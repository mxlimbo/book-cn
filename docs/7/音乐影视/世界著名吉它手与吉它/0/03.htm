Jimi Hendrix的武器

　Jimi Hendrix：一个时代的声音，由于毒品、性、腐烂的生活，而过早的离开了这世界，离开了他的Fender Tribute Stratocaster吉他。Jimi Hendrix的演奏在那个时代是绝无仅有的疯狂，和着毒品的麻醉Jimi Hendrix在舞台上晃动着身躯，手中的Fender Tribute Stratocaster 时而发出痛苦的呻吟、时而又在愤怒的嚎叫……

　　Jimi Hendrix的Fender Tribute Stratocaster是专门为Jimi Hendrix制造的“艺术家签名”系列中的一款左手琴。原木色的21品指把是它一贯的风格。琴头则是Fender传统的大波浪形状。此款吉他的音色也是继承了Fender吉他的风格——单薄明亮的色调再配上电子管音箱的过载失真，这些现在看来简单的不能再简单的配置，却在那个时代创造着神话。
