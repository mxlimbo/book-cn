剧本说明

　　《风雨兼程》是一部直接描写两性人生活的电影文学剧本。全文共分剧本正文和剧本附录两部分。其中，剧本附录是作者收集到的一些有关资料，以供普通读者更好地了解本剧的情节背景，以及更好地了解这个特殊群体。剧本中对两性人角色的性别称谓，既尊重这个特殊群体的生活方式，又考虑了普通读者的接受可能。

作者简介

　　郑军，天津市武清区人。1969年出生，1990年毕业于天津师范大学教育系，1997年开始创作生涯。现为中国小说学会会员，中国通俗文艺研究会会员，通俗文学作家。出版有长篇小说《灾难群岛》、《生命之网》，以及中短篇小说、评论文章共计数十篇。
