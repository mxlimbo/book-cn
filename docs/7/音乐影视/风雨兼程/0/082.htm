　　宋小玉的家。厨房里，宋小玉和小芳在洗菜。外面传来说笑声。

　　小芳：小玉，你怎么看出来的？我老公穿上女装比我还漂亮嘛。

　　宋小玉（指了指自己的面颊）：从颧骨上。男人的颧骨比女人高。然后再看一下有没有喉结，就能确定了。平时人们只注意头发、乳房之类的明显标记，这些细节上的区别是不太注意的。其实更多的是一种直觉。说得玄一些，我们彼此之间能够凭这种直觉在人群中认出同类。

　　小芳：太不可思议了。

　　宋小玉：不过还有最重要的一条，我知道紫芸最信任的女人就是她的妻子。她不会穿着女装和别的女人到处跑的。
