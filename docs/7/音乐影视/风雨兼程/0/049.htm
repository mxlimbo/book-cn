　　宋小玉的家。

　　写字台上摆着打开的笔记本电脑。宋小玉在读着屏幕上的电子信件。

　　镜头中依次出现墙上挂着的玩偶布饰、衣架上的女装、屋顶上垂下的风铃……

　　画外音（陌生男子的声音伴随着上述画面）：雪莲小妹，你好。前天我第一次来到你的雪莲客栈。我用了两天一夜时间，流着泪水读完了上面的每一段文字，生怕有所遗露。我今年已经快五十岁了。二十年前，我曾经进过监狱，罪名是流氓罪。当然，你一定知道那是怎么回事。从此我就失去了公职。不过因祸得福，没有退路的我进入商海，从此走向成功。现在我的个人资产已经超过了一个亿。雪莲，虽然我从来没有见过你，但我信任你。我想，姐妹们之间的这种信任是外人无法理解的。你作这样规模的个人网站负担肯定很大。我想捐出十万元钱，资助你办好它。不过，出于你能理解的原因，我希望你提供一个账号，直接把钱汇过去。这笔钱是给所有姐妹的，希望你不要拒绝。
