62

　　夜，宋小玉的轿车孤独地行驶在公路上。远处是不断升空并炸开的烟花。

　　焰花的流彩映在挡风玻璃上，宋小玉的泪水无声地在脸颊上流淌。

　　手机铃响，宋小玉掏出手机接听。

　　画外音（小霞的声音）：小玉姐，新年快乐。

　　宋小玉：姐妹们都到齐吧。

　　画外音（小霞的声音）香儿：到齐了，五个姐妹呢，还有北京的小筝。你的屋子都有点小了。

　　宋小玉：大家好好玩吧，两小时以后我就到上海了。

　　画外音（小霞的声音）：怎么，和家里人处得不好？

　　宋小玉：不，很好。只是怕村子里的人看到，让家里人难堪。你们等着我吧。
