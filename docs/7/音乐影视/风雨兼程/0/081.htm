　　站区一角，周围没有行人路过。宋小玉带着三个客人走到这里。

　　宋小玉（面对年纪最大的一位）：紫芸大姐，你和照片上一模一样。

　　紫芸：当然喽，我的照片又没用电脑修过。来，介绍一下。（拉了拉左边一人的胳膊）这是燕儿。她的琴弹得很好，这次来上海，想给小玉姐姐献上一曲。（又拉了拉右边一人的胳膊）这位嘛，你自己猜。

　　宋小玉仔细地打量着这个人。对方也歪着头，玩皮地望着宋小玉。

　　宋小玉看了看紫芸，又看了看对面的人。

　　宋小玉（惊喜地）：天啊，这是你爱人。

　　紫芸（骄傲地点点头）：是，我的小芳。
