　　宋小玉的家。大家在分吃蛋糕，坐在屋子各处聊着天。小霞：你们知道吗，重庆的阿洁快要作手术了。宋小玉：是。我听李医生说了。萍姐：真羡慕你们，赶上了好时候。我只有下辈子投胎，才能作女人喽。小芳：如果下辈子我能够投胎，我愿意作男人。周围的人惊讶地望着她。小芳：你们以为，作女人真的那么幸福。就说我母亲吧，我小的时候，她每天早上都要烟薰火烤，就是在冬天的还要洗大盆大盆的衣服，哪里有条件让自己漂亮一下。其实，我们女人从什么时候才开始有条件扮靓的？也就是这二十年吧。有了洗衣机，作饭用了煤制气，有了集体供暧，只有一个孩子需要养育。

　　屋子里的人都停止了说话，围了过来。

　　小芳：也许你们羡慕，商店里有那么多服装和化妆品是给我们女人准备的，但那又怎么样嘛，还不是把我们女人当作消费机器。我已经作了快三十年女人了，没觉得有什么光彩的。只是知道，女人永远都不会站到和男人并肩的位置上。我们只能在被男人捧着的时候才能稍微虚荣一会儿，威风一会儿。为了这短短的虚荣，还得整天提心吊胆，怕自己年老色衰。所以我想，轮到我转世投胎，大概也就是五十年后吧，我相信到那时，女人的地位也不会比今天强多少。下辈子一定要作个男人。

　　众人默然不语，各有所思。
