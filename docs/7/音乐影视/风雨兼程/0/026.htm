　　夜晚，宋小玉的家。宋小玉坐在写字台前，笔记本电脑打开着。画外音（陌生男子的声音）：你好，雪莲小姐。首先声明，我是一个普通的男性，对变性从来不感兴趣。

　　镜头上慢慢地依次出现窗台上的花瓶、墙上的饰物、枕头上毛绒绒的小熊、粉红色的窗帘、

　　画外音（陌生男子的声音伴随着上述画面）：我不讳言，如果有人在我面前穿上异性的衣服，我可能会出于习惯而感到恶心。我是在偶然间来到雪莲客栈的，也看了不少内容。我觉得我能够理解你们，并且支持你们展示真正的自己。每个人的生命都很短暂，也都有权利在不干涉别人的情况下，做自己愿意做的事情。如果有一天我偶尔遇到你们中的一位，我会愿意和她真正的自我打交道。另外我还要说一句：雪莲小姐，你的勇气令我佩服。
