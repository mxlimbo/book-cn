　　化妆台前，影楼的化妆师仔细地给身着婚纱的小霞化妆。张振鸿站在小霞的侧后方，面带震惊地望着镜子里的小霞。

　　张振鸿：你化妆的水平真高。

　　化妆师（边操作边回答）：这也是经验，其实他不是第一个提这种要求的顾客。

　　张振鸿：真的？

　　化妆师：我以前遇到过，我的同行也会遇到这种事情。所以我知道他是怎么一回事情。其实，化妆只是物质上的改变，关键是内在的东西。如果他没有一个真正女性的内心，再怎么化妆也是没有用的。

　　张振鸿：哦，不过，我不是……

　　化妆师（笑了）：我知道你不是，给你化上女妆会把人吓坏的。

　　张振鸿满脸通红，转头走向营业厅的大门，在那里徘徊。

　　张振鸿干脆走出营业厅大门，站在门口，掏出手机，拨动号码。

　　张振鸿（焦急地）：小玉吗，你还在加班吗？

　　画外音（宋小玉的声音）：你不去守着小霞的东西，在门口转来转去作什么。

　　张振鸿恍然大悟，抬头四外张望，目光停在街对面一座商厦三层的临街茶座上。宋小玉的身影隐隐在窗后闪现。

　　茶座里，宋小玉拿着手机，望着下面的张振鸿。

　　宋小玉：张振鸿，快进去吧，和我的姐妹们一起面对别人的目光。我知道你能够作到。

　　张振鸿略一迟疑，远远地向宋小玉伸出大姆指。然后转身回到营业厅。
