　　公园里的茶座。

　　一叠报纸被狠狠地甩在桌上。

　　宋小玉（边翻边说，情绪激动）：看看，这就是你们记者干的好事。喏，这个变奏酒廊，是姐妹们聚会的地方，所有营业手续都齐全。为什么这篇报导暗示它是个卖淫场所？还有这条鬼新闻，什么男女互换性器官，技术上根本就不可能。你们记者不懂，不能问专家吗？不能查资料吗？还有这个配照片的报导，照片上这个人本来是一个精神病患者，难道穿着女装的男人就是两性人？还有这篇，什么作过手术的变性人活不过四十岁，真不知道这些胡说八道的东西是从哪里编出来的！

　　张振鸿一脸无辜地望着宋小玉。

　　宋小玉：对不起，我知道这些东西不是你写的。我只是……憋了很长时间没有地方去说。

　　张振鸿：没关系，新闻记者要看到真实的东西。你的愤怒就是真实的。

　　宋小玉：我只是希望有准确的报导。要么干脆别注意我们，要么就写真实的东西。

　　张振鸿：我们一起努力吧，把真实的你们告诉社会。

　　宋小玉：不好意思，本来我不是这么凶的。不知道怎么啦。

　　张振鸿拿出采访机，摆在桌上。

　　张振鸿：咱们开始吧。
