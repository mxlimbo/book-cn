　　曲海的家。一个年轻男子正坐在电脑桌前浏览着屏幕。屏幕上是一张张女装照片。他是曲海的弟弟曲江。

　　曲海推门走了进来，脱掉外衣，看了看上网的弟弟。

　　曲海（漫不经心地）：怎么，又去看黄色网站呐？

　　曲江：这是个人妖网站，就在大陆。

　　曲海：哦？（拉了把椅子坐在弟弟身后）大陆还有人妖。

　　曲江：我上这个站很长时间了。（指了指屏幕上的字）喏，雪莲客栈，国内最有名的网上人妖社区。

　　曲海（仔细望着照片，惊讶地）：你是说，这些人都是男的。

　　曲江：当然，怎么，漂亮吧。

　　曲海（端详着）：确实不错，头一次开眼。

　　曲江：可惜，就是看不到版主雪莲的照片。

　　曲海：为什么？

　　曲江：保密呗。你看，这些是变装照，在生活中他们都穿男装，就是面对面遇到这些人你都不会认出来。可是据说版主雪莲已经作过变性手术了，在生活中也是女人，和照片上一样。所以她从来不贴自己的照片，最多只有背影。喏，这张是她在给别人化妆。瞧，闪得多严实，连一张侧脸的都没有。

　　曲海：变性手术？就是说，这个雪莲原来是男的，作过手术后才变成女的。

　　曲江：是啊，看雪莲客栈里的贴子，都把她形容得象一朵花似的，可惜就是看不到正脸。怎么，看样子你认识？

　　曲海（脸上惊喜交加）：象我的一个熟人。
