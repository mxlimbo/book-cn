7

　　画面黑暗一片，响起"吱呀"的开门声。

　　灯亮了，一间式样普通的单元住房内景出现在画面上。屋子里简单地摆着几样家具，有的家俱上还覆盖着布单。

　　一个中年妇女领着宋小玉走进屋子。

　　中年妇女：这个小区离大街远，很安静。走出去公车很多的，你就是到浦西上班也很方便。家俱也是满全的。

　　宋小玉走到里面，查看厨房、厕所。镜头跟随着宋小玉。

　　画外音（中年妇女的声音）：这套组合年前刚买的，还没有用。有线电视插头就在这里，你交了收视费，随时都可以接通。

　　宋小玉回到中厅里，中年妇女等在那里。

　　中年妇女：怎么样，满意吧？

　　宋小玉：这样吧，最迟明天上午我给您答复。

　　中年妇女：嗯，宋小姐，如果你真想租的话，要先付半年的房租，以免……

　　宋小玉：先付三个月可以吗？我刚来上海，有许多地方需要花钱，您能理解吧。

　　中年妇女：能理解，能理解。（犹豫地）嗯，还有一样，不知我该不该问。

　　宋小玉（友好地）：您问吧。

　　中年妇女：你到上海，准备找什么工作呢？

　　宋小玉皱起眉头。

　　中年妇女：哪，你看，我们两口子都是平民百姓的，把闲着的房子租出来，也就是挣个零花钱，不想有什么麻烦。

　　宋小玉（点点头）：大婶，我在大学时读通讯专业。上海这里有许多新成立的通讯技术公司，我想来试一试。

　　中年妇女（连连点头）：那好，那好。高科技企业，又是白领工作。你一定很能干。那……就祝你成功喽。

　　宋小玉（微笑）：谢谢大婶。
