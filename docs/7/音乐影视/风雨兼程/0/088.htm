　　傍晚，露天茶座，宋小玉和女装的萍姐、香儿围坐在一张靠边的桌子前。

　　宋小玉双手抱头，双肘支在桌面上。

　　香儿（压低声音）：小玉姐，这是对你的肯定呀。你能让一个女人吃醋，说明你的魅力……

　　萍姐：香儿，现在这时候别开玩笑。咱们帮小玉想想对策。

　　香儿：对了小玉姐，她不是没有直接给你挑明吗？装糊涂不就行了？

　　宋小玉（摇了摇头）：这是自尊的问题。我不想让人以为，我在事业上有成就，是靠出卖色相。

　　香儿：不行就辞了算了。上海这么大，凭你的能力还愁找不到事作。

　　萍姐（摆摆手）：小玉，我觉得还是能忍就忍。你能干到这个位置并不容易。有些人，有些事让你不愉快，就当他们不存在好了。

　　宋小玉（扶在桌子上）：我仔细想想吧。

　　香儿：唉，全国各地的姐妹都把你当活菩萨一样。她们哪知道你天天面对着什么呀。

　　宋小玉（直起身）：香儿你可别再随便对人乱说了。上次我让公安局叫去，结果你在网上发贴子讲出这件事，有那么多姐妹发来E-MAIL，让我整整回了一个晚上。

　　香儿：大家关心你嘛。

　　宋小玉：所以才不能让她们担心。这是我自己的事情，我自己能应付。
