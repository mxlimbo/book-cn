106

　　影楼营业厅。

　　男装的小霞和张振鸿坐在接待顾客的桌子旁。服务小姐走过来。

　　小霞：我们是预约的，这是预约单。

　　服务小姐：（看过预约单）好的，请问是哪一位小姐？

　　小霞：是我。

　　服务小姐（面露惊讶，但立刻恢复平静）好吧，（回身指着挂在墙边的一排婚纱）你挑一挑吧。

　　小霞（回头对张振鸿，友好地）你给参谋一下吧。

　　张振鸿（窘迫地）：不不，我没有经验。你自己挑吧。

　　小霞左挑右捡，最后用手指着一套紫色的婚纱。
