67

　　夜晚，宋小玉和李春兰走在街头。

　　宋小玉（轻松地）：拿下这个项目，今年我们组的营业额就有把握了。

　　李春兰：杭州我还是头一次来。咱们放松一下吧。呀……（手指前方）

　　宋小玉（随着她手指的方向望过去）看到什么啦？

　　李春兰：小洁！

　　宋小玉：谁？

　　李春兰：冯总的女儿。

　　宋小玉：哪一个？

　　李春兰：就是留短发的那个。

　　一个留短发的十几岁少女百无聊赖地走在人行道上。

　　李春兰：她今年才上高一。今天是星期三，冯洁不应该在这么远的地方。一定是从家里跑出来的。

　　宋小玉：走，咱们去看看。
