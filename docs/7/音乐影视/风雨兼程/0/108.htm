107

　　影楼营业厅一角，用布幔围起一个简陋的更衣室。布幔偶尔抖动一下。

　　营业厅里，张振鸿紧张地坐在桌子旁。

　　一个服务小姐走过他的身边，向远处自己的同伴作着鬼脸。张振鸿尴尬地端起一次性塑料杯，用喝水的动作掩饰自己。

　　张振鸿看着墙上的表，又对着自己的表，面上流露着焦急的神色。

　　张振鸿打开手机拨号。

　　张振鸿：小玉吗？我们已经在影楼了。你什么时候下班？

　　画外音（宋小玉的声音）：我正忙，今天要晚些时候下班。

　　张振鸿无奈地挂断，一抬头，正与一个摄影师好奇的目光相遇。两个人都不自然地转过头。
