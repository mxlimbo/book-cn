　　深夜，宋小玉的家。

　　宋小玉敲击着键盘。

　　特写，屏幕上出现一行提示：你的操作无法进行……

　　宋小玉又敲击键盘。

　　特写，屏幕上出现一行提示：你的操作无法进行……

　　宋小玉狠狠地敲击了几下键盘，站起来，走到窗边，猛地拉开窗帘。

　　夜幕中，住宅楼的外景：只有宋小玉的窗户亮着灯，宋小玉的身影孤独地站在窗边。

　　镜头逐渐推进，宋小玉双手捂着脸，肩头耸动。

　　手机铃声响起。

　　宋小玉镇定了一下，拿出手机接听。

　　画外音（萍姐的声音）：小玉，我刚刚从杂志上找到一份资料，是国家几位性学专家开的座谈会的记录。这个会专门探讨变性问题。我看了，文章内容挺不错的。我想把它用扫瞄仪扫下来，有时间放到雪莲客栈……小玉，你怎么啦？你好象哭了。宋小玉哭出了声。画外音（萍姐焦急地）：小玉，你怎么啦。宋小玉：没什么，我只是觉得很累，从心里往外觉得累。画外音（萍姐焦急地）：你挂了吧。我马上就去。
