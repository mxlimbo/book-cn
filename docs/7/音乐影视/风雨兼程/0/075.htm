74

　　冯满涛的家。

　　冯洁睡在床上。冯满涛的妻子怜爱地站在床边望着女儿。冯满涛和宋小玉站在她身后。

　　冯满涛：现在的孩子们呀，太自我中心。一点不考虑家长的感受。

　　冯妻：你还说呐，你在孩子身上投入了多少精力。

　　冯满涛：你……

　　宋小玉：冯总，没什么事我先回去了。

　　冯满涛：哦，好好，多谢你了。

　　宋小玉：没什么，正巧遇上嘛。再见了李总，（向冯妻摆摆手）再见了秦姐。

　　宋小玉转身走出门去，冯满涛送到门外。

　　冯妻站在屋子里，不满地望着门外。
