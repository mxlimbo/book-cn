18

　　画面上黑暗一片，黑暗中响起轻快的脚步声，声控计时灯又亮起来。

　　宋小玉拎着鼓鼓的塑料袋走上楼梯。她打开房门，走进去，小心地把门带上。轻轻地走向卧室，同时用目光寻找着。

　　卧室的门打开了，一个身材微胖的女人出现在门口。

　　两个人热烈拥抱。

　　宋小玉：萍姐！

　　萍姐：小玉妹妹！

　　萍姐拉着宋小玉，走进屋子，坐在沙发上。仔细地端详着她。宋小玉也仔细地端详着对方。

　　萍姐：来，让我仔细看看你。（抚摸着宋小玉的头发，抚摸着宋小玉的面孔）真漂亮，天生丽质。小玉妹妹，你知道吗。因为工作需要，我经常出国，也接触过国外的变性姐妹。（酸楚地）那时我想，我这辈子不可能活着在同胞里面找到姐妹了。

　　宋小玉（拉着萍姐的手）：萍姐，别说这些了，我们见面是高兴的事。来，（站起来，走到桌旁）我买了些吃的东西。

　　宋小玉把饮料、食品一样样从塑料袋里取出来，摆在桌上。忽然停住了手。

　　宋小玉：呀，萍姐。忘了给你买烟。

　　萍姐：吸烟是女人的习惯吗？

　　宋小玉：现在恐怕是的。象我这么大的女孩子里就有许多吸烟的。

　　萍姐：我是个保守的老太婆嘛。

　　宋小玉投之以会心的一笑。

　　墙上的电子钟奏响玩皮的音乐。

　　宋小玉：呀，十点了。萍姐，你不着急回家吧。

　　萍姐：我家只有我一个人。

　　宋小玉（愣了一下）：是这样。

　　萍姐（开玩笑地）：怎么，你现在已经是女儿身，不能留我啦？男女授受不亲哟。

　　宋小玉走过去，拉住萍姐的胳膊。

　　宋小玉：瞧你说的。明天我休息，咱们可以聊上一夜。
