　　傍晚，公园茶座。

　　宋小玉仔细包装着药品。张振鸿坐在她的对面。

　　宋小玉：得病的其实是姗姗的继母。她和姗姗父亲结婚那年，姗姗才十岁。当然，一开始肯定是家庭冷战，一直持续了四年。姗姗从小就有强烈的变性倾向。十四岁那年，有一次家里没人，她找出继母的衣裙，试着穿了一下，站在镜子前面，发现自己真得很美。从那年起，她就把这一天当作自己的生日。到了现在，相熟的姐妹也在这一天给她过生日。

　　后来，姗姗的举动被继母发现了。她以为，继母一定会暴跳如雷，但是继母很能理解她，还帮她父亲接受这个事实。上大学以后，姗姗考到了外地，继母每学期都要额外给她两百块钱，对亲戚们说是买营养品的，但姗姗知道，那是给她买女装和化妆品的。从此，姗姗的继母就成了她的贴心人。

　　张振鸿感慨地点着头。

　　宋小玉：其实，在许多姐妹那里都发生过这样的故事。这种故事听多了，让我对人生、对社会总是抱乐观态度。
