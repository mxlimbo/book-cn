　　特写：一只方便面碗端在嘴边，一双筷子迅速地往嘴里拨着面条。

　　特写：墙边的纸篓，"啪"地一声，压成一团的方便面碗被扔了进来。

　　办公桌后面，一个中年人用手帕匆匆地擦了擦嘴。

　　宋小玉坐在办公桌对面，有些紧张地抓着放在膝上的小包。

　　中年人打量着她。他叫冯满涛，私营企业主。

　　冯满涛：你学什么专业？

　　宋小玉：通讯技术。

　　冯满涛：有文凭吗？

　　宋小玉：有，但是……

　　冯满涛：（加重语气）但是？

　　宋小玉：不小心丢了，正在补办。学校那边马上就会寄来的。

　　冯满涛（打量了宋小玉几眼）：真的有文凭？

　　宋小玉（坚决地）：真的有！

　　冯满涛（往椅背上一靠）：那好吧，等你把文凭补好后再来吧。

　　宋小玉缓缓地站起来，用几乎看不清的幅度点点头。

　　宋小玉：好吧，您忙吧，我告辞了。

　　宋小玉回身走到门口。门开了，一个四十岁左右的男子走了进来，手里拿着一叠文件。男子侧身让宋小玉过去，然后走到冯满涛身边。他叫曲海，公司的业务骨干。

　　曲海：冯总，环湖小区的通讯网络施工图纸，请您审看。

　　冯满涛（用目光示意他放下）：好的。

　　曲海：冯总，刚才那个小姐是来应聘的？

　　冯满涛（不屑地）应聘？八成是来蒙事的，仗着自己长得漂亮。哼，这家公司是我一分钱一分钱搞出来的。没有真本事，光长得漂亮有什么用！
