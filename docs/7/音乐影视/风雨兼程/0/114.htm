　　列车，软卧车厢里，乘客们正在寻找座位，安放行李。

　　宋小玉焦急地沿着走廊，在一间间车厢寻找着。

　　一间软卧车厢，门打开着。镜头停在里面两个年轻男子身上。

　　宋小玉停住脚步，注视着其中的一个。

　　宋小玉：玲儿，是你吗？

　　男青年（激动地站起来）：雪莲姐！

　　宋小玉挤进门，顺手将软卧间的门关上。

　　宋小玉（焦急地）：玲儿，你真得要走吗？

　　另外一个人挡在宋小玉和小蝶中间。

　　陌生人：这就是雪莲小姐吧。不错，她们说你长得如何如何漂亮，我以为说得有水分呢。你要是到了泰国……

　　宋小玉（扳起脸）：阿龙，我知道你是什么人。

　　阿龙（傲慢地）：我是什么人？我是合法的职业经纪人。（拿出一叠文件）玲儿和我签的合同在泰国也是合法的。你自己不作可以，但不要挡住别人的前途。

　　宋小玉：我和玲儿说几句话，请你不要干扰。

　　阿龙哼了一声，退到一边。

　　宋小玉（关切地对玲儿）：我知道在泰国那个地方，这种合同确实是合法的。我只是想劝劝你，从我个人角度不希望你走这条路。

　　玲儿（流着泪）：雪莲姐，你的心意我懂。但是我又有什么办法。我不象你懂得技术，能当白领。我只能干力气活。年轻的时候凑不出手术费，将来身体条件差了又有什么用，这一生都得不到解脱。（用手揩泪）雪莲姐，相信我，在那边会照顾好自己的。出去以后我会随时上网和你联系。

　　宋小玉（无可奈何地搂住玲儿的肩膀）好吧，出去以后常联系，别让大陆的姐妹们担心。
