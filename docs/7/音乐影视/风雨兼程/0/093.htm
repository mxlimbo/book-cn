92

　　两杯泡沫红茶被放到桌上。

　　张振鸿和宋小玉对面而坐。

　　张振鸿：没想到，刚才你一眼就看出了我的身份。我还以为，自己已经进入角色了呢。

　　宋小玉（淡淡地）：也算是成功吧，在网上我就没有看出来，但见了面以后一点感觉都没有。不过，（加重语气）为什么花费精力搞这些名堂？天底下有的是新闻让你写。

　　张振鸿：其实，我给你的第一封E-MAIL，就是用真实身份写的那封，已经说了我的想法。

　　宋小玉（严肃地）：我知道，你们是一家大报，不象那些登花边新闻的小报。但我仍然不认为我们的经历有报道的必要。我们是一个微不足道的群体，不关乎国家大事。

　　张振鸿：报纸并非只报导国家大事，尤其是我们这样的报纸。（顿了一顿）你知道，人们都说，上海这个地方很现代化，甚至说这里是中国最开化的城市。我想，什么是现代化？什么是文明进步？其中很重要的一条，就是能够理解、关心社会上的每一个特殊群体。

　　宋小玉：看来你还是个社会学家。

　　张振鸿：说对了。我在大学时读的就是社会学，只是后来觉得，当个书斋里的学者没有意思，不如到报社更能接触社会。

　　宋小玉：可是媒体行业竞争得很激烈哟，尤其是在上海这个地方。

　　张振鸿（感慨地）：是，很激烈。超过我以前的想象。

　　宋小玉：所以你只有挖出轰动性的独家新闻，才能站住脚。或者说，才能暂时站住脚。

　　张振鸿满脸通红，张口结耳。

　　宋小玉平静地注视着他。

　　张振鸿看了看表，然后努力控制了一下自己的表情。

　　张振鸿（真诚地）：很遗憾，五分钟已经到了，我没有说服你。好吧，不打扰了。

　　张振鸿站起身欲走。

　　宋小玉：张振鸿。

　　张振鸿（转过身）：啊……

　　宋小玉：看来，你为这个选题计划肯定作了不少准备。我拒绝了你，你可能还会去找其他的姐妹。我想知道，如果我们每一个姐妹都拒绝你的采访，你会不会坚持把它写出来？

　　张振鸿（偏着头，不解地）：你的意思是……

　　宋小玉：我的意思是说，你会不会自己杜撰一些故事，作为你的所谓采访成果？

　　张振鸿（想了想）：你的担心我明白。我保证只写我亲眼看到，亲耳听到，亲身感受到的事情。

　　宋小玉用目光示意他坐下。张振鸿慢慢坐下，不安地望着宋小玉。

　　宋小玉：你知道，为什么那些从来没有见过面的姐妹们，愿意把她们的故事告诉我吗？

　　张振鸿：因为你是她们中的一员。

　　宋小玉：不仅如此，还因为她们知道，我了解她们的情况，是为了帮助她们，而不是为了满足好奇心。

　　张振鸿（思索片刻）懂了。

　　手机的铃声响了，宋小玉拿出手机，看了看屏幕。

　　宋小玉：好吧。我接受你的采访。不过我还有事，咱们再约时间。

　　张振鸿（兴奋地）：太好了。

　　两个人握了握手。宋小玉转过身，一边接听着手机，一边走进川流不息的人群。
