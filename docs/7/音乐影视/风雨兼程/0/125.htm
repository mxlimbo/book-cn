　　索普尔公司，经理办公室。

　　李安华和宋小玉对坐在办公桌两边。

　　李安华（严肃地）：有一件很不好的事情。我们请人作过鉴定，你提供的毕业证明是伪造的。并且我们也作过调查，北方工业大学通讯技术系九三届毕业生里没有宋小玉这个人。我希望你能作出解释。

　　宋小玉的嘴唇动了几下，没有说话。

　　李安华：你知道，我们是一家很讲信誉的跨国公司。公司的信誉建立在每个员工的信誉之上。我不想在这方面有问题。希望你能解释清楚。

　　宋小玉（喃喃地）：没有什么好解释的。毕业证书是我伪造的。但是……

　　李安华（摊摊手）：我感到很遗憾。

　　宋小玉：很抱歉，给你们添了麻烦。不过我不是有意的。这份毕业证书是我为了应聘到晨光公司才伪造的。

　　外方负责人：但是在索普尔公司完成收购以后，你并没有向我们说明情况。在新合同中有一个条款，员工必须保证自己身份证明和学历证明的真实性，否则公司有权力解除合同。

　　宋小玉：我记得。

　　李安华：很遗憾，实在很遗憾。虽然你是一个出色的技术人员，又很有管理能力，但把你留下来，对那些拥有真实学历的员工是一种伤害。并且你的行为也违犯了贵国的有关法规。我们跨国企业是很重视驻在国法律的，不希望惹什么麻烦。所以，公司决定解除与你的合同，提前发给你两个月的薪水，请你另谋高就吧。

　　镜头转向宋小玉的脸。宋小玉轻轻地咬着牙。

　　李安华：你去财务部门办理手续吧。
