52

　　公园里的茶座，宋小玉和男装的萍姐对坐着。桌子上放着两杯热饮。

　　萍姐：过几天，我要随单位的人出国一段时间。

　　宋小玉：这么忙，快到春节了呀。

　　萍姐：我去欧洲，白人不过春节的，他们还在工作。女儿在她妈妈那里。对了，你有驾照吗？

　　宋小玉：有。

　　萍姐：我有辆车，我走这段时间你开吧。上海很大，有辆车方便些。

　　宋小玉：好的。有几个离家到上海的外地姐妹在这里打工，春节不回去。我想请她们来家里聚一聚。可能还有外地的姐妹到时候也要来。

　　萍姐：你自己不回家看看吗？

　　宋小玉敏感地望着望萍姐，咬着吸管不作声。

　　萍姐：不管当初发生了什么冲突，你作女人已经是个既成事实，我想你家里的人也会慢慢接受这个现实的。

　　宋小玉：只是，我的老家是乡下，很保守，村里的乡亲们不会接受这个。

　　萍姐：所以你更应该回去看看了。家人实际上分担了你的压力。我们虽然很无辜，可他们更无辜。

　　宋小玉（点点头）：明白了，萍姐。

　　萍姐：试试看吧，先和家里比较能理解你的人沟通一下。
