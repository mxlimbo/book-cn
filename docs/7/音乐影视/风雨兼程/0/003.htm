　　鲜花的旋涡渐渐淡去，依次出现以下画面：

　　一位少女伫立在平台上的背影。少女手扶栏杆，面对着初生的太阳，长发在晨风中微微起伏。

　　熙熙攘攘的长街之上，一位职业女性身着风衣，快步走过人行横道线。

　　夕阳洒在水面上，波光鳞漓。一位身穿白裙的女人坐在水边，双手抱拢膝头。

　　夜幕，音乐喷泉广场。一位身着长裙的女性伸出右手，触摸着细细的水柱。周围变换着五颜六色的灯光。

　　以上每一个画面都要求柔美、朦胧，仅显示出人物的身体轮廓而不要清楚地展示细节。

　　清柔、和缓的音乐声伴随着这些画面。

　　以这些画面为背景，映出职、演员表。
