　　全景：晨曦中的上海市。

　　宋小玉的家，凌晨的微光照在屋子里。宋小玉和萍姐躺在床上，望着天花板。

　　萍姐：七年前的今天，我和妻子领了离婚证，（用手比划着）和结婚证一样大小，绿色的封面，一人一本，里面记载着我们曾经有过的关系，还有以后的分割协议。大概平时已经吵够了，我们分手时反而很平静。我依然很喜欢她，她真的很能干。不过如果我坚持不离，无非就是天天听她吵闹。现在想来，也许我苦苦哀求就可以不会离婚，但是我做不到，因为我爱她，就应该去满足她。

　　镜头转向宋小玉，宋小玉侧身望着萍姐，静静地听着。然后镜头又转向窗户，转向窗外的高楼、蓝天和朝霞。

　　画外音（萍姐的声音伴随着上述画面）：那天，女儿放学回来。她一看就知道是怎么回事，只是她没有说什么，默默地接受了这个事实。后来我们一直相依为命，她是个很好的孩子。我把家庭破裂的事情告诉了她的班主任，她答应照顾我的孩子，并且说我的女儿好像没有受到什么影响，成绩还可以。我心里明白，女儿的学习肯定受到了影响，本来应该更好的。女儿在生活中表现出的乐观，可能只是在自我安慰。我对不起她，也对不起她妈妈，事情本来不应该发生的，但我不知道为什么命运要这样落在我身上。
