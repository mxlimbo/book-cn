　　办公室里，曲海和方明分坐在办公桌两边。

　　中年教师（看着曲海的名片）：索普尔公司……请问有什么事吗？

　　曲海：我是索普尔公司人力资源部的。最近我们公司在整理员工档案。您知道，索普尔公司是一个跨国公司，各项制度都非常规范，其中也包括建立有非常规范的员工资料。我来是作例行调查，因为本公司有一名员工提供资料说自己是贵系九三届毕业生。听说您当时是这个班的辅导员。

　　方明：是啊，时间过去不长，许多人我还记得。只是大多数已经没联系了。你们这位员工叫什么名字呀？

　　曲海：宋小玉，一位小姐。

　　方明（皱皱眉）：宋小玉？没有这个人呐？

　　曲海（略显吃惊）：真的嘛？

　　方明：当然，我跟了他们三年，四十几个人的模样都印在我脑子里。要是说名字接近嘛，他们班上倒是有一个叫宋晓宇的男孩子。宋晓宇给老师们的印象很深。他学习成绩非常优秀，人长得也挺出色，很帅气。但是大学四年从不谈恋爱，一门心思钻研学问。这在如今的大学生里已经很少见了。哦，对不起，你问的是宋小玉。抱歉，我肯定九三届里没有这个人。曲海：方明老师，能不能给我看看那届学生的毕业合影。方明：好。（起身从文件柜里翻找，拿出一个相册，翻开一页）这就是他们那个班。曲海：哪一个是宋晓宇。方明（用手指点）喏，宋晓宇。特写：一张漂亮而忧郁的脸。曲海：这个宋晓宇，您觉得他这个人怎么样？方明：哪一方面？曲海：唔……心理上。性格上。方明（略略回忆）他一直很忧郁，除了学习上的事，很少和老师交流，和同学们往来也不多。有的老师觉得他有心理疾病，也有的觉得他就是那种内向性格。怎么，您这次来不是调查那个什么……宋小玉的吗？
