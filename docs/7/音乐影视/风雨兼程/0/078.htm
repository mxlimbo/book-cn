77

　　冯满涛的家。

　　客厅里布置成生日宴会的模样。冯满涛、冯妻和宋小玉围坐在冯洁身边。

　　冯洁吹熄了蜡烛。

　　冯妻（切着蛋糕，热情地）：小玉真是个好人。冯洁天天都提起你呢。

　　冯洁：小玉姐就是好。（冲着宋小玉）原来我以为，象你这么大年纪的人都很市俗，很没有趣味的。

　　宋小玉（玩笑地）：我这么大？我今年才二十八岁呀？

　　冯洁：才二十八岁，二十八岁不是很吓人吗？

　　宋小玉（作出一副无可奈何的样子）看来我已经老喽。

　　冯洁过来搂着宋小玉的肩膀。

　　冯满涛望着女儿和宋小玉。

　　冯妻：小玉呀，你一个人在上海生活，很困难吧。女孩子出外打工不容易哟。对了，有男朋友了吗？（声音渐渐减弱，直致无声。只剩下画面上冯妻热情的面孔和不断开合的嘴。）

　　宋小玉陪着笑。
