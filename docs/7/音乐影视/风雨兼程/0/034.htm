33

　　饭店单间里。

　　香儿充满新奇感的脸。

　　局促不安的萍姐。

　　一个女服务员站在宋小玉的座位旁，手里拿着点菜单和笔，并没有注意香儿和萍姐。

　　女服务员（问宋小玉）：你还要点什么吗？

　　宋小玉（看了看点菜单）：先要这些吧，不够再点。

　　女服务员：好的。（转身退出）

　　香儿（凑到宋小玉身边，低声地）：她怎么一点都没注意我呀？

　　宋小玉：傻妹妹，她要是注意你，不就说明你有问题吗。

　　香儿不好意思地一笑。

　　香儿：对了，有点麻烦（把嘴贴在宋小玉耳边，嘀咕着）

　　宋小玉（笑了）：萍姐，你看着东西，我带香儿去洗手间。
