　　宋小玉的家，宋小玉和张振鸿站在窗前，眺望着上海的夜景。

　　宋小玉：希望你能理解她们。如果一个普通的女孩子这么喜欢打扮、化妆，你可以说她们幼稚，肤浅。但我的姐妹们不一样。她们平时穿着男人的衣服，夸张地讲，就象受刑一样痛苦。只有在换上女装的短暂时刻，才能感受到自己的真正自我。那种解脱感难以言传。所以她们在扮装时才那么认真，因为她们非常重视这段短暂的时光。

　　张振鸿：嗯，我不能说马上就会理解这种感受，但我一直在争取去理解。

　　宋小玉：你已经作得很好了。你看过以前姐妹们照的婚纱摄影吧，都是形单影只的。其实她们希望能够有个男性站在身边，衬托一下她们的美。但这样的人总是找不到。
