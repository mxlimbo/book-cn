　　晨光通讯公司。午休时间，员工们在吃午餐。

　　宋小玉坐在电脑前，右手移动鼠标，左手在食品袋里取食。

　　周围的声音渐渐沉寂下去，仅余朦胧的画面。一个清晰的画外音响了起来。

　　画外音（陌生男子的声音）：你好！雪莲姐，我是一颗痛苦的小流星，在社会偏见组成的大气层中向下坠落、燃烧。在我的前面看不到一丝希望、只有等待着粉身碎骨。我比任何人都坚信，世上绝对没有什么上帝！如果它存在，为什么要制造出我们这些苦命人？难道就是为了作可恶的游戏吗？

　　宋小玉在桌面上的一张餐巾纸上擦了擦左手，敲起键盘。

　　画外音（宋小玉的声音）：你好！小流星，欢迎你来到雪莲客栈。把这当成你的秘密家园吧，有什么痛苦可以和姐妹讲一讲，大家一起分担。

　　李春兰走到宋小玉的隔间外，想说什么又停了下来。

　　宋小玉抬头看到她，把桌上的一叠文件递了过去。

　　宋小玉：整理好了。

　　李春兰接过文件，向宋小玉笑了笑，转身走了。

　　宋小玉继续敲打着键盘。
