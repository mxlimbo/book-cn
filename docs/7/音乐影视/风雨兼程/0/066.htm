　　一连串叠映的画面：街道上的汽车、灯火通明的高楼、明亮的路灯、机场、起飞的客机。

　　画外音（陌生男子的声音伴随着上述画面）：我已于昨天晚上顺利回到北京。这次上海之行时间虽短，但姐妹之情令人终生难忘。我遇见的每一位姐妹都是精英，人人都是那么善解人意，与我们在社会上见到的勾心斗角、尔虞我诈完全相反。是不是我们有太多不被人理解的遭遇，所以更珍惜心灵的交流？昨天与雪莲妹妹分手后，我感到非常失落。安检后我一个人孤单地坐在一个角落里，眼泪不由自主地掉下来。我在飞机上望着上海的万家灯火，祝愿我们的雪莲小妹，祝愿每一位不幸的姐妹都能交上好运。
