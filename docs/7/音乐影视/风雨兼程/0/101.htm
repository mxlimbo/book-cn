　　原晨光公司的办公地点。

　　两个员工正在墙壁上布置索普尔公司的英文标示。

　　会议室里，一个中年白人经理正在主持会议。镜头从一个个中方高级员工的脸上划过，大家表情各异。宋小玉和曲海都在其中。

　　画外音（外籍经理用中文说道）：大家好。我叫李安华，暂时担任索普尔公司中国分公司的负责人。中国是未来世界上最大的通讯市场，索普尔希望能在这个市场上有所作为。

　　镜头转向李安华，并固定在他的身上。

　　李安华：冯满涛先生和各位员工白手起家，建立了一个不错的起点。不过，企业要想进一步发展，需要先进的管理机制。索普尔公司拥有这种机制。现在各位需要一边工作，一边适应这些新的机制。
