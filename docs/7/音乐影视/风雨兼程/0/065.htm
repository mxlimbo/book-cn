　　宋小玉的家。

　　宋小玉和小霞坐在沙发上，其他几个姐妹围在她们身边。

　　小霞（神神秘秘地）：小玉姐，我和你说件事。

　　宋小玉：唔？

　　小霞：我有一个远房的哥哥，在外企工作，人很可靠。喏，这里有他的照片。

　　小霞从背后取出一个小相册，把它打开。宋小玉伸过手，把它合上了。

　　小霞：小玉姐……

　　宋小玉：小玉姐谢谢你。不过，我没有这个想法。

　　两性人甲：小玉，其实这也是大家的心愿，都希望你能作个圆满的女人。

　　宋小玉（黯然地）：谢谢大家的关心。不过，我现在已经很满足了。

　　两性人乙：为什么？

　　宋小玉：如果一个男人不爱我，我没有必要嫁他。宋小玉又不是非出嫁不能生存的人。如果一个男人爱我，我更不应该嫁他。不应该把非议和白眼分给他一半。毕竟我们生活在中国。

　　小霞：小玉姐，你说得太惨了些吧。如今你"自由"了，却要独身一辈子。

　　宋小玉（抚摸着小霞的头发）：姐姐没有说要独身一辈子哟。将来有一天，人们不再用看待怪物的眼光来看待我们，不再用"人妖"这个词来称呼我们，我就可以光明正大地出嫁啦。

　　小霞：那要多少年？三十年？五十年？

　　宋小玉（笑中含泪）：三十年以后，我还不到六十嘛。

　　小霞搂着她，把头埋在她的肩膀上。众人默默无语。

　　宋小玉：算了，大家是来过年的。为什么提这些不愉快的话题。来，别浪费这段时光。
