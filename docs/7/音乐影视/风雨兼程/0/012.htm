11

　　特写，一张毕业证书充满整个画面。毕业证书的封面被打开，一张黑白照片贴在证书内页右下角。那是宋小玉变性之前的女装照片。

　　冯满涛右手拿着毕业证书，审视着。

　　冯满涛：我以前和你们系有过技术合作关系。当时，申建教授给我很大帮助。他现在身体怎么样了？

　　宋小玉：申教授身体不好，主要是肾功能不好，已经提前退休了。

　　冯满涛：吴副教授呢？吴美珍教授。

　　宋小玉：现在是教授了，在日本作访问学者。

　　冯满涛（把毕业证书递还给宋小玉）：好吧，明天你来上班吧。我们是私营企业，重水平不重文凭。

　　宋小玉伸手正准备接过毕业证书，听到这句话，稍显惊讶，手停住了。

　　冯满涛：当然，也不等于随便就信任一个人。象我们这种高科技企业，员工的技术能力就是最重要的财富，我不希望有假货。这一点希望你能明白吗？

　　宋小玉点点头。
