　　索普尔公司，李安华的办公室

　　李安华：宋小玉小姐，今天和你谈一件重要的事情，希望你保守秘密。

　　宋小玉：我明白。

　　李安华：我们公司进入中国市场，一个重要的原则就是充分地本土化，这其中也体现在管理人员的本土化上。这个分公司我只是临时负责。现在半年的导入期已经快结束了。我们准备从公司内部寻找一位经理人员。你是其中的一个候选人。

　　宋小玉全神惯注地听着。

　　李安华：我对你的要求就是写一份详尽的报告，站在未来的总经理的角度，分析一下索普尔公司中国分公司成立以来的业务情况，并且拟定将来的业务发展方向。报告出来以后直接交给我。

　　宋小玉：好的。
