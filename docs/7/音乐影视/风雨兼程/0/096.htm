95

　　晨光公司。餐厅里布置得一片喜气，一排排桌子上摆着食品和饮料。员工们穿着礼服，挤在餐厅里，每个人的手里端着杯子。

　　冯满涛站在讲台上，手里也拿着一只杯子，面向全体员工。

　　冯满涛：晨光公司的全体同仁们。今天是公司成立一周年的日子。昨天，统计报告已经搞出来了，是没有任何水分的统计报告（众笑）。它显示，在上海地区的同类市场上，晨光公司占领了百分之三十五的份额！

　　员工们抱以一片掌声。

　　镜头从一个个员工们的脸上划过，停在宋小玉的脸上。宋小玉全神惯注地听着。

　　画外音（冯满涛的声音）：将来，当你们回想起这段时光，你们应该骄傲地说，你们创造着历史，而不仅仅是继承别人创造的历史！

　　又是一片热烈的掌声。
