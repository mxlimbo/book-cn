　　公安局，彭越的办公室。

　　彭越：这次请你来是想让你帮个忙，协助我们调查一个案子。

　　宋小玉（放松地）：您别客气。

　　彭越（笑了）：来，你看。

　　彭越将桌上电脑的监视器转了一个角度，冲着宋小玉。监视器上是一个年轻男子的照片，相貌清秀，头发浓密。

　　彭越：这是个被通辑的在逃犯，名叫许庆华，二十四岁，福建泉州人。被控在一家私营企业任职期间，利用身为财会人员的便利，卷走三万美金。案发后在逃。当地公安部门后来调查了解，许庆华从小在女孩子堆里长大，非常喜爱穿女装，平时生活方式也相当女性化。我想，你应该熟悉他的心理，或许，能够帮助我们找一些线索。

　　宋小玉（想了想）：关于这个许庆华，你这里有没有不需要保密的文字资料。

　　彭越：（拿出一份资料）：你看。

　　宋小玉接过，仔细看着。

　　宋小玉：这样，我现在就回家去，从我的电脑上给你传一个黑名单。

　　彭越：黑名单？

　　宋小玉：这一两年，国内有一些整形外科医生看中了变性手术的高额利润，都来作这一行。他们不仅手术质量很差，而且提供违规操作，靠这个来吸引力非常想保密的姐妹们。结果，作了手术的姐妹们吃了亏都没有地方申诉。但是她们会在圈子里把自己的遭遇传开。这个黑名单就是全国各地姐妹们提供的。

　　彭越：你是说，许庆华拿这笔钱去作了变性手术。

　　宋小玉：如果从案发时间来看，许庆华现在可能已经是个女人了。（指指屏幕）你们如果按照这张照片去找，肯定找不到的。不过，从我提供的名单上，可以找到那个为她作手术的医生。并不多，全国不超过五个人，你们可以一个个去查问。
