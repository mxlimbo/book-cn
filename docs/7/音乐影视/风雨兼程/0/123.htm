　　索普尔公司的冷餐会。

　　宋小玉正和几个女同事一起用餐。曲海端着一杯酒走了过来。

　　曲海：大家好。（坐在宋小玉对面）你们知道吗？我一直对小玉有好感。今天借着酒劲，就说一说吧。

　　李春兰：算了吧，你儿子都能打酱油了，还给小玉献殷勤呐。

　　曲海（夸张地）：那又怎么样，没有色胆，还不允许有色心吗？

　　李春兰：那你说说，为什么有好感，是不是因为小玉长得漂亮？

　　曲海（莫测高深地）：漂亮？不，我更欣赏她内在的素质。

　　李春兰（起哄地）：那更要说说了。

　　镜头对准宋小玉，她冷静地望着曲海。

　　画外音（曲海的声音）：我欣赏她精明、干练、果断，她的思维和行动很有逻辑性，还有一种天生的技术才能。

　　镜头转到曲海的身上。

　　曲海：只是我非常奇怪，这些都是男人才有的性格特征呀。

　　宋小玉（淡淡地）：哪条法律规定女人不能有这些性格特点？

　　曲海（眯着眼睛）：可能是自然法则吧。

　　曲海转身走开。

　　李春兰（不解地）：他怎么神神秘秘的。
