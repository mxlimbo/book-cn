26

　　小区工地，宋小玉在分检材料设备。

　　冯满涛穿着工作服走了进来。宋小玉看到他，吃了一惊。

　　宋小玉（直起身）：冯总，您怎么亲自来了？

　　冯满涛：我怎么不能来？公司开办的时候只有三个人，你们干的活我都干过。（摆弄了一下施工器材）怎么样？搞这种野外作业不如坐办公室舒服吧。

　　宋小玉：冯总，我以前并不是坐办公室的。

　　冯满涛（赞许地望着宋小玉，点了点头）：这样，公司接下一个项目时，你独立设计一份方案，直接送给我。记住，不要对别人讲。

　　宋小玉：我明白。

　　冯满涛扭头走了。宋小玉望着他的背影，若有所思。
