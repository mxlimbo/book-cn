　　大商场内部，宋小玉和香儿的身影随着电梯缓缓而上。

　　宋小玉：女装本来不是为我们制作的。不过你很瘦，应该比较好办一些。

　　宋小玉和香儿站在女装柜台前。

　　宋小玉（指点着服装，压低了声音）：正常发育的男孩肩膀会比女孩要宽一些，而且比较平。如果买带垫肩的女装，可以把垫肩拆掉，这样肩部才比较合适一些，而且袖子的长度会更适合你的臂长。

　　服务员（走过来）：两位要些什么？这里有最新的样式……

　　宋小玉：谢谢，我们自己看一下。

　　宋小玉和香儿站在另一个服装展示区里。

　　宋小玉：虽然柔软的丝裙看上去比较潇洒，但我们的骨盆比真正的女性窄，最好买材料稍微硬挺一些的短裙，然后选臀围比较宽，而且成型的超短裙作为衬裙，再套上连衣裙，效果会更好一些。
