前言

　　尽可能的做最真实的描写，是我作家生涯的主题。

　　虚构中的真实，和不虚构中的事实，是与现实完全不同的现实。

　　有假的事情也一定会有真实的事物。这真的是相当为难的问题。如果要以浅淡的话语，来为真实下定义的话，可能哲学家们要争论好几百年，所以为了要避免这么麻烦的事，我个人考虑到"有关漫画故事的真实"的是，它不就是共通感情的最大公约数吗？

　　如果以虐待小狗狗来比喻的话：大部分的人，一定会觉得"好可怜喔！"。但是，很令人意外的，也有些不同的声音："活该，我最讨厌狗了！"。不过，我想对于虐待小狗这种场面，人类最大公约数应该是说："好可怜！"吧！

　　所以，"认为被虐待的小狗很可怜"这个真实的事实，可以从这里看到。

　　为了要得到大数人的共鸣，漫画家必须将这些事实，当做许多故事中的主干。

　　话说回来，一个十五少女的真实，通常对于一个三十五岁的女人而言，并不算是真实。

　　终于，要切入正题了。

　　我想到要画《爱情白皮书》的动机是，三十岁的我突然查觉到，如果现在我再不画二十岁女孩的真实的话，以后我再也画不出来了。

　　二十岁女孩的真实。――还有，对差不多的在这种年龄左右的女孩而言，"反正，我就是喜欢啊！"这句话，在这个世界上没有任何存在的东西可赢得过它。

　　不论是父母的忠告，社会的规范，友情的说服，都不足以和"反正，我就是喜欢！"这句话的能源抗衡。那不纯熟的个性及压倒性的威力是只在这个时期会存在的。

　　看着心里憧憬却未曾谋面过的明星，光是站在舞台上就可能晕倒，这种威力实在令人难以置信。这可以说，从在披头四时代便开始继承而来的年轻人的真实。

　　园田成美，就是在最大的公约数之中的女孩子。

　　她和她那稚拙的恋情，可能就是要真正踏入恋爱前的准备阶段。

　　但，这也是二十岁时最真实的恋情。

一九九二年四月 柴门文
