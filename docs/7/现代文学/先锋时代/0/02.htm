序

　　一次在武汉大学讲座时，主办者自作主张地在海报上写道："丁伯慧，青年作家，代表作：《先锋时代》。"我有些哭笑不得，故而，在这儿，我首先声明，这只是一篇习作，是不能代表我的主流创作方向的。

　　这篇小说写的是"另类"，或者称之为"先锋"，《小说选刊》选载这篇小说时，曾有一段评语："《先锋时代》出自青年作家丁伯慧之手。（注意"青年"两个字）这里得先说一声，宏观地看，断代史上是没有所谓'先锋时代'的，它只在表述当代人中行为方式具有'新'与'异'特点的那一个青年群体时适用，可褒可贬，因为'先锋'不一定等于'先进'。这篇小说将属于这个群体的知识青年--曹丽文与郁聪的生活和恋爱方式，作了生动而细腻地展示，相信读者据此对这个群体有多一些的了解，并从这个现代故事中品了一些古典风味来。"

　　我觉得编辑可能太过仔细了。其实，所谓另类，在我看来，不是人的类分，我宁愿将其看作是一个人的一个阶段，或一个层面的生活。文字可能有些搞笑，我也恰好又是七十年代出生的，属于"另类"似乎理所当然。可我本人看起来却偏偏像是一个严肃的人，"另类"们是很少愿意接近的。我写这篇小说的初衷，是"美女作家"们正风行中国的时候。那时候，喝采声和叫骂声此起彼伏，交相呼应，很是热闹。我总觉得，喝采者也罢，叫骂者也罢，对于"另类"，都有些屈解，故而萌生写几个真正"另类"的想法。我写另类写得很苦，将近两个月时间，才写完这篇小说。一出手，即遭到一位大刊编辑的痛斥，退稿，然后谓之"粗浅"！然而接着，《大家》却看中这篇小说，发在首篇，还配了一幅整版的、并不"另类"的作者照片。只是小说被截去了结尾部分，让悲剧变成了喜剧，满足了中国人大团圆、留点希望的美好心理。紧接着，《小说选刊》也选载了，仍然不愿意要我原来的结尾。小说出来后，果然遭到了许多反对，说怎么没写完？郁聪与曹丽文似乎应该有个结局。也有人这样说：另类原来是这样的！唯有湘谭大学法语系的一位女生在发给我的电子邮件里说："幽默风趣的语言背后，我读到了一种辛酸。"我真为之感动！

　　然而，这时，我仍有些不太心甘，现在的结果似乎与原先的想法相去甚远。于是，E书准备将它做成电子版的时候，我突然萌生出一个愿望，将截去的部分补上来，再听听读者的声音，是骂声多，还是赞扬声多？也罢，骂也罢，赞也罢，都早已远远超出了我以为习作的初衷。故在电子版的前面，写出这么一堆不咸不淡的题外话，是为序。

丁伯慧
2001年4月18日
于汉口家中
