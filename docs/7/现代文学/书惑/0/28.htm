27

　　鲁晓峰和林林打上"面的"，穿过不知多少胡同，拐过多少道弯，来到W学校印刷厂。这个学校是市重点学校，国家投资建了新校舍，旧楼就改成印刷厂了。

　　林林选择这个厂印制《不生情》有两个理由，一是校办工厂是国家扶持的企业，减免许多税收，成本低，价格便宜；二是这个厂虽然是校办工厂，但印刷质量却很高，制版、印刷、装订等各道工序，都有高薪聘来的国营大厂退休的老师傅把关。这些老师傅在原单位兢兢业业一辈子，工作责任心极强，就是你让他们凑合着干，他们都不会。

　　鲁晓峰这趟来，是跟厂长谈价格和印刷日期的。走进大楼，就听见眶当眶当的轰鸣声，他们没立即到二楼厂长办公室，而是到工房转了一圈。鲁晓峰看到，这个校办厂的规模着实不小，制版车间有电脑照排机、电子分色机、进口双面晒版机；印刷车间有人民机器厂的08单色机、小轮转机和全裁米利手续机；装订车间分机器装订和手工装订两种。各道工序都在忙碌，没有闲聊闲逛的现象。

　　"这哪儿是校办工厂啊，分明是一个国营中型企业的架子！"参观出来，鲁晓峰感叹地说。

　　"我问了一下，全厂有100多人。"林林说。

　　"厉害！"

　　"学校的奖金都是从这儿出来的。"

　　"奖金少不了！"

　　他们来到二楼厂长办公室，见到了女厂长。这位厂长姓刘，原来就是这个学校毕业的，留校当了几年语文老师，工作很努力，教得也不错，可由于学历不高，评职称时总吃亏，所以一狠心打了个报告，宁愿到校办工厂当普通工人。校长就遂了她愿，把她放在厂长办公室做主任。她是发狠出来的，干起工作来非常努力，在很短的时间内，不但掌握了印刷工序的各关键环节及其关系，还拉上了许多客户，许多出版社、广告公司，有了活儿首先考虑的就是她。这除了这个厂的印刷质量和价格优势外，女厂长知识女性的成熟老练、办事得体等个人魅力，也不可小视。鲁晓峰见到她，第一眼就感到这不是个省油的灯，精明过人。

　　"刘厂长，我们老板来啦！这是刘厂长。"林林为双方做了介绍。

　　"嚯，你们厂可够厉害的！我去过那么多印厂，校办工厂像你们这么大规模的，我还是头一次见着呢！"鲁晓峰先是一通儿恭维。

　　"我们的活儿还行，质量不会出什么问题。你们看过我们印的书了吗？"刘厂长一边说着，一边从书架上抽出几本样书，递给鲁晓峰。

　　鲁晓峰接过来看了看，说："不错，印刷烫背都挺规矩。"

　　"你们打算印多少？是不是还是那本儿《不生情》？"刘厂长将目光移向林林。

　　"还是那本儿。"林林答。

　　"你们定下印数了吗？"

　　"至少30万！"鲁晓峰说。

　　"有订数了？"

　　"问题不大。"

　　"你们可别冒险！你们进来的时候，看见楼道里堆着的那么多书没有？那就是一个研究《易经》的干的。当时他说看好了市场，一下子就印了10万，可只批出去5000多，现在都好几个月过去了，还扔在那儿呢！"

　　"付款了吗？"鲁晓峰问。

　　"纸钱倒是先付了，印刷费恐怕就没日子了。"

　　"那怎么办？"

　　"能怎么办？我们找了几个二渠道的书商，三零折甩给他们，让他们去批，能收点儿就收点儿。"

　　"那作者干吗？"林林问。

　　"他不干又怎么样？欠我们那么多钱不给，我们可不就拿书来抵呗！"

　　"倒也是个办法。"鲁晓峰沉思了一下说。

　　"你们决定印了？"刘厂长双眼一弯，笑出一对酒窝。

　　"决定了。"

　　"有准印单了吗？"

　　"有了。"

　　"30万？"

　　"30万。"

　　"那得要多少令纸呀！你们得先付纸钱。"

　　鲁晓峰稍停了一下，说："能不能后付？交货后一个月内付清。"

　　"那不行。我们没那么多钱去买纸，现在纸价儿涨得厉害。"

　　"交货后一个月付现金，不要增值税发票。"

　　"那倒是省了一笔税金。可没钱买纸也白搭！"

　　鲁晓峰左右看看没其他人，咳了一下说："我们公司有个规矩，谁帮了忙都亏待不了谁。完了事儿，给您5000顾问费，怎么样？"

　　刘厂长一笑，说："我倒没什么，没纸你让我怎么印？"

　　"这样吧，我想办法筹措3万块，其余的您先垫上，表明我有诚意跟您合作，怎么样？"

　　"3万太少了，怎么也得10万！"

　　"刘厂长，您怎么这么死心眼儿呢？我不是说交货后一个月内保证付清吗？"

　　"唉，看你们也真够难的。那你就先把这3万打过来吧，我们库房里还有点儿存纸，是金城凸版纸，借你们先用。可其余的款，可得按你们说的，一个月内付清啊！"

　　"那没问题！"

　　刘厂长又问："你们接受什么价格？"

　　"包印张。"鲁晓峰与任何一家印刷厂合作，都是让厂家包印张，这样比较干脆利落。

　　"多少钱一个印张？"

　　"您说多少就多少。"

　　"一毛九一个印张，行吗？"

　　"太高啦！"鲁晓峰瞪起了眼。

　　"那你们说多少？"

　　"一毛三一个印张。"

　　"太低了，干不下来！"

　　"带封皮一毛四一个印张。"鲁晓峰退了一步。

　　"一个封皮相当于一个印张，您只添一分钱？您这个老板算得也太精了！"

　　"我这书量大呀！一下子30万，上轮转多过瘾，您这产值大，往里摊的费用就多啦！"

　　"您可真是行家！这样吧，一毛六一个印张，干就干，不干就算！"刘厂长还是笑。

　　"唉，好吧！到时候您可得按期交货，耽误我们发货，印刷费可是要扣除的！"

　　"这您放心，我们有连续加班一个礼拜的先例。"

　　"小心工人告您违反劳动法！"

　　"给加班费呗！"

　　"好，痛快！"

　　这时候，办公室的门被砰地撞开，紧接着，一个风姿绰约的女人大大咧咧地走了进来。

　　"神经病！"刘厂长笑骂道。

　　"怎么，又接到肥活儿啦？"女人看看鲁晓峰，似乎是愣了一下，又笑着说，"见过，见过！"

　　鲁晓峰也看出来了，原来是耸胸女人余泳，他的脸不由得腾地发起烧来。

　　"你们也是来找傻姐印书的？"

　　"谈点儿业务。"鲁晓峰不愿多扯，想马上走，就对女厂长说，"刘厂长，就这样，明天我让林林把支票送过来，您就立即安排印制吧！"

　　"行，你放心，我把你的活儿往前排就是了。"

　　鲁晓峰和林林刚想赶紧走，余泳不紧不慢地说："等等，什么活儿这么急？"

　　"你管人家什么活儿呢！"刘厂长笑着说。

　　"我的活儿还急呢，我可是老客户！"

　　"你从来就没不急的时候！人家印的是《不生情》，著名的小说，赶着发货呢！"

　　"哟！《不生情》让你给抓过来啦？"余泳瞪大了眼睛看着鲁晓峰。

　　鲁晓峰在社会上什么样的人都见过，就是不会跟这种女人打交道，支吾了一下就要走。

　　"你能不能给我张名片儿？"余泳笑着，双眼火辣辣地望着鲁晓峰。

　　鲁晓峰的直觉是这女人是祸水，就说："对不起，名片用完了。"

　　"把我这张给你吧！"刘厂长热心地把鲁晓峰进门时给她的名片交给了余泳。

　　余泳看了看名片，对鲁晓峰说："鲁老板，过几天我跟你联系。我想和你合作发行《不生情》，保证你的利益不比你自己干少，而且收款由我负责。"

　　"鲁老板还不知道吧，全国有好多大书商，都拜倒在余泳女士的石榴裙下啦！"刘厂长笑呵呵地说。

　　鲁晓峰感觉这气氛不对劲，就含糊地支应几句，溜之大吉了。

　　在楼门口，他遇到了张鹤鸣，问他是不是也印书来了。张鹤鸣说，他是来进点儿低价书的。

　　"《不生情》我准备开印了，你能包北京地区吗？"鲁晓峰问。

　　"包多少？"

　　"怎么也得3万吧！"

　　"包不了。第一我资金不足，第二我的发书点儿只限于南城，大多数又是国营书店，包个三五千的还差不多。"张鹤鸣态度很诚恳。

　　"那就到时候再说吧，书出来我通知你。"鲁晓峰信心十足。
