【纲12】

　　【纲12】当天晚上，黑山堡首领刘广龙回到大队部，做了一段相当重要的政治笔记。这段笔记对于他今天的政治行为是个很好的说明。而后，他被自己政治上的杰出表演所鼓舞，又对女人发挥了英勇善战的能力。这个能力第二天需要六个以上的鸡蛋来养护。

　　〖目〗刘广龙来到大队革委会办公室，在幽静的灯光下拿出专记革命的笔记本做记录。这一天的笔记绝大多数是广龙符号，仅有的几个汉字代表黑山堡革命委员会的主要成员。他实际上是在分析钱爱孔加入黑山堡革命委员会之后的权力结构。倘若把他隐秘的记录简洁地翻译过来，我们得到的是非常出色的权力平衡术。

　　看得出，刘广龙对统一钱爱孔的成功（他已确信这个成功）十分看重。过去，在黑山堡革命委员会中，他同时依靠两股力量，一股是罗元庆、罗燕、冯二苟，一股是禹永富。一个人有两只脚，就要比一只脚站的稳得多。现在，革委会内又多了钱爱孔，他就同时依靠三股力量了。一个鼎两足还站不稳，三足鼎立就万无一失了。

　　我们看到刘广龙对未来的黑山堡革委会的权力平稳结构做出的第一个公式就是：──钱爱孔──罗元庆、罗燕、冯二苟──禹永富──。当然，首尾两个破折号应该联结在一起，这三股势力其实构成一个三角形。在这个公式下，我们又看到三个小鲍式。钱爱孔──禹永富加罗元庆、罗燕、冯二苟；这一公式表明在排斥外来户钱爱孔时，罗元庆、罗燕、冯二苟和禹永富可能有共同立场。禹永富──罗元庆、罗燕、冯二苟加钱爱孔；这一公式表明对待禹永富这样的原保守派人物，罗元庆、罗燕、冯二苟和钱爱孔这些革命派会有共同立场。罗元庆、罗燕、冯二苟──禹永富加钱爱孔；这一公式表明，当罗元庆、罗燕、冯二苟沼嗌势力飞扬跋扈时，外来的钱爱孔有可能站到禹永富一边，禹永富也可能靠近钱爱孔，联合起来对付罗元庆、罗燕、冯二苟的力量。

　　刘广龙将这三派力量一一分析之后，又将自己放了进去。于是乎，就又出现了如下三个公式。刘广龙加罗元庆、罗燕、冯二苟加禹永富──钱爱孔；这一公式表明，当他和钱爱孔冲突时，黑山堡革委会原有的人马都会支持他。刘广龙加罗元庆、罗燕、冯二苟加钱爱孔──禹永富；这一公式表明，当他要打击禹永富时，罗元庆、罗燕、冯二苟与钱爱孔也都会站在他的一边。刘广龙加禹永富加钱爱孔──罗元庆、罗燕、冯二苟；这一公式表明，当他想要对罗元庆、罗燕、冯二苟这股势力加以限制时，禹永富、钱爱孔都会坚决支持他。

　　刘广龙一定觉得这种三大势力的分析还是粗线条的。罗元庆、罗燕、冯二苟根本不是铁板一块，他们三个人在黑山堡政治权力中也有争夺和矛盾。罗燕虽然和罗元庆是叔侄关系，但是罗燕是自己金刚钻捅出来的女人，说来说去，可能更是自己的亲信。冯二苟虽然跟着罗元庆、罗燕跑，那是因为罗元庆、罗燕经常打他刘广龙的旗号指挥他，只要自己一伸手，冯二苟就是他的看家狗，何况冯二苟还是刘红的表舅。这样想来，黑山堡的政权就再稳当不过了。

　　钱爱孔以后又是自己的妹夫，这又多了一层联系。

　　自己真可以高枕无忧了。

　　他写完革命的笔记，心情非常富裕。又打开专记生产和专记建设的笔记本看了看，写了写，对黑山堡新村建设的平面图又做了一些描绘。然后，又打开专记女人的笔记本，看了看一年来的累累战果。

　　最后，他兴致大发，收起笔记本走出房门，踏着月光推开了广播室。蓓蓓一见他进来，就娇嗔地一笑。温顺的羔羊一年来已经变得越来越活灵活现了。他把她睡了。睡完了，还有余兴，便吩咐蓓蓓去把罗燕叫来。他有重要的事要和罗燕个别谈话。蓓蓓瞟了他一眼，没说什么，低着头踏着月光叫人去了。

　　罗燕很快来了，刘广龙在办公室里又和她进行了个别谈话。过去的大队部现在的革委会大院早已把木栅栏门改成了大木门，办公室的窗户也都装上了窗帘。夜里没有别人，只有蓓蓓住在院内的广播室里，刘广龙量她不敢有什么动静。他又把罗燕仔细地梳理了。这个梳理建立在政治分析的基础上，便尤其有一种特殊的味道。当一个女人既是自己的女人，又是一个有点游离于自己之外的独立力量时，梳理起来，就有一种双重的占有感。

　　他不仅在占有一个女人，还在占有一种力量。
