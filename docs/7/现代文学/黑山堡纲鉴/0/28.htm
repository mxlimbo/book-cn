【纲13】

　　【纲13】广龙三年春节，东山大队正式并入黑山堡大队，成为黑山堡大队东山分队。至此，大黑山堡王国诞生。这一天，新黑山堡大队举行了空前盛大的庆祝会，山上山下红旗招展。在这一天，还举行了钱爱孔与刘小妹的结婚典礼。

　　在这个胜利的喜庆日子里，刘广龙没有一点头晕脑胀。他在想象黑山堡王国新的蓝图。这一天还传来新的消息，一大批知识青年和下放干部将从大城市来黑山堡插队落户，其中包括舞蹈学校的部分师生。有人预言，这些知识分子和干部的到来会使黑山堡局势复杂化。刘广龙却添了兴致。他其实不喜欢高枕无忧；他喜欢披荆斩棘、开天辟地。
