【纲３】

　　【纲３】兼并西山大队以后，刘广龙在黑山堡的威望如日中天。革命委员会将战争中的军事化转变为和平建设的军事化，黑山堡百姓凡十岁以上六十岁以下者，都以团营连排班的军事编制组织了起来。每天金号手的喇叭经过广播站的放大嘹亮吹响时，各路兵马就紧急集合奔赴战场。一半人千军万马地开始拆除旧堡墙。另一半人上西山帮助西山百姓摧枯拉朽，开通大火后的山路，修筑倒塌的房屋。为了表明对西山群众的关怀，又开始修建高压泵管道，将山下的水泵上山去，解决山上多少代来吃水难的问题。

　　在广龙二年初的这个冬季，黑山堡山下山上到处风展红旗如画。数千人挥镐抡锹推车担担，像蚂蚁啃骨头一样拆除着旧堡墙。在拆除旧堡墙时，发现土堡墙内还有上千年历史的古代旧城墙。对于这些历史遗迹，黑山堡百姓毫不留恋地刨掉了。那些古旧的城墙砖只配以后垒猪圈。

　　破旧迎新的热潮中，新政权革命委员会表现出了团结奋战的精神。花二嫂不顾西山战斗中身体负伤，每日奋斗在第一线，堡墙头上不时闪动着她吹着哨子挥动着小办旗指挥的身影。

　　在这个开天辟地的冬季里，她天逃诩在等待刘广龙与她个别谈话。

　　〖目〗花二嫂自从那天晚上跟着刘广龙钻地道到野外"个别谈话"之后，就喜欢上了这个男人。那是一种从心里从肉里从命里对男人的喜欢，用城市人的酸话说，就是爱上了。她是一个浑身上下都憋足女人劲的女人。当她在山上山下村庄里走路时，她觉出自己浑身放女人味。就像发情的母狗会放味几里招引公狗们一样，她的女人味自然是放给男人的。她的娘家在西山，糊里糊涂嫁给了黑山堡一个窝囊的男人，壮母狗遇到了一条没性子的公狗，只能呲牙咧嘴将这没本事的公狗咬到一边去。

　　论干活，她比窝囊男人强。论地位，她过去是生产队妇女队长，比那个什么都不是的窝囊男人高。论模样，她更不该嫁给这个叫做木堂的熊男人。男人欠了她，她便每天回家摔摔打打做老娘。该照顾男人的事她也不是不做，做个鞋了，缝个衣裳了，总让男人出门有个模样。但是，男人睡觉时若想爬到她身上来，她就一胳膊将他搪到一边。没有金刚钻，别揽瓷器活。不三不四地弄两下，只会勾起她老娘的痒痒。男人便像个没骨头的癞皮狗，眯缝着眼睛可怜巴巴地看着她。她有时候也生出对男人的可怜，便软下心来侍候一回。可是，没骨头也没劲头的男人哆哆嗦嗦地爬上她的身体，又让她从心里厌恶。对付着完了，她便扭转身冲着墙壁睡自己的觉。

　　绑来，她喜欢开着院门坐在院子里了。又后来，她就招惹上别的有性子的公狗了。她也不乱招，她只喜欢自己看得上的男人：立在那里要有模样；活在世上要有能耐；到了床上要真枪实弹有本事。满村人说她风流，她不在乎，其实她过去也就两三个相好。罗元庆就是一个。铁塔似地立在那里像个大男人；在村里过去当着民兵营长，现在又当了民兵团副团长，有能耐；床上那两下也比自己的男人强百倍。她有个三岁的女儿，村里人都背着她说长得像罗元庆；她不解释，也觉得不冤枉。

　　文化大革命了，她跟着罗元庆一道参加造反派。造来造去，两个人都成了刘广龙的兵。刘广龙在她眼里一天比一天高大起来。他见多识广的能耐很快就遮住了花二嫂眼前的所有男人，包括罗元庆。她每日看着他，就看出一心窝甜水来。她用那一心窝甜水养起一个女人的梦。

　　她一有机会就用笑盈盈的目光盯住刘广龙，浑身女人的味也都放给了他。当刘广龙的金刚指直指着她，哪怕是训斥，她都觉出透心的舒服。她愿意刘广龙当众一指捅过来，在她胸脯穿个洞，她就可以一仰身躺倒在地了。

　　刘广龙那天说要和她个别谈话，她便从早到晚思谋着，终于谈上了。她发现，这是黑山堡最好的男人。刘广龙是顶天立地的革命英雄，又是能把女人里外搞透的好男人。就像他的金刚指能把砖头捅穿一样，刘广龙把她从上到下捅穿了。她像一团有劲的面，第一次被一个如此有力的男人翻来覆去揉透了，揉出了女人又软又韧的劲头。比起刘广龙，罗元庆那几下男人的功夫又显得浮皮潦草了。那天钻地道到了田野上，她先被刘广龙绑在树上，又被他松了绑，而后顶着月光就天翻地覆地干开了。刘广龙的军大衣成了他们的被褥，一男一女发出的烫热扛住了冬天里田野的寒气。她从来没有被男人这样颠来倒去地搞过。搞完了，她觉得自己浑身上下都酥了。等穿上衣服立起身，刘广龙从地上拿起军大衣，他们发现，大衣下的冻土都被他俩的热气融化了。

　　钻着地道回到大队部下面的地下室里，她和这个黑山堡最棒的男人又在床铺上难解难分。那天，她真的舍不得离开这个人了。

　　然而，第二天一早，她家院门口就被挂上了一双破鞋。破鞋是叫她摘了扔掉了，但是再见到刘广龙时，她在刘广龙的眼里没有看到预期的情意。刘广龙接连几天没有另眼看她，更没有约她个别谈话，她心中很难受。大概是院门口挂破鞋的事也传到了刘广龙耳朵里，刘广龙现在是革命领袖，要考虑政治影响，她只能这样想这样安慰自己了。她明白刘广龙要干大事的抱负，但也不该这样绝情绝意呀。破鞋是挂在她家院门口，她当女人的受得了，他当男人的就躲得到天涯海角去了？她想来想去又想不通。刘广龙忙，暂时不再约她钻地道，她能够忍受；可是每天见面，就像没那回事了，这让她活不下去。

　　为了守护这份情意，她不再让罗元庆凑近她了。罗元庆百思不得其解地问：我哪儿得罪你了？她说：哪儿也不得罪，我不想来往了。罗元庆瞪大眼说道：你还真想立起贞节牌坊啊。她拿起扫炕笤帚指着对方说：你别嘴里没人话，快给我滚出去，要不我就不客气。罗元庆打量着她，退着离开她家走了。她撂下笤帚，坐在炕上发起呆来。她从今以后不再和任何男人胡来了，她要门清院静地守着，只等着刘广龙再和她个别谈话。

　　饱打西山那天，她冒着性命危险冲上去抢回刘广龙。当她用身体护着刘广龙逃下山来，石头土块砸在她的头上背上时，她觉得自己在护一个心爱的男人，真幸福。后来，她晕倒在刘广龙脚下，晕晕乎乎中知道刘广龙将她抱了起来，放到了担架上，吩咐人们抬去抢救。她立时觉得自己做得真值。

　　又后来，她躺在家里养伤，刘广龙专门来看望她。她硬从炕上撑起身体来，觉得自己为面前这个男人做什么都心甘情愿。可是，刘广龙也就是让她好好养伤，再也没说别的话。她将自己的窝囊男人支出家门，刘广龙也没说一句情意特别的话。后来，罗燕来了，她说了两句慰问的话，就叫着刘广龙走了。她知道他们是有急事去了，她也知道罗燕和刘广龙关系不一般。她不想管刘广龙和其他女人的事，她只想刘广龙和她的事。刘广龙现在是她心目中唯一的男人。

　　她为了他，可以上刀山下火海。

　　刘广龙要建设红彤彤的新黑山堡，她就忍住伤痛又冲锋陷阵了。
