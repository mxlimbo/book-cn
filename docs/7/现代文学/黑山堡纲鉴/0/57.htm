【纲２】

　　【纲２】广龙九年初，刘广龙终于不能忍受大权旁落的局面，开始亲自出马收拾黑山堡天下。他抓住瞭望塔拆除事件，敲山震虎削弱鲁峰的影响，又老谋深算地分化禹永富和鲁峰，实行各个击破。他将花二嫂、赵蓓蓓调集到自己的政治核心中，用"女人也要在黑山堡当一半家"的口号无懈可击地扩充了黑山堡革委会常委班子。而后，他借"批孔孟之道"为题，将黑山堡两千多年的留孟槐连根挖掉，一下子将全堡忙着刨食吃饭的昏昏噩噩气氛冲开了。

　　〖目〗作为一个顶天立地的男人，刘广龙被罗燕不实干的说法真正激恼了。

　　他用了三天时间养精蓄锐，不和蓓蓓玩投壶，也不找其他人个别谈话。到第三天晚上，他把罗燕彻底搞败了。第四天早晨，他抖擞精神，带着钱爱孔和罗燕来到了黑山堡两级干部会现场。

　　这个会集中了山上山下大小队干部二三百人，会期五天，讨论部署新一年的生产建设。这是会议的最后一天，全体都集中在黑山堡小学的一个大教室里。

　　当刘广龙走近会场时，他立刻体会到了罗燕说的迈不进去的意思。教室里一定是人塞得太满了，大冬天门窗开着，窗台上门口也都挤满了人。浓烟连同一阵阵掌声和笑声十分兴旺发达地从门窗往外冒着。当走近一个窗户时，坐在窗台上的一个瘦脸汉子转头看见是刘广龙，冲他很尊重地点点头，然后似乎不便破坏开会规矩，依然转头向着会场。

　　刘广龙顿时觉得自己身后只带两个人的身单力薄。

　　听见鲁峰正在里边讲话，老家伙讲得很从容很风趣，逗得一阵又一阵满堂拍手大笑。他又往前走了几步，是门口了。穿过几个挤坐在门口的人的背影，看见鲁峰正站在讲台上满脸红光地挥着手讲话。老家伙现在又吃得大腹便便了，听见他拍着自己的肚皮说：现在看我这肚皮，就知道黑山堡粮仓又满了。不过大肚皮的时候要想着小肚皮的时候，饱肚皮的时候要想着饿肚皮的时候，丰收了要继续争取丰收，同时要有长期抗灾的准备。我可不愿意这个肚皮再瘪下去。全场又哄堂大笑。

　　刘广龙回头看了看罗燕、钱爱孔，两个人都有点怯阵。刘广龙在这一瞬间握住了自己左手的四个指头，挺出了中指，贴身转了一转，好像在往下捅砖头，而后壮起自己的气来，踏着台阶走上门口。

　　几个坐在门口的人很匆忙地回头看了一眼，发现是刘广龙，便礼貌地站起来闪开一条缝，说道：刘广龙主任，您是想进来？这一让路，给了刘广龙最初的一点破浪前进的感觉。他威严地点了点头，挤着人缝进去了。钱爱孔和罗燕也都借着首领的气势跟了进去。讲台两侧已经坐满了人。

　　鲁峰没在意有人进来，继续妙趣横生地挥手讲着。大概是因为人群分散的目光，他才扭头看见了刘广龙。他冲刘广龙点头笑笑，而后面对整个会场说道：现在刘广龙主任也来了，待会儿我讲完了，大家欢迎刘广龙主任再和我们讲几句。说着，他继续挥着手势天南海北地讲下去。刘广龙在讲台侧后边的椅子上坐下了。禹永富抽着旱烟袋坐在他的一旁，对他轻声说了几句，算是大概介绍了会议的情况。

　　刘广龙很像模像样地点着头，这种首领听汇报的架势开始培育起他在这个场面中首领的自信。他知道，他坐在这里，已经对鲁峰的春风得意给了压力，老家伙的讲话不那么放得开了，这让他觉出自己的余威。但是，鲁峰依然讲得海阔天空，老家伙不愿意随便收场。全场二三百人也都一脸笑容地瞪大眼盯着他，不时起劲地拍手，好像一群孩子过年一样凑着热闹。刘广龙看了看台下这些穿着黑袄的人们，多少觉得自己对这个王国有些生疏了。

　　他的权威此刻有些模棱两可地立在这里。

　　他是名正言顺的首领，然而这个名正言顺的权威现在有些空；再不充实它，就可能化为乌有。他知道名正言顺是个很重要的资本，古来的皇位传来传去，虽然有各种实力的弑君篡权，但是正统的名分常常是个重要的力量。有的时候，不得不抬出一个三岁的太子来当皇帝，多少有实权的人也要拿着这样的小皇帝当招牌。曹操那么大的实力，还要挟天子以令诸侯。但是，名正言顺的招牌又是有限度的，实权派推翻名正言顺皇帝的篡权行为比比皆是。

　　刘广龙掂量了一下鲁峰，相信他在这一会场人中比他刘广龙更有实际控制力。但是，刘广龙不是一个只有其名的草包皇帝。他一直没有忘记掌握和县里的联系，他一直将黑山堡的全部抗灾成果归于自己的正确领导。现在，当有人把他看做空壳子，供在那里不当回事时，他要让他们领教一下他的厉害了。今天他撑起气势迈进这个会场，就是一个有力的开始。

　　他对自己进门前运动金刚指感到满意。关键时刻软下来，就一败涂地。他知道他今天这一下，就把鲁峰这个阵势捅破了。正是进到这个阵势中，也便看到了里边的缝隙。一个铁蛋一旦楔进缝隙，你就能够把它分化瓦解。他决定对鲁峰和禹永富实行各个击破。

　　鲁峰终于讲完了，全场很长久热烈的掌声。

　　轮到刘广龙登台讲话，掌声虽然不低，但是显得规矩，没有那种热烈情绪。就好像一群小孩玩得兴高采烈，看见家长来了，不得不摆出一副笑脸做欢迎，其实巴不得家长不要干涉他们。

　　刘广龙心里明白，他今天绝不能太凑他们的热闹，再顺着他们的势走下去，就可能扭不过来了。他名正言顺地立在讲台上，两手八字伸开扶住讲台，开始有些生疏但又是顽强地做首领讲话。他这个讲话十分经典，在广龙历史上有相当的研究价值。根据挖掘到的会议记录资料，我们可以逐层透视刘广龙在这个讲话中表现出的政治手腕。

　　刘广龙第一句话就说：看到大家很高兴，很长时间没有和大家这样面对面了，乍一见面，从心里边觉得喜气洋洋，咱们这叫欢聚一堂。众人一片开心的掌声。他又接着讲：我虽然没有天天和大家面对面，但是大家的功劳苦劳我天天都在听老禹、老鲁汇报，我对大家很满意，对你们每一个人的所做所为都一清二楚。他说着用手指点着会场：你们的贡献都在我的功劳薄上记着呢，谁也不会落下一笔。众人又是开心地欢笑鼓掌。

　　刘广龙觉出自己首领的风度已经高出鲁峰一筹。他还有意地将禹永富排在了鲁峰的前面，这是他一个不易被人觉察的策略手段。他又接着说道：刚才，老禹向我简单汇报了一下这几天开会的情况，我很满意，对他的汇报很满意，对这个会很满意。他这一句短短的话又十分老谋深算。他继续表现着他名正言顺的首领地位，同时又非常微妙地离间着禹永富和鲁峰的关系。

　　他第二次不经意似的将禹永富排在了鲁峰前面，他这样说道：我对老禹、老鲁这两三年来领导大家抗灾生产基本上也是很满意的，他们──特别是老禹──熟悉黑山堡情况，做出的很多领导决策是有水平的，我想，我对他们两个人的表扬你们也不会反对。

　　众人又开心鼓掌。

　　刘广龙接着说道：今天，我们常委会五个人都到了，我、钱爱孔、罗燕、老禹、老鲁，我们常委会对大家两年多来的工作都一致表示满意。

　　正是他这样回头指点着钱爱孔、罗燕，他们也便在讲台上坐得像模像样了。从这一刻起，钱爱孔、罗燕紧挨着鲁峰、禹永富坐在讲台上，有了一种名正言顺的感觉。罗燕在那一刻十分佩服刘广龙云山雾罩的讲话手段。刘广龙说过，政治手段的一半就是讲话的手段，一句话讲好了，可以调动三军，一句话讲绝了，可以致敌于死地。

　　刘广龙接着讲到：从广龙开元以来，七年多的时间，黑山堡百姓步步前进，节节胜利。我们敢统一，就把东山西山山上山下统一了。我们敢革命，就把张力平反革命集团、罗元庆冯二苟反革命集团粉碎了。我们敢建设，就把我们黑山堡建成了一个像模像样的新村，堡内街道横是横竖是竖，整整齐齐，全国独一份。我们敢抗灾，百年不遇的旱灾终于叫我们战胜了。

　　这段讲话无疑和鲁峰主持的两级干部会气氛是有出入的，但这种开天辟地讲过来的历史总结具有名正言顺的力量。它无疑将几百个脑袋梳理了一下。在那一刻间，刘广龙一定觉出了自己正统的权威正在罩住会场，而鲁峰、禹永富可能都会隐约感到一种没法抗拒的泰山压顶之势。刘广龙再一次自我欣赏了今天闯进门来的气魄，他不伦不类地想到"过五关斩六将"这样的历史典故，还想到"单刀赴会"这个成语。

　　他觉得自己的气势开始起来了，做出的手势也就更加云山雾罩。

　　往下他讲的话就更显出政治上的老到，他说：经过这么多年的锻炼，我们做事更要心明眼亮，要每做一件事都问一个为什么，要做好，不要做坏，不能走回头路。譬如，这两天我看见黑山堡山上山下在拆瞭望塔。他说到这里，回转身看着鲁峰很首领地说道：这件事你们事先倒是没有和我商量。

　　鲁峰坐在那里老脸一下有点漾红，点头说道：是没来得及向你请示汇报。

　　刘广龙扭回头继续对着会场说道：这件事刚才老鲁讲了，他们没有来得及向我请示汇报，事情不在请示汇报没有，还要看做得怎么样？我看了看，明白他们是想腾出瞭望塔占的耕地，走顺浇地的水渠。拆掉田中的几座瞭望塔我看也可以，但是，山脚下那些瞭望塔都是修在乱石坡上的，还有山上的一些瞭望塔也都不是修在耕地里的，一刀切都拆掉了，既没有增加任何好处，反而浪费了很多人力。再说，瞭望塔是黑山堡的一个革命风景，一走近进黑山堡，瞭望塔四面一立，红旗山上山下一升，鼓舞士气呀。古代打仗靠什么？靠旌旗鲜明，锣鼓震天，这才有士气。瞭望塔说拆就都拆了，把黑山堡的革命士气损害了，我看这件事做得不算太好。

　　他目光扫视了一下会场，在大幅度的旋转中，余光也顾及了鲁峰。老家伙坐在那里颇有些不自在。

　　刘广龙知道，就要这样名正言顺高屋建瓴不动声色地削弱老家伙的影响力。讲话不光是讲满堂和气，还要放得下脸来，讲出这些有份量的话。政治斗争关键时刻敢不敢放下脸来讲话，是决定性的。只要他的话讲得没人能跳起来公开反对，就算是把场压住了。他接着说道：老鲁他们没向我及时请示汇报，当然他们有一点责任。他扭回头很家长地看着鲁峰和禹永富，笑着说道：有一点吧？

　　两个人不得不点点头。

　　刘广龙又面对全场说道：你们大家为什么不文死谏呢？过去我们搞文死谏运动，因为出了张力平反革命集团被干扰了，今天我们还可以重提文死谏呀，你们当时为什么没有一个人站出来，找到老鲁提不同意见呢？为什么没有人站出来，去找我刘广龙反映不同意见呢？

　　刘广龙的话讲到这里，气势足够地首领了。

　　他接着说道：所以，我今天就再一次向全黑山堡干部群众提出，做事一定要心明眼亮，不要随波逐流，什么事情都要动脑筋想一想，不要盲目执行。瞭望塔是拆光了，田间的就算了，田边山脚下的要不要重建以后再商量。对于老鲁他们来讲，要多请示汇报，对于大家来讲，要敢提意见，文死谏。现在中国不是都在批孔孟之道吗？孔老夫子、孟老夫子两千多年的传统了，我们今天都不能盲目跟随，其他人其他事更不能随便跟随。大家一定要领会我今天这些话的精神。

　　最后，他似乎不经意地其实是十分经意地看着会场说道：孔孟之道有一条就讲男尊女卑，我们今天就要讲男女平等。我看今天会场上也还是男尊女卑，女的没有几个嘛。

　　众人算是轻松地一笑。

　　他指着坐在前排的花二嫂说道：这儿有一个花二嫂，到底是人单势薄嘛。他用手指了指身后说道：我们常委五个人，也就罗燕一个女的嘛。我看像这些孔孟之道，我们都要反其道而行之，从常委会开始，我看就可以增加几个女的，半边天加半边天，才是整个天。

　　〖目〗两级干部会议以后，刘广龙立刻主持召开了革委会常委会，专门讨论了瞭望塔拆除一事。鲁峰就此做出了两点检查：第一，没有及时向刘广龙请示汇报；第二，没有考虑到瞭望塔鼓舞士气的革命意义。刘广龙点头通过了这一检查。会后，他指示将鲁峰的检查及常委会有关瞭望塔拆除一事的决定通报大小队两级干部，要求大家就此展开讨论，提高认识。

　　看着鲁峰在常委会上还挺沉得住气的样子，刘广龙心中就发出一丝冷笑。老家伙以为事情就过去了，其实这只是刚刚开始。他知道，鲁峰这两年在黑山堡的威信和权力开始被动摇了。

　　在同一次常委会上，他又提出增加两名女常委：花二嫂和赵蓓蓓。因为有了处理瞭望塔事件的威风，又有事先铺垫好的反对孔孟之道、男女平等的舆论，鲁峰和禹永富提不出什么反对意见。禹永富只是说了一句：赵蓓蓓是不是没什么工作经验？叫刘广龙一句话就否了：工作经验一工作就有了，过去，男人干的事情不叫女人干，她怎么有经验？只有叫她干，才有经验。钱爱孔、罗燕自然是紧密配合刘广龙。

　　对花二嫂、赵蓓蓓的任命在黑山堡革委会全体会议上更是一致被通过。至于要不要上报县里，那更是刘广龙一手遮天的事情。

　　当天晚上，刘广龙又将罗燕、钱爱孔、赵蓓蓓、花二嫂召到他的单独办公室内，开了特别会议。花二嫂、赵蓓蓓当了常委很兴奋，像喝了酒一样红着脸。钱爱孔更觉得这是他千载难逢的好机会，咬牙切齿地将鲁峰、禹永富骂了个狗血喷头。罗燕看清了刘广龙一系列部署，为一下出现的大翻个前景摩拳擦掌。

　　刘广龙说：二级干部会我去讲话，那叫打破他们的一统天下。处理瞭望塔事件，叫做山雨欲来风满楼，给全堡上下吹一吹风，活一活脑筋。增加你们两个女将赵蓓蓓花二嫂，这叫做点兵点将，借题发挥。赵蓓蓓不理解地问：这怎么叫借题发挥？罗燕说：男女平等、反对孔孟，这是题，发挥就是把你们俩发挥进来了。

　　刘广龙与众人一起满意地笑了。他接着说道：往下，就要开始做大文章了。罗燕说：常委会定个决议，把两千多年历史的留孟槐连根挖掉，批判孔孟之道运动就开展了，咱们就可以又借题发挥了。刘广龙指着罗燕说道：这很对，但是不应该由常委会做出决议，应该先由百姓出来造反，我们再支持，自下而上和自上而下相结合，才有分量。

　　罗燕说：可以发动知识青年贴大字报。

　　刘广龙说：张力平带来的那拨知青不能再用了，背景太复杂，牵枝带蔓连到张力平那里去。罗燕说：这两年又来了一批知识青年，才都十几岁，思想单纯，好发动，我可以去做这个工作。刘广龙点头说道：这样很好。

　　几天后，黑山堡内出现了大字报，强烈要求连根挖掉留孟槐。黑山堡大队广播站立刻广播了这张大字报。黑山堡革委会常委会在刘广龙的主持下通过了决议，支持知识青年的革命行动，而后组织了浩浩荡荡的队伍开到北山脚下，挥镐抡锹将留孟槐连根挖掉。当几十条牛将大如小山一样的留孟槐连根拖出来后，一时狂风大作，飞沙走石，，山脚下留下一个池塘一样的大圆坑。数千人围观了这一壮举。

　　批孔孟运动在黑山堡发动开了。

　　[在黑山堡挖掘到的刘广龙的遗物中，还有一本《四书集注》。宋代理学家朱熹将《论语》、《孟子》、《中庸》、《大学》合编为《四书》，并做了"集注"。这本书大概是广龙九年批判孔孟之道时才做了批划的。

　　在《四书》中，刘广龙对《中庸》和《大学》一字未划。

　　对《孟子》则只划了一句话："故天将降大任于斯人也，必先苦其心志，劳其筋骨，饿其体肤，空乏其身，行拂乱其所为，所以动心忍性，增益其所不能。"刘广龙在这段话旁边划了一个很大的惊叹号。看得出，他并没有对该文批判的意思，而是赞赏的意思。

　　对《论语》他的批划较多，我们现在一一引录出来，供研究广龙年代历史的读者参考。第一句是："君子怀德，小人怀土；君子怀刑，小人怀惠。"在这句话旁边，刘广龙批了一句重复原文的话：君子怀德，小人怀土；君子怀刑，小人怀惠。

　　第二句他划出的是："子曰：君子喻于义，小人喻于利。"在这句话旁边，刘广龙批注：大义小利相结合，言不及义不行。

　　在《论语》中他划出的第三句话是："子曰：巧言乱德，小不忍则乱大谋。"对这句话，刘广龙划了三个硕大的惊叹号，而后批划了几个字：忍小忿而就大谋。

　　他在《论语》中批划的第四句话是："子路曰：卫君待子而为政，子将奚先？子曰：必也正名乎！子路曰：有是哉，子之迂也，奚其正？子曰：野哉，由也！君子于其所不知，盖阙如也。名不正，则言不顺；言不顺，则事不成；事不成，则礼乐不兴；礼乐不兴，则刑罚不中；刑罚不中，则民无所措手足。"在这句话旁边，刘广龙批注了一个公式：名正──言顺──事成──礼乐兴──刑罚中──民有所措手足。

　　在《论语》中刘广龙批划的第五句、第六句话是："子曰：唯上智与下愚不移。""子曰：民可使由之，不可使知之。"刘广龙对这两句话的批注是相同的一段话：上智下愚，民可使由之，不可使知之，这里有真理有谬误，有时，民必使知之。

　　刘广龙在《论语》中批划的最后一句话是："子曰：唯女子与小人为难养也，近之则不孙，远之则怨。"他在这句话旁边批注了女人二字，而后划了一个硕大的惊叹号，又批注了小人二字，在旁边也划了一个硕大的惊叹号。而后，在女子小人这两行字的右边，划了二道竖杠，又批上一句话：近不得，远不得。而后，又是硕大的惊叹号。

　　看完刘广龙对《四书集注》的全部批注，我们大概要认真想一下，才知道他在黑山堡发动批判孔孟之道的真正思路。]
