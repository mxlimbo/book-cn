【纲８】

　　【纲８】几天后，罗元庆、鲁峰、禹永富、罗燕、钱爱孔、冯二苟等革委会常委各带领一个排的武装民兵，分头挨家挨户给黑山堡山上山下近三千户共一万口人分发了储备粮。数量由原定的每人一斤增加为每人两斤。

　　分发粮食这两天，山上山下武装民兵处于一级戒备状态。

　　又几天后，带头冲击南河涧大桥和粮库的胡大被定性为反革命分子揪了出来，他因为成分偏高，富裕中农，接近富农，又和张力平反革命集团中的知识青年有过来往，罪大恶极，铁案如山，游街后，被送往劳改队劳改。几个剥了留孟槐树皮的人也都被定性为坏分子，游街示众后，送往劳改队劳改。对他们的揭发批判连日在广播中从早到晚地进行着。黑山堡革命委员会又重申了不经获准任何人不准出境逃荒的通令。谁破坏黑山堡的威望，绝没有好下场。

　　在这段日子里，刘广龙除了偶尔和罗燕、赵蓓蓓进行个别谈话外，没有再找过其他人。村中也只有罗燕、赵蓓蓓这样几个不多的女人因为有补助粮，还保持着模样，其余的女人大多已经瘦脱了形。

　　黑山堡的鸡早就杀光了，个别谈话也便没了鸡蛋养护。
