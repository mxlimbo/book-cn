【纲６】

　　【纲６】黑山堡旧王朝的第二号人物禹永富这顿晚饭吃得心事很重。他端着大碗蹲到门外院子里，看着院中的一个石碾发呆。筷子停在碗上像是一个思索的符号。老婆颠着脚弯着腰在灶火房转来转去。回娘家来看望的二十多岁的女儿禹莲往爹的汤面碗里滴着香油，做爹的也没有太醒过神来。

　　这时，刘广龙在罗元庆、罗燕的陪同下进了院子。

　　罗元庆、罗燕和禹永富沾些远亲，按照亲热的称呼叫完禹永富，便簇拥着刘广龙在院子里拣小板凳坐下了。禹莲赶紧把一个小方桌摆到他们面前，刘广龙摆了摆手，说：吃过了。便和禹永富说开了话。

　　刘广龙亲自登门是个出奇不意的举动，他请禹永富出山。禹永富皱着一张北方农民脸沉默不语地就着大碗吃饭，老婆和女儿站在身后眼巴巴地看着他。最后，禹永富冒出一句话：要让我管生产也行，大队小队生产领导班子的人要许我挑拣。刘广龙轻轻拍了拍禹永富的肩膀，既有领导的权威，又含晚辈的亲热，说道：由你提名，革委会最后定。禹永富又沉默了许久，撂下空碗，掏出旱烟袋说道：不怕我搞复辟？刘广龙坐在小板凳上仰声笑了：我怕什么？古人说用人不疑，疑人不用，你放手干，我给你做后盾。

　　刘广龙看到禹永富的女儿禹莲正站在他爹身后看着自己。他知道自己现在在所有的女人眼里都很英雄好汉，他两手八字伸开，扶着自己的双膝很气派地仰脸瞄了禹莲一眼。小女子高高挑挑很中看，脸被他瞄红了。
