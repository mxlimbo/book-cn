【纲３】

　　【纲３】广龙三年夏收大忙期间，刘红为刘广龙生下了一个儿子。这一年是猴年，论月份又是农历马月，猴年马月是最难熬来的喜庆年月。黑山堡首领生下了猴年马月的儿子，自然是大喜；然而，他却严令部下不许庆贺送礼，更不许对黑山堡百姓传扬此事。他以天下为公。在生下儿子这一天，他只是约了钱爱孔、罗元庆、罗燕、冯二苟几个人喝了酒。

　　那天晚上，他一个人住在大队革委会办公室，没安排任何个别谈话。

　　他通宵达旦地在专记革命的笔记本上记录了他的宏伟构想。
