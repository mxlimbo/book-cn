【纲１２】

　　【纲１２】大概是被粉碎张力平反革命集团的胜利所鼓舞，黑山堡全体百姓从广龙四年春天开始全年投入了声势特别浩大的新黑山堡建设运动，成果显著。

　　〖目〗在这一年，四四方方堡内的村舍终于完全棋盘化，横齐竖直。几十个小队各占一个大方格。小队内又有若干小组，各占一个小方格。

　　现在，从城堡正南门走进来，迎面就是大队革委会办公室。拆旧更新，比原来规模大多了，党、政、工、团、妇联、治保、贫协、民兵团办公室都在这里。再往堡里走，迎面一排就是商店、理发馆、信用社、卫生所，这完全符合刘广龙在广龙元年除夕提出的城堡规划。

　　过了这排房，前面出现的就是棋盘化的村舍。这一年，黑山堡推广商鞅变法的经验，五户一小组，十户一大组，互相监督纠正揭发，各家各户不许留宿任何外来流窜人员，不许违反黑山堡革命委员会各种规章制度。倘有一户违法，其余九户不揭发检举，就如古人连坐一样共受惩罚。

　　再往里走，经过村舍区，就到了城堡北。这里是机要区，西有粮棉库房，民兵武器库房，东有刘广龙的单独办公室。在刘广龙单独办公室的院内，还建有黑山堡广播站，因为首领有可能通过广播站指挥全堡革命行动。北城墙开有二门，一门供粮棉库房与堡外运输交通，另一门直通刘广龙办公室大院。此门常年紧闭，钥匙刘广龙亲自掌管，遇到紧急情况，首领可以从这里直接出城堡。在刘广龙的办公院内，还建了一座很高的t望塔，t望塔上有直入云天的信号旗杆。首领站在t望塔内，可以俯瞰城堡内家家户户的院落，也可以升起红黄蓝信号旗，直接指挥山下山上革命行动。

　　在这一年内，黑山堡还挖通了地道。地道的一个秘密出口，就在刘广龙办公室内。遇到紧急情况，首领可以钻地道，直接到达北山。

　　正像首领几年前规划的那样，以粮食加工为主的加工业如酒厂、醋厂、粉坊、油坊、豆腐坊等等都整整齐齐重建在东堡墙外，而以农机修理、牲口饲养为主的行业都整整齐齐排在西堡墙外。

　　现在，外来人要进黑山堡国界，一过南河涧大桥，迎面依然是跨路的拱形大门，上面写着"黑山堡"三个大字，只不过拱形大门重新修建了，比过去壮阔宏伟。旁边也建了t望塔，t望塔上也有直入云天的信号旗杆。山上山下有很多这样的t望塔，塔顶都立着信号旗杆，t望与联络相结合，形成黑山堡王国的指挥体系。

　　高压泵送水工程依然日日往山上送水，山下平原水渠已经纵横遍布。又如首领设计的，环城堡修了护城河。禹永富曾经提出，这样太浪费水；但被首领否决。全黑山堡山上山下开始实行一齐上工下工，上下工时间以山上为准。天一亮，就吹喇叭、升信号旗、敲钟、放广播，一起上工；天一黑，又是统一号令一起下工。山下天亮晚，天黑早，上工下工就要摸一点黑。

　　革命委员会每日早点名晚总结的程序现在已成为全堡人民的共同程序。在早晨上工之前，晚上下工之后，以生产小队或以民兵连排班为单位进行一早一晚的统一思想活动。按规定，每隔五天，要进行一次人人过关的反省检查。每隔十天，进行一次全民军事训练，凡十岁以上六十岁以下男女老少都必须参加。每隔十五天，进行一次全体百姓参加的批判大会，批判内容根据当时阶级斗争动态而定。每隔一个月，黑山堡山下山上进行一次革命联欢活动。每隔两个月，黑山堡开展一次清理阶级队伍运动。

　　东山西山上分散的村庄也准备拆迁重建，和山下一样成为整齐的村舍。只不过山上地势复杂，工程巨大，要在今后若干年内逐渐完成。
