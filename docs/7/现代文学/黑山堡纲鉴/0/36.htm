【纲８】

　　【纲８】广龙三年秋收大忙期间，刘广龙将生产的事完全交给禹永富和鲁峰管。鲁峰过去在省里当过大干部，对工业农业都十分熟悉。老家伙不仅协助禹永富管农业，还更多地把房屋建筑、农机修理、砖瓦厂、采石场、酒厂、醋厂、豆腐坊、油坊、粉坊、酱油坊管了起来，实际上，他等于把黑山堡王国的工业、交通、建筑业都抓在了自己手中。他像一个久经沙场的老狮子一样，不露声色地观察着黑山堡的政治风云。张力平找他推心置腹地谈过，他耐心地听着，思索地点着头，最后说：你再和禹永富谈谈，和方方面面谈谈，看看大家都什么意见。

　　这头老狮子在广龙三年的政治动荡中没有留下任何有明确倾向的痕迹。
