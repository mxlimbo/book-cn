　　"你令人家开心？你是在骗别人的爱心，你如果要找爱心的话，就应该去找一个你最心爱的人，不要四处去骗……短暂的One Night Stand。"

　　天使不明白："什么是最心爱呀？"

　　"我还以为天使是一只很纯的白鸽，但现在看清楚你的样子，愈来愈像一只鸭。还有，你刚才……不是说过晚上留在这里，不饿死便会冻死吗？"

　　"我说说而已。"

　　"你可以飞了吗？"

　　"……"

　　"可以飞的话就请你走吧。"

　　Athena愤怒进房，剩下天使与一百双鞋呆在那里。

　　夜深，Athena睡在床上辗转反侧。

　　天使抱膝瑟缩在沙发上，对着一屋新鞋，两眼也发光。忽然间看见远处的鞋柜底的墙脚下，有一对封了尘的红色高跟鞋。天使爬过去把红色高跟鞋拉出来，抹去灰尘，放到新鞋中央，趴在地上，两眼望着她。

　　突然，红色高跟鞋像落到一个舞池上，翩翩起舞起来，婀娜多姿，意态撩人。红色高跟鞋愈跳愈诱惑，渐渐舞到旧皮鞋身旁，跳出更诱惑姿态。旧皮鞋受挑动亦开始舞动起来。天使伏在地上看得会心微笑。

　　两对鞋愈跳愈合拍，愈跳愈精彩。屋内的新鞋亦好像现场观众受感染一样，渐渐开始原地舞动起来。天使伏在地上也愈看愈开心。其他新鞋亦加入舞动行列。这刻，天使和全屋的鞋都兴高采烈地在舞动。高潮处，突然……

　　Athena的房门突然打开，天使吓得飞身扑向红色高跟鞋。

　　Athena走出房门，目睹天使趴在地上："你在那里干什么？"

　　天使结结巴巴说："我……我在试飞。"

　　Athena懒不在意："飞不起来就等好了才走吧……对了，你可不可以帮我多带点东西？"

　　说罢头也不回地关上房门。天使从怀里掏出红色高跟鞋，一副惊魂未定的样子。
