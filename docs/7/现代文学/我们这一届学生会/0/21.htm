第二十一节

　　从学生会办公室出来，萧木就直接去找于勇了。他把于勇叫出宿舍，显出一副神秘的样子，一边走一边就说："我带你去找陶韵。说不定能帮你们一把，尽到哥哥我的责任。"

　　于勇听后，就欣喜欲狂，紧跟在萧木的身后，不时就小跑几步，追着撵着把满腔的感激之情表达出来。萧木一笑，并不多言，他知道于勇此刻的感激绝对是由衷的、发自内心的。

　　来到了"怡情园"，萧木让他在门外等着，独自就进去了。陶韵正在晾衣服，见到萧木就很惊奇。萧木不由分说，叫上她就走。陶韵心里忐忑不安，偷眼望着萧木，预感一定有什么大事就要发生了。来到门外，见了于勇，她就犹豫着想退。萧木制止了她，说："跟我来你俩！"说着，就向湖边走去。

　　来到无人的地方，萧木停下了脚步，转过身望着他俩，半晌没有开口。于勇满脸堆着笑，神情有些羞涩，一刻不停地搓着双手；陶韵则眼里含着不安，低头望着湖水，面有不悦。萧木掏出一支烟点上，深深吸了一口，又缓缓吐出来。突然，用手指着陶韵，就问于勇："你！一直在跟这个人谈恋爱，对不对？"

　　于勇不知所措了，脸色怪异，半晌，慌忙点头："是……"

　　陶韵白了他一眼，就想反驳。萧木又指着于勇，转而问她："你！跟这个人根本没就谈过恋爱，对不对？"

　　"对！"陶韵很坚定。但她的表情是仍不明白萧木肚子里卖的是哪块狗皮膏药。

　　于勇脸色发白，心急火燎地转向陶韵。想辩解却是没有辩解，张了张嘴就算是解释过了。

　　萧木没有看他，把两眼牢牢地盯住了陶韵，用相当缓慢但却足够响亮的声音，一字一顿地说："陶韵，我要开始追你。从现在起，你当心点儿！"

　　一时间，空气里好象结了冰，三个人全都无语。

　　于勇恨不得把自己的眼珠子挖出来举到萧木面前，来强调他是多么震惊多么愤怒多么屈辱！但他没那么做，他的嘴唇由红变白，再由白变红，哆嗦了好一阵儿，终于就发了一声悲壮的单音。然后，带着悲壮的脸色和悲壮的心情猛地转身，跑了。他跑的时候就感到脸颊上凉瘦瘦的，好象有什么液体悲壮地滚落下来。

　　陶韵没有思想准备，听了这句突如其来的话，就睁圆了眼睛。那双眼睛里的表情很复杂，混合着惊讶欣喜疑惑释然渴求不安委屈幸福等等等等，并且就这样复杂地定格了很久、很久。

　　萧木依然一脸的麻木，冷冷地对她说："晚上8点，我在这儿等你！"说完，转身就走了。

　　到了晚上，喧闹了一天的校园逐渐就进入了平静。路灯的光从阔大的梧桐叶子上滴落下来，溶化成细碎的光箔。偶尔有人走过的时候，身影就被踩得沙沙作响。高大的教学楼此刻消失了，它溶进了夜色，只留下一块又一块方方正正的光，整齐划一地码砌在原来是楼的地方。这光在湖面上反射出一片璀璨，微风拂过，荡起的涟漪就让人想起正在驶过的豪华客轮。

　　陶韵心神不宁。她坐在石凳上，垂了头，指尖缠着一张白手帕。长发飘了下来，在风的怂恿下偷偷亲吻她俏丽的脸庞。萧木在不远处无言地走来走去，同样是低垂着头，眉头紧锁。一明一灭的烟头把夜色灼得"滋滋"作响，好象有意要渲染这种沉闷的效果。

　　就这样，默默无言，捱过了20多分钟，两个人始终谁也没有讲过一句话。

　　下午，萧木放下那句硬梆梆的话就走了。陶韵却在原在楞了好一会儿，才慢慢往回走。她的脸色忽晴忽雨，迎面走来的学生，就不由得凑在一堆，猜测新的"据说"。她没有在意，极力在乱哄哄的脑子里理出一条明晰的头绪。一直到上楼梯时，她才好象突然想通了，连蹦带跳地跑回了宿舍。

　　她把衣箱搬到床头，将里面的衣服一古脑倒在床上，一件接一件地披到胸前比试。他在室内旋来旋去，反复哼唱着刚刚学会的台湾校园歌曲"坐在门前的矮墙上一遍遍幻想——"。唱着、旋着，不时就在同宿舍的大同学脸上亲一下，在小同学头上敲一下。同学们就屏了呼吸，瞪大惊惧的眼睛，望着她疯来疯去。大同学头对头低语商量，要不要问问医生，看精神分裂有什么样的前期征兆；小同学们互相依偎着缩在门后的角落，查看着发生突发性事件时最便捷的逃生路线。晚饭时，她让同学随便带了些饭菜，却一口没吃，大大方方又分给了别人。同学们更加坚信，她今天一定是受了太大的刺激，就连从不相信那些乱七八糟的"据说"的同学，这时也不由得产生了失去真理的痛苦。

　　陶韵自己也闹不清这般失常的原因。她已经习惯了被异性包围、追逐、恭维的生活，对男生们前仆后继、勇往直前的大无畏精神早已司空见惯。不仅不会感动，相反还充满了鄙夷和烦恼。但这次不同，没有哀求没有眼泪没有情书没有下跪没有上吊投湖跳楼割腕，也没有徐志摩的诗歌戴望舒的诗歌舒婷的诗歌以及不知什么人胡乱涂抹的诗歌。只有一张冷得下了霜黑得能播种的脸，还有一句硬得象棍子直得象棍子短得象棍子的口号。她审慎地问自己：是逆反心里在作祟，还是真正心仪已久？

　　要是按照自己心目中的恋人标准，萧木这个黑黑瘦瘦硬硬冷冷的家伙根本是不沾边儿的呀！可为什么今天上午在办公室谈到于勇时，自己会那样委屈那样恼恨那样急切地辩解？为什么自己以前在鄙夷他的同时，却又时时盼着听到他的连珠妙语？为什么自己会那样热心地向董力昆打听萧木的为人？而当董力昆谈起来时，自己是那样专注，那样殷切地希望谈论永不中止？为什么刚才见到萧木来找，自己会莫名其妙地心跳？而当见到于勇也在埸时，自己会那样失望，那样哀怨，那样伤心？这一切她无法不问自己却又不想多问。要回答这些问题太累太难，而她现在只想轻松只想唱歌，想蹦蹦跳跳想挑一身好衣服想着今晚八点。她愿意被自己的感觉牵着走下去，愿意被自己心中的火焰反复烘烤，就象烤烧饼一样翻来覆去地体会那种浓郁久远的馨香……

　　可是，来到湖边这么久了，期待的话还是没听到。她偷偷望了萧木一眼，见他仍在漫无目的走来走去，那样子似乎永远不打算开口了。兴奋的心连同随着心一起带来的忸怩和颤栗，都如湖面的水泡，一点一点地消失了。只有不安和焦虑象越来越厚重的夜，把她慢慢地从头蒙到了脚。

　　她不知道，下午上演的那一幕，只是萧木事先排练的一出邪恶的戏，纯粹是出于报复于勇的促侠心理，和表现自我的争胜欲望。他要用这种阴损的方法彻底割断和于勇的交情，同时也是要让陶韵这小娘们儿清醒过来！ 可是，当他念出那句精心准备好的道白时，他是清清楚楚地感觉到有一股温情被掺了进去，而这根本就不符合剧情的要求。更要命的是，不听话的舌头竟然自作主张，临走前约她晚上8点在这儿见面，真是见了丫挺的活鬼了！

　　这沉寂的20多分钟，对于萧木来说就是黑暗的中世纪了。他觉着自己苦心构筑、又时时加固的心灵堤坝正在慢慢塌毁，感情的激流有如禁锢已久的湖水，向着塌落的方向夺路而逃；他赖以生存并引以为自豪的理性屏障被捅开了一只风洞，一股异样的情绪从风洞里徐徐地溜出。他试图固守最后的阵地，但却没有任何可以支撑的东西。他感觉自己象一头又蠢又笨的熊，掉进了陷井，正可怜巴巴地趴在井口，而井口还在不断的塌陷……

　　这一局是彻底玩儿砸了！他恼火万分，就想。

　　夜风轻拂，一股淡淡的发香弥漫开来。他忍不住偷眼望了陶韵，就见她一手扶在石凳上，两眼无主地望着湖面，显出一副无可奈何的神情。远处的灯光映衬过来，她的皮肤白嫩如月光，长发飘逸如黑瀑，神态安静如一只受了惊吓的小鹿，均匀的呼吸使她的姿态充满了青春的韵律。萧木不敢再看下去，他终于明白了，眼前坐着的根本就是一个女巫，一个叫过路的行人无法抵御的林妖。

　　"走吧，回去！ "

　　萧木手指一弹，烟头划出一道漂亮的孤线，飞进水里。

　　"什么？可是……可……你一句话也没说呀。"陶韵满眼皆是霜雾，睫毛沾满了露水，晶莹闪光。

　　"你不也一言不发吗？"

　　"可，可是你叫我出来的呀……"

　　"也是你自己愿意来的，不是吗？"萧木好象在跟谁赌气，开始有些蛮不讲理。

　　"你！……"陶韵站起身，早已蕴酿成熟的泪珠也随之坠落了。她顾不上擦抹，一转身就跑了开去。但并不跑远，扶着一棵小树就停下来，小树也随她一起嘤嘤抽泣。

　　萧木上前，双手插在裤袋里，显得不知所措。半响，闷声闷气地说："我去买烟，你等一会儿……当然，也可以不等……"话未说完，就慌里慌张地消失在黑暗中。

　　陶韵听他走远，忍不住就哭出声来。这是她一生受过的最大的委屈了，忍不住就对萧木恨恼交加。她想立即拨腿，一走了之。但迈了半步，却又止下。望望天空，再望望教学楼的灯光，最后无奈地叹了口气。慢慢就走回原处，在石凳上重新坐下。她看出了萧木的矛盾和不安，知道他一定会回来。她要等对方开口，然后再响亮地拒绝他。

　　过了一刻钟的功夫，萧木果然就踢踢踏踏地回来了。他在陶韵面前站定，神情严峻，说：

　　"陶韵，我、我想过了，我……我爱你！"

　　陶韵霍然起身，一双火辣辣的大眼睛直盯着萧木，睫毛上还闪着晶莹的泪光。她想像着自己抬起了右手，把一记力大无比的耳光狠狠地抽在眼前这张黑乎乎的脸上，然后一字一顿地说："可是我并不爱你！"说罢，转身就走，决不回头！

　　这只是她的想象，真实的她是在听到这话的一瞬间，满肚子的委屈一齐涌上了心头。必须赶紧找一堵可以支撑自己的墙，必须找个能理解自己的人倾倒满腔苦水。这个人此刻就在面前，正用火辣辣的目光盯着自己。那双熠熠闪光的眼睛象一对无底的深洞，把自己强烈地吸引过去。她心跳过速，这就逼着她必须说些什么才能平缓情绪。于是，她就说："我、我也是……"

　　说着，就想扑过去，象女人们通常表现的那样，搂紧对方的脖子。

　　谁料，萧木竟挡住了她，问："你也是什么？"

　　"就是你刚才说的那个字……"陶韵羞涩地低头。

　　"你必须亲口说出来，否则……我……"

　　陶韵试了几次，却开不得口。半晌，她突然象一只松开发条的木偶，"噢"地一声就扑进了萧木的怀里，紧紧搂着他的脖子，在他耳边不住地说："我……爱你，我爱你，我爱你爱你爱你……你真坏，坏木头！"一边说一边就破涕为笑。

　　萧木任她把鼻涕眼泪尽数抹到自己脸上，双手轻轻抚着她的背部，没有说话。他面部的表情说不清是一块死肉疙瘩呢，还是不是一块死肉疙瘩。

　　陶韵渐渐平息了，他才拉着她的手肩并肩地坐下，替她擦去脸上的残泪。陶韵不好意思地将手帕塞到萧木手里，然后闭上眼，任对方轻轻抚摸。萧木第一次这么近地看着陶韵，光亮的额头，轻盈舒缓的双眉，长而密的睫毛，笔挺的鼻梁和小巧红润的嘴唇，一切都是那么匀称、标致、迷人。看了半响，他正色说：

　　"陶韵，你必须明白，在今天以前我一直是抱定了终生不娶的……"

　　"噢，萧木……"陶韵睁开眼，凑上去，企图用吻把他的掐断。

　　萧木轻轻避开，语气沉重地说："……我8岁那年，母亲出卖了我的父亲，把他送进了监狱，而她跟着上司跑了。父亲平反后不久，也组织了一个新的家庭。我一直是个多余的人，无论在哪边……"

　　"萧木，噢，我的木头，我爱你……"

　　"……他们把我抛到这个世界上就撒手不管了。我只能自己去爬去闯，我恨他们。尤其仇恨恶毒的女人，尽管她们漂亮……"

　　"木头，……我不是那种女人，真的！我会让你幸福的……"

　　"幸福？我对这个词已经失去了信心……真希望你能帮我找回这种感觉……"

　　"我会的，会的，我发誓……"

　　萧木突然把陶韵扳到面前，盯着她的双眼说："听着，陶韵！我会倾注全部的心来爱你，我也要求你这样爱我。从现在起，你是我的！唯一属于我的，你懂吗？！"

　　"懂，我懂。"陶韵轻声作答，说罢就重新闭上双眼。微微努起的嘴唇传达着她的暗示和渴望，巨大的幸福如一袭轻柔的纱，蒙在她的脸上，和心上。

　　萧木抬起手，把她额前的黑发轻轻撩开。伏下身去，滚烫的唇从额头滑下来，迫不急待地去迎合那瓣鲜红的唇。倏地，一只小巧的舌头一掠而过，那种滑润让他感觉自己就象一条被赶进沙漠的狗，急切地寻找甘露般的甜泉。他的右手不由自主地从陶韵的颈部滑下，隔着厚厚的毛衣，触到了圆润而富有弹性的双乳。陶韵小声地呻吟起来，所有的感官都在快乐地震颤。萧木急促地喘着粗气，他感觉象被一只从未触摸过的巨手握着，慢条斯理、好耐心好耐心地把自己研成了粉末，再研成水……

　　直到这时，他才发现原来自己早就在心底深处默默爱着这姑娘。是那可气的自负和可笑的自尊合谋，一直把他蒙骗到今天。

　　——这才是真正演砸的地方！
