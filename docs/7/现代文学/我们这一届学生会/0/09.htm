第九节

　　陶韵跌跌撞撞，飞一样地跑了下楼。任凭董力昆在后面连声叫喊，也是不理。到了楼的门口，才放慢脚步。抬起手拢了拢长发，擦了擦眼泪，好象若无其事一样走了出去。小干事紧紧地跟在身后，眼中充满了关切。她想对自己的部长说几句安慰的话，但搜肠刮肚也翻不出合适的句子。小干事就很着急，后悔在家的时候，没跟着当里弄干部的妈妈学上几招儿。

　　在图书馆前，不巧就碰上了蔡闽义。他掖下挟着几本书，隔了看不清眉目的距离，就把喜洋洋的表情堆在了脸上。挥手大叫："哇——好巧啦！正要去找你就碰上啦！"

　　蔡闽义的舌头总是躲在喉管深处不肯出来，这就使他讲起话来永远象含着一个饭团。他说的挺费劲，听的人也挺着急，对双方都是莫大的难为。

　　"我刚刚从图书馆来，刚刚进了不少好书啦。你看，卡夫卡的《城堡》、加谬的《鼠疫》、马拉默德的《伙计》，还有……这一本，毛姆的，《刀锋》啦。很不错的，你要看的话可以让你先看的啦……"

　　"我可以自己去借……"

　　"借不到的啦！"蔡闽义摇头："抢光啦。刚刚有很多人在那里抢啊！"

　　"那我以后看……"

　　"还不回来的啦。这些书借到谁的手里也不会还的，借不到啦。"

　　"你还有完没完？我这一辈子不看了成不成？！"陶韵很恼火，说完就侧身向另一条小路走去。

　　蔡闽义楞楞地站在原地，伸出食指把滑落的眼镜推了上去。百思不解地望着陶韵，又百思不解地看着手中的书。

　　走了一段，陶韵发现小干事还跟在身后，就停下，说："你老跟着我干什么？"

说罢，也不等对方回答，又昂头前行。小干事的满腔热情被一瓢冷水浇灭了，就十分委屈地转身回去了。

　　情绪慢慢平息了，仔细回想着刚才发生的一切，陶韵心里就觉得非常窝火。萧木那不怀好意的笑，和故作聪明的口气，都让她感到蒙受了极大的耻辱。最近好不容易强迫自己对他产生出的那一丝丝好感，倾刻间又消失殆尽。

　　能参加这次夏令营，陶韵从心底里感激萧木，并开始重新认识这个人。她发现在他冷漠和散漫的外表下，其实有许多潜在的品质是别人不具备的。发现了这一点，她就决定要换一种态度来对他，并且要在适当时候对他表示感谢。

　　可是，到了桂林以后，从于勇的言谈举止里，陶韵渐渐就明白，这是萧木为了撮合她和于勇而精心策划的。整个夏令营期间，于勇就象一个恪尽职守的保镳，寸步不离地跟在陶韵身后。有好几次，在人稀树密的地方，他就鼓足了勇气想说什么，都被陶韵巧妙地避开了。她怀着一股报复的快感，肆意挥霍着于勇的忠诚。上车的时候，于勇奋勇争先为她占座；出门的时候，于勇任劳任怨为她拎包；饿了，于勇就慷慨解囊，把饭菜端到面前。而她，象个公主一样，理所当然地享受着一切。不仅如此，到了后来，与她要好的其他几位女同学也一并由于勇承包下来。对此，于勇是有些不满的。但不满归不满，却不敢在脸上表现出来。陶韵以及她的女友也就昧着良心假装不知，继续成全着他乐于助人的坏毛病。

　　这些事，回到学校以后也就淡了。人多眼杂，于勇也不好死皮赖脸继续扮演仆人的角色，彼此就回复了原来的正常状态。今天，萧木当着那么多人的面提起这事儿，虽是没有明说，但陶韵是有心病的，有心病的人当然就知冷知热。她就不高兴，就拉了脸。可谁知道，女人们用来对付男人的招数在萧木这儿就不好使。他比你还横！看他那样儿，要吃人似地。好吧，从此往后，咱们是井水不犯河水，谁也甭理谁！

　　在心里痛痛快快地骂了一顿，陶韵觉得挺解气，就平静了，也恢复了往日的自信。她轻轻叹息一声，继续向前走，不知不觉就到了那片被称为湖的大水塘。已是黄昏，看看表，过了打饭的时间。她就不急着回去，在一处极尽雕琢却又不想让人看出雕琢的假山傍坐了下来，夕阳悄悄地，就把她的白裙染成了桔黄的颜色。

　　能在大学校园里见到象陶韵这么漂亮的女孩子，应该说是个奇迹了。19岁的她，在西湖之水和赞美之词的双重喂养下娇蛮地长大成人，漂亮的脸蛋和柔美的身姿使她拥有了比别人更多的任性和成功。就连人皆应有的外号她也比别人多得多，可见世间事绝对是没有平等、公正可言的。

　　她那一大堆外号里，流传相对较广的是"桃花运"，女生们开玩笑的时候多用；男同学们背地里一般叫她"老九"，其意思是说她的美貌排在"校园八景"之后，算作第九景吧；此外，还有一些不堪入耳的叫法，比如"果脯"、"鞍子"、"老猫"、"小码"等等，每一种叫法只是在某个系或某个班的局部小范围里流行。出处及含意均无从考据，想必是男狐狸们蹲在葡萄架下面，过于无聊的时候瞎编出来的。

　　据说，和她谈过恋爱的男生是括号手指数加脚趾数括号的平方，注解：这里的手和脚全都是六指儿；据说，至少有两打半心眼儿不太活泛的家伙被她甩了以后，就满嘴胡话沿街刷写"反标"；据说，她跟好几位留校的"工农兵"小助教上过床也钻过草堆，起码有6次是被抓了"现行"的；据说，上初三的时候她就不幸被一伙小流氓给"落实"了，而那流氓小头目从此竟立地成佛金盆洗手了；据说，有一次躺在男甲的怀里却口口声声叫着男乙的名字，而男乙正是男甲的舅舅，于是男甲就拎着头发把自己痛揍了一顿；据说，出于人道主义的考虑，为了防止自残行为再度发生，她一直是随身带了一个小本儿，上面记满了姓名班级时间地点等翔实的数据；据说，各式各样稀奇古怪叫人心惊肉跳的据说还远远不止这些，不过呢，仅仅是据说信或不信要看你是谁。
