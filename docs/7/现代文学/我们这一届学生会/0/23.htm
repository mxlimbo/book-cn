第二十三节

　　一般说来，放寒假时校园里要比暑假期间冷清得多。能走的人几乎全都走了，那些一年只能回一趟家的学生，也都把有限的路费攒到寒假来使用。毕竟春节在人们的心里有着不可替代的神圣。

　　高安照例是不回家的。这一次除了经济据拮的因素以外，还因为他参加了市委党校举办的新党员培训班，整个寒假有一半时间是在市里度过的。期末考试之前，他终于是如愿以偿地加入了党组织，成了一名光荣的中国共产党党员。介绍人汪稚握着他的手，向他宣布了这个天大的好消息。当时，他并没有象自己想象中应该的那样激动，如果是在当知青的时候，或者是在部队，就算是在车间里听到这消息，他也会激动得流下幸福的热泪，他想。

　　上次汪老师找他谈话以后，第三天班主任就到宿舍来看他了。她坐下，第一句话就是埋怨他不该在评助学金的时候推推让让，把甲等让给其他同学。她还说，班委会经过了研究，决定向全班乃至全系进行募捐，帮助他家度过难关。高安望着班主任老师花白的头发，就激动不已。半响，才反对说："不能搞募捐，尤其不能向外班同学募捐，以免造成不好的影响。"班主任见他说得坚决，就说再与班干部们商量商量。高安后来又找班长，再三陈述了自己的理由。班委们见他态度强硬，就取消了募捐的计划。只是在班干部中间凑了一些钱和粮票交给他，班主任也拿出２０元生推硬塞地交到他手里。望着与自己朝夕相处的老师和同学，高安就百感交集，激动得说不出话来。

　　从第二个月开始，他的助学金就改成甲等了，学校还为他特批了一笔困难补助。寒假前，他接到了妻子的来信，信里说困难补助和同学们的捐款都收到了，除去给母亲交医药费以外，又还了一部分债，日子好过多了。还说，除了这些钱，还收到了一笔１００元的汇款，署名是"薛雷风"。她让高安代表全家向薛同志好好致谢。

　　读到这儿，高安就暗笑妻子连这么明显的假名字也看不出来。笑完就纳闷儿，就发誓要找出这位"薛雷风"。一直到后来他毕业留校了，就从学生工作部同事口中知道这位薛同志原来就是自己一贯敬重的汪老师。从此，他更是时时处处以汪老师为榜样。家属迁来以后，还特意带领全家向汪老师登门道谢。从此，两家人象走亲戚一样地你来我往，连孩子们也都成了好伙伴。这是后话。

　　萧木和陶韵也没有回家，他俩在春节前去峨眉山玩了一趟。去之前，也就是正式放假的第三天，他们买了一些罐头和水果，专程去医院看望了李冬军。

　　虽说是省里的精神病医院，其实是座落在邻近的县城。一路上倒了三次车，好不容易才找到了那里。在门口，他们就被一个警卫人员挡住了。说了一些好话，警卫又请示了值班的大夫，他们才勉强获准进去。

　　走廊里弥漫着浓浓的酒精和来苏水的味道，不时就传出病人怪异的喊叫。陶韵有点儿害怕，紧紧挂在萧木的胳膊上。萧木轻轻拍了拍她的手，回头一笑。

　　李冬军与另外5名病友同居一室，他的床铺靠近门口。这间病室的人大概是轻病号，看不出他们与平常的人有什么不同，只是更安静、更老实。见到他俩，李冬军好象并不惊奇。他平和地笑着，搬过椅子让他们坐。他比在学校的时候胖了一些，面色也很红润。胡子刮得很干净，露出青青的胡碴，使他看上去比以前更精明，也更持重。他们拉拉杂杂谈了一些有关学校和学生会的事儿，两个人都小心地避开了罢课的话题。一会儿，护士进来送药了，萧木和陶韵就乘机告辞。李冬军起身，与他们使劲儿握了握手。

　　经过医生办公室的时候，萧木就折进去，向值班的大夫打听李冬军什么时候能够出院。大夫抬起头，透过镜片眨着眼睛说，快则一两个月，慢则一两年。这要看巩固的情况才能确定。往回赶的路上，萧木比来的时候沉默了许多。他眼前一直晃动着李冬军的影子，赶也赶不走。

　　"我看他并不是挺可怕嘛。"陶韵试图引起萧木的注意，就说。她和李冬军虽说都在学生会里，却是连话也没说过，对他就不了解。

　　"这小子绝对是一人物。"萧木把目光从车窗外收回来，信口答道。

　　陶韵乘机把头靠在萧木的肩上，车里就有人向他们投来并不友善的目光。在市里下了车，他们买了一些食品和上山的用具。放假之前，陶韵给家里写信，说是要回家过年的。她妈妈就寄了路费来，这些钱再加上萧木的助学金，不算很宽裕，爬一趟山却也足够了。

　　他们搭了一趟慢车，在峨眉火车站下了车，就搭上开往县城的汽车。眼下是淡季，游山的人几乎是没有，从县城开往山下的汽车也停了。萧木就在一家商店门口向当地人打听怎么上山，那人惊奇地打量着他俩，说："找死哟，这个时候上山！"萧木笑着说："我们就是为了看雪景才来的。"那人就摇头，用手指指大路说："一直朝前走，大概有一个钟头的路。"

　　萧木抓过陶韵的手表，看看时间还早，便说："开１１号，怎么样？"

　　陶韵亲昵地笑道："跟着你，怎么都行。"

　　说罢，两人一前一后，迈开大步就向山的方向走去。

　　冬天的峨眉山有一半被厚厚的雪包裹着。逢到睛日，从山下望去，山脚依然是郁郁葱葱，山顶上却是白雪皑皑。金顶拨地而起，高大的电视转播塔就在阳光下闪闪发亮。出了城，人烟渐渐就稀少，陶韵也慢慢感到了吃力。萧木把全部家当横披竖挎地挂在自己身上，仍要不时地停下来等她。眼看天色渐渐晚了，他们就开始拦车。一连拦了两辆，汽车减慢了速度，但瞬间又加大油门跑了，陶韵就气得跺脚。又一辆卡车从后面开过来，萧木就让陶韵去拦，自己则伏到路基下面。车到了，陶韵站在路边扬起手臂，汽车犹豫了一下，就停下来。司机把车门打开，满脸堆着笑。萧木就从路基下窜了出来，拉着陶韵坐进了驾驶室，一面向司机递烟，一面就说些感激的话。司机象一个贪吃的人，突然就被美味佳肴噎住了，原本愉悦的脸色一下子就变得很难看。他面部的肌肉抽搐了几下，无可奈何地发动了汽车。

　　他们在"天下名山"的牌坊前下了车，不等萧木多说一句感谢的话，汽车就"呼"地一声开走了。他俩对望着，终于是忍悛不禁，憋了一路的笑猛地就喷发出来，捂着肚子蹲在地下。

　　萧木一边笑，就一边说："要不说女人……活得顺当呢！那丫那脸……象……打了麻药……"

　　陶韵上气不接下气，说："……在心里……说不定怎样……骂咱们呢！"

　　笑够了，两个人就站到郭沫若手书的"天下名山"石牌坊下留影。陶韵双手叉腰，显得气宇轩昂的样子。可惜的是没人帮他们合影，这使他们有点儿遗憾。

　　沿着上山的公路继续前行，不远处又有了一座牌坊。上面是将介石所题"雄秀西南"四个字，牌坊的柱子上长满了青苔。萧木说："没被砸了真是件怪事。"

　　天色晚了，光线暗谈。盘恒了片刻，就向报国寺走去。

　　报国寺是峨眉山的第一大寺了，也是上山的门户，平时进出是要门票的。或许因为是谈季，或许是天色已晚，他们进门时就没有受到阻拦。院子里，充盈着浓郁的香火气味，大殿里透出的烛火在青石板上反着光。一位年轻的和尚正在专心致志地打扫庭院，萧木就上前问："师傅，住宿登记处在哪儿？"

　　那和尚抬头打量着他俩，用手指了指左侧的通道，没有言语，依旧低头扫地。萧木见他的模样大约是20岁左右，心下就有些称奇，他悄声对陶韵说："如果你离开我，我就来这儿当和尚。"

　　陶韵听讲，就笑在脸上。斜了他一眼，说："真的吗？尽拣好听的说……"

　　萧木正色："我发誓！"说着，就举起右手。

　　陶韵急忙拦住："我相信。如果没有你，我也当尼姑。只是不知道人家肯不肯收。"

　　"心诚则灵。我对你真得那么重要？"

　　"当然。"

　　陶韵说着紧走几步，登上了台阶。

　　 他们两个是庙里唯一的游客，一个人占据了一大间客房。陶韵有些害怕，萧木就安慰说："咱们一宿不睡今晚，我陪着你。"

　　安置完毕，就取出面包吃了。耳边传来和尚们做晚课的鼓馨声，就带上门向大殿走去。大殿里烛火通明，香气缭绕。随着木鱼的节奏，和尚们抑扬顿措地吟唱着。众多的声音合在一处，忽高忽低，煞是好听。萧木站在门外，被殿内的气氛感染，神情不由得庄重起来。举头向释迦牟尼望去，那份慈祥、那份善良，使人不由得就倾身倾心。

　　突然，萧木感到鼻孔发酸。好象心里蕴藏着极大的委屈，就想扑到佛的脚下倾诉一番。他不明白这感觉是从何来的，急忙收回目光。再看陶韵，也是满脸凝重的表情。他轻轻捅捅她，两人回到现实世界，都不敢继续观看下去，就牵着手离去了。

　　第二天一大早，沿着崎岖的山路开始上山。路边有一家农户，就买了两根竹杖。来到清音阁，萧木把相机架在石头上，照了一张合影。脚下有两股清泉飞流而下，在牛心石相汇，然后就相拥相抱，一路欢笑着流向山外。陶韵指着泉水说：

　　"我是这股白水，你是那股黑水。这块石头是咱俩共有的……"

　　"白蛇青蛇这是，她俩就从这儿修炼下山，去了你们杭州……"

　　"那我就是白娘子，你就是许仙。"

　　"不吉利，悲剧！"

　　"咱俩不会，你说呢？……下个暑假咱们一起回杭州，在'断桥'再合个影，多有意义。"

　　萧木见她说得痴迷，不觉就动了情。拉起她的手，说："把'三妻''四妾'给我召过来。"

　　他把陶韵的脸蛋儿分了区域的，左边叫"三妻"，右边叫"四妾"，嘴唇就叫"糟糠"。听了他的话，陶韵果然就把脸蛋儿凑了上来，让萧木左右吻了一下，最后把"糟糠"递过去……

　　良久，萧木抬起头，望着满目青葱，呐呐道："多美……"

　　陶韵睁开眼睛，发现他夸的不是自己。就愠怒："我呢， 我不美吗？"

　　"当然！"

　　"哦，木头，我爱你，爱你。深深的……"说着，就势滚到了萧木的怀里。双手搂着他的腰，眼里充满炽热的火焰。

　　"当然。"萧木望着他，目光专注而温情。

　　"当然，当然。你就不会说句甜甜的话？"陶韵噘着嘴，假装生气。

　　"我说了，你没听见？"

　　"什么时候？"

　　"现在，用眼睛和心。"

　　"木头，你真坏！"说着，翻身坐起。用命令的口气说："喂，把'老黑奴'带上来！"

　　她以牙还牙，把萧木的脸叫作是"老黑奴"的。当时，萧木认为这叫法很贴切，忍不住就笑了。陶韵急忙制止，说你这一笑让我觉得老黑奴正准备跳海呢，求求你别吓着我。萧木也作势要打，直到陶韵连声告饶。

　 此时，陶韵在萧木额头亲吻了一下。又说："还有'家长'"。"家长"指的是萧木的嘴唇。

　　两个人说着、笑着、亲怩着。休息得差不多了，就继续前行。越往前去，竹木就越茂密，天气也越寒冷。来到万年寺的时候，四周已经有了积雪。

　　在那里他们遇到了另外一伙年轻人，5男2女。询问了一下，知道是另一所大学的学生。他们刚刚从山上下来，见到他俩，就劝阻，说前面大雪封山，根本过不去的。萧木笑了笑未置可否，对方一位年纪稍长的就正色说："是真的，我们这么多小伙子，租了铁抓鞋都没法上去，你俩？不行！"见对方说得严肃，萧木就不再笑。抬头望望山顶，脸色凝重起来。

　　当晚，两伙人在一起吃着各自带来的食物，在烛下闲聊。得知他俩是中川大学的，就有人问："你们学校罢课是怎么回事？"

　　萧木笑问："怎么，你们都知道？"

　 "收到过你们的求援信。"　

　　"这么说，你们是学生干部？"

　 "校学生会的。"其中一位指着年长者，说："他是我们张主席。"

　　萧木一听，就觉眼晴一亮，陡地拉近了与对方的距离。他介绍说："我俩也是校学生会的。"

　　对方听了也很高兴。张主席指着他俩，竖起两个大拇指抵在一起，勾动了一下，问："你俩是……这个？"

　　陶韵不好意思了，就抿着嘴低下头。

　　"你们学校让谈恋爱？"

　　萧木笑笑："那哪能呢？偷着呗。"

　　张主席就说："其实这条规矩挺损的。正是青春年华，男同学女同学在一个校园里朝夕相处呆上4年，不碰出火花儿来那就是有病了。能禁得住？"

　　听了这话，大家都笑了。萧木就问："我听说你们学校挺开放的？"

　　张主席就笑了，答："你从哪儿听说的？还不都是一样？其实学校的初衷是好的，怕影响学习。但这种事儿完全要看自己，那些想考研、想出国留学的，你逼死他，他都不谈。"

　　又东拉西扯地说了一些，就早早歇了。陶韵和那两个女生住在一起，不象昨夜那样害怕了。

　　第二天清晨，萧木被一阵奇怪的动静惊醒了。睁开眼，看见一只肥硕的猴子正趴在窗口向屋里张望。他一跃而起，穿好衣服抓过枕旁的提包就冲出了房门。来到陶韵门前，大声叫她来看。猴子受了惊吓，就长啸一声。瞬间，就见群山涌动，万木瑟瑟，数百只猴子从四面八方纷涌而至。只听树叶哗哗作响，落雪纷纷扬扬，声涛连天动地，象猛然刮起的一股强烈的山风。它们在树枝间跳跃飞行，象一只只轻盈的精灵，转眼就到了跟前。

　　望着这壮观的场面，萧木忍不住兴奋。一面大喊大叫，一面就从书包里掏出食物递给就近的猴子。其他人也从屋里出来，兴高采烈地周旋于猴群当中。陶韵特别喜爱的是一对母子，兴致勃勃地掏出花生、爪籽递过去。

　　一位住寺的小和尚看出他们是没有经验的，就上前制止。说喂猴子应该事先把食物放在手心里，不可以让它们看见你从包里取食物，否则要出危险的。

　　话音未落，一位女同学就失声尖叫。大家扭头，只见一头又高又大的老公猴把她的裤管扯开一条大口子，鲜红的抓痕清晰可见。因为她从裤兜里掏食物，被贪婪的老公猴看见，就打劫了。

　　大家慌忙驱赶，老公猴发一阵怪叫。猴群一反刚才的温顺，不约而同呲牙咧嘴，发出威胁的吼声。陶韵连跑带叫，首先冲回了屋里。男同学们保护着女同学且战且退，往屋子里转移。眼看到了门口，最初爬窗户的那只猴子就从屋檐上探下身，很轻松地从萧木肩头摘走了他的挎包。它三蹦两跳就窜上一棵大树，坐在树杈上开始翻检包里的物件。几只小猴想凑过去，都被它一一吓退了。它很熟练，把包里的东西一件件翻出来，能吃的就吃了，不能吃的就随手一扔。萧木站在树下，又是恫吓、又是央求。对方不予理睬，照样专心致志地犯罪。

　　包里的食物并不多，猴子对打劫的战果显然不满，就把里面的衣服和用具胡乱地扔。一条裤衩挂在树枝上，象欢呼的小旗，叫萧木哭笑不得。其他同学闻讯赶来的时候，那只猴子正打开钱包，把钞票一张张抽出来，扔出去。花花绿绿的纸票子在风中飘忽，落下深不见底的山涧。陶韵忍不住就呜呜地哭出声来，萧木探头向崖下望去，山雾阻挡了他的视线。他只得回身把陶韵搂在胸前，说一些宽慰的话。有人去请教了寺里的和尚，回来就说，要是夏天还可以雇人下去捡，如今可没人敢冒这个风险。萧木明白是没有希望了，就平静下来，扶着陶韵返回了房间。他俩把各自身上的钱掏出来，凑在一起还不到8元钱。这点钱想回学校是不可能了，陶韵担忧地望着萧木。萧木尽力掩饰着内心的焦虑，装出一副充满信心的样子，说："我有力气，可以打工挣钱。人还能让钱给难倒？"

　　陶韵抽噎起来："这山上到那儿打工呢？"

　　萧木拉着她的手："咱们走回县城，在那儿可以找到工来做。放心吧你，一切有我呢！过来，让三妻四妾们高兴起来。"

　　陶韵含怨白他一眼，脸色略有开朗："都什么时候了还开玩笑。"

　　那伙大学生敲门进来，萧木连忙问："那位女同学怎么样了？"

　　"主要是受了惊吓，伤倒是不重，但也需要及时处理。"张主席说："我们决定马上下山。希望你俩跟我们一起走，否则你们没法回到学校。"

　　"这……"萧木望着陶韵，不知如何是好。

　　"甭客气。谁让咱都是大学生呢？而且还都是学生会的，这叫有缘。赶紧收拾行李走吧！"

　　萧木与陶韵对望一眼，想想再没有别的办法了，就只得这样。路上，萧木十分卖力地照顾受伤的女同学。每到一处，跑腿出力的事就由他去办，他是想用这种办法补偿人家对自己的情份。开始，大家劝他不要这么见外，后来了解了他的性格，也就由他去了。

　　这次邂逅使双方结下了真挚的友谊，后来在萧木的倡导下，经过双方学生会的极力撮合，两所大学就结为姊妹学校。这也是后话。
