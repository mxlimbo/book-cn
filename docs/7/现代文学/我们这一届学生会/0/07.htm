第七节

　　宝成铁路象一条逶迤的蛇，不疾不速地盘旋在厚大的秦岭山中。在一个叫不上名的小站，冰冷的钢轨伸向神秘的远方。钢轨的尽尖，两只信号灯闪着幽灵一样的蓝光，愈发使大山的夜深不可测。站台上空无一人，一只昏暗而单薄的电灯悬在空中，照亮了站房前面一小块地面。蛐蛐的叫声很飘、很密，偶尔还有几声蛙呜。天气冷，空气里是雾，可以想见落叶上一定是沾满了露水。亮亮的，凉凉的。

　　李冬军蜷缩在侯车室唯一的一张长椅上。越是想睡片刻，就越是睡不着。饥饿和寒冷象两头虎视眈眈的野兽，交替向他挑衅。他脸色憔悴，胡子显得更浓、更密，也更脏。一列货车轰隆隆地驶过，地面就瑟瑟发抖。他一骨碌翻身坐起来，隔着窗户望着飞驰而过的列车，感到黯然神伤。就推开门来到站台，值班员提着信号灯，表情漠然地目送着远去的列车。回头看见了李冬军，就走上来，说："还有两个钟头呢，你可以多睡一会儿。"

　　李冬军摇了摇头，说睡不着。那人就掏出烟递过来，李冬军摆摆手拒绝了。那人不再谦让，掏出火柴点燃。吸了一口，说，你们大学生都跟你一样？李冬军苦笑着，没有答话。谈话不能深入，那人就失去了兴趣。闲扯了三五句就要回值班室休息。临别，又嘱咐，说你尽管睡去，到时我叫醒你。李冬军一个人在站台遛哒着，身影忽长忽短。一阵寒风刮过，猛地就打了个冷战，一阵透心的凉。于是跺跺脚，缩着脖子就回了候车室。

　　他是因为无票乘车被撵下来，滞留在这儿的。因为每天只有三趟慢车在这儿停靠，他就必须耐着性子等上6个钟头。值班员看他面生，就好奇，主动找他搭话。听说是个大学生，就动了恻隐之心，答应下趟列车来的时候把他送上车。他吹嘘说，自己跟那趟车的车长是老熟人了。李冬军表示感谢，那人就乘机问一些大学里的事。在他看来，大学是个遥远而神秘、高贵而华丽的词，就想多知道一些，日后讲给自己的孩子听。怎奈李冬军心里有火，谈兴就不浓。那人感到十分失望，好象是白白地错过了一个好机会。不过，有一点他是明白了，那就是大学里也有穷人家的孩子。这个发现使他对大学的景仰有所减弱，甚至在心底掠过了一丝快意。

　　但他哪里知道，李冬军可不是穷人家的孩子。他的父母都是部队的高级干部，他也就是人们常说的那种"高干子女"。14岁那年，父母被下放到了湖北的乡下，哥哥姐姐也忙着串联和造反，就只有年迈的姥姥和他相依为命。捱到了高中即将毕业，就想逃避"上山下乡"。他不辞而别，无声无息在外面流浪了4个月。回来的时候，已是篷头垢面、衣衫褴褛。姥姥在急病交加之中，已经撒手人寰。破例被准许回京办丧的父母望着面黄肌瘦、营养不良的他，就不忍心多加埋怨，只能是相对落泪。李冬军对这一切无动于衷，用漠然的目光望着陌生的父母，不落泪，也不多话。父母要带他一同返回干校，也被他冷静地拒绝了。

　　这以后，他就独自一人留在了北京，靠父母寄来的生活费和哥哥姐姐的接济过活。有兴致的时候，也和伙伴们一起拣废铁卖破烂，没兴致的时候，就躲在床上读书。那一段时间，他埋头研读马恩列斯毛的著作，写下了大本大本的读书笔记，和一些政治理论小组也建立了联系。

　　到了1976年，部队大院里的邻居们就很少见到他了。好在大家对此已经习以为常，提到他的时候，心肠软的阿姨就叹息，说："小军这孩子，倔得象头牛，危险哪！"4月中旬，有消息传来，说他在天安门广场被抓了。果然，随后就有一些人到他家翻箱倒柜。那些保存完好的读书笔记成了他的罪证，使他身陷囵圄。父母也因为"把复辟的希望寄托在下一代身上"而罪加了一等。

　　粉碎了"四人帮"，他父母平了反，官复原职就回到北京。分别在内蒙古和黑龙江当知青的哥哥、姐姐也都先后回城。过不多久，他也被无罪释放了。全国恢复了高考，他就埋头复习，分别参加了77年和78年的高考。第一次因为他的问题还留着"尾巴"，政审时就卡了下去；第二次是在户口问题上出现了波折，等居委会与知青办之间推诿扯皮告了一个段落的时候，录取工作也告了一个段落；一直到1979年8月，他才被中川大学的哲学系录取了。中川大学是他的第4志愿，报的专业不是哲学而是政治。因为他在志愿表上填了"服从分配"这四个字，就近就划拨到了哲学系。为此他不止一次地发牢骚，说哲学和政治完全是两个不同的概念嘛！听那口气，他的不如意仅仅是因为录取人员弄混了这两个名词的缘故。　

　　在是否增补他为学生会委员的问题上，汪稚最初是踟踌再三的。一方面，他象发现珍珠一样在79级学生里发现了这个难得的人才；另一方面，却因为他的经历过于复杂而不敢贸然起用。几经权衡，又与高安多次商量，最终还是决定大胆提拔。"凡是敌人反对的我们就要拥护，凡是敌人拥护的我们就要反对。'四人帮'是我们的敌人，而李冬军同学呢，是反对'四人帮'的勇士，那么李冬军同学就是我们可以信赖的同志。象这样根红苗正的同志都不去依靠，那么谁会高兴呢？想一想！是敌人哪！同志。"他把这一番突然想通的道理讲给高安听，连自己也大受感动。这是多么严密的逻辑啊！可见严密的逻辑必然就引导我们的思维沿着正确的方向走向光明！

　　高安提议让李冬军担任了学习部的副部长，这在79级学生里算是破格提拔了。80年代嘛，"不拘一格降人才"的口号叫得震天价响，学生会当然也不能例外。委员们在习惯性地举手之后，就向学习部部长于勇表示祝贺："嘿！大甩，这下你们部的工作没说的！"

　　大甩是于勇的外号。不光是他，学生会里每个人都有外号的。说不清这些外号是怎样产生的，反正不知不觉的时候，一个人的符号就变了。李冬军也难逃此劫，时间不长，人们不再叫他李冬军，而是叫"五车"，据说取自成语"学富五车"。他是学习部的副部长，叫他"五车"也是恰如其份的。

　　开始的时候，大甩和五车之间的合作还算是顺利。但不久，情况就有了变化。大甩有时就当着董力昆和萧木的面抱怨，说五车那小子邪性。这样抱怨了一段时间，就干脆找高安要求调换工作。问原因，他也说不清。只是把额前的长发猛地向后一甩，说哪怕是当个小干事，也绝不在学习部呆了。转过头去问李冬军，他更是莫名其妙。疑惑地抬起头，皱着眉，说："挺好的，我们的合作挺好呀。"然后就一言不发了。无奈，高安就把于勇调离了学习部，让他担任了学生会秘书长。这样，学习部实际上就是五车这个副部长在主持全面工作。

　　一列快车从南向北飞驰而过，车窗连成了一条长长的光带。那些倚近窗口的脸一闪而过，没有人能看清楚他们的模样，正好象如没人知道他们从哪里来，又要到哪里去。和李冬军一样，他们也是这个小站的过客，只不过来去得更加匆忙而已。

　　上个学期一开始，班上的同学就发现李冬军的情绪有了一些显而易见的变化。那双忧郁的眼睛越来越多地闪出明亮的光，好象有一个天大的秘密正在激励着他，让他亢奋。他经常是毫无道理地激动，莫名其妙地发脾气，言谈和举止都叫人百思不解。好在班上同学都知道他一贯是神经兮兮、捉摸不定的，就不多在意。他在班里另有一个外号，是叫"神哥"的。这显然比"五车"包含着更多的尊重，也包含更多的揶揄。

　　他的确是有秘密的，上个学期，他的来信突然增多了。写信的大都是过去在北京时的旧伙伴，如今他们分散在各个大学，并成了那里的学生干部。无论是在思想上还是在行动上，这些人都是当前高校里面最活跃的一部分。信中，他们向李冬军通报了自己学校罢课的消息。这些消息就象纷乱的鼓槌，敲在李冬军沉寂已久的心上，叫他激动，叫他震颤不已。他想象着战友们如火如荼的斗争，脑海里一遍又一遍地重现"四五"运动那波澜壮阔的场面。这种时候，他就昂奋得象一头发情的牯牛，食不甘味、寝不安睡。

　　与其他学校的暴风骤雨相比起来，中川大学平静得就象一潭死水。这种微波不兴的死寂，被李冬军归咎为观念的陈腐、思想的萎缩、品质的颓败和行为的堕落。他一面在沉寂中急不可待地诅咒，一面却又不得不在沉寂中心神不宁地等待。他象一匹磨利了牙齿的狼，支起长长的耳朵，孤独地躲在漆黑的夜里静静地谛听。住监狱的时候，他经常就是这样整夜不眠，躺在床上瞪大眼睛，一动不动地透过窗口凝望浩繁的星空。这时，他就进入了一种不可知的神秘状态，思维飘忽体外，在一些非常复杂奥妙无穷的迷宫里兜着圈子。他非常清楚自己作为一个"政治动物"，在生活中应该扮演什么角色。他渴望行动，象离不开水和食物一样，他无法阻止自己去幻想和设计一场又一场狂飙般的行动。这种剧烈、尖锐、动荡、偏执的幻象，让他获得了人生最大的满足和愉悦，真正体验到了自己的价值。

　　长这大了，他没有过恋爱的经验。除了自己的母亲，他是从没认真仔细地看过任何别的女人。男人和女人在他的观念中不过是两个有着微小差别的概念而已，根本就不存在形象上的立体感。他面对着女同学的时候，其实他的目光早已穿透了她们的身体，而落到了一个遥远的不可知处。他与她们讲话，纯粹是人与他的同类们不得不进行的信息交流，没有丝毫感情的色彩。因此他的语音总是干巴巴的，象铁锨铲土，又象老人在倒炉灰。说起来叫人很难堪，他甚至从来就没有手淫过。有的时候，早晨醒来在被褥上发现了一些粘液，他就惊奇，就不知所措。

　　睡在他上铺的是一个外号叫"老枪"的胖子。有一次，半夜操练的时候，李冬军就起来，摸着老枪的头，关切地问是不是病了。老枪自然气恼交加，其它同学醒来，也忍不住哈哈大笑。李冬军疑惑地望着他们，很费解地摇了摇头。第二天，全班的男生就都知道老枪昨晚是放了哑炮，挨了一记"窝心枪"。见了李冬军，他们不怀好意，就打趣说："哎我说神哥，人家老枪干活的时候你甭舍身堵枪眼好不好，这不人道。"说完，就发出一阵猥亵而满足的狂笑。老枪在背地里就恨恨地骂："也不知他妈是真不懂还是装糊涂，看他胡子拉茬的不象个二尾子呀！"此后不久，老枪找了个借口，就搬到了原来堆放衣箱的那个铺上。

　　睡意朦胧之中，李冬军被值班员轻轻地捅醒了。他坐起，伸了个懒腰，抓起提包来到外面。天色已经露出了晨曦，夜色是很浅了，远远传来的鸡鸣声细微但却清晰。寒风一吹，李冬军就打了个寒战，抱紧双肩缩着脖子望着铁轨消失的尽头。值班员披着一件薄而且破的军大衣踱来踱去，嘴角的烟头一明一灭。走近了就没话找话地说上一两句，不说的时候就踱到远处。他面容平和，好象火车到与不到，早到与晚到都是一件与已无关的事。对于李冬军内心的焦虑，他不会也根本就不想知道。

　　火车终于来了，一道强烈的光柱刺破了残留的夜。这时李冬军才发现车站的前面是一个山洞，车站刚好是建在半桥半涵的山腰的。列车喘着粗气停稳了，车上下来一些农民。他们缠着头帕，背着背篓，循着陡直的石阶径直就向山下走去。值班员一边和女列车员打情骂俏，一边就招呼李冬军上车。李冬军想向列车员说一声谢，但对方却是看也不看他一眼。值班员也好象也忘记了他的存在，脸上带着猥亵的笑，十分投入地说着只有他和列车员明白的、内容肯定下流的笑话。李冬军就放弃同他说再见的念头，走进了车厢。旅客不多，大部分的人们都在昏睡。一个孩子在哭叫，他的母亲就轻声地哄。李冬军找了一个空位子坐下，几位上了年纪的老人睁大眼睛望着他，面容和善而憨傻。列车开动了，李冬军就把头转向窗外，望着小站一点点消失。心境象渐渐明朗起来的天色，现出了一抹霞红。

　　这个暑假，李冬军有一半的时间是在火车上度过的，另外的一半则是在各个大学的校园里。他象一位匆匆赶场的"票友"，马不停蹄，从一所大学赶往另一所大学。所到之处，就受到罢课学生领袖们的热烈欢迎。这一方面得益于他言词偏激的演讲，更主要的还是靠了他的头衔。学生领袖们宁肯把他看成是中川大学学生会派来的声援代表，他们在群情激愤的学生集会上，就是这样介绍李冬军的。李冬军听了也不反驳，这并不是说他是在行骗，在他的心底，理所当然地认为事实本来就是如此嘛。行骗一旦成为了一种职业，行骗者本人也会被自己的谎言感动的。中川大学在全国很有影响，它的支持就使那些准备草草收场，或者根本就将胎死腹中的罢课行动陡然间再掀起一重波澜。就象街坊大妈吵架，随便一个毫不相干的路人说上一句话，被支持了的一方就会气焰万丈，又拍屁股又跺脚，好象她真的就挺有理。

　　这一个多月的时间，李冬军实在是过足了斗争的瘾。在东北，他跳上饭桌，振臂高呼："中川大学与你们同在！"狂热的学生们就放下手里的饭盒，把他抬起来，抛向空中；在长沙，他头上缠着发带，在数千名学生的集会上慷慨陈词，极尽煽动之能事，紧接着又和学生们一起上街静坐，游行示威；在北京，他把身上仅剩的钱，连同手表一古脑就捐给了罢课组织委员会，然后不名一文又踏上南下的征途。两次在北京停留，他都没有回家。对他来说，家是一个既模糊又空洞的概念，只有眼前的斗争才是真实的，催人奋进的，欲罢不能的。

　　在北京的时候，他向自己最要好的朋友二毛详细讲述了中川大学的情况，对这种无动于衷的平静表示了极大的愤慨和失望。二毛以过来人的宽厚和先行者的经验鼓励他、帮助他，让他感到从此肩头就压上了历史的重担。此番回校，要是不能把中川大学的铁盖子掀开，那就真是愧对了时代、愧对了战友、也愧对了自己！临别，他紧紧地握住二毛的手，象出征的战士告别亲爱的首长。虽说没有战地宣誓的动作，但其坚毅的表情和握手的力度，都饱含着壮士一去杀身成仁的悲壮。正因为这样，一路上不管是餐风露宿还是蒙羞受辱，他都用那颗宽容的心承受了下来。他已经把自己摆到了伟人的位置，既然天降大任于斯，当然就要饿其肌腹，劳其心志的。想到这些，他就可怜起身边的芸芸众生，比如把那个自己赶下车的蛮横的列车员、比如车站上那个无所事事的值班员、比如这些忙着赶路的农民……
