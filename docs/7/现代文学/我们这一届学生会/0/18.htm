第十八节

　　不知是出于革命人道主义的关怀，还是受了共产主义风格的感动，女医生把高安两次献血这件事报告了院长。她说，那小子情绪不对，该不会出什么事儿吧？院长和汪稚是住在同一幢楼的，闲聊的时候就把这事儿讲了。他说，那小子情绪不对，该不会出什么事儿吧。

　　汪稚听了，心里就犯嘀咕。高安是他首先发现，又一手提拔起来的优秀学生干部，对他的一举一动就格外关心。他把高安叫到办公室，就问："小高呀，听说你一连两次到医院献血呀？"

　　高安苍白的脸有些泛红，说："我……是……"

　　汪稚为他倒一杯茶放到面前，说："你看你，脸色多不好。学校号召同学们自愿献血，可你也不能这样不管不顾呀！"

　　高安用茶杯焐着双手，抬头望着汪老师："汪老师，我……"

　　"你说说，是不是还有另外的原因呀？"

　　"没有……"

　　"你不说我也看得出来。最近，你的情绪不高，脸色也很不好嘛。"

　　"………"

　　"这就是你的不对了，有什么思想活动都可以讲出来么！我解决不了还有组织呢。小高呀，要相信组织嘛，啊？"

　　汪老师已经把"组织"搬了出来，如果再瞒下去，就说明自己真的存有"二心"了。高安把茶杯放到茶几上，从书包里翻出了一封家信。递给汪稚，说："是……家里出了一点儿小事儿……"

　　汪稚的手伸出去，但中途却犹豫了。他心中暗忖，会是什么事儿呢？高安自打入学以后就没回过家的，会不会是……？他有些后悔，不该刨根问底，这么热心地掺和别人的家事。他望了高安一眼，高安的目光中充满了信任。

　　信封皱皱巴巴，显然已被翻读过多次的。看看邮戳，一个多月前罢课期间就寄到了。抽出来，信纸是从小学生的作文本上撕下来的，字迹很潦草。因为是用铅笔写的缘故，有折痕的地方已经模糊不清了。信是高安的妻子写的：

　　高安：

　　　　你好！

　　来信已经收到，寄来的钱和粮票也都收到了，你放心吧！

　　高安，这封信是我瞒着爸爸，等孩子们睡了以后一边流着泪一边写的。按理说，你学习工作那么忙，我不该给你添麻犯（烦）。爸爸也说不要打扰你，可我实在没办法呀！你是我的丈夫，孩子们的父亲，我不找你又找谁呀！

　　去年玲玲被开水烫伤，至今还有几百元的饥荒没有还清，这你是知道的。我白天上班，晚上帮一家出版社抄抄稿子，想多挣点儿钱贴补家用。妈妈那么大年纪还要去建筑工地筛沙子，为得是早日把账还清。可谁想到，前天妈收工回来，因为天很黑，她也累，不小心掉进了工地的沟里，摔断了右腿。我好不容易凑了一些钱送她住了医院，可这一家老小的日子又怎么过呀！爸爸从厂里借不到钱，我在车间打的"会"钱也不多，朋友们也不肯再借了。如果有一点点办法，我也不会向你诉苦呀！

　　我知道你每个月就那么点儿助学金，即（既）要吃饭又要买书，连烟也戒了。自己还顾不过来，每月还得往家寄，已经很不容易了。能不能对你们学校讲讲咱家的困难，申请个补助什么的？要不就找同学老师借点钱救救急？如果你觉得为难就算了，我另外想办法，千万别为这事影响你入党，不要影响学习。

　　小勇和小玲都很想你，你已经三年多没回家了，孩子们都快认不得你了。我总是哄他们说你快毕业了快回来了，说得多了，孩子们也不相信了。

　　………

　　没有读完，汪稚的眼泪就不知不觉滴了下来，落在信纸上。

　　"小高，你……你在卖血！"

　　高安抬起头，抹了抹眼眶，甚至还逼着自己笑了一下。

　　静默片刻，汪老师抖着信纸说："家里这么困难，你怎么不早说呢？"

　　"我……不想给组织上添麻烦。"

　　"你的助学金是多少？"

　　"十七块五，乙等。"

　　"有困难补助吗？"

　　"没有。"

　　"你呀你呀，这么困难才评个乙等！……你每月往家寄多少？"

　　"五块。"

　　"嗨！剩下十二块五够干什么的！真是……难为你了。"

　　说着，他就走到高安身旁坐下，把信还给他。用力拍拍高安的肩膀，神情是既关切，又透着嗔怪，叫人以为下面的话一定是"这都是我们当领导的失职呀！"

　　然而，汪稚并没有落入这个俗套。他停顿了片刻，说："这件事儿交给我来办吧，我向学校打报告为你申请困补……"

　　"汪老师，别，这不行……"

　　"什么不行？你看你瘦成什么样了？学习这么紧张，工作又那么繁重，你呀你……嗨！这都是我这当领导的关心不够，失职呀！"汪稚终于还是没有脱出俗套子。

　　高安眼里噙着泪水，充满感激地望着汪老师。好些话一齐涌到了嘴边，却一句也说不出。是呀，"困补"虽然不多，但对他的家庭却是能够派大用场了。入学以后，他一直严格要求自己，不与同学们争抢助学金。当初评上乙等时，就些小同学背后议论，说他带工资上学还拿乙等，一定是靠了什么关系。可家中的困难又有谁知道呢？

　　汪稚起身，取过暖壶为高安添上热水："小高呀，以后可不能再去卖血了。身体是革命的本钱，身体垮了，还怎么去干四化？要相信组织，相信学校嘛。"

　　高安点点头。

　　汪稚挥了挥手，象要赶走压抑的空气，换了个话题，问："好了，不说这事儿了。小高呀，再有一个多学期就要毕业了，有什么打算吗？"

　　"这……还没想过。"高安想了想，又补充说："服从分配呗！"

　　汪稚背着手踱了几步，走到桌后坐下，若有所思道："以前也向你透露过一些。我想呢，争取让你留校。你看，学生工作还得进一步加强。以前，都是团委主抓学生工作，组织上不大顺，啊？学校正在筹备成立一个学生工作部，人手不够呀。我向党委推荐了你，不知道你愿不愿意舍弃专业，改做政工，象我这样？――当然，你还可以选择。至于家属嘛，这个问题有一定的难度，你爱人做什么工作的？"

　　"工人。在一家五金厂……"

　　"哦――这的确有一定的难度。我再想想办法……"

　　高安听到此，就站起来，脸色涨红说："太感谢汪老师了，我保证服从分配……"

　　"唔，很好！学生会的工作还不能放松，站好最后一班岗嘛！你们明年还要实习、写论文，学生会班子要进行一些调整。不过在岗位上一天，就要会心全意抓好工作，可不能松懈哟！"

　　"汪老师请放心，我一定不辜负您的期望！"

　　"嗯。你看董力昆这个同学怎么样啊？"

　　"他嘛，工作很负责任，热情高，团结搞得好。在学生会中威信还是有的。"

　　"萧木呢？"

　　"他……这个同学本质是好的，很真诚也乐于帮助同学。只是……只是同学们都不太喜欢他。"高安想起萧木给自己闹过的难堪，忍不住就皱了皱眉头。

　　"哦，那是为什么？"

　　"主要是大家看不惯他的作风……"

　　"看人还是要看本质的嘛。其他同学呢？你认为谁适合全面工作？这毕竟是你的接班人嘛！"

　　"我看，董力昆就挺合适……"

　　汪老师若有所思地点点头，站起身道："好吧，今天就谈到这儿。回去多做政治思想工作，让大家争取早日解决组织问题。哦，对了！那封信能不能先放我这儿？"
