第十六节

　　吴绍中一直在寻找接近萧木的机会，而萧木却总是有意无意地躲着他。两人虽说见面很多，但能坐下来谈谈心却不是一件容易的事儿。

　　这次罢课，给吴绍中心里留下了说不出的滋味儿。当初，他带着学生到家属区示威的时候，绝对料不到李冬军半路杀出来，就把同学们领上了罢课的路。他自己象离了弦的箭，又象是个笨手笨脚的溺水者，在潮流的推拥下身不由已就成了罢课的领导者。整个罢课期间，他的心始终处于矛盾的状态。一方面，他真切希望通过此举使凶手早日伏法，为川娃申冤；另一方面，眼看着事情越闹越大，接二连三提出新的罢课理由，就使他惴惴不安。不知事态会怎样发展，最后将怎样收场？复课以后，他听说了萧木在这期间的所作所为，就大为感动。特别是说服大黑让搓板自首这件事，叫吴绍中更是对他另眼相看了。他是个心直口快重义气的人，心里憋了好些话要对萧木推心置腹地说。他巴望着俩人尽弃前嫌，亲密相处。

　　以前他和萧木之间没有过多来往，就谈不交情。他在化学系，而萧木是在教育系。文理相轻是老祖宗落下的毛病了，传到今天就更是深入人心。不是东风压倒西风就是西风压倒东风，眼下正值"科学的春天"，科技兴国的观点就占了上风。高中学生在划分文理科的时候就唱："学理不如学工，学文不如卖葱！"。进了大学，继续唱："工科是主人，理科是佣人，文科是仆人。"文科学生在夹尾巴之余，也幸灾乐祸，唱："教育教育，粉笔讲义！"、"三尺讲台一张嘴，能把学生哄成鬼！"以此表明有教育系学生垫底，自己还不是最惨的一类。有人向他们指出，教育和师范不是一回事儿，他们就一楞，说，差不多，都是哄孩子的。

　　吴绍中不至于如此势利，但也没有发烧到跑去教育系交朋友的热度。一起进了学生会，他是体育部长，萧木是宣传部长，一个玩儿的是四肢，另一个玩儿的是笔杆子。要不是运动会期间互相之间配合几天，平时连鸡犬之声也是不多闻的。何况，萧木这家伙嘴皮子太损，油腔滑调，一肚子发不完的牢骚，听了叫人生厌。还有，这人神秘兮兮，摆出一副玩世不恭的样子，牛逼大样玩儿深沉，捉摸不透。不象这些搞体育的哥儿们，快言快语。浅是浅了点儿，可是痛快！两者是皮肉不沾连，中间垫着太多的隔漠。

　　吴绍中的朋友大多是在运动场"拍"上的。当了体育部长，他更是每天泡在跑道上，把一对小眼睛眨巴得欢欢实实，四处盯人。被他盯上的人被称为"苗子"，他上前，在"苗子们"肩上亲热地拍一下，就拍到了小食店里。等到酒精把舌头变成了多余的肿块儿，就瞪着红眼珠子，叮嘱"苗子们"今后要多多关照。"苗子们"拍胸脯挽袖子，"这位大哥您说谁吧想烂掉哪位您只要把手一指哥儿们指哪打哪儿皱一下眉头我是你孙子罚我再喝半斤，咯！""我操！不是找你们打架打架我自个儿就能对付是学校的田径队……""闹拧了怪您没说清看样子您真喝多了，咯！""是你听错了你才多了呢我操行不行给句痛快话。""不就是田径队吗你说怎么干咱这一百多斤就交给你了……"

　　吴绍中本人是学校投掷三项的冠军，两项校纪录的创造者。别的且不说，那一身结实的犍子肉，硬鼓鼓地随时准备破衣而出，当然无需"苗子们"为他挡祸消灾。酒醒以后，"苗子们"知道在鲁班爷爷门前卖弄了一回斧锯手艺，就暗中惭愧不已，执蹬衔铃之心就愈是可以敲出钢音儿。所以，吴绍中在运动场上咋唬起来，总是一呼百应的。各个运动队的队长同他的关系，烧一柱香就是八拜之交，不烧香也是磕头弟兄。特别是刚组建不久的校足球队，时间不长就声名大振， 号称"铁脚老虎队"。那队旗是吴绍中自己掏了腰包从商店买回来的，一张绣着下山虎的沙发巾。

　　萧木对吴绍中这些事儿谈不上了解，也不想去了解。他对任何人任何事几乎都是漠不关心的，至少表面上看起来是这样。罢课结束以后，吴绍中找他套近乎，他岂能不知道？但他认为这太孩子气，闹憋扭的时候彼此不说话，给两粒糖果就搂肩搭背地表示亲热，忒俗！所以，复课已经半个多月了，他一直避免与吴绍中单独相处，有意不给对方致谢的机会。

　　今天是个星期六，因为电视里转播审判"四人帮"的实祝，学校的电影晚会就被取消了。学生们围在宿舍楼前，一眼不眨地盯着小小的屏幕。电视机的音量已经拧到最大了，还是有人喊听不清。前排的学生忍不住就议论，说江青还他妈挺横，后排的人粗暴地制止说，听她的还是听你的！萧木没有同大家争挤，他跑到系办公室与几位小助教一起看。看完，忍不住就想喝酒。独自来到学校小吃店，要了一瓶尖装大曲和一碟花生米，自斟自饮就喝。

　　角落里有几位低年级的女生围坐在一起，显然她们有比审判"四人帮"更重要的事儿。其中一位正在极力模仿着电影里的镜头，举起一杯红色的葡萄酒向另一位说着"生日快乐"之类的吉祥话。那个过生日的女孩儿过去做寿一直是吃炸酱面的，乍一改成葡萄酒还挺不习惯，脸上的怯意红扑扑的；另外一张桌前，两个男同学正在低声交谈，不时发出会意的痴笑。但凡这种表情，一般只能让人联想到专门收集艳闻的私人侦探，绝对当不了太大的间谍。

　　"唿喇"一声，门突然就开了。一股冷气把吴绍中连同他身后的几个"苗子"一并就送了进来。那声势叫人以为梁山好汉们果然就打下了青州，马上就应该听到"打三坛好洒，切二十斤牛肉"的吆喝声了。然而没有。只有吴绍中那比草寇更粗俗、更单调、也更直率的口头禅：

　　"我操！木头，你在这儿！"

　　萧木抬起眼皮撩了他一眼，没有言语。片刻，就把桌上的酒瓶推了过去。顺手从烟盒里抽出一支烟，用手里的烟头对燃，再把烟盒推到吴绍中面前。

　　吴绍中抓过酒瓶喝了一大口，把嘴一抹，头也不回就喊："点菜！"

　　喊着，一屁股就坐下，椅子发出了痛苦不堪的呻吟。他从萧木有烟盒里抽出一支烟，点上。又从自己上衣口袋里掏出一包"大前门"，扔到桌上。"苗子们"忙前忙后，一面张罗着备酒置菜，一面就拖过椅子坐下。

　　或许因为萧木的表情过于冷淡了，吴绍中满怀的热情就凝成了水珠。那些象烙饼一样在心里翻过来倒过去的感谢话，一时就拎不出来。"苗子们"见头儿没话可说，自然也就不敢喧闹。掏出了烟互相递让一番，桌面上就摆满了沉寂。幸好酒菜上得及时，一桌人如蒙大赦。咋咋唬唬地指挥着杯盏的摆放，以此鼓励着自己与尴尬作斗争的勇气。

　　吴绍中用牙把瓶盖咬开，斟上一杯酒，双手捧了，说："木头，这一杯我……"

　　"块儿，今儿个只喝酒咱们，别的一律甭说！"

　　"这……"

　　"要不然我走。我带回去自个儿慢慢斟这半瓶。"

　　见他说得坚决，吴绍中也不再多言。把杯中酒一口就倒进了嘴里，伸手挟了一块鸡肉，说："我操！你自便！"
