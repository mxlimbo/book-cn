第三节

　　萧木正要离开的时候，门外就来了两个学生。很显然，他们是不习惯同领导打交道，推开了门就犹豫，不知道是不是应该进来。萧木见了，就指了指里面的套间说："团委在里面，现在已经下班了。"

　　"萧部长，我们是来找你的。"两个学生终于壮着胆子跨进门来，笑嘻嘻地望着萧木。

　　"找我？"萧木有些吃惊。

　　"怎么，你不记得我们了？我们是'哲学问题恳谈会'的……"

　　"啊－－对对！'恳谈会'。想起来了。"萧木恍然大悟："你们是问辩论会那事儿吧？"

　　"正是，不知道学生会研究的……？"

　　"恳谈会员"看到萧木依然是记挂着他们上学期提出的要求，心里就感到一阵释然。脸上的笑容越发丰富了，就象他们组织的名称一样，既诚恳，又傻里傻气。

　　萧木指了指高安和董力昆，说："得问他俩这件事。喂，两位头儿，上学期说的辩论会的事儿打算怎么办呀你们？可是来听结果了人家。"

　　高安和董力昆就双双抬起头来，冲着恳谈会员点了点头。又交换了一下眼色，董力昆就问萧木："木头，这事你有把握吗？"

　　"把握？……没有！"

　　"这可是个敏感话题，搞不好……"

　　"当然是敏感。象你们历史系举办的那些演义啦注释啦还招不来人呢！"

　　高安忧心忡忡，插嘴说："小萧，这可不是哗众取宠的事儿呀。"

　　"哎哎，高头儿，这回您可没说对。既然要取宠当然得哗众。再说，哗众不一定就能取宠这是辩证的眼光前辈们教的……"

　　高安皱着眉头，脸色就显得不悦。他走过去，小声请两位同学到门外稍等。恳谈会员把期待有目光投向萧木，想征得他的同意。萧木装作没有看见，把头扭向一边，他们就只好悻悻地退到门外。

高安把门关了，返身说："小萧，当前是敏感时期，我看这事儿还是大家商量一下再说。"

　　"高头儿，当初成立这些鸟社团时，我可是投反对票的，是你硬逼着我揽下这档子事儿。要开展活动了你该高兴呀，总得有起码的信任吧对群众？早知道这窝鸡不让下蛋当时就该养点儿肉鸭呀什么的……"

　　萧木说的，是在暑假前了。在农村正忙着分田到户的时候，大学生们却不知怎么想的，重新发现了组织起来的优越性，就掀起了一股子"社团热"。各种名目的有组织无组织的小组织一夜之间，就遍地开花，好象不入社不在团就只能进孤儿院，就不能算是中国人了。高安对此很热心，好几次开会，要求各部把组建学生社团当作眼前的头等大事来抓。仅是在宣传部的名下，就有了"文学爱好者协会"、'文学创作之家"、"莽原诗歌社"、"历史论坛"、"哲学问题恳谈会"、"社科沙龙"，还有集邮啦、灯谜啦、楹联啦等等的协会。林林总总地算下来，就不少于20多个。学生会的工作一时就呈现了前所未有的大声势，多次受到了学校的表扬。

　　萧木从一开始就千方百计逃避着为这些社团服务的机会。高安责成他招安这些散兵游勇，他就连连地摆手说，这根本不是宣传部的事儿该归学习部管是于勇的职责要不然就成立个社团部什么的专门料理他们或者主席您直接负责编为嫡系一肩担起来反正平常没事儿有事开个会大家凑把手一蹴而就挺好的你以为？但高安坚持说按性质划分该归宣传部管干工作不能挑肥拣瘦你别再推辞就这么定了。无奈，萧木就派了一个女委员，每天下午下课以后坐在办公室里，抱着登记薄守株待兔。这样，不费吹灰之力，本部的工作就有了大片的成果，仅此一项就可以在工作总结中占去满满的两大页。为此，他就夸奖了那女委员几句，让她觉着活得特别有意义。

　　其实呢，这些社团大都是三五个同仁好友发起，到学生会登记的时候就突发奇想，冠一个足以惑众的名称，为自己课余饭后的闲聊找到了一个官冕堂皇的借口。名存实亡者有之，苟延残喘者有之，象"哲学问题恳谈会"这样执著得近乎顽固的，唯此一家了。

　　从上个学期开始，"恳谈会"就多次打了报告，请求组织一次全校性的"存在主义与当代思潮辩论会"，并把辩论稿一并交给了萧木。当时萧木感到这班小同学真诚得可以，就把这事儿转告给高安和董力昆，请他们定夺。高安说再研究研究，从此就没有了下文。今天，"恳谈会"再次找上门来，萧木独自应付不过，便顺水推舟把矛盾踢给了两位头儿。

　　"小董，你看呢？"高安见与萧木谈不拢，就转而问董力昆。

　　董力昆看看他俩，略一思索，就说："我看搞是可以搞的。人家准备这么长时间也不容易，但也必须防止出现偏差。咱们把汪老师叫上，发现问题便于及时纠正。"又转向萧木："你跟那些同学说说，让他们严格按照稿件讲，把住方向，观点要明确。尽可能少留点儿自由辩论的时间，你看如何？"

　　"领导英明！只要能开成往我们部的功劳薄上添上一笔，怎么着都成啊！"

　　"那好吧！具体的时间等我向汪书记汇报以后再定吧。"高安最后这样说。

　　萧木出门，把好消息告诉了等在门外的同学，并与他们一起离去。路上他问，你们怎么找到学生会的今儿个才搬家？他们回答说，那还不知道？天一下雨就知你们要忙活一气了。萧木说，你们还挺聪明对学生会了解够深的。这样闲聊着，就走到了路口。分手的时候，萧木又嘱咐，说以后别叫我部长，我自己听着都别扭。万一让别人听见还以为上面来人微服私访呢容易造成混乱，给校领导添麻烦。他们说，那好那好，萧部长您真谦虚。
