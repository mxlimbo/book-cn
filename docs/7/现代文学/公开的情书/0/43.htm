第十四封信 老嘎致真真

　　真真，亲爱的朋友：

　　收到信了，我走出门去……

　　我一个人登上了山头，茫然地走着。

　　我说不出来是什么感觉……斑斑烂烂的一大片呈现在眼前。这里，已经不是陌生的地方，我是感到留恋还是感到冷酷？是悲痛欲绝，还是什么都有呢？

　　难道你还不理解我怎样以自己的痛苦和尊重，在用心地贴着那被撕碎了的画页，把它修饰成精美的画册么？

　　难道你不记得在一封信中我说过"我将含着辛酸的眼泪去描绘我们共同奋斗的欢乐和胜利的图景么？"我坚信，总有一天，我能拿出这样一幅创作，放到我的同志和战友面前。

　　在旅行中，每当我要离开一个地方时，就会产生没有着落、漂泊不定的情绪。我要走自己的路，但又不知现实将怎样摆弄我。

　　这些年来，我部充满矛盾和痛苦的内心世界，没有人理会过。天真的亚瑟是在生活的艰险的考验中变成了锋芒毕露的"牛虻"。而对于我，却是一颗高傲的心灵被压抑成麻木的"东西"，带上了沉重的精神枷锁。直到现在，我还没有摆脱旧的躯壳，以致于不能大胆地追求新的生活。

　　请你相信，我现在是在用理智写信。我不愿章伤害你。那天你劳动完回来，看到你是这样高兴、活泼，脸色红润，心情舒畅，我心花怒放了。多么想唱一支欢乐的歌，但我马上又产生了痛苦，竭力克制自己。我做过多少次这样的自我克制，这是多么残忍的扼杀--亲自的扼杀呵！

　　虽然，你也一直对我关闭着心灵的大门，我还是感受到了你的灵魂。就象小时候我听着姐姐在弹钢琴，我感受到她急促的呼吸、激昂的内心一样。我从心灵深处体验着。我觉得我需要这样的性格、这样的心灵、这样的热情。一旦我获得了，我将变得那么强有力，那么富有生命力！

　　岁月在飞逝着。我并不奢求什么。你是那样地理解我和体贴我。但是，你越是给我带来友爱，我越是看清了你和我的差距，我便越感到现实的冷酷。正因为我是深深地爱着你，所以我吞下了自己的痛苦，带着对朋友的友谊和热望，做出了大胆的举动。你问我为什么大胆地促成了这件事么？因为我爱你，我认为他可以给你带来你所需要的一切。我以自己的痛苦来帮助你投入事业。我相信，你一旦把充满愤怨的一切倾吐出来，就会坚强地奔向未来。我已经失去了一切，但我决不遗憾。

　　我知道，如果一个人没有能力帮助他所爱的人，最好不要随便谈什么爱不爱。当然，帮助不等于爱情，但爱情不能不包括帮助。也许，我又错了。不，不谈这些了。

　　我不想再在沉默中忍受理智带来的痛苦了，--从这里，你不是更可以认识自己的价值，从而更加勇敢地走向未来么？我第一次，也是最后一次倾吐了我对你的爱，我将会以更加勇敢的步伐走今后的路。

　　祝福你，你以你那大胆热情的心灵，获得了爱情和事业。世界上再也没有比这更动人的基于事业的爱情，也没有比这更崇高的由爱情结合起来的事业了！这才是我们的未来！

　　我从小就看懂了安徒生的《丑小鸭》，然而直到现在才看懂了《海的女儿》。

　　呵，雨后的凉风吹起来了，吹响了铃铛！是的，那心灵的铃铛一直在振响着，从来没有停息过。它是永远不会停息的！

　　再见了！
　　凄苦寒冷的雨夜
　　总会过去。
　　接着而来的必将是--
　　阳光明媚的白昼。
　　狂风暴雨能够
　　无情地摧毁--
　　腐朽的城堡；
　　也冲掉了
　　街道上的泥沙和污垢。
　　现在
　　你是多么幸福地
　　沐浴在温暖和光明之中，
　　你更懂得了
　　阳光的灿烂呵！
　　再见吧！
　　让我深情地再看你最后一眼，
　　明天，我叉将登上漫长的旅程，
　　重新开始心儿的歌唱……

　　老嘎

一九七二年三月初稿一九七九年九月二稿
（原载《十月》）
