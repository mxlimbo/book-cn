第十三封信 真真致老嘎

　　老嘎，我亲爱的朋友：

　　送你上汽车时，我曾经问过你，你当初怎么会产生把我和老久联系在一起的灵愿。你避而不答，汽车快开了，你突然对我说："我会告诉你一切的！"

　　是的，亲爱的朋友，在你和我相处的七、八天中，你一直很理智、很冷静。但我明白，在你小心翼翼的热情中包含着多么深沉的内容。（老嘎批：也有不冷静的时候，那天傍晚，我跑出去了，跑过了小河上的石桥，在黑夜中痛苦地哼着歌。）

　　其实，我这个人并不象你想象的那么好、那么完美。我只是一个心地善良、敏感、热情的人。我有无数的弱点和错误。（老嘎批：只有偶像才没有错误）

　　你不只一次地对我说："也许，将来我会很惨很惨。"我说："不会的，我们永远和你在一起。"我明明知道这种所答非所问的方式不能解决感情上的问题，但我又能说什么呢？我很清楚，友谊不能代替爱情。

　　上星期五下午，我带学生们劳动回来，给你采回两朵山花，还拾到一支老鹰的羽毛。这紫蓝色的野花，象倒挂着的小铃铛。我心中充满了诗意的幻想，好几次，我非常认真地用那支羽毛拨动那倒挂的花蕾，仔细聆听，看它是否会发出叮玲叮玲清越的声响。你说："硬要花朵和铃铛一样发出声响的心灵是多么美呵！"其实，它不会响，就象压抑在你心头的热情永远不会发出清脆的响声。（老嘎批：我说过，心灵的铃铛从未停止过振响。）

　　晚饭后，我们默默地坐着。我无意翻出一封你过去给我的信。没想到，你抢过去，一把把信撕碎了。一种强烈的情绪袭上我的心着："你在做什么呵？"我扭过头去，不愿再看你，你在撕信吗？不，你是在撕毁你心灵中美好的感情。为什么要当着我的面呢？我伤心了，我从来没有不尊重你。可你……我为了控制情绪，看起书来。当我感到你用惊异的目光看着我那在暮色中苍白起来的脸时，我走出了房间。（老嘎批：没想到这么令人心碎，无法弥补！）

　　我始终忘不了你穿着那件海蓝色的汗衫，灯光下，你的脸显得格外分明，面露深沉，只有一双眼睛充满了热情和痛苦。（老嘎批：我也忘不了，那天晚上，灯光下，你站在我身边，凉风吹起了白色的纱布窗帘，你深沉地说："闷呵！风来了。"）要知道，我希望你心头的热情伴着那痛苦的泪水一起倾泄出来，这样我可以安慰你。但我又能说什么呢？我知道，在你那颗高尚的心开始觉醒时，感情的浪涛开始汹涌时，你却不能打开感情的闸门了。（老嘎批：感谢你！）

　　老嘎，我永远尊敬你。我只是期望你更勇敢一些，在艺术创作上步子迈得更大一些。即使你一生的奋斗都得不到社会的承认，我们也永远爱着你。因为，那些作品里面凝结着你的、我们大家的感情和追求，这是我们一代人的经历啊！

　　樱花呀，樱花……

　　没想到过去的一切竟然这么难忘。

　　今年，在樱花盛开的春天，你给我带来光明和希望。明年呢？樱花重新开放成晴空里灿烂的云霞的时候，你还会来吗？你又将带给我什么？"年年都有风和雨，樱花总要盛开。"樱花呵，我们心中的春天，我们心中的希望！

　　真真
