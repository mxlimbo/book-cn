第三封信 老久致真真

　　真真：

　　可悲的是一个人没有走向新生活的勇气！
　　可悲的是一个人没有去爱自己所爱的人的勇气！

　　我不能原谅一个不敢走向新生活的人。

　　如果你的宗教教义不打破，如果你永远只追求个人内心的安宁，那么，你只能是个永远痛苦的幻想家、宗教家！

　　你爱我，又不敢爱，要和我保持一种不远不近的关系。在我们两人的关系上，我宁可死，也要在这两点之间抉择：不是爱，就是仇。我不能接受你出于一种调和的教义而提出的那种兄弟姐妹的关系。

　　"你太纯洁啦！"一阵轻飘飘的声音掠过耳际。一阵冷汗，我全身的毛孔都竖起来。我从一个甜蜜的梦里、温柔的吻里惊醒了。一盆冷水从头浇到脚。我呆呆地坐着。秒针嗒嗒地响着，我凝望着窗外黯淡的黄昏。

　　我的头脑被冰包围着。眼前是我熟悉的那个世界：女人、男人、金钱、利益、地位、权势、玩笑、调情、哭泣……那个被我思索过、研究过的世界。

　　"你太纯洁啦，我可怜的孩子。你不懂人生，不懂爱情……"一个牧师的声音。

　　一丝冷笑掠过我的唇边。我好象听到地狱里的狞笑和天堂中的喧闹……一切都消失了。无边无尽的、灰白色的、冷寂的雾漫上来了，我沉默了。

　　黄昏越来越浓，我看见一张又脏又乱的床。一个垂死的年轻数学家躺在那儿，没有爱人和亲人的哭声。孤独的黄昏中，他要死了。

　　远方赶来了他的朋友，点燃了唯一一盏灯。他走到数学家床边，喊着朋友的名字。他又听不到回答。

　　可怜的朋友呵，无名的年轻的数学家，难到你就永远听不见和你共同走过漫长道路的朋友的呼唤了吗？难道你再也领悟不到人生最后一点安慰了吗？一颗心就这样破碎了，沉没了……

　　朋友停止了呼吸。他在数学家耳边轻声而镇定地问："十二的平方是多少？"

　　"一百四十四。"这是临死的人发出的最后的声音，是对那热情和友谊所作出的最后的回答……他死了。

　　在这孤寂的黄昏，我眼前出现了数学家光辉的墓碑："144。"

　　一道眩目的光射进窗来。门开了，房间里先溢着玫瑰色的阳光。一个美丽庄严的姑娘走进来。她愉快地微笑着，抚摸着我的头发。呵，我心中光明的女神，你来了！

　　"你难受吗？"

　　我点点头。

　　"你爱她吗？"

　　"是的。"

　　"你愿意做她的兄弟吗？"

　　"我光明的女神呵，你应该提醒她一下，历史上从来没有一种强烈的爱情会以兄弟姐妹关系而告终。"

　　真真，我可以原谅魔鬼，但决不原谅庸人。

　　我的眼睛又碰到"牛虻"天蓝色的眼睛。

　　"牛虻先生，靠边站吧。现在已经是二十世纪七十年代了我们不必须那种病状的感情面制。我们这一代人将以自己豪迈的爱情，记录在人类感情发展的史诗上！"

　　老久
