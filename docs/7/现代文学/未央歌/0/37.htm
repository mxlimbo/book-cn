评论： 说鹿桥·话未央

作者：姚秀彦

　　鹿桥旋风

　　台湾冬天没有风，但在文艺界、大学及中学校园内，却吹起了一阵旋风，那就是返国的鹿桥带来的。

　　鹿桥本名吴讷孙，是西南联大外文系42级毕业，毕业后赴美深造，获耶鲁大学艺术系博士，其后在华盛顿大学任教美术史，并曾担任系主任，84年退休。有十八年之久未回来过台湾，此次是应历史博物馆及高雄师院的联合邀请，返台演讲，停留三周。三周之内，席不暇暖，先到高雄师院，然后到台北历史博物馆讲龙的传人，又有一女中等校的邀请讲演，并返新竹清华母校谒梅校长墓、又和读者网路对谈。所到之处，掀起青年学子的热潮，抢购他的未央歌，簇拥者等候签名，盛况少见，一直到他返美去机场之前，还有人在旅馆等候签名。　

　　关于未央歌

　　鹿桥中英文著述很多，学术论文外，有未央歌、人子、忏情书等，其中以未央歌最受青年学子欢迎。未央歌描述抗战时西南联大校内的生活情形，既真实、又浪漫，既入世、又脱俗，记述师生同学们细致情谊，可说是自孔门以来少有，不但汉儒宋儒及历代大学无此状况，就是后来的规划完整、设备齐全的现代大学也很难如此，因为现代大学多重视行政运作、纯知识技能的传授，缺乏那种步履相随、感情融洽、随时质疑问题的关切与呵护。

　　但是也有几位教授级的人提出疑问，在那抗战的漫天烽火中，怎么可能有这样金童玉女般清纯的生活。说这话的人，一方面不清楚抗战的性质，一方面也不了解中华民族性，对日抗战是长期性的，用时间的持久换取胜利，不是靠冲刺，一鼓作气，难免再衰三竭，而是要气定神闲的应敌，不可乱了方寸，做到战时如平时。再者中国人的特性实在与他国有些不同，战阵中有揖让、烽火中有从容。例如春秋时代的城濮之战，晋文公遵守他多年前的一句承诺"退避三舍"。鄢陵之战，晋楚激战一整天，当晚，晋竟派人去犒劳楚军。中国人好整以暇的个性，大概源于历史文化的滋养，兵法上说，祸兮福之所倚。因此越是艰难，越能浴火重生，所以虽在战时，而大学校园中有未央歌中所说的生活，是不足令人讶异的。　　

　　关于市尘居

　　时报出版社为鹿桥新书市廛居的出版举行说明会，十二月十日在金桥图书公司举行，门上大字写着"久违了、大师"，市廛居是个总集，包括历年出版的散文共36篇。首先由鹿桥本人对本书作一说明，接着是在场人士的热烈讨论和发问。我虽有一册在手，但来不及细看内容，首先映入眼中的是封面和书名，封面上是鹿桥夫妇二人对坐，有一种"相看两不厌、相爱永不倦"的感觉。书名是市廛居，市廛是一般民众的居所，既无关大庙堂大人物，也不是深山古寺不食人间烟火的高士，一切回归人生，这使我想起有同样情怀的古人，一是唐代李商隐的诗句"嫦娥应悔偷灵药，碧海青天夜夜心"，嫦娥不满后羿的专横，偷吃了不死之药而飞升到月宫，在那里不虑柴米油盐，也无恩怨情仇。无忧无虑，不病不死，守着寂寥的广寒宫，一月月、一年年、百年千年，过着同样单调的生活，作者认为嫦娥是会后悔的。同样的想法在苏轼的水调歌头中又出现，"明月几时有，把酒问青天，不知天上宫阙，今夕是何年？我欲乘风归去，只恐琼楼玉宇，高处不胜寒，起舞弄倩影，何似在人间？"仍然是回归到人生。因而千古英雄豪杰的志业，不在天上，而在人间市廛之间，能使家家柴米油盐不缺，父母子女夫妻相爱，盗窃贼而不作，就是伟人的成就。　

　　清华叙旧

　　清华人永远是清华人，学术与情谊交汇为一道长流，是切割不断的，因此鹿桥一定要到清华母校，去的目的有二，一是凭吊梅校长墓，一是史研所黄一农教授和小学弟周维强为他举办的网路对谈。

　　古人认为人生有三不朽：立德、立功、立言，立功是胜敌于战场，立言是文字流传于后世，二者都是彰彰在人耳目，可以立即举证的，居于三不朽之首的立德，却无法用数据、图片等具体形象来表达，而其影响之深广及久远，又非立功立言所能比拟，所以司马迁对孔子，也只能用"高山仰止，景行行止，虽不能至，心向往之"来尊崇他。梅校长正是立德的典范，也一身承担清华三阶段，由北京、昆明、到新竹复校，所谓桃李不言、下自成蹊，他建立了学术风气，化成学子的人格，作育英才无数，真是荡荡乎、莫能名焉，所以万里归来的鹿桥，当日必到梅校长墓前凭吊、低徊沉思、不忍离去。

　　其次就是由人文学院周碧娥院长陪同参观院中设施，并透过清蔚园与读者网路笔谈，所谈的问题包罗万象，从未央歌到一般大学生活，情况激烈、欲罢不能。　

　　后记

　　三周旋风过后，鹿桥又回到他在美国州的寓所延陵乙园，据说那儿一半是灵修地，一半是难民营，我未去过，姑且揣测一下，他是学美学和建筑的，很容易把住所设计一番，精美绝俗，美轮美奂，怎么会是难民营呢？应是像南朝庾信的小园赋中所说"落叶米床，狂花满座"，他宁舍了几何图形人造装饰之美，而让它拾回天然的野趣。再深入探究，可能是出自更超越的情怀，天地有情，赋予万物，最深最重的是家属父母子女，其次是朋友国人，再次推之于犬马鱼鸟等生命，这就是庄周化蝶观鱼的境界，最后连无生命的东西也有情，多年穿着的衣服、把玩使用的器皿，好像都是生命中的一部份，不忍丢弃，这是庄子万物与我齐一的心境，无法用实用的观点看，也不能用俭德来解释。

　　至于他的寓所为什么用延陵二字，该是与春秋时名公子吴季札有关，季札是吴王寿甍第四子，当时吴的王位仍是兄弟相传，三兄去世后，季札让国不受，及传到阖闾及其子夫差。季札封于延陵，后称为延陵季子，这一点是他有让国之贤。其次是他礼桨素养之深，他聘问鲁国，鲁君为他演奏周乐，他闻乐知政，从雅、颂到各国国风，都批评的为中肯，如评齐乐，"有泱泱大国之风"，评秦乐为夏声，能夏必大。第三点是他安顿情感、圆满无憾，他北上时路过徐国（今徐州），徐君热切招待他，很欣赏他的佩剑，但他要到上国聘问，名剑为不可少的佩饰，心想返国后再赠送给徐君，但返回经过徐国时，不幸徐君已死。他就到徐君墓前哀悼之后，将剑挂在墓前树上，以实现宿诺。唐诗曾赞颂他：延陵季子兮不忘故，脱千金之剑兮挂丘墓。他想赠剑徐君，只是心许，无口头承诺，无文字可稽，无证人证物，而且对方已死，大可不必还当初的一念，而他竟然不因齿明路隔，而丢弃他那心中隐密不为人知的初衷，这对因不得已而原谅自己的人也是提示。

　　中国文化以周文化为主轴，而季札一生将周文化发挥的淋漓尽致。鹿桥将自己居所以延陵命名，想来也有尊崇季札的隐意，何况他又姓吴。总之，从鹿桥居所的命名，到他一袭长袍的穿着，和强壮的双手，寸步不离的推动夫人的轮椅上飞机、到讲演场地，处处闪耀着中国文化的光辉，更不必求之于他的著述了。
