为了证明

　　为了证明。仅仅是为了证明。这是罗伯特·巴乔在法国世界杯足球赛中意大利首场迎战智利之战中惟一的主题。那便是战神巴乔。在战场上奋力奔跑着。有点像末路英雄。苍老而忧郁的神情。剪掉了小辫子的巴乔。一个真正成熟的意大利男人。当他终于在最后的时刻从老马尔蒂尼手中拿到了最后一张进军法兰西的入场券，他会怎么想？

　　在这一次世界杯赛前，我并不了解巴乔。只知道他是个曾经风光无限、战果辉煌的球场英雄。很多很多人热爱他，除了他足球的天赋和精湛的球艺，可能还因为他的英俊和潇洒。但，用不着花很多的时间读很多的资料去认识和了解巴乔这个人。仅仅一场球，就单单是凭着这激动人心的九十分钟，你就全然知道了巴乔是谁，他为什么是巴乔。

　　6月11日晚上。我坐在电视机前。期待着概念中充满浪漫色彩的意大利。即或是在那一刻，我对传说中神话一般的巴乔还几乎一无所知。但是我看了那场球。看到了在那样一场比赛中的巴乔。当满心的紧张和激动终于结束，我立刻跑到在另一个房间看世界杯赛的母亲面前，对她说，巴乔。在整场比赛中，我只看到了巴乔。

　　巴乔就是这样的人。他是能够在短暂的九十分钟将他所有的意志品质，人格魅力全都显现出来的那种男人。他全场跑动。尽心竭力。永不懈怠。那场艰苦的比赛打到了最后，仿佛场上就只剩下了巴乔一个人。所以电视镜头始终追逐他。那晚的电视解说员是孙正平。最后当巴乔终于为意大利赢得了那个点球的机会并终于射进了那个点球的时候，他仿佛惊呼起来，他说，又是巴乔！意大利队所进的每一个球都与巴乔相关，特别是在意大利一比二落败的情况下，老将巴乔顶住了全场的压力，从不放弃，最后凭借他那一粒点球，才最终把比分追上。

　　全场欢声雷动。

　　像浪潮一般滚动的巴乔的名字。

　　巴乔拯救了意大利。他竭尽全力是因为他知道如果那一场球不踢平，意大利在小组就很难出线。于是他奋力。为了国家的、团队的荣誉，也还有个人的，那就是为了证明，他是巴乔。

　　那一刻巴乔之所以成为英雄，是因为他的英雄伟业辉煌战绩是发生在那个最后的最危急的时刻。如果是在维埃里进球之后，或者智利队并没有二比一领先，也许巴乔所踢进的任何精彩的球都不会像终场前他踢进的这个点球这么具有价值，这么令人惊心动魄。当希望已经破灭的时刻，巴乔又将希望变为现实，而英雄便是这样成为了英雄。

　　巴乔终于完成了自己。比起四年前，他尽管已经苍老了许多，忧郁了许多，但却在岁月的严酷和境遇不顺中升华出许多新的品质。老骥伏枥，却志在千里，这才是巴乔真正的可贵之处。巴乔使我们看到了什么是成熟的英勇的男人，也使我们了解了竞技场上什么样的人和什么样的行为才是令人感动和敬佩的。

　　我们不管巴乔是不是已从他事业的巅峰滑落了下来，也不管他在未来的比赛中是不是还有上佳的表现，但巴乔已经证明了他自己，证明了他的精神，他的意志，他的勇气，和他为他的祖国所英勇承担的那一份神圣的责任和使命。
