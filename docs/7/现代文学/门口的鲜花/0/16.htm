又是秋季

　　秋总是骤然之间就来到了我们身边。包笼着你，向着深处。温暖而又满目凄凉。天高起来，然后落叶被疾风追逐着，惊慌地卷起棕黄色的旋涡。冰冷的秋雨打在枯干的树权上，发出寂寞而忧伤的响声。皮肤变得干燥，你要赶紧去买各种润肤露，还要为女儿换上秋季的衣服，然后走到窗前，便赫然看见了街对面围墙上的藤蔓中，那几片最后的红叶，像火一样燃烧着，在深秋金红的夕阳中。那是一种感人的辉煌，而且悲壮。那一刻街上静极了。

　　小时候，最喜欢秋天。出于天真，因为秋天收获。金黄色的茅草，浪一样地翻卷着，为此，家中的墙壁上很多年一直挂着我和弟弟在秋草中的照片。灿烂的秋的阳光，所有的温暖的记忆。那是幸福的儿童时代。我们不知道未来，更不懂得苦痛忧虑，就像女儿的现在。

　　可我们长大了，在轮回而永恒的四季中，人生有了很多阅历，心情也变得总是复杂而斑驳，一如秋天给我们的印象。那天，女儿指给我看，河边有一棵大树上的叶子全黄了。几乎是转瞬之间。那时候，我每天清晨要送她沿着小河去上学。那是一种很透彻的黄，黄得温暖而又浪漫。而在那棵大树边，则是一层又一层的青绿和棕红，将天地间的色彩染得丰富异常。一种感动油然而生。我答应女儿明早一定把这秋的斑斓色彩拍下来。结果，就像所有落空的愿望一样，这个夜晚骤然刮起了一场很猛劲的风。第二天清晨，当我们急急忙忙带着照相机赶到那棵大树下时，却看见那满树的黄叶，竟一片不剩地全部萎落在地上了。怎么会？仅仅一夜之隔，那黄转瞬即逝，我们方知秋的瞬息万变，方知叶的生命是何等地脆弱，方知秋的色彩是留不住的。于是感伤。踩在大树下那松松软软黄色的枯叶上，就禁不住有了一种想哭的悲哀。

　　这便是一种心境。怀着这种心境，我总喜欢在秋天开始的时候，也开始一部新小说的写作。在温暖与凄凉编织起来的复杂感觉中，写开头的那几行字。我常常是一边写，一边看着窗外的树叶怎样一层一层地将绿色褪尽，又是怎样一片一片地枯萎、卷曲，最后，在一个不期的瞬间，被一阵微不足道的轻风卷走。这时候，满目的凋零就变成了一首忧伤的长歌。然而就在那忧伤之中，却仍然坚持着你从不敢忘却的那一缕勇士般的辉煌，还有生命的坚忍和力量。

　　我喜欢描述秋天。喜欢讲秋所带给人类的丰富的印象和感觉。我人生中的很多大事都发生在深秋，它们使我快乐过也伤痛过，使我觉得经历了秋天，也就是经历了人生的一种永远。我变得成熟，变得对生活不再苛求。总之，拥有秋天的心境是一件非常美好的事情。特别是，这种秋天的心境能帮助你成为一个作家。脚下踏着悲凉的枯萎，而眼前跳动的却是火一样烫着你的金红，那么，你的作品中还缺少什么呢？一切人类的心情在这一刻已经全都拥有了，于是，你终于成为了在那个季节里最富有的那个人。

　　所以，我总是盼望着一个又一个秋季。
