第46章：白金咭计划


　　李若龙刚走，方阳晖就开始翻阅《白金咭计划》。他看着看着，脸色由青变紫"气得双手发抖，抓起电话挂给李若龙。

　　"喂，李若龙吗？你立即上我这儿来一下。"不一会李若龙就到了。方阳晖"啪"的一声，把计划书往桌面上一掼，吼道："这是个疯子计划——"李若龙想辩解。方阳晖不容李若龙分说，又吼道："你不用说，我不想听你说，你这是想出卖我——""方先生……""走吧——"方阳晖大手一挥，背过脸去。李若龙仍呆立不动，方阳晖转过身来咆哮："滚——你给我滚——"李若龙走后，方阳晖仍怒气难消，又挂电话给詹森。

　　"詹兄，你有空吗？"

　　"有急事吗？"方阳晖就在电话上骂开了："你来看看，李若龙搞的出卖我的疯子计划。""好，我就来。"半个小时后，詹森来到方阳晖的办公室。方阳晖把《白金咭计划》往詹森面前一推，说："你看看吧——简直是混帐——"詹森粗略地浏览了一下，蔼然地说："方兄，不必动怒，何不听听李若龙的意见？""我不想听他胡说八道——""既然你们已经坐上一条船，他主管业务，提出拓展业务的设想乃应有之举，听听何妨？"方阳晖气鼓鼓的，不再出声。詹森续说："再说，你听他解释时可以直斥其非，令其心服口服。"听了詹森一番劝导，方阳晖说："好吧，你给他挂个电话。"詹森与李若龙通完电话，说："他请我们上六十八楼会议室。"

　　方阳晖和詹森走进会议室时，李若龙已经在那里恭候。李若龙拉开壁柜，内装一块白色书写板，他拿着彩色水笔开始讲解《白金咭计划》。他说："这个白金咭计划我已经构思了很久，但由于条件不成熟，一直没有实施。现在方先生加入了，才打算推出。"方阳晖架着二郎腿，一副漫不经心的样子。詹森则凝神望着李若龙，认真倾听。

　　李若龙继续说道："简而言之，白金咭是一种债券形式会员咭，每张咭销售价六十五万、以三年为期限，到期原价赎回，而前两年则以固定年息率十三厘半付予购买咭。""混帐的逻辑，"方阳晖打断李若龙，驳斥道："这不过是一种虚假销售，到时还债，我占百分之四十五股份，主要债务背在我身上，这不是出卖我吗？""方兄，你让他把话说完嘛。"詹森说。

　　李若龙对方阳晖的泼妇骂街不予理睬，继续说道："为推出白金咭还有一个配套工程——就是公司架构的调整。我们必须在英属处女岛百慕达设立一间名为'四海高尔夫游艇会控股公司'的分支机构。将来我们的收益主要体现在控股公司。"这时方阳晖已下意识地挪动坐姿，把脸转向李若龙。

　　"说，继续往下说。"詹森听得津津有味，鼓励道。李若龙面露得意之色，说道："这样，换句话说，香港公司就好比输水管，百慕达控股公司好比是蓄水池。"李若龙在书写板上画了一条输水管泄水注入蓄水池的草图。"对，对。"詹森颔首称是。

　　"不过，还有一点我们要做的是……"李若龙睥睨一眼方阳晖，见他听得入神，脖子伸长了，不禁暗自发笑，说："注册资本上也要调整，香港公司的注册资本降为一万元，而百慕达控股公司注册资本为一亿。"

　　"好——"詹森说："蓄水池满时，随时可以把输水管关闭——哈哈……"詹森转头问方阳晖："方兄，你以为这个白金咭计划如何——""喔……"方阳晖一面听一面在暗中筹谋，被詹森一问才支吾其词地答道：

　　"可以考虑。不过……""不过什么？"詹森追问。

　　方阳晖说："一要调整股份比例，二我要辞去董事长之职。""为什么？"詹森逼问。

　　方阳晖并不正面回答，把暗地里打好的腹稿摊出来："先说第一点，将香港公司原来的股份比例调整为：袁先生占百分之七十五，李先生百分之十五不变，王薇小姐保留百分之五，而本人降百分之五。""那么股本金呢？"詹森又问。"这点请放心，我仍出百分之四十五的股本金，只是股份比例减少而已。同时顺理成章地我应将董事长的交椅让位予袁先生。"

　　"哈哈……方兄，你这是仿效周文王礼贤让位呀。"詹森一下子就明白方阳晖的用意，李若龙也洞悉方阳晖的诡计，但不露声色，只微微一笑。詹森又说："方兄如此慷慨大方，我也应谦让，在此仅代表内子王薇将其名下的百分之五股份完璧归赵，奉还袁先生。"

　　"我还要补充一条。"方阳晖说："新注册的百慕达控股公司的股份比例，我建议：李先生占百分之十五，本人占百分之八十……还有百分之五留给詹森先生。"詹森连连摆手说："不行不行，我不可以在外头公司拥有股份。""詹兄不必谦让，这是你应得的，如有不便那就仍然记在王薇名下吧。"方阳晖说。

　　詹森说道："却之不恭，受之有愧，既然二位一番诚意，我也不推让了。""那就这么定吧——"方阳晖习惯性地挥了一下手。

　　詹森拊掌说道："这是一个天才的计划，不过还得加上老千两个字，叫做天才的老千计划。但是要当我没听到，当我不知道这件事。你们怎么搞，与我无关。我只有一个条件，给四海高尔夫游艇会的每一笔贷款，都必须由华盈集团做中介担保人。"

　　方阳晖略略想了一下，说道："没问题。"接着又说："现在当务之急是通知袁先生到香港开董事会。"
