第52章：方阳晖一反常态


　　这天夜里，壁上的猫头鹰挂钟刚敲过十响，雨荷又下楼，坐在客厅里，荧荧的烛光映照着她的影子，像剪下的一块黑布长长地拖在地毯上。

　　就在这时，一条黑影越过雨荷身后，像猫一样窜上楼，他推开了雨荷卧房虚掩的门，摸索着，找到了床头柜上的电话机，迅速地旋开了电话机的底座，将一只微型窃听器安装在里面，然后，偷偷地溜了出来。

　　这天，方阳晖打电话给李若龙，让他到办公室来一下。

　　半个小时后，李若龙刚踏进他的办公室，方阳晖就笑容可掬地迎了上来。"李先生，请上这儿坐。"他邀李若龙在沙发上坐下。问道："喝点什么？""来杯咖啡吧。"李若龙感到纳闷，这方阳晖从来对他都是黑口黑面的，今天怎么一反常态，热情有加呢？

　　喝着咖啡，方阳晖慢条斯理地说："李先生，你最近的工作卓有成效啊！""都是董事长领导有方。""哎，我已经不是董事长，可不敢居功。""但你仍然是我们公司的掌门人。""哈哈……掌门人也好，跑龙套也好，我们现在同搭一条船，我理应支持你的工作。""谢谢！

　　""哎，我应当谢你才是，你为公司挣钱，也就是为我的荷包着想，劳苦功高哇。""哪里哪里。"李若龙越发纳闷，今天姓方的转了性？

　　一个劲地给他戴高帽，葫芦里究竟装的什么药？

　　"我们过去因为误会，存在着一些芥蒂，现在了解了，疙瘩解开了，过去的事一风吹，别再搁在肚子里。""我早就把它抛到九霄云外去了。"

　　"那好，那好！"方阳晖问："白金咭计划推行到现在快一个月了吧？""喔，差两天就整一个月了。""总共销出了多少张？""到今天上午为止已经销出四十六张，估计头一个月可以销出五十张。总销售额三千二百五十万。""太好了！"

　　"如果以平均每个月销出三十张计，三年内销售总额将达七亿多，扣除债券利息和各项开支后，单是白金咭的纯利就约有六亿。""嘿，这真是空手套白狼，一本万利的大买卖。"方阳晖喜不自胜，豪气干云地说："李先生有什么困难尽管提出来，我来替你解决！"

　　"暂时没有什么困难，一切按部就班，运作正常，只是……""说，说出来，只是什么？""只是日常的开支，流动资金支取手续繁杂，费时误事，有时就影响了正常运作。"

　　"噢，这还不容易，我给你开张授权书，今后五十万元之内你个人签字就可以向银行直接支取。""谢谢方先生。"

　　方阳晖说到做到，立即让秘书打了一份英文授权书，签名盖章后递给李若龙。"谢谢，如果没有其他事，我就先走了。"李若龙接过授权书说。

　　"等一等，"方阳晖蓦然想起了什么，招呼李若龙坐下，他挂了一个电话出去。"喂，你进来一下。"刚放下电话，接待生老女人就进来了。

　　方阳晖说："日本那边催得很急，明天的机票要改到今天。"他看了一下手表："现在是十一点多，你马上跟航空公司联系，我今天下午就得走。"老女人应诺一声，脚底生风般走了。

　　老女人走后，方阳晖叹了口气："唉，我是牛马命，公司的事一大摊，经常满世界跑，飞来飞去。所以，李先生，会所那边的事我顾不过来，全靠你了。""谢谢方先生的信任，我会尽力而为！"

　　"好，今天就谈到这里，不耽搁你太多时间。"方阳晖亲自把李若龙送到电梯口，挥着手，直到电梯门关上。

　　奇怪！李若龙虽然没有受宠若惊的感觉，但确被方阳晖反常的态度搅得如坠五里雾中。他不相信像方阳晖这样城府深不可测、心高气傲的人会真情流露、礼贤下士。但无论如何，李若龙想，他今天是颇有收获的，一则他轻而易举地得到了一张财务授权书，每次可以调动五十万元以内的资金；二则他获悉方阳晖今天将离开香港远赴日本，他可以放心地约会雨荷了。

　　回到办公室，他让会计立即把方阳晖开具的授权书呈交开户行，并办理印鉴手续。接着，他就打电话给雨荷。但是电话没人接听，响了三、四声后就自动转驳到留言机上，他再打还是转驳到留言机上，如是反复打了七、八遍，都只听到同一句话："我是郭雨荷，现在不能接听你的电话，请留言，谢谢！"

　　李若龙知道雨荷就在电话机旁，只是她拒绝接听，迫于无奈，他只好对留言机说话："雨荷，我是若龙，我知道你不愿接我的电话，雨荷，请你听我解释……雨荷，我知道你在生我的气，怪我在停车场和游艇上两次弃你而去……雨荷，你听我说，请你拿起电话，我要知道你亲耳在听……请你原谅我吧，我那样做是有难言之隐，是有苦衷的。但是雨荷，我是真心地爱着你，我对你的爱天地可鉴！好，雨荷，那就让我当面向你解释，今晚我上你家来，方阳晖今天下午飞到日本去了，今晚我一定要见你……"自始自终雨荷都没有接听，李若龙只是在个人独白。

　　但他的这番说话都清晰地传进了方阳晖的耳朵。
