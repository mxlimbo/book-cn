冰签


　　一位渴望把子女送到明星中学的朋友向我诉苦，她的孩子因为学校作弊，而未能抽签抽中。

　　"在众目睽睽之下抽签，怎样会作弊呢？"

　　朋友说，学校把权贵者和有钱者的子女的签先冰在冷冻库里，抽签时和普通的签放在一起，但是学校的行政人员即使闭着眼睛一摸，也立刻可以抽中那些冰签，到后来，明星中学就成为权贵子弟和富豪子弟的天下了。

　　这真是匪夷所思，使我想起侦探小说中常写的冰钻杀人的情景——以冰结成的利钻或子弹，射人人的身体致人于死，冰溶化了，永远找不到证据——冰签也是一样，摸上来在阳光下晒三分钟，作弊的证据就消失了。

　　我听了非常赞叹发明冰签的教育工作者，他们有超乎常人的创造力，如果他们愿意用发明冰签的聪明与细腻来献身教育，每一个中学都会成为明星中学。

　　因此，我安慰朋友说："别痛心和埋怨，我们都不是用冰签抽中的人，不也活得很好吗？"
