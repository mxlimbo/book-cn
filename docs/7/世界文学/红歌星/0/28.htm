第二十七章


　　仍然是我玛阿，对，昨天我又跟汉克见面了，就我们两人。我先给他寄去一封信，说我要跟他谈些严肃、重要而秘密的事。显然我这是虚张声势。我约他在一家大饭店里见面。他得在一个房间里等我，在88号房间。梅尔和马克呆在大厅里，他们永远不会见到汉克。他们想在我之前先看一下房间，我没同意。没有任何危险，无人知道我要占这个房间。他们犹豫了一下，顺从了我。我感到他们有些妒忌。我之所以敢于同汉克约会，是因为璐向我透露了一个信息：吕丝跟汉克吵架了，吕丝哭了。我觉得吕丝的眼泪流得妙极了。吕丝，哭吧，哭吧！我的乖乖。我来喝你的泪水，你那苦涩的露水。璐告诉我，汉克是个三心二意的家伙，他是个情种，但他好奇、独立。吕丝也十分独立自主。这样，潜沉在热恋的狂潮下的梗阻终会暴露，针尖对麦芒在所难免。"他们不融洽。"璐对我说道。可她又是怎么知道的呢？她到处掏人家的知心话。比如，汉克向柯妮推心置腹，而璐则是柯妮的朋友。如此说来，汉克与吕丝的关系恐怕不行了，我很喜欢这种猜想，我好像松了一口气，最终大家都自由了。太棒了！我不再向马尔科姆咨询。我觉得自己没有失去吕丝，让那精神分析见鬼去吧！马尔科姆在把我的钱装入自己的口袋时，目瞪口呆地注视着我。突然，我觉得他那么矮小，几乎有点可怜他。灿烂的阳光照耀着街道。梅尔和马克春风满面，充满了青春活力。马克跟着我上车前在上衣里边旋转着手枪，像个苦行僧。人们跳舞，鸟儿鸣唱……我真想干点荒唐事。我给汉克寄去了一封信，上面签着玛阿的名。

　　汉克在房里等着，坐在一张扶手椅上，一副傻相，一动不动，如钉在深玫瑰红的天鹅绒椅面上。他在顺从地等候我。保镖们呆在大厅的天花板下，在穿着朱红外套的年轻侍者和衣着华丽的富有顾客之间。一切秩序井然。这是我玛阿推动了行星，是我使这一小群男子汉有了活力。那两个偷摄者，那个精神分析专家和那个亚洲舞蹈演员，他们是我麾下的四重奏小乐团。吕丝，你就看着吧！

　　我从装在一油漆家具里的小冰箱中拿出香槟，给我和汉克斟上。我决不喝烧酒，但对香槟不在乎。汉克和我喝了一瓶又开一瓶。我始终不说穿什么是"秘密而重要的事情"。我长得很美，这就是秘密。可能由于我很高兴，显得容光焕发。我完全支配汉克，这是个悲剧。这家伙不过是条泥鳅。我怎么会那么怕他，怕他对吕丝的爱？他是个漂亮的青年舞蹈演员，但没有天分。由于我们稍微有点醉了，我对他建议做个游戏，他放松了……他发誓听我的，不是吗？我是明星，他没有别的办法。他从来就是为我而跳舞。好吧，现在跳吧！那些十足笨拙的秘密新发现，他自己会找到的，这就是我所建议的游戏。我命他平躺在床上，四肢尽量伸展，放松。我则坐在一张椅子上，稍稍在他的后面，跟长枕一般高。我们映照在贴在房间侧墙的大镜子里。于是我对他说：

　　"汉克，你听任自己……你放任自流，听凭念头自发冒出……"

　　他回答我：

　　"这有点精神分析学的滋味。"

　　我等待，然后问他：

　　"你立即想到的是什么？直截了当地说。"

　　"不是想法……"

　　"那好吧，告诉我你的印象，即使是很模糊的印象。"

　　"我有点被出卖、受威胁的感觉。"

　　"受什么威胁？"

　　"受您的威胁……您的目光，我不再看这种目光了。"

　　"是不是我有可能暴露而汉克先生要掩盖？"

　　他笑了，首先是神经质的笑，然后是乐得哈哈大笑。他开始开怀大笑。这才是年轻小伙子的笑。

　　"玛阿，您这是为了什么？好了，别躲躲闪闪、云遮雾罩的了。快说吧！"

　　"我不能说……"

　　"应该说。"

　　"这是瞬间场面。"

　　"恰恰正是……是在德方斯广场演出的最后一个晚上。您有一段时间回到您的化妆室，我经过门口，璐不敲门就冲了进去，门开了，我瞥见您，有几秒钟……您背对着门，站在一面镜子前。您上身还保留着灰色乳胶，可下面却裸露着。"

　　"你喜欢吗？"

　　"这太过分了。这就是您那过火的美。"

　　我没料到事情是这个样子。我心花怒放，一下子就把他--吕丝的舞蹈演员--逮住了。

　　"你是不是想再见那时你所见到的？"

　　他噤了口，发出他那年轻人的咯咯笑声。他喝了酒，眼睛闪闪发亮。我也是。我进一步说道：

　　"我把你的缄默看作承认。可你是不是认为我能这样在你面前展示……我过火的美？"

　　"可能……"

　　"总不能直接地观看吧！你不要朝我这一边看，看床边的镜子里。而且得等我叫你睁开眼时你再看镜子里。你先闭上眼……对啦，很好……汉克，你在想什么？"

　　"我什么也不想。我心跳得很快。现在是夜里，我将看见您。我不十分清楚将看见什么，把窗帘拉开……"

　　"告诉我，你想看什么？"

　　"我想我是疯了，又回忆起那瞬间场面，马上看到那场面，连续看到好几次。"

　　我脱下衣服，保留我的缎子胸罩，袒露我的腰部和大腿。他始终闭着眼睛，等待着。他那张纯洁而漂亮的脸表现出崇敬的神情。

　　"你是个睡美人。"

　　他不睁眼地微笑，五官依旧十分优美。吕丝的情人躺在那儿，身材颀长，像朵百合。我等了一会儿，站起身来，从手提包里拿出一面小镜子，又回到与枕头并排的地方。我始终站着，我转身朝墙，仿佛要审视自己的脸，往嘴唇上涂口红。我命他翻身转到他侧面的镜子那边。他终于可以睁开眼睛了。我不再看他。

　　"好了，你看……"

　　他喃喃地答应着，声音很怪。我让他仔细端详，我说：

　　"你对我说一句你的看法。"

　　"是件无价之宝。"

　　他用强调的、赞赏的语气说了这句话。于是我挪开手中的小镜子，看见他正在端详大镜子里我的胴体。我也观看这反映出来的使他入迷的形象。我发现自己愈显得美丽了。我仍然移动我的小镜子，他的脸和我的脸连贯地由两面镜子映照出来。我们的目光相遇，我们有点颤栗。

　　"再闭上眼。"

　　他闭上了眼。我轻轻地来到床上。

　　"千万别睁开眼。"

　　我躺在吕丝的舞蹈演员那修长而肌肉发达的身上，我吻这个睡美人的嘴。

　　"别睁开眼……但摸摸这华贵的宝贝。"

　　我引导他的手，他抚摸、揉捏我的身子。我解开他衬衣的扣子，吻他那纯洁的胸部。我占有了吕丝的情人。我紧紧搂着他，使他有点透不过气来。我用嘴唇拧他的脖子。他温柔地微笑着解开我的胸罩。他吻我的乳房。我叫喊，我听见自己发自体内的，而不是从嘴里发出来的叫喊声，一连串欢乐的叫喊声在我身上打开了缺口，我沉溺在这连续不断的叫喊声中。

　　就在第二天，我约见了吕丝，我温和地告诉她：

　　"我跟汉克睡了觉。"

　　她有些僵直地往后退，跌坐在一张扶手椅上。她闭上了眼。我喃喃道：

　　"我求你别睁开眼，仍然闭着……再闭着……睡吧，吕丝，睡吧……我跟汉克睡觉，因为我爱你。"

　　我走到扶手椅那儿，我跪下向她探过身去。吕丝似乎由于反射作用而睁开了眼，我恳求她闭上眼。她像个小女孩似的，很快又闭上了眼。我拥抱她，我用舌头顶开了她紧闭的嘴唇。她的脸变得很凶。她扑在我的怀里。

　　两天里，我接连尝到了永恒的幸福，我享受了生活的乐趣，这是来自我心底的渴望。这一乐趣带着光轮围在我四周，我闪耀着快乐的光芒。对，是颗明星，我终于成了明星，辉煌的明星。

　　第三天，吕丝寄给我一封信。信中说我们不能再这样了，这不可能。应该留下这美好的回忆。她一点也不遗憾。但她不能永远成为我的情人，这并非因为汉克的缘故。对他，她的爱情已降温。她感谢我使她发现和感受我们生活中的乐趣，但她不能再继续下去了，长此以往是不行的。她知道这样会愈来愈糟。她请求我相信她。况且，她是我的新闻专员，在工作中应当严格……区别趣味。她感到惊讶，万分惊讶。可这正是惊讶的效果，表达幸福的撞击。她对我有着强烈和复杂的感情，但这并非通常所说的爱情。她无疑处于去爱、但更长久地爱男孩子的状况，这是习惯，而且不可改变。至于我，她是以另一种方式爱我……

　　我被这封信搞得心绪不宁，我还未想到将来如何，我不敢想象。我和吕丝永远在一起，我仍然处在十分满足的向往中。吕丝拒绝我，要我相信她。我自己并不想马上再开始。我清楚我们之间能够相爱的特殊原因。现在，我们的作为使我惶恐不安。我原本害怕做得不好，害怕把下一次搞糟。可这一拒绝，把一切快乐都拒之门外了。我感到非常痛苦。她本该再等等，给我留下一线希望。我寻思可能这种难得的偶然情况还会重新出现。不是现在，也不是明天，而是以后。她生活中出现危机时，就会重新解除武装的。到那时，我就等在那儿，在她心绪纷乱、一团糟时，我伺机行事。那时就会有新的转机。既然有过第一次，肯定有朝一日还会有第二次、第三次，在她失去一切目标的绝望之夜。我等着那一天。不应把这告诉她，应该等待，应该相信，我仍然充满信心。我得到了那么多，可从未希望过这样的慷慨。我心满意足，我拥有一切，我占有了她。这是我的福分，我绝对的乐趣。现在我知道了她的一切，我永远拥有她的一切。我将等待那黄道吉日，等待雨过天晴，等待着她心里的暴风骤雨……这一天将来临，我坚信不疑。有一团永不熄灭的秘密之火，我将生活在这团火焰的强光中，虔诚地等待这一回归。
