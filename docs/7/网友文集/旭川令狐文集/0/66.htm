二十一世纪的心愿

　　大二那年，文友委托我到学校附近的一个职工家里给一名小学生当作文家教，接完朋友电话的第二天，我就来到了朋友跟我描绘的居民区找到了那个家庭。

　　我的学生是一个七八岁叫筱筱的小女孩，大大的眼睛，扎着两个小辫子，可爱极了。小丫头特懂礼貌，一见我便"大哥哥、大哥哥"地叫个没完。叫得我心里甜滋滋的。

　　每个周三、周四、周六的晚上，我都会去筱筱家辅导她。筱筱很勤奋，每次去她都会拿出好几篇她写的作文来让我批改。当我给她讲解她行文中的不足之处时，她便会仰着头看着我认真地听着，不时还会低下头去记些东西。

　　我还会带着筱筱去看一些有益的展览会，在那次烈士公园的希望工程展览中，筱筱久久地站在展览牌前，我看到了她眼中的泪水顺着脸颊流了下来，在人山人海中，筱筱竟呜呜大哭起来。

　　不知不觉中两年过去了，在这两年中，筱筱的作文水平提高得很快，由两年前的作文差生变成了几乎每次作文都被当作范文的优等生。有几篇作文还被几家小学生作文选刊采用。这使我感到一点欣慰：毕竟我的努力没有白费。

　　两千年十二月十日，我依照惯例又去了筱筱家。一路上所见都是一派新年前的喜洋洋的景象。我心里也莫名地高兴起来——毕竟新年是大家共同的节日，而且还是千禧年。敲开筱筱的家门，开门的是筱筱，她一见是我，便高兴地拉着我到她的小房间，我还沉浸在新年是喜悦之中，便问筱筱："筱筱，告诉大哥哥，你新世纪的最大心愿是什么？"筱筱沉默了一会，拉着我，让我坐在她的小椅子上，然后从书桌的小抽屉里拿出了一本剪辑本，贴满了一些报纸上关于贫困山区失学儿童的报道，本子上面还夹着一些钱。我一脸惊诧地看着她。

　　筱筱的眼睛亮晶晶的："直到大哥哥你带我去看那次展览后，我才知道世界上还有那么多同我一样大的小朋友没有钱念书，从那时开始我就把我的零花钱和得来的稿费都藏了起来。新年里我最大的愿望就是象大哥哥你一样能常常发表一些文章，我要把得到的稿费都送给那些没有钱念书的小朋友。"

　　看着那些剪报上一双双渴求的眼睛，再抬起头时，我的眼睛已是一片模糊……

　　由于工作的调动，筱筱已随她父母离开了长沙，但我仍然时时想起她。想起她小小的辫子，想起她大大的眼睛，想起她的勤奋，想起她的天真。

　　这一段时间来，我无时无刻不在想，一个小小的孩子心里所想的真的让我自叹不如。新年来了，新千年就要来了，人们在狂欢狂舞，狂欢的浪潮一直没有停过，好像新的千年来了便一切烦恼都没了一样。可是饥饿的人依旧没饭吃；受冻的人依旧没衣穿；濒临倒闭的工厂依旧无法开工；贫困人家的孩子依旧没书读。我们还有太多太多的事情要做：祖国的经济要我们每一个社会成员共同努力发展；祖国的和平统一要我们每一个社会成员共同维护；社会上贪污腐败的不正之风要我们每一个有良知的中国人共同反对与斗争；贫困山区孩子的失学问题还要我们每一个有同情心和爱心的人共同商讨解决。一个孩子有同情心，愿意为失学同龄人捐上自己的一片爱心，这固然是好事。但从另一个角度我们可以看到我们的社会并没有达到很理想的阶段。可是我们中的许多人却以为我们的国家已经很富强了，社会已经很安定了，人民已经很富裕了，于是他们一整天无所事事，端着茶杯过日子，甚至贪污犯罪，腐化堕落。不知他们在想到那些贫困山区没钱上学的孩子一双双求知若渴的眼睛时，心里会想一些什么？不知他们在筱筱这样天真可爱的孩子面前是否会觉得面红耳赤？

　　一直很喜欢冰心笔下几十年后共产主义时代那种国家富强，社会安定，人民富裕的生活构想。不知到何时我们的国家会像冰心描述的那样美好？我想这个理想并不会很遥远的，真的，我相信！
