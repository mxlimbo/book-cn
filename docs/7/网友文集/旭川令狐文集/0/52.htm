我眼中的蔡骏

　　一直很喜欢蔡骏，喜欢看蔡骏的文章。从他在榕树下发表的第一篇文章《天宝大球场的陷落》开始，就发觉咱们的蔡同志是一个很不错的网络写手。

　　而真正让我近乎痴迷地喜欢上他的文章是看到他的《爱人的头颅》和《殉》两篇文章。这两篇文章的发表中间相隔五天。但是我看它们却是在同一天。我记得很清楚：六月四日。

　　从那天开始，我便开始感觉到自己已经渐渐喜欢上了他的文字，也就是从那开始，"蔡骏"这个名字就刀刻般地深深嵌在了我的脑海里――虽然我一向不喜欢记作者的名字，但是"蔡骏"这个名字却因为他的文章而变得分外充满神秘感。

　　蔡骏的文章当中有很大一部分都是描写古代的故事。像很受读者欢迎的《食草狼》、《爱人的头颅》、《殉》、《十个月亮》、《一封家书》，写的都是古代的故事。他总能以很深厚的文字功底把那些古代的在普通读者眼里很难想象的场面描写地绘声绘色，给人一种身临其境的感觉。

　　蔡骏很擅长细节描写，不论是心理描写还是景物（场景）描写都是一流的，相信读者在看了他的《爱人的头颅》和《恋猫记》这两篇文章之后一定会有这样的感想。不管你愿不愿意，只要你看了他的文章你就会不得不喜欢上他――一个年轻、才华横溢的写手。因为他给我们带来的是一种不可抗拒的美的追求与享受。

　　蔡骏的文章中有两篇是我感触最深的，那就是《恋猫记》和《一封家书》。记得看《恋猫记》的当天，我一直沉浸于猫与人之恋当中，以至于当天晚上我竟梦见了猫――一只除了尾巴尖上有几点火一样跳动的红色之外浑身雪白，行走的姿势相当优雅，每条腿落地时都很轻柔和小心的猫。而《一封家书》则是我看过的包括《第一次亲密接触》和《我的爱慢慢飘过你的网》在内将近五千篇网络文学中第一篇让我掉泪的文章。真的一直想不透一个未满22岁的青年人居然能写出那么美妙的文字，或许这就是天赋吧！我想。

　　希望你也喜欢蔡同志的文章，更希望咱们的蔡骏能写出更多更好更好的文章，我相信这是所有爱好"榕树"，喜欢网络文学的朋友们共同的心愿吧！
