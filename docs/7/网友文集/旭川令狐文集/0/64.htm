蝶

　　从小她就想变成一只蝶，一只能在五颜六色花丛中翻飞的蝶。在很小很小的时候，她曾把自己的梦想告诉朋友，可是她们都笑话她，说她是个傻女孩，于是，从那时开始，她再也不敢向别人说她那好美好美的梦了。

　　随着年龄的增长，她变得越来越漂亮越来越可爱了，18岁那年，她考上了北方的一所著名大学。大学期间，很多男孩拿着火红的玫瑰追求她，他们说她是他们心中的女神，可是她对他们不顾一屑，因为她知道要是自己没有那么一张美丽绝伦的脸，他们或许根本不会理睬她的。

　　她就是这样的一个女孩，当别的女孩和男友花前月下卿卿我我时，她忘情地徜徉在知识的海洋里，她向往电视节目主持，她发誓以后一定要当一名优秀的电视节目主持人，她要让所有观众都喜欢上她的节目，每当她一个人静静地走在校园林荫道上时她就喜欢这么静静地想。

　　毕业后，计算机专业的她分到了一家电脑公司，工资很高，老总也很赏识她，可是她整天闷闷不乐，因为她还在怀念她的梦，她的主持梦，她就这么忧郁地过着自己的每一天。

　　那天是她出生以来最开心的日子。她和他的相遇美得就像电影里最浪漫的情节一般，美丽而充满诗意。从那天起，他就成了她心头最美的牵挂，就像，就像小时侯心头的那只蝶一般，每时每刻在她心头翻飞如云，她喜欢听他的声音，喜欢看他浅浅的笑，喜欢他用温热的嘴唇亲吻她飞红的脸庞，喜欢他搂着她的细腰，喜欢他握着她的手走过大街小巷，暖暖的。

　　他跟她说自己小时侯在果园里偷摘苹果的捣蛋顽皮，跟她说初中时暗恋临桌女生的情窦初开，跟她说他第一眼看到她时的砰然心动。于是她便在一旁嘻嘻地笑，笑得他的脸直发红。她很喜欢看他泛红的脸，她想那是他最令她喜欢最可爱的东西了。她也跟他说起自己小时侯依偎在母亲怀里细数漫天繁星的烂漫天真，跟他说起少女时代崇拜高仓健的狂热如火，跟他说每次见到他时的甜蜜满怀。最后她还跟他说起了一直深藏心头的蝶，那只美丽异常的蝶。

　　他伸出双臂，紧紧地拥她入怀。他说那是一个很美的梦，而美丽的东西是应该长久存于心间的。他的温柔与体贴让她的心猛地一颤，她知道自己这辈子再也离不开他了，就为了他的这句话。

　　他附在她耳边轻轻地说我们结婚吧？她的心一阵狂跳，她的脸一阵发烫，她转过脸，看着他温柔的眼神，轻轻地点下了头。

　　她轻快地走在通往百货大楼的公路上，不断玩弄着指间的钻石戒指，虽然依他的经济实力只能送一颗那么小的钻石给她，可是她已经很满足了，在她的眼里那就是世界上最美最大的钻石了。她抬头看看头顶的天，好蓝好蓝。

　　她又看到了蝶，飞扬的蝶，就在路边那个边唱边跳的小女孩的连衣裙上，五颜六色的蝶随着小女孩的跳动欢快地飞舞着，她停下了脚步，静静地看着，微微地笑着，脑海里又出现了那只蝶，那只活在她生命中的蝶。

　　忽然，她看到对面一辆货运车迎面冲了过来，快得令她感觉到一阵头晕目眩，眼看就要撞到小女孩身上了。

　　她猛地扑了上去，奋力推开小女孩，她的手触到了裙上的蝶，暖暖的。

　　她感到腰边一阵钻心的疼痛，然后就什么都不知道了，在她的眼睛就要闭上的一刹那，她看到了不远处有一只蝶在飞，那只蝶很眼熟很眼熟，就像那只一直翻飞在她生命当中的那只蝶一样，在晴朗的天空下飞着，艳丽异常。
