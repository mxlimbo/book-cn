和沈庆的一次同台演出

　　想起那时的情景真让我激动不已：舞台下是一眼望不到头的黑压压的一片。只能容纳一千多人的交院礼堂挤着数千名本校和外校的学生。虽然他们被挤得几乎透不过气来，但仍然为自己能够挤进礼堂来而无比自豪，因为礼堂外还有几千名学生正在因为无法挤进礼堂而急得直跺脚呢！

　　演唱会马上就要开始了！我的心不由得一阵激动。深深吸了一口气，清了清嗓子，手在吉它琴弦上拨了拨，偌大的礼堂便飘起了一阵悠扬的琴声。听众立即安静了下来，正在为听众签名的沈庆放下手中的笔，扭过头来对我笑笑，转过身抱起吉它，挂上背带。用他那特有的略带沙哑的声音对着台下的听众说："今天很荣幸到交通学院与旭子一起同台演出。这次演唱会我们将为大家献上由我谱写并演唱的一些校园民谣。希望大家喜欢！谢谢！"

　　台下立即爆发出一阵掌声和欢呼声：是的，人们太激动了。能现场听一听被称为"校园民谣三大原创歌手"之一的沈庆演唱对于一个爱好校园民谣的人来说确实是一件值得自豪一辈子的事情。

　　我轻轻拨动了琴弦，于是熟悉的音乐便在礼堂内响起，我和沈庆一起合唱那首迷倒中国千千万万歌迷的不朽之作《青春》："青春的花开花谢，让我疲惫却不后悔。四季的雨飞雪飞，让我心醉却不堪憔悴……"台下听众的掌声雷鸣，震耳欲聋。后来湖大的一个老乡跟我谈起那时的情景时说："当时，你们交院发出的掌声我们湖大都能听见。"当时的演唱会有多么轰动就可想而知了。

　　在听众近乎疯狂的掌声和欢呼声中，我又和沈庆一起唱完了《没有想法》、《岁月》等经典校园民谣。演唱会刚结束，听众便一拥而上，围着我和沈庆纷纷要求签名。刚开始我还很认真很仔细地签，到了后来手麻木了，便敷衍似地签着，到了最后由于人太多，氧气缺乏，头开始发晕，连自己签了什么也看不清了。现在我已经忘记当时自己是怎么走出礼堂大门的了，只记得迷迷糊糊中好像是工作人员和沈庆奋力拉着我走出礼堂大门的。沈庆在演唱会开完的第二天便回北京去了。在送沈庆离开时，他递给我一张歌谱，告诉我说那是他刚写的歌，送给我的。

　　直至今日，我都还在想：歌星的肺活量是不是都很大？要不然在氧气稀薄的人群中他们怎么不晕倒？
