没有一天不想你

　　从牵上你手的那一刻起，我就知道自己这一生已经不可能再有别的女孩让我如此心迷神醉，所以，我把自己所有的爱都给了你。从此以后我的心便不再流浪。我相信我是找到了我前生约定的那一半。

　　和你在一起的每一天都是我开心的日子，和你相处的每一刻都是我最美丽的记忆。我们一起逛大街，看着那些你喜欢和不喜欢的洋娃娃，看着它们静静地对着你笑；我们一起游览公园，在湖边看着鱼儿在水里自由地游走，阳光透过头顶班驳的树影打在你的脸上，泛着淡淡的光芒，你的眼眸在阳光下显得明亮非常；我们一起挤在一台电脑前上网，看文章，聊天，不浪漫，我却很满足；无数个小雨淅沥或是月明星繁的夜晚，我们依偎着坐在沙发上看电视，你时常回过头来问我节目是否合我胃口，这时，我多么想告诉你，只要能和你一起这样依偎着静静地享受夜的宁静，于我已是最大的满足，当你看着看着就睡着时，看着你恬静美丽的脸庞，我在心头暗暗发誓，一定要努力让你过上温馨甜蜜的好日子。从此，在前进的道路上我不断地追寻属于你我的未来。

　　你走了，你说你要去为我们的将来努力。看着你坚定的眼神，我只能放开你温润的手，轻轻地为你祝福。

　　你离去的日子，思念成了我对你唯一的情感表达。从此，溢满你款款深情的来信就成了我最美丽的等待。花开了数载又谢了几回，我们一起栽下的小树如今已是枝繁叶茂。每个晨曦方现的凌晨和晚霞漫天的黄昏，我都端坐在树荫底下，细细地翻阅着你的翩翩飞鸿。你的每一句亲切的话语是我最真最美的牵挂与眷念；你的一颦一笑时时出现在我的面前。显得那么朦胧而不可捉摸。正是这样若即若离的美丽让我足以用自己的一生去温柔地爱护与珍惜。

　　我知道，你此刻正在想着我，正如我正在想着你一样。远方的你或许永远都不明白我到底有多么想念你，但是我相信你是会明白你是我此生永久的唯一。没有我在身边的日子，自己照顾自己就成了我对你最大的叮嘱，我想你是会明白这句话当中所蕴涵的所有爱恋的，就像我明白你温柔绵长的叮嘱一样。

　　你的爱总是让我心怀感动，每时每刻回荡在心中的，只有你如火的热情和款款的深情。该怎么对你说你才能明白我爱你的一颗心呢？或许爱情并不需要说出口，因为心有灵犀永远是爱情的真谛。可是，我真的好想让你明白：没有一天不想你！
