血腥下的反思

　　湖南新化一名年仅16岁的初三学生一连砍了三十几刀砍死爷爷、婶婶、年仅7岁的堂弟，还烧屋灭迹，其行径令人发指，可是看了报道才知道原来直接导致他杀死三个亲人的动机竟是他父亲和婶婶有染。这件事情发生以后，人们都在谴责那个初三的学生，我也为他的行为感到不可思议，感到心悸，但是我们不妨站在他的角度看看，看看到底是什么直接导致了这场悲剧。

　　这名学生的母亲出外打工，数年没有回家，他的父亲常年和其婶婶有染，村里长期有他们的绯闻传出，连孩子本人也成了村里孩子的笑柄。可以想象，这在孩子的心里有多大的影响，可以说这孩子是在家庭的阴影中成长的，这种从小就处于阴影的孩子在不成熟不理智的情况下做出一些傻事可以说并不是很难想象的。

　　我一直无法理解一些父母在生活中是怎么言传身教自己的孩子的。在孩子面前大谈自己的各种各样贪国家财、取公众物的"丰功伟绩"；在孩子面前做出一些枉为人夫、枉为人父甚至枉为人的事情来。想想，这样怎么能让孩子正常成长起来？这样的环境下成长起来的孩子又怎么会有一个良好的心态，怎么和别人正常交往？

　　我们的国家很早就呼吁人们要加强对孩子的科技文化教育、思想素质教育、心理教育，国家还特别强调了家庭应在孩子思想素质、心理教育方面做到主体作用，可是除了科技文化教育，我们的家庭真正做了什么，真正起到了什么作用？什么都没有，我们社会上的一些父母对孩子并不是不关心、不爱护，而是只关心他们认为重要的学习、所谓的是否乖巧，每天给孩子请语文家教、数学家教、英语家教、美术家教，不问孩子是否喜欢、是否适合就剩吃俭用花几万元给孩子买钢琴、每天逼着孩子做一些他们完全不喜欢的所谓的素质教育，而真正该做的思想素质教育、心理教育却远远抛在一边。在这么的环境下成长起来的孩子的思想素质、心理承受能力怎么就可想而知了。

　　该清醒了，真的该清醒了！这样的血腥的场面，难道还不值得我们清醒与反思吗？
