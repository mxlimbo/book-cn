谁为"偏长生"护航？

——一篇来自大学校园的深度报道

　　 他，大学四年本专业成绩平平，却在计算机应用及网络方面刻苦钻研、自学成才，计算机水平超过了许多重点大学科班出身的本科生；他，虽然是理工科学生，却在文学方面造诣极深，在全国报刊杂志上发表作品百余篇，三十余万字，屡屡在全国征文赛中获奖；她，虽然就读专业课程成绩不佳，却有文艺方面特长，多次代表院校参加全国性文艺赛，成绩斐然……

　　别以为记者所列举的人物离你很远，其实，只要你认真观察就能发现：他们就在你身边，他们的影子无处不在，这类学生几乎在每一所大学都能见到。他们，因为与普通学生不一致的言行举止，在大学期间大胆而成分地发挥了自己的特长，取得了一些令人惊诧的成绩，这类学生往往有一个共同的特点：他们大学期间所学的专业课程成绩往往很是一般，甚至很不理想，正是由于这种偏向的存在，所以人们往往称他们为大学校园里的另类一族——"偏长生"。

　　虽然有极少数"偏长生"在低年级时便转到自己喜欢或是相近的专业就读，但由于大学院系严格的分划，使得"偏长生"中的绝大多数还是被迫留在了原先的专业就读。时间磨砺了他们少许的锐气，但他们仍然在课外时间通过自学打下了良好的"外专业"基础，并通过参加一些相应的活动和锻炼提高了自己的业务水平，正是这种深藏于内心的浓烈的兴趣、一些天赋和自身的刻苦钻研，使得他们在这个和自己专业并不相关的"外专业"进步很快，同时由于在这些方面花费了过多的时间，使得他们的专业课成绩落后于其他同学。这样，"偏长生"就出现了。

　　偏长生的出现，使得平静的校园出现了一道别致的风景线，同时也引起了老师、学生的关注，其中有赞赏的，也有反对的。两种意见日益对立，并在校园中长久地辩论下去。成为校园辩论赛的一个最常见的辩题。不管怎样，偏长生的出现是对校园传统价值观的一个强有力的冲击，也给沉寂的校园带来了一缕清新的风气。

　　偏长生由于在特长方面有很好的基础，并掌握基本的本专业（大学所在专业）知识，知识面相对较宽，因此有较大的潜力，在步入社会后较容易发展起来。记者所认识的几个特长生在工作后都发展得比一般毕业生强。如此文所举第一个例子中的学生，毕业时在其所在省份电信局公开招聘中以优异的成绩和很强的动手能力打败一大批重点大学计算机专业学生进入令人艳羡的电信部门工作，并在短短一年内以其出色的业务成绩得到领导赞赏并委以重任。记者的一个好友，大学期间在报刊杂志上发表文章无数，毕业后被武汉一家大报社录用，成为一名光荣的新闻记者。

　　虽然偏长生往往有某一方面的天赋和特长，但是我们很遗憾地看到，现实中真正能在社会上做出一番作为的并没有多少，一方面有他们自身的原因，比如：自学成材毕竟有一定的难度，自学很难做到既系统又有一定的深度，但有一点我们不可忽视，那就是学校没有为这批"另类"的偏长生提供一些必要的制度和硬件方面的条件。甚至很多学校对偏长生都怀着一种歧视的态度。认为他们是一批"不务正业"的坏学生。不但没有为他们提供一些便利和条件，还在纪律和制度上做出了不利于偏长生成长的规定。比如，学校会在综合测评特别是智育成绩考核上做出一番不利于特长生发展的限制，如：规定学年有成绩不及格者不得参加学年奖学金评定；学生不管在其他方面有多么好的表现，只要有一门成绩不理想就失去了几乎所有的优秀学生评定；甚至连专门为偏长生设定的"单项奖学金"也成了"综合奖学金"的另一种延伸与形式。由于所谓全面发展学生众多，而"综合奖学金"数额有限，因此教育部门设定专门为偏长生设定的一种鼓励性质的奖励在实际的运行中也成了奖励够不上"全面发展学生"格却在优秀学生之列或者是"学生干部"的奖励。而那些真正应该从此得到一些鼓励与帮助的偏长生却与之无缘。面对这样不近公平的待遇，许多偏长生思想上有了包袱，不但影响了他们发挥特长的积极性，甚至影响了他们学习本专业知识的积极性与主动性。另一方面，虽然学校或是老师似乎在言语上对偏长生有所眷顾，而在实际操作中却处处不能真正地做到公平、公正地对待偏长生，他们往往凭借着偏长生学习成绩不是很理想这一理由，在入党、推荐单位等方面对这些偏长生做一些限制。我们不能不发出疑问：一个在文学方面有天赋并被专业人士肯定，将来从事新闻、文学工作的学生电力专业考试考试是否有必要非得优秀不可；一个计算机水平远远超过相应大学本科生的学生是否有必要在机械方面门门功课都打90分？或许有人会把这种现象同前一段时间火热的"韩寒现象"联系起来，但是我们说，这两着之间是有一定差别的：韩寒是一名高中生，由于语文之外的所有考试成绩差和作文水平高而倍受媒体和社会关注，高中阶段是一个人打基础的时候，而高中所设置的课程又是对我们将来的成长有重要意义的课程，所以我们必须学好每一门功课。而大学就不同了，大学期间是一个人追求全面发展和个性发展的最重要阶段。追求个性发展应该是我们学生的第一要务，而我们深感遗憾的是我们的学校并没有给偏长生一些自由发展的空间，固然，这有教育制度上的一些问题，而笔者认为最关键的是我们的教育体制没有跟上来。我们的教育改革进行了很多年，也进行了一系列具体而深入的措施，但是我们前一段时间进行的教育改革并没有真正地进入到一定的深度，没有从根本上改变以前陈旧的教育观念，还是以那种"成绩高就是好学生"的思想指导我们的教育改革工作，没有认识到充分发挥与发展学生兴趣对于最大程度发挥学生潜力的重要性。因此也就不可能真正把我们的教育事业提高到一个全新的高度。可以说从"形式（分数）"到"内涵（水平）"的转变是我们教育改革真正进入实质的最关键的一步。

　　如何给偏长生创造一个良好的发展环境，使得这部分有极大发展潜力的学生最大程度发展自身的优势，是值得我们教育部门深思的问题。
