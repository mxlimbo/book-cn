爷爷、父亲和我的幼婴时代

　　爷爷出生在1928年冬天，那时爷爷家很穷，连一件婴儿的小棉衣都买不起，爷爷的母亲只好从自己身上的破棉衣上里抽出点棉絮，再用大人的破旧衣服制成了一件不成样子的"棉衣"以供御寒。那时，国内战乱四起，为了逃避战火，爷爷常常被母亲背在背上四处逃难。可以说，爷爷的幼婴时代是在战火、苦难与颠簸中度过的。

　　相对于爷爷，父亲可以算是幸运的——父亲出生在新中国成立后的第四个夏天，社会环境安定了，生活条件也好了一些，更重要的是，那时人们的思想觉悟提高了。爷爷当时是一名国家干部，一天到晚忙于工作，但他只要一有空，便会坐下来抱抱孩子。

　　爷爷是个"工作狂"，就是在抱着孩子时都不忘自己的工作，对孩子进行一番思想教育，虽然父亲回报他的总是一泡热尿，但他总是擦擦后又很快投入到他的"思想教育工作"当中。父亲长大之后思想特红特专，我想跟爷爷的这种"幼婴教育"不无关系。

　　我是踏着改革开放的春风来到这个世界的。父母亲事后说他们当初想要我这孩子时根本就没有想到我会出生在这么一个有纪念意义的年代。那时，家里的经济条件有了很大的好转，母亲给我买了一个在当时绝对算新奇的洋娃娃，放在我的小床头，而父亲甚至可以从他每月的工资中抽出一部分来给我买营养品补充营养。父母常说我的幼婴时代和爷爷、父亲比起来简直是泡在蜜里一般。

　　看了一家三代的幼婴时代，再看看今天那些襁褓中的孩子们身处营养品的世界、千般宠万般爱的情景，真是感叹万分：时代在进步，中国在发展，而我们人民的生活水平也在发生日新月异的变化，面对着这一切，你怎么能不发出来自内心的感叹呢？
