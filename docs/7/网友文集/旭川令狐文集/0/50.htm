书评三篇

重读博尔赫斯

　　读博尔赫斯，时常为他的智慧与绵密的深沉所感动，他有着绵长广阔的想象力，却在思考与创作中不懈地追求精确与内心的真实感受。他的文学是与他个体的存在并存的。他的价值因为其作品，而他的作品的存在更因为其个体的存在而存在。

　　他是一个为作家写作的作家，他是位不为文体所羁绊的作家，他许多的作品，我们很难确定具体属于哪一种体裁。他是一个"游走在语言和语言之间的幽灵"，他试图以整个的生命去诠释整个宇宙的变化与菁华，因此，探讨宇宙间所有的奥秘便成了他的嗜好与永不变迁的幻想，他在科幻界的地位并不亚于其在文学界的地位，成了阿根廷最杰出的科幻作家。他更是一个诗人，在一定意义上来说，他是以一个诗人的状态出现在世人的面前的，在他眼中，理性和想象成了散文和诗歌唯一的区分点。于是，他在文学创作的空间里极度地发挥了他的想象力，他的想象力能在时间与空间的范围内随意地驰骋，梦幻是其创作灵感的一个重要来源，想象在他的作品中被发挥得淋漓尽致。在《在镜子里》这篇文章里，他写道："起初，他做的梦是纷乱不堪的；不久之后，就自然而然地合乎辩证了，这个陌生人梦见自己在一座圆形露夭剧场的中央，有点儿像被火烧毁的神庙……"。他让自己的想象力极度扩张，同时又用最简洁最精确的语言进行文学创作，他给了读者一个最大的驰骋想象的空间，正如他自己所说："我是一个计算音节的人"。他为了他探索形而上学，以一种不变的、静止的、片面的观点进行着他的观察和创作，正因为如此，在他的眼中，世界是不变、黑暗而恐怖的，在他的文章中，充满了对现实与未来的不安与恐惧。他拒绝现实，逃避现实，逃避激情，甚至于逃避女人，在他的世界里，母亲是他最依赖、最值得信任的女人。他的两位妻子都叠加着他母亲的形象。对现实的恐惧使得他一天到晚以读书为乐趣，这在一定程度上又促进了他创作的再深化与升华。博尔赫斯在其创作中极力地追寻一种高定位的人性的释放与人类本真。他用自己的创作和自己的生命实现了对人性本真的最本质的解剖。

民众的呐喊
――读《对我们的环境发问》有感

　　曾有过太多的愤世疾俗，却在时光流逝后渐渐淡去；曾经棱角分明的脸孔，在岁月的磨砺下变得圆滑世故。于是，便开始用淡漠的目光看待尘世人来人往，花开花落，直到看到许多同龄人如布达拉宫酥油灯前虔诚的信徒般有了一种不灭的信念为自己、为民众大声疾呼，摇旗呐喊时才发觉内心应有的良知和对世俗的愤懑已被尘封，一时间便面红耳赤、羞愧难当。

　　他们不满于现状而在平凡中寻找生命的支点；他们不屈从于凶残与愚昧而在社会的最底层摇旗呐喊，那是一种时代的信念与理想的指引，他们对陋习与愚昧的抨击如同他们对生机盎然的现状和美好前景的赞扬同样令人振奋与深思。他们敢于对现实的一些丑恶的现象作出尖锐的的揭露与批判，令人读来颇觉酣畅淋漓。

　　他们又不是谋图一时的快意、发泄愤慨的愤世疾俗，而是以新闻事实为基础，在此基础上引出鲜明的现实针对性，真正以一种对民众的良知与爱心和一种对自己生存空间负责的态度，从政治到经济、教育，从官员的义务到民众的权利，从物质领域到精神领域，覆盖了社会的整个领域，为了一种社会信念和理想而呐喊，他们在抨击时弊的同时以极其理性和深度的目光透视了整个整体，更深刻地揭示了时弊产生的根源，为解决问题，除污去垢提出了自己极富深度的见解，这是一种充满理性的声音，他们以一种理智与理性的思维对我们的环境做出了全面而独到的拷问，用一种尖锐而不乏理性的语言真实地反映了民众的呼声，他们是我们时代的最富有激情的一族，他们是我们时代精神的捍卫者，他们是社会前进最忠实的清道夫。

　　这是真正来自民众的理性的呐喊。

作者：罗强烈
主编出版单位：中国青年出版社

伟人与你心灵的对话

　　"我的虚荣心是：用十句话说出别人用一本书说出的东西――说出别人用一本书没有说出的东西。"在尼采短段56岁的生命当中，他一直用自己的行动在努力地实现着自己的诺言。

　　他23岁参军、26岁担任古典文献学教授、26岁到56岁短短三十年的时间里写出了十七部书。他是唯心主义哲学家，唯意识论者，却受到世界各地各种阶层人们的喜欢与推崇，不论是唯心主义还是唯物主义者。他和黑格尔的一样，在世界思想史上是一个不容忽视的重要篇章。

　　尼采所处的时代（十九世纪中后期），腐败和虚伪现象在当时的历史和社会环境下已经是一曝无遗了，尼采不是像托尔斯泰一样怀着对农村社会的和平朴实，相爱互助的憧憬，走上了人道主义的新宗教主义；他也不像马克思一样研究工业结构，知道现代文化、现代社会的发展趋势，走上了科学的社会主义的探索道路。尼采用他稀有的想象力和特有的热情将希望和理想投射得很高，是他教着我们走上一条桥梁，由毁灭到重创，走向更加美好的明天。

　　尼采所有的著作都是用一种十分经典十分简练的语言来表达他的这种思想和迫切的宣传意识。毫无疑问，他是一名成功的宣传家（我更习惯于用"宣传家"而不是"演讲家"来形容他）、思想家、哲学家。在他逝世后整整一百年的时间里，他的著作被世界各地的出版商无数次地翻译、出版。直至百年后的今天，尼采，这样一个人类思想史上的伟大人物的文集仍然在世界各地热销，这不得不说是人类思想史、出版史上的一个奇迹。

　　尼采除了在思想史上有着极重要的地位之外，他在诗坛上的成就也是不可忽视的，他的抒情诗歌浪漫而又忧世，箴言诗充满智慧而哲理，在他或长或短的诗篇当中，我们能很容易地为他来自内心的一种的深层次的思想与理念而感动。

　　值得我们注意的是，由于尼采是唯心主义者，所以我们虽然可以从他的著作当中全面了解到尼采对哲学、历史的一系列看法，但是站在唯物主义上看，他的有些观点是十分荒谬的，这就要求我们在阅读他的著作时注意取其精华、弃其糟粕，以更好地学到我们所想要的知识。

　　现今书市上尼采的文集不下十六种，另外还有他的单本著作，不过最受欢迎的还是改革出版社出版的《尼采文集》，这本书以其编辑定位准确、重点齐全而在尼采的中文译本中独占熬头。
