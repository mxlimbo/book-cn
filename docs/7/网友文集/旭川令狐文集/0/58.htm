男人与女人

　　男人和女人总是有很大的不同
　　据说源于上帝造人时给他们安装的骨骼数目不等

　　男人的身材总让人想到恐龙或虎豹
　　女人的身材总让人想到魔鬼或水蛇

　　男人喜欢现实
　　所以不论什么时候总是圆瞪着眼
　　女人喜欢幻想
　　所以接吻时总是喜欢闭上眼睛

　　男人的泪是奋起的标志
　　女人的泪是求助的资本

　　女人占有了男人就想改变他
　　男人改变了女人才想拥有她

　　女人总是问：到底谁爱我？
　　男人却在问：我到底爱着谁？
