书店经营管理何时走向专业化？

　　近几天，笔者外出买一本复旦大学李良荣教授编写，高等教育出版社出版的《新闻学导论》，本以为在长沙这样一个省会城市买一本新闻学经典教材会很容易，谁知跑遍了长沙近十家大书店（书城），竟然连它的影子都没见。不但如此，在找书的同时还发现了一些值得书店经营者思考的问题。

　　一、书目分类专业化欠缺。

　　在当今这样一个信息化、高效化的时代，书店经营者为了读者查找书籍的方便，书目分类是一个简单易行、方便实用的方案。长沙市内的绝大多数大型书店都采用了这种方法。

　　但是在实施这种方案时，由于书店工作人员专业性不强，出现一些错放，甚至乱放的情况便在所难免。例如，笔者在一家大型书店的计算机软件专柜放着UNIX 和LINUX 的参考书，只要稍有一点计算机常识的人都知道，UNIX和LINUX 是计算机的一种操作系统。把其放在软件专柜实在是有失妥帖。（这家书店中关于UNIX和LINUX 的大约三十几本书全放在了软件专柜，这就排除了读者看完后乱放的可能。）令人担忧的是这种情况并不是一家两家，而是有很大的普遍性，几乎在各大书店均存在。

　　二、书籍的实用性与时效性问题。

　　书店的书籍订购是一项很重要的工作。如果书店的订购没有注意实用性与时效性，那么很可能会失去很大一部分顾客，一些书店订购书籍时并没有注意到实用性与时效性的问题，买的书中很多都是一些时效性不高的，或是对读者并没有很大使用价值的书籍。这样对于那些专业性比较强的读者的要求就不能满足，自然会失去一些读者和顾客。

　　书籍的实用性与时效性不好还会影响一些读者的读书定位，甚至会影响读者的读书品位。曾经有图书出版商作过一项调查表明：到书店的读者当中，有61% 的人并没有明确的购买目的，也就是说有很大一部分人到书店时并没有打算要买什么书，他们只是在节假日或平时空闲时间去书店随便看一看、瞧一瞧，看到有什么值得自己买的书籍才会掏钱去买。这样看来，书店书籍的订购不单单是关系到能不能满足读者的要求，而且从某种意义上说还决定了读者买什么书，看什么书。因为那一部分读者买书的前提是书店里有什么书，他才会从中选什么书。从这个角度来看，书店还肩负着向读者推荐好书的责任。这一点是任何书店管理人员都不能忽视的。

　　三、书店的全面性与专业化问题。

　　一般说来，大型的书店一般是向大型化发展，而小型书店则应当向专业化方向发展。

　　但是我们看到的情况并非如此。对于这一点，我们常跑书店的读者就深有体会了，定王台书市一向是长沙很繁荣的一个图书供应点。其中一个最重要的原因是那里的图书打折，但是按照笔者的推算，定王台本应当更繁华的。阻碍其繁华性的原因有很多，其中一个就是定王台书市没有实行书店（书摊）专业化。有一位读者说过一句很能令人深思的话"定王台只有两家书店，一家专门卖电脑书，一家专门卖畅销书。"他的这句话虽然有点过了头，但从某种意义上说也还是正确的。君不见，定王台一楼二楼绝大多数的书摊都是卖畅销书，前一段时间畅销《绝对隐私》，你就能在定王台80% 以上的书摊看到这本书，这段时间卫慧的《上海宝贝》被炒得沸沸扬扬，于是，书摊摊主们又为读者提供了这本被出版社自禁的书籍。自从两年多前我第一次去定王台起，我便一直在想一个问题：要是定王台的每一个书摊都找到一个适合于自己的专业性书籍类型，那么这个深受读者喜欢的书市会繁荣成什么样的景象？当中可能有很多的原因，当中或许有摊主的顾虑：书籍专业化后，顾客是否会减少？生意是否还能坚持下去？或许当中还有很多很多其它方面的原因，但有一点是肯定的：定王台书市一楼一家专门卖法律书籍的书店是人山人海的很繁荣的。

　　要解决以上这些方面的问题还有很多事情要我们去做。对于一般书店经营者来说，书籍专业化是一个必然的发展趋势。虽说书籍全面化是一个好的趋势，但这一般是针对大型书店而言。更何况，专业化和全面化并不是完全对立的，而是既对立与又统一的。大书店的全面化是建立在各个领域书籍的专业化基础上的，而中小书店的专业化则是建立在某个领域书籍的全面化基础上的。对于书店经营者来说，提高自身的文化素质，多关注图书最新动态，多订购优秀图书，并在书店员工上下点工夫，以使书店经营、管理工作更加专业化、正轨化。
