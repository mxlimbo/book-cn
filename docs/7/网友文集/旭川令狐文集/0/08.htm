其实不想走

　　给一个承诺，我就可以守侯一生，不需要时常相依相偎，不奢求时刻脉脉相望，你我分离时唯一所求，就是在我不在身边的日子，你能够时时想起我。

　　没有你在身边的日子，我无时不刻不在想着你。在我拥有的短暂的休假期里，我不远千里地去看了你，看我日思夜想亲爱的你。

　　对于我的到来，你很是惊奇，很是欣喜，你说你没想到我会来，没想到我会在短短的四天时间里不远千里来看你。连我自己都不知道我怎么可能在四天的假期里跑那么远去看你，亲爱的你又怎么会知道呢？

　　我把所有烦琐的世事都放下，我要陪着你，陪着你度过快乐无比只属于你和我的四个日日夜夜。你知道吗？每次和你在一起的时候，我总是能体验到淡淡的温馨与甜蜜，在我的心里幽幽荡漾。喜欢看你长长的睫毛；喜欢看你含情脉脉的似水明眸；喜欢看你含羞绯红的脸庞；喜欢，喜欢轻轻地搂着你，看着你闭着眼睛一脸甜蜜的样子；喜欢温柔地在你的脸颊上留下我深情的热吻；喜欢轻轻拉着你的手，陪着你在艳阳高照的日子里逛大街、跑小巷；喜欢在深夜里和你一起躺在宽大的竹椅上细数满天繁星，构想着你我并不富裕并不浪漫却很温馨很甜蜜的将来。

　　我没有什么好对你倾诉的，除了对你无尽的思念；我没有什么好送给你，除了我一颗火热的真心。紧紧搂着你的时候，我只要你听得见我的心跳，我想你会明白的，我的心因为你才跳得那么强烈而固执。

　　你知道我不会跟你说什么甜言蜜语，可是你说你能听得见，因为你我的心已经完完全全地融合在了一起。你说这话的时候，我看见你的眼神中淡淡的爱恋，于是，我的心便莫名地感动起来，不为别的，就为了你。

　　四天的时间很快就过去了，离开的前一天晚上，你在橘黄的台灯下看着我，满脸的笑意，可是我看到了你眼里闪烁的晶莹泪光，你一句话都没有说，只是静静地看着我，你幽忧的眼神为什么让我如此心痛？以至于当我把你搂进怀里时我竟如此清晰地听见了心碎的声音。你在我怀里嘤嘤地哭了，哭得我的心都完完全全地碎了，亲爱的，别哭，别哭好吗？你用力地点点头，可是你的泪水还是不断地流了出来。我掏出手帕轻轻地为你擦去，可是泪水就像一个不听话的孩子，刚刚擦干又马上流了出来，擦到最后，我也干脆放手不擦了，紧紧地搂着你，把头埋在你的发间，我的泪也流了出来，我没有理会它，就这么让它静静地流着，直到我们都慢慢睡去，就在那么一个下着雨的夜晚，我们就这么相拥着慢慢睡去。

　　你和我站在月台面对面地对视着，时间仿佛在此时停止了一般。铃声响了，列车马上就要开了，我低头在你脸上轻轻一吻，我就感觉到有一滴凉冰冰的液体滴到了我的脸上，我转身便走，不敢回头，不要怪我狠心不回头好吗？亲爱的，我怕，我怕我的泪又不争气地流了出来，此时我唯一能做的就只能是加快脚步上车了，其实真的不想走，如果我能选择。

　　列车缓缓驶出车站时，我看见你早已是泪流满面了，我对你挥着手，不敢对你再说一句话，虽然我心里有着那么强的渴望，那么燥热的渴望渴望满含深情地对你说一声我爱你，爱你一生一世。

　　当你的身影在背后变成一个模糊的影子时，我的泪水夺眶而出。
