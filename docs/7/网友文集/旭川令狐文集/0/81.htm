夜是心情

　　我不明白自己为什么会有这种感觉，但我却是那么真实地感受到了一种凉意，一种直逼内心深处的凉意，仿佛整个世界都要离我而去的那种感觉。

　　每天深夜，当我独坐窗前看着窗外的灯光一盏一盏地熄灭，直至最后四周一片黑暗，只剩下自己窗边的灯还亮着时，那种感觉便像自己到了一个很黑很黑的小屋，我大喊，可是却怎么也叫不出声来。仿佛是在另一个世界一般。

　　那是一份令我时时在深夜惊醒的爱情。许久以来，我一直深深地隐藏她，我想忘记她，而她却时时出现在我的眼前，一直以来，我都感觉到有一条很长很长的线把我和另一个世界的她紧紧地拉在了一起。这条线很长很长，长得以致于沉甸甸地拉得我的心在鲜血直流。

　　像所有的爱情悲剧一样，我的爱化在了如诗如画的春天。她去得那么早，以致于我连她的最后一面也没有见到。等我到时，一朵原以为会一辈子开在我心间的鲜花竟然那么脆弱地了。只剩下漫天飞舞的春雨，和我那颗早已没有知觉的心。

　　在一次玩笑中，曾经问表妹同学中的一位也把我叫" 哥" 的女孩：" 如果我27岁那年还没有找到一个爱我和我爱的女孩，你愿不愿意嫁给我？" 小丫头的脸变得绯红：" 如果那时我也没有找到生命中的另一位，我一定让你牵着我的手，陪我走过余下的春夏秋冬。" 看着她纯真而柔和的明眸，我感觉到了自己的心在发出一种很细微很轻柔的声音。那是一种足以令我感动一生的东西。

　　那个女孩现在还在念书，每周都给我来信，那是一种迷梦般的感觉。令我久久不敢相信：" 五年后，若是我还没有找到一位爱我的和我爱的女孩，我是否真的要和她一起走进另一种生活的开端？" 明天或许是阴天，或许是晴天，或许还如今日一般阴雨绵绵。但我的心呢？是否会破开阻迷，抑或依旧沉寂如昔？
