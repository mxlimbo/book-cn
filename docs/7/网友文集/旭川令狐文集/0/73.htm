那一刻，我听见了生命的声音

　　周末和同学一起出去野炊。酒饱饭足之后，我独自一人坐在小溪边，想着心事。

　　一只出生不久的蜻蜓从头顶树枝上掉了下来，落到我的肩上，我不及细想，便伸手把它拂去，蜻蜓掉入溪水中，等我回过神来时，只见它在流淌的小溪中扑腾着它那四片在阳光的照耀下显得分外刺目的翼翅。我竟如此清晰的看到它大大的眼睛里透露出的无助与对生命的渴望。我的心忽然有一种崩裂般的疼痛，我就像疯了一般跨入初夏齐腰的溪水，扑向那颗微小的生命。轻轻把它托在手心上，那一刻，我觉得我的手心竟是如此的沉重，沉甸甸到压着我。

　　同学却用一种惊奇的目光看着我，我浑身都湿透了，但我的心是暖暖的，因为我挽回了一条险些被我摧毁的同我有着一样生存权利的生命。在那小溪边上，我的耳边无数次地回响起溪水里蜻蜓扑腾着翅膀的声音，我相信，那是生命的声音。
