拆城与虚荣有关

　　最近媒体报道，舟山市在省市甚至中央的干预下，依旧我行我素，拆了舟山的古建筑，一个文化名城一夜间成了一片废墟，这样一个消息真的让人咋舌。

　　我一直在想：为什么民愤和各方面的压力这么大，舟山市为何依旧一意孤行？我想除了文中提及的与经济利益有关之外，其中还有一个很重要的原因，那就是——虚荣。

　　我并不想贬低舟山市某些领导的素质，但是从文中我们可以一目了然地看出在这场拆城风波，舟山市决策人员的虚荣心理确实起了极大的作用。

　　作为政府部门，希望自己所管辖的地区高楼林立、名声在外本就是无可非议的，这也是政府部门应该做的，要是一个政府部门不思为民做点实事，一天到晚无所事事，那么我想这样的政府部门也是没有存在的必要了。但是凡事都应该讲究实际，各地有各地的具体情况，作为政府部门的领导，一个具体的决策可能会影响到许多民众乃至整个地区的生存与发展。我们不能想象,如果各地都像舟山一样"立志"建成一个钢筋水泥的世界,那么我们山清水秀、景色迷人的大好河山会便成什么样子。反正我是绝对不希望我去黄山、泰山游览时是去爬一座由钢筋水泥堆积成的毫无生气的假山的。不讲究本地的实际,硬要把一个大好的古城建设成一个所谓的现代都市，甚至不惜违背省直至中央的指示，把一个好好的文化名城拆得支离破碎，这样的事情在以前我是没有听说过的，甚至在我看了这篇文章之后我也不敢相信——我真的不能相信这是个事实，可是，我们谁也不能否认其的真实性。我为我们的国家又消失了这样一个古城而痛心疾首。

　　我不知道作为一个市领导，该不该有自己的虚荣心，我不是一个思想境界很高的人，所以我说我也有虚荣的时候，有时甚至在工作中都很难避免这个隐藏在内心最深处的弱点，但是作为人总应该有必要的自制力，一个为百姓谋福利的政府机关把中央的指示和百姓的意见当耳边风，这样的领导，甚至是领导机关真的很令人不可思议。

　　不知道我们的祖先看到他们的后人这样折腾他们的成果会怎么想，我想他们一定会因为有这么虚荣的后人而羞耻吧！
