爱你的理由

　　你说，只要我说出爱你的十个理由，你便嫁给我。我感觉得到我的脸在发烫，我感觉到我的心在狂跳，我努力地想，努力的想你的好，努力地想你的妙，可是纵使我搜肠刮肚，怎么也找不出十个爱你的理由。

　　美丽能算爱你的理由吗？不管你再美丽，都有肌肤变皱、容颜老去的时候，到那时我就不爱你了吗？不！爱情是两情相悦、地久天长，决不会因为爱人容颜的老去而改变我对你如火般的热情。所以，美丽不是爱你的理由。

　　温柔能算爱你的理由吗？不管你性情再好，都有心情坏极、乱发脾气的时候，到时候我就不爱你了吗？不！爱情是两个人相互体谅、相互了解，决不会因为你的性情转变而改变我对你的温柔与体贴。所以，温柔不是爱你的理由。

　　可爱能算爱你的理由吗？或许你会认为我很俗，但是可爱终究会随着你的美丽的流逝随着你年龄的增长而慢慢离去，到那时我就不爱你了吗？不！爱情是两个人的相知相守、相亲相爱，决不会因为你的点点变化改变你在我心中无法替代的地位。所以，可爱不是爱你的理由。

　　体贴能算爱你的理由吗？如果你为了工作、孩子而没有尽到做妻子应有的责任，那时候我就不爱你了吗？不！爱情应该建立在互相谅解、互相支持的基础之上，决不会因为一己私利而阻止爱人前进的脚步。所以，体贴不是爱你的理由。

　　我找不出爱你的理由，但是我也不能说不爱你。我不能说不爱你，因为你是我心中最美丽的一朵花，不论春夏秋冬，永远都不会凋谢。

　　我不能说不爱你，因为你是我心中的神秘的天使，不管我遇到什么不快与痛楚，你都能用你的真心和爱心把我安慰。

　　我不能说不爱你，因为你是我心中最可爱的一颗星，不论风吹雨打，我都能在你的指引下走遍天涯海角。

　　我不能说不爱你，因为你是我今生的唯一，不管世事有多沧桑，我都要陪着你一起走过人生的每一个春夏秋冬。

　　那么，请你告诉我，什么是我爱你的理由？看着你迟疑的目光和害羞的神情，忽然间我找到了爱你的理由。其实，爱你的理由只有一点，那就是：

　　你就是你，一个世上独一无二的你。
