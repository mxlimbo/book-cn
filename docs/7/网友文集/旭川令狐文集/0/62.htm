超级网虫十八大症状与特征

　　本人近几年潜心研究超级网虫之症状及其特征，近日有所成就，不敢独享，遂于此披露，希望对有志成为超级网虫者有所帮助：）

　　一、最喜欢的音乐是猫叫。一听见猫叫保管心旷神怡、鼻血直流、飘飘欲仙。

　　二、夜"游"症。白天浑身无力，半夜精神极佳，眼睛红得能使小白兔自惭形秽。一般来说，夜晚是他们最好的活动周期。

　　三、看见".com"、".net"的文字就会心跳加快、血流加速。

　　四、偶发感想，欲挥洒大笔之时做的第一件事不是找笔和纸，而是四处找键盘。

　　五、不进"口才培训班"亦能做到"专业名词"脱口而出，遇上不开心之事不骂"他妈的"而骂"TNND" ；遇上佩服之人开口就是"你真NB"，甚至遇上爸爸就叫"GG"，看到妈妈叫"MM"。

　　六、基本掌握一门速记语言，并在工作总结报告上牛刀频试。一张报告有70%是智能拼音符，常常把老板写成"lb"，把为什么写成"wsm"。

　　七、一分钟不摸键盘心就开始发痒，一小时不碰鼠标就坐立不安，一天不收发E-MAIL就急得想跳楼，一周不进chatroom 我就劝你直接到医院那个最安静的房里找他吧。

　　八、看见带下划线的文字就想找个鼠标点击一下。

　　九、上课老师要求抄笔记时脑海里灵光一闪，想到的两个命令就是"复制"和"粘贴"。

　　十、免费信箱一大把（反正不要钱的），每天收信键入的信箱地址花时八十分钟以上。

　　十一、建有个人主页。主页访问量3000当中，保管有299次是硬拉朋友访问的，2700次是自己点击的，还有一次是某一天某一个倒霉蛋偶然窜入的。

　　十二、他的微笑比蒙纳丽莎的还有特色。一般是两只红通通的眼睛瞪成圆形，嘴巴张得奇大，整个儿就是一个笑脸符 ：）

　　十三、应邀出任足球裁判，遇到球员犯规不是猛吹口哨，而是指着球员大叫："非法操作！"

　　十四、中国电信的忠实消费者，每月会拿出工资的一半来孝敬电信组织。

　　十五、"上网益处"资源会的泰斗。说起上网的好处来头头是道、滔滔不绝："上网可以查询资料、上BBS、收发信件、收发传真、看电视直播、发表健康和不健康的评论……"——虽然自己只去过聊天室。

　　十六、屏幕英语越看越流畅，离英语六级及格线越来越远。

　　十七、超级下载狂。花时七个半小时down完office ，才发现手中的光盘里就有这个软件。

　　十八、网友遍布五湖四海、两岸三地、七大洲四大洋；网友职业遍布三百六十行，工农兵学商行行爆满，唯一空缺的就是日本天皇和美国总统夫人。

　　由于本时间仓促及本人水平有限，多有不足之处，望广大读者踊跃提出不正之处，以便作者修改，不胜感激！
