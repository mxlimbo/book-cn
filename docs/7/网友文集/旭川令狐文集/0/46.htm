不和五种人交朋友

　　一、不懂孝顺者

　　孝顺是中华民族的传统美德，也有一个人人品好坏的最重要的标准。孝顺父母、长辈是每一个有良心的人都应该做的。很难想象，一个不孝顺的人会用一种真诚、宽容的心来与朋友相知相交。

　　二、大男子主义者

　　上帝造人时虽然在性格、体格上赋予了男女一些不同的品格，但是毫无疑问，男女在实际生活中的地位应当是平等的。大男子主义者片面强调了自己作为男人的一些体格上的优势，对女子的优点视而不见，他们相信只有男子才是这个世界的主宰者，而女人充其量只是一种配角。他们把女人看作是操持家务的主妇、出外游玩的游伴。大男子主义者所体现出来的深层次的人性品格是：非平等主义，或者叫歧视主义。实际表象是对女子的歧视，引申下去就是对民族、种族的不平等的歧视的态度。剖析其内心，我们可以知道，其实导致大男子主义产生的根源是大男子主义者内心的自卑感。

　　三、一心为钱者

　　现实生活是一个很实在很无奈的社会，我们中的绝大多数人工作的最大目的还是为了谋生。钱对于我们来说还是一个很重的字眼。但是我们并不一定要把"钱"这个字眼时时挂在嘴边。我不想标榜自己有多纯洁，有多"脱俗"，但是如果要我跟一个嘴边常常挂着"钱"，一心为钱者交朋友，我还是会毫不犹豫地说"不"的。金钱只是人谋生所必须的，除此之外，钱一无是处，把钱看得过重，就意味着会把人世间最美好的亲情、友情、爱情看轻。试问，这样的"朋友"，有谁愿意与之真心相交？

　　四、婚外恋者

　　婚姻是爱情的最美好最圆满的结局与回归。和自己倾心相爱的人真心相守地共度一生是许许多多青年男女共同的追求。可是从现实中我们可以发现，许多婚姻中都存在着婚外恋的现象。婚外恋者虽然不能说是道德败坏者，但是至少可以说是道德涵养不高的一类人。

　　五、"贬低别人，抬高自己"者

　　从实践调查中，我们可以发现，现实生活中往往存在着这样一种人：他们往往不失掉任何一个时机地去打击别人，借贬低别人来抬高自己。他们中的很大一部分是农村的或者是受教育程度不高的人。他们自身往往是不自信的一族。心理学原理告诉我们：人的个体如果对本身不自信就容易导致在现实生活，与人相处时产生自卑心理（这种自卑心理往往不为这类人本身所觉察，有很强的隐蔽性），由此便产生了"贬低别人以抬高自己"的想法。

　　六、口口声声"爱国"主义者

　　这类人往往满口的大道理，其实心里极为自私，须知：真正爱国的人是不会张口闭口言爱国的。这一类人往往是虚伪的一群人。
