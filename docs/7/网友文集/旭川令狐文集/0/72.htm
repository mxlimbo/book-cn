母爱无限

　　伴随着"哇"地一声大哭,母亲用自己身上的一块肉和虔诚的祈祷换来了我的生命.这个世界并不因我的到来而更加精彩美丽,而母亲的生活却因我的出现而流光溢彩.

　　于是便同天下所有的母亲一样,母亲用自己的乳汁与满腔的爱心抚养着我,随着年龄的增长,母亲的爱也越来越浓烈,风风雨雨中,转眼已过了二十个春秋.

　　母亲的爱是慈祥的,她会让我在她的怀里细数满天繁星;她会给我讲很多很多我听过的和没听过的故事;她会在深夜起来给我盖好被子;她会给我做可口的饭菜.是母亲的爱让我感觉到了世间母爱的伟大；是母亲的爱让我感觉到作为一个有母亲疼的孩子有多么幸福；是母亲的爱让我在同学们抱怨母亲唠叨时情不自禁地想起母亲对我的刻骨铭心的爱；是母亲的爱让我时至今日仍怀着深深的爱想起母爱的温馨。母亲的爱又是严厉的,她会在我跌到时走到一边告诉我跌到了要自己爬起来;她会在我没完成作业就跑出去和小朋友玩时毫不留情地跑出来拉着我回去.正是在这种严厉的监督和鼓舞下,正是在这种严厉中，我这么一个在别人眼里不会写作文的"笨孩子"在全国各类报刊杂志上发表了数十篇文章;正是在这种慈祥中,我成了一个人人羡慕的好学生;正是在母亲的这种严厉而又不乏慈祥中,我不断地长大成熟起来;

　　念大学以后,母亲对我的爱丝毫没有改变,每周一封的家书和数个电话便连起了我和家的感情基线.虽身在他乡，我仍能感觉到身边终日都围绕着母亲温馨的爱。细细想起来，母亲似乎没有像报刊杂志上的一些母亲一样曾经为自己的孩子做过什么惊天动地的事情。母亲的爱是平淡而温馨的，但正是在这种平淡中，我体验到了我在母亲的心中有多么重要，也体验到了母亲在我的心里有多么重要。一旦离开母亲，我这个二十来岁的大男孩还会想小孩子一样依依不舍。

　　二十年来,母亲在无时不刻地爱着我,而我也在深深地爱着母亲.我为自己拥有这样一个好母亲而高兴、自豪.

　　愿母亲一生好走！
