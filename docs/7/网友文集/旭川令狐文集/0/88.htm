寻求新而有力的新闻根据
——把"旧闻"变成"新闻"的一种行之有效的方法

　　在采访与写作过程中，由于时间把握得不好，或是在当时此新闻还不适于公开发表，很容易造成一些好的新闻白白浪费，那么有没有什么方法可以让这些"过时"的"旧闻"变成适合于当前主题的新闻呢？无数的新闻采访实践告诉我们：把"旧闻"变成"新闻"是可能的。关键是看你怎么把握住新的"新闻根据"。

　　"新闻根据"又叫"新闻由头"、"新闻引子"，大多为最近一段时间发生的最能吸引读者的材料。在复旦大学新闻学院张骏德教授、刘海贵教授所著《新闻心理学》一书中就提到了新闻根据在使"旧闻"变"新闻"中的重要性："新闻根据寻求、发掘得精当和有力，既是新闻来源的有力说明、新闻事实的得力佐证，又常常能使'旧闻'变新闻，收到'起死回生'、顿生新鲜感的效果。" 是否能及时寻求到新而有力的新闻根据往往是决定一个好的"旧闻"可否能变成"新闻"的关键性因素。

　　那么怎么寻求新而有力的新闻根据呢？一般说来可以从以下几个方面入手：

　　一、以相类似的时间段或时间点作为新的新闻根据。

　　可以说这是一种最"呆板"方法，但对于初学新闻采访、写作的新闻爱好者来说又是最实用也是最常见的一种方法。比如，劳动节某某单位举行了义务清扫火车站的活动，而你又由于种种原因没有及时发稿，那么就可以等到五四青年节或者国庆节甚至是明年劳动节此单位再举行相类似的活动时再发稿。稿件可以这样写："今天是五四青年节，××单位组织本单位青年进行了义务清扫火车站的活动。这是继五一劳动节以来此单位进行的第二次义务清扫公共场地活动……"这种方法的优点是简单实用，缺点是拉开的时间长，不利于记者采写工作的开展。

　　二、以相类似的事件或地点作为新的新闻根据。

　　如某校某学生获得省奥林匹克竞赛一等奖，记者便可以在写稿时将自己以前积累的关于该校在奥赛中获得优异成绩的材料写出来："……该校教学工作成绩显赫并在奥赛中频频获奖，如……"此方法的一个特点是要求记者注重平时素材的积累，而不是像上法一样"抱着材料等时间"，此法较适合于一般记者的采写工作。

　　三、以相应的主题或背景作为新的新闻根据。

　　一个老的题材在新的背景下往往能成为一条很好的新闻。如果记者在采写此类新闻时能选择合适、巧妙的切入点，甚至可能成为一条起轰动效应的新闻。如学校教育与家庭教育的结合问题、子女与父母的情感交流问题……这种方法对实施者的要求较高，但只要善于在实践中摸索、总结，掌握这种方法也并非难事。

　　值得注意的是，以上几种分法并没有明显的界限，很多新的新闻根据并没有固定是以新的时间或新的事件地点来区分。如上所举的第一个例子就既可以说是以新的时间又可以说是以新的事件来作为新的新闻根据。新闻爱好者要在具体的实践中很好地体会并不时进行总结。只有这样才能在实践中不断进步。

　　注：1、详见1997年11月复旦大学出版社出版的《新闻心理学》一书第72页。
