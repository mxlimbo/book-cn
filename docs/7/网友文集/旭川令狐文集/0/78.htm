我在学校的一天

　　朝雾犹浓，惦记着昨夜的依衾甜梦，翻身下床，走进如火的运动场，场地上晃动的是年轻的生命。满脸或是刚毅或是娇艳的面庞迎着夏季凉爽的晨风和你擦肩而过， 带起了阵阵炙人的激情。

　　回到寝室，背起书包，匆匆吃过早餐，站在报栏边，一天来的国内外大事便随着身边悦耳的鸟鸣声沁入心田。不时会有鸟儿从脚边轻轻跳过，调皮地在脚上留下一个淡淡的小爪印。

　　一到教室，朗朗的背单词、读英语课文的读书声便在你还来不及提防时一股脑儿地往你耳朵里钻，让我也不由自主地取出英语课本和他们"同乐同乐"。

　　要说上课，那可真是意趣横生，老师或是诙谐幽默或是条理清晰的讲解总让你如饮甘醇，而下课的铃声总是不期而至，甚至于让我求知若渴的心感到一阵隐隐的不安。

　　下午课后的课余活动时间是我们最活跃最开心的时刻，操场、排球场、篮球场、足球场、网球场、溜冰场、游泳池，到处是同学们的身影。锻炼完，从运动场往寝室走的途中，我能真正感受到什么叫"百鸟归巢"，各种各样的鸟鸣声，声声入耳。如一股深山老林中流淌的清泉，静静地划过心头，声声悦耳，经久不绝。

　　晚上，我们可以到阅览室或是教室去看看杂志，了解国家、社会大事，也可以到图书馆、教室温习功课，充实自我，我能感受到那种心灵日益充实的快乐与兴奋，我喜欢那种感觉。

　　自习归来，我会和同学站在寝室门前的走廊上谈心，我们谈一天的学习；谈自己的理想；谈未来与追求。我们憧憬着自己的将来，我们同样对祖国的未来充满希望，因为我们一直坚信：个人的发展和国家是息息相关的。熄灯后，我们便上了床，开开"卧谈会"，然后静静地、带着对新的一天的向往与憧憬甜甜地进入梦乡。

　　理想便在这样的环境与永恒的追求中长成枝繁叶茂的畅想，在不懈的奋斗中细细致致地诠释了整个生命。
