浅析新闻采访中的决定因素

　　一段时间以来，在一些记者,特别是初搞新闻工作的记者中曾流传一种观点：新闻采访没有什么学问，只要在采访提问时问得有技巧一点就行。甚至有人提出了所谓的"新闻采访无学"论。新闻采访真的没有什么学问？新闻采访的决定因素是什么？在这里，笔者想结合自己担任几家报刊记者、通讯员工作中的一些感受谈谈作为一名新闻记者必须具备也是新闻采访中起决定因素的几个方面。

　　一、 强烈的事业心与责任感

　　无庸质疑，事业心与责任感在新闻采访中是至关重要的。新闻工作作为社会分工的一个重要类别，自然要求从事这门工作的人要真正全身心地投入到工作当中来。所以对于新闻记者来说新闻工作的事业心和责任感是其根本。一个记者如果连最起码的事业心与责任感都没有，就算他有再好的文采又有何用？只有记者真正把做好自己所从事的新闻工作当作自己毕生的追求时，才能充分发挥自己的才华，不断学习、不断进步；才会在日常的工作、生活中时刻保持敏锐的新闻感知力，不断捕捉生活中有价值的新闻；才会敢于冒险、不惧牺牲地在枪林弹雨中为读者采写新闻。可以说，只有拥有了强烈的事业心与责任感，一个新闻记者才能真正做好采写工作。

　　二、深厚的业务知识与技巧

　　虽然在我国，很大一部分记者、编辑并非相关专业毕业，有的甚至是土生土长的"土

　　记者"，但我们应当看到这在一定意义上可以说是由我国的国情决定的。我们不能抱着这根"救命草"来作为自己不刻苦学习业务知识的理由。要想做一名名记者，不论通过大学教育还是通过自学，都应当使自己的业务知识不断地得到提高，并在工作与实践中不断地提高自己的采编技巧。定时总结采访、编辑中的经验与教训，积极主动地向新闻学教师、有经验的同事请教。否则，要使自己成为一名优秀的记者实属难事，甚至连正常的采访工作都很难进行下去。

　　三、高尚的道德与为人准则

　　新闻采访是一种长期与人打交道的工作，这就要求记者要有高尚的职业道德，戒骄戒

　　躁、互敬互学。摆正与采访对象的关系，以诚相待，善与和采访对象以及普通人交友。只有戒骄戒躁、以诚相待，对方才会愿意把真实情况告诉你。只有和同行互敬互学，才会不断在工作中提高自己的业务水平。同时，新闻记者还应当摆正自己的位置，不以"权"谋取私利。新闻产品的商业性决定了新闻工作的特殊性。广大记者在采访中应当认清自己工作的特殊性，不谋求"小利"，不贪图"私利"，真正客观、全面地为群众采编好新闻。

　　四、广博的知识与技能

　　记者采访的对象几乎包括了社会上的所有行业，这就要求记者应当是一名真正的杂家。

　　记者要在采访中与采访对象较活跃地交谈，必须对对方所从事职业的基本情况和基础知识有所了解。如果记者对采访对象所从事的职业一无所知，那么很自然地，对方便会在心理上轻视记者，这必然在一定程度上削弱采访对象谈话的积极性。使采访受挫。记者应当在平时合

　　理地安排时间进行业务及业务外知识的学习。不要羞于向别人求教，一个记者应当明确一个观点：新闻业务知识不但要靠自己刻苦学习，还要勤于请教；不但要请教业务知识，还要向各行各业的人请教各行各业的知识。读书应当读得杂一些。尽量拓宽自己的知识面。对于自己所负责采访报道的行业还应进行较深入的学习。另外，记者应当掌握包括新闻摄影、电脑操作、外语、驾驶、辨向测时等方面的技能。对当地的方言也应当掌握。只有这样才能在这个瞬息万变的时代立足。

　　五、深厚的文、史、哲功底

　　要当好一名记者，就必须先当好一名文学家、史学家、哲学家。新闻采访是一门考验

　　文学功底的工作，没有深厚的文、史、哲功底就根本谈不上在采访中应付自如，还很可能被采访对象搞得手忙脚乱，更不用说什么提问的技巧了。

　　六、健康的身体

　　健康的身体对于新闻工作者来说是十分重要的。细心的人会发现，知名记者都是很注重身体锻炼的，著名记者范长江就把健康看成是自己成长的"四个经验"中的一个。著名新闻学教授、复旦大学新闻学院副院长刘海贵教授曾经说过一句很耐人寻味的话"新闻工作既是复杂的智力劳动，也常常是强度较高的体力劳动。"记者由于工作的需要，每天都要东奔西跑。有时候为了新闻的采写还得熬夜，没有好的身体是吃不消的，这就要求新闻记者必须首先拥有一个健康的身体。没有好的身体，有再好的业务理论、再好的文才也是白搭。所以不论工作再忙，记者都应当坚持每天抽出一定的时间来进行锻炼。

　　从以上分析的几点可以看出，新闻采访是新闻学中的一门重要课程。新闻采访不仅有"学"，而且是新闻学的一个"重头戏"。我们在新闻采访与学习中不仅要善于总结，而且要善于学习、系统学习新闻采访这门新闻学重点课程。只有这样，才能在新闻采访中真正做到"心中有学，应用自如"。
