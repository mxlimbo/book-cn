头顶有阳光的日子

　　喜欢阳光，喜欢他赋予万物生命的仁慈与宽厚；喜欢他永恒不变的热情如火。每当穿过繁华的都市，看着眼前匆匆流淌而过的人潮人海时，总会想到阳光，想到这个我们生命的赋予者。只要有阳光，我们就不会感到孤独，因为，阳光打在脸上的感觉永远都如一首诗，暖暖地洒在心头，永不变更。

　　头顶有阳光的日子，我喜欢坐在窗前，静静地看着窗外人来人往的人群，看着他们脸上的点点微笑，在阳光的照射下就像春天里散发出阵阵芳香的桃花李蕊，美不胜收。

　　头顶有阳光的日子，我喜欢和朋友相约去郊游，在青山绿水间荡漾，在柳暗花明中漫步，清新而又带着点泥土芳香的空气给予我的是回归大自然的欢乐与温馨。

　　头顶有阳光的日子，我喜欢和朋友一起带着画板，来到公园的休闲草地上，描绘眼前的万般风景，我们发誓要把缕缕清风、桂枝柳叶上的片片春意都写入画纸。

　　头顶有阳光的日子，我喜欢给父母写一封书信、打一个电话，然后想着父母接到后的情景，那淡淡的温馨总让我的心暖暖的，感动非常。

　　头顶有阳光的日子，我会给女友送去一束火红的玫瑰，在她拥着我的脖子亲吻我时，我能看到阳光在她眼里闪烁，美丽得能使我整颗心都陶醉。

　　头顶有阳光的日子，我会为每一个爱我的人和我爱的人合掌祈祷，祝愿每一个关心我和我关心的人事事顺利，我还会为每一个不相识的人企福，祝愿阳光打在他们脸上，而长久留在他们心头的则是那种暖暖的春意。
