让我离开你

　　让我离开你吧，如果你还要飞翔。虽然离开你后我将没有快乐的理由，但是我相信这是我们最好的选择。你是天空的苍鹰，飞翔是你的天分，为什么要为了我而放弃你的天性？如果我是你飞翔的羁绊，那么我愿意把对你的爱变成最美好的期盼与祝福，伴随着你神游天涯。

　　让我离开你吧，如果你没有勇气远离。虽然我也不愿放开你温润的小手，但是我相信只有勇敢地走出这一步，我才不会在将来的日子里陷入无边的愧疚与自责。你是追日的夸父，天边的骄阳已经离你很远，难道你还不出发去追寻？如果我曾经耽搁了你宝贵的时日，那么我愿意做河道边的一滴水珠，在你饥渴时给你一份琼液般的甘醇，让你离自己的梦想更进一步。

　　不要告诉我你不在意门户之见，因为我实在不能忍受旁人怪异的目光。如果你一月的工酬能抵十辆令人艳羡的宝马，那么我一月三千的工资只能够得上你一周上酒家的小费。如果说爱我成了你的一种负担，那么为什么不当机立断地将我放弃？

　　不要告诉我你不在意我们的高度之距，我只能告诉你其实我真的很在意。我不能承受别人满目异样的目光；我不愿我们走在大街上被别人怪怪瞧着的不安；我不愿和你接吻掂着脚跟的感觉。我愿意去找一个一米七四下的小女孩，如果让我选择。

　　爱情总能让人头昏眼花，我不愿你被我们的爱情遮住了你明亮的眼睛；爱情的力量真的很神奇，总能让你把我的缺点看成是我最可爱的优点；爱情总是让你看不见窗外优秀的男孩，而在你的眼里，我永远是世界上最富有魅力的。可是迷雾总有消去的一天，力量总有消退的时候，魅力也将随着我年龄的增长而慢慢老去。那么为什么不离开我，让我的魅力我的好永远地留在你记忆的最深处？

　　让我离开你好吗？在我不在身边的日子，请好好保重你自己，不要老是记得你我的相识和相知，离开爱情羁绊的你一定会很快发现其实我并不是最适合你的另一半；让我离开你好吗？或许没有爱情的日子我会找不到令自己快乐的理由，但是请你相信：只要你过得好就是我最大的幸福。在离开我之后的日子，你不必记得你我的相遇、我的出现，只要你内心深处有你我相处的最美的记忆，就已经足够了，不论是对于你，还是对于我。
