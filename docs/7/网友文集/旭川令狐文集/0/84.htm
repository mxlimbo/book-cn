服务听众、创建品牌、推陈出新、不断进取

——从湖南广播创办特色谈新时代广播节目的定位与创新

　　在当今报刊、电视电影占据宣传媒体主体地位的时代，湖南广播仍受到广大听众的欢迎，里边自然有各种各样的促成因素，但有一点是至关重要的，那就是湖南广播时刻在了解听众需要以适应听众品味作相应的创新，使得节目对观众永远富于吸引力。

　　记者经过对湖南省省会长沙的六家电台（湖南电台新闻频道、交通频道、经济频道、文艺频道，长沙电台音乐频道、星沙之声）长期的节目特色研究，总结出了以下几个方面的特色，希望能对国内电台的特色化发展起到一定的抛砖引玉的作用。

　　一、了解电台定位、搞好特色宣传

　　虽然各个电台都负有宣传国家政策、方针的重任，但从具体的方面来说，各家电台又有自己鲜明的办台定位与宣传方针。搞好特色宣传对于各新闻媒体来说是首要的。所以，只有真正明确本台的办台定位，才能在栏目策划中真正做到定位准确、依据有力，搞好特色宣传。如湖南电台经济频道着重宣传三湘大地日益发展的社会主义经济；湖南电台交通频道重点依托自身与省内外交通单位、院校、媒体的紧密联系，重点宣传国内外交通方面的及时快报和重点新闻；湖南电台新闻频道依据自身雄厚的新闻采写队伍，抓好新闻采写、编辑与评论，定点及时地为听众做好新闻宣传；长沙电台音乐频道则充分利用自身与国内外音乐界较为密切的关系做好音乐专栏节目，得到了广大听众的一致好评。由此可见，真正了解电台定位、搞好特色宣传是现代广播电台宣传的一个极重要的环节。

　　二、明确受众特点、真正服务听众

　　相对来说，每个新闻媒体都有其相对稳定的宣传听众群体，广播电台的宣传也不例外。只有真正明确了听众群体，明确了听众特点，才能做到从栏目安排、主持人员选取到具体宣传语言特点的明确与一步定位。在这一点上，湖南电台交通频道可以说是做得极为出色的。该频道建台之初就作了大量的社会调查。发现该台听众是以各类司机、高校学生为主，便在节目安排和主持人筛选上做了大量的对比工作，并在节目改版后广泛答征求听众意见，真正做到了急听众之所急、供听众之所需的办台宗旨。当然，要明确电台听众的特点并不是一件很容易的事情，需要对受众（对于电台来说就是指听众）群体做一番调查（甚至是大范围的调查），明白听众真正想听什么，什么才是听众最需要的。只有这样才能真正做到想听众之所想、急听众之所急，与广大听众心贴心。

　　三、培养名牌主持、创建名牌节目

　　名牌媒体培养名牌记者、编辑、主持，而名牌记者、编辑、主持又能无形间为其所在媒体起到一个宣传的作用。由于电台宣传的独特性，培养名牌记者、名牌编辑往往并不容易，而要培养一个名牌主持、创建一个名牌节目相对来说就显得容易多了。湖南广播有许多名牌节目，如湖南电台文艺频道的"夜色温柔"与"相伴到零点"、经济频道的"心灵之约"都是闻名全省乃至全国的名牌栏目。而以上几个栏目的主持也是深受听众欢迎的名牌主持。特别是湖南电台文艺频道心情类节目"夜色温柔"连续几任主持（如柴静、杨璟等）都是听众十分喜欢的偶像主持；湖南经济频道深夜谈心类节目"心灵之约"主持罗刚则以直而敢言的率真谈吐赢得了广大听众的厚爱。值得注意的是名牌主持与名牌节目之间往往是密不可分的，名牌主持主持的节目往往容易成为名牌节目，而名牌节目的主持又往往比普通节目主持更容易成为名牌主持。名牌主持与名牌节目的培养与创建往往是一家电台扩大知名度的关键。

　　四、不断推陈出新、敢做时代先锋

　　当代的社会是一个知识与创新的社会。如果一家电台只懂得守着原有的几个固定栏目，不思创新与不断进取，那么就算原来的栏目布置与人员安排有多么合理，这家电台都将会最终走向失败的边缘。只有在适应社会的发展中不断调整节目定位、不断推出合乎时代发展与社会潮流的新栏目，才能在发展的道路上不断向着更高的目标前进。在这一点上，湖南电台交通频道的专业策划组就做得很好。推出了周六版块的"都市休闲日"与周日版块的"都市音乐日"，节目一推出后，受到了听众喜欢和同行的一致赞赏。而该频道也在建台短短二年时间里一跃成为听众最喜欢的电台之一。由此可见，只有不断推陈出新，敢做时代第一人，才能在当代媒体的激烈竞争中稳操胜券，立于不败之地。

　　五、注重公益宣传、搞好有偿广告

　　公益事业是我们全社会都应该关注的事业。像社会环保问题、人口与社会发展问题、妇女与儿童权益保障问题等等。媒体对于此类公益事业的宣传是无偿的，所以有些媒体面对公益宣传任务时往往会找出一些冠冕堂皇的理由来推脱。其实公益宣传往往容易使得受众对媒体产生好的印象,进而能极大的提高自己的社会公信力和影响力。因为时值当代，人们对公益事业是越来越关注了。如果听众经常在广播里听到公益宣传，无形之间就会对这个无偿为公众做公益宣传的电台增加好感。可见，公益宣传不仅是其他媒体也是广播电台的责任，作为党和政府的宣传机关，我们的广播媒体应该负有不可推卸的责任。

　　有偿广告是媒体增加收入的一个重要途径。一般电台往往会安排专门的人员进行广告接收、设计和播报工作。但有一点容易被忽视，那就是广告的真实性问题。据报载，某些省份某些媒体出现了一些做虚假广告的事件，不论从社会宣传角度还是从社会价值取向角度来说这都是不应该发生的事情。虽然有些情况确实是媒体不知晓或是产品生产厂商有意欺骗，但是相关媒体的责任还是不容忽视的。湖南省会的几家电台都比较重视有偿广告的考核工作。每接收一个广告业务都要对相应产品做严格的调查和审核，遇上"名不符实"的产品一律拒绝接受广告业务，不仅维护了电台的声誉，更是对广大听众的负责，赢得了听众的赞赏与支持。

　　以上几点是笔者对湖南省会几家广播作一序列相关调查采访所总结出的湖南省会长沙广播电台宣传特色，希望能对有关想走特色化道路的广播电台起到一点微薄的作用，那么笔者便不胜欣慰了。
