爱就一个字

　　别问我为什么爱上你，在我不能自己的时刻；别问我为何爱上你，在我意乱情迷的时候。你的到来，唤醒了我前世的记忆，记起了你我相携走过的日子。在你出现的时候，我毫不犹豫地走向了你。虽然，我不能给你浪漫故事里的汽车；虽然，我不能给你灿烂美丽的钻戒，甚至，我连一块廉价的面包都不能送到你的嘴边。但是，你应该相信我，你真的应该相信我，因为，我早已把我一颗脆弱却真挚的心，连同自己完完全全送给了你。

　　你看得到我的真心，你听得见我的心声。在你的面前，我一改过去小孩似的幼稚，抛弃了远航的梦想，这一切，只因为我有了你；在我面前，你就像一个邻家小女孩，年轻可爱，永远都长不大。我们在一起的时候，我从没有跟你说过一句情话，一句都没有，但你应该相信：没有人像我这样深地爱过你，从来没有。

　　或许我应该给你很重的承诺，或许我该给你很多的甜言蜜语，或许我该用很多的话语告诉我有多么在乎你，或许我该用很多行动让你明白我有多爱你。可是我做不到，我真的无法做到，因为我不愿用语言亵渎你我的真情，因为在我们的爱情面前，任何语言和行动都无法把我对你的爱表达，因为所有的语言在你我的爱情面前都显得那么苍白与无力。所以，我只能紧紧把你搂在怀里，让你闻得到我的气息，让你听得见我的心跳。

　　你问我，如果真的有前世，你我是不是相知相惜的磐石与蒲团；你问我，如果来生可以选择，我是否还会和你相持相守着我们这段爱情，你问我，如果今生我们不能在同一年同一月同一天同一刻死去，我会选择先还是会选择后？我不知道是否真的有前生后世，但是我想告诉你：如果真的有前生，那么你我就是前世夫妻，恩爱如胶；如果真的有后世，那么我们注定还会像今生一般，穷苦而又快乐；如果我们不能同一刻死去，那么我愿意后你而去——我不愿把失去爱人的伤痛和无边的思念留给你。

　　如果有悲伤。就让我一个人伤感；如果有伤痛，就让我一个人独自承受；如果我们的爱情注定要破灭，那么就让你离我而去吧，我不愿把被人离弃的伤痛让你来承受。

　　你知道吗？爱你的我无法对你说出口，虽然我已经把你当自己生命中最重要的一部分；你知道吗？爱你的我无法对你说出口，虽然我已经把自己的一切都交给了你；你知道吗？爱你的我无法对你说出口，虽然爱就只有一个字。
