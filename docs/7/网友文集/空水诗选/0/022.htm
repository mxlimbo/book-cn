北郊垄上行

离开妈妈的时候
我投进田野
去唱
世上只有妈妈好

没妈的孩子
像根草
有草的田野
像妈妈

离开妈妈的孩子们
让我们投进田野
如果你看到田野的草
那就是我们的同胞

每当与一棵熟悉的草重逢
仿佛见到了久别的乡亲
每当与一棵陌生的草相识
仿佛走进了崭新的世界

我曾有一个
比在妈妈怀里撒娇
更幸福的奢想
奢想自己美丽的死亡
安息在
妈妈的怀里

离开妈妈的时候
我只想像草一样生活
最后
让我安息在
有草的原野

1991 05 16
