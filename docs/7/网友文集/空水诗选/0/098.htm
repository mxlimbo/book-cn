难忘恩师

当年我们像制造中的帆船
朦胧着远航的渴望
总是有四十五分钟的阵风吹过
令我陷入水手的梦想

当我惭愧自己的桅杆为什么总是摇晃
那高耸的灯塔
总会投来鼓励的目光
当我觉得黑板像夜空深邃而迷茫
那些粉笔字的星座
总会温暖地闪亮

常常觉得学校像寂寞的海港
常常憧憬去经历外面的风浪
常常怨恨锚链的羁绊
常常期待缆绳的解放

终于闯入茫茫的海洋
才觉得自己的桅杆还不够粗壮
才觉得那缆绳其实是脐带
曾经给我输送宝贵的营养

1991-1997
