序

　　余幼好古，喜欢收藏旧物，及长入学，习诵旧诗，品味研读，深受其泽，欣然忘我。遇有所感，以文字抒之，出则仿古诗之体例。今得余闲，忙此不急之务，拣拾旧作，连缀成集，计百余首。

　　又闻我等自幼所习诵者俗谓古诗，然则方家言诗分古体诗和近体诗，唐诗多为近体诗。窃以为唐宋距今逾千年，实不近也，更综观吾之涂鸦，中有非古诗更非近体，或为打油或较之打油稍雅，删之可惜，姑妄存之者多首，乃统称之旧体诗，以别于不计字数不讲押韵之新体（白话）诗。

　　是为序。

辛巳年三月十九日寅时于陆遥寺
