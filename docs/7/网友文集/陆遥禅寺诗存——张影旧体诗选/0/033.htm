儿女情长录

（对胎中宝宝的希冀）

曰强曰圣曰宝贵
真情真意真自由
大儒大亨大总统
是男是人是吾儿

　　小注：强即其本意身体强壮，圣取大仁大义品行高洁，富乃学富五车之富，贵取精神贵族之义。真情指七情六欲的率真，真意乃洞悉万物的豁达，自由乃应用自如于世之意。儒、亨、总统是对知识、钱物、权力的戏称。末句指做有血有肉的男人，又能做社会意义上的公民，归根底是继承我的血脉--历史人类一小环儿尔。

　　吾道刘氏女NN为拯救我书生小九哥之天使，又，人道儿童乃上帝派天使送与家庭的礼物。想我一介书生穷困若此之际，二美兼得而憧憬备至，遂有如上包罗万象之打油诗。不过，千般梦想还得从衣食住行一点一滴做起，遂购此育儿千答，权希冀"无灾无病到公卿"吧！
