《诗经2代》（洁本5）

　　四十二、课桌上的字

某大自古无媚娘，
残荷败柳排成行。
纵有几只野鸳鸯，
不是野鸡就色狼

　　疏：这是我飘进北师大的一间教师采写的。不过只要是教室，只要有课桌的地方，皆大同小异。全国山河一片红也。课桌上的诗，远远比北岛舒婷的更流行，远远比李白杜甫的更深远。至于那些当代瘪作家的破棉絮，就更无法相提并论了。中国课桌文学，是一种高级的文学形式。

　　四十三、芳草

天涯何处无芳草
何必定在X大找
量又不多
质又不好

　　疏：与流沙河的《明晨还得洗衣裳》也差不多呀！当然，还比不得《橘颂》。

　　四十四、Y校歌

XX师范，
阳光灿烂。
走进一看，
破破烂烂。
八个教授，
七个笨蛋。
还有一个，
神经错乱。

　　疏：哈！唱得好！唱得好！同感，同感。

　　四十五、横眉

横眉冷对秋波，
俯首干为光棍。

　　疏：这首短句，流传了二十几年了哟。我读书的时候就流行，今天还流行。看来，这作品的生命力还挺强。而今还有人读琼瑶没有？还有人读"雪米莉"没有？鲁迅笑也。

　　四十六、

老婆是个人所得"税"；
情人是调节"税"；
嫖娼卖淫是"偷"税"漏"税"。

　　疏：卖淫虽是无烟工业，但是不上税也不好。但是上也不好。我有一年去广汉，我的妈呀！满城"尽戴黄金钾，充天香气透广汉"！这还不算啥，X县长大人与我同桌吃饭，他的一番指导性发言，把我从椅子上吓落下来，象鱼一样扑在地上。县长说"牺牲两代妇女，换取广汉繁荣！"我的个妈妈呀！这是，这是……什么玩艺儿呢这是！哎！

　　四十七、考查干部的若干标准

打麻将三天五天不累，
喝茅台三瓶五瓶不醉，
下舞池三夜五夜不睡。
给新领导捶背，
对下属横眉冷对，
吹牛不会上税，
卡拉OK一学就会，
常到舞厅去找小妹，
表决心学着行跪，
基本功弯腰哈媚，
常开宴会，
巧善行贿。

　　疏：不敢疏了。呀，你别嚷嚷，你不会推我下岩吧？疏下去不是犯错误吗。

　　四十八、三级干部

省里的忙着出国，
县里的忙着吃喝，
乡里的忙着赌博。

　　疏：我坚决不疏了。我赶牛犁地去都可以。坚决不疏。反正这话不是我说的。我是在重庆市江北江静观镇水土乡张二娃茶馆里采集来的。是张二娃叫"二嫂"的人给我说的。我就记在本本上了。看上去那"二嫂"也有六十五岁的样子了。

　　四十九、头儿的的生日

　　头儿甲与头儿乙是好朋友，某日，他俩聚在一起。

　　头儿乙见头儿甲神情沮丧，便询问发生何事。

　　头儿甲叹气道："昨天是我生日，我的女秘书请我去她家给我庆祝生日。"

　　"那不是很好吗？"

　　"到了她家，她让我在客厅先等一会儿，五分钟后进卧室找她。说要给我一个惊喜。"

　　"那不是更好吗？生日交上桃花远了。"

　　"我当时也是这么想的。可五分钟后我走进卧室，发现我的女秘书和其他职员都在里面，捧着生日蛋糕等着我呢。"

　　"这也不错呀，你的职员都很爱戴你，你应该高兴才是。"

　　"可当时我是脱光了衣服之后才进去的。"

　　疏：这一段不像诗。这是一个新体裁，是中国人民近年来创造的，叫"段子"。主要发源地在重庆和成都，很快就普及了全国。"段子"引一个在这里，只是抛砖引玉。而人民的"段子"太多。我估计要一千个四库全书才收得完。如果近年我不死，我就准备注疏个全本出来。

　　五十、长恨歌　　

下岗妹，别流泪，挺胸走进夜总会；
陪大款，挣小费，不给国家添累赘；
爹和妈，半生苦，老来待业很凄楚；
弱女子，当自强，开发身体养爹娘。

做美容，隆丰胸，中外功夫都学通；
练内功，学口技，风情气质巧相配；
跳探戈，走四步，各种喜好要对路；
会矜持，巧放纵，把握时机才让弄；
多撒娇，少贫嘴，揪准口味要油水；
很舒服，也劳累，拉动内需创外汇。

他爱好，我奉陪，我和首长谁是谁？
你敢讲∶风气败，领导都夸我可爱！
搞开放，百业举，要比奉献我彻底；
排隐患，除干扰，四大恶人都放倒；

香风薰，美酒醉，红唇轻启罗裙退；
骨头酥，意志颓，谁都为我当保卫；
选靠山，定要牢，搞定书记第一条。

活思想，多宣传，无私奉献数欢场；
一不偷，二不抢，一切献给共产党；
无资金，无贷款，自带设备搞生产。
不占地，不建房，工作只要一张床；
无噪音，无污染，紧要关头小声喊；
不添女，不生男，不给国家添麻烦；
活经济，增影响，积累资金求发展。

态度灵，技术精，挣钱方式要扪清；
你舒服，我高兴，都要满意才稳定。
我规矩，你谨慎，让你老婆当陪衬；
你出钱，我奉献，都是为党作贡献。
我三陪，你三讲，都是紧跟咱们党；
你多讲，我多陪，咱两还分谁是谁；
你弄虚，我实干，保证领导回头看。

你买单，他打炮，帮你奉承大领导；
你当官，我发财，若想温柔你也来。
趁年轻，多积累，莫学前辈常后悔；
不唯上，只唯实，首长教导我合适；
大实话，我敢讲，我也够格做党员。

　　疏：《长恨歌》有三，白居一为正版，这一个无名氏所作为副版，还有一个王安忆写的，那只能算盗版了。嗯嗯呀呀的自恋，那能跟这正副版相比呀。
