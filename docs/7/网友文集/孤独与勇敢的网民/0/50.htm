元曲的升级版

　　元曲也太古了，有三四百年没升过级了。我试着给它升了一次级。这可是核心技术，我这人从来不搞那些小玩意儿，小打小闹的，像那些中国软件，要搞就搞核心技术。所以这个版本有看头，基本上让元代的人文文明，与互联网接轨了，成为一个人文技术的平台。如lunix一样，这是个开放了源代码的版本，共享版本，注册费都不要。用之玩之，不亦乐夫。

　　见于有些用户顽固守成，同时也便于喜新厌旧的用户在线升级，原版保持本来面目，同时发放出来。新的算法，跟在那又老又破的老代码后。噫吁嚱！

　　南吕．四块玉．别情

　　老版源代码：自送别，心难舍，一点相思几时绝，凭栏袖佛扬花雪。溪又斜，山又遮，人去也。

　　升级版源代码：你要跑，你跑好，网上MM如牛毛，上得网去马上泡。谁怕嗷，小莱鸟，886。 （帮助文件：what's new?新版丢弃了老版的别情方式。if相思,than上网。凭栏袖佛扬花雪那种凄清孤冷情调，过余绝望和飘眇。干嘛站在纷扬的雪花中，靠在冰冻的栏杆上望啊。到网罢去，一个大虾，聊个MM，不过半个小时，亦或三言两语耳。新恋爱的到来，不小菜一碟欤。老版关汉卿参与编写，是封闭式的，老关被送到郊外，关在一间木屋里，与外界隔绝一月又余。但今观这代码，不免哈哈大笑。那有哪么傻的恋爱哦。这次作曲为琼瑶，演唱为老燕子，拉二胡的为流德花，灯光为中中电视台，赞助商为天极网站。）

　　商调．梧叶儿．别情

　　老版源代码：别离易，相见难，何处锁雕鞍。春将去，人未还，这其间殃及煞愁眉泪眼。

　　升级版源代码：谈也好，吹也罢，芳草黑压压。一把抓，去网吧，这破屋话情话如雷炸。 （帮助文件：马自然是没有了，又岂有雕鞍在？今儿个只有汽车、自行车、人拉车和脚板，按钱权等级排列组合，如婆罗门教。这离别，哪有"难"字可说，要走就说声886，那还道一声珍重，还沙扬娜娜！撑把油纸伞，像结了愁怨的丁香，走在小巷，三步一回头，五步一颦首。这时代已经过去也，现在流行的是"数字"风尚了，再见改为"886"。上网找个女朋友，容易到像吐块口痰！所以那"难、愁眉、泪眼"，都变了风景，可谓笑遂颜开也。）

　　双调．寿阳曲

　　老版源代码：夕阳下，酒旆闲，两三航末曾着岸。落花水香茅舍晚，断桥头卖鱼人散。

　　升级版源代码：罗马远，也不远，网址一输就看见。昏逛三月不见天，还在开罗边边。

　　（帮助文件：在搜索引擎输入"罗马"，关于罗马的"类"110个，网站3000个，网页1亿5千个。"地球村"里，夕阳、酒旆、航船、茅舍、断桥头上的卖鱼人，也都有的。什么风景都有。只是夕阳不照君脸庞，显示屏亮光射君眼球。夕阳照在写字楼的楼顶。你在这楼里上网。）

　　中吕．阳春曲．题情

　　老版源代码：笑将红袖遮银烛，不放才郎夜看书。相喂相抱取欢娱，止不过迭应举，及第侍如何？

　　升级版源代码：红袖好比抹桌布，酷虫通夜只看书。为了一段烂代码，岂有美女可挡路。 （帮助文件：作曲：滚石乐队，演唱：扬玉莹。剧场：网易社区大剧院。票价：１元。用招行卡支付。）

　　中吕．阳春曲．题情

　　老版源代码：轻拈斑管书心事，细折银笺写恨词，可怜不惯害相思。则被你个肯字儿，迤逗我许多时。

　　升级版源代码：国家儿有新令，斑竹儿你记清。那BBS要管严，别让那阶级敌人把空钻。要是有人乱写了，那你就惨。

　　中吕．阳春曲．知几

　　老版源代码：知荣知辱牢缄口，谁是谁非暗点头，诗书丛里淹留。闲袖手，贫煞也风流。

　　升级版源代码：有起有落方网站，赚不赚钱靠概念。去美国找靠山，发证券，圈钱一牛圈。

　　（帮助文件：这个老版即1.0版是白朴写的。虽也精典，但太古朴。界面不华丽，思想按钮比dos时代还远。诗书今也成了e书，贫煞是万万不能，点头开口的行为，也已更为鼠标加眼球。故作以上修订。）

　　越调．天净沙．秋思

　　老版源代码：枯腾老树昏鸦，小桥流水人家，古道西风瘦马。夕阳西下，断肠人在天涯。

　　升级版源代码：上传下载解压，灌水菜虫大虾，注册换肤密码，夕阳西下，断肠人在网罢。 （帮助文件：一个城市青少年，无论如何，也不可能下午坐在小桥流水人家的生存处境里。除非他爸爸是李嘉诚蔡万霖什么的。给他买了三百亩地的别墅。乖乖呀！还有乌鸦呀！还有手粗的枯腾缠在千年老树上呀！这是何地方？是电影制的假基地，还是神农架的原始森林？今天有的只是灰尘扬起的马路，高搂导弹酒店立交桥。夕阳下，网虫们在随便一间破房子里，手忙脚乱地敲健盘。）

　　双调．蟾宫曲．叹世

　　老版源代码：咸阳百二山河。两字功名，几阵干戈。项废东吴，刘兴西蜀，梦说南轲。韩信功兀的般证果，蒯通言那里是风魔。成也萧何，败也萧何。醉了由他。

　　升级版源代码：人生如此短暂，二两老酒，一段恋爱。谁修长城，谁筑网易，于我何干？毛主席飚扬了一生，七分成绩三分错误。一日三餐，一觉一枕，暴户如何？

　　（帮助文件：演唱：小崔健。老板是马致远程序员写的。引用了历史数据库。bug也太明显了。新知识曰：资本主义，不言功名，只言圈地；帝国主义，不言获利，只言上市。新版修正了bug）

　　关于升级版：有人说，这越升级越差了。甚至差得不像话了。比元代时期的老本还差远了。这词越谱也没谱，这曲越唱越令用户受辱。话由人说去。这只是个开放平台，谁都可以修改源代码。至于元代版本和今天版本的比较，孰是孰非，且有他人说去。
