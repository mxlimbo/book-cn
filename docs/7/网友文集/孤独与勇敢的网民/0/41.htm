文章的"架子"大矣

　　我读过伍立扬先生的一篇文章：《语文忧思小札》，是批评时下语文的堕落。扬先生说，现在的汉语，被商业文本、译文文本给弄得一蹋糊涂。汉语的情意郁积、言简意赅、跌宕音韵，被糟蹋得令人心痛。扬先生说，今日的学界、著述界那种冒似精确的绕弯子语句结构，不过是缠小脚的裹脚布，缠了一层又一层、绕了一圈又一圈，其是还是个裹脚步。

　　想起来扬先生说得还真是有理。前段时间翻《读书》杂志，被一篇文章搅得很糊涂。不是那文章所涵真意,幽微缈远，而是那种"流行"的,化简为繁的句式和作者千方百计"做出"的新辞，令人堕入云雾之中。

　　是不是这样才能表现出"有文化"呢？很知识，很专业，很层次呢。其实这篇文章要表达的意思简单，几句大实话就说清楚了。比那浩浩万言书，还要一语中的，还要透彻。倒是作文的人绕来绕去，把读者绕糊涂了，反而不明白他倒要说什么了。

　　我的一个朋友说，这叫"学人架子"。没架子，用易让人瞧不起。原来文章"架子"，比文章本义还要重要哦。也就是本义可以有，也可以无，可以让其糊涂化，但不可以让其清晰化。这是"架子"，是一种"入流"的面子。试问这样的"入流"是不是可以称为文化垃圾呢？

　　文化新锐余杰，在《说，还是不说》一书中指涉过文个"文化现象"。不过没在这方面作深论。多是一语带过，像说气话。比如他评批《读书》杂志的"绕口令"文风，他说"与其读《读书》，不如不读书"。这种话可能是过激了点。但震耳发聩,有时候还是需要的。况乎今日，文风畸变糜靡，"架子"长袖当舞。世风下堕，真假莫辨。反倒只有说点气话，方解心中郁闷难平耶。

　　吸收西方优良的文化当然是好的。就怕画虎不成反类犬，西方的没学到手，把老祖宗的也丢了。做些这也不像那也不像的文章出来。扬贵妃长了高鼻子，海伦端杯白酒问青天。这样的吸收，文章就成了活脱脱的怪物。一个时代里，做文章的人，多少要为这个时代的文明形态负点责任的。

　　我们可能认为纯粹的文艺理论家，行文总是幽暗深遂的，外间人想读也读不懂。我不知道罗兰．巴特尔，福柯够不够"文化档次"。但我知道，他们的作品在法国当时就是畅销书。天体物理学家霍金，解释如此幽玄的宇宙，也是人人都能懂的畅销书。但观我们的文章，常喜习于西语者，开口闭口福柯、巴特尔。但文章读来，怎么看上去，也是在绕弯子。绕了十八公里，原来他只想从街的东头走到西头啊！

　　这样的"架子"语句你是不是也看得糊涂呢：

　　"除了屈从于宗教与社会关系意义上的'父'的膝下外，还受制于另一种经过软性处理或女性打扮的权力，而这种权力又巧妙地与'父'结合为一种'血缘'关系进入家庭或者与家庭制度的'正常'操作共谋，'自然'地发挥规训作用。"（《读书》2000年9月号第61页）

　　这说的是什么呢？通读全文，才知道，这一段作者想说的是：

　　"被迫屈从于宗教思想、父权、家庭观念。"

　　至于作者冷不丁跳出来个'宗教'，我看了全文，不知是从哪里钻出来的。"另一种经过软性处理"的权力，我更不知是从他这篇文章中哪里跳出来的。"女性打扮的权力"你听说过么？"女权"我倒是知道。可这"女性打扮的权力"作者浩浩万言的文章上下文中又末置一词，是不是个文化背境语词，比如生物进化或者天净沙一类呢。不知天下"愚昧无知"的读者哪位知道"女性打扮的权力"的文化渊源？。与'父'结合为一种'血缘'关系，家庭制度的正常操作共谋，自然地发挥规训作用。这样的句子，听起来远比中华"古文"要信屈聱牙得多了。真是中国人不懂，外国人更不懂。古人不懂，今人也不懂。作者自己更不懂。

　　这不过是个小例而已。当然不仅仅是这一段喽，这篇文章一万多字，都是这样写的。这本书也差不多都这样写。称为风格，我称之为"架子"。一篇文章，或是一本书，当然不足以论。但如果是一阵风了。弥漫整个国家了。为时已久了。那写此小文一篇，还是值得的。尽管杯水车薪。
