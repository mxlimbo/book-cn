男人有钱就变坏了

　　我现在越来越想念马克思了。他是多么具有预见性的天才啊！他就说过：

　　"钱"是个什么东西呢？"资本来到世上，从头到脚的每一个毛孔里都充满了肮脏。"如果你连马克思认识世界的智慧也不相信，你也就没必要看我这篇破文章了哟。事实上你任何文章都不必读了。反正理由是跟你讲不清楚的。

　　马克思深恶的东西抖擞着翅膀飞到我们的生活中来了。我听过一句出现频率很高的话："男人没有一个好东西！"。这句话是近几年才出现的。近几年社会大浪淘沙，淘出了"每个毛孔都充满了肮脏"的"成功人士"。

　　这不是什么冷嘲热讽，更别以为我是吃不到葡萄说葡萄酸，我也是你们的同类，级别还很高。我写出来，不过是像卢梭写点《忏悔录》。伟大领袖毛主席曾经在他的著作里一针见血地指出"人是环境的产物"。一个人在钱堆里混，这不就等于在垃圾堆里出入吗？又怎么能够不染污秽呢？这就是女人们挂在口边的话了：

　　"男人有钱就变坏"。

　　没有钱的时候，他可能信誓旦旦：假如我有了那么多钱，我要如何如何。可是一旦他有了钱，早已把那信誓旦旦当成了嘲笑当年的幼稚思想。人就是这样宿命的，没有一个可以例外。想像中一个人可以站在长江的洪流中依天而立成一柱坚定的标杆。不过人的肉体和灵魂是如此软弱：肌肉的承受力远不如一块钢锭，灵魂的承爱力远不如一块无生命的木板。待入了长江，怎么也免不了被卷入洪涛顺水下流。一个人步入了有钱的洪流里人，也大概就这等景象。又要用到马克思的话了，可钱是"每个毛孔都充满了肮脏的东西。

　　我还是说具体点吧：与无产阶级男人相比，有钱的男人会出现以下的特征，有综合症患者，也有少数单一病况者。

　　一、 有钱男人趾高气扬。竟管表面上儒雅歉和得象个大德人，素质再差的干脆就直接趾高气扬一番。这两种估计你也见多了。实际上你可以把这条终极理论套到每一个"成功人士"的身上，放之国内，皆准。

　　二、 有钱男人爱情上心怀鬼胎。你别指望他非常赞同一夫一妻制的爱情模式，别指望他用情专一，别指望他不花心花肠。如果你的智力没出什么大的毛病的话。世界上有钱的楷模，一般不是跟在一个影星的屁股后用钱作核武器穷追不舍；就是对妻子撒谎说去美国做业务，其实他正躺在豪华游轮的甲板上在地中敖游，左右两侧的十三个仪态万千的穿三角裤或根本就没穿什么的女子正在给他捶背。

　　三、 有钱男人不知天命。有人以为在有钱男人族中也能找到"达观、安静的知命者"。这完全是做梦。"钱使人疯 "早已是一句英国谚语。变疯的"成功人士"早已视自己如冀土，更何况什么亲情、友谊、道德一类东西。这道理还不简单么。既为钱生，便为钱死。举一个例吧，胡长清是副省长，因贪污被抢毙了。一个女儿才四岁。哎！不举了，反正你也知道满地皆是这种事儿。视财如命不是说某个少数的人物，而是一种类：有钱男人类。越有钱越视财如命巴尔扎克也描述过了，马克思也论证过了。总而言之呢，如果你是吴国之女性，你要选择了一个有钱人做丈夫，那么你就要作好思想准备：某一天他可能会卖掉你。你莫要大惊小怪哟！怎么可能？怎么可能？在一派清秀儒雅的气质下、在西装笔挺的身体里，怎么生出这样一堆蛆呢！读了这篇文章你就可以防微杜渐了。

　　四、 有钱男人吃喝嫖赌。他会说，"哎！没办法呀！陪客户去洗桑拿浴啊！陪上司喝酒啊！陪客户打牌啊！人在江胡，身不由已啊！不过我没洗桑拿，只是睡在外面的长椅上一边构思一首诗一边等他们。"

　　如果你信男人们的这种信口捡来的谎话，我就怀疑你只是八岁的扎着羊角辨儿的小姑娘，要不你就是被你"丑恶"的爱情欲望（阿弥托佛！）冲昏了头的傻瓜，大傻瓜！我就有个朋友。他长期让我帮他给他在电视台当主持的名星老婆撒谎，说他正在北京忙着呢，过两天再给她打电话。你猜他干什么去了。呵！这家伙一飞机坐到非洲去找当地黑人嫖娼去了！别以为这事太典型，它只是一件很普通的事儿。只不过钱少的去了澳门；再少的去了本城的红灯区；再少的去了背街的发廊，谎称说自已的头发深了，好久都没理了。

　　什么叫"人在江湖"呢。这倒也是真话。口袋里有了钱，就上了贼船。还想洗手不干啊？没那事儿。船都到江中心了。一切都晚了。人是不可以再生二次的，不可以脱胎换骨的。不然马克思就不会组张革"命"了。

　　五、 有钱男人失人性。上四条说的也是人性，不过偏了一些。像常规的"亲情、古道热肠、男子汉精神、同情、诚实、总厚、心胸宽广"一类品质，你如果能在此类男人身上发现，那这男子就不仅是个有钱人，还是个合格的演员。这是最危险的一个小分队，阴险毒辣，十足伪善。不是洞中毒蛇，就是深涧豺狼。他要是破了产，第一个想到的念头就是把你卖了。你还在叫他"亲爱的，过来看电视呀"，他一边数钱一边答应你"哦，我就来"。

　　当然你心甘情愿被折磨，不是这篇我章要说的。我下一篇来给你讲些动听过事吧。

　　别看他在四处捐款给贫困儿童、失学孩子。他年迈的父亲一直想把这狗杂种拒付每月七拾圆的詹养费把他告上法庭呢。

　　世界就是这样子，顾此必失彼，正是古典的鱼和熊掌哦。无论世界怎么变，永远逃不出简单的规律。兹以此小文献给那些正在做梦的"很有追求"的正在成长中的中国少女们。当然，如果你打定主义不入俗流，要求自己和梦中情人"做一个高尚的人，一个脱离了低级趣味的人"，你并不包含在此文的叙述中。权当一篇批判现主义的参考资料来读，以提醒你千万别搞忘了。
