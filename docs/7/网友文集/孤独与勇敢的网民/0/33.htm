上帝的程序一

　　上帝其实也没有干别的，整天无所事事，有时侯熬熬夜编编程。就把我们弄得神神鬼鬼的了，不知究里，不知这人百态是何以故。懂得查看源代码的人可有福了。可以略窥一下上帝这个高手的绰约风姿。当然，应用是简单的，写代码也没什么难，说难那是蒙你的。蒙人最厉害的，就数他老人家了。普天之下第一名。

　　上帝高兴的时侯，顺手写下以下代码：

　　if 他是男人

　　then 让他终生想女人

　　else 让他当和尚

　　这人这动物就这样搞定了。这是菜单第二项，子莱单分别有"结婚"、"少年犯罪"、"弗罗依德这混蛋"、"基督教功能和马尾巴功能"、"浪漫"、"独身"、"十八岁以下不得入内"、"受滋病"、"犯罪档案"、"情痴"、"情杀"、"奸夫"、"西门庄与贾宝主"、"薜涛及张曼玉"、"明星崇拜"、"下贱狗"等等不一而举。每个子莱单实现的功能各守其职。比如"明星崇拜"吧，鼠标左健一单击，弹也个对话框问你"你需要崇拜吗？确认？取消？看看帮助？"如果你不懂，先看看帮助，帮助的弹出对话框就出来了，从你的C盘到E盘，五肺六府都显示出来了，有你的脑灰质，阑尾、桃型的心、视网膜、骨架，等等不一而举。帮助说，"如果你对某异性明星崇拜，击健后会指向你的分泌系统的X部位，可以查看其下贱的函数列表。如果你经常苦恼，呼天抢地的叫喊，这显示你内存过大，请点击Edite莱单的cut项。

　　上帝菜单的第一项是"我的制造"，正如你可以想到的那样，下拉莱单地一项是"开天劈地"。点这个健，弹出一个标签，标 签上说"老子说有了光就有了光！"。非常霸道，像恶霸地主，像美国打南斯拉夹，像中国电信，像萨这姆，像秦始皇。你不能追问"为什么"。标签上有些选项。哎！不说这个乱来的豪无道理的选项了。还是用二位制编辑器看一段源代码吧：
