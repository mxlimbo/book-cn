"我"的市场价格及使用说明书

　　标签价：２块３角１（全国统一零售价）

　　产品简介：本产品系著名的"我"牌，始创于本世纪六十年代早期，发明人系著名科学家亚当，在"中国社会工厂"里，该产品经过三十余年的高温、高冷、度压煅造，又经不可胜数的流水线程序，反复车、刨、冼、磨、表面抛光、电镀等，其品质达到了国际认证标准，符合甚至超过了ＩＳ０９０００。

　　使用说明书：本品由肉、骨、血浆构成主要部件，用户在装机时随附装机单，请仔细检查细检查有否缺漏。肉，包含本品的皮肤、肌肉和脂肪层；骨，附着在肉内的脊椎动物的坚硬部份，起着本品的支撑作用，又可细分为骨孔、骨腔、尺骨、肋骨、胸骨、颌骨等细小零配件，这是本品的躯干；血浆，系本品内部循环系统的不透明液体，大多为红色，由血细胞和血小板组装而成。

　　硬件方面，还配备了脑、胸、腰、肩、背、屁股、手、食管、胆囊、腿、胄、肺、肝、心脏、气管、直肠、盲肠等目前的前沿科技产品。脑由１百亿个细胞组成，其储存容量仅次于联想１２Ｇ硬盘，但ＣＰＵ处理器、蕊片都比硅谷出Ｉｎ＋ｅＬ奔三要好，反应速度敏捷，肩可以承载１００公斤重物，在搬运各类杂物时，可大显用埸，仅次于骆鸵；手，颇有腕力，在公共关系方面，在看不见的战线里的运筹帷握方面，在算补他人、制造陷井方面，有其显著的优良的特性；腿，主要用于跑步，２７楼的的楼顶、下水道、矿井、地狱、街头巷尾，无论远近，都可以随利抵达，些地方大多是波音７５７不能到达的地方。另外，本腿还善于登大雅之堂那种非同一般的高贵的地方，以便雅足入高室、投机取巧；背，也有其很大的用途，如果用户一不小心搞了金融诈骗或者行贿行赂，或者其它违非好法作歹的事（这类事情的发生机率己高得让人咋舌），它可以替你背黑锅；食管、胃、肠皆设计精良，保证了最小口径，设计师是曾经设计来辐小口径步枪的资产阶级．葛朗苔先生，省"电"省"油"，保证了最低能耗（每天约大米一斤、藤藤莱７．５俩）；肺、肝、心这些附件，在制作工艺的流程中，使用了妨生技术，参照了岩石形成原理、鹰犬情感神经结构、狼、狗、猫等精神汾泌物，所以使用起来绝对安全可靠，对外工作有能力，对内绝对服从主人，绝不会产生各类形状的七情六欲的"病毒"。随机附送的"开除"软盘，用户可在关健时刻使用，以彻底消除"病毒"隐患；气管设计方面，本品使用了从蛇身上的提取物，保证耐磨、耐打、耐乱整的坚强的品质，解除了普遍用户的后顾之忧。

　　软件方面，配备了"公元前三世纪道德品质鼎"，一切古道热肠，尽在其中；配备了"汉语千年"，只要是汉语曾经叙述过的，都能查阅，相当于百科全书；配备了"西半球空间"，用户可以了解西半球发过这的一切；配备了"长袍马褂爱迪生"，充满了创造发明的功能；配备了"诸阁亮思维库２０００"，可替用户想神出鬼没的点子，也包含一些烂点子；配备了"卡拉ＯＫ一类特殊服务"，替你唱歌跳舞、走台步表演时装，准让你乐开怀。假如说，你是一个高尚的人，一个脱离了低级趣味的人。你真是会乐开怀呢。

　　在显示屏方面，其界面设置，亦令人欣喜，眉似剑，眼似电，濞如弯弓，一派清秀和玲珑惕透。使用起来，有爽心悦目之愉快。

　　本品外形不算太规则，这是唯一触缺陷（但用户可以免费升级换代下一代纯正方形产品，并不在补差价）。本品长１７６缠米，宽７８缠米，高１９缠米，净重８１公斤，合符产品检验标准。

　　本品无需防潮，大不了得关节炎，但不会影响使用效果，长期置放于地下室内亦无所谓。

　　本品无需防震也无需防整，只要不从三楼上往下扔，一般没有关系，摔不着它那点儿。本品无需防静电，但切勿把照明电路和高压电以及瞬间放电的电警棍朝它接，这样可能会造成"死机"。本品有效使用期限为35年，在此期间，用户绝对是上帝，不满意，退了就是，无论什么时候，什么情况，用户一旦"说不"，就可以退，让它滚蛋回家就是了。
