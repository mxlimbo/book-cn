上帝的程序二（函数释义）

　　分别为：名称 类型 说明

　　abort 函数 引起放弃的意外处理。放弃吧，这个世界。到处都是不公乎和荒诞。上帝也没办法。它们人娄悲伤的放弃中忙得整天顾不上吃早餐。

　　abs 属性 绝对值函数。一切都不可改变。天和地。贫穷与富裕。暴君与绵羊。流氓与弱民。疾病与木马。眼泪与病床。死亡与过生。冥冥中的绝对。认了吧。一切都是妄想，妄为。

　　addexitproc 函数 将一过程添加到运行时库的结束过程表中。比如死亡时的梦想和生存时的欲望列表。很详细。from head to tail。

　　addr 函数 返回指定对象的地址。只有一个，那是墓地。

　　align 属性 使控件位于窗口某部分。决定你生们哪个方位，哪家。是马利桑娜还是四川。是茅屋还是金銮殿。

　　alignment 属性 控件标签的文字位置。决定你的身体。是均匀分布呢还是东拉西扯。是平衡呢还是一这倒。样板软件好的有英格丽．褒曼；病毒软件有"钟楼怪人"。

　　allocmem 函数 在堆栈上分配给定大小的块。上帝的"打土豪，分田地"。

　　allowgrayed 属性 允许一个灰度选择。颜色人生，还是灰暗无光的人生，在这里决定。不过别想走后门。上帝不买那帐。

　　ansicomparestr 函数 比较字符串（区分大小写）。这是竟争。帝国主义垄断下的竟争及社会主义地区的单位，比如中国电信就用大写字母。比尔．盖茨由大写变小写了。

　　ansicomparetext 函数 比较字符串（不区分大小写）。地球上的某些部份，话它们更多的自由与平等！

　　ansilowercase 函数 将字符转换为小写。恶有恶报及恶有善报。

　　ansiuppercase 函数 将字符转换为大写。善有善报及善有恶报。

　　append 函数 以附加的方式打开已有的文件。出身。家产。关系。国外的亲戚等便是也。

　　assignfile 函数 给文件变量赋一外部文件名。比如"小资情调"、"右派"、"知识分子"、"约翰牛"、"老太监"、"刀客"、"妓女"等等。

　　assigned 函数 测试函数或过程变量是否为空。小偷式的侦探。看看XX这两年趁混水摸鱼，摸得如何？地窖里如外国银行私藏了多少美金。

　　autosize 属性 自动控制标签的大小。签大的叫大官，大老板。这是个中央控制结构。。标

　　backgrounddi2001.jpg 属性 背景色。父母及直亲亲属。及祖父母。及社会关系。学历。单位。等综合个人背境。

　　beginthread 函数 以适当的方式建立用于内存管理的线程。你的记忆力强弱在此被控。。

　　bevelinner 属性 控件方框的内框方式。你住什么房子。

　　bevelouter 属性 控件方框的外框方式 。你穿什品味的衣服。

　　bevelwidth 属性 控件方框的外框宽度 。你在某个区相比如国家的名气的大小。

　　blockread 函数 读一个或多个记录到变量中 。你的联想能力在此被控。

　　blockwrite 函数 从变量中写一个或多个记录 。你的死记硬背的功夫。

　　borderstyle 属性 边界类型 。你做什么工作，是卖冰糕？挖地？当小偷？当干部？搞剥削？当褓姆？坐牢？等等。

　　borderwidth 属性 边界宽度 。你的活动范围。是一个村呢？还是常坐飞机去澳门睹搏。

　　break 命令 终止for、while、repeat循环语句。控制事和人，叫他"住手"或投降，或屈从。

　　brush 属性 画刷。你就全靠舞动这玩艺儿了。在这个世界上，你操不操得转，就全看这一"刷"子了。如果你能舞它个每千分之一妙0.8转，那恭喜你，你可以上皇家电视台春节联欢晚会上唱歌了。并且发了！
