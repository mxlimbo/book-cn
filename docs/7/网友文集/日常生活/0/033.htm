爱断情伤

　　CD里放着蔡琴的碟，一首《爱断情伤》听得我泪流满面。

　　"天色已晚，开口之前泪光已在眼里旋转，你无波的心情比我的泪还冰凉，而再三思量，离开你又能怎样，想走却没了方向，迷乱在狂想的路上……"歌让我想起了那位杨小姐，不知她现在身处何方，一切可好？

　　杨小姐，一位陌生的女性，一天打来电话，问能否谈谈？声音里抑制着哭泣。我赶紧说，当然可以了。

　　没多长时间，一位相貌俊丽、穿戴入时的女性飘然来到我眼前。是杨小姐。

　　没开口，先哭起来。大大的眼睛溢满了忧伤。她说，自己爱了一个50多岁的有家有口的男人，不知该怎么办？

　　跟所有婚外恋的故事一样，没有任何特殊或独到的地方。听这样的故事，我会麻木。但当事人杨小姐，却觉得自己是天下第一不幸的人。

　　毕业于华中某大学的杨小姐，有学识有教养有风度有相貌。前期爱情路上虽坎坷，却也游刃有余，进退自如。先后谈过两任男友，都无疾而终，没有留下太多的伤心后遗症。

　　事情坏就坏在一年前的一次朋友聚会上。席间一位谈笑风生的中年男士，风度优雅，谈吐极佳。当目光相对时，两人都有了触电的感觉。第二天，男士打电话给杨小姐，约吃饭。杨小姐一点都不诧异，欣然答应。她说，自己其实一天都在等这个电话，而且直觉对方会主动约会。

　　他们一起吃饭，聊天，一切都很投缘。好像是前世就认识的朋友，彼此熟悉、自然、亲切、愉快。

　　后来，他们经常一起吃饭、看电影、打球、听音乐、泡茶馆……

　　杨小姐说，我一开始就知道他是有妇之夫，是有子之父。但我不想问，也不想让他有什么改变。

　　两人坠入了情网，一切来得自然、正常，水到渠成。

　　杨小姐什么都不问。男人很细心。会找来各种时间陪她。不在一起的时候，他们会留纸条、打电话、发信息。节假日，他们会互赠礼物，亲手做卡片，写上浓浓的甜言蜜语。

　　就这样，过了中秋，迎来国庆；过了国庆，迎来元旦；过了元旦，迎来春节；过了春节，迎来元宵；过了元宵，迎来五一……杨小姐爱着、快乐着、感动着。慢慢地，也遗憾着、苦恼着，她开始不超脱了，开始无法做到逢年过节别人团聚自己独守空房了。她知道男人真心爱她，但不会为爱砸碎自己的生活。毕竟，身分、名誉、地位、家庭等等，更多的因素决定了男人是一个社会人，而不单单是一个爱人。

　　男人是真的爱她，平时会编各种借口离开老婆，跟她在一起，照顾她的情绪，逗她开心，细心呵护她。春节年饭，男人匆匆回家扒了几口饭，就跑来与她相聚，一直守岁到夜里两点。每次杨小姐无法忍受提出分手时，男人会夜不能寐，大把大把地掉头发，会到处打听杨小姐的行踪，恳求她回头……

　　两难中的杨小姐说自己想走又离不开，不走又死路一条，实在不知该怎么办。无奈之中，只能来找我。

　　我也不知该说什么。从传统意义上讲，杨小姐是第三者。但站在她的立场上，她也是受害者。情感问题，真的没有对或错，一定要有答案，只能是道德和不道德。然而，道德标准又怎能约束真情实感呢!

　　我能做的，只是尽到一个听者的本分。我建议她，暂时离开这个伤心地，出去旅游，也许，空间和时间是解决问题更好的途径。杨小姐倾诉完，抹着红红的眼睛走了。后来，文章见报后，她打来电话，表示感谢。我问，好点了吗？她那边有点犹豫，说，还行。后来，就没了音信。

　　今天，听着蔡琴的歌，我真的希望，她在想走的时候能有方向，不会再迷失在狂想的路上。
