时光飞逝

　　每天早晨醒来，是心情最灰色的时候，可以说近乎绝望。

　　因为想起，昨天一天又过去了，浑浑噩噩的。也就是这一刹那，我特别地恨自己，也特别地觉得对不起自己：为什么要这样浪费光阴？

　　每天，上班，做必须做的事情，应付必须应付的杂务，开必须开的会，说必须讲的话，露必须有的笑 脸，接必须听的电话，读必须看的信，挨也许免不了的老总的训，还有，就是不饿也必须按点吃饭……忙忙碌碌，疲于奔命。然后，在夜色蒙蒙的时候，挤上挤得不得了的公车，一路晃里晃当地回家。

　　累，从身体到心里，无孔不入。洗完澡，打开电视，看看新闻，看看天气预报，然后，再看看新来的杂志。

　　只等到坐在电脑桌前，累得半死的躯体才稍微有了一点活气。趴在桌前，在网上爬，一直到凌晨一两点，最后实在盯不住了，才转转酸得发硬的脖子，站起身来，刷牙睡觉。

　　日复一日，这就是我每天的功课。

　　时光就这么流逝。

　　一个同事感慨：唉，现在觉得时间过得太快了，刚刚过了春节，马上又到五一。一到五一，就奔十一，而到了十一，又快到年底。怎么这么快呀？时不我待呀。

　　是这样的。

　　关键是，天天过，月月过，年年过，而我们又做了什么？

　　西安一个小小孩陈智，写了一本非常时尚的网络小说《给爱一点时间》，文笔流畅得无以复加，语言生动得令人捧腹，比痞子蔡完全有过之而不及。我羡慕陈智，如此小年纪，在做的，是自己喜欢的事情。把当今流行的网和恋结合起来，以自身经历作脚本，配以处处闪现亮点的网络语言，轻而易举地挥就了一部网上读物，迷倒了榕树下大批拥趸。

　　身边有个电台主持人，节目主持得不错，在反响巨大时，急流勇退，把电台经历写了书，自己又筹款开了一家书+酒吧。问她，有单位好还是没单位好？她的态度比较坚定：还是出来自由点。也许她的书+酒吧并不特别盈利，也许她每天必须应酬各方捧场朋友，但她还是选择了自己愿意做的事。尽管累点飘泊点，但心中的踏实感是无人能比的，至少她知道自己在做什么。

　　天生羡慕这种人。会羡慕，是因为自己缺乏勇气和冲劲。

　　如果仅仅是羡慕，过后能回归平静也行。但我又不能，偏要受各式各样的榜样触动。比如，今天这个出国留学了，明天那个辞职单干了。甚至，在办公时间看到某同事捧着外语书埋头苦读，或是在网上看到身边某某人建了个非常雅致的个人主页……我都会感慨良多：别人都在做属于自己的事情，我呢？

　　每当这个时候，就睡不着。而每当梦醒的时候，就会万分焦急。我在哪？我想干什么？我在做什么？我为什么活着？我活得好吗？我……

　　现在还不到喊"如果时光倒流"的时候，但时光飞逝，我该咋办？
