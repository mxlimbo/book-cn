人，千万不要做自己不愿做的事

　　老张，一位年届四十的同事，在酒桌上谈起了自己的难事。

　　老张同部门有个同事小刘，每周都找一个美术编辑为她所编的版配图。该美术编辑与老张同住一个住宅区。小刘理所当然地请老张帮忙：每次美术编辑画好图，就塞在老张楼底的信箱里。老张早上上班时打开信箱取走，带到单位给小刘。

　　事情很简单，每周一次。

　　可就这看似简简单单的事情，却把老张烦得成了心理负担。

　　有时早上上班，老张忘记了打开信箱取图，走到半截，想起来了，赶紧回家，重新打开信箱取走。有时老张想起了带图，却发现忘带了信箱钥匙，于是重新上楼打开里外三层的家门，屋里屋外找着钥匙再下楼取图。有时老张可以不上单位，却为了送图，非得跑一趟不可。有一次老张有事要外出，不上单位了，但惦记着图，就打电话找同住宅区的另一位同事，说明情况，再把图给那位同事送去，嘱咐他第二天务必带走，如此折腾，烦不胜烦。

　　老张这边，图可成了事。但话到嘴边却不好讲，因为太简单的事了。讲了，听者会以为他小气不肯帮忙。小刘那边，却无比自在。不就一个顺水人情嘛，太简单的事了。连谢都不用讲。

　　这件事，让老张吸足了教训。

　　太太有一次请他每周为小孩买一箱酸奶。老张当即就拉下脸来，愤然说：太难了，我做不到。太太不解：每星期买一次奶，还难？老张确凿无疑地说，是难。

　　老张说，难的不是买奶，而是雷打不动的规定。要他答应，他真办不到。万一哪天忘了呢!但如果没有此规定，哪天他想起来，会主动去买，愉快地去买。

　　答应的事情，就要办到。因为答应，就成了约束，就成了一件事。如果每星期这边答应一件小事，那边应承一件小事，事都很小，但积少成多，就会形成每天都有很多必须做的小事。人被这些小事所困扰，可能会因此坏了一天的心情，也可能会因此一天一事无成。

　　别人请老张为一个专栏写稿，专栏专栏，固定栏目也。老张登时双手作揖说，先饶了我吧。专栏就免了，哪天我愿意写高兴写有得写就写点吧。

　　做自己不愿做的事，是件痛苦的事。做没有负担的事，是幸事；做自己喜欢的事，是乐事。如果，每个人都能少做点痛苦的事，多做点幸事乐事，心情必定会愉快些，办事的效率会高些，成效会大些，环境气氛会更好些。不是吗？
