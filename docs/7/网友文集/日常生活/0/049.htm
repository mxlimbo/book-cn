走过的路，再也不能回头了

　　走过的路，再也不能回头。

　　有一部电影，叫不道德的交易。年轻夫妇，除了有爱之外，一无所有。一个富翁，跟男的说，给一百万，借你太太一晚上。夫妇当然坚决反对。回家后，他们辗转反侧，彻夜未眠。实在太穷了。后来想想，有感情在，怕什么!一晚上换一百万，值!他们和富翁签了合同。富翁给了一百万，带着男的妻子走了，到了一个小岛，让她享尽奢华。

　　妻子回来后，尽管说好不提那个晚上的事情。但两人之间的阴影已挥之不去。男的总要问：他比我强吗？女的实在忍受不了。后来，他们分居了。

　　女的跟富翁走到一起，过着富足的生活。男的到了一个大学当教授，他想明白了：是自己错了。心里彻底理解也原谅了他的妻子，但他想，自己没法给妻子更好的生活，可能这样的选择更好吧。

　　一次拍卖会上，富翁用50万买一头小狗给这个女孩。那个男的也在场，他举起牌子，出示了100万。他只是想把100万还给富翁，也了却对妻子的愧疚。

　　回去的路上，女孩跳下了富翁的车，追着丈夫去了。富翁对司机说，我不想阻挡，因为，她的眼里有着对前夫深深的爱。

　　这部电影，是一件作品，很美。最后的结局令人满意。但现实中，可能吗？爱有了玷污，再也难以愈合；奢华地生活过了，还能回归清贫吗？所以，走过的路，再也不能回头了。
