本不是马铃薯，不想在土豆堆里混

　　其实，这句话应该说成"本非马铃薯，岂能在土豆堆里混"。

　　如果用于人，从字义上理解，就是说，自己不愿意跟与自己不同一阶层或水平线的人打交道或同流合污。再进一步理解，就是说土豆算什么，我瞧不起。

　　呵呵。

　　想起一个故事，对了，不是故事，应该是旧新闻，旧闻。

　　约翰・莫根，英国著名男士杂志《绅士》的主笔，所著的《现代礼仪》是英国各图书馆必备的参考书，一位真正的"英国绅士"，社交界的当红宠儿。41岁，从住家三楼跳下来，自我了断生命。

　　警察在他的住所发现了三百件绣着他名字缩写的衬衫，九十双手工制成的皮鞋，还有六十套高级男装专卖店为他定制的西服。

　　正是这位礼仪权威，在教导现代人如何做真正的绅士淑女的同时，严格恪守自己的生活习惯：从不去超市或一般商店购物，牛奶面包果酱从来都是Fortnum&Mason包装精致的产品；从不去普通的银行存钱或取钱，而是在会员极为精选的Claridge俱乐部中兑换支票；更甚的是办公室的洗手间从来不去，而是不辞劳苦地来到他自己的俱乐部，享受有专人伺候的服务……

　　在公众眼里，他是一个高品味、有魅力、有修养的人。而关于他的身世，却无人知晓。几乎没有人去过他家，没有人敢询问他的隐私，没有人知道他是同性恋或是异性恋者，也没有人知道他有没有亲密爱人，因为在他的笔下有关淑女绅士的规定中，这是最没有教养的行为。

　　这致命地造成了莫根的悲剧。

　　看莫根的装束打扮，人们以为他肯定是世家出身，至少也是个没落贵族。实际上，他出身于一个推销员的普通家庭。从艺术学院毕业后，他曾服务于《卫报》，由于总是穿着三件套，与周围环境格格不入，被迫离开。后来转入刚刚创立的《绅士》杂志，沉浸其中宣扬的那一套旧式英国绅士的规范中，慢慢地竟成了真正旧式英国绅士的代表。他由此一发不可收拾，把此项工作当成了上帝赋予的使命。

　　走红后，莫根很少回老家看望父母，把家庭抛在了身后。他开始把与别人合租的小房子换成了大房子，再后来，又把普通住宅区的大房子换成了高尚住宅区的小房子……

　　此时，莫根已不是马铃薯，不想在土豆堆里混了。

　　不在土豆堆里混的莫根成了内心最寂寞的人。他由此犯了忧郁症。忧郁症不治，他只能走向了生命的尽头。

　　有心理学家剖析莫根的病因，指出，莫根的悲剧，归根到底，是英国那无可救药的阶层等级制度造成的。

　　这是社会的原因。如果究其心理原因，则可以看作：自己太跟自己过不去。

　　莫根如此，我们难道不也是这样吗？
