你不觉得我很苦吗

　　 港台明星中，较喜欢的是吴倩莲。

　　喜欢她那灰天鹅的容貌，在众多白天鹅中她并不出众，但能让人过目不忘。

　　更喜欢的是她的气质，隐忍、含蓄、有太多的内涵却含而不露，让人探寻。

　　有天她出现在凤凰卫视的《明星三人行》里。在铁嘴不饶人的窦文涛近乎"逼问"下，她侃侃而谈，不加掩饰，从容应付，更加深了我对她的好感。

　　问及为何与交往了12年的男朋友分手，吴倩莲平静地答道；是因为缘尽了，所以自然而然地分开，但依旧是好朋友。

　　这是一句男女朋友分手时惯用的词汇。为着很多解释不了或不便解释的原因，"缘尽"就像万金油一样，及时有效。

　　但这话出自吴倩莲的口，我信。

　　因为她说的时候，是沉思了以后说的，尽管很平静，却很艰难。在平静的语调和神态中，我品出了她的无奈和难言之隐。有了这份厚重，"缘尽"还能不信吗？

　　吴倩莲是港台明星中人气很足却绯闻最少的一个。要做到这一点很难。除了拍片之外，她最喜欢的业余生活是旅行，其次是足不出户。

　　足不出户的时候，她说自己在家里干一些修理工之类的杂活，比如修电视、音响、电话机甚至马桶……她对此解释为是一种自立的能力，一种独立的精神。

　　窦文涛戏言她，像个男人。窦的比喻非常有趣，说"你长的一副小鸟依人的柔弱的样子，当男人们张开臂膀想抱你时，却发现你还扬着胸脯梗着脖子，骨子里根本就不需要保护"。

　　窦文涛的话有点毒。吴倩莲并不生气，她只是淡淡而且柔柔地反问："你不觉得我很苦吗？"

　　一句轻轻的反问，倒是让窦文涛愣住了。那一霎那，我想很多人会因此爱上她，包括窦文涛，也包括我。

　　真的很苦。

　　没有一个女孩子愿意在家里当修理工，尽管那是一种自立能力和独立精神的体现。

　　如果有人疼爱，有人呵护，有人照顾，我想就是再能干的女孩子也不愿去实现自己能干的价值，因为她不需要，她只要享受爱就行了。

　　没有爱的女孩是苦的，就像花朵没有阳光的照射和雨露的浇灌，它哪怕盛开也不能璀灿。没有爱的浇灌，女孩就不成为完整的女孩，她不再水灵，不再滋润，不再娇媚……

　　吴倩莲也不例外。

　　所以吴倩莲的眉宇间有种耐人寻味的东西，淡淡地、若隐若现的，挥之不去的。

　　我寻味之后，自己把它理解为"曾经沧海"，只是少了分凝重，多了些希冀，不知对不对。
