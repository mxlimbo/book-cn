爱，千万别乱七八糟

　　朋友陷入感情僵局，找来我们几个死死党，帮着出谋划策，寻好点子。

　　细听诉说，才知道问题症结在哪：

　　朋友是个已婚男士，太太是当初学校里的校花，鲜艳欲滴，被朋友连蒙带骗入了虎穴(当然，也是两情相悦)。三年后，在新闻机构工作的朋友，以巨大的魅力又迷倒了一个年纪更轻的靓女孩子，两人迅速点燃爱火，愈烧愈烈，朋友的太太知道了，闹得满城风雨。朋友一怒之下，来到了深圳。那个女孩则一气之下，嫁了别人。

　　跟所有传统的家庭一样，随后，朋友的太太和孩子也落户深圳，尽管外表看起来相安无事，但裂痕无无法愈合了。那个嫁了人的女孩，旧情难忘，痴心不改，毅然抛家别子，去了北京，报考了研究生。只盼毕业后能有情人终成眷属。

　　朋友也是作此打算。无奈朋友的太太死活不肯，她希望朋友帮她自立以后，才离开这个家。朋友同意了，毕竟，爱情不在亲情在，况且，还有聪明活泼的孩子。

　　北京的女孩，每次打来电话，朋友都信誓旦旦：自己马上就搬出去。如此说法，一两年都没有兑现。北京的女孩开始犯嘀咕了，天高皇帝远，情人的心思谁能猜得出呀？她也打退堂鼓。两人的关系，就这么热一阵冷一阵。

　　朋友在深圳的工作有点不顺，虽说他的业务能力很强，但性格比较耿直，得罪了领导。别别扭扭中，人的心情也走入了低谷。远水解不了近渴。身边的太太是毫无感觉了(私下两人都已签了离婚协议，财产也分隔好了。只等分到房子和太太的工作有了着落，马上劳燕分飞)。女朋友又远在北京。平时连个拉着手说话的人都没有。朋友异常苦恼。

　　别的哥们，给朋友介绍了一个身高1米77的模特，人是够靓的，相处时也轻松，谈得来。但朋友说，这种身高，让人有了无形的压力(尽管朋友也是1米77的个子)。好端端路上走着，别人的回头率总那么高。

　　在单位，又有人搓合朋友和另一位毕业没两年的女孩。女孩像个新新人类，青春、亮丽，身边不乏追求者。朋友跟她打电话，问：想不想我呀？她会说，想呀。女孩出差，朋友给她打电话说：我很想你，你知道吗？她会回答：知道，我很快就回来!出去吃饭，朋友的手揽着她的腰，她也没拒绝。朋友对我们说，好像这个女孩也喜欢他。但不知女孩是对所有人都这样，还是只对他一个人这样？

　　这就是朋友的感情僵局。他说一天要给这几个人打电话，说不同的话，陪不同的小心。要是，万一张冠李戴了，说错话了，那可怎么办呢？

　　我们大笑。另一个过来人的死党一针见血地戳穿：就你这一阵一阵地，还不得头痛？!

　　按朋友的逻辑分析，他的第一方案是：与同单位的女孩摊牌，问个究竟，看她是何种态度，两人能不能发展？

　　这一方案不成，那么就走第二方案：跟模特朋友细谈，看两人深入发展的可能性大不大？

　　如果这个方案也不成，那么就走第三方案：创造机会到北京，跟有了几年感情基础的女朋友好好沟通，看以后能不能走到一起？

　　如果这个也不成，那就只好暂时做独身的准备了。

　　看着朋友那颓废的样子，我们异口同声地批驳他：就你这样，哪个女孩知道了你这三个方案，谁都得跟你黄!

　　看来，朋友的苦恼有道理，但谁也帮不了忙。
