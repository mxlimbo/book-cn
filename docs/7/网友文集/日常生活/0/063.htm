喜欢那种眼神

　　一种眼神，让我陶醉，让我感动。是那种恋人男看恋人女的眼神。是那种热恋中的男人看女人的眼神。是深陷爱中的男人看女人的眼神。

　　眼神里，有信任基础上的默契，有时时刻刻的抚慰，有溢满心头的无言的爱，有那种我即是你你即是我的不分彼此的亲近……我最喜欢看这种眼神了。

　　在朋友聚会的饭桌上，男孩看着对面的女孩，眼神里有会心的微笑，有等待女孩首肯的探询，有妙语连珠后给予掌声的希冀……男孩的眼光始终离不开女孩，生怕一不小心，女孩就会人间蒸发。我很羡慕那位女孩，如此有福气，定是前世积下的德。我会很欣赏那位男孩，爱，就应该这样，无时不在，无处不在。

　　在路上，在汽车上，在饭店里，在所有的公众场合里，我总下意识地搜寻这种眼神，百看不厌。我发现，只要沉浸在爱河里，眼神大抵相似，都像吸铁石一样，一看定格，再也难以挪开。难怪很多经典的眼神对视，让爱情故事流芳千古，让电影历久不衰，让明星大红大紫，让人们的思想行为深受影响。《铁达尼号》，男女主角情深意切的看，使沉船的大灾难变得渺小，使逃生的举动变得无足轻重；《爱情故事》，爱的眼神，让整个七十年代西方国家的人们重捡了爱情。就是《花样年华》，粱朝伟看张曼玉的眼神，也令人坠入了情网。

　　电影里的眼神，有夸张的成分。现实中，却是实实在在的。少男少女的眼神，如痴如醉，如火如荼，可以爱到吃掉死掉，无怨无悔；成熟男女的眼神，会深不可测，点到为止，心领意会，就像白开水，像呼吸，平常却不可或缺；老年男女的眼神，可能是不经意的对视，都能一眼看到心里，读懂读透。

　　看到爱的眼神，我总感慨良多。有时觉得谁跟谁不合适，但他和她却爱得死去活来；有时觉得两个人很般配，却无奈最后分道扬。爱，就这么不可思议，可以是男财女貌，也可以是男貌女财(才呢？丢了!不过，有才即有财呀，等同吧)。

　　网络时代，爱情来得快，像速食面，一冲即食，食而知味，久而生腻，腻而弃之。无痒无痛，疾风暴雨吹过，明媚阳光重现。现实社会，爱情也加了现实的筹码，"他有钱吗？""他工作如何？""他的房子大吗？""车子是进口的名牌吗？"……这些，是必须考虑的，即便口口声声说爱，但没有爱的物质基础，爱的上层建筑也会坍塌的。女的哪怕不这样想，男的也会这样想的。

　　在网络时代的现实社会中，搜寻那种纯纯的爱的眼神，有点难。一旦发现，我总是为之震撼，继而为之喝采。我想，下次，我再找到这种眼神时，一定要上前，握手或是鞠躬，向他们表示我内心真诚的祝福。
