答蒋郎憔悴之二月河比金庸

　　蒋郎憔悴，不好意思 ，今天我才看到你的"愿闻其详"。

　　当然，作家之间，没有什么谁不能跟谁相比，也没有什么谁比谁强。要说区别，只有相形之下更加喜欢谁的作品，或是更加欣赏谁的作品。

　　如此说来，我更喜欢金庸的作品。

　　因为金庸的作品，是成年人的童话，是成年人的梦。有金庸，才有杨过、小龙女、黄蓉、韦小宝……这些活生生的人物，带给我们笑、带给我们泪、也带给我们生活的色彩。没有金庸，就没有这个缤纷的世界。

　　而没有二月河，帝王将相们的正史野史也存在。只不过，不是二月河写，就是别的作家写。

　　再从我个人角度，我以为，论文字功底，论文章架构，论想象空间，论作品意境，我更喜欢金庸。

　　匆匆几笔，不尽思量，有许多不攻自破的地方，请指教。
