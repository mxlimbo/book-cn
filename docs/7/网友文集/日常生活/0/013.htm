最怕听这种声音

　　网络时代，让许多事物有了立足的借口，比如网上情缘。

　　一个膝下孩子已长大成人的老头，可以端坐电脑前，摇身一变以年轻人出现，跟网络那一端同样身份未明的美眉一来二去地谈情说爱；一个已是妈妈级的女人，也可以在虚拟里以少女的形象迷住未明世事的毛头小伙；每个情种还可以多方联系虚拟情人，并且一次次地表白：我只爱你一个。

　　在网上，可以今天甜言蜜语，明天反目成仇；可以说谎不脸红，可以许诺不兑现，可以假戏真演。

　　只不过，在网上，谁也见不着谁，各种表白各种举动可以来得理直气壮，不像现实，火候稍微把握不好，就会引火烧身，惹出串串麻烦。

　　当然，只是虚拟里来虚拟里去，还是不过瘾，于是，各种赚钱盈利的公司就挖空心思地想出多种招法，让这种网上的情缘演绎得愈发如火如荼。聪明的电信公司及时推出了手机短信息服务，不管你愿不愿意，全部给你自动开通，愿者上钩的，则乖乖地交钱来。

　　一时间，全世界不管白天黑夜，不管是工作还是休息，不管是公共场合还是图书馆，只要有手机的地方，就会"嘀嘀"地冒出各种怪异的声响来，声音短触有力，提神又醒脑，让机主昏昏欲睡中顿觉精神百倍，打开屏幕观其内容，欣喜之情尽形于色。

　　真是皆大欢喜，网络公司赚足了钱，电信公司也鼓起了腰包，电脑商喜逐颜开，网虫们情圣们也有了用武之地。最近，这短信息服务还在政治领域干了件大事。菲律宾反对派，就是用了这种新型技术，迅速动员民众，间接导致军方倒戈支持反对派，从而令国内局势急转而下。

　　看来，网络的力量无处不在，无时不有，就连短信息也延伸渗透到各个角落。难怪身陷其中的人，可以喜洋洋说服自己跟上时代的步伐。于是乎，白天里，每个人都道貌岸然，该干嘛干嘛，是局长的人前挺胸扬头；是白领的处处彬彬有礼；是车夫的可以满嘴脏话；是打工仔的老板面前照样俯首听命，完全没了网络里当大款的派头。

　　工作都在急急地进行，只盼赶紧下班钻进虚拟里，摘下面具换上新装，以新的角色出现，然后，谈情说爱，打情骂俏，曰实现自己人生价值。

　　所有的对错，都有冠冕堂皇的借口，网络时代嘛。

　　情可以放纵，欲也可以发泄，人可以滋润，生活可以有滋有味地过，这就是网络带给人类的最终目的吗？

　　我不懂。反正，最怕听的，就是手机"嘀嘀"的短信息声音。
