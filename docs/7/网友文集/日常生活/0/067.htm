文人苦吗？

　　现在不大喜欢去书店了。

　　新书出的频率太快，一摞一摞地，封面漂亮得无以复加，纸质厚薄不等，开本大小不一，只要新奇，就有卖点，无需照出版机界的老规矩。重包装形式，忽略了内容。打开书来，往往文不对题，题文不符，或是空洞无物，味同嚼蜡，读不下去。

　　最难以忍受的是，几个炒作得厉害的作家，作品重复归类包装发行，一个系列一个系列的，迷惑读者视线，不小心的，往往掏了腰包之后，才发现上当。这可能与作家的初衷相背，但毕竟有他的招牌，所以也脱不了干系。

　　日前看一篇文章，说贾平凹给《家庭》撰稿，一篇三千，篇幅长短不论。这真有点吓人。记得董桥给香港报纸写文章，如此重量级的作家，才千字千元。正为老贾唏嘘，又闻王朔向约稿的报社张口：付得起稿费吗？一字五元！天呀！不是王朔存心与该报社为敌，就是太不把自己当回事。写的是什么金文，卖得如此高价？

　　市场经济嘛，作家们敢开尊口，只要有买方市场，也不足为奇。一个愿打，一个愿挨，甲方乙方的事咱们管不着。只是，赚了钱的作家们就不要再讨了便宜又卖乖了，到处喊抓盗版，到处申辩自己的收入低。跟着凤凰台满世界跑的余秋雨，名利双收之际，竟为反盗版自动歇笔，以行动表决心。

　　且问，一篇文章重复收入多个版本出书发行，算不算另一种意义上的盗版？凭着自身名气，信手涂鸦胡诌几笔赚取稿酬，又算不算某种意义上的盗版？前者盗的是读者的口袋里的钱，后者盗的是读者脑子里的"钱"（文字垃圾堵眼塞脑污染心灵）。

　　为什么总是赚足了钱的作家们在喊抓盗版，而没有吃了亏的读者喊抓作家盗版？

　　从另一种意义上讲，如果盗版真的能解决几个下岗员工的吃饭问题的话，字里行间处处流露爱心的作家们又何乐而不为呢？就当个顺水人情，把多余的钱拱手相送好了。何况，自身又没损失什么，多的还是知名度呢！
