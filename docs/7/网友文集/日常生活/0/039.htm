情迷海鲜档--海南行有感之三

　　生平最爱吃的，就是海鲜。

　　从小长在海边，一天三餐几乎少不了海鲜。长大了，离开家乡，虽然不能天天吃海鲜，却隔三叉五地来点虾，伴着青菜和米饭，吃下去，感觉非常地爽。

　　这次海南行，最满意的，就是海鲜了。

　　海鲜海鲜，第一必须是海里的，第二一定要鲜。海南的海鲜是名副其实的海鲜。捕了鱼的船直接开到岸边，临海而建的宾馆酒店纷纷就地取材，从渔贩子手里接过海鲜，下锅烹饪，上桌即食。

　　来海南之前，就听说海南的名吃是文昌鸡东山羊。

　　踏上海南，则闻到浓浓的海腥味。朋友说，我们吃海鲜。那时，我的心就激动地怦怦跳起来，心情好到了极点，眼里的景越来越美。

　　第一顿饭，地主朋友点了满桌的海鲜，白灼基围虾、清蒸石头蟹、豆豉白昌、葱煎芒果贝，还配以海南名菜家鸡鸭、当地特产四角豆。本来坐车不敢多吃的我，这时候也顾不了那么多了。虾是我的至爱，一会功夫，面前的小碟满满地堆着虾皮。海南的基围虾，个不大，没有九节虾颜色鲜亮，但入口肉嫩，鲜里带甜，甜而不腻，再蘸以调料，味道美极了。我当众宣布：自己可以吃三斤虾，别的不要。

　　到了三亚，当晚我们驱车找到一个坐海边吃海鲜的酒店。酒店打出标语：海鲜大排档。说是大排档，其实是海鲜长廊，沿着海边，摆满桌子，身着海南岛服的服务生们在桌子中间穿行，忙得不亦乐乎。我们去晚了，酒店说，没位了，他们忙不过来，不想接活了。我们说可以等。

　　看着海景，听着涛声，我们却一点都不专心，都像身负特殊使命的特工，在长廊中眼观六路，耳听八方，密切注意哪个桌子要埋单。看着没人搭理，我们想出一招，自己动手，丰衣足食算了。于是，搬来桌子，到各桌抢出四五张椅子，派同来的小孩子端来碗筷和茶杯，地主朋友去点菜，我们去烧烤，双管齐下，一会功夫，菜也就纷纷上来了。

　　这次是清蒸膳鱼、清煮大闸蟹、开边基围虾、干煎马胶、粉丝扇贝……"吃海鲜，必须佐以小酒，才过瘾。"服务生忙不过来，幸亏我们有一个跑堂的小孩子（地主朋友的小孩），他忽前忽后地，一会拿来二锅头，一会拿来虎牌啤酒，伴着习习凉风，我们简直要把酒当歌，对海膜拜，直抒胸臆，长吁狂啸了。

　　一顿饭，把大家吃得心满意足。埋单时，服务生小姐这才笑容可掬地出现，一看，六百八十多。"不是大排档吗？有没有搞错呀？"我们猛地酒醒。借着月光细细看单，发现漏洞百出，一些别人的单算到我们头上，东减西减，减下了一百多块钱，还有一些有疑惑的，小姐说，我们是四星级宾馆，加收百分之十服务费。"什么？我们不都是自助的吗？！"

　　海鲜的确是够鲜的，只是大排档服务，星级宾馆收费。

　　第二天，到博鳌，我们痴心不改，又寻着海鲜大排档去。车刚停下，就有小姐上前招呼，跟着去，发现人满为患。好容易找到一个客人刚走的小间，急急坐下，生怕讨价还价没了位置。虾蟹鱼是必有的，此外，全是贝壳类，什么青口、圣仔、香巴瓣、花甲、沙白……按人头分，一人一个，盘子上了撤，撤了上，吃得满头大汗，满嘴油腻，满身腥味。幸好，大排档没多算钱，物美价廉。

　　天天海鲜，直到登上回深圳的飞机。

　　回来后，意犹未尽，想的，都是海南的海鲜。
