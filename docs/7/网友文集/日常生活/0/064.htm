一生的惩罚

　　那天，说了同事老张的难事，即每周一件小小事，可以把人束缚死。

　　一位朋友听了，又讲了另一个类似的故事。

　　在美国还是英国，一个男人开车撞死了一个女孩子。事故处理中判罚男人赔款几十万。悲痛中的女孩父母，向男人提出了一个请求：赔款不要一次付清。而是每星期汇一笔，一直寄到他们去世。

　　男人非常理解这对痛失爱女的父母的心情，不加思索地答应了。

　　此后，他每周寄一笔钱，给这对老人。

　　慢慢地，男人难以忍受了。因为每一次寄钱，都唤起他撞死人的回忆。这种回忆，折磨他的每一根大脑神经。后来，男人疯了。

　　这个故事，虽然简单，却告诉一个道理，要想惩罚谁，就不断地让他想起自己的过错。

　　一个作者在报上开了一个专栏，每周写一篇稿件。开始时，他非常兴奋，专栏不但提高他的知名度，赢来大批读者，而且还增加收入。该作者发自内心地感谢编辑。每周他准是提前一天把文章写好，发邮件到编辑邮箱里。时间观念极强，态度积极。编辑评价甚高。

　　渐渐地，每周一稿开始让他头疼了，本计划好的度假因写稿令他却步，临时安排的活动也得为写稿让路，就是想放松一下，也因稿子而放弃。他开始放慢写稿的节奏，推迟交稿的时间，不等到编辑催促，决不提前写稿。慢慢地，他甚至害怕编辑的电话。每周一稿已成了负担。对这种负担，他食不知味，夜不能寐。最后，他只能委婉地向编辑提出，取消专栏。

　　好事情、愉快的事情，一旦成了定规，就会向相反的方面转变，成了坏事情、痛苦的事情。要想惩罚某个人，就让他不断地干同一件事。让事情成为负担，使其烦不胜烦。

　　其实，生活中的很多道理大致如此。我们说，干一行厌一行。本职工作，不管是多么热爱，一旦成了每天必须让你完成的一件事情，就会心存厌倦，不胜重负。

　　甚至我们的日常生活，一天一天，衣食住行、平淡无奇，我们就会感慨：无聊！没劲！

　　感情婚姻家庭也逃不出此规律。天天面对同一个人，没了新鲜感、没了激情，我们就不再重视对方的感受、不再珍惜已经拥有的。我们会说：唉，就那样，日子嘛，就这么过。

　　很多东西，你没法改变。没太多想法的人，能苟且偷安；有想法的人，只能跟生活抗争，末了，留下的，只能是伤痕斑斑。
