后悔药，真的不是好吃的

　　回到家，三点十五分，我赶紧换衣服，洗脸，想好带什么东西，然后，打电话。

　　"你拨的电话已关机。"传出电脑小姐的声音。

　　我愣住了。

　　好像料到，又不愿料到。

　　我的同学真的走了。可能现在就在飞机上，所以关了机。

　　怅惘若失。 甚至难受。

　　这是大学的同学。周末来出差。昨天到的，约昨天中午见面、吃饭。

　　我第一反应就是：你来几天？同学说，明天就走。

　　我的确有事。中午是不能吃饭了。我说，晚上吧。

　　到了晚上，我又有事。急急地给同学打去电话，说，只能晚些时候，赶得及，就去泡吧。同学有点遗憾。不过表示理解，说：好，晚些时候再通电话。

　　后来，很晚了，我事情没结束，就把手机关了。

　　今天，我要命地起晚了，到十二点多才醒，又有事情，忙去了。

　　心里不断地希望，同学事情没办完，会延期走。今晚一定无论如何一起吃饭、好好叙旧，再带他来看我住的房子。

　　可是，现在，手机却是关了的。

　　几年不见的同学了，尽管上学时并不是非常地亲密，但班里就那么几个人，过后想想，竟如兄弟姐妹般，还是有感情的。

　　但我却在应尽地主之谊时这么轻而易举地错过了。其实，我有事，但事情可以推掉的，可以延迟的。而我没有。

　　我真的不够意思。

　　很多事情都是这样，在经历的时候，我总是莫名其妙地不珍惜，过后，就开始内疚、自责、难受，恨不得重新来过。

　　亲戚朋友来，我也是这样。总推托，很忙。你们自己去逛吧。然后，他们走的时候，我开始反省。父母亲来看我，我总要使点小性子，过后，又难过得不得了，觉得自己不是个孝顺的孩子，父母老了，长大的我还这么不懂事。

　　有时在办公室里接到电话，对方一件事讲半天，我很不耐烦，就会语气生硬地三两下收线。放下电话，又不安起来，想想：人家并没有错，只是想咨询得明白点。将心比心，如果我受此冷落，会如何想？

　　每天处理邮件，需要回复的，有时会措词不当，只顾自己痛快，并没考虑到对方的心情，过后，也内疚，却死抱自尊，不肯再去信承认错误。网上下五子棋，对方下错了，想悔棋，有时我会硬着心肠"不同意"，当自己也受相等待遇时，就会反思：不宽容别人其实也在不宽容自己。

　　这两天一直在看一本书，余杰与余秋雨论战结集出版的《文化口红》(可能是盗版的)，里边余杰一个劲地要求余秋雨，向历史忏悔。他同时也把自己和大家摆进去：不单余秋雨要忏悔，每个人都要忏悔。

　　不管后生仔余杰挑战红得发紫甚至某种程度成了中青年人"文化导师"的余秋雨有没有道理，有没有别的附加成分，但单纯从忏悔这个词上看，我同意余杰的观点。

　　至少，我自己需要忏悔。忏悔自己的过失和不当。

　　如果忏悔过后，能避免新的过错。我愿意天天忏悔，时时忏悔。我希望下次同学来的时候，我不会问：来几天？我希望自己会第一时间前往相聚。

　　后悔药，真的不是好吃的。
