再说缘

　　平时说惯了"缘"字，一时想写点缘的文章，却无从下笔了。

　　细细想来，缘，似乎总跟佛有点关系，有禅意，藏禅机，随便说不得的。

　　到古玩店里，远远总可看到，暗色调的店内，淡淡亮光照着墙上挂的名家书写的"缘"字，或是出自大家的"缘"画。缺了这个，好像古玩店的档次就差那么点意思，不够火候。

　　翻开佛书，没有不讲缘道的。讲因果，析轮回，普渡众生，清净六根，都少不了缘。一个国际品牌的名服装店叫"台湾小调"，卖的都是佛门弟子或信徒的"手信"（信物），衣服作工精细，挂坠精美，琳琅满目。老板说，卖得的钱是捐给佛门做善事的。

　　光顾的，都是有佛缘的人，不在乎价高，一买许多，既可穿，又行善，一举多得。皆因一个缘字。

　　又有电影＜半生缘＞，根据张爱玲小说改篇，黎明和吴倩莲将故事演绎得让人涕然泪下。

　　好一个半生缘，是生活毁了幸福，抑或生活磨灭幸福，还是生活里压根就没有幸福？只恨张爱玲，自己看透尘缘不说，还非得用老辣的笔，把本可以虚幻的东西，戳穿得清清透透，让人存不得半点梦想，只得在灰色的背景下活一步算一步。张爱玲笔下的尘缘，说白了，是孽缘。

　　再说人际关系中蜘蛛网般的人缘，错综复杂，无从下手，进去了，就出不来，越挣扎，缚得越紧。就是不进去，它还要来罩住你，防不胜防，无法逃逸。老话说，适者生存。人得八面玲珑，面面俱到，小心谨慎，亦步亦趋，这才在网中相安无事，进而如鱼得水，畅游自如。有人总结出厚黑学，估计就是网中之学，生存之计了。

　　所以说，人缘，啧啧，有缘也罢，无缘也罢，皆随缘去吧。
