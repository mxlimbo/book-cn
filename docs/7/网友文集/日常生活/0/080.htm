食色味香也

　　从小就馋，总爱吃一些稀奇古怪的东西，比如鱼肝油、酵母片、含片之类的，倒不是生活艰难，而是大人们说，这是药，不能多吃。我就记住了，是药，偏多吃。现在想想，可笑，幸亏没事，健康地活了下来，就是体质差了点。

　　长大了，馋病没改。到处打听哪有好吃的，哪又新开了什么店，又时兴吃起了什么……于是，部门聚餐，头儿第一个问的便是我："今到哪去？"我必定罗列出一大堆，任各种风味爱好者选。食过后，得到的总是夸。

　　昨晚买了本沈宏非的《写食主义》，就着咖啡店里跳跃的烛光，看得我饥肠辘辘，吃了一海碗田鸡粥，不过瘾，又要了一大盘草霉松饼，撑着胃难受，却还想吃。以前知道沈先生在《南方周末》开有关于吃的专栏，零星看过，这么集中系统地看，却是第一次。

　　说吃臭豆腐，不爱吃的人简直就是把"厕所摆上了餐桌"。那么依此推理，爱吃的人简直就是在餐桌上吃厕所（呵，被沈先生看见可就了不得了！）。我从来不吃臭豆腐，特此声明。

　　说喝茶，会喝的人，能出意境，如"枯"。无欲无心无语无情，只是对坐，就茶喝。妙的是，沈先生说，广东人不会喝茶，潮汕人除外。里边对潮汕人饮茶文化高度评价，我直乐。

　　说倪匡，尽管在香港以卫斯理出名，但到西方世界，却英语不大灵便，点菜时可急坏了。有人给他出了一招，说菜谱一般都是字大的菜贵，字小的便宜。于是，有天倪匡到餐厅，就指着字最小那一行给侍者，双方眼神对视几个回合，未果。侍者找到经理，汇报说，有个怪怪的顾客，非说要点我们15%的小费。哈哈！可怜大名鼎鼎的倪匡，菜没吃成，这点丢人的小碎事还被沈先生宣扬了一把，闹得人人皆知，就像看卫斯理的破案小说一样有味。

　　沈先生把食与文化相联，写出情色饮食。食里有色，色里有情，情离不开食。书的编后语说，这书写出了沈宏非对饮食恋爱般的细腻体味，是性灵的文字的饮食，真的很对。

　　大凡菜要好吃，必须跟文化联系起来。至少要有典故，要祖宗三代明明白白，这样别人吃起来才能有嚼头。很多美食家都是文学家，反过来，很多文学家都是美食家。如曹雪芹，《红楼梦》里，美食跟情事一样引人入胜；林语堂，《生活的艺术》岂能丢弃美食？最有印象的还要数张恨水，《八十一梦》、《魑魉世界》……把战时重庆的生活描绘得有声有色，就是清水豆芽，也是碧绿如珠，味如珍肴。看得我天天喊妈妈买豆芽，不是想吃妈妈做的豆芽，而是想吃张恨水笔下的豆芽。

　　啧啧，《写食主义》我只看了几页，就有如此感慨，再看，估计走不出来了。回单位，跟同事交流体会，说，我知道这本书在哪看最好！大家众口同词：洗手间！（哇，沈宏非非得把我剁了不可！）
