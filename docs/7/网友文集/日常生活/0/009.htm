电脑综合症

　　天天坐在电脑前，上班收稿、看稿、编稿、发稿、组版，发排……下班，只要看完新闻，就是进了书房，打开电脑，上网，看信息，收邮件，再跟网友们热热闹闹地打招呼……

　　日日如此，周而复始。所以，可以说，每天除了吃饭、睡觉，所有时间都在与电脑为伴。就是吃饭，也是端着饭盒，饶有滋味着打开电脑，边吃边看边乐……；就是睡觉，也是做着与电脑有关的梦，比如发现一个强病毒，杀死；又来一病毒，再杀死……

　　终于有一天，坐电脑前开始犯晕、眼花、恶心……强忍着，心想是一时的反应不良。试着走开，倒杯水，远眺窗外不远处的莲花山，深呼吸，调整心态，默默祈祷。然而，胃在翻江倒海，酸水不断地涌上喉咙，心跳加速，眼前发黑。完了，真的是得了电脑综合症了。最后，只能跑到洗手间，吐个天翻地覆，边吐边想，戒了戒了，打死我也不敢如此不要命了。

　　第二天，我彻底休息了，说休息，也是假的，当编辑嘛，版不可不做，所以还是尽可能加快速度，在屏幕前尽可能缩短时间，然后老老实实在打开纸质的书，一个劲地说服自己：还是纸书捧起来踏实、舒服。

　　第三天，还是休息，因为体质太弱，坐电脑前还有晕的感觉，不敢太肆无忌惮。只是，心里像揣着个小兔子一样，总惦记着那几个邮箱，当然还有别的……

　　第四天，好像身体好多了，我开始跃跃欲试，不断地劝慰自己，或许不是电脑综合症，只是那天碰巧吃坏了。然后，我理由正当地开始端坐电脑前，像久违了的情人一样，无比激动地拨号上网，对眼前一个个司空见惯的网页充满了爱意，这看看，那转转，简直是用心在抚摸每一个跳动的标题每一幅色彩鲜艳的图片……

　　从此，我又开始了往常的日子，可以一天不说话，可以几天足不出户，但不能没有电脑、没有网络……

　　在电脑前，我改变了自己的一切，比如急脾气。碰到网路堵车、网站被黑，我则不急不躁，每每保持淑女本色，捧着小书稳坐电脑前，无限耐心地翻着书，只是眼的余光时刻扫瞄着屏幕上的任何变化……

　　朋友忠告：女孩子不可以熬夜，会失去容颜。

　　我笑纳，只是不改。

　　如果钱可以换来快乐，我不要钱；如果健康可以换来快乐，我不要健康；如果容颜可以换来快乐，我不要美丽。

　　我要的很少，只要快乐。
