今天是清明

　　今天是清明节，天是阴的，昨天下的大雨停了。

　　云低低的，像人的心，有点重。

　　一早我就知道今天是清明，其实早几天就知道。但不愿说。

　　清明，一个凭吊的日子，带着哀思，带着怀念。很多人请假赶回老家，祭祀亲人。别的节日可以不回，清明不行。

　　妈妈也回老家祀拜外婆。外婆去世很多年了，在我上学的时候。我是外婆带大的，对外婆，有着深深的感情。那时候，我刚到北京，想着第二学期带外婆看长城吃烤鸭，但她却得了脑溢血，半身不遂，不会说话了。放假回家，我跟外婆说话，她只会笑，只能用手比划着表示自己的高兴。后来，外婆去世了。说是患了肺癌。我不在身边。

　　我时时想起外婆，总想，好人为什么命运总是不好？外婆吃了很多苦，二十岁出头就守了寡。外公是去了泰国打工，在那里得了破伤风，没及时救治，死了很多天后才被发现。外婆一个人把舅舅和妈妈牵扯大。

　　小时候，我记得外婆身材总那么高挑，头发又黑又亮，别人讨方子，她总笑着说，拿茶籽洗就可以做到了。外婆有空的时候，总喜欢拿着线装书看，里边是一部一部的古装戏。看得高兴，还会哼唱几句。每天下午上学前，外婆总给我煮绿豆汤或是凉茶。放学回来，则有点心。逢年过节，外婆会给我红包。那时，我小，不是很珍惜，总想，有爸爸妈妈的很多朋友给外婆红包，外婆钱很多。现在懂事了，外婆却走了。

　　我不愿去碰及内心深处的往事，尤其对外婆。外婆对我，可能比爸爸妈妈还亲。但我欠她的太多太多。

　　每逢清明，我总保持缄默,不想对自己提起，也不想提醒别人。亡灵，实在太重太重。此时言语，已显苍白。

　　总觉得，亲人已故去，往事已成追忆，应该深埋在心底，时刻怀念，而不是每年在某个固定的日子，以祭祀的形式，抒发自己的哀思，以抚慰负疚的心灵。

　　泪终归要洒的，思念终归剪不断的。祭拜的举动则是有限的，无限的，惟有乐施行善，消除业障，净化心灵，告慰天灵。外婆的在天之灵，会明白孙女的一片孝顺之心的。
