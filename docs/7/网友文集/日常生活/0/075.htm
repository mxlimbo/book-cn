生命童话

　　梅子涵肯定特年轻。

　　画像上不再年轻的梅子涵肯定活得非常年轻。

　　有颗年轻的心的不再年轻的梅子涵，注定年轻。

　　梅子涵爱讲童话故事，看到的想到的写到的，都是美丽的童话故事。

　　跟儿童打交道，是人生又一件快事。男性青年找对象，会说，找幼儿园的老师。整天在幼儿圈里打转的女孩，会有耐心，更有爱心。会单纯、无暇，更会天真、浪漫。找这样有童心的人当太太，人生足矣。羡慕的职业，是医院里的儿科大夫。整天可以哄来看病的小孩子："乖，不哭，让阿姨看看，哪里长了虫子了？"、"来，张开嘴，张大点，喊啊，哇，一个好大好大的虫洞，里边住满了小虫，怎么办？不怕，让叔叔来杀掉它们！"……甚至羡慕看小孩的职业保姆，可以整天带着两三岁不明世事的孩子，分享儿童世界里的喜怒哀乐。尽管也在家长不注意的时候吃点小孩子的水果糖果，但一旦挨主人的训，小孩却会一个劲地喊："要姐姐抱要姐姐抱，姐姐不要走……"多值得的事情！

　　梅子涵的幸福，不但在于她能跟小孩子打交道，还在于她做的是引导孩子成长的事业--讲童书，为孩子的精神导师。

　　英国第一位获得诺贝尔文学奖的作家吉卜林，写了一本童书《勇敢的船长》。小孩子当然看不懂了，得有大人讲。梅子涵就告诉孩子们：书中的主人公是哈维，家里非常有钱。他还不到16岁，正坐船从美国到欧洲留学。有钱的哈维难免染上一些不良习气，会站在甲板上说："我们要是能撞上一条渔船会有多棒！"话这么说着，吉卜林就真的让哈维掉到海里去了，当然，不让他死。而是让一条小渔船将他救起。故事就这么活灵活现地延续下去：渔船上有特鲁普船长和他的儿子。哈维告诉他们，自己是大富翁的儿子。但船长就是压根没听说也不相信。甚至觉得哈维脑子有问题，干脆一巴掌打得他鼻子出血，彻底清醒自己是在一条渔船上。哈维开始忘掉了自己的身分，跟船长的儿子一起干活，洗甲板、数捕捉的鱼、划平底船、学会开膛破肚……最后，哈维肯定是回到自己的家里，与富翁父亲团聚。当然，他已经不是故事开头时的小男孩了，他明白了人生的真谛。最后，哈维很有出息，还有那个船长的儿子也很有出息。富贵与贫穷并不重要，只要人明白一些道理，敢于吃苦和拼搏，最终都会有完美的结局的。

　　梅子涵就是这样给小孩子讲故事的。她讲过的故事很多很多，另一个印象深刻的是特拉弗斯的《随风而来的玛丽阿姨》。玛丽阿姨会夜晚的时候爬上梯子，往天空上刷胶水，再从篮子里取出一颗颗的星星贴上去，天空中就有了更多的群星在那儿闪闪烁烁。美吗？难道不美吗？

　　梅子涵讲得有道理，说她们那一代人从小没读过什么童话，想象力不够丰富。所以心不自由，精神飞翔不起来。其直接后果是除了自己给自己很多限制，也限制了自己对孩子的认识。剥夺了他们的童话年龄和生命愉快，配合着教育和学校，数理化、数理化……

　　养着孩子的家长注意了，不要再逼着孩子学余时弹什么钢琴上什么书法或跳什么芭蕾了，也不要再逼着孩子读什么奥林匹克数学班或攻几级的英语了。还不如，多给孩子心灵一点空间，看点童话故事，到郊外放一放风筝呢！
