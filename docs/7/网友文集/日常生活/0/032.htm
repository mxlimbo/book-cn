分手・别离

　　有一封来信，是用粉红色的信纸写成的，字迹清晰、行文工整。我猜想是位细心的女孩，就拆开读起来。

　　标题是《分手・别离》，里边的第一人称却以"他"来叙述。他说，那天，他和相恋多时的女友分手，在一个咖啡店里，女孩还给了他见证爱情的戒指。很长一段时间，他借酒浇愁，剃了光头，发誓将女友彻底忘记。后来，男孩听说女孩得了癌症，住院了，心里掠过一丝快意，幸灾乐祸地想女孩得到报应了。再后来，又听说女孩去世了。他心里多了几分感叹。过去的爱恨情仇纷纷扰扰算是有了了断了。

　　然而，却有一天，一位不认识的女孩递给他一封信，说是那位女友转交的。男孩有点茫然。女孩在信中说："我知道你会恨我，我那样做是不得已之举，因为我知道自己得了癌症，不想让你陷得太深，连累你……你的人生道路还很长很长，可惜我不能陪你走下去……"

　　男孩泪如泉涌。那一霎那，他以为是电影是小说，以为是上天在跟他开的玩笑。但，电影里小说里的情节，却真地降临到他身上。

　　情人节那天，男孩来到女孩的墓前，献上了一束白玫瑰，并写了一首诗，用火燃烧，希望女孩的在天之灵能看到读懂。

　　这就是一个叫程熙的男孩用粉红色的信纸给我写来的信。薄薄的两页，我却看了几遍。

　　这位男孩像是不会打字，或是手边没有电脑可以操作，手写的字一笔一划非常认真。从寄出的地址看，好像是一个镇三资企业的打工者。他可能只是想找个可倾诉的地方，表达自己对恋人的深爱之情负疚之心。

　　相似的故事我读过很多，但这封信，我却相信他是真实的。

　　很多少男少女的情爱故事，尤如树上翠绿欲滴的青果，看着诱人，尝着生涩。像小孩子过家家的小把戏，不是小女孩故意跟小男孩闹别扭，就是小男孩故意让小女孩生气。今天说分手明天又和好。阴阴晴晴像六月的天，说变就变，还没来得及安抚，又说有了新变化。难怪过来人看着会说，哼，为赋新词强说愁。

　　程熙的故事，没有这一类的娇情。它实实在在的，沉甸甸的，至纯至真，让人有心痛流泪的感觉。

　　我不知道剃了发的程熙是否还依然光头，我不知道逝去了爱的程熙是否已经抚平了心口的创伤？我只能在心里默默地祝福这位不曾相识的男孩，能在今后很长的人生旅途中，把这段不了情化为更加成熟的心智，轻装前行，更好地去把握和品味美丽人生。
