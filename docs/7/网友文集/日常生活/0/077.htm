不可比不可比

　　中午，躺在沙发上，一边看着新来的《中国图书商报（书评周刊）》，一边看凤凰台的节目。

　　报纸开篇说，现在图书市场上不外乎有三类热销的书：关于网络以及网络文学的书、关于股市投资的书、关于名人名家正史野史列传的书。归纳起来，是网、钱、人。占据了整个一渠道二渠道书摊书店。

　　想的是，尤其第一类，与网有关的书，正紧紧密密地充斥着E时代的各个角落，无处不在。

　　有人分析说，之所以网书此起彼伏，是因为准入门槛低，没有像传统时代要过编辑关。只要你手头有部电脑，有一根电话线，只要你想表达，想抒怀，就可以义无反顾地深入进行。没人管你，没人限制你，没人笑话你。你写得好，就有读者，就有掌声，就有高的点击率，就有多的回贴，就可以以文会友，就会凝聚人气。如果写得不好，也没人骂你、嘲笑你、攻击你。这就是网络时代的好处，你可以尽情挥洒，隔着屏幕，蒙着面，出丑了自己红一下脸，高兴了则大呼小叫，仅此而已，仅此足矣。

　　于是，前两年，台湾有了痞子蔡，写了《第一次亲密接触》，又出《雨衣》，今年又来了《爱尔兰咖啡》，人的期望值越高，失望程度就越大。书评家说，小伙子后劲不足了，语言已显苍白。大陆，人气旺的是宁财神。大家的心得是，在网上看，逗逗的，但结集出版后，就怎么看怎么不如网文好读了。还有，安妮宝贝、心有些乱、还有今何在，还有李寻欢……

　　网络，催生了一批小鬼美女作家，也衍生了速食时代的肠胃不良消化症，网文得一目十行地看，飞速地看，才好看，看完完了。

　　翻着报纸，电视上正放着迎清华大学九十华诞的回顾片《清华名人录》，这一期介绍的是两位大家中的大家冯友兰和季羡林。

　　冯友兰与钱钟书的笔墨官司人所皆知(确切些说，是杨绛与宗璞的笔墨官司)，但不管谁是谁非，冯友兰在历史上的地位和作用是不可抹灭的。作为哲学家，冯友兰一辈子与哲学打交道，他以《新理学》自成体系，建立了一家之言。冯老先生在北大的家，古朴书香。宗璞说着父亲，淡泊中透着怀念，衬着历史的厚重。

　　季羡林则显得人情味十足。九十岁的老人了，气定神闲。笔耕之余，最爱的是身边那只小猫了，猫趴在身上睡着了，只要一刻不醒，老人就一刻不动。 对他来讲，猫比啥都重要，哪怕破坏了精致的收藏，也是可以原谅。

　　我问在文化界小有名气的朋友，季羡林与张中行比，前者是否更有人情味，后者更智慧？他断然否定，说，在他，季比张更大智慧。

　　《清华名人录》很快就结束了。这个节目策划得好，可惜时间太短，让人意犹未尽。

　　我不仅想，一边是长江后浪推前浪、年轻得几乎傲视一切的网络写手，一边是走过差不多一个世纪、学贯中西的大学问家，假设放到一起，整个天平就会霎那间倾斜，前者轻得不经时间过滤，后者却像润玉，时间越长越亮，源远流长，不断放光。

　　网络时代是个什么概念，就是e书、e语言、e交友方式、e意识形态、e价值观……？我很迷茫。要我选择，我会选网络工具，加上积淀深厚的书、沉稳的阅读习惯、持重的思考方式、进取的精神和谦虚的态度。我会选老式的生活方式，有条不紊、不紧不慢，而不是连走路都一溜烟小跑……
