人生最快意的事情莫过于此

　　问：人生最快意的事是什么？

　　我会毫不犹豫地罗列出一系列：

　　人生最快意的事莫过于，在最忙的时候，能有一次意外的出差机会，可以关掉手机，自由自在地在外地呆上几天，什么也不去想；

　　人生最快意的事莫过于，忙了一天之后，洗个热水澡，躺到两米宽的床上，让紧绷的脑神经彻底放松、让坐了一天累得僵直的腰肢尽情舒缓；

　　人生最快意的事莫过于，下雨天，坐在沙发上听着雨声嗑着瓜子看着小说；

　　人生最快意的事莫过于，在自己过了几天盒饭或食堂的日子，极想打牙祭时，正好有心怡的人邀请，到心怡的地方吃心怡的菜；

　　人生最快意的事莫过于，自己搜寻很长时间没得到的书，突然有天在书店一个阴暗的不起眼的角落发现；

　　人生最快意的事莫过于，过了一个漫长的灰色的寒冷冬季，突然发现了树上长出了新芽，抽出了新绿，那一霎那，内心亮堂，无比兴奋；

　　人生最快意的事莫过于，一直想办又拖着没办总在心里耽搁的事情，忽然有天不经意间顺利轻松地办完；

　　人生最快意的事莫过于，丢了医疗卡，胆战心惊地挂失后，发现卡里的钱没少；在出租车上丢了手机，以为失之不归，却不料第二天一打电话，对方说自己就是的士司机，捡到的手机已交给出租车营运管理中心。雷锋又活了。

　　人生最快意的事莫过于，在路上见到擦肩而过的陌生人眼里的笑；莫过于为乞讨的人献出一个硬板；莫过于帮拿着重物的人按开电梯门；莫过于在公车上为老人小孩让坐；莫过于在别人做错事时说声"没关系"；莫过于看着漂亮的衣服表示一下赞美；莫过于看着闪亮丽人表示自己的欣赏……

　　人生不一定如意，但漫漫路上毕竟有太多太多乐事快事。

　　哪天天气不好，心情不好时，我会想，有这么多快意的事，干嘛自己跟自己过不去？
