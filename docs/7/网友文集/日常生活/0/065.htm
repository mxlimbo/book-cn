醉醺醺＋胖乎乎＝不媚俗？

　　这几天各大报纸都在炒崔键。

　　因为他来深圳的一个酒吧包装一个看好的乐队。

　　从1986年出道到现在，崔键成了老崔，八十年代出生的新生代管他叫崔大叔。但显然老崔没有意识到自己的年龄，胖乎乎的他依旧抱着酒瓶拿着喇叭当话筒，醉醺醺地吼叫一些即兴创作的"粗口歌"：多了偶像，少了才华，卡拉OK是傻瓜。

　　老崔之所以这样说，是因为不想媚俗。

　　记得八十年代时，别人假唱，他真唱，所以东方歌舞团的王昆鼓励他，促成了＜一无所有＞，也让崔键一鸣惊人。崔键说，那是个媚俗的年代，所以他打响。十几年过去了，已成老崔级的崔键，除了一无所有外，已很少也很难有别的声响。

　　王朔说，崔键的歌让人麻醉，让这些不想承担什么责任的人，觉得自己什么都没放弃，理想还在，勇气还在，希望还在。不管王朔点评金庸时说的是不是胡说八道，至少在说崔键的歌时听起来有几分道理。的确，这种麻醉，让六十年代出生的而今已是中年人的成功或不成功的人，觉得生活还有意义，这里边当然也包括崔键本人。

　　崔键教导没出道的乐队：不要因循守旧，摇滚就是不满意。他还呼吁深圳音乐人，联合起来，抵抗港台之媚俗文化。但是，有摇滚教父之称的崔键，却一直守旧在不想媚俗的自己的八十年代，死活不肯走出来。走不出来的老崔，除了胖了点醉了点以外，惟一的借口就是不想媚俗，也让大家不要媚俗｛其中的潜台词是不是可以理解为，只听他的歌，只跟着他走）。

　　如果说，当年是崔键，让中国有了摇滚，那么，崔键的不媚俗观，却让中国的摇滚停步。这当然不能怨崔键。只是，这十几年摇滚乐坛的一无所有，真的要归罪于媚俗吗？
