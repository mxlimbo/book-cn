折腾

　　回到家，头开始蒙蒙疼了起来。没准真被传染了--今天还在办公室跟感冒的同事逞强："我刚感冒过，有抗体。不怕你们这些病毒。"现在，也许病毒正死灰复燃，施施然卷土重来呢！

　　头疼，可还得上网。要不，更疼了。

　　支着脖子，打开电脑。拨号上网。右下方两小方格亮了，一闪一闪，绿得煞是可爱。每天，上网前，等着绿格出现，是我心情最好的时候。今天当然也不例外，抱病痛快。

　　绿了绿了。我习惯地拿鼠标点IE，喝口茶，等着页面显示。时间一分一秒地过去。屏幕上没任何反应。惟有一句提示：正在检测代理服务器……

　　怎么回事？我用的是一线通，以前没出现过类似问题，况且，菜鸟的我，根本不懂得使用代理服务器。那变好的心情开始直线下落，头隐隐地疼得厉害。情急之中，打开QQ，有几个好友的小脑袋在跳着。我紧急求援。好友告诉我：重启电脑。

　　赶紧依法炮制。此时，什么话，我都听得进去。只要能打开网页，本小姐就是赴汤蹈火也在所不辞。

　　满怀希望，连线，试图打开网页。再次失败。好友又告诉我，点网上邻居，在属性里添加协议。我又欣然听之，重启，重新拨号，重新失败。好友又出招：一个协议不成，再加一个。我又试之，再试之……

　　折腾了将近个把小时，网就是上不去，好像跟我有仇。说今天是强病毒"欢乐时光"的发作日，我还没进网，却中招倒毙了。

　　无奈中，心灰意冷，想跟好友道个别，却QQ 也登不上了，只见小企鹅图标在一个劲地蹦呵跳的，全然不知主人我的着急。

　　索然无味、头疼、脑子不听使唤。今天也够累的，忙上忙下，恨不得把一周的活全干完。现在，干完活了，却没有了休息的机会（当然，休息即上网）。

　　没了网，就像与世隔绝，无比孤单。躺在沙发上，拿着同事刚新出的书，看着雅致得不得了的封面，玲珑剔透的文字，想着身边的事，觉得少点什么，对了，还必须吃点东西，尽管刚吃得饱饱地回来。拿出冰箱里的西瓜、酸奶、乌梅，又打开一个鱼罐头，就着徐福记小点心，一鼓脑全吃了下去。呵呵，谁让心情不好，让胃来分散些注意力吧。

　　(后记：这文章能贴上，全得力于同事家的电脑。看来，我刚买的顶级电脑，完了！)
