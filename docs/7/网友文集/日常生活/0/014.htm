好一个元宵节

　　 今天是元宵节，有人说是中国的情人节。一星期后的今天，又是西方的情人节。

　　中国的情人节也好，西方国家的情人节也好，总归少不了玫瑰、巧克力和象征团圆的汤圆。总之，是和和乐乐，情意浓浓，爱暖心头。

　　情人节是有了，但世间有多少称心如意的爱情故事？记得大学的时候，最爱看的是美国七十年代一部风扉整个西方世界的爱情片《爱情故事》，剧情非常简单：两个相爱的年轻人不顾门第之见走到一起，他们有过摩擦，也闹过别扭，但爱始终紧紧地维系着两颗年轻的心。女主人翁因癌症最后死去，片子的结尾有一句至理名言："爱，永远不要说抱歉。"每次听到结束时的这句话，我必定泪如泉涌。记得那时候，我还没有恋爱，还不知爱究竟是什么。

　　《爱情故事》影响了我的爱情观，一直到现在。

　　高年级的同学说，这部片子之所以有影响力，是因为那个年代整个西方国家正泛滥着情欲，至纯至真的爱情少之又少，所以稀罕。

　　那么，后来又一次掀起情感波澜的《廊桥遗梦》,是不是又一次革命?（反婚外恋的革命），还有再后来引起争议的《铁达尼号》……

　　不得而知。

　　只能说，要打动人，只有真情实感。因为少，所以稀罕。

　　现在的情人节，人们多了些重视，是不是因为情泛滥，爱不够完美，婚姻的缺撼太多，所以需要用玫瑰、巧克力、汤圆种种方式和手段来弥补，来抚慰自己的良心？

　　形式有了，内容却空洞了。

　　我还是想买《爱情故事》的碟，百看不厌。可惜外边的影像店都没有。
