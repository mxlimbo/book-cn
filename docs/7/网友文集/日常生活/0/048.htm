第一次上网下棋

　　真新鲜呵，第一次在网上跟素昧平生的人下棋，笨劲傻样尽显，丑态窘相百出。亏得不见面，要不，不是像关公一样涨红着脸，就是像好斗的鸡落到汤里一样湿透发抖，活脱脱一弱智儿童，整个发育不彻底。

　　没办法。谁让平时是好孩子，不早学这些手艺，临阵露怯，而且还不是一般的怯。

　　说下棋，也是在捡好听的说。什么棋不棋的，就是那种小儿科五子棋，下的都是老弱病残者。而我是被诱导误入歧途的（很无辜呀）。

　　那天，朋友喜滋滋地告诉我，要下载一个游戏软件，下围棋。朋友已是业余几段了，本领高强。因为用的是我的电脑，我不得不在边上警惕地注视着，生怕碰了哪些私人文件。

　　好了，下载成功。朋友长驱直入游戏中心，欢喜之情溢于言表，几乎把边上的主人忘了。好家伙，屏幕上游戏大厅里，满满当当地坐着像是宋朝明朝那种朝代的古人，都正温文尔雅地挽起长袖举棋挥舞。很新新人类的朋友，此时也成了一个长满胡须的老朽，身着长衣，披着白发，满面皱纹。就是身体还敏捷点，三两步抢到一个空桌子，急急坐下（要抢座的），一会，就来了一位长须白发者，来头好像比朋友小点，朋友嘟囔一句："凑合吧，赚他个十分。"于是，举手，对方赶紧也举手应答。棋战拉开，两人进入了无人之境。

　　我悻悻退出。内心极其不平衡：首先受到了忽略，其次，这种玩意儿竟然不懂，落伍了。

　　朋友走后。我心有不甘。重启电脑，还好，游戏程序还在。

　　反正谁也见不着谁，我准备小试牛刀一把。那种激动的心情呀，就甭提了。我注册了自己的名字，摸摸索索进到了游戏大厅。有很多种游戏项目：围棋、像棋、桥牌、拱猪、升级…… 我看来看去，都不会，只能别无选择地点了五子棋。不就是不让五子成一线嘛，这还不会哪？我给自己鼓劲。

　　进入了五子棋游戏厅，也都是远古朝代的人，我也不例外，而且，还被变了性别。大厅里，清一色都是男性。变就变罢，这样下起棋来集中精神一点，公平合理嘛。我不在乎。

　　学朋友的做法，我也行动迅速地抢到一个空桌子，坐下。马上，就来了另一位应征者。他举手。我也想赶紧举手。但就是怎么动，手也举不起来，反而是人一会跳着离开座位一会又坐了回去。急死人了。我拼命双击鼠标，手还是举不起来，人却照样又蹦又跳的。心想，完了。第一次打照面，人家以为碰到一神经病呢。

　　几次三番，手还没举起来。对方已起身走人，另觅佳人去了。这会，我才看到，在下方，有个"开始"字样的钮，一点，手就举起来了。唉，吃一堑长一智吧。还好，马上又有伴找上门来了。他口里念念有词："送钱来，送钱来！"像另一神经病，正好配对。双方举手，正式进入比赛。我白子，他黑子。

　　好玩好玩。我的心快蹦出来了，天还不怎么热，却已手心出汗。顾着高兴，我把白子随意一放，才发现放错地方，想悔棋，已来不及，先输了一盘。接着，连杀三盘，我节节败退。把高兴劲都杀光了。想想，算了，换人吧，这个"送钱来"的神经病，还真有点邪气，我估计是被他中了邪了。

　　又找了个名字像是女孩子的人，信心倍增，果真，刚开盘，就赢得没商量。女孩一气之下，强行中断比赛（必须下两局），另寻出路了。

　　我越战越勇，有点所向披靡的样子。于是，自己给自己升级，专找年纪大点的人下，这样，好显水平。来了个自称是"老人家"的，我有点警惕，却也无不嬉戏，故意在下棋间隙，打点好笑的话逗他分散注意力，一来二去，我赢的次数慢慢多起来，后来，老人家甚至同意我悔棋了，我简直开心死了，这样，我几乎没有输的时候了。连杀几个小时后，老人家不跟我下了，改聊天。末了，还约下次下棋的时间，哈哈，首战告捷。

　　到夜里一点多，才猛然醒悟第二天早上要上班。

　　一觉醒来，才发现肩酸背疼，好像还得了颈椎病了。
