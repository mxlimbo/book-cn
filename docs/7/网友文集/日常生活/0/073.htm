沈从文与黄永玉

　　一个叫土蛋的大二学生，对《水一样的湘西水一样的凤凰》提出质疑。他说，这样写欠严谨。

　　我承认不够严谨。因为，我不是沈学研究者，我没有资格没有资历评价前辈大家的功过是非（当然，我也没有这样做过）。

　　我只是有感而发，对自己喜欢的作家，说点自己的感受。是自说自话，只求痛快。其中会有很多偏颇或不实。

　　土蛋说，他读过黄永玉写的纪念沈的文章，沈后来转向研究服饰，并不是无奈之举，而是自己喜好为之而已。还说他的家庭生活也还美满。

　　我却坚持自己的看法。

　　黄永玉写过一篇叫《那些忧郁的碎屑》，纪念沈从文。按辈分排，沈从文是黄永玉的表叔。即黄的父亲是沈的表哥。沈四五十岁时，黄还是二十出头的毛头小伙子。

　　黄永玉是我佩服的画家，单看酒鬼酒那憨态可掬的瓶子，就知道这老头多么有趣味有人情味。他写的很多文章，比如三毛之父张乐平，亲切随意，栩栩如生。但，纪念表叔沈从文的文章，我却觉得有点委婉有点克制，字里行间，点点滴滴，却不能淋漓尽致。

　　黄永玉对表叔当然有发言权。他22岁闯上海时，表叔沈从文就非常关心他。第一，生怕他盘缠不足，衣食窘迫，劳顿而死。第二，生怕他年幼无知，涉世甚浅，被洋场女明星缠身，落得把身子掏空的结局。为此，沈表叔写了大量的信，托上海熟人照顾黄永玉。殊不知，那时的黄永玉，正迷于木刻，食与不食都无所谓，哪还有时间精力去与女明星瞎混？

　　在黄永玉的印象中，兆和表婶特别温文尔雅，说话从来是C调，就是不高兴，也是降D调。仅此而已。

　　沈从文后转向研究服饰考古，是痛写思痛下的决心。要活下去，必须这样。黄永玉的文章里，提到了沈表叔自杀的事，是淡淡地几笔带过。倒是说，救过来活下来的沈从文，人生观有了重大的转变，死都不怕，还怕活吗？所以，他开始坦然地面对一切。黄永玉还提到钱钟书对他说过的话：别看沈从文表面很温婉，骨子里却是硬的，不想干的事，让他试试……

　　沈从文去世的时候88岁。在后边的日子里，黄永玉曾带着太太梅溪去看他，拿着三十年代沈从文在凤凰书写就的一块碑文，字体隽秀，刚劲有力。梅溪说，表叔，你那么年轻时就有这功力，现在永玉六十多岁了，还达不到你的水平。沈从文的眼睛里，泛起了一层亮亮的水雾……

　　这就是黄永玉回忆里的表叔。文章收在黄永玉的一本集子里，叫《那些忧郁的碎屑》。不知，土蛋同学，看过的是否是同一本？
