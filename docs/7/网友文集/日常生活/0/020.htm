成长的悲哀

　　星期一就要考试了，好紧张。

　　说考试，其实是说不出口的测验。就是那种学车前必须经过的理论测试。管理松点的地方可以开卷，严的地方却能严格到100分的卷子90分及格。

　　我不幸地属于后者范畴。在我之前，有很多"被杀的鸡"，所以我这只"猴"只能乖乖地复习，一点侥幸心理都没有。

　　提前五天看教材，每天下班看一点，八百道题总得知道个七七八八，要不，下场可想而知。

　　给教练场的师傅打电话，让他星期一早上八点来接，好赶上九点半的考试。

　　师傅却说，那九点到单位门口接吧。

　　"不行不行，太迟了，去到考场就考试，局促了点，还是八点吧。"我，还有一同报考的同事都吓得一旁叫喊。

　　"没事的，去早了挨冻。九点吧，我们送学员送多了，知道考试啥模样。"师傅坚持。

　　不行不行。"那八点四十五分，定了，不能迟了。我们宁愿挨冻。"我们斩钉截铁，老师傅笑了。

　　天呀，又不是什么严重的大考，我们竟被吓成这样？想想真是没道理。

　　当年高考，不也是从从容容地过来的，后来，考职称，考电脑，考创造性思维，竞争上岗……也是有条不紊地应付出来的。怎么现在碰上一个小小的学车理论测验，却束手无策了？

　　我们是怕考试吗？怕考试通不过吗？还是对自己没了信心？

　　整天稳坐办公室，干着跟自己专业对口的活，没有优胜劣汰，没有危机感，所以我们安逸地日过一日 。我们可以终日上网或看报，关心国际局势国家大事，可以侃侃而谈和平与发展，可以信口开河指点江山，可以感慨可以牢骚满腹可以发出些微的杂音，可以身处温饱感慨别人的不温饱，可以说着下岗的话题却觉得那是遥远的事情，可以……

　　我们领着工资拿着奖金到饭店吃饭泡酒吧唱卡拉OK，我们一样也不缺却一点也不快活。

　　究其原因，只能是我们在机械的生活中渐渐迷失了自己。所以，我们不敢承担责任，不想面对困难，不愿履行职责。一点点事情，就足以让我们惊吓半天。

　　所以，我们连考试也害怕。

　　这就是成长的悲哀。
