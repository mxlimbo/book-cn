水一样的湘西，水一样的凤凰

　　"五一"期间，有一条旅游线路，叫湘西三日游。从张家界出发，一路欣赏土家族的吊脚楼、漂流猛洞河、作客苗家、观赏矮寨天险、最后到凤凰古城，参观沈从文故居。据说挺火的。

　　说穿了，这条旅游线路就是奔着沈从文的名气开设的。

　　一直喜欢沈从文，《边城》、《长河》等文章是一方面，更多的是他的性情和经历。沈胜衣在《关于粱朝伟的几则笔记》中，写到粱朝伟最喜欢的作家是沈从文。对此他暗自点头。说这份文学趣味正符合多年来他对粱的好感和印象。

　　喜欢沈从文的人很多。但沈从文在文学史上的地位却曾经很模糊。

　　印象中的沈从文，个高、瘦弱、身体单薄，长条型的脸上，始终是优雅的谦逊的笑。在那个时代那一批文学人中，沈从文应该算是较英俊的一位了。

　　说当年，沈从文追求张兆和时，大家闺秀的张，还只是一名在校的学生，对师长辈的沈，谈不上喜欢，也不知该不该拒绝。张兆和的姐姐张允和，从中撮和。她帮妹妹拿主意，又给沈通风报信。后来，沈向张家提亲。张允和替妹妹出面，向父母亲开口，一说即成。她马上跑到邮局打电报报喜。那时打电报，讲究用文言，不用大白话。电报要字少、意达、省钱。允和想来想去，终于悟出一条妙计，发了一封一个字的电报"允"，即表达了家长同意的意思，又署了发电报人的名，一举两得。这在后来传为美谈。（见张允和《最后的闺秀》）

　　尽管沈从文如愿以偿，娶到了有"黑凤"之称的兆和（黑又俏的美人），但从其它有关他们的文章看，好像沈从文婚后的家庭生活并没有想象中那么甜蜜和温馨。战乱时，他们分隔两地，兆和照顾小孩，不能赶到沈从文的身边。沈非常失意，写了大量抒怀感慨的信，有埋怨也有自责。据后来一些文章，说是沈从文和张兆和在政治上观点不一，而且，在战时，沈曾经对张有过欠忠诚的举动。一度，张想和沈离婚，最终没离成。而在建国初期，沈没有融入到革命热潮中，作品没被承认，在文艺界很孤立，许多知名的文学家都撰文批判过沈，包括丁玲。沈一度自杀，被抢救过来。后来，他就一直当博物馆的讲解员，给文物贴标签。干了很多年。文革中，派去扫女厕所，扫了一年多。

　　人生就这么坎坎坷坷地过来。一直到八十年代，沈的作品在国处倍受重视，国内才开始注目他。那会，《边城》也被改编成电影，第一次编剧没编好，沈不满意，导演重换成凌子风，才拍成。电影厂送来版税，两次都被沈退还，一来表示自己的不满意，二来表示自己给大家添了麻烦。再后来，沈转向研究服饰，也有了成果。那时，兆和愿意自己住，每天，沈要走一里多的路，到兆和那拿做好的饭。

　　纵观沈的一生，文学造诣是没的说的。至今，许多文学批评家都公认，沈从文是我国乡土文学的开山鼻祖。西方学子金介甫写了一本《凤凰之子－沈从文传》，评价是，沈作为教师、导师，对很多小说家、诗人产生了重大的影响。并且在四十年代和八十年代，影响过两代的湘西乡土作家。

　　沈的婚姻家庭生活，却少有人提。

　　不如允和和周有光。这对现已九十多岁高龄的老人，每天讲得最多的话是"快乐极了"、"得意得不得了"。众所周知，周老先生是《简明大不列颠百科全书》中文版三位编委之一，著书立说的立意之高、之雅自不必说。张允和则是著名的昆曲研究者。两人喜欢不定期地请一些"亲爱的"到家里参加"一壶酒、一碟菜"的"蝴蝶会"（曲友聚会）。弹弹唱唱、说说笑笑，不亦乐乎。叶稚珊说，此景只应天上有……

　　不如黄苗子和郁风。活泼开朗的郁风，当年也看不上又矮又胖且有国民党背景的苗子。后来，阴差阳错，两人好了之后，却能风风雨雨一路相随相伴，始终微笑面对人生。去年电视台采访他们时，苗子和郁风都是一副幸福的样子，回忆起往事时两人互相补充，切切深情溢于言表。

　　也不如钱钟书和杨绛。尽管很多人屡屡把钱杨与《围城》里的方鸿渐、孙柔嘉和唐晓芙对号入座，但热闹之后，这对大智大慧的文坛俪伉终是相携相扶走过了大半个世纪。

　　有时翻开沈和张的照片，我总为他们扼腕叹息。本是一对璧人，却是不尽人意（我的感觉）。

　　很多家庭看似安安稳稳地过日子，波澜不惊。个中滋味，却有谁知？名人也逃不出此命运。
