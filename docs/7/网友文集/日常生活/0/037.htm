没了椰风椰影--海口印象

　　 到达海口的时候，夜色已浓。朦胧中，只见椰影婆娑，凉风微拂，心情开始愉悦起来。

　　这就是海南了。人们都说，到海南养养颜、润润喉、开开眼，想必我也开始了此征途。

　　吸一口空气，果真甜丝丝的，好像掺了椰汁，原来空气都是甜的。对海南，我的期望值在无限提高。

　　进入市区，人车开始多起来，道路不是很宽阔，街边的建筑也看不出什么整体风格。只是路边的小摊小铺，让人觉得海口是一个以本地人为主流的生活的城市。

　　尽管是晚上十点多了，我还是饶有兴趣地想到处观光。

　　坐在车里不觉得什么，一旦自己徒步走在街上，却发现了很多不是椰影椰风的地方。

　　想找一家象模象样的酒吧或咖啡馆吃点东西。刚看到一家比较亮堂的海鲜店，却发现门口集中了一批穿着像是工商执法制服的人员在盘问什么，大家说出事了出事了，不过去了。又掉头往前走，来到一个大排档区，热闹得很，有卖田螺的，有卖烧烤的，还有火锅……我的兴致提了起来，直喊："吃田螺吃田螺"。刚想近前问价，却见又一批工商执法人员走来，用本地话叽哩呱拉地说一通，一会，卖田螺的摊主有点不情愿地端出几盆田螺，倒进一个大塑料袋，工商人员拎着，扬长而去。我们愣愣地看着，半天才明白过去，原来是清理乱摆卖的工商人员在执法，收取了好处费的，则相安无事，还是该干嘛干嘛。

　　怪不得，说工商部门有权，夜色下，只是这样一个小小的角落，却也能看个明白。

　　没吃成饭，走在街上，心却提了起来。到处是横冲直撞的摩托车，喇叭按得天响，让你避之惟恐不及。怕被撞是一方面，怕被抢是更主要的一方面。瞧那些摩的车夫，个个面孔黝黑，眼神贼亮，我只能紧紧地抓着背包，担心把回程机票也给洗劫了。

　　水果摊始终吸引着我，那些怪模怪样的水果很能满足我猎奇的心理。终于忍不住在一个叫人参果的档子前停下，讲好价格，摊主开始把秤，模糊中，他告诉我："四斤。"我一拎，感觉不像，好像只有三斤。但摊主凶巴巴地冲我说，你可以借隔壁的秤用，少一两不要钱。我在他眼光的威慑下，只能乖乖掏钱，提着水果潜逃。刚走出一段距离，只听后边一挑担的黎族妇女追了过来。我吓了一跳，以为出了什么差错。

　　这位妇女拉着我比划说，刚才那个摊主的秤是八两秤，不是一斤秤。说四斤实际上只有三斤二两。

　　我很生气，反驳她为什么当时不揭穿，她说大家做生意，不可以互相揭底的。说罢，要求我也买她一点水果做为奖赏。无奈中，我只能重新掏钱，又买了四斤人参果。

　　酒吧没找着，饭没吃成，浓浓夜色中，提着重得很的水果往回走。一路避着摩托车，还要在路边打扮得花枝招展的"小姐"中穿行，天躁热，走得出汗，没了情趣，心里嘀咕："这就是海口呀！"刚下飞机时的那种愉悦已丧失贻尽。
