这不是胡闹吗

　　昨晚，一群人奔"物质生活"书+酒吧，泡到二三点，吃了五十串羊肉串，不过瘾，转移地点，奔鸡煲去。

　　六个人，四男二女，明显阴阳不调。男性们大喊不公平，必须另招两女才平衡。正好座中一人邓公暴露了自己下午曾接受湖南卫视的采访，于是义不容辞地担当起招女的重任。邓公离座，拿着别人手机躲到远处树底下窃窃私语。

　　座中余人纷纷不怀好意地盯着邓公的背影，算计着能否阴谋得逞。有人提议打赌，看是否有靓女中招。猜中者免酒，猜不中者罚。于是，两大阵营立即形成。

　　反方阵营说，邓公需电话半天，可见难请，人不来也。正方阵营立即反驳，并命服务生多加两座，以实际行动迎接贵客。

　　一会，邓公回来了，表情并不畅快，嘴里却慢慢吐出：一会就来。好像保险系数不高。

　　反方阵营抓到把柄，拿出表要求以半小时为限。

　　湖南卫视的靓女成了午夜一群游荡的人的期盼。

　　喝酒，数时，进入了倒计时，人却不见影。又进行电话指引。正反方互相攻击，喝酒。

　　终于，最后时刻，来了。是两女一男。

　　女的不算靓，男的却又破坏了阴阳平衡的格局，可见是不受欢迎的人。

　　反方输了，正方要求罚酒。反方不认账，说是来三人，不是两人。苦苦盼来的客人被晾到了一边，正反方趁着酒性自顾自地斗起来，什么"你胡说""你他妈的放屁"……

　　地主们斗得性趣盎然，笑逐颜开，粗口无遮拦。可怜邓公一旁无比尴尬，只能东劝劝西劝劝，想尽快息事宁人。

　　半晌大家重归平静，又开始敬酒，递名片。对着客人，黑夜掩盖下，没有了作脸的必要。你一句我一句地戏说起邓公来，"有媒介采访你，你就一个人搂，不管其它兄弟了……""你也太不讲义气了。"

　　邓公直冒汗，私底下暗瞧客人们的反应。

　　还好，尊贵的客人没多长时间就走了，说是第二天七点要拍外景。

　　人走后，邓公沉下脸，一个劲地埋怨大家，你们这叫什么呢，不是瞎胡闹嘛，要闹，自己人关起门来随便闹，不要在外人面前这么肆无忌惮嘛，要不，人家还以为深圳的文化人档次都这么低！一点都不维护深圳文化的形象。

　　哈哈……"我们还以为来的是李湘呢！""李湘，又算什么！""李湘算个屁呀！"哈哈……

　　刚刚被采访的邓公还陶醉在高大全里边，没出来，所以他着急呀！
