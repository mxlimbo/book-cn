偷窥

　　 回到家，我总喜欢把所有窗打开，把所有窗帘拉开，把所有灯按亮，把电视声音拧大，让屋外的人气、声浪进来，让里外交融，自己不再孤单。

　　住的房子是那种楼房，前楼挨着后楼，换句夸张一点的话说，前后楼的人可以隔着窗户拉手。

　　一天，同事来做客。一进门，就大声嚷嚷："快把窗帘拉上！"

　　"干嘛？这样多好！"我不解。

　　同事郑重其事地说："你不知道现在深圳最好卖的是望远镜呀！"

　　天呀？！

　　不会吧！我一边心里自我安慰，一边赶紧反省一下平时在家有否当众出格的行为。还好，我的窗帘是两层的，哪怕有人偷窥，还有一层薄纱可以抵挡一下。

　　想想同事的话，真的还是有点道理。

　　本来嘛，人的内在就存有偷窥的本能（抑或是好奇的本能）。不说别人，我自身好像都有过这方面的"劣迹"。平时在家，我总是有意无意地朝对面住家的阳台看，曾经有一家就引起我的注意。那是一对年轻夫妇，如果太太在晾衣服，丈夫肯定会在旁帮手，递一下衣架、或把衣服甩甩干。太太晾完进屋了，丈夫还顺手拿起扫把把阳台扫一扫……这种感人的场景我看过几次，对这一家无限羡慕。

　　将心比心，我在看别人的同时，对面人家可能也在看我。还好的是，我只是肉眼无意地看看而已（因为阳台的视野就这么大，很难再挖掘出别的风景），还没发展到用望远镜去窥探别人的隐私。而别人家会怎么样，我只能是不得而知了。

　　由此我也感慨万分，现代社会，人与人的隔阂越来越大，人情事故越来越淡漠。楼房里的隔壁住家会老死不相往来。但渴望沟通和交流的人们，却会换种方式，花大把钱，浪费大把时间，在忙完一天工作，已经累得半死的状态下，还能拼着命上网，找不知面不知性别不知年龄不知底细的"网人"，吭哧聊个没完。这还不够，还要买个高级望远镜，架在家里隐蔽处，偷偷地窥探对面人家的一举一动……

　　高速发展的社会，让人也变成了奇怪的动物。
