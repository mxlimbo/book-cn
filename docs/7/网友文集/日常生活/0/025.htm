街上盛开玫瑰花

　　今天是情人节。

　　路上看得最多的是玫瑰花。

　　上班路上，看到一群上学的小孩子，每人手里一枝玫瑰，嘻嘻哈哈地打闹着。心里不禁哑然失笑，现在的孩子，了不得!

　　在办公室，有个花店的送花小伙子，捧着一大束玫瑰，找某某人签收，把全部人羡慕得大声高喊："喂喂，没送错吧!是不是我的？!"

　　下班等在路旁打的，看到有抱着花走的娇羞甜蜜的女孩，本不怎么漂亮，但手中的玫瑰，却让全世界的人知道了她是最美丽最幸福的人。

　　看着送花人，一手抱好多束花往写字楼走，每束花上都有一张条，可能是地址，也可能是献花者的寄语，不禁心存感动。

　　在饭店，还没到吃饭高峰期，却已是客满，不得不加座。服务小姐说，很多都是提前订位的。

　　因为情人节，有情的人庆贺时往往一掷千金，无所不惜。爱是最重要的，钱又算得了什么？

　　一个可以说是普通的日子，但因为有了玫瑰，有了爱情，所以它成了隆重的节日。

　　这就是情人节。

　　今天特别冷，但玫瑰特别地斗艳。

　　愿玫瑰天天盛开，愿天天情人节，愿天下有情人终成眷属。
