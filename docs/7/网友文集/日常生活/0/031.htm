那种滋味，你尝过吗？

　　今天可以睡个懒觉，没有工作。

　　不用上班的日子是多么温馨。懒懒地躺在床上，暖暖的阳光已照了进来。心里盘算着起来做点什么。看着表，已经快十一点了。那就起来吃点饭，看看午间新闻，再到书报亭买几本新出的杂志……这么想着，翻翻身，不禁觉得浑身酸痛。呵，都是昨天该死的倒桩，落得如此后遗症。

　　想起练倒桩，我心有余悸。

　　才一个多小时，我就被方向盘折磨得死去活来。教练师傅不停地在边上喊："打左打左，两圈。""回正，快！""打死右，两圈，听见没有？"师傅话音还没落，只听"轰"一声，练倒桩的塑料墩已被撞倒。我猛地急刹，吓出一身冷汗。车的轮胎还没回正，而师傅的脸则像天上布满的乌云，阴森得让人害怕。我等了半天，幸亏没挨骂。"倒档，后退，回正一把。"师傅不看我，闷声喊着，一副恨铁不成钢的样子。

　　可怜我，体重只有88斤，却要跟一笨重无比的方向盘搏斗。眼看到了标好的位置，要快速急打方向盘，脚下的离合器还要稳速配合。不能死火，不能过快。这是师傅的教导。但具体实施起来，却不是简单的事情。首先，看着标志，在猛打方向的时候，胳膊就不听使唤，本来两下搞掂，我做起来就要三四下，手使劲，胳膊使劲，肩也使劲，腰也使劲。脚还得踩着离合器，不单脚盘使劲，脚底也要使劲，腿也要使劲 ……如果这些身体部位都能使上劲也行，关键是，手使劲了，脚却松了；等脚踩实了，手却转不动了。如此重复，车才蹒蹒跚跚地进了停车位，不是太靠左，就是太靠右，不是顶了后边的塑料墩，就是冲出前边划好的线。

　　师傅是一边气得话都不想说，我则气得涨红了脸。当然，是气自己。

　　下星期就要桩考了，就这水平，哪成？

　　我知道师傅心里着急。但这话他还是留情没说。

　　练一个半小时，我再也没力气了。师傅提前收工。一路没话。

　　晚上回到家里，我恨不得去桑拿一把，捏捏筋骨也行呀。

　　今天我正好没有应急的事，可以好好放松胳膊和腿。磨磨蹭蹭起了床，洗漱完毕。从冰箱里倒了杯鲜橙汁，懒洋洋地躺在沙发上，打开电视看新闻。新闻上说，钱其琛副总理正跟鲍威尔谈两岸问题。"嘟嘟"手机响，"喂？"我打开一听，"我是李师傅呀，你下午有空吧，我们两点学车……"

　　呵……

　　那滋味，你知道吗，就像从高空跌入低谷，从火焰山掉进冰窖……
