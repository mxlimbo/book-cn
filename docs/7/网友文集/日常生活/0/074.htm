好女孩上天堂，坏女孩走四方

　　太热了，简直是今年以来最热的一天。拉开窗帘，外边像烤焦了一般，凝固的空气让人窒息。据说，小区里的游泳池五一开放，水里会凉快些。但折腾过后，还是热！

　　还不如到书城。

　　新书架上，没见几本新书。倒是书评周刊提到的"新新女性情调散文书系"那几本书名用手写体写成的书赫然亮相。捡了一本水果的《让灵魂摇滚》，文字顺畅，但通篇隔三岔五地就会来一句：我是个美丽的女人……我长得像猫……如不是熬夜我会更加漂亮……让人忍无可忍。有自恋倾向的水果，怎么就不知道：美丽的女人从来不气急败坏歇斯底里声嘶力竭地到处标榜自己的美丽。

　　也是，现在流行美女写手，个个闪亮登场，不单文美，人更美。因了人美，文的美与不美或欠美也可退居其次了。俗话说秀色可餐，有了可餐的秀色，文色算什么！打开一本本制作精良的美少女书，尽管笔调敏感、温婉、清澈，诉说着个人情绪、感悟，但轻得让人一拿就想放下。

　　有个叫文京的作者（不知是否是笔名）说得好：池莉的小说之所以有一种世俗层面的好看，源自于她的某种果敢率真的文气，有一种真正踏实的市民气或坦然的小市民气。武汉街上常常骂声嘹亮，但骂人有时却能显出一个人的真，而夸人却常常会像矫饰。骂人不好，但真实好。

　　我喜欢真实，不喜欢娇情和扭捏作态。如果说，这些靓靓的新新女性中，耐心地斯文地讲述着自己经历和感受的女孩是"好女孩"的话，我却更欣赏那些字里行间我行我素、富有江湖气、话无遮拦不忌粗口、在性方面大懂特懂的"坏女孩"。

　　在书城，我上上下下找寻陈彤和陈幻的书，未果。

　　《没有人知道你是一条狗》和《没有人知道你打算坏多久》，打出了陈彤"坏"的旗帜。现在，她又出版了《风情万种》，好像有所收敛有意改邪归正，但骨子里的"坏相"却根深蒂固。比如，"我在发现如果我再不结婚可能就永远也不能结婚了的时候，临危不乱处变不惊，在一夜之间生擒了我的老公，并且以最高效率一锤定音，我在完成与一个素不相识的人喜结连理百年好合的过程中可以说是表现得大智大勇完美无缺……"哈哈，后边还有更绝的："是的，我是在一夜之间活捉了我的老公，他像一个在森林里迷路的小动物，肚子饿了正打算找点吃的，我是女巫，摇身一变成了一棵长满诱人果实的树，于是他吃了我的果子，然后在树上挖一个洞，住进去了…… "啧啧！

　　好一个陈彤，不想证明自己是淑女，非得把男女关系写得通透直白，一副大无畏的精神和胆识！

　　19岁的陈幻写了本《偏要是美女》，有人说是《恶之花》的中文版。她写爱情：爱情不是对某一个人日出日落地厮守拥抱，而是无数个可以带给你奇妙感受的瞬间。这注定了人是要花心到地老天荒的。"字句鲜活灵动，却挡不住内在涌动的邪气。

　　不管"坏女孩"能有多坏，至少鲜明真实，能让人忍俊不住，或击案叫绝。

　　前两年有一本书叫《好女孩上天堂，坏女孩走四方》，互不得罪。是好女孩的，留着上天堂；坏女孩的，至少可以走四方。经历过了，熟知过了，定会返璞归真。也许这样的生活才叫精彩。

　　新书收获甚少，还是在旧书架上，拿了一本祝勇的随笔，一本董鼎山在《读书》10年间专栏文章的集子，还有一本喻丽清的散文。心里多少清凉了些。
