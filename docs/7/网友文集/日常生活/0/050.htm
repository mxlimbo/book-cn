邮箱如"和平号"般坠入太平洋

　　我有两个邮箱，一个是公共的，一个是私人的。公共的，每天有大量的来信，读完，有的留起来了，有的当即删除，不占内存。私人的，目前濒临作废。

　　私人的邮箱我几乎不打开了，因为，每次打开，都会让我失望。那种零记录的感觉让人难受。

　　没有私人信件，等于说别人不愿意跟我私底下交流，甚至等于说我没有私人朋友。而这个结果，肯定是我造成的。

　　曾经，邮箱也有它辉煌的时候，一天几封，不管内容如何，有人惦记，总是一分欣喜加感动。后来，信件越来越少，到现在，邮箱将像"和平号"一样，完成它光荣的历史使命，顺利坠入南太平洋，噢，不，"和平号"坠入太平洋，而我的邮箱，却是黯然坠入我的内心，让我深思反省。

　　我是一个什么样的人？尖酸刻薄？歇斯底里？过敏？神经质？自怨自艾？小鸡肚肠？心胸狭窄？斤斤计较？无理取闹？呵，……所有能想像的词汇我都用上了，这会是我吗？

　　非也。我自己不承认，身边认识我的人，也绝对不会这样认为。真的，多么不可能的事情，除非太阳从西边升起。

　　可是，怎么没人愿意跟我通信呢？

　　我行文都很中规中矩的。抬头必有称呼，称呼之后必有问候，问候之后必有内容，内容即便不是洋洋洒洒，情意绵绵，至少也会言简意赅，通俗易懂呀。末了，还不忘落款，署上自己的名字。就是有时忘了把写信时间标明，如有瑕疪，也仅此而已。复信，向来也及时，都是有来有往。

　　但是，就是没人来信了。

　　有时，我实在按捺不住寂寞，求QQ上的至亲好友给我写信，但对方就是不肯。追问半天，才勉强回答，不知说什么！

　　怪了！帮我反思的机会都不给。

　　我只能冥思苦想，究竟是哪个环节出问题了？网管故意刁难我、私下截取邮件？还是我的邮箱自身有极强抗体？嗯，都不像。再说，我的邮箱开通才一年不到，决不像"太平号"那样服役了15年，年事已高，俄罗斯政府负担不起每年2亿美元的经费，才不得已忍痛让它向历史谢幕的。我的邮箱是免费的，而且正当壮年，身强力壮，是干事的时候，现在却提前让它下岗，多可惜。

　　痛定思痛。我只能作如下解释：人都有多面性，社会人、个人。网上、网下。我也不例外。别人也不例外。在邮件里，我可能表现得真实了一点，有什么说什么，喜的悲的，善的恶的，都毫无遮挡地表露出来，直把收信人当自己人。对自己人，还有什么需要情礼或面具的？有气，尽管发吧；有恨，尽管撒吧。反正，自己人能理解、能宽容、能忍让。可怜的我，这么想，也就这么做了。

　　殊不知，通过邮件这根线连着的，还是现实的双方。以个人对着社会人，怎么行呢？以网上的方式对着网下，又怎么行呢？

　　不言而喻，我的作为有了立竿见影的效果。

　　邮箱，只能坠落了。曾想敞开的心，也就这么关上。
