喜欢毛尖

　　看好了，我说的是，喜欢毛尖，不是喜欢喝毛尖；毛尖是人，不是茶。

　　我也不认识毛尖。却发自内心地喜欢。

　　知道毛尖，源于《万象》。一本精致的小32K本月刊杂志，辽宁教育出版社所属，顾问有柯灵、陈原。

　　原来看《万象》，是想找董桥，找思果，后来，看到了凯蒂，看到了毛尖，一期一期的。我开始每月找《万象》，找毛尖。董桥、思果成其次了。

　　在《万象》里，毛尖只是一介作者。没有专门的介绍，没有更多的关于他个人的资料。从行文内容和方式，我直觉他是个三十多岁的专写影评的男性作者，是不是作家，现在还不知道。(注：后来得知，毛尖为一女性作者，是陈子善的学生，很傲慢的样子，别人交换名片，她则不理不睬。)

　　喜欢的，是毛尖的用词和表达。在2001年三月期《万象》里，一篇关于影星的看，看得我如同吃了顿大餐一般心满意足、如同蒸了个桑拿一般畅快淋漓、如同听了一曲优美音乐般如痴如醉、如同拾取失而复得的心爱古玩般感激涕零……那种意外之喜，简直言语无法表达。

　　他说梦露，站在地铁通风口上，白色的连衣裙随风扬起，她急急忙忙地用手去捂，同时抬起头，风情万种地看了一眼摄影机。注意了，"梦露把摄影机当男人来撩拨，把所有的观众都在她的这道注视里变成了她的臣民，银幕布里的人和银幕布下的人立即处于一种准性爱关系中……"看看，毛尖式的语言，诠释得让人心慌。

　　不过，梦露的看还不算经典的看。真正经典的看，影史上有三次。我想，这是毛尖自已的归纳总结了。历史的东西，只要没人跟你较真，只要你说得有理，大家也就听之信之了。

　　一次是褒曼，在《卡萨布兰卡》电影里，由于导演和编剧的矛盾，她无法预知剧情的发展，不知自己究竟该爱男主角还是次男主角，所以，整个影片中，她只能以一种游移的、举棋不定的眼神去看，而这种看，主宰了一部影片的命运，赢得了第十六届奥斯卡奖。从此，"褒曼带领一代好莱坞人这样'看'"。

　　一次是海蕊耶在《莫尼卡》里的看，戏里戏外的看。像所有好莱坞传奇一样，19岁刚出道的漂亮女演员海蕊耶，与34岁的导演擦出了爱的火花，并迅速燃烧。当电影快拍完时，戏外导演的眼神越来越绝望，而看着情人的海蕊耶的注视，也意味深长，那一注视，被称为"一种无可挽回的分离力量"、"一个电影时期就此结束"。毛尖把海蕊耶的注视自作主张为"我要走了，现在轮到你作选择了!"尽管是自作主张得霸道，却不无道理。

　　一次是混血女星阿佳妮在《阿黛勒雨果的故事》里的看。毛尖把阿佳妮的美描绘成是"一种非法的美"、"贵族神经质和迷人的错乱的美"，说她的看，"令人精神错乱 "，说阿佳妮看的镜头，对观众是一个考验：或是留在一个安全的地方老去，或是进入她的王国，燃烧。观众得自己决定去留，或是分担角色的命运，或是闪开。语言能用到不负责任到这个地步，真是绝了。

　　不管毛尖的归类是否准确，这经典的三次看，在电影史上是有目共睹的。女影星的看，固然美仑美奂，我却更喜欢毛尖的"看"--賊贫賊坏的入木三分。
