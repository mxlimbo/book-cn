七

　　徐昆掐准了时间赶到了"蓝月大酒店"。他知道妻子早就等候在咖啡厅的桌边了。他不想和她呆的时间过长，连他自己都搞不清是什么原因。

　　妻子今天换了件紫色套裙。那还是前年秋天，徐昆到北京出差给她带的。妻很喜欢这件衣服，说穿了无形之中有了富贵气。她平日里不舍得穿，只有在重大节日或者是重要场合，她才昙花一现地穿穿。然后，再小心地洗净熨好收起来。徐昆的印象中，妻总是一身蓝色套装，好象随时准备给人作她钟爱的思想政治工作。

　　妻看见徐昆，欠起身，犹豫了一下，最终还是没站起来。只是抬抬屁股，又忙不迭地坐下了。她给了徐昆一个歉意的微笑，似乎想弥补自己的礼貌不周。

　　"你还是那样守时。"

　　妻指的是恋爱时的徐昆，总是提前到达约会地点。而今天，徐昆听出了妻语气中的不满了，好象从徐昆的准点赴约看出了他的应付态度。

　　"我忙。"

　　"喝点什么吗？"

　　"没时间了，有事说事吧。"徐昆直起背，好象谈话一结束，他就要忙着离开似的。

　　"你真的只给我五分钟时间？"

　　"是工作只能给我五分钟。"徐昆说，"我现在是身不由已呀！"他无奈地摇摇头，"要不来杯咖啡？不过我希望你能简短捷说。"

　　"我只想问你一句：为什么一个多星期不回家？"

　　"我真是太忙了。"

　　"骗谁？回家又不是出国，不过就三分钟的时间。"

　　"可是，家务事对我情绪的影响，决不会是简单的三分钟。"

　　"你这是逃避，是不负责任！你是一家之长，你不能对这个家不闻不问。"

　　"别说了"徐昆挥挥手，"我又没有三头六臂，哪能面面俱到？当初嫌我不求上进的是你，嫌我混不上干部混不来房子的也是你，可是当我拼死拼活的挣来你想要的一切，你又嫌我不顾家。我算是明白了，即使我累死累活也不会让你满意的。"

　　"难道望夫成龙也是我的过错？我可是全为了你，为了这个家。"

　　"我要不是为了家，还不"

　　"还不早就另觅新欢是不是？"

　　"你不要无中生有。"

　　"无风不起浪。你那些事，外面传得纷纷扬扬。我就是个聋子，耳朵多少也会进点风的。"

　　"我很失望，你竟然也这样小市民？"

　　妻直直地望着徐昆慌乱的眼神，说："但愿我是庸人自扰。我有句话放在这儿，有则改之无则加勉。常言道，纸包不住火。要想人莫知，除非已莫为。"

　　"你威胁我？"

　　"我是忠告你。我劝你不要自绝于民。只要你能悬崖勒马回头是岸，我们还是欢迎你的。"

　　"果然不愧是纪委书记，大道理一套一套的。"徐昆不屑道。

　　"听不听由你，反正我是给了你一次机会。作为你的妻子，我还有必要提醒你一句：玩火自焚，多行不义必自毙！"妻一口喝掉咖啡，抹抹嘴，头也不回地走了。
