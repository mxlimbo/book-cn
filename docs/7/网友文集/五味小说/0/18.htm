九

　　A市公安局禁毒科第XX号卷宗

　　案件名称："杜冷丁"专案组

　　犯罪嫌疑人：

　　徐昆：男，35岁，市人民医院神经内科主任，副主任医师……

　　去年10月与张枫勾搭成奸，经常虚借治疗的名义开杜冷丁供姘妇张枫注射……

　　没有发现此人有贩卖或注射杜冷丁的迹象……

　　张枫：女，30岁，蓝月大洒店"简.方达健身房"教练……

　　曾与某男同居两年之久，并生有一女，后某男弃其而去，孩子现雇人喂养……

　　其注射杜冷丁始于认识徐昆之后，并有贩卖杜冷丁嫌疑……

　　络腮胡须：男，年龄姓名职业不详，怀疑经常从张枫处购买杜冷丁，用途有待查实……
