又 见” 泰 坦 尼 克 号“


六

　　翌日清晨，天飘起了细若发丝的小雨，天地间织起密密的网，罩得人透不过气来。姐姐一脸忧郁地自言自语："左眼跳灾还是跳财，我怎么老感觉不对劲。"

　　姐夫往外走的时候，冲姐姐笑道："别老疑神疑鬼的，兴许是做家务累的。"

　　莫荷关心地替姐姐号了脉，量了血压，毫不犹豫地证实了姐夫的判断。她轻轻地抚摩着姐姐的胸口，说这样可以放松情绪。她开玩笑说："你可不要把姐夫给宠坏了。咱们女人吗，就是要对自己好一点。"

　　姐姐苦笑着摇头。半晌才幽幽地说："你们今天不要出去了吧。我不放心你们。"

　　囡囡上来抱住莫荷的腿使劲摇晃，扑闪着明亮的大眼睛说："阿姨，今天咱们玩家家吧，你当妈妈，舅舅当爸爸，我当"她歪着头，眯起眼睛，调皮地笑了，"爸爸妈妈的小可爱。"

　　姐姐烦躁地冲囡囡摆摆手："囡囡，别闹。"

　　杨军已背起药箱，站起来冲姐姐说："姐，你今天怎么了。不舒服就休息会儿。我们早点回来陪你。"

　　莫荷也安慰姐姐："不会出事的。天气凉凉爽爽的，是个难得的好天气。"

　　姐姐说："那你们早去早回。不要让我担心。军子，你要照顾好莫医生，她要有个好歹，我不饶你。"

　　杨军冲莫荷眨眨眼，双手亲热地按按姐姐的肩头，佯装生气地说："姐，看你对她那么好，我都吃醋了你没以前疼我了。"

　　姐姐轻轻拍拍他的手，有气无力地说："看你永远都长不大了，也不怕莫医生笑话。"

　　石村果然名不负实。莫荷目之所极，简直就是石的世界：石板路，石头桥，石磨，石桌，石凳，猪吃食用的石槽，石砌院墙，石砌房子，连房顶也是石板铺就……雨丝飘落到石板路上，比轻盈的雪跑得还快，眨眼之间融化得无影无踪。但细心的莫荷还是发现了石板上雨丝的脚印：清晰细密的石板纹理，被水滋润得丰盈饱满，如婴儿透明的血管。杨军图凉快，也不听莫荷担心会传染上细菌的劝告，赤脚在石板上又蹦又跳，还劝莫荷也脱了鞋，"爽！脚底上就象虫子在挠痒痒。"

　　莫荷被石板上绚烂多姿的花纹迷住了，低着头欣赏着变化无穷的美妙图案。这些图案比雨花石上的图案线条更流畅匀称，乍看起来颇象从古碑上拓下来的篆书的笔划。隔着石板上渐隐渐现的水膜看，那些生动的线条仿佛游走的青蛇，行姿婀娜，媚态十足。

　　"这是大理石吧。"莫荷瞪大了眼睛，"城里人现都时髦用大理石装修房子呢，价格贵的吓人。"

　　杨军不以为然地说："在这儿一钱不值。"

　　莫荷惋惜地"哎呀"说："真是不可思议。"

　　杨军漫不经心地说："石村人一生与石为伴，从出生到入死，等于说是一个会说话的石头。连人的名字，也都与石有关。听说，石村叫石头的有几十个，不但外人搞不清楚，连本村的人都会混淆。呶"他抬起胳膊，尽力指向远处山包上一片白花花的坟墓："那墓地里的石柱坟上，都镌有名字，清一色的'石头'。时间久了，别说外人，就是后代也没法找到自己的祖坟。山里人说，生是一村人，死是一村鬼。干脆放在一起祭祀得了。"

　　待走近那片坟地，莫荷大吃一惊。每座坟都用雪白的大理石砌成圆柱体，约一米粗，两米高。微风细雨中，显得庄严肃穆，仿佛一个个巨大的感叹号。莫荷悲戚地想，短暂的人生原来可以用一个简单的符号来概括的呀！杨军指着那一片石柱说，一群站着的灵魂。

　　"站着的灵魂？"

　　"这里石多地薄，人们视土地为神灵，就连死后也不想浪费过多的土地，所以都采取站姿。"

　　莫荷说："真是奇特的风俗。从生到死，都可以听风看雨，无言关注着子孙后代的喜怒哀乐。"

　　杨军轻轻捂住莫荷的双眼："别乱看。听老辈人讲，鬼魂最喜欢阴雨天出来散步的。如果让鬼撞见，是要折阳寿的。"

　　莫荷望望墓门前那对沉思不语的石狮子，不由自主地想起了杨军姐姐那双忧郁的眼睛，心中一激灵。只觉得脚步被一种神奇的力量吸引着，鬼使神差地一步一步走向石狮子的四个深深的大眼窝。

　　杨军诧异地望着呆若木鸡的莫荷，叉开手指，伸到莫荷的眼前晃晃，见莫荷丝毫没有反应，吓得头上冒出了一层密密的汗珠。他哑着嗓门，对着莫荷的耳朵大喝一声："哎！！！"

　　莫荷剧烈地晃了几晃，无力地歪倒在杨军的怀里。她揉揉眼睛，勉强地笑笑，梦呓般地喃喃自语："我真的遇见鬼了。"

　　杨军望望天边一团正迅速向头顶飘移的乌云，拉着莫荷就走："快点，大雨来了。看来真应该听姐姐的话：今天不宜出行。"

　　他们刚跑进石村村委会大院，闪电也追到头顶。也许是遗憾没有让他们见识一下雷公的利害，气急败坏地大施淫威，在房顶上"嘁哩咔嚓"地轰炸不休。雨借雷威，脚跟在房顶石板上狠命地踏着，震得耳膜嗡嗡作响。村长笑哈哈地对他们说："老天也凑趣，放礼炮欢迎你们呢。"他紧紧握住莫荷的手，仿佛久别重逢的老朋友："这么大的雨，改日再来就是了。"然后用征询的目光望着莫荷："要不？先歇会再工作。"

　　莫荷已打开药箱，说："现在就开始吧。天不好，查完了我们早点赶回去。"

　　村长点点头，说："那也好。"他走到墙角麦克风前，准备在村里的有线广播上发个通知，召集村民到村委会院子里来检查身体。莫荷忙制止村长，说等雨停了也不迟，这么大的雨会淋出病来的。

　　今天的检查很不顺利。第一个接受检查的中年汉子，面色腊黄，捂着肚子说痛了很多天了。莫荷怀疑他患的是肝炎，劝他到乡医院作进一步检查，不要延误了治疗。那人一脸沮丧地叹口气，说查出来也是白搭，哪里拿得起医药费。莫荷着急地说："想想办法吗，钱还能比命大？"她安慰他说，"你用不着太悲观，说不定一场虚惊。"那人勉强笑道："我才不怕死呢。到山上住石塔，惬意着呢。"

　　后来，有个满脸自豪的妇女领着一个中学生模样的女孩子走到莫荷面前，说她闺女刚刚接到通知，已经被县卫校录取了，后天就到县里体检。她愉快地接受着人们的祝贺，说等正式通知书下来了，请大家伙喝酒。她对莫荷说："先请你为她预检一下，如有不合格的，也好抓紧想办法。"

　　莫荷手往女孩子的手腕上轻轻一搭，就感觉有点不对劲。她神色凝重地戴上听诊器，认真地听着女孩子的心跳。站在旁边的母亲眼睛紧张地盯着莫荷的脸，莫荷刚放下听诊器，她就迫不及待地问："怎么样？大夫，没啥吧？"

　　莫荷难为情地望着快要哭了的母亲，字斟句酌地说："心跳不齐。我怀疑是先天性心脏病。"

　　母亲的脸刷的白了，瞪大了眼睛，扳着莫荷的肩头发疯般地摇晃："你是个晦气鬼，你胡说呀！"

　　村长上前使劲掰开被突如其来的不幸打击得失去理智的母亲的手："艳艳妈，你冷静点。"

　　母亲抓起呆若木鸡的女儿的手，跌跌撞撞地冲了出去。

　　望着母女俩消失地雨中的背影，莫荷后悔不该捅破那层纸，也许晚一天知道，她们还能多一天的幸福。

　　有人摇头叹息："真是可怜呀。石村的第一个中专生，就这样被耽误了。"

　　"是呀。眼看孤儿寡母的快熬出头来了，却又......"

　　检查结束时，天近傍晚。莫荷谢绝了村人们的极力挽留，匆匆踏上了归程。村长充满感激地指指天边绚丽的火烧云，很诗意地说："托老天的吉兆，祝你们前程似锦。"

　　来时经过的桥几乎被暴涨的河水淹没了，只在水面若隐若现着一小段鱼脊似的拱背。汹涌的河水挑衅似的前呼后拥，冲击着桥洞发出令人心悸的呜呜声。杨军无奈地冲着急得要哭的莫荷摊摊手，说只有住一夜再走了。莫荷摇摇头，坚决地说："一定要回去，不然姐姐这一夜会睡不安的。"

　　杨军沉思半晌，拉着莫荷往山上跑，说他知道还有一条路通水村，只是要多绕好几里路。莫荷笑道："你不用担心我吃不消。只要能平安到家，累点没啥。"

　　他们翻过第一座山时，天色完全暗了下来。除了他们匆匆的脚步声，天地间死一样的寂静。微风吹来，莫荷哆嗦了一下，双手习惯性地抱紧了双肩。杨军关切地问："冷吧。"为了不让杨军替自己担心，她故意大声笑着回答说不冷。还开玩笑说夜间行路真是妙不可言，省得淌一身臭汗。遗憾的是嫦娥不作美，要是披星戴月那就更浪漫了。

　　喜欢成人之美的嫦娥，好象听见了莫荷的话，不一刻便驾乘一轮弯月前来助兴。莫荷欣喜得大呼小叫，连说山中有灵，老天在保佑我们呢。杨军打趣道："嫦娥可能是看中你了，想让你到广寒宫去陪她消遣寂寞吧。"他紧紧搂着莫荷的腰，冲着月亮哀求道："求求你不要夺走我的心上人，我可不想做后羿第二。"

　　莫荷头仰在杨军宽大的怀里，无限深情地说："'嫦娥奔月'和《泰坦尼克号》上的爱情有异曲同工之妙，都是为了心爱的人甘愿牺牲自己。但我看嫦娥比罗丝更要伟大，因为嫦蛾面对的是赤裸裸的生死离别，而罗丝和杰克还有可能一起走向明天的。"

　　杨军捂住她的嘴，幽幽地说："我既不要你嫦娥，也不让你当罗丝，只要你是我心上的莫荷就行。不管发生什么事，我们永远都不分离。"

　　莫荷突然失声尖叫，杨军只觉莫荷的身体从怀中急速下滑，他条件反射地双臂用力，紧紧夹住差点随着脚下松动的泥土滑下深谷的莫荷。杨军望望月光下深不见底的黑乎乎的山谷，又看看怀中脸色煞白的莫荷，倒吸一口凉气，左手抓住路旁的树，小心翼翼地一步步往前挪。

　　莫荷睁开眼睛，晶莹的泪珠涌了出来，哽咽着说："我还活着吗？"

　　杨军怜爱地亲亲她的脸，说："有我在，没事的。"

　　好不容易挪下山岭，来到河边。望着湍急的河水，杨军懊恼地捶打着脑袋，说要是知道这里的水也这么大，就不必费劲绕这么远的路了。莫荷捡块石头用劲往河中间扔去。连个水花也没溅，石块就不声不响地消失了踪影。莫荷仰望着平静如水的月光长叹道，难道今晚就要被困在这深山野谷吗？！

　　杨军望着弯弯曲曲的河谷，嘴里念念有词。突然他双掌一拍，兴高采烈地说："命不该绝。我刚才计算了一下，我们从这里往下漂，肯定会被冲到对岸那个拐角处的。"

　　莫荷想了想，佩服地点点头："你的物理学的不错。这回可派到用场了。可是怎么个漂法？赤手空拳地下水，那还不得被淹死呀。"

　　杨军鼓励道："开动脑筋，肯定能想出办法。"他学着动画片《聪明一休》里的一休和尚，指尖敲着脑袋，不急不躁地说："不着急，不着急，休息休息。"

　　杨军眯着眼睛沉思片刻，突然大喝一声，吓得莫荷嗔怪地瞪了他一眼。杨军跑到不远处的一个破草棚处，三下五去二的将棚子拆倒，费力地将四根粗粗的木棍捆扎成一个小木排。站起来抹把汗，冲着目瞪口呆的莫荷自豪地笑："人定胜天。智慧是战无不胜的法宝。"他将木排拖到河边，冲莫荷做个邀请的手势，"娘子，上船吧，开始我们的浪漫漂流。"

　　莫荷听从杨军的劝告，蹲下身子，双手用力，十指伸进木排缝里，牢牢地抓住碗口粗的木棍。杨军象蓄势待跑的运动员，弯着腰，双腿当舵，忽左忽右，控制着船向对岸游。

　　行到河中间时，水速奇快，巨大的漩涡象鲜花一样一朵朵争先恐后地绽放，拉扯着木排摇晃着向下流冲去。杨军手舞足蹈，努力想使船逃离水的裹挟，莫荷也腾出一只手来，配合着杨军的动作飞快地划着水。一个浪从后面拍来，木排头猛地一翘，杨军站立不稳，晃了几晃，才好不容易站稳，刚要庆幸没被掀到河里，回头一看莫荷不见了。他心一沉，睁大眼睛往水面搜寻。莫荷的头从脚底下冒了出来，杨军赶紧弯下腰，一咬牙，将莫荷提了上来。莫荷双手扒着木排帮，身子紧贴着木排，痛苦地吐着水。杨军大喝一声，手脚并用，艰难地扳过木排头，缓缓地往对岸驶去。

　　一排急浪把木排冲到对岸的一棵树下，杨军伸出手，敏捷地抱住了斜伸在水面的树干，冲着莫荷大喊："快起来。"莫荷从惊悸中反应过来，跳起来抱住树干。木排完成了使命，连个招呼也来不及打，倏忽之间被水冲得没了踪影，吓得莫荷毛发直竖。树根处早被水冲得松了土，此刻缀上两个人的重量，树根处吃不住力，树干"吱呀"叫着往下沉。杨军照着莫荷的后背猛推一把："快上去。"莫荷借着杨军的托力，小心地爬上了岸。待回过头来拉杨军，树哗啦一声，没入滔滔的河水中。
