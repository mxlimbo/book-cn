　　杨歌摇摇头，说，我不知道。要不，我们找莫荷问问。

　　素心摆摆手，说算了，现在知道了还有什么用。

　　婚变的阴影笼罩在杨歌的心头，很长一段时间里，他都提不起精神。有几次，他甚至动了轻生的念头，若不是被人发现得早，晚报的几十万读者也许从此再也看不到他才华横溢的文章了。

　　莫荷磨破了嘴皮子，才勉强说服杨歌继续撰写最后的三篇文章。写到最后一篇时，杨歌无论如何也不肯动笔。莫荷好说歹说，急得差点都要哭了，杨歌还是那句话，他说，我心已死，情已干，实在找不到任何灵感了。

　　莫荷苦思冥想了大半天，最后说，要不，用那份试题对付一下吧。这个建议激起了杨歌的兴趣。他们请十对婚龄超过二十年的夫妻分别做了一次试题，然后把他们的答案统计了一下，发现没有一对夫妻的共同答案能超过50%的。杨歌对这个结果大吃一惊，他不无嘲讽地对莫荷说，你看，你的那份试题根本就不合国情。莫荷说，反正是游戏，没人当真的。杨歌差点跳起来，他说，可是我们当真了。我总觉得，要不是那两个不同的答案在作怪，我们肯定不会离婚的。我总觉素心对不起我，我对她那么好，可她呢，竟然还有贰心，这么简单明了的十个题目，就有两个跟我想的不一样。女人，他妈的太毒了。杨歌骂得唾液横飞，后来看见莫荷用异样的眼光瞪着自己，忙说你别误会，我不是骂你，我只是针对素心的，你看看，她竟然忍心把我们的孩子给流掉。

　　莫荷平静地说，那试题你知道是从哪里搞来的吗？

　　杨歌惊讶地说，不是你从美国的什么杂志上抄下来的吗？

　　莫荷摇摇头，不，不是。是素心给我的。
