雪 地 童 话


　　知情的人都说，赵婆婆的那只花猫是个通人性的仙物。因为这只花猫曾经奇迹般地救过赵婆婆的两回命。一次是在前年夏天的某个午后，赵婆婆正坐在床沿上做针线，忽然觉得胸口堵得慌，眼前漆黑一片，还没弄明白怎么回事，便象根木头似的一头栽倒在地，口吐白沫，不省人事。当时，花猫正枕着赵婆婆的小脚打盹。赵婆婆跌到床下的同时，把花猫也给划拉到地上。花猫一个激灵跳起来，凑近赵婆婆的鼻孔嗅嗅，然后飞身跑到隔壁的支书家里，将酣梦中的支书弄醒，拽着支书的裤脚把他拉到赵婆婆的床前，才把已走到奈何桥头的赵婆婆硬是给拖了回来。

　　另一次是在傻柱走之后大约两个月左右吧。傻柱的走把赵婆婆的精神推到了几近崩溃的边缘。她变得魂不守舍，常常不由自主地走神。有一天做饭的时候，她又愣了神，眼睛直勾勾地望着窗外的天空，忘记了往灶膛里续柴草。后来，火从灶膛里掉出来烧着了她的裤脚烧痛了她的皮肉，疼得她失声尖叫。花猫正在院子里的杨树上练爬高，听到赵婆婆的惊叫，箭一样窜进屋，奋不顾身地跃上前，用身体扑灭了已窜到赵婆婆膝盖处的火苗。

　　这两件事发生以后，村里人都劝赵婆婆搬到乡敬老院去住，省得一个人孤苦伶仃的没个照应。赵婆婆无儿无女，村里一直是按照"五保户"的标准照顾她的生活。她有权享受住敬老院的福利。但无论大家好说歹劝，赵婆婆死活不答应。问她原因，她也不说。问得多了，她就反问支书，你是嫌我拖累村里了吗？支书说当然不是。支书见赵婆婆这样问，自觉不便再劝；否则，好心可能会被误解，那可就不好了。

　　赵婆婆不愿去的理由只有花猫知道。赵婆婆对花猫说，谁说我无儿无女没有依靠，你就是我的孩子。

　　花猫懂事地"喵"了一声。

　　赵婆婆又说，你傻柱哥虽然没了，但我们守着这个破屋，好歹还算个家吧。我们要是搬走了，你傻柱哥在外面玩累了，到哪里歇息呀？

　　花猫又懂事地"喵"了一声。

　　但这次，花猫却无能为力，只能眼睁睁地看着赵婆婆踏上了黄泉路。赵婆婆是在今年第一场雪的夜里走的。后来医生说，赵婆婆死于哮喘引发的心肌梗塞。

　　那天子夜时分，支书被一阵隐隐约约的叫声惊醒了。他披衣坐起，屏息静听，叫声却又消失了，只有雪花落地的沙沙声。支书刚点上一根烟，叫声复又传来。支书跳下床，推开窗户，凄厉的叫声破空而来，三分悲哀，七分绝望。支书仔细谛听，觉得这声音特熟悉。嗯对了！是赵婆婆的花猫。一阵寒风吹来，支书不由自主地打了个哆嗦，心头掠过一种不祥。花猫的哀号传递的分明是一个危险信号莫非赵婆婆又出事了？

　　支书一边穿衣，一边叫醒熟睡的妻子，说快，赵婆婆出事了？

　　妻子说，咋乎啥？猫叫春呢！

　　胡说！深更半夜的，它还有那心思！

　　那就是冻的。妻子说。

　　不会的。支书说，我下午刚刚给赵婆婆送去二百斤炭球，还是我亲自帮她生的炉子呢。

　　要么，妻子迟疑道，要么就是饿的？

　　更没谱了。支书说，下午我不是刚送去一百斤面了嘛。赵婆婆说晚上擀面条吃的。

　　那就是支书妻子也觉得猫叫的有点蹊跷了，急匆匆地穿好了衣服。

　　支书夫妇火急火燎地推开赵婆婆的院门，发现地上的雪已经有半尺多厚。屋门从里面上了闩子，支书推了两下没有推开，就跑到窗前，吹掉玻璃上的干雪，脸贴着玻璃往里看。黑暗里传来"喵"的一声，吓得支书差点跌坐在地上。他划根火柴，定睛一看，原来是花猫正用四爪抓着窗棂，圆睁着两只绿眼睛瞪着他呢。花猫认出了支书，"喵"地一声，跳了下去。支书又划根火柴，贴着玻璃往里看，见花猫正卧在赵婆婆的床头，轻轻地舔着赵婆婆的脸。赵婆婆的脸在黑暗里泛着可怕的黄光，看上去象一张黄裱纸。

　　支书心里咯噔一下。

　　支书从雪地里摸起一块砖头，砸碎玻璃，从窗户爬了进去。

　　支书妻子捏捏赵婆婆已经僵硬的手腕，失望地摇了摇头。

　　没救了？支书问。

　　没一点脉了。支书妻子说，身子凉得跟冰似的，恐怕死了足有两三个钟头了吧。

　　支书叹口气，说，下傍晚那会，还活蹦乱跳的一个大活人呢，怎么说死就死了呢。

　　支书亲自挂帅，带着一班人操办了赵婆婆的丧事。赵婆婆的坟地紧挨着她的儿子傻柱的坟旁边。赵婆婆下丧的时候，花猫站在傻柱的坟头，身上滚了厚厚的雪，仿佛是给赵婆婆带的重孝。它不停地呜咽着，叫得大家都惶惶不安。有个吃斋的老太太合掌念佛，说这猫想必是傻柱投胎转世来尽孝的。马上就有人反驳她，说傻柱没死之前，猫就出世了呀；要说是赵婆婆的那个没良心的男人投胎来赎罪，还倒有点道理呢。

　　傻柱的坟，其实是一座"衣冠冢"，里面只埋了些傻柱的衣帽之类的"遗物"，至于傻柱现在是死是活，到底在哪里，谁也不知道。

　　这事说来话长。

　　傻柱五岁那年，他爹和一个外地来讨饭的女人远走他乡。几十年来，杳无音讯。傻柱那时还是一个聪明可爱的正常人。而且很懂事，从不惹妈妈生气。大家都夸柱子是块料，以后肯定会有大出息。赵婆婆也坚信不移地认为，柱子一定会争气的。岂料天有不测风云。傻柱七岁那年，害了一场奇怪的病，神经出了点问题。不犯病的时候，与常人无异；犯起病来，痴痴迷迷的，还不如三岁的娃娃知好歹。赵婆婆的心碎了，她生命中惟一的希望就这样被无情地击破了。想想自己的命真是苦呀，男人走了，儿子又成了一个废人，自己下半辈子该依靠谁呀？有好几回没想开，就闹着寻死。幸亏被人发现的早。大家都劝赵婆婆，儿子傻归傻，怎么说也还是你的亲骨肉，你不为自己活着，也得为儿子想想呀。赵婆婆于是就坚强地活了下来。

　　傻柱神经正常的时候，特别孝顺妈妈。不但饭菜侍候到床前，而且还形影不离地陪着赵婆婆拉家常。傻柱嘴甜，原又聪明，经常说些俏皮话逗妈妈开心。大家背地里议论说，人还是傻一点好；知道疼人呢。

　　转眼之间，傻柱已是快奔三十的人了。有一天，傻柱对妈说，妈，我想跟着村里的建筑队闯关东，挣点钱给你买点好吃好喝的。

　　赵婆婆的泪水刷地涌了出来。

　　傻柱替妈妈擦去泪水，安慰妈说，放心吧，你看我现在不是挺好的嘛。

　　赵婆婆说，儿行千里母担忧。你要是犯了病，会把妈急死的。

　　村里人会照顾我的。

　　你笨手笨脚的会干啥呀？

　　我不是还有身傻力气吗。傻柱就屈起胳膊，指着结实的肌肉疙瘩说，妈，你看我多壮实。

　　赵婆婆年近花甲，自觉世上光阴不多。知道总有一天，儿子要一个人想办法活下去，早点让他学着走路也好。于是就去找支书。支书边听边摇头，说放心吧，你走了之后，还有全村千把口人，饿不死傻柱的。无论赵婆婆怎么恳求，他就是不答应。心想这事岂可草率，傻柱在外面要是有个三长两短，赵婆婆哪能承受得了这个打击。傻柱听说支书不答应，就跑到支书家，跪在支书面前大半天，无论谁劝，他就是不起来。逼得支书无计而施，最后说别跪了，我答应你就是。

　　临离家之前，赵婆婆和支书千叮咛万嘱咐，傻柱还是出了事。车到长春站时，同村的二毛陪着傻柱上厕所。傻柱先进去方便完了，二毛说你站在门口等我哪也别去。等二毛从里面出来后，发现傻柱不见了。二毛急忙喊来同村的人挨个车厢找，梳篦子似的前后找了两三遍，也没有见到傻柱的影子。后来，车站派出所的民警分析说，傻柱可能是随着到站的人流下了火车。

　　赵婆婆被这个噩耗击倒了。她在床上哭了三天。支书和村长轮流陪在床前。支书说，老嫂子，人死不能复活，你还得保重自己的身体啊。

　　赵婆婆说，傻柱也没了，我还有啥活头哟。

　　支书说，我给你当儿子，给你养老送终。

　　三天来，花猫也是不吃不喝，一直静静地卧在赵婆婆的床前。

　　支书说，老嫂子，你看看这猫，都快饿死了。

　　赵婆婆爱怜地摸摸花猫瘪瘪的肚皮，长叹口气。说，命，都是命啊！

　　赵婆婆后来把傻柱的衣服拾掇拾掇，请人给傻柱堆了一座坟。赵婆婆对花猫说，我还得再硬撑着活两年，给你傻柱哥成个阴亲，不然他连个说话的人都没有，多闷得慌呀！

　　从赵婆婆坟地回来的时候，支书望着还立在坟头叫个不停的花猫说，这样叫会叫死的。

　　有人说，这家伙是个灵物，我把他抱回去吧，说不定能给我带来好运呢。

　　等他靠近花猫时，花猫"喵"的一声逃走了。大伙在雪地里追了半天，最终还是没有逮到它。

　　支书说算了，就让它留下来给赵婆婆守灵吧。

　　众人走出老远，回头望见花猫又站在了傻柱的坟头上，"喵喵"的叫个不停。

　　支书说，真是邪乎啊！我都有点迷信了。

　　大家都笑了。

　　有人说，你可是个老党员呀！

　　支书说，共产党员也是人，也有人性嘛！

　　这天半夜，支书起来方便时，听到隔壁赵婆婆屋里传来男人的哭声。谁呀？哭得这么伤心。支书琢磨了半天，也没想出个所以然来。莫非闹鬼了？支书马上又摇头否定了自己。支书会动共产党员的感情，但决不会动摇唯物主义观。

　　哭声越来越清晰，越来越伤心。支书觉得没理由再睡下去，他穿衣下床，来到赵婆婆窗前，贴着玻璃往里看。这一看不要紧，支书的三魂差点吓掉两魂半。柱子！原来是柱子！那个已经死了大半年坟上荒草都长得老高老高的柱子！

　　傻柱回来的消息，着实让全村人大吃一惊，也让大家心里多少有了点安慰。有人说，天不负人，他们母子有神灵保佑着呢。

　　全村人一拔拔地来看柱子，问他这大半年跑到哪去了都干了些啥。柱子现在状态还好，客气地回答着大家的问讯。不过，这半年的经历，显然是记不清了。他只记得一个月前到了山东淄博，在一家汽车修理厂打工，准备挣点钱回家过年。前天夜里，他突然感到头疼，特别想妈妈，就连夜坐车回来了。

　　大家听了面面相觑。目光里传递着同一个内容：莫非真有心灵感应之说？

　　傻柱在大家的帮助下，办了一桌丰盛的供菜，一个人挑到母亲的坟前。那只花猫还趴在傻柱的坟头，身子冻得挺硬。傻柱在母亲坟前哭了一整天，天快黑的时候，放心不下的支书跑来看他，见他把自己的"坟"挖开了，正将那只花猫埋进去。

　　傻柱对支书说，我还不如只猫呢，没能在妈床前尽孝。

　　支书说，你是个孝子，大家都知道。

　　傻柱说，让花猫给妈做个伴吧，她就不会太冷清了。

　　回村的路上，支书说，你回来就好，哪儿也不要去了，你的生活村里已经替你安排好了。

　　傻柱说，我想出趟远门。

　　远门？

　　我去把我爹找回来？

　　你爹？

　　嗯。

　　上哪找？

　　我有种感觉。傻柱的眼睛里放射出异样的光芒，挥起胳膊比划着，他肯定在那个方向，东南方。

　　找他做啥？

　　我要把他五花大绑的拖回来，让他给俺妈披麻带孝。他害得俺妈受了一辈子的苦，我决不能轻侥了他！我，我非亲手宰了他不可。
