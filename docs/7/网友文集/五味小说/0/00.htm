序

　　我上高中的时候，就开始写“小说”了――那或许称之为“作文”更合适些。真正意义上的小说写作，是在我工作之后。我的第一篇小说是《与感情无关》。她是我目前最得意的作品。其时，我刚恋爱，文思如泉涌。加之没有写作经验，写作的目的也比较单纯，所以写起来自然流畅。其后，陆续写了几十万字，但满意的不多。此书收录的十二篇，姑且称得上是小说吧。 后来，感情生活的不幸，湮灭了我的写作激情。这些文字，成了记忆。
