幸 福 擦 肩 而 过

（三）

　　俄倾，许大海进了教室，直奔我杀来。我全身的汗毛都直立起来，只觉得血一个劲地往头上涌。

　　许大海胳膊支在桌子上，趴在我面前。王老师来调查这事了，是你写的吧，赶快投案自首吧，争取宽大处理。

　　我头一拧。别瞎说，不是我干的。

　　嘘许大海示意我小声点，不是你干的？好，我们来分析一下。303室里，除了你有那样的文采，还会有谁想得出来？再说，你的笔迹还不就是最有力的证明。我可是全为你好。学校对此事很重视，李主任刚才亲自去把它揭下来，指示王老师一定要查个水落石出。其实大家都心知肚明，谁干的那还不是秃子头上的虱子明摆着吗。我劝你要认清形势，主动一点，对你只有好处，没一点坏处。

　　我感觉额头上冒出了密密的汗珠。偷眼望望王国庆，他正双手插进裤兜里，悠闲地来回踱着步；李央则瞪大了眼睛，紧张地注视着我。

　　我咬咬牙，别费劲了，说不是我干的，就肯定不是我干的。

　　许大海站起来，摇摇头，我可是仁至义尽了，到时可别怪我没提醒你啊。

　　我突然冲他吼道，你小子什么意思？我什么时候得罪你了，找了机会来整我！

　　许大海红了脸，讪笑着出去了。　　

　　上课铃响了之后，进来的不是教英语的田老师，而是一脸严肃的辅导员王老师。同学们的目光不由自主的全转向了我，烧得我满脸通红。

　　王老师朝站在门外的田老师点个头。对不起，耽搁你几分钟。他转向我们。说，我现在只想知道，那张轰动了校园的"征友启事"出自何人之手。请自觉站出来，不要耽误大家过多的时间。

　　同学们都似笑非笑地望着我。

　　王国庆趴在桌子上，不停地打着手势，要我顶住。可是，我却鬼使神差地站了起来。

　　好！还是挺有勇气的吗。其实也不是多大的问题。下午到系办来一下，我们好好谈一谈。王老师走下讲台，出去了。

　　我怔怔地望着王老师的背影发呆。田老师进来时，我还象电线杆一样立在那儿。

　　田老师意味深长地望了我一眼。笑道，出了什么有趣的事？

　　教室里爆发出很响亮的笑声。

　　同位的拽拽我衣服，我才缓过神来，一屁股跌坐下来。

　　田老师开始上课了，我一点都没听进去。在纸上茫然地乱画着，后来看看，满纸写的都是"征友启事"。

　　从后面飞过来一纸团，在桌上弹了两下，就静静地不动了。我哆嗦着扒开纸团，上面写着：不用怕，有老大呢。

　　我回过头，感激地朝含笑望我的苏乐点点头，泪水顺着腮帮流了下来。　　

　　午饭由李央端到床前。

　　我把头用被子捂得严严实实，在里面大声喊道，不吃！

　　苏乐用劲揭开被子，用绝食来吓唬人呀！把饭吃掉！

　　我的眼泪刷地流了下来，老大，你可要替我报仇啊，都是许大海这小子卖友求荣，把我给出卖了的。

　　别乱咬人，你们玩的那点把戏，傻子也猜得出来。苏乐说。

　　王国庆说，谁也别怨，都怪你小子沉不住气，挺过三分钟也许就没事了。

　　苏乐瞪了他一眼，你还有脸说，三个人干的事，为何让人家一个人来扛？太不仗义了吧。

　　王国庆头一挺，有福同享有难同当。下午我和李央陪你去。我就不信，还能让我们蹲几年大牢？

　　苏乐说，去你的吧，老老实实呆在宿舍里，别再给我乱捅漏子。下午我陪郭长风走一趟。是死是活，大不了头上一个疤。

　　李央紧紧拉住苏乐的手。老大，谢谢你，晚上我请你吃饭。

　　你就那么怕欠我的情，屁股一抬就急着报恩？苏乐拍拍李央的肩，别太在意，谁让咱们是好弟兄呢。

　　许大海端着饭碗来到我们宿舍，见我不理睬他，就讪笑着和苏乐搭腔，老大，这事可真不能怪我呀。

　　苏乐皱皱眉，没人怪你。看你这副贱样，软骨病？

　　许大海苦笑道，老大，你就嘴上留情吧，看郭长风这样子，你以为我心里好受？其实，我的日子也好不过呢，既要全心全意侍候好你们，还要挖空心思对付上头。夹缝中做人难呀。

　　李央说，别讨了便宜还卖乖。又没人逼你当这鸟官。

　　你以为谁想干？但现在是骑虎难下呀，撒手不干，两年多的心血岂不是白费了，孬好也得混个党员回家。

　　我就知道没有白拉磨的驴。王国庆阴阳怪气地说。

　　许大海啐了他一口，俯下身子说，长风呀，我劝你还是先写份检讨书，深刻点，我想只要态度好，不会有什么大不了的。

　　苏乐点点头，不错。

　　王国庆说，只要不往学校里报，在系里就不会有大事。

　　言之有理。大海讨好地冲王国庆笑，王老师对我们班还是非常关心的，不会干那种缺德事的；再说，学校对系里本来就很头疼，系里现在是能瞒则瞒，想必不会借题发挥的。　　

　　赵岚和密菊来找苏乐。

　　苏乐抱歉地说，实在对不起，只能改日了，下午我要陪郭长风去负荆请罪。

　　她们失望地对视了一下。

　　赵岚说，有那么严重吗？

　　苏乐说，多一个人，多一分胆。

　　他看看正痴望着赵岚的李央，笑道，要不，烦请你二位陪两位小姐走一趟？你们的技术可都远在我之上。

　　李央抢着说，好呀。

　　王国庆则连连摆手，不行，把郭长风扔这儿受罪，我于心不忍。

　　密菊掩嘴窃笑。

　　赵岚红了脸，没时间就算了吧。她捅捅密菊，别傻笑了，打道回府吧。

　　许大海喊道，赵岚，别辜负了李央的一片诚心。

　　赵岚撩了下飘到额前的一绺头发，提高了嗓门说，稀罕你提醒。她朝李央笑道，那就给你添麻烦了。

　　密菊装模作样地喊，哎哟，我肚子疼死了，你们两人去吧。

　　赵岚照密菊后背狠狠打了一巴掌，挺乖的丫头，咋学得油头滑脑的？今天死也得死在一起。

　　王国庆痛苦地叫，可怜的电灯泡。　　

　　我把检讨书毕恭毕敬地递给正襟危坐的王老师。他表情庄重，一字一句地读完后，盯着我好半天，才慢慢吞吞地说，你认识得还比较深刻，也就不用我再多说了。不过你要有心理准备，李主任可能会找你谈话。他的脾气想必你有所耳闻。当然了，你不要有任何思想负担，他说什么，你听着就是，千万注意不能和他顶嘴。

　　我感激地点点头。实在对不起，王老师，给您添这么多的麻烦。

　　王老师摆摆手，人非圣贤，孰能无过。何况你们这些还没成熟的学生呢，辨别是非的能力还不强吗。不过说实话，你们这届学生，我感觉特难带，也许我的思想有点落伍了，跟不上时代的进步了。

　　苏乐说，是我们生不逢时。这社会变化太快，难以把握自己。

　　王老师点点头，说一千道一万，还是你们闲的无聊。有事追着，就不会生出这么多乱子来。我看你的文采不错，可以把精力多朝这方面发展嘛。

　　我连连点头。

　　王老师摇摇头，你既然喜欢搞文学，就应该注意用词。看你这满纸的一派胡言，把大学生活写得是乌七八糟，真让人没法接受。不是吓唬你，放在上纲上线的"文革"期间，单凭这张大字报，就能让你坐上十年大牢。

　　我吓出一声冷汗。

　　王老师又语重心长地叮嘱了我许多，然后把检讨书递给我，让我回去再润色一下，如果李主任找我，可以交给他。临出门时，王老师拍拍我的肩，说，不要对李主任说是我让你写检讨书的。

　　我脸上堆满了感激，一一应着。　　

　　我和苏乐刚到宿舍楼下，密菊迎面跑来，气喘吁吁地说，不好了，赵岚出事了！

　　啊？！我和苏乐大吃一惊。

　　怎么回事？人哪？苏乐着急地问，不要急，慢慢说。

　　密菊说，我们在公园里的旱冰场玩的正欢着呢，上来几个小痞子，有意往我们身上撞。赵岚看不过，就和他们吵，惹恼了那伙人，他们一哄而上，把赵岚碰倒了。到医院拍了个片子，脚骨断了。

　　操！苏乐狠狠地一跺脚，都怪我没去！

　　李央呢？我问，这个脓包，一点用也没有。还指望他当保护神呢。

　　别怨他了。密菊说，李央勇敢地冲上去了，被那伙人好一顿痛打，鼻清脸肿的。还背着赵岚在医院里楼上楼下的检查取药。今天真是多亏了他。

　　我低头无语。心想，世上的事多有巧合，李央上辈子欠了赵岚的情债呢，理当还债。

　　苏乐拔腿就走，走，找那几个狗东西算账去。

　　我忙拦住他，还是先报告保卫科吧，或者到派出所报案去。要是自作主张再捅出什么漏子，不好向王老师交待呀。

　　报案有个屁用！苏乐说，区区小事，我自个儿就能摆平他们。你就不用去了。

　　我尴尬地说，这

　　密菊说，去也没用，早就跑没影了。还是先到医院去看看赵岚他们吧。

　　苏乐望望我，又看看密菊，好吧。　　

　　我们轻轻推开病房门。李央正趴在床前，轻声说着什么笑话，逗得赵岚咯咯直乐。

　　我大声喊道，看你们美的，都忘记疼痛了吧。

　　他们吓了一跳。

　　李央腾地站起来，见是我们，难过地低下头，对不起老大，我给你丢脸了。

　　赵岚挣扎着想坐起来，咧了下嘴，又歪了下去。

　　密菊忙跑过去扶着她，小心。她的眼泪流了下来，让你受委屈了。

　　赵岚勉强笑道，一点轻伤。看你兴师动众的，多不好。

　　苏乐拍拍李央的肩头，好样的，不是孬种。

　　他走到床边，俯下身去，关切地问，很疼吧。

　　好多了。赵岚笑道，就擦破了点皮。医生说，个把星期就能下床了。

　　苏乐说，你就安心在这儿养着吧，我们轮流来陪你。

　　我想现在就回去呢。赵岚面有难色，这儿太贵，住不起的。

　　没关系，身体要紧。李央说，钱的事，你就不用操心了，责任在我，我负责到底。

　　赵岚低下头，轻声说，已经给你添那么多的麻烦了，很过意不去的。

　　密菊笑道，都快成一家人了，还用得着这样客气呀！

　　我跟着说，这可能是红娘的刻意安排，你们是患难之中建爱情呢。

　　赵菊使劲地捶打着密菊的背，该死了，越来越不要脸了。

　　苏乐说，要不今晚就先让李央陪陪你，明天我们来替他。

　　李央连连说，好好好，将功赎罪。

　　临走时，我悄悄地拉拉李央的衣角，机会难得，努力表现啊。

　　他回头望望赵岚，冲我神秘地笑，你是不是想说，傻小子有傻福？密菊也不比赵岚差吗，上呀，还等什么？

　　我冲他挥挥拳头，搡！没你这样损人的。　　

　　第二天早上，我早早吃过饭，准备到医院去换李央，李央却疲惫不堪地回来了。

　　苏乐一脸的纳闷：谁替你的？

　　赵岚执意要回来，说住不惯那儿，晚上老失眠。

　　王国庆还赖在被窝里，伸着脖子喊，那正好可以玩个通宵嘛。

　　国庆苏乐吼道，别不知好歹，开玩笑要注意场合。

　　我说，赵岚怕花你的钱？

　　许大海说，药费可以报销的。

　　李央往床上一躺，有气无力地说，医院那种地方，能不去就不去。我晚上老听到有鬼叫。宿舍里多少象个家的样子，心情好。

　　大海。苏乐说，拿点班费出来吧，买点营养品送给赵岚。

　　行。许大海点点头，我早就想到了，正想和你商量买什么呢。嗯，班费还剩下不到二十元，我可以先垫点，一百元够不够？

　　多少无所谓，心意到了就行。苏乐说，话可说前头，没人领你的情，谁让你是一班之长，垫点钱不应该？再说，你那奖学金，不是还有剩余吗？

　　许大海连连给苏乐拱手，老大，都哪辈子的事了，别老是哪壶不开提哪壶，"四人帮"都打倒了，我就不能平反昭雪？

　　时候不到，继续考验。苏乐认真地说。　　

　　许大海所说的"不开的那壶"，是指曾轰动整个学院的"罢课风波"。作为这场闹剧的主要策划者和积极的参予者，许大海恍如"文革"期间创造政治奇迹的"造反派"一样，一夜之间闻名全院。

　　这要从上学期的期末考试说起。

　　当时，我们总共有四门考试课。其中的三门，老师或多或少的都划了重点，过关看来不成问题；另外的一门，也是我们最头疼的一门，是由院里最年轻的计算机博士生诸葛老师带的《微机原理与应用》。这门课本来就深奥难懂，一个学期下来了，大家都稀里糊涂的。眼巴巴地指望诸葛老师能体谅民情，睁一只眼闭一只眼，让我们都轻松过关。可恨的是，诸葛老师对我们的请求置若罔闻，不但不给任何提示，反而再三严正声明，让我们不要抱有任何幻想，好好复习；即使仅差半分，他也绝不会高抬贵手。搞得我们精神高度紧张，私下里骂他是不会做事的书呆子。

　　"罢考"阴谋出台在我们对诸葛老师一片埋怨声中，不能不说是占尽了天时、人和两大成功要素。

　　临考前的第三天晚上，我们吃过饭，顾不上休息，正准备到教室看书，许大海神神秘秘地跑到我们宿舍，说好消息好消息，弟兄们有救了！

　　我们狐疑地望着他，不知道他葫芦里卖的什么药。

　　许大海坐到苏乐的床边，轻声说，老大，复习的怎么样了？有没有把握？

　　苏乐笑道，什么意思？嫌我影响你进步了，今年不想让我抄你的了？

　　许大海连忙摆手，误会误会。老大，我可有一妙招，能让大家免受考试劳役之苦。

　　还有这等好事？！我们都围拢过来，齐刷刷地望着许大海。

　　大海压低声音说，绝对可能。到时候，我们一起喊卷子太难，让他免考！

　　嘘！我们长吁一口气，都感到被欺骗了。

　　弱智！苏乐骂道，什么馊主意，真把诸葛老师当呆子了。告诉你，狗急了还会跳墙呢，惹火了他，没什么好果子吃。

　　偷鸡不成反蚀半把米，赔了夫人又折兵。我晒笑道，许大海，别聪明一世糊涂一时。

　　放宽心，天塌了由我一人顶好了。许大海见我们不相信他的诚意，急红了脸。

　　籁蛤蟆打哈欠好大的口气。你也不过是泥菩萨过河自身难保。王国庆眨着眼睛，再说了，你小子什么时候换了一副好心肠，肯定有什么见不得人的目的，想让我们替你当炮灰。

　　唉呀！许大海一跺脚，天地良心，我可是全为了大家好。

　　别费劲了，违法乱纪的事绝对不干，别把弟兄们往火坑里送。苏乐板起脸说。

　　老大！许大海眼睛红了，带着哭腔说，我对你什么样，你不是不知。说这种话，多让我伤心。

　　苏乐站起来。别哭哭啼啼的，我们不相信眼泪。为你两肋插刀可以，但你至少应该把问题讲清楚了，不能让我们糊里糊涂的去送死吧。

　　许大海沉思半晌，才吞吞吐吐地说，几位都不是外人，我就实话实说了，不过，这话绝对不能外传。

　　我们都严肃地点点头。

　　许大海说，我们系的情况，你们肯定也知道。王老师和诸葛老师是死对头，诸葛老师这人，水平确实不俗，但就是人太狂，不讨人喜欢

　　那是王老师心眼小，妒贤嫉能，容不得才气比他高的人罢了。王国庆愤愤不平地说，搞行政的人都这德性，屁本领没有，瞽天琢磨着算计人。

　　许大海白了他一眼，接着说，王老师注意收集民情，听说我们对诸葛老师很不满，想联手给他点颜色看看，杀杀他的威风。这罢考的点子，全是王老师的意思，有他为你们撑腰，还有什么可怕的。他洋洋得意地说，放心，王老师不会亏待你们的。

　　苏乐的脸色变得很难看，王老师真是狗眼看人低，以为我们都象他一样没心没肺呢！大海，你转告他，就说我苏乐义正严辞地说了，这种缺德事，我们不会干的。

　　对！王国庆跳起来，诸葛老师与我们无冤无仇的，不能这样对待他。

　　许大海说，那你就等着补考？

　　考就考！有什么大不了的！王国庆斜睨着许大海，谁让我们没好好学的，活该！诸葛老师对我们可是认真负责的，课上得多好。

　　我说，大海，别让王老师把你当猴耍了，古往今来，政治斗争的最终牺牲品，往往都是象你这种无知的人。

　　许大海的脸红得象猪肝，哀求道，老大，就算帮帮我的忙，行不？

　　苏乐说，这事对你真的很重要？

　　嗯。许大海的泪水流了下来，王老师答应我，明年一定送我张党票。我当这个龟孙子班长，混得里外不是人，干的净是出力不讨好的事，为了什么？还不就是为能混个党员。老大，念我平日里的交情，你总不能见死不救吧。他抹把鼻涕，很响地甩到地上，我要是拒绝了王老师，不但这几年的汗水白淌了，今后肯定也不会有什么好日子过。

　　苏乐厉声喝道，够了！你扪心自问一下，良心是不是让狗吃了！为了一点私心杂念，什么缺德事都敢干。中央整天喊党风不正，有你们这种人给瞎搅和，能好得了吗？！

　　他望望泪流满面的许大海，目光移到我们身上。弟兄们，虽说这事有点太损，可大海也确实够难的了，我们能不能帮他一把？

　　我和王国庆毫不犹豫地答应道，一切行动听指挥，老大指哪我们就打哪！

　　苏乐脸上漾开了灿烂的笑容，臭小子，挺会说话。

　　他转向许大海，大海，你小子听着，下不为例。不过你放明白了，此次同流合污，纯粹是出于同情，决非与你一样卑鄙无耻。你要是过河拆桥，别怪我翻脸不认人。

　　许大海破涕为笑，拍着胸脯保证。放心吧老大，事成之后，我请几位吃饭。　　

　　才华横溢的诸葛老师万万不会想到，正有一个可怕的阴谋在一步步向他走来。

　　《微机原理与应用》的考试那天，诸葛老师把试卷发了之后，就不声不响地出去了。一个小时候，他推门进来时，看到我们一个个正襟危坐，目不转睛地看着他。他白白净净的脸上露出了惊讶的表情，笑问道，都做完了？

　　没人答话。

　　他瞪大了眼睛，望望我们，拿起坐在门边的徐昆云的卷子：一片空白！

　　啊！诸葛老师的脸变得煞白，怎么回事？

　　不会做！徐昆云低下了头。

　　难道一题都不会？！他愤怒地喊道，搞什么鬼？！

　　我们都难为情地别过脸去，不忍再看他那张被愤怒扭曲的脸。

　　许大海腾地站起来，就是一题不会！谁让你出的题都超纲了！

　　诸葛老师怔怔地望着许大海，说吧，你们想怎样？

　　重新出题。许大海说，不能超出我们掌握的范围。

　　我要是不答应呢？

　　我们就罢考！许大海喊道。

　　你诸葛老师强压住怒火，反了啊你们！闹吧，爱怎么折腾就怎么折腾。想倒退到"文革"时期？想重演历史？简直是痴人说梦！

　　诸葛老师摔门而去。　　

　　问题并没有我们想象的那么简单。

　　诸葛老师气势汹汹地从系办杀到院办、党办，非要讨个说法。

　　院领导非常震惊，感到此事不可思议。这可能是"文革"之后，全国第一例"罢考事件"吧。

　　院里派下来的调查组很快将此事查了个水落石出。并且很快发出通报，对这起事件的幕后策划者王老师给以"计大过"处分，许大海被处以警告。

　　虽然没有处分到我们，但我们也为此付出了沉重的代价。在临放假的头天晚上，我们将车票都买好了，准备翌日起程回家。李主任亲自到宿舍里通知我们，我们班推迟一个星期放假，抓紧复习，准备考试！

　　我们炸了窝，都喊着要将许大海活剐了！

　　许大海痛苦流涕地说，要是我死了能平息大家的怒气，那我情愿去死。

　　我们看看许大海那个熊样，一个个也都没了怨气。

　　王国庆说，谁也别怨，怨只怨我们鼠目寸光，看不透政治斗争的险恶。

　　苏乐瞪他一眼，值得，至少通过此事，我还能看到弟兄们的凝聚力，依我看，倒是件大好事。外来侵略并不可怕，只要我们能团结起来，没有克服不了的困难。

　　我低声对苏乐说，老大，不是凝聚力，而是你的号召力！

　　他笑道，都是一回事！

　　我们又不得不收拾起准备回家的轻松，投入到紧张的复习中去了。

　　考试结束后，我们一个个垂头丧气，哀叹这回是彻底完了补考没商量。

　　成绩当天下午就出来了，出乎全班人的意料：全班除了许大海一人不及格外，其余的人千篇一律的都是一百分！这是历史上从没有过的好成绩。我们都跳起来，连呼诸葛老师万岁。

　　我们兴奋完了，幸灾乐祸地望着许大海，心想，搬起石头打了自己的脚吃亏的到头来还不是你自己。

　　许大海丝毫不怨诸葛老师，平心静气地说，罪有应得，我对不起诸葛老师。
