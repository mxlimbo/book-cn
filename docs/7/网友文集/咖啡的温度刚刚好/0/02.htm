好不好玩

　　我把电脑的桌面换了--一种昏黄的调调，两个人伸长了腿，坐在地上，执手相看。浓浓的爱情味道从屏幕里溢出来，让我很满意。

　　酱瓜在我的身后叹气，绝对没有悠然的要人重视的味道在，而且一声比一声高，到了后来近乎于尖叫。这样的叹气我很快的忽略。于是他终于忍不住要说，这样的桌面？

　　不妥？我反问。

　　怎么会不妥？他呵呵的笑，说很有味道呀，有种老女人在春天里呼吸的味道！

　　我神情自若，继续看网上拷下来的文章，连眼神都没有挪位置。

　　酱瓜跟着沉默，然没有多久，终于耐不住，问：你做什么不说话？培养自己孤独的气质？

　　我很气定神闲的回答：我说话有两个条件，第一，看心情。第二看对象！今天阳光灿烂，我的心情确实不错，但是对象就……

　　酱瓜有着极为深厚的脸皮，可以与向来自负的我一争高下。他依旧坐的很稳，坦然说原来你是在生我的气？我还小，不要同我认真！他说的沉稳极了，压低了声线，活脱是卡通片《蜡笔小新》的台词和语气。

　　对于这样的人我有什么办法？我思考了一下，发现没有对策，只好一扔鼠标，爽快的笑。

　　这个时候我的心情确实不错。因为在一个阳光灿烂的下午我惊喜的看着酱瓜忽然在我的屋子里冒出来，而且表面看上去完好无损，于是我很愉快。这是实话。

　　每次他来之前都打电话给我说，我要抗一束长枝玫瑰过来，准备好花瓶。我拒绝，我说这样漂亮的花被你象拿鸡毛掸子一样的拿给我，什么味道都没有了，我不要！

　　于是他坏笑，他叫我阿姨。他说阿姨，实际一点，现在肯送你花的也就是我一根独苗了！你实在没有理由拒绝我！

　　我也坏笑，我说，对了！你可以送我更加实际一点的礼物，比如人民币？

　　这回轮到他拒绝。

　　于是每一次他都两袖清风的过来，然后我也用空空如也的冰箱迎接他。

　　他说，生鱼片，我们的配合实在很好！这个世界也许再也找不到比你更加配合我说话做事的人了！

　　这个时候他叫我生鱼片！

　　我们是贫下中农出生，上辈子没有解决温饱问题似的，于是一个劲拿着食物的名称作为彼此的称号。我们自欺欺人的以为这样叫叫也可以满足我们饥饿的沟壑。

　　想到这一层，我说我们还是吃点什么吧！好过现在的无聊！你真的应该带点人民币出来的！

　　他也叹气，说你真的应该在冰箱里塞点什么的！

　　没有办法，我们只好一起叹气！

　　这样的事情不是一次两次的问题，只是我们依旧如故！这成为我们的游戏和话题。原本每次我们叹气的时候，都有一个叫釉子的女孩子皱着眉头看着我们，然后说，走吧走吧，一起吃饭去！

　　她是一个好女孩，很善良！她爱着酱瓜的时候就和他一起来我的家，看着我和酱瓜斗口，也不管我和酱瓜怎么闹别扭，她总是很英勇的站出来，帮助我们填饱肚子。换过来当我到酱瓜家里偷他的东西时，酱瓜叫嚣着要釉子立刻出去买保险箱把值钱的东西全部放进去，避免某个强盗抢劫的时候，釉子也是温柔的笑着，包容着我和酱瓜从屋子里打架到屋子外。虽然她不了解我和酱瓜彼此间的游戏，但是她表现的和上帝一样宽容。这样的好女孩实在不多见。我和酱瓜说好好把握，不要辜负了人家。酱瓜笑着说不会不会，辜负了她就是辜负我们的肚子。

　　然而，釉子在我和酱瓜的生活里出现没有多久就人间蒸发了！

　　事情的发生显得有点莫名其妙。就是在一个早晨，阳光灿烂的很让人愉悦，酱瓜起身的时候很明显的感到了这点，于是他叫着釉子的名字，叫她起来看看这样明媚的早晨。

　　釉子翻了个身，说了一个字：不！

　　酱瓜呆了呆，想起了釉子的温顺。从开始以来釉子从来没有和他说过一个不字！酱瓜为了确定方才是不是早上起来迷糊的听错了，只好再强调一下呵斥她起床。

　　于是釉子也再次强调了一下说：不！

　　她居然和我说两个不字！酱瓜和我说到这里的时候表现出言之切切的样子，她从来都是那样温柔，现在居然和我一连说两个不字！

　　于是酱瓜忽然扔了一只拖鞋过去。虽然没有打中，但是足以引发一次妇女大解放。釉子当天下午就离开了！

　　我只是忽然想要测试一下抛物线的弧度，所以褪下了脚上的拖鞋扔了一下……酱瓜尽量把口气放的低点同我解释。我知道他说话的风格。假如他把小刀叉在了一个人的腿上，他也会说我只是把刀摆了一个位置，我怎么知道那个位置正好是他的大腿呢？

　　釉子的离开简直是一定的！

　　她跟着酱瓜这样久，从来都是一个人文文静静的笑，她爱着酱瓜，虽然她从来不了解他。但是女人的忍耐靠的不仅仅是爱情，而是一种信心！一种对爱情的信心。她不了解酱瓜，所以她得不到酱瓜给他的信心。她这样柔弱，实在是没有办法再坚持！

　　釉子在我和酱瓜的生活里昙花一现，唯剩下我和他继续我们的游戏，至于游戏好不好玩也就是我们可以知道了！

　　我们的台词还是：我说我们还是吃点什么吧！好过现在的无聊！你真的应该带点人民币出来的！

　　他也叹气，说你真的应该在冰箱里塞点什么的！

　　唯一的区别就是少了一个声音，少了一个声音说，走吧走吧，我们吃东西去！

　　我想到了这层，酱瓜自然也想到了。他在我身后伸着懒腰，笑着说，你知道吗，我爱着釉子！真的！

　　我吃了一惊！

　　我看看窗外，太阳高照，我想我不是在做梦，于是开始重视酱瓜的话！

　　他说什么来着？对了，他说他爱着釉子！

　　酱瓜好象还在问我你相信吗？

　　我爽利的回答相信相信！你爱着她，但是当你发现这点的时候已经晚了，她走了，对不对？

　　酱瓜说，其实并不晚！我以为不晚。你知道我一向是一个老实的人，一旦我发现这点就告诉了她，可是问题在于她不肯相信！女人真的是奇怪的动物，她每一天都在想着我是不是爱她，但是当我告诉了她答案，她却不肯相信！一直到她走的时候她还是不相信我爱她！

　　我呆了呆。

　　我看着我的电脑屏幕半晌，看着那对可爱的人儿执手相看之间的温柔，然后低低的说，换做了我，我也不会相信的！不管大家怎么样开始的，过程里你一直说这是一个好玩的游戏，到了游戏的结束你说原来一切都是真的，叫她怎么相信！她也许以为你说这样的话无非是想要延长这个游戏，或者增加游戏的趣味性！在这场游戏里快乐的只有你自己，痛苦的却是她。你要她怎么样有信心有勇气再次把自己投入到你的新游戏中呢？

　　说话的时候我没有看着酱瓜。我说话的对象原本就不是酱瓜，虽然我的对象不在这个屋子里，虽然他听不见，但是我一样的说了！

　　酱瓜长长的叹了一口气！我也叹气！

　　我们这次叹气因为我们各有各的心事！

　　酱瓜自然听的出来我话里的意思，他问，你还爱铭西吗？

　　我苦笑，忽然间在这样的场合听见酱瓜问起他，所有的脆弱从心里涌起一下子卸除了我那所谓坚强的面具。我说我也实在没有办法在他的游戏里坚持了！

　　怎么连你都有胆怯认输的时候？你不是向来喜欢问好不好玩，不好玩不奉陪吗？酱瓜问！

　　女人其实都很胆怯！在爱情游戏里我们不过拿出自己的心任由你们切割！我们觉得这个游戏一点也不好玩！我说，希望你记得给女人爱的信心！

　　酱瓜了然的叹气，说，我知道了，现在才知道！其实我也觉得这个一点也不好玩！不过知道的迟了一点！

　　怎么会？这句话你应该和釉子说去！知道吗？有时候女人做出了决定是希望男人来否定她。所以现在也许釉子虽然决定离开，但是实际上却正在等着你的这句话呢？我很和气的说。

　　酱瓜笑，你也说有时候了？那就不是绝对了？我们都知道其实釉子不是一个容易下决定的女人，但是下了决定她又是一个坚决不肯回头的人！好比你！

　　我没有话可以说！我了解！

　　那就什么也不需要说了！真的要说话的话，我可以说其实也不晚，这个经验可以留待下次用！但是我没有说，因为我看见了酱瓜发亮的眼睛里面的一种凄凉，好象孩子损失了什么心爱的玩具一样的凄凉和无奈。这样的神情我见的不多。我看着他，似乎有点欣赏他的这点凄凉未绝的样子。很有点好男人的姿态。

　　我和酱瓜就这样相互看着。我们的眼神里充满了想要给对方安慰的神色！

　　忽然酱瓜眼睛里什么东西一闪，然后说，今天是什么日子？

　　我说，也许是4月1日！

　　什么也许？简直是确实的事情！他说！

　　我们忽然笑了，笑的很可爱！

　　所以你要原谅我刚才的话也许不是真话！

　　彼此彼此！我笑！

　　这样好不好玩？酱瓜说，我看见你刚才的脸表现的特别的悲切！似乎在同情我！

　　这个游戏很绝妙！我说，我们彼此都很入戏！

　　是呀！也就是你可以这样配合我说话做事！酱瓜的脸上写满了愉悦的表情。我也一样！

　　我把手边的报纸挪了一下位置，逼开了酱瓜的视线。报纸上的日期是：4月2日！
