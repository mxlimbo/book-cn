蓝色月亮（三）
　　和南瓜认识的第10天，沈哲回来了！

　　很莫名的我忘记了很多东西，但是记得这样没什么大不了的事情时间。

　　沈哲是我的朋友。

　　快十年了。

　　没有什么波澜，就象水一样的感情。像水一样渗透到彼此生命，也象水一样不可或缺。

　　那天，主任还是老样子，打着官腔。我生平最害怕官腔，一听见就有逃的恐惧。

　　所以我要逃。

　　我的脸上含着笑，很温柔的请示："我去一下洗手间！"

　　主任没有怀疑。他从来没有怀疑过我对他的厌恶，因为我一向掩饰的很好--我知道他是主任，他有权利这样做。我的逃跑是我的责任，当然不可以因为我的责任而加重主任的不快乐。：）

　　所以我走到了洗手间。

　　没有进去。

　　没有进去的必要。

　　我就在门口作了几个鬼脸，很愤怒的很痛快的发泄自己的情绪。反正主任还是在外面训话。

　　完了以后，就听见有人说："我知道你在这里！"

　　是沈哲。

　　脸上带了一种把我了若指掌的表情！

　　沈哲回来了！

　　我的心不由开始快活！

　　"怎么回来了？"我装着闲闲的问。

　　"这里有一个奇怪的女人！"他说。我没有听清楚他的话，因为墙上闹钟提醒说是上网的时间。

　　沈哲回来了。牺牲一下上网的时间是应该的。

　　十年的感情总是强过10天的感情。

　　可是我很生硬的下逐客令："拜托，明天来！我上网了！"

　　"这样急？？呵呵，是不是有网恋？"

　　"也许吧！"

　　我只是笑。心里好象还指望沈哲生生气，或者死皮赖脸的留下来。

　　可是沈哲走了。

　　剩下我和南瓜。

　　继续我和南瓜特有的快乐！

　　至于想到我和南瓜提出见面，是因为我被人家误认为--"是陈小姐？"一个胖胖的男人走过来问。

　　"啊？"

　　"对不起，我认错了，我和网友约好在这里的！"胖胖的男人有点窘。

　　平为了这件事情笑了半天："如果你是陈小姐，他会半夜里笑醒！"

　　"什么呀啊？"

　　"当然了，聊的来的网友刚好漂亮的要命，那不是人生一大乐事？"平抱住了我，继续不怀好意的笑："也许，南瓜长的还真的像南瓜！"

　　"恩--圆圆的？没有关系呀！！那叫可爱！"我不以为然的说。确实，关于网友见面我不知道人家如何想好象有人把外貌放在第一，连痞子菜也不能免俗：轻舞飞扬没有出现的时候他很拽，等到漂亮得不可思议的轻舞飞扬出现了，他就缩短了一半。

　　我觉得既然和他聊天愉快，那么见面的话，不论他长的如何，一定很愉快！长的如何我不介意，管他长的圆圆的还是方方的呢？
