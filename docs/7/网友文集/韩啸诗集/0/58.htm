母爱 永远的嫁衣

即使世间最美的诗句
也无法形容您 万分之一的美丽
即使用恒古的生命
也无法将您对我的恩赐 代替
是您 独自用柔弱的身躯
为我挡住世俗的风雨
是您 在灯下把慈爱注入针脚
为我注入一世的情谊
无声无息 恍惚中
您已老去
但心中最温柔的角落
深藏的爱意仍是那样的浓郁
我已决议
要用短暂的生命
千倍偿还您的情谊
然而 您却说我是一片羽翼
等到成熟 丰满
就该飞去
此时 您心海的波涛
已涌上我的眼底
并且 润湿了我疼痛的记忆
风霜憔悴的母亲啊
您可知 您可知啊
母爱 才是我一生永远的嫁衣
