厄运，你何必要这样

厄运，厄运你快过来
你快过来坐在我身旁
你不必怕，快过来
让我们坐下来想一想

你何必苦苦的把我跟随
你用你的本领
让我痛苦，让我悲伤
而我用我的坚强奋力抵抗
结果我们两败俱伤
厄运啊厄运
你何必要这样

你认为生活是一场诡计
我认为生活是一场斗争
你毁我前程 毁我爱情
我用坚强扼你喉咙
捅你心脏
你看，你看
我们两败俱伤
你何必要这样

你知道
我在你面前永远不会恐惧
我也知道
你在我面前暂时也不会退让
既然如此
我们终会两败俱伤
厄运啊厄运
你何必要这样
