琥珀的爱

要多少次孤寂的雨
要多少次旷野的风
才能孕育出前世的我
才能孕育出今生的你
又是怎样的宿命
怎样的谬误
让我们相隔了几个世纪
于是 我甘心触落岁月的泪水
把自己滴入了千年的等待
只为我们今生的相遇

岁月掠过苍茫的大地
我在黑暗中祈求了千年
请让我在生命中最美的时刻
与你相遇
再把我琥珀的心捧奉给你

终于
你带着微笑向我走来
自我崩溃的边缘
自我无悔的等待
自我凝固的心灵
自我千年的企盼
是怎样熟悉的一双手啊
带着甜蜜的温存
带着芳香的眩晕
把我从乱石中拾取
这时,这时你可听到我热情的心跳
你可看到我晶莹的泪花
我那深埋的芳心
我那凝固的爱情
在你的手中慢慢融化
经历了千劫百难
我有了精致的轮廓
我有了耀眼的光泽
这一切
都是为你准备了千年的杰作
生生世世 反复雕磨

请你,请你赶快收留我
一如黑夜收留星辰
一如大地收留花朵
想必此时你也从我眼中读出了急切的哀求
然而,你为何,你为何啊
又这样轻易的转身离去
暮色中,把我遗弃在最最荒凉的
黑夜里

为什麽在岁月的轮回里
总是与你擦肩而去
为什麽用血泪凝聚的生命
你却毫不珍惜
我那曾经粉身碎骨的抉择
我那曾经全力以赴的努力
难道正是你准备了
一千年的 骗局

带着我的依恋你渐渐远去
是不是 是不是只有离去
才能平息
平息我忘年的爱恋
平息我如火的热情

终于 我又慢慢凝结
一如前世
今夜 云也寂寞
风也无语
