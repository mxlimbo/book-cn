话别

曲终人散 你终将离去
我毕生所敬仰的人啊
请不要哭泣
人生本是一场无果的悲喜
且让我们把回忆珍惜
带着依恋相互祝福
只为我们曾经的相遇

曲终人散 我也终将离去
我毕生所爱慕的人啊
请莫道别离
让我们把情谊深藏心底
在某个夏日 某个黄昏
我依旧会含着笑容把你想起
那时 我不会哭泣
无论怎样 我会庆幸
庆幸曾经与你携手走过青春的绿地
与你共过尘世的风雨

然而 曲终人散
我们终将离去
在无数个夜里
我们会深深的把彼此惦记
