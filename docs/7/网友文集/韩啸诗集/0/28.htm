心灵涟漪

当我为一朵落花流泪的时候
请你相信我是感动的
当我独自在雨中徘徊的时候
请你相信我是寂寞的
当我静静的坐在窗口等你归来的时候
请你相信我是思念的
当我在你手心写下那三个字的时候
请你相信我是真诚的
当我沉默的时候
请你相信我是渴望的
当我离去的时候
请你相信我是眷恋的
当我躲避的时候
请你相信我是茫然的
当我叹息的时候
请你相信我是无助的
当你在自己的幸福中欢乐的时候
请你相信我是带着微笑送去祝愿的
当所有往事如烟如云消散的时候
请你相信我的泪眼中是饱含着幸福的
