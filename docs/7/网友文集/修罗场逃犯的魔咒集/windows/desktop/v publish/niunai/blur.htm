落入我视网中的Blur
 

    
你有什么骄傲得可以不被我操纵
徒手把你强扭向我
对着你粲然一笑
知道你的冷淡始终还是为了诱惑我

盯住你的脸
清楚它的只是属于我 而得意
既然如此
就试着改变你 用KPT

进入 仿佛
含食了彩色小药丸 幻药发作
快乐导致心痛得直笑 掌握着你
不肯罢手

概念已不存在 满脑欢愉
欢愉等于麻木
Filter漂浮在半空中

无意识的从你脸上移走视线
竟也发现 我已将你的样子藏于白墙、窗间
察觉时间正被贪恋你的理由偷走
开始依依不舍的厌恶你

离开 但必须说
我爱你就是^S

1999/06
事后注解：酸奶99年的文字，写的是由于长时间对着电脑屏幕做设计图而迷糊了眼睛的事情。
