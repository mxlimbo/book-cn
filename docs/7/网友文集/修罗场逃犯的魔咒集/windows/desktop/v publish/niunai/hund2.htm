一封叫‘你骂我是混蛋’的信
 

    
    就算我是混蛋，也是一个诚实的混蛋，善良的混蛋。也是一个真心对待你的混蛋！！！这点我没有骗你。
    路一开始走的就不太正确。到了后来才发现离我的初衷越来越远，我尽力想去弥补，可是我不能有效的控制住自己的感情。但是你也说过，早晚要有不开心的时候，没想到这么快。。

    如果你需要我消失，我会消失的，只要你给我一句话：你消失吧。
    如果我听到了这句话，那一瞬间我会哭的。虽然我不喜欢自己哭。。。

XXX于X年X月X日

    事后：
    为什么要放他的信在这里，我不清楚。

    不过最终我还是看到他哭了。
    在初夏暗夜中的三里屯酒吧街头，他抽泣得像个真正的孩子。当时有使人陶醉的香风与热浪涌到我身边，为此我也流下了眼泪。

    知道我喜欢他什么嘛？！
    我喜欢他像个孩子，他的脸庞纯真而让人不设防。我爱他微笑时候的样子。我不敢爱他。
