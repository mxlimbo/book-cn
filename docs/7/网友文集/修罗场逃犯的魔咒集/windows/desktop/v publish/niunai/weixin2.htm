沉醉于自己的唯心
 

    
    猛一抬头，街灯都亮了。
    原本棕蓝的天空，变成了橘色。立刻，都市的感觉又回来了。

    在我坐的高台上的另一端有个忧郁少年。他抱着膝盖独自唱着歌，轻轻的，应该是情歌。即使不是情歌，他弓着的背影里也流露着情爱两字。
    他的忧郁应该是幸福的。

    烟蒂在我手指间变烫，然后被我捻熄，它的任务就是为了陪我消遣。

    扭秧歌的人散了，大家全都心满意足、意犹未尽的道别。虽然明天他们还会见面。
    我盯着四散的影子在广场上滑过。
    一下子便安静了。
    风也来加速流动，暗示着人流的减少。

    我突然又翻找不到我的打火机了。
    空气再次干净了８秒钟。
    车没有减少的企图。

    不远处的街心公园里居然有狗在叫，很惊惶的那种叫声，象在求救。
    在昏黄的天景下，凭空添了些许乱世的气氛。

    我穿着条雪白的长裤，此时它的上面全部都是烟迹，灰蒙蒙的，迷彩一样。

    估计要下雨了，风里面有这种资讯的传达。
    我没有要走的意思。
