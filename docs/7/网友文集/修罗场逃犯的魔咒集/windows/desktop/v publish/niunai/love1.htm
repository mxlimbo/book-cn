我为了爱你
 

    
苍蝇踩着自己的脚
用唾液抚摩轻薄翅翼
堇色复目突然一闭
思付美丽光芒四射的热力
从吸管里发出微微笑意
垃圾也顺畅的流入嘴里
汁水不再外溢

不知它是否有灵敏嗅觉
还是腐肉本身便格外会发出信息
食物与咬啮者彼此相互需要
也是天理

狠狠吃你
就表示非常喜爱你
吃完你
还要再甜蜜的亲亲你
希望下次快快遇到你兄弟
让他也添满我肚皮
让他也知道我甜蜜

摩擦后腿
抱着自己的头
舔净指尖
保持卫生和礼仪
得意也不能够忘乎所以

知道太阳披金挂银
所以孤独并势无可敌
抓把空气却只握到手掌指须
