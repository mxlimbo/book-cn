沉醉于自己的唯心
 

    
    周末。无事，但不想太早回家。
    随意便走到了北海。我喜欢那里。
    湖面有暗暗的起伏翻涌，我突然就想抽烟了。开始四处寻找卖烟的摊位。

    在故宫的门口，当我问起哪里有香烟的时候，居然有些人肯帮我一起找。
    终于找到一家，我一付很高兴的样子。随后，花了很贵的价钱，买了包奢侈的香烟。
    原是想在护城河边抽掉它，但没有这么做。
    那里的空气干净，虽然有淡淡的死水味道在空气中弥漫。可眼前，雾气中绿树红墙依然使人感觉到了六零年代的单纯与激情，河边都是恋人与饭后的悠闲客。他们真心的在一起拥抱、接吻，或者是育子、谈心。
    所有张望我的人都断定我喝醉了，因为我的眼神里全是醉意。一种张狂放纵的醉意。我向他们笑，笑得很单纯，不给他们以任何风情的想象。

    当走到美术馆，暮色已经黯淡得可以使人不必隐藏起自己的表情了。
    我一个人坐在美术馆高高的“萨尔瓦多·达利原画展”的大宣传牌的台子上。居高临下。
    扭秧歌的人们在反复表演着同样的动作，我淡淡的看着他们，烟迷住了我的眼。
    天色棕暗，风与云彩都浑浊不堪，本来应该没什么风景可欣赏。但是当我想再抽烟却找不到打火机的时候，突然觉得天空清冷了许多。
    我又仿佛回到了幼稚冲动的少年时代。
