一封叫‘你骂我是混蛋’的信
 

    
    你骂我是混蛋。。
    你骂我是混蛋。。。
    你骂我是混蛋。。。。。。！！

    我不是！！我从来没有骗过你！我什么事情都告诉你，我把我的感觉告诉你。我只是说“我会做我能够做的事情让你高兴，让你快乐！”我不知道你想到哪去了！我也不想知道，因为如果你真的那么想，我觉得你是在侮辱我，也是在侮辱你自己！

    如果你有一天要死掉，我会是哭的最伤心的一人，如果你吃了我的心脏，就能活下去，我会给你的。因为我很害怕你消失的日子。与其你消失不如我消失。

    我跟你在一起我从来就没有很轻浮的感觉，因为我不是那种人，你也不是。我能够对你坦诚相待，我知道也许咱们一开始就走的不那么正确，可是在不正确的路上，你和我真的是很快乐的在一起，虽然时间不长，可是你真的是我生活中的一部分了，我喜欢跟你说话，喜欢跟你去炸街，喜欢跟你饥肠辘辘流窜在马路上。有钱的时候大家挥霍，没钱的时候一起挨饿，一切都是那么的有趣，那么的开心，我从不隐瞒自己的感觉，我从不欺骗你，因为我不允许自己言不由衷，那样跟流氓没有两样。

    我喜欢你的性格，我喜欢你的思想，我喜欢你说话的语气，我喜欢你的外表。。。有很多很多。。。我没有骗你。因为这样，所以我能够做的就是让你跟我在一起的时光更加的快乐，更加的有趣，让你能开开心心的面对生活的每一天，我希望我能够做得到这一点。可是我现在发现这些要求我无能为力了。你骂我混蛋。。。我真的很难受。。。。
