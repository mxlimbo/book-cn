用文字的错觉扮演自己
 


    我总说文字会带给人错觉。很多人从我的文字上认为我吸烟很凶。其实我只有偶尔无聊才想起来抽上一两支。我告诉他们，那是文字的错觉。

    正常的人都希望受人注目。特别是那些贪恋幻想与喜爱表现自己的人。受人注目带给人的成就感总比委屈要多得多。那些诉求自己没有隐私的明星也只是想让广大平凡的民众心理平衡一些而已。

    因为文字的错觉。开始有人利用文字来表演了。他们或者她们把自己表演得醒目、特立独行、让人远远观望就有小宇宙般的光芒笼罩。实在是使我羡慕之极。
    我长得不美，没有很傲人的容颜，但是我心底却很想惹人瞩目，所以我也学人家用文字来表演，躲在文字后面重新设计自己，这是无奈的选择。毕竟还是既有声又有色的表演看着让自己更过瘾。

    我就是如此的虚荣与浅薄。
    用文字反复修饰都掩盖不住。

    于是策划开始。

    我想象自己应该是这个样子的：
    “那是一个美丽而眼神忧郁的女孩子，留着顺滑轻盈的长发，柔软的手指总是搭在苍白的另一边胳膊上。
    “我习惯沉默不语，喜欢淡淡微笑，穿白色纱裙迎着风站在窗前。
    “房间里充满着音乐，我时常躲在里面看书，画画或者其他。
    “与世无争。
    “或许我还会在一个无事的下午给自己开出些中药方子来。
    “芳草味道萦绕在我的房间与身体之中。”
