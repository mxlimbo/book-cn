好色，是你活该倒霉的理由
 


    唉，我从来没有对别人说过这些。他在那边叹气。随后便把我当成鼻涕纸篓，向我投掷被泪水与其他浸透的手纸。
    我边接着他投掷过来的湿纸，边面带同情悲伤地准确做出相应的回馈“恩，啊，就是”，同时感叹道：这么一个漂亮的人居然如此信任我，我真是欣慰。随即也流下了热泪。

    他的经历大致如此：
    原来他也如同我一样好色，他的女友好似小妖精一样迷人，长腿美肤，料想别的功力应该也会不错。
    就因为她妖精一样的美，所以，他活该倒霉。
    他为她策划美好未来，为她实现更像妖精而所需要的装扮道具，为她不负责任的妈还债，还有些我就不方便在这里向大家描述了。
    看来他也只可以对我一个人说出来了，我还能很善良的报以同情。但是如果他的小妖精女友听到，应该只会冷漠一笑而已。

    同样一个性情的女孩子或男孩子，她（他）美，所以你就有可能付出的更多。
    选择与你患难的人，如果她善良，并很养眼，简直是天赐的好事情，人人都会羡慕你。
    但是你选择她的原因只是因为她很养眼却将她的不善良视而不见，那么说你活该倒霉的人就有可能不止一两个了。

    看来我以后还是信奉距离产生美的概念好得多。
