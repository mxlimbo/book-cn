好色，是你活该倒霉的理由
 


    我喜欢漂亮的人，无论男女。
    不过，这丝毫不意味着我对所有漂亮的男女都想入非非，至少我对漂亮的同性比较手下留情。

    每当看到使人悦目的容颜，都能大大增加我乐意与他或她交谈的渴望度。
    并且对经历同样悲惨遭遇的人，如果你漂亮一些也更会使我的同情心高高升起，心里一定会想：噢，这么可爱的一个人，居然如此可怜。简直……更让人想去爱了。

    我认为与我怀有同样不齿心念的人相当多，所以一直也不觉得这会是什么遭受劫难的缺口。

    我从前的男朋友很帅，虽然不是漂亮但却十分惹人注目。
    我几乎因为他的漂亮而娇纵他。这么可爱的一张脸，谁会忍心去指责他什么那。

    但是，可能结果大家都猜的出来。我们最终还是分手了。分手的时候我没有当面和他诉说任何理由，原因是我怕看到他那张可爱漂亮的小脸儿而心软。

    没过几天，我就被一个午夜电话惊醒了。
    这是另外一个很漂亮的男生打过来的。他的美以浪子式的忧郁和冷漠而著名。
    现在看不到他话筒那方的脸，不过他的声音倒还是很忧郁。仿佛他漂亮的脸一样。
    聊了几句，我们就开始划入正题。
    原来他失恋了。
