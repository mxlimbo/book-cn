我是如何当上修罗场逃犯的
 

    
    在网路上，我的名字是我爱酸奶。我的身份是修罗场逃犯。

    我从来都没有解释过，为什么要给自己的身份设计为“修罗场逃犯”。
    不过也没有人真的问过我这个问题，他们大多都只会问我是否真的喜欢酸奶而已。

    据说，修罗场是这样一个地方。
    它惨烈壮美尸横遍野，有英雄与智者为此流血和眼泪。

    书上描述的修罗族很奇特，他们的女子都极美，男子都丑陋不堪。性格残劣，并把毁灭当成本族的神谕来信奉与执行。
    修罗场恰好是他们与天帝战斗的地方。

    我渴望如此地生死一回。
    站在他们其中，我会血流加速，欣喜若狂。

    可是我又害怕，胆战心惊，因为我总是想让自己多看一些这世界的。
    每个人都是如此。
    人不愿意死的原因大多是这样。

    所以我决定，让自己的确在那个惨烈壮美尸横遍野的地方生存过。
    不过，我很奋力地逃了出来。
    虽然，我随时都有可能会被同伴或敌人发现，并抓了回去。但，我还是会很快乐地接纳我自己的命运。
    等有人把我抓回去，或者让我留在这里继续偷看世界。
