我要写故事的原因
 


    我现在写的东西依然有很多是幻想来的。幻想不来，我就偷。
    我偷别人的经历，偷别人的梦境，偷别人的电影情节。然后把他们变成我自己的故事，但是这故事里却惟独没有我自己的影子在其中。
    我总是悬在这些故事的上面，低着头好奇的窥视这些故事里的人。那些有高潮迭起人生的人。
    我很羡慕他们可以永无死亡，即使死了，你再翻开一次书，他们又可以复活，然后再次投入地把他们的心碎，他们不可告人的秘密，他们到死都不忘的痛苦，他们的人生重新演一次给你看。

    我对此羡慕得五体投地。
    永生比痛苦更值得我向往。

    我不想多说了。
    这就是我要写故事的原因。
