看A片学日语
 


    别问你爸妈A片是什么，永远。这对你没好处。
    当然，我这里是指真正意义上的A片，而不是什么America片。
    幸好，我爸妈纯良得根本不知道A片为何物，所以我能在他们的眼皮底下，如此坦然的坐在电脑前噼里啪啦的敲着键盘打着字写着这篇小作文儿。

    下面我要揭发一件大多数群众不大能相信的事实，那就是，我从来都没有看过A片。
    我比我爸妈要进步许多，我知道A片为何物。但是我却依然纯良得从来都没有看过它。
    绝对不是装清纯，我向来实话实说。

    为此我遭受到了不少进步群众的耻笑。这些群众丝毫看不得我落后，一定要邀我共同提高。
    所以她们推荐给我不少经典的A片，想来是多年珍藏。日本台湾澎湖列岛大韩民国美洲欧洲各国各省包罗万象。她们肯借给我看，料定我是太脱离群众使她们恨铁不成钢盼我能早日成材。
    不过我这个人不思进取，借给我，我便随手丢到角落里去了，根本就想不起来学习。

    过了好久，才又有人肯借我学习材料看。
    名字与题材我都很喜欢，叫《看A片学日语》。

    听了借盘人的介绍。大致是，一对或其他数量的男女咿咿呀呀的演上一场，随即暂停，屏幕的下方就会跳出一个人，给你讲解这段场景期间发出声音的中文涵义。并对重点句式进行回放、字幕与详细分析。
