用文字的错觉扮演自己
 


    草样出来了。
    我沉迷于自己的方案里，很是得意。
    开始托着下巴去想象应该发生在她身上的故事。
    让她经历什么我才甘心那？
    不过每次我都沮丧不已，因为我不熟悉她。那个美丽忧郁的她。

    我的想象力贫乏。

    所以我放弃了这个设计。决定死去以后重新再来。

    这次的我应该是这个样子的：
    “那是一个傲慢无理的女孩子。我偏爱水蓝色的眼影、短裙与靴子、大波浪的卷毛头和在手指上带许多硕大的戒指。
    “红艳嘴唇上总是叼着黑细的雪茄，它会在我说话的时候随着嘴唇的波动而一翘一翘，并且永远抽不到头。
    “我喜欢坐在北京街头的台阶上，身后是大红门、从住家中探出来的小台子上的私人公用电话和小卖部的各式食品。我可以一个人悄悄喝着酸奶。或许可以窃听到那些公用电话的部分内容，偷笑或其他。
    “在夜晚，我孤单的坐在任意一个地方，让风把寂寞刮向我，然后暗暗流泪微笑。”

    我又是一番得意。
    我觉得她这副样子，应该故事更多一些。天马行空也不怕。

    可是我又失败了，我总是让她风尘得不太彻底。外表的颓废是有了，不过我却又惦记让她的心灵无法摆脱理智与善良。
    由于我的能力有限，她最后险些变成了一个亦文亦武、打扮古怪的小学教师。
