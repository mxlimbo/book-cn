这里的生活（代为序）
 

    
    我不知道自己需要什么样的生活，但我清楚，自己绝对不会容忍压抑的日子。被压抑所操纵我会疯掉的。

    我是个懒惰的人，我唯一不偷懒的事情就是找乐子。我几乎一度把找乐子当作自己忠贞不二的生活目标与信念。我不停的告诉自己必须要坚强的活下去，为那些我还没找到的乐子活下去。
    毕竟，人有了信念，活着就容易多了。
    可遗憾的是，我操纵乐子的手段不佳，乐子到最后总会给我带来麻烦。于是为此我又要把自己锻炼成甩掉麻烦的高手。以此类推。
    
    有个人对我说：事在人为。
    心情好的时候，我会赞同，一切尽在掌握。
    心情不好的时候，我会笑着向他脸上吐口水，自欺欺人。

    我实际上是个极具暴力倾向的人，内心歹毒，时刻想象着攻击他人时候带给自己的快乐。听摇滚乐的原因之一就是因为它能给我一个发挥暴力的理由和借口。
    不过值得放心的是，我还不属于那种走在街上听到摇滚乐的鼓点拨弦就起身打人的家伙，我还在极力扮演着淑女。

    穿着白色短裙，装饰银色项链，拥有美丽发型，微笑着与别人谈论着作者名字拗口的书籍。
    这是我理想中的自己。

    但是，无论怎么看，淑女都与我很不成一格。
