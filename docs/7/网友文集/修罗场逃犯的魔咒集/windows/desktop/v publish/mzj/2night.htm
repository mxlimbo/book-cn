陪虫子睡觉的酒醉夜晚
 

    
    正走着，前面一座废墟里已经走进的阿俐就大叫起来。啊呀呀地求救着。

    三个人慌忙奔过去，只见阿俐揽着眼镜的脖子惊恐的大喊：“虫子呀！”
    眼镜脱了一只鞋子，奋力地要去打它。
    大家都嘘了口气。

    废墟的土墙上有只美丽的金兰色硬壳虫。脊背上晶莹的闪着光。
    霏霏没见过这么漂亮的虫子。
    它此刻正被阿俐的叫声和眼镜的鞋子吓得左奔右跑，不知如何是好。

    “不要打它，很可怜的。”霏霏拦道。
    阿俐抱着眼镜还在惊恐的大叫。眼镜一边穿上鞋子一边搂住阿俐对虫子说：“丑八怪，不要再吓我女朋友了啊。小心要我下次见到你，一定扁你浑身出绿汁。你还是好好谢谢霏霏救你吧。阿俐不怕。我们走啦。”阿俐含着眼泪点点头。
    于是，两个人很甜蜜的相拥而去，阿俐满目小可爱的笑意。眼镜更是气焰嚣张得意的要死。

    李墨和小捷不知道什么时候也已经走出废墟了。

    霏霏耸耸肩膀看他们走远，便蹲下身来仔细地端详着已经安静下来的虫子，然后用手指碰碰它美丽闪亮的脊背，叹了口气，很亲爱的对它说：“快走吧，这里全部都是人，我们和你不是同类的。你以后要多保护自己呀！”
    霏霏说完，觉得虫子在对自己笑，她感到自己也很开心，于是向虫子说了再见转身追回到队伍中去了。
