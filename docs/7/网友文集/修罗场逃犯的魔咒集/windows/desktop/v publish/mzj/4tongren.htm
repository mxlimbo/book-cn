嫉妒的瞳仁
 


    “不许喜欢我，听到没有？”丁微躲在罗兵的怀里呜咽：“你只可以喜欢我妹妹。”
    罗兵忧郁地将她揽得更紧……

    中国北京天安门广场上的中轴线。
    两个仿佛死期将近、没有未来的年轻人，拥抱在一起。
    没有放松的意思。
    
    正如他们刚才所谈到的：相依为命。

    周围恰好有个西班牙摄影师，他看到了，认定这是个好场面。
    于是他拍了下来。

    几个月以后。

    首都机场。
    丁闪很快乐地奔向罗兵。罗兵抱住她转了几个圈。
    两个人笑做一团。
    丁微在旁边看着他们有些怅然。她很小心的警告自己不要妒忌。
    妹妹能得到幸福，自己应该也很开心。

    丁闪转过头来，看着姐姐说：“姐，你好瘦呀？！简直小了两个号码。”
    丁微也笑着走过去搂丁闪：“是嘛？我自己都没察觉。”同时去看罗兵，她真的瘦了？
    罗兵的眼神已经开始有些躲闪，他应当爱的人回来了。
    丁微轻轻叹了口气，当真开始有些妒忌。
