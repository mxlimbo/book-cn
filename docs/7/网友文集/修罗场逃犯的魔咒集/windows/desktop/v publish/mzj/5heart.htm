突突的心脏
 


    “我会活下去。不但为了那些我应该拿到的报酬，也为了我爱的那个人的心。”

    猫想起来她以前爱的人是如何死掉的了。
    月亮的阴影里有血的香气。猫从来都觉得鲜血是浓香的，仿佛咖啡一样得诱惑，引人兴奋。

    她掏出刀子。

    …………

    ◆◆◆◆◆◆ 
    
    猫是个相貌忧郁的女孩子。甚至有些文弱，肌肤很柔软。或许她并不忧郁，但是她相貌如此，很容易就让人联想到这个词。

    不过，她没有高估自己的魄力和冷酷。
