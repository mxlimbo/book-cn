陪虫子睡觉的酒醉夜晚
 

    
    天晚的很快。星月渐渐浮出。
    篝火已经点好了。一团橙子般的愉快颜色，黑色的枯木在火中噼啪的响。
    羊肉在这个时候已经烤好了，眼镜抱着酒瓶大口喝着白酒，李墨拿着刀子很小心地割肉，女孩子们理所应当地坐享其成。

    同伴们都在围着篝火狂欢。霏霏思量着，犹豫了许久，独自走到不远处的沙丘上观看热闹的场景。
    霏霏喜欢在阴暗的角落里观望明亮时候的感觉，她觉得这时候的 明亮更值得向往。

    仿佛期望中的一样。
    不一会儿，李墨便趟着软沙走了上来：“喂，快去吃羊肉吧！小捷在叫你。”
    霏霏起身抬头看天上的月亮，它有种要坠落的感觉：“你知道吗？大家都说月亮是有魔力的。古代的人之所以比我们聪明，就是因为他们喜欢看月亮。”
    “你不需要看月亮，你已经很聪明了。”李墨看着霏霏朦胧的脸说。
    霏霏也微笑着回看他：“没想到你会如此评价我。”
    李墨接着评价她：“不过，聪明人总喜欢做傻事情。”霏霏这才明白，没想到的话却在后面。于是盯住他：“什么？”
    “没什么。”李墨突然也微笑了。他不是很喜欢笑的人：“你也快些找个能够和你相互喜欢的人吧，我希望看到你快乐。毕竟我们同学很多年了。”他快速的补充了一句：“我有自己喜欢的人了。”

    看来她是得到答案了，霏霏忍了忍眼泪，微笑了一下。
    李墨没做什么表情。不过，这和他平时一样，他平时几乎也是如此的没有表情。
