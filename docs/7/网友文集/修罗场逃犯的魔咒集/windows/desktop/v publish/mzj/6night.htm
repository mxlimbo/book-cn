陪虫子睡觉的酒醉夜晚
 

    
    霏霏觉得这声音真美好：“那我应该怎么做才可以？”
    “我教你。按照我说的去做。如果你想达成愿望的话……”

    霏霏听到费力搬运的声音，静了一会，那声音对她说：“这是三根玫瑰花刺，明天你在花刺上涂上鲜红色的指甲油，然后将三根花刺偷偷地放在他的影子里，并且用脚把花刺踏入泥沙中，从此以后他的灵魂就会被你的玫瑰花刺钉住，牢牢地钉住。从此以后他便逃不了了。”
    “逃不了了？！”霏霏有些胆战心寒，昏沉着头想这个词的涵义。

    突然惊醒了。

    她发觉天已经大亮。
    霏霏还有些恍惚，不知道自己昨晚是否真的喝醉了。但是，当她躺在床上偏过头去看窗外的时候，却发现枕边当真有三根玫瑰花刺。

    霏霏起身走出房间，发觉李墨他们也全部都醒来了，正站在院子里欣赏清晨中微凉的沙漠。
    小捷看到霏霏走出来，立刻跟过来，关切的问她头还疼不疼。

    霏霏不好意思的笑，把头发拢到耳后，依然偷眼看李墨的表情。手里握着玫瑰花刺，捏着有些刺痛，却不由得越握越紧。
