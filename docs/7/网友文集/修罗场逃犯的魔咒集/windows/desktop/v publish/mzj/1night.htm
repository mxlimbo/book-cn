陪虫子睡觉的酒醉夜晚
 

    
    一起去沙漠旅行的有五个人，都是大学时代的同学。
    但是霏霏参加它只是因为其中的那一个。
    李墨有些让人难以琢磨，霏霏认为他也应该是在意自己的，可却不明白他为什么总是使人感觉那么疏远。霏霏想，或许多一点亲近的机会，他便能告诉自己那些无论是她愿意听到还是不愿意听到的话了。
    所以，霏霏决定应该趁这次旅行给自己一个答案。

    今天是行程的第二天。昨晚，霏霏和同床的小捷说了一夜的话，讲的事情大多都与感情有关，所以现在有些累。
    车子马上就要开进更深的沙漠了。
    阿俐、眼镜这一对情人和小捷坐在后面专心地昏昏欲睡，李墨专心地开着车子，霏霏则专心地支着下巴看窗外的风景。
    窗外几乎没有什么景色可以看。
    沙丘和荆棘在烈日下没有丝毫光泽反射。

    很辛苦才将车子开到了一家小旅馆。
    刚找好房间，阿俐和眼镜就牵着手向外跑，分明不是方才睡神一般的样子。霏霏和小捷也随后亲密交谈着走进大漠。李墨跟在两个女孩子的后面。

    霏霏回头偷眼去看李墨。
    小捷很兴奋地对她说：“也不知道这里有没有骆驼可以骑啊？我很想骑骑看那。”
    霏霏也觉得这是个好主意。便折过头来问李墨是否也喜欢。
    李墨说今天晚上已经订好篝火和烤羊肉了，所以只可以明天才去骑骆驼。他一切都依照计划进行，向来如此。

    霏霏和小捷多少有些失望。
