突突的心脏
 

    
    突突凝视着猫，摇头。
    “为什么你不试着爱我一下那？！为什么不？只要一点点，总可以吧？”猫近乎是哀求。
    “我不可能同时爱两个人，我心里没有位置了。”
    猫觉得可笑。
    这不是她所能想象到的答案，这个答案幼稚得除了令人感到可笑再没有其他了。
    猫不相信她眼前的这个杀手一样男人会说出如此的话。
    “放弃这个念头，它不适合你！”
    “我不能放弃它，除非放弃我的生命。但是我的命很值钱，我永远都不会轻易放弃。”
    猫安静下来，含着眼泪说：“我还以为你的想法会和我一样。还以为你也应该爱上我。”
    “我从来没有琢磨过你究竟会怎么想，你的想法根本与我无关。”

    看来，猫还是没有人来爱她。
    她以前爱的人死了，现在爱的人却不爱她。
    猫感到有点绝望了。
    “你不能爱我？”她突然软弱下来，苍白得好像风中的白色小瓣花朵。“我得不到那笔钱了，我那笔本应该在自己活着完成任务以后才可以得到的钱。我清楚地知道，我会死在这里。现在，你的话使我更觉得如此。看来我真的不适合做这个职业，我高估了自己的魄力与冷酷。死亡来到的期限对我来说已经不重要了。”猫拖着枪走进了战壕。

    突突看着她的背影，转回头来。
    他觉得自己此刻险些爱上了这个悲伤的女孩子。
