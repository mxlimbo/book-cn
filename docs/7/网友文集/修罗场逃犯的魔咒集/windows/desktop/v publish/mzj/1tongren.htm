嫉妒的瞳仁
 


    春天就是如此，所有都开始重新孕育了。
    花草，云彩，阳光，念头，情欲，罪恶。

    工作结束了。
    丁微走在阳光下面。有阴影从她的脸上划过，仿佛剪纸一样的停留了一会便移走了。
    她懒懒地支起头看看天空，暗暗叹了口气。心里想，本还以为是块乌云或许有雨可以下，却原来又是个假象。

    最近丁微好像格外期望有雨可以滋润一下，她觉得自己已经干涩得要枯萎掉了。今天又是无事可做，天气燥热得总在提醒你可以少穿些衣服。
    丁微抚摩了一下微微湿漉的后颈，心里依然在想：总得做些什么才好呀。
    丁微觉得自己可耻，总要努力地去想才知道自己应该做什么。她很羡慕那些一天到晚都有事情做的人，她觉得那些人真的没有烦恼可言了。

    手机响了。
    丁微多少有些开心，还是有人会记得她的。
    电话那头是个年轻的声音：HI。是她妹妹的男朋友罗兵。丁微的妹妹丁闪去法国读书已经好几个月了，罗兵在无聊的时候总是会和丁微联系，共同打发一下时间。今天电话的目的也是如此。
    丁微觉得这没什么，毕竟都是孤单的人，而且都心有所属，又有什么可顾及的。

    到了约会的地点，罗兵却早已经等她了。
    罗兵远远看到丁微便很开心，孩子气的脸在阳光下微笑。发际间有隐隐的汗泽闪烁。在丁微的印象里，他永远都是个有些害羞的男孩子。
