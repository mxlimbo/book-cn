嫉妒的瞳仁
 


    罗兵见她不说话，便问她想什么。
    丁微一冲口就把心想地说了出来。
    说完又有些后悔。

    罗兵看着丁微只说了一句：“我明白。”
    丁微对他敷衍的一笑。表示赞同，不去和他争论。
    罗兵又拉住她说：“别把我当小孩子，其实我什么都明白。只是不想活得那么累而已。”
    “我也清楚，但是毕竟我比你经历的要多。我习惯沉湎于自己营造的世界里，这其实和旁人无关的。”
    “恩，所以就喜欢中轴线？”说完两个人重新笑了起来。
    “还喜欢黑社会。哈哈……”丁微大声补充。
    “喜欢黑社会？”
    “是啊，我觉得人在其中就会有种漂泊和相依为命的感觉。”
    “相依为命？不是啊。是不知道哪一天就死了的感觉。”罗兵反对。
    “正因为，不知道哪一天就死了。所以才有相依为命的感觉。”丁微的眼神越过高高的旗杆说：“是那种总看到夕阳，觉得美好，但是却明白地知道黑夜即将来临前的悲哀。”
    
    罗兵绕到丁微的面前，看定她，微笑着点头。
    刹那间，丁微感到一股忍受不住的凄凉：“别这么看着我。”

    丁微的本意是要推开他。
    不想，他们却突然猛地拥抱在了一起，很拼命的把对方往自己的怀里压。
    分明是一付坚决致死对方，同时也让对方坚决致死自己的壮烈场景。
