突突的心脏
 

    
    突突好像也应该是这样的人。猫心里琢磨着。于是，更觉得有理由去爱他了。

    这次行动报酬很高，猫主动要求和突突一组。
    但这次行动几乎是去者必死。
    很显然，参与者要钱的决心比要命的决心大很多。
    因为他们都有资格相信自己可以拿到这笔钱，他们一个比一个优秀。
    除了猫。

    猫参加这次行动，只是为了自己的打算。
    我已经说过了，她是个有打算便去做的女孩子。

    “其实，这次我是想来死的。”
    行动开始后的休憩中，猫走到突突的身边。坐低，温存地抱住突突的腿继续：“我失去了我爱的人，所以我准备去死。但是，我还是想得到爱！如果此时有人爱上我，我就不会那么轻易的死去了。”
    猫望着突突说：“你能帮助我得到解脱吗？你能爱我吗？”

    月色很迷茫。
    突突不去看猫，躲开她的眼睛：“每个人都会死，我帮不了你。”
    “但我现在是和你同生死的人，我们的性命是在一起的。这是雇佣兵的规则。”
    “如果你需要别的理由的话，我可以告诉你，我有太太和小孩子了。我是在为她们赚钱。”
    “他有太太？！他居然有太太……？！”猫的眼睛和心情里突然有了些怨恨的感觉。“我以为你这样的人，应该没有家。不过没有关系，我不在乎这些的。我只是要求你爱我而已。”
