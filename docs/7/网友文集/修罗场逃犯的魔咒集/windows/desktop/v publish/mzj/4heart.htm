突突的心脏
 

    
    敌人又冲上来了。
    同伴一个接着一个的死去。
    突突很谨慎地注意身边的丝毫动静。
    他身边有子弹飞梭，击中了在后面为他做掩护的猫。
    猫“啊”的一声倒入了战壕，鲜血崩溅出来。

    突突继续神勇地战斗。

    总算结束了。
    他们杀光了所有的敌人，而现在只有他一个人活了下来。
    突突享受了一口空气，至少他自己还活着。他已经习惯如此。

    这时候，突突听到干脆的几声枪响，后背仿佛被人猛地咬了一口，咬住不放，穷凶极恶、毫不犹豫得好似不要命一般。
    突突撕心裂肺的痛。他向自己面前的土地狠狠砸了下去。随之，有烟尘飞腾起来。

    突突很迅猛地死去了。
    一个被猫描述为沉着、精明、敏捷、冷漠的人，死去了。

    猫拎着自己滚烫的枪，从战壕里爬了出来。
    肩头中弹。

    她跪下来，将突突翻过身，看着他温热的尸体：“我清楚地知道，我会死在这里。可是，我现在改变主意了。”

    然后，露出迷人的微笑。
