陪虫子睡觉的酒醉夜晚
 

    
    沙漠里黑夜冷得出乎意料。霏霏才觉得自己穿得太少了，不停地抖，所有的意志都克制不住这股颤抖。她知道是掩饰不住了，便搂住自己默默地说了句：“我冷。”然后丢下李墨转身跑回到围着篝火的人群里去了。                  

    篝火前的人们都在说笑着。
    霏霏坐定在沙子里，望着火不敢出声音，她怕自己哭出来。
    身边的眼镜不知道在说着什么笑话，还没有讲完自己已经先笑到前仰后合了。
    霏霏突然什么都没有说，站起身一把抢过眼镜手中的酒瓶昂头就喝。正在大笑的眼镜好像被定住了身一样，瞪大眼睛看着霏霏豪爽，直到霏霏都快饮完才想到要夺她手里的酒瓶。
    阿俐和小捷也扑上来阻止：“不要喝那么急嘛，究竟是怎么回事情？”

    霏霏从来都没有喝过酒，但是她听说喝酒可以化解愁苦。

    直到喝完，她才知道错了。
    霏霏只感觉冷和头疼更加明显了，不停地有血管扩张撑裂的感觉袭击而来。
    什么还都没有来得及想，她就明白自己已经倒在了地上。

    她隐约记得李墨把自己横抱起来，大声的对自己说：“搂住我的脖子，好嘛？我送你回房间。”

    恍惚了好一阵，霏霏才接收到了一些类似知觉的东西。
    在这之前，霏霏以为自己死了。现在她只可以知道自己躺在客房里，外面依然是沙漠，房间在月色里一片漆黑，有一种没人气的清凉，坟墓一样的清凉。霏霏明白，想是大家体贴，所以决定让她一个人安静的休息。
