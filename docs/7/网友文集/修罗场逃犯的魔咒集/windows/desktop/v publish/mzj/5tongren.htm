嫉妒的瞳仁
 


    他应该是爱上过自己的。但是她主动放弃。为了妹妹，她应该成全他们，而不是成全自己。可她又那么的舍不得，妹妹回来，他就不可理所应当地同自己在一起了。
    她有些舍不得。

    眼睛渐渐模糊。    

    不知情的丁闪快活地拖着罗兵，在前面说笑，全然没有看出姐姐和男友的异样。
    罗兵也笑得过分。好像在做功课。

    丁微更是有点恍惚。
    她是疼爱妹妹的。一点都不想她受委屈。偏却自己也当真爱上了这个男孩。究竟什么道理。
    可，他们原本是一对，自己只不过是个后来者……
    他们原本是一对。

    妒火已经烫热了她自己的全身。
    烫。
    丁微有些发烧。

     突然，说笑着的丁闪望着丁微许久才说话。
    “姐，你，你的眼睛怎么了？”
     罗兵也回过头，愕然的盯住丁微。

    丁微不清楚自己怎么了，她停住。
    很紧张地维护着自己仅有的镇静。
    她要镇静。

    不让妒忌外露。
