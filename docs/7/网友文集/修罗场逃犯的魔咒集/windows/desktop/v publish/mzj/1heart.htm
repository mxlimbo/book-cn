突突的心脏
 

    
    猫是个相貌忧郁的女孩子。甚至有些文弱，肌肤很柔软。或许她并不忧郁，但是她相貌如此，很容易就让人联想到这个词。
    几乎没有人猜得出她的职业。
    她是个雇佣兵。一个给钱就把自己的性命卖给你的人。

    猫做这份很有前途的职业还没多久，所以她杀死的人也还不算多。不过她觉得自己杀的都是应该死的人。

    猫参加雇佣兵团的第一天就结识到突突了。

    一眼望过去，突突应该是个杀手，而不是一个雇佣兵。
    做为雇佣兵来讲，突突未免有些老了。
    看上去，他比猫的父亲年轻不了多少。
    但是他脸上表现出来的沉着、精明、敏捷、冷漠，却正是猫概念中的杀手。

    这里的人只有代号，谁也不知道他们的过去。至于未来，连他们自己也不知道。

    当神父把猫带到突突的面前时，突突正在往枪膛里装子弹，他只抬眼看了她零点二秒，便说：“怎么让个孩子来送死，她会拿不到自己全部报酬的。”

    那时候，猫就觉得自己有些爱上这个男人了。
    猫以前一直很辛苦地爱着的人刚刚死掉，所以她觉得找个新的爱人顺理成章。
    猫不太在乎突突有多大年纪，她觉得只要自己喜欢就好了。猫是有打算便去做的女孩子，干脆果断得让人觉得有些可怕。这真是不很符合她那张能令人联想到忧郁的面孔。
