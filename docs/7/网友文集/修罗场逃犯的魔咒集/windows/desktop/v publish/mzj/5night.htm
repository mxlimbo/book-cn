陪虫子睡觉的酒醉夜晚
 

    
    霏霏努力想眨眨眼睛，却丝毫都动不了。几个回合下来，她决定放弃想要恢复清醒的念头，就任由自己睡去。
    这应该是个好决策。

    睡去。

    是梦里嘛？
    霏霏晕陶陶地感觉自己的手指上有轻轻的痒。
    这痒是移动的，像几只小脚在爬。快速，急匆匆，一路小跑。
    跑，应该还带着些气喘吁吁。
    这感觉由她的手指游走到手背、胳膊、肩膀、脖子，一直跑到她的耳边。

    霏霏静静的躺着，甘心当它走的路。

    “喂！”有人在说话。
    霏霏动了动嘴唇，却张不开，只能专心的听着。

    “我是来感谢你救我的。我要帮你，这是我们的规则。必须要帮你才可以。”
    霏霏头脑里微微的想：“你能帮我什么那？”
    那声音又说：“我能帮你达到愿望。你所有的愿望。不需要你付出什么给我，只因为你搭救过我。”
    霏霏继续在梦里和它交谈：“可是我现在已经没有什么愿望了。我深爱的人丝毫都不爱我。他爱别人。他爱别人。如果可以的话，我唯一希望的就是他爱我，在我们之间没有任何阻碍。”
    那声音沉默了几秒就回答道：“如果是这样的话，我会让你快乐的，我会让他爱上你的。你将是他唯一的选择，一切阻碍都将化解。只要是阻隔在你们之间的，无论是什么，我都会帮你解决掉。”
