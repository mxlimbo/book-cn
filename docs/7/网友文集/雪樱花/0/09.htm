第五章 御缘
（2）


　　这一个傍晚，这一顿晚餐，这错综复杂的渊源，把他们两人推入了前所未有的一个神奇境地。回家的路上，车子开得飞快，两个人虽然默默无语，心潮却都在澎湃起伏着。

　　车到家门口停下了，但是康治没有熄火， "洗个澡，早点睡觉吧，累了。"

　　"……你，这就回家……了？"雪儿有点奇怪。

　　"今天说了不少让你生气的话，请你原谅。"康治垂下头盯着方向盘。

　　哦，不，我不怪你！雪儿在心里说道，这个时候，她是多么渴望他有一个温柔的眼神，热情的拥抱。可是康治丝毫没有反应， "明天我打电话给你。"

　　雪儿失望至极地下了车，整个动作都是缓缓的，她希望他说点什么来挽留她。但是没有，什么都没有发生，他坐在那里一动不动。他这是怎么了，雪儿不能理解，但又不能说什么。

　　头也不回地走到楼梯口，背后仍然没有他汽车发动的声音，于是她再也忍不住，猛地转回身朝他狂奔去。他急忙摇下窗玻璃，把手伸向她。

　　开门啊，为什么不开门？下车啊，为什么不下车？吻我啊，抱我啊，为什么不？

　　康治于是再不能克制自己的情绪，两双唇紧紧地吸住了对方……

　　"康治啊，你可知道，我们相遇的那一个夏天过后……"她开始哽咽起来。

　　"怎么了？发生了什么事？"他紧紧地将她搂在臂弯里，"有我在。"

　　"可是那时你在哪里？"她迷惘的眼睛里漾起一层雾气，幽幽地说道，"那是我们建国35周年的国庆节，我记得夜晚还爬上屋顶去看礼花和焰火的呢。可是第二天，我就发热病倒了，满身都是一点点的红疹，那是一种弄不好就要死人的病。"

　　"发热？是不是猩红热？那是小孩子容易得的传染病啊！扁桃腺发言，全身红疹，舌头象草莓似的，甚至会并发肾炎和关节炎呢。"

　　"你真了解，好像医生似的。"

　　"如果持续高烧的确会有生命危险。"

　　"所幸的是挺过去了，你知道生病的时候我想妈妈，如果经不住捱不过去，那时也就去了她的世界了。也见不到你了。"

　　"小雪……"

　　雨差不多是在康治迈进家门后，就倾泻而下的。织香早已经入睡。当冲淋的热水将他整个笼罩的时候，他感到前所未有的轻松和舒畅。终于将这个多年来难解的心结一吐为快，而且面对的是这样一个倾诉对象，太不可思议了。尽管雪儿的反应远远超出他的想像，但重要的在于过程，他向她交出了心里的一颗石头。

　　但是今晚他不能陪她，尽管将她拥在怀中相伴到天明是他无比向往的美梦。在短短的时间内，经历了这许多起伏，雪儿需要一个人独自安静思考，他不愿意打扰她。而他自己更需要一个适应的过程，他要同时反省着战争，又感慨着命运的巧妙安排，不可思议的缘分啊。渗透着家族的历史和罪恶，纠葛着民族矛盾和分歧，两个人的生命在广博的宇宙之中不期而遇。然而爱的蜜网织得再牢，也难抵御世事的风吹浪打。这是他必须面对的事实。

　　窗外雨声唰唰，夜色沉浸在雨幕中，分外落寞。真希望此时身边有她相依偎，共赏雨景度良宵。唉，他的小雪，此时在香甜的梦中打呼噜了吗？他真喜欢她睡着时的脸庞，憨然的神情。他的想像顺着她的下巴，脖颈慢慢往下爱抚，遮掩在被团底下的小雪的身体啊，那是怎样柔软温润的身体啊！他感觉一阵心颤，热血猛地涌动起来，他发疯似的想要她。他是爱她的，深情地由衷地一如既往地。即使有时候他压抑和控制自己，但事实上谁都不能阻拦他成为一只扑火的飞蝇。不管她是有意还是无心，他确实已跌入陷阱而无法自拔。制造这个陷阱的猎人是无所不能的上帝，所以他们两个都毫无选择地坠落其中。他彻夜未眠，辗转而不能入睡。残酷的硝烟战火和雪儿的娇俏脸庞，在他的眼前交替切换着。他几乎不能承受住这样的折磨，两次从床上坐起来，连连叹气……

　　夏天的早晨，难得有这么清新的空气，啊，地上湿漉漉的，下过雨了？雪儿打开窗户透风，惊异地看着高照的艳阳和地上湿漉的痕迹。昨天晚上回到家倒头就睡了，不知道康治他怎么样？经过这一场波折，雪儿心里对他有了更深的眷恋，但是在话语中却没有半点透露出来。

　　坐在上学去的急行列车中，她仔细斟酌着昨天意外看到的那两封遗信。她激烈的言词，康治都默默承受了，所以现在她站在人道的立场上反思，对于他的正义和容忍产生了钦佩。中午她懒得跑去食堂吃饭，就在文史大楼的一层买了一只面包一罐饮料。坐在空无一人安静的教室里，她笔下的白纸上罗列下一串涂涂改改的日期。整张纸到处是涂改的圈圈和删除，加上大叉符号，疑问号，惊叹号，能够清楚辨别的只有这些字。

　　1936年雪子出生、1937年1月雪子丧母 1岁，南京事件，12月8日受伤，12月13日南京陷落、12月14日留下遗书、12月16日下午死！

　　"死"字周围打满了大大小小的惊叹号。

　　她想起了小时候看过的日本动画片《聪明的一休》。其中有一集：吉梗店老板出题刁难一休，让他在一卷好几十米的长纸上用最快的速度写满天下至理名言。一休两手往嘴里沾了唾沫抹在光脑门上，闭上眼睛打座思考。他喜欢把食、拇指围成一个圈，这时候会有小木鱼"笃笃"的敲击声伴奏。随着清脆的"叮咚"一声，一休调皮地睁开眼睛，办法想出来啦！

　　一休拿来拖把当毛笔，师兄们为他准备了盛满墨汁的木桶，全村的人都跑来看热闹，他们都想见识见识这个聪明的小和尚怎么解决这道难题。只见一休拖着蘸满墨汁的拖把，赤脚在长纸上飞奔起来，墨汁在他的身后留下长长的一条尾巴。跑到白纸的尽头，气喘吁吁的一休左转弯，随之添上一个弯勾。整个字形就象英文字母L，或者中文里的一个笔划——-竖弯勾，在日文里它是"死"的平假名。

　　一休用"死"一个字归纳了世间所有的真理和名言，再贴切不过了！然而死有多种多样并非一了百了，或重如泰山或轻如鸿毛，更有正义邪恶之分。普通人平安度过一生，如果不是在特定的历史环境下有特殊的行为，那既不会名垂青史，也不至于遗臭万年。所以从民族观念出发，康治的这位长辈无疑要被定为战争的千古罪人。

　　但是，但是，他出征前留给小女儿的遗书，渗透着丝丝父爱亲情，雪儿不由地心里发软。无论如何，雪子是无辜的，战争也给日本的百姓带来罪孽，多少人丧夫，失子，成为孤儿。她的思潮翻滚着，圆珠笔在纸上一笔一画地写着:

　　他是个罪人。他替国从军没有选择。他没有参加大屠杀。他死于中国军队的枪弹下。

　　就事论事，用中国的古话来讲，他已经得了因果报应。而且他如果泉下有知的话，应该庆幸他的后代已经为他做了偿还和赎罪。

　　 康治再次打电话约她，是在一个星期之后。

　　"真的吗？带我去看火花？太好了？什么时候？去哪里？"雪儿几乎要握着电话跳起来，她正走在青草葱茏的校园里。

　　"怎么这么兴奋？难道，去年夏天没有去看吗？东京很多地方都有火花大会的，你，不会不知道吧？"

　　"什么呀，火花大会我当然知道啦。可是什么时间在哪里举行，门票什么的，可没有人告诉过我。我一个外来小妞不就和乡下人一样，读书已经很辛苦，哪里还能想着法子玩呀，享受生活呀？怎么说，我们都是……"她故意停顿了一下，"二等公民呀，你不明白吗？"

　　"哦，你说的是这个意思。有什么不好过的事情，记得要告诉我。"康治的心沉了一下，不知道怎么安慰她。

　　"没什么的，早就习惯了。好吧，不说这个了，你把时间定下来以后，尽快告诉我吧，因为要提前向店里请假的。马上就要开始上课了。"她已经来到教学楼的门口，山风吹起她的长发，她舒服地在风中扬了扬脑袋。

　　经历过那场心灵和精神上的涤荡之后，虽然只短短的一星期，她觉得无比漫长。能在一个美丽的夏夜和他一同去看焰火大会，她的心突然怦怦地跳了起来，童年时的记忆涌到了眼前。

　　那是1984年的国庆节，也就是康治结束中国之行回到东京后不久，雪儿9岁，康治是20。

　　1984年是建国35周年的庆典，市政府特意在人民广场安排了大型礼花燃放活动。雪儿的记忆里，那时外滩是怎样的场景已经非常模糊，只记得父亲常带她去人民公园和广场玩。她喜欢蹲在草坪上看小蚂蚁搬家，夏天在荷花池塘边看水里的鱼儿嬉戏。缺少母爱的孩子多半有些孤僻和内向，她却意外地乐天和爽朗，这令父亲的心中不知添了多少安慰。

　　老房子拆迁之前，父女两人挤在一座三层的旧式里弄房子里。

　　朝南的一间八平方米的亭子间，虽然破落拥挤了些，也算是属于雪儿的小天地。进门左手是一个水泥砖头砌成的水斗，因了这个破水斗，房间里总是显得有些阴湿。夏夜里开着门窗睡觉，时常会有蟑螂肆无忌惮地在那里爬来爬去。父亲上中班或者晚班不在家时，她因为害怕常常会关紧门窗，然后使劲地用杀虫水喷洒。她甚至很怀念当时的那种针筒式喷雾的器具，那样子非常有意思，可惜现在早已经绝迹了。不过杀虫水的味道实在不好闻，她只能皱着眉头默默地忍受。

　　窗前有一张老式的写字桌，她小的时候觉得那张桌子很大，父亲没有上过学，家里没有什么像样的书。窗是木头的窗，因为常年没有得到修理，铰链总是嘎吱嘎吱作响。有时候忘记钩上搭扣，风一吹来就会砰地一声把窗推上。窗外，两米之内就是对面人家的晒台。印象里对面那个勤快的媳妇，终日不停地在家洗衣服，那是一个大户人家，老老小小的衣服似乎都归她洗。

　　从亭子间出门，走上三级楼梯，就是面北的后三层阁，面积更小，杂物更多。一大堆总也收拾不清的杂物中，父亲的单人小床紧靠着墙壁，好像个凌乱的仓库似的。没有多余的空间支一张吃饭的桌子，父亲拼拼凑凑地做了一个粗糙的可折叠台板，吃饭时才打开来用。

　　尽管如此，雪儿还是很喜欢这个家，如今也只能从照片上回忆童年时的点点滴滴了。后三层阁门口有一个必须猫着腰走的梯子，通向亭子间顶上的晒台，那是她玩耍嬉戏的地方。虽然晒台属于公用部分，但是她家住得近，于是近水楼台先得月。楼下的其他三户人家只是在周末上来晾晒衣物。夏夜里，她喜欢搬张小躺椅，望着天空数星星，脑子里钻出许许多多奇形怪状的念头来。

　　从半人高的墙沿上可以爬到三层阁的屋顶上。父亲在老虎天窗的边上安了一个鸽棚，养着20多只信鸽。每次他给鸽子喂食，或是在傍晚放飞它们时，雪儿也喜欢跟屁虫似的爬上去蹲在一旁看。她有好奇和热爱小动物的天性，也许就是从那时渐渐培养起来的吧。经常是雪儿放学回家后，做完功课就跑到晒台上看自己家的鸽子在头顶的蓝天上回旋飞转。她小小的勃颈也跟着一起转动，看小鸽子们的翅膀上披着夕阳的余辉，听忽近忽远的翅膀振动空气的那种特有的嗡嗡声，她觉得一切都太美了。

　　那一年国庆的晚上，楼下的所有人家几乎都涌到晒台上来看礼花燃放了。父亲值晚班，留下雪儿一个人在家。她早早地吃完饭，就拿了一张小凳子爬到屋顶上去，抢占有利地形。大家都只能在晒台上站着，而无从享受她那样的特权，因为那是她们家的屋顶呀。

　　那时还没有如今这样林立的高楼大厦，老上海电视塔几乎是一枝独秀，从城市的任何一个角落都能望见它的风采。天色渐渐暗淡下来，可以看见发射塔顶端的红灯一闪一亮。雪儿坐在屋顶上，两肘支在膝盖上，双手托着脸，眼巴巴地等着焰火升起来。鸽棚里的小伙伴们，似乎也按耐不住喜悦和激动的心情，咕噜噜咕噜噜地闹腾着。

　　哧——-喳——-嘭————终于，第一枚礼花整点在夜空中升腾起来，几乎可以感受到所有期待已久的大人小孩们同时发出的欢呼声。改革开放后焕发出蓬勃朝气和活力的上海城， 这个彩妆的夜色美人，顷刻为欢乐的海洋所包围。在雪儿所接受的教导里，国庆节是个隆重而庄严的节日。整个燃放的过程中，她都坐得端端正正，两只小手放在膝盖上，脖子跟着抬起，低下，再抬起，再低下。小嘴不时地张得老大，情不自禁地咿呀咿呀地喝着彩。美不胜收的焰火，看花了她的一双眼睛，当最后一支礼花在空中飞散开来后，一瞬间四周归于平静。在等待了片刻之后仍无动静，人们才意识到这已是结束，于是再次喧腾起来。起哄的起哄，回家的回家，人们带着意犹未尽的兴奋渐渐疏散了。

　　晒台上的人群也散了，但是雪儿仍然静静地坐在那里。太美了，太美了！但是美好的东西，是否都不能长久？礼花只是璀璨在绽放的那一瞬间，照亮整片夜空。也就是在它向全世界展示自我的那个短暂片刻后，化作碎屑纷纷散落。雪儿忽然模模糊糊地领略了一个道理，那些极致的美好的事物，往往都是稍纵即逝的……她第一次对美产生了困惑。

　　满天的星星依然在，仿佛它们遥远得根本不知道这里刚发生了什么。相对于渺小的人类来讲，星星的光芒也许是恒久的吧。

　　这是雪儿记忆中难忘的一次观赏焰火的经历，在十多年后重新想起，仍然历历在目。那一年她才读小学二年级。康治一再向她重复的在南京路巧遇她的那个故事。她开始越来越愿意相信他们曾经有过那次偶然，虽然在她的记忆中好像没有这清晰的一页。

　　雪儿没有穿丝袜，露在外面的十只脚趾头，闪着粉灰色的荧光。他在心里赞叹她修长的双腿，尽管没有丝袜的包裹和点缀，尽管这样毫不修饰，却别有性感的味道。在公司里早已习惯于穿戴严整的女同事，习惯她们格式化的点头，微笑和神态。断然找不到一双未经丝袜包装的秀腿，永远也不会知道她们裹在黑色皮鞋里的十个脚拇指上，是否也会闪耀五彩光泽。于是康治就特别喜爱享受弥漫在雪儿身体四周的清新氛围，此刻他才是轻松惬意的。她从来不刻意化妆打扮，穿衣随和自然，从头到脚没有一件叮咚作响或者闪耀光彩的饰物。

　　"不是的，我当然喜欢，我太喜欢了，我看到这个的时候都幸福得快要晕过去了。你听我解释好不好？"她用从未有的最快的速度说着这番话。

　　"已经离婚了。"她用最简单的语言在短短的瞬间将他抛向空中，又扔回地面，他的思绪在此刻忽然凝结住，似乎不能转动了。
