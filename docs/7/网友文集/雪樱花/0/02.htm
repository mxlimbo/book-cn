小说简介

　　上海女孩雪儿留在樱花国度的一段恋情。

　　康治的学生时代就曾一心向往着上海，热爱中国，却时时有家族的阴影徘徊在心底深处。爱上雪儿时，他是个已婚的中年男子，一家小公司的普通职员。

　　在日语中，"御缘"和"5元"同样发音。以前有一种习俗就是在送人礼物的时候，放入一枚5元硬币，取其谐音，示意送礼双方之间的一种缘分。

　　雪儿决意离开康治回国时，他送了一套和服作为礼物，其间夹了一枚"5元"硬币，可是雪儿最终是否能握它在手中……
