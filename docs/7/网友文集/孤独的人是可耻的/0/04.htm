刻在掌心里的冬天 （一）

　　群走过去紧紧地抱住宝宝，而后抬起头来看着天空，于是绽放在脸上的笑容如满天星光般灿烂。而后往事如潮水般倒灌回来。

　　那么是缘分吧？

　　清晨，宝宝找到光问："群去哪里了？"

　　"我也不清楚，他一大早就出去了。噢，对了，小妹我们一起去逛街吧！"光说道。

　　"不！我要等群回来。"

　　"走吧！群的衣服有些单薄，正好去帮他买件毛衣啊，也许在街上可以遇到他也不一定哟！"光的脸上带着神秘的笑容。

　　"那--好吧！"宝宝应道。

　　买完毛衣后宝宝和光正在街上走着，忽然光停了下来，用手指着前方对宝宝说道："那不是群吗？咦？他身边的那个女孩子是谁啊？"

　　宝宝抬眼望去，忽地全身一震：一个长发飘飘的漂亮女孩子轻轻的挽着群的臂弯，两人正有说有笑地朝这边走过来。
