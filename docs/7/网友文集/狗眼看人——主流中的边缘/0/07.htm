6、海与孩子

　　小男孩光着脚坐在海滩上，手里扶着一根细长的竹竿。竹竿顶上拴了一把破扇子，在随海风晃来晃去，一顶大草帽戴在他的小脑袋上，远远望去他就像长在海滩上的一个桔黄色的蘑菇。远处碧海连天，白浪穿空。时时有一两只海鸟飞到海边，用又细又长的翅膀击打着海滩上的空气。男孩能听到头顶传来的鸟翅击穿空气的声音，也能听懂鸟儿在唱的歌。他知道，远方的海面上白色的浪花们正手挽手排成一队一队，浩浩荡荡地玩着登陆的游戏，其中调皮一点的会使劲去撞半浸在海里的岩石，然后发出一声快活的欢呼：哗！

94年6月9日晨
