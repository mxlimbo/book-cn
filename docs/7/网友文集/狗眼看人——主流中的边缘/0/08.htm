7、糖水

　　二年级的小米抱着盛了水的酒瓶子走到教室门口。现在正是夏天，天很热，常常有男孩子跑渴了找小女孩们要水。每次小米都是眼睁睁看着满头大汗的小男孩在自己面前咕咕咚咚把所有的水都喝光。于是小米从前天开始在瓶子里放了一小块水果糖。当男孩冲过来时，小米就赶忙举起瓶子，用手指着瓶底水里的糖块说："俺这是糖水。"每次小男孩都犹犹豫豫地走开了。不好意思再问小米要水喝。

　　这天，小文满头大汗跑过来喊"水"，小米赶忙一举瓶子认真声明："俺这是糖水。"

　　小文汗水淋漓地歪了一下脑袋，害羞似地说：

　　"俺，就爱喝......糖水。"

94/11/13中午
