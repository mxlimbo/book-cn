5、我在黑暗中

　　我在黑暗中，望着尘世明亮的剪影。这些剪影都被一些燃烧的火鸟，悠闲地衔来衔去。

　　我在黑暗中，行走，许多黑色的带着银色的剪影，固定成一种姿势，发出一种单调的声音，被风霜雨雪冲刷着。

　　我在黑暗中做梦，重新组合自己的人生......破碎的片段。然后我在里面做一个英雄或一个狗熊，或者一片点缀的风景。

　　我在黑暗中叫着别人的名字，然后侧耳倾听这呼喊的一点点回声。在许多时候，我只听到我远去的呼喊在空空的山谷中虚无地游荡着。这时我梦中有了泪水。

　　我在黑暗中开始自己的历史，又在黑暗里结束它。

　　在有生的日子里，我走在无边无际的黑暗中，耳边听着生者的哭声和死者的呻吟。

　　我在黑暗中，学会了唱歌。
