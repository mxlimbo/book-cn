12、白石的虾

　　大师修成正果，笔下墨虾灵气十足。

　　这是一个人苦修近百年方见真精神的一例。

　　白石的虾，落于空纸之上，空纸立如一潭清水，春意盎然；那虾也已非虾，而是游动的生灵。

　　纸和墨在大师笔下皆超越自己物质的外表而上升为精神------此或为艺术之真髓。

　　大师挂角读书，乃为一懵懂书童，不为人知。晚境中显其超拔的心性却不幸故去。临死仍言，我奋斗一生，须发皆白时方才入得真境，未及创出真正自己的作品便要离世了。

　　人生百年，学艺一件，若入真境，便为人仙。

　　大师的境界不死。

94/8/5上午
