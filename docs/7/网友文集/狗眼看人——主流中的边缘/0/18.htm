第二部 灵 魂

1、 灵魂对我说

　　灵魂对我说："你见到我的机会很多，但时间很短，所以你是幸福的。"

　　灵魂喜欢平静的心境。所以在平静的心境中，你才能看到自己灵魂的影子。

　　也许你永远无法看到自己的灵魂。但它是存在的。当你问自己："为什么我就看不到灵魂呢"时，你也许不知道--是灵魂在问你这个问题。

　　每个人的灵魂都写在脸上。只不过用的密码不同。有的人灵魂直接写在脸上，而对另一些脸，你应当用另外的方式去考虑。

　　我们肉体的行为本身无高尚卑劣之分，一切取决于你的灵魂。

　　所以对同一种行为，我们有时称之为爱，有时称为兽行。
