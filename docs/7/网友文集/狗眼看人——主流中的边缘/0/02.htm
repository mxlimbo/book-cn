第一部 尘 世

1、尘世

　　太阳又从东方升了起来。阳光铺满了这个世界。早出的飞鸟相约在阳光中扇动透明的翅膀，鸣叫着上下翻飞，掠过秋天的田野，沿着村子里树与树的空间，向着自己心目中温暖的天堂飞去。

　　阳光升到了最高的地方。世界浸在一片白色刺目的反光中。许多灰尘在路上，罩着穿行于其间的行人和匆匆赶路的车。街上的钟显示着时间，显示着在每天这一时间，有多少人和多少车在由目的地出发后，走在寻找归宿的路上。

　　阳光开始下滑了，许多的光散射在这片空间中，折射着一种火的颜色。许多云聚向阳光落下的地方，像一群群匆忙归林的鸟归向每一天都要去的家。天空中响着他们火红羽翼的扇动声。

　　夜幕降临了，许多信会悄然飞上夜空，衔住断线风筝们飘扬的线，使游弋翻落的风筝有了一种与土地相连的感觉。而这一天，我却在夜空的飞动翻转之中，望着身上那段飘浮的断线。
