6、絮语

　　A

　　有一只大海鸟，扶摇而上，掠过茫茫人海飞向黑暗的远方。在阳光下喧嚣的人潮中，留下一点天堂的气息

　　B

　　太阳光照射着这片土地，几千年始终如一。

　　他在我们先人的眼中，在我们的眼中，在我们后辈的眼中，曾经，正在或者即将飞过。

　　他是一只火鸟，衔着每天的种子，把生命撒播在这片灼热的土地上。

　　C

　　大漠的风吹进了城市，飞旋于高楼之间，攀附于高塔之上，呼啸于巷陌之间。

　　大漠的风改名叫城市的风。

　　D

　　天边是太阳升起和落下的地方。也是许多新生的手们升起和落下的地方。

　　那些手象是树，树林间有许多鸟

　　衔着种子

　　和枯黄的叶子

　　E

　　有一束阳光照在一片巴掌大的地面上。

　　这束阳光中有时有麻雀飞落，有时有昆虫飞舞，有时有狗和猫小心翼翼地爬过，有时有人的脚走过，大人的，孩子的，男人的，女人的。有时有尘土飞起落下，有时有风刮着雨雪霜和落叶。

　　更多的时候，这束阳光孤单地待在那儿，看周围的一切都在变化着。
