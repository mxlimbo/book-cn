11、心如镜

　　当我们是孩子时，我们的眼睛是清澈见底的。透过它我们可以洞悉我们清纯的内心。那时我们的心是一面纤尘未染的镜子。

　　当我们是大人的时候，我们从孩子的目光中看到了天使才有的神情。看到了那曾如天使般的我们的童年。

　　而今我们的心蒙尘了。知识使我们的心蒙满了尘土。本真的我已在上面显不出影了。我们的语言与我们的心隔得那么远。

　　童年的目光已经不可得了，因为上面已蒙了几十年的风尘。童年的心我们还可以找回。

　　只要我们能找到那面镜子，并一天一天拭掉上面的灰尘。
