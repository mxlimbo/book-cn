9、两个世界

　　这个世界阳光灿烂，那个世界一定暗淡无光。

　　这个世界人口密集，那个世界一定空空荡荡。

　　这个世界充满噪声，那个世界也许凄凄凉凉。

　　这个世界的人脸上充满了焦虑，那个世界所有的人都神态安详。

　　这个世界的人都为钱而发愁，那个世界的人会不会不知钱是什么模样。

　　这个世界有那么多门也有那么多墙，那个世界一定一片汪洋。

　　生活在这个世界的人每天都想着地狱天堂的事情，生活在那个世界的人一定大不一样。

　　这个世界也许与那个世界间没有阻隔，阻隔和沟通两个世界的都是.......

　　我们的目光。
