突击车子化

一九五八年十二月二十日

劳作声，冲破了深夜的宁静。
汽灯，照彻了夜空。
月亮为之失色，星星惊疑不定。
灯光拨弄着人影,人们在紧张地劳动。
锯子、斧头发出嘶嘶吭吭的响声。
砂轮的欢唱伴着人们的笑声。
一边是安装车架，一边在制作轴承。
他们三昼夜不曾合眼，
然而，手一刻也未停。
歌声驱散了劳累和疲倦，
热情熔暖了冬夜的寒冷。

几十个人怀着一个信念：
争取车子化提前完成。
东方微白，天色朦朦，
几十双充血的眼睛，
几十张幸福的笑容，
欣赏着：
手推车的队伍在路上奔腾。

注：这年冬天，不知吹来一阵什么风，
要求消灭扁担、爬犁，实行手推车化。
八一农场机电试验站是当然的试点，
于是，出现了上述劳动场面。
