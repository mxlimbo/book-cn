七言古 ・ 辍 学

一九五一年六月二日

朝复暮兮暮又朝，挥锄低首望禾苗。
寂然远离邻家子，泪眼怅怅泣辍学。

　　注：余四六年秋入学，不久，为支援解放军南下，我地进"大拥参" 运动。我村十六岁以上四十五岁以下男子全部入伍，父亲也参军远去，各家的农事，全落在老弱妇孺身上，解放之初学生站岗放哨，开会游行，学习时间本来就少，而我身虽在校，但经常被迫请假下地干活。因而学习时间就更少。一九五一年秋，哥哥携四弟外出耍杂技，我只好辍学务农。时见同龄人，或升学，或学艺，纷纷外出，甚为伤感，作此诗以纪之。
