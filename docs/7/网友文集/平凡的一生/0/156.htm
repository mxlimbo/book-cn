七绝·壮别沧州


一九六八年十二月廿五日

山川呼啸风雪骤，汽笛悲壮别沧州。
几转危难锤丹心，一挥辛酸意更遒！

注：吴桥群专函调，一０二团回函曰：于刘是"五·一六"分子，请当地政府严加惩处！对"人命案"事只字不提，群专情知有诈，便找我谈话，要我自己买票回疆。我予以严词拒绝。群专无奈，便把我们迁送至沧州专区收容院。这里的工作人员不错，尚能以礼相待，最后，破例买了两张直达乌市的车票，上车后，又把一０二团的来函一一叫我过目，并勉励说："回去告他们"！这是一个风雪交加的日子，火车一声长鸣，我们挥手告别了沧州。
