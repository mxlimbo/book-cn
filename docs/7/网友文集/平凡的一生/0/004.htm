再见了，故乡

一九五四年五月十七日

桃李斗艳，百鸟争鸣，
春虫唧唧喧闹，
柳丝漫拂着春风。
小伙子荷锄招手，
老人们微笑点头，
和风传播着姑娘的歌声。
美丽的农村，家乡的风情，
已使我忧心忡忡；
是苦闷是高兴，无法分清，
找不到恰当的语言，
表达这别离前的心情。
明天呀，一夜之隔，万里之长，

我将离开您的怀抱，
为理想而奔忙。
我感谢您──家乡，
感谢您对我的哺养。
今后，今后又是遥远的迷茫。
啊，再见了，美丽的家乡，
我不能因您美丽而自缚理想的翅膀；
啊，再见了，姑娘。请莫惆怅，
我也不能因你惆怅而留在你的身旁。
你正当青春年少，来日方长。
故乡，明天就要分别了──再见。

　　注：一九五四年初，母亲想阻我进疆，
托人给我介绍对象。"姑娘"即指此。
