七律·"五·一三"事件

一九六八年五月廿九日

满腔怒火满腔仇，革命岂能怕断头。
挺身枪林卫真理，跃向刀丛夺自由。
天山低首悲志士，溪水哽咽泣荒丘！
侥幸留得一息在，此恨不雪实难休。

注：一九六八年五月十一日，贾樊再次支持一派群众组织，打开值班连武器库，"抢取"了全部枪支弹药，并于十三日在全团发动了武装镇压群众流血事件。我同修配厂、试验场的工人和团子校学生在逃难途中，又遭天山化工厂值班连武装袭击，当场击毙我同行者三人，活活打死一人，并继续向已逃散的群众开枪追杀。我原躲在一堆水泥管后面避弹，见此情景，气愤难抑，一跃而出，拦在一群值班战士前面，拨开冒烟的枪口，大声喝斥：不准开枪！由此而身陷囹圄，遭到残酷摧残！
