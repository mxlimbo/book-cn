歌词·军垦战士怀念红太阳


一九六八年四月廿八日

老龙河水弯又长，风吹稻麦两岸香。
天山顶上红霞飞，军垦战士怀念红太阳。
老龙河水奔流忙，阶级斗争永不忘。
革命造反擎红旗, 军垦战士怀念红太阳。
七月河水翻血浪，资反路线如虎狼，
皮鞭下面啸壮志, 军垦战士怀念红太阳。
迎腥风、战恶浪，万丈怒火燃胸膛，
同仇敌忾讨贾樊, 军垦战士怀念红太阳。

注：文革中，八一农场改为一０二团。"贾樊"即贾宝善、樊甫。是当时的团长、政委。去年七月十八日，他们公然调动值班连队，支持一派群众组织，对不同观点的干部群众大打出手。制造了闻名全疆的"七一八"惨案。
