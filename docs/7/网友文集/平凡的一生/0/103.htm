七言古·不老松

一九六二年四月十四日

我爱西邻老太太，十年不见鬓发白。
音容笑貌还如昨，老态龙重心未衰。
半世尝尽阶级苦，感恩念念不忘怀。
全心全意为集体，品高质朴如松柏。
丈夫贪心谋小利，盗窃公物饱私怀。
儿女规劝难入耳，母女联合怒相责。
夫不知悔反成恨，驱妻弃女将房卖。
饥困难挫老妪志，荷锄带女战荒灾。
齐努力，战荒灾，佳话远传十里外。
我爱西邻老太太，犹如高山一松柏。
风摧雪压质更坚，霜袭冰封永不败。
