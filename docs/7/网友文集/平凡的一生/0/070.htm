春 歌

一九五九年四月一日

春日映着红旗，红旗招展飘飞。
枝头小鸟啼不息，春耕已开始。
田野里，人声鼎沸，
公路上，车马如飞。
笑声千里悦人心，一片景色醉人！
东风频传春意，春光明媚如许，
晴空万里无云迹，大雁又北归。
拖拉机，田野奔驰，
隆隆隆，声震广宇，
昼夜耕耘不停息，干劲谁可比！
东风吹醒大地，万顷良田尽碧，
高产禾苗已出齐，望之快人心。

搞丰产，信念不移，
跃进年，大创奇迹，
待到秋登粮山顶，敢笑天山低!
