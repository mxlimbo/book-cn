别

一九五八年四月五日

同志，分别了，这并不突然，
离别的阴云早已布满高天。
莫怪我默默无语，对人冷淡，
对这一切，我只能装作漠然，

然而，心中又何尝不是熊熊烈焰！
同志，分别了，我珍惜你的纪念，
并以同物奉还；共载着今天，
或许可慰百年。
写下这寥寥数语，
我已力尽平生，挥笔如山！
同志，分别了。
在人生的道路上，
免不了遇上峻岭高山，
祝你勇敢直前！
在生活中把自己锤炼。
同志，分别了。
分别使我如释重担，
却也饱尝了别离的辛酸！
