题 一个人的照相


　　嘿，站好，给你照张像

　　对了，笑一笑。笑一笑才可爱嘛。你平常不是最爱笑吗？都说A血型的人在笑的时候是最有魅力的。看，你调皮的笑、羞涩的笑、得意的笑、痛快的笑、友善的笑、自信的笑。生命本就短暂，何必自己给自己栽培苦涩。太阳每天都是新的，我们有什么理由不珍惜呢？善待自己，让自己过得快乐些。你一向是那么乐观。

　　也许是上天的宠爱，让你事事如意，没有经历什么坎坷曲折，在洒满鲜花的大道上，你阔步走着。你觉得世间是美好的。但你也深知坎坷曲折更能磨练、造就一个人。所以你做好一切

　　准备去迎接未来的挑战。

　　也许是太年轻吧，"少年不识愁滋味"。慢慢地慢慢地你长大了。有了忧郁的时候。那时的你是那样的静，静得出奇。你没有了表情，没有了话语。你不敢抬头正视每个人的眼光，你只匆匆低下头，看着自己的脚。你不发火、不吵闹、不要求什么。不了解的人以为你和平常一样，不会在意、关心。可谁知————此时你心中正淌着泪，一滴一滴打在那流血的伤口上！你讨厌人世间的世俗、虚伪，讨厌见到忘意、曲颜附势的小人。

　　为这，你恨过，深深的。

　　好了，再来个温柔的。善意的面容、甜美的嗓音，本就是一个温柔样。和亲近的人在一起你显得那样的快活、放松。象个永远长不大的小孩子。但不要以为她很任性、撒娇，其实你是个善解人意、懂事理的女孩。在外人面前，你会很大方、得体、说话井井有条，但不要以为她很能说，其实越是对自己最爱的人，你越不会表达。因为你无法用修饰的词语来表达自己真实的感受，那样出口的东西是假的。你唯一能做的是用心去爱她。

　　你很喜欢小孩子、小动物。仿佛那是种最原始、发自内心的关爱。看着那幼稚的小生命，你仿佛感觉到一种生命的延续，一种人性的回归。多么可爱的小生命啊！可是不知怎的你又害怕它们。怕自己不小心伤害了它们，它们是那样脆弱。可对它们你并不了解，你束手无策。所以你又头疼它们。

　　你是个很丰富的女孩。天生有种对文学的偏爱，幻想那甜蜜、浪漫的故事，诗情画意的情调。还有一种音乐的敏感，那美好的音乐一从心间滑过，便荡漾起一串心弦之曲，让我终生不忘。她适合唱孟庭伟那样的情歌，但她更喜欢王菲那不落俗套的风格及天籁空灵般的嗓音，更向往张惠妹那无拘无束、洒脱自如的痛快与放松。

　　你喜欢热闹，但也不厌孤独。一个人的时候似乎很写意、一盏孤灯、一杯清茶或者牛奶、一段美妙的旋律、一本心爱的书。多美的意境。你喜欢这样的孤独。你可以静下心来，整理一下思绪，回忆一下往事，任时光慢慢流逝。

　　你很喜欢四处走走。看看周围的景色。你喜欢桂林的秀意、云南的风俗、异国的情调。

　　你还喜欢收集。从一张糖纸到一枚邮票，从昔日同学的一张纸条到友人的一只纸鹤，从恋人的一封情书到任何一件小东西。你都要保存起来。那只字片语、点点滴滴仿佛都记录着一个不老的故事，记录着我成长的足迹。

　　你喜欢蓝色，你将永远喜欢蓝色。那蓝色的平静、恬人、深远。那是天的颜色、海的颜色、梦的颜色、你的颜色。蓝色————永恒。
