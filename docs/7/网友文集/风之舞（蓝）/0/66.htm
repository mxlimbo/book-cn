愁，骐


看我的额头
我的眼睛
是否还是那样忧愁
那么思虑
你总喜欢我的熟睡
轻柔紧锁的眉头
让我知道
关怀就在身旁
无法忘记
轻柔温欣的瞬间
只是浪漫它会沉淀
会因风大而吹散
永久记忆
会慢慢的飘

看你的长发
你的眼睛
是否还是那样单纯
那么幼稚
我总喜欢你的微笑
那是世上最美的画面
让你懂得
路上有那艰难
也有解脱
唱一首属于我俩的歌
只是为了让浪漫延续
会因为距离而搁浅
忘了记忆的我
你会是我所有的未来

2000.05.28
