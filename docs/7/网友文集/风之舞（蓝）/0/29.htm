一想到她爱你

狂的风让人睁不开眼睛
灰的天眼看要下雨
我闭上眼睛裹紧被子
冰冷的心中有一口井
痛苦无奈是唯一回音
一想到她爱你
爱得很痴迷
我默默收起你的信
将他压在柜底
一想到她爱你
绝不愿分离
而我的距离
只能让你与背叛靠近
我一直不适合争夺什么
可也不愿轻易失去
熄了灯做没有你的梦
试着习惯这陌生
一想到她爱你
而你坚定的心
又给了勇气
使我的选择有了意义
也给了我忧郁
我决不要同情施舍的情意
也不要你的花言巧语
红的眼，冰的心
不要你来怜惜
只要这一切
使你真的愿意
