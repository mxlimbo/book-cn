氧气

五天的假期如五年的监狱
是你让他阴暗度日困难呼吸
想你会来探狱
无声叹息……
出狱的他如一条小鱼
四处找寻你的踪迹
游过池塘跨过海洋
一天又一天一遍又一遍
再大的力量也抵不过层层波浪
再大的希望也抵不过次次失望
历尽艰难他遍体鳞伤
无奈停下来好好想一想
为何你隐身顿去
不顾他的怜惜
为何你又对他呵护备至
默默地关心
为何你要如此
折磨他的心绪
莫非是你虚情假意
莫非是你内心空虚
还莫非是你没有勇气
可怜的鱼儿没有了氧气
渐渐地沉入海底
此时一双大手
突然将他捞起
啊？！
那竟是你
在彼此心中安家
