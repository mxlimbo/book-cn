晚秋

　　当最后一缕夕阳落下时，彩霞布满了整个天空，那彩霞是属于黄昏的，夜幕慢慢的拉下，月亮泛出鱼肚白，也不知过了多久，孤独的黑夜就只有那暗淡的月亮。

　　她是那样消瘦，孤零零的挂在漆黑的夜空中。星星从她身边擦过，编织出天河、北斗......陪伴月亮捉迷藏。月亮非常高兴、快乐一天天胖了起来圆圆的可爱之极，泛出了皎洁的目光，给人间万物带来生机，给黑夜孤独的人照亮了归路。

　　突然暴风骤雨遮挡了那目光，天河倾泻给人间带来了一种无法弥补的灾难，以至于民不聊生，月亮看在眼里记在心中。她，失去了欢乐一天天消瘦下去，又失去了那美丽的目光、可爱的笑容......

　　注：此作原作于1998年10月31日一个归途的夜里，后修订于98.11.7日凌晨4点。此作初作时心事重重，后修订时心胸开旷，所以似无伤感。希本作能有一个更完美的结尾。

逍狼于10.7日亲笔
