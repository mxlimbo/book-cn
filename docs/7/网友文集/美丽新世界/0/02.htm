二

　　秦甡在饭桌前端着晚报，看到诗人沙龙关张的一百来字的小短讯，说了声："还是关了。小漪，你昨晚不是在那儿吗？"

　　 "嗯。" 连漪边吃边面无表情地应着。

　　"怎么样？没什么感想？"

　　"没什么。关就关了呗，顺应社会发展潮流，谁有时间为它感想。"一觉醒来，连漪对此兴趣全无，当然这话还有一层她要强烈申明的意思：谁管诗歌这档子破事。

　　"王东是个不错的人。"秦甡说着，摘下眼镜，收起报纸吃起了饭。

　　又是一顿沉闷的早餐。

　　"妈妈上班去了？"

　　"嗯。"

　　"姐姐上班去了？"

　　"嗯。"

　　"哥哥也上班去了？"秦甡这样问是为了活跃一下气氛，也顺便激连漪一下。谁知连漪还是面无表情地应了一声：

　　"嗯。"

　　早餐是馒头稀饭拌凉菜。前面那三位吃的不是点心就是泡麦片，每天早晨只有秦甡和连漪能这样从容地就餐。

　　"没打算干什么，今天？"

　　"没。"连漪吃完了，把自己的碗筷拿到厨房，用水洗了洗，放进碗橱。

　　秦甡听着厨房里乒乒乓乓的声音觉得对这个女儿真是无能为力了。

　　把自己的碗筷收拾好后，连漪跑进自己的房间，然后出来，冲他喊："父亲，我出去了。"也不等回应就换上一双球鞋出门了。

　　明亮的阳光从她房间射进来，投在厅里的落地钟上，秦甡盯着看了一会儿，忽然想：我也好长时间没出去了。

　　连漪和卞铭菲在中意大厦顶层见。卞铭菲九点钟要去应聘，打扮得果然很精神，紫色纱质职业套裙，束了个马尾，看得出打了不少mousse，显得鲜亮动人。很美。连漪不确定自己能否称得上"美"，尽管连涓高傲，目中无人，但她对自己不确定。

　　"精神恍惚，怎么了？"

　　"没什么。质问你一件事。"

　　卞铭菲笑了："什么事？请质问吧！"

　　"你在引诱我哥哥吗？"

　　"你哥哥？开什么玩笑你？"卞铭菲换了副莫名其妙的笑容摸了一下连漪的额头， "我倒是对你父亲更感兴趣。迷人的学者气质，散发着古典主义的浪漫气息，深沉温和的眼神，让我感觉自己委琐卑微。在你未见识真实的事物之前千万别妄下结论。之前我一直以为诗人是个贬义词哎，是你的父亲，大诗人秦甡让我领略了什么是真正的诗人。这几天我一直在修正我的诗人观。"

　　"你在引诱我哥哥吗？"连漪并不理会她这一番听上去挺正经的话。连鸣自见了卞铭菲后就有些魂不守舍，有机会就向她旁敲侧击她的消息。之所以会这样，连漪认为完全是卞铭菲的阴谋。她那矜持的、淑女的，又茫然的一笑，她那对连鸣不理不睬的态度，都是有目的，经过自我训练，杀伤力很强的。望着连鸣那时而尴尬时而失落的表情连漪就知道他肯定会中铭菲的道。她喜欢男人对她失魂落魄，她说这是她天性的一部分。

　　况且她真的很美。几天来连鸣的态度也让她深信不疑，卞铭菲又用她那套俘获了一个人。这个人可是我的哥哥，是我干净高尚的哥哥。我要制止这件事。卞铭菲是天使也是魔鬼，可在爱情这方面，绝对是个魔鬼。

　　"我是说你在诱惑他吗？"连漪追问。

　　"昨天晚上我很难过，又想到了死亡，想到自杀。"卞铭菲忽然黯淡下去，一副失落的样子看着天。连漪也顺眼看了一下天，吃了一惊：天这样蓝？眼睛仿佛受到了刺激，一阵晕眩，恢复过来后，脑中被她的话占据了——自杀？！

　　这两个字不值得用这样强烈的符号来修饰。谁没想过自杀呢？十四岁时连漪就想了。她因什么事坐在院子里的板凳上伤心地哭泣，抬头抽搐时看到了晃悠悠的晾衣绳，她想把脖子吊上去一了百了。以后遇到的挫折和失落更多，这样的念头也多，但只是想想，是本能的想法吧？她不会自杀。尽管到现在生命还是沉寂无为，但她不会自杀，她确定。如果命运对她不公，那么平淡地完成自己的一生也是一种成功，也是对命运的一种对抗。

　　"我无时无刻不想到死亡。有时我觉得我真的要那样去做了。"卞铭菲经常这样说。

　　"据日本官方统计数字，1997年日本共有24391人自杀身亡，比1996年上升了5.6%。都是人。看到这条消息，我有一种，一种欣慰的感觉。"她凄然一笑。

　　她一定一直在心中重复着这两个数字，这会成为她的后盾或者说是动力或者说是更直接的理由吗？连漪不知该说些什么。

　　"自杀是可耻的。我可不能输给这个世界。"她又一如既往地目光坚定起来。42层的中意大厦下面，人们和车们渺小渺茫又让人有些如戏的感觉。

　　谁都在对生的惶惑中追寻死的概念。连漪常会假设自己在行将灭亡之际的情形，寻找那时的心态，她会感到骇怕、迷惘、凄凉或会觉得无所谓，到那时会露出不屈服的微笑。至于不屈服什么，她也不知道。

　　自杀又是个什么概念呢？

　　在离开这个城市往东一千公里一个挺现代化的镇上，在镇上的一所高中，五年前也是在这样一个晴朗的天气里，高一四个女生集体自杀。她们买了安眠药，买了一瓶酒，然后在宿舍里就着酒吃下了安眠药，然后又用削笔刀割破了手腕上的动脉，血流出来，因流到了别人的被褥上而遭到埋怨。――她们没死成。她们嚎哭着挣扎着不上救护车，拉她们上车的人嘻嘻哈哈地笑着骂着，一出校园闹剧。四个女生归校后把袖子捋得老高，炫耀她们的伤痕。怎能如此轻佻？连漪原先对她们充满好感，以为这是对校方监狱似的管理的反抗，却是情场失意，醋坛子被打翻。有时与她们对面而过，会感到悲哀还有些愤怒。校方对此事沉默良久，然后请了一位赫赫有名的农民企业家来作报告。农民企业家怒气满胸膛，最精彩的一句就是：你活着每一天，就要有每一天的发现，要是没有新发现，你就是行尸走肉，你就是干尸，你就是木乃伊！什么迷惘啊，渺茫啊，茫然啊，情爱啊，我给你加上两个字：该死！你确实该死！全校一片哄笑。

　　"笑什么？"卞铭菲不解地问。

　　"一件精彩的事。"关于农民企业家的报告连漪早已把它作为救拔卞铭菲的资料绘声绘色地讲给她听了。这件事百忆不厌，一忆起就会情不自禁地笑。卞铭菲总是说她精彩的事真多，其实就是那么一两件。

　　"你精彩的事可真多。细想一想我也不少嘛，那些阳光灿烂的日子。噢！阴霾！你何时驻入了我的心空？"她两手抱在胸前，夸张地紧闭起双眼，然后睁开，说："我得走了，宝贝。得抓紧时间，这是个机会啊。祝我成功。"

　　连漪笑了笑，说祝你成功。忽又想起了，叫道："哎！我哥哥――"

　　"别替我自作多情了。我连他的名字都没记住。没事看看8月份的《诗刊》。"紫色坤包在空中划了个狂放的曲线，飘然而下了。
