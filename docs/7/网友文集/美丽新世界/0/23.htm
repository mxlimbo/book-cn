二十三

　　我要跟我的世界说Bye-bye！

　　我猜你活得也不快乐。连漪拖着虚弱的身体走进连涓房里，蜷缩着躺在墙角的短沙发上。

　　连涓确实不快乐。一个极端孤傲的人失去了自我，怎么会快乐？她无语，移开眼前那本描写青楼女子的书，看着连漪。

　　看那样的书。说不上连漪这句话里有没有责怪的意味。你也失去贞洁了。

　　怎么知道？

　　我知道，连漪柔柔地说，眼睛在台灯黄色光晕之外的暗里一闪一闪，很空洞很茫然的一闪一闪。她盯着的是挂在墙上的一个红色坤包，在那里或许能翻出避孕的药物吧。失去了怎么会和没失去一样，增加了怎么会和没增加一样。残缺了的东西给人的感觉不一样，沾染上了别的东西给人的感觉也不一样。

　　真的可以看出来？

　　嗯。

　　你知道那种感觉吗？很肮脏，很可耻，也很快乐。

　　我知道。卞铭菲的呻吟和尖叫在耳边起了。为什么这快乐偏要是肮脏偏要是可耻的呢？

　　连涓不回答，掀开毛毯，让连漪过来。

　　连漪过去，脸伏在连涓肩膀上躺下，连涓把毛毯给她盖上，抱着她说："怎么了？医生说你快好了，不要想太多。"

　　连漪在床上躺了整整三个月，一家人小心翼翼，防范着她的抑郁症。他们不知道在消落困乏的外表下，连漪也在拼命抵抗，同时抵抗卞铭菲发疯似的伤感和性爱时露骨叫喊的侵蚀。

　　明天国庆节了。

　　是啊。怎么了？想出去？

　　是我们的祖国的生日。50周年，很伟大啊，可是太年轻了。连漪的声音更轻了，连涓不明白她要说什么。我们换一换吧，你做连漪，我做连涓。

　　这个决定就这样随口说出，像呼吸一样平淡。连涓一惊。她把她的想法告诉连漪时就知道连漪终会向她提出这样的要求，不过没想到她会选择这个时候。

　　为什么？

　　因为不想死。连漪转过身去，又蜷缩起来。是在自救。我痛恨我的世界，快痛恨死了。可我不想死，我还想知道我的未来什么样子。换一换吧，既然你也不快乐。

　　可是——

　　你不用担心，我不会给你造成什么损失，你也不用担心我会有什么危险。为期一年，同意吧。

　　想多久了？

　　病多久就想了多久。好吧？

　　这样平淡，没一点轰轰烈烈的感觉。

　　就是这样平淡。——像所有的阴谋一样，在开始时都是平淡的。

　　什么时候开始？

　　现在。

　　现在？

　　就现在吧。

　　可是——

　　你同意，我的病马上就好了。

　　可是——

　　同意了？连漪转过来看着连涓。我们互相交接一下。你先说说你。

　　我？说什么？

　　你的生活。你每天要接触的生活。

　　连涓笑了：你认真？

　　嗯！

　　其实我也好想和你换一下，我也早想逃开现在的生活状态。换吧。连涓觉得这话仿佛不是从自己口中说出的。我是失去了贞洁。没有准备，暴风骤雨般的失去了，在怀着对爱情的美好憧憬中失去了……

　　不用解释，我知道结果就行了。把灯关上吧。

　　台灯其实在连漪这边，可她不想动，仿佛动一动，就会打消念头似的。连涓起身隔着她把灯拧死时，连漪闻到她身上散发着的芳香，这芳香让她联想到刚来时闻到的这城市的味道――遥远又温暖。她好奇地看着这个散发着奇怪味道的高大的城市，否定了她前16年里生命的意义及一切的快乐和痛苦。

　　黑暗统治着连涓的情绪和语言：

　　"没有真正的朋友。每天像个影子一样被踩在身前或拖在身后，没有一点用处和意义。外表还是坚不可摧的城池，硬邦邦的皮鞋声在大楼里震荡，把别人的心震虚了，也把自己的心震虚了，他们看我的眼神充满羡慕与敬畏，却不知道那不过是座拼命维持着外表的废墟而已……害怕夜晚。害怕在夜晚中面对自己，害怕夜晚中那个变成野兽的自己，"连涓转过去，，"就只有跳舞，喝酒，喝酒，跳舞，更多的时候做爱，像动物那样做爱，没一点人的尊严和情调……好几次我想把他杀了再自杀算了……这些，这些就够了，知道这些你还想和我换吗？"

　　连漪已有些颤抖了，但她用平静的声调说："这些没有关系。"

　　"你会那样做？"

　　"不会。"

　　"能保证？"

　　"嗯。"

　　"想知道他是谁吗？"

　　"什么他？"

　　"那个男人。"

　　"不，不想知道。"

　　"好吧。"连涓也不确定自己是否能把周光的名字说出口，"说说你吧。"

　　"做我很舒服，只需呆在家里，闷了的时候逛逛街，爬爬山，去影院看电影，去书店看书，去花鸟市场和那些老人交流一下养花的经验教训，再就是去姥姥家看看花们草们。有时周光会约我……"说到这里她明显感到连涓震动了一下，并不理会，接着说，"我们去水手餐厅，心情好的时候吃西餐，心情不好的时候吃中餐，心情不好也不坏的时候吃快餐。我还有个朋友，就是那个常来看我的阿龙，我猜他爱上我了，你得和他谈谈清楚，我们不能这样下去。就是这些。"

　　"就是这些？"

　　"嗯。"

　　"偶尔也去中意大厦，在楼顶看风景。不过不想再去了。"

　　"那个叫卞铭菲的女孩呢？"

　　"她？就让她随我要决裂的世界一起去好了。"

　　"那个别墅里的男孩呢？"

　　郇兵？是啊，还有他。为什么在这三个月里几乎从未想到过他呢？

　　"我们偶尔通电话，他叫郇兵，旬耳郇，你可以与他交往。"

　　"性呢？没有欲求？"

　　连漪也忘了这个。

　　"那是黑暗中的事。只需在人前做我就行了。"

　　"不，我要彻头彻尾的体验你。"

　　"你无法体验。我们的精神层次不同。"

　　"自高自大的家伙。你说说看。"

　　"我不想说！是交换位置，不是身体买卖！"连漪喊了起来。

　　"这算什么？我不是说了我吗？！"连涓也喊起来。

　　"那是你，那就是你，有压抑渴望倾诉所以才会说。我是我，没这方面的困扰，我不说。"

　　"不说算了。"平息下来，连涓又问："没有一点留恋？没有一点顾忌？"

　　"没有。你呢？"

　　"没有，没有吧。还有财产呢？"

　　"你无偿使用我的，我无偿使用你的，身份证、学历证、工作证也一样。"

　　"这样让我觉得你是这个游戏的受害者，对你不公平。"

　　"是说你一直过得不公平？"

　　"表面上不是。可是今天晚上我觉得是了。我想出人头地，结果却落得粉身碎骨，我辛辛苦苦地打拼，所赚的不及你没有付出劳力获得的财产的，有你财产的百分之一吗？"

　　"我不是不劳而获，善良和真诚也价值。"她现在不想争辩，"我不想说这些。我们还需要一份合同，还需要改变一下形象。你的眉毛还能长成原先的形状吗？你的皮肤还能恢复成少女状态吗？你还学得会善良和真诚吗？"
