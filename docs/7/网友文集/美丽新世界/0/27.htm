二十七

　　连漪今天高兴，原因有二：

　　其一，她猜测的事情发生了。周光开始侵犯她。那黑色的皮沙发就是他们――连涓和周光――做爱的地方？下午，有些醉意的周光把她叫进去，没说什么，只是从他的椅子里走出来，绕到她后面，从她后面抱着她，吻她的头发和耳朵。她没有吃惊，亦无反感，一动不动地任由他摆布。她也不记得他们怎样倒在了沙发上，周光压在她的身上，她承受着他汹涌的狂吻，允许他抚摸自己的乳房，甚至允许他脱掉自己的上衣。这些观念里隆重的事，发生时发现也没什么。她站在自己之外看着一切。一个清醒的自己，一个有些邪恶的自己。周光嘴里呼出浓烈的酒气，有一种粗犷的味道，她迷醉。邪恶的自己像妓女一样表现得从容镇定，调皮地躲避，又投入地参与这游戏，与他缠绵。他最近显得可怜。她想给他安慰。她不完全懂得男女之事，她想着应该是有礼的，温柔的，浅尝辄止的，而被周光残忍地挤压，甚至窒息时，并不吃惊。她感到快乐。她想让那个清醒的自己走开，她想，尝试。周光尽管在她身体上肆虐，却不像真想伤害她，只是在试探，他与炽烈的欲火力不从心地对抗，他呻吟着，嘴里吐着爱的字眼，欲火渐渐占了上锋，不管她是谁，不管是不是他不想伤害的，他想要。他把手探进她裙下，用力地抚她，她为什么不反抗？为什么要扭动身体配合他。这个女孩怎么了？他的手探进她的内裤，在边缘轻抚，渐渐粗野了起来，也许是粗糙的手指弄疼了她，也许是那个清醒的自己无法忍受了，她叫了一声，推开他，转向一侧离开沙发。不能看他，她知道他一丝不挂，穿厚丝袜的双腿依然敏感，有坚硬的东西划过它们。她决定不去想那是什么。

　　她高兴，因为她既定了一个事实，还因为她终于有了这样的经历。

　　其二，发现公司里的人际关系其实很简单，同事间很好相处，做人做事很舒服，和"大鸟"完全不同，是国企和私企的差别？临下班 "闪亮一族"主设计关莉莉找她谈了谈新款皮鞋的构思理念，她对周光的审美取向不以为然，认为他是个观念上毫无创造性的时尚跟屁虫，表面上够IN，够前卫，可骨子里根深蒂固的小农意识决定了他不会有什么出息。

　　"他刚来时我激动得两天两夜没睡着，真的，这种激动持续了6个月，现在我做都想他什么时候离开。"

　　关莉莉已过30岁，浑身散发着一种特殊水果香气，细瘦纤巧，衣着讲究，总用七彩的皮筋扎两个又细又长的辫子，像个天真的小女孩，气质却是清高的，做事也泼辣，说话肆无忌惮且声音宏量。连漪第一次见到她就喜欢上了她，但一直找不到机会和她接触，而且她在自信的人面前向来没有自信，她觉得自己是不受自信的人喜欢的那种人。虽不知关莉莉为何主动找她，而且是谈论这个话题，有些不习惯，但是很高兴，起码说明她受她信任。就在那里乖乖地笑着，静悄悄地听着，嗯嗯地附和着，虽有不同意见，比如"小农意识"这个词就让她有些不舒服，但争论有什么用呢？实际行动产生的实际效果才是颠覆成见的最好方式。

　　最近她听到了一些对周光独断专行的做事风格的埋怨，最后都归结为一条：周光出身稼穑。稼穑何辜？连漪对这种成见极为反感。农民有小农意识，市民有小市民情结，不过是各自文化生态的一种反应，都免不了成为缺乏深彻入骨的思索、大刀阔斧的改变的借口。连漪不知道自己出身于何，十六岁之前，在各种表格"家庭成份"一栏里随着同学们开玩笑似地填上：贫下中农或贫农，十六岁之后她开始填"干部"了。有时她也迷惘，也是她痛苦的一个根源，她和连涓的不同是显而易见的，一些东西是烙进血液了。

　　关莉莉很快就把她从这种思绪中拉了出来，她说了句"小连，我很欣赏你"，话题就转了。

　　"美丽的女人有两种，一种丽质天成，美丽的花朵；一种后来修成，馥郁的芬芳。我是后者，你兼而有之。"

　　连漪心里一美："有什么区别吗？"

　　"美丽的花朵让人赏心悦目，初见之下就会赞叹，而芬芳是要体味回味品味的，体味回味品味完了感觉到一种美好的享受才是美的。"她建议连漪不要穿那些尖头高跟太过伶俐的皮鞋，头发别老盘成个高不可及的髻，这些与她1米70的个子太冲突了，暗色系的化妆品也不适合气质浪漫古典却是一张娃娃脸的她，干脆不需要化妆，别把自己的气质淹没在脂粉里。

　　最后说："小连，我看好你，好好干，前途无量。"

　　能被自己欣赏的人如此赞赏岂非最大的美事？一种被关爱的温暖感觉。真好。下班回家，一路欢歌35635，65312。连涓回来后把晚上要吃的米和菜淘了淘，洗了洗，就倚在床上心情烦躁地乱翻着一本朱德镛的漫画等连漪回来，听到外面响起她的歌声，立即起身冲出房门，不等连漪踏进家门就把她拽了出去，没乘电梯，走楼梯，从第十二层一冲而下，直到楼下的小公园才把她放开。毫无防备的连漪不停地大问什么事，俟被放开，意识到什么似的，不吭声了，只盯着气喘吁吁的连涓，一脸无辜状。有很多下班的人们经过身边，他们大都会扭头看一下她们，眼里尽是好奇――毕竟双胞胎很惹眼，而且是这种年龄，这样美丽的双胞胎。连涓的表情从未这样，激动得像个姐姐，她不断朝那些乜眼窥视的人狠狠瞪着眼，最后抓住连漪胳膊，把她拉到附近的废旧建筑工地上，还是不说话。连漪知道是在等她开口呢，就问怎么了，发生什么事了吗？

　　你同性恋？

　　什么？什么同性恋？

　　和那个女孩，卞铭菲，怎么回事？

　　什么怎么回事？我们差不多两个月没来往了！同性恋？

　　演戏！你就是会演戏，又自私又狡猾又让人捉摸不透！

　　到底发生什么事了你说啊，你不说我连解释都无从下口。

　　今天下午，连涓盯着她的眼睛，一字一顿地说，42层的中意大厦楼顶，那个叫卞铭菲的女孩，她说她爱连漪，快把她给想疯了，还……她说不下去了，眼前是卞铭菲疯狂吻她的景象，她竟那么长时间才反应过来！

　　什么女孩？她比我们大一岁呢。女孩女孩的，你总是居高临下！你不遵守合同，谁让你去那里了？！

　　别转移视线，你回答，你和她之间怎么回事？

　　什么都没有！

　　连鸣的出走就是跟她有关系吧？你和她也有关系？这是怎么回事？

　　什么我和她有关系？我们之间什么关系也没有！我声明！我严重声明！不要败坏我的名誉！更何况我们现在连朋友都不是！连漪喊了起来。她也从未这样，表现得像个妹妹。

　　那你为什么要和她决裂？

　　这个不要你管。

　　你是我妹妹，我不管谁管？

　　现在你是我妹妹！就是不要你管！管好你自己吧，可怕的黑眼圈。连漪用一副卡通的表情，噘着嘴把头扭到一边。

　　连涓看着她，无话了。她知道问不出什么。为什么这样复杂？她从未想过连漪的生活里会发生这样的事情。

　　连漪见她放弃了，反过来一脸坏笑地看着她："嘿嘿，该不会你们之间发生什么了吧？"

　　"哼！总之你好自为之。回去！"连涓说着转身就走。

　　"喂！有？发生什么了？"

　　连涓被她缠不过，就说被她从背后抱了一下，就是这样。

　　就是这样？然后呢？

　　我把她推开，她的头撞破了。

　　真的？连漪的心立马疼了，你这么不懂幽默？你可以把胳膊张开，你可以笑着说：泰坦尼克啊，你开玩笑就会化解的，你真的把她的头撞破了？

　　是啊，都流血了。我可没你那么有风趣，在这种事面前还会风趣。

　　想到那一幕，连涓又恼怒了，眼里尽是厌恶，走路的速度也快了起来。连漪落在后面，想着卞铭菲此时像个被遗弃的小兽一样，孤零零一个在黑暗的角落里蜷缩着，舔舐着伤口，可能连饭都不会吃，还得化上妆，一脸微笑地去弹琴，心疼极了。

　　失魂落魄地踱回家，把自己关在屋里，爬上床，盯着外面的灯火发呆——那别人家里亮着的、空中闪烁着的、大街上流溢着的、海面倒映着的、更远处迷离着的……这么多灯火让这城市显得辉煌、温暖又有些暧昧，多少孤寂的孤单的灵魂在这灯火后面呐喊？多少受伤的心灵在这灯火后面哀叫？连涓粗暴地把卞铭菲推开，骂她神经病，让她滚远点，她撞破了的额头血肉模糊，她的眼睛里全是毫无防备的惊讶，她感觉被抛弃了，……这一幕情景不断在连漪面前演绎着。——她的心一定被摧毁了！她伤心了！她真的爱我？真像连涓说的那样？她想起她做的那个梦，想起那个冰凉光滑的蛇一般的身体，想起蛇的手指，想起卞铭菲脸上闪过的不安……是的，一定是。她见到郇兵，立刻以伤害连鸣作为报复，她在隔壁故意大声地呻吟让我难过？！这是真的？！怎么会？太不可思议了。她想到阿龙，或许他能证明些什么。手机关机，家里的电话响了好久才有人接，是他母亲，问清楚她是谁后，说阿龙两个月前去了加拿大，不会回来了，他让我转告你一声，他祝你快乐。阿龙母亲的声音很冷，说完这句话像完成任务一样立即把电话挂了。

　　连漪呆住了。

　　连涓喊吃饭了。"病""好"了之后，她对做饭和吃饭的着迷程度让人吃惊。此时她用食指关节响亮又急促地敲着连漪的门，喊了四五遍，连漪才厌恶地应了一声，慢慢地下床，猛地拉开房门，连涓早已闪在一旁，故作一脸不屑地站在房门口盯着她，连漪狠狠瞪她一眼，说声"无聊"就往餐厅走，连涓也不说话，哼了一声，挤到她前面。

　　晚餐很丰盛，但各自有各自的心事，气氛沉闷，连涓有时大声地咳嗽一声，提醒他们这饭是她做的，至少给些赞许吧，只遭遇连漪几个逗气的眼神，没有任何回应。快饭中时，秦甡似乎终于领会了她咳嗽的含义，指着面前的一盘醋溜大头菜说这大头菜味道不错，然后"嗯"了一声，轻描淡写地说他要出诗集了，和一位诗坛新人合著，写的是一些有关生活生命命运的感想感悟，他和那位诗坛新人的风格很不同，名字不好起，请吃饭的各位参考。

　　丁秋平一听，把自己的心事放下，做出极有兴致的表情恭喜他，然后问你该有五六年没出新诗集了吧，问都起了些什么名字。

　　秦甡苦笑：是啊，五六年了。和那位诗人讨论过各种类型的，深沉的，清新的，古典的，现代的，一个字的两个字的三个字的，都有，都觉得不妥，唯一取得一致的就是"美丽新世界"。

　　美丽新世界？好啊。丁秋平说，这个名字不错。

　　这个名字让连漪心里震了一下。她知道那位所谓的"诗坛新人"就是卞铭菲，心里有种难以名状的难过。她飞快地吃着饭，又不想过早地离席，那毕竟是父亲，一个把写诗当成一生事业的父亲，沉默好久后重又开始的诗人，这时扫他的兴太残忍。而连涓，如果不是有契约束缚着早开口了，她看着连漪的样子，不自觉地叹了口气。

　　秦甡问连漪说："小涓，你认为怎么样？"

　　连漪有些别扭地说："不错，很好啊，很好。"

　　"小漪呢？"秦甡故作自然地问。

　　"我觉得很好，"秦甡话音刚落，连涓就笑着说开了，"很健康，很积极，健康和积极正是现在人们所需要的，一定火。"又问："和你合作的那位诗人叫什么名字？"

　　轮到连漪咳嗽了一声，连涓才不理会呢。

　　秦甡没想到一向沉默一向"懂事"的连漪会这样问，表情立时尴尬，夹菜的手一时竟抖了起来。连漪看在眼里，难过了，眼泪差点掉了下来。她忙掩饰，故意大声地吸了吸鼻子。

　　"今天不知为什么，鼻子一直发酸。小妹，你问那么多干嘛？你又不认识。"

　　"不认识，问问不行吗？"连涓不依不饶。

　　"是啊，"丁秋平也开始了，"叫什么名字？是男是女？很有名气？"

　　"啊，是个女诗人，斩露头角，笔名，笔名叫菲菲……"秦甡结结巴巴地说着。

　　吃过晚饭，连漪把连涓拽了出去，她很生气，重申了一遍契约的第三条和第四条，然后问连涓，对诗没丝毫兴趣的她干嘛表现得这么兴致勃勃。连涓说你问我？你该问问自己，爸爸这么多年没出诗集了哎，我们表现得那么平淡会让他多伤心你想过吗？我？连漪我我我我我地语塞，总之以后要严格信守我们之间的约定，否则后果自负。

　　自负？怎么负？会发生什么？谁知道。该不会出师未捷身先死吧？唉！乱了，开始乱了。
