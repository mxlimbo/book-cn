三十五

　　连漪看到了连涓和郇兵在一起时的模样，仿佛一对亲密无间的情侣。那天下午她跟在他们后面去了好多地方。商城，卡丁车俱乐部，汉堡店，最后是新中国的士高。这些地方她平时只是看看，现在她进去了，看到了里面的社会，看到了在里面的社会里，连涓如鱼得水，而郇兵也玩得得心应手。他们开心地笑着，跳着，畅谈着，快乐着，真正的女主角却只能站在角落里远远地看着，心痛着。

　　如果换成是她，这些情景，这些快乐，永远都不会存在。她只是一个苦行僧，一个心存大处的修行者，她能给这世界的给郇兵的或许只会是一些痛苦的教义，而不是身心的愉悦。

　　然而这是无谓的吗？她是无谓的吗？

　　她怅然转身，如果郇兵能够给连涓的人生带来转机，如果连涓能给郇兵带来快乐，如果他们在一起是幸福的，那么，那么我就退出吧。

　　她去了风月。不少人――就是卞铭菲说的那些瘪三模样的人――和她打招呼，任由他们把她拽入舞池。有人说舞蹈是人的本能之一，她从未跳过如此剧烈的舞蹈，表现得却像个天生的舞者。除了那些色情意味的舞蹈，他们一点也没有侵犯她，这些十八九二十岁的青年叫她姐姐，表现得敏感又侠肝义胆。

　　姐姐，谁欺负你了？

　　欺负？连漪用与他们一样的腔调说。谁敢欺负我。

　　那你为什么不高兴？

　　谁说的，我很高兴。

　　你高兴的样子怎么和不高兴的样子一样？

　　说对了小子。

　　她请他们去本市最豪华的五星酒店吃午餐，想吃什么就吃什么，别给我省钱。

　　这一次花去了4千块，是契约上规定的一年的花销。

　　如果有谁欺负你，说一声，小弟们替你主持公道。他们吃着龙虾螃蟹，不断重复着这句话。

　　如果在平时，连漪会对这句话大加鞭笞，然而此时，她感到莫名其妙地解气，并且感到有了支撑。

　　然后她开了一个让连涓那些华丽的纽扣们倾城而出的Party，要连涓带郇兵一起来。

　　她穿着白色的紧身低胸晚礼服，像个骄傲的公主。她把连涓当成道具，当成背景，谈笑风生，却无丝毫快乐可言。王子的手绕在姐姐的腰肢上呢。

　　请我跳舞！

　　她仰着头，眯着眼，把手伸向郇兵。郇兵笑了笑，接过了。

　　最简单的舞步，最复杂的内心。郇兵你看到了吗？我眼睛里闪烁的光芒，是连漪才会有的！

　　她爱郇兵。她的痛苦证明了这一点。

　　连涓的行为超出了常规。她变成了一个噬情的魔鬼。她始终无法挣周光强加在她身体上的魔咒。她在第一次时就被摧毁了。

　　那不过是连涓第一次见到郇兵。

　　她抬头看他推门而进时，有些不敢相信，他竟是这样一个英俊晴朗的男子，而这个英俊晴朗的男子竟走进了她的屋子。立刻有种相见恨晚的感觉。他就是她的理想，高大、英俊、稳重、风度翩翩，几乎完美到无法挑剔的地步。

　　可她先遇到了周光。可他却是连漪的。

　　连漪换好衣服倚在门边，不放心地瞪着她看了一会儿，叹了口气，然后对郇兵说，你坐啊，你们谈吧，我，我出去了。

　　把门关上！连涓冲她喊了一声。一副刁蛮的样子。连漪的样子。

　　 郇兵朝连漪做了个抱歉的笑容。真客气。

　　 郇兵又回过头对连涓笑笑，用在连涓听来非常之动听的声音说：你总是虐待自己的身体，你什么时候才能学会"珍惜"？

　　 外面，门重重地关上了，他们出去了。

　　 连涓不说话，朝他仰起了脸，眼里全是放肆的意味，又做了个非常连漪的动作：用手拍了拍床沿，让他坐在那里。

　　 郇兵犹豫了一下，笑着坐了过去。

　　 她趴到他怀里：我就要这样！我还想要你虐待我的身体。

　　 郇兵没有反应过来。

　　 听不懂？我想和你做爱！

　　 这句话也是连漪的风格。如果她想她是会这样说的，不会掩饰。但连涓没想到她会把这样的话说出口，他们毕竟是完全陌生的。我怎么了？疯了吗？

　　 郇兵震动了一下，松开她，问：你怎么了？

　　 我就是想嘛。每天都想，每夜都想，每时每刻都想。我爱你。连涓要把她邪恶的想法进行到底了。

　　 说完就把自己的嘴唇印在他的嘴唇上。郇兵手足无措，他没有料到她会这样。不过连涓温软激动的嘴唇很快唤起了他的反应，他捧起她的脸，主动地吻起了她。

　　 单是这吻就让连涓心驰神荡。――周光从未给过她这样情意缠绵的吻，那只是两只舌头玩的野蛮游戏。她把郇兵推开，用充满野性的眼神看着他，站起来，把窗帘拉上，然后慢慢把系带式睡衣解开，睡衣里面的身体是赤裸的。郇兵看着她，完全地晕眩。

　　连涓带着魔鬼般的表情看着他，跪在他面前，用手托起他的下巴，潭水般的双眼情欲的蓝雾迷离，直看得郇兵无法招架，他俯身上去，他的身体起了变化，他感到体内有种强烈的需要……

　　他显然太笨拙了，这没关系。游戏有许多种方式，许多种方式她都要体验。连涓脱着他的衣服，用哝哝细语轻唤着他体内潜伏的野蛮的雄性，扭动着身体让他知道她能带给他多么巨大的快乐。郇兵渐渐进入状态，像个躺在母亲怀里的小兽，轻轻地咬着她，头在她身上蹭着，硬硬的头发刺在身上时连涓禁不住呻吟，体内欲火炽烈，但她不想这样迅速就把这些温柔的细节结束掉。她不仅要性的快感，更要虚荣心的满足――她从这个纯情的男孩那里感受到了从未体验过的深深爱意，他简直把她当成了一件稀世珍宝，小心翼翼地，虔诚地，温柔地，小声地征求着她，倾诉着他的爱意，一切都令她满意，令她有感受到前所未有的欢愉。她疯狂地吻他，把紧紧夹着郇兵勃起的生殖器的双腿张开，开始更深地呻吟，大声乞求：

　　我要！我想要！

　　要什么？

　　要你！把你给我！

　　郇兵好像突然不明所以了起来，完全没有配合她的意思。他用胳膊支起身体，看着身下眼睛紧闭眉头深锁的连涓，然后颓然地倒在一边，盯着天花板，长喘了一口气，说：

　　我们结婚吧。
