二十九

　　丁秋平表面平静地看着周光送过来的一块散发着清香的牛皮，物美价廉，她知道这次没有理由拒绝，沉默了一会儿说好吧，我们再合作一次。

　　周光似乎并没意识丁秋平说了些什么，他的精神似乎高度紧张，眼睛里什么也没有，只有心事。然后下定决心似地直视着丁秋平，说话了：

　　你是我树立的一个中期敌人，如果打败了你，或者说征服了你，我就可以向我的远期目标进发。请你告诉我，我有没有打败你，或者说征服你？

　　丁秋平没料到他会用这样一番话应答，没想到他会这么直截这么露骨没一点掩饰，没一点委婉。这说明了什么？在她面前的这个年轻人开始露怯了？开始失去自信了？顶不住了？

　　她波澜不惊地一笑：你应该问自己。

　　不，这种事由敌人宣布心里才会踏实。

　　你们年轻人为人处世急功近利，什么都赤裸裸。

　　这与事情的实质无关。

　　丁秋平马上接住他的话头：那我不客气地告诉你，就事实，就目前的事实来说，表面上你确实胜出不少，至于"打败"、"征服"，这样的词还用不上，你放心，也不会有机会用得上。

　　周光对这番话回之以轻蔑的笑。丁秋平的脸色沉了下去，这个小子实在太狂妄了。

　　说实话你不是个十分有才能的人，你能坐上逸龙鞋业经理这把交椅是个奇怪的谜，当然我对这个谜并不感兴趣，但是你能在短短的时间内取得这样的成绩绝非一己之功。比如这张皮，你的最大能力其实就是上次送来的那张，这张，应该是连涓的功劳吧？

　　周光脸色略变，仍笑着站起，那好，先祝我们合作愉快。我们再来一轮，等这个春天过去，请你亲口对我承认。

　　说声告辞就摔门而去。身后丁秋平脸色铁青地把那块散发着清香的牛皮揉成一团，扔了出去。

　　周光这天上午去的第二个地方是总公司。董事长找他。在董事长办公室一坐定，他就把话题引到刚才的事件上，意气风发地把与万发总经理丁秋平谈判成功的经过及由此将产生的影响带来的效益添油加醋地谈了一番，董事长韦为迟耐心听他说完，然后说了声：不错，你很能干。不过——在此处他颇艺术地停顿了一下，周光心里一直齐刷刷亮着的灯这时齐刷刷地灭掉了，他会意地调整了一下情绪。

　　今天要你过来是要向你宣布总公司的两项决定，第一，照顾到你的精力，总公司决定让你专门负责制鞋这一块，至于皮革厂那边，副厂长张伟将接替你的位置，另外，鉴于你的助理连涓一直以来的表现，决定让她兼任皮革厂质量督导，负责监督检查所有皮革质量，有一言否决权。这项决定由你回去对她宣布。逸龙鞋业这一年来的成绩有目共睹，你树立了不错的形象，继续加油，好好干。

　　事情还是发生了。周光对这个公司再了解不过，明白他们开始一点点地收权，先是摘掉他皮革厂厂长的乌纱帽，然后缩减他别的权力，直到把他赶出公司。就是这样。就是这样。走出董事长办公室他脸上堆成笑容的肌肉立即散了下来。要想人不知除非己莫为，事情只要你做了，焉能有人不知？看来我在自欺欺人，公司掌握了我和董的关系无疑。

　　他想得对极了，他与董翠芬的关系在他上任之初就已是公司高层人人皆知的秘密。不过这并不是导致他的厄运的直接原因。

　　他说还有别的事，让司机先回去。司机不解。今年逸龙大手笔地改革了各项规章制度，在使用公车方面规定不管职务高低用车都需按一定比例掏钱。付了钱不坐车？司机很为他着想，问他去哪里，坚持要送他，周光说：你他妈的给我闭嘴！司机脸色猛地沉了下来，回敬道：你他妈的别不识抬举，说话给我小心点！然后把车发动，擦着他的身体开走了。

　　他妈的！一个破司机也敢和我横？！

　　――难道他也知道了？

　　他觉得自己又要变成一条狗了。

　　这天好天气。刚下的一场薄雪化了，街上到处是污水，周光哪管脚下，在人群车隙间横冲直撞，到处招惹骂声和白眼，就像他那些落魄时候一样。那些他曾在其中颐指气使的饭店宾馆，现在显得那么坚硬、冷漠、高不可攀，它们就是这样对待一个它们曾虔心欢迎过的客人，而这个客人尚未到破落的地步？它们敏锐得势利得令人厌恶！他发誓，无论如何不会再进入它们，不会扔给它们一分钱。在那些以前无处可去时逛的商场，他照例在电梯上立得笔挺，照例对无处不在的塑料model不习惯，照例总觉得有人带着挑剔的怀疑的鄙视的目光在监视他，难道5年中他没积累起一点自信？

　　从商场出来，他又去了火车站。这里的热闹仅次于批发市场，各色人等交集一起，喧闹又沉默，那些紧张的、茫然的、饥饿的、充满征服欲望的脸让他感到一种切肤的疼痛。五年前，刚毕业的他就是从这个简陋寒碜的火车站出来，带着类似的表情进入这座城市的。他上的是师范学院，本该成为一名神圣的人民教师的，然而教师不是他的理想，上师范学院只是因为那里学费便宜罢了。他只想做商人。他发誓要成为一名成功的商人。但这个城市显然不怎么青睐他，在初来此地的三年中，他做杂工，搞传销，倒服装，曾有四天粒米未进的经历。这一切随与董翠芬的相识结束，只要能实现抱负，出卖肉体又能如何？他开始意气风发了，他开始出人头地了……

　　而这一切又将随董翠芬的结束结束？是想得太多了吧？我确实是在一直努力，一直在证明自己的能力，我做得不错，他们没有理由！

　　自我安慰不能平复他内心疯狂汹涌的黑色狂涛，他需要实在的发泄。在一个公用电话亭，他停了下来。

　　连涓的早晨还没开始，厅里的电话铃声遥远又模糊得很不真实，但还是打乱了湮远纷乱的梦境，她醒了，看了看钟表上的时间：10：10分。10：10分？朱镕基的眉毛？她带着恶作剧似的表情笑了。家里又剩下她一个。谁啊这是？催命鬼！没皮没脸！死去吧！电话响了无数遍后她招架不住了，懒懒地爬起来，懒懒地走过去接了。由于从未在家里接到过周光的电话，他的声音从另一端传来时她差不多是跳起来吃了一惊。

　　出来好吗？我想见你，半个小时的时间够不够？风月五楼502房间，我等你。周光早已看穿她是连涓而非连漪，这种游戏逃不过他的眼睛，只是他不想这么早就揭穿而已。陪她们玩吧，看她们能玩出什么花样来。可今天他惊觉：，等看到她们的花样的时候，或许已经晚了。

　　连涓除了接电话的那句"你好"外，什么也没来得及说。周光的声音阴沉得可怕，一定发生了什么事。风月？她有些慌张地去迅速地洗脸刷牙，然后走到连漪屋里把那些化妆品找出来，湿粉掉在木质地板上，一声低哑的闷响让她吓了一跳。眉笔在哪里？天啊，时间快到了！原先决定化得淡淡的，可不知怎么越化越浓，决定把它们擦掉时，来不及了。当然不会穿连漪的衣服，她从自己过去的大衣橱里找了套从未穿过的领口处装饰着闪闪发光的亮片有旗袍风格晚礼服效果的衣裳。从鞋柜里找出一双白色高跟鞋，把黑色的皮大衣取出来，匆匆奔到客厅，发现忘了带包了，落地钟上的分针已指向了50，时间过了！没关系。干嘛这样慌张？不，这没什么。镇定。镇定。

　　她又去卫生间照了一下镜子，摆了一副神情迷乱的下贱的表情，甚至呻吟了一声。

　　风月。

　　周光下午三点匆匆回到公司。连漪看了他的模样，心里忽然一阵难受，情不自禁地拿起电话要拨给连涓，周光又匆匆出来，叫她进去一下。

　　面无表情地宣布总公司那边的决定。

　　连漪心里一笑。其实她早知道会有这样的结果。她不是个甘做幕后英雄的人，也不是明知连涓被玩弄而坐视不管的人。其实她表面的心无城府下报复之火一直在熊熊燃烧。

　　这一点连她自己也没意识——要应付的东西太多了，她还来不及照顾到自己的内心。在短时间内准确地摸清了周光的本质后，准确地猜到他就是连涓所指的那个男人后，准确地知道他想把万发打挎之后，在对周围的新鲜新奇感平淡下来后，她就开始一步一步实施起她的报复。尤其周光做事的冷酷、自私和绝情让她无法忍受。她开始让总公司那边知道，周光现在只会才尽地指手划脚，没任何指导意义，创造不出任何新价值地指手划脚。而她，她会给逸龙制鞋甚至整个逸龙带来新的经济增长点。

　　只要给我机会，我会将整个世界颠覆。

　　总公司那边任人唯贤，而且眼睛雪亮，知道是谁让皮革厂在短短时间有了内质的改观。

　　是啊，很奇怪。

　　是吗？周光看着她，心里在过着他们的相识过程，她有着绵软的外表，有着纯真的眼神，然而内心深不可测，比连涓凶猛十倍。表面仍无表情，继续说，在这里只要好好干，有成绩，就会得到赏识。好好干吧。

　　 你很难过？

　　说实话，有点。

　　我也有那么一丁点儿难过。

　　内心高兴的同时，确实有些难过，确实是为周光。

　　晚上回家连漪特意跑到连涓的房间看她的脸色，问她今天去干什么了，连涓说什么也没干，然后拒绝交谈。

　　只要被周光需要，其余的有什么可顾忌的呢？
