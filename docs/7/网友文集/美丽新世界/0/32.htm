三十二

　　逸龙懵了，确切地说是周光懵了。万发不是被打击得独自一个呆在角落里生闷气，而是采取了条迂回战术，不过三个月的时间他们竟在北方、南方、大热门的西部甚至东南亚等地收购兼并了约60家经济实力单薄的小制鞋厂，这些制鞋厂在短短的时间内就步入了正轨。当都在淡季里休养生息时，他们响亮地于春光明媚的三月在国内外六十多个城市的各大商场同时举办了规模盛大的春夏季新款皮鞋新闻发布会及亮相会。名曰：看吧看吧，穿吧穿吧。

　　这就是这次重要会议的内容。周光看着韦为迟甩在他面前的一撂五彩缤纷的宣传资料和亮相会入场券，哭笑不得。　

　　中低档。皮一族，革部落，皮革同盟军。走百姓线路，军事防线建在对手门前，他们考虑的是大局，根本没把逸龙放在眼里。

　　有思想工作"专工"之称的韦为迟只会拍着桌子嚎着嗓子震天响地冲他嚎，他妈的真是管屁用！

　　第二天一上班他又把这一撂东西甩到了连漪桌上。

　　连漪没料到母亲会选择这个时候采纳她的建议，她看到了她们之间的差距。一个不骄不躁，不气不馁，胸怀天下，一个急攻近利，玩弄感觉，游戏人间，她这人间还就只这么丁点儿个盛着几十万人口的巴掌大的地方！

　　万发和逸龙，本是亲兄弟。逸龙是老大，叫第一皮鞋厂，万发是老二，叫第二皮鞋厂。九十年代初，老二的厂长看不起老大的厂长，闹了独立，老大的厂长也看老二的厂长不顺眼，净拖他后腿，毫不犹豫地一脚把它"踹了"，然后兼并了当时效益还算不错的皮革厂，改名为逸龙。老二改名为万发。俩兄弟从此开始了旷日持久的龙虎斗。万发由于在改制方面牵扯了过多精力，出师不利，初时不仅负于老大，且连年亏损，董事长向龙不得已引咎辞职，丁秋平应运而生。就职演说上她说了句让老大笑破肚皮的话：敢教日月换新天！结果三年后这新天还就形成了，而且一直"浓浓地笼罩"着逸龙。逸龙直到1999年底才拨开乌云，见到一丝天光，算是出了口恶气，然而能出这口恶气并非全靠实力，大部分要算侥幸——逸龙资格最老的设计师车祸归天，怀才不遇的关莉莉直捣黄龙，拿着自己一门心思搞出的"闪亮一族"毛遂自荐，管理经营决策才能平庸，用人眼力却不俗的韦为迟给了她一个机会，当然，还有周光的误打误撞，请了连漪做了一期还算可以的广告策划。

　　现在又乌云密布了。

　　在万发的新鞋亮相会上，周光连漪他们又感受到了另一种震撼。所有款式按价格层次先后出场，风格与以往几乎完全不同，动感的、时尚的、狂放的、绚丽的、古朴的、职业的、休闲的，全透着新意，简就简得不能再简，繁就繁得不能再繁。人为本的理念又保证了所有的款式的舒服与舒适。其设计思路天开地阔，其制作鬼斧神工，就是这独特的设计，这巧妙的制作，让低档的普通的皮革焕发了新的生命。 33元、44元、55元、66元、77元、88元、99元，每个价格档次的皮鞋都有着巨大的市场潜力，不同性格，不同年龄层的人都能在里面找到属于自己的鞋。他们还破天荒地请了质量监督部门现场做抽样检测，结果质量全部过关，"完全信得过"，"真正地物美价廉"。谁都知道淡季，这个在商场上本来就不该有的概念被万发潇洒地扼杀了。

　　连漪问丁秋平选择这个时候推出新款就没考虑到风险吗，丁秋平稳稳一笑：你看看现在的人们，几个有主见？他们不知道生活的方向在哪里，随便找个领域，只要你自信得无懈可击，你怎么牵，他们就会怎么走。

　　这话不怎么好听，可事实不就如此吗？她不也这样认为吗？她只是想，母亲却把它变成了实实在在的生产力。总学不会想不到将想法与现实契合,学不会想不到把理论和实践结合起来！只能怪自己太单薄，太肤浅，缺乏胆识，缺乏经验，啊，浪费了太多的时光在无谓的虚幻的东西上！

　　现在就有机会。还有半年。她确立了目标：证明自己，打败万发。

　　妈妈，下一个弯道，我要超车了。

　　丁秋平的笑容既有英雄惜英雄的信任又有母爱的意味，她拍了拍连漪的肩膀：超吧，能被自己的女儿超过，我会自豪的。

　　周光希望丁秋平对他说点什么，但她什么也没说，目光压根儿就没在他脸上停留过，哪怕那么一秒钟也没有。尴尬、恼怒、失落,还有被挫下去的欲望，自暴自弃的颓唐一齐挤压着他,他觉得自己快被引爆了。没人看得起他，没人在意他，他为成功为引起重视为证明自己殚精竭虑到最后还是一文不名！

　　——真的到了最后吗？

　　自命不凡的关莉莉则不动声色地沉默着。听说万发新聘请的设计师是个尚未毕业的大学生，就是坐在丁秋平旁边的那位稚气的女生？不错，气质很好，朝气蓬勃，目空一切，不是这个中等规模的城市的风格。这样也好，起码有了动力。――固步自封太久了。当她开始耽于安逸，当她开始偷懒，当她对自己开始有些失望的时候，来了动力？绝非坏事。三十岁的女人需要对手，需要动力。

　　万发为这个城市大规模的各种各样的狭隘打开了一个缺口。城市又开始为万发自豪了，城市又开始了一番喧嚣。这种所谓理念的东西能被广泛地迅速地领悟深透，但在行动中，却"困难"重重。这不是个有胆性有灵性有个性有创造性有超前性的城市，灵机一动独领风骚的时候太少了。对新生事物大力宣扬，实际上无能为力，就像只老猫一样老诚地蹲伏着，对刺激敏感，被刺激起来又缩手缩脚，蹑手蹑脚，竖起耳朵四处观望一番又蹲了下去。所以住在里面的人会觉得压抑，被压抑紧紧地捆绑着，身体和精神都挣脱不出，压抑也毁坏了许多有创造性但无公关能力交际能力的人才。

　　连漪曾长时间地这样认为。现在她发现她的辩证法颠倒了。是太多平庸的人堵塞了城市的发展，平庸的城市无法阻碍一个敢做敢为能力强大实力雄厚的人。那么就不要去抱怨，不要让抱怨把自己毁掉，去做那敢做敢为能力强大实力雄厚的人吧。——要让这个城市为我感到自豪！

　　连漪用了三天的时间考虑这个问题，并发现了自己沉睡已久的野心。这种野心一直存在，从她第一次踏入这个城市起就有了。她没有把握住，在那样长的一段时间中她被一些次要的东西蛊惑了。――怎么可以用两年的时间得一场可耻的抑郁症，就因为那实际上没任何害处的命运？

　　协作精神！要学会与人协作。孤独是可耻的，它是内心的阴暗滋生的毒草。

　　她和关莉莉完完全全地走到了一起，做市场调查，四处搜集资料和情报，分析季节特点，划分消费人群，在成百上千的鞋纸图样中寻找有卖点，有市场潜力的款式，把万发设定为自己的短期对手，先是超越它，然后是成为国内最具知名度的名牌企业，然后是亚洲，然后是世界。这不容易，但想做到也不难，到母亲的岁数还有27年？够了。

　　周光一个月后才发现自己被撇开被绕开了。把连漪叫过去，心平气和地问：这段时间在做什么？

　　你知道。连漪脸上带着笑，冰冷的笑。

　　周光本来想拐个弯的，对她这样的直接没有防备。

　　这算什么？

　　工作。不是工作是什么？

　　我呢？

　　你？谁知道，你不是也在工作吗？好像更忙。

　　你是我的助理，你却不知道我在做什么，我也不知道你在做什么？这是不是笑话？

　　这怎么能算是笑话。那些落后的程式才可笑。

　　你到底想怎么做？

　　把工作做好。

　　我呢？

　　我刚才说了。

　　咆哮了：你她妈的有没有把我放在眼里？

　　没有。连漪平静地回答。我看不起你。真的。所以没办法和你合作，也没兴趣……（她是想说也没兴趣和你玩下去了）你辞职吧。

　　再怎么想也不会料到她会说这样的话。

　　太直接了？我就是这么想的，也这么希望，没必要费时间费精力地去做什么宛转。

　　周光不可思议地"哈"了一声——这当然不是笑，无力地回应：还有你这样的人？

　　今天见识了？写辞职报告吧，或者我再代劳一次？要不就别打扰我。

　　我是你上司！

　　不，只有更高的实力更高的能力才是我上司。除此之外，我谁也不认。

　　我可以炒你鱿鱼。

　　怕被你炒我还会这样说吗？

　　这句话让周光一下子想到了自己的处境。——她也知道了？

　　你听说什么了？

　　什么也没听说。不过你江郎才尽是人人心照不宣的不争事实，而我，才不过刚刚开始爆发。

　　连漪说完就出去了。这一番对话，她一连几天都怀着兴奋的心情反复回味。像是打了一次翻身仗，像是爬过了雪山，走过了草地，眼前清明开朗，一派大好气象。

　　这就是终于找到了自我的感觉吧？

　　一直如浮萍一样飘荡的她终于找到了自我的感觉。
