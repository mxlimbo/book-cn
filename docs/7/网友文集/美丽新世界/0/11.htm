十一

　　"大鸟"广告策划公司。

　　连漪第一次出来应聘工作，第一次觉得自己弱小又单薄。对方要求本科以上学历，至少两年的相关工作经验，她什么都没有。她从未真正和这个社会接触过，只是个有抑郁症史的高中毕业生。还有三个人轮到她，她们都一头鲜亮的长发，目光充满自信，都带着厚厚的文件夹――那就是社会经验吧。有那么几分钟她打起了退堂鼓，但强迫自己坚持下来――终要踏出这一步，否则逃避到几时？

　　"连漪。"秘书叫她了。她觉得这冷冷的声音里尽是不屑和嘲讽。

　　果然一上来就问学历和经历。

　　没有。什么都没有。

　　那个带着无框眼镜自称姓秦的男人做出一副不解的表情看着她，她看起来很稚气，穿的也是家常便服――白色毛衣，牛仔长裙，白运动鞋。好一会儿――仿佛要给她某种压力似的――才说："那你来干嘛？我们这里不是失业无业人员收容所。"

　　意料之中。但他的话让连漪来了斗志。

　　"我是来这里面试的，是因为觉得自己有能力胜任才过来的，至少该给我一次考试的机会吧。"

　　"好――吧，"很长一段时间的沉默后，考官似乎是叹了口气才把这两个字吐出来，"我手头上正好有一个糖果的案子，生产，销售，一切都是零，连名字也交由这里做。如果是你你会怎么策划？去写个策划案，一个小时内交上来。"

　　"糖果？就是吃的糖果吗？"办公室里的其他人笑了起来。

　　"是。"对方显得耐心十足。

　　"我现在说可以吗？"

　　眉毛一挑，作了个请的手势。

　　"名字就叫甜蜜蜜吧。可以根据不同的人――群，做出，不同的系列。比如儿童系列，以水果、牛奶和巧克力口味为主；情侣系列，以柠檬巧克力可可和，和，和那个，那个叫什么？就是牙膏里用的那种东西？"

　　"薄荷。"

　　"对。对不起，我忘了，是薄荷，情侣套装就以柠檬薄荷巧克力和可可味为主，中年人可以虾酥花生之类为主。可以根据不同的节日推出不同的系列。广告词可以说，甜蜜蜜童年，甜蜜蜜人生，甜蜜蜜糖果；酸酸的甜蜜蜜，甜甜的甜蜜蜜，苦苦的甜蜜蜜，爱情的甜蜜蜜，岁月的甜蜜蜜？"连漪脸红了，怎么说出来这样俗气这样生硬这样没劲？她几乎是做了个鬼脸："给我时间我会组织好的。"周围又笑了起来，秦考官也笑着往上撮了撮眼镜。

　　"不错。我们综合一下，再给你答复，OK？"

　　这个"OK"可不是商量的口气，连漪知趣地站起身：

　　"我想现在，就知道结果。"

　　她知道卞铭菲的命运，她不想为一个失望的结局充满希望地等待三天一个星期或更长时间。那太折磨人了。

　　"如果可以的话，三天后会通知你。"

　　"我现在就想知道。如果你觉得不行，请你现在就告诉我，我可以承受。"

　　秦考官笑了，看着她，她真的很纯，像个还在上学的高中生，和工作扯不上关系。

　　"你的策划我觉得，怎么说呢，有一定的可行性。这样吧，给你一个月的试用期，如果能通过试用期你可以正式留下。"

　　"真的？"

　　"不是因为你有什么过人之处，只是感觉，嗯，是因为觉得你可能有些潜力。"

　　这句话让连漪很不服气，想想说："试用期不是半个月吗？"

　　"多给你一些时间熟悉一下这行的工作方式方法，还有这里的环境。"

　　"就半个月吧，半个月之内如果觉得我不行，你告诉我，我会立刻离开，这样你们也不会有什么损失。"

　　"是你很有自信吧？"

　　连漪笑笑。

　　"那一切就按我们的招聘启示来，下个星期一你过来上班。"

　　"下个星期一？为什么？"

　　"因为明天双休日，没有工作我们这里是不加班的。"秦考官又笑了，向外招呼一声"高绮。"

　　刚才通报连漪进去的那位小姐应声而进。

　　"带她去熟悉一下门路和人事。连――漪，是吧？下周一过来直接投入工作。"

　　"是！"连漪敬了个军礼。她很少开这样的玩笑了。

　　回家的路上连漪简直想飞起来。一会儿高兴地笑起来，惹得路人们频频侧目；一会儿又一脸失落：如果认真地想一想不那么唐突，我会说得好一点；一会儿又脸红了：秦经理一定觉得我很好笑吧？

　　经过万发鞋业时，她想了想，进去了。

　　秘书得知她是连漪很吃惊：总经理的这个女儿从未来过啊。

　　"有急事？不巧总经理正和一位重要客户……"

　　"没关系。我等她。"

　　秘书笑了："还有十分钟下班，里面估计也谈得差不多了，你先坐吧。"

　　连漪坐了一会儿，趴到秘书的桌上看她工作，她忽然对工作充满了好奇。秘书笑着回答她的问题，问她怎么了？她摆摆手说没什么没什么，在一旁转了转，说，我出去等她。

　　丁秋平和周光一起出来。连漪正在院子中央的花坛里看美人蕉，上面还留有几朵鲜艳的花呢。小学的花坛里也种满了这种花，她们把花摘下来，吮吸里面的花蜜，很甜。周光看到她时吃了一惊，丁秋平说：那可不是连涓喔，是妹妹，连漪。周光看得呆了，连声说：真象，真象。当然，她们是双胞胎。丁秋平也觉得今天的连漪满面红晕，很有朝气。周光很快意识到自己的失态，忙说，就此告辞吧，希望您能认真考虑一下我的建议。丁秋平笑了笑，淡淡地握了握对方伸过来的手：等你们高质量的皮革出来再说。

　　周光进车后，把车窗摇下，深深看着红色美人蕉花下动人的连漪，直到车子驶出万发。

　　丁秋平的汽车上。

　　"到底什么事？过来找我又说没事，自己又偷着乐。"

　　连漪也觉得很丢人，可实在没办法控制自己的情绪。

　　"快说啊。再不说我没耐心了。"

　　"我――"连漪闭上眼睛，仰起头，又低下头，语调在车内的空气中划了一条弯弯的彩虹，司机老王都笑了。

　　"我――"又一条彩虹。

　　"别这么婆婆妈妈的，快说。"

　　"好，那我就痛快说了。我――下星期一就去工作了。"

　　"真的？"丁秋平真的吃惊。她曾为连漪工作的事联络过许多地方，连漪一直拒绝，她要靠自己的实力赢得工作，却又按兵不动。她以为是她继承的那些遗产在从中作祟，消磨了她的斗志和紧张感，一直旁敲侧击地提醒。原来这个不动声色的小家伙在背地里行动呢。

　　"嗯！"连漪羞涩地重重点了一下头。

　　"什么公司？是做什么的？"

　　"大鸟广告，是广告策划。"

　　"真的？不错啊。老王你听到了吗？不容易啊，可得好好庆祝一下。"

　　连漪脸更红了，笑着看着窗外。丁秋平看着她，百感交集。这个女儿从未和她坐得这样近，从未和她谈过心，甚至她觉得在她心里总怀着对她的恨意……她把手搭在连漪肩上，看她没有躲开，就把她揽在怀里，眼眶里竟有泪光闪烁。
