二十

　　这是你吗？连漪不敢相信，是太阳在跟我开玩笑？紧紧闭上双眼，然后睁开：那就是郇兵。

　　郇兵脸上没一丝表情，只是看着她。他们就这样对峙着，任凭好奇的不好奇的人们在身旁来来往往。良久。阿龙受不了了，咳嗽一声，说，也许我该离开？

　　"你为什么用这样的眼光看我？你凭什么用这样的眼光看我？"连漪不理会阿龙的话，盯着郇兵，不想在他的目光中落败。

　　郇兵没说话，又看了她一会儿，转身便走。

　　"喂!"连漪追上去，喂，你神经质吗，为什么不说话，是在谴责我吗，你有什么权力这样？

　　阿龙倒退着看着他们面对面地越走越远，然后转身跑开了。

　　郇兵的难过程度远非连漪所能想象。在郇兵心里眼里爱情是简单的，爱情就该是简单的，我爱你，你爱我，彼此承受彼此的感情，彼此寄托着对方的感情，全部的感情，这感情无论如何哪怕是一丝一毫也不能发生在第三者的肩膀或胸膛上，在电话里连漪用话语伤害了他，现在她又用行动伤害了他，他外表仍平静，但内心已是快发了狂的豹子，他自顾自地走着，不理会连漪的软磨硬套死缠滥打，右转右转一直右转，整整走过了三条街，直到连漪扑到他怀里，用尽全力把全部的生命紧贴在他身上。

　　"对不起，是我错了。"她并不是真地在为自己的行为道歉，是郇兵的行为深深地激发了她体内令人不可思议的母性，此刻她想用全部的生命怜爱他。

　　郇兵止住了。

　　"是我错了，原谅我，原谅我。"

　　在说话的是他爱了近十年的女孩，她在他怀里显得那样弱小，那样楚楚惹人怜惜。就是刚才，他在她家的客厅里，她的父母向他述说了一个完全不同的连漪，他不知道她有那样的身世，不知道她在疗养院度过了两年，记忆中连漪那些华丽的与众不同的裙子和不可思议的行为表达的再也不是张扬与骄傲，而是一种深深的落寞与孤单。"有一种维生素她严重缺乏，那就是爱，心灵上的爱。只要她能从任何人那里感受到爱意，都会轻易付出自己的情感与之呼应，这很危险……用你全部的爱去滋润她去感化她吧。"他一路深味着秦甡的这句话，去她的公司找她，没有想到，发生的一幕为这句话作了个很好的注脚。

　　他说没关系，然后把她从身上松开，看着她，说，真的想吃饭？

　　嗯！连漪乖乖地点点头。

　　"西餐？"

　　"嗯!"

　　"麦当劳？"

　　"嗯!"

　　"带我去，想吃多少都行。"

　　"我又不想去了。"连漪猫一样看着郇兵。

　　"怎么了？"

　　"我想吃你。"

　　这不是玩笑，她是真的想把他吃掉。

　　麦当劳太远，他们去肯德基买了十个汉堡，十盒薯条，十杯可乐，十杯热奶，打包！连漪把赠送的沙拉酱和小包的薯条给了一个独自在吃儿童套餐的小弟弟，那位小弟弟一本正经地说，我本来是不想要的，可看你拿不动，帮你减少一些重量好了。连漪冲郇兵做了个鬼脸。没办法，来这里吃东西的小孩说话都这样，普遍早熟。

　　还能去哪里？当然是42层的中意大厦顶上。

　　乘电梯到30层出来走楼梯，连漪说这样可以保证肚子有足够多的容量。

　　暮春时节，加之已近中午，亮晃晃的阳光已把这里烤热了，一上去浓烈的沥青味就扑面而来。那里已有一把红黄条相间的巨大太阳伞撑在中间，伞下坐着的正是卞铭菲，――胳膊放在支起的右腿上，左手按在已快喝光了的一瓶啤酒上。男性化的姿势。她挑着眉毛眯着眼睛带一脸奇怪的笑容看着他们气喘吁吁嘻嘻哈哈从头到尾出现，表情没有变化，姿势也没有变化。

　　不知为什么连漪见到她及她这副模样，有些尴尬，怔在那里，不知如何是好。

　　"过来坐啊。"卞铭菲笑着说。

　　"过去坐吧。"连漪对郇兵说。

　　坐在伞下，放下手中的食物，你看我我看你轮流看着，无话。

　　卞铭菲终于改变了她的姿势，兀自把这些袋子一一打开：好丰盛哦！

　　"我，我介绍一下，她是我朋友，卞铭菲。他是郇兵。"

　　"你好。"郇兵说。

　　卞铭菲拿起一只汉堡，要往嘴里填，似乎对他的招呼没有防备，回答了一声你也好，就笑开了，说我可以吃吧。

　　连漪不知道她为什么这样，可并不出乎意料。

　　"吃啊，凉了就不好吃了。"卞铭菲大口嚼着，含混地说。

　　郇兵笑了笑，拿了个汉堡和一块纸巾递给连漪，说："吃吧。"

　　 "啊——嚏！"卞铭菲一个适时的喷嚏把嘴里嚼烂了的东西喷到了郇兵的胳膊和举着的汉堡上。

　　"铭菲！"连漪忍耐不住了。

　　"啊抱歉！"卞铭菲一边忙拿起一块纸巾给郇兵擦了起来，又一边吃着，又一边说着，"伤风，是伤风，我最近伤风。"

　　是败俗！连漪心里喊了起来。

　　"啊我看我还是离开吧。"卞铭菲不好意思地说着，然后指了指食物袋子，"我想再要一只汉堡一盒薯条一杯奶，OK？"

　　郇兵照她要的，各取两份放在一个袋里给她。谢谢！吾去矣！别忘了把伞收起来交给那个带眼镜的保安。她边吃边走了。

　　连漪已彻底没了兴致。她怎么是这样的？然后又坠落到辞掉工作的难过状态之中。郇兵没有理会她的情绪，默默吃着他的那份，偶尔看她一眼，也不说话，也没有问题。

　　"你干嘛来这里？来找我？怎么知道我地址？谁告诉你的？我哥？"

　　连漪终于说话了，郇兵笑着把手擦擦干净，喝了口奶。

　　"公司要在这里开一个办事处，我是马前卒，过来实地考察，顺便看看你，正好你母亲找我，我去过你家了。"

　　连漪忽然不知该如何以对。

　　"我母亲找你？"

　　"别问我为什么，我不知道。"

　　"考察结果呢？你认为这里如何？"

　　"我会作一个详细的报告，如果你有兴趣，可找我们领导要来看。"

　　"你，你，"连漪对他忽然强横起来的态度感到恼火，又无能为力，这毕竟是她导致的。沉默好久，又说："对这个城市有什么印象？从这里看？"

　　"从这里看？一个普通的正在发展中的中小城市。"

　　"没有了？"

　　"没有了。"

　　"没觉得这里太狭窄太肤浅太幼稚？"

　　"你这样看？"

　　"这个城市本来就是这样的！"

　　"你凭什么这样看？是因为你活得很好？"

　　看来郇兵今天铁定和她对立到底了。

　　"你为什么这样？"

　　"那我该怎样？"

　　"就是不能这样！"连漪喊着，咬着嘴唇，拼命让不争气的眼泪掉别下来，可还是没忍住，低着头走过去靠在郇兵胸膛上，悄悄地说："如果你也这样对我我会彻底崩溃的。"

　　"真的？"郇兵抱着她。他其实是想过去抱她一下的，当在电梯时往上走时，他就觉得她很孤落。

　　"嗯！你在这里我觉得很温暖。你会走吗？不要走！"

　　郇兵没有回答，默默地捧起她的脸，连漪看着他的眼神，迷惑了，郇兵低下脸，连漪知道他要做什么，抬起了自己的脸，迎接着他。颤抖的嘴唇触到了一起，郇兵的舌头舔着她的牙齿，好久，她张开了牙齿，舌头绞缠一起时，她才有了感觉，这就是吻吗？

　　她忽然把他推开：

　　"你在这里我觉得更孤单！回去！永远不要来！"
