二十八

　　时间匆匆门前过，哪管门里渐白头。春秋坐尽风水转，且看今冬谁称雄。

　　又一轮充满硝火味道的冬季大战早早地鸣锣敲鼓了。商战伊始，胜负往往取决于谁先发制人，谁气势如虹。万发吸取去年轻敌的教训，先发制人的造势运动明显猛烈许多。相比之下逸龙却显得从容淡定。连漪建议冬季度放弃低档鞋的生产，以中高档应战。周光没怎么考虑就表示同意。

　　这么果断？

　　你的直觉不错，现在的商场不过赌场，商战有时就是一场赌博，什么也打败不了准确的直觉。

　　他太自私，永远受自己的情绪摆布，任何情况下自己第一，其他都在其次，当初为追连漪而把逸龙广告拱手奉上就是证明。连漪早已摸清他的本性，亦知道他最近心里有鬼，无心考虑这个问题，也好。对他的话也不辩驳,诡秘地笑了笑，说：你的直觉也不错。

　　周光当然心不在焉，无心恋战。——原因：逸龙副董事长董翠芬贪污受贿东窗事发了。

　　周光知道会有这么一天，早就提醒董收敛一下，可这个女人胃口大过天，最后简直疯狂到变态的地步，有力转乾坤之能事的也只有法律了。周光最大的优点就是头脑清醒，贪污受贿他不沾边，董翠芬给的钱他一向拒绝，然而作为董翠芬的宠臣兼第一情夫，谁知道这件事不会对他有影响？董让他放心，说枪子儿我自个儿挨无期我个儿顶，你们这些没良心的一个也不会有事。是的。不会有事，没人知道，他们最多知道他是她一手提携起来的，以他一年多来的成绩，谁也没有理由对他怎么样，江山会依然稳固。但他走不出这团阴影，笼罩其下，每天惴惴。直到董翠芬被正式拘捕，被判无期一个多月后他悬着的心才放了下来。

　　出了周光办公室，连漪直奔设在厂门口的IC电话亭，以连漪的身份向母亲建议冬季大量缩减中高档鞋的生产，把精力放在低档鞋的生产制作上。她飞快地说最近在家里看了一些关于鞋的方面的市场情况，发觉现在多数商家都把精力放在中高档产品的生产上，根本没考虑老百姓的消费能力，低档鞋又多从南方那边过来，声誉不高，质量不高，不讨好，总之啦，低档鞋的质量、款式算是市场盲点哎，建议您这个季度以制作精美结实的低档鞋为主，缩减中高档尤其是高档鞋的产量，而且要加大宣传力度，别那么四平八稳的，完全可以开个新鞋发布会嘛。我就说这些，幼稚之言仅供参考，以后拒绝讨论。说完就把电话挂了。丁秋平在那头举着电话好长时间也没反应过来。

　　连漪知道她会考虑，并最终接受。结果会是双赢，失败的是?真令人期待啊。

　　在周光精神恍惚的这段时间里连漪并没有把太多的精力放在鞋业这边，反正今冬的生产营销理念上上下下人人都领悟在心，设计出来的新款也相当有市场冲击力，由他们去做就是了。她在暗中大胆向总公司那边请缨，要在短时间内让"令她无比激动的"皮革厂"活"起来。五次职代会，岗位工资制，记件工资制，提高内退工资，提高下岗安置费，在工人中选拔一批人组成产品质量监督小组，以国内先进水平为基本标准，优秀者工资升级，三次不过关者下岗学习，五次不过关者对不起下岗……一些通行的做法，然而短短的时间内取得了不错的效果。

　　周光对这一切当然不会一无所知，他虽恼火却不理会，只想看看她能做出些什么来。

　　两个月后，不仅鞋业这边开门红，皮革厂也红火了起来，各类皮革质量大幅提高，原先周光一门心思地想把他们装上火箭统统发送到月球的那帮人仿佛一夜之间脱胎换骨。

　　连漪一点也不掩饰她的得意，活出来似地想:一年的时间有限，我愿怎么风火就怎么风火，谁管背后周光那眦睚欲裂发狂的狼似的闪着红光的眼睛？

　　改变是显然的，母亲脸上的肌肉开始有些放松了，父亲脸上青春激荡，连漪更甭提了，走路脚跟没几次着地的时候，只有我这样黯淡。连涓困在家里萎靡不振了。事情和她想的不一样。从一种状态的生活掉进另一种状态的生活，她有机会对她自己进行一次总结。她发现自己如此虚弱无力，有无亦可。连漪每天晚上都会过来，学她的样子用食指关节在她门上响亮地敲几声，她知道这是提醒她别荒度年华，这个家伙别无能事，就是会把人一眼看穿。可她能怎么过？茫然无措，不知如何应对这每天空荡荡的时日，对生活无能为力，对自己的生命亦无能为力。有时她穿着宽大的睡袍，光着脚，无声无息地从一个屋子走到另一个屋子，在有玻璃或镜子的地方看到自己一闪而过的邋遢样子或失血的脸孔，常觉得自己跟个鬼魂似的。

　　那段曾经引以为傲的经历竟没什么意义，有什么意义呢？没在她的生命里留下任何痕迹，世界并不因她的离去而黯淡而停滞，也就是说这世界并没有因为她的存在而显得有多么不同多么精彩过……想过去没有留恋，只有这种畸形的失落感，每天深陷其中，思想变得极端又病态。对现在——她曾以为会给她以新生的现在——没有热情。对做饭失去兴趣后，她又像以往那样热衷于购物，像个疯狂的庸俗的妇人。除此之外，再找不到什么来打发时间，更沉不下心去学习。周光约了她几次后仿佛看穿了她似的再也没联系，――他看穿了吗？不，不会。是太忙了，肯定也与董翠芬的事有关。她知道周光和董翠芬的关系，也知道做事干净的周光不会受什么牵连。唉！干嘛为这样一个无情无义无心无肺的人牵肠挂肚？只因为他一手毁了自己的贞洁，毁了她的纯真年代？

　　还要报复他。她不是为此处心积虑吗？而当有了机会时，却忘记初衷。

　　遥不可及的两千年也轻佻地在眼前一晃即过了，还没来得及做什么，也没来得及留下什么，一切仿佛还未开始，却老了。再回首已百年身，凭什么相信自己会从头再来？还有那个卞铭菲，如鬼魅一样如影随形，在街道拐角，在麦当劳，在商店，只要一抬眼，一转身就能看到她那副颓败的模样直撞眼帘。这些都不算可怕，可怕的是，这个有着怨毒的眼神的女孩已闯进了她的内心，有那么几次，在每月轮回不可避免的生理期到来时她心里想的不是周光而是她。

　　——人都有着不可避免的动物性，这种动物性正统治着连涓。她心中常情欲沸腾，不厌其烦地回忆与周光那些"不人道"的性爱经历，以此自慰。除了父母、连漪，在家里能见到的就是那个每周送水两次，总穿着落满灰尘的黑色皮衣，用麻绳绑着厚厚的皮护膝，总面无表情的矮个子男人。她冷漠地开门，两手搭在胸部，一副监工模样地看着他把空水桶从饮水机上卸下，然后把新送来的水放上去，心里却想着他冲过来，粗野地把她按在沙发上。有时，一阵疯狂的臆想过后，她会对自己充满厌恶，甚至掩面而泣，挣脱不出了，挣脱不出了，这一生就此毁掉了。

　　就是这样快慰着，苦闷着，苦闷着，快慰着，不久她找到了一种安全的发泄方式——网上聊天。对方叫大海盗，他说他喜欢这个城市的女孩。他在上海，言语粗鲁，方式像周光一样残暴。

　　她不是要报复吗？她怎么忘了？

　　有必要交代一下连涓使用的网名――涟漪。
