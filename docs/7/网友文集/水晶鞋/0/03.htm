“水晶鞋”的含义＊白色贝壳（序）

＊人鱼姑娘＊


　　“哎，那个坐在石头上的独自沉思的人，你是怎么来到这里的呢？

　　听这海浪声多么优美，这里是我们的家啊！我是海底深处的人鱼姑娘，我就要给你讲讲我们的故事，还有'水晶鞋'，你要听吗？”

　　那个坐在石头上沉思的人闻声一瞧，只见一个可爱的小姑娘从水中浮出，只露了半个身子，两只手臂在水面上轻轻地浮荡。她身上裹着碧绿的水草，那种水草美得他从来都没有见过。

　　他望着她晶莹的眼睛，略带茫然地说：“是啊，我不知道的，你能告诉我吗？”

　　小姑娘说：“你等我一下好吗？！”说着便翻身潜入水中，他只看到一道优美的弧线在水面上一划，闪射出一道金光便消失在蔚蓝色的海水里，不觉呆住了。过了一会才不由自主地喊道：“喂，你去哪？你还来吗——？”

　　没有人回答，只有一声声海鸥的啼鸣不时地从远处传来，和着海浪的涌动，更让他感到周围的寂静。

　　不一会，刚才那个小姑娘潜下去的地方有许多白色的泡泡向上翻滚，紧接着他看到那个小姑娘又浮上来，一头长发经海水的湿润越发柔亮。

　　他从石头上站起，微微地笑了，说：“你又回来了？！”

　　“是啊，你不是想知道‘水晶鞋’的含义吗？，看我给你带来了什么？！”

　　小姑娘说着，从怀中取出一块若大的贝壳，这贝壳也是那么特别，洁白，光滑，一些绿色的小字在阳光下一闪一闪。“接着啊，是给你看的！”

　　他犹豫着伸出手去接了过来，定睛一看，只见上面写道：

　　“水晶鞋宣言：

　　水晶鞋，透明，无暇，有着非凡的魔力！水晶鞋所在的地方，将要容纳两种感觉：青少年所具有的热情、冲动、叛逆、单纯、张狂……；

　　另一种就是隔壁的成年人偶尔会盟动的童心：生活中一切能引发你回归童真的人，事，景物……它将以接近童‘话’的面目出现。”

　　“哦,你这是从哪得到的？”他好奇地问。

　　“是一个人间的姐姐留给我的，她的声音好动听，她悠悠地告诉我：一定会有人问你水晶鞋的含义，那时，你就把这个给他们看吧！可当我问她我怎么才能遇到问话的人，她就说了一句，你只要常在这儿游玩，见到有人在岸边徘徊，你就和他说话！她说完就一阵风似地离开了！现在，你看完了，该把贝壳还给我了！”
　　说完，小姑娘便伸出了手臂。

　　“哦，原来是这样啊！”他一边轻声低语，一边把白贝壳递了过去。
