水晶鞋的邀请

俏儿

　　天很高，很蓝，云淡而远，风轻轻地吹过，阳光格外亲切，一切都浸在平和的气息里。

　　有时会怀着浪漫的心情，伫立临街的窗前，看走过大街完全陌生却似乎熟悉的行人，看打着花伞走过长长雨巷
的女子。

　　有时会默想，所有的日子其实都是有色彩的，你的心情就是你的日子的色彩。

　　生活总是这样散文似地过去。但是岁月并非无痕，他会留下一份回味，一种情调，一点怀念，令我们惦记，令我们珍惜。

　　把你的快乐写在每一片飞过的云上，把你的笑声挥洒在每一阵吹过的清风里，把你最真最纯的心情留在我们的水晶鞋里。

　　只要你留下的是一份纯净，我们都会好好珍惜，写作风格也不再限于童话。

　　灰姑娘和人鱼姑娘在海边，在花丛，在风中，等待你们静静地述说......
