＊＊＊我们的水晶鞋＊＊＊


＊人鱼姑娘＊


　　　　　　我们就要拥有梦中的水晶鞋了！
　　　　　　这灵感来自俏儿的《灰姑娘》。
　　　　　　是的，我们就要有了！
　　　　　　这姗姗来迟的幸福！
　　　　　　我们的眼睛快要湿润了，你能理解吗？
　　　　　　　　　　
　　　　　　我们到底该怎样倾诉，这心海深处的波涛？
　　　　　　她在每一个寂静的时刻悄悄涌动。
　　　　　　如今，她轻轻地来到你的面前！
　　　　　　你将如何看到星夜里，这水晶鞋的光辉？

　　　这是一双透明的水晶鞋，"她"不接纳被“污染”的事物，当你无意间把不适合的玩笑话留在这里，我们将嘻笑着用玫瑰花瓣把你隐藏，那时，星星会笑着眨眼，云儿会羞红了脸庞……

　　　如果你因此生气了，我就吐出无数的小泡泡，把你的烦燥包围！
　　　而你，会不会在此时随一阵风远行，然后在一个美丽的清晨，给我们采来一朵七色花？！
