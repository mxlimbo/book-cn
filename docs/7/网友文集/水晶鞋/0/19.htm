生活如水,微有波澜!


弱智@鸟

　　籍着音乐的力量，把自己投放入许久没有进入的心灵深处，能感受到的是近日来沉迷于外物不能清明自处的精神。脑海里仿佛只存在一条平直的线，不知所来，不知何往。

　　或许是长时间的工作，使得心灵不能自如流转，但大半还要归罪于自己的懈惰，不肯多付出些许来返回到行为的起点——心灵。

　　记得当初明了精神与行为的关系时，那心中的波动无以伦比，似乎自己会有新的生命，但还是败在了惰性的脚下。优越生活之中的我，何尝知道奋斗的涵义，恐怕更喜欢的是在意外界给予自己的多少吧。我确实太渺小了，渺小到堪不破这层纸。

　　世界确实给予了许多，无比爱着自己的家人和朋友，给我多到不需要奢望的爱，给我最大的容忍与关心。自己是聪明的，确实自私的聪明，为了自己而欢喜，为了自己而忧愁。年少时最爱标榜自己不受外物所惑，其实有几分是因为不去在意与关心。

　　忽然很想离开现在的公司，原因只是希望多给自己些时间与空间。具体会如何，我会考虑现实，毕竟已不是轻狂的年龄。至少，我不希望离开的原因是浮躁。

　　生活如水,我希望我有颗如水的心灵!
