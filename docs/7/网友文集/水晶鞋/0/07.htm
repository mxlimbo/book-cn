给哨儿和所有朋友们的说明

俏儿

　　哨儿，借这里和你聊聊。

　　那天告诉你，你一直没在这里看到我的帖子，是因为我心情好的时候在这里发，心情不好时则在品位人生发。其实也不全是因为这个。

　　看过我们的这么多关于水晶鞋的说明，你对我们的这个版块是该有些了解了吧。当初起到水晶鞋的这个名字，也是希望这里能够纯纯净净，如水晶鞋的名字。而朋友们给我们的祝福和希望也是愿这里和水晶一样晶莹透明。

　　我有时写的一些心情绝对不是透明的，那里搀进了太多太多我流露的对现实中的不如意的情绪。

　　所以，我倒宁愿这里安安静静的没有花朵没有尘埃，也不会去刻意去加进一些东西让她绚丽热闹起来。

　　这里的美丽很多，虽然我很感谢你如此执着地只是钟爱着水晶鞋，但我也会建议你去其他的版块去走走，发现另一种风情的美丽。
