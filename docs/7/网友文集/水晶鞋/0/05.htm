[ 第一个洁白的小泡泡 ]

＊人鱼姑娘＊

　　"海水越来越蓝，我在水底漫无目的地游来游去，一群群小鱼排着整齐的队伍从身边掠过，我知道他们这样是为了保护自已这一族群！

　　珊瑚静静地呆在那里一动不动，似乎在想着心事。可她们知道吗？我也有心事呢，而且是很重的心事，我已经有好多天没有浮出水面了，那天我还听到那个少年在岸边的呼唤，他在大声地问我：人鱼姑娘，你这几天怎么不来了？你去了哪里？

　　我没有回答。可我原本想回答的！

　　我还听到有一个姐姐也在呼唤我，我想我不能再不出声了，我怎么能忍心让他们这么牵挂我而又见不到我呢？"

　　人鱼姑娘 默默地想着，不觉得开始在海底向上浮，她灵活的尾巴在水中轻柔地拨动着，她身上美丽的水草随海水温柔地飘浮。射进涨水里的阳光越来越明亮，人鱼姑娘开始发出淡淡的金光。

　　她已经来到水面，她已经依稀看到那个如风的少年留下的思念，还有那个姐姐，在阳光里坦露着最真挚的情怀，还有……

　　哎，那是什么？

　　人鱼姑娘看到右边的水面上漂浮着一丛鲜艳的什么，好漂亮的，她急忙游过去，仔细地看：一朵朵薄薄的小片片围绕着一簇粉状的蕊，她们围成了好美的图案。

　　人鱼姑娘心里暗暗思量："这是什么呢？好象听我的大姐姐说过，在人间，有一种花代表着爱情，当一个男孩将它递给一个美丽清纯的女孩，那么他们俩的心就会热烈地一起跳动！我记得还傻傻地问过她：我们海里 这种花吗？将来会不会有人送给我啊？那时，姐姐只静静地看了看我，没有说话，她的眼神很特别，我以前从来没看到这她那样深沉的眸子。"

　　人鱼姑娘围着这丛美丽的花转了一圈，想："这一定是了，人间的爱情之花，可是她为何遗落在这里，她应该在肥沃的土地上开放啊！"

　　人鱼姑娘伸出手把她们捧起来，轻轻地闻了闻，一缕芬芳袭入心脾："啊，好香啊！可是，她在这里开放会有谁知道呢？她会缺少了营养，尽管我是那么的喜爱、尽管我是那么舍不得离开她、尽管……，我还是不能不把她栽到爱情花园，因为只有在那里她才能让幸福的男孩和女孩的心一起热烈地跳、只有在那里，她才能留住最美的容颜！"

　　想到这儿，人鱼姑娘咬咬牙，毅然地捧着这丛人间的爱情之花向东方游去，她要给这丛花找一个好的归宿，她相信她会在那里艳丽，在那里怒放！

（＊人鱼姑娘＊依依不舍地给爱情茶座送去礼物：爱情之花——百果之秋的《把暗恋说出口》 ＊人鱼姑娘＊的心意你们能懂吗？）
