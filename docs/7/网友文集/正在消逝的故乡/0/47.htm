昙花

　　忘了是什么时候学到 "昙花一现"这个成语的，从此便盼望着有朝一日能亲眼看看"昙花"这种想来实在是奇妙的东西。可惜，一直无缘相见。

　　十多年前的一天，在学校的传达室里，我看见一种"叶子上长叶子"的植物――一问才知道这就是我慕名久矣的"昙花"。后来我查了《辞海》，才知道那些"叶子"其实都不是叶子，而是"叶状枝"。原来这昙花不只是花开得神神秘秘的让人难得一见，连"枝"也是这样特殊呢！

　　前年的时候，孩子他舅给孩子他姥爷弄了一盆昙花。在一个小盆里，长得太密。换盆的时候，就分出两枝来，要把它们扔掉。我看见它们都生机勃勃的样子，就想何不拿回去我也养它一盆，于是就把它们养在了我的办公桌上。

　　从此我就在放着昙花的案头读读写写。常常在倦了或者得意的时候，对着我的昙花喝茶，最后把"茶根儿"倒在花盆里。一枝枝一片片的绿色迎面扑来，足以让我神游其间。更有新生的叶状枝，油光发亮的，嫩绿的"叶片"顶端泛着若有若无的朝霞，让我觉得生命虽然短暂，却不妨时时新鲜。

　　去年夏天，因为办公室里装修，我把那盆昙花搬回到家里，养在阳台上。

　　今年7月25日，我欣喜地发现，在一片壮壮实实的叶状枝的边缘，已经长出了一个小小的花蕾，有点像是个身披蓑衣的小嫩茄子。因为学校里安排我将于8月5日去济南学习，我就盼着它最好能在我离家之前开放。

　　眼巴巴地看到了8月4日晚上，那个花蕾长到前端像小鸡蛋大小，脖子弯着像个烟斗，沉甸甸地垂在那里。像是要开的样子。可是，直到第二天早晨我要走了，它也没有开开。于是我就死心塌地地到济南去了。

　　8月7日的早晨，我从济南打电话，问孩子他妈看见咱昙花是怎么开的。她却说："还没开呢！好像是要等着你回来才开吧？"

　　8月8日我回到家里，那个花蕾长到前端像小鹅蛋大小了，脖子弯着像个萨克斯管了。再看整个花蕾，又大约像一只"曲项向天歌"的天鹅（脖子是太长了些），稀稀疏疏地披着些莫名其妙的丝带或者神秘的"霞光"，准备于静夜里绽开她洁白的羽翼。

　　眼看着，眼看着，在静静的夜里，我的昙花，一点一点地绽开……

　　"雪羽分兮，仙洞开兮。冰清玉洁，幽香淡兮。银蕊丝丝，金屑点兮。极乐净土，花中现兮！"

　　怪不得书上说佛祖喜欢昙花呢！

　　子夜时分，昙花开放到极限状态。"羽翼"尽展，"霞光"四射。

　　盛开状态持续到大约凌晨3点，便转为渐渐衰微的过程。

　　天亮之际，我的昙花，仿佛又回到了开放之前的闭合状态――它的有形的生命终于圆满地结束了！
