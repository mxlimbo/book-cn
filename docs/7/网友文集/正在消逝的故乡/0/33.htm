谁还在喝黄河水

　　谁还在喝黄河水？

　　我们中国人的祖先，肯定不都是喝着黄河水长的。把中国人全说成是黄河的儿女，是不公平的；就像他们把中国人概括为"黑眼睛黑头发黄皮肤"，是不准确的。

　　我作为一个喝过且正在喝着黄河水的"黑眼睛黑头发黄皮肤"的中国人，对所有没喝过黄河水的中国人，对眼睛和头发都不黑、皮肤也不黄的的中国人，表示同情。我想，对许多中国人来说，黄河的确如同母亲，而对另外的许多中国人来说，黄河就如同大娘或姨妈吧？咱们这"好大的一个家"要是照全家福的时候，可千万别忘了请咱们的眼睛和头发都不黑、皮肤也不黄的兄弟姐妹们到场啊！

　　根据关于我们家族的文字记载和传说推算起来，我的祖先是喝过黄河水的。只是到了后来，我的祖先又离开了黄河。我不是喝着黄河水长大的，但是，我从小就对黄河有着深厚的感情，向往着有朝一日能见到黄河并且喝到黄河之水。后来，在我喝过了汶河之水，喝过了泗水之水，喝过了泰山之水以后，又来到了黄河下游，定居在黄河边上，天天喝黄河之水。

　　乍来的时候，喝着这里的水不太习惯。我们这里的地下水都是咸水，自来水是从经过沉淀的黄河水净化得来。黄河水是被送到水库里沉淀的，水库又是在盐碱地上挖出来的。这盐碱地，大概是若干年前黄河泥沙淤积和海水自然退让的结果。所以呢，我们喝的黄河水里，还兼有一点海水的味道。如果要做广告呢，就可以说："黄河口水，有点咸！"怎么是"黄河口水"呢？俺这里正是黄河入海之"口"。那有条件"更上一层楼"的朋友，或者那生着"千里目"的朋友，就会看见，真正的"黄河入海流"之景，是在俺们这里！我就是这样想着想着，喝着喝着，便彻底地习惯了。这大概就如同海里的鱼虾，浑然不觉海水的"苦咸"。我听说有人把淡水鱼养在掺了海水的水里，渐渐加大海水的浓度；最后，它们就能在完全的海水环境中长大了。我觉得我就是这样的一条淡咸皆宜、随遇而安的"人鱼"。我还曾想，假如搞一个"瓶装水"厂，就凭这"黄河之水"的非凡魅力，定会让那些"不曾听见黄河浪"的海内外华夏儿女饮水思源、一往情深！

　　但是，近年来，黄河的污染越来越甚了！

　　我们这里有长达三华里的"黄河水体纪念碑"。这是某年，从青藏高原的黄河源头开始，每隔十华里安排一个取水点，约定于某月某日某一时刻，同时从黄河中流取得水样；然后集中到我们这里，分别注入一个个玻璃箱内，做成水体标本，整齐摆放于石墙之上，成"一字长蛇之阵"。我想水体标本中是加了防冻液的，因为冬天也不见它们结冰。

　　这据说是"全球独有"的水体纪念碑建成之初，我们真地是非常自豪。来出席纪念碑揭幕典礼的，不乏有头有脸的人物。而"黄河水体纪念碑"几个大字，是国家政协主席李瑞环所题。

　　但是透过一个个盛着水体标本的玻璃箱，我看见了流失进黄河又沉淀在这里的大量的泥沙，更看见了里面那许多仿佛水草一般的化纤，扭曲腐烂的铁丝、铁片，塑料袋子，还有些说不清是什么材料的管子、片子、块子……听说，有的人看了以后，就觉得恶心。

　　我没有恶心，我只是惆怅。我不知怎么就想起了刘欢唱的那首歌，我感到一种"弯弯的忧伤"，"穿透了我的胸膛"。所幸的是，我们这里的自来水，据历次检验，都是优于国家标准的。我已经喝了十年多的黄河水了。

　　然而，我终于还是恶心了！那是在我听到下面这个消息的时候——

　　某某大员，以及某某大员，来到"母亲河"边，来到我们中间。带着华夏儿女的情怀，带着炎黄赤子的心愿；还有，还有，还有泉城的"纯净水"啊！啊！只因为，只因为——他们嫌黄河的水脏！他们在瞻仰"黄河水体纪念碑"的时候，更感到他们的英明："啧啧，啧啧，这是人喝的水吗？"

　　当泉城的"纯净水"都变成大员撒在黄河边的尿以后，当黄河边的小员们也学着从此不喝黄河水以后，我们，我们这些不是人的东西，还在喝着黄河的水。

　　黄河的水啊，黄河的水啊！你是"母亲的乳汁"吗？

　　"母亲的乳汁"啊，你何时能再"甘甜"？
