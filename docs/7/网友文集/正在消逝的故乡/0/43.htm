早起

　　昨天夜里睡得稍微早了一些，今天早晨醒得就稍微早了一点。

　　鸟儿们在窗外叫着，日出前的白光已透过了窗帘。建筑工地上不时传来几声钢管碰撞的脆响，比起隆隆不休的搅拌机噪音来，显得颇为可爱。

　　忽然想起《朱柏庐治家格言》里说："黎明即起，洒扫庭除，要内外整洁；既昏便息，关锁门户……"想起就是一直过着这样的农村生活的父亲和母亲。

　　又忽然想起"早睡早起"也曾经是我的好习惯之一。小时候吗，晚上就是听个"瞎话儿"，捉个迷藏，在煤油灯下做个家庭作业，或者找一两个伙伴去看电影；到了中学，就得赶到学校去上晚自习了。早晨就是匆匆忙忙地吃饭，上学；中学的时候，我们不住校的学生，一般是提前吃了早饭再赶到学校去上早自习――我们在中学的早自习一般就叫做"晨读"，朗读语文、英语什么的。直到大学四年、工作以后、结婚初期，我还基本上保持着良好的作息习惯。晚上看点书，看点电影或者录像，早晨则在校园、河边、田野读书或者锻炼身体。

　　我从何时起改变了原来的作息习惯？晚上看电视，上网……过着所谓"城市人的夜生活"，每天熬到很晚，甚至熬到第二天凌晨才睡。早晨呢，或者是狠狠地睡到不能再晚，然后赶紧爬起来做饭，吃饭，上班；或者虽然早起来了，却是在电脑上泡一早晨，不到上班就足不出户。

　　今天早晨我忽然有一种出去逛逛的渴望了。

　　于是就从半空中的家中下来，又从地下室推了自行车出去。这外面的空气真是清新啊！是什么使我忘记了享受这大自然的恩赐？

　　迎面过来一位白发老者，手里提着油条和新鲜的瓜果蔬菜。看样子是早去散步了一圈，顺便买着些东西回来。跟他一比，我真是惭愧了。

　　下了沥青公路，走上田野的土路。

　　水稻叶子的那种又绿又黄的颜色和整片稻田上泛着的光晕，是那样夺目和提神，是那样柔和而舒爽。荷塘里的叶子是厚实的，花朵是硕大的，叶柄和花柄是粗壮而显得有些矮的。这不是林黛玉的荷塘，也不是朱自清的荷塘；这是农民的荷塘，也是我的荷塘。

　　开始我以为我是在独享这些美好景色的，可是转眼我就发现我又错了。稻田里早就有拔草的农家女了；小船上早就有看网的渔夫了；芦荡边，早就有挥竿的钓叟了；成群的燕鸥早就在忙了；寂寞的布谷鸟早就在等了。

　　逛回来的时候，路过农贸批发市场。我也顺便买回了油条、豆汁和又便宜又新鲜的瓜果蔬菜。丝瓜一块钱二斤；零卖一块多一斤的小甜瓜才要六毛，我五块钱买了它十斤；还有扁豆。扁豆令我想起当年在农村的生活。从篱笆墙上摘下扁豆，洗净切丝，加小辣椒、香油、酱油、醋、盐，生着一拌，啊――美啊！
