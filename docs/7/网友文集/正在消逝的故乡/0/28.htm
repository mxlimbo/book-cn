孔林

　　从十七岁到二十一岁这段时间，我正在古老的曲阜——孔夫子的家乡上学。

　　那时我的生活基本上就是两大方面：读书和散步。读书最爱读鲁迅，常常觉得他在替我说话；同学们说我三四年读完《鲁迅全集》很有毅力，我觉得这话根本没说到点上。你喜欢嚼牛肉干，还得用"毅力"去嚼吗？散步常到南沙河，也常在学校周围的农田里转悠，而能让我体验到散步的最高境界的，却是孔林。当时的曲阜，没有现代城市的繁华喧嚣，自有一种高古悠远淳朴肃穆的神韵在，所以不论是读书还是散步，都可以十分投入地进行。

　　孔林一直就是孔家的坟地，直到现在还是。说到坟地好像有人害怕，我却是在里面呆习惯了。我那《空水诗选》里有一辑就叫《秋坟之花》，我那虚拟的精神家园就叫"秋园"，大概就是受了这孔林的影响。相传孔子逝后，"弟子各以四方奇木来植，故多异树，鲁人世世代代无能名者"。所以孔林不但是一片坟地，它还是一座植物园哩。其中有一种"楷树"，所谓"楷模"、"楷书"，实源于此。空水的"秋园书屋"里有一尊极具感染力的笑僧雕像，即用孔林楷木雕成。

　　这孔林着实非同一般。从名誉上说，它叫"至圣林"；从资历上说，它有两千多年；从规模上说，它有三千亩之大！ 什么"枯藤"呵"老树"呵这里多着呢（"昏鸦"有没有就不记得了），只不过与之相配的换成了碑林墓海罢了。高树蔽日，深草没人。大坟如山丘，小坟如馒头。还有石人石马石虎石羊肃立两旁的"神道"，标志着墓主生前死后的威仪。那些颇具书法水准的墓志，默默讲述着或繁或简的人生。而最能让我们读书之人生出敬仰怀念之情的，除了孔子墓外，大概就是孔尚任之墓。这孔尚任，正是那《桃花扇》的作者。

　　我为什么喜欢在孔林散步？因为我在这里仿佛进入了时间隧道，遨游阴阳，咀嚼古今；更因为我在这里可以尽情地友鸦雀而亲草木。

　　想当年我们三个好友，在孔林里一片野花点缀的草地上，人手一瓶啤酒、一根香肠，语默自由，坐卧随便，临鬼神而坦然——比那刘关张在桃园时如何？唉，如今是"天之涯，地之角，知交半零落"了。
