学骑自行车

　　大概是一九七九年吧，二哥花六十块钱从"委托门市"买了一两老掉牙的自行车。它那牙盘上的牙齿已磨得像棘针一般了，一不小心就掉链子。过了几天二哥就不想要了，打算再去卖掉。我说那你就让我先用它来学学骑车子吧。

　　开始，我姐姐在后面扶着货架，让我往前蹬。别别扭扭地，走得艰苦卓绝。村里的人们看见，大概都说道："使劲扶住把，别来回乱扭；往前面看路，脚使劲蹬！"我照着人们的教导努力了一通，汗都出来了，还是不行。

　　我干脆就一个人自学。弄了一根很有韧性的榆木棍子，绑在后面的货架上，然后专找宽敞的地方去学。

　　先学"标"车子。就是身体在车子的左侧，左脚踩着车蹬子，右脚断断续续地蹬着地前进，直到能靠着惯性滑行很远，就算会"标" （那时我们那里从来就没见到过轮子小点的自行车，大人小孩全都是这个学法）。如果歪倒了，就有那长长的榆木棍子撑地保护。因此我可以有恃无恐地练――苦练加巧练。一天就会"标"了。

　　会"标"了之后，我就渐渐地试着骗起右腿，不断保持着新的平衡，骑上去，蹬踏板……要往左歪了，就让手里的车把随着往左扭……要往右歪了，就让手里的车把随着往右扭……扭来扭去，倒来倒去……终于可以扭来扭去而有较长的时间不倒了！这样又练了一天，就能够像蛇一样地前进到很远的地方了。

　　在确定基本上没有危险了之后，撤掉了绑在后面货架上的榆木棍子，就算是学会了。

　　后来我儿子学骑自行车的时候，条件就比我好多了。他用的是"好孩子"牌的带着两个小辅助轮子的童车（除这两个辅助轮子之外，跟成人自行车几乎一样――连价钱都是一样）。他很小的时候，就能骑着它跑。可是他长到上小学了，还是不能不依赖那两个小辅助轮子。

　　我干脆给他把那两个小辅助轮子卸掉了。反正他静止地骑在上面的时候，两脚能撑到地面了――所以就不必像我当年那样绑上榆木棍子。我扶了他一阵子也不怎么管用，于是就放手让他自己去练，只讲些"随车转把"之类的心得体会供他参考。

　　结果他练了几天之后，就能跟着我各骑各的自行车去东游西逛了。

　　"随车转把"， 而不是"以把转车"，这也仅仅是初级阶段的骑车要领。到中级阶段，就应该是"以身带车"。能做到"以身带车"，便可以两手清闲而使转自如，无为而无不为矣！然而，还有一个高级阶段――"以心许车"。达到这"以心许车"的境界，便可做到"以神遇而不以目视"。

　　空水现在的骑车水平没法跟人家玩专业的相比，只是也能够于双手离休、闭目神视之际，于清净无物之平坦公路上，安然直行数十米而已。条件所限，不敢更远，见笑见笑！另注：本人自娱自乐，不表演，不参赛，说说玩玩就立即撤退。
