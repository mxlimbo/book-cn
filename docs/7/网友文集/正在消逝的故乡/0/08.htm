蟋蟀

　　如果说蝉是"夏虫"，那么，蟋蟀就是当之无愧的"秋虫"了。

　　夏天里如果听不见蝉鸣，我是有点想的；但是满耳里都是蝉鸣的时候，也是有点烦的。我和蝉的感情，确实并不像它的叫声那样热烈。

　　我和蟋蟀的交情就不一般了。离秋天还早着呢，我就在等待和倾听着了。每当在夏的末尾，在一个人的夜里，在呀呀蝉鸣的背景中，终于听到了蟋蟀先锋队的清音，那感觉，就像"久旱逢甘雨，他乡遇故知"。这个时候，我就欣喜地知道，炎夏即将过去，清秋就要来临了。

　　秋天里不能没有蟋蟀，大概就像寂寞的山谷里不能没有小溪，幽静的湖水里不能没有小鱼，雪后的早晨不能没有麻雀，梨花开的日子里不能没有微雨。

　　童年，我在老家的园子里翻开瓦砾，捉蟋蟀在瓶中，看它的鸣声是怎样发出。青年，我在远离家乡的校园外面，听着满坡遍野的蟋蟀，心中澎湃着莫名的惆怅。

　　如今，人生的路差不多走过一半了吧？童年的兴趣和青年的激情，消耗得还剩下多少呢？在我把许许多多的东西看淡了之后，秋天和蟋蟀这两样永远美好的东西，在我心里越来越显得重要。

　　蟋蟀叫起来了，就要立秋了。

　　我想我就是一只"秋虫"。在属于我的季节里叫几声，是我的最为喜悦和充实的事情。

　　我想我最好是死在若干年后的一个秋天，独自一个人，在萋萋秋草中，在满坡遍野的蟋蟀声中，"如秋叶之静美"地，于瞬间消灭了踪影，不让任何人见到。
