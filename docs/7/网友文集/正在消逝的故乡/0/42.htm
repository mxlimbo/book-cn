五月二十一日

　　五月二十一日的夜里，我就想把我这一天的生活写下来。可是几天过去了，却只有一个在电脑上打出的题目。天天都想着要写，还是没写下一个字的正文。每次坐在电脑前，觉得这一天确实该写，但是耳朵里灌满了电脑小风扇的噪声，写不下去。两年来我已经习惯于直接在电脑上写稿儿了，可偏偏这次就不行了。那么就首先诉诸白纸黑字吧。人家说：“文学高于生活。”而我愿意说成：“文学美于生活，生活重于文学。”这次我还是写写我一天的流水帐吧。也许明天就死了呢？也好留个纪念。

　　2000年5月21日，周日，农历四月十八。
　　清晨上网，先看邮件。网友说谢谢我昨天寄给她的两张图片（一张月夜，一张秋野）。然后看些关于昨天陈水扁“就职演说”的新闻。
　　六点，习惯性地让电脑播放一盘总称为《珍藏一生》的萨克斯音乐——那里面有我憧憬的生活（我想我死的时候应该用它代替哀乐，并且不要听到人哭）。同时让洗衣机工作着，同时我去做饭。

　　一边吃饭，一边看了《实话实说》的尾巴。说的是抑郁症患者康复的事情。我媳妇说，她就愿意看崔永元主持。她不爱听我的音乐。

　　吃完早饭，孩子跟着他妈的摩托车走了。学校“减负”了，孩子有更多的时间去玩。他不是跟着他妈去玩，是去找他姨家的表哥和舅家的表弟。他妈总是忙着经营她们的小店，说“哪有闲工夫去玩啊”。她还嫌我星期六星期天怎么不去监考挣钱。我说，一是不愿去夹在考官和考生中间受罪，二是不愿拿好不容易用五天工作换来的自由时间再去换钱。没有周末的自由生活，我就会失掉很多活着的意思。

　　他们走了，我试着阅读从网上下载的《Lady Chatterley's Lover》。我有一个理想，就是能欣赏原汁原味的外文作品。借着电子词典读了两段，我就被吸引住了。我大概不是那种凭着“惊人的毅力”“刻苦”读书的人，所以特意找这本据说是“很有争议”的书来读。我从十八岁以前就喜欢看“有争议”的作品，还好，至今尚未因此而走上犯罪的道路。

　　看得眼睛累了（在电脑上看书就这点不好），休息。想起去年代发工资的工商银行给的“牡丹灵通卡”连一次还没用呢（我根本就不会用），就想找出来试用一下。也不知放到哪里去了，就到处乱翻，瞎找。

　　一个大木箱。是我上高中的时候，请父亲给我做来盛书的。当时我大概有一百多本书，爱惜得很。那时我大概已经把《大破金龙阵》、《枪挑小梁王》、《小白旗的风波》、《海湾夺船》、《七把叉》等一批心爱的连环画册赠送给比我小七岁的侄子了——都早已经被弄没了吧？现在我有书架、书橱，不用在大木箱里放书了。现在的大木箱里，放着汶河滩上拣来的石子儿，小学的笔记本和初中的图画本，高中的“自由夹”和“冬梅”手表，大学时爱听的一些磁带，近二十年来所收的信件的信封和我十八岁时所写的惨遭退回的幼稚而真诚的求爱信，第一次到南方第一次吃芒果留下的大果仁（从那粘乎乎、毛乎乎的果核里扒出来的）……还有父母所给的一块银元，大哥自学无线电做成的第一只小收音机，二哥在“小西屋”里吹过的笛子，妻子在家做姑娘时做的绣花鞋垫……翻了一阵，只找到了那“牡丹灵通卡”的密码，却没有找到那卡。

　　还有几个纸箱子。里面装着些档案袋、大信封、塑料袋等等。档案袋里，有我的诗稿、字稿、画稿、印稿，还有历年的计划、总结。大信封里，有近二十年来所收的贺年卡片。塑料袋里，有我已经到过的旅游景点的导游图、风景图片、门票，等等。我有一个理想，就是漫游全球。另外，纸箱子里还有大哥和二哥早年所写的家信、历届爱好文学的学生自办的小刊物、儿子小时候的涂鸦之作，等等。这样又翻了一通，还是没有找到那卡。不过也不急着去找了。我已经沉浸在被那些“历史文物”勾起的复杂的情绪之中了。

　　最后我希望渺茫地又翻了一遍早已翻过两三遍的写字台大抽屉，竟然就找到了那张“牡丹灵通卡”——原来它就被掩在一摞稿纸的底部。

　　晾上已经被洗衣机洗完的衣物。中午喝了一瓶啤酒。下午又看了些文章之后，就想出去透一透气。我已经有好几个星期没到野外去了。

　　骑上自行车——五年前我特意为逛野地而买的变速山地车。已换过若干车轴、若干轮胎、若干脚蹬子了，现在还很壮实。
　　路过“洋子洗脚屋”、“南妹洗头房”、“梦娇按摩中心”……就到了透气的郊外。

　　从外面回望这座崭新的城市，远处有用蓝玻璃包装的大厦，和一些仿欧建筑的红的、黄的尖顶；近处有白楼、绿树，和它们在宽阔而明静的人工河中的倒影。刚刚还在它里面烦那些钢筋水泥和人车嘈杂的我，这样从郊外望着它，又觉出它的可爱了。

　　但更可爱的，还是田野。

　　风大起来了。水库里的水和洼地里半人高的柽柳都被吹得此起彼伏、前呼后拥的，不由得让人思念大海。一撮儿一撮儿在水田里劳动的，几乎全是女人——这从她们那一般是比较鲜艳的衣着就可以看出。她们那样的衣着如果是在城市的大街上会显得“土气”甚至“俗气”，可是偏偏在这阡陌纵横的田间，水光树影的所在，就美得恰到好处了。你也可以设想，把一个长袜、短裙、高鞋子、眼睛像时装店的石膏模特一般的“都市靓姐”赶下水田，会有怎样的感觉？看起来，人是在可爱的地方才显得可爱。

　　沿着一条水渠走去。渠里现在正流着从远处引来的黄河之水。堤上的白杨树耸立在我的身边，哗哗地鼓着它们手掌形或心形的叶子，让我仰望着它们，想起小时候被高高的大人们围在中间的情形。而柳树则像平易近人的胖婶子们，慈眉善目地歪着“披毛”头对着我笑，还伸出她们温柔的手来，摸我的光头，擦我的鼻涕，把冒着热气、滴着汤汁的肉骨头，优美地吮了一下，再塞给我啃。我想起几年以前，也是这样的一条水渠、一些可亲可爱的白杨树和柳树，眼睁睁被我们的城市一口一口地吞噬掉了。

　　遥望一片湖泊，来到湖泊的边上。这是一处人工开挖的湖泊。原来的轮廓大致如中国地图，是准备建一处公园的。多亏了他们的公园计划泡了汤，才使这里荒凉起来，渐渐具有了自然天趣。湖水日夜拍岸，终于融化了挖土机的爪痕；芦草遇水自生，远近修短，聚散合宜，抒发着大地的情怀。轻歌曼舞的燕鸥和若有所思的野鸭，显然会认同我对这片湖泊的观点。如果我欣赏水里那些像篱笆一样静立着的小渔网，说它们像迷宫一样曲曲折折地美丽，如果我欣赏坡上坡下的那些缓缓流动的绵羊，却不说它们具有“温良恭俭让”的品德和“甘愿为君衣，甘愿为君食”的精神，可爱的鱼们羊们，能不能理解我呢？我想我眼中的鱼羊，正如鱼羊眼中的小虾小虫和黄花青草，谁又比谁怎样呢？

　　看着想着，想着走着。走到一座废弃的花园，和花园旁边的一片废墟。瓦砾堆上蓬勃着蒿草，蒿草丛中扑闪着蝴蝶，蝴蝶飞过了残垣断壁，趴在刻着“勝一村”的大黑石碑上沉思。原来，这里曾经是一群石油创业者的家园。那时他们大概是“男油女稻”，每天回到那些极为普通的红砖平房和小小的院落里，擦洗身上的油污或者泥巴。现在，却不知他们又搬到哪里去了？在一座被我认为是“没有历史”的城市的郊外，有这样一两处叫我沉下心来遥想过去未来的废墟，实为难得。

　　但是在废墟一旁的公路，又在准备拓宽了。一大片十年来滋养我灵魂的野地，又将被那拓宽后的公路圈进去，变成“经济开发区”了。也许到了明年，我的野地——我的灵魂保护区，又要被姐姐妹妹的“洗脚屋”、“洗头房”、“按摩中心”代替了吧？没准哪一天，到了我没有野地可逛的时候，就只好逛到那“屋”、那“房”、那“中心”去了。

　　逛着想着，想着看着。天色已晚了，肚子已饿了。于是收起我的野心，乖乖地滚回城市，去准备以“饱满的热情”和“主人翁”的姿态，享受“公仆”们“全心全意”的服务。

　　在小摊上吃过晚饭，去小店里见过媳妇，从街头捉住儿子，踏上回家的征途。

　　路过工商银行的自动取款机，便摸出那张“牡丹灵通卡”来，去一试这“高科技”的效果。一试，失败；再试，又败；三试，成功！子曰：“失败乃成功之母，自学乃成才之路。”插卡，密码，选“提款”按钮，要100元钱，100元就从下边的缝隙里送出来了。“高科技”这厮端的是方便快捷啊！“哦！哦！哦！哦！芝麻开门！芝麻开门……”

　　忽然想到，这个月又该给远方的老父母寄些钱去了。

　　回到家里打开电视，电视上说，我所敬仰的赵朴初老人，已于今天下午十七时零分逝世了（享年九十三岁）。而另一个台上说，青岛的很多情侣纷纷于今天举行“海滩婚礼”，觉得“5.21”像“我爱你”一样吉祥。我想如果赵老听到这则新闻，以他的佛家胸怀，定然欢喜。

2000 05 24-27空水
