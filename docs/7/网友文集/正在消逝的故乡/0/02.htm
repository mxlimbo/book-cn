小的时候

　　我于一九六六年春天出生在安丘城西的一户农民家里。有两个哥哥，四个姐姐。

　　记得很小的时候，我吃过一些饼干。饼干被父亲锁在一个小壁橱里，不许别人偷吃。我喜欢把饼干泡在茶碗里，用匙子舀着吃。我还清楚地记得，在同意（一个小伙伴的乳名）家的墙外的碾子旁边，我端着茶碗吃着泡的饼干，看几个姐姐帮母亲推碾子……稍大一些，就和大人们一起吃煎饼、窝头、地瓜干、地瓜叶、南瓜汤、榆钱、榆叶、萋萋菜、白杨花、刺槐叶，等等一些极其复杂的东西，白面只有在节日里才能吃到。我们家人多锅大，吃饭的时候，热闹极了，那真是——"就着饥困吃，不香也香。"

　　其实我也干活。拔草，拾柴，拾粪，放猪，等等。秋天落叶的时节，手里捏着长长的大铁针，弯身轻轻地一刺，一个金黄的大白杨叶，或者一个黄绿斑驳的更大的梧桐叶，便被拣起来了。到长长的大铁针上穿满了厚厚的一叠，便将它们一抹，抹到针尾拖着的长长的一条线上……最后，拖着长长的一大串落叶回家了，母亲便微笑着，说等她再去赶集的时候，就买个好东西给我。

　　当然我更爱玩。我和我的小伙伴们，玩出各种为今天的幼儿园的"小皇帝"们所望尘莫及的花样。我们从屋檐的瓦下，摸出黄嘴角的小麻雀，到河滩的草地上去扑蚂蚱喂它们。母亲帮我用树皮和粗线做成"网包"，把麻雀养在里面。有时候不能及时捉到虫子，就嚼些煎饼、"卷子"什么的，用手指抿到它的嘴里。在我顾不上的时候，母亲就替我来喂。到它们会飞的时候，我们可以放它们飞到树上，再吹口哨唤它们下来，落在我们的头上。有一次我养的一只麻雀被父亲摔死（不知为什么要拿它出气），我伤心极了（可以用"悲愤"去形容）。我含着泪把它埋在屋后的园子里，默默发誓要为它报仇。我们还经常去"喷鱼"——在罐头瓶里放上一点干粮末儿（要是有肉骨头就更好了！），放进池塘，作好标记，等鱼进去。耐心地等一会儿，悄悄地摸下去，一下子捂住瓶口！托上来，那就有三条五条，也许还可能是十条八条的，活蹦乱撞的"麦穗儿"鱼了！我们还把小铁桶用钉子打上密密麻麻的小孔，放进去一些点燃的木片，用长绳拴着，于（农历）八月十五的晚上，在村子后面的空地里，抡得火球呼呼作响，舞成美丽的圆圈和"8"字。

　　夏天的晚上，时常在大门外的空地上铺一领草席、两条麻袋，一家人便一起乘凉。摇着蒲扇，数着星星，扒"瞎话"（童话），讲《聊斋》。但是一到秋收的时候，全家人一个个急得手忙脚乱，累得头昏眼花，我就玩不成了。我那《秋坟之花》里有一首诗说过这事，题目也是《小的时候》。但是我最爱秋天，我尤其爱在秋夜里听蟋蟀的叫声。

　　当我体会到秋天的深沉和成熟的时候，就不再是小的时候了！

　　　　　　　　　　　　　　　　　1989年10月18日写
1999年08月23日输
