我的2000年元旦

　　2000年元旦零时，我在看电视：江主席按了一下电钮，点燃了"中华世纪坛"的大火炬。在前面的节目里，一群化装成原始人的青年在山顶洞里钻木取火，然后由另一群青年把火种送到"中华世纪坛"。不知为什么，没看见用那山顶洞的火种来点燃大火炬，而是用了电钮。

　　在黎明到来之前，我被自己定好的闹钟叫醒，起来看电视。泰山的日出直播，由于雾大，没有看成。看到了海南三亚的日出。曙光中，突然从沙滩下面钻出各种人扮的动物——这个舞跳得挺好。在这期间，我踩在浴缸的沿上，从朝东的窗口远眺了几次，希望能看到我们东营的日出，也因为雾大而没有看到。

　　接着，看了天安门广场的升旗直播。

　　吃过早饭，泡上要洗的衣服，想想今天怎么过。昨天夜里刚下过毛毛细雨，今天仍是阴天，加上我那不轻不重的感冒（它要是也怕"千年虫"就好了）也跟着跨了世纪（其实我更同意新世纪从2001年开始），弄得我无心出去了，于是就接着看电视吧。

　　我看一阵大人的，儿子看一阵小孩的，合伙看一阵通俗的，这就到了下午。

　　在家里实在憋闷，决定出去透透气。

　　我骑自行车，儿子坐在后面，就出去漫无目的地瞎逛。

　　往西穿过"商贸城"，往南来到广利河。我问儿子，是回城还是下乡？他说愿意下乡。于是就故意放着较好的路不走，偏偏去走那河堤上从未走过的小路。这时就让儿子步行，我推着车子，走走停停，边看边聊。河里有迷宫样的渔网，堤上有雨水冲蚀的沟壑。枯草静静的，空气冷冷的。

　　走到一片树林。全部都长着一种我和儿子都不认识的树。这种树的枝头大都挑着一个紫红色的穗子。有点像高粱的穗子，但是小得多。扒开以后，里面是许许多多紧裹着鲜红的绒毛的子粒。从树林高低参差、自由散漫的样子来看，大概是种子和树根都能繁殖的，而且很能适应这里的盐碱土质。儿子发现了树林里的青苔——他不知道是什么，我就让他想那首古诗："空山不见人，但闻人语响。返景入深林，复照青苔上。"儿子摸了摸那些青苔说："原来我不懂诗意。"

　　再往前走，河堤上的小路分了岔儿：一股儿拐向村子，一股下到河滩里去了。于是我们就下到了河滩。河滩上不像河堤上那样干爽，泥巴渐渐地在我们的鞋上和车轮上越沾越多。我就跟儿子说："你懂'泥泞'这个词吗？"他说："早知道了。就是这样。"

　　走到一座旧桥的北头，又能回到河堤上来了。

　　这桥的北边，紧连着一个古老的村庄"沙营"。这几年，先富起来的村民大都到河南面离公路近的地方去盖了新房，形成了"沙营新村"。依然留守在老村的，好像有的是为了养牛放羊方便一些，有的是因为在这里有他的荷塘和鱼池，有的也许就是只因为留恋？这边的老村看起来，虽然是满目的断壁残垣，但是自有一种你在新村里（更不用说在城里）找不到的感觉。土圈里懒洋洋的老母猪，浅浅的泥水里抖着羽毛的鸭子，那少有人走的土道上巧妙点缀着的油亮的羊屎蛋儿……

　　在村头的土坡上，我们远远地看见一只鹭鸶（也许是灰鹤？）模样的水鸟。它静静地呆在浅水里，很长时间才转动一下脖子，或者挪动一下腿脚。

　　在村子西边的大白杨树下，我问儿子还记不记得小时候在这里坐在土坎上吃饼干、数羊的情景，他不记得了。

　　天快黑了，我们又望了两眼村子西面那些青青的麦田，就赶紧上自行车往回走。

　　先去农贸市场，买了十个馒头（两元），二斤四两芹菜（一元两角），一斤三两藕（一元三角），一包五香花生米（两元五角），六个熟鸡爪子（四元五角）。然后就回家，继续过日子。

　　新与旧，大与小，豪迈与尴尬，宏图伟业与琐碎生活，都涌上我的心头，搅成一锅粥。本来不平衡，逛一逛，写一写，搞得自己比较平衡。

　　说句心里话，我也想伟大。可是我必须，面对琐碎的生活。

　　说句实在话，我也想潇洒。可是我的翅膀，常常沾着泥巴。
