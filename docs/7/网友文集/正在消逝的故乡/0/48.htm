寻根

　　"妈哎，你当初要是嫁到个大城市多好啊，那俺们也不用憋在这个小地方受罪了！"

　　"爸哎，你当初要是趁早去参加了革命多好啊，那俺们现在也都成了高干子弟了！"

　　"爸爸妈妈哎，你们要是在千禧零点生我多好啊，那我就用不着这样费劲地想办法出名了！"

　　呸！要是那样的话，生出的会是你吗？

　　据说，最早的古人，只当那生人的种子就在女人肚儿里，与男人是没有关系的。所以就崇拜女人。

　　后来的古人，又发现如果不用男人，女人也就生不出人来，于是又以为那生人的种子其实是在男人的肚儿里了。所以就崇拜男人。

　　直到最后，才知道这生人的事情，既不像是女人的肚儿里有一粒种子，等着男人来浇点水它就发芽，也不像是男人把一粒种子种到女人的肚儿里，等着女人的好墒情使它开花结果。才知道这生人之事，其实是男女双方的对等合作。就像红酒和白酒兑成了一种新酒，就像一条鱼朝着一只蚌吐了一口沙尘，一群沙尘纷纷挤在这蚌的门口拼命地抢购彩票，忽然一个愣头小子中了大奖（目前记录，每届获将者一般仅只一名，最多不过数名），于是就进去参与珍珠生产去了。

　　念天地之悠悠，叹人海之茫茫，哪些鱼遇见哪些蚌，哪条鱼遇见哪只蚌，哪一口沙尘吐向哪一届摸奖，哪一粒沙尘得以中奖入蚌，入蚌之后又终于能做成珍珠，嗨，你说那不是偶然中的偶然吗？

　　你如果抱着史书感叹，要是秦始皇不修长城……要是岳武穆抗金成功……我们的国家……我们的民族……——如果真是那样，那大概就会引发一系列多米诺骨牌一般的事件，最终影响到你这个个体生命在这个世界的诞生。

　　所以呢，从国家前途、民族命运的角度，我们不妨对历史评头品足；若仅仅从个人生命的角度，我们也许还要感谢历史的巧合。 没有一系列历史的巧合，世界就不会有我。

　　巧合的生命，唯一的生命，岂不值得好好珍惜，好好纪念么？

　　我坚信，生命是一种资源。一个人的生命资源，是不可再生的，是一次性的。消费完了，就永远没有了！若不是为了很大很大的事情，我是决不会随便献出生命的。所以呢，我可以对诸如抢劫财物的好汉们承诺：您尽管放心，我不会在没有把握赢您的情况下，为了一百块钱就和您拼命。但是您也想想，您值不值得为了一百块钱，来冒触犯刑法抓去坐牢的风险，甚至是被我正当防卫打死活该的风险？我也可以对诸如欺骗压迫黎民百姓的狗官们承诺，我也不会在没有把握赢您的情况下，为了一个职称就和您拼命。但是您也想想，人们的觉悟可并不全都像我这样啊！

　　什么才是那值得献出生命的很重要很重要的东西呢？每个人的心里，可能大不相同。您要问我，无可奉告。这是我的第一隐私，是生命绝密！

　　常听人说，"我们中华民族，是炎黄子孙"。不对，肯定有许多少数民族兄弟是不同意的。这个说法，有失对各民族兄弟的尊重。就是这"炎黄"，也嫌混帐。据说我们这个孙氏出于姬姓，而姬姓属于黄帝部落。我宁可相信黄帝是我祖先的领导，却不敢相信怎么那么巧黄帝就正好是我的祖先。还有一说，是"黄皮肤的旗帜上，写着中华"。呸！我怎么眼看着人家日本、朝鲜、越南、柬埔寨什么什么的，也是黄皮肤呢？既然叫"旗帜"，那就得跟人家有所区别吧？这倒好，不但没跟人家区别开来，反而连一些自家的兄弟，算不算"中华民族"都难说了！那什么动不动就"黑眼睛黑头发……龙的传人"，也是只看皮毛，还强奸人的祖先。我宁肯相信，我的祖先的祖先是两头像大猩猩那样的动物。

　　胡尧著的《中国姓氏寻根》里说："孙氏有三支。一支出于姬姓。卫康叔的八世孙卫武公有个儿子叫公子惠孙，惠孙的孙子名乙，字武仲，他以祖父的字命氏，就是孙氏。另一支出于芈姓。楚君蚡冒有个儿子叫蒍章，蒍章的儿子吕臣以父名命氏。吕臣有个孙子叫蒍敖，字孙叔，是楚国著名的贤臣。……孙叔敖的孙子以他的字命氏，也叫孙氏。还有一支出于田氏。陈厉公的儿子陈完避难逃到齐国，改称田氏。田完的五世孙田桓子无宇有三个儿子……小儿子田书字子占，在齐国为大夫，因为伐莒有功，齐景公把他封在乐安，赐姓孙氏。田书的孙子就是著名兵法家孙武……"

　　哈！快快搬出我们的家谱来查查！不管我们这个孙氏是哪一支，来头都不小啊！如果是第一支，那来头就更大。你晓得卫康叔何许人也？周武王之弟，周文王之子是也！如此说来，我乃帝王之后也！

　　可惜，我们已经没有像孔家那样世系详明的家谱了。

　　明代洪武年间，我的先祖从枣强县（今属河北省）迁居山东青州。初有四兄弟龙、豹、虎、熊。我乃孙熊后裔。

　　是不是像传说的那样先被骗到了洪洞县，然后又被迫从老槐树底下出发来了山东，我不能确定。但也可以猜想，大概并不是他们自己要迁的，而是被朱元章强迫着迁的。

　　关于"先被骗到了洪洞县"，据说是这样的：朱元璋怕人们安土重迁，就先放出风说，只有洪洞县的可以不迁。待到人们投亲告友地躲到了洪洞，朱皇帝一道圣旨，叫军队包围起来，押送他们迁移。

　　这样想来，出自楚国的孙氏和出自齐国的孙氏，与我有瓜葛的可能性极小。我这个孙氏，还是出自卫国的可能性大些。

　　唉，焉知我的哪一位祖先，不是某个姓孙的乞丐在路上拣到的一个孤儿？

　　寻得个什么劲呢？

　　我，就是一个我呀！
