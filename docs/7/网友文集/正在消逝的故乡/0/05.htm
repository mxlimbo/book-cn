池塘

　　俺老家管池塘叫"湾"。

　　我小时候，俺庄儿里有三个湾："大湾"、"小湾"、"下园湾"。后来我知道那些湾都是庄里的人挖出来的。

　　那时候全庄里的房子只有"地主家"的是青砖瓦房。俺庄里就只这一家成分是地主的，所以一说"地主家"说的就是他家。"地主家"的青砖瓦房确实比人家的房子也高也大，光那砌着青石的基础就比人家高一大块。几户富农家的房子，有较高的砖基，上面是墼垒的墙，麦秆子屋顶。上中农家的房子，砖基比不上富农家的厚，上面也是墼垒的墙，麦秆子屋顶。广大贫下中农家的房子，只有薄薄的砖基，上面是土打（夯筑）的墙，也是麦秆子屋顶。

　　庄里的人家盖房必先"拓墼"，就是用麦穰和泥，铲在墼模子里成型，晒干备用。"拓墼"就得挖土，挖来挖去就成了大坑，灌满雨水就成了池塘。那个"大湾"一般情况下都还有水，另外两个湾则是季节性的：没水的时候，就又成了"拓墼"的场子。

　　我当然很喜欢那些湾里有水。在盛夏多雨的时节，三个湾里常常都长满了仿佛跟雨点一样多的绿绿的浮萍。那些背着金线、绿地儿、黑斑的青蛙，或者英俊地蹲在草窠儿里目光炯炯地抿嘴儿微笑，或者舒舒坦坦地张开了四肢趴在水面上翘起圆润的下颌。那些我总是无法看清其面目的蛤蟆藏在密密麻麻的苘棵子里，喂——哇——喂——哇——地高声合唱着不带丝毫忸怩的公开的情歌。

　　"大湾"里水多的时候可以没人。曾经有好几个小孩掉进去了却从来没有人被淹死。——看来这就是湾边有路的好处吧？所以尽管经常受到大人的警告和训斥，我们还是经常泡在里面玩水打闹。我就是通过反复地在这个"大湾"里扑通才学会游泳的。俺老家管游泳叫"浮水"。——这么一说，我倒想起"白毛浮绿水"来了。那里面还真是有鸭有鹅的哪。俺家的鹅鸭也在其中。俺家就在岸上住哪。"大湾"北面的高台上就是俺家的那口老井和那棵大柿子树。俺还从湾里摸上鸭蛋来哪。

　　冬天"大湾"里结了厚冰的时候，我们就上去玩。滑冰，打陀螺，刨冻鱼。我们滑冰基本上有两种滑法：一是把小木排底下弄上铁条的滑脚，坐上去用钎子撑；一是穿着鞋子在冰上猛跑，凭着惯性滑过去。俺老家管打陀螺叫"打懒老婆"，反映出旧社会的性别歧视的余毒。那刨冻鱼，是说有些鱼因为水面封冻而憋死了，漂在冰下，我们就刨冰取鱼。

后来因为俺庄儿里修路，把那"小湾"和"下园湾"都填没了，"大湾"也被那路分得一边只剩了一小汪儿。再后来，那条路被修成了安丘城"县府路"（后来改称"人民路"，改得好）的一部分，从此"大湾"也消失了。

　　后来我离开家乡，有一天听到朱明英唱"池塘边的榕树上知了在声声地叫着夏天……"，我就想起"大湾"边的柳树。那些柳树斜着长在那岸上，柔长的柳条探到静静的水面，引得鱼儿来咬，弄出一圈圈的波晕……
