正在消逝的故乡

　　我们的村子，原来是安丘县城西城门外的第一个村庄，所以叫做庄头。小时候翻弄父亲的抽屉，发现一张日军占领时期的"良民证"（大小犹如现在的名片），那上面写的是"西关乡庄头村"。据"地名委员会"的考证，庄头村始建于明朝洪武年间。而据老人传说，庄头村住户只有姓尹的是"坐地户"，别的都是搬迁过来的。大部分说是从"洪洞老槐树底下"迁来的。而我们Ｘ姓家族有明确记载，是从枣强县迁来的。

　　我记事的时候，早已没有了城门和城墙，只有一条护城河还在。护城河上有一座小石板桥（据说在日占时期只有吊桥），桥的东边是"城里"，桥的西边就是庄头。到我上小学的时候，"城里大队"和"庄头大队"合作，在小桥的北边建了一座可过汽车的混凝土拱桥，就叫做"大桥"。

　　在护城河上只有小桥的时候，河里经常有水。虽然那河水常带着泥土的颜色，但我们还是很喜欢在里面玩耍。记得在我们上一年级的时候，下午天气很热，张老师就带着我们去下河。女的一群在上游，男的一群在下游，玩水。也有不老实的男生侵犯到女生的地盘，泼辣的女生就骂他是"流氓"。张老师的家就在这护城河的岸上。我们从河里出来，就一块儿去张老师家里看花。她丈夫在水瓮里养着一些个头很大的中国金鱼，还有一棵一人多高的夹竹桃，开得热热闹闹。我们听说她丈夫就是她当年的老师，觉得非常美妙。后来我当了老师也总想找一个学生爱人（可恨未能如愿！），大概就是受了我这位启蒙老师的传染。后来我听我父亲说，有个媒人差一点就把这位才貌双全的张老师介绍给他！我怀疑我父亲是不是单相思——我母亲说那是他自己想的好事。

　　"庄头庄儿，东西长。"我们这个村子，有沟有塘，有坡有崖，七拐八扭，错落参差，整个一个自由散漫的大格局。也许正是因为如此，童年的我，在其中觉得特有意思。下雨的时候，有许多地段的路就变成了小河。我们就一丝不挂地玩水打闹。脚底下踩着细细软软的沙子，舒服极了。路沟里的沙子来自ＸＸ家和ＸＸ家用来当作围墙的沙丘，沙丘的沙子由西北风从田野里刮来，田野的沙子来源于村子西北面的汶河沙滩，汶河的水和沙子据我父亲说是来源于沂山。

　　那汶河的水可是清得不得了。绝对是清澈见底。过路的人不管是大人小孩，谁渴了，趴下就喝。那个甘爽！什么矿泉水，什么纯净水，没一种能比得上这汶河水的！——然而，那已是过去的事了！

　　现在的庄头村，周围包裹着安丘市（撤县设市了）的人民银行、检察院……样子快与城市里的小区差不多了。布局犹如棋盘，串门须记排号。盖楼的人家多了，买楼的人家也多了。……仿佛已是小康了。但是，小轿车碾过一堆堆垃圾，大摩托溅起一洼洼污水……

汶河的水已脏了，护城河的水已臭了。树林被砍了，沙子被卖了，我的故乡梦快完了。
