精神的瘦子

——从一则减肥产品广告说起

　　看到近期有一则广告讲到某减肥产品说，用了该产品，就可以大吃大喝（也不会发胖）。不禁颇有感慨，从某种意义上说，肥胖是上帝对奢侈者的惩罚或忠告。而现在他们却可以无视于这种忠告，其目的是为了继续大吃大喝，如果说一开始是因为疏忽，那么在接受了上帝的忠告之后，就是故意了。

　　同时也还看到一则消息，现在肥胖已成为现在许多孩子和大人的一个问题。远的先不说，我在上海的朋友告诉我他的学生中，十个里面有九个是胖子，而在九个胖子里有八个是笨蛋。我国还是发展中国家，发达国家可能更厉害。

　　如果仅仅是上面的消息，我们也没有太过担心的，至少表明人们的生活水平是提高了。但是我们又知道中国还有许多人还在贫苦之中，在非洲，许多孩子在挨饿。而这个地球的另一半人却在减肥，其原因是大吃大喝。

　　这不由又让我想起一个笑话：一个胖子碰到一个瘦子，胖子为了取笑他说看到他人们还以为你那儿在闹饥荒，瘦子回敬说那么人们看到他就知道闹饥荒的理由了。

　　现在我已经笑不出来了，我希望肥胖的朋友，我不知道造成的原因是否因为过度奢侈，或者是减肥的朋友，其实节食才是上帝给我们的唯一办法，还有孩子的父母，如果我们处在幸福之中，让孩子看看地球上另一半的朋友，不要让他们成为身体的胖子同时，却成为精神的瘦子。
