怕 生

　　我回到家的第二天，也就是说我在家休息了一夜，我是傍晚时候到家的，我从上海回家乘的是头天下午的船，经过将近二十四小时的航行，到温州是下午二三点钟，转了两回车，到家已是六七点了，当我把表从包里掏出来，我惊奇地发现表针又开始在动了，那副不紧不慢的样子，并不象是积蓄了一年之久突然爆发出来的，只像一个在外的游子回到久违的家乡，悠闲自得，一切都按班就部 。我对准家里的挂钟把表校正好，中午的时候，我再去校对了一下，表并没有走差。现在已经一个多星期过去了，表总共才走慢三分钟，这里面还包括我的视差呢！

　　一年前，我考上上海的一所大学，在我离家之前，父亲把手腕上的表摘下来交给我。那是一块双狮表，老式的，显得很旧色了，但是朴素、淡雅，线条很柔和，丰腴而没有臃肿的感觉。和另外两块同样的双狮表和一块女式的，曾经分别戴在父亲、大哥、二哥和大嫂的手上，后来大嫂的给了二姐，二哥的表给了三哥，重新买了一块，而父亲则把表交给了我，它们记载了父亲、大哥们创业的一段历史--

　　那是在十多年前，家庭承包责任制和改革开放刚开始不久，大哥、二哥都已长大成人，成为父亲的得力助手，父亲在老式农民中无疑是最新式的：精明能干，勇于创新。在一次买卖中，一下子赚了几千块，这在当时是一笔不小的数目，父亲一口气买了四块双狮表，父亲在实际的业务交往中，认识到没有时间的观念是不行的。在以后的几年内，我家很快成为村里、乡里最富和最有影响的家庭之一。但是后来因为种种原因，进入九十年代以来，以父亲为中心的事业开始走下坡路了。其中一个很重要的原因是父亲不识字，从来没有也不能到温州以外的地方，这块表也一直伴随着父亲，没有离开温州一步。同时父亲毕竟是六十开外的人了，父亲曾经是走在时代前头的人，他看起来的年龄要比实际年龄大得多。当父亲把这只陪伴了他十余个春秋的双狮表交给我时，是否意味着父亲将最终退出商场而回到一个淳朴农民的本来身份了呢？不过当时的我却没有想得这么多。

　　当我到了上海，过了一段时间，却发现表不走了。在这期间我也试过各种办法，可是一点用处都没有，而我又懒得拿去修理，所以一直放着。到今年放假的时候带回家，不想一回到家竟又能走了。

　　春节里偶尔与朋友谈起这件事，还没有等我说完，朋友就说了："这是因为地磁场的影响，从这里到上海地磁场发生了变化……地球是一个巨大的磁体--"

　　"这是很平常的物理常识，"他看到我无言的样子，得意地笑了，"亏你还是中学时物理老师的得意门生呢！"

　　我也笑了，不过我却不敢这么肯定，我还一直以为它是因为怕生的缘故呢！

　　 九八年于温故书堂
