久违了，太阳

　　一天，二天，四五天，一个星期，两个星期，一个月来，从阴历正月，阳历二月开始，雨一直在断断续续地下着，就象林黛玉的眼泪，要把男人的雄心溶解，溶进无休无止的雨水或泪水中。但是在这漫长的雨季里，我却时常感受到阳光的温暖，始终相信阴雨之后就会有太阳，也许还会因为久违的缘故，反而显得更加的明媚呢。

　　记忆里的一次雨季也特别长，雨量特别大，或者南方是一贯这样吧。那年我从浙江农业大学退学回家，刚好赶上了那个雨季。尽管我是协议退学，但是就象协议离婚，总有被抛弃的一方，当时我正被自己的雄心折磨得精神麻木，不觉得痛苦。只有等到了家里，感受到父亲的叹息母亲的悲哀时，我生命中的雨季开始了。

　　想一想大学生，在当时，不要说在农村，就是在城市都是父母骄傲的资本，现在却成了父兄的耻辱。因为我的自私和疏忽。而我带回来的除了一纸肆业证书竟是因为饮食没有节制造成的还有点严重的胃病。有时从半夜醒来，隔房传来父亲的咳嗽声，屋背雨打瓦片的声音也变得清晰，一下一下敲打着我自私而不负责任的灵魂。

　　那几个月的日子，一段时间我寄居在大姐家里，外面下着雨，而我整天呆在房间里，精神恍惚，不可终日。现在我已不知道自己是如何熬过来的，我只知道那个季节充满了雨水和泪水。

　　一直到有一天，我模糊记得开始还下着雨，到上午的时候转阴了。我心情也似乎平静下来，也许是几天之前一次老师的看望，跟我谈了一会话，我应该感激他，给了我许多帮助和有益的建议。而现在我却不能再从他那里得到有益的教诲，因为几个月前他因病过世了。

　　那天下午，我不知哪里来的兴致，和几个侄子走出家门，到了屋外的小山。田野里在连绵的春雨中爬满了青草，小树的树丫伸出嫩绿的叶芽，毕竟是春天了，只有象也在等待久违的太阳似的缺少鲜花，溪塘的水里栖息着大小的鸭群。

　　不知是见的鸭子还是哪个孩子提出讲故事，我给孩子们讲起"丑小鸭"来了。

　　"丑小鸭受到人们的轻视，有一天他离开家开始了流浪。"

　　（那时我居然没有讲到丑小鸭出走之后，家人是如何的担心焦急）

　　"经过了多少时间，经过了多少路程，丑小鸭到了森林的小湖畔，看到许多天鹅，他们看到丑小鸭尽情地欢呼，这时从湖水的倒影里，丑小鸭变成了白天鹅"

　　（丑小鸭变成白天鹅，这是遗传的因素，还是成长的过程。）

　　"阳光照耀着高山、森林、湖泊和自由的白天鹅。"

　　我也仿佛沉浸在丑小鸭沉浸在那种阳光般的喜悦之中。

　　这时，我身上也有点痒痒起来。

　　"叔叔，叔叔，太阳出来了。"

　　抬头看时，不知什么时候，太阳终于出来了，正照射在我们的身上。是的，天边还有许多乌云，但是太阳毕竟出来了，他将不会再消失。

　　第二天，果然是晴天的兆头，吃早饭的时候，我告诉父母自己出去找一份工作或者参加再一次的高考。

　　父母互相看了看，点了点头。这时我还注意到了母亲脸上偶尔露出一丝笑容，那明媚的笑容，不正象太阳。

　　久违了，太阳。

　　尽管脸上还隐藏着太多的哀悉，眼角还有不少的泪痕。但是太阳毕竟出来了，它将永远不会消失。
