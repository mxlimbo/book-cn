新杂文

　　杂文的出现是与文章同时的，但是杂文形成一种传统是晚唐的罗隐皮陆等人，那时也不叫杂文而叫小品文。罗的文章呱呱叫，但是就不能中举，一气之下不考了，写一些不平的文章。那是还没有出版业，他们的文章不能大规模的流传，所以统治者也不知道或者也管不了。到了宋代，宋代是给读书人官当的时代，文人也就不大写小品或者叫杂文。到了明朝，李贽是一个大家。他关于道学的批判和孔子的嘲讽，可以说是新思想，新文章。清朝大兴文字狱，文人自顾不暇，也就不大敢写杂文了。

　　杂文到了鲁迅才重新发扬光大。鲁迅之后大陆似乎中断了杂文的传统，这归功于共产党的文化愚民政策。但是在台湾出了两个大家，就是众所周知的柏杨先生和李敖先生。柏杨提倡崇洋但不媚外，李敖说我来剥蒋介石的皮。

　　现在是大陆重新拾起杂文传统的时候了，老实说在今天我们也会看到一些杂文，对现实不满或抨击的，但是往往只骂小鬼，不骂阎王的。就象在中国老百姓眼里，只有坏的官员，没有坏的皇帝，最多是昏君。所以大多的文章只是隔靴搔痒，总是不痛快。

　　因此我的这组文章名为《新杂文》，但是新者不新，也只不过是继承过去的传统。但是针对今天的中国，也可以算是一种新的声音吧。我只希望人们在过早对我下结论之前能完整的阅读它。
