运动会小（四）记

　　瑞安市第八中学（湖岭中学）第十五届校运动会，我恭逢其会，并且忝为径赛终点载判，克守职责之余，走笔记趣。阮日旺。十月

（一）

　　运动会是十三日上午七时开始，首先是开幕式。不知哪班进场时有四位同学分别捧着一个一尺见方的木牌。这让我产生一个可笑的联想，审判大会上的那些囚犯，前面挂着一块木牌，等待有人用朱笔打一个大大的红勾或叉。

　　"……《宪法》第X条……、《XX法》第X条……"校长的开幕词更象宣判席上法官的审判书："……根据《刑法》第几条第几款……"然后"判决如下……"。不知道这份绝不能在法院的宗卷里找到的判决书是谁写的。

（二）

　　在径赛场上，没有象田径那样"X号，下一个X号准备"的从容。随着一场枪响，六名不等的运动员一齐冲出，竟有万马奔腾之势。而其中一马当先的运动员更是人们注目的焦点。

　　但是也有这样一些运动员所受注目的程度决不亚于冠军。在比赛的过程中，好几位运动员跌倒了，但是打一个滚，爬起来又继续跑。可惜当时不允许我去问他们的姓名。其实姓名也并不重要，这不仅是哪一个人在跌打滚爬。我看到的是青春在跌打滚爬。

　　让青春跌打滚爬。

（三）

　　在运动场上，运动员们为了自己的成绩而奔跑。这时，场上也少不了许多忙碌的身影，其中就有一位骑着自行车来往奔波的连络员某老师。在他的自行车篮里放着一瓶矿泉水，累了渴了就喝一口。这时不免让我联想到一句话：吃的是草，挤出来的是牛奶。那么某老师呢？他喝的是水，挤出来又是什么？那是奉献和对运动员负责的守职尽责的精神。

（四）

　　当歌声在操场的一角响起时，意味着经过了一天半运动员们为之拼搏的运动会快要接近尾声了。各个运动员的成绩有好有坏，各个班级也喜忧参半，但是所有的运动员无一不表示出了青春的魅力，这是一次青春的盛会。

　　围在一个圈子唱歌的是高三某班的学生，他们也许更深刻地感受到这次运动会对他们意味着是什么，他们也想表达什么。于是他们想到了用他们的歌声。真的，我是被他们感染了，他们歌唱的不仅是某班，也不仅仅是这一次运动会，他们歌唱的是青春的骄傲和忧伤。
