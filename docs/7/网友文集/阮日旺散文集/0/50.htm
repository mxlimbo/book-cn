外 婆 偶 忆

　　外婆的娘家在四古山，大概既使不是大户，也会是小康人家吧（这从外婆临终所留下的少数器皿中略微得知），外公家原来也算较富裕，外婆嫁过来之后，生了四个孩子：大姨、母亲、大舅、二舅。但是就在小舅出世后几年，外公不幸过世了。那时最大的大姨也只有十几岁，大舅、二舅还不到十岁。可以想见，外公走后，家里留下外婆和孩子的那种从小康陷入困顿的境况。尽管那时会得到族人亲戚邻里的照顾，但是要把四个孩子拉扯成人，成家立业，的确不是一件容易的事，而外婆曾经是多么坚强和伟大的女性啊。

　　而在我的印象里，外婆却似乎与这种坚强无缘，她是这么的老（当然她年青过，因为岁月的流逝和生活的重负，她老了），这么的小，连走路都颤颤的，因为小脚的缘故。外婆有时牵着我的样子，我怀疑一旦我甩开手，摔倒的是我还会是外婆。

　　在我童年记忆里，我是外婆家的常客，最小的表弟，差我一岁，是我的玩伴。因为外婆家和我家只有几块田地和一条溪涧的相隔，望过去就可看到，我们叫"隔岸"。有时候我对着"隔岸"大声喊叫，这时外婆会颤颤地出来，然后我飞快地奔去。

　　在我奔到外婆家的时候，外婆已守在门口，这时她会从兜里掏出一块糖，作为欢迎的礼物。我不知道小时的我吃过多少回外婆给的糖，也不知道外婆是如何省下这些糖，我只知道每次吃糖的时候外婆脸上的笑容，也有糖那么甜。那时，我还想能一辈子都吃到外婆那香甜糖呢。

　　上了学之后（初中），去外婆家的机会少了，也还会偶尔吃到外婆留的糖。一天，母亲偶尔跟我说外婆说我长大，给我糖我也没要。我当时也不在意，不过说真的，我是没听到的，可能我正贪玩着。

　　在我上了高中之后，一年二次回家，去外婆家更少，一次回家，母亲说外婆不在了。外婆临终前说念书要紧，不让我回来。

　　现在外婆走了将十年了（我甚至不记得她的祭年祭日），大舅也在几年前的车祸中去世了，你的最小的孙子也已结婚有了孩子，我也大学后回到家乡教书，偶尔还会想起您。外婆，这一切您知道吗？

　　以后我也很少吃糖，也许是因为真的长大了的原故吧，即使偶尔吃到糖，也不觉得有那么的香甜了。
