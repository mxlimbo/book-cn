新木桶骑士

　　木桶骑士，我们已经知道，他在卡夫卡的小说里骑着木桶在一个寒冷的冬夜去一个商店乞讨一小块煤，因为遭到老板娘的拒绝，最后他飞向了，像一片鹅毛被一阵大风刮起来，飞向了又黑又冷的冬夜。

　　许多读者以为故事就此结束了，其实木桶骑士的传奇，还才刚刚开始，他在第二天的阳光里又重重地落在地球的某一端，中国的一个老镇和尚桥。关于和尚桥，有许多有趣的故事，而发生在乡村的关于木桶骑士的故事无疑是其中最有趣的之一。

　　他，木桶骑士，在和尚桥的桥头晒了一会儿太阳，心情慢慢复苏过来，他不知道那个冬夜自己是如何熬过来的，在他的记忆里那是一个又冷又长的冬夜，他曾经以为再没有比冬夜更长的了。现在他才发现比冬夜更长的是生命和桥头的阳光。这是值得高兴的，尽管离冬天结束还早，但是他还是从冬天的阳光里感受到了那种温暖，万物也似乎充满了生机。

　　他朝四周望了一下，这是一个乡村小镇。在冬天里，略显得有点萧瑟，但是充满宁静，很显然这种田园风光，是符合骑士身份的。他仿佛有种感觉，他是属于这个小镇的。他忽然感觉到缺少一点什么。真的，在这个时候，对于一个骑士，也是需要发生什么事情，还有小镇。这时，从桥头过来了一位姑娘，就像一个公主，她是乡村王国的公主，这是故事必不可少的开端。他身上的血液，充满浪漫和冒险的骑士气质，开始象桥下的流水一样流动起来。（田园式的风光，美丽善良的乡村姑娘，这往往构成了一个骑士的浪漫史。）

　　说到这里，我必须声明卡夫卡的《木桶骑士》是缺乏浪漫的。卡夫卡是现代派作家甚至迄今为止作家中最伟大的之一，是真正的弱者的天才，但是他的这个故事却不高明，也不动听，试想一下今天的人们只要打一个电话或者利用电脑网络就可以，谁还会在冬夜上街买煤气。如果说在冬夜到来之前还有谁上街，再如果他是骑士，那他唯一的可能就是为了追求爱情--爱情才是永恒的主题。

　　因此在接下去发生的就是木桶骑士在乡村的爱情故事。为了使我们的故事能够顺利地继续下去，我们还需要点什么，这也是木桶骑士需要的，他正寻找什么，他的木桶不见了，这是理所当然的，他本来骑着木桶是用来装煤用的，难道他能用木桶装爱情吗？我们又有谁见过或听说过一木桶的爱情呢？这时他发现了一辆高头大马的破自行车，这看起来是很不错的坐骑，就象堂吉克德坐下的--现在他开始坐在那辆破自行车上了。他眼望着前方，仿佛那里正有辉煌的战果在等着他。

　　木桶骑士在拥有他的坐骑，一辆破自行车之后，尽管它破损得很厉害，骑着的时候还哐哐作响，这难道就是我们的爱情故事需要的优美乐曲或者预示着这个爱情故事悲壮结局。木桶骑士尽管他又黑又瘦象个老头，骑在自行车上的确象一个骑士，当然他也正是一个名副其实的骑士。

　　这种骑士精神不仅表现在他的外观上，还表现在他思想里，当他第一眼看到那位姑娘，就是刚才一开始在桥头出现过的那位，他发现自己已经喜欢上了她。这叫一见钟情，你相信一见钟情吗？当然你并不相信，它并不存在，即使存在了也是没有，但是却反复地出现在骑士的罗曼史和当前最流行的爱情故事里，而一个骑士爱上一个姑娘，并且愿意终身为她效劳，这是骑士的传统所拥有的。何况我们的骑士还是一个未婚骑士，而且还不仅仅这样，他还是个勇敢的未婚骑士，这表现在他在以后的日子里采取的行动方式。

　　总而言之，他借鉴了所有骑士的成功和失败因素，决定每天早晚两次，跟在她的后面，当然是保护她，难道不是保护她，而是她来保护自己，这是需要坚定的信念和非凡的勇气的。当然除了爱情以外，还有许多伟大的事情等着他去做，但是现在爱情却是他最重要的最伟大事情之一。

　　我应该补充一点，那位姑娘也是值得木桶骑士这么做的，首先她是一位姑娘，其次她是一个年轻的姑娘，而最后她还是一位漂亮的年轻姑娘。但是她是怎么想的呢？很显然，她把他当作好色的登徒子之徒，因此，木桶骑士的骑士行为也成了登徒子之类的无耻行为。这让我想起了古诗《秦罗敷》中的"太守"的行为本身没有错，但是错在他是"史君自有妇"。当然秦罗敷也没有错，也永远不会错，"使君一何愚"，这已经足够了，这是对一个有妇之夫丑恶行径的鞭挞。其实她是不需要其他理由的，而不久之后，我们会看到我们的姑娘，她是有秦罗敷一样的美貌，一样的坚贞，聪明和善良。

　　当一天下午，那是一年四季里平常的一天。骑士象平常一样跟在姑娘的后面，那是一个艰难的历程，好几次他觉得那辆破自行车太沉重了，仿佛他载着的不是一百斤还不到的一个人的重量，而是一个人的生命或爱情，或者正因为爱情，生命才变得沉重，他差一点放弃了，但是我们知道他是一个勇敢的骑士，所以当他看到太阳正以悲壮的姿态坠下山去，这鼓舞了骑士，于是他追了上去。

　　"我是木桶骑士……"

　　"谁管你木桶还是饭桶……你怎能这么憨……"

　　乡村姑娘居然不知道鼎鼎大名的木桶骑士，这真是遗憾，而她的一句话或者这个中国方言字却道出了事情的本质，"憨"就是一颗勇敢的心。

　　很显然，她把他当作了史君一类的人物，她还说了什么，木桶骑士已经听不到了。

　　因为木桶骑士连同他的自行车，突然飞了起来，就像孙悟空被铁扇公主的铁扇一扇，向后飞去。

　　其实她再讲什么已经不重要，她拒绝了他，就像曾经老板娘拒绝给他一小块煤，这已经够了，真的，有时候一个女人拒绝一个男人，是不需要铁扇公主的那把铁扇的，她只要轻轻一句话，从她的口里吐出来，就像铁扇公主从口里吐出铁扇，然后轻轻一扇，就把孙悟空扇到了十万里之外，这是女人的骄傲，还是女人的悲剧。

　　天突然变暗也变冷了，又一个冬夜降临了，于是木桶骑士又一次飞向了就像卡夫卡笔下描述的又冷又黑的冬夜，永无止境，没有尽点……
