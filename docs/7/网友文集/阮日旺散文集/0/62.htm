生 命 走 廊

　　每天，在这个冬天搬进新居之后，我都会走一段不算很长，大约十米不到的走廊。我会放慢脚步，小心翼翼，因为走廊并不宽，两旁的桌子上放满了各种花草的坛子，有瓷碗的，算是高级的，有塑料的，有的就是破瓦破罐，里面放一把泥土，这是我搬进新居之后的邻居倪老人养花草的园地，倪浩忠老师，八十五岁高龄，在湖岭任教数十年，退休后以著书和花草自娱，我也仿佛怕惊动了生命的小精灵。可惜我对花草知识一窍不通，我总以为拈花弄草是传统文人雅士闲趣，在大学时专修过一年的植物学早还给了书本和老师。其中有各种颜色的花和叶子，但以绿色植物居多，我能说出准确无误的也许只有仙人掌，因为童年一段时间，家里养过。仙人掌，叶肉很厚，呈绿色，手掌形，上面有刺，生命力特别强，能在沙漠里生长。它们大小不一，大的有手掌那么大，小的只有指头甲大小，形状各异，有球形的，掌形的，给走廊增添了绿色的生机。

　　已经是冬天了，随着几次寒潮的袭来，窗外前几天还葱绿荫密的杉树树叶已落了一地，而当我打开门，绿色的走廊就映入眼帘，嫩绿的仙人掌还有其他植物，在冬天早晨的寒气里似乎还显得分外精神呢！

　　如果有阳光就更好。这不，阳光不是已经开始眷顾我们的走廊（恕我到来没几天，就把走廊当作自己，当然还有倪老师的了），那是一条朝南的走廊，从太阳刚刚升起不久到落山之前，都会有充足的阳光，即使是阴天难得一露的太阳，也会让走廊里充满着阳光的味道。冬日的阳光在别的地方似乎有些吝啬，但是在这里仿佛是对于这些生命的眷恋吧，太阳就像一个从不吝啬勤于眷顾的客人，而对于这走廊，这些生命，还有什么比阳光更好的礼物呢？

　　这时候，老人会出来摆弄花草，或者在走廊里坐下来，晒晒太阳，就象面对着一个老朋友。如果我没事的话，我会陪老人聊聊，这是我所乐意的。于是我也知道了倪老师的一些事，几十年如一日的从事科研教学，对不同学科的广博兴趣。我也许应该表示应有尊敬，对于他的博学，单单通晓四国语言，已足够让我佩服了，这时决定向倪老师学习外语，而让我更加赞叹的是尽管倪老师八十五高龄，身体明显佝偻，也许从这一点上，我们才说他是一个老人，但是耳聪目明，这在他这个年龄是很少有的，这恐怕不是我能学到的，倪老师现在还看书读报和钢笔小楷写作。在与我的交谈中，没有丝毫的障碍，还会偶尔在脸上嘴边露出妩媚的笑容，那是我们会从二八女孩或孩子们脸上所看的，但是却给我们更大生命的感动，就象一枝老树的枯枝上开出一朵美丽的小花。

　　不知怎的，我不由想到几天前走廊里一截枯白的仙人掌，上面长出片肉芽，嫩嫩的，绿绿的，那是全新的生命，就象我们每一个人都会衰老，但是有一天我们希望自己枯老的身躯上也会长出一段嫩绿的永恒的生命。
