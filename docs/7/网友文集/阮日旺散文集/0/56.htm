烂苹果的趣事

　　多少年后，我不知道自己竟会一下子买了十斤烂苹果，一个人一次买十斤苹果不值得奇怪，因为他可以吃上几天或一个星期，但是十斤烂苹果，就多少有点让人吃惊了。

　　那天当我经过桥头，看到一边坐着两位老年乞丐和一个疯子--我称之为"和尚桥八仙"之一的永达，附近有一家水果摊，地上一大箩烂苹果，一中年妇女和一位十五六岁的女孩在向来往的人兜卖。不知怎的，我突然停下来问："烂苹果多少钱一斤。"女孩说五毛钱。这时中年妇女过来说苹果进来已经八角，现在烂掉了卖五毛。这时突然从我的口里冒出一句："我买十斤。"当然我的同情心还不足以让我可怜卖水果的母女，是因为我触动的童年的情结--烂苹果情结--再说一想到只要花五元钱，如果我再压价的话，兴许四元就可以，它还不到我一天工资的十分之一，买下这一大袋东西（烂苹果），我不禁要差得露出得意的笑容了。

　　记得我小时候，家里还刚刚摆脱饥饿。家里孩子很多，很少有吃什么糖果之类的零食。但是也是有例外的时候，如在水果的旺季或者父母卖了家里农副产品，从集市上回来。这时，我们会分到一个烂苹果，烂苹果的价格只有新鲜的三分之一，但只要把烂孔剔掉，有三分之二可以吃，而且烂苹果吃起来还特别香甜呢！烂苹果往往成为我们几个孩子兴趣的焦点。新鲜的苹果是很少有的，既使有，也舍不得吃，母亲总是挑一二个放起来，母亲大概觉得那是了不起奖品，不能随便给我们，再说几个孩子，一两个不够分，她会在我们考试取得高分或完成了什么任务时才交给我们作为奖赏。但是母亲的记性又特别不好--一个即使记性最好的人，要记住七个孩子的大小琐事，她的记性有一天也会不好的--所以往往把苹果一放在抽屉里，几天过去就忘掉了。有几次，等到我们在房间里闻到了一股酒香，然后我们找到了那个苹果，在抽屉里已烂掉了大半--在以后，我常常想起烂苹果，我们在它还完好的时候，没有好好利用，甚至有时会忽略它的存在，当一天它烂掉了，发出一股醉人的酒香。这是苹果的好处，它不会产生别的污染，这时我们才发现原来在我们的身边（我们的生活中），曾经有过一个顶大顶大的苹果--这个苹果有时候会是一个人才、一种情感。

　　我不知道自己是如何把这一大袋烂苹果扛回学校宿舍的。同事们问我带了什么好东西。当我说是烂苹果时，他们问我是不是疯了。我只是笑笑，真的，有时候我也不知道自己究竟疯了没有。

　　当我坐下来吃我的烂果，吃到第四个的时候，我感觉到肚子有点发胀，而它还不到剩下的十分之一。于是我开始发愁了，我不能把这些烂苹果留到以后的几天，那样他会很快的烂掉。最后我突然想到一个绝妙的主意，于是提起剩下的烂苹果，冲出了学校，向桥头奔去。

　　几分钟之后，经过桥头的人们--里面没有我的学生，这时还在上课时间，他们会看到桥头的一道奇妙的风景：两个老乞丐，两个傻子（一个永达，一个是我），正在悠然自得地一边把（苹果）烂的地方用小刀剔掉，一边吃着烂苹果……

　　这个时候如果让我的朋友看到这副风景和我脸上自得的笑容，还以为我正在做一件十分了不起的趣事呢。
