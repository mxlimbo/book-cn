一碗饭饮的故事

　　这个故事发生在解放前，我祖父辈的一个农民身上。在旧社会，农民一般没有什么财产，一间木屋，往往是住了祖孙几代的，几个子女和几亩田地，到了实在揭不开锅时卖田地，到最后只好卖儿卖女的，当然这里面也有少数富裕的农民，一段时间人们叫富农，他们与地主不同，地主是继承的，父亲是地主，儿子也是地主，而富农是通过自己的勤劳、节俭、精明能干而致富的。富农也可能购买少量的田地当有的农民遇上了特别困难的情况，如天灾人祸，欠债之类的原因，而出卖自己的田地。

　　农民甲现在就到了出卖田地的艰难时刻，为了这件事，他转辗反侧，一夜没有睡好：家里已经好几顿没有食粮了，几个子女嗷嗷待哺，妻子生病花了一笔钱刚愈，自己去了一家亲戚，本来想借一点救济，反被疏落了一顿，空手回来，左邻右舍，也度日艰难，不好再借。倒是向乙家借过几回，有放出风来，他家想买田。想到卖田，甲的心揪紧了，田地可是农民的命根子啊！来年如果没有田地，就只能租种人家的。可是妻儿，想到今天的窘迫情境……

　　第二天清早，甲起来的时候，看了看很早饿醒，但是没有东西下锅而盼望白天不要到来，躺在床上饥肠辘辘，同样转辗反侧的妻儿，狠了狠心，向乙家走去。

　　甲到了乙家，只说了句"我要卖田"，乙说："你想卖田"，甲"嗯"了一声，接下去他简直听不见对方在说什么，他只觉得一阵头昏，眼睛在发花……

　　这时，一阵饭香仿佛从天际飘来，飘到甲的鼻窍，沁入到他的心脾，这里怎么会有饭香，在他记忆里，家里已很长时间没有开饭了，他睁开眼睛一看，原来主人家正做饭，刚好煮开了锅，而上面浮着一层白色的泡沫，透出一股饭香。

　　甲出于饥饿的本能，向女主人讨了一饭饮。

　　他从女主人手里接过滚烫的饭饮，还在冒着泡沫，他忙着呷了一大口，又连续几口下去，甲突然变得清醒，知道他正在做的事，它是如此重要，头也不昏了，眼睛亮了。

　　现在已经是五月了，再过一个月左右，稻子就可以收割了，对了，刚才经过田里的时候，怎样没注意谷穗，有的已经发白，可以先捋下一些煮成汤，可以熬一段时间，等到收割，可以再熬一年半载，来年，来年也许还会很艰难，但是总可以熬过去的。

　　故事的结局，不用说大家也知道，甲并没有把田地卖掉。"后来呢？"父亲给我讲这个故事的时候，当时的还情形已记不清了，我也这样发问，后来甲一家渡过了最艰难的时刻。对于一个农民来说，田地是最珍贵的东西，一个人最珍贵的东西，是不应该轻易放弃的，一旦放弃，就很难重新拥有，大到一个国家，国土是最珍贵的东西，以李鸿章的代表的清政府在马条约中放弃了祖国宝岛台湾，从而使台湾成为祖国统一的最艰难的问题。

　　一个人的一生总是在艰难之中度过，但是最艰难的时刻，却不会很多、很久。如果我们在最艰难的时刻都能坚持自己的东西，也不轻易放弃自己最珍贵的东西，那么它将永远不会失去。而有时候改变一个人的命运的竟是一些微不足道的东西，譬如一碗钣饮，那么我们在帮助一个人做出重大选择时，是否会吝啬一碗饭饮呢？或者我们自己即使是在最艰难的情况下，也能保持清醒的头脑，从而做出正确的决择。这大概是这个故事给的启发吧！

　　 一九九九年八月

　　注：

　　饭饮：温州方言，是指用锅烧饭，刚烧开时浮在上面一层糊状物。
