花为谁开

　　傍晚的时候，我正闲得发愁，晚上没有辅导学生，电脑因为26 号出的病毒，虽然朋友帮助弄好了，又听说8号可能有病毒，不敢打开。这时一位朋友打电话来说好久没见面了，想过来聊聊。我说好的，我去校门口等你。

　　过了几分钟，朋友来了，我们一边在校园里漫步，一边随便聊点什么，朋友是做行政工作的，我做老师，平时也可以说是为谁辛苦为谁忙。我们原来都是学校的学生，大概是关心母校，朋友问了许多学校的情况，我一一回答，开玩笑说领导来了解情况来了，我说这也是职业病吧。朋友笑着打我。

　　这时，我们走过一处草地和花圃，在这样的春末夏初，雨后的傍晚，前两天下了几阵不小的雨，今天开始有点放晴，所以花草显得格外清新。学生们在教室里自习。于是诺大的校园只有我们两人，不，还有一旁的碧草绿叶红花。

　　朋友突然问：此花为谁而开。我说：你还记得贝克莱大主教的话吗，他说你不看此花时，此花与你同归于寂，你一看此花，此花便一时明亮起来。所以现在只有我们在看花，欣赏此花，那么此花就是为我们而开。

　　朋友笑了笑，说我还没忘记书上的东西。然后问一位老师，他不在办公室，于是给他打了电话，是有事相托。

　　我突然有点失落：原来他并不是为我而来的，是因为有事才来的。也不由想起苏轼的话："唯少闲人如你我两人耳"，看来自己是愧对东坡老了。不过朋友能够想到我，也算是一分难得的友情了，那么我且享受这一分友情吧。

　　回去的时候，又经过刚才的花圃，在暮色中更添了几分幽雅。也许这花并非是为我而开，或者她也并不是为谁而开，有句诗叫"花开花落自有时"。那么我们只是在她花开还没有落的时候，在她青春美丽的时候，遇上了她，于是在我们的心里也留下了她的一份美丽，我们又何必去追问她是为谁而开的呢？
