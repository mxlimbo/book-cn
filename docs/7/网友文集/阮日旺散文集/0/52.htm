百 货 小 姐 的 故 事

（或愚人节的故事）

　　那天是愚人节。

　　至少对于故事里的人是如此，那天是他们的节日，我应该为他们祝福。

　　我不知道这个故事是什么时候诞生的，但是我想不会早于五十年，因为里面讲的是百货小姐和阿三的传奇故事，而百货公司在中国城乡兴盛还是近几十年的事，当然也不会是近二十年的，这时百货的地位已经动摇。我也不知道最初诞生这个故事的人是谁，如果有这么一个人的话，如果他还看到我通过别人的口里听到并把它转诉出来的那种模样，也许会有点啼笑皆非呢。我还不知道他最初讲述这个故事时是否会带有讽刺的意味，不过即使有那么一点，我想也可以作善意的理解。

　　在城里的百货公司有一位年轻漂亮又骄傲的小姐，尽管年轻漂亮骄傲是她不可剥夺的品质，但是为了使我们的文字不显得拖沓，在下文我们暂时去旧这几个词而称之为百货小姐，当然聪明的读者时刻也不能忘记她是一个年轻漂亮而骄傲的百货小姐。

　　今天的人们在林立的商店、超市中冷落了百货，当然还有百货小姐，而二十年前，人们对百货甚至抱有神圣的观念。人们进城一趟，最重要的事就是看百货和百货小姐，从百货出来就象现在从香港回来，而百货小奶俨然就是今天的香港小姐，昔日的公主，不，今天的公主，其实每个时代都有自己的公主，也都有公主和王子的故事，而阿三就是童话里的青蛙王子。

　　青蛙王子来自田间，不农村。我不知道故事里为什么经常会有阿三出现，而且多少会带有一点岐视的意味：癞头阿三，上海叫人瘪三。不过在我们的故事里，尽管阿三也有个毛病，就是从小鼻涕拉遢，用现代医学的概念是慢性鼻炎，而且阿三还是个穷小子，但是接下去 幸运就发生在他身上。

　　一天阿三第一次进城，当然不例外去看百货和百货小姐。正好他的鼻炎犯了，他怕百货小姐看到赶紧退了出来。这时他遇到了几个城里的朋友。

　　他们不知怎的看到阿三，突然来了一个念头，要捉弄阿三或百货小姐。原来他们都试图追求过百货小姐，一个人家里有许多钱，一个有很多房子，另外一个是大学毕业，还有几家父亲是科长。但是一个个还没有上阵就败下阵来，于是原来的情敌结成了失恋阵线同盟。他们在痛定思痛之后，决定出钱出力帮阿三上阵。

　　那一天一定是愚人节。

　　讲故事的人没有告诉我，也不能告诉我故事的原因。阿哥在转述这个故事的时候忘了日期，那时他也压根儿不会知道有4月1日的愚人节。但是阿三竟有追求百货小姐的勇气，是爱情或是无知战胜了缺点。于是我相信那天一定是愚人节，而接下去发生的事都表明这一点。

　　那天，当阿三再次出现在百货柜台前的时候，当然已经换上了全新的西装，衬衫和领带，那是一套兰绿西装，阿三的大学生朋友从"Blue is blue"（兰色是忧郁），得出兰色容易打动人，不过兰绿西装的阿三还会让人不恰当地联想到癞蛤蜡或青蛙，不王子，当然下面王子和公主的游戏就开始了。

　　阿三开始向百货小姐要了一张手帕，我不知道阿三围绕百货柜台兜了十圈还是二十圈以后，突然从阿三的口里冒出一句"一张手帕。"这是成功的第一步（？）。

　　"一元钱，不用找了。"

　　"哼，又是一个装阔的。"百货小姐才懒得理他。

　　可是过了两分钟没到，阿三又返回来，再要了一张手帕，同样说了一句"不用找了。"

　　然而刚过了一会，阿三又买了一张手帕。同样的事情在一个小时之内重复了n次。这时百货小姐才注意到不知什么时候地上、门内，柜台边抛满了不同颜色的高级手帕，要知道拥有这样一方手帕，可是妇女们一年过节时才会侈一下的行为。

　　当阿三第n次重复做的事：买一条高级手帕，擤一把鼻涕，把它扔掉，重买一条。百货小姐以前也见过许多大款，但是没有见过象阿三这样浪费，慷慨大方的。

　　如果说百货小姐因此以为阿三是哪一个华侨老爸的儿子或者高干子弟，那简直是对百货小姐和爱情最大亵渎。

　　于是百货小姐现在居然和阿三说话，最后还答应第二天到阿三家里看。

　　不过我们千万不要误解或学习阿三的行为，如果是在今天，阿三这样乱抛手帕是不文明的和不符合市民规则的行为，肯定会受到门卫的制止或大叔大婶的罚款，何况现在人们早用纸巾了。我时常怀疑阿三不是因为西装、手帕，故事里告诉我们的原因而是他的勇气和鼻涕感动了公主，就象现在一些勇士可以用一封缠绵的情书或热情的诗歌，但是他们往往不会有童话里的王子或阿三那么幸运。

　　第二天，阿三带着百货小姐来到向朋友借来的一个套房。

　　不久，百货小姐和阿三结了婚，这时才发现阿三原来是个穷光蛋。

　　故事的结局是这样的，百货小姐在羞愧之余，决定相夫教子，后来阿三在妻子的帮助下考上了大学。就象童话里的公主和王子，从此过着幸福的生活。

　　其实百货小姐是无须羞愧的，我常常想她的选择是正确的。因为一个人拥有的东西，房屋、金钱、权力等等，也正是随时可能失去的，而一无所有的人，才能够从无到有，也会更加珍惜。

　　那个愚人节过去了，百货小姐和阿三的故事也完满结束了，但是来年还会有愚人节，还会有愚人，当然也就少不了有新的"百货小姐"和阿三或阿七的故事上演，而我们在期待着。
