中国愚人节--除夕夜

　　春节是中国人民最注重的传统节日之一，而给我留下最深印象的，现在回忆起来余味无穷又不无感慨的，不是大年初一，是前一天的除夕之夜。大多数农村和农民家里，过去是没有电视观看春节晚会的，但即使是在有了电视的今天，农村的青年男女，大人孩子也不耐烦坐在电视机前，他们有着自己的方式，度过一年一度的除夕夜。

　　每年到除夕前两天，即使是平时车马冷落的小庄，也会异常地热闹忙碌起来，除夕这一天，村里的人更是突然不知从地底里钻出来似的成倍增加，出去做生意的，做工的，除了极小数实在脱不开身，都要回来和家人团聚，有的是一家人在外回家过年的，过了年又要全家搬走的。在西方，家有两个概念：一是成员的家FAMILY，一是乡土的家HOME。而在中国，这二者是紧密相连合为一体的。家成为国人的主要情结之一，没有家就象无根的游萍。一个在外的游子总是要在除夕夜来临之前漂回家。

　　随着白天的过去，当人们吃过了年夜饭从家里走到乡村的街上，当鞭炮声在庄里此起彼伏，当"吱""嘣"地在空中绽放成火树银花里，当街上的青年男女这里一伙，那里一群：那是除夕之夜到来了。

　　除夕夜也许是一年中唯一没有禁忌的。大人们可以在辛苦一年勤俭积攒下来的钱大肆霍挥，吃酒的，赌钱的，这夜可不会有不识趣的穿制服的人出现，也许在某一个赌场里就有他们大肆吆喊的声音，当然已经脱掉了穿了将近一年的制服。老人们则例行放几个鞭竹，据说这原来是一个隐居在山里的人给自己壮胆避邪的。孩子们放鞭炮是为了好玩，青年人则是为了吸引异性的注意：谁的鞭炮放得响，谁的烟花放得更高更响更亮，吸引异性的目光也就越多。这仅仅是前奏。

　　从这个时候到午夜，是青年男女的黄金时间，是他们盼望了一天，不，一年，对某些人甚至于有终身意义的时刻。呆在电视机前吗？不，那简直是浪费时间，看别人联欢。除夕夜才不愿这样。除夕夜是属于农村青年男女的。他们的歌有点跑调，也不会跳舞，说着十分笨拙的笑话，但他们是真正狂欢的，是今夜的主人。

　　开始的时候异性伙伴是分开，泾渭分明的。不久就开始了喧动，有几个伙伴也许是赌输了钱想捞本，不，捞人：或者赌赢了想再赢一场。于是把目光转向街上路边随地可见俯拾即是的年青姑娘，捕捉她们随时可能抛来的一个媚眼。年轻的心在除夕之夜尽情荡漾。有许多佳偶或怨偶就是在这样的荡漾中产生的。

　　我的一位有点羞涩的朋友在前年的除夕夜看上一位姑娘，那个夜晚我们一起追逐女孩的踪迹。我承认自己在情感方面不是早熟而是迟熟的，尽管在智力上我表现出早熟，所以现在回想起来又好气又好笑的是好几次无意识中我充当了这样的角色。在除夕夜这样的事情是不受到指责也并不算过分的。这是为情人，不，愚人准备的节日。现在我的朋友已经成功地成为妻子的丈夫儿子的父亲了。

　　当然并不是所有人只要想成为愚人都会幸运地成功的。又有谁不愿意找一个机会让自己成为这样的愚人呢？有时候甚至有跟人家走了半个村庄几里路而把人跟丢了的。而当他正着急的时候，也许蓦然回首，那人却在灯火珊阑处呢。

　　这个时候也会有不愉快的事的，青年男女会因争风吃醋而发生争吵甚至斗殴的。幸好这只不过是愚人节小小的插曲。

　　这样的时光很快过去接近午夜的时候，原来为了各自的目标走散的伙伴重新聚合，然后开始吃封岁酒，这是乡间的习俗，不同于一家人吃的年夜饭，是几个儿时的伙伴一起吃的。这一次一般是一醉方休。不知是追到了女孩还是追丢了女孩，也不知喝空了几缸红酒还几瓶白干，最后从愚人变成醉人。

　　第二天醒来之后把昨天夜里的事忘光，开始新的一年，而年轻的心期待下一次能在除夕夜成为愚人。
