似 曾 相 识

《不再相识》姐妹篇

　　我本来想为师友立传，而偶一不经意下笔，竟是一全无深交之村女。事后捧读，百感交集，啼笑全非。阮日旺记。

　　村里有一个女孩，她叫梁燕。在村里相邻年龄的女孩子中是最漂亮的之一。所以每次我回家时，我在外地读书，只在假期回家，总有朋友跟我说起她的事，当然真实的少，夸张的多。因为年轻的朋友聚在一起不谈论异性是很少见的事。不过她倒因此成了我认识的不多的几个女孩子之一。但是我是不能说我们是相识的，我们没有说过一句话（我甚至还不知道她是否也认识我），当然我并不是没有与她相识的机会，但是因为我天生的羞涩，我常常错过了那些机会。

　　一年，我在假期回家，照例先到县城，再转车到村里，从县城上车之后，我就一直沉浸在对乡村生活的依恋之中，当我从客车的颠跛中惊醒过来，我发现在左边几排前面多了一个女孩子，头微靠着窗，只能看到半边的脸，"这不是梁燕吗？"当我再一次抬起头时，她脸转过来，果然是她，不知什么时候上车的。我低下头，我感觉到她也看到了我。接下去的时间里，我又看了她几眼，有时是有意的，有时则是无意的。她也似乎看了我几眼，不过当我抬起头的时候，她的目光又很快岔开了。

　　客车驶进了村里，因为我要在外村下车，我们村子分成里、外两村，我却一直不知道它们的界限到底在哪里，所以提前提了行李包，到了门边。车停住了，我不自觉地回过头去，看到她也正望着我。这时，我还注意到她的嘴唇动了动，后来嘴唇停止了运动，而把运动传递给了嘴角--

　　"下车啦！"耳边响起司机催促的声音。我跳下车，拿下行李放在地上。门关上了，客车开走了。

　　我站在了原地，我挽起行李包。--

　　我不禁想：她原来也是认识我的。
