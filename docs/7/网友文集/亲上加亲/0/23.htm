二十二

　　刘淑汀离开后，病房里只剩下黄俊士和刘智韬。两个人面面相觑，好半天找不着共同的话题。黄俊士揣摩不透这刘智韬的葫芦里究竟卖的是什么药。是要向自己说明他拒绝与葛瑞丝合作的理由以减轻良心上的不安，还是惮于这次的遭遇很有可能会加深自己和他女儿的恋情而乘人之危落井下石呢？无论怎么想，黄俊士都希望刘智韬赶快离开病房，好让自己眼不见心不烦，心不烦则伤不痛。

　　这时，病房内被高高安置的彩电正演播着洛杉矶熊猫电视台的中国大陆新闻节目。播音员的声音衬着迭化的德国自然风景，娓娓动听，娟娟可亲：“我国将在上海浦东开发区建造首条磁悬浮列车示范运营线。据介绍，从浦东国际机场至浦东龙阳路地铁站的这条磁悬浮列车运输线，全长约35公里，建成后将成为浦东国际机场与上海市区紧密相连的快速通道。专家们指出，磁悬浮高速列车的时速可达500公里以上，是当今世界上最快的地面客运交通工具……”

　　刘智韬看着电视，借以调剂一下尴尬的气氛，顺其自然地问：“小黄，你对上海……熟悉吗？”

　　“熟悉。”黄俊士不太情愿地答道。

　　“我记得你曾经说过你的家乡在温州。是不是温州离上海很近啊？”

　　“是的，不远。”

　　“那……你们家有亲戚在上海吗？”

　　“我们家本来是在上海，文革时逃难到了温州。”

　　“哦，你爸爸妈妈身体还好吧？”

　　“还行。我妈身子骨还算硬朗。退休后还在城市中心开发建设指挥部干着呢。”

　　“虽说儿行千里母担忧。”刘智韬感喟道：“其实，做父亲的，同样牵挂儿女啊。你怎么就不提提你父亲呢？”

　　“我父亲？我父亲没有给我留下任何印象。”黄俊士见刘智韬听不太明白，补充说：“他……早没了。”说着不免伤感，咳嗽了几声。

　　“对不起。”刘智韬端上水，说：“这么说，你妈挺不容易的。”

　　“是的，她为我父亲守了一辈子。”黄俊士清楚地记得这是刘智韬自那天酒店举杯后头一回跟他这么和气地谈话。这种气氛来得是这么的突然，竟会是旋踵间事，不免使他心中顿生疑窦。

　　“我在台湾有个好朋友，他就是在文革期间从大陆逃难过去的。你记不记得你父亲的名字？”

　　黄俊士一听这个，似乎明白了刘智韬的意思，说：“当然记得。他叫……刘晨海。”

　　听到刘晨海三个字，刘智韬石破天惊，当即抓住黄俊士的手，半天说不出一句话：“孩子……”

　　这一声竟把黄俊士弄得莫名其妙：“你是……”

　　“你的爸爸……他还……还活着啊，孩子。”刘智韬声音颤抖，老泪纵横。

　　“怎……怎么……你是说你……”黄俊士几乎不相信自己的耳朵。

　　刘智韬抹去泪水，说：“我真……真为我的……好朋友……高兴啊。”

　　黄俊士刚才差点以为刘智韬就是他爸。他想，幸亏没有乱认：“我爸……他，他真地还在吗？”

　　“在。”

　　“在哪里？”

　　“在台湾。”

　　“您能找到他吗？”

　　“我已经……好长时间没跟他联络了，不过，这事，我义不容辞。”

　　谢谢您，刘总。请您，一定帮我找到他。”黄俊士握住刘智韬的手，感激涕零。

　　“哦，这么多年了……我想……最好……让他们先通个电话。你……你有你妈妈的电话？”

“有。”黄俊士拿笔拿纸。“我给您写一个。”

　　这时，张克杰踏着纷乱的脚步找到了病房。他一见刘智韬，先是一怔，然后吃吃地把车钥匙还给了黄俊士并询问了一下治疗的情况，以示关心。刘智韬称自己要去趟洗手间，走出了病房。一会儿，张克杰也从病房出来，怯怯懦懦地来到了站在走廊一头的刘智韬的面前：“刘总。”

“　　你做事怎么这样没头没脑。”刘智韬皱着眉头说。“你用他是第一错，扣留他的车是错上加错。”

　　“扣车是戴安娜的主意。”

　　“她不知道，你还不知道吗？”

　　“可是那辆VAN是她的，我说什么也没有用啊。”

　　“嗨，你们呀，竟给我添乱。”刘智韬感喟道：“那边上午给我打来电话，说你们还欠着人家的钱，有这回事？”

　　“我这两天忙得鸡飞狗跳。”张克杰讨好地说：“我马上去邮局把钱汇过去。”

　　“做生意，信誉第一。别把台湾那帮人给得罪了，免得今后被动。”

　　“下不为例。”张克杰咽了一下，说：“刘总，有件事想跟您商量一下。”

　　“什么事？”

　　“刚才我在这门口跟您女儿碰上，她提出赔偿医疗费的事，您看怎么办？”张克杰笨拙地说。

　　刘智韬思忖后说：“幸好这孩子没动手术，而且他好像已经买了医疗保险，所以估计医疗费不会很高。你回去跟戴安娜商量一下吧。依我看那，就权当是花钱买个教训吧。至于其他费用呢，我来处理。”

　　接受了刘智韬的面授机宜，张克杰行色匆匆地离开了仁爱医院办他该办的事去了。刘智韬依然站立在那里，透过封闭式的巨型窗玻璃悠然地眺望远处，若有所思。

　　据《纽约时报》报道：在美国，人们租看极其暴露的色情影带，一年要花费四十亿美元。一般激情的，八亿美元。在网际网络上，招摇过市的就有六万多个色情网站，约两千一百万人平均每个月至少光顾一次这样的网站。而根据旅馆业的资料统计，至少一半投宿的成人旅客愿意花钱看这种影片，如饥似渴，乐此不疲。

　　随着先锋影视科技的更新与发展，DVD登台亮相，其逼真的画面，更对色情业的滋生和蔓延起到了推波助澜的作用。在美国本土上，有许多

　　DVD影像制作单位，但他们的制作费却高得令人乍舌，令许多跃跃欲试的中国人望而却步，不得不舍近求远去台湾制作。虽然延长了周期，却也节省了成本。

　　刘智韬和张克杰近来交头接耳，神出鬼没，就是在做这种生意。刘智韬认识台湾的制作商，通过他的关系，既给台湾拉了生意，又为张克杰开源节流。扮演了一个掮客的角色，好不乐哉。而且，他还另外得到两点实惠：一是喜门登大酒店的色情影带在洛杉矶的宾馆中，版本最多、翻新最快，借此招徕了不少新客和回头客，；二是张克杰通过他大陆的关系，不断地给刘智韬介绍客人。当然，张克杰也互惠互利，除做生意处，还得到了刘智韬的特批，免去了长年住宿的费用。应该说，这住宿费也是一笔不小的开支。

　　尽管昨天晚上已经给周明生打了个电话请了个假，刘淑汀知道，突然的请假会给周明生带来诸多的不便。今天，旅行社那几个人一定是忙得热火朝天团团转。所以，她在大酒店吃过中饭之后，并没有听从老爸的嘱咐找一个房间打盹小憩，而是趁这会儿黄俊士正在休息，驱车到了旅行社。

　　大概是因为中午，旅行社并没有忙得不可开交。刘淑汀进门第一眼就见到了一张新面孔。这姑娘二十好几，长得眉清目秀，玲珑乖巧，是周明生从一家叫什么凯撒的旅行社给挖过来的。今天头一天试用，业务能力看来不成问题。可正是因为能力强了，开价也跟着水涨船高。周明生迫于无奈，只好答应。但他私底下对刘淑汀依依不舍地说，希望刘淑汀能够奇迹般地回到他的身边。说着，他颤颤巍巍地塞给刘淑汀一个红包，饱经沧桑的眼角忍不住地滴下了几颗伤感的老泪。

　　刘淑汀一直都没有跟周明生谈起黄俊士。这倒不是她成心隐瞒什么，而是这些纷至沓来的事情搅得她心烦意乱，好几回都欲言又止。可是今天，在这个离别的时刻，她觉得应该让周明生知道她已经找到了意中人，或者说，她的心已经找到了归宿的方向。她在陈述过程中淡化了刘智韬的所作所为，因为从今天早上开始，她冥冥之中感受到了刘智韬似乎从黄俊士遇害事件中认识到了什么，言谈举止，一改往日的固执偏见而变得委婉亲和，似乎还有点愧疚的成份。她把黄俊士的情况向周明生讲完后，表示到时候一定会邀请周明生喝喜酒，分享她的快乐。周明生听后，既为黄俊士不幸的遭遇深表同情，又为刘淑汀名花有主深感欣慰。

　　天下没有不散的筵席，刘淑汀终于恋恋不舍地离开了长城长旅行社，离开了可怜巴巴的周明生。黄俊士一见到她，就把刘智韬知道他爸爸下落的消息告诉了刘淑汀。刘淑汀犹如拨云见日，将信将疑地问刘智韬：“真的吗，爸爸？”

　　“真的。”

　　“我太高兴了，俊士！”刘淑汀拍案叫绝，激动地亲吻了黄俊士的脸颊，回头又问刘智韬：“爸，我有没有见过他爸爸呀？”

　　“你……你可能不记得他了。”刘智韬像是在回忆。

　　“爸，他爸爸是干什么的？”

　　“跟我一样，做生意的。”

　　“他现在有家吗？”

　　“有。”

　　“你能找到他吗？”刘淑汀怀着希望地问。

　　“能。”刘智韬语气十分肯定。

　　“谢谢你，爸爸。”

　　刘智韬抿嘴一笑，说：“汀汀，我回酒店看看。”又对黄俊士说：“好好休息吧。”

　　黄俊士感激地说：“刘总，我代表我妈向您表示感谢。”

　　“不用客气。”刘智韬说。“汀汀，你出来一下。”

　　刘淑汀不知道老爸又要搞什么名堂，心头七上八下。她温柔地对黄俊士说：“我马上回来。”

　　刘智韬在走廊上对刘淑汀说：“汀汀，你要知道他现在是个病人，任何兴奋激动的情绪都不利于他的治疗。”刘淑汀听着有点抓不到要领。按港台的说法，就是有点懵嚓嚓的样子。刘智韬干脆往明处说：“你和他……千万不可……过份亲热。”

　　“我知道。”刘淑汀娇羞着说。

　　刘智韬没走几步又回头，关怀备至地说：“医院的菜做得不好吃。等一会儿，你问问他想吃点什么，打电话告诉我。我让人送来。”

　　望着老爸远去的背影，刘淑汀思绪万千。她想，可怜天下父母心。老爸大概是因为认识黄俊士的父亲，所以无形之中，拉近了距离，跨越了鸿沟，消除了隔膜，凸现了关爱。如此看来，别说黄俊士的设计方案即将重见天日不成问题，自己和黄俊士的恋情也必将因祸得福，峰回路转。
