二十七

　　要是不仔细看，准以为是运动员在训练。刘淑汀泳装、泳帽、泳镜，一应俱全，站在她们家游泳池的跳台上，做着些甩胳膊踢腿的热身动作，其女性曲线美，丰姿绰约，呼之欲出。稍后，她轻身一跃，斜插水中，犹如海豚戏水，于蓝色水下，婆娑起舞。她换气自然，动作协调，双臂前后轮番划水，双腿上下交替拍打，如鱼得水，畅游而去，如鸟翔空，翻腾转身。刘淑汀对这套自由泳的动作要领，体会深刻，像一名奥运泳将，做得层次分明，丝丝入扣。

　　黄俊士手上拿着一部无绳电话，从里屋向泳池大步如飞地走来，喊道：“汀汀，你的电话。”

　　“谁啊？”刘淑汀向池边游来。

　　“是个男的。”黄俊士将听筒交给刘淑汀后，朝着他来的方向返回。

　　刘淑汀冲着电话只说了声“哈罗，是的。”就心惊肉跳地捂住听筒，叫住黄俊士：“哥，你快回来。”

　　黄俊士预感不妙，疾步回到刘淑汀的身边，关切地问：“怎么啦？”

　　“张克杰在我们家门口。”

　　“现在？”

　　“现在。”

　　“他要干吗？”

　　“他想见我。”

　　“让他等着。”黄俊士果断地说。

　　“喂，你等一下。” 刘淑汀说完，关掉电话。

　　黄俊士从刘淑汀的手里接过听筒，问：“他怎么知道我们家的，海伦告诉的？”

　　“海伦根本不知道。”刘淑汀猜着说：“也许是爸爸吧。”

　　“邪门儿了。”

　　“他不会是克格勃吧？”刘淑汀戏谑着说。

　　门外面，张克杰倚靠在他的凯迪拉克上，手抄裤袋，悠然自得。

　　黄俊士从屋里出来，神态自如，说：“你好，张克杰，找雪丽？”

　　“哦，她不出来？”张克杰支吾道。

　　“她身体不舒服。”

　　“身体不舒服？”张克杰显然不相信。“我看看她吧。”

　　“算了，张克杰，一个女孩子躺在床上，不太方便。”黄俊士想方设法加以阻止。“我把你的心意带给她就可以了。”

　　“那行，我改日再来。”张克杰悻悻地钻进他的车里。

　　刘淑汀披着一条白色浴巾躲在遮光窗帘后窥视屋外。小狗东东在她身边乖巧地转悠着。

　　黄俊士靠近张克杰的小车右边，敲了敲窗玻璃，说：“张克杰？”

　　张克杰将车窗完全摇下后说：“比尔，你说，咱哥儿们对你是不是够意思了。你跟她拍拖的时候，我找过你麻烦吗？现在你应该帮着我说话才对啊。你这人怎么这样不通情理啊。”

　　“强拧的瓜不甜，张克杰。她不愿意，你又何必缠着她呢？再说，戴安娜不是挺好的吗？”

　　“咳，掰了。”张克杰垂头丧气地说。“就因为你妹妹，彻底掰了！”

　　那天晚上的事，黄俊士后来听刘淑汀说起过，所以他知道张克杰在说什么。他从口袋里掏出一张名片，说：“这样吧，张克杰。我认识一个妞，这是她的名片。我想啊，你找的人，兴许就是她了。”

　　张克杰接过名片一瞟，不屑地说：“总经理，哼，她是美国公民吗？”

　　“当然。”

　　“她有什么要求？”

　　“Money。这你没问题吧？再见，哥儿们。”黄俊士转身欲走。

　　“等等，”张克杰引颈探问：“她跟雪丽比，怎么样？”

　　“怎么说呢，她们不是一个类型的。不过，她比戴安娜强。祝你走运。”

　　黄俊士所说的强，到底是指哪方面呢？大概张克杰听懂了，所以，他没有再问什么，而是说：“比尔，我说句实话吧，你千万别介意。自打我认识你以来，我就没有走运过。坦白地说，就算你妹妹愿意跟着我，我一见到你，心里头也不是个滋味，总觉得要出点什么事儿似的。”张克杰轰然发动引擎，震耳欲聋。

　　“你说得太对了，张克杰，我也这么想。”

　　张克杰气急败坏，倏地一下开跑，车尾喷出浓浓的蓝烟。黄俊士捂住鼻子，往后趔趄退了三步，还是躲之不及，咳嗽了两声。他进屋后，见刘淑汀仍然一动不动地站在窗边，说：“他滚蛋啦。”

　　刘淑汀没有反应。

　　黄俊士走到刘淑汀身边，亲切地问：“你没事吧，汀汀？”

　　刘淑汀突然转身趴在了黄俊士的肩膀上，抑制不住内心的感情，泪水涟涟。

　　“别害怕，有哥哥在，谁也不敢把你怎么样。”黄俊士安慰道。

　　“我不要哥哥。”刘淑汀大声哭喊。

　　“我也不希望你是我妹妹呀，但是……”

　　“我不想叫你哥哥啦。”

　　“那就叫我名字吧。”

　　刘淑汀仔细端详着黄俊士，说：“不，你不是我爸爸的儿子！”

　　“你说什么？”

　　“你不像我爸爸！”

　　黄俊士真担心刘淑汀会精神错乱：“你可千万别这么说，汀汀。你这么说会伤害我们的爸爸和我的妈妈，你懂吗？”

　　“从你搬过来那天起，我一直在观察你。”刘淑汀说话的语调异常冷静，但节奏越来越快：“你的长相、你的声音、你的性格脾气、待人接物没有一样像我的父亲。”

　　“你错了，我画画的天赋就是从爸爸身上遗传下来的。我妈妈当初不让我学文科，就是因为爸爸的事在她心灵深处留下了阴影和余悸。”

　　刘淑汀无言以对，半晌后说：“你爱我吗，俊士？”

　　“非常爱，这还用问吗？”

　　“你要是真心爱我，请你打电话问你的妈妈，你是不是我爸爸的儿子，好吗？”

　　“我妈妈从来没有别的男人，这我太清楚了。”

　　“可是你不姓刘。”

　　“我随了我妈的姓。听着，汀汀，我理解你现在的心情，因为我对你的感情和你对我的感情一样，但是，我是不可能向我妈问这方面的问题的。如果我这样做，会极大地伤害我妈妈的感情。就算我不是你爸爸的孩子，我妈想告诉我早就告诉我了，不想告诉我，自有不告诉我的道理。”

　　“你这样爱你的妈妈，只能叫我更加爱你。俊士，我也不愿意去伤害你的妈妈。这样吧，请你和我爸爸去做亲子鉴定，要真是，我认了。要不然，我这辈子再也不想找什么男朋友啦。”

　　“假如结果出来对我们两个有利，你考虑后果了吗？”

　　“那你叫我该怎么做才是对？”刘淑汀望着黄俊士。“你快说话呀？”

　　黄俊士一时想不出有什么好办法。

　　“我恨你！”刘淑汀爱恨交加，躲到一旁，泪下涔涔。

　　黄俊士走过去竭力安慰：“再等等吧，汀汀，等有机会了，我会问我妈的，好吗？”

　　刘淑汀泣不成声。

　　这天晚上，刘智韬回家后，刘淑汀去了他的房间把自己的想法告诉了刘智韬。刘智韬听后，不悦地说：“你这孩子简直在捕风捉影、无稽之谈。他不是我的儿子，他妈怎么会到今天还是一个人呢？以前的女人本份得很，哪像现在的。你不要拿现在的观念看以前的问题。”

　　“你有没有跟他妈说起我跟他的事？”

　　“说了。他自己肯定也会对他妈说的。”刘智韬显得极不耐烦。“你把你的想法告诉俊士了？”

　　“嗯。”

　　“他怎么说？”

　　“他说，等有机会了再问问他妈。”

　　“那就好了嘛。”刘智韬起身要去卫生间了。“我看你啊……快去睡觉吧。”刘智韬差点说出“神经有毛病了”。

　　“你给他妈拨个电话吧。”刘淑汀执拗着说。“你要觉得不好问，我来问。要不然，我睡不着。”

　　“你这孩子怎么这样？啧。”刘智韬在桌上翻了翻，说。“号码在这儿，你自己打。”说完径直去卫生间。

　　刘淑汀拿起电话拨号，一会儿又懊丧地放下来。

　　刘智韬走到卫生间门外回头问：“没人接？”

　　“占线。”

　　“别打了。”刘智韬说。“俊士现在又没跟别人好，你这样神经兮兮的，真会……真会心理不健康，知道吗？”

　　刘淑汀一言不发地站着不动，准备等一会儿再拨。这时，她听到黄俊士在敲她房间的门，并且气急地喊道：“汀汀，汀汀，你在里面吗？”

　　刘淑汀不知道发生了什么事，连忙开门说：“我在这里。”

　　黄俊士兴冲冲地跑过来，抱着刘淑汀，兴奋地说：“汀汀，你是对的。我给我妈打电话了。她在线上。你快来听啊。”

　　“是吗？”刘淑汀激动得几乎不敢相信，喊道：“爸，你快出来啊！”说完和黄俊士去了他的房间谛听免提电话里黄妈妈越洋的声音。

　　刘淑汀知道黄妈妈跟老爹同岁，现在也是奔六十的人了，但要是事先不知道她的年龄，单凭这声音来判断，最多会猜，是个四十多岁的中年人。她的声音听上去细腻、亲切、婉约，充满着女性的温柔、安祥和慈爱。她首先问了一些刘淑汀的近况，问阿海在不在家。她还是习惯性地叫刘智韬为阿海。她叫阿海时，那声音里充满着经久不竭的浓情厚意，简直要钻进人的心扉，震撼着人的灵魂。刘智韬披着一件外套哆哆嗦嗦地答应了一声。刘淑汀将黄俊士给她的那张黄妈妈怀抱黄俊士的照片从桌子上挪到了电话机的侧旁，静心聆听。黄妈妈说：“要不是因为你们两个孩子，我是死也不愿意再提那些不堪回首的往事。”黄妈妈停顿了一下，开始抚今追昔：“三十年前，红卫兵抓走阿海后，砸烂了我们家所有的家具。我不得不回到我的娘家苟全性命。但是，没过几天太平的日子，他们又卷土重来，硬说阿海逃走，是我把他给藏起来了。我哪里知道阿海的下落。他们就疯狂地毒打我，对我百般蹂躏、折磨、污辱，惨无人道。当时，我肚子里已经有了一个三个月大的孩子……就这样……就这样活活地被他们……给扼杀了。”黄妈妈唏嘘着，声音颤抖不已。“我痛死过去，他们以为我断气了，才肯罢休，扬长而去。我怎么……怎么也想不明白，这世上的人，为什么……为什么就不能好好地相处呢？”她声泪俱下，吸泣着，继续说。“后来，我逃到了温州乡下，靠挖野菜维持生命。那年冬天，一个寒风凛冽的黑夜，我的茅屋外面传来了一阵一阵撕心裂肺的啼哭声。我从梦中惊醒，推门一看，迷迷糊糊地只见一个颤弱的婴儿被紧紧地裹在一条破布褴衫里，冻得已经发紫。我连忙将他抱在怀里，用我的口水来喂食他。我想，红卫兵打掉我的孩子，老天有眼啊，还我一个！从那以后，我跟俊士……相依为命，活到……现在……”黄妈妈哽咽住了，再也说不下去。

　　黄俊士感激涕零，失声喊道：“妈妈，你就是我的亲妈妈！”

　　刘淑汀泪下沾襟，抚摸着黄妈妈的照片，说：“妈妈，谢谢你……好妈妈！”

　　刘智韬脸色苍白，半天说不出话来。

　　“阿海？”黄妈妈问。

　　“我在。”刘智韬讷讷地说。

　　“阿海，当我知道你跟孩子们的事之后，我心里一直很矛盾。告诉你真情，怕你伤心，怕俊士受苦。不告诉你真情，又怕孩子们误会，不能好好相处。我没有保住你的孩子，一生都觉得对不住你。其实，我一直是把俊士当作我们的孩子来抚养的。是按照你当初的理想，把他教育成人的。现在他跟你女儿好，是他们的缘分，也是我们的福分，你应该支持他们，成全他们啊。”

　　“我知道，我知道该怎么做。”刘智韬吸噎着说。“我去中国把你接过来。你会看到他们的。”

　　“我不会去美国的。我只希望，孩子们幸福，别像我们那样……”黄妈妈的话似乎说完了，又似乎没说完，然而电话却中断了，不知是线路出了问题，还是黄妈妈自己不想说了。刘智韬关掉了通话键，迟缓地站离座椅，喟然叹息。刘淑汀和黄俊士面面相觑，既为找回失落的爱而欣喜，又为刘智韬暧昧的态度而忐忑揪心。刘智韬迈出几步后又回头说：“你们怨恨我反对你们，其实，真正反对你们的……”刘智韬将目光集中到刘淑汀的身上。“是你的妈妈。”

　　刘智韬的这句话堵在了刘淑汀和黄俊士的心头，使他们刚刚重获温存的情怀顷刻之间又遭到了冷风一阵，性趣隐遁。

　　黄俊士无精打采地说：“汀汀，我想，我想明天……搬出去。”

　　刘淑汀对此并不表示异议，因为一切理由都在不言之中。她心事重重地抱着黄俊士的腰身，将头安然地依在了他的心口上，低声地说：“你要是觉得住在这里不方便，搬出去也好。”
