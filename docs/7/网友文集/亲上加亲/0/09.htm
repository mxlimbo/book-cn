八

　　原来，当黄俊士正说得娓娓动听的时候，刘淑汀看见了楼下正发生了一件让她触目惊心的事。她失声惊叫起来：“No, No, 东东！”

　　如果说昨天黄俊士把刘淑汀吓了一小跳，那么今天，刘淑汀可是实实在在地把他给吓了一大跳。他被吓得目瞪口呆，直到他听到东东两个字，才意识到是小狗出了什么事而不是刘淑汀，更不是他自己。他猛然扭头，只见楼下的小狗正在把他放在沙发上的那份酒店资料撕咬得粉碎，满地雪花飘。他绝望地吼道：“噢，上帝啊！”

　　“快去！”刘淑汀推了一把黄俊士。黄俊士冲下楼梯。刘淑汀紧随其后。

　　小狗被吓得狂吠乱窜。

　　黄俊士蹲在地毯上一边收拾残局，一边不住声地念叨着：“完了，这下可闯了大祸了。”

　　刘淑汀望着黄俊士哭丧着的脸，忍不住大笑起来：“东东对你有意见了。”

　　“还笑呢，哭都来不及了。”

　　刘淑汀都笑得打嗝了，说话断断续续：“没……没事，这只是……复印……复印件。”

　　“刚才被你吓坏了。”

　　“谁让你昨天吓我的。”

　　“这事要是被你爸知道，我就完了。”

　　“明天……明天星期天……我爸爸不会在酒店。我请绮丽儿帮我复印一份就是了。”

　　“绮丽儿是谁？”

　　“我爸的秘书。”

　　“她上班？”

　　“休息。但我可以请她帮个忙嘛。”

　　“她不会告诉你爸？”

　　“我想不会。不过，她要我帮她办的事，我到现在还没给她办成。”

　　“什么事？”

　　“介绍个男朋友。”

　　黄俊士马上想到了布莱恩，说：“巧了，我们公司有位老美，建筑师，长得一表人才，是我的好朋友，要我给他介绍个中国姑娘。”

　　“不知道绮丽儿要不要美国人……”

　　“你对绮丽儿说，我的哥儿们中文讲得可流利了，语言上绝对不会有什么障碍。而且，他说话幽默，听他说话，你要是不笑，那是你自己虐待你自己。”

　　“你的朋友有什么要求？”

　　“长得好看，会讲中国话的中国姑娘就行。”

　　“还有中国人不会讲中国话的？”

　　“有。”

　　“什么时候他们见见面？”

　　“越快越好。”

　　“就这么简单？”

　　“有缘就简单。”

　　他们将大块的废纸捡起后，留下细碎的纸屑由吸尘器一吸而净。刘淑汀没让黄俊士帮着洗碗。黄俊士也就乖乖地站在她的身旁说说笑笑。两个人你一句我一句，说来问去，越说越投机，越说话越多。就像美国人形容的那样：话题是个皮球，在两人之间，踢来荡去，没有接不住的时候。从谈话中，刘淑汀了解到，黄俊士从小在温州农村长大，是个放牛的娃娃，会吹笛，会画画，会游泳，会爬树翻墙，会登山越岭。从他考上大学背井离乡后，他母亲的印象就只有几个与寒暑假联系在一起的互不连贯的片断珍藏在他的记忆里。他对刘淑汀说：“我来到美国才知道，我跟我妈说话也要掏美金。每次通电话，我妈总要问我找没找到女朋友啊。我被她催促得都不敢给她打电话了，前天才去了白天鹅找海伦。说心里话，遇上你，我真地很高兴。我宁愿白天鹅白赚我的钱，也不要再为我介绍什么人了。”

　　同样，刘淑汀也把自己的身世告诉了黄俊士。她生在台湾，由于父母忙于事业，从小都是由台湾的外婆一手带大，直到六岁那年才来到了美国接受教育，所以她会讲一口台湾国语和地道的美式英语。刘淑汀也把怎么会想起去白天鹅的前因后果一五一十地告诉了黄俊士。两个人都相信他们昨天的奇遇是命中的注定、是上天的安排，因为即使不在大酒店邂逅也会在咖啡馆里相见。当然，他们更愿意第一次的见面像现在这样的，富有浪漫和戏剧的色彩。他们的话题从旅游延伸出来，涉及到了建筑、美术、影视、音乐等等，天马行空似的，什么都谈，就是没谈彼此的恋爱史。大概是觉得还不到时候吧，也许，他们压根儿就没觉得有必要谈什么过去。过去的都让它们过去吧，关键是现在，关键是我和你的现在。刘淑汀收拾完了锅碗瓢盆、刀叉筷子，问黄俊士想喝点什么。黄俊士想，今天收获挺大的，尽管意犹未尽，应该见好就收，别贪得无厌，乐极生悲。记住小心没大错这句话永远不会有错。于是，他问：“你爸爸什么时候回来啊？”

　　“他很忙，通常九点以后。”

　　“做老板也不容易。”

　　“挣钱都辛苦嘛。”

　　“我该走了。”黄俊士看一看墙上的时钟说：“七点整，我们认识刚好一天。”

　　刘淑汀抿嘴一笑，说：“我送你。”

　　两人朝大门走去。在黄俊士系鞋带的时候，刘淑汀问：“明天怎么交给你？”

　　黄俊士说：“我去酒店下面等你。”

　　刘淑汀扭捏一下，说：“去酒店？不好。”

　　“那你说呢？”

　　恰在这时，门外传来哼哼叽叽的汽车声，车库门随即哧溜开启，两道光束探照灯一样地扫亮了车库，使白墙反光分外刺目。黄俊士见刘淑汀豁然变色，知道刘智韬来了，来得比他预料的早，甚至比刘淑汀所说的九点以后足足提前了两个小时！真叫人猝不及防。刘淑汀想让黄俊士脚底抹油从边门溜之大吉，但黄俊士的汽车泊在院子里，刘智韬定然疑心重重，刨根问底。可见拥有汽车并不完全是件好事，由它引起的烦恼有时远远超出了人们的想象。黄俊士对刘淑汀说，与其偷偷摸摸，不如大大方方。刘淑汀见黄俊士若无其事、镇定自如，自己的表情也恢复了常态。两个人在最短的时间内达成了共识：明天手机再联络。

　　刘智韬经小门从车库走进客厅，他一见黄俊士，脸上闪过一阵惊讶，但很快以得体的微笑掩饰住。

　　黄俊士主动问候：“您好，刘总，回家啦？”

　　刘智韬热情地说：“唷，小黄，我看见外面的车，想不到是你啊。怎么样？有构思了？”

　　构思？刘智韬这时候所指的构思在黄俊士听来好像话里有话、弦外有音。黄俊士努力控制着情绪，沉着应答：“今天去图书馆研究了一下您给我的资料，有了一个初步的、大体的设想，还很不成熟。”

　　“不急不急，好好想想，等创意、设计方案出来了，我们再约个时间谈谈，我会去你们设计院的。”刘智韬一副公事公办的样子。“或者，你们也可以通过互联网传递给我，随你们方便。”

　　“好的，那我走了。”黄俊士说。“刘总再见。”

　　“再见。”

　　刘淑汀对黄俊士说：“谢谢你送我。”

　　“不谢。”

　　黄俊士离开后，刘淑汀将房门关上。刘智韬的声音从她的背后传来：“他这是准备进来呢还是准备离开？”

　　刘淑汀心里正琢磨着怎么开口说话，听刘智韬这么一问，正好来个顺水推舟：“你也不请人家进来坐坐。”刘淑汀说完想上楼去她自己的房间。

　　刘智韬不急不缓地说：“汀汀，你过来，爸爸有话跟你说。”

　　“有好消息？” 刘淑汀故作镇静。

　　“我刚刚从天地银行那里来，你陈叔叔还在为你们的事耿耿于怀，他不愿意我们两家的关系从此疏远，因噎废食。他希望你能，怎么说呢，他说汤尼这孩子胡作非为，咎由自取。但看在陈叔叔的脸面上，希望你能原谅汤尼少不更事，给他一次痛改前非的机会。”

　　陈长鸣是美国华人私人银行天地银行的董事长兼行长。他的儿子陈维加，长得一表人材。还在刘淑汀念高一的时候，年长一岁的他迷上了刘淑汀，一直穷追不舍。起初，刘淑汀也曾为他的潇洒俊逸而心有所动。旁人皆称这是一对帅哥亮妹。可是随着交往的频繁和距离的缩短，两个人对爱情的认识和对生活的态度反而越扯越远，各执己见，常常会发生那些难以求同存异的磨擦和冲撞。但是在陈维加一再感激涕零的表白之下，刘淑汀常常迷失方向，直到那天下午。那是去年圣诞节过后不久的一个下午，当陈维加自以为神不知鬼不觉地和一个洋妞寻欢作乐的时候却被本想创造惊喜的刘淑汀抓奸抓双，逮个正着，从此，美丽的谎言昭然若揭，不攻自破。刘淑汀悔恨交加，伤心欲绝。

　　刘智韬所指的汤尼就是陈维加，也就是玛格丽特所指的金融巨头的独生子。本来事过境迁，刘淑汀受伤的心灵早已抚平，不料刘智韬往事重提，又勾起了她阵阵伤痛。她没好气地说：“爸爸，这种事过去这么长时间了，我不想再提起。”说着就往楼上走去。

　　“陈长鸣说，他见你最近一段时间没去他家里，才知道你们俩闹不愉快了，他已经把汤尼臭骂一顿了。”

　　“既然他那么不关心他的独生子，现在又何必操这份心呢。”

　　“汤尼本人也表示忏悔，只要你表个态，他会向你当面道歉的。”

　　“哼，江山易改，本性难移。这种事，任何人只要对我冒犯一次，我决不会给他半次机会。”

　　刘智韬走到刘淑汀的身后说：“汀汀，你别把话说得这么绝，做人应该……”

　　“爸爸。”刘淑汀转过身哭腔喊道：“你是不是有事要求上陈长鸣？”

　　刘智韬将刘淑汀拥抱在怀里，半晌才说：“爸爸何尝不心疼你啊，汀汀。只是，你知道吗？一方面，这小子本质不坏，寻花问柳缘因年少无知，这次你给他的苦头也够他吃一辈子啦，相信他会成熟起来，懂得爱惜。念他初犯，就饶他一回吧，下不为例，啊？另一方面，你是爸爸的心肝宝贝，爸爸的一切生不带来，死不带去。盖这新大楼，也是爸爸对你爱心的体现。你手上掌握着万贯家产，往后谁还敢惹你生气不是？但是，目前的情况是，这新大楼修建需要大批资金，除了你陈叔叔，周围的人谁都爱莫能助啊。没有他的支持，新大楼无异于纸上谈兵。爸爸年事已高，再干几年，恐怕也是心有余而力不足了，你总不会让我将来带着这份遗憾去见马克思吧？”

　　刘淑汀望着刘智韬，问：“他最高可以贷多少？”

　　“百分之五十。”

　　“哼，守财奴！”刘淑汀不屑一顾地说：“要我嫁给他儿子，还这么吝啬。难道他不懂，大楼将来也是他儿子的吗？别说贷款，就算资助都不为过。”

　　“当前经济发展止步不前，企业投资瞻前顾后，百分之五十也算是仁至义尽了，你不是还没进他的家门吗？”

　　“爸爸，你让我怎么说呢？我理解你的苦衷，我也愿意为你违背我自己的心愿，忍声吞气，委屈求全。但我只听你一次，第二次就要听我自己的。”

　　“好，顾全大局，这才是我的好女儿。”刘智韬石头落地，长吁一气。

　　“忠诚是我的眼睛，容不得半粒沙子。眼睛里一旦有沙粒，我非要把他取出来仍掉，眼睛才能再次睁开。”刘淑汀接着说：“我会很快跟他离婚，然后去找我自己真正喜欢的人，一个对我绝对忠诚的人。我要和他厮守一生，无论天涯海角、贫富贵贱。就像天上的鸟儿那样，他飞我也飞，他栖我也栖。”

　　知道女儿任性，却不知道女儿会这般任性。刘智韬叹道：“你这是理想。理想和现实是有距离的。距离给了你一种假想的美感，所以想起来激动，听起来感人。但理想毕竟是理想，现实终究是现实。只有面对现实、适应现实、运用现实，我们才会幸福。你说的那种绝对忠诚的人，除了文学电影，现实生活中你就是亮着车头大灯找都没处找啊，我的傻丫头。”

　　“我相信他正在向我靠近。”刘淑汀脸上掠过一丝自信。

　　“谁？黄俊士吗？”刘智韬揶揄道。“荒唐之举！我正想对你说，你们相识不过一天，你怎么就把他领到我们家？你这是引狼入室你知道吗？偌大一个HOUSE，就你们两个，万一他有非分之想，你怎么招架得了？你了解他吗？在你心目中，他是不是一个年轻有为的建筑师，前途无量？你可曾知道，要不是我给他提供机会，他的建筑师到现在还是个徒有虚名。你知道像他这样的建筑师一年能挣多少吗？”

　　“多少？”

　　“美国不是个发展中的国家，建筑项目早就饱和。朝不保夕的建筑师比比皆是，一抓一把。就我这个项目，小小一块肉，竟然引起几家颇有名气的设计院虎视眈眈，你争我夺。何况，他中国没靠山，美国没根基，孤军奋战，必败无疑。你跟着他，到时候，你要反过来养着他！不是你爸爸不尊重你的感情，妨碍你自由恋爱。是我担心，你被他的外表所吸引，一叶障目，不及其余，将来，将来你会追悔莫及的啊。”

　　“我知道，在你看来，我只有跟着汤尼，才是天作之合，才是秦晋之好。”说完，刘淑汀走向楼梯，拾级而上。

　　“你误会了，订订。”刘智韬在客厅里大声说：“如果你对汤尼彻底绝望，没有一丝挽回的可能，爸爸是决不会把你推到火坑里的。贷款的事，我自有办法。但是，你也要答应爸爸一件事……”

　　“什么事？”

　　“从此不见黄俊士。”

　　“你刚才说过，我跟他只不过一面之交，是个理想，是个泡沫。说不定这个理想一碰到现实，就会面目非全，惨不忍睹。要是一夜之间，我突然躲着他，反而会被他认为是我在自做多情。”刘淑汀往上蹬几步又停下，说：“我不愿意给人家留下这种印象。”

　　“不，男追女，难比攻山；女追男，易如破纸。你记住我的话，汀汀，别到时间怪你爸爸心狠手辣。如果你对我的话充耳不闻，继续跟他你来我往，我会马上取消他的设计！”

　　如果刘智韬前面没有暗示这次设计是黄俊士的处女作，对黄俊士的成名和未来事业有着多么至关重要的话，刘淑汀也许还不会意识到取消黄俊士的设计是一种多么可怕的威胁，更不会被刘智韬一语中的而无力争辩。刘智韬正是瞄准了这个要害，使刘淑汀在这次发生在父女之间的唇舌之战中败下阵来。刘淑汀头也不回地疾步跑向自己的卧室，甩上房门，栽倒在床上，欲哭无泪。她知道，父亲是说得出做得出的。当年，和所有的新移民一样，父亲闯荡美国，白手起家，吃了不少苦，受了不少气，深感创业之艰辛，世态之炎凉。父亲这是心有余悸，惟恐后续无力，前功尽弃，贫穷困苦卷土重来。她体谅父亲的苦衷，理解父亲的苦口婆心和软硬兼施。然而，黄俊士留给她的印象太奇妙了，宛如童话一般的美丽，却又是活生生的可触及的。黄俊士每一次走进她的视野，都会让她脸红心跳，都会让她不知所措。这种感受从前只在阅读中获得，是那样的遥不可及。而今，黄俊士来到她的面前，给她带来了这种感受，真真切切的感受。黄俊士喜欢她，毫无疑问，他喜欢她，所以他来到她的面前。他表达爱意的方式是含蓄的、认真的、发自内心世界的。刘淑汀对此明察秋毫，不容置疑。这是一种心灵相通的感应，一种不言而喻的交流，一种梦里寻他千百度的意境。

　　夜深人静，刘淑汀却人意阑珊，毫无睡意。她仰望着星空，无计可施。她多么希望玛格丽特此刻就坐在她的面前旁观者清，指点迷津。咳，好朋友到了婚嫁的年龄往往顾此失彼：亲近了对象，疏远了朋友。这是一条定律，来自于无数的实例。刘淑汀越是睡不着，越想黄俊士；越想黄俊士，就越是睡不着。她想，黄俊士现在是不是也和她一样，辗转反侧，心乱如麻呢？
