诗卷后记·附录

　　今年的闰四月导致我25岁生日的推迟。

　　从初中就开始写诗，能看懂繁体的王力的《古代汉语》，却学不会大部分同学都懂的数学和英语，初高中时零散入选了一些诸如《当代传统诗词精选》；《中国当代诗人诗历》之类的集子并获了几次诗歌奖，使我在92年成为吉林白山作协最年轻的会员——但所有的这一切都无法改变我最终要毕业于吉林粮专这个事实。在这所学校，我最大的收获不是外贸的专业课，而是粗通了周易六爻八字手相等的实际应用。

　　出于一种变相的逃避，我选择了股票经纪作为自己的职业，我信奉这样一种说法：不要让爱好成为自己的职业。与之相仿的另一种说法是——不要与你最爱的人结婚，不知谁会信奉。岁月以惊人的毅力把一切绚丽消磨至平淡……

　　99年上网后，黄金书屋是第一个发表我诗集的文学网站。之后我到了清韵，榕树。在榕树，因为结识了志同道合的海萍、小牧、JILI、青若等，成立了榕树诗社，再加上去年得了榕树第二届网络原创文学作品奖的诗歌奖而去了上海，对榕树就更多了一份特殊的依恋之情。得的其它的网络文学奖项包括：网易#首届网络文学奖（诗歌类*铜奖）；华网文盟第一届文学奖#诗歌三等奖/散文三等奖；华风原创#情是何物/辽宁广播电视报-七夕节'情网'有奖征文二等奖。

　　有位也写诗词的MM曾问我这些奖中没有得过大奖算不算遗憾，我说诗词毕竟不是当代诗歌的主流，何况这四次我都是唯一的以格律诗获奖的，能为我深爱的这种体裁争到这一席之地我已经十分满足了。我也感谢网上一百多家刊载我诗集的网站和许多鼓励过我的网友，他们让我知道我并不孤单，仍有这么多人如此热爱唐风宋雨……

　　相信前世今生，执著地继续寻找我苦等了几世的情人，无论你是在网上还是网下，咫尺抑或天涯……

　　再略谈些我的诗歌观点：我是从13岁时开始写诗的，13年来共得诗50首左右，格律当初学的是王力的《古代汉语》，后来又接触了一些其它律书……我个人的主张是在保留骨架的基础上对格律做些改革，所谓"穷则变，变则通，通则久。"

　　正因为我喜欢诗词，才想把它们保留并发扬光大，古今音殊，诗词的音韵美体现在读时所能听到的声调上，对于古今不同读音的我主张用今音，韵部也是如此，平水韵中的韵脚归纳其中很大一部分已经不适合现在使用了，是针对当时的读音押韵，如上平声四支韵"移""吹""奇""支"同韵部，按这四个字写首七律别人是不能说你没押韵，可是现在读来是什么感觉呢？支离破碎！

　　又，把"一东"和"二冬"分成两个韵部，现在遵守起来也没什么必然道理……至于入声字，除南方的一些方言中还有所保留，大多数人不能正确读出，现在归平声的归平声；归仄声的归仄声，完全可以废弃了，只要读古人作品时知道哪些现在读平声字的古代是入声就足够了。

　　我主张保留的是必须押韵，可略宽些，但不能离谱儿；必须粘，粘是为了保持诗的内在联系和完整；必须对仗，律诗特有的形式，体现美感。当然我也不反对受过严格格律教育的人或初学者按照以前的格律写，如果写得好，我一样佩服。

　　怕的是有些人食古不化，买椟还珠，徒然继承了形式，却真正丢掉了诗词的神髓，比不懂者更为可叹！！

　　当然我更反对什么完全放弃格律的所谓诗词，通常主张完全放弃格律的人连格律是什么都不知道，谈何放弃？所谓"入乎其中，出乎其外"，未入，安能出？！

　　即使想改革格律，也得先知道格律是什么，否则就是笑话，干脆去写新体诗，何必来折磨诗词！

　　所以我的诗基本都是用今音，合粘对，押今韵的。还有如《国殇》；《自嘲》；《钻戒诗一首留赠白衣卿相未来新娘》首句平声都未入韵，方家可解其故……我另外一个常破的律诗大忌是三平调，但出现时都是箭在弦上，各有苦衷，并非刻意为之……凡此种种，且算自作自受的探索吧……
