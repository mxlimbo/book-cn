结束语

　　一直想为我的大学生活以及那样一位女孩写下点什么，可是动作起来，才发现一切只是凌乱地散落在记忆里，没有头绪，更要命的是，没有如同学校里那么多时间，所以我就象一名小学生写日记一样，一点一点写开来；所以从敲击第一个字母起到结尾，已经三个月过去了，但无论如何我完成了，我还是为自己能写下来而感到高兴，因为我就这样蜕变、成长，因为这就是生活，值得记忆的生活。

　　我的大学生活可谓丰富多采，朋友多多，聚会多多。热爱旅游的我去过两个藏寺，去过青海湖，去过敦煌，骑过骏马奔驰于广袤无垠的草原，骑过骆驼漫步于黄沙漫漫的腾格里沙漠，乘过羊皮筏子飘荡于水流湍急的黄河……可遗憾的是，这一切没有一个心爱的女孩陪伴在一旁。大概也就是因为此，我才有那样的梦想----与心爱的人手牵手，一起去西藏，一起去求那千年的佛。而当我和她相识后，这梦想就指向她了。

　　现在想来，我的大学生活就好象一朵烂漫于山冈的花，开在花丛中，将沁人心脾的花香撒在整个青春，以及人生；我那时的爱情就好象躲在角落里的一株不知名的植物，还没来得及吐出花蕾就被一只可爱的小动物当作了美味，甚至还没来得及让世界看到它的样子，然而，它留下的依然是芬芳，我依然可以骄傲地说：这世界，我曾经来过！

　　2001.02.14
