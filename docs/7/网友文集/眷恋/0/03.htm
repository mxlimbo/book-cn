　　"我不是说了吗，我四年一直在关注着你。不光我，还有好多人都在注意着你。只不过是我和我们宿舍人最先发现的你。"

　　"是吗？你知道我开始为什么会答应你出来吗？"

　　"为什么？"

　　"因为Y说这是你四年来的夙愿。"

　　"现在呢？"

　　"现在我觉得和你在一起的感觉挺好的。"

　　"那就好，不让你讨厌就好。"我高兴但又不敢表现太过地说。

　　天已经黑了下来，我们选择的方向什么惊喜都没有，而且越走越黑，这对于真正的情侣是很好的，但对我们不适合，于是我们折回头往回走.。
