　　他的短短几行字居然轻而易举地打散了她的愁结，搅开她的思路让她置身于另一个思想空间中去。他带有学术性却通俗易懂的文字，真实地将她曾经有过的梦想重新展开。相对于浩瀚的宇宙，渺小的人类中的某一个小小女人的爱情烦恼，又算得了什么呢？她宁愿闭上眼睛去天际遨游，哪怕只是片刻也好，让她的灵魂去幽蓝色的空间涤荡一下吧。

　　[如果我掉进黑洞将会发生什么？ 小资]

　　我掉进了黑洞了，一定是这样！她想像自己的星空旅行遇到了意外，她坠入那个传闻中神秘莫测的天体中。

　　邮件发送出去后，她到常去的沙龙里看帖子，她做什么事都很尽心尽力，即使是副斑竹这么个小小的无冕之王。纵然是心情颓丧至这样的极点，她还是能给出一些认真、和谐且幽默的回帖。她想，也许网络上的小资和生活中的肖栀是她个性中不尽相同的两个侧面。

　　忽地又传来一声"You've got mail"提醒她去查收新邮件，不想中贵居然在半个小时之内就给出了答案。他是在网上的！他要么竟然是感应到她会等他的答复，于是便用英文飞快地打了这封信？她回想起他的第一封信便是用英文打的，而且……看来……他还果真是专家！

　　[小资，

　　假设现在你乘坐飞船，直向星系中心的百万太阳质量的黑洞飞去。然后从离黑洞还很远的地方你就关掉飞船的动力，慢慢靠近，我们来看看会发生些什么？

　　首先，你不会感觉到任何引力。因为你是自由落体，你身体和飞船的每个部分都受到相同方向的引力，所以你感觉到失重。当你越来越靠近黑洞中心时，你开始体会"潮汐力"的引力作用。越靠近中心受到的引力越强，假设你的脚比头部更靠近黑洞中心，脚部就比头部受到的引力就强。于是你感觉"被拉伸了"，最终你将被拉成两半。

　　对于一个十分大的黑洞，潮汐力直到你进入离中心60万公里的范围内时才变得明显。注意，这时你早已跨过了视界。如果你掉进一个小些的黑洞，比如和太阳一样质量的，在离中心6000公里远时，潮汐力就已经使你很不舒服。你将远在到达视界之前就被拉成两半了。这也是为什么我让你跳进一个大黑洞而不是小黑洞的原因，我希望你至少进入黑洞内部后还能活着。

　　当你掉进去后能看到什么呢？惊奇的是你并不一定看到什么特别有趣的事情。想像中远处的物体会被奇怪的方式扭曲，因为黑洞引力弯曲了光，但并不一定。实际上，当你跨越视界、甚至跨过了视界后，你仍能看到外部的事物，毕竟发自外部事物的光还能到达你。当然，外部没有人能看见你，因为有你出来的光不可能逃出视界。

　　总的过程需要多少时间呢？我假定你从离奇点10倍于黑洞半径的地方开始，那么对于百万太阳质量的黑洞，你需要8分钟到达视界。然后，用7分钟到达奇点。顺便说一下，这些时间取决于黑洞的大小，如果你跳进一个小的黑洞，可惜你将会死得快得多。

　　一旦你跨过了视界，在你余下的7分钟里也许会十分惊慌，打算开足你火箭的马力想对逃离奇点作绝望的尝试。很不幸，这事毫无希望的，因为奇点就是你的未来，你没有任何方法避免你的未来。实际上，你把火箭开得越猛，你就越快到达奇点，越快粉身碎骨。

　　不过所有这一切都源自你的假设，我只是告诉你可能发生些什么，希望没有吓到你。

　　中贵]

　　显然，肖栀的科技词汇贫乏了些，她不得不打开即时翻译词典，用鼠标指向她不认得的专用术语。故尔读完整篇文章之后，她长长地吁了一口气。天哪，掉进黑洞便意味着毫无疑问的粉身碎骨、意味着死亡！她浑身一个激灵，3年来她付出的这份感情结果却最终成了她埋葬自己的洞穴，多么悲哀……

　　她惊异又不得不相信他所写的每一句话都有相当的权威性，是什么？也许是他字里行间的气势，不经意之中向她确证了他的真实性和可信性。她于是意识到这是一个非同寻常的人，不像有些网上混混的人那么无聊。

　　[你真是专家吗？请你再告诉我，如果我的好朋友圆缘在安全的距离，看我掉进黑洞，她会看到什么？ 小资]

　　肖栀心想，方芳虽然是自己最知心的女友，但在这场畸形的感情中，她究竟是怎样看自己的呢？科学家能不能用宇宙科学的原理也给她做一个相应的解释？

　　[小资，你好。

　　原来圆缘是你的好朋友，怪不得常常看到她在沙龙里写帖子为你撑腰呢。

　　现在我们从圆缘的角度来分析，显然她和你看到的是完全不同的。当你越来越靠近视界，她看到你移动的速度越来越慢，事实上无论她等多久，永远都看不到你到达视界。

　　一种方法是把它理解为光学幻觉，黑洞的形成并不需要无限的时间，你也不需要无限的时间跨越视界。当你越来越靠近视界，你发出的光就需要越来越长的时间爬回到圆缘那里。事实上，你跨越视界时发出的光，将盘旋在那里永远达不到圆缘。你已经跨越视界了，但那些告诉她这信息的光信号将在无限长的时间里都到不了她那里。

　　还有另一个考察的办法。从某种意义上讲，当你接近视界时，时间确实变得缓慢了。假设你乘飞船在视界外的某一点，然后你在那里盘旋一会儿（当然这需要燃烧无数的燃料为了不让你掉进去），再飞回圆缘那里。你就会发现，在这段时间里她比你老得快得多，因为时间对你来说流逝得比她慢。

　　那么，光学幻觉和时钟变慢这两种说法，哪一个是正确的呢？答案取决于你用哪一个系统坐标去描述黑洞，两种解释都是有效的，它们只是用了不同的方法说明同一件事。

　　实际上，你将在没有太多时间过去之前就从圆缘眼里消失，因为从黑洞方向来的光被"红移"至更长的波长。所以，如果你发出某一波长的可见光，她将看到波长更长的光。当你越来越靠近视界，光波越变越长，最终将没有可见光，成为红外线，然后是无线电波。另外，你得记得，光是发出的一个个粒子（光子）。假设在你跨越视界时发出了光子，它将在有限的时间内到达圆缘——在百万太阳质量黑洞的情况下，一般少于一个小时，那之后她将再也看不到你了。

　　希望我的解释够通俗，能让你看明白了。

　　中贵]

　　这一回，文章更长难度更大，他竟又是在半个小时之内完成并发送过来了。这么专业性的文章，即便不是中贵本人所作，去网上查找资源也不可能这么快啊。这究竟是如何的一个人？不管怎么说，他在她思路混乱、心情遭透了的时候竟拉了她一大把。

　　她去洗手间用冰冷刺骨的凉水冲了一把脸，嘴唇都冻得苍白哆嗦起来，然后她回到暖暖的空调房间，在沙龙里贴了篇帖子。

《彻悟》

　　[回忆起我曾经站立在黄山顶峰上的时候，觉得自己真的可以放弃所有。

　　那是一个风清云淡的秋日下午，天空湛蓝得一片清澄通透，四周包围着我的是静静山林的气息，远处的山连绵起伏着。于是我就想，这山这天都是远古便存在于那里的，并不是故意装点了来迎接我们这些观赏客。

　　山风轻轻地吹，柔云款款地飘，岁月站在那巅峰上便是如此无声地俯瞰人类变迁。在人类之前，就有了这一派景像吧，那么几万万年之后人如果立于此地，是否也要为追忆我们这些"古人"而感叹呢？

　　宁静、安详仿佛是从另一个世界飘来笼罩了我，于是就有了这一种彻悟。人类相对于宇宙已经是如此渺小和模糊的概念，人海中一滴小水珠的我，何苦又去解这纷纷扰扰的世事。宇宙无边无垠，时间无穷无，、渺小人类的烦闷愁苦却如滚滚浪淘沙。

　　大自然将每一个灵魄落定于每一个人，便是为了让这卑微、平庸、琐屑的个体用自己的方式去领略人生。也许赋予你的是各不平等的积极或消极条件，但丧失信心和信念的时候，请一定听听大自然最真切的声音：渺小中见博大、卑微间出珍贵、平庸里显才智、琐屑内找真知，能够探寻生命的意义、更好地活着，这才是宇宙间凌驾世俗之上的"人"。]

　　很快地，肖栀又收到钟规的邮件。

　　[小资，你好。

　　我刚看了你在沙龙里的新帖子《彻悟》，请恕我多嘴多事，也许你遇上了什么事情。但无论如何，在这个新年之初，送上我真挚的祝福，祝你快乐健康。

中贵]

　　[中贵，我们可以聊聊吗？用MSN。]

　　[小资，请教MSN是什么？ ]

　　肖栀对自己的过分热情感到有些疑惑不解，尽管她是个微软主义的信奉者、平日里总竭尽全力向同学好友们作义务推广。可是将这种热心蔓延给网路上萍水相逢的陌生人，在她还真是第一次。
