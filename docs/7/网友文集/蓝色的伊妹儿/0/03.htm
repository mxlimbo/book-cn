肖栀…小栀…小资…小资女人…

　　 肖栀，25岁，就职于上海一家美资化妆品公司，任部门销售助理。英美文学专业毕业，打造她一头一脑一身，从上到下、从里到外的小资情调。但她说一口流利软糯、带着浓浓上海口音的普通话，除非是在公司内部网和工作邮件往来，她拒绝用英文交流。她热爱中文，喜欢写作，时常也会用上海话发发嗲，但从不理会酒吧里和她搭讪的老外。

　　她是个经常改变计划，又很能临时拿主意、作决定的人，今天看《花样年华》，明天就跑去定制旗袍。除了多年不变的直长发，她的衣装必然要跟着时尚走，忽短忽长、一会紧身一会宽松，百变成废物终也不后悔。站在二字头的中间，她还要充分挥霍剩下已不多的灿烂阳光。热爱生活、享受生活，朝气依然蓬勃在她的笑脸上，她美丽因为她爱美。

　　她是时尚杂志的阅读收藏者和投稿人，她可以滔滔不绝地和女友唠叨上半天品牌化妆品，但深知橱窗模特儿身上的美并不是自己也能拥有。所以最后穿在她身上恰到好处的衣裙，往往是从太平洋百货逛来的中档货品，恰如其分于她在这个社会中所处的阶层。上海滩上的时髦事情她从来没有错过的机会，接受过蹦极和空中飞人的惨痛经历后，她又多了一项引以为荣的自夸。

　　她有时聪明过人，有时却狂钻牛角尖；有时温柔甜蜜，偶尔也发发嗔怒；多半时候明辨是非，也有稀里糊涂、不分青红皂白错冤好人的事情发生。她像天空中漂浮不定的一朵云，看到有伟岸可靠的高山，也想歇息下来停停脚步。不过为了一座山而放弃整片山林？她暂时还只能摇头不想答应，山外有山她想自己没有走过的世界还很大。

　　她是微软的狂热追随者，她灵敏的鼻子可以在最短时间内感触微软的各种升级讯息，并把电脑中使用的所有微软Soft及时更新，并热情地向她的每一位朋友强力推荐使用Ie和 Outlook Express。

　　她小巧秀丽的数码相机已经跟随了她整整两年，她的照片文件夹按时间年月分类。她也去各地出差旅游、也留影，但却永远生活在自己镜头里，是当仁不让的惟一女主角。她的笑容灿烂，即使作为背景的风光并不迷人，即使天空有时很不晴朗，却绝不会冲淡照片的鲜活色彩。

　　她经常积极运动，吃饭和吃零食却从来都是毫无节制、穷凶极恶地风卷残云。即便如此暴饮暴食，她的小脸还是会偶尔泛出苍白，手脚的冰冷症是与生具来、无法治愈的痼疾。她不问政治，不关心世界的哪个角落有政党纷争，反正社会主义的上海很不错。锋芒毕露的人在哪里都有，她从不去轧那种闹猛，只要她在淮海路上还能招来一点回头率，她就对自己的生活挺满意。

钟规…中规…中贵…中贵男人…

　　钟规，37岁，天文物理学家，执教于美国芝加哥大学。将近中年的门槛，四十岁的男人尚且是一枝花，他却整日将自己微缩在高倍望远镜和卫星传输回的数据中。浩瀚星空吸引着他的不是牛郎和织女星，而是神秘玄妙的天际黑洞、或是那里隐藏着的时光隧道。在美国生活了十多年，他的国语依然非常纯正，偶尔回国属于那种挨宰的对像。

　　他是个严格遵守时间，生活规划得一板一眼的人，那种刻板是紧紧依附在时钟秒针上的精确。性情稳重，待人宽厚，既有普通男人的幽默风范，又不失科学家的严谨态度，虽然其貌不扬却散发智慧与成熟的气息。他只求每天干净整洁地出门去学校，其它生活细节少有考虑。

　　他在黑洞领域的研究中小有成绩，在美国的华人科学圈里占有一席之地，拿着一份不多不少的薪水。若是以国内生活指数来衡量，该属于中产阶级新贵一类，但他不买名牌不逛商店。他爱呼吸自然的清新气息，心情好时会在阳光下的草地上躺个半天。他还很喜爱旅游，但不愿意冒太大的风险，万事有限度，点到即可。

　　他忠厚诚恳，对人对己都是如此，表里如一也许本身就是教师学者们天赋的优良秉性。他是一座可以依靠的山，无论是飞鸟还是云彩，他欢迎任何人的到来并且给与帮助，最后热情地送他们继续上路。他安心于自己的生活和工作，显得有些与世无争，这种无争源于他一路走来的人生路，平坦开阔而又精彩。满足了，他常常提醒自己，知足者常乐。

　　他是反微软主义者，坚持使用Netscape，而且每天都要对Windows98操作系统恶意诅咒几句，他自愿被隔绝在电脑日新月异的Update世界之外，尽管21世纪的钟声已经敲响，他仍然心满意足地操作着Office97和中文之星的最低版本。

　　他有一套在英国购入的完备摄影器材，他热爱业余摄影，各地的自然和人文景观是他镜头中的主角。数码照相机买了才不到一年，他已经爱不释手地在电脑里存进了一千多张照片。他保存照片的文件夹以地名区分，他的每一个足印都留在他捕捉到的风景里。他的摄影技术和摄影器材的品质一样无可挑剔，即使没有赶上好天气好时间，他的作品总能让人感觉自然的灵动无处不在。

　　他也经常参加运动，但即使万分注意节食，还照样为他生活着的那个国家体现生活水准的优越性。他为自己生为中国人而自豪，尽管从清华毕业飘落去美国自由土壤扎根的人比比皆是，他还是毫无疑问觉得自己是最优秀的。

　　他们在年龄的悬殊上差了一轮、整整一代；

　　他们被婚姻的法度分离在左右两个透明水晶球里，只能远远相望，却触摸不到；

　　他们在现实生活中远隔千重山万里水，不同的社会文化和风俗将他们各自包装；

　　他们在精神世界里分属上帝管辖的两个花园，中间隔离着矮矮的却无法逾越的篱笆和栅栏；

　　他们还有不少心灵上不可逾越的鸿沟……

　　他们……如果不是因为99年夏天的那个如果……
