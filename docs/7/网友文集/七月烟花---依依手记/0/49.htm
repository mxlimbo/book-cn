写在七月烟花之后

　　想起来写这个故事，是因为有天下班回家的车上，电台里突然放了一首歌。不久以前也听过一次，乐曲和歌词都很美，可惜当时错过了记下歌词。以为错过了就再没有了，没想到又一次听到，而且车里的音响分外真切。能做的就是马上手忙脚乱地找记事本记下几句歌词，"一万一千公里以外/ 我对你的爱变的稀薄/ 却放不下来/ 千山万水离不离开/ 你一样存在/ 只是天黑的更快"。第二天在新闻组的热心网友指导下，终于让我的机器里也流淌出这首歌。

　　我很喜欢这首歌的歌名《一万一千公里》，那英的声音沙沙的很质感。它让我想起很多故事，其中包括自己的，更多的是别人的。有眼前的，也有一万一千公里以外的。做个念旧的人其实是件很辛苦的事，写下这篇东西的过程中，我屡屡感到写作的痛苦，如果这可以算是写作的话。那种语言的无力与情节本身的深刻内涵之间的矛盾，这次才有了真切体会。老实说，我是第一次写这样的东西，越写就越觉得结不了尾，思想让我很累，可它不让我草草结束。所以我常常觉得头痛，心动过速。但我却居然还是停不了手地这样写下去，直至写到不知道算不算结尾的时候。我也觉得结局好象很不象结局，可无力再去编织一个什么美丽童话。有个朋友常常对我说，"世间事了犹未了何妨不了了之"，也许是个好办法。

　　近来流行讨论七十年代人的话题，我要说的不是这个话题本身的话，而是由这一系列的话题引出了一个新鲜的词叫"新新人类"，并用它来形容和代表七十年代的这一代人。我很坚决地觉得反感，因为它所能代表的内涵太窄了。至少我所经历的年代里，我们都是很纯真的一代人，我不愿意自己是什么"新新人类"当中的一个。也曾想过要写点什么东西来记下我们所熟悉的七十年代，和这代人成长的过程，但马上发现这太困难了，因为如果是记录一个时代，我的只言片语又岂能担负起这种负担？

　　所以只能记下我的只言片语，希望字里行间能让我，还有那些和我一起成长于同一个年代的孩子能看到些我们真真实实经历过的很纯真的故事。毕竟那样的日子和心情这辈子都不会再有了。有个朋友，我写多少，她就催着要我把写的发到她信箱里，她就看多少。她说，只要是真的情感，就一定会有人共鸣和珍惜。我想我写这些下来就是为了她这样一些人吧。

　　从来都没有勇气把普鲁斯特的《追忆似水年华》通篇一气读完。回忆是很费心力的事，不论是甜蜜还是苦涩的。偶而读一两章已经是很奢侈的感觉，就象啜饮珍贵的陈年酒。

　　我们是七十年代的孩子，生长在城市里，看七月夜空里的满天烟花，流时间的泪。我们或许什么都没有，但有质朴的回忆和伴随长大的故事。

　　我累了，要睡了。

依依

二千年六月九日
