七月的烟花特别美（4）

（七）

　　冬天来了，新年也来了。这是大学后的第一个新年，新朋旧友，邮件穿梭。每天宿舍里负责取信的同学都是最受欢迎的人。晓异也收到许多贺年卡，朋友们不论在哪，即使是在同一座城市，平时都没什么机会见面。可却都还记得她，这点就让她足够安慰。可新年还没到，因为新年总是从安旭的贺卡开始的。可他还记得吗？

　　安旭的专业是化学。他似乎特别忙碌，连雨青都经常抱怨说不知道他在搞什么鬼。在晓异和他偶而的一两通电话里，他不是说天天猫在实验室里做实验，就是说在背大本大本的英语单词准备考托福。看来他还真的是开始为出国这条路作准备了。晓异心里怪怪的，虽然她知道就算出国也要等到毕业，还早着呢，可她还是无缘由地觉得很担心，很忧郁。她和安旭此生的距离好象越来越远了，总有一天会远到一个她无法触及的范围。

　　可晓异终究还是盼到了他的信。是年底的最后一天，早上去上课前，晓异看见安旭的信躺在她的桌面上。上课铃已经响了，晓异抓起那封扁扁的信就往阶梯教室跑，满心开始都是新年的喜悦。连她自己都不知道为什么甚至连自己的新年都是由安旭来主宰的呢，虽然不看他的信也知道里面只是一成不变的几句新年祝福而已。

　　遥远的讲台上那个年青的男老师又开始不厌其烦地讲着他崇拜的莎士比亚和美丽的奥菲莉亚，有时晓异觉得他简直臆想症。于是晓异特地找了个最远的角落坐，桌面上摊开的《英美文学史》下，是那张安旭的卡片。在拆开之前，她定气想了想，想象她拆开之后会看见的安旭那熟悉的蓝黑色钢笔戳上去的几个新年快乐之类的简单的字。展开卡片，晓异却懵了，心脏狂跳：卡片上一反常态地只有蓝黑色钢笔戳上去的"安旭"两个字的落款，其它什么都没有。卡片上是一幅黑白照片，两个外国小孩的背影，他们仰头望着面前尖顶的教堂，穿着厚昵西装的小男孩象绅士似的，一只手提着一只古老的箱子，另一只手牵着那个穿着苏格兰格昵短裙的小女孩。干干净净的照片旁边只有一句话：我们的世界。

　　整个上午，晓异都不知道课上究竟讲了些什么，连下课都是周围的同学推她她才发觉的。夹在熙熙攘攘的人群里走出硕大的阶梯课室，晓异觉得头有点晕，象从刚散场的电影院里出来时的感觉。"什么才是我们的世界？是安旭的世界还是晓异的世界？或是晓异和安旭的世界？"冬天的阳光稀疏地撒落下来，脑子里就这几个问题在来回打转，好烦。回到宿舍，晓异倒在床上，捏着卡片，怎么看就怎么觉得那两个小孩子怎么就那么象她和安旭呢。"不可能不可能，安旭怎么会这样想？如果安旭想表达什么的话可能早就表达了，为什么要等到现在？他肯定是忙得多写两句的时间都没有了。但怎么可能？安旭到底在说什么？"

　　不想吃中饭，躺在床上想。想得好累，晓异好想睡。仿佛有人在叫她的名字，晓异已经觉得自己开始梦游了。"晓异！""晓异！"真象是安旭的声音呀。"晓异！"这次怎么又是室友的声音？？？有人在拼命拉晓异的衣服，"喂，你梦游呀，楼下有人叫你，好象是你那个什么同学。"

　　晓异清醒过来，从床上跳到窗口。她一伸头出窗外，看见窗外的高高的玉兰树，看见云淡天青，看见傻瓜安旭杵在她的宿舍楼下，正在叫她的名字。那一刻，晓异突然明白那个"我们的世界"是什么了。那一定是安旭带她进入的世界。

（八）

　　和安旭在一起的感觉很复杂，但有一个感觉最明显：那就是成天象做梦似的不真实。在和安旭恋爱后几个月，晓异还觉得恍如梦中。

　　那个一年中的最后一天，傻瓜安旭站在她楼下狂叫她的名字，等她跑下来还没来得及喘气，就被他一言不发地按住肩膀，他直视着她的眼睛，让她自己无处躲藏，无法掩饰自己的情感，觉得一颗玻璃做的心就要被安旭的注视看穿了。然后，他用要命的好听的普通话对她说："晓异，看到我的卡片了吧。我不想再等什么了。很早之前我就开始喜欢你了，一直到现在。做我女朋友。"他的手捏得她的肩骨生疼，最后一句话都不知道是算请求还是命令。晓异不知道，也不想知道，心里只有一种盼望了很久挂念了很久的东西终于来到她身边时的那种无措。但她毫无招架之力，过度自尊的坚硬外壳被安旭直视她的目光瓦解，她别无选择。她只能选择在这么多年的莫名的等待之后，终于听到他这句话时，站在宿舍楼下，被他捏着肩膀，无处可逃，象个委屈的小孩掉眼泪。

　　安旭纵然有再多缺点和不是，晓异都没法不去让自己投入到与他的感情当中。她常常会问安旭一些以前的疑问。有时候他们会同时问："告诉我，你是什么时候爱上我的？"然后彼此都大笑。晓异赖着不回答，一定要让安旭先说。他被逼急的时候会说："什么时候？还用问吗？就是有个厉害的小丫头片子叉着腰恶声恶气地要我擦她的桌椅板凳的时候。"然后揽住她的腰，恶狠狠地说："那时我就想，君子报仇，十年不晚呀。哈哈，看见没，现在果真还是报到仇了。"晓异信手摸着他的头发，温柔地看着他嚣张的样子，心里很喜欢，有种想一辈子看他这样子的冲动。

　　安旭的大学边是个很大很大的湖，一望无际，远远有青山，美得不得了。中学时他们还曾结伴来这里游泳，甚至和雨青还在湖边的一棵大树上刻下了他们的名字。晓异有时会和他坐在湖边，看着宽阔的湖面上金色的阳光，谈起很多过去的事。安旭很温柔地握住她的手说，当初他天天站在走廊上和群狼共乐，其实就是想能够看见她出现，因为那时能看见她的机会太少了。

　　"那时我给你写了一封情书，可想来想去，还是没敢给你，觉得你那时一直好象对我有天大的意见一样。是不是吃醋呀？"

　　"你又想呀，谁吃你的干醋呀。看见你们象狼群似的盯着人家女孩子看，真来气！"

　　"还说不是吃醋？醋得走路都走不稳，手里的东西还稀里哗拉直往下掉呢！"

　　"哎呀，你还好意思说呢，小心我修理你！你不是一直说我厉害吗？"

　　"哎，有话好好说呀，别打别打，哎呀，女人真是招惹不得呀。哎哟......"

　　 打归打，闹归闹，打完闹完，晓异还是喜欢温柔地靠着安旭的肩膀。她不知道原来自己是这么依赖他的。安旭喜欢边往湖心丢小石子，看着湖水一圈圈地扩散着密密的波纹，边和她谈以后的人生计划。安旭是铁了心打算毕了业是要出国深造的了，这是他的理想和选择，晓异没法去阻挡，虽然她从来没动过这种心思和念头。他曾和她说过："知道我为什么一直压抑自己不和你说我喜欢你吗？因为我很可能面临着出国的选择，我不想到时对你负不了责。可我骗不了自己。我想你永远和我在一起，不管我在哪里，你都能陪着我。"确实是这样，在安旭天天紧张准备着日近逼近的托福考试的时候，晓异还不明白出国是为了什么，至少她不会主动去选择这样一条路。有时安旭会逗她说："你就不用考试了，以后就做我的陪读太太吧，多好。"晓异笑笑，心里很明白，安旭要做和想做的事一定有他自己的道理，她不会阻挡，只会支持；至于她自己，恐怕也只能迁就他了，毕竟两个人在一起，总得都为对方考虑。

（九）

　　在安旭面前，晓异觉得自己好象是另外的一个什么人。她常常不相信原来自己也可以这么可爱。也许那句话说的是对的，女孩是在有人爱的时候才变得真正美丽的。晓异常常在下午没课的时候走到他的学校，和他一起看书，在球场边看他踢球。可安旭的时间依然很紧张，做不完的试验，看不完的英语。随着托福考试的逼近，他能和晓异见面的时间也越来越少。她当然不会怪他，虽然她很在意。

　　考试前，晓异有一个月没见到过安旭了。他说想静心复习，晓异只在电话里"嗯"了一声表示同意。晓异很苦恼，实际上她很难接受这种方式----她太依赖安旭的存在了。可趁着这段时间她也好好想了些问题。他们两个都是性格很强的那种人，客观地来说，一旦有什么问题或分歧，如果想指望谁能迁就谁并不是一件容易的事。晓异原本不信什么命呀之类的东西，可自从他埋头书本去啃托福以来，她开始猛找一些星座呀，八卦之类的书研究，纵然不全信，可也觉得是个参考。有时她会在看书的时候突然走神，去感觉她那些胡思乱想。她真的觉得她变得比以前过敏很多，小小的事情她都会想很久。晓异顽强地打电话找雨青，想找到个这个唯一可以说话的人倾吐。

　　坐在雨青的小房间的地板上，晓异才稍稍觉得可以把紧张的神经放松一会。雨青的家离晓异的学校很近，有时晓异会心血来潮地住在这里，和她聊个整晚。关上灯，放点音乐，再点上胖胖的卡通小蜡烛，整个房间里都摇曳着淡淡的红光。她们喜欢坐在柔软的地毯上，背靠着沙发，怀里抱着大大的抱枕。在这样的气氛里聊天，仿佛什么心思都可以泄露，没什么理由隐藏些自己。她们常常会聊到半夜，再爬起来找东西吃。有时说着说着就觉得好象天色开始转白了，两个人也都折磨得精疲力竭，于是第二天早上肯定是一起逃课，睡到中午才起床。

　　雨青也正在恋爱，一脸沉浸在幸福中的模样。那个男孩儿晓异见过，是个学艺术的，搞油画，虽然有点不修边幅，但满身逼人的才气，雨青对他刻骨的欣赏。他给雨青照了很多很棒的黑白生活照，张张都象配了哲理诗的油画，常常把晓异看得一楞一楞的。唉，优秀的男孩子太多了，"弱水三千，我只取一瓢饮尔"呀。于是又想起安旭。 "雨青，我好累。"晓异捧着脑袋恹恹地。"我老觉得把握不住他，他骨子里的很多东西是和我相通的，可形式上却好象总是格格不入。"

　　"我一早就说了，不知道那个傻瓜有什么好的，值得你那样眷恋他。我可不是说些没良心的话，这么多年的朋友，我太了解他了。他现在还是那种没长开去的小孩子呢，要等到他会真正爱人，可能要等到三十岁罗。"雨青又搬出了她那套亘古不变的三十岁理论。其实自从他们开始恋爱的第一天起，雨青就是持保留态度的，她总觉得晓异和安旭有什么不妥的地方，可又说不太出来，只是感觉而已。有时晓异和她讲安旭站在她楼下要她做他女朋友的事情，雨青总是会笑得不知所已，说："真想不到一向理性的安旭还有这一手，哈哈哈，学会点浪漫了。"

　　是的，安旭是很理性的人。这与晓异骨子里的那种敏感似乎有点格格不入。她不愿意多想，虽然她很想分析得一清二楚，得出个让人心安理得和安旭在一起的结论。"love is blue,love is blind"，也许这就是他们在一起的理由。
