七月的烟花特别美（3）

（五）

　　晓异他们这代人总的来说都还算是幸福的孩子。高三那年夏天的末尾，他们三个都分别被三所大学录取，人生开始更加富于创造性的四年。

　　可晓异却总爱翻看那些她断断续续记了六年的日记，看着她和安旭斗嘴，生气的过往片断，总能沉没到当年的那些情形当中。那时他们虽然老是针尖对麦芒，可每年里至少有几天还是很和平的，他们谁都不会和谁斗嘴，生气，或是故意招惹谁些什么。这几天就是他们各自的生日。相处的这么多年里，无论碰到谁的生日，大家都会开开心心地想法高兴一下，这种习惯从那时就一直保留了下来。晓异的生日在七月，总是在学校刚刚放暑假的时候。他们总会一起骑上车，去天然的湖泊里游泳，在碧水蓝天里感受七月的热力似火。而晓异总会嘲笑安旭的泳姿是狗刨式，气得安旭拼命向她拍水，直到她哇拉哇拉的求饶为止。而雨青则笑得在水里直呛。从那时起，记得彼此的生日成为一种长久的习惯，永不改变的习惯。

　　另外还有那些她仔细收藏的六年中每个新年里安旭送给她的贺年片。事实上，每个新年来临时，她觉得好象其它什么都不重要，校园和教室缤纷的彩带和气球都是只是一种装饰。只有看到安旭送给她的卡片，她才会觉得安慰，新年是从安旭的卡片到达的那刻开始的。

　　当然，安旭从来不会说什么柔情的话，就象雨青老是说他还没开窍一样。最神经的时候他也只会用蓝黑色的钢笔硬硬地在贺卡上戳上一句："江南无所有，聊赠一枝春"。为了这个，雨青甚至有一次专门找了一本心理学的大部头著作，戳着晓异的脑门说：

　　"看看看，这行！看见没，在青春期，女孩子和同年段龄的男孩子相比，心智显得更加早熟。所以我早就说了嘛，安旭那个傻瓜，还不知道什么时候才能脑筋开窍呢。"雨青笑着，很有把握地说："我看拉，安旭这样的人要到三十岁才会开始有点味道，就象泡茶一样，初道茶是不好喝的，知不知道？"晓异不知道雨青说的话对不对，要等到三十岁，也太遥远了。她只会每每看到日记里那些琐碎而细致的片断，自己感动不已，老是一个劲儿问自己是不是真的对安旭这么在意。

　　 进入大学，并不象前人说的那么夸张，象从地狱到天堂。晓异只觉得好象是从幼稚园到了成人社会，以前什么不该想的，不该做的，在这里全都合情合理合适地存在着，比如说恋爱。学校是名校，山青水秀，小径通幽，人文气氛迎面扑来。晓异看着校园里双双对对走来走去的情侣，想着雨青肯定地说着安旭这种人一定要到三十岁才开窍的结论，只能叹叹气。也许雨青是对的。晓异只能日复一日地走在秋天里开始飘荡金黄色落叶的校园大道上，想象着如果有一天能和安旭走在这里的感觉。

　　开学快两个月，晓异都没见过安旭了，听说学校拉他们到山区去军训了。雨青也是忙得不可开交，她的专业是学设计，生活丰富多采，不是出外写生就是做作品做展览的；而且雨青不说晓异也猜得到，虽然才入学，她肯定是被人早就追得鸡飞狗跳的了。常常连周末晚上也逮不到她的影子，晓异有时想借着打电话向雨青抒发一下自己的郁闷，可这竟也成了一种奢侈。

　　宿舍楼里的女孩子天天花枝招展风摆杨柳，今天会这个老乡明天会那个朋友，而且鸿燕传书，乐此不疲。晓异并不妒忌，可真的是打心眼里羡慕。她的床就在窗口边，她总是趴在床上看着窗外的高高的玉兰树，只想哪天云淡天青，一伸头出窗外，就能看见傻瓜安旭杵在她的宿舍楼下，叫她的名字。

（六）

　　秋天里的校园很美，处处被浓郁的桂花香味包围。已经过了国庆节了，秋天和夏天还依然同在，下午的阳光很灿烂。晓异从图书馆出来，手里抱着大摞大摞的建筑欣赏图册。晓异从来就是个好感心重的人，什么都想探索一番，所以在图书馆里很少能看到她在自己的英美文学专业的分类架前找什么书，而是在不同的架前乱找，然后抱一大堆形形色色各方各面的书出来。下午的课已经下了，学生们三三两两从教学楼的各个出口冒出来，安静的球场马上开始热闹起来。校园的下午，是晓异很喜欢的时刻。林荫大道上有个反戴棒球帽，脚蹬溜冰鞋试图扮酷的男生，嗖地从晓异身边掠过。忽然有人从后面敲了一下晓异的脑袋，晓异回头一看，那个她熟悉又陌生的安旭正站在她身后傻傻地笑，点着她的脑袋说："不认识我拉？"

　　本来就黑的安旭显得更黑了，又穿了件黑色的T恤，一只手插在米色卡其布短裤的兜里，象是刚从非洲旅行回来一样。看着晓异猛盯着他的脑袋，安旭略略不好意思地摸了一下自己短短的寸头，说："别这样盯着看行不行，好象我是怪物似的。咱们整个男生营都这发型，帅吧，呵呵。"晓异继续盯了两秒钟，突然大笑起来，抱着成摞的书捧着肚子笑弯了腰。晓异想象过很多次安旭再一次出现在她面前的情景，但唯独没想到他的头发会是这样的好笑。"哎，我可跟你说，我可不是来看你的啊。昨天才军训完回来，我们那边山头的哥儿们都说你们学校的女生漂亮，特地要我来侦查侦查，顺便呢，才是来看看你的。"

　　走在下午的阳光里，老樟树在风中轻轻摇摆，散出清新的味道。安旭又马上恢复到那个晓异熟悉的状态：眼晴闪闪发亮，狼似的盯着来来往往的女生，口水横飞。这让晓旭想起了那个原来站在中学走廊里嘻嘻哈哈，什么事都满不在乎的那个的安旭。"唉，这两个月过得真够呛，还别说我吹，我真的两个月没见过女的拉，我们营房食堂大师傅养的猪都是公的！可真把大家憋坏了，哈哈哈哈......" 看着安旭大大咧咧地讲这些晕话，晓异有时真的对雨青下过的结论表示过怀疑，不知道他究竟是不是个什么都不懂的傻瓜。但有一点可以肯定，他好象不再是那个中学时代的安旭了，他开始象个大男人了。

　　自从这次最初的造访后，安旭会久不久地来晓异的学校，可真正见到她的次数却很少。他总能保持自己的性格，每次来都不提前打招呼，有几次晓异去了自习或是图书馆，回到宿舍，总能看到书桌上压着安旭借门房大妈的小铅笔头戳在小纸片上的几个字："我来了，你不在，没什么事。"每次看到安旭这几个写了等于没写的歪歪扭扭的字，晓异总会生上会闷气，可过不了多久，又在心里温柔地原谅他。她也不知道自己是不是越来越离不开这种感觉，安旭到底在想什么，安旭心里的她是什么样子的，她一点谱儿都没有。

　　隔壁宿舍里的收录机老是在唱同一支歌："爱人的心是玻璃做的，一旦破碎了就难以再挽回......"晓异觉得她的心也象一颗玻璃心，渴望安旭来了解，却很怕不小心打碎。受伤的感觉是晓异所恐惧的，她天生保护自己的意识特别强，那就是她拿绝对自尊和敏感织成的一张保护网。所以哪怕是忍受日复一日的猜测和幻想，她也不愿轻易地去做这种尝试。她可以做的就是收起每一次安旭留下的纸条，然后把它们夹在日记本里，在夜里床头的温柔灯光的笼罩下，一个人听着音乐，静静地收拾心情。

　　有时好容易逮到久违的雨青的电话，晓异一开口却总不知道从哪儿说起好，急得雨青一个劲儿在那头叫："你还是晓异吗？你已经是大人了，还不会处理自己的事吗。喜欢他就去告诉他呀，难不成这种事还要我帮你去讲？做朋友这么久，有什么不好意思的呢。别折磨你自己了。唉，说到底，我就是想不通安旭那个傻瓜有什么好的，值得你暗恋他那么久，到现在还不敢说？！......"听着雨青在电话那头嘟嘟囔囔地说着，有时晓异的眼泪就会不知不觉地啪啪掉在听筒上，在电话这头拼命抽鼻子，雨青也连连叹气，嘴里还在说着"安旭这种人是要到三十岁才会成熟的......"

　　可无论如何，晓异都觉得也许自己这辈子是摆脱不了安旭这个梦了，不管他是个美梦还是个没有结果的梦。她愿意这样一直做下去。
