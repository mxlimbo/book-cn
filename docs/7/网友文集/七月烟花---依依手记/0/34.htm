相忘于旧事

　　那天晚上，一个人坐在滴滴嗒嗒的深夜，在明白的顶灯下，拆着信，几百封多年以前的信。窗外的黑夜里，可以看见飞机打着明亮的前灯慢慢在城市里下降。

　　这些信曾经被细心地分类，一扎扎地用丝线扎好，一捆捆地放在小纸箱里，留在那个曾经成长过的城市，很久很久都没有人再动过它们。它们身载着一段段的过往时光，静静地躺在书柜下面，等待某天，比如今晚这样，它们的主人会重新在灯下抚平它们，抚平那些旧时旧事旧心情。

　　在那几百封信里，最早的是小学五年级和同学吵架后、从作业本里撕下的纸上写的道歉信，最后的是大学毕业前、和老朋友共同唏嘘离愁别绪时狂乱的笔迹。在那些信里，还夹杂着一些当时的照片，或是简单的贺卡，上面都留下了多年以前的心迹。

　　那跨越十几年的几百封信，拆得人热泪盈盈。如果不是那个晚上，如果不是重新抚摸着这些深深浅浅的字迹和薄薄厚厚的纸张，我就不会记得，自己居然曾经有过那样幼稚的笔迹，那么多哭笑之间的故事和情境；我就不会记得，自己居然曾经被成长里那么多的事情打扰与伤害，同时，曾经被那么多人那样深深地牵挂、照顾和爱恋过。

　　十五年前，那个认真的小姑娘在作业纸上写：我先向你道歉好吗，我们永远不要吵架，我们要做一辈子在一起玩的朋友，你是对我最好的朋友；

　　十三年前，那个笑起来眼睛就没有了的女孩在薄薄的信纸上写：初中的生活多么丰富，又是多么累人，我们下星期就要去春游了，好盼望，可今天还得做大扫除；

　　十年前，那个又高又酷的男生用潦草的笔迹写：给你写这封信真辛苦，拿着纸笔到处打游击，不然被其它男生看到，会笑话我给女孩子写情书；

　　九年前，那个伴我成长直至今天的死党它它用扁扁的美术字写：如果哪天，我们都有自己的男朋友了，我们四个人就一起骑车出去玩，在夜空的星星下露营和吹风；

　　八年前，那个暗恋你却什么也不敢说出口的男生羞涩地写：一直想去学校看你，可鼓不起勇气。想想，还是想来看你，可能中秋节吧；

　　七年前，那个疯狂喜欢三毛却又多愁善感的女孩子写：女人和女人之间常因一些最小的事而心生隔阂，我是多么希望站在那个熟悉的门口，你还能习惯地挽着我的手臂，一起散步散下去；

　　六年前，那个知道你根本不可能属于他的男生写：想起以前共处时光里的友谊，留在异地的我的眼睛就湿润了，虽然不能再看到你，你永远是善良可爱的小妹妹；

　　五年前，那个善良细心却象小弟弟似的男生苦口婆心地写：在外实习，不要玩得太疯、暴饮暴食，小心又犯胃病，不吃药的你，讲课讲到嗓子哑的时候，还是吃吃西瓜霜吧，随信寄上......

　　在这些经过了很多很多年的各种各样的字迹里面，隐约看到人生一段路上长大过来的自己和身边的人，看到我们曾经共同关联的故事。我们曾在一个时空里成长，长大了，发现曾在身边的那些人，他们几乎已经全部不再在我身边了。我们共有的交集，只是生命里特定的那一段时间。现在在我身边与我共度的人们，也许也一天，也不会再返，我们正在度过的，也只是现在的这段交集而已。

　　昨天看见庄子的那句话，心里酸酸的，可随即又坚强起来。"相掬以湿，相濡以沫，不若相忘于江湖。"旧时不会再返，旧事不能重来，纵然会回想，回想之后，还是将这些心情都相忘于江湖吧，来一次抽刀断水，举杯销愁。到了命里该忘却的那刻，就要忘记；哪怕抽刀和举杯以后，明知水更流，明知愁更愁。人生有时是没有办法的东西。

　　其实也只是相忘。因为你知道，就象《笑傲江湖》里任我行说的那句话："有人就有恩怨，有恩怨就有江湖，你又怎能退的出呢！"所以，江湖永远是不能抽身而弃的东西。人在江湖，就有永远割舍不完的旧事和感情。

依依 4/11/2001大风中，降温的城市
