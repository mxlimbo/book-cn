七月的烟花特别美（1）

（一）

　　七月来临。亮晃晃的阳光好象已经把春末最后一丝潮气彻底蒸发干净了，透过绿得不真实的阔大树叶射下来。微风吹拂，什么都爽净明亮，就连马路边大大小小的广告牌也显得生动起来。这就象在平时，晓异是极少认真去看街头的广告牌的，而最近她居然发现那个《将爱情进行到底》的徐静蕾也在为某个网络做广告，年青的笑容纯不胜收；而旁边就是葛优瞪圆了他的小鼠眼说着《没完没了》，锃亮的光头让人眼光停驻不已。街边撑起阳伞的卖汽水的小店里在唱："难得有的好心情, 突然放晴的天气 ,整个城市欢天又喜地......"

　　又是一个七月。她毫无选择地对这个热度不断上升的季节抱有特殊的感情.七月是早上在穿衣镜前面临太多选择而无所适从的季节,是午后烈日下逛街时叼着吸管啜饮着草莓奶昔的季节，是纵身跳入泳池碧波那刹心里蹦出做条鱼的念头的季节，是躲在家里开足冷气跷起双腿不分天黑地暗狂看VCD的季节。当然，也是在某一天会出现些许情理之中而意料之外的惊喜的季节。晓异的生日就在七月，就在这个夏天真正开始的季节。

　　每年七月里的生日，晓异都会有些不同的感觉。在一年里的其它时候，她会很细心地把家人和朋友们的生日记在电子记事簿里，即使再忙，她至少也会打个电话或是寄张电子生日卡，想象着他们被人记挂和关心的喜悦。然后她就盼星星盼月亮似地盼着自己生日的到来，期待着那些可以想象或是不能想象的惊喜。可真正那天来到的时候，她又觉得自己仿佛很抗拒，一年比一年要抗拒。可能长大了并不好，成熟也意味着衰老。还有些伤感的是，每年的这一天都会发现，今年身边和你共同庆祝的朋友和去年的又不全相同了，不知道到底是自己在不断流浪还是别人在不断流浪。所以她偶而在这一天甚至会故意关掉自己的中文传呼机和手机，只是为了让自己去刻意淡化这种感觉。而往往在第二天，朋友们就会怨声载道地接连打来电话或留言说昨天和谁去乐了，害得我们总是说些迟到的话云云。晓异总是在电话这边浅笑，她很明白自己要找的理由不是为了应付朋友，而是为了说给自己听的。

（二）

　　早晨醒来的时候，头隐约有点痛。还没睁开眼睛，就已经感觉到夏天早晨窗外透进的亮色。晓异几乎差点忘了今天是自己的生日，而事实上，昨天她还特别向公司请了假。不为什么，就是想生日的这天能完全属于自己，按自己的意愿去支配，而不用再在公司里翻译那些永远也译不完的资料，起草那些永远也起草不完的合同。哪怕是在家一个人发呆也好呀，况且晓异一向把发呆当做是她人生之中的几大快事之一。

　　在床上翻了个身，眼睛还是抗拒着不想睁开，可脑袋已经开始舒醒。晓异一直在摆脱就要清醒开去的感觉，想把没做完的梦继续延续下去。可不管怎么努力，梦还是断了，怎么连也连不上，而楼下马路上的声音却越来越清晰地钻进耳朵里。

　　晓异很可惜地在心里唉了一声。昨晚她梦见了安旭，梦见去年夏天安旭回国时和他，还有雨青几个在咖啡语茶-----那个他们称之为"窝点"但其实很雅致清新的咖啡店里聊天的情景。

　　好久好久没梦见过他了，每次做完关于他的梦，晓异都要迷糊一天才能清醒过来，然后再习惯性地在晚上和雨青通电话，开始的第一句话永远是恹恹地说："雨青，我今天心里不舒服"，然后就开始说她做的梦。雨青总是能变着法儿来提起一些往事，说得极为滑稽可笑，两人会在电话两端笑得象抽了筋一样的喘不过气来，完全和刚开始通话时欲生欲死的气氛格格不入。等到笑得实在累了，两个人的声音都睡意朦胧起来，再各自丢下电话沉沉睡去。

　　晓异有时会十分无可奈何地对雨青说，我完了，我觉得自己需要你的时候就象吸了鸦片毒瘾发作的时候，鼻涕眼泪就想冲着你掉。

　　家里没人，晓异伸长手拧开头顶床头的音响，随手放了张好久没听的老狼的校园民谣进去。钢琴叮呼的伴奏声干净纯粹，是那首她喜欢得心疼的《模范情书》，她选择了REAPEAT功能，这首歌就来来回回地在房里回荡起来。

　　 "我是你闲坐窗前的那棵橡树
　　 我是你初次流泪时手边的书
　　 我是你春夜注视的那段蜡烛
　　 我是你秋天穿上的楚楚衣服
　　 我要你打开你挂在夏日的窗
　　 我要你牵我的手在午后倘佯
　　 我要你注视我注视你的目光
　　 默默的告诉我初恋多忧伤
　　 这城市已坦开他孤单的地图
　　 我怎么能找到你等我的地方
　　 我象每个恋爱的孩子一样
　　 在大街上琴弦上寂寞成长"

　　晓异完全醒过来，可就想保持目前这种状态。手垫在脑袋下，眼睛盯着远处书桌电脑旁去年和安旭，雨青的合影，赖在床上发呆。在这个生日的早上。
