阳光下的拉奥孔

　　早上上班，摇摇晃晃却空空如也的公车里，颠簸得厉害时，硬硬的木条座椅磕得人有点痛，偶尔几股有点冰凉的空气从车窗缝里钻进来。手捧宗白华的《美学散步》，这种有点反差的情景有点奇怪，可我却很自得。有时你得承认，反差常会带来或多或少心理上的一些快感。

　　拉奥孔的故事从大学美学课上一直分析到今天，仿佛一定要在这摇摇晃晃的公车里我才能对它有更深理解。拉奥孔因为提醒特洛伊的人们要小心防备希腊人的木马计而被袒护希腊人的阿波罗神惩罚：他和他的两个儿子被海蛇缠身并被一口口的吞噬掉皮肉。希腊晚期的著名雕像群《拉奥孔》表现的就是他们在精神和肉体上受到巨大折磨的情景。可令人争论不休的美学命题在于：在遭受如此大折磨时，雕塑中的拉奥孔并没有象维吉尔在诗歌《伊尼特》中描写的那样张大嘴痛苦的狂吼，而只是微微张嘴，仿佛在呻吟。据说拉奥孔的这种表情不单为了照顾雕塑美学上的要求，更是一种古希腊时代内敛精神的表现，代表了古典艺术的理想，即"高贵的单纯和静穆的伟大"。

　　很长一段时间以来，看到拉奥孔这几个字的组合，就仿佛看到痛苦这个词的代名词。我绝对写不出莱辛那样的传世著作来分析拉奥孔里的美学，痛苦和哀伤当然不是耻辱，我只是常常心生疑惑：如果痛苦当前，我应失声狂吼还是只微微呻吟？

　　每个人想来都会有彻头彻尾痛心疾首的时候吧，我猜，为自己，也为别人，只不过每个人都有属于自己的特别方式去表达和渲泄。有时很好奇，仿佛别人都不曾痛苦过，人们因为自身矛盾痛苦挣扎的情景好象只有屏幕上的才真切，现实中，大家都小心神秘地隐藏着自己的极端情绪。所以有时反而会怀疑自己的过激，其实人应该都是一样的，只是我没见过他们成宿未眠留下的烟头，没听过他们在无人的山顶上痛苦的狂呼和狂哭后失调的声音。对于女人，只是好象是有人说过，一个女人剪不剪长发，换不换香水，有时超越了这些东西本身的意义，而上升成为一种内心情绪的表现。她们往往借这些来表达当时用其它形式难以表达的内心挣扎，这些看似表面化而温存的东西背后有时竟是非常矛盾和痛苦的挣扎结果。

　　而我，在痛苦的时候，也许会坚决地剪掉长发，换上别样香水，竭力抛弃掉与以前有关的某些记忆，可除此之外，往往还会发狂地贪恋某些特定时刻里的感觉。那些瞬间的感觉仿佛时空停滞，把思想最大限度地表现出来。偶尔心情不好的夜晚迟归，就算急赶慢赶地从地铁站的自动扶梯上跳下月台，还是眼睁睁地看着列车呼啸离开，看看显示牌，又得等差不多十分种。想起初中的时候，就曾把一篇美文抄在自己的笔记本里，文章里说，人生的过程就是要不断地赶一些车，赶不上也许无奈，可总要继续赶下一辆，总有一辆是你要上的车，会带你去想去的地方。不知道那时怎么就喜欢这样的文章呢。很无奈的一个人站在夜晚的空荡荡的地铁月台，灯箱依然通明，在胡思乱想中咀嚼着痛苦，月台的地面被灯光照射得反着冷冷的光。铁轨铮亮铮亮地散发着金属沉静的光泽，寂寞地往黑暗的远处延伸。列车驶来时，站在离黄线最近的边缘，地道里的很大的强风扑面而来，把衣服吹得在身上紧紧贴着，忍不住地打着寒颤，眯缝起眼，却不肯后退一步，倔强地看着那一节节明黄光亮的车厢呼呼而过，然后慢慢地驯服地在身边停下来，停下又一个空荡车厢里的心情。这种时刻，总是意犹未尽，我的痛苦仿佛在这一刹停滞，停滞在明黄的速度里。

　　昨天晚上看报纸，惊闻上海的陆幼青去世的消息。这个自从八月份就将自已最后与癌症抗争的时光以《死亡日记》记录下来的人，终于还是去了，但是很有尊严的去了，留下十万字的日记，还有那句让我眼睛湿润的话，是他写在这本出版了的日记扉页上献给他的妻子的话："你能给我的都给我了，这是我能给你的。"今天早上一上班，就上到"榕树下"网站，这个由头至尾登载他日记的网站，已经有个悼念他的专题了。重新把他的那些文字又看过，不由得你不动容。陆幼青曾提到他喜欢的史铁生和《我与地坛》，忽的觉得人在某些方面真的非常相似。《我与地坛》是上中学时在《读者文摘》上看到的，其中关于史铁生的母亲每天推着他坐的轮椅到地坛去散步的情景让还不更事的我热泪盈眶。

　　考问自己，没有象史铁生那样无助地坐在轮椅里守着每一天寂寥的地坛，没有象陆幼青那样每天早上揽镜自顾时看见肿瘤在脖子上疯狂生长，对于生命的痛苦和尊严，你能体验到多少？我觉得很惭愧，我所能体验到的，远非那样深刻，不仅仅因为他们各自的特殊境况。我每天只是在幸福的平凡中痛苦而已。不管我是痛声狂吼的拉奥孔，还是低声呻吟的拉奥孔，我竟然把痛苦看得那么厚重，厚重到有时简直令自己窒息和厌恶，忘记了生命是在一天天继续，不能提前去透支任何一点东西，包括痛苦。也许只有在意识到生命是有限的时候，哪怕痛苦，也会是值得珍惜的经验。

　　这几天常常会碰到一些朋友聊天，谈到的东西和自己以前曾经的东西很相似。有时不可避免地怀怀旧，想想当年的自己和别人，觉得历史真的都还是值得的。讨论感情时，常会有人征询似的问，该不该爱，该不该等。我常常想到以前化学课上讲物质结晶的过程：升华的逆过程的是结晶，那么爱情到底是不是可逆呢？该不该执着地在其中找寻因和果呢？也许爱本身就充满了幸福与痛苦，选择它，就只能继续它，不后悔，不回头。时间往往会说明一切。

　　时间就象一个过滤器，慢慢滤去岁月的浮燥气息后，剩下的都没有了当时身处其中时的大喜大悲，而是大彻大悟后的淡淡香气，悠远而持久。哪怕在当时是再痛苦的东西，也一定会有属于它的淡淡气味永远保留在你的脑海里，就象《追忆逝水年华》里普鲁斯特关于童年吃过的下午茶里小甜饼气味的回忆一样，会在某个不经意的时刻撩拨回忆的心。

　　我问一位好友，痛苦的时候一般都做什么。他说："听音乐，抽烟，就象上网时一样"。正宛尔之际，他停顿了一下，又说："有时会痛哭。"听了很安慰，因为人都是一样的，我的痛苦并不奇怪，生存就会有痛苦，生活就会有烦恼。和朋友聊起拉奥孔，他说，怎么表达痛苦是个性上的事，可怎么面对痛苦才是共性上的事。他说得很对，痛苦的勇气也许就可以化做日后的信心。

　　一直温暖得不行的天气终于降温了，上班时再坐在公车里，窗外是冬天的小雨和湿湿的街道，没有几日前的树梢上淡淡的阳光。可翻开拉奥孔的雕像页，看见他微微张合的嘴里，稍稍拧紧的眉头里，仿佛有自痛苦中升华的阳光。
