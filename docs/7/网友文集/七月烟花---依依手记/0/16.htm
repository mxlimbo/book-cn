读书时间

　　昨天下午，在大洋的银座书城里坐了很久，居然读完了一本书。抬头看表时，才知道不知不觉就过了三几个小时。安静的书城里，有隐约轻灵的钢琴在伴奏，音符在若有若无的飘荡。整个人就完全陷在深蓝色的帆布椅里，温暖而舒适，象陷在深蓝色的文字和情绪里。

　　其实还是挺喜欢一个人安安静静读会书的。管它是什么书呢，只要读的时候身心放松，有所收获就行。所谓的收获也并不一定要是什么人生的大彻大悟，或是深奥的哲理。哪怕是一本漫画小书，能在静读之际博你一笑，也算达到读书的目地了。

　　就象吃饭一样，读书原来也是要讲环境的。朋友请吃饭时常常会问，今天是去吃环境还是去吃东西。读书的时候，也许就应该问，今天是想读书还是想读心情。如果是出于极强的功利目的，读起书来也绝不会对环境挑三拣四。就象大学时代面对第二天就要到来的政治考试，谁人不挑灯夜读，悬梁刺骨？哪个怕蚊叮虫咬，酷暑难当？可如果要透过读书，来读读自己的心情，环境这个因素的分量陡然间就变得重了起来。

　　喜欢银座书城里的装修，其中让自己特别喜欢的，是在一张特别垫高的小塌塌米上，摆了张棕色的小几，小几上大大的木制棋盘上，似乎是下了一半的围棋，旁边古朴的屏风旁边伸过来几枝葱葱绿绿的竹条。对弈之景，让人顿觉满室书香盈盈。

　　所以在这点上来说，古人应该比现代人幸福，他们的读书环境是不需要造作的清净。无论是寒夜里小红泥炉里煨温的一壶清酒，还是一局残棋，都是能和读书这种活动配合得很默契的。论讲究，其实古人更讲究，而且比现代人更加懂得讲究。他们会焚香沐浴，在明月当空，清风飒爽的窗前合书而吟；或是在静静的雪夜，似乎四下里只能听到雪飘落在地上的声音，对着在在冷月下绽放的梅花，抱书闲庭信步。而现在，我们却要在什么刻意的书吧茶吧或是CAFE里去寻找这种刻意的情调。往往等找着了情调的时候，却已经没有了读书的心情。我向往的"独钓寒江雪"似乎只能是在梦里，或是某张古老发黄的画轴里。

　　只能舍弃一些浪漫幻想，自己给自己的读书时间和读书找点环境。一天劳碌后，回想起来，竟数不出今天的收获如何，失落之际，只好在清净的沐浴之后，在床头那摞永远也看不完的书里寻些慰藉。夜风吹起窗帘，有点凉，有零零星星的雨声入耳，缩在大大的抱枕和橙色明亮的床头灯光里，也挺好。偶尔身边放点音乐，仿佛是要为文字找点背景和色彩来配衬。那些温和的或是锐利的文字以各种方式冲击心灵，让人偶尔忍不住把它们放在嘴里，咀嚼橄榄一样，来回地读，反复地念。

　　突然会在这种时候感觉久违的幸福。过度依赖电脑和网络让人有时疯狂，心情起伏，就象抓狂。而眼睛在字里行间穿梭时、那种心与文字间所做的摩挲与在屏幕上、视觉神经所做的本能而富目的性的搜寻是完全不同的。看书的目光一定会比看电脑时要柔和得多，因为捧着厚厚的纸质文本，从手到眼，从眼到心，没有一处不是安静闲适的。

　　清静的时候，就地来一段一个人的读书时间吧。管它是什么书呢，抓起来就读。我们在读的其实不是书，而是书里的心情和在书里看到的自己。

依依
4/6/2001
