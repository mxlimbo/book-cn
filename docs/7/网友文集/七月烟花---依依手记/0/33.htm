逝水流年

　　早上突然静下心来重新看看以前写的一些东西，耳边是王菲的新专集里那首《给自己的情书》，浓浓的王菲味道，心里不知怎么就委屈起来。看着以前的文字，想起了逝水流年的心情，一样的委屈，是过了多少年还记忆犹新的。

　　今天看了陶然的文章，里面提到那首爱尔兰诗人叶芝的名作《当你老了》，情绪顿时飞回到曾经的草长莺飞桂花影疏的大学，仿佛又看到那个神气极了的讲文学欣赏的老师朗诵这首诗时的情景。老师当时三十出头，长得气宇轩昂，出口如流，旁证博引，天文地理，精彩极了。当时他的课是到堂率最高的几门课之一，特别是女生们，一早就占据最好的位置，以便彻头彻尾地欣赏老师的风采。老师声音雄厚，曾为我们吟唱就要失传了的配了乐曲的《诗经.国风》，苍茫之中，心灵不知跨越了多少沧海桑田。可当他用略带北方音的普通话读起叶芝来，却是另一种让人沉醉的男人的深情："当你老了，头白了，睡思昏沉，请取下这部诗歌，慢慢读，回想你过去眼神的柔和，回想它们昔日浓重的阴影； 多少人爱你青春欢畅的时辰，爱慕你的美丽，假意或真心，只有一个人爱你那朝圣者的灵魂，爱你衰老了的脸上痛苦的皱纹......"

　　几年后，当我站在大学课堂上，身份由一个台下的学生转变为一个台上的老师时，才开始真正体会到当年老师的那种成功，哪怕只是一点点。对于一个老师来说，最成功的时刻是看到台下那么多双或微笑或欣赏地看着你的眼睛和目不转睛的眼神，那种成功在于从他们的眼睛里，你能找到自己独有的智慧。常常在课时结束之前留下的讨论时间里，我会应他们的要求再给他们念点东西。虽然我竭力想象当年的老师一样去做，可我知道，那种极致，我是不能超越的，至少在自己的感觉里和回忆里是这样。每次给他们推荐某本好书，或是念起头一天晚上睡觉前阅读过的好文，他们都会心地笑，神情专注地看着我，我知道，在听到我的并不专业的朗读声的同时，他们都在用心思考。春天的微风从教学大楼的窗外吹进来，翻动手中的书页，慢慢地给他们念着《北大往事》里孔庆东写的那篇《47楼207》，就象念着自己的过往时光，学生们会心的笑声好多次打断我的声音。那时，仿佛时空错杂，我看到的是当年桂子山头的大学里，那个令女孩子们很倾慕的老师用很男人味的声音念着"当你老了"，边念边在阔大的老式木板地的阶梯教室前踱步，老旧的地板已经踩得发白，偶尔会发出一两声吱吱声，所有人的身边都满溢着窗外飘进的桂花清香......

　　转眼过了多久，都无暇去计算了。我们每天早出晚归，不胜重荷，可晚上上床睡觉前，老是想不起今天收获了些什么。老同学见面，一年比一年可以谈论的东西少，一年比一年更加实在。风花雪月的回忆逐渐开始蒙尘，开始还经常在些安静的夜晚拿出些精神上的东西来打扫，然后就开始漫不经心，直至忽略，直至重心转移向恶俗，转移向物质。虽然我们也常常说：有些东西平日不必想起，可永远也不会忘记。其实找这些理由的同时，很多的细节正在被慢慢抛弃，抛弃到有天回忆起来的时候，突然发现什么都只剩一个模糊的轮廓了。只是在偶尔闻到桂花香时，会有些似曾相识的感觉飘过，却再也抓不住一角。

　　前些时在编一份电子杂志时，忍不住用了一篇文章，因为其中写到了玛格丽特·杜拉斯的那本书-《情人》里那句著名的话："我已经老了，有一天，在一处公共场所的大厅里，有一个男人向我走来。他主动介绍自己，他对我说：我认识你，永远记得你。那时候，你还很年轻，人人都说你美，现在，我是特地来告诉你，对我来说，我觉得现在你比年轻的时候更美，那时你是年轻女人，与你那时的面貌相比，我更爱你现在备受摧残的面容。" 那时的感觉倒和叶芝的这首诗是有异曲同工之妙。岁月远去，昨日重现。

　　星期日去华师大的伯父家。校园里一潭碧波在微凉的秋风中起皱，食堂前提着开水瓶拿着饭碗的人流熙攘，刚踢完足球黑汗直流的男生站在小卖部门口喝可乐，路边的电线杆上挂着的喇叭里播放着社团活动的通知。心里很多情结又开始作祟。虽然妈妈边走边说，你走在这些学生当中还象是学生呢，可是我知道，那样的岁月已经回不去了。中午吃饭的餐厅里正好有个60届历史系学生的聚会，里面坐的几乎全是白发的老先生老太太，他们的谈笑让我陡然地心生尊敬。虽然步态不再有当年的轻盈，两鬓早已染上风霜，但却是不知经历了多少故事后，他们才可以重新坐在当年的校园里，也许当年的有些同学已经永远回不来了......聚会结束时，他们簇拥着一位八十多岁的很有风度的老先生走出来，我想那一定是他们当年的老师吧，岁月在他的手上脸上都打上了老人斑的标记，可他戴着褐色的灯芯绒帽子，拄着拐杖硬朗地站在秋水边，微笑得很美。不知道他是不是也曾给心爱的人念过那首叶芝的《当你老了》呢？......

　　在城市暄闹的背景中，我们每天都继续着自己急功近利的生活，总希望明天就会有奇迹出现，总以为幸福就在不远的地方。然后是日复一日地失望，失意，失落，失去自己，失去宁静。只是在非常偶然的时刻里，才会出现一些转瞬即逝的奢侈感觉。有一年去武夷山，坐的是夜机。从灯火辉煌的福州起飞，下面是热闹的城市，机翼慢慢掠过繁华，进入山区，沉沉的夜色将一机人的心情包围。那是我唯一一次坐夜机。飞机的轰鸣显得心境反而特别平静。心里暗暗哼唱着喜欢陈慧娴的《夜机》，在别人的歌里体会自己的故事。舷窗外是黑暗的大山隐约勾勒出的线条，很苍茫。偶尔看下去，山区晚上零星的一两点很模糊的灯光，不知道那里是什么样的地方，有些什么样的人，说着什么样的话。以前曾经想象过有天如果坐夜班机应该会有的很戏剧化的心情，想象自己如何编织着寂寥。可那刻，凝视着黑暗，竟然可以什么都不想，事实上是什么也想不起，心静如水。人生难得糊涂，也难得真正的心灵安静。略经世事的疲惫之后，还远远没有老去，却已经可以在黑夜的轰鸣中感受到宁静，不知是喜是忧呢。

　　早上打开电脑时，顺便去看了看不久前远赴法国留学的堂妹的留言版，她已经把新近照的照片放了上去。那个曾经和我彻夜聊天的小女孩，如今站在艾菲尔铁塔下，站在塞纳河边，站在巴黎圣母院外，并没有太多想象中的兴奋表情。她在留言版上写道：刚到法国的那天，觉得一切和自己想象的也没有太多不同，所以心情平静。言语里没有太多去国离乡的惆怅，更多的是一种淡淡的超然。她的美好生活才刚刚开始，我已经忙不迭地想把所有可以让她心灵宁静的祝福都送给她。看着她在风中依着铁栏杆，大衣上飘动着白色的中国丝巾，远远的艾菲尔铁塔让我有了种想在高处感受清冽寒风的欲望。

　　黄昏的时候，弹起了《南泥湾》，钢琴声清脆如水。妈妈正坐在沙发上折衣服，她跟着琴声慢慢地哼唱，让我心里很温暖。我希望能象她那样，当老了的时候，终能心神宁静地和着琴声轻轻哼唱；更希望在冷冷的冬天，红红的炉边，我的爱人为我吟诵那首叶芝的诗......

依依　于10/25/2000　黄昏
