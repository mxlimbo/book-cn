秋天的约会

　　感觉到秋天，就是这一两天的事。秋天在风里，因为我是从风里先感觉到它的。这在广州好象不是件容易的事。

　　即使是在夏天，每个早上无论多早起来，上班前总是慌慌张张的。妈妈老是这么说，说等我出门要提前两个钟头准备都不够---似乎女孩子天生都是这样的，即使不化妆，也有收拾不完的东西往包里塞，而且总在连滚带爬地下了七层楼后，猛然想起还有东西没带。看着墙上的钟，长长短短的指针在挣先恐后地竞走，心里在打鼓。每天都是这样，匆匆系上鞋绊，下楼时总怕因为自己的慌张没系好鞋带而滚下楼梯。那天冲上人行道，突然间感觉到街道上的阳光变得淡起来，一股不同于夏日清晨的凉风卷住了整个身体，风从各个角度包围住身体里和脑海里，仿佛一夏的感觉就这么容易地被这阵风清洗掉了。当时一个念头突然涌上来，秋天来了吗？

　　秋天真的来了，来得人一点心理准备都没有。来广州这么久，一直都觉得在广州这种城市，季节是没有什么分界的，至少从没清楚地感觉到过。从小是在北方长大的，我衷心地沉醉与那种四季分明的气候，习惯于严冬后看到春天第一缕枝头新绿的惊喜，习惯于酷暑后的某个清早黄叶满街的略有伤感的秋雨。而这一切在广州的感觉是极其平淡的，甚至就要忘了这种情绪。可今年的秋天来的时候，我却看到了它，感觉到这个亚热带城市里秋天的第一缕沁入心脾的凉风。

　　清晨的时候，穿着长长的及膝睡裙，带着一晚深深浅浅的睡眠里还没完全醒过来的梦，浴室明亮的镜子前总是惺忪的脸，半闭的眼。喜欢将双手放在开始日复一日冰凉起来的水流下，长久地，让它徐缓从容地从我的指缝中滑下去，无声地落到骨色的洗脸池里，感受秋天的适宜而柔和的水流般的气质。掬起一捧水扑在脸上，仿佛看到自己的心象块透明的石子也浸在这清净微凉的水里。

　　今天是星期六，去报社编辑部坐班车回位于这个城市西边的公司上班。报社大门口是高大的走道，去年秋冬走进这里时，总是会有很大很大的过堂北风。刚进报社大门，一股强烈的穿堂风扑面而来，感觉似乎有点冷了。风里好象听到身后远远有人叫我的名字，回头看见心怡的天蓝色的半袖T恤被风吹得紧贴在身上。心怡赶上来第一句话就是，好象到秋天了哦。地下车库上的草地上种的一长排夏天的茉莉已经半人多高了，翠绿得整整齐齐的，远远看过去，枝梢头还有一朵两朵白白小小的花朵，香味已经由夏天里的浓郁转成淡淡的秋天风里的清香了。

　　我们都曾经在这个编辑部的大院里工作过，后来搬到了报社位于城西的印务中心。转眼就快一年了，时间过得真快。报社编辑部位于广州极热闹的老城区，在上下九和人民路一带，商业发达，市井繁荣，寸土寸金的地方。那个深秋开始，十几个人挤在一间二三十平米的办公室里，不知不觉度过了一整个广州的冬天，我还记得那种冬天的阳光暖暖地撒在办公室门口那架铁制的一踩下去就吱吱作响的楼梯上，同事们的工作台紧凑而整齐，楼下会不时飘上来刚刚印刷好要运往各处的报纸的淡淡的油墨味。中午坐在面积不大却温暖的餐厅的落地玻璃旁的餐桌前，看着阳光穿过玻璃幕墙的楼群泻下来，手里捧着一大碗暖暖的汤，墙上挂的是五六十代美国社会的黑白相片，心里很享受这样的秋冬时光。吃过午饭，回到小小的办公室，一部只有一个喇叭响的音箱里飘出王菲若有若无的歌声，还有同事们打帝国时代的浅笑声，ICQ里偶尔响起突兀而可爱的"啊哦"声。比起现在前卫而现代感十足的印务中心，我仍很怀念那段时间，偶尔有从当时一直共事过来的同事身上熟悉的香水味掠过身边时，这种感觉会越发强烈起来。因为很多时候很多种人生和经验就是这样在不知不觉中得到体会和变得珍贵的。

　　有人曾经很认真地对我说过，任何事情太执着都是一种错误，适可而止的喜忧才是好的，中国传统的中庸思想有时是可以显现出不一般的价值的。我却常常在某段时间里会沉醉在一种感觉或情绪里不得脱身，什么事都做不了，也许就是因为自己太执着了。这个秋天来临的时候，我沉醉在一些细碎的回忆里不能自拔。将什么都进行到底，结果也许是一片苍白。唯一的解决就是有一句没一句地听着新老情歌，自闭似的打开电脑的note pad，漫无目地漫不经心地在键盘上敲字，过一两个小时，回头看看自己记下的东西，象巫婆似的催眠自己，觉得那些符号里就蕴藏着自己心里最隐密的，连自己都不为所知的东西。当看完往往还不知道自己到底想什么，要什么的时候，只好作罢，把冰冷的可乐瓶贴在脸上，坐在转椅里发呆，想象一场可能的发生在秋天下午温暖的PIZZA HUT里的约会。

　　独自冥想，想起朋友昨天和我说，黄昏时去少年宫的溜冰场边看孩子们滑旱冰的情景。他靠着栏杆在场边看了很久，说当时风很凉，觉得秋天来了。孩子们在场里都玩得很开心，教练和家长们都在旁边呵护着他们的幸福。他说他真想也下去，在黄昏初起的秋天的感觉里，和那些孩子们一起玩玩溜冰，感觉一下秋风拂面。我说，下次一起去吧。找种适合的方式沉浸进这一场秋里。

　　秋天来的时候，我想起一件早应该做的事，就是重新挂起那幅因搬家而收起的爸爸书写的条幅，韦应物的《滁州西涧》，虽然它描绘的是"春潮带雨晚来急"的时节，但紧随其后的那句"野渡无人舟自横"的略略寂寥与凉意在现在这种时候来感觉，也应该是再适合不过了。

依依于　9/9/2000　午后
