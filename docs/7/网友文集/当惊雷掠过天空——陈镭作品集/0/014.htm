北方的八月

　　八月就要来了，这从未经历过的，北方的八月。

　　办公室里的白光冷冷地罩下来，象冰块将人凝住，凝在寒意未消的心底。生命突然在自我感觉的瞬间凝固。这是办公室的生活，没有季节，没有昼夜，没有大地上滚动着的生的音乐。

　　这是我的第一个北方的八月。

　　灰蒙蒙的天空下，是别样的沉重的土地。我却在从未生活过的古城里开始着新的一切。暮云乱卷，将斜阳印在巷尾檐头。而心中的愁绪，却如缕缕丝丝，渐上心头。

　　这是个崭新然而平静的八月。

　　在干燥的风里，北方似乎有一种强劲的力量，推动着人在躁动中向远方追寻，追寻属于自己或是大地的什么东西。尽管没有雨夜的花香，或是河流的气息－－那潮湿的、无所不在的气息，静静地扑来。然而这是另一种雨季,它似乎预示着新的生活已经悄悄地降临了。

　　人们表露着各色各样的表情。在阴雨和阳光中生活，在白天或夜晚走来走去，在一切漠然与爱恨的世界里活着。永久性地、这个或那个，不停地生育着。信仰着纵欲或禁欲的宗教，以此为生存的喜剧。而继续的繁衍下去。

　　从这一点上来看，八月无论在哪里都不会有什么不同的。

　　然而，对身在异乡的我，这终究是一个寂寞的八月啊。

2000年8月11日西安
