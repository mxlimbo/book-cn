言者无罪 闻者足戒

　　――关于科学与人文的几个问题

　　"同志们"的发言都很好。令我五体投地，佩服不已。"科学主义"原来如此可厌，"人文精神"原来如此可恶。在下首闻之。心内不由得暗自问自己，我是这种人么？

　　有三个问题似乎可供大家参考：

　　1、鄙见以为，所谓科学与人文者，究其根本，或者并不是面目狰狞的，相反却对某些事物更真正有所裨益。真正令人厌恶的，是毫无或者仅有一点科学与人文内涵的人四处叫嚷，以为面部贴金也。

　　2、退而求其次，"半瓶子水喜爱晃荡"，更何况以追求科学与人文的精神晃荡，在价值观上终究还有些形而上的东西，总比权势、金钱那样直接的赤裸裸来得有趣一些。这是个人的看法，一笑一笑。

　　3、任何人生活在世界上都有晃荡的自由，任何人有发表意见的权利，这一点，虽是形而上的说法，未免脱离实际。但作为理想，似乎也无可厚非。希望真懂的人不要蔑视假懂的人，更不要蔑视不懂而想懂的人，倒是不妨对一些不懂而自大的人多说说。

　　作为一个也喜欢常挂一点"科学主义"、"人文精神"之类不登大雅之堂内容的伪风雅者。我忍不住乱说几句，希望大家不要见怪。
