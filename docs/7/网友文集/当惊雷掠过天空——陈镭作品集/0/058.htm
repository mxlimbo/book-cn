世事岂能尽如人意

　　日本新首相小泉纯一郎的当选，与年初美国小布什竞选胜利一样，是一个对我国影响深远的行动。

　　世事岂能尽如己意，这话用来说明目前我国的处境，是很恰当的。

　　小泉的当选，或许意味着十年来日本内阁更换频繁、政局不稳的局面的结束。正如一些人分析的那样，泡沫经济后的日本，对政府、对目前主宰首相人选的自民党内部而言，都希望求"变"，事实上也正在这求"变"的形势下，导致了这少数派的小泉能够当选。

　　小泉的当选，从短期看，是自民党内最大的派阀——桥本派的失败；但从长期看，无疑是日本近十年来普通民众及一般政治力量对政府缺乏有魄力的"作为"的失望情绪的反应。更是一部分政治势力对"开拓"太慢，较为"守成"的这己届前任内阁的一个总的反映。

　　从小泉与桥本的个人背景来说，桥本内阁与美国的战略伙伴关系，由于桥本本人的西化色彩，似乎可代表日本历史上曾有过的"西潮"的一种现实的选择；而小泉虽曾负芨于伦敦大学，但作为昔日的防卫厅长官之子，日本的本民族的背景似乎更加浓厚，如果追溯日本近代的历史，可以发见，日本民族主义的政治势力，在大利益一致的前提下，也同样存在着"本土"与"西化"的双重选择。而双方各自的人才，在历史上亦曾多次争论，总的来说，在明治时期，是西化派占了上风，到一战后，民族主义的情绪强烈起来。日本内部思想的不同分歧，发展到二战后政治的实际运作中，即使在今天的日本政局中，也不难发现旧日的影子。这又验证了那句老话：历史总有惊人的相似。

　　经济的地位使日本早已成为一个大国，但随着泡沫经济的破灭，可以说，日本民族的危机感又一次伴随着这个破灭得到了各种形式的加强。相当的统治基础要求政府"变"，而且是快"变"。以往的几届内阁没有能充分满足这个要求。这直接影响到小泉的这次当选。

　　从此出发，小泉的内阁必然会以"变"为出路，只有变，才可使这一届内阁继续得到相当民众的支持。而这"变"将变往何方，变的力度如何，都关系到小泉内阁能够保证相当时间的执政时期的问题。这也是整个东亚国家所凝视着的问题。

　　无论从哪一点上看，小泉的上台都意味着十年来日本政局最深刻的变化，而这个变化是否将影响到整个亚洲的局势，目前还很难说。因为许多个人的言行，都会又相当的作用，即使在小泉个人来说，上台前与上台后，也肯定会有所不同。但可以肯定的是，在我们拭目以待小泉的下一步所作所为之时，必须作好一个迎接较差的中日形势的准备。
