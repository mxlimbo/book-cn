风生云起乔老爷

　　作为新中国优秀的外交家，乔冠华为人所熟知。

　　乔冠华一生功业斐然，那是伟人辈出的时代，在伟人的影子下，乔做到了顶尖大国世界中的强有力小国的最好的外交家。就这一点来说，他是值得纪念的。

　　然而在我读完这本《乔冠华文集》后，不由得想起了高中时读过的一本叫做《乔冠华与章含之》的书，那里面有乔冠华年轻时代写下的那些震动人心的文章的节选。准确、热情、感人。窃以为是最好的政论。

　　1983年，"乔老爷"从世界上消失了，而同时带走的，好像是一个在困难与狭窄的外交领域里所能体现的刚强自立的外交政策。这也许是外交史上一个时代的终结。

　　《乔冠华文集》搜集了1939年至1946年间乔冠华为香港《华商报》、《时事晚报》、《世界知识》杂志和《新华日报》撰写的文章及国际评论。这些文章能够激动人们的心，与当时波澜起伏的世界形势有着紧密的联系。

　　二战已经动员了整个世界，而风云突变的政治形势可以说是一日而千里，入侵者的铁蹄一波接着一波，欧洲与亚洲，从来没有联系得这样紧密。

　　民主的前进也与被奴役的鲜血结合在一起，建立了新的战前不可想象的理论与新的世界体系。从热战到冷战，人类历史上最大规模的战争后又迎来了最大规模的僵持。

　　就在这样的形势中，乔老爷的这些评论使香港的读者看到了世界的另一面，一个真实而不乏勾心斗角的卑鄙的世界，一个有战争有死亡也有誓言和坚定的朋友的世界，准确的预测是这些文章的共同特点，而这些预测之所以准确，与乔冠华个人的才学识见分不开，更与个人的天赋分不开。

　　评判豪爽的人与细心的人的优缺之处在于此人面对世界时所持有的视野，视野的宽窄不以个人的性格而定，乔老爷的个人性格偏于豪爽，但也同样具有外交家的细心。这豪爽的书生，能够以他那难能可贵的细心洞察国际间于尔虞我诈中那些细微的变化。

　　"自然，试一回首，时势全非。欧洲的捷克，奥地利，非洲的阿比西尼亚（现译埃塞俄比亚），已在法西斯的铁蹄之下呻吟；西方的西班牙，东方的中国正在和法西斯苦斗——由于几年来民主国家的不团结，人类在法西斯侵略下流的血已足以汇成长江大河，但是九折臂而成医，我们希望英政府，为挽救过去的过失，丝毫不犹豫地加强英苏关系，造成一条坚强的民主战线站立在东京——罗马——柏林集团之前，这不但关系大英帝国的利害，而且关系全人类的幸福。"

　　这是《乔冠华文集》开首的第一篇文章，题目是《国际新动向——论英苏接近》，在这文章结尾最后的短短200来字中，谈到了捷克、奥地利和埃塞俄比亚，还谈到了西班牙和中国，在局势尚未明朗时，已看到了大英帝国的利害在于何处。而文笔的流畅与气势的非凡，已看出这位当时的政论家，未来的外交家所拥有的宽阔视野和鲜明立场。这对一个外交家来说，是最重要的。

　　时下外交者，读到乔老爷的这些政论，真不知此类人其寿之木何时能拱矣。
