回顾历史 热爱山贼

　　山贼有很多种，有意思的山贼也大有人在，我比较喜欢那些在热闹的山寨里当大王的同志们。他们似乎比较可爱。在未来的世界里，山贼是不大可能产生了，每想到这一点，我就不免扼腕而叹息，毕竟，可爱的山贼能杀贪官，能有威慑的力量。

　　古今中外的山贼不少，比较出名的不多。英国有个宾汉哥，喜欢射箭；非洲阿拉伯这些蛮荒之地，不谈也罢。我中华泱泱大国，在这上头倒是从来不乏人才。成则为王，败则为寇的事多着呢。刘关张还有在芒砀山落草的时候，吴国水上的悍将甘宁在没转作正行前，也是黑道上的大哥，外号"锦帆贼"。只不过不是山贼，是水贼。

　　同志们对古代的山贼其实似乎不太热情，对近现代的山贼（尤其是文学小说中的山贼）就比较感冒。这让我很不服气。其实近现代的山贼有甚意思，希望大家一起来回顾伟大祖国的那些可爱山贼的历史。
