烟雨旧梦此心暖

　　时间一分一秒地过去，我喝了一杯光明牌的牛奶，看窗外的暮色把西部最大的城市一点一点的笼罩起来。我的版快签字了，但是我的手指却像是难以停止的起落机，一下又一下地向苍白的键盘上敲落下去。

　　有人说，在机械化的运动中人会失去思维的感觉，有时候我很相信这话。但是，时间的力量并不是无穷的，机械化的运动与麻木的感觉也并不会永远就这么继续下去。在少量的情绪后面，我们还是得想想生命并不仅仅是蒙古骑兵在草原上驰骋时的快意，还有白云苍狗会把变迁的世事告诉我们，告诉我们一切终将过去，而未来，虽然如此的不可知，但在每一个人回首过去的时候，会想起的，虽然只是生命中挺小的一件事，象在黄昏时分，烟雨间卧室窗外院子里栀子花飘来的淡淡香气之类的。

　　来吧，亲爱的朋友，想想人们是怎样享受生命的喜悦。即使外面会有天地翻覆，内心深处也是风雨交加，但只要想起那令你欢喜的一件小事，也许在蓦然中你就已经欢喜。
