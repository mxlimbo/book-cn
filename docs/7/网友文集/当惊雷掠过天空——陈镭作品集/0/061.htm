"新生活”与旧生活

　　没有一种新生活能够在旧生活的大本营里获得成功，宋美龄所努力的"新生活运动"也是一样。

　　宋美龄把推广"新生活运动"当作了一项政治事业，这是眼光，也是错误。她试图使国人在生活习惯和精神上"脱胎换骨"，不要让西方人"看不起我们"，并藉此让国人知道蒋夫人关心大家，这不免让人怀疑她努力的真诚程度。

　　"新生活运动"运动的背景却是数千年来根深蒂固的生活习惯和贫穷的广土众民，而当时的中国人，摆在第一位的是生存的问题，人民要在活下去的基础上，才谈得上改进精神生活。

　　最重要的一点在于，在一个与旧生活有着千丝万缕联系，甚至本身就是旧生活的大本营的环境里，新生活运动那不牢靠的基础，最终只能使这场运动恰如沙上作画，当时代的风雨一来，则宛然无痕矣。

（2001.03.20"边看边说"栏目）
