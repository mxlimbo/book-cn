(二)

　　第一次真正见到夜的时候,是杰去湖北读书一年后的事了.

　　夜拥有像他哥哥一般桀傲不羁的眼神,深邃的大眼睛把黑白刻画的分明.让我几乎找不出任何的杂质.当我没有表情,直直盯着她看时,她的嘴角会扬起甜甜的笑容.把她那大大的眼睛荡漾的更加清澈,也更加的不安.

　　我一直是个不安静的人.不喜欢和太安静太正规的人呆在一起.静止的东西会让我恐惧.他们的言论死板的令人沉闷.他们的沉默让人要疯掉.

　　当然,夜绝不是这样女孩.当她旁若无人的唱歌,然后,把笑声洒满整个屋子.我就知道,这个女孩和我一样,有着同样的表达方式,和极强的掩饰能力.不管她内心是怎样的在垂死挣扎,她依然会露出如花般的笑容.

　　只有活在同样世界的人，才能读懂,才能看到那痛苦已在内心无止境的沸腾着......

　　我有预感,我会和她相处在很好.就如我第一次见到杰时,执著的相信自己会不可自拔的爱上他.然后奋不顾身的奔向他,永远跟随着他如风般的身影.

　　"杰，今天我见到夜了。你的妹妹和你一样......"橙黄色的灯光，柔和的照在我温顺的脸上。我摊开纯白的信笺，把空白处一块一块的填满。

　　虽然，杰已经好长一段时间没有回信给我了。但我相信他，他一定是有事，不然他一定不会不理我的！我轻轻的在日记写着：

　　在这洒满阳光的午后,我在想:好心情的你,是否也该去打打球,或躺在草地上看悠闲的云.再细细呤听南来的风.可曾捎来我的消息.想我的时候就看看南方的天吧!希望你快乐.

　　1998年2月15号离开泉州

　　1998年3月15日接到你的第一个电话

　　1998年4月28号收到你的每一张信
