倾听自己

　　　最近夜无眠时会听到一组电台节目叫做"相伴到黎明"，其中有一档叫做"情感热线"。人往往会在遇上一些事时被一种情绪所左右，而死钻牛角尖，把自己逼得走投无路。这个时候总希望能有个人听听自己的心事。但往往身边的人又不是合适的倾诉对象，因为他们了解你，却又不完全了解你。于是诸如"情感热线"这样的节目似乎就充当了救世主向你招手。我不喜欢这样的节目，其实很多时候你需要的只是倾诉一下，而非真的需要什么人来"指点迷津"，更何况每个人的情况不同，他（她）们又真能指点出一个正确的方向来吗？他（她）们并不了解你的真实情况，不可能设身处地地为你着想。

　　大凡打进热线的，都是些十几至二十几岁的"迷途羔羊"。在我看来，打进热线的"羔羊"又面临了一场"任人宰割"的命运。主持人先听你大致描述一下你的情况，接下来，惨了！主持人抓住你描述过程中的一些矛盾点步步紧逼，一番狂轰滥炸的诘问，最后你会发现自己已经进入一个圈套，对方用你自己的原话给你的脖子上套了一个又一个套，你觉得口干舌燥，无法言语。临了，主持人再"亲切地"说："你说对吗？好了，我们就到这里好不好？"你能说不好吗？退出热线，你是否寻到了一个方向或看到了一线希望呢？我看未必，要看见曙光还要等天亮呢。有很多人又进入了下一轮的思想中矛盾双方的斗争回合。

　　我是一个极易钻牛角的人，可能跟遗传有关吧（千万别告诉我爸妈）。在近一期的《女友》上看到一个变通原则："退一步，不是永远的退却，是为了更好地前进。"平素总听到退一步海阔天空，细细思之，倍感受用。一时激动："这样的人生，活着还不如死了好！"，但又转念一想："好死不如赖活着。"人生的路长着呢，眼前的事可能只是人生长河中的一粒砂。没准第二天就有好事等着你呢（如果遇上捡钱的好事千万别忘了通知我也参一脚，我的"胃口"不大，请我吃顿早饭----三个包子即可！）

　　那么如何退这一步呢？自己再做思想斗争无疑会更矛盾，那么找一个对象倾诉，仅仅是倾诉，你需要的只是一个听众。认识的亲戚朋友不放心，怎么办？没关系，找一个你心爱的玩具，最好是有眼睛的，诉说的时候看着它的眼睛，因为它也在"盯着"你呢。这时你会觉得它在认真听，你便会勇敢地正视这双真诚的眼睛，不用担心诸如："我说了它会不会这样这样想，这样这样做。"；或者，找一盘空白磁带，按下录音键，然后请开尊口，说吧！那个转轴就是一双眼睛，它正冲着你眨呢："我在认真听啊，我会为你守口如瓶的。"其实这个诉说的过程一方面是你掏出了心里所有的话，包括你不肯面对亲朋说的话；另一方面在说的过程中其实也是你把自己放在一个宽阔，开放的环境中思考的过程。尤其是复听录音带，似乎是站在别人的角度看你自己的问题，你会显而易见地发现一些问题。

　　效果如何呢？试试就知道（怎么象是卖狗皮膏药的在做广告？如果不想讨打，还是赶快开溜吧~嗖~~~只见一道金光，咱快乐去也~~）
