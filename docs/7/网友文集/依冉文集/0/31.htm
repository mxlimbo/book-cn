失约的海


　　大约是从八月份开始吧，她一直不停地拨打那个似乎是永远不会再开机的手机号码。起初以为，日子久了她会厌倦，说实话，连我都觉得厌倦。然而，她始终没有放弃，象是成了一种习惯。

　　我并不了解她，八月份之前她住四楼，我住六楼。后来我们一起搬到了这里，住在同一个房间里。她除了整个人的比例看上去比我们小那么一两号以外，身上没有任何明显的特征，是一旦走进人群就很难找出来的那种。说白了，很普通。

　　然而有一天，当我放下手中的笔时，我发现自己在纸上写下了"这个美丽的女孩子"。

　　那一天，天相当阴沉，我坐在阴暗的房间里听她讲完了整个故事。就象她叙述过程中常用的一个形容词"很平淡"一样，故事相当平淡。象每个网恋故事一样地进行，甚至没有精彩的对白。

　　她不是一个很好的语言组织者，然而"很平淡"的背后又掩藏着多少，无人知晓。每当她停顿下来时，我能看到她脸上附着的痛苦。在她痛苦地闭眼又努力地睁开，艰难地吞咽口水的时候，我总是很不忍心地将目光移开。

　　那个人叫小艾，是个做房地产的生意人，和我们年纪相仿。可能是置身商场多欺诈的缘故吧，他似乎对一切都抱不可信的态度。然而丘比特的小箭总是会在你毫无准备的情况下射出，他们最终身不由己地陷入爱情。

　　有一次，因为朋友的欺骗他的生意失败了。他问："如果有一天，我失败了，一无所有，你会怎样？"

　　"钱是身外之物，只要人在就好，我们还有两双手。"

　　"那如果我破产而要坐牢呢？"

　　"我等你。"那单薄的身躯里包藏着怎样的一颗坚定的心啊。

　　我不知道这个男孩子如何看待生意场上的阴险狡诈与真诚的爱情许诺。

　　后来，他用自己以往的积蓄和房产抵清了两百多万的欠款，并和朋友开了一家房地产公司。再后来，那部电话就再也打不通了。非常现实而又奈的一切。

　　她常在网上期待他奇迹般地出现，常常被与他同名的陌生人扰乱了心情。所有的Email都石沉大海。

　　她用了"艾秋"这个名字等他。有人问她为什么起这个名字。她说自己喜爱秋天。对方问为何不用"爱秋"而选择了"艾秋"。她苦笑：秋是个丰收的季节，然而自己的爱情却欠收了。

　　每个故事都有它美丽的地方，这个故事也一样。我对大海没有过太多的憧憬，朋友寄来两张大海的照片，却把生动而博大的大海局限在静态而又四四方方的一帧图片里。然而当她讲起关于那个大海的约定时，我从她的眼里读到了一些我不曾感受过的东西。

　　"我现在背靠崂山，面对大海......"某个夜晚，电话里传来了大海的心情。

　　"真的？！那现在的大海一定很宁静吧？"她的声音里充满了对大海、美丽沙滩的无限向往。

　　"是的，非常。但很快风会推动波浪打破这份宁静。以后带你来看大海吧，享受大海的宁静，海风的轻柔。"关于大海的约定就在那个时候诞生了。

　　我不知道一个约定的保质期有多久，也曾听说女人的爱很长很长，到底能有多长我也不清楚，但我知道这里有一种似乎是无限期的等待正在悄悄进行。

　　终于有一天，我接到了一个找她的电话，声音里年轻的苍老告诉了我那个人是谁。

　　我静静等待他们谈话的结束。

　　她垂着疲惫的双眼，但眼神中却有一种喜悦隐透。他又失败了，目前正在努力筹钱还清债务。他说除了一身债务以外自己已经一无所有，所以也不愿再来打扰她。电话匆匆挂断了，他说晚上十一点半打来。

　　我问她准备continueornot，她答：continue。

　　然而，那个说要带她去看海而失约了的男孩子又一次失约了。于是，每晚十一点总有一个瘦小的身影抱着肩膀，在只有冷风过往的楼道里等电话。不知又该会是多久，其实我真的很想说：睡吧。

　　真希望那个男孩子能抽出点时间，带这个美丽的女孩子去看看她向往已久的大海，仅仅只是圆一个美丽的大海梦。

　　夜里梦见了海的女儿。大海，爱情，泡沫。
