猫在说话

　　她是一只猫，是故事中的主人公，她穿梭在这座城市的各个角落里。她的生活叫流浪。

　　本来她可以跟着主人一起走的，但她最终还是选择了从火车窗口蹿出来，并迅速潜入这座城市。猫，从此有了自己的生活，缘于一次抉择。

　　初踏上这座城市潮湿的地面时，天正下着雨。她逃离汹涌的人群，躲在一个安静干爽的角落里想着自己的决定。她有些兴奋，喵喵地叫了几声。这个城市里五颜六色的伞在雨中行走。

　　流浪的日子是艰辛的，猫从未想到过。当然，猫是坚强的。在这座城市里它学会了一切流浪该具备的，怎么说呢？应该说是流浪该具备的心理素质吧。

　　通常它会蹿上一米来高的垃圾桶去寻找一些食物。不论是偷窃或乞讨都是猫内心所摒弃的。然而她是一只流浪的猫，当寻找食物变得困难时，她也会发出几声凄惨的喵叫，用人类的同情心来换取一点食物，但她从不去偷。当然，并不是每个拥有食物的人都会给她食物的，有时候她甚至会挨打。

　　夏天天气晴朗，夜里她行走在建筑物的顶上，看一排排能燃亮夜色的路灯，看闪烁的霓虹，看闲散的行人和一辆接一辆汽车驶过。那个城市缺少满天的星星。当她醒过来时，阳光正暖洋洋地盖在身上。房顶是个好地方，她喜欢这里，因为没有人会打扰她。

　　通常冬天或雨天猫会选择在某个建筑的地下室里栖息。当迎着风雨觅到食物之后，她便躲在那里用自己的体温来烘干湿漉漉的皮毛。有时，她会梦见她出生时那间温暖的房。

　　尽管上顿不保下顿，但猫还是一天天长大。猫冷漠地行走在自己的日子里。她冷漠，因为没有人看到猫会露出愉快的笑容。她没有朋友，或许她不懂得孤独，或许吧，猫向来是我行我素的。

　　冰冷的雨水，飘零的黄叶，又一个冬天来临了。冬对猫来说是难熬的。一整天，她没有找到一点食物，还被打伤了后腿。行走在一个住宅小区里，路灯光影下猫看到自己一瘸一拐的影子。一只破塑料袋在空中和寒风撕扯。夜已深，她没有心思找个地方休息。来到空阔的马路上，没有了建筑群的遮挡，寒风企图侵占每一个毛孔，肠胃又一次因为饥饿而收缩。猫突然觉得自己失去了面对生活的能力，失去了爱的能力，站在现实而又尖刻的生活中泪流满面。没有人见过猫流泪，因为，她想，泪是从心底滚落的钻石。一片叶子随风划过脸颊，不知是它要为自己拂去泪水，还是仅只是一种讽刺。

　　猫失去了力气，失去了知觉，因为寒冷，饥饿，还有痛。钻石滚落的时候划伤了心。

　　猫梦到了自己出生时的那间温暖的房。

　　睁开眼，猫停留在一间四壁洁白的房间里，空气暖烘烘地裹在身上。她又闭上了眼睛，让梦延续。

　　猫听到了一个声音："它和咪咪挺配的，可以把它们打扮成一对双胞胎的样子。"不是梦。她抬起双眼，看到了另一只猫，那只叫咪咪的猫。咪咪头顶上的毛被一只红色蝴蝶结束起来，身上裹着一件红色的衣服。她们象是两种毫不相干的动物。

　　猫得到了食物，每天，每顿。然后，她被加上了和咪咪一样的装束，她觉得自己象个怪物。更可怕的是她的脖子上还套着一根绳子，出门时主人总牵着绳子的另一端。而咪咪则跑前跑后，要么靠在她的身边，要么附在主人的脚边。有时她们被束缚在主人的胳膊里，她觉得异常压抑，而咪咪则显得轻松自如。

　　猫最喜欢的是洗澡，那个时候身上的套子全部被卸下来。

　　不需外出的时候，猫和咪咪便呆在房子里。她们没有共同的话题，咪咪对她的流浪生活并不感兴趣，只热衷于趴在沙发上睡觉或谈论猫食和装束。猫明白了为什么咪咪没有套在脖子上的绳子。

　　空气中饱含着她残缺的呼吸，这样的生活只剩下呼吸。

　　猫逃走了。

　　马路上是湿的，风在耳边划过。路灯光影照着猫的影子长了又短，短了又长。猫说："至少，我是自由的。"
