阳光落在地板上

　　大雨的夜晚，在校园里捡了一枚鹅卵石，它在雨水的洗礼下褶褶闪亮，那一刻心静的只有大雨和石头的存在。把鹅卵石紧紧地握在手心里，希望自己的体温能将它暖热。然而很久很久，雨水干去，石头仍是冰凉的。这世间总有那么多无奈，那么多我们无力改变的无奈。

　　除夕夜打了一个电话给陈，只说了句："过年了，打个电话问个好。"就没词了，那种客套令人别样的尴尬。从他的声音听来，他的身体状况暂时良好。分手了还再惦记些什么？

　　小茜打电话来问："还爱他么？"

　　"不了。"言毕吃惊于自己没有经过细细思量就给了答案。才明白，很爱很爱一个人，又很无奈地不能去爱了时，原来这份爱就可以慢慢地，慢慢地从心上剥落，可以残忍地没有声音。于是想起了不知从哪里看来的几句话："人生是一本书，有些人注定了不能陪你走到最后一页。在某些章节里你可以爱得刻骨铭心，而在以后的章节里你必须走得云淡风。"于是明白了那样的刻骨铭心与这样的云淡风轻。

　　认识董是在陈住进医院后不久，那时我没有任何能联系到陈的方式，只能在每个深夜里写许多无处投递的信。厌倦了聊天室生活很久之后的那一天，我以guest的身份进了聊天室，带着极其沮丧的心情。和董聊起来时他要求我换名字，于是我用了"寂静跌落"，足够低调。

　　"寂静跌落？接下来即将是激情的到来。"居然被他解释成了这样。

　　我象练打字一样地说话，他只是静静地倾听，我诉说了生活中的种种不快，唯独没有提起真正令我担心沮丧的陈。对于他自己，他给了简单的介绍："我当过兵，出过国，坐过牢。你介意么？"

　　"每个人都有自己无法改变的过去，重要的是你如何去面对现在和未来。有些人把曾经的苦难变成了财富，而有些人没有。希望你会是前者吧。"

　　接下来的日子我再没有去过聊天室，他也就无声无息地消失了。网事如风，风过无痕。而我也终于能把自己的信寄出去了，能让陈在病房里读到我的信，是那些日子里我最大的欣慰与幸福。

　　两三个礼拜后居然收到了董的Email。字里行间透落出一个真正经历过生活种种的人的知性与理性，没有太多生活经历的人讲出来的道理是空泛苍白，缺乏说服力的。出于一种好奇心我开始了和董的Mail往来，他总是洋洋洒洒，能写几K到十几K。

　　我告诉了他有关我和陈的一切，他为我们真心祝福，然而最终我们还是分手了，haveto。

　　董开始向我表明心迹，他写了一首诗给我，我狡猾地给了回复："诗写得好不好都是美的，美的是心与心愿。"我不愿去体会那些句子中蕴含的意味，"人们把生活美化，诗化了，然而生活就是生活，是平凡甚至残酷的，美的只是心灵与眼睛。"

　　董和我通了电话，整整聊了十个小时。他说到什么时哭了我怎么也想不起，我找不到任何能安慰人的字眼。或许有些时候人哭泣是不需要直接理由的，我是说成人。一种不易觉察的寒意在心底久久酿集，在那么不经意的一刻，人就选择了脆弱，所有无法言传的苍凉悄然升起。这样的哭泣无需劝慰。

　　"依冉，你要宽待生活，宽待自己。"

　　可是，如何去宽待，如何去把握这样一个"度"，才不致使宽待变成了放纵与随波逐流。

　　"我相信，凭你的悟性，我能把我所有的思想讲给你听，让你完完全全地了解我。我会非常地珍惜你，我自信有这个能力给你幸福，幸福是可以把握的。"董说。于是我记起了他曾经说过的一句话："在你看来喜多于悲的结局，或许在他看来恰恰是悲大于喜的。"也曾渴望一个和我彼此深爱的人能陪我走得很久，很长，很远，可是我要走的路是漫长的，而生命，经不起太长的等待。

　　董和老哥一样给了我一些东西要我去思考，我都不能急于知道答案，因为生活中有许多事情单凭想象是无法真正理解其内在的深刻蕴意的，唯有经历过一些什么之后才能明了。成长需要付出代价。

　　逃开所有的人，一个人生活，总是忘了吃饭，与其说是忘记，不如说是放弃。不停地擦地板，收拾房间，然后站在寂寞的房间里，看着地板上自己模糊的影子，象是正在经历一场等待，而自己又在等待什么呢？

　　叔叔从北京打来电话，我们算是忘年之交了。他感慨良多，我一言不发地听着，末了说："你的心理承受能力太差了。"

　　"是的。"他叹口气，"不过你了解的还是几年前的我，现在的我经历了从国有企业到私营企业的转变，在商场中打拼了几年，我的身上渐渐多出了几分'兽'性。"一个近中年的人放弃了中国人民银行的稳妥工作去外面打拼的确需要勇气，谋略与野心。

　　"是的，从我们这个家庭中出来的人，受了太多人性化的教育，有时似乎也不很合时宜。"

　　"有没有想过去国外读研？你是几个孩子中综合各方面素质，灵性与悟性最好的一个，我希望你能把握好自己的未来。回头我和你小叔商量一下这件事。"

　　"可是，叔叔，我，我不想到国外去，我只想在国内生活。"

　　"不想去国外？那你自己有什么打算？"

　　"您知道，我现在正在用初中二年级的文化基础接受高等教育，我已经很努力了。我打算毕业后花三年功夫去考研，若不能考取便作罢，那时我也已年近三十了。我也只能作这样的短期计划了。"我已经厌倦了疲惫，同样是疲惫，我宁可选择自己的方式。

　　"我明白了，也好。既不要沉浸在过往的得与失之中，也不要总沉缅于对未来的幻想里，认真充实地过好现在的每一天，日子久了，把这一个个每一天，象珍珠一样的每一天串起来，那就是你的收获。"

　　夜里，一个人不敢睡，不敢在房间里走动，很晚了才躺在沙发上。关了灯，有千百万个幻影在空气中狂舞，或无声地机械移动，象是被装进了一只塑胶袋里，不敢动，亦不能呼吸。如果不去征服黑暗和恐惧，它们便会将我俘虏直至天亮。猛地坐起身来，光裸着脚板踩在冰凉的地板上，摸索到门边开了灯。灯光闪亮的那一刻，一滴汗水从额上滑落，砸在地板上却没有声音。

　　陈说，你的幸福不在我这里；

　　老哥说，找到自己的位置，生活它不相信眼泪；

　　雨萱说，每个人都是一本书，他不适合你，去寻找适合自己的那一本；

　　董说，你上的那座山不是光秃秃的，它巍峨美丽，但你不得已下来了，于是你应该去寻找有属于自己的风景的那座山；

　　二叔说，希望你能把握好自己的未来；

　　小茜说，放弃他，并不代表放弃了爱情。

　　走钟在耳畔虚空地响着，时间已不复存在；地板上有阳光落进来的影子，无声地静止着。我在地板上的方格子里踱步，从这个格子到那个，又挪进另一个。哪一个才是我的位置？
