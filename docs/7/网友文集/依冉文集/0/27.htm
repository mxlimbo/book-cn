寂静跌落

　　趴在窗口，指尖轻轻抚着手腕上的一根红线绳，已经戴很久了，是为了一个人。我给它起了个名字，叫"觊昕"，和"系（ji）心"诨音。"觊"是希望的意思，"昕"是太阳升起时的意思。

　　为自己买了一条真丝的方巾，是正红色的，象一团轻盈的火，没有毕毕剥剥，却正在忘情地燃烧。将丝巾系在窗棂上，每晚将夜色点燃，美丽火焰在轻柔夜风中绚烂。

　　系心的那个人在远方。我望着夜色出神。

　　有时候，想，到底是怎样了呢？他骄傲，而又自卑。

　　有那么几天，莫名其妙地总会哭，泪水不分时间、场合地奔赴第一线。有时会是在城市熙熙攘攘的人群中，有时会是在寂静的小屋里，有时还会在梦里；有时清晨，有时夜晚。其实，我知道，是因为他。

　　拨了他的电话，心砰砰地跳，失去了规律，每次都会这样。

　　"病了。"他说："每年这个时候都这样，总有些日子要受罪，发烧和咳得比平时更厉害了。"

　　他那里的秋来得早。我能听到他喉管上下跳动的声音。如果，我能分担一半他的痛苦，我想，我不会害怕沙哑。

　　悄无声息地流泪。他找话题。

　　织了一件毛衣。那个城市的秋风正在肆虐，他没有什么衣服，不会去买，也没人帮他买。不想矫情地说什么"温暖牌"，只想为他织一件普普通通的，甚至针脚不怎么匀称的毛衣。只想他快乐，这个冬季不会冷。

　　生活是多彩的，爱情也应该是有声有色的，我总想。

　　每周写信给他，有时是文字，有时是卡通画，有时会是别的。他不回信。我仍会继续写。有时，心里难受会写信告诉他，记得他喜欢一首刘若英的歌"透明"：我的心最透明，每天为你抹干净。久久玩味，知道我的心很透明。每封信到了接近末尾时常会笔峰一转讲些轻松快乐的事。哪怕走到尽头，也希望他轻松快乐。

　　午夜，坐在宁静的小屋里，折两个小纸人儿，贴在一起，它们伸出双臂迎着阳光并行站立。食指轻触的地方，两只手托起一颗心，伸向外侧的两只手共同托起一轮七彩的虹。伴着清晨第一楼阳光，所以的心情投递给远方。

　　毛衣寄到了。我说："不大合适吧？"

　　"挺好，真的很好。"

　　"穿了没？"

　　"穿着了。下雪了。"

　　"树叶都没了吧？"

　　"嗯。"

　　想告诉他自己穿着他的毛衣拍了照片，又没说。其实，很想知道他穿着我织的毛衣会是什么样子的。

　　南方城市的秋还很柔和。银杏叶的边缘开始泛黄，白玉兰的叶子仍坚持地绿着，仿似要做这一季最后的留守者。为他采集了许多叶子，它们绿得可爱。用叶子做成了简简单单的卡片，除了生命，绿色，还有阳光和希望。有一组叶子令人爱不释手，每一片嫩绿的叶片上毫无规则地漾开来一片嫩黄，第一次把"黄"和"枯"分了开来。秋季里鲜活的嫩绿嫩黄。

　　一直很害怕空寂。可以不求回报，但却不能没有回应。常常会有丝丝的痛，在心里最柔软的部分。我知道，如果追逐他就会跑，如果停下来他则会回头静静看着你。于是，我选择了停顿，凝视，学会了不提问题，不给要求。

　　有时觉得他心上的结正在被自己慢慢地，慢慢地松解。可是又总会重复一个梦：自己站在一个空荡在大舞台上表演，然而台下空无一人。每次惊醒，心，好饿。

　　打了一个电话给远方。疲惫穿着破旧的拖鞋形影不离地跟着他，工作的劳累令他无力招架，他虚弱地喘息。临了，他叮嘱："你也早点休息吧。"就这么一句简单的话，心里竟有太阳升起，阳光散开来的感觉，阳光穿透浓密夜色撒下来。我总会问吃了没？穿暖和点，多喝开水。这就是生活，我想，是最简朴的关怀与爱。

　　坐在地板上，月光在窗口荡漾，红丝巾在柔风中轻舞。如果这一刻，他出现在门口，从背后走过来拍拍我的头："快起来，地上凉！"他不会把我抱起来，因为他没有足够的力气；或者他用脚踢踢我的屁股："起来起来了，当心着凉。"呵呵，好象都不象他，他会怎么做呢？我对着自己笑。

　　抱着膝头，倦意如一匹轻纱轻轻覆盖上双眼。他走过来，轻唤我的名字，动了嘴巴，可我什么也听不到。

　　冷。醒了。站起来，手指和脚有些麻木。走到窗前，解下红丝巾，轻轻绕在指间。

　　夜风，无声地拂过手臂，那一团红色随着它飘洒开来。

　　慢慢下落。

　　寂静无声。
