百部酒

【配方】百部根100g 白酒500ml

【功用】润肺下气，止咳杀虫。

【制法】1 将百部根炒后捣碎，放入干净的瓶子中； 2 倒入白酒浸泡，密封； 3 7日后开启，过滤去渣，装瓶备用。

【用法】每次15～20ml，每日3次，饭后徐徐慢饮。外用时，用百部酒涂患处。凡脾胃虚弱者，及大便溏泄者均慎饮本酒。

【药材功效解析】百部根：味甘苦，性微温，功效温润肺气，止咳，杀虫。
此酒主治因百日咳、肺结核、气管炎等引起的咳嗽气急；外用可杀虫虱、疥疮，阴道滴虫等。

【方剂来源】《本草纲目》
