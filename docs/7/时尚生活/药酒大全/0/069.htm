大豆乌蛇酒


【配方】大豆100g 麻子仁100g 乌蛇（去头尾皮骨）12g 白酒1500ml

【功用】祛风通络，攻毒，消肿止痛。

【制法】1 将上述3味药材，相和令匀，放甑内蒸； 2 熟后，去掉甑底物，将酒淋入甑中，等酒热再淋，按此淋7～8遍； 3 将酒倒入瓷瓶中密封，冷却，备用。

【用法】不限时，随量饮之，常带酒气为好。

【药材功效解析】乌蛇：即乌梢蛇，又名青蛇、黑花蛇，其性平，味甘咸，有祛风湿，通经络之功效；与白花蛇同而性善无毒。
大豆：有黄豆、黑豆两种。黄豆健脾中，润燥消水；黑豆活血，利水，祛风，解毒。所以方中以用黑豆为佳。
麻子仁：即火麻仁，润肠通便，使热毒有退路而出。
此酒有祛风通络、攻毒、消肿止痛之功效，主治热毒风肿，日夜焮痛。

【方剂来源】《普济方》
