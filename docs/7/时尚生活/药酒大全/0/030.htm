丹参杜仲酒


【配方】杜仲30g 丹参30g 川芎20g 江米酒750ml

【功用】补肾益肝，活血通络。

【制法】1 将上述药材一同捣碎细，装入纱布袋内； 2 放入干净的器皿中，倒入酒浸泡，密封； 3 5日后开启，去掉药袋，过滤装瓶备用。

【用法】不限时，将酒温热随量服用。

【药材功效解析】杜仲：性味甘、微辛，温，有补肝肾、强筋骨的功效；
丹参：性味苦，微寒，有活血祛瘀，凉血消痈，养血安神的功效；
川芎：性味辛，温，有活血行气，祛风止痛的功效；
此酒补肝肾，强筋骨、养血活血、祛风通络；主治肝肾虚，精血不足，腰腿酸痛，久痛络脉痹阻。

【方剂来源】《普济方》
