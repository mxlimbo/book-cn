虎骨酒


【配方】虎胫骨1具 糯米2.5kg 曲适量

【功用】壮筋骨，强腰脚；祛风寒。

【制法】1 将虎胫骨炙黄打碎，放入锅内煎汁； 2 将糯米放入，用曲适量如常法酿酒； 3 密封50日，待酒熟，压去糟，装瓶备用。

【用法】随宜饮用。

【药材功效解析】虎骨：味辛性温，辛散温通，去风定痛，强筋健骨，可降低毛细血管的通透性，有良好的镇痛和镇静作用。
糯米：补中益气。
曲：消食健胃。
此酒主治肾虚骨弱，少腹冷痛，行走无力，肩臂疼痛。适于治疗风痹疼痛，四肢拘挛以及肝肾虚寒，腰脚软弱无力等症。

【方剂来源】《普济方》
