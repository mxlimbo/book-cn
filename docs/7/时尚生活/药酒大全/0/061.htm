大风引酒


【配方】制附子16g 枳实20g 泽泻20g 陈皮20g 茯苓20g 防风20g
大豆100g（原方有甘草） 水1000ml 米酒1000ml

【功用】祛风利湿止痛。

【制法】1 将上述前6味药材一同捣细碎，装入绢袋内； 2 放入干净的器皿中； 3 用米酒、水一同煮大豆； 4 煮取750毫升，分作3份。

【用法】每日服1份，3份为1疗程。

【药材功效解析】制附子：回阳补火，散寒除湿止痛。
防风：祛风胜湿。
泽泻、茯苓：利湿健脾。
枳实、陈皮：理气化湿。
大豆：活血，利水，祛风，解毒。
此酒有祛风利湿止痛的功效，主治风湿痛，遍身胀满。因制附子之毒性，可小量服之，观察效果，再酌加至治疗量。

【方剂来源】《柳森可用方》
