复方红花苏木酒


【配方】两背针（皮）2.5kg 50％食用乙醇7500ml 苏木2.5kg 红花500g高粱酒7500ml

【功用】活血法瘀，消肿止痛。

【制法】1 将上述药材放入干净的器皿中； 2 倒入食用乙醇、高粱酒浸泡，密封； 3 15天后开启，滤过去渣，装瓶备用。

【用法】每次20～30ml，每日2次。外用适量，擦患部至有灼热感。孕妇、有内出血者忌服。

【药材功效解析】此方选用活血法瘀之品，治跌打损伤所致的瘀血肿痛，温散润利，肿消痛止。
此酒主治因跌打损伤而引起的瘀血肿痛。

【方剂来源】《中药制剂汇编》
