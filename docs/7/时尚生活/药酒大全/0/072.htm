复方黄药子酒


【配方】黄药子1.2kg 海藻1.2kg 浙贝母900g 白酒7000～8000ml

【功用】软坚散结。

【制法】1 将药材一起研成粗末，放入干净的器皿内； 2 倒入白酒，隔水加热，不时搅拌至沸； 3 取出，连酒带药倒入坛中，趁热封闭； 4 静置10天，滤过装瓶备用

【用法】每次服10ml，每日3次。

【药材功效解析】黄药子：性味苦，寒，散结消瘿，清热解毒，凉血止血。
海藻：性味咸寒。有消痰软坚，利水之功。
浙贝母：性味苦，寒，化痰止咳，清热散结。
此酒有软坚散结之功效，性寒清热，味苦泄降，消瘿解毒，化痰散结，主治地方性甲状腺肿。

【方剂来源】《串雅内编》
