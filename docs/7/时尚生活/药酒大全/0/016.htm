刘寄奴酒

【配方】刘寄奴10g 甘草10g 白酒100ml

【功用】破血通经，散瘀止痛。

【制法】1 将刘寄奴与甘草捣碎； 2 放入锅中，加水200ml，煎至100ml； 3 再倒入白酒100ml，煎至100ml，去渣备用。

【用法】1次温服，孕妇忌饮用此酒。

【药材功效解析】刘寄奴：性温味苦，有破血通经，散瘀止痛的功效。
甘草：补脾益气，缓急止痛，调和药性，使刘寄奴破血之力缓而持久。
此酒主治血滞经闭，产后瘀阻腹痛，折跌损伤，以及创伤出血等症。

【方剂来源】《简易效方》
