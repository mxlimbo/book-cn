内消浸酒


【配方】鲜仙人掌250g 羌活30g 炒杏仁30g 白酒1000ml

【功用】清热解毒，消肿。

【制法】1 将上述3味药材捣成粗末，装入纱布袋中，放入干净的瓶子内； 2 倒入酒浸泡，密封； 3 7日后开封，去掉药袋，过滤备用。

【用法】每日10～15ml，将酒温热空腹服用，临睡前按上量再饮1次，以消为度。

【药材功效解析】仙人掌：性味苦寒，行气活血，清热解毒。
羌活：祛风胜湿。
杏仁：除肺热，治上焦风燥，利胸隔气逆，以消瘰疬之病因。
此酒有清热解毒、消肿之功效，主治风热毒气，结成瘰疬。

【方剂来源】《普剂方》
