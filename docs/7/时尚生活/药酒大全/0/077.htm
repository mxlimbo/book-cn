两皮酒


【配方】海桐皮30g 薏苡仁（炒）30g 五加皮30g 独活30g 牛膝30g防风30g 干蝎（炒）30g 杜仲30g 生地90g 白酒1500ml

【功用】祛风，解毒，止痛。

【制法】1 将上述9味药材一起捣为粗末，装入纱布袋内； 2 放入瓷瓶中，倒入白酒浸泡，密封； 3 秋夏3日，春冬7日，开封，去掉药袋，过滤去渣，装瓶备用。

【用法】每次10～20ml，每日3次，饭前将酒温热服用。重患者不拘时候饮之，常令酒色相续。

【药材功效解析】五加皮、海桐皮：祛风湿，通经络，强筋骨。
干蝎：息风止痉，解散结，通络止痛。
防风、独活、薏苡仁：助海桐皮祛风湿。
杜仲、牛膝、生地：协五加皮强筋骨。
此酒有祛风、解毒、止痛之功效，主治热毒风结成疡肿，痛不得安。

【方剂来源】《证治准绳》
