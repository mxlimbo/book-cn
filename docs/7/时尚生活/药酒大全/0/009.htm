当归地黄酒


【组成】生地黄50g 当归尾50g 黄酒500ml

【功用】补血，止血。

【制法】1 将上述2味药材一并捣成粗末； 2 放进锅中，倒入黄酒，在火上煮1小时； 3 然后过滤去渣，装瓶备用。

【用法】每次20ml，每日3次，将酒温热空腹服用。

【药材功效解析】地黄：滋阴养血，治疗吐血，衄血，妇女月经不调，崩漏腹痛等症。当归：补血和血，调经止痛，当归尾有止血的作用。但当归性滑，生地粘腻，所以脾胃虚寒，湿阻中满及大便溏泄者应慎服。此酒有补血止血的功效，主治产后血崩，腹痛等症。

【来源】《民间验方》
