史国公药酒


【配方】虎胫骨（酒浸1日，焙干酥炙）62.5g 炙鳖甲62.5g 川牛膝62.5g
枸杞子156g 干茄根（蒸熟）250g 当归62.5g 萆Z62.5g
防风62.5g 秦艽62.5g 松节62.5g 蚕沙62.5g
羌活62.5g 无灰酒（优质黄酒）12000ml

【功用】壮筋骨，祛风湿，舒筋活络。

【制法】1 将上述药材装入绢袋内，放入干净的器皿中； 2 倒入黄酒浸泡，密封； 3 10日后开启，去掉药袋，过滤装瓶密封，在干燥阴凉处存放

【用法】随量服用，每日早晚2次，不可间断。

【药材功效解析】当归、枸杞、鳖甲：滋养阴血，活血散结。
虎胫骨、羌活、萆Z、防风、秦艽、牛膝、松节、蚕沙、茄根壮筋
骨：祛风湿，活络止痛。
此酒养阴血，壮筋骨，祛风湿，，活络止痛，主治半身瘫痪、四肢顽麻、骨节酸痛、风寒湿痹等症

【方剂来源】《证治准绳》
