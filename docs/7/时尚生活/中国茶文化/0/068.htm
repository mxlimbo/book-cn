武夷岩茶

　　武夷山素有"奇秀早于东南"之誉。自古以来，就是游览胜地。武夷山所以蜚声中外，不仅仅由于它的风光秀丽，还在于它盛产于武夷岩石茶"。武夷不独以山水之奇而奇，更以茶产之奇而奇。"

　　武夷岩茶所以深受人们赏识，在于它的品质优异。优良品质的产生，不外乎一有得天独特精谌的特作工艺。

　　弄武夷岩茶的香所气馥郁，且似兰花而深沉持久，锐则浓长，三青则幽运"。滋味浓醇清活，生津回甘，虽浓饮而不见苦涩，茶条状结，匀整，色泽青褐亮呈"宝光"。叶面呈蛙皮状沙粒的点，俗称"蛤蜡背"。泡汤后叶度"绿叶红镶边"分三分红七分绿。
