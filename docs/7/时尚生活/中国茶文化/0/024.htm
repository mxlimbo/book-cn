龙王茶（龙舌）

　　龙王茶产于浙江省安吉县西天自然保护区龙王山龙王庙一带。

　　产地以海拔800米以上的深溪、石岭、南坞山、桃树湾所产为佳。那里悬崖峭壁林立，山高林密雾浓，溪水沟壑纵横。茶树王生于常绿阔叶林或针叶林下，腐殖质层深厚，土质肥沃，空气洁净，日照期短，富于紫外线照射。不使用化肥，不需浓药灭虫，也无人为污染，堪称"三无"茶。茶生其间，芽叶肥壮，白毫披覆，持嫩性强，加之精巧的采摘和加工工艺，在色、香、味、形上别具一格。

　　鲜叶于清明后10天采一芽一叶初展，经摊放青锅筛分与摊凉荤锅割末装筒等工序加工而成。

　　成品外形光扁平直，色泽翠绿，略显毫；香气高鲜持久，汤色杏绿清澈明亮，滋味鲜醇爽口回甘；叶底成朵肥嫩匀齐、嫩绿明亮。
