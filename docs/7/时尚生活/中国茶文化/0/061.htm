武夷肉桂

　　肉桂，又名玉桂，据《崇安县新志》载：肉桂树最早发现于武夷山慧苑岩，另说原产武夷马振峰上。为武夷名丛之一。

　　据行家评定，肉桂花的桂皮香名湿，佳者带乳味，香气泡尤存，冲泡四五次仍有余香；入口醇厚回甘，咽后齿颊留香；汤色橙黄清澈，叶度黄亮，红点鲜明，呈绿叶红镶边状；条索匀整，吧结卷曲，色泽褐绿，油润有光，部分叶背有青蛙皮状小白点。

　　如今肉桂已有兴旺的后代，分布于水帘洞、三仰峰、马头岩、桂林岩、天游岩、晒布岩、响声岩、百花庄、竹窠、九龙窠等峰岩石之中和九曲溪畔，面积达1700多亩以上。投入市场备受赞誉，被评为全国名茶之一。
