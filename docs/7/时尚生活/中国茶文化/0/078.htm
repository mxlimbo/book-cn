康砖和金尖

　　康砖和金尖是四川南路边茶的两大花色品种。原状产于四川雅它、宜宾、江津、万县等国营茶场（厂）加工筑制，年产量近万吨。

　　康砖茶每块净得0.5公斤，金尖每块净重2.5公斤。康砖为圆角枕形，大小规格为17X9X6厘米。金尖也是圆角枕形，大小规格为24X19X12厘米。

　　康砖、金尖的品质特点是；康砖外形色泽棕褐，香气纯正，滋味醇和，汤色红浓，叶底花杂较粗；金尖外形色泽棕褐，香气平和，滋味醇和，水色红亮，叶底暗褐粗老。

　　康砖主销川西和西藏，以康定、拉萨为中心。金尖销区的康定为中心，并转销西藏边远地区。
