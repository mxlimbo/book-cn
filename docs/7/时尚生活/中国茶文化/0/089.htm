君山银针

　　君山，又名洞庭山，为湖南省洞庭洞中的一个小岛位于岳阳城西15公进而处。

　　君山银针属芽茶，因茶花树品种优良，树壮枝衡，芽头肥壮重实，每斤茶银茶约2.5万个芽头。君山银针风格独特，岁产不多，质量超群，为我国名优茶这佼佼者，其芽头肥壮，紧有实挺直，芽身金黄，满披银毫，汤色橙黄明净，香气清纯，滋味甜爽，叶底嫩黄匀亮。根据芽头肥壮程度，君山银针产品分特号、一号、二号三个档次。

　　君山银针于1956年国际莱比锡埔览会上，因其质量优良，被誉为"金镶玉"，并赢得金质奖章。其售价也创我国当今名优茶之最。
