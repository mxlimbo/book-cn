宁红工夫

　　宁红工夫是我国最早出口的工夫红茶之一，该茶以独特的风格，优良的品质弛名中外。

　　宁红产区山多田少，土地深厚肥沃，气候温和，茶树生长根深叶茂，内含化学在分丰富，造成宁红工夫优良的自然品质。宁红输出量最盛时期达30万箱，锋苗挺拔，色乌略红，光润；内质香高持久似祁红，滋味醇厚甜和，汤色红亮，叶底红匀。

　　宁红除散条形茶外，还有一种束茶龙须茶。

　　龙须茶产于修水县漫江宁红色村，成茶身披红袍，叶条似须而得名，制作龙须茶的鲜叶原料，多系一芽一叶至一芽二叶，芽叶大小、长短要求一致，鲜叶经萎凋、揉捻、发酵、初干以后，进行"扎把"。

　　龙须茶具有独特的艺术风格和特异的色、香、味、形、多次被评为江西省的优质名茶。
