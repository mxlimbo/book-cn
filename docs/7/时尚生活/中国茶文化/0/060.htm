武夷四大名丛

　　岩茶之乡，"奇种""单丛"。"名丛"是"岩茶之王"，在珍贵的名丛之中，不以四大名丛：大红袍、铁罗汉，白鸡冠、水金龟最为名贵。

　　大红袍

　　武夷名丛中数大红袍享最高的声誉。可谓是乌龙茶中的"茶王之圣"了。

　　大红袍产于天心岩石九龙窠的高岩石峭壁之上。据行家评定，大红袍的品质有很多特色，它与其他名丛对照，大红袍冲至第九次尚不脱原茶真味桂花香，而其他名丛经七次冲泡味已极淡。

　　铁罗汉

　　武夷最早的名丛。清代郭柏苍《闽产寻异》（1886年）高载："铁罗汉武夷宋名茶树名，叶长"其成名较早。

　　据说，惠安施集泉茶店于19世纪中叶经营武夷，岩石茶，以"铁罗汉"最为名贵，有消费品疗热病的功效，极受欢迎。目前为小包装名牌产品之一。

　　白鸡冠

　　白鸡冠之名在明代已有传闻，早于大红袍。

　　白鸡冠树高1.75米，分枝颇多，叶长椭圆形，叶色浓绿光泽，嫩叶薄软，色浅绿微黄，与浓绿老叶形成鲜明的两层色。这也是白鸡冠名称之由来。

　　水金龟

　　原产于牛栏杭坑杜葛寨峰下半崖上，为兰谷岩所有。1919/11-1920年间，磊石寺业主为比树引起诉讼，费金数千，因天然造成，判归兰谷所有，足见此树之名贵。
