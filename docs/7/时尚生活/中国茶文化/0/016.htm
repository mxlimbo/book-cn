贡字龙井

　　产于浙江省杭州市西湖区。

　　龙井茶区分布于峦叠翠，名胜古迹比比皆是的西湖山区。名城、名湖、名山、名茶四者相得益彰，冠绝天下。

　　主地气候渐和，雨丰水沛，年平均温度过16.1℃≥，10℃活动积温暖5067.7℃，极端最低温度9.6℃，年平均相对温度82%，年降水量1400毫米左右。土层深厚，质地疏松，多为沙质壤土，含石英沙砾多，通气透水，土壤中含有效磷高，酸度适适中。茶树生长于溪漳涧谷地，山溪盘绕，终年不涸，小区气候极利于茶树的生长发育。

　　鲜叶于清明前采摘，以一芽一叶初展为标准，长1.0-2.0厘米，经摊放一青锅理条整形"回湖"（二青叶筛分与摊凉）一辉锅筛分归堆贮藏"收灰"等10道工序加工而成。

　　成品形似碗钉，扁平光滑，尖削挺秀，大小匀齐，芽毫隐藏，色泽糙米色，润匀鲜活；汤色碧绿明亮，香气鲜嫩高长，滋味甘鲜醇和；叶底嫩绿，匀齐成朵的待点。
