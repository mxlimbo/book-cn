铁观音

　　铁观音原产福建省安溪县。别名红心观音或红样观音。铁观音原是茶树品种名，由于它适制乌龙茶，其乌龙茶成品亦名铁观音。

　　优质铁观音茶条卷曲、壮结、沉重，呈青蒂绿腹蜻蜓状。色泽鲜润，砂绿显，红点明，叶表带白霜，这是优质铁观音的重要特征之一。铁观音汤色金黄，浓艳清澈，叶底肥厚明亮，具绸面光泽，泡饮茶汤醇厚甘鲜，入口回甘带蜜味；香气馥郁持久有"七泡有余香"之誉。

　　铁观音茶一向为闽、粤、台湾茶客及海外侨胞所珍爱，此茶一经品尝，辄难释手，现在其美名遍及全国各地，消费量在不断增长。
