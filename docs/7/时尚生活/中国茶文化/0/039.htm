信阳毛尖

　　要产于信如县车云山、集云山、天云山、云雾山、震雷山、黑龙潭和白龙潭等群山峰顶上，以车云山天雾塔峰为最。人云："师河中心水，车云顶上茶。"

　　茶区年平均温度15℃，年降雨量1100毫米，无霜期223天，年相对平均湿度77%，年日照19402180小时；土壤肥沃，结构良好PH值4.0-6.0。

　　鲜叶采摘一芽一叶或一芽三叶初展。经生锅熟锅理条初烘摊凉复烘等工序加工而成。

　　成品条索细圆紧直，色泽翠绿，白毫显露；汤色沮绿明亮，捍气鲜高，滋味鲜醇；叶底芽壮、嫩绿匀整。素以"色翠、味鲜、香高"著称。
