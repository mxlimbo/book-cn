陆虎

陆虎（ROVER 又称罗孚）原是英国一家古老的汽车公司。“陆虎”是北欧的一个民族。“陆虎”商标采用了一艘正在行驶的海盗船图案，张开着的风帆象征着公司乘风破浪、所向披靡的大无畏精神。同时，也表示该车就像陆虎民族一样具有勇猛、顽强的气概；文字商标是“ROVER”。

1884年创建的陆虎（罗孚）公司，初期以生产自行车为主。随着汽车的出现，1904年开始把主要精力用于小汽车的生产上，但直到1946年才开始开发令其后来闻名遐迩的四轮驱动汽车。1948年4月，兰德·陆（LAND ROVER）在阿姆斯特丹车展上，获得了极大的成功，从此走向世界。该公司几易其主，现是德国宝马旗下的一员，它是世界著名的越野汽车公司。陆虎公司创造性地将四轮驱动和舒适有机地结合起来，生产出轿车和越野车合二为一的全新车型。其中，该公司生产的“览胜”、“发明者”、“卫士”被称为陆虎公司的三杰，是世界公认的四轮驱动汽车的典范。
