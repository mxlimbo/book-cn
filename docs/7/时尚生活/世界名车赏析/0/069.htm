朗伯基尼

朗伯基尼（LAMBORGHINI）牌赛车是由意大利的朗伯基尼公司生产，其商标是一头公牛，它浑身充满力气，正准备冲击，寓意该公司生产的赛车马力大、速度快、战无不胜。

1962年弗鲁西欧·朗伯基尼在摩迪纳创建朗伯基尼股份公司，主要生产高级赛车。1965年，该公司在都灵车展上，推出的朗伯基尼350GT赛车，就像力大无比的公牛，冲击力十足，令人瞠目结舌。1987年，朗伯基尼公司与美国的克莱斯勒公司合并，克莱斯勒的技术和朗伯基尼的工艺结合后，使 “朗伯基尼”赛车的知名度大增。

朗伯基尼（LAMBORGHINI）

 

朗波吉尼（Lamgorghini）,1988,Countach:这部最初于1971年在日内瓦汽车博览会上亮相的康德“外星车”，没想到日后为整整一代汽车设计大师所模仿，为一代车迷所追踪，为各界所关注……她的诞生使“朗波吉尼公司成为永恒”，其设计风格奖锲型车的设计水平推向极限。这部像火一样热情的名跑车，据悉是世界上唯一一辆永远保值的跑车。本照片展示的是1987年克莱斯勒公司控股该公司后，修改设计的25周年样车（共生产了400辆）。

　

朗波吉尼（Lamgorghini），1988，Countach:尾部装饰繁杂的“雄鹰”（左图）；超低的车身，可能是世界上最低的跑车（右图）；宽大的两座车箱令车手如同在驾驶一辆飞机。　

　

朗波吉尼（Lamgorghini），1982，Jalpa：朗波吉尼·节拍，比康德车晚推出整整十年，是在原来博通设计的Silhouette车型基础上发展起来的。康德车太脱离消费阶层，只具有观赏性，而节拍车就是弥补这一缺陷而生产的，它配备3.5升V8缸发动机。马力在255匹，可跑到242公里/小时，足以让车迷体会一下朗波吉尼的风味。

　

朗波吉尼（Lamgorghini），1991，Diabol:以“革新对保守”向法拉利F40直接挑战的朗波吉尼·德宝，采用了更大马力的发动机：5.7L V12 EF1,马力492匹，最高时速 325公里/小时。这部以观赏性为主的90年代的德宝比80年代的康德跑车明显上了一个档次，去掉了尾部大而宽的扰流板，车板更趋流线型，圆润、洗练，方灯换成了流行的圆灯，但万变不离其宗：她是迷人而独特的一对朝上开启的开门，像一对翅膀，永远成了朗波吉尼的代名词，该车被誉为“世界五大名跑车”。
