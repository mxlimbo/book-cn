宝马

1916年，工程师卡尔·拉普和马克斯·佛里茨里慕尼黑创建了巴依尔飞机公司；1917年，该公司改名为巴依尔发动机有限公司，这就是巴依尔公司简称（BMW）的来历；1918年，公司改为现在的名称—宝马汽车公司。

“宝马”普译为巴依尔。由于宝马公司是以生产航空发动机开始创业的，所以商标中的蓝色为天空，白色为螺旋桨，这是“宝马商标的第一大特点；第二大特点就是“宝马”汽车的散热器（车鼻）中间那两个金属方框进气格栅。

BMW（宝马）是公司全称“Bayerische Motorenwerke AG”的缩写。“宝马”采用了内外双圆圈的图形，并在双圈圆的上方，标有BMW字样的商标。整个商标就像蓝天、白云和运转不停的螺旋桨，喻示宝马公司渊源悠久的历史，既象征该公司过去在航空发动机技术方面的领先地位，又象征公司的一贯宗旨和目标：在广阔的时空中，以最新的科学技术、最先进的观念，满足顾客的最大愿望，反映了公司蓬勃向上的精神和日新月异的新面貌。

宝马（BMW）图片赏析

宝马（BMW）,1992，3系：宝马车系------技术完美代表。经过八年创新设计而推出的318I(L4)和325I（L6）及敞逢车像一股狂风席卷世界车坛，被国际最权威的汽车杂志评为“每年最佳十大轿车之一”该车将宝马的革新传统和深沉的品格融为一体，将富有动感的车型和浪漫洒脱的个性融于一炉。宝马为现代界车回答了这样一个问题：保值、实用、安全和纯粹的乐趣。

　

宝马（BMW），1992，5系：宝马的天才工程师们决心将优雅、动力、精确、安全、豪华融于一部车内，使之成为车中的精品，她即是5系：525I（L6）和535I（L6）。在5系身上可以体会到宝马的技术精湛，应验了人们常言的“开宝马、坐奔驰”。最近，宝马又将V8缸发动机装在5系车上，推出530I，540I两款，外表俊逸潇洒，动力澎湃，加速性能令您叫绝，实在令人喜出望外。

　

宝马（BMW），1992，7系：被誉为“世界五大名房车”的宝马7系车：750iL(V12),740Il(V8)和730iL(L6),体现了宝马的设计理念：驾驶乐趣，创新极限，该车被公认为是近五年内打造得最佳车辆之一，能驾驶一部配备V12引擎的宝马是爱车之士的追求目标。在最高档批量生产的豪华房车中，宝马是唯一可以和奔驰S级轿车相抗衡并且超过她的车种。该车属二级配备。坐在宝马车中欣赏莫扎特的音乐可是一种美妙地享受。

　

　

宝马（BMW）,1992,850Ci:与宝马750iL装备同样发动机的850iL(V12)轿跑车是替代6系的新产品，但比750iL体型要小、灵活。该车利用空气动力学设计车型，车身线条流畅，车头扁小、底盘低矮、升降式隐型大灯、大斜度的车背，所以这些将宝马这部跑车的驾驶性能推向极限。850iL-----上世就对波尔舍911构成了极大的威胁，与其共同争夺欧洲市场。

宝马（BMW），M1：当宝马运动车部于1972年成立时，就开始研制用于参加F1比 赛的超级跑车。该M1跑车只生产了450辆，其中39辆装备成纯种跑车，因而非常吸引收藏者。整车跟850iL的外型接近，但比850iL更带有跑车味。那是因为吸取了意大利跑车设计精华之故。

　

宝马（BMW）,1987,M3:被誉为“披着羊皮的狼”的限量生产跑车M3,采用原宝马F1发动机（直列四缸、16阀，双凸轮、195匹马力），最高时速达237公里/小时。目前已推出敞逢车，倍受车迷的喜爱。
