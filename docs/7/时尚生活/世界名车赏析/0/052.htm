大众

大众汽车公司的德文Volkswagenwerk，意为大众使用的汽车；图形商标是德文Volkswagenwerk单词中的两个文字V和W的叠合，并镶嵌在一个大圆圈内，然后整个商标又镶嵌在发动机散热器前面格栅的中间。图形商标形似三个“V”字，像是用中指和食指作出的V形，表示大众公司及其产品“必胜―必胜―性胜”。文字商标则标在车尾的行李箱盖上，以注明该车的名称。大众商标简捷、鲜明、引人入胜，令人过目不忘。

大众汽车公司是德国是年轻的、同时也是德国最大的汽车生产厂家。使大众公司扬名的产品是“甲壳虫”式轿车（由波尔舍设计），该车在80年代初已生产了2000万辆。它启动了大众公司的第一班高速列车，紧随其后的马球、高尔夫、奥迪、帕萨特、桑塔纳等也畅销全世界。


大众·高尔夫

大众(Volkswagen)图片赏析

大众(Volkswagen),1947,Beetle:波尔舍博士毕业的理想是设计一辆价廉物美的市民汽车，1934年1月17日，他向当时的希特勒政府提交了一份生产“大众汽车”的建议书，具体包括：（1）最高时速为100公里/小时；（2）油耗：8L/100Km；（3）最大爬坡角度30主公（4）车身净重650Kg；它既是“甲壳虫”的原型经过反复试制，第一辆“甲壳虫”于1938年7月3离开了生产线，开始了光辉的生涯。而德国大众公司也是在此基础上发展起来的。1985年10月17日是“甲壳虫”车面世的最后日子，总产量达20，638，802辆。大众公司转而生产较为现代化的高尔夫车系。虽然“甲壳虫”车已停产约十年，但人们对“甲壳虫”还是难以忘怀，尤其是她那“傻乎乎”的外型。

　

大众（Volkswagen），1993：德国大众车系中目前最具代表怀的车种是高渔产夫敞篷车（Golf中图）和科拉多G60(Corrado,右图)跑车、两款的外型匀保留了传统的欧洲峁，在欧洲一直畅销不衰。高尔夫敞篷车基本上可代表标志205敞篷车、罗浮敞篷车等的风格，而科拉多G60跑车却能代表像富豪440。雷诺21等这一类跑车的风可格。
