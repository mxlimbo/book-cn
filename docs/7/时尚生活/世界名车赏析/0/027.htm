爱希

爱希（AC），1966，Cobra:该车的许多精华之处被后人愉快地采用着：前大灯非常像波尔舍911，排气管的布置与道奇Viper有异曲同工之处，车体中部用于散发发动机余热的出风槽又留有法拉利的痕迹……

　

爱希（AC）,1966，Cobar 127S/C:该部排气量7.0升、500匹马力的每一代跑车1966年面世时，就创下了赛事纪录，彻底奠定了AC公司在公众心中的崇高地位。几乎巾近地面的底盘在运动时，产生了极强的吸地力，像肌肉一样有内张力的车体给人们一种动感和坚契不拔的意志，侧向不锈钢双排气管与车体的不锈刚排气管与车体的不锈钢装饰件混然一体，光滑的轮能一地面百分之百地接触，以形成极强的抓地力，利于稳定操纵，这是一部与其它英国赛车全然不同的“野马”。
