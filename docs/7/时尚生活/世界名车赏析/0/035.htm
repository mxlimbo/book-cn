本田

本田宗一郎于1946年创建本田技研工业公司（即本田汽车公司），并用自己的姓氏作为公司的名称和商标。

“H”是 “本田”汽车和 “本田”摩托车的图形商标，是 “本田”日文拼音 “HONDA”的第一个大写字母。本田汽车商标中的字母 “HM”是 “HONDA MOTOR”的缩写，在这两个字母上有鹰的翅膀，象征着 “飞跃的本田的技术和本田公司前途无量。”

“人和车，车和环境的协调一致”是本田公司的发展方向；动感、豪华、流畅是本田公司的一贯风格；设计动力澎湃、低耗油、低公害的发动机是本田公司的技术目标；靠先进而实用的设计、卓越的制造质量和相对低廉的价格，吸引更多顾客是本田公司的宗旨。

“H”商标，这个世界著名商标，是本田公司立业之本，是本田公司成功之魂。

1960年， “H”商标首次在S500跑车上使用， “H”商标伴随本田赛车，在1965年墨西哥举办的世界F1汽车大赛上一举夺冠，并在第二年的F2汽车大赛上取得胜利；更令本田辉煌的是在1967年取得了11冠军。

1969年，本田公司为突出鹰的形象，而使用了纵长的 “H”商标。1980年，为了体现本田公司的年轻、技术先进和设计新颖的特点，决定使用形似三弦音箱的 “H”商标，该商标把技术创新、团结向上、经营有力、紧张感和轻松感体现得淋漓尽致。

本田·雅阁

本田(HM)图片赏析

本田（Honda）,1993:挑战精神是本田在当今世界车坛驰骋的推动力量。透过高科技，本田每年都推出新品，引导未来汽车发展的趋势和消费，其非凡的精神为轿车赋予了崭新的定义。根据国际汽车界预测。世界将流行微型车。这里展示的是四部未来将准备入市的微型慨念车，体现出人类渴望改善交通工具，追求自我实现、保护环境的强烈愿望：一人驾驶，方向盘中置的EP—X,1.0升3缸，长3.7米（上右图）；配备齐全的房车FS—X,3.5升V6缸，长4.9米（上左图）；轿跑车EV—X（下右图）；配有电子地图的迷你车 F—R(下左图)。

　

本田（Honda）,1993,Civic CRX/del Sol;采有本田NSX 跑车发动机技术的CRX，虽然只有1.6升四缸，但却能输出160匹马力，最高时速达211公里/小时，高速行驶下具有极好的贴地感和平稳性，它那奇特的金属活动顶可收进尾箱，既可以作敝篷又可作车顶，表现出日本汽车公司出奇制胜的设计能力。

　

　

本田（Honda）,1993,Civic Feri:又一款装载本田DOHC VTEC发动机（SiR,VTIi,Eti）的二能配备房车-----------法里奥。

　

本田（Honda）,1993,Vigor:本田的Acura 在美国推出的豪华经济车，其排气量为2.5升，马力可达190匹，为世界流行的房车款式。

　

本田（Honda）,1993 Integra:本田的Acura在美国推出的毫华经济车。VTEC发动机、排气量在1.8升，马力可达170匹，体态优美，车灯象奥迪的慨念车车灯，照射力强而大。比较符合美国人口味。

　

本田（Honda），1994,Prelude:本田的Acura在美国推出的两门轿跑车。VTEC发动机，排气量在2.2升，马力达200。由于是跑车，车体流线型较好，车头扁而低，车灯与保险杠融为一体，最有趣的是尾部的设计，车尾从两侧开始与车后盖形成一光滑弧面，煞是好看，比奔驰190E2.6还要夸张。目前，该车的金属基和陶瓷复合材料发动机正在研究中。

　

本田（Honda）,1994,Accord:最新本田“雅歌”。排量在1.6至2.2升的中档房车历来是世界汽车市场竞争的焦点，使这一档的车款精品辈出，如：三菱·Galant,标志·405，大众·Passat,马自达 ·626，日产，蓝鸟，欧宝·Vectra等都是各领风骚的好车，但雅歌却独树一帜，在美国这个世界最大的汽车市场上，能连续5次击败所有的对手，以每年 35万辆的销售额独领冠军的殊荣。她保留 了本田的一贯风格：动感、豪华、流畅，车内配备完备，油耗较低，价廉物美，是一部耐人寻味的良驹。另外，本田还推出1994年最新二款车：Ascot 和Domani。与雅歌一样，楚楚动人。

　

本田（Honda），1994，Legend:本田推出的最豪华房车。配备了一台3.2升24气门SOHC V6发动机马力190匹，发动机安置为纵置式，以取得最理想的60：40车身重量分配，四轮双叉骨式悬持系统，令车身架构坚稳程度达到最高水平。考虑细心、周到的安全设施，令车主放心、踏实，车箱内的每一细节，与您融会贯通（二级配备）。本田“里程”的诞生，确实是本田公司的一座里程碑。
