法拉利

意大利是世界艺术的主要发祥地之一，而现代汽车艺术又出了个 “法拉利”。

“法拉利”也曾译为费雷丽，是1929年由安素·法拉利创建的一家小型的汽车公司，该公司以法拉利的姓氏而命名，用法拉利（Ferrari）作为文字商标，用 “腾马”作为法拉利中国的图形商标。

自从法拉利使用 “腾马”作为赛车的吉祥物后， “腾马”的魅力和 “法拉利”跑车独特的造型，使其超出了机械工艺的范畴。在世人看来， “法拉利”不是现实的汽车了，而是超越时空的艺术品。

意大利素有 “高性能汽车王国”之称，作为菲亚特公司的王牌 “法拉利”跑车，无疑是王冠上最美的钻石。精彩的 “法拉利”有着魔鬼般令人晕眩的震憾力；它那刚劲和难以言喻的经典 “红头”造型，淋漓尽致地散发出运动与建筑的美；加速与极限速度会让你真正体会到风驰电掣的滋味，体会到 “法拉利”就是速度之神的化身。

汽车外形尺寸（长×宽×高）：4250毫米×1900毫米×1170毫米

发动机形式和排量：V8、3.5升

最大功率和扭矩：280千瓦、260牛·米

最高车速：295千米/ 小时

加速性能：从0加速到100千米/小时用4.7秒

法拉利（Ferrari）图片赏析一

 

法拉利（Ferarri）,1989,Mythos：法拉利公司第一次想要展示它未来准备生产的概念车，于是在1986年东京汽车博览会上，这部风格与其它法拉利迥然不同的名车问世了。在她身上找不到F40和Testarossa的痕迹，倒有点象本田的NSX，但比NSX更圆润、潇洒。所有的装饰件：大件、雾灯、刹车灯、保险杠、扰流器、徽标、双排气管都极为得体地安置在车体内，以减小风阻力。机械部件大多数采用了Testarssa的产品。这部车无须多夸，她静静地细说着自己的一切。

　

法拉利(Ferarri),1984 Testarossa:进入80年代后，法拉利出现了许多问题，有技术的，但主要是销售方面的。由于排放量大大超过美国环保的标准，因而很难打入美国市场。在这样的情形下，一种全新的车款应运而生了，她就是1984年在巴黎汽车展上一鸣惊人的Testarossa。这部车可谓无人不喜，无人不爱，她的侧向进风槽及尾部细槽可谓匠心独具，被称为最“美丽的车尾”。车尾与车头的联系非常自然、流畅，结构布局合理、紧凑，大块面红色色块与黑色修饰相得益彰，V12发动机的排放量完全符合标准。总之，她是美丽的、完美的，很难用形容词来描写她，就请您慢慢欣赏吧！　

法拉利(Ferarri),1990，348TS:法拉利的精神总是超前的。每当一种新款问世时，多数人都说没老款好，可是当另一新款问世时，人们才开始崇拜老款。348TS也没逃此运。该车是法拉利真正保持商业化的最后款式，348TS排气量为3.4升，8个缸，32汽门，300匹马力，它比Testarossa 尺寸要小，比Mondial座位要少，但对多数车而言，驾驶和乘坐都比较舒服。国外专家称“这辆车是法拉利边接未来和现代的桥梁”。当您在驾驶NSX时，您知道自己驾驶的是一件艺术品，而但当您在驾驶法拉利旱，您知道自己的确在驾驶法拉利。　


法拉利（Ferrari）图片赏析二

法拉利（Ferarri），1947，166：1946年恩左·法拉利决定退出他效忠长达26年的爱快罗密欧跑车了，开始生产冠以他本人名字的世界级跑车。1947年第一辆法拉利问世了！从此开始了其推崇备至的生涯。由于在赛车界的名气，加上所推了的车款风格独具美感，爆发力强，马力均匀，一直深受车迷的欢迎。这部车在赢得第一次比赛胜利后，被一位富有赛车手购去，获得到了购买第一辆法拉利的荣誉。该车的外型酷似荷兰的DookervoortD10

　

法利利（Ferarri）,1967,330P4:20年以后生产的这部4.0升V12缸、价值约900万美元、造型奇特的超级法拉利跑车会使您对名车价格有一个全新的概念吧！她曾赢得24小时Daytona 赛事完军和Le Mans的第三名。仅生产3辆。　

　

法拉利 （Ferarri）,1981,512BB:这部外型与288GTO类似的大红色法拉利被人誉为“播上翅膀的射手”，是法拉利新设计风格的起点。

　

　

法拉利（Ferarri）,1971,DINO:DINO是法拉利唯一独生儿子的呢称，正是由于法拉利出生，才迫使他放弃较为危险的赛车职业而从事跑车生产，最终形成庞大的法拉利帝国。这部半敞篷车在1965年巴黎车展上首次亮牌，推出的目的是想批量生产产，赢得利润，并参加二级方程式赛。所以外型上偏向实用，制造工艺较为简单，但绝不失其雕塑般美丽的身躯。

　

法拉利（Ferarri）,1985,GTO:1984年在日内瓦车展上首次参展。GTO在意大利语里表示已生产一定数量、可以参加超级大赛的车种。几乎每一款法拉利的问世，都凝聚了法拉利Pininfarina先生的心血。该车配备大量的电子设施，发动机马力强颈。截至1987年，当271辆GTO离开Maranello 车间时，法拉利开始着手开始超级跑车-------F40，以庆祝独立制造跑车生涯40周年。

　

法利利（Ferarri）,1987，F40:“世界五大名跑车”之一。1987年正值法拉利90高龄之际，迎来了冠以他本人名字命名的世界级跑车生产40周年，没有人在生产能享有如此高的荣耀。他成功了，而且获得了极大的成功。法拉利前后8次获得一级方程式大赛冠军，近100次的Grands Prix 冠军，9次Le Mans 冠军和数千次的战绩。这部在杂志上、报纸上、挂历上等被采用得最多的名车，外型是那么好看、协调、完美、以致于在该车上增加或一部分都会是个缺陷。视觉上平衡的高大的扰流器，光滑的车身、钢结构车架，Kevlar材料车体，F1赛车的悬持系统，双槽进风口等等都属于独创。马力478匹，最高时速321.8公里/小时。尽管受到来自波尔舍959的挑战，但法拉利在跑车界的辉煌成果甚是可观，仍然傲视全球，无人能出左右。据悉，将接替F40历史使命的新款法拉利F130将于明年面世。
