别克

别克（BUICK）商标中那形似“三利剑”的图案为其图形商标，它是别克分部的标志。它被安装在汽车散热器格栅上。图中那三把颜色不同（从左到右：红、白、蓝三种颜色）并依次排列在不同高度位置上的利剑，给人一种积极进取、不断攀登的感觉；它表示别克分部采用顶级技术，刃刃有余，是无坚不摧、勇于登峰的勇士。

1903年5月19日，大卫·别克在布里斯科兄弟的帮助下创建美国别克汽车公司，但不久公司就陷入了困境。后在威廉·杜兰特的资助下，公司才开始兴旺起来，并创造出汽车年产量居美国第一位的业绩。1908年，以别克汽车公司为中心，成立了美国通用汽车公司。当通用汽车公司扩大后，别克部成为通用汽车公司的第二大部门。它主要设计制造中档家庭轿车。别克牌汽车的销量在通用汽车公司内居第三位。

别克（Buick）,Skylark:这两部都是云雀牌，一部产于1953年，只生产了1670辆；另一部为当代产品，请看两者的风格有多大的不同！老式别克依旧保了古朴风味，前保险杠银光闪闪，装饰味道浓厚，车体古色古香；但新别克却简单明瞭、实用、车体风阻系数低。将两者作一对比，我们可发看出：轿车的演变是简化车身构造，注重性能，屏弃多余的装饰。

别克（Buick）,1985,Windcat:当通用的别克部于1985年在新闻发布会上展示该部概念车时，与会者感到惊讶：别克历来生产中档实用车，为何推出这部“明日之星”？别克此举是想在新一代青年人心中强化其公司印象。这部与众不同的概念车，从门的开启、发动机的安置（后置、外置）到车体的构成、方向盘及仪表盘布置、色彩的协调等的确独树一帜。她将目前汽车设计的流行手法全部融进她的怀抱：超低底盘、复合材料车壳、复合材料车壳、隐式车灯、侧向进风槽、流线型车体……是一部值得玩味的未来车。
