水星

福特·水星MC II型汽车是林肯·默寇利部生产的概念车。该车用美洲狮的头像作为图形商标，表示该车像雄狮那样威武气派，像美洲狮一样与大自然共舞。

狮是古今中外人们崇拜的动物，在古埃及文化中就被视为王权。如在卡夫利法老王的金字塔旁边，就有一尊20米高的伏卧着的人首狮身石雕―法老王的纪念像。在一千年前的亚庶王国的王宫墙壁上保留着《垂死的牝狮》浮雕，描绘一只雄被数只箭射中后的哀鸣，反映出谁能驱狮降虎就有英雄气概；在人们的生活中也常用雄狮来代表勇敢和威武。

水星（Mercury）,1947：水星公司作为独立的公司于1940年在福特名下开始生产轿车。本车为第一次世界大战后的三段式款式，但一直生产到1949年才停止。
