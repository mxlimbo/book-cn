现代

1947年，郑周永创建现代汽车公司，经50多年的发展，它已成为韩国最大的汽车生产厂家，并进入世界著名汽车大公司行列。其商标是在椭圆中采用斜体字“H”，“H”是现代汽车公司英文名“HYUDNAI”的第一个大写字母。

“现代”首先体现了“2000年在世界上腾飞的现代汽车公司” 这一概念；其次还象征现代汽车公司在和谐与稳定中发展。商标中的椭圆即代表汽车的方向盘，又可以看作是地球，与其间的H结合在一起恰好代表了现代汽车遍布全世界的意思。

韩国与世界汽车大国日本隔海相望，似乎在那里发展汽车工业有“班门弄斧”之嫌，但现代公司有“班门弄斧”这可贵的勇气，并正在成长为“新鲁班”。

现代汽车公司满怀豪情地进行了大胆的猜测：到2010年，世界上十大汽车公司中，将有现代汽车公司。

汽车外形尺寸（长×宽×高）：4700毫米×1772毫米×1405毫米

发动机形式和排量：V6、3.0升

最大功率和扭矩：146千瓦、230牛·米

现代（Hyundai）图片赏析

 

现代（Hyundai）,1993,Qrandeur:韩国现代汽车公司1993年上市的最新、最高档次的豪华房车。发动机为EFI SOHC 3.0V6,马力约205匹。车内装载二级配备，其豪华程度令人对后起之秀的现代公司的水平乔目相看。

　

现代（Hyundai）,1993：韩国现代汽车公司生产的两种经济类新款车：Elantra (9179)和Execl(180)。配备属一级。
