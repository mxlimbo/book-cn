大切诺基

切诺基·吉普车（Cherokee Jeep）是克莱斯勒公司的“鹰·吉普部”生产的越野车，“切诺基”取自美洲印第安部族切诺基土人。他们世代居住在山区，由于生活和狩猎的需要，他们擅长在山地攀行，以此表示“切诺基”汽车能攀过岩石、涉过泥沙，征服任何艰难险阻，到达胜利的彼岸。

美国的克莱斯勒、英国的利兰和日本的丰田，被称为世界吉普车生产厂家的“三剑客”。

由于越野吉普车具有功率大、通过能力强、底盘坚固，能配合车主完成各项任务，因此称吉普（Jeep）为有生命的汽车，它的性别是“雄性”。

在第二次世界大战前夕，美国的威利斯、奥巴朗德和福特公司联合生产的越野车被美军选中，由于威利斯公司占有的份额较大，一般称越野车为“威利斯”，并命名为“GPM”。该车在第二次世界大战中，能冲锋陷阵、勇往直前而被美军将士称为“英勇之士”，并用当时的一本流行连环画中的一个主人公的名字“Jeep”，来命名自己心爱的“威利斯”越野车。
