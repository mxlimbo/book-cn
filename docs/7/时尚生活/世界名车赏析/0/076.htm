双龙

双龙汽车公司创建于1954年，是韩国四轮驱动汽车的主要生产厂家。

“双龙”汽车商标将SSANGYONG中的“S”抽象成“8”字，形似“双龙”飞舞；在文字商标中又使用两个“S”，表示“双龙”情深意重。

“龙”不仅是中华民族，也是亚洲其他民族所信奉的万灵之长，被喻为神圣和威严。但在高丽民族还有“双龙”兄弟情深意长的动人传说：远古时有两条灵慧的龙，潜心修练，盼望有一天能上天成仙。到了1000年修练期满时，玉皇大帝就送下一颗升天如意珠。玉皇大帝等了很久也不见他们升天，觉得很奇怪。后来得知升天如意珠只有一颗，双龙不能同时升天，正在互相推让呢。他们这种仁让美德感动了玉皇大帝，就又送下一颗升天如意珠，使双龙同时升天。双龙公司正是从这个故事中悟出自已的企业哲学，以“双龙”命名自己的公司。
