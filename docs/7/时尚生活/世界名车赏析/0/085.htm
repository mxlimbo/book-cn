沃尔沃

沃尔沃（VOLVO）也译为“富豪”，商标由图标和文字商标两部分组成。其图形商标画成车轮形状，并有指向右上方的箭头。文字商标“VOLVO”为拉丁语，是滚滚向前的意思，寓意着沃尔沃汽车的车轮滚滚向前和公司兴旺发达，前途无量。

成立于1927年的沃尔沃公司生产的每款沃尔沃轿车，处处体现出北欧人那高贵的品质，给人以朴实无华和富有棱角的印象，尽管“沃尔沃”充满了高科技，但仍不失北欧人的冷峻。“沃尔沃”那典雅端庄的传统风格与现代流线型造型揉合在一起，创造出一种独特的时髦。卓越的性能、独特的设计、安全舒适的沃尔沃轿车，为车主提供一个充满温馨的可以移动的家。
