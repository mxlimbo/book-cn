凌志

90年代，丰田汽车公司推出的 “凌志”牌高级轿车，就像一匹 “黑马”，以与众不同的风格跃入人们的眼帘，以不同反响的商标 “L”标新立异。

“凌志”商标由图形商标和文字商标两部分组成。它的图形商标不是采用常见的三个椭圆相互嵌套形式，而是一个椭圆中镶嵌英文 “Lexus”的第一个大写字母 “L”，并被镶在散热器正中间；车尾标有文字商标 “Lexus”,喻示该车驰骋在世界各地的道路上。

70年代，丰田公司在美国作电视广告时，总是以一副亲切而谦逊的面孔出现在观众面前。90年代，丰田公司却一改过去那和善的面孔，如在一则电视广告中，画面是凌志LX450汽车爬上了摩天大楼，当爬到第40层时 “定格”，广告词出现”凌志LX450汽车无所不能，而任何车都不及凌志！”出乎人们所料，清高自大的美国人，不仅买丰田公司的帐，而且还为其捧场，使 “凌志”车在美国 “火”了起来。就连向来看低日本车的欧洲汽车评论界，也赞不绝口，称之为日本汽车的 “里程碑”。）

发动机形式和排量：V8、4.0升

凌志（Lexus）图片赏析

凌志（Lexus）,1993,Ls400:丰田公司投入近4000名最优秀的工程技术人员，花了六年多的时间紧张开发而成了豪华房车，其中LS400被称为凌志的元老，它集中了日本汽车工业所能表现的精华，令欧美汽车界赞不绝口，有过横扫美国豪华车市场的佳绩，是日本房车生产的设计的“里程碑”。LS400沉稳而又圆滑，外型极具气度，内外装饰豪华典雅，洽到好外，平稳的驾驶性和舒适的乘坐性足以使它鹤立鸡群。另外，凌志的完美的间响系统被誉为世界房车中“第一音响系统”。

　

　

凌志（Lexus）,1993,ES300:令人赏心悦目的ES300，蕴藏着先进的机电技术和精湛的工艺水平。

　

　

凌志(Lexus),1994,GS300:最新上市的新款GS300，车体更加结实，流畅，风格更为深沉，线条更为飘逸，配备更为完美……她完全揉进了LS400和ES300的长处，是一代凌志的“奏鸣曲”。

　

　

凌志（Lexus）,1993,LS400,Limo:能拥有一部全新的加长凌志LS400伴您商业谈判将是何等的体面！

　

　

凌志（Lexus）,1994 SC300：喜爱凌志的车迷，当开过一段时间的LS400后，就有想驾驶一部凌志轿跑车的强烈愿望，这个愿望在凌志SC300和SC400推出后，彻底地得到了满足。这部轿车外观更趋流线型，是目前最美的轿跑车之一，它轮跑并没缩短，可见它的室内布置是多么的宽敞，在照顾舒适性和驾驶性方面又是多么的精心！
