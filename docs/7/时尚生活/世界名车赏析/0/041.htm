三菱

三菱（Mitsubishi）商标是于1917年注册的三瓣菱形钻石图案。三菱体现了公司的三个原则：承担对社会的共同责任、诚实与公平、通过贸易促进国际谅解与合作。

1870年，出身望族的岩崎弥太朗创建了九十九商社，1873年改称为三菱商会；在三菱商会里有岩崎弥太朗的三个挚友，其名字都有 “川”字，因此三菱商会使用了 “三菱”文字商标；图形商标的三瓣菱形钻石图案，是根据岩崎家族海运船只上的标识和藩王内家的三帕叶标识演变而来的。从此，三菱集团中的所有成员都使用这个商标。1970年，三菱汽车工业公司从三菱集团中独立出来，该公司生产的轿车和越野吉普车 （三菱Jeep），品质优良，性能超群，使三菱商标知名全世界。

三菱Jeep是出口到中国较多的一种吉普车，由于它种种的优良特性而被我国的边防战士称为 “山猫” 。

三菱跑车

三菱（Mitsubishi）图片赏析一

三菱（Mitsubishi），1993 HSRⅡ:排名位于世界第十大汽车公司的日本三菱汽车公司长期以来一直致力于开发与人类和环境相协调的交通工具，产品极其丰富。这部风阻系数低于0.2的HSRⅡ明日轿车于1989年在美国展出时，人们对它的造型赞美不已,对它能配备如此多的电子设施而感到惊吧，尤其是它的发动机（DOHC2.0升四缸，16气阀，涡轮增压，295匹马力），更让人不可思议。开发这样一部明日之车将有助于人类更多地思考未来。

　

三菱（Mitsubishi），1993，HSR,IV:在1993年第30届东京四际汽车博览会上，三菱再创佳绩，隆重推出第四代HSR未来慨念车。两款车相比较，我们看到，第四代产品比第二代修改得更加流线型化，更加与环境融为一体了……

　

　

　

三菱（Mitsubishi），1993，3000GT：无论是起步、转向、刹向均能准确控制的三菱顶尖跑车3000GT。

　

　

　

三菱（Mitsubishi），1993，Eclipes:与普时茅斯·激光一样，三菱·Eclipse 轿跑车也为世界流行色。
