保时捷

“保时捷（PORSCHE）”曾译名为“波尔舍”，“保时捷”牌汽车的文字商标，采用德国保时捷公司创始人费迪南特·波尔舍（他曾在奔驰公司当设计师，于1930年离开奔驰公司，并创建保时捷设计公司。他的第一件成功作品就是大众公司的“甲壳虫”轿车。）的姓氏，图形商标采用公司所有地斯图加特市的盾形市徽。1948年，第一部以“保时捷”命名的跑车问世。从此，“保时捷”公司以高超的技术和优雅的艺术造型，在跑车世界占有一席之地。

“PORSCHE”字样在商标的最上方，表明该商标为保时捷设计公司所拥有；商标中的“STUTTGART”字样在马的上方，说明公司总部在斯图加特市；商标中间是一匹骏马，表示斯图加特这个地方盛产一种名贵种马；商标的左上方和右下方是鹿角的图案，表示斯图加特曾是狩猎的好地方；商标右上方和左下方的黄色条纹成熟了的麦子颜色，喻示五谷丰登；商标中的黑色代表肥沃土地；商标中的红色象征人们的智慧和对大自然的钟爱。由此组成一幅精湛意深、秀气美丽的田园风景图，展现了“保时捷”公司辉煌的过去，并喻示了“保时捷”公司美好的未来。

波尔舍(Porsche)，1995，356：德国著名天才波尔舍博士对近代汽车工业的贡献是有口皆碑的，他曾经设计奔驰，大众甲壳虫等世界级名车，但他毕生的愿望就是设计和生产一部由他本人名字命名的跑车，它就是日后的波尔舍世界超级跑车，第一部波尔舍跑车于1951年问世，排量在1.1至2.0升，而本照片是1955年356敞篷车，1.5升、70匹，最高速161公里/小时。

　

波尔舍（Porsche）,911:波尔舍911 1965年开始了它光辉的生涯，这部被誉为“车坛上的长青树”、“风格永远也不会忘却”的古典跑车精品，在至今为止的整整30年中，不知打动过多少车迷的心，唤起过多少明星的情，对它的崇拜，对它的赞誉、对它的依恋已到了无以复加的程度……车体的优美（前大灯和后尾翼）与性能(3.3升折排量,时速可达260公里/小时自动落下)的卓越巧妙地结合在一起，是小型车创造高性能的典范。但可异的是波尔舍博士并到这一切，他于1952年就与世长 辞了。波尔舍911目前已发展了多种式样，性能大大地提高了，发动机仍为L6。但排量从当时的2.0升已扩大到4.0升左右。从外观上看，有：敞篷车（Cabrio,右上图）、轿跑车
（Coupe,右下图）、半敞篷车（Targe,左上图），兜风车（Speedster）;按尾翼有否分Carrera和非Carrera;按驱动方式分四轮驱动（Carrera4）和后四轮驱动：按涡轮增压器有否分Turbo(或称波尔舍930，左下图)和无Turbo等。这些技术(Tubo 和4WD)在日后开发波尔舍959时，全部用上了。

　

波尔舍(Porsche):正如科尔维特、法拉利和奔驰这些漂亮的车在问世后被一些车厂改装一样，波尔舍911也如此。（左图）这部车是由德国AMG成员GA公司改装的波尔舍911，从效果来看，车体变得更细腻，车内变得更明亮：（右图）号车为参加一级方程式大赛的波尔舍，该车的造型与马自达RX----729P类似。

　

波尔舍（Porsche）,1994,911:推陈出新的波尔舍911，充满着怀旧、复古的情调，车体流线、光滑、内饰精细，但已失去了传统波尔舍911的风味了。

保时捷（PORSCHE）图片赏析二

波尔舍（Porsche），1989，928：波尔舍911在它的生涯中外型风格基本上没变过，虽然它不是不朽的，但车迷想驾驶新款波尔舍的愿望仍旧是极其强烈的，于是波尔舍911的两位后继者波尔舍928及波尔舍944分别在1978年和1981年相继问世。波尔舍928是纯粹的波尔舍公司产品，是波尔舍家族中体型最长、最宽的，发动机前置（符合国际习惯），V8缸，排量在1987年达到5.0升、32汽门，马力316匹，时速270公里/小时，它的提速时间小于6.0秒，另外还加了天窗，车体绝大部分采用铝合金，是现代波尔舍公司又一杰作。

　

波尔舍（Porsche），1987，944：波尔舍944只是波尔舍928的半个兄弟，因为它只有四缸，刚好是928的一半。波尔舍944的诞生被国际汽车界称为“波尔舍公司内最好的事”，到1987年为止，该车配备了EFI，四汽门涡轮增压发动机（前置），马力达247匹，时速260公里/小时。正如法拉利名款都是精品，不分高低一样，波尔舍911、944、928也不分胜负，款款如珠。

　

波尔舍（Porsche），1982，924：波尔舍924是波尔舍公司卖给德国大众公司预定的跑车，后因大众不准备继续开发它，波尔舍公司这些车，命名为波尔舍924，924的外型几乎和944一样，1982年，最后一批924装上涡轮增压发动机后离开了车间。

　

波尔舍（Porsche），1989，959：虽然波尔舍959的外型不像法拉利F40和朗波吉尼·康德那么激动人心，但确是一部实实在在的高品格超级跑车，是波尔舍公司的顶极产品，可击败世上所有跑车。被誉为“世界五大名跑车”之一。在这部车里融进了当代汽车制造一切技术和工艺：外型采用拓朴学原理放机而成，各种车外饰部件位置经过反复比较才确定，保留了传统的波尔舍风格，最神奇的是发动机和整车的控制系统，450匹马力双涡轮增压发动机(DOHC、L6缸、2.9升排量)的强大动力，将其推至320公里/小时的极速，全程计算机控制的四轮戏驱动可以在任何路况条件下畅通无阻。该车只生产了200辆，价格自然高得惊人，但这个数量足以使它挤进B组赛行列，遗憾的是，由于959排放量超过美国环保要求的标准，以及该车前后保险杠满足了不防冲击要求，波尔舍959不准进入美国，令许多美国车迷大失所望。
