雪铁龙

1990年，安德烈·雪铁龙发明了人字形齿轮。1912年，安德列·雪铁龙开始用人字形齿轮作为雪铁龙公司产品的商标。后来，雪铁龙曾组织过横穿非洲大陆和横越亚洲大陆的两次旅行，使雪铁龙汽车名声大振。

法国人生性开朗，爱赶时髦，喜欢新颖和漂亮，“雪铁龙”轿车就表现了法兰西这种性格，每时每刻都在散发着法国的浪漫气息。若穿上法国时装，开上“雪铁龙”轿车兜风，那就更能体会到亲情逸致的法国式生活情趣。在人们心中，雪铁龙是典型的法国情调，秀美、精致的图画，大弧度曲线和倾斜式的车首、轻松自在的样子。驾驶“雪铁龙”轿车能体会到一种独特的情调，无怪乎车迷称其为“法国第一夫人”。

在1997年9月的法兰克福汽车展览会上，法国雪铁龙汽车公司首次展示由该公司设计制造的全新车型—雪铁龙·萨拉（Xsara）轿车。萨拉车的名称极富吸引力，它的声调便于记忆，又充满现代感和诱惑力，能唤起异国他乡的遐想。此外，字母“X”象征雪铁龙的技术和雪铁龙家族的荣耀。萨拉为雪铁龙轿车中的佼佼者。
