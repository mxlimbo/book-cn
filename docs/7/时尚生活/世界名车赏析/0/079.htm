夏利

1986年，天津汽车工业总公司引进日本大发工业株式会社的技术生产出天津·夏利轿车。车名的日本音译为“夏来多”，公司初步确定中文名字为夏立得，当时任天津市长的李瑞环同志提议改为“夏利”，意为“对华夏有利”。

天津·夏利轿车商标，最早在散热格栅中央镶嵌“夏利”两个字。1997年开始采用形似“高速路”的新商标，表示夏利轿车是受大众欢迎的省油车，又是能在高速路上与其他车辆竞争的优质车，喻示“夏利”前程远大。

97款夏利（TJ7100UA）轿车，更具有豪华、新潮感，配有冷暖空调、宽大的行李箱，操纵轻便更省油，是又一款适合中国国情的轿车。

汽车外形尺寸（长×宽×高）：3995毫米×1615毫米×1385毫米

发动机排量：1.3升

最高车速：160千米/小时
