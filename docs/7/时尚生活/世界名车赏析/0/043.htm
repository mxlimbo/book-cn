无限

无限（Infiniti）：日本为了占领美国高档车市场，相继在80年代后期推出了一系列价廉物美的高档车，丰田公司推出了现在风靡全球的凌志，而日产便推出了性能卓越的无限（Infiniti）。它分别有Q45（上左图）、G20（上右图）、M30（下左图）、J30（下右图）等款式，排气量在款式上一目了然，如最高档次的Q45，排量为4.5升，（依次类推），与日本国首相的座驾“总统号”排量相等。Q45曾被评为设计最佳的房车，风格沉稳、品格优异、欧陆风情浓重、口味独特。
