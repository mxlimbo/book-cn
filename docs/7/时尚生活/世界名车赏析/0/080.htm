红旗

“红旗”图形商标立在发动机盖的前端；另一商标是在椭圆有一带羽毛的“1”，表示“中国第一汽车集团”。该商标镶嵌在散热器的正中；文字“红旗”商标则标注在车尾。“红旗”两个字由毛泽东主席亲自书写。

“红旗”是对中华民族最古老的两个部落“龙―风”图腾的简化。旗杆，象征着“龙”；旗面，象征着“凤”。“龙”和“凤”的结合，就是团结、统一、伟大的中华民族。腾飞的“龙―凤”，代表着东方巨龙的觉醒和美好的未来。那鲜红的“红旗”充分体现出热情、直率、大度和充满活力的中华民族风格，表现了中华民族团结向上、不怕艰险、奋力拚博的精神。

红旗（Hongqi）图片赏析

红旗（Hongqi）,1986,770:六七十年代曾经是中国首脑政务专用的红旗，外型酷似苏联·伏尔加，但具有凯迪拉克的庄重和帅气，在中国百姓心中，象征着权和地位。该车长度为5.98米，三排座，自动排档，发动机为5.7升V8缸，马力2.7匹，最高时速160公里/小时。但由于油耗太大（27升/百公里），于1981年被迫停产。但亡灵永存 。
