普利茅斯

普利茅斯也称顺风，是美国克莱斯勒汽车公司一个分部，1928年被克莱斯勒汽车公司收购。普利茅斯是当年英国向美国迁移僧侣的港口，顺风部的产品就用普利茅斯来命名；图形商标用僧侣曾乘坐过的帆船“珠夫拉瓦”号的船图案。

普利茅斯是克莱斯勒汽车公司的著名品牌，昔日以制造大功率轿车并在汽车赛事上屡创佳绩而闻名。

1997年，顺风部推出普利茅斯·潜行者（Plymouth Prowler），这款真正的美国式跑车，代表着美国及其民族的文化品质，说明了美国文化的多样性和继承性，体现了年轻的美国极富创造精神，是一次汽车文化融入世界文化大潮的尝试。普利茅斯·潜行者这款具有传统老爷车风格的双座敞蓬跑车，正在世界汽车业掀起一股返朴归真的热潮。

普利茅斯·潜行者

发动机形式和排量：V6、3.5升

　

普利茅斯（Plymouth）图片赏析

普利茅斯（Plymouth）,1992,Laser：连续三年荣获《汽车和驾驶员》杂志颁发的“十佳轿车”（1989，1990、1991）称号的激光牌双门跑车是普利茅斯公司近年来的新产品，车体灵巧、美观，0.33风阻系数的流线型车身圆润、漂亮，尤以发动机性能卓越而倍受青眯，2.0升四缸，DOHC、16阀，涡轮增压、多点电子喷油发动机可生产190匹马力，加上四轮驱动的功能，足够令车迷狂喜了。

　

普利茅斯（Plymouth），1994：普利茅斯旅行、商用高档面包车，和马自达MPV的风格同出一炉。　

　

普利茅斯（Plymouth），1993：普利茅斯两款经济类车（一级配备）：太阳舞（Sundance,左图）和赞美（Acclaim,右图）。其中，太阳舞为中国将引进的车款，赞美牌装载涡轮增压器后，马力更为强劲，为驾车人士所喜爱。
