玛希拉蒂

1914年，玛希拉蒂（MASERATI）家族的四兄弟在意大利的科隆纳创建了玛希拉蒂公司，公司 以生产高级赛车而著名。其商标是在一个椭圆中放置一个三叉兵器，相传这个兵器是罗马神话中的海神纳普丘（在希腊神话中则被波塞顿海神）手中的武器。它显示出海神巨大无比的威力。

这个商标也是公司所在地意大利玻罗尼亚市的市徽。该商标表示 “玛希拉蒂”牌汽车，就像浩渺无垠的大海咆哮澎湃，隐喻了玛希拉蒂汽车的快速奔驰的潜力。

汽车外形尺寸（长×宽×高）：4550毫米×1810×毫米×1380毫米

发动机形式和排量：V6、3.0升

最大功率和扭矩：209千瓦、413牛·米

最高车速：255千米/小时

加速性能：从0加速到100千米/小时用6.0秒
