MG

1924年，莫利斯·伽吉（MORRIS GARDGE）在伯明翰效外创建了一家汽车厂，并以创建人的姓氏作为公司名称，采用文字“MG”作为商标。公司以生产著名的MG系列敞逢跑车而闻名。后来，该公司并入英国利兰汽车公司，仍使用原来的名称和商标。

MG跑车向来以独特的设计、做工精细和性能优良而著称。如人形化了的车头五官齐全，每看到它，就像见到了老朋友一样，给人以亲切感；独有的加油口盖及其下面的网状进气口，更增添了几分幽默感。

侏儒（MG）,1955,TC：英国侏儒公司（Morris Garges）在立于1923年，以生产民用小汽车而出名，风格上有点类似于美洲虎。这里展示的是二战后的两门敞篷款式，车型古朴典雅，端庄大方，深受战后青年人的喜欢和迷恋（发动机为1.25升4缸，54匹马力）。

　

侏儒(MG),1980,Midget:产量23万的麦捷特敞篷车（1.5升）,外型，成其是车大灯酷似波尔舍911，是最受欢迎的微型车之一。
