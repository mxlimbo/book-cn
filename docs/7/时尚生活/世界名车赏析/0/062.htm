阿尔法

意大利设计和生产的运动车都比其它国家的快，由此称意大利为“快车”之国。阿尔法·罗密(ALFA ROMEO) 牌汽车又是快车之国中的快车。

1910年，意大利政府在米兰创建赛车公司，1901年，一家取名Soueta Anonimal Lomboda Fabbrica Autombilc的小汽车厂在米兰诞生。1908年，一名叫Nicoha Romeo的工程师接管了该家工厂。他把原厂名缩写后，再加上自己的姓氏，就组成了阿尔法·罗密欧（ALFA ROMEO）这个名字。80年代并入菲亚特汽车公司，现为意大利第二大汽车公司。

1911年，阿尔法·罗密欧公司将 “ALFA ROMEO”置于米兰市的圆形市徽（原是维斯康泰家族的徵章）外圈的上半部。该公司采用这个图案作为商标，主要是为了纪念米兰市的创始人维斯康泰公爵及其家族。

徽章分左右两部分，左边的十字原是米兰大公的徽章，后来正式成为维斯康泰家族的徽章一部分。右边部分是蛇正在吞食撒拉逊人的图案，对此有许多耐人寻味的传说：其中一是象征维斯康泰公爵的祖先曾击退了使该城人们受难的“恶龙”。从此，维斯康泰成为人人皆知的英雄。

爱快罗密欧（Alfa Romeo）,1993,164 Super:出自著名汽车造型设计大师家Pininfarina 之手的意大利精典作品，兼具房车与跑车的双重特性。该车揉优雅和刚强、传统和现代于一体，将 菲亚特集团的房车设计风格提升至颠峰的境界。三角盾牌的贵族图腾使人们追亿到遥远的过去，先进的电子配备令驾驶员操作自如、安全稳定。在度假期间，沿欧洲的高速公路，开一辆爱快罗密欧兜风，可谓惬意无边。（二级配备）。

　

爱快罗密欧(Alfa Romeo),1938,8C 2900B为了保持长距离赛车的统治地位，1938年阿尔法公司推出这辆装有2.9升引擎的跑车，最后包揽了赛事的前三名。该辆跑车与当时的宝马和奔腾车风格相接，水箱罩大而突出、向前倾斜，窄而长的前车体包裹着发动机，四个突出的车轮罩鲜明而富有个性，为30年代流行的车型和设计风格。爱快罗密欧曾充当过墨索里尼的座下椅，用于二中检阅军队；而每当爱快罗密欧从亨利·福特身旁驶过时，他总是将礼帽戴并且凝望着它，没人知道这是为什么。

　

爱快罗密欧（Alfa Romeo）,1989, Spyder敞篷车：于1967年开始生产至今，历经数次修改，与波尔舍911一起创下了生产历史最长的运动车之世界纪录。该车外型也和波尔舍911相似，前部像一个青蛙的嘴，两个前大灯非常像青蛙的两个眼睛。让人回味无穷。

　

爱快罗密欧（Alfa Romeo）,1989,75 SZ:外型采用Zagato 塑料壳体的新型跑车，于1989年在日内瓦车展上首次亮相，该车的前保险杠几乎碰地，两边对称布置三只四方车灯，与奥斯通马丁·拉贡达的风格极为相似，显得端庄而有生气。契型车体为时下最流行的款式，敞蓬造型更富情趣，在急驶上，可达到245公里/小时的极速。
