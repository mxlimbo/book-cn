凯迪拉克

凯迪拉克商标是凯迪拉克家族在古代的宗教战争中，使用的“冠”和“盾”型的纹章图案。“冠”上的七颗珍珠表示凯迪拉克家族具有皇家贵族血统，即凯迪拉克家族是贵族。“盾”象征着凯迪拉克军队是一支金戈铁马、英勇善战、攻无不克、无坚不摧的英武之师。

“盾”被两根深褐色棒平分为四个等分。第二和第三等分有两根相互交叉的褐色棒，表示十字军战士在遥远战场上富有骑士般的勇猛。第一和第四等分中各有三只黑色的鸟，这两等分又被黑色棒一分为二，并把三只相同的鸟分开，两只在上，一只在下。按照当时的风俗，没有腿和嘴的鸟，如果以三只同时出现（即三位一体），就表示神圣。这些鸟还表示大胆和热情的基督教武士的智慧、富有、聪敏的头脑和完善的品德。

“盾”中的各种颜色也有深刻的含义，它们分别表示婚姻、土地和丰收。如红色表示勇猛和赤胆；银色表示婚姻、纯洁、博爱和美德；黄色表示丰收和富有；蓝色表示创新和探险；黑色表示土地。

凯迪拉克

凯迪拉克（Cadillac）图片赏析一

凯迪拉克（Cadillac）,1994,Fleetwood—Brougham：虽然1994年的车型与去年的车型一致，但在性能上作了较大改进；发动机为EF1系统，排气管变成双管，马力增加到280匹。自动反光镜、电加热座椅、驾驶座记忆系统、旅程电脑等等应有尽有。是配备完整、性能卓越的“老板车”。代表了90年代的汽车设计制造水平。

　

凯迪拉克（Cadillac），1988，Solitaire:尽管这部漂亮的凯迪拉克概念车没有生产，但向人们展示了21世纪的凯迪拉克风格。车身光滑圆润，线条流畅、利落、给人一种感觉：好像该车仅由玻璃和钢制成。车内配巡航地图显示屏，可随时确定车主的位置。6.6升V12缸的发动机输出430匹马力，最高时速可达320公里/小时。该车大部分材料可再生利用，1994年的风格与她相比还相去甚远，但我们将等待这一天的来临……

　

凯迪拉克（Cadillac），1967，Limo:该车是1967年Fleetwood 车型上经过改装加长至14.49米，可称为陆地巡洋舰。车内的配备样样齐全：转角沙发、酒吧、厕所、浴室等等，整车具备三级配备。实属罕见，为大多数民众所喜爱。

凯迪拉克（Cadillac）,1931,V—16 Pheaton：通用公司的旗舰凯迪拉克是美国历来打制得最大，最好的房车，完全能体现半个世纪来美国人的追求目标--------美国梦。本车的徽标是30年代凯迪拉克特制持好处，V---16大排量的发动机为当年的跑车所装载。毫无疑问，该车是目前较珍贵的收藏品。

凯迪拉克（Cadillac）图片赏析二　

凯迪拉克（Cadillac），1962：60年代是美国汽车工业的黄金年代和全盛时期，各种款式的各车云集美国。造大车、卖大车。赚大钱是每一个汽车生产与销商子里最强烈的念头。本车即是一例，宽大四方的车体在其尾部长上了一对小小的翅膀，像鲨鱼利齿般排列的水箱罩在闪闪发光的装饰线映衬下特别引人注目，这些花俏的装饰成了当时人们生活中一件不可缺少的装饰品，一时为政界要人、财界大亨所追求。

　

凯迪拉克（Cadillac），1971&1967：两个不同时代车款：DeVille Fleeetwood。我们看见，凯迪拉克变得越来越庄重、阔气了。

　

凯迪拉克(Cadillac),1986,Allante:凯迪拉克在面临奔驰、宝马、劳斯莱斯等一些老对手的竞争下，特意推出了尺寸小、外观柔和、马力强颈、欧洲色彩浓厚的车型。这辆由Pininfarina 设计的跑车在意大利用德国钢、瑞士铝合金打制完车体，然后由德国汉莎航空公司的747专机运往通用公司组装。该车揉和了典型化贵的欧陆风范与毫放的美式气度，体现出优美、动力、珍贵三大特征，营造了新一代的美国之梦。

　

凯迪拉克（Cadillac），1994，SeVille：全新的凯迪拉克赛维利亚，其中STS在英国汽车杂志上被誉为“世界五大名房车”之一。风度翩翩、光彩照人、魅力永存。发动机采用“北极星”32汽门DOHC V8,马力输出、提速、时速不亚于宝马740i，前后车内空间足够宽敞，仪表盘造型颇有时代感，是一部值得推荐的豪华座驾。

　

凯迪拉克（Cadillac），DeVille :1994年款（左图）和1993年款（右图，左下图）的凯迪拉克帝威级房车。我们看到，相差一年的凯迪拉克风格就完全变了，变到更圆润、神气。两车均配上了二级配备。
