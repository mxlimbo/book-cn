大宇

1967年，金宇中创建大宇（DAEWOO）汽车公司，它是韩国大宇集团的骨干企业，使用形似地球和花朵的大宇商标。

“大宇”商标像高速公路大“动脉”向未来无限延伸，表现了大宇的未来和发展意志。椭圆形代表世界、宇宙；向上展开的花朵形态，体现了大宇家族的创造力和挑战精神；中部五个蓝色的实体条纹和六条白色条纹，表示大宇在众多领域内无限发展的潜力；蓝色代表年轻、活泼，而白色则代表牺牲。整体商标体现了大宇家族的智慧、创建、挑战和牺牲的企业精神，表现出大宇集团的“儒家”风范。

汽车外形尺寸（长×宽×高）：4980毫米×1720毫米×1417毫米

最大功率和扭矩：95千瓦、100牛·米

最高车速：190千米/小时
