常见酒礼

　　在中部、东部等传统汉民族习俗保留较多的农村，一般有先干门前杯之说，依各地风俗不同和酒杯大小，分从1－4杯不等。有些地方礼规严格，门前杯必干。然后才开始交叉互找应答，觚筹交错中，酒菜随之减少。在圆桌超过8人的席上，常设二名司酒者，俗名“酒司令”，分管斟、派、论、斗，一般酒量较大。宴席将终，须饮尽最后一杯。席上客不论生熟，一律排列互敬，以每两人互敬一杯计，8人席至少要五、六十杯。加上始及五花八门各种名目的共饮，小群体共饮，一席须耗白酒3、4瓶。


资料：中华酒韵
