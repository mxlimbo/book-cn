　　"爸！奕恒！你们怎么了？是不小心吃到辣椒了吗？"

　　"我本来就很乖！"

　　"如果乖，又怎么会在爷爷和奕恒的饭里加辣椒？"

　　"这可不能全怪我，我只不过是放了辣椒，又没叫他们一定得吃下去啊！"全是歪理，难怪奕廷也无心辩下去了！

　　"好吧！就算是你对吧，但是……"晓萱本来听到奕廷不计较的话，她开始暗自得意了，可是才听到"但是"两个字，她的眉又皱在一起，为什么他不一次讲完呢？

　　"但是什么？"

　　"爷爷年纪大了，奕恒也被你整够了，下次可别再玩把戏了！"

　　"嗯！"

　　看晓萱一脸心不甘情不愿的样子，让奕廷再次心疼起来，于是走到床边，双手由晓萱的背后圈住晓萱。

　　"今天你都在做些什么呢？"为了让老婆开心，奕廷温柔体贴的慰问老婆。

　　"我好无聊，我好想工作，奕廷，让我出去工作好吗？"

　　"不行，我何奕廷的老婆怎能出去工作？向且我要你是我一个人的，我才不让别人和我一起分享我的老婆。"他霸道的口气让晓萱又喜又忧，喜的是奕廷重视自己，忧的却是无法接受这种少奶奶的生活。

　　"可是，我受不了这样无所事事的日子。"

　　"你可以找晓薇啊！"

　　"还不是一样！"

　　"你再试试看，如果不行，我再安排你到公司工作，这样可以吧！"

　　"哇！廷，你最好了，"兴奋不已的晓萱抱着奕廷，奖励性的吻上了他的唇，然而才轻轻一吻，却已经激起了奕廷的欲望之火，两人不再只是亲吻而已，急促的双手，呢喃的呻吟，又为宁静的夜添上爱的色彩，两人之间的恋爱，如野火般的炽热狂烧……
