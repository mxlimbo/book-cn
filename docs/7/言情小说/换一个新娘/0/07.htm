　　缓援的打开盒子，里面放置着璀璨夺目的钻石项练和耳环，晓萱不敢相信的道：

　　"这……太贵重了。"

　　"它再怎么贵重也比不上我的小宝贝！"

　　奕廷对她的宠爱，在此刻已表达得淋漓尽致，晓萱感动的吻上他的唇，再次点燃他们之间的火苗，钻石项练、耳环已被奕廷悄悄的拿开了，他门再度陷入激情欲火之中。
