时钟

　　《时钟》写于十月革命之前，是在沙俄白色恐怖统治下完成的。最初发表在1896年11月22日《尼日戈罗德报》。写作《时钟》时的高尔基，已是在俄国和欧洲有广泛影响的进步作家，因此，文章具有广泛的号召力。这是一篇战斗檄文，也是慷慨激昂的号召书——号召人们不要屈服于苦难的压力而不能自拔，不要诉苦，也不要埋怨生活，要向"生活中的凶暴与污秽，虚伪与残忍"作斗争，要珍惜生命，热爱生活，也不要怜惜生命，要让生命永远闪光。

　　《时钟》托物言志，燃烧着革命的激情，其特点是在形象化的议论中蕴含人生的哲理。文中有很强的描写、抒情的成分，还运用比喻、拟人的修辞手法，语言洗练、形象、优美、隽永、有许多描写性和抒情性的文字。同学们自读时要注意体会。
