群英会蒋干中计

（其一）

　　今天我们学习《群英会蒋干中计》（板书），本文选自古典名著《三国演义》。《三国演义》是怎样的一部书？有谁看过吗？（引起学生回忆）它的全称是什么？作者是谁？写什么时候的事情？写些什么？（讨论，补充，教师小结）《三国演义》全称《三国志通俗演义》，是我国第一部著名的章回体历史小说，也是我国古代小说跨入成熟阶段的标志。它的作者是元末明初的罗贯中（板书）。

　　《三国演义》取材于重大的历史事件。描写从东汉到晋统一约100年间的历史变化，着重写魏、蜀、吴三国的兴衰过程以及彼此间的政治、军事斗争。它既来源于历史，又不同于历史，作者以《三国志》及裴松之注为依据进行艺术加工，既尊重历史，又增加了民间传说。所以它有助于我们形象地了解这个时期的历史，但又不能忘记，它是小说，有许多虚构的地方。《三国演义》塑造了一系列栩栩如生的人物形象，如诸葛亮、曹操、孙权、张飞、周瑜等等，这些人物可以说家喻户晓，深入人心。我们也不要忘记，他们是小说中的典型形象，倾注了作者强烈的爱憎、褒贬的感情色彩和主观的取舍原则，所以不同于历史上的真实人物，而且，有的人物是虚构的。我们阅读这部古典小说，可以增加历史知识，更可以学习作者记述复杂事件和塑造人物形象的写作方法。还可以提高我们的文学修养。

（其二）

　　 有关三国时期的文学作品，我们学习过哪些篇章？（诸葛亮《出师表》、司马光《赤壁之战》、苏轼《念奴娇·赤壁怀古》、罗贯中《杨修之死》）．谁还记得《赤壁之战》讲些什么吗？（讨论）赤壁之战在《三国演义》中是着力描写的重大战役，书中从四十三回到四十五回，叙述了这次战役的全过程。今天我们要学习的《群英会蒋干中计》，选自《三国演义》第四十五回，讲的就是赤壁之战中的一件重要事情。

（其三）

　　江苏吴卫东老师这样导入：

　　课一开始，教师先在黑板上写出了一个成语——毛遂自荐。接着问学生："毛遂自荐"是什么意思，学生回答：比喻自己推荐自己。教师又问：推荐自己，难能可贵。那么，需要什么条件呢？学生纷纷回答。教师最后作总结：毛遂自荐，首先这个人确有真才实学；其次，这人得充满自信。今天，我们结识一位蒋干，他既无真才实学，又盲目自信，在曹操面前夸下海口，愿凭三寸不烂之舌，去江东劝说周瑜投降，结果怎样呢？（板书课题：《群英会蒋干中计》）

（其四）

　　湖北洪镇涛老师这样导入：

　　我们学过《赤壁之战》，那是从司马光编的历史文献《资治通鉴》中节选的；现在我们学习《群英会蒋干中计》，这是从罗贯中写的小说《三国演义》中节选的。这两篇课文虽然写的是同一历史事件，但前者是历史著作，要求用准确而精炼的语言，原原本本地叙述历史事实，后者是文艺作品，要求用文艺的笔调和虚构的情节，塑造历史人物形象。本文是"赤壁之战"中间的一个故事，内容比较复杂，我们首先要把它的情节、结构弄清楚。下面，请同学们认真阅读课文，然后按照故事的发生、故事的发展与高潮和故事的结局给本文划分段落，还要思考这样一个问题：前面三个自然段与"蒋干中计"无直接联系，是否可以去掉？

（其五）

　　本文节选自《三国演义》第四十五回。《三国演义》原名《三国志通俗演义》，是我国第一部长篇章回小说。

　　早在晋代和南北朝时期，三国故事已广为流传。南朝宋人裴松之为西晋史学家陈寿《三国志》作注，采集了许多轶事传闻；隋唐时代就上演过刘备、曹操的戏曲剧目，刘知几在《史通》（710年作）中说三国故事"得之于行路，传之于众口"；李商隐《骄儿诗》（850年作）中也说"或谑张飞胡，或笑邓艾吃"。这些都说明三国故事已经在民间讲唱，连儿童都很熟悉。宋代"说话"艺术中，"说三分"（三国故事）成为专门科目．并有了专业艺人。元至治年间刊行的《全相三国志平活》，是长达八万字的讲史话本。元杂剧中的三国戏，多达四十余种。罗贯中在这样的基础上"据正史，采小说，证文辞，通好尚"，写成了这部巨著。

　　罗贯中，名本，号湖海散人，经历了元末明初的农民战争和社会大动乱，生卒年大约在公元1330年至1400年之间，他的好友贾仲名在《录鬼簿续编》中说他"与人寡合，乐府（歌谣）隐语（谜语），极为清新。"明代王圻在《稗史汇编》中说他曾经"有志图王"，后来才"传神稗史"（从民间收集资料编写的史籍。也指非正式的史书），还有人说他与元末率盐丁反元称王的张士诚有过联系，明太祖朱元璋推翻了元朝，统一了中国，罗贯中便从事"稗史"的编写工作。《三国演义》就是在这个时期写成的。相传他著有十六史通俗演义。现存有：《隋唐志传》、《残唐五代史演义》和《平妖传》等，不过这些作品屡经后人删改，已经失去了本来的面貌。他还著有杂剧《宋太祖龙虎风云会》等三种。

　　《三国演义》现存的最早版本是明朝嘉靖元年（1522年）刊印的《三国志通俗演义》，首页题"晋平阳侯陈寿史传，后学罗本贯中编次"，清代康熙年间，毛纶、毛宗岗父子仿照金圣叹评改《水浒传》的先例，对全书进行增删润饰，加上了评语，成为清代以来最流行的本子。 《三国演义》以东汉末年魏、蜀、吴三国兴衰的历史为题材，形象地再现从黄巾起义到西晋统一各个地主武装集团错综复杂的矛盾斗争，塑造了一系列栩栩如生的典型形象，表现了作者的忠义观念和仁政思想。小说中好多事件为人们所熟知，如桃园三结义、长坂坡、草船借箭、单刀赴会、七擒孟获、空城计等。小说塑造了四百多个人物形象，其中神机妙算、忠贞高洁的诸葛亮，老谋深算、凶残奸诈的曹操，宽仁爱民、礼贤下士的刘备，忠心义胆、英勇无比的关羽，耿直勇猛、嫉恶如仇的张飞，才能出众、心胸狭窄的周瑜等，更是家喻户晓，长久活在人们的心中。小说采取半文半白的语言，雅俗共赏，具有简洁、明快的特色和生动的表现力，这在八股充斥、文言盛行的年代尤其难能可贵。

　　《三国演义》的缺点是宣扬了封建正统思想、英雄史观和封建迷信，特别是对黄巾起义持否定态度。小说人物的性格缺少发展，而且有夸张失实之处，正如鲁迅所说："欲显刘备之长厚而似伪，状诸葛之多智而近妖。"这些都是我们学习本文时应当了解的。

（其六）

　　本篇课文节选自《三国演义》第四十五回，是赤壁之战故事中的一个著名片断，在此之前，曹操大军南下，刘琮投降，曹操得了荆州，气焰很盛，准备顺江东下。东吴孙权经过一阵犹豫之后，接受了诸葛亮的劝说和内部主战派的主张，决定联合刘备 抗击曹操。曹操的青州、徐州之兵不习水战，顺江东下要依靠荆州水军。荆州降将蔡瑁、张允"深得水军之妙"，被曹操任命为水军都督。周瑜为了要打败曹操，决心用反间计除掉蔡、张二人。曹操手下的蒋干盲目自信，以为可以凭交情去劝说周瑜降曹，结果送上门去，充当了周瑜实行反间计的助手。

　　这篇课文虽然是长篇小说的节选，但在情节上有着相对的完整性。故事情节有开端、发展、高潮和结局，叙述的线索很清楚。
