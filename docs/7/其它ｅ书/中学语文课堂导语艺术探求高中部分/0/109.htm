琐忆

（其一）

　　本文是作者为纪念鲁迅先生诞辰八十周年而写的一篇散文。标题中一个"忆"字，表明这是一篇以记叙回忆性材料为主的文章；一个"琐"字，既表明所记之事小且多，又提示了文章的选材特点：以小见大。学习本文，从哪里入手呢？我们可以先弄清楚一共写了几件事（实际都是谈话）；每次谈话的主要内容是什么；谈话在什么场合下，以怎样的方式进行；每次谈话给作者什么感受。好，下面我们一起来研读课文。

（其二）

　　我们在初中学过鲁迅的一篇散文，文章的题目和主要内容谁还记得？

　　明确：《同志的信任》，文章的中心内容是方志敏烈士于国民党反动派的监狱中，在就义前托人把他用米汤写的一封信和一束文稿，交给从来没有见过面的鲁迅先生，要求鲁迅先生设法把这一束文稿转交给中国共产党中央委员会，表现了一个中国共产党的高级将领对鲁迅先生的无限信任，从而高度赞扬了鲁迅先生的伟大品格。 今天要学习的《琐忆》，是一篇纪念鲁迅先生的文章。本文记叙了鲁迅先生的七次谈话，通过这些日常的对话和独白，鲁迅先生深邃的思想，闪光的智慧和崇高的精神得到比较充分的显现。

（其三）

　　在一个兰香清幽的处所，容貌俊秀、聪明敏锐的进步青年唐 ，不期而遇他渴慕已久的鲁迅先生。他用真挚敬仰的目光注视着自己心目中的导师：先生目光明亮，留着浓黑的胡须，满头簇簇直竖的头发，仿佛处处在告白对现实社会的不调和。先生平易近人，十分关心爱护青年人，帮助他们进步成长，点点滴滴，循循善诱。唐 虚心倾听着先生的教诲，心领神会，欢悦无比。先生谈话时从容镇定的神态，充满智慧的语言，犀利深刻的思想，真使人"如坐春风"，"如沐春雨"。唔，让人开怀，令人奋发的春风呵！唔，滋润肺腑，使人振作的春雨呵！

（其四）

　　对鲁迅的两句诗："横眉冷对千夫指，俯首甘为孺子牛"，大家一定熟悉吧。这两句诗，是对鲁迅先生一生的写照，也是他作为一个伟大作家的人格的体现。毛主席对鲁迅先生一生忠诚为人民服务，对敌人坚决斗争的精神，赞扬备至，对这两句诗作了很好的解释，可供我们参考。他曾说："鲁迅的两句诗，'横眉冷对千夫指，俯首甘为孺子牛'，应该成为我们的座右铭。'千夫'在这里就是说敌人，对于无论什么凶恶的敌人，我们决不屈服。'孺子'在这里就是说无产阶级和人民大众。一切共产党员，一切革命家，一切革命的文艺工作者，都应该学习鲁迅的榜样，做无产阶级和人民大众的'牛'，鞠躬尽瘁，死而后己"（《在延安文艺座谈会上的讲话》）

　　今天我们要学习的课文《琐忆》，就是曾亲身感受过鲁迅先生的关心爱护的唐 先生写的一篇纪念鲁迅先生的文章。"横眉冷对千夫指，俯首甘为孺子牛"这两句诗，则是这篇文章的总纲。

（其五）

　　有位教师这样导入：

　　作者唐 ，原名唐瑞毅，我国著名的文艺评论家、研究鲁迅的专家。他1913年出生于浙江镇海县一个农民家庭，靠亲友资助求学，只学习到初二就被迫辍学。他16岁考入上海邮局任职，后参加工人运动，担任文艺宣传工作，组织读书会。1923年在鲁迅的影响与帮助下从事散文与杂文创作。

　　本文写于1961年。为纪念鲁迅先生诞辰8O周年，作者怀着深厚的景仰之情，写了这篇文章。关于本文的写作动机，他说："回想建国以来纪念鲁迅先生的文章，大多强调鲁迅先生品质中战斗性的一面，对'横眉冷对千夫指'一句阐发得比较充分，但是关于他对同志、对青年的爱就强调得很不够。我认为'横眉冷对千夫指，俯首甘为孺子牛'，这正是鲁迅先生伟大人格在不同方面的两种表现，二者是辩证统一的，对敌人恨，对同志必然是爱。所以我想在我的纪念文章里应该两方面都讲一讲。30年代我作为一个青年，曾经和鲁迅先生有过一些接触，亲身感受过他对青年的关心爱护， 因此，想从后一方面多说一些，这就是我写《琐忆》时的指导思想。"那么，鲁迅先生是怎样关怀、爱护青年的呢？学习课文后，你对鲁迅先生的伟大人格，有哪些新的理解呢？
