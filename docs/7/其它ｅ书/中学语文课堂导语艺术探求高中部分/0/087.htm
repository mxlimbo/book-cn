农村调查序言

（其一）

　　邓小平同志在《讲讲实事求是》一文中写道：1941、1942年，毛泽东同志发起了伟大的整风运动，这次整风运动的主要文献之一就是《<农村调查>的序言和跋》。我们现在要学习的课文，就是《<农村调查>序言》。《农村调查》这本书，收集了毛泽东同志从1927年到1934年在农村中进行调查的一部分材料，是中国共产党制订当时的农村政策的重要依据。

　　 本文是毛泽东同志在1941年3月17日为《农村调查》一书所写的序言。序言，也叫序，是一种文体，一般写在一部书的前边，可以由作者自己写，也可以由别人写，常常用来说明作者的写作意图、写作过程，介绍书的主要内容、精神实质，或者说明同这部书有关的一些事情。序言可以有不同的写法，从本文来说，它的表达方式以议论为主，可以把它作为议论文来学习。

（其二）

　　 公元前260年，秦国发兵进攻力量强大的赵国，赵国派老将廉颇率军迎战。廉颇在长平筑垒坚守，双方相持三年。在此情况下，秦国使用反间计，故意扬言：廉颇只善守而不善攻，秦国不怕廉颇，最害怕的是赵括。赵王没有调查研究，也没有真正了解廉颇的军事才能，也不去向将士了解战争的情况，征求具体的对策，就信以为真，调回了廉颇，改派缺乏作战经验，只会纸上谈兵的赵括为将。赵括到阵前，也没有调查研究，盲目率军出击，被秦军包围，粮食供应又被切断。赵军断粮46天，五次突围不成，结果赵括被秦军射死，40万赵军士兵在投降后全部被活埋，这就是历史上有名的"长平之战"。赵国惨败的主要原因之一，就是赵王、赵括都没有深入地调查研究。调查研究对于战争是如此重要，对其他工作又何尝不是这样？今天，我们学习毛泽东同志的《<农村调查>序言》，从小就要学会调查研究,这对今后的学习、工作都有好处。

（其三）

　　山东戴继超老师这样导入：

　　抗日战争时期，中国共产党开展了具有伟大历史意义的整风运动。在这次整风中，为了教育全党充分认识调查研究的重要性，学习了毛泽东同志进行调查研究的立场、观点和方法，以肃清王明左倾教条主义的恶劣影响，克服主观主义的工作作风。党中央收集了毛泽东同志在第二次国内革命战争时期，进行社会调查的部分材料，编印成《农村调查》一书。毛泽东同志为这本书写了序言"和"跋"。

　　"序"、"跋"是文体名称。"序"作"叙"，或称"引" ，一般用以说明书籍出版意图、著述经过、作者情况以及资料来源等。有的序言还直接对作品发表评论，对有关问题进行研究和阐发。我国古代，常把"序言"列于书末，如《史记》的《太史公自序》。现在一般把"序"放在正文的前面。有的书中叫"引言"、"前言"、"编者的话"或"作者的话"等，这也是"序"。列于书后的通称"跋"，也叫"后记"或"编后"。

　　"序言"可由作者自己写，也可请别人来写。由于角度不同，内容的侧重点也不一样。它的作用主要介绍、评价和推荐，帮助读者更好地阅读、理解作品。因此，每读一本新书，应先看序言。


　　序言的写法很多：有说明的，有记叙的，有发议论的，有用抒情笔调写的，也有说明、议论和记叙相结合的。在《< 农村调查> 》序言》中，毛泽东同志用马列主义观点，以自己的切身体验，提出了"没有调查就没有发言权"的著名论断。毛泽东同志关于调查研究的理论和实践，在今天，对我们仍然有着重大的教育意义。
