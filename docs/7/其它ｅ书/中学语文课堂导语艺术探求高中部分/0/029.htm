汨罗行

（其一）

　　同学们，大家都知道"端午节"的来历吗？端午节，又叫"端五"、"端阳"、"端节"，是我国传统的节日，时间是每年农历五月初五日。这是为纪念忧国忧民的伟大的政治家和爱国诗人屈原而设的。相传屈原在五月初五这一天怀沙投入汨罗江自杀，他的爱国，正直的精神感动了人民，后人为了纪念他，把五月初五日这天当作节日，有吃粽子，赛龙舟，喝雄黄酒，用菖蒲、艾蒿熬水洗澡驱蚊祛风等风俗；而且，人们每当经过汨罗江，都忘不了要去江边凭吊一番。

（其二）

　　本文是一篇精巧、朴实的记叙性散文。作者通过汨罗之行，介绍了与屈原有关的人文景观与感人的故事，歌颂了屈原强烈的爱国主义情怀，表达了作者对屈原的尊敬与怀念。

　　全文反映的主要事件是汨罗行。其间，作者做过的、看过的、听过的有许许多多，但作者只选取了此行中与屈原有关的人文景点和与景点相关联的传说、故事、事件，做到材料集中，主题鲜明。同时、为了营造悼念、崇敬屈原的浓厚的感情氛围，文章开头，作者从子上市、端阳已近入笔，追忆童年时家乡人民纪念屈原的习俗，抗日战争时期郭沫若创作话剧《屈原》，以及建国后几登行吟阁产生的对屈原的眷顾之情，再引出汨罗之行，作者这样精心的选材组材，其用意何在呢？下面让我们认真阅读课文，对此加以体会。
