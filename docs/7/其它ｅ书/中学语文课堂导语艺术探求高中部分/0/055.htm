"老虎团"的结局

　　山东胡甲林老师的导入：

　　《"老虎团"的结局》是一篇复杂的记叙文，它生动地记叙了解放战争期间淮海战役中著名的"双堆集"全歼国民党黄维兵团"老虎团"这一次战斗的经过；"老虎团"是蒋介石"五大主力"之一的十八军的一个团。十八军是陈诚起家的血本，而"老虎团"又是十八军的一支骨干部队，是国民党反动派精心豢养的一只恶"虎"。这次战斗在淮海战役中具有重要意义。本文以灭虎的过程为主线，再现了这重要战斗的概貌，展示了我军获胜，敌军失败的原因。
