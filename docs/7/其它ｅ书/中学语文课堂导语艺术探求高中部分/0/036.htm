汉堡港的变奏

（其一）

　　今天我们学习《汉堡港的变奏》（板书），题目中关键词是"变奏"。"变奏"是音乐术语，指用变换旋律、节奏、调式等手法使乐曲发生变化。这里用于作课题，意思是什么呢？是将汉堡港的正常活动比作一支乐曲，且是一支由来已久的古典乐曲，其节奏一百多年来就是如此。可是，以贝汉廷为首的汉川号货轮来到这里却发生了"变奏"，为什么会发生"变奏"？文章的中心是什么？贯串全文的线索是什么？让我们来学习、研讨课文。

（其二）

　　湖北洪镇涛老师的导入：

　　我们一起来学习女作家柯岩的一篇报告文学《汉堡港的变奏》。（板书：汉堡港的变奏 柯岩）

　　这篇课文是从柯岩的一部报告文学作品《船长》中节选下来的。《船长》由"汉堡港变奏"、"从南市来的孩子"、"吐水的水手"、"是日本人吧？" 、"蓝色的梦"、"伦敦港的友谊"、"邓小平式的船长"、"海员风度"、"船长的苦恼"等九部分组成。本文所写的是中国远洋货轮汉川号在西德汉堡港，装载贵重而又特殊的成套设备的成功及其在汉堡港所引起的震动。下面，提出几道思考题，请同学们带着这些问题，认真阅读课文。要求边看、边划、边批注，并列出每道题要点来。（以小黑板出示思考题） 1、本文为什么取名"汉堡港的变奏"？有什么深刻含义？ 2、本文采用了倒叙写法，说说这样写有什么好处？ 3、具体分析本文是怎么样在矛盾冲突中刻划船长贝汉廷的高大形象的？

（其三）

　　黑龙江杨伯威老师的导入：

　　初中语文第三册已经选了柯岩的一首歌颂周总理光辉业绩的诗《周总理，你在哪里》。柯岩是当代女作家、诗人。从一九七八年以来，她用诗一般的语言写了不少热情奔放的报告文学，如《奇异的书简》、《美的追求》、《追赶太阳的人》、《特邀代表》等，其中《船长》、《特邀代表》荣获一九八一年全国优秀报告文学奖。这篇节选自〈〈船长》的报告文学《汉堡港的变奏》（板书课题），以精巧的构思、多种的叙述方法和修辞手段，突现了我贝汉廷船长和海员的爱国热情、科学态度、实干精神。
