一月的哀思

　　山东王牧天老师的导入：

　　《一月的哀思》，全诗共五章，课文节选的是第二章。原诗发表于一九七七年一月七日《光明日报》。

　　 一九七六年一月八日，全国人民敬爱的周恩来总理不幸病逝。举国上下，无比悲恸。当载送周总理遗体的灵车路过长安街时，万民侍立，哭声震天。诗人李瑛以一个解放军战士的身分，置身人民中间，同伫立长安街的人民一样，寸心欲碎，涕泪满襟。在诗人的心头，压抑着的是一团怒火，在诗人心底，埋藏着的是无限哀思"情动于中而形于言"，诗人把这种强烈的感受，发而为诗，道出亿万军民对"四人帮"的怨愤情绪，唱出亿万军民对周总理的深沉悼念。

　　作者李瑛，当代著名诗人。一九二六年十二月八日生于河南省丰润县，青年时代曾就读于北大中文系。毕业前夕，投身人民解放军的行列，先后任记者和《解放军文艺》编辑。诗集有《枣林村集》、《献给火的年代》等十余种。李瑛的诗，构思精巧，诗风刚健，他善于用以小见大的手法，从平凡的事物中开掘深刻的时代内容。他的诗作，深受读者欢迎。
