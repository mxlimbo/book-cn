五人墓碑记

（其一）

　　在"鸿门宴"上，项羽因为害怕担当"不义"之名，终于让刘邦得以脱逃，纵虎归山，铸成大错。其实项羽真正注重的不过是一个"名"，他企图"名正言顺"的爬上帝王的宝座。其人虽不可一概抹杀，其"义"却不足为训。

　　在我国历史上，另有一大批志士仁人，他们舍生取义，前赴后继，或藐视权贵，或为国为民。不务虚名，不图私利，他们跟项羽有着很大不同！现在我们来学习张溥的《五人墓碑记》，文中记述了颜佩韦等五人的"蹈死不顾"，会帮助我们"明死生之大"，加深对"义"的认识与理解。

（其二）

　　本文作者张溥，字天如，号西铭，太仓（现在江苏省太仓县）人，明末文学家。据《明史・文苑传》记载：张溥从小就勤奋好学。"所读书必手钞，钞已朗诵一过，即焚之，又钞，如是者六七始已。右手握管处，指掌成趼。冬日手皲，日沃汤数次，后名读书之斋曰'七录'"。这足见他学习勤奋刻苦的精神。张溥文章才气很大。传记里说他"诗文敏捷，四方征索者，不起草，对客挥毫，俄顷立就，以故名高一时"。 在文学上，他针对明末文人空疏不学的弊病，提出了"兴复古学"的口号，同时强调"居今之世"必须"为今之言"。他写过不少抨击时政的文章，内容充实，风格质朴。 他卒时，年仅四十岁。他曾编辑《汉魏六朝百三名家集》，著有《七录斋集》。

　　张溥为崇祯四年（1631））的进士，他曾和同里张采组织爱国社团名叫"复社"，集苏州名士，以复古学为号召，继承东林党（以无锡顾宪成为首，反对宦官、议论朝政的士绅集团）人反对黑暗政治的优良传统，进行文学和政治活动，影响很大。张溥 生活于东林党与阉党激烈斗争的时代。他是复社的领袖，为人崇尚气节，富于正义感，对于贵族阉党的特务统治深为不满，与他们进行了坚决斗争。今天我们要学习的这一篇课文就充分反映了这一点。
