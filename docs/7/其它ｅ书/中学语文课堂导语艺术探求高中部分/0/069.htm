论积贮疏

　　汉朝初年，采取一些措施后，社会经济虽有所好转，但地主兼并土地，商贾囤积居奇，加之赋敛过重，迫使许多农民弃农经商，造成了"百人之作不能衣一人"，"一人耕之，十人聚而食之"的局面。又加上内部诸侯扩张势力，准备叛乱；外部匈奴压境，使西汉王朝受到严重威胁。针对汉朝建立多年以后"公私之积，犹可哀痛"的状况，23岁的贾谊写了这篇奏疏（向皇帝陈述意见的一种文体）。主张"殴民而归之农"，注重积贮。这一主张有一定的历史意义和借鉴意义。
