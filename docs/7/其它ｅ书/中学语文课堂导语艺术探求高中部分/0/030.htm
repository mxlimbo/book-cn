故都的秋

（其一）

　　对于秋天，固然人们有着"一年好景君须记，最是橙黄橘绿时"的赞叹，然而面对黄叶飘零，西风残照的凄凉景象，涌上心头的更多的是"寻寻觅觅，冷冷清清，凄凄惨惨戚戚"，和"秋风秋雨愁煞人"的伤感与悲凉。郁达夫的《故都的秋》，高深莫测，体现出这样一种悲秋的情绪，现在就让我们来学习体会一下。

（其二）

　　我国古代，一些文人墨客热爱大自然，赞颂大自然。比如："不愿为五斗米折腰"的东晋大诗人陶渊明，醉心于"采菊东篱下，悠然见南山"的田园生活；写出"疏影横斜水清浅，暗香浮动月黄昏"的北宋诗人林逋，愿意终生和梅花、仙鹤为伴，人称"梅妻鹤子"，他们都渴望把自己的身心托付给大自然。现代也有一位作家，在文章中发出了这样的感叹："秋天，留得住的话，我愿意把寿命的三分之二折去，换得一个三分之一的零头。"知道是谁说的吗？是哪篇文章？对，郁达夫，《故都的秋》（板书）。今天，我们就来学习这篇散文。

（其三）

　　说到秋，可以使人产生无限的联想。她，是收获的季节――金黄的田野，橙黄的果园；是欢乐的季节――天高云淡，风清月白；是豪壮的季节――香山红叶，钱塘秋潮；是飘渺的季节――皓月当空，桂香暗动；是怀旧的季节――秋风瑟瑟，落叶萧萧……那么，著名作家郁达夫笔下的秋，又是怎么样的呢？

（其四）

　　广州卢福东老师这样导入：

　　上课之前，老师先给大家唱几句歌："我爷爷小的时候常在这里玩耍，高高的前门仿佛挨着我的家。一蓬衰草，几声蛐蛐叫，伴随他度过了那灰色的年华。吃一串冰糖葫芦，就算过节，他一日三餐窝头咸菜么就着一口大碗茶……"，这首歌名叫什么？――《前门情思大碗茶》，歌中唱的是我们国家的哪个城市呢？对，北京。（歌曲可以放录音替代）

　　今天，我们来学习一篇也是写北京的散文，不过，它写的不是北京的冰糖葫芦，也不是北京的大碗茶，而是北京的秋。大家知道，北京现在是我们国家的首都，历史上也曾是几个朝代的都城，但是，在郁达夫写这篇文章的时候，北京已经不是任何朝代的都城，也不叫北京，而是叫北平，因此，郁达夫把它叫做故都。（板书课题）

（其五）

　　湖南张帆影老师这样导入：

　　就风格而言，《故都的秋》（板书），属于隽永类型。肥称为隽，深长叫做永。吃肥美的鸟感到滋味深长，谓之隽永。隽永特别注重一个"味"字。

　　《故都的秋》，不像徐迟的《黄山记》那样气宇轩昂，气量恢宏，气度豁达，气势磅礴；也不像《天山景物记》那样五彩缤纷，金碧辉煌，绮丽迷人。

　　《故都的秋》，更像李煜的"闲梦远，南国正清秋。千里江山寒色远，芦花深处泊孤舟，笛在月明楼。"秋色秋思，境界寥廓而意味深远。更像"无言独上西楼。月如钩，寂寞梧桐深院锁清秋。剪不断，理还乱，是离愁，别是一般滋味在心头。"无言之痛，哀怨凄婉，令人回味无穷。让我们细细品尝《故都的秋》那份隽永吧。

（其六）

　　湖南文继之老师这样导入：

　　历代名家描写秋的佳句举不胜举，描写秋的名篇也数不胜数。有哪位同学能就这些佳句和名篇各举一例？

　　一学生回答：名篇有马致远的《天净沙・秋思》；佳句有王绩的"北风吹故林，秋声不可听"。

　　老师总结：对，这位同学的回答很好。现在我们再来看看关于"秋"的名篇和佳句流传较广的还有哪些。（展示幻灯）

　　 名篇有：

　　 杜甫的《茅屋为秋风所破歌》
　　 王维的《山居秋暝》
　　 欧阳修的《秋声赋》
　　 峻青的《秋色赋》

　　 佳句有：

　　落霞与孤鹜齐飞，秋水共长天一色

　　――王勃《滕王阁序》

　　烽火城西百尺楼，黄昏独坐海风秋

　　――王昌龄《从军行》

　　浔阳江头夜送客，枫叶荻花秋瑟瑟

　　――白居易《琵琶行》

　　万里悲秋常作客，百年多病独登台

　　――杜甫《登高》

　　这些名篇和佳句，有的描写了秋天的萧杀凄凉，抒发了作者深深的悲秋之感，有的刻画了秋天的绚烂多彩，寄寓了作者浓浓的赞秋之情。那么，著名现代作家郁达夫笔下的"秋"会是怎样一种景象？"秋"景如何？"秋"意怎样？请打开课本，快速阅读《故都的秋》。
