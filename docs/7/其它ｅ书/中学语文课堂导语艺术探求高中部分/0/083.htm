难老泉

（其一）

　　这是一篇游记。"难老"既是泉名，同时也蕴含着文章的中心。一看题目就可知道，作者仿佛在告诉我们：我们祖国的悠久历史、源远流长的民族文化、勤劳勇敢的劳动人民，就像这"难老"的泉水一样，"永远活泼"，"青春常在"。

（其二）

　　内蒙古宋亚军老师这样导入：

　　有这样一个颇富情趣的故事，据说有一个硕大无比的馅包，大得惊人，里面的馅儿异常精美。为了得到那精美的馅儿，有一个人手拿铁铲，一直挖进去，他挖呀！挖呀！挖了好多天还没见到馅儿，就在他失望得快要放弃时，他挖到了一块石碑，他满以为肯定快到馅儿了，谁知石碑上的字叫他望而却步，大失所望，终于扔掉铁铲，打道回府了。石碑上是什么字呢？原来是这样几个字："此地离馅三十里。"（学生大笑，教师接着说）今天我们来共同学习《难老泉》（板书），这篇游记和我们以前见过的游记有很大不同。明明是游晋祠的"难老泉"，却从山西写起，写了山西该写"难老泉"了吧，却又写到了山西的省会太原，下面总该写"难老泉"了吧，谁知作者仅点了几句，又宕开笔锋写了"桐叶封弟"的故事和晋祠的情况，然后又不厌其烦地介绍了"晋祠三绝"中的两绝——"宋塑侍女"和"古柏齐年"，七拐八拐，终于写到了"难老泉"，真是"此地离馅三十里"！下面大家读课文，考虑一下该文由"皮儿"到"馅儿"的过程，是不是罗哩罗嗦，离题万里，如不是，体会这样安排材料的好处。
