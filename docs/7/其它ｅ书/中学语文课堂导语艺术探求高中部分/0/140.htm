一个极其重要的政策

　　有位教师这样导入：

　　教师先设问：毛泽东同志1942年9月为《解放日报》撰写的社论——《一个极其重要的政策》，其标题属什么短语？学生不难答出：属于偏正短语，两个定语"极其重要"、"一个"依次递加在中心词"政策"前头。教师再设问，为什么说是"重要"的，又是"极其重要"，并是"一个极其重要"的呢？

　　稍后，教师介绍时代背景，并在此基础上指出：当解放区在军事、经济等方面都遇到了空前严重的困难的时候，为了夺取抗战的最后胜利，我党制定了精兵简政的决策，克服了财经困难，巩固了根据地。这个政策能说不"重要"吗？所以，作者在标题中用"重要"一词修饰中心词，表明此政策非同一般，又用"极其"一词修饰"重要"，突出了"重要"的程度，表明不仅"重要"，而且"极其重要"。当然，精兵简政是克服困难的一个办法，但不是唯一的办法，所以，作者又用数量结构"一个"，说明此政策仅仅是我党一系列政策中的一项。可见，整个标题精当、醒豁，值得我们写作命题时借鉴。
