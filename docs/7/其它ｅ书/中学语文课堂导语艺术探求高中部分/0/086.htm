娘子关前

（其一）

　　 初中阶段我们曾经学过周立波的一篇文章，课题叫什么？该文节选自什么书？谁能简介一下作者。（讨论后小结）我们学过《分马》，节选自长篇小说《暴风骤雨》。周立波，现代作家，原名周绍义，湖南益阳人，"立波"是笔名，由英文Liberty"解放"的译音而来。 1934年加入中国左翼作家联盟。抗战开始，在八路军前方总司令部和晋察冀边区任战斗记者。1948年底完成了长篇小说《暴风骤雨》，除此以外还写有《山乡巨变》、《铁水奔流》等著作。

　　1937年7月，抗日战争爆发，这一年的冬天至1938年的春天，周立波曾作为八路军战地记者走遍了华北前线，行程达2500余里，并于6月份撰写了报告文学集《晋察冀边区印象记》。该集作品热情歌颂了八路军英勇抗日的精神和革命人民伟大的民族气节，控诉了敌人的罪行，揭示了敌人必然灭亡的命运，抒发了作者热爱祖国和人民的思想感情。《娘子关前》就是其中的一篇。

（其二）

　　《威尼斯》是篇游记散文，通过学习，我们知道散文的特点是什么？（略）现在我们来研读周立波写的记事散文《娘子关前》（板书），看作者是以什么为中心，以什么为线索，按照什么顺序，把零散丰富的材料组织成一个有机的整体，达到"形散而神不散"的境界的。
