改造我们的学习

（其一）

　　毛泽东主席的《改造我们的学习》（板书），从辩证唯物主义的世界观的高度，从学风问题上总结了我党两条路线斗争，深入批判了机会主义路线的思想基础――主观主义和教条主义，论述了必须树立理论和实际相结合的马列主义学风，并指明了改造学风的具体途径。本文不但在当时整风中发挥了重大作用，对今天的理论学习仍具有重要的指导意义。

（其二）

　　本文题目中的"改造我们的学习"，既是全文中心论点，又标明了文章的论述范围――如何彻底改变学风方面的问题。"改造"，不是一般的改变，而是从根本上改掉旧的，树立新的。"我们的学习"，这里主要是指我们党内的革命队伍内的学风，指学习指导思想、学习态度等根本性的问题。

（其三）

　　 本文是1941 年5月毛泽东同志在延安干部会议上所作的报告，是当时我党开展的整风运动中的一篇重要文献。《改造我们的学习》这个标题中，"我们"是指全党的同志，特别是党的干部；"学习"是指全党同志的学习，特别是指对马克思列宁主义的学习。这篇重要报告，不仅在当时对党的整风运动具有指导意义，而且在当今这样一个新的历史时期，对于我们建设社会主义现代化强国的伟大事业，仍然具有十分重要的意义。我们中学生学习这篇文章，对于提高自己的思想水平、树立理论联系实际的正确学风，也是非常必要、非常有益的。

（其四）

　　大家先来读一读宋代诗人陆游的一首诗：（投影出示陆游的诗《冬夜读书示子聿》其三）

　　古人学问无遗力，少壮工夫老始成。
　　纸上得来终觉浅，绝知此事要躬行。

　　这是陆游写给儿子的一首诗，劝勉儿子要努力学习。特别是后两句，是陆游从自己一生的读书经历中悟出的一条求知的真理。书本知识固然重要，但实践比书本更重要。诗人告诉我们，仅仅掌握书本知识是远远不够的，应当把学到的知识应用到社会实践中去。拿我们现在的话来说，只有把理论和实践结合起来，才能获得真正的知识。如果理论与实践相脱离，就会犯主观主义错误。今天我们学习毛泽东同志写的专门反对主观主义学风的文章，这就是――《改造我们的学习》（板书）。
