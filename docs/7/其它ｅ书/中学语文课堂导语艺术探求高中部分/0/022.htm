订鬼

　　 山东赵广成老师的导入：

　　今天我们学习订鬼》（板书课题）" 订"在《辞海》上有这样几种解释，同学们看看哪种解释符合题意？（出示小黑板。）

　　 订， 1、约定。如订交。 2、制定。如订规章，订计划。 3、改定，核定。如订正、校对、修订。 4、评议。（"订"是评议的意思。《说文》："订"平议也。"这是"订"的本义，其他义项是引申义）

　　"订鬼"，就是评论一下到底有没有鬼，也可说是评论人们产生"有鬼"观念的根本原因。

　　世界上有没有鬼的问题，从古到今反映了唯物论与唯心论两种世界观的斗争。早在一千九百年前，东汉时期，盛行以神学迷信附会儒家经义的一类书。统治阶级用迷信思想欺骗醉人民。当时，能勇敢地对由朝廷支持下的妖妄思想作斗争的就数王充。王充出身"细族孤门"，一生只做几年小官"得宫不欣，失位不恨，居贫苦而志不倦"。用了三十年的精力写成《论衡》一书，共三十卷，八十五篇，二十余万言。课文就是节选自《论衡·订鬼篇》的开头部分。在这部分里，作者对人们产生"有鬼"观念的根本原因作了最精湛的论述。
