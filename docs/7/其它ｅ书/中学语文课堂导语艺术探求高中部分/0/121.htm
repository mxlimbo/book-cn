我国古代小说的发展及其规律

　　和《谈<水浒>的人物和结构》一样，本文也是一篇文学评论。所不同的是，前者仅就《水浒》一部小说进行评论，而本文是以"我国古代小说"作为评论对象的，所以在谋篇布局等方面与前者有明显的不同，我们学习时应注意比较。为了论证自己的观点，有时必须引用别人的评论，如何根据论证的需要选择合适的内容，以及如何围绕论点，对所引用的内容进行分析、评价，并使之与文章紧密地结合在一起，水乳交融，是我们学习本文的重点，也是本文的难点。
