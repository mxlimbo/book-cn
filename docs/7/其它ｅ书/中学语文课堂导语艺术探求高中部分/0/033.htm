归园田居

（其一）

　　 陶渊明（365一427），字元亮，一说名潜字渊明，晋浔阳柴桑（今江西九江）人，出身于仕宦家庭。二十九岁时，出任江州祭酒，不久就辞官归隐，此后，又作过几任参军一类的小官，时间都很短。因家庭经济状况不佳，到39岁时，他曾亲自参加农业劳动。后来又出任彭泽令。毛泽东同志有诗云："陶令不知何处去？桃花源里可耕田。"诗句中的"陶令"即指陶渊明，"桃花源"是从陶渊明的《桃花源记》一文脱化而来。陶渊明在任彭泽令时，郡里派督邮到县里来，县吏叫他束带迎接。他说："吾不愿为五斗米折腰向乡里小儿"，就离职回乡。这时，他41岁。以后就没再作官。死时63岁。他的作品现存的有诗120多首，散文六篇，辞赋三篇等。

　　《归园田居》共五首，当写于辞官归家的次年，即 公元406年。课本选的第一首，是最能显示陶诗风格特色的一首。诗人对人生进行了深刻的总结，决意弃旧 图新，归隐田园，全诗描绘了诗人由官场回归农村后田园生活的无上欢乐，字里行间洋溢着诗人置身于大自然的怀抱之中的欢快心情，和对官场生活的厌恶之情。

（其二）

　　山东马学声老师的导入：

　　本诗选自《陶渊明集》。作者陶渊明（365一一427 ），字元亮，浔阳柴桑（今江西省九江县）人。东晋亡国后改名陶潜。东晋著名诗人。曾任江州祭酒、镇江参军、建威参军，因不满官场的污浊，每次赴任旋辞。年四十一，出任彭泽县令，仅八十余日，因不愿折腰媚迎郡里派来的"督邮"，毅然弃官归田，为世人所熟知的"我不能为五斗米折腰向乡里小人"语，即由此出。此后二十多年，亲自参加农事，接近农民，对农村生活多有体验。他有长期的田园生活实感，描写农村园田日常生活的诗作很多，被称之为"田园诗人"。他的田园诗，画面淳朴自然，感情真挚深切，表达了诗人弃官村居，"躬耕自资"后喜悦的生活情趣。传世名篇甚多，《归园田居》五首最为著名。

　　 课文所选的是其中的第一首，这首诗大概是诗人自彭泽县归隐后第二年写成。
