劝学

（其一）

　　我们初一时学过一篇课文叫《为学》，大家还记得是什么内容吗？（用设喻方法，讲有关学习的道理。）今天我们要学习的《劝学》也是一篇谈学习的文章。

　　本文的作者荀子（约公元前313一前238），名况，字卿，战国末期赵国人，曾游学于齐，当过楚国的兰陵令。后来失官居家著书，死后葬于兰陵。荀子是我国古代的思想家、教育家、是先秦儒家最后的代表，朴素唯物主义思想的集大成者。韩非和李斯都是他的学生。他反对迷信天命鬼神，肯定自然规律是不以人们意志为转移的，并提出"制天命而用之"的人定胜天的思想。他强调教育和礼法的作用，主张治理天下既要靠"法制"，又要重视教化，兼用"礼"治，强调"行"对于"知"的必要性和后天学习的重要性，认为后天环境和教育可以改变人的本性。

　　荀子的著作有《荀子》二十卷。该书由《论语》、《孟子》的语录体，发展为有标题的论文，标志着古代说理文的进一步成熟。他的散文说理透彻、语言质朴、多排比句，又善用比喻。《劝学》是《荀子》的第一篇。我们的课文是原文前几段的节录。

（其二）

　　教师先在黑板上写一个"劝"字，问道：这个字是什么意思？学生往往会不假思索地回答："劝告"。教者趁此在"劝"字后加上一个"学"，再问："劝"是什么意思？学生才会领悟：原来它还有"劝勉"的意思。接着就讲："劝"的繁体字是形声字，凡是用"力"作声旁的字，多数有给人们勉励的意思，如"励"、"努"等。这是"劝"的本义，而在现代汉语中，"劝"，解释为"劝阻"，词义已经转移了。作者在这篇以《劝学》为题目的文章中，勉励人们要不停止地坚持学习，只有这样才能增长知识，发展才能，培养高尚的品德。《劝学》的"劝"起着统领全篇的作用。

（其三）

　　我们今天学习《劝学》（板书），这一文最大的特点是运用比喻论证。那么作者在本文中共用了多少比喻？请大家用快读法找一找，划出运用比喻的句子。一、二段中各用了几个比喻？三段中呢？（说九个的，说十个的，各抒己见。经过仔细分析，统一认为第三段中有十个比喻，全文共计二十个比喻。）

（其四）

　　一位老师这样导入：

　　如果有可能的话，我想编一本《古今劝学名言大观》，（一笑）涉及到这方面的格言同学们肯定都有积累，今天我再给大家介绍几句，以期共勉。

　　玉不琢，不成器；人不学，不知道。
　　知之者不如好之者，好之者不如乐之者。
　　非学无以广才，非志无以成学。
　　好读书不好读书，好读书不好读书。
　　书到用时方恨少，事非经过不知难。
　　蹉跎莫遣韶光老，人生惟有读书好。
　　年将弱冠非童子，学不成名岂丈夫！
　　学如逆风行舟，不进则退；心似平原走马，易放难收。
　　人一能之，己百之；人十能之，己千之。果能此道也，虽愚，必明；虽柔，必强。

（其五）

　　山东杨天顺老师的导入：

　　荀子，名况，战国末期赵国人。时人尊称他为荀卿，汉时避宣帝讳又称孙卿。是我国古代著名的思想家、教育家。他先后游历过齐、秦、赵、楚等国，曾经长时期的在齐国首都临淄（现山东省淄博市临淄区）西门（稷门）外"稷下学宫"讲学。当时稷下是文化学术中心，百家云集，各派横议，荀子得以接触各家各派思想。在稷下学宫荀子"最为老师"，并曾"三为祭酒"（学宫之长），中间曾赴秦考察，提出用王道统一中国的主张，秦不能用，于是返齐。后又去楚。荀子前255年到楚国，梦相春申君任为兰陵（山东枣庄市东南）令。中间曾去赵，与赵王议兵。晚年居兰陵著书，死后葬于兰陵。他一生主要从事教学和著述，现存《荀子》三十二篇，他的学生著名的有李斯、韩非等人。

　　荀子是我国著名的思想家，先秦的最后一位大师。他不仅集儒家大成，而且集百家之大成，荀子写了《非十二子》批判百家之短，却又吸取百家之长，融汇贯通，自成一家，堪称"杂家的祖宗。"（《十批判书》）

　　《荀子》，在思想内容上融百家之长，自成一家。在语言艺术上有它独特的风格，笔势雄健，句法整齐，文气浑厚，论点鲜明，特别是善用比喻，议论风生，说理透辟，对后世论辩文的发展很有影响。

　　《劝学》是《荀子》首篇。全文十四段，主要论述了学习的目的、意义、内容、方法等问题。本课节选的三段，论述了学习的重要性以及学习必须积累、必须坚持不懈、必须用心专一的道理。这在当时条件下是进步的，在今天讲，有其借鉴的意义，当然荀子劝学的目的是为封建地主阶级培养人才的。
