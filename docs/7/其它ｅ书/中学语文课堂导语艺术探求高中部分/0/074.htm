《<梦溪笔谈>二则》

（其一）

　　本文作者沈括，字存中，北宋时期的科学家、政治家。曾积极参加了王安石的变法革新运动，为制定并推行新法做出努力。他在晚年潜心研究学术领域里的前人成果，认真总结自己的实践经验，致力于著书立说，撰写了《梦溪笔谈》。

　　《梦溪笔谈》是我国古代科学技术史上一部重要著作，反映了我国古代科学技术的辉煌成就。著作以笔记体形式写成，涉及天文、地理、气象、生物、数学、物理、医药等诸多内容。《梦溪笔谈》不仅记录了沈括自己的研究成果，还总结了我国古代，特别是北宋时期自然科学、应用技术方面的辉煌成就。如毕升发明活字印刷（即初中语文课本中的《活版》），这是中国古代的四大发明之一，是中国人民对人类文明作出的巨大贡献，该书的记述是最早的也是唯一的记录。另外，河工高超，木工喻皓的重要经验，该书无不作详细记述。只是由于封建统治阶级的偏见，这本书没有得到足够的重视。直到近代、现代，由于世界科学技术的发展，人们才逐步发现了这本书的价值。英国剑桥大学的教授李约瑟博士在其所著《中国科学技术史》中，称《梦溪笔谈》为"中国科学史的坐标"，给予高度的评价。 《采草药》和《雁荡山》都是科技说明文，作者以事实材料为依据，提出并阐明科学见解，我们今天读这样的文章，固然可学到一些科学知识，但更主要的是学习作者实事求是的治学态度和创新精神，学习文章说明事理的顺序、方法，严谨平实的说明语言。

（其二）

　　 山东余全忠老师的导入：

　　 沈括是我国北宋时一位著名的政治家，同时也是一位卓越的科学家。他生平著作宏富。《梦溪笔谈》是他晚年退居梦溪园（今江苏镇江东部）时，总结了学术领域中广泛的知识和经验写成的综合性科学论著。内容丰富多彩，很有学术价值。英国剑桥大学李约瑟教授，称这部书为"中国科学史的坐标。"

　　选自《梦溪笔谈》的这两则短文，都是以科技为内容的笔记文体的说明文。在《雁荡山》一文中，对雁荡山特殊地貌成因的提出，比所谓近代地质之父的英国人郝登在1788年提出的流水侵蚀的学说，早了七百多年。《采草药》一文，他不拘古法，提出采草药不可"拘以定月"的观点，也是很有见地的。这些都表现了沈括的实事求是的科学态度和敢于创新的精神。
