与朱元思书

（其一）

　　本文作者吴均，字叔庠，梁朝吴兴故彰（今浙江省安吉县）人。他出身贫寒，为人性格耿直，好学，有俊才。他的诗文"清拔有古气"，曾得到当时文坛领袖沈约的称赏。当时有的人效仿他这种文章，被称作"吴均体"。吴均著有《后汉书注》、《齐春秋》、《庙记》等书，后来都失传了。

　　"与"给予。"书"，信。文章题目的意思是写给朱元思的一封信。信中述说旅行所见，寥寥一百四十余字，把富春江富阳到桐庐一段的山光水色，描绘得生动逼真，使人读后悠然神往，就像置身于奇山异水之间。本文可算是六朝骈体文中描写山水景物的妙品。下面我们就来欣赏这一篇妙文。

（其二）

　　有位老师这样导入：

　　同学们，大家知道严光的故事吗？严光，字子陵，与汉光武帝刘秀曾是同学。刘秀即位后，严光改姓变名，隐身不仕。刘秀思念他的贤能，派人去找，终于在富春江边找到了。他在那里钓鱼。刘秀派人用重礼征召他，第三次才请到，把他安顿停当，住好，吃好。司徒侯霸就对他说，皇帝很想亲自来见你，只是不合礼仪。你有闲暇，请屈驾去谈谈吧。严光不回答，口述了一封信给刘秀，祝贺他当皇帝，又说，怀仁辅义天下高兴，一味阿谀，顺承旨意就该斩首。刘秀见信骂了一句"小子狂妄一辈子不改"，当天便驾车赶到严光的住所。严光睡着不起来，刘秀便坐在床边摸着严光的肚皮，说："子陵啊，你不可以帮助我治理天下吗？"严光不应。后来又把严光请到宫内，两人谈心叙旧多日。一日，刘秀得意地问严光："我比起从前，怎么样？"严光说："只是长肥了点儿。"（学生笑）晚上一起睡觉，严光把脚搭在皇帝的肚皮上。

　　后来，刘秀多次对严光委任官职，他始终不就。严光死后，他钓鱼的地方筑起了"严子陵钓鱼台"，成了富春江上一处名胜。范仲淹为其祠堂作记时赞扬道："云山苍苍，江水泱泱；先生之风，山高水长。"

　　同学们，严光远离世俗官场，固然因为他清高脱俗，也说明富春江的秀美风光对他有巨大的吸引力。实际上许多隐士，认为尘世污浊黑暗，而大自然纯洁不染。在他们隐逸思想的形成过程中，美丽的大自然的作用不可小视。如此看来，同学们，你们看富春江美不美呀？

　　"美！"学生异口同声。那好，请大家阅读课文。
