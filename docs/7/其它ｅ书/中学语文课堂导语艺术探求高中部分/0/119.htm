我的空中楼阁

（其一）

　　这篇课文通过对一所建在山上的住宅小屋及其周围环境的描写，表达了作者对"独立的"、"安静的"生活的向往和对大自然热爱的感情。以"我的空中楼阁"为题，一语双关：既指家居小屋建于山上，在烟雾迷朦中，犹如天上的楼阁；又指幻景中的"空中楼阁"——理想中的生活环境，表明作者对超然物外的"独立的、安静的"生活的向往，可见，题目隐含了中心。

（其二）

　　山东李中喜老师这样导入：

　　我们形容一个美丽、漂亮的女子时，常用"沉鱼落雁"、"闭月羞花"，这两个词语实际上指我国古代的"四大美人"。谁能说一下这"四大美人"分别指谁？ 学生听到这个问题稍一沉思，便纷纷举手回答：沉鱼，指西施；落雁，指王昭君；闭月，指貂蝉；羞花，指杨贵妃。但同时也感到迷惑不解，不知老师要讲什么内容。于是，老师趁势点上一句：台湾女作家李乐薇就把她心目中的小屋比喻为美女"眉梢上的痣一点。"
