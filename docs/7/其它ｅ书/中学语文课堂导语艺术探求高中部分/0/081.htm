纳谏与止谤

（其一）

　　唐太宗李世民曾经说过："以古为鉴，可以知兴替。"意思是说，以古代的事情为镜子，可以知道政权兴衰更替的原因。本文采用读后感的方式，通过齐威王"纳谏"和周厉王"止谤"两件事及其结果的对比，"以古喻今"，评论现实，具有很强的针对性和说服力 。

（其二）

　　 "纳谏"的故事出自《战国策·齐策》中的《邹忌讽齐王纳谏》，这篇文章在高一第一学期已经学过。齐威王是位圣君，他那宽大明智、精神高尚的形象，虽隔几千年，仍栩栩如在眼前。"止谤"的故事出自《国语·周语》中的《召公谏厉王止谤》。文章说："厉王虐，国人谤王。召（同邵）公告王曰：'民不堪命矣！ 王怒，得卫巫，使监谤者。以告，则杀之。国人莫敢言，道路以目。王喜，告召公曰：'吾能弭谤矣，乃不敢言。'召公曰：是障之也。防民之口，甚于防川。 川壅而溃，伤人必多。民亦如之。……王弗听，于是国人莫敢言。三年，乃流王于智"。意思是：周朝的厉王，对百姓十分残暴，全国的百姓都对他批评指责。召公劝他说："老百姓受不了暴虐的政令了。"厉王大怒，找到一个卫国的巫人，派他监视批评指责自己的人，只要他有报告，就把被告发者杀掉。百姓都不敢说话了，在路上相遇，只是互相看看，心里有怨怒，可什么话也不敢说。厉王为此很高兴，对召公说"我能消除诽谤了，谁也不敢说怪话了。"召公说："你这就好比治水是用堤坝拦阻啊。想堵塞住百姓的嘴，比治洪水还难，河水被阻塞不通，一定要决口，淹死的人将更多。堵人民群众的嘴也是这样。……"厉王不听劝告，于是全国没有一个敢公开议论国事的人。只过了三年，厉王就被人民赶走，放逐到智这个地方去了。

　　《纳谏与止谤》这个标题，对比鲜明，发人深省。文章通过对齐威王和周厉王对谏谤的不同态度及其不同结果的述评，联系现实，告诫各级领导同志要正确对待批评意见，敢于"悬赏纳谏"，改进工作，推动四化建设。

（其三）

　　古人云："人非圣贤，孰能无过？"既有过，就难免受到批评。怎样对待批评呢？《纳谏与止谤》一文运用"以古喻今"的写法，采用对照式的论证结构，给我们做了明确的回答。下面让我们共同来学习课文《纳谏与止谤》。
