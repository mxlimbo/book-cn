廉颇蔺相如列传

（其一）

　　我们读书作文时，经常遇到"完壁归赵"、"负荆请罪"等成语。这些成语的出处就在我们今天要学的课文《廉颇蔺相如列传》（板书）。我们学习这些课文，不仅可以进一步理解这些成语典故，学到一些文言知识，而且将受到古人那种勇敢机智的品质，以及识大体、顾大局的爱国情操所激励，从而培养我们的爱国主义精神。

（其二）

　　 本文选自《史记·廉颇蔺相如列传》。司马迁把廉颇、蔺相如合在一起写，也就是"合传"，这是为什么呢？（让学生稍稍议论）这是因为他们两人先后在赵王麾下为将相，且一为文臣，一为武将。廉颇勇于作战，功勋卓著，为赵国立下汗马功劳；蔺相如善于外交，智勇双全，在赵国史册上写下了辉煌一页。不仅如此，他们两人还有一段由互相嫌怨，到重归于好的过程。合写则利于将他们的不同性格进行相互比较、相互映衬。
