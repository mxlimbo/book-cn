左忠毅公逸事

（其一）

　　上节课我们学习了太史公的《信陵君窃符救赵》，从体栽上看是纪传体，一般用翔实的史事全面叙述人物生平事迹，属于正史。在人物传记中还有一类属于野史的逸事，它与正史不同，往往用人生中一小部分的行事（多是散失的没有流传的），来表现人物的思想。这类文章我们以前接触过《记王忠肃公翱事》，今天我们再来学习方苞的《左忠毅公逸事》。

（其二）

　　大家还记得《五人墓碑记》中"矫诏纷出，钩党之捕遍于天下"，这句话是指什么事情说的？（明熹宗时期魏忠贤专政，假借皇帝命令杀害贤良正直的人。）"大阉之乱缙绅而能不易其志者，四海之大，有几人欤？"这句话是不是说所有官员全都依附魏忠贤？（不是，也有少数官员和在野的东林党人跟魏忠贤作坚决的斗争。）

　　我们今天学习的这篇课文所描写的左光斗，就是站在反阉党斗争的最前列的，并为此献出了他的生命。他的名字是永垂不朽的。正因为如此，方苞写了这篇文章来纪念他。

（其三）

　　《左忠毅公逸事》（板书），这篇文章是写明朝人左光斗的故事。左忠毅公，即左光斗（公元一五七五上一一六二五年），桐城（今安徽省桐城县）人，字遗直。万历时中进士，做过大理少卿、左佥都御史。他是东林党重要成员，因上书弹劾宦官魏忠贤，被诬陷入狱，受酷刑而死。 "忠毅"是左光斗的谥号。封建时代，帝王、贵族、大臣等死后，依据他生前的事迹、功绩和品德追加的称号，叫"谥号"。"忠毅"，有忠烈、刚毅的意思。"公"是对男性前辈的尊称。"逸事"，逸，通"轶"和"佚"，散失的意思。多指史书未载的事情，"前史所遗，后人所记"，叫作"逸事"。
