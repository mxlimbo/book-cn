景泰蓝的制作

（其一）

　　"景泰蓝"是我国一种传统的手工艺品。它的名字的来历，是因这种手工艺品的制作到明代景泰年间达到了鼎盛，又因当时多用宝石蓝、孔雀蓝等蓝色釉料，所以得名"景泰蓝"。后来不管是什么时代的同类产品，大家都习惯称它为"景泰蓝"。 "景泰蓝"和我国许多传统的手工艺品一样，是中华民族灿烂文化中的璀璨明珠，历来受到老人的珍爱。叶圣陶先生生前曾想写一组文章，把每种手工艺品的制作过程都记录下来，可惜只写了两篇，本课文是其中的一篇。在科学技术日益发达，手工艺绝活日见珍稀的今天，来读读叶圣陶先生的这篇遗作，会激起我们对悠久历史文化的缅怀，对我国手工艺工人精湛技艺的赞叹，唤起我们的创新意识。

（其二）

　　有位老师这样导入：

　　先找来一两件景泰蓝制品，让学生传看，并告之那上面曲曲弯弯的东西是粘上去的铜丝，整个成品经过多道工艺制成。然后说道：

　　《景泰蓝的制作》这篇课文就是向我们介绍其制作过程的，而且所需原料不贵，手工工艺甚至我们也能学会。

　　这时学生的学习兴趣被初步调动起来，很想读这篇课文了。教师趁热打铁，对学生提出了以下明确要求：

　　假如我们学校打算筹建一个景泰蓝工厂，工厂将设置哪几个车间，包括哪几道工序，需购进些什么原料呢？大家都是厂长候选人，请将以上内容分类列表。作为可行性报告的一部分，读给同学听，并以此参加厂长竞选。

　　此言一出，学生立刻兴奋起来，读书热情空前高涨，未经老师任何指导，很快就按要求列出了表格。有的同学还就某道工艺应安排在哪个车间展开争论，气氛热烈。

（其三）

　　广东钟德赣老师这样导入：

　　上课了，老师在讲台前拿出一样小小的玩艺儿笑着问道："同学们谁知道我这只手镯是什么做的？"同学们纷纷引颈细看。

　　"是金！"

　　"是铜！"

　　"是陶瓷！"

　　老师一边笑着摇头一边说："都不是，应该是景——泰——蓝。"（板书"景泰蓝"三字）

　　"同学们再猜猜我这只小小的手镯买来要多少钱？

　　 "5元！""8元！"……

　　 听见老师说是18元买来时，有同学脱口而出："怎么这么贵？"

　　 "物品价格的高低主要是由制作时投入的成本多少决定的。景泰蓝是我国传统的手工艺品，它为什么如此地昂贵呢？今天我们就来学习一篇文章，——《景泰蓝的制作》。同学们一定会从中找到这个问题的答案。"（边讲边板书"的制作"）学生纷纷打开书。
