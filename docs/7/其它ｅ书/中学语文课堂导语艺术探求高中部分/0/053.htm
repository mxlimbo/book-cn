就是那一只蟋蟀


（其一）

　　 本诗作者流沙河，原名余勋但，1931年生，四川省金堂县人。50年代任《川西日报》副刊编辑，开始陆续发表小说和诗歌，1956年和别人一起创办《星星》诗刊，1957年因此被错划为右派分子，"文革"后正式平反。台湾诗人余光中先生，抗战期间，在四川省江北县的悦来场读了五年中学，晚上在窗前做作业，窗外有一只蟋蟀伴唱。后来，他到了台湾。1982年6月17日，他在给诗人流沙河的信中谈到这件事，说："当我怀念大陆的河山，我的心目中有江南，有闽南，也在无穷的四川。在海外，夜间听到蟋蟀叫，就会以为那是在四川乡下听到的那一只。"于是，流沙河写了《就是那一只蟋蟀》，一唱一和，一觞一咏，反复吟唱，余味无尽。

　　蟋蟀是一个引线，象征着住在海峡两岸的两位诗人心有灵犀，双方同时具有了相同的心态、相同的艺术追求，才会结成这真实而又深厚的友情，………。

（其二）

　　大家已经预习了课文，一定知道蟋蟀曾是历代诗人反复咏唱的形象。在《诗经》中唱过，在《古诗十九首》中唱过，在《木兰辞》中唱过，"在姜夔的词里唱过"；"劳人听过／思妇听过"，"孤客听过／伤兵听过"，牵动无数人的心肠；现在又是"那一只蟋蟀"的鸣唱，而引发了海峡两岸诗人的无限神思。小小蟋蟀竟有这般魔力，是不是因为秋虫卿卿，最能惹人愁绪呢？ 其实这只是表面现象，咏蟋蟀不仅仅是借物寄托情思，不仅仅是某个人的个人感慨，而是我们汉民族从古至今，特有的表达感情的一种方式。正因为这是我们民族所共有的抒情方式，所以我们由蟋蟀能感受到古人的情思、今人的胸怀。蟋蟀不仅是客观自然中的物体，更深的寓意是反映我们民族共同的心理心态，反映我们中国人共同的血缘和共同的文化。也正因为如此，才能使海峡彼岸有所感，海峡此岸有所应。在本诗中，两位诗人听到同一蟋蟀叫声，从字面上理解，表现的是Y诗人思念家乡的感情以及流沙河对海峡彼岸的诗友的思念，即所谓"心有灵犀一点通"；而从蟋蟀这一形象所包含的深刻的寓意来理解，则表现的是汉民族特有的历史背景和文化传统，突出同一祖先同一血脉的感情。"中国人有中国人的心态／中国人有中国人的耳朵"，正因为有共同的心态、共同的"耳朵"（这个"耳朵"指心理上的感觉，而并非听觉），才能使相隔海峡的两位诗人听到"同一"蟋蟀在鸣叫。

（其三）

　　 湖南张帆影老师这样导入：

　　床前明月光，疑是地上霜。
　　举头望明月，低头思故乡。

　　古代诗人李白这首诗，不着一个"愁"字，却写尽了诗人浓浓的乡愁。

　　 小时候，
　　 乡愁是一枚小小的邮票。
　　 我在这头，
　　 母亲在那头。
　　 长大后，
　　 乡愁是一张窄窄的船票。
　　 我在这头，
　　 新娘在那头。
　　 后来啊，
　　 乡愁是一方矮矮的坟墓。
　　 我在外头，
　　 母亲在里头。
　　 而现在，
　　 乡愁是一湾浅浅的海峡。
　　 我在这头，
　　 大陆在那头。

　　台湾诗人余光中以"乡愁"为题，反复吟唱，直抒胸臆，同样写尽了那浓得化不开的乡愁，还有那祈盼统一的心愿。

　　大陆诗人流沙河的《就是那一只蟋蟀》，还是写乡愁，当然也还是写渴望统一，然而仅仅只写了这两层意思吗？
