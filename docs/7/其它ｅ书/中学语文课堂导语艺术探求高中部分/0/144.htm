义理、考据和辞章

（其一）

　　提问：我们初中曾学过施东向的一篇什么文章？主要内容是什么？（初中第五册学过施东向的《关于写文章》-文。文章开头说："写文章，是交流思想、传播经验的一种方法"，写文章"也是整理我们的思想和经验，使之明确化、条理化的一种方法。"这篇文章主要论述了文章要"有物有序"，即要有内容和有条理的问题。）

　　今天我们将学习他的另一篇文章——《义理、考据和辞章》（板书课题），这篇文章与《关于写文章》无论从发表的时间，还是从内容看，都很相近，堪称姐妹篇。从发表的时间看，是同年发表，从写作的内容看，谈的都是关于写文章方面的问题。特别是在《关于写文章》的结尾，作者写道："好文章是在大量材料的基础上，经过踏踏实实的研究才能写出来的。这种文章在义理、考据和辞章等方面，总是经过认真努力的。"所不同的是，《关于写文章》谈的是写各种文体皆应注意的共性问题，而《义理、考据和辞章》谈的是如何写议论文的专论。

（其二）

　　1958年"大跃进"运动以后，"左"的思想更加膨胀，浮夸风更加严重，对文风也产生了直接影响。许多人写文章不是从客观实际出发，进行科学的具体的分析，而是用一些空洞的理论和词句代替科学论证；不顾材料是否准确可靠，甚至吹牛皮，说假说；把讲究文章的写作技巧和语言表达看成是搞形式主义。《红旗》杂志针对这种有害的文风，在1959年12期和14期发表了两篇署名施东向的文章，题为《关于写文章》和《义理、考据和辞章》，驳斥了当时的一些错误看法，阐述了写作中的一些基本规律和要求。今天我们学习的是其中的一篇——《义理、考据和辞章》（板书）。

　　本文的题目借用了清朝学者戴震和散文家姚鼐的说法。"义理"指文章的观点，"考据"指文章中所使用的材料，"辞章"指文章的语言表达形式。显然，义理和考据属于文章内容方面的问题，辞章属于文章形式方面的问题。这个题目告诉我们，本文的主要内容是阐述写文章必须解决的思想内容和语言表达形式这两方面的问题。

（其三）

　　写文章，难；写好一篇文章，更难。因为，它除要有正确鲜明的义理，有准确可靠的材料之外，还要有漂亮的形式。那么，义理、材料、形式这三者之间的关系是什么呢？我们今天所要学的《义理、考据和辞章》，就是为了解答这个问题的。除此之外，这篇文章的论证结构也是我们的学习重点。

（其四）

　　本文作者"施东向"，是胡绳、王忍之和郑惠合用的笔名。他们为什么要写这篇文章呢？1958年起，由于"左"的思潮膨胀，浮夸风严重，对文风也产生了直接影响，许多人写文章不是从客观实际出发，进行科学的具体的分析，而是用些空洞的理论和词句代替科学论证，虚张声势；不顾材料是否可靠，甚至吹牛皮，说假话；把讲究文章的写作技巧和语言的表达看成吹牛皮，说假话，形成了一股不正的文风。针对当时的这种有害文风，《红旗》杂志于1959年接连发表了施东向的两篇文章：《关于写文章》、《义理、考据和辞章》，驳斥了当时的一些错误看法，阐述了写作方面的一些基本规律和要求。本课文便是其中的一篇。今天我们学习这篇课文，对于我们树立正确的文风，仍具有一定的现实意义。

（其五）

　　江苏王崇凤老师这样导入：

　　同学们，请问你们是否喜欢看MTV？（气氛立即活跃起来，学生纷纷表示喜欢）告诉你们，老师也喜欢看。特别是每年在全国音乐电视大赛中获奖的一些作品，其美妙的旋律、精彩的画面和积极的主题，浑然一体，让人们叹为观止。而时下有些VCD视盘里的歌曲，却处理得不伦不类，常让你"笑"为观止。例如《我爱你塞北的雪》这首歌，画面上连雪的影子也找不到，却只见一个外国妙龄女郎身着三点式泳装，正一步步向碧波深处走去。（学生大笑）大家别笑，其实我们作文中也常出现类似的情形。内容与形式完美地统一，就能给人以享受；反之，只能给人以难受。关于这方面，施东向同志比我讲得要好。现在我们就来读〈〈义理、考据和辞章〉〉，看他是怎么讲的（板书课题）。
