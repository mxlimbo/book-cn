送东阳马生序

（其一）

　　 本文选自《宋学士文集·朝京稿》，是作者宋濂写给他同乡后辈马君则的一篇赠序。文中生动地描写了他幼年刻苦学习、不畏艰苦的经过，劝勉后辈要珍惜良好的学习条件，发愤读书，争取在学业上有所建树。课文所叙的学习内容和我们今天不同，但作者那种勤奋好学的精神，在今天对我们仍有借鉴意义。

（其二）

　　本文作者宋濂（1310一1381），明初文学家。字景濂，号潜滨，浦江（今浙江省义乌县西北）人。元末隐居龙门山著书。明初接受明太祖朱元璋的征聘，任江南儒学提举，后负责纂修《元史》，官至翰林学士承旨、知制诰，为明朝开国文臣之首。当时朝廷上关于祭祀、朝会、诏谕、封赐的文章，大多由他执笔写成。晚年辞官回家。因长孙宋慎犯法，全家流放茂州（今四川省茂汶羌族自治县一带），在半路上生病死去。他是元末明初的散文家。做官之前，他的文章所写的社会生活面较广，文字简练朴素。做宫之后，文章内容大都歌颂"王朝贤君"，宣扬忠孝节义，风格也变得"雍容典雅"。他在明初文名最高，传说高丽、日本也出重价收买他的文集。我们今天要学习的这篇《送东阳马生序》，就是他写给他的同乡后辈马君则的一篇赠序。

（其三）

　　黑龙江杨伯威老师的导入：

　　这是宋濂晚年写的一篇著名的劝学文章。宋濂，明初著名文学家。少时家贫，常借别人家藏书刻苦攻读。曾受业于元末古文大家吴莱、柳贯、黄缙，以文章著名。他的散文简洁明快，与刘基、高启为"明初诗文三大家"。明初接受明太祖朱元璋征聘，先后两次主编《元史》，后历任国子司业、侍讲学士，最后做到翰林学士承旨、知制诰。作者年老辞官以后，洪武十一年又从家乡到南京入朝，为南京国子监一个姓马的学生写了这篇赠序。

　　赠序是一种特殊的文体，为赠人而写，它和为评介著作而写的书序不同。
