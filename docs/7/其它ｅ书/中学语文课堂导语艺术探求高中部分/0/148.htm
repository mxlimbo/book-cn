游褒禅山记

（其一）

　　在初中我们学过哪些用文言文写作的游记？（《小石潭记》、《登泰山记》、《醉翁亭记》……）大家能说一说这些游记共同的侧重点是什么吗？（抓住景物特点，浓笔重彩，精描细绘，表现山水之美。）

　　我们今天学习王安石的《游褒禅山记》（板书课题）。这篇游记与我们过去学过的那些游记不同，它不重在表现山水之美，而是在记游的基础上，用酣畅的笔墨进行大量议论，记游实际上只是个引子，说理才是全文的中心。学习这类游记有助于提高我们的思辨能力。

（其二）

　　褒禅山，在今安徽省含山县北。原名花山，唐代一位高僧，法名慧褒，在这座山上住过，死后又葬在这里，因而得名"褒禅"。课文是王安石在宋仁宗至和元年（1054年）任舒州通判时写的。文章记叙的是一次未能尽兴的游览，作者深感后悔和遗憾。照理说，这样的游览并没有多少值得可写的东西，可是，作者却由此得出两点深刻的启示：一是达到目标，要有志、力、物等主客观条件，而"志"最重要；二是治学必须"深思而慎取"。文章虽然写的是游山，实际是以游山作比喻，表现作者的人生感悟，阐明深刻的人生哲理。

　　王安石在一生从政、治学的道路上，坚毅不拔、直道前行。他在宋神宗时作宰相，认准了"变法"于国有利，决心推行新法，尽管遭到方方面面的强烈反对，但毫不动摇，仍然坚持自己的主张。列宁曾经说过："王安石是中国十一世纪的改革家。"在研究学问方面，他独立思考，创发新义。在文学成就方面，他的诗文雄健遒劲，是唐宋八大家之一。了解作者生平的这些方面，能更深刻地理解本文所蕴含的深意。

（其三）

　　河南尹连顺老师这样导入：

　　首先给学生介绍毛泽东的七绝诗《为李进同志题所摄庐山仙人洞照》："暮色苍茫看劲松，乱云飞渡仍从容。天生一个仙人洞，无限风光在险峰。"学生通过诵读领会了"无险风光在险峰"一句的含义。随后因势利导：

　　今天我们要学习的王安石的《游褒禅山记》，就含有类似的深邃哲理。下面请同学们读课文，从文中找出能够作为"无限风光在险峰"注脚的语句（"世之奇伟、瑰怪、非常之观，常在于险远"）。

　　这样，自然而快捷地把学生的思维活动引向了对新课的学习中。
