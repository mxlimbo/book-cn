谈白居易和辛弃疾的词四首

　　这是一篇诗歌评论。要读懂诗歌评论，先要做好三点准备：一是熟悉所要评论的诗歌，以便进入诗词所描绘的境界。二是整理一下有关的诗歌理论，以这些理论知识作为我们分析、评论诗词的解剖刀，有了它们，面对诗词时我们就不会无所适从了。三是自己应先对所评论的诗词作一点分析，否则没有一点自己的感受、理解，就去读别人的评论文章，有如嚼别人嚼过的馍，味道终究不够纯正，而且也不易深刻地理解别人的评论的妙处。只有自己先行作些分析，再将自己的分析和别人的评论文章相比较，看看哪些地方自己也分析到了，哪些地方是自己没有分析到的，为什么没有能分析到，别人分析评论的方法比自己的方法高明在什么地方。经过这个过程，再去读评论文章，自然会有更大的收获。

　　因此，我们学习这篇评论文章，不仅要理解、体会文中所评几首词的妙处，而且要思考、琢磨诗评者是如何读诗，又是如何把自己读诗的感受、认识表达出来的。
