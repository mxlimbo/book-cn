人的正确思想是从哪里来的

（其一）

　　工作中有些人精明能干，"点子"很多，"点子"从何而来？要解答这个问题，请认真学习毛泽东同志《人的正确思想是从哪里来的？》这篇哲学论文。这篇文章简明扼要地解说了马克思主义认识论的基本原理，阐述了人的正确思想从何而来的科学道理。我们党的方针、政策、路线的制定，都是以马克思主义认识论作为理论基础。我们要正确地理解和贯彻党的方针、政策、路线，要正确地认识世界和改造世界，就要掌握这一基本原理。

（其二）

　　山东胡甲林老师的导入：

　　本文选自《毛泽东著作选读》（乙种本）。它是毛主席一九六三年五月二十日主持起草《中共中央关于目前农村工作中若干问题的决定（草案）》时，为这个文件所写的前言的一部分，是毛主席的一篇重要哲学著作。在这篇文章中，毛主席再一次着重指出人的正确认识，只能从社会实践中来；毛主席把由实践到认识，再由认识到实践，精辟地表述为由物质变精神和由精神变物质的"两个飞跃"，科学地概括了认识论的基本规律，对马克思主义理论作出了新贡献。毛主席还从党内很多人不懂得认识论的实际状况出发，阐述了在全党"进行辩证唯物论认识论的教育"的重大意义。我们重温这篇文章有着重大的意义。
