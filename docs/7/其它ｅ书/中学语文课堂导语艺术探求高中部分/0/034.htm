过秦论

（其一）

　　 在《改造我们的学习》一文中，毛泽东同志批评了那种"割断历史，只懂得希腊，不懂得中国，对于中国昨天和前天的面目漆黑一团"的主观主义学风，教导我们要"认真地研究历史"，"不但要懂得中国的今天，还要懂得中国的昨天和前天"。为什么要"认真地研究历史"？一句话，吸取历史的经验教训，古为今用。对于这个道理，古人很早就懂得。西汉贾谊就说过："前事之不忘，后事之师也。"他的史论《过秦论》（板书），就是在这种思想指导下写的，通过总结秦不施仁义二世而亡的教训，来讽谏汉朝统治者。虽然由于历史局限，作者不可能用历史唯物主义观点来分析和找出秦朝灭亡的根本原因，但他针对当时现实而论史的精神，今天仍值得我们学习。

　　《过秦论》共有上、中、下三篇。我们学的是上篇，文中概括叙述了秦王朝的兴亡过程，揭露秦始皇的暴政，最后指出"仁义不施而攻守之势异也"是迅速灭亡的根本原因，这是从史实中提炼观点。到了中篇，作者对这个观点作了分析，并进一步指出秦二世不顾"天下之嗷嗷"，继续实行暴政，"重之以无道"，然后从正面提出治理天下的原则："牧民之道，务在安之而已。"这是作者为文的本意所在。下篇主要指责秦王朝第三个也是最后一个君主——子婴的过失。《过秦论》鲜明地表现了贾谊的政治立场。

（其二）

　　本文作者贾谊，西汉政论家、文学家，洛阳人。18岁被贬为长沙王大傅。后为梁怀王太傅。贾谊生活在西汉初期，由于此前经过500多年的战争破坏，社会经济凋敝，人口减少，所以他极力主张行仁政以"安民"，多次上疏，批评时政。《过秦论》就是为宣传这种主张而写的。论"古"是为了说"今"，这是我们读本文时首先应当注意的。

（其三）

　　本文着重分析秦王朝削平六国、统一天下及其迅速灭亡的原因，为西汉统治者提供历史借鉴。因为作者贾谊生活的年代，表面上虽然天下太平，实际上却矛盾重重，危机四伏。如诸侯王的封建割据和中央集权之间的矛盾、匈奴屡次入侵与汉族人民希望安居乐业的矛盾、劳动人民与统治阶级之间的矛盾等等。贾谊看出了这些矛盾的存在，并且从秦朝覆亡的历史事实中，意识到人民力量的强大。他企图通过对秦朝兴盛和覆亡的分析，"借古讽今"，使汉朝统治者接受历史教训，从而改革政治、调整各种矛盾，避免社会危机。所以本文明为"过秦"，实则"过汉"。这是本文的实际意义。

（其四）

　　广东刘清涌老师的导入：

　　本文是西汉大政论家、文学家贾谊的代表作之一。他写这篇文章时年纪只有二十二岁左右。当时已被汉文帝召为博士（学术顾问），随又被破格提拔为太中大夫（掌议论，为皇帝谘政，国之上卿），死时也才三十三岁。可见其少年才高，脱颖而出。可惜在那个社会里他终因气盛招谗，不得志而病亡。"过秦论"就是论述秦王朝的过错。全文包括上中下三篇：上篇总论秦的得天下及其覆灭的原因；中篇论秦始皇的暴虐无道，二世继之，因而速亡；下篇论子婴扶危无才，秦亡无救。课文选的是上篇，文章高屋建瓴，纵横恣肆，尽得史论散文之妙，成为千古传颂的名文。

（其五）

　　有位教师这样导入：

　　标题中的"过"是个多义词，作动词用时，有过从、过往、经过等组词，都有从一个地点或时间移到另一个地点或时间的意思；作助词用时，用在动词后，表示某种行为或变化曾经发生，如学过、玩过、吃过等。这里作名词用，与"功"相对，是过失、过错的意思。 "过秦论"，应颠倒过来理解，即"论秦过"，议论秦朝灭亡的过失。秦朝是我国古代历史上第一个封建中央集权制的朝代，它统一六国后，威加海内，不可一世。秦朝的封建帝制的开创者赢政，自封为始皇帝，企图帝业永传。可是秦国却二世而亡。那么，秦国灭亡的过失究竟是什么呢？贾谊分析的原因你同意吗？这篇文章结构严谨，久传不衰，称"天下第一气盛的雄文"，究竟有哪些地方值得我们学习呢？
