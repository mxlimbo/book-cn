灌园叟晚逢仙女

　　黑龙江杨伯威老师的导入：这是明代文学家、戏曲家冯梦龙写的"三言"中的一篇描写劳动人民反抗地主官僚压迫的话本小说。（板书：话本）冯梦龙与其兄冯梦桂、弟冯梦熊并称"吴下三冯"。他毕生从事通俗文学的搜集、整理、编辑工作。著述很多，"三言"是其代表作。"三言"是在"话本"，即讲故事时用的底本的基础上写成的。它以优美的故事、曲折的情节，吸引广大读者，至今影响所及，波及日本。五十年代曾把本篇原作改编、拍摄成为电影《秋翁遇仙记》，深受广大观众欢迎。

　　本文标题，点出作品中的两个人物，一个是劳动人民的代表"灌园叟"，另一个是劳动人民战胜强暴的理想的寄托者"仙女"。这两个人物"晚逢"，在整个故事情节发展的过程中，起着承前启后、推波助澜的作用；同时，为故事情节涂上了浪漫主义色彩。
