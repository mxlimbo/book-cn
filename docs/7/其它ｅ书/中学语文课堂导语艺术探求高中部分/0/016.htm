错字小议

（其一）

　　十年动乱之后，我国出版发行工作空前发展，写作和编辑队伍不断壮大，出现了一批新生力量，但由于其中一些人在工作作风、文化素质和实际经验方面存在的欠缺，许多书籍报刊经常出现各种错字，影响了书报的质量，也给读者造成许多困难。著名语言学家吕叔湘先生写作了这篇《错字小议》，目的就是为了号召所有的作者、编者、排版校对人员，乃至广大的读者都来重视错字问题，共同努力消灭书报中的错字。文章发表在1982年第2期《读书》杂志上。

（其二）

　　在许多议论文中，往往可以找出具有提纲挈领作用的若干句子，阅读时识别这样的句子并辨明它们所概括的内容，对弄清文章思路有重要意义。

　　我们今天要学习的这篇《错字小议》一文中，作者先用"天天看书看报，天天看见错字"这句话提出问题，然后针对这个问题展开论述。在论述中，有三句话概括力很强，这三句话是："错字的来源是多方面的"（第2段）"错字造成阅读上的困难，程度不同"（第3段）；"让所有写稿子的、编稿子的、排字的、校对的，乃至于广大读者，联合起来，向错字宣战"（正文最末一段）。请大家在自读时抓住这三句话，领会课文内容的要点和全文的思路。
