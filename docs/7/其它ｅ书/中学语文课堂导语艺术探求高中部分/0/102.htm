石钟山记

（其一）

　　《游褒禅山记》这篇游记不同于一般游记，它的特点是什么？（即事明理，叙议结合。）今天我们要学习的苏轼的散文《石钟山记》（板书课题），这篇文章同《游褒禅山记》在写法上有类似之处，不仅以说理透辟见长，而且以景物描写的形象生动见胜，行文缓急有致，抑扬顿挫，波澜起伏，诙谐风趣。清人方苞、刘大木魁 都认为这是"坡公第一首记文"。

（其二）

　　石钟山，在江西省湖口县城之侧的长江岸边，分为两座，县城以南的叫上钟山，县城以北的叫下钟山。山为什么以 "钟"命名，历来有不同的说法。宋神宗元丰七年（1084年），苏轼由黄州团练副使调任汝洲团练副使时，顺便送他的长子苏迈到德兴去上任。途经湖口，乘兴游览了石钟山，对山的得名原委作了一番实地考察。《石钟山记》就是这次游览考察活动的实景，也是一篇富于哲理性的游记。
