致橡树

（其一）

　　《致橡树》是当代女诗人舒婷 的一首抒情诗。舒婷是被称为"朦胧诗"派的代表作家之一，但这首诗一点也不朦胧，她在诗中明明白白地宣布了现代女性追求的积极健康的爱情观。当然，诗歌要以形象来表达思想感情，诗人首先用橡树这一高大挺拔的形象来象征男性的阳刚，然后用同样高大而又开出艳丽花朵的木棉来象征女性的阴柔，表明爱情必须是双方平等，独立而又相互依存，永不分离的绪合。下面让我们来欣赏这首诗。

（其二）

　　河南马双有老师的导入：

　　最近，我国具有最高权威的诗歌杂志《诗刊》，在全国发起中国最受读者喜欢的现当代50名诗人的评选活动。评选的结果令人惊异，你们知道按得票数排序排在第一名的诗人是谁吗？就是我们今天要学的诗歌《致橡树》的作者舒婷。我国著名大诗人艾青、臧克家、郭沫若等竟排名其后！这位年轻而又普通、对不少中学生来说还比较陌生的当代女诗人，以什么样的诗风征服了读者的心，赢得了千千万万读者尤其是广大青少年读者的热烈喜爱？原来，舒婷的诗以火一样的激情关注当代生活，从感情和理性的角度反映社会问题。她的大量诗歌表现了新时代青年人从迷惘到觉醒的痛苦、探求、欢乐的感情轨迹，表达了青年一代的心声，叩击着时代的脉搏。她是中国当代朦胧诗的开拓者，但她的诗我们读来一点也不感到朦胧；她的语言含蓄而又晓畅，坦率而又深沉，华丽而不雕琢，深邃而不做作，如山间的小溪流水潺潺，似天上的云霞绚烂多彩，给人以极大的艺术享受和思想启迪。今天我们要学习的《致橡树》，就是舒婷的这样一首代表作！

（其三）

　　江苏盛冬蕾老师这样导入：

　　教师首先用幻灯展示出另一位女诗人席慕蓉的《一棵开花的树》，让学生思考，找出表现抒情女主人公心理的词语，然后明确地描绘说：通过"求"、"盼望"、"颤抖"、"等待"等关键词语，这个女子的形象似乎就在我们面前：纤弱、柔美的她，羞怯地守候在路旁，祈祷着，激动着，渴望着，她所钟情的人会注意到她。接着话锋一转：席慕蓉表达了女性对爱情祈求的心情――被动的等待。而对爱情，并非所有的女性都如此。在今天我们要学习的这首《致橡树）中，女诗人舒婷就向我们倾述了一个全新的爱情观。
