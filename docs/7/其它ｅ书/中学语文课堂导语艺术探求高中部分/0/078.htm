母亲

（其一）

　　我们刚刚学过鲁迅的小说《祝福》，祥林嫂的悲剧一定打动了大家的心，祥林嫂的形象一定还深深地印在同学们的脑子里。今天，我们来学习另一个形象鲜明的女性――"母亲"（板书课题）。

（其二）

　　《母亲》是高尔基的代表作，也是世界文学史上第一部描写无产阶级革命斗争的不朽名著，母亲是世界文学中第一个女共产党员形象。本文生动地展现了一个革命者的母亲在革命斗争实践中，通过激烈的思想斗争逐步觉醒、逐步坚定的过程。母亲拉盖雅・尼洛夫娜在本文出场的时候，革命人民同沙俄专制的斗争已达到白热化的程度。她的儿子巴威尔・符拉索夫已经被捕，而且在法庭上发表了猛烈抨击沙皇政府的演说，被反动政府判刑流放。母亲不顾失掉儿子的痛苦，毅然接受了党组织派她带着传单（巴威尔在法庭上的演说稿）到外地去散发，把真理告诉群众，以促进革命形势的发展。本文就是从这里开始的。

（其三）

　　（先板书课题）本文作者高尔基是苏联著名作家，也是社会主义现实主义文学的奠基人。初中时我们学过他的散文诗《海燕》。《母亲》是高尔基的代表作之一，也是世界文学史上第一部正面描写无产阶级斗争的不朽名著，曾得到列宁的高度评价。

（其四）

　　《母亲》是苏联伟大的无产阶级作家高尔基的代表作，写于1906年，是世界文学史上第一部描写无产阶级革命斗争的不朽名作，列宁曾称赞它是"一部非常及时的书"。小说反映了俄国第一次资产阶级民主革命时期，在无产阶级政党领导下，工农革命运动由自发到自觉的发展过程，第一次成功地塑造了无产阶级先进战士的典型形象。小说取材于现实生活中的真人真事，具有极强的艺术感染力。小说以"母亲"命名，意味着主人公尼洛夫娜形象在作品中具有特殊意义。她从一个劳动妇女成长为坚强战士，从普通母亲成长为成熟的革命者，概括了当时俄国劳动妇女的许多特征。

（其五）

　　 山东戴继超老师的导入；

　　今天学习高尔基的《母亲》。在初中，我们曾学过他的散文诗《海燕》。试问：高尔基姓高吗？谁能谈谈他的情况？内容可包括原名、笔名、生卒年月、主要著作以及他在世界文学史上的地位。（学生议论）

　　高尔基，原名阿列克寒・马克西莫维奇・彼什科夫。马克西姆・高尔基是他的笔名，意思是"最大的痛苦"。他是苏联伟大的文学家，社会主义现实主义文学的奠基人。列宁称赞他是"世界无产阶级艺术最杰出的代表"。主要著作有，自传体小说三部曲：《童年》、《在人间》、《我的大学》，散文诗《海燕》和剧本《底层》等。

　　《母亲》是他的代表作，写于1906年，这部书是无产阶文学中第一次广泛描写无产阶级革命斗争的长篇小说，列宁说，这部书很有用，许多工人都是不自觉地，自发地参加革命运动的，现在他们 读一读《母亲》，会得到很大的益处。《母亲》对全世界无产阶级革命运动有着很大的影响。在我国，自从1930年由夏衍译成中文以来，它一直为我国革命者，特别是广大进步青年所热爱。解放前，《母亲》是禁书，人们甘冒坐牢、杀头的危险，传抄它、阅读它、珍藏它。像《母亲》这样全力传播革命真理，具有强大感染力量和教育作用的文艺作品，在世界文学史上是罕见的。
