《物种起源》导言

　　山东胡甲林的导入：

　　达尔文于一八三一年十二月到一九三六年十月的五年环球旅行，是他一生事业的转折点。旅行开始时，他是"创造论"、"物种不变论"的信徒。在旅途中成千上万物种变异的事实教育了他，使他对"创造论"、"物种不变论"产生了怀疑，并冲破了神学教条的桎梏，逐渐成为进化论者。回国后，他整理了旅途中搜集的许多资料，开始研究生物进化的问题。从一八五六年起，达尔文开始著述《物种起源》。一八五九年十二月二十四日，达尔文发表了划时代的科学巨著《物种起源》，成为进化论的创立者。这篇导言主要说明《物种起源》的写作经过及作者对《物种起源》问题的一些基本的看法。
