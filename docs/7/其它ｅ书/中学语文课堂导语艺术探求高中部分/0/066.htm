论"费厄泼赖"应该缓行

（其一）

　　本文写于1925年底。1924年，北京女子师范大学女校长杨荫榆依附北洋军阀，施行封建家长统治，压迫学生，激起学生反对校长的风潮。1925年4月，段祺瑞执政府的教育总长章士钊声言要"整顿学风"，更助长了女校长的气焰。她在5月9日开除了包括刘和珍、许广平在内的六名学生自治会的干部，并且在报纸上造谣攻击学生。8月初，教育总长下令解散女师大，另立国立女子大学，并派刘百昭雇用流氓、女丐殴打、强拉女师大学生出校。鲁迅坚决支持反抗迫害的学生，同一些教师、学生组织了校务维持会，在宗帽胡同赁屋继续上课。11月28日，北京群众举行要求段祺瑞下野的示威运动，学生参与其中，捣毁了教育总长等人的住宅。女校长被撤职，教育总长也被迫下台，女师大得以复校。在这场斗争中，以陈西滢为骨干的《现代评论》派，一直站在反动势力一方，这时又宣扬 什么"公正"、"和平"、"反对偏激"和"不为已甚"的中庸之道，为反动势力效力。林语堂的《插论语丝的文体——稳健、骂人、及费厄泼赖》，提倡"费厄泼赖"精神，主张不再打击段祺瑞、章士钊等人，认为周作人的反对"打落水狗"，足以补充"费厄泼赖"的意义。这和《现代评论》派维护反动势力的言论基本一致。"费厄泼赖"，就体育比赛和为人处事来说，可以肯定，但绝对不适用于对敌斗争。鲁迅基于上述斗争情况，针对林文，提出了"打落水狗"的主张，写出了这篇著名的议论文。

　　本文以林语堂的文章为论战的对手，而在论证论点当中，以陈西滢等走狗文人和军阀官僚的反动行径作论据，批判的锋芒直接指向了这些丑类。因为运用论据具有犀利的批判锋芒，论证论点也就具有了充沛的力量。

（其二）

　　我们高一时学过鲁迅作品《记念刘和珍君》，那是一篇记叙文。今天我们学习鲁迅的一篇议论文《论"费厄泼赖"应该缓行》，文章选自《坟》，发表于1926年1 月10日《莽原》半月刊第1期，两篇文章发表的时间十分接近，内容上也有相联系之处。

　　1925年，我国第一次国内革命战争正在形成高潮。在中国共产党领导下，"五·卅"运动、省港大罢工震动中外。全国正处于北伐进军前夜，北洋军阀政府面临覆灭。北京发生女师大事件（学习《记念刘和珍君》时已作介绍），女师大学生的斗争在广大人民群众的支持下，取得初步胜利，反动校长杨荫榆被撤职，段祺瑞执政府教育总长章士钊被迫下台。当时以胡适、陈源（陈西滢）为骨干的现代评论派，以资产阶级改良主义作为理论基础，宣扬"公平""正当""和平""稳健"的中庸之道，实质上是反对人民群众的革命，为反动势力帮忙。而林语堂的《插论语丝的文体——稳健、骂人及费厄泼赖》，这篇文章与现代评论派的反动言论是基本一致的。在这种情况下，鲁迅写了《论"费厄泼赖"应该缓行》。
