警察和赞美诗

（其一）

　　本文作者欧·亨利（1862一1910）是美国著名的短篇小说家。他原名威廉·锡德内·波特。一生写了近300部短篇和一部长篇（《白莱与皇帝》）。

　　他出生于美国北卡罗来纳州的一个小镇。出身贫寒，因母亲早逝而童年孤苦，少年辍学当学徒、放牛娃，作会计、办事员、出纳员，后因涉嫌银行缺款案被错判刑五年；在狱中为给女儿买圣诞礼物，便以欧·亨利为笔名发表短篇小说,因而出了名。由于一生屡遭不幸，常与下层百姓同甘共苦，所以他的作品常以曼哈顿下层市民为对象，表现他们的悲哀。所描绘的环境深刻地揭露了资本主义社会的黑暗、悲惨，被人们誉为"曼哈顿桂冠诗人"，"美国生活的幽默的百科全书"。 他的小说风格独特，构思巧妙，情节曲折。尤其是突然转折的结尾，收到出奇制胜的表达效果，被称为"欧·亨利式结尾"。《警察和赞美诗》便具有了这一特点。

（其二）

　　山东车仲实老师的导入：

　　我们今天学习《警察和赞美诗》这篇课文，作者是欧·亨利，大家还记得初中六册学过的《竞选州长》吗？那篇的作者是谁？（马克·吐温。）这两篇作品具有共同的幽默讽刺的艺术特色，以漫画式的、夸张的荒诞形式反映社会生活的真实，借喜剧的色彩表现悲剧的内容。如果说文学是反映社会的镜子，这两篇作品则是哈哈镜，它扭曲变形地折射出资本主义社会的污秽和丑恶，大出了美国这个最先进发达的资本主义国家的洋相。《竞选州长》批判的触角伸向美国的所谓民主选举制度，《警察和赞美诗》讽刺的矛头，则指向美国的所谓高度的物质文明和自由平等的社会秩序。令人读了，在笑过之后，深感这种社会生活的荒唐、颠倒、黑暗、可恶、可恨、可悲！

　　欧·亨利（1862一1910 ）美国著名短篇小说家，他一生写下的近三百篇短篇小说和一部长篇小说，被誉为"美国生活的幽默的百科全书"。本文可以作为他的代表作。
