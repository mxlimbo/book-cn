内蒙访古

（其一）

　　1961年7月23日至9月14日，著名维吾尔族历史学家翦伯赞与另两位历史学家范文澜、吕振羽一起，应当时内蒙古自治区负责人乌兰夫的邀请，访问内蒙，其后翦伯赞写了一篇一万二千多字的游记散文《内蒙访古》，发表在1961年12月13日《人民日报》上。全文除引子外，分六个部分：《哪里能找到这样的诗篇》《一段最古的长城》《在大青山下》《游牧民族的摇篮》《历史的后院》《揭穿一个历史秘密》。后被收入周立波同志编的《散文特写选》。课文所选，是原文的二、三部分。编者以作者原文总题为题，交待了本文内容反映的地点和时代，给读者以比较鲜明的提示。

（其二）

　　作者翦伯赞（1898一1968年），湖南桃源人，维吾尔族，我国著名的历史学家，著述颇丰。解放后曾任北京大学历史系主任，副校长等职。文化大革命期间，被"四人帮"迫害致死，"四人帮"覆灭后平反昭雪。

　　本文是1961年翦伯赞应邀访问内蒙古后写的。全文除引言外，共六部分，课文节选的是第二、三部分。课文以寻访古迹所见所闻为线索，把自然环境、古迹和历史人物的描写，以及作者的评论融合在一起，记叙了我国内蒙古历史上的民族关系，阐述了自己的见解，这是一篇独具风格，富有特色的访古散文。

（其三）

　　有位内蒙古老师这样导入：

　　先板书课题后说：提起内蒙古，我们都有一种亲切和自豪的感情，因为我们生活在这块热土上，都是内蒙古人！接着用投影仪映出内蒙古地形简图，说道：内蒙古地域辽阔，物产丰饶。东部有大兴安岭莽莽的林海，西部有"黄河百宫，唯富一套"的河套灌区，还有乌海的煤炭，包头白云鄂博的铁矿和稀土，以及水草肥美的草原和工厂林立的新城。所有这一切都展示出一幅幅欣欣向荣的壮丽画卷，吸引着一批批来访者。翦伯赞概括自己的访问观感时说："见所未见，闻所未闻。"他还在引用恩格斯的"世界史是伟大的诗人"这句话后指出："我们在内蒙地区看到了这个最伟大诗人的杰作……在世界文学宝库中，哪里能找到这样波澜壮阔、气势豪放的诗篇呢？"那么让我们在历史学家的指引下，从现实走向历史，去寻访这首历史诗篇吧。有地图投影可感形象，有语言叙述激发联想，创设情境，一下子就调动起学生探求新知识的积极性和主动性。

（其四）

　　陕西田宗昌老师这样导入：

　　首先出示打乱了顺序的八个诗句。

　　①邯郸歌舞终消歇②飞来天外有鹰扬

　　③英雄不愧武灵王④骑射胡服捍北疆

　　⑤望断云中无鹄起⑥两千几百年前事

　　⑦只剩蓬蒿伴土墙⑧河曲风光旧莽苍

　　然后告知学生，这是一首七言律诗，让学生根据七律的有关知识排出正确顺序。师生讨论排序后，教师讲述全诗大意，并告诉学生这首诗题为《登大青山访旧长城遗址》，作者是我国现代著名史学家翦伯赞（板书作者）。紧接着设问吊胃口：那么，你们想不想知道这首诗是作者于何时何地写的呢？（引出课文，后板书题目）虽是精心设计，有意为之，但却浑然天成，无斧凿之痕。
