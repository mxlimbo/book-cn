幼学记事

　　莫利兹说："穷人在想哭的时候也是常常笑的。"于是之的《幼学记事》这篇文章所写的，就有这样的特点。

　　本文是一篇回忆性散文，通过记叙幼年因家境贫困而在求学过程中所遇到的种种艰辛、困苦，表现了旧社会恶劣的政治制度的不合理、不公平，表达了对幼年求学过程中帮助过自己、曾给自己有益教育的良师益友的深切怀念和感激之情。文中所叙的事情，都是作者亲身经历，写来如诉家常，读来倍感亲切。作者以幽默的语言、豁达的气度，来描写那些幼年因求学所经历的十分辛酸的事情，使人读着感到有趣、好笑，而在这同时，又会感到那么辛酸、凄凉。
