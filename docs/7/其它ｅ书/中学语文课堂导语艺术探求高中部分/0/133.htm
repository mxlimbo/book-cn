训俭示康

（其一）

　　在高一年级，我们学过司马光所著的编年体史书《资治通鉴》中的《赤壁之战》，对作者已有所了解。据史料介绍，王安石推行新法时，司马光的思想确实保守，但仍不失为北宋的名臣之一。他律己极严，《宋史》中说他"恶衣菲食，以终其身"。对子女教育，从不放松。其子司马康，字公休，从小聪明、好学，博通古书，曾任校书郎、著作佐郎兼侍讲。平时"为人廉洁，口不言财"，受司马光的影响很深。本文就是司马光教诲他儿子司马康的一篇文章。

（其二）

　　本文作者司马光，字君实，夏县（今属山西）涑水乡人，世称涑水先生。他是北宋政治家、历史学家，主编了我国古代最大的编年体通史《资治通鉴》，我们高一时学习的《赤壁之战》即节选自该书。

　　司马光自幼聪明好学（可结合讲讲"司马光打破缸"的故事），尤其喜欢《左氏春秋》。年轻时，学识就很渊博，20岁考取了进士。到神宗初年，他已做到翰林学士、御史中丞。他作为保守派的代表，顽固对抗正在推行的王安石变法，并自己请求任西京御史台，退居洛阳，从事《资治通鉴》编纂工作。哲宗即位，保守派重新得势，司马光出任宰相，废止新法。但司马光不久就死了，追封温国公，谥文正。本文即选自《温国文正司马公文集》。司马光政治上保守，但个人品德却很端正，自律甚严，生活俭朴，为官清正，对儿女教育严格。本篇就是他对儿子的训诫，要儿子崇尚节俭，不求奢靡，并要求儿子训其子孙，永远保持。据史书记载，司马康从小聪明、严谨，博通古书，为人廉洁，口不言财，受父影响很深，甚至不认识他的人，"见其容止"，"皆知为司马氏子也"。

　　《训俭示康》这个题目中，"训"，是教育，教诲。"俭"，节俭，崇尚节俭。"示"，给……看。本文是司马光写给儿子司马康看的一篇训诫文章。是说有关节俭方面的教育。但就其内容说，所讲道理适用于所有人。就是我们今天来读，也是很有教育意义的。这对于我们保持艰苦奋斗的作风，抵制剥削阶级思想的侵蚀有很好的借鉴作用。
