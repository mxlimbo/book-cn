结婚现场会

　　山东孙宝诚老师的导入：

　　（板书课题）作者马烽，一九二二年生于山西省孝义县居义村。一九三八年春，他高小尚未毕业就参加了革命队伍，同年加入中国共产党。根据毛主席《在延安文艺座谈会上的讲话》精神，他深入到农村、工厂做实际工作，一边深入生活、一边给报纸当通讯员，写些通讯、特写、小说。当时，他与西戎合写了《吕梁英雄传》，出版后，产生了广泛的影响。解放后，他陆续写了《结婚》、《韩梅梅》、《我的第一个上级》、《太阳刚刚出山》等著名短篇和优秀电影剧本《我们村里的年轻人》，打倒"四人帮"以后，他精神振奋，又重新提笔创作了十几篇短篇小说，《结婚现场会》就是其中优秀的一篇。

　　茅盾1960年 中国文学艺术工作者第三次代表大会上的报告中，曾论及我国现作家的艺术风格，对马烽的艺术风格论述时说："淡炼鲜明，平易流畅，有行云流水之势，无描头画角之态。"这一评价，精当准确。它恰切地反映了一九六0年前后马烽的创作实践。


　　《结婚现场会》怎样体现了这一创作风格？下面我们准备用两节课的时间做一点探讨。第一步，同学们先默读全文，读完后，列出文章的情节发展提纲。
