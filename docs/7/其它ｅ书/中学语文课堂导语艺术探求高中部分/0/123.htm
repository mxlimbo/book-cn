五蠹

（其一）

　　本文节选自《韩非子·五蠹》。韩非子，战国末期韩国的公子，思想家荀况的学生，是先秦法家思想的集大成者，在先秦历代法家人物理论和实践的基础上，他提出了以法治为中心，法、术、势并用的主张。强调君主要严刑峻法，厚赏重罚，利用权势，讲究权术，才能治理好国家。他反对"是古非今"，主张"法后王"，"论世之事，因为之备"，即根据当今事势，采取恰当的治国措施。他的这些主张在秦统一的过程中，被李斯等付诸实施。在以后的漫长封建社会中，历代封建统治者往往兼用儒、法，加强统治，韩非的思想影响还是很大的。 《五蠹》是韩非表明其政治主张的重要论文，他把当时不事耕战，有害国家的五种人，即学者（指战国末期的儒家）、言谈者（指纵横家）、带剑者（指游侠）、患御者（指依附贵族私门的人）、工商之民称为"邦之蠹也"，即社会的蛀虫。主张清除他们，保护并鼓励耕战之士（指农民和军队），达到富国强兵的目的。

（其二）

　　秦始皇读了韩非的著作后说："寡人得见此人与之游，死不恨矣！"这是对韩非的高度评价。我们现在读《五蠹》，全文千字左右，不及韩非著作总字数的百分之一，对秦始皇的这一评价可能很不理解，但猜一猜也是好的。请大家带着这个问题细读全文。
