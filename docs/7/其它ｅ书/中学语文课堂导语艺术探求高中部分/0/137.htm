夜

（其一）

　　 本单元的起始课我们曾学习了鲁迅先生的《药》，这一节课我们将学习叶圣陶先生的《夜》。叶先生是我国现代著名的教育家、文学家、语言学家。作为文学家的叶先生在小说、诗歌、散文等方面都有突出成就，并且是我国童话园地的开拓者。作为教育家，他也是我国现代语文教学的始创者之一。在中学课本里选作教材的文章，鲁迅先生是最多的，叶先生的仅次于鲁迅。例如，在初中阶段我们学习过他的《古代英雄的石像》、《苏州园林》、《华队公会的供状》、《两种习惯养成不得》、《多收了三五斗》，高中我们已学过了他的《景泰蓝的制作》（以上篇目可抽学生口答）。叶先生关于语文教学的论著颇多，在语文教学方法方面他有一句名言，就是"教是为了达到不需要教"。下面我们就遵循他这句名言来进行教学，把新课文《夜》与已学过的《药》进行比较阅读。好，下面我们就来比较一下《夜》与《药》的同与异。

（其二）

　　1925年至1927年，是我国现代史上反对军阀的第一次大革命时期。1926年7月正式开始了北伐，在短期内取得很大进展。当北伐军乘势长驱攻克上海之后，隐藏在革命队伍内部的国民党蒋介石露出了反革命真面目，以清党为名，在上海发动了惨绝人寰的"四·一二"大屠杀，使大革命遭到了失败。大革命失败后，国民党又变本加厉，在国统区实行了白色恐怖，对共产党和革命群众进行了野蛮的血腥屠杀，他们对共产党员和革命群众的态度是："宁可错杀千人，也不放走一个。"在白色恐怖下，大批共产党员和革命青年被屠杀。本文中的一对青年夫妇，就是在大革命失败后被国民党反动派杀害的千千万万革命者中的两位。

（其三）

　　《夜》这篇文章难度较大，但《夜》的线索正好与刚学完的《药》有一致之处。《药》和《夜》都写了革命者和他的母亲。革命者都没有出场，最后都牺牲了，而把母亲留在世上。《药》是双线结构，那么《夜》呢？《夜》与《药》里的革命者及革命者的母亲，他们有哪些相同，有哪些不同？请大家带着这些问题来研读课文。
