我们打了一个大胜仗

　　 安徽金平老师的导入：

　　 据《中国文学家辞典》介绍，马识途，生于一九一五年，四川忠县石宝乡人。曾发表过〈〈清江壮士〉、〈找红军〉等长、短篇小说。一九八一年六月下旬到九月初的六十多天里，四川省遭到了百年不遇的特大洪灾。在此期间，有二十多个县的降雨量达二百毫米。个别县达四百七十毫米以上，长江上游主要支流岷江、嘉陵江、涪江等都出现了特大洪水，在此情况下，四川省委决定立即动员群众抗洪救灾。《我们打了一个大胜仗》选取了这次抗洪救灾的动人事迹，以饱满的激情，生花的彩笔，描绘出四川人民在党的领导下战天斗地的壮丽画卷。表现了中国人民能够战胜一切困难的英雄气概。
