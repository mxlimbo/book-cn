《<孟子>二章》

（其一）

　　孟子名轲，字子舆，战国时代人，儒家大师。《孟子》一书属语录体，内容是记录孟子的思想和政治言论等，是"四书"之一。 其实我们对孟子并不陌生，初中也学过《<孟子>二章》，谁还能背诵其中的成语、名言？

　　（成语：得道多助，失道寡助／生于忧患，死于安乐 。 名言：天时不如地利，地利不如人和／天将降大任于是人也，必先苦其心志，劳其筋骨，饿其体肤，空乏其身，行拂乱其所为，所以动心忍性，曾益其所不能……）

　　初中的《<孟子>二章》除给我们深刻的教益外，想必在大家淡漠的记忆中还留有孟子文章气势磅薄和滔滔雄辩的影子。今天重读孟子文章正可以借此再度领略一番。

（其二）

　　这两篇文章均选自《孟子》。《孟子》分为七篇，每篇又分上下。各篇又分若干章，以首章中数字标于篇首为题，并不能概括本篇内容，这跟《论语》类似。《孟子》中也有不少章很简短，但大部分篇幅较大，成为一篇完整的文章，虽然人们也称之为"语录体"，但与《论语》已大不相同。我们初中学的《得道多助，失道寡助》《生于忧患、死于安乐》还比较短，本课两章就比较长，《齐桓晋文之事》洋洋洒洒超过2000字。孟子为了捍卫孔子的学说，宣传自己的主张，经常与人展开辩论，用他自己的话说："予岂好辩哉？予不得已也。"因此，他的文章语言流畅，笔锋犀利，论辩深刻，气势磅薄，是先秦诸子中文学性较高的著作。

　　《鱼我所欲也》章选自《孟子・告子上》，论述的是怎样对待生与死，义与利的问题，提出"舍生取义"的主张。

　　 孟子有个著名主张，叫"民为贵，社稷次之，君为轻"。《庄暴见孟子》中孟子劝导齐王"与民同乐"，以实行仁政，也便是从这一"民本思想"出发的。
