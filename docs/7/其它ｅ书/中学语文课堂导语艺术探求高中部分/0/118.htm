文学和出汗

（其一）

　　 我们在中学阶段，已经学习了鲁迅的《论雷峰塔的倒掉》、《"友邦惊诧"论》、《拿来主义》、《论"费厄泼赖"应该缓行》等杂文，请大家回忆并综合一下鲁迅的杂文有什么特点？（让学生讨论，教师归纳）鲁迅的杂文是文艺性的政论文，其特点：①针贬时弊，有感而发（针对性强）；②语言精炼、幽默，讽刺辛辣（战斗性强）；③说理形象（文艺性强）。风格、特点体现在具体的文章中，现在我们学习鲁迅的另外一篇杂文代 表作——《文学和出汗》（板书课题），看看是不是具有这些特点。

（其二）

　　一位老师这样导入：

　　 "文学"是一种社会现象，"出汗"是一种生理现象，本来两者毫不相干，作者为何把它们连成并列短语呢？因为"文学"与"出汗"都与人有关，所以就有相同之处可以联系。"文学和出汗"就是"文学和阶级性"。这里作为批驳梁实秋的"人性论"文学观点，比喻得既通俗生动，又幽默含蓄。
