一碗阳春面

（其一）

　　 本文作者栗良平，本名伊藤贡，昭和十八年（即1943年）出生于日本北海道砂川市。高中毕业后，在医院当了十年职员。以后，他到日本各地搜集民间流传的故事，从事文学创作，并登台"口演"（类似中国的说书）自己创作的故事。《一碗阳春面》自1988年12月问世后，很受欢迎。日本的许多文学杂志纷纷转载。这篇小说的单行本出版后，也被争购一空。日本电视台还多次播放了作者"口演"这篇小说的情景。

　　小说的标题具有写实和写虚双重含义。写实是指这"一碗阳春面"构成了全文的线索，是小说常用的"物线法"中的"物"，如《药》中的药，《项链》中的项链一样，将人物、情节、环境贯穿起来，形成一个有机的整体。写虚是指这"一碗阳春面"在全文中还具有象征意义：阳春面，与"春"密切相关，在大年夜吃面，更寄托了吃面人对新春、对未来的憧憬。这母子三人承受着重大的精神打击，背负着沉重的债务，本可以在家吃些更省钱的晚餐，本不应向老板夫妇显露出自己窘迫的家境，但他们一年又一年地来到了北海亭面馆，虽然"怯生生"，但是有勇气三人合吃一碗阳春面。这样的典型情节使这"一碗阳春面"具有了团结、不屈、向上、奋争等多种美好的含义。

（其二）

　　江苏曾宪一老师这样导入：

　　先投影一首诗，然后朗诵：

　　给我志气吧一一我要出人头地――弟弟说；
　　给我毅力吧一一我要艰难玉成――哥哥说；
　　给我力量吧――我要向厄运挑战――母亲说；
　　团结，顽强，不屈，向上――是我们的性格――母子三人说。

　　同学们，这母子三人是谁？他们是日本著名作家栗良平的短篇小说《一碗阳春面》中的主人公。

　　《一碗阳春面》，是语文教材中叫读者潸然泪下的佳作。不信？请读《一碗阳春面》。

（其三）

　　山东王卫国老师这样导入：

　　课前用录音机播放歌曲《爱的奉献》的第一段，并给学生讲述歌曲第一次演唱时的背景和效果，然后上课。

　　一曲《爱的奉献》赢得了春节晚会上那么多的泪水，把整个晚会推向高潮。是啊，是人间的真情、真爱打动了我们，所以白居易有句话："感人心者，莫先乎情"。1988年12月， 一部短篇小说在日本问世，一时间，许多文学杂志纷纷转载，小说单行本被抢购一空。日本电视台多次邀请作者"口演"，每一次，作者都泣不成声；每一次都有许许多多的人被感动得痛哭流涕。那么，这篇小说到底写了些什么，作者用什么样的生花妙笔，使那么多的人着迷，使那么多的人，乃至于作者本人那样不能自持呢？咱们今天就来学习这篇由日本作家栗良平创作的小说――《一碗阳春面》（板书）。
