《呐喊》自序

（其一）

　　鲁迅先生有两本小说集：《呐喊》和《彷徨》。《呐喊》是鲁迅先生1918一1922年间的短篇小说集，共收集短篇小说《狂人日记》、《孔乙己》、《药》、《故乡》、《阿Q正传》等14篇。《呐喊》体现了作者彻底的毫不妥协的反帝反封建的革命精神，在中国现代文学史上具有划时代的意义。本文是这部小说集的序言，写于1922年12月。作者在序言中回顾自己从19世纪末至"五四"时期的生活经历、思想发展及探索救国之路的过程，说明了写作《呐喊》的缘由和自己的创作态度，表现了作者不断追求真理，探索救国救民道路的革命精神。因此， 《<呐喊>自序》是全面了解《呐喊》的一把钥匙，也是鲁迅前期思想发展的小传。

（其二）

　　这篇《自序》写于1922年12月3日，冠于小说集《呐喊》卷首。在内容上，《自序》回顾了小说集中那些为革命而"呐喊"的作品的由来，最早回答了作者"怎么做起小说来"的问题，初步总结了作者自己早期从事文学战斗的经验。在艺术上，《自序》别开生面，独具一格，不同于一般书序的写法。娓娓而谈，侃侃而议，在夹叙夹议中述说了自己的经历，表达了自己的社会观点和文学观点。可以说，本文既是书序，又是一篇杰出的散文。

（其三）

　　山东车仲实老师的导入：

　　我们今天学习《< 呐喊> 自序》。《呐喊》是鲁迅的第一部小说集，收集一九一八年间所写的小说十四篇，选入初、高中课文的有六篇：《一件小事》、《孔乙已》、《故乡》、《社戏》、《药》、《阿Q正传》。鲁迅写这篇自序，是为了说明《呐喊》是怎样创作出来的。他在本文中自叙了创作这十几篇小说前的生活经历和思想变化，这对于了解鲁迅早期的生活和思想，这对于深入理解《呐喊》的时代精神和思想意义，是有很大作用的。
