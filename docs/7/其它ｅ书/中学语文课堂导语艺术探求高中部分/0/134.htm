眼睛与仿生学

（其一）

　　"仿生学"这个名词是1960年9月13日斯蒂尔在美国召开的第一次仿生学讨论会上提出的。这次仿生学讨论会的副题是"生物原型——新技术的关键"。现在，人们已将这次会议召开的日子当成"仿生学"诞生的日子。在工程科学的进一步发展中，人们需要向生物界寻找启发和进行模拟是很自然的，这种努力有广阔的前景。今天我们学习的《眼睛与仿生学》（板书）就是一篇介绍有关"仿生学"的科学知识的说明文。

（其二）

　　今天我们学习说明文单元。本单元的四篇说明文，都是讲解科学知识，或是介绍一门新兴科学，或是一篇科学考察报告，或是阐述某种规律。今后我们无论是学习还是工作，接触这类文章的机会很多，因此要努力培养自己阅读这类文章的能力。今天我们学习的《眼睛与仿生学》（板书）就是介绍一门新兴科学的说明文。

　　文章标题是个联合短语。"眼睛"和"仿生学"之间，用了一个"与"字，把两者连接起来，表示两者之间存在着一定的关系。读者一看标题，自然产生一个疑问：什么是仿生学呢？"眼睛"和"仿生学"有什么关系？进而引起阅读文章的兴趣。

（其三）

　　江苏连根老师这样导入：

　　中国女子中长跑运动员王军霞一次次扬威世界体坛，一次次让五星红旗在运动场上升起，这标志着我国的中长跑运动取得了突破性进展。王军霞成功的奥秘何在？究其原因，固然与她的身体素质与刻苦训练的精神分不开，但更重要的是她采用了科学的训练方法。据报道，马俊仁教练曾潜心研究过羚羊奔跑的姿态，并让队员们在跑步时进行模仿。马教练这种科学的训练方法，正是运用了仿生学的原理。仿生学是一门新兴科学，它在生产、生活中被广泛的运用。今天我们来学习《眼睛与仿生学》，了解一下眼睛的仿生价值。
