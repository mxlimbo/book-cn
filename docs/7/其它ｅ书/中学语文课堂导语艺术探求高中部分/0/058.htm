猎户

（其一）

　　张帆影老师的导入：

　　"好好爱护动物吧，这对大家都好！"这是《新动物世界》每一集的结束语。"水的污染使鱼虾减少了，乱砍滥伐使山林面积缩小了，肆意行猎使禽兽锐减了……大自然已到处发出紧急警号。"这是著名作家秦牧在"大自然警号长鸣"中发出的呼号。《猎户》讲述的是从前的故事。那时候，老虎、豹子的存在构成了对人类的威胁，而今天，谁能想像世界没有了老虎、豹子，那将是大自然对人类的一种什么样的惩罚！"动物与我们共享地球家园"，这是新时代的最强音！

（其二）

　　叶建华老师这样导入：

　　上课铃声响后，教师手提收录机走上讲台。而后向学生们说：现在，老师来给同学们讲一段有关我国戏曲表演的艺术，怎样？"好！"学生们异口同声地回答。心理刺激的新异性，掀起了课堂气氛的第一高潮。 教师打开收录机，教室里乐曲声起。这是现代京剧《智取威山》第五场《打虎上山》的开头片断。乐曲声中，教师向学生们描述着：威虎山麓，雪深林密；一株株挺直的栋梁松，高耸入云；缕缕阳光，穿入林中。……急风骤雨般的京胡，行云流水似的笙管，以及各种和谐地统一在一起的轻重缓急的乐声 。……"穿林海，跨雪原，气冲霄汉！"一一这是未见其人先闻其声的剧中人二黄导板唱腔。头戴绒皮帽，身着皮大衣，手执道具马鞭。……一记鞭响，剧中人猛一转身，亮相造型。……

　　当讲到剧中人的亮相时，教师模仿出侦察英雄杨子荣的姿势。

　　骤然间，教室里爆发出学生们雷鸣般的掌声。课堂气氛高潮再起。

　　火候已到，教师关掉收录机，随即在黑板上写出课文题目：猎户。紧接着，便向学生们提出问题：刚才老师描述的这段京剧精彩片断，与我们今天学习的课文《猎户》相比，在人物的塑造方面，有着很大的相似之处。请同学们认真地比较思考，说说《猎户》一文是怎样描写打豹英雄董昆的。

（其三）

　　吉林魏大久老师的导入：

　　同学们曾经学过《记一辆纺车》和《歌声》。关于作者的情况你们还记得些什么？

　　作者吴伯萧，现代著名的散文家和教育家。原名熙成，生于1906年5月，病逝于1982年7月。当时新华社发布的电讯说，"吴伯萧是中国共产党优秀党员，山东莱芜人。1938年到延安参加革命工作。94年入党。……他长期从事教育工作和教育出版工作，为培养社会主义建设人才和发展社会主义文化事业作出了积极的贡献。早年他即从事业余文学创作，数十年来撰写出大量的充满革命激情的散文，成为我国当代卓有成就和影响的散文作家。"他的作品有《羽书》、《出发集》和标志他创作风格成就的《烟尘集》、〈北极星》等散文集。他的作品富有真实感，语言朴实自然，构思富于变化，情感深沉，想象丰富。本文写于年月，选自作者的散文集《北极星》。文章通过"我"亲自访问打豹英雄董昆，向读者讲述了自己的所见、所忆、所闻、所感，反映了劳动人民在生产斗争中的贡献和敢于消灭一切野兽和强盗的决心。
