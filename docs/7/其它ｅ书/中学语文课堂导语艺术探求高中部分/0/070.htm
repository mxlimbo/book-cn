《论语》二章

（其一）

　　同学们：初中阶段我们学过《<论语>六则》，谁能为大家背诵几条？（或几句）（学生背诵后转入新课）今天我们来学习《<论语>二章》。 大家看标题中的"章"能否换成"则"或"篇"？（学生讨论后明确）课文标题有的叫"则"，有的叫"章"，有的叫"篇"，"则"是"条"的意思，"章"是"段落"，"篇"是首尾完整的诗文或成部著作中的一个组成部分，这里是段落，因此不能改"则"或"篇"。

（其二）

　　《论语》是一部主要记录春秋时思想家、教育家孔丘言行的书，其中也有部分弟子言论的记录。从记录的称呼和口气上看，是孔子的门人弟子（包括再传弟子）们据自己的回忆和耳闻传说记录下来的。书中涉及的内容非常广泛，包括政治、教育、伦理、文学、哲学以及立身处事等各方面，是有关儒家思想的最重要的著作。这是古代读书人必读教材，是儒家经典《十三经》之一。全书分为20篇，每篇包括若干章，大部分章都很短，有的记录孔子和学生的几句对话，有的就只记录孔子的一两句话。初中教材我们学过《<论语>六则》，今天我们学习本课文所选的两章。这两章记录了孔子的政治观点和政治态度，且有一定的情节，比较生动，是《论语》中著名的篇章。
