回忆我的母亲

（其一）

　　谁都有自己的母亲。母亲的爱像高山，像海洋，母亲的爱是最伟大、最无私的。任何伟人也都一样，也都有自己的母亲。今天，我们学习朱德的《回忆我的母亲》，看看是怎样的一位母亲哺育了朱德总司令这样一位时代的伟人。

（其二）

　　母亲是伟大的，她养育了我们，教我们怎样做人。谁没有自己的母亲，谁不热爱自己的母亲。朱德同志的叙事散文《回忆我的母亲》，赞美了母亲的可贵品质，抒发了作者对母亲的崇敬心情，表达了作者尽忠于民族、人民和党的革命精神。今天，我们学习本文，一定会受到很大的教益。

（其三）

　 江苏朱笑冬老师这样导入：

　　同学们还记得几年前一首比较流行的歌曲吗？这首歌是"世上只有妈妈好"，也就是"世上只有母亲好"。母亲这个字眼多么神圣，多么崇高，母爱像高山，像海洋，母爱又是最伟大最无私的。古往今来有多少文人墨客、仁人志士歌颂自己的母亲，"谁言寸草心，报得三春晖"，唱出普天下儿子对母亲挚爱的心声。那么我们的无产阶级战士是怎样爱戴、报答自己母亲的呢？今天这节课，老师和同学们一起来学习朱德总司令写的《回忆我的母亲》一文。
