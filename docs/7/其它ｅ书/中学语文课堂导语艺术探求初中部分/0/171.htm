统筹方法

（其一）

　　同学们，现在我们处在知识经济的时代和竞争的社会里，所有的优化都体现在时间的节约上。走路，有高速公路、高速列车；寄信，有特快专递、电子邮件；吃饭，有方便面、快餐店，等等，无非是为了节约时间，因为"时间就是金钱，效率就是生命"。为了节省时间，提高工效，我国著名数学家华罗庚先生写了一篇事理说明文――《统筹方法》（板书）。什么是统筹方法？如何运用它来节省时间、提高工效？作者又是怎样运用下定义和列图表的方法说明事理的？我们学习课文后就知道了。

（其二）

　　同学们已经预习了《统筹方法》一文。现在我来考考大家，看哪一位同学能根据课文中的有关内容，用不到10个字概括说明"统筹方法"是一种什么样的方法（如回答"统筹方法是一种安排工作进程的数学方法"，虽然对，但超过了10个字，如回答"统筹方法是方法"则太笼统，应答成"统筹方法是数学方法"）。

　　统筹方法作为一种数学方法，它属于抽象的事理。要把这种抽象的事理说得通俗易懂，可不是一件容易的事。下面，我们一起来研究大数学家华罗庚是怎样来说明统筹方法的。

（其三）

　　我们班的同学读书一贯很细心。下面老师把"预习一"中的一句话读一遍，请同学们指出其中的一字之差："阅读这篇说明文，要注意学习作者从日常生活中举事例通俗地说明统筹方法"。（让同学们指出"举事例"的"举"应改成"设"）为什么这里要用"设事例"，而不用"举事例"呢？课文设的是什么事例？这个事例在文中先后用了几次？请大家带着这个问题细读课文。

　（其四）

　　我们已学过了介绍某一具体事物的说明文，如《松鼠）、《蜘蛛》、《兔》、《人民英雄纪念碑》、《苏州园林》等，均属这一类。还有一类介绍有关科学原理和科学方法的通俗性说明文，把抽象的（不是看得见，摸得着的）、深奥的（事理的高度概括）科学知识写得通俗易懂。今天我们要学习的《统筹方法》，就是这类说明文中颇具特色的一篇。
