藤野先生

（其一）

　　只要是上过学的人，每一个人都有自己的老师。魏巍的老师蔡芸芝先生是"一个温柔和美丽的人"，魏巍对她十分爱戴、思念；鲁迅先生在三味书屋时的老师寿镜吾先生是一个"极方正，质朴，博学的人"，鲁迅对他"很恭敬"。鲁迅在23岁时曾到日本仙台医学专门学校学医，他在那里遇到了什么样的老师呢？对那里的老师又是一种什么样的感情呢？今天，我们一起来学习鲁迅先生的回忆性散文——《藤野先生》（板书）。

（其二）

　　山东田其中老师的导入：同学们，我们曾到鲁迅先生的《故乡》，结识了闰土和杨二嫂，也看过鲁迅先生的《社戏》，领略到平桥村纯朴的风情，还到过鲁迅先生小时候的"百草园和三味书屋"，欣赏了百草园的妩媚，并聆听了寿镜吾老先生的读书声，今天我再带领大家去结识鲁迅先生的一位日本老师——藤野先生（板书）。

（其三）

　　四川黄智生老师的导入：

　　在初二册我们学过魏巍先生的《我的老师》，作者是通过什么方法来赞扬蔡老师的高贵品质的呢？（生答：通过回忆的七件事来赞扬的）对，这七件事，就是七个生活片断，用生活片断来表现人物的高贵品质。在表达方式上，以记叙为主，还用了什么表达方式呢？（生答：还用了议论和抒情）对。我们今天再学一篇通过生活片断，用记叙与议论、抒情相结合的表达方式来歌颂老师的文章——鲁迅先生的《藤野先生》。请大家仔细阅读课文，看作者用哪四个生活片断来表现藤野先生的高贵品质，勾画出议论、抒情的语句，并把它与《我的老师》加以比较，看除了相同之处，还有哪些不同的地方。
