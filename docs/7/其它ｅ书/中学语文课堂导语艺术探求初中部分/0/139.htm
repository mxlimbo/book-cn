人民的勤务员

　　雷锋同志是人民的好战士，是党的好儿子，是我们学习的榜样。他那"憎爱分明的阶级立场，言行一致的革命精神，公而忘私的共产主义风格，奋不顾身的无产阶级斗志"永远激励着亿万人民前进。革命真情绘佳篇，出于爱英雄、学英雄的无产阶级真情，多年来涌现出许多赞美雷锋同志的优秀诗文。《雷锋的故事》就是其中的一部佳作，而《人民的勤务员》一章，更是这本书中的奇花异朵。它集中地再现了雷锋同志全心全意为人民服务的崇高精神和风格。现在我们就来学习这一篇课文。
