《龙须沟》选场

（其一）

　　"旧社会把人逼成'鬼'，新社会把'鬼'变成人"这是歌剧《白毛女》表达的主题思想。话剧《龙须沟》写人民政府把一条臭沟变成了新的暗沟，并在上面修成了一条平平正正的马路，给人民群众带来了幸福。两剧的题材毫不相干，却表达了共同的主题：鞭挞了黑暗的旧社会，歌颂了新社会，歌颂了共产党。通过学习课文《<龙须沟>选场》，可以进一步增强我们热爱党、热爱社会主义的思想感情。

（其二）

　　我们刚刚读过《<白毛女>选场》，《白毛女》是歌剧，以歌唱为主。今天我们要学习新课文《<龙须沟>选场》。《龙须沟》是话剧，主要表达方式是对话。通过学习本课，我们会对戏剧这个大家族里的话剧的特点有所了解。

（其三）

　　同学们有没有看过《龙须沟》？（如有同学看过戏或剧本，可先由同学讲述，教师只作补充）这个剧本是我国著名剧作家老舍在解放初期根据真实的事迹创作的。解放前的龙须沟，原是北京一条又脏又臭的污水沟，住在两旁的劳动人民吃尽了苦头，赵老头疾病缠身，丁四的孩子被淹死，程疯子失了业，王大妈苦挨日子，而历代的统治者还巧立名目，派捐征税，却从不修沟。恶霸、地痞更是"硬抢硬夺"，猖獗横行，逼得劳动人民走投无路。北京解放了，恶霸打倒了；百废待兴，而人民政府最先想到的是修建龙须沟，为人民造福。一九五0年春派来测量队，群众一片欢腾，夏初正式动工。赵老头语重心长地说："咱们政府并不富裕……可是，还来给咱们修沟"，"不叫咱们再病、再死，再臭，再赃，再挨饿。"在修沟过程中，意外受到一场暴雨的袭击，正当人民生命和财产处于十分危急的时刻，区长亲自带人来抢救。赵老头十分感慨地说："修沟出了点毛病，政府又这么关心我们，我活六十多岁了，没有见过。"这肺腑之言，说出了社会主义制度的优越性。就在这年夏末，新沟竣工了，给人民的思想和生活都带来了新的变化。全剧共三幕六场，课文节选的是最后一场。这一场写了什么？请仔细阅读课文。
