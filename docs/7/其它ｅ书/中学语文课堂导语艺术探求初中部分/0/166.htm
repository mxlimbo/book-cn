天气陛下

（其一）

　　天气不仅与我们的日常生活关系密切，而且与工农业生产、交通运输等等有密切的关系。今天我们所要学习的课文《天气陛下》，可以使我们进一步了解天气与人的关系，认识只有掌握自然法则才能改造自然的道理。

（其二）

　　自古以来，人与天气关系密不可分。天气既可以造福人类，也可以祸害黎民。比如，大旱之雨如甘霖，滋润着干涸的土地；大涝之时如猛兽，吞噬着美丽的家园。又如风，它可驱散云雾，给人们带来明媚的阳光；又可以滥施暴行，使人民的生命财产安全受到成胁。再如雪，她会温柔地给植物盖上薄薄的被子，让他们安全过冬，又会无止境地落下，阻塞交通，给生产、生活带来诸多不便……这就是天气，一味任性，向人类展示着他的个性。前苏联儿童文学作家、工程师伊林写的科普作品《人与自然》中生动地展现了人与自然的关系，今天我们要学习的《天气陛下》就是其中的节选。
