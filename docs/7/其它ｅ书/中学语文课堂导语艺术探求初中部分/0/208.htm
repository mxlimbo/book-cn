幼时记趣

　　每个人都有自己金色的童年。这是人生最幸福、最美好的时代。儿童天真烂漫，无忧无虑，爱好新奇，喜欢幻想。童年的梦是五彩缤纷的，有说不完的故事。比如鲁迅先生在《从百草园到三味书屋》这篇散文中，回忆幼时的生活，记述了百草园的自然景物和乐趣。文章描写了十五种景物，叙述了四件趣事，描绘出一幅十分美丽的图画；还特别记叙了冬天百草园的游戏活动，如堆雪人、塑雪罗汉、捕鸟等。写得有声有色，表现了他对童年生活的深情怀念。他的《朝花夕拾》散文集里，就收了十多篇这类回忆性的散文，都是记的童年的故事，值得好好阅读欣赏。

　　今天，我们来学习清代文人沈复的《幼时记趣》，这也是一篇追记作者童年生活的极有情趣的散文。它一定能唤起你对儿时的回忆，把你带进色彩斑斓的童年岁月中去。
