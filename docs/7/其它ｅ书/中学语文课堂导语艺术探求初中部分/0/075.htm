回延安

（其一）

　　滚滚延河水，巍巍宝塔山，呵，延安，中国革命的圣地，你养育了多少中华民族的优秀儿女！如今，离别你十年的儿子回来了！同学们，你们可以想象，这该是怎样激动的心情啊！今天，我们学习的《回延安》就是作者抒发自己这种感情的一首诗。

（其二）

　　陈毅同志的诗〈〈梅岭三章〉〉，给了我们深刻的印象，这儿，我们再补充他的另外作品的几个片断，同学们看一看，它们所唱的，又是什么：

　　万里长征不计程，
　　指津自有北辰星。
　　太行山上辞残雪，
　　延安城头望柳青。
　　八载睽离望关陕，
　　五年风雨仗延安。

　　（摘自1942年冬所作《送沈、张诸君赴延安》

　　众星何灿烂，
　　北斗住延安。
　　大海有波涛，
　　飞上清凉山。

　　 （摘自1943年11月所作《赴延安留别华中诸同志》）

　　百年积弱叹华夏，
　　八载干戈仗延安。
　　试问九州谁作主？
　　万众瞩目清凉山。

　　（1945年4月《七大开幕》）

　　从这几个片断中，我们可以看到抗日战争时期陈毅同志对延安的赞颂与向往。延安，革命的圣地，党中央在这里领导全国人民进行了八年抗战，又从这里出发，摧毁了反动的蒋家王朝。延安，给了全国人民光明、希望和力量，是每一个革命者无限向往的地方。贺敬之的《回延安》，表达的同样是革命儿女对延安的一往深情。

（其三）

　　本文作者贺敬之，当代著名诗人和剧作家。歌剧《白毛女》就是他和丁毅执笔集体创作的。

　　1940年，诗人奔赴延安。1946年因工作需要离开。六年中在延安受到了革命的教育和锻炼，以历了生命史上最难忘的时期。1956年，与延安阔别十年之后，诗人从北京重回延安，参加在那里召开的西北五省（区）青年造林大会，受到延安人民的热情接待，看到了十年中延安的巨大发展变化，内心充满激动和喜悦，就写下了这首抒情色彩浓郁的诗歌，歌颂了党中央的正确伟大，表达了对延安人民深厚而真切的无产阶级感情。

　　诗所采用的是流行于陕北一带的民歌："信天游"（又名"顺天游"）的形式，这种民歌曲调纯朴、高亢、悠长，节奏自由。一般由上下两个乐句构成，歌词常为两句一段，短的只有一段，长的可接连数十段，用同一曲调反复演唱。每段常转韵，多用比兴，多用蝉联，多用叠字，多用衬字，既有急徐相替之美，又有刚柔交织之妙，诵读、歌唱时，自有一唱三叹、回环飘荡的兴味，因而有着显著的地方特色和民歌特色。
