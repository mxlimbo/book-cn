宇宙里有些什么

（其一）

　　炎热的夏天的夜晚，我们在室外躺着纳凉，仰望天空，会看到银河，看到数也数不尽的星星，有时还会找到牛郎星、织女星，找到北斗星……同学们，我们目光所投向的空间，就是无穷无尽的宇宙。宇宙里究竟有些什么呢？今天我们所要学习的课文《宇宙里有些什么》（板书），会告诉我们关于宇宙的一些基本的知识。

（其二）

　　有位老师这样导入：

　　上课时，教师先在黑板上画了两少年乘火箭在星空上飞翔的简笔画，这幅画一下子引起了学生们的好奇心。然后老师讲道：同学们，你们是否也想乘火箭去宇宙旅游观光呢？同学们异口同声地回答：想啊！教师说：那好吧，我们一起乘上想象的火箭飞向那广漠的穹宇，探索宇宙的奥秘吧！板画和老师的话，一下子把学生思维带到想象的宇宙与课文里描写的宇宙融为一体的世界里。
