闻一多先生的说和做

（其一）

　　大科学家爱因斯坦有个公式："成功= 艰苦劳动+ 正确方法+ 少说空话。"我们明白从事任何工作，说空话有害无益，重要的是做，是实干。俗话说："一万个零抵不上一个一"。" 一万句空话当不了一次实干"。显然空话是不行的，然而"说"和"空话"是两个概念。一个人不仅要脚踏实地干，而且要"说"，说应该说的，该说的，说得有成效。民主革命时期的闻一多，是杰出诗人、学者，他是"口的巨人"、"行的高标"。他是怎么干的，怎么说的，诗人臧克家给我们作了十分精彩的介绍，我们一起来学习《闻一多先生的说和做》。

（其二）

　　世界上有这样两种人：一种人是说了再做，或者说了也不一定做；另一种人是做了再说，或者做了也不一定说。第一种人说得多做得少，夸夸其谈、华而不实。第二种人做得多说得少，埋头苦干、踏踏实实。两种人有两种完全不同的说和做。（板书："说和做"）
现在，在我们面前有一个高大的人物形象：他研究学问，做得多、说得少，他参加革命斗争，敢说敢做，最后在反对国民党反动派的斗争中，英勇地献出了生命。他是谁呢？他就是永远值得我们学习的闻一多先生（极书："闻一多先生的"）
