消息通讯

　　尹庆春老师这样导入：

　　同学们，告诉大家一个好消息：我校生物老师王丽丽老师家的猫，昨天在北京电台"五彩缤纷"节目里，获巨奖啦！（学生注意力全被吸引，于是老师将事情的时间、地点、人物、情节、结果，用简洁、明快的语言讲述清楚。）同学们， 刚才我讲的这则消息若发表在报纸上，就是"新闻"。"新闻也就是消息"，是报纸最常用的文章样式，它用事实说话，用简洁、明快的语言及时报导最新发生的有社会价值的事实。如果再将消息的内容展开一些，写得详尽、具体、形象些，那么就成为另一种文章样式，那就是通讯。

　　今天，我们要学的课文就是：消息通讯（板书）。
