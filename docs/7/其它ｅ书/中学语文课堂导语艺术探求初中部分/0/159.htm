苏州园林

（其一）

　　苏州园林是我国园林建筑中的一朵奇葩。它以园林数量众多、布局别具匠心、建筑巧夺天工、景致变化多端吸引着众多的中外游客。它的风采或被摄影家收入镜头，或被文学家写进文章。今天我们学习叶圣陶先生写的《苏州园林》，要注意把握苏州园林的总体特征，学习谋篇布局的技巧，领会叶老遣词造句的匠心。

（其二）

　　我们刚学完〈中国石拱桥〉，课文是采用举典范的例子的方法来说明中国石拱桥的特点的。《苏州园林》也是一篇说明文，看看它又是采用什么方法来说明苏州园林的特征的。

（其三）

　　我们今天学习《苏州园林》。园林：是种植花草树木、供人游赏休息的风景区。作者叶圣陶，在一年级我们学过他什么文章？是什么文体？你们能简要回忆它的内容吗？（明确：学习过《古代英雄的石像》是篇童话。）叶圣陶老先生是哪里人？（明确：苏州人）他对苏州园林很熟悉，又有深刻的研究，《苏州园林》一文便是介绍苏州园林特点的，是说明文。大家看课文，把不懂的问题，不识的字，不懂的词提出来。
