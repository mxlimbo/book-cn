海燕

（其一）

　　对于同一个对象，不同的作家出于不同的目的，往往有不同的描写。如郑振铎《海燕》这样写道："海水是皎洁无比的蔚蓝色，海波是平稳得如春晨的西湖一样，偶有微风，只吹起了绝细绝细的如千万个粼粼的小绉纹，……天上也是皎洁无比的蔚蓝色，只有几片薄纱似的轻云，平贴于空中，就如一个女郎，穿了绝美的蓝色夏衣，而颈间却围绕了一段绝细绝轻的白纱巾。""就在这时，我们的小燕子，二只，三只，四只，在海上出现了。它们仍是隽逸的从容的在海面上斜掠着，如在小湖面上一样；海水被它的似剪的尾尖一打，也仍是连漾了好几圈圆晕。小小的燕子，浩莽的大海，飞着飞着，不会觉得倦么？不会遇着暴风疾雨么？我们真替它们担心呢！"

　　那么高尔基笔下是如何描写这"小小的燕子"的呢？今天我们来学习高尔基的散文诗——《海燕》，把它和郑振铎的《海燕》作一下比较，看各是怎样描写大海、天空、海燕的？为什么会有这样的不同？

（其二）

　　高尔基是苏联伟大的文学家。伟大的革命导师列宁称他为"无产阶级文学的奠基人"，"无产阶级艺术的最杰出的代表人物"。高尔基在苏联革命文学史上的地位与作用，正像我国伟大的文学家鲁迅在中国革命文学史上的地位和作用一样极其重要。一九0五年，高尔基写作了散文诗《海燕》。当时，俄国国内阶级矛盾日趋尖锐，沙皇的专制统治更加反动。此时，马克思主义在工人运动中广泛传播，促进了俄国劳动人民的觉醒。工人运动此起波伏，阶级斗争已经激化。《海燕》所描写的正是这样一个"山雨欲来风满楼"的时代面貌，它反映了俄国革命人民，对沙皇反动政府越来越愤怒的斗争情绪和迎接革命高潮到来的渴望，并预示了沙皇专制制度即将崩溃。

　　这首散文诗发表后，在群众中产生了极其强烈的影响，人们争相传诵，成为宣传革命、鼓励群众的有力武器。列宁十分喜欢这篇作品，充分肯定了它的深刻的思想意义和巨大的号召力量。一九0六年，列宁在《暴风雨之前》这篇论著中，就引用了《海燕》里的话，并且以"让暴风雨来得更猛烈些吧"来结束他的文章。
