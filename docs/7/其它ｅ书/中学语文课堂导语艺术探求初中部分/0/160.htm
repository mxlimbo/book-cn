她比幸子更幸运

（其一）

　　助人为乐，情深似海。在我们中间，一个人发生困难，就有上百、上千、上万个素不相识的人热情地向你伸出温暖的手，不遗余力地帮助你……病魔把正处于人生花季的十六岁的女中学生孙倩残酷地推向死亡的边缘，而充满挚爱深情的人们，却奇迹般地把她从死神的手里夺了回来。电视剧《她比幸子更幸运》，又一次以生动的事实反映了我们社会主义大家庭"一方有难，八方支援"的新型的人际关系。

（其二）

　　一位老师这样导入：上课前老师先给学生放韦唯演唱的《爱的奉献》这首歌的录音，然后又请一位学生独唱，使学生全身心地融入音乐的美感之中，用音乐的艺术美呼唤学生灵魂深处的爱。让那些能给人们新的生命和希望的真诚的爱，像永不枯竭的淙淙甘泉在学生的心田里流淌，鼓舞学生脚踏实地生活在充满人间真诚和爱的世界里，陶冶学生为人间多献一分爱的高尚情操，促进学生身心健康地成长。
