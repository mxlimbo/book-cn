我的小桃树

　　徐永平老师的导入：

　　同学们，你们这一年龄阶段的人，已经不再是不知世事艰难的小孩子了。你们已经走过了一段不平凡的人生里程，对社会、对人生已经初步具备了独立判断和思考的能力，内心深处也有了较为丰富的感情。你们正在用自己特有的目光观察着社会，审视着人生，你们正一步步走向成熟。但是，假如老师现在布置一篇作文，要求自拟题目，写一篇写景状物、托物言志的散文，你将选取怎样的景物作为你对社会、对人生所思所感的寄托物呢？我想不少同学首先会想到的是青松、翠柏、高山、大海等高大、有名的景物。其实，大可不必如此，生活中许许多多我们司空见惯的细微的事物中，同样蕴含着丰富的生活哲理，只是我们没有注意到罢了。罗丹曾经说过："所谓大师就是这样的人，它是从人们司空见惯的现象中发现出美来。"古今中外的许多作家都证实了这一真理。本文作者笔下的"丑石"，高尔基笔下的"海燕"，于谦笔下的"石灰"，茅盾笔下的"白杨"，无不是如此。今天，我们学习的《我的小桃树》也是这样一篇典范之作。作者通过一棵小桃树告诉了我们一条人生哲理，倾诉了一腔情思，通过学习，愿每个同学都能成为"大师"。
