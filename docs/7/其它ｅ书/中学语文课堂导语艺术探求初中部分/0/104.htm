荔枝蜜

（其一）

　　同学们从课本上已经学过不少优美的记叙文，其中有的是散文，它们选材十分广泛，组材自由灵活，为我们展现出人、事、景、物等大千世界一幅幅迥然不同的画面；无论是写人记事，还是写景状物，它们都以深刻而新颖的立意，巧妙独特的构思吸引、感染着我们。今天，研读著名散文家杨朔的《荔枝蜜》，我们将步入散文世界中的一方新天地，在诗情画意中领略到一种新的境界，现在我们来学习散文《荔枝蜜》（板书课题）。

（其二）

　　这个单元的五篇课文都是散文。阅读散文得抓住要领。其中一个要领就是理出文章组织结构的线索，弄清材料之间的内在联系。那么，今天学习的《荔枝蜜》一文，究竟是以什么为线索来组织材料的？下面，我们就来研究这个问题。

（其三）

　　著名散文家杨朔的《荔枝蜜》，同学们都读过了，我想，大家仿佛跟着作者一起来到广东从化温泉，饱览荔枝林，品尝荔枝蜜，还参观"养蜂大厦"，目睹耳闻了蜜蜂的辛勤劳动。现在我想问一个问题：你读了这篇散文后感受最深的一点是什么？好，请思考一分钟，想好了再回答。

（其四）

　　有位老师这样导入：同学们吃过荔枝蜜吗？今天咱们就一起来学习这篇介绍荔枝蜜的说明文吧。学生马上就站起来反驳：不是说明文！不是介绍荔枝蜜的！教师接着问：哦，不是啊！那它是什么体裁？又是写什么的呢？为了驳倒老师的"错误观点"，证明自己的观点是正确的，学生们的那种兴奋急切的心情，那种群情激昂的氛围，简直是想压也压不住了。老师抓住时机，稍加疏导，整篇课文的教学就在教师轻松、学生活泼的情境中瓜熟蒂落了。
