老山界

（其一）

　　"红军不怕远征难，万水千山只等闲。"1934年10月初，中央革命根据地（江西）的中国工农红军突破国民党反动派的第五次"围剿"，北上抗日，在长达一年的时间里，战胜了无数艰难险阻，粉碎了反动派的围追堵截，长驱二万余里，纵横十一个省，于1935年10月胜利到达抗日根据地陕北，与陕北红军会合，完成了举世闻名的二万五千里长征。同学们，你们知道红军在长征途中战胜了哪些艰难险阻吗？（让学生回答，突出抢渡金沙江、飞夺卢定桥、翻雪山、过草地等）你们能够想象出这些艰难险阻的具体情况吗？让我们来学习陆定一同志写的《老山界》（板书：老山界陆定一）一文，先来感受一下其困难程度远比战胜金沙江、大渡河、雪山、草地"小得多"的翻越老山界的艰难吧！

（其二）

　　（首先板书：老山界陆定一）同学们，老山界是一座什么样的山？又叫什么？（一座三十里高的瑶山，地图上叫越城岭）"我们"决定翻越老山界，"我们"是什么人？（红军）老山界在红军长征途中有什么重要地位？（是长征中所过的第一座难走的山）你们预习了课文，从课文来看，翻越老山界艰难吗？（艰难，但老山界的困难，比起金沙江、大渡河、雪山、草地等地方来，还是小得多）同学们，从1934年10月到1935年10月，中国工农红军历时整整一年，战胜了无数艰难险阻，粉碎了国民党反动派围追堵截，长驱二万余里，纵横十一个省，从中央革命根据地（江西）来到了陕北，完成了举世闻名的二万五千里长征，保存了革命力量，谱写了中国革命的新篇章，让我们先来感受一下困难程度似乎最小的翻越老山界的艰难，从而认识并体会长征途中的红军战士不怕艰难险阻的坚强意志和革命乐观主义精神吧！

（其三）

　　同学们，先背诵在小学读过的毛泽东同志的七律《长征》诗，好不好？（学生背诵"红军不怕远征难，万水千山只等闲。……"。）这首诗生动地描述了中国工农红军在长征途中战胜艰难险阻的过程；热情地赞颂了红军战士藐视困难，一往无前的革命精神，在红军眼里，险峻的高山不过是"细浪"、"泥丸"。今大学习的《老山界》，是陆定一同志写的长征回忆录。通过这篇课文的学习，我们可以具体了解红军长征部队是如何翻越长征途中第一座难走的山一一老山界的。
