《梦溪笔谈》二则

　　在古代，我国劳动人民和一些科学家很注意观察天象，观察气象的变化，积累了丰富的经验，并且有一些科学著作问世，一直传留到今天。《梦溪笔谈》就是其中的一部。课文中选的两则，一则《陨星》，是记载一次陨星现象的。在宇宙中各种天体运动蔚为壮观，像一支雄浑的交响乐曲，而流星陨落只不过是一支插曲罢了。但是在900 多年前人们就能注意观察这一过程，如实地记述，并得出陨石是一种铁石之类的物质，这就十分难能可贵了。另一则《预防风灾》，其实就是反映古人如何看云识天气，文中记述的这一经验，也是符合科学道理的。
