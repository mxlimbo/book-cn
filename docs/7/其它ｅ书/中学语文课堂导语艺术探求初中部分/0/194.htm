新闻两篇

　　1949年4月20日午夜开始，人民解放军百万大军先后横渡长江，彻底捣毁了国民党的长江防线。4月23日，解放了国民党22年来的统治中心南京。渡江战役开始后，新华社连续发表了毛泽东同志撰写的两则新闻。第一则是《大军三十万昨日渡长江》，第二则是我们将要学习的课文《人民解放军百万大军横渡长江》。现在我读一下第一则新闻，同学们听后思考以下几个问题：

　　（1）文中哪几句概括介绍了时间、地点、人物、事情？

　　（2）文中用哪些词语描述国民党反动派防线崩溃，用哪些词语描述人民解放军的英勇善战？

　　（3）《人民解放军百万大军横渡长江》一文中，哪一段是简述《大军三十万昨日渡过长江》的战况的？
