木兰诗

（其一）

　　同学们知道木兰从军的故事吗？民间传说中的木兰是一个智勇双全、深明大义的姑娘。当国家危急，皇帝征兵御敌，征兵名册上有她父亲名字的时候，她看到父亲年老，弟弟又年幼，都不能应征入伍，便毅然决定女扮男装，代父从军，卫国保家。在十多年战争中，木兰表现英勇，立下了很多的军功。战争胜利后，木兰朝见天子的时候，要求辞官回乡与家人团聚。回家后，她恢复了女子的装束，出门去见她在军队中的伙伴，这使她的伙伴大为吃惊：跟木兰在一起共同生活、战斗了十二年，竟没有认出她原来是个姑娘。宋代郭茂倩编了一部《乐府诗集》，其中就收入了南北朝时北方的一首民歌——《木兰诗》，内容就是写这个充满喜剧性的故事的。现在我们就来学习这首民歌。

（其二）

　　我们伟大的祖国，曾以古老文明称著于世。古代文化灿烂，文学艺术上的财富数不胜数，妙诗佳词多如天上璀璨的明星。美丽的神话、动人的传说、优美的民间故事，给人以丰富的想象、有益的启发。木兰替父从军的故事，就在人民中广为流传，深受大家的喜爱。

　　这个故事原见于《木兰诗》，又名《木兰辞》，是南北朝时北方的一首民歌。写作年代大约为北朝后魏时期，是古代劳动人民集体创作的民间文学作品，后来虽经封建文人改动，但仍保留着民歌特色。著名历史学家范文澜在《中国通史简编》一书中说："北朝有《木兰诗》一篇，足够压倒南北两朝全部士族诗人。"《木兰诗》为历代人民传诵，和《孔雀东南飞》诗合称为我国民间文学史上的"双壁"。

　　《木兰诗》如此佳妙，今天，我们一同来欣赏、享受这首叙事诗的艺术美吧。
