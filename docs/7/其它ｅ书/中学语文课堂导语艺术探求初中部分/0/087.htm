俭以养德

（其一）

　　同学们，我们都看过彩色故事片《周恩来》，周恩来总理的生活十分俭朴，他接见外宾、出席招待会时的礼服只有一套，而且上衣已有破损。是周总理没有条件再添置一套新礼服吗？当然不是，是周总理吝啬吗？更不是。我们以前读过《一件珍贵的衬衫》，那篇课文的作者由于骑车不慎，进入快车道，衬衫让总理坐的汽车剐破了。总理是怎样处理这件事的？请同学们回答。（周总理出钱买了一件崭新的衬衫，叫一名工作人员送给"我"）可见，周总理对人民群众十分大方、慷慨。那么，周总理的个人生活又为什么那样俭朴呢？因为总理深知"俭以养德"（板书：俭以养德），他时时关心着人民的疾苦，处处为群众作出榜样。今天，我们一起来学习马铁丁（板书：马铁丁）写的《俭以养德》这篇课文，以进一步懂得节俭对于品德修养的意义。

（其二）

　　陕西张建军老师这样导入：

　　上课一开始，教师在黑板上写出一条字谜："一人站，一人卧，一条板凳三人坐，日子就要这样过。"

　　不少学生觉得奇怪：今天老师怎么要我们猜起谜语来？有人开始窃窃私语，有人左右观望。老师示意他们好好猜。有的学生从前三句中寻求突破，有人凝神从后一句中思索着什么。突然有一女生忍不住喊出了谜底："勤俭的俭！"既然已经有人猜中，老师因势利导——一边嘴里重复着谜面，一边慢笔写出"俭"字，学生们一下子悟到了谜语的妙处。当他们发出会心的微笑时，老师讲道："俭"是我们民族的传家宝，大可安邦，小可治家，因为"俭"是"聚宝盆"，众所共认。但是"俭"的意义远非如此，"俭"还能陶冶情操、修身养性。今天，我们学习一篇课文，看看作者在这方面有何独到见解。接着转身将"俭以养德"四个大字赫然写在黑板正中。
