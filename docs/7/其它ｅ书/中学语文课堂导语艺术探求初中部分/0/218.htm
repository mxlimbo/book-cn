枣核

（其一）

　　儿时的回忆，总是充满着温馨；故乡的一切，总是充满着感情。同学们，故乡的山川、草木、人物，什么最能打动你们的心灵？在故乡的土地上，有没有留下你们的足迹？（可让学生回答）今后，无论你们走到哪里，这一切，都将成为你们永远难忘的回忆，伴着你拼搏，伴着你进取。今天，我们所要学习的《枣核》一文，写的就是一个远在大洋彼岸的风烛残年的老人，作者的"旧时同窗"，尽管"家庭和事业都如意"，但越到晚年，思乡越切，总是想起故乡院里的那棵枣树。当作者去美国时，"再三托付"为他带几枚生枣核，以便在花园里试种一下。让我们一起来体会文中那浓烈的思乡之情吧！

（其二）

　　山东朱君华老师的导入：

　　教师提前二分钟进教室，问学生：同学们，愿意听老师唱歌吗？（愿意）我为大家演唱一首《我的中国心》好不好？（好）教师便有感情地演唱这首歌，并接着提问：你们知道这首歌表达的是什么人什么样的感情吗？（游子对家乡对祖国的思恋之情）这时，教师及时把话锋一转说：像这样的歌曲还有很多。在文学界，也有不少作家作文章来表现游子思念家乡这样的主题。比如，今天我们就要学习的叙事散文《枣核》，表现的就是美籍华人思念家乡的真挚而深沉的感情。希望同学们认真学习，细心品味。

（其三）

　　四川苟俊林老师的导入：（先放歌曲《我的中国心》）这首歌是香港歌星张明敏演唱的《我的中国心》，歌中唱出了海外华人的共同心声。世界上共有数千万海外华人，他们身在异国他乡，却情系故土，因为他们的根在中国。正如歌中唱道："'流在心里的血，澎湃着中华的声音，不管怎样也改变不了我的中国心"他们是中华民族的儿女，他们是炎黄的子孙，他们用各种各样的方式表达着他们对祖国、对故土的深深眷恋。下面我们来读一读《枣核》，看看文中的这位海外华人，他是用什么方式来寄托他对故土的思恋的？
