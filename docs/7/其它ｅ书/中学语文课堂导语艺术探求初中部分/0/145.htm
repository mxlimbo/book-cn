身居闹市人未识

　　对于解说词，同学们已经并不陌生。本单元的第二篇说明文《从宜宾到重庆》，就是一篇介绍长江上游宜宾到重庆这一段绮丽景色、风土人情和历史知识的解说词，今天学习的《身居闹市人未识》，则是介绍我国最大、最重要的古钟知识的解说词，它们都与图像相辅相成，语言准确明白，讲究趣味性。现在请同学们再看一遍课文，考虑这样一个问题：课文为什么不以"给我国最大的古钟正名"为题，而要用现在的正、副标题？大家可以从本篇解说词的内容、结构以及文体特点三个方面入手来回答这个问题。
