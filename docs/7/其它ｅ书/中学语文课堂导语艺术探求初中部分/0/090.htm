晋祠

　　有位老师这样导入：

　　教师先让学生看书中的插图，让学生想象建筑物的形态，晋祠的山、树、水的色彩美，然后又做一个抛砖引玉的的导游：同学们！晋祠不仅有旖旎的风光美，令人赏心悦目，而且更有神工鬼斧的悠久历史的文物美，请同学们到晋祠一游！这样学生对晋词的形神美都有了初步的认识，这初步认识是学生了解和掌握课文内容的"加速器"，学生在这"加速器"的作用下，主动到课文里去寻找描绘"晋祠"山、树、水的自然美，以及古建筑的三绝美，从而使学生在形和神上都领略了其山清水秀的语句，了解到了它是古代劳动人民智慧的结晶。
