影子的故事

　　余中云老师的设计：

　　同学们，平时我们说话中常常可以听到两个成语："形影不离"和"立竿见影"，这两个成语中的"影"字是什么意思呢？（学生回答后板书：影子）是的，不管是人，还是物，都有影子。什么是"影子"呢？影子，就是物体挡住光线后，映在地面或其它物体上的形象。生活中只要有光线，事物就都会留下影子。影子，我们太熟悉了，可是，关于影子的知识我们能知道多少呢？好，今天我们就来给大家讲一下影子的故事。（板书课题：影子的故事）
