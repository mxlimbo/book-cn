过故人庄

　　郝明山老师这样导入：

　　在日常生活中，我们经常会遇见或感受到许多极有意义、蕴含着美或发人深省的事物与情理，或是人情中的真假、美丑、善恶，或是事物中的壮丽、妩媚、清幽，这些大千世界的种种异象风情，使无数人怦然心动，他们挥毫泼墨，诉诸文字，化为丰富多采的文学作品，光照人间，永垂后世。代表唐代文学辉煌成就的唐诗，以其独有的魅力放射出夺目的光采。美的事物、丰富复杂而又深沉高远的思想感情，都会凝炼而成高度精炼、意味深长、优秀绝伦的诗歌。《过故人庄》正是这样的诗篇（板书课题）。幽居山村，农家作客，独特的风光、自由的天地，以及朋友的感情是那样着人迷恋，令人神往；安于归隐、悠然自得的情怀虽未直接表述，却蕴含在清新隽永的诗句里，实在耐人寻味。（范读《过故人庄》）
