一面

（其一）

　　"一面"，见一面，一面之缘。本文写的是在悼念鲁迅先生的日子里，一位电车工人回忆了四年前和鲁迅先生的一面之缘。写的仅是一面之缘，怎样能表现出鲁迅的伟大人格和战斗精神的呢？本文具体记叙了什么事？作了哪些描写？从中看出了鲁迅的什么特点？请同学们认真学习本文。

（其二）

　　江苏张小忠老师的导入：

　　一上课，教师板书如下句子："两弯似蹙非蹙烟眉，一双似喜非喜含情目。"释义疏通后，教师说：这两句外貌描写，出自我国古典名著《红楼梦》，大家猜一猜：描写的是哪一个人物的外貌？有的学生很快猜出是林黛玉。教师问：为什么不是贾宝玉、王熙凤，而是林黛玉的外貌呢？学生稍作思考后回答：这两句描写抓住了林黛玉的眉目特点，写出了她多愁善感的性格。教师及时肯定，并联系学生写作实际：平时同学们的作文中时常可见这样的句子：×××大大的眼睛，高高的鼻子，樱桃似的小嘴，苹果似的圆脸。这些外貌描写好不好？学生笑着回答：不好，没有抓住人物的特征。教师顺势引出课题及教学目标：那么如何抓住特征进行外貌描写，如何通过外貌描写表现出人物的性格呢？这节课我们来学习阿累的《一面》，重点讨论这两个问题。

（其三）

　　鲁迅是伟大的文学家、思想家、革命家，他"横眉冷对千夫指，俯首甘为孺子牛"。本文中有一处议论，集中表现了鲁迅的伟大人格和战斗精神，同学们能把它找出来吗？（"这一来不会错了，正是他！站在前进行列最前面的我们的同志，朋友，父亲和师傅！憎恶黑暗有如憎恶魔鬼，把一生的时光完全交给了我们，越老越顽强的战士！"）
