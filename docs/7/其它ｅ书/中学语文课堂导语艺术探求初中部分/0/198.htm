夜走灵官峡

（其一）

　　同学们，我们已经学过了《小橘灯》和《最后一课》。《小橘灯》的主人公是谁？（小姑娘）她多大年龄？（只有八、九岁光景）她生活在什么年代？（1945年国民党反动派统治的黑暗年代）课文表现她的什么精神？（镇定、勇敢和乐观的精神）《最后一课》也有一个主人公是儿童，他是谁？（小弗郎士）他的身份是什么？（小学生）他生活在什么年代？（1871年阿尔萨斯亡国的年代）课文表现了小弗郎士思想感情怎样变化的过程？（表现了小弗郎士从幼稚、贪玩、不爱学习到懂事、成熟、热爱祖国语言、热爱祖国、理解老师的过程）今天，我们再来学习一篇以儿童为主人公的课文，它的主人公名叫成渝，"看起来不过七八岁"。他生活在什么年代？从他身上我们看到了什么？请同学们打开课本，看《夜走灵官峡》。

（其二）

　　我们已经学过了《小橘灯》和〈〈最后一课》，这两篇课文都通过对一个孩子的描写表现了重大的主题。今天我们再来学习一篇写孩子的课文，作者通过对这个孩子的记叙、描写，表现了什么主题呢？让我们一起来学习《夜走灵官峡》。

（其三）

　　1950年6月到1952午7月，我国建成了成渝铁路。成渝铁路从四川的成都到重庆，全长504公里，是四川省的交通干线。1952年7月到1956年7月，我国又建成了宝成铁路。宝成铁路北起陕西宝鸡，和陇海铁路相接，南至四川成都，和成渝铁路相联，是纵贯我国西部的运输大动脉，全长668．2公里，贯穿陕西、甘肃、四川三省19个市县。宝成铁路经过的地方，地形、地质情况十分复杂，每平均不到3公里就要凿一条隧道，每1公里就有一座桥梁，我们英雄的筑路工人克服了无数的困难，创造了无数的人间奇迹。今天，我们就来学习一篇歌颂英雄的筑路工人的课文――《夜走灵官峡》，学习时要注意理解文章是怎样围绕中心意思选材的。

（其四）

　　河北燕凤老师的导入：

　　给孩子取名字，大多数家长很认真，查字典，翻辞典，亲朋好友在一起讨论，集思广益，力争给孩子取一个好名字。或有纪念意义，或寄寓深深的期望……你的名字是谁给你取的，有什么意思，哪位同学愿意讲一讲？（指名两位学生讲一讲）

　　请看老师写在黑板上的名字"成渝"，你想到了什么？（有的同学想到铁路线，从成都到重庆）

　　"成渝"是文中一个孩子的名字。他的父母为什么给他起这个名字？他是一个什么样的孩子呢？今天我们一起来学习杜鹏程写的《夜走灵官峡》，认识一下成渝（板书课题）。

（其五）

　　《小桔灯》、《社戏》、《最后一课》和《夜走灵官峡》，在写法上有共同的特点，都是第一人称，通过"我"的口说故事，通过"我"的眼睛来摄取生活中的形象。《社戏）和《最后一课》相仿佛，其中的"我"是孩子，使作品的整个描写洋溢着儿童的情趣，跳动着一颗童心。《小桔灯》和《夜走灵官峡》则更相似，"我"都是大人，描写的都是孩子，都是在偶然的机会中相遇的孩子。一个是在黑暗旧中国艰难磨炼中的小女孩，一个是新中国在艰苦的生活环境中成长起来的小男孩。都是那么的令人喜爱，具有感动人的思想品质。
