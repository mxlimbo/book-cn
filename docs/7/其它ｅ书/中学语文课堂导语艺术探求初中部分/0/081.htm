积累资料学习办报

　　蔡澄清老师的设计：

　　同学们，你们平时搜集过报刊资料吗？你们自己编过小报吗？你们了解这方面的知识吗？如果你们还不了解，我们今天学习的这篇课文，就教给你们这些知识，让我们好好学习。

　　这篇课文包括"积累资料"和"学习办报"两则短文，都是知识性文章，都属应用性说明文。学习这类课文，我们要注意三点：

　　一是把学习理论与实践应用结合起来；

　　二是把掌握知识与培养能力结合起来；

　　三是把阅读练习与操作训练结合起来。总的要求是：学以致用，重在实践。
