江南春绝句

　　江南因其独特的地理条件，景色美，春来早。历史上有多少文人墨客用脍炙人口的诗句赞美过江南春色。同学们还记得北宋改革家王安石的一句颂春的名句吗？（学生答："春风又绿江南岸"）我们上学期学过白居易的一首《忆江南》词，请大家背诵一遍。（学生背诵："江南好，风景旧曾谙，日出江花红胜火，春来江水绿如蓝。能不忆江南？……）我们今天再来品尝一下唐朝诗人杜牧的《江南春绝句》，看他又是如何描绘出令人陶醉的江南春色美景的。
