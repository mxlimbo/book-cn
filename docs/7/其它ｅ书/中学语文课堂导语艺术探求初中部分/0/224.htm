植物世界的博物馆

　　喜爱植物世界的同学一定知道，在我国，有许多城市建有植物园，像南京市有中山植物园，她号称远东一大植物博物馆；厦门市的植物公园，被人们称为"绿色博物馆"，最云南省昆明市又建起了世界园艺博物园。我们在初一时，从课外自读课本上还读到过一篇题为《热带植物资源的袖珍库》的说明文，她介绍的是海南热带作物两院的经济植物院。今天，我们将作一次假想旅行，同课文作者一起到风光秀丽的庐山，参观地处亚热带高山上的"植物世界的博物馆"——庐山植物园。
