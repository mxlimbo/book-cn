开学第一课

　　甘肃常春秋老师是这样导入的：

　　新生入校的第一堂语文课，老师没有上教材（课文），而是让学生首先细心观察语文书封面。提出下列问题：世上的颜色种类很多，封面用了哪些色彩？哪些图案？设计者设计这些图案的目的是什么？有一位女同学这样理解："语文书的封面主要是由红、黄、灰、绿这几种颜色构成。红色代表着灿烂的阳光，绿色代表着幼苗。看！它就象征着我们在阳光下茁壮成长。"她的这一段话是从色彩的选取角度来理解象征意义的。从图案造型的角度观察、审美，她又说道："这个图案又像一双手，告诉我们牢牢抓住中学时代这个被人们喻为'黄金时代'的美好时光，学得更多的知识。"另一位男同学用形象思维的方式发表自己的看法："封面这个图案，又像一个手电筒，好像是为我们学好语文知识照亮前进的道路。"又一个同学的发言，不但综合了上面几位同学的看法和理解，还有自己独到的见解。他由图案美、色彩美，产生有意义的联想。他说："画面上的这些色彩，红、黄、绿，都是绘画中的基本色彩，它们可以调配成各种不同的颜色。这就让我们从中领悟出，各门学科都离不开语文。语文就像这些色彩一样，是绘画的最基本的颜色，我们可从中得出：语文是各门学科的基础！"这种教法，比起"语文是重要学科"等单纯的说教，力度就大得多，学生也感兴趣。美不只是具体的、形象的，而且还具有很强的感染力。任何美的事物，都能激发人的感情，使人们精神上得到很大的快乐，思想上得到很大的收获。
