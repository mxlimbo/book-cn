怀疑与学问

（其一）

　　我们都读过英国大戏剧家莎士比亚的名言："书籍是全人类的营养品。""生活里没有书籍，就好像没有阳光；智慧里没有书籍，就好像鸟儿没有翅膀。"我们也喜爱苏联大文豪高尔基的一句名言："爱书吧，它是你知识的源泉。"

　　书上说的，难道还要怀疑吗？怀疑跟学问有什么关系呢？我们知道，伟大的天文学家哥白尼创立地动学说，就是从怀疑亚里士多德、托勒密的天动学说开始的。伟大的数学家祖冲之，决不"虚推古人"，敢于怀疑前人的陈腐学说，推翻错误结论，作出了计算圆周率、制定《大明历》等杰出的贡献。历史学家顾颉刚从小就有怀疑的精神，读书时善于提出疑问，一生为祖国的历史学作出了宝贵的贡献。今天我们一起来读一读他写的《怀疑与学问》，就能进一步明白做学问为什么要有怀疑的精神，明白什么是我们提倡的怀疑精神，这对于培养开拓精神和创造精神，是大有益处的。

（其二）

山东闵凡旺老师这样导入：2300多年前，古希腊有一位著名的思想家叫亚里士多德。他认为"物体降落的速度，因其重量大小不同而不同，重的先落地"。这种思想统治了人们近2000年。400多年前，一位意大利青年表示了自己的怀疑。他亲自在180英尺高的比萨斜塔上用一大一小两个铁球反复试验，结果，两个铁球同时落地，从而推翻了亚里士多德的理论。这位青年就是伽利略，他很有怀疑精神。那么怀疑与学问之间又有什么关系呢？今天，我们就来学习第19课一一《怀疑与学问》。

（其三）

　　河南郭国选老师这样导入：

　　上课时，教师先安排一个学生在办公室等候，然后自己匆匆走进教室，故作焦急地对学生们说：刚才××同学的家长来了电话，说他离家出走了。

　　学生们一听大吃一惊，顿时嚷嚷开了，有的说要赶紧到汽车站、火车站去找，有的说要赶快去派出所报案，还有的迫不及待地站起来，要去寻找。就在教室内乱成一锅粥的时候，教师拍了拍巴掌，那个同学笑眯眯地站在了门口，教室里霎时静了下来，同学们不知道是怎么回事。教师就问：××是咱班的优秀同学，他近来有什么异常的表现吗？学生回答：没有。有的还补充道：他上次数学还考了第一。教师接着说：既然他没有一点可能出走的迹象，为什么老师一说，同学们就深信不疑了呢？这是因为缺乏一点怀疑的精神。小事如此，做学问也如此。好，今天我们就来学习《怀疑与学问》。
