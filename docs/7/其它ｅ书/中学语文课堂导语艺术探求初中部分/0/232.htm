字风和世风

　　生活中有些反常现象，我们见多了，往往也就习惯了，想不到去分析它，深究它。例如我们到街上走走，只要稍微留心一下，就可见商店的招牌很多是用的繁体字。对这种社会用字的"繁化"风，你也许会认为它不对，但为什么不对，不对在哪里，却没有很好地去思考。今天我们所要学习的课文《字风和世风》，对社会用字不规范的状况，特别是滥用繁体字的现象，提出了尖锐的批评，精辟地分析了这种风气的危害和实质。这对启发我们碰到问题，学会思考和分析，大有教益。
