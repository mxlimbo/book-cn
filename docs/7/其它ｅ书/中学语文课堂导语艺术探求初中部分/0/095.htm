看云识天气

(其一)

　　同学们，不久前我们学习了课文《天气陛下》，懂得要跟天气陛下作斗争就得知道天气的法则。天气的法则很多，而云是天的"脸色"，通过看云的形态、变化，识别晴阴风雨，预知天气变化，这是千百年来人类跟大自然斗争所掌握的天气的重要法则之一。今天我们所要读的课文《看云识天气》，具体生动地介绍了一些看云识天气的经验和知识，值得我们好好学习。

（其二）

江苏周道富老师这样导入：首先出示小黑板，上书几条谚语：

　　1．天上沟沟云，地下雨淋淋。
　　2．朝霞不出门，晚霞行千里。
　　3．天上鲤鱼斑，晒粮不用翻。
　　4．东虹轰隆西虹雨。

　　学生看后，津津有味，教师趁机启发道：这几句谚语是我国劳动人民在长期的劳动实践中总结出来的。同学们读后可以得到哪些启示呢？或者说我们可以从中看出哪两种关系呢？（学生热烈讨论后，教师总结）我们可以从中发现两种关系：第一，天气和农业生产、和人们的日常生活密切相关，因此我们必须随时关注天气变化；第二，天气变化与云的变化有着内在的联系，因而我们可以"看云识天气"（板书）。

（其三）

　　广东虞伟力老师这样导入："大家都听说过'草船借箭'的故事吧！""听过！"学生多数熟识此故事。

　　"那孔明究竟是靠什么'借'得这十万支箭的呢？"

　　学生七嘴八舌，终于有人回答道："是大雾天气。"

　　"对，孔明就是靠大雾天气来'借'得十万支箭的。那孔明又怎样知道有大雾天气呢？"学生冥思。教师解答："小说里的孔明故弄玄虚了一番，其实他真正知道有大雾天气的是通过观天象，即类似于我们今天的'看云识天气'"。（板书）

（其四）

　　天气和人们的生活有着十分密切的关系。早晨，六点钟醒来，一打开电视机，就听到播音员在播送天气预报。因为内容重要，影响着每一个人的生活，有时甚至涉及千百万人的生命财产，所以播送得特别慢。晚上，一打开电视机，新闻电视以后，就是中央气象台的天气预报，因为重要，有时还专门映出一张风和雨的布局图。过去，现代化的测天仪器还没有发明，现代化的信息传播工具还没有出现，却照样有预报气象的"报纸"和"电视"，人们往往一起身就抬头看天上的"报纸"和电视"。在长期的生产和生活实践中，人们积累了丰富的气象知识，总结了许多观察天气变化的方法，从阳光，月色，星状，风向，虫态，水温，皮肤的感觉，甚至身上的老伤是否发作，来预知天气发展的趋向。在各种观物识天气中，最重要的方法莫过于观云识天气。如果把风看作天气陛下的呼吸，太阳看作他的眼睛，那么云就是他的脸色，"关公脸，一天变三变"，通过脸色的变化，我们就可以了解天气陛下的喜怒哀乐。随着四化建设发展，将来总有一天，天气陛下要看我们的脸色行事，但目前，我们暂时还得看他的脸色行事；要改造他，首先还得了解他，掌握他的脾性。《看云识天气》这样的文章，就是人类在征服大自然过程中所写下的一个小小的备忘录。
