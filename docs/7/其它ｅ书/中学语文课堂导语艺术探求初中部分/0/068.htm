花儿为什么这样红

　　有位老师这样导入：

　　上课前教师带着几朵鲜花和稀盐酸溶液、肥皂水各一杯走进教室。学生都不解其意地望着老师，就在学生疑窦未开之时，老师请一名学生把两朵鲜花分别放入两种溶液之中，过了一会儿，同样颜色的鲜花，放在稀盐酸溶液里的花变成了红色，放入肥皂水里的变成了蓝色。学生直观了花的化学变化的美，这美像聚光镜一样把他们的注意力立刻集中到《花儿为什么这样红》上。老师及时地说：同学们，大家想要探究花红的原因吗？让我们来研读课文吧！
