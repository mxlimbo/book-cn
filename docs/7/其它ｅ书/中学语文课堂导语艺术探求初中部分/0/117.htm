马 说

　　同学们都读过《爱莲说》、《捕蛇者说》，今天再来读一篇《马说》。为什么这几篇文章的题目都叫"说"呢？"说"也称"杂说"，是文言文的一种文体，说得浅易一点，"说"就是说说谈谈的意思。"杂说"也就是"杂谈"。它的内容很广泛，可以就一事、一物、一景生发开去，为之一说；它的形式比较自由灵活，可以记叙、描写、抒情、议论；可以议论为主，也可以记事写景为主，或者与杂文、小品文类似。其目的都在于说明一种道理，或是陈述作者对社会某些问题的观点，这是"说"的一个鲜明特点。如《爱莲说》，看起来是描绘莲的形象，赞美莲的品格，实际上是借说爱莲的原因进而说明当时社会的污浊，抒发作者不与世俗同流合污而洁身自好的情操。那么，《马说》一文的作者要谈的是什么道理呢？
