生命的意义

（其一）

　　同学们，首先让我们做一道非常简单的数学应用题，请大家注意听：自从我们降生，就开始拥有了生命，假设我们的平均寿命是72岁，那么请同学们回答一下，你大约已经度过了几分之几的人生？

　　好，在座的每位同学已经走完了六分之一的人生旅途。那么，大家考虑过这样一个问题没有，即我们应该如何度过自己的一生呢？下面请同学们以自己为例，展开讨论，然后自由发言。请不要空喊口号，自己怎么想的就怎么说，开始……（学生发言）

　　同学们发言非常踊跃，而且可以看得出大家已具备了独立思考问题的能力，这很好！其实刚才我们讨论的是一个非常重大而严肃的问题，那就是"生命的意义究竟是什么"，对于这个问题，各人有各人不同的理解，它反映了人们不同的人生价值观。

　　今天，我们就来学习一篇新课文，学习文中主人公的人生观，然后大家再对照自己，重新思索一下我们的人生观应该是什么。这篇文章的题目是——《生命的意义》（板书）

（其二）

　　每当清明时节，我们总要到烈士公墓去祭扫。当你伫立在烈士墓前深深默哀的时候，同学们，你想到了什么？缅怀革命先烈，珍惜今天的幸福生活……不过，在烈士墓前，你可曾想过，人的一生应该怎样度过，生命的意义何在呢？今天，我们就来学习《生命的意义》这篇课文，看看苏联长篇小说《钢铁是怎样炼成的》的主人公——保尔·柯察金在烈士墓前是怎样思考生命的意义的。

（其三）

　　"人，最宝贵的是生命。生命对每个人只有一次。这仅有的一次生命应当怎样度过呢？每当回忆往事的时候，能够不为虚度年华而悔恨，不因碌碌无为而羞耻；在临死的时候，他能够说：'我的整个生命和全部精力，都已经献给了世界上最壮丽的事业——为人类解放而进行的斗争'"这是苏联的优秀长篇小说《钢铁是怎样炼成的》的主人公保尔·柯察金，关于生命意义的思索。他是在什么情况下思索生命的意义的？他为什么能想得这么深刻呢？今天，我们一起来学习《生命的意义》（板书）。

（其四）

　　河北史艳丽老师的导入：

　　同学们，你们现在正值青春年少，豆寇年华，你们思考过人生这个严肃的话题吗？我们每个人都要从呀呀学语到雪染双鬓，这漫长而又短暂的一生应该怎样度过呢？我们每个人都只有一次生命，这宝贵的生命究竟怎样度过才有意义呢？下面我们听一听革命者对人生的看法吧！（放录音："人，最宝贵的是生命……"）刚才同学们听到的这段话就出自课文《生命的意义》，今天，我们就一起学习这一课。

（其五）

　　甘肃王跃农老师的导入：

　　同学们，一位哲人曾经说过："你不能决定生命的长度，但你可以控制它的宽度。你不能左右天气，但你可以改变心情。你不能改变容貌，但你可以展现笑容。你不能控制他人，但你可以掌握自己。你不能预知明天，但你可以利用今天。你不能样样顺利，但你可以事事尽力。"一个人的一生究竟该怎样度过呢？生命的意义何在呢？前苏联作家尼古拉·奥斯特洛夫斯基在《生命的意义》一文中给我们做出了极好的回答（板书课题：生命的意义）。

（其六）

　　湖南李春华老师这样导入：

　　冯巩是我们非常熟悉、非常喜爱的一位相声演员，大家一定听过不少他说的相声。除了说相声，他还演电影，拍电视剧呢。获奖影片《没事偷着乐》就是他主演的。这部电影的主题歌颇耐人寻味，歌中唱的是："人这一辈子，怎么着都是过。与其皱眉头，不如偷着乐。冬天别嫌它冷，夏天别怨它热。有钱别装穷，没钱也别摆阔。别盼望天上掉馅饼，也别指望鬼推磨。苦辣酸甜都尝尝，才算没白活。"这通俗幽默的歌词中蕴含着对人生意义的思考。一个人的一生究竟该怎样度过呢？生命的意义究竟何在？前苏联作家尼古拉·奥斯特洛夫斯基在《生命的意义》一文中给我们做出了极好的回答。今天我们就来学习这篇课文。
