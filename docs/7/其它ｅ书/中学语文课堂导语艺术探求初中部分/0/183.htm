吴门桥

　　同学们，在我们祖国——这块有着古老文明史的土地上，名胜古迹星罗棋布。仅就桥梁而言，除刚刚学过的驰名中外的赵州桥、芦沟桥外，大家说说看，还有哪些历史名桥？（学生回答）教师归纳：对了，历史上著名的桥千千万万。那么，当我们登临这些名桥俯仰观望之时，你的感触如何呢？你是否会追古思今、感慨万千呢？好，现在让我们来共同阅读《吴门桥》（板书），看看作家碧野是怎样描写吴门桥这颗历史明珠在新的时代里更加异彩勃发、璀璨夺目的。
