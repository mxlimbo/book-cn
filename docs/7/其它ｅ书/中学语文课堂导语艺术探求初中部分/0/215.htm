岳阳楼记

　　我国湖南省岳阳市有一处古迹，它是这个市西门的城楼，名叫岳阳楼。这座楼高三层，下瞰洞庭湖，碧波万顷；遥望君山，气象万千。它是唐代建筑的，到宋代岳州知州滕子京重修过一次，因为大文学家范仲淹为此作过一篇《岳阳楼记》，此楼因而著称于世。在这篇作品中，"先天下之忧而忧，后天下之乐而乐"的名句，千百年来脍炙人口。解放后这座楼整修一新，成为中外人士向往的游览名胜。同学们听了我介绍的这处名胜、这篇名作和这个名句，一定发生了很大的兴趣，急于了解个究竟了吧，好，现在我们就开始学习《岳阳楼记》这篇课文。
