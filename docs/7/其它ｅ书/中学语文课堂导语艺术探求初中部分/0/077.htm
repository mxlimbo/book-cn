活板

　　电话机键盘上有几个数字？为什么用这几个数字就能编出无数的电话号码跟千家万户通话？这是因为这一组基本数字的排列顺序和组合形式，能够不断变化。关键在个"活"字，即活用这几个基本数字。我们今天读的书报虽然已经广泛运用电脑照排进行印刷了，但是铅字排板印刷还在继续使用，这铅字排印重要的特点也就是一个"活"字。要知道在古代印书是很费难的，那时文字传播全靠手抄，唐朝开始有了雕板，即把文字刻在一块木板上；印一页书，就要刻一块木板，印一本书，就要刻很多块木板。自然比手抄要快，但很不灵活，而且要花费很多的人力、物力。到了宋代，毕升发明了活板印刷，从雕刻整块的木板发展到使用可以装拆的活字板，这在我国印刷发展史上是一个很大的飞跃。我国使用活板印刷比欧洲要早四百年，活板印刷是我国古代四大发明之一，对整个人类文化的发展作出了巨大的贡献。这篇说明文就是介绍毕升发明的活板印刷术的。
