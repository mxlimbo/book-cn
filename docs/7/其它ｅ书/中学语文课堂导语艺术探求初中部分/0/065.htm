和青年同志们谈写信

　　余中云老师这样导入：

　　同学们，有谁没有见过"信"吗？（板书：信）好，大家都见过信。我国历史上最早的一封信，是春秋时代郑国的大夫郑子家给晋臣赵宣子的信，距今已有几千年历史了。现在大家看到"信"已经极其平常了。

　　每一个同学都会写信吗？（板书：写）

　　同学们都不大敢讲会写信，其实，即使不会也不要紧。今天我们给大家请来一位老师，专门跟我们谈谈写信的事。（板书课题）这就是我们今天要学习的课文。（齐读课题）只要大家认真地阅读一遍课文，我保证大家就懂、就会写信了，信不？
