向沙漠进军

（其一）

　　1980年春，我国著名科学家彭加木同志，进入塔里木盆地沙漠地区考察，他来到人迹罕至的荒漠中搜集第一手资料，最后迷路断水，失去联系，为祖国科学事业献出了宝贵生命。彭加木为什么要到沙漠中去呢？是为了要创造中华民族更加美好的未来。为了参加向自然的斗争，创造美好未来，我们必须了解过去，掌握知识。现在大家一起来学习说明文——《向沙漠进军》。

（其二）

　　全世界陆地上约有七分之一的土地是光秃秃的沙漠。除了南极洲和欧洲，其他各洲都有大片沙漠。非洲的撒哈拉沙漠面积达800万平方公里，相当于澳洲的总面积。我国新疆的"塔克拉玛干"在维吾尔语中就是"进去了出不来"的意思。你们能根据平时的积累说说沙漠是怎样危害人类的吗？（学生介绍）好，刚才一些同学生动地描述了沙漠的危害性。今天，我们要学习的一篇说明文，不仅具体地介绍了沙漠对人类的危害，而且准确地说明了我国人民怎样抵御沙漠的侵袭，怎样主动向沙漠进军，富有科学性、知识性和趣味性。

（其三）

　　一提到竺可桢，同学们都很熟悉。是的，在初中第二册语文课本中，我们学过一篇题目叫"卓越的科学家竺可桢"的文章。有哪位同学能比较全面的介绍一下竺可桢是位怎样的科学家吗？（学生回忆：卓越的科学家，著名的气象学家，在气象学研究领域里做出了重要贡献，终生献身祖国的科学事业，为科学普及工作，为发展我国农业做出了许多有益的贡献，著作有《物候学》、《竺可桢日记》等）刚才的介绍突出了竺可桢的主要方面，可大家不一定知道，竺可桢还是一位研究治理沙漠、指导改造沙漠的专家。今天，我们就是研读他的《向沙漠进军》一文。通过学习，我们可以进一步了解竺可桢渊博的学识、严谨的治学态度和献身科学的精神。

（其四）

　　江苏程言宝老师的导语：

　　从电视和报刊上我们看到了长江和嫩江、松花江流域发生了特大洪涝灾害，广大军民以顽强的毅力战胜了洪水这个天敌。人类天敌很多，除了洪水，还有台风、冰雹、地震、沙漠，人类是如何战胜的呢？洪水泛滥，固然有降水量过多的缘故，可是森林保护和植树造林跟不上也是重要原因。沙漠横行，依靠的是什么武器？面对着沙漠的肆虐，我们怎么办？著名科学家竺可桢响亮地提出"向沙漠进军"（板书课题和作者）。"举危害，抓特征，历数沙患与狂风；游击战，阵地战，引水植树相抗衡。"同学们，这歌诀能概括《向沙漠进军》的内容吗？请自读课文。

（其五）

　　河南李福祥老师这样导入：

　　也是这么一个初秋时节，13世纪初叶的一天，一代天骄成吉思汗正率部驰骋疆场，横扫千军。当他经过伊金霍洛旗地区时，深深为那里布满绿色的草原奇景吸引住了，勒马观赏，欣羡不已，当即插马鞭于地以为记，令部下在他百年之后将他掩埋于此，以偿夙愿。花开花落，日月轮回，悠悠七百多个春秋过去，如今，当我们把目光投向昔日故地时，夕阳残照里，风逐尘沙飞。沙漠吞噬了美妙怡人的绿，遮掩了高天明丽的蓝。面对沙漠的肆虐，谁能无动于衷？如何制止沙漠来势汹汹的进攻，还大地以新绿，还长天以湛蓝？现在让我们共同聆听著名科学家竺可侦"向沙漠进军"那振聋发聩的战斗召唤吧。
