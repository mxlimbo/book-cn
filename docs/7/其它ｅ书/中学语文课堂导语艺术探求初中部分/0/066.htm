核舟记

（其一）

　　《核舟记》在当时被誉为奇文。它写了一件徽雕工艺品，原材料只是一枚长不满一寸的桃核。文章的最大成功，在于完整地说明了雕刻家巧妙的艺术构思，使人能够通过这四百多字一篇短文，想象出这件久已失传的工艺品原来的样子。作者是怎样才做到这两点的？他抓住了"核舟"怎样的特征来写？让我们来认真阅读这篇作品吧。

（其二）

　　安徽徐伟老师这样导入：

　　一上课，教师先放了一首《泰坦尼克号》主题歌。优美的音乐立刻引起了同学们的极大兴趣。接下来教师叙述道：1912年4月10日，一艘巨轮从英国的南安普敦启程驶往纽约，但在第四天的途中，船体触到一座北大西洋的巨大冰山，船身折为两半，沉入海底，船上1300多名旅客与近900名船员，几乎全部遇难。后来，一位天才导演根据这个题材拍成了曾经全球最为轰动的一部电影，这就是——《泰坦尼克号》（学生齐声回答）。好，请同学们再把目光从西方转向我们古老的东方，看看我们中国。在明代，有一位杰出的雕刻家在一个很小的桃核上，雕刻了一艘非常精美的小船，船上有旅客……（1300多人，有调皮的同学接着说）不，有旅客3人，船员2人。后来，有位作家以此为题材写了一部非常有名的作品，这就是——《核舟记》（板书）。
