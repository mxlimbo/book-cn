美"挑战者"号航天飞机升空后爆炸

　　陈军老师这样导入：

　　同学们，你们知道什么叫"航天飞机"吗？哪位同学说说？大家可以查查工具书，找答案。（"航天飞机"是人们探索太空奥秘的飞行工具，是我们当代科学技术迅猛发展的象征。）

　　今天，我们学习一篇课文，内容不是说明"航天飞机"的，而是描述人类探索太空奥秘的一次悲剧性事件。（教师板书课题）现在，请同学们打开书，看"自读提示"。请一位同学找出概括这篇课文中心内容的话（"及时报道了航天飞机失事的情况和在美国所引起的巨大反响"）。这句话很重要，它不仅概括了中心，同时也给我们指明了 阅读的思路，即一方面要了解"失事""情况"，另一方面 要了解"巨大反响"请同学们依照这个阅读思路快速读课文，把说明"失事情况"和"巨大反响"的关键性语句画上线条，看谁读得快而又抓得准、找得全。
