听 潮

（其一）

　　同学们，你们看到过海吗？听到过有关海的故事吗？海，无边无际，辽阔壮美，神秘莫测，变化无常。有时它平静温柔，海鸥掠过水面，在海空之间盘旋翱翔；有时它汹涌澎湃，浊浪咆哮。生活在海边的人，目睹海的情态，耳闻海的呼啸，熟悉海的脾气，热爱大海；远离海边的人，读着描绘海景的佳作，也有身临其境之感，感受到海的壮观。作家鲁彦的《听潮》一文，着力描写了海潮涨落的情景，学习本文，我们会如同在欣赏一曲威武雄壮的交响曲。

（其二）

大海，有时平静，有时凶猛。落潮时，平静的大海波光粼粼，水天相接，充满了诗意；涨潮时，凶猛的大海波浪滔天，声如雷鸣，使人感到恐惧。可是，有人就是欣赏涨潮时的凶猛的大海，因为它充满了力量，具有雄壮美。今天，我们就来学习鲁彦的《听潮》，看看作者是怎么样讴歌大海的雄壮美和它的伟大力量的。

（其三）

　　江苏王纯老师这样导入：上课了，教室里顿时鸦雀无声。老师打开准备好的录音机，教室里立刻响起剧烈的海潮声，一阵紧接一阵，此起彼伏，掺杂着海鸥的鸣叫，航船的笛声和岸边寺院的禅声……几分钟过后，海潮渐渐退却，教室里又恢复了宁静。老师问同学们有什么感受。他们显得很激动，有的说有一种雄浑开阔的感觉，令人振奋，催人向上；有的说海潮具有无比的威慑力，势不可挡，让人感觉只能前进，不能后退；还有的说，海潮时起时落，就像一个人的一生有起有伏，能给人以深刻的思考。于是老师很兴奋他说：看来，鲁彦先生的《听潮》，一个"听"字不但不会让大家感到难以理解，而且还会引起大家强烈的共鸣。好，我们就一起去普陀山海岸边那所靠寺院的小屋，听一听那时涨时落的海潮之音吧！
