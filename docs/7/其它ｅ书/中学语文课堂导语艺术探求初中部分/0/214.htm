语言的实际运用：得体

　　河北宫艳芬老师的导语：

　　同学们，今天我先给你们讲一个小笑话。古时候，有那么一位不大会说话的女婿，在岳父73岁生日那一天，买了许多礼物前去祝寿。席间大伙儿说得正高兴，这个祝老爷子"万寿无疆，"，那个祝老爷子"福如东海"，这位贤婿也不甘落后："岳父大人，俗话说，七十三、八十四，阎王不找自己去，你可得留点神。"一句话，把老爷子说恼了。过了几天正赶上二姨子的孩子过百日，媳妇千嘱咐，万叮咛："这次你千万别说话啦！"果然，这次这位贤婿一言不发。不料，快离开时却来了一句："这回我可一句话没说，等将来这孩子死了，你们可别怨我！"（学生大笑）教师问：你们笑什么？学生回答：这位女婿太不会说话了。教师说：像这样说话不得体、不分场合的例子，你们生活中遇到过吗？（学生大笑）那么，怎样才能做到语言得体呢？请同学们打开书，仔细阅读知识短文——《语言的实际运用：得体》。
