周处

　　我们已经学过课文《扁鹊见蔡桓公》，它说的是蔡桓公患病以后讳疾忌医，终于断送了性命。这个反面例子提醒我们，一个人如果思想上有了毛病，也得及时治治，不然就会犯错误，成为一个被人唾弃的人。今天我们学的课文《周处》，举的是一个正面的例子，讲的是古代有个周处，原先品行坏，被人看作是祸害，后来觉悟了，改过自新，终于成了一个有用的人材。在日常生活中，我们有时也会犯这样那样的错误，有了错误后怎么办？从周处改恶从善的故事中，我们可以得到一些有益的启示。
