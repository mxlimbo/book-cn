弈喻

　　张红萍老师的设计：

　　同学们，作为一个初中学生，大家已经不再是一个只知道以好奇的目光打量周围世界的小孩子了。你们不仅能够从生活中发现问题、提出问题，而且能够独立判断和思考一些问题了。这无疑是你们开始走向成熟的标志。但是，同学们是否能够经常自觉地去观察生活、思索人生？如果我们能够这样做，就不难发现，生活中有许多看似寻常的小事，实际上却蕴含着一些做人的大道理。我们即将学习的这篇课文《弈喻》，就正是借生活中人们司空见惯的下棋这一普通现象，生动地揭示了一个引人深思的生活哲理。下面，请同学们先阅读一下课文，看作者究竟告诉了我们一个什么为人处事的道理？
