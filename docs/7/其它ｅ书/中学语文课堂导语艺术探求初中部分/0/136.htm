青纱帐――甘蔗林

　　张基华老师的设计：

　　今天我们学习《青纱帐――甘蔗林》。首先让我们一起来回忆一下诗歌的一般常识（结合第二册《谈谈诗歌》） 哪位同学起来说一下诗歌的分类：诗歌从形式上分，可分哪几类？从内容上分，又分为哪几类？《青纱帐――甘蔗林》属于哪一类？

　　这首诗属现代抒情诗。"抒情诗一般没有完整的人物形象和故情节，直接抒发思想感情。"然而这首诗不是凭空抒发感情，发表议论，而是借助于具体鲜明的形象，因而不失诗歌的形象、含蓄的特点。学习这首诗歌的目标是：弄清诗人借助于什么形象，抒发怎样的感情；又是通过怎样的手段把形象描绘得鲜明生动，把感情表达得淋漓尽致的。（板书课题）
