笑的武器

　　史留毅老师的设计：

　　人常说："笑一笑，十年少。"这句话告诉我们一个简单的事实：笑——具有很大的作用，它可以养性益身。不仅如此，在某种特殊的情况下，它还是一种战斗的武器。譬如大家十分喜爱的相声、小品等文艺节目，演员诙谐幽默的演技令人忍俊不禁，甚至捧腹大笑。因而人们把这些演员捧为"笑星"，像大家熟知的相声界一代宗师侯宝林、弟子马季、姜昆；小品演员赵本山、潘长江、宋丹丹等，他们在演出逗笑的过程中，或对敌人尖刻嘲笑，或对自己人善意讽刺，或对真善美热情颂扬。大家也在笑声中受到深刻的启迪教育。这种艺术是演员通过逗笑发挥其战斗、宣传、教育作用的。同相声小品一样，绘画中也有一种令人发笑的造型艺术——漫画。有不少小朋友就特爱看一种小小画刊《讽刺与幽默》中的漫画，（有谁看过，请讲讲）乍看，不由你会发笑，是不是笑过之后我们也从中明白了一些事理呢？今天，我们一起读一篇评介漫画家华君武爷爷是如何用"动物漫画"作为战斗武器的文章——《笑的武器》。（板书题目）
