榆钱饭

　　有位老师这样导入：

　　老师首先问学生：同学们，你们自己写的文章能不能编入语文课本中来？一石激起学生心中的波澜，课堂一下就活跃起来，大家都不知老师葫芦里卖些什么药。中学生的文章怎么可以编入中学课文呢？

　　老师稍顿一下，接着说：这并非绝对不可能的事。在五十年代，有位中学生写了一篇题为《青枝绿叶》的小说，不久，就编进了高中语文课本。他是谁呢？他就是少有名气的当代作家刘绍棠。不久以前。为了歌颂党的富民政策，他以自身的经历又写了一篇散文，叫《榆钱饭》（板书），今天我们要学习的就是他这篇充满泥土气息的文章。
