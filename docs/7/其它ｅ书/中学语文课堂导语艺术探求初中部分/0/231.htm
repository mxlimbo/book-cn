卓越的科学家竺可桢

（其一）

　　走进公园，来到郊外，漫步农村，看到桃花开了，柳絮飞了，听到布谷鸟叫了，你会陶醉在大自然的美景之中。但作为一个气象学家，他会怎样呢？你去过一次公园，短时期之内，恐怕不会再去第二次了，你会觉得没有什么东西可看了。但作为一个气象学家，他会怎样呢？本文所写的卓越的科学家竺可桢，就是一个卓越的气象学家，他热爱大自然的美，更把美的大自然作为观察研究对象，几十年如一日，从不间断，取得了常人想也没有想过的巨大成绩，这是多么非凡的事情！本文还写了竺可桢的其他几件事，我们一起来向"卓越的科学家竺可桢"学习吧。

（其二）

　　我们每天都收听到天气预报，天气预报对于工农业生产、交通运输和人民生活有着十分密切的关系。天气预报不但有当天的、明天的，还有在今后一段较长时间内，甚至几十、几百年的，研究天气变化的科学就是气象学。你想了解一个杰出的气象学家的情况吗？想了解一个卓越的科学家应有什么样的治学态度和精神吗？请读《卓越的科学家竺可桢》。

（其三）

　　同学们从小都有自己美好的理想，有的想长大了当一个人民教师，有的想当一个农艺师，有的想当一个文学家，也有的想长大了当一个科学家……，为祖国的四化建设贡献力量。同学们，你可知道科学家们是怎样学习、工作和生活的吗？今天我们来学习新的一课——《卓越的科学家竺可桢》，它真实而生动地记录了卓越的科学家竺可桢从事我国气候科研的几个片断，他的严肃认真、实事求是、勤勉有恒的治学态度和献身伟大的社会主义祖国科研事业的崇高精神，为我们做出了榜样。
