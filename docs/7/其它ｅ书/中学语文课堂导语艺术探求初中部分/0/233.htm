自传

　　人是社会的产物，只有在社会群体中他才能正常生活。离群索居是一种孤癖，强迫隔世常被作为惩罚手段之一。作为社会的成员，需要社会交往，需要彼此沟通，需要交朋结友，正如飞鸟需要空气，游鱼需要流水。《诗经》中有这样的诗句："嘤其鸣矣，求期友声，相彼鸟矣，犹求友声。矧伊人兮，不求友声。"鸟犹如此，更何况人呢？那么，我们该交怎样的的朋友呢？交"好"朋友。一个好朋友就像一本好书，会告诉你怎样做人，怎样生活。古语"近朱者赤，近墨者黑"，说得正是这个道理。今天，我介绍一位朋友给同学们，他就是茅盾先生。他会告诉大家怎样才能成为一个真正的"大写的人"。下面我们一起来学习《自传》这篇课文。
