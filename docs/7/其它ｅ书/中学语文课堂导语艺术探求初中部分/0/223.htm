蜘蛛

（其一）

　　蜘蛛是我们常见的一种节肢动物。人们知道它会张网捕虫，但它究竟怎样捕虫的，许多人并不清楚；它会吐丝织网，但它为什么会吐丝织网，许多人也不一定了解；至于蛛丝又有哪些用途，知道的人恐怕更少。向人们介绍这些知识，这是写说明文的宗旨。今天，我们来学习周建人先生写的《蜘蛛》一文，看看他是怎样准确而生动地说明蜘蛛结网捕食的生态特征的。请同学们翻开课本。

（其二）

　　同学们才学过布丰写的《松鼠》，知道松鼠是一种非常漂亮的小动物，同松鼠比起来，蜘蛛的外形就谈不上讨人喜欢了。对于这种人们虽然熟悉但并不喜爱的节肢动物，怎样介绍出新意？怎样使人读了感到有兴趣？这里有不少学问。我想，有不少同学一定已经认真预习过周建人先生写的科学小品《蜘蛛》，现在请这些同学来说说读过这篇课文以后有哪些启示。
