宜兴紫砂

　　如果你到过江苏省宜兴市，我想，你除了驻足陶醉于张公洞、善卷洞、灵谷洞、莫蠡洞等人间仙境，一定会流连忘返于鼎蜀镇琳琅满目的陶瓷市场，为那里的日用陶、细陶、精陶、均瓷、青瓷等几大类数千种产品所惊叹，更为紫砂工艺陈列馆里陈列的几百把出自世世代代制壶工艺名匠之手的紫砂壶所折服，这些被日本学术界人士称之为"古今中外无与伦比之名器"的紫砂壶，为什么能为世人所瞩目？要说明这样一种精致而珍贵的特种工艺品，又该怎样谋篇布局、遣词造句？今天，我们学习《宜兴紫砂》一文，可以从中得到有益的启示。请同学们打开课本——
