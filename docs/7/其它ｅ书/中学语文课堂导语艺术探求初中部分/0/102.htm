雷锋精神永放光芒

　　雷锋是伟大的共产主义战士。 1963年3月5日，毛泽东同志亲笔题词："向雷锋同志学习"，周恩来同志题词："向雷锋同志学习：憎爱分明的阶级立场，言行一致的革命精神，公而忘私的共产主义风格，奋不顾身的无产阶级斗志。"六十年代以来，除了十年动乱的那一段日子，学习雷锋的热潮不断掀起，但也有人对学习雷锋有这样那样的想法，甚至还出现了"雷锋同志没户口，三月来，四月走"的说法。到底雷锋精神有没有过时，应怎样看待向雷锋同志学习的活动，请学习《雷锋精神永放光芒》一文。
