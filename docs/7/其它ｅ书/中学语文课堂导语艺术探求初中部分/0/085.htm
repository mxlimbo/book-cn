继续保持艰苦奋斗的作风

（其一）

　　同学们，我讲两件报纸上曾登载过的事，请你们评论一下。一件是：春节时，一个青年富豪仰望着纷纷落下的爆竹纸屑兴奋得流下热泪，因为他刚刚点燃的四个爆竹是用两千元人民币卷成的。另一件：一位北京"大款"用两万元一桌的宴席招待广东"大款"竟遭奚落，随后广东"大款"用六万元一桌回请，而北京这位"大款"竟啪地打开密码箱，甩出三十五万元说"今天这桌就照这个数！"（在学生发表意见后，教师作扼要归纳）改革开放后，人们的生活水平普遍提高了，但有些"富豪"，"大款"，讲排场，比阔气，挥金如土，肆意浪费，这种做法当然是不对的。我国现在还有几千万贫困地区的人没有解决温饱问题，一些人先富起来了，应该怎么办？今天我所要学习的课文《继续保持艰苦奋斗的作风》，会给我们一个圆满的答案。

（其二）

　　上海有一个腐败分子原是某地石化总厂腈纶厂供销科科长。他见利忘义，利用手中掌握分配原料的权力收受贿赂，成了一个犯罪分子。

　　检查机关在这个罪犯家搜查时，从一大堆赃物中发现一个不小的铁桶。打开一看，里面有足足几十斤重的黄花菜，由于霉菌侵蚀，黄花菜已长出了绿茸茸的细毛，有的开始霉烂了，盖子一开，一阵难闻的气味使每个人掩鼻。

　　黄花菜为什么发霉？一桶发霉的黄花菜又能给人们怎样的启发？我们来读读毛主席三十多年前写的《继续保持艰苦奋斗的作风》，再回过头来想想这些问题。
