松鼠

（其一）

　　我们初一上学期学的五篇说明文以说明群体建筑为主，本学期第三单元学的五篇说明文以说明园林、桥梁、军事工程等建筑艺术为主，接下来第四单元要学习的一组说明文以说明什么为主？对，从课文标题可以看出，是以说明动物的特点为主，比如这一篇《松鼠》。说明的对象变了，说明的方法、语言和顺序往往也不同。让我们来研究"单元教材支配表"，看看学习的重点是什么。

（其二）

　　说明文一般根据说明对象的特点来安排说明的顺序，我们已经学习了四种说明顺序：以空间顺序为主的，如《人民英雄永垂不朽》中的浮雕介绍；以时间顺序为主的，如《中国石拱桥》中说明中国石拱桥的历史发展部分；以程序顺序为主的，如《活板》中活板印刷全过程的介绍；以逻辑顺序为主的，如《苏州园林》先总后分的安排。今天学习的《松鼠》一文是按照什么样的顺序来说明松鼠特点的？请同学们在个人预习、小组讨论的基础上作出正确的回答。
