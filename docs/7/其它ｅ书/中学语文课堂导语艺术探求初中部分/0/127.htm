牛郎织女

　　我们曾经学过《两棵奇树》，那是一篇民间故事。民间故事的作者不是某一个人，而是人民大众。它是集体创作。民间故事长期流传于民间，不断得到补充修改，内容逐渐丰富起来，故事往往生动、曲折，引人入胜，深受人民喜爱。我国民间故事很多，如《白蛇传》、《阿诗玛》、《阿凡提》等，大多反映劳动人民的生活和愿望，歌颂了劳动人民的劳动和斗争。民间故事是祖国的一份宝贵的文学遗产。

　　牛郎织女这个故事中告诉我们，牛郎由放牛得名，织女由织得一手好彩锦得名。牛郎和织女，一个在人间，一个在天上，却结为夫妻。他们男耕女织，靠双手劳动，过着幸福美满的生活。但是代表恶势力的王母娘娘拆散了他们的家庭，破坏他们的幸福，只允许他们每年七月七日在鹊桥上相会一次。

　　从天文学上讲，牛郎织女是两颗星的名称，它们之间相距十分遥远，隔着天河，两颗星只能在天河两边遥遥相望。据天文学家计算，这两颗星星之间，通一次电话也需要十四年！可见，鹊桥相会，只是人们美好的愿望罢了。
