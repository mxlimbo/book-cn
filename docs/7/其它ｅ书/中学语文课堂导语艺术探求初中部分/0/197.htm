"偃旗息鼓"与"圆满结束"

　　杨庆奎老师的设计：

　　同学们，我们说话和写文章都是为了表情达意的。有时，为了增强语言的表达效果，每每要苦思冥想，甚至还要引经据典，寻求最好字句，借以增加辞彩。这样固然是好的，它不仅锻炼了同学们遣词造句的能力，而且，大多数情况下，确也能使我们的语言增色不少。而就这一点来说，我们的先辈之中也不乏其人。杜甫就自誉为"寻章摘句老雕虫"；贾岛、王安石则以善"推敲"字句，名垂千古。但是，如果一味地去生搬硬套，而不注意词语的蕴含、词语的环境及其感情色彩等，那么，往往会导致弄巧成拙、适得其反的效果，有时，还可能闹出笑话。今天我们学习的这篇短文——《"偃旗息鼓"与"圆满结束"》就是一篇针对某些错误的语言现象进行评析的议论文。
