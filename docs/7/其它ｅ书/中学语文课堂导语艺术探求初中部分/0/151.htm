蔬菜的药用

同学们都知道，蔬菜是人们日常生活中不可缺少的朋友。我们常常听别人说这是什么菜、那是什么菜，但不一定会写出来。要是不相信，我现在说一种蔬菜名字，看大家会不会写。这种菜叫"jiu菜"。（同学试写，互评）这个"韭"字是个象形字，下面一横代表地面，中间两竖和三小横就表示它的叶子向上生长、向四处开张的状态，说明这种植物是丛生的。据研究，韭菜是我国古老的蔬菜，它的栽培历史至少有三千多年。品尝鲜嫩的新韭菜，可以说是一种享受，像唐代诗人杜甫就写过"夜雨剪新韭，新炊间黄粱"的佳句。说到这里，我要进一步提个问题：你能说说韭菜或其他蔬菜的保健、防病、治病的作用吗？今天，我们就请同学们自己来研读一篇专门介绍这方面知识的文章——《蔬菜的药用》。
