潘虎

　　同学们都看过《水浒》，里面有两个写得非常成功的人物——李逵和鲁达，他们是《水浒》起义英雄中突出的代表。李逵为救宋江反上梁山，具有强烈的革命要求和彻底、坚定的革命神，他爱憎分明，既粗鲁爽直，也纯朴天真。鲁提辖为救金老父女，路见不平，拔刀相助，三拳打死镇关西；他大闹野猪林，救下林冲，使高大尉的阴谋不能得逞。他对社会上的压迫和不平怀有强烈的愤懑，对各种罪恶势力主动挑战。和李逵一样，他既粗鲁爽直，也单纯朴实，但下级军官的生活经历，又使他具有一定的斗争经验，粗中有细，勇而有谋。今天，我们一起来学习一篇课文，这篇课文成功地塑造了中国民主革时期一位李逵、鲁达式的农民英雄。当然，他比李逵、鲁达更高，因为他热烈追求革命真理，主动要求党的领导。下面我们打开课本，一起来学习《潘虎》这篇课文。
