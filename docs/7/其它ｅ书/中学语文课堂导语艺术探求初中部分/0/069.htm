花市

　　同学们，讲课之前，先思考两个问题。先想一想：现在，我们在市场上看到卖花、鸟、鱼、虫的越来越多了，这反映了什么？（学生议论）

　　有的同学会说："下海"的人多了，做这样买卖的也就相应地多了。这只说对了一方面。另一方面，做生意是为了赚钱，要赚钱，就必须得有人买你的东西。现在做这样生意的人多了，反过来也说明买花、鸟、鱼、虫的人多了，这又反映了怎样的社会现实？

　　再想一想：你在市场上见过有人要出高价买东西，而卖东西的人偏要低价出售的情况吗？你可能见过。比如双方是朋友，认识或"关系户"……但是你见过买卖双方既不是朋友，也不认识，更不是"关系户"，而在他们之间发生了这样的怪事吗？这又反映了卖东西的人怎样的精神品质呢？下面，同学们把书翻到《花市》一课，让我们试在这个耐人寻味的故事中来寻找一下答案。
