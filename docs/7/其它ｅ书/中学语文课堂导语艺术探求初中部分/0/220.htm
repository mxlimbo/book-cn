怎样丢掉学生腔

　　有这样一个人，他不分对象、不论场合，总是以不离口的"之乎者也"来显示自己的身份与众不同。记得他在给小孩们分茴香豆时，他怕孩子们把豆子要光，于是用手罩着碟子说"多乎哉，不多也"。这就是鲁迅先生笔下的孔乙已。鲁迅先生通过这一言行刻画，生动表现了这个中毒很深的迂腐书生的形象。而在我们今天的生活中，咬文嚼字、故作深沉，找些深奥华丽的词藻来装饰自己的，也还大有人在。乍一看，或一听，倒也不错，仔细推敲，则华而不实。针对这种习气，我国著名作家老舍先生，写下了《怎样丢掉学生腔》这篇文艺短文。学习这篇课文，对于我们正在学习语言的同学是很有帮助的，可以帮助我们正确使用语言，丢掉不该有的"学生腔"。
