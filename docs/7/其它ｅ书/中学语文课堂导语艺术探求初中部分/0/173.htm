挖荠菜

（其一）

　　同学们知道，荠菜是一种一年生或多年生草本植物，作为菜蔬，它鲜美可口，有丰富的营养价值和一定的药用价值，身在农村的同学小时候大概还挎着篮子挑过荠菜呢！现在请你思考一下，如果以荠菜作为题材，"挖荠菜"作为标题写一篇散文，你将怎么写？（讨论三分钟）下面，再来看看张洁写的《挖荠菜》一文，又是怎样组织材料、写出深意的。

（其二）

　　前一课我们学了散文《荔枝蜜》，它是以作者感情变化为线索的。我们知道，散文的线索可以是事，是物，可以是时间的推移或空间的转换，也可以是作者感情的变化发展或观察认识的过程。张洁的散文《挖荠菜》我们预习过了，这篇课文的线索是什么？有的说是荠菜，有的说是挖荠菜，有的说是时间推移，也有的说是作者对荠菜的特殊感情，究竟哪一种比较恰当？请同学们在仔细阅读的基础上自己来解决这个问题。

（其三）

　　 "挖荠菜"，看题目中的"荠菜"二字，大家都明白，它是植物，加上一个"挖"字，组成了动宾词组，表示的是所做的一件事。荠菜，是一种鲜美可口的菜蔬，大多野生，现已有人工培植。以"挖荠菜"为题，可以写些什么呢？读了这一课，大家可能会获得原来意想不到的收获。
