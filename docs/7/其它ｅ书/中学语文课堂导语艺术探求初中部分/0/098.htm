口技

（其一）

　　同学们，你们欣赏过口技表演吗？舞台上只有一两个演员，但他们能模仿各种声音：鸟语、虫鸣，火车隆隆、汽笛声声，矿山里飞转的风镐、草原上疾驰的马群，绘声绘色，引人入胜。它使人如身临其境，是一种很高的艺术享受。这一种杂技，在我国明、清两代就比较流行，那时，一些笔记中也有这一类表演的记述。我们课本中就选了清代人林嗣环写的这一篇《口技》（板书），很受中学生的喜爱，现在我们就来学习这篇课文。

（其二）

　　大家听过口技没有？口技是一项杂技节目，演员运用口腔发声来模仿各种各样的声音，比如虫、鸟、走兽的各种鸣叫声、枪炮大作声、火车启动声等等。这篇课文《口技》，（板书）就是描写四百年前的一次口技表演的。现在演口技，演员站在台上，面向观众，有时一个人，有时两个人互相配合，同时还有适当的动作，以加强真实感。四百年前的口技叫做"隔壁戏"，演员隐在布幔或屏风的后面，而且往往是一个人进行表演。口技是很好听的，很不好写，因为口技模仿的是声音，声音看不见、摸不着，写起来比较困难。但《口技》这篇课文把表演者模仿各种声音的精湛技艺写出来了，写得那么妙。现在让我们来看看作者是怎样来描写的。
