狼

（其一）

　　课文《狼》和《两小儿辩日》、《乐羊子妻》一样，三篇短文都通过叙述一个小故事来阐明一定的道理。但是《狼》的故事情节发展急遽，更加引人入胜；它所要阐明的道理是由作者在篇末的评论中点明的。现在我们就来学习《狼》这篇课文，体会它的这些特色。

（其二）

　　河北王佐斌李淑敏老师这样导入：同学们都知道，昔日东郭先生遇见了危难中的中山狼，由于他过分"仁慈"显些丧命，幸亏农夫的帮助，才转危为安。今日屠夫又遇见了中山狼的"后代"，（学生大笑），那么屠夫是否从东郭先生身上汲取了教训，或又重蹈旧辙？狼也许变善良了？那么究竟如何，现在我们就来学习《狼》这篇课文。

（其三）

　　江苏唐惠忠老师教学本文，先提问："有关狼的成语、俗语和故事有哪些？"学生略微思考后，纷纷举手口答："狼吞虎咽、狼心狗肺、狼子野心、狼外婆、东郭先生……""由此可见狼的本性是什么呢？""贪婪、凶狠、狡诈。"学生回答得很顺利。"好，让我们一起看《聊斋志异》中的两只狼……"精彩的导入，如磁铁一样吸引了学生，成为教学乐章的优美序曲。
