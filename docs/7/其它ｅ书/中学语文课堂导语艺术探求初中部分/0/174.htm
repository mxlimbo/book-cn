万紫千红的花

　　人们每见万紫干红的花的场景，总会不约而同地发出"啊，多美的花"的赞叹声；每年春节，个个争相买花，旨在取其好兆头——花开富贵；外宾下机，让天真烂漫的孩子迎上去献给贵宾的是一把五彩缤纷的鲜花；到医院看望病者，为祝愿友人早日康复的又是一束鲜艳夺目的花；在公园里漫游，见到那亭亭玉立的少女，你会自然而然地暗道：姑娘好比一枝花。随着社会的进步，人类的文明，世人也越来越喜欢花。当今中国，改革开放，居民农友，生活好转，庭前院后，阳台窗前，无不栽花。

　　人人喜欢花。那怕是刚刚牙牙学语的孩子，只要一见到花，亦会即举小手，直指花朵，连连叫道："花花，花花！"这正如作者在《万紫千红的花》一文中的结尾所说的，"恐怕世界上没有人不喜欢花的。人们用万紫千红的花来点缀生活环境，用它的形象来装饰服装和用具，把它作为美、纯洁和幸福的象征。"

　　作者仇春霖撰写的千字文——《万紫千红的花》，的确把我们带进了繁花的国度，可以让我们更好地了解花。
