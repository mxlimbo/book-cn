写在第11亿位国民出生前夕

　　1995年2月15日，我国人口数已经达到12亿。今天我们所要学的课文，写于1989年4月中旬。从1989年到1995年，6年时间，人口增加了1亿。我国自80年代开始推行计划生育，经过顽强努力，人口过快增长的势头已经得到有效控制，使"12亿人口日"的到来推迟了9 年。但是喜中有忧，且忧大于喜，因为我国人口基数和每年出生人口的绝对数都很大，所以人口形势仍不容乐观，本文宣传了控制人口、计划生育的必要性，今天我们来学习这篇课文有重要的现实意义。
