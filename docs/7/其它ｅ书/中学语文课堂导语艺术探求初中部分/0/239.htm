山峡

（其一）

　　在我们伟大的祖国广阔的土地上，到处都有秀美的山川、壮丽的景色，古往今来，吸引多少英雄豪杰，多少迁客骚人为之倾倒，多少文人墨客为之挥毫，来描绘祖国山河之美,抒发对大自然的热爱之情啊！郦道元的《三峡》就给我们展示了长江三峡四时雄奇秀丽的景色，抒发了作者对祖国山河的赞美之情。让我们随着作者的指点去三峡一游吧。

（其二）

　　有位教师这样设计导语：

　　今天，我们学习《三峡》，请同学们先看这幅地图（挂出绘有三峡的地形图和彩色照片），三峡是我们祖国万里长江中游的一段，它西起四川奉节县的白帝城，东到湖北宜昌的南津关，全长二百零三公里。三峡，是瞿塘峡、巫峡和西陵峡的总称。大家想一想，为什长江在这一段称为 "峡"呢？

　　大家看这个"峡"字，是"山"和"夹"相合而成。再看这幅由高空拍摄的照片，两山相夹，一水流过，形象地给我们说明了"峡"的概念。三峡可真是山高峡窄，水深流急，好一派"万山磅礴一水通，惊涛轰鸣气势雄"的壮丽景象，好一幅优美的山水画卷。我们虽然不能到三峡一游，但读一读《三峡》这篇文章，也能领略一下壮丽三峡的自然风光，得到"江山如此多娇"的美的享受，从而激起我们对祖国更加热爱的感情。请大家把书翻到——
