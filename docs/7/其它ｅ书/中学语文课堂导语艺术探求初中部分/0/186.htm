想和做

　　美国的宇宙飞船飞奔月球，对月球背面的地形作了详细的考察，画出了精确的月球图，还对月球上的山蜂一一进行了命名。为了纪念一个名叫"万户"的人，其中的一座山峰就叫"万户山"。

　　万户是哪一国的科学家？有什么杰出的贡献？他不是别国人，就是我们中国人，是我国元朝的一个军官。他爱好天文，经常在晴朗的夜晚仔细观察月亮。渐渐地，他产生了一个念头，决心要做一件划时代的大事：到天上去！实地考察一下月亮和整个宇宙。即使失败了，也为后人积累一点经验。他设计了世界上第一个飞行器，在一只椅子下绑了一组当时打仗攻城用的火箭。一个晴朗的夜晚，皓月当空，他坐在椅子上，叫家里人同时将火箭点着，"轰"的一声，飞行椅升上了天空。可以想见，在当时的科技条件下，这样的想法近乎空想。飞行椅升到半空后，摔了下来，万户成了第一个为宇宙空间探索事业献身的人。

　　万户这样的想法是不是空想？他的这种做法，现在应不应该肯定？为什么欧美的科学家们对他评价这么高，要用他的名字命名月球上的山脉，永远纪念他？《想和做》这篇课文，可能能给我们一点启发。
