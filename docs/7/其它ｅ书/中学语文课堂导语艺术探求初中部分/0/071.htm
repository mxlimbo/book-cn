《还乡梦》自序

　　陈祖文老师这样导入：

　　名画有"一叶知秋"的惊人之作，好的"序"、"跋"也能"窥一斑以见全豹"。读本"序"，对陌生的作者的有关情况，特别是异域怀乡的感情和《还乡梦》诗集的成书及其创作意图，会有哪些了解呢？

　　作者以散文手法写作"序"，它行文自由，写法多样，选材多角度，可说放得很开，但也收得拢。读本"序"，对它的"形散而神聚"的特点又有怎样的体味和掌握呢？

　　本"序"语言准确、优美、生动，仅就修辞而言，它就运用了排比、比喻、反问、用典等手法，这些都和作者的直抒胸意结合起来，读文章就感到激情喷射，久久萦怀。结合平时所学，对这些又有怎样的感受和领悟呢？
