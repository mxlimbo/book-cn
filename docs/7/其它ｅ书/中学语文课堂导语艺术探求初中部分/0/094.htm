看戏

（其一）

　　京剧，是我国传统的戏曲艺术，已有二百年的历史，是祖国文化艺术宝库中璀璨夺目的艺术瑰宝。梅兰芳是我国著名的京剧表演大师，也是享誉世界的艺本家。他8岁学艺，11岁登台，在半个世纪的舞台实践中，形成了"梅派"艺术的独特风格。新中国的建立，更使他焕发了艺术青春。《看戏》所描写的正是梅兰芳先生演出时的动人场面。让我们也来看看这场精彩的戏吧！

（其二）

《看戏》这篇课文主要描写了群众观看京剧表演大师梅兰芳先生演出的场面。同学们，你们谁能说说有关梅兰芳的故事？（一二个同学发言后，老师接着说）说起梅兰芳，如果退回四五十年，在我们中国，很少有人不知道他的名字，用"万人空巷"这个成语来形容那时候人们为了争睹他的风采，或观赏他的表演是再恰当不过的了。这一方面是因为他有高洁的品行和强烈的爱国主义精神，（有个"蓄须明志"的故事，说的是在日本人侵占中国时，梅兰芳坚决不给侵略者演出的事）另一方面则是因为他在京剧表演艺术上，几乎已达到了炉火纯青的境界，他在京剧舞台上塑造的一个个血肉丰满的艺术形象，已经深深地根植于人们的心中。梅兰芳的名字，不仅仅属于他个人，也属于整个中华民族。同学们，你们如果有兴趣，回去问问你们的爷爷奶奶或其他大人，就一定会更加了解梅兰芳是一个怎样的人了。

（其三）

像梅兰芳这样的艺术大师的演出，所造成的轰动场面，同学们也许在生活中或在电视中是不难想见的；真要用文字把它描写出来，这就绝对不是一件容易的事。写这样的题材，既要写出演员精湛的演唱艺术，同时又要写出他的演唱在观众中所引起的轰动。要把这两方面的内容很好地统一在一篇篇幅不大的文章中，除了文字工夫以外，还要有巧妙的构思和非常丰富、形象、生动的联想。《看戏》就是这样的一篇好文章。

（其四）

　　问学生：课文题目是《看戏》，谁在看戏？怎样看戏？为什么要这样看戏？请读课文《看戏》。

（其五）

湖北周远喜老师这样导入：教师：同学们，你们喜欢看京剧吗？

　　学生：（异口同声）不喜欢！

　　教师：（笑）我小时候和你们一样，也不喜欢看京剧。唯一感兴趣的是看演员翻筋头。怎么也领悟不了这门"国粹"的艺术魅力，每当看到老人们听京剧时摇头晃脑作陶醉状，我便疑心他们是故弄玄虚。后来，我偶尔拜读叶君健先生的《看戏》，便尝试着摇头晃脑的品味，渐渐地发现那咿咿呀呀的唱腔也的确还是那么回事。随着文化素养的提高，我也逐渐爱上了京剧。今天我想通过本课的学习，力图改变一下你们的观点。
