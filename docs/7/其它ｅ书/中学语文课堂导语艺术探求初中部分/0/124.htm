陌上桑

　　在记叙文里，经常要描写人物形象，假如在一篇写人的作文中要写一个美貌出众的少女，你用什么方法能让读者和你一样觉得她的确美呢？也许你会说，那还不是多用些形容词直接刻画她的外貌长相，比如，写她乌黑的头发、柳叶的细眉、红红的嘴唇、迷人的笑窝、苗条的腰肢……但是，这样写的效果，读者并不一定会感到这个少女就美。有没有比这种正面呆板的描写技高一筹的写法呢？能不能找到一种能够充分引起读者的想象和联想，通过他们亲眼见过的少女，去补充、丰富你所写美貌少女形象的表现方式呢？我国古代许多优秀作品中刻画人物形象的成功范例，会给我们以启迪的。现在要学的汉乐府诗《陌上桑》中描写的主人公，就是一个美貌的妇女，让我们看看诗中是怎么表现她的美的吧！
