现代"千里眼"

　　在埃及尼罗河畔有闻名世界的金字塔。在这些金字塔里，人们发现了许多秘密，其中有墓道、墓室，有埋藏在这些墓道、墓室里的价值连城的珍贵文物。在考古学家看来，这些金字塔里的秘密已经所剩无几了。但是，前不久，日本科学家利用一种仪器，探测到在金字塔下面隐藏着一条古隧道。这个发现震惊了考古界。因为在过去漫长的岁月里，不知有多少考古学家和地质学家用各种手段，在这里进行过无数次的探测和研究，但都没有发现这个古隧道。这充分说明日本科学家使用的仪器具有很强的探测能力，这种仪器就叫做"地下雷达"，也称为地下"千里眼"。它能透过土和岩石找到埋在地下的物体，并对地下目标进行测距定位。这种雷达的"火眼金睛"由好几部分组成，其中起识别作用的就叫做"传感器"，它运用的是先进的遥感技术。今天我们就来学习一篇介绍遥感技术的说明文——《现代"千里眼"》。
