山的那一边

　　高瑞光老师的设计：

　　同学们，我们伟大的社会主义祖国正在进行"四化"建设，"改革、开放"犹如一位技艺高超的魔术师，它使多少门前的小路变成了康庄大道；它又使多少低棚矮户变成了高楼大厦；它又使多少昔日的"泥腿子"也敢光顾"花市"同"大款"们竞价买花；它还把"榆钱饭"几乎变成了珍馐佳肴！……总之，"改革、开放"创造了"愚化智，穷变富"的奇迹！

　　然而，尽管春风浩荡，"山的那一边"却因崇山峻岭的围困和封锁，依然还是"春风不度玉门关"的穷乡僻壤，那里的一些人依然穷得连裤子也穿不上，那里的人一角钱买一个鸡蛋，还不会算一元钱买几个鸡蛋，就算上识字的人吧，也只能将一千二百记为1000200，这一切多么荒唐又是多么令人心酸啊！

　　80年代后期的一天，山沟里进来一支队伍，住下就不再走了，于是军爱民，民拥军，短短几年间，竟然山乡巨变，如今村民们有淘金的、包工的、广州跑买卖的、科学种田的，还有几户竟出了大学生……如今，无论是"山的那一边"，还是山的这一边，都紧跟时代前进的步伐。

　　同学们．你们一定很想知道解放军在山村怎样播撒现代文明的种子的情形吧，好，让我们一起学习作家陆芸芸写的优秀纪实散文——《山的那一边》。（板书：山的那一边 陆芸芸）
