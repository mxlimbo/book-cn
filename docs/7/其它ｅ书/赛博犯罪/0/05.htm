导言：电子边疆的荒芜西部


　　计算不再只和计算机有关，它决定我们的生存。——尼古拉·尼葛洛庞帝(NicholasNegroponte)麻省理工学院媒体实验室(MITMediaLab)

　　有一个被称作赛博空间(cyberspace)的地方，那是一个与现实世界有些相似的世界，是一个既存在于现实世界，又存在于现实世界之外的无法界定的地方。许多年前人们就开始拜访这个地方了，每当他们拿起话筒与电话线另一端的某人交谈时，每当他们使用自动提款机时，每当他们站在超级市场付款处看着条码识别器扫描过他们的货物时，他们都在与赛博空间连接。也许他们并不知道这个地方有个名称，但它确实有。

　　时间和技术的无情进展已经把越来越多的人带入了赛博空间。我们不仅可以通过电话到达那儿，也可以通过传真机或无线电——还有计算机。实际上，计算机让我们能够在赛博空间做一些我们以前从未能做到的事：建立一个社区。赛博空间社区没有物理上的体积，也没有有形的屏障，但人们去那儿交谈、游戏、工作、交流信息、发布消息并相互影响。他们从世界各地沿着同轴电缆或光缆，在一瞬间到达那儿。

　　你无法步行到赛博空间，但你却能以其他方式抵达那里；你无法触摸赛博空间，但却能居住在那里。实际上，如果你尝试着去触摸赛博空间，你可能会受到电线和线路板的120伏直流电的电击。赛博空间不能用手触摸，这个社区只能通过电话线和计算机键盘来访问。

　　赛博空间的虚拟社区居住着成年人和儿童，男人和女人，白领和蓝领，富人和穷人。你无法在赛博空间里区分出社会阶级，因此它就没有。除非有人自愿告诉你他的个人信息，否则你就无法确知他的种族和宗教信仰；因此，这些东西通常也毫无意义。除非你的系统配备有最先进的高技术设备，否则你就无法看到或听见赛博空间里的其他人。除了上述不同，赛博空间是一个相当典型的"虚假世界"(pseudouniverse)。

　　那儿有做生意的地方，有进行娱乐的场所，有用键盘交谈和工作的地方。那儿有教师、银行家、商人、医生、律师、飞行员、军官、秘书、家庭主妇，以及来自其他你所能想到的任何行当的人士。

　　在赛博空间里，有彼此要好的朋友，也有定期聚会的组织，他们讨论(或诅咒)任何你所能想到的话题；那儿有狂热的爱情，激烈的辩论；那儿有愤怒的呐喊，无害的揶揄，善意的玩笑甚至哭泣；还有邻居、企业和"家庭"。躺在舒适的扶手椅中，你就能在赛博空间经历你日常生活中要做的大多数事情。你可以真实地生活在赛博空间里，体会它的便利与小毛病。但是，同其他任何虚拟的或现实的社会一样，赛博空间并非十全十美。在那里你也会受到伤害。你会遇到威胁、攻击、骚扰、抢劫。在这个新社区里，罪犯们极为隐秘，你看不见，也几乎没有听说过这些人，他们也许一直在暗中从事犯罪活动，等你意识到这一切时，却为时已晚。这个虚拟世界的犯罪同现实社会里的犯罪活动极其类似：行窃、欺诈、勒索、盗用公款、间谍、贿赂、伪造、故意破坏、盗版、走私、恐怖活动、色情活动、恋童癖、假冒、侵犯隐私、侮辱，甚至蓄意谋杀。

　　在赛博空间的虚拟胡同里，你会遭到现代罪犯的伏击：飞客(phreakers)、黑客(hackers)、赛博朋客(cyberpunks)、密码朋客(cypherpunks)指对使用电子密码来加密感兴趣的人，他们以此来增强个人隐私安全，防备独裁、集权的权力机构的专治。——译者注、病毒编写者(viruswriters)、信息战士(informationwarriors)、技术无政府主义者(technoanarchists)和技术强盗(techno-bandits)。这些亡命徒驾驭着计算机键盘进入了虚幻的极度荒芜的西部，在大多情况下，那里几乎没有什么限制，也几乎没有法律。与我们世界里带着9毫米手枪的盗车贼相比，在那个世界里他们的危险性并不逊色。

　　网上的高科技恐怖活动是一类增长最快的犯罪。执法人员无法告诉我们这类犯罪的统计数字，因为它太难捉摸；而且，很多人出于困窘、害怕或仅仅是疏忽，甚至没有报告他们受到的伤害。但是，通过查看相关的统计数字，我们可以判断这个问题的大体情况。

　　全球大约有1.1亿台正在使用的个人计算机。仅在1994年一年，美国就售出了4800万台个人计算机。全美国有33%的家庭拥有个人计算机。毫无疑问，其余的人在工作中也会接触并使用计算机。只要考虑下面一点就够了：美国比例最高的职业是管理支持工作，比如秘书性的、专业性的、行政性的或管理性的。这些职业中大部分都要用到计算机。实际上，一项由微软公司(Microsoft)和IntelliQuest公司美国一家提供调查软件、信息数据库和对高技术及计算机产业进行市场调研的公司。——译者注[ZW)]开展的调查表明，大约有87％的美国人相信，将来所有的工作都会用到计算机。

　　我们面前的世界是一个迅速进入赛博空间的世界。在闪烁的光标和发亮的屏幕后面，我们发现的边疆与我们的先辈们驾着四轮马车驰骋西部时发现的边疆没有什么不同。

　　这块电子边疆是一块陌生的、令人兴奋的、没有边界的土地，它偶尔也有些危险。这片荒芜的西部仍在发展和成长当中。随着它的发展，它将慢慢成熟，但它仍会是一个有一定危险的地方。任何到达那儿的人都必须防备可能来自藏匿在虚拟山丘后的歹徒的威胁。如果你以为自己在这里不会有危险，你就可能成为下一个受害者。我们的先辈意识到了他们面前的危险，他们中的大部分人驯服了那块边疆，并将它变成一块安居乐业的地方。作为电子边疆的移民，我们需要一些往日的常识。是该我们驱动马车，保卫自己免受赛博犯罪的伤害的时候了！
