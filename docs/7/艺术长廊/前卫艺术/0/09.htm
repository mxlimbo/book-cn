行为艺术

 
 

管策《蚊子无过》 1999年

用“波普”式的9幅照片记录“行为”的方式，
表现一个现实生活中真实有趣的生活细节。
 
　　　
  　　　

    　　　　　
　　　　　　 宋永平《新生活》 1999年

　　作者将自己的“行为”与父母的“行为”转换成照片的形式，对将转换成照片的“行为”与父母恋爱、结婚和生育后的老照片――即曾有过的“行为”重新组合，也就是再一次“行为”的方式，这种方式不仅非常明确地反映了一种“赤裸裸”的生活真实和一种不寒而栗的人生感叹，也表现了一种对中国社会生活特定时代的记忆。
