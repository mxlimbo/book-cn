装置艺术


　　　　　　　《替身》吕胜中 日本东京第四届亚洲美术展
　　　　
　 1995年初吕胜中先生应日本东京世田谷美术馆的邀请，为第四届亚洲美术展现场制作装置 作品《替身》。在日滞留时间为3月31日——4月8日，从4月1日开始到4月6日安装完成，这期间吕胜中先生以日记形式不断记录了见闻与观感。《替身》相当成功地引起社会各界的热切关注。
　　透过《替身》,艺术家说出自己的思想：首先，靶子也是一种“替身”，与民间招魂巫术不同的是，靶子是为真正的对同类进行精确伤害而制作，用于进行不懈演练的道具。而剪纸人形是作为精神上的庇护自我或伤害他人的道具。它们叠合在一起，构成人类设计的另一个完整自我。每个人都曾经善良，每个人都曾经残忍。我们其实都投出过弹丸，我们其实都伤痕累累，我们可憎，我们也可怜。人类为自己设置了一种不可逃避的圆满结 局——就是用自己的手干掉自己，让世界毁灭。
