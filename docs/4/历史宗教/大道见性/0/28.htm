第二十二回 德性本质

八十九年九月十五日 庚辰年八月十八日

南海古佛 降

　　南海屏障护众生 古佛着书引迷沉

　　人间因缘不固化 德性本质个自清

　　由净染与垢染之中来言，在于人世间必须常会运用的，是把德性来彰显。故在净染方面来言：必须明晰的正是把德性的本质作净染的动作。乃是对自己在生存中必要有的规律与作为，因此「德性」即是对有情众生，能提供其「上升与超越」的根本因质，让其德性来发挥其所长、其所用，乃在于对德性是一种「净染」的作用。又如何化除了对自己恶性相循的左右，才是让自己在人类世界中，不会阻碍自己灵性的增长。必须明白在德性中，是如何来增长。

　　德性是一切修持的因本。在有修与无修的众生中皆是会如此，即是对人类的作为，提供其向上迈进的作用。必须能以德性来提升自己的所作所为，乃是让自己在德性中如何去耕植。亦在如何把德性来作深切的耕植，即是不使其遭受些许的风吹雨打，即会凋零退却，且丧失原本的初心。乃是让人类在德性中，如何作耕植，必要能明晰「德与恶」两者之作用，因灵性的净染与垢染，皆会存在于偏悖一方，果若无法能引用德性，而只是对自己所行持的功果中来作了别。是在未来中，会有诸多的难以转圜的困难。如此必须要能明白，对德性与恶性两者之染垢净化中去作成长。才是对自己能以德性来作人间的福祉，而不会仅在福报之中，让自己存在于厚福中来造业。

　　德之作用是把自己对品性作超越。因而一个人的思想观念，人品是十分重要。不可仅持外表的秀丽，而评估个人的因质。要了知对自己在人类世界，若仅评估其为人的种种行程，即受到自己对他人一种定向的作为，此乃不可取也。因在人品的素质中，是要能清除其垢秽思想的建构，才是真正性。因各人的人品好坏，不可从其外表来评估，必要从其个己的气质中，是福德善相，或是仅秀丽的外表。尚且有不良的外在，即陋恶的气质，即可由人品的气质，对他人来了知一清二楚才是正确。因在有真修之修子来言，即有一份可亲的善良人品，即是由其外在即可观内心。如此人品的好坏是攸关真实性的深浅所致，必须明白其气质是透过了德性耕植于自身。

　　德性的因质必须具备对自己，是存在于内而显着于外的气质。乃对自己是否有修？是否具备了低声下气的作用？还是仅报个名、挂个份、谓我自己在作修持？此乃是挂羊头卖狗肉、沽名钓誉，亦谓之自己在作修持；此乃一般的修子常会遇到的事项。因在于德是行于内，而显于外的根本气质，不必经由言词即可明晰，且由其内在气质的作用中，来导引形于外表气质的根本性。如此一个真正有修之人，必要能明白自己在做啥事？是否会仅是挂个名份而已。还是早已对自己所行所为，能作一个通盘的整理与暸解，即是对自己气质的蕴藏中，了知经由内在而显现于外在，此乃真实有修之人所应具备的气质。

　　「德是作用、道是体用」。乃在于是把德与道来融合，于自己的思想观念。不可存有在德之中，仅是作福的造因者而已。必要是把德性来彰显，才是对自己未来能具足，必在福与德之中作冥想，因此德是比福来得受用。因众生要造福是相当容易，在布施中即可得到福报。但在布施中不一定会是具足德性，而是具备了慈悲的作用而已，因此要了知慈悲是拔苦，而德性是无私无为的作用关系。如此要明晰德之作用，乃在于无私无为之中，让自己去成长。才不会仅在人类世界中有福而无德也，此乃当今众生所应具备的心态，与作为才是正确。如此德之具足是以无私无为来累积，然而德之因质是把自己能利于苍生、利于大众、利于自己，才是真实的德性。否则自己仅具备了福份，而不具备德性，即难以在未来中，具足了自己上升与超越的因本，更在对自己如何作到无私又无为，必要是把德性来彰显，才是以德来培植福份，更是长远也！

　　「崇心」所教导的课程，是要将智慧来提升，在此当下来作准备，而在未来可以如理而实际。亦不会仅在祈福中而无德，必须明白在自己所行所为之中，是要具备相当深厚的智慧累积，才能对德性深植的作用，去完现于福报的来临，因此在德之净染过程，「崇心」所要求的正是如此。不可偏悖于德性之拓展，因而要以德而服人，才能在未来的拓展，可以让众生有依循规范。德之展现不仅在于无私无为，更且是利于个己之孝道，让自己亦能以孝而服天下。因而德性亦是把孝道来彰显，有孝之子孙，其德性即高超，如此德性因质之本体，必须是对自己生存利益，能分别「作用」与「体用」两者，乃是道德者所应具备的。

　　「德是用、道是体」。不可仅在于体用中来混杂，要了知对自己的德性深植中，是必要自己对上求资粮，先具备了丰盈的果实，即能应用的如理实际，更且广泛无垠。亦是让德性来彰显于自己，更是对自己德性因质的净染中作成长，才可谓之「我在修持」，反之仅是挂个名份，那一点都没有改造，何来谓修持之人否？见笑了！见笑了！或是对自己生存要改造思想；思想的改造，让自己在当下可以作真正的主人，而不会沦于为生存的奴隶。

　　德性因质作改造 深植本性即更好

　　因质本源靠己成 人类福祉德性高

　　净染垢染皆两边 德性无私无为延

　　成就超越靠己为 功果确立德妙先

　　此偈同有情众生来共勉之！在自己德性中作成长，把恶性因质来化除，即在于无私无为中来作建构与实现。
