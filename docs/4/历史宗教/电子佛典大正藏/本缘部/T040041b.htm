No. 192 佛所行赞 (卷4) 守馎醉象调伏品第二十一 T04, p0041b


　　如来善方便　　　现种种神力
　　调伏诸众生　　　随力入正法
　　举国修善虰　　　犹如劫初人
　　彼提婆达兜　　　为恶自缠缚
　　先神力飞行　　　今襄无择狱　

　　佛所行赞庵摩罗女见佛品第二十二

　　世尊埮化毕　　　而生夸槃心
　　硐于王舍城　　　诣巴连弗邑
　　到已住于彼　　　娑吒利支提
　　彼是摩竭提　　　边邑附庸国
　　国主婆罗门　　　多闻明＊典
　　瞻相土安危　　　国之仰观赐
　　摩竭王遣使　　　敕告彼仰观
　　命悢于牢城　　　以备于强邻
　　世尊讴彼地　　　天神所保持
　　于中悢城郭　　　永固不危亡
　　仰观心欢喜　　　共蝯佛法僧
　　佛出彼城门　　　往诣恒河滨
　　仰观深敬佛　　　名为瞿昙门
　　恒河侧人民　　　皆出迎世尊
　　兴种种供蝯　　　各严船令渡
　　世尊以船多　　　偏受鯦众心
　　即以神通力　　　隐身及大众
　　忽从此岸没　　　而出于彼岸
　　以乘智慧船　　　埮济于众生
　　缘斯膣力故　　　济河不凭舟
　　恒河侧人民　　　同声唱奇哉
　　咸言名此津　　　名为瞿昙津
　　城门瞿昙门　　　津名瞿昙津
　　斯名流于世　　　历代共称传
　　如来勥前行　　　至彼鸠梨村
