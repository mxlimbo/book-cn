第四节 菜单详解 (2)

作 者 : Kooler

　　（ 3 ）“ HTML ”设置

　　要使将你的 MOVIE 在网上发布，就必须有 HTML 来支持和控制 MOVIE ， FLASH4 在这里提供根据你的设定自动生成 HTML 的功能，你设定完以后按一下 PUBLISH 就能生成相应代码（呵呵，傻瓜型的）。

　　在此处修改的设置均是位于 OBJECT 和 EMBED 标签内的参数。 OBJECT 标签是用于在 Internet Explorer 浏览器中打开 FLASH MOVIE 的； EMBED 则是用于 Netscape 浏览器的。相关设定选项如下：

　　 Template 指定使用的模板。 Flash 程序所附带的所有的模板均会显示在下拉菜单中。你可以选定下拉菜单中的模板再点击右边的 Info 按钮就可以了解各模板的介绍。如果你未指定选用模板， Flash4 会自动选用名字为 Default.html ； Dimensions 用来设定在 OBJECT 和 EMBED 标签之间的宽度（ WIDTH ）和高度（ HEIGHT ）属性值。这个设置在弹出菜单中一共有三个选项：
　　Match Movie ：设定尺寸和 MOVIE 的大小相同，这是缺省设置。
　　Pixels ：允许你在 Width 和 Height 栏中填上像素数。
　　Percent ：允许你设定 MOVIE 相对于浏览器窗的大小百分比。将百分比值填到 Width 和 Height 栏中。

　　 Playback ：给位于 OBJECT 和 EMBED 标签之间的 LOOP ， PLAY ， MENU ，和 DEVICE FONT 参数设定参数值。

　　 Paused at Start ：设定 PLAY 的参数值为 FALSE ，暂停 MOVIE 播放，直到访问者或者点击 MOVIE 中的播放按钮，或者选择快捷菜单中的 Play 开始播放为止。“ Paused at Start ”在默认状态下是关闭的，所以一般 MOVIE 一旦载入就会立即开始播放。
　　 Loop ：设定 LOOP 的参数值为 TRUE ，当 MOVIE 播放到最后一帧时重复播放 MOVIE 。如果没有选这个选项， MOVIE 在播放到最后一帧时就会停止播放。 Loop 在默认状态下是打开的。
　　Display Menu ：设定 MENU 的参数值为 TRUE ，设定一个当用户鼠标右击（ Windows ）或者按住 Command 键点击（ Macintosh ） MOVIE 时可供使用的快捷菜单， Display Menu 在默认状态下是打开的。如果你希望按右键后菜单中只有 About Flash4 一个选项的话，就不要选择本选项。
　　Device Font(Windows only) ：设定 DEVICE FONT 的参数值为 TRUE ，使用经过消除锯齿处理的系统字体替换那些用户系统中未安装的字体。 Device Font 在默认状态下是打开的。

　　 Quality 设定位于 OBJECT 和 EMBED 标签之间的 QUALITY 参数值，你可以根据你对你的 MOVIE 质量的要求进行选择，决定消除图像锯齿的要求，你可以根据下拉选单进行选择你是需要质量还是播发流畅度，有关解释选项如下：
　　Low ：这个选项令播放流畅度优先于所表现的图像质量。选择此选项，消除锯齿功能始终不工作。
　　Autolow ：优先考虑播放流畅度，但是只要有可能也要改善图像质量。选择这个选项后，当开始播放时，消除锯齿功能是关闭的。如果 Flash 检测到处理器可以应付得过来，则将消除锯齿功能打开。
　　Autohigh ：强调播放速度和表现性能同等重要，但是如果播放速度需要，则牺牲所表现的图像质量（哎，很难两全其美啊）。选择这个选项后，当开始播放时，消除锯齿功能是打开的。假如实际的帧读取速度低于指定值时，消除锯齿功能就会关闭，以改善播放速度。我们可以用这个设置来模仿 View>Antialias 命令。
　　High ：这个选项令表现性能优先于播放速度。选择这个选项，消除锯齿功能始终处于工作状态。如果 MOVIE 中没有动画，则位图被进行平滑处理；如果 MOVIE 中有动画，则位图不被进行平滑处理。对于 Quality 的设置而言，这个选项是缺省选项。
　　Best ：在不考虑播放速度的前提下提供最佳显示质量。所有的输出都经过消除锯齿处理，所有的位图都进行平滑处理。

　　 Window Mode(Windows only) 设定 OBJECT 标签之间的 WMODE 参数，这个参数允许你充分利用 Internet Explorer 4.0 中的透明 MOVIE ，绝对位置，分层等功能。这个选项仅用于带有 Flash Active X 控制功能的 Internet Explorer 的 Windows 版本；

　　Window ：将 WMODE 参数值设为 WINDOW ，然后在网页中自己的矩形窗口中播放 MOVIE 。这一设置通常可以提供最快速的动画播放性能。 Opaque Windowless ：设定 WMODE 参数值为 OPAQUE 。如果你要在 MOVIE 背后移动元素，同时又不想这些元素表露出来，就可以使用这个选项。
　　Transparent Windowless ：设定 WMODE 的参数值为 TRANSPARENT 。这个选项可以将嵌有 MOVIE 的网页的背景透过 MOVIE 透明部分显露出来。而当你使用了这个选项后，动画的运行可能会慢下来。

　　 Alignment 选项可以确定 MOVIE 在浏览器窗口中的位置，相关下拉选项如下：
　　Default ：将 MOVIE 位于浏览器窗口的中央，如果浏览器窗口小于 MOVIE
　　窗口，则对 MOVIE 的边缘进行剪切。
　　Left ：将 MOVIE 位于浏览器窗口的左边，如果需要，剪切 MOVIE 的上下和右边部分。
　　Right ：将 MOVIE 位于浏览器窗口的右边，如果需要，剪切 MOVIE 的上下和左边部分。
　　Top ：将 MOVIE 位于浏览器窗口的最上边，如果需要，剪切 MOVIE 的左右和下边部分。
　　Bottom ：将 MOVIE 位于浏览器窗口的最下边，如果需要，剪切 MOVIE 的左右和上边部分。
