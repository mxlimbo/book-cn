<A NAME="126"><h1>Lesson 2: Automating the Installation of Windows 2000 Server</h1></A>

<p>Automated installations of Windows 2000 Server involve running Setup with an answer file. You can perform automated installations on multiple computers so that Setup can take place in an unattended fashion. The following installations can be automated:</p>

<ul>
<p><li>The core Windows 2000 Server operating system</li></p>

<p><li>Any application that does not run as a service</li></p>

<p><li>Additional language support for Windows 2000 Server through the installation of various language packs</li></p>

<p><li>Service packs for Windows 2000 Server</li></p>
</ul>

<p>This lesson focuses on unattended installations of the Windows 2000 Server operating system. To learn about unattended installations of other applications, see Lesson 3, &quot;<A HREF="ch03d.htm#136">Automating the Installation of Server Applications</A>.&quot;</p>
