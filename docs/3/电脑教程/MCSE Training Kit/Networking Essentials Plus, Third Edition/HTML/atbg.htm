Getting Started

This self-paced training course contains demonstration videos that enhance and supplement the text. The following sections discuss the hardware and software required to view the demonstration videos on the companion CD-ROM.

Hardware Requirements

Each computer must have the following minimum configuration. All hardware should be on the Microsoft Windows Hardware Compatibility List.

  • Pentium 90 MHz processor

  • 16 MB RAM

  • 16-color display card

  • 16-bit sound card

  • VGA or better monitor

  • CD-ROM drive

  • Mouse or other pointing device (recommended)

Software Requirements

The following software is required to view the demonstration videos in this course.

  • Microsoft Windows 95 or later; Microsoft Windows NT 4.0, Service Pack 3, or later

  • Standard multimedia player, such as Windows Media Player (which is included on the companion CD)

  • Microsoft Internet Explorer 4.01 or later (Internet Explorer 5) is included on the companion CD.
