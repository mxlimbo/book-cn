Chapter 14

Using Remote Installation Services

About This Chapter

This chapter helps you to understand the tools available to help you efficiently deploy Microsoft Windows 2000. You can use the Windows 2000 Remote Installation Services to deploy Windows 2000 Professional. You can also create distribution servers to speed up installing any of the Windows 2000 family of products.

Before You Begin

To complete this chapter, you must have:

  • A computer that meets or exceeds the minimum hardware requirements listed in "Hardware Requirements."

  • A drive on your computer formatted with Windows 2000 NTFS with enough space available to hold all the Windows 2000 Professional installation files; approximately 300 MB of space is required.

  • A Windows 2000 Server CD-ROM.

  • A Windows 2000 Professional CD-ROM or access to a shared folder containing the Windows 2000 Professional installation files.

  • A 3.5 inch formatted floppy.
