<A NAME="431"><H1>Lesson 1: Understanding the New Authentication Protocols in Windows 2000</H1></A>

<p>Microsoft Windows NT 4.0 included support for several authentication
protocols used to verify the credentials of users connecting to the
network. These protocols included the following:</p>

<ul>
<p><li>Password Authentication Protocol (PAP)</li></p>

<p><li>Challenge Handshake Authentication Protocol (CHAP)</li></p>

<p><li>Microsoft Challenge Handshake Authentication Protocol
(MS-CHAP)</li></p>

<p><li>Shiva Password Authentication Protocol (SPAP)</li></p>

<p><li>Point-to-Point Tunneling Protocol (PPTP)</li></p>
</ul>


<p>Windows 2000 includes support for these and several additional
protocols that significantly increase your authentication, encryption,
and multilinking options. The new protocols supported by Windows 2000
include Extensible Authentication Protocol (EAP), Remote Authentication
Dial-In User Service (RADIUS), Internet Protocol Security (IPSec),
Layer-Two Tunneling Protocol (L2TP), Bandwidth Allocation Protocol (BAP)), and Microsoft Challenge
Handshake Authentication Protocol version 2 (MS-CHAP v2).</p>
