<A NAME="26"><H1>Lesson 2: Windows 2000 Professional</H1></A>

<p>Windows 2000 Professional is easier to use and manage and provides
greater compatibility, file management capabilities, and security than
previous versions of Windows. This lesson discusses how Windows 2000
Professional improves the capabilities of previous versions of Windows in five areas: ease of
use, simplified management, increased hardware support, enhanced file
management, and enhanced security features.</p>

<p>
<div class="sidebar"><blockquote>
<p><b>After this lesson, you will be able to</b></p>

<ul>
<p><li>Identify features and enhancements in Microsoft Windows 2000
Professional.</li></p>
</ul>

<p><b>Estimated lesson time: 15 minutes</b></p>
</blockquote></div>
</p>

<A NAME="27"><H2>Ease of Use</H2></A>

<p>Windows 2000 Professional includes changes to the look and
functionality of the desktop, windows, and the Start menu, making it
easier to use than previous versions of Windows. Besides these user
interface enhancements, Windows 2000 Professional also contains
features that improve support for mobile users and make printing easier
and more flexible.</p>

<A NAME="28"><H3>User Interface Enhancements</H3></A>

<p>The enhancements and features that improve the Windows 2000
Professional user interface include the following:</p>

<ul>
<p><li><b>Customized Start menu.</b> Personalized Menus can be
activated to keep track of the programs you use and to update the
Programs menu so that it presents only the programs that you use most
often. Applications that you use less frequently are hidden from normal
view, making the Start menu easier to use.</li></p>
</ul>


<p>
<div class="note"><blockquote><b>NOTE</b><hr>
 To activate Personalized Menus, click Start,
point to Settings, and then click Taskbar &amp; Start Menu. Click the
General tab and select Use Personalized Menus.
</blockquote></div>
</p>

<ul>
<p><li><b>Logon and shutdown dialog boxes.</b> Logon and shutdown
dialog boxes are easier to use with fewer, better organized
choices.</li></p>

<p><li><b>Task Scheduler.</b> The enhanced Task Scheduler allows
users to schedule scripts and programs to run at specific times.</li></p>
</ul>

<A NAME="29"><H3>Support for Mobile Users</H3></A>

<p>Windows 2000 Professional supports the latest laptop technologies
based on the Advanced Configuration and Power Interface (ACPI), which
allows you to change or remove devices without turning off the
computer. ACPI also lengthens battery life with power management and
suspend or resume capabilities.</p>

<p>Features in Windows 2000 Professional that provide support for
mobile users include the following:</p>

<ul>
<p><li><b>Network Connections wizard.</b> Consolidates all of the
processes for creating network connections. Users can now set up the
following networking features from one wizard:</li></p>
<ul>
<p><li>Dial-up connections</li></p>

<p><li>Virtual private networks (VPNs)</li></p>

<p><li>Incoming calls</li></p>

<p><li>Direct connections to another computer</li></p>
</ul>
<p><li><b>Virtual private network (VPN) support.</b> Provides
secure access to corporate networks from off-site locations by using a
local Internet service provider (ISP) rather than using a long
distance, dial-up connection.</li></p>

<p><li><b>Offline folders.</b> Allows you to copy documents that
are stored on the network onto your local computer, making it easier to
access data when you are not connected to the network.</li></p>

<p><li><b>Synchronization Manager.</b> Compares items on the
network to items that you opened or updated while working offline.
Synchronization occurs when you log on, and any changes made offline to
files and folders, Web pages, and e-mail messages are saved to the network.</li></p>
</ul>

<A NAME="30"><H3>Printing Support</H3></A>

<p>Printing in Windows 2000 Professional has been improved to assist
you in providing a more flexible network of printers. Windows 2000
Professional includes the following printing features and
enhancements:</p>

<ul>
<p><li><b>Internet Printing Protocol (IPP).</b> Allows users to
send documents to any printer in a Microsoft Windows 2000 network that
is connected to the Internet. Internet printing enables users to do the
following:</li></p>

<ul>
<p><li>Print to a Uniform Resource Locator (URL) over an intranet
or the Internet.</li></p>

<p><li>View printer and job-related information in Hypertext Markup
Language (HTML) format from any browser.</li></p>

<p><li>Download and install printer drivers over the Internet.</li></p>
</ul>

<p><li><b>Add Printer wizard.</b> Simplifies the process of
connecting to local and network printers from within a program. You no
longer need to open the Printers system folder or specify driver
models, printer languages, or ports when you add printers.</li></p>

<p><li><b>Image Color Management (ICM) 2.0.</b> Allows you to send
high-quality color documents to a printer or another computer with
greater speed and reliability than ever before. ICM 2.0 is an operating
system API that helps ensure that the colors you see on your monitor
match those on your scanner and printer.</li></p>
</ul>
