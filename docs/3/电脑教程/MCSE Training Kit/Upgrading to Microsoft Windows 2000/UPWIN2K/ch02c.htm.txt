<A NAME="85"><H1>Lesson 2: Installing Windows 2000 from a CD-ROM</H1></A>

<p>This lesson looks at the four-stage process of installing Windows
2000 from a CD-ROM. These four stages are as follows: run the Setup
program, run the Setup wizard, install Windows networking, and complete
the Setup program. After you learn about these four stages, you will
install Windows 2000 on your computer.</p>

<p>
<div class="sidebar"><blockquote>
<p><b>After this lesson, you will be able to</b></p>
<ul>
<p><li>Install Windows 2000 Server from a CD-ROM.</li></p>
</ul>
<p><b>Estimated lesson time: 90 minutes</b></p>
</blockquote></div>
</p>

<A NAME="86"><H2>The Windows 2000 Setup Program</H2></A>

<p>Installing Windows 2000 is a four-step process that combines the
Setup program, wizards, and informational screens to complete the
installation.</p>

<p>Installing Windows 2000 from a CD-ROM onto a clean hard disk
consists of these four stages:</p>

<ol>
<p><li>Run the Setup program.</li></p>

<p>The Setup program prepares the hard disk for later stages of
installation and copies the necessary files to run the Setup
wizard.</p>

<p><li>Run the Setup wizard.</li></p>

<p>The Setup wizard requests setup information about the computer,
which includes names, passwords, licensing modes, and so on.</p>

<p><li>Install Windows 2000 networking.</li></p>

<p>After gathering information about the computer, the Setup wizard
prompts you for networking information and then installs the networking
components so that the computer can communicate with other computers on
the network.</p>

<p><li>Complete the Setup program.</li></p>

<p>To complete the installation, Setup copies files to the hard disk
and configures the computer. The system restarts after installation is
complete.</p>
</ol>

<p>Each of these four steps is covered in more detail in the following
sections.</p>

<A NAME="87"><H2>Running the Setup Program</H2></A>

<p>To start Setup, use the Setup boot disks. Insert the disk labeled
Setup Disk 1 into drive A, and then turn on, or restart, the computer.
If your computer supports booting from a CD-ROM drive, you can also
start the installation by using the Windows 2000 CD-ROM.</p>

<p>

For instructions on how to create the Windows
2000 Setup boot disks, see <a href="chaba.htm">Appendix B</a>, &quot;Creating Setup Disks.&quot;
</blockquote></div>
</p>

disk drive (see Figure 2.4):</p>

<ol>
<p><li>After the computer starts, a minimal version of Windows 2000 is
copied into memory.</li></p>

<p><li>This version of Windows 2000 starts the text-mode portion of the
Setup program and prompts you to read and accept a licensing
agreement.</li></p>

<p><li>Setup prompts you to select the partition on which to install
Windows 2000. You can select an existing partition or create a new
partition by using unpartitioned space on the hard disk.</li></p>

<p><li>After you create the installation partition, Setup prompts you
to select a file system for the new partition. Then, Setup formats the
partition with the selected file system.</li></p>

<p><li>After formatting the Windows 2000 partition, Setup copies files
to the hard disk and saves configuration information.</li></p>

<p><li>Setup restarts the computer and then starts the Windows 2000
Server Setup wizard, the graphical-user-interface-(GUI)-mode portion of
Setup. As in Windows NT 4.0, by default the Windows 2000 operating system files
are installed in the C:\Winnt folder.</li></p>
</ol>
