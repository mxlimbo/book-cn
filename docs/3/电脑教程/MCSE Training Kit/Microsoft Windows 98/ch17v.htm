Practice: Using Kiosk Mode

In this practice, you will configure Internet Explorer for kiosk mode, switch to another program in kiosk mode, and close Internet Explorer when it is in kiosk mode.

    To configure Internet Explorer for kiosk mode

    1. Click Start and Run.

    2. Type iexplore.exe -k //www.microsoft.com and press ENTER.

    3. To switch to another program in kiosk mode, press ALT+TAB or CTRL+ESC.

    4. To close Internet Explorer when it is in kiosk mode, press ALT+F4.

After you create a fully navigable Web site with only the pages you want the users to view, and without links to other pages, you can preset Internet Explorer to open with a start page on your site. In this way, end users can browse your site without exploring the rest of the Web.
