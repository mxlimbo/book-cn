Windows 98 Explorer

Windows 98 facilitates navigation between local, network, Internet, and intranet content through a single Explorer (see Figure 7.3).

Figure 7.3 Windows 98 Explorer

Single Explorer

Consolidating Explorer into a single utility streamlines access to information, making it easier and quicker to find and retrieve data. Simple and Consistent Browsing are important features of the single Explorer.

You can browse a local hard disk or a network in the same way that you browse the World Wide Web. Navigation functions, such as Back, Forward, and single-click navigation, used in Web browsers, appear throughout the entire user interface. The single Explorer provides the ability to view multiple types of content in any folder, whether the content consists of files, folders, or HTML. With Explorer, you can view folders as Web pages.

Transparent Browser Access

While you are looking at the contents of your hard disk, you can go to a Web page with a single click, without starting a new application. Because all content can be viewed in the same window, another click returns you to your local content. You can then move between local content and Web pages without having to start a Web browser.

Context-sensitive Menus and Toolbars

The user interface detects the type of information presented, whether it is HTML or local files and folders, and automatically adjusts the toolbar to the type of information. For example, Edit, Search, and Print replace features such as Delete and Properties. The Address bar also supports both Uniform Resource Locators (URLs) and paths, such as C:\Windows.

Seamless Browsing in Global Favorites

With new Global Favorites, you can keep track of any folders, files, or servers along with your favorite Web sites, so that you can seamlessly move between local content and Web content.
