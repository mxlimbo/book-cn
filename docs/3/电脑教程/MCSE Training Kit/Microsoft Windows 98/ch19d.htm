Lesson 1: Troubleshooting Dial-Up Networking

Problems in Dial-Up Networking usually occur in two areas: modems and connections. In addition, if you use Multilink, you may need to troubleshoot problems that occur with that function. This lesson discusses steps you can take to troubleshoot problems with Dial-Up Networking.

After this lesson, you will be able to:

  • Use the Modemlog.txt file and the Modem Diagnostics Program to resolve modem problems.

  • Use the Ppplog.txt file to resolve connection problems.

  • Describe common PPTP connection problems and their solutions.

  • Describe common Multilink connection problems and their solutions.

Estimated lesson time: 15 minutes
