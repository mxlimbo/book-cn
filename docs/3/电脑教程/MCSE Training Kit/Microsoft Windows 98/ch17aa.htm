Installing Personal Web Server

You install Personal Web Server from the add-ons folder on the Windows 98 CD-ROM disc. Before you install Personal Web Server, you must have TCP/IP configured on your computer.

If you want to install Personal Web Server, perform the following procedure:

  1. Click Start, and then click Run.

  2. In the Run dialog box, type D:\Add-ons\Pws\Setup.exe (where D: is the name of your CD-ROM drive).

  3. Follow the instructions on your screen.

  4. When you have finished, restart your computer for the changes to take effect.

By default your home page is C:\Inetpub\wwwroot\default.htm. You can edit this file to meet your needs.

If you want to connect to the home page on Personal Web Server, perform the following procedure:

  1. Click Start, then click Run.

    The Run dialog box appears.

  2. Enter http://localhost and click OK.

    Internet Explorer appears, displaying the home page on your computer.

  3. Close Internet Explorer.
