Lesson 1: The Internet and Intranets

Understanding how to use the Internet efficiently is a necessary skill in an information-intensive environment. This lesson provides an overview of the Internet and intranets. It also provides an overview of Internet Explorer 4.0. In addition, the lesson introduces basic tools and procedures involved in accessing information on the Web, and in filtering Web material.

After this lesson, you will be able to:

  • Configure Internet Explorer 4.0 to match the requirements of your environment.

  • View channel content.

  • Use NetShow to transmit and receive multimedia files.

Estimated lesson time: 45 minutes
