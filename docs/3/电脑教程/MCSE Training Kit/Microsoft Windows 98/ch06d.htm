Lesson 1: Configuring Hardware Settings

This lesson discusses configuring hardware settings in Windows 98. Windows 98 detects and configures most hardware devices, but there may be occasions when you want to customize or change the configuration. When you want to configure Windows 98, you should use Control Panel. Control Panel is a folder containing icons with which you open property sheets to customize the computer's configuration. Each configuration setting updates an internal database called the Registry.

After this lesson, you will be able to:

  • Add new hardware with Control Panel.

  • Create hardware profiles with Control Panel.

  • Configure display properties with Control Panel.

Estimated lesson time: 50 minutes
