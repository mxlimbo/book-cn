Print Subsystem Components

The print subsystem components include:

  • The graphical device interface (GDI) subsystem and its supporting modules such as the printer driver. These components translate printing instructions from applications into data suitable for the printer.

  • Files. These can be in either raw or enhanced metafile (EMF) format.

  • The print spooler. This component has two main functions:

    • To deliver data to printers for printing, either directly or by recording and releasing a hard disk resident spool file containing the print job. The spooler manages the events and format conversion that cause the data to be printed on the designated printer.

    • To manage the printing environment, which consists of printers, printer drivers, print providers, monitors, ports, and print jobs. The spooler maintains registry information for all components in the printer subsystem. The spooler also keeps track of the current status of each component.

  • The print processor. This component converts spooled device-independent data into device-specific data.

Note

For more information about the Print Subsystem, refer to the Microsoft Windows 98 Resource Kit.
