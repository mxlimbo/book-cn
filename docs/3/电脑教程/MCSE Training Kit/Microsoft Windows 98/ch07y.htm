Practice: Using the Subscription Wizard

In this practice you will use the Subscription Wizard to change the properties of a subscription.

    To change subscription properties using the Subscription Wizard

    1. Click Start and point to Settings, then Active Desktop.

    2. Click Customize my Desktop. The Web tab on the Display Properties dialog box appears.

    3. Verify that View my Active Desktop as a web page is selected.

    4. Click New. The New Active Desktop Item dialog box appears.

    5. Click No.

    6. Type www.microsoft.com in the location box.

    7. Click OK. The Add item to Active Desktop dialog box appears.

    8. Click Customize Subscription. The Subscription Wizard appears.

      You can update subscriptions on a daily, weekly, monthly, or customized schedule. You can also update subscriptions manually.

    To update subscriptions manually

    1. Click Start, point to Settings, and then point to Active Desktop.

    2. Click Update Now.
