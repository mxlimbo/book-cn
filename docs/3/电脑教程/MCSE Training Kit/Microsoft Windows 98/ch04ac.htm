Recommended Practices

Here are some recommendations for supporting Windows 98 hardware:

  • Although in some cases Windows 95 and Windows NT can share a single network adapter driver; this is due to each platform's mutual support of Network Driver Interface Specification (NDIS) and should not be confused with the Win32 Driver Model.

  • Windows 98 power management features will work with a legacy BIOS, but the legacy BIOS must implement the power management control.

  • When using USBs, if the configuration includes cascading hubs (one hub plugged into another hub), make sure that USB supports the configuration according to the following guidelines:

    • A bus-powered hub cannot be plugged into another bus-powered hub.

    • Bus-powered hubs can support no more than four downstream ports.

    • Bus-powered hubs cannot support any bus-powered device that draws more than 100 milliamps.

    • Hub cascade depth cannot exceed five tiers.

  • When configuring your Windows 98 computer for multiple displays, you must check that all display adapters meet specific hardware requirements.

  • When using multiple displays, the system BIOS picks the primary VGA display based on the PCI slot order, unless the BIOS offers an option for picking which device is to be treated as the VGA device. When using docking units, you cannot stop the VGA device.
