Managing a Local Printer

You can configure Windows 98 to manage your printing tasks (see Figure 8.3). For example, if the designated printer is out of paper or has a paper jam, you can pause printing until you can resolve the problem. If you sent print jobs to the printer in error, you can purge them. Or perhaps you want to change the order of documents in a queue so that one document prints before another.

Figure 8.3 The Printer window

If you want to manage printing tasks, perform the following procedure:

  1. From the Printers folder, double-click the printer whose operation you want to change.

  2. To pause printing, from the Document menu, click Pause Printing. This will stop all print jobs. Repeat this procedure to resume printing.

  3. To purge all listed print jobs, from the Printer menu, click Purge Printing Documents. All documents will be deleted from the queue.

  4. To pause a single print job, right-click the print job, then click Pause Printing to stop the print job temporarily. Repeat these steps to resume printing.

  5. To cancel a single print job, right-click the print job, then click Cancel Printing to delete the print job.

  6. To change the position of a print job in the queue, drag the job to the desired position in the queue.
