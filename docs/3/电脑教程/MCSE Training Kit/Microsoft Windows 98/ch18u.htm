Using Direct Cable Connection

You can use Direct Cable Connection to establish a direct serial, parallel, or infrared connection quickly and easily between two Windows 98 computers in order to share resources or gain access to a network (see Figure 18.12).

Figure 18.12 Setting up a Direct Cable Connection

Direct Cable Connection establishes a two-computer network including a host computer and guest computer. If the host computer is connected to a network, the guest computer can also have access to that network. To share resources over Direct Cable Connection, you install Direct Cable Connection and share folders and printers in the same way you would with other local or remote network users.

Direct Cable Connection supports the following cable types:

  • Serial null-modem standard (RS-232) cable.

  • Standard, Extended Capabilities Port (ECP), or Universal Cable Module (UCM) parallel cables.

Note

A parallel cable provides faster connections than a serial cable.

Windows 98 Setup does not install Direct Cable Connection during setup, unless you select the custom or portable setup type. You can install Direct Cable Connection at any time after setup.

If you want to install Direct Cable Connection, perform the following procedure:

  1. In Control Panel, double-click Add/Remove Programs, and then click the Windows Setup tab.

  2. In the Components list, click Communications, and then click Details.

  3. Click Direct Cable Connection, then click OK.
