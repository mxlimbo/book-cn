Practice: Implementing Content Ratings

In this practice, you will enable and disable content ratings.

Exercise 1
Enabling Ratings

In this exercise, you will enable ratings in Internet Explorer 4.0.

  • To enable ratings

    1. Start Internet Explorer.

      Internet Explorer appears.

    2. Click OK.

    3. From the View menu, click Internet Options.

      The Internet Options dialog box appears.

    4. Click the Content tab, and then click the Enable button.

      The Supervisor Password Required dialog box appears.

    5. Type your password in the Password box and in the Confirm Password box, then click OK.

      The Content Advisor dialog box appears.

    6. Click on each of the categories and set the rating level for each of the four rating categories to Level 1.

    7. Click OK to close the Content Advisor.

      A dialog box appears indicating that Content Advisor has been installed.

    8. Click OK.

      The Content Advisor message box closes.

    9. Click OK.

      The Internet Options dialog box closes.

Exercise 2
Disabling Ratings

In this exercise, you will disable ratings in Internet Explorer 4.0.

    To disable ratings

    1. Start Internet Explorer.

      Internet Explorer appears.

    2. From the View menu, click Internet Options.

      The Internet Options dialog box appears.

    3. Click the Content tab, and then click the Disable button.

      The Supervisor Password Required dialog box appears.

    4. Type password in the Password box and then click OK.

      The Content Advisor message box appears.

    5. Click OK.

      The Internet Options dialog box appears.

    6. Click OK.

    7. Close Internet Explorer.
