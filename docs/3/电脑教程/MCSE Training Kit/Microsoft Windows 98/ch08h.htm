Bi-directional Communication

With bi-directional parallel communication, the printer driver can query the printer automatically and configure it for a specific print task based on the printer's capabilities and limitations. Printers can also send status messages, such as when paper jams occur or when the printer is low on toner, to Windows 98 and to applications.

To use bi-directional printing, you must have the following:

  • A bi-directional printer.

  • An IEEE 1284-compliant cable.

  • A bi-directional parallel port set to bi-directional or PS/2 mode.

Extended Capabilities Port Support

Windows 98 includes support for Extended Capabilities Ports (ECP) and ECP devices, which provide high-speed printing. If you have an ECP port or device, you can connect either ECP or non-ECP devices to the port. In either case, using an ECP will improve I/O (input/output) performance, which can improve printer performance. Windows 98 configures ports according to their configuration in the BIOS.
