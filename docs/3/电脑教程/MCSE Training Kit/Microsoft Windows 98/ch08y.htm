Summary

The following information summarizes the key points in this chapter:

The Windows 98 Print Subsystem

  • Windows 98 implements printing using a 32-bit print subsystem that includes a multithread, preemptive spooler architecture. Subsystem components include the graphical device interface, raw or EMF files, the print spooler, and the print processor.

  • Some printers support bi-directional parallel communications, in which the printer driver can query the printer automatically and configure it based on the printer's capabilities and limitations.

  • Deferred printing enables you to store a print job on your computer's local hard disk. You can print the document later without opening the application.

Installing a Local Printer

  • There are two ways to install local printers in Windows 98: automatically through Plug and Play detection, or by using the Add Printer Wizard.

Configuring and Managing a Local Printer

  • Use the printer's Properties tab sheet to set printer options, including port settings, transmission retry, and device driver options.

  • To manage printing tasks, such as pausing or canceling print jobs, double-click on the printer whose operation you want to change.
