Lesson 2: Configuring System Settings

This discusses how to configure system settings in Windows 98. You can configure Windows 98 system settings to optimize your computer's use of virtual memory, hard disk cache, and CD-ROM file system cache through the System icon in Control Panel. You can also adjust the options for floppy disks and removable disks.

Users with special needs can benefit from system configurations that are adjusted to compensate for vision, hearing, or other physical disabilities. Windows 98 offers an Accessibility Options icon in Control Panel, and an Accessibility Wizard and Magnifier through the Start menu, to enhance the user interface to meet the user's needs.

After this lesson, you will be able to:

  • Configure virtual memory.

  • Configure the hard disk cache.

  • Configure the CD-ROM File System cache.

  • Configure Accessibility Options.

Estimated lesson time: 30 minutes
