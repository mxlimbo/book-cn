Lesson 5: Troubleshooting Windows 98 Setup

Failures can occur during Setup. Setup uses different procedures to recover from a Setup failure, depending on when the failure occurred in the Setup process.

This lesson discusses various support issues related to Windows 98 Setup, as well as their solutions.

After this lesson, you will be able to:

  • Troubleshoot common Windows 98 installation issues.

Estimated lesson time: 25 minutes
