Optional Practice: Installing a Network Printer

In this practice, you will use the manual method to install a network printer. You will then install a network printer using the Point and Print method.

Exercise 1
Installing a Network Printer Using the Manual Method

In this exercise, you will use the manual method to install a network printer that is hosted by a Windows 98 server.

    To install a network printer using the manual method

    1. Click Start, point to Settings, and then click Printers.

      The Printers folder opens.

    2. Double-click the Add Printer icon.

      The Add Printer Wizard appears.

    3. Click Next.

      You are asked how this printer is attached to your computer.

    4. Select Network Printer, and then click Next.

      You are prompted for the path to this printer.

    5. In the Network path or queue name box, type the location of the network printer and then click Next.

      You are prompted to enter a printer name.

    6. Replace the default name with a name of your choice, and then click Next.

      You are asked if you would like to print a test page.

    7. Click the No button, and then click Finish.

      When you click Finish, Windows 98 will start copying the files required to support the printer.

Exercise 2
Installing a Network Printer Using the Point and Print Method

In this exercise, you will install a network printer using the Point and Print method.

    To install a network printer using the Point and Print Method

    1. Click Start, and then click Run.

    2. The Run dialog box appears.

    3. In the Open list box, type a UNC for the server name where the printer resides, and then click OK.

      A window opens displaying the shares on the server.

    4. Drag the printer from the server window and drop it in your Printers folder.

      The Add Printer Wizard appears.

    5. Close the open server window.

    6. Click Next on the Add Printer Wizard.

      You are prompted to enter a printer name.

    7. Click Next to accept the default name.

      You are asked if you would like to print a test page.

    8. Click No, and then click Finish.

      When you click Finish, Windows 98 will start copying the files required to support the printer. When the copying is complete, an icon is added to your Printers folder indicating that this is a network-based printer.
