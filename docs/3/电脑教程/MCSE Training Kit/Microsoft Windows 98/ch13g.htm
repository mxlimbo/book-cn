Lesson 2: Logging on to a Windows NT Domain

If your network includes Windows NT domains, you can log on to the domain and connect to domain resources. This lesson introduces the concept of domains and discusses the configuration and administration of domain logons.

After this lesson, you will be able to:

  • Describe how a domain differs from a workgroup.

  • Configure Windows 98 to log on to a domain.

  • Change passwords for a domain.

Estimated lesson time: 30 minutes
