Optional Practice: Using Remote Troubleshooting Tools

In this practice, you will install and use the Remote Registry service, and monitor the system parameters of a remote computer with System Monitor.

You must install the Remote Registry service as a service through the Network icon in Control Panel. Because Remote Registry is not listed as one of the default services, you must supply the path manually.

    To install the remote registry service:

    1. Have your Windows 98 Installation CD available.

    2. Open Control Panel and double click Network.

      The Network dialog box appears.

    3. Click Add.

      The Select Network Component Type dialog box appears.

    4. Click Service and then click Add.

      The Select Network Service dialog box appears.

    5. Click Have Disk.

    6. Put your Windows 98 Installation CD in the CD-ROM drive.

      The Install From Disk dialog box appears.

    7. Click Browse.

      The Open dialog box appears.

    8. In the Drives box, select your CD-ROM drive letter.

    9. In the Folders box, select the following path: D:\Tools\reskit\netadmin\remotereg (where D: is the name of your CD-ROM drive).

      The Regsrv.inf file appears in the File name box.

    10. Click OK four times.

    11. Click Yes when prompted to restart your computer.

      This installs the remote registry service.

After you install Remote Registry Service, you can use it to set up your computer and the remote computer for remote administration.

    To use the Remote Registry Service

    1. Start the Registry Editor.

    2. On the Registry menu, click Connect Network Registry.

    3. The Connect Network Registry dialog box appears.

    4. Type the name of the remote computer that you want to administer.

    To monitor the system parameters of a remote computer

    1. On the Start menu, point to Programs, Accessories, System Tools, and then click System Monitor.

    2. On the File menu, click Connect.

    3. Type the name of the computer you want to connect to, and then click OK.
