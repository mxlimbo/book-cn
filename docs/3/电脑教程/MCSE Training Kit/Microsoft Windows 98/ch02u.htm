Collecting Information About Your Computer

After the Setup Wizard loads and you accept the Licensing Agreement, you use the Choose Directory window (see Figure 2.7) to determine where Windows 98 will be installed.

Figure 2.7 The Choose Directory window

Choose the Install Directory

The Choose Directory window displays the following options:

  • Existing Windows directory, for example, C:\Windows. Selecting this option upgrades the existing version of Windows 95 or Windows 3.1x.

  • If Other is selected, you must supply the name of the directory, and then perform a new installation of Windows 98.

Note

If you set up Windows 98 on a newly formatted hard disk, the C:\Existing Windows directory is replaced with C:\Windows, the default directory name.

After you select the install directory, the Checking Your System window appears. Setup checks the hard disk for errors and initializes the system's registry database.

The Preparing Directory window then appears, and Setup creates the Windows 98 folder structure, including all required folders under the root folder (for example, C:\Windows and C:\Program Files folder).

Setup then checks for installed components and determines if there is adequate disk space for the files.

Note

If there is not enough disk space, the Setup Wizard prompts you to create more free hard disk space and then restart Setup.

Selecting Setup Options During MS-DOS Setup

The Setup Options window (see Figure 2.8) lists four types of Setup. Each Setup type contains a specific set of components to install.

  • Typical installation is recommended for most computers.

  • Portable installation includes options useful for portable computers, such as laptops.

  • Compact installation does not install optional components, thus saving disk space.

  • Custom installation allows you to customize your installation options, and is recommended for advanced users and system administrators only.

Figure 2.8 Setup Options window

Supplying User and Computer Information

If you are installing to a new folder or hard disk, a User Information window prompts you to supply your user name and company name. Windows 98 uses this information for various operations, and you must enter this information for Setup to continue. With a Custom installation, a dialog box prompts you to select the components you want to install. Next, a Computer Information window (see Figure 2.9) prompts you to enter the computer name, workgroup name, and computer description.

  • The computer name must be unique on the network, and can be up to 15 characters long with no spaces. The name can contain only alphanumeric characters and the following special characters: ! @ # $ % ^ & ( ) - _ { } . ~

  • The workgroup name can be up to 15 characters long and uses the same naming convension as the computer name. The workgroup is used to associate groups of computers together for more efficient browsing.

  • The computer description can be up to 58 characters long, but it cannot contain any commas. This text appears as a comment next to the computer name when users are browsing the network, so you can use it to describe the department or location of the computer.

Figure 2.9 Computer Information window

Verifying Computer Settings

As before, Setup prompts you to verify various computer settings, such as keyboard layout and regional settings. You can change these settings by clicking Change.
