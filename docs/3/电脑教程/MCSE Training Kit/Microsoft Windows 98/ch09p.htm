Practice: Using Scheduled Tasks

In this practice, you will use Scheduled Tasks to schedule a daily task.

    To schedule a task

    1. Double-click My Computer, and then double-click the Scheduled Tasks folder to open the Scheduled Tasks window.

      Existing scheduled tasks are represented by icons.

    2. Double-click Add Scheduled Task.

      The Scheduled Task Wizard appears.

    3. Click Next.

      The second dialog box lists all of the programs on the Start menu. It also has a Browse button that enables you to locate any local program you want to schedule that is not accessible from the Start menu.

    4. Locate ScanDisk in the list, select it, and then click Next.

    5. Select Daily under Perform this task, and then click Next.

    6. Enter 3:00 a.m. as the time for the task to begin.

    7. Select Every Day under Perform this task.

    8. Enter today's date as the date to begin, and then click Next.

      The final window offers the option to Open advanced properties for this task when I click Finish. Selecting this option opens the properties window for the chosen task after the wizard closes, so that you can modify the default configuration settings for the chosen task. Do not select this option during this practice.

    9. Click Finish to close the wizard.

      The Task Scheduler icon appears on the taskbar with the clock.

    10. Close all open windows.
