Updating Local and Roaming Profiles

When a user logs off or shuts down the computer, the registry and desktop settings update the local profile and, if it exists, the server-based user profile.

If the user logs on to more than one computer at the same time, the last computer the user logs off has its profile saved to the server. The computer that was logged off first will not have its profile saved because Windows 98 will not merge profile changes from multiple computers.
