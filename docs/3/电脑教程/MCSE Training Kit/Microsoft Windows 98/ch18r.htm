Configuring Multilink Connections

Windows 98 Dial-Up Networking supports the Multilink Point-to-Point Protocol (MPPP). Multilink increases data transmission rates by combining multiple physical links into a logical bundle that increases bandwidth. Dial-Up Networking with Multilink can combine the following connection types on both clients and servers:

  • Analog modem connections

  • ISDN connections

  • Mixed analog and digital connections

A client with an ISDN adapter and with access to both 64 Kbps B channels can use Multilink to establish a single 128 Kbps connection to a Multilink server. This speeds access to the Internet or to an intranet and reduces the time required for remote connection, which can reduce the cost of remote access. Both the Dial-Up Networking client and the server must have Multilink enabled.

To configure Multilink in Windows 98, you must modify the properties of a Dial-Up Networking connection. If you want to configure Multilink, perform the following procedure:

  1. In the Dial-Up Networking folder, right click the connection for which you want to configure Multilink, then click Properties.

  2. Click the Multilink tab.

  3. Select Use additional devices, and then click Add.

  4. In the Edit Extra Device dialog box, select the additional device and enter the telephone number of the server that this device will use.

  5. Click OK, and then click OK again.
