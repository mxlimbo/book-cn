Overview of the System Information Utility

You can use the Microsoft System Information utility to diagnose and solve system problems. For example, if you have display problems, use the System Information utility to determine the type of display adapter on your computer and the video device drivers that are loaded.

The System Information utility collects information from the following sources:

  • the Registry

  • System.ini

  • Win.ini

  • Autoexec.bat

  • Config.sys

  • the operating system

    This includes verifying which 16-bit and 32-bit dynamic-link libraries (DLLs) and software programs have been loaded.

System Information displays a brief summary of information about your computer. The left pane (see Figure 9.1) displays three categories of information: Hardware Resources, Components, and Software Environment. The right pane displays specific information for the category that you select in the left pane.

Hardware Resources displays hardware-specific settings, such as DMA, IRQs, and I/O addresses. Components displays information about your Windows configuration, including the status of your device drivers and network components. Software Environment displays information about software currently loaded into memory, including version information and whether a process is running.

Figure 9.1 The Microsoft System Information utility

For some items, additional choices–Basic Information, Advanced Information, and History–appear on the top of the right pane. Basic Information for Network displays driver resource information for network cards. Advanced Information provides detailed data about the selected item, for instance, the same information plus resource allocation, registry keys, and other applicable details. History provides information about configuration changes for a particular item.

The following programs are available on the Tools menu of the System Information Utility:

  • Windows Report Tool

  • Update Wizard Uninstall

  • System File Checker

  • Signature Verification Tool

  • Registry Checker

  • Automatic Skip Driver Agent

  • Dr. Watson

  • System Configuration Utility

  • ScanDisk

  • Version Conflict Manager

The System Information utility executable file, Msinfo32.exe, is copied to the C:\Program Files\Common Files\Microsoft Shared\Msinfo folder by default during Windows 98 Setup.
