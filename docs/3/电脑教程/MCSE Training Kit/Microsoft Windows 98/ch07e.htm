Start Menu Enhancements

Enhancements to the Windows 98 Start menu include changes to the main Start menu, additions to the Settings (see Figure 7.1) and Find menus, and the use of the drag-and-drop operation.

Figure 7.1 Enhanced Windows 98 Start menu

The Start Menu

The following options are new to the main Start menu:

  • Windows Update is a Web site that contains links to the features described in the following table:

    Windows Update Selection Description
    Update Wizard Keeps the computer current with the latest operating system software and drivers.
    Technical Support Finds answers to questions about Windows 98 quickly and easily.
    Site Feedback Provides feedback to Microsoft about the Windows Update Web site.

  • Favorites is a link to the Internet Explorer Favorites folder.

  • Log Off Username on the Start menu closes all open programs and logs off the current user.

Additions to the Settings Menu

The following items are new to the Settings menu:

  • Folder Options includes configuration options for the desktop, folder views, and file registration.

  • Active Desktop provides the following menu options:

  • Use View as Web Page to enable or disable the Active Desktop.

  • Use Customize my Desktop to gain access to the Display Properties dialog box.

  • Use Update Now to update current Active Desktop objects.

  • Windows Update has the same functionality as the Windows Update selection on the main Start menu.

Additions to the Find Menu

Two new items are added to the Find menu:

  • Use On the Internet to start Internet Explorer and connect to a Web site that provides access to popular Internet search engines.

  • Click People to search your Windows Address Book, and connect to several Internet Lightweight Directory Access Protocol (LDAP) directory servers.

Drag-and-Drop Operation

With the Windows 98 Start menu, you can use drag-and-drop operation to rearrange and add items. If you click an item and drag it, the pointer changes to an arrow carrying the menu item. As you move the item, a black bar appears to indicate the current drop location.
