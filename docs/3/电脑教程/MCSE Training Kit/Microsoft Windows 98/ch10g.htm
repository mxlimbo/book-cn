Creating an Emergency Startup Disk

An Emergency Startup Disk (ESD) is a floppy disk containing utilities you can use to start and troubleshoot a malfunctioning system. The startup disk loads the operating system and presents a Windows 98 command line.

You can create a Windows 98 startup disk during the file copy phase of Windows 98 Setup, or you can use the Add/Remove Programs icon in Control Panel to create a startup disk after installing Windows 98.

Important

You should create a startup disk during Windows 98 Setup, and create several after setup, using the Startup Disk tab of the Add/Remove Programs icon in Control Panel (see Figure 10.2).

Startup Files

The Windows 98 ESD supports both IDE and SCSI (Adaptec and Mylex/Buslogic) CD-ROM drives. If you start the computer with the ESD and load the real-mode CD-ROM drivers but cannot read a compact disc in the drive, the generic drivers supplied with Windows 98 may not work with your drive or controller.

If this happens, you must copy the correct drivers for your CD-ROM drive and controller to the ESD, and then edit the Config.sys file on the ESD so that it will load the drivers. Consult the documentation for your CD-ROM drive and controller to get the correct files and configuration settings. Start your computer with the ESD you create, and attempt to read your CD-ROM drive to test your ESD.
