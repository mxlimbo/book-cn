Before You Begin

To complete the lessons in this chapter, you must have:

  • Operational understanding of the implementation and function of the following: memory, hard disks, types of CPUs, communications ports, printer ports, display adapters, pointing devices, and basic input/output system (BIOS).

  • Completion of Course 578, Networking Essentials, or equivalent knowledge.

  • Extensive experience supporting end users and the ability to set up, optimize, and customize programs.

  • Experience with Windows 95 or Windows NT 4.0 graphical user interface.
