On a Windows NT Network

A Windows NT network stores user profiles in each user's home folder. During logon to a Windows NT domain, Windows 98 connects to the home folder and begins the following comparison and download process:

  • If both a server-based profile and local profile exist, a mandatory server-based profile loads to the computer. If the server-based profile is not mandatory, the most recent profile loads to the computer.

  • If only a server-based profile exists, it downloads from the server to the computer to create a local user profile.

  • If only a local-based profile exists, it loads to the computer and saves to the network server when the user logs off.

  • When the user logs off, the local profile replaces the server-based profile, unless the server-based profile is mandatory.
