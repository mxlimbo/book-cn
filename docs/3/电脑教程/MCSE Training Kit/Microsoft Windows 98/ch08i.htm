Practice: Enabling Extended Capabilities Port (ECP) Support in Windows 98

You can use this to add an ECP device to your computer or check the configuration of the device. This example shows the general procedure for use with any ECP-enabled device.

    To enable ECP support

    1. Consult your computer (or add-in card) manual and record the interrupt request line (IRQ) and direct memory access (DMA) settings selected for each of the ECP ports you want to use.

    2. In Control Panel, double-click the System icon and then click the Device Manager tab.

    3. Double-click Ports (COM & LPT), and then select the ECP device. (You will see this device only if an Extended Capabilities port is detected on your computer.)

    4. Click Properties, and then click the Resources tab. This dialog box shows an I/O range that has been detected automatically.

    5. Click Use Automatic Settings to uncheck it.

    6. In the Setting Based On field, select Basic Configuration 0000, which contains Standard I/O ranges for LPT ports, IRQ, and any DMA setting.

    7. If IRQ setting does not appear, then in the Resource Type list, click Interrupt Request, and then click Change Setting.

    8. In the Edit Interrupt Request dialog box, type the IRQ value from step 1, and then click OK.

    9. If the DMA setting for your device does not appear, then in the Resource Type list, click Direct Memory Access, and then click Change Setting.

    10. In the Edit Direct Memory Access dialog box, type the DMA value you noted in step 1, and then click OK.

    11. If you made changes, then shut down and restart the computer so the changes can take effect.
