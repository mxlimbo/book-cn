Chapter 13

Using Windows 98 on a Windows NT Network

About this Chapter

Using Windows 98 on a Windows NT network expands the capabilities of Windows 98. For example, Windows NT allows Windows 98 users access to Microsoft Distributed File System for Windows NT (DFS), which they can use to connect to resources anywhere on the network without knowing the location of those resources. Windows NT also offers Windows 98 users increased security in the form of user-level access control.

In this chapter you learn how to implement Windows 98 on a Windows NT Network. You learn how Windows NT network resources are organized with DFS, and you learn the procedures for logging on to a Windows NT domain and configuring the computer to log on using a logon script.
