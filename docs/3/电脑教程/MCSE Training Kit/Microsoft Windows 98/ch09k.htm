Defragmenting a Hard Drive

As Windows 98 reads applications from and writes applications to a hard disk, fragmentation occurs — that is, files are stored in noncontiguous clusters. It takes much longer for the computer to read and write fragmented files than unfragmented files.

The disk defragmentation process moves these noncontiguous clusters so that they occupy a contiguous area on the disk, thus reducing access times for these files. To improve file access time, you can defragment uncompressed drives, as well as compressed DriveSpace 3 or DoubleSpace drives.

The Windows 98 Disk Defragmenter tool introduces the ability to store an application's executable file and its supporting files in a contiguous area of disk space. This new feature can decrease the time an application takes to start, in addition to improving its overall performance.

When an application starts, it typically reads an *.exe file and various *.dll files. However, if the application reads nonsequential portions of the *.exe and *.dll files when it starts, it must locate each portion of each file on the disk separately, increasing the time it takes the program to start. Windows 98 Disk Defragmenter tries to place disk clusters in the order they are read, reducing the time it takes to start the program.

If you want to run and configure Disk Defragmenter, perform the following procedure:

  1. Click Start, point to Programs, point to Accessories, point to System Tools, then click Disk Defragmenter.

  2. When the Select Drive dialog box appears, select the drive, then click OK to run Disk Defragmenter with the default settings.

    – Or –

    Select Settings to change the configuration.

  3. Select options in the Disk Defragmenter Settings dialog box:

    When defragmenting my hard drive: This section provides two configuration options:

    • Rearrange program files so my programs start faster. This option places the most frequently used program files at the front of a disk, optimizing program performance. This is selected by default.

    • Check the drive for errors. In case of errors, a ScanDisk prompt appears.

    I want to use these options: This section determines how frequently Disk Defragmenter performs the selected tasks. Select one of the two options:

    • This time only.

    • Every time I defragment my hard drive.

When you start Disk Defragmenter, the window displays a progress bar and a percentage complete counter. If you click the Show Details button, the Disk Defragmenter window changes to a graphical display that shows the progress of the defragmentation.

Tip

Disk Defragmenter takes longer to defragment your hard disk if you run it with details showing. For quickest performance, minimize the Disk Defragmenter window while the utility is running.
