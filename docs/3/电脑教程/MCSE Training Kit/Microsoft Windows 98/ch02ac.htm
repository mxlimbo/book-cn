Recovering Before the First Restart

If the computer stops responding during Setup prior to the first restart, you must turn the computer off, turn it back on, and then run Setup again.

After the Setup Wizard reloads, Setup determines that a failure occurred and the Safe Recovery screen appears (see Figure 2.10).

Figure 2.10 Safe Recovery screen

If you click Use Safe Recovery, which is the recommended option, Setup examines the Setuplog.txt file to determine where the failure occurred. Setup then resumes from the point where the failure occurred.

Caution

If you do not use Safe Recovery, the Setup process will start over from the beginning.
