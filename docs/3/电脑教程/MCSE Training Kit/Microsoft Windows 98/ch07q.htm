Practice: Using the Customize this Folder Wizard

In this practice, you will customize a folder with the Customize this Folder Wizard.

    To customize a folder in Windows Explorer

    1. Open the folder you wish to customize.

    2. From the View menu, click Customize this Folder…

The Customize this Folder Wizard gives you the following options:

  • Create or edit an HTML document

    You can customize views for any local or network folder by using an HTML document that can contain ActiveX controls, VBScript, or Java applets. Doing this lets you preview files on the desktop.

  • Choose a background picture

    This option lets you choose a picture to display in the background of the folder.

  • Remove customization

    This option restores the folder to its original state before it was customized.
