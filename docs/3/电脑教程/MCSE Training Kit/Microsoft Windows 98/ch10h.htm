Lesson Summary

Windows 98 provides two programs for backing up and restoring system configuration files. Windows Registry Checker is a Windows-based program that scans and backs up the following files: User.dat, System.dat, Win.ini, and System.ini. The scan occurs automatically when the system is idle, and when configuration changes are made.

Microsoft Registry Checker is an MS-DOS-based program that scans the system configuration files. If errors are found, it restores a good copy of these files.

An Emergency Startup Disk (ESD) is a bootable floppy disk containing utilities you can use to troubleshoot a malfunctioning system. You should create an ESD during Windows 98 installation. You can also create an ESD with the Add/Remove Programs icon in Control Panel.
