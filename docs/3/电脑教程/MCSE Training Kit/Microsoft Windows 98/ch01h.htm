Advanced Support for New Hardware Standards

Windows 98 supports powerful new hardware technologies while at the same time maintaining the broadest support for older Windows-based applications and technologies.

Support for the newest generation of hardware makes new Windows 98-based computers significantly easier to use, more productive, and easier to maintain.

The following table describes the new hardware standards implemented in Windows 98.

New Hardware Standard Description
OnNow A power management approach that facilitates faster booting. Instead of turning the computer off, it switches to a low power "sleep" state that can be restored easily, so that the computer starts quickly. The result is similar to turning on a television or stereo.
Universal Serial Bus (USB) Support Allows multiple devices to run simultaneously on a computer, with peripheral devices such as monitors and keyboards acting as additional plug-in sites, or hubs.
IEEE 1394 Serial Bus Support (Firewire) Provides enhanced PC connectivity for a wide range of devices, from audio/video components to traditional PC storage devices. IEEE 1394 provides a low cost, Plug and Play-compatible expansion interface for the PC.
Advanced Configuration and Power Interface (ACPI) Supports simpler device management on new PCs and enhanced battery performance on new mobile PCs.
DVD Support Supports the next generation of optical disc storage technology. DVD can store video as well as audio and computer data. DVD aims to encompass home entertainment, computers, and business information with a single digital format, eventually replacing audio CD, videotape, laserdisc, and CD-ROM.
Windows Driver Model (WDM) WDM provides a common set of I/O services and binary-compatible device drivers for both Windows NT and future Windows operating systems. To help decrease the effort necessary for hardware vendors to support all Windows platforms, this driver model is based on a class/minidriver structure that provides modular, extensible architectures for device support.
Multiple Display Support Supports display of applications on multiple monitors. Each monitor is capable of running at a different resolution, providing users with greatly enhanced display potential.

Note

For more information on WDM, USB, IEEE 1394/Firewire, OnNow, ACPI, and DVD, see http://www.microsoft.com/hwdev, the Microsoft Hardware Development Web site.
