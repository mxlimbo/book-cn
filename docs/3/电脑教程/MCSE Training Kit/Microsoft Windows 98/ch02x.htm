Lesson 4: Automating Setup

You can automate Setup by using a setup script. A setup script is a text file that contains predefined settings for all of the options specified during Setup, and can contain instructions for installing additional software.

With a setup script, you create an automated installation template for installing Windows 98 on multiple computers. It is recommended that you use a setup script if you need to conduct a standardized installation more than five times. Using a setup script allows you to perform unattended installations on multiple computers, thus reducing the cost of upgrading to Windows 98.

This lesson discusses the structure of setup scripts, and how to edit scripts and generate new ones.

After this lesson, you will be able to:

  • Describe the structure of a setup script.

  • Edit already existing setup scripts.

  • Generate new setup scripts.

  • Use setup scripts to install Windows 98.

Estimated lesson time: 15 minutes
