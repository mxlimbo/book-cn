Lesson 2: Controlling and Troubleshooting the Boot Process

It is important to be able to troubleshoot the start, or boot, process of your computer. If you are aware of what steps occur (and in what order) during the process, you will be better able to troubleshoot. For example, knowing that a portion of the process occurs before Windows 98 loads may make it easier to determine that a particular problem is hardware-related rather than software-related.

Various methods of starting Windows 98 provide an opportunity for you to view and control the boot process as it occurs, and to identify the commands or drivers that cause a problem with startup.

This chapter provides an overview of the boot process and also discusses several methods of troubleshooting the process.

After this lesson, you will be able to:

  • Describe what occurs during the boot process.

  • Edit the Msdos.sys file.

  • Troubleshoot the boot process with the System Configuration utility.

  • Troubleshoot the boot process with the Startup Menu.

  • Monitor the boot process with a bootlog file.

Estimated lesson time: 20 minutes
