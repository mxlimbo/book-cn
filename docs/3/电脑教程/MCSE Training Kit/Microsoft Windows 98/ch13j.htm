Synchronizing Passwords

Your local Windows password and your Windows NT domain password are maintained separately. However, if your computer is configured so that you log on to a Windows NT domain when you log on to Windows 98, you can keep your local Windows password and your domain password synchronized. You can then use the same password for both Windows 98 and the domain. Use the Passwords option in Control Panel to implement, change, and synchronize user passwords (see Figure 13.5).

Figure 13.5 Changing passwords with the Passwords Properties sheet

Changing Passwords

The Passwords Properties sheet offers two options for changing passwords.

  • Change Windows Passwords—you can set your Windows password, and synchronize it with other resource passwords such as the screen saver password.

  • Change Other Passwords—you can change your password for specific resources, such as for a Windows NT domain.

When you change your Windows password, you can also choose from a list of other system component program passwords that have this feature enabled. If you select any of these other options, Windows 98 will change the system password and will also communicate with the selected programs to change those passwords. The same password will be applied to each selected program.

Note

You can synchronize passwords only for programs specifically written to take advantage of Control Panel Passwords; Windows 98 does not automatically change all passwords; for example, a password for a Web site may not take advantage of this feature.
