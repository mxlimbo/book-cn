Configuring Floppy and Removable Drives

Computer configurations can vary widely. Windows 98 includes two tabs in File System Properties that offer options that can improve performance for floppy disk drives and removable disk drives (see Figure 6.7).

Figure 6.7 The Floppy Disk and Removable Disk tabs in the File System Properties sheet

Floppy Disk Options

If you check Search for new floppy disk drives each time your computer starts, Windows 98 directs the floppy disk driver to scan for new drives every time the system starts.

This is helpful for portable computers. Select this option if you frequently add or remove drives between Windows sessions and the drives are not recognized upon startup. Clearing this option makes startup faster.

Removable Disk Options

Selecting Enable write-behind caching on all removable disk drives speeds performance on any removable drives installed in your computer. Clear this box if you are experiencing problems with disk operation.
