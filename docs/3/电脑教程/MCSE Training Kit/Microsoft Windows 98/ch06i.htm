Practice: Configuring Display Properties

In this practice, you will modify the screen area of your display with the Display Properties dialog box.

    To configure display properties

    1. In Control Panel, double click the Display icon.

      The Display Properties dialog box appears.

    2. Click the Settings tab.

    3. Drag the Screen area slider until the screen area is set to 800 x 600 pixels.

    4. Click OK. (A Compatibility Warning dialog box may appear, depending on your video adapter card. If it does, click Apply the new color settings without restarting, and then click OK.) The Display Properties dialog box appears indicating that Windows is going to adjust your display settings.

    5. Click OK to test your new display settings. The Monitor Settings dialog box appears asking if you want to keep the new settings.

      Click Yes to accept the new settings.
