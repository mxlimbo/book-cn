Viewing NetShow Productions

You can use Microsoft NetShow for streaming multimedia over networks that range from low-bandwidth dial-up Internet connections to high-bandwidth LANs. Streaming enables content to be delivered to the client computer as a continuous flow of data with little wait time before content playback begins. NetShow makes it possible to deliver the following throughout an intranet and across the Internet:

  • Live and on-demand audio

  • Video

  • Synchronized sound and still images

NetShow uses Active Streaming Format (ASF) file format to encapsulate multimedia data types and embedded text, and synchronizes these objects within a stream. Major multimedia authoring tool vendors are adopting this file format. The ASF file format is a media compression scheme that eliminates the need to purchase special hardware. To view a NetShow production you must use Microsoft NetShow Player, or an application that supports the NetShow ActiveX control.

A NetShow presentation starts with the feed, which is the content captured through any of these media:

  • VCR

  • Video camera

  • Television

  • Digital Camera

  • Microphone

To run the NetShow presentation, you must have the following:

  • A computer running Windows NT Server 4.0

  • NetShow Server running NetShow Multicast Channel Manager

  • Internet Information Server for HTML pages (this can be a separate server)

You can connect to the NetShow production through a wide area network, the Internet, or an intranet. You can view a NetShow production in two ways:

  • Using NetShow Player. By choosing this option, you can decide whether to view the content locally or remotely.

  • Visual Basic script or other ActiveX container application embedded in a Web page. This option enables a person who does not have NetShow Player installed to see the content.

If you want to install NetShow Player, perform the following procedure:

  1. From Control Panel, click the Add/Remove Programs icon.

  2. On the Windows Setup tab, select Multimedia.

  3. Click Details.

  4. Click NetShow Player to select it.

  5. Click OK.
