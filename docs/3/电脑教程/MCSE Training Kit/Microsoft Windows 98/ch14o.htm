Mapping Network Drives

Although you do not need to map a drive in order to use resources on a NetWare server, you can use Windows Explorer to map a drive to a NetWare volume or to a directory within a volume.

If you map a drive to a NetWare volume, you must choose a drive letter to map, and must specify whether Windows 98 should remap the drive each time you log on.

If you want to map a drive to a NetWare volume, perform the following procedure:

  1. Open Network Neighborhood.

  2. Double-click on a NetWare server.

    A window opens displaying the available volumes.

  3. Right-click one of the available volumes.

    A menu appears displaying the available commands.

  4. Click Map Network Drive.

    The Map Network Drive dialog box appears.

  5. Select an available drive letter, and then click OK.

If you map a drive to a NetWare directory while exploring or browsing the server, you must configure the same options as for mapping to a volume. You must choose a drive letter and determine whether or not to have Windows 98 remap the drive when you log on again. You must also specify whether the drive letter should be mapped to the selected directory rather than to the directory's parent volume. This is equivalent to the NetWare map root command.

If you want to map a drive to a NetWare directory, perform the following procedure:

  1. Open Network Neighborhood.

  2. Double-click on a NetWare server.

  3. Double-click one of the available volumes.

  4. Right-click one of the available shared folders.

  5. Click Map Network Drive.

  6. Select an available drive letter and then click OK.

Note

You can also map a drive from the command prompt using the net use or map commands.

Using Windows Explorer to Map a Drive

When mapping a drive using Windows Explorer, you can use UNC syntax or NetWare syntax. In addition, if you are using Service for NetWare Directory Services, you can also use NDS syntax. For example, you can use either of the following to map the letter z to \\NetWareServer\sys:

net use z: \\NetWareServer\sys

map z:=NetWareServer\sys
