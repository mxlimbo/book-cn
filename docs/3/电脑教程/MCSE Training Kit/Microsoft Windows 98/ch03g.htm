Formatting Partitions

After you create a partition, you must restart your computer and then format the partition before you can store data on it. You can format partitions through Windows Explorer by right-clicking on the drive and selecting Format from the drop-down menu. You can also format a drive from the command prompt using the Format command.

Figure 3.2 Windows Explorer format operations

Windows Explorer offers the following types of format operations (see Figure 3.2):

  • Quick (erase). Erases the directory table, which contains the list of files and folders on the partition, but does not actually erase all data from the partition. You can only use this option on formatted partitions.

  • Full. Erases all data from the partition.

  • Copy system files only. Does not format the partition, but instead copies the files that start Windows 98 and creates a Windows 98 boot sector on the partition.

Caution

This procedure is presented for your information only. Formatting a hard disk destroys all data on the disk.

If you want to format a hard disk, perform the following procedure:

  1. In Windows Explorer, right-click the drive icon for that partition, and then click Format.

    – Or–

    Start your computer with your emergency startup disk. At the command prompt, type the following:

    format drive_letter:

    For drive_letter, type the letter of the partition you want to format. If you are formatting drive C, you can also copy system files to the partition by typing the following at the command prompt:

    format c: /s

    When the warning message appears, proceed with formatting by pressing Y. Then press ENTER.

  2. When formatting is complete, type a volume label, and then press ENTER.

  3. If you want to format another partition, repeat steps 1 and 2.

Remove the floppy disks from all floppy disk drives, and restart the computer by pressing CTRL+ALT+DEL.
