Lesson Summary

The following are tips for troubleshooting user profiles:

  • Make sure applications use the registry rather than .INI files.

  • Be sure to install the components and applications for each shortcut in the user's profile on each computer where the user will log on.

  • Do not use Windows NT profiles on a Windows 98 computer (and vice versa).

  • In a mixed Windows NT/NetWare environment, manually synchronize profiles so that they are available across the enterprise.
