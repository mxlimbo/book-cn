Lesson 3: Connecting to NetWare Servers

Windows 98 users can browse, connect to resources, and print in a NetWare network environment, just as they can in a Microsoft network environment. This lesson discusses the requirements to integrate the Windows 98 and NetWare environments, and how to access resources in the mixed environment. This lesson also discusses browsing the NetWare network, mapping network drives, and printing to NetWare print servers.

After this lesson, you will be able to:

  • Map a drive to a NetWare volume, directory, or server.

  • Print from Windows 98 to a NetWare server.

Estimated lesson time: 30 minutes
