Configuring Scripting for Modem Connections

A script is a text file consisting of commands. Windows 98 supports the use of scripts to configure modem connections except for ISDN or Point-to-Point Tunneling Protocol (PPTP) connections. If your ISP requires you to log in manually, you can write a login script to automate the process.

Windows 98 provides four sample scripts you can use as starting points. You can find sample scripts in the C:\Program Files\Accessories folder.

Windows 98 also provides a file explaining how to write and modify login scripts. This file is named Script.doc and is in your C:\Windows folder.

Some ISPs also provide scripts you can use to log in to their computers. Contact your ISP for more information.

Caution

Microsoft does not support login scripts you create. Also, Microsoft does not support modifications to the four sample scripts.

Figure 18.9 Using a Dial-Up Networking script

If you want to use a script for a Windows 98 Dial-Up Networking connection, perform the following procedure:

  1. In the Dial-Up Networking folder, right-click the icon for the connection you want to configure, and then click Properties.

  2. Click the Scripting tab (see Figure 18.9), and then click Browse.

  3. Locate the script, and then click Open.
