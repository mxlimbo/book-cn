Optional Practice: Interoperating with TCP/IP

In this practice, you will test TCP/IP configuration and configure TCP/IP with a static IP address.

To complete this practice, you will require a second computer that has TCP/IP configured, and a valid IP address.

Exercise 1
Testing TCP/IP Configuration

In this exercise, you will view and test your current TCP/IP configuration. You will then configure your computer to obtain an Autoconfigured IP address.

    To view the current TCP/IP configuration

    1. Right-click Network Neighborhood, and then click Properties.

      The Network dialog box appears.

    2. In the list of installed components, click TCP/IP - > Net_Card_Name, and then click the Properties button.

      The TCP/IP Properties dialog box appears.

    3. Click the WINS Configuration tab.

      The WINS Configuration property sheet appears.

      Answer

    4. Click Cancel to close the TCP/IP properties dialog box.

    5. Click Cancel to close the Network dialog box.

    6. Click Start, and then click Run.

      The Run dialog box appears.

    7. In the Open box, type winipcfg and then click OK.

      The IP Configuration window appears.

    8. Click More Info.

      The IP Configuration window expands.

    9. Under Ethernet Adapter Information, click the arrow to open the list of adapters, and click your Net_Card_Name.

    10. Record the following information from the screen.

      Subnet Mask: ________. ________.________.________

      DHCP Server: ________. ________.________.________

    To receive an IP Autoconfiguration address

    1. Detach the network cable from your computer.

    2. Switch to the IP Configuration application, and then click Renew.

      A message appears indicating that your computer was unable to reach a DHCP server, and asks if you wish to see further network messsages.

    3. Click Yes.

      Notice that you have been issued an Autoconfigured IP Address.

    4. Record your new IP address:

      IP Autoconfiguration Address: ________. ________.________.________

      Answer

    5. Reattach the network cable.

    6. Switch to the IP Configuration application, and then click Renew.

      A message appears indicating that your computer successfully leased an IP address, and asks if you wish to see further network messsages.

    7. Click Yes.

      Notice that your IP Address and Subnet Mask have been reissued.

    8. Click OK to close the IP Configuration dialog box.

    9. Type exit and then press ENTER to exit from the MS-DOS prompt.

Exercise 2
Configure TCP/IP with a Static IP Address

In this exercise, you will reconfigure TCP/IP to use a static IP address.

    To configure TCP/IP with a static IP address

    1. Right-click Network Neighborhood and then click Properties.

      The Network dialog box appears.

    2. In the list of installed components, click TCP/IP - > Net_Card_Name, and then click Properties.

      The TCP/IP Properties dialog box appears.

    3. On the IP Address tab, click Specify an IP address.

    4. Type an IP address (you may need to ask your network administrator for an IP address).

    5. Click OK.

      The Network dialog box reappears.

    6. Click OK.

      A dialog box appears prompting you to restart your computer.

    7. Click Yes.

      Your computer restarts.

    To view the TCP/IP configuration

    1. Log on to the network, click Start, and then click Run.

      The Run dialog box appears.

    2. In the Open box, type winipcfg and click OK.

      The IP Configuration window appears.

    3. Click More Info.

    4. Under Ethernet Adapter Information, click the arrow to open the list of adapters, and click your Net_Card_Name.

    5. Record the following information from the screen.

      IP Address: ________. ________.________.________

      Subnet Mask: ________. ________.________.________

      DHCP Server: ________. ________.________.________

    To test TCP/IP configuration

    1. Go to an MS-DOS prompt, and ping your own computer by typing ping localhost and pressing ENTER.

      Was the ping command successful?

      Answer

    2. Exchange IP addresses with the second computer.

      IP Address : ________. ________.________.________

    3. Type ping followed by the other computer's IP address, and then press ENTER.

      Was the ping command successful?

      Answer

    To configure TCP/IP to receive a DHCP address

    1. Right-click Network Neighborhood and then click Properties.

      The Network dialog box appears.

    2. In the list of installed components, click TCP/IP - > Net_Card_Name, and then click Properties.

      The TCP/IP Properties dialog box appears.

    3. On the IP Address tab, click Obtain an IP address automatically, then click OK to close the TCP/IP Properties dialog box.

      The Network dialog box reappears.

    4. Click OK.

      A dialog box appears prompting you to restart your computer.

    5. Click Yes.

      Your computer restarts.
