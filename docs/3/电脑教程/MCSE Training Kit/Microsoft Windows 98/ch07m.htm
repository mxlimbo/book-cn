Desktop Styles

Windows 98 provides two different ways to access and view content on the desktop: through a Windows Classic style desktop, or through a Web style desktop, using Explorer.

Figure 7.4 Windows Classic style desktop

In the classic style (see Figure 7.4), the Windows classic settings are the default settings. In this mode you select objects, open applications, and view folders the same way you did in Windows 95.

Figure 7.5 Web style desktop

Web style (see Figure 7.5) provides an optional desktop style in Windows 98. When Web style is enabled, Windows 98 adds the functions of the Web browser to the desktop. Using the Web style desktop, you can select and launch objects on the desktop, and display the contents of folders on the desktop.
