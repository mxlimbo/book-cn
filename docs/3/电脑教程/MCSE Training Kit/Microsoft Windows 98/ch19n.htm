Lesson 3: Using Windows 98 Troubleshooting Tools for Networks

In addition to the Web-based tools discussed in the previous lesson, Windows 98 provides a number of other tools to help you resolve problems when running Windows 98 on a network. This lesson provides an overview of Troubleshooters, Registry Editor, and System Monitor.

After this lesson, you will be able to:

  • Use Windows 98 Troubleshooters to troubleshoot networking problems.

  • Troubleshoot remote computers using Registry Editor and System Monitor.

Estimated lesson time: 20 minutes
