Using Disk Cleanup

Disk Cleanup is a utility introduced in Windows 98 to remove temporary and deleted files from a hard disk. Disk Cleanup automates the maintenance task of deleting excess files. You customize Disk Cleanup with a wizard that lists ways to free disk space, including emptying the Recycle bin, removing old temporary files, and emptying the Internet cache (see Figure 9.2).

Run Disk Cleanup periodically to maintain system performance, when a disk is nearly full, and before running Disk Defragmenter.

Figure 9.2 The Disk Cleanup utility

You can also use Disk Cleanup to regain disk space on any hard drive. If you want to run Disk Cleanup, perform the following procedure:

  1. Double-click My Computer on the desktop.

  2. Right-click the drive C icon.

  3. Click Properties to see the (C:) Properties dialog box.

  4. Click Disk Cleanup.

    Disk Cleanup calculates the space that can be freed, and then the Disk Cleanup for (C:) dialog box appears.

The Disk Cleanup dialog box has three tabs, Disk Cleanup, More Options, and Settings. The Disk Cleanup tab shows the following information:

  • The total disk space that can be freed on the identified disk drive.

  • Files to delete. A checkbox shows the following file types, and the disk space used by the files. When you select a checkbox, it marks that file type to remove from memory.

    File Type Description
    Temporary Internet Files Files stored in %windir%\Temporary Internet Files
    Downloaded Program Files ActiveX controls or Java applets stored in %windir%\Downloaded Program Files
    Recycle Bin Files stored in the Recycle Bin
    Temporary files Files stored in the %temp% folder that have not been used in the previous seven days

  • Total amount of disk space you gain. Shows the total disk space selected.

  • View Files button. Press this button to view the files that will be deleted. This option is not available for Temporary files, nor for the Delete Windows 98 Uninstall Information files.

The More Options tab provides these additional Disk Cleanup options:

  • Windows components frees disk space by removing Windows 98 optional components. This starts Add/Remove Programs from Control Panel, with the Windows Setup tab selected.

  • Installed programs frees disk space by removing programs that are infrequently used. This starts Add/Remove Programs from Control Panel, with the Install/Uninstall tab selected.

  • Drive Conversion (FAT32) creates more space on the hard drive by converting from FAT16 to FAT32. This starts the Drive Converter (FAT32) wizard.

The Settings tab controls whether the Disk Cleanup wizard is to appear when a user is running out of hard disk space. If active, when you copy or save a file to a hard disk and the disk has less than 3 percent of its space left, Windows 98 shows a warning and starts the Disk Cleanup utility.
