Lesson 4: Implementing Disk Compression

Disk compression shrinks data to reduce the space necessary to store it. Windows 98 includes the DriveSpace 3 compression engine and provides support for partitions that have been compressed with the Windows 95 DriveSpace compression engine.

DriveSpace 3 is an enhancement to the DriveSpace compression engine that was included in Windows 95. DriveSpace 3 became available in Windows 95 (OSR2) and Microsoft Plus! For Windows 95.

This lesson provides an overview of disk compression. It also discusses how to manage and tune your compressed disks for optimum performance.

After this lesson, you will be able to:

  • Compress disk drives with DriveSpace 3.

  • Manage and tune compressed drives.
