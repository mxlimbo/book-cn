Improved Communications and Networking Support

Windows 98 introduces improved support for networking and communications features. The following table describes the networking and communications features introduced by Windows 98.

Feature Description
Network Driver Interface Specification (NDIS) 5.0 NDIS makes it possible for any transport protocol to communicate with any network adapter card. NDIS 5.0 adds enhancements to the existing NDIS specification, such as the ability to automatically rebind when changing network cable types and support for OnNow and ACPI.
Windows Sockets 2.0 (WinSock 2.0) Provides concurrent access to multiple network transports. Unlike version 1.1, which focused on TCP/IP, the new API is transport-independent.
Web-based Enterprise Management (WBEM) Enables management of enterprise networks using a Web browser. WBEM allows disparate networks, applications, desktops, and systems management tools to work together. WBEM enables development of tools that reduce the complexity and costs of enterprise management.
Resource Reservation Protocol (RSVP) Allows the user to request a level of network bandwidth for complex data such as audio and video.
Integrated Services Digital Network (ISDN) Provides high-speed, fully digital transmission of a variety of data over a single line.
PPP Multilink Protocol Increases bandwidth by combining multiple physical links, such as utilizing the bandwidth of two or more modems.
Point-to-Point Tunneling Protocol (PPTP) and virtual private networking (VPN) PPTP can be used to create virtual private networks. In the past, connecting remote offices and users to a corporate network required either paying long distance charges or purchasing leased lines. It is now possible to use communications bandwidth on the Internet to provide this connectivity. A VPN, using a public network, provides a level of security equal to or greater than that provided by a private phone call or leased line.
Infrared Data Association (IrDA) protocol version 3.0 Allows communication links over infrared transmissions.
Distributed File System Compiles and displays network data in a single logical structure, no matter where the data is stored.
