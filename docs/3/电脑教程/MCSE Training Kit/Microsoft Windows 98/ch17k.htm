Practice: Configuring a Web Page Subscription for Offline Reading

In this practice, you will configure a Web page subscription for offline reading.

    To configure a Web page subscription for offline reading

    1. Open a Web page to which you wish to subscribe.

    2. On the Favorites menu, click Add to Favorites.

    3. Click Yes, notify me of updates and download the page for offline viewing.

    4. Click Customize to launch the Subscription Wizard.

    5. Select Download this page and pages linked to it, and then click Next.

    6. Select the number of pages you want downloaded, and then click Next.

    7. Select the desired e-mail option for this subscription, and then click Next.

    8. If prompted, enter your e-mail name and your e-mail server name, then click OK, and then click Next.

    9. Select the desired scheduling option for this subscription, and then click Next.

    10. Enter any necessary user name and password information for the site, and then click Finish.

    11. Click OK.

Note

If you do not designate a schedule for downloading updates to Web pages, Internet Explorer defaults to a daily update at 12:00 am.

You can also modify downloading and scheduling options for existing subscriptions using the Manage Subscriptions option in the Favorites menu, which opens the C:\Windows\Subscriptions folder.
