Other Installation Support Issues

Windows 98 Setup is designed to minimize the kinds of problems that can occur during installation. These issues can include:

  • Incompatible video drivers

  • Setupx.dll remaining in memory

  • Changing the installed components

  • Interrupting file copying

  • Virus protection software

Incompatible Video Drivers

When you install a video card for your computer, you should make sure you are using the correct driver. Using an incorrect driver could result in the inability to display the desired resolution or the correct number of colors, and could even cause the video display to distort.

Setupx.dll Remaining in Memory

Setupx.dll is the program that actually performs the Windows 98 installation. When Windows 98 Setup is started, it checks to see if Setup is being run from inside Windows 95 or from MS-DOS. When Setup indicates that the Setup Wizard is loading, Setupx.dll has been loaded into memory.

Setupx.dll can remain in memory if Setup fails. If you try to run Setup again without first restarting the computer, Setup may fail, and you may see an error message. For this reason, if Setup fails, shut down and then restart the computer prior to running Setup again. This will ensure that a previous copy of Setupx.dll is not in memory.

Changing the Installed Components

Because Windows 98 is an upgrade to Windows 95, there is no way to add or remove components when you run Setup from within Windows 95. If you want to change the installed components, you can add or remove them after Setup finishes by clicking Add/Remove Programs in Control Panel. You can change the installed components during Setup by either using a setup script or running Setup.exe from MS-DOS.

Interrupting File Copying

File copying must complete without interruption in order to ensure a successful Setup. If Setup is interrupted during the file-copying phase, Windows 95 may not run when you restart your computer. This is because the Windows 95 directory now contains a mixture of Windows 98 and Windows 95 system files.

If you want to continue Setup when Windows 95 will not run, perform the following procedure:

  1. Shut down and then restart the computer.

    The computer will be able to restart only from MS-DOS at this point.

  2. Run Setup.exe from MS-DOS.

  3. When the Safe Recovery dialog box appears, select Use Safe Recovery (recommended).

Virus Protection Software

As discussed earlier in this chapter, you should disable any anti-virus software or BIOS-level anti-virus protection prior to running Setup or converting a FAT16 drive to FAT32.
