Setting Up a FAT32 Partition

If you decide to use the FAT32 file system, there are some initial steps you should take in preparation, such as verifying the size of the partition(s) that you want to format as FAT32. You can enable large disk support so that all new partitions that are 512 MB or larger will be formatted as FAT32.

You can also convert existing FAT16 partitions to FAT32. In order to enable FAT32, you should:

  • Verify that your drive is 512 MB or larger. You can only enable FAT32 on hard disks that are at least 512 MB.

  • Verify that your current file system is FAT by checking the Properties sheet for that partition. The Type field indicates whether a drive is FAT or FAT32.

  • Make a Windows 98 emergency startup disk before you enable FAT32.

  • Back up any data on your partition before you convert it.

  • Disable any virus protection software before you enable FAT32 or convert to FAT32. Windows 98 will disable most virus protection software automatically, but cannot disable BIOS-level virus protection software.

Enabling Large Disk Support

Enabling large disk support determines that all partitions you create after that selection will be formatted as FAT32. It does not affect existing FAT16 partitions.

Figure 3.4 Two methods for setting up a FAT32 partition

The method you choose for setting up a FAT32 partition depends on whether you want to create new partitions or convert existing partitions (see Figure 3.4).

  • Run Fdisk.exe and enable large disk support if you want all new partitions to be formatted as FAT32.

  • Use the Drive Converter (FAT32) that is included with Windows 98 to convert existing partitions from FAT16 to FAT32.

The method you choose determines the steps that you take in order to format your partition.

In the first method, you run Fdisk—the version included with Windows 95 (OSR2) or Windows 98—on a partition of at least 512 MB. Fdisk prompts you to enable large disk support. Any partitions you create that are at least 512 MB in size will be formatted as FAT32 partitions. However, any existing partitions that are already formatted as FAT16 will remain FAT16.

Tip

If you have a hard disk that is not partitioned, use the Windows 98 emergency startup disk to start your computer and then run Fdisk to make the entire hard disk FAT32.
