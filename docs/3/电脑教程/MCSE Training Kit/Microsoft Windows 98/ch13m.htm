Lesson 3: Using Windows NT Network Resources

Windows 98 can connect to and share resources with computers running Windows NT. This lesson discusses the issues involved with sharing resources with computers running Windows NT and with other users.

When you share resources on a Windows NT network, you can take advantage of the greater security that user-level access control provides. The Access Control List (ACL) determines which users have access to network resources. You can edit the ACL to expand or limit permissions for users.

User-level access control also allows you to assign permissions, such as read only or full access, to shared files. You assign permissions explicitly or implicitly. For example, you can assign full access to a folder, which means that a user has full access to any subfolders within the parent folder. You can change that implicit permission by assigning an explicit permission to one of the subfolders.

After this lesson, you will be able to:

  • Connect to a computer running Windows NT.

  • Print to a Windows NT print server.

  • Enable user level security.

  • Edit an Access Control List.

  • Assign folder permissions.
