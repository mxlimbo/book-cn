Using Windows Update

Windows Update (see Figure 19.2) is a Web-based resource site for updating operating software and device drivers and for getting technical support answers. When you install Windows 98, the installation program places a shortcut to the Windows Update page at http://update.microsoft.com on the Start and Settings menus.

Figure 19.2 The Windows Update Web page

Windows Update is implemented through the use of ActiveX controls, which are installed the first time you connect to the Windows Update page. This page appears only if you have a browser that supports ActiveX controls.

The Update resource works by:

  • Reviewing the device drivers and system software on your computer.

  • Comparing that information to a master database.

  • Recommending and installing updates specific to your computer.

The Windows Update site provides a link to Technical Support, also called Support Online (see Figure 19.3). Here you can search through technical support articles for a particular topic or search through a glossary of terms. Windows Update Technical Support is also available through Windows online help by opening Help and clicking the Web Help button.

Figure 19.3 Windows Update Technical Support

If you want to open Technical Support from Online Help, perform the following procedure:

  1. Click Start, then Help, and then click the Web Help button.

  2. When the Web Help windows displays, click Support Online.

The system administrator may want to restrict access to Windows Update in order to maintain control of the drivers and system files installed on users' computers. Windows 98 system policies include the option to either disable the Windows Update shortcut or reroute the shortcut to an alternative URL.

Note

You can also update device drivers from a floppy disk using Device Manager, which is located in the System icon in Control Panel. In addition, The Update Device Driver Wizard in Device Manager can search Windows Update.
