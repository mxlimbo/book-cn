Lesson 4: Using the Registry to Configure Windows 98 Settings

Because the Windows 98 registry is the key to a computer's hardware and software configuration settings, it is essential for support technicians to understand how to work with it. This lesson introduces the registry, describes its structure, explains how to maneuver through it, and shows how to edit it to meet different computing needs.

After this lesson, you will be able to:

  • Describe the registry structure.

  • Use the Registry Editor to alter registry values.

Estimated lesson time: 25 minutes
