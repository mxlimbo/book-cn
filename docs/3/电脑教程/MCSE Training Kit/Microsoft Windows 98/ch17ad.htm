Lesson Summary

Personal Web Server transmits information and files using the Hypertext Transport Protocol (HTTP). Personal Web Server can be used on an intranet to create, maintain and test Web sites, with a connection limit of 10 users. Once TCP/IP is installed on the computer, Personal Web Server can be installed from the Windows 98 CD-ROM.

Using the Personal Web Manager, you can configure, administer and monitor the Personal Web Server. The Personal Web Manager has four separate views to help manage your site: Main, Publish, Web Site and Advanced.
