Practice: Viewing Configuration Information

In this practice, you will view information about your computer's configuration using the System Information utility.

    To explore the System Information utility

    1. Point to Start, Programs, Accessories, System Tools, and then click System Information.

    2. Expand category lists by clicking on the plus (+) symbols, and search for the following items:

      Which MS-DOS device drivers are loaded?

      Would this list be identical on all computers?

      You need to install a legacy modem that can use IRQ 5, 9, or 11. Will you be able to install it in your system?
