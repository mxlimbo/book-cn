Summary

The following information summarizes the key points in this chapter:

The Internet and Intranets

  • The Internet is composed of high-speed data lines connecting regional commercial networks. An intranet allows internal network users to browse and share information using the same software and hardware as the Internet without needing to connect to an Internet Service Provider.

  • After you subscribe to a Web site, Windows 98 can notify you when a Web page has been updated. You can configure your subscriptions to download Web pages for offline reading.

  • A channel is a Web site that makes it possible to receive up-to-date Web content. Once subscribed to, the channel's content is periodically updated by the provider and is sent directly to your desktop.

  • Microsoft NetShow is an Internet Explorer component that streaming multimedia content over networks. NetShow provides a continuous data flow with little wait time before content playback.

Controlling Access to Internet Content

  • Security levels and security zones enable you control user access to the Internet. You can configure rating levels for each of four categories: language, nudity, sex, and violence.

  • Proxies and firewalls enable you to regulate a Web site's traffic. A proxy server acts as a secure gateway to the Internet for a network or group of computers. Firewalls are routers that filter traffic and perform security auditing.

  • You can configure Internet Explorer to run in kiosk mode, in which a publicly available workstation is dedicated for Internet access.

Sharing Resources on an Intranet

  • Personal Web Server transmits information and files using the Hypertext Transport Protocol (HTTP). Personal Web Server can be used on an intranet to create, maintain, and test Web sites, with a connection limit of 10 users.

  • Using the Personal Web Manager, you can configure, administer and monitor the Personal Web Server. The Personal Web Manager has four separate views to help manage your site: Main, Publish, Web Site, and Advanced.

Communicating on the Internet and Intranets

  • Using Microsoft NetMeeting, you can communicate through intranets or the Internet using video, audio, and data conferencing. You can share information with remote users from one or more applications on your computer.

  • Outlook Express can be used to read and send Internet e-mail, browse and post newsgroup messages, and sort and store messages on the local drive or mail server.
