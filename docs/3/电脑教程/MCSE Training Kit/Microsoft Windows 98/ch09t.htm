Technical Support

The Windows Update site provides a link to Technical Support (see Figure 9.6), also called Support Online.

Figure 9.6 Windows Update Technical Support

From here you can search through technical support articles for a particular topic or search through a glossary of terms. Windows Update Technical Support is also available through the Windows Help toolbar or through a Windows 98 Troubleshooter.

Troubleshooters are interactive help files that ask particular questions about problems you are experiencing. If a Troubleshooter cannot solve your problem, it directs you to the Support Online site where you can research the problem more thoroughly. Troubleshooters are available in Windows Help on the Contents tab under Troubleshooting.
