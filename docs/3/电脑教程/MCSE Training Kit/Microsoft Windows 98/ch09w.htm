Recommended Practices

Here are some recommendations to help you maintain Windows 98 in a stand-alone environment:

  • Use the Microsoft System Information utility to diagnose and solve system problems.

  • Use the Maintenance Wizard and/or Scheduled Tasks to automate disk maintenance and other system maintenance tasks. For instance, use Scheduled Tasks to run a complete virus check on your system daily.

  • Run Disk Cleanup frequently to remove temporary and deleted files from a hard disk.

  • Use Update Wizard on the Windows Update to review available device drivers and system software for your computer. Windows Update can install updates specific to your computer.
