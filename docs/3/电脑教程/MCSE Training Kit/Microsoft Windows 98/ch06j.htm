Lesson Summary

Control Panel is a folder containing icons with which you open property sheets to customize the computer's configuration. You also use Control Panel to access the Add New Hardware Wizard, which allows you to install hardware devices that are not Plug and Play compatible.

A hardware profile is a record of a specific hardware configuration. Windows 98 uses hardware profiles to determine which drivers to load when the system hardware changes. You can create and modify hardware profiles through the System icon in Control Panel.

The Display icon in Control Panel opens the Display Properties dialog box, which provides options for configuring screen resolution, color depth, appearance settings, and screen savers.
