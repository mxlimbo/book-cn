Using the Windows Report Tool

The Windows Report Tool is an Internet-based bug reporting tool that is installed with Windows 98. If you experience a problem with Windows 98, you can use the Windows Report Tool (Winrep.exe), to compile information about your computer, which you can then send to Microsoft support engineers. You start Windows Report from the Tools menu of the System Information Utility.

When you run the Windows Report Tool, it prompts you for the following information:

  • A description of the problem.

  • The results you expect once the problem has been addressed.

  • Steps to reproduce the problem.

The Windows Report Tool then creates a Microsoft System Information (*.nfo) file with your current system information, and collects other system files, including log and configuration files. The Windows Report Tool compresses these files into a cabinet (*.cab) file and prompts you to save it in C:\Windows\Helpdesk\Winrep. This *.cab file can be used by support engineers to help diagnose and resolve the problem. If you are discussing the problem with Microsoft Technical Support, they may ask you to send this file to Microsoft for further investigation.

If you cannot start Windows 98, you can use an MS-DOS-based tool, Dosrep.exe, to collect system files and compress them into a *.cab file.

Note

Dosrep.exe will work only with modems, and may not recognize ISDN adapters.

To start the Windows Report Tool, you can either click Start, then Run, and type winrep.exe, or you can use the Microsoft System Information Utility. If you want to start the Windows Report Tool with the System Information Utility, perform the following procedure:

  1. Click Start, point to Programs, Accessories, and then System Tools.

  2. Select System Information.

  3. On the Tools menu, click Windows Report Tool.

  4. Type the problem, the results you would normally expect, and the steps needed to reproduce the problem in the designated boxes.

  5. Click Next.

  6. Complete the form to submit the problem.

  7. Click Send.
