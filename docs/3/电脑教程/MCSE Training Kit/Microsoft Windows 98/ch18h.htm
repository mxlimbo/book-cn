Installing and Configuring Modems and Dial-Up Adapters

If you are using a modem to set up Dial-Up Networking, you must do the following:

  • Install and configure the modem.

  • Configure the dial-up adapter.

Figure 18.2 Configuring a modem

After you physically install a modem on your computer, you can use either the Modems icon in Control Panel or the Make New Connection Wizard in the Dial-Up-Networking folder to install the modem software.

If you want to install and configure a modem, perform the following procedure:

  1. In Control Panel, double-click the Modems icon.

    The Install New Modem Wizard starts.

  2. Click Next to instruct Windows 98 to detect your modem automatically.

    Windows 98 detects your modem and installs the appropriate drivers.

    -Or-

    Select Don't detect my modem; I will select it from a list, and then click Next.

    Windows 98 prompts you for the Manufacturer, Model, and COM port for your modem. Windows 98 then installs the appropriate drivers.

  3. Click Finish.

    The Location Information dialog box opens.

  4. Enter your country and area code information. If applicable, enter the number dialed to access an outside line. Select either tone or pulse.

  5. Click Finish.

  6. To configure the modem, double-click the Modems icon in Control Panel. The Properties dialog box appears.

  7. From the General tab, select the modem you want and click Properties (see Figure 18.2).

  8. You can change the port, modem speed, and other configuration information.

Note

Although you can also change modem properties from within a Win32 application, any changes that you make from that application will affect that application only. To make global changes, use Control Panel.

Configuring Modems

The default properties work for most modem connections. You should consult your modem documentation before changing any of the default properties. For some modems, you cannot modify all settings; if an option appears grayed out, your modem does not support the feature controlled by that option.

Configuring the Dial-Up Adapter

When you use a modem for Dial-Up Networking, a dial-up adapter appears in the list of installed network components in the Network option of Control Panel.

The dial-up adapter is a network adapter driver that interfaces with the Windows 98 communications device driver. The dial-up adapter driver shields the network components from the fact that communication is through a modem rather than through a network adapter.
