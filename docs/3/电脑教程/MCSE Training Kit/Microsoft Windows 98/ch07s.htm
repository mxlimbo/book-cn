Practice: Creating a Custom View for a Folder

In this practice, you will create a custom view for a folder.

    To create a custom view for a folder

    1. In My Computer, open the Program Files folder and maximize it.

    2. On the View menu, click Customize this Folder.

      The Customize This Folder Wizard appears.

    3. Select Create or edit an HTML document, and then click Next.

      A dialog box appears, indicating that your HTML editor will open.

    4. Click Next.

      Notepad opens with Folder.htt loaded.

    5. Close Notepad.

    6. Click Finish.

      The Program Files folder appears as a Web page.

    7. Maximize the window.

    8. Click a folder.

      The following information about the file is displayed on the left side of the window: File name, file type (and association, if appropriate), modified date and time, and, in some cases, a preview.

    9. Close all open windows.
