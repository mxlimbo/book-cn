Troubleshooting Modem Problems

If your modem is unable to dial the dial-up connection, you can use the Modemlog.txt file and the Modem Diagnostics program to troubleshoot.

You can record a log file of all AT (which means Attention, a command line prefix) commands sent to your modem by TAPI programs and the results returned by those commands. Windows 98 writes the file modemname.log to the C:\Windows folder. For security reasons, the file does not record specific telephone numbers or calling card numbers.

If you want to enable a modem log file, perform the following procedure:

  1. In Control Panel, double-click the Modems icon.

  2. On the General tab, click Properties.

  3. On the Connection tab, click Advanced.

  4. Select Append to log to activate this setting.

  5. Click OK.

The modem diagnostics program attempts to communicate with the selected modem to verify that the modem is connected properly and is receiving data. If the modem passes the test, one or more of the following is probably causing the problem:

  • The connection settings

  • The program used to connect the modem

  • The receiving modem

If you want to run the modem diagnostics program, perform the following procedure:

  1. Double-click the Modems icon.

  2. On the Diagnostics tab, click the COM port to which your modem is connected to select the modem.

  3. Click More Info.

This starts the modem diagnostics program.
