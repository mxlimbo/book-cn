Summary

The following information summarizes the key points in this chapter:

User Interface Enhancements

  • Significant improvements have been made to the Windows 98 user interface, including enhancements to the Start menu, Settings menu, and Find menu.

  • The Windows 98 taskbar now supports multiple toolbars. Toolbars can be moved and resized. You can also add toolbars to customize the desktop.

  • Windows 98 combines Windows Explorer and Internet Explorer into a single utility. The single Explorer enables you to navigate between local, network, Internet, and intranet content.

Web Integration

  • A Web style desktop is an option that adds the functions of a Web browser to the Windows 98 desktop.

  • Using the Web style desktop, you select objects with hover select (the pointer behaves the same way as when it is positioned over a hyperlink in Internet Explorer), and you start objects with single-click execution.

  • Windows 98 allows you to customize the characteristics of local and network folders.

Enabling Active Desktop

  • The Active Desktop integrates the Windows desktop with Internet Explorer, enabling you to put Web content— such as graphics, HTML documents, and ActiveX controls—directly on your desktop.
