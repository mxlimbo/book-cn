Lesson Summary

The Win32 Driver Model (WDM) provides a common set of drivers that work in Windows 95 OSR2.1, Windows 98, and future versions of Windows NT. WDM allows hardware developers to develop one device driver that will function in multiple operating systems.

Cross-platform compatibility results from WDM's layered architecture. The WDM architecture is composed of device class drivers, bus class drivers, minidrivers, virtualization drivers, and kernel services (Ntkern.vxd).
