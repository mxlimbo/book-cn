Lesson 3: Configuring and Managing a Local Printer

Once you've installed a local printer, you can configure it by setting such properties as ports for printer output, separator pages, and other printer-specific device options. This lesson describes the procedure for configuring a local printer and it also provides troubleshooting tips in the event of printing problems. This lesson also describes how to manage print tasks.

After this lesson, you will be able to:

  • Describe the configuration options for local printers.

  • Configure a local printer.

  • Manage a local printer.

Estimated lesson time: 30 minutes
