Sharing Resources with User-Level Access Control

Using File and printer sharing for NetWare Networks to share resources is similar to using File and Printer Sharing for Microsoft Networks to share resources, with the following exceptions:

  • In File and printer sharing for NetWare Networks, you must apply user-level access control. Because there is no option for share-level access control, a NetWare server must act as a security provider.

  • The World account allows access only to users who have an account on the security provider. This differs from File and Printer Sharing for Microsoft Networks, which allows access to unknown users through The World account.

Granting Permissions

Permissions for File and printer sharing for NetWare Networks are the same as permissions for File and Printer Sharing for Microsoft Networks. They just use different terminology. The following table matches each Windows 98 permission with its NetWare equivalent:

Windows 98 Permission NetWare Right
Read Files Read
Write to Files Write
Create Files Create
Delete Files Erase
Change File Attributes Modify
List Files File Scan
Change Access Control Access Control

Note

The Change Access Control permission has no effect in NetWare networks. You must change permissions at the Windows 98 server.
