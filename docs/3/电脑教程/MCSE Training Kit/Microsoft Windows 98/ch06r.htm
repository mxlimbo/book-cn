Lesson 3: Creating the Software Environment

After you establish your desired hardware and system settings, you may want to consider your software configuration. You can remove installed Windows 98 components at any time or add new components as you need them. You must also know how to install and remove applications. This lesson explains how to configure your software environment by installing and removing components and applications.

After this lesson, you will be able to:

  • Add and remove Windows components.

  • Install and remove applications.

Estimated lesson time: 15 minutes
