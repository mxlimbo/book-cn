Summary

The following information summarizes the key points in this chapter:

Viewing System Information

  • The System Information utility is a troubleshooting tool that enables you to view information about your computer's configuration. It also gives you a way to access several other Windows utilities, including the Windows Report Tool, Dr. Watson, and ScanDisk.

  • You can use the System Information utility to view files from other programs, such as the Windows Report Tool and Dr. Watson. You can also save and export System Utility information.

Performing Maintenance Tasks

  • ScanDisk is a disk analysis and repair program that in Windows 98 has been upgraded to recognize and fix issues with FAT32 drives and compressed drives. ScanDisk runs automatically after an improper shutdown unless it is disabled through Msdos.sys.

  • Windows 98 Disk Defragmenter can also store an application's executable file and its supporting files in a contiguous area of disk space, thus reducing access time for files on the disk.

  • Task Monitor monitors programs you use and records their disk access patterns during their start. It also records the number of times you use programs. Disk Defragmenter uses this information to favor more frequently used programs in optimizing the disk.

  • Disk Cleanup improves system performance by removing temporary and deleted files from a hard disk. Disk Cleanup automates the maintenance task of deleting excess files.

  • The Maintenance Wizard is a Windows 98 application that can speed up your applications, delete unnecessary files, and scan hard disks for errors. Schedule Tasks is a Windows 98 application that allows you to run any program at a desired date and time.

Using Web-based Troubleshooting Tools

  • Windows Update is a Web-based resource site that updates your computer's drivers and system files. When you install Windows 98, shortcuts to Windows Update are created on the Start and Settings menus.

  • From Windows Update, you can access Technical Support, where you can search through technical support articles.

  • You can use Windows Report Tool to send information about your computer via the Internet to Microsoft support engineers.
