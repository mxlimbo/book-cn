Managing and Tuning Compressed Disks

DriveSpace 3 can create an empty compressed drive from free space on an existing partition, or a compressed drive containing the data from an existing partition.

Compression Ratio

By default, DriveSpace 3 assumes a compression ratio of 2:1. Because the largest FAT16 partition is 2 GB, the default CVF size is 1 GB. You can adjust the expected compression ratio in the DriveSpace 3 utility based on the type of data you will store on the compressed drive. For example, Microsoft Word documents can compress to 1/16th of their original size. Others, such as Video for Windows files (.avi files), do not compress as well.

By decreasing the expected compression ratio, your CVF can be larger than 1 GB. However, DriveSpace 3 still limits the amount of data in the drive to 2 GB. You can see how much a file is compressed by using the Change ratio command on the Advanced menu of DriveSpace 3. You can also use the dir/c command at the MS-DOS command prompt.

CVF and the Host Drive

When you compress a drive, the CVF completely fills the host drive that is created to hold the CVF. With DriveSpace 3, you can move free space from the CVF to the host drive. The host drive free space is not compressed, and files saved to it will not be compressed.

For example, if you compress drive D, DriveSpace 3 creates host drive H to hold the compressed drive D. If you move 200 MB free space from drive D to drive H, you can save files to drive H and those files will not be compressed. Any files you save to drive D will be compressed.

Important

To perform this procedure you must have a compressed drive on your system; however, you may decide not to adjust free space based on your needs.

If you want to adjust free space:

  1. Click Start. Point to Programs, Accessories, System Tools, and then click DriveSpace 3.

  2. Select the compressed drive or the host drive.

  3. In the Drive menu, click Adjust Free Space.

    A graphic display of the compressed drive and host drive shows the size of the drives and the amount of free space on each.

    You move free space between the compressed drive and host drive with a slider bar at the bottom of the window. When you move the slider, the graphic display shows the changes.

  4. Move the slider to adjust the free space between the compressed drive and host drive. Click OK.

Compression Agent

Compression Agent (Cmpagent.exe) is a DriveSpace 3 utility on the System Tools menu with which you can choose:

  • Whether specified files should be in UltraPack format, and whether to downgrade a file from UltraPack format. If you require maximum system performance, remember that files compressed in UltraPack format are much slower to decompress. For slower computers, consider using Compression Agent with Scheduled Tasks to compress files during idle time rather than when you save files.

  • Whether individual files, folders, or file types should be compressed, and if so, with what method.

  • Whether to use compression for the rest of your files and, if so, which type.

With Compression Agent, you can write files to a compressed drive without compressing them. When you start Compression Agent, it evaluates the files on the drive and compresses only the types of files you specified, using the type of compression you specified.

Important

Compression Agent is compatible only with DriveSpace 3 compressed drives.
