Dual Booting Windows 98

In some cases you may want to have the option of running more than one operating system on your computer. This process is known as dual booting.

Dual Booting Windows 98 and Windows NT

Dual booting Windows 98 and Windows NT is not recommended because Windows 98 cannot access Windows NT File System (NTFS) partitions, and Windows NT cannot access the FAT32 file system.

If you require a dual-boot configuration with Windows 98 and Windows NT, then you must adhere to the following guidelines:

  • Install your programs in each operating system.

    In order to run applications in both operating systems, you must install each application in both Windows 98 and Windows NT. Any application written exclusively for one operating system can only be installed in the operating system for which it was written.

  • Windows NT must be set up and configured to dual boot before you set up Windows 98.

  • Windows 98 must be installed in a directory other than the Windows NT directory.

  • The C drive must be a FAT16 partition with enough free space for the Windows 98 startup files.

    Windows 98 can be installed on any drive that has enough free space and is a FAT partition, but the startup files are always installed on the C drive.

  • To install Windows 98, select Windows 95 from the Boot Loader Operating System Selection menu that appears at system startup. If Windows 95 does not appear on the menu but MS-DOS does, select MS-DOS.

Dual Booting Windows 98 With Other Operating Systems

You can dual boot Windows 98 with MS-DOS 5.0 or later, using the f4 boot-to-previous-operating-system feature, if drive C is FAT16. You cannot dual boot Windows 98 and Windows 95 because they use the same boot files (Io.sys, Command.com, and Msdos.sys).
