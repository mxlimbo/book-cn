Implementing Mandatory User Profiles

The contents of a mandatory user profile are the same as those of other user profiles, but the User.dat registry file is named User.man instead.

If you want to implement a mandatory user profile on a network, perform the following procedure:

  1. Customize the desired settings on any computer running Windows 98.

  2. Copy the folders and the User.dat file from the C:\Windows\Profiles\<user name> directory to the server directory where profiles are stored.

  3. In the network server directory, rename User.dat to User.man.
