Optional Practice: Installing a Modem and Configuring Locations

In this practice, you will install a modem, and create a new dialing location.

Exercise 1
Installing a Modem

In this exercise, you will install a modem using the Add Modem Wizard.

    To install a modem

    1. Click Start, point to Settings, and click Control Panel.

      Control Panel appears.

    2. Double-click the Modems icon.

      The Install New Modem dialog box appears.

    3. Select Don't detect my modem; I will select it from a list, and then click the Next button.

    4. In the Manufacturers column, select a modem type.

    5. In the Models column, select the modem you want to install and then click Next.

    6. Select the appropriate COM port, and then click Next.

      Windows 98 installs your modem.

    7. Click Finish.

      The Modem Properties dialog box appears with the modem you installed appearing in the list of installed modems.

    8. Close the Modem Properties dialog box.

Note

Windows 98 can also install your modem automatically using the Add Hardware Wizard.

Exercise 2
Creating a New Location

In this exercise, you will create a new dialing location.

    To create a new dialing location

    1. In Control Panel, double-click the Modems icon.

      The Modems Properties dialog box appears.

    2. Under Dialing Preferences, click Dialing Properties.

      The Dialing Properties dialog box appears.

    3. Click New.

      A message telling you a new location was created appears.

    4. Click OK.

    5. Configure your new location with the following properties:

      Field Value
      I am dialing from Hotel
      I am in this country/region: Any country other than your own
      Area code: 567
      To access an outside line – local 9
      To access an outside line – long distance 8
      To disable call waiting, dial: Checked *70
      For long distance calls, use this calling card: Checked AT&T via 1-800-321-0288

    6. Click Calling Card.

      The Calling Card dialog box appears.

    7. In Personal ID Number (PIN Number), type 1234567.

    8. Click OK to close the Calling Card dialog box.

    9. Click OK to close the Dialing Properties dialog box.

    10. Click OK to close the Modem Properties dialog box.
