Overview of Internet Explorer 4.0

Windows 98 is integrated with Internet Explorer 4.0 (see Figure 17.1), which incorporates a suite of Internet communication tools, including an easy-to-use Web browser and ActiveX controls.

Figure 17.1 The Internet Explorer 4.0 Web browser

ActiveX Controls

ActiveX is a technology that enables software components to interact with each other in a networked environment, regardless of the language in which they were created. ActiveX controls are software components that provide a user interface. With ActiveX controls, your Web site can include multimedia effects, interactive objects, and sophisticated applications that create a user experience comparable to those of high-quality CD-ROM titles.

Additional Features

With Internet Explorer 4.0, you can also do the following:

  • Restrict access to certain sites based on standard ratings.

  • Establish Secure Sockets Layer (SSL) sessions with commercial sites. SSL uses encryption to enable secure financial transactions on the World Wide Web.

  • Run Java applets. Java applets add multimedia effects, such as background music, animations, and interactive games.

  • Customize the browser to meet the needs of your organization (with the Internet Explorer Administration Kit).

Note

Because Internet Explorer is constantly evolving, you should check frequently for product updates. For the most recent list of features and updates, see http://www.microsoft.com/ie/.
