When to Install from MS-DOS

Run Setup from MS-DOS when:

  • You want to change the settings of an existing Windows 95 configuration.

  • Setup fails when you run it from Windows 95.

  • You want to install Windows 98 into a new directory.

  • You want to install Windows 98 on a new computer or into a newly formatted hard disk.

  • You want to install Windows 98 and your current Windows system is Windows 3.1x.
