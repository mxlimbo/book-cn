Practice: Installing a Local Printer in Windows 98

In this practice, you will use the Add Printer Wizard to install a local printer.

    To install a printer

    1. Open My Computer.

    2. Open the Printers folder.

    3. Double-click Add Printer.

      The Add Printer Wizard will guide you through the printer installation process. After you add a printer, an icon for that printer appears in the Printers folder. A checkmark appears on the icon for the default printer.

Tip

You can also locate the Printers folder through Control Panel or by clicking Start, Settings, and Printers.
