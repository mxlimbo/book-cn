Lesson Summary

Troubleshooters are interactive help files that help you find the source of problems and suggest corrective action. Windows 98 provides the Startup and Shutdown Troubleshooter, the Hardware Conflict Troubleshooter, and Troubleshooters for networking, modem, printing, PC cards, sound, and memory.

Device Manager lists all of the installed devices on your computer and provides information about the status, driver, and resources for each device. The Automatic Skip Driver Agent (ASD) identifies drivers that fail to load or operations that fail at startup. ASD marks the failed device or operation to be bypassed on subsequent startups.
