Printing to NetWare Print Servers

You can submit print jobs from your Windows 98 computer to a NetWare print server. To do this, you set up a printer using either the Add Printer Wizard or Point and Print.

You can install a NetWare printer though the Add Printer Wizard or simply by dragging the printer icon from Windows Explorer to your desktop. You must specify the location and name of the required driver unless the NetWare server is configured for Point and Print.

Point and Print

With Point and Print enabled, you can install a network printer locally without user intervention. Point and Print occurs automatically when connecting to a Windows 98 print server, or to a Windows NT 4.0 print server that is sharing the proper drivers. In order to use Point and Print with a NetWare server, however, you must add the following information to the server's bindery:

  • Printer name and configuration

  • Names of the printer driver files

  • Location of the printer driver files

You can add this information to your NetWare server's bindery through Windows explorer.

Note

Point and Print works with bindery or bindery emulation only.

If you want to configure a printer for a NetWare print queue to support Point and Print installation, perform the following procedure:

  1. Log on as supervisor or equivalent.

  2. Open Network Neighborhood.

  3. Right-click a print queue on the server and click Properties.

  4. Click the Point and Print Setup tab.

  5. Click Enable Point and Print.

  6. In the Path from which clients can copy drivers for the printer box, type \\netware_server\sys\public\printer_folder and then click Set Printer Model.

    The Select Device dialog box appears.

  7. From the list of manufacturers, select the manufacturer of your printer.

  8. From the list of models, select the model of your printer, and then click OK.

  9. Click OK again.

    The NDS Administration Dialog box appears. Windows 98 prompts you with the following message:

    If this task has not been done on the current NDS tree, it will modify the NDS schema database. If so, you must have administrator rights for the [Root] object to continue. Do you want to continue?

  10. Click OK.

After this process is complete, users can install the NetWare printer with minimal user input.

Configuring NetWare Printers

When you install a NetWare printer on your Windows 98 computer, the printer's Properties dialog box displays an additional tab for Capture Settings, from which you can configure:

  • Form feeds

  • Banner pages

  • The number of copies to print

  • Additional properties for MS-DOS-based programs
