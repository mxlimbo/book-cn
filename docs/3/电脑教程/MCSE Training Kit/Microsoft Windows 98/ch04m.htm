Lesson 3: Supporting Universal Serial Bus Devices

The Universal Serial Bus (USB) is a low- to mid-speed bus that you can use for a variety of peripherals, including mouse devices, keyboards, joysticks, scanners, audio devices, and digital cameras.

This lesson explains how the USB works, what happens when a USB device is added, and how to troubleshoot problems that might occur.

After this lesson, you will be able to:

  • Describe how USB functions.

  • Install and configure a USB device.

  • Describe USB support issues and their solutions.

Estimated lesson time: 15 minutes

Before you continue with the lesson, run the Chap4a.exe animation located in the Animations folder on the Supplemental Course Materials CD-ROM that accompanies this book. This animation provides an overview of Windows 98 support for the USB.
