Lesson 2: Web Integration

Windows 98 lets you apply a Web browser look and feel to your desktop. This lesson discusses the different desktop styles in Windows 98. It also discusses options for selecting files and folders and launching programs.

After this lesson, you will be able to:

  • Configure the Windows 98 desktop for classic style or Web style.

  • Choose between hover select and single-click for the way you select files and folders.

  • Customize folders.

Estimated lesson time: 30 minutes
