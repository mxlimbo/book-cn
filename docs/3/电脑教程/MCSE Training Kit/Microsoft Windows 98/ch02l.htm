Phase 2—Collecting Information About Your Computer

After you accept the License Agreement, Setup checks your system (see Figure 2.3), and prompts you to save existing system files, select Internet channels, and create an emergency startup disk.

Figure 2.3 Collecting information about your computer

Checking Your System

Setup performs the following information collection tasks:

  • ScanDisk runs in protected mode, checking the directory structure, file allocation table, and file system.

    In protected mode, ScanDisk does not perform a surface scan; therefore, the disk is not checked for physical errors. If ScanDisk finds any problems, Setup stops and allows you to run ScanDisk manually.

  • Setup checks the registry for corrupt files.

    Setup attempts to repair any corrupt files. If the repair is successful, Setup continues. If the repair is unsuccessful, you are prompted first to run Setup from MS-DOS, and then to install Windows 98 in a new directory.

  • Setup prepares your Windows directory and verifies that your computer has enough disk space to install Windows 98.

    If you do not have enough disk space, the Setup Wizard prompts you to free up hard disk space and restart Setup.

Saving Existing System Files

After Setup has finished checking your system, you have the option of saving your existing MS-DOS and Windows system files (see Figure 2.4). It is recommended that you select Yes to save your existing files in case you need to restore your Windows 95 configuration when uninstalling Windows 98. If you select No, you will not be able to uninstall Windows 98. Instead, you will have to delete the Windows 98 directory and reinstall the previous operating system, as well as any Windows programs.

Figure 2.4 Save Existing System Files dialog

If you choose to save your existing configuration files, Setup creates the following files:

  • Winundo.dat

    The Winundo.dat file containes all of the files necessary to restore your old system settings. This file may be as large as 75 MB.

  • Winundo.ini

    The Winundo.ini file contains information necessary to restore the files contained in the Winundo.dat file.

Selecting Internet Channels

Setup prompts you to select a country for Internet Channels. The channels are tailored for various countries to provide information of popular interest for the people of that country. You do not have to live in a particular country to subscribe to that country's channels.

Emergency Startup Disk Files

Setup prompts you to create an Emergency Startup Disk (ESD). The ESD, also called the Windows 98 Startup Disk, is a system disk with a collection of real-mode Windows 98 commands and utilities that can assist in troubleshooting. If your hard disk will not boot up after Setup, use the ESD to restart your system.

Note

For a list of files contained on the ESD, refer to Appendix C, "Emergency Startup Disk Contents."

The ESD can support both IDE and SCSI (Adaptec and Mylex/Buslogic) CD-ROM drives. If you use the ESD to boot the computer and choose to load the real-mode CD-ROM drivers but cannot access a compact disc in the drive, it is probable that the generic drivers supplied by Windows 98 will not work with your drive or controller. If this happens, you must copy the correct drivers for your CD-ROM drive and controller to the ESD, and then edit the Config.sys file on the ESD so that it will load the drivers. Consult the documentation for your CD-ROM drive and controller to get the correct files and configuration settings.

After the files are copied, you are prompted to remove the disk and then to continue with Setup.
