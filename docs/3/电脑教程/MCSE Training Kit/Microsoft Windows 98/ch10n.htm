Practice: Using the System Configuration Utility

In this practice, you will use the System Configuration Utility to back up your system configuration files, limit the amount of RAM available to Windows 98, and configure Windows 98 to display the Startup menu.

Exercise 1
Backing Up System Configuration Files

In this exercise, you use the System Configuration Utility to create a backup of your system configuration files. You always should back up your system configuration files before making any modifications.

    To backup system configuration files

    1. Click Start, point to Programs, Accessories, System Tools, and then click System Information.

      The Microsoft System Information window appears. View the data under System Information in the right-hand pane.

    2. How much RAM does Windows 98 have available?

      Answer

    3. Click Tools, then click System Configuration Utility.

      The System Configuration Utility window appears.

    4. On the General tab, click Create Backup.

      System Configuration Utility creates copies of Config.sys, Autoexec.bat, Win.ini, and System.ini using the file extension .pss.

      The Create Backup dialog box appears indicating that your files have been backed up.

    5. Click OK to close the Create Backup dialog box.

Note

Do not close the System Configuration Utility dialog box.

Exercise 2
Configuring Advanced Startup Options

In this exercise, you use System Configuration Utility to limit the amount of memory available to Windows 98 and to display the Startup menu at boot time.

    To specify Windows 98 startup options

    1. On the General tab, click Advanced.

      The Advanced Troubleshooting Settings dialog box appears.

    2. Select Limit memory to and change the amount of memory to 4 MB.

    3. Select Enable Startup Menu to specify that Windows 98 should display the Startup menu at boot time.

    4. Verify that no other options are selected, then click OK.

      The System Configuration Utility dialog box appears.

    5. Click OK.

      The System Settings Change dialog box appears prompting you to restart your computer.

    6. Click Yes.

      Your computer restarts.

    To verify system configuration changes

    1. The Microsoft Windows 98 Startup Menu appears. Select Normal and then press ENTER.

      An error message informs you that there is a memory problem. You are prompted to press any key to continue.

    2. Press the ENTER key.

    3. Reboot the computer.

      Notice that the Microsoft Windows 98 Startup Menu appears.

    4. Select Safe Mode and then press ENTER.

      Safe mode will override the 4 MB setting to allow Windows 98 to start up. A message box informs you that Windows is running in safe mode.

    5. Click OK.

    6. Click Start and then click Run.

      The Run dialog box appears.

    7. Type msconfig.exe in the Open box.

      The System Configuration Utility appears.

    8. Click Advanced.

      The Advanced Troubleshooting Settings dialog box appears.

    9. Set the Limit memory to box to 16 MB.

    10. Click OK twice to close the configuration utility.

      The System Settings Change dialog box appears prompting you to restart your computer.

    11. Click Yes.

      Your computer restarts and displays the startup menu.

    12. Select Normal mode from the Startup menu and press ENTER.

Exercise 3
Restoring System Configuration Files

In this exercise, you use System Configuration Utility to restore your system configuration files from the backup you made in Exercise 1.

    To restore system configuration files

    1. Click Start, point to Programs, Accessories, System Tools, and then click System Information.

      The Microsoft System Information window appears. Notice the data under System Information in the right-hand pane.

    2. How much RAM does Windows 98 have available?

      Answer

    3. Click Tools, then click System Configuration Utility.

      The System Configuration Utility window appears.

    4. Click Restore Backup.

      The Restore Backup dialog box appears prompting you to verify that you want to overwrite your existing system configuration files.

    5. Click Yes to restore your previous configuration.

      The Restore Backup dialog box appears indicating that your files have been restored.

    6. Click OK to close the Restore Backup dialog box.

      The System Configuration Utility dialog box appears.

    7. Click OK to close System Configuration Utility.

      The System Settings Change dialog box appears prompting you to restart your computer.

    8. Click Yes.

      Your computer restarts.

    To verify system configuration changes

    1. Does the Microsoft Windows 98 Startup menu appear? Why or why not?

      Answer

    2. Select Normal and then press ENTER.

      Windows 98 starts.

    3. Log on to Windows 98.

    4. Click Start, point to Programs, Accessories, System Tools, and then click System Information.

      The Microsoft System Information window appears.

    5. View the data in the right pane.

      Was your previous RAM configuration restored? Why or why not?

      Answer

Note

Do not close the Microsoft System Information window.

    To disable the Windows 98 Startup menu

    1. Start the System Configuration Utility.

    2. On the General tab of the System Configuration Utility window, click Advanced.

      The Advanced Troubleshooting Settings dialog box appears.

    3. Clear the Enable Startup Menu box.

      Clearing this box specifies that Windows 98 should not display the Startup menu at boot time.

    4. Verify that no boxes are selected, and then click OK.

      The System Configuration Utility dialog box appears.

    5. Click OK.

      The System Settings Change dialog box appears prompting you to restart your computer.

    6. Click Yes.

      Your computer restarts.

    To verify system configuration changes

    1. Does the Microsoft Windows 98 Startup menu appear?

      Answer

      Microsoft Windows 98 starts.

    2. Log on to Windows 98.
