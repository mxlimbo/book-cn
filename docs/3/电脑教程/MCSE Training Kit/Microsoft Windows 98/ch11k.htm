Installing Additional Components During Setup

You can add additional services during installation; for example, you can add Remote Registry service so that you can enable remote administration automatically. Use the INF Installer Tool to add components during setup. INF Installer is on the Windows 98 CD-ROM in the \tools\reskit\infinst folder.

In Windows 98, additional software components typically include a setup information file (*.inf file) that scripts the installation procedure for that component. INF Installer also edits the custom.inf file that batch.exe creates, so that custom.inf includes directions to install the new component(s).

If you are integrating an application, INF Installer adds an entry to the [Optional Components] section of custom.inf. If you are adding a network service, INF Installer modifies the Services entry in the [Network] section in custom.inf.

If you want to add a Hardware component to the setup.exe script:

  1. Create a custom.inf file using the Microsoft Batch 98 tool.

  2. Insert the Windows 98 CD-ROM in your computer.

  3. When the Microsoft Windows 98 window appears, close the window.

  4. Start Windows Explorer.

  5. Copy the D:\tools\reskit\infinst\infinst.exe (where D: is assumed to be the drive letter for the CD-ROM) to your \Windows folder.

  6. Double-click infinst.exe.

    The Inf Installer dialog box appears.

  7. Remove your Windows 98 CD-ROM and insert the CD-ROM or floppy disk that contains the new hardware driver.

  8. In the Inf Installer dialog box, type the path of the new hardware inf in the Inf to add to Windows 98 Setup box.

  9. Type the Windows 98 setup.exe path in the Windows 98 Setup.exe box.

  10. Click Add Inf to add the new hardware driver. A status dialog box will appear displaying the progress of the installation.
