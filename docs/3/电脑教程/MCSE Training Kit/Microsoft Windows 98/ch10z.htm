Customizing System File Checker

You can customize the operation of System File Checker through the Settings button in the System File Checker dialog box. The System File Checker Settings window has the following tabs and options:

  • Settings. Configure backup and log file options.

  • Search Criteria. Specify the files and folders to scan. You can also limit the scan to files with a specific extension if you know the type of file that has been corrupted, but are unsure of exactly which file has been corrupted. This reduces the amount of time that the scan takes.

  • Advanced. Choose a different verification data file to use, create a new verification data file, or restore the original Default.sfc. You may want to create a verification file for each type of file. Then, if you suspect that a particular file type has been corrupted, you can load the appropriate verification file.
