Lesson 6: Uninstalling Windows 98

If you need to remove Windows 98, you can perform an Uninstall, provided you selected Yes to save system files during Setup. You can also perform an Uninstall to remove Windows 95 once you've successfully installed Windows 98 and are confident that your programs run on the new operating system. This lesson discusses procedures for uninstalling Windows 98 and Windows 95.

After this lesson, you will be able to:

  • List the requirements for uninstalling Windows 98.

  • Uninstall Windows 98.

  • Remove Windows 98 system files.

Estimated lesson time: 10 minutes
