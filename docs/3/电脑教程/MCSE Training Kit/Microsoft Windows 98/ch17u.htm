Configuring an Internet Explorer Kiosk

An Internet kiosk is a publicly available workstation intended for dedicated Internet access. With Internet Explorer and strict system policies, you can use a Windows 98 computer as an Internet kiosk.

Figure 17.6 Starting Internet Explorer in kiosk mode

You can start Internet Explorer in kiosk mode (see Figure 17.6). In kiosk mode, Internet Explorer opens without a menu bar, toolbar, or URL address bar. To keep end users from opening another instance of Internet Explorer and browsing freely, kiosk mode also hides the Windows 98 taskbar and Start button.
