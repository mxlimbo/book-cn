Prerequisites

This course requires that students meet the following prerequisites:

  • Operational understanding of computer hardware, such as memory, hard disks, and CPUs.

  • Have experience installing, configuring, and supporting Windows 95, or

  • Successfully complete Course 798, Supporting Microsoft Windows 95.

  • Have experience installing, configuring and supporting networks, or

  • Successfully complete the Networking Essentials course.

  • Experience with the Windows 95 or Windows NT 4.0 user interface.
