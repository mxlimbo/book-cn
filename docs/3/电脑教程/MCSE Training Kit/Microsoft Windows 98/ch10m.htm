Using the Microsoft System Configuration Utility

The System Configuration Utility replaces Sysedit from previous Windows versions and provides a graphical interface to control the Windows 98 startup environment. With System Configuration Utility, you can backup and edit system files, and control and troubleshoot startup through a process of elimination with checkboxes.

Creating Backup Files

You can create backups of the Win.ini, System.ini, Autoexec.bat, and Config.sys with the System Configuration Utility. Backup creates a copy of each file, gives each a *.pss extension, and puts the file in the same folder as the original file. You can restore these files to their original name and configuration using the Restore Backup option.

Controlling Startup

Use the System Configuration Utility to control your startup by selecting one of the three options on the General tab.

  • Normal startup loads all device drivers and software.

  • Diagnostic startup creates a clean environment for troubleshooting by providing the Windows 98 Startup Menu in MS-DOS format when your computer restarts. Select Step-by-step confirmation from the menu options to choose the files to process.

  • Selective startup provides options to enable and disable individual startup files through checkboxes.

Troubleshooting Startup

When problems occur with startup, use the diagnostic or selective startup to determine which system file is causing the problem. Use System Configuration Utility (see Figure 10.4) to edit Config.sys, Autoexec.bat, System.ini, Win.ini, and the Startup Group files as necessary to locate the specific driver or software that is interfering with startup.

Figure 10.4 The System Configuration Utility

The following table describes the tabs in System Configuration Utility:

This Tab Enables or Prevents
General The loading of all or specific device drivers and software based on a selected startup option.
Config.sys The loading of specific lines in the Config.sys file. Checking a line item edits the Config.sys file and adds REM TSHOOT: to remark out the line.
Autoexec.bat The loading of specific lines in the Autoexec.bat file. Checking a line item edits the Autoexec.bat file and adds REM TSHOOT: to remark out the line.
System.ini The loading of specific lines in the System.ini file. Checking a line item edits the System.ini and adds; REM TSHOOT: to remark out the line.
Win.ini The loading of specific lines in the Win.ini file. Checking a line item edits the Win.ini and adds REM TSHOOT: to remark out the line.
Startup Specific items loaded by the operating system on startup. The information in the Startup tab comes from the registry.
