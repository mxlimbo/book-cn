Chapter 6

Configuring the Windows 98 Environment

About This Chapter

In Windows 98, most hardware, system, and software settings are performed dynamically and automatically by the operating system. When it is necessary to manually add, delete, or modify configuration settings, the Control Panel is generally used. In this module, you learn to add, configure, and remove hardware and software. You also learn to configure system settings for optimum performance. Finally, you learn how to use the Registry Editor to modify settings to the registry.
