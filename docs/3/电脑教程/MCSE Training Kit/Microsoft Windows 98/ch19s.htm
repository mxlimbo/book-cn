Recommended Practices

Here are some recommendations for maintaining and troubleshooting Windows 98 in a network environment:

  • Check Windows Update weekly for new drivers and troubleshooting tips.

  • Use the Windows 98 Troubleshooters to diagnose and solve technical problems with Windows 98.

  • If you want to administer a remote computer using System Policy Editor, Registry Editor, or System Monitor, you must install the Microsoft Remote Registry service on both your computer and the remote computer. Although the remote computer requires user-level access control, it does not require file and printer sharing services.
