Installing and Configuring Client for NetWare Networks

Regardless of the version of NetWare on your network, you must install and configure Client for NetWare Networks to access resources on a NetWare server.

Before installing Client for NetWare Networks, remove any real-mode NetWare requestor software, such as NETX (the workstation shell for NetWare versions 3.x) and VLM (virtual loadable module, the workstation shell for NetWare 4.x).

If you want to install Client for NetWare Networks, perform the following procedure:

  1. In Control Panel, double-click the Network icon.

  2. From the Configuration tab, click Add.

  3. Click Client, and then click Add.

  4. Click Microsoft, and then click Client for NetWare Networks.

  5. Click OK twice.

  6. Click Yes when prompted to restart the computer.

Tip

If you do not install IPX/SPX-compatible Protocol, it installs automatically with Client for NetWare Networks.

After you install Client for NetWare Networks, you may need to configure some parameters.

Configure This Property on the General Tab In Order To
Preferred server Designate a server to attach to when starting Windows 98.
First network drive Designate the first drive letter to use when mapping network drives from NetWare command-line utilities.
Enable logon script processing Enable Nwlsproc.exe, a 32-bit NetWare logon script processor that executes the user's logon script on the preferred server.

Configure This Property on the Advanced Tab In Order To
Preserve Case Ensure the password retains its uppercase and lowercase settings. Otherwise, the password reverts to all uppercase, and prevents the user from logging on.

If you want to configure Client for NetWare Networks, perform the following procedure:

  1. Double-click the Network icon and select Client for NetWare Networks.

  2. Click Properties.

  3. On the General tab (see Figure 14.3) select the parameters you want to configure.

  4. Click OK.
