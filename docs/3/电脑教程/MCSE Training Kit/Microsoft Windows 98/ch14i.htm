Configuring Service for NetWare Directory Services

If your network includes NetWare 4.x servers running NetWare Directory Services (NDS), you must install Service for NetWare Directory Services.

Service for NetWare Directory Services requires two components: IPX/SPX-compatible Protocol and Client for NetWare Networks. If you do not install those components separately, Windows 98 installs them automatically with Service for NetWare Directory Services.

Service for NetWare Directory Services has two configurable parameters. The following table defines the parameters for each property:

Configure This Property In Order To
Preferred tree Specify the name of the NDS tree to which you log in.
Workstation default context Specify the context (location in the NDS tree) where your user object is located. If a user with a different context needs to log on at your computer, he or she must log on with his or her logon name.

If you want to install and configure Service for NetWare Directory Services, perform the following procedure:

  1. In Control Panel, double-click the Network icon.

  2. On the Configuration tab, click Add.

  3. Select Service and click Add.

  4. Select Microsoft and select Service for NetWare Directory Services.

  5. Click OK.

  6. On the Configuration tab, select Service for NetWare Directory Services.

  7. Click Configuration.

  8. Select the property you want to configure.

  9. Click OK.
