Lesson 2: Creating Policies With System Policy Editor

This lesson discusses using System Policy Editor (Poledit.exe) to create and modify system policy files. With this Windows 98 tool, you establish computer configurations for users, groups, and computers by selecting registry settings that will overwrite previous settings. System Policy Editor in Windows 98 simplifies the process by displaying registry settings in a graphical template.

After this lesson, you will be able to:

  • Install the System Policy Editor.

  • Use templates for system policies.

  • Implement system policies on networks.

  • Implement system policies on a stand-alone computer.

  • Describe troubleshooting issues and their solutions.

Estimated lesson time: 60 minutes
