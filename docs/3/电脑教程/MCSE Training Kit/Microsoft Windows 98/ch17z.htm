Overview of Personal Web Server

The Microsoft Personal Web Server network file and application server transmits information and files using HTTP.

With Personal Web Server, you can:

  • Publish Web pages on your corporate intranet, or develop and test a Web site before hosting the site on an Internet Service Provider.

  • Publish Web pages with ActiveX controls.

  • Monitor traffic on your Web site.

Personal Web Server Features

Microsoft Personal Web Server for Windows 98 is an easy-to-use and easy-to-manage Web server that supports the following features:

  • HTTP service, which is the protocol you use to connect to Web pages.

  • A Guest Book and Drop Box, which you can create to interact with your visitors.

  • Active Server Pages, a scripting environment you can use to create and run Web server applications.

  • Visual Basic Scripting, a language used to create Active Server Pages.

  • Java Scripting, another language used to create Active Server Pages.

Using Personal Web Server in Small Networks

Microsoft designed Personal Web Server for small-scale, peer-to-peer networks, or for a corporate intranet. Personal Web Server has a connection limit of 10 users. It is optimized for interactive workstation use and does not have the system requirements of a full Web server, such as Microsoft Internet Information Server (IIS).

IIS supports an unlimited number of users, offers complete security of your web site, and also includes a file transfer protocol (FTP) service. Note, however, that IIS runs only on a Windows NT 4.0 Server.
