Configuring a Dialing Location

The Telephony Application Programming Interface (TAPI) within the Windows 98 operating system makes it possible for you to configure telephone-dialing properties centrally for the entire system. For example, you can create dialing locations that define the dialing requirements for your physical location, including the area code, dialing prefix, and calling card information (see Figure 18.5).

Figure 18.5 Setting Telephony Dialing Properties

Applications for which you can use dialing locations include:

  • The Microsoft Network (MSN)

  • Outlook Express

  • HyperTerminal

  • Dial-Up Networking

You might create two locations on a notebook computer:

  • "Home" with an area code of 503, and disabled call waiting.

  • "Office" with an area code of 206, and with a "9," required to access an outside line.

Then, when using the computer at home, you can use the "Home" location, and when using the computer at work, you can use the "Office" location.

If you want to configure a dialing location, perform the following procedure:

  1. In Control Panel, double-click the Telephony icon.

  2. On the My Locations tab, click New to open a new location.

  3. Type the name, country, and area code for your new location.

  4. Enter any needed information about outside line access, call waiting, or dialing method.

  5. If you are calling using a calling card, click Calling Card to enter information about your calling card, then click OK.

  6. Click OK.

A file called Telephon.ini in the C:\Windows folder stores dialing properties; they are not stored in the Registry.
