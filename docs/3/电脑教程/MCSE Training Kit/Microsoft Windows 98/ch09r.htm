Lesson 3: Using Web-based Troubleshooting Tools

Windows 98 provides Web-based resources that assist you in accessing technical support and obtaining the latest versions of device drivers and system files. This lesson provides an overview of Windows Update, Windows Update Technical Support, and Windows Report Tool.

After this lesson, you will be able to:

  • Use Windows Update to update drivers and system files.

  • Use Windows Update to access Windows Technical Support.

  • Use Windows Report Tool to collect information about system problems.

Estimated lesson time: 10 minutes
