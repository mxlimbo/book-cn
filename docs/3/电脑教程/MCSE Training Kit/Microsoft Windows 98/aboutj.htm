Chapter and Appendix Overview

This self-paced training course combines notes, hands-on procedures, multimedia presentations, and review questions to teach you about configuring and supporting Windows 98. It is designed to be completed from beginning to end, but you can choose a customized track and complete only the sections that interest you. If you choose the customized track option, see the "Before You Begin" section in each chapter. Any hands-on procedures that require preliminary work from preceding chapters refer to the appropriate chapters.

The book is divided into the following chapters:

  • The "About This Book" section contains a self-paced training overview and introduces the components of this training. Read this section thoroughly to get the greatest educational value from this self-paced training and to plan which lessons you will complete.

  • Chapter 1, "The Windows 98 Operating System," presents an overview of the enhancements to Windows 95 provided by Microsoft Windows 98.

  • Chapter 2, "Installing Windows 98," provides an overview of the Microsoft Windows 98 installation process.

  • Chapter 3, "Windows 98 File System Support," discusses how to partition a hard disk, as well as the capabilities and limitations of FAT16 and FAT32 file systems.

  • Chapter 4, "Windows 98 Hardware Support," discusses two Windows 98 hardware support technologies, the Win32 Driver Model (WDM) and OnNow Power Management.

  • Chapter 5, "Windows 98 System Architecture," introduces key aspects of the Windows 98 architecture that affect operating system functionality and performance.

  • Chapter 6, "Configuring the Windows 98 Environment" presents techniques to add, configure, and remove hardware and software.

  • Chapter 7, "Configuring the Windows 98 Desktop" discusses the interface enhancements provided by Microsoft Windows 98, the integration of Internet Explorer 4.0 in Windows 98, and the role of Internet Explorer 4.0 on the Windows 98 desktop.

  • Chapter 8, "Printing in Windows 98," presents an overview of the Windows 98 print subsystem. It also describes the procedures for installing, configuring, managing, and supporting local printers in Windows 98.

  • Chapter 9, "Maintaining Windows 98 in a Stand-alone Environment," describes the tools in the context of the maintenance and troubleshooting tasks they perform in a stand-alone environment.

  • Chapter 10, "Troubleshooting Windows 98 in a Stand-alone Environment," presents information about the numerous tools provided by Windows 98 to assist you with troubleshooting system problems.

  • Chapter 11, "Installing Windows 98 in a Network Environment" discusses how to install Windows 98 in a network environment. It also discusses issues you must consider when choosing a method for installation and the automation features that Windows 98 provides to make multiple installations easier.

  • Chapter 12, "Configuring Windows 98 for Use on a Network," describes how to perform the tasks required for installing and configuring Windows 98 networking components.

  • Chapter 13, "Using Windows 98 on a Windows NT Network," presents how to implement Windows 98 on a Windows NT Network. You learn how Windows NT network resources are organized with DFS, and you learn the procedures for logging on to a Windows NT domain and configuring the computer to log on using a logon script.

  • Chapter 14, "Using Windows 98 on a Novell Network," describes several features that provide integrated networking support for Novell NetWare networks.

  • Chapter 15, "Managing Windows 98 User Profiles" provides an overview of user profiles, or configuration settings, which establish the user's environment when the user logs on. In addition, this chapter includes information to help you identify, enable, load and update, implement, and troubleshoot user profiles.

  • Chapter 16, "Implementing Windows 98 System Policies," presents when and how to implement system policies.

  • Chapter 17, "Using Windows 98 and the Internet," discusses using Windows 98 with the Internet and intranets. It presents information to help you learn how to browse the Internet and intranets efficiently, how to control access to Internet content, and how to set up security to protect your network from unauthorized access.

  • Chapter 18, "Implementing Dial-Up Networking in Windows 98," discusses four features supported by Windows 98: Dial-Up Networking (DUN), Virtual Private Networking (VPN), Dial-Up Server, and Direct Cable Connection (DCC).

  • Chapter 19, "Maintaining and Troubleshooting Windows 98 on a Network," describes a number of features that allow you to troubleshoot Windows 98.

  • Appendix A, "Questions and Answers" lists all of the review questions from the book showing the page number where the question appears and the suggested answer.

  • Appendix B, "Choosing the Best Windows Platform from a Corporate Perspective," provides information to help you choose the best Windows platform for your organization.

  • Appendix C, "Emergency Startup Disk Contents," describes the function of each file copied to the ESD.

  • Appendix D, "Setup Summary," summarizes the steps in Windows 98 Setup for MS-DOS (or Windows 3.x) and as an upgrade from Windows 95.

  • Appendix E, "Setup Script," provides an example of a Setup script.

  • Appendix F, "Io.sys settings," lists the common entries in Config.sys that are now incorporated into Io.sys for Windows 98.

  • Appendix G, "Contents of the Windows Msdos.sys File," provides information about the Msdos.sys file.

  • Appendix H, "Introduction to NDIS 5.0," provides an overview of the Network Driver Interface Specification (NDIS) 5.0.

  • Appendix I, "Understanding PPTP," provides an overview of Point-to-Point Tunneling Protocol (PPTP).

  • Appendix J, "Virtual Private Networking," provides information on how to install, configure, and use PPTP on computers running the Windows NT Workstation version 4.0, Windows NT Server version 4.0, and Windows 95 operating systems.

  • Appendix K, "How to Interpret the PPPlog.txt File," describes how to interpret the Ppplog.txt file. It also provides information about background concepts used in the development of the Point-to-Point protocol (PPP) and its peripheral components.
