Chapter 11

Installing Windows 98 in a Network Environment

About This Chapter

When installing Windows 98 on a network, you have to make many of the same decisions you make for installing the operating system on a stand-alone system. However, you also have additional decisions regarding the most efficient and cost-effective method of installing Windows 98 on multiple computers.

This chapter discusses how to install Windows 98 in a network environment. It also discusses issues you must consider when choosing a method for installation and the automation features that Windows 98 provides to make multiple installations easier.
