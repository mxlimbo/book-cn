Lesson 2: Controlling Access to Internet Content

Many companies want to ensure that they can control access to Internet content. This lesson provides an overview of Windows 98 features that enable you to control access to the Internet, including content ratings, security levels, security zones, proxies, and firewalls. In addition, the lesson presents information to help you set up an Internet kiosk.

After this lesson, you will be able to:

  • Enable content ratings.

  • Set security levels and use security zones to control access to the Internet.

  • Use a proxy or firewall to provide security when accessing the Internet.

  • Set up an Internet kiosk.

Estimated lesson time: 60 minutes
