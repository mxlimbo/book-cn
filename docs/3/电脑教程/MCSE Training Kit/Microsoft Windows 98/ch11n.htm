Recommended Practices

Here are some recommendations for performing multiple Windows 98 installations:

  • When installing Windows 98 on multiple machines using the same configuration options, you should automate Setup using Setup scripts.

  • You should install Microsoft Batch 98 (Batch.exe) to simplify the creation of Setup scripts.

  • If you must install Windows 98 more than five times, a pull installation can also make the process easier.

  • Although Windows 98 does not include a push installation tool, the Windows NT Microsoft System Management Server can perform push installations of Windows 98.

  • When using a batch script for a pull installation from a server, you must create a shared directory that will contain the batch file on the server and copy the Windows 98 source files to that shared directory.
