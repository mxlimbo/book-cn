Practice: Examining the Default Boot Process

In this practice, you will examine the Windows 98 default startup settings enabled by Io.sys when you remove Autoexec.bat and Config.sys.

    To rename Autoexec.bat and Config.sys

    1. Open Windows Explorer, and then click C: to select it.

    2. On the View menu, click Folder Options.

      The Folder Options dialog box appears.

    3. On the View tab, select Show all files, then click OK.

    4. In the right pane, right-click Autoexec.bat, and then click Rename.

    5. Type saveauto.bat and press ENTER.

    6. In the right pane, right-click Config.sys, and then click Rename.

    7. Type savecon.sys and press ENTER.

    To examine Io.sys default settings

    1. Click Start, and then click Shut Down.

      The Shut Down Windows dialog box appears.

    2. Click Restart, and then click OK.

    3. Hold down the CTRL key during the startup process.

      The Microsoft Windows 98 Start Menu appears.

    4. Select 4. Step-by-step confirmation, and press ENTER.

      Follow the instructions on the screen to step through each question.

      Answer yes to all questions except Load the Windows graphical user interface and WIN.

      Which device drivers are you asked to load?

      Answer

    5. Type set and then press ENTER.

      Windows 98 displays your environment variables.

      What are the default values for:

      PATH:

      TEMP:

      COMSPEC:

      Answer

    6. Restart your computer.

      Your computer restarts using Io.sys. Does Windows 98 load normally?

      Answer

    To restore Autoexec.bat and Config.sys:

    1. After logging in, open Windows Explorer, and then click C: to select it.

    2. In the right pane, right-click Saveauto.bat, and then click Rename.

    3. Type autoexec.bat and press ENTER.

    4. In the right pane, right-click Savecon.sys, and then click Rename.

    5. Type config.sys and press ENTER.

    6. Restart your computer.
