Lesson 1: Interoperating with Windows NT

This lesson provides an overview of Windows 98 interoperability with Windows NT. A computer running Windows 98 on a Windows NT network can take advantage of many network functions and resources. Microsoft Windows NT Server networks provide both client/server and peer networking with user-level security using a domain structure.

Computers grouped into domains allow multiple servers and workstations to be grouped for unified administration. With Windows NT domains, centralized user accounts are used to validate user logon and access to resources.

Windows 98 and Windows NT use the same workgroup model for browsing network resources, so computers running file and printer sharing for Microsoft Networks on a Windows 98 computer can appear in the same workgroup as computers running Windows NT.

After this lesson, you will be able to:

  • Describe Windows 98 interoperability with Windows NT.

  • Describe Windows 98 Client for Microsoft Networks support for the Microsoft Distributed File System for Windows NT Server.

Estimated lesson time: 10 minutes

Figure 13.1 Windows 98 interoperability with Windows NT

Windows 98 interoperates fully with Windows NT (see Figure 13.1). A Windows 98 client on a Windows NT network can do any of the following:

  • Log on to a domain.

  • Share resources with a computer running Windows NT.

  • Gain access to Microsoft BackOffice applications running on a Windows NT Server.

  • Download user profiles and system policies.
