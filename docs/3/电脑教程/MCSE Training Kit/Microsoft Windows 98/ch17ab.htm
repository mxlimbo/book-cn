Managing Personal Web Server

You can manage Personal Web Server by running Personal Web Manager (see Figure 17.7). With Personal Web Manager, you can configure, administer, and monitor your Personal Web Server.

Figure 17.7 Using Personal Web Manager

If you want to connect to the Personal Web Manager Page, perform the following procedure:

  1. Click Start, point to Programs, Internet Explorer, Personal Web Server, and then-click Personal Web Manager.

  2. Click the Web Site icon, and then click the animated wizard figure.

  3. Follow the on-screen instructions.

Using Personal Web Manager Views

Personal Web Manager has four separate views to help manage your Web site:

  • Main View: used to start and stop the Web service and to monitor site traffic statistics. These statistics include the number of connections, requests per day, requests per hour, visitors per day, and visitors per hour.

  • Publish View: used to add, change, and delete files that you want to publish. It also allows you to modify file descriptions of current published files.

  • Web Site View: used to create and edit your home page. This view also allows you to create and maintain a guest book and a drop box.

  • Advanced View: used to create and modify virtual directories, enable directory browsing, and create a web site activity log.
