Converting from FAT16 to FAT32

You can use the Drive Converter (Cvt1.exe) that is included with Windows 98 to convert an existing FAT16 partition to FAT32 (see Figure 3.5). This is a Windows-based program on the System Tools menu.

Figure 3.5 Windows 98 Drive Converter

Caution

The Drive Converter changes the file allocation tables of any hard disk that is converted. Because virus protection software may attempt to prevent those changes, and therefore corrupt the file allocation tables, you should disable any virus protection software before you convert a hard disk to FAT32.

The Drive Converter does the following:

  • Launches the Drive Converter Wizard.

  • Lists the partitions in your computer. Partitions that are large enough to convert to FAT32 are listed with their current file system. Partitions that are too small to convert are labeled with a Drive too small notation. You can only convert one partition at a time.

  • Warns you that you will not be able to access the partition from previous versions of MS-DOS, Windows, or Windows NT.

  • Scans your system for applications that may be incompatible with FAT32.

    The list of programs that the Drive Converter will search for is kept in the registry key HKEY_LOCAL_MACHINE\SYSTEM\ CurrentControlSet\Control\SessionManager\CheckBadApps400

    Only the applications whose Flags' X value is set to F are considered incompatible. If the Drive Converter Wizard finds any of these programs, they appear in a list. You can select an application from the list and click More Info to open a Help file that details the incompatibility and provides suggestions, such as contacting the vendor for an updated version.

  • Prompts you to back up your files.

  • Restarts the computer in MS-DOS mode to perform the actual conversion.

Note

Windows 98 does not include a utility for converting a partition back to FAT16 after you convert it to FAT32.

When the Drive Converter Wizard is done, the Disk Defragmenter utility runs. It is important that you let Disk Defragmenter run to completion after converting to FAT32. If you do not defragment the partition after you convert it to FAT32, it will result in a less efficient and slower computer than before the conversion.

Caution

Various BIOS manufacturers include virus checkers, which look for changes to the Master Boot Record (MBR). In addition, older anti-virus utilities may detect that the MBR has changed. Because a conversion to FAT32 changes the MBR, some virus checkers may incorrectly detect the changes to the MBR as a software virus on your system. If a virus checking utility detects an MBR change after you convert a partition to FAT32 and offers to "fix" it, decline this option.

The easiest solution is to uninstall virus-checking software before converting to FAT32, and then reinstall it after the conversion.
