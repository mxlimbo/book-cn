The Registry Defined

The registry is a database used for storing software and hardware configuration settings. Windows 98 creates the registry during Setup and updates it when additions, removals, or configuration changes are made to devices or software. Figure 6.11 Data stored by the registry

The registry provides:

  • A single, central location for storing configuration information.

  • Access by a local or remote Registry Editor.

  • Storage of user, computer, and application information.

The registry is made up of dynamic information stored in RAM and the following two files, both stored in the C:\Windows folder:

  • System.dat, which contains computer-specific configuration settings.

  • User.dat, which contains user-specific configuration settings.
