Phase 5—Setting Up Hardware and Finalizing Settings

In Windows 98 Setup, hardware detection and hardware driver setup take place after the first restart. After hardware detection and driver setup, the Setup program finalizes system settings.

Setting Up Hardware

You will be asked for date, time zone, and printer information during this phase in order to allow Setup to configure your system. If you installed Windows 98 from MS-DOS or into a new directory, Setup will detect both legacy and Plug and Play devices. This is different from what Setup does when you start it from Windows 95, where only Plug and Play devices are detected.

Finalizing Settings

When hardware setup is complete, Setup performs the following tasks:

  • Modifies registry and other system configuration files

  • Configures Control Panel

  • Configures Start menu

  • Initializes Windows Help

  • Configures MS-DOS programs

When these steps are completed, Setup restarts your computer.
