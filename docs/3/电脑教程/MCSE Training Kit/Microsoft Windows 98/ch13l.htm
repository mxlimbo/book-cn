Lesson Summary

In Windows 98, you can log on to Windows NT domains and connect to domain resources. In Windows NT, a domain is a grouping of computers running Windows NT that share a common directory database of user accounts and policies. You configure Windows 98 to log on to a Windows NT domain through Client for Microsoft Networks.

A logon script is a batch file that runs automatically when a user logs on. To create a logon script, use the User Manager for Domains tool on Windows NT Server. In Windows 98, the Passwords option in Control Panel to synchronize user passwords, thereby allowing users to use a single password to log on to both Windows 98 and Windows NT.
