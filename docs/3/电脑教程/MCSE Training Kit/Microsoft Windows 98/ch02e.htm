Determining Hardware Requirements

Before you install Windows 98, be sure that your computer meets the minimum hardware requirements. Figure 2.1 illustrates the minimum requirements necessary for the installation of Windows 98.

Figure 2.1 Minimum hardware requirements for Windows 98 installation

Although Windows 98 will run on your computer if it meets the minimum hardware requirements, performance will be greatly increased if you install the recommended hardware.

The following table describes the minimum and recommended hardware requirements for setting up Windows 98.

Hardware Minimum Recommended
Processor 486 DX 66-MHz Pentium or higher
Monitor VGA, 256 color SVGA 16- or 24-bit color
Floppy Drive One 3.5-inch high-density floppy disk drive CD-ROM drive 2x, or faster, speed (required if installing from CD-ROM).
Memory 16 MB 24 MB or more
Disk space for installation When using FAT16 file system: approximately 225 MB for a typical installation. When using FAT32 file system: approximately 175 MB for a typical installation. Approximately 75 MB of additional free disk space is required if you back up your Windows 95 system files during Setup.

Note

The amount of free disk space you need will vary depending upon the kind of installation you choose (Typical, Portable, Compact, or Custom). A Custom installation could require as much as 355 MB free disk space when installing on a FAT16 file system.

In addition to the required components, you may also choose to install the following optional hardware components:

  • Modem or network card.

  • Microsoft Mouse or compatible pointing device.

  • Sound card and speakers.
