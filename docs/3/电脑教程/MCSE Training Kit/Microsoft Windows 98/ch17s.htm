Practice: Adding a Site to a Security Zone

In this practice, you will add a site to a security zone.

    To add a site to a security zone

    1. Open Control Panel and double-click Internet.

    2. Click Security.

      The Security tab appears.

    3. From the Zone drop-down list, select the Trusted sites zone.

    4. Click the Add Sites button.

      The Trusted sites zones dialog box appears.

    5. Clear the Require server verification (https:) for all sites in this zone check box.

    6. In the Add this Web site to the zone: box, enter http://www.microsoft.com.

    7. Click the Add button.

    8. Click OK to return to the Security tab.

    9. Click OK.

      Users visiting the trusted zone site and downloading software will not receive the warning about potentially damaging content.
