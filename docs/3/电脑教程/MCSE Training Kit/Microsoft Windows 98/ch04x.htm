Overview of Multiple Display Support

Multiple display in Windows 98 supports several graphics adapters running as many as nine monitors at the same time. Figure 4.5 illustrates some of the practical applications for multiple display support.

Figure 4.5 Possible uses for multiple display support

Potential uses for multiple display support include:

  • Business

    You can view e-mail on one monitor and reports and customer information on another monitor.

  • Desktop publishing

    You can work with text on one monitor and illustrations on another monitor.

  • Presentation graphics

    You can project one version of the graphic for clients and privately view another version with presentation notes.

  • Games

    Multiple display support allows several views and status information simultaneously on separate monitors.

The Multiple Display Support Desktop

Multiple display support modifies the desktop so that it can span several monitors regardless of their size or position. You can set the resolution and color depth for each display separately. To achieve this, Windows 98 developers added application-programming interfaces (APIs) to the Microsoft Win32 API set to allow any application to take advantage of multiple display support functionality. Most applications do not need to be modified to run in a Windows 98 multiple display support environment.

Multiple display uses the concept of primary and secondary displays. All applications running in a window may be on any display. In addition, the taskbar and desktop icons may be moved to any display. However, full screen MS-DOS applications will only run on the primary display.
