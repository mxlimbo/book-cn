Lesson 1: User Interface Enhancements

Improvements to the Windows 98 user interface provide a high level of intuitive operation, and provide easy access to programs and utilities.

This lesson discusses enhancements to the Start menu, the Taskbar, and to the new Explorer, which combines the functions of Windows Explorer and Internet Explorer.

After this lesson, you will be able to:

  • Describe the user-interface enhancements introduced in Windows 98.

Estimated lesson time: 10 minutes
