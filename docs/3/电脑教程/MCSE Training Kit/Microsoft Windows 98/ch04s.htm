Lesson 4: Implementing IEEE 1394 Serial Bus Devices

Windows 98 supports the IEEE 1394 bus, also known as FireWire. It is designed for high-bandwidth PC devices, such as digital camcorders, cameras, and videodisc players. This lesson explains how IEEE 1394 works, and also describes how to add an IEEE 1394 device in Windows 98.

After this lesson, you will be able to:

  • Describe how IEEE 1394 functions.

  • Install an IEEE 1394 device.

Estimated lesson time: 10 minutes
