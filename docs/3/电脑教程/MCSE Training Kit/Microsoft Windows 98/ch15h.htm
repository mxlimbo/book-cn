Lesson 2: Enabling User Profiles

User profiles are available only when you enable them as a configuration option. You do this either with the Passwords icon or the Users icon in Control Panel. This lesson discusses the two methods of enabling user profiles, and also discusses how to disable them.

After this lesson, you will be able to:

  • Enable user profiles with the Passwords icon.

  • Enable user profiles with the Users icon.

  • Disable user profiles.

Estimated lesson time: 15 minutes
