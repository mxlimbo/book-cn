Practice: Installing Files on a Server for a Pull Installation

In this practice, you will install files on a server for a pull installation. The first two exercises in this practice are identical to those in the previous practice.

Exercise 1
Installing Microsoft Batch 98

In this exercise, you will install Microsoft Batch 98.

    To install Microsoft Batch 98

    1. Click Start, and then click Run.

      The Run dialog box opens.

    2. Type D:\tools\reskit\batch\setup.exe (where D: is assumed to be the drive letter for the CD-ROM) and press ENTER.

      The Microsoft Batch 98 Setup dialog box appears.

    3. Click OK.

      You are prompted to begin the installation by clicking the Install button.

    4. Click the button with the icon.

      The Microsoft Batch 98 files are copied to the specified directory.

    5. Click OK.

Exercise 2
Create a batch file with Microsoft Batch 98

In this exercise, you will create a batch file with Microsoft Batch 98.

    To create a batch file with Microsoft Batch 98

    1. Click Start, point to Programs, and then click Microsoft Batch 98.

      The Untitled - Microsoft Batch 98 window appears.

    2. Click Gather now.

      This will gather the current registry settings.

    3. Click Save settings to INF.

      The Save As dialog box appears.

    4. Click Save.

      This saves your file as msbatch.inf. You can now run Setup and use this file to make clones of your current Windows 98 installation. Notice the title bar changes to msbatch.inf - Microsoft Batch 98.

      This batch file will use the same computer name each time you install Windows 98.

    To remove the computer name from the batch file

    1. Click General Setup Options.

      The Batch 98 – General Setup Options dialog box appears.

    2. Click User Info.

    3. Delete the text in the Computer Name box.

    4. Click OK.

    5. Click Save settings to INF.

      This saves your file as msbatch.inf. You can now run Setup and use this batch file to make clones of your current Windows 98 installation. Setup will prompt you for each computer name.

    6. Close Microsoft Batch 98.

Exercise 3
Create multiple machine name batch files with Microsoft Batch 98

In this exercise, you will create multiple machine name batch files with Microsoft Batch 98. This will allow you to install Windows 98 on multiple computers without being prompted for a computer name.

    To create a multiple machine name batch file with Microsoft Batch 98

    1. Start Notepad.

    2. Type in the following names on separate lines:

      Computer101
      Computer202
      Computer303
      Computer404
      Computer505
      Computer606

    3. Save this file as C:\Program Files\Microsoft Batch 98\computer.txt.

    4. Click Start, point to Programs, and then click Microsoft Batch 98.

      The Untitled - Microsoft Batch 98 window appears.

    5. Click Gather now.

      This will gather the current registry settings.

    6. Click File, and then click Multiple Machine-Name Save.

      The Batch 98 – Multiple Machine-Name Save dialog box appears.

    7. Click Machine Name File.

      The Open File dialog box appears.

    8. Click computer.txt and then click Open.

      A message box informs you that 6 entries have been processed.

    9. Click OK.

    10. Click Destination of Save.

      The Select a target directory dialog box appears.

    11. Click Open.

      This selects C:\Program Files\Microsoft Batch 98 as the destination directory.

    12. Click Save Files.

      This will save six batch files in the c:\Program Files\Microsoft Batch 98 folder. Each batch file will be used for a separate computer.

    13. Click OK and then click Close.

    14. Start Windows Explorer and open the C:\Program Files\Microsoft Batch 98 folder.

      The six batch files are displayed as Bstp000x.inf, where x is a digit (1 through 6, in this case) representing the six machine names.

    15. Close Microsoft Batch 98.
