Appendix C

Emergency Startup Disk Contents

This appendix describes the function of each file copied to the Emergency Startup Disk.

File Function
Aspi2dos.sys, Aspi4dos.sys, Aspi8dos.sys, Aspicd.sys Real-Mode Adaptec CD-ROM drivers
Attrib.exe* Adds or removes file attributes.
Autoexec.bat Sets environment variables and loads Mscdex.exe
Btcdrom.sys, Btdosm.sys Mylex/BusLogic CD-ROM drivers
Chkdsk.exe* Checks disks for errors
Command.com Command interpreter
Config.sys Loads device drivers
Debug.exe* Debugging utility
Drvspace.bin Microsoft DriveSpace compression driver
Ebd.cab Compressed file containing several MS-DOS utilities expanded during the startup process
Ebd.sys Identifies the disk as a Windows 98 startup disk
Edit.com* MS-DOS Editor
Extract.exe Expands the Ebd.cab file
Extwrap.exe* Simple extract command that works with Extract.exe and includes prompts for source and destination path information.
Fdisk.exe Disk partition tool
Findramd.exe Utility to find the RAM drive during startup.
Flashpt.sys Mylex/BusLogic CD-ROM Driver
Format.com* Disk format tool
Himem.sys Extended Memory Manager
Io.sys System boot file
Mscdex.exe* Microsoft CD-ROM file extension for MS-DOS
Msdos.sys Contains boot option information
Oakcdrom.sys Generic driver for IDE-based (ATAPI) CD-ROM drives
Ramdrive.sys Creates a RAM drive during startup
Scandisk.exe* Checks disks for errors
Scandisk.ini* Configuration file
Setramd.bat Searches for first available drive to be a RAM drive
Sys.com* System transfer tool
Uninstal.exe* Removes Windows 98 from the system and returns the system to its previous state
*Contained in Ebd.cab, and extracted when ESD is used to start the computer.
