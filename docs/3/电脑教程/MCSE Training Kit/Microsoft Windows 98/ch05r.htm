Practice: Viewing MS-DOS Virtual Machine Resources

In this practice, you will use the System Monitor to view the resources of an MS-DOS virtual machine.

    To view the resources of an MS-DOS virtual machine

    1. From the Windows 98 Start menu, point to Programs, Accessories, System Tools, and click System Monitor.

    2. From the View menu, click Numeric Charts.

    3. From the View menu, click Always on Top.

    4. From the Edit menu, click Add Item.

    5. Select Kernel from the Category list and select Threads from the Item list.

    6. Click OK.

      Note the number of threads listed in System Monitor.

    7. From the Windows 98 Start menu, point to Programs, and click MS-DOS prompt.

      The MS-DOS Prompt window is opened. Notice that the number of Kernel Threads in System Monitory has increased by two. This is because one thread is created for the Command.com application, and one thread is created for the virtual machine within which Command.com runs.
