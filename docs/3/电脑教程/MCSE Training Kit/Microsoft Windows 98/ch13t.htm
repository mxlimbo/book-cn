Optional Practice: Implementing User-Level Security

In this practice, you will implement user-level security by configuring Windows 98, and sharing a folder to users within your domain.

Exercise 1
Setting Up Windows 98 for User-Level

In this exercise, you will configure Windows 98 for user-level security. You will then apply user-level security to shared folders.

To complete this exercise, you must have a computer running Windows 98 with Microsoft File and Printer Sharing installed. Your computer must also be logged on to a Windows NT server.

    To configure Windows 98 for user-level security

    1. Right-click Network Neighborhood, and then click Properties.

      The Network Properties dialog box appear.

    2. On the Access Control tab, click User-level access control.

    3. In the Obtain list of users and groups from box, type your domain name and then click OK.

      A dialog box appears indicating that any shared resources will no longer be shared and asking if you want to continue.

    4. Click Yes to close the Network properties dialog box.

      You are prompted to restart your computer.

    5. Click Yes.

      Your computer restarts.

    To share folders with user-level security

    1. Right-click My Computer, and then click Explore.

      The Explorer window appears.

    2. Create a new folder called MyShare.

    3. Right-click the MyShare folder, and click Sharing.

      The MyShare Properties dialog box appears.

    4. Click Shared As, and then click the Add button.

      A list of your domain's users and groups appears.

    5. From the list, select the users to which you wish to share the MyShare folder, and click Read Only.

    6. Click OK to return to the MyShare Properties dialog box, and then click OK.

      A hand appears under the MyShare folder indicating that it has been shared.
