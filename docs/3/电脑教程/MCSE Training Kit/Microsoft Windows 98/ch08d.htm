Lesson 1: The Windows 98 Print Subsystem

Windows 98 implements printing using a 32-bit print subsystem that includes a multithread, preemptive spooler architecture. In preemptive multitasking, the operating system periodically interrupts the execution of a program and passes control of the system to another waiting program. This architecture provides good printing performance, smooth background printing, and quick return to application time after a print job starts.

This lesson provides an overview of the Windows 98 print subsystem and describes how Windows 98 processes print jobs. This lesson also describes bi-directional printing support and deferred printing.

After this lesson, you will be able to:

  • Describe the Windows 98 print subsystem.

  • Enable ECP support in Windows 98.

  • Describe the benefits of deferred printing.
