Adding an IEEE 1394 Device

Because IEEE 1394 is fully Plug and Play compatible, you can add IEEE 1394 devices without having to restart the computer. Also, because IEEE 1394 is an external bus, you do not have to open the computer to add a device to the bus.

When you add a device, for instance a new video camera, to the bus, the following events occur (all of which are transparent to the user):

  1. A bus initialize signal is sent across the bus, forcing all nodes into a suspended state.

  2. A tree identity process arbitrarily designates one node as the root and the rest of the nodes as either a parent (attached directly to the root node) or a child (not attached directly to the root node).

  3. Once the tree identity process is complete, a self-identity process runs, and each node selects one of 63 unique bus IDs starting with 0 (zero).

  4. After all of the nodes have selected a unique bus ID, the nodes are removed from the suspended state, and the bus is activated.

Note

The entire process of adding a device takes less than one second. If a device is added while data is being transmitted through an isochronous channel across the bus, the isochronous channel being used to transmit the data will be reallocated prior to the completion of the addition of the device. This prevents data from being lost while adding a device to the bus.
