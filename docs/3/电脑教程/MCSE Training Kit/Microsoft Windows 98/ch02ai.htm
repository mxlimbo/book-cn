Requirements for Uninstalling Windows 98

To uninstall Windows 98, you must have:

  • Clicked Yes (recommended) on the Save System Files screen during Setup to save the Windows 95 system files in Winundo.dat and Winundo.ini.

  • Not compressed your system or boot partition after installing Windows 98.

  • Not converted any hard disks to FAT32, unless your Windows 95 version is OSR2.

Note

Windows 95 retail product and OSR1 cannot read FAT32 drives.
