Practice: Viewing My Computer as a Web Page

In this practice, you will view My Computer as a Web page.

    To view My Computer as a Web Page

    1. Open your Windows folder.

    2. Maximize the window, then right-click an empty area inside the window.

    3. Point to View, and then click as Web Page.

      A warning appears about modifying the contents of folders.

    4. Click Show Files.

      Icons for the contents of the Windows folder appear on the right side of the window. On the left side you see Windows with the warning underneath. Below that the Select an item to view its description prompt appears.

    5. Single click a bitmap file icon (.bmp) and read the information that appears.

      This information appears on the left-hand side of the Windows folder when the item on the right-hand side of the folder is clicked:

      Filename.bmp

      Bitmap Image

      Modified:

      (date and time of last modification)

      Size: number KB (KB will be replaced with bytes for files of less than 1 KB)

      [Preview of bitmap]

    6. Close all open windows.
