Practice: Customizing the Start Menu

In this practice, you will move an item on the Start menu.

    To move a Start menu item

    1. On the Start menu, point to Programs, Accessories, and Games, and then move the pointer over Solitaire.

    2. Drag Solitaire to the Accessories menu, but do not release the left mouse button.

    3. Continue to drag Solitaire over the menu until there is a black line between Paint and WordPad.

    4. Release the left mouse button.

    5. Check to see if Solitaire appears on the Accessories menu.
