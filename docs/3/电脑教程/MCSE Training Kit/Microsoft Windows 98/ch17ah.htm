Using Outlook Express

Microsoft Internet Explorer 4.0 includes Outlook Express (see Figure 17.9), which provides services for reading and sending Internet e-mail, for browsing and posting within newsgroups, and for sorting and storing messages on the local drive or mail server.

Outlook Express includes the following features:

  • Support for hypertext markup language (HTML) mail.

  • Multiple e-mail and news accounts.

  • Stationery, a feature that you can use to personalize your messages with background images, graphics, and fonts.

  • Security for e-mail using digital signing and encryption.

Figure 17.9 Using Outlook Express

If you want to configure Outlook Express, perform the following procedure:

  1. Double-click Outlook Express.

    Outlook Express prompts you to choose a folder for your messages.

  2. Click OK.

    This selects the default location.

  3. Use the Internet Connection Wizard to connect Outlook Express with your Internet Service Provider. If this connection was configured earlier for Internet Explorer, use the following steps to configure your Internet news and mail server.

  4. Click Tools and then click Accounts.

    The Internet Accounts dialog box appears.

  5. Click Add, and then click Mail.

  6. Use the Internet Connection Wizard to setup your Internet e-mail account.

  7. Click Add, and then click News.

  8. Use the Internet Connection Wizard to setup your Internet news account.

  9. Click Close.
