Practice: Customizing the Desktop View

In this practice, you will enable and disable the Web style desktop by switching from the default Windows classic style desktop to the optional Web style desktop.

    To enable the Web style desktop

    1. On the Start menu, point to Settings, then click Folder Options.

    2. On the General tab, select Web style, and then click OK. The single-click dialog box prompts you to enable single-click.

    3. Click Yes, then click OK.

      The icons should now be underlined.

    4. Move your pointer over My Computer.

      The pointer should change from an arrow to a hand, and the icon is selected.

    5. To return to the default settings, open the Folder Options dialog box, and on the General tab, click Classic style, then click OK.
