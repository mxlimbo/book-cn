IEEE 1394 Topology

IEEE 1394 uses a tree topology and a daisy chain topology (see Figure 4.4). Maximum distance between devices is 4.5 meters with a maximum of 16 hops between devices.

Figure 4.4 IEEE 1394 daisy chain and tree topologies

The IEEE 1394 Components

The four types of IEEE 1394 components include:

  • Device

    An input/output device (such as a computer, digital video recorder, or digital video camera) is attached to the bus. IEEE 1394 devices have two or more IEEE 1394 connectors that allow devices to be daisy-chained together.

  • Splitter

    A splitter allows considerable flexibility in designing and implementing IEEE 1394 topology by providing extra IEEE 1394 ports.

  • Bridge

    A bridge isolates data traffic within a specific area of the bus in much the same way as an Ethernet bridge.

  • Repeater

    A repeater extends the distance between devices by retransmitting data signals across the bus in much the same way as an Ethernet repeater.

Data Transfer Rates

IEEE 1394 currently supports the following bus speeds: 100 Mbps, 200 Mbps, and 400 Mbps. 800 Mbps transfer rates are currently in development. The IEEE 1394 specification accommodates different transfer rates simultaneously on the same bus. Like USB, IEEE 1394 supports both asynchronous and isochronous communication across the bus.
