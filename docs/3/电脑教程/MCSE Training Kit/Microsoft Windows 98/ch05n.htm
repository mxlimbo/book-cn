Lesson 3: Virtual Machines

When using the MS-DOS operating system, an application has full control of all resources on the computer. Because the Windows 98 architecture is designed to support multitasking, MS-DOS applications run in virtual machines. This allows the applications to "think" that they have control over the entire computer.

A virtual machine is an executable task consisting of an application, supporting software (such as ROM BIOS and MS-DOS), memory, and CPU registers. Windows 98 can create multiple virtual machines.

This lesson provides an overview of virtual machines. It discusses the role of the Virtual Machine Manger as well as the Windows 98 System Virtual Machine.

After this lesson, you will be able to:

  • Define virtual machines in Windows 98.

  • Describe the role of the Virtual Machine Manager.

  • View the resources of an MS-DOS virtual machine.

Estimated lesson time: 15 minutes
