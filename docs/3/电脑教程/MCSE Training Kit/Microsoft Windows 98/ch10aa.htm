Practice: Using System File Checker

In this practice, you will use System File Checker to restore a corrupt system file.

Exercise 1
Preparing Your Computer for the System File Checker Lab

In this exercise, you will run a batch file to prepare your computer for the System File Checker lab by corrupting a system file.

    To prepare your computer for the System File Checker lab

    1. Start Windows Explorer. In the Labfiles\Chapter10 folder of your Supplemental Course Materials CD-ROM, double-click Sysfile.bat.

      This batch file prepares your computer for this lab.

    2. Click Start and then point to Programs, Accessories, and System Tools, then click Drive Converter (FAT32).

      An error message appears indicating that a device attached to the system is not functioning.

    3. Click OK.

      A second error message appears. Note the error message.

      Answer

    4. Click OK.

Exercise 2
Using System File Checker to Restore a System File

In this exercise, you will use System File Checker to extract the original system file from the installation folder and restore it.

    To restore a system file from the installation folder

    1. On the System Tools menu, click System Information.

      The Microsoft System Information window appears.

    2. Click Tools, and then click System File Checker.

      System File Checker appears.

    3. Select Extract one file from installation disk.

    4. In the Specify the system file you would like to restore text box, type cvt32.dll, and then click Start.

      The Extract File dialog box appears.

    5. In the Restore From list, select D:\Win98 (where D: is the drive letter for your CD-ROM drive), and then click OK.

      The Backup File dialog box appears.

    6. Click OK to back up the file to the C:\Windows\Helpdesk\SFC folder.

      The first time, a dialog box will appear prompting you to create the folder.

    7. Click Yes.

      A message appears indicating that the file has been successfully extracted.

    8. Click OK.

      System File Checker appears.

    9. Click Close.

    To confirm that Cvt32.dll has been restored

    1. Click Start and then point to Programs, Accessories, and System Tools, then click Drive Converter (FAT32).

      Do any error messages appear?

      Answer

    2. Click Cancel to close Drive Converter (FAT32).
