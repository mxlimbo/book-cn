Practice: Preparing a Batch File for Windows 98 Installation

In this practice you will install Microsoft Batch 98 and create a batch file.

Exercise 1
Installing Microsoft Batch 98

In this exercise you will install Microsoft Batch 98.

    To install Microsoft Batch 98

    1. Install Windows 98 on a computer using the CD-ROM and configure the options that you want to include for each batch installation.

    2. With the Windows 98 CD-ROM in the CD-ROM drive, click Start, click Run, and type D:\tools\reskit\batch\setup (where D: is assumed to be the drive letter for the CD-ROM) and then press ENTER.

    3. Click the Install button and follow the directions on the screen.

      This installs Microsoft Batch 98 on the computer.

After installing Microsoft Batch 98, you can create the batch file for subsequent Windows 98 installations.

Exercise 2
Creating a Batch File

In this exercise you will use Microsoft Batch 98 to create a batch file.

    To create a batch file

    1. In Microsoft Batch 98, click Gather current registry settings (if using a clean computer).

      - Or -

      Select the system settings you want for the batch installation.

    2. Click Save settings to INF. The Save As dialog box appears.

    3. Except the default file name or type a new file name and click OK.

Installing from a Batch File

After you create the setup script, you can run Windows 98 Setup by specifying the name of the script as a command-line parameter. Depending on which options you selected in Microsoft Batch 98, you may not have to provide any additional input while you install Windows 98.

If you want to install Windows 98 from a CD-ROM with a batch script, perform the following procedure:

  1. On the Start menu, point to Programs, then click Microsoft Batch 98 and configure the desired options.

  2. Click Save settings to INF.

  3. The Save As dialog box appears.

  4. Insert a floppy disk in your computer.

  5. In the File Name box, type a:\msbatch.inf and click Save.

    This saves the msbatch.inf file to a floppy disk that can be used on multiple computers.

  6. Close Microsoft Batch 98.

  7. Remove the floppy disk from the drive.

  8. Go to a computer that needs upgrading to Windows 98, and insert the floppy containing the msbatch.inf file into the floppy drive.

  9. Insert the Windows 98 CD-ROM into the CD-ROM drive.

  10. At a command prompt, type:

    D:\win98\setup.exe a:\msbatch.inf

    (where D: is assumed to be the drive letter for the CD-ROM)

    and press Enter.

Windows 98 will install on the computer with minimal user input.
