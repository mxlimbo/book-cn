Profile Contents

For each user profile, Windows 98 creates a hidden folder called Profiles, which is located within the Windows folder. The Profiles directory does not exist on single-user systems. This folder contains the profile for the user, as outlined in the following table:

This Item Contains
user.dat User-specific registry settings.
\Application Data Information for specific applications, such as shortcuts on the Quick Launch bar.
\Cookies Information about web page visits, including the way a Web page was customized.
\Desktop Shortcuts, folders, and documents located on the desktop.
\Favorites Web sites and other items added to Favorites.
\History Recently visited Web sites.
\My Documents Contents of the My Documents folder.
\NetHood Customized shortcuts in Network Neighborhood, created by dragging a network resource shortcut to this folder.
\Recent Contents of the Documents menu (Start/Documents).
\Start Menu Folders and shortcuts on the Start menu.
\Temporary Internet Files Internet pages stored for quick viewing.
