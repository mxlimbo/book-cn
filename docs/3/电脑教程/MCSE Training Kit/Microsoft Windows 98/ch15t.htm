On a Novell NetWare Network

A NetWare 3.x server stores user profiles in the SYS:\MAIL\<user_id> directory. The Novell SYSCON utility creates user accounts with an eight-digit hexadecimal user ID number. The SYSCON utility automatically creates a \<user_id> directory under SYS:\MAIL and grants the user account read/write access rights to the directory. On NetWare 4.x servers, mail directories are not available, so the Microsoft Service for NetWare Directory Services uses the home directory.

If the Primary Network Logon setting of the computer is set to Client for NetWare Networks, Windows 98 creates and maintains user profiles in users' folders on the NetWare server.

Caution

You must enable long file name support on the NetWare server in order to support the profile folders that use long file names, such as Start menu.
