Summary

The following information summarizes the key points in this chapter:

Troubleshooting Dial-Up Networking

  • Problems with Dial-Up Networking typically originate with your modem or connection. For resolving problems with modems, you can use the modemlog.txt file, or run the Modem Diagnostics Program. For problems with your connection, you can use the Ppplog.txt file.

  • When troubleshooting PPTP connections, consider disabling PPTP filtering. When troubleshooting Multilink connections, make sure that they are operating at the same speed as other connections in simultaneous use.

Using Web-based Troubleshooting Tools in Windows 98

  • Windows Update is a Web-based resource site that updates your computer's drivers and system files. From Windows Update, you can access Technical Support, where you can search through technical support articles.

  • You can use Windows Report Tool to send information about your computer via the Internet to Microsoft support engineers.

Using Windows 98 Troubleshooting Tools for Networks

  • Troubleshooters are interactive help files that prompt you for input regarding the problems you are experiencing and the results you obtain after taking suggested actions.

  • The Registry Editor can be used to modify system settings in Windows 98.

  • System Monitor can be used to identify memory and CPU usage, processes and threads, and network I/O.
