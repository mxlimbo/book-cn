Chapter 8

Using the DNS Service

About This Chapter

Domain Name System (DNS) is a distributed database that is used in TCP/IP networks to translate computer names to IP addresses. This chapter presents an introduction to DNS and name resolution. It also provides the skills and knowledge necessary to configure clients to use the DNS Service.

NOTE
The DNS Service is not available in Microsoft Windows 2000 Professional. You must have a computer running one of the Windows 2000 Server products to use Microsoft's DNS Service.

Before You Begin

To complete this chapter, you must have

  • A computer that meets the minimum hardware requirements listed in "Hardware Requirements."

  • Windows 2000 Professional installed on the computer.

  • TCP/IP installed as the only protocol.
