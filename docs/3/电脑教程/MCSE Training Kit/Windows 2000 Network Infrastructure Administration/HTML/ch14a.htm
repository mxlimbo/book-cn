Chapter 14

Implementing Enterprise-Wide Network Security

About This Chapter

In this chapter, you will learn how to implement security on your network and how to properly plan for security. You will also learn how to establish and secure remote access to your network. In addition, troubleshooting and monitoring network resources and remote access is discussed.

Before You Begin

To complete this chapter, you must have

  • Installed Microsoft Windows 2000 Server

  • Completed Chapters 2 through 10
