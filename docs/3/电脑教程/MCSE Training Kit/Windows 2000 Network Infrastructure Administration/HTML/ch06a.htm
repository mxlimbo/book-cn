Chapter 6

Resolving Network Host Names

About This Chapter

Both clients and servers on a network must resolve the user-friendly host names to the Internet Protocol (IP) addresses used in network communications. In this chapter, you will learn how the Transmission Control Protocol/Internet Protocol (TCP/IP) protocol resolves host names. This knowledge is important as you design your network and plan how names and IP addresses will be resolved. Advanced resolution capabilities, such as Domain Name System (DNS) and Windows Interface Name Service (WINS), are discussed in later chapters.
