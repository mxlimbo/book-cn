Chapter 13

Managing Microsoft Windows Internet Name Service (WINS)

About This Chapter

Although Microsoft Windows Internet Name Service (WINS) servers are not needed in a network consisting entirely of Microsoft Windows 2000-based computers, they are crucial in most Transmission Control Protocol/Internet Protocol (TCP/IP) networks containing computers based on the older architectures of Windows NT 4.0, Windows 98, or Windows 95. In this chapter, you learn how to implement WINS on your network.

Before You Begin

To complete this chapter, you must have

  • Two computers (Server01 and Server02) with Windows 2000 Server installed and configured with TCP/IP
