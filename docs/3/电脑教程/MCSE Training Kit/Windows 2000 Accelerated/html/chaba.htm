Appendix B -- Microsoft Windows 2000 Services for UNIX

About This Chapter

Microsoft Windows Services for UNIX (SFU) version 1.0 was introduced in 1999 and quickly became a significant tool in the arsenal of those trying to make Windows NT and UNIX coexist on the same network. With the introduction of version 2.0 of Services for UNIX, Microsoft has taken a huge leap forward, adding new features and significantly enhancing the features from the initial version, while providing seamless integration into both Windows 2000 and Windows NT 4 networks.

This chapter provides you with an overview of SFU and how to install and configure SFU version 2.0.

Before You Begin

To complete this chapter, you must have

  • Installed Windows 2000 Server on a computer (Server01)

    See chapter 8, "Installing Microsoft Windows 2000" for more details.

  • Server01 configured as a domain controller
  • Experience using Microsoft Management Console (MMC)
