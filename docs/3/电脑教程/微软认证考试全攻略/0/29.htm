The Exam Braindump is a very good initiative, I want to thank Arty for giving me this adress.
Everyone should write his or her experiences down on this page.


Now a few points of my exam:

You must known how to able User Profils and System Policies and know where the are stored, as well as Windows as Novell.
What must you able when you are working in a company where sometimes power fluctuations are? Disable "write behind caching" in System, performance tab, file systems, problems.

Know what to able when you use Net Watcher on a remote computer.

When there is no WINS server or DNS server available, use LMHosts as a equivalent for a WINS server and use Hosts as a equivalent for a DNS server.

Know how to handle WINIPCFG.

Know when to "capture a printerport".

Know how to install Point-and-Print.
