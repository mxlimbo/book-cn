I've passed, NTS, NTSE, NE and Win95. With self study and purchasing trancender tests for Win95 and NE (I will not be selling either). Win95. In general there were:

Quite a few questions on printing, I was surprized at how many I got.

Net Watcher

OS2 name space

long file names

share vs user permissions

netware client, microsoft client

Dos directories are like win95 folders

Memory address space for 32 bit windows apps, 4m-2gig

How many virtual machines is win95 maintaining, know that 16 bit and 32 bit all go in one and each dos program gets its own.

Know that long file names use up more space in the root directory name table.

Swap file can be on a compressed drive and compress drive cannot be over 512 mb.

When using the Netware client your using user level security.

Use Ctrl + Alt + Delete to end programs that stop responding.
