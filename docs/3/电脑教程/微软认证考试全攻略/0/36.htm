70-63 Brain Dump


Question: User has notebook with PC network card. He want to remove it. What should he do first?

1. Logoff the network
2. Shut down the machine
3. Disable socket services
4. Do nothing,Just remove it. (my choise,hope I was right)

Question: Sue and Jim share a computer with user profiles. Both use exchange to access Compuserve. Sue is able to access but Jim is unable to. Why?

Answer: Jim does not have client for Compuserve in his exchange profile.

Question: 10 Win 95 clients are partisipating in a workgroup without NT server or Netware server. You want to share files but only allow changes on the computer they reside on. How would you implement this?

Answer: Share-level security (my choise)

Question: You have an MS-DOS app.that works fine windowed but has corrupted video when run under full screen. You have turned off FAST ROM emulation and dynamic memory allocation and still have the same problem. What needs to be done?

Answer: Run in MS-DOS mode

Question: Why does Windows 95 print faster than Windows 3.1

Answer: EMF spooling

Question: You have 2 departments, SALES and ACCOUNTING. Both run Windows 3.1 and need to be upgraded.
SALES will need to run Win 16 apps, Win 32 apps, and MS-DOS apps. that will directly access your computer hardware.
ACCOUNTING will need to run Win 16 apps, Win32 apps,and needs heavy security. What O/S will you upgrade each groop to?

Answer: SALES-Windows95, ACCOUNTING-NT workstation

Question: Users are unable to create LFN on Netware server but can create 8.3 files. What needs to be done?

Answer: Use OS/2 Name space.

Question: Wich network protocols are supported by DUN

Answer: IPX/SPX, NetBEUI, TCP/IP.

Question: What do you need to install DUN server

Answer: MS Plus.

Question: Wher are the user profiles stored on a standalone computer?

Answer: USER.DAT

Question: Where are user profiles stored on Novell server?

Answer: Mail directory.

Question: Wich Win95 app allows you to despool a print Q from a Netware server?

Answer: MSPSRV.EXE. (everyone gets this one)

Question: You access your LAN via DUN. When you run exchange it takes a long time before you can read any mail. The modem lights flicker. This happens even when there is no unread mail waiting. Why is this happening?

Answer: The copy of the exchange.exe is being downloaded every time you run exchange.

Question: What is true about process?

Answer: Each process contains at least one thread.

Question: Linda goes to a foreign contry and attempts to dial into the office using remote access. Her modem tries to dial but gives her an error message that no dial tone could be detected. What settings should you change?

Answer: Wait for dial tone (my Choise)

Question: What program will you run to speed up your hard disk?

Answer:Defrag.

Question: You want to access your mailbox on a 3.2 server. What information do you need?

Answer: Location, name and password.

Question: How do you set up Windows 95 to install from the server?

Answer: NETSETUP

Question: You are running MSDOS app. in window, the app. is not TAPI aware. You want to send a fax to a shared fax modem. How do you do this?

Answer: It can't be done

Question: Where does Win 32 apps reside in memory?

Answer: 4M-2GB
