70-064Windows 95试题回顾（2）

Win95 Test 70-064, Required to pass: 632, Total Questions: 49, Times: 90min.

I passed this test on the first try with 830. I felt the wording is easier to understand than the Network Essential Test. Probably I studied too much or so, or adapted to this type of testing. I read 2 Win95 books Published from Microsoft and some sections on Win95 Resource kit. The following are what I remember:


1. IP address & default gateway ?
2. Which installation method will have the least traffic on the network after the installation?

3. You installed a hard drive on Client computer. Which partition method will have the most free space?

4. You add a ClientA Win95 computer to a network which has one HTTP server, one WinNT Server, one Novell Netware server. You want to be able to connect to internet, access to shared resources on all servers, and shared your resources to other WinNT Server.

5. What do you need to configure a PCL printer using Point and Print on Novell Netware Server

6. You’re printing a long document on PCL printer and then you have a problem. What should you do? An exhibit shows a printing properties.

7. Users policies on a local computers & Network?

8. You installed a modem. What do you need to connect to internet?

9. What’s the safe method to connect to internet?

10. What do you need to configure a Win95 Dial-Up server?

11. A long planning problem with access problem, with about half page of requirement, a half page of proposal, an exhibit of default policies, and a half page of optional answers.

12. A long planning problem repeated from #11 above with different access problem, half page of requirement, half page of proposal, a different exhibit of policies, and half page of optional answers.

13. What should you do to dual boot Win95 with MS-DOS?

14. What file systems does Win95 support?

15. What do you need to configure Win95 Client to access shared resources on Novell NetWare Server? Choose all that apply

16. What do you need to remotely administer Win95 network clients? Choose all that apply

17. What do you need to configure PCL Printer on Novell NetWare network? Choose all that apply

18. One of your Win95 Clients on your network complained that sometimes he/she couldn’t access some files on his/her computer. What should you do? (…., Defragment, Format, …, Compress, Scandisk, Partitions,…)

19. What do you need to configure IPX/SPX-compatible protocol to let NETX and VLM Novell Netware Clients to access Win95 Server?

20. What do you need to configure the browsing for Win95 Client in peer-to-peer local area network?


That’s all I remember so far at this time. I may be able to remember more if I meet some problems similar on the test. Good luck to you all.
