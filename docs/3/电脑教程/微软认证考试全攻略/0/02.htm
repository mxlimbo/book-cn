MCSE简介

　　对于从事网络工作的人，微软提供微软认证的系统工程师 （MCSE）证书。MCSE拥有者有能力为一个使用Microsoft Windows NT Server和服务产品Microsoft Backoffice 的集 成家族的大计算机环境进行有效的规划，实现，维护和信息 系统支持。
MCSE 需要通过6门考试（包括4门必须课程的考试和2门可选 考试）。

70-058: Networking Essentials

70-064: Implementing and Supporting Microsoft Windows 95 或70-073: Implementing and Supporting Microsoft Windows NT Workstation 4.0 或70-098: Implementing and Supporting Microsoft Windows 98

70-067: Implementing and Supporting Microsoft Windows NT 4.0

70-068: Implementing and Supporting Microsoft Windows NT Server 4.0 In the Enterprise

70-059: Internetworking with Microsoft TCP/IP on Microsoft Windows NT 4.0

70-087: Implementing and Supporting Microsoft Internet Information Server 4.0

70-088: Implementing and Supporting Microsoft Proxy Server 2.0

70-026: System Administration for Microsoft SQL Server 6.5

70-027: Implementing a Database Design on Microsoft SQL Server 6.5

70-076: Implementing and Supporting Microsoft Exchange Server 5 或70-081: Implementing and Supporting Microsoft Exchange Server 5.5

70-085: Implementing and Supporting Microsoft SNA Server 4.0

70-086: Implementing and Supporting Microsoft System Management Server 2.0

70-079: Implementing and Supporting Microsoft Internet Explorer 4.0 by Using the Internet Explorer Administration Kit
