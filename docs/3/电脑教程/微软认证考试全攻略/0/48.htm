70-063　Windows 95 Braindump


I passed Windows 95 today (3/3/98). It was more challenging than Network Essentials. I passed Network Essentials 2/98.

Study Guides: MCSE Windows 95 by New Riders, Windows 95 Resource Kit, Transcender ver1.0 practice test and this Braindump site. Thanks goes to Robert Schmidt for making this braindump possible. The information provided was very helpful. I researched the questions/answers for validity.

The questions that I remember are listed below:

Warning...it may be a repeat of what's already on this braindump site. Verify answers for validity.

1. You need to install Windows 95 of four computers. Which of the computers listed will require hardware upgrade to enable Windows 95 to run?

2. Lots of MS-DOS application and Network printing questions can't remember details

3. John has a Windows 95 laptop with a Network card installed. He connects to the network and performs his daily task. At the end of the business day he wants to remove the network card and return it to his co-worker but he doesn't want to cause problems on his laptop. What should he do? How can the card be removed?

4. Your Win16 application fails. When does Windows 95 clean up the resources that the application was using?

5. MS-DOS uses what type of memory?

6. A mobile user call from oversea while using the same portable from the office. User doesn't hear a dial tone. What should the user do?

7. Approximately three (3) briefcase questions, file sharing, printing from MS Fax, a few setup questions
