70-063　Supporting and Implementing Win 95 - Frances

24 Feb 98

Question about processes and treads and how they are executed. (Each process consists of at least one thread and the Operating system decides which thread executes when in 32 bit applications)

Amy wants to make her printer available with user-level security. She is part of a NT domain and have a W95 computer. How should she log on? Log on to domain / workgroup / log on outside domain /

Question about rings and what they mean. 0 - ring highest security, only available to core O/S

Question about 32bit, 16bit, DOS programmes and their address spaces. (Each DOS programme runs in its own VM; Win 16- based programmes share a single address space in the System VM. The system VM also contains Win32-based programs and the core operating services.

Question about 32 bit program, 16 bit program, DOS in full screen all running simultaeously. You are working on the DOS program and the 16-bit program stops responding what shoud you do? (choose two) Options include *Press CTRL + ALT + DEL , select the program and End Task. * Run MS DOS program in windowed mode *Proceed working in DOS program in full screen

Scandisk and Defrag questions (know the DOS commands and options)

Question about EMF printing and the effect it had on the time it takes to return to your application.

Question about IRQ - know that legacy devices, with a set IRQ will be assigned first. Then Plug and Play devices that can use two or more IRQs

Question about nibble mode - plug and play enabled.

Amy belongs to a WAN. The Exchange Postoffice and Mail Server is located on a NT on a remote site. She cannot connect to her mailbox. Why?

Question about server based setup. Some computers will run W95 from server, others have enough disk space, will run from their hard disks. How many installation copies do you have to provide and how many logon scripts (two installation copies - one for running from h/d and one for network; two logon scripts - one to initiate installation for h/d, one for installation running from network.

Long file names on Netware server - load OS2 name space

Your are logged on to domain A and belong to Workgroup B. How to see a server in Workgroup C (also belonging to domain A) Network Neighbourhood, Entire Network, Workgroup C, Server name.

During installation your PCMCIA card is not detected. How to install? (Hardware Wizard)

Win 16-based program cannot access modem, but Win 32 have no problems. Check that comm.drv=comm.drv in system.ini

With which tools can you edit the registry - Policy Editor, Regedit

Tom and Mary are working on the same file, located on the network. They each have a copy of it in their Briefcase. They both take it home on their laptops and make changes. Mary docks her laptop the next morning and updates her Briefcas. Tom does not. Which copy of the file is updated? One on Mary's computer, the file on Tom and Mary's computers, The one on the network

Check Properties box of MS DOS Prompt shortcut from your desktop - I got a lot of questions on "DOS programs not wanting to be aware of Windows", Exclusive mouse mode etc.

When a DOS session is running in full screen left hand mouse not available, cannot copy.

What file is created when Windows crashes during installation. Detcrash.log.

What file contains hardware info? System.dat

What do you need to do to configure Win 95 as a Dial up server? Install Microsoft Plus!

What will be the DOS name of a file called My Windows File.doc in Win 95? MYWIND~1.DOC (first six, tilde and number.

Which file loads Config.sys, Autoexec.bat and System.dat - IO.sys

When you install a printer a file is added to the Win\system\color\ directory. What for? Image color matching and grey scale for lazer printers.

What file needs to be present in order for Point and Print services to work? MSPSRV.exe

Know address spaces in the system memory map.

Which NDIS driver versions are supported by W95? 2.1 and 3.1 (Plug and Play, hot docking of network Interface Cards)

Know that the LMHOSTS file is used when WINS server not available.

How to end the capture of a printer port connected to a Netware printer. THE DOS COMMAND!
