微软认证专家(MCP)

　　微软认证专家（Microsoft Certified Professionals， MCP）是全球公认的计算机软件高级人才认证，由比尔.盖茨 签发的MCP证书在全球各国都是您在计算机软件领域个人全 面能力的证明。

　　该项考试是对于那些想证明其在微软某一产品方面有专长的 人们，通过MCP后,微软将提供微软认证的产品证书。应试者 还可以通过参加其他的微软证书考试进一步增强其使用 Microsoft Backoffice产品，开发工具或桌面应用的技能。

　　在您完成了MCP认证考试后，微软公司会在8周之内（一般为 2-4周）将证书直接寄到您的手中，有关证书的任何问题您 都可以通过微软的咨询中心进行查询.

MCP证书所需通过的考试：除70-58外的任何一门考试。
