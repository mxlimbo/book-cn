第六节：注释文本

　　<！-- -->

　　注释标志符在HTML文档中留出一些空间，这里面的文字信息不会显示在网页上，这是因为凡是在注释标志符内的文字都是隐性的，你可以利用这个空间做为备忘录。但有一点要注意，那就是如果别人查看你的源代码，那么你的这些注释信息就会一览无余了。所以如果你想把自己一些最保密的事保存在这里，恐怕要换个地方了。

　　注释标志符的格式如下：

　　把下面这段文字做为注释

　　<！--把下面这段文字做为注释-->

　　我很想把这段HTML编码用网页显示给你，但既然它是隐藏的，因此谁也看不到它了。
