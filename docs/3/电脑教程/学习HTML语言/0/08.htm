第一节：段落

　　<P>开始一个新段落

　　段落标志符是HTML标志符中最常见的一种符号，它位于各段落起始部位，使用该标志符后，每块文本段落之间都会空出一行。另外对于<P>标志符来说，不必使用</P>来做结尾。
