　　你知道如何使网页的标题比其它文字更大些以便突出这段标题吗？使用标题标志符就可以帮助你实现这一点，你可以采用大标题、中标题或小标题，使用大标题可以用<h1>标志符开始，结束时用</h1>志符。

　　如果你输入下列代码：

　　<h1>号外!外星人来了，大家快逃吧!〈/h1>

　　这时网页就会显示成：

　　号外!外星人来了，大家快逃吧!
