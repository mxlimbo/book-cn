第二课：处理文本

第五节：分行

　　<br>插入新分行

　　大多数情况下，段落标志符<p>是分隔文本的最佳符号，但有时你可能会希望重起一行，但在新行与上一行之间并不空出一行间距，在这种情况下，你最好使用分行标志符<br>。

　　使用分行标志符编写的一段话是下面这个样子：

　　<p>我非常喜欢互联网<br>

　　在这里我可以足足冲浪一天也不觉得累!<br>

　　幸好我的老板还没有抓到我,<br>

　　但如果真被他抓到，恐怕我的工资就要泡汤了! <br>

　　这时网页显示成下面这个样子：

　　我非常喜欢互联网。
　　在这里我可以足足冲浪一天也不觉得累!
　　幸好我的老板还没有抓到我,
　　但如果真被他抓到，恐怕我的工资就要泡汤了!
