About This Book

Welcome to Desktop Applications with Microsoft Visual Basic 6.0 MCSD Training Kit. By completing the lessons and associated exercises in this course, you will acquire the knowledge and skills necessary to develop solutions using Visual Basic 6.0.

This book also addresses the objectives of the Microsoft Designing and Implementing Desktop Applications with Microsoft® Visual Basic® 6.0 (70-176) exam. This self-paced course provides content that supports the skills measured by this exam.

NOTE
For more information on becoming a Microsoft Certified Solutions Developer, see the section titled "The Microsoft Certified Professional Program" later in this chapter.

Each chapter in this book is divided into lessons. Most lessons include hands-on procedures that allow you to practice or demonstrate a particular concept or skill. Each chapter ends with a short summary of all chapter lessons, a hands-on lab, and a set of review questions to test your knowledge of the chapter material.

The "Getting Started" section of this chapter provides important setup instructions that describe the hardware and software requirements to complete the exercises in this course. Read through this section thoroughly before you start the lessons.
