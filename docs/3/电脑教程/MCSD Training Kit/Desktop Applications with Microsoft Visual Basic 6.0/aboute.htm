Using this Book to Prepare for Certification

Where to Find Specific Skills in This Book

The following tables provide a list of the skills measured on the certification exam Designing and Implementing Desktop Applications with Microsoft® Visual Basic® 6.0. The tables provide the skill, and where in this book you will find the lesson relating to that skill.

NOTE
Exam skills are subject to change without prior notice and at the sole discretion of Microsoft.
