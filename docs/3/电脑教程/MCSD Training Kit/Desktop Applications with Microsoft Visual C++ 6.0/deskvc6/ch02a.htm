Chapter 2

Visual C++ Development Environment

About This Chapter

In this chapter, you will learn about the development environment and the tools that enable you to create applications using Microsoft Visual C++. You will learn how to use the MFC AppWizard to generate a development project, which contains source code and resource files that you can compile into a working executable program. You will explore features of the Visual C++ development environment, and learn how to configure options for your project. You will also learn how to use Microsoft Visual SourceSafe to manage source code control for a software development team.

Before You Begin

To complete the lessons in this chapter you must have installed the Visual C++ development tools as described in Lesson 2 of Chapter 1.
