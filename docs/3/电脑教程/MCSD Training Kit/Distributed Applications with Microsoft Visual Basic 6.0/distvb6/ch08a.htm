Chapter 8

Using Stored Procedures

About This Chapter

In this chapter, you will learn about stored procedures and their benefits, including how to create stored procedures and run them from Microsoft Visual Basic.

Before You Begin

To complete the lessons in this chapter, you must have:

  • Installed SQL Server 7.0 Enterprise.

  • Installed the T-SQL Debugger in the Visual Basic Enterprise Setup.

  • Installed the SQL Debugging Server from the Supplemental Course Material CD-ROM that accompanies this book.

  • Read Chapter 5, Chapter 6, and Chapter 7.
