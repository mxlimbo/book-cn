Chapter 4

Advanced COM Programming with Visual Basic 6.0

About This Chapter

This chapter describes interface-based programming and how it relates to the Component Object Model (COM). You will learn how Microsoft Visual Basic allows you to utilize COM through interface-based programming.

Before You Begin

To complete the lessons in this chapter, you must have:

  • Read Chapter 3.

  • Installed the OLE View tool from Microsoft Visual Studio 6.0.
