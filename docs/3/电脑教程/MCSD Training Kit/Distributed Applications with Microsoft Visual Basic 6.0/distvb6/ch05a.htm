Chapter 5

Introducing ActiveX Data Objects

About This Chapter

In this chapter, you will learn about OLE DB and its role in the concept of universal data access. You will also learn about ActiveX Data Objects (ADO), including the ADO object model and how ADO relates to OLE DB, and how to use the Microsoft Visual Basic Data Environment to quickly build applications using ADO.

Before You Begin

To complete the lessons in this chapter, you must have:

  • Installed SQL Server 7.0.

  • Completed Chapter 1, including the lab.

  • Gained a basic understanding of database concepts.
