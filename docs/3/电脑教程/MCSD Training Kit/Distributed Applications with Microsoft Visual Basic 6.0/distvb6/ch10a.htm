Chapter 10

Implementing Security in the Enterprise

About This Chapter

In this chapter, you will learn why security is an important part of application architecture. You will also learn how to protect the privacy of sensitive user information in components and processes that request applications to run from unauthorized users. These issues are addressed for Microsoft Windows NT networks using Microsoft SQL Server and Microsoft Transaction Server (MTS).
