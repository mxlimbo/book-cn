Summary

A class module (.cls file) is a type of Visual Basic code module that defines one type of object. You can create class modules manually or with the Class Builder utility. You can use class modules in one or more Visual Basic projects, and also class modules that are compiled in a COM DLL. The Object Browser, which is designed to read type libraries, lets you display the classes that are available in projects and libraries.

COM components that you create are based on either the ActiveX EXE or ActiveX DLL template. You can build either in- or out-of-process components depending on the needs of your application. To make your application run quickly and efficiently, optimize the resolution of object references. Early-bound objects always run faster than late-bound objects. This is because the client does not have to cross process boundaries to use a server object's properties, methods, and events.
