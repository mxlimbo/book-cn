Summary

Although considered an art by many, learning to create and code great applications can be done. In the preceding chapters, we have discussed many techniques that teams can use during the Developing Phase. The actual Developing Phase uses these technologies to reach its Scope Complete Milestone.

The Scope Complete Milestone signifies that a team is ready to put the finishing touches on a product. All features will have been created; therefore, the team is now ready to move into the Stabilizing Phase. Through this phase, the team's risk management skills will continue to be exercised to help create the desired product.
