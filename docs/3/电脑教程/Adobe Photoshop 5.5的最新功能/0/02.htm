图象截分

　　Adobe ImageReadyTM 2.0 中新的截分工具可以对图象进行分割。分割后可以为每个用户片段指派 URL 链接、ALT 标签和不同的压缩设置。例如，对一个照片区域进行 JPEG 压缩，对文本和纯色进行 GIF 压缩。ImageReady 在生成重新组装图象的 HTML 表时添加自动片段。

　　如果您后来对原始图象进行了编辑，您可以立即更新截分的图象的 HTML 代码——即便您已将此代码复制到一个更为复杂的 HTML 文件中。ImageReady 在其 HTML 代码的开头和结尾嵌入注释，这样它可以自动识别和替换那段代码。
