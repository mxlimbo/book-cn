无缝的 Web 图形工作流程

　　Adobe Photoshop 5.5 为设计动人的 Web 图形提供了内置的 Web 优化功能。它的 Adobe ImageReadyTM 2.0 组件用于处理图象截分、动画、JavaScript 翻滚和其它高级 Web 任务。在上述任一种应用程序中，单击工具箱上的“跳转到”图标可以立即将某个文件从一个程序移动到另一个程序中。当您来回移动文件时，图层、图层效果、字体、翻滚、动画和其它文件特性都完好地保留着。在一个程序中所作的更改甚至会在另一个程序的历史记录调板中以历史记录状态加以保存，以便随时撤消和重做。
