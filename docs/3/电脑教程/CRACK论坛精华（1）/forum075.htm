****第一步：
**软件脱壳
1、用trw2000载入peepnet.exe
2、程序被拦，下
:bpx 004126da
:g
3、程序又被拦于：
xxxx:004126DA PUSH EBP
XXXX:004126DB MOV EBP,ESP
XXXX:004126DD PUSH BYTE -01
.............(略)
下
:pedump c:\peepnet.exe
4、呵呵，这样，你的C盘目录下就有一个脱好的
peepnet.exe了!
****第二步：
**修改软件
找84bd74168bce
改84bd90908bce
存盘退出!
5、这样软件就算注册了!运行软件，哈哈，
"registered to :DEMO VERSION"
如果你觉得这个软件应该是注册给自己的，
于是你可在注册表：
HKEY_USERS/.DEFAULT/SOFTWARE/MLsoft/Peepnet/USER/
修改nume的值，如改为yubing
重新运行软件，是不是看上去爽多了!
