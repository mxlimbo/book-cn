<blockquote><span class="p9"> 目标程序: Icon Catcher 3.0.1.324 <br>
  程序用途: 从DLL, EXE, ICO, ICL和BMP文件中抓出图标 <br>
  下载地址: http://www.helexis.com/ic/iconcatc.exe&nbsp; 或 <br>
  &nbsp; &nbsp;&nbsp;&nbsp; http://skyscraper.fortunecity.com/pacific/730/software/iconcatc.exe 
  <br>
  文件大小: 416K <br>
  保护类型: 加壳,使用功能限制(不能存盘或拷贝至剪贴板) <br>
  <br>
  **该程序不大,而且也很好用,有兴趣的朋友不妨下载一试. 我这里说的破解方法不一定是 <br>
  最好的,你的破解经过如果与之不同,欢迎贴出来交流. 冰毒 2000.3.16 <br>
  <br>
  1. 脱壳: 程序用Neolite 2.0加壳,很容易脱掉,而且脱壳程序不改程序入口值也可以正常运行. <br>
  <br>
  2. 去除功能限制: 利用程序的警告框,不难追到下面这段代码 <br>
  <br>
  014F:00480549&nbsp; FF5130&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; CALL&nbsp;

  014F:0048054C&nbsp; 8BD8&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 
  MOV&nbsp; &nbsp; EBX,EAX &lt;----改成 MOV BL,01 <br>
  014F:0048054E&nbsp; 84DB&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 
  TEST&nbsp; &nbsp; BL,BL <br>
  014F:00480550&nbsp; 750F&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 
  JNZ&nbsp; &nbsp; 00480561&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 
  &nbsp; <br>
  014F:00480552&nbsp; 66837E3A00&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; CMP&nbsp; &nbsp; 
  WORD PTR [ESI+3A],00 <br>
  014F:00480557&nbsp; 7408&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 
  JZ&nbsp; &nbsp; &nbsp; 00480561&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 
  &nbsp; <br>
  014F:00480559&nbsp; 8BD6&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 
  MOV&nbsp; &nbsp; EDX,ESI <br>
  014F:0048055B&nbsp; 8B463C&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; MOV&nbsp; 
  &nbsp; EAX,[ESI+3C] <br>
  014F:0048055E&nbsp; FF5638&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; CALL&nbsp; 
  &nbsp; [ESI+38] &lt;----这个CALL! <br>
  014F:00480561&nbsp; 8BC3&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 
  MOV&nbsp; &nbsp; EAX,EBX <br>
  <br>
