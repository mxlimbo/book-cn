软件 system menachinc 的它用（附：序列号破解）

    system menachinc是一个很好的系统加强软件，可以帮助删除我们上网后留在硬盘中的cache......。但你知道还可以用它来帮助我们crack吗？让我告诉你，它至少可以承担一部分regmon的工作，帮助我们发现那些有使用次数限制，且将已使用次数和/或未使用次数记录在注册表或硬盘文件上的软件所做的手脚。具体就是用它 system\safe installer功能。以下举例说明。
  软件NEVERHACK 2.2版本只能使用15次，猜测它可能将次数记录在注册表中。先启动system menachinc，执行system\safe installer功能，先给报告文件起个名字，指定NEVERHACK.exe所在的路径，再一路next即可。等NEVERHACK自动运行后会告诉你：你已使用了多少多少次，还有多少多少次可以用。ok！关闭它。按下system menachinc上的Done report按钮，过一会NEVERHACK在你注册表或硬盘上做的手脚就都在system menachinc给你的txt文件中了，如下：
NO CHANGES IN TRACKED DRIVES

NO CHANGES MADE TO C:\WINDOWS\SYSTEM.INI...

NO CHANGES MADE TO C:\WINDOWS\WIN.INI...

REGISTRY KEY VALUES CHANGED: (2)
HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion
Value "NhCount": from "6" to "7"  <---这就是记录使用次数的地方。这比用regmon监视，动辄分析几百行乃至上千行的数据简单吧。怎么样，想不到它还可以这样用吧。
