步骤：（关键是要准备target机器的东东比较多）
1、弄到一块网卡，要注意型号（目前SOFTICE只支持两种类型的网卡）：
      a  NE2000或和NE2000兼容  （大多数的ISA网卡）
      b  3COM90X系列
    做实验用的就是属于a的类型------U9008

2、把网卡装入target机中，安装网卡驱动程序，若能用WINDOW系统自带的NE2000兼容网卡驱动程序

安装成功，那是最好的（3COM网卡就不需要用啦），如果不使用NE2000兼容网卡驱动程序，也可以

。再安装TCP/IP协议指定IP地址。

3、在SOFTICE的目录下有两个驱动程序。EL90XND4.sys 和NE2000.sys，把它拷贝到以下目录：

      a  NT下，  \WINNT\SYSTEM32\DRIVERS
      b  Win95下， \WINDOWS\SYSTEM

4、重新启动机器。

5、target机上，SOFTICE用到的命令：(注意!!!!  SOFTICE需要4.0以上版本，否则没有以下命令）
      a  NET START <IP address|DHCP> [MASK=<subnet mask>] [GATEWAY=<IP address>]
          为target机指定IP地址，启动网络。
参数：
          IP address------>>本机的IP地址，格式：xxx.xxx.xxx.xxx
          DHCP      ------>>自动分配IP地址，一般不常用。除非粗通网络技术。
          MASK        ------>>为子网掩码。一般情形为：255.255.255.0  这个在小型网络中也可以不用
          GATEWAY----->>网关。这个没有什么说，在小型网络中也可以不用，所谓小型网络就是几十台

机器联网的情况。
例：
          NET START 10.10.10.1 
          NET START 10.10.10.1  MASK=255.255.255.0  GATEWAY=10.10.10.10

      b  NET ALLOW <IP address|ANY> [AUTO] [PASSWORD=<password>]
          允许哪台机器进行远程调试
参数：
          IP address------>>local机的IP地址。通俗一点，就是允许谁来远程调试。
          ANY          ------>>允许任何机器进行远程调试。
          AUTO        ------>>自动处理调试进程
          PASSWORD--->>验证密码
例：
          NET ALLOW  10.10.10.2  AUTO
          NET ALLOW  10.10.10.2  AUTO    PASSWORD=123
           
      c  NET PING <IP address>
          进行网络测试用。就象平时用到的PING命令
参数：
          IP address------>> IP地址，一般是指local机的IP地址
例：
          NET PING 10.10.10.2
 
      d  NET RESET
          中断调试进程，需要重新使用NET ALLOW命令指定IP地址。

      e NET STOP
        中断调试进程，中断网络，需要重新使用NET START命令启动。

      f  NET HELP
          帮助命令。

      g NET STATUS
        显示网络状态及连接状态。

6、在SOFEICE的命令行下，启动target机器的网络。
      NET START 10.10.10.1
      NET ALLOW  ANY AUTO
      NET PING    10.10.10.2           

7、local机，需要一块网卡（型号任意），要求不多，只要能联上网，能联通target机就行。

8、local机，在SOFTICE目录下有一个程序-----SINET.EXE
      在命令提示符下：运行SINET.EXE
语法：
        SINET <target IP address> [<password>]
参数：
        target IP address------>>target机的IP地址
        password            ------>>验证需要密码
例：
        SINET  10.10.10.1
        SINET  10.10.10.1    123

可能会遇到以下两种情况：     
第一种
c:\nugega\ice4.05>SINET 10.10.10.1
Target is 10.10.10.1
Local host name is 'ddxia'
Initialized
Opened
Authenticating...Unable to connect to  10.10.10.1

原因：a 有可能是别的机器正在远程调试
            b 没有输入密码
            c 密码错误
 
第二种
c:\nugega\ice4.05>SINET 10.10.10.1
Target is 10.10.10.1
Local host name is 'ddxia'
Initialized
Opened
Authenticating...O.K.

一看就知道是成功啦！这时会弹出一个窗口，出现对方的SOFTICE窗口。YEAH!!!!!!  想干嘛就干嘛啦！
