macromedia传统的操作方式
flash5继承了Macromedia公司软件的一贯风格，如果您有操作Macredia公司的firework和Dreamweaver的经验，即使您以前没弄过flash5，您也会很快就上手的。
  Macromedia公司软件的一贯风格有：
1.允许自订键盘快速键。您可以按照您设计的需要，把您常用到的键键盘操作的快捷键，提高您操作的效率。
2.直观的程序启动条。通过程序启动条，您可以方便地打开您需要的持续。
3.面板化了的工具配置。在这里，您可以设置各种工具的参数。
4.主选单的结构。包括各种选项。
5.浮动面板的管理。把个种元素的控制面板化，直观，方便。
6.贝兹曲线工具。用过向量绘图软体像 FreeHand、Fireworks 都会知道“贝兹曲线工具”，即时我们所说的节点作图工具。通过贝兹曲线工具，您不用买昂贵的压感手写板，也能的到满意的矢量图。flsh5在作图上的最大改进之一，就是贝兹曲线工具的引用，这必将大大改变flsh4作图能力底下的尴尬。
7.颜色选取方式。用过Dreamweaver、Fireworks 的网友，一定能很快掌握flsh5的配色法----三者使用的是同样的配色法通过提供的调色板和其它工具，你能很容易的到您需要的颜色。
