第二页 为你的世界添加色彩

首先，我们谈一下将颜色加到网页要素上。

颜色

颜色属性并不陌生，它的用法很象HTML中使用的参数值。

    B { color: #333399 }

根据这项CSS规则，浏览器将网页上所有的加重体字以设定的颜色显示，象这几个字那样。

有3种方法用于设定你所需要的颜色：

  • 颜色名称

    CSS所用的颜色名称同我们常用的称呼方式。16种基本色为aqua, black, blue, fuchsia, gray, green, lime,maroon, navy, olive, purple, red, silver, teal, white, and yellow.

    但是Netscape和Microsoft的浏览器还认可数百种其它色彩名称，参见 HYPE's Color Specifier.

  • 16进制（hex）色彩控制

    使用16进制数可实现对色彩的更精确的控制，其格式为 #336699。16进制（hex）色彩控制详见Webreference.com。

    CSS还支持某些hex值的快捷计数法。比如， #336699可以被称为 #369。浏览器会将其按照16进制格式阐释。

  • RGB值

    对于习惯于RGB计数法的用户，这里将提供一种全新的色彩设定方法。RGB法通常用于图象应用软件例如 Photoshop。利用RGB设定色彩的方法如下：

    B { color: rgb(51,204,0) }

    RGB的数值范围从0到255，R代表红色，G代表绿色，B代表蓝色。

    IE 3不支持RGB格式，但是4.0版本的浏览器都支持。
