UnderlyingValue 属性

      

指示数据库中 Field 对象的当前值。

返回值

返回变体型值。

说明

使用 UnderlyingValue 属性从数据库返回当前字段的值。UnderlyingValue 属性中的字段值可见于事务并可能是另一个事务最近更新的结果。这一点不同于 OriginalValue 属性，该属性反映的是最初返回到 Recordset 的值。

与使用 Resync 方法类似，但 UnderlyingValue 属性仅从当前记录返回指定字段的值。该值与 Resync 方法用于替换 Value 属性的值相同。

当与 OriginalValue 属性一起使用该属性时，可以解决批更新时出现的冲突。
