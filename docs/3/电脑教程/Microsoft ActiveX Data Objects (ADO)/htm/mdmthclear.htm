Clear 方法

      

删除集合中的所有对象。

语法

Errors.Clear

说明

对 Errors 集合使用 Clear 方法，以删除集合中全部现有 Error 对象。发生错误时，ADO 将自动清空 Errors 集合，并用基于新错误的 Error 对象填充集合。

某些属性和方法返回的警告显示为 Errors 集合中的 Error 对象，但并不终止程序的执行。在调用 Recordset 对象的 Resync、UpdateBatch 或 CancelBatch 方法；Connection 对象上的 Open 方法；或设置 Recordset 对象上的 Filter 属性之前，调用 Errors 集合上的 Clear 方法。这样，就可以读取 Errors 集合的 Count 属性，测试所返回的警告。
