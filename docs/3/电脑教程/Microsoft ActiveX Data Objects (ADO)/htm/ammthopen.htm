Open 方法

      

检索多维查询的结果并将其返回到单元集。

语法

Cellset.Open Source, ActiveConnection

参数

Source   可选，变体型。计算出有效的多维查询，如多维表达式 (MDX) 查询。Source 参数与 Source 属性相对应。有关 MDX 的详细信息，请参阅 Microsoft Data Access SDK 中的相关文档。

ActiveConnection   可选，变体型。计算出一个字符串，用于指定有效的 ADO Connection 对象变量名或连接的定义。ActiveConnection 参数指定在其中打开 Cellset 对象的连接。如果传递该参数的连接定义，ADO 将使用指定参数打开新连接。ActiveConnection 参数与 ActiveConnection 属性相对应。

说明

如果其某个参数被省略，并且在打开 Cellset 之前尚未设置其相应的属性值，则 Open 方法将产生错误。
