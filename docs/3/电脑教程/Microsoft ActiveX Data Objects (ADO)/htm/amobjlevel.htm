Level 对象

            

包含一个成员集，其中每个成员在分级结构内均有相同的等级。

说明

使用 Level 对象的集合和属性，可以：

  • 使用 Name 和 UniqueName 属性标识 Level。

  • 使用 Caption 属性返回显示 Level 时所使用的字符串。

  • 使用 Description 属性返回描述 Level 的有意义的字符串。

  • 使用 Members 集合返回构成 Level 的 Member 对象。

  • 使用 Depth 属性返回从 Level 的根起算的级别号。

  • 使用标准的 ADO Properties 集合获得有关 Level 对象的其他信息。

Properties 集合包含由提供者提供的属性。下表列出了可能可用的属性。实际属性列表可能会由于提供者的具体情况而有所不同。有关可用属性的完整列表，请参阅提供者的文档。

名称 说明
CatalogName 该立方所属的目录的名称。
CubeName 立方的名称。
Description 有意义的级别说明。
DimensionUniqueName 维的明确名称。
HierarchyUniqueName 分级结构的明确名称。
LevelCaption 与级别关联的标签或标题。
LevelCardinality 级别中的成员数。
LevelGUID 级别的 GUID。
LevelName 级别的名称。
LevelNumber 分级结构的根和级别间的距离。
LevelType 级别的类型。
LevelUniqueName 级别的明确名称。
SchemaName 该立方所属的模式的名称。
