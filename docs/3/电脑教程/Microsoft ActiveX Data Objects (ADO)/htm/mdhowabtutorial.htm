教程：地址簿

该范例应用程序用于说明怎样使用远程数据服务来建立简单的、数据识别的 Web 应用程序 — 联机的企业地址簿。本教程适用于想学习使用带有远程数据服务的数据识别 ActiveX 控件的 Microsoft® Visual Basic scripting Edition (VBScript) 和 ActiveX® 程序员，以及想建立以数据为中心的 Web 应用程序而且更有经验的应用程序开发人员。

本教程假定您已了解如何使用基本的 HTML 布局标记和带 ActiveX 控件的程序。

该地址簿教程将指导您学习以下内容：

  • 确定满足运行该应用程序的系统要求。

  • 设置连接数据库的环境，并运行 SQL 查询来复制范例数据。

  • 在 Web 页上建立 HTML 框架并编写 VBScript 代码处理用户事件。

地址簿简介

地址簿范例应用程序是一个用于 Intranet 发布目录的联机地址簿。地址簿使用户可以在一个或多个字段中输入搜索字符串，以便请求有关雇员的信息。为了说明远程数据服务的基本功能，范例应用程序应尽量短小，并使用最少量的对象和搜索字段。

应用程序接口包含如下部分：

  • 文本框作为有关雇员的不同种类信息的搜索字段使用。

  • 命令按钮用于建立查询，清除搜索字段，更新雇员信息数据库，或取消挂起的更改。

  • 类似电子数据表的网格用于显示搜索结果，并通过不可见的 RDS.DataControl 数据绑定对象.绑定到后端数据库。

  • 按钮用于定位网格中显示的数据行。
