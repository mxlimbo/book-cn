Columns 集合

         

包含表、索引或关键字的所有 Column 对象。

说明

Columns 集合的 Append 方法对于 ADOX 是唯一的。可以：

  • 使用 Append 方法将新列添加到集合中。

其余的属性和方法对于 ADO 集合是标准的。可以：

  • 使用 Item 方法访问集合中的列。

  • 使用 Count 属性返回包含在集合中的列的数目。

  • 使用 Delete 方法从集合中删除列。

  • 使用 Refresh 方法更新集合中的对象，以反映当前数据库的模式。

注意   如果 Column 尚未存在于已追加到 Tables 集合的 Table 中，则在将 Column 追加到 Index 的 Columns 集合时会出现错误。
