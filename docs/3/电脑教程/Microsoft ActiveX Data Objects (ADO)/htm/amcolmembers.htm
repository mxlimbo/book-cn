Members 集合

         

包含对应于轴上的级别或位置的 Member 对象。

说明

Members 集合用于包含如下成员类型：

  • 立方中组成级别的成员。这些成员包含在 Level 对象的 Members 集合中。例如，使用多维模式和数据概述中的范例，Countries 级别的四个成员是 Canada、USA、UK 和 Germany。

  • 是分级结构中指定成员的子的成员。这些成员由父 Member 对象的 Children 属性返回。例如，再次使用相同的范例，Canada 成员的两个子成员为 Canada-East 和 Canada-West。

  • 定义单元集的轴上指定位置的成员。将使用多维数据中的单元集作为范例，位于 x 轴上的第一位置处的两个成员是 Valentine 和 Seattle。这些成员包含在 Position 对象的 Members 集合中。

Members 是标准的 ADO 集合。使用集合的属性和方法，可以：

  • 使用 Count 属性获得集合中对象的数量。

  • 使用默认的 Item 方法返回集合中的对象。

  • 使用 Refresh 方法更新来自提供者的集合中的对象。
