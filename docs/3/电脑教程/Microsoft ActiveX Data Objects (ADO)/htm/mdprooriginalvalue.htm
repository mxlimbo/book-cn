OriginalValue 属性

      

指示发生任何更改前已在记录中存在的  Field 的值。

返回值

返回 Variant 值。

说明

使用 OriginalValue 属性可返回当前记录中某字段的原始字段值。

在立即更新模式（一旦您调用  Update 方法，提供者会将更改写入下一级数据源）中，OriginalValue 属性将返回在发生任何更改前（即自从上次调用 Update 方法）已经存在的字段值。该值与 CancelUpdate 方法用来替换 Value 属性的值相同。

在批更新模式（只有在调用 UpdateBatch 方法时提供者才缓存多个更改并将其写入下一级数据源）下，OriginalValue 属性返回在发生任何更改前（即自从调用 UpdateBatch 方法）已经存在的字段值。该值与  CancelUpdate 方法用来替换 Value 属性的值相同。与 UnderlyingValue 属性一起使用该属性时，可以解决由批更新引起的冲突。
