使用带 Sheridan 组合框控件的 DataControl

Sheridan 组合框有两个属性可以绑定到 RDS.DataControl 数据。只将组合框绑定到 RDS.DataControl 并不能自动填充“Text”部分（显示当前所选项目的不可绑定属性）。结果在调用 Refresh 方法后仍然显示旧的数据。

解决办法是使用类似以下两个范例 Microsoft® Visual Basic®, Scripting Edition Sub 过程的代码。二者均可在调用 Refresh 方法时，清除 Text 属性。这些范例中 Sheridan 组合框控件的名称是 SSCombo。

Sub SSCombo_InitColumnProps()
   SSCombo.Text=""
End Sub

Sub Refresh_OnClick
   ADC1.Refresh
   SSCombo.Text=""
End Sub
