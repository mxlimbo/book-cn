Name 属性

      

指示对象的名称。

设置和返回值

设置或者返回字符串值。

说明

在集合内名称不必唯一。

在Column、Group、Key、Index、 Table 和 User 对象上，Name 属性是可读/写的。在 Catalog、Procedure 和 View 对象上，Name 属性是只读的。

对于读/写对象（Column、Group、Key、Index、Table 和 User 对象），默认值是空字符串 ("")。

注意

  • 对于关键字，该属性在已追加到集合中的 Key 对象上是只读的。

  • 对于表，该属性对于已追加到集合中的 Table 对象是只读的。
