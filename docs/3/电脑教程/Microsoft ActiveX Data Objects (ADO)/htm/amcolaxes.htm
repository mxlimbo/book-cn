Axes 集合

         

包含定义单元集的 Axis 对象。

说明

Cellset 对象包含 Axes 集合。一旦打开 Cellset，该集合将至少包含一个 Axis。有关如何使用 Axis 对象的详细说明，请参阅 Axis 对象。

注意   Axes 集合中不包含 Cellset 的过滤轴。详细信息，请参阅 FilterAxis 属性。

Axes 是标准的 ADO 集合。使用集合的属性和方法，可以：

  • 使用 Count 属性获得集合中对象的数量。

  • 使用默认的 Item 方法返回集合中的对象。

  • 使用 Refresh 方法更新来自提供者的集合中的对象。
