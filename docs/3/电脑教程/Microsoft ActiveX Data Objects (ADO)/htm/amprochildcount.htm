ChildCount 属性

      

指示在分级结构中当前 Member 对象是其父的成员的数目。

返回值

返回只读的长整型整数。

说明

使用 ChildCount 属性返回 Member 所拥有的子的估算值。Member 实际的子可由 Children 属性返回。

对于 Position 对象的 Member 对象，返回的最大值为 65536。如果实际子的数目超过 65536，返回值仍为 65536。因此，应用程序将把 65536 的 ChildCount 解释为等于或大于 65536 个子。

对于 Level 对象的 Member 对象，使用 Children 集合上的 ADO 集合 Count 属性确定子的确切数目。如果集合中的子数目很大，则确定子的确切数目会比较慢。
