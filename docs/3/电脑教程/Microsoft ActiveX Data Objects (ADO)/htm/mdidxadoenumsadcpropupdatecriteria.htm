主题   指定哪些字段可用来监测在数据源的行的优化更新过程中，与 RDS Recordset 对象的冲突。常量指定如果关键字有更改（即，已经删除数据源行）、数据源行中的任何列有更改、与记录集的更新列相同的数据源行的列有更改、或者如果数据源行只是被访问过，则应当检查冲突。通过记录集的 Update Criteria 动态属性使用这些常量。
