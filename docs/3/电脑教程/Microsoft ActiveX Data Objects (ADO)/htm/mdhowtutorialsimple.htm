建立简单的远程数据服务应用程序

可以使用 RDS.DataControl 对象创建可访问活动数据库的简单 Active Server Pages 文件。

该教程采用“最小代码”法，通过开发人员指南提供使用 RDS.DataControl 的详细信息。在该教程中，您将连接到 Microsoft® Access Database，显示在数据绑定网格中设置的结果，并添加在所显示的 Recordset 中定位的功能。如下过程说明创建该简单应用程序的步骤：

  1. 标识数据库。

  2. 插入网格和 RDS.DataControl 对象。

  3. 添加 HTML 控件。

  4. 添加代码向数据库发送查询。

  5. 添加代码向数据库提交更改。

  6. 添加在所显示的记录集中移动的代码。

  7. 查看操作中的代码。

如果要查看该示范应用程序的完整版本，请访问 http://<webservername>/MSADC/Samples/ADCTest.asp。其中，<webservername> 是 Web 服务器名。
