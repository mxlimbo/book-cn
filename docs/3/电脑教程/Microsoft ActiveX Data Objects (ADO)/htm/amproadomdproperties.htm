ADO MD 属性

ADO MD 属性总结

属性 说明
ActiveConnection 指示当前单元集或目录当前属于哪个 ADO Connection 对象。
Caption 指示显示 Level 或 Member 对象时所使用的文本标题。
ChildCount 指示在分级结构中当前 Member 对象是其父的成员的数目。
Children 返回在分级结构中当前 Member 是其父的 Members 的集合。
Count 指示集合中的对象数目。
Depth 指示在 Level 和分级结构的根级别之间的级别数目。
Description 返回当前对象的文本说明。
DimensionCount 指示轴上的维数。
DrilledDown 指示成员是否已“到头”。
FilterAxis 指示当前单元集的过滤器信息。
FormattedValue 指示单元值的格式化显示。
LevelDepth 指示在分级结构的根和成员之间的级别数。
LevelName 指示成员的级别名。
Name 指示对象的名称。
Ordinal（单元） 通过单元集中的位置唯一标识单元。
Ordinal（位置） 唯一标识在轴上的位置。
Parent 指示在分级结构中是当前成员的父的成员。
ParentSameAsPrev 指示该位置成员的父是否与直接在前的成员的父相同。
Source 指示单元集中数据的源。
State 指示单元集的当前状态。
Type 指示当前成员的类型。
UniqueName 指示当前对象的明确名称。
Value 指示当前单元的值。
