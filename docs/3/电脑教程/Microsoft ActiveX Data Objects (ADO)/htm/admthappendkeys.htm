Append 方法（Keys）

      

将新的 Key 对象添加到 Keys 集合。

语法

Keys.Append Key [, KeyType] [, Column] [, RelatedTable]

参数

Key   要追加的 Key 对象，或要创建和追加的关键字的名称。

KeyType   可选，长整型值。指定关键字类型。Key 参数与 Key 对象的 Type 属性相对应。

Column   可选，字符串值。指定进行索引的列的名称。Columns 参数与 Column 对象的 Name 属性值相对应。

RelatedTable   可选，字符串值。指定有关的表的名称。RelatedTable 参数与 Table 对象的 Name 属性值相对应。

说明

Columns 参数可使用列名或列名数组。
