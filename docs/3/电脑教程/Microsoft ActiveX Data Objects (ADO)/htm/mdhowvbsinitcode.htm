VBScript 初始化代码

地址簿应用程序的 VBScript 节首先声明变量 myQuery。该变量将包含由搜索字段文本动态建立并发送到数据库的 SQL 查询。然后，Find_OnClick 子过程设置该变量的值。

Dim myQuery

其余的初始化内容被包含在 Load 子过程中。其中包括给数据网格添加标题。这样的 WHERE 子句使记录无法取出，从而只有列标题可以提供。请复制并粘贴该代码到标记 SCRIPT 和 /SCRIPT 之间。

Sub Load
   Grid1.Caption = "Arcadia Bay Corporate Phone Directory"

   ' 仅用列名称对数据网格进行初始化。
   SControl.SQL = "Select FirstName, LastName, Title, Email, " _
      & "Building, Room, Phone from Employee where 2 < 1"
   SControl.Refresh
End Sub
