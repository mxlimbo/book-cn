设置图象的位置
 

　　数组的便利之处在于数组中的每个元素都能按照在数组中的顺序自动分配到指定的位置。JavaScript数组中的第1个元素的位置为0，第2个元素的位置是1，依此类推。

　　所以bendyList[0]引用的数组项目是位于位置"0"的图象,即bendy000.gif而bendyList[1]指向位于位置"1" 的图象bendy001.gif.在本例中我们将图象的编号同其在数组中的位置编号相一致以便尽量避免将两种数据搞混。

　　下面,我们生成变量daPosition,我们用这个变量指向 bendyList数组中特定的图象。 

    // daPosition will point to successive images in 
the bendyList array

daPosition = 0;

　　由于daPosition的初始值设为0，所以bendyList[0]和bendyList[daPosition]是等值的。
