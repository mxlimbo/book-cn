用Javascript制作的闪烁动画效果
 

　　这个交互式动画使用JavaScript在页面中穿梭显示各种图象，并且无须重新载入页面就可改变对象的类型。

　　当你用鼠标滑过图象底部的5个单另图标时，闪烁动画的主体会相应发生变化。要制作和显示该动画，你需要配备Netscape Navigator 3.0以上并且支持JavaScript的浏览器版本。

　　你可以先看看屏幕快照预览一下效果。

　　在Javascript代码中设定了代码的结构和图象单元显示的方式，改变的速度，以及其它幕后技巧。
