第二页：剪切块方式

剪切块方式，以Monty Python和South Park为代表，在纸上剪切人物和场景，把它们连接或层叠，一祯祯地捕捉它们的动作。然而，数字时代使这种单调的过程过时。当Comedy Central把South Park当作合作伙 伴时，Matt Stone和Trey Parker使用纸和剪刀做动画，然后用高端SGI工作站尽量减少动画中的剪切的痕迹。在三维中渲染平面场景看起来可能有点过分，但是结果很好，而且这个过程也很快。

剪切块方式趋向于有一种脱节的、粗糙的、抽象聘贴画一样的模样。它不必一祯祯地重画。因此它可以富于细节和色彩。传统地，当动画中的人物在祯之间的姿势完全不同时，用剪切块方法是一种容易的动画方式。同时，这种有限运动的方法可以很好的移植到网上，在这里，祯重画和文件大小限制了光滑动画的可能性。

可以用扫描仪作为相机创建剪切块动画。或者扫描你的人物，然后用一个2-D动画程序，如Flash来定位和移动他们。
