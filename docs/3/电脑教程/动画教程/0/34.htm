第二页：点和线、矢量和形状

成功动画之路始于明锐的观察：以观察你喜欢的动画开始。你会注意到它们可能包含一些（如果不是全部） 以下的基本设计元素。

1.点和线

什么是点？
[ .]
点是聚焦的中心，设计中的重点。它可以是一个实际的对象，或者通过方向指引的暗指。当你创作动画的单独祯时，你需要知道点的位置。它什么时候移动和为什么移动？通过跟踪你的构件的聚焦，可以保证你的用户聚焦在你的构件上。没有聚焦，你的动画就没有要领（字面意义上和寓意上）。

线
[-----]
线是点的延续。所有的线都有方向：垂直的、水平的或倾斜的。水平线，模拟人类休息的形式，趋向于给人抚慰的、平静的感觉；垂直线有一种运动的潜能。斜线是这三者中最动态和活跃的，因为它们看起来好象要跌落到水平或升起为垂直。水平线和垂直线的组合被认为是最稳定的形式因为那是我们在日常环境中体会到的。和点一样，线可以是实际的或隐含的。

2.方向力量/矢量

方向力量，或矢量，或者是线或者是运动，并且可以或者是实际的或者是隐含的。矢量围绕和通过创作引导读者的视线，如一只带皮带的狗。作为一个动画制作者，你需要特别关注矢量给观众的感觉 - 从哪儿开始，在哪儿结束，中间的所有事物。

有三种主要的矢量：

图形矢量 - 由静态物体建立。
索引矢量 - 由指向某个方向的物体
建立（如一个箭头或伸出的手）。
运动矢量 - 由在某个方向移动的物
体建立。

每一种矢量都可用在改变方向或感知事件。混合和匹配这些矢量可以取得更多的效果：

连续矢量：不是一个图形、索引或
指向相同方向的运动矢量，而是进
一步强调了方向和提供了稳定和持
续的感觉。
聚合矢量：多于一个点汇聚到一
点。这会产生一种生动的、聚焦的
感觉。
分叉矢量：从一个点分离出的矢
量。

通过密切注意矢量工作的方式，你会很快把观众聚焦到你想要的地方。完美你的矢量可以给每祯或一系列祯一种一致的感觉。编排一系列创造性的矢量，你的动画会有目的性和连续性。

3.形状

形状，一个很难定义的术语，是一个由线、颜色、块等接合成的封闭的区域。形状可以是有机的（自然界中出现的流体的、不规则的形状）或几何的（刚性的、有角度的、数学的形状）。当使用形状时，注意你创建的形状（阳图）和背景中的区域（阴图）。这两种形状对于动画同等重要。

通过使用阳图和阴图，你可以得到一些有趣的效果。“形状-背景翻转”是翻转阳图和阴图的技术。在动画中，你不仅可以把这种技术用在一个单独的祯中，而且可以用在一系列祯中，这样使动画中的构件的活动范围增大。这种技术也可以应用到声音上，可以把背景声音移到前景，把前景声音移到背景。
