第六页：最后时刻的技巧

我们几乎到了动画教程结束的时刻了。但在我们结束之前，我愿意留给你即将逝去的礼物：九个更成功地在网上做动画的经验。

1.别把你的最后输出限制在任何一种媒体类型中

但是不管你选择什么方式，你最终还要把它集成到HTML文件中。尝试用多种动画媒体类型创建网页，然后把HTML元素和类型集成到你的最后设计中。只要有可能，把文本作为文本，而不是作为Flash动画或 QuickTime电影中的一部分。在设计中应把CSS发挥到极致。

2.在Web动画中用循环媒体

短的动画循环在Web上工作得很好，因为文件很小，可以吸引一般搜索者的短暂注意。如果处理得当，你的动画循环将天衣无缝，就象下面例子所演示的。

3.如果你在一个小的动画舞台上工作 - QuickTime动画制作者常这样 - 从大处着想。

如果动画为160*120，你会失去很多细节，所以要记住动画最终的尺寸，避免用太多繁琐的细节。

4.在Web上，全屏动画确实能给人深刻印象，所以如果可能，允许你的动画进行缩放。

在用Flash的情况下，动画质量没有损失。如果用GIF89或QuickTime，创建一个稍微比你要显示的尺寸小的动画。把它放大一点，质量损失很少，但节省了文件大小。

5.告诉人们下面将发生什么

如果你的动画格式需要很长的下载时间，别只告诉你的观众“正在下载......”。为他们提供一个小的5KB的GIF动画或声音。对于Flash用户，你可以用一个简单的、小的简介动画。一个简单的循环，有少数颜色的划线的动画可能是你最好的选择。如果你的动画很基本，就可以在坚持文件大小限制的同时最大化祯数。

6.使用替代物

如我们前面提到的，如果你使用插件或dHTML，并且想得到更广泛的观众，考虑替代物，通常是GIF89。在一些情况下，动画格式的直接转换是可能的。在其它情况下，利用某种媒体的长处创建不同的动画是明
智的。

7.使用合适的工具

经过一段时间后，各种形式的动画会看起来相同。某种程序的好处如果用得过多，可能变成缺点。某些程序或工具鼓励你以某种特定的方式开发你的动画。尽量避免仅仅因为某些技术比较方便就过分使用之。看看你正工作的动画。你用的工具帮你实现你的想象了吗？或者你正走在最少反抗之路上？

8.从其它媒体中得到灵感

在创建Web动画时，对其它艺术形式的熟知可能是你的无价之宝。看看电影、录像、照片、绘图、雕塑、表演、写作、戏剧甚至电台节目，它们之中可能有可以运用到你的动画设计中的主意或理论。记住，Web还是相对新的，Web动画更新。应用其它媒体中的经验，甚至开拓新的领域。你会发现一些精彩的东西。

9.最后一点：欺骗

如果没有好的参考，很难正确表达运动。即使事先仔细观察你的对象，也很难得到实际动画过程的足够信息。因此最好用摄像机捕捉不同物体的运动。在你的VCR上播放你最喜欢的动作系列。剪辑杂志中的好作品。抓取吸引你的注意力和感动你的东西。把这些资源集中起来，以便在你需要的时候使用。动画只是现实的夸大，但是可以把这些例子作为跳板。
