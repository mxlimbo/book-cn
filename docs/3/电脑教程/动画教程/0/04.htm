信息的合适媒体

　　接近观众的最好方式是什么？在你确定媒体方式时，文件大小、渲染速度和信息的内容都很重要。

　　有一些简单规则。假如你在制作美学照片拼贴画。因为Flash在处理位图时表现很差，QuickTime或dHTML或许是最好的方案。但是如果你打算做一个说明性的动画，Flash将是最好的方式。它快速、可调整大小，文件又小，样子好看。除了广泛分布这个因素，GIF89绝不是最佳的可选媒体，虽然它是不那么通用的媒体类型的好的替代品。

　　如果你的作品不容易被分类，就很难选择合适的方式。你不得不权衡每种媒体的优点和限制。考虑你可能遇到的bug和问题。这些值得你去冒险吗？

　　为了帮你指条明路，我们提供了各种主要动画技术的例子。我们也提到它们的优点和缺点。希望这些信息使你的对比和选择工作容易些。
