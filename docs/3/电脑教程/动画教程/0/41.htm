第四页：制造一些噪音

可以以不同的方式制作声音。

使用不再有版权的歌曲、音轨和声音效果。

购买整洁分类的声音库。不过这些声音也有缺点：因为它们不是专为你的动画制作的，你会发现这些声音不完全适用。而且这些声音也容易被认出来。因为它是如此地方便，这些声音往往到处可见，会给你的动画一种似曾想见的感觉。为了解决这个问题，你可以在用它们之前用声音编辑程序对它们进行修改。

从头开始设计你的声音。George Lucas的Skywalker Sound以创造性的声音设计而出名。当一个麦克风跌落到一个旧的、真空管电视后面的时候，Star Wars 中的轻骑兵的声音被发明了。Stromtrooper中的暴风 雪的声音是通过搅动金属电缆的声音产生的。如果你想得到愚蠢的声音，最好用自己的嘴。Terry Gilliam会把一个毛巾和麦克风放在头上，制作各种他需要的噪音。用嘴配音简单、便宜和容易 - 你可以得到你想要的。

有经验的音频工程师通常在手边有很多工具，如麦克风、合成器、取样器、压缩器、limiter，和一整套齿轮装置。这些装置可以帮你制作音频，但是如果没有也不用担心。现在是数字时代，很多工作室中的昂贵的工具已经被功能强大的、买得起的软件代替。

我建议每一个对音频感兴趣的动画制作者应该有一个可以增强、编辑、合成、扭曲和压缩声音文件的声音编辑程序。很多音频编辑程序允许你降低或提高声音的斜度，或者加速或减慢声音。音频编辑软件也允许你不费力地裁剪和编辑文件。我发现SoundEdit 16 for the Mac是一个很好的工具，因为它是捆绑在Deck II硬盘记录软件中，你可以花很少的钱得到一个强壮的音频录制套件。在PC上，Sonic Foundry的 Sound Forge是一个全能的声音编辑器。它不包括硬盘记录软件，但是程序本身的特征已经很值了。

我们希望在今天的课程里你已经想好如何把声音合成到动画中。在明天的课程里，我将讨论多种可用的动画方式，和如何把它集成到你的工作中。
