第三页：Flash特征

运动路径
运动路径允许你为物体设置直线或曲线的运动轨迹。这是一个非常直接的方式。然而，这种易用的方式趋向于诱惑动画制作者建立不必要的复杂运动路径 - 而实际上简单的路径更有效。

关于如何设置运动路径，可以参看Flash教程。

屏蔽
屏蔽允许你保持动画的某些部分可见，而其它部分隐藏在幕后。一个对象可以被全部或部分模糊化。“填充”的对象（符号、文本或形状）是可见的；“非填充”区域是隐藏的（注意：每个屏蔽层只能有一个对
象）。屏蔽影响它下面的层，因此你在屏蔽层指定的透明或半透明区域会为显示下面的内容开一个洞。记住屏蔽不一定是静态的，它们可以象其它层一样运动。

现在说明如何创建屏蔽：选择或建立一个你想透过透明的洞显示物体的层。选择层后，选择Insert:Layer 在其上创建一个新层。命名新层，然后从层的弹出菜单中选择Mask。然后把一个填充（动画或静态）形
状、文本或符号放在这个层上 - 这些是透明的，而且任何未填充的区域将阻碍这层下面的信息。每层只放一个物体。Flash忽略屏蔽层的位图、梯度和alpha通道。最后一步：从Layer弹出菜单选择Show Mask。

形状线索
形状线索特征在Flash 3.0中开始出现而且非常有用。形状线索允许你建议如何通过把两个不同形状上的点对应起来实现变形。如果你熟悉Morph或 Elastic Reality，你对这个过程会很熟悉。

为了实现形状线索，在形状渐变序列中选择第一个关键祯，然后选择Modify:Transform:Add Shape Hints。你现在应该可以看到在形状上有一个有字母"a"的红色的圆圈 - 把这个"a"移动到你想屏蔽的地方。现在到渐变序列的最后一个关键祯，这里形状线索的结束以一个绿色字母"a"的圆圈表示。移动它与起始点相匹配。现在运行这个电影看看你都做了什么。不完全满意，是吧？调整形状线索的位置直到满意为止。

“洋葱皮”
“洋葱皮”是动画制作早期留下来的术语，是指动画制作者把一层非常薄的半透明纸放在刚画的最后一祯上，用它做参考画下一祯。在数字世界里，“洋葱皮”是允许你看动画的周围祯时创建或编辑当前祯的有用的特征。

好东西太多了
各种效果的合成会使事情有趣。不要只改变物体的比例，可以试试对它进行缩放、旋转、改变颜色、位置形状等。但是别做过头 - 否则你的动画看起来可能面目全非。用各种技术的合成达到你想做的，不要验证Flash的威力。
