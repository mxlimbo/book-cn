第二页：两种动画技术

我喜欢把Flash作为Web的AfterEffects。如果你熟悉AfterEffects的话，你会注意到它的几何特征与Flash的渐变很相似。渐变是在时间期限内处理两个关键祯之间的祯的技术。它会很清晰地改变比例、位置、颜色、旋转和形状。渐变是使物体运动的最快和最容易的方式，而且得到的文件也很小。但是要小心：虽然渐变是使图像活动的最明显和容易的方式，但它不总是最好的选择。

与渐变相反，一祯祯的动画是一种手工改变每祯的动画技术。这种技术一般用在物体进化的情况。简单的运动（位置或比例的改变）适用于渐变。一祯祯的动画的文件比渐变的文件大。

最好是把这两者结合起来达到既有趣文件又小的效果。

除了这两种特征，Flash有很多帮你容易做动画的工具和效果。
