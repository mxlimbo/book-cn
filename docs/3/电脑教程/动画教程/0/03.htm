开始网页动画制作

　　在以后的几天里，我们将深入动画的各种形式。但是今天我们只讲讲如何确定用哪种动画。

　　确保你的动画有主题/故事/要点

　　不论你的动画是简单的漂亮颜色的运动，还是讲一个有人物的长故事，最重要的是做动画时记住你的"故事"的目标。它们在质量上有很大差别。

　　善于创新

　　当你准备在互联网上制作动画时，任何事都是可能的 - 只要你保持一个开放的思想，同时为了实现你的思想不断地探索新的方式。在本教程中，我们包含了我们在HotWired工作时增强动画表现力的一些经验和技巧。我们鼓励您在学习和开发自己的动画时发现我们的错误。你也可以向网上的动画和其它媒体学习。

　　事先作好计划

　　无论你在做什么，史诗般的冒险或者三祯的动画，故事脚本都很有用。The Wrong Trousers是一个很好的例子。你的故事脚本不一定是一件艺术品，只要能清晰地表达你的思想并使看到的人能理解就够了。可视化你要做的东西，试验不同的角度和方式，尝试不同方式的组合。相信我们，这会节省您很多时间。

　　你可以在计算机上做你的故事脚本，但是用笔和纸做可能更好。这些试验性的工具使你的工作迅速和轻松；在计算机上勾画事物可能把你引向浪费时间在不必要的表现上。

　　如果你的动画强烈依赖于字符，你可能要在字符表单上进行勾画。在不同的位置和角度显示字符，探索不同的情绪，集中在任何显著的特征上。另外，这些草图在与其他人或客户合作时尤其有用。

　　查看下载大小和渲染速度

　　人们不喜欢在网上等。为了避免下载失败，设置一个可行的最大文件大小并一直坚持下去。如果你发现自己经常被限制，可以重新评价动画的类型。不同的媒体对下载速度有不同的容忍程度。一个2MB的动画对QuickTime电影是可以接受的，但在GIF89中是愚蠢的。对于流式媒体（如RealVideo)，文件大小基本上没有限制。

　　有时不能怪罪文件大小。渲染的问题也使问题变慢。渲染速度是计算机把动画化在屏幕上的时间。这在使用Flash和dHTML时尤其重要。

　　了解你的观众

　　谁在观看你的工作？他们在用什么平台或浏览器？这些是你选择何种显示方式和故事类型的最重要的决定性因素之一。一旦你能指出你的观众是哪些人，你就能更好地选择动画的形式。
