第一页：Flash简介

Macromedia的Flash虽然出现时间不长，但已经成为最首要的Web动画形式之一（而且当然是最迷人的方式之一）。Flash的成功来自于它的非常灵巧的本质：既是一种制作工具，又是一种文件格式。

Flash不仅比dHTML易学得多，而且有很多重要的动画特征，如关键祯插补、运动路径、动画面具、形状变形和“洋葱皮”。用这个多才多艺的程序，你不仅可以建立Flash电影，而且可以把动画输出为QuickTime
文件、GIF89文件或其它许多不同的文件格式（PICT、JPEG、PNG等）。Flash与其中每个版本兼容得都很好。

Flash的不足之处是需要一个插件。但是Netscape 5提供了内置的Flash支持（不需要插件）。因为IE 4 的结构化图形技术不支持Flash，我们只有期望 Microsoft在IE 5中内置支持Flash。

Flash现在稳居动画格式的冠军地位。但是你可能不完全相信，就让我们看看它的优缺点吧。
