第四页：使用交互性

当然，Web相对于传统媒体的最大优势之一是它的交互性。你可以看模拟TV，但是你最喜欢的表演绝不会由于你的行为而改变。

保证交互性会使你的故事进行下去。如果你无端地使用交互性，可能比没有交互性更坏。通过鼠标点击触发一个事件是最常用的交互性的方式，但它不是唯一的一种。交互性的其它形式还包括通过窗体元素或拖放对象的用户输入。使你的交互动作有趣和有创新性，想象用户操作的方式然后按此设计交互性。
