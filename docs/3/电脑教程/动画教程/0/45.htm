第四页：绘画

幸亏是星期六早晨的卡通片，你可能很熟悉绘画方式 - 二十世纪最流行的方式 - 所以我不用做太多的解释。基本上，通过手工绘画图像，然后把每幅图像捕捉到相机或摄像机的祯里。把这种方式应用到网上，只需在计算机上组装动画的祯，或者在计算机上手工绘画。


例子：Absolut Panushka
动画制作：Jonas Odell and Stig Bergqvist
网站：Absolut Vodka
URL：www.absolutvodka.com
媒体：QuickTime

一种不太传统的技术是直接在胶片上绘画。黑色胶片表面有一层感光乳胶，可以在其上划线；线条开始时是白色的，但是你可以在胶片的另一面加上颜色。或者用干净胶片，其上没有感光乳胶。一次在胶片上画一祯，或在一长条上画一个长的图像。你也可以在长条上拼贴元素，只要它们不阻碍胶片扫描过程。这种方法在涉及到大的、有很多色彩的图像时很有用，因为精确控制运动或者不可能，或者不特别有效。

这种方法可以用Photoshop或Painter数字化实现。或者手工绘画然后扫描，或者全部用计算机做，你可以用一种计算机程序如AfterEffects和Premiere把胶片条格式输出到QuickTimes中。
