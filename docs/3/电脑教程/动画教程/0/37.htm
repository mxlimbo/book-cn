第五页：六个设计法则

除了动画和设计的十条戒律，考虑下面的规则：

大小和比例

大小和比例可以指动画中的元素或动画本身。我总是发现大小在计算机上很受限制，因为在计算机的小屏幕上很难创建宏大的东西。与黑洞洞的电影院相比，计算机屏幕周围充满诱惑（发来的传真、狂劲的音乐，还有赤膊的邻居在你的窗外玩绳球）。处理这个问题的好方法是把劣势转化为优势：或许应该朝相反方面努力，你问题变小。

一致性和多样性

当动画中的每个物体都与一个相似的物体以某种一致的方式相关时，这就是“一致性”。多样性是物体之间的差别。太多的一致性使人厌烦，太多的多样性又太混乱。在动画中应达到这两者的平衡。

重复和节奏

在动画中重复元素可以保持持续性和增加强调。也可以有一种步伐的感觉和叙事性。

平衡

有两种平衡：

1.对称平衡 - 物体的两边对称。这种平衡很容易达到，但是趣味性明显地少了。

2.非对称平衡 - 不相似的物体同时出现。创建起来困难些，但更生动迷人。

强调和部属

把一个物体/主题聚焦到另一个上，加强了动画信息。我们以前讨论过，可以用很多不同的方式进行强调。与之相反，如果你想得到“同一”，不要强调任何一个物体。这在强调缺乏多样性时尤其有效。

对比

物体之间的对比增加了强调并且使事情有趣。对比的一般形式包括：黑暗/明亮，高兴/悲伤，大/小，善/恶，纹理/光滑。对比也可以通过颜色的色度、值和强度实现。

今天的动画和设计理论课到此结束。这些理论和技巧只是一个指导方针，不必墨守成规。多实践，形成自己的理论和技巧。
