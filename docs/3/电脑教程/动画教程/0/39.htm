第二页：创建声音效果

如何使事情真实

声音效果为你的动画增加了真实感和深度，使动作和设置更可信。当你为动画创建声音效果时，不一定要录制你要描绘的真实声音。假设在你的动画中，一个玩具狮子狗在厨房地板上跑来跑去。你可以找一只狮子狗，把麦克风系在它的腰上，然后录下它在油布上跑的声音。或者你可以作假，通常比真事更可信，而且更容易。在电影Monty Python and the Holy Grail中，Terry Gillian扮演了一个诸侯，他用两个椰子壳的敲击声模拟骑马。体会一下下面例子中的声音：

例子1：马蹄声（椰子壳）
例子2：火（弄皱的纸）
例子3：泼溅的水（手指）
例子4：一群野猴（喜欢香蕉的朋友）

放置你的声音

如果你用过Photoshop 4.0，或用过层叠样式表，你可能熟悉层叠放置图像。大多数声音编辑程序，还有一些以网络为中心的技术如Active Movie Control，允许你层叠放置声音。通过把一个声音放在另一个上
面，可以指明前、中和背景声。可以尝试把前景声放到背景，或者相反。声音制作的过程是一个学习和实践的过程。
