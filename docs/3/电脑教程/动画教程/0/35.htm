第三页：块、空间和光线

4.块/深度/体积

块通常用来描述一个三维的物体。在如电影或Web这样的二维媒体的限制下，块可以通过阴影和透视的技巧实现，给出物理重量的表现。


例子：The Shirt

可以用一些技巧来模拟透视：

层叠图像
当不同的对象层叠时，它们看起来好象一个在另一个前面，这样离观众更近。这种对远和近的模拟实现了第三维。

变小
当一个对象变小，它看起来后退到远处，远离观察者。通过相同的方法，大物体显得离观察者近些。

位置
动画舞台底部的物体被认为比顶部的近。因此放置物体时要小心。

模糊
物体消失在远处时，空气中自然的模糊和我们眼睛的不完美导致颜色褪色和物体失去细节。在设计中模拟这些天然的缺陷是创建深度感觉的另一种方式。

5.空间/区域

图片的尺寸和纵横比对动画的结构和创作很有影响。图片的某些部分比其它部分更有会聚力。因此要更加注意人们最注意的区域：地平线、边界和图片的右手
边。

如我们在讨论线时提到的，大部分由水平线和垂直线构成的动画给人稳定的感觉。通过使用水平面可以使动画显得更生动。但是要小心：过多水平面的切换让观众讨厌。去过太空站吗？当天花板旋转而我们静坐不动时，你会感觉你在旋转。你的动画可以利用这种欺骗性。

图片的边界很有吸引力。任何没被放在死角中的物体会感觉到一种离它最近的角的拉力。当一个物体大到几乎可以填充整个图片时，角落会使它感到很拥挤。注意这些拉力，或者中和它，或者强调它。如果你真想给一个物体宽阔的感觉，可以让它触到图片的两边。所以在放置人物时要注意边界的拉力。

由于某种原因，人们趋向于更注意图片的右边。利用这种现象，把要强调的东西放在右边。

6.光线

光线可以帮助设置动画的情绪 - 改变光线，然后一个物体突然出现。光线的来源和方向，还有它的颜色强度，对感知都很重要。例如，日光给人的感觉与荧光灯完全不同。


例子：Absolut Panushka

光线来源也很重要。密切注意光线的来源方向。从后面或前面来的光线使物体看起来单调。从上面或侧面来的光线给人很正常的感觉。从下面来的光线有一种戏剧化或奇异的感觉。

注意投射到或隶属于物体的阴影。用阴影可以装点背景，暗示设置或特定位置，指示季节或一天中的时间，或者提供附加的戏剧化的感觉。

关于光线的一些其它技巧：用光线风格化你的图形 - 高对比值使景象更生动。柔软、精细的对比度变化显得平静温和。如同令人惊慌的音乐，光线的猛烈改变会警告观众有一个未决的变化。你可以操纵光线来强调动画的特定部分（象一个聚光灯）。
