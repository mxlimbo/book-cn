欢迎使用GIF89

　　"GIF不是作为动画平台产生的，虽然它可以做简单的动画。"，这是写GIF规范的CompuServe的代表在1987年说的。

　　事情自从80年代末期开始已经改变了。带宽从两航道的公路成长为八航道的高速公路，CPU的速度成十倍的增长，Web现在是艺术和通讯的论坛，而不只是放置物理论文的地方。然而，CompuServe的GIF专家有一件事是正确的：GIF89不是一个很大的动画媒体。但是它有自己的目的。

　　GIF89的缺陷并不是无关紧要的：文件很大，压缩不好，没有声音能力。另一方面，它可以产生几乎所有浏览器都能看的动画GIF。并且GIF89在做小动画 - 如标志、基本的故事和简单的动画图标 - 时表现很好。它在与其它动画媒体（如dHTML和Flash）集成方面也很好

　　因为它是如此地适合小动画，GIF89赢得了"讨厌的广告标语"格式的名声，很多有创造性的人不把它作为有用的艺术工具 - 因为它的恶名。在下面几页中，我们向你展示几个用GIF89创建的创新动画的例子。例子中包含一些格式上的技巧。当然，真正的格式在细节上。所以我们也会涉及重要的事实真相。

　　下面是我们的记忆库 - 我们在开发无数小时的作品后得到的技巧。
