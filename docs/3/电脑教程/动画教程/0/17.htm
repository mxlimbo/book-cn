第二页：DHTML的正反面

DHTML是为网页增加交互性和多媒体的很好的方式，Dreamweaver是制作dHTML网页的很好的工具。Dreamweaver的一个最显著的优点是它产生dHTML如同HTML一样，与其它可见即所得编辑器相比，其代码令人惊奇地干净。既然它是HTML，你就不必担心你的浏览器是否支持适当的插件。这也意味着搜索引擎可以很容易地为你的内容编目，你也可以很简单地用你最喜欢的编辑器对它做修改。下载时间可以忽略不计。同时，因为Dreamweaver编码很紧密，它产生的交互性可以与Director或Flash想媲美。如果你是一个HTML编码的新手，花20分钟学Dreamweaver后，那些Web专家就会对你刮目相看了。

不幸的是，dHTML不总是阳光和海滩。在今天结束之前，它还是未被编译的数据，所以它的表现不如插件格式。取决于你的机器及其配置，你的显示器的刷新率要受不同程度的罪。而且还有跨平台的兼容性问题。可幸的是，这个问题会随着Dreamweaver的完善和两个浏览器巨人达成一致而消失。最后一件要考虑的事是：虽然Dreamweaver可以制作一些迷人的动画，它底层的技术 - HTML - 重来没被用于动画。Dreamweaver现在没有而且可能永远不能成为象Flash，Director或QuickTime一样强壮的动画工具。但是这并不意味你不能用它做好的工作。
