第五页：Meta Tags, Mouseovers和其它


Meta Tags/Frames

Meta-refresh <meta http-equiv="Refresh" content="35; URL=xx.html"> 允许你在动画中创建更动态、全幅的页面。


例子：HotWired's RGB Gallery
设计者：Mary Spicer
媒体：GIF89/HTML

JavaScript Mouseover

可以用JavaScript把动画与静态图像合成在一起。这样你可以更好地控制时间，也可以给你的页面更多的交互性。象这样的JavaScript阵列允许你为每一幅图像设置装载时间。其它的JavaScript命令允许用户与你的故事交互 - 甚至可以改变故事的结果。


Touch me!

动画背景

4.0浏览器允许GIF89作为背景图像。这样，你可以把动画和静态图像、文本、窗体元素或其它对象交迭在背景上。与老式的静态背景图像相比，它的位置很难百分之百的准确，用的时候要小心。


例子：Red Splash [4.0 browsers only]
媒体：GIF89
文件大小：16KB

静态背景和透明动画

把一个透明动画放在一个页背景图像上面可以减少页面的文件大小。避免用复杂的背景图像（如照片）。
例子：4YB-002
动画制作：David Opp
媒体：JavaScript, GIF89, QuickTime

剪切和重新组合

如果你的动画有大的静态区域，可以对它进行剪切从而节约文件大小。你也可以从动画中移去复杂的元素，而用另一种方式实现（如JPEG）。用表格或样式表重建切片之后的动画。


例子：Webmonkey Mad Scientist Splash
动画制作：Luke Knowland
媒体：GIF89
文件大小：79KB

GIF89的介绍到此结束。多一点耐心和创造性，你就会创建感人的、艺术性的动画。
