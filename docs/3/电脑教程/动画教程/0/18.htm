第三页：好了...开始吧

你可能在听我讲了这么多之后准备使用Dreamweaver了，但是我们还没完全准备好。DHTML建立了几个Web标准，你虽然不必知道Dreamweaver的一切，但仍有必要了解HTML、CSS，甚至JavaScript。你不必记笔记，也没有考试，但是如果你对概念很模糊，最好跳过这篇文章。

最重要的是，你需要对Dreamweaver和dHTML都很了解。如果你不能百分之百地确定具有这些知识，应该读一读Taylor的Dreamweaver教程或他的DHTML教程。尤其是Dreamweaver教程，我们将涉及一些比较高级的主题，而且我不想失去任何读者。

为了创建dHTML动画，你需要有一些必需的工具。最基本的，你需要一个文本编辑器，如BBEdit或 HomeSite，和一个4-plus浏览器。然而，很少有人用这些简单的工具，因此你可能要收集一些其它工具：

一个所见即所得dHTML编辑器

除非你是一个dHTML/JavaScript编码高手，你几乎一定要有一个所见即所得编辑器。很明显，我是一个 Macromedia的Dreamweaver的爱好者，但是GoLive! Cyberstudio Pro也很有趣。大多数所见即所得编辑器允许你使用GUI或手工编辑代码。

一个位图设计程序

动画只在内容是动态时才能用到，所以要有一个好的图像（GIF、JPEG或PNG）编辑程序。位图程序包括 Adobe Photoshop、Adobe Image Ready和Painter。

一个GIF89程序

DHTML几乎消除了对动画GIF的需求，但有时还会用到。所以最好还准备一个GIF89编辑程序。这样的程序包括GifBuilder、GifConstructioin Set、GifMation、Adobe ImageReady和Macromedia Fireworks。

传统绘图媒体和轻松表格

老式媒体，如铅笔、钢笔、墨水和纸，在比特和电子的世界里还有一席之地。有时计算机产生的图像缺乏手工绘制的图像的个性。可以手工绘图后用扫描仪输入计算机。

好了，你已经准备好了，继续吧！
