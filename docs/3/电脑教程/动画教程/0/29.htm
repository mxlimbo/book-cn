第一页：使事情动起来

剪切块方式，以断奏运动为特征，是处理描摹或扫描肖像的很好的审美选择。剪切块方式由于历史需求而发展起来：在计算机动画之前，流体运动意味着大量的工作，一祯祯地处理剪切纸。这种限制不再成为问题，因为计算机承担了缓慢的流体动画。但是还不能过度使用它。光滑的、非急促的渐变可以给动画一种计算过的感觉，失去了自然淳朴的剪切块方式的魅力。然而，流体渐变如果保守地使用会给你的动画一种有趣的机械的、未来派的感觉。

当你为动画准备剪切块方式图形时，可以考虑哪部分图像需要运动。把每个运动元素分隔在自己的层内，因此你可以把每个当作独立的实体。想象每个元素如何运动和如何影响动画的其它部分。如果当一个物体（如一只胳膊或一条腿）运动时，突然出现一部分腹部或以前隐藏的后腿怎么办？你可能要通过借用材质或用其它部分遮盖来重建新暴露的区域，这有时会有些困难。所以在选择要运动的元素时要明智。

当编辑完位图文件后，输出所有单独的层为GIF或PICT，然后把它们输入到Flash中做描摹。一旦描摹了所有的位图，把它们插入不同的地方，象连接纸玩偶那样。在Flash中，轴心连接缺省地对应于物体的中心。要编辑这个点（假如说让胳膊绕肩膀旋转），选择Modify:Transform:Edit Center。同时要记住你不必使用整个物体 - 隔离或合并只对你有用的。例如，声名狼籍的Monty Python foot曾经是一幅文艺复兴时期的作品的一部分。

当可视化动画时，把它分割成一系列“场景”会很有帮助。我们以前说过，故事板可以帮助描绘动画。

在Photoshop教程中，Luke推荐用铁笔（或在lighttable的帮助下）把照片转换成线条图来实现描摹。考虑用这种方法实现整个动画的可能性。把图像描摹三次，然后把这三副图像作为动画的三祯来创建一个抖动的人物。你的动画由于有手工绘画的痕迹而有魅力。

现在进一步：从预先存在的图像系列中确定一系列祯（或者照片序列或者视频捕捉），为每一祯应用此项技术。当你描摹时，不必担心重建完全相同的图像 - 只要抓住图像的本质。此时风格很重要。夸大化和风格化来分隔最重要的元素。如果你用视频捕捉，试着重建运动的感觉。结果是一个运动的动画。

别忘了Flash很有趣。用它的不同的特征 - 在这儿调整一个值，在那儿光滑一个线条 - 然后外面很模糊，你不能确定是黄昏还是拂晓。

试验只有简单形状、线条和文本的动画。用批判的眼光看电视，你会发现很多商业片和情景喜剧使用这种技术达到一种相当好的效果。粗略看看正常光滑的矢量图形。把这些图形扭曲。这会阻止Flash制作冷酷的、有点数学完美的动画。
