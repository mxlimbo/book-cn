越少越好

　　今天我们需要一个能制作GIF89的程序。幸运的是，我们可以得到Mac和PC平台上的几个GIF89软件，它们中的一些是免费的，而且很好用。可以到Webmonkey's Toolbox查看这些应用程序的清单。凭我的经验，Yves Piguet的GifBuilder(for the Mac)把特点和易用性结合得最好。但是这个应用程序有一些怪癖，没有我们在下几页中提供的技巧，你可能会疯掉。

　　同时，你可能想更新你关于GIF animation和animated GIF versus server push的印象。

　　准备好了吗？我们现在开始演示。

　　为文件减肥

　　GIF压缩算法对单调色彩图形最有效。这里的单调不是指图像本身显得单调，而是指构成图像的色彩数和混合数保持最小。

　　较少的色彩和混合最合适用在说明和图形上。充分渲染的照片需要大量不同的颜色。黑白照片好一些，但是浏览器只能读color-safe palette中的四种灰度，因此其它程度的灰度依赖于抖动。你可以用DeBabelizer把所有图像映射到相同的调色板上，这可以减少文件大小和优化GIF89的表现。

　　最后，如果你必须用照片，可以考虑用单色调色板在维持图像质量的同时减少颜色数。虽然下面的全色版本的动画有相对有限的调色板，它看起来还是很漂亮。相同的效果可以用更经济的单色调色板实现。
