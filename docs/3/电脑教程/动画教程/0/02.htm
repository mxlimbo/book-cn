动画教程 - 第1天

　　动画的范围很广，包括从低成本的翻动的书页到超级复杂的、很长的迪斯尼电影。卡通片，泰坦尼克的下沉都用到动画特技。每一种类型的动画都有它的难处 - 预算和时间消耗、技术限制等等。但是它们中的任何一种接受的挑战都无法和网页动画相比。

　　如今，网页动画制作者不必再花费17个小时手工绘制单元格，但是带宽、平台和浏览器等给网页动画制作带入新的困境。

　　我们生活在困境中：把它做得又大又漂亮 - 只有一部分观众能看到；或者做得又小又有限 - 每个人都能看到。我们用过图形和动画软件的各种版本，得到各种经验和错误。我们尝试新的界面、查找各种bug直到找到我们自己的捷径。

　　你现在就不必象我们这样做了。

　　在动画教程中，我们的经验会使你的动画制作之路容易走些。在后面的六课中，我们为您提供关于GIF89、dHTML和Macromedia Flash的各种技巧和排错技术。下面是我们的课程安排：

　　课程内容：

　　第一课：教程概述和网页动画简介第二课：GIF89第三课：DHTML第四课：Flash第五课：动画方法 - 设计第六课：动画方法 - 声音第七课：动画方法 - 样式和集成

　　课程要求：

　　保证你有如下的准备：

　　第一、五-七课：不要求有预先的知识。

　　第二课：1. GIF动画的基本知识2. 读过Jim Frew的GIF Animatioin Refresher

　　第三课：1. dHTML和Dreamweaver的基本知识2. 读过Taylor的DHTML Tutorial

　　第四课：1. 对Flash的基本了解2. 读过Mike Kay的Intro to Flash

　　后备：

　　在开始教程之前，应该有所有需要的工具。为了能显示我们今天我们讲的，你可能需要下载最新的浏览器和插件程序。而且需要一个支持JavaScript的浏览器。否则，在我们用JavaScript调用一个新窗口时，你将看不到。

　　OK。准备好了吗？让我们开始吧！
