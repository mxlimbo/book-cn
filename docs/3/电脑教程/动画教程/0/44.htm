第三页：泥塑、木偶动画和一点3-D

泥塑和木偶动画涉及到3-D模型的一祯祯的运动。大多数模型动画来自于欧洲和日本，但是美国也有一些很有影响的动画工厂，如Tim Burton的Skellington Studios。

这种动画背后的过程随着项目的不同而复杂性不同，但基本的前提却相同。用相机捕捉场景中的一个精制祯，稍微移动人物和场景，用相机捕捉另一祯，等等。很明显，错误率非常高，一个简单的坏祯可以毁掉一个系列，所以这种技术并不易行。但是如果把相机和视频摄像机结合起来使用，就容易得多。同时，很多桌面相机，如QuickCam，可以用在这种动画中，尤其是与象Adobe的Premiere这样的视频编辑软件结合使用时。与一块视频捕捉卡一起工作，Premiere有一种捕捉功能。

令人悲伤的是，3-D计算机动画看起来替代了泥塑和木偶动画，而且显而易见。电影“玩具总动员”是完全3-D计算机动画，但看起来象木偶动画。与此相反，“圣诞节前的噩梦”，基本是木偶动画，很象计算机动画。易用是其中的差别。如果你的计算机动画不如你预想的，只需改变顺序，重新渲染动画中的错误部分。但是在泥塑和木偶动画中，错误以为着从头开始。然而，泥塑和木偶动画有一种很难用计算机重建的，自发的、波浪起伏的、笨拙的魅力，所以它们不会很快消失。

3-D动画

虽然不象早期例子中的模拟方式，3-D动画应值得一提，因为它当然是一种独立的风格。不用走得太远，我简单地描述为：三维动画用模型软件建立人物和场 的模型，然后用渲染程序进行渲染。
