渐进按钮

　　至于这个效果嘛......　其实我也不知道具体应该叫什么，用到按扭上就叫了个这名字，也可以用在其他地方，反正说也说不清楚，先看看范例你就知道了：

 

 

　　把光标在按扭周围移动移动，看到了吧，就这个效果。这是国外一个网站提供的Flash范例文件，其中有两个按扭，按扭1比较简单，按扭2相对复杂一些，而且按扭2中也包括了1中的所有技巧。我就以按扭2为例子讲解整个制作过程。

　　你可以看到，按扭2有很多圆形排列的圆点，这个东西如何画？跟我来，你会学到 Line 的灵活运用。其次，当光标在按扭周围移动的时候，带园点的环就开始闪烁，由大到小，你可以想到，能感应光标事件的就只有按扭了，这东西其实就是将环型按扭饶中心依次排列最后实现我们要做的效果。

　　ＯＫ！我们开始吧，运行你的 Flash ，建立新文件。第一步，建立环形圆点。选择Insert菜单下的 New Symbol 建立一个 Graphic .为了便于定位，选择 View 菜单下的 Grid命令打开网格线。然后点工具栏的圆形工具按扭。接着在 Line Style 下拉菜单中选择Custom... ( Line Style 在那里？我怎么没发现？好拉，给你张图，应该知道位置了吧！)


　　选择以后会弹出 Line Style 设置对话框,设置 Type 为 Dotted ; Dot Spacing 为2Pts 然后按确定。接着在刚才新建的 Graphic 工作区画一圆形，删除中间区域，注意中心位置对准工作区的中心坐标。在次选择 Insert 菜单下的New Symbol 建立一个 Button ，然后在其Hit位置插入关键侦，选择 Windows 菜单下的 Library 打开库窗口。将刚才做的圆点环形图形符号拖进去，与坐标中心对齐。这个按扭是用来做触发事件的，因为在按扭的 Up 位置没有放任何东西，在以后的拖放过程中不太方便，所以我们给 Up 里也放一个环形，过后再删除它。

　　现在，需要建立一个影片夹子，然后在影片夹子的第一侦插入按扭，然后在Timeline中选中第一侦点右键选择属性设置Actions为Stop,在第二侦插入空白关键侦关键侦，第三侦插入关键侦，并将刚才建立的园环图形符号拖进去，点右键盘属性，设置其颜色透明度为20%，接着在第四侦插入关键侦，设置图形符号透明度为100%，然后在第十侦插入关键侦，设置图形透明度为20%，并由第四侦至第十侦Motion变化，第十一侦插入空白关键侦。从而体现圆环的闪耀。然后回到第一侦，在工作区选择刚才放进去的按扭，点又键属性，选择Actions标签，点加号选择On MouseEvent ，接着在右边的控制项勾选 Roll Over ，在次点加号选择 Go to,然后在友边的控制项 Number 处填入2,并勾选Countrol的 Go to and Play,然后按确定。正确设置以后Actions中的代码应该是：

On (Roll Over)
　Go to and Play (2)
End On

　　按确定。现在回到主场景中，将刚才建立的影片夹子拖入场景中，注意中心与坐标中心对齐。对其以后在次拖入这个影片夹子，并设置大小比刚才的小一圈！如果感觉对大小控制有困难就打开Windows菜单下Inspectors子菜单的Object控制面版来由数字精确控制,排列好以后应该类似图中显示的样子：


　　马上就要完成了，还记得我前面说的，为了方便定位，给按扭的Up位置也放了一个环形吗？现在用不上他了，进入按扭编辑状态，删除Up区的图形符号。然后回到场景中，给环形的中心输几个文字，例如My Button ，然后 Ctrl+Enter测试一下，是不是很有意思？

　　这个效果其实就是将按扭的激发事件组合起来实现效果，不错的思路，你能想到吗？还有就是线的不同风格，Flash 给我们提供了非常丰富的线条效果，多试你种风格，不同参数，你能得到意想不到的效果。
