Summary

The SQL Server component and product family—including SQL Server RDBMS, visual systems management tools, distributed systems components, open client/server interfaces, and visual development tools—provides a complete and robust platform for developing and deploying large-scale applications.

This chapter is intended to only give you an overview of the various tools available. The remainder of this book concentrates on the capabilities and uses of the SQL Server engine, which is the foundation of the product.
