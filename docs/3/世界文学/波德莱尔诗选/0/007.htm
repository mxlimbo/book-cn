中魔

太阳蒙上一层黑纱。像它一样，
哦，我生命的月亮！你潜藏在黑暗；
任性的沉睡，抽烟，伴着无声的愁肠，
你的一切都陷入无聊的深渊；

我这样爱你！可是今天如果你
　像从半阴影中走出的昏星，
大摇大摆地奔向狂欢的地方，
那好！迷人的利剑！就让你出鞘！

吊灯的烛火照耀着你的双眸！
粗野人的眼中燃烧着情欲之火！
我喜欢你所有的一切，病态、狂舞；
你心驰神往，黑夜，红色的黎明；
我颤抖的全身，每一根神经都在喊：
哦，我爱你，亲爱的贝力西卜！
