赠给你这些诗

赠给你这些诗，为使我的名字，
随着幸运的帆船遥遥驶向远方，
让世人的脑海掀起梦幻的波浪，
犹如凛洌的朔风吹打那巨船，

像无稽的传言，对你的怀念，
虽象扬琴一样使读者厌倦，
但通过亲密友好的神秘链环，
和我这高傲的韵律紧密相连。

从无底的深渊到茫茫苍天，
除了我，没人理你，罪恶的女人，
――你，犹如一个幽灵昙花一现。

你用轻松的脚步和安详的目光，
践踏那些挖苦嘲弄你的蠢物，
青铜面孔的天使，黑亮眼睛的雕像。
