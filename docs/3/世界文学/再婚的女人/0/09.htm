　　"别开玩笑了。恐怕福斯特博士都没有梦想。像牛和印度牛，马和驴呀什么的，还受孕过。嘿，也就是拿海里的一点点下等动物做个实验罢了。"

　　"这样子我就放心了。"夫人对自己说的话感到吃惊，她站起来，走到床边，讨好献媚地直勾勾盯着丈夫的眼皮，说："今天研究的是什么呀？是做标本片吗？身上有一股味道。"

　　于是，夫人感觉到从她冰冷的心底涌出一股喜悦。

　　（当男人想象妓女的时候，妻子立即会有感觉，变得冰冷。但是，想象小玻璃片的男人，是自杀。面无人色倒在研究室里的丈夫的尸体。研究的牺牲。散乱一地的小玻璃片。）

　　"人？过去果然是死囚吗？"
