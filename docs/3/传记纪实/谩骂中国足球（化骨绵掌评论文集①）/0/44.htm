每周述评——让眼泪在前面等我


　　冬天悄悄来了，我们都被浓浓的寒意笼罩着，表情凝重得吓人。

　　中国足坛不会因为寒冷而停止搞笑，在经过5个月的调查、研讨、咨询，外籍门将终于把中国足球得罪了。历史将会证明这是一个超级笑话，将国产门将的水平低下赖在表现优秀的外籍门将身上，这真是大便干燥赖厕所里有鬼。我们只看到韩文海符宾江津区楚良长期占领队中主力，我们也看到高健彬虞伟亮表现相当优异，我们还看到走后门沾了国家队一点"腥味"的李伟被迟尚斌牢牢放在厦门队门前变成了甲A第一大漏勺，我们还看到被法布雷斯挤走的刘建生在上海违纪。我们不会忘了嚼着口香糖非常自信的区楚良在广东挡着高健彬、在申花拦着虞伟亮、现在不知道在云南还堵着谁。

　　外籍门将怎么了？

　　所以我们为萨沙为老布流泪……

　　我以为鳄鱼眼泪很快就干了，但一想到远在意甲还未尝着上场滋味的马明宇，心中的冬天就更加寒冷了。不知马儿是否还记得双流机场的泪水，也不知他独身闯意甲是否只是赚了个在看台上免费学习的权利。

　　有一个赛事我们不流泪，泪水在前面等着李章洙和陪伴他三年的重庆隆鑫。

　　前不久结束的亚洲杯已经告诉我们，技战术含量决定一切。比如日本，比如沙特……

　　所以战术技术都几乎是一张白纸的北京国安在纯精神的冲动中输得找不到北。面对技术超群的三个火枪手，面对姜峰姜晖赵立春孙庆罗迪，北京国安还剩下什么？我们只看到邵佳一的迷惘，徐云龙的无奈，小王涛的孤立……

　　面对重庆义无反顾地支持李章洙（即使带着当年的大牌们差点降级），北京国安难道没悟出点教训？曾几何时，辽小虎险些扮演凯泽斯劳藤的时候，有人"危言耸听"地指责张引正在毁掉张玉宁李金宇，因为光靠年轻的冲动而内容苍白无力是扼杀他们的最毒的药剂。

　　现在这句话不知能不能送给意气风发的魏克兴？不知邵佳一徐云龙薛申们会不会在低劣的训练中销声匿迹。因为刚烈的9号田野在不假思索地飞铲18号姜晖的时候，在他众目睽睽之下骂骂咧咧的时候，他已经把这只骄横的御林军的优秀品质告诉全亚洲了。

　　当年他们中的某些老队员就是这么骂走乔里奇的，他们不知道骂走了"先进"，迎来了愚昧。可悲的是竟然有高人为他们壮胆打气，还要美其名曰：一切为了安定团结。

　　为北京国安这样一支觉悟高、顾全大局的球队惋惜！！！

　　泪水在李章洙的脸上流淌着，这个以硬汉著称的韩国人太煽情了。以至于央视5频道及时制止了他的感情宣泄，他们大气地尽快结束了这次不快的转播，刚才还夸夸其谈的刘建宏吴金祥连尊重观众的"再见"都舍不得说一句。只是不知道要是北京国安夺冠后他们的笑靥是不是灿烂无比？

　　但是李章洙的泪水还在重庆流着，他被队员抛向空中，面对KBS的记者掩面而泣……发奖仪式继续着，飞利浦在尽最后的义务。

　　在这个喜剧大于悲剧的中国足坛，泪水说明一切，也最珍贵……
