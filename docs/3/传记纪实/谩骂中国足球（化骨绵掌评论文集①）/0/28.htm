央视，亮出你的舌苔

　　讨厌的信号中断，讨厌的体育欣赏，讨厌的主持人喋喋不休，使劲搓洗我们疲惫的脑袋。这个谋杀球迷无数细胞的可恶的深夜，有谁对这个飘荡在我们头上的卫星负责任？好像那颗卫星如同断了线的风 筝，放飞它的小孩不屑地擦着鼻涕回家过家家去了。

　　听说不是央视的责任，全亚洲的观众都伸脖子干瞪眼。但，您能不能在黄金时间出来解释解释，或者倒个什么歉，例如在《欧洲2000》，例如那个沧桑的马主任，郑重其事地向花了钱的观众说声：抱歉! 没有，一切迹象表明，你看不到转播，活该！

　　如果我们还是上帝，我们绝对有权让你亮出你的舌苔。仔细一看，整个一个贵族器官：苔面平滑，苔根红润。这样的舌苔怎舍得对天下劳苦大众分泌骄傲的唾液？即使发出高傲的吼声，也是："谁稀罕你这点小钱？"

　　央视的名字早已变成"中央爷们电视台"。爷们给你转播欧锦赛就够对得起你了，你花几块钱就能跟欧洲球迷的脉搏一起跳动，天下哪有这样的好事？至于信号中断，也不是什么阶级敌人搞破坏，没准是 荷兰比利时的爷们临时把亚洲信号掐断。让我道歉？倒什么歉？我比你还急，脑门上直冒汗。

　　我们找谁诉冤？能不能发一本《消费者上当受骗指南》？

　　上亿的观众被迫收看一家电视台，这本身就是新"天方夜谭"。我不知道中国加入WTO后，能不能让外国电视台跟央视叫板？在你垄断了收视权的同时，我们能不能按动遥控器，看看信号没有中断的ESPN 台？尽管这句话说出来更像新新天方夜谭。

　　于是我们咽下愤怒的口水，被迫听于大川神侃什么"德罗之战激烈精彩，双方攻防速度转换很快"。老人的眼看什么都很快，就像德罗两队公然在欧锦赛上演2000版的"侏罗纪公园恐龙大决战"，马特乌斯和哈吉叫着劲争当足坛文物标本。就像央视老是那么几个人长期霸占评球的舞台：用意大利语报比分的张慧德（大概这样才够味）、除了德国什么都不知道的于大川、中国的"霍顿"张路……等等等等。难道找不出一个思维敏捷知识渊博的年轻人？非要找几个博物馆的人唾沫星子四溅？

　　这个世界应该是年轻人的世界，上了年纪就回家溜弯儿去。50年不变？在这一点上，央视还停留在上个世界N个年代。
