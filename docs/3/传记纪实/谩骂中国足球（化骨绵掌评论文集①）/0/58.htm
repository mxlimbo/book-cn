为《直通悉尼》喝彩

　　1996年亚特兰大奥运会的时候，电视报纸连篇累牍、气愤填膺地告诉我们：美国人只播放他们国家的运动员夺金的镜头，升星条旗、奏"永不落"歌，一天恨不得80遍。而其它国家的运动员只是个陪衬，即使夺金也是轻描淡写。于是我们愤怒，瞧美国人不顺眼。后来发现我们的央视体育频道也这么导演，甚至更加超越。其实这怎么了？哪个国家的电视台不这么偏心眼？谁不巴望自己的选手为国增光啊？于是我们平衡了，尤其这次在悉尼，全场起立同唱《义勇军进行曲》的时候，那叫痛快。

　　可转念一想，这悉尼奥运会又不是我们一个国家在唱独角戏，其它国家的运动员还不是没闲着。比如荷兰，比如罗马尼亚，比如日本，他们的金银铜是怎么获得的？升国旗奏国歌就免了，全国人民都没兴趣看，但情节也得仔细交待一下吧？因为我们除了想看见"中华民族的崛起"，还想知道其它民族是不是蹲着。

　　于是我们寄望奥运频道的《直通悉尼》给我们奉献奉献。这节目特准时，连正在激战的中美女垒大战都可以中断。看来把节目向后顺延比较难，没准要各级领导盖32个章呢。可看来看去，基本上是我国选手夺金摘银的精彩画面。不是说要产生18块金牌吗？除了我国选手获得3块以外，其它那15块呢？还好，飞人大战没有忘，其余的比赛给人的感觉是还在继续鏖战。可抬头一看表，悉尼已是夜里12点，再有夜场也该散了。就连大家非常关心的足球比赛，他也就用了10秒飞快地播报：巴西输给南非，日本被美国淘汰，连中田英寿罚失点球也懒得说。也对，别让中国足球粘了巴西日本意大利的霉气，趁你这儿还没回过神来，他那儿"拜拜"了。连进球画面都没有，更别提慢镜头了。实在不是嫉妒人家，完全是派往前方的记者为咱电视台节约录像带，或是张着大嘴忘了拍。

　　看来这节目应该改名，叫什么《直通悉尼》呀？干脆叫《直通我们》。我们是重点，其它的有时间就捎带，没时间就装着心里什么都不明白。导向非常正确，咱们都拚搏了，人家是撞大运。就像前方解说员解说射击比赛最后一枪时，全体人民的心都跳到嗓子眼儿了，大气都不敢喘。他在那儿冷不丁说人家外国运动员一句："枪口有点颤抖。"这就对了，其实他应该说："枪口抬高一寸。"我不知道这种心态对不对，精彩留给咱们，恨不得人家都是摩尔多万流着眼泪差点把眼珠子揉出来。我想肯定是对的，要不然奥运频道绝对不会这么吝啬精彩。

　　如此看来，《直通悉尼》真直通了我们，为咱扬眉吐气，直至把咱们的爱国热泪催出来。只是以后千万别再随便指责人家的电视台了，谁不爱自己的国家啊？咱不是也这样吗？而且比他们做得更棒呢，别说升什么国旗奏什么国歌了，连比赛结果都可以省略。比如俄罗斯男排输给意大利，比如印度男曲玩转西班牙，比如德国"鲁冰逊"驾驶独木舟第一个到达终点，比如……凡是没中国队比赛，真懒得动嘴。

　　罢了罢了，后面还有节目，《悉尼猜想》要与您准时见面。上次我说刘建宏扛着曲棍球棒出场，并且预测第二天扛杠铃。咳，真让我说准了。不过还算好，我国游泳健儿没夺冠，不然他非穿鲨鱼装表演。没关系，跳水金牌看来不止一块两块，有他展现才华的空间。

　　喂喂！不会现场挖个水池子吧？嗯，这节目策划没准真想得出来。
