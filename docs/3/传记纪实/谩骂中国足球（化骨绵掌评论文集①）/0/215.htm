杨铭：《本色》一文之回应

　　果然不出所料，一篇小破稿就能让网上著名黑势力团伙疯狂的犬类动物露出阴深的牙，而且跟着狂吠的竟然有一群之多，数量可观。稍微没有想到的是，本以为疯狗们对付我这个“‘不知怎么就混入记者队伍’的‘白痴’”（东莱居士语）会采用更艺术一点的方式，好教育教育我这个“不是白痴是流氓”（沂水秀才语，承蒙抬爱）的小民，谁知道竟然招招是“化骨绵掌”“迎风一刀斩”之流的狠毒招式，而且居然全都以下撩式的色情武功，大有一把干脆把我搞残之势。不知道犬科动物们究竟是中了昆仑派的“七伤拳”还是“葵花宝典”终于练成顺带相聚一起狂干了两天两夜的胆固醇，飞出来的狗牙阴毒可观，恶臭连连。

　　稍感安慰的是这一年多来相信大多数同志们跟我一样深受狂犬们的熏陶，不卫生的招式实在见得太多，倒也没怎么觉得压抑。不过料想患了严重精神压抑和忧郁症的狗崽子们是不会“就当他是个屁”把我给放了，同时网络也不会肃清这群象我一样的流毒了，既然如此板砖就拍够来吧。

　　“有时候过节就是不好，过节时吃的东西太油腻了就会把肚子撑坏，如果是一个不留神的话极有可能把人脑也搞也了猪脑，这不，今天辛辛苦苦的爬上网来就看见了一篇极有可能是被五一节的油水撑坏脑袋的记者撰写的白痴文章。”（中华俊杰语）

　　过节有时的确相当不好，我以为这么满嘴猪油狗屎的话只有我这个“被油水撑坏脑袋”混入记者圈中的文盲才说得出来，没想到狗屁俊杰也可以傍着棵大树尿出来，鄙人对你的佩服只能如“滔滔江水连绵不绝”，再无它耳。

　　“名“妓”这一点与傍不上大款的妓女倒有相似之处，管他绵不绵，只要给现钱，胡乱凑合一个喂饱了底下那张嘴再说。即使派出所的找上来，她自己负责，关强奸了她的“白痴们”鸟相干？他们二者的相似第二处就是广告词惊人的一致：欢迎来稿（搞）！”（化骨绵掌语）

　　象化骨绵掌之流“熟读唐诗三百首，不会作诗也会淫”的主看来早已融会贯通顶级意淫之道，处处洋溢“黑虎掏心谢主龙抓手”之道，连“化骨绵掌”此类海大富的绝学熟贯于心，当初挥刀自裁的毅力如今在网路上更上一层，可喜可贺。不过如此肾疼的语言倒也不由使人怀疑，“化骨绵掌”兄平时究竟鬼祟到什么地步实在难以想象。

　　“嘿嘿，跟你们交个知心话，说你们"白痴"，难道你们就真的"白痴"吗？糊涂啊。谁是"白痴"，你们当然不是，俺们当然更不是，谁是？球迷才是！他们才是吃那啥吃多了，整天没事儿跑到足球场上发泄去。他们就爱看这东西，他们就爱听这消息。这市场，这机遇，嘿，俺要是不抓住，俺岂不成了真的"白痴"？”（沂水秀才语）

　　这回连球迷也拉上来了，说句实话，虽然常在网上拜读秀才居士们的大作，只是直到现在竟然仍然不知这些阿猫阿狗们是那根葱，为何长期以来一直在网上以球迷领袖自居，而振臂一呼总有心虚犬类“汪汪”响应。中国足球确实什么都不行，只有球迷聊以自豪，但绝不是你们这种在网路上操持着语言权利的伪“球迷领袖”，狂吠一年多来从来也未见中国足球有什么实质性的改变，“妓”者们纵然再不济，为足球作出的努力有目共睹。何况“流氓”只是如各位认同的鄙人之流，何必无故牵连其他。板砖尽管招呼过来，鄙人恭候，只望各位不必牵涉无辜。

　　最后想说的是，《本色》一文言辞的确过分，内意非在骂人。在这里节选cme99先生的一番话也请各位与我这一“白痴”自省。

　　“说到人与人的相处，最重要的不过是俯仰天地而无愧于心。无论你讲究自我的修行，即便独处一室也正襟危坐，循循儒雅，君子风范，也或者你豪气逼人，放浪形骸，毫无羁袢，口不择言，本都是处世的方法，不值得过多指责或者褒扬。然而如果所言都事关是非的原则问题，还请大家注意选择表达的方式。春药云云，本不象知书达礼的读书人所为，而强奸的形象化描述，似乎也完全不值得提倡。文字自古是大事，足以表现人的修养与志趣，对文字的滥用，实在是古人所不满的啊。

　　记者的话，作为对“妓者”一称的回应，实在不当，公道自在人心，远非大呼小叫，胡乱运用劣质词语所能争取的，然而“妓者”一词，似乎入不得文字，我们都还是无名小子，俗言俗语总是要避讳一下的，更重要的是，这对于从事那个行当的读书人确乎是个侮辱。文字运用之妙，存乎一心，表达愤激，自然有更缓和更委婉的方式，大刀阔斧，快意文字自然畅快，但对于网络形象的影响恐怕是难以挽回的，网上论争，实在是不能不谨慎啊。”另一血腥提议，鉴于网络形象，各位大可约定某一地点双方板砖招呼，网上意淫实无聊之极。
