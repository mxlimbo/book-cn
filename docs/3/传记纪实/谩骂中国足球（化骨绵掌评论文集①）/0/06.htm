左手戴手套


　　著名教练蔺新江刚刚跨进厕所，猛见一著名边裁正在"舒服"，他顿时脸色惨白，连忙退出来，边退边说："您慢慢屙！"不巧这一幕被一个著名记者抢拍了下来，喝问蔺新江："你刚才嘴里在跟边裁咕噜什么？是不是你们的接头暗号？"

　　蔺新江的左手正好戴着一只手套，右手腾出来准备解手。他猛然想起《红灯记》里李玉和在车站与王连举接头的暗号就是"左手戴手套"，他恼羞成怒地问："你什么意思？"

　　记者答道："没意思，我只是出于职业的敏感才抢拍下来的。"

　　这个故事发生在中国足球职业化以后的第8个年头。

　　绝不是危言耸听，前天的厦门刚刚上演一出类似的闹剧。神经末梢超级敏感的某足球记者拍到了蔺新江给一个围着体育场跑圈的裁判仅仅一瓶矿泉水，这就足够了。如果这场比赛上海申花输了，如果喝了矿泉水的裁判太"黑"，如果场面不正常，这张照片就是一个耐人寻味的证据。怎么会没什么意思呢？没意思你拍什么？这里面太有文章可做了。

　　我们的记者和教练是不是都太敏感了？敏感得像偷了邻家王大娘一袋大米。谁闹的？就是中国足协对假球黑哨打击不力，一个个冒充法官"证据证据"地狂吠闹的。闹得球迷拼命睁大眼睛，闹得记者成了包打听，闹得教练球员人人自危，这是王俊生所说的"取得了巨大成就的职业联赛"吗？畸形变态的足球环境仿佛让人们回忆起"世态炎凉人心不古"的文化大革命，面对面地斗争，背靠背地检举。

　　肇事双方都有错。首先是蔺新江，以为还是"两把菜刀闹革命"的年代，都2000年了，暴力永远不能解决问题。都多大年纪了？能不能熄熄江湖老大的脾气？上海滩的发哥吹着枪口的青烟，巨酷！可惜那故事发生在旧社会，不然您肯定第一个跑步报名。

　　而那个神经末梢过度敏感的老记，拍这张照片的时候最好先动动脑子，一瓶矿泉水能说明什么问题？

　　真要做裁判的工作，还能让你看见？上海申花真输了，你也只能拿那张照片起起哄，搅乱搅乱球迷的视听而已。没逮到通奸的，但看见人家正在提裤子，这种自欺欺人的小聪明别当足记，当娱记，那儿更热闹。
