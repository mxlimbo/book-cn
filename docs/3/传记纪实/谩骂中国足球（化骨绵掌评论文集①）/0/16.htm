谩骂中国足球之张斌篇

　　张斌不是足球节目解说员，他是斗士，一个疾恶如仇、勇敢机智的反黑侦探。他笑容诡秘地挽了一个圈，轻易地将陈亦明们套了进去，然后步步紧逼，雷霆扫穴。中国足球的黑菜心没挖出来，烂菜帮子可全都抖落在广大电视观众面前了。

　　那时的《足球之夜》还真是球迷每周的节日，我们屏住呼吸，睁大双眼，聆听张斌将足坛中的“土匪罪状”娓娓道来。当时我们谁没为他捏着一把冷汗：张斌啊张斌，别管不住你那张尖酸刻薄的嘴，千万别惹出什么漏子来。

　　果然，我们没有料到腐败竟有如此顽强的生命力，如此莫名其妙的后盾。张斌下课了，只能躲在银屏后面当配音演员，只能当一个听起来还是个官的制片人，只能在美国女足世界杯的时候偶尔露一露那张可爱的胖脸。

　　这实际上已经超越了足球本身，已经触及了某些心里有鬼的人那根敏感的神经。可别小看了王俊生们，能在足协混了这么多年，能是光吃干饭的吗？起码也吃过几碗搅得稀里糊涂的打卤面吧？

　　党中央一再重惩腐败，官当的再大，还不是拉出去崩了（当然更多的腐败活得还挺自在）。唯独足坛这些丑事捂着盖着，这中间到底谁在从中作祟？

　　摆在我们面前的事实不得不让我们垂下曾经愤怒的头颅，一个揭露腐败黑哨假球的战士就这样不明不白地锁在电视台的深闺。害怕什么？害怕他无情地扯开遮在腐败私处的遮羞布，还是害怕他深挖下去丢了中国足球本来就不存在的那张丑脸？

　　央视，能不能给我们球迷一个合理的解释？

　　千万别用“工作需要，没什么大惊小怪”来搪塞我们，因为什么事放在你们眼里都不会大惊小怪。假球黑哨不奇怪，德甲法兰克福去年怎么保的级？冲不出亚洲不奇怪，全世界100多个国家，有几个进过世界杯？你们真是宰相肚里能撑船啊，唯独我们球迷是鸡肠子心眼。

　　你们可以睁一只眼闭一只眼，唯独我们球迷没练成那功夫，非得瞪着流血的双眼认死理。

　　不明不白的“渝沈之战”，含冤挂靴的3号隋波……这些压在球迷心中的石头将越来越沉重，直到变成尘封的回忆继续羞辱年轻一代。
