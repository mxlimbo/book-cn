谩骂中国足球之京骂篇

　　谁敢说他（她）没骂过三字经？没人！大声地悄声地偷偷地念三字经恐怕都尝试过，而且充分享受过它给你带来的痛快。谁也没说你什么，尤其你是个当官的或者高级知识分子，人家还赞美你够性格。

　　我有一个温文尔雅的知识分子朋友，活了大半辈子，从不出口伤人，并一直认为三字经是人类的耻辱。然而有一天，同办公室的一个泼妇将他惹怒了。她指着他的鼻尖大声骂了一句：“肏你妈！”我的朋友浑身筛糠似的，眼镜在鼻梁上颤抖得咔咔直响，便秘似的涨红着脸。此时任何文明的语言都不足以给那个著名的泼妇迎头痛击，他挖空心思发明了一句：“我不肏你妈，我直接肏你！”全办公室的人都被他的惊世骇语震得目瞪口呆，那个泼妇更是“嗷”地一声捂着脸蹲在地下去了。我的朋友用他曾经鄙视的语言赢得了胜利，从此那个泼妇经常无限深情地偷偷盯着他远去的背影……

　　我的意思是，污言秽语的运用存在于我们每个人中间，况且足球场上“SB”“NB”的京骂还远远没有达到非常恶劣的地步，只不过几万人齐声呼喊而变得壮观。于是有些假道学便看不惯了，什么“这是北京人的耻辱”，什么“这是践踏首都形象”，难道他（她）的嘴就有那么干净吗？

　　我不相信足球场上的北京人都是粗俗之辈，既然他们敢直面“SB”“NB”，旁若无人地发泄痛快，他们绝对不觉得这是令他们耻辱的事情。既然不觉得，就让他们以他们的方式在草地上撒点野吧！

　　有人搬出鲁爷的文章《论“国骂”》来讥讽这种现象，鲁爷在路边听到一个赶牛的老农要肏牛的姐姐，于是愤怒，于是搬出法国人编纂的《骂人字典》猛翻。其中最下流恶劣的一句是：你这个畜生！于是得出外国人文明中国人下流龌龊的结论，呼吁国人摈弃三字经。也许那个时代的外国人确实文明，或者编纂字典的外国人确实不会骂人。但现在的美国人包括上流社会的贵妇人，连开个门都要说成：Openfuckingdoor!不知道鲁爷活到今天该做何评价？

　　我绝对不是提倡“SB”“NB”的京骂，而是觉得实在没有什么大惊小怪。中国人本来就没有发泄的地方，单位上发泄，第一个下岗的就是你丫的；大街上发泄，你没准是法轮功漏网分子；回家发泄，你面对的是一张怨妇的脸，她还想骂你SB呢。

　　怎么办？现在好不容易找到了地方，就让可怜的人们痛快痛快吧！
