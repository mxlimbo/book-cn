被海盗敲掉错位的牙


　　这哪里是激情四射的西班牙，活脱脱一个讲究纯艺术的哥伦比亚。在像慢镜头似的倒脚中，斗牛士的野性也被谋杀得一干二净。这场比昨天的“恐龙大战”还要乏味的比赛，你很难想象主角竟是劳尔、瓜迪奥拉。

　　也许西班牙太想克隆昨天的葡萄牙了。虽然都是“牙”，但牙口却不相同：一个是锋利无比的獠牙，咬得英格兰遍体鳞伤；另一个却是错了位的病牙，本想抱住海盗的大腿，慢条斯理地嚼烂它，没想到反被海盗挥舞着铁棍在66分钟时敲掉了。

　　98世界杯上，可恶的裁判将他们彻底葬送了，直到今日，希耶罗的泪水仍在闪烁。谁知今天埃及裁判又“从中作梗”，执法水平跟上个星期的俞元聪差不多。但平心而论，裁判不是没给你们机会，挪威守门员持球超过6秒，裁判立即判给你们一个禁区内点射。可惜面对白色森林，哪里还有大门的缝隙？可怜的劳尔只能躲在别人的胳肢窝底下露出无奈的眼色，看能不能在巨人丛中捡个漏什么的。

　　卡马乔的怒吼是苍白的，裁判没有什么可以指责。

　　也许斗牛士们太疲倦了，疲倦得懒散迟钝、表情木讷。一个丝毫没有斗志的、陌生的西班牙人颓丧地倒在挪威的脚下俯首称臣，他们那颗被海盗敲掉的“牙”被无情地埋葬在鹿特丹德科伊普球场的草皮底下。

　　他们曾经莫名其妙地败在塞浦路斯的脚下，这场比赛为那场败仗作了强有力的注脚，素以攻极力强大著称的西班牙，在蛮不讲理的海盗面前竟然毫无办法。

　　败仗来的正是时候。看到索尔斯科亚、弗洛那挂在脸上的灿烂笑容，满嘴是血的斗牛士们，找南斯拉夫、斯洛文尼亚出口怨气吧！
