一个村支书大胆质疑网辩

摘自爱球村黑板报：

　　先声明哈，俺是爱球村的村支书，比地球村的支书小一点，俺文化不多，吃得热和，属于“废话超过文化”的典型，对什么新鲜事物都要发表一点杂言。这几天俺到“村吧”游泳（冲浪有点危险），看到网上足球辩论好凶哦，差点吐口水打起来。俺仔细研究了一下，发现是一群秀才模仿人大常委会讨论法律法规的“快乐总动员”，除了那个叫什么曦的把问题说的比较透彻以外，其他的大部分人不分白天黑夜都在苦苦纠缠字眼，而且越纠缠越深远，一望无际看不到边。你们回头看看辩论的题目：“博彩合法化对中国足球的发展利大还是弊大？”现在闹来闹去的，闹成了为博彩是否等于赌博而辩。是不是有点跑题哦？俺们村的“田边答辩”可不允许这样胡搅蛮缠，那是要狠狠扣分的。就像俺隔壁的李玉和他妈，年轻的时候背诵毛主席语录：“世界是你家的，也是俺家的，闹来闹去还是你家的。你家是早上八九点钟的日头，希望这玩艺儿就搁在你家身上咧。”

　　比如东方先生说，博彩合法化可以大大缓解中国足球资金短缺问题，对方有些人就偏说中国足球仅仅是资金短缺的问题吗？那黑哨假球呢？博彩能解决这些问题吗？问得好！中国足球什么都缺，就是不缺黑哨花哨。但是，连俺这个老B头子都看出来了，人家东方说的是资金，他非要扯到一边去，这又不是辩论中国足球有多么黑暗。更可气的是，他还把俺们农民最喜爱的著名影星兼气象学家章健带领歌舞团慰劳前线的事拿出来证明中国足球不缺钱，哈哈，那点钱我们村都请得起，难道就证明我们村就不缺钱了？

　　又比如东方先生说，博彩可以疏导一些地下赌球的现象，可有些人就歪着说赌博害人，并举了许多家破人亡的例子，他马上把“博彩”与“赌博”联系在一起了，有点像偷换概念哈？国家为什么制定赔率的比率？干脆跟赌博一个赔率算了，押好多赔好多，甚至更多，那样不是要方便些？就是因为害怕有些鸡肠子小人输了2元钱哭着喊着要跳楼，所以干脆把永远的百万富翁梦淹没在几百万分之一的希望里，但是希望经常熊熊燃起，让你一颗向往致富的心永远不会泯灭。

　　俺最看不惯的是有些人把文革时期的口号搬来当作时髦，什么“火煎”“油炸”“炮轰”等等，好像姚文元的徒弟，俺好害怕。俺那个放羊的傻儿子回来了，他对俺说，那个叫“沙龙文化”，很时髦嘞！

　　又比如有一篇用对方的网名做标题的文章，只是为了说明一个问题。哪想到有些人以为对方不懂这个网名的含义，非要写出一排什么字母让别人去看看取那个网名的历史踪迹，难道他真是装疯卖傻超过文化？

　　还有一个人最讨厌，叫什么骨的，俺老婆一边炒菜一边对俺说，那叫“另类”隐晦。什么“投奔宋江”，分明在说有些人用公款“送浆”。什么“爱好者”为开嫖喝彩，分明是有些人在娱乐场所酒醉后失态说的，俺们乡里的大官经常这样自诩。这小子的文章的内涵俺是瞧出来了，他跟俺一个意思：先搞懂什么叫“法律解释”以及“政府行为”，再回到辩论中来。他的意思是……俺儿子告诉我，是《第二十二条军规》。俺没搞懂什么军规，反正俺知道“法律”是根松紧带，根据形势的需要来决定它的长短。如果连这个都不懂的话，那就再生长发育几年再说。有些人竟然还质疑，我国法律明明是禁止赌博的，难道政府……？？？

　　俺就给你说一个简单的例子，听说过“就业”吗？不是报纸上说的那个就业，而是监狱里的就业。就是说你刑期满了，因为“没改造好”强迫你在当地劳改场所就业。听来象是天方夜谭？你可以到这些场所去访问访问，你可以有工资，可以有“自由”，但是你不能随便回家，回家必须请假，否则像逃犯一样抓你。听说过没？这又是那条法律规定的？而且连“法律解释”都没有，如果有人质疑，他会说我们为劳改释放犯找了个工作。只不过这条法规已经多年没执行了，那以前的几十年可都是这样合理地存在着啊！去年俺给隔壁的刘老二送东西的时候，还看见“就业中队”在举行春节联欢，他们互相通婚，子女在当地根本抬不起头，只能又走上父亲的道路，真像《流浪者》中说的，小偷的儿子永远是小偷。所以说，什么叫“法律”？不说也罢，越说越离谱，越说越离题。

　　所以俺说，辩论来辩论去，原来是“葫芦僧乱判葫芦案”，俺下地干活去了，连拜都不拜。
