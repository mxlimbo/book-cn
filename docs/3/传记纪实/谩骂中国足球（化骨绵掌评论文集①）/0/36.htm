不转播，就是不转播！

　　《足球之夜》这节目够拽的吧？分析国足、分析米卢，两主持西装革履，谈笑风生，什么战术什么阵形都捏在他手心儿里，你丫不跟他一块儿过这节日你都亏得慌。可轮到中国队开练了，他俩却没了踪影，找得我等鼠辈眼睛酸痛。这就像街边的一混混儿，眉飞色舞地跟你描述一武林小子正日夜操练少林蛤蟆功，准备找泰森之类的过过。架倒打起来了，你这儿心急火燎问结果，他两眼一翻：我哪儿有资格啊？让人挡在门口了！

　　我想，四国邀请赛激战正酣的时候，两主持和那主任不是在厕所假装便秘，就是正在后院互相切磋，然后仰望月空，叹了口气，说：“不转播，就是不转播！”

　　我记得前几年主任说，咱这台正跻身世界先进行列，外国有的咱都有，球迷们您就放心吧！我特放心，外国有的还真有了。意甲德甲欧洲杯确实大饱眼福，可我们最关心的中国队的比赛场面却音信全无。前几天主任又说，世界体育转播的老大还是美国，可咱和欧洲老二们比，一点不逊色，在节目制作方式上与我们差距不大。（注意：不是我们与他们差距不大。）

　　确实是，我们球迷都看到了央视五套的辉煌战果，咱们的足协杯都用上虚拟技术了，这可是世界上最先进的转播绝活。可我脑子还是有一点没开窍，这么先进的转播技术，人家欧洲杯楞不用，由此可见，荷兰和比利时的电视台真落后，他们的设备跟我们比还真不灵。我骄傲自豪美不滋儿的一古脑儿窜出来特多豪情。

　　但不知骄傲的主任这次到悉尼能不能问一下欧洲的同行，他们的国家电视台体育频道是不是也不能转播自己国家队的比赛内容？然后通知我们球迷一声，我们也好死了这条心，从此不再仰着脖子苦盼央视五套出现国家队比赛的踪影。

　　我愤恨不平，瞧那些有线台不顺眼，跟五套争什么转播权？大家都是吃这碗饭的，给人家点面子行不行？那么大的电视台，前面还有一“中央”，受你们这些破台的阴气，他还能在《足球之夜》面不改色心不跳大谈中国足球，谁看了谁佩服。一般人可没这脸，肯定特尴尬特没底气，因为天花乱坠地谈了半天，楞没资格转播。中国队上半场都2：0领先了，它还在那儿装糊涂，楞播叶江川和古烈维奇的棋谱。我特想给它挂一块红布，蒙住它双眼也蒙住了天，它问我看见了什么，我说我看见了幸福。
