每周述评——让我欢喜让我忧

——中国篇——

　　真正的球迷敢于熬夜喝啤酒酷爱欧锦赛；真正的球迷敢于直面惨淡的足协杯。这一周，我们的神经要拥有足够的韧性，否则你会被两种不同的运动弄得找不到北。在欣赏完精彩的欧锦赛后，你还会有勇气观看我们的足协杯吗？看来胆大的球迷还真不少。

　　四川全兴队想"篡夺"足协杯，想尝尝这个鸡肋到底是什么滋味。于是他们骄傲地跨进中南财大体育场，趾高气扬地乜斜着灰不溜秋的蔡晟，心想：今天让你真正懂得什么叫力不从心。看看出场的阵容，豪华得耀你的眼睛，就算在甲A不怎么出场的刘成都入选过米卢的集训队。然而宝刀未老的蔡晟一个金头，一个柳成荫的远射，使甲A第一射手黎兵顿足捶胸。全场比赛有他无他都无所谓，即使马明宇姚夏刘成马荃……们再努力，他能把一个传球停在他面前不远的地方——最多距离他20米。这个距离足以让四川全兴在一支甲B面前停止向那座"梦中之杯"迈进（幸亏不是"梦的故事"）。

　　同样窥视奖杯的还有山东鲁能泰山队。在巍峨的泰山脚下，山东人冒着泰安35度高温，干净利落地3：0击败同样"不畏强暴"的深圳平安。赛前，太认为自己是明星的李玮峰大言不惭地说："我们没有放弃，连我都来了，就很说明问题。"平安队替补球员证实了他的话，全场紧逼，与山东队展开激烈地对攻，尤其是大明星们在替补席上悠哉游哉的时候。其惨烈程度令我们对深圳的替补肃然起敬，有这样的精气神，回到甲A，深圳有戏。

　　人人都知道，足协杯是爆冷的温床，但谁也没想到殷铁生带领一帮青年近卫军将不可一世的上海申花打翻在地。脾气暴躁性格外露的彼德洛维奇终于尝到了熟读《三国演义》的殷铁头的功力，就凭在球场上闪躲腾挪追打裁判那点三角猫功底，怎么能搞懂运筹帷幄后发制人的精髓？"防反"这个法宝，中国人最熟悉。什么"自卫反击"，什么"人不犯我我不犯人"，什么"广积粮不称霸"，你来攻我绝对让你
拉稀。但不知为什么，这招到了国际赛场为什么就不灵？

　　爆冷还不止这些。大连人迟尚斌率领大连人占多数的厦门厦新把大连实德夺取双冠王的梦想留给了明年，南斯拉夫人为什么在足协杯这么倒霉？还有，辽宁狂灌无心恋战的吉林敖东；涅波干掉没有了霍顿的8848；比坎尼奇率领众哥们儿险胜天津泰达……

　　但是，糟糕的场地，糟糕的水平，糟糕的配合，糟糕的央视虚拟，还有糟糕的天气。但赢了球的当地球迷高兴，足矣！

——国际篇——

　　上个世纪20年代，当我们中国人刚刚认识现代足球时（尽管有人考证，宋代的高俅是那个年代的罗纳尔多，颇受名模孙二娘青睐），最金发碧眼的瑞典人已经毫不客气地用8：1和7：0让红魔比利时找不到回家的路。

　　从感情上说，我喜欢瑞典队。曾几何时，娃娃脸布洛林、长相酷似因斯的达赫林、风趣幽默的拉维里在1994年世界杯赛场上牵动了多少中国人的心。可惜布洛林堕入瑞典皇宫的粉色陷阱，终日骄奢淫逸；拉维里不再在球网附近表演喜剧小品，达赫林也不知屯在哪里……缺少了这些人的瑞典队，面孔陌生得使人生疑。当类似神秘宗教仪式的开幕式结束时，当刘建宏用变着调的嗓音激动地告诉我们：瑞典球迷喊的是瑞典语（很难想象瑞典人会喊：雄起！），一切悬念便在牙医的指挥下开始演绎。于是我开始亢奋，开始用火柴棍支着打架的眼皮，激动人心的揭幕战奏响了华丽的序曲。

　　没有一直高兴，仅仅相持了43分，古尔的进球就将我的一腔热血彻底葬送了。他凶猛地从瑞典队后卫脚下抢断球后直奔禁区，面对已经不是拉维里的瑞典大门，他完全有理由充满信心地用左脚一击——完全够了，挡不住的风情。他的名字起得太好了，与"goal""good"之类谐音。由此强烈建议，我国的前锋要向他好好学习。比如可以改名为：郝球、宿进、黎门之类。估计从此韩国队只有在他们大门的网底暗暗哭泣。

　　然而，悲剧还在上映。下半场仅仅1分钟，黑马蜂姆彭萨就用无敌幻影手将瑞典队彻底送入渊底。于是瑞典队的"连体主帅"拍案惊奇，梳着无数小辫儿的拉尔森挽胳膊挽腿，反击战激烈得让所有观众窒息。于是，在慌乱与得意的熏陶下，红魔36岁的门将终于为自己的"表演"付出了沉重的代价。挂在德维尔德那张老脸上的尴尬是惨不忍睹的，我宁愿将这份尴尬送给央视的韩老师。（他在提前拿到双方出场阵容的名单时，向我们亿万观众郑重其事地宣布：比利时队1号门将是德尔维齐奥。并说出以下精彩句型名垂青史："比利时队，大多数人不被看好。"这真证实了一个美丽的传说：他的嘴比脑子反应快）。

　　红魔身上黑色的颜色是瑞典队涂上去的，这支94年世界杯的季军就这样喝下海瑟尔球场藏好的毒，我的心情真正落幕了，我的爱恨已真正入土。既然红魔已经初露黑马峥嵘，那么就让下一个对手意大利把它涂得再黑些，反正老佐夫已听到下课的忙音，不如成全了这匹黑马，让它利用主场之利，让它趁着首场告捷的热劲，撰写一篇《黑马过关功略秘籍》，供全世界弱队照本宣科，反复背诵学习，尤其是中国队中那些自命不凡的爷们儿们。然而，意大利今晚给了我们一个惊奇。佐夫终于不甘寂寞，挥舞进攻的大旗。孔蒂的倒踢紫金冠、因扎吉的缥缈侧飞，令土耳其人终于明白了欧洲足球顶尖水平的真谛。

　　紧接着，法兰西的浪漫对决安徒生童话。是法国人的爱情俘虏了白雪公主，还是七个小矮人掀翻艾菲尔铁塔？下周再见！
