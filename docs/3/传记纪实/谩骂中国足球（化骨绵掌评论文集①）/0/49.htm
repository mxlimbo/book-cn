魔鬼复制人（网辩版）


　　深夜，施瓦辛格趴在自己家的窗台往里一瞧，顿时惊呆了：一个和自己长得一模一样的男人正在客厅跟自己的家人聊天，聊得眉飞色舞、其乐融融。这就是好莱坞最新影片《魔鬼复制人》(The 6th Day)中的镜头，他被克隆成另一个施瓦辛格，并被莫名追杀。

　　现在机会来了，不用影片中那个怪异的教授，我可以复制另一个口是心非的自己。

　　第一届中国足球网络辩论就是这么一个窗口：你可以把自己克隆成姓名完全一样的，一个为中国足球涂脂抹粉的大毛笔，尽情挥洒你的热情。本来网络写手都是个个疾恶如仇的神鬼战士，看到中国足球千疮百孔、歪瓜裂枣的时候，个个血红了眼跃出战壕，挥舞着各种兵器，表达着我们的愤怒。何等悲壮潇洒！现在把你分裂成两派，其中的28个战士不得不痛苦不违心地拉起中国足球的遮羞布，仅仅为了荒谬的辩题。

　　本来网络评论是唯一可以看见光明的地方，现在已被搞笑的辩题污染了。

　　沂水秀才如果回头看看自己在网易的立论文章，除了笔名没变，恐怕连他自己都不认识自己了。他在复制另一个沂水，一个无奈无聊无力无厘头的网评写手。我为沂水鸣不平，这似乎不是他的风格。

　　我有点愤怒！只因为16道莫名其妙幼稚可笑的辩题。什么“球迷在中国足球整体环境中是否已经是主体组成部分了？”“30岁以上的老队员是否应该成为国家队冲击2002年世界杯的主力?”，什么“大批中国球员去国外俱乐部踢球对中国足球是利大还是弊大？”“应否在联赛中为向国外出口队员的俱乐部增加积分进行鼓励？”……明摆在那儿，谁是正方谁SB。

　　中国足协什么时候把球迷放在眼里了？阎世铎听信气象专家章健的簧舌蛊惑，讨论什么“暂停升降级”，美言什么“为了俱乐部的发展”“为了培养青少年选手”，他问过球迷的意见吗？在他眼里，看台上一片空荡荡也是正常的，没有竞争所以“快乐”，就像他屁股底下雷打不动的铁板凳，稳坐钓鱼台，管你是“鬼子还是八路军”。（阎世铎语）

　　老态龙钟的范志毅在亚洲杯大言不惭地说：“我开始不理解米卢的战术意图，现在我终于摸到门道了，我开始喜欢米卢。”这种素质的中国足球大哥大你还指望他能干什么？“发烧友”郝海东都快成病大虫了，“老马识途”正在寻找回家的路，你非要证明缺他们不可，痛不痛苦？

　　老的倚老卖老，挤眉弄眼，小的却被捂着盖着。别提上英超意甲了，就连在国内不够岁数挪个窝也不行。看看邻居日本，你才能明白什么是中国人的小聪明。眼睛盯着一寸远的地方，却不知道一尺远的地方是沃土。我们宁愿看他们在国外碰得头破血流，也不愿目睹他们在中国足球这片大泥淖里从“球星”衍变成“流星”，年复一年，日复一日。什么？还给俱乐部增加积分？这真像你要带一个孩子出去学烙夹肉烧饼，他爹说：“不行！他走了我们家谁蒸窝头？不然你多给我二两玉米面，麦糠都行。”……

　　我赤着脚跑来，以为网络有大劫难，本想拉一车砖头，一看是让我扮演另一个角色，还不如回家歇着。

　　仓促筹备，仓促命题，以为给没有足球的冬天带来一把火，结果这些辩题的苍白与幼稚便显露无疑。为一个荒唐的理由找立论，还不如组织一个网络足球论文大赛，谁写的好谁是冠军。

　　我讨厌此次网辩的评委，他们高高在上，决定各著名写手的命运。如果纯粹是各网站的代表以及众多网友作评判，我举双手赞成，毕竟这是我们的世界，就算他们厌恶我，用砖头拍死我，我也心甘情愿。现在请来电视台和报纸的所谓“大腕”，算个卵？

　　《足球报》早就变味了，在《成都商报》受够窝囊气的李乘鹏本来是一杆猎枪，就算没射中目标，听听弹壳退膛的声音也爽。投奔《足球报》后马上变成一杆变质的蜡笔，越涂抹越没边际。你能指望谢奕欣赏你射击的声音？别提董路，整个一个御用。当然最不能容忍的是我们央视装模作样的黄健翔，我不是对他个人有什么血海深仇，我无法容忍的是央视的傲慢。我知道这也许是黄健翔的无奈，就像李乘鹏为了粤菜必须在西装外边围上餐巾学两句粤语。但用这帮人来裁定网络评论，我不知是一种妥协还是历史的倒退，要知道他们根本没这个权力，网站的斑竹和千千万万的网友才是我们网络评论的上帝。

　　让我在这场倾斜的网辩痛苦地扮演另一个角色，我会寝食不安，退TMD一层皮。

　　如果这是辩论的规则，你必须为自己抽到的辩题狡辩，从而展现自己的水平，那么我可以表达我不喜欢。

　　我没选择，我必须违背我自己。因为网络是我快乐的地方，没有它我会干枯而死亡。我赤身裸体站在我厌恶的评委面前，无奈地叹着气。

　　江海红狐在我身后给了我一脚：“把脸揣进兜里！”我照办了，面容慈祥地开始进入复制程序。我删去给中国足球的癌变进行化疗的命令，将自己另存为温柔的版本，准备粉墨登场。
