谩骂中国足球之戴大洪篇

　　戴大洪，中原足球的精神领袖，曾经扛着“反叛”的大旗向威严的中国足协发起有史以来最猛烈的攻击。看他的每篇抨击足协的文章，都恰似解剖毒瘤的手术刀，我热血沸腾得血管几乎爆裂。恨不得立即丢下身边的一切，跑到河南追随他老人家“揭竿而起”闹革命去。

　　可惜他只是昙花一现，虎头蛇尾。仔细一想，可也是，整个一个丐帮头领，错把打狗棒当成了狼牙棒，起了打虎心肠。开始还有几个人在旁边起哄架秧子，如张斌类，后来便逃之夭夭，没了踪迹。

　　一个穷酸俱乐部被中国足球之怪现状肆意戏弄了，所以戴大洪愤怒，戴大洪呐喊，戴大洪奔走相告，到处喊冤。犀利的笔锋，缜密的分析，大胆的痛斥，拳拳到肉，毫不留情。我们当时恨不得跟着他振臂高呼：“戴大洪万岁！王俊生下台！”

　　但你冷静下来仔细一想，戴大洪顶多是个蚍蜉撼大树，中国足球的黑暗腐败还不是照样在光天化日之下明目张胆地上演？戴大洪列举的无数个特大???????号还不是没有答案？

　　再仔细一想，假如河南建业当年没掉级，假如黑哨不是太黑，你戴大洪还会勇敢地站出来吗？还会不会痛揭中国足球的伤疤？

　　所以，戴大洪的愤怒是弱小的，也是非常有条件的，他的勇敢肯定掺了假水。这从他的字里行间处处流露出菜贩子嫉妒开奔驰的大款心理便可以窥见一斑。

　　我们不能强求戴大洪得了便宜还必须骂娘，那肯定是傻子，但我们也不欣赏受了欺负才正直。鲁爷早就有总结：中国人有窝头吃便不会造反的。我们生活在这个巨大的酱缸里，自然逃不出酱缸魔力的污染，连牛气冲天的国际著名教练都得适合这种酱缸的臭味，何况你一个戴大洪。连窝头都没有的戴大洪，所以愤怒。

　　我们曾经天真地妄想让王俊生下课滚蛋，就像戴大洪骂的：我搞足球，你可以欺负我；我不搞足球，你算个什么东西？可王俊生真的下课的时候，我们才无奈地发现，多少个俊生在排队翘首等待！

　　所以我们的愤怒和戴大洪一样，弱小而可怜。中国足球依然说不明白，王俊生依然风光无限，而且经常被自己的冷笑所惊醒。以前他主管业务，现在他洗主管业务的人的脑子，更可怕！

　　而此时的戴大洪你在哪里？是因为中国足坛从此一唱雄鸡天下白，还是您远离了中国足球和张斌在小酒馆里暗自垂泪？依你的脾气，你是不会沉默的。

　　是条汉子你应该站出来，不然你以前的愤怒等于放屁！
