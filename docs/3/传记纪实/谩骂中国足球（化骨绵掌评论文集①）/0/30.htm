扼住命运的咽喉

　　他们铁青着脸，背负着各式轻重武器，唱着雄壮的进行曲，从巴尔干半岛森林出发了。他们是英勇的游击队战士（其实是欧洲战场上的老油子），十几个人，七八条枪。他们面对的是被卡马乔调教出来的斗牛士，没有枪，但人人握着寒气凛凛的利剑，适合一剑穿喉。主动权始终掌握在这帮游击队战士的手里，只要他们不输，他们就可以微笑着坐山观虎斗，把别人的命运玩弄于股掌之间。这感觉真叫舒服，因为他们有这个实力。我从没有见过杀得如此惨烈的场面，顽强的斗牛士牢牢地掐住了生命的咽喉，尽管对手经常潇洒地射出最后一枪。他们知道这帮老油子不出手则已，一出手必然轻易地将你置于死地，绝对不能给他们超过三次的机会，这也正是斗牛士们最终没有倒下的原因。

　　最后的几秒钟，竟然可以牵动这么多人的神经。挪威森林没有刺中同样来自巴尔干半岛的杂牌游击队，在他们颤抖着祈祷的时候，突然冒出来一个可恶的点球和一个可恶的阿方索，于是高大的挪威人只能选择尽快从荷兰阿纳姆球场逃离。南斯拉夫和西班牙，C组最好的两支队伍，从1秒钟前的厮杀中清醒过来。他们相携着扶起受伤的身躯，擦掉脸上的鲜血，一瘸一拐地向黎明走去。

　　凌晨的硝烟还没消散，空气中仍然弥漫着焦糊的气息。不只怎的，我忽然想到去年降组的广州松日，也是在决定自己命运的最后关头，他们战战兢兢，颤抖着祈祷山城的战况对他们有利。已经是困兽的沈阳海狮在最后几秒钟咧开大嘴开始咬人了，并且奇迹般地掌握了自己的命运。在我们为今天的斗牛士顽强拚搏的精神所震撼的时候，我们突然发现，原来我们的球员也有这样的能力（我是不是说漏嘴了？）只不过这种超强的能力被阴险的交易替代了，所以它不能和遥远的欧罗巴相提并论，所以它从来没在韩国队面前出现过，所以它……没有任何价值！！！
