每周述评——今夜黎明静悄悄


这个星期每个中国球迷的魂都被牵到神秘的黎巴嫩去了。在那个曾经战火纷飞的国度，在那个亚非欧交界点，在那个封闭的阿拉伯习俗和妓女同时并存的地中海边，有一群爷们儿在游泳。12支特劲的旅在那儿汇合，为了争夺亚洲最高的荣誉。

比赛的激烈程度不亚于贝鲁特的炮火甚至惨烈（不包括中国队扮演诸葛亮的后两场球），亚洲各队的实力对比仍然如故：伊朗、日本属于第一军团，韩国、伊拉克、沙特，甚至包括中国都在后面气喘吁吁地追赶着。

伊朗本来就是传统意义上的强队，金州那场窝囊球遗了多少中国球迷的梦。本来我们还瞅冷子赢过他，可那场球后，我们再也不敢正视他们的强大了。阿里代伊、巴盖里、阿齐兹、马达维基亚突然一天在我们眼前变得高大起来，成了一道几乎不可穿越的伊朗沙漠。这种力量速度技术都不缺乏的队伍是中国队最头痛的，而且我们与这种硬碰硬的队伍一交手，在身高力量都处于劣势的时候，技术粗糙这个老大难便突现出来了。

日本的神速进步大大出乎一些中国专家的预料，从他们的传球接球停球的每招每式，整体作战的战术素养已远远把其它亚洲球队甩在了后面。不管人家是不是真的"从娃娃抓起"，不管迟尚斌是不是带过日本的现役国脚，反正人家聘请外籍教练的时候没有高呼："日本的足球还是靠日本人自己，外籍教练必须符合日本的国情！"人家是真虚心，不像我们某些人表面上嚷着学习，其骨子里透着"泱泱大国"的豪情，看见谁都像是给他扛麻袋的。如果有一天日本人真的喊着："我是世界第一！"你千万别吃惊，什么技术都学到手了，他就开始露峥嵘。这不单表现在足球这一领域，看看他们的电器汽车，开始是孙子，后来让全世界尊敬。而我们的队员还沉溺在"瞎蒙"的阶段上，说什么"碰上日本队我们不一定输"。是的，我相信。但比赛看的是内容，不是你偶尔发挥好就能证明你是强队。当年我们还赢过意甲的大牌球星呢！那时我们是怎样的狂热？多年以后细细想来，人家不定怎么瞧不上我们呢！赢一场球我们的世界排位还是徘徊在60至80名，遇到韩国卡塔尔沙特，我们还是战战兢兢。

从小组赛中国队的表现，米卢的功劳还是卓有成效的。虽然看上去还没有保守的霍顿时代踢得好看激烈，但结果却相当不错。虽然后两场球"松懈""默契"得惨不忍睹，但毕竟尝到了多年未见的小组第一啊！而且还可以悠闲地看着老冤家韩国队在泥泞中挣扎哀鸣，这种久违的快感洋溢在每个中国球员的脸上，灿烂而多情。老米好像没来得及改造中国队的痼疾，倒把"运气"给带来了。门梁帮我们，裁判也帮我们，这种情景在戚务生霍顿时代简直不可想象。但愿这种"运气"一直伴随着中国队走完全程。

今夜黎明真的很静，"八强赛"前的沉寂意味着离炮火不远了。黑子说，我正想找卡塔尔报仇呢！这话说得让人热血沸腾。我们应该有点"复仇"的血液涌动，不然，像对印尼对科威特那样磨洋工，你丢得起那张脸，我还丢不下这张皮呢！玩点猛的，冲进决赛，拿一次冠军。这事挺难，真做了，捅破了那张霉气的纸，说不定真一发不可收拾了。我相信这句话，信心是在赢球的时候建立的，"稀里糊涂"赢一次，没准你就是强队了。
