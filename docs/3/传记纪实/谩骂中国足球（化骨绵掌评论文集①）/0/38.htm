刘甲B，远离甲B！

　　又见刘建宏生气，暮色罩大地。腻不腻？不腻！关于刘建宏，永远有话题。不是因为前段时间他的"本意"惹恼了蜀中各路豪杰，没必要道歉的他已经在《足球之夜》里猛拍四川球迷的大腿根，拍得曾经出离愤怒的球迷们哭也不是笑也不是。这次是因为周六他主持转播的一场甲B。

　　心中只有一个主队的刘建宏有理由对眼前这帮傻B不屑一顾，整场比赛刘建宏几乎是用悲痛的口吻向 观众述说一个不争的事实："这脚球踢得不知所云""我们的球员战术素养实在差劲""又是盲目直传""照这么提下去，观众就会昏昏欲睡"……。他音调低沉，无精打采，仿佛在告诫正在收看转播的全国球迷：甲B，没劲！

　　没劲我们看个屁？谁都知道中国足球踢得臭，甲B更是踢得球迷眼球充血，但是千千万万的球迷心中除了"我们的大中国"，他们还会不遗余力地为自己的家乡球队呐喊助威。谁也没指望甲B能踢出酣畅淋漓的足球，也没指望它跻身世界强队的行列，但是它是一方球迷欢乐的载体，曼联比不上，喊喊冲A总可以吧？所以说，你在痛说革命家史的时候，能不能照顾一下当地球迷的情绪？他们冒着烈日，为自己喜爱的主队"雄起"，照你这么说，这么臭的球都有人看，是不是都是弱智？这种骨子里的清高让你离球迷越来越远了，这也正是前段时间四川球迷对你愤怒的根本原因。

　　有人说，刘建宏敢说真话，是条汉子。这使我想起金凯瑞演的"大话"，不说真话他憋得慌 ，连在电梯里放了个变了调的屁，都要捂着鼻子对着快要熏倒的众人大喊一声："IT'S ME!"前几天一个美国著名 喜剧明星在国会招待会上大讲荤段子，弄得西装革履的政客们面面相觑，大骂让他回家漱口。他对媒体解释说："我可是说的男女间的真话。"所以说，讲真话要看时机和场合，不是一味地大放厥词。你刘建宏直击中国足球的黑暗，我们巴不得你拎起斧头一阵乱劈。但真让你说出点"真"话，你敢吗？所以"真"话只是自欺欺人，你还没具备薛×杜×那样的勇气。

　　像这种惨不忍睹的甲B，我们不需要你无休止地痛揭，我们需要无偿地鼓励。刘建宏是从报道甲B而闻名于世的，既然你已经厌倦了在这种低劣的环境中鬼混，能不能下一会儿课，换个有激情的年轻人试试？前段时间轰轰烈烈的主持人大赛怎么没了回音？那几个生龙活虎的小伙子是不是只有比赛的份儿？让他们转播甲B，观众绝对不会听着泄气。

　　我想，也许，当然，你心中确实只能有一个主队，看看韩乔生能不能闭上他的乌鸦嘴让让位？看看中央那么大的牛B台能不能争到转播国家队比赛的权利？刘甲B，你应该远离甲B，别让这么糟糕的场面污染了你的嘴，也别让你被污染的嘴剥夺当地球迷快乐的权利。
