解说不是妓女

　　荷兰南斯拉夫之战刚刚结束，电话铃声骤响，是我一个平时不看足球、小学文化水平的朋友。他劈头盖脸就是一句："这场精彩的比赛给了那个妓女一个响亮的耳光。"我一下子惊呆了："哪个妓女？""就是那个一直唠唠叨叨的解说员。"

　　接着他用他的观点阐述了对黄健翔的不满："比赛一开始，他丫的就掩饰不住很私人的观点，把南斯拉夫球员夸了个遍，什么攻击性很强，什么即使输了也要把对方大门戳几个洞，什么个个都是世界一流球员，再加上国际足联对他们很不公正，屡屡遭裁判黑判，这反而会激起这些球员的斗志。他丫的逼我们喜欢南斯拉夫，逼我们的同情心大泛滥。难道全中国球迷都喜欢南斯拉夫不喜欢荷兰？难道米哈伊洛维奇自动申请的红牌是裁判装蒜？如果他一直喜欢下去倒也罢了，那才算爷们。可当克鲁伊维特攻进去4个球后，他马上变成另外一副嘴脸，猛拍荷兰马屁，什么荷兰找到了南斯拉夫后防线的漏眼，还说什么再罚下一人兴许好看点，什么博斯科夫也许在中场教导他的球员怎样重演连扳三球的好戏……他丫的到底什么心态？这跟街边的妓女有什么区别？谁有钱跟谁走。"

　　我连忙打断他的话："兄弟，千万别侮辱我们的健翔，怎么能跟妓女相提并论？最多也就算个假老练。世界上的足球战术阵形都瞒不住他的眼，他不但给人家出主意，还会揣摩人家的心态，而且还会根据人家的口形判断出队员围着裁判是有人劝有人扇阴风点鬼火。裁判公不公正全由他那张永不停歇的嘴决定，就像前几天那个叫达拉斯的裁判，水平是差了点，可他一说就没完，整场比赛成了苏格兰足球大批判。他丫的没想到人家可能出任决赛，他没想到国际足球裁判委员会主席就来自苏格兰。他当时可能心里想：要是我来吹该有多好啊！哪怕是一场最微不足道的小组赛。你丫的不想听，干脆把声音关了。"

　　朋友说："我又不是聋哑人，想不听都难。唉，算了，有他在那儿表演，想打瞌睡也难。听听他怎么解说西法之战吧！没准还有什么精辟语言让我们暴跳如雷，有他在，欧洲杯更精彩！"
