王俊生惨死锅炉房

　　清晨7点，我照例为我老婆煎好荷包蛋，熬好黑米红枣营养粥，然后将一张伤痕累累的录像带放进录像机，设置好播出时间，便悄悄地拉上房门，到隔壁的南郊公园练九阴化骨绵掌去了。

　　8点，组合音响自动开启，王俊生那孤独苍凉、性感撩人的声音立即弥漫整个房间：'从来就没有什么救世主，中国足球全靠我们自己……''王俊生在喊我起床啦！'她一个鲤鱼打挺从床上翻了下来，穿上珍珠隆乳按摩胸罩，将叠成梯田似的肚囊皮塞进收腹带，眼屎都没抠干净，就一步跑进客厅，伴随着王俊生的'中国足球宣言'跳起了她自编自导的健美舞，健美舞的名字叫——我爱王俊生。唉！自从我们老两口退休后，憋在家里不闹点什么就觉得空得慌。

　　我倒是立即找到了自己的最佳位置，荣升成都市南郊公园九阴化骨绵掌的舵爷，是众多老娘们儿心中的偶像。我老婆可就惨了，不追濮存晰，不追张铁林，追了一个特大冷门：追中国足球的舵爷——王俊生。这不，这两天她正在积极策划王俊生与追星族的见面会，可惜没人报名。

　　昨天，隔壁被我老婆鼓捣得有点春心萌动的刘太婆悄悄告诉她，据内部人员透露，王俊生将于今日抵达成都。当夜，我老婆激动得辗转反侧，起身披衣，扭开台灯，饱蘸深情，奋笔疾书：亲爱的生生：难解百般愁，相知爱意浓。想要给你的思念，就像风筝断了线，我的世界开始下雪，冷得连隐藏的遗憾都那么地明显。而你是一张无边无际的网，轻易就把我困在网中央，我越陷越深越迷惘，路越走越远越漫长，不知我们能否相聚在双流机场。夕阳醉了，落霞醉了，默然将心偷取了，但愿我们两个是盅蒜的对窝，你是棒槌，我是钵钵，我离不开你呀你离不开我。听说你要来成都，为此，我特到伊藤洋华堂给你买了一把铜梳，希望你能经常梳一梳你那帅气的络腮胡。最后，我和你吻别在狂乱的夜，让风痴笑我不能拒绝。我老婆今天一大早就急急忙忙赶往双流机场去了。

　　一到机场，就看见成百上千的少男少女手握鲜花簇拥在机场出口处，她那么大岁数哪里靠得拢？后来仔细一打听，才知道情报有误，王俊生根本没来，而是小燕子赵薇抵达成都。

　　就在我老婆悻悻然揣着铜梳往家走时候，一个西装革履的小伙子手拿一匝五颜六色的报纸与她擦身而过，他沿街声嘶力竭地吆喝着：'快来看呀快来买！特大新闻：王俊生惨死锅炉房！'这消息犹如晴天霹雳，我老婆险些昏倒在地。

　　她努力掌握着平衡，踉踉跄跄地朝小伙子走去，有气无力地说：'小伙子，我买一张。'小伙子将一张粉红色的纸条递到我老婆手里，不耐烦地催促道：'动作搞快点！一块钱一份。'我老婆付了钱，强忍着悲痛的心情，展开纸条一看，见上面只有一行粗大的黑体字：王俊生惨死锅炉房！欲知详情，再付10元！我老婆哪会甘休？她忙追上小伙子，从收腹带里抠出10元人民币，买过'详情'一看，见印刷精美的纸张上写道：四川省成都市神经锅炉厂青工王俊生违反安全操作规程，引起锅炉爆炸，惨死锅炉房……我老婆气得七窍生烟，抬头找那个小伙子，哪里还有影儿？

　　她愤怒地在人民南路旋到下半夜，边走边咬牙切齿地骂道：'龟儿子的！利用我的偶像骗了我11块钱，逮到你娃娃，马上喊我老公用九阴化骨绵掌把你抖散！'
