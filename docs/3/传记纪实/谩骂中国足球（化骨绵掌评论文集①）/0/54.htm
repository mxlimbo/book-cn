从凋谢的玫瑰……


　　我们把中华民族所有的溢美之词送给了中国女足，从来没吝啬过。因为在她们身上还可以依稀看到一丁点体育精神，正是这点精神支撑着多少人的梦想啊！我们甚至在她们输给挪威的时候也舍不得有丝毫的埋怨。尽管她们在该死的"打平就可以出线"的大好形势下缩手缩脚起来，尽管上半场跟着挪威"磨洋工混时间"，尽管最后的15分钟小腿已经痉挛，尽管还存在那么多"尽管"，但我们一点也不会责怪她们。要知道这些姑娘曾经给我们带来多少欢乐啊！她们是我们心中永远的玫瑰。

　　玫瑰虽然凋谢了，我们还是选择赞美。但有些不中听的话仍然憋在心里不吐不快。是突如其来的荣誉把她们彻底压垮了吗？马元安在埋怨球员淹没在耀眼的光环中的时候是不是也把他自己算在内？他不是凡人，同样不可能从巨大的荣誉中超脱出来。有人说，马元安已经将女足带到了高山之巅，他没有能力再前进了。这从他死抱着霍顿的保守战术就可以窥见一斑，而霍顿的那一套是经过惨败证明了的，连女足的后卫都在嘀咕。但你能让马元安怎么办呢？要知道这可能是他和"世界接轨"的唯一通道，他倒是想周游世界取点真经回来，可惜没这个机会。

　　回头再看看国内女足的悲惨现状吧！取得世界亚军的时候恨不得是金疙瘩，谁都想站在旁边蹭点便宜，又是"改革的成果"，又是"中华的腾飞"，咱这有特色的路算是走对了。说是向女足倾斜，说是加大女足发展的力度。光刮风不下雨，唾沫星子四溅。说白了，女足的胜利是某些人脸上的金粉，一旦这层金粉失去了魅力，他们恨不得洗八次脸。

　　一个国家仅仅需要体育成绩来装扮自己的信心，来粉饰脆弱的脸面，要向全世界宣布"站起来了"，那"发展体育运动，增强人民体质"就是一句空话，白喊了几十年。试问：中国有几支女足队伍？有多少学校有金钱有精力有兴趣开展这项运动？有多少场地向我们的青少年敞开？联想到今天上午战胜世界亚军的女子曲棍球队，中国有多少人认识这玩艺？全国只有可怜的6支队伍啊！但她们必须担当粉饰颜面的责任，不然泱泱大国缺少这项运动不是被人耻笑吗？哪想到在韩国人的调教下，连克世界冠亚军。要知道，韩国人身边没有翻译，他仅仅靠"传""向前"等几句汉语指挥比赛。三百多人庞大的体育代表团难道容不下一个韩语翻译吗？是真心开展这项运动还是死撑面子？一目了然。不过这倒是给了"外国教练必须适合我国国情"一个响亮的耳光。一个根本无法和队员沟通的教练却能让女曲捷报频传，真不知道是谁适应了谁？

　　什么时候将体育的本质还给人民，而不是什么"金牌战略"，什么时候我们就真的站起来了。
