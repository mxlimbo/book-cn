葡萄佳酿醉倒英豪烈汉

　　醉倒骄傲的英格兰人只有一个绝杀：灌他！别看他们个个喝酒如喝白开水，但今天凌晨却被背着葡萄酒桶的葡萄牙人灌了个不省人事。尽管开场三分钟"农民斯科尔斯"（健翔语）黄毛一闪先给了葡萄牙一个下马威，尽管贝帅哥的眼睛和右脚同样可以杀死女人，尽管内维尔兄弟比马荃还马荃，但费戈平托们拒绝了英格兰人的潇洒，他们开始给英格兰人斟酒，一杯又一杯，看他们还能喝几杯？

　　跟巴西人操着同一种语言的葡萄牙人确实不愧为欧洲的巴西队。他们纯熟地玩弄着同样讲究技术的英格兰人，潇洒最终属于这支1989和1991年世青赛冠军队：费戈直接向贝帅哥挑战，热奥・平托的脑袋比斯科尔斯更刁钻，更有一个金发金胡子的沙维尔瘦骨嶙峋，活脱脱一个藏匿深山100年的武林点穴凶汉。他尤其适合在双方厮杀得鲜血淋漓时，戴着斗笠在山颠出现……

　　长发飘逸的戈麦斯晃得酒量惊人的亚当斯踉踉跄跄，只有醉眼长叹那可恶的球门能不能再小点；坏小子怀斯也只能瞪着恶毒的眼睛咽下不敢发作的恶气。曾经怎样酣畅淋漓的英超啊！怎么喝点葡萄酒就如此支离破碎，只能饮下千古遗憾？

　　科埃略说："这是葡萄牙的黄金时代。"是的，葡萄美酒必须用黄金铸成的酒杯斟满。后面是老迈的老男孩德罗二老在等待，别给他们任何喘息的机会，别再重演上届惜败捷克的悲剧，乘胜追击，尝尝用金杯畅饮美酒的滋味。

　　在这个夏日的破晓时分，崇尚技术的葡国人让我们大开了眼界。我不由得想起那个重复了一万年的美丽传说：中国人就是要走技术之路。现在看来，恐怕后面还要加上一个100米长的破折号――需要100年，并且不变。

　　怎样的距离啊，根本无法追赶。让中国足球腾飞，就凭哭着要奶喝的俱乐部官员？就凭装孙子的牛B球员？就凭后妈似的足协？就凭光会发明口号的教练？用一句篡改过的广告语送给它：农夫山泉――有点悬。
