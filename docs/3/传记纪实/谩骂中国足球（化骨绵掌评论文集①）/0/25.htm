三叉戟重新起飞

　　看到挂在张玉宁李金羽曲圣卿脸上的笑靥，看到他们在雷锋体育场勇猛顽强的身影，看到他们胜利后击掌相庆的神态，我不禁被这三个阳光男孩深深感染了。但同时，我的心头莫名地升起一个巨大的疑 团：到底谁是叛徒？

　　难道几天前还在悲愤的童子军们眨眼之间就可以和叛徒不计前嫌重归于好吗？不，我不相信他们的笑容是在电视镜头前作秀，因为他们的城府没有这么深。他们该愤怒的时候愤怒，就像在曲乐恒的病榻 前将某位正在窥视的"鸡"一顿狂扁。他们该流泪的时候流泪，就像在老爸的房间，面对央视的镜头也不肯让泪水在前面等待。他们该高兴的时候高兴，就像进球后千方百计将欢庆动作设计得稀奇古怪。

　　他们近似于一张张白纸，还没来得及让世俗的油滑注入他们年轻的血液，不像某些俱乐部老板，演技直逼奥斯卡最佳男演员。

　　难道"叛徒说"又是一条臭名昭著的假新闻？

　　如果老爸说过这句话，那他老人家的心眼已经退化到洗衣妇的边缘。他亲手缔造的辽宁三叉戟的表现，足以让他在欣慰的同时，心底隐隐有点酸。

　　如果老爸根本没这么讲过，或者原意并非如此，那没准又是唯恐天下不乱的鸡们蓄意制造的"辽足悬案"。因为我们在所谓传统媒体上所看到的"叛徒"与我们亲眼所见相距甚远，所以我们有权力质问，有权利怀疑那些自诩为掌握舆论导向的"工具"们，你们登在报纸上的黑字到底真不真？

　　也许我们真有点杞人忧天了，因为只有他们有权力编织天方夜谭，而我们只能被动地听或看。他们是"红势力"，我们是"黑势力"，他们可以随意洗你的脑壳，因为他们被赋予了这样的特权。

　　我们有的是闲时间的老百姓刚想发点杂言，就被扣上了"网络霸权"的帽子。

　　到底谁是霸权？说深了我就离戴手镯子不远了。

　　好歹这些事都不是你我说了算的问题，都是蚍蜉。你要为你的事业，或者鼻子底下那张嘴 ，而我们图什么？一这不是我们的工作，二拿不到一分钱，所以我们没有被肮脏的功利主义束缚，所以我们的心 比较纯——只为心中热爱的足球摇旗呐喊。

　　不管怎么样，年轻的三叉戟重新起飞了。他们用犀利的进攻，顽强的拚搏，漂亮的进球，彻底封住了周围说三道四的臭嘴，他们从恶毒的流言中觉醒了。此时此刻，什么"叛徒"，什么"车祸"，什么"自 杀"，什么"车中的神秘女人"，什么"泡吧"……在三叉戟五月灿烂的笑容中变得渺小苍白了，同时也映照出某些"窥视俱乐部"的干将们心里是何等的隐晦肮脏灰暗。

　　三叉戟，祝福你！因为你们年轻，所以你们是未来。
