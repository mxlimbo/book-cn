中国足坛歪歪时报（第一辑）


　　●川连闷战临近结束时，黎兵在本队禁区前30米飞铲孙继海，终获职业生涯第一张红牌奖励，他主要是想让大连队抓住最后一个机会罚直接任意球。他把这个想法告诉了李明，但李明并不领情，因为他没听懂，黎兵说的是贵州话。

　　●广东警方出动1000警力寻找香港弱智儿童，成都球迷协会出动2000会员寻找郝海东。原来东东吃了某度假村的皮蛋上吐下泻，川连比赛时郝兄正蹲在便盆上为大连队呐喊加油。看来，度假村的厨师身份确实令人生疑。线人密报，说厨师的长相像马明宇。

　　●彼德洛维奇又一次冲进球场，他不敢追打裁判，也不敢跟天津球员比试拳脚，他主要想近距离观摩学习一下中国功夫。哪想到裁判以为他又要施展闪躲腾挪的"跳拳"，毫不客气地将他罚上看台。其实外国友人有学习我古代精髓的强烈愿望，裁判应该鼓掌欢迎。

　　●辽宁俱乐部设下80万巨奖，结果被青岛队逼平。去年也是设下80万，结果被北京国安逼平而失去夺冠的契机。看来80万不是个吉利数，应该设下79.9999万巨奖，剩下的一块钱实在没人要就捐给曲乐恒。

　　●去年张引冷不丁问张玉宁："你姓什么？"弄得全国球迷莫名其妙，谁也没有领会老爸的周星驰式的幽默。今年在辽宁队濒临降级的紧要关头，他应该再扮演一次唐僧，问一问辽宁队的小妖们："你妈贵姓？"

　　●此轮厦门厦新连中四枪，但并没有倒下，下轮对阵青岛海牛才是最恶的一仗。每次记者让迟尚斌发表对"土洋之争"的看法时，他总是说："中外教练带的队不同。"他的意思是，他要是带曼联，可以横扫英超，只是弗格森不答应。

　　●王亮这孩子有出息了，现在成了米卢的爱徒。王洪礼说："他虽然是我儿，但去年没有得到张引的重用，不然这孩子早就是国家队的了。"而张引说："我带王亮15年，只有四个月没跟我练球，他跟我很有感情。我下课那天，王亮特地跟我说了许多话。"看来这个赛季末，王亮要进行DNA鉴定。

　　●成都五牛和武汉红桃K比赛前，郭瑞龙一反谨慎的常态，咬牙切齿地说："对付武汉红桃K，我有一套。"果然，五牛用红桃K补了血。像这种能赢该赢的比赛的教练，应该推荐给米卢当助手。

　　●十强赛结束后，戚务生说："我只能负我应该负的责。"一副无所谓的神情，完全不顾神州大地在哭泣。此轮与沈阳海狮2：2踢平，戚务生脸色依然严峻，只不过这次说："在这场对红塔至关重要的比赛中，没有取胜责任在我。不管现在还是今后，我都会负责到底。"看来红塔的荣誉比国家队重要，因为不负责是拿不到钱的。

　　●平均身高1.80米的国青队输给平均身?.60米的越南队，主要原因是越南队全是土行孙，下半场进的三个球都是通过挖地洞攻入的。祖国的希望交给从没有冲出亚洲的高洪波，这就是阎世铎提倡的中国足球战略大转移――从青少年抓起。
