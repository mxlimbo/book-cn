今天天气挺好的

　　冲出亚洲、走向世界的土耳其终于在哈坎·苏克的带领下继续在欧洲的殿堂里上课，联盟杯冠军果然不是徒有虚名。在他们欢庆历史性胜利的时候，他们应该首先庆幸自己“移民欧洲”的英明抉择。纵观今天的土比意瑞大战，土耳其人应该感谢的太多太多。

　　第一，你们应该感谢裁判。不管是先上场的巨人尼尔森，还是那个白发苍苍的本科，面对着输红了眼的东道主，他们的心没软，没有考虑那么多主场因素。换了中国裁判，土耳其想要赢东道主，简直是痴人说梦。因为我们的裁判有太多的“利益平衡”，太多的“人的感情”，太多的“可以原谅的错误”，至于比赛结果公不公正，就“不是他所能左右的”了。于是我们听到的基本上是主场的欢呼声，像土耳其今天创造的奇迹，我们真的不相信自己的眼睛。现在看来，我国的“国际裁判”像是中国足协那几爷子关门自己封的，有没有那个水平，只有他们自己心知肚明。（问题是他们心不知肚不明，还以为自己是超水平。）

　　第二， 你们应该感谢意大利，因为他们没有演绎一场欧洲版的“渝沈大战”。如果意大利是重庆隆鑫，如果托尔多是符宾，让对方猛灌几个球也是情理之中。因为已经出线的队“身体有点疲惫，意志有点松懈，加上对方玩命”，所以“今天天气挺好的”。毕竟“有中国特色的足球”不同于资本主义的欧洲，故事内容也就截然不同。人家是世态炎凉，人情淡薄，我们是社会主义温暖大家庭。只不过这个大家庭里经常欺软怕硬。

　　土耳其，你让我们中国足球眼红。不是熬夜熬的，是纯粹的嫉妒。因为你们能在欧洲玩足球。
