中国足坛歪歪时报（第三辑）


　　●八一和国力队的冲A大战硝烟弥漫，双方队员拼命比试后场倒脚的熟练度和高射炮的高度。看来“皆大欢喜”是难免的了，唯有陕西电视台的解说员看不出门道，从头到尾他一直忐忑不安地念叨着：“究竟八一、国力和江苏舜天哪个能冲上甲A，这是本赛季最大的谜。谜底一会儿便可以揭晓。”后来有人说，他是全国唯一的盲人解说员，外号：装糊涂。

　　●尽管宝鸡全场观众锣鼓喧天，掌声如雷。但场上的22名队员却无动于衷，突然变成了聋子。打扫卫生的老头收获颇丰，他在双方的休息室里发现了几十斤棉花团，可以做成新棉衣供全家人过冬。据说此棉花是双方教练和队员堵耳朵用的。

　　●肇俊哲被红牌罚下去后说：“老外就会装相，非洲球员动作一点不干净，我确实铲了他两下，他躺下后也踢了我。现在想起来那个动作是大了点，但马克回头就来报复，他就对吗？”肇俊哲的意思是，现场和电视机前的球迷在他恶毒施展剪刀脚以后突然双目失明。

　　●彭伟军小禁区前那两脚救门终于使颐中队艰难地留在了甲A，而这时俱乐部官员迟伟和总经理秦宁就像孩子似的抱在了一起。休息室内一片欢腾，桌子被敲得梆梆响，大家把憋了很久的话都用吼声表达了出来。这时，青岛市市长王家瑞端了一盆凉水，一下子泼进休息室。他说：“如果当初一直照这么踢下去的话，就不至于年年保级了。”此言精辟！！！

　　●有趣的是，每当省委书记周永康过问全兴，全兴就会胜利或扭转局面。本赛季周书记曾经三次打电话激励全兴：第一次是主场打颐中，全兴3：1获胜；第二次是主场打隆鑫，全兴4：1首次击败隆鑫；第三次是第21轮后周书记一过问，全兴立即扭转颓势，连续逼平大连、上海，客场击败天津。不少人纷纷感慨：“周书记就是厉害，他一说话全兴就赢球。”马屁能拍出一足坛奇才！幸也。

　　●厦门队降级后，迟尚斌说：“我辜负了八闽父老的期望，实事求是地讲，凭我们队今天的表现，我们也就配在甲B呆着。”迟尚斌真聪明，如果在接手厦门教鞭时讲这番实事求是的话，恐怕没有挣钱的机会。

　　●广州太阳神队凭老将曾庆高补时阶段的入球艰难保级，数千球迷不等主裁鸣响终场哨就兴奋地涌进场内，尽情欢呼。谁也没注意到新加坡主裁桑德在哪儿？原来他正躲在第四官员身后激动地颤抖。事后他回忆道：“当时我激动得裤子都湿了。”

　　●女足在奥运会被淘汰后，王俊生说：“我要保马元安，当初十强赛结束时我就保了戚务生。”看来他完全可以改名，叫王保长。电影《抓壮丁》里的王保长说：“我悔过，我悔过。总发财说了，要一起发财。”这话是说给在红塔正发财的戚务生听的。

　　●涅波去韩国执教的传言甚嚣尘上，章健说：“俱乐部不是完全靠某个教练和队员，而是靠俱乐部具体操作的水平。”他的意思是，即使李强卷土重来，只要俱乐部运作好了，照样可以取得现在的成绩，因为去年渝沈之战就是一个成功的战例。
