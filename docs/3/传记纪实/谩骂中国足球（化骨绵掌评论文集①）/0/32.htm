再说央视解说

　　今天早晨，我的《解说不是妓女》果然引来众多保黄派的谩骂，就像我写的《央视，亮出你的舌苔》。其实这正是我所期待的效果，与其说写些不疼不痒闲扯蛋的东西，不如写点让某些人青筋暴绽的逗 他们原地狂跳。不过，保黄派视我为虫豸异端一点都不新鲜，你讨厌黄刘之类，人家还当大爷供着呢。可某些曾被黄健翔斥之为"流氓"的人非要站出来展示他的流氓本色，急不可耐地把"他爸他妈"招呼出来，成何体统？

　　于是我有点失望。原以为在最时髦的Internet上都是些受过高等教育的精英们，他们比我年轻，比我激进，比我更痛恨某些垄断。但我没想到改造某些人的奴性基因光教育是根本不行的，有些人天生就 是SB，真可谓：狼行千里，该吃肉就吃肉；狗行万里，该吃屎还是吃屎。

　　我本人的文化程度很低，文革末期小学毕业（三个月前我刚刚买了电脑，打开Word97，满头大汗地问人家："输入法在哪儿？"），所以我写不出精彩的球评，只能站在一旁恶狗似的狂吠。但往往没文化 的人能一语中的，就像89年北京一个卖冰棍的大爷，外地司机问他："同志，公主坟怎么走？"大爷怒发冲冠："北京没有同志，都是社会闲杂人员。"就像我的那个小学同类，将黄刘们定性为" 人间哪有真情在，能赚一块算一块"的夜莺。

　　像我这种混混儿本没有资格对央视的名嘴说三道四，因为我工作上没单位，馆子里没座位，社会上没地位，劳改队里有铺位。我失望的原因是，竟有那么多比我还SB的SB竟然受过教育。要说骂人，请相 信我，我可以用最下流的语言和你对骂几个小时不重样，这不算本事，最底层的人都会。我最怕的是摆事实讲道理，可惜还没出现。

　　有些人劝我，看不惯央视你不如移民欧洲，并断定我付不起收视费。感谢你闲吃萝卜辣操心，由于我身处"山高皇帝远"，可以"违法"观看ESPN、香港卫视体育台，可以欣赏凤凰卫视、凤凰电影，甚至CNN，BBC，每个月只付人民币4元。正因为有比较，才感到央视近似"愚民"的傲慢（偏偏有 人至今其"愚"融融）。

　　我并不是真的厌恶黄刘们，也许ESPN的汉语解说还比不上他们。我真正的意思是，2000年代了，我们不应该只有一种声音，不应该几亿观众只能收看一个体育台。可悲的是，某些人不以为这是悲哀，而 是过着令全世界的人都"羡慕"的幸福生活，就像趴在洞房门口的父亲，冲着不断幸福的儿子高喊："你以为那是蜜罐，嘬死你丫挺的。"

　　我的意思大家应该懂了。
