影子的碰撞


　　戚务生和霍顿带着自己的影子部队，在中国有史以来最漂亮的足球场上演了一出沉闷的闹剧。之所以叫“闹剧”，皆因赛前各种媒体的老记躲在一旁，使出浑身解数扇阴风点鬼火，把这场足协杯比赛誉为新旧国家队主帅的“绝命大死拼”，惹得我等鼠辈瞪着牛卵子大的眼睛又上了一回瓜当。

　　戚务生的红塔就像十强赛时的中国队，习惯性的懒散，习惯性的失误，习惯性的单刀不入，惊心的差之毫厘，惊人的橄榄球似的射门……

　　多么似曾相识！

　　而可爱的传教士霍顿率领的8848则与九强赛时的国奥没有什么区别，超人的身体素质，超人的后场长传，超人的横冲直撞……

　　多么相识得不能再相识！

　　谁的手这么馊，偏把这两哥们儿凑在一堆：一个嘴里唠叨着“平行站位”，一个怀里掖着《论持久战》，一个拼命想重新证明自己，证明九强赛失利不是他的错，而是俊生惹的祸；一个压根儿对“证明自己”失去了兴趣，他只想负他该负的责任（我们没看到他负了什么责任）。

　　沉闷的闹剧只能让我们中国球迷心里滴血，十强九强的伤疤还未变硬，就被两个大师毫不留情地揭开了。

　　1：0，霍顿的洋理论在这场“绝命大死拼”中赢了，下岗的俊兄没准正暗暗窃笑：谁说中国足球没进步？上一任的主教练赢了上上一任的主教练，进步还非常明显嘞！

　　我的上海宝贝renny正在现场，她打电话跟我谈了对这场比赛的感受，只有八个字，精辟：“猪打死狗，狗打死牛。”

　　这句评语她曾送给拙劣的香港武打片，现在送给足协杯第一场比赛，蒋介石他妈——正合适（郑何氏）。
