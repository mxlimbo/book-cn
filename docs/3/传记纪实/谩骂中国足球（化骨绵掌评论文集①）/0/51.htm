兵马俑的白话摇滚

　　 我们耐心等待着，稍稍有点困意。凌晨2点了，兵马俑那边一片沉寂。东方晨曦说，不在沉默中爆发，就在沉默中死亡。我的弦绷得更紧了……

　　果然，天色放明的时候，对方”挣开眼睛”甩过来一个叫”实在话”的炸弹，幸好没爆炸，否则我们7个战士非得血肉横飞不可。小心翼翼拆开那颗臭弹一看，差点笑得我背过气去。原来这个叫DIO的有滋有味地抠起了这次辩题的字眼，还把本属于他的”偷梁换柱”急不可耐地抛给了我们。

　　谁在偷梁换柱？他先是绕过”利大弊大”这个辩题中心，打起”出国”这个字眼的主意，说什么”要知道辩论的是大批球员’出国’，但对方辩友却偷梁换柱为到’欧美高水平俱乐部。’……如果我们的国脚、精英、队长们硬被对方辩友塞越南、马尔代夫往这种地方（原文如此！！！），中国球员技战术的升华从何谈起！”@#$%^&*？我倒！原来对方一直把辩题中的中国球员到”海外踢球”硬理解为去越南、马尔代夫？

　　辩论的大厦瞬间崩塌了，因为所有的论点都失去意义，包括对方一辩二辩前几天煞费苦心的文字堆砌。感谢DIO教会了我们”海外”这个字眼的含义，也感谢他让我们重温了一下地理知识。但我们所辩论的”海外踢球”这个特定含义包括越南、马尔代夫这些国家吗？谁都知道”人往高处走，水往低处流”这句俗语，到”海外踢球”肯定是去比我国足球水平高的地方踢，而不是去什么水平低下的越南、马尔代夫，你怎么不推荐去尼泊尔呢？所以，我们大家讨论的到”海外踢球”说的就是到”欧洲高水平俱乐部踢球”，我们首先要有这个共识，才能在这个基础上聚在一起。这并不需要特别交待吧？就像中国人见面常说的”你吃了吗？”每个人都理解为”你吃饭了吗？”而很少有人理解为”吃香蕉或吃其它人类不能吃的玩艺”，可能你DIO是个例外。

　　如果撇开辩题中的”利大弊大”不谈，专门抓对方的字眼进行无意义的诡辩狡辩，《化骨绝学》中有这一招，我马上给你摸脉：

　　你口口声声《睁开眼说几句实在话》，说明你以前没睁眼也不说实在话。“大批中国球员”，”大批”是多少？5个？还是50个？“中国球员”的定义是什么？是职业球员还是包括业余球员大学生中小学生球员？

　　这不叫辩论，是楞往歪的地方理解。就像某个美国佬学习中文时感叹：”太不可思议了，你们中国人管’谢谢’叫’Shit!shit!’管’不谢’叫’Bull shit!’“

　　如果照此”抓漏眼”争论下去，无休无止地争论每个字眼的含义，那才是”与我们开展辩论赛的初衷背道而驰了。”我想你DIO也不愿意这么无聊吧。

　　紧接着悄然一刀《宜将剩勇追穷寇》，开始给晕头转向的我们补课。注意！姑且不论他的论点如何，此时他所有的观点又都建立在去”欧洲高水平俱乐部踢球”的基础上了。兵马俑阵营里如此出尔反尔，神秘”变脸术”随时上映，撕下一张脸，还有更精彩的脸在等待。明天早上刘德华在门口等着你呢，他要学习”变脸”绝技。

　　这种即兴否定自己东摇西摆的风格我们似乎已经习以为常了，因为悄然一刀便是这方面的先驱。再回头看看他的经典：”我方从不否认球员到国外高水平联赛上去磨练能使自身得到提高或者成功，但是一花独放不是春，百花齐放才能春光满园。”

　　再看看DIO的绝句：”不可否认，中国球员无论是从对抗能力、技战术水平还是职业水准上来说都与欧、美等足球发达地区的球员存在不小的差距。中国球员如果能够加入到他们的高水平的联赛中去磨练显然能使自身得到提高。亚洲杯杨晨的大放光彩就是例证。”

　　接着你话锋一转：”个别球员、部分俱乐部显然不能代表中国足球。令人遗憾地是我们看到出色的杨晨并没有能改变中国队落败的命运，这充分说明了个人英雄主义拯救不了中国足球。”

　　怪哉！我们说了杨晨代表整个中国足球了吗？我们只是举杨晨成功的例子证明，我们更应该下定决心把”大批”的杨晨送出去。足球是个集体项目，一个杨晨不行，那更多的杨晨行不行呢？”大批”的杨晨是不是就是”整体”了？（再次注意辩题中”大批”这两个字，悄然一刀已经做了精彩分析。）照DIO的意思，马明宇不行，范志毅也不行，谢晖更甭提，是不是他们都回来就行了？是不是只有克隆11个一模一样的杨晨同时在欧洲踢球才可以，才能提高”整体”？是不是个别球员坐冷板凳还不如全在中国足球龌龊的环境里互相比比谁的牙齿更白一些？

　　中国足球不是没在”窝里练”过，几十年来我们闭门思索，苦心钻研”横下一条心”，到时候赏给球迷的是N个”黑色三分钟”，这种教训难道还少了吗？就算偶尔伤痕累累，也不能因为”因噎废食”而否定”出国踢球”的积极意义吧？

　　我不知道你借谭嗣同的诗：”我自横刀向天笑，去留肝胆两昆仑”能说明什么问题，给我感觉像英勇就义。 像你们这种即兴编词东摇西摆自己否定自己硬说少年愁滋味的白话摇滚，除了说明黔驴技穷强词夺理，难道让我在旁边敲破洗脸盆给你伴奏助兴？

　　走好！伴着Rap编织自相矛盾歌词的兵马俑们，343向你们远去的背影敬礼！
