谩骂中国足球之老“妓”篇（反方）

　　老妓者，老记也。这里所说的老“妓”不包括正直敏感勇敢智慧型的老记，而是专指拿着报社的基本工资每月必须完成××篇稿子专挖花边不然就SB了的痴呆老记。他们手持钢笔（或电脑）歪着脑袋眨着眼睛貌似天真无邪地盯着米卢在黑板上讲阵形，其实他丫的脑子里一片空白。为了生计他必须编它一篇争取卖个好价钱，于是他睁着死鱼眼盯上一个花边凭空卵弹琴，于是便从他笔尖底下屙出一节节丑不可闻的硬屎，颠倒黑白混淆是非扰乱球迷的视线。你别担心，他说了的，写错了自然有人找他，他他妈巴心不得，闹不好闹个闻名遐尔，再孬也就是在报社的黑板报上点名批评，扣个不疼不痒的几吊钱。

　　这一点与傍不上大款的妓女倒有相似之处，管他绵不绵，只要给现钱，胡乱凑合一个喂饱了底下那张嘴再说。即使派出所的找上来，她自己负责，关强奸了她的“白痴们”鸟相干？他们二者的相似第二处就是广告词惊人的一致：欢迎来稿（搞）！

　　搞什么搞？不稀罕！于是有些长满蜘蛛网的老妓便忍无可忍地跳将出来，正好我在写《谩骂中国足球》系列，撞到我枪口上，先按翻了再摆聊斋。

　　但凡真正的妓者，她在路边兜客你也不敢骂她是卖的，不信你试试，绝对给你一个响亮的大B斗。所以痴呆老妓也是绝对不可辱的，没有捅到他的痛处，他怎么可能跳出来歇斯底里地撒泼呢？我就纳闷，他凭什么说网友们是吃了春药抠了脚趾头呢？是不是他有这个怪癖，一边抠着脚泥一边擦着印度神油在报社上班？不知不觉中他将自己的英雄形象暴露无遗了，在众多老妓中他还真有点独树一帜嘞！

　　按照中国老妓们的逻辑推理，我们大致可以演绎出许多版本。其中最著名的是：1、他边抠脚泥边抽烟，等于自己吃自己，所以报纸有点馊味。2、老板是不是个女的？如果不是，他擦神油干什么？难道性倾向错位？如果是，会不会排练一出中国版的《叛逆性骚扰》？

　　被女老板按在办公桌上一顿海扁，可惜电话没摘开，他没有道格拉斯那么幸运，只能拉起衣衫暗自垂泪……

　　哈哈哈，别以为你们老妓会胡说八道，谁要是不要脸不要皮，谁脑子里缺零件啊？

　　罢了，总纠缠着老妓不放，难道咱欲图谋不轨？没兴趣！没性趣！

　　子弹留给年轻点的不行吗？用一句古代笑话的话作结尾：给他二两银子，让他糊涂一辈子。
