足坛流行大结巴


　　公元2000年4月7日普通得不能再普通，但由于国际足联老大布拉特先生光临央视《五环夜话》而变得不普通。老早我就泡杯浓茶，叼上一根中等雪茄（其实是叶子烟），正襟危坐在电视机前，准备好好聆听布老大对中国足球的训斥与批评。不料一场好戏被个一直没露面的结巴翻译彻底搅和了。

　　“这个这个这个……足球，对，我说的这个这个这个中国足球，主要在于这个这个足球……不！是足球协会的这个这个政治领导人的……这个这个这个全面调度……的……结局……”

　　我真他妈纳闷！诺大一个中央电视台就找不到一个说话起码顺溜的翻译？

　　这真像面对一桌佳肴，刚想拿筷子，旁边突然有人擤大鼻涕。

　　难道足坛已经拥有一个超级大结巴南勇还不嫌够吗？

　　联想到五频道体育新闻几乎已是幻灯片，国际足坛上演的好戏简直就是精美图片展播，我终于明白了，我们国家穷，穷得连足球场上的著名球星都懒得动，穷得连说话顺溜的翻译也请不动。

　　但黄健翔他老人家教育我们说，我国的球迷是世界上最幸福的球迷，可以看意甲、德甲、西甲、英超，而外国球迷是要往电视机里不断地丢硬币才能听到终场的哨声。南方的城市观看ESPN和香港卫视体育台已有多年的历史，难道有外国球迷帮我们不断地丢硬币不成？

　　穷，不怕！怕的就是煮在锅里的鸭子——肉都烂了，嘴还挺硬。

　　翻译有点结巴没关系，光打字幕行不行？拜托了央视，中国足球已经污染了我们的眼睛，就饶了我们的耳朵吧
