第八个是铜像

　　当黎兵对昔日的旧主辽宁队一剑封喉时，满脸汗涔涔的马明宇无限深情地盯着黎兵说："以前你是我的偶像，现在你是我的神像。"

　　但愿马儿这句话没冒靶（出格），因为过去的几个赛季，黎兵都是四川（乃至贵州）球迷心中的神像。然而，每次在他风光无限屡屡进球时，他都会将他的第八个进球塑成容易生锈的铜像，供家乡球迷排队瞻仰。

　　职业联赛的6个赛季，他的单赛季进球从来没超过八个。"他娃来四"曾经愤怒地向成都媒体抱怨："作为一个正选前锋，全年只有几粒入球，这样的队获得第三名，难道你们还不满足吗？"

　　受了强烈刺激的黎兵精神有点"失常"，高矮扭到（非要）离开全兴，准备到外面的世界"巡回演出"。

　　四川有句踏谑没有"自知之明"的顺口溜："扭到闹，不扯票。"没扯票的黎兵果然在德乙颗粒无收，心急火燎地飞回四川追赶大部队来了。本事学到没有暂且不论，反正德乙的三个月镀金够他今后当教练时慢慢享用。

　　黎兵不是没风光过，哪个赛季开始不是带给四川人民无限的遐想（瞎想！）？甚至他还可以在日本队门前扮演神出鬼没的鱼雷快艇，乐得红着脸的霍顿像蟠桃会上的悟空。然而，这种靠"勤奋"吃饭的前锋，在大多数时间里除了偶尔头球摆渡，剩下的时间就是充当对方解围的后卫。谁也不会忘记他精彩的单刀直入：假动作做得太逼真了，高速行进中，皮球突然定在原地不动，晃得人家后卫跟着他往前狂奔数米，蓦然回首，他脚下没球。一次次无功而返，一次次脚底拌蒜，一次次盲目乱冲……四川球迷除了失望就是绝望。

　　黎兵头上的德乙光环掩盖不住他的平庸，厦门的后卫轻易地将他收拾了。在凶悍的石勇高玉勇面前，黎兵萎缩了，猎豹姚夏也不见了踪影，有点悚。（1995年全兴主场对天津三星，如日中天的姚夏被津门虎咬断了脆弱的锁骨，"罪魁祸首"就是现在厦新队队长高玉勇。大迟真会对症下药，真"了解中国国情"。）

　　第八个真是生锈的铜像吗？黎兵准备打破这个可怕的宿命，争取焕发第三春第四春，勇夺这个赛季金靴奖。（他到底还能发育几次？）

　　周日全兴对阵比厦门还要凶悍的沈阳，黎兵又开始磨刀霍霍，准备再次雄起了。四川球迷又开始睁大渴盼的眼睛……

　　黎兵，但愿你这次没有冒皮皮！（吹牛）但愿这个赛季结束的时候，超级大耳屎（耳光）铲得我自己耳聋。
