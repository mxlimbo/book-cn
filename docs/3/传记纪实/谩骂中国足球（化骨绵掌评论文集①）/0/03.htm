米卢，离那帮傻B远点！


　　现在的中国足坛，谁的脸皮最厚最有韧性？除了足协那几个不倒翁，除了俱乐部张嘴说瞎话的老板，就是那帮没事偷着乐的足球老记。说中国足球是亚洲二流，他们怒发冲冠，引证论据。那就将就是一流吧，可你听听中日之战的转播，如果你"刚刚打开电梯"不看画面听声音，绝对是中国队正被巴西队生扒活擒。

　　鲁迅早就说过，中国人的奴性在骨子里。你看那解说，整个一个保定府的"老子在城里吃馆子都不花钱，吃你几个烂西瓜？"

　　你仔细冷静客观地一琢磨，中国足球全被这帮"戴着眼镜的胖翻译"折腾来折腾去。混淆黑白，颠倒是非，是好是坏全凭他们两片回锅肉嘴皮。现在又瞄上米卢了，逼人家讲阵型，逼人家许诺带中国队的成绩,并声称与米卢的蜜月期已过，准备随时进行反击，俨然一个性压抑的小妾心理。人家在黑板上讲，你丫听的懂吗？就算你听懂了又能怎样？哦，非得你先懂，才能证明米卢真牛B？不然的话，就骂人家米卢在尊敬的记者面前空话连天，只会哼美式饶舌小曲。

　　就算你去过英国西班牙德国法国南斯拉夫意大利，就算你采访过卡佩罗佐夫维纳布尔斯扎切罗尼，就算你是中国如雷贯耳的"名妓"，难道米卢有义务非要在你面前在黑板上画来画去？米卢，闭上你那张曾经教训哥斯达黎加尼日利亚墨西哥的嘴，你眼前的这帮人可不是好惹的。没准一起哄，就把您踢回老家蹿稀去。

　　远离这帮傻B，别以为他们假模假样坐在那儿记笔记，他们脑子里转悠的是吃铁丝尿筛子――B里现编明天报纸的标题！别以为他们脑子挺好使的，看看甲A电视转播，任何中国球迷都心知肚明，那水平像是一帮智商高的人干出来的吗？钢琴上绑块骨头――狗都会弹琴。

　　米卢，你老跟臭棋篓子较劲，那你离臭棋篓子也就不远了。
