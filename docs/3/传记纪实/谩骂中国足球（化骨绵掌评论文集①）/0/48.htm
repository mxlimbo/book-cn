中国足球歪歪时报（第四辑）

　　 ●周扒皮说：给我打长工就要按照我的习惯生活，不能吃着我的干粮还自行其事。受此地主的启发，有人起床屙早尿时发现一个黑影在米卢的门前学公鸡叫，随后又听到阎世铎在门外大喊一声：“出工了――！”

　　●深圳火车站的春运工作本来计划得很好，出外打工的民工排着长队有条不紊地踏上归乡之路。不料近日火车站秩序大乱，更多的“民工”涌向站台，除了加开了30对凌晨发车的闷罐车外，还号召民工尽量坐飞机回家。站长一筹莫展，气愤地说：“有些人不自觉，到深圳开会后非要坐火车回家，还打扮成民工模样混在其中，使我们的客流量大增。”

　　●柬埔寨近日说，由于资金短缺，尚有上百万颗地雷没有排除，请求国际援助，尽快消除给柬埔寨人民带来的隐患。有人推荐中国最好的工兵李铁前去指挥排雷，他在输给伊朗队四个球被换下场时正好听到这个振奋人心的好消息，于是李铁开心地笑了。

　　●宿茂臻不止一次在赛前宣布：别看重比赛的结果，应该注重比赛的过程。于是大批球迷在比赛的过程中踮着脚尖苦苦寻觅宿茂臻的身影，结果大失所望，除了他有一个一秒钟快哭的镜头外，大部分时间他都蹲在地下系鞋带。

　　●谁都知道我国人民都还很贫穷，而阎世铎提倡的“人民足球”是不是今后只要穷人踢球？听到这个消息，身为百万富翁的国脚们像泄了气的皮球，接连输给埃及和伊朗。他们不禁大声诘问：“不是允许一部分人先富起来吗？让我们回到人民中间去，吃二遍苦，受二茬罪，我们坚决不答应！”

　　●央视和全国有线联盟为争夺“文明杯”的转播权打得头破血流，马国力提醒刘建宏出门的时候注意点，免得被有线电视的电缆线拌一个“老头钻被窝”。

　　●一家高级饭店为了拍张柏芝的马屁，准备从200多间高级套房中精选一间命名为“张柏芝房”，并让这个当红玉女终身免费享用。有人发现那个写有“张柏芝房”的木牌子背后有四个字没涂抹干净，仔细辨认仿佛是“李铁开房”。

　　●听了某领导的谆谆教诲后，外籍教练个个茅塞顿开，纷纷摒弃宗教信仰，永远不过圣诞节，并连夜写了入党申请书。有几个个性过于张扬的外籍教练还赶到云南红塔找戚务生，向他讨教“一个教练怎样在指挥席上稳如泰山”，戚务生当场向他们表演了“驴脸拉长术”。

　　●有不知深浅的记者问阎世铎，为什么他的嗓门这么洪亮？阎幽默地回答：那是当年下乡当小队长时连成的。有武林高手一语点破天机，阎的嘴唇不动，声音却很洪亮，刺人耳膜，他用的是在武林失传多年的“腹语”。
