女排中脸皮最厚的教练


　　老女排朱玲说："我有点看不懂。"其实全国人民都看不懂，上届奥运的亚军在1：3负于克罗地亚后，获得小组前两名的可能性已经很小，进入本届奥运的前四名更像是胡进一个人在梦想成真。他在梦里水乡划着船儿继续给我们勾画着女排绚丽的蓝图："希望通过总结，中国女排可以在1/4决赛中打好。"

　　胡进所说的打好1/4决赛，指的是与俄罗斯或者古巴队的比赛。"经验丰富"的胡进大概认为俄罗斯和古巴相当于越南或缅甸，明摆着是"梦"，偏有痴人大言不惭地去说，不知道他是安慰我们呢，还是纯粹走夜路吹口哨自己给自己壮胆？"人贵有自知之明"，我开始怀疑这句话的真实性了。因为像胡进这种有"前科"的教练正脸不变色心不跳地重复着8年前的拙劣表演，而他从来没有拿着镜子照照自己脸上有没有"黔驴技穷"这四个字。他的指挥他的战术都没错，错的是"球员过于紧张，有机会但没抓住。"同样的话8年前他就赏给中国女排了，在那届奥运会上，女排"意外"输给荷兰，然后树倒猢狲散，谁都敢输，结果仅仅比第八名好点。（基本上）同样是这些队员，在郎平的率领下马上突飞猛进，让久旱的女排尝了尝奥运亚军的鲜。可"英勇"挑起大梁的胡进卷土重来后，本来希望获得世界杯第三名，直接取得奥运入场券。结果"机会没抓住"，输给日本和巴西，被迫"屈尊"参加今年6月的奥运资格赛。这可是自从中国女排获得世界冠军以后20年，从来没光顾过的"破"比赛。话虽丑了点，但胡进的表演更丑。梦中想获第一，梦醒时分才知是第十二，并且在不久前举行的世界女排大奖赛上，胡进率领的这只"胡排"又壮烈地牺牲在韩国人面前。

　　奇怪的是，这一系列故事正在有条不紊地延续着，好像谁也无法阻挡"胡排"的精彩表演。看来，这只"胡排"背后的故事更加曲折精彩，其中充满了姑娘们的艰辛和泪水，也充满了幕后见不得人的狡诈和欲念。

　　眼前的这一幕幕不会让胡进的脸色变浅，反而像一层层坚硬的牛皮遮住了某些人曾经惊慌的那根神经。在这张牛皮后面，胡进已经荣升为脸皮最厚的女排教练，他准备扮演一次螳螂，撼一撼古巴这颗大树。连"人贵有自知之明"都不相信的人，有可能再让一句古老谚语作废。

　　但，不知道又一次"没抓住机会"过后，有谁能站出来为我们曾经辉煌的女排负一次小小的责任？绝对没有！！！等着看吧，脸皮厚的远远不止胡进一个人。
