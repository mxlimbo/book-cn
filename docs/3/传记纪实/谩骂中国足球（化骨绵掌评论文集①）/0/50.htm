我比较我怕谁

　　我趴在战壕里，鼻子呛满硝烟。我问队友，是否听到冲锋号令？队友回答："事业"与"产业"都开始刺刀见红了。我"悄然"一瞥，战斗已近尾声，只看见两个散落在尘嚣中的弹壳，收藏弹壳是我的任务。

　　我们知道，这次的"战斗目标"基本上都是在"假设"状态下展开的，比如第一轮："应否在联赛中为向国外出口队员的俱乐部增加积分进行鼓励？"以及这一轮"大批中国球员去国外俱乐部踢球对中国足球是利大还是弊大？"加分了吗？没加。大批中国球员去海外了吗？没去。

　　只有一个人相信它是真的，弹壳上刻着他的名字――老枪。

　　但他偏要把它作为事实加以论述，什么"按照目前中国球员的素质来说，大批球员出国踢球明显不现实。……大批中国球员出国的想法无疑是空想"。他的意思是：既然大批球员去海外踢球是不可能的，所以我们的辩论是浪费唾沫。

　　既然是浪费唾沫，看来他是不想按照"战斗"规则继续参战了。因为按照他的意思，既然没有什么大批球员去海外俱乐部踢球，还辩论干什么？不如哪儿凉快哪儿歇着去！等到中国球员真正在海外"百花齐放"的时候，老枪说不定就变成红缨枪燃放狼烟报警了。如此说来，老枪的枪筒放出来的3000颗子弹等于臭弹。

　　其实老枪的"臭弹"远没有悄然一刀的"比较足球"来得帅，"帅"得全体人民目瞪口呆："98年世界杯11名巴西天才完败法兰西可是不争的事实；亚洲杯上代伊、马达维基亚等始终未能挽回伊朗落败的命运，而反观日本，没有中田英寿的支援却依然登顶。"他借这些失败的战例说明"整体足球"胜过球星在队中的作用，从而说明即使大批中国球员到海外踢球也对中国足球裨益不多。

　　他生拉硬扯的横向"比较"完全不顾足球比赛的偶然规律，也不管法兰西是不是世界强队，仅以一次比赛的结果以偏概全。其实这种弱智的"比较"是害了许多土产大腕的，我们熟悉得很。

　　苏永舜曾酸溜溜地埋怨道："福格茨输成那样，人家也没喊他立即下课。"他跟福格茨比较，不顾人家带国家队打100多场国际比赛只输了寥寥几场球，人家给球迷带来的快乐多于痛苦。

　　戚务生也愁眉苦脸地说："人家外国球迷有素养，真正是胜也爱你败也爱你。"他忘了人家俱乐部过去曾给球迷带来的辉煌荣耀，以及生命不息战斗不止的拚搏精神。你去沙特前先下软蛋，只会在金州被伊朗连灌4球后站在场边跳"绕指Disco"，你渴望球迷爱你什么？

　　这种硬拿别人的例子横向"比较"中国足球的笑话，现在被悄然一刀和老枪偷窃了，并留下诸多笑柄："并不是只有到国外俱乐部才能培养出杰出球员，其实齐达内、舍甫琴科、皮雷斯以及现在的里克尔梅等人都是在国内联赛中扬名才得以被国外俱乐部发现，并进一步在世界足球舞台上证明自己的实力的。"他的意思是，我国的联赛跟欧洲的联赛等同，我们也能造出齐达内、舍甫琴科等球星，等他们成名后被国外俱乐部发现，再让他们去海外扬威。闹了半天还是要送出去，只是送出去的时间需要商榷。

　　"输出的球员去高水平联赛磨练能不能得到提高还得视球员际遇而定，黎兵去德乙溜了一圈回来后又给四川全兴带来多少质变？"他认为失败的黎兵就是中国足球的全部，那杨晨呢？他在亚洲杯给国家队带来的震动又说明什么？

　　况且我方从未说把"大批中国球员送到海外"就是培养11个天才独往独来，而正是要他们到欧洲锻炼在贫瘠的土壤里无法栽培的足球理念。杨晨过去只会低着头系鞋带，金志扬说："你什么时候抬头什么时候就是球星了。"所谓"抬头"正是说一个球员的大局观。亚洲杯"孤独的"杨晨已经充分证明，在欧洲熏陶后的他已走在其他队友的前面了，而且不止一个档次。而我们屡战屡败的主要症结不正是球员缺乏这种大局观吗？送他们到国外不就是学习这些吗？难道去学习后脑勺颠球？"悄然"认为，大批球员出国深造就是塑造球星，而不是"整体足球"，并拿法国和巴西的决赛作为佐证。照这种荒谬理论，既然11个天才踢不过法兰西的"整体"，那我们干脆在小黑屋里关门苦练，只要嘴上念着悄然一刀的所谓"整体"，管他什么Batigol，管他什么卡洛斯的魔鬼任意球，管他什么苏克左脚能拉小提琴，能绣花也不灵。

　　更可笑的是，"悄然"在文章最后有一句经典："我方从不否认球员到国外高水平联赛上去磨练能使自身得到提高或者成功，但是一花独放不是春，百花齐放才能春光满园。"我差点揉破我的双眼，他竟然赞同我方的的论点，只是反对"一花独放"，而是要"百花齐放才能春光满园"，也就是说，他代表兵马俑网站集体投诚了。看看我们辩论的题目：大批中国球员去国外俱乐部踢球对中国足球是利大还是弊大？注意这个"大批"，大批意味着"百花齐放"，而不是"一花独放"。

　　既然"小批"的中国球员出去不行，那就"大批"的球员出去吧！辩来辩去，闹了半天他居然站在我们这边来了。他排列论据，抛出可笑的"比较"，最终说的就是这个"百花齐放"。既然"百花齐放"可以"春光满园"，我们还需要浪费口水辩论个你死我活吗？

　　真是辩论大赛一大奇观，比较来比较去，他自己"悄然"给了自己"一刀"，我们等着接血吧。
