我气，为女排教练

　　如果说早上为赵颖慧遗憾，中午为王义夫捏把汗的话，那晚上纯粹要被女排气出汗。看得出来，姑娘们尽力了，汗水也流够了。白岩松说，实力说明一切，这厮说在点子上了。打不过美国不要紧，要紧的是女排教练胡进都干了些什么？一个教练能把优秀的孙？调教成孙悦，能把何琪训练成两眼只知道2号位，能把33岁的巫丹当成救世主从遥远的欧洲拽回来，你还能指望他干什么？

　　女排的表现确实差强人意，强攻没劲儿，拦网抬不起手腕，战术单调得连看转播的老太婆都能看出板眼儿。这帮姑娘给谁训傻了吗？胡进说，咱们的身高不占优势，他忘了当年郎平的身高也抵不上海曼。

　　郎平的突然卸任本身就笼罩了一层神秘色彩，她的压力太大了，身体太糟了，虚弱得马上执教一支意大利职业队，气喘吁吁地获得冠军，得到了意大利人的拥戴。此时，胡进俨然女排理所当然的人选第二次浮出了水面，而张蓉芳也马上宣布避嫌。这好像是一部情节跌宕起伏的故事片，在观众最需要正义的时候，英雄人物总会吹着冒烟的枪管及时出现。你要是把女排训练成"铁姑娘连"，把古巴、俄罗斯、巴西给灭了，你就是开个"女排夫妻店"又如何？避什么嫌？此地无银啊？可惜权力和利益的争夺蒙住了我们的双眼，我们只能把希望或者说撞大运寄托在那颗聪颖的头颅里面。磕磕碰碰好不容易争到一张去悉尼的船票，拚搏去了还是丢人现眼？

　　胡进早该是没招的人了，他恨不得把张蓉芳招进队里，捎带着通知郎平，祖国需要你这样有经验有实力的铁榔头。他完全忘了，奥林匹克属于年轻的一代。我们宁愿看到赵颖慧的泪水，也不愿听见40好几的王义夫说："取得这个成绩我已经满足了。"我们宁愿看到年轻人稚嫩的双臂在网上折断，也不愿看到年迈的巫丹苦苦挣扎只为了一张不值钱的脸。

　　牺牲体育的本质精神，只为廉价的奖牌，大概这就是当年北京输给悉尼的原因所在。两票，仅仅两票，够我们几十年追赶。
