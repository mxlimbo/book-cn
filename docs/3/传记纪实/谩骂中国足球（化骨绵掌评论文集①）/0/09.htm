大戚，今年突然可爱


　　有张照片拍得牛B：米卢伸手抓大戚头上的运动帽，大戚双手捂着，绽出憨厚顽皮的笑靥。照片的标题是：我左看右看上看下看，你这个大戚实在是不简单。

　　我真有点嫉妒米卢，凭什么你一逗他，他就轻易地咧开大嘴笑了呢？是因为你威震世界足坛的名气，还是你说了什么南斯拉夫的谚语？要知道，10强赛的时候，我们上亿的中国球迷挠他夹肢窝他都没吭一声的。最多在金州体育场进了伊朗队两球后，大戚高扬右臂，表演他独创的手指划圈霹雳。

　　今年的大戚突然可爱了，也许是大彻大悟后的觉醒，也许是看淡人生后的自嘲，也许是不在其位的幸灾乐祸，也许是日进斗金后的惬意，所以他才会轻易地发出诸如"可怕的是洋教练一个接一个地倒下"的惊世叹息。

　　远离了如坐针毡的国足帅位，大戚有理由可爱：无条件支持他的俱乐部，无条件崇拜他的云南球迷，盘踞着令各甲A球队闻风丧胆的高原，指挥着一群死缠烂打的愣头青，只用两个黑哥们儿在对方阵地冷不丁儿丢两颗香瓜手雷，"

　　轰隆"一声，不炸你个人仰马翻，也得把你的脸熏得黑黢黢。大戚如果头裹白毛巾，手握红缨枪，绝对是高庄赵庄马家合子一带的八路军。

　　谁也不敢小看他，包括那些瓦尔特们，否则让你见识见识什么是自卫反击。

　　现在的大戚才是最真实的，过去我们熟悉的那张愁眉苦脸完全是一张无奈的面具。我们曾经幼稚地憎恨它唾弃它，恨不得撒上一泡骚尿才解气。现在看来，大戚顶多是足协性无能的牺牲品。扭曲了人格的大戚像个受气的丑媳妇，

　　被迫在新闻发布会上抹泪打嗝擤鼻涕。返朴归真的大戚笑声朗朗，妙语连珠，恨不得跟赵本山联手演个喜剧小品。

　　难道人格真的可以这样分裂吗？难道中国足坛恶劣的天气可以这样随意戏谑一个人的本性吗？答案绝对是肯定的。陈亦明贾秀全章健……们，谁不是在这种恶劣的天气里被迫上演着人生的喜剧？

　　可惜这些人的才气，掉进中国足坛这个大染缸里，除了平庸，就是让人看了恶心。
