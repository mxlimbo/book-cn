一半是男人，一半是女人


　　有社会主义特色的中国足球是什么？是大炼钢铁超英美要多大胆有多大胆吹牛不犯死罪？仰或是死皮赖脸抱着"计划"嘴里喊着"接轨"的野交后的怪胎？真要是这样倒也罢了，最起码是一场轰轰烈烈燃烧过透明过永远不后悔。而映入我们眼帘的什么都不是（怪胎还吓你一跳呢），是一群吃饱了撑的打着饱嗝的姑娘们在慢条斯理地玩"高俅"，脚法相当花哨令我眼花缭乱。

　　呔！看了中国队在亚洲杯上的精彩表演我已经兴奋地"呕吐"了三次。谁说"小姐"没地位？呸！那是万恶的旧社会。现在这群比"小姐"还母兮兮的五尺女儿们正在黎巴嫩舞绣花针。你看那传球，细腻得让你差点流鼻血；你看那倒脚，含蓄得让你血管爆裂。什么欲望都让"她们"给挑起来了，但找不到卸货的地儿也只能单手扶南墙喝着西北风打几声干哕。一帮老爷们儿楞婀娜多姿楞扭捏作态楞靠电线杆，这"特色"可真"特别色"的。什么"战术要求"，什么"赛会制特点"，骗鬼啊？我们懂，你们是装成大姑娘引诱人家自乱方寸。你们要真是中国女足倒好了，起码你们具有中国女人的基本欲望吧？现在扮成浓眉大眼的基佬猛抛媚眼，我们还真有点不习惯，毕竟这种风俗趣向只掌握在少数人手里。不过总比太监好，太监基本没什么念头，你们还起码有"被动"的饥渴，这点让我们特别佩服。

　　本想年轻一代的爷们儿肯定有点血性，哪想到他在那儿放一蔫的：我们在甲A领先对手后就这么踢的。呜呼！我以为他有好大的脾气，最起码方刚一盘，结果"酱缸"里一染，特聪明功利起来。我们还有什么奢望？年老的年轻的都这个模样，一半是男人，一半是女人，枉自生就魁梧身材，一到足球场就变成腼腆的大闺女。你以为他们能阳刚几分钟吗？他们恨不得在后场慢吞吞地扭腰扭胯练劈叉。你再看看日本韩国伊朗，那叫猛烈！好男儿不在输赢，在乎血染疆场。好端端的改革大好形势，楞叫一群半男半女的人妖给糟踏了。你在黎巴嫩丢脸，我们能把你咬一口不成？如此说来，"几百年来任人宰割"这句话就好解释了，不然人家怎么不宰割别人？

　　如果说眼前这一切都是米卢调教的结果，那他老人家真是天下第一变性大师。如果中国和科威特真默契，那我们只能把这种默契当狗屎吃了。如果那帮大爷骨子里就是一挨打的揍行，那就让他们撅着屁股偷着乐吧。

　　失去了拚搏精神的足球，失去了血性色彩的人妖游戏。除了鼓掌喝彩，我还不是只能瞅着夜空数星星脱了裤子骂三声天？继续表演！！！化装成男人的娘们儿们。
