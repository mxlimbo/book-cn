眼睁睁地看着你却无能为力

　　下午，我正伏在案上翻阅猴年马月出版的《马津法律小辞典》，看有没有第一届中国足球网辩赛的确切定义，忽然楼下人声鼎沸：出事了！有个伊拉克大胡子在大街上游行，高喊"赌球博彩，贻害万年！"旁边还有一个人插着黄色翅膀，高声倡议"拥护嫖娼，马上开房！"最严重的是一个啃着黄元帅的，他倒没喊什么铿锵口号，只是悄声嘀咕"干什么不好？比如开发绿色球帽……"

　　他们不是参加网辩的选手吗？快把他们拽回来，免得在街上丢人现眼。有人不屑地盯着我，什么鸟网辩？黄色绿色，支援国家建设。

　　我哑口无言，眼睁睁地看着他们越走越远，只剩下温暖的背影供人仰瞻。我好想加入革命队伍啊，因为我也曾一直为"赌博""彩票""博彩""赌球"这些名词困惑，也曾像洗牌一样将他们混为一谈。如果用输赢的概率来解释这些名词的不同，百家乐（Baccarat）是最明显的例子了，押在庄家和闲家都是50%。但是别忘了你可以押在和家，一晚上也出现不了几次，如果押中了就是可观的1赔8。这跟彩票很相像，"2元钱圆你百万梦"，只不过彩票获胜的奖金更多、希望更渺茫而已。那我一直押在和家，就不算参与赌博吧？如果要用"技巧"二字来解释，赌博是纯粹赌运，而足球彩票是综合足球方面的知识，这更是自欺欺人。真要那样，黄健翔就要马上发财了，他每周运用大量的足球知识在电视屏幕前唠叨什么？买足彩不就行了？况且现在流行的体彩和福彩要什么知识？撞大运而已。再说赌博也需要技巧和知识啊！比如"出千"，"逢赌必有假"，这是赌业的真理。如果用利润的走向来解释，一个私饱中囊，一个"为了体育事业"，但二者的性质并没有改变啊？只不过一个是私开赌档，一个是政府坐庄。对了，二者的不同正在于此。

　　有时你不服气真不行。一切法律都是为了统治阶级服务的，如果足彩不需要任何法律解释，早出台了。既然跟现行法律有饽，他必然慢慢"酝酿""论证""舆论准备"，然后出台一个"满意"的解释，好让你在他规定的范围内"娱乐"。"巨大的商机"促使他必须抢占这个地盘，什么"赔率不同""用处不同""方式不同"全是真实的谎言。试想，如果你私自开一个足彩，赔率返还率跟国家的一模一样，利润也全交给可爱的体育事业，权当做一个彩票雷锋，行吗？保证不行！因为他不能控制，其中的"色彩"应该昭然了吧。

　　有人绝对诘问，"法律解释"岂是随便捏的面筋？睁眼看看你周围的一切，什么都想通了。

　　我不想在"法律"的边缘越走越远，不想再涉及敏感话题，因为那是网辩所不允许的，我只想说说我们共同面临的辩题。足彩马上就要发行了，这是一个不争的事实，如果再去苦苦纠缠它和赌博的区别它的法律界定，无疑要笑掉体彩中心的大板牙，我们只能在"大前提"下讨论它对中国足球的利弊。

　　比如它可以筹集更多的体育资金：有人说中国足球不缺钱，可以SB似的反复化大价钱挑选外援，可以让歌舞团去前线劳军，可以很豪华地到海外集训……冬季到欧洲去抠女。但是别忘了我们可怜的青少年和女足，以及惨不忍睹破烂不堪的"菜地"，这些资金如果真要用在这方面，可以大大缓解和改善这些窘境的状况。当然，如果有几个硕鼠截留这笔资金投奔"宋江"则另当别论。

　　还有，它可以起到"堵不如疏"的功能：可以肯定的是，可以分流一批参与地下赌球的人数，从而达到降低"黑箱操作"的可能性。既然博一点小运是人类的"天性"，与其让你笑，不如我用较正规的手段控制你的笑，这也是一切"政府行为"的最终目的。因为赌博的输赢落差比较大，所以容易引起社会问题，于是有人用极端的例子来证明这种害处，这是对的。但足彩的大奖得奖率微乎其微，所以人们的心态也不容易失去平衡，出现社会问题的概率也就比较低。说这话的人你会不会因为化了2元钱没有赢得500万而去抢银行呢？当然这个社会什么人都有，就像明明知道杀人放火是犯罪而偏偏有人以身试法一样，如果没得大奖而去跳楼，没人接着你。有"爱好者"看到这儿一定按捺不住了，声嘶力竭地高喊：既然疏导这些人，那是不是马上开嫖了？慢！这个建议不新鲜，讨论很多年了，估计没什么成功的迹象，等有了好消息，自然会通知你。

　　我不想再纠缠什么"博彩+赌球=赌博"之类的跑题，因为它不是本次辨题所要讨论的，非要憋红了脸咬着什么玩艺沿着"博彩=赌博=违法=弊大"这条小道走下去，我实在没时间奉陪了。因为你在没有搞懂什么是"法律解释"的时候非要自己立一条"法律"，那你这辈子可忙了，想不通的事多如牛毛。

　　这不！游行的人又转回来了，唱着"悲伤的人满街游走，打听幸福的下落"。

　　那就慢慢打听吧，累不累？
