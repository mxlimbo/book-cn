北京国安，猛药献给你！

　　看到南方薛申杨璞谢朝阳小王涛……们在绿茵场上玩命的样子，即使不赢球，也没有谁能阻挡他们拿一回甲A冠军（倒数也行）。可联赛刚过几轮，有些人就沉不住气了，这难道像是国安某领导所讲的具有"王者风范"的御林军吗？倒有点像临时拼凑的雁翎队。

　　总病名：急躁偏激中风，另附带几个疑难杂症，有几个凶猛偏方献给领导：

　　一、下课忧虑症：您现在忧心忡忡寝食难安的问题是，如果您下课了，谁来上课？快别说笑话！全国人民都不相信，诺大一个首都，难道只有"大陆100家大上市公司排名第二十位"的国安有搞足球的能力？您真要下课了，自然有人接着啃这块香饽饽。不是说，中国足球缺了谁都行吗？偏方是：您找个荫凉地儿歇着去，省着生闷气。

　　二、文革遗风症：为了维护改革开放的成果，为了社会治安的稳定，别挡道儿！让乔里奇同志先走！乔老爷做梦都没想到，不知不觉中他差点成了阻碍人家安定团结的间谍。足球事小，申奥事大。您的觉悟确实比较高，能把足球提高到祸国殃民的地步。难道北京球迷是李洪志的残渣余孽？难道北京的警察揣手不管？药引子是：阶级斗争一抓就灵。

　　三、说了白说症：北京国安在您的领导下，获得一次超霸杯，两次足协杯冠军，成绩确实斐然，令人啧啧称奇。可北京队本来就是传统强队，它起码应该得一次职业联赛以来的甲A冠军。现在的国安队就缺一个优秀的中场，何止北京国安？全中国都缺这样的天才。中国队要是有个齐达内德约卡夫之类的，早到世界杯转悠去了。偏方是：继续梦里水乡。

　　四、银两满仓症：号称中国足坛最有财力的俱乐部，那跟大连实德争什么小王涛啊？让乔老爷挑什么外援啊？拿钱可着劲儿买不就完了？当红球星不愿意来，过气球星总还凑合吧！人家日本的俱乐部还看得上贝贝托呢！体能不过关没关系，让足协废了不就行了。这次的偏方比较长：把中国国家队都给买过来，三个外援名额就凑合着买齐达内贝克汉姆巴蒂算了。中国足协不同意，就用钱一锅烩。

　　五、见谁灭谁症：都说足球比赛的结果最难测，但您最会测。既然可以随便赢国安卖出去的老队员，而这些老队员基本上还在甲A甲B的赛场上，那北京国安还不见谁灭谁？偏方是：举行"北京杯"足球赛，让新老北京球员亮开膀子掐一场，看看您预测的准不准。

　　总之，心气过高的您确实身患重病，连说话都有点神志不清。作为北京国安的忠实球迷，我们看在眼里，急在心上。偏方难听，良药苦口啊！希望您摆正心态，千万别把自己当（北京）人，除去毫无意义的傲气，同仇敌忾，一鼓作气，说不定真能见谁灭谁呢！

　　北京国安，永远真牛B！
