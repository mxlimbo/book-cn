2000欧洲杯半决赛――法葡大战之刘建宏版

　　比赛真是一波三折，中间因为裁判员的一次严重误判，导致了比赛结果的全面更改。从亨利进的扳平的这个球的情况来看，阿内尔卡有处在越位位置上的嫌疑，但是从目前情况来看，由于我们没有全景镜头，还无法判断。如果说单从局部来判断的话，阿内尔卡也许不算越位。但是我们无法判断的是法国队队员在禁区外传球时，阿内尔卡是否已经处在越位位置上，如果那个时候处在越位位置上的话，那现在他显然是从这个位置上获益了，这个进球就值得商磋。

　　比赛进行到加时赛的最后时刻，沙威尔在本方禁区内手球了，这时裁判员果断地判罚了点球！如果这位来自奥地利的裁判本科这时能改判罚角球的话，我想费戈和戈麦斯也不会说出任何一句埋怨的话。本科仍然显得信心十足，也许这时他应该首先想到的便是向助理裁判询问一下。沙威尔是用手挡住了特雷泽盖的射门，但这是个禁区内无意手球，这么短的距离，沙威尔根本无法收回自己的手臂。正是这个判罚，使一场本应是1：1的球变成了1：2，并且葡萄牙队根本没有扳平的机会，因为本届欧洲杯在加时赛采用的是突然死亡法，也就是我们常说的金球致胜法。

　　本来一场精彩的比赛却发生了让人很不高兴的事情。很遗憾的是我们的电视解说员，包括黄健翔这样的足球口形配音演员也没有指出这个球的可疑之处。另外，我想提醒葡萄牙足球界，葡萄牙足球队包括葡萄牙的球迷、媒体注意的是，你们此刻还能否想起6月11日的的意土风波，在那场比赛里面，土耳其的球队遭遇了怎样的待遇，如果你们能够想起那场比赛的话，那么你们是否有能力或者说有这样一种胆量面对今天所发生的事实。我想，当然，也许，大概，对欧洲足球界来说，发展到今天，确实需要一种非常正常的环境来发展欧洲足球，如果大家都来破坏的话，那我想欧洲足球是很难有出头之日。连我们遥远的中国足球都迅速地追了上来，我们的王景东就敢不判那个点球，而且我们的专家也一致认定那个球是无意手球。我想今天这场比赛又一次向全世界的裁判界，尤其向欧洲的足球界亮出了黄牌，希望欧足联能够警觉。

　　另外我想说的是，本届欧洲杯的环境太不正常，在这样的环境里大家都免不了成为被伤害者。提到意土之战，是想请葡萄牙的球迷和媒介设身处地为土耳其球迷和球队想想，你们现在的伤感就是他们昨天的眼泪。也许我在遣词造句上还不够完善，有些意思没有完全表达清楚。对此我深感遗憾。但我丝毫没有指责和批评葡萄牙球迷、足球和媒介的意思。从世界足球的立场出发，如果葡萄牙球迷和我之间有误会，也应该得到澄清，毕竟我们都是为了一个崇高的目标，那就是努力为世界足球的健康发展做出应有的贡献。
