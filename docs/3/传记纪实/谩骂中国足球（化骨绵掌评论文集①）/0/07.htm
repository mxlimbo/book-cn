快乐并痛苦着

　　三个月前，黎兵挎着军用水壶背着干粮出发了，为的是到德国偷偷学点手艺。可惜手艺没学到，只凑合在德乙的后院扫了三个月的地。甲A开赛第四轮，他衣衫褴褛灰头瓦脸地逃了回来，见到魏坦克便说："哥子，我来给你们抽起（帮忙）！"

　　魏坦克唉声叹气地摇着头，暗忖："你黎兵的脑壳皮皮是肿的？三个月能学什么手艺？"

　　但肿着脑壳的黎兵硬是给了重庆隆鑫致命四击。全国哗然！未必他趴着墙头学了什么绝技？

　　我那辽宁籍的女友最不以为然，瘪着宋丹丹似的嘴，说："干哈呀？等俺们辽宁那嘎达人收拾他！"出门打麻将之前还数落我："俺不用看比赛，告诉俺比分就行了。就他那水平，俺们辽宁老鼻子啦！"

　　然而，黎兵用在德国学到的方式给了昔日的东北虎致命一击。看到黎兵快乐的样子，真是"你快乐所以我快乐"。

　　可是我实在快乐不起来。在德乙，黎兵是一颗不起眼的小兵，因为经常打不上主力，所以他说："德国足球竞争非常激烈。"然而他回到我们的甲A却如鱼得水，骁勇异常。难道德乙比我们的甲A还高一竹篾片？

　　黎兵的频频进球正反映出我们和世界足坛的差距，怪不得布老大把我们和尼泊尔越南放在一起相提并论。

　　因为全兴赢球，所以我快乐。因为中国足球还没有找到北，所以我痛苦着。这种连绵不断的苦楚不知道还要延续多少艰辛酸涩的岁月？

　　凌晨1点，女友回来了。我毫不犹豫地告诉她："辽宁队赢了，1：0"

　　这个弥天大谎起码让她快乐一夜，让痛苦尽量拉开与快乐的距离，总比痛苦和快乐紧密相连好一些。

　　辽宁队真的赢了，因为他们还年轻，就像年轻的中国足球――需要"赢"球的积累。
