铁普苏丹


　　铁普苏丹（约1750～1799.5.4），印度迈索尔王国苏丹，抗英民族英雄，海德尔·阿里之子。

　　曾随父参加第第二次英国-迈索尔战争,屡建战功。1780年10月在波利鲁尔打败英军。1782年春攻占坦焦尔，迫使2000英军投降。同年12月其父病逝后掌权,继承其父“驱英出印”事业。1783年5月率军围攻门格洛尔。后因首都塞林伽巴丹受威胁，被迫议和。战后锐意改革，引进西方军事技术，加强以步兵为主的多兵种军队建设，积极备战。1790～1792年在第三次英迈战争中被英军打败，被迫割地赔款。此后继续改革，创建精锐的近卫军,奖励农耕,充实国库。1796年自称苏丹。1799年率军英勇抵抗英军对塞林伽巴丹的进攻，战死疆场。
