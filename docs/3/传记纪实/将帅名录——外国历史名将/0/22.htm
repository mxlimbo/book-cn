阿尔弗雷德


　　阿尔弗雷德（849～899），英格兰韦塞克斯王国国王(871～899)。年轻时受过军事训练。

　　868年从军，随兄艾特尔雷德国王率军支援麦西亚王国抗击丹麦人入侵。871年丹麦人入侵韦塞克斯时即位。在危难之际，他退守山林，重整军队,于876年在埃丁顿之战中击败丹麦军队,迫其议和,撤至伦敦到切斯特以北地区 (史称“丹麦法区”)。 此后，为巩固国防,派兵镇守边关,修建城堡,并积极扩充海军。885年率军击退入侵肯特王国的丹麦军队,并于次年占领伦敦。892～896年再次率军击溃从欧洲大陆进犯的丹麦军队。

　　在位期间，重视文化教育，翻译大批古典名著，并编纂《盎格鲁撒克逊编年史》，影响深远。
