汉穆拉比


　　汉穆拉比（？～约公元前1750），古巴比伦第一王朝第六代国王(前1792～前1750)。

　　即位时，巴比伦仅为依附邻国的两河流域中部一小邦。即位后，对内大力发展经济，扩充军队并给军人分配份地；对外开展灵活外交。经五年治理，国势渐强，开始向外扩张。他采取集中力量、各个击破的战略，先后灭亡伊新、拉尔萨、马里、埃什努那等国。攻打拉尔萨时，除用重兵攻城外，还采用筑坝拦河断其水源或拦水后突然放水发动水攻的战术。经35年征战，结束两河流域诸国割据局面，建立起统一的中央集权奴隶制国家。自称“强大之王，巴比伦之王，阿维鲁的全国之王，苏美尔、阿卡德之王，天下四方之王”。

　　他制定的《汉穆拉比法典》，是迄今已发现的古代奴隶社会第一部比较完整的成文法典和研究古巴比伦社会历史的重要资料。
