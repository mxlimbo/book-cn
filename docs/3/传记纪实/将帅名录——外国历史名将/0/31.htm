瓦伦斯坦


　　瓦伦斯坦（1583.9.24～1634.2.25），神圣罗马帝国统帅。波希米亚(今捷克)贵族家庭出身。

　　1604年加入帝国军队，1606年后升任军官。1615年任皇帝侍从官。三十年战争中，参与镇压1618～1620年的捷克起义。1621年任驻捷克军队统帅。1625年丹麦进军德意志，在帝国处境危难之际出任武装部队总司令，受封公爵。他实行“以战养战”政策，在被征服地区征收特别税，迅速建立一支训练有素、战斗力较强的10万军队。1626年率军在德绍击败与丹麦结盟的新教诸侯军队，并协同天主教联盟军总司令蒂利收复卢特城。1627年春攻入西里西亚，迫使丹麦军队败退,尔后在蒂利支援下将其赶出帝国领土,并挥师日德兰半岛。1628年率部攻占梅克伦堡和波美拉尼亚，强攻斯特拉尔松德要塞，把帝国势力扩大到波罗的海南岸。后因推行归还教产敕令，引起新教诸侯不满,于1630年瑞典派兵入侵之际被解职。1632年4月重任武装部队总司令后，重整军队，在吕岑之战中击毙瑞典国王古斯塔夫二世·阿道夫。1633年同萨克森和瑞典秘密谈判，谋求国内和解和外国撤军，被皇室视有通敌嫌疑。次年1月被解职，2月在埃格尔遇刺身亡。
