克洛维


　　克洛维（466～511），墨洛温王朝创立者,法兰克王国第一代国王。

　　481年其父死后成为萨利克法兰克人部落首领，竭力向高卢扩张。486年在苏瓦松附近击败罗马军队，占领卢瓦尔河以北地区，奠定法兰克王国的基础。496年打败向高卢推进的阿勒曼尼人。为取得罗马教会和高卢罗马人的支持，率3000亲兵皈依罗马派基督教。500年迫使勃艮第人臣服，势力扩展到高卢东南部和莱茵河以东地区。6世纪初率军攻打伊比利亚半岛和高卢南部的西哥特人，于507年在普瓦捷附近打败西哥特军队，占领高卢西南部地区，直至比利牛斯山。晚年为巩固其统治，剪除本部落联盟其他首领和昔日同盟者。曾汇编并颁布《萨利克法典》。
