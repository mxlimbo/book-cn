伊凡四世


　　伊凡四世（1530.8.25～1584. 3.18），俄国第一代沙皇(1533～1584)。莫斯科大公瓦西里三世之子。

　　3岁即位，由母后摄政。1547年加冕亲政,称沙皇。对内实行政治、经济、司法和军事改革,依靠中小贵族限制大贵族势力，加强中央集权，建立“沙皇特辖制”和特辖军，残酷镇压贵族反对派和人民群众，故有“雷帝”之称。对外积极扩张，于1552年、1556年率军东侵,先后征服喀山和阿斯特拉罕两汗国,占领伏尔加河下游地区。为夺取波罗的海出海口, 1558～1583年发动立窝尼亚战争，以失败告终。1563年自称“全西伯利亚皇帝”，开始向西伯利亚扩张。1581年招募惯匪T.叶尔马克组织哥萨克远征军，于翌年占领西伯利亚汗国首都西伯利亚。1584年暴卒。
