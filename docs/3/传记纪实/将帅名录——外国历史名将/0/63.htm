滂尼发秀


　　滂尼发秀（1863. 11.30～1897.5.10），菲律宾独立战争领导人。 生于马尼拉一贫民家庭。

　　因父母早逝，小学时被迫辍学。后通过自学掌握西班牙文和英文，深受法国革命思想影响，研读菲律宾民族运动启蒙者J.黎萨等人的著作。曾参加“共济会”和“菲律宾联盟”。

　　1892年7月创立秘密革命组织“卡蒂普南”(人民儿女最高贵联合会)，广泛吸收下层民众参加，迅速扩大革命力量。1895年4月当选为该组织主席。次年8月在巴林塔瓦克举行武装起义,点燃独立战争烽火；随后领导组建革命军，并率部进攻马尼拉附近的西班牙殖民军，战败后退守吕宋岛北部地区。后与温和派领导人E.阿吉纳尔多发生矛盾，革命阵营分裂。1897年阿吉纳尔多取得革命领导权后,被捕并遭杀害。
