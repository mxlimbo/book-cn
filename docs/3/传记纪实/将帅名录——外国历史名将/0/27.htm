贞德


　　贞德（约1412～1431.5.30）,法兰西女民族英雄。生于法国东北部香槟地区一农民家庭。

　　百年战争中,英军占据法国北部广大地区,并进而围攻通往南方的门户奥尔良城后，她于1429年初谒见太子查理，请求率军解救奥尔良。受命担任前敌军事指挥后，于4月29日率军突入奥尔良城。作战中，她高举战旗，身先士卒，使法军士气大振。5月8日击败英军，解奥尔良之围，被称颂为“奥尔良的女儿”。后率军东进,护送查理前往兰斯加冕，沿途收复数座城市，声望日高，遂遭贵族和廷臣嫉恨。1430年5月率军救援贡比涅时被亲英的勃艮第派军队俘获,并被卖给英军。在敌人酷刑面前坚贞不屈，最后被教会法庭判为“女巫”，在鲁昂广场的火刑柱上就义。
