阿提拉


　　阿提拉（约406～453年），匈奴国王(434～453)。

　　434年与其兄布莱达共同继承王位。约441年率军从多瑙河中游渡河南下，进攻拜占庭帝国，占辛吉杜努姆(今贝尔格莱德)等地。443年进军君士坦丁堡（今伊斯坦布尔），迫拜占庭皇帝狄奥多西二世求和纳贡。约445年杀布莱达,独掌王权。447年侵入巴尔干半岛,迫使狄奥多西二世割让大片领土并继续纳贡。451年西渡莱茵河入侵高卢，6月在卡太隆尼平原之战中为西罗马统帅F.埃提乌斯和西哥特国王提奥多里克一世击退。452年侵入北意大利，占领阿奎莱亚等地。后经教皇利奥一世斡旋，与西罗马议和撤军。

　　他的征伐给欧洲造成严重破坏，杀戮甚众。
