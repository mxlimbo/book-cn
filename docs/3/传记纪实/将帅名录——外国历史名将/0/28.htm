斯特凡大公


　　斯特凡大公（约1435～1504.7.2）,摩尔多瓦大公 (1457～1504)，罗马尼亚民族英雄。又称斯特凡三世。出身皇族。

　　1457年推翻其叔父彼得·阿隆的统治。即位后实行军政改革,发展经济,振兴国防，加强中央集权。1465年从匈牙利人手中夺回摩尔多瓦南部要塞基利亚。1467年打败入侵的匈牙利军队。1475年1月率军4万人在瓦斯卢伊沼泽设伏，击败土耳其12万大军。1476年率军在白谷地区重创再次入侵的10余万土军。后因寡不敌众而失败，被迫于1489年签订和约，向土耳其称臣纳贡。1497年率军在科斯旺森林打败波兰军队。
