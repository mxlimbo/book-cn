福柯生平——福柯其人

  

一个非历史的历史学家
一个反人本主义的人文科学家
一个反结构主义的结构主义者

——记福柯

　　福柯在西方被视为思想知识界的一个斯芬克斯，一个谜样的人物。

　　福柯之谜的产生源于福柯思想的离经叛道和难以归类性,以及福柯个人生活的云谲波诡。

　　神秘恐怖的爱滋病及其与声名狼藉的同性恋的联系，也使福柯的形象变得更扑朔迷离。

　　1984年6月25日中午，一个惊人的消息如闪电般传遍巴黎知识界。电台和电视台宣布：“福柯去世了。”当时，福柯还不到58岁。

　　福柯去世当天晚上，他的姐姐弗兰仙和德菲尔到医院挂号室办理手续时，看到病历上的死亡原因一栏里写着：爱滋病。

　　福柯曾经激动地谈到爱滋病问题。他说，这个世界，权力游戏，真理游戏，这些本身就是危险的。但情况就是如此。这就是你所拥有的。有谁会害怕爱滋病？你明天可能会被汽车撞倒。甚至过马路都是危险的。如果与一个男人的性爱使我感到快乐，为什么要拒绝这种快乐？我们拥有权力，我们不应该放弃。

　　霍尔维茨把福柯送到返回旧金山的地铁车站时，福柯笑着留下最后一句话：“如果我死了，不要为我哭泣。”

　　福柯后来真正意识到爱滋病的威胁。德菲尔说，这种威胁在福柯对希腊罗马思想的研究中起了关键性作用。“很可能，他已经清楚地知道死之将至。但是他不想把死亡变成一场戏剧，而是竭力在每天的日常生活中建立一种与其他人的新型关系。……即便他不能确切地知道自己的境况”，但是他肯定经常考虑爱滋病的威胁。“他很认真地对待爱滋病。当他最后一次去旧金山时，他把这次旅行当作一次极限体验。”

　　在医院里，德菲尔曾对福柯说：“如果你的病被查出是爱滋病，那么你最后的著作就和《恶之花》一样了。”德菲尔这么说，是因为波德莱尔的《恶之花》就是写他自己的性生活和梅毒。福柯笑着回答：“为什么不？”

　　一语成谶。福柯之死夺走了当代最伟大的哲学家。但他对于我国思想界的重要性首先在于他的哲学活动的独创性以及他所开辟的新的知识领域和历史思考领域：疯癫、刑法制度、医学和性欲。

　　福柯死于现代瘟疫，而且他恰恰死在最初用于对付当时的瘟疫——麻疯病的医院。麻疯病和爱滋病先后都被社会视为罪恶之病。麻疯病人和爱滋病患者都被同时代人视为“不可接触的罪人”。福柯以分析人类对待麻疯病人的机制开始“尼采式探索”，以现代瘟疫的牺性者和现代社会的“耻辱者”结束这一探索。

　　也许，在不远的将来，现代医学就能找到攻克现代瘟疫的办法。然而，福柯身上的耻辱烙印会被洗刷掉吗？现代西方社会能够消除它自己制造和划定出来的“反常”、“变态”现象，从而避免福柯式批判话语的衍生吗？

摘自《福柯思想肖像》
刘北成文
