跋二

南无本师释迦牟尼佛　降

八十九年五月二十六日　庚辰年四月二十三日

　　　大道明心着完成　本师降来作跋引

　　　人生修持真实谛　相互共勉相互依

　　由无极大道院来作导引，是懿敕崇心堂着作的一系列大道天书。乃彰显人类有情世界所必须要有的因本，更是对自己能作到，可以上升超越的基础。亦在于能让自己来成就于自己的本灵，即是对自己可以成就、可以学习，更对自己未来的一切过程，皆由学习中来建立基本的观念。

　　人类世界乃是在作改造，不论任何一个宗教派脉之中，皆以「明悟自己的本心、见至自己的本性」来作了别。亦可以对自己能在「明心、见性」之中，来提升自己的智慧。因此正是给予自己不但能成就，更可以对有情众生，是一种真正的向上去增长自己的见闻，在「闻、思、修」之中，来达至「思惟修」，更进入「细思惟修」之中，让自己有成就的资粮与因本。如此来彰显自己的佛性，必会在人类世界可以达至成就，而不必寄望于下世之中再来成就。即是「己立而立人」，甚至要「己达而达人」，是要对自己在成长的过成中，有确确实实的因本源。

　　大道明心着作已完成，在「明心」之中，如何来见至自己的「本心」。即是把自己的「心向意念」来作导引的动作，即对自己能由了知的过程，达至明乎自己的因缘本。若是无法达至明悟自己的因缘本，难以对自己在人类世界中，有可以超越上升的资粮位。必在是对自己能由了知之中而达至，可以作回归的主要资粮本。因而对自己在明乎自己的本心、本性来作提升，即是波罗智慧的具足，才是正确！

　　　明心明心明本心　大道大道见大情

　　　明心见性人可成　回归超越必丰盈

南无本师释迦牟尼佛　降跋于无极大道院、懿敕崇心堂

天运庚辰年梅月二十三日　戌时
