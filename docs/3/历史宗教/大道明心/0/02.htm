本堂正主席　虚空上人　登台

　　　虚也实也人间理　空也真也必自契

　　　大道明心再接着　感恩上苍德泽意

　　今夜又是崇心的大圣事，再接大道系列天书第八册，是崇心能得上苍的器重与赞赏。实为我等神人的努力所成也，高兴之至并续接玉旨临堂。命派本堂　济公活佛与　二、三关帝，于十里外接驾。命派本堂　虚空御传天尊与　虚空金笔开智天尊，于五里外恭接　昊天玉诏。其馀神佛与诸贤生，及无极大道院灵修院士等，按顺序排班候驾，不可失仪。为师暂退！

　　　大道系列扬大千　引渡众灵归天航

玉诏开读

钦奉

玉皇大天尊　玄灵高上帝玉旨曰：

　　朕掌五教升降权衡，无非是让诸天界仙佛等，皆能超越上升提引自性、佛性。亦带给芸芸世界皆能升平安乐，更是让有情众生，皆可回归超越再进入乘越的位阶，乃天地大德之展现。故　朕意时而盼之，时而倚望，何者可以引渡苍生，了知大道的运行而不悖，能明悉自己的本心、本性，来作归返的资粮位。

　　今懿敕道脉─崇心堂，在理笔贤生的领导下，能克职完成上苍一册册的「大道系列天书」计七册。给予有情众生提供回归的资粮，更具有再乘越之契机，是为　朕心之切盼也。而能一册册着作大道系列天书，彰显天地真实义与人类润生相互结合，才是真实性。今已完成第七册「大道佛心」来广宣普化，足堪嘉勉。

　　朕龙心大悦，再赐续着大道系列天书第八册，题其颜曰「大道明心」。彰显自己明心见性的本质，是为个己乘越自己，自我提升为本因。命派　南屏道济古佛为主着仙佛，其馀三界高真为陪着仙佛。旨到之日，即行开着不限时日，早日完书。广引三千、大千芸芸有情众生，来了知自我提升之本质，是在人世间来成就也。

　　崇心诸生的毅力坚志不退，是替自己与苍生开启一条回归乘越的大道天梯，足堪称赞。盼能长远广渡芸芸有情苍生来回归与乘越，并脱离轮回漩涡是为主因，大道系列之永续着作，矢勤矢勇、勿怠勿忽、坚志克职完书，神人叩首谢恩。

昊天玉皇大天尊　玄灵高上帝　降旨于无极大道院　懿敕崇心堂

天运己卯年十月十九日　戌时

　　颁旨完毕，实感恭贺崇心道脉诸神人，能坚志勇往不退，足证上苍遴选栋梁才之宏扬也。亦感心「理笔」堂主能带引崇心道脉推展「儒宗圣教真佛心宗」的开演，足堪道贺也。祝贺之！祝贺之！好。

　　吾初临观此坚志道气弘溢，是栋梁才模范之创造者。再会也！吾回天缴旨。

本堂主席　虚空天尊　文衡圣帝　登台

　　　感恩上苍颁旨降　大道明心启世章

　　　真实理谛润人生　器物世界实相扬

　　今夜实是本堂之大盛事，能恭接大道系列天书第八册「大道明心」。又诸神佛与诸贤生等。在上苍的考验过程中，本堂是通过了一次次的磨难与考验，皆不退其志，更且勇往直前，是为崇心的基本原动力。可见在主事者，理笔堂主之领导与带引努力下能有此成绩，足堪告慰与庆幸。感恩上苍对崇心道场之器重，给予更大的重责使命。来引渡有情芸芸众生回归与超越，乃我崇心之本份与职责，亦是对上苍诚信之负责。勇往直前，必有大好光明之前程来迎接众生之回归。
