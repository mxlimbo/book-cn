第十五回　因缘果报

南屏道济古佛　降

八十九年二月十五日　庚辰年元月十一日

　　　庚辰新春气象新　有德有福来相迎

　　　大道明心天书着　因缘果报实相亲

　　由通货资源入于的，正是因果报偿之累积中来促成，其正是攸关前世或是累世的因果报偿之组合。因此能了知的是前世或是累世的含因改造，是如何的形成，乃是对自己有一份责任的担负。故而在「因」之形成前必先要了知在因果与报偿中，才能具足其是否有福德或是具足通货资源的累积，才是在此世之中形成一切的原本因缘。

　　「因」─之形成必须透过「缘」之到来组成才能有入于「果」之中，继而才有受到「报」偿的形成组合。乃在于是对自己往昔所作所为的累积，来评判其是否遭逢业因、业果、业障之苦磨，或是承受福德、福份、福报之享受。皆是有关自己的前因与前果所致。故而必须要在其前世的组合，并观其含因之形成，才是促成此世的一切因果之累积。然而在因之形成是一种相循反覆的轮回循环，必在于是把累世或是前世的种种因缘来促成。然而因缘形成的主要因素，在有情世界的众生，往往欠缺那一份心灵上的滋润，更且欠缺了那一份世相的圆满。常在起心动念之中，造下诸多业因，无法圆满其作为，因此即减少了此世福德、福运、福份、福报的享受。

　　「因」─是前业的组合来观视其在未来之中，是享福报或是遭厄殃。或在前因的组合中，能见至未来是如何？皆是由前因来作评判，亦关系着人类生存思想的起心动念，亦由前因来作为未来的主体因素。故在有情众生常是人为因素不能圆满所致，才造成给予自己难以转圜的苦果，如此则由前因的形成中，替自己做创造祸福因果，更是替自己造福德与恶祸之主体原因。人类世界中若能断因，即可减少未来有恶祸组合形成之发生。如此是人类世界所难以能有其功能作用之体证。人之思考方式皆无法在当下引用圆满的作为，皆是为了贪图那一点利益的获得，常会出卖了自己的灵性，更是自己造业，自己承受偿报。

　　「缘」─也是通过了在大寰世界中，所有的缘起法则，乃是通过其缘动之主体。故而在缘之形成，必是在反覆之中、轮回之中、相循之中才能具足。然而若无缘之起动，必然是无法能将含因累积来入于果之偿报。因此缘之所动是需要通过人为因素的接续过程，才是具足缘之动。然在一切之因缘法则之中，若人类不去攀引或是不接续之时，缘之动亦无法形成，因此在对一切含因组合之形成中，方能有缘动的促成。更是不分好坏、对错、正邪、是非、○╳皆是由人心来分判，其正邪与是非、○╳对错也。因在缘之组合形成中，人类世界无法了知「缘起法则」之作为，更不了知人类是一切含因的改造界。若不经由缘动加上执持力量之行使，缘之动亦无法左右人类的果报。因此在因之组合形成中，才能促成缘之起动。故在缘不分好坏、是非、○╳，对错、正邪，乃是由人类自己来决定其因缘的组合中，观其功效的过程来作评判与抉择。

　　「果」─是经由「因」之起动，而接续了「缘」之促成，才有执持的展现于果之受偿。因此果在人类众生之心态观念之中，皆无法能实际的了知，因缘、果之组合接续。乃必须要通过人为因素执持力量之持恒，才能达至果之展现。如此「果」在于众生是福德是祸恶，皆在在关系人类前造因，而入于果之偿报。亦关系着人类通货资源之具足与否？在人类世界中，可观其气色之明暗，来了知是否能具足通货资源的累积，或是其他相关因素之促成。然而人类世界中，皆以个己之心态来评判其果之受偿，是如何承受着好坏、善恶之组合，皆由人心来定之。但在好因有时是承受人类思想观念而形成恶果；有时会将恶因，通过人类思想观念的改变而形成好果。因此人类的思想观念，正是可以决定其「好因生恶果」、「恶因生好果」之作为，乃人类世界之主要改造的基本原素。

　　「报偿」─是通过了因缘果而入于的是享受、是遭殃，皆在在关系人类思想观念的作为。如此是要在人类世界中，了知在果之承受着因之前缘，而入于果之偿报，才是在人类思想观念的主导因素。因而报之偿受即无法更改，但必须是有其前世与今生的因缘相互促成，才能入于果之中。更是人类思想观念的主导因缘，才是决定果之报偿，是好、是坏、是对、是错、是正、是邪、是○、是╳，皆必须要明晰的正是如此。在古往「好因必受好报，恶因必受恶报」，但在当下的现世中，是不能以此来论断。因其攸关起心动念与人为造就，才是攸关好坏、善恶之果报。报之偿─必要承受是好因得好报，恶因得恶报不可分其正邪、是非、○╳、对错，乃在于人类思想观念与作为中来了知与评判。

　　通货资源是必须要有因缘果报之造就，才能具足，才是改造含因的基本要求。崇心着作大道系列天书，正是改造过往无法具足的因缘果报，才能让众生来体证，「命里无时要造就」的因果报偿。通货资源之开发，必须要通过人为思想观念的造就与作为。「取之于世界，用之于世界」，乃是符合当下时代众生心态才是正确。崇心着作大道系列天书，乃在于彰显儒、道、释之主体论藏，必须要能如理实际的应用，才是真实性。在开启「儒宗圣教」、「真佛心宗」之弘扬中，必须要以人为因素的促成，才能弘展、更是无限。

　　　开展儒宗扬圣教　启发真佛立心宗

　　　通货资源任汝求　因果报偿乃人修

　　　庚辰千禧龙年开　通货自然来相排

　　　改造含因启教化　真理弘扬天地哉

　　此偈同崇心诸贤生等共勉之，努力！努力哉！哈哈哈哈！
