跋三

南无燃灯古佛　降

八十九年五月二十六日　庚辰年四月二十三日

　　　燃点心灯永无尽　无垠可成必有情

　　　人生开创明本心　达至见性更是亲

　　由儒宗圣教、真佛心宗、无极大道院、懿敕崇心堂，来着作大道系列天书。乃是把天地有情的德泽遍布于四方。在人类世界中有其大情、大爱，且天地更以大情、大爱来同众生共为互辅互勉。又人类为一切万物之灵长，是以明白自己的本性、本心，才是对自己不入于沉沦的漩涡之中。更是对自己可以上升与超越的界线，因为人类皆有其特殊之使命。

　　大道系列天书乃彰显天地之大爱，来护佑有情苍生。在天地中不论任何万物皆以人类为依归，又人类更以「圣神仙佛菩萨」为依归。乃在于是人类可以上升与沉沦之分界点中来论及，人类不但可以移改天地造化。又能移改自己往昔的「业因、业果」而成为圆满，更且对自己来作化除障碍。然而在累世的因因果果之中，若不能化除，即永远都在沉沦。因而上苍的「大慈、大悲」，「大情、大爱」，乃在让人类的世界作改造的因本，亦对自己因果之完消而作清偿。

　　明晰自己的本心，并且要能见至自己的本性。任何一位「圣神仙佛菩萨」皆在此过程，来完消自己的因缘果报。明了自己本心、见至自己本性之作为中去印证，自己在「明心」过程中，达至无为教化。又以无缘大慈、同体大悲之作为中，让自己去行持，自己可以上升、超越的因缘本质。更是对自己能成就，而不必依赖「仙佛」来成就，即是「自性自渡，佛不能渡」。更是实证于有情世界，乃是实际、更是正确也！寄望未来世，无异是一种奢求的盼望而已。那永远都在寄望于下世或未来，可成就否？可超越否？若无法在当下之中来成就，何尝又能于下世或未来可以成就否？试思即可明白自己「本心、本性、本质」！

南无燃灯古佛 降跋于儒宗圣教　真佛心宗　无极大道院　懿敕崇心堂

天运庚辰年梅月二十三日戌时
