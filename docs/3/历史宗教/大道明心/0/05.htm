南无药师如来佛　降

八十八年十一月三十日　己卯年十月二十三日

　　　消灾解厄药师依　如来如去能向齐

　　　大道明心明本性　弘扬教化真理契

　　序─之主要乃是一本天书之介绍，是在对一册的书籍之肯定，即以「序」作天书之介绍开启因缘。崇心能得天独厚并非无原因，是在于通过真正的考程与磨难之中，来作长远的教化。亦在于无任何资粮接济中能坚志不退，彰显天地之理谛；篇篇真实义、册册皆真理，乃是其他所难以比拟也！

　　　儒宗圣教扬我佛门　能彰真如显佛性永承如一

　　　真佛心宗耀理无私　体天地宗教平等博大精深

　　　大道明心扬真相　人间生存是本样

　　　润生修持两相进　圆满乘越觉有情

　　　大道系列演唯识　册册实意彰本质

　　　篇篇真理皆实际　回归超越是自持

　　序─之序，能代表一本天书之主要因缘，在于大道系列天书中，能给予众生学习「唯识理论、观念、行径」，进而入于禅密的本质。是作天地演绎的根本法则；在大道系列不以过去之深奥理论来作注解与诠释，而是盼望当今时代的芸芸苍生，皆可实际了知上苍的慈悲德意。及让有情的芸芸众生，更可一目了然的去领会其中的章节实义也。盼不以浅白而弃之、盼不以深奥而惧之，在在是自我心灵佛性的提升。灵性世界与实相世界的结合，才是对我发扬上苍慈悲的本质也！

南无药师如来佛降序　于无极大道院　懿敕崇心堂

天运己卯年十月二十三日　戌时

本堂虚空金笔开智天尊　登台

　　　感恩佛教三教尊　着作明心赞颂序

　　　真理弘扬大道情　普化教育更初因

　　今夜大道明心作序颂赞，承蒙佛教三大教尊降来着作，乃是对我崇心未来在「真佛心宗」的教化资材，亦是彰显「儒宗圣教」之传承永不变！盼了知，上苍对我崇心之器重与重任之赋予。盼在，理笔宗主之领导下，发扬上苍之德泽滋润大地众生！
