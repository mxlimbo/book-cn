第二回　世俗理谛

南海古佛　降　

八十八年十二月七日　己卯年十月三十日

　　　南海开创大道册　古佛慈悲心无遮

　　　世俗理谛造实相　根本生存了自澈

　　由灵性世界进入实相世界，必须要以胜义理谛，来作为在世俗之中的种种过程，其最主要的目的，乃是对众生未来做润生与生存的取向。其在人类世界，正是把此过程来化作生存的原动力，是以在生存当中必须要有的过程。因而即是对众生在世俗的生活中，不可悖离实相生存的过程。

　　「实相世界」皆是实际的生存过程，因此在实相世界，不论任何一种众生，皆是为了贪图对自己能图餬口的动作，即形成了食物链与事务链的取向，即是食物链的组合。又在人类世界更是追求舒适的生存空间，如此即对自我生存有一份贪求与追寻，是在人类世界正是把生存入于生活之中。因而即无任何一位众生，可以脱离餬口之食欲组合，即是食物链的的相循过程。又在食物链之中，必有事物链的组合，即是对芸芸有情苍生，乃是要有通用货币的组合具足，更是因个人的所作所为来抉择。亦关系着前世或累世来的因缘与通货含因的具足，则是能对自我生存舒适空间的创造有所改变。因在人类世界生存的空间，是以通用货币，来作为交换与购买货品的主要依据。因而在通用货币之具足中，是关系着一个人之福份与因缘的具足与否，皆有莫大关系。

　　世俗之中，即是实相世界，在众生之中没有一位可以脱离于此身躯，与其他有情众生不来相互配合。因为在于自我的生存食物之供给，亦要经由他人的提供才能具足。如此是对自我生存可以提供的组合之中，人与我等皆不可悖离群体之生活，尤以都市生存更是如此。然而在乡村来言，则是我能具足一部份食物之提供者而已。因此更不能脱离于实相世界中，又芸芸苍生皆以「自我生存」为中心，自我所具足为导向。是在人类生存当中的世俗理谛所创造与推演。因而人类生存的过程，正是通过了群体与个自生存的总体组合，亦是在对自我生存创造更舒适，更有福祉之发明。

　　「世俗理谛」若悖离了「胜义理谛」，即让芸芸有情苍生，皆难以达至超越，与更深一层的提升。在于灵性世界与实相世界的两相组合之中，胜义理谛与世俗理谛，皆受世间凡尘的众生相互推演。因而即在生存之中，通过了对自我灵性的提升，而不仅是在人类的生存过程而已，是把灵性世界来同实相世界两相结合。因在灵性世界若没有实相世界的组合与配合，即难有对自己能做回归、能做超越，而且在于实相世界是把灵性世界来作取向的印证。如此人类生活的过程，若仅用于实相世界而无灵性世界，即使当今科技所能印证者。亦有诸多尚难以用科学逻辑所推演、所主导、所证实。因此在实相世界，正是不可悖离灵性世界！

　　世俗理谛正是凡尘众生生存的过程与经验，在于自我身躯同此有情天地，来相互结合、相互印证。正是把此世俗的种种一切生存，来做润生的过程，因而是对自己能有一份长养之生存过程，即是众生所追求的。倘若是悖离了真实的生存，又有哪一位众生可以做超越，可以做回归，可以更深一层的乘越。即受到世俗生活所影响，更在人类世界中，无法能真正提供在心灵上的慰藉。即以生存的空间中，难有真正心灵上的寄托。即是无法对自我的心灵中创造往上追求的原动力，更是造罪与造业的来源。在灵性上的寄托，是心灵往上提升，更是把世俗理谛同胜义理谛两相结合，更且把灵性与实相两世界来做统合。是在人类的信仰当中，所必须要具备的。在灵性世界的追求，若没有实相世界来配合，即似当今第三空间的神灵世界，那欲改造过往的含因即难也！更且是难之难也！

　　世俗生存是人类世界真实的过程，在天地长远，宇宙、虚空的长远来论及。则是对人类世界之「短暂」提出一种哀鸣！更是感叹人类无常之短暂，又天地的长远，是把人类世界在短短的数十寒暑当中，做改造、做推演。如此即对自我一种同天地、宇宙、虚空之长远来做比较、来做印证。这是人类世界，所必须要有的作为与思考的方向。盼能了知人类生存的世界，不仅是对自我润生提供一份长远的追寻，更是对灵性提升的一种长远的追求。能达至是乘越的印证与作为，正是实相世界的世俗理谛，所必须要有的观念与长久动作行持。

　　　世俗生存人生理　背离难以相容契

　　　胜义相加互辅作　实相结合灵性依
