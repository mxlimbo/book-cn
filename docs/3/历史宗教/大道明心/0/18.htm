第十三回　命运迷思

南无弥勒菩萨　降

八十九年元月二十八日　己卯年十二月二十二日　

　　　南无胜地清净国　弥勒下生造佛陀

　　　菩萨降着明心册　命运迷思了自我

　　由立命行运之中，来言人生的前世、累世的因因果果之过程。皆与个己的前缘、前业有相当的关系。因每一位众生，皆有其不同的含因累积。是故对自己的前缘、前业的承受因素，是以平等法则对众生，每一位皆在平等法则的施行过程。故而在凡尘的一切众生，皆是承受着前业、前缘的组合方能有今日，才是对自己能了知在当下的此世中，是如何做建构，又如何来成就于自己的命运。

　　因对自己的前业、前缘所造成的种种相循因素，承受着自己所能忍受的过程，乃是对自己命运的一种相循的交替作用，其过程正是对自己在命运之中的迷思。来作为此世给予自己能有向上提升的基本资粮，乃经由宗教来启发，才能有深刻的体证一切经验，并且是由人类自己来赋予与改造。然而在自己命运之中，皆无法对未来能有实际的掌握，甚且在当下与未来中，亦无法能实际学习仙佛圣神菩萨的行径。然而无法能提供自己在未来行持的过程中来作改造，更惶论对未来能了解与掌握。因而命运即随着业因、业果的循环，而使自己入于其漩涡与陷阱之中。

　　因而人类是无法破除在其命运中的陷阱与漩涡，常会随波逐流的走完了一生。那是何等的悲哀！如此的生存过程中，请问众生有谁愿意否？皆是不愿意。但在其无奈的过程中，更且是无可奈何。又如何来化除在命运中的一切迷思？故而即必须要了知，人类是承受着前世、累世的因因果果而有此生。但其在人类生存的世界，正是「生命的改造界」。若不能改造，且反而更是造罪、造业，那又如何能有改造含因的基本资粮。甚且无法能行功立德，更去造业、造罪、造恶，请问众生汝等愿意否？皆是不愿意，因而即是无法了知在命运中的迷思过程。

　　命者乃先前既定的前业、前缘，而成为今世所承受的种种福报与恶祸。因而佛家曰：「欲知前世因，今生受者是；欲知来世果，今生做者是」。乃是在先天命中，是前世、前缘相加在一起的命格，更是无法转圜与改变，必要是自己去承受，自己去完消，才是在消除业因、业果之循环过程中，让自己达至无债一身轻，但往往是事与愿违。众生常会因为在造业之中而不自知，更是「因上加因、果上加果」。能改造的，不论是有修持或是无修持，皆无法了知，其先天命格中种种的含因累积。即是由人类的八字中，来了知其是在何年、何月、何日、何时会发生灾祸？亦是由八字之中，来演绎其在此身，所必须要承受的福报与业果。如此正是符合了「前世因、今世果，今世因、后世果」。乃是人为因素的造作，故一切正是符合了因果法则的公平性。因此在命格之中的先天命，是无法改变的主要原因亦在于此。若能回归于昔时─即前世去做化除，或许可能改造，但如此皆背离因果法则的定律。那众生所作所为即可以改过，如此在先天命中是无法如此的作为。

　　后天运势承受了先天命的主导，故而众生所行所为、所作所事的每一项事务中是都含包着前世、前业、前缘的累积，才是给予自己有承受改造的空间。如此不论前世是好？是坏？或是一切如常的平常境。即是由自己在前世所作所为的结晶，亦是结果也。但命之主理，乃关系着运势之分差，每位众生皆会对自己所行所为。所作所事，不去细研探究其来龙去脉。有之会怨天尤人，甚且一味指责他人之不是。而不思自己有否任何差错？故常在「因上加因、果上加果」，又如何能改造后天运，请问如此能改过否？皆难也，难也！

　　后天运势在人类之所作所为当下，是必要能了知如何来改造，如何来化除往昔的业因、业果才是实真。乃是以自己的过错中去反覆思考，为何我个己是会如此？又如何能完消此业因、业果之造就，不要去责怪他人，反思是对自己一种反面的成就。更在于人类是将他人之过错来作为借镜，亦是以为警惕，才是自己能改造自己，亦在于改过的含因种籽，思考观念非常的强烈才能有所行动。正因是观念主导着行动之作为，必然可在改造中来化除过往的不良错误，亦是对自己一种警惕与苛责之中，让自己来成就，才是改造的因本。

　　命运迷思在有情众生中，皆无法了知透彻。因此即常会受到前业、前缘之主导，故而即无法改变之原因即在于此。然而有情众生，皆无法能明悉其改造的作用是如何，又如何来了知人生的世相。那即在一世世中，来反反覆覆的轮回，甚且永远都在轮回之中。必是在人类世界，都无法能跳脱其陷阱与漩涡。在崇心堂着作的大道系列天书，每一册皆详实记载着如何来化除、如何来完消、如何来了知、又如何来成就，皆在在恭喜着诸贤生，能有此天机超越于未来的弥勒净土。而直超进入无极虚空，乃是一般众生所无法能有的大福缘。亦是对有情众生，能提供其回归超越的资粮。故而能有崇心的一系列着作，切盼有情众生能详细研读大道系列天书，必可在往后之中，直超极乐无极虚空世界，而不必再来弥勒净土去修唯识观念、理论、行径，而此正是其他凡尘众生所无法具足之大福缘。

　　　命运迷思靠己为　前世业因业缘堆

　　　人生所为皆影响　改造含因要自推

　　　命格前缘订今生　运势有关前世因

　　　迷途了知可化除　思虑改造归自成

　　此偈同有情众生来相互共勉，以作为在命运迷思之中的了知与化除，才是真正的大智慧者。在此千禧己卯年尾之着作，就此划下休止符，期待明春千禧庚辰年再开笔着作大道系列天书。
