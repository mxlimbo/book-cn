第十六回　福禄寿德

上元赐福天官大帝　降

八十九年二月十八日　庚辰年元月十四日

　　　上元赐福依德凭　天官大帝降来临

　　　庚辰龙年气象新　福禄寿德更亲邻

　　由先天命格来言，人类的福德祸殃皆是由个己来创造，必观视在前世与累世，整体造就过程，是为主要依据，是将福德来赐予、是将恶祸来赐予，皆关系着人类生存所必须要有的作为。且是将前世的种种含因累积，在自己的主体观念与作为中，来作为是评判的依据。故而上苍所能赐予，正是由道德的行持中来作论断。又以人类生存的过程来言，在往昔所承受的这些不论是好坏之分差，或是正邪、对错之分别，或是○╳是非的作为，必须要将这些过程的种种，来当做自己在未来的依持作法，上苍是以此为依据，即是根据着自己在此世中的心态观念与作为来评判，才是让有情众生皆一体如是。

　　福─是有情众生皆对此有一种追求与向往，必要是在现世当中去追寻。若是欠缺了此一「福」之具足，必然是无法能让自己在凡尘中有平顺的坦途。会让自己遭逢诸多的业因、业果、横逆、挫折，甚且遭逢诸多的灾殃来磨其身心。而此是对自己福份、福德的欠缺所致，亦是对自己无法能有其个己心态转换的导引。故而在众生来言，亦是对自己无法体证了知自己所作所为中，究竟有多少的「福份」、「福德」「福田」、「福报」否？皆难也！乃在福是由心态的转变，亦是依持着自己的作为来决定，且是否能具足在「福之德」中，有其真正能享之「福份」。必是要福份所倚、福份所臻、福份所得，才是人类所能享用之。如此若无福即难以在人类世界中，有可享用之福报也！

　　禄─是自己在生存当中不可欠缺，不论食禄、衣禄、财禄…或是其他相关的福禄。皆是在人类生存的当中能拥有、能体证、能享用之，必然是以禄之作为来具足福之倚也。因禄正是同福之倚中，必然会让自己更有实质的器物实相来享用。因而以食禄来言，众生皆要食也，又此食之禄是关系着过往之布施，与当下现世之作为及环境之过程中，有其真正可用之食物链。亦因食之是众生饱餐的主要资源，任何可动的众生皆是一体如是，因此食禄是有情众生不可或缺的。然而在财禄当中仅只人类可用之，其他众生亦难使用；在官禄来言，若欠缺官禄，人类是无法能有其官俸或职司的位置，故而人类可用，其他众生亦难用之。

　　寿─是一切众生生存的长短依据，有之活了百千岁，对人类无任何之功用与效应，那长短即非人类所要求与比较的。因是在寿元是通过上天之大德─为好生之德。又是大地之大德─为养生与育生之大德。然在人类之大德─正是护生与放生之大德，能珍惜其他物种的生命。又能保护与不杀生更且放生、护生，皆可在寿元之延长中作凭证。此亦是寿元长短之依据，在天生我才必有用的当下中，且每一个物种都是，而不仅是人类而已。必是在大寰天地中的芸芸众生皆谓之才是真实。若仅在人类世界而已，那只是对人类的一种珍惜，并非是真恻隐、真慈悲，乃在寿之主体的本元中，是要了知如何能获得寿之延长，乃真心放生与护生体天地万物生命之延长，才是由此来依凭。

　　德─之主体是攸关人类生存的取向，若是在取向当中无法能具足德性，也只是对自己生存的相循轮回而已。为何有人能利用短暂的生命，来创造无穷尽的德泽于有情世界。有的却是无法具足德性来遍布于有情世界，相反的更是在造业、造罪与轮回当中，让自己入于永远都无法转圜的恶殃与业果报偿中。如此那不来人世间反而可减少了造罪、造业、造恶果之机会。是否如此，不但可以体证自己在生存的环境中，是如何能替自己来创造德性于有情世界，乃必须要是以德而服天下，以德而让自己、让众生皆可滋润于德之扩展中。

　　福、禄、寿、德是相关主体的串连，因而必须要了知的正是「德」可以具足「福之主导、禄之享用、寿之延长」，皆关系着德性的发挥。若无法能具足德性，要了知在人类世界中，是难以具足福、禄、寿之依凭，故而必是以符合上苍规范，上苍来赐予福、禄、寿。有时给人一点油水，亦是一种得德性的展现。但众生往往欠缺了那一份不可计较之心态，往往在计较中都输掉了福、禄、寿之来源。那又是多么的可惜啊！因福德具足禄寿才有依持，给人方便，自己吃亏往往更是占了大便宜。果若在对方来言，那真是吃力不讨好，以为在当下中自己是占便宜，往往在未来中正是输掉了自己的福、禄、寿之依凭，那才是得不偿失。

　　　福也禄也寿也哉　德来依持齐相排

　　　若能吃亏占便宜　未来福禄寿更来

　　　千禧儒宗开弘展　庚辰圣教教化阐

　　　端月真佛启推演　元宵心宗更弘广

　　今夜观此气象，真令为师欣慰与庆幸。未来上苍能以此弘扬「儒宗圣教」、「真佛心宗」的开展，让上苍与众为师等企盼与赞赏也。此偈同崇心诸贤契共勉之，「通货能具足，弘扬能开展，教化更真实，理谛则顺和」也。
