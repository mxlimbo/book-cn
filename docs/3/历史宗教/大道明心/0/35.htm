第三十回　实心世界

九天司命真君　降

八十九年四月二十一日　庚辰年三月十七日

　　　九天旋转宇宙界　司命德泽众生谢

　　　真君荫护天地人　实心世界是本位

　　在人类生存的世界中，是以润生来作取向。去行使自己因缘中好坏之分别，且不论在有情世界、非情世界中皆是一体如是。乃在于生存的过程，让个己能入于润己生存之过程，是必须要能明悉在实相世界，若欠缺了实际的本灵，何能在器物世界来行使自己的因缘和合。所以应以实际的本心，才能让个己能在「起心动念」中，促成自己是往上行使，或是往下墬落之分界点。若不能明悉实心世界，何能达至明本心见本性，乃为谬论也！

　　实心世界的本质，是在于必须要有个「本心」，才是促成自己能以实际的本灵作润合的依持。因在实际的本灵来言：若降于凡尘的众生，必须先要具备有自己的躯体。倘若无法能具足此最基本的躯体，何能给予灵性来降生，又何能给予自己在「实心、实相」世界来作为润生依持。因而即以自己的行持，作为自己润生的结合体。乃在于实相世界是无法能脱离此实心世界，因而必须要能具备的是此一个真实的心也！实心的本灵即是实际的本心，因而在此实相的世界一切起心动念，皆离不开此实心世界。故而无法能在此实相世界中，作自己该如何行持的依据。因在于实心世界的「起心动念」，是无法离开于此「实相世界」。因而对此有情世界的众生来言：若无此实际的本心，何来给予自己在实相世界中去完成、去结合自己有利益的任何举动。且在于对自己的起心动念，皆必要能将此实心来完成。

　　因而对自己在实心世界作回归资粮的促成，如此必须要能将此实心的本质来作依归。但此实相世界因是对此「实心」，无法能作到此实际依持的润生过程。皆难有对自己行使的责任与义务，故而即难以完成在此世之中是该如何行？如何作？又如何对自己有利益也！然而即是对自己不知该如何行，又如何能了知实心世界。不仅是对自己能否有利益的任何举动，皆对自己是一种学习与成长。尤以宗教教化的过程中，若难以了知自己的实心世界。必难以对自己有明悉本心与本性，更难以对自己生存或修持有实际的功效。乃在于对自己不能了知自己的实际本心，又如何能明心见性。那无异是对自己一种的欺骗手段，亦是对自己无法深切体悟。此实相世界的一切动源，乃在是自己迷惑了自己。又如何能洞悉有情世界的种种过程，绝无法脱离于此实际的本心世界也！

　　明心最主要的功夫，乃在了别此实心的本质，然而若无法能对自己有确确实实的行使，自己作润生与修持来彰显自性佛体。那又何来能有对自己明了自己的本心，即难以对自己有明心之本能，是对自己难以了知自己明心的本质。在于有情世界之中，若无法能让自己在明乎本心，去行使自己明了此心的作用，那对自己在实相世界，根本都难以达至实心的本质。即是在有情世界，在作自己未来以及创造反覆的过程而已。即难以达明心、见性的真实功夫也。见性是以自己对自己的本性，即是自性的彰显。则在于是一般众生所言之佛性也！又在于道家、儒家又是何？仙性、圣性也？皆是对自己生存利益，的另一种向上提升的原本动能。乃是对自己能见至自己的本性即自性，又能如何来彰显，必在于是要点燃自己的心灯。若无法能明悉此过程，欲彰显佛性即难以达成也！

　　彰真如显佛性，乃是有情众生最切盼，亦是最喜爱追逐的修持境界。若是无法能「明心、见性」必难以达至彰真如显佛性。，更且无法了知如何来点燃自己的心灯，无异是对自己难有可以体证的此世界实际过程。必在于自己无法点燃了那一盏灵性深植于内心的心灯。即是：

　　　燃点心灯照佛性　心影无踪化本灵

　　　人生修持为那桩　皆要明悟了自心

　　　有情世界化生灵　皆可彰显佛性因

　　　体证明晰化有情　明心见性灵更清

　　实心世界的有情众生，皆难以达至自己心灯的燃点，即对自己无法能提供有情的资粮。若是一味对自己姑息养奸，那在未来之中，何者可以上升，何者仍在轮回，何者早已下坠皆关系自己的抉择与判别。必须是自己能了知心灯点燃的步骤，一步步来成全自己。能在此有情世界中去作更大的发挥，才是给予自己作改造自己，更能成就自己，与成就有情众生，乃是正理。

　　今夜辛苦诸贤生在此有情世界，能有此宗脉来发扬上苍的圣务。足堪印证与嘉许也！勉之！勉之！盼此有情的弘扬能带给众生，向上提升「明心见性」的因本资粮。
