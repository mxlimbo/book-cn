第廿七回　读书本意

北极玄天上帝　降

八十九年四月七日　庚辰年三月三日

　　　北极天界中枢星　玄天教化众生灵

　　　上帝德泽有情世　读书本意理更清

　　在有情众生所能了知的读书本意，是作为了知在明晰理义的本质中，将一切的「范本」来作为其生存取向的依持。故可从读书之中，来了知生存的定义，在实相世界之中，来作润生取向的依据。乃是由读书的本意，不但是对自己能有润生取向，是对自己的生存环境来作依持。因而读书即是了别识字的开端渐而获取知识，且在于了知读书的作用，是用于何？在于古往当今的读书本意，早已是变质良多。

　　古之读书，无非是求取功名利禄的取向，因而即是「十年寒窗苦读无人识，一举成名天下知」。乃是对读书一种仅求取功名富贵的依持而已，又在当今时代的读书早已是分门别类，也是对自己生存取向的依持。然而是对自己通货润生的结合，必要是作为众生，或个己润生取向的结合点。因读书不仅是对自己生存取向有其依持，更符合当今时代的种种科技来作演化，来作为学习的成长过程。因而读书是一种了别知识获得的经验过程，其目的乃是将过往之文盲的化除，是给予有情众生更是学习的经验累积，其功能乃增进众生的长远福祉。更是对有情世界能提供一种良善的表徵。则在于对众生能有学习增长经验智慧的累积当中，来作其个己生存依持的根本性。

　　「读书」的本意，亦在于对一切书籍、经典前人经验教化的过程。由书本之中，来了知，其前人经验过程，种种点点滴滴的心路历程。更是对有情世界的苍生，提供了一种社会福祉的增进，乃在于是对有情众生化除了文盲的分界点，进而长远来看，若能了知各国语言的精神所在，必然能迎奉而无往不利，必是对一切国界的语言通过沟通来了知。即可针对其问题点来解决，因而不但可以化除了各国间的相互争锋，相对的作为。即是由读书之中，所能相得益彰有效的方法！

　　「命运风水」若是无「读书」即难以在「行善」方面去作行持。相反的更是难以了知经验知识的提升，更在于无法了知前人的种种心路历程，更是无法体证其心境的转换是如何？那又能如何了知祖先或是过往之前人的种种，更且无法了知体证，圣神仙佛菩萨的心路历程。又在当下所着作大道系列天书，亦是由读书来获知其经验的累积，及作为了知圣神仙佛菩萨等的心路历程，更能体证其心境的微妙所在，乃是对后代子孙的一种心性的提升。倘若是无书籍的记载，何能了知在炎黄中土，由古代迄今早已是历经了千万年的过程。然而在中土能有历史记载，仅短短的五千多年而已，即是对中土炎黄的历史过程所了知的，无非仅是九牛一毛而已，即是针对当今社会教育的迷思与盲点！

　　书籍正是记载过往经验与心路历程的教化资本，若是无书籍的记载，岂能了知在远古时代种种过程，更无法了知在当今生存的地球之中，早已是混沌了三次之时光。亦在于有情当今的众生，皆难考据了别于古往，甚且把古往的演绎过程来化除了。因而即难有对古往一种学习成长历程的经验也！但在当今时代，又一切讲究科学证据的当下中，若一切都要由科技来证实，那必须要经过一段很长的时光，才能达至。举例：在灵魂学说的当下，科技渐而能印证的，不仅是连最浅显的过程，都难以体证与了知，仅止于摸索的阶段而已。因而在有情众生，是对当今时代一种依赖的过程，若是不依赖，又能如何也？奈何？奈何之？

　　读书是对自己知识与智慧一种的提升，若是无读书，即难以体证一切天地运转演绎的实证也。然而经由书籍的记载过程，能由读书来体证，其点点滴滴的作为。乃是针对当下有情众生，一种向上提升的原始动源。读书的本意，是在于了别如何来作为？如何来取向？如何针对自己心性的提升。倘若在任何一个宗教派脉之中，若是无经典考据，可供后代传承者来研究解读之时。必难以针对其宗教派脉的精神教义，作有情的下化过程。更惶论来作为传承也！不得也！不得也！因在于有情教化传承的过程，必须要对精义能融会贯通。若无了知经典之时，即无读书的作为，乃是无法体证与了知其过程是如何？又如何建立信心来行愿持恒也，不得之！不得之也！

　　读书的本意，是针对有情世界芸芸苍生，提供一个向上学习成长的契机。若是无法能了别其心境的明慧之时，即难以针对自己经验过程的获得。亦难以由他人经验过程的提供，即在读书的本意中来实现了。

　　　读书根本造己福　福荫自家兴旺注

　　　了别经典书中意　有情天地是自主

　　　成就与否皆人为　上升下坠分两推

　　　经验成长智慧丰　利益人间德追随

　　盼了知对读书可以提供的经验过程中，来成就自己明心见性的基本要求。即是对自己明心的本质中，有相当大的关系也。观崇心今日祝寿的气势！真令为师感欣慰也！「儒宗圣教」的提升正是崇心的天职也。不可轻忽之！不可轻忽之！
