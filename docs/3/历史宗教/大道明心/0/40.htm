第卅五回　魄之作用

豁落灵官王天君　降

八十九年五月十六日　庚辰年四月十三日

　　　豁达开启真实意　落笔妙理有天机

　　　灵官王天君降着　魄之作用述真义

　　由魄来分别其作用，乃是把佛家未能解述之部份来补足。然而若以魄乃是对自己生存「实相世界」的一切统合体，是作自己向外攀缘的主导原动力。是对自己能经由大脑中枢神经来作指挥，即是给予眼观色，主导之后才能有所起动。在于魄之作用中，若是无「尸狗」的主导即无「伏尸」的根本性。乃在于是把「雀阴」来作向外去攀缘，因此雀阴即在于对自我生存，有其好斗的原本意来言。在于雀阴是自我原本的魄体作用，即是对雀阴能了知在生存之中向外的取向，即是雀阴所能主导。亦会入于人的睡梦中，皆是由雀阴所分判的因原本。若无法能净化雀阴，一切的因因果果即一而再、再而三的向外去攀缘。不论在过去或久远之中的因缘体，或是在于当下的本体因缘。皆是主导其因缘果的发生，乃在于是雀阴的因缘本作用。

　　因此雀阴之净化，是人类难以了知的魄体。故而在魄是把七者来细分：即是在尸狗、伏尸、雀阴与吞贼，再而即是蜚毒、进而即是除秽、再则即是臭师也。因而雀阴是一切的因缘果，即在当下若是无吞贼即难以成就。因此吞贼即是对因缘果，可以接收与不接收之分判。然而若是对自己不经由吞贼，即难以开启因缘果的促成。在于此世当中，人类生存的环境，皆是受到承先启后的教育与教化课程。在于吞贼若不把雀阴的因缘果来促成之时，即难有业因果之形成。然而在魄是谓之浊气当中，如何去净化则是蜚毒所主导的功用。因而蜚毒即在让自己，能经由生存的关系中去作建构与促成，乃是蜚毒所成就的一切因缘体。

　　蜚毒不但是对自己因缘体的过滤与消化。即在于让魄的分差，来了知人类生存的意识与如何行之。因而蜚毒不但是对一切主导建构之分差体，亦是消毒的主导原本者。然而在人类当下的现世中，皆受到毒之气息所蒙敝。故而在人体功能，虽能净化的当中，但在身体皆承受了毒质的气体所侵袭。因此生魂之主导作用与功能，在魄是经由人体的大脑中枢神经，进入于魂之本体。乃在于魂之作用于主、觉、生三者，才是融合了灵魂之分别。又以灵魂的分别来了知，在魂分差中「生、觉、主」的功能与作用。

　　蜚毒即是对自己难以进入「主、生、觉」三者之灵魂中，要了知是必须经由大脑中枢神经，才可以促成向外发动其因缘体的本质。因此在魄是难以进入灵魂之中，仅在外向的内在层里面来作统御。故在于魄即是对蜚毒有其相当的重要功效，因而蜚毒即在除秽之中，来完现人体生存的魄力动作。乃对自己一种向上升华的因缘本，在于魄不但是将过往来除秽，亦是无任何好坏、对错、是非的分别。仅在是一个类似磁铁而已，一切皆在阴与阳之相吸与相斥之中。来作吸入与排挤的动作，皆受到磁力的相吸相斥作用来了别。

　　由魄之总合体是以肺部功能，即是臭师也。然而一切可以引动的「阴神」即是魄所主导也，乃在臭师即是肺之主体功能所发出的号令。即是再给与自己能经由魄之好动中，来促成攀缘的动作。因此即是对魄所建构的臭师中之作用，即给与对自己或是他者之「相吸、相斥」作用，才是将臭师的功能体来彰显。人生于世不可谓之「魄」为不良浊气，一切成就的「圣神、仙佛、菩萨」等。皆是「把魄来净化」、「把魄来超越」、「把魄来升华」。否则任凭外功或内果十分可观，亦难以达至上升与超越，必须要了知灵魂之建构体，乃在魂与魄之作用中，可以上升净化与超越，才是真实性，更是人世间成就的因本原。

　　魄之分别：尸狗、伏尸、雀阴、吞贼、蜚毒、除秽、臭师

　　　尸狗乃是魄主导　伏尸作用了知道

　　　雀阴相加必更好　吞贼何者可知晓

　　　一切作为能明了　才能超越上升跃

　　　蜚毒功能乃净化　除秽本能自然消

　　　臭师完现磁相斥　因缘本体互相邀

　　　才给人生路更妙　净化了知灵自高

　　此偈同有情众生共勉之，亦会对未来中能有切实的真实义与了别也！
