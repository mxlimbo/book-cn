第七回　假空存实

南屏道济古佛　李修缘　降

八十八年十二月卅一日 己卯年十一月廿四日

　　　南屏大道渡一贯　道德重整惟自转

　　　济世古佛利人端　降着假空存实章

　　　千禧年前尾次回　公元两千如意归

　　　开启儒宗扬圣教　弘展真佛立心宗

　　由真空妙有之中入于灵性世界，更是在引导真正有修的众生。来了知在灵性世界中，必要将人类世界的种种行径，入于能彰佛性、显真如的过程，不可存有其在人类世界种种的不良含因，即是破除了对因缘过程的总和体，又以人类世界的因缘和合才可能促成。如此即在含藏了累世的因因果果之过程，来把人类世界迈向更超越的境界。

　　真空妙有是因缘的过程，而在「因起的缘动」的过程中，若无「含藏的因缘」必难以达至成为果实〈报〉。即在于含藏了因缘的启动中，是以含因的初动过程，亦即在含因的未动前来言之。果若无含因即无因缘的造就，更惶论能入于因缘和合的起崺迄中。是在于若不能将含因来作为，即无法开展人类世界的一切创造与创进的原动力。因在是把累世的含因过程来彰显，才能将人类世界的因缘来启动，且在于「真空妙有」之中，而入于「假空存实」。

　　假空不能为之假也，乃在于含藏了其存在的因缘，即如天地之中的一切生物来言，尚未有天地之前即已含藏了。但必须迨至天地开创之后，才能有其存续的空间，如此即不能谓之假空也！因在假之中必须要有其依存的条件。尚且要有其存在的因缘，才能在未来中有其存在的价值显现。然而以此来言，人类世界中尚有诸多难以用当今的眼光，与观念来推演、是存在的种种一切。因是对有情众生，提供一种向上发展的资源。必在于当下是不存在，但在于未来之中，必然是因为有因缘和合的存在。因而有其含藏之因素，但若想在当下中见未来，那无异是痴人说梦话。但以未来的科技来言，若无含藏的因素，即无存在的空间。必以假空而不能存实，是以人类科技来印证于当今的其他星球中。是否有同人类相似的人种，或是较人类目前科技更先进千百万倍，那即是假空之中而能存实。

　　含因即以假空来谓之，而存在于未来之中能见至也！因在真空妙有，是思想上的向上提升，而假空存实是实相世界的向上提升。因此假空不能谓之假也！原因即在于此。假空若不存在，那存实更不必谈论。要了知在「无极」当时下化人类生存于地球中，若以当时的契机，是无法能印证于未来中有何可以做回归、作超越的范本。乃在于无极当时下化有情人类于地球之时，亦很难了知在人类生存的地球中，有何可以做回归、作超越的课程与工夫。甚且是对未来苍生更能达至超越的契机，果若无法能在于含藏因素中，引证未来的实相过程，必然无法能假空而存实，即以精神世界的真空妙有来作依持而已，必难以用假空存实的理念，来做未来统合的印证，即是入于顽空的境界也！

　　假空而存实即是人类世界中含藏因素的累积，即在「含因」之中，来印证于未来人类实相的科技契机。更在于是对有情的众生，给予一种向上提升进而超越。因在于当下之中，是无法了知在未来中，有何可以向上印证的实证。乃在于是对假空而不能谓之虚幻也！因在当下来言是虚幻，但在未来中「假而能成真」，「真而成实相」，必然是对有情众生可以做印证、可以做心灵的改革，且在于改造之中来做印证。人类科技所带给众生的福祉，必要是上苍能在未来中，有诸多的印证与显化，才是对有情众生，有可以依据的指标与目的也！

　　　真空妙有是灵性　假空存实更显真

　　　真真假假人生理　假假真真皆如意

　　　假空存实未来依　存实必得再显机

　　　不可言之眼未见　科技印证实相义

　　在有情众生之中，常会因目前所见之而谓之真实。殊不知在眼前一转瞬，即消失的无影无踪。必在于能否把握当下去做改造，更要是将目前所成就的往上提升。才是在未来之中，能有真正可以依循的目标。是以此在有情世界中，来同芸芸苍生共同相互勉励，亦要求能真正在灵性世界中，把实相世界来相辅相成。否则一味完全依赖灵性，那亦无法能突破实相种种的困境，因人类世界正是欠缺了此顿悟之本质。即常在渐修之中来改造自己，但往往会障碍自己，皆是受到人类世界生存的影响所致也！必要能明了生存的意义，乃在于创造宇宙生命的大结合。开创人类世界向灵性的提升，才是同实相世界来相辅相成也！盼之！盼之！
