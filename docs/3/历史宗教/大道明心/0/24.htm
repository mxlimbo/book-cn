第十九回　善德聪智

汉锺离仙翁　降

八十九年三月三日　庚辰年一月廿八日

　　　八仙降来着天书　汉钟离仙翁降着

　　　善德聪智人根本　大道明心实特殊

　　由无尽藏的智慧入于正是在人世中，必要以聪明智慧之良善来导引。故在于福德无穷之应用中，要了知正是智慧无尽藏之本体，亦在于是对良慧与恶慧之分差。因在人类世界正是此两极化之分别中，若无法引用善德聪明智慧之时，必会流于二种智慧来行使于人类的世界。倘若无法引用于善德聪明智慧之展现时，必会入于造恶慧之作为，甚且殃及人类福祉之存在。

　　聪明不代表是智慧，而智慧必是聪明之显现，然而在人类世界中，很多人皆是相当的聪明，但往往无法应用聪明，德泽于自己及有情世界的苍生。往往仅会滋润自己的生存，而无法扩及于有情苍生。乃在于个己私欲之显现，即是利于自己而无法利他，则是人类世界常会引用的做法。因在人类世界，聪明的人占大部份，亦是每位皆是应用小聪明，去面对自己生存持有一份长远之作为，即在于此小聪明，是对自己润生有相当大的帮助。果若无此小聪明，即常会在于自己之生存中。欠缺其利益自己的作为，相反的正是让自己，来滋润于通货的具足，在生存享受其福报。

　　智慧正是在聪明当中所引用的，又一切的聪明人，大都无法具备智慧。相反的却常在聪明反被聪明误之当中，让自己去造业与造罪。此也是在人类世界中，常会运用的平常事，因为无法了知天地实义之演绎过程。只知道有我个己生存的取向，而无法能泽润于有情世界。所以聪明反被聪明误之中，造下恶殃、恶业、恶果，而此亦成了人类生存当中最要不得之作为。又因无法了知天地运转之实义，更且无法能德泽于有情苍生。因在此过程，即常受到自己生存利益的取向来导引，其往聪明的路途上行走。

　　聪智正是「聪明智慧」之显现，聪明不代表是有智慧，而具备智慧正是聪明之人。乃是「天公疼憨人」，是以聪明者得之，智慧者得之，憨厚者得之，决不是太笨之人而得之。且是能了知天地运转之实义当中去行使自己，并在利于自己后，更下化于有情苍生，才是真正人类世界之聪明智慧之人。因此在人类生存的当下，必要了知生存的实义，不但是利益自己，而且还要利益苍生，才是真正有智慧之人也，亦是憨厚之人也。

　　有聪明智慧者大都不会造业、造恶、造果。因其能了知天地运转之实义当中，人类生存的世界，正是在几十个寒暑当中来渡过。若能利用其有限之时间与空间，换取无穷尽之福泽于自己及利益有情苍生，必在学习天地之大德来泽润于有情苍生。因其不但能具足善德聪智，亦可在福德无穷、智慧无尽当中，去作为人类世界之主导。正是在聪明智慧运用的十分得当，亦在于聪明智慧运用得宜，不但是利于自己与苍生，更是学习圣神、仙佛、菩萨之精神，来完现于人类世界。

　　聪明是经验的累积，其体证正是观一而知十，观十而知百。乃在于天地之大道理，正是「一理通万里彻」是天地实义之本质，如何在「一理通万里彻」必须要能通过其经验的体证、智慧的结晶，且在于善德聪智是一切过程的显现，亦在于聪智而不入于恶慧之中，才是聪明智慧之人，更具备善德聪智，才是在人类世界所能引用，且能符合人类生存的定义规章，是在有情世界中是不可抹灭的，乃在于是聪智而不可入于恶慧中，更可是导引善德聪智之行使。

　　　聪明了知天地义　智慧高超人生理

　　　善之存有体天心　德泽有情化大期

　　　智慧无垠运用当　善德人行我自偿

　　　聪明人生行大道　福德智慧两相好

　　此诗同有情苍生共勉之，亦在崇心堂中来同诸贤契等共勉之。可以运用善德聪智去导引自己在福德无穷、智慧无尽当中。去成就自己与利益苍生，才是对未来的有情世界的芸芸苍生，有一份长养教育、教化的资粮。
