第卅六回　本灵魂魄

南屏道济古佛　降

八十九年五月十九日　庚辰年四月十六日

　　　南屏道济着明心　渐至完成悦自心

　　　本灵魂魄启圣心　大道见性再崇心

　　由灵来入于魂所成就，乃在于是把「生魂、觉魂、主魂」之建构于肉体内。是以魂所成就的，作为生息与向上的资粮位，是把人类的生存在于现象世界来成就。因而是对自己能否有可以「取向」与「向取」的作为。必是对自己能给予创造、又源源不断的力量，才是促使自己在人世间来作成全。因而要了知的是把自己在人类世界中来作建构，更是把自己在「凡尘世界」来成长。必要给予自己能有不断的「精进」与「创造」，才能达至完善之境界。

　　本灵是每一位众生皆具备的条件，不论在任何的世界中，或是物种皆具备，倘不能具备本灵，必难以达至生存取向的目的。因此要以「取向」」」与「创造」，才是给予自己在人类世界中，可以向上去「学习」与「成长」。但在人类世界往往受到五尘境的迷惑，让自己无法在人类世界中，可以按照自己的意愿去作改造。皆受到「前世因、今世果」，「今世因、后世受」的侵袭。因此无法有作「创造」与「改进」的范畴。即在人类世界一世世来轮回、一世世反覆形成自己的障碍，更且让自己形成墬落的根本源。若是无法能创造与改进之时，请问众生来人类世界，不知仅是否，为了「吃、喝、拉、杂、睡」而已。还是让自己沉沦于五浊恶世，去反覆的轮回，

　　在本灵所主导的正是在「魂魄」之中来作建构。不论「有情、非情」皆是于要作「改造与创进」。如此真是很悲哀的事情也！倘若无法能「创进与改造」之时，即反覆在「人类世界」中，根本都无法能具足，其可以向上提升与攀求上进的原动力。因而常受到自己「无法来超越」、「无法来作回归」，因此在魂魄所建构的，正是要把此来改造与上升超越。才能对自己在当下之现世中去「成长」、去「超越」。然而本灵、魂魄所主导的，正是让自己来超越与回归。如此佛家所用，与道家所用必要能「相辅相成」，才是回归必须要有的基本态度与作为。倘若无法能作到此境地，必会是时常遭逢魂魄之阻碍与间隙之差误。在于佛、道两教所应用的名词必要能融会贯通。才能对自己的魂魄来了知，在意识观念之中来作建构与主导。才是对自己在人类世界中，来作自己真正的主人，而不会沦于被自己的魂魄所奴隶也。

　　魂之建构乃「生、觉、主」。又魄之建构在于「尸狗、伏尸、雀阴、吞贼、蜚毒、除秽、臭师」之中，来完现每一个步骤，皆有其真正的功能与效应。亦是在完现其真正可以导引，的主要根本性之中来成长。因而要了知在七魄，不仅是对自己的喜、怒、哀、乐、爱、恶、欲之中所建构。甚至是对自己的「唯识思想」观念与理论来作成长。必然可以在当下之现世中，来作圆满与超越。因而在人类世界所能成长了知的，无非是经由经典的明晰来作判别。但在往昔的经典之中，难以见至其真正把七魄来判别明晰。即常在含糊之中，给予众生丈二和尚摸不到头，因而即常受到文字障之阻碍。必难以了知在人类世界中，是要完现于本灵、魂魄加上「唯识观念」之主导。才是让自己在人类世界中，来作自己的主人，而不会沦于自己遭受魂魄的奴隶。

　　魄之好斗与好动之中，要了知一切因源本的建构，是由魄所主导。而魂所建构，乃是把自己的因缘体来作了别。亦在因缘体，不但是一切因因果果所促成的唯识世界。是以「万法唯心」、「万殊唯物」之作为，要了知人类世界，不可脱离于生存的任何一个细节。必在是任何一个环节之中，任何过程中皆是自己通过了一切生存的经验累积而达至。才是让自己能在人类世界之中，具备有丰富的资粮，面对一切的过程，来作生存取向的建构。把了知的知识来转化为智慧，才是提升自己在人类世界之中，来成长与获得丰富经验的累积。才能让自己明了此心，应该有的基本观念。倘若无法明白自己的心性是如何？一味的仅以为几次的明了自己的本心即是「明心」，那离见性尚有一大段路程。况且还是沾沾自喜，自以为是已经开悟了，那是对自己无法了别在明心之基础工程中，应该如何行，又在见性之中，如何去作真正的基础，才是达至明心见性所应有的态度与作为。

　　明心乃在了别自己生存一切的过程中，让自己去体证、让自己去磨练、让自己去成长，由学习之中达至渐悟之本质，来累积了诸多次的明晰，即是渐悟中而达至顿悟。亦是由顿悟中、亦非仅是几次之顿悟、就以为自己已是开悟了，那是会自欺而欺人、自误而误人。乃在于通过了学习与成长的经验中，来完现于渐悟而达至诸多的顿悟，才是累积了自己开悟的根本性。

　　见性是要了知自己在本灵之中，来将魂魄去作净化与静虑。才是对自己在见性之中，如何达至燃点心灯，照亮自己的本心。来彰显自己的佛性，更是对自己能有可以向上超越的根本源。众生皆无「六祖慧能」的根器，仅能在「神秀大师」的渐悟体证中，来完现在日常生活的动力。如何对自己了别于本灵、魂魄之净化与建构，任何「圣神仙佛」皆是通过如此的过程，来彰显自己佛性于人类世界。不可去寄望于下世再来成就，那会是对自己一点把握都没有，何况是一个未知数的下一世否？试思即可明晰。

　　　本灵彰显明晰心　魂之主导乃无痕

　　　魄之建构净化了　七情六欲人生要

　　　大道明心已完成　大道见性再着增

　　　人生因缘为根本　回归超越在理真

　　此同有情众生共勉之！至此已完成大道明心之着作，辛苦诸贤契与诸神佛等。盼在未来能长远着作，更精微与妙理之大道系列天书。一册册皆在真正进入核心，去下化有情世界的众生。不分任何宗教派脉皆可成就也，好也！老衲（为师）要回天了，下会有期，再会啦！再会也！
