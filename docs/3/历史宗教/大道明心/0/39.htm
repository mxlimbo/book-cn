第卅四回　魄之分别

南极仙翁　降

八十九年五月十二日　庚辰年四月初九

　　　南极仙乡人世少　仙翁降着真理兆

　　　魄之分别人体上　大道弘扬妙本章

　　由魂入于魄，是以魄来分别其功能与作用的同时。乃在于自我身体所必须要有的魂魄作用，必在于引动外界的一切因缘体才可能促成。更是由人体功能与外在攀缘的结合体，即是对自己来了知，魄在人体作用的一切总枢纽，才是对自己能用以生存取向所必须。然而若没有魄，即没有一切的原动力，乃在于魄所行使的功能中。要了知魄存在于身体是一切生存取向的原动力，人世间若没有魄，即无任何可以创进的取向。

　　魄分七种，在尸狗中所引导的主体动能。若是无伏尸，即难给予尸狗有任何作用。因在于伏尸，正是同尸狗相互成长，亦管辖自己身体功能的取向。然而在尸狗与伏尸中，必要能具生存的取向才能有动源，在动源的给予即是伏尸所管辖。因此一切向外攀缘，是通过了大脑神经中枢的向外发号施令，必在伏尸中来给予有向外的动源。如此伏尸所主导，即在身体向外动能之中来创取，来给予成长与攀缘。但在于伏尸中，必要能再由雀阴的主导动能，才是对自己能经由眼观色所促成。在于伏尸所主导的功能作用，即是以色相世界的存在界，即生存的当下之中，若无「雀阴」即难以引动其「伏尸」在神经系统之中，来导引其眼观色的作用。如此要了知是在伏尸与雀阴中，是经由大脑中枢神经来发号施令。因此雀阴即是以眼观色的形相世界，来「取向」、来「攀缘」。是经由尸狗与伏尸之中，去成就于雀阴的向外创取。

　　有雀阴必然有吞贼，在吞贼中有其功能的正是尝取万味的总根缘。其正是吞我、吞汝、吞其贼之所赋予，又在吞贼正是给予此身有源源的生存取向。即在于生魂之中来成长，因而生魂所接触的正是吞贼所「引导」、所「形成」的「魄」之中。且在于魄所须，正是要以吞其贼性而可以上升。但在七魂之中吞贼不能谓之浊气，要向上提升才是净化此七魄的主要关键。如此吞贼正是一切动源的主体，官能功用的主导因缘本。

　　吞其贼即是赋予生存的取向过程，在于生存之中。若无吞其贼之魄，即难以进入在魄之导引，在入于身体存在的主体因缘本。因而要了知在魄所主导的是以吞贼，再向内层来导引，其正是把魄之总合体来入于的是除秽。因而除其秽气，不但能将身体一切「内」、「外」在的不良气息来完消。更且在除秽之中「可以上升」、「可以净化」。因而在魄所成长的一切浊气之中，要能体证的不但是让自己在生存取向，将吞贼入于除秽。其正是吞了一切可动、可食的物种形成了秽气，在除秽正是魄所形成的主导功能。要了知在灵性世界，若无法能体证其身体的不良气息，正是由吞贼之内，来将身体入于浊气之中。故而除秽即是对自己生存的取向，是入于消除其不良的气息，才是正本之道也。

　　由除秽之中来完现的正是臭师，因而在七魄所导引的正是臭师所能完现。又在魂魄之成长，必须要了知在臭师所形成，正是肺部功能的吸收与排除。因而在臭师的主导原本因素之中，不但是让自己存在的身体，将「心、肝、脾、肺、肾」之主导功能，让自己存在于魄之功能。更常让自己在睡梦中，是自己的魄向外攀缘之主体功能，因而所引动的，正是魄之存在性向取动。故由魄之存在于人体的静止功能，由于魄是好动而难以静止、又难以让生存取向休止。因此由人体的睡梦之中来完现，而启发魄向外去攀缘。且由因果循环「去取向」、「去成长」，魄所引动的正是如此，乃在于众生常谓是「造业」、「造罪」，而难以转圜。又魄是引动因果形成的当下中，让魂难以招架。故而人类即以魄是浊气，而无法理解在魄之导引中，若要上升净化必是让魄能由净化中来达成。是在「修持」、「行持」的过程中，常会让自己在修行的过程，把魄来搁置。而仅是将「魂来净化」、「而不净化魄」，即在此原因。

　　果若魄能净化，相反的比魂的净化更胜一筹，因而魄所主导在于魂之动能中，让魂魄「皆能清净」、「皆能超越」、「皆能提升」。但人类往往是无法作到这一点，即在一世世的轮回之中，来沉沦皆是仅净化「魂」而不净化「魄」。故在此一切世相中，为何人殁度之后，「三魂」难以归聚在一起。即是以「秉性作事」难以「天性行事」，又让「魄流浪于三途六道四生」之中。一魄魄相加在一起，即难以作「超越」，作「统合」。故而魄即是对自己向上提升，亦是向下墬落的根本源。如此若灵性可以向上提升，亦是魂之总和体，而不净化魄之时，即难以有可以超越的基本要求。如此魄不能谓之浊气，要让浊气来净化，才是真实性也。

　　　尸狗建构主魄力　伏尸便能更得意

　　　人生了知要超越　净化魄之根本机

　　　魂魄主导灵性依　魄力有时难成器

　　　上升下墬两分岐　根本成就魂魄义

　　让此有情众生要了知，在魄力作为之中，不仅认为有魄谓浊气，是要先净化魄，才是将魂来升起、才是上升、超越的根本真实机。
