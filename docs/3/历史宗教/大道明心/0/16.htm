第十一回　人为造作

南无观世音菩萨　降

八十九年元月十八日己卯年十二月十二日

　　　南无称赞观世音　观世无情化有情

　　　音声传唤无疆远　菩萨造作人为因

　　由善恶因果中，已入于的是人为因素所作造就，其在善德与恶祸之分差，乃是由人类所行所为之主体，来化作生存的好坏。因是对有情苍生，能提供其作为的本体，是以善恶之分别。作为评判其功果福德与造罪恶祸之主体，亦是以此作为依据。乃是必须要以人为造作方能彰显，更要在其过程中，来观看「起心动念」之分差。故而所行、所作、所为，皆是在整个「起心动念」中有相当大的关系。
　
　　因人为造作，故而在人类所作所为中，其善恶好坏之分差。必须要有「起心动念」，才能在其功德与祸恶中。分别出是否「有目的而为」，或是「无目的而为」。必要了知的是由人类的思想观念，在于起心动念之中，方能有可以评判的作为与目的。又以上苍来言，希盼有情众生，皆能「显佛性、彰真如」，因而是对有情众生可以提供的真正教化课程，乃由此来作为「起心动念」的主体。更是以人类的思想观念，来作为其行持与行径的分差。给予有情众生，可以向上提升或是向下坠落之判别。则必须要观视其由「起心动念」之中而来作评判，并非仅是以已经成就的因缘果报来作评判。更是由个人的心态与目的中，将人生的因缘来化作其生存过程的总体依据。必是要将人为因素、人为造作的目的过程中，来彰显其是否存有目的与作为。

　　人为因素之重要，是胜过在因缘和合之中的果报，又人为因素更可改变上苍所既定天命。如此必须要了知人为造作，是人类生存信仰、经商、家庭与待人接物，处事的总体因素。必要能了知人为更胜其他，不论在任何一个局势来言。人为的主体，不但可以胜过上苍既定的政策与天命。更且远超其在于已经订定的政策天命之中，来作改造。在上苍而言：「天作孽犹可违，人造孽不可活」。因而上苍对人为因素的看重，是远比既定政策之订定来的重要。因此在天、地、人之作为中，是以人为因素胜于上天既定因素。又在对上苍既定因素之总体来言，必要是不脱离其在大环境的主体因素。故而以上苍的既定大环境不可变，小环境可以依照人为的善恶好坏，来作分差判别。如此必须要了知的正是在善恶因果之中，是以人为因素的造作。来给予往后的赐福、加祸之依据。甚且在人类世界的人为因素，可以改变小环境的既定天命。因此大环境的既定天命，则是无法改变。

　　由人为之主体因素，来改造小环境的天命，必然是以个己或是群体，善恶的因缘果报来分别。然而在于是对总体因缘之造作中，人类世界必须要了知的，正是将小环境来化作改造，使之良善、使之美好。必要是破除了不良与恶陋之分差，因此人为造作，是要观其善恶果报之主体因缘，则是对有情苍生，提供了一种改造的基本方式。在对人类世界的一切善恶因缘好坏的造作中，必要能将其关系着自己之善恶祸福来加以超越。更在人类生存的环境中，提供自己能上升的基本资粮。因此人有其特性，即是对善恶观念的分别十分明朗，因而即由人类之作为中来成就。在其对人为造做的主体因素，获得善恶好坏的报偿之中加以评判。更是将其功德祸恶来作分别，才是对人类世界的主体因缘，有一份判别与评定。

　　人为的重要，正是化除了上天既定的天命，乃是改造个己周遭的小环境。而在于大环境中，是无法改其上苍的天命也。因而个己业因个己承担，业因业果之主体，是符合了因果法则之通遍无碍。对人类的有情众生，若无法能提供其向上超越的主体因缘，必然是无法能得到人类的青睬。然而在有情世界的人为因素之造作，必须要以人类之主体因缘，来成就其心态的转换与改造。必要对有情众生，能提供其明了因缘果报之形成方式，更要了知其在人为因素之造作中，如何建构、如何成就、更如何来依行。「因之因」是形成了对自己一种相当责任与义务，「因而果」即是以因之形成而有受偿，方是果之报也。然而在人类世界，正是可以改造其先前既定的果报。是必须要将其化作人类生存的原动力，因此人类的主体观念与行径。皆不会受到先前既定天命的左右，必须观其后来之作为，才能评定也。

　　　因之因缘成果报　善恶如形必然到

　　　人为因素可改造　好坏善恶之受报

　　　因已形成果报缘　人为造作是本原

　　　化除心境了明悉　因缘果报人为先

　　此偈同有情众生共勉之，在人类世界的主体因素，是以人类的主要行径为依据。即是以人为主要的因素，可以改造个己的小环境与小作为。然而是无法改造其大环境─上天既定的天命也，明否？
