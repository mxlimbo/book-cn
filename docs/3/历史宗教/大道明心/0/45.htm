开天辟地药王　神农大帝　降

八十九年五月二十六日 庚辰年四月二十三日

　　　开天辟地是大寰　药王尝草赐众端

　　　神农德泽于苍生　大帝有情护万串

一、大道明心之着作是给予有情苍生，一种向上苍来撷取资粮的契机。亦对自己在人类世界中，可否成就之分界点。

二、大道系列整体着作，乃以「佛家唯识观念」配合「道家精华」之着作。在人类世界中，以「儒宗圣教」来示现于人类世界。

三、大道明心正是对有情众生在了知自己的本心、本性、本灵之中，来达成上升与超越的因缘本。

四、大道明心亦彰显在人类世界的「实相世界」去作有情的润生过程，把经验来化作智慧的提升。

五、明心见性是不论任何宗教派脉的有情众生，所企盼达至的境界，如何来成就？如何来建构？必在于自己如何来「撷取」才是也。

六、大道系列天书之着作，足见上苍对崇心道脉之器重。是比往昔的任何道脉，更有实际的理谛，来作实证与企盼也。

七、大道明心将在庚辰千禧龙年六月二十四日，恭祝 玉皇大天尊　玄灵高上帝，圣诞祝寿缴书天庭，亦对　玉帝德泽之报答及贺礼也！

八、大道系列是以彰显「般若波罗蜜多心经」的主体架构，如何由明心来达至见性。是一切有情众生皆可达至与彰显的境界，仅在于如何「燃点心灯、彰显佛性」，而回归超越。
