第廿一回　灵性本来

张果老仙翁　降

八十九年三月十日　庚辰年二月初五

　　　张开大道行万里　果之成就乃自依

　　　老师传承立世代　灵性本来作根基

　　由善德聪智入于的正是在德性深耕之中，即是分别良恶的人性。乃在于灵性的本来体中，去应用于人类世界的一切真理。其目的是在对自己来负责，对自己生存来供给润生的条件。是要以自己提供自己的资本，才能在人类世界中不可欠缺，也正是自己自给自足的具备。果若欠缺即无法顺利在人类世界，有一平衡与顺坦的生存过程。即遭逢了诸多的阻碍与横逆，其结果乃在于自己经验之不足所致。

　　灵性是每位众生所具备的的基本条件，而不论已成就、未成就、未来可以成就的圣神、仙佛、菩萨等皆一体如是。在人类世界中，若无法能具足灵性本来体的善德深耕，即难以发挥在人性中，去往上追求一切生存过程的经验累积。其目的正是对自己来负责，且是对自己在人类世界中。是否能扮演好自己的角色，或是自己无法能好好的扮演自己，反而会是对自己在人类世界中，形成了一种相当残忍的伤害。其作为在人类世界中，无法能具足上升的资粮。反而造下了无边的罪业，又人类世界中所扮演自己的角色是如何否？皆有相当大的关系。

　　灵性的本来体，众生皆是一体如是的具备。亦绝没有那位众生，可以增加或是减少，端视于自己能否具足，在人类生存当中的善德深耕。其目的乃在于具备人类生存条件的丰厚，或者是在人类世界中欠缺了什么？在其过程中有相当大的差异，因此人类灵性的本来体，正是众生平等、仙佛平等、圣神皆平等、菩萨亦平等。亦绝没有沦落于地府，或是沉沦的其他众生较欠缺，且是一律皆平等，一体如是。因此即以人类生存的取向，来达至其造就经验累积的丰厚与否，及评判自己是造业或是造福之分界点。因而必须在灵性之中，来完现于人类生存的当下，去成就自己未来的基本资粮。

　　人性是一体两面之平衡界，是上升与下坠的分界点。能上升必是德性深耕于灵性之中；而下坠必在于恶性深植于灵性当中，才是造成灵性的污染。无法破除于人类世界，来了知天地运行的真实义。更且无法能明悟在自己所作所为之中，来完现于自己生存的经验累积，即常受到自己无法了知天地运转的真实义。仅知道有我自己而已，况且是无法能了知在自己当中。尚还有广大的芸芸苍生，又一切灵性的众生等皆是一体如是。故而在灵性当中，无法能体证自己与众生的结合点，亦由广大众生来作永无止境的教育、教化的课程。必在于自己能否了知天地运转的真实义，而不要仅受到自己肤浅知识所影响。在个己心胸当中了知天地之广大的界限，来完现于人类生存的定义。必可在彰显自己灵性的提升，更且能替自己与众生，造下无边的福祉与德性，乃当今世界的众生，所应具备的心态与观念。

　　又以崇心来言，其正是造福祉于有情世界，因而是严格要求崇心，能在当下来立「制度、规章、典则、规范」以利于未来教化、教育的传承。其正是「儒宗圣教、真佛心宗」的主体课程。因而灵性的本来体，必在于将德性深耕于灵性之中才是正确，否则若让芸芸苍生，无法能具足回归的资粮，在于当下的有情众生，又如何能回归否？因而上苍即开启龙华三会的提前，是在此当下来成就，更让有情众生能加速回归超越的基本范畴。则在未来中，不必入于龙华三会去等待弥勒净土再修持，反而是在当下之中来提升芸芸苍生，作回归、作超越的基本作为，即是此时代崇心所必须肩负的圣务。了知在当下中，是如何能推广上苍的真实义，才是崇心的大任务与大使命、大责任也，哈哈！哈哈声中话别离。
