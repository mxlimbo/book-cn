南无大日如来佛　降

八十八年十一月三十日　己卯年十月二十三日

　　　南无大开普化众　日日平常本中是

　　　如来佛降崇心堂　大道系列来祝颂

　　大大的广开「儒宗圣教」「真佛心宗」的弘扬，在我佛教阐扬近二千馀年来能有此天机者，是属相当的不容易也。因在上苍遴选真正可以弘扬的道场与栋梁才来言，是以考磨其根器为主要。乃在于能否具备了大慈、大悲、大智慧、大平等、大博爱，更且要能具备相当深厚的丰富资源。如此正是崇心能在此遴选的当下，作真正弘扬的教化课程。因是宗教平等、自性平等、修持平等，亦在于对自我提升，给予成就的因本。

　　含藏的丰富与否，是为未来教化的主要原因，为何上苍在三百五十大贤劫中，能遴选出此一栋梁才来作教化课程。是在对上苍、对我佛门的一种负责，如此即表示能给予上苍的遴选之中能雀屏选，乃是我崇心道脉能坚志不退的最主要基本原则。因此必在未来中更给予自我的提升，并带给芸芸苍生，更能提升真如自性是也。

　　日常生活即是道、日常生活即修行。倘若悖离了日常生活，且来说，我在修持、我在润生，那无异是给予自己入于难以转圜的困境。在于天地之中，人能谓三才之一，是以人类有思考的能力，又是作上升与下坠的结合点，即是对自己入于轮回与超越的两极之中，来给予自己是上升或下坠的分界点。人生乃是生命的改造界，故能上升与即在平常中，但不要在平常中入于坠落轮回漩涡里。盼之！

南无大日如来佛　降颂于无极大道院　懿敕崇心堂

天运己卯年十月二十三日戌时
