跋一

南无正法明如来佛　降

八十九年五月二十六日　庚辰年四月二十三日

　　　南无众生念如来　正法明悟本心开

　　　如来如去真实在　大道明心跋亦栽

　　由无极大道院、懿敕崇心堂，来着作大道系列之天书。一册册是以理笔宗主，先天含藏的因本来促成。亦由南海古佛 南屏道济古佛 向上苍承领懿旨，来作系列丛书之着作。无非是要在当下之有情世界众生，不必俟未来再进入「龙华三会」，更可超越于「弥勒净土」，即在当下之中来作实证。更可在当下之中，把唯识观念、思想理论来作成长。亦由实行中去了知，人类必须透过生存的因缘本之取向，才能达至经验的累积。亦对自己能了知在成长中，是真正的刻骨铭心也。

　　大道明心着作已完成，给予有情苍生能明白自己的本心、本性、本灵来作自己向上成长的无限资粮。因而即对自己在了知「大道明心」的本质，必要能体证自己在人类世界中，不可脱离生存这一关，才可以作自己的主人。否则茫茫沌沌的渡过人生的一世，那是对有情苍生一种相当残忍之事情。

　　上苍在大道系列天书之着作本意，是要让有情苍生不但在当下中。更且在未来中，皆能如理实际、如实的应用天地的实义，去作自己成就的因本源。即是「如来又如去、如去又如来」才是「无所牵挂、无所障碍、无所拖延」。更可让有情众生，皆在人类世界中来成长与成就，才能在未来中入于「莲花世界」的佛国净土也。盼有情苍生皆要回归，皆可如实来成就也。盼之！盼之！将在第九册中作实际之验证！

　　　如实了知自己心　必可成就觉有情

　　　人生经验无限好　如理实际化本因

南无正法明如来佛　降跋于

　　　儒宗圣教　真佛心宗　无极大道院　懿敕崇心堂

天运庚辰年梅月二十三日　戌时
