第卅二回　灵之三魂

中界至尊　东岳大帝　降

八十九年五月二日　庚辰年三月二十八日

　　　中界普化原灵归天界　至尊德泽有情皆超生

　　　东岳管辖冥府众沉沦　大帝引渡灵之三魂醒

　　由灵之本质来分别正是以生魂、觉魂、灵魂三者之区分。乃是将此灵三魂来区别，经由第八意识之中去做建构。亦由灵之三魂来诠释道家所引用的词汇，必是对「有情众生」能明悉在灵魂之中的分别。亦在其功能作用上，能了知是对有情众生皆可明悉的因本，如是来言，魂之作用与魄之分别，才是正确也。

　　灵分三者，魄分七种，故在于灵之本质是以道家所言，故而佛家即不用之，甚且来区分其作用是以眼、耳、鼻、舌、身、意、入于末那。再入于「阿赖耶识」之建构，故而灵之本质，是通过了生存一切的取向来作区别。以眼、耳、鼻、舌、身、入于意念的起动，即是对心态作用的了别。故而「三魂」即通过了心念意识而再升华，入于的是「传导输送」的判别，即是第七识「末那识」的主体。然而在末那分判后，「即刻同时、相关作用」来进入第八意识，即「心王」的传导作用。给予当下即有反应，而此即是灵之作用。是要通过外在的起心动念，才会入于接收的同时，即可由接收来分判。是相融或是排斥，「相融」即可圆滑、「排斥」即在当下往外推。故而灵之作用，是要靠外在才能起动内在之本灵，才是对灵有一种了知。

　　灵之分别，乃心念意识之判别，在起心动念的当下中要了知。起一心即动生魂、起二心即动觉魂、起三心即动灵魂，在此三心即为魂之主体，亦在于起三心即动三魂。故而三魂必须经由外在而传导入于三心之中，即受生魂、觉魂、灵魂之主导。才是对自己能了知外在，是经由自己接收外面的讯息，才是对自己来作统合、来作建构。在灵魂区分之主魂、觉魂、生魂总括为灵魂，但此灵魂即不生不灭、不因时空、不因轮回、不因造就善恶因缘而埋没。故而不论在何者之净土中，皆无法去除。按其生存过程，是在人世间，或是在冥府，或是上升极乐世界，乃在于此灵魂之主导功用，才是对自己能生起不生不灭的灵魂本质。

　　起一心即动魂之作用，第一心即是生魂。而在此第一心之中，要了知是对自己生存取向。有其特殊功用的当下来分别，若是起第一心─即升起了生魂，应用其自己作生存取向，故不论有情、非情皆是一体如是。因而起一心即在于生魂之作用中，来判别对自己有否作到真正生存取向，亦不论天地之中任何一种物相，皆是在生魂取向中，来完现自己生存意识之建构。亦由生存的因本来提升其向上作用的主导因素。

　　起二心，即动觉魂。由外在的一切因缘和合，皆受到因缘主体的诱导，才促成在触觉、感觉之当下中，即是对觉魂之作用。亦是第二心的本质来区分其外在的因缘，通过了眼观色、耳听声、鼻闻香、口尝味、身动触之接收后，即传入意念之中来判别，是对自我有利或是不利，皆可在此第二心，即觉魂中来输送传导。故而觉魂即是对自我生存意念之统合中，来了别其「感觉神经系统」的作用。如何能由外在的一切生存过程，来判别接收与排斥，皆在于觉魂之作用下。要了知若没有觉魂，即无任何起心动念、亦无任何的因缘和合。即对自己未来的因因果果之中，去完现自己业因福报之促成。因此觉魂即动第二心，亦由觉魂来促成，在起心第二心之中作主导因缘的成就与否。

　　起三心，即动灵魂，亦是主魂，然而在主魂之中，是通过前世累世的作为，加上此生应用于当下之中来了别。其由生魂的第一心，再由觉魂的二心传导输送之中，再接收经由外在的总和讯息来作区分。其在于对自己能了知其前业、前世、前果的因缘和合，才是在主魂，亦是在灵魂第三心来引动。在作储藏分门别类之作用，亦是「无记性」。亦无任何好坏、是非、善恶、对错、正邪。乃是通过了第二心─觉魂之分判作用，要接收与排斥皆在觉魂之中来了知，亦是佛家所云的末那识即是也。故而在灵魂之作用中。要了知灵魂是一个总合名称，其尚有三魂七魄之作用，在人体是如此微妙，故而由外在的一切事物、世相中来了知灵魂之分别。

　　灵之分别，即由起心动念来促成，经由外在的因缘事物中作导引。人体类似一个磁铁，可接收、可发布、可传导一切外在讯息，入于自己身体之内。故而「磁铁」即有磁铁的吸力，即是对自己或他人有吸引力之时，即相吸，若是磁铁两者相互排斥，即产生相斥作用。人类外在的因缘和合，即似磁铁一般，果若能了知磁铁相吸、相斥的原理作用。即可以开展人类生存的取向，是由多寡来判定。

　　魂之作用，亦是如此，若魂在一切因果来言之时，即似磁铁相吸作用。若无任何瓜葛之时，即似磁铁相斥作用。乃在于将人类因缘果的造就，来化除一切外在因素的促成。要了知人类是类似磁铁作用「相吸、相斥」的功效与功能作用中，去接收与排斥。即受到灵魂主导所作用，才是对此有情世界人类的功能中，去完现于种种的因因果果之完消，才是人类所难以了知也，明否！

　　　魂之分别生觉灵　灵魂之中本是清

　　　何者因缘来接收　类似磁铁相吸引

　　　业果完消相斥应　本是有情化生灵

　　　灵之主体建构明　才是觉悟真有情

　　盼此偈同有情众生共勉之，在祝嘏声中降来崇心观此气象，足堪上苍栋梁才也！
