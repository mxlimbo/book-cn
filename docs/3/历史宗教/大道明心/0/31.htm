第廿六回　风水实相

八仙翁之吕洞宾仙翁　南宫孚佑帝君　降

八十九年四月四日　庚辰年二月三十日

　　　八仙降着明心册　吕仙翁着大道则

　　　孚佑帝君降崇心　风水实相天地彻

　　由命运与风水的迷思之中来言，其真实义乃是在破除对风水一种依赖的心理。故在于是给予由人心，来建构心理因素的结合体，乃是给予众生一种依持的取向。在风水来言，是以地理环境的气场与磁场的两相结合。因此要以气场的作用，来改造在当下的生存过程，即是对自己生存的环境，有一自己的作为。方是对自己生存环境的作用，是做主导生存舒适与否的抉择。又在于气场是融合了自己对生存取向的流通，因而若是磁场不好之时，必要能以气场来改造，因磁场乃是先天性，而气场是后天人为，可以主导的，此即可以改变生活环境，最主要关键点。

　　磁场之关键点，正是受天地中的风水地理所影响，因在于磁场的主要因素，必须要能具足福份者，方能得之。乃在于是给予有福有德者，使其具备在先天磁场的作用导引中来作分别。因在地理风水有其时效性，有其关键性，有其福德性。若是以长远具足福德者，必然能在人类世界，不论磁场的好坏，皆能感受到其福荫自己。又在平顺当中，若不具福德者，纵使一时能得有效之磁场。即是地理之好风水，若是无法具有道德者，亦难以长远持恒。故在于风水地理，是要能有福德者方能得之，先天地理风水的好磁场，必可以在其作用的主导之中，来让其自己能有更舒适，更良好的生存环境，乃是磁场，先天地理风水的主要因素。

　　因磁场在古时而言是地理风水，若是以当今时代，必难以用地理风水来论断。因为在时迁法亦迁的当下，古往帝王若没有德者，必难以长远持恒其国度朝代的兴旺。因此在地理风水，确实是关系着后代子孙的兴旺与否，有相当重要的一环。又在好风水地理，或贫脊地理确是对有情众生生存的环境，能造就其良善与否的分界点。在于风水地理之分差当中，更以善、恶、福德，业报之分别，来赐与是良善地理，兴旺地理，恶陋地理，不良地理，或是富贵地理，皆是攸关其前世，或是今生之作为，来评判其适合生存在何种风水地理中的环境。如此风水地理不但是攸关系着后代子孙，生存福份之多寡，或是遭逢恶殃之多寡，及业报福德之分别，亦是由前世来做评判，正是给予自己在分别风水地理环境磁场，与气场的作用之中来了知。

　　古曰「要好儿孙，必先积德」，乃是不变之理谛。因在先人之辛苦开创其丰功伟业的当下，或仅是平庸的庶民，皆必须要了知在时间，空间，环境生存皆是相同。若是人为取向的依持，是可以做改造其良善或恶陋之分差。因而即是上苍给予福德者，行善者，积阴功者等等，皆是一种报偿之赏赐，乃是对有情众生一种鼓励作用。因在先天磁场与后天气场之分别中，皆关系着善德阴功，或是恶陋业障之报偿有相当大的作用。乃以上苍对有情众生，能否了知在未来中，是否能具足善德，福德，阴德，功德，心德之分别点。因而即是对自己生存的气场，磁场即是风水地理的一种良好循环也。但在当下的时代中，若以天地的风水地理论断，或是在都市风水地理来论断，皆必须要有其真正可以福报自己的基本作用。其真实的理论，必须要能以有福德者方能具备，有阴德者，方能具足。乃是对自己，对众生一种良善的报偿所致也。

　　在气场是以「当今生存的取向」，又古往之作用与当下比拟，皆是早已经不相同了。若还执持往昔风水地理观念，环境的思想，能否对当下时代的有情众生达至平衡，皆难也。是以有情众生能了知风水地理环境的造就中，能否具足先天磁场的良好否？皆不能得知！但可在自己生存的气场之中，来完现于生存的取向。其最主要的因素乃在于是给予不阻碍，能通顺，能光明，能直接通外气而相辅相成，乃成为气场最主要的基本观念与作为。更是对自己有一个较能改善良好的生存空间，即是气场最主要的要求了，因而磁场的先天风水地理之中。若以当今高楼大厦来言，是否在此生存环境的人人，都能具足好风水是否？试思即明。亦可在当今时代公寓别墅之兴建中，其风水地理的良莠与否，亦关系着整个社区或是整体磁场的兴旺。但在其自己生存具备的空间中，皆可以经由人为因素之主导，来改变其气场的作用。亦是对自己生存舒适与否的最佳良善之环境也，亦是对风水地理一种实相的了知。

　　　风水地理造千秋　磁场良好更实求

　　　气场改造舒适境　实相了知能自由

　　　风生水起好运来　也要自己来创开

　　　善德阴德福自己　遗传后代理应该

　　盼了知风水地理当中的实相，是透过自己创造也，观崇心道脉有此天机与气势，乃是对我儒宗神教提引为儒宗圣教的最佳见证也。祝福之！祝福之！
