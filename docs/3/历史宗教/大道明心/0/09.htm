第四回　真空实性

无极大道院副院尊　太乙救苦天尊　降　

八十八年十二月十七日己卯年十一月十日

　　　祝嘏声中降崇心　太乙救苦天尊临

　　　真空实性兆本明　明悟了知归自清

　　由法性、空性入于凡尘中，是要能明白，在真空实性的过程中来成就自己。更能了知实相世界的作为，乃是以成就上升的资粮才能促成。故而在真空实性，是自己不可或缺的基本要求中。不论在轮回或是上升，皆离不开于此真空实性的过程。即是自我灵性的的提升，更是不变真如的如如自性。乃是在上升与轮回中，来分判其作用的分别也！

　　因在于是真空实性，即「不变真如」上升的当下中才能促成。因真空实性即真如实性，即不变真如的本灵中，是自我的自性佛性是也。亦是在自我的真空实性，不论入于任何一个层次的空间，皆无法抹灭或消失。故而在一切的法界中，若无法了知法性与空性两者间之分别，即无法入于真空实性的基础。必要是以真空实性才能有其活动的过程，因而真空实性，是一切有情众生皆具备的本质─即是真如自性、真如实性也！

　　实际生存的过程，是不可脱离真空实性，亦以真空存在的一切法界众生，在法性之中无法入于空性，更何求能有实在的生存经验。因在于真实的生存过程里，法界众生皆是在法性左右之内，又无法能明晰空性之存在。即无法破除在空性中的假我，亦是以天地之长远来分别、来比较。又在法性之中，空性是存在于有真正行持的修子来成就的。否则即无法了知在空性之中，何以能入于实相世界？正是如此。故在空性是一切法性的根本体，即在于空中假、假中空来成就。因一切的法性，皆离不开空性；又众生常会谓之─我之身躯是为假体。即常受假中空的引诱，因而无法体证自我身躯若不存在之时，即无法谓之真空实性。更难以用「阿赖耶识」去成就「菴摩罗识」的更深一层。故众生即是如此的作为，必是无法了知，在空中假去化除法性之空性也！

　　「真中假」、「假中空」是一切法性、空性的主导体制，即在于假中何能有空性否？皆难也！因在空中假之中，一切的法性是无法能入于空性的，是以凡尘的芸芸苍生来谓之。故在于假中。空是不能化除生存的基本条件。是在一切的法性中，来成就于空性的总合体。必在于是把空中假、假中空来相互应用、相互成就。故而若不能了知在真空实性的本体中，去做自己的主人，必然是无法能有可以成就的主体。因在于法界众生，皆是在法性与空性的组合之中。若无法破除其障碍，又无法了知在法性与空性之成就，是把真空实性。来让自己能有真实的生存过程─即是经验的累积。更何况是把真空存在的一切法性，来成就入于真空实性之中。因人类生存的世界，正是真空世界，又是实相世界。但在于实性的本体中，却无法了知法性与空性的主体关系。即是入于仅为了生活而生活的相循来左右而已，更无法在不为生活而生活。那即是俗谚：「不是人啦」！即可同天地来相互印证、相互成就，我即天地、我即法性、我即空性、我即真空、我即实性也！

　　天地的造化是把真中假、假中空来相互应用、相互印证。是不可脱离于此相循的相互过程，即可以入于真如实性的莲花佛国世界里。因在于是真空实性的本体，又如何能引渡自己入于真空实性的本来处？即是我自己回归的主体因素。真空实性是在三界中，不可存有偏悖、错误、讹谬之作为。在圣不增，在凡不减，即是自我的真如佛性。因此一切生存的过程，皆离不开人类所需求的过程。因而在「生命含因的改造界」中，若无法能破除在凡尘的迷思。即无法能了知一切天地、无极、虚空、宇宙的演绎循环过程。是要在人类生存的当下中去了知、体证生命的可贵处，即在于此。不可谓之我个己之渺小，因有我个己之存在，才能同天地、无极、虚空、宇宙之串连过程。因此人类的可贵处亦在于此，同其他有情的众生，更有成就因缘在于此中。

　　实相世界之真实过程不可谓之假我，亦在假中才能了知空性之存在。法性是一切众生成就的因缘过程，必在于真空存在的世界中，了知真实性之可贵处。亦是由人类自己来决定─上升或与下坠，欲如何皆是自己来创造。任何一位圣神、仙佛、菩萨，皆无法替代与升越超渡。必要是把自己入于真空实性中，才能创造人类生存的永续而不间断。
