第十七回　福德无量

中元地官大帝　降

八十八年二月二十五日　庚辰年元月廿一日

　　　中元普化众生修　地官慈悯德本求

　　　大帝德遍有情世　福德无量智慧就

　　由福禄寿德入于凡尘人世间的正是以德性来彰显、来具足其福份的依据。乃是要以福也、德也无量无边之具足，更是要以智慧两相共辅共成之。如是而言，必须要能在凡尘之内来具足德性之彰显，才是以福报来彰显德臻之主体。且要以德性超越在凡尘中的种种作为，不但在人类世界中的福祉由人类来推广，更是要将人类生存的依据，在人世间来提升，更可以润生与存活的举动，必要能将德之本体来作发挥、来作依持，因而即以人类生存的依据，更以生活取向来认定其福泽之多寡。

　　在人类世界中，不论已成就、未成就、未来能成就的圣神、仙佛、菩萨等。皆必须要具备的正是福德无量、智慧无量，乃是圣神、仙佛、菩萨之主体作为。是以「无所为」才能「无为」，更且是「真无为」，乃要以众生的福祉为依据。更且要具足「福也、德也、智也、慧也」，真正是以德润于任何可以成就的圣神、仙佛、菩萨等。在已成就、未成就、未来可以成就的任何一位众生，皆是一体如是。更要能具足的是智慧的提升，果若无法具足智慧而仅有福报而已。那会让自己入于享福份之时，而迷失于福份之中，更且是造业、造恶、造罪之来源。因而在平顺境中，即难以了知人类生存，更是在反面的成就自己与众生，如此方是对有情众生一种向上提升的因本。如是而言，必要是不求平顺、不求福份多寡，而是以智慧的提升来辅助「德性福份」之具足。

　　「福德」是众生常会追求的取向与目的，乃因福德正是众生所欠缺的，如果能用另一角度去追求福德，那众生要成就于福德是相当容易，并且要成就于福德正神的亦是相当的容易，仅在于自己能吃得下亏，更要能施舍很多的油水给予苍生，或是德润他人。乃在「自己能吃亏即是占便宜」，那这一点不知有情众生是否能作得到、作的圆满。更且是「无所计较，无所怨心」来成就于自己。但在众生往往是欠缺此一真实作为，更是真实理谛中，皆是会为了自己的利益，来出卖自己灵性的追求上升，更常会为了那一点利益而与人争执，更且是常为了那一点利益又时常吵吵闹闹，那又何必呢！

　　「福」之来源是以吃亏来论断，在因果循环之当下，众生不知能否了知因缘果报之促成中。当今现世汝会享他人之施舍，或是获得通货资源利益等，乃在于皆有其前因之主导，而决不会凭空施舍与借与，皆是有关系着前因后果之主导因素。又在于是智者能了知其正是来消业与成就的因本，但在众生则会为了那一点利益。即出卖了自己灵性与因缘果报之完消，如此形成「因上加因、果上加果」。故而无法能了知在人类世界，若无法能具足德性之彰显，往往会为了观视别人的缺点，而埋没别人优点之彰显。更且无法能完消业因、业果之偿还，故而「福」即不愿追随于汝自己也。

　　「德」之展现，乃是对有情众生一种反面的刺激作用，亦是反面的成就众生。且是在自己无法了知人类世界之因缘果报循环中，来完消其累世之含因的组合。故在德之展现中，必须要了知的正是，能否在人世间来吃一点亏。那是在无所为又无任何起心动念下来行持，又如何来具足德性之过程中来导引其完消，或是彰显德性之本来体。因而即常会为了自己是否太笨或是太愚痴，更且是人类所言之太憨了。是否自己能了知的正是失去一部份，得之大部份。不知是太憨、太愚痴、太笨，那正是印证一句俗谚：「天公疼憨人」这才是真正的大智者。如此不求功德，不求自己福份，上苍所回报的正是无穷尽的福份。乃在是自己太笨了、自己太愚痴了、自己太憨了，那又如何否！人生于世，即是自己所行所为的由上评凭判来回报。人类世界正是欠缺此一些憨人，乃自认自己是太聪明了，又是永远一点亏都不吃。并且都是在占别人的便宜，永远都以为自己是最聪明的。正是人类世界的众生最常引以为傲之事，甚且在言谈中常会把自己来标榜、把自己来炫耀。自己是多么的聪明智慧呀！人生的聪明正是如此，那这些太憨的笨人又如何来生存？皆是由上苍来导引、来生存，因而「憨人有憨福」。乃是人类世界所欠缺的主体因本。如是德性即在于憨中来成就，福在憨中来彰显，因而要了知福德正是众生所追求的主要爱欲。但往往都无法能给人一点便宜，来完消自己的业因、业果、业障，甚且可在未来之中，能替自己造下无边的福德，护荫自己以及后代子孙。而此即「能吃亏、真吃亏」之中来完消于自己的成就。因而这些憨人都是自己吃亏，别人占便宜，即是福泽于自己与他人及有情世界的众生。因此即要求崇心诸贤契，是否能在憨中来给他人占一些便宜，自己「心甘情愿」的「无为付出」、「牺牲奉献」中来达成，不知是真憨或是智也。

　　大智若愚，乃人类世界的俗谚，但众生常为了贪求那一点利益之举，来造成自己无法具足福德无量之因本。因而即是在聪明当中自以为是，自以为自己是绝顶聪明大智者，如此即非教化所应有的基本。是以要学习天地之大憨，即在「有为中」入于「无为」。「无为」之中入于「真无为」。才是可以同天地来相比，不知哪一位是大憨或大智也。必要以人类生存的取向，去泽润有情苍生，才是真正的大憨者。因而真憨者即是大智者，而并非是人类所言：「聪明学大憨」，才不会在自己认为自己有多么深的功德，那是自己欺骗自己。

　　　福也德也化油水　给人便宜自吃亏

　　　大憨之人天赐福　德臻满盈无量为

　　　智慧提升化有情　智者大愚憨根因
　　　
　　　上苍不损吃亏人　便宜他人己修心

　　　有情天地是无为　无为之中真无为

　　　学习天地之大德　泽润苍生己德堆

　　今夜降来崇心道场观此气象，真令为师感到欣慰，在完消自己的业因、业果之中，是由憨中来完消，不知智者是哪一位也！明否？乃自己去追求也。祝贺之！祝贺之！
