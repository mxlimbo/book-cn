八十九年五月三十日　庚辰年四月二十七日

　　　崇心道场着天书　大道明心已完着

　　　弘扬教化普及众　再接再厉更成功

　　今夜恭接　无极皇母懿旨来堂完书颁旨。命派本堂　虚空彩灵仙姑，虚空彩云仙姑，于十五里外接驾。命派本堂　虚空彩霞仙姑，虚空金笔开智天尊于十里外接驾。其馀神佛及学生与无极大道院众灵修院士排班候驾，不可失仪。

无极驾前董双成仙姑　降

　　　无极带旨颁崇心　德遍天地更有亲

　　　大道明心已完成　再接弘扬必有因

钦奉无极懿旨来堂颁示神人俯伏，不可失仪。

　　　懿旨开读

钦奉

无极皇母大天尊懿旨曰：

　　母居瑶池仙乡，屡观南赡部洲沉沦众灵儿，实难以归返无极圣地，故派诸天仙佛「方方阐教、处处开堂」。无非是欲广渡下凡之佛子来回归，乃　母最衷心所期盼。而今科技文明的显现，皆以科学文明自居，且不信因果报应，故难以燃点心灯、照亮自性佛性。每每沉沦于五浊恶世，难彰佛性的成就，故令　母心十分哀伤、感叹也！

　　今　母心懿敕道脉崇心堂，自开堂后即行着作大道系列天书，今已完成第八册「大道明心」。乃　母心教化传承众灵子来作回归与超越的根本，是为上苍之栋梁道脉。而今开启「儒宗圣教　真佛心宗」之弘扬，是为　母心德意所钦仰之。必为未来之芸芸众灵子来作归返的契机，是作大道的先锋，作　母心凡尘的大道院。故由　皇母心特犒赏每位神佛与学生各三千大功，以全勤者而得之，不足三分之二者减半。才是符合为大道为苍生、为自己来作牺牲与奉献。

　　真佛心宗之弘扬乃上苍德泽于崇心道脉，是为　母心所期盼，是以同　母心心灵契合。故而真佛心宗之作为是对自己，是为众灵子来作成长与教化的道脉。在上苍所颁示的是以提升心性、灵性、本性、自性的彰显，亦在当下之中来完现、来成就。即可跳脱弥勒净土而直入于「龙华三会」的当下中，是以现世来成就、是以当下来成，才是真实性，乃　母心所切盼也。

　　矢勤矢勇、勿怠勿忽、克职完成，以慰　母心也。

母颁示懿旨于儒宗圣教　真佛心宗　无极大道院　懿敕崇心堂　

天运庚辰年四月二十七日　戌时

　　好！颁旨完毕，恭贺崇心道脉之神佛与诸贤契等，能实得　皇母犒赏功勋，足见是上苍的大栋梁才，亦是大道脉也！恭贺之！恭贺之！好！吾回无极缴旨，辛苦诸神佛与诸贤契也！再会了！再会也。
