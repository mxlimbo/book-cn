第廿五回　命运迷思

八仙翁之曹国舅仙翁　降

八十九年三月三十一日　庚辰年二月二十六日

　　　曹曹暮暮人生理　国泰民安是正题

　　　舅曰命运迷思篇　大道明心实相义

　　由一命、二运、三风水、四读书、五行善，入于的是了知在命运之中，必是以后天行动力来作为，先天命格含藏的主要因素。乃在于是对有情众生之中，来演绎其前世含藏的基本，才是促成了此世的种种过程。因在含藏于原本的因缘和合之中，是以后天生存的取向作为依归。是对有情众生不可仅只执视于先天命，而不思后天运的改造，是对自己一种相当残忍之作为，因在人世间正是「生命的改造界」。若无法了知其生命的意义，是难以对自己有一份教育生存的经验过程。其正是对自己能否具足后天运势的走向，来评定个己所作所为的生存依持条件。

　　命乃是先天而无法更改，而运是后天即以人之行持而左右自己，是否能有享受福报，或是自己遭逢灾殃，即由先天含藏来主导，来演绎，其是否能在命与运当中，化除了其自己生存的依持，正也在先天命与后天运之迷思，众生常会为了自己的命运迷思前途未卜，找个江湖术士来了知，其未来生存的取向与祸福荣枯。因而即破除其先天命格之主导，然而在先天的含藏之中，是以先天既定的作为，才是对自己此生种种的行径，来成就自己有否在命运的迷思过程，作为未来的了知。但往往却是事与愿违，因在自己了知前世的种种含藏，与今世的祸福荣枯，即是破除了天机。因而在破了天机，若是能够对自己的未来有一份依持，而奋发向上去行持。即是在于自己对自己，有一份长养信仰的原动力，更对自己在未来之中能做为取向或是向取也。

　　命运的迷思，往往在第三者告知自己福分的多寡之时，若作为依持而不努力，那上苍既定的天命便难以发挥功效。甚且无法能按照先前的本命来赋予祸福荣枯，即是对自己残害了自己未来的福运，如此即是自己对自己一种相当残忍的作为。亦是『天作孽犹可违，自作孽而不可活』。乃是在命与运之中无法了知先天命与后天运势，仅是靠自己的努力与否，来改善先前不良业因的显现，更是彰显善德因缘的提升。如此来言，众生既定的天命，若不经由人为因素的造就，即难以达至因缘和合的组成。又因众生无法了知，自己之命运是掌握在自己的手里。而并非掌握在上苍或是他人的手里，那又如何来谓之「人世间是生命的改造界」否？绝不也！因而即是对自己命运迷思，必难以向上提升，反之是坠落的分界点。因在于上苍赋予众生的平等当中，是没有任何一位众生可以有多一个眼或手足等等...那是否会比别人来的怪也，亦是怪人也。如此能了知上苍的公平，平等性是如何了。

　　命运的迷思乃是自造因自尝受。在自家的共业来临时，即要自家每一位皆受到波及，更且遭殃，乃是符合了因果法则的平等性。若是自家共业的福报来临之时，自家之众人皆承受着福运的享受，乃在于是自家传承的基本因素。绝无法自家传承，来给予其他或是他人遭逢恶殃与福享。因在命运的迷思当中，众生皆无法能了知未来的祸福荣枯，仅知道一个大方向而已。而无法能了知其详细的过程与细节，及世相的作为，又无法能深切体会其过程是如何。因在命运当中，人为因素是比先天命格而来更直接，更是对自己能否遭逢恶殃与福享的分界点。

　　命运是可以左右人生的基本作为，在于有修或是进入宗教去接受薰陶，必要是对自己能否化除命运的迷思，或是随着命运的左右即轮回再轮回，那不仅是对众生一种相当残忍与不仁道的做法。因众生若无法能改过或是真正修行，必难以燃点心中那一盏明灯，更且惶论彰显佛性，那无异是缘木求鱼『不可得，不可得也』，因为是对自己命运无法了知其真实性是如何。又如何能在当下的现世，来提升自己彰真如显佛性，把心灯点燃，即对自己是一种坠落的作为，亦是反覆轮回的因循种子而已。

　　命与运、风水与读书皆会受行善的左右来改变，下世是否仍能出生为人。或是沦落于地狱畜生道，乃对自己一种无法了知的迷思。即是在自己心态转变的当下来印证与显现，其是否具备了真正明本心见本性，正是大道明心整体着作的原始本意也。

　　　命运迷思靠己为　作福造业更己堆

　　　行善福德智慧备　成就因缘乃追随

　　　有情世界器物相　明心见性是初样

　　　佛性彰显心灯亮　光明无尽造福祥
