第廿九回　大道明心

南天主宰大主席　文衡圣帝　降

八十九年四月十八日　庚辰年三月十四日

　　　南天主宰大主席　文衡圣帝阐教义

　　　大道明心着天书　崇心德泽有情地

　　由实相世界入于凡尘的，正是在此「有情世界」来作为个己成就的因本。乃是给予「有情众生」一种「向上学习」与「反面成就」的契机。因而是以在人类生存的当中来作改造，同时要了知在人类世界，不可背离了生存最主要的「润生生活」，是给予自己能具足丰富的资粮，才是能对自己有一份长养教育的根本。如是即给予人类润生，不可脱离了通货资源的因本。如此要了知明何心？即是要明悉通货资源的因本与修持改造过程，即是凡尘的『实心世界』也。

　　因是由『实心世界』来改造，故在于皆以此心来成就，一切有情世界的因缘过程。不论在家庭、社会或是经商、修持等，一切的因缘和合，皆以此心来作为其生存当下。必须要有的丰富资源，因此是以此实心作为人道基本的要求，乃在于是对自己有一份成长与学习的原动力，必要是对自己能由学习之中，去增加了诸多人类世界成长经验的累积，乃是此实心世界所必须要具备的了。因而在此唯心世界即是实心世界的有情空间、时间之内等等，皆是对自己成长的累积，亦是对自己向上与下墬之分界点。乃对自己能明悉此实心、唯心世界是必须要能明悟此心也。

　　大道者乃自然而然的孕育世界，因而在此实心世界来言，必须要能了知实心世界的种种因本。即是对此大道自然而然的孕育此有情世界，必在于是对自己向上长成的原本因素，即是对自己成长，有一切可以向上追寻的原动力。

　　追寻的原动力，即是明悟此明心的基本观念。在此实心世界，若无法能明悟此实心世界的作用。必然是对自己无法能了知一切作为，乃在于对自己是一种相当残忍的事实。在有情众生来降生于此世界之时，必然会对自己在此有情世界之中。来行使一切可以对自己有利的举动，尤以当今时代更甚如此。乃此有情众生皆会对自己行持的范围之内，来具足了一切可以追寻的因缘。是以实心世界对有情众生一种更能成长的契机也！

　　明心则是要在此有情世界，来体证其生存的经验与过程。是必须要明悉此有情世界，绝不可脱离群体的生存范畴。要了知在明心的本质之中，是必须要能进入宗教去接受薰陶与洗链。才是让自己能具足了善德因缘，更是能了知此实相世界，一切的因缘起始。乃是对此有情天地的芸芸众生，是一种明悉在『唯心世界』与『唯物世界』当中，去作更深一层的体证。如此要明白的正是对自己在当下的一切生存过程来作了知，来作体证，来作学习与成长的经验。在此当下与未来中，皆是一点一滴的经验累积过程，也绝没有任何一位有情众生可以脱离于此实相世界。因而明白此心地的通用作为，即是对「自己」与「众生」一种「福祉」与「福德」的散播，才是正确也！

　　大道明心正是实相世界种种生存经验的累积；在于不论任何圣神仙佛菩萨与有情世界种种的山妖水怪、魑魅魍魉及阿修罗…等等，皆是此有情天地的一切众生。如此在实相世界中的一切人类，是必须要能对此实相世界中的生存过程来作为依据。才是对自己能体证其在生存当中，是不可弃离此实相世界的学习成长与累积经验。尤以人类世界更甚如是，必要能明白此实相世界种种的因缘和合。即是在作一切成长的必须性！即对自己在成长的过程中，去体悟自己的心路历程，因此「大道明心」之着作本意，即在于此，果若不能实际去体证生存的经验过程，必难有实证的了知。因而大道明心即是实相世界之本来体，又在明心见性是在此有情天地之中，通过薰习过程而达至体证。此天地运行的一切因本才是正确也！

　　　大道明心实相界　大道见性更可追

　　　明心见性入佛性　成长契机再追随

　　　大道系列着长远　生活过程皆乘愿

　　　大乘菩萨觉有情　佛国净土乃自选

　　此偈同有情众生共勉之！亦勉励我崇心诸神佛与学生，及无极大道院的诸有情院修士也。盼能成长与代天宣化，成就自己与成就此有情世界的芸芸苍生，圆满回归更能乘越也！盼之！盼之！
