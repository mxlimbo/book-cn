第六回　真空妙有

元始天尊　降　

八十八年十二月廿四日　己卯年十一月十七日

　　　三清教主元始尊　崇心发扬德广存

　　　真空妙有入于本　儒宗圣教再弘顺

　　由空性真如入于真中假、假中空的过程，皆在于是中道。如此即必须要有中道因缘，才能彰显一切世相与事物的过程。然而在于真中假来言，所必须要明了的是把假中空来成就。因而即以中道的因缘过程，才可以促成在其过程中，将人类世界的因缘来开启。果若是在于对有情众生，皆不入于因缘的造就。那人类世界根本都无法能创造与进化，更惶论来做回归与超越。如此不但是有情众生的真实修子，所必须要明了的世相。亦是要对一切因缘的过程给予开展，才能有真正的入于圆满的位阶。

　　真中假的过程是入于因缘的相互循环，更在于组合的过程，让其不入于虚幻之间。因而即以虚幻为空间，亦是人类思想观念、心念意识的主导。必是要求有真正实修的修子来做总合体，才是实真也！因此真中假并非皆是「形而上」的因缘而已，更是在「形而下」的主体因缘。果若一切有情众生欠缺了因缘，更是欠缺如何了知中道思想的过程。因在于是把因缘不分好坏、是非、○╳、对错、正歪的判别中。入于假中空的主体，必是以思想的建构才能促成。如此在假中空，入于中道思想的建构。必要能了知在真空之中必有妙有，而此妙有即是真中假的建构思想。

　　因一切的众生，皆常会受到经典字义的误解，更无法体证其真正的意涵。亦因此常在文字中打转，必然是无法能脱离文字障的左右。更甚的是入于受文字的轮回限制，因在于禅宗是不立文字，有所立文字的不言禅宗。如此若欲在显宗来言，立于文字的主体。皆必要能了知是在「胜义理谛」，或是「世俗理谛」中，不论任何一个宗教派脉，皆是一体如是。果若不能跳脱文字障的左右，必难以做回归、做超越。更且常会受到文字障的牵绊，更无法脱离人世间轮回的悲哀也！

　　「真空妙有」是把「真中假」的过程，来印证于一切的人类生存的实相世界。但在「真空中」是无法见至。必然要在真空中，入于智慧精神的提升。方能达至其真中空。观其心念意识的主体，然后可在灵性世界中，导引其入于「妙之妙妙有」的境界。众生无法了知上界的天界中，有其任何可观视见至的实相物界。亦仅限于肉眼所见至之天空中，天上有的仅是白云片片，或是蔚蓝的天空而已。根本都无法见至在天界的一切实相〈虚空〉也！因而在当今科技所能印证的，也仅仅是如此而已。但尚有诸多难以实证的灵性世界，是无法用肉眼来观视见至的。乃在于是人类世界的肉眼无法能见至而已，但可以透过用科技来印证、来彰显也！

　　「真空妙有」是存在于灵性世界的主体，必要是对自己精神领域，有更深一层的印证才能见至。否则若不在心念意识的精神领域中，进入真空的实际灵性世界。那不知在凡尘的众生，是否具备了一切创进、创造的原动力？果若如此，人类的科技文明是否仅由人类来主导而已？是否存有不知的另一世界所能导引的。因此人类世界常会自认人定胜天的谬论观念，如此若不是上苍慈悲，那人定胜天的过程能存在否！不也！

　　综观当今世界，在千禧年前大地震，反扑众生的正是古谚：「地牛翻身」。那人类可以预知否！控制否！皆无此能力。因而必要了知在「真空妙有」的过程中，是人类精神领域向上的提升。必然在于「假空存实」的过程中，来将人类世界创进、创造更多的福祉。以利有情众生，又能得到更舒适、更丰盛的生存空间。是在对有情世界的人类，做一改造、做一更新，必然成就真空存在于虚空内。因此在有情世界的不仅是人类而已，尚有其他有情、非情的芸芸苍生。如此是必要以人类来做主导、来做爱护、来做更有系统的保育。否则人类世界若仅只有人类存在而已，那即在于是对人类的戕害更深矣！

　　「真空妙有」不仅是精神的领域提升，必要了知在「真中假」、「假中空」而入于「假空存实」。否则精神领域，必然会遇到诸多难以了知的瓶颈，甚且是着于顽空的境界。而无法入于三花聚顶，五□朝元的位阶与境界。人类世界的有情修子，正是在此阶段受此瓶颈的阻挡与障碍，故而无法向上提升其成就的资粮，是在于要了知「真中假」的、「假中空」的过程，而在于「真空妙有」之中建立了「假空存实」，即是精神灵性世界的大门，否则若无法入于大门内。去见至更丰盛的灵性世界。必然无法能向上提升，更且在瓶颈中而着于顽空。盼了知实相世界，正是欠缺此「真空妙有」的实际过程，而此「真空妙有」，尚存在于「真中假」、「假中空」的过程，而无法能破除「假空而存实」也。

　　　真中假性是本体　假中空因归主理

　　　人类生存是实相　灵性世界弃无明

　　　真空妙有勿顽空　假空存实化虚空

　　　因因果果皆因缘　中道实性更理真

　　以此同有情众苍生共勉之，可以破除在真空妙有之中的假相，而入于假空存实，即是向上提升的主要因缘也。
