《路得记》

　　贤惠的路得

　　在士师秉政的时候，迦南有一年遇到了大饥荒，犹大人居住的伯利恒有个人带着妻子和两个儿子逃荒到摩押地。这个人名叫以利米勒；他的妻子叫拿俄米；他的两个儿子，一个叫玛伦，一个叫基连。后来，以利米勒死了，留下拿俄米孤儿寡妇仁，艰难地度日。两个儿子长大成人，都娶了摩押女子为妻，这两个外族媳妇一个名叫俄珥巴，一个名叫路得。一家5口艰难谋生，日子过得凄凉，到了第十个年头，玛伦和基连2人也相继死了，剩下3个寡妇，这一家更是穷得难以维持生计。

　　拿俄米听人传说故乡迦南这几年光景有了好转，就打算离开摩押回去。她带着两个儿媳走了一段路程，又改变了主意，她觉得不应该把两个儿媳带到异乡去，因为要是在摩押地，她们完全可以在熟悉的人中找个合适的人改嫁。拿俄米就对两个儿媳说："你们各人回娘家去吧。愿耶和华恩待你们，他使你们找到新的靠山，使你们在新夫家中得平安。"

　　拿俄米与她们吻别，她们就放声大哭，说："不，我们不回去。

　　我们要与您一道回迦南去。"

　　拿俄米耐心地劝她们说："我的女儿们哪，你们回去吧。你们为什么要跟我去呢？难道我还能生出孩子给你们作丈夫吗？即使我丈夫还在，今天就生出了孩子，难道你们能等着他长大，再嫁给他吗？"

　　大儿媳妇俄珥巴听了这番话，就与婆婆吻别，回摩押去了。但是小儿媳妇路得仍然舍不得离开拿俄米，她对婆婆说："我们都走了，谁来照顾您呢？您现在年老体弱，我们不能扔下您不管哪。"

　　拿俄米再劝路得说："不能因为我这老婆子连累你呀，你还年轻，应当另找出路。我这把老骨头，活到啥时候就算啥时候，死在什么地方就埋在什么地方。我如果连累你，上帝一定会降罪给我。"

　　不管怎么劝，路得就是不听，婆婆拿俄米没有办法，只好让她跟自己同去。

　　她们回到了伯利恒，全城的人都很惊讶，他们相互传话说："拿俄米回来了，丈夫和儿子都没有了，只带了一个儿媳回来。"

　　拿俄米就对人们说："你们不要叫我拿俄米了，因为拿俄米是"甜"的意思；你们要叫我玛拉，因为玛拉是'苦'的意思。耶和华降祸与我，全能者使我受苦。"

　　那时候正值大麦收割。为了谋求一日三餐，路得就到地里去拾麦穗。有一天，她来到公公以利米勒本族人波阿斯的田里，跟在割麦子的人后面用心地拾麦穗。波阿斯是伯利恒城中的一个财主，那一天他刚好到田间去巡察。波阿斯看到路得在自己田里，跟在雇工后面拾麦穗，就问："那拾麦穗的女子是谁？"

　　他的雇工告诉他："她就是拿俄米带回来的儿媳妇路得。她问我们说，可不可以让她在地里拾麦穗。我们就同意了。很不容易啊，她靠拾麦穗养活她的婆婆，天不亮就来了，天黑了才离开，中间也不见她歇一会儿。"

　　波阿斯听了这番话非常感动，他对田间的农工说："这个女子非常贤惠，你们要多撤一些麦穗在地里让她拾去养活婆婆。"

　　波阿斯又走过去对正在埋头拾麦穗的路得说："女儿啊，听我说，从今天开始，你不要到别人家的地里去拾麦穗，就到我的地里来。你要是渴了的话，就到那边的水罐里去喝点水。"

　　路得听完这话，就俯伏在地叩拜波阿斯，对他说："尊敬的主人，我是一个外邦人，怎么蒙你的恩这样体恤我呢？"波阿斯回答说："你对你婆婆所做的一切，在以色列人中已经尽人皆知了。愿耶和华照你所做的赏赐你。你来投靠在耶和华以色列神的翅膀下，愿你满得他的赏赐。"

　　吃午饭的时候，波阿斯又叫路得过来跟他们一起吃。路得吃饱了，波阿斯还叫她多拿些饼放在篮子里，晚上带回去给她婆婆吃。

　　路得那天收获特别大，她拾了一伊法大麦，还给婆婆带回两张饼。婆婆拿俄米就问她："今天你在哪里拾麦穗？收获怎么这么大呢？"

　　路得说："今天我在一个叫做波阿斯的人的地里拾麦穗，他叫农工特意照顾我，所以收获特别大。"

　　拿俄米说："他可是个大好人，还是我们本族的人，是一个至近的亲属。"说完这话，拿俄米好久没有讲话。她在思索着什么。

　　临睡的时候，拿俄米对路得说："女儿啊，从今以后你就每天到波阿斯的地里去拾麦穗吧，不要到别人的田里去了。"

　　这样路得天天在波阿斯的地里拾麦穗，直到大麦和小麦都收割完了。有一天，拿俄米对路得说，"女儿啊，我想我该给你找个安身之处了，好使你后半生有福可享。我看波阿斯这个人满不错，又是我们的至亲，他的妻子去逝已经两年多了，你跟他比较合适。

　　今夜他要在场上和农工一起簸大麦，你要沐浴更衣抹膏，到场上去，不要使别人看见。你等他的工人都睡了，他也入睡以后，你就看准他睡的地方，掀开他脚上的被子，躺在他的脚下，与他同睡。他必告诉你应该怎么办。"

　　路得就照婆婆的指点做了。到了半夜，波阿斯醒来，闻到一股暗香扑鼻而来，一翻身，吓了一跳--原来有个女子躺在脚下为自己暖脚。他问："你是谁？"

　　只听得一个甜美的声音软软地回答说："是您的婢女路得。求你用你的衣襟遮盖我，因为你是我一个至近的亲属。"

　　波阿斯这才明白是怎么回事，其实他早已暗暗地爱着路得，只是他不是路得家最近的亲戚，而且年龄比她也大得多才没有提出来。没有想到路得会这么勇敢主动地提出这桩事来，他又惊又喜他说："你的贤惠早已在以色列人中传扬。少年人无论贫富你都没有跟从，因为他们不是你的近亲。我虽然是你的近亲，不过可还有一个人比我更近，明天我去问问，如果他肯为您尽亲戚的本分，就由他吧。尚若不肯，那就是上帝恩宠我了。"

　　路得就在他脚下躺到天快亮，行人彼此还看不清面目的时候才离开了波阿斯。临走的时候，波阿斯撮了6簸箕大麦装在她用斗篷作成的口袋里，帮她扛在肩上，打发她回去，又吩咐她说："不可让外人知道你到这里来过。"

　　第二天，波阿斯到了城门口，坐在那里等着路得公公家那个最近的亲戚路过。那位亲戚过来了，波阿斯就叫住他说有重要的事情商量；他又从城里请来10名长老。波阿斯对那亲戚说："从摩押回来的拿俄米，现在要卖我们族兄以利米勒的那块地。我想有资格买这地的第一人就是你，其次就是我。你可以在这里的长老们面前说明白，你愿不愿意买这块地。"

　　那亲戚回答说："我愿意买。"

　　波阿斯说："你从拿俄米手中买这地的时候，也应当娶正在替她儿子守寡的路得为妻，好让死人在他的产业上存留名份。"

　　那亲戚说："如果是这样的话，我就不能买了。你可以买我应当买的地。"

　　于是那亲戚将鞋脱下来，交给波阿斯。从前以色列人定夺什么事，如重大的买卖成交了，转让的一方要脱鞋给接受的一方，以此作为凭证。

　　波阿斯又对众长老说："各位前辈，请你们作证，凡属以利米勒和基连、玛伦的，我都从拿俄米手中买了；又娶玛伦的遗孀摩押女子路得为妻，好在死人的产业上存留他的名，免得他的名在本乡泯灭。"

　　在座的市民和长老都说："我们做见证，愿耶和华使进你家的这女子，同建立以色列民族的拉结、利亚二人一样。又愿你在以法他（伯利恒的原名）得亨通，在伯利恒得名声。愿耶和华从这少年女子赐你后裔，使你的家像他玛从犹大所生法勒斯的家一般。"

　　波阿斯就这样娶了路得为妻，成为正式夫妻。夫妻俩又把拿俄米接到自己家中奉养。波阿斯和路得相亲相爱，生活得十分美满幸福。没有多久，路得就怀孕生了一个儿子，取名叫俄备得。俄备得的孙子就是大卫，大卫是以色列的王，是以色列人民最伟大的英雄。（得1一4）
