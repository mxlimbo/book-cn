《何西阿书》

　　何西阿的预言

　　何西阿是北方以色列王国的先知，他生活的时代为乌西雅、约但、亚哈斯和希西家作犹太王的时候。这时候的以色列北方有亚兰人，东面有亚们人，东南是摩押人，南临犹大国，西临大海，西北则是胖力基人。

　　当耶罗波安作以色列王时，国家并不安定，内忧外患迭起。按《圣经》的看法，这是因为以色列人背弃了与主的立约，无视摩西以来的律法，因而失去了耶和华的眷顾。

　　何西阿便在这时得到了耶和华的启示。上帝对他说"你去娶淫妇为妻，也收那从淫乱所生的儿女，因为这大地行淫乱，离弃了耶和华。"上帝的用意在以此警惕以色列人。

　　于是何西阿娶了淫妇歌蔑，她给他生了一个儿子。耶和华说，他应取名为耶斯列，那意思便是上帝很快要讨回耶户家在耶斯列杀人流血的罪。上帝说，他必使以色列家的国灭绝，等时候一到，他要在耶斯列平原折断以色列的弓。

　　而后歌蔑又生了一个女儿。耶和华便对何西阿说"给她取名叫罗路哈玛"。罗路哈玛意即"不怜悯"。上帝宣称，他不再怜悯、以色列家，不赦免他们，而却要怜悯犹大家，使他们靠近上帝得救。

　　当歌篾生下第二个儿子时，耶和华要何西阿给他取名"罗阿米"，便是"非我民"的意思。

　　但这些都是耶和华的愤激之词，何西阿接着宣布说"以色列的人数必如海沙，不可量不可数，从前在甚么地方说'你们不是我的子民'，将来在那里必对他们说'你们是永生的上帝的儿子'。"

　　《何西阿书》以象征的手法，将以色列人背弃宗教道德的行为指斥为淫乱和淫荡，将当时国家社会的灾难归结为道德沦丧。何西阿本人的行为既是对上帝的服从，又是对以色列人的谴责。他对以色列人说："你们要与你们的母亲太太争辩，因为他不是我的妻子，我也不是她的丈夫，叫她除掉脸上的淫像和胸间的淫态……我必不怜悯她的儿女，因为他们是从淫乱而生的。……我必用荆棘堵塞她的道，筑墙挡着她，使她找不着路。"

　　然后何西阿的预言谴责了他那时代的"淫行"--祭司们弃智忘律，民众崇邪拜偶，耽于奸淫和酒，求问木偶，与娼妓同居，与妓女一道献祭……。

　　他警告说以色列的祭司庶民和王室必因自己的罪孽跌倒，连那犹大也必与他们一同跌倒。他们牵着牛羊去寻耶和华却寻不见。

　　以法莲将见自己有病，犹大将见自己有伤。他们向亚述王哀求，而上帝仍将他们抛给狮子撕裂。以色列的荣华将成为过去，"他们必如早晨的云雾，又如速散的甘露，像场上的糠秕，被狂风吹去，又像烟气腾于窗外。"

　　何西阿的预言反反复复地谴责以色列人，又反复重申上帝的矜恤，其用意在使他们除净罪孽，悦纳善行，悔罪归主，仍获救恩。因为"耶和华的道是正直的，义人必在其中行走，罪人却在其上跌倒"。
