《约拿书》


　　逞聪明约拿堕鲸腹

　　有一天，耶和华对亚米大的儿子约拿说："你到尼尼微去，告诉那里的居民，他们做的坏事我已经知道了。"可是约拿认为亚述是以色列的强敌，不断入侵以色列及其邻国，对属国进行残酷的奴役，应早日受惩罚。所以他不愿去，而是在约帕港找到一艘正要开往他施的船，付了点钱就上了船，想跟船上的人一起去他施以躲开耶和华。

　　耶和华见约拿不肯听从自己的话，就让海里风浪大作，那艘小船被巨浪颠来覆去，随时都有翻个儿的危险。船上的水手尽管与海打交道多年，却从来没有遇见过这样的大风大浪，惊慌得不知所措。他们一面纷纷向自己的神祈祷，一面把船上的货物抛在水中，以便使船轻一些。就在船上的人为求生而忙碌的时候，约拿却走到底舱，蒙头大睡。船长发现约拿居然还有心思睡觉，赶紧把他喊醒："你怎么可以这样呢？还不快起来，向你的神求助，也许神会可怜我们，救我们一命。"

　　船上的人陷入一片恐慌之中。有人提议抽签来看看是谁引来的这场灾难，结果抽出的是约拿。众人就问他："你说，这场灾难是不是你引来的？你是干什么的？从哪儿来？你是哪国人？什么族？"

　　"我是希伯来人。"约拿回答说："我敬畏耶和华，是他创造了沧海与陆地。"约拿承认他没有履行神的意志，而触怒了耶和华。

　　众人闻听大惊失色，都怕祸及自身。这时，浪头越来越大，他们就问约拿怎么才能使浪平息。

　　"你们把我扔进海里，海就会平静下来。我知道你们遭这么大的风浪袭击，都是因为我的缘故。"

　　虽然约拿愿意一人做一人当，可水手们不忍心让他死。他们拼命摇桨荡橹，想把船靠岸，但海浪却越来越大，他们离岸越来越远。于是他们知道神意已定，就向耶和华求告："神啊，我们求你不要因为这个人让我们也活不成；也不要怪我们杀害无辜的人，因为这是你的旨意。"说完，他们将约拿抬起，抛到海里，海面立刻就平静下来。耶和华安排一条鲸鱼吞下约拿，让他在鱼腹中呆了三天三夜。约拿在鱼腹中不住地向耶和华祷告，感谢神没有让他淹死，并赞美耶和华的威力，发誓忠于耶和华，耶和华就让鱼把约拿吐到地上，然后第二次对他说："你到尼尼微去，告诉那里的居民，我已经知道他们做的恶事了。"这次约拿照办了。（拿1一3）显爱心得喻蓖麻树尼尼微是个大城市，要有两天才能走完全城。约拿进城走了一天，便对城中的百姓宣布说："40天后，尼尼微就会被倾覆。"尼尼微人信神，相信约拿的预言，全国上下不分男女老幼都开始禁食，身穿麻衣，国王也不例外。国王还派人通谕全国，要百姓要诚心诚意地向神祈祷，忏悔各人所做的恶事："也许神能回心转意，不再生气，放我们一条生路。"

　　神看见尼尼微人都已回心转意，虔诚仟悔，就决定不再用所说的灾祸惩罚他们了。

　　这样一来，约拿却大大地不高兴了。他对神说："我原来不就说你不会真这样做吗？你好施恩，好怜悯，你让我来下预言，你自己却改变主意不再降灾。你让我死吧，我活着不如死了好。"

　　"你这样生气有什么道理呢？"神问约拿。约拿一句话也不说，气匆匆地走出城，在城东搭了一个棚子，然后坐在棚下静等，看看尼尼微城究竟会怎么样。耶和华在他面前安排了一棵蓖麻树，使它长高过约拿，影子遮盖他的头，使他脱离太阳曝晒的苦楚。约拿因为有了这棵树少吃了许多苦头。可是，第二天早晨耶和华又让一个虫子来咬这棵树，以至于树很快就枯干死去。中午时分，热风扑面，日头曝晒，约拿头昏脑胀，又向神求死。神问他："你为这棵树生气有道理吗？"

　　"我当然生气，那么好的一棵树，为我遮风蔽雨，却无端地死了。"约拿说。

　　耶和华说："这树不是你栽种的，也不是你培育长大的；一夜之间长大，一夜之间枯死，你跟它没处多少岁月，尚且爱惜它，为它的死痛心，那这尼尼微城的数万百姓我怎么能不爱惜呢？"（拿4）
