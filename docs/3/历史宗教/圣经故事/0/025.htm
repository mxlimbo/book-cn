《但以理书》

　　但以理人侍巴比伦王犹大王约雅敬在位的第三年，巴比伦王尼布甲尼撤带兵围困耶路撒冷，后来他把神庙中一些器皿劫回了示拿，放在他的神庙中。

　　尼布甲尼撒吩咐太监长从以色列贵族的后裔中选几个相貌俊美、聪明睿智的少年，教他们迦勒底的文字语言，让他们掌握各种知识和宫里的规矩，以便将来伺候国王。国王还传旨，让这些孩子用御膳，饮御酒。但是有一个孩子却拒绝了，他就是犹大人但以理。但以理不愿意用巴比伦国王的御赐饮食来玷污自己，求太监长允许他保全自己的贞洁。神使但以理在太监长面前蒙恩，太监长很喜欢他，对他说："我是害怕我主我王的，他既然已经指定你们的饮食，要是有一天他发现你比其他孩子瘦，怎么行呢？你这样不是让我性命难保吗？"

　　"那这样吧：你先给我们几个犹大孩子吃十天素，喝白水，然后比一比我们和那些吃御膳的孩子的面貌有什么不同，要是我们都面黄肌瘦，那就依你；要是我们没什么不同，就得依我，怎么样？"

　　太监长依准了。十天后，大监长发现他们比那些用御膳的孩子长得更加健康、丰满。于是从这天起，他们就不再吃御膳，所供给的都是素菜。

　　尼布甲尼撒预定带他们来见国王的日子到了，太监长把他们带到了宫里。为了考考他们是否真正学到了知识，尼布甲尼撒开始与他们聊天，他惊异地发现，这些孩子中的几个犹大人聪明睿智，不仅其他孩子比不上，就是全国的术士和巫师，也远不及他们的十分之一，于是他就把但以理等四个犹大孩子留在了身边。

　　但以理为王圆梦尼布甲尼撒在位第二年，有一天夜里做了一个梦。他百思不得其解，心烦意乱，翻来覆去睡不着觉。天一亮，他就吩咐传术士、巫师和迦勒底人进宫。待他们到齐后，国王说，"昨天夜里我做了一个梦，心里烦乱，想知道这个梦是什么意思。"

　　"吾王万岁！"迦勒底人用亚兰语说："请陛下把梦说出来，仆人即可为陛下圆梦。"

　　"梦我已经忘了，"尼布甲尼撒说，"要是你们说出我梦见了什么和这梦的意思，我会让你们从此大富大贵，否则的话我就把你们全都凌迟处死，把你们的房屋变成粪堆，你们快说吧。"

　　术士们面面相觑，大气也不敢出，生怕国王会填怒，但是这样沉默下去国王一样会发怒，他们相互看了看，鼓足勇气："请陛下把梦告诉仆人，仆人才好为陛下解梦。"

　　"我不是说了，那梦我已经忘了吗？"尼布甲尼撒生气地转过身去："要是你们不能把梦说出来，我就只有一个办法对待你们。"

　　迦勒底人只好回答："世上没有人能说出陛下的梦来，从来没有哪位君王大臣向术士们问过这样的事情。陛下问的事太难了，除了神，凡人是答不出来的。"

　　尼布甲尼撒为术士们的无能而大怒，拂袖而去，然后传旨杀掉巴比伦所有的哲士。命令一下，护卫长立即执行。当但以理得知这一消息后，对护卫长说："请转告国王陛下，只要他宽限两天，我就可以把梦和圆解告诉他。"

　　但以理回到住所，把这件事告诉了他的同伴，要他们和他一起祈求天上的神施恩，救救全巴比伦的哲士。第二天，但以理又来见护卫长，请他带自己去见尼布甲尼撒。护卫长照办了，对国王说："我在被掳的犹大人中遇见一个人，他自称能给陛下解梦，所以我就带他来了。"

　　"那你能把我做的梦和梦的意思告诉我吗？"尼布甲尼撒问被称为伯提沙撤的但以理。

　　"陛下问的那件奥秘的事，哲士、巫师、术士和占星士都不能回答，只有天上的神能显明奥秘的事，他把日后必要发生的事指示给陛下。神把这个奥秘显现给我，并不是因为我智慧超人，而是神要使陛下知道梦的圆解。"但以理从从容容："陛下，你梦见的是一头大象。这头象非常高大，光芒耀眼，形状非常吓人。象头是精金的，胸膛和臂膀是银的，肚腹和腰是铜的，腿是铁的，脚是半铁半泥的。正当你看的时候，飞来一块非人手凿出来的石头，打在这象半铁半泥的脚上，把象脚砸的粉碎，连带金银铜铁泥一同化为乌有，碎未就象夏天禾场上的糠，被风吹散，无处可寻。打碎这象的那块石头，变成了大山，充满天下。请问陛下，我说的对吗？"

　　尼布甲尼撒正听得不知在梦里还是在现实中，忽然听见但以理问他，连忙说："对，对，对，全都对！现在你来说说看，这梦是什么意思？"

　　"好，现在我来为陛下解梦。"但以理信心十足："陛下，你是诸王之王，神把国家、权柄、能力、尊严都赐给你；凡世人所居之地的飞禽走兽也都交付与你，使你掌握这一切，你就是那金头。

　　你之后，必另兴一国，不及于你；而后又兴第三国，就是铜的，必掌管天下；第四国，必坚壮如铁，铁能打碎并克治百物，这第四国也必能压制列国。你看见象的脚和脚趾头，一半是泥一半是铁，那个国家将来必会分裂；你看见铁和泥掺杂在一块儿，那个国家也必是各种人混杂在一起，彼此却不能和睦相处，就象铁和泥不能混在一块儿。当列王在位的时候，天上的神必会另立一个国家，永不灭亡，也不归别国所管，还要灭了其他所有国家。你看见那飞出来的一块非人手凿出来的石头，打碎了金银铜铁泥，那就是至尊的国家。陛下的梦一定是这样，解释也一定是这样的。

　　"听完但以理的这一番话，尼布甲尼撒俯伏在地，向但以理下拜，并且吩咐太监给他奉上供物和香品。国王提拔但以理，赏赐给他许多上等的礼物，派他管理巴比伦全省，又立他为总理，掌管巴比伦的一切哲士。但以理的三个伙伴也都被委以重任。

　　巴比伦王火中见神尼布甲尼撒造一个金像，高60时，宽6时，立在巴比伦省杜拉平原，然后差人召集了总督、钦差、巡府。法官和各省的官员到像前，为像行落成典礼。传令官大声呼叫："各方、各国、各族的人，有令传与你们：你们一听见角、笛、琵琶、琴、瑟、笙和各种乐器的声音，就立即俯伏敬拜国王所立的金像，凡敢违抗者，投入烈火的窑中烧死。"此话一出，四方百姓无敢不从。

　　可是有几个迦勒底人却来告犹大人的状。他们对尼布甲尼撒说："吾王万岁！陛下降旨，让听到各种乐器声音时必须敬拜金像。

　　现在有几个犹大人，就是陛下派去管理巴比伦的沙得拉、米煞和亚伯尼歌，不听陛下的话，不敬拜陛下所立的金像。"尼布甲尼撒闻听此言，吩咐人把沙得拉等立即带来，问他们："沙得拉、米煞、亚伯尼歌，你们是故意不事奉我的神，不敬拜我立的像吗？要是你们再这样做，我决不食言，我会立即把你们投入烈火的窑中，难道你们以为会有神来救你们吗？"

　　"陛下应当知道，我们决不会事奉你的神，也决不会敬拜你所立的金像。即便我们被投入烈火的窑中，神也会救我们出来。"三人齐声说。

　　尼布甲尼撒怒气填胸，脸都变了颜色。他吩咐手下把窑烧热，比平常更热7倍，然后让几个壮汉把他们捆起来扔进火中。因为国王的命令太急，窑火又太热，那几个抬他们的汉子部来不及转身就被熊熊的烈火吞灭了。突然，尼布甲尼撤的脸上露出惊奇的神色来，他问身边的谋士："被捆起来扔在火里的是三个人吗？"

　　"是的，陛下，是三个人。"

　　"可是我看见有四个人，他们并没有被捆绑，也没有受伤，他们在火中自在地游行；那第四个人的相貌，好象是神子。"尼布甲尼撒靠近烈火窑门，说："请神的仆人沙得拉、米煞、亚伯尼歌出来吧，到这里来吧！"他们三人就从火中走出来。那些总督、巡府和谋士们马上上前来看他们，只见他们体无片损，身上的衣服连颜色都没有变。尼布甲尼撒说："沙得拉、米煞、亚伯尼歌的神是应当称颂的，他差遣使者救护依靠他的仆人。现在我降旨，无论何方何国何族人，亵读他们的神的，要凌迟处死，他的房屋将成为粪堆。"这之后国工又提拔沙得拉、米煞和亚伯尼歌做了大官。

　　但以理再解王梦后来尼布甲尼撒又做了一个梦，梦见大地当中有一棵树，极其高大，而且越长越高，枝繁叶茂，结满果实，野兽在树荫下休息，鸟儿在枝头夜宿。忽然有一圣者从天而降，大声喊到："伐倒这树！砍下树枝！摇掉叶于！抛散果子！把树下的走兽统统赶跑，把树上的鸟儿轰走。用铁圈和铜圈把树箍住。"仿佛圣者又在说某一个人："把他放在田野中，让他同地上的野兽一块儿吃草，让露水落在他身上。他在这里要呆7年。"尼布甲尼撒猛然惊醒，他被梦中的异象弄得惶恐不安，降旨召集哲人来解梦，于是那些术士、巫师、迦勒底人、占星士等再次聚集到宫里。所有人表示都不能解梦，国王便再次召来了但以理。

　　但以理听完了尼布甲尼撒的梦，非常惊讶，神色变得很不自然起来。国王注意到这一点，就对他说："伯提沙撒啊，不要因为梦有不祥就不敢开口，说吧。"

　　"陛下，我但愿这梦归属你的仇人，梦解归属你的敌人。"但以理说："陛下，这棵大树就是你啊。你的威势及天，权力及地。

　　你不是看见一位圣者了吗？他说的那些话意思就是你必被赶出去离开世人，与野兽同居，每天象牛一样吃草。既然圣者吩咐要留下树木，那么你的国家还是要归你的。陛下啊，求你从今往后不再作恶，要关怀百姓疾苦，这样，你可能还有转机。"

　　所有这一切果然都应验了。12个月后，尼布甲尼撒在巴比伦宫里散步，看到眼前的一切，他陶醉于自己的功绩："这巴比伦不是我用智慧建立起来的吗？这不足以显示我的威严与尊容吗？"他的话音还没有落，就听见空中传来声音："尼布甲尼撒，你听着，你的王位到此为止。你就要被赶出去，离开世人，与地里的野兽为伍，吃草如牛，而且要过7年。那时你才会知道，神让谁掌权，就会把国家赐与谁。"

　　发生在尼布甲尼撒身上的一切都证实了神的话，他果真在野地里生活了7年。

　　7年之后、尼布甲尼撒又重新得以回归王位。他从此赞美天上的神，因为他的一切都是神赐与的，神公平无比。

　　宫墙上显字示警尼布甲尼撒死后，他的儿子伯沙撤继承了巴比伦王位。这天，伯沙撒在王宫中设筵招待他的大臣，献筹交错之间，伯沙撒吩咐人把他父亲尼布甲尼撒从耶路撒冷掠夺的金银器皿拿来，好让大家尽情享用。君臣与后妃们一面饮酒，一面欣赏那些用金银铜铁木石制的神像。

　　忽然间，出现一只人手，在王宫内灯台对面的粉墙上一笔一划地写字。伯沙撤看到那写字的手，吓得面如土色，腰象要折了一样直不起来，双膝不停地抖着。稍稍平静一点后，他吩咐传巫师和术士们进宫："你们谁能读懂这些文字，把意思告诉我，我会让他穿紫袍，戴金链，在国中位列第三。"

　　遗憾的是所有的人都不认识那些字，更谈不上给国王讲解了。

　　这样一来，伯沙撤就更觉得这件事可怕，愈加惶恐不安。消息传到太后耳朵里，她很为儿子担心，匆匆从后宫赶来，对他说："你不要着急。你可能不知道，你父亲在世时，曾立一个人为术士、巫师和占星家们的领袖，他多次为你父亲解梦，而且言无不中，他就是但以理，你父亲称他为伯提沙撒。现在你可以召他进宫，我想他肯定会知道这异象意味着什么。"

　　于是但以理被带到了国王面前。伯沙撒对他说："你就是被我父亲掳来的犹大人但以理吗？我听说神灵与你同在，还说你聪明过人，那么现在我来证实一下。现在所有的巫师和术士都不能把这些字读出来，要是你能的话，我会让你穿紫袍，戴金链，在国中位列第三。"

　　"陛下，这些东西你自己留着用吧，或者把它们送给别人，我不要赏赐，我只要为陛下读这些字，并为陛下解释清楚。"但以理不卑不亢他说："神曾将王位、大权、荣誉、威严赐给你的父亲尼布甲尼撒，但他孤高自傲，刚愎自用，被革去所有王位和所有的容誉，在旷野里与野兽为伍。你是他的儿子，你知道这一切，但仍不自重，竟然冒读天神，把神殿里的器皿拿出来跟你的君臣后妃们饮酒作乐。你们还赞美那不能看、不能听、无知无识、金银铜铁木石造的神像，却没有将荣耀归与那主宰你一切的神。因此，从神那里伸出手来，写下这些字：弥尼，弥尼，提客勒，乌法珥新。弥尼--神已算出你的王位已告完结；提克勒--你在天秤里的分量无足轻重；乌法珥新--你的国家就要分裂，归于玛代人和波斯人。"

　　伯沙撒明白，但以理说的是神的旨意。他让人给但以理穿上紫袍，戴上金链，宣布他在国中位列第三。当天夜里，迦勒底王伯沙撒被杀，62岁的玛代人大利乌开始当政。

　　但以理狮穴得神救助大利乌在全国任命了120个总督，又在他们之上任命了包括但以理在内的3个总长，一有事情总督首先要向总长汇报。但以理的智慧与能力明显地大大超出了其他的总长和总督，国王想扩大他的权力，让他治理全国。

　　但以理的才华遭到了其他人的妒忌，加上国王对他信任倍至，那些总长和总督们总想试图发现他误国误民的"罪证"，找机会参他一本。可是但以理对国王忠心耿耿，从来不谋私利，从不授人把柄，于是那些人就想出一条毒计来害他。他们一起来参见国王，说："总长、钦差、总督和巡府们商量了一下，决定下一条禁令：30日之内，不论什么人，不向陛下，却向神或其他人求什么，就把他扔在狮子坑中。请陛下同意下这个禁令，加盖玉釜，使之不可更改。"国王答应了。但以理得知这事后，就回到自己家里（他楼上的窗户，开向耶路撒冷），每天三次跪在神面前祷告。跟平时一样，他的举动被那些算计他的人看到了，他们立刻到国王面前告状："陛下，30日之内不管是什么人，除了向陛下外的任何神或人求什么，就把它扔在狮子坑里，陛下不是在禁令上盖了印了吗？"

　　"是啊，"大利乌回答："照波斯和玛代人惯例，这是不可更改的。"

　　"可是陛下是否知道，那个该死的犹大人但以理公然不遵守你的戒令，竟一天三次对他的神祈祷。陛下，你说该怎么办？"

　　大利乌非常宠信但以理，虽然但以理违抗王命，但还是不想伤害他。一整天，国王都紧锁眉头，思忖着该如何救但以理，但是直到日落的时候，也没想出办法来，那些要害但以理的人却又来了，请他处置但以理。大利乌被逼得没有办法，只好让人把但以理带来，对他说："但愿你事奉的神能够救你。"

　　但以理被扔进坑中，坑口用石头堵住，又用国王的玉玺和大臣们的印把口封死。那些想害他的人以为自己达到了目的，以为但以理必死无疑，都喜形于色，而大利乌却愁容满面。回到宫里，国王夜不能寐，为狮子坑里的爱臣担心。第二天天一亮，国王就起床带人匆匆忙忙地奔向狮子坑那里。到了坑边，他大声呼叫但以理："但以理啊，你不是对神忠贞无比吗，神能救你脱离危险吗？"

　　"吾王万岁！"但以理清晰的声音从坑里传出："昨天夜里，神派来使者封住了狮子的口，叫狮子不要伤害我，因为我是无辜的，我从来没有做过对不起神，对不起国王陛下的事。"

　　大利乌听见但以理的声音，知道爱臣在神的庇护下安然无恙，非常高兴，立即吩咐人把但以理从坑里拉上来，然后又下令把那些控告他的人连同妻子儿女一起扔进坑里，因为他们无端陷害忠良，罪该万死。这些人还没有到坑底，狮于就扑上来，结果他们都被咬死了。

　　大利乌传旨，晓谕全国，让百姓从此同但以理一样敬神、拜神。但以理在国中的地位更加显赫了。
