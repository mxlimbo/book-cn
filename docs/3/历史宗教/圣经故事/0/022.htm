《以赛亚书》

　　以赛亚的预言乌西雅、约但、亚哈斯和希西家等犹太王在位的时候，亚摩斯的儿子以赛亚被称为先知，因为他从上帝那里得到启示，这启示中论到了以色列、犹大和耶路撒冷这圣城的命运。

　　以色列人因为背弃耶和华而遭受了神的谴责，以赛亚称他们为"犯罪的国民、担着罪孽的百姓、行恶的种类、败坏的儿女"，因为他们屡次背叛主，耶和华要责打他们，使他们全身满是伤口，而且没有可滋养的膏。"土地已经荒凉，城邑被火烧毁，田地为外邦人侵吞。"这一次都因为以色列人象所多玛和蛾摩拉两座罪恶之城的人民一样。

　　他们虽然也行献祭，但那只是虚浮的形式，他们仍然作恶不止。耶和华说了："又作罪孽又守严肃会，我也不能容忍。你们的月朔和节期，我心里恨恶……你们举手祷告，我必遮眼不着；就是你们多多祷告，我也不听，你们的手沾满了杀人的血。"

　　以色列人唯一的出路是改恶从善，否则必将遭受灭顶之灾，借以赛亚之口，谴责了耶路撒冷的败坏。"可叹忠信的城变为妓女。

　　从前充满了公平公义居住在其中，现今却有凶手。你的银子变为渣滓，你的酒用水搀兑，你的官长居心悖逆，与盗贼作伴，各都喜欢贿赂、追求藏私，他们不为孤儿伸冤，寡妇的案件，也不得呈到面前。"

　　外邦人的侵入，是神对堕落的城市的审判，以此炼尽它的渣滓，除尽杂质。最终，万民必归于耶和华圣殿之山。以赛亚预言了犹太民族遭受亚述和巴比伦人倾覆之后的复兴和和平。

　　到那时，"万民都要流归这山，必有许多国的民前往，说'来罢，我们登耶和华的山，奔雅各上帝的殿，主必将他的道教训我们，我们也要行他的路，因为训诲必出于锡安，耶和华的言语，必出于那路撒冷'。他必在列国中施行审判，为许多国民断定是非，他们要将刀剑打为犁头，把枪打成镰刀，这国不举刀攻击那国，他们也不再学习战事。"

　　到那时，耶和华成为唯一的神，任何偶像均被废弃。

　　耶和华又一次警告了耶路撒冷的腐化堕落，他说："因为锡安的女子狂做，行走挺项、卖弄眼目、俏步徐行、脚下玎当，所以主必使锡安的女子头长秃疮，耶和华又使他们赤露下体，直到那日，主必除掉他们华美的脚钏、发网、月牙圈、耳环、手镯、蒙脸的帕子、华冠、足链、华带、香盒……必有臭烂代替馨香，绳子代替腰带，光秃代替美发，麻衣系腰代替华服，烙伤代替美容。

　　你的男丁必倒在刀下，你的勇士必死在阵上，锡安的城门必悲伤哀号，它必荒凉坐在地上。"

　　以色列民只有凭善行而得于列名生命之册的才能幸免。接着举了一个譬喻，上帝经营葡萄园是指望结好果子，如果结出了野葡萄，那他必撤去篱笆，使它被吞灭，拆毁墙垣，使它被践踏，使它荒废，不再修聋，不再锄刨，让荆棘蒺藜生长。

　　乌西雅王逝去那年，先知以赛亚得以见到了上帝。主坐在高高的宝座上，衣裳垂下，遮满圣殿，六个翅膀的天使撤拉弗侍立两边。天使们彼此呼喊道："圣哉圣哉！万军之耶和华，他的荣光充满全地。"那呼声使大殿的根基震动，殿中也充满了烟云。

　　以赛亚俯伏地下，连称"祸哉！我灭亡了，因为我是嘴唇不洁的人，又在嘴唇不洁的民中，又因我眼见大君王万军之耶和华。"

　　但有一个六翼天使拿着烧红的炭飞到他跟前，那炭从祭坛的火中取出。天使用红炭抹他的嘴，便除掉了他的罪孽，赦免了他的罪恶。

　　上帝差遣以赛亚作为先知去到以色列民当中，告诉百姓，及时改过，"你去告诉这百姓，'你们听是要听见，却不明白；看是要看见，却不晓得。'要使这百姓心蒙脂油，耳朵发沉，眼睛昏迷。

　　恐怕眼睛看见、耳朵听见、心里明白，回转过来，便得医治。"上帝告诉以赛亚，以色列人不到最后灭亡是不会惊醒的了。

　　到了乌西雅的孙子亚哈斯作犹大王时，亚兰的利汛王和利玛的儿子，即以色列王比加联手攻打耶路撒冷城。亚哈斯王和城中百姓处于惊恐之中。耶和华便差以赛亚同他儿子到大路上去迎接亚哈斯王，告诉他，大马色的亚汛主和以色列的比加必不会成功。

　　依据以赛亚传达的默示，亚汛在位六十五年，那以色列王的以法莲国必败亡。

　　耶和华还晓谕亚哈斯，要他向上帝求一个兆头。亚哈斯答道："我不试探耶和华。"以赛亚便对他说："大卫家啊！你们听着。你们使人厌烦岂算小事呢？还要使上帝厌烦吗？因而，主自己要给你们一个兆头。必有童女怀孕生子，并给他取名以马内利，那意思是'上帝与我们同在。到他晓得弃慈择善的时候，他必吃奶油和蜂蜜，因为在这孩子还不晓得弃恶择善之先，你所憎恶的那二王之地必然被废弃了"。

　　以赛亚预言了那稣基督的降生，也预言了他到临后的富足日子。他又预言了亚述王是剃刀，他以及埃及王都要攻击亚兰，使其败亡。以色列人，即以法莲和撒玛利亚的百姓，因为他们的娇傲自大便要遭受耶和华的惩罚，东有亚兰人，西有非利士人，将要张口吞吃它。最终消灭愚顽的以色列人的是亚述王。

　　上帝说："'亚述是我怒气的棍，手中拿我恼恨的杖，我要打发他攻击亵读的国民……我的手已经惩罚了有偶像的国，这些国雕刻的偶像过于耶路撒冷和撒玛利亚的偶像。我怎样待撒玛利亚和其中的偶像，岂不照样待耶路撒冷和其中的偶像么？"

　　以赛亚传达了上帝对以色列人的惩罚，因为他们信拜偶像，背叛了神，所以要用亚述人来消灭它；上帝又警告耶路撒冷的犹太人，不要步以色列的后尘。

　　上帝又允许了以色列人的获救，因为它毕竟是耶西的一根枝条，因为耶和华的灵必能住在它的身上"到那日，主必二次伸手救回自己百姓中所余剩的……他必向以色列国竖立大旗，招回以色列被赶散的人，又从地的四方聚集分散的犹太人。以法莲的嫉妒就必消散，扰害犹大的必被剪除。以法莲不嫉妒犹大，犹大也不扰害以法莲。

　　他们要向西飞，扑在非利士人的肩头上，一同掳掠东方人，伸手按住以东和摩押，亚们人也必顺服他们。耶和华必然使埃及海汉干枯……。

　　以赛亚的话是犹太人对尚未分裂之前的盛世的回忆，也是对犹太重新统一并称雄于西亚的憧憬。犹太人希望上帝赐他们以血为代价的和平。他们的盛世是这样的："以敬畏耶和华为乐，行审判不凭眼见，断是非也不凭耳闻，却要以公义审判穷人，以正直判断世上的谦卑人，以口中的杖击打世界，以嘴里的气杀戮恶人，公义必当他的腰带，信实必当他胁下的带子。

　　豺狼必与绵羊羔同居，豹子与山羊羔同卧，少壮狮子与牛犊并肥畜同群，小孩子要牵引他们。牛必与熊同食，牛犊必与小熊同卧，狮子必吃草与牛一样，吃奶的孩子必玩耍在虺蛇的洞口，断奶的婴儿必按手在毒蛇的穴上。

　　在我圣山的遍处，这一切都不伤人不害物，因为认识耶和华的知识要充满遍地，好象水充满洋海一般。"

　　这之后，以赛亚分别预言了巴比伦城如所多玛城一样的倾覆、非利士人的必将遭受的灾难、摩押人的覆亡、大马色的被废弃、埃及的祸乱。他专门有一篇针对巴比伦王的诗歌："明亮之星，早晨之子啊！你何竟从天坠落？你这攻败列国的，何竟被砍倒在地？你心里曾说：我要开到天上；我要高举我的宝座，在上帝众星以上；我要坐在众会的山上，在北方的极处，我要升到高云之上，我要与至上者同等。然而你必坠落阴间，到坑中极深之处。"

　　尽管以赛亚反反复复劝谏以色列入和犹太人要行公义，要唯主是念，但他们无动于衷，于是以色列和犹大的天国便是必不可免的了，上帝终于假亚述人和巴比伦人之杖消灭了它们。这一段历史故事也可见于前面《历代志下》中间。

　　终整个以色列民族的覆灭期间，直到他们最终蒙上帝的救恩，重返耶路撒冷，以赛亚作为先知的训导和预言一直是犹大人信心的源泉。他一方面告诫他们弃恶从善，奋兴事主，以个人痛苦和民族的灾难作为赎罪的代价；另一方面，又虔心地期待着救世主的来临，使善人得脱邪恶，恶人不得平安。

　　《以赛亚书》充满了未世论的色彩，反映了处于流亡之中的犹大人的痛苦、悲叹以及希望。先知以赛亚的名字，在希伯来文中，便是"耶和华拯救"的意思。
