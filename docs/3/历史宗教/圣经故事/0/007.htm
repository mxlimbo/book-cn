《士师记》

　　左撇子以笏

　　摩西和约书亚之后，以色列民族中再没有出现一个能与他们俩相提并论的领袖，以色列民族从此进入士师时期。摩西和约书亚是全民族的共同首领，士师则是地区性的首领，打仗时是群众的统帅，平时则为管理民事的审判官。

　　在漫长的士师时期，由于没有强有力的统一领袖，以色列各宗族之间不断出现争执、纠纷，他们争疆界，争权势；另一方面，以色列人的外部威胁仍然存在，按《士师记》的说法，上帝那和华为了试验以色列各支派的后裔，有意在他们的土地上和周围留下了非利士人的五个首领和一切迦南人、西顿人等，使他们作以色列人"肋下的荆棘"。

　　被以色列人打败的迦南人伺机东山再起，尚未被以色列人征服的那路撒冷、迦萨、伊矾伦等城邦，以及居住在山区、平原的其他民族，也无时不在威胁着以色列人的生存。以色列各宗族仿佛是分散在迦南人汪洋大海中的一群孤岛，被周围的海洋浸蚀着，冲击着。

　　以色列人是来自沙漠的人，他们定居较晚，文明不如迦南人发达，定居迦南后，他们从迦南人那里学会了耕作土地，也学会了对各种农业保护神的崇拜，如巴力、亚斯他录和当地其他神灵。

　　它们是专门司风调雨顺、作物丰收、避祸消灾的神，以色列人慢慢地也对这些神灵顶礼膜拜。

　　以色列人还与当地人通婚，娶他们的女子为妻，又把自己的女子嫁给他们作妻子。

　　以色列人的神耶和华看到这种情形，非常气愤，他决心惩罚这些背叛的百姓。耶和华打发天使来到百姓中间，对他们说："我使你们从埃及上来，领你们来到我向你们祖先应许之地，我要你们永不废弃我与你们立的约，你们也不能与这地的居民立约，要拆毁他们的祭坛。你们竟没有听从我的话。因此我要惩罚你们，我必不将这地的人从你们面前赶出去；他们必作你们肋下的荆棘；他们的神必作你们的网罗。"

　　没有多久，以色列各宗族一个个都先后沦于迦南人的压迫之下，他们在敌人的重轭之下痛苦地呻吟，他们可怜地向耶和华忏悔，求他宽恕和拯救。耶和华是严厉的，但又是宽宏大量的；他使以色列人中不断产生英雄--士师，一次又一次地把他们救出青海。以色烈士师时代前后共出了12位士师。

　　第一位士师是犹大支派的俄陀聂。那时候，住在迦南南部的以色列各宗族被米所波大米（美索不达米亚）王占珊利萨田所征服，过了8年奴役生活。俄陀聂领导受奴役的以色列人浴血奋战，打败了古珊利萨田的殖民军，把同胞解放出来。俄陀聂后来担任以色列士师，40年内国泰民安。

　　俄陀聂死后，摩押王伊矾伦联合亚们人和亚玛力人强占了以色列人的土地，强迫以色列人交税纳贡，事奉他们18年。以色列人不堪重负，向耶和华乞求，上帝把便雅悯支派的以笛赐给他们，让他来承担拯救以色列的重任。

　　以笏是个左撇子，他打了一批长1肘的短剑，藏在右腿的裤管里。以笏带着进贡的礼物去见伊矶伦。伊矶伦的侍从们搜查他的身子，怕他暗藏凶器，但没有搜摸他的右腿，因为通常谁也不在右腿上佩带刀剑之类的兵器。

　　肥胖的伊矶伦在侍从和宫女的簇拥下，懒洋洋地坐在王座上。

　　以笏献礼完毕之后，就对伊矶伦说："请大王叫左右回避，因为我有重要的机密要向您单独面奏。"

　　伊矶伦信以为真，便点头吩咐左右："退下！"

　　侍卫们纷纷退去，留下以笏和伊矶伦两个人单独在殿堂里。以笏跪到伊矶伦跟前，作出一副要悄声告诉他重要机密的样子。说时迟，那时快，以笏以迅雷不及掩耳之势，抽出剑来刺透了伊矶伦的胸膛。伊矶伦未来得及喊叫出声来，他那肥胖的躯体就倒在石砌的地板上，鲜血直流。以笏从里面用门闩把门插上，转身从后面溜走了。

　　有一个仆人看到以笏出来，就问他："国王呢？"

　　以笏回答说："我与他已经谈完了，他现在正在楼上解大便。"

　　伊矶伦的侍从们耐心地在外面等着楼门打开。等了很长时间，仍不见楼门打开，他们就起了疑心。他们敲门，门不开；把门撬开一看，伊矶伦已经死在地上。他们立刻组织人马追捕凶手，但以笏早已回到以色列人那里去了。

　　以笏从以法莲山向以色列人吹起了起义的号角，大批以色列人集合在他的周围，跟着他下山去。没有几天，整个迦南南部的以色列人都闻风而动，打得群龙无首的摩押人落荒而逃。以笏又派人把守约但河的渡口，在那里拦击溃逃的敌人。经过浴血奋战，以色列人共歼灭摩押人近万人，获得了解放。以笏担任以色列士师80年，其问国中安泰。（士2一3）女杰底波拉

　　似笏死后，以色列人又干起了上帝憎恶的事情。上帝惩罚他们，把他们交在夏琐王耶宾手中。夏琐王兵力强大，有900辆战车，还有勇猛的战将西西拉作统帅。他们残酷地压迫以色列人20年。

　　以色列人在痛苦中哀告耶和华，耶和华就在以色列中产生一名女英雄，名叫底波拉。开始的时候，底波拉是一位诗人。她住在以法莲山上，在拉玛和伯特利之间的大路边。她在一棵高大的棕树下接待来自以色列各地的人，给他们出主意，预卜他们的未来，调解各宗族之间、各家族之间及个人之间的纠纷。慢慢地她在以色列人中建立了很高的威望，所有的人都相信，耶和华在借底波拉听说拿弗他利的基低斯有个名叫巴拉的青年，智勇双全，是当今以色列人中难得的将帅之才。她打发人把他请来，对他说："上帝要把仇敌交在你手里，你将成为拯救以色列民族的英雄。你率领1万拿弗他利和西布伦人上他泊山去，我必使耶宾的将军西西拉的战车和全军都陷入你的包围。"

　　巴拉没有经验，信心不足，回答说："你若和我同去，我就去；你若不同我去，我就不去。"

　　在当时的情况下，只有女先知底波拉出面，才有可能使这场战争具有宗教意义，成为一场圣战，才能号召广大的群众。考虑到这个因素，底波拉同意了巴拉的要求，她说："我可以和你同去，可是我要是出马，你就得不到那样高的荣耀了，因为耶和华把迦南人的总兵西西拉交在一个妇人手里，而不是交在你的手里。"

　　于是底波拉就在西布伦人和拿弗他利人中招募士兵。有1万人响应号召，参加了起义军。

　　起义军聚集在他泊山，伺机行动。耶宾听到这个消息，就派西西拉带领夏锁国的900辆战车队和全部精锐之师，前去镇压。大队人马浩浩荡荡，开到基顺河谷，走在最前面开路的是900辆战车。

　　底波拉鼓励巴拉说："前进吧，巴拉！上帝保佑你，胜利就在眼前。"果然，上帝庇佑巴拉，那些日子连下大雨，基顺河谷变成一片沼泽。战斗一打响，西西拉的战车都陷在泥沼里，马匹也打滑、跌倒，迦南军一片惊慌。巴拉率领起义军急起直追，一直追到外邦人的夏罗设，那里是夏锁王的大本营。

　　西西拉全军覆没，西西拉本人则在半路上跳下战车，徒步逃跑，到了基尼人希百之妻雅亿的帐篷前。因为夏琐王耶宾与希百家是世交，所以西西拉想到这里来避难。雅亿见远处狼狈逃来的夏锁军官是丈夫的好朋友，以往那副大将军的派头早已无影无踪了。看到那顾头不顾尾的样子，雅亿几乎没有笑出声来。她迎接出去，安慰西西拉说："请阁下进去坐，不要害怕。"

　　西西拉也顾不上谦让，就一头钻进帐篷。雅亿把他藏在帘子后面，用毛毯把他掩盖起来。西西拉惊魂未定，感到口渴，对雅亿说："我渴了，求您给我一点儿水喝。"

　　雅亿就打开皮袋，倒了一碗牛奶给他喝。他一边喝，一边说："请你站在门日照看一下。如果有人来问你说：'有人在这里没有？'你就说：'没有。'"西西拉实在太累了，他藏在毯子下面，不知不觉就睡着了。西西拉一点也不知道，雅亿是基尼人，是摩西内弟何巴的后代，因此他们跟以色列人是亲戚。在夏锁王统治下，基尼人虽然地位较以色列人高，但他们一样痛恨异族统治。

　　雅亿听见如雷的呼噜声，撂开毯子一看，西西拉已经睡得象死人一样，她推了推他，一点反应都没有。这时雅亿拿起一把锤子和一根木撅子，悄悄走到他身边，把木撅子钉进他的太阳穴。撅子把脑袋钉穿了，一头扎在地里。西西拉就这样死在一个女人手里。

　　随后，巴拉领人赶来了，到处搜索西西拉。雅亿迎着他说："决进来吧，我要让你们看看你们正在寻找的人。"

　　巴拉走进帐篷，看见西西拉躺在血泊中，撅子还插在额角的鬓发之中。

　　就这样，女英雄底波拉把以色列人从异族统治中解放出来。在她的领导下，以色列民族享受了40年的升平时光。（士4一5）基甸--以色列人的刀

　　以色列人一旦获得自由，就又慢慢忘掉耶和华的恩德，崇奉起迦南人的神来。为此他们再度陷入迦南人的桎桔中。米甸人、亚玛力人以及来自东方的其他游牧部族都来欺压他们。每到收获季节，他们象蝗虫一样蜂涌而至，他们的骆驼在以色列人的田地里放牧，无情践踏地里的小麦、菜蔬和葡萄。这些来自沙漠的强盗见什么抢什么，不管是粮食还是牲畜。软弱无助的以色列农夫只好逃到山上去，躲进洞里。土地荒废，生活无着，饥饿和瘟疫吞噬着以色列人的生命。灾难折磨以色人达7年之久，耶和华这才宽容了他们，派士师拯救他们。

　　在俄弗拉住着一家农户，主人叫约阿施，他有个儿子叫基甸。

　　有一天，基甸忙着打麦子，以便储备粮食，好跑到山里去躲避米甸人。突然天使来到打谷场，坐在场边的橡树下，对在树下小憩的基甸说："大能的勇士啊，耶和华与你同在！"

　　基甸说："主啊，耶和华与我们同在，我们又何至落到这步田地呢？我们现在吃糠咽菜，贫困不堪，被外族人任意欺凌。耶和华的神力在哪里呢？"

　　天使满怀希望他说："这不，我不就是来派你去拯救以色列人了吗？"

　　基甸低头说："主啊，我有何能力拯救以色列人呢？我家在玛拿西支派中是最穷的，我在我家又是最小的。"

　　天使对他说："有我与你同在，你就能战胜米甸人，非常容易。"

　　基甸说："可是，你有什么证据能让我相信，你就是上帝耶和华的使者，可以使我蒙恩呢？"

　　天使说："你去预备一只山羊羔和一伊法细面做的无酵饼，我必向你显现。"

　　基甸就回家去取来了作燔祭的礼物。他把肉和饼放在筐内，把汤盛在壶中，带到树下，放到天使面前。天使吩咐他："将肉和饼放在磐石上，将汤倒出来。"

　　基甸全照办了。耶和华的使者伸出手杖，指向肉和无酵饼，立时就有火从磐石中冒出来，将肉和无酵饼烧得一干二尽。

　　行完这个神迹，天使隐去了身影。基甸这才真正相信是耶和华的使者降临到他跟前。他惊恐地呼叫起来："哀哉，主阿。我一定活不成了，因为我当面看到了天使的面目。"

　　天使安慰他说："不要害怕，你不会因此死去。"

　　当天夜里，基甸根据上帝的吩咐，拆毁巴力的坛，砍下木偶作柴，用来烧献给上帝的播祭。作燔祭那只牛是他深夜从他父亲的牛棚里拉出来的，已经养了7年。

　　俄弗拉城里的人清早起来，发现巴力的神坛被拆毁，坛旁堆着砍下的木偶，坛上摆着1只刚杀死的牛。他们问："这是谁干的呢？"

　　有人出来说："这是约阿施的儿子基甸干的。"

　　人群中有人跑去把约阿施我来，对他说："把你儿子交出来吧，我们要治死他。因为他拆毁了巴力的坛，砍下坛旁的木偶做烧燔祭的柴。"

　　约阿施是个有头脑的人，他镇定地对众人说："用得着你们替巴力说话么？谁替巴力说话，就应处死谁。巴力如果真是神，他会自己出来处置拆他神坛的人的……"众人一听这话有道理，都各自回家去了。从那以后人们开玩笑，把基甸叫做耶路巴力，意思是"让巴力自己来处置"。

　　俄弗拉的百姓后来看到，那迦南神巴力并未来找亵读他的基甸算账，变得特别敬仰基甸。基甸成了在以色列人心目中威望极高的人。

　　那时候，米甸人、亚玛力人和东方人，都聚集过河，在耶路斯列平原安营。基甸吹起号角，召集亚比以谢支派的人跟随他；他打发人走遍玛拿西地，使全玛拿西的勇士们聚集他的周围。同时，派人去联合亚设人、西布伦人和拿弗他利人。

　　基甸对上帝说："如果你叫我拯救以色列人，那么我就把一团羊毛放在地上，如果只有羊毛有露水，其他地方都是干的，就说明您与我同在。"

　　第二天早上，基甸果然看见羊毛上挂满了晶营莹的露珠，其他地方都是干的。基甸从羊毛中拧出了满满的一盆露水。

　　基甸又对上帝说："求您不要发怒，让我再试一次。我把羊毛放在地上，如果羊毛是干的，其他地方都有露水，我就知道你与我同在了。"

　　奇迹再一次照基甸的祈求发生了。基甸更有了信心。基甸率领投奔他的32000名以色列人在哈律泉边安营。基甸知道应该如何跟沙漠上的那些强梁打仗。他决定挑选一些精锐成员组成一支奇袭队，出奇制胜地打垮敌人。他对广大群众说："战争是很残酷的，需要流血牺牲。凡是感到畏惧胆怯的人都可以回去。"

　　在他的劝说下，那些意志不坚定的人都回去了。陆陆续续走掉22000人，只剩下10000人。这一万人可以组成精锐部队了，但这对他的作战方案来说仍然是太多了。

　　如何从这1万精兵中再选出最精锐的、最勇敢的战士呢？耶和华给基甸出了主意。傍晚，基甸率领战士们都到河边去喝水。他站在附近的山岗上注意观察他们。大部分士兵都把武器扔在地上，双手捧水喝。只有很少一部分人，俯伏在地上，双手紧握着武器不放，象狗一样用舌头舔水喝。这是一些经过战斗的洗礼，时刻警惕着敌人可能突然袭击的战士。基甸数了数，一共300人。基甸决定用这300人组成奇袭队，其余的奉命回自己的帐篷留守。

　　当天夜里，基甸带着仆人普拉下到平原敌人的营地里去探听虚实。月光之下，基甸看到米甸人、亚玛力人和东方人都布在平原上，象蝗虫那么多；他们的骆驼无数，多如海边的沙。

　　基甸躲在敌人的草料堆后面，听见两个米甸人在谈话。一个人说："昨天晚上我做了个奇怪的梦，梦见一个大麦饼滚入米甸营中，一直滚到帐幕那里，将帐幕都撞塌了。"

　　另一个米甸人说："这不是别的，就是以色列人约阿施的儿子基甸的刀；神已经将我们交在他的手中了。"

　　基甸听罢，受到了鼓舞。他马上回到以色列营中对勇士们说："起来吧！那和华已将米甸的军队交在你们手中了。"

　　于是基甸将奇袭队分成3组，每组100人。他让每人手中拿1支号角和：个空罐，罐内都藏着火把。基甸吩咐他们说："你们要看我行事，我到营地旁边怎样做，你们也要怎样做，我和跟随我的这一组人吹响号角时，你们也要在营地的四周吹响号角。同时齐声喊：'耶和华和基甸的刀'。"

　　基甸和跟随他的100人，在三更之初，才换更的时候，来到敌人营地四周，各就各位，把营地四面包围了。这时敌人都已入睡，一片寂静。基甸突然吹响了号角，发出了进攻命令；奇袭队员个个都打碎了罐子，取出燃烧的火把；他们左手拿火把，右手拿号角，用尽平生气力吹响号角。吹一会儿号角，他们就停十来，齐声呐喊："耶和华和基甸的刀！"

　　天亮以后，那利未人起来开了门，要出去找她。不料却发现他妻子仆倒在房门前，两手搭在门槛上。他以为她是在那里睡着了，就叫道："起来，让我们赶路吧。"

　　他妻子却没有回答。利未人把她扶起来，发现人已经死了，他就把她的尸体驮在驴背上，运回家去。到家后，利未人把受蹂躏而死的妾的尸体切成12块，分送给以色列的12个支派，让所有的以色列人都知道基比亚城中发生的这桩骇人听闻的暴行。凡看到那个女人肢体的都气愤他说："以色列人出埃及，直到今天也没有发生过这样的事情，大家应该想个办法，好好地教训这些恶棍。"

　　于是以色列入的长老都会聚到米斯巴，共同审理这个案件。他们派往基比亚的使者，一再要求交出罪犯，但便雅悯子孙傲慢地拒绝了使者的要求。最后事态发展到了宣战的地步--以色列各支派组成的联军，共有40000人，挺进到基比亚城下。便雅悯本人也从各城派兵增援基比亚，共有26000人，另外还有700精兵：他们都是左撇子，能用机弦甩石打人。

　　第一次以色列人在基比亚人城外安营，便雅悯人从城中杀出来，冲垮了联军的阵营，杀死22000人。第二次联军再次在基比亚城外摆开阵营，又被便雅悯人冲散，死伤18000人。

　　接连两次惨败，联军士气非常低下。他们退到伯特利，在那里禁食一整天，直到晚上，又在那和华面前献燔祭和平安祭。他们间耶和华："我们接连惨败，是应该接着打下去呢，还是罢兵？"

　　耶和华对他们说："你们应当迎头出击，明天我会把他们交在你们手中。"

　　第二天，以色列人事先在基比亚的四周埋下伏兵。第三天，一部分以色列人又到基比亚城外摆开阵营，与前两次一样。便雅悯人出城攻击，一交手联军便往后撤。便雅悯人杀死联军30个士兵，他们相互说："今天他们又要吃败仗了。"他们越追越起劲，一直追到离城老远的地方。

　　这时，埋伏在基比亚周围的联军趁城内空虚，一举攻下城池，不分男女老幼，见人就杀，要把城中的人都杀光。他们一边拼杀，一边在城中放火焚烧，顿时浓烟滚滚，火光冲天。

　　追出去的便雅悯人一看身后基比亚城中起火，纷纷喊到："完了，我们中计了。耶和华要灭了我们啊。"

　　诈败的联军迅速调转方向，回头杀过来，杀死了约10000名便雅悯人。那些追赶他们的人见大势已去，向旷野中四散逃跑。以色列人四面出击，齐力追剿，第四天共歼便雅悯人25000人。

　　以色列人又转战便雅悯的各个城市，杀尽烧光。便雅悯只剩下600人，逃到沙漠中的临门磐避难。

　　战争结束了，以色列人决定在米斯巴召开全族长老大会讨论便雅悯支派的去向问题和处理各种善后事宜。大会召集者向各地的以色列人发出通知："凡不上米斯巴到耶和华面前来的，必治死罪。"

　　大会讨论的第一个问题就是便雅悯支派是否应该继续存在。

　　因为以色列人在米斯巴出发讨伐便雅悯人以前，曾起誓说："我们都不将女儿给便雅悯人为妻。"现在战争结束了，便雅悯人中没有留下一个女人，虽然有消息说有600名便雅悯男子逃到临门磐，现在还活着，但是他们娶不到妻子，无法生儿育女，难道就让便雅悯支派绝种吗？众长老正在为便雅悯支派惋惜的时候，有人报告：基列雅比城的长老没有一人到会。大会当场做出决定：派联军去攻打基列雅比，把所有的男人、已经出嫁的妇女和小孩都杀了。只留下没有出嫁的处女，带到示罗，分配给活下来的便雅悯男子为妻。

　　联军派出12000人攻下基列雅比，杀死了已经出嫁的妇女和所有的男人，留下尚未出阁的处女，共有400名，带回示罗，准备给便雅悯的男人为妻。

　　长老会又打发人到临门磐去给残余的便雅悯男人说明情况，许他们回来。这时候那600个便雅悯人的种子在沙漠中居住已达4个月之久了。这些光棍汉回到迦南后，只有其中的400人得到了妻子。剩下的200人仍然没有家室。长老们吩咐便雅悯人中的光棍汉："在利波拿以南，伯特利以北，示剑大路以东的示罗，年年都有耶和华的节期。你们去，在葡萄园中埋伏。如果看见示罗的女子出来跳舞，就在她们中各抢一个为妻，回便雅悯的地方去。"

　　便雅悯人果真这样做了，他们每人都有了妻子，便雅悯支派才繁衍保存下来。（士19一21）
