《尼希米记》

　　重修耶路撒冷城墙

　　波斯玉亚达薛西20年，哈迦利亚的儿子尼希米在富中作官，他的一个兄弟哈拿尼和几个人从犹大来投奔他。他们告诉尼希米，犹大人倍受凌辱，耶路撒冷的城墙被拆毁，城门被焚烧。

　　尼希米听说这一切，一下子坐到地上大哭起来，他为他的家园，为他的乡亲而心痛。一连几天，他不吃不喝，向耶和华祈祷："耶和华，天上的神，请你睁开眼睛看一看，听一听你的仆人为你的以色列众仆人所作的祈祷。我承认我们以色列入有罪，承认我和我一家都有罪，我们没有遵守你通过仆人摩西所吩咐的戒令和律例。求你记念你吩咐你的仆人摩西说过的话拯救我们，你曾说：'如果你们犯罪，我就把你们拆开，让你们散居于万民之中；如果你们改邪归正，遵守我的戒令，你们就是分散在天涯海角，我也会把你们招聚回来。'主啊，求你仔细听你仆入的祈祷。"

　　尼希米是酒政，这一天他为国王倒酒，国王忽然问他："尼米希，你从来都是笑呵呵的，但最近我发现你老是愁容满面，想必是有什么事吧？"

　　"吾王万岁，万万岁！启禀陛下，仆人实不敢相瞒。我听到我列祖列宗坟墓所在的那座城池被毁，城门也被忖之一炬，怎能面无愁容呢？"

　　"那你有什么要求，就尽管说吧，"尼希米一边心中默默地祈祷，一边对亚达薛西说："如我在陛下眼前蒙恩，求陛下差遣我到列祖列宗坟墓所在的犹大去，好让我重建家园。"

　　亚达薛西同意了，尼希米又请求说："求你发给我诏书、通知大河西的省长，准我从那里经过，直接到达犹大；求你再下一道诏书，要管理陛下园林的亚萨供给我木料，作修城和我自己的房屋用。"国王都同意了，因为神在帮助尼希米。

　　亚达薛西还派几员大将和许多兵马护送尼希米。尼希米到了河西省，把国王的诏书交给省长，顺利通过河西，到达耶路撒冷。

　　在那里，一连三天他察看倒塌的城墙和被焚烧的城门，然后他对犹大的百姓、祭司、贵族和长官说："大家看一看吧，我们的耶路撒冷是多么荒凉，连城门都没有，再想一想我们遭的难，难道我们还要继续在耻辱中生活吗？来吧，让我们重建耶路撒冷的城墙，这是神的旨意！"尼希米把修筑城墙的工程分段包给各家各户，祭司也不例外。耶路撒冷的百姓齐心协力，都想早一天看到修好的城墙。

　　但是和伦人参已拉和做奴隶的亚们人多比雅及亚拉伯人基善却嘲笑他们。参巴拉对他的弟兄和撒玛利亚的官兵说："这些软弱的犹大人要做什么？要保护自己吗？要从土堆里拿出火烧的石头再立一座城墙吗？"

　　站在旁边的多比亚随声附和："他们修的墙，狐狸爬上去都会踩塌！"

　　尽管他们满心看不起犹大人，不断地耻笑犹大人，但当犹大人真要修好城墙时，他们就着急起来，决定合伙攻打耶路撒冷。尼希米发现了他们的阴谋，派人看守，日夜防备。他鼓励耶路撒冷的人民："不要怕他们，神在我们这一边，我们要为弟兄，儿女、妻子，也为自己宝贵的家产而战。"他让犹大人一手做工，一手拿武器，出去打水都带着武器，他本人和卫兵睡觉也不脱衣服，他还让吹号角的人站在他身边："工程浩大，我们在城墙上距离大远；你们听见号角声在哪里，就到哪里集合。"

　　然而，尼希米所面对的却不仅仅是保卫家园免遭外族侵略，城内的危机也日趋严重。许多家庭无米下锅，有的人家典当了田地、葡萄园、房子，有的人家卖儿卖女。尼希米看到这一切，非常气愤，他斥责那些贵族和做官的向自己的弟兄谋利："你们不做善事，不敬神，怪不得外邦人会诽谤我们。我和我的兄弟都将把钱和粮食借给百姓，现在我劝你们把他们的田地、葡萄园、橄榄园、房子，以及借给他们的钱、粮食、酒和油都还给他们。"

　　众人都答应了。尼希米又召来祭司，叫他们起誓，一定照所说的做。他自己也抖着胸前的衣襟说，"凡是没有照说的话做的，愿神象这样把他辛苦劳作而得的时产都抖落的一于二净。"

　　敌人见他们组织严密，防范甚严，不敢轻举妄动，不敢明攻，但想暗取。参巴拉和基善打发人来见尼希米，说："请你到阿挪平原的一个村子，我们谈一谈。"

　　尼希米知道他们是想害他，就派人去见他们说："这样巨大的工程无法停工，我实在是脱不开身啊。"他们一共4次这样派人来，尼希米都以同样的理由拒绝了。第五次打发来的人手中拿一封没有封口的信，信上写着：现在有风声，基善也这么说，说你和犹大人图谋造反。你们修建城墙，你要做他们的王；你还派一些人在耶路撒冷指着你对百姓宣讲，在犹大人中间要有一个王出现，现在这话肯定会传到国王的耳朵里，所以我们请你来，想和你商量尼希米派人去对他说："你所说的事完全是你凭空捏造的，我不会上当。"有人劝他躲在神庙里，将门锁上，他也拒绝了："我怎么会逃跑呢？怎么会躲进殿里来保全性命？"尼希米知道参巴拉他们只是想吓唬他，让他害怕，然后再传恶言诽谤他，所以他镇定自若。

　　以禄月，即6月，25日，城墙竣工，一共修了52天。然后，安了城门，派人看守。尼希米派哈拿尼和哈拿尼雅管理耶路撒冷，因为他们对神忠诚无比。他对他们说："等太阳升起的时候，才能打开城门。有敌人窥探时，要插紧门栓，"耶路撤冷地广人稀，房屋也少得可怜。尼希米召集贵族、长官和老百姓，按家谱来计算每家占多少地方，让人们各得其所。

　　7月初一，以色列入聚集在一处宽阔的地方，请祭司以斯拉讲解耶和华借摩西传给他们的律法书。第二天，各族的族长、祭司和利未人都聚集在以斯拉那里，听他念律法书上的话。书上写着：耶和华借摩西吩咐以色列人，在7月间上山，把橄榄树、石榴树、棕树和其它各种长势茂盛的树的枝条取来，照书上所写的搭成棚。

　　人要住在里面。于是，耶路撤冷的百姓都在自己的房顶上、院子里和宽阔处搭上棚子。从第一天到最后一天，以斯拉每天念律法书，众人也守节七天。

　　这个月的24日，以色列入聚集起来禁食。他们身穿麻衣，脸上布满尘埃，开始忏悔。他们承认自己和祖上的罪恶，念耶和华的律法书，并歌颂耶和华。他们发咒起誓，将遵循律法，遵守耶和华的一切戒令和律例。不让儿女与这里的居民联姻，安息日决不买任何货物和粮食；每逢第七年必不耕种；凡欠他们的债决不迫讨。以鱼列入还自己规定，每人每年捐定量的银子给神殿，以厂在燔祭、安息日等用；每年将第一批成熟的土产和果子敬奉到耶和华的殿里；把酒、面和油交给祭司，收在神殿的库房里等等。

　　尼希米从亚达薛西王二十年到三十二年一直担任犹大的省长。在这12年里，他和他的弟兄们都没有吃省长的俸禄，没有置买田地，他的仆人也跟百姓一起做工。他知道百姓负担甚重，体谅他们的疾苦。犹大国泰民安，百废俱兴，国势蒸蒸日上。尼希米放心地离开了那路撒冷，回到了亚达薛西那里。

　　几年后，尼希米想念故土，又向国王告假，回耶路撒冷看一看。谁知，那里已经完全不是他走时的样子了。多比雅居然在神殿院内为自己预备了一间房子，尼希米气愤已极，把多比雅的家具都从屋子里扔出去，然后他吩咐入把这问屋子打扫干净，将神殿的器皿和素祭又搬进去，利来人本来是在神殿里供职和昌歌，可是因为得不到工钱，他们都回家种田去了。尼希米召回了利未人重新派忠实的入管理库房。安息日里，竟有人酗酒，推罗人还运货物到耶路撤冷来卖给犹大人，尼希米斥责犹大的长官："你们怎么能做出这种事？难道你们忘了你们的祖上就因这种恶事曾受神的惩罚吗？你们要给以色列带来更大的灾难了！"

　　安息日的前一天，天一黑，尼希米就下令关闭城门，等过了安息日才能打开。同时，他还派几个仆人管理城门，以免有人在安息日进城做买卖。但是有些商人和小贩不死心，连着几次住在耶路撒冷城外，想伺机进城。尼希米警告他们："你们如果再这样"我就把你们全抓起来。"他们就再也不敢来了。

　　尼希米还发现有些犹大人娶亚实突、亚扪和摩耶的女子为妻，他们生的孩子不大会讲犹大话，只能讲异族语言。尼希米把这些违反教规的人统统叫到跟前，严加训斥，诅咒他们，惩罚他们，并让他们向神起誓，决不把自己的女儿嫁给外族人，也不让儿子娶外族女子为妻。他还把一个娶外族女子的祭司的后代赶出耶路撒冷。（尼1一13）
