《耶利米书》

　　耶利米的预言犹大王约西亚在位十三年，直到他的两个儿子约雅敬和西底家分别先后为王期间，便雅悯地的亚拿突城的祭司那利米，多次得耶和华默示。他是当时的先知。上帝在他未生出母腹时，便分别他为圣，并派遣他作列国的先知。

　　耶利米在梦中看见一口烧开的锅，从北倾倒，此前又见有一棵杏树。上帝告诉他，犹太人将有灾难从北方临到他们。北方列国的众族将兴起大军，围困耶路撒冷城，同时攻击一切犹大的城邑。

　　耶利米于是受遣到人民中去宣告神的晓谕，让他们不要惊慌，尽管遭受攻击，异族人并不能胜他们。但他们应该反省自己的罪恶，因为他们已经负恩背主，他们所受的灾祸完全是多行罪恶所招致的。

　　上帝通过耶利米说，以色列入违背了上帝之道，它奸诈的妹妹也行淫背道，她就是犹大。而犹大的罪恶胜过了以色列。尽管如此，上帝仍期待它们悔罪归诚，这样才不致被抛弃。

　　但是犹大一族愚顽不冥，毫不悔过，上帝决定使它承受所有灾祸。耶利米哀切地替百姓向主祷告："主啊！我晓得人的道路不由自己，行路的人，也不能定自己的脚步。耶和华，求你从宽惩治我，不要在你的怒中惩治我，恐怕使我归于无有。"

　　上帝告诉耶利米不用为百姓祈祷，不用为他们呼告，在他们遭难向神哀求的时候，上帝必不应允。

　　耶利米自己也预言了犹太人对他的谋害，他们会将他从活人之地剪除，使他的名不再被纪念。上帝也对耶利米说，连他的兄弟和父家都会用奸诈对待他。

　　上帝要那利米去对君王和太后们说，只有自卑才能免祸，"因你们的头巾，也就是你们的华冠，已经脱落了。南方的城，尽都关闭，无人开放。犹大全被掳掠，且掳掠尽净。"

　　上帝再次拒绝耶利米的哀祷，表示"虽有摩西和撒母耳站在我面前代求，我的心也不顾惜这百姓，你将他们从我眼前赶出，叫他们去罢。你告诉他们，耶和华如此说：'定为死亡的，必至死亡；定为刀杀的，必交刀杀：定为饥荒的，必遭饥荒；定为掳掠的，必被掳掠。'我命定四样害他们，就是刀剑杀戮、狗类撕裂、空中的飞鸟和地上的野兽吞吃毁灭。又必使他们在天下万国中抛来抛去，都因为犹大王希西家的儿子玛拿西在耶路撒冷所行的事。"

　　耶利米为了警诫耶路撒冷的犹太人，便在城中宣布了耶和华要降灾给他们的预言，他按上帝的嘱咐，以窑工的陶瓶为喻，说明耶路撒冷的兴盛和覆灭都是主的意旨。

　　殊不知，耶利米的预言触怒了圣殿的总管巴施户珥，他先是毒打先知，然后给他戴枷，下在牢里。第二天，才将先知给释放了。

　　耶利米对他说："上帝并不称你为已施户珥，他称你为玛歌珥米撒毕（意思是四面惊吓）。耶和华说，他要使你自觉惊恐，你又要使你的亲朋惊恐。他们必倒在敌人的刀下，你也必亲眼看见。他必将犹太人交到巴比伦王的手中，那王要将他们掳去，也要将他们杀戮。上帝也要将城中的一切货财和劳碌得来的一切珍主，及犹大君王所有的宝物，都交在他们仇敌手中。仇敌要当作掠物带到已比伦去。你这巴施户珥和一切住在你家中的人，都必被掳去，你和你的众朋友，就是你向他们说假预言的，都必到巴比伦去，要死在那里，葬在那里。"

　　耶利米的话无人认真听取，人们认为他是疯子，对他讥辱嘲笑，连他往昔的朋友，也窥探他，使他跌倒。他们以他为耶路撒冷的敌人，在他身上报仇。孤独之中的先知，悲愤之际，甚至发出了诅咒自己生日的哀怨。

　　耶利米的预言最终传到了犹大王的官中，西底家因为遭受巴比伦军队的攻击，便请巴施户珥和祭司西番雅将耶利米找来。要让他去求问耶和华，解救耶路撒冷。

　　耶利米让这二人转告西底家王，说上帝已经宣布，要让该王手中的兵器，也就是在城外与巴比伦王和围困的迦勒底人打仗的兵器，翻转过来聚集在城中，攻击犹太人自己。城中要有大瘟疫、大饥荒，那侥幸存活的又要沦为巴比伦王的俘虏。他必用刀击杀他们，不顾惜、不可怜、不怜悯。

　　面对生命的路和死亡的路，耶利米劝犹太人出城投降围困的迦勒底人。至于犹大王家，他们既因自己的恶行而受罚，便唯有改恶归正一条路。

　　耶利米预言，约西亚王的儿子沙龙，就是《列王记下》中称为约哈斯的，虽继为犹大王，但必被掳去，死在异乡，不能归家；约西亚的另一个儿子，约雅敬王因为贪婪不义，不能见容于他的兄妹和臣民，他必要死去，且象埋驴子一样，扔在耶路撒冷城门外，而且他要上黎巴嫩哀号，在巴珊扬声哭泣，为了他一切所有的全都毁灭干净。而约雅敬的儿子哥尼雅虽也登基，却注定要被交到寻索他命的、他所惧怕的人之手，那就是巴比伦之王尼布甲尼撒和迦勒底人。他和他的生母将死在巴比伦。他的后裔之中，无一人能得亨通，无人能坐在大卫的宝座上治理犹大。但耶和华是仁慈的，他将从大卫后人中兴起一个公义的苗裔，使他掌王权而救犹大。主也必使以色列安居。

　　总而言之，由于犹太人没有听从万军之耶和华的命，他必召北方众族和巴比伦的尼布甲尼撒王来攻击它。犹大地的居民和周围一切国民将会尽行灭绝，那里将令人惊骇，并将永久荒凉。主要使欢喜和快乐的声音、新郎和新娘的声音、推磨的声音和灯的亮光，从他们中间止息，这全地必然荒凉，令人惊骇。这国民要服事巴比伦王七十年。

　　耶利米说预言话音刚落，祭司和伪先知，还有民众都上来抓住他，他们说："你必要死！你何以敢伪托耶和华的名，竟说这圣殿，这伟大的城成为荒凉之场！"

　　犹大的首领也从王宫来到圣殿，坐在耶和华殿的门口判断。祭司对王说，应该处死那利米，因为他竟敢以预言诅咒这座城市。耶利米为自己辩护说："我说的，你们听见了么？你们若改正行动，听从耶和华的话，那他必后悔，不将灾祸降与你们。至于我，现在你们手中，你们眼看什么是善，什么是正，你们就那样处置我吧。但你们要确实知道，若把我治死，就使无辜人的血归要"你们和这城并其中的居民了。因为我是耶和华差遣来将这一切传达给你们的。"

　　于是犹大地的首领和众民便对大祭司和众伪先知说，这人不应有死罪，因他是奉那和华我们的上帝之名来向我们说话的。尽管耶利米未被处死，但另一个先知乌利亚也说犹大当招致外邦的攻击，犹大王和他的将军们便要杀他。乌利亚逃到埃及，但他们仍从那里将他捉拿回来，杀了以后，弃尸于平民的坟地中。

　　犹大王西底家登基的第四年，另一个先知哈拿尼雅在圣殿中说，耶和华已经允许犹太人从巴比伦王的轭下得释放，耶和华殿中的宝器都当回归锡安。但耶利米说，哈拿尼雅必亡，因为他给犹太人除去的是木轭，但换上的是更沉重的铁轭。

　　就在迦勒底人围困耶路撒冷时，埃及法老的军队也攻进了迦勒底国境。趁迦勒底军撤围，耶利亚想随民众去便雅悯地，但等他到了便雅悯，却被该城守门官诬为打算投迦勒底人，给关押起来。

　　西底家王便打发人将他请到宫中，问他是否从耶和华得了什么默示。耶利米要求国王释放他，至少不要将他送回守城官那里囚禁。西底家便让人把喽押在卫兵院中。

　　可城中的另外几个犹大长老如前面说到的巴施户珥、犹甲、巴示户珥却在国王那里进谗言，说这人来不是为百姓得平安，而是叫他们受灾祸。于是国王让他们处置耶利米，将他下到牢狱里，那牢狱中虽没有水，却是齐腰的淤泥。

　　王宫中的太监叫以伯米勒的，便去求国王。国王心有畏惧，便让以伯米勒去将耶利米从牢中拉出来。

　　巴比伦王终于攻破了耶路撤冷，这是西底家登基为王第十年。

　　破城的那天，巴比伦王的将军尼甲沙利薛、三甲尼波、撒西金、拉撒尼等坐在中门。犹大王见他们在那里扎营帐，便在夜里逃跑。他从王园两城中间的门出去，迦勒底的军队便追击西底家。

　　他们在往亚拉色去的耶利哥平原抓住他。将他带回尼布甲尼撒王的营帐。

　　巴比伦王当西底家的面杀了他的众子，杀了犹大的所有贵胄，再挖了他的眼，用铜链锁了他，要带回巴比伦。

　　尼布甲尼撒手下的人在城中尽悉放火，成千的民宅和王宫毁于一旦。迦勒底人又强迫犹大人拆了耶路撒冷的城墙。所有的犹太人都给掳到巴比伦，只剩下那一些毫无所有的穷人在犹大地替他们种葡萄园和田地。

　　尼布甲尼撒吩咐人将耶利米放出来，他要他们善待他。耶利米得到允许，不随大批犹太人前往巴比伦。迦勒底人又立了犹大的基大利作省长，让他统领剩下的子民并给迦勒底人进贡。迦勒底人军撒走后，亚们人支持的以实玛利又杀了基大利。然后又是约哈难驱逐以实玛利。约哈难不听耶利米的劝告投靠了埃及法老。

　　之后的岁月，亡国的犹太人便在周围大国的夹缝中苟延残喘，耶利米代表上帝向犹太人预言了埃及被已比伦打败，摩押人、亚扪人、以东人等必受惩罚，尤其重要的是巴比伦必遭报应。《耶利米书》的最后一节写到了那被掳到巴比伦的犹太王约雅斤。在被俘三十七年之后，巴比伦王以未米罗达登极，便在当年十二月二十五日允许约雅斤抬头，提他出监。又对他说了降恩的话，准他享有巴比伦城中作人质的众王中最高的地位。他脱了囚服，可以终身陪巴比伦王进膳，每日赐他一份口粮，直到他死去。（耶：1-52）
