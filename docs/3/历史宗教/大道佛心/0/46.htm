钦 奉

无极皇母大天尊懿旨曰：

　　母居瑶池圣地，屡观南瞻部洲，众灵儿沉沦于四生六道之中，难彰显真如本性。嗟叹人生奢华、巧诈难以回归原始自性本心，故流浪轮回生死，不知归返无极仙乡。实令　母心嗟叹，感伤至极也！常倚门遥望凡尘众灵儿，何时得归返？是为感叹！

　　今台疆桃园　母心懿敕道脉崇心堂，自开堂后即行着作大道系列天书，篇篇真理、册册实义，能提引芸芸众灵儿，实际归返的根本。乃是替　母心慈悲化大同，即是能将人世间净化，而成莲花佛国。又大道系列第七册，由佛教四大教主请旨着作『大道佛心』今已完成，实令　母心欣慰，更感诸灵儿等，能坚志完成，克尽职责、勇往不退，此情此志足堪嘉许！

　　母心特犒赏崇心诸神佛等，各得三千大功，并功加三等。另诸灵儿等亦各得三千大功，以全勤者得之。不足三分之二者亦减半。以鼓励踊跃出席者之辛劳！在往后中更要能续着，详实真义之天册，且广引芸芸诸灵儿回归，乃　母心特别期盼，愿尔等崇心诸神佛、众灵儿等，坚志完成，勿负 母心所切盼！

　　　矢勤矢勇　勿怠勿忽　

　　　克志完职　功果自得
