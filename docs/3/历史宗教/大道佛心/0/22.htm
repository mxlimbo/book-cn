第十七回 虚心为谦

大成至圣先师孔 降

八十八年八月十四日 己卯年七月初三

　　修持者入于宗教之后，即在教育、教化的课程中，来成就过往凡尘俗世的经验累积，是把个人修持的方式来导引，并且在修持过程中，给自己有成长经验的累积。如此必须要能真正入于『无所骄心、无所傲气』，亦要『谦虚为怀』让个己真正能累积教育、教化的资本作修持的导引。然亦给予学习者在教育、教化过程中，有所体证，不可存有『骄心傲气』，应以『谦虚为怀』才是真正的修持者。

　　骄必败、傲必坠，乃是修持者最忌讳的本因，因在骄傲之中，看不起其他众生，因此自以为自己高高在上，而且无法把众生平等、修持平等、宗教平等、自性平等的规范来展现，个己即入于败坠之境地，乃是自然的现象。修持者应是越饱学经论，丰富的累积智慧与经验当中，必然是如『稻穗越饱满越低头』，如此『谦必受益、虚心必成器』，因而谦虚之心，乃是一个修持者须抱持着，永远都在学习的境地当中，亦永远皆是保持着半瓶〈桶〉水，任何一个事务皆在学习过程，即是多闻、多看、多想、多虑、多精研，必然是个圆满的修持者。

　　谦虚受益，骄者必败，乃在于个己心性的作为与个人业因、业果的造就，有相当大的关连，如此造业形成，乃在于对他人是一种看不起，甚且瞧不起的作用中，皆是心态举高、不良心思的展现。如此而为必然是在个己无法『虚心为怀』中，来把骄傲之观念、心态化除，因而不论在修持、润生、经商一切凡尘事务当中，待人接物必是相同。因而骄者是败之因、傲者亦是坠之源，乃由个己无法把持着修持者必有的心态，因一个真正的修持者，是必要以『谦卑为怀』的心态，把他人当作是圣贤、仙佛、菩萨等，而自己是满身错误、满心罪孽之人来看待，才能以谦虚做个己成就的因本。

　　谦虚受益——是谦虚卑微能不耻下问，更是以他人做圣贤菩萨，亦是因众生皆有佛性平等看待，如此可在修持中，不论任何时间、空间、环境之中都可以成就。因『卑微自许、视他人高』，乃以个人将修持的真正涵义，去把个己在高傲的心态中，化除了高高在上的作为，是以众生平等而视之！因而即以众生视为圣神、仙佛、菩萨等，仅在于是未来能成就，未来可以达至目标。然而在当下的众生，虽然无法达至其条件与目标，即再往后是可以成就；而现今的我，必然是以众生来做归返的导引，才是真正创造教育、教化的目标。如此必然可在未来中，真正见到自己入于圣神、仙佛、菩萨行列中。

　　谦必受益、满招损。因在自满当中，常会是将自己来膨胀，如此即标榜个己的成就。因在谦是『谦卑为怀』、『谦虚受教』即是真正半瓶水，而『非是半桶水切切跳』〈台语〉。如此必可在『稻穗越饱满越是低头』中做注解。因而可观一个人的习性与品德中，即可一目了然，是真正的大德者；还是仅『半桶水切切跳』而已，因在以个己是否能谦卑受教、谦虚为怀，即是『虚心为谦』的最主要功效。但在宗教修持者，以当今时代即非是如此，往往半桶水强加灌注，必是满招损，损中见真章，亦是自己『无德称大德』乃是让仙佛见笑矣！

　　虚心正是处于虚怀若谷的心境中，可见至天地正气直透心扉之中，开启自性本体的基本。虚心亦由自己入于谦卑受教，谦虚为怀不做作、不虚假，即至真如性的行使，亦可由自己来明白，自己有多少斤两？即是如此。倘一味膨胀自己，又如何能真正带领众生回归于无极仙乡，更能超越了自己的本来，回归、超越入于虚空天界，如此是以谦者受益、卑微低下、低声下气，作众生导引的教育、教化课程，才是真正的实证！

　　满心自以为是，即是自傲自满而不再往上追求，何来能有源源不断的资粮，作下化教育、教化课程。因而自满即难精进成就，必难超越。如此何来能带领众生，彰真如、显佛性！了知谦虚即修持者必须要保持着一颗『虚心为谦』，才是真受益。

　　　谦卑为怀受益得　骄心傲气必败挫

　　　修持把持谦虚心　成就教化众生澈

　　　彰显真如明本心　佛性清盈归本明

　　　天机了悟真实义　回归无极虚空因

　　此偈同有情众灵子共勉之！在成就过程来把持『谦虚为怀』、『虚心为谦』才是真实也！
