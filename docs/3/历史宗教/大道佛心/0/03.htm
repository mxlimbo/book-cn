南无本师释迦牟尼佛 降

八十八年五月二十八日 己卯年四月十四日

　　　南无本师降崇心　释迦牟尼佛心迎

　　　大道系列挽天机　佛来作序阐真因

　　大道者——「摩诃般若菩提萨埵」的本质，即无私无为无我之本来体，作润生修持的因本。是把菩提化为种籽，必可在修持范本中，将往昔的业因业果来化除。因「大道」者——乃由菩提的转化，才能促成在「般若」中，以「大智慧」者，用人生作基本也！ 佛心者——也是圣贤心，神道心，仙佛心，高真心。即一切有情，无情，非情的众生。皆以佛之心来成就其因本的开创，如此作有情世界的教化，才是真实。佛之心即圣之心，道之心，神之心，仙之心，及一切有情的崇德心。更以崇心作基础——又在『崇高的理谛中』，是以崇心作根本，才能符合阐述『大道的演化过程』即是崇心的教化本质与课程。

　　大道佛心——是将崇心的本质与课程，化作资粮来开展，让芸芸佛子皆可以畅真理来成就自己。又能以本身含因的贮藏化作津粮，必能给予众生能有成就的规范。『大道佛心』所演绎的正是符合此一原则，此一真理也！盼之！盼之！勿轻视之！

南无本师释迦牟尼佛降序 于——无极大道院 懿敕崇心堂

天运己卯年四月十四日 戌时

好！为师暂退！
