第廿九回 宇宙大演正

燃灯古佛 降

八十八年十月五日 己卯年八月二十六日

　　　燃开宇宙大虚空　灯影无踪自性重

　　　古道传延亿万年　佛曰宇宙大演正

　　大宇宙乃在当今时代所用的名词，在于古远而言，则为﹁虚空﹂是也！在大宇宙中，正是含包着一切的银河星系与一切的星球。其演化过程最主要的目的，是在作创造众星球间的相互平衡，亦在于是把大千三千世界的一切有情与非情的芸芸众生，来做往上追求更高深的进化。乃是把有情天地间的众生，与万物物种皆能向上追求，进化改造的目的。又在现今地球上人类科技所能创发的世间中，是远比其他星球的科技，要落后千百万倍。因此更需要提升人类星球的人种与万物之改造，也须加以眷顾护佑。是故必要能更进化、更提升，又以科技文明来印证，在其他星球所彰显的灵性世界，乃我地球中的万物众生，所应加以追求进化的目标是也！

　　因大宇宙的进化过程，是融合三千大千世界的一切物种与星球间。又是在人类世界中的众生，所无法了知的过程。如是而言，必要能了知在其他银河系的星球间，来作实证。因在大宇宙的演正，正是导引一切星球间的平衡。在大宇宙的演正过程，其他星球可以做导引。而且可以在规范的本[大道院1]来中，再去成就个己星球中的物种。然而是以该星球的演绎，做向上提升的目的，是以增进该星球的福祉。在大宇宙正是将众星球来做统合，来做导引，使其不失序，相互间不会有不平衡的追击。亦是把星球间在未来中，有相互互辅的科技灵性演化。如此的大宇宙演正，正是此虚空天界，所含包的一种自然的使命。

　　自然的使命，在此虚空中，是以无极做导引回归的契机。因在人类世界中，若无「无极」的导引，即无法提升人类物种灵性的超越。如此乃是将人类世界的芸芸众生，来向上提升灵性与物性的超越。然而在此向上提升，正是科技时代所能证实。若以当今世间而言，人类的科技物性，是无法能胜过灵性的超越，更是无法做统御。因在人类科技范畴中，当今时代所能证实的，仅是灵性〈魂〉的最基础而已。尚且还是无法进入灵性世界，同大宇宙，大虚空，大无极来融合。又在人类世界中，皆因人类世界的贪求欲念的深重，而无法把人类世界来向上提升，甚而无法超越了灵性世界的真实空间。如此即受到三界所束缚，因在人类世界皆有我个己即是「私」也，又在「私」是以自己所能争取到的一切物质来做取向。因而即无法脱离此三界，更遑论能超越轮回，皆在于自己受轮回所左右，即受此观念所残害也。

　　人类世界即有我个己之私也，因而即无法脱离此三度空间，与轮回相续的漩涡。故而在人世间，必是无法摆脱此束缚。即每每受到自己贪求欲念的薰习过程，致使自己无法了脱此三度空间的循环。况且又无法了知「无极演绎，虚空演绎，宇宙大演绎」的过程。则是人类世界根本都无法可以追寻的迷思，与无此资粮了知此世相。然而在当今时代中，由于太空梭已能登陆其他星球探索奥秘，加以人造卫星的发射，探究虚空的究竟，但也只能了知其他星球的一部份而已，亦且仅在其表面，根本都无法进入其内在。故无法了知在「虚空演正」的同时，正是人类向上追求的原动力。故对人类世界的科学家而言，是尚在起步而已。至少须至一、二千年之后，方能证实「无极、虚空、宇宙的大演正。」正是符合人类世界向上追求灵性超越的目的。是在于灵性中，如何来超越回归于无极与虚空中，才是真实性。因而即以「大宇宙的演正」过程，来示现人类世界，如此的人类世界所能了知的一切，尚是肤浅也。

　　银河星系的一切，正是符合此大道演绎的福泽于大千三千世界中，如此在一切众星球的演绎过程皆各不同，每一个星球皆有其特殊的使命，在当今世界的大宇宙中，正是串连着相互因循的互辅作用，如此的微妙关系着一切进化的原本功能。然而在人类世界中的有情天地内，是受此天地〈地球〉的天覆地载。因而即以人类世界的种种一切来做导引，在上苍降下原灵来做改造的同时，更在于其他星球之中，亦有诸多的原灵，也是下化于其他的星球，故而在三千大千的世界中，即以原灵来做统御。如此在大宇宙的演正过程，是不可失序，亦不可有相互的追击。因为在每一个星球中，皆有其外在可以保护的电子光影网路，亦是人类世界所无法用肉眼来观之，且无法用科学来做了知，是在其外层皆受「电子的正负极所牵引与保护」。如此在人类科技的范畴中，若欲往上追求与证实，目前必是无法了知也！但在未来的科技中．皆可一一明朗。正是透过了人类科技的实证，然而在仙佛等，早以在此良机中，来作下化的实证了！大宇宙的演正，正是上苍大道演绎的德泽是也！不使失序，不使相互追击而造殃害，才是大宇宙演正于自然而然，来牵引着人类世界的宗教，迈向更深一层的灵性世界，更向上的提升与超越。
