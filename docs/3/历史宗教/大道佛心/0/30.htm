第廿五回 悟白骨观念

南屏道济古佛 降

八十八年九月年一日 己卯年八月十二日

　　　南屏真理渡一贯　道济古佛理无烦

　　　白骨观念展世机　宏扬佛心归天返

　　由心念意识进入于取向的行使，必要有贪求欲念的促成，才能有所行动与作为。因众生皆会在对自己有利益的举动中，来做个己在人世间润生的添加与要求，乃是人类世界所必须要有的取向目标与空间。因在贪求欲念的行使，必然给予自己有行持的原动力，即在于贪欲整体诉求中作分别。

　　一切的作为是入于自己来行使，必是以自己之所作所为的持续，也是自己造业因、业果自己来承担，更是符合因果法则的平等性。又对自己所作所为，必须自己去领受，且由自己来承担自造业因、业果的偿还及受报，任何人皆难替代之。不论是福是祸？皆是自己所招惹之本来事因，故而是自己所愿意，又心甘情愿来承受，乃是自己入于因果法则中。一切的作为，亦由自己的取向行使与受报，由自己的甘愿心，来做自己在人类世界中，所有的作为之本能。如此必要以自己做未来祸福的相等性，倘若自己都无法了知，自己未来的行持入于有情世界中，皆必须要有『无常观念』的循环。在于往后人类或物种的众生，皆会是白骨一堆，而归还于大地，乃是人类世界中，所必须要有的观念。

　　在了知无常的法则中，再入于无常的观念，更须明悟有情天地的演绎过程，亦应由自己在『无常观』中，明悟『天地德泽与运转之无私』。任何圣神、仙佛等，或是人类众生、物种众生，皆必须要将此身躯在轮回中来抛弃，而归还于天地间，能遗留的仅是身躯内在骨骸而已，其馀皆受尸虫侵蚀，仅存白骨一堆。若是以当今时代环保意念，必须用焚化，较符合卫生条件与存纳空间。若以土埋方式，能将此身躯归还大地，亦存白骨一堆，又有何可计较的！若是几年后再检骨一次装罐，再复有时重焚乙次，焚化后所得残留的皆是如粉末一般而已，尚且可以当作肥料来滋养大地众生。

　　如此是必要了知在『白骨观念』中，是将身躯所有的外在皮肉，由眼耳鼻舌身入于接收，尚须具备地之坚实性，来支撑骨骸的站立。乃在于须由皮包肉之裹层，方能让血水有流动性，因在血水是促成人体新陈代谢的主要功能效益。乃以血水承受皮肤之包裹，必须具备了肉体之存在，然而肉体之存在，必要能通过血水加上风息之流动，方能将火气来导引，即能温暖此身躯之肉体。如果将人体来解剖殆尽之后，必仅存骨骸而已，因而骨骸之坚实，乃地之主体功能，有主体必有附体。

　　在『地水火风』即为四大假合，而此四大必在一切可以活动的任何物种中，皆是如此。必要受『地之坚实性』、『火之温暖性』、『水之流动性』、『风之吹动性』，乃融合四大而谓之假结合体。但在实相界中的当下，能谓之假体否？乃真实不虚的驱体，亦是在当下的上升与下坠之分界点，若上升必要作改造的资本，果若欠缺了此一假体，而谓之修持能成，那无异是缘木求鱼也。

　　白骨骨骸之形相，在在显露了一堆可怖惊栗之存在，世人皆惧怕而谓之不谈『白骨骨骸』。但在佛家之精神领域中，是谓之能入于真实世界的了脱，倘若白骨不存在，其人应是尚在人世间，或是早已归还大地，且成为肥料来滋养大地。但以白骨之存在，亦是将此已过往的人生，作一个统合而已，在人类中因是曾经来过一趟了，是该如何，上升成就否？是下坠沉沦否？皆由白骨骨骸中见真章。

　　白骨之存在，是警惕已经来人世间走过一回了，下次是否有此良机？再来人世间作润合，皆必须要观视其个己之所作所为来决定，因此任何润合与作为，即以白骨来作结束。人世间的一切种种行径，在此则是作一注脚、作一结尾，亦由此来划下人世间的一切休止。『白骨』乃是必须要了知在形骸中，是要体悟人生之无常，由生老病死与地水火风的四大假合，亦仅是人类惧怕之物体而已。以此当作警惕，是沉沦、是上升，皆可了知自己的所作所为，必要能明晰的是以白骨作警惕、作修整而已。

　　近来台疆地区，迎奉释迦世尊的牙齿，而谓之『佛牙』。可谓之万人空巷，其所受到的礼遇尊敬，尚且不仅是对世尊之景仰与光荣而已。亦可由迎奉『佛牙』，了知同样出生为人，佛牙能流传两千馀载而不坏否？又如何由『佛牙』亦是白骨当中之一，白骨的一种躯体的组合而已，为何佛牙能受后人景仰尊荣，而我们自己却不能？如何由学习佛陀的精神，方是能对得起自己在此世当中，去领悟必要明白『

　　白骨观念』的展现，不要谓之佛陀世尊乃十分遥远也，仅是近在咫尺而已。

　　　佛牙迎奉白骨骸　两千馀年到今哉

　　　万人空巷齐景仰　白骨观念作自裁

　　　超脱轮回入回归　了悟世相悟理追

　　　无常观念启世迷　白骨观中形骸堆

　　此偈同有情众生共勉之！人为可成，不要看轻自己而消极，反而必须要积极的投入真实的理谛中作实践的本份，才可能超越于自己的本位也。明否？
