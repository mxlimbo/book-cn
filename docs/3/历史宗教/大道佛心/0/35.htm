第三十回 觉察与休歇

大日如来佛 降

八十八年十月八日 己卯年八月二十九日

　　　大开禅宗来演绎　日常平等是主体

　　　如来如去更真实　佛言觉察休歇意

　　由大宇宙的演绎过程，来了知在人世间，必须要有向上提升，自我灵性的超越与彰显。是不可在人类世界中，仅只有自我的生存取向动机与过程，在人类生存的当下中。若无法能向上提升自我灵性，往上超越的同时，必然会受轮回的相循左右，让自己入于永远都在沉沦于六道的漩涡内。然而若以佛家思想观念，来作主题、来作演绎。崇心所着作的大道系列天书当中，是已入于桁宗榈谋咎澹故而在大宇宙的演化过程。则必要能入于真正向上提升，才不再人类世界中，承受永续的轮回。

　　禅宗的本质中，是要很具『觉察』的洞悉能力，故在于能『洞悉』此世间相的一切来龙去脉。是作人类生存向上的提升，在觉察者是了悟一切的世相。能了悟人类世界的相循轮回，是必要能将人类的生存过程，在生活中不可执视于仅是为了生活而生存。以致无法脱离三度空间之相循轮回，其基本的原因亦在于此。因人类世界正是欠缺了此一『觉察』能力的行使，故『觉』者，是以洞悉明白人类生存的一 切，『察』是观看此世界的实相过程作了知、作体证，又与天地演绎的过程，同无极、同虚空、同大宇宙的演化过程相呼应。是必要能了知，其在给予人类润生的过程中，要明悉了悟一切天地，相循的运转过程，乃是实真也！

　　因天地的运转过程，正是同人类世界的芸芸苍生，有一份不可抹灭的相循关系。如此在人类的生存空间，若无法能洞悉天地的演绎过程，即难以了知如何回归，如何能超越，乃在觉察能力的展现。觉者是能洞悉明白一切的世相，察者是体证一切过程的辅助作用。即是洞悉体证人类生存的世相过程中，必要能作个己心念意识的主导，才是真正入于「禅宗」的本体，亦在顿悟与渐悟之中来分判。又在顿悟是资粮很充足，很丰富，即可闻一知百，闻百知千，便可了知一切事相的本体。但在渐悟之中，要能了知的，正是把经验的累积来做真实的资粮，是由一点一滴来堆积。如此瓒傥蛴虢ノ颚樵蚴侨死嗍澜纭⒁磺惺老嗟木跤朊裕竞与争（进与增）的集合点。

　　觉察力量，若在心念意识中，必要了知在唯识观念中，是如何来真正达至能了知一切世相的根本性，因在觉察中，人之起心动念是会影响心念意识的行使。亦是无法把心猿意马来驯伏，如此杈跣脑常复性初榧次薹ǔ闪。因在起心动念的当下，是无法把心念意识来栓住与驯服、又何能杈跣脑常复性初椤⒓词苄脑骋饴淼谋继凇Ｈ绱硕言，人类的心念意识，若不经由栉ㄊ豆勰瞀槔醋髦魈濉蠢怼担那根本都无法能了知，天地演绎，无极演绎，虚空演绎与大宇宙的演绎过程也！

　　如何在心念意识刚刚升起的当下，来把其栓住，使其驯服，则必要在休歇中使其不生枘钪动橐玻〖匆驭栊菪槿米约盒哪钜馐叮来转换为良善方面去行使。亦是以良善道德来主理一切，因在有情世界中，是必须要有所作为，方能有为，才可以进入「无为也」因此栉尬檎是一切演绎所主理的本体。在心念意识的刚刚升起中，如何导引往良善的方面去做行使与改造。则是觉察力量所必须要有的基本条件与观念。如此在一个念头刚起的过程中，如何观察、又如何审视、则在良善与不良之中，来作分判。

　　觉察力量中作分判，必要是能让一切的芸芸众生，皆受福祉来荫护。故在一个恶（不良）的念头升起的当下，必要作休歇停止的动作。亦是要能入于真正的了知在唯识观念中，把其过程来作统合，来作消融，来作清除，即是人类世界所必须要有的动作是也！因觉察正是由根基锋利，来主导其可以判别的心念意识中、下化于个己的作为与思想内。然而一切有情众生，尤其是人类世界的芸芸苍生，若无觉察力量，即无法有休歇的动作。如是而言、必要在一个念头的升起前来作断除。即是菩萨畏因、众生畏果的最佳写照。

　　「觉察与休歇」，是在唯识思想中，如何进入根基锋利之内，不可存有钝者，即是生锈不利也。乃是无法体证一切事项的来龙去脉。因在觉察力量，正是禅宗所显示的根本基础。若无法能有当头棒喝的观察力量，即无敏锐的察觉眼光，更是钝钝钝也！休歇正是以钝钝钝合成的主体，让自己在念头未起前是如何断除，能长保永续，不可须臾离。乃是对自己的具备资粮一种展现，是根基锋利或是钝者之流，皆观自己敏锐的觉察力量而定夺。

　　休歇――正是停止不使向前，乃是作顿悟的基本要求，正是人类所欠缺此一根基。才让当下的众生永远沉沦于六道轮回之中，休歇不前乃菩萨道所应具备的根本功课与动机，不能休歇，即永在轮回受漩涡的盘旋永不停止。

　　　察觉力量观世相　了知天地泽无漾

　　　演绎过程皆真实　休歇断除入本章

　　　泯心存境精又微　存心泯境再可追

　　　心念意识转含因　禅宗教化永追随

　　　大道佛心演正宗　真佛心宗真理崇

　　　崇心更有大天机　回归超越是自理

　　此偈等供有情众生细研，明悟人生主体，如何断除在起心动念中，觉察与休歇是有其存在的必要，亦是对自己在起心动念中，如何成就自己，才是真实性也！
