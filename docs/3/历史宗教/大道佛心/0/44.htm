跋三

南无阿弥陀佛 降

　　　儒宗圣教 演正天下　众灵归返西方极乐世界

　　　真佛心宗 真理引渡　大地苍生回归无极虚空

　　真欣喜！真心喜！真欣慰！佛门能在近二千馀年来，皆难有能真正做教化课程的资粮。而今上苍的懿命 无极大道院 懿敕崇心堂，能成就自己的升华，更是成就芸芸苍生的超越，是以作回归的基本，是以作升华的资粮，乃我佛门引以为光荣！能在崇心来着作，一系列的大道天书，将佛家的精髓下化于鸾笔中。盼我佛门子弟等，不可执视最高殊胜、最能回归，此乃已是分别彼此。

　　天下宗教原本一家，只因大地苍生人种之不同，个人习性亦差异极大，故而上苍将此大道分别演绎，另派圣佛降凡开宗演教，而分儒、道、释、耶、回......等宗教，是以让芸芸苍生，有一个选择依持的目标，是以自己意愿所行使，即是让自己心甘情愿的，去做学习与教化的课程，来传承宗教的命脉。

　　大道系列，皆在在演绎我『佛家整体精髓』，虽是以『儒宗立教』，所用又以『道家规范』。故在三教本同宗，五教共同源的原来本因中，是不可轻视其他宗教派脉的教育传承。在每教皆有其优点亦皆有其缺点，如何能融合、如何能契入，而不分彼此。更能相互尊重、相互融合，更且能相互赞叹，才是上苍平等的原则，是以归返所必须的基本要义。否则若有分别、我执、诽谤、贬低、轻视，即难以上升极乐世界，更是轮回下坠的种籽。盼了知在天界，是无任何一位圣贤仙佛有分别心，因此必要相互融合，才能言回归超越！

南无阿弥陀佛降跋 于——无极大道院 懿敕崇心堂

天运己卯年十月初五日 戊时
