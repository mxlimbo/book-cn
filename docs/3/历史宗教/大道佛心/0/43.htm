跋二

南无药师如来佛 降

　　　佛教教尊降来堂　药师如来真欣赏

　　　大道佛心耀禅宗　消灾解厄兆吉祥

　　真为崇心道场感到庆贺，能独得佛教四大教尊等与诸天高真仙圣神佛的眷顾，且能在一册册的大道系列天书中，彰显有情众生成就的基本条件。其因源是对上苍一份责任行持的永恒不断，且能克服任何困难与横逆、险阻，更且是化除了诽谤、轻视中能坚志达成缴书，是对上苍一种诚信，亦是对苍生有一份教化的责任。

　　可喜可贺，皆在在的通过了，任何的考程与磨难中更能成长、更能坚心。乃崇心等诸神佛与学生等的坚定毅力所促成。实为可喜可贺、亦可感心！在我佛门中近二千馀年来，甚少有之！更甚能阐述我佛门精髓要义，下化于天书中更为创举。乃在于我佛门子弟等，皆有偏执门户唯我独尊的倚见，无法将佛门的真理，普遍散布应用下化于芸芸苍生所致之，诚为嗟叹也！

　　嗟乎！能有崇心传承我佛家精髓典要，且能把唯识、禅宗、密宗等心法，能融入于大道系列的丛书中，是为可安慰矣！又以浅白易懂的文字般若，来作下化的智慧结晶，是以崇心能达成之。更要不分彼此、不分教派，皆能融合更为圆满，是为至盼，更能长远坚志，光我佛门的灯塔，照耀芸芸苍生。

南无药师如来佛降跋 于―― 无极大道院 懿敕崇心堂

天运己卯年十月初五日 戌时
