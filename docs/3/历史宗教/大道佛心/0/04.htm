东方琉璃药师如来佛 降

　　　东方世界琉璃国　药师如来是真佛

　　　大道佛心来演绎　赞日成就个自我

　　崇心能着作大道系列天书，一册册相互接续，已完成着作共六册了。今再续接第七册『大道佛心』，是一切众灵子的根本，来成就于佛菩萨、仙圣神道的本体，是把圣贤心、佛菩萨心，引用于个己自性中来成长。

　　佛心是佛菩萨最重要的根本心，又在佛心的一切，是佛的明灯基础，是把心中的明灯来展现，照耀于三千大千世界，传承永不熄灭。在『佛心』中以个己成就为根本，以下化众生为基础。让有情世界的芸芸众灵子，皆有成就的契机，乃是以此作因本。又如何来成就，先具备资粮位，才是将资粮散发〈化〉于有情世界中。

　　佛之心——一切皆以芸芸众灵子之成就为根本、为基础。又以佛菩萨同众灵子皆一脉相承，绝无法脱离有情众生，否则又何来让众灵子能成就莲花佛国。在佛之心即仙之心，即圣之心，三教所阐述的本质亦在于此，是不分儒道释耶回等，皆以无分别心而论之，亦不分宗教派脉也！才是真实！

东方琉璃药师如来佛降赞 于——无极大道院 懿敕崇心堂

天运己卯年四月十四日 戌时

　　为师亦暂退
