第卅二回 泯心泯境 存心存境

南无本师释迦牟尼佛 降

八十八年十月十九日 己卯年九月十一日

　　　人天教主降来堂　本师释迦真心赏

　　　存心存境化无尘　泯心泯境再述真

　　由存心泯境而入于存境泯心之中，乃我佛门禅宗所必须要有的敏锐观察力。故要先具备相当深厚的功夫才能致之！然而在禅宗所显示的，正是把一切可以化除的心境，来做是一种向上提升的资本。因而即以泯除了此色心同外界的色相做相互结合。又在于是对敏锐锋利的根器者，才能言及。否则若按照一般平凡众生，皆难有实质观察的敏锐力与功夫。

　　一切的观察力，是要破除了一切虚妄思惟的建构。若无法达至真正了知人世间的一切世相，即常受人世间的形形色色所左右与影响，更是无法破除在『心色』两法中，能有实际的资本，更是无法对自己能提供正确回归的资粮契机。又众生是对自己生存的过程，是无法能了知人世间的一切世相。即难以对自己的生存，给予成长改造的资粮与契机，乃是对自己一种相当残酷的事实。且在众生皆会认假为真，又以人世间的种种行径促使自己在凡尘中，一而再再而三的沉沦。

　　泯心泯境必是破除一切虚妄与实相的世界，然而若不能了知泯心泯境，则是无法能有向上提升的资本。必在于每每是受色相来左右，因而泯心泯境是必要在禅门中来发扬。即是『不动心，不动相』又在一切法界，皆是有形有色的实相界，必受外界来左右，不论任何人皆无法离开于此色相。故而永远都在轮回之中。然而在佛、道两教，要能提倡对心性、对灵性的超越，则是必要来超越泯心泯境的世相，否则必受色界所左右，且在于是对一切世相，由眼观色的世界，即会受到色界所左右，更难以超越色界，难入于气理两界，但在色界中，更要了知的，正是人类生存的实相世界中，若没有泯心泯境，亦无法达至超越提升的条件。

　　泯心即已无此心之动念，泯境亦无境之动念，乃化除了对心动与幡动的过程，皆在于能把此心之动来消除。更断除了实相与虚妄的心境，然而在心的世界中，仍是处于一切清静与净滤。因在于是消除了此心的动摇，亦在自己是对外境而动心．故无法能向上超越，如何在泯除此心中，能不受外境所影响，但外境仍是存在，故难以泯除此心境的外驰与攀缘，即是入于对一切世相，又再有更深一层的提升。

　　存在于外境的一切，即是存境的过程，又如何能了知存在于心境中，是如何能对法界的一切世相，来做更深一层的提升，则必要能了知是在『存心存境』的过程中，让自己能明了一切世间相，而并非只是入于世间又对一切世间相无所了知，无所判别。乃是对外境更是通过了经验的累积，且是智慧的借与，才更能了知一切世间相的本体。故在于存心存境，正是禅宗本体的基本架构，因在一切真实的实相中，天地的长远，又人类的短暂，又能如何！必要是靠自己去做成长，必要能给予自己在未来，通过三际迁流的过程，是在当下中去体证人世间的种种行径过程，才能保有一份对自己向上提升的原动力，如此来言，则是具备给自己本身做超越与提升的资粮。

　　存心即是对外相一切能了知，存境则是通过外在的考磨与历炼，才能达成至炉火纯青的阶段。众生可观在于年长者，正是累积了人世间，一切的酸甜苦辣的经验，更是通过了生住异灭的过程，才能累积丰富的人生经验。故而其考磨亦较减少，又在于初生之犊亦是不畏虎，更不知天高地厚。然而刚初入社会的青少年，皆因经验之不足与不丰富，才导致在人世间有种种的考磨。更是对自己无法能累积，相当丰富的人生经验，故而常受考程来磨此身心，能谓之天之考否？试思即明也！存心存境正是超越人世间的基本架构，故已经是入于『见山不是山』，而入于『见山是山』的第三层次。即在于能了知一切的世间相，且是早已开悟了，更明白人之灵性的成长是如何了。『存心存境』正是由人世间的一切世相过程，来做更上乘的追求，才能达至也。

　　泯心泯境，存心存境，是追求向上提升的基本资粮，在主体架构中，通过了人世间的种种历炼考程才能达至。故而向上提升中，若无法能有实际的资粮，即根本都无法有实际的经验过程。即难以对自己与众生灵性的提升，有实质的效果。在崇心的大道系列丛书中，将是一册册来着作，亦是累积诸多的天书，能下化于有情世界的芸芸苍生，才是崇心的基本圣务与使命。今日真是为崇心感到欣慰，能长远在大道系列，有一份真实的教化资粮。必在未来中，能实际给予不论儒、道、释、耶、回的芸芸众生，提供可以成长的资粮与契机。
