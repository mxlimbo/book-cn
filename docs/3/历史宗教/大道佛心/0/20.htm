第十五回 真心为正

通天教主 灵宝大天尊 降

八十八年八月三日 己卯年六月廿三日

　　　通天至诚化灾劫　教主德泽润众业

　　　灵宝天尊降着书　真心为正业消灭

　　众生无法以修持的方式，同润生两相结合，常在自己的周围，将个己入于轮回之限制。即是受到个己业因、业果、业障的牵缠，即因此常在反覆相循作用中，将个己陷入难以转圜困境且受其束缚。因众生是把修持与润生脱节，然而理应是两相结合才是正确，为何会是如此，即因无法了知，修持与润生是不可相互背离。

　　众生常会为了修持，即脱离生存的过程，又此境界亦是由个己作为中，认为修持是为很高超，亦是难以捉摸。因为有此观念，更是无法同润生相互结合在一起。因而修持是高高在上，而润生则是卑微，亦是形而下之作为。如此更无法了知在润生，即是正统的大道，为何修持与润生会两相背离，乃是无法了知生活即是道、修持也是道，而无时无刻皆在做润生与修持 。因而不论任何环境、地点、时空皆是如此，否则即脱离修持与润生的基本范畴，因此常会把『修持与润生』两相背离了，如此人世间的生存方式，真是无可奈何也！

　　因是无可奈何之故，在于修持中即不明了，又在修持上，该是如何来作为才是正确。乃是常在修持与润生的规范中，增加了诸多的教条与戒律，如果在自己的润生与修持中，从小即开始作计划教育，必然是可以明白，在修持与润生中是该如何？即要了知在规范与戒律当中，是针对世人规范与戒律所要求者。如此即必须要了知修持与润生皆是一体如是，更需要在人世间，用真心来彰显在润生与修持的体证。因在一切有情众生，接受业因、业果、业障来讨伐与牵缠的同时，都无法用真诚忏悔心，去化解个己的业因、业果、业障之消弭。因在众生若无法用真心忏悔，是难以达至化解的目的。即是让众生受到偿报的痛苦过程，更是无法了知业因、业果、业障之讨伐，其本身则必须付出，比世间人更加痛苦过程，来的更加厉害有之更甚一、二十倍之苦磨，而此仅是为了讨伐受报，然而此嗔恨心，若不受偿报，因果法则亦难了脱，如此众生即在因果讨伐的当下，应了知因果偿还之可怕，必了知痛苦的折磨，又是多么无奈！但在业因、业果、业障讨伐的同时，所付出的代价，何尝又是在现今的芸芸苍生，所能了知与体会的。

　　真诚之心即是作为人世间，待人、处世、接物、应对，所必须要有的心态。不论是对上苍与诸仙佛的祈求，或是面对冤欠之讨伐，皆是要用真诚心，因是以真心为正，而一切外在的色相，正是通过人生的思想观念，才有作为、才能致之。如此必须要了知真心是可以化除，一切业因、业果、业障讨伐最基本要素与化解的效果。又如何能以真诚的心，来面对上苍，与面对仙佛的化解灾劫，在在都必须具备此一真诚心，经由仙佛从中调解，才能将业因、业果、业障，消弭到最低的程度。如此必要了知在修持中，是要保持着一颗『真诚心』，而此『真心』即不偏不倚，更有正导方面之作用！

　　直心可成佛，真心为正，无心化贪欲，虚心更谦实，一切润生与修持中，势必需要保持此一『方寸之地』，使之无污染与垢秽，是最能彰显个己的所作所为，亦绝非是受外在的一切色相来引诱，因而把修持同润生两相结合，是不可背离的，否则真心为正，必偏悖错误。因在真实诚挚的心态中，要如何把此真诚之心来导正，方是对自己的未来与当下中要如何行使，亦即能以『真诚之心』为导正的主要目标，给予自己可以转圜业因、业果、业障之牵缠与折磨，必须是始终如一而无憾也！

　　真心为正是修持必须要有的功课，若欠缺即难以达至成就，众生如何把『真正的心态，做正直的行使』。才能在修持中，不可偏离其规范，因是做修持与润生的基本条件。如何将个己『心寸之地』来导引方是真实，否则若论修持真心无法为正，又如何能给予自己在修持润生中，达至成长与学习的成就，润生是十分重要，因可化除心态不平所产生的局限，必将『偏悖化除、而居正中』。众生了知否？细思之，可悟也。
