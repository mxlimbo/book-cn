第九回 立杏坛之规范

空落笔天尊 降

八十八七月六日 己卯年五月二十三日

　　　虚空天界来崇心　落笔天尊永正情

　　　立杏坛之规范章　大道佛心宗教因

　　立――立者是以正正当当之作为来行使之！给予现今之修子，立足于教化与教育的课程，因而必要能建立规范的条理才可以促成。又在于宗教是弥补社会、国家教育法律之不足，当今人心处在末法时期，常因福报之具足，而不思修持，故心灵则是十分之空虚。如此即是欠缺了那一份灵性提升的滋润，为弥补社会教育、国家法律的不足，在宗教的教化、教育课程中，确是十分的重要也！

　　杏坛――古曰私塾，现今乃是学校与学院的通称，因杏坛是儒家思想，所建立的名词。又儒家是以教育与教化的传承，普及于当今现世的众灵子，因而需要儒家思想的建构。又以大学、中庸两教材为主，旨在把大学、中庸之课程，应用于教育与教化的弥补。必须要以宗教之心灵洗涤，才可能促成学习经验。在现今时代中是可补教育、教化之不足，乃在是对众生生存的润生过程中提供资粮。往昔心灵的滋润提升则是欠缺良多，因为教育与教化不足，又以宗教杏坛的修整，更且是把心灵教化与教育作改革。如此必须要明白在宗教学院之传延，是有必须性、必要性的。杏坛乃学院，若于当今欧美国家，教育是活到老学到老，甚且早入于成人教育的课程。然而对于宗教教育的课程虽有之，但常在心灵超越的课程则是欠缺。反观中土国家之中，更是以润生动作为第一，此乃是实际也。但更在润生具足的同时，常会感到心灵上的空虚，有时亦会感觉到自己已具足了一切物质享受。实则在心灵中则是欠缺了那一份真实感的超越，又欠缺诸多的资粮。因而常会感觉在学院与学校教育之不足中，而入于宗教去修持，去学真理，去做人生心灵归宿的改造。如此杏坛在人世间中，确确实实的有其存在的必要。

　　之规范――又崇心的着作，乃是针对当今时代，宗教教育与教化改革之创发者，是以上苍所要求的规范、规章中作主导。然而在此万教齐发的同时，每个宗教派脉皆会自我标榜、自我推销、自我膨胀。如此是给予众生，一种学习与模仿的路径之选择，又在学校与学院教化、教育的促成中，当今儒宗神教已是式微了，故而上苍再开启『儒宗圣教』的传延中，是以未来之际对众生有一依循与指标的规范场所，故要求崇心道脉来建立规章与典则，以作为往后教育与教化的传承，能永续不悖。如此亦是对崇心道脉之要求，要建立规范、规章，以符合未来众生所需要的依循而设立，而此规章是把人生之典则来建立，是把章程来建构。因而崇心若不能有一规范、典则、规章的系统是难以在未来中，能有十足引渡众原灵来归返，来成就的契机。因而才会严格要求，正是对崇心道脉十分企盼，以后能建立『佛心宗脉』之传延。因佛心亦是崇心，更是圣神、仙佛、菩萨心。如果规章不能建立。又如何在未来中能引导众原灵来归返否？

　　立杏坛之规范――崇心能以此来建构未来的『无极大道学院』，是上苍对崇心道场的肯定，亦是对崇心道脉，在此近三载之磨难中见真情，又是在未来佛心宗脉之传延奠基，如此在在的种种契机，正是要求崇心诸贤生等，能立杏坛之模范典则规章中做注解，以利未来不论儒、道、释、耶、回、天道，亦不论无极先天、三清道脉或是母娘系统、慈惠系统、天道系统之建构中，能了知是如何来导引众原灵，超越三度空间而回归于极乐世界、无极仙乡、虚空天界，乃是必要的真理实证。因此对崇心诸贤生严格之要求，必观诸贤生与主事者之配合度而抉择。因在时迁法亦迁的当下，是要求崇心能做建构基础，并且以企业化、人性化、生活化来成就自己与众生。才能开展未来对宗脉的传承，如此『儒宗圣教』将在千禧年后正式开展，欣欣向荣，迈向人间光明之社会境界。

　　立杏坛之规范中――倘若崇心不能严格订定规章，必在往后中形成另一股散沙，那又如何来做众生导引的指针。然而若不能在此当下时机中，做规范、章程、典则，必难以在未来中给予众生等，去做修整之条理规范。因而崇心的『儒宗圣教』所着作之天书中，将入于禅宗唯识宗的主要课程资粮。又在『禅学宗脉』与『唯识宗脉』的理论、思想、行径的建构，是真实导引众生做回归的资粮。因而在实行位阶中，必要是有条理不紊的章程，必要能以人性化管理，企业化的经营，生活化的润生，又三者皆可相互配合，更能在千禧年后来真正开展弘化的契机。更能正正当当把上苍的大道系列天书，来下化于有情世界中，如此崇心诸贤契，是否感到任重而道远否？责任又加重否？心情是如何也？

　　倘若崇心不能做到如此严格之要求，何来能导引未来的众生，有实际回归的契机，因而必给予诸多的考程与磨难，在在的显示着，主事者与诸贤契之配合力来定夺，更是众为师等，在在的背负着一切圣务成败的关键。若能顺利的推广，正是一种自我成长的契机，并且对众生能有十足之交待，亦是对上苍一种负责。如此众为师等与崇心诸贤生等，皆是圣务与生命的共同体，正是背负着成功与失败的重大责任，更且是息息相关且不可分离。稍有偏悖，即给予惩罚与磨难，若能成就亦是对未来的社会有一贡献。因此即要求不求功德，不为个己，不为私情，而能利于天下的苍生，才是真正的行持。如此即要求诸贤生能通力配合，在崇心道脉中的众学生等，必要能了知个己有使命、责任、义务之行使，更严格要求，若不能做到于此，不论主事者或是众贤生等，必自请处分，如此可知主事者之肩担责任是十分繁重也！明白否？

　　语重心长，盼能成就于大同社会，莲花佛国的世界。因此上界仙佛圣神高真菩萨等，亦是十分企盼崇心之作为。能明白自己的使命、责任、义务是该如何行、如何做，必自了知。更在此段时间来建立规章与典则，才是能实际的开展。
