第十一回 大道佛心

南海古佛降

八十年七月十六日 已卯年五月四日

　　　南瞻部洲着天书　海池天地正自主

　　　古德教化立规范　佛国莲花了万殊

　　大道是摩诃之本体，乃以大道为主要的发源地，是在无极、是在虚空、是在宇宙之广大的空间中。大道正是一切演化、演绎的主导者，是『无所为』、是『无为』更是『无以为』。乃是以自然而然的孕育的一切星球，及在星球中的一切芸芸苍生。必以大道所共钦仰的，是把任何动作在各个星球中，来演化、来演绎着一切有情的苍生。

　　佛心是宗教的教化根本，以佛家而言，而有佛心，但也不是只有『佛心』而已，尚有仙心、圣心、神心、菩萨心，亦即是以佛心为崇心也。因在大道演化的根本中，佛心即是禅宗的另一名词，其在禅学宗脉的发展空间，必以禅学来成就其根本资粮。因是以『禅化作玄机』，更是在『顿悟』的当下，必须要能累积其根器的锋利，才能『顿悟』。否则不论修持任何宗教派脉，亦在『渐悟』之中来成长，有之更难有『渐悟』。尚且在迷途中，根本都无法脱离此困境，才造成了一世世轮回的相循，在『禅宗』所显示的无非是当头棒喝。

　　　禅化玄机了无生　智慧开展顿悟成

　　　迷途知返识天机　弘扬崇心出鸣声

　　大道佛心所能显示，是以崇心作教化的根本，又以『儒宗圣教』作规范。即以崇心建立典则、规范、制度，带引未来的芸芸苍生，来回归于无极理天、极乐世界、虚空天界、宇宙天界。乃是把沉沦的众生，化除垢秽的灵性本体，成就于莲花佛国的境界与位阶。因而『大道佛心』即是『大道崇心』也！以崇心道脉作资粮，在未来中又以『佛心宗』来导引儒、道、释，三教的修子作回归的指针，如此正是以『佛心宗』的莲花世界，成就于一切有情众生。

　　崇心以『大道佛心』作依持的规章，开展第二期的圣务，正是通过了第二期的第一阶段，入于第二阶段了。大道者——一切大道的总枢纽，乃是人世间的源头，任何一位有情苍生，皆是大道所显化、大道所演化的过程。因在大道，若以佛家来言，乃是『摩诃之路』为大道者，是一切根源的发源地。在天之道是曰：『轮转』、在地之道是曰：『相循』、在人之道是曰：『轮回』。如此天、地、人三者之『轮转、相循、轮回』之中，又能串合在一起，更是不可分开的。即是道家所云：『道不可须臾离也』。因而道之所能显示，在人之身中，一切以人行持的润生食物，由口入于喉，再入于胃消化之后，即形成了养分与废物之分别，是吸收、是排除，皆由个人体质来决定是为最主要的了。因养分是对众生皆有利益，而废物即对众生产生残害。但在阴阳相对定论中，不论养分与废物有时亦会对人体有利益与残害，过与不及皆会对自己有损害。

　　利益太过之时，必造成对人身有所伤害，但若不足之时亦是欠缺。又一切可以供给自身的营养成份，即在过与不及中抉择，然而恶漏之不洁，若多用即是对自己造成诸多的残害。但有时并非全然皆是，有时少许恶漏不洁，更能以毒攻毒来调整自己恶漏的清除。如此过与不及，又是造成更大的伤害，而此恶漏不洁，在人体之中少有利益，对个己残害较多、较深也！

　　大道乃一体平等，大道乃万物总枢纽，一切人类世界的芸芸苍生，皆以自身的大道来运转。人身宛如一个小周天、天地乃是一个中周天、宇宙虚空又是一个大周天。若人身之不动即是固化，又人身之可动，必靠食物提供营养成分之润合所促成。如此人身之可动，必是由肉身的组合中，通过了『火』来温暖自己的躯体，又以『水』来调和在水火相济，倘若无『风』之助长，水不能动、火亦不能暖，然而若无『风』之动时，及一息之不来，呼吸间即无接续。但无『地』之坚实性包裹着肉身，即难有『水、火、风』三者之组合，地之包容性、坚实性，包容着『水、火、风』三者，人类才能存活、才能走动，则是大道之源头。众生可在自己之躯体中去详思，为何我个己能因食之物品，由口入于喉再入于胃后即能消化。乃人身之小周天为之，亦是人身之大道来作用，而一切可动的根源又在何处也？能明否？能了知否？下期可分晓！大道佛心将由此处起，进入『禅宗』，可细思必能明了。
