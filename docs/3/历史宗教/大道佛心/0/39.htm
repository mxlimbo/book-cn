第卅四回 实相世界

南无正法明如来佛 降

八十八年十月二十九日 己卯年九月二十一日

　　　南无大道无疆远　正法明如来佛言

　　　降来着作佛心篇　实相世界是本源

　　实相世界由心念意识的传导，而入于众生的心性意念中，由灵性的世界来作主体，其根本最重要的目的，是把个人的因因果果来作化除。如此是以人类的世界以灵性来作主导，其目的行使是以自己能入于灵性的提升，并且把精神意念来行使。使自己能在灵性世界中作自己的主人，而不会反覆于沉轮的轮回漩涡里，更是能给予自己有实际的资粮。

　　因在实相世界，正是器物科技所形成的世界中，不论任何一个物种皆受到人类所统御，所驾驭，乃是对万物众生，给予作保护或作残害的相循动作，让人类世界在自己生存的范围中，作人类主导的范畴。又在人类器物科技所演绎的过程，当今人类世界正是在享受前人所种植的福果，给予人类福祉的德泽，也是人类世界的福份所臻，其必须要以『人类与万物平等作主体』，亦是以人类作统合的当下，不可强加人类科技的文明残害众生，即必须要了知在万物众生，亦是灵性的一种，尚且是人类科技所能给予万物众生，亦有无穷尽的福祉，才是真实性也。

　　实相世界与灵性世界是两相结合在人类生存的环境中，若仅有器物科技文明，必难以对自己的灵性，有可以生长的空间。必然会流于仅存在器物的科技而已，即会对其他万物物种的众生，相形即受到戕害。乃在器物文明的世界，若无灵性增长的提升，必难以对有情众生，在自己生存的过程中给予再超越的契机，而此也是针对当今时代的芸芸众生，所必须要有的主体观念。

　　若以器物文明所展现，正是对人类与万物提供了可创造无穷尽的福祉。在当今而言，人类的器物世界，正是我炎黄子孙，近五千馀年来所最有福祉的时代，而对于人类心性又是臻于极奢、极诈、极险的过程中，致使有一些的正道修子，来作人类世界灵性的提升，如此是以道德而为依归，是以把灵性来作超越，更是在灵性中给予净化，而入于能跳脱往昔轮回，所设定的漩涡与限制中，乃此当今时代正道修子，所必须要有的基本观念与作为，而此即以灵性世界的超越来作导引，实相世界正是人类生存，不可或缺的最主要基本，人类世界倘若跳离灵性世界，而仅在于器物科技的文明世界里。那是给予人类世界，永远无法能有向上提升灵性的超越与升华，即是难以作统合所必须要有的基本作为。在灵性与器物世界，正是入于禅宗所必须要化除了虚幻，假设、妄想的存在空间，因而即以器物实相来作人类生存的依归。亦是将人类生存的条件，把人类的生存过程来给予再进入改造，而且更是升华，才是让有情众生，不会再入于轮回的相循环境之中。

　　实相世界是以『色法』来成就一切芸芸众生，不但是真实且是实际的过程，人类不可脱离实相世界，而仅存于灵性世界。那亦无法把人类来作改造，如此人类正是处于上升与下坠的分界点。那正是，『人类一切含因累积的改造界』，若无实相世界，则一切佛菩萨圣神仙道，皆难以达至超越与升华，更遑论能成长与提升，即在于是对人类世界，又有一份长养教化的基本课程，乃在人类世界，正是以学习而学习再应用，以学习再入于超越的基本范畴，是永远都无法改变的事实也。

　　实相世界若不能处理的完善与尽责，即在人类世界中欠缺了圆满，如此更难以达至满圆。又如何在资粮位中入于实行位，再入于精进位，再入于圆满位，即根本都无法按步就班，一步步来成长。更是无法在『大道明心』中作下化的主要课程。即是对未来众生中有一份长养教育的课程。崇心的着作已渐进入密禅的阶段，在往后必是要一段段来，再去作更大资粮的行使，即是作未来『真佛心宗』的基本架构，亦不可偏悖了『儒宗圣教』的传延，则对上苍有一份交代与责任。即是在器物世界中，把灵性世界来提升，给予芸芸有情众生，能往上提升自己灵性的再超越，才是对自己未来，自己作自己的主人，而不会再受轮回的奴隶 。『大道明心』即是了知人类世界的器物世界，一步步来融合在资粮位中去行使实行位，再入于精进位，又如何达至圆满位，必要是把一切来按步就班的完成。

　　　大道明心是实相　器物世界本真实

　　　灵性世界两融合　资粮实行再精进

　　　圆满满圆入禅宗　密密行行本是同

　　　真佛心宗扬大千　三千世界皆理弘

　　　正法明如来着作　南海古佛观世音

　　　崇心因缘深且远　理笔更可迎大同

　　此供众生在入于莲花佛国之中，更有实际的资粮位，即以人类的器物世界再同灵性世界，不可相互违背，理应两相融合才是真实性。
