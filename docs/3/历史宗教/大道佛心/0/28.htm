第廿三回 生命的观念

南海古佛 降

八十八年九月十日 己卯年八月一日

　　　南瞻部州着天书　海池天地永传注

　　　古德再延启教化　佛曰生命观念殊

　　人生因在世间中作润生的动作，如此即常会遭逢了诸多的业因、业果、业障来牵缠，乃在于是自己的所作所为。又是前世与今生的总合体，故而在当世中，即以过往的业因作凭判，是否享福报或是遭恶殃，则是由自己来作抉择。如此而言即以生命中的过程，承受着前世所作所为，加上今生的所作所为，作评判其功果与业因的形成。

　　因而有的能常享福报，有的常会遭逢恶殃的折磨，然而在于业因的形成中，不论是好？是坏？皆必须要自己来承担、自己来消业、自己来享福与受偿，是任何人皆无法替代，如此才是符合因果法则的平等性。在人生的生命过程，其观念是必须要建立，是自己造业与造福，都必须由自己来承担与领受，然而必须了知自己的所作所为，不但是自己会入于平等性的法则中，且一切的因因果果，皆是平等性，而让自己来享福与遭殃。因而即以人生的生命作取向，在总合中，不论任何的高官显爵或贫穷乞儿，皆在如此的平等性法则。然而在平等性是不能抹煞其业因、业果的形成，是由自己的先前来作为，必是由自己的自我行使，也是由自己改造中才可能化除。乃在于是对众生一种的平等性，不论王侯将相或贫穷庶民，皆是一体如是！

　　生命的价值观，是把累世的种种不良积习与含因，来作改造的资本，才可能将人生生命作改造的过程。对于一切的有情众生，若无法具足此身躯之时，必然会陷入轮回的漩涡之中，必难具足躯体作改造，因而在人世间的一切过程，是必须以自己来作评判与改造。『生命的观念』是建立在有躯体，才可以做改造；倘若无躯体，是不会有造业与造福的形成。对于一切众生，在平等性的法则中，任何一位有情众生皆必是如此。不仅在人类世界而已，尚且遍及有情世界的一切动物之物种众生，皆是如此。而生命的可贵处，乃在于将沦落于地狱的种籽，亦可入于向上提升至天堂的资本。但其先决条件，是必须要具备自我的身躯，才可能有机会！否则若丧失了此身躯，又何来能有机会入于润生与修持。然而在修持是改造过往不良积习与含因的改造源，若是沦于下坠之中，必是减少了可以上升的契机；若能把下坠的种籽，再经由往上再上升，由上升再超越，才能入于仙佛、圣神、菩萨的行列。

　　生命的价值观，若论及当今人类生存的环境，是远比古昔帝王、公侯之生活，来的有过之而无不及。乃在于古远所具足的神通法术，皆幻化于当今时代的科技文明中，若是以古远之千里传音，是当今的电传视讯，比电话来的更有看头，是直接显像。古之腾云驾雾，是当今之飞机，甚且是太空梭直入月球、火星、金星：：等之中。人类的种种生活，是远比古代帝王更享受；古之出门坐轿、骑马，现今人类出门有轿车，不必受风吹雨打，且有冷气吹，以及动听音乐来享受。如此种种的一切，真是多的不胜枚举：：难以一一列举、亦难以比拟。故而不论乾坤两造，皆是能具足大因缘、大福份来成就自己。如此必须要了知润生的过程中，远比其他物种的众生，所享受的福报，更能臻于完美，即是人生的生命价值可贵之处。

　　当今时代中有一种无形的杀手，乃是『压力』。在其所受的迫害中，是无形无相，若是陷入其漩涡中，不能了知其真实性，即难以把『压力』来化除。如此正是处于人类生命中的无形杀手，也是处在当今时代中的人类，所无可奈何之事！『压力』的纾解，是要减轻自己的负担，自己如何能看清一切的世相，即是在于自己的一种责任。一种贪求欲念的推动中，常把人类的无形杀手建立了，尚且要对一切世相的明悉与了别，是不可贪求欲念过份的追求，倘若是一种责任之时，无妨可以把此责任来分担，给予大家共同来分担负责其责任，才是减轻『压力』最主要的负担与改造。

　　生命的价值观，是以作修持，了知人生的真理，不可随波逐流，入于人云亦云，人迷我迷的境界中，一切都是迷迷茫茫，又痴痴钝钝的走一回而已，更不要在此世的人世间，造就了诸多的恶因、恶果，更让自己入于恶果中。若不来人世间，就没有造就一切过错的发生；如来人世间却是造就业因、业果，又更造罪，后悔即已迟了，那真是得不偿失，汝说是否？

　　生命的价值，乃是创造人类世界永无止境的福祉，才是可以在未来中具备无穷的福德。崇心的基本圣务，不仅如此而已，尚要求创造人类继起生命之永续，更是创造宇宙生命的开拓者。如何能在了知崇心的圣命中去做行使，去做润合，更是可以在人类世界中，有一页可圈可点的见证。明白否？

　　　生命价值在改造　人类生存福缘到

　　　享受远胜古帝王　珍惜此身躯安好

　　　成就自己福众生　创造宇宙更新成

　　　无为付出皆己造　永续传承众生证

　　此偈作为今日给予有情众生的生命价值观作改造，才是在人类世界中应有的态度。
