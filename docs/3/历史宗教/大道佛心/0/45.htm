凡例

南无大日如来佛 降

八十八年十一月十六日 己卯年十月初九

　　　大道佛心已完竣　大日如来作凡例

　　　演绎五教同本源　佛教平等众生灵

　　一、大道佛心乃佛家千年不传之密，在崇心道场来阐述其真实义，是盼对我佛们弟子，破除唯我独尊的错误观念，乃是修整宗教平等而无高下！

　　二、凡我佛门子弟等，不可持视于唯我佛门可超升极乐，且在于是上苍降下宗教来引渡芸芸苍生，而作回归的根本，是以唯心所成的器物世界来作改造。

　　三、凡尘众生皆会有我之『执着、分别、诽谤、贬低』的自我设限，又是难以成就的陷阱，如是必要能破除，在此错误的宗教思想观念中来作化除。

　　四、大道佛心是以崇心作基础教育，给予有情苍生能彰真如、显佛性而成就自己、成就众生，乃是最主要的资粮！

　　五、在资粮位中，来实行于自己的精进，而成为圆满，更是乘越！乃我自己所选择的一条修持、回归的路程。是以崇心为大道系列天书的根本理论，来下化芸芸苍生，是把宗教平等的基本观念，带给芸芸苍生皆能成就，而且更是乘越，再入于莲花佛国的净土世界中。

　　六、成就的根本是不分宗教、不分高低、不分彼此、不分教脉、亦不分回归与超越，则是为平等的教育课程，是为最主要的基本观念。

　　七、引渡苍生是自己能成就，又能成就他人，虽目前推广较辛苦、较不易，但有心必可成之！有心必能成就教化、教育，平等宗教的基本观念！

　　八、本书所在之处将有仙佛护佑，不得随意丢弃以免冒渎仙佛。

　　九、助印『大道佛心』若有所求，上天自会斟酌赐与所请，助其消业障增福慧。

　　大日如来佛 降凡例 于——无极大道院、懿敕崇心堂

　　天运己卯年十月初九日 戌时
