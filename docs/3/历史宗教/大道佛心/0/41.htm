第卅六回 儒宗圣教与真佛心宗

本堂主席 虚空天尊 文衡圣帝登台

八十八年十一月五日 己卯年九月二十八日

　　　恭接昊天玉帝降　崇心神人排班行

　　　大道院士按顺序　大道佛心着完成

　　今夜大道佛心最后一回 恭接 昊天玉皇大天尊降来着作，诸神人与无极大道院灵修院士等排班候驾，其馀人员等不可喧哗，亦不可失仪，好。为师暂退！

玉皇大天尊 玄灵高上帝盖天古佛 降

　　　玄灵玉帝降来堂　诸神人等虔恭航

　　　着作佛心觉圆满　崇心弘扬真理长

　　崇心能得天独厚，在于南瞻部洲来做教化课程的圣务，乃因崇心诸神人等坚志不拔，毅力克服任何困难，在在显示主事者之坚心所成，故而由无极皇母亲身懿命来设堂，能弘扬「儒宗圣教的开启」，将其过往的陈腐作为来作改造，是以弘扬三教真理，并且为天下宗教来作相互融合，其因原来本是一家，乃在于把宗教来区分，给予芸芸苍生有一条可以选择的路径，让自己去做学习，去作进修，亦是自己所愿意选择的一条修持大道，更是众生自己所乐意而为之。

　　儒宗弘扬是以真理来作教化，又以三纲五常、四维八德，将人生的修持过程与行径，一一来彰显，在行持「孝悌忠信礼义廉耻」之当下中，把生存道德下化于芸芸苍生，正是上苍慈悲所演化，是以众生皆能成就为目的。并且把人类在信仰中，以「信实、愿意、行持」来作主理，更是在自己能入于宗教修持，且把宗教教义课程深植我心，才是自己能成就的原动力，如此宗教在当今教化里，正处于万教齐发的时代，每一宗教皆谓我最高超与殊胜，乃自我标榜的行径。

　　弘扬真理是不分彼此、不分你我等，在在是无分别其他宗教派脉的教化传延，更是针对其宗教课程的传延，又是把宗教思想深植人心，必是让自己能在修持的范围内，可以「超越轮回、超越生死」。如此的宗教教育，正是弥补国家教育之不足。即是以教育教化传承，给予后代彰真如显佛性，是宗教最主要的目的，乃在明此心、见此性，即是「明心见性之本体」，且在宗教传延课程是理所必然。

　　宗教本身是无分别，乃在于传延者，给予后代作保护之动作下，故而有所分别之阻碍，即形成了自己对其他宗教，产生了「分别、我执、贬低、毁谤」的动作，是在人类世界的宗教中，所必须打破的讹谬观念。如此应是「无分别、无我执、无贬低、无毁谤」，才是宗教平等、信仰平等、自性平等、修持平等、灵性亦是平等，才是宗教间的公平性。

　　真佛心宗是在当今时代中以唯识观念，入于禅宗、密宗，的初立宗脉，乃在替上苍作教化、作平等、彰真如、显佛（自）性基本教育课程的道脉。是不分彼此不分汝我，更且融合五教教义在三教同宗、五教同源，更能让有情世界的芸芸苍生皆可回归、皆可超越、皆可再乘越，而达至莲花佛国之净〈境〉土中，正是『真佛心宗』所必须要有的基本观念与作为。在理笔领旨以真理度化苍生的回归中，配合诸天仙佛圣神菩萨等，来集合了整体的大回归、大超越、大乘越的主体，作为以「理化玄机渡众生」之本质，且在于当下与未来中，必能真正弘扬于大千三千世界。

　　『真佛心宗』正是超越宗教的基本观念，破除了「分别、我执、贬低、毁谤」之作为，融合了三教精华五教教义的传延，把此芸芸有情世界众生，迈向莲花佛国的世界中去作实证。更在有情世界中，不可脱离生存的一切润生动作，正因在人类世界，是上升与下坠的分界点，在上苍来言，必是要渡尽芸芸苍生皆回归，才是上苍之演绎过程，不可存有任何一位有情众生，沉沦于五浊恶世之凡尘中，乃是上苍之慈悲心路历程之本质。

　　「儒宗圣教」在当今时代，已脱离了往昔「儒宗神教」教化课程的模式，盼我崇心诸神人等，不可执视我之天命与任务的殊胜，更能在低心下气中，去融合其他宗教教化后进学生，在未来皆能有实际的功勋与福德。是自成之！是天赐之！是智慧之！皆是自己来完成，自己来创造。乃是对上苍责任之行使，亦是「在上德不执德」之观念中，作最上乘之发扬。必是体天地之大爱来融合，更滋润芸芸有情苍生，皆能回归与超越，乃是实证也。

　　　儒宗圣教　演绎五教　儒道释耶回　同源本一家

　　　真佛心宗　不分彼此　显真如自性　大道普天下

　　此偈赐予崇心道脉，在往后中能带领芸芸苍生回归、超越，乃是针对当今沉沦众原灵，一个光明的大灯塔、大指引的目标之基本作为，更是广引有情众生，畅真理来回归于无极理天、极乐世界、虚空天界。正是 朕在此时代中，最大的心愿与盼望了。矢勤矢勇、坚志勇往，必在未来中将「儒宗圣教之传延永不改变」，立于教化中，「真佛心宗阐扬真理耀大同」，达至莲花佛国之净土中，乘越『金品莲台』来作回归。亦不负 朕今夜亲降崇心道场来，着作「大道佛心」之总圆满完成也！
