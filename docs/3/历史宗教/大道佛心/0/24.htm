第十九回 贪嗔痴的化除――嗔

南无药师如来佛 降

八十八年八月廿日 己卯年七月十日

　　　南无称赞无上佛　药师如来是真我

　　　贪嗔痴念起无明　化除往上必超脱

　　众生因贪欲而入于生存的环境中，是以润生来作行持的动作，如此是以生存润生的动作，必有『取向』的思想才有投入，方能把『向取』来行使。即是创进的行为，而此创进若停留于不动之时，即无法把人类生存的空间来创发，必然是以作创造、创进的步骤，而此步骤形成的原动力，即是贪取的走向，即是『向取』的原动力。

　　而在『向取』中是以物种的创发才能开展，即有贪求的欲念，而此欲念的取向中，必存在于『向取』的润合，因在润合的『向取』中，是把『取向』来成就。如此在人类的世界里，必然是将贪欲来创发，达至目的地的行使，即是贪求欲念的『向取』也！又在贪求中，必是有取向才能促成，而此『取向』中，即有嗔恚欲念，即是恚心的运转，此即在『嗔』中有逆之转向，必然存在于取向中，将向取的事务来造就。因而嗔于不动，即难有持恒力量的产生，而嗔之动作，即是把忿怒来显现，因忿怒是恚心的另一种向取，又在恚心中，若是无取向，必存在于恚心中而无忿怒也！因恚之展现的正是把忿怒来消除，故由嗔之所动中，必然是以忿怒来平息与平静在嗔恚之中也。

　　嗔对众生能入于凡尘中，是在人类的有情世界里，能出生为人是有相当大的福报。倘若不能珍惜此生的躯体，必在轮回中反覆相循，进入了自我预设的陷阱内。如此嗔于自我无法向上求取智慧的累积，来滋润个己的资粮，必会在嗔中存有恚之念也！因而恚心即入于忿怒中，即形成了火烧功德林，已是无法存在了。而此火烧之忿怒，必然是以功德作取向，而此『取向』若无『向取』中，亦是无功德之展现，又何来能有功德之谓！

　　嗔于众生能入于修持境界，反而会比凡尘的无修众生，来得更加固执，更无法转化，因而即形成了一颗顽石硬梆梆《定扣扣》，不但一点都无软化的迹象，更且是入于『阿雎锟毓塘Α唬乃人间的一种笑谚，因而必然无法转圜在修持的局限。如此是否嗔于自己的固化所致，甚且难以改变也！『阿雎锟毓塘Α唬乃人间的一种笑谚，不知在凡尘众生的自己，是否入于此局限中而无法转圜也！必是要自己能了知嗔恚之心在何处？又如何改进创造？必然是转化与教化的基本要求。能嗔于自己无缘求得无量智慧，具足源源不断的资粮，如何在未来，能实际下化于有情世界的芸芸苍生，才是修持者，必须要具备的条件与作为，更是真实性。

　　嗔于自己不足的取向，必然是自己了知自己的不足，更嗔于自己无法成就，无法超越。才受轮回相循而左右，必在人世间，若常嗔于自己之不足，又如何能创发？如何能行持于『大乘菩萨行的觉有情境界与位阶』？如此是自己嗔于自己之不足，那又能怪仙佛、圣神、菩萨之无教化所致之，又在于自己有否『阿雎锟毓塘Α恢局限里才是真实。乃以人世间的修持，必要把持着永远都是半桶水才是实际。可在『闻思修』中入于『思惟修』的菩萨境界，更甚入于『细思惟修』的佛国境界中。

　　倘若能嗔于自己之不足处，更必须往上求取大乘源源不断的福德与智慧，才可能具足往上追求之愿心，即是向取欲念，能嗔于自己所不足而作追求，而入于真正能『代天宣化、教化众生』的资粮位，更能进入实行的位阶，才是证道体悟所必须要了知的一切。

　　　贪念化除了业因　嗔心自己的不足

　　　转化半桶入丰盈　更能大乘菩萨行

　　　贪心不足往上求　嗔己难逢智慧修

　　　福德加添更丰厚　佛国净土我成就

　　闻思修中入于思惟修，更要入于细思惟修，才是把『禅宗』的要义了知与明晰，亦能于往后中，建立自己的信心、愿力、行持也！了知否！
