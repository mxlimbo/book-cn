第廿回 贪嗔痴的提升——痴之持恒

南无阿弥陀佛 降

八十八年八月廿七日 己卯年七月十七日

　　　南无称念如来佛　阿弥陀经观自我

　　　痴心永恒成就得　佛化贪嗔痴了脱

　　佛教以痴是对众生一种不良的显现，乃针对痴迷又无法转换才造成，因在痴若不能持恒，必难以达至成就的位阶。如此必须要了知，痴是做一种行动的原创发力量，而此原创发力量，必须借重于持恒力量之接续才可达至，倘若不能达至其目的时，必难有持恒力量之承续。

　　又在『持恒力量』，必要以痴于一种相信的持续力，因持恒是以创发力量做动力，才是达至目的地的主要原因。然而『痴』是在于往后能成就，又对个己的未来能回归，甚且能超越，则必须要把『痴心』化作持恒力量的创发，而此『持恒力量』是在人世间的一切过程与作为，所必须要有的心态，如果没有持恒力量的展现，势必难以达至痴心创发力量的开展。众生是以贪图欲念之『痴心』中作取向目标的行使，才能有向取得到的喜悦。不论在任何一个环节与过程，只要是人类世界的众生，即具备了此一动作的行使，因在取向中，是必须要有向取，才能有痴心持恒力量，如果没有此力量，必难以达至人类世界，一切因缘之开展和合力量的创发。

　　在于痴一件事务中，必须要具备了此『持恒力量』，而此『持恒』所具备的正是，痴于一件事务的取向中才能达至。因而若无贪求欲念，或是贪得欲求的过程，即无『痴心』之念把人类世界的因缘组合来开展。因此人类世界是不能没有『贪』之念头，若是无贪，那人类世界何来有创造、创进的过程？那天地演绎、人类演绎、万物众生的演绎过程，又如何往前创造开发，即在于是对一切事务的开展，有其必须要的实在性。然而佛家对众生是要求不可存有贪图欲念的接续，在往昔的教化方式是如此，然而现今已进入科技文明的时代，一切的演绎过程，皆是对众生创造无穷尽的福祉，更创造了未来人类世界，继起宇宙生命的开拓者。如此在创造、创进的过程中，是必须存有贪图欲念的行使，才能达至对未来众生，创造无穷尽福祉的过程与目标。

　　倘若仍处在宗教的藩篱中作自我设限，并且对众生与该教修持者，自认只要修持一种即可达至目标，而不对生活中去做改造，那无异尚在幼稚的教育过程。因而当今宗教主事者，必须要打破以往固执陈腐的修持观念，把人类世界的创造、创发过程，加以贪求上进，求得源源不断的大乘理念及智慧，与深厚的福德来下化于芸芸苍生，并对有情世界原灵的佛性提升与回归之资粮，给予在下化中能具备不为我个己回归超越与否，是对芸芸苍生、有情众生提供创造、创进人类世界继起『宇宙生命的开拓者』。而此贪图欲念的行使，是否嗔于自己的不足，更需要把回归资粮来下化，如此必须要有痴心一片，行愿持续力量的展现才能达成。

　　人类世界，正是欠缺此一继起宇宙生命开创的原动力，而在佛国莲花世界中，必须要能回归之后再做超越，然而在众生因痴心，有持恒力量的开展才是真实性。而此『贪、嗔、痴』在佛家所云：是对自己往上提升或往下坠落之分野来作抉择，果若往上寻求贪念，更在此世中能做回归。更是嗔于自己有固化否？无法具足善德与福报的大因缘，又有啥机会，能入于痴心持恒力量的开展，让自己来成就。能了知由贪求欲念再嗔恨自己不足，即必须要往上再寻求真理大道，来给予自己有成就的契机。否则自己包裹固化了，又如何有大因缘与契机，如此痴心持恒才能有目标，才能真实的达至。因而必须要了知，是往上——『形而上』的提升，或是往下——『形而下』坠落的沉沦，即由自己之作为中，要具备半桶水之见证学习，才能装得下别人的优点与见解，亦可了知天地的实义，更要做『形而上』的成就提升的因本。如是而言，即可在人类世界中的当下——此世，来作改造、来作回归的准备，不必倚望至下世再来成就。那无异也是一种痴心妄想的持恒力量，能否成就？在于自己是如何取向？如何改造？又如何达至？是必要自己对自己来负责，任何的一位圣神、仙佛、菩萨，亦只是对自己的一种教化学习过程而已。

　　　成就因本在持恒　痴心一片乃是真

　　　贪求欲念往上行　嗔恨自己难以成

　　　痴心化力更求精　真可入世化有情

　　　贪嗔痴迷悟真理　沉沦超越靠自己

　　能识得此偈之涵义，必有所得、必有所超越，即可在当下中也！
