第四回 德遍天地润苍生

虚空天尊 文衡圣帝关 降

八十八年六月月十五日 己卯年五月初二日

　　　虚空天尊任主席　文衡圣帝阐圣理

　　　德遍天地润苍生　崇心道场着书期

　　德――德之遍地，乃天地主要之任务。是提供有情的资粮，给予大地众生作润生的过程，是以个己之所需要，做个己生活物质追求的因本。如何能达成是关个己所作所为中，去追求早已具备的资粮，是给予个己在做润生的接触过程，才能开展人生的因缘组合。故在个己的润生组合，是每位众生最主要的根本，不论任何一位众生皆是如此，更是无法脱离此范围，此乃德之总合体。

　　遍――乃是把德泽平等散布下化。因遍是一切皆平视均等的范畴之内，如此均等遍布，即做有情世界中均等的具备。不论众生每位的时间、空间因缘皆是均等，亦是平等视之，共同均享此天地中有情的资粮，但在个人累世福报与过错(祸殃)之中，来相互抵减与增加。而此个人之福分与祸殃之分差，端看各人能具足多少与消减多少。遍之所泽是平等均享此天地的资粮，是无所分别与限制，在人类的世界每位众生的身体，皆是具备相同的均等质量，更是不分贵贱高低，任何物种亦是如此。因此万物众生，正是大寰天地中的有情众生的物种之一来言，每一种众生皆是个己成分质量，条件皆均等具备。然在人类生存的世界，是以人类为主要的根本，才能彰显此有情世界的德泽也！

　　天地者――万物以此生存空间中的环境做润生，更是此时代中，把润生所需的物质追求创造，再来做各种众生的生存总合体。因天地间正是具备对此空间内的一切有情众生来做演化功能，更以在有情世界中，把人类的生存规划，丰盛此天地中一切有情的润生过程。然而在天地是不言又不语的当中，又以人类做物种的总合体来导引，更以人类彰显仁慈，来德润一切有情万物众生。然而天地间的众圣神、仙佛、菩萨等，皆是配合着人类生存的演化功能与创造，来护佑天地间的芸芸苍生，更在此世界中，做改造与进化的教育课程。因而一切有情万物众生中，皆与人类生存有相循功用的过程，并提供一切资粮给予人类作润生利益，如此是以天地之演化过程，且把人类作灵长的根本，其原因即在于此。因人类能思考、可以改造、更有文字记载，其在文明进化的过程远比其他物种优越，又其他物种众生，皆难有此功能作用，有此创造演化，更无复有此契机给予改造。因在天地中，一切皆以人类作教育传承，开创人类无穷尽的福祉，才能达至开创与进化的目的也！

　　润苍生――润苍生并不仅只是人类的众生而已，尚有其他有情、非情之物种，又在仙佛之当中，皆是由人来成就之，是已通过人类生存的过程经验累积。然在各种物体的总合体中，必须要一次次的进化与改造，更以人类之大爱去做保育与护佑的动作，绝不可倚人类的强势来侵害其他物种的众生。在一切有情、非情的动植物中，皆承受着天地的德润，给予人类能有更进化与教育传承功能。纵观近五十馀年来，人类的科学进化中，皆是以提升人类众生福份的增加与累积，以人类生存舒适的福祉为目标，是人类承受着天地间的圣神、仙佛等来护佑与培植，由发明中去改造，过去人类生存种种不良与不适的生活环境。因在人类现今的时代，可谓之三千馀年来最有福报、生活环境最舒适，行动也是最方便。因为时代巨轮生活的循环是如此的进化，但人类心灵与智慧的提升，往往跟不上时代科技的文明显现，即时常受到心灵与生活难以宁静，智慧无法增长提升，因此难以与天地相契合。此亦是润泽苍生的同时，人类在生活的润合，而无法有灵性的提升所致也。

　　泽润苍生――必以先天的本灵做彰显，又人类在现世环境当中，皆难以能用宗教的教化，来弥补灵性的不足与做信仰依归的规划，若能进入宗教的薰陶，往往又受到宗教的藩篱所束缚，难以脱离此限制。根本无法有真正修持的理谛，来滋润自己在灵性的增长，又如何来彰显自己的自性佛性。其实皆会受个己之我执固化，及理障、所知障之阻碍，根本都难有真正能回归的提升资粮与契机。在此局限的固化修持观念的环境，又如何能给予众灵子，回归于无极理天、极乐世界、虚空天界，更遑论要在此世中做一世的超越。必须要了知人类在生存的空间中，是不能脱离于此限制，更是实在的生存环境。才能给予自己有充足的资粮，更有成就的基础，必以人类给予人类自己来成长与回归。

　　懿敕崇母心 德遍天地润苍生――即在做有情世界如何回归的准备，亦在于懿敕是以无极做回归，以虚空做超越。更在 无极皇母德泽有情众灵子之中，把此天地之玄机化做津粮。才能成就在此现世，有修与无修的众灵子，皆能感同身受，去领纳天地的恩惠与德泽，因而天地中有天地的轨道，人类有人类生存的条理。天地间人类与万物众生则更要在生存中，来把人类作更上一层的教化，在教化是以平实的理谛，而非是方便权巧的俗谛，才是做润生做回归应有的知识，因在修持者皆谓我个己入于宗教中，不知所作为皆不离开此人类生存的过程，必在此润生中如何把此世的经验，化做修持与润生，是必要有的主要功能，与教化课程才是实际。

　　德遍天地润苍生――是崇心的基本课程，在教化的过程中，是不能离开此规范与轨道。如何做规范，如何做条理，又如何来应用，皆是在在的考验□崇心诸贤生，如何应用智慧，去做规范、去做条理去应用下化于有情世界的众生。不分彼此与任何宗教派脉，皆一体如是，仅在如何提供资粮，不涉任何宗教间的教理与传承。处在万教齐发的当世，宗教间皆有责任与义务，做一切下化的准备。崇心必要在此时建立起，一切有条不紊的规章，做未来遵循的道统。更以发扬『儒宗圣教』为目的，带引儒宗鸾门更有一番新气象。给予未来众生共同遵守与依循，如何做先锋，如何布施功德修持与润生，皆视个人之努力，如何改造与进化，才能在未来中，有更实际的实证。
