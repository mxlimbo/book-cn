颂

燃灯古佛 降

　　　燃点心灯照光明　灯传万世永垂心

　　　古德承续传佛法　佛心大道是崇心

　　大道者――乃一切演化的根本，任何一位有情的众生，皆在大道中演化。不论天地人，无极、虚空、宇宙之演化，亦是大道的轮转而已，又演化正是把有限的时间与空间，来作彰显印证与注脚。更以演化创造万物的成长，成就众灵子的福祉能绵远长流，能给予有情众生在演化中成长自己，成就自己的「智慧与福德」。

　　佛心者――是已成就者、是为『觉者』，佛是觉悟有情的大智慧者。佛菩萨、仙神圣道皆是如此慈悲博爱心。佛之心――即仙之心、即圣之心、即神之心、即我个己之心。人心可以上升与下坠，是何种心，皆是自己所行所为中来体证，不是上苍来监督，乃是自己来监管自己之心，与一切修持与成就的作为，即是本职也！

　　大道佛心――是崇心的第七册天书，是以佛菩萨仙圣神道之慈心，替芸芸苍生作下化的津粮，给予众生在当下中来成就。然而在下世中是否尚具足此善因缘，皆难了知，切勿心存尚有下世生存的心态观念，须拉近至当下――此世中，那则已无下世，只此一世来做回归，何妨把一切有情的根本，不再寄望于下世。又在当下的仅此一世中，大道佛心才能彰显，才能提引自己佛心的光明，来『燃点千万亿的佛心』『光明于无止境』也！

燃灯古佛降颂 于――无极大道院 懿敕崇心堂

天运己卯年四月十四日戌时

　　为师亦暂退

本堂正主席 虚空上人 登台

　　　虚极空转宇宙因　上德人间无极明

　　　演化永传无止境　崇心佛心更心迎

　　今夜佛教三大教主〈尊〉降来崇心，着作大道佛心――序赞颂。亦为往后着作教化的课程共议商讨崇心第二期的圣务，必须加强符合上苍的旨意也！又本堂诸神人等，不可喧哗，不可失仪，俾免愧对佛尊也！明否！
