第二回 懿敕崇心堂

南屏道济古佛 降

八十八年六月四日 己卯年四月廿一日

　　　南屏一贯普善良　道济公德启化长

　　　古佛降来至崇心　发扬懿敕崇心堂

　　懿——懿为母德之彰显，是以大寰之天地，由普化之过程中，来引渡众原灵作回归。『懿』是慈悲的展现，亦为众原灵之本来位的回归，是把母德来彰显于有情世界中。因懿者——是美好与完善的本质，必以懿为纯真的善良。在众原灵的本质中，是要回归于『清纯的纯一』境界，方能彰显其个己在本质的美好善良也！

　　敕——敕者是上位之下化，是以上位之尊崇者作为颁示的要旨。乃于敕化作天机，即是『敕令』的基本条件，然『□令』为今昔道家符籙之首要，因符籙是以□令才能开展。在上位尊崇对下位之行使派命，乃于『□』中用之，而非『敕』令之用，故此□非彼敕也！因『敕』为尊崇之上位，下达于派命中，故为『敕』也。敕之能展现，乃于 无极皇母亲身懿命派旨敕令，下达于凡尘。又以敕为首要，更是至高无上的尊崇者，是为最圣矣！

　　崇——崇是至高无上的表徵，是以崇之所示，必高于任何之一切方能为「崇」也！『崇』之显示——是把人类生存的本质，应用于作回归的总体生存之经验过程。众灵子皆会追求舒适的生活品质，必要在崇高之理念中建构，方能有实际的达成其目标，倘若无法有崇高的生活品质思想，必难有崇高舒适的生存环境与空间。因「崇」是理想之展现，又『崇』者为至高的理念，更是以崇者之目标，作追求、作欣赏与达至其所必须要有的条件。又崇为钦仰之主体，欣（钦）仰正以『崇拜』为目的、为依归。

　　心——心者乃一切万法所必须要具备的总根源，其本质与相关的作为，由『万法唯心』与『万殊唯物』之体证过程里，把此『心』用来替芸芸苍生谋福祉，才是正确也。然而在现世环境中的人类，是有情众生之一，每位众灵子皆有责任与义务，作润生的总合体，必以创造无穷无尽的福祉于人世间，去泽润万物众生皆受其荫。是把『心』之所呈现的德泽作下化，是把此『心念意识』德润展现于当下，更让自己与有情众灵子，皆能有所成就与提升至高境界，而此『心』正是符合了一切『圣者』、『贤者』、『神者』、『仙者』、『佛者』、『菩萨者』之本体。即以心之扩展，完现于人世间，才谓之对众生有益，更是可亲可爱也！

　　堂——堂之古曰『私塾』也！此私塾是作教化学习的基本场所，即是『杏坛』，又在现今是称为学校、学院乃是圣神、仙佛作真理阐扬，与彰显其实际的理谛，来下化于有情世界芸芸众灵子的处所。因堂是一切良善的集合体，各人的善德因缘来成就于当下，更汇集一处场所来宣讲、研理、着作，且把此良善形成一股力量，将此天机化作更有效的发挥，更是精微的阐述于鸾笔中，是以教化芸芸苍生作目的，更且是由小而大，由寡而众，由众推展更广大、更深远，乃此『堂』之最主要的功用。

　　懿敕——乃 无极皇母亲身懿命『懿敕』之颁旨下来，绝非再似过往，必经过其他之手续一层层的往上呈报，又再一关关之下达旨意。乃 皇母欢喜之、欣慰之，亲身来下达旨意，故懿敕道脉之殊荣处亦在于此也！又此『懿敕』是为美好与良善之表徵。更是把 母德之慈祥、德泽散布不分任何宗教派脉之实行修子，亦不论『普渡』或『收圆』之有情众灵子，来作『回归』之基础准备，回归原来的资粮位。再更超越于资粮位，甚且达至『实行位』的体证，因而『超越』于『回归』之本来体。

　　崇心——是以至高无限所展示之真谛，作资粮、作课程，来下化于崇心的众贤生，能实际的行使于『实行位』，必要能切切实实的开展一切道务。因至高的理谛必能以『唯心所识』，来成就法界的芸芸苍生，是把『心法』应用于『色法』当中，再去用于『有为法』，来开展一切道务而达至『无为』也！

　　懿敕崇心堂——是由 无极皇母亲身派命，亲笔点召之道场，即以济世利人方便，更用真理普化来成就于芸芸苍生。且在凡尘中把『真谛』下化于当下的现世中，又崇高理念之建构，是把『真实的理谛——真谛』，再作更切切实实的阐扬与应用，而非是一般『世俗的理谛——俗谛』所能比拟。又『俗谛与真谛』中，是以『真谛』作『实行位』，结合『俗谛』应用于人世间，才可能回归于原来的无极理天、极乐世界、虚空天界中。

　　理之谛必要如理实际，乃是真谛，而非是俗谛，是以真谛作资粮，化除世俗谛才有润生可融合。因谛者是至高无限之展现，才是让世间真修的实行者，来作依归、来作指针，更是回归超越的大指标也！因『真』与『俗』之谛理，前者是为实际资粮的润生修持因本，后着是为权巧方便世俗生活的过程，即以真俗之分差，来给予是实际、是方便的剖晰。乃是以个己成就个己，个己超越个己也！崇心堂即把此『大道系列』作下化的课程。因大道系列之着作，迨至今『大道佛心』之阐述，已进入禅宗的领域境界，渐而以『禅』示现于课程中。

　　　禅修方便皆法门　理谛应用方是真

　　　佛心彰显大道依　崇心着作更真谛

　　此偈供有情众生与诸贤契，了知『禅』修之『实行位』与『资粮位』的本质！明否！盼之！盼之！
