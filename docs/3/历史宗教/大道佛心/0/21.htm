第十六回 无心治妄

南无本师释迦牟尼佛 降

八十八年八月六日 已卯年六月廿五日

　　　佛教真理润生活　本师释迦牟尼佛

　　　正知无心治妄想　虚幻众生误自我

　　众生于世皆受个己思想的左右，不论修持与润生皆是相同，亦会受虚幻所影响。常会促使个己在本身之中，受轮回相循所限制，是把个己入于虚幻世界，更由于虚幻的作为认假为真，亦常在妄痴迷中来行使。因而自己即受到业障的局限，并且常在轮回相循之内，更是入于难以转圜之境地。即一而再再而三，永远都难以将真理来成就，乃是众生所迷妄之处，亦是轮回相循之主要原因。

　　无心可治妄，因﹁妄﹂是把一切不实在的思想观念来行使，亦即自己认为是真实，且一次次，往自己所认为是该如此的作为中去行使。但在实相世界中常会为了虚妄的境地，把幻想来造就，此即每次轮回的主要原因亦在于此。但若以人世间有 智慧者，是能看清楚此一局限与漩涡，急欲跳脱此困境，必在治妄是以无心来成就，因是妄想存在的现象界，且是真实不虚假的存在界，又一切法相、色相皆是外在所显现，而一切有为法，即在心相之中来成长，一切有形有相的存在界，若是同天地、无极、虚空、宇宙之长远相互比拟时，即显得十分渺小，因而即让众生无法能了知天地正理来行使！

　　虚妄心态正是在现象界，不论修持与润生者，皆一体如是会有的现象，因在妄心虚幻之中，一切有形『色相』与『色法』根本都无法同『心法』相互比拟。甚且因心色两法，若无法应用自如时，亦难以用『直心成佛、真心为正』，若用『虚心为谦、无心治妄』来成全。若受虚幻所左右时，根本一切都是无法开展，因而此无心即是没有『分别、对错、是非、正邪』，是无任何之一切『心所法』而显现。乃在于正是『无所为心』之主要，即是无为心、即天地之正心、宇宙之正心、无极之正心、虚空之正心，此一切之无所心即『无心』！

　　无心可治妄，妄想入虚幻，人生是如此，而一切有情修持者，若无法达至此无心，必然难以更上一层，去成就自己及一切有情的芸芸苍生。因在一切圣神、仙佛、菩萨，皆是通过了『直心、真心、无心』来启发，更且运用了『虚心』去求教于仙圣等来指点迷津，又由自己去体悟人生之过程，世界之过程，一切有情、非情可以成就的过程。如此无心者并非是无此心，乃是『无之无无心』、即可破除『无之无无明』，更可达至『本明』的境界。如此必要能用无心，去化除人生短暂虚幻的过程，因虚幻的人生，正是妄心所展现。而此妄心若不能化除，欲成就圣神、仙佛、菩萨的位阶，即难矣！更难成就。

　　　妄心入迷痴　化除可成智

　　　了悟归正知　必可消业除

　　人生常是无奈，又无可奈何，乃因因果果反覆相成，而左右人生的一切因果循环。若无法了知，必然是在每一世世的轮回之中受影响，即受到智慧提升的限制，因无心可治妄，妄想即是一切生存主要作为。不可言此现世为虚假，乃在于是把现象界，一切虚幻妄想之心，化作可以成就的因本，否则若是修持了一大段路程，尚且执着于润生、修持、作功德而沾沾自喜自己是个修持者，那真是令仙佛见笑，难以达成修持的改造与成就。

　　　虚幻妄想是迷痴　了悟真实业障除

　　　一切人生真实本　仅在此心受无知

　　　无心治妄显真理　直心成佛了天义

　　　真心为正是天机　虚心谦怀更成器

　　能了知一切众生轮回的因本，乃是有所求，有所不足、有所执着。可否在『无之无无心』之中，了知天地的演绎德泽、无极演绎德泽、虚空演绎德泽、宇宙演绎德泽的过程。必然可以了知人生演绎的过程，正是化除此『无心治妄想』的观念，乃观自己的所作所为而抉择。又众生受妄心所执，才造成了一世世，反覆轮回的主要原因，盼了知治妄心，须用无心必可化除。
