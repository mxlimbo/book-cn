第廿六回 感恩天地演绎

虚空地母至尊 降

八十八年九月廿一日 己卯年八月十二日

　　　虚空人间贯一理　地母德慈众生依

　　　至尊登台着佛心　感恩天地来演绎

　　众生在千禧年前的种种恶殃，乃在于是前造业至今来完果，其过程是以个己的『私业』与『共业』的组合来作评判。其目的以个人之本身与群体的共业所促成，故在天地演绎之中，是必须要让世人在千禧年前有所警惕与忏悔，方是在天地演绎的过程中来作注解。其真正导引的目的，是让人类以万物平等、众生平等，切勿执视人类是最高、最大的物种，乃是以人类为依归，又以天地演绎的德泽来彰显。

　　众生在宇宙天地间，承受着上天的德泽，在作润生与修持的规范中，是要把人类世界在此『千禧年』前的种种恶殃来化除，也要由自己来偿还与受报。其最主要的原因，则是人类世界无法了知的过程中，必要是『众生平等』，不可执视人类倚万物之灵长来作统御，其背后正有天地间诸圣神仙佛来替众生作清算。故在于自己本身前业所定数，又今业来完除，其在人世间的业因、业果、业障的形成，乃是符合因果法则的定义。

　　天覆地载正是生存在此有情世界中的芸芸苍生，所必须要了知的过程，人类世界会常倚人势而加害其他万物苍生，天地即降灾殃来收拾顽辈以警惕世人，在生存的过程中，不可存有是万物之灵长，而残害万物众生。故在天地演绎的德泽中，是必要常发善念来护佑芸芸苍生，才不会陷入再轮回的漩涡中，乃是人类世界的有情众生，必要能明晰的。因此天地警醒人类世界的灾殃中，必要了知当今世界的灾难，更是层出不穷，并非仅是千禧年前而已，尚且更在千禧年后，观人类世界的种种行径作为，来观察是加强灾殃或是减少灾祸的降临，来收拾与警醒世人。乃是让人类世界的芸芸苍生，能更警惕不可执视自己为人类万物之灵长而高傲！

　　天覆地载是不可倚人类为万物高等物种众生而视之，乃是一视平等、一视同仁，皆在于如何了知天地的德泽中，让芸芸众生皆能明晰，此天地正是演绎着无极、虚空、宇宙、大寰天地的一切三千大千世界中芸芸苍生。因在太阳系的天地中，人类生存的地球，正是在遥远他方世界的星球中之一小部份，其他星球的科技，比人类世界的科技，更超越千百倍。如此人类世界的众生，若不能有天地观、无极观、虚空观、宇宙观，皆在此人类世界的狭隘简窄的空间中，做自己横行霸道的生存而已，必然自己无法了脱此天地间的生存。故而吾〈为师〉等掌管此天地间，为何欲冠上『虚空』之名讳，乃在于正是把此宏观的眼光放大心胸，不论是无极、虚空、宇宙的演化过程，如此在此天地间必要了知人类世界的生存环境，正是改造下世再轮回的种籽而往上提升。若能了知其生存在无极、虚空、宇宙的生存方式，必能放宽心胸去容纳更多有情世界的芸芸众生，如此之心胸正是无极演绎、虚空演绎、宇宙演绎的过程，不要仅在人类世界，亦要把此心胸敞开去开展更宽远的世界。

　　人类世界在此天地间，正是符合了此生存的空间，无论在此世界是多么的尊贵或卑贱，皆为天地所护佑的众生之一，亦不可轻视其他物种，或高傲标榜自己，乃是对自己一种无法向上提升的阻碍也。必须明白了知在此世界中的天地间，如何去感恩此天覆地载的德泽，及其他有情众生所供给的食物链。故而在此世界必须要了知其过程，正是自己能否『超越与下坠轮回』的分界点。感恩此有情天地中的一切芸芸众生，皆是自己的同胞，自己的同类物种，理应不分彼此，皆是天地所眷顾也。也是对自己的仁德展现，更是对芸芸众生，有一份长养爱护的责任与德泽。也是学习天地的德泽，相互应用、相辅相成，故在此正是把此世界的种种行径，来作改造回归的资粮，更入于实行的位阶才是真实性也。

　　今日之台疆所发生的灾殃〈大地震〉，乃上天警醒在此中土台疆炎黄子孙，必要了知的，尤其在此千禧年之前后，更要小心如是，更且要改变以往，不可存有倚人类科技来欺侮其他万物物种众生也。在此灾殃的警醒，则非是仙佛、圣神之不慈，乃在于私业与共业的形成所致。『天能奈何！地能奈何！又人类能奈何也！』是以提醒台疆的众生，来作教化的因本。因而必要能了知天地的演绎过程是如何？又以天地之不序，又对人类之收拾，更是比人类所作所为来的更可怕也！盼了知天地反扑人类的造恶，皆是因果循环也！
