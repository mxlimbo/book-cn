第廿七回 无极演绎

骊山老母 降

八十八年九月廿四日 己卯年七月十五日

　　　骊歌初唱台疆殃　山形移位众生障

　　　老者少者皆受厄　母心无极演绎详

　　天地的演化过程，乃由人类的褔祸来作导引，故在台疆之中，能以道气充塞横溢于大千世界，乃中土炎黄子孙所促成。而今上苍警醒在此天际的同时，必要是让芸芸众生来了知天地的演绎，更是胜于人间的一切。故而人定胜天亦要天所容许，才能致之，果若天地之不容，而言人类可以胜天，乃为谬论也。

　　当今时代的众生正处于古往天地中的「三十六天罡」、「七十二地煞」所幻化的大千世界，人类所倚靠的，无非是上苍降褔泽于人世间所促成，乃在是对众生有一份长养教育的责任。故在当今时代的芸芸众生，尤以人类世界绝不可倚人势，强加迫害其他万物物种的芸芸苍生。反过来言之是把人类世界中，一切有情非情的物种众生，皆能保护能眷顾，使其欣欣向荣，又无所惧怕。一切生存在此人类的世界空间之中，乃是以天地演绎所扮演的角色为基准。是对众生能平等视之，平等爱顾也。又因人类世界常加以欺侮，侵害其他弱小动物，故而上天演绎所促成的业因、业果、业障、业殃、端视人类世界的种种行径来抉择。

　　天地的演绎，是必能将天地的德泽彰显表露出来，但在于天地所展现的，正是相对的平等，亦只在有相互对待的人类世界，故而无法能有「纯一」演化过程，其乃在于是以太极所演绎的过程中，无法回复于「纯一」。因而在太极中即是两相对待的世界，即无法具备于「纯一」的境界。乃是受相互对待所左右，但在「纯一」正是超越了普渡收圆的阶段，则是入于收圆的超越。正因是三曹普渡的收圆，是必要能回复原来的｜圆，才是圆满自己，并非仅在于收归原灵而已，是必须要圆满自己，再去圆满众生，乃众原灵所必须要有的责任与使命。

　　无极演绎的过程——任何一位原灵的本质皆来自无极，又无极所演绎的正是把回归来作准备。因在回归的路程上，常会受自己无明所阻碍，故在收归原灵的本位之下，即做回归的准备。而在一贯天道所扮演的角色，正是以收复原灵作回归，在无极所演绎正是如此。然而在母娘系统，慈惠系统，天道系统，以及相关的一切系统中皆都如是。尤以收归原灵在一贯天道中，是较有成绩且有实质效果，必以一而贯之而谓之真道。如此的天道系统中，必要详加了知的，仅在于收归原灵作回归而已，是无法能超越其本来之位境。

　　在上苍所演绎的过程，是以掌新法落实于人世间，如今必以「母娘系统懿敕道脉」的崇心作为依归，不但是引归原灵，更是在超越中作资粮，甚且入于大乘菩萨行的觉有情境界，直入莲花佛国的净土中，去做更能超越的境界与位阶。如此在无极演绎的同时，即当下中能入于崇心道脉，甚且入于大乘菩萨行的觉有情境界中，再入于莲花佛国的净土中，去做更能超越的境界与位阶。如此在此无极演绎同时即当下中，能入于崇心道脉甚且是未来的「真佛心宗 」，是以崇心为根基，把大道演化的精髓下化于有情世界，能让芸芸苍生皆能畅真理而回归。因此崇心的道务即非比寻常，此乃上苍观照﹝视﹞了诸贤儿等，长达七年的时光中，所展现的坚志而致之，并能了知崇心道务整体的发展中，由最不如意的过程，并且是以最无任何接济与资粮中来作提升，来作推广。然而在上苍所能配合的正是以此作为依据，并且观视其主事者，及诸贤儿等的心态来作筛选，如此在未来中必能见真章。

　　如是而言，崇心的大道系列是以成就未来的有情众生，在提升自己的灵性，往上升华的同时，必要能了知是作无极演绎的先锋，更且创造虚空天界，回归超越的实行位阶与境界。如此必是严格要求诸贤儿等之配合，果若在筛选中脱离了崇心道脉『真佛心宗』，而在往后中，又要入于弥勒净土再去修『唯识理论观念与行径』，必然是无法在此殊胜的当下中，能直超无极理天入于虚空天界，在现今的当下中见真章。如此必须要了知，弥勒净土是须等到龙华三会才能促成，崇心乃是超越『龙华三会』直入莲花佛国的最佳见证也。

　　真佛心宗是禅宗的根本理论，加上显密两宗演化唯识宗的种种行径，再加上道家未来的精华着作而成，更是一册册接续不断，而一册册更能连贯于大道系列中，是对未来一切有情的众生在自己的所作所为，入于大乘菩萨的觉悟有情世界的芸芸众生，正是菩萨位与佛国境界了，因此必然超越于弥勒净土，根本都不必再来人世间，乃是崇心所扮演的角色，如此亦观自己的努力与否，而决定之。

　　　天地演绎众生灵　回归资粮必要清

　　　准备丰盈无阻碍　超越本位更理心

　　　真佛心宗演崇心　禅宗显密皆本心

　　　佛国进入莲花体　虚空天界自相迎

　　以此同有情众灵儿共勉之。了知着书的原意必然能对未来的圣务有所体会也。
