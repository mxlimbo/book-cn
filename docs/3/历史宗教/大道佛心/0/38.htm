第卅三回 灵性世界

南无阿弥陀佛 降

八十八年十月二十二日 已卯年九月十四日

　　　南无至上本灵生　阿弥陀佛引众升

　　　灵性世界超越乘　大道佛心归自成

　　由泯心泯境与存心存境，入于的是以灵性世界的精神领域，往上提升自己在生存的过程中来做统合。因是对自己的灵性往上提升的当下中，是必要能把精神领域，化作自己在生存与润生过程，则必须将自己的灵性来作润合〈统合〉。亦是自己的灵性来作精神领域的润合，即是给于自己能有真正的行持，更能破除了一切虚妄，虚幻的人间假景。然而在实相世界是难以见至灵性主体的本性中，又如何能了知其境界的高下否？

　　实相界为器物世界，若以实相界而言即是器物世界，在天地人三才串连的人类世界。若无法将精神领域的灵性世界来润合，必难以同大宇宙、大虚空、大无极、大天地来做演绎的过程。又在神性〈灵性〉的世界中，皆难入空幻的心念意识，来做提升超越，乃人类世界精神领域的最主要原因。在一切法界中不论色相、心相，则是必须要两相结合。否则若沦于仅有器物世界之中时，必难以在精神领域的当下内，去成就自己灵性世界向上提升的原动力。然而灵性世界，正是串连着大地众生、天地众生，在此大地、天地中的芸芸苍生。必是觉悟有情的万物物种，乃在于对自己灵性的超越与否？即仅存在于人类世界的人种而已，因而在人类世界是可以做修整、做改造，不使沉沦于『轮回』的漩涡中。

　　天地有情众生有爱，即是对一切芸芸苍生，才有的作为，是以精神领域的提升，勿使沦为器物世界，即对于自己的灵性增长与否，乃是要靠自己去追求，自己去创造，必然在于精神领域的同时中，把灵性世界来提升，但必是要了知灵性世界，不仅含藏过去的累世因因果果，来形成其对自己的本身是做改造、是做修整，必要能了知人类世界的生存过程，是一切生命的改造界，又人类世界仅在于如何能迈向精神领域的向上提升。亦不仅是在对万物苍生一种的德泽。亦是自己愿意与否的判别，如此乃作为自己灵性增加的原动力，亦即是对自己在万物同等、平等的原则下来论断其公平性的原则。

　　能同天地万物众生合为一体，即是精神领域的灵性世界往上提升的当下。必要能了知在天地循环的过程，与运转的顺序中，必要能将物我合一，即是做到『合为一体』，我即众生，众生即我，我即天地，天地即我的范畴。乃是以自己的本质来行持『无缘大慈，同体大悲』的展现，然而在人类世界的芸芸苍生，正是欠缺此一良帖，故而常仅会以自己私利的眼光，来观看一切有情众生，才造成汝攻我奸、尔虞我诈，因而即难以做到同天地，同万物众生，合而为一。更遑论同虚空、同无极，或大宇宙合而为一，乃是谬论。

　　一切的过程皆是以全体的总合体来论断，能以我即无极、我即虚空、我即宇宙也！乃是对一切的物种，皆能下化于芸芸苍生，才是将自己灵性来升华。又在于万物苍生的器物世界中，根本都无法能融入『灵性精神世界』。因而在人类世界，是必要能把精神灵性向上迈开脚步，才能一步踏稳，再一步来提升。正是要以灵性的本质来作融合，才能在精神的领域中，入于能泯除其心境，泯除其外境，正是把『一合相』、『一境性』来做更深入的展现。我即天地、我即万物、我即众生、我即无极、我即虚空、我即宇宙也！能有此更深一层的印证，不仅在灵性世界中来成长，更是把此无极、虚空、宇宙来融合为一体，那才是对自己灵性，有一份真实的长远。若是无法达至此境界，亦在于对自己的虚幻、妄念、执着、分别所影响，才是促成自己无法往上层去做提升、去做改造。

　　灵性世界，是融合了一切的世相，甚至把此一切的世相，来了知、来判别。能融入其与自己本身，必是超越了三度空间的左右与限制，一切的有情众生，正是欠缺了此一观念，更且因自己有我执、分别、妄念、虚幻，来阻挡自己能向上提升的原动力，此乃最严重的错误。灵性世界是要破除一切虚幻、妄想能融入于器物世界的存在过程，将精神领域的灵性世界，入于彰真如、显佛性，则早已在人类世界中，已经成就。而不必等到殁度之后，再来论德性与功果。更是早已具备成就、超越的资粮位，又已是入于实行位中了，再去做圆满位的行持，才是成就自己于当下，成就自己于未来中，盼了知灵性世界是物我合一、天地合一、无极合一、虚空合一、宇宙合一也！
