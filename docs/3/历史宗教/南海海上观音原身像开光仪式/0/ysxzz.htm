――南海海上观音原身像制作过程

2000年11月12日，南海海上观音10.8米原身像于南山举行开光庆典，展示了观音圣像具有的三个世界独创特点， 即"海上108米"、 "一体化三尊"以及"白衣观音形象"，这不仅是科学地继承和发扬传统造像艺术的成果，也是勇于克服由于创新而带来艰巨困难的成果。

在1999年的观音像样稿创作中，"一体化三尊"的困难得到了初步克服，但要达到"海上108米"的建设高度，尤其是要制作出同时符合宗教与世俗审美要求的"白衣观音形象"，就必须解决像体与海洋环境协调、具有佛性和飘逸感以及由此而带来的材料、承重、防腐蚀和颜色问题。 因此，海上观音圣像难于采用传统石刻（乐山大佛）、木雕（承德大悲金刚菩萨）、分层铸造（日本14米8高的奈良大佛）、以及选取锡铜采用分块铸造拼装焊接的方法（无锡灵山大佛）。

108米海上观音像要成为时代的佛教文化瑰宝，将经受百年历史的考验，首先要在10.8米观音原身像上集中解决各种造像难题。敬造工程的策划者和造像创作小组知难而进， 在1999年北京审稿会的基础上， 经过反复翻模、论证、评审、完美，确定了10.8米观音像样稿。样稿经历了泥像、石膏像等严密的造像过程，最终创造性地选取合金钢采用锻造工艺（传统上称为金牒法）。

合金钢白亮光洁、较青铜不易氧化变色，是保证白衣观音形象最合适的材料。圣像表面喷涂高分子薄膜， 15-20年内膜层不会出现粉化、龟裂、脱落及其它老化现象。定期的维护保养，可保证观音像百年的供奉期。

“南海观音”历来以白衣形象出现。“白衣观音形象”符合宗教与世俗的审美要求，并能与深色的海水形成反差，增强视觉效果。为确定能充分体现“白衣观音”“白”颜色特性，前后进行了数十次的试验，不是太深而显得呆板，就是太浅、太花而显得没有佛性。南京晨光集团的领导曾对工程负责人说：“佛像已经有了两个世界独创，颜色风险太大，算了吧。”当时也曾想恢复传统青铜色，但考虑到白衣观音特征，这么大的投资最终搞个没有特色的佛像，心不甘，经过不懈努力，最终调出理想的颜色--白色中透着青色，象玉石质地。这是继南海海上观音"一体化三尊"的又一创新，运用新工艺和新技术实现了“白衣大士”的形象，这在佛像造像艺术上是一个创举。

2000年3月30日， 原全国政协副主席、中国佛教协会会长赵朴初居士在病房审定了海上观音像小样及工程方案，高度评价造像工作"如法、如仪"。

10月18日， 原身像制作完成运抵南山文化苑。 原身像在工艺、表面处理、色泽、效果以及佛性、协调性、相容性等方面都相当完美，朝具有震憾性的佛像艺术效果更进一步，得到了高僧大德和社会各界的广泛认可。
