“南海海上观音”圣像，是正观音的一体化三尊造型。每尊观音菩萨各持不同法器，代表了观音菩萨的不同化身，反映了观音菩萨救度众生的方便法门。南海海上观音形象，集中体现了观音菩萨“千处祈求千处应，苦海常做度人舟”的无缘之慈和同体之悲。

　

持箧观音

    持箧观音代表般若德即智慧德性。经箧代表六百卷大般若部经书，大般若部中的《心经》用最精练的语言揭示了宇宙人生的真理，是一切智慧的源泉。持箧观音以实相、境界、文字、方便、眷属五种般若启迪人们的智慧，教人在日常工作中通达无碍，在生活学习中圆融自在，臻至“千江有水千江月，万里无云万里天”的境界。

     “诸事无常”应“看破”——以“智慧之剑”断“烦恼之丝”，所以持箧是心、是智、是安心、是立体、是对内、是见道、是依理、是空义、是真谛……

　

持莲观音

    持莲观音代表法身德，即清净无相本体。观音菩萨是密宗莲花部的本尊，莲花清净，出污泥而不染，是佛门中的圣花，表本体清净及六根圆通。

    “诸法无我”需“放下”——“物来则应，物去不留”，常念“心、佛、众生三无差别”，所以持莲是色、是境、是起行、是显用、是对外、是修道、是依事、是有义、是俗谛……

　

              

                  持珠观音

持珠观音代表解脱德，即彻底摆脱无明烦恼和种种束缚，达到大自由大自在的境界。观音菩萨心心与佛相印，念念不忘众生，上合佛智，观理事无碍而了达自在，下合众悲，观念一切众生之机而化度自在，表现“众生念佛、佛念众生”，接度一切有缘同等同体的慈悲精神。

 “涅槃寂静”得“自在”——无所谓“入世”与“出世”之分，“一切世间法，皆是佛法”，所以持珠是妙观、 是色心一如、 是理事无碍、是空有不二、是真俗双融……
