活得更聪明

第六节

尊重你的眼光

　　电影，卖得好的是好商品，
　　卖得差的未必是艺术品，
　　你可以得到一点吉光片羽的就是好电影。

　　对于电影，我是个业余观众，也是别有居心的观众，更是一个偏见十足的观众。

　　如果能将名牌导演、巨作及各种电影术语朗朗上口的电影专家打一百分的话，我的电影成绩连十分都不到吧！

　　我没有看过大多数的经典名作，我喜欢看好笑的好莱坞电影，有大明星的电影，紧张但不太刺激的动作片，温馨但不太狗血的家庭人伦剧（最好不要是大悲剧），香艳逗趣但不要有太多难以理解行为的法国片。歌声动听影像cute的瞳通片。我讨厌看鬼怪惊悚、坏人得到胜利、大家抱在一起哭成一团，或斗争斗得我喘不过气、每隔几分钟就会有一个人有点变态的片子。

　　所以我说我是个很业余的，没办法登大雅之堂的观众。

　　我常从电影或电影理论中"窃取"观点，所以我别有居心。有一阵子我"窃取"导演大卫·马密《导演功课》中（摄影机摆在哪里）的观点来写小说，让自己的眼睛变成摄影机，于是我比较不担心自己的小说会流于单调的问题。

　　"小说"的镜头和电影的镜头有异曲同工之妙，它必须让人愿意继续看下一个镜头，有时窃取的是一种感觉。看完一部电影，我会问自己，最感人的是哪一个镜头。比如说"麦迪逊之桥"吧！我唯一有感觉的是梅利尔斯特里普握着车门把手，想跳下车和情夫私逃，却又眼睁睁离开的镜头。这个镜头，原著中没有，有点狗血但偏又惹人怜爱，很多女生和我有相同的感觉——我找出一部电影中感人的感觉，开始研究，它究竟是如何感人的？如果是我的小说，应该在什么地方有这样的感觉？

　　有时窃取的是一种生活的感觉。用伊塔尔·卡尔维诺的形容深得我心，他说："电影是一种逃避……而这一点正是我需要的，满足我对异乡、将注意力放到另一个空间去的渴望。"我也爱看早场或午场电影，因为"半真空状态的大厅，像是只属于我一个人的"，有一种"占便宜"的快感。

　　我也偏见十足，因常看非所看，我从电影中读到的多半不是编剧或导演对观众说的话。以近来的电影来说吧！我从《英伦情人》看到谎言的正面价值、《新娘不是我？》看到的是自恋。从《接触未来》中看到的是生的偏执与死的惶惑；从《空军一号》看到人类无可避免的自私……就跟我很久很久以前看《江山美人》、《乱世佳人》、《梁祝》的N轮片而归纳发现，所谓爱与不爱，常出自于当事人的一时任性而已……

　　电影，卖得好的是好商品，卖得差的未必是艺术品，你可以得到一点吉光片羽的就是好电影。许多封上"艺术品"的都如是。

　　用你的角度和眼光看电影、看书，管他专家怎么说呢！

你大可以超越痛苦

　　不要等待别人解决你的苦，
　　只要愿意，
　　你可以超越它，
　　一抢毙了你的痛苦。

　　有一只兀鹰，猛烈的啄着村夫的双脚，他的靴子和袜子被撕成碎片后，它更狠狠地啃起村夫双脚来了。

　　正好这时有一位绅士经过，看见村夫如此鲜血淋漓地忍受着痛苦，不禁驻足问他，为什么要受兀鹰啄食呢？村夫答道："我没有办法啊！这只兀鹰刚开始袭击我的时候，我曾经试图赶走它，但是它太顽强了，几乎抓伤我的脸颊，因此我宁愿牺牲双脚。呵，我的脚差不多被碾成碎屑了，真可怕！"

　　绅士说："你只要一枪就可以结束它的性命呀！"

　　村夫听了，尖声叫嚷着"真的吗？那么你助我一臂之力好吗？"绅士回答："我很乐意，可是我得去拿枪，你还能支撑一会儿吗？"

　　在剧痛中呻吟的村夫，强忍着撕扯的痛苦说"无论如何，我会忍下去的。"于是绅士飞快地跑去拿枪。但就在绅土转身的瞬间，兀鹰蓦然拔身冲起，在空中把身子向后拉得远远的，以便获得更大的冲力，如同一根标枪般，把它的利喙掷向村夫的喉头，深深插入。村夫终于等不及地仆死在地了。死前稍感安慰的是，兀鹰也因太过费力，淹溺在村夫的血泊里。

　　卡夫卡的寓言，大部分并不好懂，这是一个稍可加以诠释的小故事。

　　聪明的你会问：村夫为什么不自己去拿枪结束掉兀鹰的性命，宁愿像傻瓜一样忍受兀鹰的袭击？我想，兀鹰只是一个比喻，它可以象征着萦绕人生的内在与外在的痛苦。其实，任何一个凡人，都会不知不觉地像村夫一样，沉溺于自己臆造的幻想中，痛苦得不能自拔，甚至，"爱"上自己的痛苦，不愿亲手挥掉它，尽管是举手之劳而已。

　　卡夫卡另有一段格言，正是深明人身种种苦痛的洞彻哲理："人们惧怕自由和责任，所以人们宁愿藏身在自铸的樊笼中。"所以，村夫与他臆想的痛苦（兀鹰）同归于尽，然而这个寓言也悄悄地告诉我们：不要等待别人解决你的苦，只要愿意，你可以超越它，一枪毙了你的痛苦。

　　卡夫卡一直在教人们，以毅力、决心与爱心解脱人生痛苦，以体会进入人生更高境界的快乐。其实，他并不是一个会让人感到言语生涩的哲学家。

"以夫为尊"与"个性不合"

　　如果你拥有很多统一发票，
　　你可能比别人多点机率对到奖，
　　但是你未必会对到奖。

　　不是很久以前，某女明星开心地宣布，她要与某"花名在外"的男明星结婚。结婚没什么了不起，天天有数不雇的人在结，但对某些女人来说，夸示自己以贤德贤能击退众多貌如花的竞争对象，使漂泊的男人愿意点头说I do，收拾得浪子归心，至少是"妇功"一项。一时间大小报刊"据实"报导了，一时让我觉得我们又回到歌咏"三从四德"的时代。

　　大众媒体影响所及，使许多女人勤练"堡汤"的技术与"柔顺"的技能。看，只有"以夫为尊的女人，才是最后赢家！"一位新婚后积极辞职洗手做羹汤的女友对我说。

　　我不敢有任何"不驯"的表情。毕竟我知道，对某些女人来说，出人意料地掳获一个浪子，就跟西部牛仔套住一匹横冲直撞的野马一样，是值得夸耀的人生光荣史，至少那是生命中足以让他"做视群伦"的时刻。

　　没多久，"以夫为尊"的女星宣布退出苦心积虑的婚姻。原因呢？外传是"第三者"，男性当事人理直气壮的说法是"个性不合。"

　　每个因"第三者"出轨的婚姻，想先说再见的那个，总不愿承认是外在诱惑使他变心（谁愿背负重色轻义、喜新厌旧的担子呢？）"个性不合"是最好的说法了，背后的意义是："物必自腐而后虫生，不能怪我。"

　　谁跟谁，个性真的合呢？个性不合，谁有错？我又听到两面说词，幸灾乐祸地说："哼，以夫为尊，还不是有这样下场！"正义使者说："这男人真是混蛋，人家这样对他，他还是狗改不了吃屎！"

　　都偏激了。

　　如果你拥有很多统一发票，你可能比别人多点机率对到奖，但是你未必会对到奖，还有，即使你拥有再多的统一发票，尽心尽力搜罗，也未必保证你能对到奖——降贵纾尊臣服于男人种种要求的女人，在这个还很强调"男性尊严"的现实世界，是拥有比较多统一发票的，不过，不代表你会一直对到奖。

　　"有台阶下，就要下"是现代人对待感情的理性法则。你不觉个性不合，他却说个性不合，彼此下台阶，还算有风度。

感情的三不政策

　　不主动的原因很多，
　　"怕受伤"应是并逼的心理恐惧，
　　偏偏感情是不能"只许成功，不许失败"的。
　　敢爱就一定也得敢受伤。
　　很趣的情感处理态度：不主动，不拒绝、不负责。

　　且莫讨论媒体曾经努力炒过的香炉问题，就让我们来分析这个男人谈感情"新三不政策"好了。

　　"一般人"如果在感情上采取这三不政策，是什么原因？还有，大概会有什么结果？

　　不主动——这是很多年逾适婚年龄又大叹芳心寂寞的男男女女的大问题。很多人自觉条件不错，但似乎无人欣赏。为什么？我想原因可以归纳为几种：

　　一、不肯开拓生活圈。如果出校门时并无男女朋友，从事的行业极度阳盛阴衰或阴盛阳衰，加上工时甚长或下班后累得像条喘气的狗，星期天只愿赖在沙发上看电影或录影带，你当然没机会。

　　二、对爱情有错误的假设。某些看多了文艺爱情悲喜诀的女生，常以为一个男人要追你追得死去活来才叫爱情，并不知道绝大多数男人没有那么勤劳（当然，勤不勤劳也因对象条件而异啦），你一摆高一点姿态，他们就掉头而去。

　　三、有些人眼高手低，以为只有自己在选人，其实别人也选你。喜欢的人不喜欢你，或你不敢追；喜欢你的你不喜欢，当然也不能屈就。

　　四、有些人则是因为自己的"类型"与自己"理想对象"所欣赏的类型无法match而空等待；有些自认"找不到对象"的人根本没认清自己，比如一个体格健壮、脾气也健壮的女人，却偏偏把自己当成"我见犹怜"的林黛玉，当然贾宝玉是不会出现在她跟前哄她的……

　　不主动的原因很多，"怕受伤"应是共通的心理恐惧，偏偏感情是不能"只许成功，不许失败"的，敢爱一定也得敢受伤。不主动，你得永远接受"没鱼虾也好"。

　　不拒绝——不拒绝的人胃口必定好，这种人的"感情盟友"类型一定广泛得不得了。我曾听过一个例子：一名男子同时有多名女友，一个是女大学生，一个则是比他大二十岁的阿公店妈妈桑，一个是会计小姐，一个是青梅竹马的邻居……他照单全收。一群女人大军压境逼问他，他到底爱谁？他真的不知道，他只是嗜爱的动物罢了！他爱谁一点都不重要，有人爱他最重要。他的情人们应看清这一点。

　　不负责——不负责说得潇洒，其实不易。在爱情中，人人自我负责，不叫情人负责，是高尚的理念。但谈感情无怨无悔又不求别人负责的人，还真难找到。刚开始说没关系的，用情一深，难免一哭二闹三上吊！

　　我想，我们都没有超能力或超魅力，还是懂得主动、拒绝与负责（不负责得先说得很清楚，还得确定对方会不会忽然翻脸），才不致惹上多余的麻烦，或空留遗憾！

爱情白日梦

　　有人通过真实生活中的爱情经历，
　　在反省自己不知不觉承袭的爱情观；
　　有人则不知不觉地作着爱情白日梦，
　　多次恋爱，多次自苦，
　　原来只是因为没在"一念之间"交通。

　　从小你一定看过很多重话，不然也一定看过一些知名的卡通片，像"睡美人"、"白雪公主"、"灰姑狼"、"青蛙王子"、"野兽王子"……还有"美女与野兽"。

　　我怀疑，对我们长大还在作的爱情白日梦来说，这些从小耳濡目染的伟大爱情，不朽童话，就是最好的温床。

　　这些美丽的故事都很温馨感人，而温馨感人的故事都有一个模式。大致可以清楚地看到两个线索：

　　一、女人等待男人来拯救或提高——她们不是像睡美人、白雪公主一样死死的躺在那里，等理想男人让她复活；就是像灰姑娘或"美女与野兽"中的美女一样，等着看上她的男人让她变得富有，变成贵族或脱离贫困的生活。

　　二、女人不要怀疑她的幸福，绝对要信守承诺、为爱拿生命当赌注——比如"野兽王子"中，女人听信姐姐们的话违背承诺，半夜拿蜡烛看王子，结果让王子"功力尽失"变成野兽；"青蛙王子"中公主没有依约吻青蛙，因而承受了捉弄和惩罚；"美女与野兽"中的美女，为了使野兽脱离魔法的控制，美女拼尽全力，才获得真正的幸福。

　　有人通过真实生活的爱情经历，在反省自己不知不觉承袭的爱情观；有人则仍不知不觉地作着爱情白日梦，多次恋爱，多次自苦，原来只是因为没办法在"一念之间"想通。

　　我的这些感慨，来自最近在广播节目访问一位新锐作家。她说她失恋失得很惨，过去常因自己没办法"安定下来"而惶惶惑惑，直到最近买了间小套房，才发现一个人生活也可以很安定。但朋友们对她"只"买十多坪的小套房，都"不自觉"地发表了看法。男生说："买这么小的房子，将来结婚、生子，地方不就太小了。"女生说："买房子做什么，将来嫁出去，找个有房子有车的就行了。"她从这一点悟到，原来男人女人都像"睡美人"一样，在等待异性的拯救与提高。

　　我也常遇到女性朋友对我说："真羡慕你可以常去旅行。"

　　"你也可以啊！"

　　"我啊，不必了，也许……等我把自己嫁出去后，再看看有没有希望了！"

　　在经济上和生活上和思想上已然独立的女性，却很难脱离"冥冥中"有个男人就有救了的心态，是现代女性不知不觉的爱情白日梦。这样的爱情白日梦，像只恐龙，是不是从小就在我们的意识中下了蛋呢？

　　请不要继续孵化它。

你为什么不爱我

　　女人特别爱"具体而微"的理由，
　　所以常因此会做胸部整型、阴道整型，
　　白白挨了一刀后，
　　往往会发现，还有别的理由。

　　"别的女人有的我也有，你为什么不爱我？"

　　下雨天一边开着车，一边听着歌。密闭的车窗把雨声挡在外头，所以我可以静静听着流行歌。

　　听见这两句悲苦的歌同时，我情不自禁地笑了，很没同情心地笑了。

　　别的女人根本没有我漂亮，为什么你会爱她不爱我？你的眼睛给蛤仔肉粘着？这是很多老公有外遇、男朋友有第三者的女人大多会发出的抱怨。

　　爱不爱，大多是跟"别人有的我没有"无关。如果他（她）说得出是因为别人有什么，而你没什么不爱你，那根本不是爱，是势利。你失去了他，又有何可惜，当庆幸的是，他没有继续伪装爱你，或沉溺于对你习惯性的爱吧！偏偏我们的头脑没那么清楚，不从"不可说"之中去寻找答案，硬要一个可说的具体答案：你为什么不爱我呢？我哪一点比她差呢？

　　女人若开始比较，总会发现，上天对人大部分都公平，她总有比你好的。

　　如果她不比我年轻，就是比我温柔。

　　如果她不是比我温柔，就是比我胸部大。

　　如果胸部又不太大，一定是比我会穿衣服。

　　纵可以找到一些理由，历历指证负心的人，如果真能使我们好过些，那么，这些理由还算是安慰自己的借口。

　　面临第三者事件时，你对变心情人的指控，代表你对曾经拥有感情的评价。分手时把情人说得烂得无以复加的人，其实老早就知道，物必自腐而后虫生，你老早不想这样的感情，只是在骑驴找马，只是讨厌他这个混蛋先下手为强，他竟敢先不要你？

　　"别的女人有的我也有，你为什么不爱我？"女人特别爱"具体而微"的理由，所以常因此去做胸部整型、阴道整型，白白挨了一刀，当你有了别人也有的以外，往往会发现，还有别的理由。
