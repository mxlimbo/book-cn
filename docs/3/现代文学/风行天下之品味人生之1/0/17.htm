"你也有今天!"

　　一位曾经年轻的女诗人，有一天给我打来电话，因为是老熟人，所以电话里免不了多说几句。正说着，我听见她在电话那头对保姆说："这个孩子没洗澡，那个孩子没喝奶。"完了她又转回来继续给我讲。原来她生了双胞胎，现在正奶瓶尿片满天飞地忙着带孩子。这哪像女诗人干的事?犹记当年，她人的风采与诗的风采并驾齐驱，曾惹来多少追逐与羡慕?我忍不住开玩笑说："你也有今天!"诗人，尤其女诗人，给人的印象应该是仙风道骨，不食人间烟火，女诗人闻言苦笑说："一旦生了孩子，走上了所有女人都走过的路，又怎么能不婆婆妈妈?"

　　不久前的一天，我走在路上，迎面碰见一位步履蹒跚的老人，正由老伴搀扶着挪步。他的目光呆滞，嘴角歪斜，流着涎水。我怎么都不能相信，这位老人就是曾经名躁一时、威震八方的某位要员!他整人的时候，多少人梦里都怕他，他坐在主席台上讲话，又是何等风光!而今他却沦落到这种地步，我内心不由得长叹一声："你也有今天"!

　　说人家容易，说自已也不难。有一次羽毛球比赛，这个我在本系统保持多年第一的项目，轻易旁落他人。明显看到的球，但是体力不支，腿脚跟不上。年轻队员打得轻松自如，我却累得气喘如牛，最后还是丢了冠军，我虽不敢自比廉颇，却也不得不对自已说："你也有今天!"

　　撇开人生追求的内容不讲，只说自然规律，每个人，都必须走生老病死这条道。你曾经年轻，如今不年轻；你曾经美丽，如今衰老了；你曾经欲玩世界于股掌之上，而今世界把你忘在角落里；你过去曾参加别人的追悼会，终于有一天，轮到别人为你送葬……这人生的轮回，谁又能摆脱?


　　所以一想到有人会在某一天，也如我问"你也有今天"一样问我，灵魂深处就有了一种紧迫感。人生苦短，上寿不过百年，我们有什么理由不珍视今天，把握自己，充分使用每一分一秒呢?一分一秒，正像我们一滴一滴的血液，时间用完了，血液也耗干了，这有点像游戏机中的人物，少一滴血，少一分生命——你浪费时间，就是浪费自已的血液，浪费自已的生命!无论你曾经得到过多少荣耀与财富，无论你有怎样说不清的恩恩怨怨，到头来，不过仍是一个空皮囊，你带不走一点点，我们每个人，最终都会有这样一个"今天"!

　　而最有资格说"你也有今天"的，不是我们任何一个人，惟有时间。
