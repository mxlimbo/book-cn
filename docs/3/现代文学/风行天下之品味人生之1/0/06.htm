西北纪行

　　因为工作的关系，我走过了全国大部分地方，就算是吹点牛吧，"曾经沧海难为水"，如今我一不向往繁华的大城市，二不愿意去名山大川，我更心驰神往的，是那些边陲风光，这些难以走到的地方，令我挂记心头。

　　今夏，终于有了一次远行的机会，地点是青海的塔尔寺和青海湖。我对青海的印象，来源于数十年前我的一位邻居。那时我还住在开封的一个大杂院里，院里有一家姓马的人家(因为时间长，我连他的名字也记不住了，只好在这里请他原谅)，他凶悍的妻子常常出他的丑，说些对他不敬的话，她对还是幼童的我们谈起当年马是怎样追她的："那时候还是低标准，吃不饱饭，他从青海回来，撇着腔说：'我们那里好得很，我们那里遍地是野兔，我们吃面都是成袋成袋地搬。'我听信他的话，去格尔木那么远的地方找他，一看，狗屁，苦得连开水都喝不上，他居然还一口一个'我们'，呸!"我们哄笑起来，而马，也开心地笑起来。也正是从这个时候，我记住了青海省有格尔木这个名字，一查地图，真够远的，真是没敢想过要去那里。

　　一路上的行程不必细说，火车上的旅行是每个外出的人都经过了的。我们一行共九人，热热闹闹地走了一天。

　　到达西宁的次日，我们就去位于湟中县的塔尔寺。塔尔寺是在全国有大名的藏传佛教圣地，我有印象的，是这里的酥油花、壁画和堆绣，但这些还都比不上"晒大佛"活动更具宗教神秘色彩，可惜我们来得不是"晒大佛"的时候，也就无缘开此眼界。在寺里绕来绕去，因为对藏传佛教一无所知，也就看不出什么名堂，最后转到后山上，迷失了路，去问路边的一位小喇嘛，也才得以结识这位名叫"罗藏热杰"的19岁喇嘛，经过交谈，知他认识汉字，我于是将我社出版的杂志送他一本，小喇嘛很随和，应我们要求，跟我们一一合影。我背了几句《心经》，有心想请教，结果他听不懂，看来没法交流了。

　　隔一日，全体人马出动，去海北州，那里有总编的熟人在等着我们。海北州的含义，是地处青海湖的北面，以此类推，还有海西州，海南州等等。汽车行驶数小时，到达海北州时正是中午时间，总编的熟人在这里当州委副书记，她安排了我们的一切活动。

　　汽车在五分钟内就把我们拉到一望无际的大草原上，在这片草原的平坦处，搭建了几座帐篷，隐隐传来的歌声，也带有浓郁的边塞风味。主人将我们安排到帐篷里，里面已经摆上酒菜，主人先是敬哈达，再是敬酒。我们来时就已经得到消息，说如果你真的不能喝酒，主人也不会见怪，所以女士们都是沾沾酒杯，男人们就无路可退，只好咬牙一饮而尽。席间，我们问主人，你们的业余活动都干什么，他说："和你们没什么区别。""那寂寞了呢?这草原上又没有什么娱乐?"主人的回答令我们大出意外："那就喝酒呗。"果然，主人的酒量，显然在我们众人之上。

　　饭后，我们一行来到一片茂盛的草原上，主人特意把汽车停在远处，让我们步行走进一望无际的草海之中。这是一片花和草的海洋!所有的客人都是第一次见到真正的草原，欣喜之情溢于言表，孩子们在花海中追逐，打闹，拣拾蘑茹。没过小腿的青草，让人不忍踩上，犹如冬天不忍踩上洁白的积雪一样。云彩白得像哈达，又低得像在山头，风吹过来，再看山坡上自由的牛羊，就不能不想起" 川，阴山下，天似穹庐，笼盖四野"的古诗。众人正抒情时，不远处的云彩下忽然出现一道道隐约可见的雨幕，主人介绍说，这是草原上特有的风景，那片云彩正在下雨。真是奇妙，下雨居然可以看见的!很快，我们这边的草原也下起了小雨，紧接着就是黄豆大的冰雹，我们担心远处的孩子们被冰雹打伤，急忙喊他们回来，一会儿他们回来了，说，刚才那边根本就没有冰雹!草原上的气候变化之大，由此可见一斑。听说我儿段蒙想写一篇描述大草原的作文，州委副书记就领着他，走进远远的草海之中，给他讲解各种花草的名字，段蒙说："你们生活在这里可真幸福!"女书记问："怎么幸福呢?"段蒙说："你们这里没有污染，地方大，人又少，有很多城市里没有的东西。""那你愿不愿意留在这里呢?"段蒙似乎很为难，吭哧了一会儿，还是没有明确回答。

　　其实也真怪不得孩子，即便是我来回答，又能答出什么妙语吗?晚上，回到宾馆，我和大伙聊起来，关于这个话题，居然还真有不少内容。一方面，我们真心羡慕这里的山山水水，一草一木，喜欢这里的空气和阳光，留恋古朴的民风，可另一方面，如果真要我们长期往下来，我们又会不大情愿，什么原因呢?我们居住在城市里，一边享受着现代文明，又一边诅咒着它。现代文明缩短了世界的距离，却又加深了人与人之间的距离，更可怕的，是我们越来越离不开现代文明这个怪物。比如，我之所以不能回答是否愿意留在草原，还不正是我舍不得城市给我的既得利益吗?我在潜意 ，实际上是还在留恋电话、电脑、音响、空调、汽车，虽然我渴望新鲜的空气，向往单纯的人际关系，想在宽阔的草原上驰骋，尽情享受大自然的恩赐，但是我终究不能活出真正的自我。同时，我能适应草原上的清苦吗?我能耐得住这份寂寞吗?也许短暂的旅行会使我激情澎湃，兴致大发，但我不敢想像长期住下后，我还会对这样的生活纵情讴歌吗?我一方面向往它，一方面又害怕它，说自己是叶公好龙，不为过吧。也许还可以倒过来想想，长期生活在这里的人们，他们向往大城市吗?好像是的，他们同样摆脱不了现代文明的诱惑，但是我专门问一位生于此长于此的青年，愿不愿意到大城市去，他的回答出人意料，是不愿去。原因是不能适应大城市的节奏，很多东西不懂，而在草原上有自己熟悉的一切，有亲人，有朋友，有酒肉，有歌声，在这里生活得很愉快，为什么要到大城市去出丑呢?

　　西北之行，原想开阔视野，增加对大自然的了解，却竟然得出人生如此之累的感想，懿欤!
