说忠诚

　　写下这个题目，想起一件旧事。是我在中学时，曾于课堂下询问教政治的老师："现在我们批判古人忠于皇帝，以后我们会不会因为在'文革'中进行个人崇拜而又受后人批判?"这话令老师吱唔了一阵，终于没有明确答复。没有答复，也许有多种原因，但今天推测，可能还是"怕"字占了上风。

　　忠，列为贤人标准之一，《二十四孝》图解里，把不忠之人，画成死后入十八层地狱，受尽苦难。《论语》，《道德经》，以及多种古典册籍，随便翻翻，都不难找出"忠"的注脚。

　　而忠诚之人，更是比比皆是。电视剧《宰相刘罗锅》，虽为戏言，却也表现刘墉对皇帝的一腔忠诚；岳飞抗金退敌，可谓一代名臣，但得到皇上一道又一道金牌，就进京受死，还不是因为忠心?宋江自落草的第一天起，就没有打算抗上，改"聚义厅"为"忠义堂"，打出"替天行道"之名，"天"者，皇帝也，后来受招安，也就顺理成章。再说个近代的例子，明末人王夫之，与清朝势不两立，几次组织武装抗清，失败后，躲于石船山下，屏迹幽居，写"清风有意难留我，明月无心自照人"为鉴。忠诚的例子不胜枚举，以下主要谈的是对不对的问题。

　　对不对，难说。过去我们的评价是不对。因为他们忠于朝廷，而朝廷是封建的象征，因而凡是忠诚朝廷的，也就是封建的，自然是列入被打倒之列。常常在读书时碰到这样的语句："尽管他有较大的成就，但由于历史的局限，还有较强的封建意识和忠君思想，因此这是要特别指出和批判的。"现在再看，忍俊不禁要笑，人家生在封建社会里，怎么可能有共产主义思想?或者以其人之矛，还制其人之盾，反问一下发问之人："假如你提前降生于封建社会，又该如何?"

　　生活里无处不有忠诚，因而这"对不对"的结论，就愈发显得难下。以家庭论，夫妻之间，忠诚当是第一位；上班，你的上司总会优先选用忠诚他或事业的人；与同事相处，言而无信，常会受人耻笑；而作为一国之主，没有全国大多数人的忠诚，怎么能坐稳第一把交椅呢?

　　像李自成，得了天下，却重用牛金星之类不忠之人，很快演变成了一场悲剧。"文革"中，全国人都在批封建思想，有位从旧时代过来的老人因为说了句"过去皇上怎样怎样……"，马上被揪斗游街，而回转身，全国大跳忠字舞，大唱语录歌，早请示、晚汇报，一句顶一万句。神化领袖，这种忠诚又该如何评价呢?有人说："'文革'中倒有一件好事，是全国人民的心齐，心诚，这是那以后再没有出现过的。"可是这齐，这诚，与封建时代的忠君思想有多大区别呢?

　　评价如此不易，只好刨老根。而老根也是枝枝曼曼，纠葛不清，手边有辜鸿铭的《中国人的精神》，谈到忠君根源，有一段可抄：

　　"……孔子的哲学体系和道德学说，即著名的儒教，能够取代宗教够使人们甚至大众不再需要宗教。那么在儒教之中，必定有一种东西同宗教一样，能够给人们以安全感和永恒感。这就是孔子在其国教中给予中华民族的忠诚之道――对于皇帝的绝对忠诚。

　　"由于这种忠诚之道的影响，在中华帝国的每个男人、妇女和儿童的心目中，皇 帝被赋予了绝对的、超自然和全能的力量。而正是这种对绝对的、超自然的、全能的皇权信仰，给予了中国人民一种安全感，就像其他国家的大众信奉上帝而得到的安会感一样。对绝对的、超自然的、全能的皇权的信仰，也使得中国人民形成了国家是绝对牢固和永恒的思想。"

　　辜鸿铭先生是有名望的人物，学贯中西，有国际声望，但其人也怪得可以：效忠皇帝(标志是自己留辫子)，纳妾(认定一个茶壶可配四个茶杯，一个茶杯不能配四个茶壶)。说他怪，是客气，敢放在数十年前，不知给他戴个什么高帽呢!但是我不能说他完全没有道理。古代中国，儒教即为国教，男子要治国、齐家、修身，遵循的标准离不开国教，而国教，首先是忠君。女子，有三从四德约束，其主旨"未嫁从父，已嫁从夫，夫死从子"，根基也在于忠，入《列女传》的绝没有不贞不忠之人。这忠诚，如果依辜先生所言，可上升为信仰和宗教，则我一孔之见，总应该鼓掌的。因为有信仰，总比没有强，至于说到这信仰是否合用，那是后代人的事，评价那个时代的人，不能用这个时代的眼光。眼光是什么呢?最好是鉴："秦人不暇自哀，而使后人哀之；后人哀之而不鉴之，亦使后人而复哀后人也"(杜牧《阿房宫赋》)。忠诚源自信仰，而皇帝，则是这信仰的具体化身。那么问题也许就明朗了，可再具体问道："忠诚乃至崇拜皇帝，何如?"意见也有两种。一说是生在那个朝代，不忠皇帝，似再无可忠之人，一说是皇帝靠不住，如张中行先生曾愤而有言："几乎所有的中原逐鹿都是狗咬狗，不管谁胜利，坐上宝座，都是骑在小民头上。"(《再谈帝王历史剧》)，一语中的。那么回头再看辜鸿铭，忠君，有违大义，纳妾，有悖男女平等,我们只好说，这老头子!

　　拉扯这么多，关于忠诚的结论，是否就是有了呢?恕我迟钝，还是难下断语，如果非要表态而不许骑墙，则我要说：忠诚如果建立在信仰上(如对事业)，可佩，如果是建立在责任上(如夫妻)，是可敬，但如果把皇帝列为忠诚对象，那是可悲外加可怜。这总算没有交白卷吧。
