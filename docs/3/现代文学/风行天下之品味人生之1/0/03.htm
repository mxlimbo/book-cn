搬 家

　　我和很多人一样，因为这样或那样的原因而搬家。搬家，难以用好不好这个框框来论定，远的如孟母三迁，近的如城建拆迁，有些是自愿，有些非所愿。许多人写过告别故居的文章，类似骊歌，读来令人唏嘘，如冯亦代写过《辞听风楼》，萧乾干脆写了一本《搬家史》，还有其他的名篇佳什，就更不计其数了。

　　至于我的搬家，简而言之，是经历了三二一和一二三这样的过程。

　　三二一和一二三，都是指人数。

　　那还是毕业分配之时，我扛着一个布纹衣箱，这就是我的家，在省新华二厂报到已毕，行政科的人将我领到马路对面的集体宿舍，上三楼，敲开其中向南的一间，对里面的两人打招呼："他分到你们这屋了。"两人的反响不同，和我年纪相仿的一位表示高兴，后来得知他是比我早分配来三天的李君，年长的一位就表示不满，追出去责问行政科的人："已经这么挤了，还要再分进来一个，叫我们怎么住?"这位是马兄，但是抗议无效，我还是在行政科的帮助下，将行李搬了进去。不过很快我就确信马兄的怨言是有道理的，是因为不很久，我在上班时间回宿舍取东西，用钥匙却打不开门，再听见里面有响动，结果是马兄开了门，他的女朋友正飞红着脸、衣衫不整地坐在床沿!

　　尽管住室拥挤，但三人还是友好相处了一个时期。不久，马兄成家迁出，我和李君合购一对枕巾，作为贺礼，前往他的新房祝贺。马兄拿出半瓶茅台酒招待，虽说我和李君平时都不喝酒，但马兄的情谊难违，只好举杯，而不管是什么酒。在我心目中，茅台酒与二锅头没有什么区别，反正都是辣喉咙，结果两杯酒下肚，我便醉倒。

　　这时宿舍只有我和李君了，从住的条件看，确实更方便一些，但马兄的迁出，也使我们觉得冷清不少，后来马兄过烦了婚后生活，还不时跑回来跟我们吹牛，甚至还为我介绍过一次对象。

　　大约半年后，我调入一家杂志社，行李再次卷成卷，搬。这次分得一间单人宿舍，约八九平方米，我满意极了，关起门来，尽情欣赏自己的这个"家"，宿舍由三人，而二人，而一人，规格像是越来越高了。有时想到刘禹锡，"斯是陋室，惟吾得磬"，也就飘飘然了。此时我年方21岁，避不开情缘，很快投入恋爱。一个时期后，领结婚证，这间小屋也就成为两人世界。新婚是欢乐多于愁怨，常常是，我们夫妻说些开心话，因为墙壁薄，欢声笑语总会被隔壁的同事听到，他有次羡慕地问我："什么事逗得你们整天笑哈哈?"

　　家庭资产很快多起来，我还年轻，不知道"戒之在得"。总的来说，共分两大项，一是生活用品增加，比如锅碗瓢盆，一应俱全，光被子就塞满一柜子，这是妻的娘家的陪嫁品，我至今都不明白为什么要送这么多被子，让我们一辈都盖不完；二是书本大量增加，那时我买书肆无忌惮，也没有章法，很快书架满了，箱子满了，床下满了，最后书们终于挤上床。后来领导开恩，我分得两间简易平房，终于像个家的样子，搬完家，儿子就出生在这处新居里，从此我的住宿又回归到三人世界。

　　这处新居给我留下很好的印象，因为是单独一处，并有半间小院，一间向阳。有一段时间，我很有点安居乐业的念头，在小院栽了株葡萄(未成活)，几棵豆角几架丝瓜，喝老酒，吟陶诗，设想"把酒活桑麻"。可是房子本身有问题，一是没有卫生间，更不用说暖气，天然气之类，二是房子漏雨，有一晚下暴雨，院里积水盈尺，我不得不用脸盆向院外泼水。更可怕是屋里常有老鼠跑来跑去，做娶亲状。若干年后，迎来住房条件改善时代，我再迁，这次连升五级，由地面上升到五楼，单元房，水、电、卫生间俱全，但无暖，当我走进卫生间时，才真正有了点上档次的感觉，看人家都有自家的卫生间，现在我也有了，尽管我的卫生间只有半个多平方，仅容一个便池而已，但这样我就知足了。

　　可是人的欲望，随着条件的改变也会改变。满意了一个时期，妻子就抱怨冬天没有暖气，夏天天热，五楼是顶层，楼板都晒透了，打开空调，只能冷下面一截，向上一伸手，就会感觉上下温度不一样，再加上有时停水，令人狼狈不堪。

　　苦，不会让我一家吃完，去年冬天，幽谷迁于乔木，我全家再搬住进一所三居室，这回基本达到设想了，有水、电、气、暖，三楼，并向阳，装修一毕，竣工乔迁之后，我在日记中大书特书，狠狠地记了一笔，认为这是我生活中的一件大事。

　　但妻子有时还会无意咕哝一句"面积不够大"，这是事实，但与我当年家在行李箱上的境遇相比，不是大有进步吗?也许妻子会说那时是一人，如今是三人，不可比。我也不辩论，只说面积虽还不够大，但多想想它的优点吧。优点之大者，是与一些居室面积更大者相比，采光面积很大，突出的例子是，别人家大白天进屋要开灯，我们家则可以在床上晒被子!中午阳光直射进来，我睡午觉时不得不用报纸盖住脸，以致于养成了睡觉盖脸的习惯!

　　拉拉扯扯写些这些琐事，会有人爱看吗?或许会有人责问："写这些有什么意义?"我无志，或无大志，把建设自己家园，侍奉父母，供养妻儿当作生活非常重要的一环，我过的虽然是平平淡淡的日子，但从住房中，也可看出我的生活水平在一步步提高。我写这篇小文时，内心充满了快乐，我确实衷心感谢我们平安祥和的国家，我也祈愿每人都住上宽宽敞敞的房子，过上和和美美的日子。
