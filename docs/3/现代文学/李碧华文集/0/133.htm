都挺小密去


　　台湾总统大选落幕後，人们耳畔不再响着这个字：「挺」。

　　不知甚麽时候流行的，「挺扁」、「挺宋」、「挺连」之前，也有艺人呼吁：「希望各位挺我的表演」、「你不挺我不行」、「挺你的倒不少」……。

　　本来这个「挺」是「很」、「非常」之类的助语词：「我挺好」、「他对你挺不错的」、「知道了挺生气」。或是形容词，例如「坚挺」、「挺立」。

　　它变成了动词，听起来相当的有想像馀地：——

　　「我挺你！」

　　「你是不是真的挺我？」

　　「不挺你挺谁？」

　　——除了支持、拥护、鼓舞、捧场之外，也有点不文。似粗口。

　　中国大陆的顺口溜，也出现了一个新名称。

　　据说高干是：

　　「左手拿文凭，右手拿酒瓶。
　　眼里看小秘，嘴里讲小平。
　　对上讲公平，对下讲摆平。
　　在外对花瓶，回家防醋瓶。」

　　这样才叫有水平。

　　「小秘」，不是小秘书，应是「小密」（秘、密同音）吧？——小小的秘密情人，亲密女友。也许真正意义是「小蜜」。
