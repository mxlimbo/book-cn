像喝墙的罗宋汤


　　俄罗斯，总令我联想起神秘和阴冷。――即使红色，红到尽，原来是恐怖无情，似笑非笑的cool。

　　公园、广场、雕像、宫殿、围墙、马路、车厢、军表、城堡……，还有冰封的土地。――全部都可以一个「大」字来形容。

　　所以俄国菜是唤作「俄国大菜」的。

　　传统的俄国烹调方式，不外烘藓陀驼ā６砺匏辜块酿蘑菇牛油、俄国牛柳丝、炸蚝、凤尾虾……，就像高头大马的侍应。大块头、大块肉、大大的一个烤面包。

　　当然少不了名闻天下的鱼子酱Caviar和伏特加Vodka。鱼子酱分量少，但种类多，有红、灰、黑……，深浅度不同的颜色和品种。配薄煎饼吃。伏特加是放在冰格中也不会结冰的，顽强的酒，可以变身成Bloody Mary或Screw Driver。

　　但我其实喜欢Russian Bortsch。地道罗宋汤，十分浓烈而鲜艳，猪骨或鸡骨作汤底，里头有红菜头、西芹、青椒、番茄、椰菜、蒜茸、薯仔、牛肉……。表面浇上酸忌廉，更是红白分明。罗宋汤像是「食物」多过「餐前汤水」，每舀一勺，都精神饱满。配上粗犷冷硬的五谷面包，感觉好「红场」，或是「红墙」。

　　本地茶餐厅快餐厅，一年到头都有（白汤和）红汤，再差，也有质感。
