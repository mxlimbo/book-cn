赤川次郎和老懵董


　　托L代索取赤川次郎的签名（还要上下款）。我没到书展。今回最想得到的也不过是赤川的签名吧。

　　我是他的读者，――当然，赤川次郎不算「最」喜欢的作家，也说不上谁是「最」。日本小说看得多，各有特色。只是我已不能得到泉镜花、谷崎润一郎……等等的签名了。

　　他曾是日本连续十年稳坐畅销书榜第一名的作家。写作二十四年，作品四百册，版税收入居首。初度来港，在书展中是最受读者欢迎，人潮汹涌期一睹风采的一位。――以上，是资料而已。像我们这些「恶客」，把书掀掀，大概会知道是否可读。一切的包装、花巧、吹捧，对我们没有作用。实实在在可读的书，才肯从头到尾看完，――因为，花不起时间呢。

　　但问是「哪一本」？又说不出来。

　　去年某一段日子，把他的幽默推理小说系列干掉（总是一看一大堆的。近日则看苏童的全部。大量有关「猴子」的专书，用来应付「渔护署」。《五体不满足》。和某人的传记……）。我很佩服赤川故事开篇那「抓猫揪脖子」的故弄玄虚手法。

　　此外，卖得最好，甚至荣登港闻版的《老懵董》，也是想看的书。最好还有「男主角」的签名啦，――怎可能？

　　但它能够出版，我们还是得享言论自由和政治幽默的（次文化堂还不快快送一本来？）。
