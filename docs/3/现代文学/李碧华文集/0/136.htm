谁说读者要看


　　今天（四月十日）阅报，关於梅爱芳因癌细胞扩散病情告急入院的消息。梅妈妈及阿梅等经常帮人，但这回帮不到自己家人，心情当然沉重。

　　希望她福大命大早日康复。还有，别受到不必要的骚扰。

　　阿梅向传媒提出无奈的警告：——

　　「唔好搞我家姐！」

　　她说：「如果当阿Ann（爱芳）是朋友，当她是个人的话，请不要骚扰她。谁偷影到她，我就不同谁做朋友！」

　　艺人身经百战，甚麽都有人说，正面负面报道，绯闻丑闻，一一承受，谁叫你在行走江湖呢？但他／她们也有底线。

　　（一）不希望影响到家人。（二）伤病、意外、不幸，最好高抬贵手。

　　人人都有私隐，知名人士的家人，与下一代小孩，却很无辜。风气所及，亦受到牵连，饱受困扰。

　　将心比心，如果记者编辑自己家中有事发生，也会奔波、担忧、焦虑、情绪低落。当事人和身边人只求安静，坚强面对，怎有馀力馀闲去应付全天候偷影的狗仔队？他们没有切肤之痛，无所不用其极，也不过是多拍一些照片多做一些新闻吧，对「受害者」而言，却是长期的阴影。尤其是小朋友……。

　　——不要「屈」读者，说他们喜欢看，要求看。
