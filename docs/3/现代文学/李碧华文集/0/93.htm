猪和羊的天梯


　　猪的喉管唤「管廷」，它的上颚，叫做「天梯」。「管廷」淡黄色，切成蜈蚣状，涮了吃很爽脆。「天梯」更隐蔽，更刁钻，体积又小，每头猪，只得一小块，血盆大口的八婆猪，上颚也不过如是。——故一碟猪天梯，像是很多张嘴巴喊冤。而非平步青云。

　　天梯月白色，有一道一道的坑纹。可以灼熟，蘸蒜泥或沙茶酱、姜黳吃。也可油盐爆香上桌。亦取其爽脆的口感而已。这道菜吸引之处，是有话题。

　　除了猪，全羊大筵中也有「千层梯丝」，应是同样材料，但切了丝，细致是细致了，没有块状那麽一见如故（因为人人都有）。

　　不过全羊大筵是中国古代骚人墨客「卖弄」之作，菜单中，共菜肴112种，点心16种。先以四乾盘、四鲜果、四蜜饯「压桌」，每道菜以冷盘八品、大件四品、热菜十二品、饭菜四品、点心四种为「格局」，分四回上完。不但讲排场，菜式还得猜一猜。再摘录几项，你知道是甚麽吗？——（一）「明开夜合」（二）「采闻灵芝」（三）「金鼎炉盖」（四）「天开寿仓」（五）「饮涧台子」（六）「喜望峰坡」。
