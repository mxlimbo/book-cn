台湾版罗生门


　　老实说，我不大喜欢葵青剧院座位的「架构」，没有中间通道，进出得骚扰半排观众。一句「对不起」说足一分钟才抵达目的地。一旦发生火警就惨了。――以上是题外话。

　　最近在葵青剧院看了台湾「复兴国剧团」的新编戏曲《罗生门》，改编自日本芥川龙之介作品《竹　中》，原着比较错综复杂，牵涉角色多，自说自话。本剧则只集中在强盗、男人、女人身上。说是「新编」，仍传统保守，不够大胆。「群众戏」尤其老土。――不过双生一旦的演出精采，音乐和服装造型设计也有特色。值得一看。不知是宣传不足，抑或地点陌生，上座率不算高，很可惜。

　　角色中以小生曹复永最难演，一时多情，一时怯懦，一时阴险，颇考工夫。花旦黄宇琳唱造俱佳，看简介知是新生代，完全可以担戏。吴兴国有台型，是簪了火百合的艳盗。这种比较Man的角色，在影视和戏曲表演界似不多见，本地的武生行当不大出色。亦正亦邪就更少了。

　　日本的《迷离花劫》也据此改编。演强盗的是丰川悦司（男女主角是金城武和「宝冢」出身的天海佑希），亦有点牵强。

　　看後，我即兴地向梅雪诗推荐此剧的花旦戏。有一回欣赏过阿嗲演「古腔排场戏」，她很用心，特别有印象。可惜她翌晚赴宴，赶不上。
