只因为在乎


　　某男与女人反目：「我一生以来乐於单身。」――但「女士们都尝试利用魅力及怒火说服我。」

　　情已逝，当初一切甜言蜜语全是笑话。所有男女都明白，有些事，有些情，是「不想记得」的。

　　不过，一个在乎的女人，尝试利用魅力及怒火说服男人，这是她的「努力」。努力不一定有用，也不一定奏效，甚至有反效果，但不紧张他，何必花这力气？

　　世间女人会尽量展示自己美好的一面，性感的体态言行，去吸引意中人，打动他，希望对方快乐。因为有些时候，你快乐，所以我快乐。――并非女人窝囊，而是把他放在首位。对一个半点「意思」也没有的人，魅力就自然而然收敛起来，以免浪费电力了。

　　所以，动用魅力是一种「积极」的伎俩。

　　至於怒火，虽有点「消极」，但何尝不是反应？

　　对方是不重要的人，才相敬如宾，犯不着为他伤心、伤身。愤怒还伤元气，赔上精神心思，女人生一顿气，马上老了几个月，需要时间复元。妒忌得流泪，抑郁，没有胃口，缺乏补给，消耗的，何止男人想像中那麽少？免得过，谁肯生气呢？为甚麽？为甚麽？――只是被一时的爱情戏弄，愚不可及。但你却控制不到自己。
