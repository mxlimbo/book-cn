「大材小用」刀枪药


　　严寒的天气，特别容易伤痛。例如带电，暖手一碰到冷金属，总是一下刺痛；脱衣服又张咀飨欤如果在暗室，大概有小火花。

　　还有经常被一张纸着损，随便一掀一扯，那麽轻、薄、看来无杀伤力的东西，离去之後，手的任何部分会出现一条红线，未几便痛起来，不见血，但原来很深！

　　有时皮肤爆坼，不知如何竟出现一道口子。奇怪，又没做甚麽粗重工夫，又已涂cream来滋润，仍难幸免。

　　再涂甚麽润肤霜都无效了，因为愈是湿润那道小裂缝愈是不肯埋口。

　　D还说：「比起受伤的心，爆坼算甚麽？――如有奇效跌打刀伤药，都留给内用！」

　　灵机一触，我以前做research，为找寻一颗玄妙的「保险子」（俗称「还魂丹」），特地买了一瓶「云南白药」。据说江湖中人刀里来枪里去，受伤不便求医时，他们至爱的防身之宝，便是此药。一度在港卖至断市。

　　但看说明书，原来得用温水或酒送服的。小瓶药粉内藏一粒深沉朱砂色小丸，则是重伤才吞下。像个最後的寄托。

　　我才不会吃进肚子中。但把白药给敷在伤口上，即使用错，果然也好多了。――比起它一向面对的刀枪伤痛，实在是「大材小用」。
