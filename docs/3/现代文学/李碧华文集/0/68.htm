孤傲的岛民


　　有朋友住愉景湾、南丫岛……，常要赶船，称之「岛民」。

　　不过，他们会说：

　　「你们难道不是『岛民』吗？」

　　——对。搬了好几次家，搬来搬去，也只锺情港岛区，从来没住过九龙，根本不打算「过海」。

　　而且很多猪朋狗友也在这一带（是文化集中营呢），父母亲人的家、出版社、好友的办公室、喜欢的食肆公司戏院子书店酒吧……甚至医生和律师，一想，习惯已成为束缚，而非自由。只绕场跑圈。

　　爱住繁盛但幽静，罪恶而安全的地区。自己才是狭隘、孤傲、懒散不愿动的小岛之民。

　　又因为不开车（减少车祸，年中拯救了不少无辜生命），总觉甚麽屯门、大埔、荃湾、将军澳、九肚山……好遥远。

　　下班时间一个女孩坐电梯下楼，问住哪儿，说是上水。天天要转几次车。——其实所花时间不算多，地面堵车时更荒废青春。

　　住港岛，只是心理上很方便。肯定有偏见；觉得这边的人斯文有礼些，九龙的人粗鲁急躁些。看的士司机就知道吧？还有，曾被旺角区的核突佬推撞过，被新界的黑狗吓唬过，坐车也被一些纹眉的肥师奶打尖……，咦，是「暴力」阴影呢。
