排除万难，争取胜利


　　市面上好些「受尽身心创伤」的女人，上门强迫交代、打骂、借钱、仰药、要求「安置」、要求「保障」、自虐、苦缠、博见报、张扬事件……，做出各样动作，她们要的是甚麽？——只差没有明说吧。

　　不止露骨，甚至「露骨髓」了。

　　记者问了又问，无非希望她们不耐烦地，把「那个字」说出口，是的，这是我要的！我要……。

　　上岸的较高贵，太有资格骂浮沉中的同行不要脸。正受恩宠的，也放心嘲笑其他人。有的，看不起没有的。

　　妙龄少女，二十多三十岁，最好的日子，跟随一位阿伯？世上当然有忘年恋，没有感情，很难吧？不过，中间不是有些其他因素吗？花甲老翁、古稀老翁，串烧三兄弟，公园中大把，天天吹水、下棋、看咸报、付油麻地最便宜的肉金，哪有情妇去争宠？可见某些阿伯，有争的价值。

　　读者们看花边，每隔几天又一宗，只觉目标愈来愈明确，言语愈来愈坦率，这些女人，也愈来愈坚毅了。——她们根本不需要旁观者完全不切实际的同情，她们往往坐拥一千几百万仍干劲十足，所有为口奔驰的事业女性多麽寒伧！

　　大都会中总有「同行相轻」吧，不能不说，她们真的不是祖国佳丽的对手。
