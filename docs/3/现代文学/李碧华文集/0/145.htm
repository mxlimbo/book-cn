执子之手


　　公路上严重堵车，因为前面发生一宗疑互相追逐的车祸，有人受伤。逞一时之快，足足耽误了很多很多人的时间和要事。——如果坐的士，便见收费表「乾跳」，可以把人逼成躁狂症。

　　怕见车祸血肉横飞情状。更不忍见祸及无辜。

　　记得上星期有一宗公路上的车祸见报。一名後座女乘客惨被铁片扯断前臂。她向好友请求：「帮我执番只手，今次全靠你啦……」

　　虽然这是无数车祸中之一，但也令我想到：——

　　祸福无常，只能面对。希望伤者断手接驳後，恢复功能，早日康复。

　　从她「执手」的请托，我们才发现，生命中，这些都是身边重要的人。

　　在茫茫人海中找到一个亲密的、沟通的、可靠的伴侣，是「执子之手」。「得之，我幸；不得，我命，如此而已」（这是徐志摩说的）。那执你手的人，或许白头偕老，但不保证永结同心。——如果他永远有第一二次执手之情，你们也从不分手，那是除了自己的双手外，再得到另一份礼物。

　　假如没有这个人，在危险、困难、伤痛……的时候，在清醒与昏迷之间，仍希望有些「今次全靠你啦……」的朋友，无言伸出援手，赶紧执回断手。

　　「执子之手」，也有多重意义。
