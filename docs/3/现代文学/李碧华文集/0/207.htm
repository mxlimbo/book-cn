「开片」和「冰裂」


　　收到冯坚白、张佩儒、招汝灼、周先生诸位传真。谢谢。对哥K的青釉开片纹饰，有了更深认识。

　　南宋时处州人章生一、章生二兄弟，在浙江龙泉县各设一K烧造瓷器，一称「哥K」，一称「弟K」。（夏婕送我一本《渡》，是她寻访名K遗迹的摄影集呢。）

　　瓷釉瓷胎由於冷热胀缩率和冷却时收缩速度不同，因而出现裂纹，原是一种毛病，谁知开片自然、古艳，在美学上而言是惊喜，人们转而研究其规律，加以掌握，反而成为哥K的一个特色了。我有空便欣赏这件给自己买的礼物，它是仿制，比起珍藏的真品，当然小儿科之至，――但也很漂亮。

　　关於坊间流传的别名，综合有识之士的意见：――（一）「壁裂」指身体坯裂、釉面下陷，已成弃品，识者不取。（二）「逼裂」是烧制过程的压逼收缩，出现了大冰纹、小冰纹。因而又传称「冰裂」。（三）「碧裂」：绿釉之金属氧化物烧成後呈裂纹，浅碧绿色。

　　「开片」是正名。南方说法看来「冰裂」音义皆美，较贴合。在《国宝欣赏100种》也见此语。

　　又，特覆张先生：（一）「三大纪律八项注意」的革命歌曲我有盒带。前曾有小小笔误，谢指正。（二）宋代五大名K「官、哥、汝、定、钧」，叁考商务《故宫博物院藏文物珍品全集》中的《颜色釉》。
