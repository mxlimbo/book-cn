穷奢极侈亡国菜


　　清代饮食业是种病态的繁荣，尤以晚清为甚。中国沦为半封建半殖民地，政治前景半明半昧，贫富极度悬殊。上海、广州、苏州、天津、北京……，都流传过一些「唬人」的烹调秘方。——不外「穷奢极侈」，失去理性的招摇。

　　着名的文人冒辟疆（买走名妓董小宛那位）曾大宴天下名士。请到一位京师头等厨娘，问她需要甚麽材料？

　　「席有三等：上等需用羊五百只、中等三百、下等一百。其他物品随用随取。」

　　主人备中等数目，看她如何处置。那天，厨娘应约，随从百多人，她自己珠围翠绕，高座指挥，下人奔走厨房刀砧间听从调遣。——那三百羊，每头只割下唇肉一斤备用，其馀全部丢弃。人惊问故，她答：「羊之美全萃於此，其他皆腥臊不足用也。」

　　又有河道总督驻清江浦，国家每年拨给治河经费高达数百万鶣白银，实际用於工程不到十分之一。他家设宴，上了一味猪脯，与会皆赞叹精美无比。客人上厕，忽见无数死猪枕藉於地，才知席上一道菜，即此猪群里脊肉。生前闭锁空屋，壮丁执竹竿追打，由它们号叫奔绕至死，立刻刮取背上一片肉，乃全身精萃集中处。馀肉失味，不堪再食了。

　　——会吃，吃得好，又吃得起。是的。但清亡了。
