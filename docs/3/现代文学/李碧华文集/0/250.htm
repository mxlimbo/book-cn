十二星座反骨会


　　那天见星座版有「十二星座反骨会」，觉得很有趣。本来我们不大看这些小朋友玩意，也是得啖笑，因为次次的运程预测都模棱两可。星座的性格倒是有点准，至於「反骨度」，究竟是怎麽推算？

　　原来分为：——

　　（一）终极二五仔：天蝎、双子、山羊。
　　（二）低调二五仔：双鱼、巨蟹、人马。
　　（三）九流二五仔：白羊、处女、金牛。
　　（四）反二五人办：狮子、水瓶、天秤。

　　如果根据渡边淳一先生的《反常识讲座》论调第九章：「阿谀奉承也是美德」，则「二五仔」何尝不算另一种卓越才能？

　　高明地溜须拍马，不露痕迹，慢慢地将对方引向自己所控制的领域，具积极意义，又皆大欢喜。敏锐、准确、柔韧。

　　当二五仔，甚麽才是一流的出卖术呢？大概应该灵活走位、全神贯注、处变不惊、有利可图、机关算尽了。

　　必须不觉得「出卖」是一回事，并且都有「苦衷」的。卖得好，成王败寇。

　　下午茶时，刚好在座的有一位天蝎，此外是白羊、天秤，我是水瓶。太好了，有共同的取笑对象，便是「二五界中王中王」。——话得说回来，要做，便做到最好。二五而迟疑、不忍、过不到自己、露出马脚，不如索性不做了。
