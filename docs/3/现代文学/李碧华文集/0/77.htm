原来那麽浅


　　纪录片中，一头小水羚浸泡在水中。它悠闲而自在，完全不觉察四下的危机。

　　在岸边，有一头体积比它大数倍的母狮正在窥伺。母狮没有冒然采取行动，不因为无把握，只是不知道水深。所以静待良机去猎杀。

　　不久，小水羚满足地站起来了。几乎没伸个懒腰。

　　是的，它犯了致命的错误，让岸边的敌人洞悉：——哦，原来那麽浅，只及你膝。

　　母狮蓄锐出击，马上中的，啮咬它咽喉不放，并撕裂血肉，大快朵颐。

　　母狮进餐，是在水中一个小浮岛上进行。它并无意思与同伴分食。

　　岸上来了些狮子，远观它吃得痛快，也馋液大流。不过晚来了一点，又不敢轻举妄动；不知道水深呀，没游过去抢食。

　　母狮死守并独占食物，得意地尽情享用。一不小心，尸体掉进水里，它下水攫住，一站起来，群狮洞悉了：——哦，原来那麽浅，只及你膝。

　　二话不说，一齐下水涌前。饥饿的狮子群，把母狮的晚餐抢走了、分享了。真无奈。

　　人人都不想倒下去，只望站起来。无意中，一个张扬跋扈的姿态，便让所有旁观者知道你是个怎麽样的人？底牌在哪儿？水有多深？——哦，那麽浅，是自己给揭发的。
