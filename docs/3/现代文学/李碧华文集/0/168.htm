机王又如何


　　D最紧贴潮流，每有新款出炉，马上换手机。换得快。那所谓「过时」的买时也贵，又不值多少银@了。

　　其实手机一天到晚变身，只是名牌和潮流效应吧，功能差不多，再多你也用不上。热卖的「顶级机王」，商人造势，扮着「限量」以示矜贵，所以有价。匪徒还去打劫，捆绑女职员後在货仓大肆搜掠，其他都不要，只掠去受欢迎又易脱手的两款。

　　玩line的wet妹，「饮食 礼物」，战利品多不胜数，新款手机更是首选，可以转卖，换钱最实际。

　　――一个人要那麽多手机干麽？何来那麽多朋友联络？

　　科技发达，目的是方便通讯，节省时间和力气。再名贵、新款、有型、漂亮，实用价值一样，但人们花上大量冤枉钱去买一个靓机，为了享受拎出来一刻，别人艳羡的目光吧。我完全没有这份虚荣。问D：「灵魂不是重要过外壳吗？」――这样说，当然亦因为我穷。

　　近日最具名牌效应的剧集是《男亲女爱》，初出街气势如虹，但过不了一两星期，开始有生涩、硬滑稽、牵牛上树的跌watt感觉了，只靠几个好演员，火花不够支撑，看来「凑细路」又几吃力。――是开始时势太好而放缓？抑或渐渐得个壳？
