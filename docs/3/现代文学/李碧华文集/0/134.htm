等待Chicago


　　百老汇音乐剧Chicago虽在四月底登场，不过早已列入必看节目。

　　光是宣传单张：黑、白、红，已是非常精采的设计。它在十一个国家九十一个城市巡回演出，也得过东尼奖（六项都是大奖，包括最佳男女主角导演等等）和格林美奖。不过这些都是一些「吸引」，最重要的还是在座上亲自感受。

　　有朋友在纽约看过三遍，极力推荐，说是「集罪恶与美丽於一身」，令我好奇。

　　百老汇式的musical，一演很多很多年，这个版本是全新演译。演员也是鲜嫩而精壮的新生代。

　　看到一些片段和访问，女主角Chelsea两年前首演此剧时，还是甫踏出校门的二十二岁女孩，自信而倨傲。

　　此剧的故事元素是歌舞女郎、杀手、律师、狗仔队、金钱、不忠、命案、仇恨、名利和道德。――大都会是一个大黑箱，大牢笼……。

　　灯光和色调刻意地简约，追寻一个偌大的黑色世界，演员歌舞表演时，一根、两根、三根指头，以及细微的挥动，都表现得十分清楚，充满戏味。

　　最有趣的，竟然见到中国式的宫廷羽扇舞，「蝶影红梨记」地把男主角围在核心。――这个联想好荒谬吧？也是飞上枝头变凤凰。
