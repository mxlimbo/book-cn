快乐是梦，健康是醒


　　记得过年时，收到一些电讯箱中的留言。有一把低沉的声音，可以用「幽幽」来形容，她说：「祝你身体健康，心想事成。没甚麽特别的事，只是拜个年。」

　　原来是阿梅。

　　後来我回电：

　　「我们最想要的，只不过是『快乐』罢了。」

　　——「快乐」其实是最贪心的梦。因为一个人必然一切如意，心想事成，浓情蜜意，名利双收，时来运到，喜结人缘，灵感泉涌，心安理得……，才会「快乐」。两个字等於所有。简单又丰富。

　　一网打尽。

　　有了「快乐」，还要其他干麽？多罗苏。

　　有朋友想了想。他道：

　　「不，『健康』更重要些。」

　　是的，要改口：明白名利爱恨恩怨，我们也不会为了别人而活，人言不可畏，一旦不开心，还不快快跑掉？还是委屈？没有时间了，当然是自己的感觉至上。

　　——但，必须有「健康」，你的「快乐」才能享用，才可保存，才有心情力气令之循环、增生。即使不快乐，但健康叫你足以承受一切而不会倒地不起。继续追求快乐去。这是身体的事。

　　所以，「快乐」是梦，「健康」便是醒，相辅相成。
