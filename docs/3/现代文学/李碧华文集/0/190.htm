「同情」一次起两次止


　　一个人失意，有时是运气不好，但很多时是不自重，不自爱。

　　别人给你机会，扶你一把，也得靠自己勉力翻身。同情是有限量的，一次起两次止。谁也不欠谁，谁也不能倚仗同情心、同情分维生。

　　如果你失业，得到一份工又不好好把握，迟到早退还偷懒；如果你失恋，遇上一个对你不错的人，却又天天跟旧爱比较，不懂得珍惜；如果你考试不及格，有得补考，还预知范围，仍不温书；如果你曾跌至谷底，重上舞台时又甩嘴、甩底、失场、闹情绪、游魂、大觉着……

　　还有没有「下次」？你说呢？

　　――人总得自己争气，怎能靠「人工呼吸」来过日辰？这些人若消沉下去，只是活该。

　　时间又过去了。

　　克林顿明年得卸任，他舍不得离开白宫，一首歌触动愁思：「当你找到自己想做的事，时间却永远不够。」

　　所以只能用尽每一天每一刻，去「感受」他的位子。连勤奋的总统也不能长领风骚。

　　每人都有一个位子？不，其实是位子太少，争位坐的人太多。一旦有得坐，便坐好它，坐暖它，然後离座。如此而已。

　　不争气的人，音乐一停，别人占了位子。他颓然跌在地上。
