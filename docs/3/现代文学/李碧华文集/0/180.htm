暴烈「牛血肠」


　　韩国烧肉重油浓酱猛火，向来十分粗野。另有一种着名街头小吃，亦同样激烈。

　　「牛血肠」。

　　先倾大量牛血，混和小麦粉，在一个「澡盆」大的容器中。韩国人壮，一男一女四手用力搅拌，惟恐天下不乱。至血半凝。

　　之後又用另一个「澡盆」混调味料，有十至十五种香料、辣椒、糯米。各式粉末和颗粒、碎屑，黄绿红白黑……，混作一堆後呈暧昧的褐色。

　　之後两个「澡盆」的东西统统给倒进一个大「浴缸」中去，再以九牛二虎之力乾坤混沌。之後，借漏斗强塞进一条一条懦弱无能乏力的软体猪肠中。

　　猪肠一下一下的充血，亢奋起来，暴胀若干倍，不但坚实，而且坚挺。不久，整缸物料都跑到肠子中。

　　之後，这一条条，一束束之物，给扔进充斥了黳蒜和调味品的大锅中煮。

　　吃时切片，再蘸酱汁。

　　中国也有糯米肠，北方和南方的材料不一，但制法相类。这种设计，是「强暴式」的。填鸭也一样。对管道进行蹂躏。

　　台北小吃，有「猪血糕」，亦是猪血加糯米，不过省了肠子。但不怎麽好吃。

　　肠子有它独特的口感和味道。——也许它才是主角。
