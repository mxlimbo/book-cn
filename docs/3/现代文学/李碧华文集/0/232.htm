云南路「排骨年糕」


　　逛上海，离不了「吃」。

　　可以吃得很精致，有派头；但路边街坊小吃，一样迷人。

　　一般人必到城隍庙，一尝百年老店名点：鲜肉小笼包、蟹粉小笼包，还有豆腐脑、鸽蛋汤圆。「长安」饺子馆则有数百种形态和内涵不同的小饺子，玩具一样。南京东路的「沈大成」，貌不惊人，以点心和食驰名。

　　云南路的美食街，可以吃足三天。各地的风味小吃，都集中於此，又便宜又美味。像烤羊肉串、云南过桥米线、咸甜豆浆、桂林劲辣米粉、素菜包子、油条烧饼、黳油饼、各式拉、酒酿丸子、福州小点……。

　　每家尝一尝，也是好节目。

　　上海人喜欢吃年糕，前身一块块硬硬的浸泡在冷水中。有炒糕、汤糕、煎糕……（日式年糕则在炭火上烤）。

　　其中一家「鲜得来」，十分兴旺。它是「排骨年糕」专卖店，一年到头，便以薄皮铁（铝？）碟子上的排骨加年糕招徕。猪扒煎得很香，年糕软软糯糯的一大片，——这便是上海人的一顿饭。每客不到五块钱。一天可卖上几百碟。排骨年糕粗糙得来口感丰富，即使很饱，也忍不住吃上两口。同淮海路名气极大的「红房子」中，那鲜腴多汁的猪牛羊排，各具特色。
