走音圣诞歌


　　一家幼稚园在太阳下举行圣诞崇拜，家长和小孩都出席。那些「不求甚解」的小孩，用稚嫩的童音合唱。透过咪高峰：

　　「远远在马槽里，无枕也无，小小的主耶稣，睡觉很安康。众明星都望主睡的地方，小小的主耶稣，睡在乾草上。……」

　　「听啊天使高声唱，荣耀归於新生王。他赐世人新生命，他消痛苦治创伤。……」

　　「齐来，一齐来，大家上伯利恒。来朝见圣婴，天使王已降生。……」

　　全皆走音，但十分动听。

　　经常会遇上走音的歌（已成时尚），但从不感觉动听，——动听是小孩的纯真，还有「拚命」，令人忍俊不禁。

　　圣诞的来源及宗教意义，所有小孩都不太明白。这阶段，只接受灌输。

　　也许长大了一点，会很诧异马利亚许配了约瑟，还没过门，她竟受圣灵感动怀了孕。约瑟只凭一个奇怪的梦，主的使者说：「你妻子将要生一个儿子，是救世主，你要给他起名耶稣。」他也不验DNA，也不表示「如果个细路仔系我，我一定负责！」一觉醒来，毫不怀疑便接收了？约瑟真是老实而忠直啊。

　　本世纪最後一个圣诞一天一天的近。我们并未得到救赎，依旧浮沉，如太阳下十分具体的尘粒。因为陌生的小孩，才快乐起来。
