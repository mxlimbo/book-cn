珍惜已有，不要哭


　　一位女歌星在颁奖礼中得了大奖。在此之前，从妈咪口中知道，已分手的男友认识了另一女孩，并再次拍拖。自己很不开心，心酸酸，更没进食数日，因为到底有五六年的感情。幸得旁人开解，又自我安慰。但在台上一时感触，仍掉下泪来。

　　另一位女歌星，与男友多年感情虽有风雨，但至今仍是一对，认不认都无所谓，心里有数。他在身边支持。但这次她没有得大奖。末了只获安慰奖，才恢复一点自信。百感交集，她也流泪。

　　――有奖没人，哭了。有人没奖，也哭了。

　　当你有了一样，总希望也得到另一样。但世事极难两全其美呀。

　　假如一下子甚麽都有了，也会一阵空虚。――因为，除了不能达成心愿之外，就数已经达成心愿最伤感。接着要甚麽？

　　还是留一点馀地，留一个愿望，留一半遗憾，明天起来才有藉口。也支撑你，不要懒惰，不进则退。

　　要知道，世上大部分的歌星，没有人也没有奖。世上大部分普通人，事业和爱情都不如意，他们唯一拥有的，是「明天」。他们也没有哭。

　　「有」，其实很幸福了。珍惜手中已有的吧。不要哭。
