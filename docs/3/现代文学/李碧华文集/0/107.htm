动物性和昆虫性


　　有些人活着，是「动物性」；有些人活着，是「昆虫性」。

　　——动物凶猛，弱肉强食，才能自保。而昆虫，六足四翅，折了翅断了足，历尽沧桑，仍在人间顽强生存，它没攻击力，只有无奈的保护色，和逃躲、避、隐蔽……的本能。但世上活得最长久、生命力最强壮的，你说是「动物」抑或「昆虫」？

　　别的不说：譬如「小强」，原名蟑螂、学名蜚蠊、俗名鹺鹹，是地球上最早出现的昆虫之一，大约有四亿年历史。活到今天，还人气急升，简直可称「大强」或「巨强」了。当年一起行走江湖的恐龙，早已是化石。还支离破碎。

　　「动物性」是阳，「昆虫性」是阴。有牝鸡司晨，雌威当道，但很多很多女性，手无寸铁，与生活中一切强权、强敌、强势抗衡，过了一天又一天。

　　你的本质是动物还是昆虫？——可能无法回答，因为必须经过试炼、考验，面临巨变，重要抉择，才可以认识自己。免得过，我们根本不想知道。含糊就是平安。

　　人一向瞧不起动物和昆虫，骂人「衣冠禽兽！」已给面子，「禽兽不如！」则更不堪。

　　看过本报「抱拥大自然」专版的海豹、非洲象、浣熊、爱情鸟、海马、雄狮、环尾狐猴、……多有型可爱，简单纯朴。它们相骂时，也可用：「连人也不如！」
