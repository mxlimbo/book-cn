scan了我一次


　　阿Sam来信，写某回尴尬的「邂逅」，非常有趣。

　　他等巴士（屯门至湾仔线，「奉天承运」冷气巴）时，跟在一个短发女人身後，视线自然落在「粉颈」。她的汗，由头发一直流到颈，湿淋淋，好一阵也没拎出纸巾来抹。此情此景，如何捱到上车？

　　平日不大热血的阿Sam，竟渐渐热血起来，经过一番挣扎，掏出纸巾。

　　「小姐……」／「 啊？」／「要唔要呀？」／「――唔使啦。」

　　过程极短。却很「委屈」。云云。

　　那位女士，根本谈不上香汗淋漓，潜意识中好难有性联想。阿Sam想做上等人，因他以前在鼻涕呼之欲出之际，身边有个三唔识七的女人递了纸巾给他，所以他「感恩」，向另一三唔识七的女人「图报」。谁知……

　　最盏鬼的，是阿Sam小朋友现代化笔法：「　反应好快，一侧身，用七分面对住我，十一分警觉的眼神，『习』一声已经将我scan了一次，重有埋结果。我当堂好似联交所停电，唔知点　，都怪自己唔好，错着个掣。」

　　以後电脑scan图，便联想起前卫男女邂逅时四目交投极速过滤「受唔受」的决定。

　　代邮：湄、半边人、Ann、梁敏滔诸位，来信收到。又：谢孙有恒先生指正。
