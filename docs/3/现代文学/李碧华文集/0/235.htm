夸张的黄金粟米


　　只流行了一个夏季的台式饮料「沙冰」热潮已过。这沙冰曾大受欢迎，真莫名其妙，因为只是一些「颜色水」弄成的东西，假得像喝塑料。成本低，机器还轧断过店员的指掌。

　　近日沙冰店纷纷变身，热腾腾的牛油粟米杯新兴，又更莫名其妙了。玉米是「粗粮」，好些连锁快餐店常有出售，不算太受欢迎。用锡纸或胶袋封好才烤熟的两家，比较鲜嫩可口，抹上牛油洒点盐，是我很喜欢的小点。一般快餐店下午茶时份那泡在水中煮得「历尽沧桑」的老玉米，粗糙无味，不大恭维。

　　爱吃玉米，已被损友取笑：

　　「与猪争食。」

　　但便宜好吃，还高纤低脂，是健康食品。没想过忽然流行，需求量当然遽然激增，粗粮的价格也飙升四成，令人气结。粟米粒而已，不但销售一空，还有人囤积居奇。不知市面上那些罐头黄金粟粒，是否受影响？不知猪的口粮，是否也受影响？——这才是集体与猪争食。

　　奇怪，它一向也在，味道也没变过，为甚麽没多少人「发现」？首个领导潮流的又是谁呢？把平凡的东西「推展」成人气商品，有时是无心插柳的。一窝蜂之可怕在於此。

　　幸好抢购的不过是一杯黄金粟米。在北京，人人担忧人民币贬值而不安，他们去抢购金条。
