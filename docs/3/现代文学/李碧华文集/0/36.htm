七姐盆和纸电脑


　　同是农历七月的传统节日，但人们忘记了「乞巧节」，反而「盂兰节」更update。

　　七姐盆，那些「式微」的胭脂水粉圆镜针线剪刀……，送也没人肯要。三四十年代，一个女性重要的节日，今天谁会关注？谁还去「乞巧」？――时代女性自食其力，自给自足，怎肯「乞」？开玩笑！

　　一年一度鹊桥相会也是一种过时的浪漫。当中有太多妥协、委屈、老套，甚至交通不便，没半点想爱就爱，说做便做的激情。新一代都看不起牛郎织女。

　　但七月十四年年有话题。

　　大家都爱鬼故事，所以报刊电台有特辑。还有，打开港闻版，我们会发觉，很多死者都不过十多廿岁，――死亡、孤魂、烧衣、鬼门关、思念、哀悼、赎罪……的体验，已趋年轻化。连自杀也是。

　　所以烧衣祭品，总是潮流资讯。在高科技、电子化的年代，如果你要烧一部手提电脑给泉下的他，外形、处理器型号、功能、RAM数，也得追上潮流，以免他在阴间用的还是旧款，又慢又大件，笑坏其他新鬼。

　　祭奠是仪式，也是「送礼」。送甚麽给人，显示你的心思和关怀。七姐盆？　鬼收！同理，烧一些对方不合心水的纸扎品，也是「失礼」。――不要以过年糖果那麽马虎的大路货来应酬啊。
