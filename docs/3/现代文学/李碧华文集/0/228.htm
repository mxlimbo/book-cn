人人都是「乜水」


　　别人不知你是谁，有甚麽所谓？——基本上，人人都是「乜水」（「乜谁」）。

　　董太日前出席公开活动，在台上致词完毕，冲口而出：「我都系唔讲多，你都唔系睇我啦，系睇阿『乜水』！」说时示意身旁的「天王」、「天后」。

　　气氛尴尬？

　　不，二人後来一笑置之也罢。

　　有些人喜欢问：「你知唔知我系边个？」有些人摆足架子：「响、闪、扬」，生怕人家失觉了。——但也有些人不介意做「乜水」。世上并非人人都有义务去「知道」你的。

　　记得有一位名作家，每认识新朋友，握手时必自我介绍：「我是××。写小说的。」在华人社会，谁不知道他阁下××的身分？但仍谦抑自持，避过「咦？未听过」的疑惑。名人没有架子，正因他已臻毋须摆架子的豁达境界。

　　反而有些二打六，总爱道：「我已经在专栏中写过了，不想重复。」——但，我们都没看到呢。是哪份？几时？很好奇。

　　最有趣的，有作者在框框中怨人给请柬时弄错了真名。某写稿时用的笔名也没有多人认识，何况真名？那发柬的公关好可怜。

　　——人生舞台那麽大，角色那麽多，上台下台那麽频密，你才明白：人人都是「乜水」。面是人家的。
