火百合无色的血


　　在花店见到漂亮的花，常会买一束回去。——不管你是否知道它的名字，几天後便凋零，成为垃圾。其实任何鲜花，只要一剪截下来，已经半死不活了，不但受伤，还没救。我们闻到的香气，是那种生命一步一步走向死亡，苟延的残喘，花有无色的血，都混在水里，分不清楚。说不定那香，还带血腥。

　　刚买回来的花，是它生命中最奋发的一刻，拚尽力气亮丽招展，然後天天走下坡路，颜色渐憔悴，垂首不语，至萎谢大去。

　　再清的水都有细菌，叫浸泡的茎枝腐烂发臭。要好花多活一阵，可以加保鲜液、阿司匹灵，或在剪口处用火烧灼，使局部碳化不易烂。而灼伤後更可促进花苞开放。不过以上方式，都懒得采用，一切顺其自然。还有，我不待完全萎谢，通常到了九死一生便移走，不太伤感。

　　有人喜欢清香纯净的百合，不过我较爱火百合。买了一束火百合加婴儿眼泪。有盛放、有渐开，还有两个花苞。青着脸闭着嘴，一言不发。但一天两天，它胀胖了。夜里埋首写稿，天刚亮，一抬头，原来忽地绽放，青白中泛着绯红，酒醉一样，还有紫红色的火点。六个朱色花粉囊十分娇艳，一碰即要溃散，「挂」在蕊上，像一只「吊」在女人足尖上的高跟鞋，晃荡而危险。
