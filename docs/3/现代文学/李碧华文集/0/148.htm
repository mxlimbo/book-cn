他戴了多年的眼镜


　　现代商业社会，人不适合与机构发生感情，因为「机构」没有体温。——但原来很多时，人也忘记了谁是提拔你的人，还有，人家花了心思力气把一个略有潜质者捧红了，或是将自己的心得教会了，中间没有争拗，只是利害关系，便已另觅高枝。

　　「我要在你身上得到的，已经得到了。」有人觉得这是市侩，不过，也许这是聪明。

　　世上还哪有死忠派？

　　看张玉凤的回忆录。她二十岁出头时是首长专列的列车员，被毛主席看中了，调入中南海当「生活秘书」、「机要秘书」。关系耐人寻味之馀，她也得悉很多主席的秘密。一九七四年，毛患「老年性白内障」，眼珠晶体变得混浊，失去阅读和批示文件的能力，都要她代劳。此事只有极少数人知道，全国人民则蒙在鼓里，不知是一个小女子在「处理」政务。

　　忠心耿耿的周恩来非常着急。除了安排眼科专家会诊外，还把自己使用多年的一副眼镜送了给主席。总理的信写道：

　　「这副眼镜是我戴了多年，较为合适的一副。送给主席试戴，如果不合适，告诉我，给主席重配。」

　　根据常识，这很可笑，行不通的，度数也不合。根据人情，则为「愚忠」。——但，坐第二把交椅的，从不思变，永不背叛，实在太稀罕吧。有感动过他吗？
