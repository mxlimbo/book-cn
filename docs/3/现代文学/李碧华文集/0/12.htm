扬州炒饭可否注册


　　究竟「扬州炒饭」是否土产？抑或扬州当地根本没有这道菜？一直存疑。

　　据说江苏扬州市近期欲把「扬州炒饭」进行商标注册，则很可笑。

　　我们平日到「任何食肆」，都可点到。其实它只是大众化的杂锦炒饭，材料不外虾仁、叉烧粒、青豆、鸡蛋、黳花、火腿……，大同小异，只看是否镬气够、调味足。好不好吃。如果是上述物体，做得再cheap，它仍唤「扬州炒饭」。——用夜香花瑶柱蛋白来炒饭，便不是「扬州炒饭」。此乃共识。

　　一个名字有公众认同的形象，雷同的便是跟风抄袭侵权。

　　但「扬州炒饭」并非一个打出名堂的商标，它甚至是「以讹传讹」。扬州市要保住这块招牌？注册专利後，不符合标准的餐厅禁炒此饭？别说饮食界，就是我们消费者，也觉匪夷所思。

　　那麽「上海粗炒」、「厦门炒米」、「东北拉皮」、「福建炒面」、「北京填鸭」、「新疆羊肉串」、「云南汽锅鸡」、「桂林米粉」……等等呢？也要采取行动？

　　「印度拉茶」、「星洲炒米」、「瑞士鸡翼」……便是国际纠纷。「东坡肉」、「麻婆豆腐」、「宋嫂鱼羹」、「左宗棠鸡」、「毛家菜」……的主人，全得自地底爬上来了？
