无印的member


　　香港有不少「无印」迷。间中便听得有旅游公司的职员或空姐，顺道在东京入货带回来转售。有些还开了小店，专卖「无印」（或同类）产品。――必然是有需求，才有供应。

　　九七年大举撤离本城的「无印」，原来「品死品还在」。

　　有人问要不要些新货？――他们大概不知道，我本身是「良品计划」的member。至今还收到公司寄来齐全的catalogue。所以第一时间欣赏到一批简约、迷人的新设计。

　　其实是很久之前入会的。

　　多年前在日本，朋友都是member。日常起居一切用品，大都同一个tone同一形格的。如果「无印」有西装，大概便全部邮购算了。

　　「生活」在一个地方，不免入一些方便而有好处的会。入「坂急」有折扣，但入「无印」无折扣，因为新货抢手。不过他们也定期有些减价货品，好抵。

　　我喜欢掀着他们的各种产品册子（不是小册子，像书一样厚呢！），细细欣赏，好舒服。草木染料的夏衣，看上去又cool又整齐的收纳箱和架子、素淡的寝具、寂静的文具、温柔的和纸木灯。还有一切家电，即使甚少举炊，也想拥有一个如外星人用的金属电饭煲，还有今年炭灰衬银白的吸尘机。
