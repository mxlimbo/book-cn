百花齐放工展会


　　一年一度的书展：七月十九至二十四日。——不过对我来说，也是「平常心」。

　　书展诚然是热闹的盛会，有一个目标，也有dead-line，大家得交出功课。不过真要为书展「赶」一本书，也没试过。

　　记得去年七月，真任性了。

　　已经把《烟花三月》的初稿完成，在修订中。忽然一觉醒来，觉得还不够，决定不出。但出版社已做了宣传预告，孩子没生下来，老板几乎没气坏了：「都临盆了，怎会『不肯』生呢？」

　　还是把初稿扔掉，重整架构，重写。近二十万字，多花了时间，今年才能出生，我没有後悔。此外是《水云散发》、《梦之浮桥》和《矿泉水》新版本。

　　去年大会首度印制一套十张书签，很精美。我还收藏着陪伴成长的作家偶像签名呢。可惜部分在外国或没签上。今年是第二批。看项目，除「夜市」和「速递」外，更新设了「网上天地」，那麽到了电子书时代，书签是出土文物了？

　　——但我仍然爱书。希望年年有书展。或像文化地，百花齐放。或像ball场，名媛们早早准备华衣礼服，在镁光前向各界招呼。或像工展会，凑热闹的游客不会空手而回。或像假期之前的加班，它开幕了，很多人便可收工了。……
