水龙头是否开着


　　有一回朋友随便聊到一个有趣的问题：「你每天早上刷牙，把牙刷洗净然後挤牙膏的过程中，水龙头是否开着？――有没有想过这是最无谓的浪费？」

　　有人这才想起，对，自己一直让水溜走。太不环保。有人根本答不出，说要回去做一遍，下次相告。有人不大关心。水龙头是否开着，同生命有甚麽牵连？

　　我的习惯是停了水龙头。不算甚麽「环保」意识那麽严重，只是不喜欢水白白流去，还有，我顶讨厌半死不活的，滴滴答答的水声。没公德心的人总是开了水龙头，任由食水浪费便离去，见了你会顺手代关牢吗？――我觉得电子感应的开关掣最好了，文明又无後顾之忧。

　　说到环保和不作无谓浪费，其实我们浴室中很多东西都是「多馀」的。

　　最近用一只日本绿茶厕纸，它芳香柔裕擦手同纸巾差不多，但旁边仍然有好一点的纸巾，方便抽取即用。且忘了毛巾。

　　还有洗手液，难道挤点沐浴露洗手不可以吗？但又多了好几个瓶子。洗面用另一些。

　　润肤、润面之馀，双手同时滋润了，润手霜是否多此一举？特地买一瓶润足的？实在太过「斩件」了。当然，这叫做提高生活质素。

　　你们呢？数下去还有吧……
