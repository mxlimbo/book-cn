放心放心


　　曾有港姐在录影节目时，因一个镜头直落讲十句对白，心情十分紧张，NG四次。事後她说：「死啦，听日实做头条，实俾你话啦！」

　　——放心放心，头条不是那麽容易做的。太——正——常了。

　　多少人等争做头条，各出奇谋，甚至不要脸，才稍占一席位。念不出对白吃NG便想「跃登」头条？一天二三十人NG，有甚麽特别？是否太平庸了些？读者会骂的。他们已被宠坏了，等闲小事怎获青睐？要上娱乐版头条，必须：——

　　（一）卖淫、偷食、同性恋

　　（二）卑鄙无耻「完唱」

　　（三）粗言秽语，疯言疯语

　　（四）三角（最好是多角）情仇

　　（五）老幼不和朋友反目互揭阴私

　　（六）妻离子散

　　（七）家破人亡

　　从前，上头条是一种光荣，但今天，千万别虚荣，这要付出代价，不值得恭喜。

　　当然，有些相对而言比较「轻松」：佳丽努力地把对白念好表现称职，无论如何，比不上装好身娇俏地搀扶年事已高的老板去社交，那麽容易见报的，而恩爱情深的艺人专访，则用来摄期。对不起。
