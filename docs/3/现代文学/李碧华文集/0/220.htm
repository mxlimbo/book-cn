纪州青红梅


　　和歌山县有两大片梅林，是日本第一产梅「圣地」，所以常见的梅制品，都标榜「纪州梅」，传统口味。

　　光是酸梅，已有几十种，颜色深浅个子大小，分门别类。餐前作开胃用，酸得整个胃都涤荡一空似的。

　　原来小小的梅，可以制作那麽多不同项目：——

　　梅汁、梅味天然水、梅味黑糖糖、蜜糖梅渍物、梅胡麻、梅味芝麻、梅肉酱、紫苏梅醋、梅味蒜头粒、梅味　头、梅味沙律酱、梅蛋糕、梅虾饼、梅鱼饼、梅茶、梅酱豆腐、梅酒（青红两色）、小梅昆布……。

　　当然，除了梅外，柑、柚和柿，也有不少姊妹作，但没有梅的「排山倒海」。

　　梅有一种独特的香味，而且颜色浪漫。台湾有梅子汁，名字索性唤「调情媒」，像个牵线的中间人，不过用的是青梅，有「青梅竹马」的少艾情怀。日式弁当，小菜分成数格，当中饭团必放一颗红梅，娇艳欲滴之状，比较诱惑，引起一阵欲望（食欲）。而我最喜欢的，是纪州赤梅酢渍物，有茄子、小黄瓜、姜、昆布、紫苏，和一些不知名的花。胶袋装，十几元一份。下时以之当配菜，「喧宾夺主」，比主角还高贵些。於是，连可怜兮兮的午夜即食，也减去几分凄凉了。
