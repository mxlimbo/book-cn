「哭泣室」的留言


　　Isabella在电邮上说，某天，她在食店厕所内发现了这样的两行字：

　　「我爱他，他却偏不爱我。」

　　这是要领钥匙才开启的厕所，所以留言者一定是女的。

　　她用油性笔书写，字体秀丽而工整，猜想是个清秀又带点害羞的女孩吧。笔迹并没被工人清洗掉，所以在这厕格内「解决」的便人人都看得到了。

　　大家常在公共厕所看到不同的留言，有些很猥琐，有些很哀伤。字体也有好有坏（不过是差劲的多，而且错别字），还有附上自己的电话号码。

　　选择在厕格内展览心事的人，不知是甚麽心情？是拿着预备好的笔，写上预备好的字句？抑或躲在里头哭泣，怨恨命运，一时有感而发，顺手便写了？

　　女人很喜欢躲在厕所里伤心流泪，彷佛这是一个安全而固闭的「哭泣室」。不知过了多久，眼泪乾了，出来洗把脸，重整身世才见人。自叹的句子，留待同病相怜的女人去发现。

　　――但世上最寻常的痛楚，不外是：

　　（一）他爱我，我却偏不爱他。

　　（二）我爱他，他却偏不爱我。

　　没有对错，痛在「偏偏」，不能勉强。为甚麽？也没得解释。写出来，不一定便好过点，但，你们哭吧。
