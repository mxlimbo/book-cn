青苹果汁拌凉面


　　在文化中心看完表演，很多时会顺道到「沪江」吃夜消。他们的菜式精美，而且厨师经常有「新意」。每隔一阵便创作些新点子，让来客贪玩一尝。

　　黄油蟹当造，有几款菜式点心以它为主角。不过那晚我们都不饿，只吃小点。一荤三素蚕豆火丁臭豆腐之类。还有萝卜丝酥饼。特别点了店中特色拌凉面。

　　上海拌凉面有很多「器皿」，各盛不同酱料，像重温儿时「煮饭仔」。（「功德林」也有这一款素食。）

　　上桌时，一盘凉的黄黄绿绿蛋面和菠菜面，还有一碗碗的怪味汁、芝麻酱、辣油、醋、腰果茸……。

　　有一碗，黯绿色酸酸辣辣的，不知是甚麽？一问，原来是「青苹果汁」，厨房的试验成绩：青苹果磨茸汁，加上Tabasco（美国的「辣椒仔」汁），还有一些调味料，给弄成这「物体」。拌起凉面，口感清新，好有趣（不过颜色差些）。

　　做起来不难（当然，最难是度桥！）。便想，如果我们弄一盘日式的梅子冷面或绿茶冷面，青苹果、红苹果、梨子切丝，浇上芝麻酱……等，再加上新鲜的「青汁」，一定好看又好吃。

　　Sorry，师傅我抄了你的桥。
