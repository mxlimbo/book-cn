好闷的茄汁薯条


　　对於薯仔，不算特别厌恶，只是不大喜欢炸薯条。混在咖喱汁中的薯仔就比肉类还好吃，但我永远不会买M记的套餐，因为他们总是强迫性地非要我吃炸薯条。还有茄汁，为甚麽一定要蘸茄汁？既难看，又难吃。

　　台北的M记薯条一度「不必」蘸茄汁。曾推出一种有趣吃法，便是买薯条附加一个袋子，和自己挑选的调味包（有四种口味供应）。吃时把薯条往袋子一倒，再加调味粉，自己摇，「摇」出不同滋味。如果是这样，而调味包又愈出愈多的话，吃薯条不再枯燥。——但我只碰到过一次，还没试齐全，便已消失掉。依然给你一包茄汁。

　　好闷。

　　英国一间酱汁公司还研究出一个巧妙的方式，便是「薯条内含茄汁」。我不明白，是每条薯条的「里面」都包裹了茄汁？抑或只是沾上一些茄汁的「味道」而不用蘸着吃？——这创意似乎适用在洁癖主妇上，还有，把茄汁费劲地自瓶中倒出，动作激烈以致一塌胡涂的猛男，比较欢迎它。

　　美国人热爱薯条，但英国人冷静分析。也是英国，「薯仔总会」的心理学家又就人们吃薯条的方式，窥探他们的性格和性爱取向，分为（一）食家型。（二）狼吞虎咽型。（三）细嚼慢咽型。（四）牢骚型。（五）抗拒型。——真是平常谈。毫无新意。
