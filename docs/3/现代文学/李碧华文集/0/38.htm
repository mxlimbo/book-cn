黄金奇异果


　　在超市买水果，见到一新西兰的新品种，唤「黄金奇异果」Golden Kiwi。

　　它看来同我们常吃的奇异果差不多，不过身形较苗条，首尾较尖，茸毛较少，像一个大橄榄。价钱也较贵。内部构造则是黄肉黑籽，带点蜜瓜味，也有桃子和柑橘味，酸中有甜。

　　这是不同植物以配种及接枝手法「弄」成的混血儿，并非绿血人，而是金血人。

　　也许有人认为「改良版」好吃。不过原装奇异果更有特色。

　　那种「翠绿」，十分独特，没甚麽雷同者。

　　在外国称Kiwi，中国黄山最多这野果了，并不「奇异」，名曰「猕猴桃」。黄山的猕猴，不但爱吃猕猴桃，还爱「种」猕猴桃，——那一大堆黑籽，生命力很强，随着猕猴满地拉矢，八百里黄山，到处繁殖，触目皆是果子。

　　　猴桃丰收，猕猴开rave party吃不完，采了果子搬到洞穴储藏，果子发酵，酿成「　猴酒」。　猴喝了酒，酩酊大醉，不省猴事，常被猎人不费吹灰之力，手到拿来。

　　明代有书，称这酒为「悟空仙露」呢。

　　不过做水果或蔬菜沙律时，把黄金翠绿冷暖双色奇异果切片，铺在上面，黄绿相间，真令食欲大增。最好也弄些红色和紫色的来。
