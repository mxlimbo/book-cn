瞪大眼睛说瞎话


　　大坂下 着大雨，但日本进步团体、华侨、中国留学生……，都冒雨张示横额，在「国际和平中心」外抗议示威。

　　南京酷寒，五百多名市民顶着大风雪，在「南京大屠杀纪念馆」外进行声讨大会。——因为右翼分子、教授和战犯，举行「二十世纪最大的谎言——彻底检证南京大屠杀」集会。在此之前，日本政府已无理判决揭露大屠杀真相的老兵东史郎最後败诉，判罚五十万日圆。

　　可见是一种丑恶的「铺排」。

　　一直以来，我们关注东史郎案，因为这并非一场「是否毁损战友名誉」的诉讼，而是右翼势力借此为突破口，乘机全盘否定「南京大屠杀」史实。

　　十二月底，我还收到南京遇难同胞纪念馆馆长朱成山先生的来函，告知他带了六万多人的签名信函和横额，到东京最高法院给东史郎支援。八十八岁的老人，仍精神矍铄，声音洪亮，理直气壮，律师认为根据各方面的实证，胜算颇高。一行人一点也不担忧。

　　——但，证据确凿之下，那麽多的纪录片、照片、幸存者见证、挖出的尸骨……，都被判作「捏造」？除了侵略之外，是最大的侮辱！世上有是非黑白吗？法律是公正的吗？为甚麽面对国际，仍有无耻地「睁大眼睛说瞎话」的活剧？实在太奇怪了。
