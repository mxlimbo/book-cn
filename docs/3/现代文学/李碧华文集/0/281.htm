悲痛只因为得闲


　　感情生活不如意的人，如果很快以另一段感情来填补虚位，如果寄情於事业挣扎，如果移爱普及世人……，总之，如果他忙碌，一定没有时间悲痛。

　　有些人悲痛只因为得闲。

　　闲来无事，把新仇旧恨一再反刍，积怨日深，――除非遇上更大的悲伤，方可覆盖了较小的。

　　记得某回看过一则报道，是法国一名二十一岁的电单车骑士，驶经一条村落时，被汽车撞倒，肇事车辆不顾而去，这名男子太过震惊，急截顺风车回家去。母亲来照料，除下外套时，赫然发现他整条前臂不见了！

　　他的前臂断後，遗在现场路边，而他懵然不知，只因「忙於惊慌」，忘记痛苦。

　　我们如何辨别究竟哪一桩哪一件更重要？

　　很容易，你的下意识会决定，――当你不肯为它痛苦时，一定有更重要的必须面对。

　　睚眦必报的人，记性那麽好，他再没有其他任务，只好花尽心力去记挂一些小眉小眼的不愉快经验了，否则多寂寞，闲来也难以打发时间。

　　在古战场，壮士心甘情愿为更值得的事（或人），为两害相衡取其轻而断臂，这不但是一种决绝，也是一份气派。

　　或者，是以一个伤口去换另一个伤口。
