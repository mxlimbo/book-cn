十四岁的后冠


　　「二千年世界精英模特儿大赛」於北京举行。各省市数千名报名叁选者中，挑出四十多名少女作最後冲刺，然後代表中国九月出战瑞士日内瓦的总决赛。

　　结果令人感到意外，因为冠军得主是全场年纪最小的南京姑娘黄欣烨。看照片，黄小妹直发中分，并无慑人艳光，也非身材出众。十四岁，个性、风格、气质，若有，亦在雏形吧。

　　她出色之处，是站在一众女孩当中，高挑、苗条、自信、清秀、淡定。

　　完全输得起。不在乎。不当一回事，大不了回去再碰机会。就这样，她蠃了。

　　以为「模特儿」是选形格，不过原来她曾以手风琴绝技，夺得了才艺表演的冠军呢，可见不是「一片空白」。

　　早前报载，本港某一宣传活动中，有数名被包装成甚麽人气偶像的玉女歌手站在台上当嘉宾。司仪当众挖苦：「十几岁无乜读书就出唱歌……」她们只能面懵懵地傻笑，云云。

　　现今娱乐圈充斥的，起起跌跌的都不过是这样的一批。——十几岁，当然无乜料，但连歌也唱不好，已经有fans，卖的不外最宝贵的青春。

　　是的，任何一行，成名得趁早。十几廿岁经历过，才可不当一回事。一把年纪才出道？扮细便很尴尬，拼搏需更如意。
