新马仔打蠃李小龙


　　广东俚语对白传神，在粤语陈片中时见。

　　一回黄曼梨形容某男：「花花Fit Fit，扎扎跳，『削仔』o黎o既唧！」

　　一回陶三姑形容某男：「松毛松翼，无衣食，无件头。」

　　她又极讨厌有租交又扮鬼扮马的寒飞：「靓仔又唔食得　C」 - 奇怪，当年的「男主角」，红的都是这种型。因为红，女主角不得不选他们。其中之一是新马仔，「人见人爱」，玉女投怀，非君不嫁。实在十分有趣。

　　看过一个弱智喜剧《诈癫纳福》。戏中的新马仔暗恋白露明，竟然「扮」十二岁被养父母（当然是父母双亡的孤儿才可怜）虐待逃家的「小童」来亲近，白居自然十分同情，收为弟弟，时加拥抱抚慰，冷落了英挺的胡枫。连白的表妹女配角也爱上了「早熟男孩」，不理睬青梅竹马的李小龙。新马仔与李 - 小 - 龙拗手瓜，竟大蠃！（布鲁士大概为此远赴英国卧薪尝胆去了。）

　　旧戏中的「爱情」，原来很势利。
