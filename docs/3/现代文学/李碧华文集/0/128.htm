微醉红酒糖


　　买水果，很少拣葡萄。走丝绸之路时，在吐鲁番的葡萄沟吃过最新鲜香甜的，之後，不大瞧得上才一小串一小串的物体。葡萄有皮有核，吃时很麻烦――不过某些「副产品」倒不错。

　　有种日本精致的冰淇淋，《彩》系列：「玉露」、「香草」、「巨峰」，是颗粒极大的黑亮葡萄。小杯冰淇淋还包含果肉，葡萄味清香，虽然比较贵，但很好吃。

　　近日十分热门的夏威夷排毒Noni（以火山灰泥土培育）果汁，本身苦涩，得加入多种味道来「中和」，像石榴、蓝莓、葡萄等。後者较易入口。但能否有治疗奇效就不知道了。

　　一天朋友请吃有趣的零食，有咸有甜，包括我永远不会买的巧格力威化饼和红酒糖。――红酒糖？包装的色彩艳丽诱人，每颗糖果中心还有少许红酒粉末，当然，它是葡萄味的糖，带点酒香，不算浓，但设计十分吸引。心理上，已微醉。
