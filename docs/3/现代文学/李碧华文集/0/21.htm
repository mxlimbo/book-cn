勇破情戒一代男


　　程介南大错铸成，欲挽无从，但最难受的，是他已不能进，又无从退，请大假欧游避风头固然不行（不似其他出事的名人般可以高飞远引来减压），连躲在下了厚窗帘的巢穴中舐伤口的权利也没有。

　　男人应该为所作所为承担。

　　——不过女人的角度：他有九分不是，我们还给予一分同情。这一分（不是「一份」），当然因为他过不了「情」关，为情所困，做出种种不理智的傻事和错事。

　　那些扰攘地打落水狗的「骚人政客」，封之为「惩戒南」，但，程先生亦同时分饰勇破「情戒」一代男。

　　五戒、八戒、十戒……，必然包括了最难守的情欲之戒。再狠辣、睿智、霸道、正义、伟大、富裕、骄傲……的人物，他们的死穴往往便是「情戒」。何况区区一位议员？

　　弃发妻、开公司、失诚信、想方设法多挣些钱、两头（或三头？不清楚）瞒、荐入学……，都是为了女人，一个迷梦。

　　既然多情，何必恋栈政坛？诗人作家教授律师食家电影人艺员音乐家……，即使中港货柜车司机，包二奶都没包得那麽惨痛。富商甚至可包四五奶。

　　落魄白头的「情戒男」，仍得女人天涯相伴拍部爱情片？便是新生。

　　浪漫，是由「不在乎一切」开始的。
