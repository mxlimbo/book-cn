一些东西留下来


　　活得尊严，老得体面，走得安详，这是所有经历过风浪起跌老人们的愿望。也真不容易。所以上海郭家四小姐戴西是令人敬佩的。

　　看到日本第四十一届「全国唱片大赏」中一项，企划奖。这CD是音乐、朗诵、摄影的结合，没知道来龙去脉和名字，但见一位九旬老人，在镜头前朗诵了诗作。

　　他那麽用心、感性，而且对艺术尊重。主持人、观众和合作者都泫然。

　　他自己亦带泪光竟全功。

　　道：在八十五岁那年，爱子也辞世了，身边的人一个一个的走了。他还活着。眼看春去秋来，亦悲从中来，心情起伏，不过有些东西留在人间，得到共鸣，还是幸福的。

　　生命短暂无常。「你留下些甚麽东西呢？」——只要这样一问，你就不敢放弃吧？抑或，一切顺其自然？

　　一些东西被「留下来」，很多时是无意的，被挑选的。而非个人费劲铺排。

　　不过我最记得这位老人家同节目主持人开玩笑：

　　「恭喜你得奖了老爷爷。」

　　「不，你不能这样说。」

　　「——？」

　　「不应叫『爷爷』，你得叫我『哥哥』。」

　　——仍是充满生之喜悦的……。
