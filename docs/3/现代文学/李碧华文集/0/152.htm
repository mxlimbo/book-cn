Tea Masala


　　以前卖印度香料茶包的店子「不见了」。很怀念。

　　如果它只是搬走，店海茫茫，大概还可找到。但「不见了」便是不见了，应该死心。

　　所以我决定另闯天地。

　　在铜锣湾霎东街，见到一家印度店子，它既陈旧又杂乱，还发散奇怪的香味。货品是各种印度土产、香料、果酱、咖喱、酥油、麦粉、药材……。还有米。

　　我问：

　　「有香料茶包吗？」

　　店主年纪不算大，是印度北部旁遮普邦人士。他的广东话十分流利，原来是香港出生的。他说：

　　「有香料，有茶包。——不过两样是分开的。」

　　他拎出一种印度茶包。我向来喜欢「Twinings」的紫包「Darjeeling」，其产地是喜马拉雅山，十分香浓。不过我也尝试一下他另一推介，以配合另一盒香料Masala。内容物是大茴（茴香）、肉桂、丁香、黑椒、乾姜、小豆蔻、肉豆蔻、胡椒根……，研成黄黑的粉状。我们在正宗印度餐厅餐後来杯玛沙拿茶，是「煲」出来的。好喝如毒品。——所以自己做一杯茶，加茶叶，加香料，加奶，得狠狠的「煲」，不是「泡」，才不辜负这麽浓重的印度风味。好正！
