大家醒来的时间不一样


　　某回在大陆某电视台，听到剧集主题曲：「千百个梦只是一个梦，我的梦里只有你，我的梦里只有你，只有你。为你生生死死，为你悲悲喜喜，千百个梦都离不开你……」然後打出四个大字：《痴心如火》

　　多麽悲壮激情。我想，柔情似水，痴心更应如火。——谁知看下去，都是「坦白从宽」、「抗拒从严」、「首恶必办」、「胁从不问」、「立功受赏」。

　　我误会了，这是个上海刚解放的故事片，在夹缝中的人，作革命的梦。「全国只爱一个男人」的样板戏。梦中都是主席？真是辜负了一个「梦」字。梦的歌，应该是：「情爱，就好像一串梦，梦醒了一切亦空。或者，是我天生多情，方给爱情戏弄……」这样。靡靡之音。

　　梦，无休无止，无始无终，极度虚幻，又栩栩如生。每一步都怕落空，小心地，舍不得不走。我们在梦中，心底经常明白，这是一个梦，快要醒过来……。

　　如果情爱只是一个梦，中间怎会有伤痛寂寞？——两人同做一梦，然後，其中一个醒了，离开了，甚至起床了，梳洗了，上班了，同别些人相遇了，重新去做梦了，不回来了。而另一个犹在昏昏沉沉的梦中。别人劝：「你醒醒吧！」「不，我不要醒，我爱这个梦！」在空局泥足深陷。

　　没有对错。是大家醒来的时间不一样……。
