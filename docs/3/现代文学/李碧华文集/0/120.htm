红线女热潮


　　北角区愈来愈多一楼凤聚集。淫业兴旺，嫖客上落出入令良家妇女好尴尬。如何「划清界线」？总不能像娱乐版，有负面新闻时，个个都说「清者自清」解围。问题是，浊者自浊时也浊人。有朋友住这区，她无奈：「自由社会，商业行为，小市民能做些甚麽呢？」

　　像251性女，她不惜下体起茧，亦以性斗士打真军纪录片来证明自己的勇毅独立，人各有志，不伤害人，她大可继续搏杀。

　　但看近日特稿，性女在港岛最热闹的铜锣湾漫步，窄衣短裙吃猪红，又有记者随街拍摄，「居然」没有被人认出来。

　　十小时、九十名陌生男子、做爱251次，与各网站争取浏览人次一样，也是争取一个「数字」，数字可确定人的位置，也可得到广告和名利，那麽努力，没有被认出好唔抵，应向特区政府借镜。

　　不知本港报章两级制的建议进展如何，但有一项确实有助识别，十分凄厉。便是「含淫亵或不雅内容的报章，自行在首页刊登警告字句及每页印上红色对角线」。

　　若然每间一凤楼大门髹上红色对角线，每位一楼凤穿上红色对角线制服，性女在脸上加上红色对角线……，这个标签，更是免费而奏效的宣传。

　　万一流行起来，时装波鞋手袋和化妆，都引发「红线女」热潮。
