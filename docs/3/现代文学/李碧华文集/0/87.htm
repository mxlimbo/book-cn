355料理


　　在日本吃过一顿十分高贵的「诚意答谢，珍重健康」美食。——「高贵」是指价钱。不说了。但经理出来跟我们商议，提供一份供选择心水的「度身订做」餐单，表示每种食物的益补好处，还有所含卡路里（这个很重要，因为必须算计好，每人每客不超过355卡路里才合标准，云云）。

　　只有五个项目：——

　　（一）黑芝麻手长海老、虾汁意粉。（113）

　　大虾拆肉再滚满黑芝麻香烧，果然美味。但最好吃的是虾的脑汁拌意粉，令人聪明，可惜分量太少，补不上我脑汁的消耗。

　　（二）春野菜清汤（意大利式）。（33）

　　主要是人叁（甘麼）、西兰花。汤清见底如水。洒上香草很清新。用料鲜美。

　　（三）鲷鱼、苦瓜圈、青豆汁（和风）。（151）

　　以当日最新鲜的鱼作料理之用。苦瓜青涩而味甘，但纤细，圈住的那块白身鱼我见犹怜。我喜欢鲷鱼，慢慢干掉。

　　（四）梅杏冰淇淋。（58）

　　极具特色的不太甜的冰淇淋，梅和杏，有鲜果，有果乾，混成一球，颜色娇艳。——这是我吃过最好的冰淇淋之一。

　　（五）薄荷茶。（不含卡路里）

　　薄荷叶切碎泡茶，既消脂，又凉又香，感冒也好了。

　　——欣赏这精算料理，不过仍有点饿。卡路里，你真可恨！
