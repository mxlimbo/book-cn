示范单位电冰箱


　　明星的网页、作家的专栏、高官的报告、名人的演辞……，其实都是：——

　　（一）「示范单位」：面积有规限，但却能把希望你看到的，全都呈现。乾净企理，设计时尚，——并且，没有一粒灰尘。

　　怎麽可能？你我的家居，天天不知如何的便沾了尘。杂物又多，堆积如山，看不完的报刊一天後摇身变为扔不掉的垃圾。有人来了，赶快狠狠地收拾，把讨厌的东西给藏起来。「示范单位」是逢人只说三五七分话，未可全抛一片心。太真实了，反而不好意思。

　　（二）「电冰箱广告」：更加美观、整齐而虚伪。穿上刚买回来新熨的围裙的主妇、稳重爱家薪优粮准下班後仍不带一丝油垢的丈夫、聪明活泼敬老护幼的孩子（总是「兄妹情深」）。电冰箱每格的内容物恰如其分，各归其位。水果永远红、橙、黄，新鲜艳丽。鸡蛋、牛油、乳酪、鲜奶，当然，还有半打啤酒和可乐（必是一公升装）。而火腿、鱼、鸡、肉、蔬菜（必是番茄、生菜、小黄瓜、胡萝卜。因为衬色）都眉开眼笑地招展。

　　现在，你试试打开自己的电冰箱吧。不是有昨天剩菜、半个饭盒、吃不完的冰淇淋、几个颜色平庸的水果、乱七八糟的物体……？不是随随便便摊在某一格，懒於分类吗？
