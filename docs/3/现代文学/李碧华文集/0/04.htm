误入歧途泰语版


　　因为没得比较，所以不知算不算当黑。

　　《人妖打排球》靠宣传造势，和粤语配音作卖点，上座率不错。想看但不知如何没看成。以为落画，所以也就算了吧。那天天气差，在商场附近，刚好有戏院子放这个戏。为了避雨，时间配合，便入场。——谁知坐定後，才发觉，天呀，这是「泰语对白，中文字幕」。

　　我已一身湿，还要捱——泰——语——片！

　　有没有得退票？

　　要不要把它捱完？

　　为甚麽我明明想听葛民辉黄伟文吴君如谷德昭……，现在「原汁原味」？本城最「误入歧途」的观众竟然是本人？……

　　但看下去，又觉得现场收音几生鬼传神。尤其是妖妖地的对白，用妖妖地的原装泰语来表达，　晒嘴形。全场最趣致的「活跃基」，十分可爱。——虽然我错过了本地加工加盐加醋，也不知道他们给改了甚麽大号，但「凤爪基」、「忧郁基」、「含羞基」和令女人妒煞的「美艳基」，人物立体。

　　戏就一般了。间中还有闷场。忠奸分明不忘说教，人妖在球赛胜出也以妖艳浓妆来眩人耳目而已。说到底它好泰国，但谈不上甚麽体育精神。

　　开心笑半场。入错戏院睇错戏，也有意外收获。
