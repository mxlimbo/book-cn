奢侈的二十分钟


　　日本白领一天工作累了，近日流行到「氧气中心」吸氧，松弛神经。二十分钟收费近二千圆（港币百多元）。

　　吸氧不是「新风」。记得多年前，中国大陆也有过「氧吧」的潮流，供经济改革开放後，有馀钱的大腕，和港客去吸入新鲜氧气，养生减压云云。但这些雨後春麼的「氧吧」，不久便做不住了。

　　呼吸还得付钞？所得不过是宣传中较清新的空气吧？半信半疑。

　　日本人竟然也当作新兴享受，收费昂贵。

　　其意义也许是找一个好地方，躺下来休息，甚麽也不想、不做，光是呼吸便够了。「吸氧」只是手段。

　　除此之外，还流行一些短时间的按摩。指压中心设在热闹的地下街，格局高尚，布置得舒服、安详，如海洋的感觉。自称Refresh Hands。客人多半是行政人员、OL、「小姐」、大学生。可以做脚部或上半身的按摩，所以只坐着做颈背部。一般而言，二十分钟的收费是二千到二千四百圆。我试过，比起深圳的盲人师傅连做二小时不欺场，差太远。——但这里乾净、摩登。人们shopping累了，不去下午茶，去吸氧或refresh更时髦。

　　老实说，以二十分钟为单位？时间太短，甚奢侈。
