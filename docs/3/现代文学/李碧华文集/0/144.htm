钢铁是怎样炼成的


　　在地铁，一个肥师奶（对了，师奶都是「肥」的）展开霸位攻略。一屁股坐下，左右一挪，马上又多夺得了两个空位。在她一声号令下，丈夫和女儿也得安身立命之所。「快来坐快来坐，有位！」－女儿理所当然面有得色地坐下来，她幼承庭训，长大後肯定也是一个擅长霸位好好安顿家人的肥师奶。

　　只是那个男人，有点讪讪的，在众目睽睽下，爱妻摆布下，他得享区区一个座位，但付出了尊严吧。－理论上，他应去护妻，何须女人出头强悍？

　　遇上肥师奶一家子，通常我们都「让座」。车程那麽短暂，想不通为甚麽非要一矢三雕？也许只是女性「权威」和自保。

　　记得在武昌，「辛亥革命纪念馆」前，有一个个小摊子，出租军服和配刀给游客拍照。有个女人，十分扰攘地为男人整装、移帽、造型、觅外景地。还教他摆出威武雄姿。因她的「巴闭」，所以引得旁人驻足，我们还拍了十几张照片留念，－世上很多男人爱「扮」军人，原来，对女人来说，除了「制服诱惑」，还有幕後总司令之乐趣。

　　又见一恶妻，在火车卡上，抱着小男孩当众小便，面不改容。她的丈夫木然地站在一边任由她干此勾当，束手无策。

　　这些无力的一家之主，没感到不好意思吗？真有趣。
