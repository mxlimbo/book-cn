目的是「蠃」便无情


　　贺岁波的演出，香港联赛选手队大部分都是外援。

　　香港球迷不满：「为甚麽不重用自己友？」

　　负责人说：「港联伤兵问题严重。落场不会有表现。所以用外援是合理的。」

　　又道：「球赛目的是蠃波，而不是出甚麽球员。」

　　球赛的结果我们不谈。但出赛之前，唯一目的是「蠃」。不过是这样。非常简洁、明确、势利。没有同情分，不能将就，谁也不必给谁面子。

　　上阵交锋应该「无情」。适合，上；不适合，不用去了，在一壁凉快吧。这是「供」、「需」的问题。教练最爱你？一向受器重？你表现优异？人缘奇佳？你最多fans？你万人迷？……假如这一役，你状态不足以「蠃」，一切等於零。这是人们在社会立足的第一课。不光指球赛而言。

　　因为每一个个体，同老板都「非亲非故」，这种关系最好了，不会拖泥带水，没甚麽不好意思。当失去利用价值时，双方皆不勉强。心知肚明。

　　一些年事日高的艳星脱星，欲重出江湖拍三级片，她们道：「我重得……」不，这句话总不能自己说。

　　现实当然是这般残酷，丧尽天良了。我们从不怀疑。
