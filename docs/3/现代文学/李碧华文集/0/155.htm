执了便执了


　　有百货公司和快餐店全面装修，当然得注资翻新，更上层楼。但更多店结业。

　　走过铜锣湾，当初人们大排长龙轮购一个日式芝士蛋糕的「名店」，已是「全店器材大平沽」。一带的商场街，每隔一阵，换一个主人。都是消费品，总有人肯多花一点来提高生活质素。你看，化妆品公司便客似云来。

　　不过，开开关关起起跌跌虽已惯见，「突如其来」，也令人唏嘘。

　　J他们办公室楼下的一家茶餐厅，在单边位，中心点，生意还不错。前天还吃过蛋牛治和奶茶，下午上去看设计图，刚想充电，谁知重门深锁，一点徵兆也没有，便「消失」了。问对面的摊贩：

　　「前天还好好的，今天便关门？——是休息吗？」

　　「执笠了。」一个木然地答。

　　「不会吧？」

　　「做不住便执了它，有甚麽会不会？」另一个不带一丝感情或街坊交情。她也帮衬过下午茶餐吧。非常之cool，像谈着南北韩或非洲莫三鼻给的事。

　　「转了几手啦。」

　　——朝夕相见的老板和夥计，一下子不知散落本城哪个角落。事件的结论也很简单、凉薄、理所当然。

　　没有人怀念任何人、任何店。
