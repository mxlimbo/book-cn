情场「五常法」


　　原来企业管理有所谓「五常法」。此意念也许早已有人引进公司，并且实施奏效。——但我不是现代商人，也非管工，所以只是从报道中无意得知。「五常法」的心得是：

　　（一）常组织：判断物品的重要性及使用频率，作出组织计划。

　　（二）常整顿：每件物品都有一定的存量及摆放位置（它们的「家」），避免乱放，寻找费时，影响工作效率。

　　（三）常清洁：每位员工每天要执行清洁岗位的责任，大家才有归属感。

　　（四）常规范：连续及不断坚持这管理方式，经常一一检查。

　　（五）常自律：关键在於每天自动自觉，养成习惯，共同建立一个良好及舒适的工作环境。

　　「五常法」是大原则，实施起来项目很细碎。但日积月累，便有成就。

　　——情场上，也应该叁考，便是：（一）常组织（令爱情增值）。（二）常整顿（负面的破坏力）。（三）常清洁（身心）。（四）常规范（保鲜）。（五）常自律（向魔鬼的诱惑说不！）。
