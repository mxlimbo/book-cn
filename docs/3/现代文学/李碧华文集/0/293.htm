冷的咖哩失分


　　嗜辣的人喜欢咖哩，大辣，热吃，几乎喷火。但咖哩是不堪冷落的。──冷的咖哩失分，变得惹嫌。

　　一回同朋友吃印度菜，意犹未尽，再多叫了一客羊肉外卖，预备赶稿时消夜吃。放进冰箱中，第二天，整个冰箱都漫着一阵咖哩的味道，像宿臭。昨晚的浓郁芳香，竟无觅处。打开一看，卖相也欠佳。

　　当然，翻热之後，它好像也「改过自新」，回复一点状态。

　　咖哩有红黄绿黑白多色。香料中黑色小茴香据说可消灭癌细胞。好的咖哩令夏日恹闷的胃口大开。我也爱吃杂菜蛋，不吃肉，光是汁也可淘饭。下午茶来一件咖哩酥角、批、包子，加一杯又醇又浓的奶茶，抖擞精神。

　　──但，全得趁机，在它热情如火的时刻享用。餐後，滴在桌上的咖哩汁，沾衣的咖哩汁，都带过气的，沧桑的怪味。

　　记得杜杜曾介绍「咖哩西瓜」的烹制法。怎可能？西瓜是冰镇可口，它还又名「寒瓜」呢。他教人在西瓜汁内加辣椒粉、姜黄粉、蒜茸、芫荽及盐，倒进加油烧热。以小茴香子爆香的锅中，慢火煮五分钟。汁稠後加糖加柠檬汁，再煮西瓜粒。三四分钟上桌。

　　一种热不得，一种冷不得，竟可配合成主菜？不知是谁发明？
