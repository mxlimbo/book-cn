虐待不是慈善


　　每年农历七月，坊间都有盂兰胜会派「平安米」活动。相同之处是：——

　　（一）轮米的多是老人家。

　　（二）在露天场所例如足球场排队。

　　（三）派米时是下午，但老人们多於凌晨已来等候。

　　（四）在酷热太阳下，十分辛苦。

　　於是，每年都有老人，即使带备雨伞、纸、废纸皮……等遮阴　凉，在33℃高温下仍无济於事，守候整日，起码十多位67-83岁白发苍苍的老人曝晒、昏晕、倒地、受伤。惟恐错过派米的更不肯进院。

　　二千多枯候者，极为坚忍。有因为贫困，也有因为「平安米」是经法事超度祈福，拜正神可保平安，为儿孙去轮候的。

　　一年一度的话题。有机构已举办三十多年了。——但我很奇怪，「虐待老人」事件，负责人不是不清楚的呀，他们多有经验！

　　所谓「慈善」，是行好、方便、济世，总不能年年都重蹈覆辙，「平安米」不平安，多人晒晕也不思改进吧？方法一定有，很简单，可以早一晚派「筹」、派饼卡、米卡，一两天内任何时间到取，不必顶着毒日排长龙。

　　——除非，目的是「造势」。

　　明年，请你们把善事人道化。
