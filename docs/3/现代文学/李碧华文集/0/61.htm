绿茶和绿茶豆浆


　　夏日饮品，除了矿泉水，还有冰冻的绿茶。当然，乌龙茶桂花茶也很清凉。有一款无添加之「六条麦茶」，和有十六种草本植物天然素材（包括大麦、绿茶、玄米、黑豆、昆布、灵芝、熊、柿叶、红花……等）煮成的「十六茶」，是常喝的。冲绳是日本「长寿之国」，有「冲绳健康茶」，标榜材料共二十一种。味道较苦涩，喝後则甘美。绿茶的牌子更眼花撩乱了。

　　我们都明白，绿茶是冲泡热喝才好。瓶装只是大路货。有些以「生茶叶抽出物」作宣传，有些自高身价称「玉露」（但玉露是不堪搁久的，会变腥），有些则加了蜂蜜、梅、桂花、茉莉……，反而不伦不类。

　　我喜欢「伊藤园」的一番绿茶，味道比较清香，茶色也漂亮，每回都买二公升装的，－小瓶的又贵又不环保，塑料瓶子即弃太浪费了。「伊藤园」还有其他品种，还是绿茶好些，也许心理作用，也许因为它有「新俳句大赏」，由喝茶的人投稿，佳作印在瓶子上。第十回的短句，是一位名古屋十五岁女孩加藤荣子所作。－忽觉在33℃下「冒汗」的瓶子，除了绿茶之外，还有点甚麽。

　　此外，我也爱在city'super的豆腐专门店子，买一大瓶无糖的绿茶豆浆。因不含防腐剂，要放冰箱最冷的一格。最好即日干掉。无以尚之。
