半明半昧最动人


　　信用卡广告：——

　　女孩「经常」光顾同一书店，恋栈不去。目光偷偷地溜，若无其事，不知是看书，抑或看人？在书店工作的男孩，忐忑地想：「无，真系无。成日，都系因为……咋。」

　　女孩「经常」光顾同一咖啡室，恋栈不去，总是示意添咖啡，若无其事，她是在等他吗？在咖啡室工作的男孩，忐忑地想：「无，真系无。成日，都系因为……咋。」

　　宣传信用卡签帐有优惠和抽奖的两个广告，都用同一对主角。少男少女，乾净清爽，而且带一种久违的含蓄书卷气。——书和咖啡，总不能「所托非人」吧。

　　掀到周刊一篇特写：「蛊惑仔身边的女人」。十五六岁的wet妹，流连公园，即晚受媾，恨做「阿嫂」。她们的「男人」，也不过是又寒又削，人家的「」而已。你会觉得太可笑，——但，在街上走过的少男少女，都是这些人。谁还泡书店咖啡室？

　　广告拍出若即若离，又疑又惑，乍惊乍喜，自欺欺人的情怀，而这个阶段，很快过去。——深恐道破，可能是另一开始，也可能是结束。心乱如麻或心如刀割。怎会「无」？——爱在半明半昧最动人。
