台前皆幻影


　　翻看任姐逝世十周年纪念特辑的录影带（那天我不在港），发觉上节目的大老倌大明星，忆念前尘，全都变成又嗲又怯的小女孩，彷佛又回到几十年前，初踏台板，又惊又喜表演欲又强的青黳日子。阿刨尤其「嗲」，比阿嗲还要「嗲」，叫观众一时忘了她曾是戏班中的「鹤」（担戏文武生）。

　　当然，没有安排「雏凤」合唱一曲，是最大遗憾，也令这个特辑欠圆满。

　　也许是阿刨已退出了，彻底地不开金口。而她的好拍档阿嗲又另组班，不便有所比较。——只在访问中谈谈师傅轶事，难以「重拾旧欢」。

　　演唱折子戏的，除了「八和」新秀和当年老倌外，还有嘉宾张国荣和梅艳芳，他们也是「情侣档」，可惜亦不能合唱，——因为两个都唱平喉。

　　所有台前的「生旦」，都只是幻影，都不长久，难以永恒。反而幕後任白是经典一双。

　　《花院盟香》中一折：

　　「鸳盟初订莫相猜，便似金坚难破坏，任天荒地老，莫折此紫鸾钗。苦相思，能买不能卖……。愿作双鹣鲽，情深永无懈，一夕恩深记紫钗，赤绳长系足，那得再图赖？」

　　《紫钗记》其实是雾水姻缘一夜情，——但得一知己是死而无憾的。缘分就是这样了。
