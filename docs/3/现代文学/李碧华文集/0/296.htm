「狗不理」sell甚麽？


　　天津「狗不理」，是包子的美名。——其实不过是菜肉包。

　　它之所以好吃，与众不同，关键在於肉馅是三分肥七分瘦，调料独特，包子上的花摺整整齐齐十五下。以前的包子只有蒸的，现代改良下有煎和炸。不过还是蒸熟了，一口咬下，肉汁微溅，烫嘴，香香的吃，最好。

　　「狗不理」的典故，是清光绪年间，河北有个少年高贵友，天生倔强脾气大，人人不敢接近，连狗也躲开不理。他长大後拜师学艺做包子，闯出名堂，索性唤「狗不理」。又有一说，高贵友小名阿狗，他开店成名後，客似云来，亲朋戚友来找他，没工夫搭理，大家怪他架子大，「狗不理」。

　　——不管哪种说法，「狗不理」sell的是「寸」。

　　东西好，不必应酬、宣传、到处攀附、拜票。谁都不理睬。这是比较孤傲的方式。当然有些人非议，认为不懂人情世故。

　　至於「狗不理」是否真神话般好吃？也不见得。

　　包子好吃，首要的推动力是饥饿。——不够饿，两三个已撑死了。

　　一回在大陆的餐卡吃早点，套餐是：玉米粥、酱菜、十个包子。十个！

　　干掉这顿，整天不用吃饭。血液全跑到胃中主力消化，呆滞得狗不理。
