凌晨的凄厉男声


　　报载，深水湾发生斩人事件。一个一层六伙板间房分租单位，其中一户夫妇收听粤曲，声浪太大，造成滋扰，另一户住客为油站夜班入油员，收工後无法休息，所以理论争执，酿成血案。

　　贫困的住客因经济压力、摩擦口角、嫌隙渐深，冰冻三尺，非一日之寒。

　　——不过，在最疲倦最希望入睡之时，任何声浪都令人暴躁。而那些「享受」中的人，是无法得知自己有多可厌。最具杀伤力的是卡拉OK。以前这勾当流行时，家家户户传来杀猪声，狼嚎叫，五音不全，对邻居乃酷刑。理智的人会报警寻求解决；意气用事，便在港闻版见报。

　　如果能为人着想，收细声量，或用耳筒，便大事化小。若自私，甚至负气故意加大噪音，容易制造悲剧。——我有立场，因为我比较喜欢静。

　　在晚上或通宵写稿，正因为万籁俱寂，寂寞但寂静，特别有灵感。

　　也有例外，凌晨一时二十分，忽然不知何处传来一阵凄厉的男声：——

　　「你怎能这样对我？」

　　「你不要这样！我好辛苦呀！」

　　「你为甚麽不爱我？」

　　在黑夜中飘荡，半山，远望只馀几个亮灯的窗口，是哪一个？站到窗前，在最静的一刻，反而好奇地希望它「继续发展」下去了……
