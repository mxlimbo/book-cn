穷人、性大检阅


　　城中小市民，都追看富豪和名媛私人恩怨演变成悬红起底事件。低下层终於有机会嘲笑上流社会了，还不得意？一众成为动物故事的读者（鸡、龟、淫虫……物语）。

　　悬红目的好「正义」。报载：「呢个正一鶷家铲，一边扮名媛，一边去勾人老公，害到人妻离子散，你话系咪为害社会？……我今次样做，系为民除害。」而爆料对质，更加活色生香兼露骨。陆续扰攘多天，人人自危。不啻是「人性大检阅」，或「人」、「性」大检阅。

　　一旦出事，那些利欲　心努力刮料揭秘的，是甚麽人？是新知旧雨、十几年前老相好、手帕交、姊妹淘、食拍档、好朋友、闺中密友……，挽手并肩，言笑晏晏。第一个出来的，便是这些。

　　平民老百姓，身边总也有几个这样的人呀：——聆听、开解、抚慰、同情、和你一齐去识人。一交换情报、交换意见、提供方法、有福同享……，有时还对你赞美欣赏讨你欢心，所以你们是知己。——有了这些知己，还需要敌人吗？敌人对你的了解，对你的杀伤力，永远不及「知己」强。遗憾的是，那「掷地有声」一百万悬红，竟然令高级程度会考新闻，三万五千名考生中的7A超级状元，也退过一旁。

　　优秀的人才，读书何价？——要赚一百万，原来「陪赌」和「卖料」最容易？
