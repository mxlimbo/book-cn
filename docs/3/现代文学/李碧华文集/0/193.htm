有时不必五分钟


　　「只是五分钟的时间，整个世界已变得完全不同了。」

　　最近澳洲网球公开赛一场男单准决赛中，森柏斯与阿加斯苦战五盘，历时二小时五十五分钟，最後，他输了。

　　比赛当然有胜有负，有上位有出局。——而决定性的手势，也许五分钟，也许，不过是五秒。

　　如果相扑，纠缠才五秒，一秒可定江山。

　　很多人在生活中遇上了「整个世界变得完全不同了」的时刻，你不能预测，也无法掌握，但这一刻来了……。

　　一个女人自白，她一直以为有孩子有爱自己的丈夫，就有了全世界。——原来当某一天他不再爱你了，基本上是「一无所有」。一下子十分孤独、伤痛。

　　报载，某失业经年的男人，不但没拿家用给妻子，反苛索金钱去包二奶。妻子为此愤怒悲哀，但为了小孩，忍受很久。那个晚上，他竟然表示「仔都不要！」在睡梦中，男人被她挥刀狂斩，重创。妻子被捕，小孩吓呆了。

　　今天跟昨天不一样，此刻跟刚才不一样，这个变化，太微妙了。有时可能是一句话：「对不起，我决定跟他。」或者：「从头再来好吗？」或者：「原来我如此挂念你。」或者：「我愿意。」或者：「算了。」
