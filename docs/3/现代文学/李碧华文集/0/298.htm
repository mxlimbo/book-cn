没有伤痕的「猫耳朵」


　　沪、杭的咸点心，有「猫耳朵」。

　　这是假的猫耳朵。

　　但当然有人爱吃真货。

　　记得看过报道，在江门麻园镇有好些猫肉专门店（其实深圳广州也有）。每头肥猫，遭铁索勒颈，木棍击头、滚水沐浴、脱毛屠宰。

　　猫的肉、肠、尾、脑、眼、耳、爪……，都是一道道菜式。

　　不知香炸猫耳朵尖尖翘翘，又薄又脆，像不像常吃的「千层峰」猪耳朵？

　　动物的耳朵很脏。如吃烤全羊，千万别碰这部分。――事实上，不少人的耳朵也藏垢纳污，不知可掏出多少奇怪的东西来。耳洞还有毛。

　　灵敏的猫，耳朵受少许刺激已动，在备战状态。柔顺的猫，你一爱抚它，耳朵酥软得耷拉下来，十分享受。有些猫长得恶形恶状，连耳朵也特别尖，像火星人。

　　我们点的「猫耳朵」上桌了，它是一锅汤。有虾仁、火腿粒、冬菇粒、青豆、蘑菇、蔬菜、玉米、胜瓜、笋粒、瑶柱……。基本的鲜美来自羹汤。

　　主角由面粉摊薄了，切成小块，轻轻一扭一捏，便成就了一片片小猫的白耳朵，驯服乖巧。不带伤痕。

　　也曾吃过类似的小点，有些地方只唤它「面疙瘩」，听来似一个微突的疤。杀风景。
