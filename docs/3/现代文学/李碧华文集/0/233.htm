恐坏郎君美前程


　　据说，女艺人虽与相恋多年的歌手男友分手，但对旧爱仍馀情未了。某次在一餐厅遇到娱乐圈好打不平的肥妈，含泪向她说：「希望可以帮帮（人气急跌的）他……。」云云。

　　当事人一直只对记者说：「我无讲。」任凭人家如何旁敲侧击，她都淡淡应付。半句不好听的话都没说过。

　　反而前男友一度把新欢旧爱品评。道出无奈分手「理由」。

　　她还是没有回应。人走茶凉，说甚麽也无谓。

　　——原来在背後讲了这麽一句。即使是难以求证的传言，似乎大家相信，她会为他开口向人求助的。

　　失去了爱情，蠃得了全世界又有何用？但「馀情」，便是这样。

　　你会不会为一个已分手的男人费心？

　　多数不会了。就算已经没有工夫去恨这个人，还不致伟大到设法提升他。——也许努力把自己提升还来不及呢。也许，不曾遇过「黄衫客」。

　　霍小玉花前遇侠，一曲泣诉衷情後，竟回头叮嘱豪迈仗义的黄衫客：「我伤心莫向人前说，恐坏郎君美前程。紫钗遗恨写炎凉，少提薄幸人名姓。」

　　——这一份「宽大」，我一直以为只有戏剧中才有。
