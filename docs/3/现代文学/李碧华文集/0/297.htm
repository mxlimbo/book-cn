「心思」也要靠追捧


　　一回路过湾仔，见一小店很热闹，原来卖圆圆白白的甜品，名字好「娘」，唤「雪莓娘」，又是日本小食热潮，杀过香港来。

　　「雪莓娘」像大型雪米ｘ，豆浆皮粉团，裹着低脂忌廉、甜馅（有士多啤梨、芒果、荔枝、冬栗四款可选择）、海绵蛋糕（在日本则是芝士蛋糕作底）。——以上都不是甚麽「新猷」，但有点「创意」的组合，又吸引了贪新鲜的人注目。在日本则已流行了几个月，并开设二百三十家分店。

　　有时我很佩服日本人在这些生活小节上的点缀，和那种「一窝蜂」式追捧的天真可爱。他们的格仔饼、芝士蛋糕、巨蛋面包、他妈哥池、水母、象腿袜、渔夫帽……，总有人当作呼吸的一部分，「不敢」落後於形势。

　　还有极具心思的「初恋情人侦探社」、做一百件好事便可偿心愿的「天国护照」、电子求偶感应机、减肥小拖鞋、跳舞机……，还有「贞子」热。

　　最近又有气功师展示了一幅注入了「气」的画作，肥仔肥女只要每天望着十分钟吸「气」，可以减肥，云云。为了keep fit，他们还付月费去看画。

　　忽然省起，在我的背包某一格，原来有一块「宇宙波动」增加念力的「太阳、神秘图形、水晶」三合一卡片，不知是哪年的「发明品」了。一度，在纪伊国屋书店有专柜呢。
