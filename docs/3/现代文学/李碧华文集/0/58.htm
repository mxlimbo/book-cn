长在自己身上的锁


　　绮年玉貌的女艺员失恋，遭男友抛弃。男友另结新欢，并辞职离港。她则情绪低落，厌食消瘦，体重暴跌(由125磅跌至110磅)……。

　　瘦得连颈部的锁骨也突出来了，十分明显。

　　——这是一把自困的「锁」。

　　长在自己身上的锁。告诉所有人你活得不快乐的锁。令自己食欲不振病奄奄的锁。

　　还有，是无法接受新人的锁。

　　情变当然比癌还痛，比伤还重。不管你是甚麽人，你会难过、卑微、失眠、恐慌、心在淌血。连最喜欢吃的东西也会呕吐出来。

　　尤其当「猜想」、「忖测」、「幻觉」他和身边另一位的种种时，你是「落难凤凰不如鸡」。——你对他那麽好，不计较，全付出，他却选择了别人。

　　女艺员本来还强撑去宣传新剧的，只是当在场的工作人员因为知悉而关心，纷纷劝她多吃一点时，终於感触，在人前落泪。

　　人们坚强，是因为一点定力，很多人安慰，反而令之软弱、崩溃。

　　——但大哭一场多麽好，多麽痛快。流泪比忍泪更能释放自己。狠狠地哭，像咳嗽、打喷嚏、打呵欠……，不必抑压。然後狠狠地复元。

　　颈上的「锁」，便早日砸掉了。这得靠自己。
