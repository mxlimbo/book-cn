「漏油」的烦恼


　　约两个月前，听L说，他有个朋友，服用医生开的减肥丸(每粒约售十元左右），天天早上都大便，排出一层厚厚的「油」，已瘦了若干磅，云云。

　　後来还有更奇趣的「後遗症」。

　　原来这减肥丸不但「排油」，还会「漏油」，即是身体不吸收的油，会随时自动漏出，不能控制。於是，好些服用者只好使用成人纸尿片。

　　一天看花边，前女艺人提到男人也用卫生巾「索油」的妙事。（似乎尿片覆盖范围大些。）

　　看来，要应付「油污泛滥」成风，超市得推介这两样货物，周刊及消委会亦举行不同牌子的「吸力」和「防渗漏」测试，令市民有所选择。

　　天天都有新鲜话题，——但这个，好似有点核突。

　　通常，资深基佬有失禁之虞，才会暗中使用，想不到减肥人士也来抢购。是文明的推动吗？

　　女人最烦恼的日子，每月不过几天，「漏油」则全天候，年中无休，除非停服药丸。——但，减肥是很多人的终生事业，怎能荒废？

　　说到「回春」，这便是了。因为，各人都重新受到尿片的束缚，虽「行动不便」，却回到婴儿期的天真。
