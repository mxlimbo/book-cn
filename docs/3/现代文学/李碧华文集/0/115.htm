产後抑郁马上怀孕


　　剧集大结局了，一众演员有一阵短暂的失落。尤其是主角，他道：「十分寂寞。」

　　其实任何project，在一段特定的时间内，上下一心，共同合作，朝夕相处，各尽所能，一旦完成了，新project又未开始，也不知是否同样的一班人，总有一点说不出来的「空」。

　　承诺实践了，心事了结了，竟怅怅落空。——除了无法达成心愿之外，就数心愿达成了最伤感。

　　但这是无法避免的。除非没有「心」，或是一早抽「身」而退。

　　人一生中，便是由这两种程度不同的伤感相间着。创作人经验最丰富了，因为工作是一个一个排列的，有时重叠，有时中间的空白很长很长，不过「最後一天」并不快乐。不知其他人如何？我完成了一个近二十万字的长篇後，惯性产後抑郁兼失调。

　　先不说作品好坏，但怀胎十月，他吸尽心血，然後离开了。他有自己的命运和发展，我只馀一个掏空了的子宫。多麽的难过。

　　——但这有甚麽大不了？都会过去。

　　每个人都为他自己的事忙碌着。

　　曾搁置了一个写作计划。为此老板体谅地放了我一年假。今天催：「出书了吧？好，马上开始新大纲。」又得收拾心情。

　　没有时间抑郁。因为生活迫人。
