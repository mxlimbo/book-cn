没道理、没原因、没解释


　　一般二十三岁少女，谈恋爱，简单而又甜蜜，二人世界，干卿底事？

　　不知如何，GiGi那麽「惨」。又狐狸精又大话精又送上门又末路又淡出……。

　　又不是作奸犯科，没有天天在传媒面前「伤害他人身心」，完唱。——是传媒天天在「干预」。读者观众们再不喜欢，也不能忘情投入人家的恋爱中吧，难道大嘘便可以拆散？

　　交代得含糊，处理好失败，公开太拖泥带水，之类，又如何？到底是私事。

　　原来做人，自己开心快乐，是不容易的。易地而处，你要「红」？抑或要「爱」？你「好胜」？抑或「低调」？

　　记得有位朋友，她虽不是艺人，但在办公室也是一强人，爱上了条件不及但浓情蜜意的男人，还比她小几岁。别人都在背後非议，她不在乎：

　　「我又不是同他们拍拖。我的快乐他们不明白。」

　　又道：

　　「即使年龄、爱好、地位、条件、timing……，全部相配合，也不一定永恒不变。既然如此，为甚麽不先享受了目前一刻？」

　　GiGi说：「爱一个人是可以没道理没原因的。」——她这样说了，大家还穷追猛打吗？

　　爱没得解释。反而，恨有千般因由。
