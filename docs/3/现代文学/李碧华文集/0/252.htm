永垂不朽的眉


　　奇怪，现在还有女人去纹眉和绣眉吗？

　　我以为十年前才会流行呢。——而十年後，她们都後悔了。

　　男人帮女人画眉是调情，女人自己画眉是美容，但好处是，画得不好，擦掉再画，只会愈来愈美。

　　纹眉？一锤定音，一箭定江山，乃终生烙印。即使今天纹得还不错，但明天、明年、一生，全天候都是这个样儿，变不了。手势再好，也平板呆滞，了无生气。

　　你看那些成群到深圳罗湖商业城或东门市场shopping的肥师奶，十个有三个是纹眉的，物以类聚，纹过的眉：——（一）呆、（二）假、（三）凶、（四）淡。

　　像一些标签。用粗粗的指头「按捺」上去一样。

　　日子久了，均变成褐色，不化妆，仍剩两道眉永垂不朽。整张脸好生硬。眉毛剃掉再逐针色，意义等於面具吧，呼吸都有困难。

　　若自己也看不顺眼了，後悔了，好麻烦，要「洗」。

　　在相同的地方再下针，以药水漂淡，用激光消除，不能一次成功。据说，有些还要做上一两年，才勉强把「凶眉」略为改善。

　　染发会变猪头，纹眉会变炭头，脂粉浓艳又生黑头，——女人好「色」变「色」，不免付出代价。却前仆後继。
