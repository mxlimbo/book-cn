江湖再见


　　年前报业大战淘汰赛，很多同业一夜之间失去了工作。谁知这个夏天，互联网公司也纷纷裁员，他们之中有些是朋友，有些还有点工作上的联系。到了八月某个黄昏，连「博学堂」这香港首间华人网上书店，也自动清盘。

　　他们为本城很多写作人制作网页，分工很精细。负责像我这样对电脑不精且懒散的作者，是编辑部阿Ben。——好不冤枉，近月他们花心思帮我网页设计更新（全部用饮品做栏目），才run了三天，便成梦幻泡影……。世事无常。

　　Ben在和同事们往返劳工处与公司收拾细软之间，还抽空为我留一份file。说：「要趁一切还没『消失』之前快快做了。」我很谢谢他。

　　但一切总会「shut down」的。

　　最近才加入失业大军的一批IT人，都是很fresh很有冲劲的精英分子。裁员或清盘，是经济问题，决非他们的工作能力问题。

　　我很想在空气中送给失意者一个「烘面包」的鼓励。这是数月前在日本（现已登陆香港且大热）见到的人气公仔。它本来是一天限量发售二十个的高贵甜蜜红豆包，可惜被人烤焦了。无奈又消沉。

　　但这不是自己的错呀。终有一天拍去灰尘碎屑，振作起来，重新烤制一个一个更新鲜美味的面包。向前看。

　　江湖再见。
