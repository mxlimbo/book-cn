第一件事是换锁


　　一个师奶型职业女性向另一个师奶型职业女性诉苦。她租住一个单位後某天，忽然有人用钥匙开大门，她大吃一惊，原来是经纪。大概因业主放盘时，把钥匙分配给几家地产公司。租出後又没有知会及收回。经纪不知，所以带人客上来看楼。都在那儿抱怨，说业主不对。

　　旁观两位，既出来打工，见过世面，又是师奶，理应精明持家。这种事能怪人？怎麽那麽笨，吝啬一项基本动作？租住单位，第一件事是换锁。正如业主收楼後，第一件事也是换锁一样。

　　我们到了任何陌生环境，先了解进出口，还有保护自己不受外间侵扰。同任何人分开，不打算有瓜葛牵连，除了搬，最方便是一个只有自己才可打开的锁，一把只有自己才保存的钥匙。任何密码，即时更改。这样并非完全没後顾之忧，但起码把危险减低。

　　前一阵娱乐版报道艺人与业主的纠纷。她发觉对方竟可带人到屋内指手画脚。原来一直有她家的钥匙，所以好惊。云云。不知道前因後果，只就表面来看，奇怪？有没有换锁？若有，则擅闯民居者便居心叵测，太可怕；若入住後沿用旧匙？是自己责任。

　　世上可信的人并不太多。小心、警觉、落闸、上锁，是「自保」。这些都做足了，仍欠安全，便属不幸。
