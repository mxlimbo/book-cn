血管大暴露


　　春夏季流行一款铺满「血管」的名牌波鞋。血管有黑、蓝、黄、绿等色，如蛛网般，攀布鞋面，好有趣。

　　球鞋，其实即是时髦便服鞋。

　　很多穿球鞋的人不懂踢球，正如很多穿运动服的人不运动一样。而「运动服日」的贡献是募捐。

　　这款血管球鞋的面世，也是一个「透明度极高」的标志。原本隐藏在血肉脂肪底下的，神秘的血管，竟然暴露得巨细不遗。

　　而血管的分布，则象徵着Web的征服力，无处可逃。延伸得理直气壮。

　　於是，甚麽质料、是否有气孔、气垫、凹槽……，都是次要了，绝非消费者目光焦点。

　　展示完血管之後，不知下季还有甚麽要「揭发」？

　　我同朋友说笑：

　　「应该设计些透明T恤，在三点要害，就用『格仔』作为装饰花样，这样人们穿着上街，看似裸体，但『自律』地打格仔，多麽诱惑。——见报时也不劳编辑部加工，原装可以登出来。」

　　在香港，血管球鞋并非间间有货，还没到。日本是先行者。

　　但不要紧，——

　　很快，深圳就会出现翻版。到时可能甚麽颜色的血也有供应，如同廉价血库。
