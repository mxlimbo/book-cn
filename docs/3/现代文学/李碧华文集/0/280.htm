恩将仇报的人


　　法官判一名犯案的男子入狱十年半。──他犯的都是盗窃、抢劫罪，何以如此重判？

　　是因为「恩将仇报」。

　　一回，他凌晨潜入一单位行劫，事主的丈夫把他制服，他苦苦哀求放一条生路，事主夫妇一念之仁，只记下其身分证号码後，便让他离去。谁知数月後他再次闯入，以刀指吓女事主，还将她绑起，用床单塞嘴，凶相毕露地拳打掌掴。事主丈夫刚回家，又制服之，这次立即报警查办。

　　虽然他没得手，但这样对待有恩於己的人，十分卑鄙。

　　我们憎恶对不起自己的人，但特别痛恨那些当初对他很好，在他落难时借过钱，需要援手时给予协助，甚至护短，把他看作是朋友的人。一头狗饱食後回过头来咬你一口，你认命，因那是畜生，你信任一头狗是自己蠢。但恩将仇报？无论如何，总有点不甘心。──遇上这种人，他还会令你受过教训後对长得相像的都有点戒心，也算功德无量。
