Cashmere和Cashmore


　　今年秋冬流行灰色，市面充斥深深浅浅的灰。基本动作是先买一双灰色的鞋子。这天政府公　的最新失业率数字又创新高，此时必备一双鞋，方便「走佬」，真是「诙谐」。心灰意冷？不，仍盼「死灰复燃」。

　　找几件秋着出来时，发觉前年新买穿不了几次的灰色茄士咩羊毛衣，正合timing。──灰和黑，是永远不会过时的颜色。而茄士咩羊毛衣。虽略贵，却不管甚麽年代，可循环再穿。纤维细密，又轻又薄又暖，暖意就保存得长久，且一点也不臃肿。

　　名牌的小手袋，也用上茄士咩。炭灰色，配银扣银带。拿着它，寂寞的富女富婆，像握着一只手。可惜它没有生命，也不会自行送暖。当她们珍重而怜悯地抱之入怀，体积又太小，并且盛的仍是个人私己。──有些女人情愿要一只真正的男人的大手。即使她买不起这款接近五位数的手袋。

　　茄士咩是克什米尔山羊的毛，是严寒地区动物天赋自保的绒毛。英文名称是Cashmere。

　　今时今日，我们除了在莫测的冬季，身穿Cashmere保暖，同时，还希望拥有它的姊妹：──Cashmore。才觉得安全。
