道德重整、家庭计划


　　报道一切绯闻，传媒还兼职「道德重整」以及「家庭计划指导」，多麽忙碌！

　　圈中人胆战心惊，不知哪一天会见报，上头条。所爆的料大部分是男女关系。l骨寻欢、拥吻缠绵已是指定动作，还有谁同谁「终於肯拖手拍照」、谁同谁之间有第三者、谁未婚有孕、谁已婚有私生子、谁一脚踏X船……，都是人家私生活。男女之间，一个愿打一个愿捱。两个人走在一起，十分微妙。分开了，可以说是前生债务今世清还，更不知究竟谁欠谁呢？老实说，又关局外者甚麽事？

　　而且有关「感情」，不可理喻，亦不可置评。疯狂追逐日夜守候，浪费了记者青春精力。但传媒观念保守，大惊小怪，小事化大。单身艺人拖手仔，一众哗然，还可上港闻版，难道我们回到男女授受不亲的封建社会？

　　当然，大家也不必为艺人难受，作不平鸣，他们有心理和生理准备。而且不一定全部都痛苦不堪，一半一半啦，有些是布好局，做足「辅助动作」热好身，等狗仔队上钓的，你们不来，他们才失望呢。

　　只是某些明明仍可挽回的关系，有时被高调公开，欠下台阶，因面子而无转圜馀地，还影响小孩，相当可惜。――做完一单，又开始第二单了，他们怎会回头为当事人设想？
