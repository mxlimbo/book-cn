给我一个好的excuse


　　立法会宣传活动中，小歌星被大会司仪问及有关立法会选举的简单问题，她答不出（2000年）。理由：「太早起身，所以未清醒。」

　　壮汉袭警抢警枪，被捕後道：「鬼上身。」

　　港大副校长对「锺庭耀事件」的发言是：「唔记得　清楚。可能有。记忆好唔准确。」

　　港大校长对同一事件的解释是：「刚刚得悉。极感震惊。（有没有施压或授权施压？）要翻查纪录才知。」

　　明星涉嫌偷拍裙底春光。他说：「我饮醉酒。一场误会。」

　　名女人被指「有特殊药瘾」，自辩：「我贪玩。食又得，唔食又得。」

　　甲追乙还钱。乙失忆：「有吗？我记不起了。」

　　女人质问男人为甚麽变心。男人说：「你太好。我配不起你。」

　　一个人很久没同另一个人联络了，支吾：「我找过几次，找不到呀！又不知你是否在香港……」（几多次？甚麽时候？真有找过吗？）

　　——任何人做任何事，都有他的理由，或者「藉口」。听了你也不相信，没有意义的字句。

　　「给我一个好的excuse！」再好，是双方易於落台吧。
