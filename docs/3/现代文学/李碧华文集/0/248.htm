小小的红色圈套


　　昨天抄《紫钗记》歌词，有「赤绳长系足，那得再图赖？」一句。

　　记得有男人谈到他初识的女友，道：「不知如何，有一种似曾相识，亲人的感觉。我想我是逃不掉的。」

　　现代人都认为是缘分、过电、有feel。「赤绳」典故出自《续玄怪录》：――

　　元和二年，韦固离家旅行，住宋城一家旅店。晚上，见一位老人在月下翻检书信，布袋中还装很多红绳子。韦固问：「这些红绳有甚麽用处？」

　　老人答：「是用来绑夫妻的脚的。即使仇人之家，贵贱悬隔，远在天边，身处异地，只要系上了，就无论如何逃不掉。――你的脚也系上了，再找别的姑娘也不行了。」

　　很想知道将来的妻子是谁？老人带他到旅店北边，卖菜陈老婆婆家，她正抱一个三岁女孩。――一看，满脸鼻涕，哭哭啼啼，丑陋无比。他不觉大怒。随後，还派手下人用刀把女孩杀死……。

　　後来，韦固已得功名，但多次求婚，姻缘没结果。很多年过去了，才娶得一个年纪相差甚远的少妻，她眉间有一刀疤。追问之下，才知三岁那年，无端被人刺伤，但没中要害，被人冲散了。愈後留下疤痕。

　　是的，他没有能力逃出这个小小的红色圈套。――或者，世上所有人也一样。
