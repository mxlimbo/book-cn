雄蚊的叫春声频


　　不是「讨厌」蚊子，――是「憎恨」它们！

　　为甚麽世上有一种如此微小，只消「一只」已令人怒火中烧难以自控的大敌？你们针刺、吸血、遗留病毒，叫人又痒又痛……，那也算了，何以非要嗡嗡怪叫吵醒人，然後带着嘲弄的胜利微笑来瞅着拿它没办法的万物之灵？

　　请大家告诉我，蚊香、电子蚊香、蚊怕水、电子驱蚊发声器……，哪一样最有效？

　　上回我发现了这款使用电池，二十四小时，六至八尺范围之内，发出「蜻蜓翼拍」或「雄蚊翼拍」频率的发声器。最近，有朋友看到新产品消息，告知有「驱蚊钟表」。可以放在床头。窍门也是内置了雄蚊声频。用以驱赶雌蚊。

　　是雄性的威猛吗？不，与我理解不同的，是他们表示：「吸人血的都是雌蚊，很多时已有孕在身，所以很怕雄蚊来打扰。听到这些声频便跑掉。」

　　原来「神秘面纱」不是雄蚊够恶，只是有馅的雌蚊拒绝性骚扰。走夹唔着。

　　男人追，女人逃，半推半就，欲拒还迎，可能更加滋扰。

　　男人求欢，但女人为了腹中块肉狂奔，你说是否好「母性」？

　　「性」是男人攻击女人的手段。也是雌蚊暂且放过了甜美消夜的无奈。想不到人类一觉好睡，要靠雄蚊的麻甩叫春声频。
