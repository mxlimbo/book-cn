没有比这更伤心了


　　看到日本国宝，人瑞孪生姊妹中，一○七岁的金婆婆逝世消息，觉得有点难过。——我是为银婆婆难过。

　　一○七岁已是笑丧，而且金婆婆是在梦中辞世的，她患上感冒，连日来时睡时醒，说没有胃口吃早饭，家人再去看她时无反应，原来已长眠了。

　　去得安详毫不痛苦，睡着睡着，就不愿醒来，还有家人照料，多麽快乐呢。

　　——只是，她先行一步，银婆婆再坚强、乐观、机智、笑容满面，再积极生活，已经不大容易了。

　　她俩没想过可以活得这麽久，活着，因孪生姊妹，心灵相通，互相扶持、依赖。她们需要对方，甚於世上任何一人。一个走了，留下一个，太寂寞了，就像把身体砍掉一半。银婆婆低头流泪：「没有比这更伤心了。」她会很快地，找她去吗？

　　有时人活着，不是为了自己，是为了另外一个人。——你最爱的人，你最恨的人，你最亲的人。能有这样的「伴」，是很重的缘分。

　　两个人，有「情深缘深」的，金银两婆婆，和很多白头偕老的爱侣，都不枉此生。也有「情深缘浅」，好事多磨。「情浅缘深」，注定不能拆散，淡淡一生也就过去。「情浅缘浅」，男女难有突破进展，兄弟姊妹虽上天配给，但也变成陌路。
