江湖阿姐的玉腿


　　在一家茶餐厅，见两个「出来行」的女人撑着脚。

　　我自着底观赏她们的脚。

　　这些被称「X姐」或「X家」的女人，她们的玉腿是有模式的。

　　不知如何，十分喜欢穿窄脚裤，还是三个骨。颜色相当「斑斓」、瑰丽、闪。起码具备五至七种色。或是一些斑马、斑点狗、豹、虎之类的兽纹。露出一截小腿，间中有青绿的筋脉。戴一条脚链。趾甲呈尖锐的杏形，甲油不是紫、蓝，便是深红。踏在厚底凉鞋上。

　　如果翘起一条腿，那凉鞋便吊在脚上晃。如果没有翘起，通常便分张大腿而坐。――然後提起一脚踩熄刚扔下的烟头。

　　别看她们玉腿七彩纷陈，其实不大乾净。脚皮厚，尾趾结了些茧。

　　这样粗糙市井，才显出风尘味，以及行走江湖，不靠男人而挣一口乾饭一杯香浓奶茶一笔养老金，有多麽吃力。还得虚张架势。

　　最近看电视上有反盗版的广告宣传片，各式各样的罪犯，因阁下购买盗版而赚个盆满钵满身光颈靓，向大家好寸地道谢。其中之一是位阿姐，用眼角瞅着你，讲句江湖味十足的「多谢」，是又「拆」又「沙」的鹅公喉，真是神似形似。英姿勃发。

　　她的下半身，是否穿了上述「制服」？是否有上述甫士？
