中间那一阵短暂的迟疑


　　好些被强行力捧的年轻艺人、歌手，不过长得好看些，但「无料」。演戏不怎麽样，唱歌也欠造诣，个人没甚麽特色、性格、内涵、应对技巧、表达能力……，不过由一群幕後策划者，包装成悦目商品上市。

　　有的比较幸运，竟然成为「偶像」，有的不幸，起步不久即滑落。

　　年中不少艺人、歌手，是无声无息地，变回一个普通人。

　　他／她们失宠、迁出豪宅、身边再也没有三五七名公关保母簇拥、记者的镜头迅速跳过去另觅焦点、去看表演亦静悄悄入场坐侧边位置、转业、离港、到大陆登台耗尽剩馀的名气……。

　　易上易落。快上快落。

　　那些捧「红」他／她们的老板呢？另携爱将出来见客。

　　老板没有表情。但一一均有「嘴脸」。

　　你可以与大机构发生关系，但千万不要与之发生感情。在商言商，老板们（任何一行）变脸最快（华仔不必远赴四川学变脸了）。在一个商业社会，人与人之间，只有过，没有功。只有利，没有情。

　　——不，也有些「有人情味」的呀。

　　你误会了。所谓「人情味」，不过是他应当对付你，却还没有下手，中间那一阵短暂的迟疑。
