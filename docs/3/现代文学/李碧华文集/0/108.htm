世上确有伤心地


　　（一）长城。（二）桂林山水。（三）杭州西湖。（四）安徽黄山。（五）江西庐山。（六）长江三峡。（七）苏州园林。（八）江南三大名楼之一——滕王阁。（九）深圳世界之窗。（十）湖南张家界。

　　这些地方我差不多都到过，——除了（九）和（十）。那深圳世界之窗是送钱给我也不肯去的了。

　　何以表列？

　　原来是十大不想重游的「伤心地」。

　　中国大陆的劳动节长假期由五月一日至七日，当局估计出外旅游者高达一亿人次。广州《新周刊》一项调查：一度魂牵梦萦的祖国山水名胜，皆令人失望。

　　山水名胜，都有特别风貌，值得一游。我还记得桂林的马肉米粉、长城雪景、黄山烟雨、苏州园林的惊艳感……。这些地方，不知变得怎麽样？是各界的评语：人山人海、颓垣败瓦、水质污染、厕所脏臭、骗财陷阱、复制造假……吗？

　　或有叫人不想重游（甚至不想重提）的地方。但再脏再乱再不堪，如果你同喜欢的人去，心情好，不会在乎。

　　——只因在某处，曾有某人，或发生某事，叫你伤心。事过情迁，重游更添愁和恨，你避开它忘记它，连路过也不愿……。

　　世上确有「伤心地」。
