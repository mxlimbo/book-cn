无情无义，无牵无挂


　　一位在情场打滚数十年的大侠，一直有不少女人投怀送抱。他是老手，年纪稍大了，仍有「铁布衫」：――

　　向对方讲清讲楚，没有结果的。

　　「要无牵无挂，就要无情无义。」

　　早早说了，仍然肯？便是心甘情愿。成年女人向自己行为负责，承担後果，哪有被迫？――连「挟持小孩以达目的」之类招数也不会灵光吧。

　　这种心得，必是游戏人间，玩得炉火纯青的领悟。可见十分奏效。而且不会产生伤害的毒素，间接造福人群。

　　一个人为甚麽会牵挂？当然是动了真情，担了道义。

　　牵挂是痛楚的，明昧不定的，在午夜醒来便再也睡不着的。虽然心细胞不会增生，所以从不患癌，但心细胞死一个算一个，脆弱的人经不起几回创伤。

　　除了和尚，便数情场老手最「禅」了。

　　――不过，无情无义似乎说来轻松，实践又不那麽容易。而且，无情无义的交往，根本不会引发跌宕的惊喜和悲哀，――这种过程才最诱人，才最令人心跳吧。

　　「无」，便无牵挂。但也无哀乐了。

　　好的恋爱，你会为了对方而忘我。坏的恋爱，原来也是忘我，强迫自己不存在。――为甚麽竟是一样的原理？
