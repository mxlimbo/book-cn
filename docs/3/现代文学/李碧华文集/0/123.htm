荔枝对你的嘲笑


　　有时，人们会懊悔，当初没有尽情，尽兴，时间又过去了。

　　虽然，去得再尽，结果还是一样的。再抵死缠绵，全抛一片心，终亦归於平淡。——只是机会不回头了。

　　今年的荔枝因严寒冻伤，以及一年丰收一年歉收的「天命」下，它们不但涨价，还肯定不够香甜。你们知道了，有否懊悔去年没有「狠狠」爱它一场？

　　记得九九年六七月的荔枝吗？

　　它们反常地任性地狂放地，鲜甜美味，芳香丰腴，无穷无尽地供应，大家为所欲为地享受。又便宜，又好吃。

　　「一颗荔枝三把火」，所以有人吃得口鼻喷血，中了荔枝毒。受新闻惊吓者，马上止步。那麽的自保、克制、健康……，却在抬头之际，方悉大势已去。一想到此，你也许情愿在过去的日子，中中毒，喷喷血，不枉风光。但盛况已成为回忆。情迷意乱，难以自拔，好歹悲壮过。——这是荔枝对你的嘲笑吧？

　　欧阳修说：

　　「直须看尽洛城花，
　　始共东风容易别。」

　　把洛城花看得那麽深刻和深沉，其实更不容易说「再见」。不过现实是：尽不尽，也得分别。

　　在「小年」、「小造」之际，总觉一阵苍凉。错失了的，就是错失了。
