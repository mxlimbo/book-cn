梦之浮桥失踪了


　　世间所有米酒，最最最重要的，是米，和水。

　　把纯米精研，选取最好部分，配上清甜川流，酿制过程不外蒸米、散热、发酵、上槽、定时、试饮、入瓶。米不纯，水不正，清酒又怎会「清」？共产党严抓的「出身」、「成分」论，对人不公平，但酿酒必须查血统。

　　我喜欢逛酒场，叁观酒库。还抄酒名。

　　酒的名字，有时因产地、河川、天气、酒场主人……而改，有时是意境。以下是找得到的：——

　　醉心、山吹色、田苑、天河、罗生门、珠王、甘雨、飞之鬼、天狗舞、梅锦、绫菊、春香彩、瑞秀、加贺鸢、水青、水芭蕉、凤水人、红乙女、玄海、司牡丹、和乐互尊、上善如水、天草四郎、七笑、五百万石、小鼓、天山游花、不思议……。

　　而这些：「米泉、美少年、都美人薄绢、花风月、烂漫、热恋之呗、梦十夜、酒魂、幻、失乐园、隐藏、无」——是否更像小说情节？

　　日本酒分了等级：大吟酿、吟酿、本酿造、清酒……。会喝的人当然分得出。但若时间地点心情，还有：人，都很快乐。清酒也就等於大吟酿了。

　　其实我最喜欢从前偶喝了一杯「梦之浮桥」，但一直再找不到，连在酒库也失踪。是太冷门？抑或已热卖掉？

　　抑或我记错了？
