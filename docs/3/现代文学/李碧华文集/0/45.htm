最口密的瓶


　　大部分的纸都包不住火。——但如果坚决不说，又真的可以「地老天荒」。即使不致於带进坟墓那麽深沉，一些秘密的埋藏期亦相当长。

　　前一阵有艺员出事，被廉记带去饮咖啡。共事数载，朝夕相见的同僚，竟不知她已经结婚多年。

　　女歌手为了爱情放弃事业，拍拖未几即同居。三四年後才发觉男人变心，他俩计划结婚了，原来新欢已识尽他周围的朋友，自己是最後「知道」的一个，——甚至没有见过。

　　有人对她的恋情一直秘而不宣，私生活没甚麽人知悉。原来「他」那麽威猛。

　　「私生活」，当事人不泄不露，人家不会知；略知，不认，又如何？毕竟不是真人show。

　　又，如果有些事当事人不想知、不要知，也就不会「见到」了。见了，装作不见，等於没发生，把自己欺骗到底。

　　这一种本能，也是与生俱来的自我保护。世上真有「守口如瓶」外加瓶塞胶纸火漆密封。特别是感情。

　　但坊间不是有很多揭秘的报道吗？3被出卖，3露出破绽，3是当事人有意无意让人知道的。可以见报、宣传、互相利用、明白游戏规则……。这世界，「无鬼唔死得人」。
