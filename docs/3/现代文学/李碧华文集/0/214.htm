血泪中无声叹喟


　　昨天写过：「我们受尽折磨，才有今天的强悍。」

　　今天想想，其实我们受过甚麽「折磨」呢？而「强悍」，也是一个过程，够不上深沉的回应。

　　历炼比较肤浅，故而虚张声势而已。

　　因提到杨绛的小说，今天再写写她（和整整一代知识分子）的感受。

　　杨绛是钱锺书先生的夫人。两位并列，一双气质高雅、温文慈祥的璧人。「书卷气」不因一场浩劫、知识分子群体接受「脱裤子，割尾巴」改造而变质。

　　心灵所受到最巨大而悲凉的伤害，可以从杨绛的《洗澡》、《干校六记》、《从「掺沙子」到「流亡」》等小说和纪实散文中看到。

　　最近重看她的作品，原来她写来十分平静、冷峻、嘴角带着似有还无的浅笑似地，不加一丝控诉评议，当中却有千言万语。真是高手。

　　而我们无法做到，正因为我们没有经历。

　　以为失意、跌倒、失恋、贫困、被出卖、生离死别……，是很大的痛苦，但我们的灵魂不曾遭到大时代莫须有的屈辱，天天渗着血泪，生命如同破屋中一张千万人踩踏的旧地毯，放不下，提不起。还祸及无辜的下一代。

　　化为文字，血泪中才有无声叹喟……
