没有「回忆」，可以放手


　　有否得空清理衣柜？——只要彻底清理一趟，谁都会把预算换季的项目减半，原来有很多衣物我们已经「忘记」了。有些只穿过一两次便搁置，有些还是被冷落的後宫佳丽。

　　为甚麽竟可忘掉？当初买的时候不因为「喜欢」吗？喜欢得不够，或不长久，可见是逞一时之快。如何处理其後事？

　　见社区服务团体，有「旧衣回收大行动」，於指定日期时间地点，收集大家清洁的旧衣物，将分发运送往第三世界贫穷国家，循环再用。

　　响应的人真不少，堆放一起如小山丘。点收运送的都是义工吧，在圣诞前後都花了整日时间。而我发觉，说是「旧衣」，很多仍新净，旧的只是款式而已。厚暖的冬衣，一定叫需要的人受惠。

　　本城街头巷尾也有不少贫寒的人需要送暖，路有冻死骨，得靠热心人士寻找。我们没有这种奔走的精神。

　　不要的东西有落，知道它们有用，自己「奢侈浪费」的罪孽似乎减轻了一点。

　　某些「旧衣」不同「旧人」。我们总有最心爱的几件旧衣，永远把它收藏好，再陈旧再古老，也舍不得放手。而旧人，即使在眼前，心已不在，几乎认不出来便擦肩而过。

　　是的，可以送出去的，都是没有「回忆」的东西。
