拆卸情墙也一样


　　两名工人，为一间布厂内部装修「进行拆墙」工程。他们挥动大铁槌砸毁墙顶部分砖块。加班时竟改变方位，向墙脚埋手，数分钟，整幅砖墙塌下，二人走避不及，被压在瓦砾下，大喊救命。幸送院後仅受轻伤，二人频呼好彩。

　　令人骇笑的「好彩」。而传媒用「乌龙」来形容，没有嘲弄，更是宽厚。

　　我们从来没听过，四五十岁资深的装修工人，拆墙是由「墙脚」拆起的。怎麽可能？小孩玩积木，或层层叠游戏，都知道最底层的，是建筑的根基命脉，最重要。它们移动不稳，便危险了。这些拆了半生砖墙的叔叔，竟然那麽「反叛」吗？

　　——把砖墙拆掉，也像把一段关系结束。

　　男女之间，密切坚牢的关系，本来也是一块一块一块砖头合力砌好的一道墙，万一要另外装修、改建、迁移……，总之令之消失，其实是分手艺术，也是拆卸功力。

　　慢慢拆，从上而下，渐渐适应，不中要害，不伤死穴，它夷平了，也许惆怅惋惜，亦很无奈，但不会压死人。一下子「釜底抽薪」，墙脚砸起，是用错方法使错力，也是一种最笨最险恶的手段。任何具基本常识的人也明白，砖块迎头塌下，两个人都不能幸免，必会愤怒、受伤、痛苦，还祸及无辜的旁人。

　　建设和破坏，都有它的游戏规则。
