陆离的作风


　　因为争取一线机会，为陈伯与小猴「金鹰」而努力的其中一分子陆离，近日统筹各报专栏文章，所以向作者一一知会：「剪报会作请愿用。」

　　虽然有时她会很长气，又好劳气（Sorry，石琪请谅！），不过陆离对署名文章的处理，是十分周到的。她认为必须得到作者同意。

　　这点，与她当年当过总编辑的关系不大吧，是一种个人的修养。

　　我们既然可以公开表态，便不介意被用作请愿或抗议。若时间太急迫，不来「电」知会也无所谓。她是「陆离」，又不是其他的甚麽人。

　　我也有不愉快的经验：稿子让一些没甚麽交往的人一字不易的搬到电影戏场上；小说被粗劣翻版；专栏和书名也有人抄；――而自己是最後知道的一个。

　　世上各式各样的人都有。因而更显「尊重」的珍贵。

　　陆离还向我提及，前一阵《矿泉水》中写过一位女孩，香港人，长驻中国山东工作，来信问一本书的资料。回邮太慢了，所以我传真过去。她因而得知我的号码，但她表示：未得允许，怕引起不便，或过程中无意公开了，所以还是回信致谢。――她对人家私隐，也有份久违的「尊重」。陆离说：「这个女孩真可爱！」

　　「不止」可爱，真的。
