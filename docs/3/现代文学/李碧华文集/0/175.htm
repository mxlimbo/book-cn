当年的「连容骚」小姐


　　那一天，在超级市场，忽见一位头发有点花白的师奶，挽着两大胶袋的杂物，很眼熟。想了想，啊，原来她是当年粤语陈片中的茄喱啡，没有名字，但经常出现。在陈宝珠萧芳芳的电影中，是工厂妹群众、派对中大跳阿哥哥的坏学生、飞女、好姊妹、村姑甲乙丙、妹仔丁戊己……。我见过她做戏。

　　还冲口而出：「我见过你做戏。」

　　她有点惊讶，一时之间笑不拢嘴。没想到有人给认出来。有点腼腆。後来某夜，又在一出陈片中见到绮年玉貌的她了。

　　这戏唤《乌龙王发达记》，是丑生王梁醒波担纲的。合演都一时之盛，各具特色，罗艳卿、李香琴、郑君绵、邓寄尘、谭倩红……，还有我好锺意的，那风情万种酣歌热舞恶死能登的谭兰卿。

　　戏中有一个「连容骚」小姐选美大会。位位36、22、36，泳衣出场，搔首弄姿，全部熟口熟面的茄喱啡上场，牺牲色相。这位安详而平淡的师奶，也是佳丽之一。

　　看真点，原来上台的还有罗兰姐，身材甚正。而在龙年，罗兰姐「龙婆效应」，亦当上了影后。是一份略为迟到的安慰奖，但有便是好。主角们各有各的际遇，配角们也各有各的命运。
