把演员腌出味道来


　　有人说姜文的新作《鬼子来了》十分焦虑和沉闷，看得辗转反侧；但也有人说这个戏由人性出发，颇有味道。

　　我想看看这个戏，不因为它刚得了康城的电影节大奖，而是「人性出发」。好的小说，好的戏剧，最重要的是人物、人物、人物。——有血有肉，艺术的骗局变得真实。

　　也许姜文自己演戏时特别注重钻研角色性格，所以也同样要求他的演员。

　　在一个访问中，他认为「谁都可以当演员，只要你给他合适的角色」，故「适合」和「素质」是他选择的标准。并要他们实在地生活，扔到农村去，养猪杀鸡，每人派一个唐山人，天天学习唐山话，五六个月下来，几乎没成了农民。

　　演日军的日本人派到部队接受军训，培养「杀气」，勾起中日仇恨。故事以人的「心」，人的矛盾作主线。

　　於是这些演员，就像腌咸菜一样，被放进酱缸里，一段日子，腌出味来了，等要拍摄的时候，再把咸菜捞起，切片、切丝，都有味道，都好吃。

　　作为导演、编剧，如果演员肯「受腌」，多令人感动，但这样的一个戏，非一年半载不行。有没有这样的演员？有没有这样的时间？——要双方都「肯」？愿打愿捱？难过恋爱。
