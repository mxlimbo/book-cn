只要一点安全感


　　不知道禁止司机驾驶时使用手提电话的规例是否正式生效。本人非常讨厌司机一边开车一边煲电话粥，一心二用。议员们质疑规例有灰色地带。我也认为应清晰界定：——

　　（一）身分：司机。

　　（二）地点：交通工具在道路上。

　　（三）动作：驾驶中。

　　（四）禁止：使用手提电话。

　　把车停泊在路旁讲电话，当然不算「驾驶中」。也不在「道路上」。

　　司机把手提电话放在大腿上通话、用手指按开关、透过免提的麦克风通话、声控……，以上统统在「使用」中。

　　立例，当然为大众的安全和利益着想。把一两句中文组织得密不透风，没有漏洞，竟然不容易？

　　交通意外原因复杂，但近年因司机使用手提电话不专心驾驶的个案日多。

　　坐的士，那司机一边「喁喁细语」嬉笑，毫无职业道德，——与手术进行中的医生犯同等错误。何况还迫乘客「聆听」他的废话，污染空间？

　　一回朋友一边开车一边答覆法律谘询。我说：「你能不能停下来专心地用脑呢？」

　　见过车祸的图片，手提电话在血泊中，似乎仍启动着……。——不管「女人」抑或「乘客」，是的，都追求一点安全感而已。
