一袋垃圾也有梦


　　宽频广告以war game为主题的「西环快枪手」，好盏鬼。主角年轻、威猛，以一敌众，「上天下海」般大开杀戒，是梦幻英雄。甚至因「唔到对手」而非常之寸。

　　——梦醒了，他却得死死气去倒垃圾。

　　在现实中，很多很多二十岁出头的大男孩，都会被他老板、长辈、家姐、女友、阿妈……点去倒垃圾。「雄风尽失」。

　　如果是家人爱侣，那还算了。但并非每个人都能如意，可以进修或找到份好工。芸芸众生，他们的工作，是清洁、地盘、搬运、装修、售货、楼面、厨房、烧腊、派传单、送外卖、贴街招、送报纸、水吧、信差、洗头……。当然也包括倒那永远倒不完的垃圾。

　　这就是生活。

　　记得某次看过一篇访问，是一个市面上最常见的普通二十二岁男孩。打过很多份工，「通通都系耷低头做。」——只有在深宵，跑到的士高廉价酒吧，可以狂舞，可以「呼吸」，又上到台，有人赞，有掌声，才算找到自己的存在，有机会讲几句话。

　　一个失败者，一笃屎，一袋垃圾……，他们也希望在某特定空间，得以「抬起头来」呀。虽然梦会醒。

　　虽然「西环快枪手」也要倒垃圾。

　　这个广告是笑谑，不过，实际上有点悲哀。
