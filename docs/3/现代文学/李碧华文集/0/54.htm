满怀甜蜜的巴马火腿


　　M形容一个女人：

　　「肤色黝黑，看上去又咸又乾又瘦，——就像一片金华火腿。」

　　他不说「火腿」，而说「金华火腿」，因市面上可买到的舶来品，颜色比较娇俏，鲜粉红色，肥瘦大小调味适中，还可即食。不怎麽刻薄。出於祖国金华的火腿，盐腌、烟熏、风乾，所以很深沉。

　　但不要侮辱了真正的火腿。

　　高贵的火腿，非但价钱不便宜，还不一定有货。瘦肉红如火艳，肥肉白似透明，切成不规则薄片，其香扑鼻，可以佐酒下饭放汤送粥。烹调时只下一点，味道马上得到提升……。

　　意大利的巴马Parma火腿，非红非紫，如纸般薄，可以透视所环抱的一片蜜瓜，咸甜难分，甘美可口。

　　法国的白火腿，奇怪，像白种女人的肤色，还有肌理。

　　我们吃上好的火腿，有时要碰运气。——正如男人遇上不堪的女人，只是他运气不好。

　　当然，有些火腿也不争气。

　　是可怕的瘀红色，接近黑。一味重盐强腌，别说烟熏，连风吹日晒的磨练也没经过，像一条不甘心腐化的死尸的腿。又咸又乾又瘦，出土文物，怎能引起食欲？

　　有些中国大陆的冒牌金华火腿，还会长霉、出蛆、发臭。真是树大有枯枝了。
