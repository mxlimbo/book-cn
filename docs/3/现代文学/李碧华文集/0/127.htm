性感的《鬼妻》


　　《鬼妻》故事简单，但拍得幽异伤感，想不到那麽多观众为它流泪。

　　同时，我发觉这百多年前泰国水乡的年轻爱侣，造型性感，体态动人。不知道未来会否流行这种「反璞归真」但又「好虾人」的装扮？

　　男女都是好型仔的短发，脸部轮廓突出。皮肤黝黑，充满阳光气息。长年赤足，自由随意，毫无束缚。

　　男主角麦，固然俊朗、原始、很man。女主角娜，一身「古装」，原来是春夏的tube top，露肩低胸，下身窄窄的布裙，棉麻质地，调子含蓄，却是现代高雅的杏、啡、灰、黑系列。裙子一裹一缠，打个结，又似三个骨裤。把女人瘦长灵巧的身段，衬托得好舒服。比起刻意装扮的庸俗，一张素脸，却嫌脂粉污颜色。

　　连男观众也欣赏这份简朴的美态，道：

　　「好sexy！」

　　——不过，千万不要笑。

　　Cool到底好了。因为一咧嘴，人人露出日夜嚼槟榔的「成绩」——一排发黑的牙齿，异乡风情，几核突。

　　电影好看，渐渐大家都不在乎了。

　　後来，只听见鬼妻一声声的：「麦！麦！……」，发自内心的低唤，言语没甚麽花巧矫饰，但尽是：「我舍不得呀。我舍不得呀。我舍不得呀。……」真是生死契阔。
