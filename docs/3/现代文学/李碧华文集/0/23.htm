你是我的奶昔抑或咖啡


　　毕彼特和珍妮花的荷李活世纪浪漫婚礼已於加州举行。他俩向世人说出誓词：——

　　毕彼特：「我已找到世上最完美的女人。你是我的最爱、我的过去、我的未来。」

　　珍妮花：「希望每天早上起来都见到你，每晚都与你一起。我更会每日做你最爱的香蕉奶昔。」

　　——香蕉奶昔？非常甜蜜、天真、旖旎的饮品。

　　「奶昔情话」不会腻。几乎所有人，最初起誓时都那麽浓甜。部分可以永恒，部分的奶昔变成山埃。

　　其实催情的应该是咖啡。

　　心理学家一项最新研究指出：「咖啡的香味以及微苦的特质，可令显示兴奋情绪和性欲的脑电波变得异常活跃。」换句话说，令人心跳加速，血脉沸腾，激发欲望，勾起回忆……。效力胜过美酒。——女人（特别是现代女强人）最易中招。在地铁的大灯箱，看到一个手表的广告。重点不在手表，而在咖啡，画面又不见咖啡，只见一位时代女性。「我当然认得你，每天20分钟的咖啡小聚，让你快乐地回去工作。至於我，我只喝水，你已经是我的咖啡。」

　　这段心声可以是「她」的，也可以是「他」的。
