「精神分裂」的青釉


　　送了份礼物给自己：一件青釉瓷器。宋代有五大名K：「官、哥、汝、定、钧」，各有特色，价值连城，以我卑微的能力，当然只能买仿制。景德镇的青花、彩瓷、颜色釉，是元明清时期的主流，我最爱青花，――不过，後来还是挑了青釉，因为上面有开片纹饰，十分漂亮。笔筒太小，伞筒太大，所以要弓箭筒（不是弓箭的箭，是执着鸡毛当令箭的箭。――损友如此嘲弄！）。用来当花瓶。

　　本来还有一个选择，是口沿涂酱紫色釉的，但因这「豉油边」，就贵一倍。我想，也许以後再买更好的吧，也许，会添一件「金丝铁线」的……。此刻量力而为算了。

　　仿哥K的青釉瓷器，釉面上开片，其实是「缺陷」，因为烧制时，釉的收缩率大於胎的收缩率，所以出现严重精神错乱，精神分裂，表面有纵横交错毫不规则的大大小小片纹，细碎迂浚反而成为一种「缺陷美」。

　　东西买回家，才开始「了解」它。我问过三家国货公司五家商店，那些裂纹都有不同说法，而且他们全认为自己是对的：（一）壁裂：指身体。（二）逼裂：指烧制过程。（二）碧裂：颜色碧绿。（四）冰裂：像冰上的纹。――但我查书，册籍上没有这俗称。都叫「开片」。

　　希望对此有研究的读者告诉我正确写法。
