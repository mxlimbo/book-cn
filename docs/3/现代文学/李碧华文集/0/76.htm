只因为恻隐


　　在六月九日的港闻版，大家得知：即使有万多名市民签名及传真，支持九十高龄的陈伯与小猴「金鹰」重聚，度过晚年，但渔农自然护理署仍秉公办理，为这人猴相依为命的关系，强画句号。

　　陈伯闻「判」，呆站一旁，落寞地接受事实。我们也很失望。

　　照说，与陈伯非亲非故，甚至不会帮衬他买「疳积散」，但那麽多市民关心，多方求情，动气骂官，悲哀失落，只因为，这是一份「真情」。经传媒广泛立体报道，陈伯的憔悴思念、晚境凄凉，大去不远，也触动了很多人的心。

　　不肯让此事给画上句号，——是否有点不理智？但议员和时事评论员仍继续向特首致信表示意愿，也希望在两局会议时争取多些支持。

　　爱猫的陆离最感同身受，负责把全港有关专栏剪报统筹。她有很多天真的想法同我商量，其中包括「志莲净苑可申请牌照养猴吗？」之类。去电一问负责人，他们也关注这事。不过不可能。

　　後来，又轮到我有天真的想法。把胡雪姬的电话告诉陆离，托上托，不如问问李嘉诚先生会否为老人家和市民的「人情味」发言一两句吧？

　　——虽然，社会天天有新闻，这小事却是香港人久违的恻隐。
