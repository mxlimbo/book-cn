久违的尊重


　　早一阵，我收到一位女孩寄到出版社的信，问及一本专栏中曾写过的书。她很想买来一读。

　　一看，原来寄自山东青岛。

　　她是长驻中国大陆工作。还有地址、电话、传真号码。我想，寄信太慢了，所以把书名、作者、出版社、价目传真过去。

　　这只是小事一桩，举手之劳。我已忘记了。

　　但这天，竟收到回信。她已托亲友在港代购，特来信告知，以及致谢。――还有一句：「虽有你的传真号码，但未得允许，还是寄信给你。」

　　附言真可爱。显示这读者是个礼貌而周到的女孩。因为她懂得「尊重」。

　　人生在世，除非自困自闭，不与外界来往，否则，他的私人电话、地址、相片、车牌、往事、旧爱、家人状况、私生活习惯……，一定有「知道」的人。你传真，自己的号码就让人知道了。

　　我不介意她回一个传真。或许有点愕然，但马上明白。――她仍情愿寄信，也不令对方稍一愕然，同时尊重了对方私隐（运作中途也有其他人知悉的）。

　　一个小小的号码，侵扰杀伤力不算太大。女孩的笔迹和用语虽年轻，实在是一番心意。也是现今社会久违的。
