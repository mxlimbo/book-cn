食新血路


　　据说，广州目前有数十名外省民工，只靠一叠自己的名片和一部传呼机，进行「服务行业」：——陪人聊天、与人分忧、为人减压、替人道歉、代做孝子、出租伴郎伴娘、提供新闻线索……等等。

　　看他们的服务项目，都无本生利，也是比较高贵一点的「卖艺不卖身」食招数。

　　若从丰富人生的乐观方面想，不过是「全能演员」吧。——演员还不一定得到那麽好的演出经验呢。

　　做得好，还真能杀出一条血路，名成利就。如果狗仔队跟进，或索性投身入行，定发掘不少精采的题材，总好过日夜伏在人家楼下去追问是否拍拖那麽无聊呀。

　　这天看电视《两岸点滴》，便报道了重庆一名以「擦鞋」而声名大噪的阿安。

　　帮人擦鞋，虽是低微职业，但阿安动用了不少「战备」：布、垫、箱、鞋油……，都花样百出。工作起来，投入肉紧，不但像跳舞，还似耍杂技，皱眉表情既痛苦又陶醉，——如同做爱。与对方同达高潮。

　　这项街头艺术表演，吸引无数路人围观，阿婆还递茶水给他解渴呢。电视台拍摄了片段，竟可套入交响乐，节奏感强劲。成名後的「擦鞋王」阿安，赚大钱，买靓楼。又是一个都市传奇。
