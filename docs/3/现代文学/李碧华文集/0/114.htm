月光花的淡香


　　沐浴露、香薰油、温泉浴剂……，不同的香味，因心情、兴致而经常更换：花香、草香、海洋、水果和植物精华。不开心的时候，短暂的香薰掩盖了一切因由。——虽然过後仍然有点内在哽咽，起码纾缓了一阵，对得起自己。

　　有些人爱上了一种味道，永恒不变，这是因为味道以外，有心事。是「人」的幻化，而不光是嗅觉和感官的刺激。——我们其实也希望只为一种味道颠倒，如果不可以，便让众多不同的味道来调剂。

　　个别香薰油不大安全，还有致癌物质：苯。吸入过度，可致皮肤敏感、情绪不安，反而更抑郁，真是「落井下石」。

　　有一种印度来的香薰烟，约一尺长，黑色，貌不惊人，燃点时散发「鸦片」味道，令人产生兴奋感觉，一时不察，便成了瘾君子。

　　即使「香味」是危险的，莫测的，但遇上清清淡淡又精神一振的空气，还是一饷贪欢。

　　一天走过店子，竟闻到罕有的芳香。原来是「月光花」。据说一九九八年厄尔尼诺横渡美国西南部一个沙漠时，Moonflower前所未有的开得灿烂，黄昏过後，月亮初升，千千万万盛放的月光花令四周充斥特别清新、浪漫的芳香。

　　我买了一些沐浴露、喷雾和香薰油。他们送了个丝绢香囊。浴室马上又换过一种清淡的香味。叫我忘掉前尘。
