冰冻的果实


　　阅读星新一作品，同时也看了一大堆日本「掌中篇」。同期有一个唤《冰冻的果实》，以为是好有趣小品？原来不。

　　作者黑岩重吾，1924年生於大坂。他是个历尽沧桑，换过无数职业的作家。这篇不算太吸引。

　　以一宗绑票案开始。河内大作是一家制度非常严格、无情的大公司经营者，读小学三年级的独生子治夫被绑票一个多月了，刑警人员追查可疑人物。

　　发现河内妻子美代子与公司一个中级职员，长得标致小生水木茂有染。但水木不是绑票者。冢口是旧部属，与他前妻和代亦有染。对河内有刻骨之仇。

　　原来河内大作的身分，是当年驻中国东北的关东军军官，以妙法狠惩之，可留活口。冢口深夜被押至零下三十五度的郊道，军裤和棉裤拉下到膝盖，他的局部很快为白霜所覆，并活活冻成冰块。他麻木了。送到医院後，那话儿已像石榴一般裂开，不痛，但废掉。

　　故事没甚麽。但我是个喜欢联想的读者。――历史上，关东军在哈尔滨建立了「七三一」石井部队，进行活人解剖、细菌、截肢、冷冻……等等实验。中国人是被实验的「马路大」。

　　那是说，在冢口之前，这个非常奏效但又不致命的酷刑，早已由无辜的中国人在军官的狞笑下实验过了……。
