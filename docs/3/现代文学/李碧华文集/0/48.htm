年年都有荔枝骗局


　　外表甚为精明醒目的歌手，向记者报料，他在深圳买荔枝时被骗了。

　　荔枝「小年」，虽比不上去年的丰收大造，甜美多汁，不过深圳的联检大楼和闹市街头，仍有一筐筐的荔枝摆卖，小贩让你「试食」的都肉厚核小，看上去十分美艳，和香港的售价一比，便宜了一截。很多人过关，都顺手买一两筐回港。不过这些竹筐都很「神秘」，你们永远无法「亲近」，付钞後肉帛相见时，便知：(一)底层的多是混种、次货。(二)下面垫了纸皮、树叶、枝干。(三)染色。(四)……

　　前车可监，为甚麽仍有人上当？当然是诱惑力太大，贪便宜之心受不起。还有，总以为自己不容易被骗，也不一定处处是陷阱吧。

　　一个成年人上当了，不能怨恨奸商，或怪责次货，——完全是自己的责任。个人眼光不够。

　　万一我「被骗」了，肯定不会公告的。已经那麽生气，还要让人知道自己笨？受不了双重打击呀。打落门牙和血吞算了。

　　——但各位千万别像这些人，自尊心过强。

　　世上若无受害人「挺身」揭发，便是对罪恶（即使小如两个荔枝）纵容。

　　看，荔枝骗局不是又警告了很多人吗？一个千年道理：「便宜莫贪」，道来不觉重复累赘，竟历久常新。
