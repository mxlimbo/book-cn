剧场内外


　　看了百老汇音乐剧Chicago的演出，虽没预期的「震撼」，但仍觉娱乐性丰富，性感而激情。

　　它是个讽刺名利、法律、野心、道德……的荒谬剧，所以一切都是毋须解释的。而很奇怪，发生在二三十年代的故事，放诸今日社会，仍贴合得天衣无缝，是人愈活愈回去？抑或世道根本是一个循环？二十二岁的音乐剧，不显老，但主角则换上最鲜嫩而充满活力的。

　　两位女主角，短发的韦玛比洛斯更抢镜。在监狱中，一群以各种方法杀掉男人的八婆女犯，亦以她最醒目、生猛，一来歌舞难度高，角色有性格；二来，她又演得好放。

　　此剧反应不错，还加开数场。

　　我很喜欢他们的宣传单张，以红黑为主调，配合整个戏剧的风格，简约但又邪恶艳丽。它的场刊也是收藏品。

　　题外话：在架上陈列各界的宣传单张和小册子，大都很精美。以前粤剧京剧话剧演出，设计很老土，近年来已改善，并且花了心思。每次都能遇上一些赏心悦目奇招迭出的「招徕术」。

　　《艺讯》（artslink）美术水准一向高。「法国五月」有多项节目，其中「百年法国侦探电影」的小册子亦以红黑为主，拎上手，竟有几个子弹孔洞。江湖片的「母乳」，是阿伦狄龙的《独行杀手》，你们记得吗？
