繁华的古国


　　在日本花街柳巷风月区，常见一些阿伯，不分日夜，站在一角，高举一个大大的宣传牌，上面是挑逗淫亵字句、图片、电话号码、价目表。为年纪可当他孙女的小姐们招徕。

　　这些阿伯在烈日当空或寒风凛冽的街头，木然地站上大半天，即使来来回回，也不过是同区。——一名寒伧的阿伯，究竟是否可以勾引性欲？不有点倒胃和不忍吗？

　　不知道这些人肉广告牌的功效。但既一直存在，或者有点作用。又或者没有人愿干，只得找耆英。

　　是为了钱。

　　如果阿伯「肯」，还可以拍写真，与另一阿伯互相玩弄拍同志AV。同性恋杂志和AV市场，可以容纳大肥佬、壮男、核突佬、苦海孤雏，和——阿——伯。完全不以美学角度出发。

　　打真军的AV女郎，当然以鲜嫩漂亮为主，备受蹂躏的小妹妹，敬业乐业，交足功课。某些也飞上枝头变凤凰。如饭岛爱小姐，还可晋身主持人、饮食评论员、旅游节目的嘉宾……，上「清谈小(?)」，穿上衣服的她，观点得到尊重。

　　不过，也有好些变态性虐AV，女主角是孕妇、丑女、七十岁的阿婆、红潮泛滥的少女、刚产子乳房可挤出人奶的师奶……。还吃屎喝尿。为甚麽她们「肯」？——除了为钱，还有其他理由吗？
