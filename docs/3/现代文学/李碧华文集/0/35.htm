不，这还不是定局


　　世事无常。有时以为已是「最後一秒」，「全世界」也视为定局、定案，——忽然，峰路转。

　　不，不是这样的。山穷水尽疑无路，柳暗花明又一村。那个「疑」字，告诉我们，仍有万分之一的变数呀。

　　专栏永远跟不上一些莫测的变数。因为，专栏早两三天截稿，绝非今晚写明日出，即使我这样常有「不情之请」的作者，注明：「急。请打尖先排」之类，也有时差。

　　最近便「撞板」了两次：——

　　（一）关於陈伯与「金鹰」事件，原来在六月底大家已收到渔护处语气坚决的公函，七月初陈伯上庭已准备心情面对无奈的分离。我们都以为没有甚麽希望，法官在最後关头行使酌情权，人猴可以团聚。

　　（二）「博学堂」在八月一日宣布清盘，遣散了员工。与之有关系有联系甚至有点感情的作者都纷纷发表「哀悼」文章。才数天，忽有大股东招待记者宣布抢救收购。人人当它已大去，连员工也在找新工（几乎便签约了）。竟又可「复活」？

　　专栏见报日，作者「蒙在鼓里」？其实没能力预知三日事，也来不及。就算作者今天大去，是的，有三天存稿，好似尚在人间。
