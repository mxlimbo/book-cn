午夜的千层纸


　　十一二点的夜中环，兰桂坊下来地铁站外，见到一个男人在摆小摊子。

　　他的「货」是小孩的古老玩意，一根竹签，顶上是「千层纸」美术工艺品，把摺叠起来的薄纸拉开360。，再合成个一层一层的球状、葫芦状、伞状物体。小白猪、孔雀、熊猫、士多啤梨、金鱼、罗伞、灯笼、苹果、菠萝……，薄纸给染成红橙黄绿颜色，久违的古朴和乡土。

　　看来五十多的小贩很斯文，穿白恤衫西裤，操不纯正广东话。他卖这些小玩意，似乎带点不好意思。但仍落力向我们解释民间工艺。

　　「一袋有十个啊，才十五元。」

　　旁边有比较贵的，原来是小蜜蜂，打开後见黄褐和白相间的头胸腹，还有翅膀。

　　「每袋四个二十元。」

　　老实说，买这些小玩意干麽？不过还是买了一袋，外加一只小蜜蜂。付了二十元。为了小贩有一点点知识分子的落泊吧。大白天的时候，他不会出现，只在午夜，灯火阑珊处，推销没有人需要的已经不合潮流的货品。——自食其力。这就是生活。

　　有外籍人士走过，驻足，我正想多嘴推介一下。男人已用简单英语，示范那个待开展的小小千层纸球。

　　不知如何，走下地铁站时，总是联想到一个中国的教书先生，那内心摺叠起来的东西……
