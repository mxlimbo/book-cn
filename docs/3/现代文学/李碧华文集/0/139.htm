当他骂他是贱妇


　　看一个男人怎样骂人，可以分出「孪」和「直」。

　　男人相骂，语气姿态好野兽，盛怒拂袖而去，或大打一场。某些基佬（强调「某些」）却是「磨烂席」的，絮絮叨叨，而且很少开打（兰花手怎动武？），哭泣或一时语塞也有。不过常见的嘴脸特色是：——

　　（一）慢条斯理。每词或句之间，有一段小小的拉腔。身子微晃。

　　（二）喜欢「咬牙切齿」。

　　（三）七分脸，昂首，翻白眼「厉」人。骂时扬眉（一条），还有「藐嘴」。

　　（四）夹杂冷笑。而且训练自己唇部尽量不动，以达cool的效果。

　　（五）最具杀伤力的不是昂藏七尺孔武有力，而是狠毒、致命、闪过一丝「悍」意的眼神。骂时都有「蛇蝎美人」的自我感觉。

　　（六）坐着骂多过站着骂。除非拉扯。

　　（七）即使最恶最泼辣的也是风情的。即使痛哭也是妩媚的。

　　（八）爱用的词汇是「贱」：「贱人」、「贱精」、「贱妇」（即使对方是男）……。——因为这个字可占时五秒以上。

　　（九）人家累了，他永不言倦，想想又再来过。——争取做最後收声的一个。

　　（十）终生事业。
