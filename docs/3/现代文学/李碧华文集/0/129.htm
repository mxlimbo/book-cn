过分的宁静


　　你有没有被一种「过分」的宁静吓着呢？

　　深宵写稿，万籁俱寂，本来十分惬意，但太静了，反而一点点的细碎的声音都变得巨大。一只小小的飞蛾自窗外悄悄进来，忽地如眨眼「巴习」细响，伏在案头的笔座，大吃一惊。那飞蛾还很漂亮，透明的青色翅膀，泛了红丝，好似血管，实在恐怖。

　　如果四下有残留的电视或风雨声响，就不会诱发太多联想。

　　还有一回，有朋友在书房工作，为了不想打扰，我出入都「蹑手蹑足」。好一阵，问：「要不要喝杯好茶？」——话还未了，对方吓了一跳，是真的整个身体「一跳」。因这受惊，良久才能定神。太好心，做了坏事。

　　也不光是我们这些普通人。

　　据说，毛泽东晚年有位颇得欢心的贴身服务员小孟，可她刚进中南海时，热情的主席忽然不悦，同他说话也不愿理睬，还摆手让她离开。受冷落，甚惶惑。原来他房间特静，一次她进去时一点声响也没有，突然出现在面前，所以吓着了。——以後她进屋前，懂得咳嗽、鼻子出点粗气，这样才有「思想准备」。

　　原来人人都经不起绝对的、彻底的、完全的宁静，在一个这样的环境，松弛脆弱，不堪一击。——但「意外」，往往是措手不及的。

　　山雨欲来风满楼，反而是最好的思想准备。
