π水是甚麽水


　　看到一段怀疑π水能否治癌防病的报道。接着又看到π水为有益健康优质水的声明。

　　我很好奇。

　　早前在日本，和在香港的city'super，也见过瓶装的π水出售。水就是水，同矿泉水、蒸馏水、天然水、加味水……一样，无色透明。——蕴藏「特别的东西」？肉眼根本无法测知。

　　但这些π水，强调自己含氧量高，还有高能量，喝後促进新陈代谢，排泄畅通，排毒美颜，用来浇花，花也茁长得好些。云云。

　　π水是否「神仙水」？它不算便宜。买过一瓶，如常喝了，如此而已。不明白为甚麽唤「π」？

　　「π」是希腊文圆周一字的起首字母。这个符号的使用约始於十八世纪初。其实公元五世纪，我国数学家祖鶻之从天文和器械制造的需要出发，推算出圆周率在3.14159265与3.1415927之间。

　　我们念数学，「π」是3.1416。毋须去到精确的八位数字。但「π水」同圆周有关吗？「π水机」又如何令水摇身一变而「π」了？

　　生化学系专家说：「水是H2O，只有气态、液态及固态三种。」医学会副会长说：「哪有饮水可以治癌？根本无可能。」

　　当我拎着一瓶纯净而平静的「π水」时，很希望有报刊跟进一下，究竟里头有甚麽？
