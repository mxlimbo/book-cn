长期抗战也辛苦


　　「长期抗战」的新闻，记者也好惨。

　　像寻找失踪的十五岁智障男童庾文翰，父母固然疲於奔命，劳心劳力，情绪跌宕……，跟这单新闻的传媒，一样辛苦。

　　我们对那些过分侵犯个人私隐的狗仔队不以为然，甚至觉得无意义。但很敬佩所有日以继夜跟进寻人案的传媒，十几二十日都呈「胶着」的死守状态，又不能（不敢）走开，生怕就在这分钟有突破发展，生死一线，万一刚巧有消息，竟然错失？怎办？同香港出境处，罗湖入境处比，罪名当然不能同日而语，但也是「错失」呀。

　　但一直以来的虚报，啼笑皆非的勒索、假消息满天飞。刚想休息，突接线报，午夜包车赶赴广州，长途跋涉。证实流料，又得返回「总部」standby。传媒把酒店七楼住满了。

　　为甚麽我会知得那麽详尽？因为请托了以前在帮慰安妇袁婆婆寻人时，代起了易卦的桂姐，看这回有没有灵感，可以尽点绵力代寻文翰。我把卦象同本报的驻深圳记者联络。发觉有些也病了，还换过人轮班。

　　我看他们也很闷吧。说：「还好，可以到楼下做按摩。」

　　「有一次刚浸脚，突然有料到（流的），马上飞去。哪能安心做？」

　　按摩目的是松弛。一天未了，一天仍极度紧张。
