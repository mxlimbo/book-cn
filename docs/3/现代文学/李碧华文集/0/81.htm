为甚麽人反而次要


　　小猴「金鹰」与九十高龄的标伯重聚，机会微乎其微了。

　　渔农署接管了小猴，想不到引起各界强烈反应。当局忙发放照片，已为小猴找到代母，日渐适应同类愉快生活，来「反驳」热心人士。

　　读者来信说感到悲哀，想代求情。其中一位，还用上了「急件」，附注：「希望代传真不会石沉大海！」

　　其实我们也是小市民，同情老人被「横刀夺爱」，「生离死别」，但面对这样僵化的，不近人情的制度，不满也没办法。

　　我问过渔农署新闻组的佘小姐，她承认收到很多投诉和签名，明白老人心情，不过「标伯清楚养猴需要领牌」。署方不鼓励任何人非法收养野生动物，亦无人有牌。她说，这样回答我，是因为公职身分。……

　　後来，我又问自然护理署负责人黄先生：是否一线生机也没有呢？他也许已重复了千百遍，仍很有耐性地阐释小猴始终是野生动物，为它着想，安排到内地野生动物园。标伯最後一见，可以隔着笼子。

　　——特别照顾猴子？以此为重心？不管老人悲伤憔悴消瘦，依依不舍晚景凄凉？「人」反而次要？市民去电骂「无人性！」。无论怎样，他只尽职责，「只讲马骝！」

　　我问：「万一老人伤心过度，有事发生才受理吗？」

　　他无奈：「不回答假设性问题。」
