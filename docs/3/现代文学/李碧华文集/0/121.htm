黑羊薄荷鸡


　　天气湿、热、闷，心情和胃口不好的时候，最佳选择是「辣」。——一辣解千愁。

　　不喜欢吃日本咖喱，款式不多，甜而不辣，欠缺「杀伤力」，无法以毒攻毒。

　　东南亚各国都有不同口味的咖喱，我偏好印度的，香料的刺激是另一种感受。印度店子昏黯气氛，薛他乐韵的性感和「姣」，又是附送的。

　　有时并不专程去吃印度菜。湾仔区的「国际咖喱馆」，它有十七年历史，做八国（印度、新加坡、马来西亚、印尼、斯里兰卡、缅甸、巴基斯坦、苏门答腊）咖喱远近驰名。但因其貌不扬，还与一间舞厅并列，出版社在附近，我们又常路过，太熟了，竟很少光顾，算来不足三回。所以尝不尽它的特色。

　　另一方面，餐牌太迷茫，譬如鸡，有椰汁鸡、鬼鸡、队长鸡、马士打鸡、东印鸡、南洋鸡、乾炒鸡……；虾，有吉黅虾、古马虾、马六甲虾、白咖喱虾……；羊，有北印羊、酸奶羊、玛沙拉羊、甜酸羊……；鱼、牛……，不数了。

　　这天点了黑羊、非洲鸡、苏门答腊牛，以上全都是黑漆漆的，加了菠菜芝士、青瓜乳酪，才有点「起色」。最奇趣是薄荷鸡，以薄荷叶、薄荷汁、薄荷酒调味。不辣，但清凉，颜色太青翠了，——绿得好假，像一盆漆，髹古老电车的那种漆。
