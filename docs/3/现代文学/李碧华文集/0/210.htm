司机的一念


　　本报二月二十日的头条，是继「史诺比」之後另一次意外。

　　当日新闻，可以做的是郑中基飞机上酗酒闹事的判决，煽情的更有某与男友辟室寻欢，身中三十刀伏尸酒店浴缸，够吸引，与前「丰丰与『男友』同游大连」竟可作头条相辉映。

　　不过这天的头条是个大都会小故事：「南京少妇火坑前的徘徊」。没甚麽「震撼性」，但我觉得很动人。

　　来自南京，三十五岁少妇阿琪，嫁了两个丈夫都去世。她这次来港，便是得警署、社署之助，证实丈夫早於一年半前病逝，并安排暂住临时庇护所。但因生活无着，她带着十岁女儿坐的士到夜总会找工作维生。司机老赵，看她彷徨，便介绍了她陪自己的朋友上班赚点钱救急。

　　妈妈上楼陪另一个男人「睡觉」，已懂事的女儿在楼下忍泪默默等候。老赵想到自己的子女，突然产生一种怜悯之心，所以与报馆联络，记者跟进……。於是有了这新闻故事。

　　大陆妇女新移民卖肉求生不是新闻（甚至「娱乐名人版」也充斥这样的花边，但个个身光颈靓），司机对绝境母女施以援手，是「一念之仁」。有时人一念间做了好事，是他预料不到的。香港人「谂头」多，「思路」也够复杂，简单原始的「一念」，又有多少？——相信他助人後也很快乐，这是最好的回报。
