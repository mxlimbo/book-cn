最不会哭的人哭了


　　绝少在人前流泪的人，忽然一哭百应，哭作一团，直令旁观者有点感动了。

　　高收视处境剧《男亲女爱》一百集大结局了，观众舍不得，演员五个月来朝夕共处，合作愉快，擦出火花，当然更舍不得。

　　最意外的，是大家心目中最不会哭的子华、嘟嘟和苑仔（你看娱乐版这样的称呼，简直「好熟」似的），全属资深演员，见过世面，经历丰富，亦惯於悲欢离合戏如人生了，——他们却哭得最厉害，足足十分钟也停不了，成为传媒焦点。

　　原来亦「人之常情」。

　　巴辣硬净的女强人、缩骨衰格的cheap佬、神化豁达的表姑婆……，这些「人物」，理应不易动真情，他们表现坚毅、冷静、嬉笑怒骂，如同我们四下，都有好些强撑至最後一秒，也不肯「衰着人睇」，以致仪态尽失的友好和仇敌，把「我」压抑成一块冷硬金属，碰上去连手指也冰冻了。

　　偶一不慎，才释放自己。一哭不可收拾。

　　在一段特定的期间内上下一心完成工程，「最後一天」总是令人依依不舍的。释放自己的动力不必很大，只因「意志力薄弱」，才舍不得。面临「失去」的一刻，这些「哭相」，几可爱。

　　——又，会不会是有些事情发生了，大家都不知道呀？
