老少平安生死恋


　　在娱乐版看到一款菜式：——「生死恋」

　　那是祥嫂在「楚留香」官司中，被法官指「伤害家庭」，最终判她输给对簿公堂的子女後，她却拣了个吉日，冒雨哭坟，向亡夫祥哥致祭。

　　别人的家事，我们局外人还是不清楚的。所以只想谈谈这「生死恋」。

　　它有个好凄厉而浪漫的名字，原来是咸鱼拆肉，与新鲜鱼混成的一煲稀饭。乃祥哥生前「至爱」。

　　咸鱼和鲜鱼都美味，这碗粥，设计得很有心思。无骨、肉感，生死厮混，共处一锅，作为同心祭品，是贴切的。

　　我们知道，任何食物，若果加入感情、真心、眼泪……，它的味道得到提升，才特别好吃，吃的人也难忘。

　　只要不是道具。

　　不过「生死恋」上日常饭桌，又似乎有点哀怨了。前吃过家常小菜，曰「老少平安」，也是无骨、肉感，材料是鱼肉和豆腐。听起来舒服，不过略为平淡。（你爱哪一个？）

　　最好吃的鱼，是上一秒它仍鲜活地在水中游动，下一秒，便成为一道菜。对鱼而言，只有「生死」，没有平安，没有爱恋，它甚至没有心理准备。
