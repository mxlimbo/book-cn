赠品大多无用


　　有些人天天去买包，储存若干印花，便可换一只瓷碟。那只碟其实很便宜。大部分因买卖而附送的赠品，都便宜且无用。申请信用卡，可得小家电、杯碟、背囊、笔……，这些人人基本上已有。入油送纸巾，看电影送T恤，还有其他的甚麽滑鼠垫、毛公仔、相架……，买化妆护肤品，送你一瓶恐怖的橙红色甲油，一支连　尸也不会擦的口红，或金色的塑料提包，你根本就不想要。拈上手已变形的劣质衣物，全印上电影名、商号、产品，穿後做人肉广告板，而且马上「过气」。

　　世上必有人十分受落。但我觉得，一切不切实际「用不」的东西，都是负累。很多废物，我们不会携同过渡千禧年。

　　近日银行纷纷大减楼按利息，低於优惠利率（P），或予贷款额现金回赠。精明的业主选择後者，因为他们明白，如果不是长久拥有物业，不如即时受惠上算。

　　他爱你，胜过无数小动作，甜言蜜语比不上实际的关怀。任何交易，送大堆五彩缤纷的赠品，不如给折扣。

　　个多星期前，区议员选举，我们在街上老是遇到热情亲切的拉票笑脸，候选人像男朋友一样附赠殷勤问候，令你如沐春风。选举结束了，市民所得又已失。是的，无用！
