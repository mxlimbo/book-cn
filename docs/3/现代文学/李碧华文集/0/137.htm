带回味的邮票


　　常觉得天天吞进不少废物的大嘴巴便是我们的信箱。

　　总是无聊的宣传单张、楼盘广告、地产公司说谁谁谁肯出×××价钱要来买阁下的单位（最空洞的勾引）、帐单、薄饼速递优惠……，莫名其妙的，称不上是「信」的物件。

　　很久很久都收不到值得高兴的信。

　　现代人沟通方式多，都利用电波。——仍在写的人，他们不是「通讯」，而是「通信」，写得好，信纸不但有馀温，还有馀情，是下一回的延续，又不担心成为甚麽「把柄」。

　　邮费浪费在人家收到後无奈厌恶得要扔掉的垃圾上？花了一元三角，又不环保。

　　近一两年，好些办公室也有「邮票行街」兜售。他们只消拎一个胶袋，已盛了价值不菲的「货」，大量批发。

　　这些邮票各种面值都有，全可打折。连邮票也减价直销？——不，原来是炒了的纪念邮票，最多是九七、九八年的「货」，那时炒家囤积炒卖，在大陆甚为抢手。……如梦的幻影，如露亦如电。邮票无人问津大贬值，只好回流到香港平卖，脱手套现。

　　我们或有很多邮票，即使有折扣，但我们寄不出那麽多的信，也没那麽多可收信的人。带点回味的纪念品，依旧通用，却是用不了。彼此都在嘲笑……。
