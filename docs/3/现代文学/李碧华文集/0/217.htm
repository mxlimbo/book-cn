一点也没有


　　听到或看到这样的话，总是有点难过的。

　　女人，曾经有个男人给你很多的钱，也曾经有个男人哄得你很开心。但後来……

　　大家全都明白，「变心」，只要心一变，九牛拉不转，回头也无益。各有前因，比较复杂。

　　後来某日，任凭人问：

　　「当初你爱她甚麽？」

　　「她有些甚麽优点？」

　　「有没有一些可取的地方？」

　　「你们总有过难忘的回忆吧？」

　　「……」

　　――他们都没有说半个字。倒是难听的话滔滔地说了一箩筐。数落半天，却「不肯」说――一，句，好，话。

　　难道真的一无可取？一点快乐的回忆也没有？当然不。世间男女，都是自由恋爱，邂逅锺情，都共度一段美好辰光，水乳交融，牵肠挂肚。只是一旦决裂，便决绝得全盘忘掉了。人是彼此挑拣的，再不堪，当初也是你的心水。――可惜心如止水，还相敬如冰。总是欷　的。

　　这种「风气」很可怕，对下一代更是「言教」和「身教」。

　　有的女人庆幸没遇上任何一个，甚麽都不曾发生便免於互相攻讦。有的女人情愿这样，曲折离奇，百年一遇，聊胜於「无」。你们呢？
