不愿像木村拓哉


　　帮我做《烟花三月》美术设计的，是Catfish（猫鱼）和他的得力助手Joey。找他们，有小小「屈」的成分吧。因为比起那些甚麽SK-Ⅱ、天王网页……等Project，我的书，是「友情客串」。「客串」也得苦战个多月？猫鱼恨不得通知管理处不准我上楼。

　　主力设计的J，合作愉快，但不能催。他几乎每晚都是尾班船回家。因此笑不出。因此更像一个人。

　　某日上来出版社交图，编辑部的女孩已很有反应。J走在闹市，总也有些人多望两眼，或遇星探。但没在日本厉害，妹的表情，据说是惊喜、疑惑，好奇、窃窃私语……。

　　那晚有其他公司负责人上去开会。她一见，又道：「咦？木村拓哉？」

　　Joey苦笑。

　　他有点抗拒。再「像」，又如何？不像，自己是自己，没甚麽烦恼。稍像，常被陌生人打量、比较，好尴尬。所以，禁止我们再谈这个Point。

　　不愿无端像另一个人，大家很理解。――而且，靠的是「脑」，又不靠「样」，我猜他是这样想的。

　　为了「淡化」，划清界线，把头发长得很长？（有用吗？）

　　这种「痛苦」，一般人哪会有？
