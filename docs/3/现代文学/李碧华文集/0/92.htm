还记得邮差叔叔吗


　　刚看到报上一个广告：——

　　「还记得邮差叔叔吗？」

　　搞不清楚卖甚麽广告，只见一个「重量级」的邮袋。

　　虽然世界变了，是个甚麽电子时代，「邮件」像个负累，「邮递」似乎落伍，——但我们仍怀念从前传统、细心、经过等待、手上有温度和重量的沟通方式。真的。直到今天，我还是喜欢收信。可惜没甚麽好收的信。

　　记得曾为「天天负重的邮差」说过几句话。他们依旧天天背负六七十磅的邮件沿门派送呀。

　　有一位，给我回应。他说，用手推车当然可以减轻负担，但上落楼梯、石级，还是靠体力的。背囊式的邮袋较贴脊椎，不过大部分工作多年的同行，都有职业病，腰背尾龙骨，几乎都移位、受伤。

　　在33℃的酷热下，衣服已湿透。邮差叔叔只能苦笑：

　　「我们的夏季制服是深绿色的，特别吸热，这设计真与别不同。」

　　看电影，邮差常有艳遇。小镇中迎风骑脚踏车派信又多麽的逍遥。可是香港的邮差是被重担压得歪曲的小人物。他一再叮嘱：

　　「请不要公开地区和资料。在这个时势，我们谁敢吐苦水？弄不好，连邮袋也没得背。」
