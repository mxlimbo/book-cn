《男亲女爱》的火花


　　《男亲女爱》有个婆妈老土而不吸引的名字，但剧集却出奇地好看。

　　最精采的，是演员擦出意外火花，谁也不会被比下去。我们以为郑裕玲强，黄子华却不甘示弱，而「小强」更强。其他合演者，大都伶牙俐齿，抵死够放。转眼又完了一集。

　　以为大状师爷之类的知识分子专业人士题材，师奶会抗拒。谁知她们亦被俘虏，口碑不错。我错过了头两集，还要师奶覆述。原来影射「向富婆索回内裤及咸碟」事件，那麽无聊又戏剧性丰富，大加利用，先抓住一批八卦观众。等於捉猫揪脖子。

　　或者，大家喜欢看「有缺陷」的人生。

　　饱读诗书职业高尚，仍各有难处。内心的阴暗面，感情死穴，带笑自嘲。

　　醒目得来衰衰　的小男人，外表精锐却是午夜悲泣小女人的泼辣货，总是火星撞地球。但斗气冤家最後终成一对，——现实中多半没如此美满。所以看得过瘾。

　　红花绿叶高手过招，对比一些剧情沉闷或演技稚嫩的一小时长剧，更令人欷嘘了。——为甚麽只得半小时？为甚麽「光阴似箭日月如梭」？那麽快？

　　恨不得给它喂伟哥，可以延长播映时间。是的，这是对此剧最大的恭维。
