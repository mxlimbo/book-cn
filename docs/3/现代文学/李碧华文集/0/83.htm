吃掉九层塔


　　吃越南菜时，有伴碟的小小香叶，奇特芳香，带着薄荷叶柠柠叶的味道，但不是此二者，问店主，告知是「罗勒」。又称「九层塔」、「零陵香」。

　　啊原来是它。

　　我的沐浴香薰油中有「罗勒」，可提神和解头疼。感冒风寒时，在水中加几滴罗勒、薄荷、柠檬草，便是一缸奏效的热汤。

　　京都有家卖香的专门店，当中便有「零陵香」线香，燃点来冥想。据研究报告，它含有某种物质，能令脑细胞活跃、灵敏，并刺激思考和领悟力，更预防老年痴呆。——若是这样，人们何须吞服药丸？

　　多用这叶子滚汤煮菜伴饭，既有香草特色，还有食疗保健作用。

　　忽想起，我听过「九层塔」这名字是在台北，而且还在冰淇淋店呢。

　　台北西门町武昌街有家店号，专售奇奇怪怪，匪夷所思的冰淇淋，如猪脚、麻油鸡、咖喱、人叁、糙米、当归、肉松、辣椒……，都拎去做冰淇淋。当中有款「九层塔」，看来面有菜色，很有趣。但它是甜品。

　　「九层塔」何以得名？不知是否一茎九叶？它的种子是黑褐色的小坚果，曰「光明子」。可明目，故名。

　　日本有「五重塔」，中国有「七级浮屠」，「九层塔」比之还多两层。加上「光明子」，你说是否好禅？
