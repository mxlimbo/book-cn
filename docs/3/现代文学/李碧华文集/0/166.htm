同志们的初恋情人


　　报载，汪明荃担演以海关为题材的剧集《雷霆第一关》，与李修贤合作。这对拍档不但有新鲜感，而且有台型，也是好戏之人。

　　李「珍藏」了二十五年的乌蝇眼镜重见天日，则刚好是潮流。

　　今日的潮流，是实力派火花，郑裕玲和黄子华便叫观众看得过瘾。

　　不知师奶观众是否接受李修贤？――不过，同志们一定捧场。据说有导演一直渴求找他演戏，一提到李，已「杏脸含春」，可惜至今尚未如愿。也见过有读者给信箱主持人写信诉心声，道尽同性倾慕之情，还拜托不要公开这位男星名字，可他的字句细节，呼之欲出，或许是故意的。

　　近日娱乐版报道姜大卫做心脏病「通波仔」手术，当年打不死又cool又潇洒的游侠浪子，英雄也怕病来磨。王羽「固执」捉奸新闻，有点迟暮，也有点欷嘘。令那些暗恋他好多年好多年的死忠派，百感交集。

　　狄龙对记者说，他现在最想演的角色，最具挑战性的，是「基佬」。不知哪位导演成全心愿？――其实他的《刺马》就是基片！你说，张文祥为甚麽要刺马新贻？让我研究一下这心路历程。

　　王羽、姜大卫、狄龙、李修贤……，统统是当年同志们初恋「梦中情人」呢。那时只能如此寄托。
