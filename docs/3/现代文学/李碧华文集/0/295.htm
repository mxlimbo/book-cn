士多啤梨．奇异果．火龙果


　　某些味道，或某些食物，平平无奇，是因为『回忆』，才加了分。

　　那天我们去吃水果宴。他们点了一个汤，用雪梨、苹果、木瓜来煲猪肉。因为店主相熟，便说，这汤清润可口，还可以加些士多啤梨。

　　本来我以为它会血肉模糊，又易变酸，不堪折腾。不过上桌後点缀得十分艳丽。

　　士多啤梨人馔，美味又含维他命C。配合香煎的猪排，好过甜酸芡汁。

　　日本人用来做大福饼，裹以豆沙、樱花、糯米粉团，色香味都诱人。

　　其实士多啤梨因本身色相，最好是鲜吃，磨茸榨汁做奶昔，总觉可惜，欠了一重观赏价值。做蛋糕、冰淇淋、果酱那些，一定是比较不那麽登样，只好大混战。

　　士多啤梨有精致小巧的，也有放肆地扁着一张脸的。有些颜色鲜艳，但色泽深沉的更香。它成熟了，任性地横陈肉体，不顾身世，人们又吃得痛快淋漓。

　　它们的香甜，全靠体内一种『庶糖运输机』蛋白质，把糖分运送至每一个角落。──英国科学家已研究改造及控制基因的技术，令它们更香甜。──不过，高科技也令水果失真，并且对身体有害。我倒情愿一篮子中，有中有上（当然不要下！），也有特上，否则怎颢得珍贵？

　　今年三四月，对士多啤梨特别有印象。

　　意外地发现，当造时，山东盛产，不但亮丽清甜，而且太太太便宜了。大街小巷都见小贩摆卖。是堆在木头车上，一个小丘似的，十块钱三斤。可以让人吃死了……

　　在香港、台湾和日本，好的士多啤梨用透明胶盒子一个一个的铺好，永不重叠，尽情展示。贴上的标签令它们矜贵。

　　我们从没想过，在中国的东北，它的出现，『当时只道是寻常』。──如同某年在北京，见瓜山桃海，丰腴甜蜜的水蜜桃，也是论斤买，一大堆，捧个满怀。

　　如果说这是老天爷某些『礼物』，那一定是不能用价值衡量的『慰劳』了。因为见到红艳艳的，吃极吃不完的水果，已经很开心。而士多啤梨凄美之处，是无法久存、强留。要尽情。很快变坏，得扔掉。

　　士多啤梨教我们有酒今朝醉。

　　菜单上还有好些水果。例如芒果鸡柳、苹果橙黄椒花枝片、菠萝奇异果鸭脯、橙花带子……甜品是火龙果荔枝果冻。

　　有没有留意，士多啤梨、奇异果、火龙果这三者有何关系？

　　外衣分别是血红、褐（还有体毛）、紫红（多层）。肉亦相异，红、绿、白三色。──但，原来它们　满了黑点。这些极小的黑米粒，成为与众不同的点缀。没有哪位高手可以去除得乾净利落，也不会这样煞风景。当我们吃这些水果时，同时也咀嚼它们的下一代。辟啪作响。

　　若由搅拌机运作，果汁的底层，总是沉淀着一堆芝麻，每次倒出来，店主必再搅拌均匀，好令每一杯都拥有些，如同我们无法摆脱的，生活中的乐与哀，感情上的烦恼（或麻烦），希望和失望。──如果没有这些，当然也活得太乏味了。

　　──要吃得快乐？幸遇三杯酒美，况逢一朵花新，便得忘记明日阴晴未定。……
