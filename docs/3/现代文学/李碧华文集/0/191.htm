改名「果皮」的女人


　　改名的真正目的是求「转运」，必然在灰心失望之馀，才对一直伴随的私有物厌恶，欲去之而後快。一个人恨自己的名字，是它再没有可供憧憬的空间了。

　　女星半红不黑，最爱找相士给改些「刁钻」而似意头很好的名字。可惜总是愈改愈糟。前一个名字人家都记不住，哪有工夫重新再认识另一个？

　　换两三个字，无力改变一生。

　　一名苦命女子，初恋因身分悬殊，遭男方家人棒打鸳鸯，诞下私生子，她改了名字。後来遇人不淑，被骗财骗色，沦为伴唱，为免辱及家人令儿子蒙羞，又改名换姓。谁知患上乳房肿瘤，男友拒婚，大受困扰，终走上自杀绝路。命运坎坷，令人欷　。

　　她两番改名，也希望有线生机，把过去的苦难「甩」掉了。

　　原来命中注定，如何「脱身」？

　　——「名字」不重要，是「八字」主宰一切。

　　如果你叫做「垃圾桶」，但命途多舛，充塞废物和噩运，你承受不了，所以改名「果皮箱」，改名之後，你以为你真的高贵优雅得只接受果皮吗？你的运气好转了？

　　那些言不及义的高官，同帮人改名的相士有甚麽分别？

　　高明的改革，大概是从根基上把「八字」给修订一下，扭转乾坤，才是一项成就。
