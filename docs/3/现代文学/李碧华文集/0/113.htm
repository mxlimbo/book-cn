唐墨抑或唐砚


　　在《饮膳札记》上看到，台湾的尊贵美食乌鱼子，日语汉字，取为「唐墨」。——「唐墨」是个美丽又典雅的名字，当中且有故事感。

　　但乌鱼子（乌鱼的卵巢）盐渍、晒乾、压扁後，成左右对称暗红色椭圆形厚片状的乾货，像大脚板，像肺，不大像墨，点水磨成芳香光采墨汁的「墨」，呈条状，再扁，也是长形。——我以为乌鱼子称「唐砚」，还神似些。

　　我吃过两次乌鱼子。

　　第一次是朋友送的，拎着两块大脚板，只晓得基本皮毛的烤法，怕腥，又忘了帮它「宽衣解带」，把身上那层薄纱（薄膜）给挑开撕下，但吃得很「好玩」，一来贪新鲜，一来为虚荣。

　　真正吃得好，在台北由会家子调弄。切片、蘸酒、以青蒜红椒萝卜……佐食下酒，果然是上佳的美食，像形容不出的带油咸蛋黄。

　　原来最重要是火。

　　不是甚麽电炉瓦斯炉BBQ。而是炭火，还得有距离，温文地，微弱地。反覆、慢烤、变软、冒泡、渐熟、渗入酒香、微焦……。然後热吃。

　　有一种欲拒还迎，若即若离的煎熬和忍耐，等、等、等，终於可以了。

　　既是磨墨待书，又似动情思念。付出时间和闲情，还得有人，才可共醉。很奢侈，对不对？
