520，7456，886


　　继「我爱你」电脑病毒，又有「我爱你」淡菸袭港。

　　台湾禁售这滤嘴上有鲜红心形图案的淡味香烟「520」，谐音「我爱你」，以十多岁或稍长的年轻烟民为目标，少女也因它可爱而「中招」。立法院下禁制令後，存货囤积，所以公卖局决定往外推出，向香港青少年招手。

　　本城青少年已「进步」至深圳大啪fing头丸，并顺道寻欢了，年龄低至十三岁的也有。那些甚麽淡菸，难道用来引诱小学生？

　　一句「520」，竟可温暖新一代寂寞的心。——是因为，在现实中，得不到亲　的抚慰吗？他们要靠一根纸卷的淡菸，一座冰冷的电脑，来寻找「爱」。

　　网络上，中台最流行的　称是「哥哥」、「姐姐」、「妹妹」、「弟弟」（GG、JJ、MM、DD）。「伊妹儿」是e-mail，「美眉」是漂亮的姑娘。可以用「BB」代表拜拜，或不肯拜拜的Baby。

　　还有非常的「嗲」，人人撒娇地道：「不来乐」/「我要去吃饭乐」/「7456气死我乐」/「明天找你乐」/……

　　那个「乐」，就是广东懒音「叻」，在电视广告可以欣赏到：「喊叻」、「咪喊叻」、「即系喊叻」。之类。DJ和艺员的口头禅。

　　——这是新文化：886（「拜拜叻」）。不觉得肉麻吗？
