「无意地」中奖最开心


　　我没有中奖命。——但如果中奖，我很希望是「无意地」便中了大奖。没有苦心孤诣，并非逢赌皆博，而是，中了？自己也不知！忽然间，无限惊喜，快乐必是双倍的。

　　像美国纽约的「千禧超级大抽奖」，彩金高达一亿美元，一次过提取并除税，也有四千四百万美元（约港币三亿四千万元）。结果公告後，这笔全美有史以来最大的彩金，却一直无人认领。

　　原来中奖的六十六岁平凡老厨师买了彩票，但忘记此事，到了新泽西州探望朋友。後来听说头奖得主仍未出现，他姑且查看，赫然中奖。为怕老眼昏花看错了，还先後叫来妻儿也核对一番，才知千真万确。一家人狂喜不已……。

　　领取奖金、退休、接受访问、庆祝、送礼……，这是後话。

　　但无意地，一刹间的「迷惘」和「快感」，最好的毒品也提供不到。六十六岁才中奖？晚是晚了一点，如果二十六、三十六便已中奖，即使奖金减半或四分之一，亦更美满。——不过，中好过没有中。

　　我就没中过了。甚麽？你也一样？真是同病相怜。只好继续工作。

　　但这位突然提前退休的厨师伊利伯伯也说：「我仍旧每天凌晨两点半醒来，不过如今醒来只是望着窗外，不知该做些甚麽。」——继续工作吧。
