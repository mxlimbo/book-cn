素玉怨


　　凄冷寂寥之夜，远处传来猫儿叫春。阿玉心生感触泪流披面。三姑向伏在枕头绣枕上的阿玉道：「乖女，阿妈都知你委屈——」

　　「呜……，我真系顶得好辛苦！初头重要对人欢笑，扮晒无 死撑南哥……」

　　「 本来谂住有主好人家，门口又高狗又大，下半世唔使忧啦。」

　　「阿妈，点解我条命　苦？一个冰清玉洁女子，点解要成为呢个　顽固制度下牺牲品？」泣不成声的她忽愤慨道：「过大礼，但未入门　。南哥肺痨无得医，瞒着我　耐！　有事，理应由　个人承担。　家作为老公　能力都所剩无几　，好应该退文定解除婚约，唔好霸住个位，等我同第个——」

　　「阿玉，亲家老爷　睇你呀！唔敢当，坐啦。」

　　成老爷无奈：「阿玉，生仔唔知仔心肝，阿南系俾大夫捉住把脉，我先知　医好都　药费。　，封建制度唔容许话分就分，阿南起唔到身，我　都预备只生鸡同你拜堂，吹吹打打，做到好好睇睇为止。」

　　「鬼叫我当初唔查清楚　祖宗十八代？验晒身照过肺？……」

　　阿玉思前想後。这是命运的安排。就算千拣万拣，他日良人亦会负心，食鸦片，或到塘西行搞生花柳。　！

　　「又边有写包单呢？但系我重有道德、诚信，同埋贞节。守三四年生寡，第时再　过罢啦……」

　　阿玉决定挑战命运。
