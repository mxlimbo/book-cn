我们的阴乾生涯


　　「阴乾」是一个新近流行的古语。

　　因为「锺庭耀事件」，我们不但见到很多嘴脸，还增加不少常识。不赘。

　　「阴乾」之所以被广泛报道，据说是财政上某种很难明言的威吓，俗语连翻译也费周章。不赘。

　　大学副校长在回答一些关键性问题时「失忆」，又认为「校长『阴乾』你是一个非常粗俗的说法」。不赘。

　　——我只是想说：

　　「我们对『阴乾』二字一点也不陌生。」

　　近年来，大家的荷包已遭「阴乾」。你看「傅姿灿」的巨着《老懵董》多有共鸣便明白了。此外，在高科技电子化的社会，很多传统文化、礼貌、感情、伦理、书法、生活质素和「人」的质素，也在「阴乾」之列。

　　最痛苦的，我们的衣服，其实大多是「阴乾」的。

　　你不发觉吗？在人口密集的住宅区，湿衣服很少有机会在晒架上，接受大太阳的洗礼、消毒，最後乾爽得带着香气（对，太阳原来是「香」的！）。经日晒的棉被，温香软玉；经日晒的衣服，特别舒服。烈日下，刚洗好的湿衣，一阵子便乾了，好不痛快。但并非很多人拥有天台、露台、天井、屋後空地……。用的是乾衣机，强暴地烤乾。晾在稍通风的地方，阴乾。遇上潮湿天气，还有味道。

　　「阴乾」真的好讨厌啊。
