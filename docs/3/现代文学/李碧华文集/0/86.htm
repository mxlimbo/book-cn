横刀夺爱，生离死别


　　不知道只能获准「见最後一面」的人和猴，是否已面临一个句号。但在电视画面和各报道中，见到九十高龄的陈伯，与相依为命的小猴「金鹰」被迫诀别，心里很难过。

　　陈伯养猴卖药（疳积散），走遍九龙大街小巷四十年，也不过为了自食其力，老有所依，不是今天才发生的事。同住共食，训练有素，感情深厚。他说：「我锡『金鹰』锡到上头壳顶。」所以失去了它，他茶饭不思，消瘦得连裤子也松到穿不到了。

　　追溯原因，是五月九日，渔农署接获市民投诉，指陈伯饲养一只猴子，在九龙城某街头献艺。署方人员当日下午即往上址把猴子带走，还考虑起诉老人，後来因「同情分」，撤销起诉，但「金鹰」则被送往遥远外地「安置」，不会发还。老人思念再苦，再憔悴，再依依不舍至昏晕倒地……，总之，人和猴，将被拆散。

　　这不止「横刀夺爱」，还是「生离死别」。想不到政府部门的办事效率那麽高，对两个没有杀伤力，也对社会没滋扰性的主角，那麽不近人情。

　　去电「投诉」的市民，只「一念」之间；执行的官员，得硬石心肠，结果令事件如此悲哀。他们回想起来，心中是否有一丝牵动？

　　我想代「求情」。希望人猴团聚，度过晚年。
