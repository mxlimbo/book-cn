很难因不服输而蠃


　　电脑最可恨之处是无故中毒或失灵，甚麽也不能做，即使已在死线，——你就死吧。它冰冷的心肠不会施舍丝毫的同情，也无妥协途径。

　　电脑不是我的工具，怕被它牵着鼻子走。不过，它是身边同僚的工具。——我发觉他们遇上这种情况，只能急召维修，或设法抢救以免「全盘失忆」。除此之外，便平白可得一个停工的机会。相对而言，他们的小休时间比我多些。

　　电脑那麽横蛮，可是我总同它战斗至最後一秒。

　　某回Y帮我做一张设计图，不知如何断了线或甩了线或搭错线，连基本动作也办不到。Y试了几回，又打电话问专家，束手无策。他说：「有以下办法：（一）把东西拿回家做，然後速递来。（二）明天再来一趟。（三）另找人家的电脑来做。」

　　设计很赶，我要在他身後马上决定修改。第二天二人都忙没工夫再来。一时间不知附近谁有空出来让你用的电脑，人家也有工作。

　　多试一次，不行，放弃了。我真不甘心，难道输给它吗？不，「请再试一下，或者可以呢？」……

　　——是的，结果忽然又恢复功能了！但这份坚持，非常累。而且不一定每回都因不服输而蠃。

　　只是意外的幸运。人生很难打性格牌。
