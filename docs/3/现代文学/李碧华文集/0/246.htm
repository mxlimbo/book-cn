究竟谁是保母


　　北京有则令人啼笑皆非的「新闻」，出於「北京电影学院保卫处」。

　　原来自《还珠格格》播映以来，由各地出逃，跑到学院，赖不走的「小燕子」迷，前後不下三十人，最大的十九岁，最小的九岁，全是女孩。保卫处的队长和员工，不得不轮流给她们买饭吃、安排住处、联络家人……，还要捐出路费，等她们点头答应回家。

　　有的肯「合作」，省点麻烦。有的坚持不走，他们说：「真是招架不起！」

　　最近的一位赵薇fans，十三四岁，从湖北省荆门市偷跑出来。这小姑娘三岁时父母离异并双双抛弃了她，只好跟随又聋又哑的奶奶生活。小学毕业後，经人介绍当保母，在雇主家看了电视，迷上了：「小燕子又勇敢又可爱，找到她，一定会帮助我的。」

　　又央求道：「我可以当她保母，搞卫生，这样便天天见到她了。」

　　已经一个多星期，她当然没见。北京也下起雨雪了，——保卫处担忧得很，不知要「照顾」到甚麽时候？

　　一个「童话」，捧红几个年轻的明星，诱拐一众无知小女孩，也愁煞电影学院的大男人，——他们才是保母！

　　但有精神寄托的人，心底快乐自己明白。只恨我们太老，铁石心肠迷不起吧。
