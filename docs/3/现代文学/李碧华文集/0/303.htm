吃鲸像吃人


　　全球大概以日本人最嗜吃鲸鱼。他们不但公然捕鲸（因为国际捕鲸委员会在八十年代已投票通过禁止商业捕鲸的协议），还把鲸鱼肉拿到市场出售。

　　在日本有一家专门店，看他们把鲸鱼身体的各部分做成刺身、寿司，或火锅用料。鲸鱼的肉、舌头、唇、鳍、触须、乳头、乳房、阳具、阴部、内脏……，均分门别类，全部可吃。

　　鲸是哺乳类动物，是动物界中最高等，人也属於这一类。——不过在日本，连小孩也享受鲸鱼便当。

　　当他们吃着小鲸鱼时，会不会想到，鲸是有强烈家庭观念的动物？

　　小鲸鱼诞生在水下世界，不懂得用肺呼吸时，是母鲸用头部把它托到水面学习。若它是一个死胎，或遇难身亡，母鲸会一直抵住其背部，直至躯体逐渐腐烂了才黯然放弃。有时，无微不至的关怀照料，往往暴露了目标，遭到捕鲸者杀害。

　　日本有个很受欢迎的男女「相亲」电视节目，也用了「红鲸团」的名字。但它们还是爱吃多情的鲸鱼。
