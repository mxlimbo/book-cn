记忆多，回忆少


　　有次写个小说，其中有少少关於手机的「技术性」问题。基本上我的手机只是工具，这方面知识好贫乏，只是做足资料搜集「扮」识。问各方损友，但不能给我专业答覆，也懒得「暴露」其实他们比上不足，比我有馀而已。所以指使我去深水湾或铜锣湾的店子问人。

　　我说：

　　「你想我去到被人笑死吗？」

　　如果自己问得幼稚，又无帮衬，虽然不致像港客在深圳「罗湖城」般被恶人殴打：

　　「你今天一定要买，否则休想返回香港！」

　　但那些机佬会令我丧失自尊的吧？

　　C已经嘲弄：

　　「手机若用上超过一年，已是出土文物了。」

　　我决不是潮流的奴隶。不过我知现今即使烧衣给仙游者，也有手机，且是新型号。机械冷、硬，且愈来愈小，容不下人的「长情」，决不能视作贴身的纪念品。它不是一块玉。人们对手机没多少回忆——虽然它们自诩能记忆很多东西。

　　我终於想起，不如打电话给天天推陈出新网络版「专业人士」。接电话的刚好是Tony。幸好他没有笑我笨，还兴致勃勃地「附送」一些情节发展的可能性。转数快，那是今天年轻人的心得。谢谢。
