为一本书坐牢


　　我还是第一次听到，向图画馆借书迟还会遭票控入狱。——虽只入狱一天，但坐这种牢？是不是好过分？

　　美国佛罗里达州一间公共图书馆在一周之内，严惩两名失魂鱼。

　　一名十九岁男子，欠下价值八十美元的书籍未还。一名二十四岁孕妇（七个月身孕），拖欠图书十六个月，又搬了家，没理会一百三十美元的欠书追讨通知，缺席法庭审讯，终於被警方拘捕，二人齐齐入狱。

　　幸好不是在香港。

　　本城图书馆的借还方式，其实十分方便。除了亲身办理，还可使用电话及网上续借服务，亦不必到借书的那间还，任何一间亦有分送的处理。

　　每本图书限定不可连续续借超过五次。若有人预约，也得还。根本没有机会「霸占」十六个月。除非你很卑鄙地偷窃或报失，付出代价。——但，还没有太值得的书吧？我常到的几间，图书故旧，都不够update，市面上一些新书和好书都欠奉。还是要买。

　　不知中央图书馆的阵势如何？

　　如果在图书馆「遇见」一本书，是100％爱书，因种种原因，你很想但无法拥有它。你会为它犯点小罪吗？坐一天的牢？留案底？……我实在希望知道，那是一本甚麽书？

　　如果是人，则有缘无分。留不住。
