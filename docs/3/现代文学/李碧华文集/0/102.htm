泼出去的水


　　忘年恋渐渐「明朗化」，是因为男主角有意无意的默认。一个人爱另一个人，关其他人甚麽事？有没有结果，当局者不在乎不介意，也不必交代吧。

　　年轻的一方，常是奋勇向前的一方。——没有甚麽是输不起的。他的「经历」便是日後的宝藏，无事发生，青春一样挥霍，当然是多采多姿更好。今天是明天的回忆。

　　接受访问时，被问到如何形容目前的感情生活？他道：

　　「覆水难收。」

　　据说就像当时他拎住一卷厕纸，把厕纸筒拉了出来，再也塞不回去了，不能控制一样。

　　番书仔乱用成语，真是有趣。大概不明白，这「覆水难收」其实不是甚麽好事。比喻（一）已破碎的东西不能重圆。

　　（二）马前泼了一盆水，势难重收，去妇不能复返，休妻理直气壮。

　　——但是，想深一层，也是错有错着，而且另有寓意。

　　倾心、倾情、倾倒……，是这一下动作。

　　我爱他，他爱我，总是迅即丧失理智，在一种迷醉的状态，认为在对的时间遇上对的人做对的事。投入感情，付出了代价，先爱了再说，必然把心一横像把水泼出去。泼出去的水，不问下落，也收不回来。就是这样了。
