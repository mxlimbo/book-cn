吃了一尾「活化石」


　　「活化石」先生：

　　对不起，

　　我吃了你。

　　──我後来才知道你那麽珍贵。

　　人是生物的一种，但人不算「珍贵」，只属苟且生存，大量繁殖，素质日差的一百五十多万种生物中之一。

　　但你，「活化石」先生，是中国176种珍稀动物里头，甲级受保护的「国宝」。同熊猫一样举世闻名。你的先祖曾出现在中生代的三叠纪前期至侏罗纪，距今约有二亿三千万年到一亿三千万年。──吃你的时候，完全不知道像吃一条恐龙。

　　事情是这样的：
　　我有一个朋友C，常常到深圳打高尔夫球，不知如何同一位食家赌一局，结果蠃了这顿。C只是告诉我：「星期五上去吃一顿特别的罕有的饭。」
　　「深圳？」我一听，迟疑。
　　「是你从没吃过的。──当然，我自己也没吃过。」C引诱我。
　　「不，我不吃猫狗，也不吃猫头鹰、果子狸、老虎、狐狸、穿山甲……」 「是鱼。」
　　「鱼？」我爱鱼。更爱吃鱼。
　　很大很大的鱼，可以做八道菜。鱼子酱是俄罗斯最贵重的珍品。一尾雌鱼一次就可以生产两大水桶黑亮的卵，相当五十头肥牛的身价。

　　席上，我尝着根本不需要任何调味（虽然有日式的青芥辣、豉油，和顺德鱼生的姜、葱、椒丝、油、盐……），已极其鲜甜的刺身，它是浅粉红色，肌理紧密，入口清爽，整整一大盘，只是八道菜中甘美的一道。

　　我才知吃的是你：「中华鲟」。
　　接着刺身上桌的，由食家简介：──鱼好，不必花巧求工。
　　天麻炖鱼头汤。（胶质十分软滑）
　　蒸鱼腩。
　　椒盐双飞。
　　鳞骨小炒。（骨也酥脆）
　　鱼肠蒸蛋。
　　露笋猴头菇炒鱼片。
　　还有，撑得吃不下的鱼骨鱼汁稀饭。
　　菜一道一道上时，我一点一滴的认识你。
　　如同大熊猫、东北虎、儒艮、白暨豚、蜂猴、金丝猴、白唇鹿、野　牛、朱灣蠦白颈长尾雉、扬子鳄、绿孔雀、大鲵（娃娃鱼）、丹顶鹤、文昌鱼．．．均属珍稀国宾，求之不得。

　　中华鲟属软骨硬鳞类，活在水底层。最大的可长达三米，重五百多公斤，肤浅的外号是「长江鱼王」，但尊称「活化石」，因为老资格，有江湖地位。

　　身体是梭形的，像一艘银灰色潜水艇。背部拱形弯曲，腹部平直，吻尖向前方突出，头部背面有光滑的骨板，侧有两条鳞软骨。他们说你有四只眼睛（其二已退化）──是不是年岁太大，视力欠佳，所以「四眼」？但依比例，你的眼睛实在「具体而微」。

　　在国内，你是人工繁殖研究所中的主角，为了不致绝种，後继无人，育幼成功後，全向长江投放，并因「国家第一类保护」，渔民即使发现搁浅的中华鲟，也忙着拖离险境，送回深水。怎麽敢吃？「你有什麽办法得到一尾？」我问。
　　神秘食家说：
　　「这尾不适宜放流。──它受伤了。」
　　「受伤？」
　　「是。即使『轻伤』，下场也活不了。只好吃掉。」他笑。
　　「哦？」我明白了：「『轻伤』！」
　　因为你「轻伤」。一切合情合理呢。
　　你身体「不太强壮」，怎能负担长途跋涉？太吃力了吧？

　　生活在长江，但江里出生，海里成长，是溯河洄游鱼类。──那是说，你们的卵由长江顺流而下，在江口附近的海域长大，活了十几年，便历尽艰辛，沿着固定不变的道路，逆流而上，千里寻根，非得回到长江上游祖先的产卵场，才择偶婚配，生儿育女，繁殖下一代。

　　一别十数载，你们重寻故里，不会迷途吗？为什麽那麽坚持？为什麽情深一往？你们小得看不见的眼睛，为什麽目光远大、奋不顾身？

　　──是不是你们的名字中，有一个「寻」字？所以不能辜负它？

　　原来，最高身价，最稀罕珍贵的生命，不外追寻原始感情。

　　你的辈分，是地球上元老派，吃你的时候，非循正途在市场买得（有钱也不易买到！），带着罪感，但实在──

　　我有点虚荣呢！

　　吃了「活化石」，不奢望长生不老，不过，日趋冷血的我们，希望沾染你的深情、坚毅、信仰。

　　「若寻找，便寻见。」──连上帝也这样抚慰罪人和失落者。

　　爱（吃）鱼的女子
