金不换的回忆


　　谢谢温增荣先生来信，告诉大家关於「九层塔」的特色。他是潮州人，已七十八岁了，看《吃掉九层塔》（五月三十日）之後，补充了宝贵的经验谈。

　　原来九层塔是年老一代潮州家庭不可或缺的调味香菜，乡间食物，自幼已吃成习惯了。因为可在家中花盆种植，作烹调之需。他们极少生吃。

　　（我们在越南菜馆时则当伴碟香菜生吃。）

　　九层塔也名「金不换」，是草本植物。名唤「九层塔」，并非一茎九叶，而是开花成串，花白叶绿。下层较大，花细小排列成圆形，一层层向上生长，逐渐缩小，至顶端变成尖形，虽不外二三寸高，似一座塔。「九层」非定数，不过形容「多」的意思。

　　如把它种在泥地，花可四五寸高。粗生，不需太多肥料，也长得茂盛。又名「金不换」可能是珍爱之意。

　　他们小时，待其成熟结子，是黑色坚实颗粒，小心逐串连茎剪下来，明春抖出种子再撒於花盆，不久长出新株。自然跌落泥中的，自己发芽长大。

　　乾花连种子成串，是乡间煮卤肉时加入的调味料，令卤汁更香。……

　　当我看着九层塔的「前世今生」，去年今年明年的乡间童趣，如见图画。有色香味。――想温伯伯也得享回忆之乐吧。
