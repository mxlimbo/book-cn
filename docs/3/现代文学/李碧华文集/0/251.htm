他癫你也癫


　　近日言行怪异出位的歌星，重拾昔日风华开演唱会，反应不错。不管是「支持」，抑或如老板所言：「观众的心理，是想看看他是否真癫！」，门票售出，双方都开心。

　　去看看一个人是否真癫，也是「娱乐性」之一。观众又是否全属「正常」？

　　其实人人都有压力，有潜在的疯癫因子。不过某控制能力强些，某失控，如此而已。不信？你试四下一望，总遇上一两个在角落喃喃自语七情上面嘴带傻笑目露凶光者。

　　有时我也遇上这些不知是「真癫」抑「假癫」的人，他／她一个人坐，但对面好似有倾诉和怨怼的对象。同空气做对手戏，喜怒哀乐，滔滔不绝，自导自演。好奇想听真点，已被恶毒的目光狠狠扫射，得觅地逃亡。

　　问过精神科的医生朋友，都市人十分之一有躁狂症（已有人办）、抑郁症、妄想被迫害症、精神分裂、强迫性神经官能症……倾向。也有连中二奖打　上，如「躁郁症」，一人分饰两角，他们一定要由专业医生治疗，局外人帮不上。

　　有人说：她一天到晚听到四周的人用暗号来通消息，要迫害她，全港都是同谋……云云。我没有办法令她相信：势利的香港人合力来迫害一个小市民有甚麽好处？有甚麽利益？他们才不肯弹药呢。
