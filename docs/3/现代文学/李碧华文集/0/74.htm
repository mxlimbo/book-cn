六月里的22℃


　　六月上半月，酷热得很难受，33℃（部分地区甚至更热）的高温令人昏眩。受热浪袭击，不但冷气机耗电超负荷而停电，还导致有人中暑死亡，和不适晕倒。

　　谁知六月下半月，狂风暴雨加台风，还来了个冷锋。有几天，气温低至22℃。是自1886年以来，最凉快的六月天。

　　这几天，真舒服。

　　六月的风，一向惹嫌：暖热、潮湿、闷。还似乎有怪味。但我们无法逃避，正如逝水年华，随风而去。来了，走了，夏天又过去了。——看，只有岁月，我们完全明白。它的结局，它的下场，明年亦是一样。

　　但措手不及的「冷锋」？气温一夜之间相差十度以上？还是第一次感受吧。一百年之前，你我在哪儿？当年的骤冷，又是否心头一凛？

　　雪中送炭当然一番好意，酷暑送凉何尝不是一份礼物？

　　晚上吹来一阵天然清风，叫人打了个寒噤。但我们翳闷汗濡的心情，彷佛得到薄荷香味的冷敷，变得紧密清爽。上着还牵被，睡得特别沉。不愿起来。

　　——明知它好，寻找不着。只可偶遇，无法强求。来去匆匆的22℃，甚至不需要你怀念，已无情作别了。
