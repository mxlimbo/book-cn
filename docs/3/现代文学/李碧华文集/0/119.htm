一切以分贝定夺


　　我竟然参与了一个永远也没想过会到的「盛会」。

　　在铜锣湾，忽下了场急雨，到时代广场避雨，顺便到city' super shopping。谁知正举行日本人气偶像的签名握手会。宣传《另一个天堂》这电影。

　　在台上的有江口洋介、柏原崇，这两位我认得。还有一位市川实和子，不知是谁。还有一位，瘦瘦削削，长发蔽目，更不知是谁。——而这位哥哥，却得到最多的「夺命狂呼」，有少女公然索吻。

　　雨不停下着，三四百个fans，穿校服的年轻学生，少男少女。镁光闪闪，更多是用摄录机，拍下偶像一举一动。尖叫呐喊直至完场。

　　忽地有人朝一个方向狂奔。羊群效应，又集体狂奔，看来是「追车」，但影儿也不见。追呀追，结果一个两个……，滑得跌倒，急忙爬起，奋力再追。

　　铜锣湾一隅，由震动至人去台空，回复凄冷雨景，——不过是大半个钟头的事。

　　江口洋介和柏原崇，一度是fans疯狂簇拥，忘情哭喊的对象。这一回，风头都被新上位的Sugizo（《月之海》摇滚歌手）抢去。

　　那种「势利」十分明显，坦率。完全不顾台上人的感受，一切以「分贝」定夺。也很难受。

　　江口洋介早退。柏原崇笑容很少。——也不过是数年间的事吧。
