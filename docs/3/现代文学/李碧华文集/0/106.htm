空中奇案


　　看到一则新闻，疑点重重，令我百思不解。

　　美联社引述印度一份报章报道，印度航空公司一架由新德里飞往香港的航机，今年一月发生了商务客位女乘客在飞行期间被迫「罚企」事件。报载过程是这样的：――

　　（一）航机飞行期间，女乘客被一名空中服务员要求更换座位。

　　（二）女乘客按指示站立。

　　（三）不获安排座位。

　　（四）她被迫站立在原位的左面。

　　（五）直至航机抵达香港，才得座位。已历时六个半小时。

　　（六）她在机上投诉，不获理会，还遭到空中服务员的粗鲁对待。

　　――真是奇怪极了。

　　为甚麽乘客被要求更换座位？是座位的问题抑或乘客的问题？难道她无登机证？

　　为甚麽没安排新座位她便站起来？而且一直乖乖站在原位的左面？没位坐，应坐回原位等。在机上站立多危险。

　　为甚麽她竟可委屈苦忍六个半小时？是理亏吗？是怯懦吗？是精神病吗？

　　空中服务员胆敢在众目睽睽这样粗鲁对待一个女子？甚麽原因？即使无票也可补票呀。

　　其他乘客的反应呢？无人仗义挺身而出？

　　印航应该交代一下此「空中奇案」。说不定案中有案，是个曲折故事。
