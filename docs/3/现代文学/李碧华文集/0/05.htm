一夜之间不理我


　　走在街上，有点寂寞。

　　路边争妍斗丽的宣传告示牌当然已作废，大局既定，无人来讨我们欢心了。好怀念刚刚过去的一阵子，候选人是多麽热切需要我们，街头总有各团各组连同主角，打扮光鲜、态度诚恳、笑脸迎人。尤其是冲刺前夕，皆轮班出动，花车游行。（出尽法宝来追求那吊起来卖的我。）

　　除了爱人，就数候选人最温柔体贴，这些甜言蜜语，多想一直听下去：

　　「衷心感谢你支持。你拣我，无拣错！」

　　「知您心，代表您声音。」

　　「系时候换人啦！」（我们心思思想「换画」。）

　　「你好，请你记得我！」

　　「食着晏未呀，我都未呀。」（自恨扮　没接着说：「一齐　？」）

　　「好声呀，拎住　多　。袋番屋企慢慢睇啦。」

　　「十载风雨十载情。」

　　「我用诚信保证，你可以放心。」

　　忽又只闻其声不见其人，原来站在敞篷车中，向大众公告示爱：「我好挂住你　呀！」（我都系呀！）

　　感动到几乎爱上你了。――为甚麽？为甚麽一夜之间，你得逞了，或失落了，便不再理睬我们？（又失恋了。）

　　按摩目的是松弛。一天未了，一天仍极度紧张。
