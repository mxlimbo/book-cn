女 巫 辞 典


　　「贪婪」：一种不必花本钱而令人上进和美丽的推动力。

　　「尸肝移植」：找到一位高龄而疏爽的大老细照顾。

　　「活肝移植」：找到一位年青而疏爽的大老细照顾，但手术过程风险极大。

　　「心肌栓塞」：一个人已不爱另一个人，又未遇上新欢足以变心时的过渡阶段。

　　「质感」：马「英九」和陈「水扁」的差距。

　　「重新考虑」：为自己一个错误的决定遮丑。

　　「合法」：
（一） 与法官旨趣相投的一刹。
（二） 对方明知你非法也没有行动。
（三） 律师在骗你钱之前的甜言蜜语。

　　「设计师」：把女人装扮得很漂亮让她出去见世面的「姊妹」。（多半是无杀伤力的基佬。）

　　「发型师」：同上。

　　「名媛」：把男人装扮得很漂亮带他出去见世面的「姊姊」。

　　「Ball场」：女人们穿上肚兜社交的高贵场合。

　　「安慰」：当知道有人比你更不幸。

　　「祝贺」：一种皮笑肉不笑的妒忌。

　　「妥协」：大婆止，二奶回升。

　　「高纤低脂」：连「伟哥」也无法达致的理想形态。

　　「流动炒作」：媾（仔或女）。

　　「全幢招标」：结婚。

　　「呆坏帐」：做完唔俾钱。

　　「外来因素」：所有不肯自我检讨的乌Where大婆的唯一藉口。

　　「明春复苏」：每年年中开始的盼望和口号。和壮阳药广告。

　　「一脱求生」：三四线艺人最易陷入的，「求生不得，求死不能」的僵局。

　　「师徒关系」：在娱乐圈，涉及名利和矛盾前的一种邂逅。

　　「经理人」：高级淫媒。拉不到客是失职；太多客你的「女」又肉刺俾佣。

　　「安全套」：等於权势，不能用完一次再用。

　　「灵柩」：厚实笨重，长方形的新子宫。

　　「回忆」：在追想过去之际，不忘把一些从未发生过但又相当甜美的配料加进去，令自己充满快感的思想自渎。
