不痛不痒不骂


　　「静」，不但恐怖，还有从此被遗弃的寂寞。你看弃妇、弃婴、弃猫、弃犬、弃男、弃父、弃母、弃卒……，连骂的人也没有。

　　女人向闺中好友诉苦：「他连骂我也不肯——我们很和平，没吵架很久了。」

　　学生向老师提出：「你偏心，你只骂他，不骂我。」

　　风头趸生闷气：「上ball场比吃饭还勤快，为甚麽没人拍走光相来揶揄我？」

　　名人忍不住了：「这个那个他都为文大骂，总是轮不到我，真不够朋友。」

　　作家纳罕：「咦，写得那麽糟，都没读者『指正』？」

　　女同志抗议：「社会各界及传媒只顾讽刺基佬，根本漠视我们的存在。」

　　两个男人去寻欢，甲问乙：「你有向她招供吗？她准你去吗？」乙道：「如果她『不准』我就开心了。」

　　——不准，因为开始「管」他。若不在乎，不关心，不妒忌，不痛不痒，他天天鹟骨夜夜春宵，交两个深圳女友，间中还包个三线小明星……，不停在她跟前明示暗示，有甚麽意义？在女人而言，街外人啪了100粒fing头丸般癫狂，又有甚麽反应？

　　大家应该明白：有noise，才有「存在感」。此外，「肯骂」，「肯捱骂」，是一种双方皆付出的代价。
