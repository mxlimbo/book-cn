道理龙加菜根谭


　　风气所及，那些被怀疑或明确地向记者爆料者，同当事人关系愈来愈密切。——情侣、性伴、师徒、谊亲……，有些相识十年，有些暧昧得来呼之欲出，有些是事业拍档。

　　爆料，其实即是「出卖」。为了钱、名利、博见报、泄愤、报复、达到某种目的。即使甚麽好处也得不到，还是出卖了再说（「知道」也是一种虚荣）。知道得多，自因为信任，不设防。当初必然融洽亲密，否则也成不了「密友」，一旦反目，目的竟是令对方「翻不了身」，想想也心寒。

　　认识到这样的人，是个人不幸，眼光不够。

　　近日狄龙亮相，中英双语，道理滔滔不绝。有访问称之「维园阿伯俊朗版」。没想到当年寡言沉实的大侠，出口都是四字真言。但他有感而发，还送了一本《菜根谭》开解沮丧的陈宝莲。久违的正气。他本姓谭。关於《菜根谭》，看似好浅，做起来不易。但见人与人之间关系，腐败不可互信，不如抄些简句，介绍此明代奇书吧：——

　　「路要让一步，味须减三分」／「交友须带三分侠气，作人要存一点素心」／「凡事当留馀地，五分便无殃悔」／「藏巧於拙，寓清於浊」／「处世要道，不即不离」／「守口须密，防意须严」／「人生本无常，盛衰何可恃」／「祸福苦乐，一念之差」／「恬淡适己，身心自在」／……
