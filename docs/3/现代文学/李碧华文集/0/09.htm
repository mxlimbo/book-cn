人的素质


　　在中港追寻十五岁智障男童事件中，我们可以看到「人的素质」。

　　有人义助、悬红、带同子女上大陆代寻人、捐款、送月饼……，即使帮不上甚麽也为父母亲友担心和安慰。

　　父母都是小市民，母亲还不良於行，他们把一个智商只有两岁的孩子带大，可想而知好辛苦。不嫌不弃，失踪之後焦虑四出寻找，「踏破铁鞋」，电视画面还见他们长了水泡伤痛的脚，――这种原始的关爱团结，令人感动。

　　但另一方面，因消息传扬，亦惹来大量恶作剧的虚报、歹徒垂涎勒索、骚扰、恐吓、索钱求神问卜……。完全罔顾伤心人疲於奔命被玩残。这些人为了钱，甚麽也做得出。

　　同种同文，一河之隔，为甚麽某些人的素质，竟是这样？

　　「素质」（即赌王名句「质地」），不关乎贫富贵贱智愚美丑，是人的本性。但如果很大部分都是同类型的，便基於民生教育了。

　　黑与白接近、掺和，受害的永远是白。

　　我们看报，深圳轻微的车辆碰撞可演变成买起人的血案。还有，凶杀案中的歹徒，入屋不由分说乱斩，老人倒地後，他们还在其脸上划上数十刀以查验是否死亡。……

　　十分恐怖。
