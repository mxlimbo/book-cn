何必提供「联想」


　　市面上新兴的排油减肥药引起公众注意，声音很多，有人说它好有效，有人认为好烦恼……。——其实，它最核突之处，是提供了「漏油失禁」的联想。如果对方告诉你他／她正在服用这排油丸，你脑海中不免出现了成人纸尿片或卫生巾吸收漏油的「画面」。

　　某强调，没那麽严重，不是时刻漏油的。她说：「吃了含脂肪丰富的食物，才感觉到肠道不吸收，跑了出来。」又举例：「那天去自助餐，吃了几只肥美的生蚝……」

　　拜托拜托，不要再形容了。

　　那天看娱乐版，还见有女艺人自招，在游泳时漏过油。这同月事期间下水一样「污染」呀。你看，一众细意打扮，衣饰趋时的摩登女性，或身居高职，才智兼备的都会男士，行行街、开开会、食食饭、睇睇戏……，原来暗中有「後遗症」，试问大家怎能接受这丑恶的事实？情愿不知。——又没有严刑逼供，当然含蓄些或否认上算。

　　一般社交还罢了。同亲密的人烛光晚餐，喁喁细语，拥抱亲热，共度良宵……。解除武装——？漏油失禁？

　　世上还有比这更杀风景的恐怖事件吗？

　　即使有神效，三个月减二十磅，自知便好，过程是不重要的，做人何必太坦白？
