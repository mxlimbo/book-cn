寻找鸡蛋衣的女人


　　日本流行一种橡皮一样的保湿面膜。粉橙色。「OrangeMask」卖点当然是水果。（看来还会出士多啤梨、奇异果、桃、苹果、西瓜、柠檬、柚子、柿子、柑、凤梨……。反正能吃的都行。）

　　一般最能哄人的面膜是湿的，「多汁」。只消提到「骨胶原」，便有女人相信。——如同「我很挂念你！」一样。

　　医生明明说，「骨胶原」并不可能透过皮肤渗透，吸收进体内。但这片面膜又明明在你脸上渐渐的乾了，「汁」不见了，它一定是跑进皮层，叫肌肤回复活力、弹性。青春紧密。但橡皮一样的橙面膜，既不见橙汁，也没有橙肉，还是乾黏的，只有香味，香味一向虚幻。它比较适合在浸泡温泉、风吕浴时，水气氤氲中，敷十五分钟以上。

　　不必专家，我也可告诉你：「最好的水果面膜是现打鲜做」。新鲜水果榨汁搅茸，混着粉、蜜糖（之类），成糊状，敷好待乾，比任何「高科技产品」都好。——只不过费时费工夫，人们宁买高昂的次货。而把水果吃掉，已收同样效果了。「面膜」是比较虚荣的过程吧。

　　但我们在大都会，实在找不到一个没用过面膜的女人啊。没一张脸是鸡蛋，弄不好还是皮蛋，统统都误会终有一天，像剥掉鸡蛋上一层衣。
