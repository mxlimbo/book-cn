人肉汤汁


　　最近柬埔寨金边某家面店附近的垃圾站，工人处理废物时，发现两个装有人类残肢的胶袋内藏带刀痕的趾骨、腿骨、肋骨、一个肺和少量红棕色的头发。

　　警方因切割「恰到好处」，怀疑出自屠夫专家之手。推测受害者为一外籍少女。被肢解、煮熟，用来炮制汤面给客人吃。

　　这也不是新闻了。在一本《日本真人妙事实录》中看过，某社团大佬和四名手下把一名叛徒杀死了，放血斩件，用车运走碎尸。——谁知黑社会也有乌龙王，竟剩下一只断掌。留守大本营的两个男人慌起来，手足无措，最後给扔进一个汤锅，与猪骨头熬个稀烂，发出阵阵肉香，吸引了无数来光顾吃拉面的客人。二人不肯卖，被投诉，揭发凶案。

　　中国人在这方面最有经验，古时候，舍不得吃自己的骨肉，便易子而食。在战争、饥荒、文革期间，很多人吃过人肉。据说，人肉除了鲜美，还带奇香，我有个故事收在《吃眼睛的女人》中，便用人肉熬了一桶卤斗去卤鹅，延伸数十年……。
