「减的艺术」和「加的艺术」


　　那天我们一行九人到「志莲净苑」的仿唐佛寺组群叁观。原来的领队因为广告公司临时要开工，所以他委托我代领队。同去的是我们尊敬的艺术界前辈，她宽心、舒畅，我们也很快乐。

　　其实前一阵也到过志莲。那时还在建设组合中。每回再去，它都多完成了一部分。大概在年底，便可竣工。现在仍未正式开放。

　　我们下午在各佛殿、长廊漫步，欣赏不设一根钉的唐风建筑，心静自然凉。池中还有紫色和绯红色的睡莲。释迦像、直棂窗、千岁的树、仍在呼吸的桧木巨柱、斗拱、莲灯、水月观音、樱木挖空造的鼓、经幢（不分年龄辈分都去沾点经幢上的尘呢）、鸱尾、……。

　　有一座汉白玉卧佛像，在四川某山顶找得时原重二十吨，塑像後只重十吨。雕塑，是一项「减的艺术」。

　　他们说，农历四月初八佛诞日，亦称「浴佛节」。前夕（五月二十一日），释迦佛真身灵牙舍利将由专机由北京运抵香港，一连七天，供奉於红　香港体育馆，公开让市民瞻礼叁拜。

　　吃一口斋礼一下佛行一下好，不自觉种下小小善因，将来某一天忽有所悟，或逢缘，才恍然。──说来，这也是一项「加的艺术」了。
