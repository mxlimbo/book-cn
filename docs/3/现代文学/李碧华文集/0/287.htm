先 看 什 麽 信 ？


　　我喜欢人不在港，但信箱塞得满满的──虽然不一定都是「有意义」的信，但这种丰盛的感觉令人有安全感，和存在感。堆在信箱中的信件，抱个满怀，先拆什麽呢？也许，这便是你的性格：

　　先看熟悉的字体、期待的私人信件、朋友的来信……，因为渴求沟通，希望藉此得到安慰。毕竟有点疲倦。

　　先看账单，先了解财政，这不需动脑筋，但同切身太有关系了。这种人乐於货银两讫，平息纷争。一切若可以钱解决，二话不说，因为还有其他重要的事待办。

　　先看宣传单张，看完即扔，没有手尾。快速筛选，把精华和美点留到最後，先苦後甜。最初的「漫游」只是过渡。

　　先看收据，急於清楚是否有预期的结果？不爱拖泥带水，耻於求人。

　　先看「有体重」的礼物，不似信件单薄，是附加的惊喜。即使拎上手，知是书本、也明白是经过挑选的心意，感恩图报。

　　先回信？──太克己、自苦、怕得罪人了。只有机械人和电脑才那麽老土。
