操你妈的泼妇


　　在湾仔一个地铁站外，见一名女人大骂男人。

　　这里有球场，附近很多无所事事的阿叔阿伯，路过时都向男人行注目礼，甚至很捧场，当忠实观众。
女人是北姑（没有说是北菇鸡呀）。操流利普通话：

　　「他妈的！『媾』（call）你不回。手机也关了。找半天找不着，在这儿站了好久……，你怎麽失踪了？想甩我吗？」

　　「你他妈的拿不拿我当人？」

　　「我操你妈！」

　　「我操你妈你为甚麽不作声？你是不是男人？手机拿来，我砸了它！」

　　十分扰攘。

　　一个女警行过，看她两眼。稍靖。

　　女警走後，北姑继续大骂，完全不给面子，——我是说，她来到香港，男人的地盘，找上他了，在人家地方撒野，「不给香港一点面子」。若果这是「歧视」，过来踩场的北姑肯定受到众人目光歧视。

　　男人竟然窝囊得不敢吭一声，他是慑於淫威？抑或问心有愧？附近看热闹的年纪相若的阿叔阿伯，又羡又妒，——因为他们得不到一个女人当众大骂，人生未免有些欠缺，所以心底酸溜溜。表面吃吃笑：

　　「老友，活该！」
