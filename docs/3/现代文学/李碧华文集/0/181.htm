「十字帮主」应从政


　　看到一篇「十字帮主」（「穷、衰、贱、烂、惨、老、弱、伤、病、残」）阿杜的访问，好有趣。

　　杜惠东是「嘉禾」的宣传部经理，也是跨界传媒人，他居此位二十年，见尽兴衰，所以「没甚麽大不了的事」。他的专栏就像他的人那麽豁达好玩，有时又十分抵死。

　　阿杜也是个传奇人物。当过写字楼信差、印花厂配色师、行过船、在大西洋死里逃生、囚於南非黑狱、在纽约跳船偷渡上岸打扫跟车做赌馆……，几乎开了第一枪。後来才当了记者和电影宣传。年轻时，仍是斯文憨厚四眼仔，邂逅日本红颜知己，牵手合照，好不腼腆。

　　我与阿杜并不太熟，只在道左偶遇。――不过每次推动他写小说，他都不答应，说要退休才做。

　　他又怎有机会退休？「交代小龙女」的记者会，那天来了22家电视台的crew，200位记者，香港回归後最大最哄动的场面，便由他一手一脚办妥。成龙一边「演讲」，他一边满足微笑，发自五内。

　　如果台湾总统府秘书长章孝严早点认识阿杜，「留精不留种」而已，何致於狼狈鞠躬下台？一身蚁的李登辉，以及所有「尚待揭发」的风流下流史，都需要一个帮得手的宣传去化解。――阿杜先生，你从政去吧，他们比成龙更需要你！
