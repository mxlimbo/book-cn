幸好是一个噩梦


　　那天本来几个人约好了。谁知感冒，沉重的头疼得无法思想。打电话给A，A说：「B刚来电，病了起不来了。」

　　又道：「下午的一个会也取消了，因为对方病了。」

　　还有：「我自己也感冒。」

　　後来C非常抱歉地告诉我：「要去看医生……。」

　　个个不约而同感冒，谁也理直气壮，好了好了，大家都长眠不起吧，足足可以睡上二十四小时以上。――因为「断食」，瘦了两三磅。如果能量全消耗在病和休息上，未尝不是好事。

　　还有消耗在噩梦中。

　　不知如何，我的银包、电话记事簿、信用卡、证件……全被偷了，十分的「空」（像我的胃）。我报警，去卡中心报失，但连号码也忘掉。一看电话，键盘上不见数目字，全是ABCDE。四下一个人也没有，我还拚命打电话。一时不通。一时接通了听的人又不理我，粗暴地截线。竟不停地打电话，眼看天又黑了……。梦中一秒，也许是人间一小时，也许已经打了二十四小时的绝望电话。後来被人杀死了。

　　――幸好醒过来。幸好是一个噩梦。最凄惨，最恶劣，最伤心，如果末了可醒过来，有甚麽关系？

　　就这样，我的感冒好了一半。
