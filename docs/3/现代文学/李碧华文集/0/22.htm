最团结的豉油党


　　近日看到一则新闻：——某夜，土瓜湾海心庙公园女厕外，一群男人早已互相口传，几日来苦候目标人物。有些佯扮乘凉，有些装作打球。

　　九时许，他们忽一窝蜂拥至女厕门口及两旁，寻找有利位置偷窥。原来有位三四十岁，有精神病纪录的女人，经常赤裸在厕所内走来走去，嬉水和摇晃纸巾，自得其乐。

　　满嘴馋液的，大部分是阿叔阿伯，甚至印巴裔人。最高峰时聚集人数逾百人。

　　忽闻厕内传出「呀！」一声，众男即血脉沸腾，引颈翘望。愈来愈哄动，期待女人疯癫地出来门口做骚。

　　警员接报驱散人群，他们仍靠於铁丝网边不愿离去。後见女人被带出时已穿好衣服，十分失望，报以嘘声。

　　——「画面」是否好经典？

　　但我觉得疑惑：「究竟这闹剧历时几长？有几次？」

　　晚上九时，应该好热闹吧。（一）女人赤裸嬉水、（二）百几名咸湿佬围观哄动、（三）厕所是公众地方、（四）不止第一次……，为甚麽窥浴的戏场仍可做足起承转合？

　　是因为街坊不好管闲事？抑或那些咸湿佬太齐心合力，自成一党（豉油党！）？

　　原来香港最团结的群众，在海心庙。
