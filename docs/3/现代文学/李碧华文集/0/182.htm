十滴艳红的蔻丹


　　前几天报载一名患上「眼癌」的老翁，因受不了病魔折磨，自缢身亡。

　　我们听过肝癌、胃癌、血癌、乳癌、肠癌、鼻咽癌、前列线癌、子宫癌、皮肤癌、骨癌、脑癌、肺癌、舌癌……，但没怎麽听过眼癌。这实在罕有。

　　据医生说，眼癌即视网膜胚细胞肿瘤、眼皮的皮肤癌、眼白及眼球内的色素癌，或身体其他癌症经血液扩散至眼部所致。

　　印象中眼睛不会生癌，因为它并没有失控乱生变癌的能力。――但凡不会自行「分裂增生」的器官组织，就没有转化成癌的危机。人的心也一样。

　　因为心肌细胞只那麽多，输一个少一个，死一颗少一颗。没有後备，无法修复，永不再生。――故目前也没有「心癌」。将来若是上了报，医生又有解释了。

　　当最不可能生癌的部位，最不可能生癌的人，忽然有一天也逃不过，我们便明白，一切都是无常的。抗癌痛苦寂寞，要靠自己的意志力，艰辛熬过去（咒诅人癌病复发者好狰狞）。

　　有一位脑癌女病人，两个月前刚切除肿瘤，近日有复发迹象，手术和药物令她身体无力，浮肿变形。但她仍有愿望：「希望可以和家人去沙滩、山顶。」还让义工为她涂上指甲油带来喜气。看到十滴艳红的蔻丹，我有点感动了……。
