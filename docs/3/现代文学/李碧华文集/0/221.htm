找埋条数先啦


　　本届工展会上有阿灿的「栋笃酱」展销，而且生意不错。——但廖本人却因宣布破产，并无法与债权人（香港银行）取得协议，因此来不了。

　　据说身在马来西亚的阿灿稍後还上电视拍教人煮节目，亲自为其主题曲《永不言败》填词。歌词一般，但道尽心声：

　　「……当黑暗消逝，日头实出来。

　　拒绝换来施舍，扑散那旧框框，

　　肯砌实会蠃。

　　找埋条数先啦！」

　　——人最耿耿於怀的，相信是「仍未找数」吧。

　　虽然世上无耻的人不少，一身街数虱多不痒；有钱也不肯还仍吃喝玩乐；遇上债主脸皮奇厚面不改容；……甚至叫鸡也唔钱。各式各样的债，千奇百怪的债，有形无形的债。

　　任何稍有尊严的人，都不愿长久的「欠」。「欠人一文钱，不还债不完。欠人一生债，不还不痛快！」，这是古老的情操，即使活在现代社会，「终於清还」的心情仍是快乐的。所以连一首歌也自我提醒：——「找埋条数先啦！」

　　钱债易还，只要有心有力。较难还的，是血债。最难还情债。而花债，则是前生所欠，自己都不明所以。
