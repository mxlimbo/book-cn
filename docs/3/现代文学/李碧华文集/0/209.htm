鸟的故事


　　Y从北京回，除带手信外，还有些「顺口溜」和笑话。来自老百姓对政治的调侃，是民间幽默。《南华早报》报道，法轮功信众被捕，关入精神病院，被猛打「懵仔针」。——不过在大街小巷流散的笑话，不知始於谁终於谁。难道中国便是一座最大的精神病院吗？

　　有几个笑话关於李鹏。例如「降落伞的故事」，和「鸟的故事」。

　　後者说某天，李鹏和夫人到某城市巡游。该处得张起横幅来欢迎，但人民文化水平不高，所以把李「鹏」写成李「朋」。李鹏夫人见了，大喊：「李鹏的『鸟』怎麽不见了？」

　　於是市长连忙着人加上去。她见了，又大喊：

　　「李鹏的『鸟』怎麽那麽小？」

　　虽然，人家的「鸟」关你们「鸟」事？但面目模糊的老百姓便传诵得乐了。

　　——其实，这个笑话也可以「发挥」成几个版本。例如：

　　（一）市长着人把边旁加上去後，李鹏夫人见了，冷笑：「李鹏的『鸟』哪有这麽大？」

　　（二）李鹏没戴上老花眼镜，所以看後质疑：「加上去的是『鸟』吗？我看是『马』字。」——於是，除了赵高的「指鹿为马」之後，典故中同义词是「指鸟为马」。美点双辉。

　　你们有甚麽「发挥」？
