走到更不堪的地方去


　　我有点怀疑：一项调查预测，未来十年将有一成半港人有意移居深圳？按比例计算，人数超过一百万。

　　深圳是甚麽环境，难道大家不清楚吗？不要说五步一鸡十步一匪二十步一毒针了，这个充斥劣质假货和领牌恶霸的地方，大家嘴里瞧不起双腿却跟队上，年中总有好些人死在它手上，庸俗但安定的小家庭亦遭拆散。

　　港人北上，并非白鸽子朝亮处飞，而是生活迫人，走到更不堪的地方去。

　　大家并没活得更好。水准下降，沦落得成为「风气」，多麽唏嘘？——偷渡南来的先辈，没想过下一代如此不争气吧。

　　说香港物价太贵，日子难捱。北上消费，也不见得有着数。

　　随便说个笑话：——

　　香港女孩放假去深圳玩，贪东门一带shopping便宜，购物扮靓。她们去商场set头梳髻；「十五元？好抵！」於是坐下来弄头发，店主说不如影埋贴纸相，收三十八元。她们说好。之後，又被游说加发夹装饰，原来五元一个，加了五个。再染色、洒金粉……搞到像只夸张的七彩鹦鹉般。结果盛惠九十八元才放人。

　　深圳好杂？

　　——但，此处的腐败和贪婪素质，港人（特别是香港男人）催发功不可没。
