大鹅蛋和开信刀


　　看见一只冰绿色「大鹅蛋」。——那是新产品，流线型设计的雪柜。

　　首先，我是爱上了它的肤色。浅绿得好柔和细致，冰凉悦目，可以消暑。还有是「新概念」。

　　雪柜向来总是方正、灰白、板着脸。但这款却有「曲线」，形象十分鲜明，突破传统。打开之後，更加抢眼。为沉闷的生活添点情趣。

　　想换。看看价钱牌：

　　「$11598.00」

　　一只万多元的鹅蛋！

　　那些「顶天立地」的巨型四门雪柜，也不过万多元。而这个，设计费竟占了绝大部分。我们为一个漂亮的包装而付出，是否值得？它是单门的。隐蔽式门柄在高处，并不就手。融雪系统得手动。因呈蛋形，故实际容量打了折扣，放不了很多东西。不同冷度的食物无法分门别类。……

　　找出一些缺点，用来减低锺情。

　　问自己：

　　「究竟需要不实用的工艺品？抑或一个真真正正的——雪——柜？」

　　就像问在书展中狂奔的年轻人：

　　「究竟你们是来买书？抑或买刀剑？你们根本不写信、不收信、不爱信、不懂信——一窝蜂地抢购『开信刀』？为甚麽？」
