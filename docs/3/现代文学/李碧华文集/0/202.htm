三元一角


　　一回，见人拎一封信出来，问：

　　「寄美国。邮费多少？」

　　掌管「邮务」的女孩，用手一拈量，不假思索，马上答：

　　「三元一角。」

　　——她的手便是一个磅？

　　我觉得神乎其技，但磅一磅，寄美国，再查表，是这邮资。

　　天天拈量信件，大小厚薄，心中有数，差毫厘也分得出，不会超重欠资，也没便宜了邮局。她「一上手便知」，这是职业本能吧。

　　就像按摩师一摸，马上知道哪一块肌肉劳损，有感觉得到的「结」。算命的一摸，洞悉你前世今生。母亲一摸BB额头，说得出发热度数，一摸他大便，知道出了甚麽问题。我们一见广告牌或启事，必找得出错别字，还有拈上手的书，掀掀跳着看，已知是否好看？值不值得看下去？——这是保护自己。次货，怎肯浪费时间？因为时间不多。

　　但一山还有一山高，校对部的高手又能把我们的错别字迅速给找出来了，可见他们更加专业。

　　有损友笑，「资深」欢场女子，也是一拈量之下，已知男人有多长？有多久？有没有病？她们也像答「三元一角。」那麽快捷和专业呀。——说是笑话，不过阅人到这伟大程度，无论如何也不应是个笑话了。
