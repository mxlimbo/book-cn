过门都是客


　　据说，城中名媛阔太，不满名牌时装被受赞助的「小歌女明星仔」cheap晒，因而杯葛。云云。

　　这些报道太有趣。

　　真正的名媛阔太，有教养，说话得体（也要扮大方），怎肯口出恶言成为是非呀？得保持仪态。

　　藐嘴藐舌的，也许五十步笑百步而已。

　　若非心里有鬼，便是比人家先行上岸了，所以回头去瞧不起浮沉中人。物伤其类。

　　商业社会，谁付得起钞票，谁便可以入货。名牌也要做生意，「拣客」同自己过不去。——而且，怎麽「拣」？

　　小歌女明星仔固然无谓得罪，那麽城中的鸡鸭鹅女伴好友红颜知己，消费力强，亦不能小觑。大婆瞧不起二奶，二奶瞧不起三奶，三奶瞧不起四奶……，相对而言，四奶瞧不起三奶，三奶瞧不起二奶，二奶瞧不起大婆，也合情合理，她们都希望自己是「前有古人，後无来者」。

　　女人千万不要认为女人没品味，cheap。——也许娱乐大众自食其力，用血汗钱去买名牌的小艺人，还觉毫无工作能力的男权附庸寄生虫cheap呢。

　　「平等机会委员会」，你们又有着做了！

　　名牌，哪会轻易着cheap？

　　同理，名牌亦不能令人高贵，这是真的。
