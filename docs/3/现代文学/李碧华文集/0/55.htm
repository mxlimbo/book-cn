大大小小的礼物


　　有位男生说他最疯狂的一回是给喜欢的女孩送礼物。还不是「女朋友」，所以不清楚她生日。只知在四月。他想送一份礼物，又怕送错，变了贺「愚人节」，但也有可能就是明天。四月有三十天，究竟如何是好？

　　他买了三十份小礼物。然後花了一整天，亲手做了一个大盒子，分开三十格，放好礼物，上面再糊一层宣纸，好像以前的童玩，在摊子上付小钱戳一格，看看自己的意外惊喜。

　　这一盒的「心意」，也盛载了他的浪漫和周到，还有爱。一定令女孩很感动了。

　　她很感动。

　　但没有爱上他。——因为没有那感觉。

　　男生也豁达，不後悔。

　　有些礼物不贵，但有趣。你我收过某些难忘的小礼物吧。礼物也有「大」的，指大堆头。我是从来不去甚麽签名会的（太懒了！）。有时读者会把一叠书送到出版社，托编辑女孩交我签名，日後再取回。记得一回有位很客气，买了一大袋零食，有咸有甜有软有硬……，太丰富了，大家一星期也吃不完。

　　後来有朋友搬办公室，便抄桥，学她买了一大袋零食上去当贺礼。以为定可让人纪念五七天了？——谁知一个晚上全干掉，真是一群蝗虫！（当然自己也有份。）
