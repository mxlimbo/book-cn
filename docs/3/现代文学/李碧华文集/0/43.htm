电话旁的观音像


　　T业务发展不错，公司装修过。并且加了好些摆设。

　　在进门口处沙发小几，设有电话座。电话旁边，竟然是个观音像。

　　T不是「善男信女」吧，观音像也许优雅漂亮，用作工艺品观赏。也没有开光。——不过不知如何，他们近日常头痛。

　　我说，不一定与甚麽迷信物事有关。工作过劳睡眠不足也会头痛。

　　但，假如我是那尊观音像，就一定头痛、心烦、气躁。

　　供奉神像在最当眼处，人人都见到，都有礼，本来是好事。那公用的电话，却是一天到晚在响，接听的人，坐下来大谈特谈，你说有多吵！审讯犯人酷刑之一是「噪音」。

　　我对宗教认识仍浅，不过但凡神、仙、佛，都爱清静，平淡天然，不喜扰攘吧。观音大士当然一样。普渡众生很累的，也希望静静休息。

　　凡俗的闹嚷，以寂静中的电话铃声最讨厌。电影院隔世光影中一下手机乍响，特别惹嫌。我们工作中不愿被突如其来的铃声吓一大跳。午夜来电（除非是挂念你的人），多半没有好消息。

　　整天价响，哪有一刻舒适？对观音像不敬。电话的「磁场」，也有负因子。——老实说，才不喜欢闯头的电话，只是我们无法摆脱而已。
