日本菜


　　市面上不断涌出一批批只读至中一二，顶多中三，便念不上，或无心向学，出来打工的小妹妹。每份工做几个月，又做不下去，再转另一份。

　　所以我们经常遇上一些很「生手」的洗头妹、售货员、侍应生。她们完全不认识自己的工作，当然不会用心。

　　某回「悲剧」发生了（我的悲剧）。

　　到一间日本食肆。它新开张不久，有几个招牌菜。我们问：

　　「你们这儿比较多客人点的是甚麽菜？」

　　「日本菜！」

　　——难道是俄国菜麽？。又问：

　　「有甚麽好介绍？」

　　「寿司 ！」

　　「今天甚麽最新鲜？」

　　「鱼 ！」

　　当她答我们的时候，不但埋直气壮，还一脸无辜。她瞪 大眼睛，XX，YY……， 是最负气的答法，觉得这两个客人特别蠢，问些幼稚园问题。——我们花时间同小妹妹纠缠，又真的好蠢，几乎失去自尊。悲哀得只能苦笑。

　　这店，有食家介绍过的。老板招呼好又如何？应该给夥计补习一下。——不过又要「学习」？那些小妹妹肯定跑掉。再来，更生手。

　　就这样，青春漂流。
