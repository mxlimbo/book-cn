距离之美

　　为什么多数夫妻不再卿卿我我?主要原因是双方朝夕相处，失去了神秘感；有个诗人偷偷爱上一个女人，仅仅为这位女子的长发，他就写了几十首诗，但是后来当这位女子也真的爱上她，两人开始幽会以后，诗人从此不复有诗，他说是因为缩小了距离。我这几年喜爱读回忆录和忆旧文章，也是因为感觉有时空感，对那些写家庭琐事，儿女情长的文章，即使是大家手笔，妙品佳章，我的印象终究是距离太近了。

　　口语有"水中赏月"，俗话有"看景不如听景"，歌曲有《雾里看花》，从美学角度上看，都是说明距离之美，朦胧之美，想像之美。"亲密无间"、"亲如兄弟"、"亲如手足"等等，这些也是好词，可是人无千日好，花无百日红，现实常常把人们良好的愿望打破。比如两个朋友合伙做生意，分红时，甲想，既然是朋友，我多得点你也该体谅；乙想，既然我付出的劳动不比你少，为什么要少得?原本好好的朋友，一下子就散了。现在我常常想起在学校时，我与同寝室的刘君为了表示亲热，把饭票合在一起，结果事与愿违，由于两人的饭量不一样大，有时碰上合胃口的就想多买点，但对方未必认为好，所以常常双方为了互相迁就，就委屈了肚子，而交情深了没有?也未见得，毕业后仅仅见过一面，十余年来再无音讯。饭票本已合在一处，如果中途分开，又让人以为是闹什么矛盾。结果为了维持这种局面，令人尴尬了半个学年。说到底，还是有点距离好。

　　还有一种情况，多数人都会碰到。你到某地，原想游山看水，逛逛百货商场，溜溜地摊儿，轻松一下，可就偏偏碰上当地的热心朋友，他把你的日程全部排满，然后一步不拉地陪伴左右，充当免费导游。所以每到这时，我都赶紧说明，除了接受朋友的美意之外，自己还有个没出息的癖好，是爱逛地摊儿，比如什么旧货店，旧书摊，旧古玩铺，都愿意自己去信步遛哒。是不是要买，不一定，但独自漫步，说起感受，因为隔着距离，那肯定是独特的。

　　再说两件说明距离的事，一件大俗，一件大雅。大俗：民俗俚语曰："妻不如妾，妾不如偷，偷着不如偷不着。"是说距离的；大雅：维纳斯雕像之所以成为精美绝伦的艺术品，就在于她是残缺的，正因为缺了两臂，才把审美距离一下子拉大，增加了无限的想象空间。
