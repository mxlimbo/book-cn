王冠军：人称江湖一盏灯

　　我现在开始写王冠军了。我所写的王冠军，可能与你认识的王冠军不一样。但是这有什么关系呢?千人千面，起码他在我眼里已经是传奇人物了。

　　先说名字。但凡比赛，总要有输赢，总要产生一个冠军，下围棋也不例外。王冠军起了个夺冠之名，这为他争得荣誉提供了心理基础。有时候他也跟他的对手开玩笑："我已经是冠军了，你还给我争什么?"

　　再说成绩。实际上，他真没有辜负这个名字，在第一、二届全国围棋名手赛中，曾经夺得冠军之衔，在第一届"梅花杯"邀请赛上，他以九连胜的辉煌战绩呼应了自己的名字。1982年他定为四段，当年即升为五段，1985年升为六段，1988年升为七段，1993年晋升八段，现为河南省围棋队教练。

　　说起他的围棋史，其实也很简明。11岁上三年级时，学校兴起围棋了，老师挑学习好的去报名，王冠军此时荣任班长之职，理所当然入选。在这40余名学棋的学生中，坚持到最后的，惟有王冠军一人。事隔半年，学校又选一批学生去学棋，这回坚持到最后的，不是别人，恰是王冠军的弟弟王海军，老师说，也该着这兄弟俩学棋。学棋靠天赋还是靠努力，这要因人而异。学棋不久，王冠军跟一个比自己小许多的孩子下棋，棋至中盘，王冠军胜势已定，这时只见那位少年棋手看似随意地走了一手，随后便长嘘短叹，好像后悔不已的样子，王冠军受此迷惑，想着反正也是自己赢了，也跟着随手一应——不想那少年暗藏杀机，紧跟一步，只此一手便翻了盘，这回轮到王冠军后悔了!而后来有一次与方天丰八段的比赛中，他又尝随手的苦果，本来赢定的棋，方天丰却下出一手连连叹气的棋，王冠军想，这算什么，随便走走不就可以了吗?结果让方天丰出棋了!

　　围棋又需要天份，有两件小事可显示王冠军在这方面的天赋，那是在他学棋不久，有一次到当时著名棋手陈岱先生开办的棋社去玩，陈见一帮小孩子进来，先是轰他们，一听是来学棋的，又赶紧热情相待。陈先摆了一个定式，问大家怎么走，王冠军说："先立后扳是正着。"陈脑子里已有原来的次序，随口就否定说："不行不行。"继而一看，这一立一扳还真可下，较之原来的定式更完善，陈岱不禁对王冠军刮目相看："学了多少年棋了?"王说："刚学。"陈连声赞曰："可造之材，可造之材。"遂收为弟子。

　　还有一次，是在去北京的火车上。这次王冠军是作为河南围棋队的成员参加全国比赛的。火车上两位围棋爱好者正在摆擂台，王站在一旁观棋不语，两位成年人见他年纪小小却看得入神，就问："你也会下?"王也是少年气盛，在刚才观看之时，已知两人的水平，便说："我跟你俩可以同时下。"多面打?两位成年人自然不服，结果两盘下来，成年人的汗就湿透了衣服，连连惊呼："棋童神功!"

　　棋是长进了，但如何形成风格，这成了成年后的王冠军思考的问题。起初他曾下仿子棋，即对手走什么，他也跟着走什么，往往惹得对方上火，结果走出漏招。例如有一次大型比赛中，他的对手是著名的力战型国手刘小光九段，刘执黑先行，王冠军一直仿刘小光至72手，刘在天元投子，王冠军这才变化，据后来人统计，好像围棋史上还没有仿子棋下这么多手的。不过王冠军并没有打算这么下下去。他根据"金角银边草肚皮"的理论，反其道而行之，以围中腹为特色，以做大模样为主攻方面，结果这种思路，一直影响到他今天，以至于别人跟他下棋早已知道他的大模样构思，总想去破他的大阵营。有位棋手在一次正式比赛中，执黑先行，王依例以二连星应对，结果对方根本不给他机会，第五手就在白棋的二连星中分头，下在二连星中间的星位上，意在破白棋的大模样。而连我这样的业余棋手，在王冠军授四子的情况下，一见他想围大模样，也急不可待地打入，当然以我的棋力，这打入的阶级弟兄，多半凶多吉少，但也由此可以看出，王冠军的"模样风格"已经被众人接受。果然不久，王冠军就以"江湖一盏灯"的绰号，在中国围棋界独树一帜了。"江湖一盏灯"，我的理解是：棋盘四周，别人据为已有，但他在中间的大模样，雄恃四周，即可瞻前顾后，又能左右逢源，宛若明灯；深一层的寓意，是在当年围棋并不像今天这样兴盛时，王冠军能固守自已这一方阵地，不左右摇摆，依然执著于中原的围棋事业，这也宛如一盏明灯，推动着围棋事业的发展。陕西一位书法家，有一次在给王冠军写条幅时，起先犹豫，因为他见有"江湖"二字，怕容易沾上"江湖气"，及至领会了"江湖"的含义，便欣然命笔，并题长跋以抒钦慕之情。围棋杀手刘小光的理论是："我把别人的子吃掉，不就有了空吗?"王冠军的格言却是："你占实地不要紧，只要我围的中腹比你多，多半目也是赢，何必去搞肉搏战?"

　　棋风，可以反映出人的处世风格，更进一步说，棋风甚至可以显示出一个人的境界。今年初秋我到中国棋院，与其他专业棋手聊到王冠军，我问他们对王的印象，他们却比我高深，反问我如何评价，我说："好像是执著有余而灵性不足。"众人笑曰："你已经说了，我们还说什么?"结果是把我给套住了。其实，以我这样业余围棋爱好者的境界，竟敢评价一个专业棋手，实在是授人笑柄了，每思及此，总不能不连说几声惭愧。

　　回头说说我的围棋史。最初学围棋，是一个也是刚会走几步的熟人教的，教会我后，他算找到了出气筒，常常把我杀得几乎片甲不留，过足了胜利之瘾。我想报仇，却苦思不得其法。恰巧，院里一个被邻居暗地称为"坏孩子"的年轻人，不知怎么居然喜欢上了围棋，而且下得很好!当然这个好字，是我当时无偿赠给他的。他教我如何识别"拐羊角"，即今天术语中的"征子"。这一招，开发了我的智力，使我在以后对那位最初的"老师"较量中，屡战屡胜，令他大为惊奇。但是很快，我就遇上了敌手，是一位工厂的同事，他至今仍是我强劲的对手，而令我更为惊讶的是，我们的厂长，居然可以对我屡战屡胜!这使我意识到围棋的玄妙，而当我认识了王冠军八段之后，我跟他所有的对局，我都没有赢过!当然王冠军还是授四子。输了棋，我的结论常常是："围棋真玄啊!"至今我对围棋终于有了一点新认识，是：杀得赢就杀，杀不赢就忍。一个忍字，多少道出我一点心得!
