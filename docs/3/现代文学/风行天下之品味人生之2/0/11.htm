张迪华换笔

　　换笔，大家一般理解为钢笔换电脑，这是时下正流行的想法。可是张迪华的换笔，却是反其道而行之，竟是钢笔换毛笔，退回去了。不明就里的人，也许会觉得奇怪，但说起个中原因，也就不难理解了。

　　1950年出生的张迪华，在调入《洛阳经济报》做编辑之前，曾经做过小贩，当过农民，干过民工、坑道掘进工、砸焦炭工、炉前工、打包工等多种工作，总之，都是又苦又累，又脏又险的活计。跟很多人一样，他同样逃不过命运的摆弄，文化大革命的冲击，不仅击碎了他刚刚萌生的画画梦，也击碎了他的写作梦。那时他还是个小孩子，对画画有一点兴趣，学着杂志的封面封底比葫芦画瓢，后来借到一本《芥子园画谱》，这本现在到处可买得到的画画启蒙读物，成了他的宝贝。不久学得像了，俗语云：既在江边走，必有望海心。他就有当画家的念头，老师也说，你是块学画的料。可是那个时代，不是能让人这么容易实现梦想的，文革的狂风一刮，就把一个少年的梦想刮得无影无踪。直到1981年，已过而立之年的他，才被调入厂党委办公室，从此命运才有了转机。得知他的这种经历，我就不能不想到这样一个问题：文化大革命究竟浪费了多少人的青春?这笔帐怎么算?

　　调入报社，自然要拿笔，这是张迪华的强项，他原来就是因为经常在报纸上发表文章而被厂领导相中的，所以写文章，对他来说正是编织理想的最好锦缎。也就是几年功夫吧，他写出并发表了逾百万字的作品，成为河南作家协会会员和报社的排头兵。 可是人的梦，只要有一丝机会，就会死灰复燃，张迪华也没有放弃他的画家梦。当妻子下岗，家中经济日渐拮据时，妻子准备学画牡丹，好换些钱财贴补家用。身居牡丹甲天下的洛阳，张迪华认为有这个条件，便全力支持并耐心教妻子作画。教着教着，勾起了他的画画瘾，索性疏远了钢笔而亲近了毛笔。于是有一天，他站在画案前，自己开始涂抹。不过这一切好像迟了一些，因为此时他已经是近43岁的人了。岁月无情，不但更改着时间，也更改着人生，他又能抱怨什么呢!

　　钢笔换毛笔，更难。古语说，退笔如山始通神，好像是苏东坡的名言。退笔，即坏笔，天赋的成份不考虑，只说苦功夫，苏居士的意思是用坏的笔要如山一样高，才能达到自如境界。从这一方面看，张迪华的天份比下的苦功夫大，这要归结于他少年时代的良好基础。我是书画方面的门外汉，在张迪华的画室里，我还是大大吃了一惊，像《牡丹芳》、《千里共婵娟》、《童年的月亮》等等，我几乎要大声喝彩了!《牡丹芳》着色淡雅，构图精致，尤合我心。如果说让我这不懂行的看了喝彩不算数，那书画界的名流能说不懂吗?他们的评价却比我更上一层楼，河南省书协副主席李进学先生曾书赠张迪华，称其文章尔雅而不从俗，书画风流自有神韵。菲律宾普陀寺主持、精通中国书画的道元法师在看了张迪华的画作之后，特意在《菲华时报》撰文评述："花鸟画中以牡丹最为大家喜爱，因此自古以来，牡丹画散布最广。凡事一多便滥，很多追求品位的画家不屑于画牡丹，正是因为牡丹画太多、太俗的缘故。而张迪华先生由于有着深厚的文学功底，又熟知牡丹的情性，故张先生笔下的牡丹生动传神，又寓意藏趣，还讲究笔墨，真可谓雅俗共赏。"

　　已经画得这样好了，是不是可以满足了?本着食不厌精，脍不厌细的标准，他当然还想更上一个台阶，但这一次就不同了。懂行的人知道，想让众人喝彩容易，想画出传世之作却不容易，有些人终其一生，也没有画出一张可供后人留存的。张迪华苦苦思索着，勤勉地在洁白的宣纸上勾、皴、点、染，似乎要将逝去的岁月抓回来。

　　后来他到了菲律宾，在那里度过了一段客居的日子。一家华文报纸约请他做主笔，并提供不菲的经济来源，他可以在那里安心画画，搞他的创作。本来这是令人满意的生存环境，但是在菲律宾，文化毕竟不如他想象的那样发达，人们一心做生意，文化只是闲来消遣的东西。一张报纸，可以满篇错字别字。而且客居，不管你的生存环境有多好，终究还是栖息在人家的屋檐下，所以也只是一年多时间，他便又和妻子一起返回故里洛阳。"洛阳亲友如相问，一片冰心在玉壶"，在1998年长江洪灾之时，他将自己义卖的14000多元画款全部捐出。那个充满爱心的张迪华又回来了。不久前我们通电话，他告诉我，他已经加入省美术家协会了。

　　和他夫人闲聊，知道张迪华也有过不顺心的时候，但是作为男人，他不愿讲自己的苦，更不愿在夫人面前喊苦。比如最早上街卖画，他没有勇气，就是夫人拿了画，引颈叫卖，价格还往往不高，这使张迪华大为不满――不是对夫人不满，而是对自己不满，如果画得很好了，岂是可以如此贱卖的?如今他一幅画的价格，一下子翻出原来的数十倍!所以人的动力，你有时真就不知道是怎么来的。

　　钢笔换成毛笔，张迪华终于实现了儿时的梦想，如今已是一位名副其实的职业作家和画家，但是反观他走过的人生之路，我不由得感叹人世沧桑。张迪华的这一天还是来得太迟了一些!像他这样有天份且又肯用功的人，应该更早些成名成家，可是十年文革，生生把他给耽误了。这不怪他自己，而是文革造成的。也许有人会抬杠，说事情往往有它的反面，如果没有文革的磨难，他还会保持这种进取心吗?也许正是文革成就了张迪华。可是要我说，对于经受了生活磨难的张迪华来说，不是荣幸，而是不幸!这结论不仅仅是对张迪华一人适用吧。
