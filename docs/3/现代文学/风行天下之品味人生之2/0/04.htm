张宇吓了我一跳

　　张宇先生是著名的小说家，在全国多次获奖，风头正劲，属于那种既有名气，又有潜力，前途光明的实力派作家。他吓我这一跳，其实连他自己也不知道。

　　事情是这么回事。有一天偶尔翻报纸，是当天的《大河文化报》，副刊上有一篇张宇的文章，《来自猪饲料添加剂的恐惧》，题目煽情，也就想看看，一看不要紧，真让我吓了一大跳。读完，不禁陷入沉思，顺着张文的思路，想理出一些枝枝蔓蔓。

　　我们常常抱怨肉不香，饭不香，有一段时间闹不明白，后来一深想，就吓了一跳。猪饲料中的添加剂，催着猪像吹着长，本来一年才能长成的猪，半年就速成了，肉，当然不会香。鱼，野生野长的，一般一年才能长成，可是现在把鱼围在鱼塘里，定时定点地喂人工饲料，鱼们都傻头傻脑地只会张口，哪里还有追逐游戏的心思，跳跃龙门的豪情呢?还有鸡，有用灯光照射一说，不让睡，只让吃，结果鸡们都肥大，却呆若木鸡，像是膨化了。如果只是膨化倒还罢了，要命的是这膨化剂里，无一例外地含着化学元素，而无一例外地还有，这些化学元素并不能被动物们完全吸收，动物们肉体里残存着多种催化剂元素，而人又不甘口淡，于是催化剂就这样被人类吞进体内。动物吃催化剂，长成傻大笨粗，人呢，吸收催化剂，后果，我不敢设想。

　　显而易见，我不是自己吓自己，更不是想吓别人。总的感觉是，人们不肯下功夫了。有一年，有人请张中行先生吃饭，席间，上了一道酱肘子，张先生一尝，说是味道差点，别人说，这还是"天福号"的呢!张先生一听，说，快别说"天福号"了，一说，觉得味道更坏，真正"天福号"的味道，这份比不了。那是用80斤的小猪作主料，现在，谁还肯下这么大的功夫?我在郑州请张中老吃过一次京东肉饼，佐以玉米渣粥。先生的评价是，肉饼，勉强能吃，玉米渣粥则是太稀了，滑嘴，而真正的香河老家的做法，是用柴锅熬。饭熟撤火，慢慢闷，掀起锅盖，还见玉米渣粥在"突突"冒泡，喝起来，沾嘴，所以先生是照例两碗。我不用过多动脑，也知道今天的玉米渣粥是用天然气煮的，客人在催，饭店图快，没有那么多时间磨功夫。

　　从享乐主义出发，口腹之欲，精神丰宴，都是人所共求的。有一种口号现在提得很少了，那就是共产主义时代，物质极大丰富，人人各尽所能，按需分配。这个口号曾使我确信：共产主义就是让人吃饱了睡，睡醒了吃。这不好吗?然而眼下我们不但自己说还是社会主义初级阶段，而随着香港的回归，我们又接受了资本主义。越倒越远了。然而资本主义社会里的人，很多也是希望能吃饱了睡,睡醒了吃，这和信仰共产主义的人，不是走到一条路上去了吗?

　　这笔糊涂帐，只好留给政治家去算。

　　还说张宇先生的恐惧。张先生说，他有了这个恐惧之后，一度不敢吃肉，后来耐不住口淡，只好举箸之前，先猜测哪块肉里含的添加剂少，打定主意，才去下筷，算是自我安慰一番。明眼人看得出来，这是自我欺骗。可是又有什么办法呢?自我欺骗的也不是张宇先生一个。为什么会这样呢?原因很多，只好用兵法：擒贼先擒王，抓主要的。我的个人观点是，只怪这个社会太浮躁了!

　　浮躁，意即不踏实，纵观以上所言，哪一点不跟浮躁有亲戚?人不但催化动植物，现在连自已都要"克隆"，人是不珍惜自已了。而随着电脑的发展，人脑将逐渐退化，我近来看到一篇文章，说终有一天，人类还要生产出电脑性交椅，无论何时，只要你有了欲望，坐在电脑性交椅上，就可以完成自慰——人类真是越来越不自爱了!在这样的动机驱动下，给猪喂点添加剂，催猪快长，岂不是小儿科吗?所以我的悲观想法是，人类终有一天会毁灭自已!

　　就算是杞人忧天吧，我的这种担心能是多余的吗?这个推论会不会又把张宇先生吓一跳?
