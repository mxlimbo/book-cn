心灵与情感的自白与渲泄

——读段海峰《品茶斋随笔》

辽宁 王和利

　　拜读段海峰先生所著《品茶斋随笔》，掩卷沉思，颇有许多感触。文集中收录了作者自1993年以来的新作共65篇，从中可以看到海峰先生勤奋笔耕不辍的足迹和一段不同寻常的心路历程。

　　这本文集，与其叫作"随笔"，不如称为"随感录"更为贴切，海峰以他敏锐的目光和深邃的观察力去审视社会和人生，善于从人们因不经意而被忽略的生活中，去捕捉、挖掘、沉淀，升华出深蕴哲理的思维方式，为自己的思想、感情和心态寻觅到了一份平平常常的定位与座标，且都是作者真实情感的自然流露与渲泄，质朴无华、舒畅自如、通达洗练、水到渠成；如呢喃自语的心灵自白，亦如面对一位无话不说的知音挚友的衷肠倾诉，毫无矫揉造作、刻意修饰的斧凿痕迹。文如其人。这也同海峰先生待人真诚、热忱、坦荡、直率相吻合，谦虚质朴、不肆张扬、平易近人，在文章中都有很多印证和体现。

　　海峰始终以一份平常心态去解剖灵魂的自白，与读者的心态情感息息相关。他将作家所特有的丰富想象倾注于尘嚣凡俗的事物之中，发现、观察、思考，他笔下每一篇短文都如那晶莹的露珠，虽然细微，却是可以折射出太阳的光辉，给人启迪、发人深思!

　　我与海峰是作者与编者的关系，因稿子与他曾有多次电话交谈与书信往来，至今尚未谋面，属神交之友，感觉他对本职工作的热情及敬业精神之外，还有平易近人、坦率真诚的一面。闻知他出版这本《品茶斋随笔》文集后，从对他的良好印象出发，我邮购了一本。这其中主要源于尊重作者的执著、痴迷和勤奋，也不无祝贺、捧场之意。毕竟，作为同道，我深知写作的艰辛与不易。待邮递员将文集送来时，我正伏案忙于偿还业已拖欠数日的笔债，疲乏之余，权作休息，信手翻开，即被书中的文字所吸引，以至于爱不释手，一气读完书中的全部文章，唏嘘感叹之后，深感自愧弗如!同为此道中人，海峰小我七八岁，思想却远比我成熟，思路广阔，难怪备受著名学者张中行老先生关注，不仅为之写序，且发出"后生可畏"的叹喟!

　　文人于写作中所追求的至高境界是物我皆无，这很有些佛学经纶的意味，其实，写作亦如出家，是为一份执著，坚信不疑的笃信驱使，心净如水、淡泊宁静、抛弃尘嚣世俗的繁杂欲念，潜心体会揣摩，自会渐入佳境，禅悟出一份心灵所依的境界。当然，这是从文学写作的入道之说谈起，是一份心灵与精神的净化与升华。仅此一点，若想达到物我皆无的写作境界还远远不够，应如我们评价某位德高望重的长者那样"大智若愚"，表面看起来很普通，甚至有些愚钝，但实际上却是大智大勇，亦即"真人不露相"也。作文亦同此理，初学者走过下笔生涩，文理不通的沼泽泥泞之后，往往会堕入华而不实的歧途，将华丽的词藻堆砌起来，追求形式的完美而忽略了内涵、外延的底蕴。从这种意义上讲，海峰的作品已步入这一为人瞩目的境界的门坎，循此路走下去，便会领略其中的无限风光了!

　　当然，这是作者多年来勤奋笔耕、孜孜以求的必然结果。我虽从未与海峰先生谈过有关文学写作的话题，但从文集中已不难看出他在潜心练笔中已积蕴了很深厚的写作功底，能够舒畅自如地抒发内心深处的感受，那娓娓道来，且又自然舒缓如行云流水般的文笔，显示着作者日臻成熟的写作技巧与表现手法，质朴归真。

　　在写作技巧之外，我还感到海峰先生是一位非常谦虚的人，作为一名青年人，在写作方面已有成就，并得到了社会与文学界的认同，成为中国作家协会为数不多的年轻会员之一，他丝毫没有自负自傲自得之意。他虽因工作之需、之便，常与驰名全国的文化界、艺术界前辈、名人接触，甚至过往甚密，却丝毫看不出他有故意卖弄、炫耀和自吹自擂之意，这除了海峰先生的谦虚美德之外，更重要的是他对名人前辈除了敬佩、求教之外，少有攀附、阿谀之意，这一点，已是超越写作方面之外，去感受到一个人的待人处事的人生哲学，不攀高大亦不贬微小，始终以常人心态对人。如果我们工作、学习的人文环境和人事关系中，身边多有几位海峰这样的人，我们用于人际关系相处的精力便会大打折扣的。似乎这些句子与写作离题太远，其实却不然，要想写好文，先应学做人，心怀叵测之人，或为世俗功利污染太重之人，其文又有几分真实灵气呢?即便是蓄意的掩盖，其文也是言不由衷，矫揉造作，难于与读者产生共鸣，既使文笔再熟络老练，也绝然写不出《品茶斋随笔》这样自然质朴的文章，也绝然不会营造出茶香飘溢、浓淡相宜的这份怡然和宽松的氛围。

　　在笔耕繁忙中，我停止了案头正在写作的文稿，去记下以上文字，对我来讲，真正是不吐不快，一吐为快，能够为这部随笔集子记下这篇为数不少的文字，不是情之所至、欲罢不能，是没有具他解释的!

　　相信远在千里之外的海峰先生，也未曾预料到他的文集会令一名读者如此分神，竟至于写下这样多的无从发表的文字，这也是我掘弃了功利之杂念，为海峰兄的作品净化了心灵所至，相信海峰先生在日后的人生之旅，会为我们留下更多的发人深思，令人"分神"的文字，从而赢得更多的知音!
