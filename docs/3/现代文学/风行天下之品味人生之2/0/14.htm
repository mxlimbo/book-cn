灶 王 爷

　　只听过，没见过的人物中，灶王爷是其中一位。

　　中国因为没有国教，所以信奉的人物，也都带有"中国特色"，如玉皇大帝，龙王，土地爷，灶王爷等等，并根据想当然的推测，给土地爷和灶王爷各配了一位土地奶奶和灶王奶奶，这样喊起来亲亲热热，像是一家人。别看这些人物都远在天边，可是在老百姓的心目中，却是近在眼前。过去每到年关，家家户户打扫庭堂，清洗厨房，总不会忘记在灶台旁贴一张灶王爷的像，客气一些的，也许还会惦着灶王奶奶。为这些神灵供奉的祭祀，除了水果糕点之外，还无一例外地供着祭灶糖，一听这名称，就知道是专为灶王爷预备的。 这种糖，后来都被孩子们分而食之。在贫困的童年，这是一种极大的享受，所以我从小就对灶王爷有好感，大概也是尝了他这点甜头。不过这种糖虽甜，却非常粘牙，据说是麦芽糖，后来才知道，这是老百姓玩的一个小手段：让灶王爷吃了这种糖，粘住了牙，就不能开口讲话，人间的一些事非，就不会传到玉皇大帝那里去，他老人家有很多公务要处理，就不要为人间这些小事劳神了。可是老百姓就没想想，你供奉灶王爷的目地，是让他上天言好事，下界保平安，你把他的牙粘住了，还让他言什么好事?终于有一天，我把这个顾虑讲给父母，他们听了，吱唔了一阵，最后大概是说本来就是糊弄灶王爷的事，不必去认真，从来都是心到神知，上供人吃，只要心里装着灶王爷，他老先生是会知道的。

　　这笔糊涂帐，直到今天也没有算清。

　　其实灶王爷的任务还不止于"上通下达"，他还有个更大的责任是司火。谁家的灶火不旺，先不管是不是柴火潮不潮，炉道通不通，都要先给灶王爷打个招呼，提请他注意调整火势。《西游记》里，有一处写到孙悟空与羊力大仙比手段，只见孙悟空跳进热油锅，在里面翻筋斗，竖蜻蜓 ，而当羊力大仙跳进油锅，护住锅底的冷龙一撒手，油锅成了真油锅，羊力大仙一命呜乎。据老百姓说，冷龙归龙王管，龙王和灶王是亲戚，所以终究还是灶王帮了孙悟空。过去百姓家办红白喜事，总是支几口大锅，几十号人吃饭，灶火旺不旺十分关键，这时候，厨师总要特别提醒，要好好侍奉灶王爷。有一年，我参加一次这样的大宴，亲眼见到一位厨师因厨房没有贴灶王爷像而拒绝出场。大概在厨师眼里，灶王爷的地位该像木匠眼里的鲁班。我曾在年前出差到 县，见人们都在购年贷，一个小摊吸引了我，是卖灶王爷像的，我兴奋，一下买了很多张，回来送人，虽说现在都用上天燃气了，但我送的灶王爷，人家就贴在天燃气灶的旁边，可见在人们的心目中，管他是柴禾、煤火、天燃气、煤气灶，只要有火，就该有灶王的地位。可是有一点，虽说灶王是管火的，可是不需要火的时候，该不该叫他呢?叫他，他会不会帮忙灭火?不叫他，火的问题又是他份内的事，让别的神仙来，岂不有越俎代庖之嫌?

　　迷信不可取，科学更靠得住。有一次我到一户新装修好的朋友家去参观，只见灶王爷的旁边，赫然放着一个红色的灭火器!看来他也清楚，不管灶王爷多么受人敬重，真到了该灭火的时候，还是用灭火器管用。他既不得罪灶王爷，又不违反安全规定，真是一个明白人!
