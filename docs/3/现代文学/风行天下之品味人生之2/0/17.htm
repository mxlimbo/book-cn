中原才子段海峰

北京 李保田

　　中原大地，自古以来英雄才子层出不穷。我认为文友段海峰可称为当今中原大地涌现出来的英雄才子之一。
　　我所说的英雄，并非指在古战场上骑马举刀拼杀的斗士，也不指那些在邪恶面前挺身而出除暴安良的那类勇猛之人，而是指那些在改革开放的大潮中，自身有着优裕的环境不去享受，偏偏自找苦吃，放下许多人看着眼红的编辑、记者饭碗，甘愿去冒风险，专心致志地去干自己想干的事业的人。文友段海峰便是开此类先河的英雄。

　　很早就听《光明日报》的朋友常联久谈起过段海峰。得知他是一位有文化有胆识有作为的年轻人，所以一直就想见见他，但是因为他工作很忙，像一股强劲的"海风"(段海峰常用的笔名)在全国各地刮来刮去，踪影难捉，于是就很难与这位中原才子晤上一面。

　　机会终于来了。去年秋天的某一天下午，在联久兄的引荐下，我终于在北京团中央招待所见到了比我年轻十几岁的海峰弟。

　　见到海峰可真不容易。那天我和联久按照海峰事先与我们约定好的时间准时来到团中央招待所。但我们等了他近一个小时也没见到他的影子。我们知道他是个非常讲信用的人，断定他碰上了特殊情况，便耐下心来静静地等待。又等了他约半小时，只见海峰弟满头大汗匆匆赶回招待所。"失礼失礼!"他双手抱拳向我们表示歉意，亮出一副江湖豪杰的架势，看着他这风风火火的忙碌样儿，我们都笑了。

　　此次迟到不怪他。那天上午海峰去海淀采访著名歌唱家胡松华。二人一见如故，谈得非常投机。谈着谈着就接近了与我和联久兄约定见面的时间，当他将要起身告辞时，胡松华谈兴丝毫不减，突然心血来潮，说他要马上画一幅《骏马图》赠给海峰。既然是胡松华先生自已提出送他一张画，岂有推却的道理?而这也正是海峰求之不得的事儿。待胡松华的画画完，海峰急忙搭车往招待所赶。海峰那天没带手机，途中无法和我们联系，况且路上又堵车，把个中原才子急得汗流浃背。

　　海峰长得非常普通。戴一副轻度近视眼镜，一派知识分子模样。不像我设想的那种身材高大魁梧、嗓音洪亮如雷的样子。海峰是河南开封财贸学校毕业生。本来他的专业是预算会计，但因为他对文学的喜好，工作后一年，即被调到郑州《时代青年》杂志社从事编辑工作。当年他才22岁，正是年轻有为大干一番的最佳年龄。然而他经过慎重思考之后，毅然决定放弃掌握作者作品生杀大权的编辑工作，留职停薪，去当一个在经济上没有保障的"三味书社"老板，专门收售处理过期的旧书刊，令许多朋友感到不可思议。

　　但是朋友们的震惊并没有影响海峰一心一意地去干自己的事。经过一番拼搏和努力，居然把"三味书社"干得很好，获得了社会和经济效益。使他一跃成为河南省内具有五个分社的"三味书社"总经理。他的壮举受到了社会各界人士的关注和赞赏，全国有多家广播电台、报纸杂志为海峰的事迹做了广泛的宣传和报道。

　　海峰年轻好学，更爱名师。在北京，他和张中行、肖乾、季羡林、金克木、王蒙、冯亦代、林斤澜、胡絜青等文化老前辈结成忘年之交，从他们那里学到很多宝贵的东西，他和这些名家、大家、作家、画家们结成了友谊很深的好友。

　　比较起来，海峰和肖乾先生的交情和缘分更深一些：1993年夏天，海峰在郑州西郊某地摊上，看到一本小32开、繁体字竖排版、布面精装文集《小树叶》，不禁喜出望外。这是肖乾先生早期的散文集，连中央文史馆里都不曾保存这本文集，居然在地摊上碰见了它!于是海峰重金将其购下。尔后，立即给北京的肖乾先生打电话，并说要亲自将《小树叶》送上门去。肖乾得知这一消息后大喜，说："好，好!你来北京吧，我用我的三本书《未带地图的旅人》、《红毛长谈》、《北京城杂忆》跟你换，如何?"

　　海峰听罢，甚喜，速来北京为肖老奉上《小树叶》，同时手捧着三本肖先生亲笔签字的集子，兴奋得像顽童一样手舞足蹈起来。此后，海峰每来北京，必去看望肖乾老先生，而肖老先生也待海峰如自己子孙，很是热情。

　　海峰学习刻苦。他从16岁时就开始写日记，这个良好的习惯一直使他持续至今。目前他已整整写满了50余本，塞满了一木箱子。这一箱子日记本里，记载着他并不幸福的童年、少年和青年时代的足迹，记载着他拼搏成长的过程，记载着他经历过的风雨人生和酸甜苦辣。 海峰所涉猎的知识面非常广泛。他写的小说内容曲折引人入胜，写的散文优美文雅，写的随笔老道而凝重，杂文写得幽默辛辣入骨三分，今年，他成为中国作家协会中为数不多的年轻会员之一。

　　为了增加书本以外的知识，提高智慧，增加对人生的反思与感悟，海峰曾到嵩山少林寺"挂单"当了一段时间的"业余和尚"。在那个宁静又神秘的院落里，他和僧士们同吃同住同打坐同悟禅，在那个既无时间观念也无争夺竞争的寺院里，海峰真正脱去尘俗外衣静下心来，审视自己的灵魂，倾听山风的呼啸，仰观神秘的夜空。夜晚，他大着胆子去每一个殿堂里朝拜每一尊佛像，声声佛号使他的灵魂获得了净化。

　　目前，海峰已成为"水陆"两栖"动物"，既下海捕鱼又在岸上耕耘，开办一家电脑文印商店由妻子经管，他本人还在办杂志，写文章，过着"一家两制"的生活。

　　他已经出版了两本自己的诗集《潇洒的青春》、《罗曼地带》)和两本散文集《有话要说》、《品茶斋随笔》。

　　海峰，你不愧是文化界的一位才子！
