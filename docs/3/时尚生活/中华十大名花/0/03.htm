月季—花中皇后


牡丹殊绝委春风，露菊萧疏怨晚丛。
何似此花荣艳足，四时常放浅深红。
　　月季，花容秀美，色彩鲜艳；月季花月月盛开，被誉为“花中皇后”。月季花在中国已有千年以上的历史，是中国十大名花之一。每当月季花盛开之时，便会博得游客的赞赏。


戴高乐月季
cv. "Charles DeGaull"

　　戴高乐月季属杂种香茶月季类。株型矮，花丁香紫色，具有浓香，花形雅致。


信用月季
cv. "Confidence"

　　信用月季属杂种香茶月季类，株高70厘米左右，花大，重瓣，多花，花色粉色，基部金黄，具香味。



亚利桑那月季
cv. "Arizona"

　　亚利桑那月季属大花月季类，直立，强壮灌木，花红偏金黄，花瓣反卷，具浓香气。
