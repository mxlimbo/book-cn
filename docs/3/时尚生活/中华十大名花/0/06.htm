山茶花—花中珍品

似有浓妆出绛纱，行光一道映朝霞。
飘香送艳春多少，犹如真红耐久花。
　　山茶，通常叫茶花，是一种名贵观赏植物。山茶花为我国原产著名花卉，常绿灌木或小乔木。栽培历史悠久，盛栽于江浙地区，品种繁多。大多在2至4月间开花，花期一个月左右。



花露珍茶花
Camellia japonica Linn. cv. "Hualuzhen"

　　此品种原产于浙江，花大，牡丹型，花深红色并夹有白色斑块与条纹，叶绿色，边缘下卷，长势旺，是优良的园艺品种。


粉香变茶花
Camellia japonica Linn. cv. "Fenxiangbian"

　　外来品种，是杂交后经秋水仙素处理后培育出来的新品种。花深粉红色，花瓣20片，微扭曲，牡丹型。叶色淡绿，叶缘具明显锯齿，花具香气，颇受人喜爱。



期望茶花
Camellia sahuenensis Stap f. ex Bean
X C. japonica Linn.

　　该种是怒江红山茶与红山茶的杂交品种。花大型，直径达10厘米以上，花色深紫红色，牡丹型。叶深绿色，偏小，叶面粗糙。
