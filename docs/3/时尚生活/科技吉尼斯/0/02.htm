探险英雄

在现代的土著木筏上进行的伟大旅行（图片）

　　1947年，为了让科学界古时候的水手们定期穿梭于世界的大洋之间，挪威的探险家和考古学家托尔·海尔达尔出发了。他特别急于证实移居到波利尼西亚的居民并不是像大多数人认为那样从东面移居而来，而是随着太平洋洋流从西而来。为了达到这个目的，他依照土著的乔木木筏复制了一只木筏，并命名为“康提基”，然后与5个同伴从秘鲁的卡里奥穿行8000公里，历时101天，到达波利尼西亚的居民的确有可能是秘鲁人。由此，海尔达尔形象被刻画成康提基的形象。

最大的古代文明的发现

　　在19世纪20年代晚期，逃兵查尔斯·马松发现了世界上最大的古文化遗迹——印度哈拉帕（现今的马基斯坦）的印度河谷文明。1920年，拉伊·马哈杜尔·达亚·拉姆·萨尼在此处遗址进行了广泛的发掘，结果表明此处文明可追溯到公元前3300年。在摩亨朱达罗发现的第二处遗址显示出在长达1600公里区域内哈拉帕人所使用的砖块尺度都相同，重量标准都一样。而且还显示出在某些时期达到这种文明程度的人口达5万人在这里活动。更近期的发掘显示此文明的范围比想像中还要广，它沿着瑟勒斯活蒂河的两岸伸展并超过它范围，西至巴基斯坦的俾路支，东到印度的北方邦，南抵印度的孟买。此文明的大部分城市现仍需进一步发掘，它的文字尚无人能辨识。

印加的最大发现

　　1911年至1912年和1914年至1915年，由美国历史学家海勒姆·宾汉姆率领的两支耶鲁大学秘鲁考察队，发现了两座已经“遗失”的印加城市马丘比丘和维特科斯——两处美洲最重要的考古发现。

拥有最多代冒险者的家族（图片）

　　1999年1月，雅克·皮卡德（图上）正与他的儿子贝特朗告别。在此之后，他成功地进行了一次乘气球的环球旅行。雅克和贝特朗代表了一个著名的冒险者之家中的第二代和第三代。雅克的父亲奥古斯特·皮卡德在1931年5月首次成功地完成了进入平流层的气球飞行。此次飞行，升到了德国奥格斯堡上空15785米处。随后，他又成功地进行了第二次升空，并且继续修造一个深海潜水器——这是一种能够沉入海底的改进型潜水艇。1960年1月，雅克把深海潜水器定位在水深10911米处，创下纪录，实现了他父亲的梦想。

最快的沙漠穿行

　　1998年，摩洛哥冒险家穆罕默德·阿汉萨尔在16小时22分20秒的时间内完成了马拉松沙漠越野长跑。此项马拉松长跑自1986年起每年举行一次，长跑持续6天。参赛者必须穿越日常温度高达49°Ｃ，跨度达220公里的撒哈拉沙漠。选手们携带食物、衣服、一个睡袋和一个急救药箱，还要自己野炊。完成马拉松长跑的最年轻的人是一个16岁的男孩，最年老的是一个76岁的老人。1999年4月，来自27个国家的584人参加了此项马拉松长跑。

最长的皮船之行

　　1976年，为了证明所说的由圣·布伦丹完成的新世界之行是可行之举，英国冒险家蒂姆·塞韦林开始了他的行程。根据中世纪文学所示，他用橡树皮把牛皮硝制成皮革，并把皮革伸展拉开，罩到一个木制龙骨上，再用皮线将其缝制成一条皮船。最终制成的皮船,样子同现在爱尔兰仍在制作的小船一样，它载着蒂姆和他的船员们，从爱尔兰共和国克里郡的特拉里海湾到加拿大纽芬兰的海岸，历时13个月，历程7240公里，中途停留赫布里底群岛、法罗群岛和冰岛。

空越大西洋的最快单人划行（从东到西）

　　1969年12月至1970年7月，西德尼·亨得尔斯（英国）单人划行，从加那利群岛的拉斯帕尔马斯到西印度群岛的安提瓜，行程6115公里，历时73天零8小时。

应用太阳能穿越太平洋的最快航行（图片）

　　1996年，堀惠健一（日本）完成了迄今为止用以太阳能供电的小艇航行，穿越太平洋的最快航行。此次航行，从厄瓜多尔的萨利纳斯至日本东京，行程1.6万公里，历时148天。他的雪匣烟开头的小船“玛尔特美人鱼”号（在堀惠健的左边）长8.7米，重369公斤，由面积为12.08平方米的太阳能板供电。船的一部分由相当于2万多只易拉罐的再生铝制成。

第一个到达北极的人

　　人们普遍认为，美国的北冰洋探险家罗伯特·皮尔里是第一个到达北极的人。1909年3月1日，从加拿大的埃利斯米尔岛的哥伦比亚海角，皮尔里开始了他的远征。同行的还有他的密友马特·亨森，其他7个美国人，17个爱斯基摩人，19个雪橇和133条狗。3月底，考察队到达了北纬88度地区，最后的一批后勤保障队伍返回，留下皮尔里、亨森、４个爱斯基摩人和40条狗，准备进行向北纬90度――北级所在地的冲刺。4月6日，皮尔里的观测表明他到达了目的地。尽管弗雷德里克·库克（也是美国人）向他提出异议并宣称自己是在当月初先到达北极的，可是美国国会还是于1911年承认了皮尔里的成就。

第一个到南极的人

　　1911年12月14日，一支由罗阿尔·阿蒙森率领的由5名挪威人组成的小分队，从南极洲威尔士海湾出发，乘坐狗拉雪橇，历时53天，到达了南极洲。他们仅比一支由英国探险家罗伯特·福尔肯·司各特率领的由5名探险家组成的小分队早一步到达南极。

最长的南极洲跋涉

　　历史上在南极洲进行的最长的无人协助的长途跋涉是由雷纳夫·法因斯先生（领队）和迈克尔·斯特劳德博士完成的，长达2170公里。两人于1992年11月9日从古尔德海湾出发，1993年1月16日到达南极，并于同年2月11日最终放弃在罗斯冰架的行走。

步行至地磁北极的最年轻的人

　　1997年5月7日，贾尔斯·克肖（英国）在行走至少580公里之后，到达了地磁北极。他当时的年龄为22岁零351天。

首次穿越大西洋的单人飞行

　　第一个单人飞越大西洋的人是美国明尼苏达州的查尔斯·林白空军上尉（后来成为空军准将）。林白于1927年5月20日格林威治标准时间下午12时52分从美国纽约长岛的罗斯福基地起飞，1927年5月21日格林威治标准时间晚上10时21分在法国巴黎布尔歇机场降落。全程5810公里，历时33小时29分。林白驾驶的是装有165千瓦（220马力）发动机的圣·路易精神号瑞安单翼飞机。

第一个穿越大西洋单人飞行的女人

　　1932年5月20日，美国堪萨斯州的阿米莉亚·埃尔哈特成为第一个完成穿越大西洋单人飞行的女人，也是第二个完成空成大西洋单人飞行的人。她驾驶一架单引擎洛克希德韦格飞机，从加拿大纽芬兰的格雷斯港起飞，飞至爱尔兰共和国的伦敦德里，历时13小时30分。后来埃尔哈特成为第一个从美国夏威夷火奴鲁鲁至加拿大奥克兰，穿越太平洋的进行单人飞行的人。接着，她又继续进行了两次环球航行的尝试，但未成功。1937年7月2日，在进行第二次尝试，完成35400公里的航行之后，埃尔哈特又与她的领航员从巴布亚新几内亚的莱城起飞，飞往太平洋的豪兰岛，从此人们再也未看到他们。

完成冒险行动大满贯的第一人（图片）

　　英国冒险家大卫·亨普尔曼-亚当斯（下图）是第一个完成冒险行动大满贯的人。这是一项令人历尽煎熬的挑战，它包括攀登每个大陆的最高峰和到达所有四个极点。亨普尔曼-亚当斯于1980年攀登阿拉斯加州的麦金利山，开始了对此目标的追求。18年后的1998年3月5日，他与他的冒险家伙伴鲁内·耶尔得恩斯 完成了向北极的行程。
