身高和体重

最重的模特（图片）

　　美国模特泰勒在90年代早期达到了326.14公斤的最大体重，但她的模特生涯非常成功，她演电影，做过广告，玉照还出现在明信片上。当体重影响到她的身体健康时，她开始减肥并减掉了81.65公斤。

最重的人

　　医学史上最重的人是住在美国华盛顿州班布雷奇岛的焦·米诺克，他身高1.85米，当他1978年因心脏病和呼吸困难被送进医院时，体重为635公斤，但他的心脏病和呼吸困难的主要原因是尿潴留。12名消防队员用一副临时担架将他送到渡口，然后又送他到医院，在医院里他的庆是两张病床绑在一起的。此后他开始节食，每天吃含1200卡执量的食物，体重降到了216公斤，但他在1983年去世时，体重又升到了362公斤。

　　最重的女性是美国人罗莎莉·布拉德福德，1987年1月她体重最高达到544公斤。之后她得了心脏病，为了保命，被迫减肥，到1994年她的生日,体重达到了128.3公斤，1998年8月4日她在《黄金时段：吉尼斯世界纪录电视展播》中露面。

最重的单胞胎

　　加拿大人安娜·贝茨于1879年在美国俄亥俄州的塞维尔生下了一个体重为10.8公斤的男婴。

最重的双胞胎

　　世界上最重的双胞胎是美国北卡罗来纳州亨德森维尔的比利和班尼·麦克拉里兄弟俩。他俩6岁前的体重都很正常，但1978年11月比利和班尼的体重分别达到337公斤和328公斤，他们的腰围都为2.13米。当他们报名参加摔跤比赛时的体重为349公斤。

腰围最大的人

　　美国人沃尔特·哈德逊1987年时的腰围是3.02米，当时他的体重是543公斤。

体重最轻的人

　　露西亚·沙拉特是墨西哥圣卡洛斯的侏儒，身高只有67厘米，她17岁时的体重是2.13公斤，到她20岁时，体重也才增加到5.9公斤。

体重最轻的单胞胎新生儿

　　据报道，1989年6月27日在美国伊利诺伊州的罗耀拉大学医疗中心降生了一个早产女婴，体重仅为280克。

　　存活下来的有记录的最轻单胞胎婴儿是玛丽安·塔格特，她出生时的体重仅为283克。她是于1938在英国泰恩-威尔郡早产6周出生的，当时只有30厘米长。在她出生后最初的30个小时内，布勒医生每隔1小时就用液体注入器喂她果汁、葡萄糖和水。

腰围最小的人

　　身高正常者中，腰围最小的是英国剑桥郡彼得伯勒的埃塞尔·格兰杰，她的腰围只有33厘米。1929年至1939年，她将正常的56厘米的腰围减至33厘米。19世纪的法国女演员埃米尔·玛丽·布沙尔也声称她的腰围是33厘米。

在世的最矮的女性（图片）

　　南非约翰内斯堡的马奇·贝斯特尔（左侧第3位）是在世的最矮的女性，身高仅为65厘米，体重30公斤。她患有成骨不全症（一种遗传疾病），这导致了她骨质疏松和其他的骨骼疾病。此处的照片是她在台北的一次新闻发布会上与中国台湾地区最矮的男性和女性的合影。

最高的人

　　无可争议，最高的人是美国人罗伯特·沃德洛，1940年他去世前的身高为2.72米，如果他能再多活一年，身高就会超过2.74米。世界上“真正的”（无病理原因的）的巨人是安格斯·麦卡斯基尔，当他1863年在加拿大去世时，身高为2.36米。

　　美国目前最高的人是马努特·博尔，身高为2.31米，他1962年生于苏丹，现为美国公民，他曾在费城76人队和其他得队打过球。

　　英国目前在世的最高的人是克里斯托弗·格林纳，他身高为2.29米。

　　突尼斯在世的最高的人是拉德胡阿尼·沙尔比，他于1999年4月22日至23日接受严格的测量，身高为2.359米。

　　最高的女性是中国湖南省的曾金莲，她1982年去世高为2.48米。

　　桑迪·艾伦是在世的最高的妇女，身高为2.317米，她10岁时身高为1.905米，她目前的体重为209.5公斤。最高的加拿大的安娜·哈南·斯旺和美国的马丁·冯·布伦·贝茨，他们于1817年结婚时身高分别为2.27米和2.20米。

最高的双胞胎

　　在世的最高的双胞胎是美国密歇根州托利的麦克尔和詹姆斯·托尼尔，他们生于1969年，现在两人的身高均为2.235米。

　　最高的女性双胞胎是美国加利福尼亚州帕洛斯弗迪斯的希瑟和海迪·伯奇，他们生于1971年，现在两人的身高均为1.95米。

平均身高最高的民族

　　世界上平均身高最高的民族是中非卢旺达和布隆迪的图西人，该民族年轻人的平均身高是1.83米。

在世的最矮的双胞胎（图片）

　　美国佛罗里达州西棕榈滩的约翰和格雷格·赖斯身高均为86.3厘米，使得他们成为在世的最矮的双胞胎。虽然身体矮，但这并没有妨碍他们成为成功的商人。70年代他们因投资房地产而赚到了一大笔钱，目前他们经营一个拥有几百万美元资产的名为“奇思妙想”的公司，该公司主要组织研讨会，探讨创造性地解决问题的方法。

最矮的人

　　有证据表明最矮的成年人是印度新德里的古尔·穆罕默德。1990年他身高57厘米，体重17公斤。1997年他36岁时因长期患哮喘和支气管炎而引发心脏病去世。

　　最矮的女性是保利娜·马斯特斯，她1876年生于荷兰奥森德雷赫特，出生时身长30厘米。她9岁时身 高仅55厘米，她19岁时因肺结核和脑膜炎死于美国纽约，死时身高为61厘米。

　　最矮的双胞胎是匈牙利布达佩斯的马蒂纳兄弟，他们的身高均为76厘米。

最高的人（图片）

　　在世的最高的人突尼斯的拉德胡阿尼·沙尔比。他在1999年4月22日测得的身高是2.359米。

平均身高最矮的民族

　　刚果民主共和国（前扎伊尔）的姆布蒂人中男性的平均身高为1.37米，而女性的平均身高烧1.35米。姆布蒂人的孩子在青少年时期比其他的孩子要矮得多，因为他们分泌的类似胰岛素的生长因子极少。

身高变化最大的人

　　奥地利人亚当·赖纳21岁时身高仅为1.18米，此后他的身高飞速增长，到32岁时达到2.18米。身体由于以史无前例的速度增高而变得非常虚弱，以后的日子他不得不在病床上度过。1950年他51岁时又长高了16厘米，达到了2.34米的高度。

身高差距最大的夫妻

　　1990年，当94厘米高的纳塔莉·卢修斯与1.885米高的法比安·普雷托在法国的塞斯耐特-帕里赛特结婚时，创下了新郎新娘身高差距最大的纪录 。
