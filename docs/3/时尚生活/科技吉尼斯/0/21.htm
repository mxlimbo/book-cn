发明创造

最小的轻便录像机（图片）

　　索尼CCD-CR1 Ruvi的体积为12.5×厘米×6.7厘米×4.4厘米，它可以储存30分钟的移动画面。摄影机有一个6.35平方厘米的LCD屏幕和一个光学变焦镜头。录像带与录音及播放磁头一起封闭在一个塑料暗盒里，每换一个暗盒，磁头就要插进或退出。

最小的蜂窝电话

　　由日本电报电话公司生产的PHS（个人手提电话系统的英文缩写）是一种手表式电话，舍去了传统的按键，用声音识别电路来选择号码。每部电话机重70克，体积为5.5厘米×4厘米×1.6厘米。

最小的固体存储器

　　珊迪多媒体卡是由珊迪公司及西门子公司开发的，用于移动电话及数码声音录音等移动装置，体积为3.2厘米×2..4×厘米×0.14厘米，可以储存10兆字节的信息。

最小的录像机

　　索尼公司的EV0220微型8毫米型机的重量是680克，体积为6厘米×21.7×厘米×14.6厘米。它可以有和8毫米录像带录像5个小时。

最薄的微型激光唱盘机（图片）

　　索尼MZ-R55只有1.89厘米厚，加上锂电池及碱性AA电池，它的重量仅为90克。

最小的图像发射器

　　AE公司生产的VID1可以无线传送方式把图像传送到609米远的基地。它的体积为1.5毫米×2.28×毫米×0.76毫米，它发送900MHz的PAL或NTSC制式的录像，而不需要大输出功率，可以连续使用11小时。

最小的VCD机

　　自带屏幕的最小的VCD机是松下SL-DP70机，体积为13厘米×3.6厘米×14.4厘米。它可以用6节AA电池连续播放2小时，价格为528美元。

最畅销的MP3机（图片）

　　帝盟里奥PMP300MP3机自从1998年11月首次上市到1999年5月，已出售400万台。可以在互联网上得到的MP3格式是一种存储及检索视听文件的数码方法。PMP300机的体积是9厘米×6.5厘米×1.5厘米，可以储存60分钟的数码质量的声音。MP3机比一盘磁带还小，它没有可活动的零件，所以从不漏读。

最小的DVD机显示器

　　“松下掌上影院”的面积为40.7平方厘米，厚24.2厘米，不带电池的重量为911克。它14.73平方厘米的液晶显示屏幕有28万象素，可以放映16:9与4:3纵横比的电影。

最防震的CD机

　　由三洋-菲舍公司制造的PCD-7900是第一台装置了40秒防震记忆的个人CD机，它可以弥补由外来震荡而引起的唱碟跟踪错误。这防震功能非常显著，即便是更换音乐碟时都不影响使用者的收听。

最昂贵的功率放大器

　　Audio Nlofe Ongaku 价值9.3万美元，是世界上最贵的功率放大器。Ongaku是一种A级输出结构的整流放大器，只要电力充足，它能给你一个纯净的音质。价格如此昂贵的主要原因是输出变压器上缠着一道固体银圈。

最周到的抽水马桶（图片）

　　由日本东陶公司于1997年5月首次出售的小洗神Z0马桶有一个座椅、一个可以自动打开的盖子及“冲水模拟效果”棗 即可以掩盖任何令人尴尬的声音的一种声响效果。座椅的温度可以调节、马桶可以为使用者进行冲洗并烘干。整个装置可以遥控，并且每次使用后它可以自动使空气恢复清新。它的零售价是699美元。该公司下在开发另一种马桶，它可以验尿、测血压，并能通过嵌在里面的调制解调器把资料输送给大夫。

最贵的35毫米SLR照相机

　　佳能EOSIN-RS价值3840美元，是当今世界最贵的35毫米SLR照相机。IN-RS的快门速度为从1/8000秒到30秒，可以接受从25到5000ASA的胶卷速度，并且每秒钏可拍摄3张照片。它的五棱镜式取影器可以或得映到胶片上的全部景物。相机的主体部分可以配装任何佳能EF变焦镜头。

最小的电脑

　　最小的手提电脑是“塞恩系列5”，它的重量包括电池在内是345克，它有一个打字键盘和一个触觉键盘。

最小的计算器

　　位于瑞士苏黎世的IBM公司研究实验室的科学家们设计出一种直径不到0.000001毫米的计算器。

最畅销的“媒人”（图片)

　　由日本爱尔弗公司制造的“电子求偶机”，从1998年2月上市起已售出130多万台。这种手掌般大小的电子信号机的零售价为22美元，它能够识别周围4.6米范围的异性手中的“电子求偶机”所发出的信号。它是由高藤发明的，其意图是创造出一种能够识别他人思想的仪器。

“智能”最高的笔

　　“智能笔”由英国电讯公司开发，具有日记、日历、数据库、警报器、记事本、计算器、电子邮件信箱及普通笔的功能。在内部信息下载出去前，它可以储存相当于10页A4张纸的内容。硅条可以感受到手指的压力，这样不管你的书写有多糟糕，这种笔都可以辨别出来。“智能笔”预计将于2001年之前上市，价格大约在200英镑。

最小的文件扫描器

　　佳能扫描器300S使用的是佳能公司的LED间曝光技术，它的重量只有1.5公斤。

最小的文件粉碎机

　　“皮兰海”PR026体积为17厘米×6厘米×4厘米，可以粉碎任何文件，它先咬掉纸片，再把纸片粉碎成纸条。

最小的传真机

　　飞利浦公司新型电话的附件连接在飞利浦PCS1900数字电话上业发送传真和电子邮件，还可以上网和提供通讯服务。PCS1900只有17厘米，159克重，它是最小、最轻便的移动传真机。

最小的双筒望远镜

　　U-C8×18系列双筒望远镜的放大率为8倍，可以聚焦2米，它的重量为145克，体积为8.5厘米×7厘米×1.8厘米。

最便宜 GPS接收器

　　在今世界上最便宜的便携式的卫星全球定位接收器（GPS）是“GPS先锋”，它是由美国麦哲伦系统有限公司制造的。“先锋”是第一个价格低于100美元的GPS接收器，它可以识别发自由24个同步卫星组成的“导航星”网络的信号，这样一来使用者就可以在地球上随时确定自己的位置。
