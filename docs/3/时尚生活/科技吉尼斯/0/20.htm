卫星与通讯

太空最大的建筑（图片）

　　国际太空站是太空中最大的建筑。该建筑将于2004年竣工，它长为79.9米，翼展为108.6米，重达456吨。它将由44次发射来完成，是目前最大的国际太空工程。美国、俄罗斯、加拿大、日本、巴西及11个欧洲国家参与了此项工程。

移动电话的最大覆盖面积

　　1998年5月，“铱星”移动电话网发射了最后5颗通讯卫星，从此该网的规模已达到66颗卫星棗这是目前最大规模的通讯卫星网络。该系统由摩托罗拉公司代表“铱星”网管理，为人们提供了世界上最大的移动电话网。该网的电话比一般的移动电话稍大，铱星网可以让用户从地球表面任何地方拨打电话。图中所示是一位科索沃难民在马其顿难民营里正使用铱星网的卫星电话与亲人联络。

最受欢迎的通讯卫星型号

　　世界上最畅销的商业通讯卫星是休斯太空通讯卫星HS601,于1998年接下73份订单。使用HS601的用户包括美国海军、美国航空航天局。英国伦敦ICO环球通讯公司有12个卫星，该公司是HS601的最大用户。

从地球上能看到的最亮的人造卫星

　　“和平”号空间站是从地球上能看到的最亮的人造天体。用宇航员的术语来说，它差不多是零等星，与在夜空中离地球最近的半人马座星同属一个等级。

从地球上能够看到的最远的人造天体

　　1998年，美国国家航空航天局的“近地小行星会合点”宇宙飞船绕地球做了个弹弓式飞行，然后冲出去，奔向巨大的行星—爱神星。

　　1月时，该飞船稍稍向地球摆动一点，这样它就成了肉眼可以看到的星际飞行器。在4月1日，它被澳大利亚新南威尔士州鲁伯拉的天文学家戈登·加拉德拍摄焉。位于距地球3365万公里处，它成为能够在地球上看到的最远的人造天体。

商业人造卫星离月球最近的一次飞行

　　1998年，通讯卫星HGS-1因其运载火箭在发射不久后发生故障而被留在一个没有用的椭圆形轨道上。为了纠正它的轨道，飞行任务操纵器把卫星弹到月球周围。在这次意外飞行中，它从月球6212公里的上空飞过——这是发射商业卫星以来离月球最近的一次飞行。

信号最强的通讯卫星

　　休斯太空通讯HS702卫星能够发射15千瓦的信号，是世界上信号最强的商业通讯卫星。它之所以有这样大的输出功率，靠的是两个高效太阳能电池。

仍绕轨道运行的最老的人造卫星

　　1958年3月17日，美国的“先锋I号”被发射到绕地轨轨道上。它是目前仍在绕轨道运行的世界上最老的人造卫星。

由航天飞机发射的最重的卫星

　　由美国航天飞机载送并放置的最重的卫星是“康普顿·伽玛射线观测卫星”，它重达17.27吨。该卫星是一架天文卫星，已在轨道上运转了8年，它的任务是研究高能量的射线。

发射量最多的发射者

　　俄罗斯及前苏联的加盟国已总共成功地发射了1337颗卫星。这是世界上发射卫星数量最多的国家。

通讯卫星产量最高的卫星制造者

　　休斯太空公司的总部设在美国加利福尼亚州洛杉矶。该公司已向通讯部门提供了137颗通讯卫星棗 占目前正在使用卫星中的40%。

最大的商业卫星工厂

　　美国加利福尼亚州艾尔·西戈多市的休斯整体卫星工厂占地5.6万平方米，是世界上最大的商业通讯卫星厂家。该厂现在是休斯太空通讯公司卫星制造中心。

损失最惨重的无线电通讯事故

　　1998年5月，出现了无线电通讯系统最严重的事故。由“潘安赛”制造的价值2.65亿美元 的信斯太空通讯卫星“银河4号”发生了故障。结果，美国大约有4100万人的传呼机立刻失灵。

最大的太空垃圾制造者

　　对于太空飞行任务的设计者来说，包括残留在轨道上的火箭残骸在内的太空垃圾是一个不断困扰的麻烦。美国已在地球周围的轨道上留下了总共3173片太空垃圾，是最大的太空垃圾制造者。

最大的射电望远镜

　　我们安放在宇宙间的最大的无线电耳朵是波多黎各的阿雷西沃无线探测仪。该望远镜上的巨大的反向镜的直径为305米。阿雷西沃探测仪目前被用来搜寻空中的由外星智能生命发射来的信号，它也曾在电影《黄金眼》（英美合拍，1995）及《接触》（美国，1997）中担当主要的道具。

效率最高的太阳能板

　　尽管地球上大多数的太阳能板是用硅做的，而卫星上的太阳能板的材料是砷化镓—一种比硅轻却更耐辐射的化合物。这些太阳能板是世界上效率最高的太阳能板，能把它们接受的27%光能转化为电,几乎是从前的2倍。

最细的光纤

　　英国巴斯大学的物理学家们研究出世界上最细的用于通讯的光缆。每根光缆长为10公里，每个结仅有0.00000001毫米粗。若按它的直径与长度的比来计算，这相当于所英吉利海底隧道从地球延伸到木星。
