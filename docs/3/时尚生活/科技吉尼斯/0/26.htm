杀伤性武器

最大的潜水艇（图片）

　　俄罗斯的“Akula”潜水艇（见下图）全长1715米，排水量为26500吨，北约给它起的代号为“台风”号。据北约报道，第一艘“台风”号潜水艇是1980年9月23日在位于白海北德温斯克的秘密造船厂下水的。据说目前尚有6艘“Akula”潜水艇在服役，每艘都配有20枚装有多头核弹头、射程为8300公里的SS-N-20导弹。

原子弹杀伤人数的纪录

　　1945年8月6日，美国在日本广岛投下的原子弹顷刻间使10万人丧命。而后一年中,又有5.5万人死于辐射病。

威力最大的核武器

　　威力最大的洲际弹道导弹（ICBM）是前苏联的SS-18（第五代）。人们认为该导弹装有1075万吨梯能量的多弹头分导再入飞行器（MIRV）。所有SS-18洲际导弹都于1995年底之前运回俄罗斯，有150枚洲际导弹还在那里担负战斗任务。如果第二期削减战略武器会谈能够完全实行，其余SS-18洲际导弹和其他多弹头的洲际导弹都被销毁。

核能量最大的热核装置

　　1961年10月，在新地岛地区,带有相当于57兆吨梯恩梯的热核装置由前苏联爆炸成功。其冲击波绕地球3周，第1周用时36小时27分钟。估计该热核装置携带的能量达62兆吨—90兆吨的梯恩梯。

航程最长的攻击

　　1991年1月，海湾战争伊始,7架B-52G轰炸机从美国路易斯安那州马克斯代尔空军基地起飞，运送打击伊拉克的巡航导弹。每次运送往返航程22526公里，用时35小时，其间加油4次。

　　1996年9月，B-52轰炸机从太平洋西部的关岛起飞，在空中直接发射巡航导弹打击伊拉克巴格达周围的目标，往返航程为20284公里。

射程最远的导弹

　　1959年，美国大力神导弹开始服役，射程为16669公里，比在西方导弹基地打击苏联领土内任何目标所必需的射程还多4828公里左右。射程最大的俄罗斯导弹是SS-18,代号“撒旦”，它从20世纪80年代初开始服役，射程为12070公里。

最昂贵的军用飞机（图片）

　世界上最昂贵的军用飞机是美国的“鬼怪”B2号（见上图），耗资13亿美元。作为远程多用轰炸机，“鬼怪”B2既能运载常规炸弹，又能运载核炸弹，并有许多隐蔽性能，其中包括特殊覆盖层和机翼的设计。这使得它能突破敌人的防御而不被发现 。“鬼怪”B2能一次运载18.144吨弹药，航程可达9600公里而无需再次加油。它于1989年7月17单日试飞成功。

最先进的战斗机

　　美国F22“猛禽”战斗机是由洛克希德·马丁航空系统公司和洛克希德·巴西上沃思堡公司以及波音公司于20世纪90年代末联合联合研制的。耗资约133亿美元 ，这相当于欧洲战斗机公司耗资的两倍。F22的内舱能够运载多枚不同的空对空导弹：包括雷达制导空中截击导弹120C和中程空对空导弹（AMRAAMS），较长鳍的空中截击导弹120A和热跟踪短程AIM-9M空以空响尾蛇导弹。它的内舱还配有M61A220毫米机关枪，这是M61加特林式机枪的改进型。F22机身长18.9米，翼展13.4米。

最快的喷气式战斗机

　　世界上最快的喷气式战斗机是前苏联研制的米格25战斗机。北约给它起的代号是“狐蝠”。单座‘狐蝠A“机身长为23.83米，翼展为13.95米，最大起飞载重量为37.4吨。据雷达观测，侦察机“狐蝠B”的速度约3.2马赫。

最快的轰炸机

　　美国通用动力公司生产的可“变后掠翼”FB-111A的最快速度为2.5马赫。

　　俄罗斯图波列夫公司生产的“可变以后掠翼”图-22M，即北约称之为“逆火式轰炸机”，其速度估计为2马赫，也可能达到2.5马赫。

最快的驱逐舰

　　排水量为3251吨的法国“恐怖 ”号驱逐舰的最高速度达到了每小时45.25节（834公里）。该驱逐舰1935年建造于法国的布兰维尔，由4台亚罗小管锅炉和2台拉托传动涡轮机牵引，动力达10万马力 。于1957年底退役。

最先进的战舰

　　1998年7月，位于英国南安普顿的英国国防评估研究所（DERA）受命制造出世界上最大的铁制远洋三体船，它可能代表下一世纪海军舰艇的发展趋势。这艘柴油-电力并用牵引的“海卫1”号长达97米，排水量为1100吨，最高速度达每小时20节（37公里）.它先进的设计会比常规单体船减20%的阻力，在舰艇上装备的侦察系统稳定性更高.“海卫1”号将于2000年4月服役，在此之前要进行18个月的试航。

最大的无人驾驶飞机

　　正在发展中的最大的无人驾驶飞机被称为“巨鹰”号，1997年2月20日在美国加利福尼亚州圣地亚哥市的泰莱达因·赖安航空中心首次露面。1998年2月28日在美国加利福尼亚爱德华兹空军基地进行了试飞。它的翼宽35.4米，航程可达22526公里，将为美国空军担负空中侦察任务。

威力最大的鱼雷

　　俄式65，即66厘米鱼雷，装有一枚将近1吨常规炸药的弹头或一枚1.5万吨梯恩梯能量的核弹头，它的威力略小于1945年轰炸广岛和长崎的原子弹。它能通过声纳追踪位于80.5公里以外的舰艇的尾流，以每小时43.5节（80.5公里）的速度发射，这远比最快的水面舰艇的速度还快。

最先进的直升飞机

　　俄罗斯Ka-52“短吻鳄”号是世界上最先进的直升飞机。它是由双座Ka-50“黑鲨”号改装而成的，原本主要用做武装直升机，装备有各种设备和武器，在极端恶劣的天气条件下也能完成特殊的飞行任务。

精确度最高的便携式防空导弹（图片）

　　在军事演习中，有人目睹过美国海军使用的“螫针”导弹架。在20世纪80年代初引进的“螫针”导弹长1.52米，重10公斤，射程为4.83公里，速度约每小时2092公里。“螫针”导弹的低温冷却红外线自导头把飞机红外信号和照明弹一类的反措施物区分开来。在20世纪90年代初，美国军队验收了“螫针”无源光学自导头技术（POST），通过预定程序微处理机，导弹飞向目标。该导弹会“思维”，一旦锁定，目标飞机的导向器使导弹或目标更快飞去或弹射出来，没有多少其他的选择。

转速最快的机枪

　　口径为7.62毫米的M134微型机枪（见上图，安装在科布拉武装直升机前部的旋转枪架上）是世界上转速最快的机枪。在20世纪60年代末，它设计安装在武装直升机和装甲车上。在多枪管加特林设计的基础之上，它装有6根枪管，由一台电机使其旋转和一条4000发子弹，这一速度大约相当于普通机枪的10倍。
