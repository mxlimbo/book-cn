深水英雄

最伟大的沉船发现者（图片）

　　1985年，罗伯特·巴拉德因确定了1912年因撞冰山沉没于大西洋底的“泰坦尼克”号轮船残骸的准确位置而闻名于世。图为巴拉德和他发现的这艘倒霉的船的模型。他还发现了1942年被英国皇家海军击沉的德国战列舰“俾斯麦”号、1915年被德国潜水艇用鱼雷炸沉的英国“卢西塔尼亚”号豪华远洋客轮，以及“安德里亚·多里亚”号，“大不列颠”号等沉船。1997年7月他精确探明了位于突尼斯海岸附近古地中海商船航线上的8艘古罗马沉船的位置，这些沉船距今约有2000年，沉于水下762米处，这是迄今为止所发现的古罗马沉船最集中的区域。

最深的海洋下潜

　　1960年1月，瑞士制造的美国海军深海潜水器“特瑞斯特”号在雅克·皮卡德博士（瑞士人）和唐纳德·沃西海军上尉（美国人）操纵下下潜到太平洋底马里亚纳海沟中10911米深的查林杰海渊，创最深下潜纪录，这也是地球表面的最低点。

　　1989年8月11日，“真贝6500”号潜艇潜入日本三陆附近6527米深的日本海沟，这是迄今为止载人船舶所能下潜的最大深度。

深潜的最长时间纪录

　　1992年5月6日至7月14日，美国人理查德·普雷斯利驾驶一艘潜水艇在美国佛罗里达州基拉戈的一个潟湖里度过了创纪录的69天零19分。这次深潜试验名为亚特兰蒂斯工程，旨在考察人在水下环境中的生存状况。

最长时间的模拟饱和深潜

　　法国人阿诺德·德尼肖德·德菲哈尔于1989年10月9日至12月21日，在一个高压舱里进行了为期73天的模拟饱和深潜，创最长时间纪录。此次试验的模拟水深为300米，属于科梅克斯公司在法国马赛进行的“九头蛇9号”计划。德菲哈尔呼吸的是一种氢气和氧气的混合气体。

最长时间的模拟深潜

　　在为期43天的潜水试验中，法国人陶·马夫罗斯托摩斯于1992年11 月20日在高压舱里深入深度为701米的海水中，这是科梅克斯公司在法国马赛进行的“九头蛇10号”计划，他呼吸的是混合气体（由氢气、氧气和氦气组成）。

最深的戴水肺的潜水

　　戴水肺（自携式水下呼吸器）潜水的最深世界纪录是由美国的吉姆·鲍登保持的。1994年4月他在墨西哥的萨卡坦淡水洞穴中创造了这项纪录，下潜深度为305米。

最重要的潜水发明（图片）

　　图为法国探险家兼电影摄制师雅克·库斯托和他发明的水肺型潜水器。1943年他与法国工程师戈南合作发明的这种自携式压缩空气呼吸装置结束了潜水员通过与船相连的呼吸管呼吸的历史。他乘“卡利普索”号船进行海洋勘探，所拍摄的《寂静的世界》（法国，1952）等影片向人们介绍了数百万种海底生物。

　


最深的海上打捞

　　海底作业的最大深度是5258米。1991年8月一架直升飞机坠毁于太平洋，机上4人全部遇难。为调查事故原因，美国一艘打捞船的船员及伊斯特波特国际求援人员于1992年2月27日在打捞这一失事飞机残骸时创造了海上打捞的最深纪录。

　　潜水员海上打捞的最深纪录为245米，是在打捞1942年5月2日沉没于挪威北部巴伦支海的英国皇家巡洋舰“爱丁堡”号残骸时创造的。在1981年9月7日至10月7日历时31天的打捞期间，12名潜水员每两人一组轮番下潜作业，随“爱丁堡”号沉入海底的460块金锭全部被打捞上来，创造了迄今收益最多的打捞纪录。

价值最高的沉船

　　1985年，由20世纪最著名的寻宝者棗已故的美国人梅尔·费歇匀美国佛罗里达州基韦斯特海岸附近发现的“圣母湾”号沉船是价值最高的沉船。该船于1622年9月遇风暴沉没 ，船上载有40吨黄金和白银，以及约31.75公斤绿宝石。

发现海盗船残骸最多的人（图片）

　　美国人巴里·克里福德发现了3艘经核实的海盗船残骸，创造了发现海盗船的数量之最：1998年7月他于美国马萨诸塞州科德角半附近海域发现了18世纪的海盗船“寡妇鸟”号残骸，同年11月在委内瑞拉海岸附近发现了两艘17世纪的海盗船。图中克里福德正拿着一枚从17世纪海盗船上发现的炮弹。

失事潜水艇上发现最多的黄金

　　1995年5月，美国人保罗·蒂德维尔从一艘长108.8米的日本失事潜水艇上发现两吨黄金、228吨锡、54吨橡胶和3吨奎宁，创造了失事潜水艇上发现黄金最多的纪录。1944年6月23日该艇遭美国226.8公斤深水炸弹的攻击而沉没，当时艇上载员109人。该艇残骸位于大西洋佛得角群岛以西1931公里处5180米深的海底。

沉船中发现陶瓷最多的一次

　　1994年，寻宝者多里安·鲍尔和他的马来西亚文物打捞公司（MHS）从1817年沉没于马六甲海峡的英国商船“戴安娜”号中打捞上来价值500万美元的中国清瓷器，创造了从沉船中打捞陶瓷价值的最高纪录 。
