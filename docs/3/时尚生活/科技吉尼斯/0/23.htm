自行车与摩托车

最小的独轮自行车（图片）

　　最小的能骑行的独轮自行车的高度为20厘米,车轮直径为1.8厘米,无附加设备。该车由瑞典人西格纳·贝尔格伦德造。1998年3月29日，在德国下福林市的ZDF电视台播出的《让你惊讶》节目中，瑞典人彼得·罗森达尔多次表演骑自行车，最远行程为8.5米。

最大的自行车

　　以车轮直径的大小而论，世界上最大的自行车是“弗兰肯”自行车，其制造者是美国加利福尼亚州罗斯米德市的戴夫·摩尔，由加利福尼亚州莫尔帕克市的斯蒂夫·高登于1989年6月4日首次骑乘。该自行车的车轮直径3.05米，车身高度为3.4米。

速度最快的摩托车（图片）

　　据报道，铃木“游隼”Gsx1300R型摩托车的速度可达每小时312公里，是世界上最快的摩托车。该车为1298毫升，每缸有四个阀，阀的角度为14° ，有冲压空气式电子燃油喷入装置。转速为每分钟9800转时的功率为173制动马力，车身重量为215公斤。

最长的自行车

　　最长的“真正自行车”（其定义为不带稳定轮的两轮自行车）是由新西兰巴希亚特瓦市的坦利·塞斯曼设计和制造的。该车长22.24米，重量为340公斤，1988年2月27日四人共乘，行进246米。该车只能直线骑行,不能转弯。

最小的自行车

　　世界上最小的能骑行的自行车的车轮直径为1.9厘米。1988年3月25日，该车由其制造者澳大利亚昆士兰州格拉德斯通市的纳维尔·帕登骑行，行程为4.1米。

最高的独轮自行车

　　1980年10月，在美国内华达州的拉斯维加斯，美国人斯蒂夫·麦克皮克骑一辆31.01米高的独轮自行车，行程为114.6米。他身上系着安全索，安全索的另一端固定在一架起重机的悬臂上。

最大的三轮自行车

　　1998年7月，美国密歇根州埃斯卡巴纳市贝德诺克社区学院的16名学生，设计制造了世界上最大的三轮自行车。该车前轮直径为4.67米，后轮直径为2.23米，车身高7.13米。

最小的串座双人自行车

　　法国大西洋沿岸比利牛斯省波城的雅克·比尤制造了一辆双人自行车，长度仅为36厘米。该车由比尤本人及其妻子骑行。

最昂贵的山地自行车

　　“鳍足”型自行车是世界上最昂贵的山地自行车，售价为7244英镑（12 025美元）。该车由英国的斯蒂夫公司生产，重量只有9.07公斤，是专门为越野赛设计的，世界上最昂贵的自行车零件组装而成的车—车闸、齿轮和脚蹬来自日本，车架来自美国，前后叉和车所来自英国，鞍座来自意大利。

折叠式自行车的最大生产厂家

　　美国加利福尼亚州的达宏公司折叠式自行车的最大生产厂家。自1982年以来，该公司生产的自行车已超过1200万辆，其中尤以“经典”型车最畅销，在世界各地售出40万辆。

最昂贵的摩托车（图片）

　　世界上最昂贵的摩托车是“摩比德利850V8”型车，这种车的零售价为102872美元。

最长的摩托车

　　西澳大利亚珀斯市的道格拉斯和罗杰·贝尔兄弟俩设计制造了一辆7.6米长的摩托车，重约2吨。

最小的摩托车

　　英国大曼彻斯特郡进步工程公司的西蒙·蒂姆伯利和克利夫·威廉姆斯共同设制造了一辆摩托车，勘化府长10.8厘米，车座高9.5厘米，前轮直径为1.9厘米，后轮直径为2.4 厘米。有人曾骑乘该车行驶1米。

最大的摩托车生产厂家

　　日本的本田汽车公司是世界上最大的摩托车生产厂家。该公司在34个国家开设95个工厂生产汽车和摩托车。1998年本田公司向世界各地的零售商售出510万辆摩托车。

最轻的折叠式自行车（图片）

　　由日本的全国自行车工业公司、松下电器工业公司和东日本铁道公司联合设计生产的“三角牌”6500B-PEHT23型自行车的重量只有6.5公斤。车架和前叉有钛制造。车身很小，可以携带火车或放进车站的存衣柜里。

最早的摩托车

　　第一辆由内燃机驱动的两轮车锷 “爱因斯伯车”，这是1885年德国巴德-康斯塔特市的哥特利勃·戴姆勒制造的一种机动车，车架用木头制造。威廉·梅巴赫首次骑行该车。发动机为单缸264毫升四冲程，每分钟700转，最高车速为每小时19公里。

容积最大的小型摩托车

　　铃木－伯格曼400毫升摩托车是目前生产的容量最大的小型摩托车。这种车在转速为每分钟8000转时的动力输出为31.5制动马力，转速为每分钟6000转时的转矩输出为每平方英尺14.46磅。

最昂贵的小型摩托车

　　铃木AN400型是目前生产的最昂贵的小型摩托车，售价为6628美元。

历史最悠久的小型摩托车制造厂家

　　意大利的维斯珀公司是历史最悠久的小型摩托车制造厂家。最早的维斯珀小型摩托车于1945年10月试制成功，于1946年4月开始生产。1949年该公司获得英国皇家特许证，为爱丁堡公爵提供小型摩托车。
