电视和录像

最成功的连续剧（图片）

　　《达拉斯》一剧由拉里·哈格曼（图中左侧）和帕特里克（图中右侧）主演，于1978年开始播出，到1980年为止每周吸引8300万美国观众（占全部电视观众的76%）。该剧有356集，从1978年4月播放到1991年5月，还在130个国家播出过。

最多产的电视节目制作者

　　从1956年到现在，阿伦·斯佩林共制作了3842集电视节目。如把这些节目首尾相接播放一遍需128天。他的作品包括由法拉·福塞特-梅杰斯、凯特·杰克逊和杰克林·史密斯（下图）等主演的《查理的天使们》，《星空和棚屋》，《王朝》以及由他的女儿托莉主演的《贝弗利山庄90210》。他得到过两项金球奖和两项埃米金像奖。

标的最大的电视合同

　　据1998年12月24日宣布的消息，金世界制作公司将预付给奥普拉·温弗里的哈珀制作公司7500万美元，以购买2000/2001年和2001/2002年这两个季度温弗里主持并制作的记谈节目。

最昂贵的电视转播协议

　　1998年1月，美国全国橄榄球联盟（NFL）与CBS、ABC、福克斯和ESPN签订了一份总额为176亿美元的协议。协议规定从1998/1999赛季开始的8年内，上述各电视台有权转播NFL的各场比赛。ESPN每年付给NFL6亿美元，ABC和福克斯每年各付5.5亿美元，CBS每年付5亿美元。

最昂贵的电视节目

　　1998年1月，全国广播公司（NBC）同意为每集1小时的医学剧《ER》支付1300万美元。NBC曾为每集支付过160万美元。该节目在美国黄金时间播放，每周观众达3200万人。这一协议为期3年，共66集，支付给《ER》制作者华纳兄弟公司的金额总数达8.7368美元。

观众最多的电视台

　　国有的中国中央电视台（CCTV）的观众占全中国观众的84%，人数估计超过9亿。该台有8个频道，每天播出时间总数为162小时，有普通话、粤语、英语、法语播出。收视率最高的节目是每天的《新闻联播》，观众达3.15亿人。

最大的全球性电视网

　　CNN国际台可以通过23个卫星的电视网在212个国家和地区的1..49亿户家庭中收看到。

最多的电视观众

　　美国的救生员剧《护滩使者》由大卫·哈塞尔霍夫和帕米拉·安德逊主演，成为世界上收视率最高的电视连续剧。据估计在142个国家和地区内每周收看该剧的人数超过11亿。

　　关于朝鲜战争的黑色喜剧片《野战医院》的最后一集《再见吧，阿们》于1983年2月28日吸引了全美国77%的电视观众，收看的人约有1.25亿。

观众最多的审判

　　1995年1月至10月，每天平均有550万美国观众收看O·J·辛普森一案的审判。辛普森是美国橄榄运动员和演员，他被控谋杀了前妻尼科尔及其男友罗纳德·古尔德曼。

规模最大的电视连续剧制作者

　　巴西的环球电视台是拉丁美洲规模最大、赢利最多的电视连续剧制作者，每个剧平均都要超过100集。该电视台每晚6时开始播出连续剧。60年代的连续剧《憨玛丽》卖给每一个讲西班牙语的国家，有一次其收视率竟超过了世界杯足球赛。

　　连续剧《女奴》，出口到所讲西班牙语的国家，并被译成汉语在亚洲播出。

播放时间最长的连续剧

　　英国连续剧《加冕街》描写英国曼乇斯特市一个并不存在的威瑟菲尔德区居民的生活。该剧从1960年12月9日开始播出，直到现在每周至少播出2集。该剧由托尼·华伦设计，由格拉纳达纳电视台摄制。

最长的宗教连续剧

　　《摩诃婆罗多》是印度最受欢迎的连续剧，从1994年到1996年连续两年每星期日早上播出。

耗资最大的连续剧

　　巴西连续剧《巴别塔》摄制费用为1700万美元，每集耗资10万美元。故事由西尔维奥·德·阿毂罗编写。

耗资最大的中国连续剧

　　《三国演义》耗资1296万美元。该剧于1996年摄制，共44集，参加摄制者达30万人。

演员最多的连续剧

　　《巴别塔》创演员人数最多的纪录，该剧长期签约的演员有300人，但每次用到的演员不超过180人。

耗资最大的电视智力竞赛

　　日本电视台（NTV）于1990年摄制的《泛美超级智力竞赛》每集费用高达657030美元，比普通智力竞赛高出3倍。这一节目是在外国的不同地方摄制的，包括夏威夷 、欧洲和南美洲。

参赛人数最多的智力竞赛

　　参加日本的《泛美超级智力竞赛》的人数一次达到28523人，创最高纪录。这一竞赛从1977年到1993年长达16年间，共有参赛者213430人，其中只有31人进入在美国纽约的自由女神像前举行的决赛。

产品最多的智力竞赛节目制作者

　　马克·戈德森制作的智力竞赛节目数量最多，他共制作了3.9万多集，播放总时间达21240小时。

一部电影最昂贵的电视播放权

　　在斯蒂文·斯皮尔伯格导演的《侏罗纪公园 失去的世界》在国外上演之前，福克斯公司以8200万美元的代价购买了它的电视播放权。

一部微型连续剧最昂贵的电视播放权

　　1991年，以CBS为首的美国和欧洲几家投资商以800万美元的代价习下了亚历山德拉·里普利的《斯佳丽》的电视播放权，该剧是玛格丽特·米切尔1936的小说《飘》的续集。

出租收入的最高录像片

　　乔治·卢卡斯的《星球大战》（美国 1977）在北美一地的录像带出租收入即达2.709亿美元。这一收入包括该片原版录像带发行的收入及1997年数字新版发行的收入。《星球大战》三部曲的原版和新版，包括《帝国反击》（美国 1980）和《杰迪人的归来》（美国 1983）已从出租录像带获得总收益6.364亿美元。

最快的录像带制作

　　1986年7月23日，安德鲁亲王和莎拉·富格森的婚礼的录像带由泰晤士录像公司制作。现场摄像于当天下午4点42分结束，5小时41分之后，于晚上10点23分首批VHS录像带于英国的伦敦出售。

电视和录像制品的最大消费者

　　电视和录像制品的最大消费市场是日本，每个日本人每年在这方面的平均支出是43.66美元。

拥有电视机最多的国家

　　中国有2.275亿个家庭拥有电视机，为世界之最。

拥有录像机最多的国家

　　81%的美国家庭（7812.5万户）拥有至少一台录像机。

耗资最大的直接上电视的影片

　　多米尼克·斯万在影片《洛丽塔》（美国 1997）中任主角，该片共花去万美元，是在美国有线电视上首次放映的最昂贵的影片。

　　该片未获得在电影院首映的机会，后来有线电视网就购买了它的演出权。1998年8月2日，在美国电视上首次播出，一个月后才在美国电影院里上映。

拥有录像资料图书馆最多的城市

　　印度的孟买有1.5个录像资料图书馆和500家录像厅，超过世界上任何其他城市。
