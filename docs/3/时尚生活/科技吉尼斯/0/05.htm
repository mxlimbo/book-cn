速度明星

陆上最高速度（图片）

　　1.6公里（1英里）距离内的陆上行驶最高速度为每小时1227.985公里，是英国的安迪・格林于1997年10月在美国内华达州的黑岩沙漠上创造的，他驾驶的是一辆理查德・格林设计的推力SSC型汽车，装有罗尔斯・罗伊斯双缸喷气发动机，可产生22..68吨推力，该车是第一辆超音速汽车。

火箭车的最高速度

　　1970年10月23日，在美国犹他州的邦纳维尔盐滩，加里・盖布里奇驾驶一辆装有火箭发动机的蓝火牌汽车，以每小时1016.086公里的速度跑完开头1公里，创造了发动机给汽车提供了9979公斤的驱动力。

轮子驱动汽车的最高速度

　　轮子驱动汽车的最高时速为696公里，是1991年8月21日在美国犹他州的邦纳维尔盐滩由美国人艾尔・蒂格驾驶1976年产的“速度―动力/勇气”汽车在1.6公里路程的最后40米时创造的。他全程的平均时速为684.322公里。

女子陆上最快驾车纪录

　　世界上驾车最快的妇女是基蒂・姆勃莱登,她于1976年12月6日在美国俄勒冈州的阿尔伐特沙漠驾驶火箭动力的“SM1动力”型汽车，时速达到843.323公里。

最快的特种减重短程高速汽车

　　1998年5月，在美国新泽西州英吉利敦的一次比赛中，美国人约翰・福尔斯驾驶他的“98福特・马格斯腾”特种高速赛车从停车处出发行驶402米，车速达到每小时518公里。

蒸汽机汽车的最高速度

　　1985年8月19日，罗伯特・巴伯在美国犹他州的邦纳维尔盐滩驾驶“蒸汽魔王”时速达到234.33公里，打破了保持79年之久的蒸汽机汽车的纪录。

最快的灵车驾手

　　罗克・格里菲思1998年10月25日在美国加利福尼亚的波莫纳驾驶一辆1964年产庞蒂亚克灵车在16.058秒内行驶了402米，时速达到141公里。

倒车行驶速度之最

　　连续倒车行驶805公里以上的平均时速最高纪录是由美国的约翰・史密斯创造的，他倒车行驶的平均时速为58.42公里。1996年8月11日，他驾驶一辆1983年产雪佛兰超级变幻车在美国明尼苏达州弗格斯福尔斯的I-94号高速公路上,在13小时48分钟内倒车行驶了806.2公里。

两轮行驶的最高速度

　　1997年4月19日，瑞典的格兰・伊莱亚逊在瑞典哥德堡驾驶一辆沃尔沃850型涡轮发动机汽车，依靠两个边轮在100米距离内快速启动，创造了时速181.25公里的世界纪录。

最快的一级重力议程式赛车选手

　　经一级重力议程式汽车赛认定机构“国际非传统运动协会”确认，美国的德怀特・加兰于1998年9月28日驾车达到了时速103.3公里，成为最快的选手。这种赛车装有管钢底盘，使用充气磨光轮胎。车手有滚柱罩保护，还有一个五点安全套把他固定在座位上。

最快的高级燃料减重短程高速汽车（图片）

　　如图所示，采用高级燃料的减重短程高速汽车是一种由吉姆・巴特勒发动机扒动的加长汽车，其最高速度纪录是由1998年全国减重高速汽协会（NHRA）温斯顿杯汽车大赛冠军、美国的加里・斯尔兹保持的，他于1998年11月在美国得克萨斯州的休斯敦举办的一次比赛中驾车从停车状态出发行驶402米后，时速达到522..3公里。

冰上速度之最

　　冰上（无轨道）行驶的最高速度纪录是1981年2月15日由萨米・米勒在美国纽约州的乔治湖创下的，他驾驶着他的“氧气”号火箭动力冰撬，时速达到399公里。

最快的摩托车手

　　1990年7月14日，美国的戴夫・坎坡斯在美国犹他州的邦纳维尔盐滩创下摩托车最高速度的世界纪录。他驾驶的是7米长流线型的伊兹来德摩托车，装有两台拉克斯顿・哈利-戴维森发动机，发动机的汽缸容是来1491立方厘米。坎坡斯的全和平均时速为519.59公里。

摩托车后轮行驶的最高纪录

　　瑞典的帕特里克・弗斯坦霍夫1999年4月18日在英国莱克斯的布林迁索普试验场只用摩托车后轮跑出了307.9公里的时速。这一壮举在《吉尼斯世界纪录电视展播》中播出。

最快的自行车骑行速度

　　脚踏自行车的最高时速为268.831公里，是1995年10月3日由荷兰的弗雷德・罗姆贝尔埔格在美国犹他州的邦纳维尔盐滩所创。这一纪录的取得在很大程度上益于他的前导车产生的低压气穴。

最高飞行速度

　　1976年7月28日，埃尔顿・乔尔斯上尉和小乔治・摩根少校在美国加利福尼亚州的比尔空军基地附近创下时速3529.56公里的空中飞行最高纪录。这一纪录是在25公里的航程中完成的，他们驾驶的是一架洛克希德SR-71A型黑鸟飞机。

横跨大西洋的飞行速度之最

　　詹姆斯・沙利文少校和诺埃尔・威第弗尔德少校于1974年驾驶一架洛克希德SR-71A型黑鸟飞机，从美国纽约飞往英国伦敦，飞行时间是1小时54分56.4秒，这段航程全长5570.80公里。他们的飞机因途中在空中加油而减速，最后达到的平均时速为2908.02公里。

　　单人飞越大西洋的最短时间是8小时47分32秒，平均时速为426.7公里，这一纪录是约翰・史密斯上尉于1978年创造的。他驾驶的是一架罗克韦尔・克蒙斯上尉于1978年创造的。他驾驶的是一架罗克韦尔・克蒙德685双涡轮冲压式喷气飞机，从加拿大纽芬兰岛的甘德飞到英国西苏塞克斯的加特韦克机场。

最快的直升飞机

　　直升飞机的最高平均时速为400.87公里，是由约翰・埃金顿和德莱克・科鲁斯于1986年8月1日驾驶一架韦斯特兰・林耐克斯公司生产的示范直升飞机按国际航空协会的规定在英国萨默塞特郡的格拉斯顿伯里上空创下的。

最快的自行车冰川骑降速度（图片）

　　迄今为止最快的自行车冰川骑降速度为每小时212.139公里，是1998年3月由山地车选手法国的克里斯蒂安・泰勒菲骑乘一辆“标致”牌自行车在法国瓦尔速度滑雪场创造的。

最快的旋翼飞机

　　1986年9月18日，空军中校肯尼思・沃利斯驾驶着他的装有60马力富兰克林发动机的WA-116F/S旋翼飞机在3公里的直线航线上创造了时速193.6公里的飞行纪录，这一纪录是在英国诺福克郡的马海姆创下的。

水上最高速度

　　水上最高速度的正式世界纪录为每小时275.8节（511.11公里），是1978年10月8日由澳大利亚的肯・瓦尔比驾驶“澳大利亚之魂”号滑行快艇在澳大利亚新南威尔士的布洛尔英水坝湖上创的。1977年11月20日，瓦尔比曾驾驶着这艘快艇在同一湖上创下估计时速为300节（555公里）的最高速度，但这一纪录没有得到正式认可。
