内蒙古大学

学校概况

内蒙古大学是由内蒙古自治区教育厅主管的国家重点大学。1957年10月建校。

学校坐落在内蒙古呼和浩特市新城大学西路。校舍面积15万平方米。

学校现有教师总数577人。其中教授51人，副教授171人，讲师253人，教员48人，助教54人。

国家重点学科

蒙古族语言文学

本科专业设置

哲学、经济学、国民经济管理、企业管理、法学、经济法、蒙古语言文学、汉语言文学、新闻学、英语、日语、历史学、马克思主义民族理论与民族政策、数学计算、数学及其应用软件、应用数学、物理学、应用物理学、化学、应用化学、生态学、生物学、微生物学、计算机及应用、应用电子技术

授予硕士学位的专业

中国古代文学、少数民族语言文学、中国民族史、中国古代史、世界上古史·中古史、基础数学、应用数学、理论物理、无机化学、动物学、植物学、微生物学、生态学、通信与电子系统、马克思主义理论教育、民法学、翻译理论与实践、物理电子学与光电子学、物理化学、计算机应用。
