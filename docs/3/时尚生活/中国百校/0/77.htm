华中农业大学

学校概况

1952年秋成立华中农学院。1985年10月改为华中农业大学。（学校最早的历史可追溯到1898年张之洞创办的农业学堂。）是一所由农业部直属的国家重点农业大学。学校坐落在湖北省武汉市洪山区。校舍面积19.35万平方米。

学校现有教师总数718人。其中教授77人，副教授188人，讲师278人。

国家重点学科

作物遗传育种、农业微生物学、果树学

本科专业设置

作物植物遗传育种、植物保护、土壤与植物营养、农业微生物、畜牧动物营养与饲料加工、兽医、农业机械化、食品机械、果树、蔬菜、林学、观赏园艺、经济林、淡水渔业、农牧业经济管理、农业贸易、房地产经济管理、土地规划与利用、食品科学、食品工程、社会学

授予硕士学位的学科、专业
