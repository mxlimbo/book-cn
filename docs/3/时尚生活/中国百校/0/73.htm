西北农业大学

学校概况

1934年创建国立西北农林专科学校。1939年改名为国立西北农学院。1949年易名为西北农学院。1985年10月更名为西北农业大学。是农业部直属国家重点农业大学。校舍建筑面积21万平方米。

学校现有教师790多人，其中教授83人，副教授208人，讲师294人。

国家重点学科

农业经济及管理

本科专业设置
