南开大学

学校概况

1919年10月建校。1937年9月与北京大学、清华大学合组长沙临时大学。1938年4月长沙临时大学迁昆明，改称西南联合大学。1946年返津复校，更名南开大学。

学校现有教师共1686人。其中教授284人，副教授662人，讲师614人，助教126人。
