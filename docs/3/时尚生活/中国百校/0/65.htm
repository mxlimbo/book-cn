湘潭大学

学校概况

湖南省省教育委员会直属的国家重点综合性大学。坐落在湖南省湘潭市西郊。校舍建筑面积23万平方米。

湘潭大学创建于1958年6月。1959年7月停办。1974年3月恢复。

师资力量

学校现有教师总数838人。其中教授79人，副教授266人，讲师333人，教员75人，助教85人。
