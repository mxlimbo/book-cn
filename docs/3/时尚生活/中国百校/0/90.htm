对外经济贸易大学

学校概况

对外经济贸易大学原名北京对外贸易学院，成立于一九五四年，是对外贸易经济合作部所属的一所全国重点大学，也是我国第一所国际经济贸易专业门类比较齐全的多科性大学。

自建校以来，对外经济贸易大学在党的教育方针指引下，为国家培养了数万名本、专科毕业生、研究生、干训生和外国留学生。该校毕业生遍布祖国各地对外经济贸易部门和进出口公司以及驻外商务、贸易、金融机构，他们为发展我国对外经济贸易事业，为社会主义现代化建设发挥着重要作用。

学校校址在北京市朝阳区，占地面积400余亩，校舍面积17万平方米，其中，建筑面积达28000平方米的新教学主楼于一九九二年八月交付使用。

该校于1995年6月通过国家高等教育“211工程”预审。

学校现有国际经济贸易学院、国际工商管理学院、国际交流学院、法学院、继续教育学院和海关管理系、人文科学系、信息经济系，共二十四个专业，并设有研究生部和留学生部，培养国际贸易、国际金融、运输经济、国际经济、国际企业管理、国际经济法、海关管理、会计学、进出口商品公证鉴定、市场营销、行政管理学、经济信息管理和英语（经济贸易）、日语（经济贸易）、法语（经济贸易）、西班牙语（经济贸易）、俄语（经济贸易）、德语（经济贸易）、意大利语（经济贸易）、阿拉伯语（经济贸易）、越南语（经济贸易）、朝鲜语（经济贸易）以及国际贸易、海关管理第二学士学位等各类专门人才。

本科各专业均有权授予学士学位；国际贸易、国际金融、国际经济法、国际企业管理、经济信息管理和英语（经济贸易）、法语（经济贸易）、日语（语言文学）等专业有权授予硕士学位；国际贸易、国际经济法专业还有权授予博士学位。另外还设有会计专科、运输经济专科。函授对外贸易专科、经济英语培训中心、外贸英语培训中心、中国中小企业培训中心、中国-联邦德国语言强化培训中心、中国-意大利语言培训中心、中国-法国高级企业管理干部培训中心、中国-加拿大人才开发项目、在中型外贸企业经理岗位培训斑、对外开放干部培训班、亚太工商管理培训中心、对外汉语培训中心、夜大学等。由于在教学和科研等方面取得突出的成绩，该校的工商管理硕士学位于一九八八年十一月得到了美国纽约州立大学董事会的承认。

为了在深化改革和进一步开放的形势下主动适应我国对外经济贸易事业的迅速发展，加强学校与社会的联系，依靠各方面的力量，加速培养德、智、体全面发展的对外经济贸易人才，我校于一九八九年十一月成立了校董会。国务院副总理李岚清任第一届校董会主席，国家副主席荣毅仁、原德国经济合作部国务秘书冷格耳先生任第一届校董会名誉主席，现任校董会主席是对外贸易经济合作部部长吴仪。对外贸易经济合作部有关部门、中国国际贸易促进委员会、国务院特区办、中国银行、中国人民保险公司、海关总署、国家进出口商品检验局、部属和其它部委所属进出口贸易总公司的有关领导及海内外有关人士任董事。

随着我国对外经济贸易事业和发展，学校规模不断扩大，招生人数逐年增加，目前在校的本、专科、二学位生、研究生及培训人员共三千多人；留学生约三百人；函授生、夜大学生一万多人。

学校师资力量雄厚，现有教授83人、副教授211人，讲师357人，此外，每年还聘请外国专家三十余人。

目前，该校已和美国、加拿大、英国、德国、日本、法国、比利时、荷兰、澳大利亚、新西兰、俄罗斯、南斯拉夫、匈牙利、捷克、波兰以及香港等二十多个国家和地区的四十多所院校和研究机构建立了校际学术交流关系。

该校十分重视科学研究工作，设有国际贸易问题研究所、语言文学研究所、跨国公司研究中心、世界贸易组织研究会、台港澳经济研究中心和韩国经济研究中心等科研机构。学校除承担了国家一些对外经济贸易方面的重点科研项目，公开出版了许多外语及对外经济贸易方面的教材、专著、译著和辞典工具书，还办有在国内外公开发行的《对外经济贸易大学学报》、《国际贸易问题》、《日语学习与研究》，以及《国际贸易译丛》等学术刊物。随着出版任务增多，我校还成立了对外经济贸易大学出版社。

学校图书馆现有藏书46万余册，其中外文书籍16万余册。藏书重点是有关世界经济、国际贸易、经济法、语言等方面的专业书、教学工具书和参考书。

学校的电化教育中心拥有先进的电教设备和卫星转播系统，15个语言实验室。专业广播级的录音室和录像演播室可以录制各种录音、录像节目。视听中心可容纳七百多人，既可放映电影和录像，又能进行同声传译训练。

学校计算中心配有IBM-4331中型双主机和数百台微机，1990年该校计算中心曾用作第十一届亚运会计算中心。

该校还有一个标准的运动场和可供正式比赛使用的游泳池。
