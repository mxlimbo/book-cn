云南大学

学校概况

云南大学是云南省教育委员会直属的全国重点综合性大学。坐落于“春城”昆明市翠湖北路。

气候温和，四季如春。校园内花团锦簇，林木葱郁，环境优美。校舍建筑面积24.15 万平方米。

1922年12月创建私立东陆大学，曾用名为省立东陆大学、省立云南大学、国立云南大学。全国解放后为云南大学。

学校现有教师总数811人。其中教授80人，副教授282人，讲师356人，教员32人，助教61人。
