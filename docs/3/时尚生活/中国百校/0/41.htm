中南工业大学

学校概况

中国有色金属工业总公司直属的中南工业大学，原名中南矿冶学院，是1952年由原武汉大学、中山大学、湖南大学、广西大学、南昌大学和北京工业学院的矿冶系（科）合并而成。1985年7月改为现名。学校坐落在湖南省长沙市。校舍面积40.01万平方米。

学校现有教师1185人。其中教授157人，副教授332人，讲师518人。

国家重点学科

矿产普查与勘探（含应用地球物理）、选矿工程、有色金属冶金

本科专业设置

地质矿产勘查测量工程、热能工程、材料物理、计算机及应用、应用数学、技术经济、复合材料、应用地球物理、选矿工程、工业分析、机械设计与制造、粉末冶金、应用化学、英语、设备工程与管理、勘察工程、有色金属冶金、金属材料与热处理、机械电子工程、冶金物理、化学、应用物理学、制冷与低温技术、工业外贸、采矿工程、化学工程、金属压力加工、工业自动化、思想政治教育、管理工程、国际金融
