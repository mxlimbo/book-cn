兰州大学

学校概况

兰州大学所在地甘肃省省会兰州，位于祖国地理中心，是大西北的交通枢纽和工业重镇，又是古“丝绸之路”和今天“亚欧大陆桥”上一颗璀璨的明珠。兰州又是西北文教科研中心，这里有中科院兰州分院8个研究所，兰州分院和兰州大学的9个国家级对外开放实验室。国家级实验室拥有数量在全国排名第三，是全国科技力量最强的九大省区之一。兰州气候温凉，冬不严寒，夏无酷暑。夏秋之交，瓜果满城，浓郁飘香。

兰州大学校园占地面积近千亩，建筑面积35万平方米。

兰州大学创建于1909年，始为甘肃政法学堂，1949年定名为国立兰州大学。1953年起列为国家教育委员会直属全国重点大学，是我国首批博士、硕士学位授予单位和乎批建立博士后流动站的高校之一。也是国家教委第一批设置理科和文科“国家基础理论研究和教学人才培养基地”的高校之一。1996年5月顺利通过国家“211工程”部门预审。

兰州大学在人才培养上，形成了“以研究生教育为重点，以本科教育为主，以专科教育和成人教育为补充”的办学结构，本着“改造、充实、发展、创新”的思想，对学科、专业进行大幅度改革，形成了以基础学科专业为依托，以重点学科专业为核心，以博士点学科专业为骨干，以边缘、新兴、交叉学科专业为特色，以应用学科专业为主导的新型学科专业体系。特别是物理、化学、数学、生物、地理、历史等6个“理科基础科学研究和数学人才培养基地”及“文科基础学科人才培养和科学研究基地”的建设和发展，使我校学科建设向前迈进了一大步。

兰州大学现设有5个学院、22个系，55个本科专业、18个专科专业、4个直属教学部、建有26个研究所（室 、中心）、84个实验室；设有出版社、计算中心、分析测试中心和电化教育点实验室。有博士学位授予权的专业14个，有硕士学位授予权的专业46个。首批建立物理学（理论物理、原子核物理、凝聚态物理）、化学（有机化学、分析化学）和生物学（细胞、植物和生态学）、博士后科研流动站。有8个学科、专业接受国内外访问学者。现在校各类学生10000余人，其中研究生850人。

兰州大学历来重视师资队伍建设，师资力量雄厚，阵容整齐。学校现有教师、科研及其它专业技术人员2000多人，有教授、副教授（含具有相应职称的其他毓人员）近700名。其中中科院院士5人，博士导师78人，硕士导师500人，他们中许多人是我国著名专家、学者，是学校教学、科研和管理的中坚力量。

兰州大学本科学制四年，实行学分制，修满必修学分和总学分的学生可提前毕业，品学兼优的学生推荐免试攻读硕士学位。学校在教学中强调对学生的素质培养，注重拓宽学生的知识面，大力加强实践性教学环节，培养学生的动手能力和工作能力；重视因材施教，对学习特别优秀的学生，制定了优秀生选拔条例和重点培养措施，对优秀生设有各类奖学金。兰大在长期的办学实践中，不断探索、创新，形成了“勤奋、求实、进取”的优良学风。毕业生素以基础扎实、知识面宽、勤奋实干而著称，深受社会各行各业的广泛欢迎和赞誉。80多年来，学校为国家培养了五万余名各类专业人才，有许多成绩卓著，成为各个领域的知名专家和学者，或政府部门、军队和企事业单位的领导和中坚。

兰州大学既是教育中心又是科研中心。在科研方面，重视基础研究，发展具有多种特色的应用研究。1980年又红又专来，先后获得国家、部位和省级科技成果奖励330项。编著出版各种专著、教材、译著500多部，每年发表学术论文近1000篇，据《SCI》和《ISR》统计公布，近两年兰州大学发表的论文数量已跃居全国高校前4-6名。学校主编、出版发行的学术刊物有《兰州大学学报》。（自然科学版、社会科学版、交叉科学版三种）、《西北史地》、《敦煌学辑刊》、《科学·经济·社会》、《西北人口》、《秘书之友》等。

兰州大学广泛开展国际学术交流活动，已与美国、日本、英国、法国、加拿大、德国、澳大利亚、波兰、捷克、斯洛伐克、匈牙利等国家和香港等地区的30多所高校和科研机构有学术协作与交流关系。几年来，中青年教师进修和攻读学位的360余名，先后有1200多人次外籍专家、学者来校讲学和进行学术交流。
