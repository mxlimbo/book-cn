四川联合大学

学校概况

四川联合大学是1993年11月，由四川大学和成都科技大学两所委属重点高校合并创立的。四川大学的前身是1704年建立的锦江书院和1874年建立的尊经书院，1896年建立的四川中西学堂。1902年奉清廷圣旨在以上三校的基础上建立四川省城高等学堂。1931年11月，定名为国立四川大学。全国解放后，国立四川大学改称为四川大学。成都科技大学原名成都工学院。1978年10月，曾更名为成都科技大学。

四川联合大学是国家教委直属的全国重点综合性大学。校舍建筑面积78.65万平方米。

学校现有教师总数3517人。其中教授329人，副教授1619人，讲师1275人，教员20人，助教274人。
