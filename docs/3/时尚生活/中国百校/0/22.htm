中国纺织大学

学校概况

　　中国纺织大学位于上海市延安西路1882号，是全国重点大学，创建于1951年。是首批具有博士、硕士、学士三级学位授予权的单位。学校设有校办总厂，并在江苏省设立无锡校区。中国纺织大学已建设成为一所以工为主，兼有理科文科的多科性大学。中国纺织大学于一九九五年九月顺得通过了“211工程”部门预审。进入了新的历史发展时期，决心用十五年左右的时间把学校建设成为以工科、管理学科、应用艺术类学科为主，兼有理科和人文社会学科的，在教育质量、科学研究和管理等方面处于国内先进水平。

　　全校设有服装学院、艺术设计学院、旭日工商管理学院、材料科学与工程学院、纺织学院。机械工程系、纺织化学系、自动化及电子信息工程系、计算机科学与工程系、环境工程系、社会科学系、外语系、基础部、体育部等五院十九系、二部、五个研究所。设置二十九个本科专业七个专科专业。

　　学校拥有大规模的图书馆、设备先进的实验室和纤维科学技术测试中心、计算机中心、电化教育中心、图像处理中心。其中纺织工程，化学纤维二个学科（专业）已被列入全国重点学科（专业）点，并建有一个化学纤维改性国家重点实验室。

　　中国纺织大学是首批具有博士、硕士学士三级学位授予权的单位。目前，学校有博士点、硕士点共三十个，博士后流动站一个。

　　在现有近1100名教师中有正副教授近500名；在校种类学生8000余名。建校四十多年来，已培养多种层次毕业生四万余名，为四十八个国家培养了450余名留学生。学校设有成人教育学院，夜大学和函授学生近2000名。

　　中国纺织大学重视培养学生德智体全面发展，加强基础教学，拓宽知识面，注重能力培养，增强学生适应性和竞争力。

　　学校对学生实行按学科大类进行培养，基础课程和技术基础课相同或相近的专业组成同一大类，在前两年至两年半内按学科大类统一进行基础课和技术基础课的学习，后一年半至两年，按专业进行分流，允许学业优良的学生在学科大类内自主选择专业。实行学分制，贯彻因材施教，进行分层次教学。

　　学校开设了经贸管理类、外语类、计算机应用类、服装类、人文社科类等20个跨专业选修课组。工业外贸、管理工程、会计学。计算机应用、涉外秘书五个辅修专业，计算机应用、自动化、管理工程、纺织材料检验与贸易、英语五个第二专业，供不同层次的学生选读。学业成绩合格，颁发相应证书。

　　学校还建立了选拔培养优秀学生制度，实行人民贷学金与人民奖学金并存的办法，并设有由著名人士、教授、专家赠款设立的“五爱奖学金”、“钱之光奖学金”、“陈维稷奖学金”、“梁溪唐氏奖学金”、“周承佑奖学金”和由国内外著名企业、社会团体设立的“纺织工程学会奖学金”、“宝钢奖学金”、“美国杜邦公司奖学金”、“仪征奖学金”、“欧德力奖学金”、“桑麻奖学金”、“东阳·蜻蜒奖学金”等，奖励优秀学生，攀登科学高峰。

　　中国纺织大学是磁头德、智、体全面发展的工、管、经、文、法、艺术等学科高层次人才的摇篮，并负担着发展科学技术尤其是纺织科学技术的重任。我校纺织工程、纺织材料、针织工程、纺织品设计、服装、染整工程、纺织机械7个专业具有学士学位的毕业生，均可获得国际性的英国纺织学会认可，享有在英联邦国家免试开业资格（LTI），前5个专业可获得该学会会员资格（ATI）。由此可见，我校有些专业教学和学术水平已达到同类专业国际先进水平。
