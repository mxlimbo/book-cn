武汉大学

学校概况

  武汉大学坐落在湖北省武汉市风景秀丽的武昌东湖之滨、珞珈山麓。校园依山傍水，层林翠染，湖光潋滟，山影葱茏，景色迷人。

  武汉大学拥有校园面积3000多亩，建筑面积54万平方米。武汉大学是国家教委直属的重点综合大学。

  武汉大学的历史溯源于1893年湖广总督张之洞创办的自强学堂，后数易校名，1928年定名为“国立武汉大学”。

  武汉大学具有光荣的革命传统。解放前，周恩来，董必武，陈谭秋，邓颖超，罗荣桓，郭沫基，李达，李汉俊，恽代英，伍修权，郭述申等 都曾在这里从事过革命活动。解放后，武汉大学更受到党和国家的翔与重视。毛泽东，董必武、李先念，叶剑英，彭德怀，陈毅，张闻天，陆定一，乔石，李岚清，李铁映等曾光临学校视察。在1993年建校一百周年庆典之际，中共中央总书记、国家主席江泽民为学校题词：“发扬优良传统，培养优秀人才”。

  武汉大学具有优良的学风和较高的学术地位。辜鸿铭、华衡芳，许德衍。李四光，孙大雨，黄侃，朱光潜，竺可桢，邬保良、查谦，桂质廷，叶雅各，闻一多，叶圣陶，郁达夫，沈从文，刘颐，刘永济，杨端六，李剑农、邓初民、曾昭抡等众多蜚声中外的著名专家学者曾在这里任教或工作，王世杰、周鲠主、王星拱、李达等先后担任过校长。1948年英国牛津大学致函中国国民政府教育部，确认武汉大学的文、理学士毕业生成绩平均在80分以上者享有“牛津之高级生地位”，1995年学校顺利通过国家教委“211工程”部门预审，确认我校为“居于全国同类高校前列，并有一定国际影响的社会主义综合性大学”。

  建校以来，武汉大学为国家培养了6万多名高级专门人才，他们大部已经成为各条战线上的骨干力量，先后有33人被评选为中国科学院院士或工程院院士。

  学校现设12个学院，8个校直属系，64个本科专业。各类在校学生15000余人。其中本专科生8700余人，博士、硕士研究生2000余人，成人教育学生5000余人，外国留学生100余人。教职工4800余人，其中教师2100余人，教师中有教授近400人，副教授近800人。教授中有138位博士生导师。有36个学科专业享有硕士学位授予权，9个学科专业被定为国家级重点学科，数学、物理、化学、生物学、法学5个一级学科设立博士后科研流动站。

  武汉大学有12座大型教学楼，28个研究所，12个实验研究中心，31个独立研究室，122个大型科研实验室（其中国家级重点实验室4个），19000多台实验设备，20余万件动植物标本、模型以及一座综合性标本楼。学校图书馆藏书280余万册，订有国内外期刊杂志5000余种。

  自70年代末以来，武汉大学承担国家及省市重点科研项目2920余项，其中近140项成果达到国际水平，468项自然科学研究成果和700项社会科学研究成果获国家或省部级奖。1992年学校获准社科青年基金项目、国家中华基金项目、国家社科规划项目数分别居全国高校的第一、二、三位。1995年国际科学界权威的期刊之一美国《科学》周刊根据在国际科学期刊中引用中国科学家的论文成果次数，评选武汉大学为中国十三所最杰出的大学之一。学校出版发行《武汉大学学报》等10余种学术刊物。

  近十年来，学校派出各类出国人员1800余名，聘请70余名外国学者或世界名人为学校名誉教授、兼职教授或客座教授，共邀请1200余名外藉专家和教授来校任教或讲学。学校与26个国家和地区的120多所大学和科研机构建立了合作交流关系。

  自1978年以来，学校不断深化教育教学改革，率先实行学分制、主辅修制、插班生制、奖学金贷学金制等制度，率先创办中法数学试验班和人文科学试验班，举行一年一度的大学生学习竞赛等活动。1994年，经国家教委批准，学校首批进行招生，收费制度改革试点，采取了一系列配套措施，以招生收费制度改革为契机，以教学改革为核心，面向21世纪深化课程体系、教学内容改革，加强文化素质教育，全面提高学生质量，允许学生在学好本专业课程的基础上，根据学科发展需要和自身志趣，选修其它专业的有关课程，以加厚基础，拓宽知识面、优化知识结构、增强主动适应社会需要的能力和自身发展的后劲，并允许学生提前毕业或根据需要推迟毕业，最长可延长到七年，为缅怀先辈、激励后学，学校设立了多种奖学金以及大学生科研基金等。1994年以来，为激励学生努力学习，冒尖成才，又增设了新生奖学金、基础学科奖学金等10余项专项奖学金，广泛开展勤工助学活动，并开展多种形式的学习竞赛，学术交流和社会实践活动，为学生积极主动地学习创造了良好的条件，培养出了一大批政治思想品德好、基础理论扎实，实践动手能力强的优秀学生，得到了社会的普遍好评。
