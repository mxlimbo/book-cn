南京航空航天大学

学校概况

　　南京航空航天大学是中国航空工业总公司直属的国家重点大学。校舍建筑面积29.71万平方米。

　　1952年10月建立南京航空学院。1993年3月改为南京航空航天大学。

　　学校现有教师总数1081人。其中教授151人，副教授348人，讲师397人。

国家重点学科

　　飞机设计、机械制造、空气动力学
