武汉工业大学

学校概况

武汉工业大学的前身是1958年8月成立的北京建筑工业学院。1971年10月适至武汉市，与武汉建筑工程学校合并改称湖北建筑工业学院。1978年5月更名为武汉建筑材料工业学院。1985年8月更名为武汉工业大学。是一所隶属于国家建筑材料工业局的国家重点大学。学校坐落在湖北省武汉市洪山区。校舍建筑面积26.62万平方米。

国家重点学科

无机非金属材料
