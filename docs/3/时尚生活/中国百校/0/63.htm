山东大学

学校概况

山东大学是国家教委直属的国家重点综合性大学。1901年创建山东大学堂，历经省立山东大学、国立青岛大学、国立山东大学、山东大学。是我国历史最悠久的著名大学之一。

全校共有58个本科专业、59个硕士点、17个博士点、3个博士后流动站、3个国家基础科学研究和教学人才培养基地，2个国家重点学科，2个国家重点实验室，1个国家教委部门开放实验室，1个国家科委重点技术研究推广中心，6个山东省重点实验室，17个山东省重点学科。学校现有教师总数1905人。其中教授131人，副教授454人，讲师974人，助教346的。另外有中科院院士3人，博士生导师86人，教授、副教授近千人。学校聘请了200位国内外著名专家、学者为名誉教授、客座教授或兼职教授。

国家重点学科

固体物理微生物学
