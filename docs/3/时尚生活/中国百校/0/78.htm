华南农业大学

学校概况

它位于广州市天河区五山街，交通方便，校园面积420多公顷，属亚热带兼有热带特点的气候，校园风光秀丽，景色迷人，连续六年被评为广州市花园式单位，是读书求学的好地方。华南农业大学是农业部属高等学校，是一所多科性、综合性的全国重点大学。

目前，该校已通过了由电力部组织的“211工程”部门预审，成为国家“面向21世纪重点建设的100所左右高等学校”之一。

华南农业大学始创于1910年，迄今已有86年的办学历史。现在已形成了包括农学、林学、生物技术、工程技术、畜牧兽医学、理学、资源环境科学、食品科学、经济学、管理科学和人文社会科学等以农为主的综合学科体系。

该校现有博士学位授权专业点10个、硕士学位授权专业点38个、学士学位授权专业点32个，建有博士后科研流动站，现有在站人员14人。该校博士、硕士学位授权专业点学科函盖了农学一级学科中87%的二级学科。1993年国务院学位委员会委托农业部对全国农业系统的博士点进行评估，该校有6个博士点综合实力列全国前三名，兽医药理学和禽病学科两博士点是全国同类学科中唯一的博士点。现有国家级重点学科1个，农业部和广东省重点学科1个，农业部重点开放实验室2个。

现设有资源环境学院、林学院、经济贸易学院、工程技术学院、生物技术学院、人文科学学院、理学院、成人教育学院和农学系、园艺系、动物科学系、动物医学系、蚕桑系、食品科学系及体育部共15个院系部。经过几十年尤其是改革开放以来的建设和发展，该校现有教职工1881人，教师中正高级职称81人，副高级职称270人。有中科院院士2人、博士生导师28人、硕士生导师193人。中乘法院院士人数在全国高校中排名第九，有广东省高校唯一的现任国务院学位委员会委员1名，还有两名中国工程院院士受聘为该校兼职教授，有国家级有突出贡献的中青年专家5人、省部级5人。

学校有开展教学和科研的良好条件，有110个常规实验室和较好的体育教学场地与设施，5个实验工厂及各种动物养殖场，有独立的电化教学体系和正在发展的多媒体课室，图书馆占地面积1万平方米，藏书67万册。实验中心建筑面积1000平方米，拥有电子显微镜等5个系统的先进仪器设备，教学信息计算机网络中心已连通全国教育与科研网、Internet网。教学管理实行学分制、主辅修专业制。学校有标准的足球场、篮球场、网球场、排球场、羽毛球场、游泳池等运动场所，校舍总面积28万平方米。
