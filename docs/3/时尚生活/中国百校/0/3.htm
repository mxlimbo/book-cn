北京航空航天大学

学校概况

1952年10月成立北京航空学院。1988年4月改为北京航空航天大学。

学校现有教师总数900人。其中教授133人，副教授351人，讲师288人，教员51人，助教77人。

国家重点专业

航空发动机、航空陀螺及惯性导航、空气动力学、机械学、飞行器导航控制系统
