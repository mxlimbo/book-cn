青岛海洋大学

学校概况

青岛海洋大学前身是1924年10月成立的私立青岛大学。1930年5月与省立山东大学合并创办了国立青岛大学。1932年国立青岛大学改为国立山东大学，直到1951年3月国立山东大学与华东大学合校成立山东大学。1959年成立山东海洋学院。1988年1月改名青岛海洋大学。是国家教育委员会直属的全国重点大学。

青岛海洋大学坐落在山东青岛市市南区。建筑面积20万平方米。

现有专职教师620人。其中教授101人，副教授217人，讲师230人。

国家重点专业

本校有两个国家重点学科：物理海洋学和水产养殖。

本科专业设置

海洋学、物理学、海洋物理学、应用化学、海洋化学、应用数学、计算机及应用、国际经济法、气象学、机械设计与制造、化学、海洋生物、生态学与环境生物、海洋地质勘查、地球物理、国民经济管理、电子学与信息系统、工业自动化、水文地质与工程地质、海水养殖、淡水养殖、食品工程、海洋渔业、渔业资源

授予硕士学位的学科、专业

应用数学、物理海洋学、环境海洋学、海洋气象学、海洋物理学、海洋化学、海洋生物学、海洋地质学、水产养殖捕捞学、水产品贮藏与加工、企业管理、光学仪器、信号与信息处理、环境地质、渔业资源
