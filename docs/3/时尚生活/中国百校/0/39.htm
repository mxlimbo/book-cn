武汉测绘科技大学

学校概况

武汉测绘科技大学创办于1956年，是一所以测绘学科为主干，理、工、文、经各学科相互依托，协同发展的全国重点大学；是国务院批准首批可以授予博士、硕士学位的单位之一，也是国家教委批准可接受国内外访问学者的单位之一。学校现有地学测量工程学院、城市建设学院、信息工程学院、土地科学学院、光电工程学院、印刷工程学院、人文管理学院、成人教育学院八个学院；拥有工程测量、大地测量、航测与遥感、光学仪器、电子工程、建筑工程、国土信息与地图科学、印刷工程、计算机科学与工程、外语、十个系。

本科专业

测量工程、摄影测量与遥感、信息工程、计算机科学与技术、地理信息系统与地图学、土地管理、资源环境规划与管理、测控技术、包装工程、城市规划、城镇建设、应用数学、市场营销、英语

授予硕士学位的专业

光学工程、测试计量技术及仪器、物理电子学、信号与信息处理、计算机应用技术、城市规划与设计、大地测量学与测量工程、摄影测量与遥感、地图学与地理信息系统、地图制图学与地理信息工程、印刷工程、固体地球物理学、马克思主义理论与思想政治教育
