华中理工大学

学校概况

美丽的校园位于武汉市风景秀丽的东湖之滨，校园面积270公顷，是一座读书治学者向往的大学城。

华中理工大学是国家教育委员会直属的重点大学，是全国规模最大的学府之一。

70年代中期，由单一工科院校转变为综合性大学，并既是教学，又是科研中心。近几年，因为学校的教学质量与学术水平明显提高，综合实力上了一个大台阶。1995年，学校顺利通过了“211”工程部门预审。1996年12月，国家教委组织对华中理工大学本科教学工作进行了考察评价，评价结果为优秀。

学校内设有研究生院和成人教育学院，并组建了文、理、生命科学与工程、经济、工商管理、建筑、机械科学与工程、信息科学与工程、能源科学与工程、体育运动技术、交通科学与工程等11个学院以及直属学校领导的力学、外语、材料科学与工程、土木工程等4个系。学校现有57个本科专业，82个硕士点，31个博士点，7个博士后流动站，5个国家重点实验室和专业实验室。

现有教师2627人。其中教授290人，副教授964人，讲师947人。

机械制造、压力加工、电厂热能动力及其自动化、电机
