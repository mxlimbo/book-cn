武汉水利电力大学（宜昌）

学校概况

武汉水利电力大学原名葛洲坝水电工程学院，创办于1978年，直属电力工业部领导，是一所以水电类工科专业为主体，管、文、经等方面专业兼有的本科高等院校；1996年5月经国家教委批准，原武汉水利电力大学与葛洲坝水电工程学院合并组建成新的武汉水利电力大学，原武汉水利电力大学为武汉部分，葛洲坝水电工程学院为宜昌校区，校部设在武汉。学校是电力工业部主管的多学科全国重点大学，已在1996年11月顺利通过“211工程”部门预审。

宜昌校区坐落在举世闻名的风景胜地长江三峡的西陵峡峡口、中国电都----湖北省宜昌市内，宏伟壮丽的葛洲坝水利枢纽工程就在学校近旁，举世瞩目的长江三峡水利枢纽工程距学校仅28.5公里，湖北省最大的清江隔河岩水电站和水布垭、高坝洲、西北口等一系列大中型水电工程分布在学校周围约50公里的范围内。这些工程为师生的教学实践和科研工作提供了得天独厚的有利条件。校区占地25.3公顷，建筑面积15.5万平方米。校园依山傍水，层林叠翠，湖光山影，景色宜人，是一处良好的学习和研究园地。

宜昌校区遵循“立足水电、服务电力、面向社会”的办学宗旨，开办有20个本、专科专业或专业方向，均为电力建设和社会经济建设发展急需的专业，有着良好的择业前景。本科专业均可授予学士学位。校区设有建筑工程系、电气工程系、机械工程系、管理工程系、社会科学部、基础课部、成人教育中心、军训及优育课部等8个教学单位。办有国内外公开出版发行的《武汉水利电力大学（宜昌）学报》（自然科学版），还出版有《水电高教研究》、《水电科技情报》、《水电大学（宜昌）报》等刊物。

宜昌校区现有教职工915人，其中专职教师358人；教师中正、副教授和高级工程师136人，讲师和工程师158人，常年聘有外籍专家参与教学；校区已与加拿大、法国、俄罗斯、乌克兰、越南等国的多所院校建立了校际联系和合作关系，经常有国内外专家、学者来校访问或讲学，1996年1月湖北省教委批准校区具有接收外国留学生资格，并已经接收培养法国等国派来的留学生。学校同国内外高校间的交流与合作将随着三峡工程的兴建日益广泛。

宜昌校区教学、科研设施完善。设有33个学科教研室，29个教学研究实验室，9个科研所和建筑设计院（国家建筑乙级）“水工结构工程”为省重点学科，“水电站仿真实验室”为部级重点实验室。有设备齐全先进的计算机系统、电化教学系统和外语语音训练系统、有宽敞明亮的教学大楼、实验大楼、学生公寓，有标准运动场和游泳馆，有藏书40余万册和计算机检索系统的图书馆。建筑面积为5000平方米的体育馆正在兴建之中。

学术成果

1991年以来，学校获准国家自然科学研究基金和部研究基金15项，横向科研课题54项，其中三峡工程科研项目22项，共获得纵横向科研成果170余项，其中国际会议交流50余项，获得国家专利10余项。校区正在向着既是教学中心又是科研中心迅速发展。
