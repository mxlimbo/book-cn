石油大学

学校概况

石油大学的前身是北京石油学院，创建于1953年10月。1969年迁校山东省东营市，改名华东石油学院；1988年更名为石油大学。磁针本部设在北京，由石油大学（北京）、石油大学（华东）、石油大学（广州）和石油大学北京石油管理干部学校组成。

现有教师总数1480人。其中教授192人，副教授369人。

本科专业设置

石油地质勘查、地球物理、矿场地球物理、钻井工程、采油工程、油藏工程、应用电子技术、有机化工、应用化学、矿业机械、化工设备与机械、石油储运、热级工程焊接工艺及设备、生产过程自动化、石油加工、应用物理、工业管理工程、计算机及应用、会计学、思想政治教育、机械制造工艺与设备、工业与民用建筑工程、化学工程、机械电子工程、工业外贸、市场营销、涉外企业财务管理
