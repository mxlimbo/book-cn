复旦大学

学校概况

复旦大学是国家教委直属全国重点综合类大学。

1905年5月创建，原名复旦公学。1917年改为复旦大学。是我国近代创建历史最长的大学之一。

学校设在上海市。建筑面积58.7万平方米。

学校现有教师总数2177人。其中教授303人，副教授644人，讲师863人，助教367人。
