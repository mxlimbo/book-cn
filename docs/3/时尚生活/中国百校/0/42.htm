华南理工大学

学校概况

1952年11月成立华南工学院。1958年分为华南工学院和华南化工学院。1962年合并为华南工学院。1970年再分为上述两学院。1978年再合并为华南工学院。1988年改名为华南理工大学。学校坐落在广州市天河区。建筑面积 62.45万平方米。

学校现有教师1601人。其中教授138人，副教授607人，讲师655人。

国家重点学科

通信与电子系统、化学工程、制糖工程、制浆造纸工程
