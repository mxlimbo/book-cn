大连海事大学

学校概况

　　大连海事大学是交通部直属重点院校。坐落在辽东半岛的南端，美丽的海滨城市大连市。校舍建筑面积25.5平方米。

　　该校于1953年由上海航务学院、东北航海学院、福建航海专科学校合并而成，定名为大连海运学院。其前身可追溯到1909年晚清邮传部上海高等实业学堂船政科。1994年2月更名为大连海事大学。

　　目前在校教师总数为684人。其中教授62人，副教授210人，讲师291人。

本科专业设置

　　海洋船舶驾驶、轮机管理、电气技术、船舶电气管理、电子工程、船舶通信导航、通信工程、计算机通信、自动控制计算机、应用管理信息系统、交通运输、水运管理、外贸运输、综合运输航政管理、航标管理水上交通安全、管理港航行政管理、国际经济法（国际海事国际商法国际金融与法律）、国际贸易（国际商务）机械制造工艺与设备、（船机制造与修理）

授予硕士学位的专业

　　航海技术、轮机工程通信与电子系统、船机修造工程、电力传动及其自动化运输管理工程、专门用途外语、国际经济法（海商法）管理信息系统
