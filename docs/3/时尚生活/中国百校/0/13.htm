辽宁工程技术大学

学校概况

　　辽宁工程技术大学是煤炭工业部直属国家重点院校。1958年建立阜新煤矿学院。1978年改为阜新矿业学院。学校坐落在辽宁省阜新市。校舍建筑面积16.35万平方米。

　　当前在校教师总数448人。其中教授30人，副教授121人，讲师204人。

本科专业设置

　　采矿工程、矿山通风与安全、建筑工程、矿井建设、给水排水工程、管理信息系统、建筑学、地质矿产勘查、水文地质与工程地质测量工程、工业自动化、电气技术计算机及应用机械制造工艺与设备、热加工工艺及设备、金属材料与热处理、机械设计与制造、机械电子工程设备、工程与管理工程、投资经济、国际贸易

授予硕士学位的专业

　　采矿工程、矿山工程力学、工程检测、矿山建设工程、矿山机械工程、机械制造、电力传动及自动化

授予博士学位的学科及导师

　　采矿工程 孙宝铮（教授）
