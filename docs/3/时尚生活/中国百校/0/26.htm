中国矿业大学

学校概况

　　中国矿业大学隶属于煤炭工业部。其前身是创办于1909年3月的焦作路矿学堂。1931年改名为焦作工学院。1950年9月以焦作工学院为基础在天津建立了中国矿业学院。1952年并入清华大学、天津大学、唐山铁道学院的采矿系。1953年9月，学校迁至北京改名为北京矿业学院。1970年学校迁至四川省合川县，更名为四川矿业学院。1978年6月在江苏省徐州市重新建校，并恢复中国矿业学院校名。1988年4月学校定名为中国矿业大学。建筑面积39.4万平方米。

　　学校现有教师1450人。其中教授173人，副教授331人，讲师528人。

国家重点学科

　　采矿工程、煤田地质与勘探、矿山建设工程
