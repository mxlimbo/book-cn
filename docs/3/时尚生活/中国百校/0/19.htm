同济大学

学校概况

　　同济大学地处上海市东北隅，分三个校区，校园占地113公顷，建筑面积80万平方米。

　　同济大学是国家教委直属全国重点大学，其前身是德国人创办的“同济德文医学堂”，1927年成为国立同济大学，是我国早期建立的著名高等不府之一，拥有医、工、理、文、法五个学院。1952年，经历了院系调整，又通过四十多年的建设，同济大学得到了全面的迅速发展和壮大，现已发展成为一所以工为主，理工结合，兼有经济、管理、文、法的多科性大学，并在建筑和土木学科等领域形成了明显优势，在机电、应用理科等领域获得了突破性的进展。70年代末开始，同济大学恢复对德国的联系，已发展成为我国对德科技文化交流的中心。

　　同济大学目前设有经济管理学院、建筑与城市规划学院、土木工程学院、机械工程学院、环境工程学院、计算机学院、材料科学与工程学院、商学院、文法学院等9个学院以及研究生院、函授与继续教育学院、高等技术学院和由德国政府与德国高样支持办学的中德学院、职业教育学院。全校设有34个系、42个本科专业、68个硕士点、22个博士点、5个博士后科研流动站（含14个学科专业）。近年来，同济大学积极进行专业结构调整和建设，发展新兴学科，使学校种类专业教育与国际同类高层次专业教育接轨，使学校的毕业生成为高层次外向型科技人才。

　　同济大学目前拥有38个研究所，17个独立研究室，4个国家重点实验室和国家专业实验室，3个国家教委开放实验室，拥有城市污染控制国家级工程研究中心，上海市建设机器人工程技术研究中心，还有汽车设计与技术工程研究中心等9个跨学科综合研究中心，以及与建设部合办的3个高级行政、技术干部培训中心、与法国合办的国际燃气培训中心、与德国合办的CIMS培训中心、留德预备部、科技德语中心和为教学科研服务的现代教育管理信息中心、测试中心、CAD软件开发中心、电教中心、出版社、图书馆等。

　　同济大学师资力量雄厚，拥有教师3300多名。若中国科学院院士4人，中国工程院院士2人，博士生导师135人，教授等正高级职称407人；副教授等副高级职称926人。他们中有许多是蜚声中外的学者专家，如曾获德国大十字勋章、国际结构工程功绩奖的李国豪院士、曾获“陈嘉庚地球科学奖”的马在田院士、曾获“蒋氏科技成就奖”的张曙教授。

　　同济大学科技成果丰硕，自1985年以来，获国家级和省部级成果奖430多项，完成国家攻关项目、国家自然科学基金会和部委有关项目2000多项，承担科技咨询项目20000多项，每年发表学术论文逾千，出版专著近百，申请国家专利110余项。同济大学还积极参与上海和华东地区及北京、广东等地的重大工程项目的建设，学校不断开拓新的研究方向，以占领高新技术领域为目标，使科技向高水平、出大成果、发挥综合优势的方向前进。

　　同济大学在国际上有着广泛的交往，现已和德国、美国、法国、英国、日本、澳大利亚等国60多所大学及工业企业界建立了密切的合作关系。1988年以来，学校主持召开了40多次国际学术会议，订立了50多项国际科技合作项目，派往国外留学、访问和参加国际学术会议教师1000多人次，来校长、短期讲学的专家2000多人次，其中包括贝聿铭、林同炎、哈斯、派尔等多名国际上著名的学者。

　　同济大学在过去漫长的历程中，为国家培养了10万多名高级科学技术人才，贝时樟、李国豪、许厚泽、夏坚白、童第周、王之卓、王守武、唐有祺、吴孟超、裘法祖等70多位中科院院士和工程院院士都曾任教或就读该校。
