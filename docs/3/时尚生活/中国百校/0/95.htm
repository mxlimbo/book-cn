上海外国语大学

学校概况

上海外国语大学的前身创建于1949年12月，原名上海俄文学校。1950年11月改名为上海外文专修学校。1952年易名为上海俄文专科学校。1956年6月改名为上海外国语学院。1994年2月改为现名。是国家教育委员会直属国家重点院校。

校舍建筑面积13.7万平方米。

学校现有教师580人。其中教授48人，副教授129人，讲师263人。

国家重点学科

俄语语言文学

专科专业设置

外事秘书、对外汉语、新闻学、外事管理、教育传播技术、英语、俄语、日语、法语、德语、西班牙语、意大利语、葡萄牙语、阿拉伯语、国际贸易、国际会计

有权授予硕士学位的专业

英语、俄语、德语、日语、法语、西班牙语、阿拉伯语语言文学、语言学与应用语言学、比较文学
