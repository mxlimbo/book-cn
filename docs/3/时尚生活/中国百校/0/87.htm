上海医科大学

学校概况

1927年创建第四中山大学医学院。1932年改名为国立上海医学院。1952年更名为上海第一医学院。1985年改名为上海医科大学。建筑面积24.23万平方米。

现有教师791人。其中教授174人，副教授163人，讲师207人。

国家重点专业

人体解剖学、病理解剖学、生理学、中西医结合基础、流行病学、劳动卫生与职业病学、儿科学、传染病学、耳鼻咽喉科学、肿瘤学、内科学（心血管病）、外科学（神外）

本科专业设置

临床医学、基础医学、医学检验、法医学、生物医学、工程预防医学、卫生事业管理学、妇幼卫生学、药学、药物化学、药理学、护理学
