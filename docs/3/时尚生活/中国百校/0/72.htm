山西农业大学

学校概况

1950年10月成立山西农学院。1979年11月山西农学院、大寨农学院、运城农学院、雁北农学院四校合一，更名为山西农业大学。是山西省教育委员会直属全国重点农业大学。

学校设在山西省太古县。校舍建筑面积16.24万平方米。

学校现有教师565人。其中教授47人，副教授147人，讲师260多人。

本科专业设置

作物、生物、果树、蔬菜、植物保护、土壤和农业化学、土地规划与管理、药用植物、农业机械化、畜牧兽医、公共卫生、农畜产品加工、农业经济管理、林学、森林保护、水土保持

授予硕士学位的专业、学科

植物学、植物生理学、昆虫学、生物化学、作物栽培与耕作学、作物遗传育种、果树学、蔬菜学、土壤学、兽医内科学、作物营养与施肥、动物遗传育种、动物生产学、传染病学与预防、兽医学、农业机械化
