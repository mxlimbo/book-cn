湖南大学

学校概况

1903年为湖南高等学堂。1926年改名为湖南大学。1953年院系调整改为中南土木建筑学院。1958年改为湖南工业学院。1959年定名为湖南大学。是机械工业部直属的全国重点大学之一。

湖南大学坐落于长沙市岳麓山。校舍散布在绿树浓荫之中。建筑面积67万平方米。

学校现有教师1241人。其中教授133人，副教授401人，讲师514人。
