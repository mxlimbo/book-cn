中央民族大学

学校概况

中央民族大学是国家民族事务委员会主管的全国唯一的重点民族大学。

1951年6月建中央民族学院。1993年11月改为中央民族大学。校舍面积18.22万平方米。

学校现有教师1531人。其中教授62人，副教授277人，讲师596人。

国家重点学科

民族学
