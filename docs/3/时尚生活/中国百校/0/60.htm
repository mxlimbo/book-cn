吉林大学

学校概况

吉林大学是国家教委直属的全国综合性重点大学。其前身是东北行政学院和哈尔滨大学，创建于1946年。1950年更名为东北人民大学。1952年成为东北地区第一所文理兼备的综合性大学。1958年改名为吉林大学。1960年被批准为全国重点综合性大学。1984年首批建立研究生院。1985年首批设立博士后科研流动站。

吉林大学坐落在吉林省长春市。占地面积2175亩，各类房舍建筑面积44.8万平方米。

全校拥有科研仪器设备2万余件，装备着101个实验室。其中3有4个国家重点实验室，4个是国家教委开放实验室。

学校的现代化图书馆藏书251万册，其中整套《亚细亚文库》；古籍收藏珍本近4000种；中外期刊3400多种。该图书馆被确定为联合国教科文组织、联合国工业发展组织和世界银行教材中心图书室和国家教委文科文献信息中心设于该馆。

学校现有教师总数1698人。其中教授255人，副教授553人，讲师573人，教员65人，助教252人。全校有中国科学院院士6人，国务院学位委员会学科评议组成员8人，博士生导师107人，硕士生导师800人。担任省、市和全国各种学会、研究会常务理事、秘书长、副会长、会长等职务的有200余人。此外，各类专业技术高级职称职务人员175人。
