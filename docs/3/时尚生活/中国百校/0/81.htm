北京师范大学

学校概况

北京师范大学是我国最早建立的高等学府之一，它的前身是创办于1902年的京师大学堂师范馆。1931年与北平大学女子师范学院合并，称国立北平师范大学。

1937年曾西迁陕甘九年，为祖国西北地区高等教育的发展做出了开创性的贡献。1952年原辅仁大学与北京师范大学合并，同时并入的还有中国人民大学和原燕京大学的教育系科。目前，北京师范大学已发展成为一所教育、文、理、管理等兼容、教育质量和科研水平高、综合实力居国内高校前列、并有一定国际影响的多科性师范大学，是国家重点投资建设的大学之一，1996年1月，被国家列入“211工程”建设序列。

北京师范大学具有近一个世纪的光荣历史和求实、精勤、奋进、刻苦的优良传统，北京师范大学的校训是“学为人师、行为世范”，激励着师生不断地开拓进取和完善自己。在“教育要面向现代化、面向世界、面向未来”的方针指导下，学校把建成“以培养高水平师资和各类教育人才为主，整体教学质量和学术水平居国内一流大学的前列，在国际上有重要影响的社会主义师范大学”作为奋斗目标，正在为振兴祖国的教育事业，提高中华民族的科学文化水平而努力奋斗。

学校现设有18个系，38个本科专业，设有研究生院、教育与心理科学学院、教育管理学院、继续教育学院、文学字、理学院、经济学院、资源与环境学院、管理学院、应用科学与技术学院、对外汉语教育学院，并受国家教委委托建立了高等学校师资培训交流北京中心。研究生院招收攻读硕士、博士学位的研究生，并举办研究生班以及承担代招国家研究生的任务。教育管理学院培养高等教育管理和研究方面的高级人才。对外汉语教育学院招收外国学生学习汉语。学校还设有马列主义毛泽东思想、教育科学、国际与比较教育、高等教育、俄罗斯文学、史学、文籍、发展心理、低能核物理、现代化教育技术、环境科学、数学与数学教育、非平衡态系统、应用化学、学校体育等15个研究所和一批研究室，以及计算、分析测试、教育技术、亚太经济。中学政治教育。中学计算机教育、特殊教育、未来教育等研究中心。

学校现有本科生近6000人，研究生1500余人，进修生500余人，港澳台学生20多人，外国留学生400多人（其中有博士、硕士留学生100多人），夜大学和函授学生6000余人。学校实行学分制，部分本科生可攻读第二学士学位或辅修专业，优秀本科毕业生可免试保送为硕士研究生，优秀硕士研究生可提前直接攻读博士学位。

学校现有教职工4000余人，有正高级职称的300余人，副高级职称700人，中级职称1100余人，其中专职科研人员700余人，他们中间有学术造诣精深、蜚声海内外的老学者，还有一大批崭露才华的中青年学术骨干。

学校现有国家重点学科7个，博士点学科40个，硕士点学科92个，还有一批新的学科生长点，基本构成了多种层次、门类齐全、质量较高的学科体系。中文、历史及数学、生物、物理、地理、心理等系科已分别被确定为国家文科及理科基础科学研究和教学人才培养基地。目前，该校博士点学科的数量在全国高校中位居前列，研究生人数已超过本科生人数的四分之一，从结构上看，已形成本科教育与研究生教育并重、教学和科研两个中心的格局。成为我国培养高层次人才和科学研究的重要基地之一。

北京师范大学具有良好的学习和研究条件，学校图书馆是全国教育中心馆，藏书280余万册。理科各院、系、所设有117个基础及专业实验室。学校出版社是全国有影响的高校出版社之一。学校出版《北京师范大学学报》等十余种高水平的学术刊物。学校有6个校办工厂，一些有前途的科技和文化产业已经起步。学校设有三所附属中学以及实验小学和实验幼儿园。

北京师范大学积极贯彻改革开放的方针，开展了广泛的国际学术交流。到目前为止，学校已和20余个国家和地区的50余所大学或研究机构建立了包括交换学者、留学生、进修生和共同开展科学研究以及交换科技信息、资料等内容的经常性的交流协作关系。

国家重点学科

中国民间文学、教育基本理论、比较教育学、基础数学、概率论与数理统计、细胞生物学、生态学

专业简介

北京师范大学有七个国家文、理科人材培养基地。

北京师范大学中国语言文学和历史学专业是国家教委批准建设的“国家文科基础学科人才培养和科学研究基地”，数学、物理学、生物学、地理学和心理学专业是国家教委批准建设的“国家基础科学研究和教学人才培养基地”。

建立基地的目的，是为适应二十一世纪的需要培养一批高层次文、理科基础科学研究和教学人才。国家为每个文科“基地”，每年投入建设经费15万元人民币，为理科“基地”每年投入建设经费约60万元。学校还有相应配套经费。“基地”培养的本科毕业生，除一部分被优先选送到国家科学研究单位和高等学校等实际工作部门外，多数将进入研究生阶段学习。

“基地”点的确定是依照扶强保重、合理布局的原则。在全国高校中，选择坚持正确方向、教学科研水平高、在国内同类学科中能起骨干带头作用的学科点。基本条件包括：有一个以上博士点或五个以上硕士点；有一支能够满足本学科专业教学和科研需要的高质量师资队伍；有一套深化教学改革的思路和措施。有教育改革和重点建设的目标，有切实可行的人才培养和学科建设方案等。

该校中国语言文学、历史学两个文科“基地”，1995年开始招收本科生，物理学、生物学、心理学等5个“理科”基地从1997年开始招收本科生。进入该校文、理科基地学习的学生，可享受优惠待遇：（1）奖学金获奖者的比例高于非基地学生；（2）在图书资料借阅、教学设备使用等方面，提供更充分的条件；（3）主要课程由具有高级职称的教师讲授；（4）配备优秀教师担任导师，负责指导学生的学习和科研；（5）统一考虑本科生和研究生培养的连续性，在保持本科生、研究生现行学制的前提下，试行“本科生-硕士生-博士生”连读制，即在本科四年级学生中经严格考试选拔品学兼优者直接进入硕士研究生阶段学习，二年后，选择优秀生提前攻读博士学位。（6）“基地”内各层次的毕业生，学校将根据有关政策和毕业生的志愿优先推荐到有关高校、科研单位和实际部门工作。

中文系

中文系有着90多年的悠久历史，在国内外均有重要地位和影响。90多年来，为国家培养了一批批的语言文学专家教授、高级教师、著名作家诗人、编辑记者和各级党政领导人。

1994年，中文系被确定为首批“国家文科基础学科人才培养和科学研究基地”。

中文系现设汉语言文学和编辑学两个专业。

毕业后，可到各类学校、国家机关、新闻出版、文化、科研等单位谋职。

历史系

北京师范大学历史系是国家教委首批确定的“国家文科基础学科人才培养和科学研究基地”。主要培养面向21世纪高质量的教学、科研人员、各新闻、出版、党政部门工作人员。

历史系设有历史学专业。

哲学系

哲学系设有哲学、思想政治教育、法学三个本科专业

经济系

教育系

建于1915年。历史悠久，学科齐全，实力雄厚。在国内屈指可数，在国际上也负有盛名。

现设有教育学、教育管理、学前教育和特殊教育四个专业。

外语系

北京师范大学外语系历史悠久，其前身为国立北方师范大学英语系。该系现设有英语语言文学专业、俄语语言文学专业和日语语言文学专业。目前共有教授、副教授46名，年龄层次分布合理、梯队健全。每年还聘有十几位外籍文教专家及语言教师在本系任教，全系教学科研力量雄厚。英语语言文学、俄语语言文学可授予博士学位和硕士学位；外语教学法、翻译学、日本语语言文学可授予硕士学位。优秀的本科毕业生可直接保送读硕士研究生，优秀的硕士生可直接攻读博士。

各专业学生还要学习两年第二外语。
