西北工业大学

学校概况

西北工业大学坐落在举世闻名的古都西安。它是党委书记任职由中组部批准、校长由国务院管理的“七五”、“八五”国家重点建设的15所高校之一。1995年11月被国家教育委员会和国家计划委员会批准首批进入“211”工程，并已正式立项，重点投资，启动建设。

西北工业大学成立于1957年10月，它的前身是1938年成立的西北工学院和1952年成立的航空精英荟萃的西安航空学院即原华东航空学院合并而成，1970年哈尔滨军事工程学院空军工程系迁并入校。

西北工业大学是全国唯一一所以发展航空、航天、航海为特色，以工为主，工理管文相结合的多科性科学技术大学。

西北工业大学占地1300余亩，校舍建筑面积50多万平方米，图书馆藏书140余万册，学校设有中国航空科学技术研究院西工大分院、研究生院、试飞员学院（亚洲唯一，世界6个试飞员学院之一）、民航工程学院（中国首家）、电子信息科学与工程学院、管理学院、航天工程学院、航海工程学院、材料科学与工程学院和继续教育学院等10个学院和15个系，在校学生逾万人，其中博士、硕士生1700余人。现有44个本科专业、59个硕士学位点、22个博士学位点、5个博士后流动站及三个重点学科。有8个国家级重点和专业实验室。2个部级重点实验室，3个省级工程研究中心。有28个研究所和20个研究中心。无人飞机研究与发展中心被国务院发展研究中心公布为“全国最大的无人飞机科研生产基地”，并颁发《中华之最荣誉证书》。

学校现有教职工近4000名，其中高级职称1075名，中科院院士1名，工程院院士2名，国际高校科学院院士1名，俄罗斯宇航科学院院士2名，博士导师115名，国家级有突出贡献的专家和国家重点攻关项目的科技顾问150名。

建校以来，西北工业大学为国家培养了5万余名高级科技人才，其中包括6个学科的中国第一位博士（航空宇航制造工程、飞行力学、航空发动机、固体火箭发动机、水中兵器、铸造）。截止目前，已有27位博士和青年教师获得世界著名的洪堡研究基金，居全国高校第一。

西工大科研实力居全国高校前列。1978年以来获国家级及省部级科研成果奖1000余项，获奖数名列全国高校前茅。

　

毕业生就业方向

主要集中在航空航天系统、船舶工业总公司系统、国家部委及大中型城市的相关单位。

国家重点学科

铸造、航空宇航制造工程、一般力学
