大连理工大学

学校概况

　　大连理工大学创建于1949年4月15日，原名为大连工学院。1988年改名为大连理工大学。是国家教委直属的全国重点大学。

　　学校坐落在辽东半岛风景秀丽的海滨城市大连。校舍建筑面积51万平方米。

　　本校现有专任教师1822人。其中教授270人，副教授680人，讲师580人。

国家重点专业

　　计算力学、机械制造、海岸工程学、精细化工
