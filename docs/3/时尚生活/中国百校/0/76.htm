江西农业大学

学校概况

1952年由南昌大学农学院分出建立江西农学院。1980年改名为江西农业大学。

学校设在江西省南昌市郊。省教育委员会直属。

校舍建筑面积15.18万平方米。

学校现有专职教师450多人，其中教授14人，副教授146人，讲师175人。

本科专业设置

作物栽培、植物遗传育种、土壤与植物营养、林学、兽医、畜牧、农产品贮运与加工、植物保护、果树蔬菜、农牧业经济管理、农业机械化、农艺教育、园艺教育、畜禽生产教育、动物营养与饲料加工、土地规划与利用、园林会计

授予硕士学位的专业

动物生理学、动物生物化学、作物栽培学与耕作学、动物遗传育种学、动物生产学、动物解剖学、组织学及胚胎学、兽医微生物学与免疫学、兽医内科学、森林植物学、农业经济及管理、森林地理学
