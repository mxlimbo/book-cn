北京科技大学

学校概况

北京科技大学是1952年由北洋大学等5所国内著名大学的部分系科组建而成，现已发展成为工、理、管、文项结合的多科性全国重点大学，是全国第一批成立研究生院的22所院校之一。

学校现有8个学院，29个本科专业，3个第二学士学位专业。并设有41个硕士点、17个博士点几个博士后科研流动站。学校有6个国家重点学科，建有两个国家级重点（专业）实验室，两个部委级开放实验室和一个国家工程研究中心。目前有在校本科生5607人，研究生1424人（其中博士生413人）、在站博士后27人、外国留学生77人，成人教育学院学生3705人。

北京科技大学名师蔡萃，师资力量雄厚。在1132人的教师队伍中，有教授232人、副教授412人。其中，中国科学院院士5人、中国工程院院士1人、国务院学科评议组组员4人、博士生导师98人。近年来，一支高水平、富有创新精神、治学严谨的跨世纪学科梯队正在壮大，涌现出一大批思想活跃、才华横溢的中青年学术带头人。其中，国家级有突出贡献的科技专家16人，省部级有突出贡献的科技专家13人，国家教委跨世纪优秀人才培养计划入选专家2人，北京市青年学科带头人、优秀骨干教师105人。

学校现聘有100多位国内外兼职（客座）教授和8名国外名誉教授。学校还与11个国家23所著名大学建立了校际合作关系，并与国内许多省市和著名企业集团建立了办学及科研协作关系。

北京科技大学在国内学术榜上占据重要地位。1987年以来，安装工程系列索引收论文排序，该校在全国高校中的排名为第二十至十六位。在科学研究方面，该校积极开展新兴学科和边缘学科的研究，取得了一系高精尖重大科研成果，以及应用基础理论方面的重大突破。1978年以来，共有821项科研成果获国家、省部级及其他各种奖励。其中，国家级奖91项。为国家的建设创造了巨大的经济效益和社会效益。
