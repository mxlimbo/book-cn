东南大学

学校概况

　　1902年建立三江师范学堂。1920年至1927年曾用名东南大学。1952年改为南京工学院。1988年改为东南大学。东南大学是国家教委直属重点大学。学校设在南京市。建筑面积41.76万平方米。

　　现有专职教师1403人。其中教授137人，副教授510人，讲师543人。

国家重点学科

　　建筑设计、通信与电子系统、电磁场与微波技术、自动控制理论及应用
