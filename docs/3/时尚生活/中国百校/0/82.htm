华东师范大学

学校概况

华东师范大学是国家教委直属的全国重点大学，创建于1951年10月，是新中国成立后为发展高等师范教育事业而首批建立起来的一所新型的社会主义师范大学。经过四十多年的辛勤耕耘，已发展成为一所学科比较齐全、师资力量雄厚，具有自己办学特色、颇具声望和影响的高等学府。

学校位于改革开放的前沿城市上海，校园环境优美，素有花园学校之誉。校舍建筑面积45.6万平方米。

华东师范大学现有8个学院，6个系，42个本科专业；1个国家级重点实验室，2个国家教委级开放实验室，4个国家重点学科，5个国家人才培养和科研基地；27个研究所，88个实验室及一批计算、测试、培训及综合研究中心；学校设有研究生院，现有32个博士点、97个硕士点、4个博士后流动站。全校现有各类学生14,000多人。

学校现有博士生导师100人，正副教授和具有其他高级职称的教师940人，其中不少是在国内外享有盛誉的专家和学者。

四十多年来，特别是党的十一届三中全会以来，该校在党的基本路线指引下，端正办学指导思想，注意师范性与学术性的统一，坚持高水平的学科建设，高水平的教学与科研的同时努力为基础教育服务，为国家的经济和社会发展服务，倡导“求实、创造、为人师表”的校风，学校工作实现了新的飞跃。现在，该校已形成了兼有教育科学、社会科学、人文科学、自然科学、技术科学和管理科学的多科性师范大学的新格局，承担了以培养高质量师资为主、兼顾培养其他各类人才的重任，在体系和功能上理炙明显地反映出全方位、多层次、多规格、多渠道的新特点。

1996年4月，该校顺利通过了国家教委的“211工程”部门预审，确定了15年发展规划：把华东师范大学建成以教育学科为重点、多学科高水平协调发展、国内领先、国际上有影响的社会主义师范大学。

该校坚持社会主义办学方向，不断深化教育改革，强化为国家培养合格的社会主义事业建设者和接班人的意识。该校强调把学生的德育教育放在首位，加强“两课”建设；该校长期抓主干课程建设，迄今已有150多门主干课程验收合格，保证了基础教学的质量；该校实行主辅修专业制，已开设了辅修专业25个，增强了毕业生的社会适应性；该校还加强了社会实践和师范生必备的基本功训练，注重文理渗透，重视外语、计算机和体育教学。目前，学校已着手扩大受教育者学习自主权的探索，逐步推行完善的学分制，并努力推进面向21世纪的课程体系和教学内容的改革，探索体现新世纪教育观念和教育需要的新的人才培养体系。

学校的许多学科专业具有明显的优势与特色，在基础研究、应用研究、开发研究和高新科技研究上均取得丰硕成果。据不完全统计，十一届三中全会以来，承担国家和地方科研项目近3千项，其中国家自然科学基金及社科基金375项。1996年科研经费超过2千万元，列国内同类学校首位、一大批教育、文、理学科的科研成果达到国内外先进水平，不少具有显著的社会效益和经济效益，683项成果分别获国家、部委、省市级奖励；出版各种著作2300余部，在中外杂志上发表论文逾2万篇，近5年来，被世界三大检索系统登录的科技论文数在全国高校中排名20-50名之间。

学校积极发展与海外的学术交流与合作，现已与英、法、德、日、美、俄、加、澳等19个国家的近80个高等学校、研究单位和产业部门建立了合作交流关系。100多位外国专家受聘担任了该校名誉教授或顾问教授。目前有500余位留学生在校学习、该校每年派出200余人到国外学习、交流或进行合作研究。

国家重点学科

自然地理学、概率论与数理统计、中国教育史、教育基本理论
