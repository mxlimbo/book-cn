哈尔滨工业大学

学校概况

　　哈尔滨工业大学于1920年创建。是中国航天工业总公司直属的全国重点大学。校舍建筑面积52.24万平方米。学校位于哈尔滨市南岗区。

　　学校在职教师总数1632人。其中教授274人，副教授535人，讲师706人。

　　金属材料及热处理、铸造、焊接、计算机应用、机械制造、飞行器导航控制系统、光学
