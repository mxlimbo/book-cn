成都理工学院

学校概况

1956年10月建院时名为成都地质勘探学院。1958年11月更名为成都地质学院。1993年5月更名为成都理工学院。

学校现有教师总数888人，其中教授73人，副教授309人，讲师393人，教员29人，助教84人。

国家重点学科

煤田·油气地质与勘探、工程地质

本科专业

马克思主义基础、英语（经贸英语）、地质学、地质矿产勘查、应用地球物理、石油工程（油藏工程）、核技术、计算机及应用、建筑工程（岩土工程）、水文与资源利用、数学地质、地质矿产勘查（铀矿地质）、勘察工程、应用电子技术、工业管理工程、工业分析

授予硕士学位的学科、专业

矿物学、岩石学、沉积学（含古地理学）、探矿工程、矿床学、地球化学、古生物及地层学、构造地质学（含地质力学）、矿产普查与勘探、煤田·油气地质与勘探、水文地质与工程地质、应用地球物理、放射性地质与勘探、数学地质
