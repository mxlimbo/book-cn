西安电子科技大学

学校概况

1931年2月建立军委无线电学校。1933年3月改为中国工农红军通信学校。1935年12月改为中央军委通信学校。1937年1月改为延安通信学校。1942年改为电信工程专科学校。1947年3月改为晋察冀军区电讯工程专科学校。1949年11月改为中央人民政府人民革命军事委员会工程学校。1952年5月改为中国人民解放军通信工程学院。1960年1月改为中国人民解放军军事电信工程学院。1966年4月改为西北电讯工程学院。1988年1月改为西安电子科技大学。是电子工业部直属的国家重点院校。建筑面积30.92万平方米。

西安电子科技大学现有教师1500多人。其中教授128人，副教授303人，讲师698人。

国家重点学科

通信与电子系统

本科专业设置

通信工程、信息工程、电子工程、应用电子技术、生物医学工程、自动控制、电气技术、工业自动化、计算机及应用、计算机软件、机械电子工程、精密仪器、光电子技术、微电子技术、电子材料与元器件、电磁场与微波技术、管理工程、工业工程、科技信息、应用数学、计算数学及其应用软件、英语、工商行政管理、市场营销、国际企业管理、思想政治教育、应用物理学、检测技术及仪器仪表、电子仪器及测量技术

授予硕士学位的学科、专业

通信与电子系统、电路与系统、信号与信息处理、计算机组织与系统结构、计算机应用、计算机软件、计算机器件与设备、机械制造（电子机械）、物理电子学与光电子学、半导体器件与微电子学、电磁场与微波技术、应用数学、无线电物理、系统工程、生物电子学、电子材料与元器件、密码学、自动控制理论及应用
