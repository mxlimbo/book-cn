华西医科大学

学校概况

华西医科大学是卫生部直属的全国重点医科大学。前身私立华西协和大学，创建于1910年。1953年更名为四川医学院。1985年更名为华西医科大学。学校设在四川省成都市。校舍建筑面积24.98万平方米。

现有专职教师1152人。其中教授148人，副教授193人，讲师537人。

国家重点学科

口腔科学（口外、口矫）

本科专业设置

临床医学、医学检验、法医学、口腔医学、预防医学、卫生检验药学、药物化学、临床药学、英语、妇幼卫生护理学
