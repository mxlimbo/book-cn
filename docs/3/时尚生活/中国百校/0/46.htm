西南交通大学

学校概况

西南交通大学的前身是山海关北洋铁路官学堂，成立于1896年。1928年2月改唐山交通大学。1930年9月改交通大学唐山工程学院。1952年5月改唐山铁道学院。1972年3月改西南交通大学。学校设在四川省成都市。校舍建筑面积55万平方米。

学校现有教师1010人。其中教授122人，副教授248人，讲师501人。

国家重点专业

桥梁·隧道及结构工程、机车车辆、铁道牵引电气化与自动化
