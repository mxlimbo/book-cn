北京邮电大学

学校概况

北京邮电大学是隶属于国家邮电部的国家重点大学。位于北京市海淀区。校舍面积24.78万平方米。

其前身北京邮电学院于1955年7月成立，1970年10月更名为北京电信工程学院。1973年7月恢复北京邮电学院。1993年11月改名为北京邮电大学。

北京邮电大学现有教师864人。其中教授124人，副教授283人，讲师327人。

国家重点学科

通信与电子系统、电磁场与微波技术

本科专业设置

通信工程、无线通信、无线电技术、图象传输与处理、电磁场与微波技术、邮电管理工程、邮政通信管理、邮电经济管理、管理信息系统、计算机通信、计算机及应用、计算机软件、电子设备结构、电子精密机械、信息工程、信息科学、应用物理、应用数学、科技英语、应用电子技术、思想政治教育

授予硕士学位的专业

通信与电子、系统信号与信息处理、电路与系统、电磁场与微波技术、运输经济管理工程、计算机应用、机械学、机械制造应用、数学、半导体器件与微电子技术、光学密码学
