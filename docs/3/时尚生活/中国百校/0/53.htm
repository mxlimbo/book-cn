哈尔滨工程大学

学校概况

哈尔滨工程大学的前身是我国第一所高等军事科学技术学府-----中国人民解放军军事工程学院（简称“哈军工”）。学院创建于1953年。它曾有过尽人皆知的辉煌历史，为我国国防现代化建设培养了大批高级科学技术人才。

1970年，根据国务院和中央军委的指示，在该院完整建制的海军工程系的基础上组建了哈尔滨船舶工业总公司，是我国船舶工业系统唯一的一所以工为主，工、理、管、文相结合的多科性全国重点大学。

1996年1月，学校通过了“211工程”立项审核，成为全国第一批通过可行性论证和立项审核的20所院校之一。

学校现有两个学院、15个教学系部（中心），有本科专业39个，专科专业11个，在校生总数7000人，29个学科专业具有硕士学位授予权，8个学科专业具有博士学位授予权。学校的水声工程专业被国家列为重点学科点，设有船舶与海洋工程博士后科研流动站，可以接收博士后科研人员。学校还设有水声研究所、新能源研究所、海洋综合技术研究中心等11人科研机械，以及近50个校内外实习基地、上百个专项实验室，形成了教学科研并重的全方位办学体系。

学校现有教职工2400人，其中教学科研人员1200余人，教授、副教授、高级工程师600余人，讲师、工程师700多人。在教师当中，有中国工程院院士、国务院学位委员会学科评议组成员以及博士生指导教师等学科带头人。他们当中既有身为船舶与海洋工程诸学科奠基人的老一代科学家，也有近几年破格晋升教授、副教授的后起之秀。近年来在师资队伍中充实了大批博士、硕士毕业的研究生。

学校十分重视教学质量的提高。从1978年起，学校就积极探索教育改革，试行学分制，同时试行双学位制、主辅修制、“本科一硕士”连读制，为培养社会主义市场经济需要的复合型人才和拔尖人才，采取了一系列教育改革的措施和办法。

在教学中学校贯彻“加强基础、突出主干、拓宽专业、注重能力培养”的基本思想。同时，学校非常重视学生的外语教学和计算机应用等实践能力的培养，为学生参与人才市场竞争创造了有利条件，在多年的教学实践中学校摸索出一整套提高教学质量的新路子，使人才培养的质量稳步提高，先后为国家输送了上万名合格人才，近几年，学校本科生报考研究生的升学率逐年提高，全国大学英语四、六级统考的通过率年年保持在全国重点院校的先进行列。学校的毕业生以基础扎实、适应性强、思想素质好而受到用人单位的广泛赞誉。

在不断提高教学质量的同时，学校重视科研水平的提高，近年来已取得重大科研成果近千项，其中获国家和省部级科技成果奖200多项。

学校重视开展国际合作与学术交流，已与美、日、法、德、意、加等二十几个国家有影响的大学和科研机械建立了广泛的交流与合作关系。

毕业生就业去向

国家部委所属院校担负着为本系统培养高级技术人才的任务，它的毕业生大部分要到本系统的单位去工作。该校隶属中国船舶工业总公司，其下属企事业单位200个，大多分布在沿江沿海城市，因此每年除相当部分毕业生考取或免试推荐研究生，蓁大多通过供需见面、双向选择到这些城市与船舶工业、海洋开发有关的研究所、设计院、工厂和高校工作，也有部分同学通过部委交换或地方留成等渠道，到外系统或地方企业、事业及科研单位从事工程技术及管理等方面的工作。
