中国协和医科大学

学校概况

卫生部直属。1917年9月建校，校名为北京协和医学院。解放后改为中国协和医学院。1959年改为中国医科大学。1978年改为中国首都医科大学。1985年改为现名。

校舍面积23.43万平方米。该校教学科研力量强大。

现有教师总数886人。其中教授227人，副教授481人，讲师71人。

国家重点专业

生物学、生物化学、药理学、内科学（心血管病、血液病）、外科学（胸心）核医学、药物化学

本科专业设置

医学（临床医学，学制8年）、护理（护理学）

授予硕士学位的专业
