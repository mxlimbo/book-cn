东北大学

学校概况

　　东北大学位于辽宁省沈阳市。校本部南滨浑河，北畔南湖，学校占地面积198万平方米（3000）亩，建筑面积75万平方米。

　　东北大学始建于1923年4月。1928年8月，著名爱国将领张学良将军兼任校长。1949年3月，在东北大学工学院和理学院（部分）的基础上成立了沈阳工学院，1950年8月，定名为东北工学院。1960年被列为全国61所重点大学之一，是国务院首批批准有权授予学士、硕士和博士学位的大学。1986年进入全国33所试办研究生院学校的行列，1996年10月正式成立研究生院。1993年3月8日，经国家教委批准，东北工学院复名东北大学，张学良先生亲自提定校名，并出任名誉校长、校董会名誉主席。1996年，经国家计季、国家教委、财政部批准，东北大学首批进入国家重点建设的“211工程”。

　　党和国家对东北大学的建设与发展十分重视，许多领导同志曾先后来校视察。国家主席江泽民、国务院总理李鹏、副总理李岗清先后为学校题词。

　　东北大学是一所以工为主，理、工、文、管、经、法相结合的多科国家重点大学。学校现高有理学院、资源与土木工程学院、材料与冶金学院、机械工程学院、信息科学与工程学院、文法学院、工商管理学院、黄金学院、外语系；还设有辽宁分校、秦皇岛分校、成人教育学院、冶金部东北大学继续教育中心。学校共设有46个专业，有53个学科有权授予硕士学位，3个学科有权授予博士学位。学校有国家级工程研究中心2个，省级工程研究中心3个，国家重点实验室1个，冶金部开放实验室1个；国家重点学科2个，部级重点学科9个，省级重点学科8个；博士后流动站5个，“中国教育与科研计算机网？东北地区节点设在东北大学。

　　学校现有教师1950人，其中教授245要，副教授706人，讲师651人。在87位博士导师中，有中国科学院院士1名，中国工程院院士1名，国务院学科评议组成员7名，挪威皇家科学院院士1或，澳大利亚工程科学院院士1名。现在校各类学生20621人，其中博士研究生541人，硕士研究生1266人，外国留学生72人，建校70多年来，东北大学为国家培养种类专业人才11万余人。

　　学校现有28个科学研究所，78个实验室和5个教学、科研服务和实验中心。学校图书馆建筑面积21826平方米，藏书165万册，有各种语言的期刊杂志2779各。东北大学出版社每年出版各种学术专著、教材及其它图书近200种。

　　东北大学坚持面向国民经济建设主战场，不断探索产学研相结合的道路，形成了教学和与科研项目达9056项，累计科研经费3亿多元，其中910项获国际、国家和省市级奖励。1995年国家科技进步奖、国家 发明奖等级和数量综合排序，东北大学名列全国高校第五名、第六名。“八·五“期间，东北大学主办国际学术会议12次，国内学术会议50余次，编写出版专著、教材359部，发表论文7000余篇，其中被国际三大检索系统收录419篇，被《EI》收录79篇。

　　东北大学的高科技产业迅速发展，先后建立了校办高新技术企业二十多家，与日本、新加坡、香港等国家和地区创办合资企业几十家，科技产业年产值已突破亿元。东北大学科学园已成为冶金行业、辽沈地区高新技术的辐射源。占地面积810亩，总投资5亿元的东北大学软件区，将成为中国最大的软件生产基地。

　　近年来，东北大学的国际交流不断扩大，已先后同美国、日本、英国等16个国家和地区的50多所大学、科研院建立了国际学术交流关系，邀请了2000多位外籍专家来样讲学、任教及合作科研，先后选派了1400多人（次）到17个国家和地区的大学、科研院所讲学、合作科研、参加国际学术会议和攻读学位。

　　东北大学顺应中国改革开放的大潮，充分利用地处东北亚经济圈的地理优势，正在逐步把学校办成在国际上有圈套影响的国内一流大学。

专业简介

　　计算机科学与技术专业、电子与信息技术专业、管理信息系统专业、生物医学工程专业、自动化专业、测控技术与仪器专业、机械工程及自动化专业、真空技术及设备专业、流体传动及控制专业

　　学生毕业后可以继续攻读硕士、博士学位，可以到高等学校和科研单位从事教学和科研工作，或到工厂企业从事机械、电子、液压与气动一体化系统产品的设计、开发、运行管理和市场营销等方面的工作。

管理工程专业

　　本专业、主要培养从事各项专业、管理（部门及企业单位）与综合管理，既懂经济又懂技术，掌握现代管理理论，能运用现代管理方法和手段的高级管理工程人才。

工业工程专业

　　本专业、是我国管理学科中新增设的专业，侧重培养懂经营的高级管理工程人才。要求学生掌握扎实的工程技术基础理论、经营管理知识和现代管理技术，具有综合运用所学理论与方法，研究并实现生产与管理业务系统整体优化的能力。

市场营销专业

　　本专业、培养适应市场经济及国际化营销发展的需要，具有企业经营管理，包括企业经营战略制定、市场调查及营销、生产组织与计划、人员组织与调配、资金筹措与运用等知识和能力的高级经营管理人才。

会计学专业

　　培养适应社会主义市场经济发展的需要，懂得一定的工程技术知识，又具有扎实的专业、技术的高级人才。

工业外贸专业

　　本专业、培养从事进出口业务、对外经济技术合作、外贸企业管理等涉外经济活动的高级经贸人才。

　　国际金融专业、金属压力加工专业、材料科学与工程专业

　　矿物加工工程专业、环境工程专业、国际经济法专业

　　思想政治教育专业、行政管理学专业、应用数学专业

　　应用物理学专业、应用化学专业、材料物理专业、经贸英语专业
