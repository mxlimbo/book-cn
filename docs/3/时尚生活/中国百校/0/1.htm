清华大学

学校概况

　　清华大学的前身----清华学堂，是清政府利用美国政府“退还”的部分“庚子赔款”于1911年办起来的一所留美预备学校。1925年设大学部，1928年改为国立清华大学。抗日战争爆发后，南迁长沙，后又迁至昆明，与北京大学、南开大学联合组成西南联合大学。1946年迁回清华园复校。新中国成立后，清华大学成为一所多科性工业大学。1978年以来，清华大学已成为一所以工科和理科为主体，兼有文科和经济管理学科的综合大学。

　　在校学生数，普通专科生708名，普通本科生10061名，夜大学生1205名，外国留学生266名，硕士生2228名，博士生1018名。

　　现有专任教师数2175人，其中：教授326人，副教授914人，讲师609人。
