江苏理工大学

学校概况

　　江苏理工大学是隶属国家机械工业部的全国重点大学，是国内首批有权授予博士、硕士、学士学位的高校。

　　建校三十七年来，学校坚持为机电行业和江苏及沿海地区经济服务的方针，紧密依托国民经济重点发展的农业、交通、机械、汽车、能源、环保等部门，形成了以工为主、理工结合、文、经、营、法相配套的办学特色，是国内一所具有较强办学实力和较高学术知名度的综合性理工大学。

　　“团结、进取、勤奋、求实”是江苏理工大学朴实的校风。学校把学生的德育教育放在首位，努力加强学生思想教育和党建工作，深入开展“争先创优”和“优良班风”达标活动，以及形式多样、丰富多彩的课外科研、文化、娱乐和社会实践活动，培养跨世纪的合格人才。

　　江苏理工大学坐落在江南著名历史文化名城----江苏省镇江市，前临就沪铁路、沪宁高速公路，后依长江，校园绿树成荫，素有镇江“东郊公园”的美称。校园占地75公顷，各类建筑面积23万多平方米。

　　目前，各类在校生12000余人，图书馆藏书65万余册，教职工2000余人。学校有正副教授、研究员、高级工程师410余人，讲师、工程师700余人。

　　学校现设农业机械工程学院、汽车学院、工商管理学院、人文社会科学学院、动力机械工程系、机械制造工程系、电气工程系、计算机科学系、材料工程系、工业工程系、工业设计系、数理系以及成人教育学院、机电学院、冶金学院、研究生部等院、系、部，建有一个博士后流动站，六个部级重点学科、二个省级重点学科、三十多个本科专业和方向、二十四个专科专业。经国务院学位委员会批准，农业机械设计制造、内燃机、农产品加工工程、流体机械及流体动力工程、汽车设计制造五个学科、专业有权授予博士学位，农业机械设计制造、汽车设计制造、内燃机、流体机械及流体动力工程、机械制造、金属材料及热处理、固体力学、机械学、振动冲击噪声、农产品加工工程、计算机应用、热能工程、管理工程、工业自动化、会计学、电力传动及其自动化、铸造、实验力学等十八个学科、专业有权授予硕士学位。

　　江苏理工大学重视科学研究以及科研成果的转化，积极为经济和社会发展服务。几年来，学校先后承担国家自然科学基金、“863”高科技项目、“火炬”计划项目；博士点基金项目、“八五”、“九五”攻关项目等纵向高新技术课题。已有10余项成果获国家级奖励，200多项成果获部、省级科技进步奖，100余项成果获国家专利。

　　江苏理工大学重视国际交往，积极开展与国外的学术交流。目前与美国威斯康星----密尔瓦基大学、日本三重大学、俄罗斯国立顿河技术大学、泰国清迈大学、巴西巴拉那州立联邦大学签订了长期合作协议。并先后与美国、日本、德国、奥地利、英国、法国、加拿大、澳大利亚、意大利、巴西、新加坡等国家和地区的一批高校、科研机构建立了友好联系。
