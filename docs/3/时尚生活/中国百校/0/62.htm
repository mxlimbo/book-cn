厦门大学

学校概况

厦门大学环境优美，办学条件良好。花园式的校园，坐落在鹭岛东南之滨。依山傍海、风光秀丽。占地面积145.9公顷（2188）亩。校舍建筑60万平方米。

厦门大学是爱国华侨领袖陈嘉庚先生于1921年建立的。是目前唯一地处经济特区的国家教委直属的综合性大学。1994、1995年厦门市、福建省先生与国家教委共建厦门大学。1995年6月通过国家教委“211工程”的部门预审，已被国家正式立项为“211工程”重点建设的高校。

厦门大学学科较为齐全，人才培养系列相当完备。改革开放以来，学校主动适应对外开放、经济发展、特区建设的需要，更新老专业，创办新专业，发展边缘、新兴、交叉学科。目前，学校已形成了包括自然科学、技术科学、人文科学、社会科学、管理科学和医学多学科的结构合理门类较齐全、内容比较先进的学科体系。厦门大学现有27个系，62个本科专业，71个硕士点，22个博士点，4个博士后流动站。

厦门大学既是教育中心，又是科学研究中心。学校建立了物理化学研究所、经济研究所、南洋研究所、台湾研究所、高等教育科学研究所和细胞生物学研究室等40多个科学研究机构。物理化学、会计、财政、计划统计、高等教育学、动物学、专门史（中国古代经济史）等学科被评为国家重点学科。固体表面物理化学国家重点实验室，是国内首批建成并对国内外开放的国家重点实验室，海洋生态环境研究实验室、材料和生命过程分析学科研究实验，经国家教委批准，成为国家教委开放研究实验室；肿瘤细胞生物工程实验室列为国家重点专业实验室。化学系化学专业、生物系生物学（含海洋生物）专业和历史系历史学专业被确定为国家“基础学科研究和教学人才培养基地”。

学校目前设有研究生院、经济学院、工商管理学院、政法学院、工学院、化学化工学院、海洋与环境科学学院、医学院、艺术教育学院、海外教育学院、南洋研究院、成人教育学院。形成了包括学士、硕士、博士、博士后和留学生教育等完整的高等教育人才培养系列。

素称“南方之强”的厦门大学，蔡萃着众多的学者名流，造就了大量栋梁之材。在中国科学院、工程院院士中，有厦大校友29人。现有1440多名教师和专职科研人员中，有教授（研究员）210多人，副教授（副研究员）730多人。中科院院士5人，国务院学科评议组成员4人，博士生导师83人，国家级有突出贡献专家10人，在国际学术组织中担任重要职务18人，国家跨世纪人才3人。学校不仅集中了一大批造诣精深的专家教授，而且拥有不少出类拔萃的后起之秀。学校还聘请李远哲、杨振宁等一批国际一流学者为名誉教授。

学校教学、科研设施较为先进，拥有国家教委厦门大学文科文献信息中心、测试中心、计算中心、电教中心，校园高速信息网络等一批现代化的实验室。图书馆面积18200平方米，藏书186.9万册，建立了光盘检索系统和计算机流通系统，并建成了先进的电子文献信息部。学校与厦门市邮电局合作，建设了厦门大学数字程序电话局，学生宿舍也铺高设了线路，我校将率先成为全国高校可以使学生宿舍的计算机并入校园高速信息网络的高校。

自建校迄今的76年来，厦门大学为国家培养了5万多本科生和研究生，其中不少成为蜚声中外的专家、学者。现有在校各类学生1万多人。其中本专科生近万人。研究生1400多人，外国留学生200多人。还有遍布世界五大洲国家和地区的海外函授生5000多人。

厦门大学的科学研究积极贯彻为社会主义现代化建设服务的方针，取得显著成果。1980年以来，理科各系，所获得国家和省级科研成果奖229项。其中，获国家自然科学奖9项，总积分居全国高校第七位。国家发明奖2项，国家科技进步奖3项，国家教委科技进步奖41项，国家专利80项。在国内外学术刊物上发表论文5200多篇，出版专著200多部，90-94年被国际四大检索之一的SCI收录论文230多篇，其中95年被SCI收录数居全国高校第19位。1985年以来文科各系、所、室出版著作800多篇，发表论文8500多篇，获省部级以上人文社科成果奖235项。学校科研重视为厦门经济特区服务，在厦门经济特区发展战略、自由港实施方案、国际经济法、国家贸易、国际金融等方面的研究，为解决经济特区建设的重大理论问题和政策问题，提出有参考价值的意见。理工科各系在特区引进技术、企业技术改造等方面也作出重要贡献。

学校大力发展国际学术交流，先后与加拿大、美国、英国、法国、澳大利亚、日本、比利时、荷兰和菲律宾等国家27所高等院校建立了校际关系。共派出850多名教师、研究生分别到20多个国家留学、访问和讲学。还与加拿大联合举办“工商管理教育中心”。学校先后聘请国外专家、学生800多人次来校讲学，主办第四十六届国际电化学年会和第十九届国际统计物理会议等国际性大型学术会议。

厦门大学实行按系（或院）按专业类招生。

所谓按系（院）按专业类招生，就是学生填报志愿时，不填写系内或专业类内所设的专业，只填写系或专业类名称，学校按系或专业类录取新生，学生入学后，前一至二年所学的必修课程在系内完全一样，第三年、第四年学生根据社会需求状况及择业志向，在所学系或专业类的范围内选择专业或专业方向。比如财政金融系高有国际金融、金融、财政学、税务四个专业，考生凡是想报考这四个专业之一者，不必填写专业名称，只需填写财政金融系即可。

按系（院）按专业类招生与培养，有利于拓宽学生的知识面，这种宽口径培养模式使学生具备基础知识扎实、专业面宽、适应性强的特点，使学生毕业择业具有灵活性和竞争力。凡有能力、肯奋斗的学生，只要坚持不懈、刻苦努力、进入厦门大学后取得优异成绩的，都有机会在所学的系或专业类内选择自己喜爱的专业或专业方向。
