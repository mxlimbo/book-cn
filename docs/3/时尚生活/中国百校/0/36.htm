中国地质大学

学校概况

1952年11月成立北京地质学院。1975年初迁至武汉，更名武汉地质学院。1987年11月更名中国地质大学，并在京、汉两地办学，其下辖四个办学实体即中国地质大学（武汉）、中国地质大学（北京）、中国地质大学武汉管理干部学院、中国地质大学北京管理干部学院。中国地质大学隶属于地质矿产部。校舍建筑面积37.8万平方米。

现有教师1193人。其中教授246人，副教授348人，讲师426人。

国家重点学科

矿物学、岩石学、古生物学及地层学、矿产普查与勘探、探矿工程

本科专业设置

地质学、地层古生物学、岩石矿物学、构造地质学、矿床地质学、煤田地质勘查、自然资源管理与保护、地质力学、地质矿产勘查、工业分析、石油地质勘查、水文地质与工程地质（含岩土工程）、应用地球物理、探矿工程、矿业机械、安全工程、工业管理工程、矿产资源管理、企业管理思想政治教育、地球化学、计算机及应用（含软件）、专门用途英语（外贸）、地质学、探矿工程、环境工程

授予硕士学位的学科、专业

岩石学、古生物学及地层学、矿物学、地球化学、矿产普查与勘探、煤田·石油地质与勘探、水文地质与工程地质、探矿工程、数学地质构造、地质（含地质力学）、矿床学、应用地球物理、沉积学（含古地理学）、地貌学与第四纪地质学、环境地质、应用地球化学、分析化学、遥感地质、安全技术工程、油气田开发工程、自然地理学、人文地理学
