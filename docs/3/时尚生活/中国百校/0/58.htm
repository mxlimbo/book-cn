南京大学

学校概况

南京大学的前身是三江师范学堂，1902年建立。1905年改名两江师范学堂。1915年改名南京高等师范学校。1902年改名东南大学。1927年改名国立第四中山大学。1928年2月改名江苏大学。1928年5月改名国立中央大学。1948年8月改名国立南京大学。1950年10月定名为南京大学。

建筑面积48万平方米。

学校现有教师总数2251人。其中教授276人，副教授867人，讲师761人，助教347人。
