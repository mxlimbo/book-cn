北京大学

学校概况

北京大学地处中国首都北京西郊，北依圆明园，西临颐和园，校内湖光塔影，风景秀丽。校园面积230多公顷。

北京大学的前身就师大学堂成立于1898年12月，当时的京师大学堂既是中国的最高学府，也是中国最高的教育行政机关。1912年5月京师大学学堂改名为北京大学。

1916年12月蔡元培先生出任校长，实行“兼容并包”的方针，使北京大学的面貌发生根本的变化。到1919年，北京大学发展成为一所拥有14个系，2000多名学生的大学，是当时全国规模最大的高等学府。

1919年中国爆发反帝反封建的“五四”运动，北京大学是“五四”运动的发祥地，是中国新文化运动的中心，是马克思主义在中国传播的最初基地。中华人民共和国成立后，学校决定以5月4日为校庆纪念日。

这所古老而又年轻的大学，聚集了百年中国社会的痛苦和追求，也呈现了不屈不挠追求真理的蓬勃朝气。

这里延绵着不会熄灭的火种，它不是一种物质的遗传，而是一种灵魂的塑造和远播。生活在燕园里的人，都会感受并把握到这种无形而巨大的存在，这种存在超越时间和空间，已成为北大永存的灵魂。

1952年全国进行院系调整，北京大学从沙滩迁到燕园，逐渐成为一所侧重于基础科学教学和研究的文理科综合大学。

今天的北京大学把培养现代化建设需要的高级专门人才同重大科学问题的研究有效地结合起来，在继续加强和发展基础学科的同时，重点发展经济建设、科技进步和社会发展急需的应用学科、交叉学科和新兴学科，使学校成为一所拥有自然科学、技术科学、社会科学、人文科学、管理科学、教育科学、语言科学、医药科学和新型工程科学家等多种学科，集人才培养、科学研究、社会服务为一体的新型综合性大学。

1993年，菲律宾马尼拉评出的亚洲十强大学中，北京大学是中国大陆唯一榜上有名的大学。

1995年，美国《科学》杂志评出的中国十三所最杰出的大学，北京大学名列榜首。

北京大学师资力量居全国之首----现有教授846人，其中博士生导师536人，副教授850人。

北京大学是国内杰出科技人才最多的单位----现有中科院院士29人，中国工程院院士2人，国务院学位委员会及学科评议组成员34人，第三世界科学院院士4人。

北京大学拥有国家级重点专业42个，博士学位授予专业101个，硕士学位授予专业149个，居全国高校之首位。

院系构成

现在北京大学共有8个学院、21个系、87个学士专业、3个第二学士学位专业、149个硕士专业、101个博士专业，52个研究所、63个研究中心、2个国家级工程研究中心、42个国家重点学科，已建成国家重点实验室11个及国家重点学科专业实验室4个。

师资力量

北京大学现有专任教师2100多人，其中教授846人。现有中国科学院院士29人，中国工程院院士2人，第三世界科学院院士4人，国务院学位委员会学科评议组成员34人，博士生导师536人。

教学管理

近年来，北京大学大力推进教学改革，按照“加强基础，淡化专业，因材施教，分流培养”的方针，修订了教学计划，加强了基础课教学，充实更新了教学内容，注重培养学生的实际能力和创造精神。在教学管理上，实行学分制，提高了选修课的比例，并推荐品学兼优的应届毕业生免试攻读硕士博士学位。营造丰富多彩的校园文化氛围，加强学生课外学习和实践活动陶冶学生情操。大力提倡“勤奋、严谨、求实、创新”的学风，促进学生在德、智、体诸方面全面发展。

科研方面近年来有很大发展，一方面继续重视基础研究，另一方面大力开展应用研究，特别注意多学科、跨学科的研究。理科在注意发展创新的基础研究同时，大力加强应用学科和科技开发工作，促进科技成果的转化，努力开拓高新技术。文科增加了理论与现状方面的研究课题。“八五”期间，理科共承担研究项目680项，出版著作317部，在国内外学术刊物上发表论文8321篇，获各种奖项462项，包括第三世界科学院数学奖2项、联合国教科文组织颁发的科学家奖和青年科技奖贾乌德·侯赛因奖各1项、国家自然科学奖36项、国家发明奖7项、国家科技进步奖29项；文科共承担研究项目430项，出版著作2603部，在国内外学术刊物上发表论文7660篇，获各种奖660项，包括获得北京市首届社会科学优秀成果奖特等奖7项、一等奖11项。1978年以来，北京大学从国内聘请了400多位著名专家、学者担任兼职教授。1985年以后，在数学等15个学科设立了博士后科研流动站，到1996年9月已接受国内外360名获得博士学位的人员开展博士后研究。

国际交流与合作

北京大学开展了较为广泛的国际交流与合作，已与40个国家和地区的128所高等院校建立了校际联系，十多年来先后派出1650多名教师出国进修，短期出境参加国际会议、开展合作研究、进行讲学访问的教师已达5900多人次。

中国语言文学系

中文系早在京师大学堂时期就高有中国语言文学方面的教学机构。近一个世纪以来，众多蜚声海外的专家学者曾执教于此，文学方面的如鲁迅、周作人、胡适、陈独秀、吴虞、沈尹默、俞平伯、闻一多和朱自清以及游国恩、杨晦、吴组缃、林庚、浦江清、王摇、季镇淮诸先生，语言学方面的如刘师培、黄侃、刘复（半农）、钱玄同、沈兼士、罗常培、唐兰、魏建功、王力、朱德熙诸先生。

北京大学经济学院

北京大学经济学院的前身是经济学系、国际经济系。现有教师名，其中教授26人，副教授14人，讲师、助教30人。此外，还有11名教授离退休后被返聘回院从事教学与研究工作。

经济学院下设五个本科专业，八个硕士学位专业，六个博士学位专业，一个博士后流动站。

经济学专业：培养适应社会主义市场经济发展需要的经济工作者和理论工作者。

专业课程主要有：政治经济学、现代西方经济学、会计学、统计学、财政学、货币银行学、国际贸易、发展经济学、比较经济学、高等数学、计算机原理与应用、资本论、中国经济史、外国经济史、中国经济思想史、外国经济思想史等。

国际经济专业：培养德、智、体全面发展的、能够从事国际经济问题研究和涉外经济单位、跨国企业工作的专门人才。

国际金融专业：培养德、智、体全面发展的、毕业后能够从事国际金融问题研究及国际金融业务的专门人才。

保险学专业：培养德、智、体全面发展的、能够从事金融骸保险问题研究及实际工作的专门人才。

国际贸易专业：培养德、智体全面发展的、能够从事国际贸易问题研究及具有国际贸易业务的专门人才。

该院毕业生由于基础理论坚实，专业知识广博、适应能力较强，有后劲和开拓精神，因而深受社会用人单位欢迎。

国际关系学院

北京大学国际关系学院是中国综合大学中建立最早的国际关系学院，是我国培养国际问题和外交与涉外工作专门人才的重要基地。

国务院副总理兼外交部长钱其琛任名誉院长。

该院现拥有3个系：国际政治系、外交学与外事管理系、国际传播与文化交流学系以及4个专业和专业方向。设有5个硕士点，2个博士点，其中国际政治学博士点在全国高校中是独一无二的。

该院师资力量雄厚，在52名教师中大多数具有高级职称，其中教授16人（含博士生导师13人）、副教授18人，还有国内外著名学者担任名誉教授、客座教授和兼职教授。该院拥有丰富的中外文图书资料和供学生学习的电脑室和各种现代化设备。

目前，国际关系学院招收本科生有2个专业：国际政治专业和外交学与外事管理专业。

国际政治专业分3个专业方向：（1）国际政治专业方向。培养具有扎实的国际政治理论基础和广博的国际知识，能独立从事外事调研和科研工作的专门人才。主要课程：美国政治经济与外交、西欧政治经济与外交、当代亚洲、非洲、拉丁美洲政治经济与外交、世界环境与发展、军备控制与裁军等。（2）国际传播与文化交流专业方向。培养从事对外交流和国际公共关系工作的专门人才。主要课程：传播学概论、中西文化比较、中国社会与文化、美国社会与文化、西欧社会与文化、专业英语听说、专业英语精读、英文写作、翻译理论与实践等。（3）台港澳与世界事务专业方向。培养有关台港澳与世界事务方面的理论与实践工作的专业人才。主要课程：台港澳政治经济与文化、台湾政治与中美关系、香港政治与中英关系、台港澳法律；台港澳在大陆的三资企业及其管理、两岸关系与一国两制等。该专业前两年学习共同的专业基础课，第三常年根据需要和个人志愿，分三个专业方向培养。

餐交学与外事管理专业：培养具有综合性外交学与外事理论、涉外政治、经济、法律和国际文化方面的知识，具有外交和外事人员所具备的业务知识和技能，能适应各种涉外工作环境的专门人才。主要课程：外交学、外事管理理论、涉外秘书学、外事礼仪、公共关系、涉及外企业管理、经济外交、国际旅游、中国涉外法律概论等。

该院毕业生社会需求量大，一般被分配到国际问题研究机构、外事和涉外部门，高等院校和国家政府部门及企事业单位工作。

历史学系

本系是中国最早设立的史学教育科系，迄今已有90多年的历史。世界近代史为国家级重点学科。本系在全国率先设立了历史学博士后流动站并成为国家文科人才培养和科研基地。

光华管理学院

经济学家厉以宁教授任院长。学院师资力量雄厚、年龄结构合理，教师中有正教授13人，博士生导师11人，副教授15人。学院经常聘请海外的专家、学者来院讲学。

本科、硕士和博士研究生多层次教育，学院新的教学科研大楼将于1997年秋投入使用，大楼有一流的计算机实验室、图书资料室、教室和办公设施。

学院有五个本科专业，体现文理科交叉的特点，重视理论与实际结合。各专业还提供专业必修课和选修课，学生毕业后，可报考研究生继续深造，也可到政府经济管理部门、金融部门和各类企业工作。

各专业从我国市场经济和对外交往的实际出发，着重研究国际会计理论、实务与方法，培养具有会计、审计、理财等高级管理人才。

信息管理系

始创于1947年，是我国自己创办的最早的图书馆学情报学教育中心之一。已培养出一本科毕业生1600多人，研究生250多人，函授生5700多人。

现在，该系拥有知名的教授和信息管理专家。下设图书馆学、情报学、编辑学等3个本科专业，2个硕士生专业，1个博士生专业，以及信息资源管理，咨询学和文献学等相对独立的研究方向。
