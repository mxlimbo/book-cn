新疆大学

学校概况

新疆大学是隶属于新疆维吾尔自治区教育委员会。是我国目前仅有的两所少数民族综合性重点大学之一。坐落在新疆乌鲁木齐市。校舍建筑面积19.17万平方米。

1955年建校。

现有在校教师1184人，其中教授36人，副教授252人，讲师565人。

本科专业

汉语言文学、维吾尔语言文学、哈萨克语言文学、新闻学、历史学、马克思主义基础、思想政治教育、法学、汉语言、维吾尔语言、哈萨克语言、俄语、英语、数学、计算数学及应用软件、计算机及应用、物理学、电子学与信息系统、化学、生物学、自然地理学、水资源与环境、政治学、应用数学、应用物理、应用化学、国民经济管理学

授予硕士学位的学科、专业

中国现代文学、突厥语语言文学、中国近代史、中国地方史、辩证唯物主义与历史唯物主义、政治经济学、国际共产主义运动史、语言学基础、数学、无线电电子学、有机化学、动物学、自然地理学、物理化学、文艺学、应用数学、理论物理
