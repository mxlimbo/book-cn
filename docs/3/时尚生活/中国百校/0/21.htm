华东理工大学

学校概况

　　华东理工大学是国家教育委员会直属的全国重点大学之一。学校设在上海市梅垅路。校舍建筑面积32.42万平方米。

　　1952年建校，名为华东化工学院。1972年8月改名为上海化工学院。1980年恢复华东化工学院。1993年2月改名为华东理工大学。

　　现有专职教师1099人。其中教授98人，副教授377人，讲师453人。

国家重点学科

　　化学工程
