大庆石油学院

学校概况

　　是中国石油天然气总公司直属的全国重点大学。学院坐落在黑龙江省安达市，建筑面积26.17万平方米。1961年9月东北石油学院正式建院。1975年7月改名为大庆石油学院。

　　学院现有教师733人。其中教授56人，副教授215人，讲师268人。

本科专业设置

　　石油地质勘查、矿场、地球物理勘查、地球物理、采油工程、油藏工程、钻井工程、机械制造工艺与设备、化工设备与机械、矿业机械、计算机软件、生产过程自动化、石油加工工业与民用建筑、应用化学技术经济

授予硕士学位的专业

　　热能工程、化工过程 、机械工业自动化、有机化工、煤田·油气地质与勘探、应用地球物理、油气钻井工程、油气田开发工程、石油·天然气、机械工程、安全技术工程
