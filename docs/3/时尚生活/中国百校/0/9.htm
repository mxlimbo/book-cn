天津大学

学校概况

天津大学前身是北洋大学，成立于1895年。1951年北洋大学更名为天津大学。

天津大学是国家重点理工科大学。

现有教师1531人。其中教授181的，副教授631人，讲师583人。

国家重点专业

内燃机、测试讲师技术及仪器、光学仪器、化学工程、有机化工、系统工程
