西安交通大学

学校概况

坐落在唐兴庆宫南苑的西安交通大学，占地面积1980余亩，建筑面积67万平方米，布局合理对称，造型美观。校园内有花园草坪，假山亭台，并和喷泉雕塑相映影，展现出独具匠心的优美景观和育人环境。

西安大学是一所以工科为主，理工管文相结合的多科性大学，是国家教律 属的全国重点大学，是“七五”“八五”期间全国重点建设的大学之一。

西安交通大学，其前身是南洋公学，1896年创建于上海，1921年改称为交通大学，至今已有100年的悠久历史。1955年，国务院根据国家建设和工业布局的需要决定交通大学从上海迁往西安，1959年定名为西安交通大学。

在校教职工近5000人，全日制学生13000人，其中研究生3000人。

全校设有机械工程学院、电气工程学院、电子与信息工程学院、能源与动力工程学院、管理学院、材料科学与工程学院、建筑工程与力学学院、化学工程学院、理学院、人文社会科学学院、工程与科学研究院、研究生院和成人教育学院，43个系共覆盖53个专业，有49个专职研究所和1个国家级工程研究中心，9个国家重点实验室，78个实验室和为教学和科研服务的教学实验中心、计算机网络与信息中心、电教中心等；有6个校办工厂、2个出版社和1座以校友、著名科学家钱学森名字命名、造型别致、藏书180万册、总面积达3万平方米的现代化图书馆。

全校现有教师1800余名，其中教授、副教授800余名，博士导师140名，国家级专家14名、中国科学院院士3名，中国工程院院士2名；有71个学科（专业）可授予硕士学位、33个学科（专业）可授予博士学位，其中被国家列为重点学科的有11个，有9个博士后流动站。

在长期的办学实践中，西安交通大学形成了重视基础、严格要求、注重实践的优良传统。严谨的治学精神，稳定的教学秩序，保证了培养人才的质量。1989年，在建国以来首次举行的全国普通高校国家级优秀教学成果评选中，该校共获奖9项，其中特等奖2项，优秀奖7项，荣居全国高校之首。1993年，全国第二次优秀教学成果奖评选中，该校获奖数与清华、北大并列第一。

自1978年以来，全校完成鉴定的科研项目共计1096项，其中获国家级科技成果奖109项，省部级奖494项，是全国高校成果最多的学校之一。

目前该校已与美、英、法、德、加、日等20多个国家和地区的70多所大学建立了校际协作关系。近年来共派遣了800多名留学人员，并每年派遣近200名教师出国访问、讲学或进修。每年举办4到5次国际学术会议。平均每年约有30我个国家的1500余名外国专家、教授来校讲学、参观及参加学术会议。

建校及西迁以来，该校共培养了75000多名毕业生，其中杰出人才灿若星河。如上个世纪末的白毓昆、蔡鳄等，20年代的陆定一、邹韬奋、茅以升，邵力子、黄炎培等，30年代的饯学森、吴文俊、张光斗等，40年代的江泽民以及王安博士等，50年代的丁关根、胡兆森等，60年代的熊有伦、苏君红、陈惠波、屠规彰、李义怀等等。这些都是作为交大人引以自豪的，是广大师生心中的楷模。
