第三天

猎物

　　好的猎手会在适当的时间，适当的地点去等候它的猎物。

　　你是否在叹息为什么自己总是孤身一人，而你朋友却在周末通宵的狂欢。我知道你很优秀，而且也被大多数人夸奖，但电话就是没有在周末响起，是那里出了问题。好吧，一个人就一个人，这也不挺好吗。希望这是你真实的想法，否则就看下去吧...

　　你想要的爱情不会有人寄到你的家里或者送到你的手上，这很可笑吧...但你却总坐在那张要垮掉的椅子上，你真该换换那把椅子了。或许你该站起来，然后走出门去，看看街上的人流，你发现没有，你很孤独。但别担心，那些你认为快乐的人也曾经像你一样，只是他们可能运气好一点，最多是他们主动去找了。你为什么不呢，为什么还在叹息呢，没有人给你阻力，所以，找一个适当的时间，一个适当的地点去等候你的猎物吧...

　　猎手们都很明智，因为他们只去捕获自己能捕获的猎物。如果他们想去捕获一只饿了三天的老虎，那么他们的后果是什么？可能老虎真的能被他们捕获，但更多的是他们将从虎口里消失了。捕获爱情没有那么可怕，但你也得学习猎手，去捕获那些可以捕获的猎物，除非你想冒险，否则别碰那只老虎...

　　按下CD唱机的按钮，来段音乐，回答我为你准备的问题：(别犹豫，把真实的想法写下来。)

1.你认为你怎么样？
2.你自信吗？
3.你会主动同陌生人说话吗？
4.你会主动表达你的爱吗？
5.你能忍受孤独吗？
6.如果你失恋了，你能哭三天，然后就忘了他吗？
7.你怎样看待爱情的？
8.你害怕承诺吗？
9.你很冲动吗？
10.你有什么缺点？
11.你的人生目标是什么？
12.如果别人拒绝了你，你会怎么想？
13.你是否愿意独身？
14.你有好的承受能力吗？
15.你的对爱情的要求是什么？
16.你讨厌什么？
17.你害怕什么？
18.约会对你来说意味着什么？
19.你记忆力好不好？
20.你觉得自己应该改变什么？

　　上面的20个问题用来测试你到底有什么样的爱情思维模式，当然它是我准备问题的一部分，但足以说明你对爱情的期望。这里没有标准答案给你来判断什么，但以上问题能够帮助你更好的阅读我以后的文章。这些你今天才发现的爱情思维能引导你更审慎的去看待以后每一次的相遇及相识，它将告诉你是否放弃或者继续，在你头脑里，这些你回答过的问题会在适当的时侯指示你，你那里出错了，它将纠正你的错误，让你回到正常的爱的历程上来。好，今天就到这里...
