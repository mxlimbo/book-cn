你相信网络爱情吗？

　　你相信网络爱情吗？你期望经历一次网络爱情吗？你有过痛苦的回忆或者甜蜜吗？

标准不标准

　　网络爱情的童话使人向往属于自己的那一次相遇，然而，一种怀疑始终让我们担心。网络虽然是个开放的空间，但你把思维限定在狭小的范围内，始终无法改变自己在现实生活中对爱的要求。也就是说,你订立了若干标准，然后去找符合标准的人。网络带给你容易相识的朋友，你订立的标准也同样跳了出来。

需要不需要

　　网络爱情很特别,你能用心去导演这出“戏”。你是编剧、导演、又是演员，你所构筑的情景都是你曾经梦寐以求的。网络让人投入，因为幻想是无限的。你用美好的话语来建造这个“海市蜃楼”。

　　当你感到孤独时，网络成了你的感情寄托。可你很容易遗忘。当你春风得意时，你不会想起网上的那一位。网络爱情让双方在孤独的时侯才产生彼此需要的意愿，并非在任何情况下双方都能保持热情。

真实不真实

　　媒体开始关注网络爱情故事，展现的主角几乎都是“淑女”和“君子”模式。当然，这种模式的确存在，只是我们容易看到事物好的方面，容易被假象所蒙蔽。其实你能遇到任何人，包括你不希望的那一类。

　　事实上，在网上你有更多机会去寻找，去交流，去脱网见面。如果你不断推翻自己曾有的感情，那么必将为自己的行为产生矛盾的心理负担，换言之，你一面高谈如何注重精神交流，一面在看到对方的照片后，就和她（他）说再见。

　　网络爱情已被媒体谈得太多，而人们对它的认识却变得越来越混乱。大家各持己见，但有一点是相同的，每个人都希望网络爱情能降临到自己头上。当然，我们期望网的那一端是位“淑女”或“君子”。

　　网络爱情就是这样让人无法琢磨，而我们也不必为它的真实性辩论。一位台湾网络作家说过，实着虚之，虚着实之。这也许是网络爱情最好的映照。
