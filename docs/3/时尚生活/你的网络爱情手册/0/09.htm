第九天

爱的旅程

　　激情并不能把你们带到最终的目的地，你需要的是冷静。面对你们共同建立的良好的关系，你是否该明白当一切摆在眼前的时侯，什么才是你需要的，而什么是你想放弃的。我没有试图影响你的选择，只是认为在平静中的选择要比虚幻情景中作出的选择要真实的多。

　　网络建立的幻想空间能淹没掉对方的任何缺点，你一味思考的是对方是如何的优秀。网络提供的思考是一种建立在“已经发生故事”的承接模式上的，我的意思是，当你看到一个网络爱情故事后，如果这个故事包含了你也期望的结果，那么这个故事就会印在你脑海中，当你真的遇到这样的事情时，这个故事就会跳出来，而你把你的相遇贴上了“已发生故事”的标签，这一切当然是潜意识在起作用。这种意识会给你在网络上带来帮助，因为你将变得更懂得你和她什么才是良好的沟通，但如果换到网下，现实将揭露一切本已摇摇欲坠的幻想。

　　从一开始你需要建立和她健康的关系，这样，无论在任何情况下，你都能从容面对。也就是说，无论对方是否是你期望的人，你都将已健康的心态去面对眼前发生的一切。请记住“不要给你和她之间提出太多要求，保持平常的心情，随时面对你和她出现的转机。”

　　网上能找到真正的爱情，就如同你在网下去约会或者在单身俱乐部见面一样。但这一切没有绝对的保证，我没有告诉你，网上一定能找到真正的爱情。这其中包含的意思是你对网络爱情的期望和付出，以及你是否能以正常的心态面对你所遇到的一切，从某种意义上来说，网络是检验你真正感情的地方。

　　很感谢你能和我共同经历这一次爱情旅程。我也深知这里没有包含爱的全部，但我知道，当爱情降临到每一个人身上的时侯，我们能得到它所带来的所有的喜悦与幸福。

　　相遇就是一种缘分，也是一种注定，冥冥中的安排会眷顾每一个肯付出的人。无论你和她在那里相遇，也无论你是否真的想得到这份感情，刻意的愿望并不会因此而变得真实。如果你现在感到开心，那么我希望你能享受眼前的一切，细细体会来自于爱的甜蜜与温暖。永远的祝福你...并且让我们共同继续这段爱的旅程...
