网络爱情的优势及劣势

　　爱情会让人迷失了方向，网络爱情也不例外，让我们一起来看看网络爱情的优势及劣势：

优势

1.即时，你可以在任何时侯去寻觅网络爱情，比如你看完我这篇文章后，马上就可以行动；

2.方便，你只需要一台上网的电脑和一份热情就可以了；

3.节约费用，你是否担心自己是否有能力为你和她的恋情“付费”，但网络让你不用担心；

4.不会发生意外的事情，与你感兴趣的女生会面时，你是否担心发生意外的事情呢，比如把汤洒在身上了；

5.让你更好的洞察她，网络的交流是靠文字的传递，那么你能从她的文字中冷静的获得一些对她的认识；

劣势

1.对方欺骗你，因为你无法真的知道她是谁，所以很容易陷入她所设的陷阱；

2.让人容易冲动，因为网络只体现一个人的思想境地，而非加上她现实生活的行为和态度。但你却容易被“言词”所打动；

3.逃避责任，对方很容易就可以逃避，即使昨天还在说她爱你，今天就和你再见了；

　　我告诉你网络爱情的优点及缺点，不是让你相信或者怀疑网络爱情，只是当你在遇到上面所提及的问题的时侯，能冷静的发挥及接受它。无论如何这些问题也存在于你的现实生活周围，所以无需担心，重要的是你要真诚的对待你能遇到的每一个人。
