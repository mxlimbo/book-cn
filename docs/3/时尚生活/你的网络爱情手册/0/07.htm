第七天

别让你的女孩逃走

　　我知道你要说什么，但要先想想说出来的后果...

　　我们都很容易犯错误，特别是在网络里。感情变得容易付出，而且这种感情是借由某种忽然的冲动产生的，比如看了一部爱情的电影或者听一首爱情的歌曲。实际的经验告诉我们，每当要说出那句话的时侯，你的女孩开始注意到了，你能感觉到她那种逃避。等到你真的说出来的时侯，在网络那一头，你的女孩却逃走了...

　　是什么地方出了问题呢，你和她之间的某种默契没有产生，而且你低估了网上女孩的理性度。过去你认为女孩是感性的，事实上许多故事告诉我们，在网上女孩更加的谨慎，她能和你高兴的聊天，但每当你想进一步发展的时侯，你会发现你们的友谊也结束了。当然，一些故事也告诉我们，当一种默契从你们身上产生的时侯，这种你所期待的网络爱情就产生了。一切的发生不总是好的，所以你需要做好准备迎接来临的成功和失败。

　　我们有太多关于爱情的经验，那是来自过来人的金科玉律。事实上照本宣科无法让你找到真正的爱情，爱情来自于你的付出，而不是一味的索取。网络相对来说比现实更容易让人付出，不光是感情还有其它很多方面。这种网络具备的特质，能让我们构想属于自己的感情方式，是活泼，还是冷静，是浪漫，还是现实。不一而足的都说明，你是网络爱情这场戏的导演，同样你也是演员，从一开始你就要想好，怎样演好这出戏。

　　越是费心追逐一样东西，它越会离你远去。这就好像猎手追逐一只野兔，他越追，野兔跑的越快。如果猎手停下来，野兔有时会停下来看看猎手为什么不追了。有太多想找爱情的人，都急于抓住这段他以为难得的感情(事实上有许多这类人会在未来后悔)。那么，你已经明白，你越想得到那分爱情，而那分爱情却和你失之交臂。在《做你想做得事》这本书中有这样一句话：“每当我们迫不及待地想达成一件事情时，会在周围形成一股看不见的能量，阻隔了你所期待的东西”。请记住这句话...

　　如何知道网络那头的女孩对你感兴趣呢：

　　1.通过网络的交流频繁；
　　2.她向你索要照片；
　　3.她给你说她的故事；
　　4.当下雪的时侯，她来信要你小心身体；
　　5.你们开始通过电话交流了；
　　6.她要求与你脱网见面；
　　7.如果你不写信给她，她会来询问为什么；
　　8.她在信中与你探讨爱情的问题；
　　9.她向你述说她的烦恼，希望你开导她；
　　10.她通过邮局给你寄小礼物了。

　　还有很多的迹象表明她对你感兴趣了，而且你需要细心观察，别过度敏感，也别轻易放弃机会。

　　如何增加你们的网络感情呢：

　　1.告诉她你的过去恋情，并告诉她你很怀念；
　　2.每天都写信给她；
　　3.打电话给她；
　　4.告诉她，你生病了；
　　5.告诉她一个你的秘密，让你们共同拥有一个秘密；
　　6.主动的提出寄照片给她；
　　7.通过邮局寄小礼物给她；
　　8.不时的给她祝福；
　　9.告诉他你曾经犯过的可笑错误；
　　10.突然来到她居住的城市，然后打电话告诉她你马上要走了。

　　我知道你要说什么了...是什么?
