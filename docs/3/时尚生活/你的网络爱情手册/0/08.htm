第八天

转移爱情

　　牵起她的手，从网上走下来......

　　你们的历程告诉你们的信息是，当爱情从一开始就建立在幻想中的时侯，它是不牢靠的。除非你想永远这样，否则你们必须选择的是从网上走下来。这样对于你的选择，你的决定以及你曾经做过的承诺在那一刻将变的更真实。当你真的用眼睛看见她的时侯，一种熟悉到陌生，一种陌生又到熟悉的感觉油然而生。这将是你和她能否继续下去的重要的时刻。

　　你会把约会从网上搬到网下吗，有点耽心，还是有点害怕。爱的冲动让你在幻想和真实之间作着选择。我知道你想和她发展有前途的关系，在网络上你没有耽心她会是什么样子，只是感觉那是一段愉快的经历。但如果把爱情转移到网下，你面对的真实能告诉你过去你曾经的付出是值得的吗，还是发现这是一场大笑话。感情在网上和网下的变化，所带来的是对你心最深处对爱情刻划的印证。

　　你是否一直在怀疑网络爱情的可信性，但是你又被许多媒体上的文章所左右，它们告诉你一个个网络爱情故事，与此同时，你也幻想自己也能深入其中。不过你可能还是无法抓住你所构想的网络爱情的情景，因为你所能遇到及相识那一位，你们的发展也是普通的让人乏味。你开始怀疑那些媒体上的文章说的是真实的吗？我想你开始阻止你自己去投入了。所以，你无需去下定决心，也无需为了网络爱情特别培养自己的热情，这样你就少些压力。

　　每当你紧张一件事情的时侯，你就无法获知在这件事情周围的信息，你把自己摆在“非生即死”的境地，好吧，你需要放轻松的面对它，不管你是有意识还是无意识去寻找网络爱情，你只需要知道，网络是另一个相识的机会，仅此而已。

　　你和她终于要跨出重的那一步了，是的，你需要脱离网络来确定你和她的关系。当然，你和她都在担心对方是否是自己期盼的那一位。如同我过去说过的，你的紧张来自于你的过高要求，也就是说如果她复合你的要求，你就愿意和她进一步发展，否则就和她说声再见，这种想法会让你再次陷入“非生即死”的境地。所以，你永远不要把关系定位在更高的要求上，只需要明白你和她的见面是一次友谊的聚会。

　　还有提醒你的是，你在和她见面后，无需为过去你的言论后悔，比如说错了或者不该说。我再次告诉你一个真理，如果与你见面的女孩是一位“漂亮”女孩时，你会对你过去的网上言词感到庆幸，但如果你见面的女孩是一位长相平平或者还有某种“缺陷”的时侯，你肯定会为过去网上的言词大感后悔，而那份热情也消失了。你不要反驳我，这是千真万确的事实。我告诉你这一切，是希望你能在见面之前就确定你自己的要求，用纸写下你的对她的要求，然后确定那些可以妥协，那些必须坚持。

　　当然，一切也可能顺利的发展，无论如何，你需要保持平常的心情，而不要自己给自己出难题，而且你也不用担心对方会缠着你，你很容易就可以走掉。

　　下面是第一次见面的提议：

1.在公共场合；
2.在白天见面；
3.为了安全，可以邀朋友同去。

　　最后的忠告

1.期望值不要太高，除非你不打算和她见面
2.警惕过于亲密的举动
3.不要在还不了解的时侯就告诉她你的一切
4.谨慎公布出你的信息
5.别说的太多，给她留点幻想的空间
6.如果对方拒绝你的要求，要知难而退
