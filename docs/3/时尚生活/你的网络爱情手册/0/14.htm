网上，你的爱情飘呀飘

相遇是注定的

　　相遇是偶然的结果，那种随机的抽签，让你的心悬在那里。你无法了解面对你的将会是什么样的情形。或许一切正如你所料到的那样，或许一切都是在行进中发现。相遇是让我们感到人生不可预见性的最好例证，无论你做出任何努力，有些人总会和你擦肩而过，你可能在担心，你所最期望的人可能就在一群人当中。就因为此，你不断去尝试相遇，不断去尝试那种相遇的激动，相识的甜蜜。你也不断在检验自己的期望和对方的暗示，你所作出的种种努力是为了创造某种交汇的机会，而你过去立下的宏愿在这一刻变的真实而具体。

　　网络所为你建立的相遇的模式紧紧在于你是否能主动而适时的发出你的真诚的愿望给对方，而接下来你所能做的就是等待。或许你会锲而不舍的去与还没有对你发出回应的她，给出更多的信号，这就是你能在网络上所做到的。作为你来说，一次刻骨铭心的相遇在你脑海中已经存在了相当长的时间，只是你把它沉积在心的最底层，你需要有人能诱发出你对爱真实的渴望。你一直在担心自己是否能让自己的爱给对方安全和幸福，你需要有人来开启你那扇大门。它现在还关闭的很牢，你需要得到的是一次相遇。这样，你从没有过的幸福将遍布你的全身，你能细心体会眼前你所能感觉到的一切。

　　你只是坐在电脑面前，但你却能得到一切爱所带来的喜悦。这就是网络的魔力，一个精灵在跳跃前行，它为你斩断荆棘，并为你的担心给予更多的鼓励和暗示。它一直告诉你，这一切对于你来说是千真万确的，而不会因为某些人的故事而让你停滞不前。这是一种上帝开的玩笑，上帝没有时间去仔细为你的爱指明方向，它只是说，你有机会了，其它的你得自己去把握。时间还在流动，心情在这一刻得到宁静，一幅美景展现在眼前，远远的一个影像若隐若现。你还在盯着电脑吗？闭上你的眼睛，然后在睁开，你会发现，一次偶然的相遇......

你们的今天

　　你们终于相遇了，是的，没有错过。那么，你们是否在建立起你们彼此的对爱的追寻的一条路。在这一条路上，你们是否会携手前行，或许一切只是相遇，没有更多让你们感到希望的东西存在。她总会无所谓的在偶然的时刻注视着你，她不是在给你希望。而只是说，当你们在网络上遇到的时候，应该打声招呼。你是否会自然不带杂质的去回应呢？什么占满了你的头脑，是爱吗？那种爱好像是虚无缥缈的，是的，你没有看见她，你只能去想象那种爱。对于网络来说，爱的体验在你们.田.的交谈中，或许是在一种符号里。每一个细微之处都体现了你对爱的渴望，你不断去检验自己的想法，是否开始寝食难安了，是否该向她说出你的心里话了。你是否想到，当每一次太阳升起的时候，爱会在那方停留呢？在你心里吗？在你勇敢的表达爱的情绪里吗？或者是在她对你的良久注视里？你捕捉每一个细节，让后动用全部的智慧去破译那道难题。你试图去解释你现在的心情和她对你的所有的暗示。你开始陷入困恼当中，你无法清醒的告诫自己，网络爱情真的对你意味着什么。

　　等待开始成为你一种习惯的生活。你等待着她上网，然后告诉她，你很想她快一点出现，是的，你的所有激情被她诱发。你无法去扑灭已经雄雄燃起的火焰。你不愿意去自编自演一出戏剧，你需要她也是剧中的主角。你开始怕她逃跑了，你只能小心翼翼的去以玩笑的口吻来表达你对她所有的喜欢和爱护。你知道这种方式能让她知道你的心情，而又不会因为你的冒然而吓住。你想探听她的想法，或许一切正如你所料到的。是的，我们在犹豫中错过了许多的机会，同时我们又在自己的漫无边际的幻想中迷失了自己。眼前一切对于你来说，可能是从没有过的那种让人感到神奇，感到喜悦，感到谜题一般的悬疑。这是一种吸引，一种你们所创造的吸引，你看到了吗？隔着墙你就能想象，隔着网络你就开始真的让自己去幻想了。

　　这是一个值得纪念的日子，你们没有在人海中错过。是的，你的爱让你知道面对一些难以解决而且复杂问题的态度。你是否会歇斯底里的去怀疑眼前的一切，然后快速的恢复到难以置信的平静继续你们的交往。事情有时候就是那么的让人感到奇观。无论你在什么时刻(田涛)去面对网络另一头的她，你都会自然的过渡到让你们彼此间产生某种幻想和希望的道路上来。然后顺着这条道路继续你们早已经开始的那种默契之间的感情的交流。电脑屏幕还闪烁着它的光芒，你的心却因为这一次你们的相遇，进而的相识感到甜蜜，那种笑藏在你心中好久了，现在你可以把它拿出来，对她说：我爱你。

上帝的玩笑

　　你会站在街角的一个电话亭前等着她出现吗？你是否在她没有出现之前，去想象着她的样子和当你如果失望以后的对策呢？然后你又想到你们曾经在网络上相识的那种让人感到幸福的所有细节。你想到好多值得你们记得事情，但现在你却在做一件会让你们感到永远遗憾的事情----见面。上帝总会给我们开个玩笑，你开始担心这一次上帝会放过你吗？什么一直在你头脑中盘旋，是紧张，担心，希望，祈求，还是脑袋中一片空白。突然，你的肩被人拍了拍，你吓了一跳。她来了吗......

　　当你感到是时侯了，你会想见她。因为你想这样才能衍变成为一种真正意义上的爱，更重要的是，你能吻到她。你在头脑中不停想象着那种见面的情形，你在深入的去找寻你们的契合点。有时候，当你第一次遇到她，你就会想见到她。时间并不是影响你是否想见她的促进器，你会因为某一刻的感触让你容易生成一种见面的渴望。有时候，那种信念是很强烈的。无论她在那里，你都能在很快的时间出现在她的面前，这就是爱的魔力。当加了一点神秘的时候，爱不单纯是爱了，它含包含了希望。当一个人充满希望的时候，爱对于他来说更显得清馨透明，不留杂质。你还在继续你的想象吗？或许你该冷静想想见面对于你们来说将意味着什么，上帝不会永远把好运赐予你。其实，你可以把握自己的命运，但在见到她之前，你开始祈求上帝了。你相信冥冥中的力量会改变某种情形，你开始相信魔法师的力量，你要找到他，你对他说，你想要什么样的见面，什么样的她。

　　我能帮你什么吗？蒙上你的双眼然后在她面前慢慢打开吗？或许你该偷偷躲在某一个角落里去注视着她，然后决定你是否该出现在她面前。当然，你可以喝点酒，然后以一种壮志未酬的情怀出现在她面前，然后告诉她，你看，一切不都不错嘛。喔，你累了吗？我知道你累了。一次见面让你调动了所有的想象力和安慰的话语，你给自己制造难题，然后再自己解决她。真的，我开始担心你们的见面了。如果你花掉许多时间去想象你们的见面，而不是感到见面对于你们来说只是一次真实的接触，并不因为见*面的后果会替代你们曾经所付出的所有感情。一切都是自然发生的，没有刻意的安排去让你对自己的世俗感到难过和悲哀。你无须为她的真实感到难过，一切本来就是上帝为你指明的方向，他最先告诉你直走，然后到现在你该转弯了。

　　你的肩很少有人拍的，你回过头去，发现一张苍老的脸，你吓了一跳。"请问，这个地方怎么走"，这位老人拿出了一张写有文字的纸片放到你的眼前。你看，上帝为什么给我开了一个玩笑......

结尾

　　头脑之间传递着某种讯息，它指示着你的行动方向。你该在短时间内做出某种判断。没有人真正能告诉你，你该怎么去做，只有你自己能掌握你的方向。你或许该告诉她，或许该静静的呆在那儿，你不能因为别人而改变你的想法。爱对于你来说，是你这一生用尽全力该找寻的，无论她在那里......
