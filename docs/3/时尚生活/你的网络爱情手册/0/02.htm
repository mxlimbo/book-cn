第二天

成为你自己

　　成为你自己，这样才能遇到真正的爱情。在这一点上，怀疑及犹豫占据了大多数人的思维空间。我们都在怀疑对方是否真的喜欢自己，还有他(她)到底是不是你最终要找的那一位。在能触摸的生活中，这些问题都令你迷惑，换到网络上，又该如何面对呢？

　　有太多的问题需要我们去解释清楚，在这其中有些问题必须要跟随感觉去做才能知道它的答案。冥冥中的指向会让你寻获你愿意得到的爱情。要相信自己，你一定会得到...

　　网上，我们都相聚在一起，彼此更融洽的的交谈。对待生活的态度我们能更理性的看待它，无论是什么问题，通过网络的缓冲都将被化解。网络的这种生活态度能让我们更加真切的投入自己的感情，虽然你可能无法看到和你对话的那一个人，但你能感受到自己通过真心付出而得到回报的那种你不曾有过的激动。笑也好，泪也好，你能不加杂质的细心体会。

　　每当你夜深人静坐在电脑面前的时侯，你是否感觉电脑变得是那么的柔和，在它背后有无数的人在那里等着你，他们有的会成为你的朋友，上司，恋人...看到没有，这是一个丰富的世界。

　　可能你从未去体会网络所带来的改变，这种改变不是让你升级硬件，下载软件，这种改变是：你的生命历程从网络那里变得平静，安祥以及一种想有人为你在网络停留的那种冲动...它的确是你真实的感情，你不要被习惯的生活所蒙蔽了双眼...

　　有一位男孩因为无法确定是否喜欢一位女孩而犹豫不决，他总期待下一位一定更好，我们都这样想。事实上我们根本就不知道自己到底要什么。如果拿张白纸给那位男孩，要他把对爱的要求写出来。你看着吧，他一定会很快动起笔来，但他会逐渐发现"我到底要什么呢..."，白纸一片空白...当然，我知道你一定能把那张白纸写的满满的，你看到没有，你的爱情有那么多的条件，虽然条件是未来舒适生活(只是一种可能)的保证，但事实上爱情是不需要条件的。我们既想要爱情又想要物质的丰富，这是一个可笑的你却习以为常的生活态度。

　　要想成为你自己，就要从今天开始想这样一个问题：我到底要什么？不管是在网上还是网下，你必须去想清楚。我不是要你抛开条件去寻觅爱情，我知道那根本不可能也没有必要，你只是要理性的看待你的期望，而不是以模糊不清的自己来面对未来的相遇。可能就在明天，你就能遇到他(她)了...因为你想好了，那么还担心什么呢?

　　我们所期待的相遇是偶然的，因为那样才会被认为是"浪漫"。网络能给你偶然的相遇吗？一切都在自己的掌握中，除非你真的愿意相遇，除非你真的确信你能做到，否则远离网络爱情的诱惑。当你开始期望的时侯，你才有动力去做每一件事情，当你相信自己的时侯，即使在遇到挫折的时侯还能勇往直前，我想说的是：你可以去怀疑，但在开始期望与相信自己的时侯就别在回头了...
