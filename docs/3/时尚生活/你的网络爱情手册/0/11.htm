我们见面吧

　　ANNIE说："既然两个人也互相了解了，不如见上一面，或许感觉会更好"。我问ANNIE的时候，她还从来没有见过网友，不过她并不介意去见。只是需要确定那种在网络上的感觉是否真实。

　　见面是两个在网络上相识的人，当他们成就了某段感情时共同需要去面对的问题。无论彼此出于什么原因，见面去揭示真实前，担心总会伴随着几个不眠之夜。小石头告诉我，她见网友前一天竟然没有睡着，因为她们交往了一年，很担心见面破坏她们之间的关系。不过最后，她们彼此都很满意对方，现在还是要好的朋友。为什么见面对于网络上认识的人来说是件满难的事情，特别是交往的双方涉及到感情的时候。

　　笔者感觉见面前的担忧是因为双方当事人在网络上交往时产生过多的幻想而留下的后遗症。这种情况发生在男生和女生之间，那么见面会感觉更为犹豫。网络上建立感情不需要靠外表来帮衬，需要的只是善解人意，风趣幽默的文字。凭借人类的幻想本质，一切所呈现出来的会是我们所追寻的美丽图景。当我们都掉入那个陷阱，享受到了幸福。只有当要从陷阱里爬出来时，才会发现一切只是幻象。把感情寄托于网络，所带来的会是对自己和自己感情的考验。

　　见面是必须的吗？我们是否可以选择一直和对方这样的交往下去呢？两个人在网络上遇到，然后随着交往的增多，当感情提高到一个新的阶段时。见面成了可以让她们继续交往的惟一出路。我的意思是说，精神交流所能维持的时间是有一个限度的。无论你们在网络上交往的情形如何。见面将是双方都挥之不去的结，越是感情深，越是会想见面。人类对于感情是重视实质的接触，而不是靠幻想和文字过日子。如果你反对我的话，那么我会告诉你，当你遇到那个人时，你一定会想见面的。那个人是谁？你说呢。

　　笔者曾经看过一个娱乐节目，其中的一个环节是让一对在网络上认识的男女第一次在电视上见面。她们都带着面具就这样相对而坐。当时，笔者心里也相当紧张，其实不为别的，而是怕那个女生太漂亮了，我会感叹老天不公呀，呵呵。笔者听到主持人在到数，5，4，3，2，1，面具轻缓的摘了下来。或许是电视的原因，她们彼此感觉好像都还不错。主持人要那位女生从黄玫瑰和红玫瑰中选择一朵以确定是否还和这位男生继续交往下去。女生拿了红玫瑰递给了男生。不过笔者到是感觉这段感情也不会维持太长时间。因为笔者注意到了那个男生的眼神，那是一种略微失望而无可奈何的表情。

　　见面真的是一件很难处理的事情吗？到底是什么让我们去担心她？有些人担心是外表，有些担心来到网下是否还聊的来，还有些人担心那个人的品质是否很差。有很多原因会让那份担心在见面前出现。如果有人告诉你，他并不担心这一次的见面，那么可以肯定，他撒谎了。许多人对于网络和现实的差别总是估计不够，一面在网络上轻易的释放感情，另一面又在担心对方是否正如自己希望的那样。我们会相信在网络上交往的日子是真实而又开心的，同时我们也相信自己的想法是怎么样的。人的幻想多数时候是帮了我们的，但在网络上幻想所得到的结果却是令自己相当不好的一次经历。害怕在于自己的变化如此之块，昨天还在网络上说爱，今天却因为揭示了真实而say bye。其实见面后，分开是满自然的，如果彼此感觉不好，以后在网络上遇到肯定不会在答理对方了，有的更绝，直接把对方删掉。

　　在网络不算普及的头几年，见面会是一件很慎重的事情。慎重并不是因为担心失望，而是觉得该如何去发展在网络上建立起来的友谊，这种想法是很单纯的。而现时，见面不在让彼此感到慎重。有可能在聊一次就提出见面，而当事人对这类要求多数都会答应。因为这样的随意，见面成了寻求满足自己愿望的机会。有些人不断的见面，然后不断的放弃。这种循环开始让见面变得不在成为网络上值得褒讲的话题了。见面现在成了一件很容易的事情，没有人再很看重它。

　　见面前，请参考下面的建议：

　　1.确定你的希望和你的承受能力；

　　无论处于任何的激动当中，你都该冷静自己而去思考自己的希望是什么，是想得到爱情，还是顺其自然的发展。同时，你也要考虑自己对网络的认识是否成熟，以及当一切无法挽回的时候，是否还能重新站立起来。希望和承受能力对于网络上的你来说都是相当关键的，因为只有你明晰了这些状况你才能在遇到难题和挫折时马上逃离出来。当然，这一切不会很具体的出现在你的脑袋里，只是随着事态的发展，你应该学会如何去面对和处理，有时候跟随自己的感觉去做，很容易走到绝路。更多的时候，你该学会相信自己，无论最后会是什么，你都应当相信你们彼此的交往是真实而值得记忆的。

　　2.携同你的朋友们一起去见面，万一感觉不好，也可以有人帮你；

　　你想增加彼此独处的时间，但这不应该在第一次见面的时候。第一次意味着会发生许多变故，而有一些是你无法处理来的。网络建立起你们彼此的信任，感动，幸福，但回到网下，你们所建立起来的这些感情基础将在网下得到改变。这种改变来自於你对于你自己的要求，网上那些要求是虚幻的，因为它被幻想所迷惑。来到网下，是你清醒的时候，你重拾了要求。虽然你很相信你们未来的发展，但变数总在里面。在憧憬的同时，你应该学会如何开始处理你们的第一次见面。告诉对方你将会和朋友们一起去，不过为了让对方放心，你该选择在一个人比较多的地方。当然，你也可以告诉对方，希望她们也来更多的朋友。

　　3.晚上见面，这样因为这时的人是比较浪漫和包容的。不过也要考虑安全哟；

　　我的朋友在后悔，因为他的女朋友是他在晚上遇到的，他告诉我，那一夜，她是那么的迷人。但到了现在，我的朋友又告诉我，晚上让他失去了辨识的方向，原来夜晚欺骗了他。我不是来告诉你，用夜晚来伪装自己，只是希望别在第一次就让你们the end。夜晚是让人浪漫的时刻，人的感情此时最容易释放和感动。同时，夜晚也是人心最为真实的时候，而你也不会担心路人会看清楚你们的第一次见面。夜晚的确有相当多的优点，唯一的需要注意的是安全，选择人多的地方，这总是避免出现问题的关键。

　　4.最好在见面前交换照片，这样可以心中有数；

　　在见面之前交换照片是比较稳妥的方法，这可以减少因为过度幻想而带来的烦恼。虽然照片无法显示一个人各种品质，但却可以从照片了解对方是否正如你希望的那样。交换照片并不是一种很俗的方法，那种完全不知情的见面是"危险"的。如果你愿意去冒险，那么你可以不提出交换照片的要求。不过，提出这个要求希望你应该平常的心态去面对，无论对方是否同意，你都应该感谢对方对于你的信任和好感。

　　5.见面只是很普通的面对面交流，而不是男女约会。

　　你为什么要把每一次的见面都想象为男女的约会呢？是否这样会增添更多的浪漫呢。或者你们已经在网络上建立起某种感情基础，见面无非是继续你们的感情和让感情更为真实。无论你们在网络是如何去做，去说的，在见面的时候，你应该相信这一次仅仅是一次face to face的交流。你要学会很自然的去看待它，任何的心情波动在见面前都是不恰当的。在网络上，你应该最想寻找的是友谊，而不是爱情。当你把友谊排在爱情之前时，到最后你往往会得到爱情。

　　最后的建议是：网下才是你寻觅生活乐趣的地方，一切都是可以看得见，摸得着的，这样还担心什么呢？

　　相遇是你们的福气，好好珍惜吧......
