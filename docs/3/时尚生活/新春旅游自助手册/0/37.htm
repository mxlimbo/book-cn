如何申办旅游护照


(2000-12-18)

　　护照是旅游者在国外的身份证明。以前，中国公民出国旅游，旅游者必须先到当地公安局出入境管理部门申请护照，需要提供的材料多，手续也较为繁琐。但现在情况有了很大改善。如果你到中旅、国旅、中青旅等三家较大的、具有出国旅游经营权的旅行社参团报名时，你可以在该旅行社直接申办护照，市公安局出入境管理处为方便人们办理护照，在这三家旅行社派驻一名民警审核办照。具体做法是，到旅行社接待中心领取“中国公民出境申请审批表”，填妥后，由本人到档案所在单位的党委、人事或保卫部门政审盖章（档案在“人才交流中心”的，由该“中心”盖章），同时注明“同意自费旅游”并由负责人签字，贴一张二寸照片，然后携带户口本、身份证原件，以及由旅行社开具的全额团费发票，就可以直接办理护照了。
　　
　　办理护照时间较长，一般从受理日起15个工作日后才能拿到护照。所以，若您没有护照，报名参团应至少提前45天，特别是准备在元旦、春节期间出国旅游的客人，更需提前报名，以免因名额已满而使出游计划落空。
