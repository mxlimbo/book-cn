冬季购物到巴黎！

(2000-11-30)

　　冬季的巴黎，不只是艾菲尔铁塔、凯旋门，更吸引人的莫过于每逢冬季的商品大减价所带来的购物狂潮。置身其中，在领略千年皇城、浪漫花都那独特情调的同时，玩一把购物的疯狂与心跳，亦是漫漫冬季里的一次潇洒。

　　巴黎的时装，足可与其绘画、雕塑、音乐一样列入艺术领域，同样享有盛名的首饰、皮具、香水犹如通行全球的时尚标志。凭心而论，巴黎的时装比起香港来并未便宜很多，但却胜在款式新、选择多，而且买够1200法郎便可以退税，当可令您大省一笔。

　　购物地点除了著名的香榭丽舍大道林立的Bally、G.Armani、Chanel、Saint Laurent、Dior、Givenchy等品牌店，尚有rue duRivol大道这个商店密集之地和据此步行10分钟便可到达的Forum des地下商场，260多家商店足够您逛一气儿了。

　　另外，适合“一气买个够”的人，不妨前往春天百货公司或老佛爷百货公司；愿意“购物当享受”的人，亦可前往拱廊购物街去体味“巴黎性格”；如果您是收藏爱好者，您一定会被摆满九桥与叙利桥之间的塞纳河岸边的旧货摊和克利昂库尔门、蒙特勒伊门、沃韦门、星期六、星期日的跳蚤市场吸引，您可以充分显示你的鉴赏与谈判能力，满载而归。

　　时逢冬季，何不前往巴黎过一把购物瘾！
