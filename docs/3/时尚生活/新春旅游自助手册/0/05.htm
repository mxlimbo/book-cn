滑雪自助游

2000-12-18 13:46:18

　　下班后，从北京站乘T17次（18：20）或T71次（17：00）特快列车出发，一路笑语欢歌奔向北国冰城哈尔滨。

　　经过一夜休息到达哈尔滨，稍待休息伸展肢体后，换乘T409次（7：50）子弹头特快列车到达亚布力（10：50），再换乘雪原bus到达风车山庄度假饭店，沿途欣赏亚布力滑雪中心（风车山庄）的外景，闻名遐迩的风车网阵以及东北辽阔的雪原风光。

　　午餐后，身着艳丽的滑雪服，脚踏时尚的滑雪鞋、滑雪板，手拿轻盈的滑雪杖，迈着自信的步划，在热情的滑雪教练指导下，在宽阔的初级滑雪场上学习犁式制动、转弯等基本滑雪技巧，体验驾驭冰雪的新奇的感觉。

　　晚餐后，走出温暖的小屋，抬头眺望满天群星，呼吸一下清新无比的雪原空气，或是观赏烟花篝火，或是自助烧烤，或是参加夜间灯光滑雪场的滑雪运动。青少年可以在风车度假饭店的康乐宫内参加多种室内活动（游泳、台球、保龄球、乒乓球、电子游艺、网吧、夜总会、桑拿等）

　　第二天，早餐后，在初级滑雪道继续练习半犁式制动、减速、转弯。滑雪过程中，可以欣赏茫茫雪原，听吱吱雪声，呼吸清新空气，并尝试沿雪道自上至下滑翔的美妙感觉。休息时可选择雪地摩托，滑轮胎，打雪仗，冰上拔河等娱乐项目。

　　午餐后，参观雪上芭蕾铜像、中国石、风车别墅、天印湖，也可以乘缆车上山参拜五指神树，从山顶欣赏森林雪景，乘坐滑车沿世界最大滑道飞速下滑，新奇刺激，有惊无险。

　　下午乘坐T410次（16：34）子弹头特快列车返回哈尔滨（19：32到站）。哈尔滨站乘T18次返京。
