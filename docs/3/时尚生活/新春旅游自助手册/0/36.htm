越南旅游自助手册

(2000-10-25)

　　越南实在并不是个特别好玩的地方，但好象已经成为黄山九寨之后的热点了，所以还是有必要介绍一下。

　　主要行程是北海-芒街-下龙-河内-海防-北海，在7日内可以轻松完成旅途，费用在1800以下已经豪华的了。没有特别值得推荐的季节，因为在东南亚只有干湿季只分。只要不在中国的大假去就是舒适轻松的旅途，是没有护照又想出国尝试异国风情的最简便推荐。

　　北海和办证

　　北海有飞机和火车直达，但火车每天只有从四川来的一班，飞机的班次就经常变动，如果从桂林来最方便是走高速公路，虽然是腐败官员修建的路，但毕竟是好路。

　　北海住宿方便便宜，即便是海边标间也不过一百多元，北海大陆唯一推荐就是银滩，沙非常细白，浪出奇的小，海滩很宽，值得去看看，另外北海还有两个外岛比大陆有特色，但由于时间紧迫重点在越南，此线就不走那里了。

　　去越南要办证，办的是边民互市的证件，有效期就那么六天左右，而且不能去西贡。证件需要委托当地旅行社办，需要五张证件照。

　　如果走水陆，可以在北海码头凭证件买去越南（海防、下龙都有）的船票，行程需要一个晚上，比较舒适，顺便叫旅行社买票可以省些事情。如果走陆路，可以搭车到东兴，但过境手续有时候会很麻烦，可以委托旅行社帮忙混过去，顺便叫他帮忙在越南方面联系好旅行社。　　

　　芒街和越币

　　过境是越南的芒街，其实只是个边贸的批发转运点，市场边坐满了越南妇女，是换钞票的，最好就在那里换上那么点越币，现在是1：1500，反过来是1700：1，去鸿基和下龙的车也在这里就有，不跟旅行社就得在这里坐车。芒街没有什么特别的东西，基本是玉石、牛角、拖鞋、香水，胜在便宜，牛角摆件比国内便宜得太多，不怕麻烦就买吧，整个越南就这里便宜了。芒街的香水比下龙贵，但是“绿毒”50ml也就20元人民币，自己看着买了。看见三五快钱的尖顶草帽赶紧抢先买，因为谁后买谁就是跟风，跟风买就没意思了。　　

　　越南的语言环境很糟糕，没有英语提示，基本不会听说外语，所以如果想行程自由些还是请个越南导游的好，包不包车再说。有一些地名的越南语供参考：河内hanoi，下龙halong，海防haiphong，芒街mangcai，鸿基honggai，河北省habac，广宁省quangninh，环建湖hoankiemlake，胡志明hochiminh等等。除了旅游点外，基本不收外币，所以越币一旦用完，要尽快找旅游点换，否则连厕所都进不去。在芒街逛逛就可以走了，当天就可以到下龙。
