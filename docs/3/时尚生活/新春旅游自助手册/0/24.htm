国外旅游八大锦囊

北京晨报 2001-01-02 08:43:29

　　行装 早些拟出一个单子，写出要带的东西，避免出发前临时忙乱。如果到东南亚旅游，那里永远都是夏天，一定别忘了带夏季的衣服。东南亚国家的多数饭店都不预备牙具、拖鞋，如果您忘记带，小事也会很烦心。

　　药品 一旦到了异国他乡身体不适，身边的小药就能救大急。几种药应该在行囊中必备：感冒药、乘晕宁、息斯敏、黄连素、创可贴，心脏病人还千万别忘了带硝酸甘油！另外，可能的话，带一支体温表也不算多余。

　　风俗 记住！泰国小孩的头摸不得，印度的小孩抱不得。到泰国，进入寺庙要脱鞋，女士见到僧侣要避让，游览大皇宫女士不能穿短裙和凉鞋。在新加坡街头随地吐痰、乱扔废物，高额的罚款会让你的脑袋大一圈。

　　护身卡 到语言不通的国家去旅行，一到饭店，先在总台拿一张饭店的卡片。如果迷路，卡片上有饭店的地址、电话，出租车司机会把您送回来。

　　付费 国外饭店房间里有些电视节目看了是要付费的。冰箱里的东西用过后多数也要付费，但一般有两瓶带标签的矿泉水是可以免费用来解渴的。东南亚的饭店房间里一般没有热水，但备有电热壶，要沏茶需要自己烧水。

　　储物 贵重物品存放在总台保险柜比放在房间里保险。一般饭店的总台都有这种保险柜，免费给客人存放贵重物品，钥匙由饭店留一把，您自己拿一把。顺便说一句，外国的饭店很多都有规定，您的贵重物品如果在房间里丢失，他们并不负责赔偿。

　　购物 假冒伪劣产品，并不只是中国才有，您在东南亚一些国家购物，一样会碰到。选购金银首饰、宝石制品等高价值商品，千万不能只听信商家可退可换的承诺，一定要想好了再买。为帮客人退货折腾几回仍办不妥的事例，好些旅行社都遇到过。

　　现金 在国外单独上街，身上带的钱一定要适量。因为中国人大多不爱用信用卡、旅行支票，总爱带大量现金在身，所以国外的有些强盗专门打劫中国人。但身上一分钱不带也不好，在美国治安不好的城市，为保安全，无论如何也应带上２０美元的保命钱在身。（携程）
