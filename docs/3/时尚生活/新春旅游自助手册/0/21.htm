曼谷、芭达雅、普吉岛、珊瑚岛8天游


第1天：广州━曼谷━芭达雅━云石公园

　　於广州白云机场出发，搭乘豪华客机前往泰国首都━曼谷，抵达後由穿着泰国古典服装的美女献上迎宾花环，随後前往云石公园叁观，在这里有许多百万年以上的奇石和动物化石，矗立於一片绿草如茵中，另有鲤鱼养殖场及梅花鹿饲养区，在此您可以与老虎合影留念。晚餐後可至世界闻名的泰国「人妖秀」，豪华盛大的场面、瑰丽的舞台布景、高级的音响效果，开启您的新视界。

第2天：芭达雅━格兰岛━金沙岛

　　今日换着轻便服装搭乘游船前往格兰岛，穿越暹逻湾，享受休闲渡假生活，途中安排小艇，透过艇上玻璃可观看海底珍奇珊瑚奇景及热带游鱼（气候不好则取消），期间可自由玩水或自费骑乘水上摩托车享受无拘无束的大海驰骋，亦可体验海上拖曳伞遨游蓝天白云的绝妙快感。中午可游览附近一美丽岛屿━金沙岛，岛上沙滩绵长洁白，海水湛蓝清澈，透明见底，游鱼可见，於岛上可游泳、浮潜、海潜等，岛上无人居住，您就是岛主，自由自在享受别的岛屿所不能的清静幽闲。下午点心为 BAR-B-Q 炭烤活海鲜，新鲜的鱼、虾、螃蟹、烤肉串让您大快朵颐，乐不思归。夜间自由活动，或可享受泰国着名之指压（按摩）。

第3天：巴达雅━东芭乐园━小人国

　　早餐後前往芭达雅近郊有名的「东芭乐园」游玩，可观赏猴子摘椰子特技，兰花园中百花盛开、珍异花卉，并可欣赏园内文化村各种精彩泰国传统民俗表演如斗鸡、泰拳、民俗舞蹈、婚礼习俗、比剑等，结束後还有大象表演，最大型的动物表演各种动作、踢足球、跳舞、骑车、拔河、可爱又有趣。下午叁观「小人国」缩影城，中有世界知名及泰国重要建筑的缩小模型，构造精致华丽，逼真动人，颇值一看。

第4天：芭达雅━曼谷━皇宫玉佛寺━毒蛇研究中心

　　今日返回曼谷，前往泰国拉玛王朝大皇宫叁观、金碧辉煌、富丽典雅的宫廷建筑及庄严神圣、古意盎然的玉佛寺，会使您大开眼界、赞叹不已。下午顺道叁观国立毒蛇研究中心，欣赏空手抓蛇特技，再叁观民俗特产店、珠宝、皮件等泰国特有产品专门店後，晚餐於泰式自助餐厅，餐後返回酒店休息。


第5天：曼谷━水上市场━赛福瑞野生动物园━海洋世界

　　早餐後，搭船游览湄南河风光，实际体会泰国水上人家生活，继而叁观水上市场，特殊的交易方式新鲜有趣。上岸後前往规模庞大建设完善的「赛福瑞野生动物园」乘车观赏各种热带奇禽猛兽如角羚、犀牛、长颈鹿、斑马、老虎、豹、河马、熊、猴群、鹳鹤、孔雀等许许多多动物，生动的呈现在您眼前。继而至海洋世界观赏精彩的各项表演，有「美国西部牛仔秀」、「海豚、海狮P 秀」、「美国划水特技秀」、使您赞不绝囗。晚宴於湄南酒店、享用海鲜 BAR.B.Q 自助餐，在悠扬乐队演奏声中，一边晚餐一边欣赏湄南河岸迷人夜景，轻松渡过美好时光。


第6天：曼谷━普吉岛━攀牙湾━回教村

　　今日搭乘泰国国内班机飞往南方岛屿━「普吉岛」，普吉岛为泰国第一大岛，比较於曼谷的繁华与芭达雅的璀灿，普吉岛像是颗明珠，洁白无瑕，屹立於印度洋上，气候怡人，椰林婆娑，遍植像橡胶树，民风淳朴，抵达後前往游览攀牙湾，搭长尾船穿梭於热带原始林间，仿若蛮荒探险，类似桂林山水的奇岩怪石遍布四处，抵因「007 金枪人」而声名大噪的铁钉岛，更可感到自然神奇造化之工，在此处发挥到极致，中午在海上人家━回教村落午餐，特殊的文化产生了有趣的比较，下午返回普吉市区，叁观普吉着名的贝壳工厂及腰果工厂，夜间您可自由於街上购物、观光。


第7天：曼谷━珊瑚岛━神仙半岛

　　今日早餐搭乘专车前往码头，再换乘船至「珊瑚岛」、珊瑚岛为普吉邻近之一小岛，风光秀丽、海水透明，海底珊瑚发育完整并且种类奇多，为泰国政府明令保护的岛屿，换上浮潜装备，整个海底世界都属於您的，各种彩色热带鱼就在您周围游过，绮丽的海底世界，开启您从未有过的新视界。下午游览吉甫赛村及神仙风景区，愉快晚餐後，返回酒店。


第8天：普吉━中国

　　今日搭乘豪华客机飞离普吉返抵中国，结束旅程，回到温暖的家，并期待下回再见。
