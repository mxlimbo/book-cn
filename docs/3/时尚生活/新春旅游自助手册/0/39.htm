办理签证小常识

(2000-10-30)

　　材料应附原件及复印件，中文材料应翻译成相应的英文或法文；

　　拿申请表应提早，访美申请表和商务用日程表8：30在驻华使馆门口的箱内可以取到，访德申请表8：30―8：40领取；

　　各驻华使馆收取申请、签证费时均不备零钱，只收取准确数目；

　　有的大使馆申请签证需预约，如美国在6月至9月实行预约制，周一至周五下午到签证处预约，德国的旅游签证应打电话预约：010-65325560-100；

　　各国签证从申请到签发所用的时间不同，如美国的探亲签证批准后需一个工作日，其他一到十个工作日不等，泰国需三个工作日，德国需五个工作日等，所以申请者应根据自己的出发时间提前申请；

　　因地域不同，申请者应就近申请，如泰国有驻广州、昆明领事馆，美国有驻成都、广州、上海和沈阳总领事馆，上海、江苏、浙江的申请者应向法国或德国驻上海总领事馆申请。
