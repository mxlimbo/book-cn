冰雪热了新世纪旅游

(2000-12-12)

黑龙江：冰雪旅游好戏连台

　　今冬黑龙江捧出的盛宴令人向往，从2000年12月2日到2001年3月，黑龙江各地各种类型的冰雪节日将会把北国的冬天串成一连串最“火热”、最令人心跳的日子。

　　――哈尔滨冰雪大世界的活动于2000年12月25日至2001年2月28日举行。

　　――龙珠二龙山滑雪场，将成为第三届中国黑龙江国际滑雪节举办地;

　　――具有中国第一雪乡美誉的海林，今年将举办中国雪乡旅游节。

吉林：冰雪旅游异彩纷呈

　　2001中国体育健身游暨吉林长春冰雪旅游节将于新世纪的第一天拉开帷幕，延续到2月18日，从长春开始，经吉林市，至长白山，各具特色的冬季冰雪旅游活动一路展开，异彩纷呈。

　　――长春冰雪节将于2000年1月1日开始，以净月潭滑雪表演及冰雕艺术作品展、冰灯游园会为主要内容的冰雪节，盛况空前；

　　――吉林国际雾凇冰雪节将于２００１年１月２日至２月１６日在北国江城吉林市举行。展现五彩纷呈的场面：雾凇奇观、冰雪世界及有体现关东风情的踩高跷表演；

　　――北大湖国际滑雪节，领略碧天银海的冰雪狂欢世界；

　　――第二届长白山国际冰雪节在延吉市举行。
