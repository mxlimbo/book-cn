旅游衣着之冬装


(2000-12-18)

　　过去，每当冰天雪地的冬季来临，人们总是穿上厚重的棉衣御寒，出门则更是穿了一层又一层，恐怕不够暖和，根本不去考虑轻便与否，总是将自己裹的似皮球一般才能放心的出门。现在可谓是冬依旧、衣不同。随着科技的发展和人们生活水平的普遍提高，各种皮革服装、羽绒衣等冬衣已逐渐取代了传统的棉衣。从质料上讲，皮革服装、丝棉衣等御寒效果强，花色品种多，且轻便、耐穿、美观、便于携带，应为动基础有的首选。
　　
　　如果你喜欢穿大衣，最好再配上一条毛绒绒的大头巾，既雅致又兼有保暖作用。一般来说，偏矮的女士，最好不要穿过长的大衣；而个子高挑的小姐在配裙装时，最好是长大衣，且大衣最好比裙子长一些，这样穿里外一致。短大衣配裤装也比较适合，特别是宽松的短大衣，配上一条灰色或黑色的紧身裤，在穿上一双高腰皮鞋和一双毛针织袜套，会有一种现代典雅的风格。
　　
　　冬季若一皮装作为外套，里面的服装则可以相对薄一些，用不着再穿那些厚厚的防寒套装了。在皮装内配穿白色高领毛针织服装，加上一条同色西裤及与皮衣同色的高帮鞋，是很迷人的装扮。
　　
　　男性的冬季着装首选是皮装。皮装最大的好处就是防风。男士的皮装有茄克、西服、中褛、风衣之分，穿的最多还是茄克和中褛，茄克由于下摆是松紧带收紧的，所以一些腰围大、肚子大的男士不适合穿。在穿皮装时，别忘了配一条漂亮的围巾，增加几分帅气和休闲的情趣。围巾的花色与皮装的颜色要协调。黑色的皮装，可以配蓝、灰、白等冷色调的条格围巾；如果是棕色的皮装，围巾最好选择与之相近的带有棕色的暖色调围巾，这样会显得和谐，很有气度。
　　
　　各种棉褛、棉茄克也是常用服装。在这种防寒服里面，可配穿各种颜色的针织高领毛衣或电脑提花的纯毛马球衫，再配一条与防寒服色彩协调的围巾，既挡风，又可体现出男士的稳健、诙谐。
　　
　　此外要说明的一点是，人体自身有一定的调节体温的能力，但这种能力并不是无限度的。如果穿衣太多、太厚，人体热量散不出去，热量增大，皮肤血管扩张，流向皮肤的血液就会增多，从而降低了肌体对外界温度变化的适应能力，可能会引发或被传染各种疾病。冬季过厚固然增强御寒效果，但也可能因此而降低人体的抵抗力，使人染上流感等传染途径较广、较强的疾病。
　　
　　另一方面，服装也有一定的重量，人穿衣服就要承担着一部分重量，从而对人体产生压力，这些压力主要有肩部和腰部来承担。冬装太厚，必然会增加肩、腰部的负担。若负担过重，则可能直接影响人体健康，引发某些疾病造成身体不适。
　　
　　那么，冬季出门的服装以多少重量为宜呢？据服装专家研究，男士冬装的重量应以3千克左右为宜。女士则为2千克，这不包括帽子、手套和鞋袜。帽子和手套是必备的，最好是皮质皮品，既保暖，又轻便、耐脏。
