昆明春节旅游专列精选

2001-01-08 08:53:31

　　为在春节喜庆的气氛中让游客悠闲地坐着火车出国出省观光，昆明铁路客运公司将开行一批春节旅游专列，目前有关各方正积极组织游客。

　　春节“异国风光游”将带游客到越南河内、海防、下龙湾等迷人景区感受异国风采，昆明客运公司的专列共开行２趟，第一趟大年初一即１月２４日开行，第二趟于大年初二即１月２５日开行。专列于第一天的１６时９分从昆明站开出，次日６时３９分抵南宁，经越南芒街出境后，共游览中越大桥、北仑河、商业一条街、下龙夜景、“海上桂林”、斗鸡石、天宫洞、还剑湖、巴亭广场、胡志明博物馆、福林寺、“卡西诺”赌场等１４个旅游景点。同时，从芒街入境后到北海，游览海滨公园、水族馆、大型音乐喷泉“潮”、城雕“南海魂”、北海银滩等景点，专列于第七天的7时9分抵达昆明。

　　春节“南国风光旅游”专列特意为在春节期间到深圳、珠海、广州的旅游者组织开行，为８日游。该专列１月２４日（初一）２０时３０分从昆明站开出，１月２６日６时４２分到达深圳，游览锦秀中华和世界之窗；１月２７日前往珠海，游览拱北海关、九洲城、免税商场、渔女像、情侣路等５个景点，并乘船澳门环岛游；１月２８日到广州，游越秀山、云台花园、中山堂等４个景点，并安排市容观光。１月２９日２２时２２分从广泛站乘专列返昆，于１月３１日（初八）９时２０分到达。

　　大年初二还将开行为期５天的“南宁、北海风光游”，在南宁游览青秀山、文物苑、博物馆３个景点，并安排观光南宁市容。在北海游览音乐喷泉“潮”、水族馆、城雕广场、珍珠基地、干海味市场、大银滩等６个景点。

　　云南省内景点也将开行旅游专列，春节期间将开行新近旅游热点罗平的专列。此趟旅游列车的开行日期视客流情况而定，有客即开，无客停运。
