忘俗蜀南竹海

圆舞儿 2001-01-05 10:37:13

　　记得苏东坡曾写诗言道：“无肉使人瘦，无竹使人俗”。在这个世纪末的冬日，对于既没有太多时间，也没有太多金钱的，特别是不想落俗的我来说，蜀南竹海便是首选了。

　　从成都新南门汽车站出发，大约四小时后进入长宁县城，山间满目皆翠，河里碧水幽幽，偶尔还有一两只竹筏漂在河中。竹海距长宁县城只有20公里，在长宁县下车后，可以搭机动三轮车直达竹海西大门。

　　一走过三里半竹海大门，便是一望无边的竹海了。竹海的形成，主要依赖于得天独厚的地理条件，竹海所在地海拔600―1000米，冬季最低气温不低于0℃，夏季最高气温不超过30℃，雨量充沛，土质酸性，含氮含铁特别高。竹海景区内分布有楠竹、人面竹、凹竹等30余个品种，面积达120多平方公里，为天下所罕见。仙寓古洞，忘忧幽谷，五彩飞瀑，墨溪流水等景点，如颗颗明珠，闪烁其间。最具特色的要算那坐在竹楼中细细品尝的“全竹席”，香喷喷的竹筒饭、清凉的竹根酒，炒的、拌的、煮的、烧的，各式“竹菜”，让人在大长见识之余也大饱口福。

　　入夜，没有灯火的竹海也并不寂寞，琴蛙声此起彼伏，犹如古琴，为举世罕闻的佳音，令人产生高山流水之思。

　　提示：

　　成都、重庆游客乘汽车、火车先到宜宾，再转乘旅游专线中巴到蜀南竹海。

　　成都有直达竹海的客车，新南门汽车站上午8：00（可提前购预售票）出发，车程约4小时，票价85元／人。
