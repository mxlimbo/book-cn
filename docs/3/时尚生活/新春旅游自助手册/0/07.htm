九寨沟冬之旅自助游

2000-12-06 15:04:27

　　第一天：成都――九寨沟

　　早晨八点前往成都新南门汽车站乘旅游直通巴士前往九寨沟，车票价格是85元人民币，约需10小时左右可以到达九寨沟沟口，中午会在茂县等县城停车用餐，最好自备一些零食及干粮。

　　抵达九寨沟后您要先选择酒店，建议先在成都咨询及预订好您满意的酒店。

　　晚餐最好在所住的酒店进餐，酒店外几乎没有餐厅，价格一般20-30元/人就可以了。

　　第二天：九寨沟内游览

　　在所住的酒店用早餐后，可以坐三轮车前往沟口售票处，如果步行的话，约需要20-40分钟。购票时您需要将门票及观光车票一起购买，总费用是150元，另需要购买3元钱的保险费。

　　如果您在九寨沟只有一天时间，今天您就需要将三条沟一起游览，九寨沟的景点呈“丫”字分布。因观光车每次停靠后不会在原地等待，所以最好减少下车的次数以节约时间和体力。原则上可以从沟口直接坐到原始森林下车，原始森林的游览时间一般大约只要20分钟，草海，天鹅海不必下车，在车上欣赏即可。箭竹海和熊猫海是两个姊妹海，在箭竹海下车后，拍好照后不用花时间等车，沿公路走到熊猫海只需十分钟。沿着熊猫海栈道下行，下行到五花海、珍珠滩瀑布、镜海，参观拍照后以后的几个景点只需在车上看看即可。当车来到诺日郎中心站时，你可以在这里的餐厅用午餐，上洗手间。补充胶卷(最好在出发前戴足)。休息之后等车，前往另外一条沟-则查洼沟。上下季节海都只需在车上观看。接下来需要花时间的是九寨沟最小的海子-五彩池。参观之后，乘车返回到第一条主沟-树正沟。游诺日郎瀑布、犀牛海，老虎海下车后可以走到树正瀑布的脚底。芦苇海，双龙海，你可以一直走到火花海上车，之后，你值得停留的最后一个景点是盆景滩。一天的游览充分而不松散。

　　第三天：九寨沟返回成都

　　早晨八点前往九寨沟口乘旅游巴士返成都。您最好自备一些零食及干粮。 (如果还想到牟尼沟，则买票到松潘，下车后租马到牟尼沟或者租车到牟尼沟，晚上可宿松潘，可以在晚上买好回成都的车票。松潘到牟尼沟的面包车租车费约300元，如果租马约需要60元/马，这一切都可在松潘办好，当地有专业马队，有很多老外也在当地租马，最好先讲好价格，以防被宰)
