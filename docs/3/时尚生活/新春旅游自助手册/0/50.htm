学生证的妙用

　　火车半票

　　这可是一个省钱大法。一般来说，学生半票的价钱是硬卧全票价格的四分之一。如果我们来回双程购买学生票而不是硬卧，就可以节省下一笔可观的费用。路费在整个旅程的花费中的比重一般来说是最大的，如果在路费上节约下来，整个旅行的成本也将大大降低。但是铁道部门关于学生票问题有较严格的规定，必须是本人居住地到学校所在地的客运段，并且在学生证上注明出发地和目的地，才可以在这个区间内购买学生票。

　　方法一：借学生证。 找居住在你所要去的目的地的朋友借他们的学生证，前提是：你的那位同学假期不回家。或者是住在目的地线路以内的火车线路上的城市也可以。比方说：我的学校在广州，我想去北京玩，我就要事先准备好北京或者沈阳的学生证。

　　方法二：办理新学生证。 此法为方法一的补充。可以先让方法一中所说的朋友报失学生证，一般学校交一定的费用即可办理新的学生证，而此费用相比火车票则是九牛一毛了，可是用小钱省下大钱哦。记得最好把相片换成自己的。

　　景点也可以获得学生优惠。
