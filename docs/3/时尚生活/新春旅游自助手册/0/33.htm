日本旅游的基本知识


(2000-10-10)

　　时差

　　比格林威治时间早9小时；比北京时间早1小时。抵达日本后，请将手表拨快1小时。
　　
　　电压

　　日本的电压均为100伏。如想使用中国制的电器产品请跟饭店的总台打招呼。
　　
　　饮用水

　　日本的自来水一般都可直接饮用。但毕竟是生水，谨慎一点的话，还是饮用矿泉水为好。
　　
　　电话

　　日本的通讯相当发达，但中国手机在日本因制式不同不能使用。公用电话到处可见。打电话可用投币和插卡的方式。标有国内、国际电话兼用字样的电话机（墨绿色）及IC电话机（桔红色），均可直拨国际长途。打回中国时先拨001（或0041、0061）＋86＋去掉0之后的国内地区号＋对方的电话号码。
　　
　　这里很难找到网吧，另外饭店中的市话、长话都收费（有些还要加上15％服务费）。用100日元硬币打国际长途（投币），大约可讲3—5句话。
　　
　　邮件

　　如果您想给国内的亲朋好友寄信、明信片或包裹时，最简单的方法是交给饭店的总台处理（海外明信片邮费／70元，信件／90元）。
　　
　　货币

　　日币以元为单位。分纸币和硬币二种。纸币有1万元、5000元、2000元和1000元四种。硬币则分为500元、100元、50元、10元、5元和1元。外币的兑换以当日牌价为准。人民币在日本不能直接兑换。饭店前台全天可以用美元换日元，去银行换钱应注意营业时间（一般周一至周五是9：00AM—3：00PM）。
　　
　　自己随身应备10元、50元、100元的日币，以便在公用电话打市内、国际长途。
　　
　　交通

　　与其它岛国一样，日本的道路都靠左通行。日本道路的车流量很大，务请不要横穿马路。出租车较贵，建议搭乘地铁。
　　
　　治安

　　日本是社会治安较好的国家，但为以防万一，请您注意自身贵重物品的管理及尽量避免单独行动。
