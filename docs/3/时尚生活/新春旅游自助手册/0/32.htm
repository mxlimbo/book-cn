尼泊尔旅游资讯


(2000-10-17)

　　历史久远的古老街巷，层层叠叠的宝塔，古韵悠长的石雕和恐怖的面具；虔诚者手中不停咕咕转动的经筒，色彩斑驳的唐卡；悠长的低诵，激昂顿挫的圣歌，还有欢乐喧闹的印度音乐；四弦琴的弦音混合着短笛发出的诡异音符飘荡在空气中。民歌者、舞者喧闹的集会给加德满都山谷带来了生机。　　
　　
　　作为世界上唯一的印度教国家和释迦牟尼的祖国，宗教文化溶入到了尼泊尔国民的血液之中，印度教和佛教在相同的世界观下和谐相处。
　　
　　尼泊尔的本土饮食却没有如宗教一样融汇了两个美食邻居的精华。可概括为简单而无趣。多数情形下，一顿标准的尼泊尔餐包括一碗内容不详的菜汤、米饭和咖喱味道浓重的蔬菜和肉块。奇异的土地上总有奇异的事，在加德满都泰末区的自助餐馆里你可以有以下选择：墨西哥玉米饼、日本寿司、中国蘑茹、意大利洋葱肉汤、酥皮馅饼、自选三明治、苹果派、杏仁饼等等。享用如此美食，还可以佐以土产啤酒或者喜马拉雅山地家酿的大麦酒。
　　　　
　　旅行推荐加德满都

　　尼泊尔的首都是个传统与现代混合交织的大杂烩，众多的庙宇夹杂在简陋的贫民区中。而在市郊也不乏豪华酒店、餐馆和高档商店。加德实际上是两个城市：一个是遍布着红墙庙宇的宗教圣地；另一个是充斥着猴子、乞丐和汽车废气中的混乱都市，行进在加都曲折、狭窄且时常毫无指示的街道之中，创造力和耐心是必须的修养。　
　　
　　加都的中心是德巴广场，而市北的泰来区则是自助旅行者的大本营。多数的经济旅馆座落在泰末区中心地带，古老、幽深而毫无规律的小巷可以满足你的怀旧癖好。
　　
　　价格低廉，味道醇正的泰末区的餐馆，仿佛是一个环球美食的博览会。旅馆里的告示板上贴满了提供挑夫、导游服务和寻找旅伴的字条，给人一种天下一家的感觉。
　　
　　卡拉延佛庙的白塔，贾格纳斯神庙中让人脸红的情色雕刻和旧王宫都是加都的名胜。弗瑞克街是加都人生活的缩影，她的污秽与美丽都让人无法回避，街道中飘荡着浓劣的印度香味道，摇着转经筒的孩子，贩卖水果，尼泊尔、唐卡和纸草画的商贩，多如牛毛的廉价旅馆和灯光闪烁餐馆，让这条街的“FLEAK”（奇特的）名符其实。
　　
　　帕坦（Patan）和巴克塔布尔(Bhaktapur)是加德附近的两座古城。她们都以古老的庙宇和精湛的手工艺品闻名。
　　
　　世界上最大的佛庙之一——巴德纳斯（Bodhnath）也建在谷地之中。这里同时也是尼泊尔藏人的聚居地。　　
　　
　　远足（Treking）尼泊尔被称为远足登山爱好者的乐园，人们在远足中即可以观赏许多著名的雪峰，又可以在边远村落了解人们尚未被现代化冲击的古老生活方式、奇异的风俗习惯。自助旅行者一般都会参加一个一周至二周的远足项目。
　　
　　漂流（Rafting）在奔腾咆哮的河水中划皮艇，玩激流将是毕生难忘的经历。从高山野林西藏边境的上游一直划到与印度接壤的广阔平原。仅在一百英英里内人们可以看到变化无穷的地形，耕作的梯田和古老的民风。笔者强烈推荐此项目，但其有一定危险性，参加者务必做好心理、生理准备。　　
　　
　　波克拉（Pokhara）对于寻求平静安逸的度假者来说，尼泊尔西部的湖城——波克拉实为理想选择。在雪峰倒影下的湖中划艇，玩独木舟、钓鱼和游泳。这里海拔900公尺，属于热带地区。奇特的是仅距离40公里，竟可观望整个西部的安纳布纳山脉，其中最高雪峰达8000公尺以上。
