庐山雪中行

herry007 2001-01-05 13:58:14

　　日照香炉生紫烟”，大约指的就是庐山总是云腾雾绕，面对面竟看不清对面的景物。但这就是庐山最美最撩人的地方，就像是美女的面纱，所有的美丽都若隐若现，让人浮想联翩。去年12月12日庐山下了今冬头场雪，雪气与雾气相合，这层面纱比以往蒙得更紧了，却又让人更想一见她的真面目。

　　早上天空放晴，太阳出来了，雾气渐渐散去，蓝天、白云、雾凇、雪凇等景色分外绚丽多彩。如琴湖（原叫花径湖，因形状像提琴而得名，在牯岭镇的南边）湖水清澈，岸边树上挂满了雪，湖中竟还有人在划船撒网捕鱼。原本千岩竞秀、万壑回萦的锦绣谷，现在只能看到树上千奇百态的雾凇、雪凇，听到谷间溪水的哗哗声，沿锦绣谷小道西行500米到达仙人洞，洞系自然风化石洞，洞顶有一个高6米，宽12米，深14米，形似佛手的岩石覆盖，故又名“佛手岩”。洞边有一座道观，有几个道姑常年住于此。

　　再继续向西，道边尽是被雪压得折弯了腰的竹林，湿润空气让人可以感觉到水气，路很滑，几次差点滑倒。

　　到了中午，晴空万里，阳光普照，雾凇雪凇在太阳的照耀下化得很快，走在树林间就像在下雨。可一到傍晚时分，太阳的照度不断下降，小风飕飕地刮着，雾凇雪凇又很快变成了冰凇。

　　眼下是庐山的旅游淡季，据说有95％的宾馆、招待所停业了，所以价格都相当便宜。包了一个房间，有彩电、卫生间、两张床、淋浴，才40元一天。

　　提示：招待所里没有暖气，屋里屋外一样冷，晚上睡觉前，最好不要洗澡，以免感冒。可以用两壶热水烫烫脚，穿着棉毛衣裤和袜子，盖上两床被子才能入睡。

　　乘火车到九江站，再花7－10元车费，行车24公里，就可到庐山牯岭镇。门票50元。
