汉城滑雪四日游

2001-01-05 14:31:19


　　第一天：北京-汉城，汉城-阳智。北京出发，抵汉城，游览乐天世界,晚乘车赴阳智度假村，住韩式度假村。

　　第二天：阳智-汉城。可自费在京畿道的雪场滑雪，晚乘车赴汉。

　　第三天：汉城。景福宫、青瓦台、韩国第一高楼大厦、88奥运会场、，汉城购物（人参公卖局、紫水晶加工厂、南大门、明洞等）晚餐后参观华克山庄。

　　第四天汉城-北京。金浦国际机场出发，抵北京，结束愉快的旅程。
