出境游的特别关照


(2000-10-31)

　　1．目前参团游比较方便的是新、马、泰、韩国、日本，到这几个国家旅游都需要在国内办理护照、签证。像越南、缅甸、柬埔寨、尼泊尔等边境国家则不需要持护照前往，只在边境办理证件即可，一般的旅行社组团时都可办理，国内的游客一般按北海、越南或云南、缅甸游的较多。

　　2．在取消旅游护照的情况下，除个别国家外，出境游一般都需办理五年因私护照，可通过旅行社或自行办理，在国旅、中旅、北京神舟国旅参团的游客，这三家旅行社将为您提供办理护照的服务。

　　3．特别需要提醒游客的是在游览新马泰的过程中，一定要把握好购物的尺度，购买珠宝首饰等贵重物品一定要慎重，当地的一些有纪念意义的精美物品不妨买一些回来送给新朋好友。
