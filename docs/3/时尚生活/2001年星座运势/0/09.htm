星座EQ

天秤座是一个在人际关系中很会委屈自己的人，所以生气的时候，表示对方已经欺人太甚，让他忍无可忍，或是认为某件事的处理方式太不公平。

至于天秤座的生气方式，是不像火象星座那样火爆的，他会对着对方把这件事情条理分明的说清楚，企图说服对方同意他的看法。之所以会如此，是因为天秤座十分讲究优雅的仪态，即使生气时也不例外。

就如天秤座象征图形：一把秤，生气的目的，就是要追求公平与正义，好让杠杆的两端能达到完美的平衡。

星座爱情

天秤座：公平正义一肩挑

天秤座族群是很容易辨认的，他们亲切和蔼，笑容可掬，很容易和别人打成一片。

一方面因为天秤座在占星术中代表「别人」，包括亲密的伴侣和朋友，所以他们喜欢「人」，也喜欢在人多的地方打转，很多职棒球员都是天秤座的。天秤座爱凑热闹的个性，和射手座、狮子座合得来，可是金牛座、处女座和天蝎座就很难忍受这一点。

代表和谐与美感的金星，同是金牛座和天秤座的守护星，但由于天秤座是气象星座，活泼开朗的本性，使金星的特质更明显，也更具有艺术方面的创作力，这一点和叛逆的水瓶座很有默契。

代表天秤座的图形，是十二星座中唯一的「东西」：一个天平，这个非生命体的象征，点出了人心中不属于本性的一面：公平与正义。每一个人都是自私的，受到后天的养成，才懂得什么是平等、如何尊重别人。

天秤座开朗亲切的特质，擅于交际，这点使双鱼座和巨蟹座十分羡慕。然而天秤座追求公平的本性，使他不愿意忽视别人的权利。

过分重视别人，也等于忽略了自己，天秤座族群不自私，可是心中不见得平衡，这一点和他们优雅的外表，恰好成为有趣的对比。于是许多天秤座积极投入公益事业、艺术、神秘学等，企图在其中找到内心的平衡点。

天秤座能带给你精致的爱情，不过当你爱朋友的天秤座走失时，记得到人多的地方去找他，他正在弥补他的不平衡！

星座上班族

第一印象

天秤座新进同事通常会是个显眼的人物，即使他的外貌未必亮丽，但是他很勇于发言，而且口才不错，通常说话不会得罪任何一方，会顾及两面的言词。也就是说，他可能说了一长串的话，但却没有什么结果，很可能他会为了维持和谐的气氛，而扮演居中协调的角色。

在团体中，天秤座部属也是个讨喜的人物，他的个性随和，你要他做什么都好，各项活动他也都会积极参与。

天秤座不懂得拒绝别人，当你身处孤单时，你可以很容易找到天秤座的新进人员，因为他的亲和与爱交际，很快就能与你建立起友谊的桥梁。

天秤座给你的初次印象，会觉得你和他已经很熟了，然而事实上，你又好象走不进他的内心深处。但是他不会因为与你相识才短短数日、数小时，说话就有所顾忌，他与你聊天时，几乎会将他家里的小狗、小猫等各项大小事物，统统告诉你。可是你却会发现，他好象和每个人都很好。这些内容好象也不是只有你知道而已，他似乎和大家都很好、而不是和某一个人特别好。

一开始，你或许会觉得天秤座的新进人员，好象与你特别投缘。他会与你交谈很多事，甚至也很友善的与你相处，你可能会以为你们很好，应该是很好的朋友了。慢慢你又察觉，其实他很爱交谈、能言善道，像个广播电台。不懂拒绝别人的个性，有时也反应了他的体贴，令人有一股温暖的感受。

激发其工作潜能

将天秤座放置公关部门可以说是最适合不过，举凡要面对各式人等，不论是采访、交际，他都能应对自如，起码他能很轻松处理人与人间的相处之道。

天秤座的口若悬河，也蛮适合业务部，加上他的脸皮又厚，所以面对陌生人，他依然能泰然自若、丝毫不紧张。尤其天秤座生来就有股优雅的魅力，在举手投足间，很轻易就能建立起别人对他的好感，这是天秤座相当得势的一点。另外，对于创意企画，他也能应付自如。

隐藏其弱点

天秤座较弱的一环，就在于财务的观念很差。天秤座的意志力相当薄弱，从他对很多美的事物无法抵抗来看，就不难窥出端倪，对于预算的规划，天秤座可以说无招架之力。

一般而言，天秤座的实践力很弱，如果以出版刊物的工作来看，采访是负责将文字内容交出即可，然而编辑则必须做好编排的工作，将所有的东西汇集而成，完成一份作品。为避开他可能较差的项目，可以尝试让他参与和美有关的各种工作，诸如美编、设计、摄影、服装设计等比较需要创意的工作，最好是能让天秤座动脑想些新点子，以满足其爱幻想的空间。

天秤座还有另一个缺点，他们容易神经质，容易不平衡，容易为了一点小事变得很愤怒。虽然他平常看来修养不错，但是当遇到一些临时的变故时，他会很容易为了一点小事觉得被亏待、或处置不公，他很在乎上司对他的态度，对于那种不平，他会放在心里，然后私底下才发表他的感受，传播他的想法。

让自己跑在他前面

天秤座的图案象征是一座天平，所以他时时力求自己能维持平衡，而他却常常不平衡，因此很容易被煽动。通常如果你和他说，某某主管对他好象有点不满时，这时他会张大眼睛或竖起耳朵，可以说有点神经紧张的询问你，为何口出此语？是否听到了什么？很可能他手边的工作就必须因此停顿下来，因为他的情绪已经大受影响，整座天平已呈现倾斜状态。

注重人际间的和谐，所以对于人事上的口语或谣言相传，天秤座会特别敏感而易受影响。不只是主管，如果你提到某位同事对他好象也有意见时，他会神经紧绷，甚至变得歇斯底里，心情左摇右摆无法平息。这正好可以衬托出你雍容大度的自信。

知人知面也知心

天秤座有种与生俱来的魅力，常常他一进一个新环境，便会使所有人感染到一股欢乐的气氛，而且很容易便成为异性注目的焦点，通常天秤座是那种「异性相吸、同性相斥」最典型的人物。或许是他一身有品味的穿著，非常有亲和力的举止及活泼的态度，很容易吸引异性的目光，有时会令一旁沉默内向的同性土象星座同事眼红。

或许很多人会发出疑问：我也不是比较丑，或者比较笨，为什么就是无法像天秤座那般自然的散发光芒？天秤座的一举一动几乎不会令人感到厌恶，相反地倒有些温馨，他的亮丽与亲切，常会赢得异性同事的好感，但是很可能会造成同性间的排挤或嫉妒。

小心敌人就在你身边

天秤座最大的优点在于灵活的公关能力，他一到新团体内，就马上将你的伙伴给吸了过去，天生的群众魅力与高超的社交手腕，使得他能很快跟人们打成一片。

他懂得主动参与别人的聚会，不像有些星座，个性保守，不爱参加各种活动。天秤座不然，只要有活动，他一定会热烈参与，不仅如此，还会主动发起某项活动，邀大伙一同参与。天秤座是个挺有乐趣的星座，很可能他的降临，会将你原本辛苦经营部署的人际网络完全打乱，而暂时让你失去重心。

天秤座同事长袖善舞，会跟上司、同事保持良好的互动关系，他的反应机敏，令你自叹弗如。面对他的能言善道，自己倒像个木头。他给你最大的竞争力，就在于他的人缘极佳，很自然就笼络了其它人的心。

天秤座的异性缘相当好，容易讨异性的喜欢，其它人很难抵抗这种自然的魅力。天秤座同事本身也很了解自己的迷人之处、也清楚自己拥有的魅力。天秤座女性的笑容灿烂，善于打扮、交际，使她拥有强大的磁力，很容易便适应新的环境，而她的亲切有礼，也常博得长官的好印象。

说得更贴切些，天秤座女同事通常晓得自己如何引起某一种人的好感，包括甜美的声音、迷人的笑容，适宜的妆扮与妩媚的肢体语言，或散发何种香水气息等。
公关的本能，使天秤座在十二星座中稳稳的独占魅力鳌头。

以彼之短衬己之长

天秤座同事的缺点是散漫、做事缺乏计划性，通常会想得多、做得少，理想伟大、行动力却薄弱。注重外表的妆扮，常使他的荷包倾囊而空，相对而言，财务观念也较薄弱。

天秤座同事很重视第一印象，所以你会发现他像只花蝴蝶，常常可以看到他飞到东又飞到西，似乎镇日忙着拓展自己的人际关系，不停的到各处采蜜。

或许在人际关系的广度上，是其它星座望尘莫及的，然而在晋升的道路上，天秤座未必会有同等的际遇。他对工作的野心与企图并不强烈，加上有时显得散漫，上班常迟到，花太多时间应酬交际……等等，反而会阻碍了晋升的管道。

此外，交报告时，你可以准备得充分些，并且有计划性，在面对天秤座常常急就章的处事行为时，你会有很好的机会。虽然天秤座同事常能将自己的疏懒掩盖过去，但是主管在判断事情时，仍会有一定的原则的。

天秤座的长袖善舞，换个角度看也可能是虚伪。如果你是个沉默寡言、不善交际的人，不妨继续保持自己的特色。久而久之，天秤座的能说善道，会造成别人眼中的不真诚，你的真诚与木讷又能重新赢得别人的认同了。

成为最佳工作拍档的绝招

如果你想拉拢天秤座同事，找他一起吃喝玩乐准没错，只要是享乐的事情他都非常喜欢。如果你找他参加读书会，那你只有自讨没趣。举凡任何与玩有关，不用大脑的事情，天秤座都会乐于参与的，譬如开幕酒会或一群人的聚会，能让他多认识朋友、能开心的吃喝玩乐。至于知性及精神食粮对天秤座同事而言，他铁定会将其排列在后，在他的生活哲学中，没有任何事比能过得快乐更重要了。

共享双赢成果的秘诀

其实天秤座是个藏不住什么秘密的人，通常你可能不需花费太大的力气，便能由他那得知某些情报。天秤座的防卫心不高，兴高采烈时容易说漏了嘴，或不经意传出秘密，所以藉由与他共餐，他就自然而然的将他所属部门的事情一一告诉你了。

天秤座不会有很强的占有欲或地域观念，禁止其它部门使用自己部门的设备，他算得上是个相当慷慨的人，只要你主动，他多半不会拒绝，天秤座是个很难向别人说「不」的星座，如果你提出需求，他多半会满足你的需要。

拢络人心的秘诀

天秤座会喜欢主动与他搭讪、主动与他做朋友的人，如果你有任何的活动、聚会、酒会……，举凡各种玩乐或能够认识新朋友或有口福的邀请，他铁定会非常高兴。除了喜乐方面别忘了找他参一脚外，你的私事或你的喜怒，也可以向他倾诉。

天秤座的鸡婆个性，会希望能多出些点子或帮你多分担点哀愁，他会适时的安慰你，而且会使你觉得，他是你的好朋友，你才会与他共享事物。

此外，天秤座的正义感，喜欢为朋友两肋插刀，尤其请他替你主持公道时，他更会乐于相助，并且衍生替你出口气的正义使命感。

小心自暴其短

天秤座的人讨厌做事情极端且自私的人，尤其是那种不顾别人、只顾自己，又喜欢自我表现的人。天秤座相当重视团队合作，他会觉得如果某人鹤立鸡群、独领风骚，会破坏甚至抢走了团体的风采。

天秤座的人其实有些懒惰，他可能希望藉由群众的力量分担自己的劳苦，所以常会吆喝一堆人帮他做一件事。如果你比他更懒，没办法满足他的支配欲，当然他会讨厌你了。

教战守则

好的开始，是成功的一半，没有开始，永远也不会成功。
天秤座常常在犹豫不决间，将通往成功的路给封闭了。人生的机会往往只有一次喔！

星座运势

整体运势：你即将迈入人生高峰，你所有的抱负及努力都将开花结果。你必须坚定你的生命价值。但也不要过於自以为是，应以开阔的心看待这个世界，让生活哲学引导你，而非束缚你。

而且目前木星进入你的迁移宫，有利於你的心灵提升，使你常怀热切的希望，思想清明，趋向於哲学或宗教等较为高深的学问的追求。有远行的机会。你可藉由旅行、远离、出走，带著某种东西走出你的生命，给你自己一分敞开的空间。

爱情运：家庭生活运佳，充满天伦之乐，在另一半闹情绪的时候能保持冷静。要能适度表达自己的感受，他们才会明了。要能有耐心的听取另一半的心声，不要老是以为你够了解另一半。

工作运：工作运不错，当贵人出现时，你的问题就迎刃而解了，对人与人之间的分寸要能拿捏得当，有助於对工作专业的掌控。如果你对一个看似未经计划的事件思考过滤，可收集到很重要的线索。意外的对你的事业帮助很大。

财运：你的财运甚佳，适合处理有关有价证券、动产、流动资金，上班族有加薪的可能，投资者则眼光独具，常能获利。但千万不可随便挥霍，因为这样运势也有可能导致邪门歪道的投资，变成不正当的赌博，甚至惹上官司。
