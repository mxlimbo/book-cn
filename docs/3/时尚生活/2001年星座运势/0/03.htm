星座EQ

白羊座的人发起脾气来就像漫画「灌篮高手」中的樱木花道一样，是那种平地一声雷式的爆发型情绪，发作时很强烈，但来得急去得也快，正是典型的火象星座。

至于白羊座发脾气的原因，多半是因为他不耐烦，而目的就是希望别人能跟上他的速度。但白羊座的特点就是脾气发作过了，问题却还是没解决，在处理情绪问题的能力上，与他的表达能力是两个极端。因此在情绪特征上，白羊座可被归类为婴儿级。

星座爱情

白羊座：敢拚爱冲怕落后

白羊座是十二星座中的第一个星座，它是整个循环的新开始，因此果决、勇敢的特质，在牡羊族群中表露无遗。这种决断能力，只有天蝎座比得上，也是让双鱼座、处女座最佩服的一点。

在交异性朋友方面，魔羯座男性对未来有很明确的规划，只是他们个性慢吞吞的。

火星是白羊座的守护星，它带来明快果决的个性。这种阳刚气息，很容易吸引阴柔细腻的男性，譬如巨蟹座。

白羊座擅于冲锋，拙于守成，如果你身边有一位白羊座，记得踩踩煞车，让他超车跑在你前面吧。

星座上班族

第一印象

白羊座在团体中，是个不容忽视的角色。一进公司就是个非常醒目的人，他们在团体中，总会想办法凸显自己，想出锋头，引起他人的注意，进而在团体中成为领导者。

如果你的事业地位很不稳固，白羊座的新进人员，将来会是个取代你的人，尤其如果你们被编列在同一组，担任同样的职务，你的安全感会饱受威胁，你的地位也会受到动摇。

白羊座在团体中会表现主动出击，喜欢当一个行动者，公司开会时会提出提案。他总是领先行动，领导群伦。譬如午餐时间，他会吆喝大家一起去吃饭。

他给人的印象，是个野心勃勃的人，一进来就开始组织团体，主动切入团体。绝对不是个呆坐在那里，没有意见的人。由于他的积极、投入性高，意见又多，因此在团体中很醒目，而给人压迫感。有时会让人看不顺眼，因为他会没大没小，一付爱现又跋扈的样子。

激发其工作潜能

白羊座的优点是主动，很有行动力。新进人员的学习能力非常强，他会主动要求分担工作，承担责任，积极参与公司的一切，而且做事速度快，效率又高。在同事之间，表现得很有担当的样子。

白羊座新人非常适合开创性的工作，尤其是在第一线担任基层业务人员。业务人员通常底薪低，奖金高。对于金牛座和魔羯座而言，非常没有保障，没有业绩就没有奖金。但对白羊座来说，很有挑战性，他觉得业绩做得高，表示能力好，可以依此证明自己的能力。底薪低，奖金高得吓死人的工作，最适合白羊座来做。
白羊座也适合开创新市场。假如你有个新市场、新的区域需要开发，或者该区域不在你们公司的势力范围，需要和其它公司厮杀一番，这种工作最适合白羊座的个性，可以派他去开拓新市场。白羊座不怕死，愈是没有人敢尝试的新领域，对他愈具有挑战性。他会认为只要自己开发成功，就可以成为他的地盘。他喜欢到新的地方画地为王；旧的地方对他而言，并没有太大的挑战性。

白羊座一旦为你所用，就会尽全力为你效忠。虽然他最终的目的，是要取代你的地位。但是若你能在职权上，充分授权给他，让他得到出锋头的快乐时，他也是个很好用的部属。你只要给他一点饵，他会马上上钩，而成为你出生入死的部属。

隐藏其弱点

白羊座新人很容易给老同事压迫感，认为自己什么都懂，喜欢表现自己，不放弃任何机会。另外，他对上司没有畏惧感，会比你更快和更主动接近上司，让你产生威胁，怀疑他和上司是否已经建立良好的关系。

白羊座部属的缺点，是冲劲有余，思考不足。一般人是三思而后行，白羊座是不思而行，他们是不懂得思考的人。很多时候，他们明明知道不可行，却不肯接受忠告，宁可犯错。

白羊座属下带给你的困扰，除了压迫感外，他还是个惹麻烦的人物，常常去招惹一些不必要的麻烦。因为白羊座非常铁齿，通常不了解公司团体里，有些禁忌存在，有些事情不能去谈，也不能去碰。

还有些事牵涉到其它主管；或者是存在已久，没有办法解决的问题，白羊座偏偏不信邪，想去打破规范，而让人感到头痛。

让自己跑在他前面

想要拉拢白羊座，最好派他去做些难度较高的事情。白羊座的新进人员，目标就是求表现。希望在一堆新进人员当中，赶快被认出来，让大家知道这个团体有个白羊座×××，他的目的就在此。所以在团体中要拉拢他，就是让他有表现和出锋头，使他有成名的机会。

白羊座是个机会主义者，凡事都要试试看，如果你给他机会，他会很高兴，而向你靠拢。尤其是给他挑战性高和高难度的工作，他觉得在你这边可以出头，就会自动靠拢过来。

由于白羊座具创新性和冲劲，要避免让他做些因循敷衍的工作。另外不能让他管理财务，因为白羊座非常浪费，对金钱没什么概念。如果让他担任会计部门的工作，往往预算打得过于宽松，不符合经济效益。

想要防止白羊座属下爬到你的上头，可以将他派至闲差事的部门。在竞争性不高，只能养老的地方，他就无从表现起，如数据库、图书管理员。有些地方，人事已经冻结，一动不如一静，不希望有新的动作，一切因循旧规行事。白羊座喜欢改革，求新求变的个性，会引起所有人的反感而排斥他。

另外，白羊座是个有创意的星座，喜欢创新和表现。你可以问他意见，但把差事派给他人执行，让他手痒，却也无从表现。

知人知面也知心

白羊座同事，在团体中有两种极端：一是大家都非常欣赏他，成为他的附庸，他不见得是个上司，但会是小团体中的领导者，能够在团体中呼风唤雨。一是他很容易被孤立，大家都不理他，独来独往，但他也无所谓。白羊座不会在乎别人的想法，或别人怎么看待他。他认为只要自己多努力，表现良好就行。因此他的人缘常表现极端，不是独行侠，就是领导者，当个大哥大或大姊大。

公司里如果有白羊座同事介入，必然会引起一阵骚动，因为白羊座是个竞争的制造者。除非他在外面有第二事业，表现良好，才会在公司做个隐姓埋名，没有声音的人；或者他在过去曾经冲过、拼过，有过一番历练。

如果一个公司的走向是以平等为原则，凡事按部就班，白羊座的进入，会破坏整个生态平衡。他的表现会非常突出，而破坏了原来的结构。所有表现平平，而只是资历很深，人缘很好的人，都是白羊座所瞧不起的。他认为资历很深的没冲劲，唯唯诺诺的是人渣。就好象在一个鱼缸里，放入一条食人鱼，如果喂不饱它，它就会把别人吃掉，白羊座绝对是个专门破坏办公室平衡的人。

小心敌人就在你身边

白羊座是个高竞争力的敌手，他是个非常可怕的竞争者。有个白羊座同事，你就很难出头，他一定会把锋头出尽。他们习惯拚命的表现自己，很容易树敌。如果你在私人机关，又有个白羊座同事，由于升迁不受资历影响，奉劝你要拚命努力，因为他是个高度竞争者。无论处在如何艰困的地方，白羊座都会不顾一切，勇往直前，横冲直撞地拚出一条血路来。

在团体中，每个人都必须缩小自己，以求立身之地。白羊座会把别人撞开，扩大自己的地盘。碰到利益冲突时，他是个完全以成就为导向的人，「优胜劣败」是他遵奉的法则。他是进化论的支持者，认为胜者生存，败者淘汰。他最讨厌人渣型的人物，在办公室里能力很差，毫无建树，只会唯唯诺诺，拍马奉迎，是他最瞧不起的人。

白羊座属于大炮型人物，有任何提案或不公平的事，他都会插上一脚。他认为人生就是要竞争，而且适者生存。如果别人有本事，白羊座欢迎有人来拚一拚，把他挤到一边去；如果你不挤人，反被别人挤出去，那是你活该，弱者本来就该遭到淘汰。

白羊座同事的优点，就是他的行动力很强，做事有冲劲，凡事都能自动自发，不会推卸责任。同事之间，愿意帮助他人，非常有正义感，常常见义勇为，喜欢拔刀相助。他的心胸宽大，不大会和别人计较，于公于私，他都非常热情，随传随到。

以彼之短衬己之长

他的缺点，就是常常刚愎自用，明知山有虎，偏向虎山行，不听他人劝告，非常的「铁齿」，你愈告诉他事情很难做，他偏要试试。

另一个缺点，就是非常爱现，常常抢了其它同事的锋头。他喜欢一个人把所有的事都做了，自己出锋头，造成其它同事的不满；久而久之，干脆把所有的事情都推给他，懒得插手。他也乐得自己做，不在乎别人的想法。他没有团队精神，也不想把工作让给别人表现，使得其它人没有成就感。

成为最佳工作拍档的绝招

拉拢白羊座同事最好的方法，就是把他拉进来当领袖，告诉他这个团体需要他这种人出力。譬如告诉他，大家要上阳明山玩，需要他开车带路。这种属于领导的位置，由他掌控方向，他会很快切入这个团体。他喜欢当个领导者，即使他不当上司，也会想在任何一件事上当个领导者，所以要拉拢他，就是让他当个行动派的领导者，什么事都由他出面，一切就搞定了。

拉拢白羊座同事并不难，但要小心的是，他缺乏团队的精神。能把他拉拢到你的小团体，并不代表在工作上，他会密切和你合作，或放水给你。他不会因为这件案子很好，而来找你分担工作。想和他合作，利益共享并不容易。白羊座是个个人主义者，在精神上，他是孤独的，在行动上，他是个领导者，缺乏团队的精神。

共享双赢成果的秘诀

白羊座对情报并没有什么概念。他的个性开朗，并非容易受骗。你愈是神秘兮兮的样子，反而让他觉得奇怪。所以想要套话，要装得一副若无其事的样子，好象寒暄一般，如「你们部门最近怎么样啊？」或者用拜托的口气，如「这件事非你帮忙不可，只有透过你才能知道，麻烦你帮我问看看。」让他觉得这件事非他莫属，他就会赴汤蹈火，在所不辞。

获取情报的最佳时机，是在你们一起出差的时候，或你们一同做业绩时，稍为略施小惠，礼让他一些业绩，他自然会把情报告诉你。

想要向白羊座借用设备，最好的办法，就是让他觉得他是这个设备的主控者，由他来控制资源设备。如问「你桌上的专线可否借用一下？」或「你办公室的传真机可否借我使用？」一切以他为主，他自然会借你使用。

拢络人心的秘诀

要让其它部门的白羊座人员对你有个好印象，首先要让他觉得你是个有魄力、有冲劲，敢做敢当的人，和他属于同一类型，虽然你们分属不同部门，也要让他确定你不会侵犯到他的势力范围，跟他不会有利益冲突，他会基于英雄惜英雄的心态，而替你制造口碑。

小心自暴其短

基本上，白羊座喜欢凡事明着来，在台面上作业。但是如果想预防他放冷箭，你必须对他采取低姿态。即使你和他不在同一部门，而且在你的单位锋头很健；在他面前，也要表现怕他三分的样子，敬重他如大哥大、大姊大。一旦你瞧不起他，觉得他没有什么了不起的时候，他会马上打压你。

给办公室里有白羊座同事的忠告：「请尊重他的个人主义和自主权。」

教战守则

白羊座总是害怕别人说他反应慢，所以常常未经思考，就急于表示意见，其实停下来，听听别人的意见，并不代表反应慢，反而可以表现自己的气度。

星座运势

整体运势：今年木星进入白羊座的兄弟宫，意味著今年白羊座在人际沟通及心智启发上有较显著的表现。学习新的事物或者短期的旅行，将有助你今年的运势。

今年必须注意平时习以为常的动作，并不时检视每个行动的背後动机是否纯正，因为土星也将在四月份时进入你的兄弟宫，你的生活就会出现一些压迫性的状况，让你不得不去面对它。

表面上看来你的人脉拓展将无往不利，同时让你的心灵注入一股新的活力，但如果过於强力的改变自己的本性去迎合别人，或过度坚持自己的主张，却会让你陷入进退两难的窘境。

爱情运：由於天王星和海王星均在你的朋友宫，木星和土星在你的兄弟宫，暗示你和情人将聚少离多。

但，对於仍单身的白羊座来说，这却是好消息，因为今年你们将有分活络的社交生活，爱情能从中滋长。

工作运：你必须开放心胸接受所有的可能性，并随缘吸收各方的讯息，不妨出国考察或再进修。想换工作的话，最好在年初就决定，另外，有到国外工作的可能性喔！

财运：今年白羊座财来财去，留不住有形的钱财，和亲友间容易发生财务纠纷。最好能够将你的钱财做更好的分配和运用，或者将钱投资在自己身上，让自己有更多的心灵成长，对你未来反而有好处。
