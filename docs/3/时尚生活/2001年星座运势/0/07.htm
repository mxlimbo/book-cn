星座爱情

狮子座：爱现火爆急惊风

狮子座所象征的，正是源源不绝的活力。你会看到病狮子、懒狮子，可是不容易见到缺乏创意的狮子。因为狮子座的活力，是心智上的灵感，这一点和水瓶座、天秤座倒是相似。

很多狮子座给人家爱现又霸道的感觉，他喜欢「被注意」的滋味，因此积极地走向幕前，射手座和双子座会认同这种感觉；狮子座的守护星是太阳，免不了有点君临天下的霸气，这一点使金牛座和天蝎座很受不了。

而火象星座急惊风的特性，使狮子座相当没耐性。他很不能忍受冗长的会议，给他一种愚蠢又没效率的感觉，这点倒是和牡羊座有共识。

慢工出细活的处女座和魔羯座，很容易为此和狮子座意见不合。双鱼座虽然同样追求完美，但鱼类的包容力，勉强可以忍受狮子的坏脾气。

与其说狮子座目中无人，倒不如说他们是把没有自信的一面，用高贵的言行包装起来，这一点是比巨蟹座划算多了。

如果你的身边有一只狮子，记得尊重他，一句适度的赞美，他可以为你鞠躬尽瘁。

星座上班族

第一印象

狮子座是一个野心勃勃的角色，他会很热心于公司的各种福利，去参加各式各样的活动，庆生、爬山、郊游、社团等等，他并不怕生，而且积极的提供很多意见想法，也不在乎别人的看法。所以在新进人员中，他算是一个很引人注目的人，因为他很懂得主动在各种场合里表现自己、贡献自己的能力，不论是财力、人力或智力，所以非常非常耀眼醒目，通常老板很快就会注意到这个人，狮子座太亮了，根本无法让人忽略。

激发其工作潜能

狮子座是一个长于开创而拙于守成的星座，这是蛮重要的，如果把一个新进的狮子座人员，安排到业务部或企划部都非常适合。以创意而言，狮子座是属一属二的，他们思考灵活，而且又有很强的实践力。

另外一点，狮子座的人业务能力超强，一般常常可见狮子座担任业务主管，他们都是从基层做起的，由于重视荣誉心，所以他在基层人员部门不愿意待很久，会尽量往上爬。狮子座的人有偏财运，业务要靠业务奖金，这种依靠奖金的制度，很适合狮子座，他是一个喜欢有荣誉、论功行赏的星座。

对狮子座而言，赞美是很重要的，鼓励和赞美的方式，比什么都有效。最重要的是，对于他的错误尽量视而不见，因为他很注重颜面，平常多夸奖他的优点，多忽略他的缺点。你一旦不小心打击到他的信心，或忽略掉他的优点，令他产生难以下台的感觉时，他就会觉得这个地方待不下去，在这个部门很没面子，因为他总觉得别人永远记得他出糗的那档事，其实我们早都忘记了。因此要多多让他获得尊重，对他就能起很大的激励作用。

其实狮子座是一个很简单的星座，他做任何事情就像小孩子一样，他要有赞美的糖果才肯做，你只要给他一个头衔，他就会效忠领袖，牺牲奉献。狮子座的另一个简单特质是，他永远要当一个高高在上的强者，但又不希望让你察觉他有一种目的，他喜欢人家来求他、拜托他，当他身为部属时，会很讨厌去讨好上司，认为升官发财全凭自己的实力，所以如果你能主动地照顾他，给他一个名誉、一个地位，给他表现锋头的机会，那么他很快就被收买了。

隐藏其弱点

狮子座的人最不适合进财务部门，他的财务观念很差，只重视消费不重视储蓄，懂得开源不会节流，狮子座的财务概念是：「我只要钱赚得多，这些都是小钱，不算什么。」问题是以一个公司而言，预算是件大事，一定要妥善控制好预算，而粗心的狮子座不仅缺财务观也缺预算观，这种人一进财务部，公司就岌岌可危了。

狮子座的人也不适合任职于人事部门，他比较率性，常以主观操控一切，有时候待人会有任性的反应，所以，喜欢他的人会非常欣赏他，而受不了他的人一刻也待不住，这是他的大问题。

让自己跑在他前面

面对狮子座属下，你不但不能讲他的缺点，而且不能涉及到任何人身攻击，他对这种事情很敏感，若是你经常批评他做得不对，或是当众纠正他的错误，甚至经常封杀他，不准他发言，不采纳他的意见，对他的表现视而不见、听而不闻，如此久而久之，他的士气就会低落，萌生去意。

知人知面也知心

狮子座是一个光芒四射的人物，当他一进公司你就会注意到他，这个人穿著得体，很会打扮，风度优雅，待人热情慷慨，很轻易就可以拉拢到很多同事的心，而且很容易就能发挥他的影响力，成为公司里的新领导者。

当狮子座的人进到一家新公司时，通常会破坏原有的生态平衡，跟原有的领导者产生冲突，在一个团体里原本就会存在一些小圈圈，有一些小小的、非主管的领导者，他们是同事里面具有影响力的人，但是粗心大意的狮子座一踏进来，便会忽略掉这些既有的领导者，虽然他们并不是处心积虑的争取领导权，但自然而然的就产生领导风范。

领导人的特质之一，就是你要好面子、出手阔绰、常常请大家吃饭，这种事情做多了，就会形成领导主权、意见领袖。每个人的人性中都有贪小便宜的心态，往往都会跟着那个花钱大方的人走，所以狮子座很容易就变成一个迷人又显眼耀目的人物，如愿以偿地跻身领袖地位，但是这通常也使他变成引起别人排挤的对象。

小心敌人就在你身边

狮子座同事做事很积极，有冲劲又有活力，很多时候会主动去做事，主动去分担，主动去学习，大多数的人都会觉得，狮子座同事蛮讨人喜欢的，他总是表现得很光明正大，因此他的竞争力在于，他很懂得在适当的时候表现出自己的才华，这样他的能力很容易就赢得别人的赏识。

任何事当老板交代下来，询问有没有人会做的时候，他通常会第一个举手说：「我来做做看」、「我来试试看」，相当有勇气去尝试、去冒险。

狮子座的最大优点是生性乐观，很少有灰心气馁或钻牛角尖的时候，这是他的强势竞争力之一。另外，狮子座的人没什么心眼，很多的女同事会因为小心眼遭受排挤，可是狮子座的女同事，通常都能跟异性相处得很好，在无感情压力之下，任何异性都会喜欢大而化之、开朗型、可以当哥儿们的女同事，相对的，在同性之间可能就不这么吃香了。

以彼之短衬己之长

狮子座的人做事效率很高，但由于非常粗心，所以错误率更高，品质很低，经常缺少画龙点睛那一笔，他可以把重点掌握得很好，但无法兼顾细节，再加上不是每个人都受得了狮子座的火爆脾气、自以为是的高姿态、绝对主观和霸道，他认为「是」就是「是」，应该「这样」就是「这样」，你只要能掌握高品质、善用良性沟通，就能略胜一筹。

狮子座的人常因过度的自信和坦率而冒犯别人，而且是得罪人却不自知。他在同事中常干扰到别人的领导地位，又不懂得敬老尊贤，欠缺辈份资历观念，主观地认为谁的能力强，谁懂得自我表现，谁就是领导者。刚开始他可能挺受欢迎，但在团体中，不见得是会像双子座那么有人缘，也许他们对外的人际关系会很好，但在自己公司之间的人际关系，就不那么转得开了。

狮子座的人最专精于业务和创意职位，要尽量避免让他有被注意的机会，例如在会议程序里，将整个流程及时间安排得很紧凑，不要让他有自我表现的机会。狮子座的人拥有一个极佳的强火力武器，就是当他站上舞台的那一刻，在镁光灯下，他的缺点都会变成优点，即使只有三分能耐也能展露成五分，他非常擅长于自我表现，切记，切记！千万不能给他有机可乘，上司就不会注意到他，这是比较不擅于表达的星座所应该学习的，你要安排让自己大秀特秀，全然杜绝他的机会，因为狮子座是钻石，光芒很亮，很容易被发现的。

成为最佳工作拍档的绝招

狮子座的人很好玩、很注重享受，你只要慷慨大方的对待他，他也会慷慨大方的对待你，常常请他的客，跟他一起出去玩，邀他一起参加娱乐活动，让他进入你的小圈圈，给他表现的机会，帮他制造出锋头时机，多利用各部门的各种比赛活动来拉拢他、抬举他，推崇他成为一个领导者，他很快就会靠近你这个圈圈，成为你这个派系的人。

共享双赢成果的秘诀

狮子座很讨厌小里小气的人，因此你必须先付出，让他觉得你不是一直在占他的便宜，尽管他不是很计较别人占他便宜，但他很讨厌那种你明着、暗着利用他的感觉。狮子座的人生观是当冤大头可以，但是要让他知道你到底想要什么东西，千万不要暗着来。狮子座本身光明磊落，他看不起那种阴险的小人，你要光明坦诚、摆明的讲，我想要借用你的什么东西。

嘴巴愈甜，脸皮愈厚是愈能掌控狮子座，不须交换或付出代价，只要送他一顶「高帽子」戴，愈高效果愈好，例如夸赞他是该部门最行，能力最强的人，所有的事情都一定要来请教他才可以，或说「公司若是没有你，早就关门大吉了」，「你是元老功臣」等等，只要口头上的推崇，就能得到大丰收。

拢络人心的秘诀

你最好是一个勇敢而有担当的人，狮子座的人会很欣赏你，他不屑于胆小怕事、软弱无能，然后爱在别人背后说坏话的阴险小人，所以今天即使你跟他吵架，正面起过冲突，只要在事后你勇于跟他认错道歉，他就会很欣赏你这种大公无私的坦荡荡作风。只要你是一个是非分明，肯承认错误的人，而不是满嘴谎言，死不认帐的小人，就能博得他的好感。一旦你得到狮子座的欣赏，他会大力替你广告，在别人面前毫不客气地推荐你。

小心自暴其短

与狮子座相处，你会被他们的坏脾气、急性子气死，但是唯一的优点，就是他们不擅长于放冷箭。狮子座喜欢当面向你挑战，他们觉得偷偷摸摸告状、扯后腿的事非常丢脸。

只要你顺着他的毛摸，切记不可在别人面前损他，狮子座是个超级好相处的星座。他希望每个人都喜欢他，只要你不存心讨厌他，他一定会尽量喜欢你，让你高兴。放冷箭实在不是狮子座的专长。

狮子座常常为了追求掌声与喝采而逞强，与人争辩；害怕自己不受重视，所以忙着表现自己。奉劝狮子座：偶尔谦虚一下，是不会矮人半截的！

星座运势

整体运势：今年是你的理想和人生目标确定的好时机。如果你一直很用心，会带来你一直努力想得到的结果。但要多注意身体健康，特别是急性胃肠病。生病时千万不要拖。

心态上你必须做调整，因为有很多时候你将只是你生活圈中的配角，若你能处之泰然，将是你人缘极佳的一年。社交活动增强，心情可因此而变得愉快。且可能会与某种人物相识而成至交。

你的运势走到了一个被许多人需要的地步，有时你可能会觉得很烦，但只要你能接受这种转变，将会带给你前所未有的感受。

爱情运：正在寻找爱情的人，此时能够吸引异性，并获得爱情，但是要避免纵情於享乐。要常给对方一些惊喜，但不要过於滥用想像，反而会让你沈浸於虚幻之中。

工作运：完美主义的你，想用所有可能的方法来改造你周遭的环境，如果你的眼界很广，你将会把这个企图化身在广泛性的社会重整。由於你的社交活动分活络，只要拿出你诚信温馨的一面，往後的事业、财运方面便能无往不利。

财运：不要随便乱花钱。注意跟亲朋好友之间的金钱纠纷，请客、红色炸弹将使你两袖清风，不过由於你的慷慨大方，也使你赢得了友谊。此外，你也有机会去赚取一些外快。
