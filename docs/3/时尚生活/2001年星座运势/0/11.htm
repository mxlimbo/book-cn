星座EQ

射手座在火象星座中，算是较能控制情绪的，比起狮子座、牡羊座，处理情绪问题的方式也潇洒得多。

因为射手座较理想化，所以他的情绪问题多数来自于对现状的无力感。也因为控制情绪的能力较佳，所以发脾气的频率较低，也会在情绪过后自己去试着解决问题。至于生气的目的，则是希望藉此改变令他不满的现况。

星座爱情

射手座：不记恨的开心果

木星是射手座的守护星，它象征个人的理想，也可以说是「超我」。这种带有哲学意味的特质，使射手座的族群多少有点悲天悯人的倾向，虽然自己不能让世界大同，但总是喜欢帮助别人，或排解纠纷。

另一方面，木星理想化的特质，也让射手拥有乐观的思想，凡事都能看得开。双鱼座虽然不强求人生，但是比较悲观。这种开朗的人生观，和牡羊座颇谈得来。

射手座追求公平，但不记恨，这一点比天蝎座、巨蟹座更宽容一些，与天秤座比较接近。

如果你有烦恼，别忘了找射手座的朋友谈谈，他们乐观的天性，以及俯瞰人生的本能，一定能解开你心中的千千结。

星座上班族

第一印象

你会觉得射手座是一个很不甘寂寞的人，在团体里面意见很多，花样也多，像猴子一样爬上爬下，问题一箩筐。他不见得是像牡羊座那样爱现，可是他会一直举手，频频发问，让人总觉得这个人很有趣，跟他在一起好好玩，又讲笑话又逗趣，甚至提出一些可笑好玩的建议来。通常射手座的人不太沉得住气，有什么事情马上就表达，而且他不计较利害得失，常常考虑欠周全，偏偏又容易妥协，有时又冲动不计后果，讲一些不该讲的话，让旁人为他捏一把冷汗，而他还浑然不自觉。

激发其工作潜能

射手座在创意、企划、设计上能发挥的空间都很大，尤其做企划时，那种天马行空的水准和大胆的见解，以及他对整个的人事布局，全盘翻新的构想，非常非常令人吃惊。而关于海阔天空的创意专长，或与外面接触的开创业务能力，射手座都能胜任愉快，凡是具有挑战性的、活跃的工作他都喜欢。

射手座很容易建立很好的人际关系，因为他不怕生，事情都会想得很正面，很愿意主动去跟人家建立交情，算是人缘蛮好的，所以像公关这类需要跟人接触的第一线工作，也挺适合射手座的。

大部分的人喜欢找射手座诉苦，因为他很乐观，很正面，让别人觉得讲完话心情都很愉快、烦恼经射手座一点，全部云消雾散了，这也是射手座颇讨喜之处。

隐藏其弱点

射手座的组织能力不够细腻，往往不适合待在财务部，也不适合当秘书。他的个性不够坚持，常会做牺牲短打，也不能当福利委员。他做事随意，无法按部就班，这种沉不住气的粗线条，也不适合去刺探商业机密。千万不能派射手座去卧底，他很容易被收买，别人跟他讲几句好话，引起了他的好奇心，他就心软，很容易相信别人，以为全世界都是好人，天真又乐观。

射手座也不适担任有时效性的工作，他可以做一件长远的规划，但无法完成当日时效性的工作，因为射手座很好玩，经常玩昏头。射手座大多缺乏时间概念，容易误事。

让自己跑在他前面

射手座的人行事不稳定，常常为了玩而改变计划，给人好逸恶劳的感觉。你可以在适当时机表露出，自己是一个稳定有担当，而且不会因为玩乐享受而放弃公事的人，这就是打压他的好方法。

知人知面也知心

射手座同事经常不见人影，他喜欢出差，到处串门子、主持正义，又好管闲事，什么活动都要参加，很热心，蛮讨人喜欢的。而且他很慷慨大方，不计较金钱，又随和，没什么太多意见，常常跟着人家一起起哄，所以通常人缘蛮好的；只不过他也常会讲话得罪别人，不小心揭露一些别人不愿意听到的事实，这时候就叫人咬牙切齿了。

小心敌人就在你身边

射手座同事感觉很敏锐，但不像天蝎座那样常常摆着一张神秘的脸孔，射手座大都是以开玩笑的方式来说出事情，所以没有人会相信他的第六感，不把他的话当真。他的洞悉能力很厉害，例如开完一个会，他便知道下一波升迁的是何人，因此他很快明了自己该如何应对进退，既然已经退出这场战争，反正升迁无望了，他马上宣布放弃，于是开始吃喝玩乐。

射手座最大的竞争力，来自于他能够俯瞰整个公司，所有的状况看得一清二楚，什么时候该表现会有希望，什么时候该保守明哲保身，他对整个公司的潮流抓得很准，因此不会为了权力斗争而徒劳无功，也不会介入利益纠纷，因为他非常明了何时该进何时该退，所以他不容易真正得罪别人。

他也许常在言词上说错话，但久而久之大家还是会喜欢他，了解他就是大嘴巴的个性，而别人通常也不会很怕他，认为这个人好象都无所谓，事实上他是很清楚自己的角色。

射手座的人有创意，很敢做、有魄力，在团体中属于能够排除万难，去做一些事情的人，而且他所提出来的建议都很大胆，很有远见，常常做一些前无古人的创举，令人耳目一新。

以彼之短衬己之长

射手座是个欠缺恒心与毅力，一遇到挫折时，就想换一条路来走，人生目标换来换去的，对理想非常不坚持的星座，而且个性又散漫，常常说得多做得少，或是说完以后根本不做，推诿给他人，懒得执行，一味推卸责任。射手座光说不练的成分很大，如果你的实践力比他强，这就是你能够表现自己的地方。

在现代化的管理里，预算是很重要的，偏偏射手座没什么预算概念，也欠缺数字观，他可能提出一些漫天喊价就地还钱的建议，或他所提的企划案，到最后预算被删到只剩尾数。另外一点，射手座喜欢用钱来解决事情，只要做好就好，总觉得钱不是问题。只懂得开源却拙于节流，所以个人也不容易致富。

射手座同事的另一个大毛病是粗心，譬如在他们完成的企划案里，可以挑出一大堆错字，或者文字不通顺。还有就是不专心，经常顾左右而言他，然后讲到一半就话题一转把原来的主题断掉了，很多老板极不满意射手座这个毛病，一件事情都还没做好，又提出新的提案来，花样太多，怎么去执行呢？

成为最佳工作拍档的绝招

方法之一，是找他出去吃喝玩乐，这种事他最有兴趣，而且越刺激越好，越冒险越好。

方法之二，是满足他的好奇心，射手座的好奇心是多方面的，包括知识性的、趣味性的、生活中的、理财、两性……各式各样的好奇心。

他对所有不知道的事情都很好奇，你若能提供他点子、信息，满足他的好奇心，让他可以花钱，可以撒野，可以增广见闻，舒散身心，他都会很开心。

共享双赢成果的秘诀

基本上射手座是一个很慷慨的星座，不管你跟他熟不熟，他都不会那么现实，把事情分得很清楚，这是你的，那是我的。射手座一向不会很计较，所以共享资源根本不是问题，而且他很乐于提供他拥有的一切，相当乐于与人分享，是个非常大方的星座，即使你不回报他也无所谓，射手座的天真，也因此而很容易被人家所利用。

拢络人心的秘诀

射手座本身没什么心眼，开朗大方，很够朋友也喜欢帮助别人，他最讨厌不够义气，在背后出卖别人那种耍阴险、玩弄小手段的小人，尤其是爱讲他人坏话、凡事锱铢必较、心胸狭窄自私自利的人。射手座喜欢有话直说，不要在背后搞文章，任何事明说都好办。虽然如此磊落，射手座总是很容易在背后被人家放冷箭。

射手座的人喜欢把人跟事分开来谈，今天你如果对他有任何不满，可以到他面前去理论一番，但事后大家言归于好，交情归交情，事情归事情，不要因为一件事做得不好，而影响到你们的交情，然后在背后随便批评他，令他有被背叛的厌恶感。有什么事说清楚，一切好商量，如果先暗中搞了一大堆小动作再来沟通，那就甭谈了，他会很率性地「呯」一声，把你关在门外，表明他对你的厌恶。

小心自暴其短

射手座很喜欢结交各个不同部门的同事，天生有着常人所不及的乐观态度，再加上激发、开朗与富有正义感的个性，使你可以很容易的和别部门的射手座同事打成一片，他们喜欢与心胸开朗的人做朋友，若你因工作上的事而找他一同讨论，他们也会很乐意，因为射手座是一个心胸很宽大的人。

想要赢得射手座的好感，就得避免使自己成为一个孤僻、内向又喜欢自以为是的人。射手座尤其讨厌喜欢阿谀奉承、拍马屁的人，如果你因为想讨好其它部门的射手座同事，而向他那个部门的上司拍马屁，看在射手座的眼里，他反而会瞧不起你，不屑与你交朋友，产生适得其反的效果。

教战守则

射手座的眼光远大，却总是看得远，而做得少。别忘了，就算有再好的创意点子，若不确切执行，依然只是个点子而已。

如果今年男射手结婚的话，你的伴侣较可能年纪比你稍长，或比你更早在社会打拚。你对这个人的敬重，是她吸引你的主要原因之一。

爱情运：你的爱情，相当顺利，将获得恋爱的乐趣。男射手尤其容易碰到年龄较大的女蝎子。你渴望艺术家的生活，幸运的话，可享受知性和美感交融的快乐，使你的生活充满艺术质感。

财运：财务易出现波动，也可能碰到把你顾倒的顾问。亲友间金钱往来频繁，免不了遭受红白炸弹之殃。不论如何，动作愈多，就愈要如履薄冰，不要急著做决定，用心找个可靠的财务顾问吧！
