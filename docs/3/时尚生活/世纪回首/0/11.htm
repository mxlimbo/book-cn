二十世纪世界十大影片

　　新千年来临之际，英国最著名的《电影爱好者》杂志在影迷中进行了一次本世纪最佳电影评选。结果《星球大战》第一集、第二集分别荣膺第一名、第三名，斯皮尔伯格70年代拍摄的《大白鲨》获得亚军。《电影爱好者》评先的十部最佳影片是：

　　1.《星球大战I》（Star Wars）　导演：乔治·卢卡斯
　　影片讲述了银河帝国推翻了银河共和国，丽亚公主和卢克等人奋起抵抗，最后战胜帝国军队的故事。《星球大战》是美国历史上票房收入最高的10部影片之一，全球盈利达4亿多美元。影片还在第50届奥斯卡金像奖的角逐中一举夺得了最佳艺术指导、最佳服装设计、最佳音响、最佳视觉效果等6项金像奖。卢卡斯在运用先进技术的同时，也十分注重影片中的精神力量刻画，该片对“力”的描述带有神秘的东方色彩。

　　2.《大白鲨》(Jaws） 　导演：史蒂文·斯皮尔伯格
　　《大白鲨》是好莱坞传奇导演斯皮尔伯格70年代拍摄的作品，是灾难片的鼻祖和恐怖片的经典。影片的故事十分简单，讲述的就是3个勇敢的男人如何追猎一条杀人的大鲨鱼的故事。然而在70年代，正是这部电影让观众真正领略了什么叫“惊心动魄”。正是这种经典的悬念技巧使得《大白鲨》至今仍拥有大批的观众。

　　3.《帝国反击战》（The Empire Strikes Back） 　导演：艾瑞恩·凯斯纳
　　《帝国反击战》是《星球大战》的续集。该片的主要内容仍是卢克、丽亚公主等人与反扑的银河帝国的斗争。影片推出后再次获得了巨大反响，受欢迎程度比起前一集有增无减。较之第一集，《帝国反击战》更具气势，特技场面的运用更为高明，场面的精美也有过之而无不及。

　　4.《肖申克的救赎》（ The Shawshank Redemption） 　导演：弗兰克·达拉邦特
　　成功的银行家安迪被指控谋杀了自己的妻子，并被判无期徒刑。在肖申克监狱中，安迪并未绝望，他获得了其他犯人的尊敬。在朋友莱德的帮助下，安迪终于实现了自己的目标，由于有了安迪，莱德对生命意义的追问也有了答案。

　　5.《盗亦有道》 （Goodfellas） 　导演：马丁·斯格西
　　《盗亦有道》是斯格西最出色的电影。影片表现了美国的爱尔兰－意大利裔黑社会分子的生活。主人公亨利的两个同伴一心要往黑社会上层爬，以前只是小偷小摸的亨利也随之受到很大的压力。《盗亦有道》具备了优秀影片的所有元素——好的剧本、难忘的音乐、出色的演技、精彩的对白。斯格西在此片中对黑社会的塑造堪与《教父》媲美。

　　6.《低俗小说》（Pulp Fiction） 　导演：昆丁·塔伦蒂诺
　　这部拍摄于1994年的影片是昆丁·塔伦蒂诺继《落水狗》后的又一力作，影片总共包括4个部分，充满暴力场面和救赎意味，职业杀手、拳师等人物在各个部分穿插出现。影片富于活力，是对观众感官以及美国人常挂在嘴边的“政治正确”的一个挑战。

　　7.《教父I》（The Godfather）　导演：弗朗西斯·福特·科波拉
　　《教父》被电影评论界誉为自《公民凯恩》以来最出色的美国影片。全片通过一个局外人－－黑手党新一代首领迈克的妻子凯的视角，表现了纽约黑手党科莱昂家庭的种种内幕。该片在1972年第45届奥斯卡金像奖的角逐中赢得了最佳影片、最佳改编剧本与最佳男主角3项大奖。影片气氛紧张、风格雄浑和凝重，具有了慑人心魄的画面冲击力和造型效果。

　　8.《拯救大兵瑞恩》（Saving Private Ryan) 　导演：史蒂文·斯皮尔伯格
　　《拯救大兵瑞恩》是中国观众十分熟悉的影片，该片以1994年6月6日盟军诺曼底登陆为背景，讲述了美军一支8人小分队深入敌后寻找失踪的普通士兵瑞恩的故事。斯皮尔伯格在他的历史题材电影中关心的并非历史事件本身，而是充满人性和道德感的“小人物”的故事，以及个人在不可抗拒的人道主义环境下表现出善与恶。影片中深深的人道主义精神和真实的战争场面再现使得该片成为美国历史上票房最高的10部电影之一。

　　9.《辛德勒名单》(Schindler’s List） 　导演：史蒂文·斯皮尔伯格
　　表现德国投机商人、纳粹党党员辛德勒冒死拯救犹太人的《辛德勒名单》代表了斯皮尔伯格的最高艺术成就。该片曾获第66届奥斯卡最佳影片、最佳导演、最佳改编剧本等六项大奖。身为犹太人的斯皮尔伯格把影片拍得既具艺术性又有观赏性。他大胆地用黑白色调来表现故事，使得黑白和彩色之间的反差十分强烈；其熟练的电影语言用也相当出色。

　　10.《泰坦尼克号》（Tianic） 　导演：詹姆斯·卡麦隆
　　《泰坦尼克号》大概是90年代最火的影片。该片获得了第70届奥斯卡最佳影片，最佳导演，最佳摄影等11项大奖。20岁的男孩和17岁女孩在泰坦尼克号上持续了三天的疯狂爱情让整个世界也随他们疯狂，这是商业动作的成功。

(摘自：中国文化报　达巍编译)
