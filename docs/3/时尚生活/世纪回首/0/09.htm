二十世纪世界娱乐界十大人物

　 电影天才——卓别林

　　卓别林的喜剧所带给人们的快乐和思考，触及到了几代人的内心深处，他在默片时代的伟大贡献，使他的深远影响跨越了国界。据统计，至今全世界已经有3.5亿人看过卓别林的每部喜剧片；历史上没有任何一位喜剧演员像卓别林那样被那么多同行模仿；不仅观众和同道对他赞美、倾慕，而且文艺界巨擘们也给予他高度评价，譬如肖伯纳就赞誉卓别林是“电影业造就的惟一天才”。更可贵的是，卓别林数十年的从影生涯和积极的生活态度，使他从一个普遍的人道主义者成长为一位批判现实主义大师和坚强的反法西斯主义战士。卓别林曾经声明，他不是“为电影”而拍电影，他相信笑和哭的力量，他努力的目的是带给人们欢乐。卓别林用行动实践了自己的诺言，他的电影已经成了当今人类的文化瑰宝。


　　“现代舞蹈之母”——邓肯

　　邓肯为20世纪开了一个好头，她存在的价值和意义决非一个“现代舞蹈之母”就能论定。她是旧时代与新时代的桥梁，同时又将平民与贵族的注意力集中到一处。她使古希腊雕塑的各种造型复活，并为古典的芭蕾注入新的冲动。她最早的舞蹈教师是“风和波浪，还有那鸟和蜜蜂和展翅飞翔”。舞蹈是邓肯个人对各种感官的和感情的刺激所做出的反应：它在瞬间内将一种情调，无论是理性的，还是非理性的，都变成了动作。她是这样教导她的学生的：“用你的灵魂去倾听音乐。当你倾听的时候，难道没有感到某种内在的自我正在你的内心深处觉醒吗－－正是这种力量使你仰起了头颅，举起了双臂，向着光明缓缓地走去”。

　　邓肯是上帝的奇迹，她与叶赛宁的恋情，她与红色苏维埃的情结，她的恐怖而神秘的死去，无不与这个时代息息相关。


　　卡通巨子——沃尔特·迪斯尼

　　当1996年12月15日他去世的消息传来时，迪斯尼电影公司的股票一反常态地涨了10点多。这并不是源于人们对他所遗留下的事业有更加兴旺的期待，而是因为失去他的约束之后，他的事业可以拆散了赚钱，几乎人人有份。

　　迪期尼到底为美国、为世界带来多少欢乐与利益，他的机智谦和、他的兄弟之爱和动物之情似乎就决定了一切。于是便有了真人爱丽丝周围的一群动画动物以及“幸运的兔子”奥斯瓦尔多；以后就是“米奇”和“唐纳德”，伴随它们的猎狗普路托和古菲。在每一个孩子的成长道路上都曾有过《白雪公主》、《美女与野兽 》、《幻想贡》、《灰姑娘》和《木偶奇遇记》的阶段。商人如果使用迪斯尼的名字或他的动画形象要交一笔数目不菲的使用费，政治家们在讲话中经常提到“米奇”的名字等于义务为迪斯尼做广告。当然，一切梦想的最终实现，还是“迪斯尼乐园”的建成，它既使人得到欢乐，又能获得知识。


　　伟大的音乐制作人－里格

　　瓦尔特·里格是20世纪古典音乐唱片历史上最伟大的制作人，唱片业能够达到目前如此繁荣的局面，与他的开创密不可分。他早年主要从事音乐评论，1927年开始了与EMI公司长达半个世纪的合作，他在经营、发行和销售上常有不同凡响的手法，这使得公司的业绩十分显赫。最值得称道的是，他把握时机、敏锐识别音乐人才的卓越能力。1945年第二次世界大战硝烟未息，他就一人周游欧洲大陆，抢先与无数当时仍然隐居的艺术家签约；1946年由于他的只身斡旋，促使联军总部解除了被指控为纳粹战犯的卡拉扬的演出禁令，这使得卡拉扬在感激之余，一口气和EMI签下长之达10年的专属演奏合的。在里格的运筹下，当时的EMI公司名家云集，为以后音乐唱片业的大步发展打下了良好的基础，终于使录制唱片与现场演出在20世纪获得了同等重要的地位。


　　“欧洲的音乐总监”——卡拉扬

　　卡拉扬几乎可称是20世纪古典音乐界的“皇帝”，他被公认为是本世纪最受欢迎的大指挥家，还被称为“欧洲的音乐总监”。卡拉扬曾执掌柏林爱乐乐团34载，气势之盛，无人可比，此外卡拉扬先后担任过英国的爱乐乐团、维也纳爱乐乐团等最重要的乐团指挥，在欧洲乐坛占据着举足轻重的地位，可以说，他在世之时，有着顾盼自雄，傲视整个乐坛的态势。卡拉扬指挥的德奥音乐作品，到今天几乎都已成为这些作品的经典诠释，他的名声不少来自他所指挥广泛曲目的录音，据不完全统计，他指挥的录音有800多件，包括最著名的贝多芬9首交响曲，布鲁克纳、勃拉姆斯和舒伯特的全部交响曲。虽然对于他也有着极为两极化的评价，而且他与纳粹关系的阴影跟了他一生，但这些都不足以影响他在各类指挥家的排行榜上始终名列第一的地位。

　　“展现了歌剧艺术魅力的女神”——卡拉斯

　　卡拉斯是20世纪女高音歌唱家中的泰斗。从声乐艺术来讲，她与世纪歌王、意大利的男高音歌唱大师卡鲁索共同代表了人类声乐艺术的最高峰。在可预见的未来，恐怕也没有人能够超越过她。卡拉斯以对角色深刻的理解和细致而富于创造性的处理，使得听众进一步领略到歌剧的音乐美，她那一次传奇的《托斯卡》演出，至今成为歌剧乐迷不能忘怀的记忆。卡拉斯一生演出歌剧百部以上，音量幅度极宽，从轻巧的花腔女高音到最壮实的戏剧性声部都能胜任，她能有机地把富有强烈戏剧性和俏丽灵活特点的花腔唱法结合在一起。她还善于形体表演，具有崇高隽永的雕塑美，是世界公认的全能女高音歌唱家。她最杰出的贡献，在于她复兴了古典美声歌唱学派的精华，同时在她的舞台实践中达到了声乐与戏剧的全面平衡。可以得出这样的结论，即卡拉斯已经不单纯是一位歌唱家，在她的身上展现了歌剧艺术的全部魅力。

　　20世纪流行音乐的鼻祖——猫王

　　20世纪流行音乐的鼻祖无疑该属“猫王”－－艾尔维斯·普莱斯利。是猫王永远地改变了流行音乐的模式。他的唱片销售量至今已超过了100亿张，超出历史上任何一位歌手。他的单曲中有56首全球销量超出100万张。他在美国创下18首冠军曲的纪录，英国则是17首，全世界估计有10亿人观看过他的现场演唱会。作为20世纪的娱乐先锋，他还主演过33部电影。时至今日，还有些猫王的狂热歌迷不相信他已经死去，依旧梦想着猫王重返舞台的神话。但是，除了以上这些，他给了全世界的年轻人一年声音：“猫王－－永垂不朽的传奇人物。”


　　性感明星——梦露

　　20世纪初，好莱坞开创了电影界“明星制”的先河，到1948年，玛丽莲·梦露凭借一张裸体照片跻身影坛，成为一个头脑简单的金发女郎的性感象征。“绅士们更爱金发女郎”“如何嫁给百万富翁”“女人最爱的是珠宝”，梦露这个孤儿院中长大的女孩搔到了人们浮华之梦的痒处。1962年，梦露服用过量安眠药而死，又成为一个在好莱坞种种压力下毁灭的敏感的、生活动荡的象征。她的轻佻变成凄美，她留下的谜团使之进一步成为“商业明星”。梦露和她的裙子因此也就成为美国象征的一部分，她向全球提供关于偶像制作的价值：死人也好，活人也好，演员也好，政治家也好，只要足够有名，就有无尽的商业价值。这就是所谓“名人文化”。


　　创造双重财富的电影天才——斯皮尔伯格

　　20世纪最成功的娱乐行业是电影业，电影业里最成功的无疑是好莱坞，在好莱坞，斯皮尔伯格便是最为成功的导演和制片人，从1975年执导的《大白鲨》开始，好莱坞25税最卖座的影片中有6部是由他执导的。其中包括《侏罗纪公园》、《夺宝奇兵》、《外星人》、《辛德勒的名单》、《天地大冲撞》、《拯救大兵瑞恩》等等，无不惊天动地，使世界为之疯狂。斯皮尔伯格以其居于电影界的赫赫业绩，成了整整一代人流行文化的风向标，而他自己也被称为“创造双重财富的天才”。


　　“当代音乐娱乐界的首富”——安德鲁·劳埃德－韦伯

　　音乐剧是20世纪娱乐界最亮丽的风景，但真正的奇迹却是一位年轻的英国小伙子创造的。他的作品《万世巨星耶稣基督》、《群猫》、《歌剧魅影》、《星光快车》、《日落大道》以及《歌与舞》和《爱薇塔》所创下的种种纪录，使音乐剧的历史在最后1／4世纪重新改写。他曾名列《戏剧周刊》杂志编写的美国戏剧界最有影响力的100人之首。英国女王授予他爵位也是为了表彰他对戏剧的贡献。他是他自己作品的经纪人，所以在美国和英国以及全世界，任何一个时候，都至少有他的十几个巡回演出团在城市里演出他的音乐剧，他的“真有用”公司就是一个印钞机，如此说来，他被称为当代音乐娱乐界的首富也就不足为奇了。

(摘自：中国文化报)
