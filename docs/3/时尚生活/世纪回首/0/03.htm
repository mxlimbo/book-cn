二十世纪中国十大娱乐形象

　 知名度最高的中国卡通形象——孙悟空

　　明末作家吴承恩笔下的精灵，这位从石头缝里蹦出来的“美猴王”聪明机智、武功高强，以七十二般变化和翻驾斤斗云著称。因大闹天宫被如来佛压在五指山下，后经观音菩萨点化，伴唐僧西天取经。一路历经磨难，降妖除魔，最终取回无字真经。《西游记》的流传使得孙悟空成为中国最著名的艺术形象之一，其“大闹大宫”“三打白骨精”等英雄事迹被广泛传播。本世纪，它作为本土艺术形象的杰出代表频繁出现在舞台艺术及影视作品中，是知名度最高的中国卡通形象。

　　机智幽默——阿凡提

　　从新疆少数民族民间传说中走来的他骑着毛驴，机智幽默，能够解答各种难题。聪明的他用智慧惩恶扬善，嘲讽以巴依老爷为首的权贵，为贫苦的老百姓带来欢乐。

　　寻求心理平衡的——阿Q

　　让阿Q出名的是他的精神，即“阿Q精神”。精神的“精髓”在于自欺欺人、自我安慰，打不过别人时，他说“就当是儿子打老子”；被吴妈拒绝时，他说“她的脚太大”；直到被当做革命党人抓去砍头，他还能自我宽慰“二十年后又是一条好汉。”鲁迅塑造阿Q是希望从他自身反映国民普遍存在的劣根性，大师的伟大之处在于直到今天我们还可以从阿Q的身上看到自己的影子，“精神胜利法”也依然是许多现代人寻求心理平衡的一个“好”办法。


　　受苦受难的——杨白劳

　　在诸多的反映旧社会劳动人民凄苦生活的文学作品中，贺敬之的《白毛女》最为著名。歌剧《白毛女》上演时，喜儿及父亲杨白劳的悲惨遭遇曾令无数观众垂泪。饱受地主黄世仁欺凌而悲愤自杀的杨白劳身上集中体现了旧中国被压迫阶级的苦难。如今，提起他人们就会联想到被欺负、被压迫、受苦受难的人。作为一种幽默，现代人在表达自己境况不佳时也往往以“杨白劳”自喻－－“我都成杨白劳了”。

　　命运坎坷的——三毛

　　漫画家张乐平最有代表性的漫画人物，在张乐平的虚拟空间里，“三毛”流浪街头、被迫当小偷、给富人当养子、被抓去当壮丁……这个只有三绺头发的大脑袋男孩带我们流浪、从军，也让我们看到了旧中国的人情百态。

　　久难忘情的偶像——许文强

　　香港电视连续剧《上海滩》的男主角，大义大勇、英俊潇酒。和许文强连在一起的是香港影视巨星周润发，正是得益于他精湛、细腻的表演，使得许文强血肉丰满地站在人们面前。《上海滩》这部连续剧于80年代初在内地播放，当时港台文艺作品在内地还是“物以稀为贵”，远未像今天这样繁荣。《上海滩》向人们展示了“黑社会”里边缘人群的生存状态，故事惊险曲折，其中还包含了一段跌宕起伏的三角恋情，这些在当时都是非常“前卫”的，一时万众争睹，许文强也成了人们久难忘情的偶像。

　　小人物——李东宝

　　国产电视连续剧《编辑部的故事》中的男主角，他是一名摄影记者，一个机智有趣的普通男人，身上还有普通人共有的虚荣心、名利心，苦苦追求办公室的女同事，在失意的时候用精神胜利法来自我解嘲。李东宝这个形象迎合了80年代中期人们在思想进一步开放后，对“小我”的关照，不那么高尚，不那么伟大，没有轰轰烈烈的大作为，热爱的是平淡简单的日常生活。自此，摈弃高大全的人物塑模式，关注“小人物”的生存状态，在寻常百姓的日子里发现生活的鲜活与美丽，成为文艺创作的一个新的热点。

　　完美无缺的——刘慧芳

　　电视连续剧《渴望》中的女主角，“痞子作家”王朔的一个严肃作品。她的身上集中体现了中国妇女的所有美德，勤劳、善良、宽容、忠贞，这一完美无缺的形象受到了无数观众的喜爱，许多个夜晚，全国各地的人们守在电视机前，与她“同呼吸，共命运”撤下无数泪水和叹息。故事的结尾，这个善良的女人为救养女被撞成瘫痪。看到心爱的女主角坐到了轮椅上，观众们不由掩面而泣，继而投书剧组，要求“让慧芳站起来”，民意不可违，在当年的春节联欢晚会上，《渴望》剧组主要演员出演了一个小品，小品中坐在轮椅上的慧芳缓缓地站了起来，观众这才罢休，欢欢喜喜过新年。这恐怕也是来自中央电视台《春节联欢晚会》开办以来，惟一的一次以电视剧“续集”的形式作为节目。“刘慧芳”还引发了什么是新时期女性美德的讨论，人们针对这个凡事忍让、处处自我牺牲的人物争相在媒体上发表不同的意见。一部电视剧竟能产生如此大的震荡，在证明主创人员的成功之时，也成为中国进入电视文化时代的鲜明标志。

　　要个说法的——秋菊

　　明星导演张艺谋与中国第一女星巩俐的杰出作品，在这部名为《秋菊打官司》的电影里，张艺谋使用了纪实片的拍摄手段，巩俐换上了农家妇女的服饰，挺着大肚子，为了追求真实的效果，演员边说话都用方言，片中大量起用群众演员，着力追求零表演的效果，上演的效果证明了张艺谋艺术探索的成功，巩俐也以其突破自我的出色表现巩固了她中国第一女星的地位。除了创作手法上的成功之外，真正使这部电影扬名海外的是它深刻的主题，“秋菊”让人们看到了新时期的人们对公平正义的追求，“要个说法”成为当年流行词汇。

　　走哪儿火那儿——小燕子

　　台湾作家琼瑶的近期作品《还珠格格》的女主角。该剧轻松活泼，满足了压力重重的现代人抛弃说教，制造快乐的心理需求，但得到的欢迎和受到的批评一样多，出演该剧的青年演员赵薇以其活泼本色的表演博得观众的喜爱，同时又遭许多记者、业内人士的围攻，认为她的表演肤浅、苍白。一夜成名的赵薇面对媒体由茫然无措到学着应对，让我们看到90年代艺人的日渐成熟。“小燕子现象”同时向人们证明当今文艺的检验标准中，商业利益的重要地位。无论褒贬，赢得观众也就赢得了市场，尽管饱受指责，赵薇依然走哪儿火那儿，据称，她的演出计划已排至2000年1日底，真可谓红透半边天。

(摘自：中国文化报)
