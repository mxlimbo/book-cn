外国千年文化人物

　　但丁-阿利吉耶里（1265年-1321年）

　　诗人

　　但丁的史诗《神曲》乃是人类新世界的开山之作，艾略特曾经说过：“现代世界为但丁 和莎士比亚所分享，再没有第三个人了。”

　　达-芬奇（1452年-1519年）

　　艺术家、科学家、工程师

　　文艺复兴的永恒代表人物，伟大的业余者中最伟大者，在历史上留下了一个任何后来人 都无法涉足的峰顶。

　　威廉-莎士比亚（1564年D1616年）

　　剧作家、诗人

　　莎士比亚对世界各地的舞台、电影、课本以及人们日常说话的影响无处不在，无可匹敌 的语言的神。

　　路-冯-贝多芬（1770年-1827年）

　　音乐家

　　西方传统音乐最伟大的破坏者之一，面对时人的怀疑态度，他傲然答道：“我的作品不 是为你们所作的，而是写给以后时代的。”

　　列夫-托尔斯泰（1828年--1910年）

　　作家

　　一个永远置身于矛盾之中的伟大天才，他总是不断地深思着有关生活的问题，无论是在 鸿篇巨制的作品中，还是在现实里。

　　西格蒙.弗洛伊德（1856－1939）

　　心理学家

　　由于弗洛伊德强调潜意识对行为的影响力，从而拓宽了我们对人类本质与性的认识，使 我们进入了真正认识自身的时代。

　　卡夫卡（1883－1924）

　　作家

　　在他谜一般令人不安的作品中，总是力图表达非一般的生存体验，人类的疏离、孤独、 残缺的观念，以及对暴虐之国家机器的恐惧心理，后人从中读出20世纪的良知和焦虑。

　　毕加索（1881－1973）

　　画家

　　毕加索统治了一个世纪的艺术，不断尝试全新的艺术手段，并以其强有力的个性与旺盛 不衰的精力，激发着全世界人们的想象力。

　　切-格瓦拉（1928－1967）

　　诗人、摄影家、战士

　　有史以来照片在世界上流传最广的人物之一，本世纪最纯粹的战士，在把世界改造成理 想社会的长河中，他走过了一条必经之路。

　　约翰-列侬（1940－1980）

　　音乐家

　　披头士乐队主唱，一位人类灵魂的不可控制和不可抵挡的开拓者，他所代表的过去是--现在仍然是世界上最令人惊讶的音乐。

(摘自：精品购物指南 江熙)
