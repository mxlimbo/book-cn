一千年前全球城市谁主沉浮

　　在新千年来临之际，在世界各国城市化进程日益加快的今天，美国学者向我们展示了他们多年来潜心钻研的学术成果，其中包括1000年前、即公元1000年前后曾经盛极一时的全球15大城市，虽然其规模和影响与如今的大城市相比简直有天壤之别，而且有的早已淹没在漫漫历史长河之中，有的则已降格为毫不起眼的中小城市，只有少数城市至今依然闪烁着风采和辉煌。那么，当时这15大城市究竟是哪些呢？

　　1.科尔多瓦（在今西班牙）：人口45万。公元1000年时，该市是全世界人口最多、也最繁华的城市，而且是当时的文化交流中心，尤以其建筑成就、工艺水平和学术造诣而闻名遐迩。

　　2.开封：人口40万。这座位于黄河岸边的城市当时乃宋朝首都，因其临近大宋帝国的工业中心和运河网络而日渐繁荣起来。

　　3.君士坦丁堡（即今土尔其伊斯坦布尔）：人口30万。该城扼守欧亚两大洲的战略要冲，当时是拜占庭帝国的中心，也是一个重要的贸易港口。

　　4.吴哥（在今柬埔寨）：人口20万。作为高棉帝国的首都，该市当时是东南亚的政治中心，也是该帝国灌溉生产的高产稻米的主要集散地。

　　5.京都：人口17.5万。自8世纪末日本就定都于此。它是当时日本的宗教和文化中心，并以盛产丝绸制品而著称。

　　6.开罗：人口12.5万。该城是法蒂马王朝的首都，以图书馆众多和大学云集而闻名。

　　7.巴格达：人口125万。作为阿巴锡德哈里发王朝的首都，该市在公元1000年时被公认为全球的知识中心，其市政建筑、文学创作和宫廷生活等诸多方面无不折射出波斯帝国留下的深远影响。

　　8.内布沙尔（在今伊朗）：人口125万。它是波斯帝国最具进取精神的城市之一，当时还是绿松石的主要产地。

　　9.艾尔哈萨（在今沙特阿拉伯）：人口11万。该城是当时卡尔马特运动中心，该运动推崇普遍的社会平等，由什叶派穆斯林的一个激进组织发起。

　　10.安希尔瓦达（在今印度）：人口10万。与当时印度的其它许多城市一样，该城的规模和城址也随着附近河道走向与河水流量的变化而发生变化。

　　11.莱伊（在今伊朗德黑兰附近）：人口10万。该市以其精美绝伦的丝绸和陶瓷制品而闻名，被誉为是一个“美丽得令人惊讶”的城市。

　　12.伊斯法罕（在今伊朗）：人口10万。该城俯瞰肥沃的平原，盛产粮食和丝绸，其金属制品和手工地毯也颇有名气。

　　13.塞维利亚（在今西班牙）：人口9万。它是安达卢西亚穆斯林国最为富庶、最具教养的城市之一，其科学和艺术成就可谓是卓尔不群。

　　14.大理：人口9万。该城在公元986年达到了鼎盛时期。那里的优质大理石至今仍被源源不断地开采出来，以满足建筑和雕刻等方面的广泛需求。

　　15.坦贾武尔（在今印度）：人口9万。它是印度朱罗王朝的首都；拉贾国王曾在该城兴建一座庞大的石庙，专门用于供奉湿婆神。

(摘自：哈尔滨日报 郑勒力)
