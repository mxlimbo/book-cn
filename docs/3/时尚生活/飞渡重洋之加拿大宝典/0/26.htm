加国过境签证


　　凡途经加拿大前往第三国者，需办理过境签证。申请人需填写签证申请表格1张，交照处1张。加拿大方只发给允许停留2天的过境签证。

　　在加拿大如需要办理从加拿大去美国的签证，在加拿大各大城市中都有美国领事馆或签证处。你去办签证手续时，必须携带下列证件：护照及加拿大的签证； 填好的签证申请表（申请表可以事先去要，或到旅行社去取）； 护照免冠照片（38X38毫米）1张； 证明你签证到期后不会留在美国的证件（如往返飞机票等）；如果受邀请去美国参加会议或参加科研项目，带上邀请信，手续就会办得更快，更方便。

　　你最好提前1个月办理签证手续，如果你居住的地区没有美国签证处，你可以写信到距你住地最近的美国签证处要申请表。填好表格后，连同上述证件（第4、5项可用复印件）和手续费一起寄往签证处。注意，你必须用一个较大的信封，里面附上一个写上你自己地址和贴好足够邮票的空信封，以便签证处办完签证后寄还给你。
