加国移民常见问题（三）

　　21． 到加拿大后马上就能申请福利吗？

　　理论上，在加拿大移民后，如生活有困难，可以去社会服务部申请福利，但中国人几乎没有去申请福利的。

　　22．去加拿大后一定要在申请时选定的城市定居吗？

　　不用，一旦登陆加拿大后，您可以去加拿大任何地方定居。加拿大没有类似于中国的户口政策。

　　23．去加拿大后，可以去学校读书吗？

　　登陆加拿大后，您可以做任何法律允许的事情，包括读书和旅行。

　　24．移民加拿大后，对申请学校有什么帮助？

　　对于一流的大学，所有申请入学的人标准都一样，移民没有特殊照顾，但移民交的学费比留学生要少得多，而且移民还可申请加拿大学生贷款。

　　25. 需要带多少钱进入加拿大?

　　加拿大政府规定移民必须有一定的存储以支付加拿大的生活费用。一般地说，您需要至少带足在加拿大半年的生活费用。

　　26．到加拿大后何时可以去美国？

　　移民加拿大六个月后就可以申请去美国。可以申请多重签证。

　　27. 到加拿大后能享受什么样的待遇？

　　除了不能参加选举和被选举权以外，移民加拿大后能立即享受加拿大人同等待遇，包括定居，工作等等。

　　28．在加拿大居住多久才能申请公民？

　　移民加拿大后，并在加拿大居住三年（实足），就可申请加拿大公民。一般经加拿大常识考试通过后并等待大约一年就可得到批准。加拿大公民可申请加拿大护照。

　　29． 加拿大护照可以免签证去哪些国家？

　　持有加拿大护照可以不用签证自由进出美国、西欧、日本、英联邦成员国等世界上一百多个国家和地区。持有加拿大护照可以在香港合法停留三个月而不需签证。

　　30. 为什么还有魁北克移民？移民魁北克是否更困难?

　　按照加拿大法律，移民是由加拿大联邦政府和省政府共同管理的部门。但只有魁北可省的移民政策比较特殊。魁北克省对法文的要求比较高， 在魁北可读书甚至到过魁北克的都可以加分。但一般不会讲法语的人，移民魁北克不占优势。
