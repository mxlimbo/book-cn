加国移民常见问题（一）

　　1．为什么要移民加拿大？

　　在过去的几年里， 加拿大已连续四年被联合国评为世界上第一个最适宜于居住的国家。加拿大的人均期望寿命是世界上第二高的。加拿大的人均资源占有量也占世界第二。加拿大具有优美的自然环境, 良好的教育体制及世界上最好的社会福利制度。加拿大实行全民公费医疗，看病连挂号费都不要。加拿大政治稳定, 加拿大人享受充分的民主和自由。加拿大提倡多民族的多元文化，对少数民族有特别的优惠。由于良好的社会福利，加拿大治安稳定, 城乡安全, 犯罪率低，加拿大爱好和平，不当世界警察，加拿大人在海外受到保护。加拿大人讲究宽容，没有死刑。拿加拿大护照可以自由方便地进出美国和世界一百多个国家和地区，包括西欧、日本、澳洲、东南亚和香港。

　　2． 谁可以有资格申请技术移民？

　　理论上，具有特定职业技巧和经验的专业人士可以申请加拿大移民，这些技巧必须可以就地在加拿大劳工市场上转化应用。实际上，申请加拿大技术移民的人士必须具有至少大专学历和一年以上工作经验。

　　3. 为什么叫技术移民？技术移民的要素？

　　技术移民是指申请人不依靠别人担保，而靠自己的技术和才能独立申请移民加拿大。具体要素有下列10项：

　　1).年龄：21－44 岁之间，超过范围一年扣两分。
　　2).教育：至少大专毕业，一般要学位。
　　3).教育工作训练的要求（ETF）：与职业有关。
　　4).职业：由加政府根据劳工市场要求决定。基本上包括所有学科。
　　5).加拿大的工作安排：不需要。
　　6).语言能力：英文或法文。英文或法文听，说，读，写有一定水平就可以，会两种语言能加分。
　　7).工作经验：至少需要一年以上工作经验，四年以上最佳。 　　　8).地区因素：在同一时期，每个申请人的分数是一样的。
　　9).个人适应能力：申请人到加拿大的适应能力，由面试确定。　　10).亲属：如果有亲属是加拿大公民或移民，加5分。

　　4．谁可以同时申请加拿大移民？

申请人的配偶和所有需要抚养的小孩。需要抚养的小孩是指那些19周岁以下的未婚小孩。在经济上不能独立而需要父母的小孩，如在校学生和伤残人，也可以作为需要抚养的小孩申请。 因此，只要一人申请，全家人都可以移民。

　　5．聘用大律师或前移民官是否能受到特殊待遇？

　　不能。加拿大移民官对所有申请人都会一视同仁，但移民顾问能帮助您更好地回答问题和填写表格。

　　6．要不要请移民顾问？

　　如果您要提高申请的成功率，那一定要请有经验的移民顾问。申请加拿大移民不是一件简单容易的事, 移民顾问对加拿大移民规则的掌握及经验是不可能由一个申请人通过任何量的研究及阅读材料所能代替的。有些申请人往往由于一些小节的失误而造成申请案被拒。为了确保您的申请能最大限度地成功, 需要花一些钱委托一个经验丰富的顾问为您代理申请。这样，您就可以少走很多弯路, 节省很多时间, 同时增加成功的几率。

　　7．移民公司大的有名的服务就好吗？

　　一般来说，移民公司越大的，服务越不好。这是因为移民顾问要花很多时间与申请人讨论有关资料。顾客多了，自然不是所有的申请人都能及时地得到相关的服务。很多大的移民公司都只是在各地遍设分公司，在这些分公司，大多只有几个秘书小姐或销售经理应付申请人。一般这些公司都不为您做任何事情。注意，有些公司还会骗您做这做那，但实际上没有人帮助您办理，或帮助您的人水平还没有您自己高。 所以，判断一个移民公司的好坏主要是看那个经常与您打交道的咨询顾问。（1）他有没有能力，是否在加拿大居住过？（2）他是不是加拿大公民？（3）他有没有时间帮助您？很多大公司都有一两个专门顾问，他们的任务是在全国各地宣传其移民公司。但他们只是忙于在台上对大伙演讲，根本没时间帮助你，而真正于你打交道的人都是些水平还不如您的销售人员。这就是所谓的"挂羊头，卖狗肉"现象。

　　8．怎样评价移民咨询顾问的能力？

　　评价一个移民咨询顾问主要是看其自身的教育素质和经验。最理想的顾问应该是那些在国外长期学习、生活和工作，又熟悉中国国情的回国留学生。外籍人士，特别是有移民、人事和销售经验的，虽然也可能会提供好的服务，但其不懂中国国情是一个大缺点。因为中国和外国的情况不一样，例如中国的职业和加拿大就不一定相同，所以辅导起来就不知从何下手。但现在大部分移民公司雇用的都是一些秘书或销售员一样的职员，一般只要求大专文凭。对国外没有多少感性的认识。大部分申请移民的人都有资格当这些人的教师。他们怎么能胜任移民、留学咨询呢？

　　9．是否需要护照？

理论上，申请加拿大需要有效护照，而且护照必须是因私有效护照。但实际上，中国大陆公民可以先提出申请，再办理护照。

　　10. 如何请我们代理申请移民？

　　以下是请我们代理申请的步骤：

　　（1）免费评估。
　　（2）签约立案: 双方有意合作的话，我们可以签约，双方签字后生效。收到第一笔订金后, 我们就可以为您正式办理。
　　（3）准备材料: 准备材料十分重要，有些申请人的材料准备需要的时间较多。材料可能要在申请人和我们之间来回审查。待双方满意后再向移民局递表。
　　（4）等待期间：我们会指导您怎样有针对性地学好英语。并为面试做准备。另外，我们还会为某些申请人联系加拿大的工作。
　　（5）IETLS 考试和面试：如接到移民局面试, 或考IETLS通知, 我们会为您做有关辅导。对面试辅导，我们将有针对性地作适合于您个人特点及加拿大具体情况的面试问题解答。有很多其他移民公司的人都采用我们的面试辅导，成功率非常高。通过了面试，移民基本上大告成功。
　　（6）体检和补充材料：我们将指导您体检和补充移民官所需要的材料。
　　（7）拿到签证：我们将指导您做赴加准备。
　　（8）移民后的服务：我们将免费提供您去加拿大以后的生活和工作咨询，包括第一天如何度过？怎样找人接送？怎样找旅馆？怎样租房等等。
