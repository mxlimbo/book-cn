　　高丽大学的前身是李勇义（Yong-ik Lee）于1905年5月5日创建的坡孙私立学院。当时朝鲜正处在巨大的动乱时期，朝鲜的独立正受到威胁。李先生认识到，对新一代领导人的教育是朝鲜独立和发展的唯一出路。当时，这是仅有的一所完全由朝鲜人自己创建并管理的高等教育机构。虽然它面临着许多困难，但它的规模和重要性却不断提高。在1919年3月1日的独立运动中，该校成员发挥了突出作用。

　　1932年，金孙松（Sung-soo Kim）接管了处于严重财政困难中的学院，他在汉城的阿那地区修建了一个新校园，即现在的综合大学所在地。

　　1946年，即朝鲜解放后一年，金孙松建立一所具有4年制教程的综合性大学，实现了广大朝鲜人珍藏多年的美梦。坡孙学院被吸收入这所学校，并正式使用“高丽大学”这一校名，8月，学校举行了开学典礼。1949年，授予了第一批学士学位，成立了研究生院。

　　经1955年4月和1959年4月的两次改组，学校由一个研究生院和法学院、商学院、政治经济学院、文理学院及农学院等5个学院组成。1963年12月，开办了韩国第一个商业管理研究所。

　　1972年6月，乌苏大学并入高丽大学。

　　1978年3月建成的大学中心图书馆，是韩国最大的一家图书馆。学校于1980年3月，在村光那多省的约池湾开设了一个分校。1982年10月，约池湾的分校（1987年改名为“首昌分校”分设文学院、理工学院和商业经济学院。

　　1983年7月，扩充医学院和医院的体制，重新组建了大学医学中心系统，下属4家校医院落。同年9月开放的理科图书馆，成为理工研究中心和学术信息中心。

　　自50年代起，学校一直着手扩充兴建校园，到1984年，各个学院都有了自己的大楼。

　　1986年11月成立了国际教育部，以加强与国外大学进行学术交流。最后，高丽大学被接收加入国际大学生交流会。这是一个以美国为基地的世界性大学生交流组织。

　　目前，学校有12个学院，它们是法学院、商业管理学院、文学院、农学院、政治经济学院、理学院、工程学院、医学院、教育学院、人文学院（首昌）、科学技术学院（首昌）、商业经济学院（首昌），下属84个系。除研究生院外，还有6个专业研究所，它们是商业管理研究所、教育研究所、食品与农业研究所、政策研究所、工业科学与技术研究所、社区发展研究所，承担了广泛的研究项目。此外，还有35个研究学会，经常组织研讨会、讲座等。其中如亚洲研究中心、高丽文化研究中心等，都享有国际声誉。

　　高丽大学现有学生近3万名。目前，已有11万多名学生在这所具有80多年历史的大学获得了各种学位。学校现在教学和研究人共2000多人，其中教授有600多人。学校围绕汉城有多个校区，主体校园坐落在汉城东部，距市中心约两英里。校园总面积一平方公里以上，建筑物占地面积约20多万平方米，运动场7万平方米。此外，学校还拥有50万平方米的实验农场和14平方公里的森林。

　　高丽大学的最高决策机构是管理委员会，该委员会由一名主席、12名董事和两名审计员组成。行政机构的首脑是校长，由副校长协同共同管理学校。校部行政机关有计划和公共关系部、学务部、学生工作部、普通事务部、设施管理部等。

　　高丽大学的资金来源于政府及大学校友会与个人捐赠，特别是庞大的校友会为学校的建设作出了很多贡献。此外，每个学生除缴纳一般的费用及实验费用外，还要缴纳一定数额资金给学校作为发展基金。但奖学金有140多种，奖励面可达18%。

　　学校的图书馆是全国最古老、条件最好的图书馆之一，建于1973年，现在已发展了几个分馆，包括中心图书馆、研究生图书馆、科技图书馆、联合国藏书馆、首昌分校图书馆、医学图书馆、亚洲研究中心图书馆等，总藏书量90万册，其中有不少珍贵的稀有版本；还有1万多种微缩收藏品。与本国其他大学图书馆不同的是，该校图书馆特别收藏了联合国出版刊物，自1957年建成以来，联合国藏书馆容纳了约2.5万册书及300多种期刊。学校博物馆是全国第一家大学博物馆，建于1934年，有近10万件收藏品。

　　高丽大学要求每个学生所学课程包括：必修基础课程、跨学科课程（人文科学、社会科学、自然科学）、普通课程、普通专业课程、基础选修课、专业必选课程等。必须获得所在院或系要求的最低学分数后，学生才能毕业。

　　学校每年招收一定额的特殊学员。政府雇员由所在部门推荐，外国学生通过资格考试后，也可以特殊学员身份入学，修完规定的课程，可获得一份证书。

　　自从国际教育部于1986年11月开创以来，学校已与世界各地140多所大学进行了人才和学术交流。从1985年起，学校就开始每年举办一次有关朝鲜与亚洲研究的国际合作夏季学校，为外国大学生提供机会，了解朝鲜和亚洲社会的各种学术研究及有价值的交叉文化经验。这两者在现今全球性相互依赖越来越多的社会中，对现代教育起着越来越重要的作用。国际教育部还安排学生到外国的大学暑期旅行，以加强语言训练，了解外国文化。同时也邀请各专业的外国教授前来讲学。

　　高丽大学有35个研究学会，活跃于各个研究领域，其中18个属于人文科学和社会科学，17个属自然科学和医学。得到国际公认的研究学会有：亚洲研究中心和病毒疾病研究学会，为国际学术的发展作出了贡献；朝鲜文化研究中心是研究国家传统文化最主要的研究机构；商业管理研究中心、工业技术研究学会、食品来源研究学会及合法药物研究学会、，把学术研究之风带到国家工业、农业和法律生活中去；其他研究学会在各个领域也都有可喜的突出成就。

　　亚洲研究中心　是第一个促进亚洲当代问题的共同理解和学术研究机构，如今已成为全国最大也是最有声望的亚洲研究中心。据称，由于对共产主义集团国家、亚洲的现代化、朝鲜的统一、国家与地区关系问题等的领先研究，它已得到国际学术界的承认和重视。

　　朝鲜文化研究中心　通过把朝鲜古典著作翻译成现代朝语，并开展其他研究活动，如召开学术会议、公告、收集研究文化遗物等，在公众中产生了积极的鼓励作用，在朝鲜传统文化之间架起了桥梁。

　　德国研究学会　在1976年的显著成就是发现了汉坦（Hantaan）病毒导致的肾出血热综合症。1981年，该学会被世界卫生组织指定为肾出血热综合症的病毒咨询和研究的协作中心。

　　高丽大学通过各研究机构和部门开展了社会服务活动。农场是实验和生产的双重性果园，由老师和同学培育出的许多果树新品种和生产出的水果，都低价出售给居民。样品农场也为全民造林计划和地上花园建设提供了多种样品。

　　工业技术学会在对基础和应用科学研究的同时，为工业公司提供技术指导，推动了国家科技和工业的发展。

　　商业管理研究中心通过出版刊物等，使该中心的研究和考察成果得到普及，并把重点放在对商业和工业发展设置指导和咨询服务上。

　　合法药物学会承担了工厂卫生工作人员的培训工作。

　　工业发展学会为工业公司完成各种教育计划，并进行管理指导。

　　高等工程材料及技术研究中心，帮助工业公司技术人员的再培训，以提高国际竞争能力。

　　工业科学和技术研究所，通过提供理论基础加强工业与高技术知识的联系，造就了一批具有创造力的、领导高度工业化社会的专业人员。

　　联合卫生学初级学院对公共卫生人员和医务技术人员进行正规培训，适应了快速增长并专业化的公共卫生医务事业的需要。迄今，已为国内外培养出5000多名从事各种公共卫生和医务活动的工作人员。

　　环境卫生学会开展的活动研究工业区的职业病，审查危险工作环境，对工厂的工人进行体质检查等。

　　学校拥有一批能干、活跃的毕业生队伍，他们中的许多人在国家社会生活的各个行业处于领导地位。校友会为学校捐赠修建了许多建筑物。

　　理学院的生物系通过对韩国昆虫的基础研究表明，陆地上的农作物、植物和动物每年遭受昆虫的巨大伤害，但却缺乏强有惫的抑制方法。为此成立了韩国昆虫学会，研究减少这种危害的长久之计。

　　环境卫生学会完成了对环境污染的生活监测及影响的评估。

　　亚洲研究中心举办的多次国际研讨会，吸引了美国、欧洲和亚洲其他国家的许多参加者。除创办的半年性期刊《亚洲研究杂志》外，至今已出版了145种关于亚洲事务的学术刊物。

　　学校的医疗中心拥有第一流的医疗队伍和最现代的医疗仪器及设备，下属4家医院承担教学和学生临床实习工作，同时也为公众提供医疗服务。

　　大学有许多活跃的学生俱乐部和社团，如计算机俱乐部、文学社、摄影社、学生红十字会、英语会话俱乐部等，学生可加入任何一个团体。

　　学生会每年5月5日为校庆举办大型的、精彩的花岗石宝塔节庆祝活动。从1946年起，高丽和永圣两所大学每年举行一次运动会，促进了大学间的友谊，在发展现代民族体育方面发挥了先锋作用，在韩国现代体育史上具有重大意义。

　　学校创办了许多期刊和报纸，有学生主办的报纸、英文期刊、商业管理通讯、教育通讯等，还有其他一些各系主办的出版物。

　　具有现代化设施的学生联合大楼、女生娱乐场，为学生课外活动提供了便利。学校还拥有一个在韩国所有大学中最大的运动训练中心，一个具有国际水平的溜冰场。此外，学校的两处海滨建筑在暑假都对师生开放。

　　学生咨询中心还为学生经常面临的人生、社会、感情、教育等方面的问题提供咨询服务，进行心理测试等，帮助学生解决困难。

　　学校的象征：老虎。高丽大学，是一所年轻的大学，但它的确犹如一支充满生机的老虎，向世界一流大学的目标奋进。
