　　布朗大学是一所私立、男女同校的综合性大学，创建于1764年。是美国历史上第七所、新英格兰地区第三所最早的大学，常青藤联合会成员之一。

　　布朗大学设有分科精细的众多学院。它的主要学院称为大学学院，其次是医学院。前者是一所文理性质的学院，后者为专业学院。其他的学院都不是实体，如研究生院、韦兰学院等。系和部共44个，如美国文明、考古学、应用数学、艺术、生物医学、社区保健、临床神经科学、家庭医学、心理学和人类行为、放射医学、外科学、医学、病理学和实验医学、化学、古典学、比较文学、计算机科学、东亚研究、经济学、教育学、埃及学、工程学、英语、法国研究、地理科学、德语、西班牙研究、数学史、历史学、意大利研究、数学、音乐、哲学、物理学、政治科学、心理学、斯拉夫语、社会学、戏剧舞蹈等。学校没有限制性的课表，每一个主攻方向或专业都有自己的要求，但学生可以从全校所开出的名目繁多的课程中选择超越这些要求的课程修读。大量跨专业的交叉性课程供学生选择，鼓励学生在主攻专业之外进行探索。

　　学校可授予的学位有文学士，包括人文学科、社会科学、数学、工程学、自然科学和体育学等80个领域；理学士包括应用数学、水生生物学、生物学、生物化学、生物物理学、计算机科学、工程学、地理科学、物理学、心理学、纯粹数学等；文、理双学位，两个方向可以相关，也可以独立；文科硕士、理科硕士；医学博士，采取8年一贯制的学习，前4年主要学习文理学士课程，后4年医科学习时可以同时加深某一本科专业的学习，他们在获得医学博士学位时，还有望获得某一其它专业学士以上的学位。
