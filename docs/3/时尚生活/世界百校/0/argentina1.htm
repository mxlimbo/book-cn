国立科尔多瓦大学
Universidad Nacional de Córdoba

通讯地址：Calle Ob. Trejo 242, Córdoba, República Argentina

　　国立科尔多瓦大学（以下简称国立科大）是阿根廷的第一所大学，也是整个美洲3所最古老的高等学府之一（前两所大学分别是墨西哥国立自治大学和秘鲁的利马圣马科斯大学）。该校地处阿根廷第二大城市科尔多瓦，正式创建于1622年，其历史可以追溯到17世纪之初的两所教会学校。

　　国立科大的10个系分属4个部分：

  1. 社会科学类。法律与社会科学系的法律专业学制6年，学生毕业时获律师学衔；另

    2. 外可授予法律博士学位。

  2. 人文科学类。哲学与人文科学设有文学、哲学、教育学、心理学、历史、绘画、雕

    3. 塑、版画、乐器、图书馆和档案等专业。

  3. 医学类。医学类的两个系（医学系和牙科学系）分别培养医生和牙科医师，前者学

    4. 制6年，后者学制为5年。

  4. 基础科学和技术科学。建筑与城市规划系的两个专业（建筑、城市规划）直接培养

　　建筑师（学制均为6年）。化学系则以培养化学、临床生物化学和药物化学的硕士生为宗旨。数学、天文学与物理学系共开设3个专业（数学、物理学、天文学），学制均为5年，学生毕业时获相应的专业硕士学位。

　　在国立科大培养的众多杰出的人才中，有共和国总统、政府部长、省长、参议员、法官、律师、作家、艺术家、科学家、军事家和教授。

　　国立科大的医学系和牙科学系为国家培养了大批优秀医生，其中有代表性的杰出人物是：西尔维亚·伯尔曼（他和他的父亲是母校不同时期的毕业生，他们于1958年访问中国时曾受到毛泽东主席的接见），路易斯·密里西·巴勃罗（著名外科医生，为纪念这位医术高明、著述甚丰的外科大夫，现设有巴勃罗基金会），埃德蒙多·A·阿莱斯卡诺（这位医学系毕业的优等生曾在乡镇从医近40年，1981年被阿根廷外科协会授予“外科专家”称号。1965年作为众议员曾访问过中国）。
