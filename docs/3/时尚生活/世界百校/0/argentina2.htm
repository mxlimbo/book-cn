国立图库曼大学
Universidad Nacional de Tucumán

通讯地址：Ayacucho 491, 4000 San Miguel de Tucumán, República Argentina

　　国立图库曼大学（以下简称图大）是阿根廷最著名的大学之一，创建于1914年。

　　图大的11个系是：农艺学与动物饲养学系、建筑学与城市规划系、艺术系、生物化学、化学与药物学系、精密科学与工艺学系、自然科学系、法律与社会科学系、经济科学系、哲学与文学系、医学系和牙科学系。

　　3个高等专业学院是体育学院、护理学学院和心理学学院。

　　6个中等专业学校是技术学校（隶属精密科学与工艺学系）、大学体操学校和萨米恩托职业学校（属于哲学与文学系）、农业与制糖技术学校（隶属农艺学与动物饲养学系）、音乐学校和美术学校（均属于艺术系）。

　　图大的办学特点是实行免费教育；学生入学完全根据学校的专业设置和个人的爱好；在全校11个系和3个高等专业学院开设的50个学位专业中有18个专业直接培养硕士。以下是各学术单位的专业特点。

  1. 人文科学和社会科学

    2. 艺术系共开设7个学位专业，其中造型艺术和戏剧两个专业直接培养硕士生（学制均

    为5年）；艺术、室内装璜、现代舞蹈、戏剧和音响等5个专业则分别培养教师、装饰家、舞蹈演员、教师、技术员等学衔和称号（室内装璜和音响两个专业的学制为3年，其他3个专业的学制均为5年）。

    法律与社会科学系开设律师、诉论代理和公证员等3个专业，前者学制为6年，后两者均为4年。

    哲学与文学系开设的8个学位专业全部直接培养硕士生和教师（学制均为5年），这些专业是哲学、文学、教育科学、历史学、地理学、英语和法语。

    经济科学系的学制为6年，经济学和管理科学两个专业直接培养硕士生，公共会计学专业培养会计师，其中经济学硕士学位包括7个专业方向，即经济发展与公共财政学、农业经济学、统计学与计量经济学、货币理论与国际贸易、数学经济学、历史与社会学、劳动经济学与工业经济学。

  2. 自然科学和技术科学

    3. 自然科学系开设5个专业，培养目标是植物学和动物学硕士生，地质学家，建筑师，

    大学教师，学习年限均为5年。

    农艺学与动物饲养学系的学制为5年，直接培养两个专业的工程师（农艺学和动物饲养学）。

    建筑学与城市规划系仅设一个专业，在6年时间内培养建筑师（3个专业方向是城市规划、住宅设备设计和施工员）。

    生物化学、化学与药物学系共开设3个专业，其中生物化学和药物学两个专业在6年时间内分别培养生物学家和药剂师、化学专业则直接培养硕士生（学制5年）。

    精密科学与工艺学系共开设13个专业，其中物理学和数学两个专业直接培养硕士生（学制均为5年）；9个专业（制糖技术、结构工程学、通讯工程学、水利工程学、机械工程学、工业电力工程学、电子工程学、化学工程学、大地测量学与地球物理学）则以培养工程师为宗旨，前5个专业和后4个专业的学制分别为6年和5年；土地丈量术专业培养测地员（学制5年）；物理学大专班的学习年限为3年。

  3. 医学科学

该学科包括医学系和牙科学系，分别设医学和牙科学专业，前者学制7年，直接培养医生；后者学制5年，以培养牙科医师为目标。

图大的3个高等专业学院分别培养体育教师（学制4年）、护理学和心理学硕士生（学

制均为5年）。

　　除此之外，图大在图库曼省的阿吉拉雷斯市办有大学交叉学科学院，宗旨是培养艺术和护理学两个专业的硕士生。
