瓜达拉哈拉大学
Universidad de Guadalajara

通讯地址：Av. Juárez No.975, Guadalajara,Jalisco, 　　　　　　México

　　瓜达拉哈拉大学创建于1792年，在墨西哥全国高等院校中仅次于墨西哥国立自治大学，居第二位。经过整整两个世纪的发展壮大，现在已是一所包含人文科学、社会科学、自然科学、技术科学、经济管理科学、卫生科学在内的综合性大学。

　　瓜达拉哈拉大学的教学工作分为高等教育和中高等教育。前者由24个系和2个高等学院承担，后者由中高等教育委员会负责。

　　24个系是：管理系、经济-管理科学系（设在拉戈斯·德莫雷诺市）、农艺学系、建筑学系、生物科学系、物理-数学科学系、化学系、公共会计学系、法律系、设计系、经济学系、哲学与文学系、政治、政体与国际研究系、地理系、信息与计算机系、工程学系、医学系、兽医学与动物饲养学系、牙科学系、心理学系、社会工作系、旅游业系。

　　2个高等学院是：体育运动学院和现代语言学院。
