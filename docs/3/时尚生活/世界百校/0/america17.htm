　　密歇根大学（以下简称密大）属男妇同校的州立大学，是美国10佳综合性大学之一，也是世界上主要的研究性大学之一。

　　密大校园占地2,842英亩，其中安阿伯本部占地2,600英亩，有200多幢建筑物，600多万册藏书，9个博物馆，7家教学医院，几百个实验室及研究所，以及1.2万台微机。安阿伯本部共有18个学院：建筑与城市规划学院（1931年成立）、美术学院（1974年）、商业管理学院（1921年）、牙医学院（1875年）、教育学院（1921年）、工程学院（1895年）、拉克哈姆研究生院（1912年）、信息与图书馆研究学院（1969年）、法学院（1859年）、文理学院（1841年）、医学院（1850年）、音乐学院（1940年）、自然资源学院（1927年）、护理学院（1941年）、药学院（1876年）、运动学院（1984年）、公共卫生学院（1941年）、社会工作学院（1951年）等，其中商业管理学院、牙医学院、法学院、公共卫生学院及社会工作学院均列美国高校同类学院的前5位。这18个学院共有588个专业，开设3,000多门本科课程。
