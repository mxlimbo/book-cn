爱资哈尔大学
Al－Azhar University

通信地址：public Relations, AL－Azhar University,
Kasrel Zaafran Abbassia Cairo, Egypt

　　爱资哈尔大学在埃及的首都开罗，是一所在伊斯兰世界最进步、最负盛名的大学。

　　爱资哈尔大学是由爱资哈尔清真寺演化而来的。

　　现在的爱资哈尔大学，实际上是一个包括附属中小学、附属师范学院、古兰经朗读学校在内的庞大教育系统。在这个系统内工作、学习的人员，不下数十万人。

　　爱大的行政系统很复杂，处于校一级的有大学校务委员会、校长、副校长、秘书长等。校务委员会由校长任主席，其余成员有4位分管教育、科研、外地学院、女生院的副校长，各学院的院长，政府高教部长任命的高级代表，3名伊斯兰研究方面的学者，3名大学教育方面的专家。

　　教育活动的实体是学院，爱大的各学院的名称及其分布地区如下：

　　在开罗的男生学院有：伊斯兰神学院、伊斯兰教法和法律学院；阿拉伯文学院，伊斯兰研究学院，伊斯兰圣职学院，教育、语言和翻译学院、医学院、药学院、牙科学院、农学院、商学院、工程学院等；在各地区的男生学院有：阿斯尤特伊斯兰神学与圣职学院，阿斯尤特伊斯兰教法与法律学院，门苏拉伊斯兰神学与圣职学院，塞宾·科姆伊斯兰神学与圣职学院，塞宾·科姆阿拉伯文学院，塔米尔伊斯兰教法与法律学院，塔米尔阿拉伯文学院，阿斯尤特医学院，阿斯尤特牙医学院，阿斯尤特理学院，阿斯尤特药学院，清拿伊斯兰与阿拉伯学院，阿斯旺男生伊斯兰学院等，共21所，在开罗的女子学院有：阿拉伯与伊斯兰学院，文学院，医学院，理学院，商学院等；在地区的女子学院有：阿斯尤特伊斯兰学院，门苏拉阿拉伯与伊斯兰学院等。

　　爱大就是通过这些富有个性和特色的教学实体，十分成功地担当了自己的使命和职责。

　　为了在各个不同的专门领域进行研究，像当今世界的其他大学一样，爱大建立了许多科学研究中心，其中主要有：国际伊斯兰人口研究中心，赛力·阿卜杜拉·卡梅尔伊斯兰商业研究中心，心脏外科中心，肾病中心，产妇和儿童保育中心，妇女和保健研究中心，医药副作用研究中心，病毒医学中心，小家畜集约繁殖中心，新沙漠种植中心，埃及奶酪制作标准化中心，谷物生产和存储研究中心，工程咨询中心，各种交通设施的互效单位的确定中心，妇女和社会发展中心，以及爱资哈尔大学环境保护中心等。

　　爱资哈尔被称为“伊斯兰学术文化的朝向”，她的影响不仅在中东，而且遍及全世界，正如现任校长阿卜德·法塔赫·侯赛因·谢赫（Abde Fattah Husseiny E1?/FONT>Sheikh）博士所述，这是因为“爱资哈尔的研究和学习是传统和现代的结合，他们不丢弃自己的祖先从白手起家中开创的珍贵遗产，也不照单全收故纸堆中的一切，”她不仅接受各国来此学习、研究的留学生，而且派出自己的学者前往40多个国家去传播伊斯兰和阿拉伯文化。在穆斯林世界，人手一册的《古兰经》，必以印有“业经爱资哈尔审查”字样者为准。许多学者，频频应邀出席各种国际会议。爱资哈尔图书馆，藏有许多极为珍遗的手抄本图书，虽然藏书量在埃及只能屈居第二，但是只有她的藏书，才被称为东方伊斯兰文化的精华和源泉，吸引了世界各国的学者纷纷前来拜读。据有关史料记载，1921年，中国派留学生到爱资哈尔大学学习。1938年7月，我国著名教育家陶行知受各界救国联合会委托，访问了埃及的千年学府棗爱资哈尔大学。

　　爱大学生的社会服务活动，也具有自己独一无二的特色。为了实践其传播和保存伊斯兰文化的使命、培养专业水平和组织能力，每年的伊斯兰斋月，爱大都要组织自己的学生前往各地乡村和城市举行宗教座谈会、古兰经宣讲会。他们还组织学生到麦加朝觐，到边远地区进行巡回医疗、扫盲，组织学生走出宿舍，举办夏令营等。

　　爱资哈尔大学是一所令人肃然起敬的古老的高等学府，它跨越时空、继往开业、服务于现实社会的治学精神，确是当今世界大学的楷模。
