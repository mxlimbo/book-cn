阿默斯特学院
Amherst College

通讯地址：Amberst, Massachusetts 01002, U.S.A

　　阿默斯特学院建于1821年，是美国一所著名的私立文科学院。目前共有教师161人，本科生1,570人（来自美国各州和世界各地）。师生比例约为1:10。就其人数上看，是一所“微型”高等学院。

　　阿默斯特学院共有30个系，系名按英文首字母先后顺序排列如下：美国研究系，人类学系，亚洲语言与文化系，天文学系，生物学系，黑人研究系，化学系，古典文化学系，舞蹈系，经济系，英语系，欧洲研究系，艺术系，法语系，地质学系，德语系，历史学系，法律、法理与社会思潮系，数学与计算机科学系，音乐系，哲学系，物理学系，政治学系，心理学系，宗教学系，俄语系，社会学系，西班牙语系，戏院系，性别研究系。

　　学园位于马萨诸塞州的先锋谷，占地964英亩。

　　由于与汉普夏艾学院、史密斯学院、芒特霍利奥克学院和麻省大学4所大学相距很近，所以尽管该院本身规模较小，依然具有某些综合性大学的优势。它与邻近的4所大学达成协议，互相交流学生和教员；在某些专业领域，如非洲研究、国际关系等领域共同合作，并定时有汽车往返于各校之间（免费）。学生可以在这4所学校选读任何课程并获得学位，而不需另付学费。

　　由于学院的教育面很广，每个毕业生的体会都与其他毕业生不同，成绩单上的课程因人而异，每个学生与教授和同班同学间的关系也相差迥异。

　　凡在学院学习过的人都认为若要度过一个充实的人生，文科学习是最好的准备。他们学习过各种思想体系，实验过多种思维方式，懂得质疑的方法和解决问题的途径，理解创造力和评论、批评的重要性。他们在形成自己的人生观和世界观之前，首先分析了别人的准则和观点，他们与亚里士多德、莫扎特、马克思、弗洛伊德等人文科学、社会科学领域的巨匠们进行对话，然后就其中的疑惑和教师、同学展开讨论，最终将带着所有学到的知识和受到的启踏上社会，服务于社会。凡在该院学习过的人都有一种成就感。这种成就感并不以掌握的知识、通过的考试、读完的书本、发表的文章来衡量，也不以文凭和掌声为标志，这种成就感的的确确是一种微妙的内心体验……“我变了，不再像刚进学院时那样，而变得对各种信息十分敏感，见识广博，有自制力，较为成熟”，阿院学生深知“上游无止境”，他们知难而进，锲而不舍。

　　学生的成长在某种程度上应归功于学院的精心培养。学院挑选有才华、有抱负的学生入学，给他们建议、支持，为他们提供各种资料和设备。同时，希望每个学生在大学4年中按照适合个人特点的方式挖掘潜力，走自己的路，基于这种考虑，学校开设的选修课多于必修课，不用强制性的规定来约束学生，尽可能地发挥每个学生的主观能动性。

　　几乎所有阿默斯特学院科学家和研究者都获得联邦拨款或民间资助。所以，许多现代化的研究设备是其它类似规模的文科学院所不具备的。一些高精尖仪器如电生理学放大器、激光系统、BEC?/FONT>VAX计算机等可供学生使用。

　　学院不仅在传统的文科教学方面很有特色，而且尝试着开设跨学科的新教程，一年级学生所接受的文科学习入门教育就是一例。1989一1990学年的该类教育包括8门课程：浪漫主义的启蒙运动、对其它国家艺术的理解、决策及其不定性、前景展望、记忆学、大规模绝种棗对一个科学争端的案例分析、核武器学、光学等。所有这些课程都是几门学科互相渗透的产物，如“浪漫主义和启蒙运动”由英语、德语、历史和美国研究3名教授共同执教；“核武器学”由历史、化学、政治学3名教授共同执教；“决策及其不定性”则由数学和经济学教授一起负责。

　　学院与日本最古老最著名的大学之一棗东北大学有非同寻常的联系，因为东北大学由学院1870年的毕业生约瑟夫·哈代·那斯玛（Joseph Hardy Neesima^创立，而此人是日本第一个在西方受高等教育的人。
