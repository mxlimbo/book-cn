坎皮纳斯天主教大学
Pontificia Universidade Católica de Campinas

通讯地址：
Rua Marechal Deodoro, 1099-13020 Campinas,
SP, Brazil

　　坎皮纳斯天主教大学正式创建于1955年，它是根据巴西管理教育的最高权力机构棗联邦教育委员会1955年6月15日的决定，在原有的几所学院的基础上建立起来的。这所私立大学是巴西较年轻的大学之一，但知名度却相当高，原因之一是它位于巴西的硅谷坎皮纳斯市并为该市的高科技事业作出了贡献。

　　坎皮纳斯天主教大学的学术机构分为3个部分：生物与保健科学、精密科学与技术科学、人文科学。

　　在生物与保健科学领域，有1个学院和3个系：生物科学学院（以生物学专业为主）、医学科学系（共开设药物学、医学、营养谠、理疗学、职业病治疗等5个专业）、护理学系（以护理学专业为主）和牙科学系（以牙科学专业为主）。

　　在精密科学与技术科学方面，有两院两系，它们是：精密科学学院（数学专业）、信息学学院（系统分析专业）、建筑与城市规划系（以建筑学和城市规划专业为主）和技术科学系（有土木工程和卫生工程专业）。

　　在人文科学领域，共有5个学院和6个系，学院包括艺术与传播学院（开设艺术教育、新闻学、广告学、公共关系和旅游等5个专业）、人文科学学院（主要专业有社会科学、地理学和历史学）、哲学学院（以哲学专业为主）、文学院（开设文学和3个学士学位专业：语言、秘书和翻译）、心理学学院（以心理学和声听觉学等两个专业为主）、神学与宗教科学学院（以神学专业为主）；系科包括图书馆学系（以图书馆等专业为主），经济科学、会计与管理系（共开设管理科学、会计学和经济学3个专业），法律系（以法律专业为主），教育系（设教育学、师资培训与特殊教育、师资培训与幼教等3个专业），体育系（以体育专业为主）和社会服务系（设社会服务专业）。

　　硕士生专业有图书馆学（信息系统设计与管理）、哲学（教育哲学与科学哲学）、教育学（高等教育方法论）和心理学（临床心理学与学校心理学）。专业学位专业是市场管理、系统分析、法律、企业经济、人力资源、保健卫生的规划与管理，学校与学习心理学、图书资料自动化系统、外科与美容等。
