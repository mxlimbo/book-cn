科英布拉大学
Universidade de Coimbra

通讯地址：Paco das Escolas 3049, Coimbra Portugal

　　科英布拉大学创建于1290年，是葡萄牙最古老的大学，也是伊比利亚半岛上历史最悠久的高等学府之一，仅比西班牙的第一所大学－萨拉曼卡大学（创建于1218年）晚成立70多年。该校所在的古都科英布拉位于首都里斯本以北，1139-1260年曾是葡萄牙的首都。

　　1990－1991学年，科英布拉大学共拥有7个学院，它们是文学院、法学院、医学院、理工学院、药物学学院、经济学学院、心理学与教育科学学院。同期在校学生为16050名，80年代末的教师人数超过千人，其中获博士学位的教师约300名，研究员近百人。

　　科英布拉大学正在进行的科学研究分为以下4大类：

  1. 人文科学和社会科学

    2. 该学科领域的研究课题计15项，其中较重要的有《葡萄牙文化与政治史》、《葡萄牙社会史》、《葡萄牙文艺复兴人文主义研究》、《古典文化研究》、《葡萄牙教育史》、《教师进修培训教育心理学》、《欧洲法学》、《企业法》、《经济刑法》和《比较法律》等。

  2. 保健科学

    3. 正在开展的25个科研课题几乎涉及实用医学的主要学科，如《全国眼科病理学统计》、《白内障外科水肿研究》、《古巴硬毛鼠弯曲杆菌研究》《肝昏迷发病机理》、《微生物研究》、《食品化学、水和公共卫生》、《药用植物研究》等。

  3. 自然科学

    4. 全校共承担21项自然科学研究任务，如《葡萄牙孢粉学目录》、《动物系统分类学》、《植物细胞电子显微镜检查》、《细胞生理学与神经化学》、《葡萄牙内陆、岛屿及原葡殖民地植物史》、《生态学》等。

  4. 精密科学和技术科学

　　28项涉及精密科学和技术科学的研究课题由这两个学科的5个研究中心承担，较主要的项目有《数学史与数学方法论》、《材料结构》、《放射物理学》、《理论化学》、《量子力学基础》、《光化学》、《自动化与系统分析》、《信息学与系统论》、《材料研究与应用》、《分子物理化学》等。

　　此外，大学还与工业界签订科研合同，使学校的科研成果直接为社会经济服务。

　　科英布拉大学作为全国优秀大学，曾为祖国培养了大批杰出人才，其中有著名诗人卡蒙斯（1524-1580年），19世纪末期优秀作家埃萨·德格伊罗斯（1845-1900年），作家安德鲁以及安东尼奥·诺贝尔、阿方索·高斯达等著名人物。此外，还为巴西输送了许多教授学者。1789年，巴西的文人志士曾要求按照科英布拉大学的模式建立一所大学，只因葡萄牙的反对，希望在殖民地兴办大学的计划终成泡影。然而，科英布拉大学仍是巴西早期大学的楷模。
