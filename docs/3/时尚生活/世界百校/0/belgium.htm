鲁汶天主教大学

　　鲁汶天主教大学（以下简称鲁大）是比利时历史最悠久、规模最大的私立高等学府，初建于1425年，当时是在教皇马丁五世（Pope Martin V）的特许下由布拉班特（Brabant）王国的让四世爵（Duke Jean IV）创建的。这是当时荷兰的第一所大学，共有4个学院：法学院、医学院、文学院和神学院。

　　鲁大按教学和研究内容共有10个学院，分属社会科学、自然科学和医学3大部，每个部由副校长下面的一位科学顾问负责。10个学院为：神学院（下设神学、圣经语言学、宗教科学、宗教法规等专业），哲学院（包括哲学研究所），法学院（下设私法系、公法系、经济与社会法系、国际法系、犯罪学与刑法系等），经济、社会与政治学院（下设社会学系、公共与国际事务系、社交系、经济系、行政与管理系等），哲学与文学院（下设历史系、希腊、拉丁与东方研究系、传奇文学研究与传奇文学语言系、日耳曼研究与日耳曼语言系、考古学与艺术史系等），心理学与教育学院（下设实验心理学系、社会心理学系、临床心理学、教育科学及心理学系等），医学院（下设生物化学与细胞生物学系、微生物、免疫学与遗传学系、生理学系、正常与病理形态学、放射学与医学成像系、内科医学系、外科系、妇产科与儿科系、神经病与精神病学系、特殊病理学系、牙科与口腔学系、流行病学与预防医学系、医院管理与社会医学系、职业医学与环境卫生学系、药理学系、化学与制药技术系、体育与康复系等），理学院（下设数学系、物理系、化学系、生物系、地质与地理系等），应用科学院（下设材料科学与加工系、建筑、采矿与地质系、电学系、机械系、工业管理、应用数学与计算机科学系等），农学院（下设应用生物学与农业生产系、应用化学与生物工业系、环境科学与土壤使用系等）。

　　鲁大的教学分3个阶段进行。第一阶段根据不同的专业，为期2-3年，目的在于为学生提供其所选择专业领域的基础训练。第一阶段结束时，学生可得到一张“候选人”证书，以获得进入第二阶段学习的资格。第二阶段长达2-4年，主要是对在第一阶段所学的专业作进一步专业化的学习，从众多的选修课中为自己确定一个学习计划。到第二阶段结束，学生一般离校就参加工作了。但现在越来越多的学生又进入第三阶段的学习，也就是进行攻读博士学位的学习或研究。目前鲁大有约850名博士研究生。

　　鲁大的教学在10个学院的51个系中进行，共设有约340个专业，3,000多门课。每学年分成两学期，第一学期从9月中旬到次年1月，第二学期从2月到5月，考试通常安排在每学期初，每年8月还为那些考试不及格或缺考的学生举行补考。

　　除了教学以外，研究也是鲁大的主要任务。从鲁大在研究上投入的众多人员及巨额资金看就可显出其特殊的重要性。

　　鲁大的研究计划由学校的学术委员会研究分委会制定后提交校决策部门决定。

　　鲁大以多种方式为当地的社会生活服务。它在新鲁汶和瓦隆两地都建立了科学领域。吸引了许多与大学有关联的企业到两地来开办公司，不仅鼓励了大学研究同工业的联系和交流，且增加了当地的就业机会，繁荣了当地的工业。

　　另外，学生服务部还设有各种学生咨询处，提供有关专业选择、法律、勤工助学等方面的咨询；外国留学生、残疾学生和已婚学生等遇到各种问题时，服务部也都会提供咨询和帮助。

　　鲁大在为学生服务等福利和医疗制度上是别的大学难以相比的，这也是鲁大的一大特色。
