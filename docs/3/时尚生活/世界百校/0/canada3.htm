滑铁卢大学
University of Waterloo

通讯地址： Waterloo， Ontario N2L，2G1，Canada

　　1956年，加拿大安大略省的滑铁卢市只有一所附属于西安大略大学的人文学院棗滑铁卢学院。当地一批有远见的教育和企业家为了扩大高等教育的就学机会，在这所学院里建立了一所开设理工科教学计划的附属学院，在加拿大首次引入合作教育计划。1957年5月，这所附属学院正式开始教学。1959年3月，安大略省立法机构批准了一项私人法案，承认滑铁卢学院为独立的滑铁卢大学（以下简称滑大）并授权它可以授予学位。在学校成立至今的仅仅30多年里，它已迅速发展成为全安大略省规模列第三位的大学，滑大之所以成为加拿大发展最快的大学，主要是因为这所朝气蓬勃的学府以其创新教学法（主要是合作教育计划的引入）和税易发展新科技（尤其是计算机技术）的办学思想吸引了众多的一流人才以及政府和私人机构的大量拨款。现在，滑大在工科和理科设施方面，大部分北美洲的大学都难与匹敌。

　　校旗：金、黑、白三色旗。

　　校训：坚持真理

　　滑大有6个教学学院：

　　应用保健科学学院。下设5个系棗舞蹈系、健康研究系、人体运动学系、娱乐与闲暇研究系、老年医学系。

　　人文学院。有如下系科棗会计、人类学、古典研究、戏剧、经济学、英语、艺术、法语、地理、德语、历史、拉丁语、中世纪研究、音乐、哲学、政治科学、心理学、宗教研究、俄语、斯拉夫研究、社会发展研究、社会学、西班牙语。

　　工学院。所有的本科生计划都以合作教育制进行。下设5个系棗化学工程系、土木工程系、电气工程系、机械工程系和系统设计系。

　　环境研究学院。是加拿大最大的环境研究学院。下设2个系和2个专业学院棗环境与资源研究系、地理系、建筑学院、城市与地区规划学院。

　　数学学院。下设5个系和1个分部棗应用数学系、组合数学与最优化系、计算机科学系、纯数学系、统计与保险计算科学系、用于工业和商业的数学分部。

　　理学院。下设4个系棗生物系、化学系、地球科学系、物理系。另有验光配镜学院。

　　此外，还有4个学院与滑大建立了联校制或附属于滑大管理，它们是圣杰罗姆大学学院、瑞尼森学院、圣保罗联合学院和康拉德格雷尔学院。

　　科研机构包括：会计研究与教育中心、应用保健研究中心、信息理论中心、分子束与激光化学中心、新牛津英语词典中心（用计算机编辑）、职业写作中心、“社会、技术与价值中心”、圭尔夫/滑铁卢生物技术中心、遗传资源中心、计算机研究所、地下水研究所、提高质量与生产率研究所、高分子研究所、风险研究所、多孔介质研究组、第四纪科学研究所、太阳热工程中心、滑铁卢人文研究团体、滑铁卢计算机集成电路制造中心、滑大过程开发中心、滑铁卢核磁共振中心，滑铁卢研究所。

　　滑大图书馆拥有一个庞的计算机系统，包括WATMARS、一个联机检索系统（WATCAT）、CD Rom数据库和其它计算机网络。WATMARS 即“滑大图书馆联机检索系统”，能对滑大及其附属学院的所有藏书进行计算机联机检索，校外用户只要拥一个终端和调制器也能直接进行检索。CD Rom 数据库则给用户提供了心理学、社会学、医学、计算机、图书馆学等学科的软件和数据库，大大方便了这些学科的教学和研究。

　　滑大的微机中心允许师生试用最新型的微机，中心还进行硬软件选用的咨询，并设法用低价帮师生购买计算机设备和软件。同时，大学还免费开办关于计算机的短期讲习班。

　　在滑大的校园里，广泛分布着各种计算机设施。一些大型IBM主机被用于教学、科研和行政管理。DEC、SUN等强有力的中型计算机系统则用于复杂的计算和图形处理。此外，师生员工还可单独使用已联网的PC机和MAC机。

　　滑大最有特色的教育计划首推“合作教育计划”。合作教育是指大学和社会上的一些单位进行合作对人才进行培养教育的一种方法。合作教育计划规定，学生每隔一定时间（通常为一个学期）交替在工作场地和校园里进行全日制的工作（是与学习内容相关的工作）和学习。这一想法最早是由美国辛辛那提大学工程学院赫曼·施纳德教授于1906年提出来的。

　　以计算机出名的滑大自然在计算机及其相关学科里的科学研究最有特色。滑大作为加拿大计算机软件设计与开发的领导者，学校的计算机科学家开发了一系列国际知名的软件，如WATFOR、WATFIV、WATBOL、SPARSPARK和MAPLE等教育软件。计算机研究所与计算机科学系的教师组建了一个中心，用计算机帮助洋洋20卷5.7亿个字的牛津英语词典建立了数据库，并将莎士比亚的全部作品存入了计算机。
