开罗大学
Cairo University

通讯地址：
Information and documentation Center,
CairoUniversity, Cairo, Egypt

　　开罗大学（以下简称开大）正式成立于1908年12月21日，是埃及和整个阿拉伯世界最古老的高等教育机构之一。

　　开大学生入学基本条件为：具有高中毕业文凭或同等学历者；身体健康，必须符合大学最高委员会和学院委员会提出的健康标准化，如未患传染性疾病等；政府职员和部分时制学生入学前必须提交单位允许入学的证明书；必须品行端正；必须按时交纳学费、医疗保险费；国籍、宗教信仰、出生日期和部队服役情况证明等。此外，决定学生是否被录取的两个基本因素是：学习成绩必须平均70分以上；地区分布。

　　开大每年向一些优秀学生提供资助或奖学金，其类别分为一般资助、特别资助和学习资助三种。

　　开大学生除学习规定的课程外，还必须接受军训。1973年的关关条文规定：军训是学生学习的一项基本内容和学生升级的基本条件之一。其目的是在青年中增强军事意识，一旦需要，可为国家提供军事服务。军训的内容包括：武器训练、格斗、军事传统与文化教育、有关军事工程的知识和训练、武术、埃及国家史和军事传统与文化教育、有关军事工程的知识和训练、武术、埃及国家史和军事史等10余种活动。训练时间一般为3周或108小时，之后，进行射击比赛，成绩优异者给予奖励。军训结束时，送给每个学生一套军服，包括衣裤、衬衫、帽子和一双黑鞋。受训时间一般是在学生预考前或一年级，多安排在暑期。

　　开大学生的业余生活丰富多彩，包括体育、旅游、社会考察、社会服务、文化艺术欣赏等活动。这些活动主要由学生联合会组织安排。联合会由各学院或学校的埃及学生组成。外籍学生和部分时制学生，只要交纳一定费用，也可参加联合会的各项活动，但无选举权和被选举权。

　　开大的学科门类很齐全，但最具特色的当属医学类、文学类和法学类，这些科类在埃及乃至整个阿拉伯世界中都受到极大的重视。其次，商学类、工学类和农学类科目在开大也是举足轻重的。在开大众多的学院和学校中，比较著名的有：文学院、医学院、口腔与牙科学校、达尔·爱-奥伦学院、法学院、商学院、理学院、药物学院、兽医学院、工程学院、农学院、考古学院、非洲教育与研究学校以及3个分校。
