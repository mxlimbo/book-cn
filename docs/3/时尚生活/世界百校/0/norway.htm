卑尔根大学
Universitetet i Bergen

通讯地址：5014 Bergen Universitetet, Switchboard,Bergen,
                   Norway

　　卑尔根大学是挪威人引以自豪的高等学府之一，它创建于1946年，在1825年建立的卑尔根博物馆的基础上发展起来的。现在是挪威规模名列第二的综合性大学。实际上，这所国立大学和第二结下了不解之缘；它位于挪威西南海岸的第二大城市卑尔根，是全国建立的第二所大学（第一所大学是1811年建立的奥斯陆大学）。

　　截止1991年，卑尔根大学共有7个学院，它们是人文科学学院、数学与自然科学学院、医学院、牙科学院、社会科学学院、法学院和心理学学院。

　　该校和许多全国性、地方性研究机构保持着友好关系，其中较重要的有汉森环境与遥感中心、挪威人文科学计算中心、挪威社会科学资料服务部、挪威组织与管理研究中心、卑尔根IBM科技中心、挪威语言与术语中心、克里斯蒂安·米切尔森研究中心。

　　卑尔根大学有着广泛的学术交流和校际联系，国际关系办公室是负责与外界合作和交流的主要机构。1977年以来，该校每年派出一定数量的教师和学生出国进修和深造，同时邀请外国学者前来讲学。目前，该校与20多个国家的大学建立了交流和合作关系，如加拿大的纽芬兰纪念大学（互派研究生和教师）、美国西雅图的华盛顿大学（互派教师）、加利福尼亚大学（互派学生）和俄勒冈大学（互派学生），法国的卡昂大学（互派师生），德国的基尔克里斯蒂安·阿尔布雷希特大学（互派教师），荷兰的乌得勒支大学（互派师生），苏丹的喀士穆大学（互派师生和合作科研），尼泊尔加德满都特里布万大学（互派师生和合作科研），坦桑尼亚的达累斯萨拉姆大学（互派师生和合作科研），哥斯达黎加大学（互派师生）。

　　卑尔根大学还参加了欧洲共同市场高等教育机构与工业合作规划（COMETT）中的化学、信息学与自然科学、环境工程学项目。

　　卑尔根大学校历是：秋季学期（每年8月15日至12月15日），春季学期（每年1月15日至6月15日），暑假（6月15日至8月15日），圣诞节假期（12月15日至1月15日）。
