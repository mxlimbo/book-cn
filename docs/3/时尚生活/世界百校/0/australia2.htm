墨尔本大学
University of Melbourne

通讯地址：Parkville, VTC 3052,Australia

　　墨尔本大学（以下简称墨大）是澳大利亚维多利亚历史最悠久的大学，也是澳大利亚建立的第二所大学。男女同校，由澳大利亚联邦政府和维多利亚州政府共同资助。1853年维多利亚州立法机关批准创建这所学院，1855年正式开办。成立初期，该校仅有3名教授和16名学生，只开设文科课程。1857年增设法学院，1860年设工程学院，1862年设医学院，1895年成立音乐学院。1904一1924年间，随着澳大利亚农业、工业、矿业及畜牧业的蓬勃发展，学校又相继成立了牙医学院、农学院、兽医学院、教育学院、建筑学院和商业学院。1881年该校文理科开始招收女生：1887年医学院开始招收女生。第二次世界大战后，该校迅速扩大，60年代成立了原子科学与核工程研究委员会和应用经济研究所，后来又新设了印度，印度尼西亚和马来西亚研究所。1991年，哈桑教育学院、维多利亚艺术学院、维多利亚农学院又并入该校，使学校实力更为强大。

　　在成立至今的137年中，该校始终不渝地遵循“为子孙后代着想”的校旨，取得了卓越的成绩。

　　墨大现有学生15652名，教师1153名，其中教授127名。墨尔本大学坚持“继承、精炼、发展知识，为公共文化、社会经济发展作贡献”的教学方针，开设大学本科生、研究生以及更高层次的教育课程。目前，共有12所学院。

　　农学院：主要培养服务于农林业和相关工业的人才，授农业科学学士、林业科学学士。

　　建筑规划学院：授规划与设计学士、规划与设计荣誉学士、建筑学士、城市与区域规划、建筑硕士、环境研究硕士、美化环境建筑硕士、市区规划硕士学位。

　　文学院：授文学士、社会工作学士等学位。

　　经济与商学院：授理科准毕业文凭、社会科学毕业文凭、数学毕业文凭、社会科学准毕业文凭和教育学士、社会科学学士学位，另外还授成人教育与培训、计算机教育、早期儿童研究、评估、信息、成人与儿童心理学、数学与数学教育、教育研究、运动与舞蹈、特殊教育研究生毕业文凭和教育硕士、教育心理学硕士学位和哲学博士、教育博士学位等。

　　工程学院：授工程学士（包括农业、化学、民用、电机、机械和制造工程学）、测量学士学位。另外还设有交叉课程，如工程与科学、工程与商学、测量与科学等，学制5年。

　　法学院：设法学士学位。

　　医学与牙医学院：授医学士、外科学士、医科学士、牙医学学士和牙医学硕士、哲学博士、牙医学博士学位等。

　　音乐学院：授音乐学士学位，另外还为其它学院的学生开设音乐课。

　　理学院：授理学士学位，另外还开设交叉课程如法律与科学、工程与科学、文学与科学等，学制5年。

　　兽医学院：授兽医学士、动物科学学士以及兽医学硕士、兽医学研究硕士和哲学博士学位等。

　　管理研究生院：设商业管理硕士、管理硕和哲学博士学位以及管理研究生毕业文凭。

　　学士学位学制为3至6年，硕士学位学制为1至2年，哲学博士学位学制为3年以上。
