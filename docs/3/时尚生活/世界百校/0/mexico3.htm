韦拉克鲁斯大学
Universidad Veracruzana

通讯地址：Lomas del Estadio S-N, Xalapa, Veracruz,
　　　　　México

　　韦拉克鲁斯大学正式创建于1944年，这所年轻的高等学府是墨西哥5大国立大学之一。该校总部设在韦拉克鲁斯州州府哈拉帕市，其他4个分校分散在该州的另外4个城市。

　　韦拉克鲁斯大学的教学和科研在5个分校按学科进行。

  1. 哈拉帕分校

    2. 哈拉帕分校是韦拉克鲁斯大学总部所在地，除校行政管理机构外，集中了31个系：土

    木工程系、药物生物化学系、电力机械工程系、化学工程系、建筑系、物理学系、数学系、企业管理系、会计学系、统计学与信息学系、考古学系、哲学系、历史系、文学系、教育学系、社会学系、法律系、护理学系、生物分析系、医学系、营养学系、牙科学系、心理学系、农业科学系、生物学系、造型艺术系、现代舞蹈系、音乐系和戏剧系。

  2. 韦拉克鲁斯分校

    3. 共拥有16个系，它们是化学科学系、土木工程系、电力机械工程系、船舶工程系、传

    播科学技术系、教育学系、企业与旅游企业管理系、会计学系、生物分析系、医学系、营养学系、心理学系、护理学系、体育系、牙科学系、兽医学与动物饲养学系。

  3. 科尔多瓦一奥里萨巴分校

    4. 设在该分校的9个系是：建筑系、化学科学系、电力机械工程系、会计学系、医学系、

    牙科学系、护理学系、农业科学系和生物学系。此外，还有两个语言教学中心、10个图书馆和两个艺术创作室。

  4. 波萨里卡一图斯班分校

    5. 该分校的13个系是：化学科学系、建筑系、工程学系、教育学系、社会工作系、会计

    学系、医学系、牙科学系、心理学系、护理学系、农业科学系、生物学系、兽医学与动物饲养学系。

  5. 夸察夸尔科斯一米纳蒂特兰分校

共有8个系设在该分校，它们是：化学科学系、工程学系、社会工作系、会计学与企

业管理系、医学系、牙科学系、护理学系（夸察夸尔科斯市）、护理学系（米纳蒂特兰市）。

　　韦拉克鲁斯大学是一所包含人文科学、社会科学、自然科学、技术科学、经济管理科学、农业科学和保健科学在内的综合大学。

　　韦拉克鲁斯大学共设有21个科研机构，即生物学研究中心、考古学研究所、造型艺术研究所、基础科学研究所、保健科学研究所（下设心理学研究中心、心理学与康复医学部、公共卫生部）、公共会计学研究所、工程学研究所、管理科学高等调研所、经济社会高等调研所、人文科学研究所（下设教育研究中心、历史研究中心、语言文学研究中心）、法律研究所、生物医学研究所、法医学研究所、考古学博物馆和热带原始森林动植物园。

　　韦大著名的学科有：建筑学、工程学、生物学、统计学、信息学、心理学、教育学、人类学、牙科学、医学、护理学、音乐、现代舞蹈和造型艺术等。

　　较著名的研究生专业有：企业管理、国际贸易、海关法、法医、公共卫生、教育学、墨西哥文学、环境工程学、海洋工程学、水利工程学、气候学等。
