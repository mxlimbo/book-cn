皇家墨尔本理工学院
Royal Melbourne Institute of Technology

地址：GPO BOX 2476 V,Melbourne Vic 3001, Australia

　　皇家墨尔本理工学院于1990年7月1日正式成为一所大学，因为在这一天维多利亚大学的法规被批准生效。维多利亚大学是由福兹克雷理工学院、西部学院，以及皇家墨尔本理工学院等合并而成，而皇家墨尔本理工学院是其主要部分。

　　教育结构分为高等教育（在6大系科内进行）、技术教育与继续教育（在6大学院内实施）两大部分。

　　高等教育类的6大系科为：应用科学、应用社会科学及通讯、艺术、商学、工程、环境设计及建筑。

　　技术教育与继续教育类的6大学院为：应用科学及数学学院、建筑构造技术学院、商学研究学院、通讯及社会科学学院、电工技术学院、机械技术学院。
