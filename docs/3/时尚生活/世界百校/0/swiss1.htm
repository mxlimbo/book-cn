洛桑联邦综合技术大学
Ecole Plytéchnique Fédérale de Lausanne

通讯地址：Ecole Polytéchnique Fédérale de Lausanne, CH-1015 Lausnne, Swiss

　　洛桑联邦高级综合技术大学（以下简称洛桑高工，法文缩写为EPFL）是瑞士首屈一指的理工科大学。1853年，学校已有雏型，1889年后并入洛桑大学（当时该大学称为瑞士科学院）成为该校的自然科学部。1890年改称洛桑大学工程学院。从本世纪40年代起，学校的教学已自成体系，行政系统也有独立自治的趋势。因此在1943年，学校迁址至莱蒙湖畔靠近乌锡湾的赛德尔地区，1946年正式取得自治，取名为洛桑大学综合技术学院。它包括一所工程学院及一所建筑学院。到1969年时，该校成为联邦大学，正式定名为洛桑联邦高级综合技术大学。但原有的校舍规模不能满足教育的发展，因上该校所在州及联邦政府计划在几公里的艾居勃朗地区建立综合技术大学新校。1974年，新校舍建成。1980年以后，大部分课程都在新学校讲授。

　　今天的洛桑高工和它的近邻洛桑大学成为瑞士法语区两所具有相当水平的大学教育及研究中心。这是一所中等规模的理工科学校。拥有3,800名学生，其中1,000多名是外国留学生。学生的比例组成：数学专修班4.9%，建筑学19.8%，化学6.3%，土木工程9.0%，计算机8.5%，电子工程12.2%，材料科学3.5%，数学3.6%，机械工程5.7%，微生物11%，物理10.1%，农业工程5.7%。138名教授、1,400名研究助理及850名行政及技术服务人员来自世界50多个国家，他们的工作保证了该校成为世界一流学校。此外学校每年还邀请国外各类大学及企业界的科学家、教授前来访问和讲学。

　　教学按照系科的分类，在十一个学科进行。近年来，为了更好地改进教学，学校对微电子学、材料科学、建筑学的教学计划作了重大改进。考虑到微电子学的飞速发展和它在工程技术中的重要性，作了三个方面的改革：加强对电子技术、电子学、微电子学及信息学的教学；把微电子学的实验操作同计量学、自动机械学、机械电子学的实验操作结合成一体，以培养学生的实践能力，增强对现代科学研究环境高自动化、高技术化及多学科化的适应能力；在三、四年级，为学生提供两个发展方向的选择，那就是集成电路的设计及制造、微电子学在生产中的应用。在材料学方面的改进主要是：加强第一学年化学的教学难度，第二年教授材料物理学的知识体系，因此把教学内容更科学地分成四个有机部分：建筑方法理论、建筑环境学、建筑技术、建筑视觉表现方法；另一方面，该系提出用“教学单元”的概念，加强教学在理论演示，实际操作和研究诸方面的配合。具体而言，在第一、二学年，学生以“教学单元”为单位选择所学的课程。每个教学单元是几门课程的有机结合。二年级的教学加强了专业化水平。三年级及四年级的教学单元分别是4个小时及10个小时，它们包含了理论的演示、实际操作、土地勘探学及各科类型的研究。
