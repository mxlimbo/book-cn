萨拉曼卡大学
Universidad de Salamanca

通讯地址：Patio de Escuelas 1, Salamanca Espana

　　萨拉曼卡大学创建于1218年，这所国立大学是西班牙最古老的大学，也是世界上历史最悠久的几所高等学府之一。从建校起到16世纪末，它一直是欧洲的重要学术中心之一，与巴黎大学、牛津大学和波伦亚大学（意大利）齐名。

　　著名的神学家洛佩·巴连托斯和佩德罗·德奥斯马是中世纪时期萨拉曼卡大学的象征，而文艺复兴时期的代表则是胡安·德尔恩西纳和弗赖·迭戈·德德萨。

　　萨拉曼卡大学既是西班牙高等院校的典范，又是拉丁美洲地区早期大学的楷模，其中最古老的大学有墨西哥国立自治大学（创建于1551年）和秘鲁的利马圣马科斯大学（1551年建成）。

　　萨拉曼卡大学至今仍在国内外大学中享有盛誉。

　　截止1992年，该校拥有的学术机构为13个系、12个专业学院、47个研究室和2个研究所。

　　萨拉曼卡大学的13个系是：美术系、生物系、自然科学系、经济棗企业科学系、社会科学系、法律系、药物学系、语言学系、语言学与教育科学系、地理学与历史学系、医学系、心理学系、化学系。

　　12个专业学院（学制全部为3年）是：图书馆学与文献学学院、护理与理疗学院、护理学院（设在萨莫拉市）、企业研究学院、教师培训学院（共三个，分别设在阿维拉、萨莫拉和萨拉曼卡）、社会科学毕业生学院（设在萨莫拉）、工业技术工程学院、测绘工程学院、工业学院、阿维拉医学院。

　　2个研究所是教育科学研究所和生化微生物学研究所，前者除开展有关教育学的研究外，还独立开课，培养教育学硕士生；后者除承担13项科研任务外，还配合微生物学与遗传学研究室共同培养生物科学研究生以及微生物与植物及其相互关系专业的博士生。

　　萨拉曼卡大学作为重要学术中心，曾举办过31次国内外重要学术会议，其中14次为大型科学会议，17次为小型专题讨论会。

　　现任校长胡利奥·I·费尔莫索·加西亚曾获秘鲁的利马圣马科斯大学名誉博士学衔。

　　安德烈斯·乔尔迪·科尔沃教授获国家电台奖。

　　何塞·安东尼奥·卡韦萨斯椃讯系滤菇淌诒皇谟杩ㄋ沟倮菞莱昂委员会奖。

　　约瑟夫·佩雷斯教授获内布里哈人文学奖（为纪念西班牙人文学家、萨拉曼卡大学校友和教授安东尼奥·马丁内斯·德卡拉·内布里哈而设的人文科学奖）。
