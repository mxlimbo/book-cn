格勒诺布尔第一大学
Grenoblé Université Joseph Fourier de Sciences, Technologie, Medecine

通讯地址：Batiment Administratif,
621 Avenue Centrale Domaine Universitire,
Saint Martin d’Hères Gieres, France, BP 5-38041 Grenoble cédex

　　格勒诺布尔第一大学位于法国东南部的格勒诺布尔地区，全称是“格勒诺布尔约瑟夫·傅立叶科学、技术、医科第一大学”，一般简称为格勒诺布尔第一大学，是根据法国1968年为实行高等教育改革制定的《高等教育方向法案》成立的男女同校自治高等学府。

　　学校的学术结构很复杂，其主要的教学与研究机构构成如下：

  1. 宇宙和空间科学气象台；
  2. 技术科学学院。
  3. 12个学科的教学和研究单位：体育运动、生物学、化学、地质学、信息科学与应用数学、地理学、数学、机械学、医学、物理学、自然科学和技术等。

    学校的科学研究领域如下：

    数学、信息科学及信息处理。包括数论、代数及计算几何学、

　　群理论、调和分析的概率论方法（布朗运动）、微分几何及与精细物理有关的频谱分析、复数几何及及复数分析、拓扑的可计算性、形式计算、数字信息科学的数学方法等。

　　力学。包括流体力学、材料动力学、自然环境和土木工程等。

　　物质及辐射。包括核物理学、原子物理与分子物理学、量子突变、超低温状态、磁场、无序及混乱状态下的物理现象、表面物理等。

　　材料科学。包括对磁性材料、半导体及微电子材料、超导体材料、晶状体合金材料、非晶体或微晶材料、聚合物等的研究。

　　仪器和设备等的研究和制造。

　　分子及高分子化学。

　　生物及医学遗传。

　　医学。

　　宇宙、大地、海洋、大气、山脉方面的研究等。

　　学校有自己伟大的创始人是值得骄傲的，在其悠长的历史中，还培养出许多杰出人物。它有两位诺贝尔奖的获得者，其一是路易·内艾勒敖教授，他在1970年获诺贝尔物理学奖；其二是克劳斯·冯·克利金教授，他因在研究强磁场方面的杰出成就而获1985年诺贝尔物理学奖。此外，学校还有一批法国国家科学院院士。

　　这所历史悠久而又年轻的大学，正如它的创始人的名字一样，在法国、欧洲乃至全世界都有很高的声誉。而更重要的是，经过5年前的重新调整和组合，它现在正以跨世纪的速度发展着。
