马德里孔普卢顿大学
Universidad Complutense de Madrid

通讯地址：Avda Complutense s/n,Ciudad Universitaria,
28040－Madrid, Espanna

　　马德里孔普卢顿大学（以下简称马德里大学）创建于1508年，是西班牙最古老的高等学府之一，在全国29所公立大学中占有重要地位。

　　经过近5个世纪的发展壮大，这所男女合校的国立综合性大学已是全国规模最大的高等教育机构。

　　马德里大学的学术机构共分为19个系、11个大学学院、6个专科学校、20个研究所和10个专业进修学校。

　　全校的19个系分属人文科学、社会科学、自然科学和医学4大类，每个系设有教研室、图书馆、书店和复印社等部门。

　　马德里大学共有11个大学学院，其中有5个是大学基础普及教育学院，6个是大学专业学院（企业研究学院、光学学院、统计学学院、护理与理疗学院、社会工作学院、图书馆学与文献学院）。学院的学制通常为3年，主要从事大学基础教育。

　　马德里大学的研究所除承担研究项目外，还开展培养硕士生的专业教学工作。全校的20个研究所是教育科学研究所、犯罪学研究所、现代语言与翻译研究所、环境科学研究所、统计学研究所、人权研究所、拉蒙·卡斯特罗维霍眼科研究所、胚胎学研究所、宗教研究所、药品研究所、比较法学研究所、肉食品科学与工艺学研究所、人力资源研究所、工业药学研究所、梅嫩德斯·皮达尔历史研究所、应用磁学研究所、奥林匹克运动科学研究所、妇女问题研究所、俄罗斯文化与科学研究所、无线电通讯研究所。

　　西班牙作为欧洲共同体成员国，积极参与共同体资助的欧洲大学生流动规划棗伊拉斯漠规划（ERASMUS）。马德里大学除支持和参与ERASMUS规划外，还同下列学术交流项目有密切关系，欧洲共同体学分转移规划（ECTS）、新技术教育与培训规划（COMETT），增进外语知识规划（LINGUA）、青年学者交流规划（YES）、欧洲大学流动规划（TEMPUS）、职业供求传播系统（SEDOC）、促进经济科学规划（SPES）、共同研究中心（CCI）等等。
