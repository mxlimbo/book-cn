早稻田大学

地址：东京都新宿区西早稻田1-6-1
电话：03（3203）4141
传真：TOKYO（202）8583
电报：2323280 WASEDA J

　　它的名字来源，正如它的校徽图案：棗两束稻穗之间嵌上“大学”二字所显示的意义，早稻田刚建校时并不在繁华的都市，它的前身是日本明治时代著名政治活动家、自由民权主义者大隈重信1882年在东京郊区一片稻田里创办起来的东京专门学校。建校时，由于国内政治派系纷争，从大隈重信当时的处境看，不便于插手学校的事务，校长由大隈英磨担任。当时仅设有政治经济、法律、理学及英语四科，规模很小，设备异常简陋。只有一、二年级学生80名，11名教员，主要建筑物有二层楼的教室、宿舍各1幢和食堂。1890年增设的文学科为以后成为早稻田大学雄厚实力的文学部奠定了基础。1902年建校20周年时，正式改名为早稻田大学，成立了大学学部和专业学部，以前设立的政治经济等四科置于大学学部之内。几年之后又设置了商学科、高等师范部，学校教育规模和机构不断扩充，已超出了专科学校的范围。1907年，学校体制改革，废除校长--学监制，实行总长-校长制，高田早苗为校长，大隈重信就任首任总长，他将自己家庭原先借给学校使用的大量房屋等地产全部捐献给学校。后来，原有的学科均上升为学部，增加了理工方面的一些技术学科，成立了研究生院，扩大招收研究生。至此，从机构设置和学科规模看，早稻田大学已初步具备了综合性大学完整的体系和学科类别。在这期间，由于日本发生关东大地震和第二次世界大战，早稻田大学受到严重的破坏，但它还是顽强地保存下来。

　　战后，早稻田大学获得了新生，清除了军国主义教育思想，办学规模也大幅度扩展。在新的学制下，早稻田大学为适应战后大规模经济复兴的需要，在扩大全日制招生规模同时，开办了夜校班、函授班。研究生教育亦蓬勃发展，许多系科都设立了博士课程。学校机构也有了相应扩充，设立了体育局、国际部，加强同海外交流和选派接受留学生。各种各样的研究所也似雨后春笋般建立起来，终于成为今日具有5万学生、3000教师、9个学部，12个研究中心和研究所以及配套齐全的附属学校的一所规模宏大的综合性大学，成为名符其实的日本“私学之雄”。

　　早稻田大学现有的机构与规模如下：

　　学术机构

　　早稻田大学现有9个学部：政治经济学部（政治、经济），法学部，第一文学部（文、史、哲），第二文学部（美术、文艺、东西方文化），教育学部（教育学、英语、国语、社会学、心理学），商学部，理工学部（理学、工学等共13个学科），社会科学部（夜间制），人间科学部（人基础科学、人类健康、体育科学）。研究生院包括8个研究科：政治学研究科、经济学研究科、法学研究科、文学研究科、商学研究科、理工学研究科、教育学科研究科、人间科学研究。12个研究中心和研究所：社会科学研究所、语言教育研究所、产业经营研究所、现代政治经济研究所、人间综合研究中心、日语教育研究中心、情报科学教育研究中心、环境保护中心。此外还有全校共同教育中心、学生活动中心、校史编辑室、戏剧博物馆等共同利用设施为全校教员和学生提供学术研究活动。

　　除了9个学部之外，还有行使教学任务、职能等同于学部的机构如体育局、国际部、日语研究教育中心、语言教育研究所、情报科学研究教育中心。它们为教学、研究合二为一的单位，聘有专职教员，开设固定课程；为本科生、研究生、留学生乃至教职员授课。

　　在早稻田大学的学部系科中，大多数是传统学科，几乎所有系科都能招收研究生，其中尤以政治经济学部、文学部、理工学部最为有名，多年来成果累累，培养的学生质量誉满日本。

　　政治经济学部　历史最为悠久，它是该校前身东京专门学校时就有的学科。从东京专门学校到早稻田大学演变的几十年中，经过数次的学制改革，涌现出了不少改革志士，如协助大隈创校的小野梓、高田早苗、天野为之等。在以后的发展壮大过程中，政治经济学部学术成就更加辉煌，学子更是层出不穷。仅在政界，该学部的毕业生中就涌现了日本参众议员90余人，这样的业绩在世界大学中是罕见的。

　　文学部　的前身是东京专门学校时期坪内逍遥博士等人创建的文学科，后成为早稻田大学的文学部。1949年大学改制，为区别于夜间制与日间制而分为第一、二文学部。在以后艰难的岁月中，它以坚韧的精神，独创的学风不断前进，在学术、教育、戏剧、电影、广播、新闻、出版、艺术等领域培养出了一大批小说家、歌手、俳句艺人、评论家等。文学部分哲学、史学和文学3个系，各种专业19个，拥有900多个授课班级课。

　　理工学部　是该校最大的学部，它亦起源于东京专门学校。当时为理学科，只有8名学生。早期只设工学几个专业，以后逐渐增设了数、理、化等理科。现在共有13个系，已有毕业生6万余名。在1962年早稻田大学建校80周年时，理工学部已从原校本部区迁往新建的大久保校区。

　　人际科学部　是早稻田大学建校100周年时建立的一个适应世界潮流的全部学部。它一改过去那种分门别类的学习和研究方法，从整体、综合、运动的观点来研究人类如何适应环境，包括生命过程、健康保健、体育运动等。该学部设有人基础科学、人健康科学、体育科学3个系，另有人际科学研究中心，学部位于东京都外琦玉昇的所泽新校区。

　　师生

　　到1990年5月止，早稻田大学共有各类正规在校学生50,944人，另外还有函授生7,598人。在校生中本科学生44,039人（其中女生8,518）人，研究生3,146人（其中读博士课程725人，硕士课程2421人）。此外，还有留学生和附属学校的中、小学生。平均新生录取率为9.1 ：1，即9个申请者中才有一名被录取入学。1953-1988的35年间，早稻田大学共培养博士800人，硕士21,332人。为国外一些名人授予博士称号的有30余人，其中有我国的谬承志和孙平化。现在早稻田大学有40万名校友，遍布国内外。

　　早稻田大学共有教师3,230人，其中半日制教师1,984人。全日制教师中有教授794人，副教授138人，讲师55人，助教150人中大多数由研究生充任。学生与教师的比例为15.8 ：1。

　　早稻田大学的领导体制经历了校长、学长制—总长、学长制—总长制的演变过程。现在是总长、校长合一制，现任总长是小山宙丸，为早稻田大学的第13任总长。由总长、副总长及财团代表共13人组成的理事会，作为法人代表，是学校的最高权力机构，商讨决定学校重大事宜。与之平行的还有由教职员中教授组成的评议会，共有89位成员，另一委员会是商议员会，由全校职工和校友代表682人组成；此外还有监事2人。这几个委员会的任务是从不同方面对理事会制定的大政方针进行讨论、评议，监督执行，并不断提出新的修改意见和积极的建议。

　　早稻田大学是所私立大学，来自政府渠道的财政资助当然不能与国立大学相比。即使如此，早稻田大学每年仍有近500亿日元的经费预算，反映了它充足的经济实力，以1990年度为例：总收入是472亿日元，各项经费来源如下：校产收入和其它收入占9.3%，捐赠2.8%，国家和地方财政资助12.3%，募集和委托培养所得9.4%，学生学费收入66.2%。支出总数为479亿日元，各项开支比重如下：偿还债务1.3%，助学金4.1%，教学与科研经费30.1%，教员工资奖金64.5%，其它0.1%。收支方面出现不平衡，有入不敷出之感，校方正千方百计向校友和财团广泛募集。到1990年8月止，早稻田大学对外募集经费有10万多件，总额达165亿日元。

　　图书馆

　　全校图书总藏量为331.5万册（其中和文211.8万册；西文119.7万册），杂志36,349种（其中和文22,940种，西文13,409种），外文报纸152种，缩微胶卷、卡片11.8万个，唱片1.2万张。还有不少贵重图书，如一些珍本、孤本等。每年图书购置费达16亿日元。原校本部校区图书馆是日本国内最大图书馆之一，藏书140万册，面积6,800平方米，座位1,200个。1989年竣工的新图书馆，面积29,100平方米，有座位1,900个，可藏书400万册。

　　校区

　　本部校区在东京都闹市区的新宿，全校主要学部和行政机构均设在这里。另外，还有户山、大久保、所泽、本庄等9个校区。全校建筑面积1,957,661平方米。各校区除了有关部的教学、科研设施外，在本校建筑面积1，957，661平方米。各校共除了有关部的教学、科研设施外，在本部校区还有大隈讲堂、戏剧博物馆、井深大纪念厅（国际会堂）新建综合情报中心等反映早稻田大学历史和特色的建筑物。该校还准备在海外设置校区，1990年已完成可能性论证，相信不久将具体实施。

　　“早稻田大学以保全学术之独立，有效地利用学术，造就模范国民为建校宗旨”，这个优良学风，经创始人在隈重信倡导以来，经历任总长的精心扶植和全体师生的不懈奋斗，使之代代相传，不断发扬光大。当前，在这个学风主导下，校方采取了一系列措施：实行学分制，学习方式灵活多样，学习活的学问；设立体育局，统一进行全校体育课程管理，培育身心和谐的学生；利用戏剧博物馆等一些设施，对学生进行情操陶冶、培养文艺方面的才能，活跃学校生活；充分开掘图书馆的功能，使它真正起到“教室”的作用；为顺应世界潮流，相应地在教学上采用了“综合性”的改革；改革新生入学考试，除了招收学业优秀、全面发展的学生入学，也招收有一技之长者。招考方式采用多种渠道推荐与考试相结合，一改过去“一考定终身”的传统做法。如理工学部1991年入学新生中，传统考试录取的有1,100名，附属中学及挂钩中学推荐入学的有350名，推荐入学率平均达15%左右。此外，对归国子女、外国学生、体育尖子均有较为灵的录取措施。

　　另外，在教学管理改革上也有不少新的措施。1984年正式开始对学生入学考试、学籍管理、学费、选课、学生健康管理、就业指导、奖学金等方面进行系统化管理，采用现代化的手段，做到明确、快速。

　　学术研究

　　在自然科学技术方面，早在1964年就开始了机器人制造和使用的研究，研究水平走在世界大学的前列，最近又设计了一种能模拟人类动作，为人类服务的机器人。在高分子合成方面，人工脏器的研制有的已进入临床阶段，人造血液亦已进入临床。建筑设计上，理工学部自己设计了18层的理工学部大楼，运用“人与自然和谐统一”的新理论和指导思想成功地规划和设计了新颖别致的人间科学部的新所泽校区。

　　人文科学和社会科学的不少学部在日本高校中更是闻名遐迩，最大成就是在政治、经济、新闻、商业、文艺、学术思想各领域涌现出一大批出类拔萃的校友。在政界，培养的议员、内阁成员、国家公务员的数量仅次于东京大学，日本历届首相中有竹下登、海部俊树两位出自早稻大学。此外还有在野党的首脑、政治家、民主主义者多人，著名的有石桥湛三、鸠山一郎、浅沼稻一部、安部矶雄、绪方竹虎、松村谦三、川崎秀二、河野一郎等。在财界，以前曾流传有“理财在庆应”的说法，但早稻田毕业生也有不少在财界大放光彩，如丸井百货公司40年代就是早稻田人当经理，还有闻名世界的索尼公司创始人井深大就是早稻田大学1933年理工部的毕业生。在文学界，更是名人辈出，充分显示出早稻田大学的雄厚优势。就如日本文学界著名的“芥川奖”，至今早大毕业生就有20人获奖，高于东京大学；另一重要文学奖“直木奖”获奖者中，早大校友就有21人，占第1位。新闻出版界，杂志主编、新闻社社长，很大一部分人是早大学生，难怪有人说“早稻田人掌握了舆论工具”。在音乐、戏剧、电影、广播、学术思想界等更是群星闪烁，如大家熟悉的电影明星林繁久弥、古川绿波、宇津井健、吉永小百合。可以毫不夸张地说，近代以来，日本有名的艺术家、能工巧匠绝大多数都出自早稻田大学。

　　此外，早稻田大学培养了90余位国会议员，数目仅次于东京大学，他们在国会中形成了声势浩大的“稻门会”；1990年国家公务员工级合格者占第2位；全国大企业中担任重要职务的有13,003人，占全国大学的第1位；1989年司法考试合格者共有95人，为大学的第2位；至今获日本学士院奖的人数是唯一能进入前3名的私立大学，其中法学、政治学占第2位，哲学第3位。

　　根据教育宗旨，大隈重信倡导了3个精神：在野精神、进取精神、庶民精神。建校之初，大隈重信及其伙伴，如天野为之、坪内雄藏、小野梓、浮田和民、松平康国、盐泽昌贞、金子马治、中岛半太郎、高田早苗等，坚守这个办学思想，不仅在学校行政政治理过程中，同时在对学生学业传授时，都有意向学生灌输和熏陶这个优良传统，使之成为有别于其它大学的独有特色。建校20\25周年纪念时当时校长高田早苗再三明确大隈重信的“造就模范国民”方针：早稻田大学不仅仅要培养留芳于世的英雄豪杰，更重要的是培养胸怀大志的一流人物。时经80年之后，前任（第12任）总长西原春夫更加强调这一宗旨：“早稻田大学不是以培养国家上层领导的少数尖子，而是以培养分散在全国五湖四海的、为各个地方普及传播文化学术的、从广大地方基层支持和发展国家的人才为培养目标。”早稻田大学创办以来的100多年间，就是靠这种传统和精神吸引了成千上万的青年来校求学，造就了数不胜数的身于民众之中、以天下为己任准备一展其才的“模范国民”。

　　“东西方文明调和”，这是大隈重信又一远见卓识的办学思想。早稻田大学最早吸收欧美学术文化，向亚洲各国开放门户，吸收各国留学生在日本众大学生中始终走在前列。早在我国清朝时代，就接受了中国多达800名留学生。而今天为沟通留学生与日本人民的交流更是作出了自己的贡献。早稻田大学集中了来自世界14个国家地区87个学校1,000名留学生，数目之多在日本大学中首屈一指。同我国的北京大学、复旦大学、南开大学、上海交通大学建立了学术交流关系。

　　十分重视校友工作是早稻田大学的又一传统。今天，早稻田校友已遍及世界各地，世界各大都市差不多都有“稻门会”，大大提高了早大的知名度。日本全国的早大校友会设有46个支部，以下分布于区、市共有270个机构，海外也有25个，加上各种专业行业的稻门会，共有600个校友会组织。在庆祝建校100周年时，为了联络校友、维系感情、发放了数万张“联系卡片”。校友凭此卡片在母校使用设施、听课、旅社、饮食店等方面享受优惠服务。而办理“联系卡片”的手续费所得1700万日元用作学生奖学金。在向未来的发展中，早稻田大学更是以“培养世界人的世界性大学”的宏大目标高瞻远瞩、精心筹划。在庆祝建校100周年之际，西原春夫总长提出了早稻田迈向21世纪的设计蓝图，提出了早稻田大学适应国际化、信息化时代的教育和研究体制，宣布早稻田大学已揭开了“第二世纪”帷幕。小山宙丸总长上任伊始就雄心勃勃地表明：要把早稻田大学办成具有日本特征的世界通用的大学，培养世界通用的人才，100多年来的实践表明：早稻田大学全体师生员工前赴后继，众志成城，时刻准备着迈向创始人大隈重信的办学终极目标：世界的道路通向早稻田！
