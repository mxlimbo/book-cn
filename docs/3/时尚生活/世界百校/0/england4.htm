伦敦大学
University of London

通讯地址：Senate House, Malet Street,London, England
　　　　　WCIE 7HV, U.K.

　　伦敦大学正式创建于1836年，它是根据英国皇家特许成立的第三所大学。经过近160年的发展壮大，现已是英国规模最大的一所大学。

　　伦敦大学的学术机构大致可以分为以下4个部分：

  1. 大学高等学院（均为独立机构，有自己的行政管理机构）

    2. 伦敦大学学院 该院诞生于1828年，是19世纪初自由主义运

    动的产物。也是伦敦大学数十所学院中建校历史最长规模最大的学院（有学生8,000多人）。建校伊始，它就将校门向全社会敞开，不分其阶级、宗教、种族和性别，这种打破当时流行于牛津大学和剑桥大学的学术传统的胆略，对国内外的高等教育都有较大的影响，大学学院下设7个系（艺术系、法律系、自然科学系、工程学系、环境研究系、医学科学系和临床科学系），74个教学与临床研究室和5个学校（美术学校，牙科学校，图书馆、档案、信息研究学校，建筑与规划学校和米德尔塞克斯医学校），其中工程系是英国最早的系科。大学学院可授予学士、硕士和博士学位，同时开展博士后研究工作。

    伦敦英王学院 1829年经英国王室批准后成立，是伦敦大学的第二大学院（有学生6,000多名）。1985年，伊丽莎白女王学院（建于1908年）和切尔西学院（建于1891年）并入英王学院。该院的艺术、医学和自然科学等学科素享盛誉，且仍在发展壮大。下设7个系（艺术与音乐系、教育系、工程学系、法律系、生命科学系、数学与自然科学系、神学与宗教研究系）和一个附属医学与牙科学院（设1200张病床，牙科医院每年可接待10万名患者）。

    皇家理工与医学学院 皇家理工学院于1907年经王室批准成立，1988年圣玛丽工学院（建于1854年）并入后改现名。下设4个学院：皇家自然科学学院，矿业学院、城市与基尔特学院和圣玛丽医学院，前3个学院以自然科学、材料科学、工程学等理工类学科的教学与科研为主，圣玛丽医学院则是一所集教学与研究于一体的教学医院。全学院约35%的学生为研究生，还有来自90多个国家的留学生。

    伦敦经济学与政治科学学院 该学院成立于1895年，1900年并入伦敦大学。在社会科学研究领域享有国际声誉。全院学生达3,890人（含外国留学生），最高学位为博士学位。主要科目有经济学、经济史、社会科学与管理、社会统计学、人口学、国际关系、国际史、法律和社会心理学等。

    皇家霍洛威棻吹赂５滦卵г?/FONT> 该学院位于伦敦市外的萨瑞，校园占地面积达100英亩。在建校后的80年间只招收女生入学，现在为男女同校的新学院。1982年，学院投资2,000万英镑扩建校舍，1986年竣工。主要专业有计算机科学、艺术、数学、地球科学、生命科学和物理学，最高学位为博士。

    东方与非洲研究学院 成立于1916年，是欧洲地区研究亚洲和非洲问题的主要中心之一，主要课程包括经济学、政治学、历史、地理、宗教研究、人类学、语言学、艺术、建筑、音乐以及近30种语言文学课程。

    伯克贝尔学院 由伯克贝尔基金会于1823年创办，是伦敦大学最古老的学院之一。主要从事成人教育，分全日制和夜校制两类，可授予学士和硕士学位，也开设继续教育课程（颁发文凭和结业证明）。

    戈德史密斯学院 这所文理学院于1904年并入伦敦大学，有2,500名全日制学生和800名全日制学生以及4,000名成人教育学生。3/4的学生（不含成人学生）攻读学生和硕士或博士学位。

    伦敦大学的高等学院还包括教育学院（成立于1902年，1932年起隶属伦敦大学，是高等教育和研究的最大中心之一）、瓦伊学院（建于1947年，是一个集农业、园艺学和农村环境教学与研究于一体的高级研究中心）、海思罗普学院（下设神学系和艺术系，一半以上的本科生为在职人员）。

  2. 大学医学院（和高等学院一样均为独立机构）

    3. 英国医学研究生联合会 于1945年成立，两年后并入伦敦大

    学，宗旨在于发展伦敦地区的医学研究生教育。在英国医学界颇有声望。下设7个专业医学院：癌症研究所、儿童健康研究所、牙外科研究院、神经病学研究院、精神病学研究院、眼科学研究院、心胸疾病研究院。

    查林·克罗斯与威斯敏斯特医学院 1984年将查林·克罗斯医院医学院（建于1818年）和威斯敏斯特医学院（成立于1834年）合并后正式成为伦敦大学所属学院之一，可颁发学士和硕士学位。

    伦敦医院医学院 伦敦医院建于1740年，1785年改现名，现在是伦敦大学附属学院之一。医学专业招收全日制学生，临床医学课程只对牛津大学和剑桥大学开放。

    伦敦卫生与热带医学学院 建于1924年，为伦敦大学附属学院之一。下设的热带病医院是一所教学医院，主要专业包括社区卫生、传染病学、人类营养、医学人口学、医学微生物学、医学寄生虫学、医学统计学、职业健康、热带医学等。该学院的学生来自90多个国家，最高学位为博士。

    玛丽女王与韦斯特菲尔德学院 1989年将玛丽女王学院（建于1887年）和韦斯特菲尔德学院（建于1882年）合并后正式成为伦敦大学的附属学院之一。扩建工程按计划于1992年完成，它将是全校最大的学院之一。

    皇家医学研究生院 1931年成为伦敦大学附属学院之一，最高学位为博士。

    皇家兽医学院 建于1791年，1875年并入伦敦大学。可授予学士、硕士和博士学位。

    圣巴塞洛缪医院医学院 圣巴塞洛缪医院建于1123年，1662年开设第一个讲座，招收学生听课。1921年改现名并成为伦敦大学附属学院之一。最高学位是医学哲学博士。

    大学医学院还包括药物学院（建于1842年），圣玛丽医院医学院（建于1854年），圣乔治医院医学院（成立于1751年），皇家弗里医学院（建于1874年），英王学院医学与牙科学院（成立于1831年）。

  3. 高等研究院（由伦敦大学委派的行政机构管理）

    4. 巴黎英国学院 建于1894年，现为伦敦大学附属学院之一。

    设法语部（主要为英国本科生服务）和英语部（主要课程包括语言、文学、翻译和英语培训等）。

    考陶尔德艺术学院 成立于1932年，主要从事艺术史研究以及该学科的学士、硕士和博士学位课程的教学活动，研究生课程还包括服装史、绘画、壁画、纺织等。

    法学高等研究院 1947年成立，作为伦敦大学重要研究机构，是为研究生提供法学教学与研究的学术中心，同时为其他学院法律专业的本科生提供教学服务。

    古典文学研究院 建于1953年，主要从事古希腊和古罗马语言和文学的研究工作。对伦敦大学各学院和其他大学（含海外大学）的教师、研究生、学者开放，但不直接招收学生。

    英联邦研究院落 建于1949年，是伦敦大学的一个研究生院，主要从事社会科学和英联邦史的教学与研究。

    日耳曼语研究院 1950年成立，主要从事德国语言文学研究。

    历史研究院 建于1921年，是伦敦大学历史研究中心，正在力争成为世界历史学界的中心，下设美国研究中心。

    拉丁美洲研究院 建于1965年，是伦敦大学乃至全国拉丁美洲地区研究与教学机构的协调中心。

    沃伯格研究院 建于1905年，1944年并入伦敦大学。主要从事欧洲思想、文化、艺术和科学史的教学与研究，还开设哲学硕士学位课程，并负责哲学硕士和博士学位的论文答辩。

    高等研究院还包括传奇文学研究院和斯拉无与东欧研究学院（主要从事对原苏联和东欧国家的语言、文学、历史、经济与现状的研究）。

  4. 校外学院（由伦敦大学选派教师授课）校外学院是伦敦大学和其他机构共同负责的联合学院，授课教师全部由伦敦大学选派，5所校外学院是：

　　犹太学院 建于1855年，在艺术系共开设希伯来语、阿拉伯语、阿拉米语（为古代西亚的通用语言）、叙利亚语、犹太教法典以及中世纪希伯来语言文学。

　　伦敦商业学院 建于1965年，主要开设两年制的硕士课程和3年制的博士课程，招生对象是27岁左右有5?/FONT>6年工作经验的大学本科毕业生。

　　皇家音乐科学院 建于1822年，由音乐系教师授课。

　　皇家音乐学院 建于1883年，由音乐系教师授课。

　　三一音乐学院 建于1872年，由音乐系教师授课。

　　伦敦大学的其他主要学术机构还包括伦敦大学天文台（1929年建在市外的弥尔山上），伦敦大学视听中心（建于1968年），伦敦大学植物园（位于皇家霍洛威棻吹赂５滦卵诘貤萨瑞），大学计算机服务部（下设7个服务中心），大学海洋生物学实验站（建在苏格兰的坎布雷岛上），大学图书馆（馆藏图书140万册，期刊5,500种，此外每个学院和研究院均有自己的图书馆，其中大学学院的图书馆藏书量达90万册）伦敦大学原子反应堆中心，伦敦大学国际与地区研究中心和大学计算机中心。

　　伦敦大学是一所综合性大学，主要学科包括人文科学、自然科学、经济学、教育学、工程学、法学、医学、音乐和神学等。该校颁发的各类学位证书、文凭和结业证明均为世界各国普遍承认。

　　伦敦大学之所能发展成为全国最大的高等学府，主要是因为它始终坚守创业者所奉行的办学原则。160年来，该校稳步发展并且在许多领域开创了先河，如在全国率先建立了化学、物理学和生理学3大实验室，最先确认英语和英国文学的学术研究价值，最早开设工程学专业，于1878年在英国第一个准许女生入学攻读学位，并于1912年首次任命女教授，率先于1948年在英联邦委任一名女性副校长。
