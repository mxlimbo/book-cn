对于愿意打开这本书的读者来说，不需要拿出一个喜欢《大话西游》的理由“先”了。事实上当我们开始编撰该书时，只是觉得这样的书也许早就该面世。
　　如今《大话西游》已形成了一种特殊的文化现象。在特有的“大话”氛围中，大家交流起对这部电影的爱与心得，已经不需要更多的语言，只希望能够让大家在读到这本书时发出会心的一笑。
　　尽管钱钟书先生说吃了鸡蛋，就没必要去看生蛋的鸡，但对于狂热的影迷来说，肯定是多多益善，面面俱到才好。关于《大话西游》的文本研究与开发，有许多人从民间到网络已经做了非常出色的工作，本书的“中篇”部分便是其中的精华。而我们则更多地做了一些幕后挖掘工作，通过几次飞机旅行和越洋电话，以及几年沉积下来的资料，力争勾勒出这部电影的细节和全景。尤其令人感到欣慰的是，我们终于采访到了在众多影片中为周星驰配音的石班瑜先生――凡是喜欢周星驰电影的人都知道，这个人的声音对周星驰来说是多么重要，遗憾的是关于他的任何资讯都没有见诸内地报端。为了找到他(此前我们也和大家一样，甚至不知道他的名字)，我们付出了许多努力，好在大家都是“大话迷”，因为喜欢，所以乐此不疲，只希望这本书能够让真正的“大话迷”看着过瘾。
　　我相信许多“大话迷”都集中在大学校园，而当这本书开始筹划编写时，已是初夏时分，为了赶在这一年的应届毕业生走出校园前能够见到这本书，能够让他们将这本书作为一种特殊的纪念留给睡在上铺的兄弟，或是在以后的岁月里无法忘怀的女孩，或者就是他自己，我和几位同事努力将出版时间往前赶，为此留下了一些遗憾，甚至会使书中出现一些仓促的纰漏，请大家见谅。
　　《大话西游》永远又说不完的话题，在这本书付印时，我们也还有许多意犹未尽的想法，所以我们决定结合这本书的出版，在网络上开辟《大话西游》专题站(http://www.cinvo.com)，欢迎大家来共同探讨一下对这部影片的微妙的感情――很需要的。
　　当我拿着这本书的策划文案找到现代出版社的吴江江社长时，与他这个年龄的许多人一样，他对这部电影一无所知；但与他这个年龄的许多人不一样，他最终同意了这本书的出版计划。我们向吴先生对年轻人欣赏趣味的理解和尊重致以敬意。
　　这本书是现代出版社出版的《现代影音丛书》第一本，在不久的将来，我们将陆续推出这套丛书的其他几本。我们相信那几本也同样又让大家喜欢的理由。
　　Only you――敬请关注。

张立宪　　　
2000年6月・北京
