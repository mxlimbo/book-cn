吴承恩的齐天大圣和周星驰的至尊宝，最后都成了孙悟空，上西天取经去了。不过紧箍咒的动力系统改变了，只有绝情灭欲的至尊宝才能保住脑袋不吃苦头。不谈感情，投身于伟大的取经事业，以这样冠冕堂皇来作为悲剧的理由，已经不是周星驰了。周星驰从悲剧中寻找力量就像王朔严肃认真地说事一样，不灵。
　　更糟的是，周星驰走的还是红颜误事的套路，中国的男人一着急要做点儿大事业，第一个要干掉的就是自己的爱情，做他的情人就要倒霉了，这种又自私又老土的调调，谁唱都不好听，即使是星爷唱也不好听。
　 "如果再让我选择一次，我会说我爱你，如果要给爱加一个期限，我情愿是--一万年"；"我的男人是个盖世英雄，他一定会驾　着七彩祥云来迎娶我"。
　　可惜了这样动人的对白，说在一个过时的故事里。
　　对已有文本的解构和重述，是重要的创造资源，当然也是喜剧资源，这是一条荆棘路，玩得好，惜力打力，靠原有文本固有的名气和影响力衬出新作；玩得不好，失败和嘲讽都是双份的。比如笛福的《鲁滨逊漂流记》，后人的改写本多达五百种以上，但据我所知，只有其中一本为作者带来了荣耀，那就是图尼埃的《礼拜五--太平洋上的灵薄狱》，1967年获法兰西学院小说大奖。这部小说里，图尼埃的新酒味道是这样的：荒岛上的鲁滨逊在"野人"礼拜五的影响下，蜕皮般将物质与征服的可笑观念弃掉，最终成了一个能与风和太阳交流的纯粹的人。也就是说，改写的成功率是极低的。
　　吴承恩的西游记是没有爱情的，涉及男女之事的，只有猪八戒偶而有一些动物性的冲动。把这当作爱情戏的旧杯子，真是容易出彩，可惜至尊宝就像是士大夫笔下的回头浪子，先滥情，后摄心，爱护兄弟、服从师父、尊重佛祖，成长为仙界劳模。
　　现在周星驰的新片是等不来了，在那些旧碟里反反复复寻开心时，我把《大话西游》重新剪辑了一下，让它符合我心目中的星爷模式：开始，至尊宝是一心礼佛、虔诚西行的行者，后来听从内心情感的呼唤，成了多情种子，爱白晶晶、爱牛魔王夫人、爱牛魔王妹妹、爱紫霞仙子。最终，从滥情走向专情，从杯水主义走向地久人长，其间，当然穿插无数爆料，比如观音姐姐掐死唐僧弟弟；比如三藏做小妖的思想工作。最重要的是，得让至尊宝对那些来劝阻的神神鬼鬼做一番爱情宣言：我只要美人，拜托别用西天那些破经来烦我了。然后于众目睽睽之下，携紫霞纤纤玉手，走先！
　　你也许会说，如来说翻脸就翻脸，怎么能走先？这就不用你担心了，星爷有办法解决，比如让至尊宝掌握如来泡妞的秘密。
