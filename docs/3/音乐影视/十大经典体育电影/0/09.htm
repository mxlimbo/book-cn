《愤怒的公牛》


　　《愤怒的公牛》（Raging Bull）1980

　　导演：马丁・斯柯西斯（Martin Scorsese）
　　编剧：保罗・施拉德（Paul Schrader）
　　演员：罗伯特・德尼罗（Robert De Niro）凯西・莫拉琪（Cathy Moriarty）乔・帕西（Joe Pesci）佛兰克・文森特（Frank Vincent）

　　该片阵容强大，男主人公拉莫塔（LaMotta）由演技独到的罗伯特・德尼罗（Robert De Niro）饰演，导演则是大名鼎鼎的马丁・斯柯塞斯（Martin Scorsese），保罗・施拉德（Paul Schrader）编剧。

　　与其他人一样，拉莫塔靠着自己的拳头一步一步地打出了纽约的贫民窟，从一文不名的小混混成为了腰缠万贯的拳击明星，还赢得了一位美人的芳心。但无法遏制的坏脾气成了他的致命弱点。他甚至怀疑妻子和自己的哥哥关系暧昧。多疑暴躁的性格使他最终陷入了孤立无援的悲惨境地。虽然他在拳台上无比的凶狠，但却结局凄惨。最后，这位孤苦潦倒的前世界冠军为了能多换点钱，不惜把象征着荣誉的金腰带扯开，设法从里面找到些金子。具有讽刺意味的是，稍后当铺老板告诉他，一条完整的金腰带反而更值钱。据称，罗伯特・德・尼罗为了更好地饰演这一角色，特地增重60磅。他的出色表演为影片增色不少。而它也成了著名导演马丁・斯柯西斯除了《出租汽车》外最著名的一部影片。
