电影表演的特性

影象幻觉

　　电影表演是通过摄影机纪录下来的表演。是胶片或磁带上纪录下的影像的表演，而不是真人的表演，这是电影表演区月十案高爱 全众表演的艺术，电影则是用影像表现的艺术。

　　电影中的影像是真人实体表演的复制和转现。按照巴赞的照相影像本体论的观点，似乎影像与客观现实中的被摄物完全同一的，用克拉考尔的话说：“物质现实的复原。”这种本论观点后来被电影符号学所否定。符号学认为电影图像是一个文化符号系统，是人运用符号系统对现实的转写，必然带有人的文化背景所产生的主观意图和选择，不可能是对外部世界纯客观的纪录。思斯特·卡西尔说：“像所有其它的符号形式一样，艺术并不是对一个现成的即予的实在的单纯复写。它是导向对事物和人类生活得出客观见解的途径之一。它不是对实在的摹仿，而是对实在的发现。”同一个对像在不同的人眼中看到的东西是不一样的。

　　两个画家不可能画出同一个风景。影《黄土地》第一个发现了黄土高原粗犷、强悍、凝季、永远充满一种原始生命活力的美。电影影像本身就是艺术家对现实的发现，它不仅是自然的，也是假定性的。 所以体现在银幕上的影像实际上是电影创作者所要让观众感知的影像幻觉。

　　例如：进行过有名的“库里肖夫效应”试验的苏联导演库里肖夫还搞过一个实验，他用属于不同身体的肢体造出一个合成人。电影《湘女潇潇》中母亲给孩子喂奶的特写镜头就是用另一个人的乳房代替的。美国的一个录相片中出现了这样 一个场面，罗纳德·里根驾驶一架俯冲轰炸机，兴高采烈地向一支摇滚乐队发动进攻；电影《兔子罗杰》就是一个演员真人同动画角色——兔子罗杰一起表演，相互激烈搏斗的别开生面的有趣影片。此外，动作片中普遍使用的替身演员，各种光怪陆离、神奇惊险的特技镜头的使用使詹姆斯·邦德、超人、蝙蝠侠等等英雄人物在观众的眼中真正成了武艺绝伦、出神入化的超人。如此这般的种种影像幻觉只有电影、电视能够创造出来。

　　这种似实而虚的影像表演也激发了观众的偶像崇拜心理：演员栩栩如生地在影像里，那种“在场”感甚至比看室内剧时还更强烈，但却穿不透，银幕成为一道穿不透的屏障，可望而不可及。因此，每一场拍摄下来的表演都具有一种挑逗性的既在又不在，既得而又失的感觉。演员离观众这么近又是这么远，这么亲切又是这么高不可攀，在观众心理上增强了偶像崇拜的乐趣。

　　但是，把表演化为偶像崇拜的同一架机器也可能造成完全相反的结果：解构表演。演员的表演不再橡剧场演出时那样是“过眼烟云”了，而是铁板钉钉地被机器纪录在胶片或磁带上，表演变成一个本文，让人去读解，特别是有了录相机，现在人们可以在家里一点一滴地看它，可以把一段表演动作从影片整体中剥离出来，成为独立的存在，反复地看、逐格地看、让一个角色把吃进去的东西吐出来，再吃进去地看……。用纳尔摩尔的话说：“机器使观众变成后现代主义者：他们对演出的 内容保持一定的距离，对表演——那伯是接近日常生活的表演——人工造作的地方高度敏感。”崇拜感消失了，变成了某种间离和隔膜。
