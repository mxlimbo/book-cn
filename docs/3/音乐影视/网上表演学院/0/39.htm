电影表演的特性

演员在电影中的作用（一）

　　前面讲的是表演的一般规律和创作方法，它们既是话剧表演的根本，也是电影表演的根本，这些基本法则对于电影表演和话剧表演是共同的，完全一样的。理清楚了这一部分，现在再来谈电影中的表演。

　　电影做为近百年来在科技发展基础上的新兴的艺术，声势夺人，成为独具特性的一门艺术，由于它的媒介材料、制作流程、演出方式的特性，使电影拥有几乎无所不能的表现手段和其它艺术前所未有的美学特性，而后来居上的电视更是大有以磁带取代胶片的趋势，那么电影、电视中的表演当然也有不同于舞台表演的自己的特性。电影表演的特性是由电影的特性所决定的。所以要深入了解电影表演的特性，必须熟悉和了解电影的特性，我认为做为一个专业电影演员，与舞台演员最大的不同是他应当具有全面的电影文化修养。这个题目就不是这里能够详谈的，我们仅从表演的角度概括地谈谈电影表演的特性(电视表演的特性与电影表演是大体一致的。)。

　一、演员在电影中的作用

　　艺术的对像是人，电影也不例外。一部影片的情节内容、思想内涵都是通过剧中人物，就是演员扮演的角色体现的，观众关注 人物的命运，自然最注意扮演人物的演员，演员就成为最引人 注目，受到褒贬最多的中心目标。但是演员身上的光辉，取得的成就，只有一部分是属于演员个人的气民魅力和表演功 力，大部分成就和功劳是属于剧本、导演、摄影、照明以及化妆道具等部门的。一部成功的影片把所有成功的荣耀都汇聚于员身上，同样，一部失败的影片，演员也要承担许多并非是他的责难。我们在分析一个演员的表演时往往把编剧的、导演的因素加诸表演中，很难脱离开这些因素单纯地去谈表演。

　　亚历山大·诺克斯说：“伟大的剧本提供伟大的角色，伟大的角色促成伟大的演员。”这话是完全正确的。但是，在电影中还要加上一句，伟大演员要依赖于伟大导演的创作。 相对于舞台表演来说，演员在电影中的作用比之在舞台上的作用是降低了。舞台是演员为主导的艺术，电影是导演为主导的艺术。美国导演马摩里安(《瑞典女皇》的导演)说：“舞台上只有一种运动，即演员的运动，唯此而已。银幕上有三种运动：演员的运动、摄影机的运动，蒙太奇的运动。这三者必须浑成一体，才能拍出一部完美的作品。”这三大运动都是在导演总体构思的统一意图下进行的。演员在电影中处于一种很矛盾的奇特位置，他既是最核心、最显要、最引人人注目的又是最被动、最缺乏创作主动性发挥余地的。总体上说，他只是电影符号系统中的一类编码。但是在不同影片里演员的作用是不一样的。例如：喜剧影片、传记片、常规的戏剧性影片等等，在以描写人物和塑造完整性格为主体的影片中，演员表演起着重要的主导作用。目前，这种以演员表演为主的影片仍然是电影中的绝大多数和电视剧的全部。另一类影片，多数是实验电影和现代派电影，则把演员表演的作用缩小到最低限度。 例如：阿仑·雷乃的《去年在马里昂巴德》、日本影片《裸岛》、塔尔可夫斯基的《镜子》以及陈凯歌的《黄土地》等。

　　在不同导演的手里，演员所能发挥的作用和创作的能动性也是大大不同的。有两类导演，一类是以摄影机为主体的，一类是以演员为主体的。
