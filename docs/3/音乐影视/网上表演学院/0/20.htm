分析剧本与角色

人物和人物关系

　　剧本中每个人物都应该是全剧严密完整结构一个组成部分，不是随便写进去的，每一个人物在剧中都有特定的作用，!揭示作品的主题思想密切相关。 分析剧本时，演员研究剧中人物和自己所扮演的角色，先要从人物与全剧整体的关系开始，了解剧作者为什么要写这个人物，他对于揭示作品主题思想有什么意义，并深入了解人物之间的相互关系。

　　分析剧本时，演员研究剧中人物和自己所扮演的角色，先要从人物与全剧整体的关系开始，了解剧作者为什么要写这个人物，他对于揭示作品主题思想有什么意义，并深入了解 人物之间的相互关系。 《祝福》中柳嫂这个人物的戏并不多，但是她反映了当社会的一个侧面，对于揭示作品主题有很重要作用。柳嫂是受剥削受迫害的劳动妇女，她同情祥林嫂的遭遇，可又由于的愚昧无知，受封建迷信的毒害太深，不知不觉做了封建统阶级的帮凶，间接地起了从心灵上摧残祥林嫂的作用。作者通过这个人物进一步暴露了封建势力的统治之深、毒害之大，人不寒而栗，可悲可愤，加强了主题思想的深度。从情节结上，柳嫂的一席话给祥林嫂更加沉重的精神负担，促使她怀渺茫的希望苦苫攒钱，捐条门槛，为后面的戏剧高潮——砍槛埋下伏笔，推动了情节的发展，

　　《创业》里秦发愤这个人物也是不可缺少的，他为成为一名熟练的钻井工作，不顾自己膝盖下还有一块美国炮弹，每天跪在井架钢板上擦机器，刻苦地锻炼。作者这一笔就生动刻划出这个部队复员的青年工人的优秀品质，从另一个角度展现了中国工人阶级的宏伟志气。同时，在"我不趴下"这场戏里，秦发愤这条情节线又有力地起到烘托华程、周挺杉的英雄形象的作用。 演员深入了解角色在全剧中的作用，明确人物对于表达主题思想的关系，就能摆正人物的位置，从全剧整体的要求出发，发挥自己的创作能动性，完成好角色的任务。如果演员偏 离了剧本给人物确定的任务和作用，不适当地突出、强调自己所扮演的角色，结果必然适得其反，会破坏戏的完整，引起不应有的效果，造成通常所说的"抢戏"，这是十分错误的。

　　剧中人物是"由剧本的统一的贯穿行动联合一起，彼此之间有着一定的相互关系的人物的集合体。"人物之间有着剧本规定的特定的人物关系，"戏"就是在人物关系中展开和进行 的。分析剧本需要把每个人物之间的关系搞清楚。有的戏人物关系比较简单，容易搞清，有的戏人物关系则相当复杂，像《红楼梦》这样的宏篇巨著，要把每个人物之间的关系了解清 楚就得费些功夫，细心研究了。再如《雷雨》这个剧本，结构严谨，情节曲折，人物关系不是一下能够掌握的。这个戏写了两家人，周家和鲁家，而这两家人之间的关系又是错综复杂地交揉在一起。繁漪是周朴园的妻子，周朴园的小儿子周冲是繁游 生的，大儿子周萍则是周朴园同周家三十年前的丫头鲁妈发生关系后生的儿子，鲁妈再次怀孕生下了第二个孩子才三天被赶出了周家，以后鲁妈同鲁贵结婚，生了女儿四凤，四凤的哥哥鲁大海就是周朴园的第二个儿子。三十年后鲁贵同四又到周家当仆人，周萍同后母繁游有着丑恶的暖昧关系，同周萍与周冲又都爱上了四凤，四风在血缘关系上是周萍异 同母的兄妹，而四风又与周萍发生关系有了身孕。这样复杂人物关系，如果演员不认真研究是无法进行表演的。

　　即使比较简单的人物关系，演员也要细细揣摸，才能在演中具体地、细致地、分寸适当地体现出人物之间的关系。祥林嫂同卫老二来说，这两个人物之间的关系比较简单，但具体表演时就要求演员知道这两个人物的关系应该是怎的。剧本里交待了卫老二是祥林嫂的远房亲戚，祥林嫂称呼"二哥"，以此推断，卫老二是祥林嫂的族兄。封建宗法制度的族权是束得农民的一条极大的绳索，族兄对于样林嫂就相当的权威，对他只能敬畏，不敢稍有件犯。祥林嫂平日对游手好闲，不务正业，奸滑狠刁，近乎无赖的族兄就又讨厌害怕，加上祥林嫂是寡妇在男人面前更得少言寡语，回避分。所以剧本里祥林嫂对卫老二的问话只用一个字回答，"卫打柴呀?祥：暖。卫：你婆婆在家吗?祥：在"她不会多说一字的。这种人物关系决定了祥林嫂同卫老二接触时该是怎的心情和态度，这正是演员需要理解和掌握的。 分析剧本，大体包括研究剧本的主题思想、时代背景、盾冲突、划分事件搞清情节脉络以及人物与人物关系等几方面，这是演员着手准备角色、进入塑造人物必不可少要进的工作。在剧组和摄制组，这项工作是由创作人员集体研究论的，以求得创作集体对全剧的基本统一的理解和认识。但演员一定要掌握分析剧本的方法，具有独立分析剧本、准备色的能力。
