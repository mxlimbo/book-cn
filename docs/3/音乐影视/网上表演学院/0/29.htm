理解人物和体现人物

性格化（三）

　　电影《红色娘子军》的编剧、导演和演员塑造琼花的人物性格则突出地着力对于琼花眼神的刻划。导演谈到：“《红色娘子军》剧本中，作者描写了琼花的一个很突出的外形特征‘火辣辣的大眼’，戏的开场，琼花吊在梁上，用仇恨的眼光瞪着老四；遇到南霸天时，眼睛里闪出复仇的怒火。所以，在确定演员上我首先考虑的也是‘眼睛’，眼睛成了扮演琼花这个演员外形特征上很重要的条件。”眼睛是灵魂的窗户，演员通过眼神 传达出真切细致的感觉和复杂的内心世界。斯坦尼斯拉夫斯基说：人物的“精神世界和内部形象是通过外表――通过面部；通过眼神来认识的，因为它们直接反映着内心世界。”扮演琼花的演员充分运用了眼神这个有力手段，展现人物的性格特征和人物性格的发展、成熟。全片第一个镜头是琼花的特写，她那双黑亮、火辣辣的大眼睛充满着仇恨、反抗、至死不屈的顽强意志，有一股“跑!看不住就跑”的猛劲，开始就强烈突现出人物的鲜明个性。在侦察这场戏时，拿起武器的奴隶又看见了不共戴天的敌人南霸天，她两眼紧盯着仇人，燃烧着复仇的怒火，略略眯紧眼睛瞄准，这时候的琼花忘记了侦察的任务、忘记的军纪、听不见战友的警告，眼睛里只凝射出“复仇!打死南霸天!”这唯一的意愿。当琼花目睹洪常青英恿就义，被南霸天用火活活烧死时，刹那间，巨大悲痛，压抑的愤怒，极端痛苦复杂的心情在演员――琼花的眼睛中十分真实、准确地体现出来了。经历了哪些激烈的冲击的琼花独自来到分界岭阵地，她又看到了洪常青留下的皮包、四个银毫子、自己的入党申请书，上面有着洪常青的签名……，当她擦干眼泪，紧紧捏着四个银毫子，慢慢举起拳头，两眼注视前方，默默地宣誓时，从她的眼睛里看到了人物思想感情的升华，这已经是一个成熟的共产主义战士具有的坚定、刚毅、深邃的眼神了。

　　另外，抓住某些特定的情节，着重于去展现人物性格，也有很多成功的例子。如：电影女上甘岭》有一场战士们在坑道里捉松鼠的戏，这个情节不仅表现出了我们战士的革命乐观主义精神，更重要的是这一笔生动地刻划了张连长的可爱性格。在敌人重兵围困、日夜轰炸、进攻、缺粮断水，几昼夜不眠的艰苦环境中，张连长竞带头捉起松鼠来了，这位身经百战、勇敢、顽强的连长这时候更像个调皮的小战士，他眼睛闪着喜悦的亮光，军帽挤歪了，聚精会神，劲头十足地同战士们――起，在狭窄的坑道里跑来跑去地捉松鼠。有了这场戏，就把张连长性格中纯厚、开朗、甚至有些天真稚气、深切地热爱生活等丰富色彩浓墨重笔地描绘出来，给观众以极深的印像。
