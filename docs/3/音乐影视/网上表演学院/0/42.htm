电影表演的特性

蒙太奇构成的表演（一）

　　尽管运用景深镜头和摄影机运动的长镜头已经成为重要的和时髦的表现手段，但是一部故事影片终究不能用一个镜头拍摄下来，蒙太奇依然是基本的电影语言。那么电影表演也就是由蒙太奇镜头组接构成的表演。电影的这样一种表现手法给电影表演带来如下的特点：

　①表演的非连续性。 演员在一部影片中的表演不是由头至尾，有顺序地、完整的、连贯的表演，而是每一段戏的表演都被分切割裂为若干单镜头(通常一部90分钟的影片是由500至1000个镜头组成)，演员的任务只是去完成每一个单镜头中的表演，最后完成片中完整、连贯、有丰富含意和感染力的表演，不是由演员而是由导演去完成的。而且由于生产组织的需要，不可能按照戏的发展顺序由头至尾地去拍摄，只能把一个时间、一个季节、一个场景的镜头集中拍完，再去拍另一个时间、另一个季节、另一个场景的镜头。于是，可能先拍戏的结尾部分，主人公已经年迈衰老地死去，再去拍戏的开头部分，主人公正值青春年少，意气风发地走向生活。即使是同一场景里甲、乙二人对话的镜头，出于拍摄的方便，避免来回移动机位和重新布光等时间的浪费，也要把同一机位、同一角度的镜头拍完，再去拍另一个机位、另一个角度的镜头，完全不顾甲、乙角色之间一问一答、相互交谈时的情绪连贯和交流反映的连接。当然，多机拍摄可以使一场戏里的演员表演情绪连贯一气呵成，但是目前具备多机拍摄条件的摄制组仅仅是极为特殊的个别宠儿，绝大多数电影和电视剧还只能是单机拍摄。所以，这种拍摄方法就造成了电影表演的“非连续性”特点。 这种单镜头的隔裂表演和时空乱跳的拍摄方法，给电影演员的创作带来了特殊的矛盾，即：演员要在分切割裂的零碎表演中完成人物形像和人物性格统一完整的塑造；在非连续性拍摄中保持表演的连贯性。这是电影演员必须掌握、具备的创作本领。(如同摄影师要有在非连续性拍摄中保持每一场戏的影调、色调、光线统一的本领是一样的。) 本来电影演员就缺少舞台演员拥有的优越条件，可以用几个月的时间排练全剧，在排练中逐渐掌握自己的角色。再加上顺序颠来倒去的拍摄，有些演员往往在影片快拍完的时候才感到自己找到了角色的准确感觉，可是绝大部分镜头已经在把握角色不理想的状况下拍过去了。懊悔、遗憾已经为时晚矣。所以电影演员在开拍前准备角色阶段要特别下功夫，认真、细致地去研究剧本和角色，在总体上把握住角色的自我感觉和角色性格，把握住人物发展变化的脉胳，把握住角色内心变化的依据，那么不论怎样颠倒拍摄，都能做到心中有数，万变不离其宗。

　②蒙太奇构成的表演效果 蒙太奇是通过镜头组合进行叙事和揭示思想感情的艺术，他不仅仅是把零散的表演素材剪接、编辑成完整、连贯、流畅的表演，还具有渲染、强化、突出表演细节，延长和缩短活动时空的功能，赋予表演更强烈的冲击、感染力和隐喻的丰富内涵，蒙太奇构成的表演效果，单凭演员自身表演是无法达到的。

　　试看一个经典的例子：在无声片《母亲》中，半醉的丈夫回到家里，准备取走挂钟的钟锤换钱买酒，遭到妻子阻拦，酒醉的丈夫从椅子上摔倒。导演普多夫金将这场戏分切成16个镜头：
　1．全景 丈夫走近挂钟。妻子明白他的心思，默默地注视着他。
　2．中景 丈夫站在挂钟前，他在看钟。
　3．中景 妻子不安地望着丈夫。
　4．近景 (大角度俯拍)丈夫的一只手，手抓住椅子，把椅子拉到挂钟旁，他登到椅子上(我们只能看到他 的皮靴)。
　5．近景 (变换拍摄角度)丈夫的皮靴，他站在椅子上，踮起脚尖。
　6．特写 丈夫的头部，头已经靠近表盘，他抬起双手，准备摘下钟锤。
　7．特写 妻子喊了一声，冲过去，出画面。
　8．大特 丈夫的双手在摘钟锤。
　9．中景 妻子冲到丈夫身边，阻拦他，扯动他的上衣。
　10．近景 (稍俯)妻子抓住大夫的上衣，拼命地拽他，丈夫用脚把她踢开。 　
　11．大特 (变换拍摄角度)妻子的手死死抓住丈夫上衣，不停地拽来拽去，使丈夫失去平衡。
　12．近景 丈夫的双手摘下了一个钟锤，但他的上半身在摇晃。
　13．特写 (稍俯)站在椅子上的双脚。椅子在滑动。
　14．特写 丈夫的一只手又抓到另一个钟锤，画面上看不到头部。
　15．近一中(俯) 丈夫从椅子上摔下来，躺倒在地上。在他身旁可以看到椅子、摔碎的挂钟、表盘和钟 锤。
　16．大特 (贴地板拍摄)挂钟的齿轮在地上滚动，而后嘎然停住。
