电影表演的特性

思维过程的外观化（三）

　　但是，任何手段的可能性都有它的局限，如果认为电影众多的表现手段可以导至完全替代或是取消演员纯熟、精湛的演技，表演艺术在电影中是不需要的，或者是无足轻重的，那就趋向到一个错误的极端。一位导演曾说：“《人生》中扮演巧珍的演员，好就好在她不会演戏”。如此说来演员越不会演戏越符合电影表演的要求，越能体现导演的构思意图，这样的观点似乎很时髦，却是不切实际和有害的。同样，对于非职业演员在电影中的作用也估价过高了。实际上，创导起用非职业演员，不用职业演员的新现实主义的导演们几乎无一例外地放弃了最初的观点，都转向使用职业演员，并且是有声望的大明星 、大演员。他们不得不承认：“非职业演员从来没有正面接触过全面刻划性格的问题。每当把这个问题交给非职业演员去解 决时，他往往就变得极不自然。全面刻划性格需要由职业演员来担任”。我国的“第五代”导演对这一点也已经醒悟了过来。

　　在大多数故事影片中，许多充满激情，．揭示人物复杂内心活动的戏，只能依靠演员的表演，是任何其它手段无法代替的。 《克莱默夫妇》中那场父子俩做早餐的戏，妻子乔安娜出走了，早晨父亲泰德第一次为儿子比利做早餐，以往他从来没干过家务事，现在他要在儿子面前装出一切都很正常，不能让儿子感觉到他已经失去了母亲。 泰德问：“早饭想吃什么?” 比利：“法国煎面包片。” 泰德：“你要吃什么爸爸都会给你做。” 胡子都没刮的父亲忙忙碌碌地干起来，嘴里还要逗着孩子高兴，卖弄着：“你看啊，一只手，就用一只手打鸡蛋。”他拿起一只碗，用一只手打碎一个鸡蛋，把弄碎的鸡蛋倒进碗里。 “世界上最好的厨师都是男人，你知道吗?真是愉快的事，一年到头不干这个可不行。” 比利带着不信任的心情：“鸡蛋壳都掉进去啦!” “那没有关系。法国面包片会做得又松又脆。” 当他准备把面包浸到鸡蛋里时，才发现碗太小了，于是用一双脏手把面包撕成两半硬塞进碗里去。 “是不是忘了加牛奶啦?” “牛奶是必需最后加的……人是太高兴的时候就会把重要的事情给忘了的，爸爸是想考验你的注意力啊……。” 儿子不信任地挑剔，他在尽量装得老练高兴。突然比利叫：“爸爸，糊啦!糊啦! ” 秦德看到煎锅里冒烟，一着急，伸手就去抓煎锅的铁柄，烫得受不住，“哇——”地叫喊着一松手，“哐啷!”一声锅子掉在地上，东西撤了一地。这时他禁不住失声骂出来。“这浑蛋女人!”又用力去踢煎锅，比利吃惊望着他的表情又使他惊醒过来，连忙重新装出若无其事的样子：“没关系!没关系!一点问题也没有，没事儿!”这场戏，感情、语言、台词的处理是那贴近生活，合情合理，又非常生动感人，霍夫曼把人物内心情绪同外表流露完全相反的窘态，压抑不住的怒气怨气的爆发，又怕吓着孩子、伤害孩子幼小的心灵，急忙又压制去，掩饰自己的这些变化，都非常准确、有分寸地演了出来。

　　希腊影片《囚徒》，女主人公维基在电视台接受记者采访，述说自己全家被白人雇佣军残暴杀害的罪行，大段的回叙是演员面对镜头的直接讲述，没有用闪回镜头，维基越说情绪愈激动，眼中含着泪花，终于难以控制趴在椅子扶手上嚎啕大哭起来。这是一个镜头下来的情绪逐渐高昂，直到无法自制。戏的需要看，导演是把重点放在表达人物的情绪上，得到有力地揭示人物的精神状态，大大增强了维基这个人物给观众的深刻印像，如果用闪回镜头就达不到这样的效果。这场戏主要是依赖演员的表演功力，而以呼喊声，射击等音响做为辅助的烘托效果。
