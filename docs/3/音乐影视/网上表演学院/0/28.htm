理解人物和体现人物

性格化（二）

　　那么，表演艺术在人物性格化方面起什么作用，以及怎样在剧作的基础上完成人物的性格化呢?别林斯基对这个问题有过正确的回答，他说：“演员创作在于实现二个字中。你读汉姆雷特，理解他，但看不见他，一个有着一定的面貌、一定的头发颜色、一定的发音器官、一定的姿势的人，一句话，一个具体的活人。……如果没有舞台艺术，戏剧作品将是不完整的，要完全了解人物，只知道他如何行动、说话、感觉还不够――必须看到和听见他如何行动、说话、感觉。”可以说演员的作用在于使人们性格“具体化”，把人们理解、想象的形象变成直接可见的形象。

　　为了突出人物的个性特征，达到鲜明的性格化，演员要在深入理解人物的基础上，把握住人物性格基调，探索人物的精神气质、思想感情，从而找到人物特有的眼神、姿态、步伐、语气、语调、符合人物个性的动作以及人物的装束打扮、化妆造型等。法国演员哥格兰说得很形象：“演员先在想象中给自己塑造出一个模特儿，然后，像写生者似的，抓住这个模特儿的每一条线条，画在自己身上，而不是画在画布上……。只要他看见答尔丢夫穿的服装，就穿上，只要他看见答尔丢夫的步态，就模仿，只要他看见答尔丢夫的相貌，更铭记在心。 他使自己的脸部适合角色的外貌，可以说是他改变自己的外表，贴贴补补，一直到认为已经做到活像答尔丢夫时为止。但这决不等于就完事了。因为这仅仅是外形上的相似，它只是角色的外形，而不是角色本身。演员还需要学答尔丢夫的声音来说话；为了使自己的活动和角色的活动取得统一，他必须强迫自己像答尔丢夫那样走路、谈话、倾听。他必须把答尔丢夫的灵魂贯注进自己的身体。这时一幅人像才算画成，只要加上镜框 ――那就是说，登上舞台，观众就会说：‘真是个答尔丢夫，’或者说：‘这个演员演得糟透了!’

　　人物的个性特征溶化在整个人物之中，但是演员在塑造人物过程中需要善于抓住最能体现人物个性特征的某一点感某个典型动作，突出地加以强调，使它能给观众留下鲜明的深刻的印像，这是性格化创作的一个重要手法。《列宁在一九一八》中的克里姆林宫卫队长和《政府委员》中的女主人公的丈夫索阔洛夫都是由演员瓦宁扮演的，他是一位善于突出人物鲜明个性的优秀演员。他扮演克里姆林宫卫队长选取了一个很有特色的动作：每当紧要关头，他都要从上衣口袋里掏出一把小梳子。吹一口气，然后仔细梳梳他几根稀疏的头发。这个小动作很别致地突出了这个外表并不起眼，实际却十分机智、勇敢、忠心耿耿的优秀布尔什维克的性格特点，看来有些粗俗、漫不经心，但又带出他特有的沉着、机敏和幽默感。在白匪特务用金钱收买他为反革命暴乱打开克里姆林宫大门的这场戏里。梳头发的动作既迷惑了敌人，使特务感到这是个可以收买的浅薄愚蠢家伙，又体现出了卫队长在任何紧张时刻都能从容不迫对付的性格特色。我们也不会忘记瓦宁扮演的索阔洛夫那出色的无声干笑，那种无声的笑带有俄罗斯农民特有的狡独和幽默，也体现出索阔洛夫对自己非常自信，对别人却蔑视以待的性格
