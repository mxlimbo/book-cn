《千王之王2000》

(1999)
导　　演：王晶
其他主演：王晶
　　　　　张家辉
　　　　　林熙蕾
　　　　　吴君如
收藏指数：★★☆


　　星哥以赌片成名，但他的上一部赌片还是1991年的《上海滩赌圣》，星哥重返赌坛，恐怕也体现出他希望再度辉煌的一种心情吧。但可惜，这次“星哥”已经成了“星爷”，没有了当初小人物的心态，而且赌片风潮已过，再加上他和王晶在片中力捧的张家辉表现实在差劲，所以失败是理所当然的事。星哥的表演本来已经脱离了早期影片中王晶式的“屎尿屁”笑料，但这次与王晶的再度合作，又被王大导演江郎才尽后的疯狂无聊所腐蚀。像什么“失传一百多年的肥螳螂”之类的无聊手段竟然被用来结束影片，而舔鞋子、吃蜥蜴之类令人翻胃的噱头就更多了。

　　星哥这次以澳门“千王之王”的身份重出江湖，有点“跩”得过份。影片一味强调他的至尊地位，以致十分单调乏味，而且体现他高明“赌术”的桥段也牵强做作，只是把观众当傻子罢了。其实之所以还能给影片打上第三颗“☆”，也是因为这毕竟是一部新片，恐怕短时间没法看到50遍以上而已。
