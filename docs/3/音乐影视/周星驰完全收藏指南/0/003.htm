《喜剧之王》

(1998)
导　　演：李力持、周星驰
其他主演：莫文蔚
　　　　　张柏芝
　　　　　吴孟达
收藏指数：★★★★


　　20世纪末，世界影坛三大喜剧巨头：罗贝托·贝里尼、金凯利和华人至尊周星驰仿佛约好了似的，先后向自己一贯的表演风格进行挑战。贝里尼获得了最广泛的承认，以《美丽人生》一片夺得戛纳和奥斯卡诸项大奖，万千宠爱集于一身。而金·凯利主演的由彼得·威尔执导的《真人表演》和米洛斯·福尔曼的《月亮上的人》虽在奥斯卡铩羽却也在金球奖项中赢取荣誉。

　　相形之下，周星驰的《喜剧之王》虽打败《玻璃樽》成为港产片票房冠军，在艺术上的追求却显得浅尝辄止，一触即收，因而给人的感觉不伦不类，既不是艺术片，又不象纯粹的商业片，两头都沾着却又两头不讨好。影片成为票房冠军几乎完全是周星驰一人的票房号召力——尤其又是“喜剧之王”这样的唬人片名。其实这个故事本身的素质非常好，如果能够继续深入挖掘出那种小人物的卑微与自尊相交织的复杂的心态，相信会很有意思，和《月亮上的人》完全可能成为东西“双璧”。但在影片里我们虽然感觉到了这种东西，同时又看出创作上既想有所追求又时时瞻前顾后迎合票房的自身矛盾。

　　此外，这部影片里，无论整体框架还是段落细节，都存在着致命伤。最重要的是以周星驰为首的编导人员没有把握好生活真实与喜剧夸张之间的关系，而这样的问题在星哥以前的作品中，几乎都是不存在的。以前那些影片的风格被确立为漫画式的，即使是当代小人物生活的题材，它的细节再夸张、再无稽、再荒谬，我们也可以接受，因为它的规定情境完全是假设的和独立的。可是《喜剧之王》的影片基调是一种弥漫着伤感与浪漫的人生描摹，人物形象也和一般漫画式的喜剧片截然不同，因此生活的合理性、真实性也就成了很重要的问题。

　　许多看过本片的朋友都会有同样感受，就是影片的虎头蛇尾和虚张声势。由追求个人表演的终极目标而引申成为生死大事，可以说想法非常有创意。这个不成功毋宁死的情节，透出周星驰作为一个演员，所带着的近乎悲壮的心情。一个喜剧演员如果成功完成这样一部“笑中带泪”、笑中带出庄严的影片，那他应该可以登上喜剧艺术片的殿堂了。贝尼尼作到了，金·凯瑞也有希望了，而星哥却在关键时刻“落马”了。这个本应把影片推向高潮的段落看完后却让人觉得不满足，空空荡荡没有着落，实在是影片最大的遗憾。

　　无论如何，周星驰已经作出了尝试，我们有理由相信，也怀着无尽的希望，在不久的将来，他会奉献出更成熟更经典的电影作品。出于对星哥追求的支持，给这部本来应该三颗★的片子加上一颗，希望更多的朋友收藏这部能给人希望的尝试之作。
