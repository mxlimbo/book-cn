《流氓差婆》

又　　名：雌雄双辣、雌雄杀星、来自江湖(1989)
导　　演：刘镇伟
其他主演：吴君如
　　　　　成奎安
收藏指数：★


　　这部刘镇伟执导的片子，让你无论如何无法和《大话西游》联系起来，除了周星驰。漫长地等待了40分钟，我们才看到了周星驰的正脸，在此之前我们只是看到吴君如和一些乱七八糟的人打来打去。影片本身绝对是失败之作，但如果能熬过开头的无聊，还是可以看出星哥已经开始有了一些后来的风采，不过最后决斗场面的失败就不是一个周星驰所能弥补的了。有鉴于此，给各位“星迷”收藏者一个无可奈何的建议——收藏影片的下碟即可。
