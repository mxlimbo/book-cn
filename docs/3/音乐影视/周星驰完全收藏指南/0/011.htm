《食神》

(1996)
导　　演：周星驰、李力持
其他主演：吴孟达
　　　　　谷德昭
　　　　　莫文蔚
收藏指数：★★★


　　这是一部典型的贺岁片，也是星哥亲自上阵又编又导又演的影片，不知道是哪个原因，反正莫文蔚最后居然没有死，还变“漂亮”（？！？）了，画外音还生怕你不愿意似的提醒你“过年吗，什么事都有可能发生的！”。

　　谷德昭以前只是编编剧本、导导片子，最多在星哥身边弄点“米田共”（《破坏之王》）。在这部影片里，谷德昭亲自上阵，和星哥大唱对台戏，只是表演太过火了。整个故事是个“得而复失、失而复得”的戏剧过程，给了星哥发挥演技的巨大空间，而且烹制佳肴美食的题材也适合星哥绝妙想象力的挥洒。果然，星哥的“撒尿牛丸”就有十足的噱头，打乒乓球的创意可以在《破坏之王》中找到来历，最后的“叉烧饭”更是匪夷所思。星哥的“湖南少林寺”之行，是食神大赛的一个极为精彩的“插曲”，老方丈和十八罗汉铜人阵成了片中最有喜剧效果的人物造型。十八罗汉的十八般“兵器”甚至比《凌凌漆》一片中家英哥的杀人武器霸王“要你命三千”还酷。食神大赛中星哥的造型也还不错，蛮嚣张的。影片最夸张，令人印象最深的莫过于大赛最后的品评了。两次夸赞手法不同，各具妙处，尤其是“为什么我流泪了呢，哦，原来是洋葱”现在已经成了“星青年”经典语汇之一了。

　　还是那句话，这是一部贺岁片，也是星哥自编自导自演，局部细节喜剧性很强，但作为一部完整的电影就比较差劲了。很多场面实在无聊，影响了对整部影片一看再看的心情。。
