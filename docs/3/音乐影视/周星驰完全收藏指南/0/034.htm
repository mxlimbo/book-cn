《师兄撞鬼》

(1990)
导　　演：刘仕裕
其他主演：董骠
　　　　　冯淬帆
　　　　　陈德容

收藏指数：★★

　　周星驰和老一辈的董骠、冯淬帆等人合作的老式鬼怪喜剧片。星哥很多后期作品的风格已经有所体现，只不过还未能尽情发挥。像片中星哥身穿T恤打领带装模做样念出“月上柳梢头，人约黄昏后”，已经完全是后期的喜剧风格；和冯淬帆在饭桌下“对脚”的段落也让人想到《点秋香》中的武打；而片中对抗邪魔法力的“法宝”也正是王晶电影的三大件“屎尿屁”——老猫屎、玉女尿、神仙屁，可以想象为得到这三件“法宝”，影片会搞出什么样的恶俗情节——不知道香港电影的“屎尿屁”传统是否由此流传下来，如果是的话那真是罪魁祸首了。

　　不过，影片的大部分情节和笑料实在是九十年代以前的产物，如今看来不堪一笑。
