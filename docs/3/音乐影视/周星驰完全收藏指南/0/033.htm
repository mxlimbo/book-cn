《情圣》

(1991)
导　　演：李力持
其他主演：毛舜筠
　　　　　恬妞
　　　　　午马
收藏指数：★★


　　如果你还没有收藏这部片子，那么千万要注意版本问题，很多版本把开头很长的一个段落给砍掉了，其实也没有必要那么敏感，任何时代、任何地域，都有它稍微过分的玩笑。

　　这部作品只能算是周式喜剧的中游之作，片中相当成份的笑料都是依靠“波霸”叶子楣那“骇人”的形象制造出来。这种手法本身的不雅，在香港影片中算是很平常的了，只是缺了一点幽默感多了一点恶心，而且还有滥用之嫌。比如程胜抱着占便宜的目的和喝醉酒的“波波”打kiss，突然“波波”酒劲儿上来一个呕吐……未免让人产生生理上的不适感，这就落了下乘。倒是阿星和老将午马的对手戏还挺精彩，两人在豪宅客厅里的一番推手以及对话很有料。“我一生孤独，和我相依为命的只有项上这颗头颅”，这么经典的台词应该是出自某部经典武侠影片了。

　　影片一再强调“用脑子赚钱”，反而恰恰是太注重这种观念，设计来设计去实在太做作了，绝对是麦嘉、黄百鸣和许氏兄弟的典型风格。午马、毛舜筠等人倒还罢了，周星驰却几乎被束缚住了，他的长处没有得到充分发挥。不过毛舜筠的笨到家了和星哥的自作聪明倒真实名副其实的一对活宝。

　　这部影片在语言上的一大贡献就是星哥常挂在嘴边上的“一坨屎”，影片结尾他来了一句“我只是在那个新式马桶上拉了一沱屎而已”，妙就妙在这个“沱”字上，这个量词实在是对汉语言文字的丰富。这句俚语有一段时间常在同学中传颂。“沱”，妙不可言。
