《龙的传人》

(1991)
导　　演：李修贤、李力持
其他主演：毛舜筠
　　　　　李修贤
　　　　　元华
收藏指数：★★

　　和同时期的几部影片一样，《龙的传人》是属于星哥正在趋向风格成熟的作品。香港喜剧片的传统笑料虽然也在片中占了一定的比重，但星哥自己的表演风格还是占了上风。影片开始很长时间，我们似乎看到的是一部周星驰的“龙少爷”，元华饰演的父亲也是名为“周飞鸿”的一方武术世家的家长。这种半古装与时装的混合，也正是与《新精武门》等片一脉相承。但从香港部分开始，星哥就恢复了大家熟悉的角色——既身怀绝技又傻乎乎的土包子样，这才是星哥安身立命、成名立腕的经典形象。毛舜筠虽然戏份不多，但憨憨的性格甚至比《情圣》中扮傻的角色要发挥得更好，很多笑料都更加自然。元华形象凶恶，演起喜剧来也是丝毫不落下风，反而给人冷面幽默的印象。片中周星驰背上“龙的传（全）人”的精彩笑料，也绝对非元华的冷面处理不可。星哥在片中大玩桌球，显然灵感来自美国88年的《金钱本色》，而片中居然真的邀请了Jimmy White，对桌球迷来说，也别有情趣。
