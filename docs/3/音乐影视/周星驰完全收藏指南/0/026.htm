《回魂夜》

又　　名：整鬼专家、整鬼大帝(1995)
导　　演：刘镇伟
其他主演：莫文蔚

收藏指数：★★★


　　这部影片之所以还有一点回忆的余地，全因为是由周星驰主演，至于影片情节实在不堪一顾。如果这部片中没有了星哥，那就将和十足的港产烂片没什么分别了。

　　影片中的很多细节摹仿了当年轰动世界的法国动作枪战影片《杀手莱昂》（这个杀手不太冷），只不过把情节换成了“捉鬼敢死队”。不过片中还真死了不少人，这对于一部喜剧片来说未免有点煞风景。星哥头戴棒球帽和黑墨镜，身穿暗色风衣，还做到长了一点络腮胡子这么有突破的扮相，难免令人觉得“有点冷”。至于文蔚姐实在是比《杀手莱昂》中的小女孩老很多，而且刚一出场时还陷在《堕落天使》里没出来，甩着一头黄毛乱蹦。法国原片中那盆著名的花，在这部影片里成了“整鬼专家”探测恶鬼的左膀右臂。另外，影片中星哥在精神病院的情节实在令人想到俄国文学大师布尔加科夫的惊世名作《大师和玛格丽特》。扯上这部名著并非是为星哥抬身价，只是又是精神病院又是大师的，难免会让人联想到那部辉煌巨著。

　　整部影片是个畸形的怪胎，喜剧和恐怖两种类型搅和在一起，又各自抢戏，不分主次，以至说恐怖又不吓人，说喜剧又让人别扭。而且就恐怖片来说，影片也毫无创意，只是简单把“头七”、“还魂”、“厉鬼”之类的旧套路摆出来而已。喜剧部分的噱头和笑料也不是由情节决定的，而是来自星哥的个人魅力和无厘头对另外一种类型片模式的戏仿（这部影片还谈不上颠覆）。可以说，影片的剧本和导演是非常失败的。

　　不能不说，港产影片的一大特色就是在配角和细节上比较用心用力，相对于编剧导演来说，这些才是能令人留下点印象的东西。这部影片里，几个特别出色的笑料，除了来自星哥之外，不少配角都很出色，捧逗具佳，留下笑声一片。尤其是喜剧导演李力持客串的小保安，戏份不多，但喜感十足，尤其是他不相信星哥的“鬼怪”之言走出门后看到厉鬼马上转回来向星哥极为“诚恳”地“道歉”一段，从台词到表情都是令人嘴里有什么喷什么的。

　　星哥在这部影片中还是以一种“专家”的身份出现，这正是他最劲的形象，“玩牌专家”、“整蛊专家”、“武功专家”“书画专家”、“卧底专家”，种种专家身份为星哥蔑视社会一般规则的“无厘头”表演特色提供出强有力的支持。而影片中捉鬼的窍门正在于突破普通人的心理限制，这种“精神胜利法”也是星哥演技的主要心理动机之一，因此星哥演来并不觉得困难，甚至还有超越自己以往角色的地方。如果影片剧本和导演能够达到哪怕只是普通水准，这部影片也能在星哥的作品年表上有一个独特的位置，但现在看来，只是一部勉强之作而已。但对于广大的“星迷”来说，既然有星哥在，那当然是必看影片了。至于那些特别着迷于星哥特色表演的观众，恐怕看了不止十遍八遍而已吧？！
