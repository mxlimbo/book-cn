《家有喜事》

(1992)
导　　演：高志森
其他主演：张国荣
　　　　　黄百鸣
　　　　　张曼玉
　　　　　吴君如
收藏指数：★★★


　　黄百鸣的喜剧片在八十年代还比较有市场，但到了星哥主宰的九十年代，就显得落后了很多。不过，这部影片的阵容真是整齐得没话说，成功也是理所当然，但也显出黄百鸣对把握时代潮流的力不从心。

　　大哥大嫂：黄百鸣、吴君如；老二张国荣和表姑妈毛舜筠；老三周星驰配张曼玉，这三对当时的“超豪华”演出阵容的确令观众大叫过瘾。不过，真正出彩的还是星哥这一对“超级影迷”。粗粗数来他们在片中调侃的美国大片就有：《与麦当娜同床》、《漂亮女人》、《幽灵》、《剪刀手爱德华》、《米赛莉》、《终结者》等等等等。尤其是周星驰装疯卖傻的段落，堪称星哥角色经典之中的经典。就凭星哥在医院里一句“四眼田鸡，你唬我呀”和一脸贱相被挂在晾衣杆上“淫荡啊谁淫荡”的镜头，打上三个“★”应该不算过分吧。
