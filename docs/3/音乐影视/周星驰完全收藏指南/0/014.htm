《九品芝麻官之白面包青天》

(1994)
导　　演：王晶
其他主演：吴孟达
　　　　　张敏
　　　　　徐锦江
收藏指数：★★★


　　这部古装公堂影片在笑料的堆砌上达到了登峰造极的地步，应该说比同类题材影片《审死官》更加肆无忌惮因而也更加自由。在《审死官》中比较“文气”的表现手法，在本片中则发展得更加世俗化，星哥也能够摆脱束手束脚的包袱，放手施展搞笑手段。

　　影片开始部分，对于“人民反对龙星·包”的描写夸张太过，反而令人反感。戚家的灭门惨案、豹子头办案、包龙星被陷害以及含冤“六月雪”等等情节都徘徊在正剧和闹剧之间，星哥也没有发挥出“无厘头”本色，影片总体氛围、风格也被破坏，是片中的败笔之处。从包龙星遇到莫再提、莫再讲兄妹开始，才算是好戏开场，而最出色的部分莫过于包龙星妓院练功和四堂会审两个重头戏。

　　从妓院练功到智斗权贵，真可以说是匪夷所思，但又是那样符合底层小市民的幻想。这个构思来自于星哥自身的写照，多年来他一直从市井语言中获得灵感，并使之风靡香港一登大雅之堂。把武打片的模式应用到练“嘴功”的过程中，实在是星哥的独创：偷艺、对着大海练功、吹蜡烛、舌战群鸡等传统武打片的经典学艺练功方式被星哥一一嘲弄个够。尤其是创造出令人目不暇接叹为观止的“螃蟹鱼虾”的海货“咸湿”语言和“&^$%$#*?…”的独特字幕，真不愧始终是搞笑界第一高手。会审公堂之上，星哥以来自民间的市井方式挑战了管道尊严和官场黑暗，一番唇枪舌剑真是淋漓尽致大快人心。这几个经典段落在星哥的“绝色”段落排行榜上恐怕都不会在二十名以外，这是影片成功的基础，但可惜的是影片的整体水平不算出色。

　　本片演员阵容堪称大卡司，除了周吴梦幻组合外，尚有张敏、钟丽缇、蔡少芬、苑琼丹、徐锦江、黄一山一干人等加盟。这些配角虽然绝非大星大腕，但却大都是星哥的老班底，对于营造完整的氛围是必不可少的。星哥的表演也保持了他的一贯水准，甚至在几个段落里绝对属于超水平发挥。
