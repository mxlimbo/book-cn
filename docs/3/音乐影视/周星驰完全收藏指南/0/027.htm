《玻璃樽》

(1999)
导　　演：谷德昭
主　　演：成龙
　　　　　舒淇
收藏指数：★★
　　1999年，成龙和周星驰各有一部“贺岁片”问世，为了振兴香港电影业，两个人互相在对方的影片中客串一个角色，以示相互支持。所以，成龙的《玻璃樽》里就有了这么一个周星驰镜头。可能是盲目的喜欢，一直认为这个镜头是整个片中最搞笑的。星哥的表演风格与成龙的影片很难相融洽，所以星哥的表现就像在这部片中的角色一样——十分勉强地牵着一只大狗，或者说是一只大狗牵着一只周星驰。不过，就冲这个镜头，对于星星的影迷来说，也应该买一张《玻璃樽》的VCD。
