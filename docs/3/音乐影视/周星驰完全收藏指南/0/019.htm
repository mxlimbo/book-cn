《赌侠》

(1990)
导　　演：王晶
其他主演：刘德华
　　　　　向华强
　　　　　张敏
收藏指数：★★★

　　星哥这部试图将《赌神》和《赌圣》的名牌“发扬光大”的作品，也是他早期的代表作品之一。还记得当时的海报上印着刘德华、周星驰破天荒联手合作之类的字样，以示其“超豪华”阵容。但刘德华不疼不痒的演技只能招徕喜欢他长相的小女生，而真正出色的还是周星驰和吴孟达这一对“黄金搭档”。单是星哥片头一段满嘴面条的“赌神”扮相，就够观众从头笑到脚的了，而赌船上星哥满脸唇印横着闯出房门的镜头更是经典之中的经典。整部影片与刘德华的正面“交手”的胜利，实际上已经预示了两个人以后的“星运”，刘大帅哥始终不能冲上票房的顶峰，而星哥则前途无量。这部影片的成功，也使得星哥下一年再度推出《上海滩赌圣》，继续自己的“赌坛”神话。
