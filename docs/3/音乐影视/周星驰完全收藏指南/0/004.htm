《破坏之王》

(1994)
导　　演：李力持
其他主演：吴孟达
　　　　　钟丽缇
收藏指数：★★★★


　　这部当年十大卖座片之首的片子，也是星哥和老搭档李力持合作的一部佳作。主人公阿银从开始到后来，始终是个普通小人物，只不过是个能以自己的生命和全部真诚去维护尊严和爱情的小人物，而且他成功了。这种故事和角色不用说，当然是给了星哥最佳的发挥空间。

　　影片开始的阿银憨憨傻傻，中间部分一根死筋，后来又勇气十足，从头到尾都贯穿了小市民品格中的高贵和猥琐之处，这个复杂真实而又夸张适度的角色，把星哥的表演发挥的淋漓尽致，绝对是星哥的最佳角色之一。这部影片也是星哥现代题材作品中比较完整的一部，剧本比较严谨，层层递进，前有铺垫后有呼应，只是断水流的出现略显突兀，失去了一气贯通的紧张感。

　　利用对传统类型片的颠覆性模仿搞笑，是星哥的拿手绝活。阿银出现时对《终结者》的模仿，就是所有“星迷”津津乐道的经典场面之一。阿银练“功夫”的段落，则是星哥对早期武打片的“致敬”，阿银一本正经地练着各种传说中的“绝技”，并在干活时“不知不觉”地施展出来。

　　影片最后的擂台场面，是星哥的擂台戏之中最出色的一个。早期《新精武门》中尽管一身“邪派功夫”，但到关键时刻还是正经的武功打斗；到了《漫画威龙》，擂台比武有了很大的进步，漫画式的夸张得以尽情发挥，但其基础还是一手超人的绝招。在这部比较写实的影片中，心理战术被夸张到极至，没有天赋神力、没有超凡绝技，有的只是坚韧的毅力和“不要命”的精神。值得注意的是，这个段落中很多手法脱胎于星哥早期作品《龙的传人》中的台球大赛。尤其是电视台的解说，既是心理战术中的重要环节，又能够笑料迭爆，在《龙》片中只是初具雏形，在本片中才得到绝佳处理。

　　另外，片中对“月亮”的处理也令人联想甚多。早在《家有喜事》中，星哥就以这个好莱坞爱情电影中的必不可少的“道具”征服了曼玉姐姐；在《九品芝麻官》之中，硕大的月亮也一定给各位观众留下无比深刻的印象；到了《大话西游》中星哥还是和牛夫人晒月亮、叫人家小甜甜。而在本片中，阿银的失意时刻，所到之处灯光尽熄，最后连一角弯月也熄灭了，对这个“道具”的创造性应用，可以说是“无厘头”表现方式的极品。

　　不可否认，影片中还有一些庸俗无聊的噱头夸张，但其整体水准却绝对是第一流的，周吴组合各有各精彩而又配合默契，奉献出很多经典场面和对白，像练功夫、下战书、地狱式魔鬼特训等段落都能长留“星迷”心中。
