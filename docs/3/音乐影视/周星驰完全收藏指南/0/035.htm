《义胆群英》

又　　名：龙蛇争霸(1989)
导　　演：吴宇森、午马
其他主演：姜大卫
　　　　　李修贤
　　　　　午马
收藏指数：★★

　　这是一部值得回忆的影片，倒不是因为曾经逃课出去看过这部大银幕上的大制作枪战片（当时的电影院中这种片子还是太少了），而是影片的阵容实在强大。这是武侠片大导演张彻从影二十五周年（？）的纪念之作，吴宇森为这位广受尊敬的老导演监制，据说还代为完成了一部分枪战场面。主演中有李修贤、姜大卫、邝美云等一大批男女明星，其中最受瞩目的年轻人就是周星驰。星哥饰演的小齐是个典型的义气少年，被人利用，在开头不久就惨死枪下。星哥戏份不多，但形象鲜明可爱，是他又一个出色的正剧角色。如果说这部影片还有一些收藏意义的话，那就是因为片中大量出色的枪战场面。也许是因为很少在大银幕上看到这样激烈的场面，也许因为有了吴宇森参与的印象，在我的记忆中，这部影片的枪战场面甚至可以直追《喋血双雄》、《英雄本色》等经典杰作。。
