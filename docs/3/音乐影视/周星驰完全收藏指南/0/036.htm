《算死草》

又　　名：整人状元(1997)
导　　演：马伟毫
其他主演：葛民辉
　　　　　邱淑珍
　　　　　莫文蔚
收藏指数：★☆
　　这部影片是我最不愿意回忆的，我实在没有勇气面对自己最喜欢的演员拍出这样一部蒙垢遗羞的作品。该片是星哥后期作品中最惨不忍睹的一部，实在难以和《审死官》、《白面包青天》等同类“公堂片”相提并论。影片中从头到脚太多的“屎尿屁”臭烘烘地令人恶心，大便小便、屁股肛门，所有能让人生理上起恶劣反应的事物都被拿来加工成笑料。如果挪用一句经典的评价就是：这是一部不能不看的影片，也是一部绝不愿意再看的影片。作为一个“星迷”，怎么能不看自己偶像在成名之后最烂的片子呢，而再看一遍无疑是对自己的一种折磨。加上半个“★”完全是为了照顾情绪，给星哥面子而已。对于这部影片，最积极的想法就是星哥重新剪辑一个“洁版”，哪怕只有半个小时长度，也能让人多看两遍。。
