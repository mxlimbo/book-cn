《新精武门1991》

(1991)
导　　演：左颂昇
其他主演：钟镇涛
　　　　　张敏
收藏指数：★★★

　　一个少年英雄，一只右臂天生神力，几经周折拜师学艺，最后成了擂台冠军，抱得美人归。从故事大框架上上，几乎看不出这是一部“星派”电影。但也需知，“星派”影片以细节取胜，故事框架倒无关宏旨。天生绝技而又憨笨可笑，这个人物形象是典型的“星派”主角。而最有意思的是这个主角的学艺过程，四个老得牙都掉了的糟老头一心想把他培养出来，去争夺大赛奖金，而且传授的“绝艺”竟然全是韦爵爷的拿手绝活。这个创意就的确不错，再加上“潇洒哥”钟镇涛的“鼎力相助”，片子想不可笑都难。但这毕竟还是1991年的作品，表演风格的不够成熟和落后的时代感，将有碍现在的观众喜欢星哥这部早期作品。
