《百变星君》

又　　名：百变金刚(1995)
导　　演：叶伟民
其他主演：吴孟达
　　　　　梁咏琪
　　　　　徐锦江
收藏指数：★★★


　　几乎没看到过确实的文章报道美国的金·凯瑞向周星驰学习了什么，但不可否认的是，周星驰的确在跟金·凯瑞的风，至少这部《百变星君》就明显在跟《变相怪杰》的风头。其实这只是好莱坞的势力影响所至，星爷的影片别有天地、自成一家，以一贯的态度面对复杂的人生，比金·凯瑞过于夸张但缺乏内部精神基础的表演实在要高明。不过，星爷作品中太多受到低级趣味的商业因素影响，很多剧本瞎拼胡凑、低级的“屎尿屁”笑料充斥其中，使作品水准大打折扣，《百变星君》也自然不能幸免。

　　和星哥大部分影片相反的是，影片前半部分中，星哥的角色是个没有本事、靠钱拽起来的败家子。而以往的影片，星哥多半都是很穷或是很倒霉，但有一种特殊的“功夫”帮他战胜压制他的环境，这样的角色更适合大多数中底层小市民的幻想和带入。在这半部影片里，“百变星君”的“变”还谈不上，只是迎合香港影片一贯的低级口味大肆炮制无聊的笑料。星哥的表演虽然偶有发挥，但基本上特色并不突出。李寿星的胡闹整人手段实在有够低级，手法和想象力甚至比不上几年以前的《整蛊专家》，只有几句星哥特色的对白还保持了一贯的风格和水准。最抢眼的就是星哥对金·凯瑞的服装和夸张动作的模仿，但这既非原创，又有点味道不正。此外，对94年美国当红影片《低俗小说》的调侃和戏仿也是值得一提的，但这一段调笑过度、生硬突兀，既比不得国产007成系统成规模的颠覆性戏谑，也不是随兴而至、即兴而发的片刻幽默，总之是一段不成功的情节。

　　在李寿星被炸得只剩一张嘴之后，影片才真正进入周星驰自己的天地。虽然主要的噱头是在模仿金·凯瑞，但戏法人人会变，各有巧妙不同，星哥一手“拿来”，再消化消化就成了地地道道的港产绝活。尤其是星哥的角色是废品拼凑出来的低级货，这更适合星哥以往的路数。既然星哥当老师的瘾头还没过去，那就再回学校，暴力加怀柔，彻底制服这些“坏孩子”。可惜，星哥玩得有点太过火了，再加上王晶这个低级趣味的家伙，把这个本来挺有可能出彩的段落弄得鸡飞狗跳，不但比不上星哥的代表佳作《逃学威龙》，甚至连王晶自己胡闹的《逃学英雄传》都比不上，令人愤愤。

　　此外，前面出现的铁嘴钢牙妹虫虫女大十八变，居然这样漂亮，真是想不到、想不到啊！星哥在夜总会中追求虫虫，捉弄校董的段落与金·凯瑞的同样段落算是各有千秋吧。一个跳舞，一个演魔术，比较起来《百变》本来在喜剧性上还略胜一筹，但就是其中的不雅镜头落了下乘，尤其是洗手间的镜头太低级了。随后两人的恋爱段落夸张得可爱，虫虫、星星打排球等发骚的几个爱情镜头，明知道“骚的过火”，但还是觉得骚得可爱，骚得人心痒痒。这种肉麻也能当得有趣，大概就是星哥的功力所至吧。

　　影片结尾的一段“超级市场”廉价货对名牌高档货的大比拼，想象力丰富，分明是个现代版本的孙悟空七十二变。比起金·凯瑞一个形象毫无变化的片子，这部影片似乎更适合华人观众的口味。
