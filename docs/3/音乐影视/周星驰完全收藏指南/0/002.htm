《国产凌凌漆》

又　　名：凌凌漆大战金枪客 (1994)
导　　演：周星驰　李力持
其他主演：罗家英
　　　　　袁咏仪

收藏指数：★★★★★

　　第一次看完这部片子，一屋子的人都快要笑疯了，当时我就认定它几乎能和《大话西游》同列为星哥的极品代表作，没有认真学习二十遍以上的不要妄称“星迷”，如今再看仍然有常看常新、常看常笑的功效。能把好莱坞类型片进行颠覆性戏噱而又能做得如此完美的，舍周星驰再无第二人，白头尼尔森和查理・辛之流只能破坏而不能建设，实在难比星哥的才气。

　　相信任何观众都不会忘记这部影片的片头，星哥把共舞美女扔出画外先、拔枪射杀后，还能如此“优雅”地摆出pose，也算古今中外第一人了。也相信任何一个“星迷”都能和我一样背诵“你以为你躲在这里我就找不到你了吗？没有用的，你是那样拉风的男人，那忧郁的眼神、唏嘘的胡渣子、神乎其技的刀法，还有那杯dry马蒂尼，都深深地出卖了你。虽然你是那样的出色，但行有行规，无论如何你也要付清昨晚的过夜会（费）。”

　　随着一声“力拔山兮气盖世”，星哥开始了和家英哥的合作，也开始了自己的“凌凌漆”身份。从和司令部女秘书的肉麻调情、家英哥的无聊发明到潇洒扔出一块猪肝、照例问一句“不去行不行”，所有007影片的情节和细节模式，星哥一样不落，全盘招呼，作足了功夫。星哥和袁咏仪见面之后，好戏才正式开演。一番刮胡刀、吹风机的调侃大大启发了007影迷的想象力；“古灵精怪枪”果然名副其实；当然还有那句“飞是小李飞刀的飞，刀是小李飞刀的――刀”。

　　接下来的情节都不完全是搞笑，更多的是刻画了人物，把“凌凌漆”的“另一面”展示给观众，这是和以往星哥影片不同的地方。星哥的绝大部分影片，如《赌圣》、《逃学威龙》、《点秋香》等等，都是一上来就把人物的性格、身份、能力、绝招等等全部展现出来，不留余地，即使如《大话西游》由人成仙这么大的变化也是随着情节发展而来的，过程让你看得一清二楚。惟独这部“凌凌漆”，星哥开始只是“犹抱琵琶半遮面”，一直到最后才完全显露身手，堪称星哥作品的一个独特的“变奏”。这个变化的结果，就是一直有个悬念，甚至维持到影片最后几分钟，观众也总是处在一个兴奋的状态中。

　　“古有关云长全神贯注下象棋刮骨疗毒，今有我凌凌漆聚精会神看A片挖骨取弹头。”不用说，这句经典台词当然是凌凌漆受伤后在“漆女郎”袁咏仪的家里取子弹的段落里留下的豪言壮语。这个段落是所有人都不会忘记的，一方面我们看到了A片居然还有这种功能而为之振奋，另一方面也是被编导在处理这个喜剧构思时表现出的伤感所感动。画面的色调构成以黄蓝两色为主，呈现出一种很奇特的情绪色彩，衬以纯白的西服、殷红的鲜血，成为笑与愁的怪诞组合。

　　虽然影片是以大陆军区为势力背景，但这是当不得真的虚构，也没有人如此对号入座地去看星哥的影片。所以，最后一百块钱贿赂枪决执法官的大笑料也只不过是个“笑料”，恐怕只有做贼心虚的某些人才会提出“亵渎党和国家”之类的非议。而影片最后的镜头定格在那把著名的杀猪刀上，“民族英雄――小平赠”七个字也同样是个无伤大雅的玩笑。

　　总之，这绝对是一部星哥难得的代表作、极品，如果一定要说影片有什么缺陷，那么只能说片子有点稍短，略感遗憾而已了。
