《济公》

(1993)
导　　演：杜琪峰
其他主演：张曼玉
　　　　　吴孟达
　　　　　黄秋生
收藏指数：★★


　　照理说，93年正是星哥创作上的黄金时期，虽然没有95年时候的成熟机智，但佳作还是不少的，但这部《济公》却是周星驰最差的几部影片之一。片中降龙披头散发的造型实在难以用“另类”来形容，太糟贱周星驰了。吴孟达扮演的弱智儿伏虎也让观众觉得自己ＩＱ出了问题。其他无论张曼玉之类的明星还是玉帝王母等等各个普通角色，不是根本就演不出喜剧感觉，就是自己糟蹋自己，简直是对喜剧的侮辱。要知道星哥是用他的智慧和风格让大家开心的，而不是仅仅靠制造愚蠢贬低自己。

　　该片剧情混乱不堪，如同一堆乱麻，太多的无聊插曲既打乱了情节发展又不够搞笑。最要命的是，细节和噱头的处理上也粗制滥造，胡闹的地方太多。从故事上说，一个神仙下凡点化一个九世乞丐一个九世野鸡一个九世恶人，其实满有文章可做，济公这个人物也满可以通过星哥的重新演绎而成为香港喜剧电影的经典人物。不过，事已至此，也没什么好说的，总不成让星哥重新再演一遍吧。这就是电影，金大侠可以一再修改重写十几本书，可周星驰却不可能重新再把失败的片子重演一遍。。
