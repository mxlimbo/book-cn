《豪门夜宴》

(1991)
导　　演：徐克
　　　　　高志森
　　　　　张同祖
其他主演：很多，或者说没有

收藏指数：★

　　这一年，大陆水灾肆虐，香港演艺界众多明星纷纷义演募捐，这部号称香港“全明星”出演的义演片就是其中一项重要活动。几乎所有的香港艺人都在片中出现，90分钟明星不断，当然少不了正当红的星哥。虽然影片是几十年前粤语长片的重拍版，但这样紧急展开的庞大活动很难要求它有完整出色的情节，各位明星也无非是亮个像，表表心意。所以，星哥几乎没有什么值得欣赏的镜头，所以对于“星迷”来说，收藏与否无关宏旨，但对于喜欢香港影星的朋友，这倒是一部香港当红明星的收藏纪念册。
