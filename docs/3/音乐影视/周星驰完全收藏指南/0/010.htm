《大内密探零零发》

又名：鹿鼎大帝(1996)
导演：谷德昭
其他主演：刘嘉玲
　　　　　李若彤
　　　　　罗家英
收藏指数：★★★


　　虽然都是模仿007的开头，但这部古装戏其实和007系列没什么关系。1993年到95年是星哥的全盛时期，95年的《百变》、《回魂》已经有了走下坡的趋势。《食神》虽然选题独出心裁，但剧本、表演都未能尽如人意，这也是几乎所有贺岁片的大缺陷。这部《大内密探》属于星哥费尽心思打造的暑假强档，应该代表星哥96年的最高水平，但可惜我们并没有看到能和《大话西游》、《国产凌凌漆》同等级别的佳作。

　　从片中大小噱头来看，星哥肯定对这部影片灌注了不少心血，甚至寄望很高。如片头的“紫禁之巅”的一段“大话小凤”，从一开始就有个“碰头彩”，再有零零发的种种发明，搬演解剖外星人，最后的“奥斯卡颁奖”等等，都是大有可为的精巧设计。可惜的是这些古怪零碎太多了，其实只要拿出一样大做文章，都能十分精彩，但太多的堆砌反而事倍功半。最关键的失败之处还在剧本，前后两段虽然都是关于“无相神君”，但脱节现象严重，仿佛两个不相关的故事硬拼在一起的。此时的星哥可能是太注重设计精彩的“小节”，而忽略了整体的效果。

　　不管怎么说，作为一部完整的电影，《大内密探》并不算成功，但作为星哥的一个主题小品集还说得过去。应该说，星哥的个人表演并不逊于他稍早的几部代表作，留下了几个经典级的段落。最有意思的是结尾前，一段模仿奥斯卡颁奖的场面中，星哥对于没能领到奖项抱怨。罗家英教训星哥：你表情生硬动作浮夸。星哥的回答很有点自我辩护的味道：“那是世人对我的看法。”不知道星哥说这话的时候是不是真的有底气。

　　片中的配角发挥不如以往，李力持、刘以达的搭档有点闹过了头，刘嘉玲毕竟不是星哥的班底演员，罗家英的戏份本来就不多。
