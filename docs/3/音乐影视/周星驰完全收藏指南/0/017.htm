《鹿鼎记II之神龙教》

(1992)
导演：王晶
其他主演：林青霞
　　　　　丘淑贞
　　　　　李嘉欣
收藏指数：★★★

　　同是《鹿鼎记》中的情节、人物，主演也大致相同，但这部影片就是不如上集。最主要的改变是林青霞的加盟和吴孟达的退出，以林青霞的身份自然要加重她的戏份，但她演喜剧片实在是个天大的误会，而陈百祥的多隆也远没有吴孟达的海公公与周星驰配合默契。此外，星哥在影片中没能控制住自己表演的夸张程度，以致失去了漫画式表演的适度平衡。像片中独臂神尼与“我爱一棒槌”之类的狂闹段落，不免有些过于胡闹。冯锡范和神龙教的出现，很大程度上打乱了影片的喜剧氛围和节奏，其结果就是喜剧性被削弱，正剧部分也不伦不类。

　　当然，以星哥出演韦小宝，再怎么样也是值得一看的，多多少少也会留下一些难忘的经典段落和镜头。君不见互联网上有多少论坛名为“有间客栈”，又有多少网上高人化名“西方失败”，自然都是广大“星迷”所为。
