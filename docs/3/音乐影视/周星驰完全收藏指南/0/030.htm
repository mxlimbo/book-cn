《97家有喜事》

(1997)
导　　演：张坚庭
其他主演：黄百鸣
　　　　　吴镇宇
　　　　　吴倩莲
　　　　　伍咏薇
收藏指数：★★


　　黄百鸣其实应该意识到自己的喜剧风格和周星驰相比绝对是上个时代的产物，生硬拼凑的结果就是两部风格凌乱的《家有喜事》。从整部影片上来看，情节和第一集雷同的太多，以至于没有什么新鲜感。不过，这种贺岁片的好处就是不用管情节、主题、结构，只要细节把观众逗笑就行了，所以星哥在其中稍显突出也就不是什么大事了。和上一集差不多是装疯卖傻，周星驰的角色在片中仍然是最有笑料的，少了张曼玉配戏的星哥反倒更自由一点，就是黎姿和钟丽缇的表现差了很多。星哥的戏份肯定没有自己单独主演的那么多，但也足够看的了，虽然没有上一集那么多调侃美国大片的噱头，很多独有的细节魅力还是能令广大“星迷”满意的。像“老头，没见过萝卜”、“丧尽天良啊——，禽兽不如啊——”等经典台词也成了星迷们生活中模仿率很高的对白。

　　片中其他角色几乎没有一个和星哥站在同一个水平线上。黄百鸣的搞笑还是那样做作，伍咏薇的表演和她的名声一样稚嫩；吴镇宇和吴倩莲并不太适合这样一部喜剧贺岁片，吴倩莲平常的表演都还不错，就是一想搞笑就离了谱，像包公上身那一段闹剧就实在令人不敢恭维。不用星哥自己的班底拍星哥的喜剧片，就像不用成家班而拍成龙戏一样，失败是注定了的。
