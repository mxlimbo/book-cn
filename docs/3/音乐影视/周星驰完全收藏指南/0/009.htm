《逃学威龙》

(1991)
导演：陈嘉上
其他主演：吴孟达
　　　　　张敏
收藏指数：★★★★

　　这部王晶制片、陈嘉上执导的影片可以说是周星驰风格的代表作，无论是剧作结构还是笑料噱头，都还是挺认真的。周星驰和王晶很多影片中的屎尿屁笑料，由于本片是学校题材而干净了许多。而正处于上升期的星哥，演技发挥的极为出色。努力就不用说了，单是笑料的设计就费了不少苦心，很多经典的周星驰语言和行为就是从这部影片开始的。由于影片在学校题材上的成功，还带动了一大批港台两地的学校影片出现，成为一股风潮。

　　影片的故事固然很多人已经烂熟于胸，但还是不得不说上一句，影片的剧作应该是星星哥这么多作品中的佼佼者。飞虎队第一杀手卧底学校做学生，这种角色简直就是为星哥度身定造的，一种情况下是绝顶高手，一种情况下是绝对菜鸟，身份的大反差正是星哥发挥演技的最佳条件。

　　影片从周星星进入学校，就开始营造喜剧氛围，空旷的大门前孤零零的星星显得好无助啊，而学校里面简直象个集中营，还有那多得可怕的校规。学校里的老师都有点“病态”：物理老师是个喜欢拿板擦丢人的尖刻女人，化学老师什么都记不住，训导主任林作栋象个纳粹军官，好在还有MISS胡——张敏姐姐。同学里当然有收保护费的，但这些难不住飞虎队的星星哥，倒是作业和考试比较麻烦些。不过，既然有美女胡老师辅导，当然会多花点心思学学。

　　影片中的笑料当然很多，随口就可以说出一大堆：化学课上的那一段应该算是经典了吧；用钱买来作业后对正在埋头抄写的小龟说的一大段声情并茂的台词简直就是《英雄本色》周润发经典台词的校园版；历史课作弊的段落也不用多说了，把作弊夸张成赛马恐怕也只有星哥才想得到；星哥在片中挨吴孟达和警察老总的几顿痛扁，都达到了很好的喜剧效果。

　　另外，几个喜剧性很强的配角起到了绝佳的陪衬作用，老搭档吴孟达的默契配合只能用四个字形容“舍我其谁”；张敏姐姐最佳表情将永远留存在星迷们的记忆簿上；贼兮兮的小龟一脸贱相；该片编剧黄炳耀客串的老总以火爆脾气和结尾处的神勇现身令人印象深刻；化学老师更是让人难忘，直到几年后的第三集中，他还又出来露了一脸；苑琼丹饰演的地理老师大概是她最早出现在星哥影片中的角色，片中一番作弊理论深得人心。影片喜剧的噱头和笑料尽管很多，但基本上并没有流于低级无聊，比起王晶和周星驰稍后时期的很多娱乐片强了不少，可能这也是这部《逃学威龙》比那些影片耐看的缘因吧。

　　这部电影尽管是喜剧动作片，但动作场面并不太多。虽然如此，不多的几个动作场面设计得都是干净利落，不像很多动作片，打起来没完，拖泥带水，甚至最后的打斗场面时间也很短。这固然是星哥不善打斗、影片以喜剧为主，但也体现出王晶对处理动作场面的简洁熟练。其实，这部影片倒是应该归为喜剧爱情片更合适一点。

　　此外，对你收藏的版本也应该稍加注意，粤语版本中的几个镜头可能在大部分国语版本中被删掉了。
