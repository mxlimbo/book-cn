热情如火

　　热情如火(Some Like it Hot,1959)

　　提名者：《桃色血案》、《安妮少女日记》
　　　　　　《修女传》、《金屋泪》

　　得奖者：《宾汉》

　　有的时候，高尚与拘谨加在一起，会毁了一部电影。导演佛烈辛尼曼的奥斯卡提名作品《修女传》就是最佳例子，它的高尚与文雅的确值得鼓励，但是就仅只于此。现在我们请它一边让，真正的好片出头。

　　比利怀德历久弥新的经典电影《热情如火》，是一部黑道喜剧电影，由杰克李蒙与汤尼寇提斯主演，他们因为目睹谋杀案而被黑道追杀，为了逃命，两人化妆成女人，混在一个女子交响乐团中。不久后他们发现，逃命根本是小事一桩，要如何向性感的乐团女子玛丽莲梦露表白，才是比较重要的呢！

　　比利怀德因此片获奥斯卡最佳剧本奖，剧本中充满了诙谐的智能，加上杰克李蒙(获最佳男主角提名)、汤尼寇提斯与玛丽莲梦露顶尖的演技，还有比利怀德指导的摄影，也获得奥斯卡最佳摄影奖的提名。但是，这样一部好片中的好片，却没有获得最佳影片的提名。

　　有人喜欢喜剧，但是奥斯卡却鲜少看得上眼。这一回，《热情如火》没有提名奥斯卡，可是自己闹了一个大笑话呢！
