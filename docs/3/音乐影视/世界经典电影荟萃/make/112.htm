好心与王冠


　　1902年8月7日，在英国伦敦的彭顿维利监狱里，因犯谋杀罪而被判处死刑的查尔方特十世公爵正赶在上绞架前夕，匆匆地写着回忆录：

　　我叫路易斯·达斯科因·马齐尼。母亲是查尔方特公爵七世的女儿，因爱上了一个意大利歌唱家，并与之私奔，遭到了家族的唾弃。当我刚踏入人间时，父亲的心脏病突然发作，与世长辞。由此带来的生活艰难，迫使母亲委屈求全，与自己的本家和解。她接连写了几封信投寄查尔方特城堡，可恨的是，俱都泥牛入海。她只得去做各种卑微的杂活，挣钱维持我们母子的生活。在我童年时，母亲常给我讲她家族的历史和家谱。她认为我将来有可能成为公爵，因为在达斯科因家族里有一条皇上特授的权力，即家族中的母系与父系的后裔均有权继承爵位。为此，她省吃俭用，让我从小接受良好的教育。

　　我长大了。继承爵位始终是一个不切实际的幻想，因为达斯科因家族还根本不承认我是他们中的成员。拮据的生活迫使我放弃学业，无奈在布店里当了个小店员。亲爱的母亲终于也追随父亲去了。她的死仅仅是为了没钱修理打碎的眼镜，愁怅地在街上被车撞倒的。我遵照她的遗愿，写信给公爵八世，恳求他同意葬在家族墓地。谁料他们竟会十分残忍地拒绝。我只得将母亲安葬在郊外一个荒凉的墓地，在坟前我发誓：对那个百般侮辱她的家族，我将实行报复！

　　厄运接踵而来。不久，我那青梅竹马时的女友西贝拉也拒绝了我的求爱而同另一个没有教养的纨绔子弟莱昂内尔结婚了。原因很简单：他有钱，我穷。这变故使我下了决心，不惜代价实现继承爵位的幻想。我详细地了解了家族成员的情况。如果我有了继承权，在我前面还有八个人是障碍。难呀！除非出现奇迹，这八个人一夜之间全都死光……由此，我想到了谋杀，而且考虑起周密的计划来。

　　命运竟然出乎意料地帮了我的忙。一天下午，家族中的一员--银行家达斯科因勋爵的儿子，和女友来到我们店里买布料。他傲慢地侮辱了我，我仅仅顶撞了他一句，便被当场解雇。我的报复决定从他开始。几天后，我悄悄地尾随他们来到梅登黑德，当他俩在野外河边的一条小船上，用阳伞遮着作爱之时，我偷偷地解开了系船的绳子，让小船在他们的忘我中顺流漂去。我知道河的下流有一段十分危险的水域，那船肯定漂向天堂……回来后，我在我的家谱中毫不犹豫地划掉了一个名字。同时，我给他的父亲、我的表舅--银行家写了一封深情的信表示哀悼，我只是想以这种方式来寻开心。因为以前，母亲曾为了我的职业，向他恳求而遭到过他的拒绝。真出乎意料，那银行家竟然感激我对他痛苦之时的关心，邀请我在他的银行里做个小职员。

　　清除的下一个目标，该轮到小亨利·达斯科因了。他是个摄影家。我投其所好，很快我们成了"朋友"。小亨利的夫人伊迪丝，长得非常美丽，性格有点古板。她不许丈夫喝酒，而小亨利却常常瞒着她躲到一间放花盆的小木棚里偷偷地喝。我发现了这个秘密，于是一天夜里，我爬进木棚，将煤气灯里的煤油全都换成汽油。第二天，小亨利躲到这里过酒瘾时，煤气灯突然爆炸引起火灾，他被烧死了。葬礼在查尔方特教堂举行。遗孀伊迪丝在最痛苦的时刻，仍然仪态端庄，确是一个才貌双全的女人。我暗下决心，有机会便向她求婚，因为她也了解了我的身世并对我怀有好感。由她我又想起了西贝拉。啊，这也是个令人着迷的女人，可惜她不配戴未来公爵夫人的冠冕。尽管她目前不满婚后生活而一直与我暧昧地来往。

　　好心的银行家不知是出于对我母亲的忏梅还是其它什么原因，他承认我是家族中的成员，并任命我为他的私人秘书。这个职务原来是留给他的儿子的。如此看来，我走的第一步确是成功的。

　　有了地位、有了钱，事情就好办多了。我继续执行着谋杀计划：我装扮成主教来到牧师老亨利·达斯科因的教堂里，对他下了毒……我用暗箭射死了妇女运动的倡导者--阿茄莎·达斯科因……我又在鱼子酱罐头里装上炸弹，邮寄给陆军上将鲁弗斯·达斯科因，让他飞上西天……至于海军上将霍雷肖·达斯科因，用不着我下手，便在两艘军舰相撞的事故中葬身海底……家族成员一个个意外地死去，使银行家相信是因过去大家作了恶，上帝给了应得的惩罚。于是，他不再要我做他的雇员，而要我和他合伙共事。我真感到意外的荣幸。不久，他得了脑溢血，医生说最长只能活一个月。感谢上帝，使我解除了对这个多少于我有恩的老人下手的为难。愿上帝也保佑他吧！就这样，我唯一的障碍就是公爵八世本人了。要解决他我毫无办法，因为我对他的情况一无所知。正当我一筹莫展时，那摄影家的遗孀伊迪丝同意了我前不久向她求婚的要求，并把我俩的关系告知公爵，公爵邀请我们去城堡小住几天。

　　我觉得自己成了全伦敦最幸福的人，但并非没有烦恼的事。一天，西贝拉的丈夫莱昂内尔前来向我借款。因为他买卖失利，再加上西贝拉的滥肆挥霍是个填不满的无底洞，坐吃山空、行将破产。他苦苦哀求说，如果我不帮他的忙，他只有一条出路--自杀。我冷冷地拒绝了他，报复他当年夺我所好的行为。当他被我赶出门去不久，西贝拉也来了。她告诉我：我与她之间的苟合被丈夫发现了。她想与丈夫离婚，重新嫁给我。她想得倒美！我也谢绝了她，并告诉她，我已经和伊迪丝订了婚。她闻此大为恼火。我却感到痛快，谁叫她当年因我穷而拒绝过我！这叫一报还一报！可是，我确没想到这一举已播下失败的种子……

　　几天后，我和未婚妻伊迪丝来到查尔方特城堡。在城堡小住的几天里，我寻找到一个机会，在野外打猎时，神不知，鬼不觉地将公爵八世枪杀了……大病在身的银行家听说自己继承了爵位，成为公爵九世时大为震惊，没几天也一命呜呼了。我的愿望终于实现了：我为不幸的母亲报了仇！我成了查尔方特十世公爵！

　　那天，当我和伊迪丝在城堡举行隆重的婚礼时，伦敦警察厅逮捕了我。打击来得真是突然，我真不知所有的谋杀活动倒底哪里出了纰漏？会不会是西贝拉告发的？！因为她早已怀疑我家族成员的死亡与我有关，到了警察厅，我才知道，果然是西贝拉捣的鬼！不过她告的却是说我谋杀了她的丈夫莱昂内尔。在法庭上，她竭尽手段编造谎言，说是莱昂内尔要告发妻子的不贞行为，而我怕为此影响当公爵，影响同有钱又美貌的伊迪丝结婚，就谋杀了他，并且十分残忍地抛弃了她。这纯属无稽之谈！我向法庭辩白，说莱昂内尔死前曾到我这儿，他露出过企图自杀的念头。然而法庭不相信我，却相信西贝拉的指控。于是，最终的审判一致通过对我处以死刑！天哪！这真是天大的笑话！我杀了那么多人，法庭一概不知，而莱昂内尔的死与我丝毫无关，法庭却深信是我下的毒手。好罢！对这样的法庭我也无话可说了。反正从自己的良心深处自我判决，我也是罪有应得。平静地等待上绞架吧！今天上午，我夫人来向我告别，时至如今，她仍深信我无罪。她走后，西贝拉也来了，她向我暗示说她能找到说明莱昂内尔是自杀的遗嘱。但救我的交换条件是，将来要我除去伊迪丝，让她当公爵夫人。啊，真是一个活脱脱的西贝拉！除了答应她，我还能有什么办法呢？不过以后究竟谁当公爵夫人还得由我决定，我不喜欢他人左右我……

　　路易斯的自传写到这里为止。第二天一早，上绞刑前的一分钟，典狱长跑来对他说："大人，恭喜你！莱昂内尔的遗书找到了，证明他是自杀的，西贝拉撤回起诉，你被无罪释放了！"路易斯早料到这在他人眼里看来是奇迹的事会发生，他太了解西贝拉了。他平静地走出监狱大门，看见伊迪丝和西贝拉都在等他，他想："只要两位美人不在一起，和其中的任何一个，我都会无忧无虑幸福无比……"这时，从欢迎他的人群中，钻出一位记者对他说："我是'珍闻杂志'社的。听说公爵在狱中写了回忆录，特受命前来与你协商出版事宜。"

　　"回忆录？"路易斯一听顿时露出极度惊愕的神色。天啊！回忆录！！他把自己写的回忆录遗忘在死囚牢里了……
