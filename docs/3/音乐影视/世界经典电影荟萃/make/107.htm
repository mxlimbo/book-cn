布贝的未婚妻


　　一列火车在原野上奔驰。车厢里，玛拉坐在靠窗的座位上，神情严肃，凝视着窗外……七年前，她的未婚夫布贝为战友报仇杀了警长的儿子而被判处十四年徒刑。七年来，玛拉没错过一次探监的日期。从认识布贝到现在，玛拉的遭遇是痛苦的，但也有幸福的时刻，在去监狱的途程中，玛拉总是禁不住要回想这些往事。

　　那是1944年的夏天，意大利刚从德国的铁蹄下解放出来。当时的玛拉还是一个天真动人的少女，她认识布贝完全是因为她哥哥的缘故。哥哥是游击队员，在战斗中阵亡了，作为战友，布贝极力安慰她一家人。他那坚定深沉的性格在玛拉的心中留下了深刻的印象。不知不觉的，玛拉在心里总期望着看到布贝。布贝也感到了玛拉的炽热感情。他们很少见面，即使有信件往来，也都非常简短，但玛拉还是从布贝信中的字里行间看出，他是爱她的。

　　一天，布贝终于又来了，但他却是来告别的，他要到一个很远的地方去工作。玛拉生气了，她觉得布贝什么事都自作主张，一点儿都不尊重她。所以，分手的时候，她赌气似地和他冷冷地告了别。

　　谁知，不久布贝就出事了。在他们工作的地方有个警长，他是个十足的法西斯分子，他对游击队员恨之入骨，总是挑他们的刺。一次，他故意寻事开枪打死了他们中的一个。为了报仇，布贝他们打死了这个警长。当警长的儿子前来厮杀时，布贝开枪打死了他。这样，警察局下令通缉布贝，布贝逃回了家乡。

　　布贝又和玛拉见面了。一天，他把玛拉带回家，让父母见见自己的未婚妻。可是到布贝家的第二天清晨，玛拉被一阵紧张的叫喊声惊醒。布贝的战友来告诉他，警察在到处搜捕他，党组织决定让他立即躲避起来。玛拉不管布贝怎么反对，也要跟他一块去。

　　村外的一座工棚，四周是一片青翠的草地，小溪蜿蜒，绿树婆娑，本是谈情说爱的好地方，可现在两人都焦虑不安。"我们要在这儿呆多久呢？"玛拉问。"不知道。"尽管布贝很轻松地说，但玛拉仍感到他内心的沉重。她一下子变了，她不再像以前那样任性了，她处处为布贝想，她要安慰布贝、保护布贝。每当玛拉用炽热的目光望着他的时候，总遇上他同样炽热的目光……在这样残酷的处境下，两颗挚爱的心连在了一起。

　　几天后，党决定把布贝送往国外。这对热恋中的情人是痛苦的。犹为痛苦的是布贝，他太爱玛拉了，一心想使她幸福，但又办不到。分别时，玛拉要布贝像丈夫那样拥抱她时，布贝终于痛苦地对玛拉说："你重新考虑自己的婚事吧。"玛拉流着泪对布贝说，现在就是让她为他去死，她也心甘情愿。

　　和布贝分别以后，玛拉去城里的一家洗衣店工作。一天，玛拉和女友约好晚上去看电影《魂断蓝桥》。但到了晚上，女友却改变了主意，带着两个小伙子来，想叫玛拉一块儿去跳舞。玛拉拒绝了，执意要去看电影。两个小伙子中一个叫斯戴法诺的表示也愿意去看电影。于是，他便和玛拉结伴去看电影了。

　　电影散场时，玛拉显得很激动。显然，她被影片女主人公"玛拉"的不幸遭遇深深打动了。斯戴法诺对玛拉很有好感，在送玛拉回家时，他很坦率地谈了许多自己的事。而玛拉始终很严肃。

　　斯戴法诺在印刷厂工作。一天，他邀请玛拉到印刷厂去看看。在斯戴法诺的办公室里，他向玛拉介绍了很多书籍和杂志，还朗读了自己写的小说和诗歌。玛拉既羡慕又感激地看着斯戴法诺，显然这儿比洗衣店对玛拉更有吸引力。她愉快地和他交谈起来，还告诉了他自己的经历。

　　玛拉发现和斯戴法诺交往以后，自己的心情不再像以前那样孤苦压抑了；斯戴法诺像朋友似的和她交心，使玛拉产生了从未有过的自信。但她意识到斯戴法诺对自己的感情在一天天地加深，她怕自己的感情防线要被摧毁，断然决定不再和斯戴法诺来往了。然而，在以后独处的一段日子里，玛拉发现自己思念得更多的是斯戴法诺而不是布贝。她没有忘记布贝，但现在，唯有斯戴法诺能够安慰自己。玛拉陷入了矛盾的痛苦之中，她常常自问："我是否有权过自己的生活？！"

　　玛拉无法压抑自己的感情，她终于又去找斯戴法诺了。他见到她也非常激动，抚摩着玛拉的长发对她表白了自己心中的爱。

　　几天后，玛拉的父亲神情严肃地来找玛拉，他告诉玛拉，布贝在回国途中被警方抓获了，现在关在伏尔泰拉监狱。同志们都希望玛拉立即去探望布贝。

　　探监室里的气氛是令人难堪的。布贝面容憔悴，头发蓬乱，两眼直愣愣地盯着玛拉，玛拉嘴角上挂着一丝苦笑，神情紧张地等着布贝说些什么。沉默了片刻以后，布贝突然掩面痛哭起来："玛拉……这么长的时间我们没有见面……但我无时无刻不在想念你……但……决不应是在这种地方见到你……"玛拉看到他这样垂头丧气，感到一阵心酸，昔日的恋情顿时涌上心头。她像母亲那样安慰他。但是，当布贝问玛拉是否还像过去一样爱他时，玛拉没有回答。此时此刻，她既不愿欺骗自己的感情，又不愿伤布贝的心。

　　探监回来，玛拉就去找了斯戴法诺，她本想立即和斯戴法诺结婚以使事态无法挽回，但布贝那痛不欲生的神情却时时刻刻萦绕在她脑际，使她狠不下心来。现在，玛拉只希望审判结果对布贝有利，这样，他也许就不再需要自己了。

　　判决的那天终于到了，旁听席上人山人海。此刻，玛拉的心里异常的平静。前几天，一向崇敬布贝的父亲曾暗示过她，如果审判结果很糟的话，她应该重新考虑自己的婚事。不知为什么，父亲的暗示反而一下子坚定了玛拉对布贝的感情，玛拉决定，无论审判结果如何，自己总是布贝的未婚妻。她找到斯戴法诺，把自己的决定告诉了他。审判即将开始，玛拉走到布贝的面前，以热烈的目光注视着布贝，亲切地对他说："不管发生什么事，我永远不会抛弃你！"

　　审判的结果是：布贝被判了十四年有期徒刑。

　　火车继续在原野上奔驰，玛拉凝视窗外飞驰而过的景物，想象着马上见到布贝时的情景……玛拉的神情是坦然的，坚定的。当年，宣判布贝十四年徒刑时，玛拉觉得自己活不下去了，但结果呢，却是这样的简单。她想，再过七年，布贝出狱后，他们将生儿育女，他们的家庭一定是美满的吧！……
