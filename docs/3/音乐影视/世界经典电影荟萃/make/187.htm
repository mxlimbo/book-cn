上吊树


　　在日本北阿尔卑斯山的可良寿村外的斜坡上有一棵老橡树。这棵树高达25米，树木足有8米粗，郁郁葱葱，生机盎然。它高傲地屹立于村落之上，睥睨着发生在这里的一切。村民们把它看作守护神，看作德高望重的长者，有什么不幸，有什么期望，都愿到这里默默地向它叙述，甚至连生命旅程的步伐，也愿到此终止。传说很久以前，村里有户人家穷得过不下去，全家7口人都吊死在这棵树上。于是这棵树又得了一个称号--上吊树！

　　一年夏天，城里来了一对到此旅游的男女青年，发现这棵老橡树距离地面最近的树枝也是高不可攀，很难想象7口人是怎么吊上去的，而且其中还有小孩，"看来这是一棵无法上吊的上吊树啊！"

　　他们的谈论，恰恰被一位头戴草帽、一身农妇打扮的中年妇女听到。这位农妇名叫寺冈美佐于，43岁，她的眼睛里透露出愤怒与轻蔑。为什么？也许在今后发生的故事里才会使你们逐渐明白。

　　美佐子是20年前从外村嫁到可良寿村的寺冈家来的。她有两个孩子，大的是儿子叫弘一，小的是女儿叫纪美子。当弘一8岁、纪美子6岁那年的冬天，她那壮得如牛的丈夫突然患了感冒病倒了。那天夜里，风狂雪猛，她在全村人帮助下，拉着雪撬上路。很多人在前面扫雪探路，她和儿子女儿在后面奋力推着。当看到诊所的灯光时，丈夫却在雪撬上咽了气。这个意外打击，使她下决心，不管付出多大的代价，也要让两个孩子将来考上国立大学医学系，毕业后当个医生，回到可良寿村，为死去的丈夫"雪恨"。

　　在丈夫死去12年后的今天，她的愿望终于到了将要成功的边缘--18岁的纪美子在东京的一所学校读高中，马上将毕业参加全国高考；20岁的儿子弘一虽然已高考过两次，都名落孙山，但未灰心，现时仍在名古屋的一所补习学校里学习，准备参加第三次高考。而她自己却在可良寿独自经营着一爿杂货店，过着孤独清苦的生活，以微薄的收入支持着孩子们。

　　又一个冬天到了，美佐子为了使弘一能安心复习，便驾驶着自己的旧车，去看望儿子。没想到，这一去竟造成一件轰动一时的案子。

　　纪美子在东京见到报上一条消息说名古屋发生砍手凶杀案，通缉犯正是她哥哥，她怎么也不相信。几天后，她读到母亲的信，才知案子的来龙去脉。母亲在信中说："那天，我到名古屋，到弘一住处，他不在家。我很恼火，我忍受了那么多年的痛苦，就是要你们能考上大学，当个医生，而弘一不在家好好复习功课，我在他的信件和通讯录中找到了久留岛晴子的住址。在公寓的门口，我碰见一个叫深见的中年男人，他是名古屋塘鹅玩具公司计划部主任，久留岛晴子是他属下的临时职员，他是来取玩具样品的。我们一起走到晴子的房门口，门没锁，便推门而入，只见一位身穿白连衣裙的年轻女子倒在血泊里，整只右手不见了，她就是晴子。不一会，警察赶来。他打电话到弘一住处，弘一还未归家，让我及时报告我儿子的消息，我火了，因为他们怀疑弘一就是凶手。我知道，弘一决不会是杀手。离开现场后，我驾车回村，深见先生要求搭车，我就让他上了我的车。我最恨弘一性格懦弱，畏首畏尾。弘一的事你别担心，只管一门心思学习就是了，……"

　　收到母亲的信后不久，警探来向纪美子调查，她说了自己的想法，杀手可能是深见。但警探告诉她，解剖调查结果，在晴子死的时间里，深见在自己的公司，而她母亲在弘一的房里。

　　转眼到了新年，弘一还未露面，警长来村里找美佐子，"……当然我们还没断定您儿子就是凶犯，因为我们目前还搞不清他有无杀人动机。据说，你儿子学习成绩不理想，似乎对上大学彻底丧失了信心……"

　　美佐子说："不管你们怎样认为，弘一他决不是凶手！"

　　一次意外的雪崩，在可良寿村的斜坡上露出一具尸体--那就是弘一。

　　经解剖，弘一的死亡时间和晴子相近。在弘一的肚里发现了一把钥匙。

　　美佐子从警署领回儿子的尸体火化了，她没让纪美子回来参加弘一的葬礼，现在唯一支撑她信念的是希望纪美子考试能榜上有名。

　　纪美子考试结束后的一天，深见来访。他向纪美子说出了他发现的疑点，"当我们走进房间时，我曾感觉到地毯上有一个亮晶晶的小东西，而当我去打电话报告警察归来后，那东西不见了。后来当我搭乘你母亲的车回家时，我发现她外衣上少了一颗钮扣，同时，我觉得那些应放在行李箱里的工具却全堆在后座上，显然是腾出车后的行李箱载着别的物品……"

　　纪美子一听大吃一惊，突然悟出什么："先生是取玩具样品的，哥哥肚里的那把钥匙会不会是玩具手铐上用的？"

　　深见惊讶地说："你的推理能力真令人吃惊！不错，那把钥匙是玩具手铐上用的，做工相当精细，没有它，很难打开手铐……"

　　两天后，纪美子写了封长信给母亲。

　　"妈妈，考试终于结束，闲来无事，我把那件案子从头到尾理了一遍……当你赶到晴子寓所时，发现地上躺着两具尸体，那是哥哥和晴子，他俩是一起服毒自杀的，而且两人的手是用一副手铐相连着。你愤怒地拿来一把菜刀砍下晴子的手，将哥哥的尸体背了出去，放进车后的行李箱。当你准备回家时，发现外衣丢了一粒钮扣，不得不重返作案现场。于是和深见相遇。……你与深见分手后，午夜时分回到可良寿，把哥哥背到墓地旁的斜坡上，砸去哥哥手上的手铐，将哥哥匆匆埋在那里。……"

　　几天后，纪美子高考中榜，美佐于欣喜异常。这时，警方在斜坡上搜索到晴子那只戴手铐的手，他们正式传讯美佐子，明天到警署听审。警探走后，美佐子给女儿写了一封最后的回信。

　　"……妈妈佩服你，这案子的经过，确如你想象的那样。不过我砍断她的手，不是为了你，……我生弘一的气，更恨那个女人。我把弘一带回来，因为他是我的亲骨肉，谁也抢不走！埋了弘一回家的路上，我向老橡树祈祷，求它让弘一安息，求他保佑你考试成功……现在看来，老橡树答应了我为你做的祈祷，但在弘一的事上却不肯原谅我……"

　　"……既然你有所察觉，我便直言告诉你我和深见之间的事。在第二次相聚时，我为他付出了身子，想堵住他的口。我这样也许很可卑、很可笑？但为了瞒住这事，我什么代价都肯付。深见没有趁火打劫，他向我求婚。我没答应，因为我是永远属于可良寿的，我有自己的束缚。可我也是女人，自那次后，我的心和身体如火烧火燎，但我却不能再这样做！一旦事情传出，人言可畏……纪美子，妈妈决定走了……妈妈没什么给你，如果说有，那就是不受任何约束的自由，不论是丈夫、孩子，还是居住的土地。可良寿和我已不能束缚你了，你就自由自在地奔向你喜欢的地方，按照你自己的意愿，振翅飞翔吧！永别了，我的女儿。"

　　美佐子将信投寄后，带着一只小凳和一盘绳索，向村外的老橡树走去……

　　夏天，纪美子回到可良寿在寺冈家的墓地祭扫。她来到老橡树下，仰望着纵横的枝条，忍不住泪水顺着面颊滚下来。是啊，这老橡树将来也许会召唤她的，这就是命运的安排！她向老橡树凄声高呼："在那个风雪交加的日子里，我妈妈是怀着一种什么样的心情走向你的？我的妈妈，为了43载的冬天啊！"
