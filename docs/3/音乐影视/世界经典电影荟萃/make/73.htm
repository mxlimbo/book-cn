轰天炮


　　美国洛杉矶。某高层公寓的7楼阳台上，一个年轻妖艳的女郎，像喝醉似地斜着身子，依栏下望。恍惚中，她突然张开双臂，向迷幻中的奇异世界扑了下去。

　　黑人警长罗杰获讯，立即和青年警官马丁来到出事地点。一名警察告诉罗杰：昨晚这儿有个姑娘坠楼，据一个名叫狄茜的妓女说，出事时她在现场，能证明那女人是跳楼自杀。摔死的姑娘叫雅梅埃，22岁，也是个妓女，父亲叫迈克尔，母亲叫姬丽。

　　"是迈克尔的女儿？"罗杰吃惊地说，因为迈克尔是罗杰在越南战争中的战友，还救过罗杰的命，"快去查明，雅梅埃受什么人控制！"

　　验尸结果：雅梅埃跳楼前曾服用过大量的迷幻药，即使不坠楼，半小时内也会中毒身亡。一般吸毒者都严格控制毒品用量，而她却超出了好几倍。

　　罗杰和马丁去雅梅埃经常去的一家体育俱乐部侦查。在警车上，罗杰对马丁说："你精通柔道和武术，精于射击，这些都是能致人死命的技术，可不能胡来。因为你自己就已是一门威力无比的轰天炮！"马丁仍然沉默不语。

　　车子在一幢豪华的别墅前停住了。他们来到一个露天游泳池，只见池边的圆桌旁，有两个姑娘正在分装一包白色的药粉。正要上前查问，一个壮汉突然持枪从屋里冲了出来。罗杰放了一枪打中那人大腿，他刚要上前察看，那人对准罗杰举枪欲射，马丁眼疾手快，抢先开枪。壮汉中弹掉进水池。罗杰朝马丁怒吼，"见鬼！你不该杀他！该打他的腿！"

　　天黑了，罗杰请马丁上他家吃饭，他在路上向马丁道歉，"我只是不愿看见你多杀人！""我杀人是迫不得已的！"马丁在越南丛林里曾一人对付一百多个越兵。

　　在晚餐桌上，罗杰和妻子、儿女们欢乐地笑着逗趣。罗杰发现，他的大女儿深情地望着马丁。马丁因妻子在几天前突遭车祸身亡，心情一直非常悲痛，现在受到罗杰一家的热情款待，在这个欢乐的家庭气氛中，心情也变得愉快起来。

　　第二天，罗杰应约在餐厅见到迈克尔。迈克尔告诉罗杰："我的女儿被黑帮控制了，他们杀害了她，你要查出凶手，杀掉凶手！"

　　罗杰和马丁去找狄茜。在她家附近，几个孩子看见警车，就问，"你们来抓狄茜吗？"罗杰正要走近屋子，突然"轰"的一声巨响，屋子被炸坍了，燃起了熊熊大火。一个孩子告诉马丁，刚才有人进过狄茜家，那人与马丁一样，手臂上也纹一把短剑。马丁听了大吃一惊，这种纹身是越战特种部队的标记。他对罗杰说"我们遇到麻烦了，这些家伙是什么？得去问问迈克尔。"

　　"雅梅埃也许是为你死的，你参与了什么事？你得告诉我真相。"在罗杰的逼问下，迈克尔终于讲出实情；在越战中，迈克尔参加了"影子特别部队"，成员都是训练有素的杀手。"影子部队"摧毁了越南军方利用的走私毒品组织。回国后，这批人自己干起了毒品走私勾当。他们利用迈克尔公司的银行帐号，"这几年我洗手不干，他们就杀了我女儿，以此来威胁我……"这时一架直升飞机掠过头顶，扫下一梭子弹，迈克尔倒在血泊中。

　　黄昏，罗杰和马丁驾车来红灯区调查。突然，一辆汽车驰过，车里的杀手焦色华朝马丁连开两枪，马丁中弹倒地。罗杰以为马丁必死无疑，没想到他骂骂咧咧地爬了起来，原来他贴身穿着防弹衣。两人回到车上，突然对讲机传出话语："罗杰警长，有人绑架了你的女儿，赶快回去！"

　　罗杰赶回家中，妻子痛哭流涕地告诉他，莱茜被人劫走。电话铃响了，听筒里传出一个阴沉的声音："你女儿在我们手里，我们明天在指定的地点会面，只许你一个人来……"

　　在远离洛杉矶的一片旷野里，罗杰靠在车旁等候。远处，马丁手持带有望远瞄准器的步枪，伏在灌木树丛中。不一会，开来三辆车，还有直升飞机保驾。一伙人从车里出来排好阵势，然后，焦色华陪退伍将军梅阿里斯特钻出林肯牌轿车。焦色华大声喊，"罗杰，你过来，跟我们走！""你们得先放我女儿，否则我们同归于尽！"罗杰说着举起一颗手雷，并拉出保险针。莱茜被拖出车，被两个匪徒夹着逼近罗杰。远处传来两声枪响，两名匪徒应声倒地。莱茜钻进一辆空车，"快开车！"在父亲的提示下，莱茜驾车逃命。焦色华立刻指挥直升飞机去追赶莱茜。马丁瞄准了焦色华的眉心，刚要开枪，一支枪口抵住他的太阳穴。罗杰和马丁都被绑了起来。

　　匪巢的大厅内，罗杰被绑在靠椅上，被打得遍体鳞伤，"迈克尔对你说了什么？你告诉过其他人吗？"梅阿里斯特向罗杰逼问。"我不知道，他什么也没说！"

　　马丁被绑在储藏室里，悬吊在一只挂钩上，"上次没打死你，算你小子命大，快把知道的情况说出来，否则让你比死还难受！"这个手臂有短剑纹画的焦色华吼着。马丁破口大骂，焦色华叫打手拿来一只"电拳头"。这玩艺，马丁在越战中见过。"电拳头"向马丁的胸口触去，马丁发出撕心裂肺的惨叫。一连用了两次刑，马丁不肯招供。焦色华去报告头。屋里只剩下打手，马丁忽地用力荡到打手身边，双腿猛地夹住他的脖子，打手挣扎片刻，断了气。马丁奋力将手腕脱出吊钧，跳下来磨断了绳索。

　　罗杰虽被打得死去活来，仍旧骂不绝口。梅阿里斯特叫人把莱茜带进来，"你女儿是个漂亮姑娘，你再不说可害了她。"两个彪形大汉淫笑着向莱茜逼近。"爸爸救救我。"莱茜恐惧地叫起来。正当打手们扑向莱茜的一瞬间，门被砰地踢开，马丁双眼发红，杀气腾腾地冲进来。他猛地一拳，一个家伙的脖子被打断，又一拳，另一个家伙的颈椎骨被劈折。就在他对付几名打手时，焦色华和梅阿里斯特乘机夺路而逃。

　　马丁救出罗杰父女后，又去追赶逃犯。街上一片大乱。几名巡警飞奔过来，罗杰把女儿交给他们，也去堵截逃犯。

　　在街道中央，梅阿里斯特的车突然朝罗杰撞来，罗杰并不躲闪，对着驾车的保镖连开数枪。失去控制的汽车撞上一辆卡车，车翻起火。梅阿里斯特半截身子被压在车下，正在挣扎。罗杰咬牙切齿，正要对准这个作恶多端的罪犯扣动扳机，只听轰隆一声，汽车爆炸了。

　　逃脱的焦色华驾车来到罗杰家，见屋内无人，端枪向四处乱扫。突然，一辆警车驶来，焦色华对准车头打枪。他正想上前察看，马丁的枪抵住他的后脑勺。焦色华被罗杰缴了械，押到屋外，马丁愤怒地说："你又杀了我们两个弟兄，我要亲手宰了你！"

　　焦色华狞笑着向他扑来，这两个受过特种训练的格斗高手，展开了一场拼死搏斗。马丁使出柔道招术对付焦色华，焦色华挥舞木棍，二人又展开激烈的棍战。焦色华一棍打空，被马丁蹬脚踢飞。最后，马丁用中国武术中的腿功，双腿夹住焦色华的脖子，将他制服。趁警察弯腰之际，焦色华抽出警察腰间的手枪，向马丁举枪--马丁和罗杰同时拔出手枪，两声枪响，焦色华倒在血泊中。

　　阳光灿烂，马丁将一束鲜艳的玫瑰放在妻子的墓碑前，"亲爱的，再见，我永远怀念你！"他转过身子，罗杰正在朝他微笑，"老伙计，到我家去吃饭，别忘了今天是圣诞节……"

　　两个朋友迎着阳光，并肩走去。
