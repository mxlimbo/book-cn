夜幕降临的时候


　　在法国通往莱德隆的乡村小路上，两个从英国来的姑娘凯茜和简，各自骑着一辆自行车，在蜿蜒而漫长的大路上行进着。那个金发碧眼，身材丰满，名叫凯茜的姑娘，显得精疲力尽，脸上已露出焦躁不安的神色，那个留齐耳短发，穿着短裤的姑娘简用力蹬着车子，遥遥领先着。

　　很快她们就到了小镇上。由于天气炎热，凯茜和简在一家小店里要了冷饮和酒，一边休息，一边商量着下一步的计划。凯茜主张在这儿停留两天，简则认为应该在开始的时候多吃些苦，多走些路。

　　凯茜见简不采纳自己的意见，心里很不高兴，于是她拿出照相机漫无目的地乱拍起来。忽然，她被邻桌的一个戴墨镜的青年男子吸引住了。那男子也注意到了凯茜的目光，回过头来打量着她。凯茜被看得飘飘然不想走，简硬拉着她上路了。

　　半路上，这两个姑娘终于吵翻了。她们在一个小树林里休息时，凯茜一躺下来就不想起来，简再三劝说也无济于事。只得委屈地推着车子独自离开了小树林。

　　简走了以后，凯茜悠然自得地打开了半导体，一边听着音乐，一边从包里拿出洗过的衣服晾起来，然后又用防晒油涂着大腿、手臂和颈脖。然后重新躺了下去，一心等着那个法国男子的到来，很快就朦朦胧胧地睡着了。

　　突然，她被一阵"嗡嗡"的飞机声惊醒，睁开眼睛一看，发觉周围是那样的寂静，只有风吹着树叶发出沙沙的声音。她终于害怕起来，赶紧收拾起东西想马上离开这儿。这时，她才发现晾在树杈上的短裤不见了。紧接着，她又听见一声"哐当"的响声，转身一看，只见自己的自行车倒在地上，轮子的钢丝也被剪断了许多。就在她想去扶自行车的时候，树丛里悄悄地伸出了一只手，像抓小鸡似的把她一下拎了过去，"啊！"凯茜发出了一声撕人心肺的喊叫……

　　简赌气离开凯茜后，没多少时间就到了一个名叫拉萨尔的乡村小店。她想在这里等等凯茜，于是就把车子靠在门边，走进了小店。

　　这时女店主过来与她搭讪，告诫地说，一个女孩子不该孤单单地到这个地方来旅行，因为这条街的名声很不好，就是人们说的"坏路"！听女店主这么一说，简就更加焦急了，她两眼一刻也不离开公路上来往的车辆和行人，生怕凯茜从自己的眼皮底下溜过去。等了好久，仍然不见凯茜过来。

　　简决定再返回小树林去找凯茜。可是到了那儿，只见小树林里空无一人。除了在草地上发现了凯茜的照相机外，根本不见凯茜的影子。当她弯下腰去拾起照相机后抬起头来的时候，却惊讶地发现，戴墨镜的年轻男子突然站在了自己的面前。

　　简吓了一跳。"我在找我的朋友！"她向戴墨镜的男子解释说，生怕他听不懂，还结结巴巴地用法语重复了一遍。

　　只见戴墨镜的男子摘下了墨镜，冲着简一笑，竟用英语说起话来："我们还是说英语吧！"简将与凯茜分手的经过说了一遍。她知道了他的名字叫保尔。

　　保尔向她走过来，执意要带她去找凯茜。他的态度显得热情而殷勤，使简又犹豫不决起来，最后还是被保尔扶上了摩托车的后座。

　　简总觉得这个叫保尔的人行动诡秘，心里暗暗提防着他。他们到了一家咖啡馆，保尔离开了，她就和身边的人搭讪起来。可是他们都是和她说法语，简只能听懂一些诸如"谋杀"、"警察"一类的词句。最后，还是一位女教师用英语向她说，三年前这儿发生过一起凶杀案，被杀的也是一个姑娘。简听后更是忧心忡忡。她为自己害怕，更为凯茜的命运担忧。那个保尔却又迟迟不见回来。她决定赶快去警察所报案。可是到了警察所门前，她用拳头敲了半天，也不见有人来开门。她不敢久等，只好骑着自行车重新回到拉萨尔小店。当她第二天去警察所时，那个保尔却像个幽灵似地又出现在她面前了。

　　"那里面没人！"保尔在简的身后说。

　　简一见是保尔，紧张地后退了几步，叫了起来："你离我远点儿！你，你是个骗子！"简惊恐不安的样子使保尔暗暗发笑，但他很快又认真起来："是的，这儿是发生过一起凶杀案。死者是个女性，二十二岁，未婚，死因是被勒索窒息而死，下体划破并遭受过猛击。头发是金色的，像你的……朋友一样！"他满口行话，一口气说完了案情，然后轻轻吁了口气，又说，"我是巴黎警视厅的探长，研究过这桩案子！"

　　简听得目瞪口呆，仍不放松警惕地说："你以为我会相信你吗？……"

　　"我不想吓唬你！现在我在那个小树林里找到了一些东西。你一起去认认，看是不是你女友的物品。"不等简同意，他就抓起简的胳膊把她硬拉上了摩托车。

　　他们很快又到了小树林里。这一次进来，简更觉得里面阴森可怕，不敢往树林的深处走去。忽然她觉得有人在她肩上拍了一下，心顿时紧张得收缩起来。回过头，只见保尔目光灼灼地盯着自己。

　　"你害怕了？"保尔伸出手托住简的下巴。

　　简一扭头，用力甩掉保尔的手，一下窜到前面去。她不顾一切地冲出树林跑上了大路，这时保尔又出现了。她怕他再追上来，用力推倒了保尔停在路边的摩托车，然后拼命朝警察所方向奔去。

　　简一路飞奔，到了警察所已经气喘吁吁，半天喊不出声来。这时，从里屋走出来一个高个子警察，他见简的手里拿着一条短裤，还在大口大口地喘气，只好耐心等待着，直到简平静下来。

　　"你懂英语吗？"简急切地问。

　　警察操着生硬的英语回答说："会一点儿。"

　　于是简扬着手中的短裤，把凯茜失踪的事叙说了一遍，说着说着，她竟呜呜地哭了起来。

　　"她长得是什么模样？"警察问，并掏出个小本子，在上面做着记录，不时又提出一些问题。这时，简又强调补充说："还有一个，他说他是警视厅的探长，叫保尔・西蒙！你知道这个人吗？"

　　"保尔，保尔……，"警察自语着，然后又问简："他说他是从警视厅来的？"

　　"我知道他是在说谎！"简忽然惊觉起来，对警察说，"凯茜一定是遭到了不幸，一定是这家伙干的！他还骗我说，在树林里发现了一些东西，其实什么东西也没有。这是他的奸计！"她指了指那条短裤，"这短裤还是我找到的！"

　　听完简的叙述，警察收起了小本子，束上皮带，佩了手枪，然后斟了一杯酒递给简，叫她先在这里等着。说着朝简和蔼地笑了笑，然后走出门去。

　　警察所的后院是一排破旧的小屋，在小屋的檐下挂着的空罐头盒子，风一吹，就发出"丁了当当"的响声。不远的空地上停放着好几辆旧汽车，再远处，就是一片树林。原来，那小树林的尽头正好和这院子相接。简好奇地环顾着这破旧不堪的小院，感到一阵阵恐怖袭来。不一会儿，警察回来了，说什么也没有发现。那个叫保尔的家伙也没有追到。天越来越黑，简更加焦急。警察想了想又说："这样吧，你在这儿等着，不要走开。我再出去找一次，说不定会有收获的。"说完又骑着摩托车走了。

　　久久不见警察回来，简感到忐忑不安。忽然桌上的电话铃响了，她抓起电话筒后，电话里又没人回答，简感到十分蹊跷。这时，门外响起了摩托车的马达声，她以为是警察回来了，就一个箭步冲出了门外，可定睛一看，骑着摩托车过来的不是警察，却是保尔。她大吃一惊，迅速退回屋里，把门销死死插上。她躲在门后，屏住呼吸，静听着外面的动静。只听得摩托声戛然而止，接着就是越来越近的走过来的脚步声。

　　保尔敲了半天门，终于破门而入，在院子里追寻着简。转眼不见简的影子，他又追到小树林里，大声呼喊着："喂，你出来！你的朋友已经找到了，我带你去看。"

　　简畏缩着躲在一辆旧汽车里，清楚地听见保尔的喊话，但是她看见保尔手里挥动着一件凯茜用的毛巾，断定凯茜已遭到他的暗害。尽管保尔喊破了嗓子，简还是躲着一动不动。突然，她不知碰到了什么，"通"地一声，只见一个黑黝黝的东西从她身上滚落下来。她推开汽车的隔窗，通过透进来的一束光线，吃惊地发现，滚下的正是凯茜的尸体。凯茜身上的衣服被撕得破烂不堪，两只恐惧的眼睛睁得大大的。简顿时两腿发软，不敢再呆在车里，硬撑着爬到车外，然后没命似的朝小树林冲去。没想到保尔也在那里，并且一眼看见了她。简急忙隐在一棵树的后面，悄悄地从地上捡起一块石头，准备和保尔拼命。

　　保尔追到大树旁边，简又不见了。正待他转身走开，简一下子冲了出来，将手中的石头打在他的脸上，鲜血立刻顺着他的额头流了下来。简趁机扑了上去，发疯似的再用石块猛打保尔的头部，终于把他打倒在地上不动了。

　　这时，站在小树林里的警察，把这一切都看在眼里，但是他似乎并不想去帮助简，却像看白戏似的站在一棵树荫下冷眼旁观着。当看到简已经把保尔打昏在地上，他才缓缓地朝简走过来。惊恐未定的简一看见警察，就像见到救命人似的，一下子扑到了他的怀里。

　　"她死了，你知道吗？凯茜死了！"简抽泣着说。警察却趁机将她紧紧搂住。

　　"他也死了！……我打死了他！"简流着泪，抬起头望着警察。

　　警察的手抚摩着简的背，像哄孩子似地说："好了，……别怕了！"他一边说着安慰的话，一边把手伸到简的腰里，最后手又向简的短裤后袋插过去……

　　简突然止住哭泣，警觉地要从警察的怀里挣脱出来。

　　"和我在一起，你不用害怕。"警察把简搂得更紧了，他另一只手伸过去解简的腰带，半天解不开，又去扯她的上衣。

　　简这才明白警察是一条恶狼，她奋力抵抗着，用手乱抓他的脸。但她终于敌不过身材高大的警察，被他按在了地上，身上的衣服也已经撕裂开来。正当简束手就擒的时刻，只见警察却被一根木棍狠狠击了一下，立即倒在了地上。

　　原来满脸血污的保尔醒了，是他赶过来搭救了简。简整理着被警察撕乱的衣服，忽然一下全明白了……
