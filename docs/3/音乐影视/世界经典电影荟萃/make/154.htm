湖畔奏鸣曲


　　风光绮丽的湖畔，住着女教师劳拉一家。劳拉的丈夫里奇因酗酒打死人被关进监狱，她带着一对年幼的儿女和年迈的婆婆、尚未出嫁的小姑一起过日子。婆婆思想守旧。小姑维亚又不问家事，整天不是在外游荡便是跟母亲呕气。维亚和里奇不仅是同母异父所生，而且还有着杀父之仇：当年，维亚的母亲被这所农舍的东家奸淫，生下里奇后被撵了出去。战后，东家出逃当了土匪，维亚的母亲带着她的丈夫回村住进这所农舍；后来她丈夫当了这个村的农庄主席，不久却被当了土匪的东家暗杀了。维亚就是这农庄主席留下的女儿。维亚的母亲含辛茹苦地将维亚兄妹抚养成人。家里没有了男人，简直成了个烂摊子。劳拉得承担全部家务，还要定时去探望狱中的丈夫。狱中的丈夫里奇也把劳拉看作是自己生活的支柱和希望，一次探狱时，她丈夫对她说；"如果没有你，我真的不想活了。"

　　这年夏天，村里来了一个到此度假的医生鲁道夫，住在劳拉邻居家。他因借小船认识了劳拉，对这个容貌清秀而带几分忧郁的女人产生了好感，鲁道夫的出现。使劳拉的婆婆想起了自己疼爱的儿子，便数落起劳拉，怪她当初关心丈夫不够，不肯陪他去酒店坐坐，劝他少喝点酒，否则不会酿成这个悲剧。劳拉没有吭声，只是独自来到湖边小木桥上解闷。她想起丈夫每次来信，总是充满哀怨。她又想起与丈夫一起生活的情景，那时，她常常独守空房，丈夫到深夜才醉醺醺地回来……

　　鲁道夫是个鳏夫，他与妻子离了婚，十四岁的儿子跟了母亲。现在，他倒是自由了，可以在这充满诗情画意的地方尽情玩乐，度一个愉快的假期，每逢清晨或黄昏，他驾起小舟在湖面上随波逐流，淡淡的雾霭像轻纱般抚摩着他。在碧空万里、群雁高飞的日子里，他跃入碧绿、清凉的湖水里畅游，困倦了便倒在浓荫下打盹。雨天，他就到劳拉门前的湖里去摸鱼。在那里，他总希望能见到劳拉。

　　一天黄昏，劳拉的女儿扎伊卡患急病，劳拉一时找不见卫生员，就来请鲁道夫帮忙诊治。鲁道夫见孩子的病情急需开刀，便用自己的小车送劳拉母女去附近的医院。然而不巧，那里的外科医生又出诊去了。鲁道夫努力说服了值班医生，成功地为扎伊卡做了手术，终于使扎伊卡脱离了危险。劳拉心里由衷感激。

　　又有一次，劳拉去学校办完事出来，遇上鲁道夫，便请鲁道夫坐自己的小船一同回村，鲁道夫一边划船一边注视劳拉，眼光充满柔情。劳拉有所意会，心里也激起了复杂的涟漪。

　　时间一天天过去，鲁道夫对劳拉的感情日趋强烈。他连做梦也看见劳拉的倩影。他忽然感到自己已经离不开她，于是想方设法跟劳拉亲近，对她表示关切。有一次劳拉去城里办事，回家时赶上天下雨，她从公共汽车下来，见鲁道夫已开车等在那儿。劳拉因此而没淋着雨。

　　一个星期天，劳拉一大早就去厨房忙碌。太阳升得老高，维亚才懒洋洋地起床，起床后竟又拖着她的侄儿、劳拉的儿子马里斯一起去湖里游泳。他们到了那里，却见鲁道夫早已坐着小船在湖上悠闲地荡漾了。鲁道夫见到他们就热情邀请他们上船。劳拉来到湖边，远远地看见鲁道夫认真地教自己儿子划船的情景。不由得感慨万千。她由此联想到自己丈夫过去对待孩子的态度是那么粗暴。眼前的这个男子燃起了她心底的火花，激起了她无比的遐想……然而，当她想到孩子的父亲目前正在狱中，愧悔地呼喊着她、等待着她时，脑子里轰然一声，不由得克制地转身走了。

　　次日傍晚，劳拉在湖边割草，鲁道夫过去帮忙，顺便问她，为什么她总避开他？劳拉只是露出了一丝苦涩的微笑，算是回答，随后背起草捆就走。鲁道夫怅然若失地望着她沐浴在一片夕照的金辉里的背影。

　　劳拉的回避，仍没有使鲁道夫死心。她去城里开两天会，他总开车在会场外等候。终于，劳拉凝望着这位耿直的男子，接受了他的好意。在回家的路上，鲁道夫想起现在恰是午餐时间，后悔自己太疏忽，没有请劳拉一起上馆子吃完饭再走。劳拉毫不介意，顺手从包里拿出开会休息时买的面包，请鲁道夫一起充饥。吃完面包，当两人下车去溪边喝完水往回走时，鲁道夫忍不住亲吻了她。

　　车刚进村口，劳拉就望见女儿神情焦急地等在那里。女儿对她说："不知为什么，我怕你不回来，你会有一天离开我们。"

　　女儿的话，叫人听了心酸，劳拉心里震动很大。她深深感到，不仅丈夫需要自己，连儿女们也离不开她。虽然她丈夫比不上鲁道夫那么值得她爱，但她觉得自己不能卸下这副沉重的担子，离开这个不幸的家。

　　鲁道夫并不知道劳拉的心思，第二天干脆带着她的儿女，一起进城接她去了。

　　对劳拉与鲁道夫的感情注意已久的维亚，这时再也忍不住了，她责怪嫂子一面那么认真地读着丈夫的来信，装作贤妻良母似的，一面却又不甘寂寞去跟别人鬼混。劳拉沉着气，委婉地反问她："如果这真是爱情呢？"

　　维亚满怀嫉妒地转身出去，到湖里去游泳。不一会，她就在水中失去了平衡。正在划船的鲁道夫听到呼救声，急忙过去把她救起，并把自己的外套递给她防寒。可维亚没有领情，硬要鲁道夫立刻上岸。临走时，她故意用话激鲁道夫，"你老爱在我家门口的湖上转悠，鱼儿上钩了没有？"

　　维亚回到家里，真诚地劝劳拉寻找自己的幸福去。可劳拉坚定地表示，她要等丈夫回来。维亚惊异地看着嫂子。在她看来，这所旧房子及屋里的一切旧东西，有什么可留恋的呢？而她维亚对这一切早已厌倦了。劳拉没有被她说动，照旧埋头操劳家务。

　　天淅淅沥沥地下起雨来，劳拉冒雨走到井边打水，这时鲁道夫忽然悄无声息地又一次出现在她身边，并且热烈地亲吻了她。

　　"我们不会幸福的，因为在我们之间，不光是我们两个人。求求你，走吧。"劳拉说。雨水直泻而下，冲刷着她那动人的脸庞。

　　鲁道夫凝视着她，随后默默地转身走了。他完全能够理解劳拉的心。他虽然没有得到她，但他愿意永远爱她，敬重她。维亚见劳拉拒绝了鲁道夫的爱情，非常为她着急。她劝嫂子无用，又去说服鲁道夫，求他把劳拉带走。她激动地对鲁道夫说，"你应当争取，为了劳拉，里奇毁了她一生。"

　　鲁道夫赞同她的话，但他更理解劳拉的心思："如果要她毁了里奇，她将不能原谅自己。"

　　维亚回到厨房，见劳拉在洗一大堆碗碟，她像什么事也没有发生过。维亚再也按捺不住，一把夺过劳拉手中的盘子，爆发地骂嫂子是块木头、石头，是一块冰！

　　"要是你的话，你会怎么做呢？"

　　"该把阻碍我生活的东西统统砸烂！"维亚说着，把一大堆碗碟，乒乒乓乓砸了一地。

　　马里斯从外面进来，踩着碎片走到劳拉身边。他见东西都砸破了，天真地问，"我们是不是破产了！什么也没有了。"

　　"以后会有的……"劳拉搂着儿子，脸上闪耀着坚毅的光彩……
