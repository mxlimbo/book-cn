烈火战车


　　英国剑桥大学曾以培养了一大批杰出人才而闻名于世，现在迎来了战后第一批新生。哈罗德也在这批新生中。他朝气蓬勃，才华横溢，特别是在田径运动方面，他具有一个短跑运动员所有的素质。但他偏偏是个犹太人，社会不让他有显示才能的机会。年轻的哈罗德发誓无论如何也要在这片赖以生存的土地上为犹太人争得一席之地，他要让社会承认他。所以他压抑着满腔愤懑，抱着显示自己、战胜别人和冲决一切社会偏见的信念进了剑桥。在学校里，他既能尊重所有的人但又谁也不放在眼里。他的原则是谁瞧不起自己就制服谁。他参加学校里的一切活动，在别人面前，他眼里总闪烁着刚毅和倨傲的光芒，仿佛在说："我比你们中的任何人都行。"尽管这样，他还是捕捉到了那种令他"怒火中烧"的眼光和他们言语中的弦外之音。他要和他们较量，以自己特有的武器--跑步，战胜一切蔑视他的人。

　　几星期后，哈罗德宣布他要进行一次赛跑，这是一场以大钟为"对手"的传统赛跑，七百年来还没有人在比赛中获胜过。挑战者必须在正午十二点，趁学院大钟敲二十四下的当儿，沿学院大草坪跑一圈，全程三百十二步。换句话说，就是要在不满一分钟的时间里跑完大约四百码的路程。

　　这个大胆的、无所顾忌的挑战把学生们都吸引到操场上来了，连院长也皱着眉头看着操场上的动静。哈罗德做好了一切准备工作，站在大钟下显得胸有成竹。离十二点还差一分钟了，发令员--哈罗德的好友奥布里严肃地大声宣布："鉴于无人主动要求参战，哈罗德先生将单独冲刺。"这时一个年轻的贵族从人群外挤了进来要求参赛。他叫林赛，以风度潇洒、酷爱运动而闻名全校。他跟哈罗德友好地握手以后站在了起跑线上。大钟的分针慢慢指向"十二"了，全场鸦雀无声。紧接着大钟"吃"地敲响了第一下，两个赛跑者像脱弦的箭飞离起跑线。人群中吼声四起，哈罗德领先一码，可在第二个拐角处，他踩着了一块石子，身子滑闪了一下险些摔倒，林赛乘势追了上去。大钟不停地敲着……两人跑上了最后一段直线跑道了，林赛使出全身的气力想保持领先地位，可哈罗德这时不知从哪儿涌出一股最后冲刺的猛劲，他把满腔激情贯注在最终的目标上--取胜。他健美的身躯奋力冲向前，就像一团火--一团长久郁积在心中的愤懑之火喷涌而出。林赛被他抛在后面了。操场上一片沸腾，人们为他们大声地数数："二十、二十一……"在哈罗德最后的冲刺中，人们终于看到了这位犹太人身上那股令人惊羡的获胜意志。"二十二、二十三……"哈罗德一鼓作气，飞也似地越过端线，投进了奥布里张开的双臂中。落后几码的林赛紧接着也越过了端线。人们发出了狂烈的欢呼声。院长关上了窗户，他很为林赛功亏一箦而惋惜。

　　这场赛跑以后，哈罗德代表英格兰多次参加国际比赛，并取得了很好的成绩。这使哈罗德名声大振，所到之处总有学生拦着他签名。但他并不满足，现在他全身心地盯住了下一届奥运会，他要在奥运会上证明自己是世界上跑得最快的人。但令他不安的是"苏格兰飞人"埃里克。哈罗德早就听说这个传教士的后代、虔诚的基督徒埃里克有着一股超然的神力，特别是在看来毫无取胜的情况下，他会以他特有的跑步姿态，从上帝那儿汲取力量，疯狂地力挫群敌而取胜。哈罗德曾亲自赶到爱丁堡去观看埃里克和法国健儿的比赛，他的神威使哈罗德害怕。在奥运会前哈罗德有了对手，他很希望他俩能在运动场上决一雌雄。他很自信地想，"不致于输给埃里克吧。"因为他还从未败在谁的手里过。

　　这样的机会终于来了。被英伦三岛誉为"百年大赛"的比赛安排停当了。全英最优秀的选手云集在伦敦体育场，可人们瞩目的却是哈罗德和埃里克。报上大肆宣传："苏格兰飞人戎装南下，准备同剑桥英秀交手"，"哈罗德声称已准备好了迎故"。赛前的紧张时刻终于过去了，哈罗德和埃里克在成千上万群情激昂的观众的呼喊声中蹲在起跑线上，等待发令枪响。埃里克显得那么轻松自如，他挨个祝他的对手取得好成绩。哈罗德直勾勾地盯着埃里克，他为了迎战埃里克不惜劳其筋骨，苦其心志。运动员今天的胜负只能取决于个人的气质和临场的竞技状态了。发令员举起了信号枪，场上每个人都凝神屏息。枪声响了，哈罗德像离弦之箭，飞也似地向前冲，埃里克同样如此。两人并驾齐驱，迎风疾驰，接着埃里克越过了哈罗德。为了超过埃里克，哈罗德鼓起了积蓄在体内的力量猛然加速，这一招是任谁也休想与之匹敌的。哈罗德冲到了埃里克面前。然而埃里克也使出了他的招数，只见他把头往后一仰，手臂旋风似地划动起来，又追上了哈罗德。看台上的掌声如狂风暴雨，最紧张激动的要数埃里克的姐姐珍妮和哈罗德的女友茜比尔了。珍妮不希望弟弟赢，她要埃里克全身心地侍奉上帝；茜比尔则急切地希望哈罗德赢，她怕从未输过的哈罗德经受不起失败的打击。哈罗德死命地狂奔，出于一种不可名状的心理，他朝埃里克瞟了一眼，而就在这一刹那，他的冲力中断了；埃里克则身、心、神紧密地结成一体，着魔似地冲断了终点的红线。哈罗德仅一步之差败给了埃里克。

　　他呆了，在空荡荡的运动场上坐了很久，满脑子全是"输了，输了……。"茜比尔只能用言语来宽慰他。这时萨姆出现了，他是全欧洲最杰出的田径教练，他看出了哈罗德身上的潜力和不足，他愿意帮助他。

　　哈罗德在萨姆教练严格、科学的训练下，跑速在提高；埃里克也在群山原野中苦练。他们的目标全想在奥运会上夺标。哈罗德还有个心愿，就是想挫败埃里克。

　　盼望已久的奥运会终于到来了。哈罗德、埃里克、林赛等人作为英国优秀运动员参赛。哈罗德和埃里克是百米跑；林赛是跨栏和四百米跑。林赛以少有的认真夺得了跨栏的银牌。可谁知，百米跑恰巧安排在星期天，埃里克这个虔诚的信徒怎么也不肯在安息日赛跑。英国队失去了主力队员，哈罗德失去了报"一箭之仇"的机会，为了顾全大局，林赛把自己的四百米跑让给了埃里克。

　　星期天，埃里克到巴黎市中心的教堂布道去了，而哈罗德胸有成竹地走进了运动场。赛前他作了充分的准备，现在他全神贯注地进入了竞技状态。他是为了取得社会的认可才置身于这竞技场上的；击败偏见，才能确立人的价值。他正告自己：今天获胜了，你就可以凌驾于阶级和宗教的种种无聊的标签和束缚之上了。发令枪一响，哈罗德像箭矢一样飞离起跑线，他把一切都置之脑后，只觉得身体里那团火在熊熊地燃烧。他凝视着前方的终点线，飞速朝它压去，其他运动员都被他抛在后面。全场观众为他那海一样奔腾、火一样喷涌的力量而狂呼。当他冲断终点线时，"哈罗德"的呼声响彻整个体育场。哈罗德成了世界上跑得最快的人了。奥运会上第一次响起了英国国歌《上帝拯救吾王》。埃里克赶来祝贺，两人握着手毋需任何言词已息息相通了。埃里克在四百米赛跑中，也以无与伦比的速度打破了世界纪录。他特有的跑步风姿使全场叹为观止，运动场上再次响起英国国歌。

　　哈罗德和埃里克这两位体坛巨星勇敢而庄严地经受了考验。他们都如愿以偿了--哈罗德以自己杰出的体育才能得到了社会的认可；埃里克以他的金牌为上帝争得了荣耀。他们像民族英雄般地回到了英国，成了家喻户晓的传奇式人物。
