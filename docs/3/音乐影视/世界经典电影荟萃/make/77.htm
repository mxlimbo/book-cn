虎口胭脂


　　在纽约这个大都市里，只要有钱，什么都能买到，只要为钱，什么都能卖出，甚至毒品、枪械、肉体……

　　她就是一个从事出卖肉体的女人，这一行里的同仁管她叫"公主"。公主有个四岁的女儿。女儿是那样的天真可爱，每当傍晚她将女儿托人带给外婆照管，女儿总依依不舍，她像许多母亲一样哄孩子："妈妈要去上班……"

　　正当她准备出门，女友葛晶打电话来，葛晶也是个妓女，因为没有上交150元保护费，那个凶狠无比的保护人雷洛随时找上她，她的处境可想而知。公主劝她加倍小心，别轻易走动。

　　然而，雷洛还是找上门来。在他花言巧语下，葛晶开了门。雷洛闯了进来，他一手揪住葛晶的头发，一手卡住她的脖子，"你这个忘恩负义的婊子！你忘了，是我在火车站发现了奄奄一息的你，教你打扮，教你挣钱，如今你想甩了我？没那么容易！"他把葛晶摔到床上，把她的手脚绑在床上，然后把衣架拗成一条铁鞭，朝葛晶的下身狠狠地抽打……

　　扫"黄"小组组长霍斯在这地方和罪犯打交道已整整十年了。他刚来时总想通过自己的努力，使生存的环境变得干净、清新，可是年月流逝，这批小小的罪犯并不见少，而真正的大罪犯又难得"光临"。他变得麻木了。

　　今夜，轮他当班，报话器传来紧急通知："霍斯注意，马上到医院去，那儿有案情……"

　　他带领小组成员赶到医院，见到正在抢救的葛晶，她遍体鳞伤，奄奄一息，"谁干的？"

　　葛晶不敢开口。霍斯从她那惊恐的脸上想起一个人，便追问："是不是雷洛干的？""他……"葛晶哆嗦了一下嘴唇，便断了气。

　　面对着这个被折磨得惨不忍睹的形象，霍斯发誓："捉不到凶犯，决不罢休！"

　　不一会儿，公主被带到医院，"警察大人，有什么交易？"

　　"去把雷洛诱出来。"

　　"不！我不敢！这家伙杀人不眨眼。"公主赶紧回绝。

　　"你来看！这是谁？"霍斯掀起蒙在尸体上的白布，对公主吼道。

　　"葛晶！"公主一见好朋友的尸体，发疯似地扑上去，嚎啕大哭。

　　霍斯将悲痛万分的公主拖开，安抚她的背脊，给她安慰。现在毋须多说，他知道这种女人的性格，愤怒会使她不顾一切地去干。

　　公主打扮得花枝招展，在一家小酒店里招摇而过。雷洛正在酒店里，他一下子被公主的风度所吸引，立即上前搭讪："你好像独自干活，这可需要保护了。"

　　"是吗？你不怕我是警察？"

　　"你是警察，那我就是处女了。咱们是不是换个安静的地方谈谈？先让我看看你的本钱，值不值得保护？"雷洛很在行。

　　公主狡诈地一笑，鱼儿上钩了。

　　他俩离去，这一切都在警察的监控之中。

　　不一会，雷洛将公主带到自己的寓所。一进房门，他就急不可待地拉着公主上床……

　　随着令下，早已潜伏在四周的警察破门而入。雷洛大吃一惊，但很快故作镇静："你们想干什么？有搜查证吗？""还用得着这个？"霍斯一挥手，两个警察扑上去扭住雷洛的双臂。

　　公主从拎包里取出微型收录机，举到雷洛面前说，"证据都已录在里面了！你害死葛晶，我要为她报仇！"

　　"臭婊子。你设了圈套来害我！"他猛地挣脱，想抓住公主当"人质"，可霍斯他们眼疾手快围扑上去，雷洛束手就擒。

　　雷洛被警车带走后，霍斯带公主来到一家餐厅，要了两客三明治，算是对她的慰劳。

　　见他生活如此简朴，公主很惊讶，"你成家了吗？"

　　"离婚几次了！"霍斯答道，"谁愿意嫁给脑袋拽在裤腰上的警察？"

　　公主心头一沉，没想到，这些专跟她这号人作对的警察，竟然也有不如意的命运。同是天涯沦落人，相逢何必曾相识！他们之间的心沟通了。

　　"你还要去接客？"见公主起身告辞，霍斯关切地说。公主脸一沉："我有女儿要吃饭！"

　　霍斯闻言，心头泛起一阵酸楚，立即从兜里掏出钱来，递给公主。公主扭头便走，"你这几个钱，还不够一夜开销的！"

　　当霍斯回到警察局的时候，一件意想不到的事发生了。下属报告：雷洛在押送途中，击昏警察，造成警车翻身，趁机逃之夭夭。霍斯很快意识到公主将面临葛晶同样的遭遇。他立即召集小组成员，全体出动，追捕逃犯。

　　公主回到她经常营生的黄金大酒家。这里有她许多姐妹，"可可"、"蓝天使"、"花衣女郎"，她们相聚喝酒，借嘲骂男人解脱内心的辛酸。

　　协助警方抓住雷洛后，公主接连接了两次客，这时又有一个找上门来。在客房里，她机械地完成"服务"之后，那主顾竟把她打倒在地，把她钱包里的一夜血汗钱囊括一空，"你浪费了我的时间，这些算作赔偿费。"说罢，扬长而去。

　　公主欲哭无泪，只能忍受，强打精神，趁着夜色还未退尽，再拼命干活。还算幸运，她接到一笔好生意，上门为一个有钱的阔佬服务。

　　她前脚刚走，雷洛后脚赶到。他打听了公主的下落，又匆匆追了出去。他在街角抢了一辆白色货车，把路边寻客的可可诱入车里，从她口里逼问公主的去向。

　　霍斯找到被抛弃路边被毒打得奄奄一息的可可，知道雷洛正直扑公主而去，赶紧指挥人马追赶。

　　公主离开阔佬的寓所后，在一家店铺里给女儿买了一只绒布小白兔，又到电话亭与妈妈通电话，问女儿的情况。谁料，雷洛赶来发现了她，他一踩油门向电话亭撞去。蓝天使和花衣女郎见状，连忙穿过马路，去招呼公主。雷洛的车一下撞倒花衣女郎。公主闻声回头，大吃一惊，跃出电话亭，夺路而逃。

　　雷洛将公主押上六楼他的另一个据点。里面有床有各种刑具。他将公主推倒在床上，以他习惯的方式去报复。当他剥去公主的丝袜时，公主双脚一蹬，正中雷洛腹部，公主跳起逃脱。但她还是逃脱不了魔爪。

　　雷洛重新逮住她，将她拖到床上，亮出匕首挑开她胸前的钮扣，要她"品尝"即将到来的死亡是什么味道。

　　此刻，霍斯他们赶到楼下。他发觉电梯的门已被雷洛锁上，就兵分两路，从楼梯和消防备用梯包抄上去。

　　在六楼，公主的裙子已被挑破。雷洛举起用衣架拗成的鞭子，猛力向她抽来，她发出一声惨叫。

　　霍斯用枪击穿门锁，冲进来。

　　雷洛发现不妙，纵身跳出窗口，沿着梯子逃到底层，开枪打伤拦截的警察，跳上白色货车，仓皇逃走。霍斯也追到底层，跃上警车追赶上去。

　　雷洛的车被逼到一堵大墙前，无路可走，急忙下车，端着枪，见霍斯逼近他，就举枪扣动扳机。枪声响，倒下的是这条凶恶的豺狼。

　　受伤的公主被担架抬出大楼。霍斯向她走来，手里拿着那只她为女儿买的玩具小白兔。

　　公主现出一丝苦笑，对霍斯说，"你没有改变街头的罪恶，又为什么要当警察？！"

　　霍斯望着公主被抬上救护车，陷入了沉思……
