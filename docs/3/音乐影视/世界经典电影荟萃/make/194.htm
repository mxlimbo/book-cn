流浪者


　　印度上流社会中很有名望的大法官拉贡纳特信奉的是这样一种哲学："好人的儿子一定是好人；贼的儿子一定是贼。"这种以血缘关系来判断一个人德行的谬论害了不少好人。一天法院正在审理一件谋杀案件，原告是法官拉贡纳特，被告是流浪者拉兹，罪行是拉兹闯进拉贡纳特的住宅，企图刺死拉贡纳特。可是拉贡纳特万万没有想到，要想杀死他的拉兹正是他亲生的儿子。

　　24年前拉贡纳特从他的哲学出发，曾错误地给强盗的儿子扎卡判了罪。扎卡不服，怀恨在心，越狱以后，被迫成了强盗，到处烧杀抢掠。他为了报复，在一天晚上将法官拉贡纳特的妻子里列突然抢去。当他知道里列已怀孕，就动了坏脑筋，故意把里列放了，并四处放风，引起拉贡纳特对妻子的猜疑。果然，里列回到家后，拉贡纳特认为妻子已经受辱，对他不忠了。他不顾妻子如何解释、反驳，硬是将她赶出家门。从此里列过着流浪的生活，在大街上生下了儿子拉兹。母子俩相依为命，住进孟买的贫民窟，生活十分艰苦，但是里列是个坚强的女人，她辛勤劳动，省吃俭用，让拉兹上学，希望拉兹有朝一日也能成为一个受人尊敬的法官。小拉兹读书很用功，在学校里认识了女同学丽达，两人志同道合，很快成了好朋友，经常一起玩。但是好景不长，家庭艰难的生活迫使小拉兹上街去擦皮鞋，帮助母亲维持家庭。这件事触犯了学校的制度，他们毫不留情地将小拉兹开除了。小拉兹开始流落街头。有一天正好碰到窃贼扎卡，小拉兹因为母亲在家里病得厉害，非常急切地要求扎卡发发慈悲，帮助他找找工作，扎卡非常乐意，认为机会来了，他利用小拉兹爱母亲的感情，迫使他去偷东西，以便让母亲不挨饿。小拉兹开始不愿意，但经不起扎卡的威胁引诱，还是去干了，结果被抓进监狱。出狱以后，扎卡又盯住他不放，要他继续跟自己一起行窃，拉兹又一次犯罪坐牢。一个天真无邪的孩子，此刻完全变成了另外一种模样，他一共坐了12年牢，等他出狱以后已成了一个青年。扎卡的罪恶目的经过长期努力终于达到了。

　　生活又发生了戏剧性的变化，当拉兹在街头流浪时，有一次意外地碰到了童年时代的丽达，两人一见钟情。丽达的父亲生病死去，临死前委托法官拉贡纳特做丽达的保护人，拉贡纳特一口答应。从此丽达就和拉贡纳特住在一起，如同父女。拉贡纳特非常喜欢丽达，能满足她的一切要求，但对她又非常严格的控制，怕她天真无知，迷入歧途，于是一场悲剧开始酿成了。

　　拉兹和丽达完全掉进了情网，他们白天去游玩，晚上各人做着美梦。他们幻想幸福、美满的一天就会到来。但是法官拉贡纳特对拉兹非常反感，他认为像拉兹这样的人无论如何是配不上丽达的。他出面阻止丽达和拉兹接近，并要她考虑后果。丽达希望拉贡纳特能了解拉兹，允许他们继续来往。为此拉贡纳特大为生气。

　　拉兹自从再次遇到丽达以后，良心促使他悔悟。他后悔当初不该错一步而错百步。他害怕这一切要是让丽达知道，后果不堪设想。他陷入极度痛苦之中。有一天他做梦，在街上又碰到扎卡，扎卡露出狰狞的面孔，威胁他再去偷东西。拉兹不愿意，扎卡就拔出刀子一步步逼近他和丽达。正当万分危急的时刻，拉兹从恶梦中醒来。他把一切都原原本本地告诉母亲，向母亲忏悔，抱住母亲痛哭，表示今后再也不干这类事了。爱情给拉兹带来了新生，他开始寻找工作，走自食其力的道路。然而工厂的老板、经理一旦了解他过去偷过东西，坐过牢，就再不愿意让他工作了。这一切像一盆盆冷水浇在他火热的心上，他痛苦万分来找丽达，可是看门人根据拉贡纳特的指令，不让他进入住宅。拉兹无路可走，只好在街头重新开始流浪的生活。于是那悲凉凄修的"到处流浪"的歌声又响起来了。

　　拉兹心里一直想念着丽达。他想起了过去同学时的美好岁月，想起了12年前自己曾经答应在她21岁生日的时候送一件贵重的礼物给她，他默默地等待着那一天的到来。为了实现自己的诺言，为了表达自己多年来深深埋藏在心底的爱情，他不得不向扎卡借钱。可是扎卡不但不肯借，而且硬逼他再去偷。拉兹在珍宝店门口徘徊，最后终于一咬牙关又去偷了。正巧这天拉贡纳特也来给丽达买生日礼物，拉兹并不认识他，趁他买了项链走出店门时，上前一撞，把这项链偷了过来。一转眼，他又怀着喜悦而惴惴不安的心情把它送给了丽达。丽达非常高兴，说这是世界上最珍贵的礼物，她要永远带在胸前。拉兹没有听完她的话，就匆匆离开了。

　　这时拉贡纳特也来了，他告诉丽达自己给她买了一条金项链。当他打开盒子时，里面什么也没有，而当他抬头看到丽达脖子上的正是自己买的礼物时，一切都明白了。

　　丽达非常难过，她想弄清真相，立刻赶到拉兹家。她看到拉兹的母亲，孤苦伶仃地一个人坐着。她忍不住问起拉兹的情况。母亲只好一五一十地把拉兹从小受到的折磨，以及受坏人引诱走上犯罪道路的情况都如实告诉了丽达。丽达十分感动，她认为拉兹是值得同情的。她并不因此而冷淡他，反而更加爱他了。丽达决心尽自己的一切力量来挽救拉兹，让他重新做人。她在海滨找到了正在孤独、痛苦中徘徊的拉兹，安慰他，鼓励他，并把他带到自己的保护人法官拉贡纳特那里。拉贡纳特用怀疑、鄙视的眼光看着拉兹，并像在法庭上审问犯人一样质问他。拉兹受不了这种侮辱，气愤中离开了丽达家。一路上他绝望了，知道有拉贡纳特存在，他再也不能回到丽达的身边了。他诅咒这不公的命运。

　　这一天警察开始大搜捕，发现扎卡在行窃，就拼命追赶。扎卡无处可逃，溜进了拉兹的家。拉兹母亲看到扎卡，义愤填膺，旧仇新恨一起涌上心头，痛斥扎卡不仅毁了自己，而且毁了她的儿子。她想把扎卡赶出去，但遭到了扎卡的反扑。扎卡欢手卡住里列的脖子，企图杀害她。恰在这时拉兹赶到，为了救出母亲，他将扎卡刺死了。警察闻讯闯进大门，拉兹就此被捕，审查这个案子的正是拉贡纳特。

　　有一天里列想去法院申诉，被拉贡纳特的汽车撞伤。消息传来拉兹悲痛欲绝，他在警察的押送下，前往医院看望病危的母亲，里列这时才告诉他拉贡纳特就是他亲生的父亲。拉兹悲愤交集，他万万没有想到他的悲剧是父亲一手造成的：不仅被赶出家门，而且断送了爱情，眼下母亲又死在父亲的车下。他忍无可忍，从监狱中逃出来，潜进拉贡纳特的住宅，决心把生父刺死。拉贡纳特这时正在看书，忽然发现有刺客，就急忙进行自卫。拉兹体弱抵敌不过，于是再次被捕。行刺法官，他的罪名也就更大了。

　　丽达在得悉了拉兹的身世和犯罪的原委后，亲自担任拉兹的辩护律师，在法庭上和起诉人拉贡纳特法官进行了针锋相对的斗争。她义正辞严地宣布：有罪的不是拉兹，而是他的父亲拉贡纳特，是他为了维护封建观念和社会名声，无故将妻子和儿子赶出家庭，是社会的黑暗势力把拉兹推上犯罪的道路。拉兹也在法庭上向人们呼吁：贫穷和饥饿逼得自己受苦受难无法摆脱，像自己这样的人社会上太多了。他们有什么罪？法庭判了自己一个能消除社会犯罪的根源吗？法庭上鸦雀无声，人们被他们的激昂言词强烈地感染了。

　　由于丽达的辩护和公众的舆论压力，法庭最后判处拉兹三年有期徒刑。拉贡纳特拖着沉重的步伐去监狱探望拉兹，丽达也去了。她决心等待拉兹，她告诉拉兹未来是充满光明的。

　　一线阳光慢慢地射进监狱，拉兹放开了丽达的手，满怀激情地看着丽达。监狱的大门关上了，那动人的歌曲"到处流浪……"又响了起来。
