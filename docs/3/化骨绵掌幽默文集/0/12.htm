三、泡妞绝技

　　老鬼和贾老练经常躲在阴暗的角落里讨论女人。

　　老鬼说，女人是祸水。贾老练说，错！女人是水货。

　　老鬼说，痞子蔡语录：女人只迷信她们的感觉。因为女人可以不介意你是否高大，可以不在乎你是否酷毙，可以忍受你不够温柔体贴，可以接纳你不够细心呵护，可以宽恕你不够聪明幽默，但绝不能原谅你不会制造浪漫，她们经常在浪漫的感觉中体验幸福。我的语录是：女人是音乐的俘虏，是琼瑶的跟屁虫，因为只有音乐和琼瑶才能编织女人们所需要的白日梦。

　　贾老练说，错！女人只迷信她们的耳朵。因为女人是一种非常奇怪的动物，她们美丽的眼睛只是一个摆设，所以与其做十件体贴的事让她欣慰，倒不如说一句好听的话让她感动。

　　老鬼说，怪不得有位女作家说男人的甜言蜜语就是迷惑女人的汤药。

　　贾老练问，你晓不晓得？男人在向女人的耳朵里灌这些爱情汤药时，最大的障碍是啥子？

　　老鬼毫不犹豫地答道，最大的障碍是女人的耳屎没掏干净。

　　贾老练说，错！是男人的胃。因为当你讲出一扒拉阿谀奉承、谄媚巴结的恶心造句而一点也不觉得翻胃呕吐时，你绝对已是泡妞九段。
