《大话西游》四川话版之《民歌王子孙悟空》


　　主要人物：孙悟空，三轮车夫，做了一个美妙的梦。

　　 唐僧，水果摊主。

　　 猪八戒，贩卖猪鞭。

　　 沙僧，奶牛饲养场主。

　　 卡拉OK评委数人，中年妇女甲乙，记者一二三。

　　（画外音。四川话）师徒四人从西天取经转来，已经有5000年历史了。那么，这么长的时间他们在抓子喃？唐僧卖水果，生意秋得来遭不住。猪八戒卖猪鞭，只能算是将就。沙僧养奶牛，但奶水严重不足，只能偷偷给奶牛做隆乳手术，结果……咳！惨哦！罚款罚得他娃娃想跳楼。只有孙悟空混得还不错，是个很成功的三轮车夫。现在他还有了更高的生活追求，准备参加电视台举办的第1000届卡拉OK大奖赛……

　　（●场景一）一间破旧的小屋。唐僧、猪八戒、沙僧在一张小桌前正襟危坐，表情严肃。

　　唐僧：（用川普高叫）一号选手入场。

　　孙悟空从屋外迈着杂技演员的步伐入场，一个亮相。

　　猪八戒：（不耐烦地）重新来过重新来过。鬼眉鬼眼的，跟猴子扳疙蔸两个样。电视台的导演看到你龟儿这个样子就够了，肯定喊你：（川普）爬哟。

　　沙僧：（对猪八戒）不是爬哟，是：（川普）出去出去！哈（傻）戳戳的。

　　猪八戒：你的普通话更Y，Y得来有盐有味的。

　　唐僧：（对孙悟空）你要严肃点哈，请我们来扮演评委，你不要醒儿和桑的，大家都是有工作的。

　　孙悟空：好好好。

　　孙走出去，重新进屋。这次是弯脚杆，膝盖不能并拢。

　　唐僧：你中间咋个稀开那么大的缝缝喃？并拢噻！

　　孙使劲并，还是并不拢。

　　猪八戒：（讽刺孙）骑马蹲档式，有点提劲。

　　沙僧：有点像猴子屙尿。

　　孙悟空：蹬三轮蹬惯了，下把下改得脱嗦？

　　唐僧：姓名？

　　孙悟空：孙悟空，号称齐天大圣。

　　唐僧：家庭住址？

　　孙悟空：水帘洞B座16楼A面。

　　唐僧：职务？

　　孙悟空：曾经担任弼马温，高级干部。退休后反聘为三轮车管理员，一般群众。

　　唐僧：性别？

　　孙悟空做出扭捏状，猛眨眼。

　　孙悟空：你看不出来嗦？

　　猪八戒：（不耐烦地）我看你像幺哥。老子今天还要去高老庄火锅店送猪鞭，时间就是金钱得嘛！

　　沙僧：我还不是要去挤牛奶。

　　唐僧：我看你倒像泰国人妖。人是人生的，妖是妖生的。人是人他妈的，妖是妖他妈的……

　　孙悟空：（普）你妈贵姓？（川）我就晓得你要问这一句。

　　唐僧：（一楞）当真！（普）你妈贵姓？

　　

　　（●场景二）某电视台。大奖赛评委们正襟危坐。

　　主考官：（普）请一号选手入场！

　　孙悟空还是采用杂技演员的姿势入场。

　　主考官：（对评委甲耳语）姿势倒是蛮特别的。

　　评委甲：（川普）可能吃过专业饭。

　　主考官：姓名？

　　孙悟空：孙楠，不！孙国庆，不！孙悦。（自语）孙悦是女的得嘛。（大声）我叫孙悟空。

　　主考官：年龄？

　　孙悟空：50高龄。

　　评委们交换眼色，纷纷说："精神可嘉。"

　　主考官：性别？

　　孙悟空刚想摆出扭捏相，转念一想，放弃了。

　　孙悟空：男的。

　　主考官：你参赛的曲目是……

　　孙悟空：我参赛的曲目是贵州民歌。

　　评委甲：（对旁边的评委乙）很难得不唱港台歌曲，嘎？

　　主考官：（对旁边）不要高声喧哗！影响选手正常发挥。（对孙悟空）你可以开始了。

　　孙悟空开始唱贵州民歌。（由巴登自由发挥）

　　主考官满眼含着泪水。

　　主考官：我好感动啊！这么多年没听到这么好听的，原汁原味的民歌了。

　　旁边的评委哭成一团。

　　

　　（●场景三）破旧小屋。

　　（镜头一）唐僧正啃西瓜。

　　（镜头二）猪八戒正用猪鞭下酒，嘴里嚼得卡哧卡哧响。

　　（镜头三）沙僧正抱着奶瓶喝牛奶。

　　孙悟空从外面进来，趾高气扬的，一副吃不完要不完的样子。

　　孙悟空：（对唐僧）你有点吃相行不行？好赖你还是一个名人哦。

　　（唐僧愣住了）

　　孙悟空：（继续）你完成了西天取经那么伟大的事业，实在让小弟佩服。但你从此就沉湎于过去的辉煌之中，不思进取，浪费前程，我真的为你惋惜。你看你吃西瓜的样子，跟猪八戒差不多。一代明星从此堕落。（对猪八戒）还有你！一天到黑贪酒贪色，你以为你吃了猪鞭就能雄起嗦？就算你吃了100斤猪鞭又能抓子？有没得伟哥凶嘛？就算你很凶，你耍得到女朋友吗？就算你有女朋友，你买得起房子和汽车吗？你还不是只有到梦里水乡当编编匠。（对沙僧）最后是你！（看见沙僧喝奶，吃惊地）你在抓子？几个月的娃儿嗦？

　　唐僧：（不屑地）婆婆妈妈、炸炸哇哇，像个苍蝇"咻――"，飞进我的耳朵。

　　猪八戒：猴哥，你今天吃了耗儿药啊？

　　孙悟空得意地摇头晃脑。

　　孙悟空：我现在地位不同了，看到就不顺眼。

　　沙僧：（拿开奶嘴）你看哪个不顺眼嘛？

　　孙悟空：（恶狠狠地盯着沙僧）我看你就不顺眼。我现在是……（杂技演员的亮相动作）民歌王子。

　　猪八戒：哪个给你封的？

　　孙悟空：电视台的导演噻。

　　唐僧：悟空，我早就跟你说过，不要追求毫无价值的虚荣。虽然现在你仅仅是个三轮车的司机，但你一不怕苦二不怕死艰苦奋斗的革命精神已经得到全社会的充分肯定。现在骗人的Y导演多得遭不住，还不是唬你的烟烧。

　　孙悟空：这个不是Y导演，人家是获得过五一工程奖的。

　　猪八戒：现在的导演还要包工程嗦？

　　正在这时，门外闯进来两个中年妇女。

　　女甲：（对女乙）找到了找到了，民歌王子在这里耍。

　　二女拿照相机给孙悟空拍照，镁光灯猛闪，孙悟空连忙作亮相姿势。

　　女甲：（拉住孙的左手）（港普）人家找的你好辛苦好辛苦耶！

　　女乙：（拉住孙的右手）（港普）人家好喜欢你的歌耶！

　　唐僧、猪、沙目瞪口呆。

　　猪八戒：（嫉妒地）（模仿港普）人家好肉麻耶！

　　女甲：（白了猪八戒一眼）又没跟你两个说。

　　孙悟空：（春风得意）我马上要推出我的第一张个人大碟，主打歌是《不敢问路在何方》。

　　二女作"V"型手势：Yeah――！！！

　　孙悟空：我一定不辜负歌迷的厚爱，谢谢！非常谢谢！

　　女甲：可以给我们签个名吗？

　　孙悟空：可以可以。

　　女甲拿出签名笔，递给孙。

　　孙悟空：签在哪里喃？

　　女甲：（转身）就签在我的背上。

　　孙悟空：洗不脱的哟？

　　女甲：人家喜欢嘛。

　　孙给女甲女乙分别签名，然后欣赏自己的签名。

　　孙悟空：（连连摇头）写得不好写得不好。

　　（签名特写）类似阿拉伯文的弯曲的字母。

　　猪八戒：（凑上来看）简直是鬼画桃符。

　　唐僧：（对猪八戒）明星的签名都是这样的。我早就喊他练习书法，他总把我的话当成耳旁风。给他买一支毛笔，他就丢一支，买一支丢一支，砸到小朋友的脑壳咋个办？

　　（正说着，孙已经和二女握手告别）

　　孙悟空：二天来耍就是了，不关事。拜拜。

　　（二女出门后，孙忽然想起什么。）

　　孙悟空：（大叫）哎！等到起！我拿三轮送你们！（说罢追了出去）

　　唐僧、猪、沙三人面面相觑。

　　猪八戒：耶？就这样就可以当明星了嗦？

　　唐僧：这叫一炮走红。

　　沙僧：我们要给大师兄抽起噻。

　　猪八戒：咋个抽起？

　　沙僧：你们看！我们的房子像不像个猪圈。

　　猪八戒：你不要出口伤人哈。

　　沙僧：（继续）到处是脏兮兮的，还有那么多蜘蛛网网吊起吊起的。

　　猪八戒：那是我的爱人春三十娘留下的爱的足迹。

　　沙僧：（继续）一点都不符合大师兄民歌王子的身份。要是有啥子记者来采访，未必就这样子曝光嗦？

　　唐僧：你说咋个办？

　　沙僧：我们一家吐点血，好生装修一下。

　　猪八戒：一家出好多喃？

　　沙僧：起码1000元嘛。

　　猪八戒：不干。

　　沙僧：为啥子？

　　猪八戒：我要卖好多根猪鞭哟。

　　唐僧：你不干也得干，大师兄待你不薄哟。

　　猪八戒：（委屈地）要得。（心疼地）可惜我的猪鞭呀！

　　

　　（●场景四）一组三人粉刷墙壁、抬沙发、抬办公桌的镜头。

　　

　　（●场景五）屋内焕然一新，俨然一个公司的办公室。师徒四人端庄地并排坐在办公桌前，每人面前有一个牌子，上面分别写：民歌王子、唐总、朱总、沙总。众多电视、报纸记者就座。办公桌上方有一个横幅。（横幅特写）民歌王子新闻发布会。

　　唐僧：民歌王子新闻发布会现在开始。记者们可以自由提问。

　　记者一：请问王子，你成功的秘诀是什么？

　　孙悟空：（微笑）我成功的秘诀就是没有秘诀。（记者们纷纷点头）

　　记者二：请问你最崇拜的明星是谁？

　　孙悟空：我最崇拜我自己。（记者们议论纷纷）

　　记者三：请问能不能提一个很私人的问题？

　　孙悟空：（一伸手）请讲。

　　记者三：你结婚没有？

　　（孙不答，做出羞涩状。）

　　猪八戒：请记者不要向少年儿童提这种问题。

　　记者三：可他不像个少年儿童啊。

　　猪八戒：我说他是少年儿童就是少年儿童。谭咏麟都说他永远25岁，你相不相信嘛？你再纠缠这些无聊的问题，我马上撵你出去!

　　（众记者怪罪记者三，纷纷说：不要紧到缠到别个的隐私嘛！）

　　记者一：请问王子，能不能告诉我，谁对你的演艺生涯影响最大？

　　孙悟空：要说哪个对我的演艺生涯影响大，还是我身边的几位专家。

　　（唐僧、猪、沙立即挺直腰杆。）

　　记者一：请问唐总，能不能介绍一下你自己？

　　唐僧：我是专门研究人类营养学的，比如西瓜里含有多少维生素C。营养丰富了，歌喉才能滋润，才能展现他优美的歌声。

　　记者二：请问沙总，你也是一个营养学专家吗？

　　沙僧：确切地说，我是一个美容教授，比如我专门研究牛奶对人类皮肤的影响。喝多了，就会变成奶油小生，喝少了喃，脸上的皮肤就会干滋滋的，一点没有弹性。这对一个演员来说非常重要，因为你要给观众带来美的享受，而不是让观众看到恼火。

　　记者三：请问朱总，你也是一个美容专家吗？

　　（猪八戒白了记者三一眼。）

　　猪八戒：我看就你是个假老练，顺到杆杆爬嗦？

　　记者三：那你是个毁容专家哦？

　　（众记者怨声载道，纷纷撵记者三。）

　　记者一：出去！哪个小报的记者哦？

　　（记者三被撵了出去，门外传来一声惨叫。）

　　猪八戒：这些小报只能下课，莫搞。我……（环视四周，众记者安静下来。）既不是营养学专家，也不是美容教授。我是专门研究男性科学的。

　　记者一：也就是研究伟哥喽？

　　猪八戒：伟哥是门伟大的科学，但只适合美国。拿到我们中国来，就不一定符合我们的国情。第一，广大老百姓消费不起。第二，据广州一个科研单位调查，99%的妇女认为，伟哥让她们只痛不快。所以我正在研究一种适合老百姓消费的壮阳药，这么大一盘，才人民币10元。比如说猪的……（停顿）今天在场的有很多未婚女青年，我就不多说了。

　　记者一：吃了你的药，对唱歌有帮助吗？

　　猪八戒：当然有帮助，增加阳刚之气噻。（指孙）他最近吃的少了，所以有点母兮兮的，我要加大用量，争取把他塑造成中国的史瓦辛格。

　　（众记者鼓掌）

　　记者一：我提议，请民歌王子现场给我们演唱一首新歌，好不好？

　　众记者：好！！！

　　孙悟空：（起立鞠躬）谢谢！现在我为大家演唱一首我最近创作的维吾尔民歌《****》。

　　（由演员自由夸张发挥）

　　（掌声如潮）

　　

　　（●场景六）"哐啷"一声巨响，类似玻璃破碎的声音。紧跟着夹杂着拳脚相加的声音。

　　孙悟空被打得滚在地下，"妈呀妈呀"地叫。唐僧、猪、沙均穿着内裤，上身精光。

　　猪八戒：（狠狠踹了孙一脚）半夜三更的唱个铲铲啊？还让别个睡不睡？

　　唐僧：（踹了孙一脚）老子还要卖水果。

　　沙僧：（跟着踹）我还要到牛棚挤奶。

　　猪八戒：（又踹）老子还要到屠宰场进猪鞭。

　　孙悟空：（躺在地下，哭腔）我还不是要蹬三轮车。

　　三人：你晓得嗦？（说罢三人打着哈欠，转身上各自的床睡觉）

　　（特写）孙悟空鼻血流了出来，嘴皮也肿了。

　　孙悟空：（自语）原来是个梦嗦？好安逸哦！

（剧终）
