菜鸟办网站的艰难历程

　　我是个狂热的网虫，在军事，政治和文学音乐体育等领域有着很高的造诣。除了吃喝拉撒睡，我基本上都在网上游泳。"冲浪"的动作有点高难，所以我只能选择"游泳"，且经常被呛得伏尸荒岭。

　　想当初我还是个菜鸟级的人物，我认识电脑，但它不认识我。虽然我只读到小学第四册，但为了跟时代同行，我用光了多年的积蓄买了一台三代奔腾。我最大的特点就是厚着脸皮不耻下问，而且从来不脸红。为了解决在安装软件过程中有可能出现的尖端技术，我时刻与卖给我电脑的付二娃保持热线畅通。第二天，我打电话给他："二娃，电脑弹出一个菜单，叫我按任意键。这个任意键到底在哪儿？我已经找了8个钟头。"

　　气得二娃在电话那头大叫："你个瓜娃子，任意键就是随便按一个。"Yeah!我懂！

　　一星期过后我又打电话给他："二娃，你说你的电脑质量很过关，我刚刚买了一个星期，电脑上的茶杯架就坏了。"

　　"电脑上哪里有茶杯架？"二娃不解地问。

　　"就是一按就出来，放茶杯的那个架子。"

　　"我马上过来铲你一耳屎！"二娃气急败坏地怒吼。

　　后来我才知道那个东西叫光驱。

　　真正使我脱离菜鸟级别的还是我现任的女朋友。那是我刚刚认识OICQ的时候，都说那东西好，可以在网上勾兑女朋友，我立即迫不及待地安装了一个。因为我自认为自己跟周星驰一样无厘头，所以我的登录名叫周星星，并且加上一个：-)~~，以示我渴盼女朋友的迫切心情。（那个符号要把头偏向左才能看懂，意思是：流口水。）

　　我是在一个叫"逃学威龙"的聊天室认识她的，她是重庆美眉，昵称：暗恋周星星。这不是摆明了在寻找我吗？于是我们立即一拍即合，携手开了一个爱情单间。自然而然她变成了我的老师，我不懂就问，她有问必答。我们的网恋在一夜之间迅速升温，成渝高速公路已经在我们的键盘中化为乌有。她是个电脑专家，所以她想亲手把我塑造成黑客级的IT界精英，并帮我建立了个人主页，专门介绍我的帖子。在她的帮助下，我的主页日访问量不断创下吉尼斯纪录，只有一个叫吴孟达的网友紧随其后。后来我成了各大网站BBS上的老油条，我不断进去刷新我创作的《菜鸟帖》，以便把我的创作水平及时升级。最新的版本是："每当夜深人静的时候我总是睡不着，我怀疑是不是只有我的奔腾Ⅲ没有变得更好。OICQ会怎样，究竟有哪个美眉知道？￥的价值和爱情的尊严哪一个重要？我是一个成都的菜鸟，想要飞却飞也飞不高……"

　　最近有两件事说明我的巨大进步。第一，由于我周星星的名气越来越大，让许多人仰慕，我的OICQ号码12348500几乎被人打爆。于是我被邀在一个网吧与一帮网络辩论高手过招，我凭着高人一筹的打字速度、良好的键盘性能和胡搅蛮缠的本领，将那帮高手驳得哑口无言，屁滚尿流。网吧里的电脑全被我的唾沫星子吹到了浩瀚的天空，掉下来的时候险些砸烂我的脑袋。就连一个13岁未成年美眉的夜饭――一盒面霸也被我吹得跳了起来，正扣在我的头上。烫得我"哇哇"乱叫，大喊一声："我走先！"便头顶着卷曲的面条狼狈地逃到街上。

　　第二，我准备建立一个网站，虽然只有十几台机子七八个猫，但凭借我独有的虎狼之心，我一定能战胜搜狐网易新浪，将我的网站打入美国市场。但暗恋周星星奉劝我说："你的实力还远远不够。"

　　我自信极了，岂能让她阻挡？我耐心地向她解释："虽然我是小学水平，但我的智力却是博士后。网站都是养狗的发明的，难道我身上的零件比他们少？"

　　她惊问："哪个说网站是养狗的发明的？"

　　"你晓得网站的后缀.com咋个读？"

　　"我啷个不晓得？dot com."

　　"对喽！dog come.狗来了。这充分说明网站是酷爱养狗的人发明的。"

　　她马上昏倒。还没开始赚钱，这么激动干什么？要是在美国上市，她肯定昏死。这个网站我办定了。

2000/7/31
