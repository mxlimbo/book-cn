女人刺伤我的眼

　　小胖娃的眼睛肿了。我晓得今年的太阳紫外线特别凶，但和小胖娃的肿眼没有任何关系；我还晓得他隔壁子就是一个建筑工地，那儿通宵达旦都有100个电焊工在热浪中挥汗大干，这些都对小胖娃的眼睛没有任何刺激。小胖娃的眼睛是被今年夏天的女人晃肿的。

　　小胖娃的眼睛肿得来眯成两条缝，他叹着气对我说："人家说女人走光是养男人的眼睛，不料今年夏天女人们走得都是电焊光，把我的眼睛刺得筋痛。"

　　"你不会仰起脑壳走路？"

　　"我也不是没试过，结果身后跟了一趴拉瞅热闹的。有个穿红色吊带衫的老嬷嬷问我：'是不是有不明飞行物？'我对她说：'我看你就像个ET，而且还是成都出生的。'哎！现在真是：上街不看人，看人走不成。"

　　"有那么严重？"

　　"女人五大露：胸腿背膀肚。不管她是肉感还是骨感，统统暴露。但有些女人养眼，有些女人刺眼。"

　　"不要停顿，赶快摆。"我迫不及待想知道答案。

　　"自从宁静号召'做女人挺好'，再加上三级电视广告铺天盖地，效果非常明显，全体女同胞从封建的束缚中彻底解放出来，犹如脱笼的惊兔。尽管形状各异，重量不同，但毕竟给我们这座城市带来无数道亮丽的风景线。听说已经有三个男人因为眼睛太贪婪，撞倒了直径一尺的电线杆。当然，有些是以假乱真，这个不必讲究只能将就，爱美之心谁也不能责难。我最不能容忍个别女人没胸露胸，装起多凶，那样最费我的眼睛。再说这腿：一双腿的质量决定一个女人是否高贵，它最能让一个正常男人想入非非。有些人愿意在腿之前加上'粉'啊'玉'啊之类，我觉得这不仅是简简单单的赞美，完全是赞歌。有些女人的腿质量确实相当过关，随便可以和玛丽莲梦露、莎朗斯通的腿相媲美。不过有些暴露的萝卜腿就惨了，那简直是在糟踏美丽的夏天。"

　　我点头称是，心底已经暗暗佩服这小子的钻研精神。

　　"现在讨论背。"小胖娃继续发表高论，"一个美好的背是让人永生难忘的背，它应该没有许多肉，但看上去有一种丝绒般的柔软。当你看到这样的背正是她离你远去的时候，所以显得珍贵，你必须加快步伐才能继续享受。我为了看清一个有这种质量的背足足在二环路跟踪跋涉了一圈，看安逸了，一点不觉得累。而有些女人的背，确实不敢恭维，赘肉太多，再加上颈项后面有个肉包，像小时候担扁担落下的疾患，揉都揉不散。一看到这种背，我就好想找根扁担给她递过去。第四说膀膀，也就是胳膊。美丽的胳膊只能用嫩藕来形容，这是女人最普通的暴露，已经很少引起男人的惊叹。我想要说的是，女人千万不要随便张开她的双臂迎接大自然，除非她像芭蕾舞演员那样来一次大扫除。目前大商场里对付这类丑恶现象的工具和药品琳琅满目，她可以随心所欲地自由挑选。但有些女人却忽略这一点，长得像吴倩莲，其实比吴君如还要悬。尤其在公共汽车上，为了掌握平衡，最能暴露她们的弱点。这要是乘夜车倒也罢了，偏偏是在光天化日之下。我只想把那英的一首歌送给她。"

　　"啥子歌？"

　　"《白天不懂腋的黑》"

　　我差点卧倒。

　　"最后说最关键的：肚。其实露肚也就是露肚脐眼儿，美脐最能捕捉男人的目光，当你看到一个妙龄女郎露着精美的肚脐眼儿像你翩翩走来时，你难道不惊慌失措吗？以前我们只能在印度电影里才有幸欣赏到这类美丽的画面，现在遍街都是，数都数不完。我认为这是本世纪初女人对陌生男人最无私的奉献。"

　　"你的眼睛到底被哪种女人刺伤的？养眼的还是刺眼的？"

　　小胖娃丝毫不掩饰我们男人的劣根，理直气壮地说："爱好很广泛，啥子都要看。"
