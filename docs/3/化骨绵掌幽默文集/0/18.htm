九、浴室惊魂记

　　贾老练司职的公共浴室每星期都要彻底打扫一遍卫生。为了工作方便，男女浴室之间设有一扇木质小门，平时都是铁将军站岗，只有星期天晚上打扫卫生时才临时捅开，以便将男女浴室一次性冲洗干净。开这道门的权利惟贾老练独享，他经常为自己有特权堂而皇之地跨进女人的世界而洋洋得意。

　　这个星期天，中央台5套正好直播意甲联赛一场重头戏——米兰双雄会，贾老练心里像猫儿抓似的，恨不得立即关门，插上翅膀飞回家里看罗纳尔多和舍甫琴科的较量。无奈浴池里有几个中年人紧到在那儿磨皮擦痒，有一个竟然躺在浴池边上眯着眼睛"晕"味儿，嘴里断断续续地哼着："你说过两天来看偶（我），一别就是一年多……"贾老练气惨了，心里暗骂："我今天撞到鬼老二了，看到一个一丝不挂的情哥哥。"

　　好不容易熬到一个鬼影子都没得，贾老练急急忙忙用水龙头草草地冲洗了一遍男女浴室，然后骑上他那辆扯拐赛车飞也似的朝家里蹬去，边骑边想："那两个米兰兄弟害怕都打到8比8了。"他还算幸运，现场直播的下半场总算看到了，但他忘记了一件最最重要的事情，男女浴室之间那道门没有上锁。

　　星期一，老鬼肩搭一条印花毛巾来洗澡，走到门口时对贾老练说："一会儿你如果有空，进来帮我搓搓背，在家里的卫生间洗澡，永远洗不干净。"

　　不一会儿，贾老练赤身裸体地走了进来，手拿一条干毛巾对老鬼说："趴到门上去！"

　　浴室的墙壁太滑，那道木门上倒有一个铁把手，扶着它正好可以搓背，况且这是老鬼长年累月搓背的老地方。老鬼乖乖地抓紧门把手，屁股翘起，屏住呼吸，闭上眼睛，准备好生享受一下。哪想到贾老练的莽劲有点大，轻轻一搓老鬼的背，"哐"地一声将那道没上锁的禁区之门撞开了，同时也将一丝不挂的老鬼撞了过去。女浴室那边顿时像炸了营的马蜂窝，一片"嗷嗷"怪叫，有个老孃儿还算清醒，歇斯底里地大叫一声："赶~快~关~门!!!"

　　木门"砰"地关严了，贾老练赶快耳贴门缝，只听见老鬼在那边像一只误入狼群的羔羊，不断无助地哀鸣着："千~万~不~要~关!!! 我还在这边！"
