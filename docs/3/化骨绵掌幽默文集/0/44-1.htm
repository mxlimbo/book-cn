情景喜剧（5）：我爱麝香

阿忆

　　（画外音：二娃自从练牛皮掌成功以后，得意惨了，到处展示他的武林绝技。但是他的爱情却丝毫没有进展，谢丽亭还是不来电。二娃恼火惨了，为不能施展自己的男人魅力而苦恼。这天，谢老五又在给他出谋划策……）

　　谢：照理说牛皮掌是武林绝学，谢丽亭应该哄动春心，要晓得她以前的男人只是个卖打药的，只晓得喊"每天坚持练，打断高压线"，只能骗一些退休老职工。二娃：我比她男人高一个档次噻，牛皮掌已经练得炉火纯青，但是她好像并不在意我，她说她崇拜知识型的武林高手。像我这种骨节越练越粗的人，只能淘汰。（谢观看二娃的手掌）谢：咦？奇怪，咋个骨节这么粗喃？二娃：还不是你教我的？拿自己的尿擦洗，可以消毒去瘀。谢（沉思良久，忽然猛拍大腿）：我想起来了，不能拿自己的尿，那样肯定有副作用。二娃：那拿啥子喃？谢（紧盯着二娃，一字一句地）：拿--麝--香。二娃：啥子麝香？谢：麝是一种鹿科动物，它在森林里狂奔的时候肚脐眼儿是张开的，于是啥子蚊子苍蝇就钻了进去，钻进去肯定就痒噻，于是它就把肚脐眼儿闭到。等一会儿又张开，又闭到，久而久之肚脐眼儿就变成了一个硬砣，味道有点臭，稀释以后才香，这就是麝香。二娃：恐怕有点贵哦？谢：肯定噻，你消费不起不说，主要是不好找，有点珍贵。但也没关系，我们可以制造人造麝香。二娃：人造麝香？谢（神秘地）：千万不要外传哦！二娃猛点头。谢：我这个是已经失传的绝技，听到哈！现将竹子的上半截砍去，往里面灌一些猪肠子鳝鱼骨头之类的东西，于是蚊子苍蝇蠛蚊小咬之类的就钻了进去，你马上用砍去的上半截把竹子扛（盖）到，记住，要用橡皮筋拴牢，不能漏气。等半年过后那个地方就变成一个硬砣，这就是人造麝香。二娃惊喜地：真的？！谢：我还骗你嘛咋个？以后你就用麝香擦洗你的牛皮掌，骨节不但不变大，还可以柔情似水，谢丽亭一定喜欢惨了。二娃：要得，我喜欢麝香！

　　（字幕：三个月过后……）

　　（二娃见到谢老五就急忙解裤子，谢惊慌失措）：喂喂喂！你在抓子？实在憋慌了到公共厕所噻。（二娃边解裤带边激动地）：我给你看看人造麝香。谢（吃惊地）：啥子？人造麝香？二娃：竹子确实不好找，我就将就我的肚脐眼儿，撒了一些蜂蜜豆腐乳水水。现在我的肚脐眼儿已经变成一个硬砣，你闻一下，看像不像人造麝香？

　　谢老五两眼翻白，马上昏倒……
