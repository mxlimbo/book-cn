裘大爷看跳水

　　裘大爷自从给中国足球搞科研失败以后，一直闲赋在家磨皮擦痒，不晓得整点啥子来丰富自己的业余生活。他先是喂了一只跛脚鹦鹉，准备寒夜凄凉时跟它摆摆知心龙门阵。不料随便你咋个教，那只烂鹦鹉整死不开腔。后经过鹦鹉鉴赏家鉴定，原来是个哑巴，气得裘大爷把它蒸来吃了。后来他老人家又不知从哪儿搞来一只秃毛泰国斗鸡，心想等它翅膀长硬了，带它到红牌楼耍耍，哪怕客死疆场也威风了一盘噻。哪想到秃毛斗鸡到来的第一个晚上就把隔壁王老孃儿非常威武的大公鸡的眼睛给啄瞎了，没过一个星期，方圆5公里之内的公鸡母鸡包括鸽子，全部双目失明。在左邻右舍的强烈抗议声中，裘大爷只有忍痛把它卤来下酒了。

　　实在闲得无聊，眼看着渐渐苍老，2000年奥运会及时拉开了帷幕。裘大爷哪肯放过这等大开眼界的机会，立即奋不顾身融进悉尼赛场。看来只有观看体育比赛能使裘大爷年轻。

　　那天三米板跳水决赛，熊倪和萨乌丁铁青着脸正在较劲，裘大爷紧张惨了。眼看着萨乌丁秤砣般的身躯一次次砸向水中，居然没溅起一点水花儿，急得裘大爷直喊："熊娃儿，往萨乌丁刚才入水的那个地方栽，那里肯定有个缝缝！"轮到最后一跳时，熊倪还差萨乌丁11分。裘大爷的头顶像蒸笼盖两个样，直见冒烟烟。此时萨乌丁已小心翼翼走上踏板，他平端双臂，深吸了一口气，然后微微闭上眼，嘴唇轻轻抖动着。就在萨乌丁从踏板上弹起的一刹那，裘大爷猛地跳起来，大吼了一声："绊！"

　　萨乌丁好像被裘大爷吓到了，"扑通"一声栽进水池，水花溅起多高，几乎跳了个门板，才得了60多分。裘大爷笑安逸了："熊倪雄起！"

　　当夜裘大爷越想越不是滋味，心想：如果不是我那么惊炸鼓响地吼一声，他娃娃就是冠军了，这好像对他很不公平，毕竟体育竞技是建立在公平竞争的基础上的。他翻身披衣，挑灯奋笔，连夜给萨乌丁写了一封道歉信：

　　尊敬的萨乌丁先生：

　　我是一个来自中国的超级老体育迷，我深深佩服你良好的体育道德风范和你高超的跳水技艺。你可能不知道，你那最后一跳可能是我给吼砸了，在你入水的一刹那，不晓得你听到一声"绊"没有？我想你可能听到了，不然不会绊得那么惨。不过话又说转来有一句说一句，你独霸了"跳坛"好多年了，用我们四川话说就是有点跳颤。这次也该让我们的熊倪翻下梢了噻。以前他被洛加尼斯欺到欺到的，后来又换成你，水土要轮流转噻。不过，体育要公平竞争，如果是我那一声"绊"影响了你的正常发挥，在此我向你表示千万次的深切歉意。本来我想把你吼成亚军，哪想到一下子吼飙了，被一个墨西哥选手薅了去，想起来有点可惜。二天到我们中国四川成都来耍，我请你吃麻辣烫。

　　写完这封信，裘大爷舒服多了。当晚睡了一个好觉，还做了个好梦。他梦见和萨乌丁一起在府南河练跳水，动作非常优美，就是河水有点浅，裘大爷的脑壳被鹅卵石盅了几个包吊起……
