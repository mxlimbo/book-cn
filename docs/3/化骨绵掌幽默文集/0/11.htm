二、女人的心思

　　老鬼在舞厅里旋过去旋过来，贼眉贼眼地寻找漏网之鱼，只要看到孑身一人的女人就想打别个的"启发"⑴。哪想到女人们跟他跳舞都扳起僵尸似的脸嘴，腰杆挺得梆直，身体隔起帽子远，老鬼的右臂早已麻木酸痛，哪里拖得动？稍微有点苗条的都被隔壁的街娃⑵们请去了，剩下的都是泰坦尼克号，而且是清一色的爆眼子老嬷儿。老鬼连蹦了两曲便索然无味，悻悻然靠在角落里生闷气。但见舞池中那些窈窕靓女伏在街娃们的肩头一会儿殷殷浅笑，一会儿咯咯浪笑，一会儿粉拳捶胸，一会儿纤指掐背，看得老鬼直流鼻血。老鬼愤愤地说："女人真是奇怪的动物，她偏偏喜欢有点坏的男人。"

　　此时，坐在黑暗中一直没开腔的贾老练纠正道："不！女人喜欢有点好的男人！"

　　老鬼顿时大悟。他准备第二天先砍翻几个人摆起，在社会上操出点名气，然后再到舞厅找那些粉子些献殷勤。

　　⑴"启发"：鬼主意

　　⑵街娃：街上的混混儿
