一九八七年

 

获奖最多港姐：杨宝玲
　　出身豪门的杨宝玲本应做商界女强人，最终却因选美晋身娱乐界。智慧与美貌同样惊人的她，在1987年多个世界性选美大赛中打破港姐参选大吃零蛋的记录，在“世姐”选举中，得到“亚洲皇后”荣衔，而“环球小姐”又得到第5名，为香港争了光。

　　虽然在娱乐圈发展平平，但她以其知名度大行投资，成为圈中亿万富姐，同时醉心主持电台节目，是圈中少有的“才女”。

本届出了一位最性感港姐：邱淑贞
　　1987年港姐竞选最大新闻便是大热门邱淑贞的神秘退出，原来因为当时被人揭发她整过容，她因受刺激愤然退出，此举却令她成为风云人物，无线也趁机与其签下合约大加力捧。

　　1993年闯入影坛的邱淑贞，在大哥大王晶的安排下一弃清纯形象，以性感却不低俗的形象出演了40多部电影，部部票房爆收，她也成为性感代言人。1998年与王晶分手后，邱淑贞一改形象，以实力派出演争议性电影《愈快乐愈堕落》，令人耳目一新。近来忙于恋爱的她已经鲜有露面。
