一九八三年
香港小姐1983 冠军：杨雪仪

亚军：张曼玉

　　张曼玉是迄今为止从港姐选举中走出的最为成功的国际巨星。17岁时从英国返港的她被星探发现做了一年模特，随后夺得港姐亚军走上影坛。起初，张曼玉都是扮演花瓶角色，经过自己不断努力，1989年以《三个女人的故事》夺得金马影后，1990年又以《不脱袜的女人》夺得金像奖影后，而1991年一部《滚滚红尘》令她再夺金马奖最佳女配角。

　　令张曼玉扬威国际的是，1992年反映30年代电影人的传记片《阮玲玉》，该片令她成为第一位在国际上四大影展封后的华人女性，而1997年、1998年，她在《宋氏三姐妹》《甜蜜蜜》中的出色演技，令她夺下金像奖两届影后。目前已嫁为人妇的她长居英国，正拍摄王家卫新片《2046》。
