《 小技巧大用途 》

{　　1.判断用户的屏幕分辨率而转向到不同的网页
将以下代码加入到<head></head>标志对之间。640htm，800.htm，1024.htm三种分辨率下的页面。
　　<script language=″javascript″>
　　if (window.screen){
　　 var w = screen.width;
　　 url_640 = ″640.htm″;
　　 url_800 = ″800.htm″;
　　 url_1024 = ″1024.htm″;
　　 if(w<740){
　　 self.location.replace(url_640);
　　 }
　　 if(w>=740 ＆ w<835){
　　 self.location.replace(url_800);
　　 }
　　 if(w>=835){
　　 self.location.replace(url_1024);
　　 }
　　}
　　</script>
　　2. 给文字链接加上描述
　　只需在链接标记中加上title属性，如：
　　<a href=″http://www.pcwclub.com″ title=″欢迎光临电脑报读者俱乐部″ >读者之家</a>
　　当你将鼠标指向链接文字“读者之家” ，就会显示出″欢迎光临电脑报读者俱乐部″这句描述。
　　还有很多此类的技巧，只要你多留心别人网站的制作，就会有很多意想不到的收获。
　　(广东　叶纪宁)}
