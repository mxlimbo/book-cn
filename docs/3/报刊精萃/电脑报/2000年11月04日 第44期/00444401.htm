《 FlashGet实用技巧 》

{　　一、快速添加下载任务
　　添加下载任务可以通过下面四种方法进行：
　　1.通过IE的右键快捷菜单开始下载：在FlashGet(网际快车，原名为JetCar)安装完成后，用鼠标右键点击它，在弹出的菜单中会添加“Download All By FlashGet”（下载页面中所有链接的文件）和“Download using FlashGet”（下载选择链接的文件）两项命令。当你在浏览页面中点击鼠标右键，就可以选择下载本页某个链接的文件或本页所有链接的文件。由于这种方式不需要事先启动FlashGet，同时也便于在安装了几个下载管理工具的情况下选择使用FlashGet程序而不使用其它程序，因此得到了笔者的最爱。选择右键菜单命令之一后，系统就会弹出一个下载任务窗口，其中的有关参数已经自动填充，假如你不想改动的话，只要直接点击“确定”就可以按照事先设置的参数下载。
　　2.手动添加下载任务：首先启动FlashGet，在程序的界面窗口中点击工具栏中的第三个工具图标――“新建”或“任务”菜单中的“新建下载任务”命令，会弹出手工添加下载任务窗口，你必须在其中的URL文本栏中填入你欲下载的链接地址，一般情况下不需要对其它选项再进行设置，点击“确定”即可开始下载。
　　3.使用拖动链接到悬浮窗口：在FlashGet启动后，我们会看到一个支持拖动链接地址的悬浮窗口，这个窗口总在一般文件和程序窗口的上面，你可以在浏览器中将一个文件下载的链接地址拖动到悬浮窗口中，此时FlashGet同样会打开添加下载任务窗口，并自动在URL文本栏中填充刚才你拖动的链接地址，点击“确定”即可下载。
　　4.点击网页的下载链接开始下载：FlashGet可以监视浏览器的点击，当你点击网页中URL时，它可监视该URL，如果该URL链接是你设定的下载文件类型时，则同样会弹出下载任务添加窗口，供你进行下载。
　　二、编辑下载任务
　　上面我们多次谈到下载任务窗口，实际上不管你使用哪一种下载方式，都会弹出这个下载任务窗口，不同点是手工添加下载任务时需要我们输入URL，一般情况下我们并不需要关心它。但有时也许你下载时会有一些特殊的要求，因此学会编辑下载任务窗口就是一个非常实用的技巧。
　　1.更改下载文件的保存目录：程序默认的下载文件保存文件夹是FlashGet程序安装目录中的Download文件夹，如果你需要将下载的文件临时保存到你所指定的文件夹，可在下载任务窗口的“目录”文本框中进行更改，或直接点击右边的按钮进行设置。
　　2.选择类别：FlashGet可将下载的文件默认为软件、游戏、MP3等类别。在添加下载任务窗口中也可以预先将该下载文件进行归类。下载结束后，该下载文件就保存在预先设定的类别目录中，在“类别”中进行选择即可，甚至你还可以自定义不同的类别。这一点笔者将在“归类整理下载的文件”一节中介绍具体的方法。
　　3.重命名下载文件：在网上下载的文件经常都是一些扩展名为zip的文件。为帮助我们记忆，你可以对下载的文件进行改名，在“重命名”文本框中进行更改即可。同时，你还可以在“注释”框中添加对该文件的注释，以后就可以根据这个注释进行查询。
　　4.设置代理服务器：许多用户可能要使用代理服务器来上网和下载，因此设置了很多的代理服务器。FlashGet可以设置通过代理服务器来下载，不过要在“代理服务器”窗口手工添加你使用的代理服务器地址，然后就可以选择使用代理服务器或是不使用代理服务器了。
　　三、归类整理下载的文件
　　下载文件的管理有时候是一件比较麻烦的事情，特别是像一些新手和网络“懒虫”们，下载的文件全部按照默认设置放在d:\Jetcar\download下，时间一长，看着这些名字相似的压缩包，自己也搞不清哪个文件是干什么用的。笔者以前用的是最笨的办法，即重新解压安装这些下载文件，实在是很麻烦。实际上，FlashGet最为重要和实用的功能之一就是对下载文件进行归类整理，每种类别可指定一个磁盘目录，所有指定存放到该类别的下载文件，下载完成后就会保存到对应的磁盘目录中。具体方法是：
　　1.创建不同的下载目录：可以在“E:\下载文件管理”下创建若干个不同的子目录，如“杀毒工具”、“工具软件”、“驱动程序”、“BIOS文件”、“图像浏览”、“网络工具”、“压缩文件”、“临时文件”等，当然你也可将它们放在其它路径下，但笔者建议不要与FlashGet的安装路径放在一起。
　　2.新建下载类别：进入FlashGet主界面窗口，选择菜单“类别”下的“新建类别”命令，建立新的类别，你需要做的只是在“类别名称”文本框中输入类别名，再在“默认的目录”文本框中输入该类别所在的磁盘目录，或通过浏览的方法查找到后，点击“确定”即可。
　　3.下载文件时选择相应类别存放：上述工作完成后，当你以后下载文件时，就可以在下载任务管理窗口的“类别”中选择相应的类别，FlashGet就会将下载文件帮你放在相应路径下，以后找起来就很方便。
　　四、定时下载
　　定时下载的功能对于网上飞虫和下载狂们非常实用。你可以在任务添加窗口中设置自动拨号来定时下载文件，这便于选择在深夜网络拥挤程度较低的时候通过无人值守的方式下载比较大的软件。你可以从“工具”菜单的“选项”命令中的“计划”选项卡进行设置（具体设置可根据各人的不同要求进行设置），不过这需要同时配合“拨号网络”选项卡里的一些设置才行，同时还需要将计算机一直开着，并且下载结束后还要能自动关机，这些设置都比较麻烦，因此笔者并不推荐使用。
　　五、限制下载速度
　　据测试FlashGet是目前国内下载工具中速度最快的软件，最新版本的最大进程可以达到10个，但如此高的多线程下载会占用几乎所有的网络带宽（建议选择默认的5个进程足矣），所以我们经常会发觉在使用FlashGet下载文件时浏览网页的速度非常慢，特别是使用OICQ时表现得尤为明显。因此，下载速度的限制应当是一个比较实用的功能，从菜单“工具”或者右击系统托盘中的图标弹出的快捷菜单中选择“速度限制”就可以了。
　　六、不用FlashGet而用浏览器下载
　　一般情况下，我们都是将“工具”菜单中的“选项”窗口中的“禁止监视浏览器点击”项不选中，这样在浏览网页点击超级链接来下载文件时，FlashGet会自动弹出，否则你就只能右击下载链接，从快捷菜单中选择“Download All By FlashGet”或“Download using FlashGet”了。但有时你可能想使用IE的下载功能，这时你可以右击系统托盘区的FlashGet图标选择“禁止监视浏览器点击”及取消“监视剪贴板”，这样就可以暂时关闭FlashGet了。
　　七、再次下载
　　有时，我们会发觉FlashGet在下载过程中会告诉你说“××文件下载失败”，这是最常见的情况。你可以试着换一个下载链接继续下载，也可以先点击工具栏上的“暂停”下载，稍后再重新点击工具栏上的“开始”按钮，一般多试几次就可以奏效了。
　　就笔者个人的使用感受来看，FlashGet可以当之无愧的说是目前最好的中文下载工具之一，和网络蚂蚁相比，FlashGet的下载速度和下载界面都要友好得多。假如你只想安装一种下载工具的话，那么FlashGet无疑是最佳选择。如果各位朋友对FlashGet的使用有什么问题，欢迎与我联系（wzj9999@yesky.com）。（江苏　王志军）}
