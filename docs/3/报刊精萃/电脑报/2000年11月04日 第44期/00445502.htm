《 OICQ泄密 》

{　　许多朋友是在网吧里上网的，我也常去。OICQ是我们经常使用的，网吧里共用OICQ也是很正常的事,由此安全就成为了一个很重要的问题。
　　从资源管理器入手，一切秘密都会在此暴露。在C:\Program Files\oicq下(在网吧里面多半是这个目录，即使不是看一下oicq快捷方式的“属性”也就能找到其安装目录)你可以看到许多账号。双击进入任意一个文件夹，可以看到.cfg的文件，那是账号的配置文件，其他有很多.msg文件。那些.msg文件的文件名就是他OICQ的朋友的OICQ号码。现在用记事本打开sysmsg.1st，里面记录的是某某账号在何日何时将本账号用户列为好友（当然是没被删除的）。再用记事本把friend.v30打开，里面有好友的登记资料以及用“收发信息”交谈的内容，还包括陌生人、黑名单方面的内容。再用记事本把bookmark.dat打开（如果有），看看也会吓死你。如果有tempfiles.tmp文件，通常是其他人用“语音传送”发来的声音文件，tempfiler.tmp通常是本账号用户发给其他人的声音文件。只要用录音机就可以打开播放。够了吧，还有什么可以保密。
　　这毕竟还是手工的，在网上很容易找到一个几百K的软件，轻松破解密码，看到他人的聊天记录，危害更大。
　　解决的办法：
　　简单也有效。在网吧下网后删除自己用的账号目录（不要只是删除到回收站，那跟没有删除一样）。下次使用时选注册向导，选“使用已有的OICQ号码”，再逐步注册就行。
　　（泥巴）
　　鸟语：本来刊发这一篇文章可能会造成一些恶意的破坏，到时候我就成了“千古罪人”。但我最近在网吧看了一下，一台网吧的普通机器上居然有几十个账号共用一个OICQ，我实在是有点为这些朋友担心。所以刊发这篇文章希望大家能提高警觉。如果你说“我没有什么怕泄露的”，那就当我没说。　　}
