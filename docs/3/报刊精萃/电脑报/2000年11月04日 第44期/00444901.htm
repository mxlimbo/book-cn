《 用Dreamweaver3.0定制链接样式 》

{　　1.链接文字的样式
　　选择“text”菜单中的“CSS Styless/ Edit style Sheet”，在弹出对话框中选择“New”，接着选择“Use CSS Selector”，在“Selector”下拉列表框中选择“a:link”，接着在弹出对话框中的“Decoration”栏中钩选“none”项，在“Color”项中选择黑色（＃000000）。这样链接文字显示为无下划线的黑色文字。
　　2.当鼠标移到链接文字上时的样式
　　选择“text”菜单中的“CSS Styless/ Edit style Sheet”，在弹出对话框中选择“New”，接着选择“Use CSS Selector”，在“Selector”下拉列表框中选择“a∶hover”， 接着在弹出对话框中的“Decoration”栏中钩选“underline”项，在“Color”项中选择蓝色（＃0066ff）。然后点中“Extensions”，在“Cursor”下拉列表中选择“hand”。这样当你把鼠标移到链接文字上，就可以看到鼠标变成手形，链接文字变成蓝色并有下划线。
　　3.访问过的链接文字样式
　　选择“text”菜单中的“CSS Styless/ Edit style Sheet”，在弹出对话框中选择“New”，接着选择“Use CSS Selector”，在“Selector”下拉列表框中选择“a∶visited”， “Decoration”栏中钩选中“underline”项，在“Color”项中选择红色（＃ff3300）。那么你访问过的链接就变成有下划线的红色文字。(广西　黄胜锦)}
