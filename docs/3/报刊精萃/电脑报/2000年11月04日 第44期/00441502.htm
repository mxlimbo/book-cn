《 {让Windows Me与Windows 98共存} 》

{　　1．硬盘准备
　　将硬盘分成三个区：C盘（至少1GB）用来安装第一套系统Windows Me；D盘（至少1GB）用来安装第二套系统Windows 98；E盘安装所有软件（例如笔者是在第二系统安装游戏和所有的应用软件。而第一套系统只安装了一些常用的网络专用软件，目的在于专用第一系统上网, 当然你可随意地按照自己的需要来设计多系统的软件配置）。
　　2．安装Windows Me系统
　　按正常方法安装第一套系统，并安装所需的应用软件和网络软件。
　　3．安装Windows 98系统
　　在DOS下安装第二套系统，安装第二套系统时要备份C盘已有的msdos.sys和io.sys，然后把安装盘符选择为D:\，然后再安装Windows 98系统和应用软件。
　　4．创建选择启动系统的文件
　　当电脑中有两套独立的系统时，将由C盘根目录下msdos.sys文件里的WINDIR和WINBOOTDIR两个参数来决定启动哪个系统，而开机画面是在io.sys文件里。先在Windows 98中把msdos.sys和io.sys两个文件的只读和隐藏属性去掉，在E盘建一个存放切换系统启动文件的文件夹（也可以在C盘或D盘）2WIN（名字随意），将msdos.sys和io.sys复制到此文件夹中改名为ms98.sys和io98.sys。在此文件中建一个批处理文件（名字随意）Win98.BAT，用记事本编辑Win98.BAT内容为：
　　copy ms98.sys c:\msdos.sys／y
　　copy io98.sys c:\io.sys／y
　　再把备份的Windows Me的两个系统文件复制到2WIN文件夹中再将两面个系统文件去掉只读和隐藏属性并改名为msMe.sys和ioMe.sys，同样再创建一个批处理文件WinMe.BAT，并编辑其内容为：
　　copy msMe.sys c:\msdos.sys／y
　　copy ioMe.sys c:\io.sys
　　如有必要可在Windows Me桌面上建立Win98.BAT的快捷方式，在Windows 98的桌面上建立WinMe.BAT的快捷方式，以后要进入某套系统，只要DOS或Windows下运行相应的WinMe.BAT或Win98.BAT后重新启动即可进入相应的系统。
　　5．修改Windows Me的环境变量
　　Windows Me只要重新起动就会在C盘根目录下产生一个自动批处理文件autoexec.bat,这样会让Windows 98“找不着北”，因为它里面所指向的文件Windows 98不可用。在Windows Me里单击“开始→程序→附件→系统工具→系统信息”，选择“工具”菜单中的“系统配置实用程序→常规→选起选择性启动”，然后选择“环境”一项，把里面所有的选项前面的钩去掉，再单击“新建”，在变量名里填上“COMSPEC”，不填变量值，确定后重新启动。
　　（编者注：Windows Me刚一出道，就引来“百家争鸣”。要是你拿不准的话，就安装双系统吧，慢慢地体会这两种系统的优与劣。我们也在寻求从Windows 98到Windows Me的过渡方案，要是你有好的想法，请来信与我们联系。）
　　（湖北　黄哲）}
