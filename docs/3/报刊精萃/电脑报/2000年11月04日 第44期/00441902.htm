{克隆与刻录――CloneCD

　　世界上自从有了第一只克隆羊“多利”之后便又多了争论，很让不少人伤了脑筋。CloneCD也是这样一个让某些人颇为头痛的软件。针对目前光盘加密的方式，它程序可以通过特殊的刻录方式将具有防拷的光碟进行复制。不仅程式版本更新速度快，注册方式也换得快，堪称目前最具话题的烧录软体。其“Raw Mode”功能(原盘完全拷贝模式)能够完全地复制一些经过处理的“防拷保护”光碟。当然也能复制其它任何种类的CD。不过其试用版有以下限制：1.程序内部设定不会储存。2.刻录时程序会自动限制烧录机在最低倍速运行。目前版本号为2.8.1.2，大小在1.73 MB左右，下载网址：http://www.t－online.de/home/CloneCD/download/CloneCDen.exe

其它

　　CuteFTP是不少老资格电脑玩家最常使用的FTP利器，也可称得上是FTP软件中的元老级代表。其版本多次更新，在新版本中加入了目录比较、远程文件编辑、目录上传下载和剪贴板监视等更加完善的功能，版本号为4.2 Beta Build 10.26，网址为：ftp://ftp.cuteftp.com/pub/cuteftp/cute4232bc.exe。
此外，快猫加鞭(Speedcat)的作者左轻侯又发表了自己的另一款作品iCompanion(网络伴侣)。该软件作为一款计费软件具有拨号管理、计时计费、 完全支持多账号，支持分段计费，支持多种折扣方式，符合电信最新计费标准，还能当天、当月全部计费即时汇总，极大地方便了你的网上旅程。更新的2.1版修正了一些小BUG,程序设计更完善，极力推荐大家试用！！下载http://qinghou.myetang.com/soft/ic21.exe，大小为423K。

下载工具火拼――Net Vampire网际快车

　　作为一个老牌的断点续传软件，Net Vampire（网络吸血鬼）除了下载速度快之外的最大特点是不易出现文件破损的情况。最新的4.0（http://www.netvampire.com/ftp/nv4pro_b.zip）版其内置的ZIP文件管理器可以实现对ZIP文件的校验，修复因网络不稳定造成的下载文件破坏，让你在疯狂下载时充分保证文件的完整性。
　　当然如果真的对英文不熟悉也没关系。国内著名的网际快车（FlashGet）完全可以满足你的需要。它支持多语言界面，支持多线程、断点续传等常见功能。在管理下载文件上是更胜一筹，其注释功能可让你保留更多与下载文件相关的信息。最新的版本0.91版（http://www.amazesoft.com/fgf091.zip）加强了“全部使用网际快车”下载URL选择功能，加强了成批导入文件管理的功能；还可按字母顺序显示子类别服务器。它的功能强大全面，并不亚于Net Vampire，更适合国人的使用习惯，定会让你爱不释手。极力推荐下载！

语言之王“巴比伦”――Babylon

　　如今的翻译软件如过江之鲫，种类繁多。从大家最熟悉的金山词霸到东方快车，然而Babylon更受到我们的关注。它是一款多语言支持的在线翻译系统。可支持诸如中文、西班牙文、日文、德文、法文、意大利文、葡萄牙文、荷兰文、希伯来文等等。其巨大的词库放在www.babylon.com的服务器里，节约了我们的硬盘空间，它拥有三百多万单词，可将英语单词翻译成其它11种语言。还可以进一步了解英语单词和短语的确切含义和具体用法。至于详细的使用在《电脑报》39期上已有详细讲解。最近更新的3.1b Build 12版（http://babylon.com/files/babylon31.exe，大小为1469KB）运行速度更快，界面更加清晰、简洁。建议上网条件比较好或者需要经常需要在网上查看外文资料的用户下载使用。}
