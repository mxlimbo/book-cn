《 IE的错误处理 》

{　　在浏览网页的时候，我们常常会遇到IE的错误提示，这些浏览网页的错误常常来自两个方面，一方面是线路、WWW服务器和网页本身的问题，大多数错误都是这方面问题引起的。另一方面是IE的设置或者某些功能没有安装。当然，也有系统故障和拨号网络的问题。在IE5.0中，浏览器遇到访问错误，将显示一个错误提示页面，并给出解决的建议。在IE4.0中，浏览器遇到访问错误，将显示一个错误代号，我们可以根据这些错误代号来判断错误原因。
　　一、 IE 5.0的错误提示
　　IE 5.0的错误提示很简单，常见的只有两种类型，当然，这两种类型归纳了各种错误情况，简单明了，使用者不至于糊涂。而且IE 5.0对错误的解决建议比较详细，操作也简单方便(如图)。
　　● HTTP 404——无法找到文件
　　此网页可能已经不存在或者已经被移到另外一个网址。
　　● 找不到服务器或发生DNS 错误
　　想连接的计算机可能没有上网，或者可能出故障，或者可能太忙碌。可以过一会再试连接。
　　二、 IE 4.0的错误提示
　　IE 4.0的错误提示非常多，错误提示一般是显示错误号，使用者根据错误号来解决网络浏览的问题。但由于错误号很多，使用者也很难搞清楚，所以很多人遇到常常无所适从。下面我们介绍一些典型的错误提示。
　　● 401 UNAUTHORIZED
　　提示访问网页必须有一个正确的用户名称及密码，一般在浏览一些收费的网页时会出现这个信息。
　　● 403 FORBIDDEN
　　该信息表示未在访问的网站注册。在一般情况下，可以在网上进行即时注册，但一些完全“封闭”的网站则会产生以上的信息。
　　● 404 NOT FOUND
　　这是最常见的错误信息，表示IE浏览器不能找到所要求的文件（即网页），该文件可能被移到别的地方去了，或根本就不存在。
　　● 500 SERVER ERROR
　　这个信息通常是对方网页程序设计错误而产生的，只有对方网页纠正错误才能正常访问。
　　● 503 SERVICE UNAVAILABLE
　　这个信息表示不能连接上对方网站，可能是因为网络线路繁忙。可以等一会儿再试。
　　● BAD FILE REQUEST
　　你可能在网页上填写某些表格的时候填入了不正确的资料，导致在处理资料的时候出现错误。处理方法就是按“BACK”键返回修改。
　　● FAILED DNS LOOKUP
　　用户所键入的URL位置不能被翻译成IP地址。这种错误通常是因为网站负荷过重，常发生在商业性网站。
　　● NOT FOUND
　　这个信息说明你想找的网页已不存在。可能是用户输入错误的URL，若确定输入正确，但仍然不能接上，则该网址也许已移到别的地方了。
　　● TOO MANY USER
　　这表示网站已经饱和，不能接受更多用户了，唯一的办法是过一会儿再尝试。
　　● UNABLE TO CREATE HOST
　　这表示你所键入的网址或其他URL不能找到所要的目标位置，你可能是打错了字，或者该网址并不存在，也可能是你的网路连接上出了问题。
　　注意，这些错误号在其他浏览器的错误提示中也可以看到。
　　三、 常用解决办法
　　一般情况下，IE浏览器遇到的访问错误都是外部的原因，如网站本身、WWW服务器问题、网络拥挤和堵塞，少数情况是电脑的软硬件设置的问题。可以按照下面的顺序来排查。
　　1． 如果可以访问其他网站，则说明本身电脑的软硬件设置没有问题，可能是访问的网站或者网页的问题。这个时候，可以刷新一下，看是否正常。
　　2． 如果多次刷新，还不正常，可以到上一级的目录去访问，看是否正常。如果正常，可能是该网页有问题(如不存在、正在更新、页面错误等)，也有可能是浏览器设置上的问题(如安全性比较高)。此外，还可以试着从上一级网页访问该网页。
　　3． 如果某个风站的任何页面和目录都打不开，则检查输入的域名是否正确。如果正确，则可能是该网站有问题，或者域名解析有问题。这个时候，只能另择时间访问。
　　4． 如果任何网站都不能访问，可以看其他程序能不能连上Internet，如ICQ、OICQ、Outlook Express等。如果能够连接上，一般是你的IE浏览器出现问题，这个时候，就要查看设置是否正确，程序是否被损坏（需要重新安装）。如果任何程序都连接不上Internet，那就是你的拨号网络或者ISP的问题了。}
