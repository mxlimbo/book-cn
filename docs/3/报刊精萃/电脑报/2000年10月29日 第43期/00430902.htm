《 {成长中的第三代用户界面――SUI} 》

{　　引文：在9月的苹果世界大展上，苹果公司展示了新一代操作系统MAC OS X的公开测试版，它提供了一个称为AQUA（水）的新式用户界面，预示了用户界面的进展。
　　用户界面（User Interface）是用户与计算机进行交互操作的方式，即用户与计算机通过一定方式互相传递信息，包括输入和输出，早期称为人机界面。
　　用户界面自从计算机诞生以来，至今已经出现了三种类型，按出现时间先后又可以划分为三代。最先广泛应用的是字符用户界面（Charateral User Interface），简称CUI；第二代用户界面也就是现在普遍应用的图形用户界面（Graphic User Interface），简称GUI。第三代用户界面则被称为社会用户界面（Social User Interface），简称SUI。相比于GUI，SUI有以下显著优点：
　　第一， SUI是全媒体用户界面。GUI广泛采用图标和图形作为表达媒介，偶尔采用声音和小型动画，至多可以称为多媒体界面；SUI则采用了一切可以表达信息的媒介物，文字、图标、图形、图像、动画、声音和视频，并能以三维方式显示。这样能够全面刺激用户感官，充分调动用户注意力和实现全方位互动，提高用户使用效率。
　　第二， SUI是动态仿真的用户界面。GUI通过简单桌面即一个几何平面来显示各种资源和操作对象，SUI把这个平面改进为一个实际生活和日常工作的三维模拟场景，更能接近用户心理；GUI通过图标加文字来表示各种对象，SUI则把各种对象都实物化，更加直观。
　　第三， SUI是具有智能引导功能的用户界面。在GUI中，几乎所有操作都得用户自己去摸索。SUI中专门设计了卡通人物或动物，在必要时它能通过说话和动作给用户提供帮助。
　　正是因为以上优点，SUI软件深受普通用户的欢迎，尤其是SOHO用户群。随着计算机硬件技术的发展，SUI极有可能融入未来的操作系统，成为基本用户界面，最终成为GUI的接班人。
　　SUI的缺点在于它还不能完全实现基于自然语言的人机对话。一旦实现这一技术，SUI将趋于完善，成为真正人性化的用户界面，促进计算机应用的“傻瓜化”。这将是用户界面的一次巨大革命。但正如GUI从出现到普及经历了二十年一样，实现GUI向SUI的过渡也需要一个时期。SUI技术本身也需要改进，还需要硬件技术协同发展。
　　SUI的发展是否意味着CUI和GUI的消亡呢？答案是否定的。上面说过，SUI是一个全媒体用户界面，这就表明CUI和GUI仍有使用价值，因为文字和图标也是表达信息的媒介物，全媒体当然包括它们，正如CUI也有图标操作、GUI并不排斥CUI一样。而且有些应用软件还必须使用CUI和GUI，例如字表处理软件。所以三者是和平共处的，只不过是个应用范围的问题，它们都是一种基本用户界面。
　　也许有人要问，SUI的发展方向又是什么呢？ 根据当前的科研成就和专家观点，最有可能就是虚拟现实（Virtual Reality）。(河北　王玉)}
