《 {借你一双火眼金睛——PC主要信息软件大观} 》

{　　购机难，难在不识货！
　　我们经常会听到一些朋友诉苦说在购机时如何上当受骗。采购电脑配件确实是件很棘手的事情，现在各种配件名目繁多，而且新品又不断推出，即使是老鸟也有看走眼的时候，更遑论新手菜鸟了。
　　但道高一尺，魔高一丈，我们并非没有办法对付。笔者今天就向各位朋友介绍几款小巧玲珑的工具软件，为大家装备一双善识真假的“火眼金睛”，让虚假配件无处遁形。
CPU篇
　　Intel Processor Frequency ID Utility　这是英特尔公司专门提供的检测CPU信息的工具软件，它可以使大家轻松识别手中的Intel处理器是否已超出额定的运行频率，同时它也可以识别系统中是否已安装英特尔处理器。目前该软件的最新版本是2.53版（可在http:∥www.intel.com.cn处下载），安装后执行FreqID.exe文件即可。程序界面看起来非常简洁，只有“频率测试”、“CPU ID数据”两大类：“频率测试”中显示了处理器和总线的报告频率和预期频率，假如这两者不一致，说明你的CPU肯定是被Remark过了；“CPU ID数据”则分为处理器分类（提供了CPU类型、CPU系列、CPU型号、CPU步进四项信息）、处理器细节（提供CPU的二级高速缓存、一级数据高速缓存、一级指令高速缓存和封装形式）和处理器特性（提供处理器数量、当前处理特性以及是否支持Intel MMX（TM）、Streaming SIMD(SSE)扩展指令集等CPU特性）三大部分。
　　AMD CPU Information Display Utility　AMD公司的这款CPU信息软件不仅能识别AMD的Athlon、Thunderbird、Duron系列CPU，而且还支持Intel处理器和VIA的Cyrix系列CPU。它主要显示CPU的类型和厂商名称、CPU的设定主频率、CPU内部一级数据缓存和指令缓存容量大小、CPU二级缓存容量大小、CPU系列、CPU型号、CPU步进、是否支持MMX和3D Now!技术。使用AMD系列CPU的朋友们可不要错过它哟（下载地址：http:∥www.amd.com）。
　　第三方的检测工具软件　笔者试用过多款检测CPU的相关软件，发觉还是WCPUID实用一些，大家在 http:∥www.h－oda.com下载该软件后直接执行即可。除提供芯片制造商信息、CPU型号和封装方式、CPU内频和外频信息、CPU缓存和多媒体指令集功能等信息外，WCPUID还特别提供了AGP插槽信息的检测，其中包括有关AGP插槽的版本以及传输速率（AGP2×或AGP4×等）、是否支持边带寻址和快速写入等最新功能告示；另外，它还能够识别出主板所采用的芯片组。当然除上述软件外，还有如CPU－Z（可以识别CPU是否被超频）、CPUCOOL、CPUSPEED等工具软件可供使用，大家可以从http:∥www.newhua.com找到它们并下载。
内存篇
　　目前内存的价格大跌，正是添加内存的好时机，但它也是各位朋友购机时最担心被Remark的配件之一了。为此，笔者给大家介绍大名鼎鼎的Sisoft Sandra Standard 2000。这是一个非常全面的硬件信息测试工具（这里笔者仅介绍与内存有关的部分），它总共提供了55个测试功能模块，操作界面不仅直观清晰而且操作简单方便，我们可以从http:∥www.sisoftware.co.uk下载它的最新试用版本（未注册版本有些功能无法使用，但不影响检测内存信息）。
　　双击主界面窗口中的“Mainboard Information”图标，便会出现如图所示的窗口，当前内存容量、内存的SPD信息、CL3和CL2信息都一览无余，够清楚的吧！当然，Sisoft Sandra Standard 2000的功能远远不止这些，它还能检测出最新的Windows Me操作系统版本呢。
光驱篇
　　CD Speed99一直是各大IT媒体评测光驱的必备软件，我们可以从http:∥www.cdspeed2000.com下载得到它的试用版本，主文件cdspeed99.exe大小仅394KB，下载解压后无须安装，直接运行即可。接下来你只要放入一张光盘，然后点击程序窗口右侧的“Start”按钮便可开始测试。该软件的测试项目有：速度测试（Speed）、DAE质量测试（DAE quality）、寻道时间测试（Seek times）、CPU占用率测试（CPU useage）、冲击传输测试（Burst）、电机起速和降速时间测试（Spin up/Spin down）。另外在菜单栏中的Extra里还有一项驱动检查（Drive check），运行后可以检查出关于驱动的很多信息，如操作系统、ASPI版本、主机适配器、驱动器支持的盘片种类及光驱所支持的命令等。
综合篇
　　笔者向朋友们推荐Dr Hardware 2000硬件信息检测软件（最新版本下载地址：http:∥www.drhardware.de），运行后点击窗口右下角的“Refresh”按钮就可以看到当前系统的硬件信息了，它是从BIOS、I/O端口直接读取硬件信息，可以说是非常权威的。该软件共有六大检测项目，下面笔者就一一介绍它们的功能：
　　Overview（常规信息部分）：这里主要显示系统信息和Windows信息等内容，也就是一些基本的信息。
　　Hradware（硬件检测部分）：这里主要有CPU、主板、内存（除显示内存容量、品牌外，还可以显示出额定工作频率是PC100还是PC133，甚至你可以看到它的Cycle和Access信息呢）、BIOS信息、芯片组、CMOS设置、I/O信息、PCI插槽、端口信息等。
　　Devices（磁盘检测部分）：这里主要有驱动器（这里包括Name、Type、Capacity、free、FAT、KB/Cluster、Details）、显示设备（显卡、显示器等）、ASPI设备、EIDE/ATAPI设备、Modem、Printer等。
　　Windows（操作系统部分）：从这里我们可以看到操作系统的版本和版本号、发布公司和浏览器版本等信息，还可以了解到配置情况。
　　Resources（系统资源部分）：主要内存资源、CPU资源、磁盘资源、系统监视、日记记载等。
　　Benchmarks（比较测试部分）：这里主要对CPU、内存、显卡、磁盘、CD－ROM等设备进行比较性测试，只要选定相应设备后点击窗口右下角的“Run benchmarks!”按钮就可以看到结果了，当然这些结果只能当作参考。
　　笔者文中所介绍的工具软件，除Sinsoft Sandara 2000和Dr Hardware 2000外，其它的你都可以将主文件装在一张软盘上随身携带，非常方便。假如各位对这些工具软件的使用有什么疑问的话，欢迎与我联系，笔者的E－mail是：wzj9999@yesky.com。最后衷心希望各位朋友都能买到货真价实的PC配件产品。(江苏　王志军)}
