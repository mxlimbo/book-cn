《 拥抱宽带网――ISDN 》

{　　ISDN(Integrated Services Digital Network，综合业务数字网)是以电话综合数字网为基础发展而成，它能提供端到端的数字连接，而以前只能提供端到端的模拟――数字――模拟连接；支持一系列广泛的话音和非话业务，而原来只能提供单一的话音业务或非话业务；提供一组有限的标准多用途用户/网络接口，而过去只能提供单一接口。ISDN理论上可以提供8个终端同时通信，但因为目前设备限制，所以暂时只能提供两个终端同时通信。
　　ISDN给用户带来的好处有：使用方便、费用低廉、高速传输、传输质量提高、网络互通性强等。
ISDN主要硬件设备介绍
　　ISDN从硬件分类上看有很多种，比如：NT1、NT1＋（NT1 PLUS）、ISDN TA、ISDN路由器、ISDN测试仪、ISDN内置PC卡、ISDN笔记本卡(PCMCIA卡)、ISDN数字电话、ISDN桌面视频会议、ISDN小型交换机（PBX）、服务器端ISDN PRI卡、服务器端ISDN BRI卡等。从低端到高端都覆盖到位，因此支持面非常广。
　　1.NT1和NT1＋
　　NT1和NT1＋(图1)是用户传输线路的终端装置，它是实现在普通电话线上进行数字信号传送和接收的关键设备，是电话局程控交换机和用户的终端设备之间的接口设备。它完成线路传输码型的转换，并实现回波抵消数字传输技术。它能以点对点的方式最多支持8个终端设备接入，可使多个ISDN用户终端设备合用一个D信道，并在用户终端和电话局交换机之间传递激活与去激活的控制信息。该设备完成维护功能，使电话局能通过该设备进行环路测试。NT1和NT1＋具有功率传递功能，能够从电话线路上吸取来自电话局的直流电能，以便在用户端发生停电时实现远端供电，保证终端设备的正常通信。
　　NT1和NT1＋唯一的区别就在于，NT1仅提供数字接口，也就是说用户侧的所有设备都必须是数字设备，原来的模拟电话、模拟传真、模拟Modem等都不能用。而NT1＋就能让原来这些模拟设备继续使用，这样可以减少投资。说得更简单些，你只要有了NT1＋，其他什么设备都可不买，照样能在基于ISDN的线路上通信。而NT1就不行，你必须有TA(Terminal Adapter，终端适配器)或者有数字电话、ISDN PC卡等数字设备。所以，用NT1＋肯定比用NT1省钱、方便。
　　NT1设备背面接口如图2所示。
　　NT1＋背面接口如图3所示。
　　比较图2和图3，我们可以清楚地看出NT1＋比NT1多了两个PHONE（电话）口，这两个电话口就是供接模拟电话用的，数字电话接在S/T口上。很多读者还弄不清楚什么是RJ11接口?什么是RJ45接口？上图中ISDN进线接口、PHONE1、PHONE2即是RJ11接口，S/T 1、S/T 2即是RJ45接口。大家仔细看一下实物，RJ11中有两根金属线，RJ45中有4根金属线，而且RJ45的接头比RJ11明显大一些。
　　在使用NT1＋的过程中通常会碰到因为NT1＋难于激活而无法上网和通话的问题。这是因为在不使用ISDN的时候，NT1＋通常处于休眠状态，使用时需要有个激活的过程。当碰到PC卡或TA难于激活NT1＋时，很简单，只要拿起电话，用电话激活NT1＋即可，这算是人工激活方式。如果连这种方式都不行的话，那只能重启NT1＋，操作方法如下：把ISDN进线和电源线从NT1＋上拔下，等待5秒钟后先把ISDN线插上NT1＋，等待NT1＋前面板上ST指示灯常亮。如果ST指示灯一直闪烁，则把ISDN线拔下再插上，重复几次，一直等PW和ST指示灯都常亮为止。然后再插上电源线即可，这时NT1＋肯定激活。
　　以前的NT1＋功能比较简单，没有来电显示功能，现在市场上已经可提供带来电显示功能的NT1＋，它把来电显示器集成在NT1＋中，这样你无须买数字电话就可享受此功能。但是遗憾的是，往往大家不愿把NT1＋像电话机一样放在比较明显的地方，所以这个功能有时候被忽视了。还提醒大家一点：在雷雨天气，建议把ISDN线路从NT1＋上拔下，因为雷击有可能损坏NT1＋或者引起火灾。
　　2.ISDN TA和ISDN内置PC卡
　　与Modem分为内置Modem和外置Modem一样，ISDN也有内置ISDN PC卡(图4)和外置ISDN TA(图5)的区别。ISDN内置PC卡和ISDN TA连接方式与普通Modem也差不多，插在主板插槽内或者通过RS232串口与电脑相连。
　　TA的功能就是使得现有的非ISDN标准终端（例如模拟话机、G3传真机、分设备、PC机）能够在ISDN上运行，为用户在现有终端上提供ISDN业务。
　　ISDN TA上有电源接口、S/T（RJ45）接口、模拟设备（RJ11）接口和RS232连接线接口，TA的接口和NT1＋的差不多。其实最早的时候NT1＋不太多，大多数用户都只能用NT1加外置TA的方式组网，但因为这种上网方式设备费用比较高，因此人们想到了把NT1提供S/T接口和TA提供模拟接口的功能集成在一个设备里，NT1＋就是这样一种设备。所以NT1＋是在NT1和TA基础上设计的，因此用户只需购买NT1＋和ISDN内置PC卡就可上网。NT1配TA售价要比NT1＋配ISDN内置PC卡高很多，很多地方电信部门免费提供NT1＋给用户使用，所以用户只要花三百多块钱购买ISDN内置PC卡即可上网。现在还有一种更好的TA设备，也是把NT1和TA设备的功能完全集成在一个设备中，称为U口TA，EICON的U口DIVA T/A ISDN Modem就是这样的一种设备。
　　ISDN其他硬件产品，如：ISDN笔记本卡(PCMCIA卡)、ISDN数字电话、ISDN桌面视频会议、ISDN小型交换机（PBX）、服务器端ISDN PRI卡等设备的介绍，你可以上网（www.bisdn.net）查询。
ISDN的安装和调试
　　鉴于目前多数用户采用NT1＋配ISDN内置PC卡上网，所以下面以台湾盛达产的BILLION ISDN内置PC卡为例详细介绍ISDN的安装和调试。
　　系统要求：CPU 486DX2－66以上、8M内存、硬盘空间5M、支持Win95/98/2000/ME/NT4.0。
　　1.准备工作
　　主要的准备工作是添加“TCP/IP通讯协议”及相应的网络组件并设置拨号网络组件，因为多数用户对此都比较熟悉，所以这里不再赘述。
　　2.安装BILLION ISDN PC卡
　　安装前确认PC处于关机状态（避免触电）。打开机箱盖，把BILLION ISDN PC卡插入一空闲PCI插槽，用RJ45连接线（NT1设备随机提供给你的连接线）将卡与NT1＋设备连接起来。开机，进入操作系统，操作系统将会报告：“已找到新设备，正在为其安装驱动程序”。把随卡附带的驱动盘插入计算机中，然后选择从厂商提供的驱动盘上安装驱动程序。此时系统会显示安装的进度，用不了多少时间就能安装好。最后系统提示要重新启动计算机后才能生效，选择重新启动计算机即可。
　　3.新建ISDN拨号网络
　　Windows重新启动后，如果没有报告错误，则ISDN PC卡安装成功；若有错误，可在控制面板的“系统”中找找原因。如果是I/O地址冲突的话，手工调整即可解决。不过在日常安装过程中也发现有时就是装不上，后来换一条PCI插槽即可解决问题，所以实在找不到原因，你也可以尝试换一条PCI插槽。如果ISDN PC卡已成功安装，接下来就要新建ISDN拨号网络。下面以新建一个“Shanghai Online”的连接为例介绍其具体操作方法。双击桌面上的“我的电脑”→“拨号网络（Dial－up Networking）”→“新建连接(Make New Connection)”建立新连接，出现图6所示对话框。
　　
　　在最上面编辑框中为要拨号的计算机，键入一个名字(可以是任意的名字，这里取为“Shanghai Online”)，在Select device（选择设备）框中，可以选Billion ISDN Line0，也可以选Billion ISDN Line1，分别代表ISDN的两个B信道。再单击“下一步”，出现图7所示的界面。
　　在区号框中，键入要拨入地区的区号(例如上海为21)；在电话号码框中，键入要拨的电话号码（例如上海为8888）；在国家/地区代码框中，选择要拨国家代码（86代表中国）。然后单击“下一步”，弹出的界面如图8所示。
　　单击“完成”，这样就建立了一个连接。现在进入“拨号网络”，可以看到一个新的图标(图9)。用右键单击它，并选择属性(Properties)，弹出图10所示的界面。
　　在Use area code Dialing Properties复选框处，若为上海本地用户，则可以将其设为不选中，转至“服务器类型”，在对话框(图11)箭头的复选框处，一般用户将其设为不选中，然后单击“确定”，至此，ISDN拨号网络就成功建立。
　　4.拨号过程
　　ISDN的拨号过程和Modem的拨号过程基本相同，不再赘述。只不过ISDN与Modem拨号稍有不同，比如Modem从拨号到上网这个过程比较长，一般需要15～30秒时间，而ISDN只要3～4秒即可。Modem拨号过程会发出“猫”叫声，而ISDN拨号却没有声音。
ISDN路由器
　　ISDN路由器是局域网通信的必备设备，一般用于大、中、小型企业。有些有经验的读者会说，我在服务器上安装代理服务器软件，通过TA也能使局域网上网和通信，不需要什么路由器。但是就像《电脑报》第42期61版《网关服务器VS ISDN路由器》一文中提到的那样，从价格、安装、使用维护和速度、稳定性、安全性能方面，路由器都胜过代理服务器。所以我们建议使用ISDN路由器，特别是在稳定性、安全性、维护使用上有比较高要求的大中型企业尤其应该这样。
　　这里以EICON的DIVA LAN ISDN Modem这款ISDN路由器为例，连接方案如图12所示。
　　从图中可以看到局域网中的计算机都连接到路由器的相应端口上，如果计算机很多，可以通过HUB级连后再接到路由器上，最多可以连50台计算机。两个模拟端口可以接电话机或者传真机等模拟设备。路由器的设置方式比较简单，一般花上15～20分钟就能搞定。EICON的这款设备通过图形化界面很方便地就可完成设置。路由器的具体方式请参见《电脑报》第43期61版《ISDN路由器的设置》一文。
RVS－COM使用手册
　　一般用户在使用ISDN时会碰到一些以前使用Modem没有的难题，比如：无法使用来电显示功能、无法通过电脑收发传真、无法通过电脑拨打和接听电话、无法与其他计算机连接等问题。为此，原德国RVS有限公司推出了一款非常实用的ISDN专用软件：RVS－COM。这款软件都是随ISDN硬件赠送给用户，在国内无销售。所以，在购买ISDN硬件时，提醒你最好买有赠送RVS－COM软件的产品，这样，你就可以充分发挥ISDN的优势了。
　　RVS－COM的功能很多，如：发送和接收传真、计算机之间的文档传递、使用联机服务器、借助RVS－COM和其他的应用程序使用ISDN－CAPI适配器等。

　　将随机赠送的RVS－COM盘插入驱动器，启动RVS－COM中的SETUP安装程序，屏幕上显示如图13所示。

　　4.发送传真
　　发送传真很简单，打开“RVS通信中心”，点击“新建传真文”，进入发送传真文件的设置界面(图20)，在姓名、号码栏中输入对方的信息，然后点击“下一步”，在弹出的对话框中填上传真主题和传真正文内容，然后点击“下一步”，在接下来的对话框中设置传真的紧急程度，再点击“下一步”，设置传真精度等内容，即可完成设置。发送传真过程即可开始，传真发送后，你可以到RVS收件箱查看发送的状态，一般会显示发送成功或者失败的信息。
ISDN常见问题
　　1.ISDN用户终端设备分为哪几种？
　　答：ISDN用户终端设备主要有网络终端NT（Network Terminal）和终端适配器TA（Terminal Adapter）以及终端设备（Terminal Equipment）等。NT又分为NT1和NT1＋。NT1是早期产品，只提供S/T数字接口接入标准的ISDN终端设备（如数字电话等），现趋于淘汰。NT＋（也称NT1 plus）是目前最常见的，它是NT1的改进型，除具有NT1的所有功能外，还添加了两个模拟端口可接拨普通电话、G3传真机等，从而大大节省了用户改造设备的费用。TA又称为ISDN Modem，它分为内置TA和外置TA两种。内置TA也叫ISDN PC卡，它接在PC机的PCI插槽上，主要功能是配合NT将PC接入网络。外置TA是目前较好的ISDN终端设备，设置简单功能齐全，有的还内建Flash存储器支持软件升级，缺点是价格高，两个信道全开时难以达到128K。
　　2.为什么有的ISP能用ISDN拨号上网，而有的却不行呢？
　　答：有些ISP提供ISDN端口拨号上网，而有些ISP不提供ISDN端口拨号上网。上海的ISP，如上海热线、81890、8163现在都提供ISDN拨入，而有些小型ISP由于种种原因没有提供ISDN拨号方式，因此在选择上网ISP时应注意这一点。
　　3.我能否在一台PC机上装多块ISDN PC卡？
　　答：这好像不行，笔者专门对此问题做了试验，试验结果是一块PC卡能正常工作，而另外一块怎么也拨不上号，并且还会发生其他怪问题。如果真要把多条ISDN线路接在一块卡上，可以用BRI卡。
　　4.我是笔记本电脑用户，哪种ISDN设备是我需要的？
　　答：对于笔记本电脑用户有专门的ISDN设备，叫PCMCIA卡，它类似于笔记本电脑上使用的网卡。比较好的PCMCIA卡如EICON的PCMCIA卡不仅可以提供ISDN连接，而且提供Modem连接。这样使经常在外地出差人员可以随当地接入方式的不同而选择接口，而不需要额外的其他设备。
　　5.来电显示、多用户号码功能是否要付费？
　　答：来电显示和ISDN多用户号码功能按月要收取一定费用，有些地方电信部门提供某些方式的优惠措施，减免费用，不过按照固定是要收费的，费用一般不超过10元/月。
　　6.通过ISDN与PC相连有几种方式？我听说串口有115K的上限速度（小于ISDN的2B的峰值速度），其他呢？
　　答：ISDN与PC主要通过插槽或者串/并口连接，目前卡类设备一般都通过插槽连接，外置的以串口连接居多。串口连接方式因为串接线本身的速率限制，所以只能达到115K，这一点没错，对用户来说是有些损失。因此建议你最好使用PC卡或者并口外置设备，解除串接线的瓶颈。
　　7.有没有设备可以同时连接两条以上的ISDN线路？
　　答：有，EICON公司生产的DIVA Server 4BRI CARD最多能接4条BRI ISDN线路，而他们的DIVA Server PRI 30M可接入PRI线路，特别适用于大型企业和ISP企业、金融企业、电子商务网站等。
　　8.为什么我在用ISDN上网的时候，它的速度很慢（与56K的猫差不多，甚至还慢）？
　　答：ISDN一般正常情况下，1B信道的下载速率是6K～7K，2B的下载速率是12K～14K左右。在日常上网过程中我发现，某些国外的网站和某些服务器硬件较差的网站本身就很难连，就算用ADSL也很难连。有些平常很好连接的网站也发现访问较慢的现象，主要是因为访问该网站的人数过多，造成无法满足所有用户的请求。还有因为ISP上网人数太多也会造成速度很慢的现象。因此碰到这种情况，你可以换其他网站访问，或者更换ISP上网可以基本解决此问题，与ISDN接入方式没什么关系。
　　9.为什么在上网过程中常常掉线？
　　答：虽然ISDN使用的是普通电话线，但在传送过程中采用全数字信号，误码和信号失真少，所以掉线一般不是线路问题，主要和计算机的系统资源有关，注意在上网时不要同时运行大程序（要不，加大内存也行）。
　　10.使用2B信道捆绑拨号，为什么有时候拨通后只显示速率为64K？
　　答：可多试几次，一般拨通后，速率不会立即达到128K，需要等若干秒，速率自动升为128K。如果确实迟迟不能升为128K，多为线路问题，用户可向电信部门反映。
　　11.听说ISDN能有144K的速率，可为什么广告中却说最高可以达到128K的速率呢？
　　答：我们所说的ISDN是有144K的速率，分别为两个64K的B信道和一个16K的D信道。B信道是用户传送数据信息的，而D信道是传送和交换机联系的信令信息的。所以我们通常说ISDN的上网速率最高可以达到128K。
　　12.怎样从NT1＋上判断ISDN线路是否连接上了？
　　答：很简单，一般每个NT1＋都有LP、PW、ST、B1、B2等指示灯，无通信情况下，LP、PW灯亮，有信号进入或出去时ST和B1/B2灯亮。测试方法是拿起电话，只要看到PW、ST和B1/B2灯亮即可。
　　13.如何在线增加一个B信道，或者减少一个B信道？
　　答：双击状态栏的上网图标→单击“Details >>”，会出现此连接的详细情况，并有一个按钮“Resume”，单击它，会重新来一次128K连接的尝试。当此按钮为“Suspend”时，单击它会挂断一条64K的信道，只使用一条。
　　14.我用的是苹果机，有没有适合我的ISDN设备？
　　答：BILLION的迷你USB TA128就是一款支持苹果机的ISDN设备。它支持Mac OS8.6及以上版本，并可以在Windows与Mac下用G3收发传真。
