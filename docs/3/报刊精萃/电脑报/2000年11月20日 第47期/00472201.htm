《 {善用Fireworks的“效率工具”} 》

{　　在Fireworks中，有两组可以令工作效率倍增的工具，我们称之为“效率工具”，它就是查找、替换、批处理及脚本工具。
　　查找和替换可以方便迅速的帮你替换图像中的文字、样式及图像的样式特效等；批处理工具可以自动完成对一组图像的特定格式的处理，还可以将这种特定的批处理保存起来变成脚本工具，以便以后处理同样的工作。
　　如果你对此它们感到困惑，请看下文，我们将详细的介绍一下Fireworks特有的“效率工具”。
一、查找和替换
　　网络图像在网页设计中占着非常重要的地位，对它的要求已越来越高了。我们常碰到这样的情形，在做好一批Web图像后，又发觉还需要对这些图像中的文字、文字格式、图像颜色及样式、图像的URL链接等等进行一一修改，工作简单却重复。这时，利用Fireworks的查找、替换效率工具就可以轻松的一次帮你解决。
　　我们可以通过如下两种方式打开查找和替换功能：
　　1.选取Edit\Find and Replace命令。
　　2.选取Windows\Find and Replace命令。
　　这时，出现查找和替换的浮动工具条，如图1所示。主要功能分为Search Document（搜索文档）和Find Text（查找文本）。
　　在Search Document栏下还包括：
　　Search Selection，对选中的文档进行查找和替换。
　　Search Frame，对文档中的当前帧进行查找和替换。
　　Search Document，对当前的文档进行查找和替换。
　　Search Project Log，对Project Log（工程日志）中存在的文档进行查找和替换。
　　Search File对指定的文件进行查找和替换，可以是多个文件。
　　而Find Text又分为：
　　Find Text，对当前的文本内容进行查找和替换。
　　Find Font，对当前文本的字体效果进行查找和替换。
　　Find Color，对当前文档中的颜色效果进行查找和替换。
　　Find URL，对当前文档中指定的链接网址进行查找和替换。
　　Find Non－web216，将当前文档中存在的非Web安全色进行查找和替换为最接近的Web安全色。
　　3.查找和替换文本
　　Find Text（查找和替换文本）的功能实用而又简单，操作界面请见图1，只需指定要Find（寻找）的文本和Change To（替换成）的文本即可。另外需要注意的是浮动工具条下侧的Whole Word、Match Case和Regular Expressions对查找和替换的限制条件，含义分别为：
　　Whole Word（符合整个字），选中该项，表示查找和替换的内容以整个单词为准，不能用一个单词查找和替换一个单词的一部分。例如，将or替换成and，而不会将word中的or替换成and。
　　Match Case（符合大小写），选中该项，表示查找和替换的内容严格区分大小写，主要针对字母。
　　Regular Expressions（正规表示法），选中该项，表示查找和替换的内容可以遵循类似通配符的表示法，有？、＊等通配符。
　　这样就可以轻松对文本内容进行查找和替换了。
　　4.查找和替换字体
　　在寻找和替换的浮动工具条中点击Find Text，从下拉菜单中选择Find Font，就会出现如图2的对话框。主要分为查找和替换两部分的参数设置，在上半部分可以选择要查找的字体和字型，以及在多少字号的区间进行查找；下半部分是可以选择的要替换成的字体和字型，以及字号的大小。
　　参数设定好以后，可以单击Replace进行逐个替换，也可单击Replace All进行全部替换。
　　5.查找和替换颜色内容
　　在寻找和替换的浮动工具条中点击Find Text，从下拉菜单中选择Find Color，就会出现如图3所示的对话框。只需轻松地在颜色井中选择要查找和替换的颜色，就可以实现查找和替换。
　　需要特别说明的是，Fireworks除了简单的查找和替换颜色外，还可以对文档中对象的属性进行操作，主要包括Fills（填充）、Strokes（笔画）、Effects（特效）等。如选择了Fills，则只对文档中的填充色进行查找和替换。
　　6.查找和替换URL
　　利用Fireworks的查找和替换URL功能，可以查找和替换文档中对热区和切片指定网址，其操作方法同查找和替换文本。这里不做重复介绍。
　　7.查找和替换非Web安全色
　　在文档中存在非Web安全色，则在不同的浏览器中会出现不同的颜色效果，导致色差。（所谓Web安全色，指的是在Microsoft IE和Netscape Navigator这两种浏览器中都能正确识别的颜色，简单说来，一共有216种颜色。）
　　在寻找和替换的浮动工具条中点击Find Text，从下拉菜单中选择Find Non－web216，就会出现如图4所示的对话框。通过选择，可以对填充、笔画、特效的颜色查找和替换成最接近的web216安全色，使图像颜色在不同的浏览器中有共同的效果。但是需要注意的是，替换之后可能会出现意料不到的效果。
二、批处理及脚本工具
　　当你需要对一组或多个文档进行相同的处理时，Fireworks 提供的批处理功能，就可以方便的实现重复操作，提高工作效率。而且，Fireworks还允许你将这种批处理保存下来，留待以后使用，批处理保存后，就成了脚本工具。下面就分别介绍一下它们的操作。
　　从Fireworks菜单File中选择Batch Process（批处理），就会出现如图5所示的对话框。其操作的过程程为通过Current Open Files选定要进行批处理的文件，然后设置要批处理的动作，包括进行查找替换、输出格式，然后确定即可自动执行批处理操作。
　　其中的查找替换同于上文所讲的有关文本的查找替换，可以参考上文。当选中Export（输出），点击后面对应的箭头时，会出现如图6所示的对话框。通过｀这个对话显示，可以设置你想最终处理的结果，包括对文件格式、文件名称前缀后缀、缩放方式及大小的设定，操作界面一目了然。
　　当你在进行批处理时，在批处理的设置完成后，可以通过点击批处理对话显示中的Script按钮，将当前的设定保存起来，以便以后使用，它的文件格式为.jsf。
　　小技巧：将生成的批处理文件复制到Fierworks安装目录的\Settings\Command下，就可以在菜单Command下看到这个批处理，此时，它变成了脚本工具。
　　如果要运行已保存的批处理，则打开菜单File，选择Run Script（运行脚本），就会出现已保存好的.jsf文件，也即批处理文件，选中后确定，即可运行。
　　灵活运用Fierworks的查找替换和批处理工具，可以减少你对众多图像文档的重复操作，极大提高提高工作效率。如果你有什么疑问，可以与我联系，共同探讨。我的联系信箱是yuyu@163.net 。
