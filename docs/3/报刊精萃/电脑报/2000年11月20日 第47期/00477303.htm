{华彩软件举行Tanmino产品研讨会
　　日前，华彩软件与德商赛克科技股份有限公司隆重推出全球第一套 XML Information Server － Tamino。
　　Tamino是世界第一套以标准的XML格式进行资料存取的信息服务器，提供资料交换和应用程序集成环境，同时又是支持WEB的完整资料管理系统。它也是莫扎特的著名歌剧《魔笛》中英雄王子的名字，Tamino王子凭借一支神奇魔笛，历尽艰辛后与美丽的Pamina公主终成佳偶。
　　德商赛克大陆地区总代理华彩软件将于11月30日在北京嘉里中心举办“跨平台电子商务沟通新脉动――XML技术及Tamino产品研讨会”，欢迎业界人士与技术爱好者莅临参加。
方正电脑卓越Z系列激情面世
　　11月18日，方正科技电脑公司正式发布卓越Z系列家用电脑。方正电脑卓越Z系列不仅具有强大的硬件配置和丰富的应用软件，而且第一次实现了家用电脑Internet接入与Internet服务的“无缝”整合，家用电脑与掌上电脑的一体化产品设计，以及家用电脑的Internet智能家居管理功能。在方正电脑卓越Z系列的帮助下，用户可以获得半年免费上网和半年免费远程教育，在家用电脑和掌上电脑之间方便快捷地实现功能联通和信息传递，利用家用电脑对数字消费产品(例如数码相机、数字摄像机、MP3)和传统家电产品(例如电视机、空调机)进行远程智能管理。
　　戴尔公布第三季度业绩
　　戴尔高级副总兼亚太区总裁宋德智先生近日在北京公布了戴尔第三季度业绩。报告显示，公司在全球的出货量较去年同期上升21％，领先于业界增长率。继续成为全球第二大计算机系统供应商。亚太区(包括日本)的季度收益由于大型企业方面的销售强势而增长39％。至于在中国，销售额增长超过了70％。
　　戴尔在中国还致力于为客户提供卓越的应用解决方案。1999年9月，戴尔在上海设立了亚太区首家应用解决方案中心，帮助中国客户成功转型到下一代戴尔服务器、工作站及英特尔构架的产品。从启用以来，中心已协助多家跨国机构、本地企业和独立软件开发商在戴尔系统上进行概念验证、性能方面的调试、应用程序的移植和基准测试。}
