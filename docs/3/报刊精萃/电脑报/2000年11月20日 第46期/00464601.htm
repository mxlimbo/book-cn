《 我的名字 我的邮箱 》

{　　NamePlanet公司提供一种新的免费电子邮箱，你可以用自己的名字去申请，获得像YourFirstName@YourLastName.com这样的电子邮箱，如：Jason@Wellings.com，或者xiao@yang.ac。这样的E－mail非常容易让人记住，因为它和你的名字是一样的，简单明了。
　　下面简单介绍一下怎样申请这个E－mail以及申请中的一些技巧和要注意的问题。
　　1.首先，登录http:∥www.nameplanet.com网站(如图)。在First name和Last name中输入你的名和姓的拼音，或者你想注册的其他名字，然后按下Check It(检查)，看看这个名字是否可以使用(有没有被其他人先注册了或者系统不支持这个名字)。如果检查通过，那么，你就可以获得以你的名字命名的电子邮箱了。接下来，在(A)选项中按下Click to register selected address(注册选择的邮件地址)。
　　2.名字起好了，剩下的事情就是填表了，和申请其他的免费E－mail没有什么太大的区别，此处不再赘述。由于它的界面全是英文的，对于英语不太好的人，借助东方快车、金山词霸等汉化工具，就可以轻松搞定。需要注意的是，用黑体字表示的栏目一定要填写，其他的可以不去管。如果漏填了，系统会用红色的字来提示你重新填写。填好后，你的邮箱就立即生效，可以马上使用。
　　3.如果不巧，你申请的名字正好不被系统所支持或者已经被其他人先用了，系统会提示你申请的名字不能使用，给出一些其他相近名字的组合，你可以从中挑选一个满意的，再按下(B)选项的Click to register customized address(注册用户定义地址)继续第2步。
　　4.如果想申请到以com结尾的邮箱，往往需要以英文的名字注册，如：Tom、Wellings等等，否则，往往只能得到以ac、as、cc、to、nu等等这样结尾的邮箱。
　　NamePlanet提供的邮箱不支持POP3方式，只支持Web方式，这是一个遗憾。但是，你可以把它设置为转信信箱，就是说所有发到这个邮箱的电子邮件都会被自动转移到你的另外一个支持POP3的邮箱中去。在缺省情况下，转信功能是被关闭的。如果你要使用这个功能，首先要在Nameplanet的主页上，用你的名字和密码登录到你的邮箱中，在屏幕的左边选择My Settings(我的设置)，然后再选择Configure forwarding(设置转信)，输入要转到的地址，按下Add(添加)即可。你可以设置多个转信信箱，但是其中只有一个是活动的。
　　NamePlanet邮箱支持邮件过滤功能，在My Settings的Block senders中，输入那些你不愿收到的E－mail地址，就可以拒收他们的信件了。
　　另外，NamePlanet公司和Maunt公司联合还推出了一个域名服务，只要你申请了FirstName@LastName.com这样的邮箱，就可以免费获得http:∥www.FirstName.LastName.com这样的域名。如果要使用这个域名服务功能，和上面一样，首先要登录到你的邮箱中，选择Personal Web Address(个人网站地址)，依次输入你已经有的网站的地址、网站名称、说明和关键字，最后按下Update(更新)，它就可以自动帮你将网站登记到几个大的搜索引擎中去。使用http:∥www.FirstName.LastName.com这样的域名很有特色，别人看过以后很难忘记。不过它会自动地在屏幕上方给你的网站加上一个广告条，真是有利也有弊，鱼和熊掌不可兼得也。
　　如果有问题，欢迎给我来信(shao.zi@yang.ac)。
(SZY)}
