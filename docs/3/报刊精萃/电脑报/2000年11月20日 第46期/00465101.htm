《 IE的高级功能 》

{　　1.使用JAVA脚本
　　虽然JAVA脚本的应用可以为聊天室增添许多色彩，但是一些人却使用它在聊天室里踢人，为了防止这种情况的发生，可以做如下设置。
　　执行主菜单“工具／Internet选项”命令，在出现的对话窗口中，选中“安全”标签页面，接下来，选中“Internet”图标，单击“自定义级别”，打开“安全设置”窗口，找到“脚本”，将“Java小程序脚本”和“活动脚本”两项全都设置成“提示”或“禁用”，这样就不必担心有人在聊天室里踢人了。
　　2.使用搜索助手
　　在网上冲浪，经常要用到如Sohu、Sina、Yahoo等搜索引擎，但使用它们必须首先进入这些引擎的站点，在IE5中提供了可以不进入搜索引擎站点就进行搜索工作的搜索助手，使用方法如下。
　　单击工具栏上的“搜索”按钮，这时浏览器会分成左右两个窗口，左边是搜索栏，在这里可以使用IE的excite搜索引擎来查找信息，右边的部分就是网页了。这时，只要在搜索框中输入查询关键词(如图1)，单击“OK”按钮就可以了，然后单击搜索栏显示的结果，右边便会显示链接到相应的网页上。如果需要开启一个新的搜索窗口，可以单击搜索栏左上角的“新建”按钮。 使用搜索助手，最简单的方法就是，直接在地址栏中输入要查询的关键字，然后回车就可以了。
　　3．离线浏览网页
　　IE 5.0提供了离线浏览的功能，可以使我们在离线状态下浏览Internet页面。
　　首先打开要脱机浏览的网页，然后执行主菜单“收藏／添加到收藏夹”命令，打开“添加到收藏夹”对话框，选中“允许脱机使用”复选框，然后单击“自定义”按钮，启动“脱机收藏夹向导”，单击“下一步”按钮，会提示“要收藏夹的该页包含其它链接，是否要使链接的网页也可以脱机使用？”，选择“是”，指定“下载与该页链接的网页层数”后单击“下一步”按钮，当询问如何同步该页时，选择一种后单击“下一步”按钮，系统会提示“是否有密码”，根据自己需要选择是否设定密码后，单击“完成”按钮。
　　将网页添加到收藏夹后，就可以进行脱机浏览了，执行“收藏／整理收藏夹”命令，进入到“整理收藏夹”对话框(如图2)，单击选中要脱机浏览的网页，再选中左边“允许脱机使用”复选框，然后单击“属性”按钮，选择“Web文档”标签页选中“允许脱机使用”复选框，单击“下载”标签页框选择下载的层数，完成后，单击“确定”按钮返回。
　　连接Internet以后，执行主菜单“工具／同步”命令(如图3)，单击“同步”命令按钮就可以开始下载脱机浏览的网页了。
　　4．用IE听广播
　　IE 5.0提供了网上收听广播功能，可以使我们在线收听一些著名电台的广播内容。
　　执行主菜单“查看／工具栏／电台”命令，打开IE的电台工具栏，输入相关的网络广播站点网址后，就可以利用电台网络提供的工具进行收听了。不过网上收听广播需要安装有Windows Media player或Realplayer等媒体播放软件。
　　5．关闭Cookie
　　Cookie可以记录用户上网的浏览信息，并保存在本地硬盘上，当用户再次访问相同网站时，系统就可以有针对性地进行服务，这虽然方便了用户的使用，但也带来了一定危害，如果将Cookie功能关闭，就可以避免这些不必要的麻烦。
　　执行主菜单“工具／Internet选项”命令，选中“安全”标签页，在“请为不同区域的Web内容指定安全设置”列表框中选择“Internet”选项图标，单击“自定义级别”按钮，打开“安全设置”窗口，找到Cookies，将“允许使用存储在您计算机上的Cookie”和“允许使用每个对话的Cookie”两个选项设定为“禁用”，然后单击“确定”按钮返回。同样道理，在“安全”标签页将其他几个选项使用Cookie功能禁用就可以了。如果嫌以上方法过于麻烦，可以将Windows文件夹下的Cookie子文件夹设定为只读属性，也同样可以关闭Cookie功能。
　　6．自动查看邮件
　　IE浏览器除了能浏览信息外，一个很重要的功能就是直接利用集成在浏览器中的电子邮件软件来收发信件。一般来说，如果我们想在Outlook Express中查看邮件时，每次都要单击“接受和发送”按钮很麻烦。其实还有一种很快捷的方法来查看是否有新邮件，那就是直接在IE 5.0浏览器的Outlook Express中自己设定下载邮件的时间间隔，具体操作步骤如下：
　　首先，打开Outlook Express邮件操作窗口，用鼠标单击菜单栏中的“工具”菜单项，接着再从弹出的下拉菜单中单击“选项”命令，随后程序将会弹出一个标题为“选项”的对话框，在该对话框中单击“常规”标签，然后在其中的“发送和接受邮件”设置栏下，选中“每隔X分钟检查一次邮箱”，在X的位置填上数字，最后再单击一下“确定”按钮，此后用户的邮箱就将在设定时间检查一次是否有新邮件。
　　7．用浏览器来运行程序
　　当用户在浏览网上信息时，由于特殊原因需要及时运行系统中的另外一个程序来处理突发任务。我们可以直接使用浏览器把需要的程序打开。先在IE地址栏中直接输入运行程序在硬盘中的绝对路径，例如我们在地址栏中输入“c:\windows\write.exe”，输入完毕后再单击一下回车键，接着浏览器就会在浏览窗口中自动打开指定程序的界面，在这里我们将会看到一个打开的“写字板”程序界面。
　　8．IE升级
　　IE的升级可以到国内各大下载站点或者微软的官方站点下载最新版本，也可以在各类出售的光盘中获得。在IE5安装盘中执行ie5setup.exe程序，按照屏幕提示去做就可以完成IE的升级工作。另外还可以到http:／／www.mypatch.net下载Internet Explorer 5.5 在线安装中文升级版，大小497K。
　　9．IE补丁
　　IE的功能虽然很强大，但是也少不了一些系统方面的漏洞，这样会给一些居心叵测的人一个可乘之机，为了确保我们的系统安全不受侵犯，可以到http:／／www.microsoft.com去查看和下载微软发布的最新补丁程序。
　　10．卸载 IE
　　关闭所有应用程序，打开“控制面板”，双击“添加／删除程序”图标，在“安装／卸载”标签项中，单击“Microsoft Internet Explorer 5 和 Internet工具”，然后单击“添加／删除”按钮。选中“恢复以前的Windows配置”单选框，单击“高级”按钮，在“恢复以前安装的以及下面所选的Windows组件”中，列出要恢复的组件，选择要卸载的组件，单击“确定”按钮，系统开始卸载IE5，完成后重新启动Windows。须要注意的是，在卸载时不可直接删除IE目录及文件，否则会造成不能正常恢复以前的Windows配置。
（李军　周勇生）}
