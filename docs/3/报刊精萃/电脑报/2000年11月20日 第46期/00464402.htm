《 离线浏览工具Leech 》

{　　Leech是一款相当不错的离线浏览工具，它能够稳定地运行在Win 95/98/NT/2000系统之上，具有多线程及断点续传的下载功能，并且它还具有强大的文件过滤功能，能够限制下载文件的类型、文件的大小、文件所在的路径以及文件的层数。
　　一、软件的下载与安装
　　Leech是一款共享软件，需要有IE4.0或4.0以上版本的支持，当前版本v3.0.4，文件大小1884KB，未注册的版本可以免费试用10天，下载地址：http:∥www.aeria.com。下载的是个zip压缩包，解压缩后，运行其中的setup.exe文件便可进入安装。安装完毕后，在开始菜单的程序中会出现“Leech”的快捷项，点击该快捷项便可启动Leech，该程序的界面和IE浏览器的界面差不多。
　　二、添加下载站点并进行相关的设置
　　在Leech开始工作之前，我们需要把准备下载的网站添加进站点列表。点击菜单栏“Site”下拉菜单中的“New”项，弹出“Site Properties”对话框。该对话框中共有五个标签页面，在这里我们可以进行站点的添加和设置。
　　1.“Site”标签页面：在“Address”项后的正文框中，我们可以把需要下载的网站地址添加进去；在“Name”项后的正文框中，我们可以给这个网站取个名字。另外，如果你添加的网站在访问时需要用户提供用户名和密码，请选中“Use authentication”项，并在相应的正文框中填入用户名和密码。
　　2.“Options”标签页面：如果你选中“Erase existing content before leeching”项，当Leech在进行新的下载任务之前就会清除下载文件夹中原有的文件；在“Default document”项后的正文框中你可以填入浏览下载网站时的起始页面，如：index.htm。另外，你在这里还可以设置Leech连接服务器的超时时间、服务器回应的超时时间以及出错后重试的次数。
　　3.“Limits”标签页面：如果你想限制Leech下载文件所占用的磁盘空间，请选中“Restrict Disk Space Usage”项，并在“Maximum”项中填入合适的数字，单位为MB；如果你想对Leech下载文件的时间进行限制，请选中“Restrict Time Interval”项，并在“Maximum Time”项中填入合适的数字，单位为分钟。
　　4.“Link Filters”标签页面：如果你希望限制对站点列表中的某些站点进行下载，请选中“Restrict Server”项，并通过“Add”按钮把需要限制的站点加入该项下的站点列表框中；如果你想限制对站点中某个目录下文件的下载，请选中“Restrict Directory”项，并在正文框中填入其详细的路径（这是Leech的特色功能，因为过去大多数离线浏览器在为你下载网站时，不管你需不需要，它们会把所有允许下载层数内的东西都给你抓下来）。另外，如果你希望限制下载网站中网页的层数，请选中“Restrict Level”项，并在“Maximum Level”项后填上合适的数字。
　　5.“File Filters”标签页面：在该页面中，你可以设置Leech下载网站时过滤的文件类型和文件大小。如果你想限制下载的文件类型，请首先选中“Restrict file types”项。当你选中该项下面的“Include file types”项时，Leech只会为你下载列表框中出现的文件类型，你可以通过“Add”按钮添加你需要下载的文件类型；而当你选中“Exclude file types”项时，Leech会拒绝下载列表框中出现的文件类型。另外，如果你想限制下载文件的大小，请选中“Restrict file size”项，并进行相应的设置，可设置最大值和最小值，单位为KB。
　　此外，在这里顺便提一下Leech的参数设置。点击菜单栏“File”下拉菜单中的“Preferances”项，弹出参数设置对话框。在该对话框中主要可以设置下载文件的存放路径和下载的线程（该功能试用版本不能使用）；另外，如果你使用的是代理服务器，也可以在这里做相关的设置(该项功能试用版本同样不能使用）。
　　最后需要提请大家注意的是，当你把需要下载的网站添加进站点列表并设置完毕之后，一定不要忘了保存你的设置，否则，当你下次启动软件时，你会发现站点列表框中什么都没有了。
　　三、软件的使用
　　1.下载指定的网站：首先选中主窗口中站点列表框中需要下载的网站，如果你想全部选中所有的网站，按下快捷键“Ctrl＋A”即可，然后，点击菜单栏“Site”下拉菜单中的“Leech”项。如果你这时候已经拨号上网，Leech就会开始为你进行下载工作了。如果你在下载过程中想暂停下载进程，按下“Pause”按钮即可。这时“Pause”按钮会变为“Resume”按钮，想要重新开始下载按下该按钮即可。
　　2.离线浏览已下载的网站：选中站点列表框中准备浏览的站点，点击菜单栏“Site”下拉菜单中的“View”的子项“Leeched Content”，你就可以慢慢地欣赏Leech为你抓下的网站了。
　　四、不足之处
　　Leech的主要不足表现在断点续传功能上。虽然在下载过程如果发生断线的情况，它能够从上次任务中断处为你继续下载，但如果你在下载任务没有完成时退出了该软件，当你重新进行下载时它就不能够断点续传了，而只能从头开始下载。另外，笔者在试用时发现当Leech无法正常下载时并不能给出相应的提示信息（此情况发生在刚开始下载时），而仍旧显示为正常下载状态。因此，如果用户发现下载进程条一直停在“0”处不动时，必须重启软件后再次执行下载命令才能开始正常下载。}
