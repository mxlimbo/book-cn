《 {Windows Me小技巧两则} 》

{　　一、自制显示文本
　　在Windows Me中，我们经常见到当鼠标指向一个文件或文件的快捷方式时，会显示出一个简单的文本信息。其实，我们是可以对这些信息自行设置的。操作方法是：右击这些文件或文件的快捷方式，在弹出的菜单里点选“属性”命令，即可见到一个对话框。这时我们只须将“备注”栏里的文本进行修改或添加，点击“确定”退出即可。
　　二、设置文件夹查看方式
　　Windows Me可以记录每个文件夹的查看方式。但在我们需要对所有的文件夹进行同样的查看方式，或要恢复文件夹的默认查看方式的时候，就可以同样在“文件夹选项”对话框里进行设置，只不过这次是选取“查看”选项卡。点选了“查看”选项卡后，我们见到“与当前文件夹类似”和“重置所有文件夹”两个按钮。前者是将所有的文件夹的查看方式都设置成与当前打开的文件夹的查看方式一样或类似；而后者则是恢复所有文件夹的默认查看方式。
　　（广东　廖年旺）}
