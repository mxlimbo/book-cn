《 {硬盘初始化的好帮手――MaxBlast} 》

{　　对硬盘进行分区和格式化几乎是每一位PC用户，特别是DIY爱好者要遇到的事情。随着硬盘容量的迅速增大，这项工作已变得既耗时又枯燥，而且许多人或许早已厌倦了Fdisk和Format、Partition magic等软件的操作界面，想换换“口味”，那么MaxBlast可以实现我们大多数人的愿望，它能在极短的时间内高效而又安全地完成硬盘的分区和初始化过程。
　　MaxBlast是迈拓(Maxtor)公司向所有Maxtor用户推荐的一款免费的IDE接口硬盘的实用工具，它支持EIDE和Fast ATA硬盘接口标准，支持多驱动器、多分区系统以及Windows 95B/98的32位文件分配表。目前MaxBlast的最新版本号为9.20,支持的硬盘最大容量为32GB。不过，相信很快在下一个版本中就会突破这个限制。下面就让我们来看看它的“神奇”之处：
　　一、功能简介
　　1.对硬盘进行快速分区和格式化：MaxBlast省去了用Fdisk和Format操作时繁琐、耗时的缺点，只须按提示进行简单操作，在1分钟之内即可完成对硬盘的分区和格式化。
　　2.对硬盘进行诊断和测试：MaxBlast能侦测出硬盘表面的各种故障(如坏的扇区、坏的磁道)、分区表被损坏以及硬盘不能被操作系统正确识别等问题。当检测出问题以后，它会报出一个错误代码，Maxtor用户可以藉此拨打迈拓公司的全球技术支援电话，迈拓公司的技术服务部可以根据用户提供的错误代码进行远程技术支援服务。
　　3.其他辅助功能：包括对硬盘的零磁道进行备份和恢复、将一个分区的数据精确复制到另一个分区等等。另外，该软件的帮助文件中还收集了市面上许多常见硬盘的跳线说明，非常实用。
　　二、如何对硬盘进行快速分区和格式化
　　1.在DOS或Windows下运行Maxblast.exe，程序将提示你插入一张软盘（干净空白软盘），按提示操作，程序将会生成一张专用启动工具盘。关闭计算机，用刚才的软盘重新启动计算机，它将自动引导机器进入Maxblast界面。程序首先对你的系统进行检测并提示你将检测到的硬件信息记录下来(主要包括硬盘的各种重要参数以及CPU信息等),接着是教你如何在CMOS中设置硬盘的提示信息，以及使用本软件限制的声明信息，你可按回车键跳过它们，然后进入如图1所示的主操作界面：
　　2.在图1左上方的“Main Menu”框中，自上而下分别为硬盘设置（Setup Hard Drive,即对硬盘进行分区和格式化）、高级选项（Advanced Option）、查看帮助文件（View Help File）、开始测试（Start Maxdiag）、退出Maxblast（Exit Maxblast）。右边为当前菜单提示信息框，正下方Identified Drives框列出了已被Maxblast检测出的IDE驱动器（包括硬盘、光驱等）。
　　3.选择“Setup Hard Drive”后回车进入图2所示的画面，这时系统会列出当前硬盘上的所有目录，并提示如果继续进行下一步操作，将会删除掉当前硬盘上的所有资料。如果确定要继续，可在对话框中输入“Yes”并回车确认，这时系统又会弹出一个“Copy system files”提示框，提示你插入一DOS或Windows95/98启动盘（注意：如果要对硬盘进行32位分区，必须插入Windows95B或者是Windows98启动盘），插入后按回车，系统在对软盘上的系统文件检测后会询问你是进行32位分区还是16位分区。
　　4.如果选择进行32位分区,系统提示将按照已列出的分区信息进行分区(图3,系统默认将整个硬盘作为一个分区)。另外如果要将硬盘分成两个区,必须选择“Enter new partition Sizes”(输入新的分区尺寸)。在对话框中输入第一个分区所需尺寸(1400)后按回车确认，程序计算后默认将剩余的硬盘容量作为下一个分区的尺寸。
　　5.按回车确认，系统再次提示你将按重新设定后的尺寸对硬盘进行分区，选择“Use this partition size”，按回车确认。这时系统最后一次警告你即将对硬盘进行分区，如果决定不对硬盘进行分区，请选择“Cancel Setup”；若选择“Continue Setup”并回车确认，系统将对硬盘进行分区、格式化和拷贝系统文件，如果你的硬盘不太大的话，这一过程大约需要30～40秒。
　　6.取出软盘，按Esc键重新启动计算机。
　　怎么样？是不是比Fdisk和Format快多了。事实上，Maxblast其他的几项功能也很不错，有兴趣的读者可以自己慢慢去挖掘。大家可以到迈拓公司的网站http:∥www.maxtor.com下载这个软件，大小为558KB。不过，该软件目前只适用于Matrox硬盘，这不能不说是一个遗憾。
