《 {RealPlayer操作指南} 》

{　　现在，有不少的网站提供RealMedia格式的电影和音乐下载，也有很多RealMedia(后缀为RM、RAM等)格式的电视节目和电影压制成CD－ROM光碟。一般一张CD－ROM光碟可以存储400分钟的电视节目或者3～4部电影。当然，这些电视节目或者电影的分辨率大都在300×200左右，画面稍显模糊，比VCD差，比电视的效果稍差，但一张CD－ROM就可以存放7～8集的电视剧节目，还是受到很多朋友的欢迎。
　　RealMedia格式的影像文件一般都用RealPlayer播放。它支持播放各种RealMedia文件，包括RealMedia格式的声音和RealMedia格式的影像。它最重要的功能就是能通过Internet播放网络上的音乐和收听网上广播，是网络音乐和网络广播的标准工具。当然，在国内网络带宽还不能让所有的人享受网络音乐和网络广播的情况下，RealPlayer一般就作为一个播放器用来播放本地的RealMedia文件。这里，我们就介绍如何使用RealPlayer来播放RealMedia格式的电影和电视节目。
　　1.RealPlayer的版本
　　RealPlayer的版本比较多，一般经常用的是7.0和8.0，其中还有不同的版本，如RealPlayer7.0 G2就是一个比较经典的版本。而RealPlayer 8.0的版本又有Basic Minimum（简化）、Basic Standard（标准）、Basic Complete（完全）三个版本。其中Basic Complete版本的下载文件大小为11MB，Basic Minimum版本的下载文件大小约为4.3MB。一般国内各大下载站点都有下载。也可以访问RealPlayer的官方站点http:∥realguide.real.com/。下面我们就RealPlayer 8.0 Basic Minimum版本来介绍RealPlayer使用中的一些问题。
　　2.注册
　　RealPlayer的安装比较简单，但安装完成后，要弹出一个注册窗口，注册的过程比较复杂。如果你一般不用RealPlayer来收听网上音乐，可以选择“Cancel”取消。不过，以后每次启动RealPlayer，程序都会要求你注册。RealPlayer安装完成后，将在开始菜单、任务栏（只针对Win98/Me/2000）和桌面添加快捷启动方式。另外，每次启动系统，RealPlayer还将启动RealPlayer的控制程序在后台运行（可在任务栏的时间状态栏中看到运行图标）。
　　3.界面
　　相比Winamp、媒体播放器，RealPlayer的界面比较复杂（图1），但实际上很多都是和网络播放有关的，如果本地播放，很多界面元素都可以取消。
　　取消频道栏：频道栏和IE的频道类似，通过频道栏可以收听网上广播。但如果RealPlayer仅仅用来播放本地的影音，则完全可以取消。取消的方法是选择菜单“View/Content Panel”即可。
　　取消Real.com Media Bar工具条：Real.com Media Bar工具条是用来快捷访问Real.com网站资源的，在RealPlayer界面的下部，一般情况下，都可以取消。方法是选择菜单“View/Real.com Media Bar”即可。
　　取消状态栏和地址栏：状态栏主要显示播放文件的一些信息，地址栏和IE中的地址栏相似。选择菜单“View/Status Bar”可以取消状态栏，选择菜单“View/Location Bar”可以取消地址栏。
　　精简窗口模式：RealPlayer也定义了一个精简的界面窗口模式，这个模式只需要点击工具栏最后一个按钮“Campact Mode”即可出现，也可以使用快捷键“Ctrl＋M”在精简模式和普通模式间切换。图3就是最精简的RealPlayer窗口模式。只有菜单和播放控制工具条。
　　4.播放本地文件
　　使用RealPlayer播放本地的RealMedia文件很简单，这些文件已经和RealPlayer建立了关联，直接点击即可启动RealPlayer播放。
　　另外，可以启动RealPlayer程序，然后选择“File/Open File”菜单，选择要播放的文件即可。
　　RealPlayer支持的文件格式很多，包括图像（GIF、JPEG、PNG）、Flash动画（SWF）、声音（MP3及其列表文件）、文本（TXT）以及各种RealMedia格式文件。比较遗憾的是，RealPlayer不支持RealMedia格式文件的列表播放，不过，可以参考《电脑报》45期23版的相关文章解决。
　　5.播放控制
　　RealPlayer相对Winamp、媒体播放器等其他播放工具，播放控制比较简单。主要的进度控制按钮在菜单下面，只有播放、暂停、停止、快进/慢进五个按钮，另外还有一个进度条，拖动进度条可以设置播放的位置。
　　设置声音均衡：选择播放工具条的倒数第四个按钮“Equalizer”，或者选择菜单“View/Equalizer”，弹出均衡调节窗口，不过这个只有高、中、低音的3段调节，如果需要更细致地调节声音的高中低音比例，可以点击旁边的“Upgrade now”，下载有10段均衡的插件（图3）。另外，该插件还有其他一些声音调节功能。
　　调节画面大小：调节画面大小比较简单，直接拉动窗口大小就可以了。也可以使用RealPlayer的“Zoom Level”按钮（在界面的左下角），点击该按钮后，有三个选项：原始大小（Original Size）、放大一倍（Double Size）、全屏幕（Full Screen）。
　　调节画面质量：在调节画面大小的“Zoom Level”按钮的旁边，就是调节画面质量的“Video Control”按钮。点击该按钮弹出调节窗口（图4），该窗口只有两项调节：颜色“Color”和画面锐化（Sharpness）。另外，也可以去下载有5项调节功能的插件。
　　可视化画面：RealPlayer也和Winamp一样支持可视化效果。在播放音乐的时候，点击“Visualizations”按钮（在调节画面质量的“Sharpness”按钮旁），就可得到各种效果画面。另外，在界面的右下角，可以调节声音的大小或者静音。
　　6.控制中心
　　当RealPlayer安装好后，默认是启动系统时自动启动RealPlayer的控制中心，该控制中心能够让你更好、更快捷地播放RealMedia音乐。但如果我们不是经常播放，也可以将它关掉，方法是用鼠标右键选择控制中心的图标，在弹出的右键菜单中选择“Close StartCenter”。另外，如果要让系统在启动时不启动RealPlayer的控制中心，可以选择“View/Preserences”，在弹出的设置窗口中选择“General”标签，在“StartCenter”栏中选择“Settint”按钮，然后确认不选择“Enable StartCenter”项即可。
(楠　竹)}
