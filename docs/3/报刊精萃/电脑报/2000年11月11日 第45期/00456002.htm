《 {Windows CE下的音乐软件} 》

{　　Windows Media Player
　　微软终于推出了WMP的Windows CE版本了！此软件不但可以播放MP3，而且还可以播放WMA音乐！WMA的音质好、个头小，绝对适合Windows CE的用户！
　　系统：Windows CE 2.0以上
　　硬件：任何CPU
　　版权：免费软件
　　大小：500KB
　　下载地址：http:／／www.microsoft.com／windows／windowsmedia／en／download／default.asp
　　PalmPlayer
　　MP3的出现，风靡了所有的音乐爱好者。现在你可以不用买MP3随身听，也可以享受到MP3音乐，赶快安装PalmPlayer吧！最好就是在Palm OS的使用者面前展示一下，看看他们流口水的样子吧！
　　系统：Windows CE 2.0以上
　　硬件：任何CPU
　　版权：共享软件（14.95美元）
　　大小：616KB
　　下载地址：http:／／www.conduits.com／ce／player／
　　XAudio MP3 Player
　　用PDA来听MP3总有一种痛快和很超值的感觉！此软件和上面的差不多，还是免费的喔！
　　系统：Windows CE 2.0以上
　　硬件：任何CPU
　　版权：免费软件
　　大小：782KB
　　下载地址：http:／／www.mpegtv.com／
　　PocketTV
　　有没有想过用Pocket PC来看VCD呢？有了PocketTV，这个便不是梦想了！PocketTV可以播放＊.mpg格式的文件。
　　系统：Windows CE 2.0以上
　　硬件：任何CPU
　　版权：免费软件
　　需求：彩色
　　大小：592KB
　　下载地址：http:／／www.mpegtv.com／
　　WavePSGDriver
这个WavePSGDriver可以让你的Pocket PC播放MIDI文件呢！MIDI文件的体积很小，最适合爱听音乐的你！但此软件只支持MIDI Format 0格式，使用前请用Cakewalk等软件进行转换（Cakewalk的详细介绍请参看本报第30、31期23版文章）。
　　系统：Windows CE 2.0以上
　　硬件：任何CPU
　　版权：免费软件
　　大小：14KB
　　下载地址：http:／／www.lares.dti.ne.jp／～k－akixx／
　　Pocket MusicBox
　　这个Pocket MusicBox可不像普通的音乐盒那么简单，应该叫Pocket PC上的“跳舞机”才对。画面和音乐也做得很好！千万不要用脚来玩，用你的Stylus吧。下载了这个软件，你将可收录许多喜爱的歌曲，随时播放！
　　它的特色有：
　　1.有高品质的MP3数字音效
　　2.可以自由选择音乐游戏内容
　　3.明亮的软件桌面背景以及流畅的播放速度
　　系统：Windows CE 2.0以上
　　硬件：MIPS CPU
　　版权：共享软件（9.95美元）
　　大小：4.04MB
　　需求：彩色
　　下载地址：http:／／www.flux2game.com／
　　（广东 LinYuanShan）}
