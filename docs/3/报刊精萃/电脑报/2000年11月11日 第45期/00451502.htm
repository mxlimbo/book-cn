《 {快捷启动多个程序} 》

{　　我们经常要同时启动多个应用程序，如在启动“记事本”后（运行“Notepad.exe”），又想听一听MP3歌曲（运行“Winamp.exe”）。通常是要分别运行这两个软件，能不能一“键”双雕呢？
　　其实很简单：首先在桌面上新建一个文本文件，打开记事本输入以下命令：
　　start C:\Windows\ Notepad.exe
　　start C:\Progra～1\ Winamp.exe
　　保存后，把文件的扩展名改为.bat的批处理文件，（注意：由于批处理命令是在DOS方式下运行，所以不支持超过8个字符的长文件名，所以要将文件夹“Program Files”改为“Progra～1”），然后直接运行这个批处理文件就可以了。当然，你还要读读朋友的来信，再用“start”命令加载你的电子邮件软件就可以了。　　（浙江　石磊）
编者注：这可方便了我们众编辑，不必拖着鼠标一阵乱点了。只需轻轻一点，常见的程序便一股脑儿地开始运行了。}
