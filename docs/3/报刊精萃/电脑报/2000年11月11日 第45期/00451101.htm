《 {Windows也要换新衣} 》

{　　朋友，用了几年Windows了？有没有看它不顺眼？想不想打扮打扮它?
　　下面我要跟你说的就是给Windows换衣服的“换肤大师”WindowBlinds V1.22 Enhanced（由Stardock公司出品）。
　　为什么敢封它为大师呢？你瞧一瞧它的本事就知道了。这个安装完不到10M的软件，可以完全改变Windows的桌面环境，从“开始”菜单到全部Windows兼容程序的界面，来了个彻底改头换面。在它的主界面里，供你选择的界面类型有：BePC（BeOS）,WarpPC（OS/2 Warp）,MacPC（Mac OS）,NeXTPC（NeXTSTEP）这几种不同类型的操作系统界面；还有一些自创的界面，例如BrokenWindows（一种带有创伤般的Windows界面,很有趣），Sigma Shadows、Sphereworld等等；甚至还有一款称为Win3000的界面，绝对令人耳目一新。这个评估版本提供了13种使用界面，如果你仍旧不满意的话，还可以到Stardock公司的站点下载你喜欢的类型。这可是它的一大特色呦。据称该网站有超过750种不同的Skin!
　　软件使用：
　　该软件只需一路回车就可轻松安装完毕。打开该软件后,你会看到该软件简洁的主界面(图1)。上方是公司的标志,公司标志的下方有四个选单的标签,左下方是Skin的选择框和几个用于定制Skin种类的按钮,而右方是几个复选框,主要用于微调每款Skin的作用范围。那么,怎样给Windows换肤呢?1.在图1所示的界面选择框中单击左键，选定你喜欢的界面。2.点击选择框下面的Apply Now按钮。好啦，就这么简单，看一看是不是够酷？图2就是采用BeOS后的窗口界面，还是3D的哟。
　　右下方的Preview the skin按钮，使用它就可以轻松预览各式Skin的特色了。
　　软件设置：
　　我们再来看它的细微设置。在第一张选单中（也就是Persenality选单）有如下设置：左下方的三个按钮Install a new skin、Delete skin和Refresh list分别用于给skin选择框中填加新skin，删除旧有的skin和刷新选择框。右方的复选框分别是“应用于多选按钮和单选按钮”（Apply skin to checkbox ＆ radio）,“应用于标准按钮”（Apply skin to standard buttons）,“应用于任务条按钮”（Apply skin to taskbar buttons）,“应用于菜单边界”（Apply skin to menu borders）,“应用于工具条按钮”（Apply skin to toolbar buttons）,“应用skin的色彩方案”（Apply skin color scheme）。下面的几个是“使用3D工具条取代平面工具条”（Replace 3D toolbars with flat one）,“使桌面图标及文本的背景透明化”（Make desktop icon text background transparent）。再下面是两个按钮“改变UIS1（旧式skin）的设定”（Change UIS1（old skin）setting...）和“预览选定的skin”（Preview the skin...）。最后右下角还有一个帮助按钮。
　　在第二张选单中（也就是Background Bitmaps选单），见图3。该选单用于更换窗体的背景图。左侧是选择背景图的下拉框，并且下方有一个按钮用于更换下拉框中默认的位图文件（Change Bitmaps...）。右侧也是几个复选框：“应用于对话框及多窗体（MDI）界面的背景”（Add bitmaps to dialog ＆ MDI background）,“应用于Windows的主窗体”（Add bitmaps to explore window）,“对Windows的主窗体使用自定义颜色”（Use a custom colour for explore window）,下面的按钮是“更换Windows的主窗体的字体颜色”（explore text colour）,最后是“使用系统指定的窗体的背景图”（Use bitmap backgrounds specified by system）。
　　在第三张选单中（也就是Per Application Settings），见图4。这些选项均是针对每个应用程序进行设置的。上下两个窗分别是“对单个应用程序使用skin／去除对某单个应用程序的skin应用”（Set a Per App Skin/Remove Per App Setting）和“除去某应用程序外，其他应用/去除skin”（Add Application Exclusion/Remove Exclusion）。
　　最后一张选单是Startup Setting＆News，也就是说“启动设置与更新”，见图5。顶端的窗是显示更新信息的，其下的三个按钮分别用于连接Stardock公司的网站和阅读该软件的发放信息。下面还有三个复选框：“当WindowBlinds启动时显示该对话框”（Show this dialog when WindowBlinds starts）,“将WindowBlinds图标置于系统托盘以便于使用”（Place WB icon in system tray for easy access），和“启动时自动运行WindowBlinds”（Run WindowBlinds automatically on system start）。最后右下方的按钮用于配置Internet选项（Configure Internet Option）。
　　最后再透露一个秘密：还记得吗，上面提到过，界面和背景都是位图（BMP）文件，也就是说，如果我们自己找到或者制作一些位图来替换原有的位图的话……那么就是自己独一无二的一套skin了。真正的DIY哟！
　　说了半天，难道它就没有不足吗？所谓“金无足赤”，WindowBlinds也有不足的。其一，该软件占用系统资源稍多，所以不建议配置较低的机器使用。其二，该评估版本有使用次数的限制，并非免费软件。其三，该软件无法替换Windows的图标，因此如果能配合Microangelo98之类的图标替换工具，效果则更佳（见《电脑报》2000年第10期上的介绍）。
　　但是，毕竟“暇不掩玉”，WindowBlinds的确是一款相当优秀的“换肤”软件，能让你的老Windows焕发青春！心动的话，请到www.stardock.com或www.windowblinds.net下载，文件大小只有1.4MB。
(北京　马强)}
