《 {美丽的延续（上）——Photoshop 6.0精彩新功能详解} 》

{　　一、Photoshop界面的变化
　　乍一看Photoshop 6.0在界面上和以前的版本基本没有大的变动，只是在菜单条下多了一个可移动的“Options Bar”(选项条),用来替代了原有Options Palette选项浮动控制面板。但选项条比原来的选项控制面板增加了许多新功能。例如：在“Crop Tool”（裁剪工具)的选项中新增加了透视选项，这样我们就可以裁剪出具有透视感的图像，如图1。
　　不仅如此，Photoshop 6.0的这一改进还充分体现了Adobe对于用户的体贴。就一个简单的例子而言，在工具条中选择选取工具，在Options Bar中就会出现图2中的4个按钮，它们分别是“选区、选区加、选区剪、选区交”。这些都是我们以前制作复杂选区的基本工具。通常都是要按住Ctrl或者Shift等组合键，然后再进行复合选取，一旦制作过程中松开组合键，后果就严重了。有了这些工具按钮就简单多了。例如：选区相加功能。按下Options bar上的选区加按钮之后做出的任何选区都是添加在原有选区上面的,如图2所示。无需再按shift键。
　　二、增加的工具
　　“Slice Tool ”（切图工具）：网页的平面设计要求越来越高，切图也成为网页设计师的一种必备手段。Photoshop 6.0也顺应潮流集成了这个工具。切图工具可以建立高质量、灵活的图像，可以直接产生HTML页，包含了全面优化功能，以尽可能小的文件制作出最优秀的Web图形。
　　“Shape Tools ”（图形工具组）：Fireworks的矢量和位图的结合曾经让很多的Photoshop用户羡慕不已，这次Photoshop 6.0也不甘落后。推出了一系列矢量图形绘制工具“Shape Tools”。 此工具包括了“Rectangle Tool”（矩形工具）、“Rounded Rectangle Tool”（圆角矩形工具）、“Ellipse Tool” (椭圆形工具)、“Polygon Tool” (多边形工具)、“Line Tool”（直线工具）、“Custom Shape Tool” (自定义形状工具)，这些工具所建立的图形都是矢量图形，可以把它们作为工作路径或作为矢量的层蒙板或普通的矢量图形，这些矢量的图形可以被“Path Component Selection Tool”（路径选取工具）和“Pen Tool”（路径工具）修改编辑，并且不受像素的干扰。想象一下，制作一个可以随心所欲调整的矢量蒙板的超爽感觉吧！！！
　　“Annotations Tool”（ 注解工具组）：你也许曾经做过一个令人佩服的效果。但是过后自己也无法记得其中的一些细节。现在Photoshop 6.0为设计师提供了一系列的注解工具，可以随时将你的灵感和创意以及文件一起记录下来。其中“Notes Tool”（记录工具）和“Audio Annotation Tool”（声音注解）工具，它们的作用是对图像进行注释说明，也可以用PDF 的格式输出，用于 Adobe Acrobat 进行阅读。当以BMP、JPG、TIF 等格式输出时并不影响图像的画面。
　　三、文字工具新特性
　　这是Photoshop 6.0比较革命性的变化之一。以往的版本中文字处理功能都是在对话框中进行，和Fireworks以及Photoimpact等图像处理软件相比显得略微有些生硬。 现在可以直接在图像上直接对文字进行修改、改变颜色、改变字体、风格等编辑工作，取消了原来的文本编辑对话框。并且可以在图像上对文字进行旋转、倾斜、比例改变的操作。
　　在“Type Options Bar”（文字选项条）上点击“Created Warped Text” （建立弯曲文字按钮）立即会跳出“Warp Text” 对话框，其中有15种扭曲风格供你挑选，每种风格可以根据用户的需要调节参数，如图3所示。
　　Photoshop 6.0 还新增加了“Character Palette”（特征控制面板）和“Paragraph Palette”（ 段落控制面板），通过这两个控制面板可以改变文字的字间距、字符的水平或垂直的距离等，还可以控制文字的排列方式，如图4所示。
　　四、更强大的层效果
　　现在除了原来的5种层效果以外Photoshop 6.0还增加了“Satin Effect”（光滑）效果、 “Color Overlay”（纯色覆盖）效果、 “Gradient Overlay”（渐变色覆盖）效果、 “Pattern Overly”（图案覆盖）效果和“Stroke Effect”（勾边）效果，并且层效果的对话框选项更加完善直观，如图5所示。
　　五、Style Palette 风格控制面板
　　Fireworks和Imageready有的Style样式功能，可以让你很方便的快速制作各种按钮和文字效果。Photoshop 6.0新增加的Style Palette 控制面板可以选定其中一个风格样式赋予给当前层、图形或者文字，并且可以进入Layer Style对话框随意编辑修改，还可以作为自定义的风格式样保存在风格控制面板上，以便再今后使用，如图6所示。
　　六、Layer Options层控制面板
　　在Photoshop 中，层具有很大的灵活性，一副图像可以建立上百个层，Photoshop 6.0的层控制面板新增加的功能可以更好、更有效的帮助用户组织管理层。
　　Photoshop 6.0在层控制面板上新增加了4个复选框，它们是：“Lock Transparent Pixels”（透明锁定复选框），当此复选框被选，当前层透明部分被锁定，不能够被绘图工具或编辑工具在上面添加任何像素。“Lock Image Pixels ”（图像锁定复选框），此复选框被选就意味着当前层不能够被任何绘图和编辑工具绘制和修改。“Lock Position ”（位置锁定复选框），此项功能表明当前层的像素不能够被移动工具移动。“Lock All”（全部锁定复选框）：当前层不能够做任何改变，如图7所示。
　　功能增强后的层控制面板能够让用户更有效地操作层。例如：你有一个层特效想附加给另外一个层，那只要拖动这个特效到目标层，全部的特效设置就会立即赋予到目标层上。
　　新增加的填充层“Solid Color”(纯色填充)、“Gradient”(渐变色填充)、“Pattern”(图案填充）这三种填充层不会破坏下方的图像像素，并且可以随时调整修改和丢弃此层。
　　当你安装了Photoshop 6.0和它完善的ImageReady 3.0那一刻起，你可以立刻感受到驾御图像的欣喜感，它可以让你从而更快更好的制作出高质量的图像，还可以提高工作的效率，让你进入一个新的突破。
　　至此为止，我们只是把Photoshop 6.0新增功能走马看花地快速浏览了一遍，更多细节上的改进则是无法三言两语说完的，Photoshop 6.0给用户带来的是更多的方便和实用。}
