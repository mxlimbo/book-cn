《 {Comdex远离PC时代} 》

{　　随着互联网兴盛，人们对纯上网用途的简单上网设备兴致大增，可上网的PDA、移动电话等设计新颖的上网工具成为市场焦点，PC独霸的时代已成过去，Comdex中非PC产品公司磨刀霍霍，在这次展览中大显身手。通过这次Comdex Fall 2000可以看出以后我们关注的将不再是电脑配件，而是无线通信、信息家电、因特网应用设备这些和生活联系得更紧密的东西。蓝牙是现在最火热的无线通信标准，有关蓝牙的设备已经进入了原型机测试阶段，但正式投放市场却要等到2001年中期。PDA和移动电话之间的界限正在变得越来越模糊，移动电话变得越来越像PDA，而PDA又通过安装扩展卡增加了移动电话功能。随着HPNA（home phoneline networking，家庭电话线网络）的不断深入，许多信息家电设备都开始进攻市场，其中以帝盟和3Com的HPNA音频外设最为醒目，再加上无线家庭网络，用电脑控制所有家电的时代已经到来了。
　　说到底，技术终归应用到产品上才能被人们接受，到底以后的焦点产品是哪些呢？看看下面这些东西吧。
掌上电脑
　　不要以为在Comdex上只有PC之类产品才是主角，针对个人的移动类产品在前两届Comdex大展上就出尽了风头，这一届当然也不例外。
　　美国最大的PDA类产品公司——Palm宣布，开始提供与手机连接后就可以使Palm掌上电脑接驳因特网的套件“Mobile Internet Kit”。该套件除了电子邮件功能以外，还提供电子邮件短信服务“Short Messaging Services(SMS) version 2.26”以及“Hand PHONE SMS version 1.3”，利用“MultiMail”可以接入POP3以及IMAP4帐户，以达到使用手机就可轻松上网的目的。
　　一个名为Pocket Linux Technology的公司，将Linux完全移植到iPAQ内。它使用笔触式操作，操作界面有别于现时大部分出售的PDA，仍然以图像界面操作，但储存操作系统的ROM模组就不同。内置的显示功能亦有所增强，除了以传统直立式显示外，最特别的是此操作系统还支持横向显示，以前只有Apple的Newton可以做到这个功能。该操作系统内置了一般的记事软件如记事本、通讯录及电子邮件，另外在多媒体支持方面也同样出色，使用内置的音乐及影片播放器可以领略到该操作系统的威力，通过iPAQ专用的PC卡扩展装置，再插入支持IEEE 802.11b无线网络标准的网卡，这个装置就可以对外进行沟通。
　　我国台湾厂商CMC展出一部具MP3/FM功能的PDA相机－Cyber Boy。该机背的液晶显示屏有240×320的分辩率，以4色显示，可于拍摄时预览相片，拍摄用CMOS感光，最高分辨率有640×480，相片以JPEG格式存储。机身内建8MB SDRAM及Flash RAM，以33MHz ARM－based处理器为核心。除此之外，三星的电话PDA手表亦引来不少注意，它只重50克，具30种电话功能，包括了WAP，也可收发电子邮件，而且可以语音操作。
MP3产品多元化
　　今年Comdex大展上各大厂商的MP3产品也呈现多元化，一方面走向整合式多功能；另一方面容量上也有突破。
　　在三星的摊位中，展示了一部Yepp'Plus（YVP－P600D），它是一部小型数码相机加MP3的混合体，外形采用折合式设计，相当可爱。它除支持MP3之外，同时支持WMA及AAC音乐规格，并可播放MPEG4动画。Digital Wallet是全球第二部6GB容量的MP3播放器，相对于创新公司的Juke Box，Digital Wallet体积比较小，只比一个5.25英寸硬盘大一些。它另一个优势是可插入PCMCIA Type Ⅱ卡，换句话说，只要有Type Ⅱ转换卡，就能兼容Memory Stick、Smart Media及Compact Flash存储设备，接口仍然使用USB为传输界面。
蓝牙产品成为最大焦点
　　蓝牙是由Nokia、东芝、IBM等大厂所共同开发，想不到以制造PC周边产品为人所熟识的罗技公司，在今年的Comdex之中就展出了一系列采用蓝牙技术的产品，包括鼠标、轨迹球、游戏手柄及键盘等。移动电话制造商爱立信在场馆中展出采用蓝牙技术的行动电话用免提装置，这样以后就可减少免提装置接长线的不便。Your Logo公司这次在Comdex 2000中展出多件利用蓝牙技术的产品，无论桌面电脑，手提电脑或个人数码助理中，都可找到其品牌的踪影。另外，Your Logo公司出品的蓝牙打印机适配器。只要接上打印机的打印机连接口，就可进行无线打印，传输速度可达460Kbps。如果连接了蓝牙网络，更可作网络打印机使用。
　　Sony这个业界巨人当然也有蓝牙产品展出，不过大部分都还是属于原型机。展出的样品有Bluetooth的Vaio C1笔记本电脑。还有采用蓝牙的通讯概念产品，蓝牙的Wireless Access及PDA，不过都还没有正式推出。
　　在这次的Comdex上最受关注的蓝牙产品是由瑞典Anoto公司开发的“Digital paper”。乍一看它与普通的纸别无两样，但使用被称为“Digital pen”的笔在上面书写时，它能自动地以无线方式将书写内容传送到附近的信息设备中。Digital pen的外观与普通的圆珠笔没有什么两样，里面还装有墨水，在普通纸上也可以书写。利用Digital paper可以将写入电子记事本的内容，同时保存到PDA及个人电脑中。
　　今年的Comdex已经表明未来是网络的世界，和网络相关的设备以后会很流行，从各大公司的行动可以看出，以后的产品会远离PC，更好地和信息家电融合。
(重庆　Joe)}
