《 网络吸血鬼的独门绝活 》

{　　众所周知，现在网上绝大多数的软件都是压缩成ZIP格式文件，以便下载。可是你有没有碰到过辛辛苦苦花了×××分钟拉下来的ZIP包竟然在打开的时候提示“压缩包已损坏”之类令人沮丧的情况？相信各位经常在网上冲浪的朋友或多或少都曾经为此头疼不已。究竟是下载过程中出了问题，还是服务器端的ZIP包原本就是坏的？难道我们只有先把××××K的ZIP包拉回来，才能发现这份迟来的“告白”？现在好了，老牌的网络吸血鬼（Net Vampire）推出了V3.3版，问题得到了解决。
　　也许，网络吸血鬼的下载速度比不上网络蚂蚁；它也不具备Flash Get那样实用的下载文件管理功能；更没有令人称羡的多进程下载功能。网络吸血鬼除了具备断点续传、定制目录下载、文件搜索、自动下载页面链接等常用功能外，最新的3.3版本还支持多语言功能。更令人高兴的是，新版的网络吸血鬼竟然还增加了一项不引人注目但却极为实用的功能――实时检测下载过程中ZIP包的完好情况。如果发现服务器端的ZIP本身已损坏，它会及时提示“ZIP file corrupted”（ZIP文件已损坏）。反之，如果文件完好无损，则显示“ZIP file OK”或“Partial ZIP file”（局部的ZIP文件），并在下载过程中实时显示该ZIP包中的有关文件列表。
　　这项功能真是酷呆了，从此我们再也不用担心辛辛苦苦拉回来的ZIP包成为一堆无法打开的“神秘”垃圾了。
