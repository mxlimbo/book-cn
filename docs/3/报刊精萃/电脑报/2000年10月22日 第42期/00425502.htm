《 给你的文件上把锁 》

{　　随着计算机互联网的普及，电脑用户已习惯将涉及个人隐私的文件如日记或个人金融账号密码，或通过电子邮件发送信件、资料，甚至一些至关重要的商业机密存储在电脑的硬盘中，由于电子资料有易复制性与易传播性，加之一些恶意的黑客捣乱，稍有不慎就有泄密的危险，从而使你遭受重大损失或伤害。我们知道，Word、WPS及一些压缩软件如Winzip、Winrar等都有密码保护功能，但不幸的是，用这类软件加密过的文件已无安全性可言，很容易就能找到相应的工具软件将其密码破译出来。这里我要向大家推荐一款极佳的文件加密软件Absolute Security Pro！
　　Absolute Security Pro是一款使用简单，功能强大的加密软件，其加密引擎采用了448位密钥长度的blowfish技术，使用它，你的文件在网络上就多了一分安全保障。Absolute Security Pro软件最新版本为3..7，下载地址是：http://www.pepsoft.com/absecpro.zip,体积很小，仅0.75M。
　　Absolute Security Pro的安装非常简单，打开压缩包后双击setup.exe就开始安装，然后再“Next”接着就“Finish”了。
　　一、文件的加密　
　　程序的界面(图1)相当的简洁美观，右部类似于资源管理器的窗口，能够很方便地选择你需要加密的文件。它既可选择任意文件类型，又可选择任意数量任意目录的文件，还可选择整个目录。
　　选择完毕后按“OK”，即出现界面（图2）：
　　密码的长度必须有十位以上，任意键盘字符均可作为密码。
　　Options下有四个参数，“keep original files”表示保留源文件。要注意，如果不选此参数则会将源文件删除！“pack all files in one” 表示将所有的文件压缩成一个加密包，“creat self－extracting”指生成自解压文件, 如果该参数不选，生成的文件后缀为.as2，解密需要Absolute Security Pro 软件或去Absolute Security Pro 站点下载免费的Absolute Security Decryptor 软件。考虑到方便起见，建议选上。但由于生成的是自解压文件，体积要比普通的＊.as2文件大190K左右。“send via email ”指通过E－mail发送生成的加密包。如果选择“creat self－extracting”则“keep original files” 和“pack all files in one ”自动选中。
　　一切准备就绪，点击“OK”！出现界面（图3）：
　　在对话框中,选择生成的加密文件的保存路径及文件名后点保存，待出现“×××.EXEcreated ”的提示框后,就表示已完成加密操作。生成的加密文件是自解压的压缩包，压缩比与zip文件不相上下，运行时也不再需要Absolute Security Pro的支持。
　　二、文件的解密
　　执行加密文件后在对话框中输入正确的密码，然后点“OK”：
　　如果你的加密文件包含文件夹的话，建议你将“create directories if needed”选上。完成上述操作后点击“decrypt”即完成解密。
　　三、文件的彻底删除
　　我们知道用Windows自带的删除命令删除文件后并没有真正将文件删除掉，依然可以使用一些工具软件将它恢复。Absolute Security Pro非常体贴地提供了将文件彻底擦除的功能，主程序界面（图1）右部有“wipe”参数，选中此参数，选取你所要彻底删除的文件，点“OK”：
　　“Number of Iteration” 让你选择擦除的次数，当然次数越多恢复的可能性越小。所以进行此操作之前千万要三思而后行。
