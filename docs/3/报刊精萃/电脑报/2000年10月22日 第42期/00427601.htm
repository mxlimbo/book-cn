《 {钻石CS65－SC主板评测报告} 》

{　　对于Intel 815芯片组的主板，我们在以前已经做过多次评测了，相信广大的读者对i815芯片组所支持的功能，以及目前i815芯片组主板的规格都已经耳熟能详了。因此，对于这次钻石CS65－SC i815主板的评测，我们觉得没有必要对上述两个方面再作过多的介绍了，我们将把这次评测的重点放在这块主板的性能上。作为BX芯片组的后继者，i815芯片组的性能已基本上得到了大家的认可，正逐步取代BX芯片组的地位，成为市场上的主流产品。i815芯片组主板存在两个版本，i815和i815E。它们之间的区别，我们已经在前几期的评测报告中作过比较详细的解释了，所以也不在这里重复介绍了。如果还有不清楚的朋友，就请查看一下我们前几期的《电脑报》。从总体上说来，i815E在性能上更为优异，支持的功能更多，但是在价格上要贵上一两百元。而i815支持的功能虽然并没有i815E多，但是它已完全能够满足用户对目前所有比较流行功能的需要，像Ultra ATA/66、AGP4×等。而且，i815芯片组主板沿用了i810整合型主板的成功经验，板上已集成了显卡和音效芯片，因此对于手头不宽裕的朋友，应该说也是一个不错的选择。说了这么多，下面我们就来看一下这次送测的钻石CS65－SC主板到底在性能上表现如何。
产品简介
　　这块钻石CS65－SC主板和其它的i815芯片组主板相比，无论是从规格，还是从整体配置上看都大同小异。整块主板采用了黄色的PCB基板，用料很足。主板提供了一个Socket 370架构的CPU插槽，支持Pentium Ⅲ 533EB～933EB 和500E～850E FC－PGA架构的处理器，以及支持566MHz ～700MHz的FC－PGA架构和300A～533MHz PPGA架构的赛扬处理器。更为有趣的是，这块钻石CS65－SC主板还支持VIA的Cyrix Ⅲ处理器，这在其它同类型的i815主板中是不多见的。我们都知道，英特尔和威盛一向都是冤家对头，两者正在主板芯片组市场上争得不可开交。本来在低端CPU市场它们也将有一番激烈的争夺，只不过是由于VIA的Cyrix Ⅲ 处理器的性能根本不能与Intel赛扬系列以及AMD Duron处理器相比，才避免了又一场大战的爆发。因此，采用Intel 815芯片组的钻石CS65－SC主板能够支持VIA的Cyrix Ⅲ 处理器，多少令我们感到有些意外。虽然目前Cyrix Ⅲ处理器在市场很少见，但是钻石CS65－SC还是为低端配置的用户提供了另一个选择。另外，该主板还提供了五个PCI插槽，一个AMR（Audio/Modem Riser）插槽和一个AGP（2×/4×）插槽。与众不同的是，这块主板为用户提供了四根168线DIMM插槽，而其它主板最多只有3根，但它最高还是只能支持512MB的PC100/133 SDRAM内存。四根DIMM的设置也有所不同，第一根和第二根DIMM插槽和其它主板的DIMM插槽没有什么区别。但由于i815芯片组最高只能支持512MB内存，因此在第三条和第四条DIMM插槽的使用上就比较特殊。第三条DIMM插槽只有在第四根DIMM插槽为空时，才同时支持单面和双面贴片的SDRAM内存。如果第四条DIMM插槽插有内存时，第三根DIMM插槽就只能支持单面贴片的SDRAM内存了。第四根DIMM插槽则只支持单面贴片的SDRAM内存，而且是要在第三根DIMM插槽只插有单面内存的情况下才能够使用。因此，用户在配制内存时要注意搭配。除了上面我们介绍的这些设置以外，钻石CS65－SC主板还提供了两个USB接口；两个Bus Master IDE连接埠，最多可支持四个Ultra DMA 33/66设备；内置了Intel i752图形加速芯片，支援AIMM （AGP Inline Memory Module，AGP板上内存升级模块）功能。除此之外，主板上也整合了AC’97音效子系统。从钻石CS65－SC主板的配置情况来说，已经可以满足大多数用户以后扩展和升级的需要了。
　　友通公司还随主板为每位用户提供一张驱动程序安装盘和一本主板说明书，说明书包含了多种语言版本，像英语、法语等。但就是没有中文，看起来比较令人头痛，友通公司在这方面的工作还需要加强。
超频设置
　　钻石CS65－SC主板的超频设置和其它i815主板并没有什么不同，仍然采用跳线和BIOS相结合的设置方式。而和这块主板一同送来的另外一块钻石i815E主板，却采用了先进的线性超频技术，能够实现从66MHz到157MHz的线性超频。钻石CS65－SC主板的右下角上提供了两个跳线，通过它们就可以把CPU外频、系统总线频率切换于66/100/133MHz三个标准外频之间。在一般情况下，这两个跳线只需被设定为Auto状态，它就会根据CPU的类型自动选择上面三个标准频率中的一个来运行。这里的跳线只是一个粗略的设置，更多的设置就需要用户在BIOS中完成。BIOS中提供了丰富的外频选择，最高可选外频达到了157MHz，应该说钻石CS65－SC主板的外频选项设置方面做得不错。另外，BIOS内还提供了异步系统总线/内存时钟调节选项，用户可以根据自己的情况进行设置。但是，钻石CS65－SC主板上没有提供CPU核心电压以及I/O电压的调节。
性能测试
　　测试平台
　　主板：钻石CS65－SC主板
　　CPU：Intel Coppermine 550E(FC－PGA)
　　内存：128M KingMax 单条
　　硬盘：IBM DTLA 307020支持ATA/100
　　显卡：大力神GeForce2 MX 32MB SDRAM
　　系统：Win98SE
　　从这次评测的各项测试情况来看，钻石CS65－SC主板的表现是比较令人满意的。在测试各部分子系统性能的WinBench 99 测试中，测试数据表明钻石CS65－SC主板的各部分子系统的性能是比较优异的。表现最突出的地方出现在磁盘子系统部分。在133MHz外频下，钻石CS65－SC主板的Business Disk WinMark和High－End Disk WinMark得分，分别取得了6030和21500的高分，基本上与支持ATA/100的主板处于同一水平上。由于i815芯片组并不支持ATA/100，因此应该说钻石CS65－SC主板在磁盘子系统性能方面不如i815E芯片组的主板。从这次钻石CS65－SC主板磁盘子系统的测试数据来看，好像这种差距并不是很明显，ATA/100还远远没有真正发挥出它的水平。但是，从另一个方面来说，钻石CS65－SC主板的IDE驱动部分确实是做得不错，它使得ATA/66的性能得以充分发挥，即使与支持ATA/100的i815芯片组的主板相比较，也丝毫不落于下风。
　　由于测试中我们使用了附加的大力神GeForce2 MX显卡，这块显卡在我们前几期的《电脑报》GeForce2 MX显卡横向比较测试中的表现就很优异，所以钻石CS65－SC主板在WinBench 99图形部分的Business Graphics WinMark和High－End Graphics WinMark中的表现也不错。这次，我们并没有对钻石CS65－SC主板内置的i752显卡进行测试。因为对它的测试我们以前已做过多次了，相信读者们对它的性能也有了一定的认识。
　　Winstone 99、CC Winstone 2000和SYS Mark2000主要是针对整机性能的测试，不过它们的着重点有所不同。Winstone 99测试偏重于常见的办公用软件，包括文字处理，电子表格等，而CC Winstone 2000适用于配置较高的、主要用于图形图像处理、多媒体制作等的电脑。无论是从Winstone 99，还是从CC Winstone 2000的测试成绩来看，测试成绩与CPU速度和内存容量、速度关系密切。具体反映在100MHz外频下，这时的CPU运行在标准的550MHz，钻石CS65－SC的Winstone 99的测试成绩为28.6。而在133MHz外频下时，这时的CPU运行在733MHz，Winstone 99的测试成绩则上升为32.5，系统性能提升高达13.7％，升幅是相当可观的。CC Winstone 2000的测试成绩与Winstone 99的测试成绩几乎成相同的趋势。其中，内存速度的影响也是一个重要的因素。从另一组测试数据我们就可以清楚地看到，当系统总线运行在133MHz下时，如果把内存总线的频率设定为100MHz，其测试成绩与内存运行在133MHz下的测试成绩相比较，都有不同程度的下降。特别是在随后进行的MDK2以及Quake3测试中，其下降的幅度更加明显。说明内存总线的速度对系统性能的影响还是比较显著的。因此，我们建议使用i815芯片组的用户，尽量使内存运行在133MHz内存总线频率下，以期获得更好的性能。
　　SYS Mark 2000是类似于ZD Winstone 99和CC Winstone 2000的综合体，它同时包括了办公软件和制作软件，是一种综合性的测试软件。因此，SYS Mark 2000的测试成绩具有与ZD Winstone相同的规律。
　　在MDK2和Quake3的测试中，我们想看一下钻石CS65－SC主板在真实的游戏环境中究竟有怎样的表现。从测试成绩来看，在大力神GeForce2 MX显卡的帮助下，钻石CS65－SC主板取得了非常好的成绩。在133MHz外频，640×480分辨率，16位色下，MDK2取得121.36帧的好成绩，Quake3的成绩则更高，Demo 001的测试成绩达到了125帧，可以说是非常出色了。
超频测试
　　超频测试的重要性再也不用我多说了，因为它越来越被电脑爱好者所关注，尤其是对于那些对电脑硬件知识有一定了解的爱好者更是如此。所以在我们的测试中，超频测试也被当作一个不可缺少的步骤来进行。就钻石CS65－SC主板的超频设置而言，我们也在前面提到了，由于它是采用跳线与BIOS相结合的方式，因此在超频设置方面并不是很方便。但这块主板的外频选项则设置比较丰富，而且整块主板超频后的稳定性比较令人满意。我们就在钻石CS65－SC主板上把Coppermine 550超到了733MHz，并且非常稳定地跑完了所有的测试，得到了测试数据，没有发生什么异常情况。随后我们又把它超到792MHz（144MHz外频），开机后顺利通过自检进入系统。我们运行了一下Quake3，测试非常顺利地就完成了。我们再接再厉，把外频设置为146MHz。这时，机子能够顺利点亮，但在进入系统时出现了错误，没有成功。我们在BIOS中调整了一下有关内存方面的设置，因为在前面的测试过程中，我们对这些值都进行了重新设定，以期发挥内存最好的性能，但现在必须把它们设定回来。结果正如我们所料，现在已能很顺利地进入系统，但在运行Winstone时出现了错误。不过，总的来说还是比较成功的。接下来，我们又把外频调到150MHz，结果情况和上面的一样。从超频测试的情况来看，钻石CS65－SC主板表现还是比较优秀的。
总　　结
　　从整个测试情况来看，钻石CS65－SC主板的各部分性能表现都比较优秀。特别是它的磁盘驱动部分，给我们的印象很深刻。目前，这块主板的市场售价大约在930元左右，这可能是目前i815芯片组主板的最低价格了，非常超值。我们认为钻石CS65－SC主板完全可以满足普通家庭用户和要求较高的发烧友的需求，特别是对于采购预算不多的用户，它是一款性价比非常高的主板。}
