←　◇　差猜·裴皮亚（Chatchai Puipia）
1964年出生于泰国马哈萨拉卡穆，现居曼谷。长久以来，他从他自己身上感受到一种幻觉。在经济危机爆发的数年前，他和社会乃至批评保持着距离。因为一切变化来得太快。今天他的作品表现更多的是他内在世界的神秘性。┈┈下一幅┈┈

←　◇　陈妍音(Chen Yanyin)
《一念之间的差异》
1958年出生于上海，现居上海。作为一位长期从事装置艺术创作的女性艺术家，陈妍音的风格是敏感、细腻而又充满个性的，她对女性心理世界的描述具有异常丰富的表现力。装置作品《一念之间的差异》通过对玫瑰花输液这一行为表达出艺术家对生命与爱情等问题的关注浑噩感慨。

←　◇　赫利·多诺(Heri Dono)
《被网住的天使》装置 1996
1960年生于印度尼西亚雅加达，现居雅加达。赫利·多诺知道这个世界到处充满了精神。他研究了一些大城市的人们，他们的头脑中失去了精神，表现的是空虚。

黄步青
（黄步青 野宴 威尼斯 装置 ）
1948年出生于中国台湾彰化，现居台南。《野宴》是黄步青的代表作，他用各种各样的植物种子和中国古典艺术的视觉图式构成了一个颇具东方色彩的仪式化空间，其返朴归真的语言具有一种全新的美学色彩，艺术家同时也提示着土地和自然对人类生存的重要性。   ┈┈下一幅┈┈

←　◇　施慧（Shi Hui）
1955年出生于中国上海，现居杭州。作为一位杰出的女性艺术家，施慧的作品显示着她对日常生活中常见材料的持续关注，以及由这些材料所衍生出来的关于增殖与共生的观念。参展作品《飘》用悬挂空间中的灰、白色云状物，提示着人们对空气质量等环保指标的关注。

←　◇　施慧隋建国（Sui Jianguo）
1956年出生于中国山东青岛，现居北京。在《衣纹研究》这组系作品中，隋建国把西方经典雕塑与中国经典服饰结合在一起，是一种以经典的重新审视以及对自身角色进行清理的尝试。这些身着中山装的古希腊和文艺复兴雕塑，形象而不无幽默地展现和探讨了全球化时代东西方文明的融合与冲突。

←　◇　舟越桂（Funakoshi Katsura）
《冬之书》
1954年出生于日本盛冈，现居东京。他善于刻在木雕上的人以生命，但他们生活在另一个世界上，一个永远沉静的世界。舟越桂非常熟悉那个世界和那里的人们。他的作品灵感来自日本传统雕塑中精神多于自然的启示。
