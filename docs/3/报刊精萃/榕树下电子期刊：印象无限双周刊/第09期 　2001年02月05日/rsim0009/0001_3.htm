布拉格的桥和城堡
　

　　进入布拉格，第一个印象就是觉得它的大街小巷到处都有酒吧和咖啡啡厅。喝一杯啤酒，晴空下的布拉格显得分外明媚，伏尔塔瓦河把布拉格分为东西两岸，西岸是布拉格城堡区，东岸是老城区，连接两岸的是远近驰名的查理大桥。
　　这座全长520公尺的石板桥建于14世纪，本身已经是艺术品，两旁三十座雕像，把世人形象勾画得栩栩如生，加上满桥特色地摊，增加了查理大桥的艺术氛围。现在已经有13座桥可以作为连接两岸的工具，这个桥已经变成一条步行街，人来人往非常热闹。桥的两旁有很多卖工艺品的小商贩，桥上面也有一些艺术家在表演他们的艺术。
　　所有来布拉格的记者和摄影师都不会错过布拉格老城广场的市政厅大钟。它是汉努斯在15世纪初设计的，设计完这口钟，当时的议员们怕汉努斯再为其他人设计类似的作品，就弄瞎了他的双眼。这个钟的钟声非常悠扬，但背后却有着非常悲惨的故事。
　　用石头和灰浆建于9世纪的布拉格城堡，分为三个庭院。其中第三个庭院建有圣维特大教堂，这座自1344年建起到1929年才精雕细琢而成的哥特式建筑，是布拉格最古老的罗马式教堂之一，也是欧洲第四大教堂。总统府的阳台正是古代皇帝以及历届总统发表重要讲话的地方。
　　走出城堡外，站在拜克山上，眺望整个迷人的布拉格城，伏尔塔瓦河好象银色的飘带，古老的城堡和教堂上镀金的金塔，在阳光照耀下金光闪闪，难怪诗人歌德把布拉格形容为王官上的珍贵宝石，这说法一点都没有错。布拉格也被称为著名的影城，许多国际级电影都在这里取景拍摄，例如好莱坞电影《碟中谍1》，电视剧《印第安那琼斯》等等。
