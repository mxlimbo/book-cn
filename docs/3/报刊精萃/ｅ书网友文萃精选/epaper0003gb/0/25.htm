乡 间 小 路

丑草草

　　映山红花开的时候，满山遍野红彤彤的一片。一放学，十一、二岁的我们便成群结队的欢呼着向那令人心醉的景色奔去。

　　男生爬上山去采摘最高处的映山红，我们几个女孩子则沿着山下那一条向远处延伸的小路慢悠悠的一路游去。

　　小路的一边是山，另一边是低矮的坡地。山壁上常有俏皮的树枝斜斜的向下垂吊着，一些小小的花草杂居在树根处，摆弄着优雅的姿势，发出幽幽的叹息；而坡地上则有一些藤状物努力向上攀爬着，在小路旁边露出小小的端倪。

　　长得最旺盛的是黄色的野菊花，平坦的草地上、浓密的灌木丛中，到处都可以觅到它的芳踪。它算不上美丽，更因其的繁多而显得平凡。可伊是那样执着的，并不因为别人认为自己卑微而卑微，努力的绽放着生长着繁殖着，像为夜而闪烁的群星一样成为大地的缀点。

　　山坡篱笆上大朵的五瓣花，是我们的至爱。成熟的五瓣花有着圆圆的花瓣，小小的黄色花蕊，极淡极淡淡到只有凑到眼前才能闻到的花香。含苞欲放的五瓣花最是迷人，饱满的瓣紧紧的拥在一起，诉不尽的千言万语尽在欲露还羞中。它们就那样娇俏的、自然的、平和的、静静的、毫无拘束的望着你，直望入眼睛的深潭里。

　　还有那些小小小小的非得很仔细才能看得到的深紫色的"小鸟花"。它将纤小的身子隐藏于墨绿的草丛中，像正在引颈啾啾鸣叫的小鸟，却是无声的，仿佛是为自己歌唱，又仿佛是在为谁守望，有着不与人知不为世争的快乐和淡泊。

　　回去的路上，我们会找一些刚刚长成的嫩竹枝，抽出分叉上的芯，把红的黄的蓝的紫的各种各样的花插进去，而未开的花苞自然要插在最显眼的地方。一束美丽的竹枝花就可以随风招展了。

　　男生呼叫着从山上狂奔下来，一把把映山红在他们的手中灿烂夺目。好些个男生嘴里含着两根狗尾巴草，眼皮处还支着两根，像极了似醒非醒的小猫咪。

　　会合之后，尖叫声四起，原来是那淘皮的男生把一朵朵花瓣扔在了女生的头上！大家追逐着，欢笑着，吓飞了回巢的小鸟，逗红了天边晚霞的脸……

　　一路美丽的还有我们花样的少年。
