我看网络，我看网友，我看网恋

秋歌

　　网络是个虚幻的实体，它可以给你提供可以想象的所有的快捷和方便；它也可以给你带来无妄之灾，没由来的把你的世界搞的一团糟，包括你的电脑和你的生活。

　　网络交友是现今生活在都市中人的热衷方式，不依年龄界定，但基本上以生活无忧，衣食不愁为低限。在网上，你可以抛开你的现今的一切，可以很容易的成为另外的一个人，与己毫不相干的人。你可以轻易的重生、克隆，更名换姓不必通报任何人。你可以坦诚相对，也可以鬼话连篇。网络给你提供了这样的环境，这样的机会。可以结识很多你生活圈子以外的人，但是，你不能肯定，你所遇到的人，是不是抱有同样的真诚。你可以相信一切，你也有权怀疑一切，因为这是网络，让你身陷其中，却看不清，触不到的网络。

网友

　　――可以只通mail，相交就象笔友，只是若没有共通的思想与话题，这种关系不会保有太久。
　　――可以只通电话，隔三差五拨通电话，聊天说地，相互问候，慢慢形成默契，彼此珍惜，相交逾久逾无法见面，于是选择了最佳方式，相见不如怀念。
　　――可以从网络走入生活，定期的网友聚会，大家相处甚欢，择三两个投缘的进而交往引为知交。若是男女两人，两厢有意，一段恋情就此展开也未可知。

　　至于网恋，本人拙见还是网恋止于网的好。在网络中寻一个虚幻的情人，投入一段真实的情感。如果可以的话，把它与生活分清楚。只是让自己在面对诸多的问题与压力时，有个倾诉对象；快乐有人可以分享；索然无味时有个人可以思念罢了。

　　（曾有朋友批评我将网恋的定义说的太狭窄了，在他看来，应该是于网上相识的进而相恋，都算是网恋，无论网上还是网下。）

　　但是，要做到上述状况，委实很难。人毕竟是情感动物，理智在与情感相抗衡是，常常显得那么苍白无力。于是恋情便进入了生活，于是一切的幸福、一切的悲哀便由此展开。

　　幸福的个案是肯定存在的，我知道！只是看到身边的更多的是掉进感情的旋涡，欲罢不能，自己为难自己的伤心人。

　　网恋，恋的很苦。

　　于我个人而言，在网上，可以不必计较，不必认真，可以调侃无状，山盟海誓全当过眼云烟。然而，网恋进入生活，不由得人不认真。我会投入，我会在乎。如果有伤害，将会是惨痛的一击，固然不足以致命，却足以伤心。

　　依然期待美丽，依然渴望奇迹，只是会很小心，不愿做为难自己的伤心人。
