大 连 散 记

草本植物

　　走近大连，宛若走入欧州的某座小城。早已失去了传统风格的新中式建筑，与随处可见的红砖日式房，气派的俄式公寓，尖顶的带有哥特风味的欧式别墅。

　　沿着最繁华的人民路走，两旁是那些高楼林立与国际接轨的大厦，殖民地时的带有明显异族风味的建筑群，像幅拼图，美丽但不完整。

　　人民路的尽头是港湾广场。广场中心修筑着用铜板搭建而成的，郑和下西洋时所使用的航船的微缩模型，离此不远便是竭尽繁容的都市。那一切给人的感觉仿佛是一个时髦的女郎，穿得是eitoky，脚上踩得是vivian。可不经意间，一转身脑后想梳得依旧是盘髻，只是找不到了束发的玉簪。

　　大连建市不过百年，可是每每大连人说起，总是说大连是座历史悠久的城市。也许的确如此，大连前身又名旅大市，先后沦为德国、日本等国家的殖民地，经历坎坷。在它的中山广场，环绕着八个国家的代表建筑，就像是给那个时期所竖立的一块记念碑。可大连是座鲜少见有革命历史博物馆的城市，居说都建在了离市区几十公里以外的旅顺。

　　大连以广场众多而闻名，大大小小不下于三四十座。其中以中山广场，人民广场等最富有胜名。人民广场是座有名喷泉音乐广场，每当夜幕降临，喷涌而出的泉水被闪烁的霓虹灯涂抹上了浓艳的色彩，在洁白的大理石映衬下，宛若一幅抽象的水彩画。背景音乐是郭阑英唱着：一条大河波浪宽……。所有不可调和的东西在夜色中被揉杂成一团，然后在观看者模糊的表情里再被忽略了。

　　在胜利广场的露天咖啡座上听音乐，朦胧的灯光下，不经意的飘过几个肥皂泡，色泽斑斓，不仅令人心生恍惚之情。眼中的一切宛若手里的这杯饮料，散发着浓浓的茶味与咖啡味。不知该确切地称呼它是茶，还是咖啡。可唯有如此，才是这个时代的象征，所以它叫鸳鸯咖啡。 长江路上的有轨公车也是大连一道独特的风景。这种旧迹在大连完好的保存着两条路线。老式的火车头带着特有的节奏行驶在车轨上。路许是老路，可路旁的风景早已是白云苍狗，物换星移了。坐在公 车上，听着车轮与铁轨的啮合声，觉得也许那就像新旧历史的交融，总是在碰撞中吻合。

　　沿着海弯游行的是滨海路。路面几公尺以下便是幽静，沉蓝的勃海。淡蓝天空里飘浮着荷袂似的白云，天际一线处有只欧鸟飞过，啾啾响亮的鸣叫声打破了这快要沉寂了似的油彩画面。微风吹过，搅乱的是 它投注在海面上矫健的身影。

　　日薄西山，远处的海渐渐被染成了血红色。也许日夜更迭，月涨潮起是无法摆脱的命运，可是海依然有它自己的意志。平静的海面底下暗涌的浪流，是它永远渴望奔跑的冲动。忽然间，便会涌起一朵浪花， 尤如镶嵌在海面上的一朵含苞欲放的白兰花。它拥着海浪向岩石冲去。倾刻，宛若一块巨大的翡翠碎成了细细的粉末，呈扇形四散开去，在空中停留了一下，又落回了大海。极美的东西总是瞬间即逝的，可 那尤如飞雪一般撒落的浪花却深深得铭刻在了记忆中。

　　还有大连那不能不提的生猛海鲜，可惜当时去的季节不时候,恰巧碰上封海期――大连出于对卫生和海洋生物繁衍的需求，在每年的七月至九月实行封海。不过在中国也就没有什么是绝对不可能的，友人带我们去了一家名叫富哥的海鲜楼。据介绍孙继海与郝海东是这里的常客，友人说起颇显自得。也许与这里的海鲜相比，他们是要更具有知名度。

　　友人强调海鲜要讲究鲜活，而且最好是生吃，这样才能品出个中的滋味来。客随主便，当一盘兀自欢跳的竹节虾端至桌面，我方明白生吃的含义。要剥开还在你掌心中跃跃欲试的小生物的壳，品尝它还在颤抖的肉，那滋味首先给你的是口感以外的冲击。无需任何作料的生虾肉自有一种天然的鲜咸，风味独特。只是我在朋友殷勤的规劝下品尝了几只后，便随即没有了这种食欲。

　　而后上来的一律是不经加工的海鲜，但都以切割好。没了视觉上的撞击，我这个远庖厨的君子方显得有些安乐。三文鱼与芥未仿佛已经成了一对焦孟不离的老搭档。切成薄片粉红白条相间的三文鱼片，绿色的芥未消融在深咖啡色的醋中。鱼片蘸了些醋液，先深深吸口气然后才将它纳入口中，吃时尤要张大了嘴，不然那股属于芥未的冲劲会使你憋过气去。席间有朋友因受不了芥未那股特有的味道而咳起嗽，惹来大家的一阵哄笑。

　　在酒楼下面的示范鱼缸中见到的悠游自在的海参，也已被切成了薄片躺在盘中，吃起来即脆且韧。拳头般大小呈圆形的海胆长满了黑色的长刺，现在外壳被掀开了一块，里面是尤如蟹黄似的膏状体，吃起来也颇有蟹黄的味道。还有各色贝壳类，最后上的是一道名叫吉利鱼的海鱼。比之湖鱼，鲜味虽然不减，但论起那份入品即化的口感却要差着好许。

　　饭后乘着酒兴，友人开车带着我们再次游了遍大连。港湾广场，人民广场，中山广场，霓虹灯下那千姿百态的世界显得扑朔迷离，叫人流连往返。

　　离开大连的那天清晨醒得很早，从丰源大厦十一层楼望下去。那些充满异族风味的建筑在晨曦的阳光下朦胧的映射着。长江路上传来车辙的撞击声，郑和下西洋时的起锚声，一切都在拂晓微风吹送下渐行渐远。这是个在历史间梦游的，美丽的大连。
