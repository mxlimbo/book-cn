第一章 99行动计划

'99何处去

　　冬去春来,岁月的轮回,又迎来了新的一年。
　　兔年的吉祥，应是责无旁贷。我的新的一年也应该一改往日的沉迷，重振当年的雄风，雄风的始处应从何而起呢？
　　也许首先的一点，应从抖擞精神而起吧！生命在于运动，一切革命的动力来源都在于有一个强健的身体，在所有摧残人意志的要素中，再没有比长期病魔的缠身最巨大了。故此，要一扫"东亚病夫"的形象，从强身健体、改变精神状态和生理状态双重要素着手：锻炼身体，早睡早起，保持一个良好的身体才能干自己所想干的事情。
　　其次，一个人必须有一份工作。找到一份工作，而且是能够找到一份薪水丰厚的工作，对于一个独立的人来说是比较重要的，至于工作的劳累程度和贵贱高低则应该是其次的考虑。一个人要独立，首先是经济上的自立，而且还应该肩负起自己所处家庭角色的责任，养家糊口。不然，他就无法奢想任何东西。为此，他就必须摒弃其他任何的陈旧理念，一心去想挣取更多的货币。
　　基于以上两点，99之年应做的事情如下：
　　1、找一份薪水丰厚月薪≥1000元的工作，无论如何。
　　2、加强身体锻炼，养精蓄锐，以利战斗。
　　3、放弃专业炒股挣钱的幼稚想法，股市还是最好不涉足的为好。
　　4、无论如何筹措资金，应离开目前的租房环境，买上一套新的住房住进去。否则，一切将无从谈起。

路在何方

　　经过历时四天的奔波,现在看来,厦门的经济并不怎样景气.欲想在此觅到一个职位，也并不那么容易。由于专业的限制，道路环境的不熟，亲自登门造访，效果实在太慢，一天最终也只能造访三个招聘单位。常常不是坐过站点就是坐错线，而待找到单位时，又适逢其人不在，或已近下班被人拒之门外。故此，采取以信函形式一次五六封的发出求职信，合计起来，既省力又经济而且还效率高，因为如果对方没有诚意，即便去到也是枉然。

下一步如何

　　房产与物业，这些都需有巨大的资金做雄厚的支持，也不能选择。其他文理专业，都没有长期发展的潜力，可以保持三年甚至五年或十年内拥有巨大的发展机遇。
　　故此，从根本上讲，电子通讯与计算机行业才是真正的未来十年内不会衰败的产业。基于此，下一步是必须学习计算机技术。

关于出行的行李多少及如何熟悉陌生城市的问题

　　出行在外，对于不确定因素太多的事情，应该尽量轻装简行。对这件事，还是马虎不得。如果这次来厦门，我要是只带一个行李包的话，我就拐头直达深圳了。何况这次带的许多行李都没有用，只是成为累赘，束缚人的行踪。因此，初来乍到一个城市，首要的至关重要的便是吃住两项问题，吃一般很好解决，自带干粮或到饭店就行了；而住通常比较难解决，住旅社太贵，租房时间短，还要自带被褥，折算起来，既不经济也不省力，此次厦门之行便是明证。
　　安排了吃住之后，就是对这个城市的熟悉过程了。据我个人的感觉，一般说来，初到一个陌生的城市，首先的困难就是地理环境的不熟悉，导致出行的不便，要么走过，要么走错，要么走得不够，经过三五天的奔跑，几个常用的主要乘车路线及交通要道便基本熟悉了，此后，就会少走许多弯路。
　　所以，初到一个陌生的城市，想尽快熟悉他，首先要做好以下基点：
　　1、首先安排好吃住问题。
　　2、买张交通图，花上三至五天，把重要交通干线及乘车路线和重要站点搞清，尽量避免走弯路或减少走弯路。
　　3、观察此城市的人文、经济、政治有何特点，适宜自己此行的目的否？

第三章 郑州反思

迟到的觉醒
——漫谈机遇的存在理论

　　人生中难免不走弯路，对于我来说，弯路也许较别人更长，究起源，也许这正是我本人的弱点之一，反映迟钝，于是觉醒总是落后于别人，而要取得成功更是要花费比别人更长的时间和辛苦。
　　回想毕业十年来的弯弯曲曲，总有太多的遗憾和惋惜。从信号的传播理论来讲，输入的信号总是一直向下去，而反馈的信号总是延迟太多的时间，以致到调整时，已发现贻误太多的发展良机。
　　但从机遇的存在理论来看，机遇的出现和存在相对于某一个人具有辩证式的公正性，或者称之为公正与不公正的双重存在性。
　　换句话说，当整个社会或时代出现某一机遇时，则这个时代的每一个人都同时拥有这一机遇。可是具体到每一个人由于环境或者其自身主客观条件与地点的不同，有些人可以利用，有些人则无法利用，这就体现出机遇的不公正性一面。也就是说，机遇的出现和存在是随着时、空和人的条件的变化而变换的，正所谓天、地、人三者的同时出现吻合才方为真正的机遇。否则，天时、地利皆有，而人和并不出现（此时人和特指个人自身素质条件同天时地利的吻合），则对其个体来说，机遇便体现出不公正性的一面来，表现为机遇的白白流失。故此，抓住机遇，从这个意义上理解，首先就是要让人这一主体，去迎合客体天时、地利。当天时出现，而地利不出现，可以通过迁移居住生存环境去获取地利，如我国古代的孟母三迁其家终助孟子成为教育家的故事；当地利出现，而天时尚未出现时，可以通过事前筹备待时而动；当人和的条件尚未出现，可以通过武装自己，改善自身的素质条件，如学习技术，扩展社会关系，主动寻找同天时、地利、人和相匹配吻合的条件。如此，和着机遇跳动的脉搏前进，才能永远立于不败的潮头。
　　如果要从机遇的发展历史来看，任一机遇如同自然界中的万物一样，都有一个从无到有，再从有到无，随后，新的机遇再从无到有、从有到无的循环变化过程，而每一机遇在其存在期间，又会形成一个出现、成长、衰减到死亡的过程。故此，我们可以这样认为，机遇在宇宙中是永远的存在的，如同能量和物质，只不过是从一种能量转换为另一种能量，与其同时，一种形态的物质变化为另一种形态的物质。或者说机遇是一种特殊的能量物，它伴随着一种事物的出现而出现，然后生长、壮大到鼎盛，再逐渐衰减以至消失，同时另外一种新的机遇又在渐渐产生、成长。由此，我们可以把机遇所蕴含的这种能量称为机遇能。
　　若用E表示机遇能量，G表示机遇物，EG表示机遇，机遇的运动规律可用公式表示为：
　　能量式：E1=E2=E3=E4=…… 其意义表示机遇运动过程中能量是守恒的，当后一机遇物的能量作用逐渐增长时，前一机遇物的作用同时逐渐缩小；
　　物质式：G1→G2→G3→……其意义表示机遇运动过程中机遇物即机遇的表现形式由一种物体向另一种物体转移；
　　故此，上述两式，可以启示我们在行为中，如果遇到机遇G1在衰减，就应该保持敏锐地知觉注意正在增强的机遇物G2正在何种事物上体现，以便及时把握新出现的机遇，从而取得制胜的先机。
　　根据以上两式，可以认为，机遇在总能量上是永远守恒的，但在机遇的表现能和表现物上却是随时变换的，新旧机遇间保持一种动态平衡关系。我们把机遇的这种特点称为机遇永恒定律，上述二式称为机遇方程式。

1999.4.15于郑州

关于机遇点和线的存在理论

　　鉴于以上所论机遇的运动规律，我们可以认为宇宙世界是一个三元世界，它有天、地、物（人是物元的一分子）三元，每一元作为一轴，天元，即时间轴，具有单向不可逆性；地元，即空间轴，而空间轴是立体的，又可分为三个亚轴，具有三向性，即我们日常所见到的三维现实空间；物元，即对象轴，包括自然界中万事万物，可分为生命对象，具有主动性，尤以人为代表对象；非生命对象，具有被动性，如水；物元的特点具有主动被动二重性，也即是说，物元的任一个体，对天地二元所赐予的机遇，表现为生命对象的主动可选择性和非生命对象的被动不可选择性。
　　故此，在天地二元之间就构成一个时空界面，姑且称之为时空面，在这个时空面上，就存在着无数的机遇，有的转瞬即逝，称之为机遇点；有的可能保持一段时间甚至相当长的一段时间，称之为机遇线，他们本身也在不断的运动和变化。当物元上的任一对象，在运动中，如果碰到机遇点或机遇线，对于非生命对象具有被动不可选择性，就必然受到机遇能的作用；对于生命对象具有主动可选择性，可以主动选择机遇，比如武装自己，迎接挑战，发挥机遇能的作用，当然，这种作用的大小，还要结合生命对象自身的状况；也可以主动放弃机遇，比如离开机遇点和机遇线所在的行业和具体的事件，从而避免机遇能的作用；也可能不知觉地放弃机遇，事过之后，回首时深感痛惜。

1999.4.15于郑州路寨

关于机遇的变化动因论

　　前文从机遇的运行轨迹表现形式阐述了机遇的特点，下文试图对机遇的变化动因进行分析。
　　我们知道，要想准确地把握机遇，除了对机遇的既往运行轨迹知晓外，更重要的是要搞清机遇变化的原因及以后运行的方向，从而使自己在今后的行为中，能够随时和机遇的运行节奏合拍，以便永立机遇的潮头。
　　故此，首先要明白机遇迁移的原因。从时空面上可以看到，机遇线上的任一点都是一次机遇，也就是说，机遇的任一次表现形式都对应着物元上的一个机遇表现物。故此，机遇本身就是一种能量，它并不能单独存在，必须借助一定的物元对象来表现出来。随着时间的推移，它又向另外的物元对象迁移，以至于完全脱离原来的物元对象，实现一次机遇的转移。而机遇的每一次转移，都同时伴随着一次机遇物的从无到有、从小到大，又从大到小以至消失的过程。据此，机遇的特点是催生某一事物使之产生、壮大，然后再衰减甚至完全消失。当某一事物衰减之时，也正是机遇正在转移之中。
　　根据机遇的变化过程，我们可以看到，机遇所钟情的表现物，正是代表着事物的未来发展方向、具有远大前程和强大生命力的那些事物。从以人为中心，人为天地之主，万物为人所用的人本宇宙论的观点来看，无疑那些在不久的将来，能够为人类带来巨大利益和幸福的事物（真实的或虚拟的），正是机遇下一站所追踪的表现物。
　　因此，当我们追踪机遇时，要时刻注意那些代表未来方向、可给人类带来巨大利益的行业或事物，因为机遇追求的正是未来的人类最大利益物。

关于机遇的绝对存在论和相对公正论

　　根据前文的论述，我们知道，当机遇物出现时，它就是客观存在的，这对于任何对象人或物来说，都是绝对存在的。然而，由于每一对象的素质及主客观条件不同，不同的对象对机遇的把握度是不同的，致使机遇在不同的对象身上产生的能量是不一样的，故此，我们又说，机遇对于对象来说只能是相对公正的。这有两方面的因素：一者是客观的非己力人为社会因素。比如不同地域、行业或国界下的政策、法律等种种限制。再者，对象人的主观愿望及其客观成长历史背景中已固化成个人内在一部分的自身素质。这两种因素，有的通过人的自身努力可以改变，有的通过自身努力一时难以改变，甚至就不能改变，种种情况都限制了机遇面前的竞争者并不是在相同的起跑线上竞争，所以，我们又说，机遇对于竞争者只能是相对公正的。
　　鉴于此，任何对机遇要求绝对的公正论者都是不太现实的空想主义者或消极抱怨论者。

与机遇同步

　　机遇作为一种特殊的物质，他具有能给予与之同步的任何物质一种能量，使之滋生、成长、壮大，这种同步的物质就是机遇物。然后旧机遇物上的能量逐渐转向新机遇物，使得原有同步物质滞后于新的机遇，最后逐渐走向衰亡。此时，新的机遇物已经形成。机遇的这种不断的从一种物质向另一种物质迁移的特性，我们称之为机遇的变迁性，这就是机遇的根本运动规律。
　　所以，任何事、物、人若想成功，必须不断的给自己补充能量，以便有能力判断出机遇，并抓住机遇，从而实现和机遇同行的理想。故此，成功可以用一个简洁的公式表示：
　　成功=1%机遇+90%汗水+9%天赋
　　这表明人生中，有90%的时间和努力都是在自我积蓄能量，提高自己判断机遇并抓住机遇的能力，以便当1%的机遇出现时，才不至于造成遗憾和惋惜，所谓俗语所说："养兵千日，用兵一时"。

机遇的永恒波动特性

　　虽然机遇是一种能量，而且这种能量是永恒的，但它却是以波的形式不断的向前传递的，在传递过程中还必须依附一定的载体或介质，即机遇物或称机遇媒才能运动，所以，我们要想取得成功就是要不断地捕捉并抓住这种机遇媒或机遇物。
　　以上论述了机遇的各种特性及其变化规律，是我本人多年来对机遇的观感,在此推出,以便人们在实践中及时准确地抓住机遇，促成事业早日成功。

1999.4.22于北京知村里

北京初闻录

　　初到北京，第一个感觉，就是北京人大门口前卖假毕业证和身份证者较多，而来到中关村卖盗版游戏带者应接不暇，他们主动迎面私下询问，对人极有耐心，如果你一答话就会纠缠不休。可见假盗者多矣！究起源，也许如此的成本及其低廉，相反利润就颇为丰厚。
　　第二个感觉，就是这里的求职者众多，来自于四面八方。故此，北京的住宿费及其昂贵，为了降低单位住宿费用，求职者则以长期包房的形式向旅社租宿，相对来说，这也是外来打工者在京能够居住的一种方法。通常单间内有八个铺位，每一铺位每月260-300元不等，而单独居住一间打工者往往是租不起这样昂贵的房租的，就在我所住的旅社里大都是如此。
　　第三个感觉，这里的就业机会相对较高。昨日上午在北京市民族文化交流中心北京市人才市场分部参加了周三的人才交流会，由于来得较晚，约11点30分左右，主流人群已基本散去，但就所需职位行业来看，北京的新闻、计算机、营销人才的需求相对较高，也许这是因为北京为全国的政治、经济、文化中心所致吧。但由于近年来国有传统经济的全面停滞，目前从产业的角度来讲，唯有计算机信息产业是最具活力的一个行业，未来计算机通信、英语、生物技术将拥有无限的胜机。

1999.5.23于北京沙河

流落北大

　　今日上午计划应聘的两个单位，一个是海淀图书馆二楼中和讯电脑培训公司招聘的图书编辑一职，因专业所限被拒；第二个是一文化发展中心招聘的文史类图书编辑，据报载和电话联系，位于北大西北芙蓉里小区附近，但我围着小区转了整整一圈，也未见其公司，足见其单位实力也并非很是雄厚，时近11：00，故此，进入北大校园暂休。
　　初入北大，的确恍若进入了花园，我看不出这里有一点学校的味道，那幽静、葱郁的小路，及个别骑车穿行的行人，使我不相信这是一个校园，而感觉这就是一个公园，因为所有的楼房都是仿古的飞檐倚角结构；再往里走，又见一湖泊，查地图方知此乃北大校园内的未名湖，就在未名湖畔，有个别的学生和行人在湖边凉风习文，我便也在此稍憩。
　　未名湖，未名湖，在这里我才真正地感受到能在此学习的北大学子的确是一种幸运，而这里的一切的确给北大学子增添了诸多的灵感，比其人大那真是远胜一筹。
　　校外夏日炎炎，酷暑难耐，这里却凉风习习，微波荡漾。极目远眺，弯弯的垂柳随风摇曳，与其同时，唧唧的蝉鸣声和远处不时传来的喝——呼——喝——呼——的鸟叫声，使这里的一切完全摒弃了都市的喧嚣，我也忘记了诸多的烦恼，仿佛不知校外有"汗"，无论"胃馑"。
　　下午，我就再访问两个单位，如此，每天三四个单位，尽快把工作落实好，现在的目标就是先找一份有基本工资的工作，然后再说别的。

1999.6.3 12:06 北大未名湖畔

第五章 后记
