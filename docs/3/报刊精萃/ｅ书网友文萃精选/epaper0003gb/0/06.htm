烟 在 指 间

一凡

　　我是一个烟民。五岁时抽了我的第一根烟，那是一个不负责任的亲戚给我的。

　　初中时开始从老爸不设防的烟盒里取烟。高中，有了更大的压力，也有了更多的时间和空间，烟成了我生活的一部分。在大学的宿舍里，烟是唯一能伴我面对枯燥课本和无聊生活的情人。从国际装三五到纯万，从纯万到双喜，我现在只抽软盒椰树，烟的价钱越来越低，买烟的钱却越来越多。我讨厌烟味，不知道地道的烟民会否因此而否定我，反正我是讨厌烟味。有很多人抽烟是为了那句广告词：投入真正男人的世界。烟成了他们的必然装饰品。有些人抽烟是因为有人不想他抽。我的烟，给我一种淡淡的满足，淡淡的释然，淡淡的自怜。如果觉得烟可以使人放松，使人飘然如仙，那实在太傻了。深深地吮一口，烟云掠过喉头，经过气管支气管，撞到肺泡壁上，带来一丝丝的刺痛，干涸和不适。长长的舒一口，烟雾迷蒙了双眼，真实的世界不复存在。空虚的心只会更空，心痛的人只会更痛。没有人会可怜你，只有自己可怜自己。这世上还有什么比可怜自己更叫人心碎绝望呢？

　　烟之所以美，在于它的惨淡，在于它对身体的荼毒，在于人们对它的恐惧和斥责。没有烟的日子，我裸露于众人目光下，看着世界把我胡乱摆布，听着旁人说三道四。与其让别人伤害自己，不如自己伤害自己；与其让别人可怜自己，不如自己可怜自己。我选择了烟。

　　那个女孩告诉我，我指间的烟是她不考虑我的主要原因。我笑了笑，说，算了吧。朋友问我，既然那么喜欢她，为什么不能为她做点实事，把烟戒了。我也只有笑笑，无话可说。心里清楚得很，吸烟是自己的事，戒烟也只能为了自己。可能会为了一个催人向上的故事戒烟，为了一个叫人活下去的理由。而我最希望会有一天，在那个告诉我生存意义的女孩耳边，轻轻跟她说：“我戒烟了。”
