迷失的爱情

lwj

　　刚下了一场雨，天气闷热异常，康在这样的一个午后又不可抑制的想起了眉，那个清纯可人又有点娇柔的女孩。眉是康大学几年的女友，最先吸引康的是眉的背影，颀长苗条，留着质地很好的长发。康于是和朋友打赌一定要追上她，接下来的几年康统治着这个有着绝好身材的女孩，常搂着眉的纤腰趾高气扬出席各类朋友聚会、舞会，让班上的哥们羡慕不已，都说这小子艳福不浅。然而康知道自己不是一个能忍受麻烦的人，毕业了，面对复杂的调动手续，康放弃了眉。回来后，康常不由自主的想起眉，康为自己常思念眉而烦燥不已。刚毕业那会，眉时常会打电话给康，然而康总是以很忙来搪塞眉，讲不到几句就挂了，以后眉的电话有就少了。

　　“铃铃铃……”电话一个尽的响着，康只好从无谓的想念中走出来，“谁呀？这么催命的打电话。”“康，是我，雯，今晚我值班，你来陪我好吗？”“反正反正闲着也是闲着，好吧！”“那好，我在值班室等你。”搁下话筒，康有一种想打电话给眉的冲动，不知眉现在好吗？

　　“最近比较烦，比较烦……”，康哼着小调出门。雯是康新结识的一个妞，没几天就对康一往情深，有事没事爱找康，康也正需要有人来冲淡自己对眉的想念，两人一拍即和。雯今天显然精心打扮过，穿着一件较透明的真丝连衣裙，隐约可以看见内衣。两人有一搭没一搭的闲聊着，……“怎么，这段日子在家修身养性呢？”“男同志夏天最好待屋里，免得犯错误。”……康心里知道雯大概期待发生点什么，两人坐得很近，康似乎已经感觉出雯的呼吸有些急促，月光下，雯显得很甜静，康不由得搂过雯，寻找雯的唇，雯积极得配合着康，康的手在雯的身上流移……。猛然，康似乎看到眉那幽怨的眼睛，康一把推开了雯，“对不起，雯，我先走了。”康不顾雯的错愕，径直走了出来。

　　“难道我真没法忘记眉？”康在夜里反复的问自己。康清楚的记得他和眉的初吻，是那种轻轻的有如品尝甘汁的感觉，很纯。康闭上眼睛也还可以清楚的看见眉那诱人的胴体，曲线玲珑，康可以回忆得出眉那对小巧而令人心醉的乳房，以及那令人愉悦的神秘地带。在学校期间，康公然的在校外租了一处小屋，两人吃在一起，睡在一起，眉的身体很能激起康的男性欲望，面对眉，康很容易冲动。分手时，眉哭得跟泪人似的，回到家乡，也写过几封情意绵绵的长信，康总是狠下心来不去理会，然而随着日子一天天地消逝，康发现自己依然如往昔一样深深地爱着眉，越是想忘却越不由自主地想起眉。

　　“也许我该联系眉。”康想。
