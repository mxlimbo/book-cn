秋 悟

丑草草

　　太阳一点点向西沉去，大山投在地面的阴影越来越小。空气已不是那么灼热了，潺潺的小溪变得幽清起来。

　　段老汉拄着根木制的拐杖站在田梗上，眯起眼睛透过老花镜的镜片注视着眼下的一片稻田。

　　饱满的谷粒，金黄色的稻穗，一簇簇一丛丛在泥土的清香味中已有些声嘶力竭的蝉鸣中随着傍晚的浅浅轻风缓缓晃动着。

　　老汉将拐杖往田里戳了戳，被磨圆了的拐杖底触碰到了潮湿而松软的泥土。再过一阵子，等泥土变干了，就可以割稻啦。如果我身子好使，真要下田去大干一场。段老汉心想着，干皱的嘴角泛起一丝微笑。

　　小路那头传来嗄嗄的鸭叫声。一只领头的鸭子一摇一摆的跑过来，伸嘴去啄食冒出田梗的穗子，却卟嗵一声掉进了稻田里，惊起了小小的飞虫，压塌了一大茬稻禾。老汉慌了神，他抡起拐杖在空中虚晃着，嘴里"嘘嘘"的吆喝着。那只鸭子惊慌而又笨拙的爬上田梗，沾了浑身的湿泥巴，一摇一摆的扭着屁股跑远了。

　　老汉挽起裤角下了田，水的表面是温热的，一脚踏下去，泥温柔的向下陷了一个脚印大的坑，泥里凉凉的水便汩汩的向外流开来。老汉佝偻着背小心的扶起那些被压得东倒西歪的稻禾，不一会儿，手上就全是泥了，一种熟悉而异样的情绪向老汉袭来。

　　赶鸭的贩子停下脚步，拿着草帽扇着风："段老爷子，您老今年的稻子不错啊。"

　　"哼，如果没有你那鸭子，就更好了。"老汉在田间的空隙处一小步一小步的走着，上了田梗。

　　"嘿嘿，我向您赔不是啦。那只鸭子就是调皮。"鸭贩子顿了顿，"嘿，老爷子，您老怎么亲自下田了？您怕有好些年没下了吧？"

　　段老汉拂去手上的泥，挺直了脊梁骨说："三年啦。要不是我眼睛不好使，身子不大好，还真要和他们一帮年轻人比划比划。"

　　鸭贩子恭维的笑道："就是，就是。这眼镜是草草新给您配的吧。"

　　老汉的脸稍稍缓和了一点："嗯，这是我家草草从外地买回来的，嘿，听说还是什么'民牌'呢！"

　　鸭贩子道："嘿，可不，名牌呢。老爷子，你看我家顺发和你家草草的事……"

　　老汉甩了甩手，一脸不屑道："你家顺发哪里配得上我草草！我草草将来要做大学生哩，你家顺发是个土包子哩。我草草可不能跟着他吃一辈子的苦。"

　　鸭贩子却不恼，涎笑道："老爷子，你不也是土包子出身么？"

　　老汉像个泥塑紧闭着嘴唇，没有答话。

　　鸭贩子赶着鸭走了。

　　老汉凝望着一片片稻田，轻叹道："土包子。唉，这田地，我享了它一辈子的福，也吃了它一辈子的苦啊。"

　　粗糙而松弛的泥色的皮肤、过得紧巴巴的日子全是这土地给赐的呀。这一生就像个被岁月捏成的土包子，风霜雪雨侵蚀过的痕迹全成了高低不平、长短不一的皱纹被刻在脸上。可是，一年年耕种插秧收割，看着这些稻子像自己的孩子一样一点点拔节长高成熟，心里全是疼爱全是舍不得哩。这田地，我吃了它一辈子苦也享了它一辈子福哩。

　　风吹着沉甸甸的稻穗，像微微泛起的波澜。太阳仅剩下个红弦儿附在山边了，天的那头淡白色的月亮早已升起，周围变得安宁静谧，连蟋蟀都收拾精力歇息去了。

　　段老汉心满意足的咂咂嘴，拄着拐杖转身溶入了暮色之中。远处，传来了妇人家叫饭的声音。
