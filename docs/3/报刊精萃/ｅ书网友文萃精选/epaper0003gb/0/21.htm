《大话西游之大圣娶亲》我的修改篇（二）

心思 （gdxudawei@sina.com）

　　有生以来第一次这样握着一个女孩子的手，他感到这一双手很冷，冷得令人发抖，仙子用她那最后的一口仙气感到孙悟空的存在，她缓缓的睁开了双眼，看到了这一切情景后，仿佛已经找到了她心底中最后的一个答案，微笑着看住孙悟空，用喘息着的声音对孙悟空说："我的意中人是一个盖世人物，终有一日他会踩着七彩云来迎娶我，我估中了开头，但我估不到个结局……"说完，仙子默默地含笑闭上了双眼，同时也随下了头，已经无悔地逝去了，孙悟空看见后，心里面更加激情荡漾，但这样反而会使他头上的紧箍咒收得更加紧，在不得不放手的情况下，孙悟空终于放开了仙子的手，眼睁睁的看着仙子的尸体继续直垂千尺，最终消失在茫茫的黑夜之中。

　　此刻的孙悟空才想起自己的师傅和师弟们还在正下降中的空中楼阁里，很快收拾好不欢不快的心情后，便立刻飞回楼阁中救自己的师傅去了，原来在楼阁中，姐姐因为牛魔王的那"世纪一招"的影响下，为保护唐三藏而受了重伤，正喘着粗气在一旁休息，嘴角还溢出大量的鲜血，看来姐姐也是难逃一劫的了，孙悟空轻轻的走到姐姐的面前，告诉她仙子和牛魔王的死讯后，姐姐只是牵强地笑了一笑，然后姐姐从自己的口袋里拿出了那个并不属于自己的"月光宝盒"，重新交还到孙悟空的手中，并用低沉的声音对孙悟空说："这个'月光宝盒'是我趁牛魔王不注意的时候偷过来的，现在我把它还给你，你们快点趁现在利用它逃离这里吧！这里就要爆炸了，你们快点走吧！"说完姐姐硬是把宝盒塞给了孙悟空，孙悟空用双手紧紧的抓住了宝盒，此时楼阁不断地颤抖着，火势更加凶猛了，孙悟空见时间已经不多了，便一手将宝盒仍给了八戒，叫师傅和师弟们赶快打开宝盒，趁着月色逃离火海；师傅和师弟们都离开了以后，孙悟空回过头来看着姐姐，然后走前一步，并微微弯下了腰，同时伸出了一只手，对姐姐说道："来吧！我们一起走吧！"姐姐默默地摇着头，有气无力的回答道："我妹妹都已经不在人世了，我活着又有什么用呢？我想去找寻自己的妹妹去了，我最后总算是做了一件好事，我知道妹妹是会原谅我的，"孙悟空知道姐姐已经不行了，他听着自己熟悉的声音，但却陌生的笑容，在姐姐的目送之下，通过"月光宝盒"离开了这个充满着仇与恨、情与爱的世界。

　　我再一次见到那好美丽的芦苇啊！那好蓝的天空啊！还有那恬静如镜的湖水；远处还是那一艘渔船慢慢的划过来，顿时打破了湖的宁静，泛起了一阵阵涟猗，传到了我的面前，连芦苇也在微微的颤动，我看了那船一眼，真的，除了船尾的渔夫正在撑船以外，在船头上还站着的依然是那一位姑娘，但我依旧看不清她的脸，于是我眨了眨眼，果然，这次我仿佛就站在了船的侧面，姑娘的脸也看得清楚了，她就是仙子，她的右手拿着一把剑，左手正拿起一些什么来看似的，她的脸顿时沉重了几分，慢慢的抬起了头，向远方更悠远之处望去，她左手上拿着的东西就是我的玉佩（镯）啊！但为什么我依然不能再走到她的面前呢？我多么想亲口对仙子说"我爱妳"啊！可惜我依旧不能，我也不想打破这美好的画面，就让它永恒、永恒、永恒地带着那一点的凄怨吧！我又眨了眨眼，这次我看到不同的景象了，那是茫茫的沙漠，很荒凉啊！远处我又看到了一个人，背着我走的就是晶晶吗？但我已无法再追过去了，我多么想留住她啊！可惜此时风沙已经挡住了我的去路，风沙过后，晶晶不见了，只见仙子就坐在了我眼前的山丘上，她把头转过来，对着我轻轻的问："嘿，你爱我吗？"我只是静静的回答："其实我是骗妳的，"仙子听后，只是默默的笑了一下，然后站起来，走到了我的面前，慢慢的伸出了双手，把那一块玉佩（镯）还给了我，最后看了我一眼，跟着便悄悄的离开了我，很快就消失在那茫茫的沙漠中，现在的我真的很累了，于是我再次闭起了双眼。

　　孙悟空终于回到了五百年后的世界中，此时他就好像发了一场梦似的，突然之间梦醒起来，却发现原来自己现正处于盘丝洞之中，而八戒、沙和尚正化缘回来给师傅吃的东西，看来他们一点也不曾知道那一段漫长而曲折的岁月，孙悟空此时才真正醒悟到珍惜每一段美好时光的重要性，想起以前对待师傅和师弟们的种种不敬，顿感惭愧；孙悟空站了起来，他突然觉得自己手中好似还拿着某些东西，他把左手伸到了自己的眼前，打开一看，原来是那一块玉佩（镯），孙悟空只是笑了一声，然后迅速地把它收起来，很快他们一行四师徒又要继续往西取经去了，在洞口旁孙悟空刚好碰上那个囚犯首领五百年后的后代，他是带着一班乡亲父老来盘丝洞里参观的，还说起那五百年前自己祖先在这里所发生的一切，孙悟空也驻足聆听着这前尘，深感所欠囚犯首领的太多了，在一轮对白之后，孙悟空让囚犯的后代在自己的胸口上连挫了三下，也当是还清了前世之间所欠的债，然后，孙悟空走出了洞口，在众人的异目之中赶上了师傅他们。

　　正当他们差不多走到城门的时候，突然之间前面人声鼎沸，人们争着跑到城门外的有利位置，观看着在城头上所发生的一切，有的还边走边说："快些看啊！要动手了。""他们已经在那上面站了三天三夜了，看来终于忍不住了。"唐三藏知道自己的徒弟是最喜欢看热闹的，于是他叫了孙悟空一人去看个究竟，孙悟空离开了队伍，为了不影响其他村民，他只是慢慢的靠近了城门下左边的人群中，他看着城头上的情景，原来那里正站着两个人，一男一女的，男的站在左边，看上去很熟面口，啊！原来他正是至尊宝五百年后的后代，现在他的名字叫"夕阳剑侠"，手中拿着一把剑，正侧着身，在晚霞的习习凉风中紧盯着站在右边的女子，孙悟空沿着那位剑侠眼神所看着的方向，一直由左边望到了右边，但因为光线及角度都不是太好，孙悟空还是看不清楚站在右边的这位女子的面孔，于是他开始慢慢的往人群的右边移动过去，但自己的眼睛一直是盯着那个女子的，因为他更加觉得，就凭她的身型来看，更像另一个人，慢慢的他终于看到了，于是他停留在一个适当的位置，他是看得再清楚也不过了，只见孙悟空的眼神是这么的激动，表情却是多么的克制，他看到了，那个不就是仙子吗？准确一点来说应该是仙子的后人才对，但孙悟空已经完完全全把她当成是仙子了，只见仙子身上只穿着一套再普通不过的衣服了，手中也紧握着一把剑，不过是女装的，柔柔的长发正随风飘舞。正在此时，男的那方首先开口说话了，他说："我们在这里已经站了三天三夜了，不如就这样算了，我们和谈吧！"女的说："你说什么？今天如果你不在世人面前说'你爱我'这三个字，你是不可能完整的离开这里的，"男的回应道："看来妳死了这条心吧！从始至终，我都根本没有爱过妳，我是不会对妳说'我爱妳'这三个字的，看来我们只有靠武力才能解决问题了，"说完只见他用手迅速拔出了半截剑峰，露出了一道耀眼的白光，看来，一场恶战就要开始了；现在再回过头来看一看孙悟空的表现，孙悟空的眼睛一直都是盯着右方的，而他的表情是显得那么的冷俊，两排的牙齿已经紧紧地咬在了一起，那种炯炯有神的目光使他勾起了很多以往的记忆，看来自己是要做一些事情的时候了：就在恶战一触即发的时候，突然之间城门口狂风大作，风沙使那些凡人都睁不开眼睛，在城头上的那对男女也不会例外，只好马上用衣袖遮掩着脸，这时候一道闪光闪过，城头上的那个"夕阳剑侠"突然全身颤抖了几下，很明显是孙悟空上了他的身了，几秒钟后，风沙停了，仙子缓缓的把衣袖卸下，凡人也都把眼睛睁开，只见孙悟空把那伸出来的半截剑收回原处，然后直盯着仙子，仙子也盯着他，孙悟空微笑着快步走到仙子的跟前，然后干净利落地用双手搂着仙子的纤腰，用最瞩目的眼神，最响亮的声音，对仙子亲口说出了"我爱妳"这三个字，仙子虽感突然，但却相信面前的誓言，世人在城下正看得兴奋，为一对仇人重修旧好而鼓掌，更为一对情侣的最终结合而欢呼，孙悟空看着城下的情景，再看看面前的这个已经不再是仙子的仙子，自己再也掩盖不住那份伤感的感情了，孙悟空紧紧地握住仙子的双手，然后当众和仙子狂吻了起来，吻着吻着，突然"夕阳剑侠"全身再颤抖了几下，孙悟空已经离开了那个曾经属于自己的肉体，此时，"夕阳剑侠"才苏醒起来，发现事情已经发展到这种地步了，于是将错就错，继续让它狂吻下去算了。

　　这时候不是仙子的仙子突觉得自己的手中好像拿着些什么似的，拿起来一看，原来是一块玉佩（镯），她没有说什么，只是望着"夕阳剑侠"，"夕阳剑侠"也看了看她，然后他们几乎同时发现了，在人群当中有一个背影正默默的离开了他们，他们也许根本就不知道，就是这一个浑身是毛，手拿一支金刚棒的人，才使自己从此改变了命运。

　　孙悟空一直背着他们向前走去，师傅们已经走到离城门很远的地方了，猪八戒还不时地转过身来挥手催促自己的大师兄快些赶上来，孙悟空离开了人群，也离开了那一片欢呼雀跃的叫声之中，拿着金刚棒顺手舞动了几下，然后把它托在肩上，随手在衣袋里取出了一只大苹果，在自己的衬衣上擦了一擦后，就放在嘴里大口大口的吃起来了，他走得是如此的轻快，好像已经完全放下了自己心底的所有大石一样，也没有回头再看一眼，因为他已经对那个女孩子说了自己想说的话了，他很快地赶上了队伍，前面的夕阳已经西下，茫茫的沙漠正等待着他们，路是始终要走的，但看来要受的苦难已经不多了，突然，孙悟空停住了脚步，就在那城楼将要看不到的时候向后凝望了过去，之后嘴角边笑了一笑，再把头转回来，继续沿着前方的路走去，很快便消失在无穷无尽的风与沙的彼端了……

　　这一切都已经结束了，看来这一切都只能够用几个字来形容：悠悠岁月风里吹，亲一亲妳转身去。
