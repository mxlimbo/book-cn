在我最美丽的时候，遇见了谁？

安惠

　　寂静的夜，冲上一杯咖啡，在丝丝香浓中细细回味，在我最美丽的时候，遇见了谁？

　　那年花开的时候，你出现在我面前，嘴里嚼着口香糖，你灿烂的微笑像当时的阳光，从此记住了你。

　　那时正值苦闷的高考前夕，而你每天的电话给我最大的鼓励，你总是在电话里教育我要认真看书，而你的和声细语的教育总让我心里甜甜的，想像着你说话时的表情，很想很想见你，而你问我想不想你时，我会干脆的说：不想。

　　我会在看书时不由自主的想起你，很久。在家人的提醒下才惊醒过来，恍惚间像做了一场白日梦。

　　每天傍晚，你会一下班就陪我出去散步，因为我只能这个时间出来一会儿，我们踩着夕阳的影子来到山脚下，然后并排坐在山脚怪石了，看彩霞一点一点散去。你说着各种各样的笑话，让我笑得喘不过气，那时我总很想问你，你还没回家，还没吃晚饭，饿不饿？可又说不出口，怕你会说那句：该回去了。

　　初春的早上，你穿着厚厚的衣裳从外面进来，然后你让我看书，你默默坐在旁边看着我、看着我，然后你说要走了，我的怅然无法说出。晚上睡觉时，我在枕头下发现了一盒巧克力，在这冬季寒冷未散尽时分，我分明感到了暖意融融。

　　时间越来越紧了，你总说你不来找我了，叫我好好看书，可我仍经常看见你，听你温柔的笑语：忍不住要来看看你。

　　一天，你一只手搭在我的肩上，一只手捋着我的头发，目光直视着我，以你惯有的温柔问我：喜欢我吗？我点了点头，你又说：爱我吗？我又点了点头，就往你怀里钻，你把我的头抬起来，看着我，不笑了，对我说：说你爱我。我摇摇头，你用异常严肃又坚定的口气让我说，而我依旧说不出口，笑笑地靠在你宽大的怀里。第二天，你又来了，谈笑间，问我喜欢你什么，我只傻傻看着你笑。晚上，我在日记里写下了我想说的话。

　　时间一天天过去了，你的电话越来越少，我不明白为什么会这样，我给你打了电话，只问了一句话：你还喜欢我吗？你说身边有同事，过会再给我电话，我就坐在电话边等、等.......你没有再来电。

　　我把你送我的东西扔进了垃圾筒，打电话让你把你借给我磁带拿回去，因为你很喜欢那些磁带。你来到我家门口，依旧笑笑的，我也微笑着，把磁带放在你手中，你说还有事，我说那你去吧，转身时，我抬头看了看天空，湛蓝湛蓝的，没有一片云朵。那一天是我最后一次见你，那一刻我觉得自己长大了。

　　从头到尾，你给我带来的都是欢笑，这段日子划上了句号。几年里，我总在心里默念着一句话：我爱你，你的一切我都爱.......

　　每到花开的时候，我就会想起你的笑脸，渐渐地，已是那么不真实，像是雨后的某个晚上做过的一场梦。

　　窗外是夜，微微透着寒意，我披上了外衣，端起咖啡走到阳台上，望着淡淡紫色的天空、月色清凉，手上的咖啡渐渐泠去，仍隐隐透出醇醇香味........

　　这晚，我落泪了，是因为被往事感动，感动的落泪是幸福的。
