改 换

凯子（laikanhai@sina.com）

　　哼，一切都变了吗？不知道。

　　一夜之间就是两个世纪，甚至一刹那间就是两个世纪，所以在广场上接吻的男女，那一吻就是跨越了两个世纪，那一串串的鞭炮声也从上个世纪响到到了这个世纪，我的同学在新千年的钟声即将敲响前一刻给我打来电话贺新年，到钟声敲响后，同学打趣说我们的电话从上个世纪打到了这个世纪，美其名曰世纪会谈，那些早就上床睡觉的没节目没情调的人，那一觉却也不普通，从上个世纪睡到了这个世纪，而且还是千年等一回了，真爽，躺在床上睡觉也可以赶上时髦，这是天上掉馅饼，还是叫做时势造英雄，生在这个时代就是过瘾！

　　人还是那些人（当然那些在新世纪诞生的世纪宝宝可就是新人了），却都是新世纪的人了。电视台里走来走去的还是那几号人物，娱乐圈里跳来跳去的还是那些面孔，那主持人却是要带我们去听那21世纪的歌，和看那21世纪的舞了。缘份来了挡也挡不住，是福不是祸，是祸躲不过，嘿嘿，是拖了谁的福？连垃圾都成了21世纪的垃圾。这就是改换。

　　当然，过年嘛，特殊的日子是有牺牲的意义和象征的，谁人又会没有梦想呢？改换是有的，时钟每跳动一下，世界都在转动，只是有些改换不像在平地里扔下炸弹所起的反应那样神迅，马上就让大地开花，但是那慢慢地变换，时光的流逝，岁月的无情，却也已经够令人感世伤怀的了。

　　时光催人老，多少少年，那美妙的身形变了样，多少黑发变成了白发，在那一片片的白桦林里，藏着多少动人的爱情故事。每一颗心灵都藏着多少心情。一切都藏在那老阿婆的皱纹里。妈妈替我缝枕头，叫我替她穿针引线，然后自嘲的笑道，以前她小孩子的时候是要帮别人穿针，现在是要别人帮她穿针了。妈妈问我有没有交女朋友，然后看我不好意思讲的样子，用过来人的理解和同情笑着说，年少的时光很快就会过去了。我从她微笑的眼神中看出她对自己过去那青春美好时光的回忆与眷恋，以及对我们的美好年华的羡慕。

　　再看一眼我们的娱乐圈，那谭咏麟刘德华关芝琳张曼玉，有的人还染着头发一派天真活泼样，但那不经意间流露出来的皱纹却让他们成了祖父祖母级了，当年的金童玉女们，都成了大叔大妈级，当年的姐姐成了阿姨，而赵薇章子怡也从妹妹变成了大姐大。那后街男孩也已经OUT，把火爆让给了西城男孩。现在的年轻人在听些什么呢？那敖包相会康定情歌已经击不起共鸣了，那纤夫的爱跟COOL一点也沾不上边，我们要的是COOL，要的是HIGH ，我们要听的是那些人类的真性情的东西，最直接的表白，人类最真实的呼喊，而不是那些所谓的艺术的，而实际上是矫揉造作的虚假的令人压抑的东西（混蛋），听到它们怎么也感受不到放松和享受，倒是一种煎熬了。

　　这些改换，自然规律使然。却给人类留下了一串串的惆怅。当然，科学家们都在预言，由于人类基因图谱的问世，人类可以活上几百岁，上千岁，这将会是一种什么样的改换了。

　　前面我们提到了垃圾，这垃圾很不幸地却让我想起了彭丽媛在舞台上用她的那副甜美的歌喉所歌颂的《我的祖国》。"为了开辟新天地，唤醒了沉睡的高山，让那河流改变了模样——"是啊，为了经济的发展，我们砍光了高山，哪里还有豺狼的踪影，一条大河波浪宽却在时不时的断流，那污水让那长江黄河改变了模样，这就是我的祖国的新解，你还敢唱我的祖国吗？你还敢跟着江泽民先生在那迎新晚会的结尾一起高歌我的祖国吗？我是从此不敢再唱我的祖国了。垃圾是跟着鸡犬升天了，升级为21世纪的垃圾了，只是不知道这21世纪的垃圾会是什么样的，会不会比20世纪的垃圾更可怕，那核废料不就使20世纪的垃圾比起19世纪的垃圾要可怕得多。

　　——

　　改换吧，慢慢地去吧

　　不管怎么样，新世纪总该有一个美好的梦想，我才是花样年华了，所以我的梦想是一段爱情。那就让我唱着那支情歌来走进新时代吧，那首动人的歌儿也叫《改换》，这才是最美的改换：

　　"因为这小屋中有一位姑娘，
　　所以他的简楼变成了天堂，
　　因为这姑娘有一颗芳心，
　　所以他的憔悴变成了轻畅，
　　爱把一切视野改换，
　　刹时丛林虚幻，
　　是真爱才是生的永恒。"
