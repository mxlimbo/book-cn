怀 念 祖 母

沉舟
（zhouyuanxin@mail.ahwhptt.net.cn）

　　祖母的去世，是二十世纪我们全家人心中永远的痛。

　　我的祖母是1994年底去世的，迄今一晃近6年时间过去了。可以说，在我们的脑海里，奶奶的音容笑貌依然如昨，我们一刻也未曾忘记过她……。

　　谁人没有祖母，可谁的祖母与我的祖母相似乃尔！我的祖母是13岁时就到我家做童养媳的，整整服侍我们五代人呀！她这一生吃尽了苦头，可却没有享几天福.我的奶奶是个异常勤劳而坚强的人，由于爷爷英年早逝，奶奶便一人靠自己辛勤的劳动把当时才六岁的爸爸和二岁的叔叔拉扯成人。奶奶曾经对我说，一位算命先生给她算过命，说她是"一天挑水十八担，一天忙碌到天明"的命，是干活的苦命。的确，我的祖母以前什么苦都吃过，什么活也干过，唯一的目的就是想把两个孩子培养成人；一天到晚没黑没夜的给别人干活，什么纳鞋底、挑塘泥、采茶、卖饼卖毛豆腐、赶猪斗呀，什么活没干过。终于盼星星、盼月亮，含辛茹苦、呕心沥血地把两个孩子都培养成人，在家境困顿时并不惜变卖家产让孩子读书，这对于一个没有上过一天学堂的旧社会出来的妇女来说，是相当不易的。左邻右坊看到我们祖上的那点家产都变卖了，甚觉可惜，都来劝她，而我的祖母却说"我卖家产是为了给儿子读书，我想老祖宗是不会怨我的；现在家里的东西卖掉了，以后等我儿子读书出来，我再把那些东西赎回来。"在旧社会，一个目不识丁的妇女能说出这样的话，是很有远见的。我的祖母不但人很聪明而且还非常善良，自己日子过得并不富裕，但只要看到别人有难，她都愿主动相帮。有一年的冬天，乡下的一位穷苦人家房子不慎着火烧了，她得悉后二话没说就把自家的一床棉被和一套炊具给别人送去；住隔壁的人家破旧房子倒了，一家老小十几口人没处安身，她便主动腾出房子让人家住下，一住就是十多年，也从未要过别人一分钱房租。她的慷慨好施、乐于助人赢得了邻居们的普遍好感，所以她的人缘极好。我清楚地记得，在我祖母去世的时候，整条街上的人们都主动前来慰问帮忙；出丧的那天，整条街上的人都主动出来送行，场面令人十分感动的。那一幕我至今仍然历历在目。

　　我的祖母是突然去世的。那一年，祖母已是86岁的高龄，身体已经不太好了，再加上前不久坐竹椅上不慎坐在了地上，大腿骨头便像炭一样断裂了；一卧床就是三个多月，而后又搬到休宁叔叔家去住了，而且还特意请了一位阿婆来服侍她。我是祖母一手把我带大的，因而对祖母有一种特殊的感情，而且这种感情已远远超过了我对其他任何人的感情。于是，我每隔两个礼拜都要骑车去看她，并顺便带点她老人家喜欢吃的食品。可就在她老人家去世前的那个礼拜，我不知是因为工作还是其它什么原因就没去了，噩耗突然从星期一早上传来，我不禁感到五雷轰顶，刹时潸然泪下。事后叔叔告诉我，奶奶在临死前，不断地呼喊着我名字，她是觉得我人太老实，不放心我呀。祖母呀，请您原谅你这个不孝的孙子吧！在你临死前，也未能在你的身边陪伴你，侍候过你一天。此事虽然已过去多年，每每想起我仍然心存愧疚。

　　由于父母长期支边，我们姐弟三人主要都是跟随奶奶长大的。我是长孙，上有个姐姐，下还有个弟弟，所以奶奶也特别溺爱我；我与姐弟之间发生争吵，奶奶也总是喜欢站在我这边，无原则地袒护我。我们姐弟三人是1970年回到故乡休宁万安的，那时中苏关系紧张，说是可能要打仗，政府便动员遣送我们回家乡。回来时，我六岁，姐姐十岁，弟弟才两岁，尚在襁褓中。父母亲在新疆支边也始终调不回来，我们姐弟三人便在奶奶一泡屎一泡尿中带大的。那时我们一点也不懂事，奶奶为我们吃了多少苦受了多少累，现在简直无法用语言来形容了。我在故乡万安跟随奶奶一直到高中毕业，祖母在生活上对我们关爱备至、呵护有嘉真是无法用言语来描述的。她每天为我们做饭烧菜，为我们洗衣服、料理家务，由于那时父母工资不高，经济还比较拮据，我们自己还开辟荒地种菜，自己养猪。我记得，那时我们姐弟跟奶奶四人在一起生活，父母每月从边疆给我们寄65元的生活费来，除留用家用外，每个月也就几十元的生活费罢了。那时，大家生活都很清苦，我们有这点生活费就已经是让别人羡慕的了；那时什么都凭票供应，我们全家每个礼拜能吃上一次肉，就已经是很不错了。由于奶奶的省吃俭用、勤劳持家，使我们的日子过得也还算平坦。由于我们姐弟三人都要上学，每天奶奶都是起早贪黑地为我们张罗着，她那老人家操劳的背影至今都无法从我们的脑海里抹去。

　　真是"天有不测风云"。1974年的时候，奶奶吃饭突然难以下咽了，平日只能吃些流质的东西，后到医院一检查，经确诊为食道癌。那时，我父亲恰好在杭州进修，结果奶奶被送到了浙江医科大学附属医院动了手术，主刀的正是我父亲的昔日同学、著名的心胸外科专家严志焜教授，手术进行得很顺利。奶奶在杭州住院的一个月里，可苦了我们在家的几个孩子。我记得有一次，下午放学回家生火做饭，姐姐好不容易把饭烧好了已是万家灯火，可我们烧得还是夹生饭，可我们却吃得很香。经这次食道癌手术后，奶奶还活了二十多年。

　　奶奶是个异常聪明、善良和有主见的人，她常跟我们讲做人的道理，她说，一个人来到这世上便是来吃苦的，而不是来享福的。一个人在世上，看到别人有困难要想尽可能地帮助别人；人活着要多做善事好事。如果要说这方面故事的话，简直可以写厚厚的一本书了。奶奶对事物深刻的认识能力和判断能力是惊人的。妈妈跟我讲过一个故事，那还是在文革期间，正是"万岁不离口，语录不离手"的林彪红的时候，有一次看电影回来，奶奶对家里人说"那个林彪长得就像个奸臣相。"那时，一句不慎可就要遭至杀头之祸的，当时吓得我妈忙对奶奶说，你可千万别乱说了，万一让别人听到可不得了。妈妈至今还说，真奇怪，奶奶怎么看事情这么准呀！

　　奶奶的故事是永远都说不完的，对于奶奶的这篇文章我一直都在心里写着，可我总一直不敢下笔，因为我总担心写不好，而且也不是三言两语所能写完的。在描写奶奶的时候，我感到语言是那么地贫乏和单调。奶奶虽然离我们而去了，不知为何，我总觉得她似乎还总生活在我们的身边，在某个的地方在看着我们；在我们遇到挫折和不顺心的时候，总是在教我们做人的道理，给我们以生活的信心和力量。

　　奶奶，清明节又要快到了。我们到时再到坟前去看你好吗？我还有许多心里话要对你说……。
