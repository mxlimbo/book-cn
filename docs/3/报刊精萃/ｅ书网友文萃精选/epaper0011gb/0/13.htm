附带产品的吸引力

凯子（laikanhai@sina.com）

　　人类做事有时候是喜欢舍本逐末的，不管是有意识还是没有意识，是明明白白还是不明不白，也顾不了是傻瓜、昏蛋还是颤头了，总之，或明或暗的，仿佛有着一种快感在吸引着，并牵引着我们。

　　于是，才有了买椟还珠这个寓言。那个精明的古代商人为了促销他的珠宝，为他的珠宝做了些非常精致的盒子包装。没想到有人竟然看上了那个漂亮的盒子而去买他的珠宝，并且他要的仅仅是那个漂亮的空盒子，于是在花了大价钱买下那套珠宝后，便将珠宝还给商人，而只拿走那个让他爱不释手的，讨他欢心的空盒子。对于这个顾客的行为是不是，很可笑，很昏蛋，用不着再讲什么仁者见仁智者见智这样小题大做的话了，不过，我倒是觉得，从某种角度来讲，他的行为并不见得就那么可笑，因为他知道他喜欢什么东西，他要的是什么东西，他只要他想要的东西，他便去要了他想要的东西，那东西给他带来了满足和喜悦。虽然他付出的代价是大了点，或者说他完全可以无须付出这样的代价便可以获得他想要地东西。其实，这个寓言反应了人性的某一个方面的东西，每一个人都会或多或少的，这时那时的碰到这种情况。

　　人类为了想得到某样东西时，有时候是无比疯狂的。

　　所以这个世界有那么多的爱情傻瓜也就不足为奇了。为了自己心中爱的那个人，可以抛弃一切东西，财产可以不要了，王位也可以不要了。一切的一切都变得那样的无足轻重，一切的一切都可以不要了，只要能够和我的心上人在一起，就满足了。哼（冷笑），这到底值不值了，是不是还可以套用经济学里的边际效用的方法来分析，每个人扪心自问。当然，许多浪漫的故事只能躺在文学书籍或电影镜头里面，或许在这时候你还会被那感人肺腑的场面搞得心潮澎湃，热泪盈眶，跟着欲罢还不能。但是过了一会儿，当你回到现实中来的时候，如果这种事情发生在你的身边，你是不是会认为那个人是傻瓜，很不现实了，而且这种情况恐怕也很难发生在你的周围，要不然怎么会人人都在怨叹现在的人太现实，与其说是在找对象，不如说是在找房子车子票子位子。美女都嫁给了有钱人，甚至可以去当二奶当三奶,就因为这样就可以马上得到一切。而那些买椟还珠般的爱情故事也只能留给那些文人骚客一类的风雅之士去演绎了。或者偶尔也可以在那虚似的幻境中沉浸一下。当然，举这个关于爱情的例子，并不是想说这个世界上占大多数的比较现实的人不会买椟还珠，而只是想说，在一般人的眼中，为了爱情而舍弃财产和地位，是那样的不现实，简直就是在买椟还珠了。

　　很多时候，人们做一件事情，关注的并不是事情本身，或者说关注的不是事物的本质的东西，主要的东西，而是将目光放在事物的附属物上，而且或许还是事物的附属物吸引了他的目光，而使人爱屋及乌，才使人将目光投到事物上面来。这样的例子就太多了，到处都是。获许你买一样东西只是因为你喜欢它的包装的精良可爱，就是那个包装打动了你的心，而使你动了购买欲。或许你会买某样商品只是它会赠送一点小玩意儿，或者说是为了那中奖的诱惑而买下了你根本就不需要的东西。所以商家才会那么挖空心思的搞包装，使也浑身解数搞促销。开运动会的时候就要找些美女来跳舞唱歌助兴，以期吸引人们的目光，或许也是这个道理。

　　看电视剧不是在注意它的剧情的发展，而是在欣赏那里面的客厅的布置。想去当兵不是为了保卫祖国而是为了让人看看我穿上军装有多帅，上大学不是为了学知识，而是为了能够多踢两年足球，考个研究生也不是为了那个文凭，而是为了继续我的爱情故事，当博士，出国留学不是为了搞研究，而是为了可以多生一个孩子。有的人就是这样想的，到底什么才是目的，什么才是附带产品。
