　　“我可以坐下来吗？”她没等我回答，就已一屁股坐在我对面。那么多的空位，偏偏坐在有人的位置？

　　她在未征得我的同意时喝了口我的酒，当她把手伸向我入在桌上香烟时，我主动替她点燃了。

　　她眯着眼，深深吸了一口，一会儿，烟从她的嘴里缓缓出来，犹如群蛇乱舞，她的脸亦模糊被在烟雾缭绕中。

　　“你不来一支吗？”

　　我摇头，烟对我来说仍是陌生客。

　　我们坐着，谁也不说话。

　　忽然，她痴痴地笑了，一种诡魅！一丝无奈！

　　“你知道我为什么要坐这个位置吗？”她自问自答：“因为这是我认识他时坐的位置！”

　　莫明其妙的女人，莫明其妙的开场白！

　　“我喜欢这儿，”她好象就为了找个人听听她的故事：“很安静，没有人会来打扰你。一杯清茶可以坐上半天。我是个无业人士，说得好听点是自由攒稿人！”说到这儿，她自嘲地笑笑：

　　“他也常来，跟这儿的老板娘很熟。我们常常会碰到，但是从未有过任何交流！那天他走过来问我可以坐下吗？我有些惊愕，不明白他想要做什么？”

　　“而他的第一句话是‘小姐，你真美！’又是一个无聊的人，走到哪儿都能碰到这号人。我厌恶地看了他一眼，希望他能自动消失！他毫不在意。你知道，”她朝我喷了口烟，我被呛了，她咯咯地笑了，眼睛更加迷离了：

　　“这种男人对付女人很有一套，首先他们必须对心理学懂一点，尤其是对女人颇有研究。

　　他说介不介意今天的帐单由他结？他以为自己是什么/大众情人？慈善家？我不会接受一个陌生人的任何恩惠。他仍不死心，又开始自我介绍并给了张名片。我瞟了一眼：原来是建筑公司老板，怪不得那么嚣张！老板又怎样？如果我阿情想要男人，那还不容易！”

　　这句话不是盖的！她的确是个相当漂亮而且有气质的女人！她有双摄人魂魄的媚眼，但是她的脸看上去又显得有些稚气，我无法形容她是怎样的女人！

　　“但是，此后，我每去一个地方，总能看到他，真有那么巧吗？

　　他每天会送花给我，当然不是直接交给我，而是放在我常去的那张桌上。他以为这种骗小女孩的把戏就可以蒙我吗？

　　后来，我有一段日子没有去那家茶座，不是避开他，他还不至于成为我的心事。我只是去看一个朋友，她是我唯一的朋友，但她却得了绝症！”

　　她幽幽的叹了口气，看得出，她仍然沉浸在失去好友的痛苦中。

　　“那段日子我长时间地陷入无边的伤感中，不知为什么而活，只是莫明的伤心，我可以特别特别地沉静，非常深地陷入一个回忆之中，让自己跟着它难过，走不出来！

　　我没有心情做任何事。有天，我在街上走，经过茶座时，我无意之中瞟了一眼，然后就看到了桌上盛开的红玫瑰。

　　我心一动，推开门，我问侍应生花是送给谁的？

　　就是你呀！

　　是谁？

　　龙先生！他每天都有送花给你，你没来的那些天他也每天送！老板娘说这个位置以后就是你的了！

　　我有些感动了！也许是那时的心情真的很需要有个人关心，有人来陪！

　　我默默地坐在那儿，我知道他一定会来！

　　他果真来了！我们相视无语。

　　让我来照顾你！他忽然握住了我的手。

　　我任由他握着，就象把自己的一生都交给了他。其实那时我甚至不知他是否有太太？”

　　她笑了，笑得比哭还另人难受！

　　“他是有个太太，他太太也才是他们公司的真正老板。他只不过是个傀儡！

　　我很佩服那个女人！她真的是个很聪明的女人，如果她象个泼妇似的跟他吵，或许他会选择我，但是她什么也没说，也没有来找我。只是更加温柔地对他，甚至以她手中的百分之二十股份诱惑他。

　　他是个男人，一个很懂得享受，而且是个已习惯享受有钱人生活的男人，他更加明白金钱和女人哪样更重要！”

　　她的侧影很美：秀挺身而出的鼻子，长长的睫毛，唇有些上翘，很性感！她的头发是未经加工的天然直发，乌黑柔亮！额前的那络发丝垂下来，遮住了她另一侧的脸。那半侧脸在阴影里就显得更加妩媚，更加妖艳！

　　“今晚是我最后一次来了，我从来不在晚上到这儿来，今天是第一次也是最后一次，以后将不会来了！”她的眼中有些晶莹闪亮的东西：“因为我要结婚了！”

　　这句话真让我吃惊，如果刚才的那个故事让我有些难过，那么她的这句话则令让我心痛！

　　“好了，我该走了！”她站起来：“谢谢你的烟，还有你的酒，也谢谢你陪着我！”

　　就如她来时那样鬼诡，忽儿就不见了她。

　　“小姐，你醉了！”

　　有个人在推我，我茫然地抬起头，触到一双困惑的、关心的眼睛，这是个很年轻的男孩儿！

　　“小姐，你没事吧？”年轻的男孩儿小心翼翼地看着我。

　　 不，我不是毒素！也不是发了霉的面 包！不要这样看我！

　　“她呢？”我四处张望。

　　“谁？”他困惑地：“今晚这儿一直只有我跟你！”

　　她留下来的烟味儿还在，她无奈、悲伤的叹息仍留在空气中，但是她不见了，仿佛从来就没有这个女人似的！彻底消失了！又或许她从未出现过，只是我的一个幻觉而已！

　　可是我明明看到她笑，听到她讲的故事？

　　“你已经在这儿坐了老半天，抽光了我一整包烟，而我也被熏了老半天，你对我说了许多话，只是，”他不解：“你真的要结婚了吗？为什么你。

　　 我已听不见他在说什么了。
