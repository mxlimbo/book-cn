这绝望的声音让我心里好难过

xujiang（yuanfan19@263.net）

　　紧了紧身上的衣服，我默然的注视着来往的人群。

　　下雪了……

　　一片白茫茫。

　　早已过了触景生情的年纪，对于雪没有丝毫的感动。一个人在这座城市游走，没有目标，没有方向。

　　距离我要赶的火车还有两个多小时的时间，我提起步子，像一个异乡来的游客审视着这个既熟悉又陌生的城市。风吹的我有冷，我站在中兴商城的天桥上，看着来来往往的车辆，我想起在北京的那段生活，想起在人民大学西门天桥上的那个夜晚，想起曾经的那个四哥，大飞，和那个车莉。想起我们共同走过的日子，想起我们现在的尴尬。我漫无目的的走着，走过兴隆大厦，走过盘锦大酒店，走过紫澜门饭店，走过每一个我熟悉的地方。一切都是冷冰冰的，没有一丝的亲切感。

　　坐在世纪广场的长椅上，点燃一支烟，我看着它一点一点的燃尽成灰，我想起我正在逐渐枯萎的生命，是什么东西在无情的吞噬着它？没有人能给我答案。

　　喜欢这个广场，喜欢这样坐着，不知道什么原因，每次回到这个城市，不管多么劳累都要来这里，来这条长椅坐坐，点一支烟，看着它燃尽。有几次执意带几个朋友来这个广场，我告诉他们："我向往一种漂泊的生活，路途太过遥远，未免有点凄凉。我喜欢在这里坐着，看身后的草坪，看面前不断变幻的喷水池，看人群起舞的翩翩舞池。"我感伤的告诉他们："这么多年走过不少地方，没有一处让我这般的留恋。"

　　身边走过一对对情侣，让我莫名其妙的想起佳欣，有点怀念她诡秘的笑声。我默默的坐着，想起跟佳欣的那段柏拉图似的感情，站起身想给佳欣打个电话，走到电话亭的时候我停了下来，突然之间我觉得自己很可笑。佳欣的笑声像一道魔咒紧紧的箍着我，任我怎么甩也摆脱不掉，有一阵子我怀疑我来这个地方是不是为了佳欣，我说不清楚。我告诉自己：我只是想默默的在这里坐着，没有人打扰，安静的想些事情。但事实上并非这么简单。

　　穿行在茫茫的暮色中，城市像个害羞的孩子隐去了她羞涩的脸庞。我站在喷水池的中央，任凭水滴洒落在我的身上。晚风轻轻的吹过，我裹紧了衣服，有点冷。我绕着广场的华灯无聊的转悠着，我不知道下一步我将去做些什么，迎面走来一个女孩，修长的身材，飘逸的长发，我冷漠的看着她，只是冷漠的看着，直到她回避我的眼神。然后是另外一个，如此反复，我比较着每个从我身边走过的女孩，比较她们的眼神，我企图从这无聊的比较中找到点什么我也说不清楚的东西。

　　我躺在那片整洁的草坪上，看那暮色笼罩的盘锦大酒店，我想起妍，想起我和妍的海誓山盟，想起我对妍的每一个谎言，那都是什么时候的事情了。13楼的旋转餐厅依然闪烁着诡秘的美丽，而对于我那一切仅仅是可望不可及的海市蜃楼。我买了一听啤酒，一个人无聊的喝了起来。啤酒的酸涩滋味刺激着我老不争气的胃。哈一口气，一溜逝去的白烟。九环路在我的身边一趟趟的穿过，我瞟了一眼车上的行人。依旧漫无目的的走着，谁能告诉我：我将去哪里？手里的啤酒被我一口气喝去大半，我的胃开始隐隐作痛。我停下来，跪在地上大口大口的吐了起来，我出了一头的汗，风一吹，打了个寒碜。站起身，我一脚将手中剩下的那半筒啤酒狠狠的踢向空中。啤酒筒在空中划出一道美丽的弧线，"哐啷"一声落在地上，我一个箭步冲上去，潇洒的一脚将啤酒筒踢到马路中央。耳畔响起一阵阵怒骂声。我放声大笑，很满意的走开。

　　悲观无声无息的袭击了我，我沿着马路继续无聊的走着，我希望这个时候有人能陪着我，告诉我该往哪里去。我开始一根接一根的抽烟，走到市交通局的门口，我停了下来，坐在门阶的石梯上，许久许久我这样默默的坐着，我看着每一个从交通局走出的女孩，她们或带着微笑，或一脸的冷漠。我不停的猜测这个或者是那个会不会是佳欣。我觉得自己很可笑，我爬起来，头也不回的走开。我不想给任何人打电话，我陷入一种模棱两可的境界，对一切都很麻木。

　　我折回身，回到世纪广场，在那条长椅上又坐了一会。6点的时候，我起身最后看了看这里。在路旁的电话亭我给妍打了个电话，说了些祝福的话，随后是一阵急促的"嘟""嘟"声。放下电话，我买了一张昨天的体坛周报。找钱的时候，那位大妈多找给我10元钱，我又买了一包烟，转身离开。这个时候，一辆1路车从我的身边驶过，我踏上车，车上空无一人，只有一个快要睡去的售票员和一排排空着的座位。我打开那张体坛周报，无聊的翻看着，不一会，困意袭来，我昏头睡去。恍惚中我作了一个梦：佳欣坐在我的身边，我靠着她娇小的肩膀，昏昏的睡去。她给我唱着歌，边唱还边玩弄着我的耳朵，她的小手是那么的轻柔，我握着那支小手，一时间我感觉我是多么需要她。不一会我睡着了，当我醒来的时候，佳欣已经不在了，一张残留着佳欣体温的纸条被我紧紧的握在手里。突然之间一阵风吹来，将我还未来得及看的纸条卷出了窗外，我惊呼一声，拼命的把头伸向窗外，但是脑袋却卡在窗玻璃中间，怎么也探不出去，我放声大哭。一遍遍的喊着佳欣的名字。这个时候车子一个紧急刹车，我的脑袋一下子撞到了前面的坐椅上，我醒过神来，窗户关的好好的，车子里除了那个售票员一个人也没有，灯不知什么时候被人给熄灭了，一片黑漆漆的，我揉揉脑袋，觉得有点疼。打开车窗，冷风吹过我的脸庞，我想起我做的那个梦，忍不住哭了起来。车子驶过物探南区，驶过油气金河小区，一头扎进了茫茫的暮色中，我想起了我的那帮哥们，想起了那支解散的足球队，想起了那首记不清楚歌词的队歌，想起了和他们一起耗着走过的那段日子，想起了我的初恋。不一会我又昏头睡去，恍惚中车门一次次的被打开，乘客走了上，上了走，售票员不耐烦的报着站名。再次醒来的时候车子已经开上了通往盘山的大桥，路边矗立的华灯飞快的向后面跑着。我的心情抑郁，叼起一颗香烟，大口大口的裹了起来，胃痛残忍的折磨着我的身体，我几次昏厥过去。体坛周报上的消息味同嚼蜡，我把报纸撕成碎片，随手撒出窗外。车子晃晃悠悠的向前行着，漆黑一片不知道到了什么地方。刚刚喝进肚子的那罐啤酒不争气的顶了上来，我打开窗户呕吐起来。车行到商业城的时候，乘客一股脑的全下去了，我想起唯真，鹤成的那帮哥们，下了车，买了一根烤香肠，一杯热牛奶，胃稍微好受了一些。路过唯真，路过鹤成的时候，我想起了小旭，很久没有联系他们了，此时此刻他们都在做些什么呢？

　　7点10分的火车，我起了车票，站台上空空荡荡的，我顺着暮色一路走去，渐渐的连人影都看不见了。下了站台，我掏出最后一支香烟，坐在铁路上抽了起来。我想起和妍分手的那天下午，一个人沿着这条铁路走着，走过的那一片稻田，走过的那一片旷野，我拼命的哭号着，一时间觉得被这个世界抛弃了。上了火车，找到自己的座位。从背包里拿出那本石康写的《晃晃悠悠》看了起来，当我看到阿莱和周文分手的时候，我想起了妍，想起了我所过的19年晃晃悠悠的生活。又翻了几页，悲观和失望像两道枷锁牢牢的捆住了我，我开始思索我的过去，思索我百无聊赖的现在，以及我那遥不可知的未来生活。这种徒而无功的思索让我更加悲观，绝望。我看着车厢内的芸芸众生，我发现我对他们提不起丝毫的兴趣，我用我冷漠的眼神审视着每一个我视角能覆盖的地方，我企图去发现人类的某种共性，但事实让我绝对的失望。我放下书，把脑袋埋向窗外。黑夜向一道灰色的长廊无休止的向前延伸着，紧接而来的是无形的困意，车厢内的每个人都是那么的怪异，所有的人都对交流了失去了兴趣，彼此冷漠的对视着，就像进入冬眠期的青蛙静静的坐着，没有嘈杂的聊天声，没有打扑克的热闹景象，整个车厢陷进死一般的寂静。每个人都固守在自己的角落，或倒头睡去，或读书看报，或听着音乐广播。列车在夜色中呼啸的穿行着，不停的向前奔去，奔去……满脑子都是乱七八糟的事情，我听着郑均的《极乐世界》，昏昏睡去…………不知道这样过了多久，我醒来，耳边是那首郑均的《门》，我想起了妍，想起了那个我曾经深深爱过的女孩，在火车到达沟帮子的时候，我下了车，站在站台上，我在茫茫的夜色中寻找着，妍的影子从四面八方飘过来包围着我。我伸出手去拥抱她，却什么也抓不到，恐惧悄悄的袭击了我，我莫名其妙的朝出站口走去，下意识的抬头看了一眼偌大的站台，那站台上醒目的写着三个大字"沟帮子"。想想上次来这个地方都是什么时候的事情了。一阵汽笛的长鸣声刺破长空，我迅速爬上火车，挥挥手向空无一人的站台告别。突然之间，我觉得我失去了点什么，但是我却不知道我到底丢失了些什么。我反复听着那首《门》：也许本来就不该让我进来，你应该把我拒绝在大门外…………我听到了哭，这绝望的声音让我心里好难过…………
