烟

依丫（blue771108@sina.com）

　　烟飞灰灭……

　　所有的痕迹停留在手指上那抹淡淡的味。烟嘴带着点点红光急速飞旋，画下美丽的弧线迷失在黑夜里。在地面爆发最后的火花就象我嘴角边爆发的笑一样无力。

　　这不是我第一次拿起，也不是最后一次放下。

　　烟对于我并不是忧郁、沉闷、孤独的替代品，也不是我快乐、灵感、源泉的朋友。我只爱它在我手指间冉冉上升的体态和燃烧后留给我的灰。

　　烟飞灰灭……

　　"万宝路"热烈的辣味犹如它外表耀眼的红，让你忘不了来自美国西部隆隆的奔马，带着邪笑粗狂的男人。"555"带来的是微微的苦涩，那是在世界沉没的曾经辉煌的国度传来的苦味。"剑牌"慢慢释放着的辣苦和干涩，"摩尔"里薄荷掩盖着那征服不了的辣，"七星"那股淡淡的不留意的甜……就这样慢慢的品着烟，从轻盈的烟里找寻着不同国度里的故事，体验着烟里那不同的人生。

　　烟飞灰灭……

　　灰会夹着不同的颜色，有黑有白有灰，就象我眼里的世界，每天虽然过着同样的日子可是却有着不同的颜色和味道，哪怕是苦和涩的轮流交替，同样会带给我惊喜、欢乐、郁闷、孤独……燃烧是一个辉煌的过程，烟飞得无影无踪，只独自留下灰证明那曾经有过的精心策划，摄魂猎魄的火。风轻轻的滑过，摔裂成无数的灰，象蝴蝶般消失在地尘里。

　　烟飞灰灭……

　　划在手指间的那淡淡的淡淡的烟味，嘴里呛呛的是苦、涩、干、辣混合的气味，头发里、衣领里、停滞在空气里的味道，可以让我深深的躲藏在里面怀恋着逝去的烟。

　　夜晚没有白昼里的吵喳，味觉那时是最好，其实是偏爱在冷冷的月光里看着手指间灵动的烟和微闪的火光，体会着内心最深处的独白。

　　烟飞灰灭……

　　烟嘴带着最后残存的能量，画着美丽的弧线绽放最后的辉煌，终于在人间熄灭自己
