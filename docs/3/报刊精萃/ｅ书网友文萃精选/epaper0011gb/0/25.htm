网路情哀（之二）

鹰飞雁舞（yingzifei@sina.com）

　　"你打的到太和桥，南头。我等你。"

　　"好的，我马上来。"关了手机。我挥手招的。深青蓝的大衣随我的手势摆动起来，这再一次提醒我此行的隆重。

　　"太和桥。请问先过桥北还是桥南？"

　　"当然是桥南，现在是在城南嘛。"

　　"那到桥北吧。快到桥边时请开慢点，提醒我一下，我想看看桥上风光。"瞩咐完司机，我裹紧衣服。

　　小城干净得有点晃眼，人少车少，静若处子。我想起她说的一句话："来之前可要抖落尘埃，这里干净得容不下异乡尘土的飞吻。"我说："放心吧，我保证梳洗得比上供的猪还干净。"

　　整洁就不用说了， 我现在大可以保证自己象寒雨良一样有型。今天我是城市芳心猎人。

　　见面我该叫她"舒蕾"，还是"沾香的字"。叫网上昵称，象说梦话，不妥。可她的名字跟一种洗发水相同，有次我说我每天都用舒蕾，她气得被一个"你"字哽得半天说不出话。

　　尽管我发誓不会把场面弄得象相亲一样尴尬，可我没有把握，因为我的确喜欢她，那香喷喷的字。大半年的神交，我迷上了她的机灵和俏皮。

　　"桥南到了。"司机的话音一落，我忽然一紧张，象当年参加面试。桥头只有一人伫立，一定是她了，穿着苔色套裙，披着一件黑色大衣。她玫瑰色的嘴唇开得娇艳。她的目光好奇地探进车来，我被灼了一下，脸发烧了，车象逃兵一样静静溜走，载我象偷窥者。我后悔让司机把车开这么慢。

　　我打断好心的司机好心的介绍，停下车来。

　　我向她走去，风吹动我的长衣裳，手里的电脑也象装钱的密码箱一样扮酷。哈，我象周润发一样地来了。

　　我生命中值得纪念的一幕就要展开。

　　她远远的就在笑。不是迎宾的那种。而是好玩，好玩得害羞。我也笑，为我的浪漫念头。

　　"你是13980120338？"奇怪，我居然没叫她的名字，竟吐出这一长串手机号码。她点头，笑。眼神象一双舞蹈的手在我眼前一挥。"我还能叫你阿飞吗？""我习惯，你叫吧。"

　　"阿……飞。你好。"她又笑。

　　 趁她冲着地笑的空当，我的眼睛舔她。对于我这双经常在美女身上歇脚的眼睛来说，今天既没有惊艳，也没有给饿着。我心想着要煽眼睛的耳光，平时太贪嘴，滋养得刁了，让它稍息吧。我鼻子这下机灵了，冲出来报告，她身上有股淡淡的茉莉香。

　　"路上累吗？"

　　"不累。飞机上花了两小时，长途汽车也霸占我两小时。现在是下午两点，可能我只能呆两个钟头。还好，都有一个'两'字，好事成双，吉利！"

　　"你要不要吃点东西？"

　　我本来想说你秀色可餐，算了，在网上是老相识，可她的模样毕竟是陌生的。可陌生中又夹带一点亲切，象是忽然遇到失散多年的一种气味，努力捕捉，又稍纵即逝。

　　我期盼能到一个培育爱情的咖啡屋去坐坐，于是说："是有点饿，你随便给我找个地方来点东西吧。"

　　她带我到了一家拥挤的面馆。"这店儿生意特别好。牛肉香粉、酥脆薄饼，是这里的名特小吃，你该尝尝。"

　　为了那两个小时不被严重剥削，为了她那句'该尝尝'的忠告，我吃，我饿虎吞食地吃！

　　我的周润发衣袖扫了一点面汤。我指着说："沾香的衣，跟沾香的字配。"

　　"你那是吃香的衣！你别太给我面子，你的馋相是在间接夸我哦。""夸？我是要好好夸你。恩，还是留到网上说吧。"

　　吃完后看看时间，连等带吃，花去20分钟。

　　她说要带我逛这个我从没来过的小城。我有点沮丧。我工作的这三年，一直在外奔波，小乡大城，可以说阅尽人间春色，我何苦千里迢迢地来逛这座小城呢，尽管它很干净。

　　"我挺累的，我们找个地方歇歇脚吧。"

　　她恍然大悟似的："哦，对不起，我只想让你别白跑一趟。我真不知该怎样款待你。好吧，前面就有一间茶屋，我们去坐坐。"

　　她真象天使一样可爱。

　　总算找到了我想要的气氛。飘着音乐，蒸着茶香，整个背景象是一张图案温馨美丽的纸签，等着有情人书写情话。

　　我想起以前曾经胡说的一句话："年青的鹰嘴里叼着沾香的字，飞到它梦见的天堂。"

　　字说话了："你怎么尽想美事？要是这个字是个'针'字，你消受得了？"

　　"香针？针香，真香啊。"

　　"真香！这茶真香。"我说。她淡茶似的一笑，若有所思。

　　"你比我想象中要稳重得多，你在网上特别能吹，我以为你现实中也那么油腔滑调，幸好不是。"她好象还满意。不过使我为难了，腹有诗书语自华，能叫我的满腹经纶不开花吗？

　　好吧，我给她讲走四方的奇闻妙事。流浪的人身上都有各色尘土的味道，混合起来就是钢筋混泥土一样结实的男人气概，这对留守的女人有着独特的吸引。《廊桥遗梦》中那个老风流的摄影师就证明了。我比他有优势，我年轻，也有劣势，我的脸皮没他的结实。

　　这一点一年前就得到证实了。

　　那是个令耳膜尖叫的夜晚。和几个同事在酒吧里喝得半醉。色心陡起的他们非要拖我下水，说是趁兴给我举行成人仪式。我的抗议把肚里的酒精都煮沸了，天可明鉴。可是不由分说，我被关进一间小笼子，后来进来一头小兽，我看不清她，只觉得她的味道象酒精和庙里的香混合起来的，太冲，她的脸象面具，我真怕她揭了面具显出狰狞相来。我可以玩她？我是男人呃，不能让她笑话。我对自己说，这是一个女人。她抱着我啦！天，我象是一具等她雕刻的泥胚，来吧，刀。我象粮仓着火。闭上眼，我用嘴拱，她象一条健壮的鱼，在我怀里摇摆，等她消停了，我睁开眼，天啦，原来她想摆脱的不是我而是衣服。她在我身上胡摸，寻找解剖的入口。忽然我弹尽粮绝。心想，成人仪式的香滨也算开过了。我撒开她，有点委屈，说：够了，你走吧。她意犹未尽，我推开她，有点恨她。总算拣了一条命，香滨开得正是时候。

　　那次经历让我的眼睛厌食三月，街上美女的身影没得到我如日目光的照耀，晴转多云。

　　拨开这片乌云的正是眼前这个端庄的女子。我常在看到一个顺眼的女人时想到她，希望某天一张乖巧的嘴在生动的双眼的掩护下说：我就是沾香的字啊。

　　现在，这双生动的眼在一句句不紧要的话的掩护下说：我喜欢你。

　　等我讲完了早已打好腹稿的精彩故事，她象是在蛋糕上面敷奶油一样，惊喜地说哪些地方她去过，和我有相同的感觉。爱相随吗？我心里象农夫山泉，有点甜。

　　两个小时已经过去，我打算冒险再停留一个小时。

　　我想我爱上她了。其实我早就爱上了她，在形神合一的时节，我的爱更完满。我说过，她没有惊人的外貌美，合乎我的理想，因为太美了就显得不真实。我可以打开我的电脑证实给你看，谁认识了她的内在，还无动于衷的话，那才真的是呆子。网络就是这样一个翻云覆雨手，把爱情生产线上的工序轻易调转。

　　拭去了开始的害羞，她显出她的大方和热情，表演她那妙语如珠的口才。以下就是她的语录。

　　"北京象香糟扣肉，切得方方正正，很威武，边上有红枣簇拥，红通通；上海是个玻璃城市，莴笋片一样，晶亮爽脆；广州好挤，象是藕片炒肉末，房子一格格，到处填满了人。"

　　"我喜欢我这洗发水的名字，因为我喜欢研究头发，你看，那边那个女士的头发好整洁，上面还满是梳子的脚印，那个男的摩的油太多，老有虫子冒着生命危险飞上去，我想它们是想上去贴告示：小心路滑。"

　　"茶比咖啡好。茶是慢慢泡的，茶的文化也是泡出来的，泡成了历史悠久，茶的味道，慢条斯理地出来，很文质彬彬。我们很多人的生活也是在泡，什么滋味都是渐渐渗出来的。咖啡是冲，很性急的样子，不比茶来得理智。味道也很冲。喝进去，在血液里冲，我不喜欢这种心急火燎的态度。"

　　听着她的茶理论，我想，我这不是在泡她吗，这个方式她喜欢。可我愿意我们能象咖啡，赶紧冲出点什么，哪怕一句要紧的话。冲有冲的道理。

　　我感觉自己是沸腾的开水了。我准备倾出我的感情，冲上一杯浓浓的咖啡。我提起了开水壶把。

　　她轻轻几句话，象一个忧伤的手势，拦住了我。

　　"本来应该请你去我家的，这么远跑来不容易。"我心想不去也合情合理，谁知道会不会遇上人面兽心的家伙。引到家里岂不是引狼入室？女孩子警惕一点是可以谅解的。再说去家里就要面对她的家人，我不是要有新女婿上门的尴尬？我说用不着客气，我今晚上得赶回广州，时间上不允许。

　　她放慢了语速："家里已经象个货仓，不再象个家了。很多东西已经打包。"

　　"要搬家了？搬到新房子里去？好啊，下次来我就可以参观你们的新家了。"

　　"我们的新家在英国。"

　　我吃惊："怎么没听你说过？"

　　她有点抱歉："因为我想见你，我怕你不来。我从小就爱读书，没谈过恋爱。但是你让我听到了好多的甜言蜜语，我象真的在谈恋爱一样。我感到幸福极了。"

　　我开始摸我的烟，但这身新衣服上没有。我有点不知所措。她骗了我吗？没有，她没让我见面就爱上她，没想到我会有和她长相守的打算（我曾想过辞掉现在这个工作）。仅仅是见面，谁对谁都没有承诺更多。网络的暧昧就在这里。

　　"我爸爸在那边工作好几年了。我准备过去深造。"

　　那么我能不能请她留下来。可是我有这个资格吗？去留学，还是留下来陪这个会说甜言蜜语的陌生人，我想她会象泡茶一样理智地思考。

　　忽然觉得，当我们真正面对时，我们在网上所说的一切一笔勾销。我叫她老婆也好，她说赏我个飞吻也好，都象梦话，你听见了吗？我又何尝听见。

　　在见第一面时，我能要求她些什么？我怎么叫她相信，挂在网上的不是一张羊皮。我又怎敢承诺，我能带给她实实在在的幸福？茶一样的生活需要慢慢泡啊。

　　我试图遮掩自己的失落："没关系，我们还可以在网上聊啊。今天不来，你就是人在英国，我也感觉不出。"我牵强的笑遇上她为难的笑。

　　我们静静坐着，直到她提醒我快赶不上末班车。

　　我仓促地把自己塞进一辆出租车。与她挥别。"阿飞，再见。"她的声音有点抖。我故做潇洒地一笑："再见，祝你一路顺风。"我不知道是不是抢了她的话。我看见她的眼睛里闪了一下。但愿不是眼泪。我转去跟司机说话。

　　车开远了，我嘲笑自己：网络不是神，你不要期待它给你一切。

　　"天上掉下个林妹妹！"我无精打彩的调子让我感到忧郁了。
