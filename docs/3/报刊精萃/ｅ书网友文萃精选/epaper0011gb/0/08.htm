回来的伤

gaoyuan

　　18的花季,谁没有那青涩的回忆、温馨的体恤和迷茫的恋情。内向的我总祈望能有一位恬静的女生轻轻的进入我的生活，可只有18岁的我把梦想梦的太美，在现实的生活中上帝只得怜悯的递给我教训和一片片破碎的心，我无法把它再贴合，使像原先天真欢快的跳动。

　　也许原平庸的我，只品味到友谊的甜美，不幸一时受宠，增添了一份责任感在我的学习生活中，的确令我手足无措，同学的鼓励和鄙人的自勉，使我努力的爬着。就这样，我又有了许多朋友，有更多的同学认识了我，我也慢慢变得爱说话了，可不知为什么一见到她，脸上热热的，舌头仿佛也打着她头上那条鲜艳的蝴蝶结，千言万语都默默的放进了回收站。我不断的为自己创造机会去接近她，去了解她，慢慢的胆子大了，不太懂的脸红了，但还没忘掉不好意思的低头这个动作。起初我们见面时只互相打个招呼，有时托人传个纸条捎封信什么的，时间久一点了，我们也有一同出去聊聊天，只是随便聊聊，应该算不上那个叫"约会"的吧。为了让她开心，我不时的给她讲有趣的故事，不时的送礼物给她，在生活上帮她，在学习上鼓励她，尽我所能去关怀她，对她……对她无限的……无限的……因为我想这样应该是"爱"了吧。可，可我还是错了。喜欢一个人不是见面脸红，凡事唯令侍从，为了让她入天堂，自己下地狱十八层，给自己点自私，都给了最爱的对方，你如何生存……可能有点太激动了，言语中不凡有些偏激，对于18岁的我，也只能理解到这里了。

　　我们，我们相识286天，只有49天是快乐在一起的，剩下的237天都在沉默中渡过。之后的我变的令人不可理会，无名火烧在我朋友们的身上，我可以闻到那淡淡的焦味。脾气变的易怒，开始旷课，在游戏室里消磨时间，自己一个人懒懒的躺在床上，听着伤感的音乐,有时会有泪流下,轻轻的滑落在雪白的枕巾上。我也曾低首写血书给她，可换来的不过是几天的强笑，我不敢再去想天长地久，怕流干最后一滴热血，换来的也只不过是一段带着创伤的感情，随时都有再崩裂的可能，那样的话太累了。现一到冬季，原写血书时的指尖，一一榨裂，反刻在心上，那感觉谁，谁又能感受……

　　现我们早已失去了对方的联系，过着个自的生活，学习和工作使我忘却了那段回忆，但在昨夜的梦中，她对着我灿烂的微笑，我回头望，看是否有别人站在我的身后，答案是：没有。那里只有我和她，我又挠着头慢慢的向她走去，脸上挂着害羞的笑。我抬起头看着她晶晶亮的眼睛，她依旧笑着，我轻轻的牵起她的手，可一手抓空，在我的面前，就在我的面前消失的无影无踪，我慢慢的蹲下，泪水顺着脸颊滴落在这一片洁白的空间……

　　是我左手背上那道因打架而留下的刀疤使我保持清醒，不再冲动，使我冷静的看待事物，让我再安心的学习……我恨且懊恼这道伤疤，自作自受，让情绪主宰了自己，可也是这道疤痕，令我暂时不再释放受伤的心，让这颗心好好的养伤，待日后再为至圣的爱情火热疯狂的跳动。
