永别了，我的爱

echo911（echo911@163.net）

　　今天，我六点多就醒来了。朝阳刚刚升起，天空透出了醉人的蓝，那是阿豪最爱的颜色。阳光为大地染上了一层金色，路旁的树努力地散发清晨的气息。这个世界仍然美好，美好得像是不曾发生过任何事情；美好得让我不敢相信，他已不在。

　　那天，那一个电话，我多么希望那只是朋友的玩笑。可是，为什么开玩笑的偏偏是上天，为什么偏偏要对我开一个如此残酷的玩笑。我愣了半天，没留一滴泪，因为我根本不能相信，他就这样离开了。今天，他的身躯即将被厚厚的黄土覆盖，我不敢去参加他的葬礼，我害怕自己会失去最后的，最后的一点勇气。

　　今天的黄昏格外浪漫，沉重的浪漫令人心碎。我支撑着疲惫的身躯，等待着，等待我们最后的一轮夕阳沉沉地坠入心底……

　　我一直是个执着的女孩，从小时候，阿豪打跑了欺负我的人时，我就执着地认为，他是我最好的哥哥。却不知道，他从来都不希望我只是他的妹妹。一直，我都享受着他对我的宠爱，享受着朋友们羡慕的眼光。我觉得，自己大概是世上最幸福的人了。

　　那一天，枫对我表白了。枫是我们学校众多女生心中的"白马王子"，我不加思索的就答应了，毕竟，枫是一个很出众的男朋友。当我向阿豪介绍枫的时侯，他淡淡的祝福我，眼睛闪过了一丝难以察觉的痛苦。不知道为什么，我的心忽然觉得很不安。

　　第二天，我向阿豪介绍了小柔。小柔是我的好朋友，她的人就像她的名字一样，温柔可爱。我不知道自己为什么要这样做，可是当我看见阿豪和她快乐地说笑，我心中的不安似乎消失了。故事的发展如我所料，小柔成了阿豪的女朋友。

　　那一年，阿豪到了新加坡留学。临走的那个晚上，我没去大家为他办的欢送会。那天正好是枫的生日，那晚，我留在了枫的家。后来，听小柔说阿豪那晚喝醉了，一直在喊我的名字……

　　阿豪去新加坡后，一直没给我打电话，我想，他大概是生我的气了。直到我生日的那一天，我收到了他的礼物――一盒从新加坡速递回来的"肯得基套餐"。我幸福得哭了，想不到阿豪还记得小时侯的承诺。那时的我最爱吃肯得基，所以他答应过每一年我的生日都送我"肯得基套餐"。真想不到，阿豪会比我更加执着。

　　在阿豪去了新加坡一年后，我和枫分手了。枫爱上了别的女孩，而我却怀孕了。我努力的在朋友面前装作若无其事，可是当在电话中听到阿豪的声音时，我却忍不住哭了，忍不住流露出了我的彷徨跟无助。不到两天，阿豪就风尘仆仆地出现在我的面前，像过去我被人欺负一样，心疼的把我搂在怀中，擦去我脆弱的泪水。

　　在医院的走廊上，阿豪握着我冰冷的手，不停地安慰我："不要怕，我会陪着你。"医院充斥着消毒药水的味道，令我昏眩。忽然听到护士喊我的名字，不禁用力握紧他的手，他的声音变的低沉而痛苦："不要怕，我会在这等你的。"我用力吸了口气，跟着护士走进了手术室，当我躺在手术床上时，那位护士体贴的问："要不要帮你买杯热牛奶？"我咬紧苍白的嘴唇，用力摇了摇头。当护士搀扶着我走出手术室时，我看见了焦急得正在来回踱步的阿豪,手上正拿着一杯热气腾腾的牛奶……

　　那天是小柔的生日，阿豪带着一大束的百合和气球出现，吸引了所有人的目光，大家都笑着说小柔真幸福。可是小柔却一把打掉了阿豪手中的花，哀怨的说："为什么，为什么你还是不爱我？喜欢百合的不是我，是她。"所有的人都顺着小柔的手望向了我，我连忙惊慌失措的解释："小柔，不是的，你误会了，我们就像兄妹一样……""你不用说。"小柔打断了我的话，认真的对阿豪说："如果你爱我就把你手中的气球给我，如果你爱的不是我，就请你把气球放掉，从此也放了我。"我不知所措的看着阿豪，他缓缓的松开了手……

　　从此，我一直躲避着阿豪，一直不敢正视这份感情，因为我伤害了我最好的朋友――小柔。我忘不了她的指责，"所有人都知道你爱他，为什么只有你自己不知道？为什么你要把还要把他推给我？"没错，所有人都看得出其实我是爱他的，为什么我自己却不知道呢？终于，我忍不住给他打了电话："带上你的百合花来见我好吗……"

　　平安夜的教堂，到处是拖着手祈祷的情侣，不知是什么时候流传了一种说法，要是一对情侣平安夜在教堂拖着手，许同一个愿望，那这个愿望一定会实现的。阿豪紧紧拖着我的手，说："我们永远也不要分开，永远……"

　　为什么？不是说天主是最仁慈的吗？为什么？不是说平安夜的愿望一定会实现吗？为什么仁慈的天主要从我的身边把他夺走？为什么要发生这场车祸？为什么在经历了那么多磨难后，才让我们永远的分离？为什么？为什么？

　　今天的天气很晴朗，可是我却希望能有一场大雨，让天空陪我流泪；夕阳红得醉人，可是我渴望的却是另一种醉，让我忘记心碎！生命的河流干涸了,河底交错着裂纹,像一篇用古老甲骨文写成的悼词，看不出内容，只有沉重的标题："永别了，我的爱！"
