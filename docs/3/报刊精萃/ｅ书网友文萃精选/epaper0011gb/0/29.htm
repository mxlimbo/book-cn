选 择 默 然

思炜（zhs402@mail.nbptt.zj.cn）

　　世界上没有相同的人,也就没有相同的爱情.有人爱得轰轰烈烈,有人爱得细水长流;有人爱得幸福甜蜜,有人爱得伤痛欲绝.而面对你我的爱情,我除了选择默然,还能选择什么呢?

　　虽然你也不比我大几岁,可按照我们农村的规矩,你是我的长辈,我得喊你舅舅,而实际上,我们没有任何血缘关系,你只是我舅妈的弟弟而已.从我们成为亲戚的第一天起,我的眼睛已经再也离不开你.那一年,你16岁,我10岁.

　　在接下来相处的日子里,我明白我是幸福的.你骑着单车带我到处兜风,你陪着我四处玩耍.你不止一次地抱起我,让我靠在你温暖而宽阔的胸膛上.那时的我曾梦想,能永远地靠在你的怀里.那一晚,我住在我的舅妈家里作客.我先在你家玩,一直到很晚,刚好我舅妈和我自己的亲舅舅都上夜班,只好由你送我回舅妈家.吹着夜风,睡意袭来,我早已靠着你有点儿迷迷糊糊了.但我依然能感觉到,你抱我上床,帮我脱去外衣,替我盖好被子.然后,我感觉到你温热的唇贴了上来,你的舌已经试探着我的反应.我心紧紧地缩住了.年少的我明白,这是吻,而不是单纯的亲.哦,我的初吻!但我又是如此慌张,只好继续装睡.接着,你把手伸进我的衣服里边,你温柔的手在我尚未发育的身上游离着.我无法用语言来表达我当时的心情,只是十年后,我依然对每一个细节都记忆犹新.你并未有更多的行为,见我仍紧闭双眼,就替我盖好被子,关了灯,走了,留下我一个人睁着眼细细回味.在之后的家庭聚会中,你又曾吻过我几次,都在彼此清醒的情况下,都照在亲情的幌子下.

　　然后,你考取了远方的一所大学,而我升入了初中.只有每年过年时,我们会碰上一面.可不知是什么原因,你很少跟我说话,而我又笨拙地不善找话题.我们之间,开始显得有点冷漠.我总觉得你在逃避我,我的心里空空落落的.

　　之后,你大学毕业回到了家乡,开始工作,而我则经历了父母的婚变,考上了师范学校.这些变故更使我沉默.接着,我毕业分配,也参加工作,此时却传来了你有女友的消息.那一刹那间,我听到了我的玻璃心破碎的声音,泪水刹那间在我眼中滂沱.

　　我心灰意冷,接受了身边一个对我极好的男孩.他忠厚老实,并不知道我心中无数的秘密,给了我很多快乐时光.

　　细数时光,已过去了整整十二年,我还能有什么企盼呢?面对这样的爱情,我除了选择默然,还能选择什么呢?

　　正当我的生活日趋平静的时候,又传来了你与女友分手的消息,而且听说你因此不再有恋爱与结婚的念头.我的心,又泛起波澜.

　　由于我的主动,我们之间的话开始多起来了.我发现自己原来一直都无法忘怀往事,发现自己原来心中对你是无限依赖,常常不禁在你面前撒娇,常常发疯似的一天之内给你打好几通电话.可这一份情,却始终没有勇气明诉.我怕,我好怕!我怕会伤害无辜的爱我的人,更怕物是人非,怕你早已忘了一切,怕一切只是自己的自作多情.

　　面对这样的爱情,我除了选择默然,还能选择什么呢?
