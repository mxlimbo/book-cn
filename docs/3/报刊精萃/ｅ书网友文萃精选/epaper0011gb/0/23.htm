感悟人生——分享快乐

杨信伟

　　修长茂盛的小树，清脆的鸟鸣，红绿相间的花草，宁静的小湖……这里所有的一切都在向人们展示着勃勃的生机。漫步在这大自然的美景中，我不禁陶醉了。

　　突然，一幅更美的画面展现我面前。一个身穿白裙的小女孩正坐在小湖边的一个石凳上双手托腮，静静地凝神面对着那恬静的湖面。明媚的阳光照耀下的像绵绵美玉一样的湖面上正漂荡着一艘行进在盛开的荷花群中的小舟。真是一幅绝美的风景画，无法用言语表达的惊喜之余的我赶忙用随身带着的相机把这难得的景致拍了下来。

　　为了向她表示自己的冒昧，我走到了小女孩身边。脚步声惊动了她，当女孩扭过头来时，我才注意到她是一个盲人，秀美的面庞也掩饰不了那因眼睛所带来的遗憾。造物弄人，我不禁深深地为她感到惋惜和不平。

　　当女孩听完我的解释后，那原先满是忧郁的脸上稍稍有了些笑容。

　　"真的有那么美吗？"女孩问道。

　　"真的，非常非常的美。"我给了她一个肯定的回答。

　　于是我就给她讲起了周围的景色、自己所知道的一些旅游景点，同时还不时地讲一些曾经从书上看到的小笑话……我们就这样天马行空地谈着，这时连我也惊奇平时不善言谈的自己竟也有如此长侃的能力。"原来世界这摸美丽，今天是我最高兴的一天……"小女孩的愁容没有了，灿烂的笑容这时早已展现在脸上。

　　看着她那快乐的样子，我突然有所顿悟：原来我的眼睛并不仅仅是自己一人所独有的。
