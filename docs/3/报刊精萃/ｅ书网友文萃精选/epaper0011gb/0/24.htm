网路情哀（之一）

鹰飞雁舞（yingzifei@sina.com）

　　时间是水，泅在其中，有肿胀的感觉。

　　空调在卖力地搬运，煮我的热量。我无事可做，我在工作。

　　他们羡慕我独自享有这间办公室。妈的，这玻璃墙的小杂间，各种噪音抢着冲进来发言，每个过路的人，都把眼睛的脚踩在我身上，领导来了，象把晾衣叉一样把我顶起身来，我装模做样地东晃西晃，象衣服随风做摇摆状。后来我干脆不摇摆了，管它工资是不是做抛物线运动。

　　生产线上传来的声音一定是把我当成重金属乐器在敲，来势汹汹，很恣意的样子，我他妈真想唱首歌，来个好竽充数。脑子里的搜索引擎却出现程序错误，怎么也找不到一首上口的歌。只有一句"让我悲也好，让我喜也好"陷入死循环一样，哄哄的，去不了。

　　我尝试想一些浪漫的事。脑袋里烟雾迷茫，象半熟的水，偶而冒出个气泡。

　　我想她是爱上我了。我吸着烟，吐出来一个拥抱状的烟圈，她就爱上我了。

　　昨晚真是一夜春宵啊。你别想歪了。我可是衣冠楚楚地端坐着的。我们说了一夜的情话。她有一个太骄傲的名字，叫"绝色天使"，天使已经够级别，何况绝色呢，我口水娇艳欲滴。我记得我对着麦克风喊第一声"绝色天使"时，口水把进音孔都堵住了，她说你近香情怯啊，别捂着嘴说话啊，我又看不见你有几颗暴牙。我说我如果有暴牙，也都是因为你，只只朝着你的方向。她笑，铃儿响叮当的那种。女人花开在我艳福不浅的音箱上。

她说你为什么叫鹰飞雁舞啊，让人联想到莺歌燕舞，以为你是个美眉，甚至会联想到鸡飞狗跳，骂你个狗血淋头。

　　我说天使啊，你还不明白，我整天飞来飞去干什么，不就是为了我们能多一点美丽的相遇吗。你的翅膀，我的依恋；我的翅膀，我的机会啊。

　　她奖励我又一个笑。可是还不满足。

　　其实鹰和雁不是鸽子那种"扑啦啦"飞的动作明星，它们是讲究姿态到位的舞蹈专家。每一个动作都深思熟虑似的，有点优美，有点沧桑。我在犹豫该不该告诉她这些，怕她笑我故做深沉。

　　上次有个哥们就呸了我一句，说他妈的鹰和雁能够忧郁，耗子都能认猫作干妈。我只好降级处理，做"扑啦啦"飞状。

　　这个美眉是我用几十句宋词钓上的，千万不能甩给我一个"扑啦啦"。于是说，天高任鸟飞，不能只让天使霸占。落霞与孤鹜齐飞，天使共鹰雁同舞。

　　她说你的嘴可真甜啊，要记得刷牙，小心长虫。我说嘴甜了，虫子找甜吃，只会伤牙；嘴不甜，虫子就往心里钻，伤心。余欲嘴甜乎，余不得已欹。

　　我说的是实话，我的第一个女朋友（姑且这样称呼），就是因为我嘴不甜，说我不够浪漫，不够爱她，就去找甜嘴了，独剩我伤心。我那时候是很爱她的，我记得为了她的离去，我走了一个通宵，走到了很远的地方，冬日深夜的冷渐渐冻住了我的伤口。当两个巡警拦住我盘查的时候，我已经愿意和人说话了。

　　为了不再有那样的冬夜，我开始象蜜蜂一样采蜜。当我吸得饱饱，抬起头来，身边的女孩们都已为人妻。于是我在网络上呕吐。

　　"原来你是个老光棍啊。"有个十八岁的妹妹这样笑我，我发了一阵呆，原谅她的青春逼人，却不能释怀自己的老火柴样子。我的一个哥们大篆就是通过网络找到了他的颜如玉，他说网络上的女人心理不设防，很好进攻，与现实中比，事半功倍。女人是迷信灵魂和才华的动物，让她崇拜你，就胜却人间无数（小伎俩）。

　　我举着大篆教我的"帅哥"，"有钱"两个招牌，飞舞花丛中。有几回眼看要收线庆功了，自己却没了胃口，或者鱼摆了腰身。我数不好到底失恋了多少回，不过是两三天情绪低落，之后又露出鹰爪来，扰得依人小鸟们喳喳叫。

　　绝色天使不是只小鸟，是只大的，象天使那么大。她很聪明，有时候还很幽默。女人少她那样的。男人虽然喜欢大放厥词，让女人做崇拜状，有时候不免独孤求败，希望女人能回应，不只是"呵呵"的呈漩涡卷叶，有去无回状。天使垂问我："你吸烟吗？"我怕答错，只说吸烟有害健康。她说："你怎么象个烟盒啊。其实我喜欢男人抽烟的样子，周润发，梁朝伟，都是我的偶像，因为他们吸烟的样子太迷人了。"没想到还有人有"烟情结"。我说我可以吐出50种以上的烟圈，她惊呼了，我以为那一刻她爱上我了。其实我心里想，不同品牌，不同的尼古丁含量，不就是不同的烟圈吗，再加上不同的流速，不同的吞吐量，等等，何止几十种！吹到这里，忽然悟到，女人喜欢烟，怎么也是叶公好龙，来真的就玄。连忙说不是老烟客，偶尔做秀而已。果然得了她的欢心，肯定我不是不良少年出身。良宵苦短啊，就这样打着字，说着话，不觉已是凌晨三点。正是意气风发的时候，我不免得意忘形，说，你做我的女朋友吧，现实的，我这就飞去看你。她这次不笑了，说，我已经结婚了。

　　我的眼皮张大得足以容下那个十七寸彩显。

　　我忘了吸烟，烟烫了我的手，妈的，连伤痛都那么老套！我就不能换个方式受伤吗。

　　是啊，我没问过她结婚没有。天使，并且绝色，怎么还会等我这只大笨鹰呢。互联网看守不严，应该让已婚者一律不准进聊天室，一律不准Q来Q去的。免得我辈窥见人家后院梨花。

　　我还对她说了些什么，不记得了，大致是可惜的意思。其实我不只可惜，我差点哭出来了。做男人真累，需要安慰。

　　我又尝到那年冬夜的滋味。久别重逢，寒入心肺。

　　可我还以为她爱上了我，天使的爱，不体己啊。

　　半透明的办公室还在暴露我的颓废，我真想对每个过路的人喊：你相信网上爱情吗？你他妈美吧！
