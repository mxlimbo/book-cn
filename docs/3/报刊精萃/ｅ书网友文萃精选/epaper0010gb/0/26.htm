放在音乐盒上的巧克力

文茜（szadam@sina.com）

　　这些天我买了一大堆的电池放在家里，就是等着你回来，看到――放在音乐盒上的巧克力。

　　差不多已经快有一年没有见到欣儿了，闭上眼睛就能够想象得到她的一笑一颦，眉头紧锁时的天真，开怀大笑时的烂漫，噘着嘴巴生气的样子。也许你不会体会到我现在的心情，失落、空虚、寂寞，一切都是源于对欣儿的思念。无数次在心中呐喊：我的欣儿，你快快回来吧，你怎知我有多么需要你。

　　还记得欣儿走的那一天，眼中挂着丝丝泪珠，眼神中充满了对我的愤怨，走的时候撇下了一句：你要的不是我。还有那最后的只言片语，放在素白的纸巾上，以前这是我们交流彼此最好的载体，可现在它成了欣儿对我进行最后声讨的武器了。纸巾上还能依稀看到欣儿那双伤心透顶的双眸，还能感觉到欣儿淡淡的体香，看着她写的每一个字，我的胸口都在拼命地滴血：

　　阿毅，难道你心中真只有那一大堆程序代码吗？那我算什么，我已经受够了。我在你面前就象是若有若无的影子，你累的时候，需要我跟你冲一杯咖啡，我冲了；饿的时候，需要我跟你做可口的饭菜，我也做了；成功之后，你说要庆祝一下，我会买来香槟跟你共同分享。然而，认识的这么多年里，你有没有真正的关心过我，你生日的时候，我会送你我亲手织的毛衣，而在我生日那天总会听到你对我说对不起，今天又要加班了。在我最需要你的时候，你总不能在我身边。昨天是平安夜，本来打算跟你在一起，可是你又失约了，徘徊在江汉路口，一对对幸福的人儿从身边走过，一时间突然觉得我的幸福早已失衡，我爱你爱得太重，我已经不是从前的那个我。路过街边的小摊，看到了这个可爱的音乐盒，只好买下来送给自己作为今年的圣诞礼物，就好象是在没有情人的情人节里买个礼物送给自己作为感情的寄托。本想将它带走，可是最后还是决定送给你，算作是最后的纪念吧。珍重！

　　往事悠悠，历历在目，曾经的你说过我不知道怎样去哄女孩子，你说你生气的时候又百分之八十不是因为事情的本身，而是因为我的当仁不让；曾经的你怪我这个人不懂得浪漫，不知道在你生日的时候送给你六种颜色代表六种不同含义的水晶之恋；曾经的你曾经给了我无微不至的关爱，而逝去了你美好的青春；曾经的你……

　　而曾经的我又做了些什么呢？曾经的我，在你面前信誓旦旦地许下诺言要照顾你一生一世，可是你最终选择了离去；曾经的我，只知道埋头写我的程序，把你一个人丢在了相约的地点；曾经的我忽略了你的感受，带给了你无数次的失望；曾经的我……

　　还记得吗，欣儿？你总抱怨我不会在情人节里，象其他情人那样浪漫，送给心爱的人一盒心型的巧克力。听说今年圣诞节也流行送巧克力了，我买了一盒很漂亮的巧克力，就摆在音乐盒上了。这些天我买了一大堆的电池放在家里，就是等着你回来，看到放在音乐盒上的巧克力，旋转的音乐，衬托着巧克力的温馨。

　　欣儿，我知道你一定会回到我的身边……
