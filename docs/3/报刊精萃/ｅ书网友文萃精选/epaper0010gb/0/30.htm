目 标

贺伟（jn2981222@public.jn.sd.cn）

　　现代生活，有太多太多的竞争、每个人都尽量去适应，全身心地投入拼搏的大潮。我就是处在这滚滚浪潮中的一个柔弱的划舟人，我只有支持者那太过柔弱的身体进行一场殊死搏斗。这是我才发现，我的柔弱并不等于软弱，我在顽强的抗争者我的自尊心不准许我体力上的不支，它让我不停的奋斗，直到胜利。

　　当我的眼前是一片黑暗，找不到行驶的方向时，我想起了徐特立先生的话：[一个人有了远大的理想，就是最艰苦困难的时候也会感到幸福]也就在忽然之间，有一盏航标灯亮了！我心中的疑团打开了力量，因为我没有目标，我没强大的精神力量支持着我，没有方向，只能是摸索前进，不知是对是错；因为我没有看见航标灯的光亮，就没有坚定的信念向困难挑战，失去了勇气。

　　当我从迷雾中重新获得方向，重新拥有失去的无限力量，这股力量支持我，鼓舞我，是我振作，我重新我紧手中的舵，向着光明前进，迎着风雨前进！有了战胜自我的目标，我恢复了自信，我相信，在生活的海洋中，我会成为勇敢的舵手，无论使风浪，还是巨礁，都不会让我退缩、徘徊，我会用自己的力量到达金色的彼岸。我只是想告诉我所有的朋友：树立一个伟大的目标，才能投入你的全部力量去实现；如果没有目的地，那么你将失去方向，失去力量……
