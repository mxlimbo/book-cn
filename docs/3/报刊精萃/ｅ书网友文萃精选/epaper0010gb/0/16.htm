　　到杭州已是第二天的两点钟了，女孩的房子在玉泉。车子在求是村里绕来绕去，终于到了。帮女孩拿下拉杆箱，

　　"如果那女孩不跟你好，就来找我吧"女孩笑嘻嘻的仰起脸看我

　　"手机？"

　　"No"我迟疑了一下，"Sorry"，我笑笑。

　　看着楼道里的灯次第打开又次第熄灭，我坐回车里，看看表，三点。

一四

　　 正是西博会期间，城里城外挤满了各地的来客，宾馆很难找，车子在城内兜来兜去，一家家问过去，每家的前台小姐都露出少有的不屑，客满、客满、又是客满。到杭州饭店的时候终于找到了套房，八百八，小姐一脸凛然，没有丝毫让步的姿态。罢了罢了，我退出来。

　　司机一脸茫然，车子在城内转来转去，时间一分一分过去，在家乡深夜的街头，我竟无处可去。
