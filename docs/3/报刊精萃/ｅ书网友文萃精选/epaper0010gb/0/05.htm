没有结局

凌小小（xiao-xiao123@263.net）

　　想起这个题目是在百转千回之后，也许你会不理解。没关系，其实我也不理解。这世上有很多事是无法讲清楚也无法预测的，就像是我昨晚跟朋友讲起的那个故事，一切都可以归为两个字：宿命。

　　不知从何时起，经过校园里的每一个有电话的地方时，都会禁不住的去拨动那个熟的不能再熟的电话。不为什么。是的。但当一些事情真的已经不理智的走向一种极端时，危险已经潜伏在你的生活里。

　　我就是这个时刻可能被危险袭击的人。我宁愿在一个很黑的夜晚，被一个类似于查理的抢劫犯抢劫，从而引发一种很难去预测的生活，也远比这种把心情完全寄托在一根电话线上好得多。是的。在这个时候，归结起所有的事情，惹祸的，只是那根无所不能的电话线而已。

　　我是在网上认识他的。在这个时代里很不时髦的一种相遇，却让那时还是一只准菜鸟的我刻骨铭心。他如一个飘忽不定的幽灵，出没在那时叫做凌波水榭的oicq聊天室里，常常是刚要与他说话就找不到他，这让我从一开始就有了一种欲求不能的痛。尽管后来知道这是因为他那里的教育网很破，不稳定，一个全然不浪漫的原因，但起初的想法却丝毫没有改变。

　　第一次的对话是很有趣味性的。

　　他在一次掉线以后，又挣扎着上来了。

　　我焦急的问你怎么了，他的回答是我晕倒在电脑前了。

　　于是我就觉得自己无可救药的爱上了他。从此陷入了一个及其可怕的漩涡。足以使自己沦陷的漩涡。

　　在看过类似于《第一次亲密接触》之类的书以后，我就幻想着我们是不是也应该有上一次极其浪漫的相遇。即使这个愿望至今仍未实现，我却从未停止过这种幻想。网上过于煽情的文字可以使任何两个异性的人产生对爱情的想象，我想是不是我们也只不过是其中的很平凡的两个。而不是因为我们一直以来生拉硬扯的缘分或是什么与众不同的东西。――虽然这个答案让我有些受伤。

　　那是在给他写过五封没有回音的信之后，我们在聊天室里再次相遇。我们聊了好多，聊的内容杂乱以至于我忘了聊的内容，只记得后来他说，能知道你的真名字么？

　　我等这句话等了好久么？我不知道。不过我要提醒你你在电脑前很无聊时可以注意观察你旁边的一些人，也许他们正在经历着恋爱，一些在这个时候很能感动自己的恋爱。我忘了我那时的神情，不过你要是选择那个时候观察我的话，一定会得出一些跟恋情有关的结论。

　　我们于是就交换了名字，还有电话。那时我刚装电话不久。我很兴奋的把那几个还不是很熟悉的数字敲在我面前的电脑屏幕上。他说好了，我记下了，我会给你打电话的。为了这句话我当晚等到深夜才失望的上床。

　　他的第一次电话是在一个周五的晚上来的，我很惊喜的听到了他的声音。我想我的耳朵后来爱上了他的声音，所以我才会一次又一次的拿起听筒，去拨他的号码。而我本身没有。

　　我们聊了好久，第一次聊天就没有隔阂，这让我有些没想到。放下电话时，我有些紧张。我曾在那个长长的假期里拨过他的电话，传过来的只是一连串的嘟嘟声。这是我所期望的，我那时还没有勇气去听他的声音，只想感受一下离他很近的东西。

　　珍贵是一个人的感觉。你逾爱惜就逾觉得珍贵，而逾觉得珍贵就会逾去爱惜。现在我才知道这原来是一种恶性循环。我觉得我和他之间的友谊珍贵我就去小心翼翼的爱惜，直到把这种友谊培养成了温室里的花朵。它是经不起风吹雨打的。

　　我开始依赖他。在我无助的时候，在我孤独的时候，在我被这个世界遗忘的时候。我无法给他一个确定的定位，好友、好朋友、还是很好的男朋友。我想我不用去管。他常在信中说，这是他第一次给一个女孩子。。或者为一个女孩子。。这种甜言蜜语似的第一次让我开始迷失方向。也许危险就是这时开始滋生的，我没有察觉。不让人察觉的危险才是最危险的危险。

　　那段时间我们去实习，我们住的地方没有电话。这就等于离开了他。

　　条件很苦，我每天给他打电话。有一天下大雨，我站在马路旁边一个孤独的IC卡电话机下，对他说，我好想你。他沉默，然后笑了起来。他似乎不适合这种气氛，我当时想。但是后来，我也觉得我自己可笑。如果我不是把他当成了一个想象中的爱情寄托者的话，何来的想念。我们一直都是这样的距离，何来想念！

　　那天我打家里的电话，却习惯性的拨了他的号码。室友说你晕了吧，还是爱他太深？我笑了笑，习惯，只是习惯而已。

　　我没有想过要跟他在一起。因为太无可能。但在他说了那句你等着我我也等着你以后，我的想法开始改变。曾经有人说我太容易被欺骗，而我却说他的想法太复杂，没有那么多的人会去骗你。其实骗你的人并没有骗你的初衷，但是后来他的话成了谎言，于是他就只好去骗了。我就常常作这种过期谎言的牺牲者，我在一种毫无防备的情况下被谎言所俘虏，然后在他的酷刑下遍体鳞伤。

　　他从不在电话里说情话，他是不是在隐藏他的心情或是因为某种原因不愿表达我也不知道。但这让我误解。我难以想象一个在和我说了三分钟的话后就说困了要去睡觉的男孩是爱我的；我难以想象一个因为时间太久记不起上次是什么时候给我打电话的男孩是爱我的。我于是有了太多不了解他的地方，他也有了太多不懂我的地方。

　　结局并没有出现，一些乱七八糟的事情还在进行着。

　　朋友问我，现在对他是什么感觉。我不知如何回答。也许我期待的是一种很单纯的爱情，理想爱情，把一切这其中所能碰到的挫折忽略不计。

　　我宁愿他没有在那天一时冲动的要我的电话。

　　那么这个时候，在离我千山万水的地方的那个人，我还会为了他伤心和流泪么？
