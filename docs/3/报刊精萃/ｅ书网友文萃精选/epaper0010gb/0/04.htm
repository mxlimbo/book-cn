再见猫儿

鱼儿（susan7777@yeah.net）

　　他是个孤儿，跟着又聋又哑的阿婆长大。安静的世界使得他少言寡语，几乎没什么朋友。

　　他十一岁那年认识了猫儿。猫儿是邻居家的女孩。他第一次见她时，她穿着脏兮兮的裙子，躲在公园的角落里哭泣，象只迷途的小猫。猫儿父母的婚姻是个错误，婚后便为着这个错误争吵不休。每每他们又开始大吵大闹乱扔东西的时候，猫儿就只有跑出来，躲在角落里一个人掉眼泪。那晚，他装作小狗汪汪地逗她开心，还编了花环戴在她头上扮成小公主，把她放在秋千上荡得好高好高。猫儿说她第一次觉得夜晚也不那么怕人了。

　　猫儿有一双乌黑的眸子，总是潮潮的罩着一层湿气，黑漆漆的。他觉得世界上不该有什么人不喜欢这样一个有好看眼睛的女孩，他说猫儿的父母是怪物，反正他是拿她当作宝贝来爱护的，他要一辈子保护她。

　　他没有父母，猫儿有父母却没有时间爱她。有一天猫儿问他，如果你有一个小孩子，你会疼他吗？他说我会，你呢？猫儿说我也会。他说那么我长大了娶你，我们一起疼我们的孩子好吗？猫儿开心地说好，亮亮的眸子象天上的星星。

　　那一年她七岁。

　　他和他的宝贝无忧无虑地度过了六年快乐的时光，几乎天天形影不离。他觉得有自己可以保护她，日子过得很幸福。

　　猫儿十三岁那年轻轻离开了他。她死于心脏病，先天性的。猫儿的父母搬离了小城，屋子里住进了新的邻居，而这些对于他都是麻木的。猫儿的离去，本该改变很多。

　　他的世界一下子空了。

　　那晚他一个人坐在公园的长椅上。一只脏兮兮的小白狗蹲在地上，呜呜地望着他，象是迷了路的。它的两只乌黑的眸子亮亮的，象是罩在湿气里，一动不动地盯着他看。他呆呆地望了它一会儿，说：“猫儿，我们回家吧。”

　　他的宝贝回来了。

　　猫儿成了他的影子，总是快乐地摇着短尾巴，跟着他跑来跑去，一步也不离开。晚上，他就给猫儿讲好听的故事，看见它歪着脑袋很认真的样子，他就眯起眼睛冲着它笑。他也会给它讲他和她以前的事，点点滴滴他都记忆犹新，也会自己莫名地开心或者忧伤。猫儿会把脑袋搭在他的胳膊上，静静地不出声。

　　他十八岁那年考上了一所大城市的名牌大学。那座城市是他和曾经的猫儿梦寐以求的。他们翻看杂志时曾不约而同地一下子喜欢上它，有着宽阔的圆形广场和欧式的建筑。

　　他把录取通知书和城市广场的图片摆在猫儿面前，你想和我一块儿去吗，猫儿？猫儿嗅了嗅，快乐地摇着短尾巴，亮亮的眸子闪着光。可他眼睛里的忧伤，它一点也看不出。

　　他说我还是带你出去走走吧，猫儿。

　　街道在一阵刺耳的急刹车声中静了下来。人群从四面八方一齐拥过来，围着躺在地上的他，和旁边手足无措的一只小狗。它雪白的毛沾了鲜红的血迹，格外刺眼。他是为救猫儿而被车撞到的。目睹的人都说这小伙子肯定疯了，真是可惜啊，就为只小狗。可他们不知道，猫儿，就是他的生命。

　　他模糊地看见猫儿趴在旁边，呜呜地唤他起来。他又看到那双眸子，罩在湿气里一般，黑漆漆闪亮着，象天上的星星。

　　他闭上眼睛的那一刻是微微笑着的，他在心里轻轻地说：猫儿，我知道会有一天再看见你。
