心有千千结

cq（xpychqgyy@etang.com）

一

　　初秋的清晨，我踏着单车穿越在清冷的小巷间。远处，偶尔几片枯叶缓缓飘落，似乎还在留恋着枝头的余温。秋风绪绪，吹乱了我的发丝，也吹乱了我的心。

　　"他过分！难道是我太小心眼，还是他对我不信任——如果我现在死了，他会千里迢迢赶来为我送别吗，他会用双手为我抚平凹凸的墓壁吗，他会直到白发苍苍还在我墓前一遍又一遍重复地拉着〈〈梁祝〉〉吗，他会吗？"我的视线不禁有些模糊。

　　正当满是愁绪时，一辆摩托横冲而来，将我重重摔倒在地。喧闹中，一双厚实的大手把我扶起。"啊，我没死？"我睁大迷糊的双眼，突然间感到了生命的可贵。当跛着脚来到学校，听到老师、同学送上的一句句关切的问语。当看着父母火速赶来时那紧张的神情。我的心竟是暖暖的。就像含着一枚青橄榄，苦涩中荡漾着一丝甘甜。我不再是一只忧伤的鸟儿，所有的不快都已被那浓浓的亲情友情所熔化。

二

　　"通知书来了没有啊，强哥哥？"我直闯进他的卧室。

　　"哦，嗯——"他慌忙拉上抽屉，转向我，目光却游离在窗外。紧张的气氛让我有些不安。

　　"收到了，那做啥还吱吱呜呜的，是不是舍不得那顿肯德鸡啦！"

　　"不，不是的——"他紧张地望着我，"谣谣，对不起，原谅我？"我顿时被他那没头没脑的话给楞住了。"我就快要去北京了。"他的声音有些低沉。

　　"这——，"我开始有些乱。"那浙江大学呢？"

　　"对不起，其实我没报浙江大学，因为——"

　　"不用说了，我也知道，你骗我。"此时望着陌生的他，我的心很痛。

　　"你太小心眼了！"甩下这话，我捂着脸跑出了他的家。

三

　　合上信笺，我脑海中一片茫然。双腿却不由自主地跨进了那片童年的"游乐园"。我沿着河岸慢慢地走着，脚底踩着厚厚的草丝。（尖细尖细的野草稀稀落落爬满坟头。）风中那孤零零挂着几片嫩叶的细枝，楚楚摇曳。天铁青着脸，偶尔一两只乌鸦飞过，发出嘶哑的叫声。心中不免涌起一丝惆怅。

　　低沉的琴声缓缓飘入我耳中，这是一个个久远而优美的音符在跳跃：他牵着我的小手去上学；放学后他递上一串甜甜酸酸的糖葫芦；他手把手教我写那些他也只能写得歪歪扭扭的铅笔字；后来，他偷偷送来一对可爱的娃娃；他生气地扯下我耳中的耳塞，命令我做作业——琴声嘎然而止，美好的思绪也随之飘逝。

　　我停下脚步，对面的他提着琴向我走近，说："谣谣，我知道是我不对，可我是怕你伤心，才瞒着你的。""强哥哥，你别说了，我不怪你。"说完我慌忙别转头，拨弄他手中的琴，掩饰着夺眶而出的泪水。"我知道你还在伤心，"他轻轻地抚摸着我的发丝，低声说："谣谣，原谅我吧，哪怕让我说上一千遍一万遍我也愿意。""强哥哥，"我终于拭干泪水，"我不要你的任何承诺，我只想要回以前的那个强哥哥。""谣谣，只要你能原谅我，我们依然能想以前那样。不是吗？""不，如果再能回到儿时，我一定会把你当成我最亲最亲的哥哥。"泪水再次夺眶而出。他默默地低下头，许久才说："外面太冷了，我们回去吧。"这时我分明看到他的眼睛有点湿润。

四

　　暖暖的灯光下，我打开日记本：

纸飞机
一只纸飞机，
轻轻擦过我的发丝，
在我面前静静飘落。
纸飞机？！
一只洁白的、小小的纸飞机，
我惊喜地望着它。

哦，我久违的大哥哥，
——快上来！
——我怕，大哥哥。
——别怕，抓住我的手。
望着闪烁的星空，
我拍着小手，坐在大树枝丫上，
我摔着小脚，折着纸飞机。

就是这只纸飞机，
我童年的纸飞机，
我轻轻抚摸它，
因为它，
满载着我的笑声，大哥哥的微笑，
——我飞得比你远，
——不，还是我的远。

一只洁白的、小小的纸飞机，
在我面前静静飘落，
我缓缓举起沉沉的它，
它使劲挣脱我的双手，
飞向远方——
不！！
哦！
高飞吧，纸飞机！
高飞吧，我的大哥哥！

　　我心中默默念上一遍，挂满泪水的脸上露出一丝微笑！
