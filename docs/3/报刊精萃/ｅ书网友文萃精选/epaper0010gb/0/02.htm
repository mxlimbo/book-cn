死 亡 婚 姻

zhouyuanxin
（zhouyuanxin@mail.ahwhptt.net.cn）

　　我的婚姻突然死亡了。这使我感到很悲伤，但也很无奈。结婚的时候可能没有人想到过离婚，我也是其中这样一个人；以前，我看到别人离婚，总觉得这些事有些蹊跷，似乎离自己是那么地遥远，不想这件事又突然降落到了我的头上。

　　离婚的要求是妻子首先提出来的。起先我以为妻是开玩笑的，没想到妻的态度是如此坚定而执着，我不禁感到有点震惊。妻对我说，为了有个孩子，我们还是离婚吧。当时，我想这可能只是她的一个借口，我们结婚都快十年了，生活中的什么风风雨雨都过来了，而且没有孩子我也从未责备过她，没想到她在这种时候还要提出离婚。我不知道这是不是她寻找的理由，古人说“不孝有三，无后为大”，而我们结婚十年来没有孩子，我从未责怪过她，我父母也比较开明，对此事也从未怪过她。我不知道她何时有这种离婚的想法，她说，上次去上海做人工授精未成功，我当时就有离婚的念头了。是荷，我不知上天对我如此不公，看到别人结婚后一个个都有了孩子，过上了充满欢乐的生活，而我们两个人只能在平淡的生活中品味枯涩和单调，有时心里也真不是个滋味。可这一切又有什么办法呢？这一切似乎是上天注定的。

　　在上海医院检查时，我们俩之所以不孕的原因，是所谓的抗原抗体关系所致：而因这种原因不孕者，在现实生活中几乎微乎其微；而这不幸却恰恰降落到了我头上，我真是有点无法明白。为了有个孩子，为了治疗，我们夫妻俩数次请假往返奔波于皖沪之间，不仅人财两空，人也被弄得精疲力竭，可还是没有成功，我们原先的那点希望又化成了泡影，我们不禁又跌入了失望的底谷。日子又在一日复一日地过着，生活中没有任何奇迹出现。我的妻比我大一岁，当时是别人介绍的，我们在了解不到半年后就结婚了，婚后俩人感情还好，就像大多数人那样“过日子”呗；不瞒你说，中国老百姓大多数都是凑合着过着，电影小说里宣扬的那种完美的爱情和婚姻在现实生活中可真是难找呀。我们也就日复一日这样麻木地生活着，也不曾想过生活的变故。

　　寒流突然来临了，我没有想到我的婚姻也遇到了寒冬。我发觉我的婚姻就像这初冬里树上的枯叶一样开始凋零了，婚姻的枯萎是没有任何办法可以挽回的，就像我们不能把枯叶重新返青一样。我心中的痛苦是不言而喻的，说实在的，对于离婚我内心是有些矛盾的；结婚十年，无论妻有多少诚意，但人毕竟是有感情的；不管怎么说，妻对我还是相知较深的。妻对我说，不是我对你无情无义，关键是我们在一起没有孩子，而且医生说你是好的，你会有孩子的。这一切都是为了你我都好。我们的婚姻谈不上美妙，但总体来说还是可以过去的。我想，这世界上没有两个完全相同的人，也不会有完全相同的婚姻。难道我们的结合与分手，都是上天注定的？

　　我还是随其自然发展吧。

　　对于大多数人来说，对于婚姻普遍有两种态度：选择维持还是逃避。大多数选择了前者，而我们却选择了后者。我们的婚姻就像枯黄的秋叶一般，已经没有任何新鲜的内容可言；奇怪的是，爱情竟像绿叶一样褪去了。相处十年，应该说我对妻子是了解的，她是一个极不现实的人，这一点也可能就是我们真正产生分道扬镳的原因所在。婚姻的失败让我更多的是反省，这一点也是不容置疑的。

　　这个冬天是如此寒冷，我与她的距离似乎是越来越远。这个冬天我不知在干什么？日子过得是那样孤寂和无助，是那样悲凉和绝望棗在生活中几乎看不到一丁点的希望。天是那么地阴沉，又是那么的寒冷。走在黄昏的街道上，我感到我现在的生活就像这幽暗的黄昏一样，似乎没有一丝亮色，是那么孤寂和陌生；地下暗流的积水就像我心中淌出的鲜血，分裂的地面就如同我裂开的心脏一般。一切都是不可挽回的，一切都是那么地悲伤和无奈。我不知我走得走不出这个寒冷的冬天，也不知前路有多少风雪在等着我。

　　这种“名存实亡”的婚姻使我感到烦恼，可我又实在说不清我的感情，似乎有一种剪不断理还乱的心绪在里面。妻子对我说，你看我们在一起有什么意思呢？我们这种婚姻再维持下去已经没有任何意思了。妻的一番话使我警醒，是的荷，继续维持这种没有多少爱情可言的婚姻是可悲的；可是，当我们抛弃这一切的时候我们难道真能找到幸福吗？我不知道。

　　我的妻不是个现实的人，我们结婚十多年这点我对她是了解的；而我扪心自问，在婚姻这一大事上，我也不是很现实的，而我太注重了外表而却忽略了其它内在的东西，这是我所犯的一个无法弥补的错误，这一教训是值得深刻记取的。我没有想到我们的婚姻会迅速“消亡”，而对这点妻是早有心理准备的。她说，离婚我也很痛苦，可是为了你我都好，为了有个孩子，我们必须要走这条路。话已至此，我还有什么话可说的呢？

　　回首我们这十年所走过的路，应该说我们的婚姻是缺乏深厚的感情基础的；我们互相之间的了解并不够，我对她以前的状况并不了解，她对我也所知不多。当时，她在一所厂办的幼儿园里任幼教，也只有初中毕业。是我们的一位热心的同事介绍我们相识的。至今回想起来，我都感到有点可怕。因为我的妻子是个功利色彩十分重的人，我之所以这样说并不是基于诋毁她的目的，而是有我们十年的生活基础作证的。我们之间的婚姻并不是个纯洁的婚姻，有许多事我在此都无法启齿，姑且还是暂时不说吧。

　　我忽然想起了结婚前谈恋爱时的一件事。那也是个初冬的早上，天空出奇地寂静，空气中弥漫着一股袭人的寒意。我和她走出家门散步来到半山坡上一所干休所大宅院的门口，大门前没有一个人，只有一块石板搭在树下的路旁；我和她就静静地坐在石板上，静赏着这冬天的枯涩的风景。正在我们慢慢体味这冬天的景致的时候，突然一位算命先生手提着一只鸟笼走了过来，我们很觉得有点奇怪，于是我们就姑且叫他给我们算一下；没想到，这种算命方式还很奇怪，我们俩似乎也没说什么，只见那只鸟在主人的口诀指挥下，迅速地在那签袋上跳来跳去，很快就啄出了一只签。算命先生就给我们讲了属龙的应和属什么的相配，现在我只记得他说了一句“龙见兔，看不清。”当时我很觉懊恼，也很奇怪。——懊恼的是本来算命我们是算算玩玩罢了，不想还引出一些烦恼；好在那时我们关系还好，也就没把此事当真了。现在回过头想想，这位突然不知从哪里钻出来的算命先生可真是有点灵了，他竟然说出我和其他属相的没有冲突，惟独不能和属兔的在一起。

　　这个冬天是如此地难熬，如此地让人失意和绝望。我变得似乎什么都没有了，我没有孩子，没有幸福，没有爱情，没有理想，也没有了婚姻。我又变成孤零零的一个人了，我没有了希望，没有了任何前进的方向和动力。我不明白，缘何我的命运总是布满坎坷布满艰辛？缘何幸福总是离我那么遥远？工作不但不顺利，现在连婚姻这把剑也重新砸向我的心坎。我感到我活得这么大，似乎一切都是空的，我仿佛我白活了，白来了世上这一趟。

　　我用手指在敲打着键盘，来描述着我正在结束的婚姻，心里有一种说不出的惆怅。婚姻仿佛就像一团乱麻，说不清理还乱，就像我现在的文章写得没有头绪一样。

　　应该说，我们的婚姻不是没有一点爱情的基础的，我们曾经携手并肩漫步在冬天的寒夜，也曾经共同面对生活的不幸和挫折，所有的一切我们都挺过来了，可是，我们却迈不过冬天婚姻的这道门坎。婚姻生活中所充满的变数是难以预料的，婚姻也不是一团死水，它也会随着生活的变化而变化。

　　婚姻是什么？有人说，婚姻是两个人在走人生之路时， 在走累的时候能够互相在对方肩上靠那么一会儿。现在，我仿佛一个人在一片森林里孤独地行走，没有伴侣，没有歌声，也没有人理解我的举止，更没有人了解我的痛苦；我孤独地在这片森林里行走着，人世间的悲伤和荣辱已离我远去，我孤独地一个人快乐地行走着，这时，突然不知从何处传来一股颓废腐臭的气息厖 。这气味倒底是什么？我也不说不清。这是我昨天夜班下来后走在路上的一种感觉，感觉也真有点奇怪。

　　婚姻的死亡是我无法挽回的，正像爱情的突然消失一样悄然无痕。我对婚姻的失败感到过痛苦，感到过茫然，但我却从未怨恨过。坦率地说，我们结婚十年来，不知经历了多少风雨，她也陪我吃了不少苦，我也没有给她带来多少幸福的日子，这是我心存愧疚的地方。她对我说，你的确没有什么不好，也没有什么对不起我的地方，只是我们没有个孩子啊！我不知她说这话是不是安慰我，没有孩子是不是她的借口呢？我不知道。我感到我的确是应该一个人静一静了，好好想想自己该怎样重新生活了。

　　我知道她是个不现实的人，是个爱慕虚荣的人，但是回过来说，我们男人就没有虚荣吗？现在，我已经落败到了这步田地，她的离去我也丝毫不能怪她。人不是生活在真空里的，我混得不好，不能给她和她家带来利益，帮助她家解决一些实际的困难，我们这种婚姻是无法长久维持下去的。这一点我很清楚，只是她不好说罢了。婚姻的失败让我很苦恼，但也很可能是一种人生的解脱；不知为何，说起这段婚姻，我心里竟然有些激动。是的，我并不想去离婚，没有孩子我也从未责怪过她，我父母也未因此事怪过她，而她竟然还是执意要离婚，这是没有办法的事。

　　回想起来，我们的婚姻就像一场梦，一场似真似幻的梦。这场梦让我昏睡了十年，现在终于到醒的时候了。应该说，这十年我们有欢笑的日子，也有悲伤的时候，但更多的是互相间的包容和迁就。坦率地说，我们的婚姻不是完美的，其中许多差异婚后才渐渐表现出来。她只有初中文化，虽然脾气不是挺坏，但许多方面我们无法沟通，甚至我和别人说的许多话她都听不懂，在这里，我说这些并不是想贬低她，而是想很客观地把我这段生活描述出来，最好不带感情色彩，这是很困难的一件事，但我也要努力去做好它。可又不知这是不是件荒唐和无聊的事。

　　我现在一个人坐在空荡荡的房间里，敲击着键盘，突然感到内心很落寞，一股说不出感伤叩击着我的心灵。我在写这篇文章的时候，实在不想把她的名字写出来，不为别的，只是想今后不给她带来麻烦而已。应该说，一桩婚姻的失败，有时不应该只归罪某一方，可能双方的因素都有。有时我又不敢回想，我的婚姻是一场恶梦吗？古人说，梦里求梦原非梦，无中求无便是无。的确，仔细回想起我们十年的婚姻生活，有许多事还是难以忘怀的。

　　恋爱时的情景有好多我都忘却了，我只记得，那时我们肩并肩地在寒风中行走时，她有时把身子紧贴着我，使我感到别扭和她的有些做作。我们第一次在做那事时，我感到她是那么老练，而我是那么地手足无措，那时我也没有想到，因为这对于我还是第一次。在这里，我不知该不该说这些；这些对我来说也并不光彩。但我却感到不能不说，对于此事，我也不是一无所知，那时别人背后说起，我心想哪怕是事实，我也只当着没听见算了罢。对于此事，她也一直缄口不提，我也不想去问她。直到等她准备与我离婚的前夕，她才告诉我，她是不应该的。虽然，我想之所以她不告诉我可能有多种原因，但她也不应这时才告诉我。她不是个诚实的人。我这话可没对她说过，虽然到现在她还不愿承认这件事。

　　我不知为何，在描述过去的事情时，语言竟然是那么地支离破碎，我确实很难描述那段婚姻。我们的家也不像个家的样子，几乎很少在自己的家里烧饭；这其中固然有我们住的地方经常断水的原因，但主要是我们俩都有很强的依赖性，这么大了还是在父母家吃的多。要说她对我一点都不关心，也是不事实的；但如果说有多关心，也是没有的。我们这种婚姻的确缺乏坚实牢固的基础，一遇生活风雨或失意的打击，便摇摇欲坠了。我的很多想法跟她不一样，爱好也完全不同。她爱逛商场买服装，而我则喜欢逛书店买书；她喜欢唱歌，而我只能是写写文章。可以说，这么多年来，我写的东西几乎她都未仔细看过，更谈不上交流了。这便是婚姻的悲剧和无奈。我这么说不知是否说重了。

　　我的确很难理清自己的思绪，可能是我目前还不适宜写这些吧。但有一点是我没法改变她的，那就是她的不现实。这种不现实是源于她们思想深处的，是源于她性格和生活环境使然的。棗这是任何人都无法改变的。另外一点就是，我自己目前混得这么差，也根本无法解决她家的困难，而这种负担是要背一辈子的。我是个命运不济的人，是一个对命运能逆来顺受的人。可我却无力改变自己和别人的命运。

　　我不是个强者，是一个卑微和庸俗的人。

　　我在断断续续地描述这段死亡的婚姻，其中的一些片段我现已记不清楚；有许多事情到现在我还可能没弄清楚，也就不须为难自己了，待以后再说吧。有一点我敢说的是，这可能并不是我真正的婚姻，我真正的爱情可能还没有到来，这里有算命先生和妈妈在香港黄大仙庙所抽的签为证。棗我并不是个宿命论者，但冥冥之中为何似乎总有一种东西在主宰着我们的一切呢？我实在真不知道。

　　前不久，我在报上看到这样一句话，说的是“对失败婚姻的一种补正”。我觉得这句话说得是太好了。结婚离婚虽然只是一念之差，但已经没有爱的婚姻再维持下去是没有意义的了。看看历史，再看看我们的父辈，那些不幸的婚姻令我们痛心扼腕，我们再也不应重复那样的悲剧了，我们也没必要去重演那样的悲剧。棗毕竟人类和社会都在进步着。历史和昨天一直向我们说明这么一个道理：如果婚姻不幸福我们应该去摆脱它，因为追求幸福和向往美好的生活是人应有的权力和起码的要求。我们没有任何理由去破坏这一自然和人类的法则。

　　最近这几天，电视里在播放五部剧的电影《红楼梦》，曹雪芹对诸多人物描写之到味、人性刻画之深刻是令人叹服的，那些红楼女子命运之悲惨和不幸简直达到了令人难以复加的地步，特别是看到晴雯和尤二、尤三姐等青春少女的蒙冤而死，确实令人十分感动。曹雪芹的确不愧是语言和小说大师，社会的复杂多变、人性之善恶、世事之无常、人心之诡诈被他均描写得惟妙惟肖，的确现在许多肤浅的小说是不可望其项背的。虽然斗转星移，时代在变迁着，但人性却没有变，因为它是永恒的。在这方面我们的确应该向前辈和大师们学习。

　　有人可能觉得这是不是跑题了，可我觉得一点都不。《红楼梦》里有爱情和婚姻，而且那里还有不是庸俗的爱情和婚姻，有的甚至可以说比我们许多现世的婚姻还要纯洁得多。看着一个个妙龄少女怅然离去，人生之梦逐一破灭，我们不禁十分伤感。这时，我不知为何却想到了那么一句话：宁可别人负我，我决不可负别人。虽然时代不同了，可人物的命运均是那么地坎坷曲折，这似乎是任何朝代都是相同的。在这点上，我们和古人也没有区别。

　　在这段婚姻行将结束的时候，匆匆写下这些文字是想对自己已经过去的感情进行梳理和抚慰。我现在似乎感到我一身疲惫，满心伤痕，我的确应该用文字为自己疗疗伤了。度过这个寒冷的冬天以后，我不知在前面等待我的将是什么？无论是多大的风雨还是荆棘，或者是转瞬即逝的鲜花和掌声，我都要勇敢坚定地像个人那样活下去。
