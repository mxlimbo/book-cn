诗 样 情 话

lovecong （lovecong@xinmail.com）

　　还记得我们初次相识的那次新年晚会吗？那次你以一曲琵琶独奏倾倒四座，而我则被台上端庄秀美的你深深的吸引了，隐隐的感到这就是我一直在寻觅的女孩。正所谓"众里寻她千百度，蓦然回首，那人却在，灯火阑珊处"。

　　记得我们初次约会，晚上我送你回家，看着你的身影渐渐地消失在楼梯拐角处，而我却不忍离去，久久地在你家楼下徘徊，只希望能在窗前看到你的身影闪过。就这样不知不觉的过了几个小时。哎，欧阳修说的没错："人生自是有情痴，此恨不关风与月"。可没想到第二天当你得知这一切时，竟感动得泪留满面，看来情痴又何止我一人呢。

　　你知道能得到你的感情我有多高兴吗？当你满含羞涩的告诉我我是你的初恋时，我心里想的只有一句话："愿做一生拼，进君今日欢"。

　　朝阳的晨曦中，我们曾一同在学校的花园中背诵共同喜欢的诗词。可能是起的太早吧，你还有些睡眼惺忪，可这却令你更加娇媚可人。怪不得苏东坡在形容美人时曾用过："困酣娇眼，欲开还闭"的词句。

　　在夕阳的余辉中，我们曾携手登山观日落。你说真希望这无限美好的夕阳能多留一会，于是我便满怀豪情的"为君持酒劝斜阳，且向花中留晚照"。了

　　小心眼和坏脾气可能是我与生俱来的缺点，没想到这竟令我差点失去你。在一次完全由我引起的争吵后，望着你含泪离去，我想我可能永远失去你了。在没有你的日子里，我每天魂不守舍。生活中如果没有你，"便纵有、千种风情，更与何人说"。

　　能够重归于好不仅使我更加珍惜我们的感情，也使我一定程度的改正了我的坏脾气。还有一个意外的收获，就是我那"岿然不动"的体重竟减轻了不少，这大概就是"衣带渐宽终不悔，为伊消得人憔悴"吧。

　　有一次我们一起赏析《诗经》中的名篇"氓"，我来朗诵，你来翻译。当我念到"将子无怒，秋以为期"两句时，你忘情的翻译道"求你别生我的气，到了秋天的好日子你来娶我吧"，那一刻，我明白了，你是我唯一的新娘。
