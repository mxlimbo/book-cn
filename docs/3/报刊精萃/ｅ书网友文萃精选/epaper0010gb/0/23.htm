我们彼此只是陌生

无情（cappuccinoo@citiz.net）

　　女孩的名字叫静。

　　她不喜欢说话，她喜欢沉默。特别在深夜，她会把房间的门窗都关起来，黑暗中，她只开一盏台灯，柔和的灯光散发着诡异的光芒，映照她苍白的脸色，象幽灵般飘忽。临睡前她会躲在温暖的被窝里看书，她最喜欢看的一本书是安妮宝贝的《告别薇安》，她说里面的女孩象她，喜欢喝冰水，穿棉布裙，头发像水一般倾泻，有明亮漆黑的眼睛。

　　她习惯孤独，她说她小时候常常一个人趴在阳台数星星，明亮的夜空中，有流星划过的痕迹，一闪而逝。不过她害怕寒冷，她不习惯冬天，冬天使她心乱、让她心寒，所以她不得不用药物来平静自己，医生说，这些镇静药物可以帮助她。

　　失眠的深夜她也会披着棉袄上网，她在网上认识了他，他的网名叫陌生。

　　她：晚上好。

　　陌生：好。

　　她：这么晚都不睡？

　　陌生：你呢？

　　她：睡不着。

　　陌生：我也是。

　　她：呵呵。

　　陌生：呵呵。

　　陌生是个奇怪的男人，他的话简洁明了，但又牵动着静。每晚深夜，他们会不知不觉在OICQ上相遇。

　　聊天的时候静会把那盏台灯熄灭，然后安静地面对电脑敲打键盘。她聆听熟悉的键盘声有节奏的在黑暗中清脆的响起，她苍白的脸上就会露出一丝欣慰的笑容。他们下线的时候也会象其他同伴一样，数3，2，1，然后一起下。

　　睡觉前静习惯对着镜子道晚安，不过她发现认识陌生以后，她道晚安的时候，心里却默念着321。

　　每天早上静乘49路公车从上海体育馆到人民广场，然后穿过西藏中路再到延安东路，中间塑着一块电视大屏幕，播放着早新闻，然后另一辆公车会在这个时候驰过，车号为123。

　　她忽然对数字极为敏感，早安她习惯123，晚安她习惯321。这种感觉就象她小时候趴在阳台数星星，永远都只能感觉熟悉的陌生，却无法让自己清楚。

　　她：我喜欢孤独。

　　陌生：孤独也是一种美丽。

　　她：我习惯了让自己变得美丽。

　　陌生：你是一个美丽的女孩。

　　她：我从不否认，呵呵。

　　陌生：你笑的也美丽。

　　静的公司处在商业区，竞争激烈，事务繁忙，乏味的填制定单使她的思绪简单重复。不过她喜欢，她不想让自己变得忙乱。

　　林说，你是个安静的女孩。静微笑。林是她的上司，比她大六岁，高高瘦瘦的，英俊帅气。在她刚进公司开始，林就对她关怀备至。静懂，可她没有感觉。

　　她感觉自己需要的是象123那样简单的数字人物。

　　陌生：其实简单也未必好。

　　她：哦？

　　陌生：因为我就是一个简单的人。

　　她：呵呵，这么说你不是个好人。

　　陌生：你说呢？

　　她：我......没有感觉。

　　其实她知道自己是有感觉的，特别是深夜。她已经不需要药物来平静自己了，当她冰凉的手指触摸键盘时，她心里前所未有的欣慰和平静。

　　静想，也许我应该见他。

　　林把她调到了办公室打定单。林说，冬天外面冷。静默默地注视着电脑屏幕，脸色苍白，手指不听使唤的僵硬，她有些忙乱。她想对林说，我不适合坐这儿。可是她沉默，她不喜欢说话。她希望林能从她冷漠的表情中明白。

　　公司内部组织舞会，林邀请静一起跳舞。静说，我不想。林说，那么我们喝酒。静说，我不会。林尴尬的看着她，脸色微怒，他开始独自喝酒，他说，酒苦涩的让他想呕吐，可是他愿意，他心甘情愿。灯火柔和，林有些醉醺，恍惚中，他上去握住静的手，静的手指微微颤抖，随后漫布了全身，她把手从林手指间抽出，冰冷的。她看着林，她说，我要回去了。

　　已是午夜，城市空气异常清新。静走出DISC BAR，林在她身后追喊着，难道你就不明白我吗？明白......我明白，可是我们不可能。静转过身，冷漠。空气变得沉闷压抑，他们彼此僵持在人群中，林的视线开始模糊，泪水沿着眼角滑落，他冲过去，他想再次握静的手，静一转身，走了。消失在茫茫人群中，泪水象酒一般苦涩。林说，我不明白。

　　午夜，三点零四分，街道冷清，独自一人徘徊，没有黎明，黑暗笼罩，我想，我是灵魂，今夜，没有归属。

　　静不想回去，幽暗的天空，没有繁星闪烁，暗红色云层大片遮盖着，随着寒风在城市上空呼啸而过。她忽然感觉自己开始害怕孤独，有种心痛的寂寞。她拿出镇静药，吞下，努力平静。她伤了林的心，她不想，可是又不得不。她安慰自己，林会找到一个更好的女孩。

　　她：相信缘分吗？

　　陌生：事实还未证明。

　　她：那我和你呢？

　　陌生：也许。

　　她：也许什么？

　　陌生：也许也许。

　　也许我们只是陌生。

　　静不知道陌生住在哪个城市，可是这并不重要，她需要的是感觉，那种熟悉的陌生，就象她走在深夜的街道上和陌生的人群擦肩而过，她好几次都相信那个人是陌生，就是陌生。他们会彼此漠然的看一眼，很平静，没有火花。只是这样，仅此而已。

　　她在拥挤的人群中穿梭，她试图让自己变得和普通人一样，可以开口说话，不再沉默，可以享受欢笑，不再孤独。有时她会到人声鼎沸，充满喧嚣的地方，她想让自己习惯热闹，而不至于见到陌生时无话可说。她应该是个美丽的女孩，她不需要任何镇静药来麻木自己。

　　她约了陌生在那块电视大屏幕下见面，明晚八点。

　　林好几天都没来上班。公司说他上次舞会酒喝太多，病了，需要休息。静开始加班，许多定单填制都需要她来完成，可是她不在乎，她想，也许这是一种补偿。

　　天空是暗蓝色，有流星划过的痕迹。静站在窗前，远处那块电视大屏幕播放着晚新闻，时钟刚过七点。静把定单重新整理了一下，然后面对着镜子练习着说道，123。她不由的笑了，美丽动人。

　　门突然开了，进来的是林。

　　静的笑容变得有些僵硬，她无话可说，她沉默。林说，有些客户的定单忘交了，客户急着催。他从她身旁擦肩而过，她的心跳的厉害，她不敢正视，她低头，她说，我要回去了。

　　空气稀薄的让人窒息。静冰凉的手指拧开房门的一瞬间，林从她身后冲过来，林说，相信我，给我一次机会。他握紧她的手，他们彼此颤抖。我会给你幸福的，相信我。林有些激动，全身开始颤抖。放开我，放开我。静大声叫喊，剧烈的疼痛从手中蔓延，她哭了，她反抗。林一把抱住她，她在他怀中奋力挣扎，她无法动弹，她感觉自己的肌肤在摩擦中扭曲。

　　林粗暴地亲吻她，她的头开始晕眩，她听见自己抽泣声埋葬在林急促的喘息中，她什么也没抓住，她没有依赖。她感觉自己下体焦灼的疼痛，泪水在黑暗中遗落，她有些疲惫，她绝望......

　　时钟指向了八点。

　　她从房间里失魂落魄的走了出来，衣襟零乱，神情暗然。她的眼神枯萎碎裂，颈部有红斑的伤痕，恍惚中，她走过那块电视大屏幕，她抬头，看见了一个男人，陌生的男人。

　　他穿着一件暗蓝上衣，洗白的牛仔裤，干净清爽的短发，略显消瘦的脸庞带有骨感，露出邪邪的微笑，他看着静，漠然。他们彼此交换眼神，熟悉的陌生。男人忽然说，321。

　　静无言以对，她冷漠，低头，从他身旁蹒跚而过，很平静。男人看着她的背影，长时间停留，他没有感觉，昏暗中，他们的视线彼此模糊......

　　静从身上拿出镇静药，一口吞下。一辆公车从她身旁飞驰而过，车号，123。

　　――他们彼此只是陌生。
