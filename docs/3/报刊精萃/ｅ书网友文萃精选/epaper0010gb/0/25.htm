爱 情 负 伤

无情 （cappuccinoo@citiz.net）

　　我渴望拥抱，渴望亲吻。我只是害怕寂寞。

　　——前言

（一）

　　他想，我应该见她。

　　她的名字叫爱情负伤。她在e-mail告诉他，爱情就象负伤的鸽子，在猎人的追击下，无处可躲，黑暗中，她看不见慰蓝的天空，她需要的只是孤独，可以平静她伤口的疼痛。

　　他可以感觉她是那么柔弱的女子，爱情曾经摧毁了梦想，她的翅膀已经折断，无法飞翔。他注视着那些文字，心纠缠般的痛楚。

　　空旷寂静的荒野，暮色迷离，天空是暗蓝色，象一张染满斑点的宣纸，绚丽的，但又沉闷。他住在这个偏僻的乡镇，有幽绿的沼泽和草地，空气清新。阁楼顶上成群的鸽子不分黑夜白昼忙乱飞翔，黄的泛灰的泥路有自行车轮扎过的痕迹，路旁几乎干涸的河流飘浮着大片大片厚浓的水草鳞，散发腥味的气息。远方，夕阳模糊了他的视线，梦想在昏暗中沉落。

　　有些事一开始就已经注定，无法改变，我们只能接受，现实残酷。曾经，他有想过要出去闯闯，可是家境不允许，他无法丢弃老弱病残的母亲，父亲在两年前已经去世。每晚深夜，他独自面对电脑，他希望可以在虚幻中模拟梦想，他害怕绝望。

　　认识爱情负伤是一次偶然，他发错了信箱。他说这不是我的错，她说这也不是我的错。他们开始互相通信，直到两天前的那个夜晚，他想，我应该见她。

（二）

　　城市上空高楼耸立，空气商业般的腐烂，大片大片暗灰色云层呼啸而过，人潮涌动。他握紧手中的地址，这是他梦想过的城市，他有些兴奋。夜色临近，他终于见到了她。

　　她穿色彩绚丽的宽大衬衣，洗的发白的牛仔裤，前额几簇丝发染成咖啡色，喛味的眼神，笑起来有倾斜的嘴角，她对他打招呼，HI，你是痴情？呵呵，你是爱情负伤？他看着她，傻笑。

　　他们彼此是那么的不协调。他穿起皱的麻布上衣，灰暗的棉裤，头发零乱。他有些不敢正视，他低头，脸色微羞。她说，还没吃饭吧？走，一起。他跟在她身后，艰难移动。他们彼此保持距离，象擦身而过的那些车辆，害怕接近，躲避危险。

　　她说，今晚你睡我那儿吧，我去朋友家。她用餐巾纸擦着嘴唇，然后甩了甩头发，她问，有没有吃饱？他拍了拍自己的肚皮，满足地打个咯，她笑了。

　　分手的时候她把房间的钥匙交给他，告诉他地址。她说朋友那边有事，她要先走了。他目送她消失在暮色中，他发现自己变得迷茫，黑暗中，他无法躲藏，他赤裸裸地表现，面对着一个城市有教养的女子。他们来之不同的世界，他们有着不同的梦想。只是，他们相遇了。

　　午夜，空气异常新鲜。他徘徊在街头，不想回去，他想看清楚这个现实与梦想错位的城市，他穿梭在人群中，和他们摩擦、相对、漠然.......拉近距离。他让自己开始习惯。

　　忽然间，他又看到了她。灯光昏暗的地铁入口处，米色的瓷砖墙壁，她和她的朋友在一起，他们相互拥抱，互相亲吻，男人压住她的身子，紧贴着墙壁，手放肆的触摸她修长的大腿，她在一阵急促的喘息声中发出微弱的呻吟。男人臃肿的身子显的有些苍老，头发灰白，脸庞皮肤粗糙，带有些许的皱纹。他是个年近六甲的老人。

　　他有些惊讶。他从他们身侧急步驰过，他忍不住低头看了一眼。他和她的眼光在一瞬间遭遇，她象负伤的鸽子，无法躲避。

（三）

　　凌晨的时候，他听见房门开了。她疲惫的走了进来，低着头，试图埋葬自己。她衣襟零乱，颈部有红斑的伤痕，她倒在沙发上，象一堆棉粟。

　　他沉默不语。他感觉自己心跳的厉害，他在被窝中裹紧身子，很小心的看着她。空气僵持着冻结起来，让人压抑窒息。他不敢喘息。

　　她说，你什么都看见了。

　　他没有回答。泪水从她眼角晶莹的滑落，焦灼般疼痛。屋内没开灯，她开始抽泣，黑暗中他听着泪水滴落地板清脆的响声，她倦缩起来，不停的颤抖。

　　她说，你们不明白。你们不会明白。

　　她开始叙述自己，她需要发泄，她对他说，我渴望拥抱，渴望亲吻。我只是害怕寂寞。

　　她的父母在她十一岁时抛弃了她，她没有选择，她需要生活，孤独的，寂寞的。我只是想和普通人一样，在失眠的深夜可以有人拥抱我，亲吻我，给我温暖。她说，我不想一个人活着。可是我要活着。

　　她象负伤的鸽子，在猎人的追击下，无处可躲。可是她不想躲，她害怕孤独，她不需要平静，她的伤口剧烈的疼痛。她说，爱情负伤，我们只是选择逃避。就象她的父母一样。

（四）

　　空荡荡的房间里他们彼此梦想，一个破碎的夜晚，他们象花儿簇拥在一起。她哭着在他怀中入睡。她在恶梦中喊着自己的名字，她在哭声中挣扎，她冰凉的手指掐入他的肌肤，痛楚的瞬间他不敢动弹，他们彼此依偎。

　　阳光斜斜地从窗口照进来，透过蓝色窗帘映出柔和的光芒。她醒着，脸庞倾卧在他的怀中，她说，如果可以，我希望永远都不要起来。永远。永远。她抚摸被她手指掐红的伤口，她说对不起。她用舌头添着它们，温暖而潮湿。他的身体开始有微弱的变化，他想起身，她压住了他，她把嘴唇贴过去，他无力拒绝，他们缠绕在一起，让伤口相互融化......

　　她说，你走吧，谢谢你来看我。他不知所措，他不是那种不负责任的男人。他不想让爱情负伤。可是她说，我一向只选择逃避。我只是一个出卖肉体的女人，我没有爱情。

　　她把他推出了门口，他看着房门被用力的关上，他的眼睛开始酸楚、发痛。他喊着她的名字，他说，你不是爱情负伤，你不是。房间里寂静的可怕，她把脸埋在被子里，湿透了大片。

　　我们没有爱情。

（五）

　　夕阳已经坠落，城市高楼依然耸立，这是一个梦想的城市。

　　他徘徊在街头，人群从他身旁擦肩而过，保持距离。他去敲她的房门，没有反应。夜色弥漫，他倦缩在门口，他可以等她回来。他想对她说，我要留下来，陪你，然后实现梦想。

　　时间在黑暗中流逝，午夜的天空异常清澈明亮，他看见她的影子从远处渐渐恍惚着走来，他的心跳的厉害。他迎上前去，他想大声喊叫。那一刻他没有，他忽然看见一个男人跟在她的身旁，他们拥抱在一起。

　　他无法自控，他冲了过去。他说，你不应该这样糟蹋自己。她冷漠的看着他，她说，你走。他一把拉住她的手，从男人怀抱中挣脱。男人有些发火，冲过来，和他纠缠扭打在一起。男人说，我他妈是出钱的，你呢？他们没有更多言语，他们在明亮的夜空下象野兽般搏斗。

　　她看着他们疯狂，她沉默。她象一只负伤的鸽子，无法躲藏，只是等待胜者来捕获。

　　他抓住了男人的双肩，他们彼此用力的喘息，翻滚中，他拿起了路旁一块大石，狠狠砸向了男人的脑袋，瞬间，灰白色脑浆喷射，粘稠血液迷雾般洒出，飞溅到他的身上，还有脸上。他的手冰凉，有血液的温度，他无法让自己镇定下来，恍惚中，他整个人都脱虚。

　　她说，我没有选择，我需要生活。我不想一个人活着，可是我要活着。可是你呢？

　　他神情漠然，呆滞。他仿佛感觉到冰冷的手拷套上他带有血液温度的双手，他开始晕眩。天空变得暗蓝，暮色迷离。他说，我们没有爱情。

　　有些事一开始就已经注定，无法改变，我们只能接受，现实残酷。

　　爱情曾经摧毁了梦想。

　　——爱情负伤。
