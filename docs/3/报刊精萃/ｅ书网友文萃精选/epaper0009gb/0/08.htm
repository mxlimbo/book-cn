折断的翅膀

fly

　　那天，风折断了我的翅膀。我好希望风可以折断飞机的翅膀，那么你便不能走，那么至少你可以多陪我几个小时，让我多听听你的声音。可是，你终于还是走了，而风，也把我的翅膀给折断了。

　　我站在风中，手脚冰凉，这样的天气，怎么说冷就冷了呢？可我只想多和你待几分钟，你怎么就不能理解，而一个劲的要我走呢？我只是想和你多待几分钟啊！也许，该走的总是要走，谁也留不住。可我从来没有挽留过什么，为什么就不能为我破一次例呢？我只再任性这一次，以后再也不了，我只想抓住时间的脚步。可我，却什么也没有抓住……

　　我们都是理智的，所以我们不会去提时间，我们也不敢说将来。也许我们的相识便是一个美丽的错误。其实很久以前，你就提醒我了。只是我总是怀着一种玩火的心情，只是不知火势是怎样蔓延到了我的身上。直到有一天，我清楚的感觉到生活中你的存在，我清楚的感觉到你离去时的楚痛。我想我是回不去了！

　　 回想着认识以来发生的一切，谁也没有想到会闯入彼此的生活。如果说我傻的话，我想你更傻，你怎么可以爱上一个乳臭未干的小女孩呢？她都不懂你，她根本不能理解你，她也不能给你快乐，而你怎么可以就这样爱上她了呢？至少你比她傻。可是，此刻讨论谁傻又有什么意义呢！你就要走了，这种心被慢慢撕裂的感觉，谁又骗得了对方。尽管我们不去想时间，我们不去想别离。可是，外面的雨却越来越大了，我也越来越冷了。好想牵你的手，让你捂暖我冰冷的手，可你却离我越来越远了，我快抓不住你的手了……

　　 我第一次感到了时间离去的恐惧，我也感到了人的无能为力。你走了之后，雨一直在下着，我迎着雨走了好远，我没有告诉你，在我的背包里静静的躺着一把伞。

　　那天，风折断了我的翅膀。我好希望风可以折断飞机的翅膀……
