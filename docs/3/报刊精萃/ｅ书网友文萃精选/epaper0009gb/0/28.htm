爱情，在愚人节这天翩然而至

小玲珑 （syanet@21cn.com）

　　在我的感情世界里，虽一直有着这样那样大大小小的邂逅，但它们或如鸟羽飘落于湖面般波澜不惊，或如惊鸿般一闪而逝，始终未能在我平静的心湖泛起动感的涟漪。直至25岁的那一年，再一次一个人度过情人节之后，我紧闭的心扉依然固执地为最早的初衷沉默着：这一生里在没有遇到心仪且相悦的人之前，我绝不松懈自己对真爱的执着。其时，我的情感依然如一叶孤舟，在茫茫人海中搁浅。

　　就在这一年春日的某一天，我突然收到一封寄自本市的快件：我已于今日抵汕参加为期两天的学术交流会，三天后返京，若无不便，周六晚见一面如何？8点钟我在阿波罗等你。落款是江洋洋。

　　我愣了半天才醒悟过来：和我相交已久的笔友洋洋她到汕头来了，并且想见我。刹那间我的心莫名地一阵激动，与素未谋面却书信往来甚密的人见面，于我这还是第一次。虽然我们相识已有三年，一直以来大家的友情也还很不错，但这毕竟是两个陌生人之间的约会呀。我迅速瞄了一眼台上的日历，白纸上赫然印着2000年3月30日星期四的浅蓝色字样，周六，不就是后天吗？

　　"后天晚上，我和江洋洋在阿波罗见面。"我这样想着，可是我又听到心里有另一个小小的声音在嘀咕着，"这个家伙，来汕头为什么不提前通知我？说见面就见面，真是的，也不先征求一下我的意见，还8点钟我在阿波罗等你呢，你过门是客了不起啊？就该你说了算？"

　　但当我再一次看到阿波罗三个字时，我笑了：洋洋她毕竟是一个不错的朋友。我记得我只跟她提过一次说我喜欢到阿波罗吃雪糕，在那一秒钟，我决定原谅她的自作主张，无论如何在周六晚我一定去赴那一场别开生面的约会。

　　星期六那天很快就到了，在连续开了两场例行周会之后，我又马不停蹄地完成了两份会议记录，很快地，下班时间到了。公司的同事们在这时盛情地邀请我去参加今晚的节日Party，我连丢下"不了谢谢我有约"一句话就走了。

　　我匆匆地回家、吃饭、梳洗，然后我在衣橱前略为考虑之后取下了那套浅紫色及膝的碎花套裙换上，最后浅浅地化了一个淡妆，一切就绪，时间刚好是7点半，我于是坐车前往洋洋指定的地方，欣然去赴约。

　　与洋洋的相识开始于三年前，那时刚好是我毕业参加工作后的第一个年头，闲来无事，我于是业余参加了广州某校开办的一个文学函授培训班。我是在接到通讯录半个月后收到洋洋的第一封来信的。她在信中简要地作了自我介绍，说她是京城人氏，就业已经几个年头，是个文学崇拜者，并说她钟情于南方的青山绿水，也喜欢南方姑娘似水般的柔情，希望藉此机会结交一个志同道合的南方朋友云云。她的字很是中性化，文笔却蛮流畅，这是在信中她留给我的第一印象。自那以后，我们由于多方面的兴趣相同，而且同是酷爱文学的缘故，很快便成了无话不谈的笔友，几乎保持一周一封的书信往来。

　　大约在通了两个月的书信之后，在某一天，我再次收到洋洋的来信，这一次，通篇文字包括信封清一色是打印出来的，除了署名。洋洋在信中告诉我，由于工作上的需要，她正在拼命练习打字，所以现在只要是可以当成练习来打的东西她都尽量不用手写，并说请我支持她。我为她的这种敬业精神很是感动，但我还是从落款中看到了些许不同，在回信的时候我问她是不是连签名都在重新塑造。这一次，在寄出信件三个星期之后我才再次收到洋洋的回信。她在信中对我提出的问题作了肯定的回答，她说现在练习打字之余也在练字，对于我的发现，洋洋大为赞赏，说我很细心，洞悉力很强，但是她很遗憾地告诉我最近工作很多，所以经常加班加点，以后回信可能会稍为晚一点。她很诚恳地请我原谅她，并表示无论如何她会争取时间尽早给我回信，我于是彻夜给她回了信说些让她放心工作多多保重身体之类的话以示理解和鼓励。

　　从那以后，洋洋依然给我来信，只是如她所言，每一封信件的发出时间大都比原来稍晚一周左右，但却依然规律如从前般。在信中，洋洋与我谈文学，也谈她的生活和事业，我发觉聪颖的她事业心很重，作为一名白领，她常常是谈事业多于其它，我想这大概是因为南北观念的不同吧。洋洋有时也与我谈起她的一直一片空白的感情，她说她是一个执着的人，缘份未到之前，她不会勉强自己；缘份来到时，她会不顾一切。我记得在相识半年左右，洋洋兴冲冲地来信告诉我，她最近强烈地感觉到丘比特已开始青睐她，为此她表现出少有的高调，我也深深地为她感到高兴。

　　交往多了，我又发现了一个问题，那就是有一些她认为比较重要的信件一般都是用快件寄出，有许多次她在信中对我说，她真的很想让我尽早知道她的想法，了解她的思维，她说很庆幸能够结交到我这样的朋友，她深深珍惜着和我的友谊。偶尔，洋洋的来信也很特别：或是一个只贴着一张漂亮邮票的空白信封，或是一张只写着几句歌词的小卡片。逢年过节，洋洋都会让我在节日当天收到她写着祝福的贺卡，而每年的10月28日，我都会如期收到洋洋的生日祝福和礼物。初识那时，洋洋在我的印象中是个温柔可人的女子，渐渐地，我发现温婉的她其实是个刚柔并济的人，有细致的一面，但更多时候我体会到的却是她为人中线条简单作风豪放的一面，在越来越深的交往中，我同样越来越深地欣赏着这个不曾谋面的远方朋友。

　　对于她的突然到访，我虽是有些措手不及，但我的内心其实也同样想会会这个相识已久的老朋友。正想着，车子已然来到了海滨花园，离阿波罗已越来越近了，我感觉到心跳有些加速。春夜的海滨花园灯火盈盈，在这个高级商住区里，错落别致地包藏了具有各种各样独特韵味店名的休闲娱乐场所，阿波罗就座落在这个区域的中心地带。此时，马路两旁全是挂满星火作装饰的小树，在那当中，我注意到了春意正浓的木棉。车子停在门口时，我感觉到从二楼大厅里透出来的融融气息，此时虽是初春，但是对生长在四季如春的这座城市的人们而言，只要有兴致，什么季节吃冰都可以，何况今天还是周末呢。

　　我下了车，在推门而进的一刹那，我看到了橱窗里一长串正在发光的幸运星。这时，我陡然想起曾经给洋洋许诺要亲手折99颗她喜欢的幸运星装成一瓶送给她，这件事我早就完成了的，那一个装满星星也装满幸运的瓶子此时正在我的抽屉里。当是见面礼也不错嘛，而且，如果我拿着这个瓶子，肯定会让她更容易从人群中发现我。天，我怎么就给忘了呢？正在那时，我突然发现隔壁是一间精品店，看看手表，还有8分钟才8点，我于是决定先去凑合着买份礼物再上去。权衡再三之后，我买下了一个形如幸运星的卡通气垫球，我想，这同样能使未曾谋面的洋洋认出我来。

　　阿波罗一楼是营业大厅，我顺着欧式风格的楼梯来到了二楼的餐厅。此时装饰得典雅而不失堂皇的大厅里烛光摇曳，在一片灯光幻影中，一台台地坐满了或缠绵低语的双双丽影，或谈笑风生的三五良朋，遍寻大厅，唯独不见本应形单影只的我的洋洋。来这之前，我已绝对确定洋洋她是独自来赴约的，而且在我的印象中，洋洋她会是一个守时的人，可是现在，这大厅里只有靠马路窗边的一张台上有一个男子背对着我站着的地方单独坐着。我看看手表，刚好是8点整。

　　我于是走到大厅中间，在<<罗密欧与朱丽叶>>的旋律中仔细地搜寻：每张披着大红格子台布的台上都燃烧着一支在荡漾在水杯中的蜡烛，每张台上都绽放着一支灿烂的玫瑰花，每张台上的每张脸孔都在流露着暖暖的笑意，我的目光掠过之处，我感觉到我的到来似乎并未引起在座的任何一个人的特别注意，由此我确定洋洋她不在这人群中，但我也相信她不会爽约。

　　我的目光再次落到那个男子坐着的那张台，这一次他正好是面对着我坐着：好帅气的一张脸。我的头脑中飞速地闪过这句话。正在那时，他也不期然地抬头看着我，我感到那不是不经意的目光相碰，他几乎是在——直视我，或许是看到我正在看他，旋即，他很有礼貌地冲我露出浅浅的笑，我突然意识到自己的失态，迅速地收回我的目光，想走。就在我转眼的那一刻，我蓦地觉得那张台的台面上有些什么东西不对劲，我放缓已迈开的脚步，那一瞬间，我清晰地告诉自己：那张台上的花瓶里插着的是一朵木棉花。﹗﹗﹗

　　就在我转身离去的那一刻，突然有一个颇具磁性的男音在响起："小姐，一起喝杯冰果汁可以吗？我想我就是你要找的人。"

　　我停下来，回头，那一张帅气的脸再次露出浅浅的笑。

　　"我是一直和你通着信的'江洋洋'，如果你是'玲珑'的话。"他依然面带微笑，我惊诧地从那浅笑中听出了这是一句纯正的汕头话，那句话里说的第一个名字正是此时我要与之见面的那个人，而第二个名字则是我的昵称。

　　"'江洋洋'？"我有些不置可否地加重语气，并重复着，"江洋洋？你？"

　　"是的，是我。很高兴见到你，"他还在笑，"坐下来喝点东西吧？"

　　我有一种被愚弄的感觉，但我不是一个没有修养缺乏风度之人，我跟着他的步伐朝那张台走去。他很绅士地直身为我拉开了一张座椅，并做了一个请的动作，我坐下了。

　　"给这位小姐来一杯 '鲜橙汁圣代'，我要一杯'红粉佳人'。"他对前来记单的服务员说，"我记得这是你喜欢的。"他朝着坐在对面的我扬扬眉。

　　因为人物不符，我依然在为"江洋洋是他""他就是江洋洋"这个事实而不高兴。

　　"请先接受我的歉意，好吗？"他收住笑容，目光直视着我，"原谅我一直以来没有向你表明身份，我知道这是很不对的，但我担心我说明了之后你会不相信，或者你会因为我欺骗你而拒绝继续和我通信。"笑容在他的嘴角隐去，稍为停顿了一下，他接着有些艰涩地说，"我珍惜你这个朋友，喜欢和你聊天，喜欢与你共享我的快乐，喜欢听你分析问题，甚至听你的戏言、气话，和你对我的尖锐的批评。"他再度停下来，目光依然直视我。

　　这时，服务生端上两杯色彩鲜艳但柔和的饮料上来。他很仔细地在杯子里插上吸管，然后起身将其中橙色的那一杯端到了我的面前，接着才转身坐下来端自己的那一杯。

　　在他起身的一刻，我凝视着眼前这个人：五官分明，棱角深刻，笑容绽放的时候，他的脸显得很生动，此刻收敛了笑容，却也凭添了一股英气，深蓝色的格子衬衫配卡其色西裤，这装扮令他近1米80的身材显得更颀长、挺拔，看样子，也不过是27~28的光景。要命!他那双始终荡漾着笑意的眼，和从那眼里流露出来的逼人目光又落在我的身上了，这让我有些不自在。

　　"先喝点东西好吗？"很柔和的语气，就像这杯里荡漾着的液体的颜色，"我说过，你是个细心且洞察力很敏锐的女孩，这是你给我的第一印象，也是为什么在第一次与你通信之后我依然拿着这个名字当晃子一直给你写信的原因。"

　　我在记忆中搜寻，有些东西很清晰地在慢慢浮现，是的，就是在那次，江洋洋，不，是他告诉我他在练习打字的那一次。

　　"你给我写信是从你所谓的练习打字那次开始的，对吗？"

　　"是的，你很聪明。"他用勺子慢慢地搅拌着杯子里的冰淇琳，"这就是吸引我后来一直和你通信的力量。"他啜着杯子里的冰水，"其实江洋洋是我大学时候的同学，她是北京人，在学校时，我们都是文学社的成员，那年夏天我去北京旅游，正巧赶上她结婚，她请我去喝喜酒。席间，我们谈起了毕业后的一些事情，她告诉我前些时参加了一个文学函授班，说是很有趣，她开玩笑说反正她结婚了以后的业余时间可能要全力放在相夫教子上，问我有没有兴趣参加那个函授班，并且表示可以让我延用她的档案，我那时只当是玩一玩无所谓，所以就答应她了。不曾想在我回来之前她真的把那时整理起来的一些资料全部转交给我，其中就包括你给她写的信。江洋洋对你印象很好，她一直在我面前夸你有灵气。她知道我崇尚缘份，所以她笑言要为我创造一个世纪童话。她说反正你们都是汕头人，有机会谈一谈，找个时间见一面，算是多交一个朋友嘛，但是她警告我千万别吓着你，在她的心目中，你是一个稚气可爱，需要呵护的小妹妹。开始我也没在意，只当她是在说笑，后来，我看了你给她写的信件之后，不知道为什么，我竟不能拒绝她的提议，突然有一种想与你相识的冲动，我毅然决然地接受了江洋洋的建议，于是当即给你打了那一封信，没想到你真的一眼就看出了破绽，洋洋把你的回信回寄给了我，她趁机打趣说我是缘份已到，就在那一次，我也索性请她帮忙，让她在以后的日子里把你以为寄给她的信件转寄给我，然后我写好了信装在打上寄自她的地址和贴好邮票的信封里，外面再装一个写她的地址的信封寄给她，再由她转寄你。可是这样很费时间，所以才有了后来寄快件的事情。江洋洋说她很乐意为你和挂名的'江洋洋'效劳，所以我们就这样一直通着信了。其实我有想过向你解释这件事的，我真的不是故意骗你，但我怕你误会，所以我选在今天这个节日向你澄清这件事，希望你可以原谅我，如果不行，那你就当它是一个节日的小插曲，可以吗？"他态度诚恳，目光炯炯。

　　"节日？"我一直在听他娓娓而谈，谈这个荒诞离奇的童话，忽然对这个词有些不解。

　　"是的，今天是4月1日。"我看到一直被强忍着的笑意在他的嘴角游移——他在忍着笑。

　　天，我真是忙得昏头昏脑了我 。"你可真客气，一见面就送我这么大的礼﹗"

　　"对不起，我真的不是故意的。"他有些心慌地低下了头。

　　我感到我的手在紧紧地捏着一个什么东西，对于刚刚发生的一切，我是有些愠怒的，可是他的这个童话令我有些飘然，有些陶醉，我回想这些年来"江洋洋"为我所做的一切，想起刚刚在路上还在给自己下着越来越深地欣赏着这个不曾谋面的远方朋友的结论，尤其是在听到这个故事之后，想到一个男孩不惜费尽心思冒名顶替，为的只是编织他美丽的世纪童话，这份执着让我阵阵激动。我的怒气慢慢地平和，心情也渐渐地释然。

　　"如果我告诉你现在坐在你面前的也是一个冒名顶替的人，"我故作姿态地眨眨眼，"或者我碰巧只是一个路过而被你错认为是你在等而坐下来听你讲述童话的人呢？你怎么办？"我的表情一本正经。

　　"哦，那我很遗憾，"有一丝落寞滑过他的眼角，"但我不信，"他垂下眼帘，眼光有些飘忽，旋即，他充满自信地抬头，"我确信这是幸运星在牵引着我们的缘份。"他眨眨眼，"这个星星是送给我的吗？"

　　我抬头看到他正指着我的面前，一看，一直被我捏着出气的竟是那个气垫星。

　　"是的，送给你的见面礼，'江洋洋'，请笑纳吧。"我故意似笑非笑地加重语气。

　　"谢谢。这表示你已经原谅我了，是吗？"他一脸灿烂。

　　我摇摇头，笑容在他的嘴角僵住；我听见我的心在大笑，我有一种报复的快感，然后我感觉到我的脸也在笑，于是我点点头，我看到他的脸在笑，他的嘴在笑，笑得见牙不见眼。

　　"谢谢你，玲珑。"

　　"客气了，'江洋洋'。"

　　他迅速地从包里掏出一张名片，双手递给我："这是我的卡片，请多指教。"

　　我接过卡片，看到那上面印着"意念广告公司 陈浩扬 总经理"的字样。

　　他又递过来一个粉红色的信封，我小心地拆开，一张相片从里面滑了下来，那相片上是一个高挑女子笑意盈盈的倩影。

　　"她是真正的江洋洋。"他说。

　　我取出装在信封里的卡片，展开，那似曾相识中性化字迹再次跃然纸上：

　　亲爱的玲珑：

　　请允许我再一次这样称呼你，你是我见过的女孩中最动人的一位，而浩扬，他是个执着的浪漫才子，我曾想，你们的认识会造就一段美丽的童话，所以我善意地欺骗了你，能原谅我的冒昧吗？

　　"于千万人之中遇见你所要遇见的人"，这是缘份，也是童话，但它就在你的眼前，珍惜这一切吧。﹗

　　祝福你们。﹗

你诚挚的朋友：江洋洋

　　 "玲珑，做我的女朋友，好吗？"

　　我看着他，他目光炯炯，有一种令人心痛的温柔。我有些愕然，但我很快镇定下来，下意识地看看墙上的挂钟：今天是世界欢庆的节日。

　　他起身举起花瓶里的木棉花，满是真诚地献给我："这是你最喜欢的花，热烈而不失质朴，就像我对你——的感情，请你接受我的见面礼，也接受我的感情，好吗？我想在10月28日给你送玫瑰花已想了很久，希望今年你能让我如愿。"

　　我还在犹豫，我怀疑自己是否在做梦，我还在想着"江洋洋"，我……

　　我注视着那朵娇艳的花儿，此刻，它正无比灿烂地盛开着，被一个帅气的人细心地呵护着。我的木棉情结由来已久，我能不为眼前的一切感动吗？

　　"请你做我的女朋友吧。"他重复，并且改变了语气助词。

　　"我可以拒绝吗？"

　　"借用你的一句话：'我不是最优秀的，但我比很多人出色'。这是我对自己的要求，也是我对你的承诺，所以你不能拒绝——这朵木棉花。"

　　"原来爱情可以只是一朵木棉花。"我听到自己在笑，我看到我接过他送过来的花。

　　"今天——是个可爱的节日。"他说。然后我看到笑容在他的脸上绽放，犹如我手中的花儿。

小玲珑
2000年8月22日于梦园
