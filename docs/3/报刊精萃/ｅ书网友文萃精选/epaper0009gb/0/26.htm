我 的 初 恋

清风月（qingfengyue@263.net）

　　已经是深夜了，可我还没有睡意，脑海里总是想着我初恋的情形，那是我一生都无法忘记的。说来可笑，那时我还在上初中一年级。@0@

　　认识她是4年前的冬天，我还依稀记得那天的天气很好，我在约定的时间到了一个朋友家，一进门我就看见了她。一头长发，瓜子脸，大眼睛，红润的唇，是标准的梦中情人，经朋友的介绍知道她叫丹，是学舞蹈的，第一次见面难免有些拘束我们只聊了一些关于学习，朋友之间的事，但我敢保证我的视线一直停留在她的身上，我想她也在看我，因为我能察觉出她总是有意无意的向我望一眼，聚会很快的就结束了，幸运的是我要到了她学校的地址。那次见面后我就回到了北京。：（

　　很自然的回到北京后我们开始通信，再过一段呢就用到了电话，在那一段时间里我们之间的感情在一天天的加深，彼此在一天天的了解，我觉得我们走的更近了。：）

　　放假的前一天我给她打了电话，我们约好回本溪后，在本溪的望溪公园见面（是我第一次约会呦^0^）离公园越来越近了，我的心也在怦怦的跳着，我努力的使自己平静下来，但有过恋爱经验的人都知道那是不可能的，呵呵！看见她了，白色的连衣裙，乌黑的长发，打着一把小花伞，美！真的，美极了，我走了过去，脸上的笑容是傻乎乎的那一种^0^我们什麽也没说就一起进了公园。

　　拉着她的手慢慢的走在石子小路上（什麽什麽怎麽拉上手的？？这还用问，当然是我主动啦，就是找点什麽借口拉住就不放了，嘿嘿）她低着头，默默无语，我偷偷的看着她的眼睛，长长的睫毛一眨一眨的，我不由得把手紧了一紧，她好像也没有反抗的意思（当然啦，我们都明白是怎麽一回事）就这样我们之间的浪漫恋情从此开始了，没有任何约定，没有任何承诺，平平淡淡的走到了一起。还记得第一次吻她，，我不知道，我是怎N吻上去的，只知道她左W右W，最后还是让我吻上了让一股电流，侵袭了我，也侵袭了她，我吻得好狂热，吻得好激动，她的手此刻也紧紧抱住了我………

　　又是黑夜，又是昨天的那个时候，我继续着我的回忆

　　那个暑假应该是我们最快乐的一段时光，她很喜欢看电影，听音乐，所以电影院，音像店是我们最常去的地方，我们看电影很有特点总爱盯准一个片子不放，看了一遍又一遍（你说烦不烦，烦，当然烦了，但有她在身边就不一样喽）那时候看的最多的应该是黎明和吴倩莲演的《都市情缘》吧，在我的记忆里看了4遍以上是不夸张的，她说那部片子每看一次都有不同的感受，所以每次看的时候她都很认真，静静的。而我呢？当然也很认真只不过我看的是她罢了^0^我们最爱听的歌是张信哲的《直觉》到现在我也时常放来听听，心是一个容器不停的累积关于你的点点滴滴……直觉我们应属于彼此，否则我不会常常若有所思（这首歌的歌词被巍巍放在精华区，谢谢）每当我们听这首歌的时候，总认为那首歌就是为我们而写的，浪漫缠绵的歌词总让我忍不住去抱她，亲吻她，而她呢也是闭上双眼，来体会我给她的温柔。（呵呵，写到这里眼睛已经湿湿的了，谁说男人坚强？我说男人最脆弱了）整个一个假期我们都出双入对，去哪里都是两个人手牵着手，面对双方的家人也是这个样子（呦呦呦那时侯这样子可是很严重的）经常留在她家和她的父母一起吃饭，他们也很喜欢我：）只是总说像我们的年龄要以学业为重（这个我当然知道）就这样我们度过了一个甜蜜的假期，回北京的那天晚上，我们一直从她家走到火车站（很远的）因为是晚上十点多的车，路上的行人很少，我们互相依偎慢慢的走着，什麽话也没有，她还是像我们第一次约会时的样子，低着头，眼睛一眨一眨的，只是气氛不同了。啊~~~~~多希望这一刻能永远的停留。

　　开往北京方向的28次列车就要发车了……当站台广播响起的时候，我知道我们即将分开了，透过火车的窗子，一直望着她的眼睛，大眼睛里闪烁着晶莹的泪花，列车在缓缓的开动我们的距离在一步步地……我的双手放在车窗上，脑门贴着玻璃，眼睛努力的在送站的人群中寻找她的身影，那身白色的连衣裙，她一头让我痴迷的长发。我的视线在渐渐的模糊。眼泪充满了我的双眼，思念从那一刻起开始了他漫长的旅行！

　　秋天的北京有点凉，手里捧着一杯刚刚冲好的咖啡，坐在电脑旁，看着杯子里一缕缕的热气，我的思绪也回到了从前。

　　一月五日，星期五。那天是我的生日，记得那天收到了许多的礼物，最喜欢的当然是她送给我的了，打开邮包看见的是一副手套，白色的，左手有一只熊GG，右手有一只熊MM，虽然有点像女孩子戴的，但我依然天天的戴着它。那晚我们通了电话，聊了很久，但说的好像就是那麽几句，我多麽的想她，她多麽的想我（你说闷不闷？当然不，甜蜜的很）后来要不是我妈妈来说我，我想我们能一直聊到天亮。

　　二月十四，情人节。那时我回到本溪已经有几天了，那天的我特别的高兴（当然第一次过情人节能不高兴吗？）一大早我就去花店买了十一朵郁金香，十朵白色，一朵粉色。（浪漫吧，真正的浪漫还在后面呢！）她送我的呢，是一盒巧克力，还有一件她亲手织的贴心小毛衣，为什麽说是小毛衣呢，因为我穿在身上的确很小呀，那天我们玩了一整天，她手里拿着一块热气腾腾的烤红薯在前面又蹦又跳的，我呢在她后面左手捧着送她的花，右手拿着她给的礼物：（一会儿她好像想起了什麽，转过身来拉着我的手就向我们初次约会的公园方向走，（我手里的花差点掉在地上，好贵得！）我们进了公园，和上次不同，这次我们是漫步在雪地上，每走一步都能留下我们共同的足迹。而我手中的那束郁金香在这样的气氛下就显得更加的美丽尤其是那一朵粉色的：）就这样我们来到了一座铁索桥，相传那是一座情人桥，那儿的情侣来到这里都要买两把锁头，把他们锁在那座桥的铁链上，到这时我才真正的理解了她的用意，当我们把两把银白色的小锁头锁到铁链上的时候，我看见了她脸上的笑容，而且她还吐出了她可爱的小舌头，粉红色的，呵呵，真的可爱极了！离开的时候，我忍不住的回下了头，我要记住今天，记住那两把银色的 锁！

　　临回北京的前一天，外面的疯很大，我们相互依偎在她的床上，我揽着她的腰，她的头靠在我的肩上，我们一起听着那首我们最喜欢的歌，直觉我们应属于彼此，否则我不会每次无法停止……慢慢的我吻上了她的唇，她的眼她的鼻，我的手抚摸着她的头发……这时候我们之间已经没有任何阻碍了，在一阵晕旋过后我们彼此结束了生命中的第一次。（此时我眼中的泪水直打晃，仿佛这些事情就发生在昨天）

　　回忆在那飘雪的季节，分手在一个漆黑的雨夜。

　　第三年，冬，雪。独自一人走在我们曾经约会的公园里，还是那条小路，还是那一排老杨树，不禁想起去年的情人节，身后留下了一串孤独的脚印。来到了那座铁索桥，不争气的我眼泪充满了眼底，两把银白色的小锁还挂在那里，冰冷的手指伸了过去，可很快的又胆却的缩了回来，闭上眼睛呼吸着这里的空气，仿佛这里还留有她一丝丝的余味，忽然间我的眼前一片模糊，又很快的清晰，看见她很认真的把那两把小锁锁在那里，微微一笑，露出她可爱的小舌头，当我要去牵她手的时候，她慢慢的离我远去，两只手扶在冰冷的铁链上，链子上的一对对小锁好像风中的风铃，轻轻的摆动，在向我抗议，打扰了它们的甜蜜。大年三十的夜里，穿着厚厚的大衣，一个人坐在雪地理，望着满天的烟火，它们多么的美丽，但为什么，它们匆匆的来，风一样的离去。

　　第四年，盛夏。已经知道我们同在一座城市，知道她每天会站在窗前，懦弱的我只有在楼群中的角落里凝望着她的身影，那一头在微风中飘舞的长发，她！还是那样的美丽。

　　黑色星期一，雨。打着一把雨伞，穿着趿拉板，在雨中缓缓的走着，不知不觉中来到了她家的楼下，望着那扇紧闭的窗，我知道她一定在那里，鼓足了勇气给她打了电话，电话中淡淡的声音，值得回忆，她告诉我让我等她。终于，我们又见面了，她让我闭上眼睛，她温暖的唇吻上了我，而我像一块冰山上的寒石，呆呆硬硬的站在那里，当我睁开眼睛的时候，雨水和眼泪混在了一起，唇上的温暖已经消失，而她也离我而去，我的手中多了一封长长的信……信尾这样写道：我们的恋情真的很美丽，但已经不能在继续，它将是我一生的回忆，谢谢你！

　　任雨水在我的身上拍打，地上有一把雨伞，和一封长长的信……
