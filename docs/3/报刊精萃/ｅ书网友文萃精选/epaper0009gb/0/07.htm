奶奶，再见

隐士（hong@mail2.powerise.com.cn）

　　奶奶终于离我而去了。据说她走的时候很安详，中午母亲还给了她最喜欢吃的蛋糕，晚上便静静地走了。

　　去年春节离别时随口说的“奶奶，再见！”，竟成了与她最后的话语。

　　奶奶，再见！今日一别，何时见？

　　奶奶，再见到您时，您还会这样喊我吗？“我大孙子回来了”。您还会这样叮嘱我吗？“大孙子好好干，闯出个样让奶瞧瞧。”

　　奶奶，再见到您时，您还会牵我的手，走过故乡一望无垠的平原吗？您还会站在村口，翘盼孙子的归来吗？

　　奶奶，再见到您时，我会对您说：我不会忘记，您送我南下求学时拄着拐杖的背影；不会忘记，我结婚时已瘫痪在床的您，头插一朵小花与我合影时幸福的微笑；更不会忘记，我俩之间的约定棗您为我带儿子。

　　奶奶，多想再见您，听您唱“东北二人转”，听您讲“月儿玩玩晴满天，膀靠膀来肩靠肩，知心话儿说不完……”。

　　可是，奶奶，我们难说再见了。此刻的您安详地睡着，没有人能打扰您。再无世事烦恼，您可以好好休息了。

　　奶奶，再见。今年春节我一定回去看您，为您唱一首歌棗你静静地离去，一步一步孤独的背影，多想伴着你，告诉你我心里多么爱你……。

　　您一定听的到的。
