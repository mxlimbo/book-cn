网络——我的灵魂所在

雨果 （whq26@163.net）

　　网络是由信息网络和情感网络组成的。信息网络是为物质社会服务的而情感网络服务于精神世界，我想说的是情感网络那部分。

　　 情感网络——精神的存在的空间。

　　人的一生可以单纯的分为两个部分，一半是在现实的物质世界里，另一半则是在看似虚幻的精神世界里，就象人之所以为‘人’之一撇一奈，一撇是精神，一奈是物质，物质支撑精神。你想把它写大一些，两者都不可缺。人追求物质是精神的需要，而精神的升华是对物质的超越，但也是建立在征服基本物质上后的一种有权力的超越。所以我个人认为‘人’的高大与否很大程度上还是取绝于撇的长度，当然你的意识已经完全超越了这一切，那么你就不是人了，也许是鬼，神，佛、、、、、、在现实生活中每个人都有他的权力和义务，人们因此而活着，同时也受到现实因素的制约。在网上也大相径庭，那是一个更自由的世界，是由你我的精神构筑。

　　 网络——我的灵魂所在。

　　你想宇宙有多大？它有边吗？有的，但它的边的另一面又是什么？所以、、、你想你的思想有尽头吗？所以、、、你想网络有多广阔，不用说了，它可以传达无数人的思想，所以、、、我爱网络一如我爱自己的思想，一如我爱这世界。它们的浩瀚才匹配的上我的信服。我把网络当成我生活的一部份，因为我需要，我可以很真诚的在那里展现自我，与好友跨越时空做心与心的交流。你可以说我很虚幻，没错！我属龙的，注定虚幻。但如果你能告诉我宇宙是真实的还是虚幻的？感觉是真实的还是虚幻的？那我就知道该如何去做了、、、、、、

　　 网络——现实生活的倒影，灵魂的镜子。

　　孔子很强调的一点是‘慎独’，即人最该在乎的是你独自一人时的行为和思想，因为那时你才是真的你，那是你人格和精神真实展现的时候。

　　我觉的在网上也是一样的，你拥有尼采所言的觉对自由，无需考虑太多的现实因素，你可以用灵魂去碰撞，这是一种天大的福份！福兮祸所倚？就有许多人把网络当成游戏，更可悲的是有些人把它当成垃圾桶，随意的发泄自己的垃圾。呜呼，哀哉！俗不知他因此出买了自己的灵魂，当然也许他们的灵魂就那样。我只是不希望我生存在又一个被污染的世界里，更不希望他们失去一个这么好的的精神世界 ！

　　请大家拿出真诚该变网络被污染的天空！

　　让我们的灵魂同在一片蓝天下——自由翱翔！！！

——雨果26 尽心下
