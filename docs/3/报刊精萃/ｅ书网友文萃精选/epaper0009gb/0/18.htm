恨是毒药爱是酒

baobc（baobc@citiz.net）

　　仇恨会毁了一个人。

　　这是句老话，也是真理。

　　与爱相比，恨是一场旷日持久的艰苦战役，大恨大痛小恨小痛，从来不曾听说过恨会让一个人快乐。那种不经意间就会翻腾而起的仇恨象一把得意洋洋的鲨鱼刀，一下一下玩儿似的在你心上左划右刻，用不了多久就会让你重新恨的牙痒痒。可是痒痒归痒痒再恨你也奈何不的，于是便有满怀仇恨的人去挺而走险，以生命的代价去摧毁另一个生命，结果是玉石俱焚。就算是不去摧毁别人，仇恨也会让人自残，很简单，因为仇恨而使自己的每一天都乌云密布，从此进入不快乐的人生，这难道不是最彻底的自残？！人这一生总共不过几十年，去掉少不了更事的几年，再去掉吃饭睡觉的时间，余下的已经不多了，再让仇恨折磨着自己，这一辈子不如及早结束及早轮回开始新的生命还好过一些。

　　无论是谁，这一次的生命都是单程旅途，不会有机会重新再走。等到转世轮回的又一生，早已是沧海桑田物是人非，要爱的和爱你的都已经换成了另外一些人，切莫让仇恨早早了此一生。

　　恨是毒药，药性的浓烈和仇恨的程度成正比，且多用于他杀不成后的自杀。

　　恨的背面是爱，有爱才有恨，恨是爱的另一种变形的延伸。

　　爱是酒。

　　爱是酒，酒多伤身。

　　会喝酒的人很懂得适当的控制，深知酒要喝的恰到好处才嘴美妙，界限就是微醉。

　　微妙的时候人最快乐，既有醉酒的飘飘欲仙，又能够保持一点清醒不至于出丑或是做下不值得的错事。

　　酒喝得太醉，会头痛，会呕吐，唯独没有美的感觉；酒喝得太少，又不足以让人放下警惕和矜持，不能够完全地放松自己，便不会有释放心灵的那份惬意。

　　酒意朦胧是微醉时的专利。

　　爱也一样。太浓烈的爱就象是一个人喝了太多的酒，不仅会弄伤自己，也会连累你身边的人，而且是离你越进越遭殃；可若是没有了爱，或是只有一点点喜欢罢了，又无法领略由爱而生的那份牵挂那份温情那份生生世世永不分离的信念与渴望，以及爱一个人时的那份羞怯那份幸福那种心跳的感觉……没有了爱，如同这世上没有了风景，如同一个人失去了耳朵、眼睛和心灵，剩下的只是程序化的穿衣吃饭，这样的人生真的是不要也罢。

　　恨是毒药爱是酒，这一生尚要走的快乐一些，就要学会化解毒药、学会适量饮酒，在微醉里去细细品位每一点喜悦每一缕芳香，才真的对得起自己，也不枉来些走这一生。
