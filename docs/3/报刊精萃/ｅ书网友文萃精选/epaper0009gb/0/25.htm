爱，也需要承载

fly（flyinsnowing@21cn.com）

　　我以为爱可以让我变得美丽，却不知道爱是需要承载的。

　　一直想做你的新娘，你的美丽而纯洁的新娘，所以，愿意为了你而变得美丽。我相信丑小鸭的故事，也相信女为悦己者容的故事。我不在乎别人的赞美，却在乎你细微的动作、言词，那怕只是一个字。我感到了生命存在的意义在于你，而不在于自己。

　　苍白而无力的爱着你，却希望成为你永远的新娘。你用声音吞蚀了我的心，爱你简单得没有理由，也没有给自己找个理由。只是，第一次为你流泪的时候，你却一句话都没有说，让我无法知道你会不会心痛，可我却心痛了，因为你的无言。

　　那些日子，我因为喜欢雪，而希望这个城市能下一场雪，可我知道，这也许比六月飞雪更难，而你却告诉我，你那里下雪了。

　　于是，我便期盼你到来的日子，因为我相信，你会为我捎来片片雪花，你会为我打造一个雪花飞舞的世界。等待的日子，心痛而漫长……

　　也许，是我把我们的世界建造得太完美，而乎略了现实，乎略了我们各自生活的现实；也许，那只是我的童话世界，我也只是那个童话世界里的你的新娘。海市蜃楼终归只是虚无飘渺的景象……

　　我想，是我悯诚的期盼感动了上苍，你终于来到了这个城市。而我，也终于可以做你美丽的新娘了。我高兴得忘了，那个日子对我的特殊性。我不停的告诫自己：不可以太激动，不可以太开心，不可以……在你来之前要一切维持原样，才不会使一切在我的慌乱中变得糟糕和不真实。我激动、慌乱、开心而急切的等待你的到来。

　　却没有想到，你给我捎来的是寒冷的雨天，而逃避这场雨的人，是我！

　　我以为爱可以让我变得美丽，却不知道爱也需要承载，在那个雨天，我成了爱的逃兵……
