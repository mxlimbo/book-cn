家 书

文茜（szadam@sina.com）

　　记得第一次听李春波的一封家书时，还是在上高三的时候。那时的我只觉得这首歌分外煽情，甚至有些佩服李春波的天才创意，居然把自己情真意切的思家之情以家信的方式配合当时大学校园名谣的风格表现得淋漓尽致。但至少在我心里是不能打下任何烙印的，因为我毕竟无法体会身处异地的那份念家情怀，我还沐浴在父母关怀备至的亲切呵护之下。

　　今年七月，我随着滚滚“创业大军”，来到了离家千里之遥的深圳。只因为爸爸那句好儿女志在四方，你要去哪儿，爸爸决不反对，我知道你是一个有抱负的孩子。想想如果当初父母对我有一丝的挽留之意，我也绝对不会跑到这么远的地方。

　　既然下定决心要独自闯天下，当然要作好一切随时需要自己能独立承受挫折、面对阻挠、挑战自我的能力。然而把社会叫做大熔炉也好，叫做大染缸也罢。总之，短短的几个月，发现理想和现实总是背离得遥遥无及，只知道白天活在现实中，晚上活在幻想里。人也渐渐地变得很颓废。

　　一日偶然在同事的书架上翻到了一盘磁带皮都有些泛黄的磁带，是我大学时最钟爱的校园名谣。夜晚一个人躺在床上听里面的歌曲，也会是一种意犹未尽的享受。

　　“亲爱的爸爸妈妈，你们好吗？现在工作还忙吗？身体好吗？厖斠桓鋈嗽谄岷诘囊雇恚俅翁秸馐浊拥氖焙颍丫巧钣刑寤崃恕５谝淮味雷宰鸪道吹揭桓瞿吧某鞘校谝淮未钇鹈浩钭隽艘欢傧痰梅⒖嗟姆共耍谝淮伟だ献艿穆睿谝淮瘟斓叫剿畢呎庖荒甓晕叶裕翟谑怯刑嗵嗟牡谝淮瘟恕?p>

　　然而让我最难忘的还是第一次收到来自家中的书信，那是爸爸写的：

　　欢儿，你好！

　　不知道一个人独自在外，生活是否还习惯，工作是否顺心。

　　还记得以前你一放学，总爱不顾一切的冲向厨房，品尝爸爸给你做的饭菜，你总能赞不绝口地给爸爸塞一大堆的糖衣炮弹。前阵子你打电话回家，说你做的饭菜比学校食堂的还要难吃十倍。做爸爸的，首先要做一个自我检讨，都怪爸爸以前只知道注重对你学习方面的教育，过分的强调你在学业上所取得的成绩，而忽略了其他方面的培养。不过，为了让你给我一个将功补过的机会，我特意去书店给你挑了一本儿菜谱。高考你都过三关闯六将的，挺过去了，更何况是做饭呢？我相信这对你来讲根本就是小菜一碟，不是吗？

　　一个人出门在外，万事都得当心。你毕竟还小，没有经历过什么曲折的社会阅历。你会因为很多不顺心的事儿受尽打击，可是谁又不是在一边受挫折，一边流泪中成长的呢？

　　你今年的考研成绩不太理想，不过爸爸知道你已经尽力了。你不要嫌爸爸罗嗦，这年头你只有拼命的学习，才能出人头地。爸爸知道你在外面工作很辛苦，但无论如何可不要让你的大脑生锈了，要知道工作对你而言只是一个新的开始，你前面要走的路还长着呢。

　　欢儿，你在爸爸眼里始终是个长不大的小姑娘，爸爸对你总是有一万个不放心。一个人在外面的日子，要好好照顾你自己，你嘴唇总是爱裂，记得多买些水果。家里人都很挂念你。

　　每每看到家里来的信，心中都有很多说不出的思念和寄托，总恨不得马上回到父母的身边。可是这终究是不可能的事情，只好紧记父母的教诲，老老实实做人，踏踏实实干事，不敢有一丝一毫的懈怠。

　　今天，刚进公司便有人喊我名字，“文茜，这儿有你的一封信。”看着信封上熟悉的字体，便迫不及待的看信：

　　欢儿，你的来信已收到。爸爸要在信中纠正你的一些观点，你信上说，你很想赚一大笔钱，觉得有了钱你就会活得很快乐。你突然萌生的这种念头是爸爸一直以来很担心的。要知道你现在还不具备出色的才华，我希望你用一颗平静的心情去看待金钱。

　　前一阵子爸爸在报纸上看到了两篇文摘，现在我将它寄给你，希望你能好好去领会。一篇是人生十乐，一篇是金钱的忠告。我想告诉你的是，人生的快乐不仅仅来自物质生活的富有，更重要的是辛勤收获的喜悦，被人认可的慰藉，心胸开阔的豁达，乐于助人的品质。我的女儿啊，我们可不想你这么年轻就误入歧途，我们多么希望你能本本分分做人。

　　看着看着，眼睛不知不觉地湿润了，此刻的我多么想在爸爸妈妈面前唱完那首一封家书啊。

　　“亲爱的爸爸妈妈，你们好吗？现在工作还忙吗？身体好吗？爸爸妈妈，要保重身体，不要让儿放心不下。 ”
