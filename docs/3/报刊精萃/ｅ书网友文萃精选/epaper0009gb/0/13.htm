无情小说系列：虚幻

无情

　　我盖好被子，蒙住脸，呼吸沉重。

　　那个女人倦缩在阴冷角落，漆黑头发遮住了脸庞，衣襟破碎不堪，断断续续发出凄凉的抽泣。我不敢走近，她苍白的手中紧握着一把刀，刀沿着她纤弱的手臂慢慢刻划下来，撕破光滑如丝缎的肌肤，暗红色血液如迷雾般洒出，顺着白色地砖缓缓蔓延，没有疼痛。女人微微抬头，露出她枯萎的眼神，憔悴的表情，嘴唇在昏暗灯光下没有一丝血色，抖动着...抖动着...

　　她怨恨的看着我，看着我......突然，她发疯似的冲了过来，我来不及闪躲，刀冰冷坚硬的刺入我的腹部，粘稠血液贴着刀面喷射出来，洒在她扭曲的脸上，红色的，鲜红。房屋里充满了血腥的味道，我的眼前一片黑暗，女人发出野兽般的尖笑.......

　　冷汗润湿了背脊，我在恶梦中惊醒。心口依然疼痛难忍，我捂住，轻微的咳嗽几声，有些缓解。我擦去额头上的冷汗，深长沉重的叹出一口气。

　　这是幢极其普通的私宅，一星期前我因工作关系搬来这儿，听说以前房屋的主人是个女人，不过在半年前死了。邻居们还说，她是在一天晚上被杀死的，死状恐怖，惨不忍睹。我倒无所谓，看在房租的份上，我还是留了下来。

　　这个梦境在一星期内已经第四次出现，再这样下去的话我迟早会发疯的，我还清楚的记得那个女人扭曲的脸庞沾满了鲜红的血液，她野兽般的尖笑在黑暗中象一根丝线勒住我的咽喉，让人窒息。所以，我决定明天就搬走。

　　我看了一下闹钟，2：00。时间滴嗒滴嗒象滴水般流逝，在寂静无声的黑暗中清脆响亮，窗外寒风呼啸，刮过透着缝隙的窗户发出嘶裂般的尖细声，就象女人凄凉的抽泣。那扇破旧的木门在寒风中叽嘎叽嘎颤抖个不停。我点了一盏灯，暗红色，血液的颜色。我不敢睡，拿了一本书，随手翻阅。

　　铛、铛、铛

　　闹钟响了三下，我的眼睛有些疲倦，睡意来临。我把书放好，关上灯，盖好被子，蒙住脸，呼吸沉重。

　　挞、挞、挞

　　黑暗中，忽然传来几声轻微的响声，我翻了个身，意识有些模糊。

　　挞、挞、挞

　　又是三声，似乎是破旧木门的响声。我迷茫地睁开眼睛，看到窗外有飘忽的影子闪过，几片枯叶紧贴着积满灰尘的玻璃窗户，艰难的移动，不时又飘散几片。我仔细聆听。

　　挞、挞、挞

　　的确是敲门声。我一看闹钟，3：14分。深更半夜，怎会有人敲门？我再仔细一听，似乎有女人凄凉的抽泣声，我的心一寒，浑身起了疙瘩。我点了那盏暗红色的灯，披了件外衣，向门道走去。

　　挞、挞、挞

　　敲门声继续。我站在门前，疑迟了一会儿，开门。门外一片漆黑，一条人影歪歪斜斜站在那里，漆黑头发遮住了脸庞，衣襟破碎不堪......

　　原来是我自己的影子！

　　第二天大清早，我卷了行李马上走人。

　　后来听说那间房子被拆建了，在那扇破旧木门的地下，发现了一倶女人的尸体......

　　后记：

　　第一次尝试写鬼故事，本来是想写长一点的，来个中篇什么的，可刚写了个开头后，自己都有些害怕了，呵呵，所以就改成了短篇，当然故事情节也变了。唉，以后再也不写这种东东了，看看恐怖片倒还可以，自己写还是有点血腥带呕吐。（写好了都不敢看，想想你家门口，就可怕，呜......~~不过，我还是建议你看第二遍）
