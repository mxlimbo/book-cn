我的江妈妈

吴秋丽（zxg2429@163.com）

　　问起江妈妈，友人告诉我：“她老人家巳去世了”。听到这个消息，我禁不注泪如雨下――江妈妈，我那可亲可敬、淳朴善良的老妈妈，我还不曾报答您思情，您怎麽就离去了呢？甚至在您逝世的时候我都没能见上您最后一面。

　　离开水城，离开与我相处多年的邻居们巳有两年半了。

　　两年前人们走进总段院子，也常能看见我的江妈妈――她巳年近古稀，短发灰白，个子不高，身体稍微有些发胖。她那一口亲切而悠扬的玉屏话，那张慈祥而开朗的笑脸，总是让我如沐春风，就象见到了久别的母亲。她每天有规律地买菜、做饭，总是那么有心肠、乐乐和和地照顾着与她相亲相爱相敬一生的江伯伯。他们二老的最大的特点就是快乐、平和，有着博大的爱心。

　　认识江妈妈他们二老，是我刚搬到水城防疫站附近的“老基地”以后。他们二老常到我隔壁的胡叔家玩，由于胡叔、龙娘两位长辈也很爱护我，所以，有时在他们家玩就会遇到江妈妈和江伯伯。

　　当时，我的孩子小，自己身体以不好，父母为了让我工作之余能好好地休息一下，就把我还未满两岁的孩子接到盘县扶养。但孩子不在身边，剩下我一个人，生活并不轻松，反而觉得孤独、落寞，常常因为思念父母和孩子而然伤怀。那时，别人家看彩电巳经很普遍，而我却连台白电视机都买不起。所以，无聊时就到隔壁胡叔家找他们的发儿玩，或者看电视。的确，我那段时间的情绪很低落、很迷惘，就仿佛一个人站在黑黑夜孤岛上┈┈这种情况下，我是多么渴望亲情和友情啊!也许是与江妈妈有缘吧，一见到慈眉善目、笑容满面的江妈妈，我阴郁 的心情就一下子烟消云散了。她让我感到亲切、温暖、轻松、安全，就象回到了母亲的怀抱。我的江妈妈，她是多么平凡普通的一位老太太，却那么豁达，那么爽朗，笑语里那么富有感染力。

　　记得那是一九九二年的秋天，我患了急性胃肠炎，上吐下泻，吃不下东西。折腾了好几天，到附近诊所打针吃药也不见效，眼看整个人又黄又瘦支撑不下去了。江妈妈知道后，就来到家看我。当她看到我孤伶伶一个人病在家里时，拉着我的手竟掉下泪来：“姑娘，你快点去市医院住院治疗吧!这样不吃不喝是不想活了。”热泪滴在我的手上，呼醒了我麻木的知觉；这一句看似责备的话语，却让我寒冷的内心涌进了一股暖流。我想起了我的亲人、我的孩子，我又有了求生的勇气。

　　在市医院住院治疗的几天里，江妈妈每天都给我送一次吃的东西，陪我聊会儿天儿。从她家到医院很远，还要挤公共汽车，这对年迈的老妈妈来说，辛苦自不必说，这份情是何等珍贵啊。每天也只有江妈妈来看我的时候，是我最快乐的时候，感觉就象回到了童年、回到了家。江妈妈语言并不很多，便只要见到她的微笑，你就能从这真诚的笑容里读到很多慈爱和关怀。江妈妈说：“穷人不生病，当得行大运。要有好身体才有好心情!”她是这么会开导人。她的乐观、开朗不但帮我度过了难关，而且让我受益至今。

　　九五年冬天，我搬到总段院子里住巳经两年多了，并且和江妈妈家住对面楼。由于每天要上班、做家务，接送孩子读书、……总是忙忙碌碌。离江妈妈近了，反而很少有时间享受江妈妈给我的那份亲情。不过，只要我需要帮助的时候，江妈妈总会约我最亲切的援助。诸如帮我照看一下孩子，做些我爱吃的“盐菜肉”之类。非常让我过意不去的是――此时，我住在高高的六层楼上，当孩子放假去外婆家时我又病了，我那象母亲一样的妈妈又开始来照料我。

　　那一天是我终生难忘的日子。窗外下着小雨，天气很冷，我睡在病床上思绪万千。一想到自己的姻、生活以及不可知的未来，忧伤和痛苦就一起向我袭来，我不禁掩被恸哭。不知过了多久，我听见了“嘭嘭、嘭嘭”的敲门声，继而听见了江妈妈轻轻的喊声：“吴秋丽，姑娘，你睡着了吗？”我急忙擦干眼泪去开门。打开门，我的泪水更奔涌而出――江妈妈显然是刚爬上楼来，正累得手扶栏杆，大气直喘。汗水巳把她稀疏的头发打湿。有几缕贴在额头边上，汗水还在顺着脸颊淌。她喘着气，笑着说：“这六层楼，是爬不动了，腿脚不行了。我刚给你炖好的鲜鱼汤，”说着用手指了指脚边的一个大汤钵：“姑娘，傻站着干什么？快，进屋，乘热吃了。”这时，我鼻子发酸，嗓子发辣，除了傻站着，就只想扑进江妈妈的怀里大哭一场。可是，当时血压很高又患有糖尿病的江妈妈几乎站不稳了。我只好把最感动的心情和最辛福的泪水甩在一边，先把老妈妈扶进家里坐下。还在气喘吁吁的老妈妈边进屋边指挥我：“快把外衣穿上，刚好一点，不要又受凉了。我在汤里给你多加了些姜丝和胡椒粉，你快偿偿看好不好吃。”面对这份沉甸甸的母爱，语言巳变得多余。

　　“好吃，非常好吃!”至今想起那钵又香又鲜的鱼汤，还十分想吃。可是，江妈妈，我的好妈妈，您怎麽就走了？――再也不来敲我的门，再也不管我病中是否有人照料!我经后的路还很长呵，怎能没有你的关怀!我就是您的亲生女儿，您是

　　刻在我心中的母亲，您将与我的生命同在!
