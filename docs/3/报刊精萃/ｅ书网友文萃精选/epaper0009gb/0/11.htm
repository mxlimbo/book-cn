逝去的青春之二

江南浪子（superjune@sohu.com）

　　?不知道哪个名人说过的“国人对女人的想象力是无可限量的，看到一个漂亮的女人就想起了她的胸部；看多她的胸部就想起了她的大腿。”确实如此，中国本来好男人很多，都是被女人毁掉了。所以男人总是大叫红颜啊，祸水啊！！就连我们的啊Q也有时想起了“女——”，何况是我辈风华少年呢。

　　班上的女生有十多个，可以于以考虑的大概有五六人。我所中意的也是大众情人是杨阳和琴书。阿阳纤细如秋草，有东方女孩子的柔弱之美。秀发长长的批在肩上，就像一株温室里的水仙。倾城的微笑，让我痛恨前生不是给她浇水的侍者。无数的男生梦想着有英雄救美的机会，而每人的对象则是阿阳。

　　琴书的美丽有是另外的一种：成熟的身体配一个童贞的心。身材颀长，发育的好丰满，总是让人疑心她也许前世是玉环。孩子般灿烂的笑经常挂在那如花的脸上。她老爸是局长，老妈也是国家公务员。处于那样的家庭的独生女当然是宠的厉害，所以虽然也是高中生，还是嗲的令人难以忍受。可是即使如此，暗地里男生还是喜欢她那种嗲声嗲气的，男生真是贱，我也是这些男生中的一个。如此良人，倘若能红袖添香夜读书，那时何等惬意的事情啊。我曾经做了无数次这样的梦，可是始终没有变成现实。有很多的想法，总是一个个的被我否定掉。就象是遇到一道实在是很难的题目，我用很多的假设来猜想它，可是总是证明不了它。其实说的明了一点就是我他妈的太胆小。不可否认我长的不蟀，但我有颗温柔的心，可叹的是“美人涓涓如秋水”，我却始终是在水了另一方，只能隔雨望红楼。其实我也不是没有一点机会，事实是我的座位在阿阳的后面，而在琴书的前面。每次值日我们都在一组，有很多交谈的可能性与可行性，如果我有勇气和她们聊天，那么我就要机会博得她们的芳心，可是我竟一次也没有把握住，真乃次生一大憾事。而且，琴书还经常向我讨教数学题。说来也怪，琴书的题目对我来说是小菜一碟，不论那道题是几星级，我都能在极段的时间内搞定它。有次琴书对我说，阿豪如果你能获得全国希望杯数学竞赛获奖我就当你的女朋友。结果我真的获奖了，可是她并没有当我的女朋友，因为我醒来时候，发现那是一场南柯一梦。迷蝴梦中难比目，独向春风诉衷肠”，可笑我的未央歌梦啊。人生不如意的事情，十有八九，这也算是其中一件吧。

　　我心理虽然如此的不堪，可是现实生活中我却是出奇的老实，在女同学的心目中我绝对是安全系数很高的男生，只有阳痿的男生的安全系数比我过。我对女生从来都是，严肃的一B，对女生目不斜视。久而久之，我的视线就出了问题，成平行的了。上课的时候，我一直往前盯着看，好多的老师都被我迷惑了，以为我是听课最认真的学生，其实事实恰恰相反，我根本不是在盯着某中现实中的实体，而是在考虑中午吃什么，如果不吃大排，我就没法和我的室友一起吃饭，如果吃呢，我的银子实在少的很，超出了我的承受范围。我的外语老师刚刚结婚，教书也没有几年。她在津津有味的讲课，同窗们都在认真的记笔记，只有我两眼直视，黑板上的英语字母渐渐地连成了一片。我想从字母的字里行间读点什么东西出来，不定里面可能藏着三维立体画。因为练习的比较多，所以就举重若轻的学会了看立体画。我看我的画不会对别人产生任何的影响可是得出了我的英语老师，开始的时候有几次我发现她讲着讲着就红起脸来。后来我猜想她认为我是在偷看她的胸部。我在一本书上看见有这样一种说法，那就是无论你以多快的速度去浏览女性的胸部，她总能感觉到。可是我却相反，我并没有兴趣去看我老师的胸部，大家都说她是飞机场。因为这样有点失态，所以后来我就改了，老师讲课的时候我就自己思考复杂的哲学问题，我的脑子让我用坏了，里面全是些希奇古怪的问题。
