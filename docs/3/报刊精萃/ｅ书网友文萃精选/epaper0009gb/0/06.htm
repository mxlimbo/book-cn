疯狂的城市

第三空间

　　疯了，你们真的疯了，这些城市里的人。

　　你们拥来拥去抢购廉价的商品，一场混战到了高潮。有小偷，有人在尖叫着。你们的目光都注意到了身边的人，警惕地捂了捂装钞票的口袋。有人从人堆里跑出来，跟着又有人跑了出来，最后出来的是一个中年妇女，一只脚已经跑丢了鞋子。

　　在马路上有三个人在跑，人行道上的人纷纷驻足。

　　"是抓小贩吗？"

　　"快看，抓小偷呢。"

　　"前面的哪一个是小偷？"

　　"怎么没来警察，有人打110报警了吗？"

　　"在拍电影吧。"

　　女人跑不动了，索性一屁股坐在地上嚎啕大哭起来了。

　　一群人立刻围成了一圈，纷纷议论起来。

　　"蛮可怜的。"

　　"被偷走多少钱啊？"

　　"我看她是个下岗工人，挣点钱不容易，各位每人捐助一点吧。"有人提出建议。

　　围观的人纷纷慷慨解囊，这时候又发生了一件事把人们的视线吸引了过去，两辆的士在前边发生了事故，撞在了一起。围观的很多人都奔了过去看。

　　"还好还好，人都没事。"

　　"我看就私了算了，等交警来处理没准谁都走不了。"

　　"我看还是叫交警，车有点撞坏了。"

　　闲话声中两个的士司机怒目相视。

　　马路旁边一个卖彩票的摊子悄悄地移了过来，接着卖人寿保险，卖进口香烟的都移到了这段路上来。

　　"这些人来干什么呢？"

　　"哈哈，想赚我们的钱。"

　　围观的人冲着卖彩票的、卖保险的、卖香烟的调侃着。这时，外围密匝的人群让开一条路。一个戴墨镜的交警走了进来。

　　"怎么了？"他表情严肃的问。

　　围观的人七嘴八舌，司机互相指责。

　　交警不耐烦摆了摆手，冲着司机说："把驾驶执照留下，车我会叫人来拖走，等候处理吧。"

　　司机提上红中华说："帮个忙……车我们要做生意。"

　　"那每人先交两千元押金，驾驶执照留下，三天后来支队处理。"交警甩开提烟过来的手，拿出一本收据快速地签了一串看不太懂的字撕下来。

　　交警走了。两辆车也被开走了。围观的人还在议论着不肯散去。

　　有人传来消息，那个商场里偷钱的小偷被抓牢，失主根本和中年妇女无关。人们这才发觉那女人早已无影无踪了。

　　"她可能是个骗子。"一个围观的人忍不住说。

　　"怪不得有点面熟，上次在广场见过她骗钱呢。"又有人轻声说。

　　"你怎么不早说，我还给了她十元。"有人在大声抱怨。

　　"太不像话了！"

　　一辆巡逻的警车呜呜地开来。几个警察和刚才闯祸的两个司机又下来。

　　警察喊到："所有的人都不要走。"

　　"怎么了？"

　　"我托人打电话到支队问情况，根本没有那个交警。"一个司机向围观的人解释着。

　　警察说："你们每个人都看到了刚才这一件事，都是目击证人，跟我们到警局去录一个口供吧。"

　　围观的人互相望了一眼，突然作鸟兽状四面逃窜了。
