无情武侠系列：不幸

无情

　　他把脸埋葬在空气中，零乱的头发遮住了脸庞，他的伤口开始流血，剧烈的疼痛使他接近麻木，他低着头，不敢正视对手。

　　这是他第一百零一次挑战失败，据说已经打破了江湖上连续失败次数最多的迪尼斯记录，他被邀请出席下个月的武林风云人物颁奖典礼。

　　刀歪歪斜斜插在地上，对手从他身旁慢慢走过，踩了他一脚，吐了口唾液。他没有反应，战战兢兢倦缩在那里，脸色苍白，粘稠血液顺着泥土缓缓蔓延。对手轻蔑地瞟了他一眼，冷笑着离去......

　　他没有勇气站起来，过多的败绩逐渐消磨了他的锐气。

　　大街上，人声鼎沸。他握着刀，松松垮垮，疲惫不堪。人群在背后议论、嘻笑。他们说，他是江湖的耻辱，他是江湖的不幸，他是江湖的悲哀......他深埋着头，快步退去。

　　他忽然很想回去。三年前，他发誓一定要在江湖上闯出个名堂。师傅告诉他，你还年轻，再过几年下山也不迟。他笑了，他说，不，正因为我年轻，才要下山，等象师傅您一样老了我就没有机会了。

　　如今，时过境迁。山谷还是一样耸入云霄，树丛还是一样郁郁葱葱，泉水还是一样清澈甘甜，可是人呢？

　　师傅坐着，静静地注视他，眼里充满了慈祥和关怀。他突然有股想哭的冲动，经过了那么多的失败他都没有流过泪，可是在今天，他还是忍不住放声大哭起来。他扑倒在师傅怀中，泪水放肆的泻出，三年来所有的委屈一瞬间跟随着泪水......

　　师傅轻轻拍着他的头，柔声道，不哭，乖，不哭......师傅把他的头抬起，擦去他脸上的泪花，问道，这么多年来，你有没有把师傅的名号给报出去呀？

　　他稳定了一下情绪，连忙道，没有，没有。弟子一直牢记师傅的教导，在还未闯出名堂之前，从不敢向外人提起师傅您的大名。

　　师傅苍老的脸上皱纹慢慢舒展开来，他笑了。恩，很好，不愧为我卑鄙派的正选大弟子呀，哈哈......师傅的笑声还未落，他只觉得背脊上一阵冰凉，一把刀从他身后刺入，胸前穿出，血沿着刀尖滴水般流出......

　　他最后抬头看了一眼他的师傅，疼痛间，他问道，为什么？为什么？......

　　师傅一把推开他，他重重的摔在地上，血掺入泥土，渐红。你是江湖的耻辱，江湖的不幸，江湖的悲哀......留你在卑鄙派，只会败坏我派名声......

　　刀歪歪斜斜插在身上，师傅从他身旁慢慢走过，踩了他一脚，吐了口唾液。他没有反应......

　　他死了。
