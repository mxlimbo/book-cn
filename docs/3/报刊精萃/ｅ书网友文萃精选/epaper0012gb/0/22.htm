沉 湎

甘力（garywdz@21cn.com）

　　比尔，一个普通工人。它的生活很简单，三部曲：上班，下班，啤酒。

　　像比尔那样的酒鬼倒也潇酒，哎不，潇洒。一个人，无牵无挂，只要不犯什么大的错误，拿稳铁饭碗，这辈子也就算对得起自己了。

　　可是，我们的这个酒鬼，竟决计要结婚了。原因很简单：房子。

　　妻子漂亮。分到的在三楼的房子也很好。最令比尔满意的是朝西开的装有防盗网的阳台。比尔拿来一罐啤酒，一个枕头，爬上阳台，背靠着枕头半躺在阳台上，任凭落日的余辉肆虐地洒在它的脸上，他那本来就已通红的脸显得更红了，比尔惬意地咂了一口啤酒，又吮了一口，再仰脖灌了一肚，最后舔了舔罐边，手一捏一放，那瘪了的啤酒罐伴着落山的太阳完成了它的使命，落到楼下的死胡同里。比尔拿经过昨夜洞房花烛建立起的信心，此刻也随着太阳的西落而熄灭了；然而，他不会知道，太阳的西落是为了明日的东升。

　　小比尔已不小了。一天，比尔下班回到家，拿了罐啤酒，还有那个枕头，径直走向阳台，眯起眼，略带疲倦地望着那每天都是新的太阳，微微地咂了口啤酒，他习惯地把手伸出防盗网，用力一捏，轻凹的啤酒罐却不再顺从，"啪"地一声回复了原状。于是他又把另一只手伸了出去，双手用力地捏，不料，年久失修的防盗网终于承受不了如此轻飘飘的啤酒罐和比尔，整个脱落了，比尔便不由自主地跟着掉了下去……

　　比尔苏醒过来，发觉自己躺在一大堆瘪了的啤酒罐里。他抱着昏沉沉的头拼命地回忆：阳台，死，妻子，小比二；想到酗酒前的梦想，想到……但回忆很快被那占据了他一生，吞噬了它的灵魂的东西所替代。他下意识的在脸上摸了摸，"还好——嘴还在，"他的嘴抽搐者似乎在笑，"去找人来修好防盗网。"他试着起来，然而，稍一用劲，便觉得一阵昏眩，又昏死过去了……

　　入迷是件好事，然而，沉湎却是可悲的。
