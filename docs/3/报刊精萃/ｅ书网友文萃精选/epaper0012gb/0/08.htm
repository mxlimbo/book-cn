"微观经济学"历险记

兔罗杰（RabbitRoger@hotvoice.com）

　　曾经有一则故事，美国以前的风云总统罗斯福，在他破天荒地第四次被选为总统后，有记者问他的感想如何。罗斯福笑而不答，先是客气地请记者同志吃一大块三明治。记者受宠若惊，加之"肠中羞涩"，于是欣然接受，乐此不彼。然而吃干抹净后还没等开口，罗斯福又客气地递上了一块三明治。虽然这块三明治的边际效用早已大大递减，然而盛情难却，记者同志只得继续米西米西地。谁知第二块三明治刚下肚，罗斯福的笑脸和第三块三明治又接踵而至。记者早已吃饱，但为了完成采访任务，只得委屈一下自己的胃，这第三块三明治生吞活咽终于解决了下去。然而"罗氏服务"还是太到家了，第四块三明治又出现在了记者的面前。记者终于不堪重负最终开了口，罗斯福这才彬彬有利地对他说："记者先生，现在你知道我的感受了吧。"

　　也许这个故事可以换一个荒诞一点的结尾。罗斯福拿出了第四块三明治，然而这却是一个炯然不同的顶级霸王美味三明治，聚合了三明治所能够拥有的一切，鲜香热辣，噬魂摄魄，令得记者不得不产生了非理性食欲，身不由己，欲罢不能，终于"营多营多，吃了再说"。结果是记者在超级美味带来的"痛并快乐着"中肠胃爆炸而死，临终前回光返照中发出了清醒的感叹："哇，原来这才是真正的终极三明治"．．．．．．

　　上学期已经是我第四次学微观经济学（Microeconomics）课了，国内上过两遍不说，来美国后也炒过一次冷饭了。无奈"书到用时方恨少"，真正现在做起这方面的research来，还需要去"充电"。然而仅头两堂课下来，残酷的现实便活生生地令我困惑是不是我这辈子压根就没学过Microeconomics。课程所用的教材是其他学校的同志也闻名的"杀手教材"，而课堂上原本印象中Microeconomics中那迷人的曲线直线推移相切相交的图上作业法也消失殆尽，取而代之的是满黑板的你所能想象到的和不能想象到的数学符号。这些数学符号才是决定了所有经济学模型的构建和繁衍的底层基因，原来这才是揭开了面纱的蒙娜丽莎的微笑。只可惜面纱揭开的一刻同样意味着普通人的理解力所能接受的问题表象特征也同时被揭去。当这些来自希腊、罗马和阿拉伯的符号排列在黑板上组成了一个又一个多元、高次的微积分方程时，它们看上去就象是刻在刚出土的文物上，显得那样的深澳而有距离感。当形象思维再也无法作为把握逻辑过程的依赖时，在这里科学和抽象派艺术之间的界限似乎早已不那么明晰，这到底是经典意义上的严谨还是后现代主义的解构只有去问上帝了。

　　教我们这门课的老师大约四十多。从他的姓名、脸型、身材、生硬的欧洲口音，和推导公式时那一板一眼的作风来看，任何一个见过《辛格勒名单》里的德国军官的人都不难判断出他的日耳曼血统。然而该同志却也还不是一个刻板的人，而是充满了美国式的自由作风，爱笑，爱开玩笑。当数次早上八点钟我睡眼惺松匆匆冲进课堂惊魂未定时，他也会只穿条短裤，拖着拖鞋慢悠悠地从办公室中溜达出来给我们上课。他同样是那种天才般的选手，据说可以把那本"杀手教材"倒背如流。每当推导公式兴起时他便会面露微笑，一蹦一跳，手舞足蹈，作天真浪漫状。那些在一脸迷茫的我们看来无法逾越的抽象思维屏障，在他万般沉醉的奋笔疾书下，变得竟如同波兰或比利时那些小国横在强大的第三帝国装甲集团军面前的脆弱的防线一般不堪一击。每当下课时我们彻底找不着北了以后了以后，他就会带着将军征服时特有的踌躇满志的笑容离去，留下一黑板的"般若波罗密经"。

　　最痛苦的是作业。谁曾想象过会有作业多到连看书的时间都没有的地步？还好同班的战友搞来了上届学生的答案，就这样大家连抄带讨论，每次交作业的时候还得连夜加班。要想一个人自己做除非是喝多了大内烧酒。每当夜色温柔中不断地往早已肿胀的大脑中塞入作业题并不断刺激其清醒时，心里的气就不打一处来。那些批评国内的教育是"填鸭式"教育的人真是一群"井底之蛙"。听说芝大的同胞们那里Microeconomics课情况也一样，天天作业也是"质量万里行"，而且都是费了半天劲做也拿不到几分，可人家那儿毕竟教课的是诺贝尔奖的得主，"受虐"也找对了对象。我们着又是受得哪门子洗礼？想到这不禁脑子里又浮现了"将军"那征服时的笑容，这样精力充沛的德意志人如果真的身上长满了如太阳光线般灿烂的金毛，脸上挂上了长靴，那岂不是会很对卫慧的胃口?

　　期末考试是"将军"向协手走向光明寒假的"盟军"学生发动的最终一次阿登战役。要想成功pass不外乎两种办法，一是参透那如"葵花宝典"般的课堂笔记，可要想通过苦练"葵花宝典"成就神功所要付出的代价也可想而知了。二是捉对"将军"出题思路，于是收集往年考题成为了在海外创造性运用"新东方"指导思想实行"两国一制"的实践活动。然而题库实在太小，而且题太难，就算会做相同类型的题目，考试时光计算时间就来不及。

　　"将军"在考前最后一次的发布会上放出讯息给人以一丝希望：考试分两批进行，第二批在第一批考试完三天以后进行，选择在哪一批考试随便，两次考试题目一样。我敢发誓我听到了那最后一句话。于是这里就有两个疑问需要分析：第一，岂不是所有的人都会选择第二批考？得出这种简单的"最优策略"并不需要复杂的"博弈论"知识；第二，这不明摆着漏题的可能性很大？纵然老师可以收回考题，他又怎么可能让所有第一批的考生集体失忆？

　　事态的发展越发出乎意料。首先，竟然有超过一半的学生选择了第一批考。经过抽样调查统计，显著性多数人的理由是为了提早考完提早放假。当然没有出乎意料的是，大多数人准备并不充分从而倒在了炮火之下。然而那些勇士们，尤其是一些外国学生显得很洒脱：为什么要那么在乎考试呢？当被问及如何准备考试时，对方总是一笑：你所需要做的应该是去relax，而不是着急去"临时抱佛脚"，因为道理很简单，做不出的再怎么样也做不出。好个"引刀成一快，不负少年头"！

　　其次更不可思议的是，当第一批考的学生肆无忌惮地将考卷带了出了考场时，"将军"竟然也没怎么加以阻拦。天下居然有免费的午餐？虽然冥冥中感觉到一丝不祥，担无奈这门课如浩海般的内容早已造成了"记忆不留痕，样题无价宝"的形势。就算这份考题不是真正的"真理"，至少也可能是个"局部的真理"什么的。总之最后终于有一份考卷到了我们手里，其珍贵程度不亚于二战时日本远东军在菲律宾埋藏下的亿万"山下奉文藏金"。顾不了那么多了，拿着这"不义之财"，心中一阵窃喜：现在的我离胜利只有0.01公分，七十二小时之后，这份考卷的出题者将彻底被征服。看来情报战在现代战争中的地位应该是不言而喻的了。

　　然而事情远非想象中的那样顺利。两天过去后，居然只有百分之八十的题目被解决了，还有些难点竟依然婷婷玉立。而且还是那句话，就算是会做，到时候仅三个小时的时间也肯定来不及。实在没办法，只好召开"政协扩大会议"了。于是原先从不同渠道搞到考卷并相互之间闪烁其辞的各党派人士，终于在共同的生存压力下走到了一起，红一方面军和红四方面军正式会师了！更有人提议应该请经济"高手"出山，但奇怪的是两位同样也要参加这门考试的"高手"居然还不知道有这样一份考卷流传在民间。

　　考试前两个小时终于找到了两位"高手"。两位"高手"刚考完了经济系的另一门考试，看那样子早已内力尽失，气息焉焉。对于我们手中有这样一份考卷他们略显惊讶，不过稍作思考后他们便放弃了努力：不会做。这样的情况并不出乎我们的意料，我们并不认为我们两天的集体努力会不及某位"高手"两分钟的灵机一动。只是令我们不痛快的是两位"高手"消极的态度。他们并没有指望弄懂这份考卷，甚至没指望记住这份考卷考了些什么。他们的理由很简单：不一定就考同样的题，而且就算考同样的，这么短时间内做不出就是做不出。这使这份考卷更值得怀疑，更糟的是看着那一堆乱七八糟涂着我们东拼西凑出来的答案的纸，我们都没有信心考试时能准确地再将这些内容完整地重现在试卷上。然而事实是，就算不考这些，除了这根 "救命稻草"外，还有其他的"诺亚方舟"吗？连这份可能性最大的"准考卷"都解决不了，还有什么其他的指望吗？时间在一分分地流逝，唯一的希望只有是"将军"不做或只做较小的改动，我们还可以尽可能地缩小损失。

　　时间到了，我们走进考场。"将军"早已等候在那儿。在发考卷之前，"将军"致了针对这门课程的简短的谢幕之词，其眷恋的笑容让人联想到小学课本中都德的《最后一课》中那法语老师的神情。那时候正是"将军"的祖先－－普鲁士军队的侵入造就了那伤感的最后一课。如今"将军"同样转过身去拿起粉笔，然而黑板上出现的却不是"Viva, France"，而是一张简单的地图。"将军"转过脸来微笑着对我们说，考完试后他请我们去图上标明的那个小酒吧喝beer。那充满善意天真的微笑和特有的日耳曼豪情令得一股莫名的羞愧感在我们心理油然而生。

　　考卷终于发了下来，在那一刻，一切都结束了。

　　那是一份完全不同于我们"破译"的那份的考卷，尽管题目类型有部分相近，但难度反而大大提高。弹指之间，头脑中早已拼凑有序的思路重新又被打乱成无序状态，"将军"仅用一纸考卷就向我们无言地阐明了史蒂芬·霍金在《时间简史》中反复论述了的一点：你无法靠记忆"未来"从而逆转心理学时间箭头。抬头看着"将军"，那善意的笑脸似乎早已转变成了华夫人对唐伯虎那呲牙咧嘴的笑脸："嘿嘿，你有张良计，我有过墙梯。"再看看我们手中的试卷，那怎么还可能会是我们费尽心思要点中的"秋香"？甚至有些新概念在两批考试的试卷中都出现了，然而在第一批的试卷中，"将军"还予以解释，而在给我们的试卷中，"将军"居然连解释都懒得解释了。这说明了什么？很明显，"将军"和我们"盟军"心里都一样的清楚。

　　怎么办？我们根本没有资格跳起来指着"将军"的鼻子痛陈道德操守问题，而且时间也不允许我们搞清楚究竟是应该打落门牙往肚里咽还是留着门牙咬他一口。我们所可以做的只有用连思维都似乎跟不上的书写速度，将我们自己也不太明白，老师可能也看不懂的另一份"般若波罗密经"涂满考卷。事实上我们根本没有靠打"时间差"占到一丝哪怕是心理上的便宜。

　　也许是一生中过得最快，最没有感觉的三小时结束了，其间的这场考试之于我们，就如同滑铁卢之于拿破仑，诺曼底之于隆梅尔，中途岛之于山本五十六一样。当我们手中的考卷再次被送回到了"将军"的手中时，考场内外便顷刻间充满了由各种语言组成的哀叫。

　　"It is over!......"

　　"完了！......"

　　"#@$^&*~......"

　　"φδαηκιρ......"

　　"※§‰№￥￡......"

　　...... 然而我们已没有机会改写历史，而"将军"却依仗抛弃六十年前第三帝国装甲师团在敦刻尔克海滩上对四十万英法联军的慈善心理而谱写了新的历史。万念俱灰时好象《大话西游》中的唐僧来到了我们面前不紧不慢地教导：

　　"所以说作老师就和作将军一样，不能有仁慈的心，没了仁慈的心，老师就不再是老师了，是将（僵）师（尸）。"

　　又是一个宁静而温柔的夜晚，不同的是在这个夜色中不再会有做不完Microeconomics作业。在前三次中从未体会过的"终极"的感觉中，这第四块"三明治"已勉强下肚，虽然结局是不成功的。然而故事并没有结束，下学期还有另外一门这个老师教的Game Theory在等待着我们中的幸存者。原来如果那个记者没有死在第四块"三明治"下，他很可能也不会就因此一辈子不再吃三明治了。他还是得吃，因为就象一部平庸的美国喜剧片《黑拷贝》中主角说的那样：三明治是百年来受苦受难的结晶。

　　它还是主食，它就是生活。

　　这还不是结局。

　　接下来是寒假，转眼一周后又到了圣诞节。然而我们还没有机会高兴起来。一是因为在假期中我们还要不停地忙research，直到现在我还没抽出时间去给自己做顿饭，将堆积如山的脏衣服洗了，以及用轻松使自己从期末考试的恶梦中恢复元气。工作只是使我暂时忘记了还有这么档子事。二是在这个南部小城中，冬天没有雪。

　　没有雪的圣诞节和没有礼物的圣诞节一样缺乏光彩。

　　圣诞节的前一天，我们仍照常来到学校。忽然有人提醒期末的成绩应该已经出来了，可以在网上查到。一想起期末就又想起了Microeconomics考试的那个月黑风高的夜晚，它能使我体会什么是美国的"天才教育"，也能使我更加明确地意识到我其实不是一个天才。说老实话，成绩对我们来说已经没什么意义了，只要不太差，到PhD毕业时没人在乎你的成绩是多少。只是在Microeconomics这个问题上总还是有点觉得对不起自己，从来读得没这么辛苦过，到头来还考得这么差，真不合算。这辈子也不是没有考差过，只不过与其花了很大地力气而考差，还不如压根就别花力气去考差，这样还能顾上些其他的东西，说白了前者从经济学角度来说就不满足所谓的"帕累托效率（Pareto Efficiency）"。如果你的收益一定是"差"，那你为什么要多花成本呢？

　　结果成绩出来了，我期待着能有一个"B"，但我眼前却赫然出现了一个"A"。看来今年的圣诞并非完全没有礼物。

　　说老实话，这样的成绩并没什么可值得高兴的。第一，如前所说，它的意义不大；第二，很多其他人也得了"A"，那帮人也是考完后以为自己非拿"C"不可的人；第三，就算是"A"，并不能说明我实际上考得很好。以为我的确还是考得很差。只不过那老师，就是"将军"，并不是完全按照对错来给分，而是按思路。也许我那堆没有做对的题中可能有些"将军"认为可贵的思路。其实这对我来说很占便宜，因为我一直就是那种灵感有余而严谨不足的人，很多思想我都无法坚持到最后把它彻底完整实现，"将军"这样正帮了我的忙。

　　但人是有虚荣心的，尤其是读书人。于是我还是不禁沾沾自喜了一把。同样人是势利的，一下子在我们的谈论中，"将军"的民意支持率升得恶高，这个"柔情铁汉"一下子又赢得了无尽的赞许，那架式可比几个月前参选的戈尔在民主党全国代表大会开完时的势头猛多了。

　　短暂的热烈过后，终于一切又恢复了平静，就好象刚考完的那个夜晚。忽然间我又想起了那一夜"将军"在黑板上标明的那个小酒吧。在那个一切都结束了的夜晚，会不会有人去陪"将军"喝酒？还是"将军"在"战后"的寂静中一个人独饮？后者的可能性很大。经济学家很多很寂寞，因为许多经济学模型根本就是少数人的游戏。也许"将军"根本就没指望我们能做出那些试题，他只是希望我们思考。也许精力充沛的"将军"确实好战，但对手不是我们，而是那些更加深不可测的经济学问题。虽然"将军"因为资历还远称不上经济学界的"大腕儿"，但我还是曾无意中读到过"将军"发表在一流经济学杂志上的文章，那深度又岂是那份小小的期末考试卷所能比的？也许真的有什么难题困扰着"将军"，他不甘寂寞，让我们陪他思考。我们当然没能力帮他解决什么问题，他让我们思考的肯定也不是他自己正思考的东西。然而也许在那个"全民"都为试卷中经济学问题搅尽脑汁的考试的夜晚，也许在那一刻，经济学不再是少数人的游戏了。也许在我们走后，"将军"还有新的战场，只不过此时"将军"孤军奋战，挑战的是其他难题和自己的思维。对于一个真正的研究者，失去思考就如同对于一个象"铁血"巴顿那样真正的将军失去战争一样，那就如同失去了生命。不同的是，如果是真的战争，我们可能都无法幸免于难，而在这里，我们在"将军"手下安然无恙。

　　不管怎么说下学期还要上"将军"的课，还会碰到"将军"，也许还会有无尽的作业，高难度的考试，和百思不得其解的题目。不过不同的是，那时不管考得多差，一定要记得去陪"将军"喝一杯。

　　"将军"，圣诞快乐！
