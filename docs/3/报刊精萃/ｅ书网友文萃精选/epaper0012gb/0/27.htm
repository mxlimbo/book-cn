门

lnj（askrin@china.com）

　　在以前，身材肥胖与否是辨识领导的一个重要线索。而如今，减肥中心遍布，身为领导却瘦骨如柴的也遍国都是。这种领导通常是能带领一部分群众有效地快速致富的。所以，我们迫有及待地要找出另一种特征来辨识这些领导，以便更好地为人民服务。经过长期的观察，我选择了门。

　　每个领导都有家，有家就有门。而这些门又具有一些相同的特征。下面，就让我以参观的方式带领大家来看看。

　　首先，你看到的通常是一道大铁门，而这道大铁门一定要装修得富丽典雅，而且要能衬托出其领导地位。通常，滑动铁门是官阶最低的官爷的。而到市级领导干部，门口往往还要站上一个荷枪实弹的卫兵，以显其在人民心中所处的重要地位。到了某些高级干部时，铁门也就进化成自动门，有的还装有红外线自动报警器，毋庸置疑，这对其人身安全起了很大保护作用。这也是好人短命，而坏人却长命百岁甚至遗臭万年的缘故。

　　走进大铁门，到其住所后，如果没有出错的话就会看到一道防盗门。对，防盗。因为当官嘛，总有许多重要的文件或隐私要锁在家里。否则丢了，人民的利益就会受到损害。但其真正的作用是，锁住人民的嘴巴，以实现其地方专制的目的。开了防盗门，又会看到一扇朱红色的木门，上书两个苍劲有力的墨字"人和"。不要被骗了，这是一种障眼法，用来迷惑一些并不雪亮的眼睛。看到这种情况，我们就会知道，他大约已经众叛亲离了。所以，就要躲在门后烧香求佛，示神明保佑其仕途顺畅。所谓有因就有果，这种官不久主会受到神明的眷顾，受召升天与它们为伴。

　　在房子里，我们转了一圈，发现一个极其有意义的门――后门。你去拜访一些领导，从正门走一定看不到人。这时，我们不妨逆过来，走一趟后门。这些领导，大约是对自己管辖地区的治安非常满意吧，这后门一直是对外开放的。这也是诸位领导们与外界保持信息交换的唯一通道。不要怀疑，这没有与防盗门矛盾，防盗门是防某些有心人士的，而后门则是招贤的地方，而且通常拴着一条看门狗。所以，奉劝一句，两手空空的人勿进。

　　步出后门，本来以为已经参观完毕，孰不料，这领导家还真是机关重重。我们差点误入一扇门。而门后，就是上述领导的最终归隐处。这是什么门？这是牢门。
