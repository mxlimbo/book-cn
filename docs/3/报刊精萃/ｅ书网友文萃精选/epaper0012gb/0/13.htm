摘下一颗孤寂的心

思漫（simansiman@sina.com）

　　此刻，我呆在热闹城市的偏僻一角，想念并不太遥远的你。你的名字充满淡淡的书卷味，只要在我意念中出现，心就变得温暖而潮湿。我想，如果不是你激情的渴望，我会摘下一颗孤寂的心走近你吗？

　　夏初，我带着好奇的心走进了网络。在那乱糟糟的聊天室，我明白这里并不适合我的个性，网络聊天的新奇两天后便荡然无存。我想逃离这热闹的场面，寻找一个安宁的角落，可网络的魅力不包括"安静"两字。

　　那天例行公事般在里面呆了几分钟，在我想出门的瞬间，你向我打招呼，出于礼貌我也点头，网络作为虚拟社区也有其生存法则，我无意破坏。不经意间我们谈了很多，时间眨眼从我们身边溜走，你成了我第一个聊友。

　　聊得多了，挑剔的我发现你并不俗，我们彼此淡漠而又真诚地交流思想。不知何时，我对你有了一份眷恋之情，而你越来越变得烦躁不安。我们彼此小心的避免谈情感话题，毕竟我们不再年轻，我们必须面对太多的现实，我们必须承担太多的社会责任。你的出现丰富了我的网络旅程，你温柔的体贴慰藉我的孤寂灵魂。我们的接触就是在这种不迷人但很动人的情况下进行。

　　在长长的相思，苦苦的煎熬后，我出差到了你的城市，我们相见却是两无语。在一个雨后的黄昏，你邀我游泳。往日清澈的水面变得浑浊，滔滔江水如我们皮肤一样，只有两岸的芦苇，绿的醉人，我们放弃了游泳，选择了草坪，听柔柔的江风……

　　话说到很多的时候你问我：为什么是我？

　　我说：因为你不太帅。

　　你笑了，笑若灿烂朝阳。我也笑了，如涩涩青果。

　　你突然说：咏萍，嫁给我吧！你面颊潮红，期待的小眼痴痴地望着我。而我脸上刹那飞出两片红云。

　　你急忙补充：我不是开玩笑。

　　听了你的话我凝思良久，默不作声，你轻轻用手托起我的头，我早已是梨花带雨。你不知所措又极为怜惜为我拭去一脸泪花。"对不起，咏萍。"我摇头，泪眼迷离地望着你。为你一个永难实现的邀约，我愿意守侯千年。你冲动地把我搂进怀里，狠狠地吻我。你呢呢道：你让我疯了，为什么不？为什么？

　　那晚是一夏最清爽清新的夜，我们一遍遍数桥上来来往往的车辆；看夜幕天水间偶尔随风掠过的水鸟；看远远近近的点点星光。当露珠打湿你的额头我的发鬓，已是凌晨时分。你说：今晚不回家。相拥走上长长的街，路灯将我们的影子拉长、缩短，心里异常的幸福，我们只想永远这样走，不要声音的陪衬，不要醒来。那个黎明，我们坐卧沙滩看彤红的太阳徐徐升起，它点燃了我爱的火苗。

　　我的旅途不再单调平淡，你用男人的温情一天天渗透我落寂的心灵。我爱了，没有更多的原因，没有更好的理由，爱上一个虚无又真实的你。坦然说。你不象我文章中杜撰的凄婉动人的爱情主角，你很特别。我知道，你不是我虚构的恋人，你之所以独特，因为真实的存在，你的一举一动一言一行生动着我的生活，你不再虚无。

　　我们依然游泳、依然散步、依然喝茶。记得那次在海底世界看鱼，我惊喜赞叹每一种鱼的可爱，你笑说如果我前世是鱼一定是美人鱼。几天的旅程如夏天的风景，风一吹就走了。我们真想变成鱼，自由游弋在海的怀抱。

　　我们挽手漫步河堤。你告诉我，不长的时间里，已习惯我的存在，习惯每天看见我甜甜的笑脸，习惯每晚轻念我的名字入睡。我又何尝不是呢？我一直都是那种不轻易接受也不轻易放弃的女孩。相识于偶然，相逢却是必然，爱也是冥冥中注定的！我一直这样认为。我所有的柔情便给这夜幕吞没了，就要和你分别，心中难以割舍。

　　回来后我才发现，有爱真苦，思念的利箭深深刺痛自己温存的心。我们又开始发电子邮件，开始通电话。那支写惯了深情文章的笔用来写情书时，竟然难以写下整段的文字。满脑子是你殷殷的话语和朗朗的笑声。你的关切，你的忧虑、你成熟而又易感的心，袭击了我。渐渐我无法摆脱你远方的那种声音，我是那种声音固执的感受者。无论我心情多糟，无论我发多大的火，只要一听到你富有磁性的声音，我的怒火便在你的温柔中消融。多少个夜晚，我目光游移，只为等待你的声音。你常常拿起话筒，和我一谈四、五个小时，我们彼此毫无倦意，感受你并聆听你成了我一大思绪。

　　渐渐我无法摆脱你网络的那份真情，我们总是在回忆相处的点点滴滴。你一次次追问我：你爱我什么？"我不知道。"我的回答让你失望。我发去E-mail问：你会不会尽全力改变我？你的回邮让我感动：你真傻，喜欢看你在坦荡的沙滩上自由打滚，也喜欢听你哭诉，看你忧郁的笑容，看你欢快的歌唱，看你愤怒，伴你沉思。你写道：我的最大痛苦是被别人背叛，更大的痛苦是背叛自己。如果你和我相处，而改变许多，那是我人生最大的失败。懂吗？看你的邮件让我落泪，无能如何我已为你改变初衷，我背叛了自己吗？

　　长时间里，我依然落寂，依然带着忧郁的眼神在这城市忙碌。我的心已千疮百孔，但我却一直保留着一块圣洁的地方，牵挂着你，惦记着你。我是你心中的一朵云，尽管我的泪水已干，但你看见风中有朵雨做的云，那是我不肯附落凡俗的灵魂，沧海桑田也不会把它湮灭。

　　我生日那天，蛮以为能接到你的电话和贺卡，可我失望了。次日在线，你问我生日过得好不好，有没有很多人为我祝福。你说想来看我，为我祝福，可怕打破我宁静的生活，你说昨天你醉得一塌糊涂，只为心口永远的痛……看你字字传达的信息，我仿佛看见你独自的彷徨，我甚至感觉到你的伤感情怀已溅落我心。你说好想好想赶来送我一份真实的礼物。够了，够了，我的泪水飞流在键，你的真挚情感，就是我最丰厚的礼物。

　　每天每天，我将快乐或委屈的信笺发给远方的你；每天每天。等你远程的邮件悄悄叩开我的心扉；等你熟悉的清音；等你点燃所有的灯笼，满世界寻找又一个我时，我却睡成一帘幽梦，梦中轻轻呼唤你的名字……

　　你在网络那端一次次轻问：想我吗？

　　我的回答简单的不能再简单，想！这一个想字，把我女性的矜持全抛弃，把我全部的爱情投资给一个人，累了，倦了，也不想改变。

　　想你，想成新月如钩；想你，想成素烟如云。

　　想你的日子，我熬一杯浓浓的咖啡静静地品味，默默咀嚼相聚的时光，咀嚼熟悉又陌生的你。

　　想你的日子，会有太多莫名其妙的思绪豁然布满全身血脉——甜蜜的忧虑、伤感的美丽、微笑的眼泪、流泪的欣喜。

　　我掉入了你温柔的陷阱，我们深深陷入痴迷的情感，我害怕，我恐慌。我不想和你谈自己任何的感伤，我逐渐变成了冰山，拒绝任何暖阳的照耀，包括你的关切。

　　你在网络那端感觉我的变化，你发来了一封封言辞诚恳的邮件。你说，我是个太坚强的女孩。你说，作为男人，被人依赖，被人依靠，被人需要，也是一种价值的体现。你说，如果仅仅为了维护尊严而拒绝关怀，不让我分担你的痛苦，你这样打击了一个男人做为男人的尊严。"咏萍，我希望看见你无暇的笑容，真的希望你快乐！"你最后永远会带上一句。

　　我好不容易竖起的坚冰再次被你击溃，其实我真的真的需要你！为了告别那伤感的季节，我剪下一地长发。短短的碎发在金色阳光下熠熠生辉，我写信告诉你，我要开始为你留一头长长的秀发，等待我们相聚的日子……

　　在那长发柔魂的季节，你抚着我的乌发，贴着耳朵对我说："咏萍，我想吻你，不长，就一生。"
