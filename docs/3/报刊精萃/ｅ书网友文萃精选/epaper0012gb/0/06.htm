冬 复 冬 来

安惠（wzj0729@sina.com）

　　我想哭，但是哭不出来，我怀念过去，想哭就哭，是多么自由，有一片睛朗的天空，任我遨游，释放所有的情感，缘份的天空下，片片黄叶在秋风中飘摇，渐渐的逝去的是我所有的情感，我所有的拥有的只留下漠然，我并非不想恋爱，我再也爱不起来了，像断了翅的风筝，飞向半空就跌落下来，理想只是手中的线，它永远握在我的手心，我无力张开手掌。

　　我很感谢你让我有了一份感动，我只怕我不能同你一起飞向蓝天，我会半路就跌落下来，我只想一个人静静的老去，不要再伤害到任何人。 也许你又会说我高傲，我只是不想别人再同我一起受伤，我无心亦无力。我眼中的世界，那是旁人的世界，我的世界里只有我一人，，慢慢地习惯一个人的生活，幸福着自己的幸福。

　　爱来爱去，爱到不明白爱到底是什么？我慢慢地不理解"爱"这一词，感动在心底扬不起半粒沙。爱去爱来，你还明白爱吗？冬复冬来，你还会冷吗？

——给 一凡
2001/1/11
