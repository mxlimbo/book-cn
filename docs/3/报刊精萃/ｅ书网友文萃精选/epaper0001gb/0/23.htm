血 迹

marret

　　在我的手上，居然都是血，那腥的厉害暗红快要凝固的液体，我拿起那本《挪威的森林》翻看着以前认为经典的片段，开始在那儿专心的看下去。那是个纯粹的爱情故事吗，可我象是爱上了那个**的“我”，其实拿什么词语去定义他我都觉得不是很恰当。而现在我只是想去了解他看过的书，想去聆听他喜欢的音乐，想去体念他的思想。想是着迷了似的，我沉浸在那个年代的那个时刻，好象能和他一起漫步，我在前面走他在后面，而偶尔才说出一两句不相关的话语。说是爱上了，其实也许是种依恋，就象小时候依恋安徒生童话一样，而用爱这个比较夸张沉重的词只是因为那已经足够让我不想别的可以依恋的东西，应该可以说是爱了吧，要重就让它重吧。放下书，才发现他终于留下我的痕迹了。

　　已经下午三点了，洗干净手上那些杀了用作晚餐的鱼后留下的血迹，想起晚上约了人见面，那本想养起的“长发飘飘”看来是寿命不长了，我还是回到那个短发的我。短发，听过那首歌吗，我听过但早把歌词忘了一干二净，只记得朋友说，你剃光算了，把那些个都扔的远远的。我笑了，说，我的形象也跟着跑了啊，呵呵，不过省下洗发水的票票来上网哦！可终是短发适合我吧，何必让自己被那些乱七八糟的多余物遮住视线呢，在这些日子里我有些犯傻，就是因为那不长不短的发，让我和它一起乱，它遮住我的眼遮住我的耳朵遮住我的嘴遮住我的脖颈，我无法呼吸我无法倾听我无法接触，只顾自己傻等，可总算是等空了才明白过来。呵呵，在找借口给自己呢。

　　走出路口，看到好友勋，招呼过后，说起厌。那个在我印象里单薄清高的厌，读完本科后，也不知道在哪儿混。从勋这儿，我知道了厌前些日子自杀过，虽然没成功，但最近很是需要人探望。再和勋寒暄几句，我说我有事要先走，他开始耻笑我，我大叫：滚。真的滚了呢。

　　回到家的时候，镜子里的那个我已经回到一年前的这个时候，想起些东西让我又开始麻木，找个地方躲起来，就是这儿了。（……）

　　路过超市，正看见一对男女，神色紧张的在商量什么，我很难得的有点好奇。他们从那个小弄堂里进去，我也跟着走过去，当然不能让他们发现我是在窥视他们。男的长发披着到肩下，满脸的胡扎，就看到这些，而女的，身材很好，穿着一身粉红色职业套装，在脑后长发挽起一只粉红的发夹，她始终都是背对着我。他们时不时有些争吵，我看了看手表，是早上九点四十五分，反正今天没事就无聊一次吧。

　　跟着他们越来越往里走，我有点点担心，如果出什么事，我该怎么办，我找到包里那把小瑞士军刀放在口袋里。前面是些富裕农民造的别墅房子，有些俗气的屋顶上居然养着几只鸽子，在造房子的时候想着怎么让自己多占点地，就剩下只可过两个人的通道了。那个男在前面走，女的低着头在后面跟，我走在最后，当然要远远的。走了好一会，在这些房子里左转右转的，我有点糊涂过会怎么出去了。终于他们停下脚步了，我看到他们说了一两句话就把一幢别墅的房门打开了。很快门就关上了，我发着呆，我在干吗？也许什么都没发生，好象自己是个癔想患者。我想找到回去的路，就往前面去看看，有没有大的出路。这时，我听到了一阵混乱打闹声，女的尖叫了一声好象就被捂住了嘴，接着是这物品的移动声。我的第六感告诉我，有事发生了。我心跳已经是100以上，我看了一下周围，这里几乎是没有什么人来往，人们都去工作了。我拿出小灵通，给我朋友打了一个电话，让他们过来。然后我鼓起最大的勇气，走过去，在门前我想了好一会，好象上面也平静了，没什么声响，我只能听到自己的呼吸声。我找到门铃按下去，等待，三分钟也没人应，我好象觉得手心湿了。我试着想去打开门，锁着，我拿出一根小发夹，以前也曾干过这事，不过只是为了开忘了带钥匙的门。很顺利，我轻轻推开屋子，看到一个大厅，正面是楼梯，下面的摆设我也没多注意就往楼梯上。我慢慢的走上去，这个屋子装修的很是别致，在外面根本无法想象主人有这样的情调。楼梯转弯角墙上挂着一张向日葵的画，我本对这也没什么研究，但画和房间很是协调。我走上去，看到一个很大的客厅。低矮的沙发摆成L形，茶几上有一只杯子，杯子空着什么都没有，只是在杯子旁边放着一把打火机。我立即把目光投向别的房间，走过去看到一间主人房，门虚掩着，但我看到地上居然是正一点点往外流却快要凝固的血……。

　　也许是我的职业让我并不恐惧鲜血和死亡，我打开门，我想这时候我的表情一定是空白的那种，因为屋子里除了地上一滩的血什么都没有。血倒是很多，有划过的痕迹，接着我环视屋里的每件东西。先注意的是中间墙壁上一张画，好象在哪儿见过，不过怎么就想不起来了。然后是房间里豪华的装饰，梳妆台上摆满了名牌的化装品，不过很是杂乱，女主人不是个喜欢整理的人。大理石的地板亮的有点刺眼，透出冰冷，可屋里的东西似乎和它一样没什么变化,似乎只是我一个人在幻想着发生了什么事。我还是不甘心就这么走了，打开可以打开的柜子和抽屉，却都是空的，什么都没有，我在脑子里想着各种的可能性，没有正确的答案。我手心又湿了，不知道下一步该做什么，就顺势坐在了那张大床上。突然想起新闻里说到谋杀案里有把尸体放在床下的，我一下子象抽筋似的跳起来，脑后的头皮很是酥麻。我不知道自己有没有勇气去抬那张床，也许我会看到恐怖极了的血腥极了的让我都不能接受的现状。我感觉到背后有人，我往后看，什么都没有，可总觉得好象有人在看着我，在某个角落。我额头开始出汗，顺着鬓角一直往下流，可能是这六月的天太热亦或是紧张的缘故。我有些窒息，手足无措，整个空间象是凝固了似的没有一点声响。这时我的小灵通响起来，把这儿所有的生灵都吓的全身发软，我按下通话。是朋友，问着我的方位，我无法告诉确切位置，只是粗略的说了点，让他们找吧，好象是很难的事。我开始冷静下来，想着继续怎么做。我走到窗边，靠着往外看，也没动静，而那一男一女象是瞬间消失似的。我开始回想所有的过程，还是没一点头绪。这时，我听到了楼下开门的声音，有人进来了……。

　　我在犹豫该怎么做，最后象大多数那样躲进了大衣橱，开了很小的缝隙让我可以看到关于屋里的事情。接下来更让人诧异的是，进来的人居然是个男孩。看上去大致只有十八九岁的样子，皮肤很是白皙，他象是在找什么东西，也开始翻看那些抽屉，当然很失望的他也坐在了床上。现在我可以看到他的五官，长的很是清秀，有一双大眼睛，似乎好象也在哪儿看到过。他用手摸着床，象是想起了什么，我想也许那床上有什么让他感兴趣的东西了。他走出房间，一会拿着一把剪刀回来，开始剪开窗沿，他背对着我以至让我无法看到他所做的，可我想他在找东西。他好象停下来，手里摸索着，我听到奇怪的声响。
他看着手里的东西，可我什么都看不到，好奇的很。这时，他速度很快的把东西整理好了放进自己衣服的口袋，我想那东西一定不是很大。他环视房间的每个角落，也许是刚才的动作让他脸上泛出些红晕，而我注意到他手上居然都是血。猝然我开始发冷，象是被关进了狭小而不能动弹的箱子，已经感觉不到心的跳动。他起身，看了看手，我想他会去洗手所以他走出了屋子。我只有等待。

　　大概过了半个钟，我猜想他应该已经离开，因为我几乎听不到任何声响。我打开衣橱，全身有些不适，我快速的跳了出来。我看到了那床上的东西，凌乱，还就是血迹。是些很污秽而不明的物体，我正想走过去。这时在我身后突然响起一个人的声音。
　　“你看到什么了？”

　　我回过头，看到门口站着那个男孩，他面无表情地看着我。我停顿着，然后回答：“不知道。”

　　他突然在嘴角露出一丝笑，那种笑似乎更让人无法接受，然后说：“哦，是吗？！”我有些停滞，我看了看周围，然后也冲着他笑了笑，故意笑的大声些。他向我走过来……

　　我站在那儿，而他就这么走到我面前，他看着我的眼睛，而我发现他居然比我长的高大。他就这样站在我面前，而我都能听到他的呼吸声，我本打算试试能不能把他打倒后逃出这个鬼地方，可我始终没动，其实我很清楚自己那细胳膊的能力。我，他，就这样面对面站着，大概过了数个世纪，我听到他说话了。
　　 “你是没看到什么对吗？”

　　我开始张开嘴，想说，可，，，，，，不知道自己到底想说什么。

　　“如果你没看到什么，那你走吧。”

　　我想利马抬起腿就走人，可还是没动，我听到自己说话了。

　　“我看到了，看到一男一女，然后是你。”该死的，就这样说出口了。

　　他突然真的笑起来，然后象一个愚蠢透顶的人说了最后一句话。

　　“你走吧。”

　　我怎么走出来的，我后来也回想不起来了，只是记得走出那座别墅后，左转右走就到了大街上。我听到一种声音，那是肚子抗议的叫嚷。我到了四海豆浆店，点了最喜欢的甜豆浆和甜饭团。可我突然想回家，想极了那张只属于我的床。

　　我躺在床上，就这样睡去了。突然我看到了那男孩的眼睛正看着我，我听到他说，你什么都没看到啊，而我却叫嚷着，我都看到了我知道了。我醒过来的时候，我看了时钟，凌晨三点。前年父母一起去了父亲的老家青岛，而留下我一个人在杭州，我就靠着自己在一家外贸公司里做跟单兼一些文秘工作养活自己，还算过的去吧。我走出家门，小区里那一幢幢黑色的楼房旁有昏暗的路灯，我看到转角上一块计划生育的牌子写着“优生优育，只生一个好”。想起他曾近拿这个和我说笑，在他家的楼下也有一样的牌子，他便说我偷拿他家门口那块牌子放我这。我说，呕吐呕吐。走出大院，我看到黄色路灯下通明的马路，偶尔有的士在我面前缓慢路过。我在一家银行门口的阶梯上坐下来，一阵风吹过，我感到有些冷了，缩了缩身子，才知道自己只穿了一件廉价的白色T恤。

　　这时我看到一个人，正朝我走过来，让我更觉得该回家去找件棉袄之类或是干脆盖上厚厚的被子更解决问题呢。

　　“你也住这附近？”他，对，就是那个男孩。

　　我只有笑了笑，点了点头，而不知道说什么。。。。。。

　　你们一定以为我在瞎编故事，其实这些都是曾发生过的，哪怕是在梦境中，那些恐慌、惊栗、血腥、莫名、无聊都在你们身边无时无刻的发生着。你们都没感受到吗？我坐在夜了的大街上，看着曾有着来往不息人流的道路，那些被践踏的土地，那些被侮辱的空气，难道都可以视而不见吗？那还有人感受到我吗？那我是谁？我究竟又是谁？这个问题被问过千万遍了吧，可还是没有答案。而生存还是死亡？噢，那是个对现实选择妥协还是抗议的问题。记得我站在山顶，看着可以让自己瞬间消失的深谷，我数落着自己的生与死的理由，我幻想消失后那份轻，而重的是什么，是责任吧，是生之起源的责任。

　　而最终我仍这样走进人群，象是什么都没发生过一样生存，是，我选择了生存。

　　我忙碌着，为了所有的过程，也许你们会说我错了，那样的人生观是让人可以接受的吗？我那种失望、空洞以至轻飘的已经不能被哪根绳索牵制，而或许在将来的某个时间我会嗤笑一声，对我走过来年轻而幼稚的日子。

　　现在的我却还幻想有一天，割开手腕上的动脉，那充满搏动的地方，鲜血象是被释放喷涌而出，我用眼睛羡慕地看着它就这样做着自己想做的事。而地上那些斑驳是我的血迹。

　　又插开主题了，那是些梦话吧。

　　那晚我和男孩说过什么？我好象没多说什么就象逃亡一样回了家。而他居然会说“再见”。进门，我打开冰箱的时候看到那瓶沙司，我喜欢薯片大部分的原因是和沙司一起吃很是美味。我拿出沙司的瓶子来，倒出一点在碟子里，用手指点着吃，想起小时侯母亲喜欢把用手指点些炼乳给我吃，那个年月的好东西，到现在已勾不起我的食欲了。

　　时钟敲响五点，我想我该睡会，早上七点要起床去上班的。

　　天亮的刺眼，我才发现我没有拉窗帘，我走出盥洗室的时候发现自己的衣服已经是皱的厉害，换上一件绿色的连衣裙打算上班去了。而电话居然响了。

　　“晚上有空吗，一起去个地方。”原来是勋，“这时候不打给你怕找不到你啊。”

　　答应后我走出家门。

　　我和勋在壶笑天的门口见的面。他穿的很讲究，带着微笑在那儿等我，我远远的就看到他傻傻的样子坐在自行车后座上。我走到他面前，说：“比我早啊！”他笑的把眼镜都快掉下来，“那当然，是我约你呀！大小姐！”

　　“带我去哪儿？什么好玩的地方？”

　　“哪有好玩的，是个很有意思的地方。”

　　“哦，你什么时候有空找有意思的地方了，嘿嘿，不会是想找个相亲的绝佳地点先去探察探察吧？！哈哈~”

　　“是啊是啊，你这也知道啊！”

　　我们一起骑上自行车往柳浪闻莺方向去。

　　在长桥公园，我们停下来。我说，怎么到这儿？他笑了笑说，有人想见你。

　　在湖边，远远看到石椅上一个瘦弱的背影，长发披散在背后，是厌。

　　她眼圈隐隐的显出些黯色，没有一点血色的脸，连唇上都不施一点红脂。厌看着我说，“你来了，我不知道是不该见你，我知道你一定有些恨我，可还是要来见你。”

　　如果说我现在唯一可以做的恐怕也只有很淡的笑笑了，说声：“那些过去的我们不必要再计较了吧。”呵呵，可心里是没有可能没有一点痕迹的忘记过去啊。

　　而那个叫帆的人在我的大脑里也难免还留有些东西，至少他借了我的CD还没还。厌在我不经意的时候把他从我身边带走了，好象很多爱情故事都这样发生过，我也不想细细去追究谁对谁错了，感情没有对错是吧，我想。

　　“他在医院，想见你。”厌有些无力地说。
　　“是帆？哦，怎么了？生病？”
　　“医生说他没多久了，胃癌。”

　　我，手心里又湿了，而空气象是凝固似的，没有一点风。

　　半小时后我去了医院。

　　医院，对我而言可能是除了家再熟悉不过的地方了，那种气味也没让我感受什么，而白色在人的眼里也许是所谓的恐惧吧，而我径直往里走上二楼。在病区的门口写着“肿瘤科”。我是一个人来的，勋和厌去了哪儿我不知道，我只是对他们说我想单独和他呆会。
六号床位，我看到了他，除了脸色苍白了许多似乎并没有多大的变化。他正坐着发呆，不知道他在想些什么，象是一座雕塑似的没有声息。我的出现他并没有所动，走到他面前，我的声音让他抬起头来。

　　“好久没见了。”

　　他眼睛盯着我看着，深深的让我有点不知所措，也许是因为他的近视吧，看了许久才知道是我来了，我只能这样解释给自己。他动了动嘴角，象是想说些什么，但什么也没说出来，然后他慢慢的往后靠去。我帮他躺下，也许是没有生存的意志，我知道现在他完全可以坐着或是走着象个正常人，而他却似乎只剩濒临死亡似的无力。我笑了一声，说：“你不欢迎我吗？”他不说话，手却伸出来想要抓住什么，我把自己的手伸给了他。他紧紧地攥着，惟恐就这样再也无法触及。我离他近了些，说：“现在觉得有什么不舒服的吗？”他总算摇了摇头，说了第一句话：“没有，很好。”

　　接下来是停滞，谁也没说一句话，而空气好象也就这样滞留着，而象是在另一个世界我和他面对面说出了心里想说的。

　　“爱一个人是什么感觉？我始终以为我是爱你的。”他说出口的时候我的手心又湿了。

　　“而现在你出现在我面前我又开始觉得我是多么的自私，好象我那自私的爱只是为了自己，而并没考虑你以后也许会为了我而难受。而我做不到，只是想在我离开这个世界之前你能听我说完我想说的。也许对你来说，是那么的不公平，因为我总在欺骗你，哪怕是善意的，可本质也是一种欺骗。你会原谅我的，我知道。可我会原谅自己吗？那一定不会。我在你来之前想了很多，想和你说很多，可现在好象什么都没有了。是的，我不知道要说什么了。”

　　“我都明白，可我一样不原谅你，因为我始终是我，不会可怜或是同情曾那样伤害我的人，我本就不是个善良的人，干吗要让自己受伤害而没有一点还击？！你不用说什么，我想我知道你想说什么，也别认为我无情，你在做这些事的时候本就没把我放在某个位置，现在说什么对我而言没有任何效应了。别想我真的会同情你可怜你，谁又知道谁先离开这个世界，谁又知道谁真正值得同情呢？！我想这是我第一次来看你也是最后一次，我不想听什么你想说的那些，只要我做过我想要做的一切也就这么过去了，不需要留下什么。我想现在我已经做完了我想做的，仅此而已。”我感觉着自己的脸越来越僵化。

　　他不说话了，但仍紧紧的抓住我的手，我并没有想着去挣脱。过了数分钟后，护士来给他打针了，我起身走了出去，再也没回头。

　　我又回到忙碌中，我要看书要考试要上班要写文章，生活也许就是这样，在莫名中时间就这样过去了，而回头看看也没什么后悔的，快乐过了痛苦过了，就算是也会怪缘分没有我的份，而我只有匆匆再匆匆的走过。大概太会幻想，那些浪漫的场景一幕一幕地上演，我却在那个阶梯上远远的看着，动情的时候流下几滴眼泪。可屏幕上打出“END”后居然会留下一片血迹，我有些茫然，而这时好象只有我一个人。是不是在人的心里都会有那片血迹呢？为逝去的留下的痕迹？而红色总是那么让人心惊让人刻骨，血就是红色的，而我不希望那是泪迹。

（完）

为什么我总不能写个完整的结局出来，不太清楚，而这次我又多了很多客观问题，这个星期末要接受很多考试也就没时间写故事了。但我是那样喜欢写的感觉，在我下班的路上在我的大脑里居然是“我”和“他”“厌”“勋”“男孩子”等等，我可以想着每个场景，象是自己在自编自导让那些人物说着他们想说的话做一个性格所支配的事，很有意思。可最近我没有时间了，有很多现实的问题等着我去解决，我实在不希望自己象条鱿鱼那样被某些事物东动西挪的，饭碗先拿在手上再说了，呵呵~有空再见了！
