永不言败

醺儿

　　由史蒂芬索德柏执导的，朱莉娅.罗伯茨主演的《永不妥协》一片让我深受感动。

　　故事的女主人公是位经历了两次离婚，并拖着三个孩子的单身母亲，在一次十分无奈的交通事故之后，这个一贫如洗，既无工作，又无前途的可怜妇女的帐户上只剩下十六美元，几乎到了走投无路的绝境。

　　这部片子的开始，已经为我们展示了一个狼狈不堪而濒临绝望的女人的形象，她还美丽漂亮，但接踵而来的一连串不幸的事情却总是将人逼迫得透不过气来。于是她的脾气变得暴躁不堪，对邻居无意的噪音骚扰更是怒不可遏，但她在怒骂之后并不能真正释怀，生活如何继续，这是摆在这个弱女子面前的一个重要的问题。

　　看到这里，我强烈的感受到那种叫做“残酷”的东西，而在绝望之时，她只得恳求自己的律师雇佣自己，在律师事务所所里打工度日。在调查一件不动产诉讼案中，她无意中发现了一些可疑的东西，在不懈的调查工作中，她发现了当地社区掩藏着重大环境污染事件，是造成致命疾病的根源，艾琳用自己的真诚赢得了全体居民的信任，成为他们的核心和代言人，终于使污染事件得到了令人满意的赔偿，并创造了美国历史上同类民事案件的赔偿金之最。整部电影让人感受到平常人真实的生活，也让人信服的看到一个弱女人如何于绝望之中开辟出一片崭新的天地。并在此份工作中寻找自我存在的真正价值，帮了别人也帮了自己。

　　当我们仔细玩味那种坚苦卓绝的奋斗过程，是不是会心痛，在现实生活中又有多少人背负着这份沉重，压抑而无奈着。

　　于我，感人的东西是在她的内心世界所袒露的勇敢、独立和坚持自我的个性，当外在所给予的一切信息都是不安和无助时，我发现她的潜意识当中所呈现出的求生的欲望是非常强烈的，其实这就是相信自己能够处理好任何事情，因为这是她必须要做的，也是唯一的出路。这种对未来抱有的坚定信心和希望的安全感，是唯一属于她自身的持久的安全感。财物终将会耗尽，名声会随毁誉而浮沉，爱情也不会亘久不变，但是惟有自我可以依赖终生。人始终要相信的是自己的内在力量。

　　内心强大，毫无畏惧，你就会活下去。

　　不知道在生命濒临绝望之崖的时候，我会有着怎样的坚持，那份勇敢、坚定和背水一战的微笑，会不会与她一样。
