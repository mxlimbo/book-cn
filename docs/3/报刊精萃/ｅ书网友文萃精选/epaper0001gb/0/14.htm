行为散记

秋歌

　　很久没有一个人在街上游荡了。晚上一个人走在街上，晚风习习，抚过裸露在空气中的肌肤，柔的恰似情人的手，温情无限。

　　漫无目的的走着，很想打通电话给某人，想听听那种遥远而熟悉的声音，想知道这个世界上远远的地方还是有人认识甚至是惦念的。只是如此这般的想着，却一路走下去并不想停驻。

　　路上的行人大都在打量着我，眼里的神情是我不能够确定的，或许是奇怪吧，亦或许是莫名。没有妆彩的脸在街灯之下，是一种苍白。不能确定自己在旁人眼中的样子，但我想,至少现在，此时,自己身上一定有着一种低沉的颓废的味道。

　　从一个路口到下一个路口，来到一条黑暗的小街，周围是寂静的。我停下了脚步，望向昏黑的前路，忽然有种放足狂奔的冲动，于是我开始——跑。开始释放、冲破自己。那感觉，很棒。仿佛一下子年轻了回去。

　　路边的大排挡生意好的不得了，入夏以来尤是如此。空气中弥漫着焦碳，啤酒，肉串，男人的体味与女人的香水的混合味道。一直以来，我都没有办法接受路边滩、大排挡的方式。除非是与我对坐的男人有足够的魅力可以使我的大脑中某些神经无法正常的发挥其功效。（一般我与女友是不会选择那里的，虽然那样可以使我们看上去豪气倍增）

　　在路边看到一个摆满瓷器的地摊，那些有着漂亮色彩与可爱形状的瓷器让我不由的停下了脚步。于是我想起自己需要一个水杯。当我在地摊前以从左到右，再从右到左的顺序徘徊了三圈之后，终于相中了一个长的高高瘦瘦的杯子。从形状上看有点像自己，掂在手里很沉，有种厚重的安全感，极淡的颜色也是我所喜欢的。在与摊主讨价还价之后，我拿起了杯子准备带它回家，让今晚我的无目的行程有一点的收获。然而，这时我却发现了杯子上竟有一点瑕疵。我不能完全肯定自己是个完美主义者，但最终我也没能带它回家。重新让它有些孤傲的站在一群瓷罐之中了。

　　而我继续漫无目的的上路。……
