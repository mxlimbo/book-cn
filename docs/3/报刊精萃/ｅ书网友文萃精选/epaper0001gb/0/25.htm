致吾友的一封公开信

秋歌 （viento2000@sina.com）

　　每次，每次，当你满怀希望的对我说，写些什么吧，我等着的时候，我都禁不住要感动。因为有一个可爱的朋友，在我的世界里与自己相知、相望。然而，那些因久未曾拜访而生疏了的文字，于我而言，要把它们合情合理的拼凑在一起是多么的不容易呀。那个曾经用笔当舌，在纸上流泻所有的言语的女孩子仿佛已经消失了，至少是模糊了，淡化了。久不提笔，久不思考，所有的感情都已麻木，不再新奇于身边的事物，不再为快乐而歌，不再为忧伤而泣，不再……

　　稍稍恐怖了一些，我还是没有那么的麻木不仁的。只是比少年时少了些许冲动，沉寂了一些而已。那些少年心事已然离我远去了。所有的瑰丽的梦和那些我们曾经感动过的许许多多都淡忘了，消逝了。于是，窗前灯下，尚未安眠的我，便和着繁星无数，漫想着前尘如梦。任思绪万千，如丽日下的沙，潮水不侵，嗅着微咸的气息，懒散着自己，放任着自己。

　　有人说，当你去拜访别人时，为你拿出糖果的是你的朋友，而为你端上一杯白开水的是你的挚友。想来这句话是有道理的。真正的朋友正是那些于你生命中默默相守，不求形式的人。忧伤相随，快乐相忘，当爱情为你而歌时，朋友给你一声祝福，一句珍重，便消失于你的生活中；当你为爱情而伤时，朋友给你一方帕子，一个有力的臂膀，让你籍此疗伤。

　　每当想于此时，我就禁不住要快乐，因为我有朋友，与我一起快乐吧，我的朋友。

　　因为——你有我

　　人间自是有情痴　此事不关风与月

秋歌
写于千嬉年前夕
