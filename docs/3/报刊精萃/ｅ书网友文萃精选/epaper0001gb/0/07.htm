女人心情（二）

秋歌（viento2000@sina.com）

　　刚刚与男友分手不久，不是失恋，因为分手是自己提出的。只是心中依然有一丝淡淡的惆怅与另一种解放的、重归自由的心情。这种感觉真的很奇妙：一个人的时候偶尔会寂寞，但两个人的时候却又觉得他仿佛入侵了我的生活。于是，很想让时间重归自己，完全的属于自己，可以任由自己支配。男友，不应当是被呼来唤去的，可以招之即来，挥之即去。那样对他太不公平，自己本性也不是那种人，不允许自己做那种小女人……于是，在不经意间说出了分手。可以想象的出男友的愕然，实在是没什么先兆，对他来说是没由来的一件事。只是想现在说对男友来讲也许愕然，但并不难接收，总比日久，更深的投入之后的那份伤害来的小的多……

　　夕日的朋友各自有了同伴，而自己依然是一个人的时候，寂寞会一点一点的吞噬自己。祈望着生活之中能有一些感叹与奇迹出现。然而，日子在手指缝中一天一天的流掉，一切如常，没有任何的不同出现……
