吴晓明

你说 你也喜欢孤单
从不说什么未来
你说 你已习惯孤单
爱你就不要再为难

偶尔
看见你还站在那个地方
眼中已没有了往日的神采
就象是无忧的平凡
如今
你已远离这个地方
而我却始终还在期待
就象是多余的答案

说不出心里有多哀
要怎样才能不再徘徊
说不出对你有多爱
只能站在原地等待
——等待
一个满意的答案
让我能用爱去感化——你那冷若冰霜的情感

(等待是一件非常幸福的事，也是非常辛苦的。本来想继续地等但…………….)
