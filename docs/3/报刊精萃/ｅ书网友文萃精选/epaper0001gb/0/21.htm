手 机 情 结

嘟嘟

　　日下同学聚会，手机密密麻麻放了一桌，如同商场里的手机柜台。漂亮女士的手机小巧轻盈，坤包里摸出流线型的摩托罗拉，盖子一翻，朱唇微分，低语含情，尽是淑女气派；潇洒的男士们则业务繁忙，左手麦克风歌未起，右手爱立信声先鸣，赶忙摸出去回话："喂，我在歌厅，你大声点……"远方的听众永远比面前的重要——也是社会通病。

　　报纸上说如今正"手机横行"。BP机被贬成了"屁屁机"；磁卡电话成了学生专利；手提电脑倒是新贵，惜其体积稍大，要秘书提着方显气派。唯有手机，火遍中华，市场大得让国外的手机制造商恨不能要胁本国政府，派海军陆战队来助其抢滩。

　　什么东西一多就要分三六九等，用价值高的东西不一定能证明品味高，但至少能证明钱包鼓。据说英国真正的豪门公子，从小就被教不要显山露水，衣着饰物不要名贵的，得体就好。身份藏在口袋里——他可以随手拿出一个一万多美金的打火机给你点烟——那根烟在天有灵的话，恐怕都要笑得腰折——火光一闪，身份又收了回去，只看对面坐的人有没有识珠慧眼了！

　　这种打火机在中国恐怕销路不大，大款们青睐的是上得了台面的洋酒及西装。XO当水喝是时髦，当街打手机也是早两年的时髦之一。现在不行了，在街上大声打电话的，要不在闹市区，要不就只剩了信号不好。手机带来的现象种种，都有点社会心理在做怪。

　　最伤心的往事当属模拟机，买进时万儿八千变了今天的白菜豆腐价，还在用模拟机的同事开玩笑说，这机子除了省话费，还可以打狗及防身——重量正合适。它象一个未老先衰的美人（只是老得未免也太快了点！），让曾经量珠而聘的拥有者变了黯然神伤，取舍两难。

　　未曾头一个吃螃蟹，当然也不会被螃蟹咬。我们这样的普通人等着等着，终于也拥有了自己的平价手机。信号强，功能少，短话多，长话少，关键时候挺管用，可以昼夜不关机。普通人也有普通人的小奢侈－－谨小慎微的自得其乐。

　　朋友当中有两个手机怀旧派——或者叫骑墙派，腰里总别着手机加CALL机的"双响炮"。我笑话他们说，再来个"商务通"，就成了广告活标本。回答倒也有趣，一个说手机上无时间显示，CALL机当表用；另一个说CALL机是女友送的，不能喜新厌旧。不过他那CALL机倒也争气，人前从不响——情意绵绵而默默无语，有这样的女友才是福分。

　　同窗中没有做到大老板的，自然也没有用高档手机的。广告中刘德华的"回头便知，我心才有你"是非常成功人士的风度，轮到我等不知何年何月，不想也罢！我认识个关之琳般漂亮的女孩，追求者甚众，有来势汹汹者发豪语："一旦追到，当先送她一部摩托罗拉V998！"我问伊："动不动心？市值四千多呀！"聪明的不回答我，放出一个比蒙娜丽莎还神秘的的微笑，莫测高深。

　　由他（她）去！好象卖手机的柜台，既有人老珠黄、门前冷落的"五折酬宾"，也有青春年少、美艳动人的"政策性价格上调"，既准你V998招摇过市，也得准我小老百姓的C2588暗渡陈仓。我的美人，应该是摆于柜台水晶格内的那一款，日日拂拭，历久如新，标签上赫然醒目——"非卖品"。
