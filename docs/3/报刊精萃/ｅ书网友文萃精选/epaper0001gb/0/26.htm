岁 月 凄 凄

陈威

　　 微微有些阴的下午，我站在车站等车。天色已经不早，街面上行人寥寥，偶而的几个也步履匆匆，所以当一个少女婷婷袅袅走过的时候，我忽然感到眼前亮了一下。

　　十七八岁的少女无一例外地充满了自信。因为自己光彩照人，所以都相信自己是美丽的焦点，相信整个一条街上，每一双眼睛都在注视自己。所以她尽力维持着自己的美丽，昂首，挺胸，目不斜视地迈着一字步，散开的长发在风中款款飘动，神情则一丝不苟如古希腊神坛前的女祭司。 我看着她艰难地沿街走过去，忍不住轻轻地笑起来。多美丽的女孩，白衣蓝裙，恰恰正是一朵欲开未开的小蓓蕾。虽然整整一条街上除了我，再也没有人去注意她，那样的清丽与纯净，那样刻意又自然。年轻多好，风霜还未侵蚀，忧伤还未袭击，正值韶华，无忧无惧，连她的高傲都是多么可爱。 与她擦肩而过的，是一个中年妇人，手提一只菜篮，步履匆促得仿佛要赶去打仗，顾不得头发有些蓬乱，鞋面上沾染了灰尘，更顾不上去感受路上是否有人在注意她。她只是急急忙忙地赶路，急急忙忙地用脚步去追赶流逝的时间。

　　女孩也许是去约会，在她这样的年龄，或许再也没有比约会更重要的事了。那么，那个妇人呢，想必是赶回家做晚饭，或许在她的世界里，再也没有什么比做一桌子好菜再看着家人一口一口吃下去更实在的事了。与是少女和妇人步履匆匆，在街上不经意地擦肩而过。

　　诗酒花开少年时。若干年前，那位妇人也正值豆蔻，也正是满怀挥霍不尽的青春。那时，她必然也是与擦肩的少女一样，明眸皓齿，是碰也碰不得的娇嫩。那时，在她的心中，也正是除了花与梦，梦与诗，再也容不下其余什么的。那么若干年后，那个少女呢？那颗曾矜持的心是否会在开门七件事中不落痕迹地变成不修边幅的粗糙与坚实？

　　“书画琴棋诗酒花，当年件件不离它。如今七事都已改，柴米油盐酱醋茶。”什么是前戏？什么又是后戏？只在那擦肩而过的一刹那，所有的悲欢聚散，风雨沧桑都暂时省略了。是啊，那一刹那间，少女可以成为一个平实的妇人，妇人却无法再能还原成少女。从书画琴棋诗酒花，一变而成柴米油盐酱醋茶，这中间的，就是岁月。
