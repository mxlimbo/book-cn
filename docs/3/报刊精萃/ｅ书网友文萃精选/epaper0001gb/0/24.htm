经典电影点评

草本植物

  1. 乱世佳人 我喜欢费雯丽说明天还有明天时绿油油的眼睛和赫思嘉计算明年棉花收成时的经济头脑。
  2. 永不消失的电波 我喜欢孙道临最后一刻的从容不迫，和导演给了他从容不迫的时间的睿智。并且还喜欢这种手法对以后的电影、小说、电视剧等描写壮烈牺牲结尾的深远影响。
  3. 简爱 三部我都喜欢，每当我看到选拔出来的女主角并不像书上描写的那样其貌不扬，就有种由衷的欣喜。
  4. 七十二家房客 我喜欢赵丹睡在躺椅上，嘴里哼着：“一块金子变两块金子，两块金子变四块金了，四块……”的那副表情。（他扮演的角色在炒金价）我若炒股必定这副尊容。
  5. 熊 我喜欢熊的宽厚仁慈，也喜欢人的残忍贪婪。两者我都爱。
  6. 霸王别姬 它让我思考男女性别究竟是由心理还是由器官来决定。
  7. 甜言蜜语 我喜欢它的片名，尤其是当我知道它的男主角是个哑巴。
  8. 夜访吸血鬼 我喜欢汤姆克鲁斯，那时他还不算老。还有那个与人仿若的孤独的世界。
  9. 欲海沉浮 它是电视版的〈荆棘鸟〉，我喜欢，拉它凑数。
  10. 铁达尼号 我喜欢它的票房，除此之外，没有别的。
