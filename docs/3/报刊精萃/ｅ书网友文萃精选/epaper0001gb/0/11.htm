记忆拾遗（二）

草本植物

蚕豆花、爱情故事与狗

　　写到这个题目，我就想到了足球与拳头。两种事物原本是没有关联的，可有了人这个媒介以后，就多多少少有了些必然的联系。也许世上万物都是因为人，才有了必要的或者是不必要的关联。

　　我小时候住在郊区的工厂宿舍里，每天都要穿越大片的田地去市里上学。每到春天，我所喜爱的蚕豆花便开了。紫色的花朵，黑色的花纹，一株一株的在路边排列着。天是洗练过的青白，俯瞰着被春染翠了的大地。近处的人家院里种着歪脖柳。风一吹，柳丝能飘出老远，枝叶缠绕着，又仿佛是被什么牵绊住了,柳锁春魂，花翻蝶梦，蚕豆花在这片广袤无垠的天地间，就越发显出它的轻盈，独特的美来。

　　我每天都与小伙伴在它的迎送下来来去去。那时，我很有些说故事的天分，在周围的同伴间也小有些名气。在这条开满蚕豆花的路上，我不知曾跟他们说过多少故事。当然，也不是白说，常常有些小伙伴从家里带些零食给我，算是酬劳。我便将那些有酬劳送来的人安排在离我最近的地方。至于那些没有酬劳的就只好在身后跟着。

　　可是转眼过了十数年，那些常常给酬劳的小伙伴都已记不起来了。唯独还记得一位从不付酬劳的同伴，她叫红。

　　红若是没有零食那倒也罢了，可是她每每有了零食却当着我的面给了沿路那些农家的小狗。所以，我不爱带她。其它的小伙伴们也不是很爱与她走在一块儿。可她却偏偏上哪儿都要跟着。有时，大老远就能看见她在路口等着我们。

　　一路上最热闹的除了我响亮的说故事声，便是那些小狗远远地闻到她的气息，就像见到亲人似的飞奔过来。接着，便是那些怕狗的小姑娘们一惊一乍尖叫声。红与小狗们搂抱，打滚，混成一片。等我们快步走得很远了，她又气喘嘘嘘的追了上来。

　　不管是什么样的故事总会有男主角、女主角，只要有了男女，就必定有爱情。我很小的时候就懂了。所以那时我不管说什么故事都离不开爱情。而我的故事也鲜有说完整的时候。也许是因为我给男女主人公制造了麻烦，却不知该怎么替他们收场，每次我总觉得到了最后，他们会分开，我就会很快得厌倦了他们。

　　那些故事就像是一个梦，从梦开始就注定了它要破灭。所以，我常常是用另一个更炫丽，更迷人的梦的开始，去避免告诉同伴前一个梦的破灭。每当我的故事说不完的时候，红的小狗们表现的热情却从来是有始有终。

　　后来略大些，看到了罗曼.罗兰的那句：爱我的人，我不爱他；我爱的人，他不爱我；而相爱的人总是要分离的。这句话道尽了天下爱情故事的精髓，至此我便很少再与人说爱情故事。

　　再后来，看到张爱玲的那句：我除了天才梦，一无所有。每每有人说那句话有种破碎的凄凉，我便笑。因为梦开始就是要破碎的，那才是一个完整的梦，被实现了的那部分，永远不会是梦。张爱玲拥有了一个完整的梦，这是别人所不懂的。

　　童年的往事，其实都忘得差不多了。只是偶尔还会想起紫色的蚕豆花，飞奔来的狗和我的爱情故事。
