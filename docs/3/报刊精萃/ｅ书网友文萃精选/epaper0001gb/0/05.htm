不说再见

草本植物

　　我与恽在这个城市属于点头之交。他偶尔向我买点货，我偶尔向他供点货。

　　那是在一家供酒的陶吧内，恽一分分的压着我的价。他是个精明的商人，凑巧我也是。所以我们在喝了点酒，消磨了大半夜之后，还没有谈拢价格。

　　也许是那天外面突然下起了雨。雨丝打在透明的玻璃窗上，凝结成的雨珠在来来往往的车辆照射下，竟有几分似女人的泪珠。也许是夜深了，萨克斯风的音量小了，也少了几分爵士的味道，也许是什么别的原因，恽对我说，想跟我讲个故事，一个爱情故事。我说好啊，早知你要讲爱情故事，我就该叫两杯cappuccino。他说也不知为什么忽然想对我说这个故事，也许是因为我和他的初恋情人一样住在化龙巷。也许是因为我们们今天又刚好在这条路上的小吧内喝酒。我笑着说，也许是喝了酒后，甚至于觉得我和她还有几分相似。他笑着回答，是啊，不过她不像你喜欢把感情放在心里，她的喜、怒、哀、乐都是放在脸上的，像块透明的水晶。所以我们就称她为水晶。

　　恽是在南京的某所大学毕业的，因为在某次学潮中，他的某些政治倾向上的错误，在档案上有了一笔，所以没有什么大单位肯收他。恽就只好来到我们这个小城市的一家二流单位里面混着，也不怎么受重用。

　　在那个酷暑的季节里，恽一趟又一趟得跑着人才交流中心，希望最终可以找到一个合适发展的空间。他说那时最喜欢经过化龙巷，因为街道两旁长着枝繁叶茂的梧桐树。每次经过那里，便会感到一阵清凉，像是快要窒息的人忽然闻到了一口清凉的空气。

　　认识水晶，是因为她推着车子停在他面前，给了他一个微笑，然后问："你能不能帮我装下车链条？"

　　恽说，他喜欢上她，也许就起始于那个微笑。那个微笑在恽看来是混和着真诚和信任的。结果是不言而明的，他们很快做起了朋友，又很快成了恋人。一切都是那么快。

　　水晶来自个有点背景的家庭，父母在市政府做着不大不小的官。她时常发点小脾气，却又马上和恽和好。她任性但又不是很有主张。她喜欢坐在恽的那辆破车后面，一边晃着双脚一边叫道："嘀灵灵，嘀灵灵，大家小心，这辆车子即没刹车又没铃。"然后悄悄地在恽的耳后说："我是你的铃，我是你的铃！"我想她在说这话时的声音一定也像水晶一般剔亮，清脆。所以恽说他在好多年之后，还能听到她在他的耳边说，我是你的铃。

　　当秋天将至，水晶兴冲冲地问他要了厂里的地址、电话，说是她的父母会去找他。恽惶惑不已，在提心掉胆地过了几天后，接到的却是水晶的电话。

　　水晶站在巷子口等，恽看到她的脸色就有了某种预感。水晶对他说，她的父母已经去厂里了解过他的情况，一位老朋友对他们说他有些政治问题。恽马上想起那些档案里的字，那些字他从未见过，不过在那时浮现在脑海的却是红色的。水晶说他们家很在乎这个。水晶还说，她这次高考没有能考上好的大学，所以家里准备送她到新西兰去留学。水晶还说，她会给他写信……

　　可是恽知道，他与水晶的关系在这一刻就已经定格了，不是进行式，没有未来式，有的只是过去式。他慢慢地转回身离开她，走出化龙巷才知道外面已经下起了雨，只不过化龙巷的树叶太浓密而他没有发觉。等他转弯时，恽发现水晶还站在那里。恽知道水晶在等他和她说再见。但他始终没有说。
恽说希望水晶不要误会他是在责怪她。他只是不想说再见，说再见是多么容易，要想再见是多么的不容易，所以他不说。

　　我和恽走出陶吧，他一个人慢慢地向前走去。我不知道恽能不能找回当初的那份感觉，因为现在化龙巷的树早已经都砍了，路也拓宽了，即使走在这条路上也能感觉到外面的斜风细雨。但我还是站在那里，直到他转弯。

　　恽也没有和我说再见，但我知道我们是要再见的。因为我和恽，不是情人。
