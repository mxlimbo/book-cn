等你一百年

草本植物

　　嫣然与展悦相识于南方的校园。嫣然相貌普通，而展悦长得帅气。嫣然第一次见到展悦，他在笑。那是种粗糙的美，带着些许放肆。这让她想起了曾有过的一段爱。那是原，嫣然与他长在苏州的绿水石桥边。他死了。好像春天，还未开始，便已经过去了。他就像嫣然生命中的第一缕阳光。是朝阳。温和，带着属于早晨的清新。现在嫣然又见到了她生命中第二缕阳光。展悦就像午后的阳光，热烈而奔放。

　　展悦见到了嫣然。奇怪的是，他并不是第一次见到她。只不过从那刻起，她才真正走进他的心里。展悦聚集了一群人在他的木屋。严格的说来她并不算他的客人。她是应他的室友相邀而来。茑茑燕燕围绕在他的四周，可他却在她们的夹缝中见到了嫣然。

　　她坐在那儿，好像坐了好多年。很安静。好像还会坐下去。很平静。可当展悦再瞥一眼时，又觉得她随时会走。她就像一缕烟，浓重时，会遮住你的双眼。风一吹，她又会消失的无影无踪。从那刻起，展悦心里有了个强烈的愿望。他希望可以永远牵住烟的手。

　　展悦就这样闯进了嫣然的世界。他幽默，她慧诘。他喜欢苏州的一切，因为嫣然。他会大声说，我知道去天堂的路径。汕头/广州/……上海/苏州，坐335次火车。嫣然笑说，去苏州不用从广州转。但展悦坚持要，因为那是他的故乡。嫣然说，那我们就生活在天堂好了。因为这还包括了她对原的承诺。原对她说，若是他不能带她生活在天堂，就请她带着他生活在天堂。他要亲眼看到她幸福。她把对原的承诺告诉了他。她愿意对他回忆和原的点点滴滴。因为，她觉得她已经爱他够深。

　　每当嫣然约会迟到，展悦便会笑着说，别急，我会等你。但是不会等太久，就等一百年。那是句台词，她笑。但是她真得相信，相信他会等她。在一次，他们俩单独待在展悦的小屋时，展悦忽然说，他希望能与嫣然生活在广州，那是他们的天堂。嫣然很吃惊，她以为他会与自己生活在苏州。结婚，生孩子，再一起老去。展悦突然粗暴的说："你想在那里与之一起生活的不是我！是原，是他！"他们开始了相恋以来第一次也是最后一次争吵。嫣然责怪他不该如此狭隘。他死了，死了。说到这个字，她终于流泪了。见到她的眼泪，展悦沉默了。最后他说，正因他死了，所以才成了永远的胜利者。

　　毕业前，他们又重新聚到了展悦的小屋。嫣然陪他喝着酒，争论着爱与被爱哪个幸福。嫣然选择了苏州，展悦没有同往。嫣然记得自己喝醉了，隐隐间听到弹词乐在唱着玉蜻蜓。嫣然走了，她走时，展悦说留在汕头。我等你。他又说。嫣然真的相信，相信他会等她。

　　回到了苏州，嫣然实现了她对原的承诺。她离得他如此之近，却不觉得生活在天堂。他不能，再也不能给她任何慰藉。一年后，嫣然终于明白，原是要她生活在天堂，苏州已不再是她的天堂。她觉得解脱了。她要回到展悦的身边。那么，无论在哪里，都是天堂。原希望她幸福，只要她幸福便是实现了对他的诺言。

　　再回到展悦的小屋，已是一片萧瑟。他正在收拾行礼，他要走了。嫣然站在那儿与他四目相对，感觉像是有一个世纪那么久。他是那么狼狈，完全没有准备。她就这样无期遇的闯进了他已认定了的生活。他要回广州了，也要结婚了。他满目后悔，他摇着头："我不知道，我不知道……"嫣然却明白了，命运对人，如手对陶泥，塑成什么，就是什么，不容抗拒。他在哭，她却听到了外面细细碎碎的脚步声。嫣然转身往外走，一去不回头，她不想再让他看见自己泪水弥漫的双眼。一生一世尽系一念之中，此一念，彼一念，全盘皆落索。

　　他结婚了，听人说他在婚宴上喝得仃伶大醉，嘴里唱着不知名的曲子。人们爱的是一些人，与之结婚的又是另外一些人。错过是一种美，它让人懂得珍惜。

　　嫣然留在了她的母校，那是他们开始的地方。每当夕阳红，她就会想起曾经有过的阳光。再一年，她出差去广州，在熙熙攘攘的人群里竟然又见到了他。他光洁的额头有了岁月的痕迹，她才发现自己从未曾真正地忘却过他。抬起头，她在心里暗暗念着，看着我，让我们做个最后的了结。在那一刻，她真得好像已过了百年。等你一百年，等你一百年。就在这一秒间，他们俩无声无息得擦肩而过。

　　人们希望之事往往不会发生，命运另有安排。 他们都说，嫣然如愿以偿去了那么美丽的地方做她喜欢的事，真好。" 是吗? 是为了这海、这城市、这草长莺飞的校园吗? 也不是，只有这儿才有幸福啊，可是我来了，义无反顾。 然而，我千山万水的来了， 才发现初衷早已在千里万里之外。" －－嫣然
