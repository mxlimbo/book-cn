坐时间的河岸看风景

希子

　　我说，再见。
　　于是，你转身离去。
　　你走了，每个清晨我睁着眼醒来，才发现早己发不清是睡还是醒。
　　今天我穿着那条深紫的长裙，赤着脚却象午夜时分仓皇逃奔的公主。

　　那天阳光的味道很淡，我们站在街边的站牌下等公车。我用双手从身后环着你，笑。你把我拉到面前，那时我的模样应该是有些苍白的，于是你怜爱的把我轻轻揽到怀里。这一幕重复了很多次，乐此不疲，后来终于想起我们要等的车根本不经过这个站。忘了那条街道，记忆里是二月的阳光。

　　你把我的手放在你的手心，我的手指是冰凉的，你让我感觉你的体温一丝丝渗过来，多久以后我常想这种感觉的重叠好象电影里恋人重逢相向而奔的慢镜头。

　　电话里的你有些遥远，我想见到你，我知道那隔着山，还隔着海，然而我还是孩子气的要你答应在我明天早上醒来时就童话般的出现，我用娇纵的泪水灼痛了你的天空后无梦的睡去。究竟睡了多久？醒时，敲门声，开门，是你，和南国的细雨。

　　月照纱窗，萤火在游，我想这是六月的夜空，寂寞中我拥着窗棂死去。

　　我在等你，而你失约了。

　　是我私定了这个约会，还是我又站错了站台？

　　没有人哭,只有时间的河泪水长流.
