网络骂辞之概要

草本植物

　　我混迹网络这些日子以来，常能见到聊天室里的网友们开战之时乱骂一气，毫无章法可言。要知网管虽多是两眼一抹黑，可总也有偶尔睁着一只眼的时候，冒被三振出局的危险，讲些不入流的话，实在是得不偿失。我往往见到网友辞不达意，入旁门而不走正道，又尤如大禹治水三过其门而不入，每每我便恨不得能越俎代疱，代而为之。所以我在此略略总结一下前人的经验，希望大家能有所领悟，提高骂人的境界。

　　说起骂人也与上网一样分为三六九等。上网之人大家都知道可分为网虫、网皮、网仙等。这骂人也有，大致可分为次品与正品，往细里说，这正品又可分为下品，精品与极品。

　　次品大多是一些骂中新手，却急于在网络上小试牛刀，开口不是骂人sb就是阳萎，言词乏善可陈。而且速度奇慢，偶尔他们想打一句自认高明或者比较长的句子，你甚至于可以在其中打个盹。可看性极差，所以说他们是次品。

　　正品中的下品开始略领悟到网络骂人的技巧，学会使用板砖，大大提高了骂人的速度。可是他们又往往表现得像个唐门的低级背暗器的弟子。虽然满身武器却不懂如何使用。例如有人骂他阳萎，他抛出：sb。等别人骂人他sb时，他倒又抛出：不要相信女人尤其是裸体女人。牛头不对马嘴，自己打自己的嘴巴。这种人也多混迹于聊天室，是骂人中的下品。

　　骂人中的精品，开始自重身份，轻易不说脏话，往往将之代以性器官相称，并且开始登堂入室进入bbs骂人。他们多是有些文学功底，也已略略懂得含沙射影，指桑骂槐等伎俩，但致命伤便是常常以洋文来骂，动不动就是shit、fuck。要知我国骂辞之洋洋洒洒，更有着五千年的文化历史为背景，岂是简陋的外文可以相提比拟的。例如：fuck便涵盖了性交、 欺骗、利用, 诅咒、 杂种, 性交对象、他妈的如此之多的中文骂辞，我见了每每有暴殄天物之感。中国人用洋文骂人，套句钱仲书的话，尤如政治家躲进了租界活动。

　　最后我要说的是网络中的骂人极品，此中人不但文武皆备，根骨其佳而且极居穿透力，敢与专业对掐，敢与名人叫板。骂人已可以达到拈花伤人，杀人于无形的骂界至高境界。这种人不但在网络上是可遇不可求，即便放眼中国也是屈指可数的。他们当中不乏国骂，可以骂一代人，甚至于可以骂整个民族，骂得隔了几代后别人都不能说他骂得不对。但这些人大都已驾鹤西去，我们就不在此讨论了。他们当中就算略次点得也大有一骂成名，或者咸鱼翻身的骂界高手。这些人不但懂得如何去骂，去骂谁，而且深知进退之道。每每有人骂他们无知，往往就落入了夸奖他们无畏的圈套之中。真是让植物钦慕不已，视为毕生追求的至高目标。可惜修练起来，常因打不通任、督二脉而功亏一篑，至今只能在门外徘徊。至此不免又想起钱仲书的一个比喻：由婢做妾易，要由妾扶正做太太那就是难上加难了。所以由骂人中的下品修炼成精品易，再由精品修炼成极品，那就是难上加难了。

　　植物仅以此献给全网络骂界的朋友，希各位能在骂人上更上一层楼。
