从“第一次亲密接触”看网络文学

第三空间

　　一句话，网络文学唯美、颓废一边倒。

　　其实文学就是文学，之所以称之为网络文学是目前流行的IT行业的价值观了。新世纪新人类棗网络棗文学的组合，当然特有的商业价值也就不容置疑了。那么文学套上网络的光环是好还是坏。我们来做一个比较：

　　鲁迅用白话文体尝试写的第一部小说《小草》在当时影响很大，文坛开始流行白话文。《小草》不是鲁迅最好的小说，但确实是他成为一个文豪的过渡。

　　八十年代的王朔现象应该引起我们思考，王朔的创作特点已经不用多说。可以用一句剑走偏锋来概括，他现在成了家喻户晓的名人。

　　从“第一次亲密接触”看文学，也许可以说是文学套上网络的光环。但从它的内涵来说文学还是文学，只是创作题材更广泛了，有了一个新的空间和一种新的语体。这是好的一面。不过也有不尽人意的另一面，那就是网络文学唯美、颓废一边倒。

　　“第一次亲密接触”引发的不同凡想的商业价值，网络文学就把自己定位在边缘文学和非主流文学中。绝大部份作品是一个模式唯美、颓废。真正反映现实的几乎没有。这也许又是网络的光环，偏向另类。而这不得不成为一个误区，是制约网络文学发展的一条界限。同样有很多开始读者群最终成为网络文学或大或小的写手，他们对自己的出路和多年的创作同样会产生迷惑。即使有一天他们的作品在创作上超过了“第一次亲密接触”因为相同的风格和定位。他们还是成不了蔡子痞。

　　所以还是相信文学就是文学吧，网络只是一种借口。你可以心领神会，但千万不要被它的光环掩没了你的创作动机。
