今天，你WAP了吗？

张诚 （zhangcheng@xinhuanet.com）

　　20世纪末，互联网经济席卷整个中国，在这场被称作“第三次革命”的风暴中，众多网络公司诸如新浪、网易、首都在线、8848等脱颖而出，一时间，网络成了最耀眼的名词，似乎是谁和网络沾边谁就会一夜爆富，于是众家公司不管三七二十一纷纷上网，甚至连“WC”后面也恨不得加上“.com”以炫耀身份。然而不论你相不相信，如今，二十一世纪的第一个年头，“.com”公司已不再被人看好，WAP（无线应用协议即Wireless Application Protocol）正“光芒万丈”的走来，在这片充满神奇的世界里大声宣布：无线互联的时代已经到来。让我们拿出当年欢呼互联网的热情来庆祝这个伟大时刻的到来吧！

　　一场“瞄准指尖”的新圈地运动正在起步棗无论是那一块地，圈下来就占据了先机，至于如何变现则是后话。于是江湖]各路豪杰，无论辈分高低，竞相亮出招事，以新浪（wap.sina.com.cn）、网易（wap.163.com）、搜狐（wap.sohu.com.cn）为代表的传统ICP均开通WAP服务，以掌上网（wap.byair.com）、365do(www.365.com)、灵通网为代表的一批新玩家则看准时机，纷纷杀入重围。而实力派的移动运营商的动作更大，拥有4600玩手机用户的中国移动，5月17日宣布其特服号为172的WAP业务将覆盖全国。相比之下，拥有1000万手机用户的中国联通则略显逊色，饶是如此，联通也宣布将在全国20几个城市开通WAP服务。他们完全有理由相信，把代表通信领域发展趋势的两大产业移动通讯和互联网结合起来的无线互联也将和互联网一样，成为网络时代的一种平衡力量棗近两年来人们对旧经济里的企业乃至传统的IT企业抱怨的最多的一句话就是：面对互联网反应迟钝。无线互联时代的到来，同样使得这些站在浪尖上的“.com”公司感到类似的威胁与恐惧：“你忽视了无限互联，忽视了移动电子商务，那麽你就输掉了未来。”

　　但是无线互联会带给我们什麽，发展前景如何，到目前为止，仍然是个未知数，一般来讲，公共政策的制定会滞后于技术革新。去年9月，主管部门关于“外资不能投资电信增值服务”的政策阻碍了产业的发展速度，使得有关部门为互联网的政策制定大费脑筋。现在同样陌生新兴的无线互联网来了，市场虽远未成熟，但我们也得吸取互联网政策方面的教训。一个简单的例子，大家都说可以用手机炒股，手机信号不好是很常见的，如果正在交易的过程中手机失去信号，给用户造成损失，责任谁来承担？现行的条款里并没有一个清晰的界定。

　　政策的内容会涉及到方方面面，哪一方面最为重要？回答当然是隐私。就WAP而言，它意味着人们开机后便差不多一直处于联网的状态，由于手机是“个人化”的物品，它的身份是唯一确定的（写在SIM卡里），这样，运营商、企业、政府包括个人都可以定位指定用户，并跟踪了解他们的行动。一两年内，带小的摄像头和麦克风、有实时广播功能的移动终端将面世，届时跟踪每一场谈话，每一个动作就会变得更加轻松。

　　毫无疑问，有关部门得为无线互联时代的到来做好政策上的准备。欧洲在发展互联网上以采用了这样一个政策：个人在选择链接、提供个人信息方面必须是自愿的，获得个人信息的互联网企业如果想向第三方出售有关个人的信息，必须经本人同意。这一原则，同样应该运用在无线互联网上。

　　企业也得调整自己的策略。无线互联网络给了员工随时随地工作的能力棗无论是在办公室在家，还是在移动中得汽车、列车、飞机里。老板们完全可以利用这一点，在工作时间之外增加员工的工作量，用马克思主义政治经济学的理论来说，就是最大化“剩余价值”：在下属旅游休假的时候把一些文件资料传给他，让项目经理在周末的时候也作项目计划……直到把他们弄得筋疲力尽。

　　技术从来都是一把双刃剑。对于那些今天还在迷恋无线互联网的人们来说，也许用不了多久，他们就得学会如何才能摆脱它的纠缠，寻求一些真正属于自己的时间。

　　预言家们说，未来几年内，移动通讯和电子商务都会渐入佳境。可要真正做到善始善终，需要政府和企业做出合情合理的决策。惟有此，无线互联才不会成为人们的一场恶梦。
