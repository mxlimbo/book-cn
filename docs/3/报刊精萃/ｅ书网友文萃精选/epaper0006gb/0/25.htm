单 身 日 记

碎梦一刀

　　6月11日凌晨晴

　　端坐在电脑面前，脑海中一片空白。思绪突然间的休克，并不使我惊讶，这已是很平常的事了，至少对于我来说是这样。读着一篇不知名的写手写出的文章，觉得很可笑，为什么人们一天到晚总是在编织着一个梦，一个永不可能实现的不真实的梦。更可笑的是人们往往是在知道其不可能的真实之后，还是愿意去相信它。或许童话人们读的太多了吧。很多时候是对许多文章或事物的看不起。或许有人要说，你丫有什么了不起，凭什么看不起其它。是的，我无权责怪任何的人，因为我并不比他们其中的一个聪明，或是看的更广。我只是想把自己的看法说出来，否则我会发疯的。看不下去的可以走开了，当你看到一个侏儒还要赞美他的高大魁梧这是很恶心的事，我至少还没学会。当然这并不能责怪任何人，无论写手还是读者，当你挚着的相信了一段故事的真实之后，便再没有什么能够改变它了。人们本就是如此，更多的相信自己而不愿意相信其他，哪怕它是虚幻的。其实说到底，很多时候自己一边为着无法改变的现实苦恼，一边却又在不停的为着那些丑恶而呐喊助威，这或许就是现实吧。真的很可笑，我正在为着无法摆脱的媚俗而申斥，却又在写着附庸它的文章，是无奈还是自甘堕落的沉沦，我不知道，真的不知道。或许只为了摆脱那种濒临疯狂的思想崩溃，我在这里大放厥词。很想找个地方呐喊

　　狂吼，却还是选择了用笔。这大概是我没有完全消灭的思想在发生着作用吧。

　　前几天和一个朋友谈到对许多事物的看法，朋友问我是否对不满的事物表示批评或是包容，我告诉她我选择独行，虽然可能比较难走，可是既然我无法改变一些现状，只有让自己不被它改变。希望能保持自己人格上的独立。其实今天晚上，在瑟瑟的雨声中对镜自观，忽然发觉自己的轮廓已经不再鲜明。才感悟到在年轮中消逝的不只是青春更多的是思想的棱角。再几年后或许照镜子的时候已认不出了自己。现在把自己的想法写下来，至少以后还有了一些可以的回味。

　　6月28日晴

　　鱼缸中的那条金鱼悠然自得的在我面前游来游去，炫耀着它的闲适和安详。看着它轻轻的吐出一连串的气泡，我忽然产生了一种不真实的感觉。衣食无缺，又不用担心外界环境变化。这样的生活是否真的令人羡慕。

　　呆呆的看着这条红色的金鱼，我不禁想起曾经认识的一个女孩。那是一个喜欢鱼甚至把自己也当作是鱼的女子。灵性而睿智的她，聪明得可以看透每一个人的心语，却一直看不懂自己。她总是不愿让人接近，因为害怕别人会象她梦中的水草，在纠缠住她的同时，伤害了她美丽的鳞。

　　女人是聪明还是愚蠢？我无从解答。

　　人们总是害怕别人的伤害而忘了能够伤害自己的其实是他本身。

　　朋友阿江终于要结婚了，当他告诉我这个消息的时候，我的感觉竟然不是高兴。我甚至为他的处境而担忧。因为我认为阿江和他的未婚妻小玲可以说是一对典型的疯子型结合。很不幸我和这对活宝从小就认识。目睹他们八年抗战的恋爱惨状，亲眼看着他俩一次次分分合合，聚聚散散。如同上演一出经久不衰的多幕闹剧。我不知道他们是否相配，但如果要征求我的意见，那么我认为他俩不合适。就象把香蕉和洋葱放在一起那样的不协调。阿江是香蕉，样子可爱但是没有水份，女孩和他在一起，很快便会感到枯糙。小玲则是洋葱，长的不怎么样，却很有一些思想。属于性格泼辣的一类，相处越久，越能发现她更多的吸引力。

　　我不知道他们是如何走在一起的，反正我是看不出他们的共同之处。我始终倔强的认为，婚姻是一场爱情游戏的终结，婚后的生活将是夫妻重新认识对方的开始。婚姻得以维系的基础存在于夫妻双方对某一事物的共识，没有共通就谈不上交流，这样的婚姻终将失败。

　　但是我还是祝福他们，恋爱中的男女除了祝福还能听得进什么呢？

　　阿江和小玲正在紧锣密鼓的准备结婚，我觉得自己忽然变得孤独了起来。

　　‘男大当婚，女大当嫁’是老人们最爱挂在口头的话，或许是他们尝试了太多的孤独，可很不理解为什么人一定要婚嫁呢？父母又在旁敲侧击对我进行提醒，到了该找女朋友的时候了，可是曾经沧海难为水的那种感觉，使我真的很难付出全部的感情去面对别的女孩。我猜想自己可能是害怕付出后得不到结果的伤心，所以不敢全身心的投入，这样的结果其实是可能一辈子得不到真爱。我无可奈何，但我无计可施。感情总是这样那样的令人捉摸不透。

　　朋友曾经劝我：“婚姻就象是在超市中买商品，太高档的买不起，太廉价的不想要。最后凑合个价格适中又不算差的中档商品。所以不要太挑拣，凑合就行了。”

　　朋友也是一番好意，有时孤独的感觉确实难耐，可是却无法苟同朋友的看法。因为我是理想主义者，所以一直执着的认为婚姻不是商品，并不能够随便的凑合。如果硬要把其当作是商品也应该是一生只一次的大件，需要的是彼此的配合和相互的吸引。

　　我知道我的想法太过天真，也不够现实，可是我就是这样一个人，已经无法改变了，所以对于阿江和小玲的婚姻才有那样的不理解。阿江和小玲无疑都是我不错的朋友，可我担心他们并不了解婚姻的意义。说到底，我并不希望他们的婚姻是一个为了摆脱寂寞感觉而参加的游戏。因为如果是我在那样的情况下一定不会快乐。我宁可把孤独当作是一种美丽，端一杯清茶，静坐一旁，用一种超然的状态去欣赏其中的宁静和祥和。

　　8月2日晴

　　最近工作比较多，忙了一整天，回到宿舍里只是想睡觉，其他的事什么都不想做。床下面搁着满满的两盆脏衣服，其中的一些已经保存了有两个星期。很想从床上爬起来做一次大的清洗工作，可是有些说不出的慵懒情绪在缠绕着我，使我只想躺着不动。闭上眼，看到的是无尽的深远，令人疲惫。睁开眼，看到的又只是白色天花板的无聊。

　　忽然间没有了睡的意识，只是想要平静一下思绪，体味什么都不想的短暂宁静。可是莫名的，在无所想的过程中竟渐渐变得心绪不宁了起来。

　　那是一种无法形容的感觉，好象是宿命的召唤，又象是邪恶之灵的引诱。灵魂的脱壳在有意识的情况下发生了。我觉得自己好象面临着一种哈姆雷特似的抉择。生活的目标何在？活着真的很累，我不知道人在没有了目标之后的浑浑噩噩是否可取？很容易的问题？^-^ 是否有人准备对我进行开导了。那我先告诉你吧，我讨厌有人对我说教。美丽的世界！呵，至少我没有发现。看姜武在城市中傻傻的等着中奖后的奖品，我只是笑。说到底，我还是更多一些喜欢伍佰，无尽的落寞和孤独。

　　是否人们不喜欢的事物就可以归结为不好？那么从这个意义上来说，白痴应该是世界上最幸福的人，因为在他们的心中根本不存在喜欢或不喜欢。或许有人要说，遵从大多数人的意见确定好或坏的标准。那么是否可以说搞笑的商业电影是有史以来人类最伟大的发明，因为它受人们的欢迎程度绝对名列前茅。

　　胡说八道了一通，其实只是想发发牢骚，现在好象舒服多了。明天又可以坚持下去了。

　　8月8日晴

　　不知从什么时候开始，人们对8字开始青睐起来。车子落户要带8，买手机要买带8的吉祥号码，甚至连结婚也都要选带8的日子。

　　阿江和小玲选择了今天结婚是否会在以后的日子中开始连‘发’我不得而知，但我却真的要发了。天恒酒店的大厅里挤满了不知从哪冒出来的许多人，我觉得自己在那样的人潮中挤进挤出，充当接引使者，身受热浪的侵蚀，吸纳从四方挤压进我体内的热能，几乎要成为了一个蒸锅里的发面馒头。

　　晚上的节目是很普通的鬼子进村，新郎和新娘反串花姑娘和小鬼子一点新意也没有。大伙都有些兴味涣然。这时在茫茫人海中突然跳出了一位高人，提出了让新郎新娘配演大话西游的高招。我当时只是忙着说话没太在意，以至后来不知如何寻找那位无名的高手，丧失了向他请教的机会。这确实是个令人高兴的提议，让很多已经奄奄一息的朋友精神为之一振。大家动手立刻找了不少的道具。当阿江头戴鞋盒身披被单站在小玲面前的时候，我们都为之倾倒。面对着架在脖子上的苍蝇拍，阿江信誓旦旦。

　　“曾经有一份真挚的爱情，放在我面前，而我没有珍惜……如果上天给我一次重来的机会，我会对这个女孩说：‘我爱你！’如果一定要给这份爱加一个期限，我希望是：一万年。”小玲的眼中含着泪，

　　这一刻我被深深的感动了。

　　我相信阿江是用自己的心在说这段话，“希望你们白头到老，我的朋友”我在心里默默的为他们祝福。

　　一直不太喜欢周星驰的电影，可是对于这部大话西游却情有独钟。其中周星驰扮演的悟空是我见过的最为人性化的英雄。在悲凉里隐含着激情，在诙谐中透射出豪迈，这才是我所能接受的英雄。英雄首先要是一个人，要有人的七情六欲，有人的爱恨别离。因为只有成为了人，有了人的特点，才可能成为英雄。就好象你要装水首先要有装水的容器。要装载荣誉和名声同样需要载体，这个载体便是人。

　　8月15日阴有雨

　　阿明今天要搬走了，心中莫名的有了些不舍。阿明是我的舍友兼同事也算是朋友，近两年来一直和我同居，算得上是患难之交。因为他的存在避免了我被母蚊子攻击的危险。我因此对他感激莫名，所以对他的去向表示了自己的最大的关注。蚊子为何如此的喜欢和他接近，我并不知道原因，当然也从来没有试图得到答案，因为我并不喜欢蚊子，更不喜欢被母蚊子亲吻的感

　　觉，所以对于他的好运从来没有嫉妒过。但是女孩喜欢和他接近的原因我却是知道的，因为阿明无疑是一个很不错的家伙，高大帅气并且幽默搞笑。我想女孩如果不愿和他这样的人接近那就是有病了。

　　阿明从来不缺女朋友，但是始终没有想过要离开我到外面租房。

　　难道他也会堕入情网？我问他搬走的原因。他告诉我，准备暂时和女友同居一段时间，看看彼此是否合适，然后可能会结婚。

　　看到阿明要走，不禁加倍的怀念起一起生活的那些日子。每个难以成眠的深夜，当我被窗外楼下的烧烤人群搅得心烦意乱的时候，总可以发现阿明也因为蚊子的骚扰而辗转反侧。那时便莫名的高兴了起来。呵，终于有人陪着我在孤寂中无眠了，我可以在同伴的陪同下静静期待黎明前的那第一缕光线……“我得意的笑，我得意的笑”每每这个时候，我便忍不住想要笑出声来。可是听到阿明噼噼啪啪的举掌自责，面对母蚊子作忏悔状，又殊为不忍。只有躲在被子里面拼命的咬被子角。

　　为此，阿明专门还买了两包耗子药，据他说时常在半夜中听到老鼠咀嚼衣物的声音。虽然我们搜索了整个的宿舍一直没有找到老鼠存在过的痕迹，并且我向他保证在七楼的高度上基本不会存在老鼠，可他发誓确切的听到了老鼠磨牙的声音。我只有在他没有想到检查我的被子之前，尽可能快的将被褥送到了二十里以外我的父母家中，以免被他发现被角上面那两个深深的洞。

　　阿明和我如此的亲近，以至于对他的离开总觉得有些怅然。想到从此后，我将独守空房，独自面对群蚊的骚扰，真个是十分的恐怖。

　　而更令人伤心的是，从此后我将不得不亲自打扫宿舍，直面积尘扑面的难堪。我的MONEY并不能够在维持我的生计之外再承担一个保姆的费用，所以看来以后的不少事情得我自己亲自动手了。我为之心碎，几乎悲痛欲绝。

　　“ONLY YOU ，ONLY YOU CAN HELP ME。

　　ONLY YOU ，能够替我挡住蚊虫骚扰。

　　ONLY YOU 能够充当菲佣不需费用。

　　ONLY YOU ……”

　　8月的昆明窗外飘着雨。
