青 春（二）

Terry

　　那天我很高兴,因为对我来言在网上找一个知心的红颜知己并不容易,那时我以为我已经找到了。不过回头又想其实就是玩的个景嘛。在网上能有什么真实的爱情啊?况且两人相隔千里,见都见不着面。老婆,老公这些称谓也只是在网上喊喊罢了,说不定人家身边已有男朋友了,在网上也说不定已经有n个老公了。我心想:她不会是"妖精"吧。（注："妖精"是指搞男人没商量的女人，她们打字速度奇快，与四五个男人周旋不成问题，并以戏弄男人为乐，到底是不是MM就不一定了。总之，这些女人没安好心，是我们这些"痴情派"最大的杀手。）她才18岁，不太可能这么年轻就"成精"吧。我猜她应该是和我一样的真诚，不会那么随随便便的处理感情问题的。

　　我的心早已飞到星期六的下午了。我期望着这一时刻赶快到来，因为我好想再见到她。与她聊天我真的感到无比快乐，有种轻飘飘的感觉好像梦游天界一般。盼呀盼，终于盼到星期六了，我如时地见到了她。这一次我们依旧像上一次一样投机，不过更加浪漫了。我都没想到我的嘴竟然会这么甜，这不像我，也许这就是爱情的魔力吧，它让我变的更讨人喜欢了。但只有在我跟[相思甜心]聊天时才会说出那种让我过后不敢再想的情话。时间很快地在充满玫瑰芳香的气氛中流逝，虽然不想但无力流住。这一次，我们互相通告了地址，并许诺一回家就给对方写信。关于那首《今生共相伴》由于我忘带歌词了，所以只好以羽泉的《最美》代替。不过我答应我将会给她一个惊喜。是什么样的惊喜呢？其实是我突发奇想：何不用电脑打完歌词，再用打印机把它打印出来呢？这样她可以看到它就想到我。于是我马上照办，回家后没几分钟就打完歌词了，现在只剩下把它打印出来了。可上哪儿打印啊？对了！吴寿家有打印机，只可惜是黑白而不是彩喷。不过总比没有好啊！我想。于是第二天便把装有歌词的软盘给了吴寿，跟他讲明原由，让他助小弟一臂之力。吴寿说："没问题！"第二天便把打印好的《今生共相伴》拿来了，我一看效果虽然没像我想象的那么出色，不过还说的过去。但它看上去不像歌词，倒像一首小诗。我迫不及待地将它和早已写好的信件一同装进信封里，跑到邮局，贴上80分的邮票，小心翼翼的把它滑进了信筒里。回来的路上，我的心里没滋滋的……

　　没想到第二天就收到了她的信，信中的开头和结尾让我脸红耳赤，竟然是"帅老公"与"靓老婆"。我是又高兴又无奈，心想：原来她比我还肉麻啊！看完她的信后，更坚定了我对她的信心。起码我们又进了一步，从网络发展到"鸿雁传书"了。没过几天，我又收到了她的信，上面说她已收到了我的信，并把《今生共相伴》放在包中随身携带着。看到这儿，我非常感动，如果她说的都是真的，那我真是没话说了。我想我已经找到我的另一半了。我感觉她是一个聪明可爱，温柔体贴的女孩子。她在我心中是完美无暇的，我把她想象为一位容貌秀丽，仪态大方的公主。我敢说每一个上网的男人都渴望那个远方的她长的美丽动人，但现实中的女人不可能都是MM。当然，男人们也不可能都像谢霆锋那么帅。

　　其实人的容貌漂亮与否并不重要。刘超说的好："80岁不都一样？"对于一对真爱的情侣来说。最重要的是能在成功时互相鼓励，在受挫折时能互相安慰，风雨无阻地共同走完人生的道路。倘若长相可人，那可称之为"郎才女貌"，但如果不是建立在真爱的基础上，两个花瓶在一块儿总是很危险的，就怕哪天碰到一块，那后果将不堪设想。

　　我有着一副普通人的长相，就像张朝阳，王志东他们一样。但人不可貌相，海水不可斗量。说道这儿，想起COOL这个词来了。有些人觉得穿靓装奇服叫COOL，有人板着个老K脸也叫COOL，那样的话你在大街上可以随意找到COOL哥，COOL姐，只不过COOL的让你发冷。其实我为什么自己跟自己过不去，我也属于冷COOL型的啊！只不过是外COOL内HOT罢了，你别说你自己是内COOL（内裤）啊！ ^_^

　　我觉得张朝阳，王志东他们才叫COOL，不单因为他们是我的偶像，更重要的原因是他们的创业史感染了我。他们几乎都是白手起家，凭借自己的才智与勇气和不懈的努力才有今天的地位的。SOHU造就了中国的第一个中文搜索引擎，也造就了至今也广为人知的"朝阳神话"。王志东则是从一位编程员一步步走向成功，最终成为新浪网的总裁。据说他是国内第一个写出能在WINDOWS平台下运行的程序的人，现在大名鼎鼎的RICHWIN97，2000系列都是出自他之手。他现在不但是新浪网的总裁，也是RICHWIN公司的总经理。张和王的成就不是天生注定的，也不是家人所给予的，而是自己闯出来的。我钦佩他们的勇气与智慧，再加上一股不怕失败，积极进取的精神就是我毕生所信赖的成功的信条。而我缺少的他们天才的非同于普通人的大脑以及上天能否赐予给我的机遇。如果二者缺一，恐怕我只能做一个无名的SOHO了，但我坚信我会在IT这条路上走出自己的足迹的。记得高中时的前任班主任曾在我的高二学年评语表中这样写到：你对生活付出多少，生活就回报你多少。我觉得这话很有哲理，让我察觉到原来生活对每个人都是公平的，关键在于你的付出，你的态度，你的为人，你的处世。我现在是一只小小鸟，不过将来会飞高的！

　　上面讲了一大堆我的人生观，与主题无关，只是想发发牢骚。请大家函量。

　　其实我的观点无非就是：如果你真的喜欢上一个人，你是不会计较他（她）的一切的。也许我现在是头脑发胀，语无伦次，夸夸其谈，不重现实，但那是别人（我说的别人是那种我不能和他相提并论的人，那种集合了这"四大优点"的人）的看法。我的观点则是：实践是检验真理的唯一标准。你不尝，怎么能知道它到底是酸还是甜呢？我已经做好了一切准备品尝这个虽然已经咬了一口但仍酸甜不知的圣果了。不过现实总叫人迷惘，我唯一担心的就是网上的她是不是现实中的她，我怕我现在把她想象的太过于完美，而真正面对对方后，却不以为然了。她也一样，说不定一见到我就开始打退堂鼓了。我是面对现实才说这番话的。因为未来的事你我都不直，而它并不是一帆风顺，甜美宜人的，肯定会经历大风大浪，狂风暴雨。丛香，这些波折就是考验我们感情的砝码，我相信它越重，说明我们的感情越真越深，而仅仅喝了一口水就弃对方而去的话，那说明我们之间从来就没有过感情，只是那个先走一步的人一直都在找寻一个既漂亮又大方的浮板罢了。在陆地上这种对象就成了白马王子。丛香，我一直都把你当成白雪公主，而那位吻你的黑马王子到底是谁呢？

　　继那封信后，我们之间的沟通差不多全靠邮差帮忙了。开始我平均能一周两封收到她的来信，而自从我跟她通过电话后，信就与日增多了。那次电话是我们感情的一吨催化剂，是我们感情的一次飞跃。

　　那是在我们相互通了大约有两周信后，我觉得我们俩都是认真的，至少我是已经真的彻底的爱上她了。我觉得是有必要听听她的声音的时候了。于是一天中午，最讨厌打电话的我拿着借同学的IC卡，跑到邮局并拨通了她们学校的电话。

　　嘟…嘟…

　　"喂？请问丛香在吗？"

　　"她不在。吃饭去了。"

　　"那她一般什么时候回来？"

　　"不知道，她一般中午不回宿舍。"

　　"她下午有空吗？"

　　"4：30以后。"

　　"好吧。你告诉她有个叫TERRY（泰瑞）的人找她。还有，我会在6：00给她打电话，让她等着我。谢谢你！"

　　"好，不客气。"

　　"再见"

　　"再见"

　　咣~的一声轻轻挂上电话，满脸遗憾地走出了邮局。

　　下午的时间一秒秒地过去……好不容易打了晚自习的下课铃。我不再像往常那样悠闲自得地抡起书包，双手插进裤兜，哼起小曲回家，我冲向邮局，看了看已经6：05分的表，拿起话筒，怀着莫明的紧张与激动勇敢地拨下了那长长的11位号码。等待我的是令我呼吸越来越困难的拨号声。

　　嘟…嘟…（通了）

　　"请问丛芳在吗？"

　　"我就是。"（别提我当时有多激动了，那一瞬间我脑子里一片空白。）

　　"你能猜出我是谁吗？"

　　"……不知道。"

　　"你收到我几封信了？"

　　"喔，我知道你是谁了――TERRY。"

　　"是啊！知道吗？我上午已经给你打过电话了，不过是你同学接的。"

　　"我知道，她都告诉我了。"

　　"就你一个人在宿舍吗？"

　　"是的。" "你同学呢？"

　　"她们都上课去了。"

　　"你为什么不去上课？"

　　"嗓子疼。有点感冒。"

　　"吃药了吗？"

　　"吃了。"

　　"现在怎么样了？"

　　"好点儿了。"

　　"上过卫生室吗？"

　　"去了，药就是从那里拿的。"

　　"再不行就上医院打个吊瓶得了，接着就好了。"

　　"没事，你不用管我了。"

　　"身体要紧嘛。"（其实我好想说：我怎么可能不关心你？你痛苦我心里更难受。）

　　 "行了，别谈这个啦。你学习忙吗？"

　　"还行吧。反正有做不完的卷子和题。你呢？"

　　"我们还算轻松吧。比你们要好的多。"

　　"那肯定了，高三是最黑暗的一年。"

　　"你一定要好好学习，考上一个好的大学。"

　　 "我会努力的。"

　　…………

　　…………

　　由于是第一次通话，或者现实中的我们原本就很含蓄，我们没有像书信和聊天室中说的那样肉麻。我们宛如两个好朋友一样的交心。她的嗓音很甜美，充满磁性，我不敢相信她的嗓子当时疼的厉害。一口流利的普通话出乎我的意料。此时，一幅《美女接线图》映在我的脑海里。我兴奋不已，没想到她说话如此迷人。感觉对初恋中的情人非常重要，而这次通话是我对她的感觉更加强烈，我打算"非她不娶"了。因为我觉得她真是一个百里挑一的女孩儿，我不会轻易错过的。人家都用"百里挑一"用来形容女孩子的漂亮，而我用其来形容她的温柔体贴，她那与众不同的气质，她那透露出的十足的女性魅力。我知道爱一个人就要爱她的全部，我已经做好所有准备了。

　　"明天6点等我电话，好吗？"

　　"好，不见不散，我等你。"

　　"BYE BYE。"

　　"BYE BYE。"

　　但我们都没有挂电话，而是等着对方挂。"啪"，我先挂了。其实如果不是同学的IC卡，我多么想与她多聊一会儿啊！（就一会儿，不多，2个小时罢了。）

　　于是我第二天就向同位李鹏飞借了30元大钞买了张IC卡。晚上如约而至，拨通电话。"是香香吗？"这次我大胆的像信上一样叫了他的小名。"老公！"她的回答比我还要命，我真是哭笑不得。丛香，为什么你老是带给我惊喜？你可真是现代女性啊！佩服，佩服。我们接着昨天的话题聊了起来，从我们第一次在网上相识到相知相爱全都在电话中温习了一遍。她说她以后的网名想叫"搭你的肩是我一生的天"，我心里当然不愿意，因为我不想让她叫那么肉麻的名字，被别的男人点来点去的。其实我早就想让她改名了，[相思甜心]太显眼了，而且并不适合她这样温柔含蓄的女孩子。但处于尊重和信任，我没有反对。"这个嘛，挺好听的。你喜欢就行了。"我表里不一的说。我在电话中表白我已经彻底的喜欢上了她，她说：SO DID I。（我也是。）"你不怕我很丑吗？"她突然问到。"再丑我也要你！"我立即回答到。快要挂电话之前上演了激动人心的一幕，就好像电影片段一样。

　　"快说一句话作为结束语吧。"我说。

　　"……"

　　"快一点。电话费不够了。"

　　"……我爱你！"（我怔住了，还以为我在做梦）

　　"我也是。"

　　在挂上电话的一瞬间，我一看电话机上的显示屏，只剩下0.4元了。再一看表，原来已经不知不觉地在温馨浪漫的气氛中度过了45分钟，几乎用完一张30元的IC卡也不足为奇了。这是我们第二次谈话也是我们高考前最后一次了。我在电话中对她说："芳芳，我觉得以后我们还是写信吧。打电话对我们来说有些奢侈。因为我们是学生，不可能一天拿出30元用来打电话。今天这30元还是我向同学借的。"其实我给她打电话的目的就是想听听她的声音，看看她是不是网上的那个她。果然没让我失望，她比我想象中的还要好，比网上更含蓄，更稳重，再加上活泼开朗，温柔体贴的性格，我已经彻底地被她征服了。她在我心目中的地位已经无人可以替代了。我对自己说："好好珍惜她吧，不要错过机会。"

　　丛香同意我可以不再给她打电话，说明她是一个体贴且明白事理的女孩；说明她并不是爱慕虚荣，不求实际，也不是把恋爱当玩笑的"妖精"。她是个普通但又特别的女孩儿，普通在她和其他女孩一样，喜欢浪漫，喜欢白马王子，言谈举止透露出她那十足的女性气息；特别在她比别的女孩儿要成熟，更注重现实，在浪漫的同时不会忘我。"百花丛中唯你最芳香"形容她一点都不过分，也许这是"情人眼里出西施"的缘故吧。

　　从此信成了我们唯一的交流方式，我的生活中从此多了一份牵挂，一份祝福，我对此乐此不疲。虽然我知道未来的事谁也无法预料，而且网络与现实是两个不同的"二次元空间"，但我无所畏惧，我相信时间是考验我们是否真爱的最好凭证。我相信她跟我一样是认真的，而且是极度的认真。

　　我没问她要过照片，因为我并不在乎她长什么样。如果两人相爱的前提条件是对方长相好的话，那么这段感情我想是不会长久的。看了许多电影和电视剧，一般那些花言巧语的帅哥最后都没有赢得女孩儿的芳心，女孩最终还是和那些淳朴善良并一心一意地爱着她的男孩走到了一起。像最近青岛电视二台正在播放的《情书》就是个很好的例子。

　　当时《青岛晚报》正在征集网络美文。我看过几篇前辈的文章，几乎都是关于网恋的。我若有所思，打下了一篇题目为《我的眼里只有你》的800余字的短文，通过E-mail发了出去。可是不幸的事情发生了，征文活动竟然在我寄出稿件的同时拉下帷幕，对此我失望了很久。我对我的作品充满信心，我已经打算买下个星期天的报纸了（征文在每个星期天刊出）。我写的生动形象，真实感人一点也不比那些"前辈们"差。我还打算等报刊一旦刊出，便把它剪下给丛香寄去。现在看来已经不可能了。于是我打算把它打印出来，起码要让她看到我对她的感情是多么的认真。在文章的最后，我问到："相思甜心，嫁给我好吗？"

　　还是托吴寿给帮忙打印了出来，这封比上一封的《今生共相伴》要好看的多。寄出后没过几天，丛香来信说："我不只想做你网上的新娘，我还要做你现实中的新娘。"看到这儿，我觉得自己真的很幸福，我们的心已经完全交给对方了。我对她的爱已经超出了我一开始的想象，所有顾虑和猜测都没有了，正如那篇文章的题目一样――我的眼里只有你！
