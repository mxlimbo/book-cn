写在时间停止之前

无情 （cappuccinoo@citiz.net）

　　如果我还可以听见自己的心跳，我一定会给你幸福，相信我。

　　――题记

　　暮色苍茫的天空，只有鸟群寂静地飞过。她的手从他手指间冰冷的滑落，泪水从她眼角如流水般倾泻，温暖而潮湿。

　　我要走了。她低下头，闭上眼睛，努力掩饰自己快要崩溃的伤感。

　　那天她走的时候，他听见自己内心的寂寞呼喊着她的名字，静。火车呼啸着离开，他看见她的影子在他眼中渐渐弥漫开来，最后留下的只有黑暗。他对自己说，这只是一场尴尬的梦。

　　他和她十一岁认识，十五岁学会彼此之间的拥抱，十八岁第一次初吻.......他以为和她永远会在一起。不过就在半个月前，她父母说要带她去日本。其实他知道，她的父母只是在躲避他而已。

　　烟雾顺着视线垂直升起，他疲惫地倒在床上，天花板的灰暗色刺痛了他的眼睛，他很无奈的闭上，感觉着自己微弱的心跳。他从毕业以来就没有好好做过一份工作，他不喜欢被人束缚的那种感觉。他现在每天很努力的坐在电脑面前写稿子，那些无聊的稿子压抑的让他窒息，可是为了生活，他不得不去忍受这一切。

　　房间里充满了苦涩的香烟味，夹杂着厕所里几天未冲洗的熏味，桌子上还剩下半瓶矿泉水和几片快要发霉的淡面包，床上的手机自从她走后就没有开过，幽暗的屋子里可以照出自己孤独的影子。他喜欢这种感觉，蓝色的大蓬窗帘把狭小的房间笼罩在昏暗的世界里，他一个人静静地坐在电脑前，用他冰冷的手指敲打着键盘，那些碰撞的声响在寂静的房间里特别脆亮。

　　她走的那天，他没有开口留她。他知道自己不配。他连自己最起码的生活都还没有足够的保障，他有什么权利留住她？他凭什么留住她？他给她写信，写电子邮件，他告诉她很想她，他很爱她。可是那些信、那些邮件都没有发出去，他盼望有一天她能够回来看这些东西。

　　有时他会对着电脑发呆，直到他发觉眼泪已经无声无息流入他的嘴角，有种苦涩的味道。每天晚上他对着窗口看天空，寂静幽暗的天空，繁星漫无目的的闪烁，有种堕落感从他内心蔓延开来，空荡荡的大街上，昏暗的路灯寂寞的照耀着。他拉上蓝色窗帘，无力的躺在床上，他忽然发现原来那只手机已经很久没有打开过了。他按下开关，"嘟"的一声，有中文信息。

　　"我打你手机，你已关机。""你还好吗？我很想你。""不要抽烟了，有空出来透透空气。""你为什么一直关机，你不想我吗？""我很好，只是有点想你。""你为什么不打电话给我？""你是不是不爱我了？"......."我决定回来看你，你来接我。4月25日。虹口机场。"

　　他忽然从床上跳起来，电脑上显示：5月2日，20：15。他黯然低下头，继续看信息。

　　"你没来，我很伤心。我知道也许你已经不爱我了。""我这次是溜出来的，如果你不来看我，我不知道自己应该怎么办？""我现在住在华亭宾馆，512房。""我会等你。".......

　　她回来了？他听见自己寂寞的声音从干涸的喉咙里慢慢爬出来，他感觉到自己的手有些兴奋的颤抖。四个月了，想不到他还能见到她。

　　还好吗？在她明亮的房间里他觉得有些不习惯。

　　好。她用她红肿的眼睛注视着他，有残余的泪花。

　　他把她接回家，在他昏暗的房间里，她埋倒在他怀里放声的哭泣，她说为什么你不来接我？为什么你不打电话给我？为什么？为什么？他紧紧抱着她，对不起，对不起。他的声音是颤抖的。那天晚上，他疯狂的和她做爱，放肆地在她身体里缠绕，他很用力的吻着她，她在他身体压抑下痛苦愉快的呻吟，他喊着她的名字，静，不要离开我.......

　　他开始去尝试着努力工作，他对她说，我会给你幸福的，相信我。她用力的点头，直到勃子开始酸痛。他和她都没有什么好的工作经验，找一份工作不容易。他还得维持一段没有工作的日子。以前那些无聊稿子的稿费只能支撑他自己，可是现在情况不同了。

　　他每天很晚回家，然后疲惫不堪的坐在电脑面前，看着烟雾顺着眼睛平行升起，有时会刺痛他的眼睛。他的稿子写的越来越臭，再也没有网站录用。常常半夜的时候，他会爬起来对着窗口看天空，寂静幽暗的天空，繁星漫无目的的闪烁，有种堕落感从他内心蔓延开来，空荡荡的大街上，昏暗的路灯寂寞的照耀着。他从新拉上蓝色窗帘，无力的躺到床上，他不敢转身看她，他怕自己会流泪。他只是默默的闭上眼睛，冷冷地抱紧她。他希望至少她还可以给他温暖，可是他发现，她也一样的冷。

　　那天晚上，他告诉她，今晚不回来睡了，他朋友有些事要他帮忙。她从窗口看着他经过大街的时候，感觉眼睛有些迷糊，她使劲的用手擦亮眼睛，才发觉是那样的温暖和潮湿。其实自己不应该回来的，她这么对自己说。她忽然很想在他消失之前能再看他一眼，可惜再一次抬头的时候，他已经走远。

　　走在寂静寒冷、空无一人的大街上，他第一次感到恐惧。一阵寒意从心底串流而上，他使劲把自己倦起来，走到了大街尽头。那里没有孤独路灯的照耀，抬头望着天空，也没有繁星的闪烁，四周一片寂静，偶尔传来几声遥远的狗叫。他插在衣袋的手冰冷，在黑暗中，他的脸色苍白，孤独的影子不停的颤抖。

　　从远处依稀走近一个人，他抬头看了一眼，低头走了上去。就在他们相遇的刹那间，他忽然从衣袋里抽出一把刀。

　　"快，把钱给我交出来！"他的声音冷的有些发抖。"快！"他急促的催促着。

　　"我没......没钱。真的，没钱。"

　　"妈的，没钱老子就杀了你！快！"

　　他把刀用力的往那个人身上顶了顶，幽暗的夜空中空气瞬间变得冰冻起来。他们僵持在那里，各自等待着什么。就在这个时候，远处传来的狗叫声似乎越来越靠近他们，他忍不住向那边的黑暗里看去，可惜什么也看不见。

　　忽然，他觉得他的刀一下子落空了，那个人牢牢抓住他的手，他们扭打在一起，在寂静无声的黑暗中，他还没来得及反应过来，一阵冰冷的寒意刺入他柔软的小腹，然后身体里感觉到有股温暖的激流涌出，异常的舒适。他倒下去的时候，看到寂静幽暗的天空中，闪过一颗流星，空荡荡的大街上，他听不见任何声音，他无力的闭上眼睛，他看见她的影子他眼中渐渐弥漫开来，最后留下的只有黑暗。

　　这是一场恶梦，他这么对自己说。其实他真的好想再看她一眼，可惜已经再也没有机会了。他和她十一岁认识，十五岁学会彼此之间的拥抱，十八岁第一次初吻.......他以为和她永远会在一起.......

　　远处传来一阵凄凉的狗叫声，那个人在黑暗中慌乱的离去.......
