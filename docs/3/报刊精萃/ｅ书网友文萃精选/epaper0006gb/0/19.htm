该 或 不 该

蕊

　　你转过身去 不给任何的理由
　　我没有离开 也许还想等你回来
　　当你的爱已经都不在
　　我还当一切只是因为无奈
　　你的洒脱让我心酸
　　我却当这只是应该
　　我从来没有太多的期待
　　只是希望你真的还是原来
　　当我已经感觉不到在你眼中的存在
　　我又怎么相信
　　你的心 还在这里没有离开
　　纷乱的世界里 穿梭的人群中
　　我怎么可能握得住你故意闪躲的手
　　望着渐行渐远的你的背影
　　我却要不到一个属于我的未来
　　你看不到我的脸上有泪水的痕迹
　　是因为你已经不在乎我内心的伤感
　　当爱你开始变成一种对自己的伤害
　　我是不是还要继续这没有结果的等待
　　如果清醒还不算太晚
　　那么现在
　　我是不是该就这样
　　安静地走开
