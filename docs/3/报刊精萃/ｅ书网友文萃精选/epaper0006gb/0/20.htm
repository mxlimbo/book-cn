楼的对面的树上的鸟巢的对面的楼里的鸟

风柜来的人

　　1）她最初看见那个鸟巢的时候是早晨，她站在自己的窗台前向外望，看见了对面的一棵树的树顶上有一个鸟巢，一只鸟飞上又飞下。那时树的叶子已经掉得只剩很少，所以她可以把那鸟看清楚，那是一种个体很大的鸟，灰色的羽毛，看起来很矫健。它重复地从树顶飞到树下，一直不停。

　　开始她很奇怪，仔细看了后，就明白那鸟是在树下衔树叶枯枝以及泥土，那鸟是在修补鸟巢。但更让她感到奇怪的是以前怎么从来没有看到树上有个鸟巢，可能是以前叶子太密的缘故，她想。

　　女人站在那里看了半天，直到他走到她身边，给她披上一件外衣。他其实刚刚睡醒，起来却发现她不在他身边。

　　“看什么呢？”他问她。

　　她看了他一眼，却不说话，象是厌倦一样。于是他不再理她，也往前面看去，也看见了那只鸟。他笑着说，“一只小鸟有什么好看的，站这里会着凉。”

　　说完搂住她，想把她带回房间里面。她强不过他，就随了他。在转身的一瞬，她正看清楚那鸟巢里其实是有一只鸟的。

　　她想那应该是只雌鸟。

　　2）“我上班去了。”他说。

　　她看着他给自己系好领带，打好摩丝，擦亮皮鞋，然后提起文件夹走人。

　　他走了，她就守住这空荡的房间，象一个幽灵一样，她自己这样觉得。她看着空荡的房间，却没有看见自己，仿佛自己给房间溶解了一样。

　　她走到电话前面，想打个电话给谁，拿起话筒了，却发现自己根本不知道要打给谁，她在这个异常空荡的房间里和这个异常空荡的城市里竟然找不到自己可以说话的人。

　　于是她又走到了窗台前，看那个鸟巢。

　　深秋的阳光是这个世界上最好的东西，没有人可以拒绝。她就坐在阳光里，这样消磨着时间，让时间流去象阳台上挂着的那些衣服里的水分被蒸发一样，不知不觉。

　　她记得他说过，只要她愿意，她可以去做任何事，去任何地方，他将给她充分的自由。她就是这样，从来不相信自己可以改变的现状，而更相信他所承诺的，所以失落了以后，她就想着他似乎没有履行对自己的诺言。

　　这时她突然想起了那双鞋。

　　她要离开房间的时候又看见鸟飞出了巢，还是那只雄性的鸟。

　　3）他在公司里忙碌着，象个被驱使的蚂蚁，没有歇下来的时候。而他一旦可以停歇，他就会想起她。

　　他其实已经觉察出她的变化，她总是对他默然不语，如同陌生人。他不知道她为什么这样，因为他觉得自己对她好的已经是无以复加了，可她就是不知道是怎么了，老是没有一个好脸孔给他看，和他在一起也没有一点热情。

　　他和她其实是同居，开始她不愿意，而他近乎强迫，他是个固执的男人，而她就这样顺从了。他知道她爱他，而他也是爱她的，这是个接近于笑话的过程，因为最后就是由于这种无距离的靠近产生了距离。

　　他开始怀疑距离。

　　他开始怀疑自己做的一切，但他无法也无处寻找答案。

　　蚂蚁有敏感的触角却不能看到很远的地方。

　　4）她把自己的鞋子拿去修补，因为鞋跟掉了下来。

　　鞋子是他买了送她的，很合脚的一双鞋子，也很漂亮，那时她很喜欢，但她终于将它穿旧了。

　　什么都会陈旧，特别是最爱的东西，她知道。

　　他曾对她说，“掉跟了？不要算了，改天再给你买一双吧，昨天我路过##鞋店，那里又进了些女鞋，很好看。”

　　她看了他一眼，觉得有些累，她不喜欢他象女人一样过分地贴近她的隐晦的世界里，因为他除了给她买鞋，还为她买贴身衣物。

　　他记得她的三围，清楚她的脚的尺码。

　　修好的鞋好象又能穿了,她试了试，就是觉得有些别扭，可是她还是坚持穿了。

　　她心里原来还是有些指望的。

　　5）他回来的时候果然带回来一双鞋，很漂亮的女鞋。

　　“店里的女人说这是最新的款式，你试试。”他把鞋拿到她脚前面。

　　她看见这男人此时就象一个恭顺的妇人，她厌烦。但她也知道，一会他就有可能变成专制的帝王，尽管他是无意识的，可她也厌烦。

　　“别这样，把脚给我。”他耐住性子把她鞋子脱下，却看见她的脚上起了两个水泡，她没有穿袜子，“怎么不穿袜子？”

　　“不关你事。”她冷冰冰地抛写一句。

　　他终于再忍耐不住，他把鞋子往地上一摔，“他妈的你到底想什么？老子这样伺候你，你却给老子脸色看。”

　　“谁要你伺候了。”她回转身朝窗台走去。

　　她看见那只雄性的鸟又衔了几根草回巢。

　　而他坐在那儿，象一头失败的狮子。

　　6）夜里，他和她躺在一张床上。

　　原先他是想要和她亲热的，她却死活不肯，他勉强不过来，只好做罢。

　　 月光象水一样铺在床上的一角和橡木地板上。开始他是要拉了窗帘的，拉上了，他不喜欢这样的暴露自己的睡眠。而她又把窗帘拉开，他再拉上，她又拉开，坚决不肯妥协。于是他就妥协了，他真的很想睡了。

　　她看着身边打鼾的他，竟然感觉到陌生，以前从来没有过这样的想法。

　　她想，树上的鸟应该不会相互感觉陌生。

　　“我如果不在你身边，你会习惯吗？”她问已经睡去的他。

　　他没有回答，因为他已经睡去。

　　7）第二天他出门的时候她对他说，“你回来的时候看不见我会吃惊吗？”

　　说这话的时候她是笑着说的，笑容很灿烂，他很久没见了，很惊异，所以他说，“开什么玩笑？我先走了，宝贝。”

　　她看着他象个绅士一样走下了楼梯，突然记起什么，便叫住了他，“等等。”

　　“什么事？”他有点厌烦。

　　她把早上送来的新鲜牛奶给了他，他摇了摇头，却也返身接了过去。

　　就这样，蚂蚁吮吸着牛奶走下了楼梯，却没有回望一眼。

　　8）又留下她一个在空荡的房间里，她想起夜里的那个想法，不由得有些落寞。

　　她开始收拾她的东西，她只想带走自己的东西。她收拾了一下，却发现在这个房间里，她曾以为那些属于她的东西只剩下了不足装满一个行李箱的衣物，没有什么是她所想带走的。

　　她提了提轻盈的行李箱，觉得自己也象箱子一样空荡。

　　她坐到了地上，听见了鸟的叫声。

　　她从窗台那里看出去，发现那只鸟就歇在阳台上的晒衣杆上，连忙向阳台走去。她踮着脚，尽量地不发出声音，慢慢靠近了鸟。

　　那鸟其实已经看见了她，却不飞走，可能意识到她的无威胁。

　　她和鸟只有咫尺的距离，她站在阳台上，仿佛伸出手就可以触及鸟的羽毛，她是多么想摸一摸这鸟的羽毛啊。但她伸出手时，却发现她够不着。

　　鸟没有飞走，鸟停在那里象是一种引诱。

　　阳台的栏杆下面有几个花盆，里面原来种过茉莉的，后来茉莉都死了。她小心地踩到了花盆上，这样能够缩短鸟和她之间的距离。

　　果然，她的手要触到鸟了。

　　她的重心前倾。

　　这时，花盆突然被踩翻了。

　　9）他坐在没有了她的房间里，知道即使她没有踩翻那个花盆，她也要离开他的，因为他看见了她收拾好的行李。

　　他开始回忆她那天早上说的话，原来她真的想要离开他，只不过命运作弄了她一下，把她从预想的路带到了另一条路上，这条路没有人可以走得回。

　　他想到这里，心里竟然欢喜了一下。

　　他觉察到自己的欢喜，凭空有了一种罪恶感。

　　为此他搬出了这栋楼。

　　搬出之前，他买了一支气枪。

　　一天傍晚，他对着那个鸟巢总共发射了三合铅弹。

　　从始至终他都没有看见鸟的影子，他想鸟应该是在巢里，鸟不在巢里又能去哪儿，他这样对自己说。这坚定的意识促使他打完了三合铅弹，打完之后，他的手就象酿过的黄瓜一样软了下来。

　　10）楼下的老人早上做晨练时，照样会看见树顶的那个鸟巢。

　　鸟巢里还会飞出两只鸟。

　　原来三合铅弹并没有伤害到它们。

　　“鸟的羽毛是灰色的。”一个阿婆对老伴说。

　　“瞧你那眼神，羽毛是黑色的。”她的老伴取下老花眼镜下后这样说。

　　而不久以前，两个老人曾看见楼上飞下一个人来。
