如风（网恋系列之三）

第三空间

　　有这样一种男人，他的笑很温和象春天的风。他可以是沉默的，你不在乎看不看得见他的心。你可以看着他的眼睛，象冬天的风，冷峻而凛冽。

　　你每天生活在一条必需的路上。从家到公司，从公司回家里。以前你没有去留意过这样的男人。有一天你偶然坐到电脑前在网上遇到了他。你去的地方就是他的个人主页，里面有图片和笔记。你就有了这样一个深刻的印像。

　　这很莫名，是吗？

　　不，你相信有这种男人，他和你周围的男人都不一样。他可以很潦倒，但拒绝平凡；他可以孤独，但身边一定会有痴心爱他的女孩。就象古龙古大侠。

　　你想你是小说书看得太多了，那也是年少花季的时候。很多事都遗忘得干干净净，直到这天被这个网上的男人唤醒……

　　如风，这是你给他取的名字。

　　你还是每天生活在一条永恒的路上。从家到公司，从公司回家里。但你的生活不知不觉起了很多变化。你开始留意身边的每个男人，看他们不同的笑容和眼神。

　　你的丈夫也感觉出你的变化。你快乐了，不再为琐碎的家务而沉着脸。你温柔了，笑容挂在脸上，房间里弥漫着是你经常买来换上鲜花的香味。似乎五年前恋爱时的氛围有回来了。

　　你对一切都重新有了一种满足感。

　　生活可以简单而浪漫激情。

　　女人都希望按照自己的想象体验生活，你也不例外。

　　花开有时落无时。

　　几个月后，你又特意去了他的个人主页，在他的留言簿里留了一句话：多谢你，我曾经爱过的男人。
