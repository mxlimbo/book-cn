明天你还爱我吗？

无情 （cappuccinoo@citiz.net）

　　她独自安静的坐在那里，阳光慵懒的从对面大街上斜射过来，感觉暖暖的。她要了一杯咖啡，冰冻的。她喜欢这种感觉，冷冷的，触摸到内心。她觉得自己还能够承受。

　　这是一家露天咖啡屋。独有的帕格尼尼小提琴演奏"爱情的一幕"，熟悉的音乐缠绕着四周掺叉整齐的桌椅。她轻轻端起咖啡，很小心的呡了一口，有一种冷的心痛的感觉。她抬头看着对面大街，迷茫的眼神似乎在等待着什么。

　　她好几次都忘了去喝那杯咖啡，直到它又变得滚烫起来。她迷惑的看着那杯咖啡，感觉到自己全身的颤抖。她很努力的使自己镇定下来，然后一口闷了下去，依旧还是冰冷的。

　　她很无奈的抬起头，看到了一个男人。

　　阳光依然温暖，那个男人挽着一个女孩的手，嘻笑着从那条大街上相拥走过，女孩紧紧依偎着他，幸福的笑容蔓延过整条大街。

　　她默默地注视着他们，泪水没有感觉的滑落下来，滴入那杯咖啡，瞬间变得冰冷。

　　"今天我在那条街上看到你。"她低着头，把整个身体埋在沙发里，眼眶还有残余的湿润。

　　"是吗？在哪条街？"他整理着上班的文件，力不从心。

　　"我已经看见你们三次了。"

　　"什么？什么三次？"

　　"你和那个女孩。"她抬头，怨恨的看着他。

　　一堆文件忽然不小心从桌上掉了下来，他没有蹲下去，他有些迟钝的停住。

　　"我......"他不知道自己应该怎么解释。

　　"林.....你爱我吗？"她小心的看着他，感觉到自己的内心已经冰冷。

　　"......当然，我爱你。"他僵硬的笑着，弯身拾起那堆文件。

　　"也许，今天你爱我，可是明天呢？明天你还爱我吗？"她又低下头，房间里充满苦涩的味道。

　　"明天？...呵，当然，我依然爱你。"

　　"......我们结婚吧？"她忽然从沙发上起来，抱住他，泪水滴落在他手心，温暖而潮湿。

　　"我不想失去你......."她埋在他怀里，放声的哭泣。

　　"结婚？"他感觉到她在他怀里用力的颤抖，用力的抽泣，泪水浸湿了他的胸口。他轻轻抚摸她，吻着她，"好的，我们结婚。"

　　一个朋友曾经这么跟她说，婚姻可以拴住男人的心。

　　所以结婚的那天晚上，她第一次觉得自己是真正的拥有他。他在她身体里放肆地没落、缠绕。他们拥抱一起的时候，她发现自己的内心没有想象中的那么冰冷，至少她可以感觉到他的温暖......

　　阳光懒懒的斜射过那条大街，对面咖啡屋传来独有的帕格尼尼小提琴演奏"爱情的一幕"，熟悉的音乐渐近渐远、忽隐忽现。他挽着她的手，嘻笑着从那条大街上相拥走过，她紧紧依偎着他，幸福的笑容蔓延过整条大街。

　　她忍不住回头看了一眼那家露天咖啡屋，掺叉整齐的桌椅在阳光的照射下显得有些零乱，她突然发现那里有许多熟悉的眼神，迷茫的，似乎在等待着什么。那杯冰冻的咖啡，冷冷的，有些心痛。她抱紧他，努力使自己镇定下来，可是依然很冷。

　　他们每次经过那里的时候，她都发现了，那么熟悉的眼神，迷茫的，冷冷的看着她，就象她当时看一个女孩一样。每次她都快步的离开那里，她不想看到原来的自己。

　　阳光依然温暖，他挽着她的手，快步从那条大街上相拥走过，她紧紧依偎着他，忧郁和恐惧渐渐蔓延。她不敢回头。她觉得自己内心很冷。

　　就在这个时候，她突然感觉到身后一阵刺痛的冰冷，冷的心痛。她无力地从他身旁倒了下去，血从她后背如迷雾般洒了出来。他吃惊地扶住她，血喷在他颤抖的手上，温暖而潮湿。

　　一个女孩呆呆的站在他们身后，用迷茫和恐惧的眼神看着他。她的手，同样的温暖而潮湿。

　　他发疯似的大喊，快叫救护车......

　　那个女孩痴痴地看着他，"从你们结婚的那一天起，我就对你说，我不会放过你的，我得不到的东西，别人也别想得到！"

　　他没有理会那个女孩，只是紧紧地抱住她，她的身体冷的让他心痛。她缓缓地睁开眼睛，迷茫的看着他。

　　"...你...爱我...吗？"

　　"爱......"他握紧她的手，感觉到彼此的颤抖。

　　"那...你...明天...明天还爱我吗？"

　　"爱......永远...都爱......呜......"他忍不住放声哭泣，哭声渐渐淹没了对面咖啡屋传来独有的帕格尼尼小提琴演奏"爱情的一幕"，世界突然之间变得冰冷。
