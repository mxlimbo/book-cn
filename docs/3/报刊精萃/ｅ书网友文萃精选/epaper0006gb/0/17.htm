一个杀手的悲哀

无情 （cappuccinoo@citiz.net）

（一）

　　暮色苍茫的天空，只有鸟群寂静的飞过。他静静地坐在门口，握紧她的手，感觉到彼此之间的颤抖，他的眼神迷茫而忧虑。

　　你是在害怕？她依偎在他怀中，抬头很小心的看着他。

　　不是。他抱紧她，低头闻着她清香的黑发，漠然的回答。

　　明天你有把握杀了无情？她继续深埋在他怀中，感觉着他微弱的心跳。

　　没有回答。

　　远处，褐色的鸟群飞翔过那片空阔的田野，然后消失在那里，偶尔传来几声零乱的鸟叫，隔着灰白色的云层，让人捉摸不定。他低下头，努力感觉她带给他的温暖。他告诉自己，他还是个杀手，这是他的职业，他需要用它来维持他们所应该拥有的幸福生活。也许过了明天，天空将变得蔚蓝而明亮，阳光灿烂，鸟群也不再寂寞。他这样想着，僵硬的笑了起来。

　　那天晚上，他睡的很香，他做了个梦，梦见他们象鸟群一样飞翔过那片空阔的田野，然后停留在白色的云层上，他很温柔的吻着她……

（二）

　　"你找我？"幽暗的夜空，繁星漫无目的的闪烁，无情冷冷地看着她。

　　"明天会有个人来杀你，我知道他不是你的对手，你可不可以放过他？"她的眼神真诚、忧虑。

　　"我为什么要放过他？他是来杀我的，他会放过我吗？"无情依然冷漠的看着她，夜空下，她美丽的影子孤独的站立在那里。

　　"就算我求你……求你放过他。"她无力的哀求。

　　"哦？你求我？你用什么求我？如果你带来些钱什么的，我可以考虑。"

　　"我……只有……"

（三）

　　当第一缕阳光从窗外斜射进来时，他握紧刀，垮出了门口。离开的时候，他感觉到自己内心的寒冷，他全身无力，头晕脑昏，他最后又回头看了她一眼，她的眼睛有些红肿，头发还没有整理，有些零乱。

　　鸟群在暮色的天空上飞翔，那片空阔的田野还有晨露依附在上面，天空是明亮的，阳光也很灿烂。她看着他的背影消失在远处，泪水忍不住从眼角滑落下来。

　　她知道他是爱她的，她也一样。她还知道她不会站在门口等很久，很快他就会回来了。

（四）

　　"你找过他？"他疲惫不堪的站在她的面前，狠狠地看着她，"他对你做了什么？"

　　"没有。"她显的力不从心，她低着头，不敢正视他。

　　"你骗我！"他用力抓住她的衣服，零乱的头发在他摇晃中飘荡，"说，你倒底给了他什么好处？"

　　泪水从她眼角如洪水般倾泻，她在他粗暴的双手下整个人都快崩溃。鸟群从他们头顶盘旋而过，寂静地没有发出任何零乱的叫声。

　　"说！你们是不是……"他的声音接近颤抖，刀在腰间拼命的镇定，不让自己轻易的滑落下来。

　　她抬起头，用红肿的眼睛凝视着他，那是一种对他的怨恨。

　　"是，是又怎么样？我都是为你好！"她大声的喊叫，发疯似的挣脱他的双手，倒在了地上，放声的哭泣。

　　"好...很好...为我好？呵，为我好？"他断断续续的自言自语着，内心的恍惚使他开始迷失自己，他走到她的面前，抓起她的头发，冷冷道："你是个不要脸的女人。"

　　寂静。死一般的寂静。

　　她抬高头，用惊讶的夸张的眼神迷茫的看着他，那么的熟悉而陌生。她永远都不会相信他竟然会对她说这种话，她忘了怎样去反击他，她的心碎成了千千万万片。然后她有所醒悟的给了他一个响亮的耳光。

　　他愤怒了，腰间的刀轻易的滑落下来，他看着她，他的手颤抖不停。

　　"你杀我呀？你杀呀？"她发疯的吼着，零乱的头发盖住了她，象一个落魄的疯子。

　　他的全身开始颤抖，刀在他手中发出风铃般的响声，有风吹过。那片空阔的田野传来她吼叫声的回音，你杀我呀，你杀呀。鸟群消失的无影无踪，一片灰白色的云层遮住了灿烂的阳光。他看着她，眼眶湿润，刀从他手中无声的滑下。

　　"我不想再见到你。"他转过身，头也不回。

　　她倒在地上看着他的背影渐渐从她视线里远去，忽然起身拿起那把刀…….
