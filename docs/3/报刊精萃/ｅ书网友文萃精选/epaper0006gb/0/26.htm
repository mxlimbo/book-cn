踢 球 者

碎梦一刀

　　眼看着那个白色的足球似陨星一般急速的向面前袭来，我迅速的侧身抬右脚，将脚内侧迎向足球，在球和脚接触的一刹那，轻轻一收，便消去了球急速前冲的势头。

　　抬头飞快的看了一眼周围，对方三个球员从不同的方向围了上来。我轻轻用右脚一挑足球底部，将球挑至半人高度，然后摆起右脚，对准球门作势欲踢，正前方两个对方球员忙收住脚步，做出了保护动作。我便又用膝盖将球顶起，身子便往前冲，从两个尚未合拢的后卫防线中挤了过去。当球还在空中飞行的时候，赶上几步，用头将球顶往前方。在对方拖后迅速前移准备封住我出球路线，将球破坏的时候，我忽然的一停身，变魔术一般的用右脚将球勾过顶，紧接着便侧身起左脚往球门内侧下方一脚抽射。对方门将显然没有想到我会在这样的情况下射门，虽然扑了一下，却连球皮也没有碰到，眼睁睁的看着足球滚入网底。整套动作如同行云流水，干净而又流畅。

　　好球，场内外立刻沸腾了起来。紧接着主裁吹响了全场结束的哨声。1 ：0 ，中国队终于战胜了宿敌韩国队，闯入了世界杯决赛圈。

　　足球评论员黄健翔兴奋至极点，几乎口不择言的称我为亚洲乃至全世界最杰出的球员，盛赞我的左右脚都能拉小提琴，并且拉的是‘梁祝’棗中国的经典。

　　我兴奋至极点，因为中国队终于走向了世界。

　　大声的笑着，仿佛中看到了球员们站在大力神杯面前合影的情景，我伸手想要从萨马兰奇手中接过大力神杯，可是哪个小子在拼命的拽我？

　　“起床，快，要迟到了，今天公司开职工大会别给抓了典型。”

　　我听见有人在叫，努力的晃了晃脑袋，睁开眼，朦胧中发现站在面前的是舍友兼同事小冯。

　　公司老总一声会议结束，大家立刻纷纷作鸟兽散，逃也似的，迅速离开会议现场。我也赶快溜走，准备到外面吃碗豆浆油条。

　　“Hi，小桂等一下。”

　　我听见有人叫我，只好停步。不会是头在叫我吧，说实在的，每次头叫我留下来总没好事。经常被蛇咬，我已经到了见条状物就害怕的地步。当我怀着极大的勇气回头看时，却发现叫我的人是我们球队的队长老黑。

　　我们队长为什么会叫老黑呢？其中有个故事。据说在我来公司之前，我们队有一次和一支比较强的队交手。当时，两队正处在胶着状态。你来我往多次射门均无建树。终场之前，突然对方发动了一次反击，锋线球员带球突破我方后卫防线拔脚怒射，被我方门将奋力挡出。

　　但球没有击远，禁区内一片混乱。这时侯，忽然一员猛将快马杀到，迎着球就是一脚。真是急如流星快似闪电。只见那白色皮球，划出一道奇异的弧线，直奔球门远角而去。门将虽奋力扑救终是无法抵挡这一记势大力沉的劲射，皮球应声落网。场下立刻掌声雷动，对方拉拉队纷纷询问那个射手的名字。结果没人知道这匹黑马是从哪里冒出来的。后来是我方球迷给他们解答了这个问题。原来射门的是我们队的托后。自此之后，他有了老黑的称号，直到老队员纷纷‘退役’，他当了队长，这个称号还是一代代的流传了下来。

　　知耻而后勇，老黑作为我们的队长确实还是尽心尽力，我们也因此对他由衷的尊敬。他叫我干嘛呢？我投射出征询的目光，他立刻的知道了我的意思。

　　“周六下午两点在云工足球场和设计院踢球，准备一下，你到时候可能首发出场，早点到。”

　　“好。”我回答的干净利落，当然了，能首发出场和人一起踢球总是很愉快的事情。特别是对于我这种精力过剩的人来说更是如此。

　　虽说是接着打了好几场球，可我总是作为替补出场，这一次总算轮到首发了，这不能不算是一大喜事。我为之而兴奋不已。

　　“操，真他妈的水货。你丫从后场跑前场去就为了给对方解围啊。那一脚矢球连门框都没挨上边，还赖在上面。滚下来，让老子上吧。”

　　土狗难得的不和我争吵，屁颠儿屁颠儿的在我眼前跑来跑去，炫耀着他的愉快和满足，我可以看出他的得意。

　　“懒得和你多说，就不下来，你要怎么着？”这小子虽然没说，可我总觉得他就这意思。在场下大声的嚷嚷着，看土狗一次次的把机会浪费掉，我心中越发的不平起来。说好让我首发出场的，这松一来，便使我沦为了替补，真是可怜啊！我为自己的不幸而悲痛莫名。

　　环顾四周，和我一起坐场下的还有不少人。除了我，三毛和白胖他们都挺安祥的样子，如同北京四合院中的老太太，悠然自得的抽着烟，聊着天。N多个家属围绕在球场的四周，试图对对方球员造成心理上的压力和威慑。土狗的女朋友也在其中。

　　一般来说，我们球队的出场阵容根据几条原则而订。主力队员的位置是铁定的，主要是留给“巴乔”、“小罗”等几个少有的队中球星。其他出场队员通常采取轮换制，轮流首发。如果有亲友团助战，则可以象分房一样取得优先权。其中特别是有队员女友莅临的时候，一定给予最惠待遇，往往获得出任锋线位置的机会。

　　本来这是一件很公平的事，可令我气愤的是这个月接连踢的三场球，土狗这家伙每次都带了女朋友来，结果当然是夺走了我们多次难得的首发机会。我怀疑这小子是为了踢球故意把以前的老同学拉来又或是让他的表妹充当女友角色。为此，我做过详细的调查工作，然而据消息灵通人士报告，那女的确实是土狗的女友。我无话可说了，只有在心中想着，这小子一定是为了踢球，骗取了某位女士的同情。

　　虽然我一向对自己单身贵族的生活表示很大的满足，并且对那些成了家或被女友缠身的家伙表示了极大的同情。这时却第一次的发现有个女友确实还是挺不错的，至少多了不少上场踢球的机会。

　　实事求是的说，我的球技确实不怎么样，反正就是混球踢的那一类。可其他的人好象也不怎么样。真要踢的好的，也不在亚洲待着了，至少不会在中国待着。就国家队里的那几扎，还什么将军什么的，不也就只会用身体撞对方，捞多几张红黄牌吗？

　　三十分钟，白胖被换上了场，四十二分钟，三毛也被换上了场。而我竟然还是坐在场边，望穿秋水，衣带渐宽。怎么说白胖也是队长的老部下了，咱能说什么？至于三毛更是没法比，有个公司老总做老爸总是有好处的。闲极无聊做了N次准备活动，在场边拼命的射墙，维护场馆的老头因此一个劲的替我跟队长说好话，“你们还是快让他上场吧，否则，我们这房子就被他踢倒了。”我怀疑老黑是贝尔格的关门弟子，对于这样的民主呼声竟然听耳不闻。

　　我虽然不是杨晨可咱队伍也不是法兰克福啊。群众和家属纷纷呼吁，“让他上场吧，不然他老在场边乱踢，别踢到了我们。” 尽管周边群众的眼睛雪亮，支持我的群众多不胜数，可就是上不了场。

　　上半场结束前两分钟，对方9号连过我方三名球员，一脚抽射，皮球直奔球门右上方而去。门将“张跳墙”伸了伸手，姿势优美潇洒，使我联想到了拈花摘叶手的高雅和白鹤舒翅的美丽，只可惜对方射出的球素质比较低，对这样的心灵沟通没法领悟。只是让他碰到了一下球皮，足球依旧执迷不悟的钻进了网底。

　　“真矢，都跟他妈木头似的。”我大声的嚷嚷，没有得到上场机会的气愤在这时充分的发泄了出来。

　　中场休息，大家都没有话，我也懒得说，就听老黑在拼命的自圆其说。

　　“大家踢得都不错，我们有很多机会只是没把握住。坚持这样的打法，一定能追回来……”

　　没人搭理他，他还在一个人说个不停。

　　这家伙怎么跟周星驰似的，挺能说的嘛。看他的两片大嘴上下翻飞，如同急速旋转的水车板，唾液如同水车带起的水花四处飞溅。这一刻，我想做的就是尽快的找块饼塞他嘴里。‘这家伙要找机会送到足协说不定又一王俊生呢。’我忽然冒出了这样的一个念头，但随即又迅速的打消了主意，‘没点路子，你以为就嘴皮行就能做主席了。’看着老黑的振振言词，看着队员们一个个跟蔫鸡似的。我突然想起了中国队，没有大气，没有自信，多的只是不切实际的空谈。中国队在亚洲杯上能走多远，看的完全是运气。那我们队如果要反败为胜又要靠什么？

　　当然了，下半场开场还是没我上场的份。谁叫咱话多呢。五十多分钟，胖子忽然的躺到在了地。

　　远远的，我看见老黑对着我们这个方向招手。是叫我吗？我表示怀疑。伸手指指自己，对老黑表示征询。“就是叫你了，快，准备上场。”

　　“好。”我忙不及的脱去外衣，心中一万次的对胖子的摔倒表示了自己由衷的感激。撒腿就往场上跑，庆幸一直没有穿上长裤，不然又得麻烦一番。

　　比赛结束的时候，我们3：2反败为胜。我一个球也没进，就赶上射了两次门。其中的一次被对方门将扑救之后击中门竿。我因此引以为得意，先前的不快早已灰飞湮灭。

　　回家之前，一起去饭店撮了一顿，我听见了没能上场的那几个在赞扬了队长的‘精彩绝伦’的球技之后大声的问“下次踢球什么时候？”
