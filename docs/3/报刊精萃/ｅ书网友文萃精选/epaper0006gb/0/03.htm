指尖情人（网恋系列之一）

第三空间

　　键盘。当你用手指尖轻触它思念着某人的时候……

　　也许，你并不想在现实里见到他。但是你对他有一种奇特的感觉，在网络里。你和他认识不久，在聊天室里秘语一番后交换了ICQ号码。你每次上网他都来找你，不知道是哪一天谈到了感情。在围绕这个主题聊了几次他就说，我喜欢你。你突然有了一个奇怪的念头，你想他会送你一朵火红的玫瑰花。后来，他真的说，我会送玫瑰给你。于是，你就期待着这一天的到来。

　　终于，有一天你的办公桌上出现了一束火红的玫瑰，和一份精致的礼仪电报，上面印着：I Love You,Forever。走进洗手间里看着镜子里快乐的泪水，你很想永远就这样看下去。

　　键盘。当你用手指尖轻触它思念着某人的时候，你的心已经飞得很远。你感觉你就在他身边，和他吻个你死我活。你觉得这样还不够，就让他进入你的身体在灵魂深处去感受温柔的被爱。你的想象和你的指尖下隐喻着的嗳昧字句陪你到午夜零点。

　　可是，你只是想象。你不会真的和他见面，更不用说生活在一起。原来的生活不容许去破坏。你有一个谈了四年的男朋友，你的父母很中意他，你也觉得他不错，朋友们说你们很般配。这些话概括了他所有的优点，所以你静等着他来娶你。你知道这一天不会让你等得太久，你已经无所谓他是不是象刚开始谈恋爱的时候对你甜言蜜语，温柔浪漫。你不想去想得太复杂，你很少对他说你心里真实的感受。语言变得累赘，就连细致的微笑也会是装的。你只有一个目标，让他娶你。

　　 但，你需要一个情人。一个很安全的情人。你要把你的感情释放出去，所以一束火红的玫瑰，使你动了情，也动了春心。那端电脑前坐着什么样的人和你的生活无关，你喜欢他是喜欢他的存在，一个确确实实的情人。他会浪漫地送你玫瑰花，并说：我爱你，永远！

　　这是指尖情人，上网男女的专利。
