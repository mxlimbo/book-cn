青 春（四）

Terry

　　7月23日，终于发榜了。不出我所料400有余。这当然不是我期望的，这种分想考烟大简直是痴人说梦。25日到校，碰到刘超和李鹏飞，他们也考的不甚理想，前者因为大失水准，才考了458分，气的好几天没吃饭。而李鹏飞也是出乎意料，他应该和我一样的水平才对，我想肯定是英语把他拌到了，因为他英语极差。不过除了我们之外，许多在我看来平时学习不错而且都比我强一大截的人考的都不过如此，有的甚至还不够400分。历年高考都有这种现象，谁也说不清为什么？不过有一点中国人都清楚：考上是英雄，考不上是狗熊。但我在报纸上看过一个笑话，是这样的：有一种说法，高考就是落榜者立即失业，而在榜者延缓三到四年再失业的一种考试。

　　晓荷是我在网上认识的另一位女孩儿，是我在网上最要好的朋友，我和他第一次聊天时是我和刘超一块用[万诱引力]时。当时因为认识的人太多，所以对她的印象除了知道年龄和所在城市外别的都记不得了。后来我在齐2又碰到她时我们开始了彼此的交往，慢慢的成了"哥们"。她心直口快，但心地善良加上热情大方让我很欣赏她的为人，希望现实中的她也是这个样子。她的真名叫张梦娜，跟我一样同样是高三，她是济南盟光高中文科班的学生。除了数学外，她的功课都不错。有一次，我们谈到了学习，我对她说："我数学很差，有一次才考了30分。""那算什么，我13分。"我随即打了一个字"晕"。她应该是经常上网以至于我几乎每次叩进聊天室的大门都能见到她的身影。我和她也偶尔通信，大约是一周一封。每一次她写的都很认真，字也瞒漂亮的，尽管她自己不这么认为。临近高考的一封信她写了有正反面满满三页，当时我很感动。但我每次回信都是一页纸，因为言简意赅应该算是我的风格吧。

　　张梦娜有个同学叫白玫，也是个十足的网虫，她的网名叫"紫苑花"。这个世界可真是说大就大，说小就小。"紫苑花"竟然是刘超在网上最要好的朋友，他们当时的关系就跟我和晓荷一样。而更有趣的是在这之前我还跟白玫聊过，只是我们好象说不上话来。我们四个相识不能不说是一个奇迹。刘超和白玫可以说是超级网友，一天不见，如隔三秋。他们俩经常打电话，写信更是家常便饭。现在他们好象已经认识了十几年似的，真让人羡慕啊！说道刘超，他可真是个"万人迷"。在他上网的短短几个月里，就相继让4个女孩中了电，其中有2个还是在网吧认识的，我不大相信一见钟情，可这个东西偏偏发生在这些女孩身上了。

　　关于这两个女孩我想以后再谈，还是先让我们看看让刘超一开始心魂颠倒的网上情人"H.O.T"吧。她叫杨蕾蕾，是浙江一所艺校的学生。除了古筝达到8级外，她还会谈钢琴和竖笛，唱歌也不错。现在已经开始给别人教课了，听刘超说一个月2000块。而且她今年又的了5000元的奖学金，可以说是一位十分有才的女孩儿。

　　说到这儿，让我突然想起了那些40元/时的家教。我真很不得抽他们耳光。他们真他妈的不要脸。既然是老师，教书育人就应该是理所当然的事，就算你想得一点酬劳也不用40元/时啊！你知不知道现在的家庭挣钱都不容易？你他奶奶的的1个小时收的钱相当于普通工人几天的工资啊！你不就是发几张卷子出几道题给受害人作吗？就算你动动你那金牙玉口，也不用这么贵吧？对，你是在育人，是在教书，不过我请问：你以前是老师吗？你当初为什么要当老师，莫非你早就盘算着要当"夕阳红"吗？对了，据我观察，这些老师还真实名副其实的"老"师，这种不交税还丧尽天良的像刽子手一样狠宰学生的全是些老东西所为，他们粘污了老师的名号，他们根本不配当老师，他们更扭曲了老师圣洁的形象，他们应该被所有人包括他们自己所唾弃。亏他们还念过书，个个都是相当年的学习尖子，都背过《毛泽东语录》，都学过《思想政治》，都曾经不止一次的说过："打倒资本家。""为人民服务。"之类的豪言壮语，看来这些都是在"文化大革命"时昧着良心不得不说的话，而他们现在却一个个地都成了资本家。只不过资本家剥削的对象是工人，而他们的对象确是那些"傻啦吧唧"的孩子，之所以说这些孩子"傻"，其实是因为他们甘愿受剥削，甘愿让自己的父母给那些刽子手送钱而自己还麻木不仁，无动于衷的行为让我感到不解，所以才说他们傻。其实他们不是傻，是蠢才对！我认识的一个同学，先是高一就在即墨一中借读，因为吃不了那个苦第二年又回到四中，本以为在那种地方就算待上一年学习也应该在你我之上，没想到他回来后我们班原来的最后一名终于可以摆脱"大拉"的称谓了，因为让这名从即墨回来的"高才生"给取而代之了。于是，高二，他就四处请家教，什么化学，物理，数学，英语统统都请遍了，可是不知道是这些老东西们没教好还是他脑子真有点问题呢？一直到快高考前他还是班里"前几名"。最后，高考成绩下来了，乐得他合不拢嘴。也是的，他竟然考了321分？！我们本以为他能250就不错了。（虽然他是个二百五）看来这次他是超常发挥了。像我们这样的破学校，不用说没请家教的，就是只请一个家教的都是凤毛麟角，我属于前者，所以在里面算得上稀有动物了。

　　上面一堆"废话"并不是针对杨蕾蕾的，因为它们是两回事。人家杨蕾蕾是在卖艺，是在课余赚点零花钱罢了，你们这些老东西是没有退休金还是怎的？难道是适应社会大力发展市场经济？现在你们一个个都成万元户了，可你们在我眼里一文不值，你们是一群披着人皮的狼寄居在世界上一个叫"学校"的地方，口蜜腹剑，假装仁慈地赚取每一只没有大脑的羊，我再次为这些羊感到惋惜，谁叫他们没有大脑的？这些老东西不配得到"老师"的称号，我看应该叫"老屎"才对，他们让我感到恶心！因为我一看到他们就想起了大便。

　　好了好了，不谈这个了。再谈我可真要吐了。

　　下面稳定一下情绪继续我的正文。

　　杨蕾蕾和刘超可以说是一见钟情。他们第一次聊天就认定对方是自己要找的人了。老天把两个都是第一次上网的人结合到了一起，这就是网络的奇妙，他可以让两个互未谋面的人成为私订终生的恋人。但我当时直言相告：网恋不可信。不过，这句话在他们以后的相处中显得越来越微不足道。他们真的相爱了，而且都不能自拔。但其过程并不是一帆风顺，他们经历了吴寿的一颗炮弹，差点这段刚刚开始的感情就此结束。事情是这样的：一次，刘超与杨蕾蕾如约来到齐2，他们一见面就像久违的恋人一样亲密地攀谈起来。可没多会儿，杨蕾蕾的言谈让刘超觉得不对劲，她突然问到："你在和别的女人聊是不是？""没有啊！只在和你一个人聊。""你骗我！"随后，杨蕾蕾不说话了，任凭我们怎么对他说，他都不再理我们。突然，屏幕中出现"foreign悄悄的对[万诱引力]说她好骚啊！" "foreign？这好象是吴寿的网名。"我对刘超说。"肯定是他在捣鬼。"果然没错，不知是什么原因，吴寿对杨蕾蕾说了不少刘超的坏话，如：刘超是个大色狼。我们气不打一处来，刘超更是怒不可斥，他显然是不明白吴寿为什么要这么干。他打道："死吴寿，赶快向H.O.T道歉！""为什么？我不干！"吴寿回答到。我猜那天吴寿是吃了豹子胆，要不就是他神经错乱了。哪有拿这种事开玩笑的？

　　结果就是杨蕾蕾对我们说："刘超，我恨你！"之后就离开聊天室了。刘超对着屏幕发呆，他不敢相信今天的"约会"会是这个样子收场。你们猜他会干什么？当然是找到罪魁祸首吴寿并揍他一顿了。"他在哪儿？""一般是信息城。""走，我们一块儿找他。""好。哎——不行。（看了看表）都7：30了，再不回去我妈真"杀"了我了。""那我自己去。"刘超握紧了拳头，直奔电子信息城去了。

　　没想到第二天确是另一番景象。

　　"喂，昭阳？"

　　"是我，刘超？"

　　"是，告诉你个好消息。"

　　"关于我的？"

　　"不是，关于我的。"

　　"看把你乐的。肯定是你和杨蕾蕾——（话没说完）"

　　"对，是我跟她之间的。"

　　"你们俩好了？"

　　"不简单，这都让你猜中了。"

　　"要不你怎么能这么高兴？"

　　"对，现在我确实是非常非常高兴。"

　　"到底怎么回事？"

　　"她给我打过电话，我都向她解释了。还有吴寿这家伙也给他打过电话说都是他一手造成的。"

　　"太好了。对了！你昨天晚上找着他了？"

　　"何止找着他？我还给他来了个'天马流行拳'！"

　　"他怎么说？"

　　"他说他确实不知道我和H.O.T的关系。"

　　"看样是不知道，要知道他有哪个胆子吗？"

　　"就是嘛！别管他了。你不知道杨蕾蕾的声音有多甜啊！"

　　"是吗？南方人都这样。"

　　"真的是非常好听……"

　　就这样，一波在网上掀起的大浪在电话中平息了，而且是杨蕾蕾与刘超的关系更加牢靠，更加亲密了。我真为他们感到高兴。

　　从那以后，刘超与杨蕾蕾的接触更加频繁。他们想尽一切办法来联系对方，刘超的BP上动不动会有杨蕾蕾的电话，信件也如同雪花般的飘来飘去。他们好象失散多年的亲人一样不放过任何一个可以拥抱的时间。那时侯，他们的爱真让人羡慕，虽然有点近似疯狂。肯定是刘超开的口，有一天杨蕾蕾把自己的一张照片寄了过来，照片的背景是杭州西湖，他亭亭玉立的站在湖边，她不算很漂亮，但很秀气。笑的很甜，给人一种大家闺秀的感觉。看过照片后的刘超，对杨蕾蕾的长相可以说是非常满意，用他自己的话说就是"比我想象中的要好"。而在这之前刘超也已经给杨蕾蕾寄过一张照片了，据说上面的他很COOL，不知道杨蕾蕾是否像刘超这样满意对方，别忘了刘超当时可是对杨蕾蕾说他长的比古天乐强点有限啊！（再让他吹牛）那他们到底能不能走道一起呢？——请看下集。（磕倒了）

　　其实刘超长的并不是很帅，不过他躯干结实，块头十足有所谓的"安全感"加上穿着时髦，所以很受女孩子的喜爱。而且他号称"球王"这称号对他来说一点也不过分，他好象除了玻璃球以外，别的球像什么篮球，足球，排球，网球，乒乓球，台球等都玩的很棒，尤其是篮球，在我们学校可以说是一等一的高手吧。（眼看就快赶上我了） ^_^ 其实他最大的优点就是人很善良也很丈义，对朋友如对兄弟一样。（这点跟我一样） 综合以上各种优点（缺点咱就不提了）他成了名副其实的万人迷，而迷他的全是女生。（废话！）

　　除了杨蕾蕾，刘超在网上还结识了不少女孩子光青岛就有2位，先前也跟大家说过，不过只是一笔带过。现在就来详细介绍一下这两位。

　　一个叫孙丽，网名叫雾霎霎。她和刘超相识在[避暑山庄]，不对，应该说是中新商厦4楼的网吧才对。因为他们花了不足20分钟就从网络走到了现实。（大家鼓掌，祝贺他们"友情速递"成功。）那是一个星期六的下午，我和刘超放学后便来到了已挤得密不透风的那间只有十几台电脑的小网吧。没办法我们只好在一边等着有人下机。突然我们看到了同校的两个同学也在这儿，于是告诉他们下机后就把机器让给我们，他们满口答应说："我们这就快了。"这时我们看到有两个女孩也在等电脑，其中一位长发披脸，摆出一副很COOL的样子坐在一边，也许是我一向冲动，乱下结论吧我即对刘超说："MM！"他却不以为然，说道："什么啊？"看样他要求瞒高的嘛。我们最后实在等不下去了，那里面又热又闷谁能受的了？于是刘超提议上别的楼层逛逛，我同意。但没想到当我们回来时先前说好把机器让给我们的两个同学以走了，坐在他们位子上的正是那两个女生。我埋怨刘超："看，都是你！"刘超："早知道就不逛了。""接着等吧。"没办法，这次我们只有乖乖的等了。正巧，我们坐的位置再那个女孩的正后方，于是毫不费力的看到了他的网名——雾霎霎。接下来的事猜都能猜出来了，我们好不容易上了机，然后"跑"到[避暑山庄]，装做不认识的跟这位"雾霎霎"聊了起来。没想到她的口调很硬，活象个男人婆似的，不过冲着她的美貌，我们就给他三分面子，继续与她聊天。不过她很聪明，最后竟然猜出我们就在网吧。不对，这句话应该说成"猜出刘超在网吧"才对。因为在我跟她聊了有半个小时之后，她的一句"是你吧？红衣服。"（刘超身穿红耐克上衣，坐在我的旁边）我不禁问自己："是我太没有魅力，还是刘超太有魅力？"她明明知道是我打的，也许他以为那些话是刘超发自内心说的，而正是这些话让她心花怒放，我真后悔对她说那些话，其实就是些"你很漂亮"之类的话。但结果怎么样，他和刘超结识了，在我的掺和下结识了，我真佩服自己怎么老是替人做媒。（一般，一般）但更加难忘的是我和她第二次在网上聊天，世界真是太奇妙了！那一次，我们好象互换了角色又稍微改动了一下剧情。口调生硬的我竟然被比第一次温柔十倍的她（也许是刘超就坐在她身边的缘故吧）说成"自卑"以及"心理不正常"，我随即打下"对，我是自卑，但我更自傲！"那天也是星期六，因为在学校忘了因为什么事了憋了一肚子火，想到网吧发泄发泄，正巧遇上她，所以我的口调可以说是十分生硬，十分消极，跟第一次比有天壤之别。我就是这样一个人，所以我现在的网名叫"仙水"（《幽幽白书》中的反面人物，有9重性格）于是自找没趣，没把人家骂了，反倒落了个"自卑"的名声。我当时真是气不打一处来，这简直是对我的人身攻击和人格的侮辱。我真后悔为什么要认识她。此时，也许你们不相信，我正在青岛东泰家世客的游戏厅写这些玩意儿。走过的人都觉得不可思议，其实我也是鼓足了勇气才这么做的，并不是想出风头，而是想挑战自己的心理，我有个癖好：愿意尝试一下自己打心里不敢做或不好意思作的事情，也就是战胜自己内心的恐惧和虚荣心。虽然耳边轰隆隆的，但丝毫没影响到我的思绪，我发现"一个人最大的敌人就是自己"这句话说的很对，当你不为外界所动时有自己的人生观与价值观时，我相信你已经迈向了成功的重要一步。

　　前卫这个词我不知是从哪蹦出来的？反正是近几年才流行的词。这个词的意思好象是形容人的打扮很新潮，很COOL。但在我眼中前卫就是虚伪，把自己打扮的跟别人两样，看似很酷很有性格其实都又蠢又虚伪，他们之所以打扮的跟人两样还不是因为想吸引别人的注意，渴望别人投来惊讶的目光的同时自己却把眼神投向另外一边，好象自己满不在乎，让别人以为："哇塞！好COOL啊！好有型啊！"型你个头啊！这只能说你太肤浅了，告诉你他们就好象一些涂着颜色的空壳的蛋，虽然外表华丽但里面却空空如囊。

　　我和雾霎霎应该是两个不同世界的人，不管是我们之间存在很深的误会或真像刘超所说的那样"有缘无份"，我都不管这些。但我容不得有人在背后说我坏话，而她竟然明目张胆的在网上给我灌水，她以为这是BBS啊！她以为她真的是MM，而我这只青蛙对她来说就不屑一顾？我确实是个小心眼儿，但只有小心眼儿的人才能当一个成功的领导者，我不愿当领导者，我只愿能像未来的中国首富丁磊那样作一个在别人眼里不可理喻却总用行动证明自我价值的英雄主义者。刘超在雾霎霎的眼里和我正相反，没交往多长日子，两人就称兄道妹了，现在刘超又认了个妹妹正是雾霎霎的同位。这难道有个人魅力问题？这也难怪，因为一般人是看不出我的魅力的，他不在我的外表，也不在我的大脑，而在我的内心深处。这种魅力只有两个人最懂，一个是丛芳，另一个是——我自己。@_@

　　刘超认识的第二个青岛女孩叫徐美静，网名很美，叫香雪。她是我们邻校九中的学生，而且是体育生。与她相遇并不是那么"无耻"的投看了网名才相识的，而是我先当了雷锋又当了媒婆。刘超遇到我，怎么老是交桃花运？认识她很有戏剧性，正是在我和孙丽的网上大战后，她和同学来到了网吧。这时间间隔了只有5分钟。下面让我详细说来给大家听听。
