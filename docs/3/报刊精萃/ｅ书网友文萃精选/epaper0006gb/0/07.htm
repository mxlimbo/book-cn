在深夜死去的我的爱人

阿煜

　　在深夜死去的我的爱人
　　若干年后一个阴雨绵绵灰暗乏味的下午
　　我的烟蒂飞舞着落在了我的裤子上
　　我的皮肤和我的心脏在熟悉的音乐中咝咝作响
　　他把手指伸进我的生殖器象在寻找儿时的玩具
　　他的脸在头发的深处不停的摇摆
　　他的眼睛溢满了可怕的欲望
　　他说你看你看你耳朵红了
　　你的乳房象热气腾腾的包子
　　他还说我爱你我要操你一辈子
　　我觉得很刺耳然后我信了
　　
　　没有人再赞美春天的杨柳再唱激情的歌
　　麻雀在路上慢吞吞的走
　　爱情象瘟疫象妈妈裙子上的塑料珠子
　　
　　我的爱人在深夜死去
　　他脸上的表情很无邪
　　他还是个孩子
　　他背对着我他在看夕阳
　　他的影子长长长长碰到了我的脚
　　我尖叫我哭泣我躲到了橱子后面
