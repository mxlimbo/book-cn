非 常 非 常

yemiaoer

"天涯遥望，每天通一个电话，说30分钟，剩下的时间用来想念，这样，多好。"
"那还有呢？"
"还有就是喜欢你。"
"那还有呢？"
"哦，非常非常。"
"哦？还有呢？"
"还有？没有了。"
"没有了？呵呵，小傻瓜。"

小柯很忙，总是海阔天空的奔来跑去。而我却只能固守着我的城市留在原地踏步，在等待他的日子中将思念慢慢凝聚成沙，再点点累积成塔。于是日渐苍白。

认识小柯是在网络。很现代的相遇。在icq上说声你好，都好，便开始熟络起来。从不曾见过面，只是通电话或网上聊天，发发e-mail。想想起来现在的爱情真的很速食，这么的简单就可以心意相通。可以在一起多久，未来是怎么样，都已不再重要。甚至不必知道你的长相，身高，体重，以及其他零零碎碎的细节，只要开心，就是相爱，还说的那么理直气壮。
可是这些真的重要吗？

我是不管的，只要想你便好了，难道不该是如此的吗？

小柯经常会寄来一些各种各样的花果的果实。知名的不知名的，花花绿绿开满了整个阳台。坐在阳台浇花时，阳光暖暖的打在身上，有懒洋洋的快乐。

喜欢春光明媚的天空，流光溢动的带来漫天的光华。小柯会在这时拨一个电话来，听他的声音在电话那端温柔的低语，嗤嗤的呼气通过听筒传至耳里，呼呼做响，有痒痒的感觉。

在一家幼儿园里任教，教小孩子弹钢琴。五线谱象小蝌蚪在指尖游来荡去，孩子们总是快乐的围在身边听悠扬的音乐飞跃，或用脚尖在地上踩出属于他们的音符。喜欢孩子，他们的心灵就象雨后的彩虹一样七彩剔透。总是爱拉拉我的长发说老西老西好，然后推着我到处转圈。这就是他们表示喜欢你的方式，直接明了。

可是有的时候还是常常会莫名的担心。挂念外面世界的他，想可以去到那五彩的世界看个通透。可是，这一切不过只是想想，我早已被剥夺属于自己的资格。

便开始情绪低落。
天也阴阴的，叫人毫不痛快。
心里渐渐犯起点点的白毛。
小柯依旧天马海空。
依然收到各式各样的花果种子，和他的电话。
孩子们在身边快乐的游戏。美丽的小脸荡漾着幻彩的光环。
而我，依旧一天比一天天苍白。
……

电话轻响，接起。
"嘟嘟…嘟嘟…嘟嘟…"
是谁的恶作剧？
门上的风铃轻摇。
叮叮咚咚，叮叮咚咚。
老西，有人来了。
门外，光环的笼罩下是哪个健硕的身影？不曾见过的笑脸，和手里的那盆开的正欢的紫罗兰相映成辉。
小朋友聚在身边，审视的望着。
开始紧张，手心出汗，脸孔发热。
"走了那么多的旅程，我总是在想，或许你会愿意陪我一起去看。"
"哦？"
"天涯遥望，每天通一个电话，说30分钟，剩下的时间用来想念，这样，不好。"
"嘎？"
"我只想能有你可以陪着慢慢去看？"
"那还有呢？"
"还有就是喜欢你。"
"那还有呢？"
"爱你。"
"啊！"
"是的，非常非常。"

眼前慢慢升起的迷雾模糊了视线。孩子们的欢呼响彻云霄。车轮被缓缓向前推动，终于他的手拉着我的。温暖的体温瓦解心中所有的防线，我知道世界将从此不再苍白。
因为爱你。
非常非常。
