如果可以，让我们相爱

无情 （cappuccinoo@citiz.net）

　　阳光温柔地撒在身上，懒懒地，微风如细沙一样拂过他的脸庞，就像是情人的手。

　　他静静地躺在轮椅上，迷茫的看着广场上人群来来往往，心有些痛。

　　自从那一次车祸以后，他不得不接受事实，而事实往往是让人不愿相信的——医生告诉他，他这一辈子都可能站不起来了。

　　这两年来，他承受了别人所不能承受的，他的朋友们已经很少来看望他，他的妻子也远离他而去，他的父母因过度悲伤相继过世。每天他所能做的唯一一件事，就是躺在这个广场上，让阳光温柔地撒到他身上，让微风像情人的手一般抚摸着他.......

　　爱情，对他来说，永远是遥不可及的。

　　每天，都有无数对男男女女相拥着在他面前走过，每当这个时候，他就忍不住去摸自己的脚，他恨它，他打它，可惜他永远都感觉不到肉体上的痛楚，而内心的痛苦，自从他看到那个女孩以后，更加变得迟钝，因为只有迟钝才能让他感觉到痛苦来临前的平静.......

　　那个女孩，很像他的妻子，他是曾经那么的爱他的妻子，可惜，世事多变，有一句话说：这个世界上永远、唯一不变的，就是这个世界每天都在变。所以当她离开他的时候，他没有再看她一眼。

　　如今，看着那个女孩，他忽然有种陌生而又熟悉的感觉，也许那是离他太遥远了。

　　女孩有着丝一般的长发，很好看的眼睛，柔柔地，就像天上的星星，她的一双脚更是细长而均匀，让人羡慕。

　　可是这一切对他来说，都是那么的遥远。

　　每天，他都会看着那个女孩从他轮椅面前经过，然后渐渐地，女孩的背影从他视线里远去，终于消失在广场的尽头。可是他的眼光还依然停留在那个遥远的地方。

　　女孩没有发现过他，也许是因为他太藐小了，或者是因为他太平凡了，所以他有一种自卑，他希望他们就这样可以一直到老，就算那个女孩永远都感觉不到他的存在.......只要他可以。

　　可惜这种所谓的幸福生活没有持续多久。

　　有一天，女孩来的时候，身边多个一个男孩。男孩是很英俊的那种，有着高高的身材，清爽的头发，还有对女孩甜甜的笑脸。男孩很温柔揉着女孩的腰，陌生地从他轮椅前走过。

　　他忽然变得很迟钝，阳光温柔地撒在身上，懒懒地，微风如细沙一样拂过他的脸庞，他居然变得异常的平静，然而，他的眼睛却不敢去看那个女孩，因为他永远都容不下那个男孩。

　　他开始喝酒，不停的喝酒，不停的醉，在醉生梦死的世界里，女孩抱着他，他揉着她的腰.......

　　第二天醒来，阳光还是一样的温柔，微风吹过他有些晕的脑袋，他睁开眼睛，迷茫中他又模糊的看到那个英俊的男孩揉着女孩的腰，渐渐远去，终于消失在广场的尽头.......

　　每天，他还会静静地躺在轮椅上，迷茫的看着人群来来往往，可惜，他再也没有看到那个女孩。

　　这种迟钝地生活过了很久以后，有一天，那个女孩终于又出现了。 她来的时候，是坐在一张轮椅上，和他一样的轮椅。那个英俊的男孩推着她，熟悉地从他面前走过。

　　他忽然又变得很迟钝，也许他是真的反应不过来，阳光温柔地撒在身上，懒懒地，微风如细沙一样拂过他的脸庞，他居然还是那么平静，然而，他的眼睛却看着那个和他一样躺在轮椅上的女孩，因为他的视线里已经没有了那个推着轮椅的男孩。

　　他有些迟钝的微笑，也许微笑对他来说，都已经变得陌生。他突然想起了他的妻子........

　　阳光温柔地撒在身上，懒懒地，微风如细沙一样拂过女孩忧愁的脸庞，就像是情人的手。

　　她静静地躺在轮椅上，迷茫的看着广场上人群来来往往，心有些痛。

　　自从那一次车祸以后，女孩不得不接受事实，而事实往往是让人不愿相信的——医生告诉她，她这一辈子都可能站不起来了。

　　英俊的男孩陪着她，眉头有些皱起，他握着女孩的手，眼睛却看着远方，远处，变得弥漫，一眼望不到尽头。

　　这一个月来，女孩每天都来这个广场，她喜欢阳光的温柔，微风的感觉，因为今天来陪她的是一个老婆婆，女孩知道，不只是今天而已，也许以后都不可能再见到男孩了。

　　女孩忽然变得有些迟钝，可是她很平静，阳光温柔地撒在身上，懒懒地，微风如细沙一样拂过她忧愁的脸庞，她漠然的抬头，看到了一个和她一样坐在轮椅上的男人，他正在温柔的看着她，女孩觉得这种眼光真的好熟悉，好象他们已经认识很久似的.......女孩笑了，男人也跟着笑了.......

　　如果可以，让他们相爱。

　　《尾声》

　　昏暗的灯光下，他静静地躺在轮椅上，迷茫的看着屋子里的另一个男人。

　　于是他开口。

　　“你做的很不错，这是给你的酬劳。”

　　一堆现金摆在桌子上，在灯光的照耀下，显得异常的耀眼。

　　男人笑了，“你放心吧，肯定没有人知道，别人都以为这只是一场交通事故而已，哈哈.......”
