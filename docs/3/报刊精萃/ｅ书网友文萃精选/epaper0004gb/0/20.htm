广场的经典性与商业空间――关于城市文化之五

李华新

　　山东济南的第５４期<<齐鲁周刊>>话题新闻中有一篇谈到了泉城广场的商用价值问题的文章，其中对于城市广场的商用价值及其商用价值的开发提出了很好的建议，这是让人感到兴奋的事情，也可以感到当我们的城市在出现了“真正意义”的广场之后，人们就已经把关注的目光给予了作为城市文化载体的广场，但是城市广场所要承载的是文化的功能还是商用的功能，这就自然引起了各方的注意，作为注意力经济的一种发展趋向，城市经济的发展一定要和城市的未来发展有最直接的联系，但是作为城市的广场，它一定要走商业发展的路子，或是一定要体现出它的商用价值来，才能说是城市广场的充分利用吗？或是说一定要最充分地利用了它的商业价值或是体现除了商业利益来，城市的公用设施的建设才是最合理的开发吗？商用价值的开发，商用价值的迅速升高与膨胀，才是城市广场的唯一任务吗？当然这里不想偷偷地换掉这样的“发展商用价值”的概念，而是觉得我们的城市所缺少的一定不是商业的氛围，而是广场的氛围和文化的氛围，城市文化的发展与走向，包括我们现在所要充分利用和开发的泉文化，一定不是绝对地没有商用性的，但是至少它不包括城市文化的商用命题。

　　于是，笔者想到了城市的文化经典和商业的空间问题，在这里提出以引起更多的人对城市的发展及其未来趋向的注意，并有一个更清醒的认识。

　　显然作者提出这一广场商用价值的说法，可以说是经过了自己的思考的，但是不得不指出的是，用这样的说法来解释城市广场其实已经把城市广场的文化功能和作为城市文化载体的作用大大弱化了。城市广场的使命和作用，所具有的商业空间不应该用这种方式来体现。广场本身的作用不应该也不可能用庸俗的方式来完成都市的人文责任。

　　因此从这一点出发，城市广场的商业空间的命题就显得有些牵强或说是不妥。

　　首先城市广场的建立不是城市的商业流通的集散或是城市消费的物流渠道。我们现的城市为什么有太多的商业情景而绝少文化的气息，就是因为一个时期以来都市的功能区域的划分是随意和总体规划的滞后，或是规划之后在执行、实施过程不能以城市的文化情景为标志，并围绕这一标志诠释城市文化现象的。当我们所面对的是一个充满了文化的韵味和泉城泉水特色的广场时，我们能说我们现在广场的时空交叉和城市文明精神的载体，是因为有了商业理念而形成的吗？或是说我们城市的广场在城市人已经对于济南的城市经典缺乏必要的信念和不能移位的历史感时，我们还要把城市广场的商业气氛非要再搞得足一些才行吗？

　　其次，我们还不能简单的用商用价值来为广场定位，即使我们现在所建的这些广场即使没有一处体现出商用价值，我们也不能随意地把所谓的商用价值抬高到一个不应有的位置上来。当然，和一些人所设想的搞一些旅游纪念品的设想想比，这些还是有一些可取之处的，问题是当我们把广场的商用价值放在一个必须考虑的位置时，我们会发现这是一个错误。我们的城市已经有不少游离于城市文化的东西，而且这样的东西已经被证明是败笔了。我们还有什么理由再重新心提起这个话题呢？

　　第三，现在人们在议论城市败笔的时候，几乎无一例外的把城市对于古典的破坏斥之为败笔，而且也历述了败笔产生的不少原因，其中一个最直接的和最要命的因素就是城市当代人对于城市情感的寄托没有了，城市建筑体现了城市文化的表象，城市文化的这种表象，所能给予城市的几乎是凝固的城市文化的一部分，而不会是它的全部，它的全部是什么呢？是藏匿于城市建筑之中的那种不同风格的体现。因为当我的城市刚刚想把城市广场作为一种载体悉心打理的时候，商人们已经在用商业的眼光来看待城市仅存的一种空阔。就不能不说是一种城市商人的眼光了。当然，对于这样的眼光，我们所能做的和应该做的就是拒绝。

　　不提倡广场的商用价值，好像让一些商家感到不舒服，或是认为我们是在阻碍城市商机所能够给予城市百姓的某种好处，可是我们能否静下心来想想，当我们的城市到处都是一种商业的世俗或是商界的呐喊时，我们的城市不是很悲哀很无奈吗？我们现在城市的不快或是因为建设所存留的遗憾难道还少吗？难道我们还应继续这种商业的炒作，而使得我们的城市人或是慕名而来的外地人，再来见识一下千篇一律的城市建筑它所依附的商业气氛吗？

　　广场作为一种文化的所在或是说广场就是广场，它所拒绝的现在就应该拒绝，不要让一种游离于文化理念之外的东西来左右广场的存在和它的发展，如果我们不能下这个决心，那么我们的城市不是会很惭愧的吗？

　　说实在的城市文化是体现在城市的建筑和人文景观与风景区的，因为只有在这些幽静和适宜于保留文化痕迹的地方，文化的存量才能大一些完整一些，当初我们建设广场的初衷是什么，不是会提供能够在今后作为某种标牌的商标的东西，而即使是用做了商标，也是因为其中有文化的含量。不然景阳岗的“武松”或是景芝酒厂的“武松”，也不会发生纠葛；同样如果我们不是有了舜帝耕耘历山的传说，我们也就不会来为构建大舜文化而不遗余力。而现在当我们的城市稍微有了些文化的气氛时，我们同样会说现在我们的城市所缺少的不是商业的操作或说是炒作，而是文化的含量还不够。再说我们对于一些城市商品或是旅游商品的出现也好，或是什么人为的制作也好，它毕竟不是文化城市的部分，因为这些都不是城市经典的所在。既然不是城市的经典那么它的存在形态就不是我们所要求的，因为一个广场的缘故，就会不成为什么城市经济的增长点，因为城市经济的发展它所要依靠的还是城市的综合实力，城市广场它的存在先是文化效益或是说历史文化效应中所体现出来的或是由此而带来的旅游效益，而不会是它本身的效益体现。

　　热情的城市商人或说是城市商界从广场的出现和它的空间存在，能够立即联想到广场商场的出现，并且还以一些品牌的意识以及相应的产业出现搭建某种构架的心情是可以考虑的，但是我们即使是一个最普通的市民我们也会向我们的市长一样说，我们投入巨资所希望的就是能够使都市人拥有一块让心灵徜佯的空间，当然不会是希望出现某种熙熙攘攘的商业效果。

　　没有足够的理由说服别人，但是有一个例子，在我们庆祝特区成立２０周年的一个记录片中，笔者注意到了这样的一个场景，在珠海特区有一个和我们济南显著不一样的地方，就是在它的市区公交车上是绝对没有车体广告的，因为一个拉着广告满城市跑的地方，它能够出现静谧与幽雅吗？当一个城市拒绝了商业的躁杂或是地摊般的喧嚣后，这样的城市或许是最适宜于人们居住。因为人们在对城市的美进行评价时，城市的宁静其实就是一个相当重要的标志，充满噪音的城市不是美丽的，同样一个充满喧嚣的城市广场也不能说是美丽的，“一座美的城市应该是紧张而又宁静的”。从这个意义上来看待商家的良苦，或是说用这个尺度来分析这种构想至少是多了些商业的味道而少了些城市的宁静之美吧。

　　另外既然笔者谈到了城市的经典和城市广场的经典问题，其实对于这个问题即使是现在也有不少人存在这一个误区应指出的是城市的经典是有具备一定的时间跨度，并被历史的时空所检验或是说被人们所普遍认可的一种文化或是建筑文化、民俗文化、历史人文文化等等，而我们这里既然是指城市而言的，就应该是充分体现了这座城市的人文精髓的东西，经典不是主观臆造的或是刻意创造出来的。而且从这一理论来看待城市广场它至少目前还不能承担或还没有承担过这样的重任，城市广场可以经典但是不是现在。如果我们现在赋予了它太多的商业的任务，那么它就更不会成为经典了。
