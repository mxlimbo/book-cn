（十三）

　　等了很久，她还没反应。

　　——不会吧，这么冷漠。

　　"我爱你！"这次我更加大声，旁边一个女孩用迷惑的眼光看着我。

　　——看看看，没见过帅哥对美女说我爱你吗？孤陋寡闻！！！

　　不过小洁还是没有反应，我急了，转过脸——

　　然后我就看到她软软地躺在椅子上，枯黄的脸上还残留着几滴冷汗…….

（十五）

　　医生说了些什么我几乎都听不懂，也听不进去，我唯一留在脑海里的是医生说了这么一句话："唉，这种病，在国外有过几例，国内很少的，不过上个月这儿有一个孩子也是……"

　　我呆住！

　　可不可以让一个人有两次生命呢？你说，不，只要有一次就已足够了，因为人世间除了生死离别还有别的大多太多痛苦。

　　可是你知道吗，我情愿去承受别的痛苦也不要和你生死离别，然而为什么会是这样？为什么？难道说是……我不由想起那天上课时——

　　"那个八戒竟然紧挨着宋小洁，一副小鸟依人的狗样，气的我当时心跳一下子超过了100下，那绝对不是紧张，是愤怒。而且让我想去撞玻璃窗的行为还真的出现了，宋小洁她，她竟然去主动拉那个八戒的手，不会吧，你不是存心和我过不去，我要死啦！！！"

　　——会不会是感染？那天医生也说这个病是会感染的，而那个时候他们根本就不知道，而她弟弟就已经有病了，要不是我去揙他，他还不会去看医生呢……

　　同样的医院，同样的14层楼，同样的514房，不同样的人。

　　"你不能进去，医院规定的！"那个医生拦住我。

　　"让我进去……让我进去！！！"我大声吼叫。

　　"不能进！"

　　"我再说一遍——让-我-进-去！！！"我的脑海一片火热，几乎快要丧失理智。

　　那个医生还想拦住我，我一把推开他，冲进了房间。

　　"快，快去叫院长，快！"身后传来急切的幺喝声。

　　房间里很安静，同样的桌子椅子，同样的床，同样的药味。

　　我看着她，眼泪如泉水般涌了出来。

　　"坐！"她像我们第一次来看她弟弟一样的礼貌。

　　可是我怎么也控制不了我的泪水往外溢出。

　　"谢谢你来看我……"她说这句话的时候还是那么平静。

　　我忽然觉得我们之间有种陌生的感觉，就像第一次看见她一样。

　　"不要哭了，乖，想想以后……"她说话时的表情有些僵硬。

　　以后？我们还有以后吗？

　　我坐在她的身旁，握起她的手，她的手很冷，就像我的心一样。

　　"放心，我不会走的，阿泰生病时他的脸上有红肿，而我没有，所以，我不会和他一样离开你的……"

　　听了这句话我的心马上被撕成了千千万万片，你知道吗？阿泰脸上的伤痕是我造成的，是我叫人打他的，都是我的错……这一切我没有说出口，我不是不敢说，是因为这些秘密起码还能给她以生存的一丝希望。

　　"不要再哭了……至少……我还没死呀……呵呵。"她苦笑。

　　——原来，一个人还没死是那么快乐的事。

　　可是听了这些话，我却怎么也快乐不起来，泪水反而越来越多。

　　"你快走吧，医生马上就要来了。"

　　——远处，一阵急促的脚步声。

　　我无奈的看着她，又看看了门外，然后起身，把脸贴着她，轻声道："记住，我爱你！"

　　说完这句话一滴泪从她眼里落了下来，我忽然不想看到她落泪的样子，转身，痛苦的走了出去……

　　"阿料"身后传来她呼唤我的名字。

　　我不敢转身，默默道："还有……什么事？"

　　"如果我有翅膀，我要从天堂里飞下来看你。"

　　心一下子就被撕裂，泪水又如洪水般冲了出来。

（完结篇）

　　学校里又在举行"献一份爱心，得一份爱心"的捐款活动。

　　每天我都会去看她，虽然我不能进去，但我相信她会感觉到我的存在，还有我的爱。

　　有一天，我在房门外徘徊的时候，有个医生过来对我说："你是阿料吧？"

　　我点点头。

　　然后他给了我一张小洁写的纸条，上面写着一行字：去告诉我的爷爷奶奶，地址*****。

　　我握着那张纸，手在颤抖，我甚至可以感觉到两个老人痛苦欲绝的心情。

　　考完试后，我就按照纸条上地址去找小洁的爷爷奶奶，我花了一天一夜终于来到了一个偏僻的山村，可是很可惜，没有找到他们，等待我的只是一间空荡荡的平房。

　　后来村子里的人告诉我，房子里的两个老人在几个月前就已经死了，男的先死，女的因痛苦万分而在一个星期后也相继死去，只是女的在死之前告诉她的邻居不要把他们的消息传给远在城市读书的两个姐弟，因为他们不想让姐弟俩伤心，他们希望姐弟俩好好读书，以后大有出息。

　　听到这里，我忽然明白，你说人只要有一次生命就已足够了，因为人世间除了生死离别还有别的大多太多痛苦。

　　的确，我很痛苦！

　　回到医院，我真的不知道该怎么对她说才好，也许，她活着就是为了等我的回答。

　　我很无奈的乘上电梯，上了14层楼，来到514号房，却看见房门大开着。

　　医生告诉我，就在昨天，这里的一个病人被火化了……听说走的时候，非要医生在她身上藏一只蝴蝶，有翅膀的那种。

　　我忽然又有了一种冲动——哭的冲动！

　　走出那家医院，我回头又看了一眼顶上的那几个红色大字：爱心医院。

　　今天，阳光真的很灿烂，照在我的脸上，一片枯黄……

《尾声》

　　几天后，阿荟陪着我去医院查身体，来到医院，我有种熟悉又陌生的感觉。

　　看着医生，我问："医生，怎么样，有没有异常？"

　　医生抬起头，也看着我，然后说："……"
