另一只眼睛看“孝”

唐君虞 （Anonymities@sohu.com）

　　无意间又忽然想起那久未露面的字“孝”。作为中华民族的传统美德之一，它被千百年中华历史所提倡至今任不褪色分毫。。视孝为为人处世基本品质之一的中华民族对于这个字有着千丝万缕、难以割舍的感情。

　　中华民族的孝是建立于五千年封建史之上的。但正缘于此因，却使这个“孝”字发生了些不经意的变化。而这些微妙的变化却成为了一出出本可避免的悲剧的原由。

　　从古至今，由父母之因而误国误民者屡有所见，因父母之命而贻误终身者更不胜枚举。究其本原，竟皆处出于这个“孝”字！难不成这“孝”真出了什么问题了吗？其实不然。

　　曾经三千年的帝王史在华夏儿女的心中铭刻下了一个难以磨灭的烙印棗服从，无条件地服从。作为封建史的象征之一，其早已“深入人心”。从对于国家地服从继而转变为对于长辈地服从，而后又自然而然地融入了“孝”字之中，并将其视为“美德”之一。

　　曾经的农业大国也给中华民族留下了一个深深的印记棗小农思想。贪图个人利益而无视他人及集体；为了自我眼前的安逸而不顾明天。在急功近利与个人利益至上的名义下，曾经有多少中国人以“孝”为掩干了愧对下一代的事！

　　回眸曾经走过的路，在历史车轮下辗过的被充作政治工具的儿女的亡魂有多多少少，在历史长河中由那变了味的“孝”字而至国破家亡、民不聊生的故事岂止一二？而面对此情此景，一旁冷眼看戏的戮民却依旧愚蠢至极地高呼：“那是多么伟大的一种品质啊！”伟大？何以伟大？是轻身而昧大意之伟大？是为了昨天而忘却明天的伟大？是为了没有理由的理由而否认自我表现价值的伟大？还是延续这样一种“传统”的伟大？何以所言，何以所对！

　　但我们依旧存在于这个世界之上，好好地存在于这个世界之上，而且将继续存在于这个世界之上。因此我们没有理由否认现状。既然占世界五分之一人口的文明古国也被这样欺骗了几千年并无大碍，那在被蒙骗数个千年又有何妨？
