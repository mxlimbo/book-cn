情感的月亮

李华新

　　我们现在还不能对中国传统诗歌中的运用了多少次月亮作一个详细的统计，但是，我们可以明显的感受到中国的月亮，在传统文人笔下的作用，是深蕴的是有着种种心灵的慰籍的。月亮是离我们最近的一个星体，可以用来寄托我们的许多飞天的想象和逃脱的情怀。我们所有的人生的幸与不幸仿佛都可以通过对月亮描写的比兴来达到这一目的。而现在我们所能认识和接触到的则是月亮对于人的情感所能产生的缠绵的思念和情感浓缩和放射的张力。

　　“月上柳稍头，人约黄昏后”。这是我们再熟悉不过的情景了，这也往往是对恋人们幽会时的一种心情表露和场景的向往；而李煜的“花前月明笼清雾，今霄好向郎边去”则把一个美丽而又神秘的夜晚之时一位女人等待的心情，融入了心理的活动。而晏几道的<<南乡子——新月又如眉>>的词中则把黄昏新月时女主人公的愁思微妙的表达了出来。因为紧接着的下一句“长笛谁交月下吹”，把那种即使是吹笛也枉然的心情表达了出来。这里的月亮是什么呢？显然月亮在这里所要表达的是与人的心理活动相关的一种内在的心情表露。随意地翻翻唐宋词，类似这样的心情表达非常之多在，只不过由于心情的不同让月亮所承担的比兴“任务”不一样罢了。五代词人牛希济在<<生查子——新月曲如眉>>中以比喻的手法、明快的语言、开朗的意境，表现了相思女子的苦恋，热望同所爱的人早成佳侣的深挚情感，感情质朴清新。而在第二句中就把在新月还没有形成圆形时，也就隐约的把恨新月不圆的感情表达了出来，含蓄而又双关。苏轼的<<水调歌头——明月几时有>>，是一首中秋词作，抒发的是作者外放无俚的茕独情怀。明月几时有？把酒问青天。不知天上宫阙，今夕是何年。这是一篇咏月的词，全篇月是中心的形像，但是却处关和着人情时事，和自然与社会有着密切的关系。词的上片借明月以抒发自己的清高和遁世的情怀，而在词的下片衬托的也是离别的思绪。

　　这种以月来抒发个人的情感，或是在咏月时搀杂上自己的对时事炎凉感触的不在少数。月的凄楚和月的欢欣这好像是古代的圆月的一种即定的比兴手法，月便自然成为写作的是一种，承揽了离别愁绪；同时月圆又多半是一种团聚之情，而一些格调高远的词人，则能够把一种激越的情绪表达出来，就显得有些不同凡响了，这一点李清照的词中就表现了不少。“为报今年春色好，花光月影宜相照。”据说这一词的意境和唐代诗人刘禹锡的“淮水东边旧时月，夜深还过女墙来”一样在特定的时局面前抒发的是一种沉郁苍凉的感觉，而所有的这些万千感慨，又都称为词人笔人的月之形像。当然，人们大多习惯的是晓风残月、花好月圆这种完全不同的月亮的比兴手法。人们的心情其实就是对月亮的驱使和比兴的所需，其实月亮本身的变化是一定的。它也不会对人间的喜怒优思有什么不同的影像，它就像我们在所学的那篇经典的英语课文那样，在宇宙飞船上我们所见多的蓝色的地球是脆弱的，如同是一枚上帝所创造的最美丽的弹子一样，而对于月球人们其实也已经认识到它是一个昼夜温差极大的星球，它不会有什么生长的桂树和什么玉兔的东西，而即使再寂寞的嫦娥，也是根本不可能存在的。但是，诗人们却把这一想象牢牢地给予了它。 其实，我们也知道月亮所承载的这些人间的情感，就是人本身的情感，月亮的存在不过是一种心情表象的再传达，或是可以寄托的一个都公认的物体一样，它是可以通过视觉而作用于人们的心理感触的，再加上它的可望而不可及的远离尘世的高洁和浩淼于如水一样的情景中，所以这种高悬的形像就只能用什么比兴、比喻等等文学的手法来达到，于是思念、沉思、相望就称为一种书怀的方式，也便让它有了许多寄托的表白用途。

　　当然现在的科学早就把月亮对人体的影响作了充分的说明，而至少那些思春的女子，或是沉浸于爱河的男女要受月亮的影响，这已经成了定论，至少月亮所形成的潮汐或是女子的月经等等，以及人们的情感随月亮的圆缺而对人们身体的影响，就已经说明了这一点，人们已经不持异议。所以我们在皓月之时的情绪会好一些，而在残月入钩时情绪会低落，包括人们的男女幽会之情也是在圆月十分会使女子更性感，会使男人更有激情的说法，就是在外国也是一样的，因为圆月之时人们的性激素的分泌会更多一些，这也已经为科学家所证实。

　　一轮圆月承载了多少人间悲欢离合的故事，一轮圆月让天下多少人在充满想象的场景中驻留，而所有的这一切都是人们所给予的，于是月亮就有了许许多多的情感寄托，也就寄托了万万千千的情感，并让这许多的情感和许多寄托活化与诗化。
