秦 淮 河

海平 （susu0289@sina.com）

　　不用脑筋真是一件轻松的事，可也是一件痛苦的事。我现在在听张震岳的CD，真是奇怪，只有在听CD的时候我才能写点东西，看书的时候可以翻书，看电视的时候可以玩打鸡蛋游戏，走路的时候手一摆一摆的，睡觉的时候不必动手，除了这三件事我想像不出平时我还能干什么。偶尔我也听听歌，听歌的时候看不下去书，多半也没有电视看，双手就觉得没处放，只能打字了，毕竟双手是需要活动的。我现在懒得我自己都难以自信，近两三年来我一直搞不清楚我身体里那个叫头脑的东西到底蜷缩到哪里睡大觉去了。没有头脑着实轻松，因为不必再寻找自我，没有头脑也着实痛苦，因为找不着自己毕竟烦恼，没有人习惯把自己只当一具躯壳存在的。我现在在听“干妹妹”，以前听这歌CD总是出毛病，现在竟然能完整地听完，真是奇迹。现在换成了阿牛，在许美静和阿牛之间我犹豫了一阵，到底还是选了阿牛，因为最近我迷上了傻乎乎的歌，我喜欢许美静，可是她有点故作深沉，我不是怪她，所有的歌都是这样，连张震岳也是这样，可是他怪声怪气的又冲淡了深沉的感觉，真是好玩。说到底我喜欢的还是声音和曲调，歌词是狗屎我也不在乎。这又是失去头脑的佐证之一。

　　我讨厌杨佩佩，恨她糟蹋金庸的小说，可是我老是记得她拍的《倚天屠龙记》最后张无忌对赵敏说的话，他说他一生最大的愿望不是要建功立业，只是希望有所小房子，房子前面有几棵树，最好再有一条小河。（他原话还要哆肉麻得多（杨佩佩的特点），我概括了一下）。我家门前也有一条不算大的河，不幸它叫秦淮河。我从小就有的愿望很多很多，其中一个就是门前有条河，想不到它竟快于其它想头率先实现了。所以我才很烦它，因为我心中所有诗情画意的摹想全被它破坏殆尽了。春天来的时候，岸边栽的垂杨柳弯着腰把叶子浸到水里，那时我上学每天都要路过那里，一年三百六十日至少可怜它们三百次，盼自己来生千万别做秦淮河边的柳树。每天我都看见有人端着痰盂哗的往河里倒的，当然也有倒剩菜倒垃圾的，这许许多多不相类的物事进入水中，散发出一种混合的味道，咸味。很浓烈，像炒瓜子的味道。所以我一直认为这就是秦淮河的气味，后来我走到河那头才知道，原来真有一家炒瓜子的小摊子。虽然如此，以后不管走到哪里，只要见到秦淮河，不管是哪个地段的，是长干桥旁的还是文德桥旁的，鼻子里好像总闻到一丝淡淡的咸味，就像炒瓜子的味道。不过有点恶心。我家那边的秦淮河边还开了一家印染厂，于是河水从此变得多彩多姿，有时是纯蓝的，有时是墨绿的，有时又有点发红。有一阵我望着蓝蓝的河水发怔，我不是一直希望有一条蓝色的小河的吗？怎么会是这样？我要的小河，是波光粼粼的，阳光照射的时候，它要闪着金光，让人无法逼视。有月亮的晚上，水面要镀上一片银光，天好的最好可以看见游鱼，也不要一清见底，要深邃一点，显得有内涵。而不是像一块粗布一样，对光没有感应，对季节没有感应，对污物没有反应，一味的溶和，溶和，溶和的都有气味了。

　　有一天，我记得那年我上小学五年级，因为那天我是穿那条小学时才穿的黄色背带裙出去的，我出去的原因是因为那天傍晚我趴在我家的阳台上居然远远看见秦淮河在夕阳照耀下闪光！于是我换了衣服就出去了，走到河岸边，才知道刚才不是幻觉，河水真的在闪亮，是那种清粼粼的河水才有的亮泽，因为是傍晚，那天还有晚霞，所以河水也有点发红，不是那种死板的红，是那种会跳跃，会抖动，会发亮的，流光溢彩的红色。衬在幽绿的发清的河水里，河水不再浑浊，也不是一望到底，正如我想像的那样恰到好处，既深邃又清彻。让人感觉像是在做梦。那天我沿着河岸走了很久，具体在想什么已经记不得了，只知道当时认为幼稚园时的梦想已经实现，生命里第一次出现了一种叫诗情画意的东西。直到――

　　我在河岸边散步的时候，听见有人好像在叫我，我侧头一看，是个大胡子，我那天心情好，看谁都有点儿似曾相识，那个大胡子有点像小时候住的大杂院里那个爱罚儿子跪搓衣板的人。我刚想跟他打招呼的时候他却板着脸问我“怎么搞的，跨过人家放在地上的茶缸，这是能随便跨的呀！”我一低头才发现，地上果真有一个茶杯，我刚才没留意。再抬头发现那个人还虎着脸瞪着我，我当时可能是说了一句对不起就逃开了。

　　我接着散步，可是好像被人从诗情画意的梦里敲醒一样。我满怀善意地去要和人家说话，却被训了一顿，不能不使我感到不痛快。正在这时，透过晚霞反射出模糊的金光，透过不太清晰的暮晖，我看见了一个人。在对岸。是个胖子，看不清是男是女。穿着一件汗衫，晃悠悠地走着，手里拿着一个痰盂。我怀着一种近似恐惧的感觉看着那个人，看着他走近河边，看着他爬上栏杆，看着他把痰盂向着那么清的河翻转过来。至此我的情绪被破坏无疑了。

　　后来我常常想，为什么我会恨那个黄昏。那个发现家门前的清彻小河黄昏。在那个黄昏里，我人生第一次接触到的那种细腻敏感，带点儿忧伤，也许还有点浪漫的情怀被摧毁殆尽，至使以后每次心头泛起类似感觉时，总是想起那个黄昏，那样的晚霞，还有那个大胡子，还有那个看不清面目拿着痰盂的人。我人生第一次开始迷离模糊地企盼和猜想未来，就像想看透那天天边闪烁的夕光和缤纷的晚霞，却没有看见脚下的杯子，还被人骂没有礼貌。

　　从此我也开始恨秦淮河。每次我对它抱有希望的时候它最后总是让我失望。它那样逆来顺受，对命运不知是侮辱还是挑逗没有半点反应。让我搞不清我恨的到底是它还是我自己。我恨那些把我脑袋搞混的东西。

　　再后来就到了某一天，是白天，我和爸爸可能还有我妈，还有什么人我记不得了，我们去了夫子庙，路过文德桥，秦淮河照例在脚下凝固着，不时有船桨捣咕进去，泛起几缕纹漪。有几条船在河里划过，上面像是游客。我不知道他们怎么受得了，我又闻到那种咸味，有点儿像炒瓜子的咸味，却让我恶心。也许是我敏感，因为船上的人看起来都很高兴。这时我爸爸说了一句话，彻底让我对这条河的感觉定了性。他说：“这些人真好玩，居然在阴沟里划船。”至此我对它完全死心了，也对我自己死了心，原来这么多年在我家门口的不是小河，是条阴沟。从小我就希望家门前有条河，有这个愿望时，那时我家门口有口井，我常把小桶吊进去玩，盼望能吊条鱼上来。那时我多希望这是一条河啊。河里一定有鱼的。后来家门前真的有了一条河，我虽然觉得有点不对头，可它毕竟是条河呀。我有那么多的幻想，其实这是最美丽的一个。虽然它的实现不是我想的那样。可是它竟然是条阴沟！它算什么，我又算什么？一切全不对了，也许人的许多幻想都不过是肥皂泡，肥皂泡都会破灭，命运最后留给人只是惆怅，可能还有安宁。可是它居然给我留了一条阴沟。一条赶也赶不走的阴沟。它映照我自己还有我全部的理想。还有我最初对未来的憧憬，像我十一岁那年某一天看到的晚霞一样。

　　就是我爸爸说了关于秦淮河那天，也就是我对秦淮河以及我自己定性的那一天，那一天晚上，好像天上还有一轮残月，我们从一座房子里走出来，听自己的父母和自己的亲戚告别说亲热话，我背对他样，面对秦淮河的方向，我猛然发现原来白天和晚上的感觉完全不一样，不知是夜晚能麻痹人的嗅觉还是心。我不再闻到那种咸味，耳中却好像隐隐听到打更的声音，几艘观光船泊在河里，黑灯瞎火想是里面没人。夜色让它们变得黑黝黝的看不清楚，只是隐约看出是船，却也洗掉了它们白天张灯结彩的俗气。它们现在就是船，秦淮河上几条孤独的摇船，让我想起枫桥夜泊的情景。河面上笼罩了一层像青纱又像簿雾的东西，不知道是不是冬天的冷气。再也看不出它真实的表面，可以尽情幻想它清彻无比。我又想起碧血剑里袁承志和青青夜游秦淮河的情景，青青真是爱干净，歌女吹过的箫她要用手绢擦过了才肯接着吹。那时她一定不会闻到咸味。忽然我的眼眶里有一点湿润的感觉，就在那时我和这条河的宿怨仿佛已经被抛到一边，心里想起神雕侠侣结尾时那首歌（那时郭襄也哭了）：秋风清，秋月明，落叶聚还散。寒鸦栖复惊。相思相见知何日，此时此夜难为情。
