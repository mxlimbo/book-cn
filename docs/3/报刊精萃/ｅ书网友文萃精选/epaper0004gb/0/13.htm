清扫爱情的房间做心灵的家务

――读《此时此刻――寻找你和你心中的爱》
李华新

　　每个人的一生是不会不和异性发生交往关系的，由此，也就演化出许多的两性关系的话题。是的，没有什么能比两性关系使我们更加了解我们自身和我们自己的生活。爱情与夫妻关系和人际关系一样是无法驾御的吗？以爱的名义和以被爱的名义之后就不会使我们的爱情爱情生活出现危机，或是不被人所爱了吗？我不得不筛选或是舍弃对我自身或是与对爱情的，就像打扫房间，力图挑出那些灰尘蒙面或破旧不堪的东西，将不在需要的时候把它和属于感情的东西一起扔出屋外，爱情需要在巨大的混乱中突围。选择了Iyanla Vanzant这本书，我的本意是要从中发现一些我们和美国人在婚姻问题上不同的价值观的，但是在我没有对于这一问题开始涉入时，我首先是感到了这种比喻的新鲜。

　　“将为你的头脑和心灵的免疫消毒，为的是让你做好准备，拥有你和你想要的东西。”

　　我们要想什么呢？爱情从旅途开始。生活的全部就是爱。假如我们把爱比做一个房间的话，那么我们就是爱之屋的主人。而在这个时候当我们把爱作为快乐、健康和财富接触爱的时候，我们就会发现，我们就会把一种强烈的目标感带入两性关系之中。在书中她提到了一种爱情的自我感，我感到了不妥，因为爱情的东西，会使人的自我感流失。为了爱而进入一种两性的关系，然后男女开始伥惘。在这个时候，他就会被这个小小的信息片段所震颤不已，然后是一方求婚，一方接受。然后就会像所有的第一次发生两性关系的时候那样，每次都想接近婚姻的那种最后的辉煌。但是每次的两性关系都是相伴到永远的关系吗？这是远比我们现在一些有关这方面的书籍，阐释的如此明白的一本书。

　　因为在现实中我也曾经见到过这样的一个例子。只要是这一对很年轻的没有性经验的恋人在一起的时候，就必然要发生性关系的。好像她们的婚姻或是前婚姻期就是为了准备性欢愉而来的。为爱情的信息实现而来，或是为了缔结那个婚姻的情缘，我们的作者把这样的婚姻做了一个我们所不曾想到的例子，从自己家中的地下室开始打扫，清理爱情的房间。

　　也许这是我对这本书产生兴趣的一个原因。

　　相信大家也会有一种同感。

　　这本书完全不同于过去我们所读的一类，其实对于这样的书籍，我们并不非常熟悉，比如它说爱情的生发处是在地下室，这一点就和我们的生活习惯有所不同。她又说，两性关系的房间在春天到来的时候，是要清扫的。因为这是一个极好的季节和机会。不知道为什么作者对地下室有这样多的钟情。或许是他们的做爱最先就是在地下室完成的缘故，也许是在生活开始的时候他们是居住在地下室的，所以清扫是在地下室开始的。

　　在这里作者提出了不少富有哲理的解决两性关系问题的方案。在地下室这一章中作者说，爱的出现是一剂治病的良药。这种治疗的效果，是我们大家生来就要在自己的生活的方方面面去发现并体验。这话说的深刻。因为婚姻关系是无法避免性关系的，在这里所说的性关系，并不是我们唯性关系至上的提法，而是把一种夫妻关系的性提到了一个新的层面来进行分析。因为，感觉和情感是促使我们行动和发言的能量。感觉通常是掩藏在我们的两性关系中的问题之后的答案。因此在一种两性关系中，你有必要时刻不忘自己的感觉，以决定你行为的动机所在。在这里好像作者所截取的地下室是一种夫妻生活避免和外界接触的一个场景，而这个时候仅靠你自己的判断力好像不足以使爱情得以逃脱怅惘期。然后作者选取了不同的夫妻组合，有的在一楼，有的在二楼、三楼或是在顶楼，而要清除的东西也是很多的比如澡盆上的污垢、梳妆台、壁橱等等。作者所要人们达到的就是让爱情到达爱情的顶楼然后进入甜蜜的套房。这一切所要达到的就是让爱情在经历了怅惘期之后的唯一目标，让我们学会生活，让我们享用生活，让生活本身教会我们如何生活，然后在投身于对我们的自己和我们彼此的嘉年华会的同时，拥抱生活简单的真相。　　而这个时候你必须让自己沐浴在爱之中。你必须将爱当古龙香水使用。你必须用爱刷你的牙齿。你必须通过你的思想、语言和行为，将爱倾注进万事万物之中，倾注在芸芸众生身上。　　在这本书的封面折页上有这样的一段话，不妨照录：每一个活的生命都渴望体验到爱之光，问题是我们的窗子太脏了！我们的心灵之窗被旧日的痛苦与伤口，旧日的回忆与失望所蒙蔽。这些窗子被恐惧、自我怀疑和错误的信息这类的阴云所积满，以至于爱之光已经不能透过这重重迷雾照射进来。在此时此刻，我们透过这扇雾窗，努力使自己我们所看到的就是真实的情形。实事却不是这样的，我们也明白这一点，但是我们看上去并不想要找出要做的事，除非真正的事情发生。我们必须要做的就是清理。我们必须打扫这些窗子、地板、墙壁、盥洗室以及我们心灵的角落。我们必须要把那些使我们失败、让我们迷惑，令我们此时此刻感到痛苦的灰尘清理掉。在这本书中，作者告诉我们怎样去做这个十足的心灵家务活儿，如果我们很好地这样做了，那种爱之光就会照耀进来。一旦这一切发生，我们的精神就会照耀在真爱与幸福的灿烂之光下。　　也许这些话我们今天听了不仅是新鲜的，我们还能感觉到其实我们的人生中，即使是爱情也是要不断更新的，要不断地把以往的产生误解或是迷惑统统的在一个早晨或是晚上清除出去。也许这种方法是使爱情更新的最好方式。　　正像这本书中写道的那样“我们将清扫内心你的房间，你在房间里保留了你所有对爱的无用的信仰。我们将为你的头脑和心灵免疫消毒，为的是让你做好准备，拥有你说你想要的东西”。

　　不知这样的表述伊雅拉.范森特的著作是否准确，但是， 我以为爱的寻觅和到来其情形几乎是一致的，这就是爱的表现相同性，而在这个时候以一种相同的形式走进你的内心世界――走进内心深处――那么亲爱的，我或你所需要的就是轻轻扣打心灵之门，在此时此刻。

　　此时此刻我的所有享用爱情生活的朋友们，你们想到了要清扫爱情的房间了吗？如果你不想这样做，那么我就 提供这样的一个机会，这是所有的男人和女人要干的一件事情，是一件家务活――相爱的人相亲的人儿，做做自己心灵的家务吧！ （<<此时此刻――寻找你和我心中的爱>> 昆仑出版社出版 2000年第一版）
