罗觅鸥和朱履夜（武侠版）

碎梦一刀

　　朱履夜一仰头，将杯中逃情酒一饮而尽。如果罗觅欧不能够在三天内找到苦大师，并带回大师所珍藏的那颗天香豆蔻给她服下，她将永远不能够再醒过来。可是她没有一丝的犹豫。

　　明天就是家里逼她嫁人的日子，为了防备逃婚，家里人把整个的庄院围了个水泄不通，虽然自幼修练的履夜神功已经到了第七重，可是想要不伤人而离开是绝不可能的事情。罗觅欧家传的觅鸥八式虽然是无上的轻功绝技，可是也只能做到自己出入而已。三个月来，他们想尽了办法，可是无一例外的以失败告终，时间一天天的接近，没有梁祝化蝶的本领，除了假死，她很能做些什么呢？

　　或许在逃情酒起作用之后，她将再也见不到窗外美丽的星空了。可是她知道她的心已经被罗觅欧所占满，怎能接受别的男人。如果没有爱，她宁愿死。脑海中忽然浮现出和罗觅鸥初识的那个晚上。

　　在泰山绝顶观看日出无疑是一件很惬意的事情，可是朱履夜却更喜欢泰山的夜晚。离明天清晨的日观峰决斗还有很长一段时间，朱履夜并不打算用这段时间来调整体力，她对于罗朱两家不断的争斗已经感到了厌倦。虽然这次她是作为长白朱家的代表迎战东海罗家，可是她对于这个并不在意。世界上有那么多美好的东西，为什么一定要为了一些虚名而争来斗去呢？

　　一直不理解，三十年前一直是世交的长白朱家和东海罗家，仅仅是因为一个“南罗北朱”还是“北朱南罗”的排名问题竟然会从此反目成仇。真的很难理解，想想都头疼。还是好好的欣赏一番夜色吧。在四周茫茫无边的漆黑之中，头顶的星空显得特别的接近，皎月如银，光辉照人，抬头时，那满天的星子似乎都可以随意的伸手采到。在蓝色和黑色交织的夜幕中，周围的一切都显得特别的静，在凄冷落寞之中的这份宁静是她最喜欢的感觉。她以为可以独自享受这份孤独的美丽直到天明。可是不经意之中，她却在不远处发现了另一个孤独的人影。

　　她想这个人应该就是东海罗家这次决斗的代表了，可是她没有理会。而对方也一直只是站着不动。

　　在日出之后，她便看到了他。

　　她的意识立刻告诉她他应该就是罗家年轻一代的第一高手罗觅鸥了。黑色是江湖中人穿的最多的颜色，本来是很普通的，可是穿在了他的身上却是那么的不同。她想如果面前有一万个人，她还是会一眼就从人丛中发现他，因为他就好象一条美丽的七彩神仙鱼游弋在众多的土鳖虾蟹之中。使人不可能忽视他的存在。山顶的疾风鼓动着他的黑衣猎猎作响，给人一种剽悍的感觉，可是他的人却是清秀儒雅的那一类。除了双目间的镇定从容，在他身上简直找不到一分的武人之态。

　　不用看他出手，她就知道他是一个难以对付的对手。武功一道和很多东西是相似的，比如绘画又或是作诗写文章。总要能够达到返璞归真的程度才算是大家。所以如果能够修练到如平常人一般模样的人均是属于武功很不错的那一类了。

　　很难相信，他那么年轻便达到了这样的境界，有人把武功的提高寄希望于异果奇药。可朱履夜知道这只是一种幻想而已。武功的深浅其实还是看人下的功夫而定。没有苦功是不可能有任何的成功的，武功如此，其他别的也是一样。当然还要看教的人是否能够因材施教。而所谓的天赋异秉其实是看学的人是否能够有自己独到的见解和领悟。

　　朱履夜知道自己今天的决斗将是凶多吉少。虽然作为朱家的希望，她自小修炼的履夜神功已达到第七重，可是面对的是一个难知深浅的对手，对于胜利的信心和勇气便成为了决定胜负的关键。不幸的是对于这场毫无意义的决斗，她丝毫没有兴趣，所以不用比试就几乎已经可以知道决斗的结果了。

　　然而最后的结果，是她始料不及的。罗觅鸥主动的提出认输，他说他没有决斗的兴趣，一定会输，所以不用比试了。

　　从来没有想到过结果会是这样，也从未想到过在东海罗家竟然会有一个和她同样想法的人，所以那时是一种很知己的感觉。这或许就是他们相爱的开始吧。

　　实在太累了，一种睡意开始侵袭她的神经，大概是逃情酒开始起作用了吧。朱履夜缓缓的闭上了眼。

　　罗觅鸥骑在千里追风的背上，心却比跨下的神驹驰的更快。朱履夜现在到底如何？苦大师出游了两天，取到天香豆蔻后三天的时间已所剩无多。不知道还能否及时赶到她的身旁，他心似火燎，却知道千里追风已无法再快半分。

　　他的眼充满了血丝，虽是疾风扑面却毫无所动。毫无缘由的，在这紧急的时刻他想起了初见朱履夜的那一个晚上，确切的说应该是黎明。

　　那一天，他提前到了泰山绝顶。因为他喜欢那种寂静无声的感觉。

　　就是那一天，他见到了朱履夜，她站在那里，为周遭的自然所感动着。在黎明前的微光中，她的一身白裙是那样的显眼，在山风中她的裙连同她的整个人好象都在轻舞。

　　当他面对着她的时候，他感觉一种感动从心中升腾了开来。无可否认，老天是不公平的，因为他按照他的喜好给予了人们不同的外表，有的人受到了他的遗弃，难以面对世人。而有的人则加倍的受他的呵护。获得了最多最多的优点，面前的这个她无疑是受上天眷顾的一个宠儿。她的如雪肌肤、春山黛眉，还有那玲珑剔透的双瞳无一不是上天的佳作，老天爷把她造出来一定下了很大的功夫，因为那样完美的五官配合，那样动人的气质神韵绝对是妙手偶得之的佳句。

　　不用问，他就知道她一定就是长白朱家的小姐朱履夜了。对于这位小姐他知道的并不少。因为她是那样的不同于一般，关于这位小姐的传说实在是数不胜数。传说这位朱家小姐，虽是貌美如花，却偏偏喜欢寂寞，曾经到蛾眉山要求静木师太给她剃度出家，最后还是静木师太看出她尘缘未了，才没有剃度成功。也有人传说她三岁开始练武，七岁时便能够使全一套变化烦琐的三才越女剑法，是个真正的神童。据说她虽然一直对武功不是特别在意，很多时候会偷跑出家门游山玩水，却是朱家两百年来履夜神功练到第七重的第一人。

　　这决不是偶然，因为武功和人的领悟有关。就象围棋一样，少年不成国手终身无望，而武功如果没有自己独特的见解和领悟即使是苦练一辈子也只能达到一般高手的程度。决难达到武功的巅峰。武功练到后来，修炼不但是体力还有更多的是智力。能从自然之中寻找到学问，才能够达到武功的大乘。罗觅鸥知道这位朱家的小姐其实就是因为在自然中悟到了真谛，才得以迅速的提高了自身的武学修养。真是一个可怕的对手。他知道面对这样的对手只有全身心的投入才可能取得成功，可是他无法对这样的女孩狠心，不仅是因为她的美丽，也有一种惺惺相惜的感觉，为了这样无意义的决斗而出手实在是不值得。他知道自己对决斗没有兴趣，又无法对敌人狠心，在这样的状态下，实在不合适动手。

　　所以他坦然的认了输。

　　准备任凭对方处置，可他没想到朱履夜竟然告诉他。她同样没有决斗的兴趣，所以这是一场平局。

　　这真是个奇怪的女孩子，她这样的做法不知会在家中掀起怎样的风波。他想，无论如何她总是个女孩子，这样的结果，对于父母长辈如何交代……

　　千里追风终于倒在了地上，它已经耗尽了它的全部精力。罗觅鸥看了看这个忠诚的伙伴，它已经不行了。“朋友，谢谢你，再见！”罗觅鸥轻轻的说了这一句，便立刻的掉头飞掠而去。朱家庄院已经在望了，巍峨庄严的庄院犹如一只巨大的准备择人而噬的怪兽蹲踞在长白山下，在白雪皑皑的银色世界中是那么的狰狞。罗觅鸥一向不喜欢朱家这样的大庄院，同样的也不喜欢自己家。他觉得它们都是一个顽固守旧的家族的象征。身在其中，只会加倍的感受到压抑和不自在。那里面没有自由的人，有的只是一些听话的木偶。

　　门前的灯笼是白色的，罗觅鸥悄悄的潜入了庄院。三更时分，灵堂中的人都已各自回房休息去了。罗觅鸥打开了棺木，便看到了朱履夜静静的躺在里面，她的神态是那样的从容和安详。罗觅鸥把天香豆蔻塞进了她的嘴中，他感觉她的唇是温暖而湿润的。醒来吧，我的爱人。他在心中默默的呼唤着，轻抚着她的秀发，他想起了那一次的经历。

　　从泰山下来后，回到家中，他不可避免的受到了责罚，可是他当时只是当心着朱履夜。他悄悄的潜到了长白朱家，在那一天晚上，他看到了她在夜色中的寂寞。

　　没有人和她说话，她是那样的孤独。在这个家中可能没有一个人是真正了解她的。

　　一曲高山流水，知音何在兮？

　　“汤汤乎流水，巍巍乎高山。不知小姐为何所忧？”他忍不住出了声。

　　她诧然的一回头，那满头的秀发便飘散开来。

　　“你终于来了。”她嫣然一笑。

　　在那刹那，他知道他们是如此的了解。那以生俱来的相知感觉好象是从混沌初开的时候便存在着的身体的一部分……

　　灵堂的门忽然的被撞开了，面对围上来的众人，罗觅鸥只觉得无奈，无论如何他都不能伤害这里的任何一个人，因为这会使两家的仇恨加深。而朱履夜还是没有醒来。

　　罗觅鸥身上已经伤痕累累。可是他还是没有倒下去，他知道他倒下去之后便再也没有和朱履夜见面的机会了。朱履夜的二哥又一记重拳击在他的肋下，他的七经八脉几乎全部已经断裂。没有人能在只挨打不还手的情况下支撑这么长的时间。他晃了晃却还是没有倒。

　　时间似乎突然间的停滞了，他清楚的知道自己即将离开这个人世，只要有人再来最后的一击。他的嘴角渗出一滴滴的血落在寂静无声的灵堂地上，嗒嗒作响。似巨鼓敲打着每一个人的心。他的体力象时间一样悄悄的流逝着。他努力的回过头恋恋的看向朱履夜的‘尸首’。她还没有醒来，可能永远也醒不过来了。

　　缓缓的，他终于倒了下去。恍惚中似乎看到朱履夜站在云彩中间对他招手。

　　“你终于来了。”她笑着。

　　“是的，我来了。再也没人可以分开我们了。”
