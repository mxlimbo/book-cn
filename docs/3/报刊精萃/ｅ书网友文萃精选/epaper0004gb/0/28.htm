眼泪的美丽

李华新

　　眼泪是最美丽的饰品，这是在过去的年代人们所不能想到的，现在一些另类的 女孩就是这样装束了自己，她们在自己的脸上粘贴上一滴用玻璃做成的眼泪，让自 己有一颗美丽的眼泪挂在自己精心修饰的脸上。

　　也许人们会问这是一种何样的美丽，也许现在人已经不好再说什么是美了。

　　对于美的理解已经不是一种简单的对人心灵和外表的评价，而更多的是对于人 们心灵的震撼和让人目不暇接时的那种惊骇。

　　美已经成为标新立异的一种实现和你是否敢于这样的装扮自己的勇气。

　　也许这就是现在人们常说的出位。

　　读瘦马的<<出位>>就能感到这位青年时尚评论家对现在这些时尚问题的看法， 也是在列举了种种出位女人的表现之后，作出了前卫意义上的评价——女人在工业 社会昏头转向。现在在城市找一个完全不化妆的女人，完全素面朝天的女人已经不 大容易了。女人的每一个部位都受到了化妆品的侵犯，而女人们对此很乐意，并且 乐此不疲。

　　因此在对这些许多的涉及到女人本身的问题时，首先要回答的就是女人什么时 候最美？其实这一问题几乎是和社会的风尚相关联的，女人自己并没有权利来作出 回答，因为女人的美丽不是为了自己而美丽的，她所要达到的就是对于异性的取悦。 而当已经认识到，这样做的结局是何等的不公平时，就会反过来，以自己的出位来 得到女人所可的美丽。男人对于女性的美丽从来就是站在男性社会的角度来审视的， 女人现在的打扮已经越来越多的把自己的性感作为服饰的唯一理念，而男人们都是 非常坦然的接受了这一切。并不是说女人为了争取得到象男人一样能够穿上裤子， 几乎用了一个世纪时间的艰难已经不在有意义了。在这样的一种背景下，我们刻 意看到在着装上有另外一种低俗在涌来，说什么某某泄露了春光，说什么几乎是“ 全日空”的装束，意思是不穿任何的内心。就是当那些什么所谓的蕾丝开始向女人 展示卧室与闺房的神秘时，解放自我的女人其实已经要在公开的场合把自己的玉体 横陈了。在这样的时候，把一滴眼泪挂在自己的脸颊处，就是简单的不能再简单了。 我们也许在这个时候能够真切地感受到什么叫做异类的美丽，而这个时候女人 的美丽，其实就是一种出位的东西，而唯有这样才能在马路上让人家回头呀。 而在这个时候，女孩就能作出一副挺高傲挺自得的神情，让人们向她的美丽投 放目光。

　　美丽的眼泪就是这样的不用顾忌的挂在脸上，而且非常简单，对自己的身体没 有任何的戕害和损伤。

　　眼泪也能做成美丽的饰品，一滴晶莹的泪珠终日挂在你的脸庞，美丽成为一种 刻意随时可取的异类，美的价值好像就能格外的楚楚动人？
