今天在家闲来无事，忽然有种冲动想给宋小洁写封情书。

　　这种老掉牙的追女孩子方法每次我用的时候总是屡战屡败，古话说的好，失败乃成功之母，我大概可以跟失败的老母拜把子了，幸好我的脸皮比一般人厚了一点，否则我们学校的那仅有的几幢宿舍楼也不够我跳的。

　　写情书可是我的拿手，当然是拿别人的手写，要不然像我这种只有文字认识我的人怎么出去混呀，还好，我这个人别的没什么好骄傲的，死记硬背的功夫连我自己都有点佩服自己，什么闭月羞花、沉鱼落雁、国色天香、倾国倾城，魔鬼的脸庞，天使的身材，哦不对，是魔鬼的身材，天使的脸庞。。。

　　哇，连这种小学生的台词也在用，真是瘪龟！不过这一招可不像我的情书，它是屡战屡胜哦，由此可见，老师倒底不是什么冷血动物，他一定会说："以后记得要注意哦。"

（七）

　　"哇，你好帅哦！"阿荟从出来到现在都是一副惊讶的表情。

　　"当然，所以......"我一脸的苦笑。

　　"所以干嘛？"

　　"这个月让你也帅一下。"这次我是真的笑了。

　　"滚，你这个重色轻友的家伙！"阿荟也笑了。

　　"不会吧，你这么绝情！眼睁睁地看我饿死呀？"

　　"你怎么会饿死呀，你还有很多兄弟的嘛，哈哈……所以……

　　"她的表情看起来有点奇怪。

　　"所以干嘛？"这次该轮到我问她了。

　　"今晚的晚饭就让你继续帅吧！"

　　"什么？你！有没有搞错！说话不算数呀！"

　　我回头，就看到了一个女孩子――

　　宋小洁！！！

（八）

　　"在这里！"忽然从身后传来一句轻微柔弱的声音，当然，那绝不可能是上帝的。
