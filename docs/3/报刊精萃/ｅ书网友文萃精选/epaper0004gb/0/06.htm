城间的女人

——《人间四月天》剧本中张幼仪印象
陶冶

　　在中文系文学史的课堂上，受徐志摩与林徽音浪漫爱情故事的吸引，我买来了刚出版的《人间四月天》剧本来读，但读后给我留下最深刻印象的却并不是那个才子佳人间的传奇故事，而是编剧所塑造的张幼仪形象。

　　在我过去的印象中，张幼仪是封建传统婚姻中的牺牲品，婚姻枷锁里的死囚，虽有经商的才能，却没有觉醒的意识。但读了剧本后我对她的印象一下子改变了，张幼仪在剧本中是一个勇敢坚强的女人，虽然她受封建礼教、封建旧道德影响很深，但她却试图走出围在她身边的那座封建势力围起的城，这可以从她同意与徐志摩离婚，敢于创国内第一例离婚案时就可看出。在没有先例的情况下，她不与封建旧势力、旧道德妥协的勇气是值得我们肯定的。从这一点来看，张幼仪似乎已经走出了封建势力围住她的那一座城。然而，张幼仪性格中妥协传统的那一部分仍然暴露于剧本中，这在她离婚后仍为名分而留于徐家，认同无爱的婚姻也能继续等观点也可以反映出。不过，她的这些举动并不能否定她敢于抗争的那一部分功绩，因为在那个时代能做出那些举动已是很了不起的了！她的犹豫反而更加体现出了她在封建势力围起的那个城的里和外之间徘徊的事实，这便引出了“城间人”——这个属于她的概念。她实际上并不想主动走出这个城，但事实又让她不得不去试图走出，因为这是她所不得不接受的，所以她的选择是偶然中的必然，她在无意间成了首吃螃蟹的人，她的徘徊与犹豫是可以理解的。

　　离婚后的幼仪更加坚强，她先是只身在德国生活和学习，后又回国经商理财，办学校，着实干出了一番事业来。这里的幼仪又给人一个当代的女强人的印象，成为了新时代的典范。于是，她又似乎走入了一座新城，一座当代文明的新城。但她在这座新城中的位置也不稳固，她依然要时常走出来，因为她身上的传统意识并未彻底消失，她依旧经常在处理与志摩和徐家的关系上很为难，难以忘怀自己的名分和地位，未能完全摆脱传统的束缚。从这个意义上说，幼仪仍是一个城间人，一个位于对于她来说一座新城的里和外之间的人。

　　剧本中的张幼仪就是给我留下了这个“城间人”的印象，她矛盾的一生，也就是这样的平凡而传奇，她无法抓住志摩的心，也摆脱不了徐家对她的影响。她只是轻轻的来，又悄悄的走，却依然留下自己的足印。
