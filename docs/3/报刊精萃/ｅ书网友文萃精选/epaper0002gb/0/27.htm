小 粟

Piao

　　已经很久没有过好心情了，但在前些天去张家界三天的旅游，却一直都让我感到愉快，而且那段快乐至今仍充斥在我的心里。长时间枯燥而辛苦的工作，在那里身心得到了一次很好的放松，也让我几乎忘记了那段焦心烂肺的感情纠葛。固然张家界山川俊美，是喜爱登山的我的理想去处，能让我心情愉快，但如果没有小粟――我们的导游，这位2 2岁的女孩，我相信，这次旅行会失去大部分精彩。

　　小粟的性格活泼开朗、善解人意，天生就是名好导游的料。加上她乐于助人，对工作的尽职尽责，让她深受大家的喜爱。和小粟从相识到分别刚好三天，作为我们团三十多个人的导游，小粟自然是忙得不亦乐乎，要陪大家爬山，一边又为大家解说，中午晚上又要为我们能吃好住好而费神费心。

　　我们团是坐晚上的飞机到张家界的，第一个冲出机场的我也是第一个见到小粟的，样子长得一般，有点胖，由于有点胖，所以个儿也显得不高。如果不是此后三天的相处，这个女孩根本不会留在我的记忆中。但是缘分总是不经意中到来和结束的。

　　等所有人都从机场出来后就跟在小粟后边前行，我到她身旁问她：“明天会早起吗？”结果爱睡懒觉的我听到的却是肯定的回答，然后我开玩笑说：

　　“是不是要走路到酒店？”

　　“是呀，你怕走路吗？明天爬山还要走很多路哟。”

　　“怕？我怎么会怕？”对这句对我严重小视的话，我很不服气。

　　“那好，只要你明天能跟上我，就算英雄，我会敬你一杯。”好个巾帼不让须眉。

　　“是吗？一言为定！我会比你快的。”堂堂七尺男儿，岂会输一女子？爬山可是我的强项。

　　第二天我们5点半就起床了，乘旅游公司的大巴到景区只用了短短的40几分钟，一路上，小粟边为我们介绍三天的行程，边向我们娓娓的说着一些张家界以及土家风情，小粟的伶牙俐齿和诙谐活泼让整个车上充满愉快的笑声，4 0分钟像四分钟一样短，我开始发现小粟的可爱。

　　到了景区，上黄石寨，小粟不忘提醒我昨天的约定，显然，这有鼓励我的意思。团里大概有一半的人选择了坐索道上山，剩下的一半和小粟一起爬山。爬山本来就是我的所爱，何况还有和小粟的约定，这使我在前一半一直走在最前，到了天书宝匣，我已经领先后边包括小粟在内的人很大截儿了，就乘着那里的凉风等后边的人。而且，我还没能听小粟的解说呢。

　　等了不久，后边的朋友们和小粟一起上来了，“hi，小粟，怎样？我领先你很多呢！”由于我的领先，心里很得意。

　　“嗯，你还不错嘛，我可是在等后边的朋友呀，不过后边要走的路还很多哟。”小粟似乎并不服输，“还是别走那么快，我们一起走吧，听我给你讲解。”

　　“OK，不过，我不但跟得上你还领先你了？！”虽然我正有此意，也不能就此作罢呀。

　　“好了，英雄，我会敬你一杯的，OK？let's go !”

　　一路上，听听小粟的讲解，和朋友说笑聊天、留影，到了山顶竟一点不觉累，一路的欢声笑语，和张家界的奇峰异石、绮丽山水融在一起编织着难得的快乐。下山时，步行的更少了，我也有了更多时间和她聊天，除了张家界的风景，话题也更多了，她知识面广，说起话来口若悬河、滔滔不绝，我也凭着三寸不烂之舌，说得天花乱坠。真是香味相投，相见恨晚。从音乐到足球，从学生时代到未来的成家立业，我们聊起来就像滔滔江水连绵不断。

　　她爱唱歌，声音甜甜的，偶尔哼出的一两句也能沁人心脾。她似乎并不拘泥于所谓的淑女原则，走起路来大步流星，更能和男孩子大谈特谈 任何问题。她爱笑，而且总是哈哈大笑，每次逗她笑她都总笑得前俯后仰。她说：只要死不了，活着就是快乐。

　　每天的行程大约都在下午5点钟结束，然后6点多钟开饭，这中间的一个多小时是小粟一天内最闲的时候，我们总会不约而同的一起坐在宾馆的的大厅聊天，然后我知道了她的童年有多可爱，她也知道了我的童年时多顽皮，知道了她怎样做了导游，我也告诉她为什么我干着一件与我专业毫不相关的职业，我们相互鼓励，相互开解。小粟不仅是可爱，而且总让人从心底的兴奋。

　　我们像老朋友般谈得亲密无间，和她聊天，我好像活在一个无忧的世界，而时间也总是过得非常快，现在想起来，才知道什么是“一寸光阴一寸金”。

　　一开始真没想到，张家界的三日会如此的暇意和自在，但是幸福和美好的事情总是轻易的逝去，三天的旅游很快结束了，在送我们去机场的大巴上，小粟唱了一首“送亲人”，她的眼里噙着泪水。她请我唱支歌，我唱了首“友谊地久天长”。尽管意犹未尽，但双脚已踏入归途。

　　走下大巴，小粟向每个人告别。她向我招手，轻轻的微笑，我看着她的眼，无语凝噎，脸上的微笑难以抹去我们眼里的不舍。

　　我匆匆的钻进人群，从此各别西东。

“轻轻的我走了，
正如我轻轻的来
我轻轻的挥手，
做别西天的云彩……”
别了小粟，湘西的云彩。

　　飞机在夜幕中起飞，载着我和我那颗深深印着小粟影子的那颗心，渐渐的远离了张家界，渐渐的远离了小粟，渐渐的什么也看不到了……

后记

　　和小粟的相遇，就象两条直线的相交，在短短的相遇后，又彼此分开，然后重新过着彼次不同的生活。路过却不留痕迹。
