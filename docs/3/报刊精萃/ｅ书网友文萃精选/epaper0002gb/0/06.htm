必须做梦

醺儿（vyvian@163.net）

　　那个午后，我们从大亚湾驱车回来，天气阴沉，大片大片墨色的浓云，遮天蔽日。笔直而宽敞的水泥路上几乎看不到其它行车。

　　这时，忽然一道白色的光影在我寂寞的视线中闪现，如同矫捷的白龙。一辆福特面包车从我们的车旁擦肩而过，看它纵横阡陌的姿态，优雅而生动。

　　第一次感觉车的品质，带着它与生俱来的性格，还有一丝灵气……

　　虽然在车的世界中，它远远算不上是极品，但这一刻的亲切确实是种深切的打动，原来车并不只是乘坐的工具，在这宁静的风中，它也拥有灵魂。

　　始终还是喜欢欧洲的风情，自然的以及人文的情调，那份风韵甚至在汽车制造业的工艺上也体现得如此淋漓尽致。

　　迷恋意大利汽车风格中的浪漫情怀。

　　记得007中的女主角吗？美丽、性感、艳气逼人，狂野而招摇的法拉利超级跑车正是她的坐驾；还有热情奔放的兰博基尼，浪漫造型的老牌跑车玛萨拉蒂，无不显示着作为意大利半岛所弥漫的诗画般的情怀。

　　迷恋英国汽车高贵典雅的气质。

　　宾利和劳斯莱斯是归属于贵族和皇权的象征，陆虎越野车的设计也几乎全是为了所谓上流社会的人士而设计，每个细节都渗透着名贵的味道。

　　迷恋德国汽车的一流品质。

　　这是世界上最具权威的汽车国度。

　　当二战之前，世界很多地方还依赖马车作为代步的工具时，德国的大小城镇之间已经开始铺设平整而宽敞的道路，汽车工业已经在此获得了长足的发展，直到今天，奔驰、宝马、奥迪、保时捷、大众依然属于国际上顶级的汽车典范。

　　当工业时代的滚滚烟尘给所有荣华都遮上一层重重的沉埃时；

　　当城市污浊的血管匍匐着迟缓爬行的大大小小金属甲克虫时；

　　我们所崇尚的关于汽车精神的故事也越发遥远。

　　那个故事这样写道：亮和她的德国朋友驾着红色的宝马，飞驰在欧洲的大地之上，越过葱郁的丛林，越过哥特式尖顶的教堂，风儿轻柔抚面而过，恩雅的歌声带领我们走入纯美极至的境界，天堂般的圣洁之感。

　　在人行天桥上伫立，看车水马龙，尘埃永远无法落定，我居住在一个如此糟糕的城市。

　　必须做梦，才能进入一个鸟语花香、生命飞驰的世界,如发动机的澎湃动力，梦也源源不绝……
