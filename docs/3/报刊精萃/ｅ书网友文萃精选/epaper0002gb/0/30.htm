京 广 线 上

一凡

——京广线全长2千多公里，两条平行的路轨相距约1.23米。 走过了2千多公里，它们始终没有走在一起。 甚至，连接近一点的机会，也没有。 或许其中的一条很想去挽起另一条的手， 但路轨始终是路轨，勉强走到一起，就成了一堆废铁。

　　2000年7月30日夜11时，我到达北京西站。手里拿着明天凌晨5时回广州的车票，呆坐于第六候车室的椅子上。突然想起了一些什么，我神经质地冲向公话亭。“你拨的用户已关机”话筒里传来依旧冷漠的声音。每次听到这把声音，我总是想砸掉电话，然后大叫。是了，她总是习惯夜里关机睡觉的。站在公话亭旁，想到将有30多小时听不到她的声音，我好害怕。作为一个男性，我讨厌自己的脆弱，我不能面对孤独，我不能纵情自如。而为了让自己觉得自己还是个男人，我要找个方法去驱散恐惧。最常用的方法是什么都不想，让脑袋停下来，也就是发呆。

　　候车室里没有多少时间让我发呆，我很快就被人带到了一个在二楼的小房间。在那，花上10块买个床位，我可以安稳地睡一宿，到点了就有人把我叫醒。这就是市场经济，到处是商机，只要找准了供求关系，钞票就向着你来。我没去深究他们能赚了多少钱，虽然这是我每发现一种新的经营手法时的爱好，但现在我不想让脑袋转得太多。

　　4点半就被喊醒了，但还是到了5点钟我才能进站。由于是中途站的缘故，车只停5分钟。我这时才发现，这趟车居然那么多人。我费了点劲上了车，站在车厢中间。走道上站满了人，真的很挤，有点不知所措，难道就这样24小时站着回广州？

　　把不值一文的行李随手放到坐椅下，还踢了两脚，那位置是最方便，最安全的。然后我逃离了这节车厢。其他车厢没有给我什么惊喜，这在概率分布来说很正确。两节车厢间的接驳处会是个好地方。那是吸烟区，人比较少。找个位置坐下，只要不挡路，就不用担心路人和小卖车的骚扰，还可以睡得香。这比车厢里的走道优越多了，只要你不怕脏。

　　这班车的确是个典型，就剩下这个空档了。算了，坐下吧，要不然怎样熬回广州。还好，不算脏，有一层烟灰，几滩水迹。可能是西瓜的汁液，可能是碗面的味精汤，可能是某君的上呼吸道分泌物，不会还有别的，火车上的水资源本来就珍稀得很。我一直在那，吸烟，看书，睡觉。

　　看书能让时间过得快些，于是手中的书成了我的珍宝。我看着看着就想到自己。书中的女孩，小妖，很少回家，也很少想家。一放假就跑到北京去的我，也不是留在家的料。我不是好孩子，我也没说过我是。我还不想回家，只是没钱了。不能靠自己双手赚钱，这是我和小妖很不同的一点。于是，我比较现实，无法唯美，无法堕落。而最不同的是，她可以不爱，我不能。小妖的不爱是因为怕爱得太深而受伤。我算不上，我不唯美，我很现实。我不能伤害自己以伤害别人，因为这样伤害不到别人，所以我没有自杀的念头。于是我选择了去爱。尽管一段没有结果的感情注定要受伤，但现实的我只会轻伤，我想。

　　我想起了她。大约在19小时前，我们在呼市到北京的班车上。天意弄人，我们居然买了一张作废票，只好三人两座了。我坐在最外面，位置很少。后来她的左手放在我身上，让我靠着她睡。这样我会有更多的位置。我硬是抱着她的手，环在我的腰上，用左手紧紧捉住她的手。因为我知道这是最后的一晚了，就当是满足一下自己的虚荣吧。后来，她靠着我的背睡了，我感受着她头部的重量，发梢对我后颈的刺激，和她那说不上肌滑如丝的小手。

　　都过去了，我身处拥挤的车厢，而她，应该在济南某间温馨的小旅店的床上吧。

　　脑袋转了起来是一件令人讨厌的事，我想到了她思念的人。那也是一段没有结果的感情。但她比我幸福，因为那个男孩也喜欢她。或者因该算是不幸。但我比较现实，我不能接受自己喜欢的人也喜欢自己算是不幸。十多天的旅程，她没少提他。为了他的一个电话而雀跃，发呆，为了一个音近字又想起了他，她爱他。他得知她一个人去泰山，便叫她小心点，所以她决定小心点。于是，我告诉自己送行时说句再见就算了。当然，最后我还是很不放心地重复了他的话。直至现在我依然懊悔自己的无能，说了，还是白说。

　　想到了他，我总想哭。没想过作为一个男人会那么软弱。在这拥挤的车厢内大哭一场会是一件快事。但我很现实，所以我不哭。

　　列车停的第一个站是石家庄，时间9点15分。还有近20小时我才回到广州。尽管广州没有给我什么好印象，但毕竟是我生活的地方。我可以回家，可以洗澡，可以看看堆积已久的电邮。原来，我还是想家的。我又想起了她的手，她的头和发梢。这时，我知道下车后要立刻去做。我极少计划未来。因为经验告诉我计划不可靠，事情的发展总是出人意表的。我最近的一次计划是在认识她不久。我知道她是我想要的，于是决定向她表白，时间订在考完四级后。但她提前了两天告诉我，他的故事。于是我就不再计划了。这次例外，如果列车不出轨，而我在这十几小时还没有死的话，这个计划是可以实现的。我要打电话给她。很想一下车便打，但我不能那么傻，因为我现实，我要她睡醒后，清清楚楚听到我说的每一个字。

　　我好念你，真的，好念。车上不辛苦，因为挂念你。有了挂念你的心情，有了打电话给你的计划，所以不苦。放心吧，别以为我又犯傻了。我只是挂念你，并没有说想要你。虽然我的心很想，但我不要，因为你的心在别处。没错，我们只能是好朋友，但爱上了好朋友并没有和宪法有任何抵触，于是，我唯有爱你。你不好？我知道啊！老早就知道啦！既然那天我决定爱你，那我已决定爱你的一切，你的好，和你的不好。他？关我什么事？那是你自己的事，自己解决吧！一个你已经够我烦的了，还想怎么样呢？反正我爱你，我的感觉，我的私事。同样，你爱他是你的私事。那就是说，我爱你，与你无关。

　　列车飞驰，看完了手头的书，我百无聊赖。这样的时间是最难过的。如果她在，48小时的车程我也会嫌太少。这就是爱因斯坦相对论又一有力佐证。其实现在离广州已经很近了，只剩4个小时左右。我不敢停笔，怕停下来，4小时会变成16小时。我睡不着，因为没有她的手，她的头和发梢。静静想想过去的二十多小时，好象过得很快，又好象很慢长。车上的人已不如原来的多。我终于可以与我的行李在一起了。里面没有值钱的东西，但抱着行李包，我知道它是属于我的。在这列车上，就剩下它是我的。我望着它，神经地发笑。
