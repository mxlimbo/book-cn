拾 趣

闲庭落花（vyvian@163.net）

　　我的预产期原本是六月七日，从怀孕那天起我就希望我的孩子能够在六月一日出生。

　　结果，那天成了他的生日。

　　今年的儿童节，他已经四岁了。

　　记得第一次去动物园，儿子就很喜欢老虎，左看右看看不厌，那老虎也的确使出了浑身解数，显示了它作为森林之王的威武。

　　回来后，当有人问他叫什么名字时，他竟然回答：“我叫华南虎。”

　　后来，电视上出现了一则广告：一个年轻貌美的女子在山溪边洗头，老虎走过来，那女子将美发一甩，黑森亮丽的一道风景，老虎昏倒了。

　　儿子走过来说：“妈咪，姐姐头发太漂亮了，连老虎都晕倒了，那老虎真没用。”

　　我也没有去细想，广告嘛，那上面什么事情都有可能发生。

　　等再有人问他叫什么名字时，他便指指上衣胸前绣的红字。

　　朋友送来一套《101忠狗》碟和画册，儿子一遍遍的看着，然后偷偷告诉我：“妈咪，你以后叫我斑点狗啦，斑点狗很聪明也很勇敢的。”

　　于是，家里装修后的洗手间里，用淡蓝色的斑点狗瓷砖贴上了一条漂亮的围线。

　　想起自己还是孩子的时候，最喜欢的国产动画片就是《孙悟空大闹天宫》，不论人物造型，还是情节的层次，音乐的制作，都非常精品，于是专门去买了一套回来，和儿子一起看。

　　小家伙很快就被孙悟空上天入地的本领迷住了，有段时间，只要他一放学回家，大闹天宫的故事就会反反复复、无休无止的播放，最后的结果是儿子竟然能够跟着故事主要情节的变化，达到同步配音的本领。

　　“孙悟空是谁呀？”

　　“五百年前得道的猴精，能够降龙伏虎、移山倒海，……后来跟着师傅唐僧去西天取经。”

　　“太白金星引诱孙悟空上天时说天上是如何美丽的呀？”

　　“到处金碧辉煌
　　处处奇花异草
　　星星铺成银河
　　彩虹架座飞桥……”

　　当然，Monkey King又成了他新的名字。
