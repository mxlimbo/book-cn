没有母亲的生日

风之缘

【开头的话】我写《母亲的眼泪》这篇文章数次被抑制不住的眼泪所打断和终止……到现在我的心情一直都很不好。

因为这篇文章同时刊登在了“证券之星*星光灿烂”和“天涯虚拟社区”，很多网友给了我回复，有的还发了妹儿给我，另外“海魂”网友还为我传来了陈百强的《念亲恩》： “长夜空虚令我怀旧事，明月朗相对念母亲 ，父母亲爱心柔善像碧月 ，怀念怎不悲莫禁 ”大家给我安慰、鼓励和共鸣。我再次鞠躬向你们表示深深的感激和感谢！

我明白，大家对母爱的真诚的感悟。每个人都有母亲，每个人对母亲的敬重的心情都是很真切的。我失去了一位好母亲，可是，母亲给我的爱将永远留在了我的心里。

我衷心地祝普天之下的每个人都在伟大、博深的母爱里幸福一生！


　　昨天是我很特别的日子，按照农历推算就是我的生日。因为工作繁忙，自己对自己的生日都记不清了，已至于上次竟然将自己的生日提前过了一个星期，闹出了天大的笑话，觉得很对网友不住的。

　　赶在自己的生日来临之前，我终于将早就应该写完，早就应该献给母亲的一篇文章在农历六月二十八日的凌晨两点前脱了稿，那一夜，我再也无法入睡了……

　　我是一个很爱好清静不好喧闹的人，对自己的生日我从不张扬，昨天来的朋友都是我极为要好的。我们彼此之间都有一个约定――就是每当各自的生日到来的时候，彼此都互不送礼，相互间或一束鲜花或一盒蛋糕就是最为奢华的了。面对朋友的祝福，我很感动，就如同上周六面对网友的祝福一样的心情，唯一不同的是，昨天的生日我很沉重，心里老是觉得丢失了什么。是啊，这是我度过的第六个没有母亲的生日，虽然已经是六年了，随着这时间的推移，我的这份情感，这一份想念母亲的感情是逐年的在增加，逐年的在深浓……

　　孩提的时候，家里是很清贫的。所谓吃东吃西，都只能够图个肚饱。尽管是这样，每当我的生日的来临的时候，母亲是绝对不会马虎的。在这一天，母亲必会早早地起床，先是冲好一碗鸡蛋，然后再唤醒睡梦中我，拿出一件靠她省吃俭用而买回来的新衣服为我穿上。睡眼朦胧的我是很会依耐地在母亲的怀里撒娇的，身子老象没有骨头地躺在母亲的胸前，母亲就会拍拍我的屁股说：看哪，今天长这么长的尾巴了，还这样子，不象话咯，你再这样我就会告诉你未来的媳妇……。欢笑声里，我就开始了幸福的一天。不管怎么的忙，母亲在这一天是不会外出的。母亲可能是因为生活艰难的原因，我是很少的看见她的笑容的。但是，在这一天，她总是很开心地以各种的方法逗我乐。在我的记忆里，我生日的这一天是绝对没有现在这么多的吃的，零食顶多就是那5分钱一袋的杨梅。母亲有一次就象是孩子一样，她爬上一棵“黄蜜芽”树，给我摘下那很嫩的叶尖，冲洗干净给我吃，那清凉甘甜的滋味至今还留在我的脑海里。

　　我的生日是在炎热的夏季，在南方，那树梢上到处都有那不停地鸣叫的蝉（也叫知了）。母亲操起柴刀到后山砍下一根细长的竹子，劈去枝条，然后牵着我的小手，在屋掾下将蜘蛛刚刚吐的网缠绕到细软的竹尖上,等到蛛蛛丝有一粒黄豆那么大时，母亲把它用水打湿，蛛蛛丝就变成黑色了，变得极具有粘性。这时候，母亲就教我去粘蝉，那撅臀昂胸一个劲地嘶鸣的蝉一旦被这蛛蛛丝粘上翅膀后，不管它怎么的拼命挣扎也都逃脱不了了，母亲就会帮我把它从竹竿上拽下来，扯掉翅膀，放进袋子里，不要多久的工夫，几十只蝉就到手了。母亲把它们一一剪去翅膀、脚肢和尾部，再拿出只有过年的时候才用的茶油，放到锅里煎炸，不多久，一份放有红红的辣椒、黄黄的姜丝、香脆可口的“野味菜”――“油炸知了”就出来了。可以说，这就是我生日里最为“奢华”的一道菜。

　　尽管在我孩提时候所过的生日是那么的简洁和清贫，但是，因为有母亲和我一起度过的生日，有母亲对我无微不至的关爱，我一直都觉得很快乐，很无忧，很充实的。随着我的慢慢的长大，尤其是自从我参加了工作后，家里的条件和环境也逐步地改善和好了起来，也该是我那吃尽人间疾苦的母亲享享清福的时候了，但是，母亲却因为过度的操劳而过早地离开了我们，今生今世，作为儿子的我是无法报答她老人家的养育之恩了的，只有这无尽的忏悔和愧疚伴我终生。

　　母亲说：这个时候就是我降临人世的时刻。我一遍遍地端详着被我深情地凝视了几多回的遗像；我反复地抚摩着母亲留下的遗物：这是一件怎么旧的毛袄啊，父亲说，母亲穿了它20年，里面已经发黄，发硬，但是被母亲用一些布料精缝细补的，一直没有舍的丢；那又是一个用了15年的茶缸，母亲每次的喝茶全是用它，尽管满是班驳……，近几年，不论家里装修，还是乔迁，这些东西一直我都妥善地保存着，乃至我都不舍得去抹和清洗，我想那东西尽可能永远地保留着母亲的气息。

　　昨天的生日午餐里，我特地的点了这么一道“油炸知了”，我把它摆在母亲的灵前，用母亲生前所用过的遗物默默地为母亲斟满酒、倒上茶，我点燃香、拨亮烛，将打印出来的《母亲的眼泪》的文稿会同那寄托我无尽哀思的冥纸一一焚化……

　　哀痛和着怀念，花瓣伴着泪雨。

　　愿天堂的母亲快乐、幸福……
　　愿天堂的母亲接受孩儿永远的思念……
　　愿天堂的母亲安息吧……
