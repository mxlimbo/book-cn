今日九时四十八分夏至

逐渐逐渐

今天中午我又喝了酒。不知道为什么。
并且喝的不少。天气够炎热的。再用酒去烧心。
有知了叫唤嘛就更热了。尤其心里面——热。
一种燥一种烦，烧得我通身难受。体内火光冲天。
它们在浓密的绿色里，交换着自己对夏天的看法。
我不知道该说什么好。如何表达会更贴切。

由于酒精的缘故还是别的，我觉得自己变成了火山。
变成了什么能源之类的东西。
我看了看摆在桌子上的日历。知道了今天为什么这么热。
不是心理的原因。也别往自己身上找原因了。
因为——今日 ，今日九时四十八分夏至。

我变成了一种能源，这说明我退化了。
我们除了退化还能干什么呢。
看看能源，我明白。
就是指存在于自然界中，并能够转换成为热能。
机械能/光能/电能等各种能量的资源。
连这个我都这么清楚，还有什么明白不了的呢。
我仍然清醒着。在燃烧过程保持这种状态很不容易。

等待算什么。我们知道结果。谁都知道。
但结果仍然还没有降临在自己身上。
但知道的结果。并且是好结果。
我们就一起等待那个好的结果吧。
所以这几天我一直很幸福。等待是幸福的。
即使等待的结果是一种灰烬。


既然自己是能源，我们一起分析分析。
我们不就是爱分析自己和别人，分析别人和世界吗。
按照能源的来源分，一次性能源是天然能源。
原煤/天然原油/天然气/油页岩/核燃料/风能/太阳能。
不需要转换加工就可直接利用。
不喜欢加工转换的能源产品。蒸气/焦碳/汽油/煤油/柴油/
电。
够深奥的吧。我自己都快糊涂了。
你不明白别人说什么的时候，你就装糊涂。
刚才别人和我争论什么的时候，我还眼看着天空。
装做什么也不在乎的样子。
实际，那时我在估计着今天是否下雨。
别人以为我生气或者烦躁了。
所以我很高兴。特别特别高兴。

看看，还有特殊的什么原因没有。
按照能源的性质分，可以分成燃料能源/非燃料能源。
矿物燃料/生物燃料/化工燃料/核燃料。
非燃料能源指太阳能/风能/水能/电能/潮汐能/地热能/海洋能/热水。
我想我就是燃料能源，那多牛呀。
牛就是厉害的意思。就是特能穷显的意思。
按照使用状况分，常规能源/新能源。
目前科学技术下，已被广泛应用的能源，煤炭/石油/天然气/水能/桔杆柴草。
新能源有，核聚变能/核裂变能/太阳能/风能/地热能/氢能/潮汐能。

说到主题了，我想我怎么快成了潮汐能了。
原来可能是太阳能呢。当太阳能太累。
不如潮汐能潇洒。有个月亮罩着你就行了。
你就什么也别想了。要想也要正经地想。
想出个所以然。
现在我是潮汐能。
所以我很轻松，所以可以轻而易举让我的感觉淹没别人。
别人也会说我伟大得可以浪漫得可以痴情得可以傻得可以。

按照能源生成难易程度与所需时间长短分，有可再生能源和不可再生能源。
可再生的有太阳能/风能水能/生物质能/潮汐能。

潮汐能就可以再生。所以我愿意是潮汐能。
尤其你看夜晚的大海，那种平静的出奇的浪。
岩石都那么沉默。所以我愿意是潮汐能。
这种想法由来已久。
不可再生的终将耗竭,也称矿物能源。
原煤/天然气/天然原油/
在今后相当遥远的时期内不能再生。

今日九时四十八分夏至，我幻想我变成了潮汐能。
因为月亮的存在，而去冲刷海滩，冲刷记忆，冲刷岩石。
摧枯拉朽，势不可挡，但悄无声息。

那月亮，就是致使我在今中午醉酒的女人。
