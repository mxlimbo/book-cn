冬天，再是夏天

妙儿（yemiaoer@eLong.com）

　　这个夏季走的极快，还未曾好好将热力拢在手心细细揉捏，转眼便已是秋凉如水。风儿微微吹过，叶子是一片一片飞舞着落下的，掉在地上，枯黄憔悴的模样勾起心底一阵接一阵的凉意。

　　小柯极喜欢秋天，说这是属于他的季节。尤其是一到风起的时候，便爱拉着我四处东走西逛，全然一改夏天温吞的德行。

　　可是我不喜欢。

　　我喜欢极端的事物。夏天便吃火锅，冬至吃冰激凌。那种夏天大汗淋漓，冬天冷至心骨的畅快才是我迷恋的。不象秋天，只一径的金黄炎炎，没有半分特点。

　　所以夏天过了，在秋天中我只期待冬天，再是夏天。

　　小柯会笑我是个太冷漠的人。他说我总是站在他的身边，不多语，静静的，安详的仿如影子。只是影子也会随着主人的变幻一同变幻，而我，依旧还是那副静静不动的样子。初始，他还尽力逗我。久了，知道我是这副脾气，便由我去了。

　　但小柯很宠我，象宠个孩子。

　　睡熟了，他会在身边微笑望我。睁开眼睛，便是他含笑温柔的眸子。

　　他还会买了很多铃铛给我。挂在我的手腕上。走路时就拉起我的手轻轻摇晃。

　　“叮叮当当，叮叮当当。”

　　铃铛发出丁当的声音。小柯的笑脸在秋风下是那么的耀眼。

　　“丁当，你听，多好听的声音。叮当叮当。”小柯说，低头轻轻吻我的手指。我的指尖冰凉，象那铜铃的温度。

　　有时静下来便会细细端详手腕的这串铃铛。金黄的色彩，是秋天的颜色。上面有古朴的花纹。一道一道，象秋天麦田里成熟的麦穗。小的时候会将麦头折下来，拨开，把小小的穗粒揉进嘴里，细细的咀嚼，有香甜的味道。想起，觉得嘴里还是一口的香气袅袅。

　　初秋的时分还依稀保留着夏季的余温。天气乍寒还暖。

　　铃儿在手腕叮当做响。

　　小柯拉着我走街窜巷。

　　叶子还在落着。一片两片。

　　熟睡，小柯微笑的在身边望我。

　　一天，两天，时间过去。

　　我一直在等着，冬天，再是夏天。
