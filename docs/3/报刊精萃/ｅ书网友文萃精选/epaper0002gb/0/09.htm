　　我才不相信她会舍得放弃我呢，她绝对上瘾了！可是。。。。

　　我真的在以后的日子里再也没见这名字，用IP呼她，倒真有人应答，但已经不是月季了，说话我能听出来，因为，别人比我还油腔滑调。。。。。

　　难道我这经常骗人的人也被人骗了一道？！不行！我太有失败感了，打小抄作弊被寻考的捉出去，也没这么七上八下的难受呀。。。。。

　　一个月过去了，俩个月过去了。。。。。果真和头发乱蓬蓬的蒲公英一般从我的视野里消失了。。。。。

　　我真的打了那电话。是个男的接的，我难以启齿，后来脸皮厚了，索性把酸得我都说不出口的的网名说出了口，"你们这有个用月季上网的吗？那人很冷漠说："不在我们单位了，你说的是才女刘辉兰，她2个月前拿到研考的成绩，后来复试，考到厦门去了。。。她等成绩的那段时间，嘴角全是泡。你是谁呀？喂！说话――，谁呀――有病呀你"

　　我的思绪一下变成了犹如连接话机和话筒的黑色而有规则地弯曲着的电话线，在办公桌的左边的抽屉沿儿上，悬挂着，荡漾着。

第三章

　　早春土壤解冻后，是种植月季的最好季节。月季起苗时一定要带土团，尽量不伤根。为降低枝叶水分的蒸发，应当适当剪去部分枝叶。栽后要及时浇水遮荫，经过一周的缓苗期以后，早晚要接受弱光，并逐步置于日照下，进入日常管理。月季比较耐寒，土壤不宜过于潮湿，尤忌积水。地栽或者盆栽月季应一次浇透水，待土表干时，再次浇透水。特别是移植后的头次水一定要浇透。以后根据土壤见干见湿的原则，适时浇灌。如浇水过多土壤积水，会影响月季根系的呼吸，使根部腐烂导致死亡。月季的施肥要以粗肥，基肥为主，多施有机肥有助于土壤改良。所用的有机肥范围很广，如腐熟的猪牛羊鸡鸽粪及宠物粪便均可施用。及时灭病治虫是栽好月季的关键一环，要注意病虫害的发生，防止白粉病和黑斑病的蔓延，可用500倍多灵菌与与治虫药物混合使用加以防治。

结　局

　　爱情是只春天里的虫子，它的思念和它的肉一样风起云涌地滋长着，蚕食着月季那样的花儿。
