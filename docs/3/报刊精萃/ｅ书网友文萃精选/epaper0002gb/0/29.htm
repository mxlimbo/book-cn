我心底里的三毛

阿璇

　　那个长发中分、皮肤白皙、身材高挑、一笑露出一口不太整齐的牙齿的女子缓缓的走过来。她左顾右盼着（并不是纯情少女的那种矫揉造作），善意的的眼神轻轻的抚过周围的一切。慢慢的，近了，近了，到了你的眼前。你看到了她脸上跳跃着的温情，那种温情令你忍不住想伸手拉住她。她却如轻纱一般飘荡着，离开你渐渐远去，只剩了飘渺和空灵，还有淡淡的心灵震撼围绕着你。

　　这就是我心底里一直的三毛。
　　
　　最初知道三毛是因为她的死。混沌初开的日子里，总对那些为情所困的红颜女子的故事特别感兴趣。在人人都哀悼三毛的同时，我满怀着好奇走进了她的世界。

　　手捧着《哭泣的骆驼》坐在高高的石阶上。葱绿桃红的日子被春末夏初的风轻轻的拂走；拾阶而上的行人踏着轻灵的步子缓缓飘过；初初开始鸣叫的知了从密密的树枝里倾泄出它的快乐。我不在天上也不在人间，只是行走在三毛的字里行间，行走在属于她的世界里。

　　坐在船舷上，《撒哈拉的沙漠》翻开着铺在膝盖上。那干枯的沙石被三毛的温情滋润着，长出了绿色的小草。船儿迎着风踏着浪，掀起一层层白色的浪花。微微弯腰，将手伸进水里，水波儿荡漾，穿过指缝流动，类似柔顺的黑发在指尖掠过的感觉。

　　夜阑人静的晚上，半卧床头，清香的绿茶袅绕，《夏日倾城》伴枕。英姿飒爽的军官，在遭遇那不算漂亮然而美丽的女子时，内心是怎样的一种 "众里寻她千百度，那人却在灯火阑珊处"的欣喜啊；在车船的轰鸣声中，他们擦肩而过的缘份又令他如何的痛心和惋惜？抽动的心，微微起伏的情绪，触手可及的爱转眼化成泡影的绝望。
　　
　　更深一点的了解是令胡慧中一举成名的《欢颜》。

　　许就是三毛自己的梦想吧。那个长发披肩清新脱俗的唱着《橄榄树》的少女；那个为了理想而飘泊四方的英俊少年；还有那对离去的妻子怀有深深眷念的满脸沧桑的男子。

　　假如我是个有自闭症的孩子，那我的行为在你的眼里是怪诞而不可思议的。我不太爱说话，因为我要面对自己的思想。我想着遥远大漠里孤独的落日；晴朗蓝天上独自散步的白云；在清晨摇曳露珠的不知名小草；在夜间悄然展现自己美丽的夜丁香；在荒野中突兀的丑陋岩石；被清水磨得光滑的鹅卵石；在异地他乡的飘泊旅人；在小桥流水边打闹嬉闹的孩童；坐在门槛上敞胸给孩子喂奶的妇人；憨厚纯朴黑脸白牙相映成趣的乡人；满脸皱纹却笑得灿烂的老人；污脏破烂沿街乞讨的乞丐……

　　他们一一在三毛的眼前出现，在三毛疾走的笔尖流淌。或许，她的地理知识是浅薄的，可在那一块块贫脊的土地上出现的人和物却给了我们足够的感动。因为，三毛是踏着梦想的足迹流浪在远方。
　　
　　为了梦想而流浪的孩子，让父母牵挂了一生的孩子。

　　三毛只是一个平凡的孩子罢了。那慈祥和蔼的父母让我感到三毛原来是触手可及的。年幼的无知、年少的反叛，都在父母沉甸甸的无言的爱下融化。

　　再有，再有的就是那个满脸大胡子的荷西。哦，荷西，令三毛在一夜之间绽放出所有美丽的男人。他用痴情和执着感动那个骄傲而又自卑的女子；他让她不再独自筑巢在林梢；他陪伴她学着用柴米油盐来直面生活；他给了她在平凡的生活中发现更多不平凡的灵感；他用海一般宽阔的胸怀容纳着她的一切。
　　因为有爱，她的平凡与不平凡完美的结合在一起，她与生俱来的温情与善意恣意生长着。

　　那因了不愿白发人送黑发人的悲剧在自己的父母身上重演的女子，在丈夫不幸罹难后悲痛欲绝，却不得不做一只没了心的不死鸟。
　　
　　这就是三毛，在我心底里的一个因爱而充满温情和善意的女人。
