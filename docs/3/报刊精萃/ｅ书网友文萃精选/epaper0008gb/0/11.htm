寂 寞 旅 途

无情

　　寂寞是一种深入骨髓的疼痛，没有人能给她解脱，只有自己。

　　――前言

　　她微微斜着脸庞，依靠在窗口。窗外，昏暗的天空，大片大片浅灰色云层呼啸而过。

　　她说，她很寂寞。她象烟花一样寂寞。瞬间即逝。黑暗中，她抓紧着他的手，埋在他温暖的怀中。

　　他们相恋三年，终于在那个下雪的冬天结婚了。

　　她是伤感的女孩。她喜欢黑暗。黑暗中，她可以触摸自己光滑如丝缎般的肌肤，可以嗅闻自己漆黑头发的清香，可以听自己洁白牙齿磨擦有节奏的响声.......她说，那是因为寂寞。她本以为结婚可以填平一切，那是两个人亲密世界。原来，寂寞是一种深入骨髓的疼痛，没有人能给她解脱，只有自己。

　　她开始抱怨。他公司太忙，每晚深夜回家。她可以忍受寂寞，但不能忍受孤独。

　　每个失眠的夜晚，她都在黑暗中度过。有时她也会出去逛街，午夜的街道，空气异常的新鲜，她放肆的呼吸，感觉内心的空虚，暗红色的天空，繁星忙乱的闪烁，她在暮色苍茫中飞舞.......那是她一生中最快乐的时光。

　　他说，你晚上出去干嘛了？她笑着，我是去旅途，一次生命的旅途。

　　她喜欢这样的旅途，孤独的，但不寂寞。她甚至把自己想象成一只蝴蝶，自由自在，无忧无虑，在空旷寂静的田野上空飞舞，和花朵簇拥在一起。蝴蝶的生命是脆弱的，但美丽。她这么想着，脸上笑容灿烂。

　　那天深夜，飞舞中她感到一双温暖而有力的手从身后抱住了她。她再也飞不起来了，她转身，就看到了他。

　　他说，我观察你好几个晚上了，你是不是病了？我知道结婚后没有好好待你......她看着他，笑了，不，我没病，我是在做一次旅途，一次生命的旅途。孤独的，但不寂寞。

　　他晚上回来早了。他要陪着她，不让她再出去。他说她可能是病了。他给她买回来许多药。有安眠、治忧郁的、精神方面的......她说，她没病。她只是寂寞，一种深入骨髓的疼痛，没有人能给她解脱，只有自己。

　　她不需要吃药。他在房间里追上她，紧紧抓着她的手。黑暗里，她无法逃避，她在他怀里挣扎、颤抖。他用力掐开她的嘴，泪水就在脸上遗落，她哭了。

　　他说，你需要它们。因为你病了。她倦缩在阴冷的角落放声哭泣，她的胃里翻腾着那些肮脏的药物，她的嘴角因过度挣扎而流血不止。她累了，在药物的强烈作用下，整个人都晕眩......

　　他说，也许你应该去看医生，你的病越来越严重了。她容颜憔悴，皮肤粗糙，神情忧郁。她站立着长时间发呆、沉默。她是真的病了。她的生命已经脆弱的经不起拥抱。

　　熟悉的肮脏药物，雪白的粉刷墙壁。医院里，她把自己关起来，黑暗中，她可以触摸自己日渐粗糙的肌肤，可以嗅闻自己零乱头发的熏味，可以听自己枯黄牙齿磨擦凌乱的响声.......她说，那是因为寂寞。她在狭小的房间里忍不住飞舞，把自己想象成一只蝴蝶.......

　　一年后，她完成了旅途，在那家精神病医院自杀了。
