最后的爱情故事

无情 （cappuccinoo@citiz.net）

　　她转过身，看了他一眼。那是她最后一次看见他的脸庞。苍白的，没有血色，让人窒息。

　　她倒下，带着那把刀。血从她身体里如洪水般崩溃。黑暗里，她的泪水从眼角慢慢滑落。

　　她爱他。

　　窗外，粉色的樱花，一片一片地落下。她说，她好喜欢看樱花坠落的样子，那是一种生命，生命坠落。

　　她爱上他的那天起，就为他付出了一切。她以为，他会爱护她，她以为，他会给她幸福，她以为，他会一生一世都爱她。

　　可是，当那个女人过来找他的时候，她才知道，原来自己很傻。

　　他握紧她的手，告诉她，他和那个女人没有任何关系。没有，真的没有。他是爱她的。

　　她的眼睛放肆的凝视，他把脸低下。她说，不要骗我。不要，真的不要。她的手冰冷，内心恍惚，泪水溢出。

　　他抱紧她，感觉彼此微弱的颤抖。

　　那个女人疯狂的冲过来，拼命的捶击他。他承受。那是对他的惩罚。他说，我不爱你，你走。

　　你骗我，你骗我。那个女人倒在地上，放声的哭泣。你说过，你会爱护我，你说过，你会给我幸福，你说过，你会爱我一生一世的.....

　　她的头发零乱，精神崩溃。她在他身后茫然，痴呆，随后顺手拿起桌上一把菜刀......
