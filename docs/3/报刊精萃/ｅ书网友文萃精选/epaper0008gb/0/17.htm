今年冬天她将要死去

无情

　　她说，今年冬天她将要死去，就象窗外的雪花，阳光照耀的时候，它们需要融化。

　　一年前，她得了缓期不治之症。医生告诉她，明年的冬天将是你生命坠落的时刻。

　　她把自己关起来，黑暗中，她的呼吸瞬间停止。她感觉到自己内心的恍惚、恐惧。她捂着脸，倦缩在阴冷角落哭泣，她对自己说，她还年轻。

　　每晚的深夜，她都在恶梦中惊醒。她梦见那个下雪的冬天，雪花大片大片从她身边飘落，她无法动弹，她被活活埋葬。她曾经期待阳光的出现，但在长期的黑暗中，她绝望......

　　她说，今年冬天她将要死去。

　　有时，她会独自坐在阳台，看着那些雪白鸽子在城市上空尽情飞翔，每次它们从她视线远去消失的刹那间，它们又会突然飞回来。她想，人死了也会不会象鸽子一样再飞回来呢？

　　她的容颜日渐憔悴，皮肤枯萎，眼神暗淡，她老了。空气里蔓延着死亡的气息，冬天即将来临。

　　她开始不停的喝酒。她说，酒能够使人变得更加迟钝。她想让自己对死亡的敏感变得更迟钝一些。喝醉的时候，她会捂着脸，倦缩在阴冷角落哭泣，她对自己说，她已经老了。

　　她无法控制，她无法平静，生命变成了一种负担和罪恶，在那个下雪的冬天，她彻底沉沦。

　　医生告诉她，也许还要过一段时间，或许，明年的冬天才是你生命坠落的时刻。

　　她的表情漠然。雪花在窗外忙乱飞舞，没有阳光的照耀，可是它们需要融化。她笑了，凄凉中，她告诉自己，她的恶梦结束了。泪水在她脸上慢慢凝固......

　　她说，今年冬天她要死去。
