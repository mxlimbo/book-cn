人 生 何 为

若虫

　　多年以前，当地的一位颇有才气的青年作家对我说，以前我总是不知人生干什么，现在我终于懂得人生何为了。当时我对他说的话不以为然，心里似乎还在好笑：在这世间，有谁真正晓得人生何为呢？谁又不都是在浑浑噩噩地过呢？

　　好多年过去了，我的生活竟然还是没有任何改变，人生之路似乎也越走越窄。我记得，也是很多年以前，《中国青年报》曾发表一篇一位女青年对人生之路感到迷惘的文章，题目似乎是“为何人生之路越走越窄”，结果该文在全国产生了轩然大波，并展开了广泛地讨论；人生的目的、生活的意义到底在那里，似乎也总是没有完整统一的答案。不知怎地，随着时光的流逝，随着年龄的增长，对这一问题我似乎感到越来越模糊，越来越弄不懂人生究竟何为了。

　　一个人活着，不知究竟为什么，生活的意义到底在哪里？这是有点可怖的。可是，又有谁说得清呢？在一个人的一生中，我们在大部分时光是麻木地度过的；因为我们无法准确地设计我们的人生之路，一个人在这社会中是渺小的，它就像一粒粉末或是一滴水珠，在社会大潮中根本无法主宰自己的命运；人被动地在这个社会中生存着，只能按照社会的需要来扮演着自己的角色。

　　有的人可能并不同意我的观点，认为作为万物之灵的人类又岂能不知生活的意义呢？一个人活着，不就是在于实现自身的价值，就在于使自己和别人过得更好；有的人认为，人生的价值在于奉献，在于使这个世界变得更美好。诸如此类的大道理，我也明白。可是，我在想，大多数人在平常的生活中，有谁会花许多时间去想过这个形而上的问题呢？这显然是不现实不切实际的。

　　对于我来说，人生并不是我可以设想的，也不是我想怎么过就可以怎么过的；著名作家莫泊桑在小说《一生》中曾经说过：人生既不像我们想象的那样好，也不像我们想象的那样坏。这是一句精辟的语言，它概括我们生活本身的一种基本状态。在我们人类的思维中，生活的变数是我们难以预料的，生活本身远比我们想象要丰富广阔得多。于是，我也就不太刻意去设想去过怎样的生活，还是随其自然地生活吧。

　　说了这么多，有人可能觉得我的人生观是灰色的，同我不敢苟同。假如生活没有一点亮丽的色彩，那么人活着又有何意思呢？是的，一个人活着应该有意义；可是，我们大多数活得都不太如意，我们的愿望总是与我们的现实产生较大的差距，我们无法总是生活在理想的天空中，而只能屈从于生活对我们的限制和压抑。这也就是为何那么多人像我一样沉迷于网络的原因。对于我本人来说，我之所以上网，是因为我在现实生活中感到压抑，感到失望和无奈；有人对我说，你能力不行。对于这些贬损的语言，我只有一笑了之；我想，我没有办法来证明自己，于是我只有借助于网络这个虚拟的媒体来宣泄自己的感情，来驳斥那些无耻者的虚伪和可恶的嘴脸。

　　人活在这个世上的确很难，我的父母经常对我说，人活在这世上不就是混口饭吃吗？他们都是已经退休的人了，经历了无数人生的风风雨雨，说此话无疑是对他们人生经历的概括。是的，在这世界上，无论是过得好还是过得不好，不都是在吃一口饭吗；在这世界上，无论是富人还是穷人，活着忙忙碌碌一生，吃口饭还是第一要义，而生活质量高低则是次要的了。于是，我们应该对人生的挫折、人生的荣辱淡然处置，应该尽快忘记生活中的屈辱、不幸和痛苦，而使自己尽量从中摆脱出来，重新寻求生活的乐趣和生命的意义。

　　人生何为？这的确是一个值得我们深思的问题。从古到今，无数人对此发出了不停地追问，同时也诞生了数不清的智慧和文明；人生何为，这是个历久弥新的问题，每个人一生的行动都会对它作出最好的诠释。我们的确不应该再麻木地生活着，无论成功还是失败，我们都应用自己的努力和奋斗去尽量弥补人生的遗憾和不足，使自己的人生之旅尽可能地亮丽和丰富多彩起来。
