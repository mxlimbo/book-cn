戏 城

blackapple

　　来到这个城市的时间并不长，对它的印象也不好，我很纳闷：为什么我对每个地方的感觉都会很差，难道说我理想中的城市并不存在？想到这，我不由地惶恐了起来。

　　城市并不大，小有小的好处，我可以比较快地了解它。从前我对人生如戏这一说法不很赞同，可是在这座城市呆了一段时间后，我忽然觉得这个说法很有道理，因为我每天都在看戏，每天也在演戏，不但如此，我甚至每天还要编戏。在这座城市中生活，我觉得应该这样，只有这样才能学到些什么。

　　听说这个城市有过一段经济上的辉煌时光，我听别人说起那会儿时，也同时看到了他们脸上流露出的羡慕与回味：身无分文，白手起家，不到一年便身价百万。诸如此类的事举不盛举，我并不怀疑这些事的发生和真实，让我难过的是现在和过去不同了，很难再有以往的时光，这也就告诉了我，我是没有机会演或看这么样一场戏了。我走在大街小巷，总可以寻找到从前的风光与繁荣，但我也深深明白，这些对我而言只不过是一出早已谢幕我再也不能够看到的大戏。我的命运注定我只能看些小戏，发点小牢骚。我并不因我的命运而惋惜，而只是因戏剧的精彩与否而感叹。我再也不能看到那些令人至今仍向往不已的激动人心的戏剧，这岂不是人生的一大憾事？更何况这些戏的演出离我到这座城市的时间相距并不长，一个时间差我便错过了一道风景。

　　但我还要在这里生活下去，只要在这儿生活，就总能听到当时的盛况，总会被那些神往的表情所感动，也总盼望着能重新再来一次，不论我是观众还是演员，我都会觉得值得。也许是因为这些，我虽然不喜欢这座城市，但也没有多少离去的念头，我的贪婪与违心可窥一斑。

　　同我一样，这座城市中还有许多希望大戏开台的人群。我们希望的根源来自于城市昔日的辉煌，我们想着有一天，城市中又重新铺满了黄金和珠宝，等待着我们去捡。虽然在此之前我们必须忍饥挨饿，但总是值得的，我们把希望寄托给明天。虽然我们明白这个希望是很渺茫的，对我们而言，其他城市也是如此，因此不存在希望嫁接一说，我们只想创造希望。大戏在我们看来是明天才演出的。

　　和我不同，这座城市中更多的是怀念大戏的人群。老板们怀念当年日进斗金的时光，业务员们怀念当时小小奋斗便衣锦还乡的岁月，坐台小姐们怀念当初一晚挣一万的日子。希望对他们而言是昨天的，他们是在等待希望。大戏在他们看来不但是明天才有，更主要的是昨天才演过的那一场。

　　但也许我们都错了。

　　这座城市紧紧挨着大海，也就使我们有更多游泳的机会，我躺在沙滩上，看着水中嬉戏的人群，睡眼朦胧，看到一位极漂亮的少女之后睡意顿减。朋友催我快走，我一直舍不得，因为我不知道漂亮少女明天会怎样，是不是还来这儿游泳，我是不是还能见到她。一连好几天，我都到那片沙滩游泳，可再也没见到那位少女。

　　这儿的天气很炎热，夏天尤甚，火辣辣的太阳晒得人如坠烤炉，可是为生活奔波操劳的人照样要在烤炉里行走蹦跳。晚上的时候海风吹过来，整个城市霎时凉爽下来，街上的行人比白天甚至更多出了一些。在歌厅里可以看到莺歌燕舞甚至先生搂着小姐，在酒楼里可以看到拇指大动甚至有人吆喝着下注，可却很难再觅到白天奔波的身影，虽说晚上的娱乐活动是如此地丰富多彩，但他们更多地只能在家中睡觉，对他们来说，或许睡觉是一种最好的打发时间的方法。但想着他们总让我感到没有希望，我也说不清是他们还是其他什么没有希望。

　　写这些的时候看到了王俊先生的一段话，我改些词就权当结尾吧：“城市生活”是场游戏，在游戏中获胜了就趁早逮个机会退出，否则你就只能留在游戏中，被游戏。
