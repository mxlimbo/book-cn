抽烟的女人很寂寞

梦秋 （hy0011@163.com）

　　抽烟的女人是寂寞的，望着手里的香烟和那节已燃烧得长长的纸灰，再看看空气里漂浮的丝丝缕缕的烟雾，她会熟练的用修长的手指轻轻一弹，那烟灰便掉进了小而玲珑的烟灰钢里，然后，再将烟放进嘴里，深而轻地吸一口，随着呼吸而出的便又是丝丝缕缕的烟雾，就好象她的忧伤与烦恼都随着烟雾飘走了，此时，这女人尽管看上去很风度、潇洒，但却是很寂寞的；然尽管是很寂寞的，但心里却是很平静的。

　　我曾经抽烟，并很习惯似地抽，一天没烟也不行，熟悉我的人都叫我烟鬼。但后来又不想抽了，在没人劝说的情况下便莫名地戒了。这一戒便是四、五年，一闻到烟味便恶心，并且挺烦那些抽烟的人。朋友们对我的这一现象总结了两点：一，认为我突然良心发现，女人是不能抽烟的，只有坏女人才这样。二，终于明白抽烟对健康的危害。其实什么也不是，我既没认为抽烟的女人是坏女人，也没对抽烟对健康带来我危害引起重视。反正，我突然不想抽了，就是这么回事。但近来，我又开始抽烟了，并且抽得很厉害，仿佛这几年我从不曾间断一样，任谁的劝告也没有用，我老是拼命地抽。身体也仿佛很需要似地，一点也没有觉得不适，反而从身理到心理都觉得异常地平静。朋友们开始觉得我有些怪，但很快也就觉得正常，便不时地给我散烟。但在所有人已习惯我抽烟的时候，我却很清楚地知道，自己是不正常的。

　　记得我刚开始抽烟的时候，决不是因为哗众取宠，而似乎也是因为一种从身理到心理的需要。那年我十八岁，在初涉的爱河里却狠狠地淹没了自己，尽管我下定决心离开了他，狠命似地斩断了那份青涩地爱情，然而我仍被深深地伤害。我烦躁，我恨，我痛苦，我不能平衡自己，而每到这时，只要我手里拥有一支烟，我便获得了心理上的平静。我似乎一下了变得冷静和安慰，便不再一味地折磨自己睡不着觉，整天以泪洗面。我开始把自己关在房间里抽烟，不与人交往，即使偶尔在和别人谈天，也是嘴里叨着烟，那一刻，腾云驾雾的我尽管看上去很风度和高傲，但确不是真实地我，真实地我已被切底地隐藏。我不断地抽烟，也就等于不断地给自己加防护，我不把自己交出去，也就不会受到伤害，不会有那么多的希求，没有那么多希求也就没有那么多失落和失望。于是，我不对任何人报有目的，别人也休想对我报有希望，一切都是无所谓的，只有烟对我才有着至关无比的重要。那时，一个人尽管很寂寞，但心却似乎开始平静。

　　后来我遇到了他，他在我吞云吐雾的时候，给我点烟，并静静地看着我。无论我怎样地冷漠、无理。不在乎、烦躁和无所谓，他都在一旁，在我抽烟的时候，给我打燃火机，把烟递到我面前，然后仍静静地看着我。没有繁琐华丽的语言，也没有热烈激情地表示，只有日复一日，年复一年的宽容、理解与照顾。我那颗顽固，因受伤而变得冰冷的心也渐渐开始融化了。在他朴实、稳沉、耐心与迁就的关怀下，我不经意地开始渐渐放纵自己。我竟然在他面前开始撒娇，竟然丢下了傲慢、苍伤、无所谓的外表，重新学会了期盼和等待。我慢慢地恢复了自我，那个热爱生活、充满自信的、感情丰富、渴望爱与被爱的真实的我似乎又重新苏醒过来。于是，我开始觉得抽烟渐渐不适，从身体上，我开始感到，头有些昏。嗓子也发干，浑身都似乎不舒服。从心理上，我似乎已不再烦燥，不再充满恨与不平，相我：我心里装满了爱与平静，每天都在甜蜜的梦里睡去，在平和的心情中醒来。渐渐的烟成为我多余的负担，直到后来，我不但自己不抽，并且一看见别人抽就烦，因为一闻到烟味，我就有些昏烟。朋友都说我是先前一支接一支抽得太凶了，那时没昏烟，却留到了现在。总之，我不再抽烟了。

　　戒烟的几年里，我经历了许多事，谈恋爱到后来的结婚。当然我的夫仍然是让我戒掉烟的那个他。然婚后的日子，又开始让人困惑不解。他整天似乎都在忙碌。每天除了上下班，就是看电视，然后就是一躺上床就呼呼而睡。我在他眼里渐渐成了一一个隐形人。他经常下乡检查一去就是几天，我孤孤地守在空旷的房子里企盼他早点回来，然他一回来，最关心的不是我，反而是他的电视新闻和节目。即使他在家，也经常是在你把饭菜煮好后等他回来时，才接到他的电话，今天有事不回家吃饭了。于是，我又只得孤零零一个人在家，上顿吃冷饭，下顿吃冷饭。耳听着别人家在夜幕降临时在饭桌上演奏着晚餐曲。眼看着邻居大人、小孩坐在饭桌上享受天伦之乐。而我······可我又是一个在乎家的感觉的人。慢慢地我们开始吵架，到后来很凶地吵。他说他很累，工作上等等的事，他说我不理解他。我也确实感到我们之间缺乏沟通，我试着去改变，也做了许多努力。然一切的一切都是徒劳的。他在意工作，领导、同事、朋友之间的所有感觉，就是不在意我的感觉。下班后，他可以因一些可有或可无的所谓公事，和同事、朋友喝得大醉而归。回到家里，倒床便睡。他何曾考虑到我还在苦苦等他回来，陪我吃晚饭。（因我有一个人吃不下饭的习惯）有时，等到夜里11点他才回来，一进门，便对他说，求他等我吃了晚饭再睡。然而，当你把饭菜端出来还没来得及吃时，喝得太多的他，已在沙发上呼呼睡了。每到这时，我便按捺不住怒火与他吵。然得到的却是我不可理喻，难道他连睡觉的权利也没有了吗？我也确实开始变得有些不可理喻了。每次他打电话说不回来吃饭时，我就忍个住大发雷廷，坚决不同意。即便有时真是公事，我也一概不管，让他也下不了台。我也不知我为什么会变成这样，反正他从不在意我，我又何必去管他，但心理却总认为，他只是在应付我，骗我。有时，他不回来，我就拼命地给他打传呼，到后来，他连传呼也不给我回了，任我不停地打，我的心悲哀到了极点，我终于把自己至于了多么可悲的境地。我跌倒在沙发上，心里的痛苦到了极点，那一刻，我真想立即死去。我开始反醒自己，这是河何苦，真那么在意他吗？同时，我开始恨我自己，为什么要对他报那么多爱与期望，以至于把自己搞得如此狼狈和不可理喻。

　　在难以入睡的深夜里，我打开火机，点燃了一只烟，深深地吸了一口，却莫名地觉得好受多了。于是，我重操旧业，开始狠狠地抽烟。然而，老朋友毕竟是老朋友，它没有让我失望，它似乎让我找到了另外一个我。

　　当电话铃再想起的时候，我会从容地拿起电话，无论他说什么，甚至于晚上不回来睡觉，我也会很轻松地说一声“OK”。随着手里抖落的烟灰，而轻松地将电话挂上。jy

　　烟又占据了我生活中的主导地位，它让我平静多了，我再也不会无端端地生气和发火，也再也没有让自己变得难以理解。我看上去正常多了。

　　许多个孤独的深夜里，我在灯光下，床头上，一支接一支地抽，我不再去企盼门外的脚步声，和开门的钥匙。他们对于我来说己经远没有烟重要了。

　　在这样的夜里，不知有多少女人在抽烟。但抽烟的女人是寂寞的，然心却是平静的。
