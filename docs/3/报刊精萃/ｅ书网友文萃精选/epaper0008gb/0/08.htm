雨 夜

思敏 （towzh@163.net）

　　来信收悉，我亲爱的朋友，谢谢你给我寄来的《雨衣》。

　　这是我第二次读痞子蔡的书（第二次亲密接触？^_^），又是一部以悲剧结尾的故事，人们往往为失去的或得不到的去大动感情，看完了，心中有说不出的痛，不是因为雨姬和加藤智，不为他们的结局惋惜，我不能简单到为虚构的人物大肆感慨。

　　只是想起了我的女友。

　　爱恋一个人，一定要隐藏自己的情感？如果失之交臂，那，留下的是什么？

　　我认识我的女友很简单，我们在同一个地方上班，那时我听命与父母的安排和一个他们喜爱的女孩交往，我感觉得到她有与我同样的无奈，我们互相喜欢，但谈不上爱。

　　我们经常外出，不为谈情，只为暂时逃离工作、家庭。

　　我的女友并不知情，在她无可救药的爱上我之前只想着她的好友能常来看她，轻松的、无聊的打发她的日子。

　　压抑的生活，紧张的工作使我和那女孩频繁外出，回家时不忘买只玫瑰给她来安慰我们可怜的父母。

　　一切都在那晚改变。

　　生活中改变人的往往就那一刻，甚至一分钟。

　　我不明白那晚为什么我在那女孩的房中会留到那么晚，一夜嘻闹，就我和她，我喝了酒，那是她几天前为招待朋友买的，饮料酒，20度，但我不胜酒力，一瓶下肚已经满脸通红，直到她告诉我现在已经3点钟了，她从来没有留男孩子到这么晚。

　　我知道我该走了，口中残留的酒气令出租车车司机一路不敢说话，可我不愿回家，我知道老妈不是出租车车司机，她会把我外公的老爸是怎么死的不厌其烦的再告诉我一遍。

　　来到上班的地方，我的女友在值夜班。我还没有睡意，坐下来玩起电脑。

　　许久突然想起不见我的女友，想她可能在休息室，不再理会。

　　突然听到呻吟声，月黑风高的，心中一紧，一丝寒意升起，酒醒了大半。寻声而去来到休息室，敲门，没回音，试着开门，从里面反锁了，一会又传出呻吟声，心中大急，撞门、砸门半天，总算里面开锁了，急推而入，只闻一股酒味，女友蜷缩一边。接下来叫车、急诊一通忙活。

　　总算安全了，做下来细想这都是怎么了？

　　隔日，她告诉我她爱我！我不明白、我不知道、我脑子一片空白！

　　她真幽默，我想，这种时候还能开玩笑，我呵呵的笑着，希望她告诉我是那个男人把她伤害到这种的步。

　　她哭了，她说我不该酒气冲冲的跑到公司，不该那么晚去，不该在她在的时候，一定是和那个女孩喝的酒，她说：那么晚你还精神那么好，一定刚从她那回来，还哼歌！还玩游戏！

　　而且还喝了酒！好，我也喝酒，我也能喝酒！

　　我的朋友，我是无辜的，公司没规定喝了酒就不许去，再说我没在上班。

　　我辩解道。

　　她还在哭，女孩子一哭，上帝也没了主意，我木木的站在一边，忘了她刚才说她爱我。

　　她爱我？为什么我不知道，是的，她常烧东西给我吃，我的体重一半是她造成的，另一半是因为不锻炼的缘故，对了，

　　自从她来了，我连一杯水也没倒过，可，这是因为她爱我？！

　　我的头好象疼起来了，我呐呐的说，靠近她坐了下来。

　　她看起来神色憔悴，没有了红晕的脸上显得更加白晰，我俯身检起满地的纸巾，我得找点事做，我想。

　　我感觉身后有双眼睛在随着我的身影晃动，转身我捕捉到那双眼睛，四目相对时，我的心碎了。

　　弯曲的睫毛被泪水打湿，辛苦的贴在眼睛边，一只单眼皮，一只双眼皮无力的张开着，充满血丝的眼球直愣愣的看着我，上帝不该教会女人哭泣，我想。

　　昨天你辛苦了，她说。

　　没什么，到是护士问我怎么了，我说你好酒，遗传性的，我低下头避开那双眼睛说道。

　　突然她咳嗽起来，我忙起身去倒水，心想，你也有倒水的时候。见她的嘴角动了一下，是微微上翘的感觉。

　　该为你弄点吃的堵上你的嘴，她再次咳嗽说道。

　　好男人容易饿，没吃的时候只能贫嘴，我笑道。

　　哎，就此我那可怜的父母又开始忙碌起来，张罗他们认为必须的零零碎碎去了。

　　今晚上海很冷，细细的小雨象绵缠的情人，几天了，始终挥之不去。

　　我裹上所有看起来能保暖的东西，甚至试图卷起地上的地毯，很快我还是放弃，因为我清楚的记得我已经几天没吸尘了。

　　脚很冷，看着今年春天买的皮鞋，布满灰尘的鞋面上有几道擦痕，看来坚持不到明年了，我想。

　　现在明白为什么女友说：你会后悔的，你该买双运动鞋。

　　那天经过小泽春她是这么说的，没想到我还记得，没想到我真的会后悔。

　　曾经有份真诚的爱情摆在我的面前，我有好好珍惜，如果老天愿意给我再来一次的机会，我会对那姑娘说：爱我不会太辛苦，告诉我你爱我，让我们在真爱中完美人生，不要虚度人生来寻找真爱。如果上苍非要我选择谁先开口，我会说：

　　我愿意！如果上苍同意，我将一万年不变。

　　　　　　　　　　　　　　　　次致

　　　　礼

你的朋友：思敏
