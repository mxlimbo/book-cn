听说他死了

无情

　　他象一墫石像般坐在那里，剑在他手中安静的躺着，无声无息。天空划出一道幽暗的色彩，无星无月。

　　剑神谢小峰死了。

　　这个消息在今天凌晨4：44分传遍了整个江湖。没有人不知道。听说剑神死于意外，在吃夜宵时不小心咽死，本来，今天应该是剑神的六十大寿。

　　他练剑三十余年，挑战无数名门剑客，从无败绩。在他四十四岁那年，惜败于当时名满天下的剑神谢小峰。那天，黄历上记载，初四日，无风，宜见光，忌见血。

　　他不服。他对自己发誓，终有一天，他要打败谢小峰。

　　之后的几年，生命没有了黑暗白天。他不停的练剑。最后他决定，在剑神六十大寿那天，打败他，名扬天下。每次想到这个时候，他就莫名的兴奋，他感到自己全身都充满了力量。他必胜！

　　可是他不信。剑神死了，他的尸体象他的剑一样安静的躺着，无声无息。

　　他已经没有机会打败剑神！可是他不服。

　　江湖上近日又涌现出一批名门剑客，他们不停的向他挑战。他们说，你是剑神，我们都败了。其实他明白，江湖人心目中真正的剑神是谢小峰。他永远都没有机会超越。

　　他开始堕落。他没有对手。他把自己关起来，黑暗中，他的手已经有些不听使唤的抖动，他的白发象云朵般被风吹散，他的皮肤扭曲，精神恍惚，眼神枯萎。生命随着时光流逝。

　　他还是不服。可是他已经没有机会。

　　 他象一墫石像般坐在那里，剑在他手中安静的躺着，无声无息。天空划出一道幽暗的色彩，无星无月。

　　他死了。

　　这个消息在今天凌晨4：44分传遍了整个江湖。没有人不知道。听说剑神死于意外，在吃夜宵时不小心咽死，本来，今天应该是剑神的六十大寿。
