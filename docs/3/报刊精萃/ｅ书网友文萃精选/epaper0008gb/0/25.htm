无 解

因为所以

　　这件事发生在我的高中时代。

　　你知道的，高中是个什么样的地方，尤其是重点的那种。

　　 那时班里有一个女同学，和大家并不太一样，但仿佛又没什么差别似的，就那么一直被我记住。

　　我在她的左后侧。看着她。

　　她平时是个很安静的人，常常在课桌下偷看小说，也常常被抓住，很笨，又很倔。班主任是个三十开外的男人，骨瘦如柴，个子又高，是个教历史的，生平最痛恨的就是学生不听话棗我看他只差没拿以前私塾先生的戒尺了！每一次被发现，书总是被撕碎的。

　　那时她一般都不敢回嘴，不像一般的差生敢和老师对着干；也不辩解，就一直呆立着，静等暴风雨过去。很多同学觉得她古怪：不生气，不理睬，总和班里任何人都不协调棗她几乎没有什么朋友。

　　其实我知道那时侯她是害怕的，她的手指总是在神经质地扭拧着什么东西：要么是衣角，要么是书的碎片，要么就是那些细瘦的手指互相扭绞着棗她害怕所以一直眼光无焦点地看着别处。

　　上英语课，老师常常发现她又走神了：窗外的天气变化、树掉了几片叶子都会令她一转眼忘了自己在哪里。那时侯他就会叫她起来翻译或念诵刚刚讲到的句子，她永远只有结结巴巴念着或在旁边同学提醒下如梦初醒。

　　教室里老是有一片笑声。

　　但我知道她的秘密，那个秘密总是能令她涣散的焦点突然地凝聚，然后悄悄地追随着，然后在极短的时间里熄灭。

　　那个男孩子是同年级理科班的高才生，拥有几乎那个年龄的女生喜欢和崇拜的一切条件，很多女孩子都在暗恋着他，整个情节就好象一部很菜的日本少女漫画。嗯，就是这样的，他有一副人见人爱的笑容，体育万能，头脑又好，对女生们又很体贴，无数女生都对他抱着幻想棗就是有这种人的存在！

　　她也是一样，后来我知道她和他在高中入学的那天曾经有过一次交谈，之后整个高中时代相互间便没有过任何一句言词。她喜欢他，三年。

　　从她眼中发现这个秘密的那天晚上，我又做了那个梦。

　　梦里有一座房子，很欧式的红屋顶黄灰色墙面的房子，孤零零地立在很远的地方，我要穿越过大大小小的田埂以及小溪、杂树丛才能到达那里。去时的路还好，干燥、结实，土筑的田埂边长着丛丛簇簇的杂草，有点枯萎。我耐心地在那如蛛网般凌乱的小径中寻找，寻找通往房子的路。

　　终于我到达了。那房子看上去并不大，并且在黄昏的阳光下略略反射着微弱的疲倦而悲伤的光。

　　门锁一触即开，屋子里有淡淡的霉味儿，灰尘和枯死的花的气息；透过破碎的彩绘玻璃，阳光在屋子里留下一个个小小的光斑。半明半暗中，我看见有许多和我一样大的小孩棗总是小孩，一直都是，自从我上小学时第一次做这个梦以来，一直都是小孩，包括梦中的我。那些小孩子面目模糊如同身边任何一个人，都很陌生，可恍惚中又很熟悉：就像我的亲人，即使是死去了，也不会感到恐惧。

　　突然有什么东西闯入棗总是有东西，第一次是猫之类的，最近一次又像是一匹马棗小孩子们四下里慌忙躲藏起来，手足无措间我莫名其妙地撞破一扇窗户拼命地逃跑着，跑着，感到心脏仿佛就要从胸膛里迸裂出来了棗转眼间，我看到有一个小小的影子就跟在我身边，蓦然回首，那张脸棗是我……

　　醒过来时是半夜。

　　后来上大学时我又做过一次，那时和同寝室女生挤一张床，睡得很不舒服，醒过来时我不敢扭头看身边人的脸，就那样一直睁着眼望着无尽的黑暗，一直到再次睡着。

　　那个女孩子一直就那样重复着她一生仅有的三年：早自习，悄悄地望着窗外，因为他一定是几乎要迟到了才会急急赶来；课间操，永远懒懒地提不起精神；上课，看小说，偷偷画喜欢的漫画，和几个比较要好的女生聊天；放学时她希望能和他在一起，哪怕远远看着他仅仅十分钟也是幸福，但若真的有一天他来了，我相信她一定会飞快地逃走……

　　她不是你想象的那种样子：温柔美丽或者很另类，不，她不是。

　　她总是显得别扭而格格不入，过于苍白，有点浮肿。长发，老扎着辫子，衣着简朴到寒酸的地步，我见过她最合身的一套好象是在那个夏天的百衬衫和半新的牛仔裤。

　　不，她不漂亮，也许连我也仅仅记住了某些神态和眼神，受伤而又自尊的坚韧。别人怎么想我不知道。

　　日复一日，我看着她，缄默，孤独，即使笑也不说明开心的她。

　　我的心在某个地方痛着，某种倾诉的欲望日甚一日地积累积累……

　　三年就那样过去了，她和他之间没有发生任何事情。她一直在单恋、暗恋，并且拒绝着恋爱棗虽然有朋友听说他有努力追过她。就像她后来告诉我的一样：她如果那时和他在一起了，今天的我也就不存在了。我机械地点点头。她后来也有喜欢过别人，但都很短暂棗她已经明白了，只要她一旦喜欢或依赖着谁，哪怕只是在精神上的一点点依靠，那个对象总会很快地消失了踪迹。数不胜数的微小事件将这道堤筑得日益加高，控制着她心中满盛的爱和痛苦。

　　她就那样销声匿迹了。

　　他不算她的初恋，这我知道。也许，她会在今后的生活中找到她的精神之寄托和依恋的真正恋爱，但对象不一定是人。

　　我还知道一件事。

　　她是我的初恋。即使时至今日，我仍然会想念起那个别扭、苍白而普通的女同学；在那些重复的梦中，也许有一天，那屋子里的小孩子中，有一个会长着和她一样的脸。
