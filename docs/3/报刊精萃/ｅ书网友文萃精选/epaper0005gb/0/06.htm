《肖申克的救赎》的感悟

kevin

　　有一次我在一本杂志上看到一篇文章，作者说他曾买过一张盗版影碟，叫《刺激1995》，看后感觉很不错，后来才知道那就是著名的《肖申克的救赎》（THE SHAWSHANK REDEMPTION），感慨颇深。几乎是同一天，我在一分报纸上看了一个排名，说美国人评出的90年代最好的十部影片中，就有它。这是一部什么样的片子呢？

　　当我从朋友那里借来这张影碟，放进影碟机的时候，心中还是猜不出它会讲一个什么样的故事。爱情？战争？美国大片历来如此的。

　　说实在的，影片的前几分钟并没有给人什么视觉的冲击，音乐怪怪的，画面一种很乏味的老旧感。再过了一会儿，主人公进了鲨堡监狱，我想，这又是一个冤狱片，能比《因父之名》高明到那去呢？除了监狱长的那句“把灵魂交给上帝，把狗命交给我”的话让我感觉很酷外，别的我还没有什么感觉。

　　然而再往后看，主人公安迪的监狱生活逐渐展开，我越来越感觉这部影片的不俗。它叙事很平静，娓娓道来，像一个历尽苍桑的老人（事实上，影片就是由摩根・菲曼演的安笛的狱友瑞得讲述的）。片子的语调充满了朴实的睿智，对生命的豁达的态度，对命运的慨叹。在那流畅自然的电影画面后面，我总能体会一种温暖的苍凉。我不知道这个词是否用的很矛盾，但那就是我的感觉。影片并没有一味表现监狱的残酷啊什么的，那太平面化。影片的伟大之处在于将社会和人生浓缩在了小小的监狱中，监狱中也有各式各样的人，有友情，又仇恨，有残酷的现实，有光辉的梦想。

　　导演没有刻意去渲染某个人，随着影片的一个个小高潮出现，不知不觉中，安迪、瑞得、狱警海利、监狱长和其他每个人就在你脑海中凸现，愈来愈清晰、具体，每个人的痛苦与骄傲是那么鲜明。修房顶安迪争得片刻自由的一幕，盖图书馆及盖成的那一幕，安迪在监狱里放《费加罗的婚礼》的那一幕，安迪教别人学习中学课程的那一幕，都是那么感人至深，而且这种感动像水，一点点渗入了我的神经，我的感情。

　　当然，最感人的还是安迪成功越狱的那一幕。安迪穿走了监狱长的皮鞋，是那么的大快人心。安迪从地道爬出，来到排污管处，外面正在电闪雷鸣，每当天空发出一声雷电的劈响，安迪就用手中的石头砸一下排污管，直到砸出一个洞。那刺耳的声音，不禁让我想起《命运交响曲》开头的巨大的音响。看安迪越狱的过程，相信每个人都会为那种对命运不屈不挠的精神所感动。当安迪越狱成功，站在雨中，让雨水冲涮去身上与灵魂上的污秽，我由衷感到一种战胜命运的痛快淋漓。

　　影片中包含了太多的东西：生存、梦想、命运、希望、友谊、悔改、正义、罪恶等等等等。片中的对白也非常精彩，幽默、简练、睿智，充满了哲理与丰富的感情。而迪姆・罗宾斯和摩根・菲曼的表演也是超一流的。总之这部电影是一部不该错过的电影，我这里说的不及全豹之一斑。

　　写到这里，想起瑞得在安迪越狱后，思念安迪时，心中说：“有的鸟毕竟是关不住的，它们的羽翼太光辉了！”是啊！
