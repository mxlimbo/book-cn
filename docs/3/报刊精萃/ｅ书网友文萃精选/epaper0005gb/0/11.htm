来生再续缘

童可人

　　与子健谈恋爱是在大一的时候。

　　自从第一次看了他在校篮球比赛之后，我的心就不平静了，当寝室里的同学谈论他的时候，我总是莫名其妙地心跳加速。那时我最喜欢的颜色是粉红，几乎所有的衣服都是粉红色的。

　　我开始为了能够在他面前出现而情愿绕一个大圈去图书馆，因为我们星期二，星期五下午都没课，而他们又是固定在那个教室上课。从他们的教室可以把窗外的景物看得一清二楚。

　　几个星期之后，我发觉，每次在我经过他们的教室到了图书馆，过不了多久他就会出现了。连续几次都这样，我开始有点欣喜。为了证明他可能真的为了我而去，我故意不经过他们的教室，果然他没出现。以后，我又用不同的方式，每次用不同的时间经过，而他总是在我达到3分钟之后达到，而且我一出现就可以听到他们教室发出的口哨声。就这样，大半个学期我们都在进行着“无声的约会”。

　　终于，我所盼望的事发生了。那天是星期五，快到图书馆关门的时候下起了倾盆大雨，等雨渐渐小的时候，只剩下我们两个人了。好心的图书管理员给了我们一把伞：“男同学，你送女同学回去。”就这样一句话，改变了我们两个的生活。我们第一次在同一把伞下，居然能够象个老朋友一样谈笑。以前从图书馆到寝室那条漫长的路，在那天仿佛几秒钟就到了。“下次我们一起去还伞。”到了寝室楼下，他很自然的说道。

　　就这样，我开始了生命中的第一次轰轰烈烈地恋爱。我们拼命地学习，拚命地恋爱，拚命地玩。有一次我问他是不是故意逃课去图书馆，他也马上反问我是不是故意走远路经过他们的教室。结果是我们两个同时哈哈大笑。他说，那时他一看到粉红就以为是我了。

　　这样快乐的时光过了一年，我22岁生命中所有美好的记忆都在这一年中。我们一起去海边，去爬山，去打球，偶尔也一起逃课跑到离学校很远的地方去看书。可是就在我们庆祝认识一周年纪念日的两个星期之后，子健消失了。那天我们说好了6：00在图书馆等，到了9：00也不见他的人，我开始去找他，因为他从不失约。一个一个教室找，打他寝室里的电话一直没人接。第二天，第三天还是没他的消息，我开始不去上课，整个城市地找，从我们以前常去的地方一直到我们只是路过或者提到过的地方。还是找不到他，最后我终于决定去他们的寝室，我以为他是因为某种原因躲在寝室不肯见我，因为他爸爸妈妈在日本，他在这个城市是没有亲戚的。因为女生不可以进男生寝室，我趁宿舍管理员不注意，一口气跑上七楼，听到的却是他自动退学的消息，子健一向成绩优异，他为什么要退学？所有认识他的人都说不知道。我只觉得脑袋嗡的一声，什么都不知道了。之后，我发了三天三夜的高烧，退烧之后我开始变了一个人，不再奔奔跳跳，我整天埋在书里面，连周末也不例外。

　　一个月以后，我开始恢复正常生活。可是在那个下着大雨的夜晚，我却不能控制地一直走到以前我们一起去过的博物馆，因为子健极其喜欢欣赏梵高的作品，我们经常来这里。刚要进门的一刹间，我看到了他和一个极其瘦弱的女子再一起，他显然没想到会在这遇见我，很是惊讶，而我却异常地平静。他身边的那位女孩似乎是大病初愈的样子。

　　“很高兴再见到你，祝福你们，子健。”我强忍住在眼眶里打转的泪水。

　　“你好。”“这就是我跟你提过的我的好朋友童可人。”他轻轻地对那女孩子说。他一边说已经一边走开了，把伞明显地偏向她，极其呵护。我的泪水排山倒海一般的下来，我对我们的爱情终于彻底的绝望了。或许真正的爱情是一种奢侈品，就象是雨后的彩虹，绝对是可望而不可及的。那种王子与公主天长地久式的爱，只是童话故事里面的片段而已，不可能会在现实生活中实现的。

　　以后的2年里我开始忙于参加各种各样的活动，记者团，学生会，舞蹈队，能参加的我都参加，学习上的成功终于让我渐渐忘了那场轰轰烈烈的恋爱。

　　今年我夏天大学毕业，忙于准备出国留学。我以为我已经彻彻底底地忘了他，但是这样一个下着大雨的夜里，我根本不能控制自己的思绪，我不能停止想他，想那段刻骨铭心的日子。我打开电视机，一频道接一个频道地看，忽然我发觉被采访的那个女孩有点眼熟。

　　对了就是她，2年前我看到的，跟子健在一起的。我象被针刺了，马上关掉电视。但几乎是在同一秒我又打开了电视。

　　“请问你现在最感激的人是谁？”记者在问她。

　　“我最最感激的人是我的老同学，林子健。为了给我鼓励我做最新的治疗，他承诺我病好了，他跟我结婚。其实，那时我也知道他是不爱我的，但是我爱他，所以我……他甚至放弃了他的学业和他最最心爱的女朋友……,原因只是因为在日本的时候我爸爸曾经帮他爸爸还了债……”她已经开始抽噎了。

　　“那既然你现在的病已经好了，他也可以再去找他原来的女朋友啊。”

　　“子健说他当初已经让那个女孩伤透了心，他再也不会去打扰他了，但是他这辈子是不会再结婚了，他今天乘飞机去日本了……”她实在是说不出话了。

　　我的心忽然如刀绞般的的疼痛，眼角划下的是冰凉的泪水……

　　上帝啊，你好残忍。

　　子健，我等你会来，永远我等。假如今生不能，我们来生再续缘……
