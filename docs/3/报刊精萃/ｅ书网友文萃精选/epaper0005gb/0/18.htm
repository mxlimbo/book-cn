当我想说爱你的时候

司雯

　　又一次，在噩梦中被惊起的我，不愿再合上双眼去面对漆黑一团的世界。

　　如果这世界仅仅只是黑暗也就罢了，或许从来就十分胆怯的我仍会继续的前进。但我在黑暗中摸索前行时，周围却总会突然冒出一些莫名的东西。虽然黑暗包围着我的一切，虽然我无法了解被黑暗掩饰着的危险。可我却清楚的感觉到，如果再向前走，我将堕入一个未知的深渊。

　　久久的，只是木然的呆坐在一个阳光无法曝射到的角落。生命虽在延续，但我与我周围的一切似乎都凝固了。我在回忆，正在用曾经真实而如今虚渺的回忆来满足一颗，在现实中总是失望破碎的心。

　　我似乎永远都生活在你的影子里，永远被黑暗所遮掩。明媚的阳光对我而言，永远只是奢望与不现实。

　　虽然我渴望阳光能给我和大家一样的一切，但为了你的快乐，我只能默默地隐藏在众人视线的角落里。在不经意间，偷偷的感受你与太阳的温暖。一瞬间，只有短短的瞬间。如果被大家发现，我还要承受你们的奚落与嘲笑。

　　我总在压抑着自己，压抑着一颗深爱你的心跳动的不要那么激烈。虽然你听不到我的心跳，但你可曾感受到，你的身后总有一双关切的眼尾随着你。你的一切无法逃离我的视线，你的梦想是我人生的目标。

　　我似乎已成为你爱的囚徒，爱的枷锁将我所有的感觉都和你的所连接，你的身边就是囚禁我的监牢。虽然如此，可我却是心甘情愿为你犯罪，受尽爱的煎熬。

　　我想你一定没有感觉到，在爱河的对岸有一个寂寞的人，总在默默地凝望着你。虽然他在你的对面不停的挥手，不停的喊叫，想引起你的注意。可或许是因为你我相隔太远的缘故，你终没有发现。

　　你需要的，是一种浪漫的爱情。这注定你是一个容易动情的人，我无法改变这一切，也不想改变。

　　而时间永远是感情的敌人。

　　我似乎不能再等待，爱你的欲望在本已不平静的心里愈加膨胀。我不能再作一个你精神上的守护者，我不能再用幻觉麻痹自己。酒精虽然能让我暂时的忘记爱你给我带来的苦痛，可在酒醒之后，留给我的或许就只有永无休止的思念。

　　我需要爱你，需要用你的爱弥合我干渴的心裂开的伤痕。

　　我鼓起勇气，来到你的身边。

　　可当我想说爱你的时候，你却用你纤细的手指轻轻地在嘴边摆了摆，用泪水封住了我想爱你的心。
