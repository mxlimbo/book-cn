初 恋

阿煜

　　我一直觉得那就是我的初恋。

　　那年我十九岁，单纯又燥动的年纪，大学生活给了我渴望已久的自由和行为艺术的空间。

　　于是，开学不久，我们几个就被老师和同学归为“另类”学生，衣着古怪，抽烟，不上课，作业做的又快又滥，和自己不喜欢的老师形同陌路。那种生活让我感觉自由，使我做了太长时间“学奴”而有些扭曲的性格得到了纠正。我开始觉得生活有点儿意思，但仍时常感到痛苦。

　　这种痛苦无根无源，不请自来。新人新事，亲情，友谊，含糊的爱情都不能让我远离这种无聊的痛苦，我想也许我是天生忧郁的孩子。

　　我就是在这时碰到他的。那是一个凉快的夏日夜晚，姑娘们的花裙子看上去有些神秘，空气里飘动着青春、浮燥和爱情的味道。那天我和一个他说他喜欢的男孩一块去爬山。山就在我们学校旁的公园里，黑黑的一片，我们象两个小偷一样疯狂地往上爬，很快爬到了山顶，那个人的鞋已经坏了，我忘了他和我讲了什么，好象有点儿山盟海誓的意思。然后，我们就下山了。

　　后来我回女生楼。刚要进去，花坛上有人叫我“阿煜”。我回头一看是六六，六六是我们班同学，她和好多人一字排开坐在花坛上，象在看电影，地上有好多啤酒瓶。

　　“干嘛？”

　　“过来坐会儿。”

　　我犹豫了一下，就过去了，和她坐一块的都是我不认识的人。六六给我一支烟，我们聊了一会儿，这时最那边的一个男生大声对六六说：“介绍一下你的同学吧。”六六说：“噢，都忘了。这是阿煜，这是韩冬。”我说你好，他也说你好。过了一会儿，他说阿煜你坐这儿来吧。我问为什么，他说你坐过来男女就平等了，否则你们那儿阴气太重了，我说好呀，就坐了过去。

　　他穿了一件蓝格子衬衣，不身是休闲短裤，黑色的耐克，长长的头发遮着眼睛。他那双眼睛属于那种溢满了东西的眼睛。那天晚上我们聊了很久，什么内容我已经忘了，什么情绪也因为时过境迁而无法站在公正的角度叙述了。我只好把那时的日记翻出来，上面是这样写的：

　　“昨天我碰到了他，韩冬。他的出现令我感到惊奇，我从未奢望过这种人的出现，我以为我的忧伤、迷惘是孤独的，永远的。不能说他和我很象，只能说我象他的一部分。他的思想要比我宽广、坚定得多，我觉得在他的面前我就象个小孩，我喜欢静静地听他说话，不管他说什么。因为我知道他说的一定是我想听的，也是我想说的。我还喜欢他的玩世不恭和他的那种欲言又止的语调，喜欢他弹我的脑门，摸我的头发和脸……”

　　（抄完这段话，我觉得心里很难受，差点儿哭出来，我不太清楚我想为什么哭泣，是因为回想起了彼时彼地的情景，还是为19岁时我无法理喻的单纯？）

　　这以后的几天我象着了魔一样一直想着他，渴望再次见到他，听到他的声音。六六跟我说他是北京流氓，说他昨晚还在楼下和某某调情，他睡过的女人比他的头发还多。我听了挺不高兴，但没办法，我仍想着他。

　　我又继续写日记。

　　“寝室里没有人，没有电视，也没有飞扬的音乐，只有满屋子的杂物和受伤的我，痛苦，寂寞，无聊。我想看书，想抽烟，想找人说话，却没有精力。稍不留神，他就钻入我的脑子，肆意折磨我。他到底是个流氓，还是我的亲人？他失踪了几天，又去构造他的世界，又去远航，却把我扔在海里挣扎。我爱上了一个我应该去恨的人，荒谬可笑。我对他太不了解了，也许我了解的，只是他的一副面具。也许我现在想他，只是想扯开他的面具，看看他是谁。

　　我抽着烟说着胡话，杉、漾也胡乱地听着，笑着。我四处看看，他没来。过了很长时间，他还是没来。花坛上坐满了我厌恶的厌恶我的人，我坐在地上抽烟，一根接一根，地上散乱的烟头，象我此刻的心情。那张纸条，写着他寻呼号的纸条，当我需要它时，才发现它已被我撕的粉碎，就象他再发现我时我已经破碎的不可收拾。生活就是这样无奈。吐了，差点儿把心都吐出来，胃里一片空虚，脑中也一片空白，只有他的影子在晃，睡在床上，我感到我的思想和身体正在分离，我的身体飘向天宇，无依无靠，一直在飘，我的思想却他他攥住，飞不起来。

　　晚上，我去找他。我象一个幽灵一样在街上走着，寻找。有骑车的男的朝我看，以为我是“鸡“。有狗朝我猛叫，以为我是小偷。我摸摸自己的心跳，以为自己疯了。夜很黑，我根本找不到他的小屋了。我越走越远，我筋疲力尽，有一个近乎全裸的男的朝我吹口哨，我害怕了，打了车就回去了。他知道我为他我做的吗？听音乐听到凌晨三点，我渐渐在音乐声中死去。

　　我想那是我第一次不可抑制的爱上一个人，那种激情与单纯让我现在想到都感到羡慕，仿佛是另外一个人。

　　然后我找到了他。那是在一星期以后。我在日记中写道：“一见到他，我破碎的心就愈合了。“我不再感到烦燥与不安，很快安静了下来。我们一起吃饭，聊天，散步，读他的小说。小说写得是个有点儿文化、时常思考并感到痛苦的流氓。我喜欢他的小说，喜欢和他有关的一切。和他在一起是我的那种感觉我叫它“幸福”。这种幸福维系了两年时间，在18岁至20岁的我的眼中，他始终是个扑朔迷离的家伙，无休无止地急争吵，没完没了的姑娘夹杂着这种刺骨的幸福构成了我和韩冬奇怪的感情。后来毕业了，我们分手了。

　　我一直觉得记忆应该是美好的，即使有痛苦，也应该忘记它。是我下意识地已经忘掉了这段初恋中灰色的部分，还是我不敢去触摸已经似乎养好的伤口？不知道。

　　我想当我再回忆起这段感情时，也许想到的不是他，而是自己。因为我知道我再也不会象19岁时那样单纯、疯狂、执着，再也不会象飞蛾扑火那样地爱上一个人。

　　我很难过。
