海 鸥 与 海

司雯

　　岛是海中的疮疤，难以抚平。

　　岛是海中的风景，不应抚平。

　　增添几座小岛，就会增添几分海韵。

　　海鸥与海不可分离，她是海的同伴，而海中的小岛是海鸥的落点。海固执，任性；海鸥自由，无缚；然而海却是永恒不变的。

　　海风是海的思绪，海鸥在这清新的海风中飞旋，徜徉，很是惬意。

　　然而有一天，大海起了风浪，海鸥受惊尖叫着飞开了。海气愤，他不想海鸥在他心里留下任何印象，甚至她曾经有过的东西。海鸥撞击着小岛，想把它撞碎，想把它淹没。

　　然而当他发现这一切都是徒劳时，他平静下来，伤心的抚着小岛，用泪水轻吻着他冲动后在它身上留下的伤疤，但这伤疤也许会停留终生。

　　这似乎是一个错误——当初海不应起风浪，海鸥不应飞走。然而这的确没有什么错棗是海就总要起风浪，是海鸥就总要飞翔。

　　可当海再一次平静后，海鸥还会回来吗？

　　其实对海来说，他不应有任何的担心。因为海鸥永远不会离开海，总有一天，她会再回来，因为这里有海，有小岛，有她曾经的梦，有她曾经有过的一切。

　　只有海鸥与海在一起时，才是一幅完美的画面。

　　“除非海已不是海了；除非小岛真的消失了，那么海鸥也许将真的永不再来了。”

　　海是永恒不变的，而海中的小岛，是会慢慢地被还一点点的侵蚀，直至最后的消亡。如果海鸥十分眷恋别处的风景，不及时的返回。那将不会在看到她曾经栖息的小岛。只有那永恒不变、苍茫的大海。

　　或许海鸥只会悲切的哀鸣几声，盘旋着直至最终发现小岛已不复存在，便离去了。

　　也许在几十年后，当海鸥对于飞行的往复感到厌倦和疲惫后，她会又回到她当年离开的地方，站在那惟一的一块伸出海面的礁石上，呆呆地望着、等待着。

　　虽然眼中只有单调的一种色彩，虽然耳中只听到孤单的声音。但似乎却在单调中看到了往日的缤纷，似乎在孤单中听到了过去的旋律。

　　日子一天天过去了，海鸥仍痴痴地站在海中那惟一一块伸出海面的礁石上，日复一日，年复一年，终于也成为了礁石的一部分，永远的守候在她曾经深爱的大海边。

　　在月光的映射下，她圣洁的如同女神一般。

　　或许海鸥会在海底发现仅剩的一点岛的痕迹。但为了这些，海鸥将付出生命的代价。可海鸥却会永远的依偎在大海的怀抱中，直到沧海再一次变为桑田。

　　到哪个时候，纵然没有小岛，没有海鸥，甚至没有大海，那又能如何。

　　海鸥与海只不过是宇宙中的一颗沙砾，即使生命都已变得毫无意义，即使时间都已成为虚幻的概念，可就在飞满尘埃的空气之中，都会有海鸥与海的踪迹，都会有海鸥与海曾经所拥有的一切的影子。

　　用生命来换取永恒。

　　这一切，或许是值得的。

　　海并没有将等待海鸥回来的礁石腐蚀，海依旧是海，他在耐心地等待着海鸥的归来，他信守着一个长久的诺言。

　　可海鸥却没有回来。

　　海没有生气，没有咆哮，没有失去海的风度，只是默默地抱抚着作为诺言的礁石，静静地用泪水冲洗着被时间击碎的海鸥与海构成的那一幅完美画面的碎片。

　　海做的一切，值得吗？
