红（二）故事的名字

慕文

　　眼下，路旁橘黄的街灯已经亮起，辉映着窗边缓缓流动的云霭。山脚，依旧可以看见那条曲折通往山头的Z小路，在晚霞与晚风轻抚下的欧式古城依旧是那样的破落。

　　渐渐地，那无数的繁星充斥着暮色，就似天际悬满了无数悲惨号叫的灵魂，残艳。偶尔映挂着残留的晚霞，为这秋夜加了一丝血色，但依旧体验不出一丝生命的影象。

　　也许为它添上些生命的色彩是画笔的职责。但每一副都是这残喘无力的夜与月。木然的眼神映射出的星光也似乎变成了惨绿，仿佛并不是在描绘这“清风吹拂花满路，月色无凝照前澜”的夜景，而是用那眼神中的绝望涂补着这夜色的空隙。

　　我不知道他的名字，也不了解他的过去，总之，我无法了解一个所谓艺术家的心态。应该有好几个月了吧，这座山头，每晚，只有我跟他，我总用那双被酒醺懵的夹杂着血丝的双眼上下打赏着他，他也偶尔用那双木然灰涩的眼神瞥了我一下；我回报给他的不过是对弱者的嗤之以鼻，而偶尔我框入他的画中，就似是一棵被扭曲了的榆树。

　　至于后来我是如何开始与他相互理会的，我想应该是因为那条手链吧，淡黄色很精致的那条。

　　每次他做完画的时候，总是在底部添上一条手链，很抢眼，就像是我身上的那条。因为黄与黑很容易区分，而那黄色似乎是那画上唯一的生命色彩——尽管是那般的懒洋洋，就像是我头盖下的那团脑。

　　当我掏出那条手链时，他让我做了一件事，重重的摔在地上——手链被他抢走了。当然我……最后，两人都有气无力的瘫在地上，像泡烂泥。

　　最后还是我先开口：“你就是小柳？”

　　我猜到了他的问话。

　　……她死了。

　　就像是一片乌云，又或是一块岩石拦住了我，这一次我确确实实地体验到了鲜血溅到脸上的感觉，热的，还有点咸。

　　我不知道在这个风大的有点凉的日子里，她站在那高高的楼顶上做什么？也不明白那头长长的秀发和那件柔软的长裙为什么要学蝴蝶一样地飞？更不了解自己为什么要看她那双迷茫却又安慰更显得满足的眼神，在嘴角上还留着微笑，迷人。

　　溅在我身上的还有那条手链，它掉到我的靴子里了，我只好坐在她身边，从靴子里费劲地取出了手链。她似乎也在看着这机械的动作，又或似那张被血浸暖了的白色信封，收信人似乎是小柳，里面是空的。

　　后来我报警了吗？我不知道，我也不知道当时为什么要带走这条手链。人生本就是会随时殒落，却不曾有过这样的心悸，如魔般地缠绕。我根本不明白自己的头脑中在萌想些什么，居然会对生命的殒落感到窒息，还是无法弄清那双失神却又显得满足的眼眸？

　　而他呢？这个所谓的小柳呢？在空洞的画布上添加了一丝血色了吗？

　　笑着扯着画布在地上打滚，使得这幅深邃的月与夜慢慢地变得立体起来了。我叼着根狗尾巴草站在一旁望着，当时我真的怀疑他是由山那头过来的棗我指宁化。

　　我没有听清他在嘀咕着诅咒着些什么，也不清楚那只从他怀中掉出来的白色羽毛代表些什么，羽毛中应该有故事吧。后来他哭了，画中那棵曲扭的槐树业已泛浪成灾棗溶解的色彩更似无情的讥讽，流流潺潺。

　　我没问他与她的关系，也没问她的名字。我只是觉的似乎可以睡得安稳些了。几个月来，梦中总是那双苍白却又略带满足的眼神，那浓稠稍咸的鲜血溅到脸上的感觉时常让我夜半惊魂，虽然只是冷汗虚流。

　　有时，我在诅咒自己带回的那条手链，甚至诅咒那位还未知是谁的小柳。为什么…这一切与我无关的事竟使我悴缩，燥恼。为什么我的脑中总盘旋那阵凛凛的凉风，那颤动着翅膀的蝴蝶，还有那鲜亮的红。

　　不时从夜空中穿飞的山雁在深秋的风中冻得直哆嗦，也使人感到了些许寒意。看着发狂、疯笑、痛哭的小柳，我居然有股找女人的冲动。

　　但即使躺在女人的怀中，仍然隐现出那双带着满足却又令人畏惧的眼眸，嘴角边那淡淡的微笑。她为什么不去找小柳呢？我甚至产生了幻觉，溅在我脸上的不是鲜红的血，而是那洁白的乳汁。噢，不，这是什么念头，我将头埋入女人的胸膛，喘着气。

　　入夜，我依旧步履颠簸地迈入寝室，也许是酒精的缘故吧，有点轻飘虚浮。她已经穿着睡衣在我的床上，我早已倦恶了这种你有我需的关系。可她似乎并不在乎，她看人的眼神就是那样的满足，老天……为什么，为什么又是满足？为什么女人的眼神总是那般的容易流露出那已令我厌惧的满足？使我的胃又一次地翻动，收缩，最后只有一涌而出而不可收。嘴边流出的乳白秽物好似女孩嘴角的鲜红血痕，我觉得轻飘飘的，就像那天她在风中学蝴蝶般的旋转，轻飞，直至她拥着我入睡，也许女人就是懂得满足吧。

　　从那天以后，我就再也没看到过小柳，我想他不会来了吧。其实他早就知道她死了，从他的画中，我还分辨得出那一层深灰色，只是他不愿承认而已，也许他宁可自己欺骗自己吧。我只不过是那只捅破窗纸的手指罢了，而他就是那只眼睛和那双不肯回程的脚吧……

　　我烧了那只信封，我想也许我该代小柳为她做一个弥撒吧。但，她的名字呢？
