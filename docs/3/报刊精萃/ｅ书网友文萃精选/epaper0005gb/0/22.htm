我真挚的友人

lijia

　　今天又是她的祭日，我默默低头前行。天越来越沉暗了，似乎乌云要将天地间唯一的空隙塞满。走到一个供农民休息的草亭时，雨淋淋地下了起来，越来越大，泥路上的低坑处很快有一些积水。我望着眼前那条迂迴的小路。远处那湮盖小路尽头的一片树林，眼睛渐渐模糊了。不知什么时候，泪水沾满了我的脸颊……

　　三年前的今天，她默默地，无声地倒在了这条她曾熟悉的，留着她脚印的泥路上，就这样悄然地离去了……

　　她是我的友人，我真挚的友人。

　　初次的相逢，初次的帮助，都是这小路对它的见证。

　　我自身的残缺，常给我带来无限的痛苦与难堪。双脚在一次意外的事故中，造成左脚向外撇，有时还带着“叭，叭”的声音。在同龄人中，过早地失去了女孩子爱美的心，常常在同学的异样的眼光中独自行走。

　　四月的天，象是小姑娘的脸，时睛时阴，令人难以捉摸。脚的不便使我更害怕阴雨天，望着那丝丝小雨，胆怯占居了内心。

　　泥路上艰难的我正小心地挪着步子，“叭”地一声，我摔倒在地，两眼冒金花，屁股痛得我实在想哭。

　　“让我帮你一下好吗？”一个柔而刚的声音传到我的耳朵里。

　　“不！”

　　倔强的我，猛地用袖子擦掉脸上的雨水，想站起来，可站了几次也没起来，就在我感到万分难堪的时候，一双廋小而有力的手，用劲托住我的胳膊，顺着这股力量，我慢慢站了起来，回头望了一眼，十分生气地对她说“不用你管”。想甩开她的手，不了自已却差点摔倒。 “这条路不好走，我扶着你吧”她再一次用肯求的确良语气对我说。这回我不再强了，我俩默默地走完了这条小路。

　　这偶然的相逢，一次无意的帮助，我内心却受到一丝的振动，如同死水里起了一轮水纹……

　　又是一个雨天，又是这条小路，我依然独身前往学校，走到路边的草亭时，我发现不远处有一个打着花伞的女孩，正朝我招手微笑。

　　“是她”。不知为什么从内心深处涌上了一股暧流，眼水涌上了我的双眼。

　　我自残废以来，几乎得不到同学的关心，如今，……

　　以后，在这路上，在我的身边常常有一个短头发，长得很清秀的女孩子，伴着我同行。

　　在小路的尽头，有一汪碧绿的湖，水很深，周围长了许多的柳树。这里变成了我们的乐土，在这里种下了我们的欢乐……

　　春天里，我们将各自做好的花环戴到对方的头上；夏天，坐在这幽兰深静的湖边，畅谈自己的未来……

　　就这样，两个春秋一晃而过，我从一个自身残缺的痛苦阴影中解脱出来，重新塑造了自我。

　　秋后的雨总是来的这样快。我俩来不及从湖边坐起来，暴雨便如泼水一般从空中倒下。手忙脚乱的我，不慎从那泥滑的湖边掉入湖里。

　　“救命呀，救命……”我本能地叫了起来。

　　“别慌，千万别慌”，她一边说一边跳入湖中。

　　她抓住了我的腰，使劲把我往上推。雨越下越大，大量的雨水流向湖里，经过多次的努力，我精疲力尽地爬上了岸，回头拉她时，只看见一双瘦小的手一晃就不见了。我喊叫她，可周围除了我的声音外，余下就只有雨声了……

　　她就这样匆匆而去，留给我的只是那甜甜的笑……

　　我在她那甜甜的笑中，走完了高中两年的历程，成功地考上了大学。

　　今天是她三周年的祭日，我要在今天，在这诅咒的雨中，静静地献给她一个花环，一枚大学校徵……

　　我永远也不会忘记——我真挚的友人
