在那桃花盛开的地方

愚人（hiqingdao@china.com）

　　"在那桃花盛开的地方,有我可爱的故乡...".前天,当蒋大为出现在<同一首歌>的舞台,唱起让我魂牵梦绕的老歌时,我双眼润湿,情不能禁.我家就在那桃花盛开的地方.

　　小学的时候,我的老师就很自豪告诉我们,歌是写给我们村的."人间四月芳菲尽,山寺桃花始盛开".每到这时,漫山遍野的桃花开放,娇艳欲滴,整个村庄弥漫在桃花的芳香里,沁人心肺.然而,在美丽的色彩也遮挡不住过去破旧的房屋,桃花香味挥不去人们的疲倦.水蜜桃很有名,西哈努克曾尝过,赞不绝口.然而,那又怎么样呢?70年代到80年代初期,家乡水深火热,十分艰难.

　　我的家,普普通通;我是家里的第七个孩子,也是第二个男孩――在那时,一个男孩是靠不住的.父亲是家里唯一的整劳动力,但又体弱多病.从我记事起,就看到父亲佝偻着身子,匆匆忙忙,早起贪黑,不此劳作.平时,几乎看不到他的笑容.过多的劳累使他失去了生活的乐趣,家庭的压力让他举步维艰.不到五十岁,父亲的头发已经开始泛白,腰有些驼.他,看上去有60多岁了.而那时的我,无忧无虑,是家里的一块宝.然而,好景不长.

　　在父亲的辛苦努力下,大姐,二姐,三姐和哥哥结婚了,四姐去外地上学.父亲却累到了.于是,生活的压力更大了.五姐和我的童年也结束了.我记得,那是在初一,也就是13岁,当我矮小的身子第一次担起70-80斤的粪桶时,我竟直不起腰,肩膀痛疼.每次咬紧牙关,走上10几步,便要歇一会儿.我哭了,非常委屈的哭了.在我们村,在同龄人中,我是太不幸运的一个.晚上,我摸着肩上高高肿起的大包,恨父亲,恨我自己生在这样一个家里.我发誓我一定要考上大学!远离无休无止的劳作.我讨厌体力劳动,讨厌那种'锄禾日当午,旱滴禾下土'的日子.我与父亲的抵抗情绪也越来越重,话也越来越少.母亲看在眼里,疼在心里.可是,她能说什么呢!生活,还还要继续!

　　幸运的是我如愿考上了重点高中,多少远离了劳动.但是,回家还是要干活,挑水,锄地,刨地....看到同学都无忧无虑的生活学习,我心里很不平衡.不懂事的我,越发恨父亲.话越发少了.心里知道父亲的苦楚,却从不安慰.现在想来,那是多么的无知,愚昧.

　　桃花三开三落,瞬间即逝.我在焦急中等待着决定命运的通知书.当我终于手拿录取通知书的时候,心情怎么也不能高兴起来.由于填报志愿不当,我尽管超过本科线20多分,但只能上专科.父亲挺高兴,有说有笑.我永远忘不了那一天!我竟让父亲气哭了!那是我第一次也是唯一一次看到他哭泣.父亲啊,我对不起你!我太任性,太不懂事了!你能原谅我吗?

　　后来的日子还算顺利,大学的生活丰富了我的人生,我渐渐地长大了,能体谅父母的艰苦.每到周末,我不管怎样都要回看看他们.我开始去理解父亲.但还不能完全做到.总觉得有代沟,没有多少话说.父亲仍然种地,所以我去种田,因为地少,我去的次数也少了.生活也有了起色.


　　'年年岁岁花相似,岁岁年年人不同',桃花依旧,人事已非.转眼大学毕业.父亲又开始忙着送礼给我找工作.尽管工作不尽人意,但是终于实现了我儿时的梦想――远离种地.虽然,我在那个工厂没做多久.后来,经过自己的努力,我找到了一份不错的工作.每次回家,父亲都很高兴.父子的关系也好多了.1996年,我出国去培训.在国外,我打回电话,父亲说了一句然我感动一生的话,'你又出息,我和你娘都安心了'.对很多人来讲,这是再也不能普通的话,我却终生难忘.我发誓一定让他幸福,快乐.

　　然而,1997年初,父亲一病不起,日渐消瘦.一年后,他才好了些.但是,还要出去种田.家里人怎么劝他,他都不听.也许是寂寞吧.那时,兄弟姐妹完全可以让他幸福余生,但他拒绝了我们的好意.坚持自立.那时他已经65岁了.

　　有时,好人却不一定好运.1998年12月,父亲突然离去.我如雷轰天.父亲啊!为什么!儿子还没有来不及好好孝顺你啊!老天不公平.你才68岁,你应当再活二十年.

　　'在那桃花盛开的地方,是生我养我的地方......'我怎么能够忘记我的故乡!父亲啊!你永远在我心中.
