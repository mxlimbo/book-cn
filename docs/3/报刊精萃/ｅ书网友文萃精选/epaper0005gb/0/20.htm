　　——这是一家城市电台所说的一个关于性话题的谈话节目，这样的话题如何让听众参与呢？这样的话题听众参与之后会留下什么呢？其实人们对于性问题的关注并不是什么见不得人的坏事，爱情在人类文明的历史进程中，它的作用是显而易见的，男女两性关系问题，之所以有如此之大的影响力和号召力，这时因为爱情是和性爱联系的最密切的一个问题。当年，我清楚的记得与一位下乡知青带队领导有一番谈话，我说爱情是一个最美好的字眼，他听了之后，那神情就像今天的我对待现在的年轻人，认为她们是一群另类的眼光一样，目光中充满了不解与惊异。而那时候对于男女之事也的确不入现在的年轻知道的如此之多。以后作了医生对于人体的了解又多从生理和病理的角度，对人体的构造结构明白了。但是对于性爱的话题，对于现在如此之多的有关性问题的涉足，也不能和现在相比。但是，当自己开始负责策划一个夜话节目的几年中，感到最让人激动不已的依旧是人们对于性问题的渴望了解和愿意从性无知中走出的那种急迫的心情。但是，这样的希望总是不能让人满足，总有一种很强的正统的观念不允许这样做，把人们最隐蔽的私生活成为公开亮相的话题。而这些人正恰恰是现在在公开的场合大谈什么荤笑话的人。

　　受经济不景气的影响俄罗斯的红灯区已经呈公开的经营景象，连同它的电视台前一阵子就已经有了女播音员边播音边报告国内外的重要新闻，边脱衣的惊世骇俗之举，新闻与脱衣女郎成为最抢眼的俄罗斯的一道靓点风景；而现在我们又得知意大利的两家国家级电视台在有关新闻报道中竟有儿童的色情画面，而因这一原因有关高层和记者已经全部引咎辞职。联想到前些年一家城市的区级有限电视台，因为一位在此打工的男孩。因对于淫秽黄色光盘太急于想知道里面的内容，在悄悄地借来之后，就在电视台停播之后，和值班的两人共同欣赏起来了，没有想到的是他们忘记关闭向外发射的机器，结果让一位晚上找台的看电视节目的老同志目睹了这一幕，自然这事发生之后引起了有关部门的重视，严肃进行了查处。

　　人们的性意识和道德文化是有底线的，我们在对于这一问题采取所谓保守的态度，并没有要阻止人们对正常的性行为采取愚昧的策略，而是说我们切不可放纵性话题，也不能把这些当成生活中唯一的佐料。

　　性爱的美好不会是集市和菜市场上肉摊的展示，而是清流中泊动的激动与兴奋，性爱的美好与并不邪恶之所在就在于"任何值得我们做的事情都是在爱的光芒下完成的，任何值得你认识的人都闪烁着爱的光辉"。用这些作为评判的标准，性话题和荤笑话也许应该有了尺度。
