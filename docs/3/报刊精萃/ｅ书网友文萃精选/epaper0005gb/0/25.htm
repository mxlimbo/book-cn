荆 轲 刺 秦

碎梦一刀

　　荆轲迅速的从地图中拔出形如匕首的鱼肠短剑，顺势刺向嬴政。看着嬴政惊恐无助的眼神，荆轲忽然想到了太子丹。他们两人是多么相似啊，同样的抱负远大，同样的俊俏潇洒。如果不是处于敌对，或许他们还可能成为好朋友，因为他们是同一类人。手中的鱼肠顿了一顿，嬴政便闪了开来。

　　殿下的武士如梦方醒，开始围了上来，荆轲知道自己已失去了刺杀嬴政的最好机会，现在只能迅速的逃跑，寻求别的时机。可是他不想逃，因为他知道再也不可能有这样的机会面对嬴政，为了这个机会，燕国的人已付出了许多，等待了良久，为了燕国他只能成功。

　　嬴政即将逃向后殿，荆轲放弃了逃走的最后一次机会，迅速的追了上去。嬴政这时却已拿到了武器，回身应战，那是一把长达四尺的旷世名剑巨澜。荆轲知道如果让天生神力的嬴政挥舞开这把巨剑，将再也没有人能够靠近他。因此他身剑合一扑了上去，嬴政的巨澜斩了过来，荆轲却没有退，他已算准了这一剑的力量仅能将他的右肩斩伤，他还可以用左手持鱼肠痛贯嬴政心腹。巨澜斩到肩上的力量果然如荆轲所料，在嬴政斩到荆轲的同时，荆轲左手的鱼肠也同时刺中了嬴政。但嬴政却没有受伤，荆轲计算了千万遍，万万没料到的是嬴政竟然在衣内穿了软甲护身。

　　荆轲心中忽然生出一种无奈，苍天弄人，为何一切总是失败。嬴政停止了进攻，也喝止住了围上来的武士，向荆轲喝道：“说出你是谁？让你死了后不至于被人忘记。”荆轲摇了摇头，忽的将手中的鱼肠掷向嬴政，嬴政闪身避过，手下的武士已一齐把剑刺入了荆轲的身体。荆轲缓缓的闭上了眼睛，脑海中忽然闪出了那挥之不去的身影，以及她身旁的燕太子丹。终于可以不想她了。荆轲吞下了最后一口气。

　　嬴政默然良久，让手下将荆轲葬了，心中却一直奇怪。不知道为何荆轲第一次出手时会出现迟疑，以至失去机会。
