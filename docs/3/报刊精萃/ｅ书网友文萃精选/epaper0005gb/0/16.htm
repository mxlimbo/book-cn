拼 图

司雯

　　微风略略带斜的雨，毫无顾及地洒落在了没有被遮挡的一切上。手中虽然有伞，却无力擎起，只是任凭它自由的随着胳膊地晃动来回游荡。

　　象幽灵似的游弋在充满诡秘的街道上。暗黄的路灯笼罩下的一切都是那么朦胧，就连路口的那棵曾遭雷劈的老槐树，也显得婀娜。枯死的树枝，歪歪斜斜的树干，在这种氛围中，仿若舞动水袖的女子，让夜归的人痴迷。

　　雨，似乎的大了。因为周围的噼啪声愈加的急了。某一种声音听得久了，静的概念也就随之被它替代了。于是，时间伴着细雨在这一片单调的宁静中，沿着路边的沟渠，急急地流走了。

　　寂寞的时候，心中仿佛总有无法填补的空虚。无论做什么，都只是略略带过。就这样，送走了一夜的流水，又迎来了一天的起始。

　　本来只是想小憩片刻，或许是一夜未眠的缘故，待到再次睁眼时，生命中最具“创造性”的上午与最具“挑战性”的中午，已经在周公的陪伴下，度过了。

　　夜晚，是一天中最难度过的时间。若大的房间里只有我与倒映在惨白墙壁上孤独的影，周围的空气将自己的呼吸声往复传播，墙上的挂钟也寂寞的只发出单调的嗒嗒声。

　　无聊的近乎死亡，或许死去比我这样空虚的活着更有意义，至少会有无数的虫豸与我做伴，将我的灵魂蚕食的只留下丝丝叶脉。

　　愤怒的将连接光明的细线扯断，把惟一陪伴自己的影也无情的驱逐。无力的倒下，倒在一片黑暗中的尘埃里，心中似是充满了绝望的哀求。

　　无论我如何怎样用黑暗掩盖自己，也不可能将自己完全与黑暗融合，虽然我竭力的紧闭双眼，也无济于事。我仿佛是一张尚未冲洗的底片，永远不能暴露在光线之下，永远只能是黑暗的奴隶。

　　将身体蜷缩成一团，双手紧紧地抱拥着自己，用忏悔的心等待黎明的判决。

　　时间决不怜悯惧怕光明的人，黑夜的八小时，在我眼里，永远是那么短暂。

　　无论怎样，接下来等到我的将是十六个小时的光明，我只能面对，没有任何的选择。就像待宰杀的牲畜，生死是由不得自己选择的。将自己作为黑暗与光明斗争的奖品，谁赢得胜利，我就臣伏在谁的脚下。

　　战斗在预料中进行，在我耳里，凄冽的喊叫显得那样苍白无力。眼前腾起的烟雾，掩盖了打斗的景象，我用手随意的挥了挥，转身离开了。不管是谁胜利，对我来说都是一样，只不过是折磨我的对象改变而已，我永远只是没有选择、等待宰杀的牲畜。

　　日月交替，生死轮回。无法改变。

　　铛、铛、……，接连不断的十二声钟响，让我迷惘。眼前的景象是黑或白、是明或暗，都已不重要。

　　只有过去却在时间无情的撞击下，裂成了无数的回忆。我小心的收集了所有的碎片，梦想着有一天会将一切恢复原样。

　　但空气中的灰尘已将我记忆的拼图，蒙蔽了。
