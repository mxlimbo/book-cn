镜 子

jackyandsky

　　我爱镜子，爱镜子里的他。

　　那时我们有一样的梦

　　棗他说要和我活在镜子里，我们永远也不分开。

　　我依偎在他怀里看着镜中泛着红晕的脸感受渐渐膨胀着的幸福。

 

　　他从镜中走来，让我知道那不是飘渺的盼愿；

　　让我感受爱的充实和温暖。

　　他的手从我的指尖游走到我的唇；

　　他的唇从我的发梢划到我的脖颈；

　　我们紧紧的交融。

　　镜中的我像灿烂的玫瑰由他来点缀。

 

　　他走了，带走了我的温度。

　　我呆呆的望着镜中的自己，镜中的玫瑰在枯萎。

　　他走了，他是爱情放纵的侩子手。

　　我摸着冰冷的影子哭泣，我看见殷红的液体在镜子破碎的脉络中徘徊。

　　我再也看不到镜中美丽的憧憬，渐渐的失去了知觉……

 

　　我爱镜子，爱镜子里的她。

　　那时我们有一样的梦

　　棗她说我们的爱要像镜子一样长寿，我们永远也不分开。

　　我把它搂在怀里感受自己跌宕的心潮，希冀着我们爱的永恒。

 

　　她从镜中走来，她是上帝赐予我对爱的渴盼；

　　让我脆弱的灵魂不再孤寂，不再寒冷。

　　我轻轻地扶她纤细的指；

　　我轻轻地嗅她美丽的发；

　　我们紧紧的交融。

　　镜中的我像茂盛的绿叶抚慰着她。

 

　　我走了，带不走我的魂魄。

　　我呆呆的望着镜中的自己，看着挣扎着的瞳孔。

　　我走了，我是病魔的奴仆。

　　我看着冷冷的自己哭泣，我跪下乞求上帝，让我再看一眼我的她。

　　恶魔咆哮着将我撕碎，破碎的灵魂仍在流泪。
