高 潮

阿煜

　　我没穿鞋，我在半空中摔死。

　　我在公交车里，我睡在床上，我在男人的身体里游走，我张开嘴，唱不出声，我在流泪。

　　一个光头的婴儿在邪恶的朝我笑，它长了一口整齐的白牙。

　　我是太阳，我是一切，我是臭狗屎，我是妈妈的坏孩子，我是爱流泪，我是性冷淡。

　　我象个白痴一样无所畏惧无所畏惧。

　　那天我终于给了他一耳光，他的脸上长出牵牛花，他的嘴角开始流出血和口水，笑容在我嘴唇上肆意的绽放。他问你为什么打我为什么笑，我是你的七七，你的男朋友，我说我一直想搧你耳光的，从见你第一面起每天都想，我的手在你脸上形成的那个弧度和声音使我一想到就面色潮红兴奋不已，他说你疯了你不爱我了，我说我没疯我是爱你的，就象星星爱月亮露珠爱青草爱的心慌慌，这和搧你没有关系，你仍是我的宝贝我的心肝，我走上前去亲他的鼻子和下巴，他的胡须扎的我好痛，我说我们做爱吧，他说你这个魔鬼我总有一天会死在你手上，然后我们就做爱。窗外的月亮象个眼睛一样一直看着我们，看着世界上最美妙的动作，星星跌落在阴沟里，蝈蝈在撕破嗓子唱歌。

　　我的七七大汗淋漓，圆圆的屁股一起一伏象在水面跳舞的皮球，七七的汗顺着我的鼻尖流到我的嘴里没有味道，七七突然很使劲很努力的样子，龇牙咧嘴，我说你是不是到高潮了，七七不说话过了一会儿才说是的，我说我没有高潮从来都没有，高潮是什么感觉，是不是从21楼跳下来的感觉，七七说不知道，他象个士兵拔枪一样把生殖器从我的器皿里拔出来，我说你不告诉我我就同其他男人做爱然后去问他们，七七把衣服穿好洗脸梳头换鞋抽烟然后吐出一堆烟在空中织了一件毛衣，七七说你去问吧问谁都行。

　　我把枕头扔在他脸上，把我的日记本扔在他脸上，七七又开始出血，我说你这个流氓，你这个没心没肺的臭流氓，你除了操女人你什么都不想，你单纯的让我觉的恶心，七七不说话一直不说话，血顺着嘴角流到了他的脖子上，喜儿的红头绳飘来飘去。

　　我开始哭，我并不难受但我又开始哭，象个成年的小怨妇死死盯着窗外，泪水滴滴嗒嗒的流到了桌角，屋里到处飘着棉花和鼻涕。

　　我偷偷看了一眼七七他没在看我，也没擦嘴角的血，脸上一点表情也没有眼睛成了小石子，我突然觉得他很窝囊，又开始向他脸上扔东西，我把电饭锅扔过去，把避孕套扔过去，还有我养的蜥蜴，我的七七突然站起来，够了！我受够你了！你给我滚，你看看你现在象什么样子，整个一个妓女泼妇变态狂！

　　我看到七七漂亮的脸涨的通红，用我从未见过的厉害眼神盯着我，我说我终于等到你说这句话了，我知道你早就不耐烦了，我也不耐烦了，我烦了，我烦了！我拿起麦克风站到桌上玻璃碎了，我对着窗外大叫，七七是个臭流氓！我要和七七分手！分手！分手！我的声音象定时炸弹一样陆续在空中爆炸。

　　我看到一只只蚂蚁探出了脑袋，我砸了一只热水瓶又砸了一只，啪——啪——我听到眼珠摔裂的声音，我披头散发，我把七七的欢欢拉了上来，准备扔下去，七七在下面拉我的腿把我的丝袜拉了一个大洞也没有把我拉下来，七七说你把他扔下去我就把你扔下去你这个泼妇，七七把我的腿抠出了血他已经一点都不心疼我了。

　　我没有把欢欢扔下楼，我把它从桌子上扔到了地板上，我要让七七清楚的看到它的死状，小狗的眼珠掉出来脑浆迸裂。

　　我在桌上手舞足蹈，轻歌曼舞，我看到七七的眼睛变成了紫红色。

　　然后他上来推我，我们扭打了一会儿，一分钟后我们俩双双坠楼，突然上了一辆高速列车，我还没有闻到死亡的味道，只看到七七的衰草发了疯的生长起来，他的动作很美象在花样游泳，我手上七七送我的钻戒滑落下来，变成一颗泪珠在空中飞来飞去，我在14层时一下到了高潮，因为死亡的逼近让我顿时感到兴奋不已，我终于知道高潮的感觉，就是一颗飞速划过的流星，我想两秒钟后我和七七将象烂西红柿一样躺在马路上，那是多么有趣，我已经原谅七七了，我在空中艰难的翻了一下身，抱住他，然后别人就会觉得我们是双双徇情……

　　哐！在落地的一瞬间我听到了一个细小的声音：

　　我爱你。
