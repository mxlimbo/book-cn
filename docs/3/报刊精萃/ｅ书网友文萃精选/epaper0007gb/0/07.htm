缘

苹果梨

　　缘，并不是虚无飘渺，不是梦中注定；缘是心灵的感应，心灵的默契。

　　缘，是冥冥之中的感觉，有千里相会的缘，有一面之交的缘，有终生相守的缘，也有情深缘浅的缘，有无缘的缘。

　　缘，需要我们用心去感受，用真诚去换得，用汗水去培育，否则缘在眼前却终生不得，本来一份相当不错的缘也会枯萎、夭折、甚至变成恨，变成仇。

　　缘，也是一种机遇，靠我们自己好好把握，好好珍惜，我们登上泰山应蓄收它的壮观与博大，观日出就应该摄下它的阳光和瑰丽，经历了人生的花季就应该珍惜友谊棗这份难得的缘。也许我们会是一对好朋友却由于你的沉默，又由于我的冷傲而留下一份并不美丽的遗憾，也许以前我们已错过太多，难道我们错过太阳，月亮，还想错过今夜的星星吗？就让我们从现在开始留心身边这最美好的风景棗友谊，有它人生会更美丽，生活也永不寂寞。

　　然而缘既是心灵的感应，那么就是不可强求的，缘犹如手掌中的沙土，如果它不够分量，一起风就会随风而去；如果它够分量，不管风吹雨打始终会留在你的掌心，与你共度苦乐人生，心心相印，肝胆相照，但如果你怕它被风吹起，而紧紧地握住它，它会从你的指缝中悄然滑落，给你留下一份失落和惆怅，一份无言的感叹！

　　缘，既是一种机遇，那么它也是可以创造的，并非命运注定，如果不好好珍惜，用心去培养，一切随缘也将注定也这份缘无缘！
