等着你的我，等着我的你

眼泪

　　等一个人的时候，就像子夜等待着黎明。

　　等一个人的时候，就像冰川等待着蛇火。

　　等一个人的时候，就像亚当等待着夏娃。

　　等一个人的时候，就像饥饿者等待着食物。

　　如果子夜等不到黎明，就意味着不会有光明。

　　如果冰川等不到蛇火，就意味着不会有温暖。

　　如果亚当等不到夏娃，就意味着不会有爱情。

　　如果饥饿者等不到食物，就意味着死亡。

　　我还不想死，所以我要等下去。

　　“当你左脚疼，我会感到我的右脚疼！”你是否也有这种感觉呢？

　　是否在我等着你的同时也在等着我呢？
