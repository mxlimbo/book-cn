你一定要死

无情 （cappuccinoo@citiz.net）

　　我是一个杀手，在我刚出道的时候，我的师傅告诉我，一定要杀了一个人。

　　一个女人，她有着魔鬼的身材，天使的脸庞，曾经是我师傅的初恋情人。

　　我不知道那个女人倒底有什么错，值得要我去杀，不过我记得师傅曾经告诉过我，女人都是应该死的。所以，杀女人不需要理由。

　　见到那个女人的时候，已是黄昏，夕阳西下，残阳懒懒的照着天边灰白色的云层。

　　我握紧刀，默默的站着，低着头，象一个害羞的孩子。

　　我师傅说，杀一个女人时千万不要看她的眼睛，因为女人的懦弱是会传染的。

　　"是你师傅叫你来杀我的？"她说话的声音柔柔地，象一堆屎。

　　我依然低着头，轻声道："是的。你一定要死。不过我师傅说，他可以给你50元，让你自己杀了自己，这样可以少受些痛苦。"

　　"哦？是那样吗？如果我不愿意呢？"

　　我把头低的更低，诺诺道："如果你不愿意，师傅说他可以给我100元，让我杀了你。"

　　"哦？你有把握杀我吗？"

　　"我知道要赚一个女人的钱不是那么容易的，所以......我不敢肯定。"

　　"呵呵，既然你不敢，我不愿意，那我怎么死呢？"

　　我转过身，头有些晕。

　　我知道是该走的时候了，所以我对那个女人说，"我师傅说还有一个办法。"

　　"哦？不会是他自己来吧？"她的话语中充满了嘲笑。

　　"不，我师傅说，他可以等你，直到你——老死！"
