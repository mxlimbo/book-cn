首届记者节有感

沉舟

　　今天是新中国首届记者节，曾经干过数年记者的我，突然间有了自己的节日，心里竟然有一种异样的感觉。

　　应该说，我不是个合格的记者；因为在我看来，在今天这个时代，记者的根本使命应该是能够主持公正、声张正义的，可是，我却深切地感到，一名中国记者的能量其实是很有限的，有时连自己的正义和合理的要求都不能捍卫，更别谈为社会主持正义了。

　　在报社我干过编辑，干记者前后也有四五个年头；可是，很惭愧的是，虽然写了不少应景的文章，但却没有写出真正有震撼力、有份量的文章，这是十分遗憾的。但应该说，干记者我是尽心尽力的，各种酸甜苦辣也尝到了，也作出了不少奉献和牺牲。当时，我的总体感觉是，报社里整个写稿和用稿的机制大多时候是盲目的；除了领导和会议的报道好发稿外，其余的时间都是“放羊”，也就是说随我们自己出去采写什么，虽然这也挺自由的，可辛辛苦苦采写回来的稿子往编辑部那里一扔，别人何时发就不知道了。有时，月初写的稿月底发，有时放在下个月发，有时时效性过了也就“枪毙”了；自己按照自己的思路采写的稿子，当然大多不如会议和领导的活动重要，加之版面也有限，自然自采的稿件要往后压了；单纯往后压还事小，倘若自己觉得比较重要的稿件，被删除得不成样子，即使能够发出来也就不错了；当然你还不能作声，不然要得罪人，以后发稿就更困难了。在这种机制下，每个月的任务把你压得已经喘不过气了，你再想写一些特别的东西，也就没辙了。于是，我想有些报社采编分开是有道理的，这一点可以消除不少“弊病”。

　　我不知道为何设立这个记者节，据说是为了纪念碑1937年在上海“孤岛”上成立的那个中国青年记者协会的；事隔六十多年之后，又重新庄重地板设立这个记者节，真是别有一番滋味了。不过，我感到庆幸的是，中国的记者倘能真正扮演出一个公正和正义的角色的话，也就是说是国家和人民的大幸了。
