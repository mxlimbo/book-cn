生活是种鞭策

jackyandsky

　　穷人奢盼富人的成就，富人则怀念做穷人时的进取心。

　　穷人觉得时间很漫长，富人却看着表杞人忧天的哀叹。

　　大家似乎活得很平衡。

　　上帝把弱者推到了墙边，有的弱者学会了攀岩变成了强者，剩下的成了真正的弱者。

　　而我，我是个穷小子。

　　我是个穷小子。尽管我物质空前缺乏但我有十分饱满的精神空间，所以我不是很穷。

　　我成天坐在家里，对着镜子，有时笑，有时哭，当我想到ed格式的我就会笑，当画面切换到ing格式的我，我会哭。

　　有时我会想，我应该站起来，去找个象模象样的工作好填满我的肚子。然后我就可以继续坐在这里胡思乱想了。

　　当外面的风把我剥得赤裸裸的时候，我发现自己一无施处。我用全身力气爬回家，并没有带回太多的愤怒，我只是想拿刀把自己杀掉。

　　在死亡意识的瞬间，我想起了自己的惨淡。

　　过去的我涂的香水是PARIS，而且我会奢侈的把他们洒在脖子和两个袖口处；现在我只能用MADE IN CHINA，谨慎的把他们涂在我的领子上。

　　过去我不知道零钱有什么用，把他们到处乱放；现在他们体面的进入了我的挎兜，而且我要小心的对待他们。

　　过去我总会去存钱，我对自己说再过几个月那会是一比惊人的数字；现在我的银行是朋友的挎兜，我对自己说如果不吃不喝再过几个月我就能还清贷款。

　　过去我觉得打D是件不体面的事，总有一天我要买车；现在步行对我来说很体面，我也不会再去买什么汽车杂志。

　　过去我经常去理发，我会告诉服务小姐用NO.1给我洗头；现在我不去理发，我在家里扮泰山，不知道头发上什么时候会长出蟑螂。

　　过去我认为朋友是最重要的，为了朋友可以舍弃一切；现在我知道朋友不能当饭吃，填饱肚子才是最重要的。

　　过去我会不客气的对我的BOSS说最近我有点不舒服要休息两天，身体是革命的本钱嘛，然后用他眼中的苦涩填充我的高傲；现在我会恭敬的对我的老板说，这个周日我能加班吗？我要把身体换成钱，然后所有的自尊被他眼中的藐视吞走。

　　过去我是网络CHAT族，我象个HAPPY KING在网上冲浪；现在我是个伊妹儿派，象个MELANCHOLY PRINCE（忧郁王子）在E-MAIL上贴酸文。我在考虑是不是要把我的猫当掉，因为它总会让我一时冲动挥掉我的饭钱。

　　高傲和懒惰要埋没我的青春，而我不要让他们得逞。生活的残酷鞭得我浑身是伤，我不奢望富人的成就，更不平庸于穷人的堕落，我要用青春挥洒我自己的灿烂。

　　我不再幻想持之以恒的堕落是否真的有一天也能拨开云雾，我要塌实的踩出我的每一个脚印，写下我无憾的辉煌。

　　我把刀从脖子上拿开，用它把我全部的沮丧做成一道菜，然后大口的咀嚼。我想我正在乐观的扶平自己的伤口，我要站起来了。
