第三空间

第一篇 人之初

　　上帝很寂寞也很无聊，于是用从学校里学来的魔法克隆了一个自己。为了有区别于自己，他把那个克隆的自己取名为人。

　　上帝造人是一时的心血来潮，他知道魔法过段时间就会失效，人也就会很快死去消失，变成他克隆时的那种掺杂物——泥土。

　　在人以前，上帝还用幻术建造了一个大花园。里面有日月星辰，飞禽走兽等好玩的东西。上帝把人放在花园里代表着自己管理着里面的一切事物。

　　人非常聪明，他本来就是上帝身上的一个细胞，所以他做了花园的主管后把花园弄得象天堂一样漂亮，上帝看了很高兴。他要奖赏人，问他要什么。人说，你赐我不会死的方法吧。上帝很为难，这是严重违背天条的。人不过是他玩的一个魔术，怎么可以象宙斯一样不死呢。可是他答应了奖赏人又不能食言，于是上帝想了一个两全之策，既不会让天廷责怪，又能让人的生命延续的方法。

　　上帝在人睡着的时候从他身上取了点东西然后吐了一口唾沫念了一句咒语，一个女人就出现了。人一觉醒来，发现自己的一对翅膀没了，眼前却有一个胸脯很大的怪物。人问她从什么地方来。而她只会傻笑。人就离开她找自己的翅膀去了。

　　女人没事到处走走她碰到了蛇，她和蛇成了好朋友。蛇带女人去它住的地方，那里有一棵很高的树，树上的果子悦目诱人。蛇说，那果子很好吃的，你可以把她摘下来吃。你再带点给人去，两个人在一起吃会有奇妙的效果。女人深信不疑，采了好几个。她找到人把果子交给他。人正找不到自己的翅膀而感到口干舌燥就毫不犹豫和她一起分着吃了。吃完后就和她睡在了一起，那果子在身体里起了反应，很快使人禁不住和女人两体合一，最后又分开。人和女人就有了他们有生命以来的第一次，他们觉得很快乐。

　　这一切都是上帝在暗中安排，他为了使这件事能瞒过天廷，最后拿蛇去替罪，对外宣称是蛇诱惑了女人。上帝把蛇的脚全部都砍掉，这样它永远上不了天廷去揭露真相了。

第二篇 挪亚方舟

　　上帝把男人取名为亚当，把女人取名为夏娃。让他们同寝以便繁衍后代，这样当人死了以后，他的生命就以另一个人的方式延续了。到挪亚的时代已经是人第50代子孙。

　　上帝的同门师弟魔鬼撒旦在地狱里居住，他一觉醒来就看到上帝造的大花园和里面的人类。他嫉妒上帝的快乐，他要把一切都毁掉。上帝觉察了撒旦地险恶用心，他托梦给挪亚，要他立刻造一条大船，把大花园里的生灵和他的子孙都带到大船里面避难。

　　挪亚醒了以后立刻动员了所有的人参加了大船的建造，他们把船做得方方正正，给它取了一个船名——方舟。挪亚和他的子孙带着大花园里的生灵和足够的粮食进了方舟里。

　　魔鬼撒旦在地狱里念着咒语，让银河里的水泻下来，大花园被淹没在一片滔滔的洪水里，洪水泛滥成灾。方舟飘浮在水面上将近四十来天，水才慢慢地退了下去，裸露出泥土的颜色。

　　挪亚带着躲过灾难的人类和生灵从新又回到地面上，大花园虽然没有了，但他们可以重新在建，最关健的是人类没有灭亡，他们将走想繁荣。

　　上帝为了怕魔鬼撒旦再有什么行动就暗中派了四大天使去保护人类。撒旦见人类依然生活在地上，知道上帝对他早有提防。他不敢再用他的魔法去毁灭人类，却又对这次失败很不甘心就想方设法把一些邪恶的东西装进包装精美的箱子里，把它丢在人类生活的地方。人类捡到那些箱子对它好奇就打开来看了，于是，疾病、灾害、战争等这些恶的东西在人类里传播开来。

　　 上帝的四个天使用尽了所有的办法还是没能彻底清除恶，恶还在蔓延。他们只有把美和善教给人类，以求对抗恶的势力，使人类世界不至于沦为地狱一般的丑恶。他们还找了一些没有被恶毒害的人去四处宣扬上帝的学说，那些人成了最早的传教士。

第三篇 法老

　　人类在躲过一场劫难以后，重新造起了大花园并在外面修筑了坚固的墙。这个建筑物被称之为——空中大花园，也就是人类最早的城池，巴比伦。

　　魔鬼撒旦在向人类投掷了恶以后，人类之间变得自私自立，互相间有了矛盾。于是一批又一批的人离开了巴比伦城去寻找自己的乐土。

　　这样在以后的数百年出现了很多新的领地，人类把它们叫作国家。意思就是这块地是有谁先找着的，把它圈起来就是谁的家了。国王在自己的国家里纵情极乐着，上帝看到人类这种无知的做法很痛心，他一方面让他派下凡来的天使到处找人宣扬："上帝是人类唯一的主宰，那些自封为王的都不能算数。"另一方面他要惩罚那些所谓的国家，让他们明白，人类最终也不过是一个上帝认为好玩而制造的玩具而已。他要他们生就生 ，他要他们死就死。（这样看来上帝也不是什么好人）

　　当时最大的国家是古埃及，埃及的国王也叫法老和上帝的派下凡来的一个天使私教甚好，天使常在法老的宫殿里大吃大喝。这一天他告诉法老上帝要降灾给人类了以惩罚他们只顾自己享乐而忘了赐于他们生命的上帝。

　　法老问天使有什么解救的方式，天使在他的耳边密传了机宜。

　　于是法老下令全国总动员，在尼罗河畔建造起一座高高耸立的石塔，又挑选了很多骆驼、山羊、驴等牲畜来放在塔旁边的祭台上作为供品奉献给上帝。

　　上帝来察看人世的时候首先看到在建的高塔，又发现埃及的国王法老奉献给他的供品。于是当上帝降灾给人类，所有的国家都遭受了七年的讥荒的肆虐，唯独埃及到处都丰衣足食。

　　以后的埃及的每一代法老都忘不了造一个塔奉献一些好的供品来讨好上帝，并在自己死后就葬进塔里密封起来，据说身体能不受外物侵扰，灵魂可以飞上天界。

　　举世闻名的埃及金字塔由来于此。
