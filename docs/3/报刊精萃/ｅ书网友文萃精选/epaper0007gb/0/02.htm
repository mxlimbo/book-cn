请你一定要骗我

无情（cappuccinoo@citiz.net）

　　如果到那个时候
　　我还没有回来
　　请你把这块手巾交给她
　　告诉她
　　我已经爱上了另一个女人
　　叫她不要再等我了

　　――序

　　我独自坐在客栈门口，静静地看着一群商人骑着一群骆驼从眼前走过去，又走过来。我从来没有问过他们从哪儿来，又要往哪儿去。我想我也没有必要知道。

　　远处，那阵风沙渐渐弥漫过来，沿着沙丘翻腾着。天空是幽暗的，浅灰色的云层跟随着风沙一起覆盖过来。我知道我应该进去了，很快这儿就要来临一场沙漠风暴了。

　　我到这里已经七天七夜了，我在等一个人，一个女人，她的名字叫孔雀。听说每年的这个时候，孔雀都要经过这里，有人出钱要我杀了她。忘了告诉你，我的名字叫无情，我是一个杀手，专门替别人解决性命的一种人。

　　等待是一种痛苦。每天晚上我都对自己说，也许明天她就会到了。可惜，有些事你是永远都猜不到结局的。前几天，这儿曾经来过一个男人，他告诉我，他要去杀一个人。他说那个人乘他不在家的时候强奸了他的妻子，所以就算到天涯海角他都要找到那个人，然后杀了他，替他妻子报仇。

　　我看得出他的武功不是很好。他在临走之前交给我一把藏刀，他说如果他不能回来，就叫我把藏刀交给他的妻子，因为他曾经答应过她，就算他人不能回来，也要把刀给她。那把藏刀是他们的定婚信物。

　　那天晚上他走了之后就再也没有回来，我看着那把藏刀，眼眶有些湿润。也许那个男人根本就不想让他妻子知道他已经死去，因为他连他妻子的名字都没有告诉我。我知道他本来是想自己回来拿的，可惜他已经再也没有机会了。

　　那次沙漠风暴一直持续了三天三夜，在第二个阳光明媚的早晨，我终于等到了孔雀。

　　阳光有些耀眼，那个叫孔雀的女人安静地站在那里，脸色有些苍白。在我拔刀的一瞬间，我看到在她眼里闪过一丝湿润的光芒，刀冰冷的从她前颈穿过后颈刺出，血如迷雾般洒了出来，倒下。阳光照在那把滴着鲜血的刀上，让人心寒，泪水在孔雀脸上慢慢地干了.......

　　我带着孔雀的头颅走出那片沙漠的时候，见到一个男人寂寞的坐在一棵树下，焦急的眼神似乎在等待着什么，我莫名的看了他一眼，从他身旁走过。他忽然起身拉住我，问我有没有见到一个女人，她的名字叫孔雀。我握刀的手有些颤抖。

　　他告诉我，每年的这个时候他们都会来这儿约会，不知道为什么，今年她还没有来。其实我好想告诉他，她已经来了，可惜你看不到她。在我离开的时候，那个男人转过身，偷偷擦拭着自己的眼睛。

　　后来有一次，我看到这个男人和另外一个女人走在一起，我认识那个女人，她就是曾经出钱要我杀孔雀的人。

　　我收钱回城的那天，思思站在城门口等我。见到我，她哭了。她抱紧我，问我为什么这么晚才回来，问我知不知道她很担心我，问我还爱不爱她。我没有回答，只是紧紧地抱着、吻着她。泪水滴落在我脸上，温暖而潮湿。

　　回家的路上，思思说，她在城外已经等了半个月。每天看着太阳从这边升起，又从那边落下，人群从城里走出去，又从城外走进来，她多么希望那些人群中有一个就是我。

　　在那里思思还碰到过一个女人，女人说，她也在等她的丈夫回来。她告诉思思，如果她丈夫不回来，一定会把他的藏刀带回来给她，那是他们的定婚信物。她已经等了好几个月了，如果他丈夫再不回来她就决定去找。我忽然想起，好象那把藏刀被我忘记在客栈里了。不过我希望那个女人不要找到它。

　　思思安静的依偎在我怀里，很认真的看着我，她说，如果有一天，你也不回来了，求你一定要骗我，把那块手巾还给我，说是你爱上了另一个女人，让我死了这条心，不再等你，起码我还可以恨你......说到这里，她的声音已经哽咽，泪水在我怀里不停的翻滚，我紧紧地抱着她，在她眼里，我看到了两个人的泪水......

后记

　　写完这篇作品，我自己也觉得有一丝伤感，原来每一个人都有他（她）自己的方式来解决痛苦，就算明知是在骗自己，也要让希望永远的存在。

　　所以，我希望，你们会喜欢无情的作品。
