初 夜

第三空间

　　她醒来，首先映入眼帘的是洁白色床单上那一朵粉红色的小花，然后感到自己的身体在蓬软的床上像突然苏醒并绽放的雪莲。晨光透过窗棂照在裸露着的纤细的腿上，有种沐浴一样的快感，这一瞬间她不知道自己是幸福还是别的什么。她曾经梦到过这样的情景，她曾经微笑地接受……曾经，意识中一些很迷乱的幻影；过去，确都过去了，如风。她还能找回来点什么，倦曲着身体微微颤栗着，她吻着身旁熟睡的男人。

　　“嗨，这么早……你再睡会儿吧。”身旁的男人张了张眼睛又立刻闭上了。

　　她只能自己一个人冥想。

　　乐呵呵小女孩的日子做着五颜六色的满天星，可可过来抢。

　　“别动。”她打他的手：“做好了串成风铃再给你玩。”

　　“我现在就要。” 可可却是抱着她的身体，捏压着吻着她。

　　“你这个小坏虫……”她虽然挣扎着骂着他，却还是闭上眼睛和他接吻。

　　“巧巧，我想要。”可可哀求着。

　　“不行。”她阻止着他进一步深入，毫不犹豫从他的怀抱里挣脱出来……

　　她翻了一下身，对自己想到的过去有种说不清的感觉。

　　她不想再想下去却又睡不着。她不知道自己应该做些什么，也许就只有这样睁大眼睛躺着。

　　很久前看过一本叫什么的VCD，一个男人死后变成了一个幽灵，他借用别人的身体来找他爱的女人。他们真的又相爱了，但结局还是分开了。这只是一段精彩的感情，不需要结果。她的双眼曾湿了，感动或有点感伤她可能分不清，眼泪都是真的。

　　她的身体在床上轻轻移动了一下，她用手去覆盖那片阳光。她一直以为她可以抓得住一些她所认为是属于她的东西。所以她并不着急把自己给可可，即使可可要去日本留学前，她也没给他机会。

　　她，叶巧巧告诫自己一定要等到新婚夜才是一个女人完全奉献自己的时刻。

　　可可去了日本三年。

　　他回国了但没有来见她。她听说了他回来的消息就一直在静静的等，这是一个等待中渡过的漫长秋季，梧桐叶子一片一片变黄飘落下来，暮色霭霭中她期望他的出现。床头柜子上的镜框里他的微笑捉摸不透。

　　她终于等不及，放下女人的矜持去找他。

　　当天空飘着小雨，她和他在以前相约过的亭子间见面。一条米白色的围巾是她特意围给他看的，他的目光有点漠然，虽然脸上含着笑但已经没有记忆中很她喜欢的那种调皮了。

　　他的话很少几乎没有提及起他在日本的事。他使她陌生，完全是因为他的表现。他看着她不想试图拥她入怀，不想疯狂地吻她，为什么？是他有了新欢早就变心了吗？她的眼泪禁不住流下来，为了不让他看见她扭头就冲进了雨里……

　　她的生活肯定会变，在忘掉他以前她想做一件事，就是想和他做一次爱，让他永远欠她的，永远忘不了她……他曾经想要而得不到，他会抵得住这种诱惑吗？她的一瞬念头带着邪恶的阴谋隔着电话线流到他的那端。她声音迷人优雅：“可可，你不是想要我吗？那你来找我。”电话那端的他居然迟疑了片刻说了一句莫名的话：“太晚了。”太晚了？她一直捉摸他这句话的真实含意，直到她终于知道了真相！

　　他是在圣诞节快来的前几天走的，没有人知道他去了哪里……她在圣诞节还是收到了他为她早就准备的礼物，是礼品公司的一个小姐送来的一束灿若繁星的鲜花和一个包装精致的礼盒。

　　礼盒里是一枚钻戒。还有一封他写的信：

　　钻石恒久远，一颗永相传。

　　巧巧，原谅我不能给你戴上这枚钻戒。它早在我去日本的第一年，我用打工赚来的钱为你买好了的。总以为会有那么一天，但我终于挡不住对你的相思之苦和那个物欲横流的声色世界。我回国前夕被医生告之在我的血液里查出被染上了爱滋病，你一定听说过这种病的可怕。我知道我已经没有希望和勇气只想回来再看看这里的人，爱我的人和我爱的人。我后悔我曾经的放荡，现在说什么都没用。我已经离开这个城市，去一个陌生的地方，然后安静地去天国。

　　别了，这枚钻戒是为你买的，所以还是来送给你。它不代表什么……希望你幸福永远。

――可可

　　信纸轻轻地滑落，不禁泪流满面……

　　她望着天花板想起那枚钻戒被珍藏起来一次也没被戴过。一年后的昨天她结婚了，新郎是一个长得很象可可的男人。她为他付出了第一次，也许早就该这一天了，她想。
