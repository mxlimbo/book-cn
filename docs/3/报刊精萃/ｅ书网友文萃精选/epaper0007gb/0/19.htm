你的剑很快

无情 （cappuccinoo@citiz.net）

　　他的名字叫阿呆，他的剑很快。

　　十五岁那年，他第一次出剑，"夺命双刹"之一衡力海死。

　　十八岁那年，他第二次出剑，"武当快剑"落破死。

　　二十岁那年，他第三次出剑，"浪得虚名"飘逸死。

　　二十二岁那年，他没有出剑，"虎头蛇尾"懦弱败。

　　二十四岁那年，他说他的名字叫阿呆，"无敌剑客"荆要命猖獗而逃。

　　他笑了，那把剑在他腰间不停的摇晃，发出美妙的声音......

　　那天，他依然独自坐在家门口。远处，隐隐约约可见一人影飘之而来。近了，是个女子。

　　你又来找我？

　　是，今天我一定要打败你。女子握紧她手中的剑，看着阿呆，她的脸色有些苍白。

　　是吗？阿呆迷起眼睛，起身，拍着女子的肩膀，道，你还记不记的？两年前我们第一次碰到的时候，你的剑的确很快，可惜你还是败了。去年你来找我的时候，你的剑已经没有两年前快，我告诉你那是因为你恐惧，因为你紧张。呵，今天......今天你又来找我，从你的眼神中，我可以看出，你已经败了。所以，我不需要出剑。

　　阿呆舔了舔嘴唇，干净的，湿润的，带着骄傲的斜斜的嘴角。

　　女子感到自己全身的无力，她低下头，漠然。她的手有些寒冷，脸色更苍白，她败了。她没有出剑就败了。

　　阿呆想笑，他还想告诉那个女子他的名字叫阿呆，也许，那个女子就会象荆要命那样猖獗而逃，于是，他转过身，偷偷地的笑了起来，他说，你知道我是谁吗？我......

　　话还未完，他就感觉到身后一阵冰凉，他看见自己胸前刺出一把利剑，带着粘稠的血液，他的笑容突然间僵硬。

　　女子在他身后发出铜铃般的笑声，她说，你的名字叫阿呆，我知道......

　　他倒了下去，他未出剑，就败了。

　　女子迷起眼睛，蹲下，拍着阿呆的肩膀，道，你知道吗？其实每个人都他的弱点，你的弱点就是太骄傲，太轻视你的敌人了。我花了三年时间让你来轻视我，所以，今天你败了。

　　她的名字叫丁零，她的剑也很快。
