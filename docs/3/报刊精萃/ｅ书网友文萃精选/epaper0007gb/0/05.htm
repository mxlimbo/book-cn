生 存 状 态

沉舟

　　对于生活我不知说些什么，在风雨的打击下我始终只能沉默不语。

　　——前记

　　对于生活我真不知该说些什么，生活中的太多的不如意几乎让我丧失了对生活的信念。为了寻求一个真正公正和合理的世界，我上了网。在网络这另个世界里，

　　我是编造些卿卿我我的故事以博取读者的一掬同情泪，还是写一些矫情的文字来欺世盗名，我觉得这些都是我不太愿意干的事。我该如何描述我们目前的生存状态呢？

　　或许一些人的生存状态，在某种程度上能反映出这个社会的本质。

　　我不想对生活发表一些什么评论，我想我们每一天都在生活中体味着人生。我们无法把握自己的命运，因此我们也难以预料生活中不虞的打击。几个月前的一次网上遭遇，几乎让我彻底丧失了对网络的信心。那是在我上网后的一天下午，我在进行电脑垃圾清理时，在用添加/删除程序删除两个无用的程序时，突然之间我在D盘上几乎所有的文件都不见了。我不禁大惊失色，因为在D盘中我几乎保留着自己所有重要的文件，里面有我一年多来写的文章、日记和下载下来大量的文件。在此之前，我根本没想到会出现这样的情况，于是我的大部分文件也未备份，这次误删除把我打个措手不及。不是说电脑中几乎有生活中的一切吗？于是我连忙上网寻求帮助，进聊天室去询问专家，我连续敲击键盘可没有一个人回答我的问题，我感到很失望；于是我又去有关电脑网站的站点，下载能进行磁盘文件误删除恢复的软件。在一个小时之中我一连下载了四五个软件，可对这些软件如何操作我又很陌生。于是忙乎了一两个小时还是没解决问题。我感到特别伤心的是，并不是我的那些文章和下载文件的突然丢失，而是我近两年来在电脑上所写的日记都不见了。这些日记可都是我心血和真实内心世界的记录呀，不行，假如有百分之一的可能的话，我也要把它找回来。于是我又上电脑公司请教，发Email求援，可最终也未能挽回这次损失。我只是用RECOVER软件恢复了一小部分被误删除的下载文件，而最重要的日记却永远也找不回来了。我一直以为，日记是一个人生命最真实的记录，是写给自己的将来和后人看的。前不久，在东方时空中看到一位网友把自己的日记在网络上公开发表，并得到了某位著名作家的首肯，虽然事出有因，但我总觉得有一些赚取名利之嫌。不过，这只是我个人的看法。这些都是废话，暂且按下不表。

　　网络为我们参与生活和表现生活提供了极大便利和自由度，我们可以按照自己的想法随意描摹生活和反映生活，这也就为我们在文章的内容和形式上的创新提供了极大的可能性。我不知道“生存状态”一词是否能涵盖我们目前生活中的一切，我只知道这四个字能把我生活中的一切遭遇概括进去。我不知为何我总觉得自己正生活在一场“危机”之中，我不知其他人是否与我有同样的感觉。在目前这个飞速发展、瞬息万变的世界里，我感到了一种深沉的危机感，这种危机感来自工作，来自生活，来自家庭。在目前这个激烈竞争的时代，我们很难对生活中所发生的一切保持一颗平常心。我们传统的价值观和人生观在剧烈的社会变动中被撞得“头破血流”，面对这一切我们只能随遇而安、麻木地生活着。

　　我很难描绘好我们的生存状态，我们是一天希望、一天失望地生活着。在生活中，我们的希望越大结果我们的失望也就越大。我们的理想，就像心中无数的美丽的肥皂泡一样，转瞬便破灭了。我这样说，并不是我对生活是悲观失望的，而是想说明我们现今的生活并不完美，理想与现实是那么地遥远；当我们所有的一切努力都变得徒劳无益时，我们该对生活说些什么好呢？！

　　我很羡慕我们的童年，童年是人一生中最快乐的时光，没有忧虑，也没有人生的烦恼。可是，随着年龄的增长，人的烦恼是越来越多，生活的艰辛和人事的繁杂有时令人喘不过气来。名利虽然总是身外之物，可我们总难看穿这一切。我不知我在写这一切究竟是为了什么？是为了给网站投投稿，还是想彻底表白一下自己的心声？我不知道。

　　我觉得我的生存状态并不理想，可我却发现在生活中大多数人也并不如意。在社会这个“金字塔”的底部，我看到有无数人生活在疾苦之中。他们为了生存而艰难的度日着，这部分人中有下岗工人、失业和个体劳动者等。在社会表面的光环笼罩和光芒四射下，我们决不能漠视基层广大人民的疾苦，因为人民才是我们真正的母亲啊！

　　匆匆写下这篇文章，用以表白我们的“生存状态”，并以此纪念我们广大的、勤劳的、善良的劳动人民。
