爱情的神秘源头

李慢臭美娃


　　英国查尔斯王子与卡米拉的恋情，为世人所不解，但查尔斯王子对卡米拉深情迷恋25年。他宁可冷落如花似玉的娇妻戴安娜，而追求一个长相平凡、看起来比他大许多的女人。在电话中，查尔斯王子狂放热情："哦，天啊，我真想住在你的裤子里！"他与卡米拉的偷情话语，简直无法与他严肃的形象联系在一起。是什么解开了查尔斯王子的情感枷锁，为什么在世人眼中毫无魅力可言的卡米拉，却启动了他的爱情电波？关于爱情，过去一直是诗人、艺术家的课题，现在，科学家也开始进人爱情的领域。

　　爱情的魅力究竟从何处来？又是什么让两个人相爱一生？科学家、心理学家，以及坠入爱情当中的男女，正努力地寻找爱情魅力的神秘源头。

爱情是一种化学变化

　　心理学家荣格曾经说过："两个人的邂逅就像两个化学物质的接触一般，如果其间发生任何反应，两人都会因此而改变。"

　　当一个人坠入情网，那种"火烧心头无药医"、"被爱冲昏头"的种种生理现象，其实正是爱情魅力发威的时刻。

　　人类学家海伦·费雪，曾经长期研究爱情的行为，她发现我们称之为迷恋或吸引的强烈情感波动，可能起源于一种叫苯氨基丙酸、简称PEA的化学成分。它会在脑中引起兴奋、狂喜和欣快的感觉。

　　但是PEA如何在脑中发挥作用？又是什么触动了PEA的上升或下降？

　　事实上，PEA不只能在体内发挥迷恋的效用，任何追寻刺激的情况下，它的量也会高涨。

　　已经有研究显示，人们在危险的情况下比较容易坠入爱河。还记得电影《泰坦尼克号》中，男女主角杰克与罗丝，如何在泰坦尼克号开始下沉时，爱情反而步步生长。罗丝在救生艇即将开走的一刻，又跳回泰坦尼克号，宁愿与杰克沉沦海底，也不忍独自逃生。而杰克在最危急的一刻，猛然感觉到爱情的力量。他拉着罗丝的手，感性而感伤他说："在这船上遇见你，是我生命中最美好的事。"

恋爱是脑部暂时短路的现象

　　爱情使人心荡神迷，再理智的人，都恐怕无法拥有抵御爱情的武器。事实上，科学家认为，每一次坠入情网的刹那，都是一次脑部短路的现象。

　　爱荷华大学脑神经学教授达马俊发现，当人陷入异性的吸引中，脑部的情感中枢就主导了我们的行动，我们脸红心跳、手心冒汗，满眼都是蝴蝶飞舞。

　　这是因为理性脑通常会探出头去预防危险，了解环境的复杂性，再决定采取行动或不采取行动。如果男女之间交往，每次都需事先想清楚所有可能的危险与后果，他们可能都会落荒而逃。

　　然而，脑中主导感情的部分，有时会独立运作，正如人遇到危险时，脑部会发出"战或进"的立即反应一样，当遇到两件相吸的时候，理性的思维便会让路，而靠直觉反应采取行动。

爱情基因

　　男女之间的调情、诱惑，甚至基因，也都解释了爱情魅力的部分来源。

　　张爱玲的小说《红玫瑰与白玫瑰》中，谈到女主角红玫瑰，拥有婴孩的头脑加上成熟妇人的身体，是最具诱惑的组合。

　　美国基因专家指出，女性的最佳爱情策略是精挑细选，只和愿意花时间和心思在自己身上的男人为伍。

　　但是爱情从来电的刹那到步入交往的过程，脑部理性的功能慢慢恢复了作用，原始行动不再主导两人的关系，爱情也进入更复杂多变的阶段。

　　为何爱上的是他，不是他？

　　如果单从生物演化观点解释爱情的魅力，则无法解释人世的许多爱情，就像查尔斯王子会爱上卡米拉，是完全不合生物演化逻辑的。

　　爱情的迷人之处，就在于它无法单纯用科学解释，因为它不仅受基因、生理现象所影响，也于个人情感的产物。西方有一句谚语："想要观察一个人，最好观察他怎么恋爱。"

　　关于恋爱的心理研究显示，一个人保护自己的方式，往往决定了他会成为什么样的情人。例如一个缺乏安全感的人，往往会成为一个焦虑矛盾的情人，一旦关系步入稳定期，就会开始挑剔对方的种种不是。一个自信主观的人，经常可成为一个完美的照顾者，什么事情都安排得好好的，可是却会逐渐成为一个冷漠的伴侣。一个处处为别人着想的人，可能会成为一个温暖、善解人意、愿意为爱人牺牲自我的情人，但是最后加也不免感到疲累、空虚。

　　这些特质引领人们寻找与自己相似或可互补的情人。例如，查尔斯王子之所以深恋卡米拉，也许正是她所散发的母性温情特质满足了王子渴求依靠、温暖的内心世界。《泰坦尼克号》的男主人公杰克，也正是以他不驯、冒险、狂放的特质，解救了被上流社会的虚荣世俗所窒息的罗丝。当然，这只是一些粗略的典型，在真正的爱情中，是不断变化、成长的，两人的角色也会随时演变。而这其中改变的过程往往才是决定爱情是否能够持久的关键。

喜新厌旧，也是基因在作祟

　　在脑部深处，有一块叫做"伏隔核"的区域，释放出多巴胺使人觉得非常舒服。如果多巴胺减少，人会变得严肃、自制、沉静，不愿追求新鲜感。

　　多巴胺是一种酪氨酸，是许多食物中都可找到的常见氨基酸。光是多巴胺还不足以给我们冲劲，它只是开锁的钥匙，而钥匙则称"受体"'，这是在脑部细胞表面的大块蛋白质。多巴胺进人等待的症状之后，旋即启动，脑部开始一系列变化。

　　一个以色列的研究小组发现一个奇特的基因，就是生产多巴胺受体的基因--D4DR。拥有较长D4DR基因的人，追求新鲜感的分数比拥有较短基因形式的人高。

　　要了解追求新鲜感是否对爱情美满造成影响，学者以祖克曼的测验来测试和异性交往达3个月以上的人。结果发现，追求新鲜感的程度，对双方关系及性生活有至关重要的影响。

　　追求新鲜感的程度是高或低并不重要，重要的是情侣双方要相当。在关系之初，追求新鲜感程度高的伴侣也许对程度低的伴侣并无不满，因为两人之间还有太多新事物有待发掘；而程度低的人也可能因高程度者"让人兴奋的个性"而受吸引。

　　但随着时间渐长，高程度者因低程度者对新事物毫无兴趣而感沮丧，而低程度者则因高程度者的不可测和无定性，而觉得困扰。

　　追求新鲜感不只会影响长期的两性关系，也会波及露水情缘。程度高的人乐于和许多异性伴侣交往，喜爱尝试新的性行为，视性为享乐游戏；而程度低的人与异性交往较少，性行为的姿势传统而有限，并且视性为一种承诺。

　　下次找伴侣可能该问：他（她）的D4DR和我相当吗？

（于丹丹摘自《女友》）
