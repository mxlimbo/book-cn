名人轶事

盛世良


永远的赢家

　　处在研究所中的玻尔无疑是严肃的，但生活中的他则轻松诙谐，他的同事甚至可以鼓动他一起去看牛仔片。正是在那儿，他提出了一种关于自卫者先放枪的著名理论。银幕上，英雄把企图枪击他的每一个匪徒都打倒了。玻尔认为，这种成功的心理原因就在于，一个匪徒决定掏枪来射击英雄，他的动作就会较慢，而必须立即动作来保卫自己的英雄就能够更快地掏出自己的枪来把他打死。

　　另外3个电影迷不肯承认这一点，因此就买了几支玩具手枪来检验玻尔的理论。4个人各拿了一支手枪，怀疑者们装坏蛋而玻尔装英雄；用的是"砸炮儿"手枪，因此枪声可以清楚地听到。而使每个人都大为惊异的是，玻尔永远能够第一个开枪。他的同行们不得不感叹："玻尔永远是赢家。"


去船厂打工的沙皇

　　要知道彼得大帝为什么被称为"俄罗斯之父"，就要先了解在他之前的俄罗斯和当时文明世界的发展差距。

　　在欧洲人发现新大陆时，他们很多人对东部斯拉夫民族的印象，不过是"传说那里有一个莫斯科城而已。草原上众多的斯拉夫部族，几个世纪以来便是一盘散沙，厮杀不断。直到15世纪，俄罗斯人才勉强形成中央集权国家的雏形。不过，这个拥有整个西伯利亚的国家，却没有工厂、商业、不冻港、海军……农民几乎全部目不识丁。

　　1697年8月，荷兰的萨尔丹造船厂来了个新手。船厂的学生和技师不是没看过外国人来学造船，不过这个叫彼得的新手特别引人注目。一来是他那2．13米高的个头，二来是他做事快手快脚，比谁都勤奋，随身带的本子记下了满满的字，也比谁都认真。不过造成更大轰动的，是荷兰商人在一次探访船厂时，认出他竟然是当今的俄国沙皇。

　　身为沙皇的彼得，梦想俄罗斯能够夺取出海口、建立强大海军，进而取得霸权，这就是为什么他会出现在萨尔丹的原困。告别萨尔丹后，他又去了阿姆斯特丹、伦敦、维也纳……

　　这次西欧之旅，是俄罗斯大整顿的前奏。彼得似乎觉得凡是过去的、拜占庭式的，都必须废除。一名守旧的贵族曾酸溜溜地说："他最希望的，是一觉醒来，莫斯科突然变成了巴黎。"

　　于是，莫斯科人也开始喝咖啡、吸烟，学校和专门技术学院多起来了，一些留学生被派到欧洲去。

　　为了获取一个出海口，1700年，俄罗斯和北方强国瑞典在纳瓦尔进行了一场艰苦的战争，配备和训练都不足的俄罗斯军队一败涂地。这次失败后，彼得开始了一连串的整顿军力行动。

　　3年后，彼得再战瑞典，拿下芬兰湾的纳瓦尔后，他决定在此兴建新的都城。

　　他发动4万农民，连冬天也施工。结果疾病、辛劳，加上瑞典军队的威胁和破坏，导致许多人丢了命。不过这种与天斗的意志力终于有了成果，1725年，被封为大帝的彼得去世时，圣彼得堡已成了北欧最大的都市。

　　而彼得大帝曾给新俄罗斯的，除了这座先进、典雅的都市，还包括20万精兵、50艘军舰、横跨欧亚大陆的版图，以及全新的生活形态。

（庄林炎摘自《广州日报》

"音乐皇帝"

谢承志

　　据说，拿破仑在歌剧院里看歌剧，见另一个包厢里坐着著名的作曲家罗西尼，就叫侍从请他过来。罗西尼赶紧来到拿玻仑的包厢，跪下请罪道："皇帝陛下，我没有穿晚礼服来见您．请恕我大不敬。"拿破仑语出惊人："我的朋友，在皇帝与皇帝之间，礼仪是不存在的！"从此，罗西尼便有了"音乐皇帝"的尊称。

（李鸥摘自《交际与口才》）
