网络富豪

　　《邮报》刊登的"1000年财富报会'中，前300名最富者中首次出现18名因特网企业家。

　　《星期日泰晤士报》也刊登了一份年轻百万富翁名单，其中因特网企业家的数量占很大比例。

　　"自治"软件公司总裁迈克·林奇在《邮报》的财富排行榜中名列第28位。他拥有的市场价值在不到6个月的时间内从5000万英镑猛增到7．5亿英镑。

　　特里·普卢默和韦恩·洛克纳通过向英国电信业巨人--派达丰--空中联系通信公司销售困特网视频游戏和软件而发了财，如今他们每人拥有3．68亿英镑的资产。而就在一年以前，他们还债台高筑。

　　提供旅游网上预订服务的"电子预订"公司（eboker．com）总经理迪内希·达米贾也出现在《邮报》的财富排行榜上，资产价值达1．53亿英镑。

　　在《星期日泰晤士报》的财富名单上还有30岁的查尔斯·纳赛尔，他1996年创办了因特网接入服务公司ChaNET，如今拥有资产3亿英镑。

经济人自利不自私

　　经济学家说，经济学研究的是人类的行为。但是，世界上没有相同的两个人，他们的行为可能大相径庭，经济学家如何克服这个矛盾？很简单，经济学家从不关心具体某个人的行为，他们关注的是人类行为的共性。在他们眼里，千差万别的活生生的个人都是经济人：不懈地追求自身最大程度满足的理性的人。显然，经济人是自利的，但自利不等于自私。举个例子来说，一个虔诚的基督徒由于相信上帝的原因，会充满行善的愿望，他人得到幸福时，他会觉得自己更幸福--他是自利的，但并不自私。

（摘自《画说经济学》一书）

世界正在进入"E时代"

　　《数字时代》杂志发行人认为，世界正在进入"E时代'。随着"运算科技"赋予的力量和"运算成本"降到不可思议的低廉，"E时代"即将来临，它将改变一切定义、一切规律，这涉及经济、生活、学习和阅读等很多领域。

　　E是英文第5个字母，它可以是人类听觉和视觉感官的来源--耳朵（ear）、眼睛（Eye）；也可能是人类感情起伏的讯号--情绪（Emotion）；也可能是地球生态（earth、ecology)的环保(environment）主张。E更代表一切由电子（ElecttOlllC）出发的沟通和活动--电子邮件（E－mail）、电子商务（e．busines。）等。综合分析"E时代"，它就是结合人文、科技和自然元素，让彼此交互影响而形成的数字时代。

（钱涛摘自《世界知识》2000年第6期）
