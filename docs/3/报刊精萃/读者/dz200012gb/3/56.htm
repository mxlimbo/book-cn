零储蓄族的消费观

曹伟容 石原


　　在现代都市，有这么一群人，他们出生于70年代，他们既无银行存款，又无自己的住宅，二十几岁仍与父母住在一起，交很少一点的伙食费，大多数收入花在服饰、化妆、泡吧、旅行及交友上。他们总是叫穷，买一件贵一点的时装或出去旅游一次，都需要父母或亲友支援，而他们的年薪却往往超过父母的总和。他们就是时下颇为"时髦"的"零储蓄族"。

零储蓄不等于不投资

　　尽管现实情况与预期目标仍有差距，但有一部分人已勇敢打破了"积谷防饥"的传统消费理念，将收入再次投入流通领域，以期产生更大的效益。张先生即属此类人：今年28岁的张先生自办一家有关橡胶、化工方面的公司，他所得收入的一大半均投入股市，其余的也都用于生意上的资金周转。在他看来，将钱存入银行，光靠"吃"利息是发不了财的。

　　其实，像张先生这样存有此类消费理念的人还不少，他们既不算"保险柜主义者"，也不算信奉"今朝有酒今朝醉"之人。"如此消费理念，能考验一个人是否具备经营头脑。"张先生一言以蔽之。

　　周小姐每月近两千元的收入，她坦然承认自己是无储蓄族，让咱们瞧瞧她一个月是怎么花光1000至2000元的收入。

　　租房：周小姐是外地人，所以租房成了一件大事。周小姐与人合租，每月平摊下来，房租加水电费200元。

　　生活费：没有起点，也没有止点，但周小姐给自己的生活费打了一个保守的数字；四百元左右。

　　置装费：女人爱靓，周小姐也不例外，每月的置装费要300元。

　　三样加起来已是近千元，然后再加什么化妆品、小饰物、交通费什么的，一个月一千多元，一眨眼，就不翼而飞。用周小姐自已的话说"开销那么大，我哪里有钱剩！"至于嫁妆，到时候再说啦。

零储蓄族：心无杂念享受眼前

　　有八成的零储蓄族不得不与父母住在一起，虽觉得不太方便，但自己没钱买房子。只得将就。对于零储蓄族，买商品房，近乎于神话，就算父母帮忙预付一半，另一半每月按揭支付，也是一个梦想，除非父母一次性拿出几十万，一下子担纲。

　　然而，大部分零储蓄族并不以此为忧。他们的口号是：心无杂念享受当下。并解释：如果你思虑过多，是当不得零储蓄族的。他们认为；为了要储蓄，就会使自己的生活从"气态"转化成"固态"。犹如小鸟进了樊笼、鱼儿离开水一样来去自由。如此看来，零储蓄族好像对未来的生活看得很高远。然而，疑惑的是，享受当下的理由；正是否决了通向未来的数条路，却又没有探出自己真正要走的那条路来。

新人类的消费观

　　零储蓄族的消费很大-部分在信息、技术，再学习以及文化投资等上面。在许多场合实际分不清楚是学习，工作，还是享受，如交际、看演出乃至游览等。

　　有位老父对客人抱怨："我这个儿子，身上有10元钱，就敢'打的'，好像有十万八万似的。"。儿子却说："有十万八万干吗还要'打的'。有5万够付首期，我就买小汽车去了，有了小汽车，我就能挣更多的钱。"

　　在新人类看来，"只挣不花，多挣少花"实质是当守财奴，算啥美德？而拼命地挣，积极地花，做一头勤耕地、又勤吃草的牛，竭尽牛道，才是好牛。

过来人看零储蓄

　　"零储蓄""说对大多数整天围着锅、碗、瓢、盆打转，为柴米油盐发愁的"过来人"而言，实有"空中楼阁"之感。五十开外的许先生如是说："我们年轻时，虽说工资不高，只有见十块钱，除了孝敬父母，贴补家用外，剩下不多的钱还得攒下来，作为往返探亲的旅费。当时情况下，我也称得上是零储蓄族，然而不是不想存、只是没钱可存。"'

　　从"没钱可存'"到"有钱不存"，、从"多挣少花"到"能挣会花"，从两代人身上看到的不仅是消费观念的变化，而且更多地感觉到了时代的变迁与进步。

（丁零摘自《广州日报》2000年3月25日）
