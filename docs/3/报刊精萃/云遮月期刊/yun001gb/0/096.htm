两只鸟

地点：云遮月客厅
主持：蚊子

蚊子：

　　在一片田野的边上，有两棵树，一棵高，一棵低。

　　在这两棵树上，有两只鸟。

　　有一天，站在高树上的鸟看了看低树上的鸟，它想了想。

　　说了一句话："你看它过得多好呀，那么低，可以清楚的看到下 面庄稼上的虫子。也不用飞得那么远。多好呀？" 同时，站在下面的树上的鸟也在想："上面的那只的日子过得多好呀，它吃饱了以后就可以站在上面看风景，那是多么美妙的生活呀。如果我站在上面就好了。"

　　于是，它们俩就互相的换了位置。

　　可是，很快的，它们就又想换回来了。

　　因为，上面的那只鸟发现在下面虽然看得清虫子，但是吃饱了以 后就没有什么事可以干了，很无聊。

　　下面的那只鸟发现在上面确实可以看风景，但是上面的风雨太大了，每一天都很担心。

　　于是，它们就又换了回来 。

　　再然后，它们就在这种换来换去中过起了日子，但是谁也没有找到它们理想中的好地方。


举杯邀明月：
　　是呀，每个人对自己所在的环境都是不满意的。

　　没有满足的，也许很多人把这个叫做追求。

　　我不喜欢那样的生活，我很容易满足的。

　　虽然也有自己的追求，可是我追求的是一种。

　　恬静、一种潇洒，最好是与世无争的生活。

　　可是这样的生活是没有的，我只好继续的努力……拼搏……奋斗……，实现自己的理想和目标！


乔燕：

　　我曾经是一只鸟！

　　我想我一定做过其中的一只鸟，但我还不能肯定地认出哪一只鸟身上有我的影子。

　　在低外的那只？对！是在低处，我感受过对别人的风景的仰视，羡慕......，可在我想要高飞的时候，我想到了高处的风寒。

　　是在高处的那只？对，是了！我曾经拍拍翅膀心想，其实不用飞这么高吧，我累了！......而当我想要降落时，我却又担心，以后我还能飞起来吗？

　　我就这样徘徊了很久，现在，我明白了，如果只看到失去，我永远不会安宁，而我可以在放弃中保留，需要奋斗时我就降低自已的要求，需要享受生活时，不妨给自已心灵的自由，我在同一个时期只能选择一个侧重，但我的一生将包容这几种风景。

　　于是，我拍拍翅膀，飞向云遮月这棵小树，因为它需要我的鸣叫，我还得谢谢蚊子这第二个好故事，我因此又成为一只鸟！！！

幽草：

　　在这世上，万物生灵，都有不满足于现实的时候，这山望着那山高看来想做个知足的人真是很难！！！！

　　如果有来生，我情愿做一棵小草，默默把大地绿化

蚊子：

　　我想不论是谁可能都有过两只鸟的经历。

　　不过风雨过后就会有晴天的。

　　所以经历过两只鸟的经历的人才会懂得人应该知足就好。

　　希望朋友们以后都能够时常的用两只鸟的故事来跟自己谈谈心。
