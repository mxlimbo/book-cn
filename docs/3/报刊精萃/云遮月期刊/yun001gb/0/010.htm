闲　言

奇石

　　四周很静，看看表，已经凌晨四点，裹了裹厚重的棉衣，点上一根烟，深深的吸了一口，盯着外边的雪花，人还在，景色也没有变，变的只是时间和被时间改变的心情。在这一刻，我感到了刻骨铭心的清冷，清冷的就如日本俳句里的古池，蕴一池的寂寞，等了很久，也没有等来一只入水的青蛙。

　　活动了一下已经被冻的有些僵硬的双手，脑海里不知怎么了又回荡起那已经忘记歌名的老歌，禁不住哼唱起来：

　　当初背负多少期望和祝福／才肯面对茫茫的旅途／而今储蓄多少创伤和收获／才肯回头／曾经是你是我最怕的风波／都能化成彼此的承诺／而今不必为你归航的心情／才肯想起那个日见荒芜的港口…………

　　歌声在静夜里回荡，一遍又一遍的唱着，渐渐的声音竟有些呜咽，年少时轻狂的梦早已和现实撞击的支离破碎，唯一记得的是一直的向前走，不想，不能，也不敢停泊，可等到真的想要停下来的时候，却发现前方却是没有方向的路口。痛苦在一口一口的吞噬着感情，绝望在一点一点的消耗着生命，当初的青葱岁月，书生意气，剩下的，就只有这一首无调的歌罢了！

　　爱和被爱，就如坐风卧云，似乎都没有发生，那在我耳边喃喃的说着要永远做我妻的女子已经远在天涯，天涯真的很远，和永远一样，不是脆弱的心灵能承受的距离。几百年前那艘沉入海底的泰坦尼克号，和几百年后的IF I JUMP DO YOU JUMP？让人想起时，恍惚间总有些疼痛。若把爱当成梦，梦醒了，也就不会隐隐做痛，伤心人自有怀抱，断肠人各有因缘，该来的还不知道，该去的都已经去了。百年来的感动，都交给了三个小时的泪水，当走出电影院，一切都没有变，灯火依旧阑珊，现实还是现实，回归到自己的位置，没有奇迹发生。
　　杨过从公孙萼手中接过情花的花瓣吃了几瓣，入口香甜，芳甘似蜜，更微有醺醺然的酒气，正感心神俱畅，但嚼了几下，却有一股苦涩的味道，待要吐出，似觉不舍，要吞入肚内，又有点难以下咽。这，便是爱情的的味道。爱情是最奇妙的一种感情，奇妙之处就在于它可以让快乐和痛苦并存，就如回忆一样，从来不问你是否愿意，仿佛锐利的刀锋，它的速度和力量没有人能够抗衡，当你感觉到剧烈的疼痛和快乐的时候，它早已深深的刺入了骨髓。当你我都以老去，就让我们躺在摇椅上，一边感受着回忆的温柔和刺痛，一边低吟着这首情诗！

　　"当你老了，头白了，睡思昏沉，／炉火旁打盹，请取下这部诗歌，慢慢读，回想你过去眼神的柔和，／回想它们昔日浓重的阴影；多少人爱你青春欢畅的时辰，爱慕你的美丽，假意或真心，只有一个人爱你朝圣者的灵魂，／爱你衰老了的脸上痛苦的皱纹；垂下头来，在红光闪耀的炉子旁，／凄然地诉说那爱情的消逝，／在头顶的山上它缓缓的踱着步子，／在一群星星中间隐藏着的脸庞。"

　　正象我不知道我从何而来，我同样不知道我往何处去；我仅仅知道在离开这个世界的时，我就要永远的或归于乌有，或则是落入到一位愤怒的上帝手中，而不知道这两种情况那一种应该是我永恒的归宿。这就是我的情形，充满着脆弱和不确定"人生大概就是如此吧！！拥有的只能是失去，就象手中捧起的水，无论时间的长短，水终究会从指缝里流走的。我尽量努力尝试摆脱酒神精神在我体内的影响，有时候曾经幼稚的想，既然这个世界已经这么样了，何必那么认真呢，不如尽量的自私、贪婪、玩世不恭、取消深度、玩弄虚无……。帕斯卡曾经说过"人必然是疯狂的，不疯也许是疯的另一种形式"可是我却发现我还疯的不够，我没有力量面对真诚，善良，温馨和感动。在这些情绪的面前往往会不知所措，象个傻瓜一样不知该怎样。我想也许就只能在痛苦和毁灭中得到悲剧性的陶醉，而不能如日神那样，只爱生命的过程，不去追究它的实质和归宿。

　　一切悲剧皆因死亡而开始，一切喜剧皆因婚姻而告终。就如讨厌悲剧一样我向往快乐，所以时间在一点点的过，日子在一天天的熬，不想让悲剧发生，也不愿喜剧的结束，既然如此，那就……那就……不如……这样吧！

　　亲爱的朋友啊！请再一次暂时的最后握我的手，等下一次落雪，便是我们相聚的时刻。
