传统灰色

主持/璋鲈

　　前天中央电视台《生活》中讲了一件事，一栋楼房的外墙刷成了红色，引起周围群众的不满，于是，北京建设主管部门决定今后北京的建筑外墙装饰以灰的调和色为主。看到这里，使我想起了很早以前读过的一篇文章，讲的是"中国文化由善的出发点向人们提出了过高道德要求，由于它无视人的基本欲望和正当权利，因此便产生了人们一方面要表现自己是个仁人君子，另一方面又不可能灭掉所有"人欲"的心理矛盾。这种矛盾使人们在不敢面对现实也不敢面对自己，因此也就势必造成全民性的虚伪，这种虚伪促使人们努力把自己封闭起来，使自己汇入人的群体而不为人所注意，在这里缺乏个性的灰色无疑成了一种最佳保护色。

　　因为灰色是无法让人洞穿人的性格，这给我在他人面前的表演提供了余地"。中国的传统色彩我看就是让人感到晦涩的灰色，黄色，红色也只是权利的象征，主管部门的决定我不知是忧还是喜？若是迎合传统，人有自己的看法，若是提倡个性化，人也会有自己的观点，是亦非亦？


乔燕：

　　"使自己汇入人的群体而不为人所注意，在这里缺乏个性的灰色无疑成了一种最佳保护色。"

　　说的好！

　　是这样的，人们既不想让自已在人群中显得太突出，又不希望别人比自已突出，这种随众的心理不是一日养成的，它暴露了我们的文化传统和教育方式中存在的问题。


璋鲈：

　　中国是一个文明古国，有着悠久的历史，但不可排除其低劣的一方面。任何事物只要是真实的，就应该有缺陷，就不完全合于"善"，现实的缺陷和趋向于"善"的文化心态，使我们民族审美意识极少采取参与型更多地是把自己和现实拉开一段"心理距离"从而在"静观"和想象中去完成对象的完美化。这种审美心态表现在色彩审美中，就是对灰色的偏爱。

　　乔燕：你的话又使我想到了这些，中国有句成语："木秀于林风必摧之"也是在讲这一道理。对吗？
