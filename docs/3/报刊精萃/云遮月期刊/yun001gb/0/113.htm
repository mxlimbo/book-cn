好象有一朵云在空中飘

敏感之花＊阿明/文


　　如果看到了地面上的冰雪融化了请通知我，我没有看到过冰雪融化时的情形......也许没有这种机会，也许。。。。也许没有，也许！化成了水的的冰雪汇成了小溪，流进了小河，再奔向大海的怀抱。将一口即将喷出的鲜血咽了回去，我知道了血的味道了！我真的品尝到了鲜血的味道了，自己的鲜血的味道......
　　忽然好象有一朵云在漫无目的的飘，真的不知道飘向什么地方，连它自己都不知道......
