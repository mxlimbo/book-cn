北海故事——狼桥遗梦（一）

萧十一狼/文

一

　　看到了太多的关于网恋的文章和小说，从来也没有想到过网恋这种东西也会落到我的头上。

　　刚学会上网的时候，成天都沉迷于痞子蔡的《第一次亲密接触》和李寻欢的《迷失于网络与现实之间的爱情》及《边缘游戏》这些关于网恋的小说之中，无数次的被小说中的情节所打动，觉得网恋是一种很纯的情感，让我很是向往，总希望能够有一天在我的身上也能发生一个类似痞子蔡和轻舞飞扬或者乔峰与风影那样的故事，虽是悲剧，却也能永恒。对于很多只苛求曾经拥有的人来说，人生中最完美的一段情感莫过于此。

　　我也不能例外，那时的我成天都泡在云南信息港的星缘聊天室里，尽管我已经有了女朋友，但我还是希望能从聊天室纷乱的人群中找到属于我自己的一个轻舞飞扬，找到属于我自己的一份真挚的情感，即使它只是一段插曲。

　　但很快的我便对这种游戏失去了兴趣，在聊天室里没有人会把你的真诚当真，大家都在嘻嘻哈哈的开着玩笑，真即是假，假亦是真。你可以对别人真诚，但你却无法让别人相信你是真诚的。

　　而让我对寻找我的轻舞飞扬真正失去了兴趣的原因是通过对网络慢慢的熟悉，我发现那些无数次让我感动不已的《第一次亲密接触》和《迷失于网络与现实之间的爱情》原来也是假的，原来也都是一些虚构的故事。《第一次亲密接触》本身就有一些很站不住脚的情节，

　　《迷失于网络与现实之间的爱情》也由于写得太过于真实而变得也不是那么的真实了。而最具讽刺意味的是李寻欢的女朋友曾无数次的质疑李寻欢那个叫风影的女孩子到底是谁任李寻欢如何解释都不相信。

　　我也逐渐远离了云南信息港的公共聊天室而投入到了云之南这个虚拟社区里。那时的时间大概是2000年5月。我象一个孤魂野鬼似的在云之南里四处游荡，看看帖子，偶尔看到能引起共鸣的帖子也就跟上个诸如'写得真好，我真感动'之类的回帖。我不再有任何寻找轻舞飞扬的想法，不再有任何一丝的感情。我对那些风花雪月的网上爱情故事我总是嗤之以鼻，我甚至都懒得去打开那些帖子看上一眼。在我的心里，那些都是些虚无缥缈的东西，都离我是那么的遥远。

　　就这样，我在云之南里无忧无虑的飘着，没有任何名气，没有人跟我说话，没有人跟我聊天，没有朋友，甚至没有人认识我。我象一只孤独狂傲的狼一样独来独往。

　　我以为自己已经看破了网上的一切。

　　这种想法一直陪伴着我直到7月份的一天乔的出现。
