飞 燕

乔燕/文

燕儿剪剪破九天，
清风扶翅意翩跹。
东南西北无觅处，
暗筑小巢水云间。


晚风/文

今日有雅兴，与燕子和一首：

小燕弄风风低语
展翅高飞向九天
可有凌云抒壮志
春风再入水云间


leyang/文

大家有兴致，我也小漏才华

沮丧时，我引亢高歌
悲伤时，我开怀大笑
病痛时，我加倍工作
恐惧时，我勇往直前
自卑时，我换上新装
不安时，我提高嗓音


永远的方舟/文

燕儿剪剪欲破天，
可知寒冬春尚眠。
不如筑巢蓄羽翅，
来年更上九重天。


燕十八/文

燕儿剪剪向九天
断翅难飞心不甘
待到春暖花开日
冲天而起弹指间


八神苍月/文

燕儿剪剪破晓间
春洒大地艳阳天
春风吹破寒冬雪
乳燕出巢到人间


空谷清音/文

寒冬，深夜
故人写信
心结


乔燕/文

几度浮沉云遮月
我心依旧燕归来


如云/文

但愿冬天快些过去。
我在盼望着春天。
盼望着苏醒。
