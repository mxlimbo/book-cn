整体的逃亡（诗论）

――给纯净诗歌的话
和菜头/文

　　我参加了本月初诗歌方阵的朗诵会。以下的文字是针对参加本次聚会的诗歌爱好者写的，权作为我在朗诵会上发言的补充。估计到文章将相当的枯燥无味，对诗歌本身并不感兴趣的朋友可以完全不用继续看下去；同时，对于沉浸在诗歌创作中的朋友，此文希望得到你们的激烈反对。我个人比较偏好思想上的冲突，以达到深刻和全面的彼岸。

　　一．在命名中的现代诗歌

　　在我看来，中国现代诗歌面对着两大阴影，一是中国古典诗歌，二是西方现代诗歌。

　　诗歌作为一种艺术表现形式，按照兰石先生的理解，具有净化心灵，追求美好的功用。这里强调的是诗歌的社会功能，而没有涉及诗歌本身的性质。诗歌本身我觉得是一种"命名"。诗人自己通过诗歌，对世间万物、人类情感进行命名，从而对事物打上自己的印记。我们之所以在阅读诗歌的时候会感动，是因为诗人独特的对事物的定义-

　　――命名，打动了我们的心灵。命名的意义在于：对人们没有感受到的东西的表达；以及对人们感受到却没有表达或者无法表达的东西的表达。

　　在中国的古典诗歌中，由《诗经》的无名作者开始，到各代诗人的丰富，可以说已经使用古文的形式对所有的事物和题材进行的命名。古典诗歌是完美而成熟的，我们不必遗憾于今人中没有写古体诗的巨匠，没有任何一个近代人能达到李、杜的艺术成就和水准。因为在漫长的历史长河中，各种诗人已经将几乎所有的事物的命名权瓜分一空，中国古典诗歌有成熟走向衰亡是一种必然。在两千年后，中国古典诗歌爱好者几乎没有重新命名的空间，在极其狭窄的空间里窒息而亡。

　　举例而言，崔颢写了关于黄鹤楼题材的七言律诗《登黄鹤楼》，对黄鹤楼的命名由于他诗歌的极高艺术成就，给后来者的重命名造成了极大的压力。在他的命名中，涵盖了几乎所有诗人想表达的东西，如寻仙的出世想法、思乡之情、对时间空间沧凉的感叹。。。。。。《唐才子传》说："李白登此楼曰：眼前有景道不得，崔颢题诗在上头。无作而去，为哲匠敛手云。"李白面对的这种尴尬正是现在中国古典诗歌诗人正在面对的。古典诗歌在唐代达到了发展的p

　　峰，怀乡、思人、咏景、自伤、悲秋、伤春、讽世等等题材都已经被完全命名完毕，这也就是后来宋朝词风大盛的真正原因。

　　词突破的诗在字数和格律上的限制，以一种更自由的文体重新对事物进行命名。所以在艺术上有了创作的新空间，然而，词也迅速发展到了成熟阶段，之后出现的戏曲、散文甚至是小说也说明了这一点。到了白话文写作的出现，正式宣布了古文体写作的死亡。白话文的出现并不是学者们临时的发明，也不是出于通俗的考虑，古汉语的发展成熟死亡，本身就指向了白话文的出现。作为成熟发展的肌体，古汉语必然走向衰亡，新的白话文必然取代。我们幸运的拥有了成熟完备的古典汉语文学库，同时生长在新语言的环境之下。这也就使我们的现代诗歌创作成为了可能。

　　正因为这样，古典诗歌的命名对于我们今天的重命名造成了直接的影响。我拒绝使用"压力"这一单词，是考虑到真正的压力承受者其实是今天的古典诗歌写作者，而现代诗歌的作者其实只是在古典诗歌的阴影之下，并不需要他们直接面对沉重的历史重压，寻求没有命名的小角落。现代诗歌的作者，面对的是一个成熟而完美的古典诗歌的传统，试图用新文体去重命名。而如何去重命名就是个很重要的问题。直接把古典诗歌里的东西用白话文重写一遍？甚至是直接翻译成为白话文？这样的做法是永远不能超越古典诗歌的，这仅仅是文本的复写。我们需要的是精确的重新命名，每个单词、每个句法、每个比喻都需要精确的重命名。

　　举个例子：眼睛是心灵的窗口，这是一个命名。黑夜给了我黑色的眼睛，我却用它寻找光明，这也是一种命名。新的命名还将出现，不断的丰富这眼睛这一事物。如果我们考虑到对眼睛的反射内心的命名，那么眼睛就可以和"暴露"、"投射"、"倒影"等等单词联系起来；如果我们考虑到眼睛搜寻的命名，那么我们也当然可以和"穿透"、"飞掠"、"漂浮"等等单词构成新的表现形式。这样的好处在于，我们终于可以在新语境下形成沟通，创造属于我们自己的东西。这种严密的体系是必须的，否则我们就可能看到"月升于野尻之侧"之类的句子，根本不知道对方说的是什么，诗歌变成了语言的堆砌，毫无意义和节制的自我复制和繁衍。同时，我们也避免了对已有的古典诗歌的简单复制。因为即使是相同的题材，由于我们的新命名方法，也可以成功的规避走入古典诗歌的语境。

　　古典诗歌掠夺走了大部分的诗歌爱好者，对现代诗歌不能说不是件遗憾的事。摆在现代诗歌写作者面前的事就是要创造自己的东西，足以和古典诗歌抗衡，或者平分秋色的东西。目前，这里依然是一片巨大的空白，这给了我们足够的空间去创作，去尝试。我相信，走出了传统古典诗歌的阴影，现代中国诗歌会有明媚的晴天的。

　　二．西方现代诗歌的影响和借鉴

　　中国现代诗歌的出现，是伴随着白话文运动一起的。在新诗的创作过程中，西方的诗歌对我们有非常巨大的影响。随着大量的外文诗歌的翻译传入，我们学习到了很多的写作方法的技巧。所以在某种意义上说，现代中国的诗歌是白话文和翻译体的综合，一点也不过分。在我们的时代里，最具成就的诗人几乎都是在使用翻译体写作，通过这种崭新的文体，达到了较高的艺术成就。

　　然而问题也因此产生，首先是翻译的问题。众所周知的一个事实是：语言之间的转化必然导致美的损失。无论是如何高明的翻译家都无法将原著一点没有损失的转化到另一种语言中去。所以，当我们阅读翻译的文本时，不得不遗憾的发现，在指向上，会有偏差。引申到命名上，我们无法还原作者本人的本意，容易发生误读。另一个危险是在于，我们很可能按照翻译的文本，以为西方诗人的水平不过如此，很容易超越。这样一来的结果是，我们放大了自身的实际水平，在弱小的参照系下，觉得自己的伟大，从而在艺术追求的道路上出现了偏差，和真正伟大和动人心弦的东西擦肩而过。或者是以为对方已经伟大到难以逾越的程度，对自己的努力和追求失去信心。无从构筑自己的精神花园。

　　无法正确解读西方现代诗歌的结果，将使我们失去一个宝贵的参照系。那么所有的尝试都将被视为合理的。好的诗和坏的诗之间的区别和分野将不复存在。诗歌将脱离了美学上的追求，变成一种信手的涂鸦。我们在阅读的西方文本中，基本无法看见西方诗人对语言的使用的微妙的精致的美，更无法体会他们的诗歌中音乐的成分。那么语言脱离了规范，意象失去了控制，必然的结果是诗歌的创作没有规律可寻。体现在读者与诗人的关系上，那就是双方无法沟通，诗人以晦涩为美，读者无法阅读。虽然说诗歌的创作方法上存在很多的手法，命名的方式各有不同。但是，诗歌应该是：准确、充实、清晰、深刻、丰富、坚硬的。而不应该是：浮夸、浮肿、重复、琐碎、堆砌、浅薄、混乱的。

　　试看海子诗歌中的这一段：

　　泉水白白流淌
　　花朵为谁开放
　　是这样美丽负伤的麦子
　　吐着芬芳
　　站在山冈上

　　在麦子这一命名上，它和生命之间存在准确的对应关系，麦子成熟了，意味着生命的成熟。生命因为没有聍听的对象，因此是孤独的。海子借用麦子，成功的表达了他对生命的看法。我们再看：

　　芦花丛中
　　村庄是一只白色的船
　　我的妹妹叫芦花
　　我的妹妹很美丽

　　这里的芦花，它的颜色和漂浮的性质，让村庄变成了船，而妹妹因为名字和芦花的关系，也变得美丽而纯洁。芦花的命名也是异常准确、清晰的。我们可以在第一首诗歌里明显的发现句子的音乐之美，单词具有的金属的光泽，坚硬而丰满。后一首里芦花的指向的任何一次改变都是无比的清晰和准确。我自己并不会写诗，但是我还能清晰的知道什么是好诗什么是坏诗。这样的诗歌读来令人感动，读者至少能知道他在说什么？如果仔细体会，更能发现其中的美。

　　我尤其反对在阅读了一些西方诗歌之后，完全放弃了诗歌的音乐美，或者将形容词、意象大量的堆砌，或者变化文法，使句子变得异常艰涩。这些东西都是不美的，也无从谈起打动人心的力量。我也反对超越文字的方法，重复日常生活中琐碎的部分，用所谓后现代的方法解构文字。失去了美，诗歌将全无意义。

　　我的建议是，大家努力做个双语诗人。读原著永远是真正理解作者的不二法门。唯其如此，我们才能全面的理解西方现代诗歌，这样也才能谈借鉴。否则，在误读之下，什么都可能出现，就是没有好的诗歌。

　　三．关于诗人之死

　　诗人必须活着，我们才有可能欣赏到更多的作品。虽然死亡也许是一种仪式，是不可缺失的一环，否则就会出现生命的残缺。但是我还是希望诗人能够活得长久一些，尽量晚点去完成他大仪式，我也就可以阅读更多的诗歌。

　　一种死亡的过程是来源于精神世界的完全崩溃。诗人中有很大一部分集中精力于自己的内心世界，希望能通过这种方式最大限度的贴近他所追求的艺术上的终极目标，去揭示更深刻更广阔的世界。但是这种方法是极其危险的，一个人的内心世界并不是波澜不兴的死水，愈是深入，就会发现下面的风浪愈是汹涌澎湃。当距离足够的时候，作为观察者，你是超然的，然而随着你的深入，你终将身陷其中，无法自拔。我相信一个人的精神世界如果是丰满而宁静的，这样的深潜是有益无害的。但是作为诗人，往往过着一种异乎常人的精神生活，因为自己的追求过于炽热、急切，内心世界其实是沸腾燃烧的地狱。过于的接近，它会燃烧你的双翼，让你做出最极端的选择。我希望喜欢在内心深处寻找的诗歌爱好者们，和你心里的猛龙保持一点距离。它的毒火炙烤着你，迫使你写下那些句子，流传在这世界上。但是别太靠近它，美永远都是危险的。我希望你们活得长久些，再长久些。

　　第二种死亡的方法是在于被"招安"。当一个人头顶"桂冠诗人"的头衔时，这个诗人就已经死亡了。今天的世界，是个媚俗泛滥的疯狂时代。当我们被流行价值穿上圣袍的时候，其实距离唱挽歌的时间已经并不遥远了。如果我们被官方语言承认为"伟大的诗人"，不用看就可以知道你写的是些什么东西。为了什么人，为了什么利益而写诗歌，本身对诗歌就是极大的亵渎。伟大的艺术家从来没有一个是因为报纸的吹嘘而长存于世的。另一方面，如果大众文化给你戴上了"诗人"的花冠，那其实是耶酥头顶的荆棘王冠，他们其实是在玩弄你，称呼你为"犹太人的王"。随后他们就会无情的将你抛弃，你的价值和一个歌星并无区别。所以我恳求你们，继续在没有太多世俗功利的网络上写你们的诗歌。拒绝所有招安的企图，纵使你一时拥有了掌声和鲜花，但你同时也就被埋葬了。我们之前有太多人已经如此死去。要相信自己的努力是会有被发现的一天，不要因为自己今天没有被主流文化接受而遗憾。我记得有一段话，是写耶酥的，也很适合你们：他一个人来到这世界上，安静的站立在湖边，凝视着湖水。人们匆匆经过他的身边，没有一个人知道他的名字。他不是为他们而来，他是为了将来千千万万的人而来。会有一天，后来的人会记住他的名字，他用手指写在水面上的名字。

　　四．在逃亡的路上

　　今天的中文已经被污染，各种沉辞滥调充斥在我们的生活中。领导视察，必然要"冒着"什么，比如：烈日，暴雨。贫困地区致富了，必然有一句："他们不等不靠，而是。。。"白话文中的成语和套话，已经增产了不知道几百倍。在目前的语境下，很难有大的作为和突破。作为真正追求美的人们，需要从这种语境中逃离开，避免受它们的影响。

　　作为一个诗人，注定一生都在逃亡，个人的逃亡和集体的逃亡。
　　走吧，逃离主流价值。
　　走吧，逃离大众文化。
　　走吧，逃离下山招安。
　　走吧，逃离所有的虚假和矫饰
　　走吧

　　在这篇文章中，按照我一贯的行文习惯，使用了大量强势语态，如：。。。就是。。。。必须。。。否则。。。这仅只是我表达方式上的习惯。并不代表我认为这是一定的、必须的。写出来的原因，也是想听听各位的意见，并无"奉天承运，皇帝诏曰"之意。期待着大家的反对意见。

和菜头
于中甸
