我眼中的大话西游

燕十八

　　前天，又看了一遍西游记。记不得是第几遍了，本以为不会再感动，不会再流泪了，结果当悟空的和紫霞的手最终分开的时候，忍不住低下了头，又一次流泪了……胸闷的难受。取什么西经！救什么世人！世人岂是一部经书能救得了的？如果那样，孔孟之说又何止流传了千年……佛祖如此，观音如此，唐僧如此。……世人也如此么？如果世人也如此，又何来西天之行。心中有经则处处有经何用去西天取，伸手便得之。

　　一部经书是死物，心有所悟才能万年不死，所有人都执着了。悟空呢？他悟道了吗？也未必。一个责任罢了，只有化身齐天大圣才能救紫霞。护三藏西天之行，也不过是和佛祖，玉帝，观音的一个约定吧。当至尊宝感叹为什么恨一个人可以恨五百年的时候，他悟了吗？他悟了！但那决不是一部经书能解决的问题。但他为责任，而不是为信念！最终把金钢圈套在了自己头上。去救唐僧等人，临行前那段经典的话大多数人都能背了，无奈与不甘。。。我实在不忍再回忆一遍结局了，我心中的故事从他们的两只手分开的时候就结束了。看故事的时候，我早已入戏了，好象我就是至尊宝，我为他不甘，也为我不甘。不知道其他的人有没有这种感觉。

　　故事的结局，悲剧。就算孙悟空是这个悲剧的始做蛹者，他的下场却是最惨的，从春三十娘和白晶晶说起，她们来世做了状元夫人也算一个不错的结局，按佛家来讲，她们最后保住了转世的唐僧和孙悟空，也算种下善因，最后有个善果，情理之中。青霞万念俱灰，最后回佛祖那里继续做灯芯，是她悟了？不！在最后关头，青霞终于终于知道自己其实是很疼自己那个唯一的妹妹的，尽管斗了那么多年，可最终上天也没有留下她这个妹妹。我想，她选择回那盏破油灯，决不是为了赎罪，而是因为那是她和妹妹的家，她们曾在那里呆了许久的岁月吧。。。紫霞的来世又是个悲剧，全因孙悟空的出现改变了一切，所以她不算最惨。孙悟空最惨！惨到要吻自己深爱的女孩，并且对她说我爱你的时候，都要借用别人的身体！还有比这更惨的事吗？一世轮回，所有人的都忘了前世，也许还有淡淡的一丝记忆，如紫霞，她也只是对孙悟空有那么一点点形象。。。"他好象一条狗啊"，紫霞对悟空的最后评价！悟空是象狗么？不象！但他连狗都不如！旺财都不用记起它曾咬开过菩提的乾坤袋，还被牛魔王的妹妹上过身，不管旺财的来生是做狗还是做人，它都会比悟空更开心。而悟空呢，他要背负上上下下五百年的记忆，痛也好，快乐也好，他都只有一件事做－－－－取西经。西经很快就能搞定，师徒四人也会有个封号什么的。然后呢，悟空要与他五百年的记忆一起痛
苦下去，永远没有结局。悟空一定会痛苦的，因为他是孙悟空，他又是至尊宝，其实他也未必知道他是谁……

　　好了，说到这里也算完了。我看过很多关于这个故事的文章。我不求和人比谁写得更好，谁更专业。我只是把我看到的故事写出来，也许就是我的故事。

　　每个人头上都有金刚圈。我头上也有，但我不松手！
