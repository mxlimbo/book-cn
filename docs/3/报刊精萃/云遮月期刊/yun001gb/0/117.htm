给好友乔燕

千子晨

乔燕姐姐：

　　近来好吗？还记得我这个朋友吗？！认识你不久，我就被派外出学习了，一直没有时间再来论坛。在这个论坛上认识你是我这段时间上网来最大的收获。虽然身在外地这段时间一直没有上网。但我仍然在想念这里。希望你一切都好。也祝朋友们都好。等过段有时间了，我们再见！！

千子晨
12-01-2000 18:20

千子晨妹妹：

　　我当然记得你，一个可爱而又善感的姑娘 ！

　　我怎么会不记得你？

　　你优雅地坐在弯弯的嫩黄色的月牙上，微微闭着眼睛任思绪在夜光里飞翔，你给我留下了很美的印象！

　　对，是"美"的印象！你的名字，你的语感，你的文字风格，和帖子里淡淡的絮语。

　　想念你，我相信这里的朋友们和我一样，惦念着你这个给我们带来清新的姑娘 ，我非常喜欢你坐在月牙上的模样，我觉得就像是梦中的另一个自己！

　　其实在我看了你这封信点了"回帖子"时，我本想叫你"子晨妹妹"，但由于不知"千子晨"所要涵盖的全部含义，固而未敢冒然。

　　你说认识我是一件快乐而又是相当于收获的事，我有同样的感受――和我在这里认识的每一位朋友一样，你们给了我浓浓的喜爱与祝福，我很快乐，我想我会因为这一切而加速我在曲折之路上的行进，当我有一天能够自由地出现在朋友们面前，我相信那时，我们脸上的笑是这个世界上最美丽，最动人的风景！

　　我们的牵挂会随你走到每一个地方，无论你去的地方是闹市还是临近乡镇，当你在黎明醒来，请你侧耳细听，那一声轻轻的啼鸣便是我们想念你时给你捎去的音讯！

　　我们会耐心地等你回来，回来坐在嫩黄色的弯月上给我们讲述你的收获！

你的燕子姐姐
12-01-2000 19:22
