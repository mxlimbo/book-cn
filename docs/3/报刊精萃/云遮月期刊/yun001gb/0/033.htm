　　1.不要挑战帮派.物以类聚,人以群分.这是很正常的事情.总是有无数的小集团生存在网上.很著名的就有拥金倒朔派,依王屠庸派等等.当年我有幸观看了两派血战的壮观景象.

　　从聊天室到BBS,从网易到新浪,无数高手纷纷或用绝妙文章,或用深情口号,或相互问候对方亲戚,各种武功不足一而论.反正是杀得日月无光,面红耳赤,网站老板笑逐颜开.我当时不慎说了一句:老金凭什么是20世纪中国十大文学大师.结果当时就被板砖和口水淹没,几乎薨了.这还算好的,如果你得罪的是江湖上的秘密团体,你的死状要惨十倍.他们会从各种角落里出来,乔装改扮各种角色对你进行全面攻击.或云你道德败坏,或云你文章狗屁不通,或云你精神变态,或云你哗众取宠.你往往误以为自己已经被人民战争的汪洋大海所包围,神经比较脆弱者一般都因崩溃而自杀.其实他们就一群高呼:神功护体,刀枪不入的义和团拳民.拳民虽愚,勇不可挡,不可不慎啊!

　　2.不要挑战35岁以上的老人和中产阶级.人过三十五,该立的早就已经立了,不该立的还躺着.人到了这年纪,人生经验丰富,思想早已定型,最是刀枪不入,水火不侵,油盐不进.你要是想改变哪怕一丁点儿他们的思想都是白费力气.而且别人的人格魅力比你强多少倍你知道吗?人家随便讲一点个人的感情故事就可以让MM们哭晕菜了,以为他就是杨过,自己就是小龙女,你行吗?你深沉吗?你有故事吗?你有宽厚的肩膀和臂弯吗?你当不了乔峰大哥的.如果你没有35岁,你的行为本身就是大不敬.在我们中国犯上做乱是不赦的十恶之一!说到中产阶级,人家住的是洋楼,养的是番狗,开的是Benz.人生完全达到了巅峰,你凭什么跟别人拼?论学问,别人随便就是美国的PH.D;论事业,人家没事已经开始赞助慈善基金了;论文学修养,别人卧室里那套<水浒&gt;还是全国唯一的老施亲笔呢.别人看世界的高度和你已经不在一个层次上了.即使你告诉他地球是圆的,他也会回答:"Really?这个project是哪个Company负责的呢?我们如果把它改回平的,应该有很大的benefit,right?"即使你真的象阿甘一样的执着,辩得人大败.人家依然可以对你说:"Noproblem,我来这里就是为了一点fun.现在没fun了,不好玩了,seeyou!"你就真的象个阿甘一样站在原地.活活气死你!

　　3.不要挑战权威,领导,变态者,本地人,网上阿飞,黑客,别人的网上男友,别人的网妹,聊天室主人,食素者,恐龙,神父,阿甘,球迷,写手,玻璃,和异型.和上述三种人保持了距离以后,你基本上就是安全的了.我很遗憾的说,这也就基本没有什么人可以交流一下了.当然,这并不是你的错.你环顾四周,这里曾经是夜郎的属地,也曾经成功的抵抗过中央政权的进攻,并先后以古茶马道和驼峰航线而闻名于世.如果你研究一下她的地图,你会惊奇的发现她的形状和中国大陆全境的类似.她曾是战争年代的大后方,北方语系在中国南部的一块飞地,以及法国的殖民地.所以你可以看见文化的化石的时候,必然为今天的闭塞所侵扰.你看到她不为时代的变化所动的时候,也应该明白这里顽固的守旧力量有多么强大?如果你曾经在更广阔的世界里呆过,如果你曾经和无数其他地域的人沟通过,你就会知道这里的荒漠程度,思想观念上的僵化落后.在历史上的云南,曾经因为战争和西南联合大学的缘故,空前的有一次与世界接触的机会.

　　在那一瞬间,历史上的一瞬间,云南发出了耀眼的光芒,我在其后近大半个世纪里不复见那种光彩夺目,仿佛是美人迟暮.历史说明了只有张开双臂,欢迎沟通和交流,文明之树才会在这里变成无数人可以得以栖息的思想森林.亲爱的朋友,这是不可以逆转的潮流.在网上的交流和交锋只是开始.

　　没有人能阻挡这一潮流的前进,不管是妄自尊大的学者,还是自号帝王的妄人.对于各国边疆无限扩大,世界缩小为村落的今天,我们再也不可能安于世界的一角.强大的文化的,经济的冲击将彻底改变几个世纪以来人们自以为不变的根深蒂固的东西.

　　每个人都将飞奔在各自的路上.你必须越过失败者和守旧老人羁绊,超越小我对脉脉温情的渴望,相信有更美好的生活,更自由的环境,更深厚博大情感的到来.

　　忘记这里无聊的小善和小恶吧!你得在它们到来之前在思想上先做好准备.在我们的时代里,我们将最大程度上尊重个人的意见,个人发展自身的要求,独立的个性和人格.你没有必要局限在一城一地,某时某刻,在你的沉默中勇敢的准备吧!成功将属于那些有独立思考能力,自立自强并在头脑上做好准备的人!

　　亲爱的朋友,到这里我不得不和你说再见了.我很困了,手也很酸了.希望这不是你心目中的假冒伪劣商品.1到3节,我想你应该知道,我所说的情况不止局限于一地一时.希望你能明白我的用意.而第4节比较而言,是为了让你开心一点.也希望你喜欢.PS.关于诗歌,你可以看看《读书》今年3,4期里黄灿然的《在两大传统的阴影下》.他对新诗有完全不同的,乐观的估计,找来看看.
