第一次和老婆吵架

网命徒/文

　　无论什么事情，大凡"第一次"的都有点纪念意义，比如第一次恋爱，人们美其名曰初恋。但我现在说的和初恋大相径庭，但和感情有关。

　　和老婆吵架自然不能提倡，但因为是第一次，就显得特殊。还有个原因就是今天是我们结婚五周年的日子，我想以这种特别的方式来纪念它。

　　第一次和老婆吵架，是在她刚刚成了我老婆不几天，旅游期间。

　　兜里揣着红红的结婚证，我们先到了上海，然后去了苏州。虎丘公园，就是我们第一次吵架的地点。至于因为什么，记不起来了，我那时的日记里肯定有记录，但再去找也费劲，只好略去。反正是吵了，而且很凶。最后老婆甩下一句话――那我们各人走各人的！我也在气头上，走就走，看她向北走去，我便朝南迈步。这样我们就算分道扬镳了。但走出去几步后，我就后悔了，这算怎么回事啊，我们是来旅游结婚的呀。我马上返身，老婆已经看不到了。

　　可把我急坏了，我此时才意识到问题的严重性。老婆说各人走各人的，她去哪里了呢，是回上海了还是回山东了呢？我们是打算再去上海的。她要是回了山东，那可麻烦了，两人一块出来的，却一个一个回去，那象什么话啊？！我疯了似的满公园里找，但非常的遗憾，没人。

　　她能上哪里去呢？我直奔火车站。

　　在候车大厅，我顺着一溜一溜的椅子找过去。此时的我眼睛已经绿了，看见女的就觉得象我老婆，更要命的是我记不得老婆是什么发型，穿什么颜色样式的衣服了――我急糊涂了。我必须挨个端详每一个女人，不然我无法确认她不是我老婆。当时，她们还有他们肯定把我当成疯子了，而我也确实和疯子差不多了。现在如果问我苏州火车站的门朝哪，我可能不知道，但如果你问我候车大厅里有几排椅子，每排有几个，我使劲想想可能还想的起来。

　　就这样我仔仔细细找了一遍，还是没有！

　　我彻底失望了。

　　我不再考虑我老婆在哪了，我开始考虑回去后怎样向亲朋好友解释，怎样向丈母娘请罪了……唉，我还有颜面回去吗？说出来不让人小掉大牙，千古奇闻啊！……

　　我垂头丧气的走出候车大厅。

　　突然，我眼睛一亮，哇！！！！！！我老婆！！！！我老婆在广场前面的马路边上站着！！！！！夕阳从她的右边照过来，在左边拖出长长的影子。我象见了亲人一样，踉踉跄跄地飞奔过去，从后面一把抱住她，生怕她再跑了。老婆知道是我，没动。我把头埋进她的头发里，泪水止不住的往下流。

　　周围车水马龙熙熙攘攘，我们却象是在一个孤岛上。在这异地他乡，我真真切切地感到还是和老婆是一家人。

　　半天，我把她转过身来，，老婆也泪流满面。

　　我说对不起，我来晚了。

　　我不好，我检讨，我不对，我有罪。

　　我领你出来，却让你受了这么大的委屈……

　　………………………………….

　　就让我们永远生活在一起吧，好让你有足够的时间来惩罚我！

　　老婆终于破涕为笑，我们相拥着走向候车大厅。

　　这时，周围已是华灯齐放……
