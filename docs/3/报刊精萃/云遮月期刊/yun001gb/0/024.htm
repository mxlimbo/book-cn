永远的照片

凤木/文

　　中原农村的清晨，特别是春天的早晨，似乎亘古以来就如此清爽﹑新鲜。静静晴空，柔柔红日，偶有晨起的农人，白云般洁净的炊烟，还有那清凉大气中幸福歌唱的鸟儿；滋润露珠才几日的新嫩枝叶，让人心里只有一个劲的舒服，特别是我这日日呼吸繁华浊气的城里人。

　　这实在是我久违了的晨光。

　　可我却不是为这久违的清晨而来，我来只是为了赚钱，是的，就为了赚钱。我是开照相馆的，在县城一条不算繁华的街上有一间小店，生意时好时坏。昨天中午，店里来了一位奇怪的顾客。他一进门就迫不及待地说要给他的学生——一群农村的孩子们拍些照片，拍一整卷，还让我一定随他到30多公里外的学校去拍，并且除了必要的合影外，其余要用新闻照片的形式。听到最后一句话，我不禁抬头打量了一番这位特殊的顾客。首先冲入我眼帘的就是他那一脸的沧桑﹑困苦和几十年营养不良给他的清瘦身躯，我猜他年龄最多40多岁，但看起来却象近60岁的人，不说他满面的皱纹是多么沟壑纵横，就连黑发也成了他满头白发的点缀。他穿的那件洗得发白的皱巴巴的蓝色中山装，在花花绿绿的顾客中间，实在显出扎眼的寒酸。右边手里还拿着一只书包大小却空瘪无物的破提包，门口他骑来的自行车更是惨不忍睹，这辆破车拉到旧车市场能卖20块钱就算不错了。我脑中打了个大大的问号，怎么？这样的人也要上报纸吹吹自己﹑为自己脸上贴金吗！这种想法实在让我无法与眼前的他联系起来。但疑惑归疑惑，生意还是要做的。于是，我报了一个比市价略高的数字，报价的瞬间，我发现他象是被人掐了似的一怔，紧接着还起价来。下乡是苦差，我再次强调这价钱已够合理了，不必再讨价还价。像是经过了激烈思想斗争，过了分把钟，他才同意，交来20元钱定金后，千叮咛万嘱咐让我明天早早去他们学校。说毕，蹬上破自行车匆匆离去。

　　第二天大早，我骑上摩托车启程。在感受一番农村美好清晨后，顺利到达目的地。这所小学校很破，没有门，只有两根用砖垒成的门柱，柱子的顶端有几根钢筋成弧状连在一起，几块斑驳的圆铁皮焊在钢筋上，用红漆写上字就是校名，铁皮虽破，但那字却苍遒有力，让人不敢小瞧了这所简陋的小学堂。似乎是早有准备，摩托车刚进校园，就有三位教师模样的人迎了出来，其中就有昨天联系业务的老者。后来我才知道，他们就是这所小学的全部师资力量，老者是校长，姓陈。老师少，教室也少得可怜，只几间破瓦房老态龙钟。那几十个充满活力的孩子才让这所学校显出生机和希望。

　　到得老师们那间只有几副桌椅的简陋办公室，几乎是被他们按在了椅子上。老师们似乎早已心照不宣，寒喧几句便忙碌起来。陈校长去了外面，象是在组织学生。屋子里这两位则为我倒茶递烟，万分客气，似乎我真是从哪家报社来的记者一样，热情得反倒让我不好意思起来。不大会儿，陈校长快步走进屋子，说一切准备就绪，开始上课。同时示意我随他去拍照。还未走进教室，机灵的小学生们就已经发现了我们，喧闹的教室立刻安静下来。走进那间破旧但整洁的教室，迎接我们的是一双双充满兴奋和期待的黑亮眼睛。孩子们无一不规规矩矩地坐着，目光几乎都聚焦在我脖子上那架带变焦镜头的长嘴照相机。带着好奇，带着渴盼，似是向往已久。面对孩子们，我简单交待了几句，大致意思就是让他们只管听课，就当我别在教室里就行了。孩子们一个劲重重地点头。

　　陈校长缓步走上讲台，显然他"打扮"了一番，先是花白头发梳得整整齐齐，昨天那身破中山装也已被一套档次虽低但却干净的西服所代替，人顿时年轻了许多。他翻开课本，先讲了本节学习的内容，这时我发现，他的目光不再是昨天那种有点阴郁，疲惫，而是炯炯有神，整个人也变得神采奕奕。果然，没多长时间，陈校长的抑扬顿挫就把学生们的目光从我身上收了过去，一个个都专注起来。乡里孩子那种与生俱来的田野味的纯真和对学习求知真实的渴望溢满了简陋的教室。一幅幅绝佳的画面汹涌而至，我的手指不由自主地捺动着快门，这会儿，我只恨闪光灯充电太慢。等知道自己已沉迷其中时，才发现相机计片窗的数字已跳到了20。于是我默默收拾起照相机，退到教室后边，没有打断正讲得意气风发的陈校长。又过了几分钟，陈校长才发现我停了下来。他停顿下来，走到我身边，客气的对我说："请你先去办公室歇着吧，喝点水，我给孩子们上完这节课咱再说话。"

　　到办公室没有十分钟，几声"铛、铛"的铃声打破了有秩序的读书声和办公室里显得有些沉闷的静寂，校园里顿时热闹起来。随着陈校长和另两位老师的陆续回来，办公室的门边和窗下也聚满了好奇的孩子们。陈校长不驱赶他们，学生们也约法三章似的不踏进办公室一步。在这热闹的气氛中，陈校长和另两位老师洗了洗沾满粉笔灰的手，依旧极客气地请我随他们去照相，仿佛我真是他们请来的极尊贵的客人，而不是来赚他们的钱的商人。此刻陈校长已来到院外，对那帮似有无穷无尽精力的孩子们号令起来，看得出孩子们与陈校长之间的融洽和亲热，他们对陈校长，就象孙子面对一位既严厉又亲切的爷爷，每个孩子都听话且不拘束。在他们默契的配合下，没多长时间，所有的合影照片都宣告完工，我又被他们请回办公室。陈校长亲自给我倒了杯热气腾腾的茶水。这种水，是直接从地下抽出的，那种甘甜清纯要比各种品牌的矿泉水和纯净水要好的多，反正今天有得钱赚，索性就在这里品会儿甘泉吧。

　　陈校长先是说了照片的冲洗范围和时间，接着又闲聊似的说起他的学校和孩子们来。"这些孩子们，家里大都不富裕，长到7、8、10来岁也没照过几张相片。孩子们苦啊，学校的条件不好，到冬天教室四面透风，咋修也修不住，有的孩子们手都冻烂啦，可孩子们还那么下劲学，让我这当老师的心里热乎乎的啊。"他喝了口水，像在控制自己的情绪。"我18岁就来这儿教学啦，开始连个民办的也不算，后来也想过干其它的，可咱没本事，啥都干不成，干脆就安于穷困吧，后来我渐渐无法离开这一切啦。特别是我的学生们……"他脸上带种空灵的表情。接着他又用一种奇怪的神情问我："照片不会洗坏吧，多少钱不要紧，我是想跟孩子们留个影，我太喜欢孩子们啦……"。这多少让我这个小县内有点名气的摄影师有些不舒服，我正想反驳和解释几句，忽然看见旁边一位小女老师的眼圈有些异样，一种欲泪的表现。我话到嘴边又强咽下去，只轻轻说出"绝对不会"四个字。

　　走的时候，陈校长和他们的学生们送了我好远，陈校长那种期待和渴盼的目光竟令我的心莫名压抑起来，迎着庄稼制造的甜美的风，我长舒了一口气……

　　陈校长是在五天后才来拿照片的，脸色又分明憔悴了许多，本来就瘦的身体更显干枯，似一阵风就能刮倒。见他来了，我一边热情寒喧，一边把早就精心扩好的照片如数拿了出来，并给他找了张桌子，从饮水机里倒了一杯水，让他慢慢欣赏。他看的极慢，且极认真，象是把玩一件价值连城的宝物。他脸上的表情也极丰富，一会儿沉思，一会儿浅笑，一会儿竟似要落泪。我不敢惊扰他，只是一面接待顾客一面留意着他。一卷照片他看了40多分钟，他起身走到我面前，我惊奇地发现他脸上的憔悴已被兴奋欣慰的光芒代替。陈校长神情庄重地把照片认真收好装进破皮包，紧接着突然一把抓住我的手，用力握了握。接着便如数将钱付给了我，我则又退给他10元钱，我说陈校长你拿这钱吃点饭，坐城乡公交回去吧，这么远的路。陈校长走时对我说，你的人和你的作品一样，谢谢你。我目送着他骑着破自行车有些飘摇的身影，心里说不清是什么滋味……

　　时光匆匆，转眼离陈校长拿照片的日子已一月有余了。忙乱的生意让我学会了淡忘一切，我又回复到那个没有感动、只一门心思赚钱的小老板的角色了。恰巧这两日女友非要我陪她出去闲玩，闹的我心烦，老想着女友如今就这等娇气，结了婚岂不更了不得，于是也没了多少好脸色。这天，我正在无聊地捧着本无聊地杂志瞎看，听到耳畔一声轻问：请问杨师傅在吗？我听的一愣，我姓杨，可哪儿有"师傅"这个象出土文物似的大号啊，我抬头看了看眼前的人，似曾相识，可就是想不起他是哪里的。出于礼貌，我回了一句："你好，你找哪位？""唉呀，杨师傅，可找到你啦！"面前的这位男子夸张地向我伸出手来，我伸出手象征性地与他握了握又坐下，然后说："你是……"。他见我忘记了他是谁，脸色暗淡下来，说："你忘啦，一个多月前，我们陈校长把你请到我们学校照相的……"我的心猛地一动，是小男老师，我带着愧疚从椅子上快速起身，笑容满面，倒茶递烟，他却没有喝茶也没接烟，只是从包里掏出一张照片，说："今天来，是想放大一张相片，是陈校长的……"他说着，竟有些哽咽，我从他的表情中猜到了什么，但我还是问了一句："陈校长他怎么啦？""他、他去世啦，昨天，癌症，他早就知道…"这近似有些语无伦次的话语告诉了我一切。"这么好的一个人，一位老师，怎么……"我说。"他说相片好的很，他说他死后用就用讲课的相片，他死前不让学生见，他说那会吓着孩子。他穷的什么都没有，他把心都掏给了学生们，他……"。小男老师自顾自的言语着，泪水甚至都淌到了嘴里。

　　他拿出的是陈校长在讲台上抑扬顿挫时的半身照片，照片上的他神采飞扬，丝毫没有死神阴影笼罩下的痛苦。用这样的照片做遗像可见他对生活的挚爱和淡泊生死的勇气，本就尊敬他的我此刻更为佩服受小男老师的感染，为一个好人的离去伤心起来。告别了小男老师，我没按他的要求扩16寸的，而是扩了张24寸的，我又专门去进了一趟照片框，钱都是我出的，我不能让自己穷得只剩下钱。同时，我对那次收他们的钱于心难安，一直想着该怎么补回来……

　　陈校长办完后事时已是初冬，我专门找到小学校所在乡政府上班的一位朋友，委托他把我买的铅笔、本子还有100多只暖袖送到那所小学，我告诉他，一定要把东西发到每个学生手中，一定别让他们冻着，你要还想要我这个哥们你就得给我办到，朋友被我说的一怔一怔的，他说我今天有点奇怪，怎么这么唠叨，还说我这是发的哪门子善心，他还说他还是头一回听说有捐暖袖的呢。我没理他，我又仍给他一包好烟，让他明白这事儿的重要。

　　我死活没有吃朋友的酒宴，骑上摩托空着肚子往回赶，我骑的很慢，于是30多里的路便给了我许多或远或近的田野，我望着这些奉献了一季膏血的黄土地，竟泪流成行……
