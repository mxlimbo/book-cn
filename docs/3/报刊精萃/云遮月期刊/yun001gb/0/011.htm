惊"燕"——忆小乔初会

燕十八

　　第一次听"乔燕"这两个字，是从小妹的嘴里。小妹说，"她是个透明的人，象一滴水一样纯洁"(我想，应该还是纯净水的那种纯洁~ 当时小妹的表情让我感到有些虔诚。我的心随之一动，是个怎么样的人，居然也在洛阳，我一定要见到她。甚至连为什么要见她都不是特别清楚。(也许和我嘴角那滴差点流出来的口水有那么一点点间接的关系吧。。。)

　　第二天中午，和小妹还有兔子一起去上网，下午送小妹离洛。离开车站，我就又一头扎进网吧的怀抱，几乎看遍了乔燕的贴子，总算找到了她的QQ号。(当时居然有一种刚上网时偷偷找MM的感觉，天。)在Q上找到她的时候，她不在线，发个消息过去，静静等。她上了。刚开始也许是QQ有点问题，老是联系不到她，二十分钟后才正常。不记得和她聊什么了，反正主题是要她的电话，在那时，我坏坏的。晚上七点多钟，以一种胜利者的姿态飘出网吧，因为那时候，我有她电话了。（有点象时下网吧里小弟弟的样子。）路上，跟她通了第一个电话，（在网吧里曾打过两个，没打通。）约好八点左右通电话。

　　到家了。三口两口吃完饭，茶杯香烟摆上桌。一副严阵以待的样子。要开始了，*******。两声铃声后，她出现在我电话线的那头。电话从八点多一直聊到十二点半，聊了很多，谁也不想挂。但实在太晚了，最后还是要挂掉的。我对自己说：不行！我要去她家。

　　一觉睡到十点半，起来补上昨夜没看的电视。给她去了电话，问清地址，就出门直扑七里河联通。（因为顺便办点小事。）路上无话，从她的卧室门离我有1.5米时说起。

　　当时她的卧室门离我还有1.5米。我的心，扑通，扑通。。。象往常一样，照例没有停止跳动。正在寻思自己是不是有点紧张的时候，她看见了我，但我没有看见她，只看见她的笑。我差点脱口而出：我又来了。（为什么要说"又"？）让我感到那么熟悉亲切的一个人，以至于以后的时间每隔一会儿我都仔细观察她的脸，试图记起在什么地方见过，最终还是失败了。我们是第一次见面，但又象见过很多次，没有陌生，只有亲切。她的小屋只有十平左右，对我而言，那天的世界也只有十平。整个下午，我被包围了，被她的笑，被她。甚至我的思绪都没有跑出窗外，尽管窗口离我很近。我们象多年老友一样，小屋里充满了那种叫快乐的东西，世界也充满了那种东西。我和快乐，在她的小屋里，又见面了。乐不思归。。。

　　不写什么时候离开了，有点伤感。如果不是离得太远，这篇文章我也不是在自己家里写了。当然，我会经常去的。比如明天，我就又在她家了。

　　惊"燕"!　

　　后记：写到一半的时候，死机一次。唉，回头再写，有些东西就写不出了。还好写一半的时候存过一次，要不就死翘翘啦。曲折，象她，也象我。我们的生活道路都不是太顺，也算一个共同点吧。
