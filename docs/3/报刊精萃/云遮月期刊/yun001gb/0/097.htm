投河的妇人

地点：云遮月客厅
主持：蚊子

蚊子： 　　一个老渔翁看到一个妇女投河，就把她救了上来并问她为什么要死。

　　这个妇女就告诉他说："两年前我嫁给了一个人并生了一个孩子，可是前几天男人死了，前天孩子也死了，我伤透了心，我不知道活着还有什么意思，不如死了算了。"

　　老翁问她道："你两年前是怎么过的呢？"

　　她说两："两年前，我没有遇到他的时候一直都是快乐的。"

　　老翁说："那你现在也不过是恢复到了两年前的状态，又有什么好伤心的呢？"

　　妇女想了想，说："也是。"

　　当她又想了想，重新又高兴起来的时候一抬头，那老翁已经不见了。她也没有再看，就走了。

兔巴哥：

　　但是如果一直没有也就不会失落，人在地上的时候怎么走都可以，但是人爬上了山再落下来，会摔四的！

小风铃：

　　有道是：

　　曾经沧海难为水！


乔燕：

　　我觉得蚊子讲这个故事真是很深刻！生活中有太多变故，谁又能保证：人这一生一定不会遇到些灾难呢？既然我们无法左右生活，既然我们不得不在失意中继续生活，那么如何接受灾祸就成为一个问题了。

　　这位妇女觉得活不下去了，因为她这时考虑问题是站在"我已经有过丈夫和孩子"的心理背景下，因而她感到自已失去的太多；而老翁是在引导她改变思考问题的方式并调整心理背景，使她能这样想："是呀，我原本就是这样，丈夫和孩子是人生中的获得，现在失去了，只不过又回到了原来的地方，我并没有到了不得不死的地步呀！"这时，她已经是回到以前的心理定位上了，也就是降低了期望值，所以尽管现实状况没变，只因为期望值与实现值之间的落差缩小了，她的失落与痛苦也随着变小了。

　　这个故事意义在于给我们一个重新考虑问题、衡量得失的角度，对我们这些活着的现代人同样有启示意义。
