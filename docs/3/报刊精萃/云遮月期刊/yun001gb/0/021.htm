印　象

知逢/文章

　　凭心而论，乍一见她的感觉与其说是"惊艳"，倒不如说是"忘情"。就如天涯过客踏着漫天的风雪，正叹息天地之茫茫的一刻，猛然发现眼前一株傲雪的寒梅，他心里的第一印象，也许并不是它的颜色、它的美丽，而是那一种跃跃的生机、勃勃的朝气……

　　在印象中，她给人的感受，并非绝美的那一种女孩。也许是因为嘴太秀气，（常言道："锦心秀口"。）太聪明的女孩容易使人敬而畏之；或许是由于眼睛太亮，虽也不失妩媚，可却总有一种让人无处而遁的感觉。所以，她笑的时候特别可爱，嘴儿一弯、眼儿一线，可谓："大笑足以赏心，小笑也可怡情"……

　　如果说，最可爱的是她的笑容，那最迷人的就一定是她的声音！大而不失娇媚；小而不减清脆。远听近闻，叮叮咚咚，像秋日晨曦中的风铃；似春夜小雨后的清泉。只是那一股甘爽、清脆，就足以溢满你的心田……

　　记得毕加索曾经说过，描绘一个人的外貌只是暂时的，而把握一个人的神韵才是永恒的。如果说她的声音和笑容我还可以用言词去表达，那她的神韵我也不知用什么去赞美了，只觉空谷幽兰，逊其娇艳三分；雨后长虹，输她馨香一缕……如果硬要用一个词来表达她的神韵，那一定是"清新自然"！所谓"清水出芙蓉，天然去雕饰。"

　　印象中，她就是天边那一缕闲淡的白云，悠悠远远，若有若无；她就是雨后那一朵傲放的青莲，恬恬静静，似真似幻；她就是你心底一个最美的梦境，每次想起，都会有一丝甜意，铺满心头，那种细细密密的美好，如丝如网，如砂如粉……
