最神圣的仪式

海边小屋

　　门突然格格地响起来，如同一百只老鼠的狂欢。我从蝴蝶梦惊醒。开门，只见两个黑衣人出现，面目不清，目光冷酷。他们向我打了个手势，就转身向楼下走去。我感到一股磁性的力量，不由自主地跟随而去。

　　黑夜沉沉，一股腥凉的气体直沁心脾。

　　一直向前走，仿佛走进了时间深处。

　　渐渐听见松涛的呼声，脚下软绵绵香喷喷的不是芳草就是野花。

　　有人像一棵槐树竖在前面，蒙面，看不清楚。

　　"来了？"

　　"来了。"

　　"可以开始了？"

　　"可以开始。"

　　两个黑衣人远远地站开，这片荒野一时间无比广阔。星星半眯着眼，无声的风大块大块地跌落。他的双眸闪闪发光。

　　我摸了摸怀里的飞刀，好像听见了某种远古力量的召唤。

　　突然，一束凌厉的剑气向我逼近，压得我的胸部奇痛。他的眼珠渐渐收缩，在光与影中凝成一颗方方正正的黑点。

　　这时，不死鸟的灵魂复活了，它背负青天从大洋升起。

　　我的翅膀刷刷飞举，厚实的大地陡然下沉，越来越远……这时——我的飞刀出手了。一道暗红的光芒含着最愤怒的力量向他激射而去。

　　剑气对刀芒，无敌对至尊。

　　剑花，刀雨，岭南。

　　夜。深沉。

　　"扑咚"——两具高大的躯体重重地同时摔倒在地。喘气声，越来越弱……

　　"沙沙沙沙"——两个黑衣人踏着落叶归来。

　　"他还没死。"

　　"还没死，不过伤得不轻。"

　　"虽然伤得不轻，但他赢了。"

　　是呀！我吐出一缕灿烂的血花，像新出蛹的飞蛾，颤颤栗栗，怯怯生生，还是站起来了。

　　一个黑衣人要伸手扶我，我冷冷地拒绝了。他讪讪地停止动作，幽幽地发出一声叹息。我略略抬起高傲的头颅，只见他目光里面充满温柔、平和、关切。我疑心他是我的一名故人。但这又如何呢？

　　对手已经不再是对手，对手归于万物寂然无声。

　　往事不再是往事，再也无法挽住那熟悉的云彩。

　　从对手身上拔出炙热的飞刀，迎风长吁，不禁涌起一点点落寞的伤感。

　　"我可以走了？"

　　"可以。"

　　我不再说话，迈着庄严的步子，如同进行完一次最神圣的阅兵式，回到我应该回到的地方，继续我未继续完的梦想。
