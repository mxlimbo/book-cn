小镇笔记

富顺才子/文

（一）

　　小镇是我的故乡，它和川南腹地的许多小镇一样，毫不起眼，似乎一直没有什么改变，历经岁月变迁依然宁静的卧于山水之中。

　　在我的记忆之中，小镇有一个非常热闹的水码头。旧时主要靠水运，自贡的盐、当地的家产品均要在这里转运，从沿着河滩而上的那些石梯几乎磨光的层面上，以及官厕风蚀得很深的墙面，可以想象当年的繁华景象。在那些很窄很窄的街面，铺一些磨得发光的青石板，街两的屋檐几乎要相互接拢，只有一线光可以进入，一个人独自走在街上，抬头望一望窄窄的瓦蓝瓦蓝的天空，街两旁的屋门口，坐一些闲聊的老人，他们大多头发斑白，或抽着叶子烟，或喝着粗茶，总上重复着那些前三皇后五帝的话题。见到这种情形，那种甜恬与淳厚的气氛自然会带你进入一种古朴的意境，产生与世隔绝的感觉。

　　我的童年便是在这样一座小镇中渡过的，在那些如水般空灵的日子里，小镇每一个地方都留下了我的足迹，和儿时的朋友们玩耍那自然是高兴的事，但最让我兴奋的莫过于小镇来了戏班子，在戏院拥挤的小坝里，我经常躲在大人们的后面混进去，在锣鼓喧天中望着戏台出神，我印象最深刻的莫过于看川剧《十五贯》，那个小偷尤阿鼠（或许叫娄阿鼠吧！）脸上意是沿着鼻子周围涂一块奇怪的白斑，头上的帽子边缘向前歪歪的倒着，看人总是鼠眉鼠眼的，走起路来东瞧瞧西看看，一幅偷偷摸摸的样子，锣鼓一响脖子忽的一转，眼睛滴溜溜乱转。

　　据说，现在我那故乡小镇再也不演什么"尤阿鼠"了，偶尔有现代歌舞团来这里演演，那是怎样的一咱场面就不得而知了。

（二）

　　别小看了故乡这个小镇，在历史上还出了一个响当当的人物，那就是刘光弟。小时候，对刘光弟先生还不大明白，我的老祖母曾经告诉过我，说刘光弟是我们小镇的人，他在京城做大官，由于见了皇帝娘娘偷汉子，被杀了头，尸首运因来的时候好气派哟，河两人山人海。当时我就想，一个大官也死得太不值得了，怎么看了别人做什么事就会死呢？后来，我才知道，那个皇帝娘娘大约就是慈禧吧！于刘光弟之死已弄明白，他是为变法而死，死得有价值，不是见了什么见不得人的事而死的，虽然现在不乏这种死法。

　　刘光弟先生出生于小镇正街的一间宽宽的屋里，那时叫什么"福建会馆"的，现在那间铺面已经做了间杂货铺，外间堆满了碗碗盆盆的，里间是店主人住的地方，我曾经走进去过，想从那里找一点我先辈乡贤的足迹，但什么也没有找到，两间破破烂烂的屋子，然而却在此诞生一位了不起的的人物！

　　最近，在这间铺面的门上方悬挂的一块匾额，上书"刘光弟字裴村，生于1895年"，我想刘光弟先生泉下有知，也应更添一分欣慰了！

（三）

　　小镇没有什么特殊产业，在它的西街有一个很大很大的酿酒作坊，那个作坊的屋子很宽，里面打了很多很多的大地灶，不管哪个时候，里面从来都是水雾弥漫，负责酿酒的大汉也都是赤裸上身，忙忙碌碌的跑来跑去，或者上火，或者出糟子（一种高粮酿酒后剩下的残渣，于乡下却是喂猪的好东西，乡下人还要到这里来出少许钱购买），这里出产的酒，除了行销本地外，还远销泸州以及其它一些地方。

　　而今，那个酿酒的房子四壁早已倒塌，一个个空空的大地灶躺在那里，成了老鼠们嬉戏的乐园，听说一个城里来的开发商要把那地方买了去，修什么商住楼。

　　据我所知，小镇周围的乡村没有什么工业，年年种水稻玉米高粮。而现在小镇的人们却喝着其它地方运来的酒，据老人们说那些酒大多是用酒精勾兑的，不好喝。

（四）

在小镇生活最美的莫过于在临近傍晚的时候到江边散步了，一个人沿着松软的河滩，踏着散乱的芦苇随心所欲的走，河中晚归的渔船很悠闲的划行，有些渔翁在船尾生起了火，炊烟袅袅升上天空，落日的余辉给这一切涂上了一屋橙黄色。

　　邻近的乡村不知谁家的女人在呼唤未归的鸭群："鸭儿，嗲嗲嗲"那些外出觅食的小家伙们知道该回家了。

　　而我！却没人唤我！
