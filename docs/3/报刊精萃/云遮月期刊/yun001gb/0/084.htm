加菲的故事

加菲 等

　　加菲掀开"云遮月"的帘子进场：

　　"太阳悄悄从地平线爬出来，天亮了。慢慢从网下爬上来，似乎有些不对劲儿，哎呦呦，盐不见了，（难怪没了一番滋味），你在哪儿？？？？

　　哦！原来变成一只小燕子飞跑了（无可奈何花落去，似曾相识燕归"去"），呜呜呜，说好102年以后就带着面包和盐去天堂去看上帝哥哥的（上帝也口重？），如之奈何？

　　慢慢咬咬牙，狠心吞他几个小丸子（嘻嘻），对着月亮一阵怪叫"嗷...嗷...嗷..."，哈哈，竟然变成了一只大灰猫，（大灰猫？等等，大哥，有没有搞错，我要变狼的，不行，猫没品位的，什么？现在就流行变猫？变兔子？不不，那儿已经有兔子了），于是就这样，慢慢变成了喜欢小丸子的加菲猫。一边哼着"only you..."一边驾着几只老鼠拉的南瓜车，一路绝尘，直奔洛阳而来。

　　悄悄绕过老马的铁蹄，听了会儿南腔北调，拨开遮挡的云层，终于看到皎洁月光下的燕子，她还蛮幸福的。

　　想要呼唤，可又想到燕子一向是不太喜欢猫的（本地猫除外），只好用绝招了："小姐，我们在哪儿见过"？（好象有点儿老土）。万幸，或许看在小丸子的份儿上，我终于成了普通会员。还有一只兔子来热情的指点我（什么？那只兔子是版主？哎呀！当初要是也变兔子多好。）

　　先四处走走，噢，好多版面，好多版主，好多帖子，还分楼上楼下？冰比水还冰，石头也很硬。不错，难怪乔燕流连忘返。连我也有些此间乐，不思东北矣。

　　乔燕：能原谅我的欺骗吗？（早有图谋，不过是善意的），只想知道你在这里开心吗？于是耍了一个小小的阴谋，（我对你说过我喜欢加菲猫的，我说过我会来的，我说过我认识你的，呜呜呜，我是无辜的）。

　　看到你这里有很多朋友我很高兴，别忘了那边的朋友啊，（还有那个叫字字慢的坏家伙），哦，如果有时间的话，（象今天）加菲会常来的。

　　有欢乐，就有忧伤，有幸福，就有痛苦，就象月亮也有圆缺，生活或许有些不如意，但别忘了朋友牵挂的目光永远在你身边。

　　就这样，加菲慵懒的躺在壁炉边，听着小丸子那美妙的歌声，一边看着云朵后面那时隐时现的月亮，呼呼的睡着了。

　　乔燕端坐于"云遮月"正厅：

　　"哎，还以为本小姐不知道？怎么会忘得了？"乔燕看着桌上那盘盛开的米兰，心中暗想："怎么会忘得了那篇《我说我怎么渴得象条鱼，却又很快乐！》"

　　"这不就是那个总把我这'阿妍'写作阿盐的那个人嘛！我都快咸死了！给我倒水来！"往舒适的老板椅上优雅地一坐，乔燕清脆的嗓音落地开花。

　　加菲连声称是，把一杯热气腾腾的水轻轻放在乔燕面前，自言自语道："哎，我这是何苦来着，给自已找点事！不过，我好象还很乐意啊，真是奇怪了！"

　　说着，加菲转身走出去，只留给乔燕一个俊朗的背影。

　　"他是去找随便乱走了吧"乔燕心中暗想："他们怎么能分离得开哟，一个要扒墙，一个就搭梯子；一个说扫地，一个就赶快把扫帚藏起来的……。也不知那个随便乱走又随便乱走到哪了！又去雪山采雪莲了吗？"
