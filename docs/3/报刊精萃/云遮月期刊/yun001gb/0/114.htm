小人云

敏感之花＊阿明/文

　　如果几个人在沙漠中陷入绝境的时候发现了一杯水，想自己喝的是小人、想在经过推让最后别人能让给自己的也是小人、想趁人不注意的时候独吞的是大小人、为了自己能喝到水而不惜伤害别人的是小人中的小人，总之都是小人！发现自己是小人的时候是痛苦的事情，发现自己是小人还想继续做小人的称之为可悲。人人想做君子，但是谈何容易！做小人虽然比做君子更容易，不过做小人就容易吗？小人会有巨大的心理负担，因为小人自己并不真的想做小人，而是没有勇气做君子，做小人比做君子更加痛苦，需要承受的心理压力大过做君子！为什么能得出这样的结论呢？因为最后得出的结论就是——自己是一个十足的小人！
