北海故事——狼桥遗梦（四）

萧十一狼/文

四

　　时至今日我都不明白为什么当时我会如此的忧伤，我也一直没跟她说起过我为什么流泪的真正原因，她也从来没有问起过。

　　在我和桥交往的这一段时间里无论是在网上还是在现实之中，我们彼此之间一直没有主动问过关于对方的任何事，这似乎已成了我们之间一种不成文的默契。而后来发生的所有的事表明正是这种默契使我们能够走到一起的。

　　应该说我是一个很极端的人，爱情在我的心中是完美无暇极其神圣的。我不喜欢那些轻易言爱谈情的人，尤其在网上我更不喜欢轻易说爱情两字。在我的眼里那些在网上谈爱情的人都是虚伪不可相信的。至今我还记得有一次在星缘聊天室里一个冲过来就对我说了一句'我爱你，真的'，你们知道我当时的反应是什么吗？我起了一身的鸡皮疙瘩。

　　从那以后，在聊天室里只要看到'爱情'两字我扭头就跑。

　　如果桥一开始就跟我说她相信网上爱情的话也许我会从此不再跟她说话了，可后来跟她聊天时我发现了她居然和我一样都不相信网恋这种虚幻的东西。我们能一直在一起聊天也许正是因为我们都不相信网恋吧。

　　在那个让我悲伤的夜晚之后，我有两天没有上网。第三天我上线了，让我惊讶的是桥居然在线上。

　　桥：你那晚是怎么了？突然就跑了，害得我一直担心呢。

　　狼：没什么的，心情突然变的很糟糕，再呆下去也说不出什么了，就下了。

　　桥：能再看到你真好，这下我放心了。

　　狼：谢谢你对我的关心。

　　然后我们依旧唱起了我们熟悉的歌，唱了一会之后，桥开口了。

　　桥：狼，你相信网恋吗？

　　狼：我不相信，你呢？

　　桥：我也不相信，你为什么不相信？

　　狼：我也不相信一见钟情的，

　　狼：网恋和一见钟情一样都很不真实，都是虚幻的东西。

　　桥：为什么这么说？

　　狼：我不相信我能在没见过一个人的情况下而喜欢上她，我喜欢日久生情的那种，

　　狼：是朝夕相处而产生的日久生情的那种。

　　桥：我也一直都觉得网恋很虚幻很不实在，但看着那些小说写得那么好还又那么真实，又不由得自己不相信了。

　　狼：别相信那些小说的，小说都是瞎编的。你没听过一句话吗？

　　桥：什么话？

　　狼：太过于真实的东西反而变得不真实了，网络小说就是这样的。

　　桥：我明白你的意思，看来我们是志同道合的了，握手！

　　狼：握手！

　　就这样，我和她聊了起来，渐渐的熟悉了起来。

　　我和桥都属于很能瞎侃的人，用时下流行的一句话来说就是我们都属于现代的活愚公——用嘴侃大山，每天不止。我们还都是那种很喜欢开玩笑的人，和她在一起聊天的时候总能很放松，可以无忧无虑的说话开玩笑以及肆无忌惮的放声大笑。

　　通过长时间的聊天，我们都发现了我们之间的共同语言越来越多，我们的相同点也越来越多。这些共同语言和相同点也使我们逐渐走得很近了。
