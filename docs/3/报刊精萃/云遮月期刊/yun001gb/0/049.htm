寻找劳拉

和菜头/文

　　一切都是从我的不确定性开始的。有人说，一天24小时，8小时拿来工作，8小时拿来睡觉，剩下的8小时根本不知道去干什么，所以只好找个人来谈恋爱，在相互的欺骗吹捧和拥抱亲吻中熬过这艰难8个小时。而我不同，我不清楚我具体将在什么时候上班，上多久，也许是24小时，也许是48小时，甚至是72小时。然后我就可以休息，在没日没夜的昏睡结束后，迎接我的也许是阳光明媚的早晨，或许是更深更冷的夜。我经常在午夜时分醒来，穿着条内裤，倒在我的椅子的怀抱里，点上一根烟，发呆。我凝视着黑暗的窗子，倾听着过往车辆偶尔发出的声音，觉得自己荒谬已极。手机和传呼静静的躺在桌上，等待单位里什么人再次对我下达上班的命令。明天一大早，我也许就必须飞往某个不确定的城市，在不确定的时间内完成我某个不确定的任务。明天对于我来说，永远都是不确定的，象是魔术师的盒子，我永远不知道里面将会跳出来什么东西。

　　我有时会很怀念我的朋友们，不知道他们现在在做什么？是在什么地方喝酒？还是和自己的女朋友躺在温暖的被窝里？当我休息的时候，他们都在工作；等我工作的时候，他们正在休息。正如我现在清醒着，而他们已经进入了梦乡。我一般在这种时候，都会想起童年读过的一个故事《被忘记的小老鼠》。小老鼠起床的时候，家人都已经上班去了；小老鼠放学的时候，家人都已经睡了。于是大家渐渐的将他忘记了，居然在某天搬家的时候根本没有想起他来。于是小老鼠开始了他漫长的旅途，去寻找他的家人。我每次想起这故事，就更加觉得这世界真是荒谬透顶。我自己当然没有被我的家人遗忘，老两口分了新房子，说是要补过蜜月，留了新家老家的两套钥匙给我，就从我的生活里消失了。中国人一辈子为儿为女，直到老来的时候，只有很少的一部分人才终于清醒过来，去追寻自己消逝了的美好岁月，我的父母就是这样的人。我不再是他们生活的重心了，他们决定现在开始为自己活着，享受生活而不是照顾生活。我被遗忘在市中心的老房子里，现在居然是我时不时打电话问候他们过得怎么样？而他们象两个乐而忘形的年轻人，说他们很快乐，叫我忙自己的事去吧。我顿时觉得自己老去了。在老房子里，我一个人独居。一年前，我的女朋友终于无法忍受我不确定性的出现和失踪，和她全家一起去了美国。我一直也没有搞懂，我们在大学四年里为什么相爱？又是什么让我们在这城市里度过了三年破烂不堪的爱情生活？是什么让她坚持三年的游戏，每天她找我，我找她？最后什么又终于成为了骆驼背上的最后一根草，使她决定终于放手？我一直在问自己，我真的爱她吗？或者她有没有爱过我？最后的三年时光里，我们之间还存在爱情吗？或者只是因为惯性而勉强的在一起？我记得在最后一次约会的时候，我刚下了班。我们拥抱在一起，她在我的耳边不停的在说着些什么。而我由于极度的困倦，靠着她的肩膀睡着了。我想她一定说了很多话，因为我是被背上冰凉的一大片泪水惊醒的。可她究竟在那一天说了什么？我根本不知道。然后她就走了，留给我一把她家大门的钥匙，叫我有空的时候去看看。我很久以后才终于能从那件毛衣上闻出泪水的味道，听见她的哭泣声。

　　我把这件毛衣挂在我的床头，在无数个午夜，我醒来躺在床上大口大口的喝黑牌威士忌。在那令人怀旧的酒香里，我伸手拉住毛衣的袖子。空荡荡的感觉和毛衣的触感，让人觉得自己是在从什么地方高速往下猾落，没有尽头的滑落。我拉过毛衣遮住自己的脸，先是酒的香味，然后就是她脸上的清香，最后弥漫在空气里的都是泪水的味道。我透过毛衣的经纬线，看见了毛绒绒的温暖灯光。不知道是酒精的温暖还是那粗糙灯光的关系，我觉得非常温暖，看见了许多昨天里的美好场景重新一幕幕在一个个毛衣的小格子里上演。这几乎让我微笑起来了，甚至可以听见她的笑声又一次在耳边清晰的响起。随后就是突如其来的椎心的痛苦，让人根本喘不过气来的痛苦。无数次里，我都以为自己要死了。在那些伤感的味道里，我的心绷成了一跟弦，有一根无限长的琴弓粗糙而火辣辣的拉过，于是它绷得越来越紧，发出越来越高的音调。最后我的头在最高音处被炸得粉碎，泪水混合着酒，毫无羞耻的奔流而下，畅快淋漓。

　　在某一次大醉之后，我把一幅世界地图钉在了床边的墙上。用剪刀小心翼翼的将美国全部剪去了，并把那一大块空白染成了黑色。最后再用飞镖把她的照片钉在了阿尓巴尼，纽约州，她现在所在的地方。而我每次进卧室的门，第一眼就可以看见她在漆黑的美国朝我微笑着。象是站在深不可测的黑洞的门口，正要旋转着被吸进去，消失在连光线也逃不出来的黑色深渊里。我当然知道她就在这地球上和我一样的生活着，在某个确定的点，某个确定的城市。但是，她从我的生活里消失了，永远的消失了。我拥有的仅只是一件毛衣和一张照片，以及无数的回忆，我再也不可能听见她的笑声，不能拥抱她熟悉的身躯，闻到她发丝间的清香。简单的说，她从我生活里消失了，她对于我来说已经不存在了。而她现在在美国的某个地理概念上的地方，与我其实一点关系也没有，没有任何的意义。因此，美国对于我来说也没有了任何意义，美国也消失了，只是一片墨迹染成的黑色地带。我想这是很荒谬的，我的生活里再也没有了她，因此也从来没有一个叫美国的地方可以证明她的存在，美国也是不存在的。说她去了美国，其实跟说她去了天堂没有任何本质上的区别。她没有存在过，美国也不真实的存在，甚至是我，由于我不知道我下一秒出现在什么地方，由于这种不确定性，连我自己也是不存在的。

　　作出了如此荒谬绝伦的结论之后，我想我怕是快要疯了。在长期的远避人群之后，一个人很容易在自己的小房间里成为一个伟大的哲学家，或者象我一样疯了。我想我最好还是出去看看走走，重新和人类在一起。所以在某个清晨，我拖出自己的自行车，清洗，修理，上油，在早晨8点钟骑了上街。我汇入了上下班的车流，缩着脖子感觉清晨的凉风从后领灌进脊梁凉嗖嗖的刺激，眯着眼睛感受朝阳照耀着我苍白的脸上所带来的暖意。加上我和如此多的同类生物在一起，我快乐得几乎大叫起来。看着别人自行车前兜里的奶瓶和蔬菜，我又开始对生活有了感觉。我只是那么跟着车流前进着，象是珊瑚礁边集群的鱼，不知道为什么大家要在一起，但是我就是喜欢在一起的感觉。我明知道我和他们其实不是一类的人，过着一样的生活，但是在那个清晨，我喜欢那种在人群中的感觉。我以为自己是他们中的一员，一个活生生的、有感觉的人，我是存在的。就象是鱼群里的一条热带鱼。

　　在某个不确定的清晨，我，一个不确定的人，在街上骑着自行车，不确定的走着。10点31分，我终于停了下来，在我前女朋友家的门口。我想这没有什么可以奇怪的，就象是一个喝醉了酒的人，凭着最后一点残存的理智，也能找到回家的路。我想，她家和我家都是我意识最深处的终点站。假使有一天我变成了个痴呆病人，也根本不用担心我丢失了。我不在自己家门口，就一定在她家门口。因为除此以外，我别无选择。我茫然的下了车，掏出钥匙去开她家的门。钥匙长期不用，槽里都锈蚀了。门锁发出非常尖锐的声音，仿佛是在抗议我的遗忘，手上的感觉相当的涩。一年了，自从她走后，我都没有再来。门后面是什么？我不知道。不知道还剩什么？我们的"浪漫沙发"还在吗？我犹豫在门口，不知道自己应该不应该进去。身后响起了脚步声，我不再迟疑，推开门走了进去。

　　门在我身后关了起来，空气的流动把地上的灰尘扬了起来。我站在玄观，环视四周。所有我熟悉的家具都用布盖了起来，上面是厚厚的灰尘。空气里有种什么味道，非常的干燥，没有一点人的味道，一片死寂。可以看得出，走的时候有些匆忙，地上还留下了一些散落的纸片。使灰尘看起来有隆起的地方。电视机柜的几个抽屉还是半开着，好象还在等待主人回来把他们关上。我缓缓的前进，在地上留下了清晰的脚印。每一步落下去，都有极细微的灰尘四散开去，打着滚向四周蔓延，最后又无声无息的落下。我一直走到她的卧室前面，推开门。我笑了起来，墙上还是那个小丫头的画：焦黄的小辫，翘翘的小鼻子，脸上满是可爱的雀斑。我们当时一眼就看中了这幅画，买了回来。多少次，我们躺在床上，她靠着我的胸膛，指着那幅画说："将来我们也要生那样的一个女儿。"可是一直到她走，我们也没有给她叫什么名字有个统一的意见。我想现在，如果我有一个女儿，我会叫她无雨。用来纪念她和我的爱情，没有一点水份和杂质的爱情。这将是我的大秘密，将来连我老婆我也绝对不会告诉。我怀疑每个坚持要给自己女儿起名字的男人都有个秘密，不过谁也不知道罢了。我吹落我女儿脸上的灰尘，摸了摸她的小脸，退出了她的房间。

　　我不确定的毛病又犯了，我在客厅里来回的走着。用脚踢着散落在地上的东西，突然我的脚踢到了什么东西上，它飞快的移动开去，扬起一大团灰尘。我的目光跟随而去，阳光透过窗子照射在它上面，反射出耀眼的光芒。我走过去弯腰拾起来，发现是一盒CD。封面由于太阳的曝晒，已经变了颜色。不过我还是可以看出来，是李宗盛的早期作品集——《和自己赛跑的人》。李是个天才，捧红了一个又一个歌星。在适应市场需求的时候，他的歌越写越复杂精致，很能赚一大批中青年妇女的眼泪，引起她们的人生感喟，引起唏嘘一片，却不复当年的简单纯朴和打动人心的力量。《和自己赛跑的人》正是他早期的作品，歌词简单，旋律朴素，可以用一把木吉它就能演唱。当年我第一次听的时候深深为之感动，然而他继续的却是《寂寞难奈》的道路，在《凡人歌》之后，我再不听他的音乐。而我奇怪的是，我和我的前女朋友相处那么长的时间，并没有看她家有这片CD。也不曾听说过，她喜欢这片CD。我试图打开它，手一滑，整盒CD落在了地上，四分五裂。

　　我暗暗骂了一声，我知道这是早期慢性酒精中毒的表现——手足震颤。我想自己用不了到老，就会因为这而得上老儿痴呆，每天快乐的流着口水烤太阳。等我去捡起散落一地的CD盒时，却意外的发现了一张写满字的纸条。我拿起来读了下去："朋友一定要借这片CD去听。我想她仅只是喜欢这CD的名字，为自己即将下的决定多一点决心。我自己很喜欢它，而对于无法欣赏它的人来说，这片CD却不具有在我心里同样的价值。我相信有一天这片CD会流转到其他人的手里，如果有可能，您看见了我的字条，请您把它还给我。谢谢！"签名：劳拉。下面是一个网址：www.laura1999.king.net。我觉得有什么东西鲠在了喉头，我想我完全同意她的话，因此决定把这盒CD还给劳拉。不管她是谁，就算是因为她和我一样喜欢同样的歌的缘故也好，反正我要找到她，把她珍视的东西还给她。因为，一个人能失去的美好的东西太多了，却往往无力挽回。能为一个喜欢同样美好事物的人去做点什么，我认为意义非凡。

　　出了门，我急CALL睡猫。他是我的朋友，搞电脑的。他应该知道那一行WWW是什么意思。睡猫在大怒中从睡梦中醒来，很不情愿的听我讲完。告诉我说，也许我需要一台电脑上网了。我的生活如此简单，所有的东西都可以用一个包一提就走，我喜欢有效率和高速。所以在12小时后，我的桌上出现了一台新电脑，以及一大堆睡猫推荐的电脑书籍。在接下来的24小时内，我没有合眼。我又开始了学生时代的生活，在墙上贴了一大张纸。很快的，上面布满了各种颜色的字、即时帖、复杂的线条。等这张纸已经被我画满的时候，我基本上知道了什么是上网，我需要些什么，注意些什么。我用颤抖的手建立了一个新连接，取消了登录网络和除TcP/IP之外其他的连接协议。用睡猫给我的帐号上了网，在IE的窗口里输入了：www.laura1999.king.net，静静等待她的"竹叶"的出现。IE的进度条缓慢的象前伸展，我好象是在等待法庭最后宣判结果的罪犯，我眼前即将出现的是什么呢？我是个不确定的人，第一次去做一件确定的事，多少使我不是很自在。很多年来，这是我第一次去做一件具体而微的事：寻找劳拉。

　　78号，睡猫先生转。必复，欢迎光临惠顾。而劳拉呢？不知道性别，不知道年龄，不知道是一个人还是一群人，不知道活动地点，不知道是否还活着。唯一手头的线索是：一张纸条，一盒CD，一个主页，以及一堆问题；还有的就是我对"她"如同滔滔江水的、无来由的、莫名其妙的好感。我的论证过程是：1，如果她和我在同一城市，而且活着；2,如果她的年龄在19到25岁。3，如果她能终于联系上并且见面。4，如果她能爱上我。以上条件如果成立，则：我有可能获得一个女朋友。以上条件不成立的情况：太多，无法用15G硬盘装下。我点上第43棵香烟，考虑我身上到底发生了什么？我的生活就象是原始的地球海洋，劳拉就象是一道闪电，击中了海面。于是，本来根本无序的大海出现了生命的自组织，变得有意思起来。所以无论如何，我都要找到她，表示一下我的感激。想到这里，我深深的为自己是这样一个充满理性的人而感到高兴！而等我看见墙上的地图和照片的时候，我却深深的惭愧起来。好好的一个人，怎么就这么虚伪呢？明明知道别人去了美国，自己被放弃了，就用美国不存在来安慰自己，而自己内心的伤痛却根本无法掩饰。痛就是痛！明明被CD上的纸条所打动，被主页上的小把戏所吸引，却满口的逻辑推理。爱就是爱！即使劳拉不存在，我也是爱上了一个虚拟的人物！说那么多费话是干什么？看看自己，什么德行？披头散发，人模狗样，貌丑如猪，卑鄙如狗，怎么追求幸福？如何闯荡江湖？！！！我根本就不是什么Sphinx，整个一机器猫里的小孬！看见人睡猫幸福了，就小资产阶级情绪发作，还是个男人吗？我凭什么难过？凭什么再拖累一个女孩子？我值得另一个人等我八年吗？

　　我一把掐灭烟头，拍案而起，把《古墓丽影》安装进我的机子，决定开始和劳拉一起去冒险。前进，后退，下蹲，拔枪，侧滚翻，加速跑跳跃。。。。。。我迷上了另一个虚幻的劳拉。只要我不去睡，她就陪着我，让一个小孬感觉作为一个英雄的快乐和成就感。我和她在世界各地周游，为了寻找传说中的宝藏，我把大把的金钱和时间放在计算机里烧了。我想我爱上了戴墨镜挎双枪的她，她每一次受伤我都心痛不已；她每一次倒地牺牲我都泪流满面；她每一次取得一关的胜利我都欢呼雀跃。我把地图撤了，换上了劳拉的巨幅画像！我接到命令的时候，都和她依依不舍，执手相看泪眼，竟无语凝噎！我开始问我的领导，有没有到埃及金字塔或者长城附近的任务？因为我的劳拉在那里。每次回到家，我都会迫不及待的打开电脑，马上和她在一起。我开始学习电脑绘图技术，想把劳拉变得更完美。因为我觉得她的胸部过于饱满，很明显她的作者缺乏母乳喂养，所以把我的劳拉变成了个性感女郎，而不是我的爱人。终于有一天，睡猫的婚礼到了。没有铺张的酒宴，他和小璐就把我们几个老朋友请到他们的新房里喝了顿酒。我看着小两口简单而温馨的家，舒服的沙发，精致的踏脚垫，对每一寸空间的充分利用，感觉心头堵得厉害。他们在厨房里忙出忙进，偶尔在擦身而过的时候，彼此对以疲惫而幸福微笑，我觉得鼻子被谁打了一拳。坐也不是站也不是的熬到吃饭时间，除了我，每个人都和自己的女朋友或者老婆一起出现在餐桌上。那天我说了很多的笑话，喝了很多的酒，每个人都笑得上气不接下气。我大笑着抱着酒瓶倒在了地上，满脸的泪水。朋友们说我喝高了，送我回了家。送我回到冷冰冰的家里，除了我一个活人，连一盆花都没有的家里。那一夜，我翻箱倒柜的找出所有的酒来喝，第二天，我发现自己在浴缸里抱着那件毛衣睡了一整夜。

　　我清醒过来以后的第一件事就是把游戏从我的硬盘里删了。当我伸手准备去撕墙上的画像时，我想了想还是没有动手。而是上网，进入"劳拉的家"，在她的留言板上写了一段话："我们彼此都不认识，我是通过你在一盒CD里留的条子找到你的主页的。我想把你的CD还给你，或者你已经忘记了。这都不重要，重要的是我已经对你的话做到了我能做的事,和我将要做的改变。"劳拉没有回答我的话，也许她忘了这件事，或者忘记了自己曾经有那么一个主页，甚至这世界上就根本不存在一个叫劳拉的聪明的女孩子。一切都是我的想象，一切都是一个我女朋友安排好的计划，只是让我去熬过没有她的艰难的时间。我把我女朋友的照片重新挂在劳拉旁边。在每一个寂静无人的深夜里喝着酒，微笑着看着她们两个人。我开始频繁的上网，看了许许多多的网恋的悲剧或者喜剧，看着他们的分分和和，哭闹或者欢笑，我都会对着墙上的人微笑。时间又匆匆过去了一年，睡猫他们有了个女儿。我去看了，很象我画里的女儿。她怕生人，但是只不怕我。让我用胡子扎她，她却咯咯的笑个不停。睡猫的生活依然很清贫，但是小两口依然非常恩爱。看着他们，以及他们生命的延续，我也感觉到他们的那种幸福。我在匆匆的流光里又老了一岁，却再没有醉过。我依然在各个城市之间飞行，去完成任务，只是不再喝咖啡了。我在我生日那天，把我女朋友的东西，李宗盛的CD，以及墙上所有的东西都打了个包，封存起来。我没有邀请任何一个朋友，关闭了手机和传呼，在我的房间里喝了一天的酒。

　　新的一年很快就要到了，我也习惯了一个人的生活。在每个午夜梦回的时候，我还是会想起我的女朋友和劳拉，但是已经没有了悲伤。我开始相信每个人都有一个属于自己的劳拉，或者没有到来，或者拥有过而没有珍惜。我不知道我是属于前者抑或后者，也不知道谁才真是我的劳拉？每天早晨，我都怀着对太阳感激的心情醒来，骑着自行车穿行在我的城市里。在滚滚的车流里，你根本不能分辨出我来。也不知道在我们这座城市里，有我这么一个人，在寻找劳拉。

后记:

　　首先我感谢您看完这篇小说，从2000年11月10日凌晨零点开始，到早上9点结束。我花了九个小时完成了这篇一万多字的小说。我在网上已经泡了三年了，时时刻刻困扰我的有几个问题：为什么现在的人总处于对爱情的渴望状态中呢？为什么网恋不成功的如此之多呢？为什么那么多人深陷网络和游戏而不能自拔呢？我在这一篇短篇小说里试图找出一点原因。当创作的冲动来临的时候，我无法停止它。而在这么短的时间里，完成这么一个东西，很明显是很粗糙的。我知道这也许远不是在真正文学意义上的一篇小说，而是在网络生存状态下的一篇笔记。我刻意的回避了一段真正网恋的发生，我知道很多朋友会很失望。我也想写一篇精彩的网恋故事，但是我觉得目前的样子对Sphnix是最好的结局。我不想欺骗他说真的有一个美好的劳拉在等待着他，更不想让他又一次被现实残酷的伤害。最后，他回到了平静的心态，重新过上了正常的生活。就让我们看着他安静的生活，继续寻找他的劳拉好了。这是我真正意义上完成的第一篇网络小说，还希望得到你们的喜爱和批评。最后，我再次感谢大家的耐心阅读!

　　今天是周末，希望大家周末愉快！
