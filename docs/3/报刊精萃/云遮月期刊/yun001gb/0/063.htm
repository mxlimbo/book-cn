北海故事——狼桥遗梦（六）

萧十一狼/文

六

　　虽然那一晚上在我们之间发生了一些不快，我以为我以后再不会找她聊天了。

　　我总以为那天晚上发生的不快会在我们的聊天中产生一些阴影，至少会使我们在聊天的时候有所顾及有所拘谨最后导致我们再也无法聊下去了。

　　可事实却恰恰相反，那次不快的事似乎根本就没有在我们之间留下任何隔阂，反而使我们的距离更为靠近了，从那以后我们什么都聊包括以前我们很少提及的一些彼此情感上的事，每天晚上我们都能毫无顾忌的聊个天昏地暗飞沙走石哈哈大笑或是嚎啕大哭的，我们彼此之间所有的快乐和忧伤都能如实的让对方分享或是分担。

　　每天晚上我一上云之南就直奔聊天室而去，每天晚上她也都在，然后我们就一直开心的聊直到12点她下线。

　　那真是一段快乐的时光啊，任何开心似乎都能扩大几倍，任何忧愁烦恼似乎都已不再重要，只要我们能够在一起就是最好的了。

　　我想这就应该算是我们的网络友谊吧。

　　我们的网络友谊似乎已经变得如磐石一般坚固。

　　大约是在8月中旬的一天吧，远在北方的女友给我来了一封email，在email里女友说她即将出国了，到加拿大去，她深深的怀念了我们几年来的感情以及残酷的现实与美好的愿望之间巨大的距离迫使我们不能在一起的一些话，最后她说了："我们分手吧"。

　　我是心无波澜的看完了女友的email，应该说对于这个结局我是早已料到了的，再坚固再深厚的感情也都会有崩溃的一天，何况我们经过了长年空间上的距离而变的彼此有那么一点陌生了。

　　虽然对于我和女友这种结果早已料到了，但伤感始终是会有的，毕竟那是我一直不愿意看到的结局，毕竟那是我几年来一直辛苦耕耘着的初恋，毕竟我在这份感情里投入了太多太多的东西。而要我在一夜之间放弃这段3年来一直苦苦经营着的感情又是谈何容易的事啊。

　　那晚我很早的来到了我的聊天室，很忧愁，没有唱歌没有说话。

　　过了一会桥来了。桥：你好，我来了。狼：你好。桥：你今晚没唱歌？桥：也没说话？怎么了？狼：没什么的，心情不太好。桥：又怎么了？被老板骂了？狼：不是。桥：那钱包丢了？走路摔交了？狼：不是。桥：吃饭噎到了？呵呵狼：都不是，我女朋友给我写了封email。桥：那是好事啊，你女朋友怎么跟你说的？想你了？狼：不是，她说我们分手吧。…………桥：不好意思啊。狼：没什么的。桥：那你是怎么想的？狼：我觉得分手是迟早的事。我们不在一起的时间太长了，都会有变化的。桥：是啊，还是理智一点的好。狼：可让我一下子就放弃了还是很难以接受的。桥：那你以后准备怎么办？狼：我也不知道了，混着看呗。…………狼：其实我和我女朋友感情很深的。桥：我看得出来的。狼：我们从大学二年纪就在一起了。狼：其实说实话，我也不知道她是不是喜欢我的，狼：也许我一直就在自做多情的。狼：我从来就不知道她在想些什么东西。狼：她也许不爱我的，狼：也许她跟我在一起只是逢场作戏。桥：我在听着。狼：不过我又觉得她一定还是爱过我的。狼：她虽然从来不说爱我。狼：但是她们宿舍的人都跟我说她每天在宿舍里至少要提10次我的。狼：看得出来，她还是很在乎我的。…………

　　我就这样和她聊着，应该说是我一个人在说话，我跟她说女友的种种好处；跟她说女友的种种缺点；跟她说女友的小脾气；跟她说女友对我的恶作剧；跟她说我对女友的深深的感情；跟她说我和女友在一起时快乐的时光；跟她说我是如何怀念那些日子的；跟她说我当时是如何发誓要等女友几年的；跟她说我和女友在一起的一切…………

　　她什么也没有说，一直都在听着我的倾诉。…………狼：我现在真觉得不应该离开她回云南的。狼：也许我不回来的话我们是不会走到今天今天这一步的桥：我累了，要下了。狼：你累了吗？我还没说完呢。狼：再陪我一会好吗？桥：我知道了。狼：你知道什么了？桥：我知道你很爱她的。桥：其实你一直都在爱着她，虽说你们离开了很长的时间。狼：是啊，我一直都还是很爱她的。桥：我要下了，我现在心情有点不好。狼：怎么了，我勾起了你的伤心往事？桥：不是的，我知道了你一直都爱着她，所以心情不好。狼：为什么？是不是觉得我的付出得不到回报而心情不好？是不是觉得我特可悲？桥：不是这样的，你一直都很爱很爱她的。桥：我现在很伤心，再见了。

　　小桥依依不舍的离开了聊天室

　　她走了。

　　她离线了。留下我一个人孤孤单单的在聊天室里。

　　我不知所措的看着显示器，眼中一片茫然。
