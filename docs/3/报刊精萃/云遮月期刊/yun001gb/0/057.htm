北海故事――狼桥遗梦（前言）

萧十一狼/文

前言

　　这是一个发生在云之南的真实的故事。

　　这是一个在现在这种年代已经被炒得泛滥了的关于网恋的故事。

　　我是一只狼，一只孤独的狼。

　　狼本是多情的。

　　狼的外表凶悍而残忍，狼的感情世界细腻而丰富。

　　可狼的内心却永远都是孤独的，因为更多的人看到的只是狼凶残的外表而忽视了狼的内心。

　　还好狼都不会在乎别人对他的看法，狼是孤傲的。

　　因此狼必须承受孤独。

　　狼必须孤独。

　　而乔则是一个很神秘的人。

　　我是一个很笨的人，我从来都不知道她所说过的话到底哪句是真哪句是假，她说的话象她人一样的神秘而猜不透。

　　直到现在我对她的了解也仅仅限于她在网上的名字叫乔以及她是一个女人并且是一个年龄比我至少大三岁的女人。

　　我不知道其他的关于她的任何消息，不论是她的过去，现在还是将来。我从来没有问过，她也从来没有对我说起过。

　　其实她也告诉了我很多的关于她的东西，包括她的真实的名字，包括她的工作单位，甚至还包括她是一个结了婚又离了的苦命的女人。

　　不知道怎么的，对于她告诉我的姓名和她的那些故事，我一直都持怀疑的态度，但我也没有为此而质疑过她。

　　她在我的眼中一直都是一个神秘的人。
