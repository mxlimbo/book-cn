家事（外一首）

富顺才子/文

黄土地的日子
平凡而又离奇
家事如父亲手中的锄把
光滑圆润
家事如母亲的皱纹
布满艰辛
家事 如粮食般
喂养我一生
我曾魂游原野
家事 一声声唤我
于炊烟袅袅时
泪湿家门
待到麦香四溢
青蛙走穴
母亲的引领
一群晚归的鸭子
摇摆归来 它们
仿佛是家的标点
极不规则
又富神韵

山那边，是祖坟

黑油油的土地
长麦穗玉米
黑油油的泥土
垒成一座座坟
我的祖辈们
用一生的汗水
年年 换回
金黄的麦粒玉米
也换回一群结实的儿孙
老了 最终
入土为安
年年有清明
纸钱和爆竹
烧的烧 叫的叫
先辈啊
在另一个世界里
你们 还流不流汗
而我们
年年的祭奠
是不是打破了
你们难得的平静
