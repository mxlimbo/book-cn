麦皮，豆豆，青蛙，还有网络情人

地点：云遮月客厅
主持：麦皮

（旁白：活跃的麦皮一推门，只跨进一条腿，便大声嚷嚷：）


　　我有好多名字哦！都很动听的，你想知道吗？想知道你就说嘛，你不说我怎么知道你想知道？

　　没道理你想知道我却不让你知道，你不想知道我又偏偏要让你知道，

　　大家讲道理嘛，你真的想知道吗？难道你真的想知道？

　　你不会是真的想知道吧？？？

乔燕：

　　麦皮呀，你知道我想知道啊，怎么和我一样，一高兴就会一大串一大串地来，象那春天，田野上因春风的来到而成片成片盛开的迎春花，星星点点，全都睁着眼睛要我回答？

　　那你就静静地听啊：我要知道，要知道！

　　你说吧，用你眼睛上细细的睫毛，用你鼻尖上晶莹的露珠，用你手背上轻落的蝴蝶，用你前行时带出的微风……

　　够不够啊，如果不够，还有啊……


麦皮：

　　麦皮——

　　高中时同学戏称我为麦皮，源于本人之真姓，俺和燕子一样姓乔，被称为荞麦皮，后为了表示亲切的称呼，遂省去俺的姓，直呼俺为"麦皮"。上网后，一时想不起什么好名字，就以此为名，后来就出现了一代那个什么——麦某人！（常用名）

　　豆豆——

　　98年在碧海银沙聊天市里起的第二个名字，来源是为了纪念我认识的一个女兵，她在部队被称为"豆豆"，嗓子特好，我喜欢听她唱歌、喜欢和她聊天，她是川妹子，遗憾的是我们退伍后就再也没见过！

　　青蛙——

　　在湛江碧海银沙里的第三个名字，（那时《第一次的亲密接触》一文还没在大陆流行，也就是说我还不知道网络上的青蛙是代表什么）我是那里闹的最厉害的一只青蛙，每天晚上都爬在我的泥塘里面，和一些朋友来大闹泥塘，闹的筋疲力尽的时候才想起要睡觉。。。

　　皮皮——

　　到洛阳的聊天市里后（先在讯业金网，后去信息港）也是闹的人人皆知，网友们又亲切的称呼我为皮皮，于是我连这个也去抢回来了。

　　网络情人——

　　呵呵，看到这个名字就该知道这完全属于我用来开玩笑的名字，不过不要以为是在网络上找到自己的情人，真正的意思是，这份感情既然来于网络，它依然还属于网络，如果你要把它变成现实，那么，你会哭着从我身边走开。。。因为，这个情人只属于网络！！

　　其他的都是一些随着心情而起的名字，

　　高兴的时候（名字）：

　　偏偏爱上你、知道我在等你吗、青蛙王子

　　不高兴的时候（名字）：

　　走过路过偏错过，一世情缘一线牵、笨蛋加傻瓜、等等。。。一时不能全想起来，就这么多吧。。。


百果之秋：

　　其实他们就象你的心情名片，记述了你一路走来的"酸、甜、苦、辣、盐"。他们值得你永远珍藏！


老马：

　　皮皮的经历真是丰富啊~

　　完全可以写个《皮皮网络历险记》，保证畅销！

麦皮：

　　其实在很早以前我就有一种要我把我网络生活写下来的冲动，但是，文学水平有限，不敢献拙，每个朋友都有一段故事，不知道要写多少。。。。

　　唉。。。

　　还是让他留在记忆里吧。。。。

　　落花有心随流水

　　流水无意恋落花

　　就象秋秋说的那样，这些都是我的心情名片！


百果之秋:

　　才几天呀，我的名字怎么又换了，我看，你不光是自己喜欢给自己起名字，给别人起名你也是乐此不疲呀，还秋秋，我还蝈蝈呢！！


俏儿:

　　皮皮？

　　皮皮鲁？？

　　鲁西西在哪？？？

　　：）


~ 燕子 ~:

　　 噢，想不到你有么多有趣的名字，真得好好编个童话故事，让你都进去！什么，自已编，当然好了！

　　 俏儿，皮皮（鲁）不是在故事里吗？啊，你问故事啊，我，我，我还没写呢！


麦皮：

　　呵呵！见笑见笑。。。

　　名字都很一般，只是随心情而已。。。。

　　现在这些名字几乎都不用了，

　　回到麦皮时代。。。

加菲：

　　还记得郑渊洁写的姐妹篇《皮皮鲁外传》，《鲁西西外传》，《荞麦皮外传》，我也曾用过这三个名字，不过是不想忘记一些事情。

　　你的许多名字，不正象当时的心情一样，记录了你的许多喜怒哀乐。

　　麦皮，一个很好的名字。


伊人可可：

　　麦皮真情倾诉——名字小传


麦皮：

　　每个人的名字都应该有他的意义，有的是纪念，有的是心情，有的是玩笑，有的只是无意中随手敲上去的，但这些可都代表着你当时的心情，比如说，如果我看到一个叫"青蛙公主"的名字，那我会马上去换个"青蛙王子"进来。。。

　　名字多了也不是什么坏事，虽然我可以用他来隐藏我的身份去做一些无聊的事情（骂人，发泄）但现在看来，我却对这种方式不屑一顾，还不如把自己的心情表现出来，这样心里会好受点，如果当时有人肯听我的故事，我一定会告诉他，有时，有人分享自己的忧愁也是一件好事！
