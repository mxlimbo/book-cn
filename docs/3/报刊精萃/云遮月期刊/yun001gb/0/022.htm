十八＊燕＊鱼

燕十八/文

　　晨，燕来电，遂起。燕曰：君记否昨夜之事？余大惑，问道何事，燕曰：汝昨夜言，今午将烹鱼予我，忘否？吾大窘：昨夜恍恍之言何以为真？燕佯怒，微嗔：君子一言，驷马难追，何敢欺我.吾忙陪罪，燕转喜。

　　出。入丹尼斯，少顷，提鱼。寒风中立路旁做打车状，后跳上四十路公交，绝尘而去。

　　直奔燕巢。欣然而入，燕大泣，吾不解，问曰：卿何至于此。燕以手指鱼：我不杀伯仁，伯仁却因我而死。吾忙劝道：此鱼并非死于我手。燕又转喜。笑道：汝先陪我小坐，少时烹鱼。

　　鱼熟。置于案。燕复指鱼，曰：浮想鱼游于水，而今竟成我盘中之物，不胜唏嘘。遂取箸于手。可怜此鱼生前二斤六两，片刻鱼骨一堆，不复存也。

　　余立于旁，观罢，呜乎，鱼若能言，必放声大哭：既生鱼，何生燕。
