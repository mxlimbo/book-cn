我的2000年――一个砖手的自白

萧十一狼/文

　　最近我一直在思考一个问题：我们网络生活的意义究竟何在？

　　我不能说我们的生活就是网络，这很明显是不可能的。但是网络早已变成了我们生活中的一部分，这也是让我们无法逃避的说法。

　　最近一直有朋友对我说要远离网络但又很难痛下决心割舍这块乐土。这未免让我有点难以理解，既然网络在我们的心里已经深深的扎下了根，已经成了我们生活中也许是不可或缺的一个重要环节，那么我们为什么又要去逃避它呢？

　　我上网已经快一年了。在我最开始上网的那一段时间里，我被外面花花绿绿的东西迷花了眼，整日都在网上不停的冲浪，从一个地方冲到另一个地方，不停的浏览着从身边掠过的一切新鲜的玩意。

　　记得有一天，我突然发觉自己很累很疲惫而身边却没有一个朋友来陪着我跟我说话的时候，我分明感觉到了一丝悲哀。上网那么长的时间，一直都把注意力放到了浏览新闻和观看上面，而忽视了在网络世界里最为重要的人与人之间的交流。一时间我不知所措，我的网络生活似乎一片黑暗。

　　刚好这时候，我来到了云之南。

　　当我带着满腔的怀疑和压抑来到云之南的时候，我的表达的欲望终于强烈的勃发而出。

　　作为现在云之南里一个还算有点知名度的砖手来说，错过了打破云之南宁静也是社区里的第一场板砖大战未免有点遗憾。很多人在那场大战中一举成名，包括现在赫赫有名的和菜头和笑哈哈也都是在那时成长起来的。

　　在很久以后我翻出当时的所有帖子来看了一遍，仍然可以感觉到当时的惨烈。而作为那场板砖大战中的两个主角――桑海和春风化雨，现在已经很少在社区里出现了，而越来越多的有着更强实力的新面孔不断的涌现在我们的眼前，这就让我不得不想起一句歌词：不是我不明白，这世界变化快。

　　之所以会错过第一次板砖大战，是因为当时我正躲在青青校园那个小世界里以一种两耳不闻窗外事的状态努力的写着我的第一部回忆体小说《今兮何兮》。

　　那时我根本就没有想过我会成为一名板砖手，那时我一直都在试图表达着自我。

　　先说一句实话：我在网上发帖子从来没有想过出名或是其他。

　　不记得在哪里听过这样一句话：很多东西当我们无法再拥有的时候，我们唯一能做的就是不要把它忘却。我当时的想法很简单，我只是想把一些回忆用文字的形式保存下来，让自己在多年以后翻到这些文章的时候还能够回想起当初的点点滴滴。仅此而已。

　　虽说没有想出名或是其他，但是虚荣心总是有那么一点点的。毕竟那是自己花费了大量时间精力与感情所写出来的，总是希望得到别人的承认与肯定。于是当我看到我花了整整两个星期所写出来的3万多字的回忆小说《今兮何兮》就那么摆在青青校园里而无人问津的时候，我的心分明感觉到了一种莫名的迷茫，它游荡在空气中。就象是房间里的一炉香，一截截无声无息的燃烧成灰烬然后落下，一切仿佛都凝固了，甚至是烟也笔直上升，四周寂静得可怕。

　　这种感觉在我丢出第一块近1000字的板砖然后马上就在社区里引起轩然大波之后变得更加的强烈。

　　很难清晰的对当时的感觉有什么样的印象，我一直在不停的问自己：为什么我花费了无数精力与感情写就的3万字的小说很少有人能看得完，而我不到半小时写成的一块近1000字的板砖却能让大家都认识我，这究竟是为了什么？

　　从我的第一块板砖出手之后，我在这个社区里似乎变得有那么一点名气了。每天都不停的有人给我发来在线直呼，有吹捧我夸我写得好的，也有不少是骂我的。对于这些传呼我不知道该怎样应付，我仍然困惑在自己那个看似很简单的问题中。

　　尽管那个问题我一直都没有想出一个合理的解释出来，但从那以后，我便在社区里不停的扔砖头，不停的找人论战。

　　我说过，在我来到云之南社区之后我的表达的欲望终于喷发出来。而当我在写完《今兮何兮》之后，我没有灵感了。我没有那么多的感情来让我描写，我所有的感情都在那一部小说里被我写完了，一丁点都不剩。而潜伏于我身体内部的一些不安分的因素仍然在作怪，那种表达的欲望仍然没有一丝消退的意思。

　　当我再没有可以让我写的东西的时候，我所能做的也只有不停的找人论战了。

　　这，何尝又不是一种悲哀！

　　从最初的炮打和菜头；到和笑哈哈关于社区条例问题展开论战；到和雨人就围棋的讨论；然后就是轰轰烈烈的大理网盟事件；而后是关于网络文学的论战，再到刚刚结束的边缘事件，我一直都站在战斗的最前沿。

　　我不知道是什么让我如此的冲动，每次当我看到让我看不惯的东西的时候我总是拍案而起，二话不说一块板砖就咆哮着招呼过去。记得我在现实中是一个很温文尔雅很善良很有礼貌的青年，又是什么让我一到网上就变了呢？

　　也许正如netplay说的那样，因为我们在现实社会里不得不圆滑一些，不得不做一些自己本不愿意做的事，因此当我们到了网络上，到了云之南这个虚拟的社会里面，我们便不再愿意扮演我们所不齿的那种角色。

　　这也许就是砖手的普遍心态吧。

　　最近很流行一种说法：上网就是为了找乐子，玩玩而已，不能当真。

　　我对这种说法很不以为然，但也谈不上反感。我不反对上网消遣的做法，但我上网一直遵循着一个自己的原则：我对网友一直都送出我的真诚，但我不会要求你们相信我的真诚更不会要求你们对我也同样的真诚。

　　记得在电影〈东邪西毒〉里面有这样的一句台词：我日复一日的坐在这里看着远处的高山，可是我从来没有真正读懂过眼前的这片沙漠。

　　我也很害怕自己会陷入这种境地，于是我不断的扔出板砖，希望能够了解别人也能让别人所了解。

　　还算不错，我在互相拍砖和共同战斗的过程中也找到了真诚的碰撞，那是一颗颗真心的碰撞。

　　我拍过和菜头，与笑哈哈论战过，与nimor在日本人的问题上辩论过，也和netplay在卫慧这个新生代的美女作家身上泛泛的做过简短的讨论。在这些互相拍砖与讨论的过程中都有过一些思想与心灵上的碰撞，而当这些碰撞到了大理网盟事件的时候终于有了共同点：我们都崇尚网络的民主与自由；我们都渴望着在不同思想碰撞中擦出来的火花；在我们的世界里都拒绝虚伪与矫情。

　　于是便有了滇之驴。有了这个让大家欢呼过也颤栗过的ID。其实在我们的眼里，滇之驴是以一个民主与自由的化身出现的，它本身并不代表什么小集团的利益，这也是我们在社区里广泛公布滇之驴密码的原因所在。

　　但是很遗憾，我们的初衷没有得到体现。也许是我们行事上的不妥，在众人的眼里，滇之驴似乎与我们的心愿背道而驰的走向了借自由的名义行使独裁的道路上了。现在滇之驴的ID已经很少出现了，但是它也曾经辉煌过，也曾经为我们的理想而奋斗着。而最重要的是，作为驴子的我们，友谊一直都存在于我们彼此的心灵中间，变得愈加的牢固。

　　不管滇之驴在众人心中是怎样一个印象，对于我来说是无悔的。在那些共同战斗的日子里，给予了我无尽的欢乐与笑容。在各位驴子一直为生计而奔波劳累的今天，我总是如此的怀念那一段晚上总是聊天到3点多的日子；总是那么的怀念聊天时为一个问题而讨论的不可开交的情形；也同样怀念为了一个玩笑而开怀大笑的开心的镜头。那真是一段快乐的时光啊，那段日子在现在大家东奔西走而很少碰面的今天显得尤为的珍贵。

　　在云之南这个虚拟的社会里，我付出了我满腔的真情，换回了数倍的真心。我已经很满足很满足了。

　　很多人都指责我，说是我把一批大理的优秀作家赶出了云之南，而导致了现在的云之南里没有什么好帖子出现。听到这些说法的出现，我同样不知道该如何回答他们，一如我最开始拍砖的时候面对着很多人发给我的在线直呼那样。

　　对于大理作家的退出，我也很痛心，但不会后悔。说实在的，我也很钦佩李智红等一帮人对文字的应用水平，至少我无法达到那样的高度。而如果能够让时光倒流回去再来一次的话，我想我还是会奋起而拍之的，因为他们的做法与我的网络道德标准相冲突。

　　而到了现在，在很多人眼里，我们追求的网络民主与自由变成了横行社区的借口；我们所期待的思想的碰撞变成了谋取虚名的工具；我们付出的真诚变成了虚假的义气；我们心灵的交锋变成了虚伪的表现。

　　我不知道我还能说些什么？

　　还是让我们回到文章开头的那个问题上来吧，我们除了以彼此心灵的交换来证明我们狰狞的面具下面拥有的也是一颗颗善良的心；除了以各式各样的辩论来提高自己的学识；除了以不断的网络论战来证明自己也是有才学的之外，谁又能告诉我，网络生活的意义到底在哪里？

　　时光永无休止，而年华却随岁月流逝。

　　很难判定我的选择正确与否。我只能走下去，不停歇的走下去。既然要做个旅人，目标就是远处的地平线。在我的每一年结束的时候，我都习惯的停下来休息一下，整理自己的思绪。把过去一年的回忆，好的、坏的刻在一座碑上，然后沉入心底的大洋深处。等我回顾我的一生的时候，凭借蜿蜒向前的石碑，我就可以知道我走过哪些路，错过了哪些地方。而在那条蜿蜒延伸的碑林间，有一条几不可见的细线，那是我的脚印。

　　我感谢上天，感谢上天让我在2000年接触到了网络，在度过了极其失意的1999年后，我以一种全新的容貌迎来了2000年。在这新世纪的第一个年头里，我寻找回了我曾经失去的很多珍贵的东西。自信、真诚、帮助、友谊，这些我最希望得到的东西一样又一样的从网络上飞下来，扑进了我的世界。

　　在今天赶去上班的路上，我看到一个擦皮鞋的人，他戴着discman随身听的耳机，正在随着音乐的节拍而扭动着身体。我第一次发现，这个世界也充满了阳光。

　　我不禁加快了步伐，在公司的电脑和电话线的后面，还有无数的朋友在等待着我。

　　2000年即将过去了，我很怀念它。
