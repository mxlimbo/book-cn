丑陋的中国足球

闲云野鹤

　　一场极具"中国足球联赛特色"的国际比赛在黎巴嫩的体育场悄然上演，出任主角的中国队远离了国内球迷，难得的轻松，当真胜似闲庭信步，把个"中国联赛"演绎的淋漓尽致，一个个是如痴如醉，全然没有了以往的那份落魄和尴尬，好一幅心高志远的休闲。

　　可怜的韩国队终于尝到了二十年对中国保持不败的苦果，亲身体会了被他人算计的苦涩。中国队是应该好好休息庆贺了，只是不知道这回是不是需要乘飞机到太平洋去冲冲浪什么的……卑鄙、无耻、下流、肮脏、恶心、丑陋的中国足球，你终于找到了证明自己、表现自己还有"足球意识"的机会，你终于凭借"神气教练"的魄力，将自己锻造成了一个不折不扣的"下三赖"的球队，你终于籍借他人之手，为自己的无数次失败找到了令人蔑视的宽慰，你还沾沾自喜，恬不知耻的吹嘘是战术的合理安排，你违背了体育道德精神的基本要求，你丧失了作为运动员的资格，你甚至失去了做人的资格，你的拙劣表现玷污了这神圣的绿茵球场，你蹩足的形象侮辱了那高高飘扬的五星红旗……我们为有你这样的球队而感到羞愧、愤怒，你将韩国队送到了伊朗的脚边，却将中国足球踢向了没落的深渊。

　　我天真的守在电视机前看着这场比赛，尽管我渴望看到一场激烈的比赛，但我再次失望；尽管中国队赢了，但我却感觉比输给印尼还难以接受。我们看到的是一场怎样的比赛——充满了消极色彩，散发着腐朽气息，洋溢着功利主义，一幅小市民、小心眼的恶心肮脏嘴脸装扮着这理应纯洁公正的体育场……七年的足球联赛没有提高我们球员的足球意识，却轻而易举的教会了我们假球的运作……

　　我已经说不清看球的动机了，也许只是在和宿命斗气。我渴望看到一场真正胜利——荡气回肠、酣畅淋漓、赏心悦目，但我却一次次的失望，即便对手弱小的可以只用五个人来对付，你依旧需要绷紧心弦，中国也不是总默默无闻，配合对手创造一个奇迹，对他们来说就象认出硬币中的百元大钞一样简单。
