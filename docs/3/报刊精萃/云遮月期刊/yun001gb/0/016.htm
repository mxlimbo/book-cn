我好失望

八神苍月

　　现在，人们都在追求时尚，特别是一些女孩子，染发、化妆、穿着特别时髦的衣服……可能是我有些保守，不是很喜欢这样的另类女孩。（声明：我并不是讨厌它们，只是不习惯这样的打扮）

　　前几天，我到街上去买东西，就看到了一个这样的女孩子，短寸的头发染成黄色，虽然天冷还穿着短裙，看上去象冻得一样的紫色唇膏，蓝色的眼影，我轻轻的摇了摇头。

　　买完东西了，往回家的路上走去，也许是机缘巧合，我又看到了她，她还是漫不经心的横着小调。

　　忽然觉得脚下一软，差点把我摔倒，低头一看，路面年久失修，一个大坑，哎，现在的街道都是这样的，没办法。正在边走边想的时候，听到身后咚的一声，回头一看，一个老汉，拉着一架子车的煤，架子车的轮子陷在坑里了。老汉奋力的往外拉，可是架子车还是我行我素，说不出来就不出来。忽然眼前晃过一团红影，我仔细一看，是刚才那个"另类女孩"，跑到了架子车的后面，用尽力气在架子车的后沿上推着。"老大爷，我数一、二、三咱们一起用力！""闺女，脏得很呀，别把你的花袄弄脏了！""没事！一、二……"当时在场的只有我们三个，路人最多回头看一眼……

　　架子车出来了，在一个"另类女孩"的帮助下，它出来了，不知道为什么，在那个时候我在原地足足的站了三分钟，也不知道心里是什么感觉，什么滋味……

　　这件事情深深的在我的大脑里印着，时不时的回想起来，现在，我明白那是什么感觉了，那是一种失望的感觉，我好失望，对路人，对社会，也对我自己……
