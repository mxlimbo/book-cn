八神苍月/文

一柳弯月
宛如姑娘的眉毛
悄悄的
挂在树梢
那么温柔
那么皎洁
一片青云
宛如姑娘的裙子
轻轻的
飘在旁边
那么轻盈
那么婉约
夜晚的天空
象姑娘害羞的脸
柔柔的
那么清澈
那么妩媚
看着它
不由得
痴了
醉了
仿佛
月看到了我
随手撕下一片云
遮住了脸……

　　水晶鞋

看着晶莹的水晶
仿佛恋人的心灵
高雅而不高傲
清醇而不失味道
透明而不苍白
精巧而不渺小
到了这里
忽然眼前一亮
“水晶鞋”
好美的名字
好象童话中的灰姑娘
在这里静静的等待着

　　爱情茶座

爱情难免失落
爱情难免差错
爱情难免挫折
爱情难免难过
当你面对这些问题无法解脱
有一个地方让你诉说

　　品味人生

有人说人生象一杯酒
苦辣酸甜全都有
有人说人生象杯茶
苦后就是甘甜
我说人生象杯水
想要什么滋味自己去准备
让我们慢慢的品味
人生的美
——爱情茶座
等待着……

　　心灵小憩

体会心灵最深处的感受
弥补心灵最深处的感伤
找一个让心灵能够得到安慰的地方
心灵小憩
