追踪雨来

乔燕/文

　　在青年话题滞留了片刻，环顾左右，忽然看到一个标题怎么这么眼熟，有"云遮月"还有"燕归来"，忙推门进来，没想到，竟一脚踩进另一别样的洞天。

　　原来是一神秘所在，有水声哗哗，只是光线欠佳。

　　侧耳静听良久，隐隐听到远处有雨来微弱的气息，只是还难以辨别方向。

　　终于醒悟了雨来潜入了一条暗河，此河直通"云遮月"，我已多次于此河水回游，雨来是如何摸到这艰幸之途？

　　想那雨来初次到访，暗河中波涛如此汹涌，暗道岔路又多如牛毛，真是辛苦了他，此时也一定是会渐感疲惫。

　　想至此我急急纵入水中，摸熟路向"云遮月"潜去。

　　燕归来了——

　　却只见雨来已酣然入梦，额上细珠尚凝。

　　不觉有些揪心，想这数九寒天……遂命阿妍取来毛巾，拿在手中轻沾。

　　又命阿妍传话备饭，好于雨来醒时便有热菜热酒端上，

　　然后再命阿妍唤众英雄会聚"云遮月客厅"，单等他醒来有要事相商。

　　一切安排停当，乔燕长叹一声，心中暗道：怎一个难字了得！

　　客房内，雨来轻酣仍在，众英雄纷纷手机来报：已赶在来到云遮月的途中……

　　忽又想起，当他醒时，当问此节选乃第几节，于全局中处于哪一地段，何时延伸令众英雄略知一二。

　　袅袅兮，云遮月内清香浮动。
