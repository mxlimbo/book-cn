解读乔燕的一首诗：《到深谷里去》

张卫宏/文

（原作）：

到深谷里去
乔燕/诗

　　泥土已不能托起沉静的灵魂
　　送行的雨悄然而至
　　临行，脚步沉重又孤独
　　一颗头颅的深处
　　一盏灯照亮的世界
　　穿透一座又一座山峰

　　那盏灯
　　给起点与终点同一个影子
　　谁要击开影子的门
　　令星月辗转难眠
　　霞光泪花飞溅
　　千万颗露珠将造就一片
　　无法捡拾的思想
　　一个人犹豫于走出痛苦而清醒的梦

　　到深谷里去
　　发现写在大山眼里的咒语
　　到咒语里去
　　种一棵会结果的栗树
　　世界的边缘藏于某一颗果子
　　果子坠落
　　深谷里升起一条尘封的道路

（2000年2月25日）

[ 解读 ]：

张卫宏/文

　　对一位诗歌初学者而言，“表达自我“的意识往往占据了创作一首诗的全部诗学内容。（如果这种意识只是为了抵抗世俗自我的诱惑，而不至于造成诗人心灵或人格的缺陷。）但摆在眼前的这首诗已经足以打消我先前的顾虑，整首诗中没有出现诗人惯用的”我“，这个简明的却至关重要的事实表明作者将站在另一个尚难以看清的高度向我们讲话。据我对作者的了解，这种转变并非无意识的即兴发挥，而恰恰出于一种写作的自觉。

　　“泥土已不能托起沉重的灵魂。“这个不太引人注意的悬念在我看来可以为这首不长的诗定下一种”宣叙“的基调。仿佛一支弦乐重奏，这第一组音符来自一把低沉的大提琴。诗人的灵魂也许正如音乐家的音符一样，需要一件乐器的把握和支撑。然而作者对此是有所疑虑的，就像乐器发出了奏音，那是灵魂得以沉静的泥土已经失去托起灵魂的支点。接下来的宣叙调由一把小提琴娓娓道来。也许是第一行的悬念对作者来说过于紧迫，也许是作者考虑到短诗必要的简洁和准确性，从第四行，演奏就进入了第一节的转折点。“一颗头颅的深处 / 一盏灯照亮的世界 / 穿透一座又一座山峰”。灵魂从“头颅的深处”进入“一盏灯照亮的”广大区域，最终在灯光的指引下跃上反观自身的制高点。诗人从这里回望灵魂跋涉的起点，终于发现了阴影存在的矛盾性质。“那盏灯 / 给起点与终点同一个影子 / 谁要击开影子的门”读者不难理解“正是阴影赋予事物真实性”这一事实，而作者所关心的并不止于揭示这一点，她要为赋予事物真实性的影子建造一扇可以进入其中的门，这种愿望的世俗性质就像一对恋人要为爱情建立适宜生活的家园一样急切。然而相对于晦暗的灵魂世界来说，这种愿望要比爱情之于复杂现实生活更难实现。

　　于是我们不得不面对作者创造的如下情境：“千万颗露珠将造就一片 / 无法捡拾的思想 / 一个人犹豫于走出痛苦而清醒的梦”我不得不对作者的言说感到由衷的赞叹！她为我们指明了思想存在的另一种状态：思想为大家所共有，但却并非能被我们轻易地“捡拾”，一旦错过机遇，它会象露珠一样消失得无影无踪。不仅如此，我也同时认定作者对梦的性质的准确把握，人们可以找到足够多的论据来质疑梦的可靠性，但从梦之于意识存在的本源来看，我们无法否定这样一个事实：只有做过梦的人才有可能透过梦与非梦相交叠的状态，接近意识存在的真实。

　　因此当诗人的灵魂再次从“一座又一座山峰”向下观望，我们才和作者一样有可能发现一条通往精神谷地的必由之路。那“写在大山眼里的咒语”仿佛具有一种魔法的力量，吸引着我们，当我们来到“一棵会结果的栗树”下等待有所发现的时候，我们再一次惊讶于作者独特的言说：“世界的边缘藏于某一颗果子”这有如天启的语言使我们不得不相信有限的生命可以和无限的世界在一权包蕴着种子的果子里合二为一。而我此时此刻能说的话正如另一位诗人已经说过的：“作为一种光线，我们就是历史 / 被神秘的使命所创造 / 与动荡的门窗相适应的 / 是橙红的太阳缓缓下沉 / 不需要理解，只需要相信 / 风吹着太阳，觉悟是黄金。”

2000年12月21日夜
