眼睛和心灵的矛盾

敏感之花 ＊阿明/文

　　错误的开始来自于自己的眼睛。心灵在颤抖“什么时候能带来正确的信息”？眼睛狂笑着对心灵肆无忌惮的肆虐，心灵默默的承受着欺骗。视觉带来的刺激并非真实的，最能利用这一点的就是魔术师，但是眼睛乐此不疲。心灵开了这扇窗口是为了安慰自己还是为了麻醉自己谁也说不清楚，虽然经常是错误的判断。于是闭上双眼用心灵透视世界，得出的是最痛苦的结论，因为太真实！真实的令自己难以忍受。有一个理论“眼睛其实什么都没有看到，所谓看到的一切其实就是心动所产生的”，仔细一想也并不完全的对，比如说一个生来就是双目失明的人心再动，他能看到什么，难道他因为心动就看到了一个桌子？好象不是这样的吧！如果这样的话，肯定有很多的人宁可毁坏自己的眼睛。哦？这好象和上面的论点有点自相矛盾！不过有一点好象是正确的，那就是如果心灵真正能看到本质，就是非常痛苦的事情，因为那将是非常真实的！注意，我说的是“如果”！早上醒来的时候第一件事情就是睁开眼睛，为的就是给心灵这个瞎子领路，虽然看到的一切并不一定就正确或者说存在，总比处于黑暗要强多了，于是，我睁开了狼一样贪婪的眼睛. ..
