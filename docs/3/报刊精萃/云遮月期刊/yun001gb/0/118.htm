乔燕：

　　坦率地讲，我是从"那边"慕名来到你这里的，主要是喜欢阅读你撰写的文章。在几个月前，无意中发现你在那个栏目的一篇文章（致大家一封公开信）立即被清新流畅的笔调所吸引，一种留连往返的感觉油然而生，从那时起，你的名字就隽刻在我的记忆中了，真有一种挥之不去的感觉了。

　　随着时间的推移，相继阅览了你的一些具有思想性、艺术性、可读性及深刻思想内涵的作品，觉得文坛上一颗新星在古都洛阳冉冉升起。当我从其中知道你目前处境后，更加关注你了。特别是你用微笑面对人生旅途上的风雨，尤其是这种自强不息，积极进取乐观向上的人生观，深深地感染了我，在这里我决非是怜悯与同情，只有敬佩，其百折不挠的精神与蕴藏在乔燕身上这种奋发向上令人荡气回肠的民族魂，就是值得我们效仿的楷模。具有这种精神是一个人非常难得宝贵的精神财富，它往往是一个人走向成功的关键，我想这也是大家喜欢你的真谛所在。

　　有点过于严肃了，轻松一下。我希望早日见到乔燕具有自己特色，独巨匠心的个人主页早日问世。另外用中西医结合的方法（譬如针灸、按摩......）早日使自己康复。是医疗事故造成的结果吗？事情即使过去了很多年，现在可以在法院起诉，要求赔偿。据报载，一些事故的赔偿判决都是很高昂的，胜诉后你可以用这比费用医治自己的疾病呀。

　　最后希望乔燕早日康复，回到自己熟悉热爱的岗位上去，早日见到乔燕迎着晨曦，迈着轻松的步履徜徉在洛阳一条条熟悉的街市上，多么希望你《兰色多瑙河》流畅的旋律中展示自己轻盈的舞姿。我想这一天一定会来到的！只要你坚持这种信念，伴随着医疗水平的进步，什么样的疑难病症都可以战胜的。请接受一位远方既熟悉又陌生的准朋友，对你这个清丽多才女孩儿的祝福。

　　乌云散去了，月儿露出了笑脸儿，置身于银色月光下的乔燕，彻底摆脱了疾病的困扰，心情格外的好。她乌黑的秀发和裙角在习习暖风中飞舞着，更平添了这位女孩儿几分成熟的魅力，仰望着圆圆月儿，她清纯的脸上荡漾起了层层幸福的涟漪，久违了的轻松和甜蜜顿时涌上心头.

　　如果有人问我你崇拜的英雄是谁？我可以毫不犹豫地回答在足球场上我喜欢罗马队占神巴蒂思图塔，托帝还有长发飘逸的佛罗伦萨前锋哥麦斯；在乒坛上我喜欢在这个领域叱咤风云，纵横乒坛十几载，横扫欧亚大陆的瑞典名将瓦尔德内尔，观看他的比赛就是艺术上的一种享受。在新闻媒体上，我比较喜欢湖南电视台、东南电视台的节目，因为具有创新与挑战性。现在网络上，我就爱看乔燕的文章，我就是崇拜她，至少她在我的心目中是个英雄。

　　英雄是千差万别的，有的闻名遐迩，也有活跃在万千人海中平凡的英雄，虽然他们与那些大英雄相比，暂时默默无闻，其知名度不高，但是今天的平凡并不意味着明天的平凡，倘若明天依然平凡，还有后天，经过锲而不舍的努力，总有一天她会脱颖而出，鹤立群鸡，宛如一只矫健的雄鹰掀动有力的双翅，自由翱翔于蔚蓝色的天空之上！
