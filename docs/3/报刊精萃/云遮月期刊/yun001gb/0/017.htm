走出网恋的误区

霜儿

　　网恋的美丽之处，便在于它的神秘，正是虚幻使网恋有挡不住的诱惑和吸引力。网络的虚似性，过滤了金钱、容貌、身份、家世等外在的条件。

　　网恋没有国界、地域的限制，给爱情的想像留下了足够的空间，在现实中寻觅不到的东东在网上都能感觉到、找到。网恋就象雨天过后的彩虹，夜空划过的流星，美丽璀璨夺目而短暂！

　　试问谁又能永远抓得住那刹那间的美丽呢！网恋，也许寄托着我们很多的浪漫，完美的幻想，骨子里的渴望。

　　但是它现实吗？我们敢信吗？我们该在它身上寄托点什么呢？是一刹那的浪漫感觉？还是一网情深的真诚？网恋的结局是一场空。

　　在现实中却又是不堪一击的！

　　只要把网络和现实分清！

　　你会享受到它的美好的！其实网恋是一样很美妙的东西。

　　网上情人相爱不用顾虑那么多，也不用急着知道对方的样子，只凭感觉就好了，只凭缘分的安排就好了。

　　"不在乎天长地久，只在乎曾经拥有"。

　　世上的许多东西都是难以猜测它的未来，最重要的是珍惜现在拥有的。

　　因为我们年轻，所以可以做自己喜欢的事情。但做人应该以真心待人。

　　爱过就不要说抱歉，毕竟我们走过这一回。

　　时间有时可以证明一切，但有时又并不代表什么。

　　几十年的相处可以转间就消逝，短短的一刻却又能成为永恒。

　　只要真正相爱过，每次回忆起都是醉人的，难忘的……

　　谁能评说真正的网恋能现实吗？
