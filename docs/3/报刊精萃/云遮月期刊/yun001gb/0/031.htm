中甸游记之（五）碧塔海散行

和菜头/文

　　六月二日，儿童节后一天。在我庆祝完节日后，我在昆明的初中时代同学来到中甸"看望"我。由于他们的休息时间不多，为了看望我，他们的亲人们都或多或少的生了一点小病。想我何德何能，竟然让如此之多的人蒙受这样的无妄之灾，简直是令人感到惭愧。

　　六月三日，晴。我们前往碧塔海游览。我仔细的讲述了前后两个入口的优劣之处，然而没有骑过马的他们坚持要过过马瘾，所以我们就从前口进入了碧塔海。

　　从前口进入碧塔海要走12公里的山路；也可以骑马，需要两个小时。进了公园的大门，就是一道独木桥，桥下有水。河水很凉，明显是从远处高山上流下来的;

　　也很湍急，所以水声很大。站在桥上，风从河谷里吹来，带来远方神秘群山的气息。仿佛天地之间都是经幡飘动之声,和天界的美妙的梵音。

　　在刚进山的那一段路上，我们觉得自己都失陷在了密林深处。山路左盘右旋，看起来好象根本没有尽头。视线又为树木所阻，耳听得流水之声淙淙，却无法看见，心中不由得有些焦躁。加上我的老马对身上0.1吨的重量十分不满，常常往树上蹭，试图把我蹭下去，简直是要了我的老命。

　　终于到了山的垭口，眼前忽然为之一开。只见两山之间一条绿色的河流奔腾而来，那就是传说中的高山草场。草场中有星星点点繁密的野花盛开，有红的、白的、黄的、蓝的和紫色的。其中以黄色和紫色的为最多，而且喜欢攒在一堆，非常的醒目。高原强烈的紫外线辐射之下，美丽的野花竟依然娇艳动人，明丽非凡。我问我们牵马的藏族老阿妈，她们中谁是格桑花，可是她也不知道。我不由得想起了那首民歌的歌词：

　　亲爱的阿妈哟
　　你就象那美丽的格桑花
　　在太阳下你是如此的年轻漂亮

　　我仔细的端详，希望找出一种可能是格桑花的野花，但是我失望了。因为她们都是一样的美丽。而所有人的阿妈也都是一样的动人。

　　在巨大的草场上，有许多的牦牛。它们如同黑色绸缎一般的毛发闪闪发光，鼻子上是木制的鼻环，明显是驯服的家畜。雄性牦牛的犄角很特别，先是往前和往下弯曲生长，到了最后又复向上挑起，看起来似乎全身的力量都集中在尖尖的那一点上。再加上数百公斤的体重，你不禁会对他们肃然起敬。

　　他们静静的在草间休息或者漫步。当你走近他们时，他们并不怕你。你可以和他们对视，这时候你就会发现他们的黑色眼睛竟然是那么的美。黑色的，湿润的，有光泽的大眼睛凝视着你，然而神情间却又是满不在乎的样子。很明显的，他们才是这块土地的主人，你仅仅是一个闯入者，一个参观者。也许你会被他们的傲慢伤害，可是我宁可喜欢这伤害，也不愿意和《与狼共舞》里的凯文.

　　科斯特那一样，用捕猎野牛来显示人类征服自然的伟大。我更喜欢他们如此自由自在，平静从容的生活态度。

　　藏族人有放生的习惯，在每年的大法会上，都有牲畜被放生。喇嘛在牛角上捆绑特殊颜色的布条，这条牛就获得了自由。没有人可以再驱使它干农活，也不会有人敢杀害它。它可以悠闲的活着，一直到老死。据说，有从汉地前往拉萨的人，在路过唐古拉山的时候，偶遇牛群。其中的一只径直向他走了过来，就象是认出了他一样，舔他的手，往他的怀里钻，当它抬头看他时，两眼溢满了泪水。于是藏族人告诉他说那牛与他前世有缘，叫他买了放生。由此看来，牦牛还是很有灵性的动物呢。

　　走到了草场的中部，是一个很小的驿站，名字叫做岗擦坝。驿站门口有一根根的木桩，用来栓马。这里没有食物卖，仅有新鲜的牦牛奶可以喝。我们叫了七碗，也给了牵马的藏族阿妈和小妹妹们。我和藏族阿妈坐在廊檐下，喝着可口的牦牛奶，看着我的朋友们在草场上打滚、照相，一切都是那么美好。

　　接近中午的时候，我们继续赶路。太阳很毒，每个人都昏昏欲睡。走完了草场，我们渐渐接近下一个垭口。阿妈说过了这个垭口就是碧塔海了。听到这话，大家一下精神又来了。当我们刚刚接近垭口，一片杜鹃林突然迎面杀到！以前看过的杜鹃都是种在花盆里的，而这样的木本花卉是我第一次见到。杜鹃树高大约有两层楼那么高，树上是淡粉红色的花朵。花很大，几乎有玉兰花的大小。微风吹来，满树的杜鹃摇曳生姿，花瓣纷纷坠落尘埃，花落如雨，落英缤纷！你骑在马上，看着这奇异的景象，会误以为人间就是仙境，桃源就在前方。

　　转过垭口，一路下坡，不一会儿就到了山脚。沿着木头的道路往前走，在一个坡顶，我们看见了一方简单的石碑，刻着碧塔海几个字。从坡顶往远处眺望，群山之间就是著名的碧塔海。这里的人很少，清凉的风从湖面吹来，人躺在柔软的草地上，仰望蓝天白云，简直就是人生最大的享受。

　　远离了城市的喧嚣，你在这湖光山色之间流连，根本不会觉得时光的飞逝。多么希望就在这儿搭一间小木屋，从此告别文明世界，做一个闲人。每夜清风明月，醉卧长草之中，看漫天流星如雨，直到世界末日来临。
