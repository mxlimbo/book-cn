安静的时刻——写给乔燕

张卫宏

安静的时刻你渴求温暖
你的眼睛悄悄变蓝
你的钟表忽然倒转向凌晨三点
它想起一个久别的人
一个似曾相识的夏天

安静的时刻你的母亲消失在远方
她默许一只蜜蜂飞入你的房间
这仅有的一点香甜你难于品尝
你难于在这小小的生命面前陷入绝望

安静的时刻一群麻雀飞过你的窗前
它们与冬天相依为命
饥饿是它们的父母
总不让它们飞得更高，更远

安静的时刻整座城市陷入剧烈的摇撼
我走进旷野，像一个词
落在《自然》这本大书的封面
它有多么幸福就有多少孤单

安静的时刻本来无需召唤
可是你已经开始歌唱
我怎能拒绝倾听
