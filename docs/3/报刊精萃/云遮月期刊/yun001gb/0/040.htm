冰原·剑客·恋曲或……

——给鲛
海边小屋

　　题记：诗非诗，文非文，痴情应笑谁？

　　雪花飘，带走了多少思恋……

　　雪中情，雪中情。

　　冰儿从冰屋里走出，门口是冰山。

　　古有藐姑射之山，神人居焉，肌肤如冰雪，绰约如处子……

　　冰儿不是神人，冰儿还算一个美人。

　　冰儿不餐风饮露，冰儿喝冰水，烹冰鱼，寝冰床，划冰船……

　　冰儿悬空眠于小龙女的细绳上，冰儿冰清玉洁、冰雪聪明。

　　冰儿有点冷。

　　一个剑客要寻找冰儿，一条无边的遥途，一把生锈的铁剑，一双草鞋，一个旅行箱，一本中世纪的骑士小说。从南方的热带丛林开始，他一直向北走，他袒胸澡严寒，衣不蔽体游荡在冰与雪之间。

　　寻找冰儿，甘愿被缚成大粽子，飘流在历史与现实之间。

　　寻找冰儿，即使成为最后一个匈奴，最后一个寻找草原和成吉思汗的蒙古武士。

　　金帐汗国，无尽的荣耀，雪中的豪情。

　　但愿不是，悲情莫斯科。

　　试剑石，遥古的感动；精卫填海，愚公移山，望帝春心托杜鹃。剑客已经来到第三百一十九道难关。雄关漫漫真如铁，而今迈步从头越。冰儿在荒原，他在寻找生命之杯。

　　谁注定是冠军，谁能亲吻大力神杯，亲吻冰儿冰冷的额头，在雪花纷纷的草原上。

　　他在寻找生命之杯，新世纪的又一片荒原及其英雄的主宰者。生命之杯，神灵铭刻的咒语："快乐，因你的行动而存在。"

　　然而冰儿不信，冰儿始终像一座静静的冰雕，美，美得令人心疼。

　　剑客疲倦的双手依然紧握着铁剑。

　　寻找冰儿，说出来谁也不相信，包括他自己。我爱，我欲，我为，我愿……他已经将一切交给魔鬼。

　　集市上没有超人，只有耍猴的老人。打扮得花枝招展的小猴已经进化成妖精。正如所有的人都已经退化成规律的奴隶一样。在规律中沉浮，谁能跃出三界之外。

　　把吴钩看了，栏杆拍遍，无人会，登临意。

　　青山遮不住，毕竟东流去……

　　他在不知疲倦地突破，他不知痛苦地从地上爬起，然后又不知方向地将球射向球门周围1米多宽的地方。

　　剑客既用手，也用脚。

　　剑客最终要靠自己的大脑。

　　剑客也会凭烛阅春秋，但他注定不能成为春秋五霸。

　　长途汽车上的客人有得选择吗？没有。没有厕所，没有空间，只有一种麻木的忍耐。

　　剑客只能浮起齐秦式的酷笑，剑客想重回北方的狼群。

　　冰儿在冰屋，编织着冰一样透明幽冷的故事。

　　冰儿在风中，风将她的故事吹远，冰儿来不及存盘。

　　剑客充满自嘲，对月惨笑。

　　剑客的剑喷着青芒奋力击向某位玄衣的蒙面大盗，于是夜色更加朦胧。

　　叶儿纷纷退场了，空荡荡的剧院。

　　最后一个主角抓着谁的衣带和余香，痴痴地定格成谁眼中的风景。

　　冰儿不会来，荒原狼的呼唤也没用。

　　冰儿不会来，太冷了，风太大，你要保重身体。

　　冰儿不会来，雪下吧，让我们的回忆堆叠起我们的金字塔，让后人破译我们的司芬克斯之谜。

　　冰儿在家中，冰儿在冰雪中。

　　冰儿幸福吗？生命之杯在保险箱。

　　一声叹息，世纪末的最后一部大片。

　　搂着你的名字取暖，蜷缩在一个不为人知的破庙。这是夜，深的夜，长而无边的夜，谁也不相信，还有日出的可能。

　　小楼昨夜又东风。

　　昨夜，她梦见一个橘子……
