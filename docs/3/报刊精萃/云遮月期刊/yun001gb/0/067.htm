致乔燕

张卫宏

(我能给你的是河水，不是岸，我看见它把你带出深渊)

你已经回来
而我坠入深渊
在我面前你优雅地端坐
而我竟然对近在眼前的深渊视而不见

你在飞翔
可你的双脚固执地踩疼大地的神经
大地痛苦的声音被你模仿
却在天空变成歌唱
而那里没有人
那里有的只是宇宙的运转和大自然的呼吸

你在哪里
我一开始朗诵
这深渊便显露出深不可测的晕眩
我如何相信端坐于面前的这个人
不是在倾听
而是在深渊里翱翔


小记：

乔燕：

　　不要为痛苦而痛苦，不要为迷失而迷失！写作是一个连续而完整的过程，在你经受痛苦的时候，创作可以塑造你的痛苦，使它变成文字，接近人类普遍而永恒的精神家园。塑造痛苦的时刻就是塑造灵魂的时刻，而灵魂可以带来发现——发现痛苦的必然、发现痛苦的根源、发现自己其实不是自己，痛苦不只属于某一个人。痛苦可以中断生活，但不能取消生活，谁能说某位圣贤，某个伟人的痛苦能够毁灭生活，伟大的人之所以伟大和不朽，只是因为他有一颗会说话的灵魂他向这个痛苦的世界说放，说出世界的希望所在。
