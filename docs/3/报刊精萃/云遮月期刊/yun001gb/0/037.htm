刹时花开

和菜头/文

　　是什么时候和她初一相逢，我已经完全模糊。我看着她的笑脸，仿佛今生之前的生生世世里，就早已反复端详。又如流水刻蚀下的崖画，因年代久远而并不真切。只是在她目光流转的那一瞬间，我越过无数光年的旅途，停顿了思维，屏住了呼吸，世界在那一刹那，绽放漫天的鲜花。

　　她已经完全忘记了啊！我曾经是在那个沾满露珠的清晨，把一束白色的栀子花插在她衣襟上的那个青年；我也曾经是在那残阳如血，倒在沙场上，口中喃喃低语她的名字，渐渐死去的战士；或者，我们曾经是在山路上偶尔相逢的旅人，她微笑着走过，我在她如同水神一般的风华里，目眩神迷，天地昏黄，心如鹿撞。

　　就是因为那一刹那的擦肩而过，我决定挽留她的微笑，她的凝眸。我想她是无边无尽的黑夜，轻轻的就将我完全笼罩。沉浸在她的芳香里，我遗忘了很多很多世纪。她并不知道，为了今天这一刻的重逢，我是如何的挣扎在生命的洪流中，奋力向她游去，为的是在在这一生里遇见她。

　　甚至是不需要她对我来说承诺什么啊！甚至不需要她认出我来，我只想让她看看我摊开来的手掌，请她沿着我手心的年轮缓缓上溯，看见这些年代久远的故事。而她只是在笑，只是在轻轻摇头，手指着一片白桦林给我看，那些岁月留下的创痕，那些无数悲伤的眼睛。

　　我拥抱的手臂在瞬间变成了枝叶交通的花丛，我绝望的吻盛开在她经过的路上，我的泪水迷蔓在空气中。她走在我的怀抱里，身边是我盛放的花朵。
