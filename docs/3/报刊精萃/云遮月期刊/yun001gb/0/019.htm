久违了，这里的一切

艳子

　　昨晚下雨，细细的雨丝夹杂着烦烦乱乱的心情，那种不高不低不好也不坏真有点说不出的感觉，活着的不堪重负挺累人。老妈打电话来说，我没有带伞，要不要接？

　　当然不要。

　　一个人多好。一个人走了一会儿，突然想起一个朋友说，现在太乱。算了，早点回去吧，一大早还上班呢！

　　我回到了温暖的房子里，想起了好久没有上网，想起了那么几个网友以及曾经遭遇的一切……

　　我用最快的速度打开了计算机，因为太晚，网上几乎没有了熟悉的一切。

　　中午吃完饭，我急匆匆地来到了老马的主页，果然有几个熟悉的朋友，还挺热闹呢！嘻嘻……看了一下ＢＢＳ和乔燕的贴子，那种真实有点让人彻心彻肺……
　　久违了，这里的一切，熟悉和不熟悉的朋友们，因为你们，这里的空间那样精彩！

　　一样的春来秋去，不一样的是每一季的心情。

　　快乐原本真实贴切，所以它那样流畅，不含杂质，因此，我们庆幸！
