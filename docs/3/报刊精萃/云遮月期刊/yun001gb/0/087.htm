渴望艳阳天

路遥/文

　　我等待用我的真诚，因为我只有这个了。我等待的是你我的将来。相思总是缘于一种悠远的寂寞，在雨夜里（我这里现在下着雨）思念远方的人以及往日种种情怀一同复活了起来

　　总想让一些如水般的语言，淌成一条爱意融融的河，流向你在的那方天空。我知道，我爱之所系在水之湄，情感的指向，注定我要在源头作长久的伫望。梦中的你，在眺望的下游，在情感河流的那边。 也许可以重新将段段往事，牢牢捆扎成筏, 在结束一段迷惘的眺望之后，沐着此时的雨雾， 破釜沉舟般乘槎而来，在某个夜晚闯入你长夜孤灯的世界。今夜的雨叉让我找回了失落的往事，让我和已离去的点滴近了些，我知道此时在河的那边依然有你遥远的关切，我想，雨过之后，应该有一个月圆的日子了吧？今天的你我不再，昨日的风雨依旧——


乔燕/文


　　是这样的——"我等待用我的真诚，因为我只有这个了"。

　　我等待的是你我的将来。相思总是缘于一种悠远的寂寞，在雨夜里（我这里现在下着雨）思念远方的人以及往日种种情怀一同复活了起来。"

　　雨滴会使相思更浓更纯净，这悠远的情愫徐徐而来，太多的心事便会泛上心头！

　　听听雨声，想想自已的现在，以及那飘眇的未来，是的，我就是在自已水一样的寂寞里期待！
