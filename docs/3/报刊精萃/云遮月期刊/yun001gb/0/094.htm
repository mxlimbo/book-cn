仍能唱出的最爱

主持/乔燕

地点：云遮月客厅
主持：乔燕


　　总觉得音乐是有魔力的。

　　记得还在荥阳车站时，每逢休息便坐着火车，晃晃悠悠在来往于洛阳到郑州的这一段铁路线上。除了偶尔坐慢车人少些，平时车内总是很拥挤。冬天，车窗紧闭，吸着各种版子的香烟的男人们和妖艳的红唇女子吐出的烟在车厢内弥久不散。下了车身上便带上了烟的印记，久久不去。这些味道追随着我，钻进大衣，毛衣，手套，也钻进我随身听所释放出的音乐里。

　　在火车上听音乐的感觉也象这烟所呈现出的弥久不散，不过这不散的是心底那一丝温暖的迷离。记得那时最爱听粤语歌，觉得那样的发音最能传达出心底迷惑又新鲜的情绪。

　　反复播放的《千千阙歌》是至今仍能唱出的最爱。

　　来日纵是千千阙歌，飘于远方我路上，
　　来日纵是千千晚星亮过今晚这月亮，
　　都比不起这宵美丽，亦绝不可使我更欣赏，
　　啊哈……
　　留下今晚共我唱！
　　来日纵是千千阙歌……

　　岁月悠悠，许多往事已在风中模糊了容颜，淡淡地散于水流般的琐碎日子，唯有这音乐保存了最初的感觉，点点滴滴地提醒着那曾经逝去的岁月。


垭口：
　　那首歌里我最欣赏的一句：
　　当某天雨点轻敲你窗，当风声吹乱你构想，
　　可否抽空想这张旧模样"
　　不怀旧了，我走～～～～～～

booking：
　　我喜欢听歌，可是我又怕听歌……
　　唉……
　　这种感觉……难受！

愤怒的葡萄：

　　曾经是我最喜欢的歌
　　曾经让我魂牵梦绕的歌
　　……
　　我的心像软的沙滩
　　留着步履凌乱
　　偶尔有些悲欢
　　总是去而复返
　　人越成长 彼此想了解似乎越难
　　人太敏感 活得虽丰富却烦乱


小乞丐：

　　我始终不变的最爱！


俏儿:

　　在这里，我有一个原则，不认识武汉的朋友，不和网友见面。不过后来我还是见了一个朋友，因为他经常会来我们这个城市，见面的机会，实在很多。

　　那次，他要离开我们这个城市时，我们一起去了卡拉OK。他唱歌很好听，极有水准，应该是那种容易让女孩子动心的声音。

　　那天，我唱的最后一首歌，就是这首——千千阙歌。

　　我想，这歌，和当时的气氛是极贴切的。

　　来日纵是千千阙歌，飘于远方我路上
　　来日纵是千千晚星，亮过今晚月亮
　　都比不起这宵美丽，亦绝不可使我更欣赏
　　因今宵的我可共你唱
　　因不知哪天再共你唱

　　然后，我一直没再见他......


加菲：

　　喜欢听歌，不知从什么时候，喜欢在夜里听歌，那绝对是另一种滋味，难以言表，是淡淡的忧伤？些许的凄凉？无奈的离愁？我也说不上来，或许是情绪在做怪吧，那些欢快，激昂的曲调，也会变得怪怪的。不知你们是否有同感 ？

　　现在就在听，是赵咏华的《最浪漫的事》，词写的真好，当青春不在，年华老去，还可以和最爱的人在一起想想经过的日子，聊聊往事，还真幸福。


worstboy：

　　昨夜难以入睡，往事在脑海中放电影，一遍又一遍，哎，今天又睡不着了。

　　眼睛呆滞的看着天花板上，忽见一片月光不知从何时潜入室内，伸出右手，月光照到我的手心上，皎洁而又令人感到恬静，犹如无声的小提琴在演奏。

　　难得有这么明朗的月光，也难得自己今天这么仔细的感受月光的美好。

　　突然一片黑暗，室内一片黑漆漆，"月光"从我的手心溜走了，往窗外一望，原来只不过是对面楼房的阳台上的电灯熄灭了。刚才自己所认为的美好的月光原来只是灯光而已。

　　一刹那间，感悟到一个哲理：重要的不是表面上的东西，而是自己心的感觉，正从心里发出的那种...


如云：

　　那天去一个朋友家中，当原本平静的背景音乐换成一张拥有老歌的碟子时，所有的人都静了下来，那一时刻，大家都在伶听着，思绪象被久远的故事深深吸引着。


乔燕：

　　真没想到在记忆深处久久地侵占我的这首歌也是这么多人的至爱！

　　加菲，那首《最浪漫的事》我也是非常喜欢，悠悠的旋律，真挚动人的歌词，是的，对我来说最浪漫与珍贵的人生之事莫过于此，相信也是大家心底的期盼！

　　worstboy，在远方多多保重！

　　哑口，你说的那句是国语版里的吗？这首歌似乎粤语的更有味道。

　　俏儿，也许来晶方长的一天里面，有一天我们会合唱！

　　如云，好久没有好好叙叙，看到你的感触，温馨浮上心头!
