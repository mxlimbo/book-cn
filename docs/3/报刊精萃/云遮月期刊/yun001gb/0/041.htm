秋天1944

肥眼泡鱼

　　什么都不是 / 我们什么都不是 / 只是被遗忘在世界的一个角落 / 要爱 / 只能够向天乞求 / 不论是年代 / 为什么伤害 / 人性随手可卖 / 随手可买

　　你希望我陪你 / 回到那一年的上海 / 风不断的吹起 / 你眼里的怜爱 / 我看着我爱人 / 仿佛看着更爱的人 / 提一盏风灯 / 她从少女模样变成妇人

　　我握着你的手回到那一年的上海风不断的吹起却吹不断伤害我看着我爱人心疼我们更爱的人留一盏风灯仿佛看见你流着眼泪演唱：黄大炜 。

　　郎骑竹马来，绕订开表格。书上说这是青梅竹马。欢颜一直遗憾青梅竹马只游戏在过去，她从六岁开始失去了拥有青梅竹马的机会。没有人骑竹马，没有开青梅，需要好好读书的孩子不可能拥有青梅竹马。

　　读书时候的欢颜喜欢画美女，常常在老师唾沫横飞时在课本的某一角落画上一个长脖子细腰满头珠宝的古代女人，画得多了欢颜老是觉得自己生错了年代，她应该是一个长在深闺绣花扑蝶荡秋千的大家闺秀，在个宝哥哥一般的公子骑着竹马潇洒而来，绕床弄青梅。

　　欢颜小学三年级时已经读完了《红楼梦》，当知秋第一次出现时，她眼前一亮，那一刻明白了宝玉对黛玉的感觉――这个妹妹我认得。

　　知秋那年十三岁，比欢颜大四岁，比所有的同学都有大三四岁。知秋出现以后，欢颜总是想在笔下的弱不禁风的闺秀身边现上一个俊秀佳公子，可惜画惯了女人的她画的才子佳人怎么看怎么像一对姐妹花。欢颜不管这个，倔强地继续画，常常画着画着，就盼着长大。

　　十三岁的知秋丰朗俊秀。爱美之心人皆有之，九岁的欢颜于是很喜欢跟随他讲话等到她喜欢偷看他时，已经十三岁了，读初一。

　　再大一些后，开始对小纸条有着特殊的热情，其实没什么事，无非是交流一个小笑话，或是一句打趣的话。就是因为这个，偏执的班主任为了保护欢颜这个好苗苗，坚持不让知秋升学，知秋接受不了，于是就辍学了。见不到知秋的欢颜迅速地成就，倔强地沉默。

　　欢颜考上大学那年的暑假，知秋去看她，当时欢颜已经有了一个感情稳定、如胶似漆的男朋友，而知秋也在那次见面告诉了她他已经订婚了。农村的孩子结婚都很早，知秋已经算是晚了。

　　欢颜继续经营她那日渐苍白的爱情。爱情曾经热烈过，如今只可能是细水长流。无意中说起初恋，她取笑自己的爱人当初稚嫩的初恋，关于自己，始终只字不提，她的初恋是单纯的爱情想象占据着所有情窦初开的日子，以至于若干年后，欢颜还清楚的记得知秋第一次经过她身边的样子，骑着一辆自行车，撑下一只脚，回头说，来，我带你一程，她忙不迭的拒绝。其实是当时她不知道该怎样座上自行车的后座，自行车在那年头是稀罕东西，她只能拒绝，眼睁睁地看着他的背影远去。她对于初恋的自卑使她牢牢守住它，不让任何人有接近的可能。

　　后来听说知秋结婚了。后来又听说他做了父亲。只是一年半载而已。

　　再见知秋，欢颜已经大学毕业了，而知秋已是一个纯粹的农村汉子，甩着手走在一个抱着孩子的少妇身边。欢颜实在看不出当初那个总是服饰一丝不苟、清新挺拔的知秋。她睁大眼睛，他却避开了视线。

　　欢颜坐在地板上一大堆知秋以前给她的纸箱上，信手拈起一两张来看，看着看着忽然想哭。知秋就在这时敲开了门。慌手慌脚的收拾，一张照片硬是飘了出来，照片上青春的知秋神采飞扬，靠着一棵树，冲着农村汉子的知秋微笑。欢颜捧着一大堆纸条愣在了门口。

　　走的时候，欢颜偎在男友身边看着钟爱的音乐节目，一位久久没唱歌的歌手突然唱起了关于秋天、关于古老秋天的歌，"我看着我爱人，心疼我们更爱的人"，她不由自主偎紧一些。那一刻，知秋也许正在逗弄着走路蹒跚奶声奶气唤爸爸的女儿，幸福着他的幸福；欢颜只记得在她长长的学生生涯的最初，有个叫知秋的男孩子，他不是女人的丈夫孩子的父亲，他只是他，骑着自行车经过她的身边对她说："来，我带你一段路！"
