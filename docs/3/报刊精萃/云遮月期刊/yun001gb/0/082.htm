小城之夜

海边小屋/文

　　小小的港城，临海的小屋，平静的港湾，呼呼的北风。初冬的小港城，没有星星的夜晚。

　　小城的灯一如既往的昏暗而软弱，影子布满所有的街巷。网吧突然走出一个长发男孩，满脸茫然，一天的网事如风，明天的路像风中的落叶。

　　一个三十岁的男孩，青春痘早就随深陷的眼眶一样沉入岁月的黑洞，越来越看不清的前程，越来越模糊的世间万物。从一条街走到一条巷，从一扇门走进另一扇门，从一个故事进入另一个故事，从一个梦跌进另一个梦……

　　有时求索的过程就像屈原的心结：我爱你，你为什么不爱我？我为你付出一切，你却永远不会给我一个微笑。但我依然无悔，因为我的使命铸造了我的悲剧。一生于是定格于一个至美的意象——静静飘流在历史长河中的大棕子。

　　小城的灯是沉默的，黑色的海是沉默的，走在空荡荡的街上的人也是沉默的，只有风永远不甘寂寞地向永恒的宇宙空间诉说着什么。

　　

发现与表达

乔燕/文

　　我的梦在夜里穿越时空的阻隔与小城相握……
　　一个三十岁的男孩，青春痘早就随深陷的眼眶一样沉入岁月的黑洞，越来越看不清的前程，越来越模糊的世间万物。从一条街走到一条巷，从一扇门走进另一扇门，从一个故事进入另一个故事，从一个梦跌进另一个梦……
　　诗意的发现与表达更显夜的深渊，我们就在这边缘行走，不时地在风中回头，纪念那紧紧跟随着，从来就不愿也没打算要离开的寂寞。
　　寂寞在月光里碎了一地，将那个深夜未眠的男子包围，青春痘已随年华远去了吗？还是汇成岁月里一条更汹涌的暗河，水面已然光滑如镜，而生活的潜层里，你听到哪一尾小鱼轻轻地叹息……？
　　你寂寞的叙说给寂寞的听者盈满了诗意，这是一个诗人在一个夜里最可能的远行！
