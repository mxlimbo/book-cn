红颜

和菜头/文

　　在匆匆的岁月的流光里，我还来不及叹息，你的容颜就已经在风中破碎，无声的跌落尘埃。我曾经无数次幻想我们在路上迎面相遇，无数次幻想那一瞬你的风华，我们的无语，以及比一个世纪还要长的擦肩而过。或者是在汹涌的人潮中，我大声的呼唤你的名字，可是我们的路标指向不同的地方。你仿佛充耳不闻，我挣扎在我的队伍里，狂乱而绝望，被卷向远方的车站。甚至是我已经死去了，隔着玻璃窗，看你在灯下沉思。每一句我说出口的话，瞬间被凝固成了雨滴。你抬起头，惊奇地看到夜雨如此狂野，撞击在你的窗上，无助的滑落。那是我想向你倾诉可无法说出口的话语，我曾经埋葬在了岁月里的泪水。

　　如果要让我重新在秋天第一次遇见你。我希望上天把我变成一个又盲又聋的人，我宁可看不见你的灿烂夺目的笑容，听不见你如同羽毛一样轻柔的声音。在你的微笑里，我曾经真的看见天堂。但是，我又如何能够不会再一次迷醉在你的气息，你衣袖间的清香，你如同月华一样的沉静里？就算是在生生世世的流转中，在千人万人漫天烟花的狂欢之夜里，我还是可以一眼认出你，脱口而出的还是你的名字。

　　我知道，我不能给你你要的生活。我永远是那条奔腾咆哮的河流，不羁的猛兽，奔跑的旅人。我甚至不能给你哪怕一个宁静的下午，你失望的眼睛里满都是我的远去的背影。我注定一生不停的奔跑，流浪，远行，寻找。因为我不知道我倒底要到什么地方去，我倒底要的是什么？我害怕黑暗，惧怕停留，我必须一次次越过黑夜和白天的界线，追逐明天，直到永恒的黑夜将我的灵魂收留，给我永无梦境相扰的安睡。

　　你终于绝望，你终于厌倦，你终于放弃，你决定离去。我开始思念，我开始失眠，我开始痛苦，我还是再一次上路。在以后的日子里，我见过和你一样的笑容，听过和你一样的声音，甚至是和你一模一样的拥抱。可是，没有谁能和你一样灿烂，一样甜美，一样使离别变得痛彻心扉。每一个深夜，我一次次陷入回忆里你如同海一样的目光深处，你如同山谷里的风声一样叹息的尽头。每一个清晨，我象个不愿意上学的孩子一样告别黑夜，一个一个时区的计算你的时间，用最伤感的声音祝你晚安。

　　在某个时间，某个地点，有个人会遇见你，他会给你你要的东西。你会爱上他，对他打开你的心扉。可是，我知道，你的心是世界上最复杂最精致的锁，而我有那唯一的一把钥匙。我会去参加你的婚礼，他们说："每一个新娘都是最美丽的。"只是，你的美丽不再为我盛开。你把你的幸福，你的未来，你的梦想都交在了另一个人的手里。我会穿上最好的礼服，戴上最真诚的微笑，送你我最真诚的祝福，再在众人的欢呼和掌声中悄悄悄离去。

　　沉浸在幸福中的你，会不会知道，在一条通往遥远的大路上，有一个孤独的旅人。他形容憔悴，他步履艰难，但是，他的脸上有着谁也不明白的微笑。夕阳下，他的项间有一把形状奇怪的钥匙，在闪闪发光。
