鱼和飞鸟

主持/千子晨

地点：云遮月客厅
主持：千子晨

千子晨：

　　今天心情不是太好，得知朋友的哥哥不久前离婚了，他们的婚姻纠缠到最后终于还是以结束而告终。但当初的他们也经历了那深切和炙热的情感阶段！他们为了感情摆脱阻力而不惜抛弃了很多事物的勇气曾为我们这群小女孩津津乐道。可是，婚后只是短短两年时间，他们的婚姻之船便搁浅了。

　　那个男孩碰到了另一个女孩。一个很有气质和品位的女孩。我想，也许，在这个男孩和他妻子结婚时并没有想到自己会遇到一个更好的女孩吧。。。可是，如果仅仅这样感情就要发生变化，人生可以预见的事情又有多少呢？我们可以肯定和值得信任的感情又会有多少呢？我不知道。

　　今天偶尔看到了腾翼的《鱼和飞鸟》的帖子，我回过帖后，突然又想，假如一切感情都逃不过时间的安排的话，也许鱼和飞鸟无法走到一起的结果是最好的结果吧。。。

乔燕：
　　千子晨，我很理解你此刻的心境！和朋友在谈到爱情婚姻时也听他们说：哎，现在不太敢相信什么了，未知的事太难预料，在这个多变的时代，太多的事会偶然的情况下发生。

　　可是，尽管如此，我们内心谁不是仍在期望一份真正的能够恒久的情感，不在渴望一个属于自已的家呢？在落寞时有人来安慰，在晚归时有人在一盏灯旁为自己守候，这一切在现今社会变得如此难得与珍贵，这到底是为什么啊？

　　我们的身边有人走到一起了，有在一起的开始分离，一切都在变化，我们的渴望变了吗？我们能够幸福的条件变了吗？没有——也许这正是现实生活的悲哀！
　因而真心的希望正在拥有的朋友珍惜为你而跳的那颗心，不要奢求它的完美；也衷心地祝愿正在寻觅的人能如愿以偿！

　　我们到底该如何来爱呢？


愤怒的葡萄:

　　你的心情不好不会也和感情有关吧？


路遥:

　　千子晨！

　　缘分是偶然和必然之间的产物！

　　当它到你的身旁时，就象水和天连在一起的时候，那时鱼就会和飞鸟在一起了......


乔燕:

　　鸟与飞鱼也许永远有距离，也许他们永远因此而彼此向往！
