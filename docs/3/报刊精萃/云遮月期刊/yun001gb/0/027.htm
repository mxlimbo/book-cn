建议取缔股评家

雨来/文

　　中国股市历经十年，可谓不容易。中国股民队伍的壮大，也让人从一个侧面感受到经济生活及其原则对人类社会的重要性和必要性。

　　可是在这样的经济生活里，一方面，国家鼓励更多的人参与股市；另一方面，不协调，不合法的环节还很多。"基金黑幕"是最近的新闻，许多人在感到气愤之外，并没有多少的震惊。因为，这与大家看到的，感受到的情况，大抵一致，稍有些想象力的，便可猜个八九不离十。只是面对那些庞大的机构或利益集团，个人的声音是如何地微弱！

　　而股评家，无疑是普通民众讨伐的对象。他们的利益，和哪些人相关？这在多数人的眼里，越来越成为共识。

　　总体而言，面对规模尚小，制度尚不完善，中小投资者尚不够成熟和缺乏经验的中国股市，机构资金雄厚，大手遮天。整个股市氛围，投机成分浓厚；少数利益集团，便可操纵一只股票、一个版块、甚而大盘的趋势。在这样优势占尽的情况下，一些机构并不满足，利用政策、法规的漏洞，和某些政府机关的分子、媒体及其股评家联手牟取众多散户的利益。许多所谓的股评家，在其中充当了不光彩的角色，这样的情况，也越来越成为共识。

　　有文章说，股评家也要混饭吃，他们往往在夹缝里求生存。说得很好。股评家其实成了一个特殊的职业，他们往往和媒体互为补充，替庄家散布假消息，传达假意图。这当然有难度，因为必须懂得和熟练掌握虚实之道，一味说假话没有人信，必得加上些真话才行。但说真话点破庄家关窍也不行，不是让庄家反做，便是连饭都混不了了。于是股评家即便要让散户受些利益，也总是在庄家收集完筹码开始拉升时大喊散户跟进，而绝不会"不明智"地在此前就大喊。这样他既帮助庄家找到了抬轿的人，又象征性地帮散户赚上些风险更多的小钱。

　　再从信息传播的先后上分析，股评家的评论比起庄家的动作，总是滞后的。作为一个特殊的面，姑且叫他"股评家面"，其实也很好地成为庄家采取下一步行动的参考。每天下午收盘到晚上，各种股评一出来，庄家必在第二天早九点半以前据此想好当天的策略，是打压进货，还是拉高进货？是反技术分析面操作，还是利用股评家造成的影响，将计就计，制造虚假突破？散户如果盲目迷信股评家，在优劣之势极其显著的情况下，其结果势必屡屡受制，平白损失钱财。

　　当然，也有智慧高超，判断力很强的散户，他们看股评家的评论，犹如当年国统区里看当局的报纸，能从"朱毛被擒"，"全歼共匪三千余人"的消息里，看出反面文章，看出真相所在。可是试问天下，精于此道的，又有多少？

　　所以，我个人认为，股评家是中国股市发展中一个特殊的产物，甚而可以说畸形的产物。它最初的存在，有其合理性和特殊的历史原因——中国股民素质的确普遍偏低，但随着股市健康发展的需要日益迫切，股评家应该尽早退出历史舞台。

　　国家在加紧出台各种法律、法规规范还很不够完善的股市的同时，要努力培养两类投资者。一是培养更多的机构投资者，扩大股市规模，使股市力量日趋多元、治衡的状态，而不再是少数利益集团便可操纵行情。二是培养更多的理智型、知识型中小投资者。普及股票知识、证券知识，和其他相关的经济知识，以及增加他们的手段，例如上网培训等等。使参与股市的人群，更多独立分析的能力，而不是盲目地跟风，从而树立他们长期投资股市的信心。

　　股市，是经济生活中极为重要的组成部分。它使得更多的人关心国家大事，将自身利益和更广泛人群的利益联系在一起。这是一个没有股市存在的社会所不可能做到的事情。因此，我们真心地祝愿，中国股市日趋健康和完善，中国经济日益强大。
