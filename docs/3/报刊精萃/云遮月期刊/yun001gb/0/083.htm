明天不会下雪

苦　恋

加菲/文

　　他依然清晰的记得，春天里，她穿着那身牛仔服在暖风中嬉戏的样子，已经过去好多年了，他一直记得。

　　她已经结婚两年了，他还是一个人，曾经有许多人给他介绍对象，可他每次去见面都会想起那双大大的眼睛，于是他又精神恍惚，痴痴的了，不觉中她的样子就成了比较，结果都是不欢而散。慢慢的再没人给他介绍了，于是他一直单身。

　　他们住的城市很小，他俩一个月可以碰见好几回，相互都很尴尬，他想：她或许还是喜欢我吧。于是拼命想从她的眼睛里看到什么，结果相互更尴尬了，就敷衍几句，匆匆告别。他就越发不甘，越想见她，而她就有意无意的躲着他。

　　他就越发的恍惚，这种时候他一般去喝酒，其实他不能喝的，最多也就两瓶啤酒，喝得醺醺的，就去唱歌，唱《让我一次爱个够》，唱《爱的代价》，唱《灰姑娘》，他最喜欢的是那首《鬼迷心窍》：

　　斩了千次的情丝却断不了，百转千折它将我围绕，有人问我你究竟是哪里好，这么多年我还忘不了，春风最美也比不上你的笑，没见过你的人不会明了，是鬼迷了心窍也好，是前世的姻缘也好，然而这一切已不再重要，如果你能重回我怀抱，虽然岁月总是匆匆的催人老，虽然情爱总是让人烦恼，虽然未来如何不能知晓，现在说再见会不会太早。

　　去酒吧，歌厅，但大多数是在大街上，声嘶力竭的唱，声嘶力竭的吐，吐得胃和大脑一塌糊涂再回去。

　　他变得很爱看书，那种相思的，《苦恋》，《半生缘》什么的，有时喝多了，在半梦半醒间也填几句词，很凄凉的那种。然后细细的撕掉。

　　他竭力想回忆起两个人在一起时的情景，可印象最深的就是一次下大雪，她摔倒了，他就扶她起来，一直挽着她的腰，在纷纷的雪里走了很久，于是他就开始喜欢下雪，越大越好。

　　有时很晚了，他还是睡不着，就开始胡思乱想，朦胧中又看到那秋水般晶莹的眼睛，于是，他的心酸酸的，他知道，他已经陷的很深了。

　　可他还是想着：明天会不会下雪呀。

 

明天不会下雪

笨笨0626/文

　　剪不断，理还乱，此情此景谁明了？

　　慢兄，其实你的心境我真的很理解。我也想讲一个"他的故事"。他也有过你那种无奈、彷徨、那种借酒消愁的经历。他和她是同班同学，他那时很调皮，三天两头逃学、旷课，学习很差，时不时还传出把谁谁又打伤了的新闻。而她却不仅成绩优秀，而且是班干部，母亲又是学校教师。所以深得老师的宠爱。那时我们班两极分化，学习好的坐在前边，差等生坐在后面。（直到现在还是不理解某些灵魂工程师的苦心）坐前面的视坐后面的为洪水猛兽，避之唯恐不及。但她率先打破了这种格局，和他建立了友谊。在学习上、生活上关怀他、照顾他。随着时日的增多，他对她的感觉也由抗拒到敬佩再转为爱慕。（虽然是早恋，尽管是帕拉图式的，但却刻骨铭心。）这样的心情维持了一年多，直到突然有一天，他收到一封信，在信中她说她爱上了一位体育老师。在那一刻，他觉得天好象塌了下来，眼前一片混沌。

　　他其实属于"先天"酒精过敏的人，但却一连喝了六瓶啤酒，然后带着满身酒气，搂着几位陪伴他的死党，满世界乱走。吼着今天已记不起词的歌，那种近似乎狼嚎的竭斯底里，充斥着午夜空旷的大街。同样是因为他当时很年青，更因为那是他的初恋。

　　于是他渐渐变得有点沉默寡言、不苟言笑。以致于有些女同学调笑着说他是冷血动物。他不再理睬别人说他什么，整天寄情于填词、写诗。他虽然不懂谱曲，但却作了几首悲哀的歌，整日介自个儿唱给自个儿听。

　　后来的几年里，他也认识了不少女孩，但相处时同样是找不到那种爱的感觉，其中理解他的几位后来成了他的红颜知己。

　　终于有一天，他突然感到自己长大了，天空也变蓝了，昔日的幼稚不见了。他于是写了一张热情洋溢的祝福卡片寄给她。之后他尝到了一种如释重负的快意。再后来他就遇上了现在成为他妻子的那位女孩，（其实也是当年的同学）他逐渐重新学会了去爱别人，去爱这个世界……

　　读了这个沉长的故事，慢兄你是否有种似曾相识的感叹。不过可以肯定的告诉你，故事绝对真实。

　　最后衷心祝愿字字慢早日"长大！"

　　哦，忘了跟你说声，明天不会下雪，明天会有新的阳光出现，包括爱的阳光！
