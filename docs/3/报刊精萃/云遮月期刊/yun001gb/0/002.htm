写在前面的话

乔燕

　　生活中总有那么多遗憾，我们不得不面对！

　　残疾的障碍、贫困的老人、失学的儿童、他们明亮、晶莹的眼睛溢满了热诚的渴望；天空不再蓝、河水不再清澈、蝴蝶不能放心轻舞、藏羚羊就要从我们的视野里消失……我们不能再保持沉默、不能不忍住泪水！

　　我们要呼喊、要歌唱、还要愤怒地控诉！

　　我们爱他们，爱我们生活的这个世界！

　　欢迎你留下心中的感悟、呼唤、愤怒与呐喊......可以是长篇，也可以是只字片言。

　　我们的生活要由我们来关爱！

　　我们的家园要由我们来创建！

　　来吧，朋友！别再犹豫！

　　让我们踏上这块爱之土，共圆这残缺的梦！翘首企望那隐入云后的月亮！
