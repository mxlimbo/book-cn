菜头三愿——我的生日怀想

和菜头/文

　　再过三个半小时，和菜头就围绕太阳公转整整25个年头了。一个半小时以前，刚刚和Netplay、deepsky、小乌龟等老对手狠狠恶战了一场。作为社区板砖手，能和这样一群够份量的老对手酣畅淋漓的进行一次脑力和反应的大战，真是痛快！怎么一个爽字了得！而用这种方式庆祝我的生日，是一个砖手梦寐以求的事。四个小时以前，我的老朋友"元谋人"和他的女朋友第Ｎ次分手了，而我第Ｎ＋１次又用一个小时令他们破镜重圆了。生日到来之际，我在聊天之时，又做一件功德无量之事，听见朋友欢乐的声音，顿时觉得自己形象高大起来；八个小时之前，我的两对朋友分别打电话通知我他们结婚的消息。一对终于结束了九年的爱情长跑，另一对则是女方最终坚持嫁给了我一直为理想奋斗至今，仍头破血流、不名一文的老友。爱情真伟大啊！我几乎又要相信她一回了。

　　令人开怀的事接二连三的发生，我不由得想起前人的《玫瑰三愿》。在我２５岁生日到来的时候，我也想作《菜头三愿》，表达我对未来的希望。

　　一愿：中国电信忘记登记我的电话号码，我上网的费用永远全免。

　　二愿：生个儿子有屁眼儿。

　　三愿：我老岳母终于开始跳皮筋了。她老人家终于上小学了！跳吧跳吧，快长大，生下女儿把我嫁！

　　我想对我的母亲说："老妈，你生了我这个儿子，二十五年间，我们在一起的时间是十二年，分开的日子却有十三年。记得第一次回到你身边的时候管你叫阿姨，你伤心了很久。我的童年里没有关于你的记忆，我也很伤心。这十三年里，我们学着相互接受对方，却不是很成功。你比其他的妈妈更难理解我，而我到今天也不曾学会如何牵你的手上街，甚至也不习惯女孩子接触我的身体。但是无论如何如何我感谢你为了给我生命而感受的痛苦，我感激你把我平安的带到这个世界上。我真的很喜欢这个世界，谢谢你给我机会去认识它。无论我曾经在这广漠的世间曾经有过如何的困惑，如何的伤害，如何的失望，我喜欢它，而且一点不后悔。告诉你一句话，其实你不用去染发的，你满头的白发非常的灿烂，非常的美。我最后唯一对你的希望是：千万不要给我介绍女朋友了，您眼里的美女和乖媳妇我真的是实在吃不消啊！"

　　我想对我的父亲说："老朋友，你三十九岁生我，使我赢得了菜头三十九的美称。多少年来我们冲突不断，知道你一直想我继承你对物理的天份，但是我文章真的写得好，也真的做不了一个伟大的物理学家了。我是你的血中之血，肉中之肉。所以我和你一样的倔强，一样的顽固不化。我听说我十岁那年，你因为怀疑我受辐射得了白血病而流泪。我没有看见，是的，我也现在也和你一样，不善于表达自己的情感。当我们面对面的时候，当我试图变成一个男人的时候，我们之间的冲突终于爆发。我想男人之间永远都是大型猛兽之间的关系，只有当双方都足够强悍了，彼此之间才终于有了沉默的尊敬。我们现在沉默了，我们彼此无言的尊重对方了。我要感谢你一直给我买书，教我狩猎的技巧，也感激你用耳光教会了我男人不可以流泪。我现在是条汉子了，谢谢你。希望你能在晚年终于学会表达爱的技巧，看到你的生命在我的身上的延续，直到永远。谢谢。"

　　我想对我在天堂里的刘老师说："老师，今天我很乖，请你狠狠的表扬我。在我已经取得的荣誉里，没有一项能和你当年对我的温言表扬而使我更感到高兴和满足。在我２５年的生命里，你是继我母亲以外第二个改变我生命的女人。我从一个打群架的幕后黑手，转变为一个好学生。是你发现了我的闪光之处，是你教我知道生命的价值有更高的体现方式。你没有责备过我，看见你对我失望的眼神，我比被打了一顿还难受。我还记得你给我讲解题目时的情景，在梦里我还是为自己能帮你抱着作业本而高兴。老师，你看得见我吗？看见今天的我，你也开心吗？在你的关怀的话语中，在你鼓励的眼光里，我才没有自暴自弃，没有和我当年的很多'朋友'一样今天烂在土里。我想你是上天派给我的守护天使，我象是儿子眷恋母亲那样眷恋着你。在你张开的双翼下，那就是我的天堂。不管多少时光匆匆流走了，一想起你，我就又变成了那个淘气的孩子。老师，今天我生日。老师，我很想你。"

　　我想对Amio说："你在美国还好吗？请你原谅我，你是我的第一个女朋友，也是我四年大学生涯里的阳光。可惜的是，我没有好好珍惜，你的灿烂再也不会披在我的身上。年轻的时候，我犯下了多少错啊！回忆里，你还是那只在南京长江大桥上蹦蹦跳跳踩我影子的小兔子。你还是那个每次考试都能考九十多分，而且四年里一直罩着我的聪明的家伙。你依然和我牵着手走在校园里，继续着我们之间的小游戏，握一下手是YES，两下是NO。我到今天也没有明白你是如何不转动手腕，就可以让自己一双手看似旋转起来的。还记得那次我从山里回来，送你的那一抱金盏菊吗？现在有人送花给你了吗？Amio,我已经不会再踏上江南的土地了，我想我再也不能去杭州了，我真的怕了。你是我心底里最柔软最柔软的地方，轻轻碰一下，我都会痛到发狂。是的，你是最好的，没有人能可以和你相比。我明白了有许多事是无法挽回的，也终于看懂了你送我的毕业留言：人人只道江南好，游人只合江南老。春水碧于天，画船听雨眠。垆边人似月，皓腕凝霜雪。未老莫还乡，还乡需断肠。只有你能配得上人似月，也只有你能有一双凝霜皓腕。我合该在杭州终老的，而不是在昆明怀念你。在昆明的种种遭遇，都是我应该得到的惩罚。都过去了，今天我生日，希望你一切都好。忘了告诉你，我很怀念你妈妈烧的牛肉，真好吃。你说我就知道吃跟睡，是真的。"

　　我想对我的朋友和网络世界里的朋友说："请你们大声说：'认识和菜头真不错！'人在生日时难免虚荣心强烈一点，而且我真的觉得自己作为朋友是非常难得的好朋友。我们在一起的时候，共同分享了多少快乐的时光。而能带来如此之多欢乐的朋友，恐怕并不是很多见吧？那为什么不大声点告诉我你们的感受呢？我也很感谢你们对我的理解和关心，尤其是对我的诸多怪癖的宽容。你们是我生命中可以确定的门，生命中我会遇见很多门，大部分不可以打开，更不知道后面是些什么。而我确定在我叩响你们的门的时候，你们会开门，门里有温暖的灯火，柔软舒适的沙发，当然，还有你们的拥抱和醇香的美酒。有了你们，我免于自以为是怪物而变得自闭而孤独，培根说：友谊是不会飞走的爱情。你们的友情从来没有离开过我的身边，象冬日里的一口烈酒，在我的胃里暖暖的燃烧着。所以，我五音不全的歌唱着欢乐的歌，作为我的答谢礼。我想，你们大概是上帝对我最仁慈的表现了，你们就在我身边的感觉真好。生命里有了你们手持宝剑的环绕保护，使我得到了帝王而不能得的安甜梦乡。我拿什么报答你们呢？那就让我们尽情的欢乐吧！来吧！把这酒杯斟满！趁黎明到来之前，将美酒饮尽！"

　　我在今天回头看的时候，自己的生命居然已经过去了四分之一个世纪。在我还很年幼的时候，曾经在无数个中午，躺在自己的床上，盯着日光灯胡思乱想。我幻想它也在看着我，十年、二十年。。。日光灯几乎没有什么变化，而我在镜子里却发现自己满头白发，衰老，然后就死去，再也无法看我的日光灯了。我不知道我死了以后是什么样子，如果是什么都没有了，那么一点点没有掉的感觉是什么？我对死亡恐惧得无法呼吸，而只有日光灯同情的看着我。

　　我终于明白了人生苦短的道理，所以我想应该积极的去面对我的人生。这世界古老而仁慈，有太多我不知道的，没有领略过的东西。我仅是个匆匆过客，这不是吗？一眨眼我就已经年过半半百了。'时光永无休止，而年华却随岁月流逝。'我因为这句冰岛诗人的诗句而想欣赏诗歌，开始看小说。我终于了解到，世界上有些东西是不朽的，比如美，比如爱。她们远比个体的生命而长久，人类也因为她们的存在对生活有了新鲜的期待，对和谐和完美的渴望。我不知道我能多达程度上接近她们，也不知道为什么要去接近。我想那是本能，对阳光的渴求的手指，对完美合声的期待的眼光，以及对自然神秘内在和谐的好奇。

　　很难判定我的选择正确与否，更难说我能完成我的目标的百分之几。我只能走下去，不停歇的走下去。既然要做个旅人，目标就是远处的地平线。在我的每个生日的时候，我都习惯的停下来休息一下，整理自己的思绪。把过去一年的回忆，好的、坏的刻在一座碑上，然后沉入心底的大洋深处。等我回顾我的一生的时候，凭借蜿蜒向前的石碑，我就可以知道我走过哪些路，错过了哪些地方。而在条蜿蜒延伸的碑林间，有一条几不可见的细线，那是我的脚印。

　　再次感谢你们一路陪我共同走过。

2000年11月5日7：09
