纪念我的老师刘培菊先生

和菜头/文

题记——给天下所有的好老师

　　到今年这个夏天，我亲爱的刘老师已经离开我整整五个年头了。听说今夏的昆明很多雨水，没有明媚的蓝天。而我的刘培菊老师永远都站在我的记忆里，永远站在昆明夏日明媚的天空下，永远微笑着看着我。老师，我真的很想你！

　　我和老师是在我的初二遇见的。她是个干瘦的老太太，教数学。一副黑框的眼镜，老式的发型和似乎永远不变的灰色外套。我对她的第一印象是：矮小，不美。但到今天为止，我都还记得她的目光，从眼镜后面透射出来，直盯着你，仿佛在问你："这道题的解法在哪里？！"

　　由于她的目光是如此的严厉，我在一开始的时候非常的害怕她。

　　在我的初中时代，没有什么是可以被称之为另类的想法。你可以适当的在学习领域里小心的施展你的才华，在同学或者老师斥责你标新立异和哗众取宠前闭上嘴巴，保持沉默。当时，我并不擅长数学，尤其是几何。我时时为那些复杂的辅助线的画法而头痛，所以我尽我所能的虚构出一些不存在的线条，用各种匪夷所思的荒谬方法来论证题目。往往到了最后，我根本忘记了我原来倒底是要证明个什么？

　　你可以想见，当刘老师下了课要我出去谈谈的时候，我的脸在同学们的哄笑声中是什么一付表情？"谈谈"是我整个学生时代最害怕的事情，甚至是今天。

　　刘老师用她那人人敬畏的目光凝视了我三十秒。在她沉默的注视下，我几乎窒息而死。

　　最后她开口说话了："你真是一个奇怪的孩子。不过你很聪明。你的方法不一定对，但是你总是尝试这用不同的方法去解题，这很好。"我万万没有想到，我所期待的焦雷居然变成了春雨，她-说-我-的-方-法-很-好！我抬起头，看见了她微笑的神情。我从一个刚刚还瑟瑟发抖的可怜虫一下子变成了个聪明的家伙！接下来她又说："但是，我同时也发现你不老实。在证明过程中弄了些小花招，是这样吗？"我当时一定是开心坏了，居然回答说；"不！老师！是鬼聪明！"在她佯怒举起三角尺的一瞬间，我溜之大吉。

　　从那天开始，我是真的爱上了数学，几乎每天都做题到很晚。但这并不和电影和小说里的情节一样：男孩夜夜苦读，镜头一转，他的考卷上就都是一百分了。我一般只有八十多分，偶尔上九十。因为数学根本就不爱我。

　　刘老师还是经常把我叫出教室，在走廊上给我讲解我的错误之处。我到今天还记得她瘦弱的手指居然那么有力量，可以轻而易举的画出准确而直的线条。而且，她的手指仿佛永远都是沾满了红蓝墨水的墨渍和粉笔沫。她总是就那么一下，就画出一条标准的直线，在我的错误分析的线条上打个叉，然后匆匆用手把它擦去。她从不指责我什么，我就在旁边看着她一下一下的画，一下一下的擦，内疚的心情一英寸一英寸的升起。最后，当她拍拍手，吹吹她身上的粉笔沫，表示讲解完毕时，我会清不自禁的弯下腰去，深深的鞠一躬，向刘先生告别。她总是很开心的看着我，说："你呀！调皮鬼！"走廊里的风吹拂着她单薄的身体，她的短发在空中飘散，露出更多的白头发。廊外花坛里的紫色花枝也在微微点头，我觉得我的老师很美。

　　最让我意想不到的事发生了，刘培菊先生居然要我去参加昆明市中学生数学竞赛！我听到这消息的时候，以为自己的神经出了问题。可是刘老师却对我有十足的信心，她对我说："没事的，你会考好的。要勇敢的去想。别管对不对。"事实后来证明刘老师是智慧的。当我进了铁三中的考场，发现百分之七十以上的题目我根本不会。但我却发现，题目里的多数参数都是很小的数字。我立即问监考老师要来了所有的草稿纸，凭一把量角器，一把尺子，一个圆规。我几乎是丈量了所有题目要问的长度和角度。一个月后，结果下来。和菜头荣获昆明数学竞赛初中组三等奖。她创造了奇迹！

　　在我的整个童年和少年时代，我过着狼狈不堪的家庭生活。父母的不和，对一个孩子来说简直是致命的伤害。当他面对他至亲至爱的人怒目相向，他的天空完全都是黑暗的；当他目睹家庭暴力和口角时，他却无从躲避残酷的场景。他只觉得自己的心被放在盐碱地里活生生的磨出血来！我时常逃离母亲的泪水控诉和父亲的咆哮如雷，一个人跑到学校的操场上去。学校里有我的朋友们，有我的刘老师，她家的灯火总是那么的温馨。夜深了，学校里没有人了，我不记得我有多少个夜晚是在老师家的窗下度过的。我总是觉得那里的灯光可以给我很多的温暖，我只是看着看着就已经觉得很安全了。曾经多少次决定勇敢一点，结束自己可怜的小小生命，但是我的脚步总带我转回学校，回到老师家的灯光下。在那里，我仿佛又可以看见您笑着骂我："调皮鬼！"老师啊！您知不知道，您家的灯光多少次挽救我本来早就应该结束的生命？您家的窗台下曾经有一个多么绝望无助的孩子的灵魂？

　　对比冰冷的家庭生活，在我的刘老师的身边，我可以感到时时刻刻的关怀，她对我的信任和理解，在她的帮助下，我可以创造奇迹！我甚至怀疑我生错了地方，刘老师才应该是我的妈妈呀！在她瘦小的身躯里，埋藏着巨大的、无微不至的、包容一切的母爱！只有在她的怀抱里，我才拥有了世界，阳光和欢笑！只有在她目光的凝视之下，我才可以安然入睡，无梦相扰。

　　我的事情刘老师多多少少听说了。她没有对我多说什么，只是告诉我她相信我的明天会很美好，我会有光明的未来，要我努力的去争取。她说："到那一天，一切都会好起来的。"

　　我当时根本不知道，那一天到底在哪里？

　　初三了，刘老师的爱人突然间去世。我从没有看见我的刘老师会一下子多了那么多的白发。追悼会那天，她几乎是被两个人拖着在走。刘老师之前还是坚持来和我们道别，她很遗憾的告诉大家，她不能继续给我们上课了。她停顿了一会儿，抬起头来，对同学们说："因为我的爱人去世了。"她好象已经用完了全身的力气，脸上的肌肉终于无法控制，失声痛哭起来。我因为她不能继续教我而万分伤心。

　　我考上了同一中学的高中部，我又可以经常看见刘老师了。经过几个月的大病和休息，我在开学以后又看见了她熟悉的身影。当她得知我还在本校时，她的欢愉之情也是溢于言表的。我时常去看她，路上见到她的时候尽量讲一些笑话让她开心。她很喜欢我们的初中同学去她家里玩，我依然要装出那个大孩子的样子，逗得她经常大笑不止。可是我还是可以看出她内心世界里已经消失了火焰，她实际上是在等待与师公的重聚。

　　日子一天天过去，刘老师看起来气色好多了。不知不觉中，高中最后一个元旦来临了。我预感到我会离开云南到远方去读书，所以专程买了一个新年卡给刘老师。当时的我，已经是个1

　　74公分，86公斤的庞然大物了，而她却依然是那么的瘦小，站在我的面前是如此鲜明的对比。但是这个矮小的老太太却有一种神奇的力量，可以使我这头凶暴的巨兽在她面前俯首贴耳。我是这么写的："无论我走到哪里，只要您轻轻一声呼唤，我就会象一头狮子一样跑到您的身边，象只小猫一样乖乖依偎在您的脚下。"她很感动，也很高兴。

　　我大二回昆明探亲，由于我春节是在杭州女友家过的，没有回来，所以我兴冲冲的跑去看刘老师。在学校里我碰巧遇见另一个老师，当他得知我的来意时，他告诉我说我的刘老师已经过世了。我的笑容僵在脸上，世界又如同七年前一样的黑暗。我再也见不到我的刘老师了！我在也看不见她的微笑了！我再也听不见她说我是"调皮鬼"了！曾经伴我在最黑暗的少年时代，鼓励我，支持我，给我信心的刘老师我永远的失去她了！

　　我是她最喜欢的学生，我是她最得意的门生。我许诺过要奔到她的身边，但是我没有。当我得到一个我深爱的人的时候，我却同时失去了另一个。我不知道她最后是时光里有没有呼唤我的名字，这样的想法让我发疯！我深深陷入自责之中，我觉得刘老师在冥冥中对我很是失望。每每念及此处，我都无法控制的愧疚。这是我的错！刘老师！

　　今天的我，已经不在是那个可怜的少年。刘老师，现在的我无论在精神上还是肉体上都已经非常强悍，对所有的伤害无动于衷。我已经学会了如何对付痛苦，如何面对生活。亲爱的老师，您说过的"好起来的那一天"我还没有得到，但是我已经知道在哪里了！刘老师，可是我还是无法平静的回忆起您，回忆起您家的灯光，回忆起你您身影、您的声音。我曾经多么希望我真的"好"了的一天，能让您一起分享我的快乐和幸福。哪怕是一天！可是上帝把您带走了，这不是他的第一次，也决不会是最后一次！您无私的给了我温暖和关怀，我甚至没有一个回报您的机会。您就走了。

　　您离开我已经五年了。这些年我很乖，没有做很多坏事，真的，您得相信我。公司里的薪水不错，但我也没剩几个钱。我还是没有信心去和谁结婚，老师，你又要笑我了。不是我长不大啊！老师。我是真的没信心。您走了以后，世界变了很多，我也不清楚是变好了还是坏了。反正这里有了网络，可以把我要对您说的话都絮絮叨叨的说给您听。没人知道我是谁，他们也没有办法笑话我。您听烦了吗？老师？我真是越大越没有出息了。可是全世界我只有你可以诉说了，老师。只有你愿意听我的费话，就象你是我的妈妈一样。

　　刘老师，学校里的花又开了，很漂亮。老师，我真的很想你！
