我的1999――失落的世界（下）

萧十一狼/文

二

　　1999的下半年

　　我说过人生是如此的富有戏剧性，有时候在一瞬间就足以改变一个人的一生，比如说在7月11日那天我匆匆赶到单位报到的时候领导笑眯眯的对我说："小乌龟啊，今年有点难办啊，你们的公务员指标人事局还没有批下来"。

　　我楞了！

　　而接下去领导依然笑眯眯的对我说道："小乌龟啊，不要紧张，这个指标问题我们正在向人事局催呢，估计很快就能办下来了，你就耐心等吧，你给个电话，到时候指标下来了我们就通知你来上班"。

　　我相信了。

　　后来我就经常的到单位里去打听关于指标的问题，领导总是笑眯眯的对我说"年轻人别着急，总会解决的，你就等着我们通知你上班的电话吧"。

　　后来在一个很偶然的机会我遇到了一个在市人事局做办事员的同学，他悄悄的对我说我们单位的指标早就批下来了。

　　我了解我这个同学，我没有理由怀疑他。我总算明白了一切。以前我一直都以为官场上的尔谀我诈只是一种被夸大其辞的电视剧里的说法，没想到在我满怀希望的迈出人生第一步的时候，就走到这个急速的旋涡边上，然后傻忽忽的跳了进去。

　　再后来我再也没有见过那个领导以及他那张和蔼可亲的笑眯眯的脸，而通知我到单位上班的电话到现在为止也没有响过一声。

　　此时，摆在我面前的路有两条，一条是回到老家――一个盛产油菜的县城；另一条是背着行囊走上打工的道路。

　　很快的，我那在老家的县城里还算神通广大的哥哥就为我联系好了四家单位供我挑选，都是那种低风险高收入的工作，一旦我回到老家去，我的人生就将按着工作――下乡锻炼――提拔――结婚――生子――再提拔这样的路途走下去，一切都早已安排好了，一切都不需要我的努力，房子、车子、老婆孩子，都会慢慢的出现在我的生活之中。我就会象一列火车一样，只能顺着铁轨一路走下去，再没有选择的余地。

　　我很难清晰的回忆起当时我的想法，我的家人都希望我回到老家去，他们无数次的苦口婆心的对我说回老家去的种种好处。在他们的思维里那是最好的生活方式，安稳平静。

　　我当时正年轻气盛，总把前途想得太过光明，总觉得任何困难都能迎刃而解的，总觉得是没有问题能难倒我的。在我的印象里，我当时好象是很清楚的意识到老家那个偏僻闭塞的县城是不适合我发展的，我的想法是要在纷乱的社会环境中实现我自身的价值――就这么简单的一个想法。

　　在7月31日，我正式回绝了我大哥为我找的工作，我对全家说我决定了要到昆明打工，去体现我的价值。满脸的憧憬。

　　为此我和家人翻了脸，父母出奇的安静，一句话也没有说，更没有骂我，但我已从他们的眼神可以看出他们对我的失望。

　　我失去了家人对我的信任，失去了我的事业，也失去了在我人生旅途中最后一次自主的选择生活方式的机会。

　　从那以后，我就仿佛一叶飘忽的独木舟，在社会这个浩淼的大海里孤独的漂泊着，周围的海水不断的拍打着我的身躯，海水不断的灌进我的舱里。我没有舵没有帆没有推进器，只能在变化无常的洋流的推动下毫无目的的漂流着，尽管我也在极力的苦苦挣扎着，但是我却没有任何能力去改变这一切。

　　一年半已经过去了，我不知道我当时那个草率的决定是否正确，我一直都在思考我的人生目标是什么，我人生的意义又是什么。而当我看到一个人在整个社会之中的渺小之后，我对生活更是失去了信心。

　　为此我总是很迷茫，也许当初我回到老家之后可以很好的生活，没有任何的压力，也许我早已有了自己的电脑，每天下班回家之后可以以极度放松的心态在网络世界里遨游，也许……而我现在却还什么都没有，还在为自己的温饱问题在这个城市里穿梭奔走劳累不堪。

　　我一直都在用'老家的县城不适合你，你出来打工是为了体现价值的，吃过苦才有更大的收获与发展才能更好的生活'这样的话来安慰自己。其实我是个自私的人，我从来不会去想为社会做什么贡献，而我的所谓的收获和发展其实也只是一个为了能让自己有更好的物欲享受的借口，因此每当我想起我就算事业有成之后所过的生活跟我回到老家之后的生活应该不会有什么本质上的差异，再想到为此我在打工路上的坚辛的时候，我就忍不住为自己感到悲哀。

　　当然我不会用'后悔'这样的字眼，在我的字典里没有这两个字。我是一个男人，且不管我当初为什么会选择走这样的道路或者是怎样草率的选择的，既然我已经选择了，我也就只能顺着这条没有方向的道路走下去，前行。

　　在我向全家宣布我的决定而我大哥也回绝了那几家为我准备的单位之后，我背起了行囊义无返顾的走上了打工之路。我到了昆明，住在我大姐家里。

　　而此时我还根本没有意识到我的厄运远远没有结束。

　　还记得我说过的人生充满戏剧性的话吗？在我的这一出话剧里这样的戏剧性情节还没有演完，还在继续的上演着，根本就不以我的意志为转移。再比如说当我满怀豪情去找工作的时候发现了我的身份证在7月的第一天就不见了的。

　　毕业分工我没有正式的工作单位，户口和档案还挂在曲靖市人事局里，而我又没有身份证，任何一个有过这种经历的人读到这里都会明白了――我被剥夺了作为一个正常的中国公民的资格，我是一个标标准准的'三无盲流'。

　　没有身份证和能证明自己身份的有效证件对一个尚未落下户口和没有工作的人来说就意味着没有了一切。

　　在以后的日子里，我四处奔波的找工作――我用我的大学毕业证和学位证来证明自己。我总希望能够碰到一个开明的老板不考虑我的身份证问题而接收我。

　　可是当有一天我晃荡到一个建筑工地里去寻找一份干苦力的民工的工作的时候，那个傻兮兮的四川工头把我的毕业证和学位证丢在一边，伸出粗糙的大手问我要的却是身份证――那是唯一能证明一个人身份的证件。

　　在那一瞬间，我彻底的绝望了。

　　大姐和姐夫对我都很好，但是他们的工作都很忙，没有闲暇的时间来管我的工作的事。

　　从那以后，我就再没有跑一家又一家的公司去试图寻找工作了。我兜里揣着毕业时还剩下的几十块钱，整日的在昆明城里漫无目的的游荡着，在人行天桥上和乞丐一起晒太阳，在广场边上看着老头老太们跳着蹩脚的三步，在街边看那个已经和我很熟的刚下岗的擦皮鞋的大叔套拉着脑袋的一副无精打采的样子……

　　我就这样的浪费了半年的时间，1999年很快的就要过去了，21世纪即将到来。

　　1999年的12月31日的晚上是一个有雨的夜晚，大姐上夜班，姐夫在单位值班，外甥由于受不了瞌睡虫的侵袭早已熟睡。留下我一个人孤孤单单的看着电视，电视上播放着一个学校为了创造吉尼斯世界记录而耗时几个月在两个体育馆里码起来的上百万块的多米诺骨牌，当然播放的是最后推到牌的壮观的场面，看着多米诺骨牌一块块的接连倒下，一丝久违了的快感涌上心头，很难说明在那一瞬间我是怎样的一个心态。

　　2000年终于到来了，当零点的钟声敲响的时候，整个城市一片欢腾。

　　看着电视上北京各高校的学子们在世纪坛前尽情欢呼雀跃的时候，我的心却分明感到了一种凄凉的冰冷。

　　我站起身关了电视，关了灯，推开窗户，外面下着小雨，淅淅沥沥的。远处一颗颗烟花腾空而起，在夜空中绽放成一朵朵各式各样的漂亮的花儿。

　　我的眼睛逐渐湿润，眼前的一切渐渐的模糊，直至一片黑暗……
