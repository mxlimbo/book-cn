On the way

和菜头/文

　　Ontheway,是一种流动和躁动不安的状态。随着自己内心的感受而不停顿的奔跑，无间断的风驰电掣。是这个城市里最有活力的脉搏的跃动，是将灯光撕裂成流星的速度之魔。我们的内心世界无法安于平静和无变化的生活，我们的心底里有一只猛兽在夜夜嘶吼，让人一刻不得安宁。我们必须在它安睡之前立即上路，找到它要的那片森林，让它在草原上自由奔跑。

　　Ontheway,我无法告诉你风在往哪个方向吹，你只能一刻不停的跑，去寻找。梦想肯定就在前面某个地方在等着你。也许那是你一生都无法企及的空中花园。但是我仍然只能流着泪对你说："她就在那里啊！"你没有选择的权力，因为你已经在路上。纵然你最后你两手空空，一身伤痕，神情憔悴，黯然神伤，你也只能默默接受。因为在无数的路标之下，人谈不到永恒，更没有最后。

　　Ontheway,你准备好了吗？世界很大，可以漫步在香榭里舍大道上，寻找一家合适的咖啡店；可以骑在独峰骆驼背上欣赏胡夫金字塔上的日出；可以泛舟于安大略湖上，欣赏瀑布的壮观景色；可以站在雅典卫城之上，凭吊远古时代的战士和哲人；可以走在非洲大陆的长草间，看斑马静静吃草，河马在喷水嬉戏，成千只鸟从你头顶掠过；可以住在南太平洋的珊瑚岛上，躺在白色的细沙上，在海风和涛声中沉沉睡去......在路上的意思就是：在你离开这个美好的世界的时候，你已经用你的脚拥抱过她的每一寸肌肤。

　　Ontheway,你要飞了！看看你身下的世界，你会看到很多渺小和可笑。你的时间的沙漏在一点点流逝，你在用你的心在阅读这世界。你根本没有时间顾及别人的愤怒、仇恨、嫉妒和嘲笑。你远远的把它们抛到脑后，它们还根本没有发出声音就已经在你的世界里腐烂了。你关心的是在你身后紧紧跟随的两个敌人：空虚和迷惘。你不停的在问自己为什么？你的灵魂在每夜准时前来，拷问你相同的问题。你无法呼吸，你只有不停前进。你并不知道你在一点点变得强大，你一点也没有意识到你的力量。直到有一天，你终于和他们相遇。你会直面你赤裸裸的内心，你终于知道答案就在你自己身上。真实的领悟和巨大的宁静一起降临，你在一瞬间粉碎了虚妄，你在一刹那越过了永恒。

　　Ontheway,这是最自由的方式,在我们这个疯狂的数字时代里的最终解决方法.你注定是个行色匆匆的过客,你清醒的意识到你没有什么东西是可以带走的.所以,你是不可用矩阵和算法来解构的,你无法在数字世界里被分解为比特；你是光滑的,别人无法用世俗价值中的珍贵之物来束缚你的心,你拒绝收买,因此,你避免了曲膝面对生活,你如同游鱼一般掠过浮华的现实的沙堡；你将是数字洪流之中独力逆流而上的一条船,你将是这暗夜里的最后的古典骑士,孤独而骄傲,优美而典雅.因为你在路上会发现传说之门,游离于远古和现代,传说和现实之间.你在时间之雾里,缓缓上溯,一直退到结绳记事的时代,学会早已失传的美,你以手述心,以心通心,重新看见完整的和谐的世界,人性的世界,生命间息息相关,循环不息.

　　Ontheway,这意味着你要能够习惯孤独，习惯一个人的世界，因为你没有人追随，没有掌声和喝彩；要足够坚强，可以在人群中生存下来，因为社会凶险，动物凶猛；你要学会微笑着沉默，因为在这一路上的美丽，只是为你一个人盛开，是你自己生命的庆典。广袤的大地是你荣耀之堂，无尽的时间长流是你的光荣勋章。你用你的沉默保有这天地之间最大的秘密；你还要学会勇敢对人大声说出："我爱你。"千万别怕对人说出你对他的感受，千万别在不得不分别的前一秒才说出口，因为这三个字会燃烧，会发光，会在你的眼睛里点燃绚丽的焰火。在路上的日日夜夜里，它是神奇的咒语，可以给你光明和希望。可以柔软你被生活磨砺得坚硬而黑暗的心。

　　Ontheway,在路上。天空清澈而遥远，大地坚实却又苍茫。你一个人上路了。陪伴你的有清风流岚，朝霞暮霭，星光虫鸣，渔火疏钟。大路向无尽的天际延伸，风沙吹过，掩去了你的足迹，消失了你的背影。远处依稀传来你沧凉的歌声，你一路走过，一路歌唱。
