中指

敏感之花＊阿明/文


中指承担“力”的煎熬已经习以为常，
手中的老茧是最好的证明。
也许收缩的时候他不是最有力的，
但是展开的时候他冲到了最前线。
中指没有蜷缩在手心的习惯，
他向往外面世界的辽阔，他是雄鹰。。。。
鼠标上，食指的点击执行着命运的审判；
鼠标上，中指的点击打开命运另一扇门。
如果你忘记它的存在，
决定命运的路只有一条，你只能选择“执行”；
如果你重视它的存在，
你有了选择的余地，你才知道柳暗花明的含义
中指有两个最亲密的兄弟——食指、无名指，
三兄弟之间的战争源自于一枚戒指，
于是，爱神安排他们记录恋爱的过程——
食指上的戒指在苦苦寻觅着，
无名指上的戒指抱怨爱已经到了终点，
只有中指享受了爱的最美好的阶段，虽然短暂！
