可能是我

敏感之花＊阿明/文

　　在镜子中看到的"我"似乎比我还要真实，对于自己来说我并不能看清楚自己，只存在于心中一个概念——"我"！通过它我却能清楚的看到自己。于是，明白了哪一个才是虚拟的，有可能是我！我存在于自己的观念之中，生活在镜子里，我的思想犹如遥控器，遥控着镜子里的"我"的一举一动。有一天终于发现并不是我在遥控镜子中的影像，而是影像在遥控我！我无从知晓镜子中的"我"究竟在想些什么，也不知道他怎么看待对他来说一张玻璃对面的我，同一个样子不同的思想？或者说......！

　　希望这个影像有一天不要怀疑我是假的，而他是真的！哈哈，我终于给自己想到一个方法确认谁真谁伪！我大声的喊叫，虽然他也同样的张口，但是只有我发出了一个声音，而他......!也许他正在怀疑自己的真伪,也许他永远也不会明白。忽然我感觉他好像想要打破常规，不按照我的举动行动，他似乎正在努力的摆脱我对他的影响，但是他最终还是失败了！当我正想嘲笑他不自量力的时候，忽然听到他冥冥之中也在嘲笑我"哈哈，你不也是照样摆脱我对你的影响吗？难道我做的一个动作你就能不做吗？"我又重新开始和他讨论起来，究竟我们是同一个人还是只有长像相同而各自独立的两个人呢？最起码我们应该属于不同的世界吧！镜子中他的表情好像也和我一样的困惑了起来，不过最终还是没有结果。忽然有一个犯罪的念头——怎么能准许在这个世界上有一个和我一模一样的人存在？！不行，我必须毁灭他！不假思索地抓起一个烟灰缸向镜子对面的"我"扔了过去，好像他也正想毁灭我，也抓起一个烟灰缸向我扔了过来！我下意识的向旁边一闪，没来得及看看镜子中的他是否也躲到了一旁，就听啪啦啦的声音.

　　.........虽然没有两败俱伤，但是我想我可以安心的作我自己了，镜子中的他也放心的作他自己了！我昂首阔步的走出了大门，忽然我发现走廊的对面出现了我惊讶的面孔，"原来你还存在！"我们两个不约而同的张大了嘴巴.

　　.........他妈的!谁在走廊的尽头放上了一面镜子?
