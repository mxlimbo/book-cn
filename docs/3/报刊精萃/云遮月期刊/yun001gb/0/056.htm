今兮何兮（后记）

萧十一狼/文

　　昨天跟几个同学会面，谈起校园生活的种种，不禁感慨万千。

　　其中一个同学说道：在念大学时谈恋爱是最不现实的，这种爱情与生活无关，但正因为如此，这种感情才是最纯洁的，如果一个人念大学的时候如果没有谈过恋爱，那么她的人生就是不完整的。

　　《今兮何兮》写的就是我在校园生活中的几段感情经历，也许我这些经历并不能算作恋爱，但毕竟这也是一种感情的经历吧。

　　时间可以摧毁一切。

　　想把这些经历写下来是因为害怕这些往事会因为时间的流逝而慢慢的在脑海中淡忘，也或多或少有为这些故事的遗憾而希望用文字这种方式来补偿的原因。

　　只希望这些故事不会很快的从我的脑中悄悄的溜走。

　　写这几篇文章时我的心里都有不同的体会。

　　写《此情可待成追忆》（一）时最快，几乎是一气呵成的，由于那时我想写东西的欲望特别强烈，而我在这段感情中也没有太多的表现，可以说是只有一种比较朦胧的感觉罢了，但这段感情在我的心里的位置还是十分的重要，这始终是我的第一段真正意义上的感情。

　　写《不是我的错》（二）时由于我几乎和这段感情无关，所以写起来比较轻松，但写得比较马虎。

　　在写《微笑只是为了掩饰》（三）时我遇到了前所未有的困难，由于我在这段感情中投入了太多太多，我太想好好的记录下我在这段感情中的点点滴滴了，我太想把它写好了，反而造成了欲速则不达的情况。并且在写这篇文章时我太想把我们所经历过的全部写进去，但我们经历的也太多太多了，想完全写下来谈何容易，于是我便时常在这些经历中痛苦的抉择着，我总是不知道该写哪些才能完整的表达出我真实的情感与内心世界。

　　也许一个人的内心世界和情感是无法用文字所能表达出来的，也许这些东西只有自己的心里明白，想写出来的确太难。

　　还记得大学快毕业时一个比较熟悉我经历的女生对我说：你真是很可怜很惨，你付出了那么多，却没有得到你应有的回报。

　　在我写完《微笑只是为了掩饰》的那一瞬间，我明白了：其实在我付出的同时，我得到了很多的快乐，对于我来说这就足够了。
