爱的殇歌

○福建　周国文

  

    也许年少轻狂是猝不及防的一种沦陷。没有人能准确的知道自己
的归途，也没有人能清楚的明了自己的前路。凭着一种感觉，我们上
路了。但前路漫漫，谁又知道午夜的心到底要在哪里下站？
    这个季候，人心如鸟群，一散漫天。我们的大惆怅与大悲哀显得
是那么浅薄，而生存的底色则显得是那么厚实，深入你我不安的心田
与苍白的肌肤，留下一段不可言说的凄婉。
    张爱玲说得好：“生命有它自己的图案，我们惟有临摹的份。”
是否我们归根到底只是一棵棵不想死亡的树，把所有的梦幻和期待都
托付给了那簇像血一样浓郁的东西。
    因此等候与梦想说来都是一种奢侈。也许我们总是这样，由于失
去童年，才知道自己长大；因为丢了理想，我们才恍然憬悟自己的心
中还有一样叫做“忏悔”的情绪。
    当最后的指纹试图将美丽的斑驳指点人间时，我们会否哭泣？
“世必有非常之人，然后有非常之事，有非常之事，然后有非常之功。”
说其实，步步泣血从生命中来，我们每个人都希望自己的生命小说恍
如一篇完整的时间日记。
    在静思与噪动之间，三十功名和八千里路的日月红尘，在刹那间
扑面而来。也许一个时代的苦痛化烟化雨也终难掩饰让人魂牵梦萦的
血脉相连，只为羁旅行役的“活着”更有伟大美丽的力量！
    如同我们不是鱼而终于不能属于水，我们也不敢想像没有翅膀的
自己如何属于天堂。世事的为难从来都是那么的不可言说，在记忆里
回味，在想像里厮磨，在怜爱中咀嚼，只是不堪用语言来诠释。
    我们的双手还能捧住一篮子的相思吗？我们的双眸还能圈住最陌
生的空白吗？时光在一次次的骊歌中走去，冷然无声的走出。我学会
了以沉默来取代喧噪！也许难以意料的遭遇是人世浮沉沧海漂泊。但
我只能如鱼饮水，冷暖自知。
    一蓑烟雨任平生。我们的感悟在烟中腾云在雨中行走，当爱恨交
加都无动于衷的时候，情又何以解呢？
    遥想当年，爱情是心灵的翅膀，洒在空气中的一把情绪、一把眷
恋与一把忧思，都可以如晶莹剔透的四月天说尽人间的芬芳。而今雁
群飞过，尘埃落定，落笔纸上的爱情还有当年的纯洁否？
    爱又如何？不爱又如何？风月不可解之物，是一股粘稠缠绵的东
西，还是像真心一样是清清爽爽的。假如爱一散漫天，不是几句无心
快语就可以呼来唤去的。那在寂寞无涯的日子里，又是谁来重修爱之
诗篇？
    爱，不能逃不可逃。尽管个人有限哀愁般的故事如一湾浅浅的海
峡，总是源于自己无关他人，仿佛极尽曲径通幽之妙。
    但正如情歌中所唱：“昨日你眉上的恋栈仍近如发梢浮起的星，
明天你心中的渡口还远似街头未亮的灯。”如若只是一个生活惯例的
日常归宿，那爱还能是爱吗？
    是否两个人的寂寞会胜过一个人的寂寞？是否爱上了陌生就选择
了永失家园？是否少年人梦凤化凰的惜情已是如烟云过眼？爱的殇歌
是在伤逝背后，动容来得太早，醒悟来得太晚，痴迷来得太深。只奈
得，一段缘起缘灭的感情，一个曲终人散的故事，一位始乱终弃的爱
人，一条扑朔迷离的来路，在城市生活中终究没有了幸福可言。
    哀情城市，凄婉爱人。末了，这个城这份爱，或者是美，或者是
温柔的感伤，或者是惆怅的浪漫，或者是决绝的悲痛；貌合神离的怅
惘又会是哪一种暖昧。谁又揽你入怀中，谁又轻唤你的名字在来来往
往的十字街头？
    醉花渐欲迷人眼，曾经有过的约，让所有新来的爱一笔勾销吧！
爱如果唱的是一曲流血的殇歌，那也会让我感到孤寂与无聊的。“青
山原不动，白云自去来”，但我分明听到绝尘马蹄声中传来最世俗的
心跳。
    一种爱说时依旧，一种爱挥手自兹去，一种爱看罢江湖夜雨十年
灯，还有一种爱欢也零星悲也零星。说到底，爱终是人世间那片永不
摇落的彩云。爱终不会无声无息断了琴弦。
