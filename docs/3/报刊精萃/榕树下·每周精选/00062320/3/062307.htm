劳动创造财富

○上海　_WulALa_

  
    有父子两人，山居，营果园。父病后，子不肯耕作，园渐荒芜。
一日，父病危，谓子曰：“园中有金。”言讫而逝。子翻地寻金日久，
无所得，甚怅然。是年秋，园中葡萄，苹果之属皆大丰收。子始悟父
言之理。
    这则小故事告诉了我们一个千古不变的真理，那就是财富是要自
己去挣的，天上是不会掉馅饼的，这两句话都是正确的。
    人活着是要吃饭的，饭是靠人来烧的，不劳动就没饭吃，不吃饭
是活不下去的，所以我们要劳动。劳动可以创造财富，虽然说金钱乃
身外之物，它不是万能的，但没了它却也是万万不能的，所以说我们
要劳动。有了钱才可以吃喝嫖赌，有吃喝嫖赌才会有犯罪，有犯罪警
察叔叔才会有饭吃，有饭吃他们才能活下去，所以说劳动是一种可以
拯救人类的活动，劳动最光荣！
    如今，我们已跨入了二十一世纪，二十一世纪是劳动的世纪。正
如前面那位伟人所说的那样，劳动创造财富，现在的东东是越来越贵，
我们不劳动就没有钱，没有钱就买不起，买不起东东放在那里就会发
霉，发霉就会产生细菌，细菌多了就要威胁人类的生命，人没了生命
就不能再劳动，人不劳动就不再有商品，没有商品就不会再有发霉，
那样细菌就无法生存，这样的对待小生命简直就是灭绝人性，漫无人
道！同为地球妈妈的宝宝，人类无权决定细菌的生死，所以，我们要
劳动！
    如今读书是为了拿文凭，有文凭才有工作，有工作才能劳动，有
劳动我们的精神生活才会充实，哪怕是掏大粪。精神得到充实才可以
使我们不会变成白痴，不会变白痴才不会别人当白痴看，不被别人当
白痴看不代表不会被别人当神经病看，神经病是不会劳动的，所以，
只有劳动才会被别人当正常人看，被别人当正常人看才会有人喜欢你，
有人喜欢你才会有人嫁给你，有人嫁给你你才不会去当和尚，不去当
和尚才可以吃喝嫖赌，有吃喝嫖赌我们才能花钱，钱从哪里来？劳动
创造财富！！
