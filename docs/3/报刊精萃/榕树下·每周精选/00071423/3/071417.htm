○上海　软玉馨香


我在没有星子的夜里,吹奏月光
当花瓣酿成了鲜血的颜色 
黑色的天幕在沉睡 
玫瑰为谁绽放 

蓬勃的夏天,猝然地写霜 
没有人知道,其实池塘里的蟾蜍 
也一样会哭泣 
她只是没有泪水 

曾经的回忆脱落成 
透明,薄尔轻的禅翼 
飘飘扬扬的细线旋着 
执线的人站在地上,没有影子 

青春跌入了传奇的花丛中 
汩汩涌动着生怯怯 
那是岁月的憔悴模样 
被风的轮子推着 

涉步汲水,经过木棉树 
燕雀飞散,原来是闻香识舞 
翻转青苔下的石砖 
看到四个字:语焉不详
