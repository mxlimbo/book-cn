害怕看见心爱的人

○重庆　四月风飞


　　
害怕看见心爱的人，在十一月
冬天的风景足够令人窒息 
何况她的脸如此冰冷

十一月，害怕看见心爱的人
月色水银般凄冷
何处觅回昔日的温暖
在这冰冷的腊月
阳光瓦解成薄片
而我心爱的人
却骑着一匹白色的马
向着更远的背阴处驰去

我心爱的人，怕再见到你
十一月的天气寒冷
寝舍的被窝暖和
我从此裹于棉之中
我说我怕冷，其实我怕看见你

心爱的人
冬天到了，马车已驶过
我的双眼不复有昔日的执着
因为你的眸子是一潭无尽的深水
不敢扔花于水中
我怕激不起一点涟漪啊

心爱的人，电影票没有送出
就在黄昏的空气中燃成灰
火焰是蓝色
十一月的蓝色火焰
为何只有寒冷

心爱的人，怕见到你
马车在路上晃悠
而我却在路口抽着一支不熄灭的烟
马鞭从手心跌落
我不敢去追
这是十一月，冬天的暴风雪肆意呼啸
而你的脸又是如此的冰冷
