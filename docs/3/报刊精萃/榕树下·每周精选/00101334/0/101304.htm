飞天

○吉林　风里的梦

　 
大漠里我伫立了几千年，
我见过秦时的明月汉时的关。
悠悠的古道是我飘动的裙带，
悠扬的笛声里有我的眷恋。
我挥动着长鞭，
挥不走心头的思念。
我反弹着琵琶，
弹拨不去那曾经的万朵烽烟。
我也曾是个花样的女子啊，
也曾痴痴的等着我的汉。
他金戈铁马战死在沙场，
春去秋来，季节的变幻中我盼穿了双眼。
在他倒下的地方，
热土已变成沙漠，沃土也变成荒原。
我久久的不肯离去，
黄沙飞舞中，我化成了飞天。
是飞天，我不愿飞去，
轮回中，我还在祈盼。
如果有一天再与你相遇，
如果你的眼还注视着我的眼，
我情愿再为你化作飞天！
