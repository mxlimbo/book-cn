怀念

○桑葚75


　　他是我在这个世界上最爱的人。
　　可是他离开我了。这个世界上再不会有如此爱我而也能让我如此
去爱的人了。
　　日历上被我画着猩红圆圈的日子提醒着我，他离开我已经整整一
年了。
　　一年过得真的很快，好象到现在我都不觉得他是是真正离开我了。
我总是想他去远方了，过一阵子就会回来。
　　可是，事实告诉我他确确实实地不在了。我无法再听到他的声音，
更无法再次感触他，唯一还记得的感觉是冰凉的，那是在他被从冷冻
柜里推出时我碰触他的脸颊的感觉。
　　很多时候我依然无法相信一个活生生的人怎么就会象吹过的风一
样不再回来。
　　不过，也许，生命在那一刻终止于他而言是幸福的。我已不忍心
看着他的疼痛。我甚至想过要求医生去实施安乐死。
　　瘦，已经瘦到皮包骨头，整个人已经脱型。健康时最是红润的脸
庞一片青灰。
　　深度昏迷。
　　我坐在病床前陪着他，尽管医院的消毒药水味会让我一阵阵的颤
栗，一如我生病的时候他整夜整夜地陪着我。可是，我都不知道他是
否能够认得出我来。
　　我握着他的手。干瘦的。手上的皮已因年老和突然的瘦弱打着褶
皱，有很多很多的老年斑。这手最喜欢紧紧地握着我的小手，温暖而
厚实。我喜欢它牵着我的手带我去散步，走好远好远，他总喜欢用一
只手牵着我，然后用另一只手在我的手背上摩挲，痒痒的，但很舒服。
　　我让我的手也那样摩挲着他的，我拿起他的手放在我的脸上。
　　这一刻，不在觉得电视里病床前的镜头如此矫情，的确，在这样
的时候，你只是想握着他，握一辈子。
　　我不忍心放开他的手，因为我不知道我还有多少机会可以这样陪
着他。
　　第一次我觉得如此无助，因为我知道这一次我没有任何挽留的力
量。
　　那一夜。我想他大概是认出了我。他澄大了眼睛看着我，嘴蠕动
着，但发出的咿呀声使人难以辨认，我一遍遍地猜测他的意图，但是，
都不对。他很生气。他用手拔身上七七八八插着的管子，我不知道他
哪里来得那么大的力气，我拼命地按住他的双手，可他的脚还在四处
乱踹，我哭着求他，那个是不能拔的……他恼怒地瞪着我，他从来没
有和我发过脾气，可是他是那样恼怒地瞪着我。大概他很气愤，他这
么爱的孙女居然不知道他想要表达的是什么，而他一贯地向外人宣扬
他的大孙女是他的贴心小棉袄……
　　我用棉花棒沾了水给他润湿舌头，他已经有半个月没喝过水了。
他体内的水分完全依靠输液。我无法想象一个如此长时间未进水的人
体内会是怎样的干渴。我不敢用纱布，尽管那个水分可以更大一些，
但是，他会紧紧地咬住纱布不放口，渴，太渴了！
　　同样的，他也很久没吃过东西。医生说不可以。因为他已经没有
消化的能力。在他还能说话的时候他曾经那样乞求地看着我，说，粥，
喝点粥总可以吧。我笑着对他说妈妈在家给你做呢，一会就可以了。
他满足地点点头，闭上眼睛。我却泪如雨下……我把葡萄压了汁给他
喝，他笑，伸出大拇指……
　　那一夜他不肯睡。他一直睁着眼睛看天花板。我怕他太累，用手
遮住他的眼睛，但是，手掌能感到他的睫毛的触动。我忽然想也许他
是想最后看看这个世界，我没有权利阻挡。
　　想到他最后能看到的世界只是如此狭小的一快天花板，我的心揪
扯着疼痛……
　　四点多的时候他大概是疲累了，睡去。
　　我走上医院的阳台。天灰蒙蒙的，有买早点的小摊贩已经在准备
着，早班车呼啸着驰过。我想，他终于又熬过了一夜了。
　　早上我拖着疲惫的身子去上班。心始终惶恐。我在纸上一遍遍地
写着“人生之大限，生命之不可强求”。很小的时候就考虑过，如果
他不在了我可怎么办呢？总是没有答案。因为总是安慰自己，他会在
那里，他会一直在那里。可是现在他要走了，他真的走了，我可怎么
办呢？！
　　下午再去医院的时候他已经完全昏迷。而且迅速的浮肿。那本是
青灰的面容竟因浮肿而泛出几许红光，让我害怕。
　　妈妈说，他昨夜能那么精神实在已是回光返照。
　　医生说，他熬不过那一夜。
　　晚上七点，监测器开始严重报警，所有的指示灯都是红色，所有
的指标都已降到了最低……医生打了数针强心剂才使指标暂时稳定。
两辆急救药品车推进了病房，姑姑说那是做后抢救和料理后事的。
　　我知道，任何侥幸都不再有存在的理由。
　　我依然坐在床边握着他的手。我要他知道我在。可就连那手都已
经没有丝毫的生气。
　　我看看他，然后就紧盯着监测器，每一个指数的跳动都牵动着我
的神经，只要有一项下到最低点我就要叫医生。妈妈说，你不能这样，
你去睡一会吧，没事的。可是我不能。
　　我就要看不见他了，我怎么能看不见他呢？我不能看不见他的啊！
　　我太累了。朦朦胧胧中我好象又听到了监测器的报警声。我睁开
眼，自己躺在旁边的病床上，妈妈说我刚才突然就晕到了。她不让我
起来，她说没事的。
　　怎么会没事呢，他的床前站着三个医生。监测器上的心率曲线已
经趋于平直。
　　他们大概是在用电击吧，他的身体弹起床面，我几乎以为是他醒
了。打针，我不知道他们给他打了多少针。再然后，医生用双手叠着
在他的胸口按压，只有不断的按压监测仪上才有心跳的指数，只要停
止，心率线就是直的……
　　医生摇摇头，宣布死亡时间。
　　他们都跑过去围在床前哭喊着，只有我扶着床脚弯腰咳嗽，剧烈
的咳嗽直到干呕，我抖，不能抑制地颤抖。
　　这是我第一次如此接近死亡。
　　他们在给他穿衣服。那衣服是我买的，他喜欢的藏青色的中山装。
他软软的，手脚都还带着微弱的温度，只是他的心凉了。我的也是。
　　离天亮还有一段时间。医生说可以再停一会再送太平间。他就那
样整齐地躺着，她为他蒙上了白单。
　　我缩在墙角还是在干呕。
　　我终于可以不再抖了。我走过去掀开白单想再看看他的脸。泪眼
迷梦，我什么都看不清楚。妈妈在提醒我，你站远一点，不要让眼泪
滴在他身上……
　　我还记得那宽旷的走廊，黑的，只有墙根处的小壁灯在指引着方
向。有其他病房的人探出头来看，窃窃私语，那老头走了……
　　太平间里好脏。他被装在一个黄色的袋子里推进了冰柜，我知道，
几天以后我还可以再见他最后一面。
　　大家都打车回家，我前一天骑了车，我说，明天再来拿。
　　妈妈说，明天你就不用来了……我呆了一下，拿了钥匙去存车库，
是啊，他不在了，我还来干什么呢？
　　两次，我骑着车撞在马路牙子上。撞击的那一刹那的震荡让我清
醒。
　　那是个阴沉的清晨，我记得很清楚。
　　他们在大屋里争论着丧葬的事情。我关上了房门。我坐在自己的
小床上。以前，他每次来都睡在这里。我拿着他的照片，我刚刚从影
集里翻出来的，很慈祥的笑容……
　　我很怕这时候会有人进来，我不要有人打扰我的思念。
　　我们去太平间取他的尸体火化。
　　他在那个冷冰冰的冰柜里停了两天。他们拉开那个黄色的带子让
我们确认的时候，我用手轻轻地碰了一下他的脸，好凉，那是我最后
的感触。
　　她没有过来看他，只是在车上问我，他还是那样吧？我点点头。
　　葬礼匆忙地让人觉得有些好笑。
　　那个地方一点也不肃穆神圣。全是人。不象是在祭奠到象是庙会。
　　一拨拨向遗体告别的人被安排得那么紧凑，连最后怀念的时间都
没有……
　　我一直呆愣愣的，只在他被推进火化间的时候号啕大哭。
　　我好象不会说话了。
　　在封墓的时候爸爸让每个人给他铲一铲土说句话。
　　好象从他离开起我就没再说过话。爸爸说，你就说点什么吧。我
死命咬着嘴唇摇了摇头，我的眼泪滴在铲起的土上，落下。
　　思念是我自己的事情，我不要告诉你们。
　　葬礼结束以后我开始高烧不退。没有什么是支撑我的了，似乎连
站立的力气都已失去。
　　我有伤痛无法表达，于是我就生病……
　　已经一年了。
　　在最初的日子里我无法写下这段文字。
　　如今回味起来依然连细节都历历在目。
　　我终于写下了我的怀念，眼睛依然潮湿。
　　你在天堂听见了吗？
