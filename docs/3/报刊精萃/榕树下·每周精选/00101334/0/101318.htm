也是人间四月天

○河南　生命如斯.开到荼靡

　

　　你是一树一树的花开
　　是燕
　　在梁间呢喃
　　你是爱，是暖，是希望
　　你是人间四月天

　　康桥的水流潺潺，徘徊哀婉，似泣似诉，告诉你我一个故事，一
个诗人的爱与死，一段梦萦魂牵的生死恋。我爱上了这个故事，爱上
了这诗人一生一世的浪漫，那神秘的情感天平，一端是三个各有追求
的奇女子，一端是绝代才子断肠的企盼：是爱，是暖，是希望，是一
生追求“爱，美，自由”的多情诗人徐志摩化风，化雨，用他空灵的
生命化作的“人间四月天”。
　　我没有去看那据说是风靡世界半边天的电视剧，即使我也知道黄
磊会有徐志摩脱尘违俗的忧郁气质；刘若英会有张幼仪传统坚毅的闺
秀风范；周讯会有林徽音才思敏捷的淑女窈窕；伊能静也会有陆晓曼
敢爱敢恨的情操……可他们过于完美，完美到没有了缺憾就少了一份
令人遐想的空间，我又不喜欢看这一个个鲜活丰满的人物在字幕上打
出由XX饰演的标识，那意味着你我被直白的告知，这是演戏，只是一
部戏而已，可我要的是感受他的存在，感受他的生活！
　　于是，在街角书舍一个被时光遗漏的角落，我找到了这本寻觅已
久的书――《人间四月天》。
　　买书的那天下着小雨，淅淅沥沥的四月的雨。诗人的四月是春天，
那么，我爱春天，我爱四月，不单因为我出生在这个美妙的季节、美
妙的月份。我把书放在我透明的袋子里，斜斜的挎在肩上。那洁白淡
雅的封面就这么素面朝天的显露着，很是惬意。书的封面是我喜爱的
感觉，有眉目传情的主角，有淡墨浅出的诗句，仿佛翻开来不是一页
页逸散着油墨清香的纸张，而是一卷卷时空编织的唯美的人生诗篇。
茗品袅袅，竹椅摇摇，月凉如水，烛影婆娑，是读它最美的境界。任
思绪跟随书中那游离的时间空间天马行空的流淌，以小说的手笔剧本
的形式来表现一组组情感交融的片段对于读它的人有种致命的吸引。
它瑰丽的文字荡漾着淳朴的乡土气息和旖旎的异国风情，还有那坚贞
不逾的生死相许，肝肠寸断的春闺落寂，又怎不令人“执手相看泪眼”、
深沉难以入眠呢？
　　幼仪是个充满悲情的人物，她付出了一个女人应当付出的一切，
她渴望自己的婚姻有一个承诺，沙粒是大漠的承诺，浪花是大海的承
诺，可她的爱情承诺却是志摩一句：“我要作中国第一个离婚的人！”
幼仪彷徨、失落，作为一个从小被灌输封建父权、夫权的弱女子，她
一直天真的以为那一纸婚约，一抬嫁娶的红花轿便是她成为另一个男
人的女人的“如山铁证”。
　　幼仪有一双未曾缠裹过的天足，这为当时的人们所不齿，可兄长
拍着胸膛信誓旦旦的保证会为她找一个好的归宿，她是嫁掉了，夫君
也是一表人材的出类拔萃。可幼仪不快乐，不幸福，她一生梦想的破
灭也就在这双天足上。试想，如果她是个迈着三寸金莲的小脚女人，
又怎会有气力飘洋过海远涉异国伺候丈夫，又怎会使历尽艰难生下的
婴孩夭折他乡；她本可以安分的在家孝顺公婆，教养儿子的，有怎会
使善良的徽音得知她的存在后不忍背负道义的谴责割舍自己的至爱。
剑桥和硖石隔着一个烟波浩渺的海，若非此番的阴差阳错，志摩完全
可以和徽徽杳无音信的为爱存在，携手并肩走过的。徽徽也不会背弃
自己“许你我整个人”的誓言，毕竟，志摩要她许自己一个未来的，
可只凭一个许诺，就能将爱情进行到底吗？
　　把所有的过错都推到幼仪身上是残忍自私的，没有感情的婚姻是
一副枷锁，向往自由，渴望灵魂共鸣的志摩有怎能不惜牺牲一切冲破
这局限乏味的樊笼。可他忽略了另一个同样为这家族专制下祭旗的福
物――幼仪，一个同所有女人一样需要有一个坚实的臂膀去依靠，有
一副铜骨脊梁撑起天地，可以遮风避雨。可是最终呢？她只得到了两
个字：抛弃！粉饰的自由下离婚对于从未苛求过什么的幼仪来说只是
意味着被抛弃。但她毅然签下了名字，告诉自己：“原来我张幼仪这
三个字还能完整地代表我自己！”那么，选择坚强，就有了对自己未
来的希望！
　　当男人爱上女人，这爱便是火；火可以熄灭，但那颗爱的深沉的
心也亦如烈火燎过的荒原一样没落憔悴，毫无生机。当女人爱上男人，
这爱便是水；水可以抽干，但那份爱的细腻的情也亦如人去楼空只剩
曾经载水的空间，爱已不在但一颗深爱过的心却容不下任何等待。爱
与恨只是短短的的一线间，可两个在缘分造化下相恋的人又怎样去体
会这选择爱与不爱的尴尬哪？
　　正如徽音的无奈：“第一次逃，是回家；第二次逃，是离家。”
对那份炙热烫手沉甸甸的爱情，徽音柔嫩的肩扛不起，脆弱的心更负
不起：有封建伪道德下世俗舆论的偏见克制她不能释怀真正的情感；
有一辈子心高气昂的母亲指望她嫁的个体面风光从此不用在二姨太那
里矮人一截；有另一个曾被唤作徐太太的名正言顺的幼仪使她尴尬不
安。于是女孩子最初的那段菁菁恋情还来不及吐幽如兰的宛然绽放就
默默地凋零、枯萎。心死了，寻爱的灵魂飞远了，只剩下一副清纯依
旧的躯壳承载着林徽音运啦这个象征性的名字循规蹈矩的活着，抓不
住、留不下的，只有一段段前尘往事舒卷成云烟袅袅的散去、逝去，
是康河波心安理得荡漾着的轻解罗衫，小泛兰舟；是萋萋河畔韧韧燕
草上的侬语呢喃；是青灯古刹石阶上痴人一样的志摩一遍遍低唤着
“徽徽……徽徽……”正所谓缘，妙不可言！当徽音和思成双宿双飞
远赴美国掏空了志摩的整颗心时，被誉为北京城一道不可不看的风景
的陆小曼又如风而至。小曼是个没有心的人，单纯的可爱。她喜欢软
软巧巧穿上去跟长出的蹄子似的意大利小牛皮鞋舒适的顺脚；她喜欢
凭着自己的天赋灵感信笔鸦涂出一幅幅令人赞叹不已的佳作墨宝又毫
不吝惜的揉擦掉；她喜欢轻描唇眼淡扫峨眉字正腔圆的来一段票友京
剧让你误认为哪儿的当家花旦颦轻浅笑；她喜欢用纯正的英语法语携
着翩翩舞姿旋的租界里的老外们惊艳爱慕的纷纷在石榴裙下拜倒……
可小曼依旧是没心的，会哭倒在志摩怀中诉尽相思委屈后说很想吃个
面包；会捧着一把糖炒栗子乐的满街撒欢儿满足的笑；会毫不犹豫的
撕掉王赓这张已荣升到参谋长一级的“饭票”为了爱甘心情愿陪伴志
摩守着诗守着爱这精神的富饶。
　　因为爱，志摩给小曼安上了一颗心，一颗兰心惠质；因为爱，志
摩令小曼懂得了无物似情浓的挂牵煎熬。天使为爱而堕落凡尘，整日
沉浸于金迷纸醉八圈牌局的欢娱，甚至在鬼魅幽灵一般的鸦片烟枪中
彻底迷失了自己。似云似雾似梦似幻的青烟氤氲，哪里能够找回最初
的真实呢？
　　无情不似多情苦，岂知聚散难期，翻成雨恨云愁，思悠悠、恨悠
悠。幼仪也许始终理不清对志摩的感情，再回首，云遮归途、高城望
断，灯火已黄昏，似梦里，泪暗滴……
　　沉思前尘，多少蓬莱旧事，空回首，烟霭纷纷，曾经的红笺小字，
诉尽平生意。又是碧云天，黄叶地，秋色连波，波上寒烟翠，而志摩
却以生命化作了另一个永恒，只剩下徽音默默珍藏着那些个海枯石烂、
地老天荒，依别了至爱，此去何时见也，襟袖上，空惹啼痕……
　　人间的四月，四月的天，风轻云淡……
