○湖北　雄性干花
　　　　　　 

　　
总要有个盛开菩提的地方 
不一定要设定在有地心引力的空间 
收留那些被流浪，花不起钱去旅行 
的贫穷眼神 

不用天天默背童话中的幸运或 
羡慕以斧头换来的三个愿望的樵夫 
端着发着星光的摇控器是 
仙女棒 

那里 
犯罪的过程是打着聚光灯的 
作爱时的愉悦程度不用取决隔音墙的厚度 
人格分裂的角色变换 
是被怂恿充许的荣耀 
种族融合的理想在一个 
食指的往下动作被实现 
一座剧本是法律的地球村 

又譬如说：象座传道场 
透过排练过的肢体，糖渍过的语言 
共鸣第一次温度最高的泪水 
虔诚眼睛功能退化的崇拜 
再用加厚了的纯真 
粹链忠诚。无以名状的瘾 

那里 
总是个充满矛盾的地方 
明明在切割的画面中预见自己的泪 
仍用摇控器实现眼泪泛滥的灾难 
甚至忘了下午被情人画破的笑颜 
还淌着油漆未干的血 
险象环生的报复潜意识中 
眼泪是那把刀。 
逆时针的救赎论 

有一群寂寞的眼神 
集体信奉那里的教条 
宣布放弃思考的国籍 
决定皈依那人工的制的桃花源 
认为在那里找到人生最大的宿命 
磨尖真理 

于是路上美眉的吊带裙 
不再有陌生眼神捧场及悦耳的口哨声 
而那些找到新天堂的臣民 
在化学色彩中漂白 
立地成佛
