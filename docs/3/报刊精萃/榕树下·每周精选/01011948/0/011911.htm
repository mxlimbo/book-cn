路盲

○上海　鸟三
　　　　　　 


　　路盲，顾名思义，就是那种走在大城市，和走在沙漠没有直接区
别没有方向的人。
　　鸟三是个名副其实的路盲，如果游人在南京路上问我：“小姐，
请问到外滩走哪个方向？”我会郑重其事的告诉他：“向前15米。”
“啊？这么近啊，我怎么看不见？”“哦，不是，向前15米有一个报
亭，那里有地图买。或者向后30米，那里有一位警察叔叔。祝您旅途
愉快。”
　　当然，好像鸟三的脸上就写着“我是路盲”四个大字一样，从没
遇到过这样的能让我风光的事情。倒是我自己曾在南京路上问别人淮
海路怎么走。后来朋友警告我，这种时刻，千万别说上海话，太丢人。
　　约会顶令我头疼的事，和熟悉的朋友出去，他们都会自觉的为我
设计好线路图。不太熟的嘛，我尽量不答应去方圆2 公里以外的“荒
山野林”。一次约了去看电影，是我所居住的区内最有名的一家，也
只需要乘一辆车。下车，满眼的霓虹，立即有一种不详的预感，其实
这家电影院从小学开始，统共来过大概20次。照着记忆，走，走，走，
嗯，这所学校好眼熟，刚才在车上看见过……啊？又走到下客站啦！
这就要说到上海的马路的不合理性，类似伦敦的，弯的，斜的，岔的，
要像纽约的多好，棋盘型的，横一路，横二路，纵一路，纵二路，好
找得很！别费事了，找个福利彩票售票亭问问吧！唉，老兄，让你久
等了。
　　下一次，还是老地方吧，比较有把握。表妹说，终点站下比较远，
前一站下更近，好啊，多谢指教。嘿嘿，有高人指点，担心什么。很
有把握的下车，走，走，走……嗯？怎么还是这家售票亭？原来足足
溜达了一站路，回来了啊？真没面子。
　　最伤脑筋的是和网友见面。和小错的那次，他在长途汽车上才通
知我，先斩后奏。“在哪里呢？”“人民广场吧。”“好啊，大屏幕
下面。”“什么大屏幕？”唉，外地人，不和他计较。“那你说在哪
里？”“大世界。”大世界？什么年代的玩意儿了，我已经有几十年
没去了。“大世界在哪里啊？”“啊？你是上海人，还是我是上海人？
你一到人民广场，就可以看见它的顶了。”“哦。”姑且牵就他一下。
　　顶？啊，那里有个顶！依稀看见“世界”两个字。哈哈，难得这
么聪明过。就是远了点，一会骂小错一顿。足足一站路啊，终于接近
了，好耶！就要看到小错了。拐了个弯，顶上的三个大字赫然入目：
新世界。惨了，百货公司新世界，娱乐总汇大世界，两家浑身不搭界，
这下弄得“一天世界”（沪语，一塌糊涂，乱七八糟之意。）。一打
听才知道，犯了南辕北辙的方向性错误。见到了在风口里等了一个小
时的小错，立马就来气，“你怎么不早说，大世界的顶是尖的，新世
界的顶是圆的，这样我就不会搞错了嘛。”“……”
　　路盲最大的忌讳就是“自作聪明”，高中毕业时体检，要去“虹
口区中心医院”验血，找对了汽车，好得意。身边两个女孩子正说着
话，“不知道要到几点，早饭不许吃，饿死了。”“哦，我到习惯不
吃的。”嘻嘻，我也从不吃早饭的。突然身后追来一辆车，两个女孩
子赶紧紧跟上去，我也跟着跳上去，反正是同路嘛。呼哧呼哧的开了
几站路，两个女孩子准备下车了，一听站名：新华医院。呃！当我乘
上反方向的车，赶到目的地，全班同学早已队列整齐的等到着Miss鸟
三的大架光临。由于我的迟到，不得不让后面的班级插到前面去，45
个人，集体为我的“自作聪明”多花了一个半小时，如此的风光，很
难再找到机会。
　　以后学乖了，人生一张嘴，问，不就行了？早已习惯了他人的诧
异和乐不可支。要说上海人还真热心的，特别是对一个说话温柔，目
光真诚的小女孩。“老伯，请问北站怎么走？”老伯扳过我的身体，
向前一指，“不就在对过吗？好大的一座房子。”哦，北站什么时候
变得这么漂亮了。“谢谢，谢谢。”
　　为了给这些好心的老人家（问路最好找年纪比较大的，有经验，
又热情。）少添些麻烦，不知打什么时候起，我也习惯了出门自备地
图，一图在手，万事无忧。虽然还难免看错路名和车号，但，总的来
说，确实便利了许多，又有探险的感觉，好好玩！
　　昨天有了第一次的成功经验，找一家公司办事，所在的路很奇怪，
分成两段，我也只不过比最经济路线多花了45分钟。所以，以后不要
再叫鸟三“路盲”了啊！诸位朋友，到上海来的时候，也可以请鸟三
做向导啊！只要，当我花了一晚上，详细的替你看了资料，找你要去
的地点，然后很简单的给你两个字：“打的”的时候，你不觉得太意
外就好，呵呵！
