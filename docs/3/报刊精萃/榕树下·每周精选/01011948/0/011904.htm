各得其所的慷慨

○夜行天涯
　　　　　　 

　　在美国，华盛顿虽然算不上大都市，但作为政治文化中心，值得
去看看的观光地点还不少。大多数景点是自由参观的，而且，陈列丰
富，布局考究，以真迹、文物、实物或现场为主，免费资料和说明材
料十分精致、细致而全面，足以满足普通游客的好奇心，以至于经常
流连忘返，难以辜负。这样一来，如果时间比较短，又打算跑遍所有
景点，就不太现实了。但是，我还是留了点时间，留给印刷出版博物
馆。准备到那里去走一走。
　　印刷术是中华民族对人类的四大贡献之一。应该说，整个美国也
不过几百年发展史，区区华盛顿的一个印刷出版博物馆，能有什么惊
世骇俗的收藏？然而，我之所以想去浏览浏览，是想体验一下往日的
心情。
　　恰好整整二十年。当初，读到这个故事时，并没有料想过，自己
会有机会，来到华盛顿。一个遥远而陌生的城市，一个现实而熟悉的
故事。小时候，感觉这世界好大好大，随着年纪的增长，感觉这地球
越来越小，所有的一切，变得越来越具体，越来越伸手可及了。
　　故事说的是，很久很久以前，在美国加州的一个山谷小镇，一位
全城最好的经纪师，老人莱里，因独生子意外而去，经不住如此沉重
的意外打击，日渐消沉，业务也几乎丧失殆尽。正在他终日借酒浇愁
之际，城里最老的老人，九十九岁的老太太登门拜访。早年，在美国
淘金大潮中，她千里迢迢，到西部来投奔叔父。到了这里之后才知道，
就在她来投奔叔父的路途中，叔父已被劫匪抢走了黄金，连性命也丢
了。但是，她没有退缩，在这个无法无天的乡镇里，勇敢顽强地生存
下来，以教书育人为生。社会逐渐安定后，她嫁给了当地的一位木材
商人。不幸的是，丈夫和他们的子女都先她而去。此后，她便与孀居
的曾孙女及其小孩住在一起。到了八十多岁时，她几乎已经耳聋眼瞎，
便成日坐在打字机前，写她自己的回忆录，努力把乡镇的百年风云记
录下来，尽一份心意。象大多数作者一样，老人家不准曾孙女看她那
些没有完成的手稿。
　　平凡无奇的光阴就这样度过着，一晃多年。直到眼前，曾孙女突
发重病，住进了医院，她的小孩被亲戚接去临时照看，经济上越发捉
襟见肘，但老太太坚持自己独居，也不受任何人的接济。直到实在过
不下去，她才摸索着，登门拜访莱里，这位她印象中本镇最杰出的经
纪师，询问莱里，是否可以帮助联系书商，请他们看看自己写的回忆
录，有没有出版的可能。老太太年轻时，经常给旧金山洛杉矶这样的
大都市报纸投稿，描写乡镇生活，也算小有名气。不过，已经歇笔二
三十年，而且，那些编辑们也早已告老还乡，人去楼空了。
　　莱里接过手稿，只翻了一翻，顿时犹如芒刺在背，无地自容：是
啊，老太太这么大年纪了，还如此顽强地生活着。自己呢？小时候还
去老人家的院子里偷过桃子吧？幸亏老太太老眼昏花，根本看不清眼
前这小子目前的穷途潦倒，还这么信任他。莱里还有什么好说的呢？
莱里没话可说，大声告诉老太太，他立刻进城，去找他的老朋友。
　　接下来的一切，顺理成章。莱里从城里回来后，高兴地告诉老太
太，他已经找到了老朋友，他们答应修改后出版，并且，情愿每月支
付一百美圆的定金。老太太喜出望外，立刻把曾孙女的小孩从亲戚家
接了回来，用自己挣来的稿酬抚养他们，包括支付曾孙女的住院费。
一百美圆，在当时，这是很大的一笔收入呢。同时，莱里也倍受鼓舞，
恢复精神，努力工作，拼命挣钱。小镇的乡亲们高兴地说，“能干的
莱里”又回来了。
　　只是，老太太毕竟年事已高，没有来得及看见自己的书出版。我
听说，老太太的那本手稿，仍然存放在印刷博物馆里。只是，它根本
无法阅读，许多段落都漏掉了。老人家既看不见，又听不见打字机走
到末尾时发出的提醒铃声，以至于大段大段的篇幅中，只有堆积的凝
重黑墨。当然，事后，小镇上的人们才逐渐明白，为什么莱里玩命挣
钱，还仍然衣衫褴褛，狼狈不堪。
　　在欧美，人老不服老的现象随处可见，并成为社会推崇的美德。
男士老当益壮，不是继续工作，就是热心锻炼。冬季的华盛顿分外寒
冷，但是，依然可以看见这些老年人在冰天雪地里奔忙，活跃在各种
场所。最难能可贵的是，他们并不勉强，而是精神十足。许多时候，
我们年轻人都感觉疲倦了，而老人们除了一头银发，根本看不出他们
已经退休或年岁已高体力不支的样子，始终兴致勃勃。老太太们呢，
体态虽然一般比较老态龙钟，但依然打扮得花枝招展，漂漂亮亮，比
年轻姑娘更加喜爱鲜艳的色彩，浓烈的化妆，精美的服饰，讲究的搭
配，周到的礼仪。她们并不谦卑，不甘示弱，拒绝寂寞；相反，花样
年华的姑娘倒绝少艳装，大多素面朝天，最多也只是用一点唇膏蔻丹，
画龙点睛。如果走过妇女服装铺面，留心看一看，在那里热心选购艳
彩时装的，大多是老太太。年轻姑娘对衣着似乎比较随意，更相信自
己的年轻所特有的秀色可餐。因此，总之，感觉无论在哪个年纪层次，
中心思想只有一个：青春艳丽，永不凋谢。
　　星期六上午，我用完早餐，准备出门去，乘地铁，前往博物馆。
纷纷扬扬的雪，下了一整夜，地上落满了厚厚的积雪，路边的常青树
木绿少白多，空气寒冷而清新。酒店门口的服务员热心地低声叮嘱道：
当心脚下路滑。
　　美国人通常以车代步，酒店门口，建有方便多辆汽车同时出入的
宽广斜坡。服务员看到我要步行，当然是提醒我，在这个坡度上，虽
然不停地打扫，还是免不了结了层薄薄的冰雪，千万要小心，脚下留
神。我停住了脚步，准备和他寒暄一下，感谢他的提醒，这才注意到，
服务员在对我说这番话时，眼睛却没有看着我，而是越过我的肩头，
眺望着我的身后。我顺着他的目光，自然而然地转过身，一眼就看见，
外面，有一位老太太，正在小心翼翼，步履蹒跚地，走回酒店。她满
头银发，身着一件鲜红色的滑雪长大衣，足登一双防水越野软靴，头
上和肩膀，已经粘了些雪花，皮靴面子也有些潮湿的水渍。看起来，
老太太是酒店的客人，一早就出了门，自己去雪中散步了。从她的神
色上看，这个时候，显然已经疲倦，力不从心，只好返回酒店了。
　　我顿时明白了服务员的心思。他正为这位老年妇女客人担心呢。
担心她到底能不能自己走上斜坡，不至于滑倒，安全地回到酒店。虽
然不过几十米的距离，但是，对这样一位筋疲力尽的老人家来说，恐
怕还真有点艰巨。
　　我还明白了服务员好意提醒的另外一层含义：作为门童，在这样
的天气条件下，他有义务向每一位步行出门的客人打个招呼，提个醒。
可是，如果客人坚持，一意孤行，万一有个三长两短，就怪不了服务
员了。他之所以这么和我说一声，恐怕也是担心万一老太太摔倒在大
堂门口，论起赔偿来，我也可以为服务员做个旁证，证明他的确没有
失职。
　　我也明白服务员的难处。欧美人不服老，已经司空见惯，习以为
常。我也遇到过类似的尴尬。在某些场合下，不由自主地，想帮助老
人一下。可是，经常被老人断然拒绝。即便陪同熟悉的欧美来宾，他
们也经常不给这种面子，仿佛只要打算搀扶他们，就有损于他们的公
众形象。哪怕就是机场迎送，不要我帮助提行李，而总是说，让我自
己来，我自己可以做。因此，我懂得这位服务员的心思。他担心，如
果过去迎接老太太，会破坏客人的自尊心。毕竟，好不容易，她才创
造了这么一个雪中漫步的独行记录，也许，这就是她一生中最后一次
了。既然这种努力已经接近尾声，为什么要让老人家功亏一篑呢？
　　但是，这个时候，我也突然明白了服务员提醒我的深层内涵：这
鬼东西，也把我拖入了面前的沼泽：难道我们这两个大男人就眼睁睁
地瞧着老太太冒这么大风险，在我们面前摔一个大跟头么。怪不得，
这个时候，服务员才看着我，脸上带着些微机敏的样儿，有些狡黠。
　　既然服务员有求于我，我没有退路，只能前进。所以，我迈动脚
步，朝外走。动作猛了些，一个趔趄，地上果然很滑，地皮已经冻透
了。老太太正在颤颤巍巍，一见我差点摔跤，也禁不住叫了一声：哎
呀！我自然抓住机会，顺水推舟，说了声，真滑啊，就站了下来，问
候老太太。老太太果然已经不想走了，也站下来，问我，上哪里去。
我回答说，要乘地铁去，不知道外面的路面好走不好走。老太太看着
我，想了一会儿，才回答说，外面路上的雪还没有来得及扫，很难走。
估计清扫工这会儿正在什么地方，忙得不可开交呢。我犹豫起来，沉
思着，随口说道，应该说，没有问题，我还年轻。老太太看着我，说，
是啊。我说，不过，我这鞋，恐怕禁不起，会弄湿的。美国人最怕伤
风感冒和咳嗽，老太太立刻说，不值得，你为什么不叫辆出租车呢？
我说，应该这样，或者，也许应该回去坐一会儿，等雪扫掉，至少，
不要把鞋弄湿吧。
　　于是，很自然地，我彬彬有礼，做了个动作，请求她让我陪伴同
行，返回酒店。老太太当然不会拒绝，只是，在我挽起她臂膀的刹那
间，她似乎从前面大门口那位服务员远远传递过来的微笑看出了名堂，
有些明白了我们这两个男人的伟大动机，立刻甩开我的手。不过，至
少，有我在她身边，这样，我们就放心许多了。
　　也许，老太太和我走到大门前时，脸色和动作都不十分自然，服
务员没有说话，咕哝了一声，就垂下头，飞快地为我们推开门。老太
太果然有些生气，生硬地对他点点头，就不辞而别，走向自己的房间
走廊，消失在幽静中。
　　第二天。早餐时分，我在餐厅又碰见老太太，连忙点头，打过招
呼。我想，需要解释一下，便征求她的意见，是否可以与她同桌进餐。
她立刻问，那是谁的主意啊？我说，不是他的主意，只是我们民族的
习惯，尊老爱幼。所以，包括我的父母兄弟，他们都会要求我这样做。
否则，对我们来说，就是十分不得体的事情，就会受到良心的责备，
并且，这种痛苦，要持续一生的。
　　老太太端起麦片碗，一点一点地，把麦片倒进热牛奶里，搅拌着，
沉思着，然后，才说，我们不是这样的。我急忙分辨说，没有必要坚
持啊。有时候，接受帮助，也是一种慷慨，以便让对方尽一点心意。
　　老太太严厉地看了我一眼，才徐徐地说，年轻人，你需要的是，
理解。我想，老太太一定是指对民族风俗差异的理解，就说，我理解。
但是，在中国，照顾老人，是全社会赞许的美德，古老的美德。老太
太说，不，不，你需要一个更合适的理由。你应该说，你是一位男士，
而我，是一位女士。
　　我们相视一笑，起了身，愉快地走出了餐厅。当然，自然而然地，
我搀扶着老人家的手臂。大厅的地板一点儿也不滑，这举止应该更算
是公共场所中的礼仪。我开始喜欢老太太这样的提醒，这样的解释，
这样接受搀扶的理由了。临别前，我就如实对老太太这么说。不料，
老太太先是一怔，然后，认认真真地，端详着我的脸色，末了，才说，
你真没有想到，以这样的名义，可以更容易被一位固执己见又有些虚
荣的女士所接受？我诚恳地说，真的没有，请原谅我。老太太突然由
衷地高兴起来，微笑布满了脸庞，意味深长。她看着我，目光慈祥，
然后，痛痛快快地说，不过，年轻人，你刚才介绍的，古老的中国，
淳朴的传统，也很有道理。嗯……，的确，有时候，接受帮助，也是
一种慷慨。
　　我感谢老人家的理解，并且，从她意味深长的神情中得到了启发，
大步流星，返回了大门口。未等我开口感谢门童事后的补台，他已经
必恭必敬，抢先对我说，昨天的事情，非常非常感激你，我尊敬的先
生。我会心一笑，答谢道，后来，你对老人家解释成“女士优先”？
这说法也很好。我们就算中西合璧吧。
