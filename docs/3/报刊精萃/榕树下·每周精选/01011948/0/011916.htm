桃花前渡

○浙江　秦桑儿
　　　　　　 


　　光流托着木筏，她恳恳哀求篙者让她记住献之，依稀也好。回首
桃花万里路，她不顾一切撩取绯瓣，裙裳手掌尽是，痴痴不落。篙者
黑衣深裹处掠一丝苦笑：“何必如此，真是，徒劳。”于是点头，渡
她入今世。
　　　　　　　　　　　　　　　
　　应邀参加晚会，只因单子上有陈献名姓，三月暗夜踏过砂石小蹊，
千树桃瓣委风而散，有另一番轮回的决绝。风萧萧兮易水寒。托桃花
于手心掌纹，进门。灿灿灯光碎水晶珠般叮叮当当落在身上，霎时厅
内噤声。她着沉朱底遍绣紫金浮花印度厚缎大撒裙，同色缎带缠绕盲
黑发辫，清寒素脸仅一抹唇点得血红欲滴，看上去莫名的庄肃妖气，
正好CD放Loreeda “The mummers' dance”，加上她，一派阿拉伯
风的繁奢淫逸。也不说话，就这么径直走过去，浅靥斜睨陈献。陈献
一愣，仿佛一些绰绰回忆，欠谁的。但毕竟是社交公子，微笑，吻手，
递过酒杯，优雅姿势，道，“见到你，才知道惊是怎样惊法，艳是怎
样艳法。”后来她想，当时就注定了，只是关于陈献无数流言中的小
小一缕罢了。惊是惊，艳是艳，然过于传奇，“传奇没有一个是有好
结果的。”像盛开得太辉煌的桃花。而那时她亦不知为何，只不顾一
切要接近，哪儿管得瞻前因顾后果。倾花入杯，漾一漾，还陈献，澄
冽白葡萄酒里一枚凄红。无语，傲然折转，裙裾旋开，异香细细。陈
献方寸微乱，从未见这种近于做作的风格。她。
　　回家，打灯，极好的朋友挂电话，劈头一句：“看到你了，必要
吗？”撩开纱帘，落地窗外灯火焕然，酒中冰晶，烟里乳香，好一场
世纪末的华丽。舍得放下？停了一会儿说，“我也不知道。我想，陪
他一段。”朋友放下电话，素来看到是她的不羁，原先也想好是要吸
引陈献陪她的，她在暗处；可是现在，主角和导演全换了，她一个人
暴露于聚光灯下，凄美却，孤单。必晓利害关系，而头脑清醒地自投
罗网。他知道无法再拉她。宿命的故事，他的力量，无法阻止。
　　从此往后，她把自己盛开给陈献，完全不再给自己留条退路。罢
了罢了。
　　她的时装展示会极成功，也热闹，云鬓花颜金步摇，雀啼莺声的。
几个记者过来称赞她巧妙处处描下桃花影痕，三月春色都在衣香襟褶
中。她远远衔着陈献身形穿梭，施展手腕，和着笑，心底却明澈清冷。
握不住的。可是，她宁可活得像陈献那些衣服披得少，想得也少的女
朋友们――一时狂欢。入夜，人散后，一钩新月天如水，华裳凌乱，
这样妖冶的废墟。她排排关掉灯，独自靠墙吹尺八，一叠二叠三叠，
流水落花春去也，握不住的。
　　他过来，圈她入怀：“一直以为你很异国的。”她暗想，他所要
的异国，恐怕是投入时纵情，分手时理智，如此轻松现代的一则艳闻
罢了，像两只气球，再亲热也是贴贴肌肤，又会超然分开的。而她一
旦玩真，便是要风筝般缠绕到底，结发绾袖的那种。于是深深看到他
眼中微小的闪躲，便问：“你视我为何？”那是她唯一讲过的责备话。
他脸上一僵，只不说话，搂她更紧。她心里一抽一软，疼他到怕他受
一点儿委屈的地步。
　　踏青之月，拉他去野村看桃花。东风过处，万千桃瓣纷落游荡，
犹若她心恍惚。陈献亦小怔，随即振作孩子脾气，一把蒙住她眼：
“说，桃花是什么。”本是玩笑，她的心却慢慢沉静下去。说：“表
面喧闹哗嚷，开得轰轰烈烈，云蒸霞蔚的，因为有恐惧，盛开得便放
弃似的格外不顾后果，每朵桃花都绝美而寂寞，每朵花都是想结果的，
却永远是宿命的牺牲，至多在扇子上点染几滴，勾勒渲染一个故事罢
了。”仅仅作为个可有可无的故事。
　　她早已知道无路可退，可没料到竟是陈献给她退路。直至，她参
加陈献葬礼，直至她知晓陈献竟然、最终死在女友刀下。她在葬礼上
看到他无数女友；她在法庭上看到那个女孩，脸色明净平宁，不起波
澜。转身离去，她没想到也有人将这种爱发挥到极致，只是，女孩杀
了陈献，她杀自己。
　　后来，她每天带陈献骨灰在贴身包中去跳舞，灯色惝恍陆离，音
乐沸反盈天。仿佛，昨日重回。
　　那天夜里闹腾得晚，开门，站在花园树影森森，茉莉细疏里，看
楼里的灯光，和突然出现笼罩一切的静寞，就有那么一种灯火下楼台
的味道。歌笑篱落。千里长亭，都没有不散的宴席。她紧紧握着寂寞
走过去。兰叶碰臂。轻轻上楼听朱哲琴《吹箫人》：在边缘的边缘，
在尽头的尽头。那些倒影中的倒影，回声中的回声，无限中的无限；
那些真形和灵幻，实在和荒唐，或然和虚无。朋友临窗执箫。她以为
已经看得开朗。但箫声思慕微微，是他处于时间之外的淡漠与冷嘲。
看得开？放得下？眉间心上？席地而坐，他微笑发牌，关于迷宫、游
戏的纸牌交叠，象镜子一样，重叠起来有无限的可能性，奇怪的是，
也有无限的相似性。
　　那些周而复始的轮回。
　　看他黑色长风衣散拖在水磨石上颇有巫师之风――掌管时光的巫
师。她心里一凛，有些不自在，有些不想深究的记忆，便不理他，自
顾翻书，“岛村不知为什么，很想再强调一声‘完全是一种徒劳嘛’。”
旧年的桃花瓣刚好夹在川端的《雪国》里，句子下面重重划条黑线，
是他手迹。
　　当头一喝。
　　忽然那些时日都回来，光影细细：落红成阵半掩门。渡口她送献
之，湘裙六幅染苔痕，潭水千尺尽桃花。献之拈花入她袖，笑语：
“艳色沁瘦骨，归来尚馀香。”皂葛渡人唇角一丝淡笑掠过。这一等
便是数百年。花瓣像隔年桃符，风蚀雨侵黯了颜色，那抹菲薄微红却
一如她临逝前执著牵恋。
　　冥河，时间在船舷边哗哗流过，黑衣人一再低语，何必如此，这
完全是一种徒劳嘛。
　　徒劳！她惊起抬头。正视黑衣深处的笑容。终于豁然。呢喃，
“到秋日才懂春天的花，这完全是，徒劳。”声音里枫叶荻花寒意瑟
瑟。他见她端然捧出骨灰盒子，扬灰入风。压花瓣的书页，余几点残
脏斑迹。他握书，昂首兀自萧萧一笑。
　　　　　　　　　　　　　　　
　　三月，墨服篙者身旁立一玄衣女子，眉目清明肃然。
　　风拂漫河漫天桃花，擦身而落，数枚入衿袖，薄香沁骨。
