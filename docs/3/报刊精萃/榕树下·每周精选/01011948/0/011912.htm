猫妹

○湖北　许映雪
　　　　　　 


　　猫妹是我们家二十五年前喂的一只猫，久而久之被我们视为家庭
成员之一，我们称之为猫妹。
　　收养猫妹还经过了一番曲折。
　　1971年春节，母亲从干校解脱出来回到城里，分到一所条件比较
差的小学继续教书育人，带着弟弟妹妹和我住在小学简陋的宿舍里。
父亲则仍留在干校继续劳动、锻炼。
　　小学操场西边院墙外是省某研究所在文革中荒废的一片试验场地。
田鼠、蛇、黄鼠狼、野猫等动物在这片杂草丛生野花芬芳的土地上和
锈迹斑斑的机器之间按需分配住房和活动场所，大家和平共处，相安
无事。动物们乱中取静，倒也不失为一个桃花源似的好去处。当然，
太平盛世偶尔也有个别不幸事件发生：时常有小动物大胆地从院墙的
残垣断壁中溜进小学，向往到人的领地观光访问，却往往遭到灭顶之
灾，被勇敢的孩子们围歼而暴尸校园。
　　那是在一九七四年春夏之交的一天傍晚。我放学后从小学操场西
边院墙边走过，忽然听到墙的外边有“喵喵”的叫声。稚嫩的叫声忽
高忽低，充满了期盼，也夹杂着失望和凄凉。我那时已十七岁，正是
没事找事的年龄。好奇心使我鬼使神差地翻过院墙去探个究竟。顺着
“喵喵”的叫声，我很快在一个一人多高、尚未启封就无人问津的机
器的木质包装箱里发现了一只黑色的小猫。我把木箱一边的木板轻轻
搬开，这只小猫就暴露在我的面前。
　　原来它并不是一只全黑的猫，在它的胸前还戴有一朵洁白的心形
胸花，四只脚亦白如踏雪。它看到我的到来，似乎害怕，又似乎欣喜，
似乎躲我，又似乎恋我。我轻轻地把它捧起来放在我的胸前，它在我
的手掌里呼呼地呜咽，叫人怜悯，并没有逃走的意思。我本来就喜欢
小动物，看到捧在手里的这只小猫更是爱不释手，也不管后果如何就
把它带回了家。
　　弟弟和妹妹一看我带回来的小猫，都欣喜若狂，争先恐后地要承
担抚养它的责任。可妈妈那一关怎么过，妈妈每天的课是那么多，还
要洗衣买菜做饭，照顾我们的学习，家里再多一个小动物，她受得了
吗？
　　妈妈下班回家看到家里多了只小猫果然不高兴。问清了事情的缘
由后，说：“我们连人都顾不过来，你们还要喂猫，这不是害性命吗。”
妈妈毫不留情，命令我们立刻把小猫送回去。我和弟弟妹妹的苦苦哀
求都不起作用，我们兄妹三人只好依依不舍地把小猫送回原处。
　　我和弟弟妹妹都惦记着小猫，没精打采，晚饭谁也没吃好。
　　吃完晚饭，天快黑了。我洗碗，弟弟和妹妹扫地抹桌子，妈妈一
人出门去了。不一会儿妈妈回来对我说，“大毛，我听见小猫还在那
里叫，你去看看它的妈妈在不在？”原来妈妈也惦记着小猫！我和弟
弟和妹妹赶紧跑到院墙外，顺着“喵喵”的叫声一看，小猫果然还在
包装箱里，那里有它妈妈的踪影！
　　我捧起小猫就往家里走。我们回到家里时，妈妈已经用剩下的饭
菜为它煮了一碗稀饭。小猫大概也是饿急了，毫不客气地吃了起来，
一边吃，嘴里还发出满意的呜呜声。
　　猫妹到我们家时，谁也不知它是男是女，大家都叫它“猫咪”。
在妈妈的指导下我们兄妹三人齐心协力给它洗了个澡。妈妈用旧衣服
给它在纸盒子里铺了一张床，又给它一只专用的小碗，解决了食宿问
题。那时有户口才有稳定的粮油关系，猫妹既解决了食宿问题，就等
于有了户口，开始和我们一起生活。
　　猫妹刚来我们家时还有点陌生和腼腆，走到哪里都有点小心翼翼
的神态。大概因为是我把它带回家，妈妈给它管饭，所以只跟妈妈和
我接近点。只住了两三天，大家彼此熟悉，它就无拘无束。我们放学
刚回到家里，它就欢天喜地跑到你的跟前，一边亲热地“喵喵”叫，
一边踮起后脚，两只前爪搭在你的裤腿上急切地向上爬，以示迎接。
只要你有空，它就会缠着你玩，一会儿煞有介事的把一只乒乓球扑来
扑去；一会儿打个滚逗你；一会儿用两只前爪来拨弄你几下，你要是
还击，它就会四脚齐上，急了还会嘴脚并用，咬得你心里痒痒的。
　　猫妹特别爱劳动，只要是它力所能及的，就决不会袖手旁观。我
生煤炉，它会把我燃火用的纸团和小木柴扒得四处都是，算帮我生了
煤炉。弟弟扫地，它会围扫帚扑来跳去，把扫拢来的渣滓扒散，也算
帮弟弟打扫了卫生。小妹晚上做作业，它会蹑手蹑脚地不知从那钻出
来，跳到小妹的书桌上，端坐在她的书本旁，偏着头，一本正经地看
着她学习。看到它那副认真的模样，小妹常会忍俊不禁笑出声来。不
知猫妹打算如何给小妹帮忙？家里空间小，妈妈要等我们做完功课腾
出桌子才能批改学生的作业，我们做功课时妈妈就抽空织补我们穿破
的毛衣。这时猫妹最忙，它会把毛线团从床的这头扒到那头，有时把
毛线缠作一团，费好大功夫才能理清，给妈妈帮倒忙最大。
　　猫妹特别爱学习。只要是它感兴趣的，它都要用心地加以研究。
它会拨一下乒乓球，然后专心致志地研究球的运动；它会蹲在地上，
聚精会神地观察灯线随风摆动；它会伏在书桌上，细心地琢磨闹钟指
针走动；它会坐在收音机旁，好奇地探索声音来自何处，金鱼缸里喂
的鱼，树上飞的鸟，凡是有生命的，会动的、能响的、有颜色的，都
在它的观察研究范围之列。
　　猫妹特别讲卫生。每天都要认真地洗几次脸，身上的也总是收拾
得干干净净，毛衣永远光泽如新，人见人爱。
　　猫妹有时也撒点娇。父亲工作时我们兄妹三人谁也不敢打扰，唯
有猫妹有恃无恐，敢于趴在父亲的工具书和文稿上悠然自得地酣睡，
谁叫它是猫妹呢！
　　猫妹真是懂事。那时家里生活条件不好，有时烧点鱼肉，不是我
们盛在它小碗里的，放在桌子上它连看都不看一眼，偷嘴的事更是从
未有过。猫妹越是懂事，我们越是喜欢它。
　　猫妹也有点傲气，每次都是妈妈把饭添在它的碗里，喊它，才肯
享用。嗟来之食是决不领受的。
　　猫妹在我们家把它天真活泼，淘气调皮的天性表达得淋漓尽致。
它不仅给我们家带来不少欢乐，也给我们带来切实利益：自从猫妹来
我们家后，以前常到我家拜访的老鼠们立即销声匿迹，衣物书籍，赖
以保全。
　　可是，由于生活所迫，猫妹还差一点儿被过继给别人。
　　那时父亲还在干校，我们兄妹三人又都是长身体、正能吃的时候，
而每月的粮、油、鱼、肉、蛋、豆制品都要凭票供应，日子过得紧巴
巴的。妈妈总是想方设法尽量保证我们的营养，其实哪里有什么办法，
无非是从她的口中省给我们而已。家里一个月里煨一两次排骨汤，也
是学孔融让梨，互相谦让，平时三餐基本上是粗茶淡饭。现在每谈到
当年的生活，妈妈总是为此伤心。
　　妈妈有一位同事家境优于我们，苦于鼠害很想养一只猫，她很喜
欢我们的猫妹。她不好意思说要，委婉向妈妈提出来借猫妹去她家灭
鼠。猫是食肉动物，猫妹跟着我们粗茶淡饭绝少鱼肉，长到三个月，
骨骼高大却颇清瘦。妈妈看着猫妹在我们家过着如此清苦的生活，心
里十分难受。如果猫妹在这一位同事家吃得好一点，也是给它一条生
路。
　　猫妹被带走了。没有猫妹活泼的身影家里顿时冷清了许多。猫妹
的小碗洗得干干净净，孤独的放在碗柜旁，留给我们许多关于猫妹的
记忆。这次我和弟弟妹妹没有抱怨妈妈，只是希望猫妹在别人家过得
比在我们家好。
　　半个月过去了，我们渐渐地淡化了猫妹的故事。
　　一天夜里我已上床正要睡觉，忽然门外传来熟悉的“喵喵”声。
是“猫咪”！我赶紧起床开门，果然是它！只见它满身尘土，疲惫不
堪，妈妈同事的家离我们家有两公里之远，真不知道猫妹是如何找回
来的！弟弟妹妹闻讯起来，看到猫妹回到家里都非常高兴。妈妈赶紧
给它盛了一碗饭，猫妹狼吞虎咽，一碗白饭一会儿就吃得干干净净。
　　第二天清早，妈妈看到在猫妹的小碗四周呈放射状整整齐齐摆着
六只老鼠的尸体。妈妈很感动，说，猫咪是用这种方式告诉我们，它
会帮我们做事，不愿意离开我们。
　　从此以后，无论我们的家境如何，猫妹都是我们家庭不可缺少的
一员，直到它生命的结束。
　　后来听妈妈的同事说，猫妹在她家食必有鱼，待遇颇优，可它食
不甘味，卧不安席，就是要回家。孟子曰：富贵不能淫。猫妹实践了
这种精神，叫人如何不敬重！
　　猫妹会自力更生改善生活。鼠、鸟、蛙、蛇等小动物都是它的佳
肴，又知道妈妈十分讲究清洁，这些猎物从不带回家来。所以虽然我
们家的生活条件并不好，可它还是长得非常健康。形体丰满，皮毛黑
的如缎，白的似羽，目光炯炯，行动矫捷，与其说它是只猫，不如说
它象只小老虎。
　　尽管猫妹长大后也有它老虎般的个性，可在家里人面前却十分温
顺，十分热情，从未对谁发过脾气。不管是谁回到家，它都会跑到你
的身边蹭来蹭去，表示欢迎。你要是不理它，它会委屈地“喵喵”抗
议。夏天它会殷勤地伴你乘凉，冬天它会热心地钻进你的被窝给你焐
被子，可你还没睡着，它却早已“呼噜呼噜”地先去见了周公。
　　猫妹在家里人面前非常温顺，可它在外人面前却表现得十分清高，
谁和它打招呼它都不理睬。妈妈的同事和学生到我们家来，看到它都
很喜欢。可是没有一个人敢逗它：不要说被它咬一口，就是被它锋利
的爪子抓一把就受不了。从它喉咙里发出的低沉地吼声，足以使任何
一个对它不怀好意的人感到害怕。
　　时间一晃就是两年多，不知不觉猫妹就做了母亲。事情来得很突
然，晚上睡觉前看猫妹和平时并无两样，第二天醒来，猫妹的床上就
多了四只干干净净、胖乎乎、漂漂亮亮的小猫崽。这时我们才知道它
是女性。全家人都为猫妹做了母亲而高兴，妈妈还专门为它买鱼煮鱼
汤，表示慰问。父亲看到妈妈专门为它煮鱼汤，笑着说它的地位挺高，
仅次于小妹，今后就叫它“猫妹”好了。猫妹在我们家的身份和地位
由此而定。
　　我和弟弟先后高中毕业，都经历过上山下乡，到农村接受贫下中
农的再教育。七七年恢复高考，弟弟妹妹和我都先后考上大学。父亲
也得以平反到大学教书。平时我和弟弟妹妹住在学校，猫妹独自在家
和爸爸妈妈为伴。周末全家团聚在一起，吃完晚饭后看电视聊天，此
时猫妹就格外地高兴，常常趴在我们的肩膀，在我们的脖子上亲昵，
其乐融融。
　　修短随化，终期于尽。猫妹渐渐老了，不爱活动，饭也吃得少，
还经常生病。一九八四年九月最后的一个星期六晚上，猫妹终于安详
地在我的床上走完了它的生命之旅。我们全家人在一起，用一只木匣
把猫妹安葬在我家院子里的桂花树下，让它回到大地母亲的怀抱，和
大地母亲溶合在一起。爸爸还为猫妹写了墓志铭，铭曰：“生尽所能，
死留其德。人不如此，宁无惭色。呜呼猫妹，可以安息。”从那以后
我们家就再也没有喂过任何动物，因为我们相信，每当桂花飘香的时
候，猫妹就会回到我们的身边。
