守望爱情

○辽宁　云在心空
　　　　　　 


　　今夜以什么样的方式开始，又以什么样的方式结束，这已都是最
后的一次，生命中最后的一次。
　　彼情已逝，我在记忆的长河中徜徉，携一颗洁净的心于你的梦里
缠绵——生命中的最爱，生命中的唯美。
　　向异乡深处招摇的姿势和异乡的遥远灯火一样地飘渺。我还要在
鸟鸣过后一如往昔地坚守着这空洞的文字。传说中冬天的大海和北国
的飘雪于无尽的守望里总会有一个交错的节点，思念的心底倏忽间便
点燃起无数阑珊的灯火。此刻你在我的守望的召唤下轻盈地走来，不
期然地似曾无比熟悉的海洋气息再一次地将我紧紧地包裹。
　　而今夜灯光柔和得叫我好心痛。耳边总是传来你那恍如隔世的歌
声；你所有轻盈的身影和最后的祝福都成为季节交替的风景，在时间
的摆渡下渐行渐近，重新回到我的身边，到海边重拾几颗七色的贝壳。
　　再一次地搁浅自己以及与之有关的心情。在些些许许的往事里，
在你轻轻浅浅的笑容里，今夜我再次地不可避免地迷失我自己，交付
我自己。
　　蓦然回首，往事如烟如云，随风而逝。不经意的风，轻轻地吹动
了那串窗前悬在头顶的紫色风铃。于是，我的心开始在祝福和思念间
摆个不停。摇落了一串串叮咚作响的往事，声声地扣击我初醒的灵魂，
在北国的冬天精心地呵护一片轻薄的雪花，不愿再看到你因为怆然离
去而跌落的泪花。
　　找个理由总想重走那段来时的路，在每一个黑夜来临的时候，把
自己交付到往昔痛彻体肤的回忆之中，顺理成章地期待着、无眠着。
而此刻，我情愿地再度跋涉在情感的泥沼里，丢掉任何苍白无力的理
由，任凭自己无力地憔悴。陪你经历那个生命中最庄严、最心碎的时
刻。
　　还记得吗？你伸展的舞姿，在我的眼前停滞；还记得吗？万家灯
火时我点燃的最后一支烟已悄然地熄灭；还记得吗？蓝色的“心”海
里，末班的客船已经没入那边的天；还记得吗？生命中这最后的最后
的一次。
　　我虔诚地触摸在生命两端摇摆不停的紫色风铃，满山火红的杜鹃
和家乡成熟的麦子，温饱着我们远隔万里的牵挂。记忆的时空里盛开
着如血的花朵，不经意间有一种力量悄然地涌上心头，让我于今夜的
此刻禁不住地泪流满面。
　　你会看到在那个虚幻的天堂里有心雨飞扬吗？！
　　那是你梦想中的“国度”吗？正如我心底深埋着永生不泯的期待。
　　那蜿延的山路；那潮汐潮涨的大海；那携手相牵的日子；那最后
离去的凄美的轨迹，还有你那最后的一个婉然的身影，生命中最惨然
的微笑，可曾被摇落，在心底跌荡。
　　可曾摇落，七色贝壳里彩色的梦想；可曾忘记，日渐趋短的夏天
里爱情无根地疯长。
　　回忆夏日大海的风情，我站在北国的冰天雪地里漠然地冷视着一
辆又一辆从身旁碾过的汽车，心灵深处骤然地响起一个轰鸣的爆裂。
　　我是一个流泪的雕像，在这个城市的边缘孤独地流浪。热烈而孤
独！
　　你还好吗？我们的爱情还好吗？
　　我的诗行曾因了你的到来饱满而丰富。
　　我的诗行也因了你的离去瘦小而哀伤。
　　两年了，我的心在情感的旋涡里不停地打转。
　　两年了，我的思念悬挂在冬天里寒冷的枝头。
　　今夜，请你带着欲望归来；今夜，让你再一次地轻轻地抚摸我思
念里悄然生长的发梢。
　　此刻，任紫色的风铃再次地摇摆。咀嚼崭新如故的记忆，让所有
的诅咒、泪水和思念，将往昔的日子和所有与爱情有关的影像修饰得
不再拙劣和粗糙。但一切又都已如云影掠过，扶摇直上的只是缭缭环
绕的青烟不散。
　　当遐想的触角伸向四角的天空，再次地噙着流不尽的泪水，一读
再读依旧如昔的你，一直是我想和的情诗；到头来生活在现实的无奈
的世界里，我还是躺在残缺的边缘，独自地啜泣。
　　我那将要把持一生的爱情，就这样在那个停滞的时刻变得虚无又
飘渺，逐渐地隐没在无边的黑暗的世界里，苍白无血。留在被时空的
尘埃於积的记忆里，如火般地燃烧，接着便堕入窒闷的黑夜。
　　今夜，我将躺在那段用心浇注的落花时节里——用心守望来时的
爱情！
