审美的可怕

○（广州）　亚雄
　　　　　　 

　　
　　审美这个话题，可大了去了，不知有多少人谈过。我在这里拾人
牙慧，很有点不好意思，想不理这事了，可天生是那种爱凑热闹的料，
而且眼见关系这话题的事儿又那么多，我这人又刚好与艺术扯上点缘
分，就只好瞎说一气了，管别人怎么看？
　　我现在住的地方，是广州一个有名的城中村，叫什么名我就不说
了，反正，在广州这样的“村”多的是，不过我住的那儿有点特别，
它离那栋出了名的全国第二高大厦近在咫尺，我每天早上起床，都先
从窗里往外看一眼那大厦的芳容，以证实一个现代中年人正身居繁华
都市，以为如此一来，一天会过得充实。但是这一眼就来了问题。我
在看到这栋叫做“中信大厦”的楼宇时，还连带看见一些很讨厌的建
筑，我所在的位置就在这一群建筑当中。这些小楼，栋栋面目都一样，
外墙都是用细长条浅白瓷片贴面，楼顶一律有个天台，天台上有个方
型小储水池，若不是这些楼有七八层高，它们看起来就像厕所。我住
这儿虽然来气，但图它个人气旺，偶尔还能看到来自五湖四海的人打
架，所以心理还能平衡。
　　中信大厦据说是日本人设计的，去年我在市长大厦上班时，很有
些为身在这大厦而自豪，但有一天中午抽空光临旁边的中信大厦，眼
见进出大厦的白领昂首挺胸，不觉心虚。且不说别的，光是人家那电
梯，就宽大豪华，习习生辉，设计很见功力。我不明白，广州媒体为
何提起中信大厦，常连带提市长大厦，好像两栋楼同一档次似的。现
在我看明白了，它们的差距就像孔雀与鸡的差别。转回说民宅。我跑
的地方也不算少，诺大的中国，民宅建筑模样竟惊人的相似，好像起
楼的人通了气似的。我去过小国家越南，刚从我们的地面上跨过去，
没走几步就令我大惊讶：人家越南穷是穷，可普通民宅有模有样，那
大幅的山墙竟敢大片涂黑！可配上灰色的瓦顶和别致的铁拦栅，就是
透出一股子大气、洋气。我听到身边也有人赞叹，但叹后来一句：那
么黑的墙，是不是死了人。我看他不像开玩笑。难道中国人对黑那么
忌讳吗？为什么没有人对他的穿黑裙子的女朋友说：嘿，穿这么黑干
什么，死了人似的！至于像在北京天安门附近耸立起一幢幢高楼大厦
这样只顾“小美”不顾“大美”的事例，在中国可就太多了。使我感
到迷惑的是，小百姓不懂审美就算了，连建筑设计师也不懂，那就说
不过去了。
　　现代人，房子、汽车是人生追求的两大件，说审美，不能不提提
汽车。为什么本田、别克这类新开发的中高档车那么抢手？具我这个
汽车爱好者了解的情况，一半以上的原因是“外观漂亮”。看看另外
那些满街跑的国产车吧，长的什么样啊！既然售价十几二十万，为何
不在外观上多下点工夫呢？总想一上来就猛赚一番，美观的事儿以后
再说也不急。是的，这些样子丑丑的车当初之所以好卖，是因为没得
选择。现在好了，销量明显跌了下来，你降价吧，降价人们也不想掏
钱，WTO 马上就要进来了啊！像房地产，七八年前人们掏钱买房，有
个窝就觉得挺好；才几年就不满足了，要看看地段和物业管理；现在
倒好，物业管理这一条是基本的，开始追求“生活方式”了。也就是
说，你房地产发展商在想做这个行当的一开始，就要想好你要搞的房
子是给谁住的，你不要又想给白领住，又想给老太太住，又想给二奶
住，想给谁住就一个劲地盯着谁，别见异思迁。你有多少套住房？一
个楼盘不就是几百上千套吗？买你房的人占近千万的城市居民的比例，
不过是九牛一毛。所以现在广州的房地产商精得很，什么“健康”啦、
“山水”啦、“花园”啦、“公园”啦、“蓉树园”啦、这个“家”
那个“家”啦……愣是人为赋予房地产一种生活方式的理念，你懂也
好不懂也好，反正抓住了人爱做梦的特点。但不管怎么地，这些房地
产都有一个共性，就是高档、漂亮，注意内在美和外在美。我最近进
一抢着买的楼内，我的妈，我以为我是来到中信大厦呢，这也是人住
的？但一套套都超过一百万，买的人你挡也挡不住，人们疯了？是的，
疯了，那是被美逼疯的。我要说的意思就是，不要以为汽车便宜一点
就可以忽视外观的美，你高漂亮一点，钱来得更多，所以我又不得不
说，中国的商人也没聪明到哪里去。
　　广州算是大都市了，在中国怎么说也数前三名吧，可是走在广州
的大街上，看一看满街的人吧，穿得多么老土！特别是男人。开始我
想，大概八成的男人不会穿，可是有心的我就是不怕累，在一个假日
里，站在代表广州最现代的天河区的街口，做了一次调查，调查的结
果远高于我的估计：三十个人里只有一两个会穿，这里包括了女人。
也许我挑剔一些，但是为什么不挑剔呢？这儿可不是内地的城市，它
可是大广州啊！我指的“会穿”，并非是料子高档，而是协调。关于
这一点，我们老老实实地向外国人学一学吧。怎么学？有空你到麦当
劳，买一个两块钱的冰其淋，坐在你的位置上观察一番，来来去去许
多外国人，无论是欧美人也好，日本人也好，阿拉伯人也好，甚至印
度人也好，总之他们穿得就是不一样，哪儿不一样？细细看去吧。
　　在穿方面，我得承认，上海人是懂的，他们对衣着的搭配，似乎
天生就会，但是他们有点过分讲究细节了，多了一份紧张，少了一份
自然。北京人的穿着是大气的，在这点上，这个城市的男人比女人要
好。休闲装比较合适广州人，但有些广州人穿着休闲装看上去怪怪的，
细看原来他们的脖子、手上戴了几件闪闪发光的东西。重庆、成都的
女人长得好是公认的，但她们为什么老爱穿黑衣黑裤黑裙？并不是说
穿黑的不好，而是太多的人这样穿就显得俗了。穿，这件事是很奇怪
的，穿得对头，会使一个长得不怎么样的人也变得怎么样。有一次，
我参加一个学术研讨会，与会的人中，有几位港台人士，穿着就是不
同，特别是一位五十来岁的台湾美术评论家，他个子不高，敦敦实实
的，头发花白，戴一副眼镜。他穿一件暖调子的粗麻纹休闲西服，裤
子是暗灰色的灯芯绒，鞋是粗质的圆头泥黄皮鞋，鞋上有外露的粗麻
线，这也不算什么，妙就妙在他的衬衣。在几天的会议中，他没换外
装，而是每天换衬衣。他的衬衣都是厚实的有暗格子的那种。这样，
这个人就变得有份量了，出众了，整个人既稳重有富有情调，很适合
他的体形面貌和职业形象。在那次会上，我们的主持人是一个有声望
的业界人士，也有五十来岁吧，但他在五天的时间里，自始至终都穿
着那套灰不溜秋的低档西装，他人长得高，站得也挺，但是西装裤脚
短短的，尖尖的皮鞋积了灰尘，袜子松松垮垮，坐下来时，里面的黄
颜色毛裤露出小半节，可以清楚地看到毛裤套在袜子里，这样还不行，
他在脖子上还围了一条后围巾！他这个形象连我都感到丢人。我想，
他代表着中国的艺术精英，他应该有更好的形象才对。
　　美，之于人们的生活真是太重要了，如何审美？应该怎样学会欣
赏？中国的教育体系好像不把这个当一回事儿，只知道向孩子灌输认
为将来用得着的东西。中国的家长也极少关心孩子学会优雅的，宁可
花大钱和孩子吃大餐。怎么坐？怎么站？怎么说话？怎么倾听？怎么
走路？怎么布置房间？甚至怎么吃饭？至于怎样欣赏一幅画，怎样欣
赏一支曲子，怎样发现事物的美，不说家长，就是学校校长也不会去
理会这些的。我曾与一位作家参观画展，在一幅幅西方现代绘画面前，
这位作家直挠脑袋，说，这个是不是像一个人？这个是不是像一个什
么动物？哎呀不懂了，搞写作可以，画就不懂看了，尤其是抽象画。
我以为作家应该不至于是这么个欣赏水准的，艺术的审美其实是相通
的，为什么非要从画里看出形象来呢？赵无极的画像交响乐，它在鸣
唱胸中一泻千里的激情；毕加索的画像一匹野马肆无忌惮地狂奔在原
野上；高更的画像悠远宁静的田园诗；米罗的画像稚嫩的孩子在梦幻
的田原中撒野；林楓眠的画像中国端庄的仕女在闲庭小园里弹凑着古
筝……每一个艺术家的作品都反映出这个艺术家的性格特质，我们应
体会看到（或听到）一幅（或一首）艺术品时最初的感受，用心些，
再用心些。
　　我不相信西方人良好的审美是与生俱来的，但是有一点是肯定的，
那就是你不去专门教育，他（或她）也能从环境中获得感染，就好像
我进入中信大厦，自然就捡点自己的行为，尽量把自己弄得像那么回
事。西方的审美教育已经有数百年的历史，在他们国家的空气中，都
能闻出幽雅的气味，所以他们能出奔驰、克莱斯勒、宝马、瑞士手表、
摩托罗拉、诺基亚、法国香水、意大利服装……这些精致的东西。即
使这样，人家该教的也照教不误，丝毫不松懈，这是为什么？一定是
西方人从社会普遍良好的审美中获得了实惠，这种实惠不象销售额那
么好计算，但是几乎每一个西方人都看得到这一点。其实我们中国人
也懂，比如我的许多朋友，他们就愿意买日本的产品，他们说看着人
家的精美的包装和精美的说明书就产生了购买欲望，而且要命的是这
种欲望产生了持续的影响。他们当中一些人现在拽着抄票等着买日本
的汽车。我承认我也是。这就是审美的可怕。
