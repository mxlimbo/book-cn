中国历史：武后暮年

○江苏　柳谌
　　　　　　 


常常一个人独坐到天明
细细数着念珠
好快
一生憔悴
只须掌心捏弄

年轻轻就将长发锁进太祖遗冠
苦苦背着，江山抱负
断了容颜，杀了孩儿
一点伤心说不得
前度姑娘，久做君王事

大殿上百官的嘀咕
二十年前就变调
日渐苍老的太平颂词
声声功过归何处？
唐诗三百首，中有不姓唐

这是季节的错，不是一个女人的
牡丹花儿守不到深秋
梳洗罢，
推了张生筵席
如何渡黄昏？
