美人痣

○浙江　死海印花
　　　　　　 


人说深山里的花比平原上的香气更浓
我们的开始好比一声无心消逝的道别，嗨，你
然后便无影无踪，手已经挥过，榕树下，绯衣人

你披上芙蓉花从般绢罗锦衣的片刻
以非物质的抱负为始，暮后无暇顾及的青春期为终
哽咽的视线一早归去，花影凌乱

可有所谓的玩世不恭者，杀气，艳气一并隐在袖中
遥遥望灯火，看，春江化做梨花细雨
机会主义的忧闷掺和无止境的倒霉，无情亦飘零

呆滞的裸体总让人苦恼，急于堕入爱中的女人
满不在乎地在海滩抽搐般震动的乳白烟气里摇晃
有着有限的野性与顺境中的消沉

花市画灯，穿行其间，死亡和真理均遭你禁忌
当你羞羞答答欲言又止，眼波似水横……唉
体香如故的个中滋味，比真实更缺乏真实性

我们在爱中对抗，补偿问花花未知的无言结局
表面的世界透过屏息的天鹅绒的呼吸绘出
一些化不开的祝福词，无形，安宁

不象死者本人，更不象揉皱的哲理
何日你种植麝香葡萄，泪透的春衫暗解
我从山中来，山中不见伤心色

我扮采花人，采摘沦落的千千结
看见红烛香雾，却寻不着烟中的手，丁香结愁
画眉残脂凝，你依旧不动，又不回答

似乎是个永远年轻的女人，分泌迷人的预谋
某种境况下，秋墨，楼台
种植得更为紧密，后来我们发现雨一直在下

一个春天过去了，一个秋天过去了
敞开的吻，脆弱又神秘
那是场腐蚀性的不幸，你冷吗？在她不经意

的上灯时分，象个焚香的妖孽，忍耐着不值钱的输赢
跪在有毒的幽香中，相信自我繁衍的爱情而不相信生活
憔悴的是谁，恐惧的思维里又是谁，谨慎的时间的变幻

不是歌声，是宿梦中的醉蝶，无心地拍动了翅膀
在这最后的四季，寥落的旧相思
魄散魂飞的七弦桐，胭脂不曾落满小径

幽灵般虹膜上不由自主的倒影，再说一次
香红远去，心依然在入春的花眠夜里徘徊
我，唯一的墨血赫然倾滴在你额眉

几千载的峰回路转，公主和牧羊女永远是正确的
戏衣中的纤手夹在蝴蝶屏风后，莫问，莫问
莫问何处正此时，一切都因唤做美人痣
