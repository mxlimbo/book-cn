吃饺子

○山西　老莫非
　　　　　　 

　　记不清是在93年还是在94年，我到省城出差，碰上了老同学，老
同学做东，又召集了些老同学做陪，请我赴了一场饺子宴。我们去的
那家饭店好象是叫红房子，十分气派豪华。我们先喝酒。服务小姐等
我们酒喝的差不多了，便把桌子上的剩菜连同餐具台布统统撤下，换
上新的。这是专为吃饺子准备的，成套的小碟小碗小勺小叉小笊篱，
件件是工艺品，小巧玲珑，非常精致。摆好餐具就开始上饺子。饺子
一次上六个因为我们一共六个人。每次上来的饺子颜色不一样，有粉
红、浅绿、淡黄色的，还有棕色的、褐色的，还有透明的。名堂有红
粉佳人金屋藏娇玻璃心醉罗汉辣妹子等等。形状也不一样，当然，味
道也不一样，酸甜苦辣都有。不同的饺子要蘸不同的佐料，而面前的
佐料有十好几种之多，要没有服务小姐的介绍和帮助，我根本不知道
该怎么吃。可惜那天我多贪了几杯，喝得云山雾罩，不仅没能记全那
些饺子的名堂，连究竟端上来多少种饺子也搞不清。光记得最后端上
来的一个大铜盆是汤，上面漂着一层“珍珠饺子”，一个只有指甲盖
那么大。服务小姐把灯关掉，让我们自己拿小笊篱从盆里捞。大家黑
灯瞎火地乱捞一气，再打开灯，从我开始，看捞到几个饺子，服务小
姐就讲一段与这个数字有关的吉利话或是典故。这样，饺子宴在一片
欢声笑语的喜庆气氛中进入高潮和尾声。我瞅了一眼帐单，好家伙，
七百多元。还不算酒，两瓶五粮液是老同学从家里带来的。

　　虽然六、七年前的那场饺子宴在我吃饺子的历史上创下了最排场、
最讲究和最昂贵的记录，但还是比不上二十多年前的那顿饺子印象深
刻。

　　那是在我插队的头一年。我们盼星星盼月亮盼到腊月二十九，突
然接到县里通知，规定知青不准回家过年，要在农村过“革命化的春
节”。大队书记一宣布完通知许多知青就哭了起来。全村六十名知青
赖在大队部迟迟不肯散去，有些类似现在的请愿。最后双方（大队干
部与知青）都作出让步，达成协议：大年初一只上半天工，中午吃饺
子。那时我们的伙食极差，也吃不饱，所以把这顿饺子看的很重。双
方就饺子的所有细节，包括由谁来做，肉和萝卜干的比例，饺子的大
小等等全部经过讨价还价的谈判，然后作出详尽的规定，并规定了限
额，男的100 个，女的70个，不够就吃窝头，窝头不限数量。
　　初一那天上工修大寨田，书记不在工地。书记在知青食堂监督和
指挥几个女社员给我们包饺子。书记不在工地，大家干活就比较轻松，
手头有一下没一下地挖土搬石头，嘴里却兴高采烈地谈论中午的饺子。
那天早晨谁也没让食堂送饭。大家都不傻。中午有100 个饺子在向我
们招手，谁还肯喝玉茭面糊糊？所以尽管肚子饿的咕咕叫，还饶有兴
致地互相打趣，说，看谁谁谁的口水流出来了等等。谁知收工回去书
记又修改了限额，说是由于饺子包的太大，改成男90女60，否则怕把
我们的胃撑坏。有人嘟囔了一句，饺子再大也大不过包子，让书记发
了火，说：不行立马蒸100 个包子看着你吃，剩一个扣10分工！明明
书记这话说的不对，人家只是说饺子大不过包子，又没说能吃100 个
包子，要拿饺子打赌，120 个也敢来。可是谁也没再吭气，主要是怕
把90个饺子也黄了。不吭气并不说明心里没有气。我挤过排队打饭的
人群，对维持秩序的大队干部说，先给我个窝头！我这样做一方面是
要给书记一个难看，另一方面也是为自己着想：先吃窝头后吃饺子肯
定比相反次序的效果要好。我拿住窝头刚啃了一口，一个女知青拽了
我一下，悄悄说，别吃了，过年还能吃窝头？等会儿我给你剩10个饺
子。
　　90个饺子没有多少，也就是一饭盒加一海碗。不过我们的饭盒大，
不是从商店买的那种，而是在板金店用白花铁皮订做的。大家端着自
己的一份，蹲在地上闷头吃起来，谁也不说话。平心而论，饺子做的
不错。也不小。平时萝卜干吃得不怎么样，然而与猪肉一搭配，马上
变的不再是萝卜干了，香的不得了。但由于大家的情绪太低，那天的
饺子吃的实在不高兴。吃到一半的时候，那个女知青苦着脸对我说，
你该吃窝头还吃吧，我连一个也给你剩不下了。现在想起来真可笑，
当时却不但没笑，还恶狠狠地骂了她一句。因为她说这话丝毫没有开
玩笑的意思，她也不是一个有幽默感的人。当然，我也不是。沉闷的
空气最后还是由书记打破的，所谓解铃还需系铃人。或者也是为了表
示他的歉意吧，他站在院子中央大声说：“日它娘，哪怕初二都去要
饭，初一的饺子也要吃饱。每人再加10个！”那天我吃的最多，一共
吃了108 个饺子，跟梁山好汉的数量一样。其中有我的100 个和女知
青的8 个。这是我在二十多年前创下的记录。我相信能打破这一记录
的人不会太多。至于我本人，既然二十多年始终未能破此记录，这一
辈子就别作妄想了。

　　要说吃饺子给我的最深印象，恐怕还要数三十多年前的童年时代。
　　过去，在老家的县份，饺子叫“扁食”，被视为最高级的饭。我
想是因为它成本较高，不但要花钱买肉买菜，还要花很多时间来做，
剁馅拌馅和面擀皮一套工序下来花费不少功夫。所以要说饺子是有钱
阶层和有闲阶层的食品大概没错。既没钱又没闲的一般百姓人家平时
不但不吃饺子，但凡听见有人家偶尔传出来剁饺子馅的声音，还会说：
“不年不节的，张狂甚。”剁馅儿的声音也就随之变的羞涩和胆怯，
断断续续地渐渐隐去。不过一到过年情况就不同了。老百姓再没钱再
没闲过年也总要或者说必须吃一顿饺子。包扁食吃扁食是过年的主要
内容，甚至是过年的另一种说法。不吃扁食算什么过年呢。于是一到
过年，家家户户大剁而特剁其馅，剁得无比夸张，剁得酣畅淋漓，剁
得理直气壮，剁得神采飞扬。这时大街小巷咚咚咚的声音连成一片，
象滚滚春雷，一阵响似一阵。它不如爆竹那般震耳，但更为持久，更
为激动人心。
　　那时每逢过年，简直要把我们这群十岁左右的小男孩忙坏。先是
把家里攒了一年的肉票拿出来，拉帮结派去排队买肉。买出肉来不先
急着回家，而要等大家都买出来进行一番比较才算，比谁买的肉肥。
谁的肉最肥谁就最骄傲，谁的肉最瘦谁就懊丧的不得了，还需要伙伴
们口头安慰半天。我们把肉交待给大人处理，便摩拳擦掌地等着挤菜，
就是把剁碎的白菜用纱布包住，把里面的水分挤出来。我爱干这个活
儿，干得非常卖力，总要挤得我妈说行了行了看把纱布挤破我才作罢。
这时我妈就会端上挤干了的菜到邻居家炫耀：“看我儿子的手劲有多
大？看看这菜挤得！”挤完菜，就该洗手准备包饺子了。那些有姐妹
的小伙伴在自己家里轮不上包饺子，便跑到我家来。我没有姐妹，这
成了我的优势，我颐指气使地命令他们摆桌凳、洗手，教他们怎么包，
包好了往哪儿放等等，仿佛给了他们多大面子似的。我们一边包饺子
一边听大人叨古，大都是说一些与饺子有关的趣事。说我们周围的几
个邻县吃扁食不跟我们一样蘸蒜蘸醋，而是用红糖豆腐海带等物做成
“头脑”浇到扁食上吃。我们就惊讶地说：“哎呀，那多难吃！”好
象只有我们的吃法才正统才文明而邻县们还属尚未开化的蛮夷之地。
当说到一个从京城回来的农夫吹嘘他见到过皇帝，皇帝每天早晨吃扁
食，管饱吃。我们的嘴角就会泛出鄙夷的嘲笑和城里人的优越感来。
　　案板上的饺子越来越多，摆的齐齐整整，远远看去就象接受检阅
的军队，近看则形态各异，谁包的就是谁包的，形状不一样，好象性
格也不一样，不象是一个模子扣出来的工艺品，倒象是一个个有生命
的小动物。桌子上至少放有两盆馅，一盆肉的，一盆素的。这就给包
饺子增添了更多的情趣。本来饺子的魅力正在于它的含蓄和藏而不露，
馅儿的种类一多饺子的内容就更加丰富。你不咬破它，就不知道它是
什么馅儿的，够不够咸，味道如何等等。肉馅不能生吃，素馅能以，
它是用炒熟的鸡蛋和油炸过的豆腐做的，拌上韭菜和各种调味品，生
吃也很香。我喜欢趁我妈不注意时舀起一小勺素馅迅速塞进小伙伴嘴
里，受到奖赏的小伙伴会意地眨眨眼，等我妈再次不注意时把嘴里的
素馅咽下去，咽得尽量不动声色，然后干的更加卖力。这其实就跟现
在基层领导瞒着上级给手下的人塞红包是一个性质，效果也相同。
　　饺子下锅，小伙伴们每人拿个小碟子在锅边等，等啊等，等老半
天饺子才煮熟。我妈往每个小碟里捞三个饺子，这就是他们得到的酬
劳。当然只能是三个饺子，又不是皇帝，哪能管饱吃呢？这时小伙伴
们连筷子也不用，手指一扒拉就弄到嘴里，烫得嘶哈嘶哈，眼睛直流
泪，还鼓着腮帮子说：“哈，真香，肉的。”或者说：“哈，真香，
素的。”……

　　只有过年才能吃上饺子的年代，有什么值得怀念的呢？可我就是
怀念。其实要为这怀念找些理由也不是找不出来，比方那清新的空气，
比方那湛蓝的天空，比方人与人之间那温暖的关系，比方……我吃108
个饺子那天，你知道书记、大队干部和女社员们吃的是啥吗？窝头！
他们从早晨5 、6 点一直忙活到下午3 点为我们包饺子，吃的却是窝
头。最后每人只喝到一碗煮饺子的汤，连烂在锅里的破饺子都被分的
非常均匀，并不因为你是书记就能多吃一块饺子皮。你说我不怀念他
们怀念谁呢？

　　写到这里，我想起一个老朋友的话：到了咱这岁数，去年的事情
不如十年前的事情记的清楚，二十年前的事更清楚，三十年前的事最
清楚。
　　诚哉斯言！信哉斯言！
