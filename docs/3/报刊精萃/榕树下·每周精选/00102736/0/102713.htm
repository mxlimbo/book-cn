我的姥姥

○河北　斌子

　　 


　　静谧的夜晚，妻儿已进入了甜甜的梦乡，坐在计算机旁，不知怎
么的，突然又想起了我的姥姥，姥姥去世已经多年了，儿时的记忆里，
总是和姥姥联系在一起的。姥姥的音容笑貌，也不时地浮现在我的眼
前。
　　我的幼年时期是在姥姥家度过的，一直生活到临上小学。姥姥是
典型的农村妇女，裹着小脚，不认识字，连她的名字都叫苗田氏，但
有着中国妇女传统的品德，对小辈们的呵护和爱怜，我们几个外孙深
深体会得到。
　　听人说在我二岁的时候，一头老母猪猛然把我撞倒在地，我吓得
惊厥昏迷，不省人事，姥姥和姥爷每天背着我，到十几里外的邻村去
治病，硬是把我从瘟神手里夺了回来。
　　记得还是生产队的时候，有一天到地里收花生，当时还没有包产
到户，生产队长开会说不许带小孩去，怕孩子们偷吃。姥姥说：“让
我的这个外孙去吧，他肯定不会偷吃的”，于是我就跟姥姥到了地里。
工余期间大人们挑出一些好花生烤熟了，喊我来吃，因为姥姥的那句
话，我死活不肯吃，而且躲得远远的，姥姥因此觉得给她争了气，得
意之情溢于言表，逢人便讲，到处炫耀：“看我那外孙，看我那外孙”，
以至我近40岁回村探亲时，所有的老人见到我，都要提到这件事。
　　记忆中的姥姥总是带着慈祥的微笑，很少有发脾气的时候，即使
我犯了再大的错误，也只是笑骂几句，有时把姥姥气得没办法时，姥
姥就把我拉到井边，吓唬我说“你再淘气我就跳井了”，可我还是淘
气不止，现在想起来着实愧疚。
　　从内心讲，姥姥的位置远远超过了父母，因从小在姥姥家长大，
姥姥给我的爱要多一些。上学后回到父母身边，父母性格不和，经常
吵架，家庭气氛不是很好，于是就常常想起要回老家（总是把姥姥家
当成老家），“就是吃糠也要去”，成了我必说的一句话。
　　放假期间总是要去的。当时的生活很艰苦，鸡蛋近乎是一种奢侈
品，姥姥攒下好多，不舍得吃，不让别人动，都留给了我这个外孙。
每次去的时候，离得很远，我就会心情激动，因为我知道，姥姥一定
就在村口，她总是早早地拖着两只小脚跑到村口，一等就是几个小时，
远远地望见姥姥的身影，心里有种说不出的感激。
　　姥姥是我的避风港，每当我遇到不愉快的事情，都是在姥姥家得
到解脱的。只要见到姥姥，任何事情都会化为乌有。临到放假的时候，
我都会心情振奋，因为我又能见到姥姥了。
　　上初一的那个暑假在姥姥家住了一些日子后，骑自行车返回，刚
到家发现父母正在吵架，于是一气之下我骑上自行车又回到了姥姥家。
姥姥家离我们家有四十多公里路，到家天已经很黑很黑了，姥姥看到
蓬头垢面的我，很是吃惊，我见到姥姥，委屈的眼泪立刻流了下来，
害得姥姥陪我流了很多眼泪，也害得父母半夜开车到姥姥家找我。姥
姥发火了，我第一次看到姥姥对我父母发那么大的火，这样我又得以
在姥姥家住了一阵子。
　　在姥姥家的日子总是快乐的，但幼时的记忆还是少的可怜，想起
来，点点滴滴都凝结着姥姥对我的感情。
　　姥姥不知从哪里学会的算卦，算出“我十八岁就不跟爹妈吃一锅
饭了”，于是对我更加爱护倍至，果不其然，我十八岁考上大学，毕
业后又分配到了外地，回家已经很少了，但姥姥对我的牵挂之情，虽
相隔千里，也感受得到。
　　我结婚那年，带着新婚的妻子去看望姥姥，这时的姥姥，因为脑
血栓神志已很不清楚，眼睛也更不好使了，村里的人包括亲戚，她几
乎都不认识了。我一到家，有人就指着我问：“你认识他吗？”姥姥
笑着说：“怎么不认识，斌子！”“那她呢？”“斌子的媳妇！”所
有在场的人都惊愕不已，可见姥姥对我的爱已经到了何种程度。晚上
姥姥非要我陪她睡觉，躺在床上，姥姥是那么怜爱地看着我，半夜醒
来，发现姥姥在抚摸我的脸，借着月光，看到姥姥脸上还挂着那种慈
祥的微笑，心里顿时感受到了万般慈爱，眼角也湿润了。没想到的是，
就此一别，永远见不到我的姥姥了，但姥姥那和蔼的面容，却深深地
印在了脑海里，每当想起姥姥，就想起了那天姥姥的微笑，仿佛就在
眼前，是那样的分明，那样的真切。
　　姥姥去世时我正好去郑州出差，没有赶得上她老人家的葬礼，而
且几年来因工作繁忙，也没有时间回老家到坟上祭拜，心里总是块心
病。去年到北京见到表哥，谈起我们的姥姥，俩人禁不住潸然泪下，
姥姥给予我们的太多太多了。于是相约春节无论如何也要到姥姥的墓
地祭扫。
　　我自小少言寡语，姥姥话也不多，站在姥姥的墓前，我没有说一
句话，而是在深深地感受，感受姥姥的无私，感受姥姥的关爱，感受
一个小脚老太的伟大。姥姥生前没有跟我说过怎么喜欢我，我也没有
跟姥姥说过我爱她，也许爱本来就不需要用言语去表白。想起姥姥去
世多年了，生前对我的爱无以言表，撰此文以志纪念。
　　聊以藉慰的是，我遗传了姥姥的近视眼。
