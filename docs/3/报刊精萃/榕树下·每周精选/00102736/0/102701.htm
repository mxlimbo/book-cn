惶惑中开花

○北京　蕊瑶



　　在我小时候爸爸妈妈曾经误认为我是个天才，因为我五岁就上学，
上到小学四年级的时候还因为品学兼优而跳过一回级，十六岁我顺利
的考上了重点大学，在大二的时候我考过了英语六级，大四我就得到
了中华人民共和国注册会计师的证书。
　　我学的是财会，我学的这个专业让妈妈为之骄傲，妈妈在一家国
有的大型企业里做了一辈子的会计，不过她没有会计师证，当然，像
她这把年纪的老会计是不需要那么一张纸来证明自己的，但是我却需
要，因为我没有经验，而且我的路还正长。妈妈认为会计是一个很好
的职业，它不像秘书什么的是吃青春饭，这个职业像红酒一样历久弥
香，妈妈认为历久弥香的职业最适合女人，她使女人一生都有自己稳
定的收入，而女人只有在经济独立之后才能谈得上精神的独立。可是
妈妈也教导我说将来一定要找个好老公，我问妈妈好老公的意思是不
是要有很多的钱，妈妈说当然要有很多的钱，否则我就会像她一样，
每天沉迷在计较财米油盐钱中不能自拨，我松了一口气，说我这么年
轻漂亮，找个有钱的男人并不难。妈妈像被针扎了一样跳起来，说，
不是不是，你找的那个男人不光得有钱，他还得末婚、英俊、学历高、
有良好的家庭背景，并且对你体贴入微。我听了就笑了，说这么优秀
的男人能和我结婚吗，能不和我离婚或是在外面包二奶吗？妈妈说不
会的，并不是所有男人都那么坏的，就像她的单位的许多男人，他们
全部本份、顾家、从不在外拈花惹草。我心里仔细想着那些不英俊、
没有学历的出身农村的男孩子们，觉得他们确实有着妈妈说的种种好
处，于是我一时心动，便说，那你做主把我许配给他们其中的一个吧。
妈妈又说：不行不行，你这么年轻，这么漂亮，又有知识，他们怎么
配得上你？我盯着妈妈，温柔的笑起来，这个世界上最爱我的人总是
喜欢教导她我，然而她却常常不能自圆其说，在我诡异的笑容中，妈
妈忧郁的望着我，妈妈的忧郁如此单纯，可是谁也劝不了她，她是多
么的伤心啊，她觉得她越来越无法挽救她心爱的女儿了。
　　妈妈一天比一天更觉得她无法挽救她的女儿了，从我到北京读大
学开始，妈妈总说我和她格格不入，我也不想这样呵，我多么爱她，
可是我总是让她失望。当我大学毕业后，我没有固定的工作单位可去，
我找不到妈妈所希望的铁饭碗，今年，愿意接收应届毕业生的国家机
关企业的员工们自己还面临下岗，所以我只得自谋出路，后来我决定
去公司应聘，妈妈痛苦得好几天失眠，但是她帮不了我，她是一个多
么柔弱的，不善交际的女人，而她同样老实、丝毫不具备呼风唤雨能
力的在外地政府机关做最低职位小公务员的老公也束手无策，这更坚
定了妈妈要我找个有钱老公的信念，可是妈妈又怕那些坏男人诱惑和
欺骗我。总之这个可怜的女人伤心极了。她强迫我答应她考研，并且
还给我看了她一直珍藏着的嫁妆，那是一些二十世纪初期发行的印着
袁世凯头像的银币和一些古老的漂亮首饰，据说地主家庭出身的外公
为了保存这些财产曾在土改时被打折了腿，妈妈一直很珍惜这些宝贝，
即使在她曾经有过的下岗的两年中也没有变卖，那是我初中的时候，
爸爸为了每天省三块钱而中午不在机关食堂吃饭，他饿着肚子回家，
晚上要吃三碗米饭，爸爸妈妈经常吵架，两人都大把的掉头发，不过
我记得那时我仍然每个月有十块钱零花钱，同学们都很羡慕我。妈妈
给我看她的嫁妆，还对我说，他们两口子每个月有两千块钱的固定收
入，这也就是说他们每个月都能存下来一千五百块左右，而且他们还
有那么些家底儿，所以如果我想出国留学，他们可以倾其所有，他们
愿意让我去，只要我有锦绣前程，他们就再也别无所求。五十年代出
生的人有一种特殊的天真，在我小的时候，他们认为只要我考上大学，
就什么都好了。现在他们认为只要我能读研，或者只要我能出国，就
什么都好了。至于读研和研究生毕业后的事情，至于出国后的事情，
则是想也不用想的。有时我妄加猜测，觉得这一代人从少年起就是这
样，给他们一个单薄的信念，他们就可以持之以恒的幸福下去。

　　在四月份我去了国际展览中心参加招聘会，我读大学的时候经常
去家乐福超市，可是从来没有注意过国际展览中心，在静安庄下车的
时候我还在想我应该找一个年逾古稀的老太太给我指路，老太太肯定
会不厌其烦的告诉我正确的路线。可是下车之后我立即发现自己真是
愚蠢极了，因为根本不必费心去寻找，我所能看见的所有的人都涌向
国展。于是我就被夹在人群里卷进了国展。那么多人与我竞争工作，
大家都像革命者散发传单那样散发着手中的一摞简历，个个简历都印
得比我漂亮，我沮丧的说不出话来，我把我难看的简历随时手放在了
几家公司，这些可怜的简历和我一样的抬不起头来，我转身跳跑了。
　　现实居然比我想像的乐观。通知面试的电话一个一个打来，我每
天都要面试两家以上公司，忙的喘不过气来。直到面试过到那家跨国
贸易公司后，我就不再接受面试了。
　　那家主营跨国贸易的公司在钓鱼台酒店办公，钓鱼台酒店在三里
河路上，有一个门通往闻名遐迩的钓鱼台国宾馆，这个五星级酒店的
灯光使我着迷，所有的灯都躲在看不见的地方暗暗的炫耀着自己的暖
昧，地上铺着深绿色的印着花纹的地毯，电梯里的大理石和明亮的镜
子反射着彼些的光芒。我多么愿意在这样的地方工作啊。
　　和我一同面试的还有两个女人，一个三十多岁，一个四十多岁，
她们应聘的分别的融资部经理和项目经理，两个女人都非常美丽，并
且衣着高贵无比。那个三十多岁的女人一进屋就不停的打电话，这使
她看起来像一个交通警，交通警是指挥车辆来来往往，而她则在电话
中不断的安排人与她约见，从她嘴里说出来的都是相隔很远的地方，
可是时间却互相离得很近，我想这个女人一定是忙极了。那个四十多
岁的女人先是锐利的审视我一会儿，忽然就奇怪的开始对我笑，我有
点不理解，可是她笑得那么友好，于是我也对着她笑了，她开始和我
低声聊天，在女交警指责面试我们的姓陈的老板既然没有时间就不该
约她，因为她很忙时，那个四十多岁的漂亮女人告诉我，她也是一家
公司的老总，因为自己公司解散了所以不得已才出来应聘的，可是只
要能和这个老板合作成一个项目，筹集到一笔资金，她的公司就可以
重振旗鼓了，她还说她最喜欢像我这样聪明漂亮的小姑娘，等她的公
司开始做了以后，她想高薪挖我走。她给了我她的名片，上面果然写
着什么公司的总裁，她还向我要了呼号，她说不管这次她能不能聘上，
都希望我和她保持联系。我心里奇怪，她为什么不说不管我这次是不
是能聘上呢，我刚刚二十岁，我没有任何工作经验，我想我会聘不上
的，我想，她大概是出于礼貌吧，可是我很得意，还没有工作呢，竟
然就有人想挖我，多荣耀呵！我还想如果今后她来挖我，我该说什么
呢，是坚持不去吗，这样矜持既能够使她相信我是个忠实的职员，又
能够使她开出更高的价钱，总之我得意死了。当然这些话都是那个四
十多岁的漂亮女人悄悄对我说的，陈总和女交警谁都没有听见，毕竟
我们俩现在谁也不想砸掉饭碗。
　　陈总和女交警说完话，四十多岁的漂亮女人凑了上去，陈总始终
对我一言不发，我就恭卑的等待着。四十多岁的漂亮女人讲话的声音
总是那么柔那么轻，给人的感觉亲呢而暖昧，我想她是肯定因为长期
从事管理工作才变得这么具有亲和力的。
　　陈总和四十多岁的漂亮女人聊了一会儿，就礼貌的请她先走了，
他站了起来，对女交警和我说，我们到里面去聊吧。
　　陈总带着我们两个女人下了楼，坐上了一辆凌志车，我欣赏着这
辆我从小就倾慕的流线型的车，一分钟以后他们开进了一个很庄严的
门，门里绿草茵茵，还有一个美丽的湖和一些分散的、极为端庄美丽
的小楼，我从来没有奢望过都市里还有这样美丽的桃花源，这里没有
城市的喧哗吵闹，只有娇滴滴的鸟在鸣叫。
　　我们下车上了其中的一幢小楼，楼里铺着木地板和价值不菲的红
地毯，大厅里站了一群穿着旗袍，貌若天仙的姑娘，陈总低声对她们
说，这些都是些获过国际国内大奖的名模。我艳羡的看着她们，故意
走近了一点以便能看得更清楚些，结果是这些名模也都看着我，她们
深深的艳羡的看着我，这真是让我迷惑。我一不小心看到大厅里的镜
子，名模们美丽的脸在脂粉里微显老态，可是我还如此年轻，素着一
张脸却发着光，自信哗啦啦的回到了我身上。
　　在一间房子里坐定了，我发现女交警不再像刚才那样的张扬，她
说，陈总，国宾馆也可以做写字间包出去啊？陈总深沉的笑笑说，不
是所有人都可以包，得中央办公厅批示，哦，对了，我以前是政协的。
女交警又问，那收费一定很高吧？陈总又深沉的笑笑，说，不算太高，
一天才一百六十美元。
　　现在我知道了我在哪里：钓鱼台国宾馆，我开始发抖，我真的在
发抖，因为她从来没见过这么大的世面。我只顾发抖，根本听不见陈
总在和女交警说些什么。
　　我迷迷糊糊的四处张望，忽然看到了陈总的眼睛，他的眼睛盯着
我的胸部，看见我看他，陈总闪电般的收回了目光，由于收的太快了，
使的表情一时间显得颇为怪异，他显然不愿意让我发现他很欣赏我的
胸部，可是我却并不是太在意，其实在国展那天我就发现这个四十多
岁的男人看我时的目光有些特殊，可是我不怕，我从小就相信人正不
怕影斜的这个道理，我是凭自己的本事吃饭，我不会让对我图谋不轨
的男人有可乘之机的。
　　这时，女交警又恢复了她刚才的倨傲，她喋喋不休的说他和一个
市行的行长很熟，贷来钱没问题，问题只在于陈总肯给她几个点做为
回报。陈总问她，那个行长姓什么，女交警说姓朱，陈总又问她叫朱
什么，女交警就答不上来了，连我都能看出她是答不上来了，可是她
面无惧色，还拿出了一个小巧的商务通，调出朱行长的手机号码给陈
总看，给陈总看完号码后她又重复了一遍：我和朱行长很熟。重复的
时候她的眼神迷离，这种眼神使我猜她一定是个无耻的女人，她一定
与朱行长有过某种染指。
　　陈总也拿出了商务通，问女交警，朱行长家的电话号码是不是********，
女人冷笑着，说不是，可是我不能告诉你是什么。陈总就拿起了摆在
桌上的传真机电话，拨通了这个号码，对方接了电话，是朱行长，陈
总和朱行长打了几句哈哈，挂了电话。我想如果她是女交警，此刻势
必夺路而跳，并且永远消失。可是女交警仍然面无惧色，她娇慎的望
了陈总一眼，说，早知道你和老朱这么熟，我就不瞒你了嘛！
　　后来女交警又说，她老公是总参三部的少将，很有权有势的，可
是总参的工作要求保密，所以不便透露他是什么身份。临走的时候女
交警说她曾经在南方做过六年的酒店管理工作，所以对酒店的用品有
着强烈的感情，她请求陈总同意他去拿走卫生间配发的印有钓鱼台国
宾馆字样的用具，陈总宽大的一挥手，说：去吧。
　　女交警走了以后陈总亲切的对我说，多让你听听这些，对你增长
社会经验是有好处的，我虔诚的点头，丝毫也不奇怪陈总为什么对我
这般照顾。有时候我会有一种强烈的冲动，想发个布告昭告天下的男
人们，当你们爱上一个女人或是打一个女人的主意时，再笨的女人也
会立即明白你们的心思的，如果她们假装不明白，你可以这么辩证的
理解她们的态度：要么是她们根本就懒得搭理你，要么是她们假扮纯
情，目的是期待你猛追她们，要么就是她们装傻，将计就计，既想从
你的追求中得到好处，又不能让你占到她们的便宜。
　　陈总又问我，你猜她拿走那些卫生用品干什么。我想都没想就开
口说，当然是出去后有技巧的卖弄给别人看，让别人相信她与住在钓
鱼台了某些人有着神秘莫测的关系。陈总眯缝着眼看了我一下，这一
眼中有些警戒。接着他哈哈大笑，说你这孩子真是聪明。

　　后来我就在这家公司里当了出纳，月薪两千元。我打电话给妈妈
向她报告这个消息，妈妈让我勤快点，嘴甜点，多和公司里的会计学
些业务知识。我想让我勤快没有问题，让我嘴甜也没有问题，可是这
个公司根本就没有会计。他们要么就不做生意，要做生意就是成百上
千万的做，事实上公司里只有几个项目经理，还都不是专职的，他们
动用各种各样的关系为这个公司拉来生意。因此我来了这么多天，做
的最多的一件事就是把各种各样报销的发票贴在一起，并且不断的从
银行取钱给他们。
　　直到有一天帐户上的钱快取光了。我和陈总一样着急。陈总开始
加紧到处拉融资，我跟着他，别人都以为我是他的秘书，或者叫做小
蜜。
　　有一天我们请朱行长唱歌，我们去了朱行长指定的一家歌厅，朱
行长不让我们像平时那样称呼他，他让他们称呼他叫做苟总。
　　他们一行三人走进了歌厅，几个小姐迎了出来，说，苟总，好久
不见啦，哟，又带新客人来啦？就和我在古装片里看到的杏花楼之类
的地方的女人们一样，不过说实在的，这几个小姐丑极了，我心想我
要是长得这么丑，一定没脸出来做小姐。
　　我们在歌厅里坐下，我要了一盘杏仁，我最喜欢吃杏仁啦。很快
杏仁端了上来，盛在一个我所见过的最小的盘子里，一个挨着一个的
摆着，没有一个是摞着的。我愉快的拿起一个放进嘴里，一秒钟之后
我又把它吐了出来，杏仁上散发着肥沃潮湿的泥土的味道。我想这杏
仁不会是马王堆出土的文物吧，可是他们怎么舍得卖给客人吃呢，而
且一盘只卖80元人民币。
　　一个老女人走了进来，对朱行长点了点头，然后向陈总自我介绍
说她是妈咪。我瞠目结舌。在我的印象中妈咪这个词是在骂人的时候
才出现的，如果我们想用最极端的词汇来污辱一个女人时，通常我们
会说她是鸡，而当说一个女人是鸡时也不能遏制我们的愤怒时，我们
会说这个女人是妈咪。可是现在这个词就从我面前的这个女人嘴里吐
了出来，她齿白唇红的说，我是妈咪。我认为她这是一种自渎，可是
自渎就自渎吧，又有什么关系，反正她的脸藏在那么厚的脂粉后面，
谁也没法儿知道她是谁。
　　朱行长说我要五花，妈咪很报歉的说，五花坐台了，换一个行吗？
朱行长失望的眼神都暗淡了，他说，那把你们的小姐们都叫过来吧。
　　一会儿来了许多小姐，有的美有的丑，所有小姐都穿得很少，我
想她们老了后统统会患上极为严重的关节炎，心里极为得意。我看上
了两个小姐，一个长发，涂着粉色的唇膏，像粘腻的水藻中开出了一
朵鲜艳的花，另一个小姐剃着板寸，没有表情的脸看上去so cool.朱
行长选了一个小姐，那个姑娘看上去像一小堵墙，她比朱行长高一头，
重大约二十五公斤，她如果一发狠，一定能成功的压死朱行长，可是
朱行长选了她，她的胸部像两座小小的山丘。
　　陈总看了看我，说他不要小姐，我很气愤。
　　朱行长和小姐开始唱歌，朱行长点了《杜十娘》，朱行长唱，你
要是饿得慌你对我十娘讲，十娘我给你做面汤：你要是冻得慌你对我

们的妙笔能生花，可是能生出钱来吗，我们赚谁的钱去呢，还有那些
老东西们，他们全是靠我们赚来的钱吃饭，可是他们还瞧不起我们，
你现在快和他们一样无知了。我松了一口气，还好，林主任说我无知，
他没有说我和那帮老东西一样无耻，我猜他本来是想说我们无耻的。
我感激的看了林主任一眼，林主任继续说，他就是喜欢郁子这样的事
业型的女性，郁子是他的左膀右臂。可是，林主任话锋一转，郁子和
我不是一个层次上的，郁子只是到最基层的企业去打版，那个婊子，
她最擅长的就是出卖自己的色相，而我比她层次高多了，我所做的是
直接找省市领导批文，让他们把文件转到下面的企业，我一抓就是一
批。总之，我和那个婊子不是一个层次的，那个婊子老想勾引我，我

我在等人，浦东有一个非常非常爱我的男孩子，他从小就是那么爱我，
现在他长大了，他还是一样爱我，他没有钱，因此他从家乡来到了上
