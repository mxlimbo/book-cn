人间嫦娥的失败婚姻

○北京　老谷


　　夏朝太康时候，后羿成了有穷国的国君。这有穷国建立的是军人
政权政府，自然，后羿也就是有穷国最大的军人。在传说中，后羿的
名声可不算小，他勇武过人、猿臂善射。尧之时那会儿，天上有十个
太阳，地上的庄稼草木被酷日暴晒而死，哀鸿遍野，成了野兽长蛇的
天下。历史造就英雄，后羿成为人民的救星（当然不是东方红，太阳
升，那会儿的太阳都多得成灾了），他奋力攘外，挽弓劲射，射去天
上的九个太阳后，又致力于安内，扫杀大地上害人的兽蛇，换来了天
下的平安。
　　大家都说，美女爱英雄，大月氏最美的女子嫦娥爱上了英雄后羿，
英雄后羿也一见嫦娥的国色天香而不能自拔，恨不得一切都免了，那
样最好，就正所谓樵夫农人和渔民之名言，烈火遇着了干柴。
　　我们的历史从来都是这样，打天下者不能坐天下，后羿也不例外。
于是，居功自傲的后羿在自己小小的封地中无所作为，加之看到太康
昏聩无能，暗地里就有了篡权夺位的想法，对自己的死党则说是替天
行道。
　　那年秋天，太康要出宫去河洛秋猎，他要后羿与他同去，后羿推
说自己的有穷国有好多事要料理，不能同行，请太康原谅。太康有些
不悦，但又不好说什么，便带了自己的亲信踏上了去河洛秋猎的旅途，
他哪里知道这是一条不归之路呢。在路上，太康还说，后羿有什么了
不起，给他脸不要脸，到时候别怪我给他一把一把往下撕！
　　一个亲信也说，他不去正好。他这个人仗着自己的箭术好，到处
都喜欢出风头，连天子您也不放在眼里，我们就更不在他的话下了。
他要去了，我们倒扫兴了。
　　太康走了，后羿马上发动政变，守宫的军人见风使舵，都投到了
后羿的门下，后羿轻而易举地就抢占了太康的王宫。然后，后羿叫来
太康的信使，说，你去河洛告诉太康，他爱上哪儿上哪儿，就是不要
回来。这里好些人要杀他的头呢，到时候我可帮不了他。
　　太康很没出息，听了信使的话，眼泪一下子就流了出来，从此流
亡在外，勉强在阳夏找了一块地，修筑了新城，不再打猎。太康本盼
望着三十年河东四十年河西，回到故宫的高墙上，就像我们在电影中
看到的南霸天那样，趾高气扬地大喊一声：太康我又回来了！可他没
能盼来这一天，每当在夕阳中望见飞向故都的大雁时，他的心就要碎
一次。就这样，不几年他便忧困而死了——他根本就不是一个卧薪尝
胆的人，倒像是一个内心脆弱的诗人。
　　太康一死，后羿便毫不迟疑地自封为天子，施令天下了。
　　在后宫，后羿对嫦娥说，过去我不自封天子，是给夏康一个面子，
也是故意气他。他是天子怎么样？天子也拿我没奈何。我要称了天子，
他见自己没了希望，也许会疯狂反扑，给我添不少麻烦。这样多好，
不费一刀一箭，他自己就歇菜了。记住，下次到了太康墓，别忘了烧
一炷香，以寄托我的哀思。
　　哪个时代都有特立独行的人，都有砍头不要紧，只要主义真的人。
但也有历史的玩笑，这个叫伯封的少年诸侯便是这个历史玩笑中的悲
剧人物。这伯封并不是一个为民请命，或者说拼死维护天下理义的人
物，他只不过是一个小小的诸侯。也不知道是怎么回来，这诸侯伯封
一点也没有贵族气派，倒有一副流氓无产者的心思和嘴脸，所以他也
就逃不过一个有着天生性格悲剧的人的命运。这伯封最喜欢占别人的
便宜，臭名远扬，最要命的是他还长了一张碎得要命的臭嘴。为此，
他得了一个外号，封豕，按现在的话说就是猪的意思。。
　　伯封不管是在自己的诸侯国，还是在诸侯会聚的时候，都总是摆
出一副瞧后羿不起的样子，总说一些后羿的坏话。这还不是伯封倒楣
的根本原因，根本原因是当时那个世界有一个叫玄妻的女人，和嫦娥
并称为“两朵金花”，可见其艳名之大。玄妻有一头像是常年用海飞
丝浸过的头发，又黑又亮，且丰软纤柔，长可拂地。你道这玄妻是谁，
她就是伯封的寡母。
　　我们知道后羿是一个很伟岸威武的男人，男人嘛，何况又是权势
齐天的男人，他又如何不想把这“金花”据为已有呢？这也许是考验
一个天子威仪的试金石呢？后羿听说伯封对自己的大不敬，心里暗笑
了一声：真是天助我也，小子，你的死期到了！
　　很快，后羿就把攻伐伯封的计划提上了朝中的议事日程。大家都
说战争死人事小，天子的名誉事大，伯封这小子侵犯天子的名誉权，
实属死有余辜。大家都鼓掌赞成，一致通过了攻伐伯封的计划。其实，
后羿嘴上没说，他这是一箭双雕，杀了伯封，玄妻不就成了自己的床
上恩宠了。
　　他的这份心思瞒得过别人，却瞒不过嫦娥，嫦娥一听后羿要攻打
伯封，就知道后羿醉翁之计不在酒。于是，后羿一回后宫，便看到嫦
娥一脸的不高兴，嫦娥晚上睡觉也背对后羿。后羿见状，故作关心状
地问嫦娥：爱妻，你咋了你，哪儿不舒服？
　　嫦娥说：心头不舒服！
　　后羿色迷迷地把嫦娥的身子扳过来，就要解嫦娥的衣裳，说：天
子我给你揉揉吧。
　　嫦娥把后羿的手挡开，说：你攻伐了伯封，去给妖精玄妻揉吧。
　　后羿心中有鬼，不好对嫦娥霸王硬上弓，便很索然地躺下了，说
道：这是国家大事，你不要东想西想，胡说八道的。
　　其实后羿还是喜欢嫦娥的，他那么大个人物，有权有势，在个人
生活中多喜欢一些漂亮女人不能算是腐化堕落，就是克林顿不也闹了
个拉链门事件吗？可见在这方面，嫦娥的政治觉悟不算高，思想不够
解放，更缺乏政治家的风度。
　　为讨嫦娥的欢心，后羿在攻伐伯封之前，还是想和嫦娥温存一番，
算是告别的纪念。他把嫦娥抱在自己的怀里，从怀里掏出一个贝壳，
说：我的爱妻，这贝壳里装的可是天下第一等的宝物，西王母的灵药，
吃了可以不死，还可以升入天界。要知道，就是西王母也不易得到，
非常珍贵。你好好收起来，等我打败伯封回来，咱俩一同吃下，到天
上成了神仙，仍然做那快活夫妻。
　　听了后羿的这番话，嫦娥这才面露喜色，两人便滚倒在了床榻上，
连在门口保卫后羿安全的卫兵都听见了动静，实在忍不住，只好悄悄
躲在墙角，自己安慰了自己。安慰自己的时候，这卫兵嘴里还念念有
词：
　　心是媒娘手是妻，
　　眼睛看东又看西，
　　忽然一阵白云出，
　　可怜娇儿跌泥里。
　　话说不堪一击的伯封很快就被善战的后羿打败，后羿如愿以偿，
把玄妻抢到了手中，更出乎后羿预料的是，玄妻很顺从，也很能干。
两人一个合回下来，竟欲罢不能，只好又承前启后。完了，两人躺在
床榻上说闲话，后羿说：玄妻呀，这些年守寡，你可怎么过哦？
　　玄妻说：唉，熬呗！寡妇门前是非多，我可不敢做那偷汉的丑事，
天下人知道了，我还不遗臭万年啊。有时候，实在忍不住了，就自己
和自己戏耍一番，恨不得破门进来个男子，把自己抢了去。
　　后羿就说：玄妻呀，你受苦了！要是我早知道这样，早就来了。
唉，我来晚了。
　　玄妻说：唉，八年了，别提它了。然后翻身又骑在了后羿的身上。
　　两人这样颠三倒四地过了几天生活后，后羿就说要把玄妻带回宫
中，玄妻却说：你就在这里多住些日子吧，回去了嫦娥还能饶过我？
　　后羿说：没事的，有大王在，嫦娥不敢对你如何的。
　　玄妻说：你还是在这里多呆些天，等嫦娥消了气，我们再回去不
迟。
　　后羿觉得玄妻的说法也对，便又住了下来。
　　后羿乐不思归，另寻玄妻新欢，而空守后宫的嫦娥忧涕不止，过
着以泪洗面的日子这件事，传到了嫦娥娘家人的耳朵中。娘家人听说
嫦娥被后羿遗弃，过着寡妇般孤独痛苦的日子，非常生气，家族商议
之后，便暗地里派了人悄悄潜入后羿宫中，与嫦娥接上了头。
　　来人说：你要觉得在后羿这里再也得不到人生的幸福的话，还是
趁青春还在，离开这个鸟地方，回我们大月氏那里去。你又不是不知
道，我们大月氏有那么多健康英武的小伙子，至今都仍然暗恋着你，
你回去也是给他们一个机会。
　　嫦娥说：这事可不像你说的那么简单，后羿要知道我跑回了大月
氏，他还不发兵打我们大月氏啊！再说，宫里看得这么严，平时我出
宫一步都有护兵跟着，要逃出去哪那么容易。
　　来人说：只要你决心回到大月氏，我们就有办法。
　　嫦娥心里很矛盾，和后羿生活了这么些年，两人还是有不少幸福
时光值得回忆的，当年自己看上后羿，不也因为他是一个天下少有的
英雄吗？我要离他而去，真的会像歌中唱到的那样，“分手时候说分
手，不要说难忘记”吗？但她转念一样，他如果还像过去那样爱我，
他会和玄妻两个寻欢作乐，任我日日以泪洗面，仍不思归朝吗？想到
这里，嫦娥不仅怒从心头起，咬牙切齿地说：好吧，你想法把我暗渡
回国，让那条公狗和那个老女人过一辈子去吧！
　　虽然玄妻已有了伯封这么大的儿子，但也并没有老得如何，其实
正是如狼似虎的年纪，要不后羿哪会乐此不疲呢。
　　来人说：这样吧，我们的计划一定要万无一失，我过两天再来找
你，我们都想想有没有一个万全之策，既能安全回国，又不能让回宫
后的后羿使蛮，到时候打杀到我们的大月氏的门上去要人。
　　从大月氏来的人走之后的第二天，嫦娥突然想起后羿临走时留下
的西王母的灵药，心想：狗东西后羿还说要和我一起吃了药成仙，在
天宫中做什么快乐夫妻呢，他现在倒和玄妻做了地上的活神仙。我才
不自己一个人吃了药，孤零零地一个人在天宫中过苦日子呢。但我却
可以用这灵药做掩护，假装吃了它升了天，然后再悄悄潜回大月氏，
不就得了。
　　第二天，大月氏谍人再次来到后宫与嫦娥密谋潜逃之事，嫦娥说
了自己的打算，谍人拍手叫好，说这是一个天衣无缝的计谋，与他设
想的利用祭月之时，月夜逃走之计相组合，可以人不知鬼不觉地完成
潜逃回国的计划。
　　那天一早，嫦娥便叫来宫中的大内总管卜施，卜施从小与后羿一
起长大，是后羿最得力的助手，好些后羿国内国处的交往礼义都由卜
施安排。
　　嫦娥对卜施说：按我们大月氏人的习俗，今晚是一年之中最重要
的祭月之夜，让人准备一下，我晚上要去祭月。还有，你知道前两天
从我娘家来了一个祭月的巫师，他是为今晚的祭月大仪专门来的，到
时候他得在祭台那儿等着，别耽误了。
　　卜施对此并无丝毫怀疑，满脸笑容地诺诺称是；其实就是后羿本
人又何曾想到，嫦娥会暗地潜走呢——不就守了一些日子的空房吗？
有啥大不了的。好些两地分居的夫妻不也是旱时旱死，涝时涝死吗？
　　到了晚上，嫦娥带了几个贴身的宫人和护兵来到宫外的祭月台，
台上早站了一个人，正在那里做法，一些护兵认出了他就是前些天来
到宫中的大月氏祭月巫师。你道这巫师是谁，他就是潜入宫中和嫦娥
接头的大月氏谍人。
　　大月氏人以月亮为图腾，崇拜月亮，嫦娥嫁给后羿时，可谓集所
有宠爱于一身，自然她故国家乡的习俗也得到了后羿的尊重，为此，
后羿便令人在宫外专门修筑了这一处祭月台。这几年，嫦娥常到此来
祭月，这也是卜施没有想到嫦娥此次拜月有诈的主要原因。
　　嫦娥走上台去，祭月巫师说：今晚祭月，非比寻常，西王母将来
此与嫦娥娘娘相会，诸人必须回避。
　　在台下的宫人便走到远处的树丛，侧耳听着台上的动静，听见祭
月巫师说：嫦娥娘娘，你手里拿的是西王母让后羿天子转交给你的灵
药吗？
　　嫦娥答道：是的。
　　祭月巫师说：那你现在吃下它，就可以和西王母会面了。
　　不多一会儿，只见祭月台上一道眩目的白光一闪，宫人伸颈望去，
刚才还在月光下的两个人影现在却只剩下了一个。
　　过了一会儿，祭月巫师大声喊道：尊敬的宫人们，速来拜月！
　　大家忙来到祭月台下，巫师便说道：嫦娥本是天宫的月神，现在
她下凡期满，西王母便让她吃下灵药，重返天宫归位。另外，西王母
还要你们待后羿天子回宫之后，如实向他禀报，不得有误。
　　听了巫师的话，大家深信不疑，便齐齐地在台下跪下，向着天上
的圆月顶礼膜拜起来。
　　宫人们回到宫中，向卜施作了嫦娥吃了西王母的灵药飞天而去的
汇报，卜施额头上的汗顿时便如豆粒般涌现了出来。要知道，后羿叫
他在宫里负责王宫族人及财物的安全，现在他竟然把天子的老婆弄飞
了，这还了得。所以，他心里虽对嫦娥飞天一事抱有怀疑，但却不敢
对他人说，要大家等天子后羿回来后，大家再向天子汇报。
　　卜施不敢怠慢，差人赶快去见后羿，向后羿汇报嫦娥飞天而去这
件事。后羿听了来人的汇报，虽心中有些惋惜嫦娥如此一去不返，毕
竟一日夫妻百日恩嘛，但当着玄妻的面，却淡淡地说：既是天宫的安
排，我在也留她不住，由她去吧，在天宫做神仙也好。
　　第二天，后羿便携带着新宠玄妻踏上了回宫的归途，总之嫦娥已
经走了，玄妻也就没有理由不愿跟着后羿回后羿的领地了。
　　一回到宫中，后羿便寻了个机会把卜施叫到密室中细问，卜施只
好如实说，自己那晚并没有同嫦娥一同去祭月，关于嫦娥飞天而去这
件事，是听随去的侍女丫环和侍卫说的。卜施把那天所有跟随嫦娥祭
月的人叫了来。一个侍卫说，他过去从未到对祭月台祭过月，所以很
好奇，虽然祭月巫师要大家远离祭月台，他还是躲在了祭月台不远处
一个野草丛中，一直偷偷地观看祭月台上的祭月典仪。他说：真是神
奇呢，嫦娥娘娘吞下灵药，一股奇香便飘进了我的鼻孔。这会儿，隐
隐约约的乐声从天空中飘来，一道白光从祭月台上射向天空中的月亮，
嫦娥娘娘怀抱一只白兔从台上飞起，转眼就不见了。
　　后羿又问了其余几个人，大家的说法大同小异，后羿便问道：哪
那个从大月氏来的祭月巫师呢？
　　卜施说：祭月之后，他又在我们的宾舍中住了几天，完了他向我
说，他来这里帮嫦娥祭月已经结束，要回大月氏去。我劝他多住几天，
他不肯，便让他走了。
　　听到这里后羿便“哦”了一声。
　　后羿是一个赳赳武夫，但这个赳赳武夫也还有心细的地方，譬如
攻伐伯封的一箭双雕之计，还有就是他听说嫦娥飞天而去时自己内心
的怀疑，他认定嫦娥并没有飞天，倒很有可能是大月氏人用计把她弄
回了国。后羿想，有时候女人是很狡猾而且比男人更决断的，当她不
能忍受自己的男人被别的女人占去时，她便选择了离开，选择了独自
上路。现在，她一去不返，连自己也不知道自己是更喜欢似火热情呢，
还是喜欢似水柔情，如果自己同时拥有这两样情怀的女人该是多么幸
福啊！可惜！可惜！后羿并不是没有想过找个理由攻伐大月氏，再把
嫦娥抢回来，但这个计划的最大障碍便是玄妻，她猜透了后羿的心思，
是想去大月氏找回嫦娥，自然坚决反对，后羿无法，便罢了，心想，
今后找机会再说吧。

它揣到自己的衣袋中的，她记得自己从夏的领地潜逃回来之后，就把
这个装着西王母灵药的贝壳放在了床榻边的柜屉中。这个抽屉锁着她
情爱的秘密和有着人生记录的历史见证之物。这个被紧锁着的西王母
灵药现在却不胫而走地来到了她的衣袋中，并从衣袋中掉了出来，被
自己注意。
　　嫦娥把它拿在手上，意外地发现了上边的字迹：外遇是上天都可
以原谅的错而另一面则写着：TO BE ，OR NOT TO BE？
　　嫦娥不由自主地一遍一遍地念叨。她恨自己知道这句话太晚了，
如果自己当初知道这句箴言的话，她就不会离开后羿了。要知道，爱
情和自己爱的男人被别的女人分享的时候，如果你坚持、你争取，他
可能就会回到你的身边，最不济，还可以分享；而你一当撤退和放弃，
你就一无所有了。大家都奉行这样的原则：离婚了，就别来找我！
　　嫦娥抬头看见一个男子正从山坡上往下走，这个男子不是那个被
悲伤击垮的人。
　　这个男子的脚步声越来越近了。嫦娥打开贝壳，把药粉倒进了嘴
里。
　　嫦娥缓缓地仰倒在草地上，看见天空飞掠而过之后，倾斜了。
　　嫦娥忍不住呻吟起来，她看见了后羿，后羿就在她的眼前。然后，
她的呻吟声慢慢地小了下来。
　　草原上几只鸟儿在嫦娥身体的上空一圈一圈地盘旋，鸣声清脆。
　　后羿在鸟儿的指引中来到嫦娥的身边。他单膝跪下来，这是他一
生中第一次也是惟一一次下跪。他用手指轻轻地拭去嫦娥嘴角边的血
迹，然后采摘来四野的花，把嫦娥掩埋起来。
　　后羿轻手轻脚地躺在了嫦娥的身边，又挪了挪身体，他想尽量躺
得安逸舒适些。
　　他对自己说：曾经，有一份真诚的爱情放在我面前，我没有珍惜，
等我失去的时候我才后悔莫及，人世间最痛苦的事莫过于此。如果上
天能够给我一个再来一次的机会，我会对你：我爱你。如果非要在这
份爱上加上一个期限，我希望是……一万年！
　　然后，后羿看见一道锋利的夕阳从剑页上一闪而过——他把剑深
深地送进了自己的心窝。
