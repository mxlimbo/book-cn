下午的茶

○山东　子而

　　 
　　饮茶原本是一种生理需求，据说古人也用茶治病。究竟什么时候
饮茶演化为仪式，成了一道，说不清楚，然古往今来，有文化的茶人，
容易追求极至。往往从简单的喝，渐而寻觅名叶，向往名泉，追求名
器，循习规矩，终于走向了极端。本为清心消渴，反到成了桎梏，使
人不能随心所欲。
　　世上的事情，大约是不懂其中妙处，方能随心所欲，一旦入了范
畴，则必受规矩的制约。饮茶亦当如是。
　　曾读过一篇《下午的茶》。作者对茶，可谓精通。且看他对茶的
评判和领悟：乌龙茶好比高人，喝一口就可指点迷津。花茶非常精灵，
可惜少了点儿韵味。……白毫乌龙耐品，像温厚而睿智的老者。……
英国的皇家红茶，恕我直言，镀金皇冠。铁观音外刚内柔，佛手喝来
春暖花开。柚茶苦口婆心，至于陈年普洱，好比走进王谢堂内，蜘网
恢恢疏而不漏。龙须茶，真像圣旨驾到，五脏六腑统统下跪。

　　浓列是早晨，清香已到了中午，淡如白水合该熄灯就寝。喝茶顺
道看杯中茶，蜷缩是婴儿，收放自如到了豆蔻年华，肥硕即是阳寿将
近。

　　何止是精通，简直就是茶的精灵。你看他写起茶来，那种空灵飘
逸，使人仿佛置身于中国水墨画的氤蕴浸润之中，致使我在写出我的
感受的时候，不自觉地引用了那文章的题目：下午的茶。
　　然而作者却坦然宣称：
　　随心所欲画画，随心所欲饮茶。
　　于是知道，那些识茶至深的高雅之士，原来并不尽受清规戒律的
束缚。
　　大智若愚。真正懂茶、喝茶的人，末必会写茶经，也不一定讲究
茶艺。而精雕细刻，深谙茶艺的人，却未必得了茶的真谛。
　　一生喝茶，晚年消闲，每天寻觅于街头小店，去找可口的茶叶。
块二八角三块五块，回家来一壶塞进大半两，浓如苦口中药。数碗下
肚，满腹舒畅。立即不辞辛苦，分送茶友共享。不在乎茶壶无把，水
壶生锈，人生一乐，适意而已。
　　垂钓水边，或烈日曝晒，或冷风砭骨，一壶浓茶，装在不锈钢壶
内，挂耳投杆，静待漂沉，小品一口，虽非名壶名器，也无朗月清风，
那种心境也足让人无欲无求。无牵无挂，随欲罢了。
　　曾在福建朋友处吃过一种台湾加工的乌龙茶，据说此茶选用上等
茶叶，用参液浸泡，而后加入多种香料薰制而成，每斤要上千元。小
心饮来，味苦回甘的感觉到是发挥到了极至，但茶的雪花膏味极浓，
总让人怀疑那送茶的小姐用香皂洗过手。尤其是面对叮叮咣咣一桌杯
盘和主人的一脸殷切，喝茶就仿佛喝了一顿排场，累人兮兮，情趣少
少。
　　中国人习惯排场，往往见不得随心所欲。饮茶不过排场中之一事。
家乡某官绅，炎炎夏日在家身穿棉袄，为的是偶尔出席一二宴会，能
够不流汗水，不露窘相。王安石当年要苏东坡从三峡取水，并非要中
峡水不可，真正目的未必在于饮茶，不过故弄玄虚，挫一挫苏氏的傲
气，扬一番雅士的清名罢了。为虚名掩蔽真我，大约是历代中国文人
和官场的通病。并不是说茶道有什么不好。若有闲情逸致，又有雅致
的品味，美茶美器，细细品来，自是赏心乐事。也不是说尊崇中国的
茶文化就是迂腐不化。至于杭州酒吧用龙井名茶洗手而奉英国红茶为
上宾的纨绔不肖，就是中国人的异数了。
　　然而，凡事中规中矩，就觉不及随心所欲。随心所欲是一种宁静，
是一种悠远，是一种澹泊，是一种超然，是人生难得的高妙境界。孔
夫子曾说，三十而立，四十而不惑，五十而知天命，六十而耳顺，七
十才能从心所欲。看起来，随欲之境，非长者不能达，非智慧不能及。
　　因而，向往随心所欲，企盼能够随心所欲。
　　于是，首先随心所欲地饮茶。
　　早晨，中午，或是下午的茶。
