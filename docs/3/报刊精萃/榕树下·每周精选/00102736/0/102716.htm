小妖的红色打火机

○浙江　菲凡.胡

　　 
　
　　那个戴着黑框边眼镜的女人问：你叫什么名字？
　　她想了想，说：小妖。
　　小妖刚丢了工作。在昨天之前，她还是一家广告公司的文秘，干
活轻松不说，拿的钱还不少。如果不是那个混蛋经理，小妖想她也许
会在这家公司呆上很长的一段时间。那个年过半百的经理成天晃着一
双色迷迷的绿豆眼，象只讨人厌的黄蜂在她身边绕来绕去。昨天这只
胖头黄蜂终于亮出了它的蜂针，他毫不避讳地当着公司同事邀她共进
晚餐。谁都知道晚餐只不过是个愚蠢而直白的幌子，只要她一点头就
立刻摇身变成了坏女人之类的角色。于是小妖选择了做贞女。胖头黄
蜂恼羞成怒，用一个极不光彩的理由解雇了她。最可恨的是小妖那个
不明真相的男友，以为在自己眼皮子底下居然被不知不觉戴了一顶绿
帽子。他在死死地盯了她一眼后愤然地甩手而去。就这样，在同一天
里小妖失去了工作和爱情。
　　小妖很快在一家声讯台找到了新的工作。每个刚加入声讯台的小
姐都要为自己取一个好听又好记的名字。小妖记得曾经在一本杂志上
看见过这样一句话：妖是天使在东方的称呼。所以当那个戴着黑框边
眼镜的主管问她：你叫什么名字？她回答说：叫小妖吧。那个主管愣
了愣，但很快就释然了，面无表情地在记录本上写着：小妖。无可置
疑，这是个令人记忆深刻又回味无穷的名字。
　　小妖的工作就是每天坐在电话机旁等着别人来和她聊天。这里所
指的别人多数都是男人。起初小妖还对这个工作抱有一些期望。她多
少带着一点救世主的心理想：每个人都会有倾诉的欲望，但有些事是
不能和朋友或家里人说的，而她就在这个时候适时地出现在他们身边。
小妖是善解人意的，她自信能把这个工作做得极其出色。但事情似乎
完全不象她想的那样。小妖很快发现打电话来的人大多都很无聊，并
对声讯台小姐本人表现得过分好奇和幻想。他们喜欢用一种低沉而暧
昧的口吻说话。这天小妖接到了一个电话，电话里的那个男人甚至问
到了小妖的三围。小妖冷冷地说：我们是声讯台小姐，不是坐台小姐。
挂上电话，小妖觉得像在不经意之间吃了一只苍蝇。
　　这是1998年的冬天，再过几天就是情人节了，冰凉的空气里到处
弥漫着玫瑰和爱情的味道。小妖的情绪一直低落，她想起了去年2 月
14日那天，她捧着一大束玫瑰花的幸福模样。有人说，如果情人节那
天女孩没有收到花，就最好不要上街。小妖闷闷地想：看来那晚只能
一个人呆在家里了。
　　情人节这天下午，小妖六点钟下班。五点五十分的时候，忙了一
天的电话终于安静了下来，小妖开始收拾自己的东西。她想着下了班
该去超级市场买点吃的，家里的冰箱已经空了，然后再去附近租几张
影碟。这个夜晚，她打算抱着一堆影碟过一晚上，但绝不能是爱情片。
桌上的电话忽然铃声大作，小妖看了看墙上的钟，指针已经停在了六
点。她迟疑了一下，还是拿起了话筒。
　　喂？电话里传来了一个男人的声音。
　　你好。小妖说。
　　情人节快乐。这是个普通话纯正的男中音，很斯文的样子。
　　情人节快乐。小妖说。她想，我一点都不快乐。
　　今天收到了几朵玫瑰？
　　一朵都没有。小妖叹了一口气。她重新在座位上摆正了姿势，点
上一支烟。她隐隐觉得这个男人似乎和以往的那些男人有些不同。她
开始有了一些交谈的兴趣。
　　怎么会呢？男人不紧不慢地问。
　　我和男朋友一个月前分手了。我总不能自己送花给自己吧。你呢？
花了多少钱？今年的玫瑰好象比往年又贵了。
　　我和女朋友也刚分手。他的声音显得有些落寞。
　　小妖重重吐出一口烟，说：看来我们同是天涯沦落人了。
　　电话那头出现了几秒钟的空白。小妖说：喂，你还在吗？
　　我在。男人说。你在抽烟吧？
　　你怎么知道？小妖吃惊地叫了起来。
　　我感觉得出来。那个男人的声音停顿了一下，象正在思考着什么。
接着他说：也许我有些唐突。小姐，今晚我能不能请你吃饭呢？
　　听电话这头小妖没有了声音，他赶紧又说：没有别的意思，今晚
我只是想找个人一起吃饭。请放心，我是好人。说到这儿，他笑了一
下。
　　好吧。小妖说。
　　约会定在合欢酒店三楼的月桂包厢。直到坐在这个男人对面，小
妖仍觉得有些恍惚，连她自己也很奇怪她会真的来到这里。这个男人
的举止很绅士，和他的声音感觉相似。他有着一张白白净净的脸和一
双狭长的眼睛。小妖注意到他身上穿的衣服的牌子，好象都是名牌。
男人让她点了菜。在等上菜的时候，他们开始聊天。小妖显得很快乐
的样子，不停地说着话。那个男人起初还有些沉闷，后来也渐渐活跃
了，以至于好几次，小妖的一句话竟然逗得他放声大笑起来。晚饭快
结束的时候，男人从口袋里拿出一个黑色的小盒子，说：这是送给你
的。情人节快乐。小妖疑惑地接过盒子，打开。她看见盒子里面躺着
一只火红色的zippo 打火机。她抬头看着男人，男人说：女孩在情人
节是应该收到礼物的。第一次见面我不太好意思送红玫瑰，既然你抽
烟就送一个红色的打火机吧。他的话多少令小妖有些感动，她向他微
笑了一下。
　　在酒店门口分手的时候，男人说：谢谢你能来，今晚很愉快。如
果明年的今天，你还没有找到人陪你过情人节，你就到这里来。我会
在我们今天吃饭的包厢里摆上红色的玫瑰等着你。说完他张开手臂拥
了拥小妖，转身走了。
　　小妖愣愣地站在原地看着他的背影，心里涌动着一种莫名的伤感，
她忽然想到甚至还不知道他的名字。她摊开手，那只打火机在黑夜里
像一团火燃烧在手心里，极尽它的艳丽。
　　第二年的冬天出奇的温暖，情人节那天晚上，小妖考虑了很久，
最后她还是穿上了一件火红色的粗花毛衣去了那家合欢酒店。她走上
三楼找到了那个月桂包厢。当侍者打开门，她的眼睛瞬时就被桌上的
一束红玫瑰捉住了。她走过去在桌子旁边坐下，一股异常甜美的花香
很快包裹住了她的身体。就像是爱情的味道。她点了几道菜和一瓶红
酒。她感到自己那颗因等待而发烫的心在剧烈搏动着。
　　小妖不停地看着手表，时针正指向十点。桌上的菜凉了，红酒冰
壶里的冰块也已经化成了水。小妖几乎抽完了一包烟，那只火红色的
打火机静静地躺在一堆烟蒂旁边像一个沉默的胜利者。小妖想：他是
不会来了。侍者进来结帐的时候，对面包厢的门几乎是同时忽地开了，
一对情侣模样的人从里面亲密地走出来。小妖惊讶地看到那个包厢的
桌上同样摆着的一束火红的玫瑰，一样的粉色透明包装纸，一样用古
色的景泰蓝花瓶插着。小妖顿时瞪大了眼睛，指着那束玫瑰：那束花……
侍者立刻微笑着殷勤地说：小姐，很漂亮是吗？这是我们老板为每一
位来我们酒店过情人节的客人专门准备的，希望能给您一个温馨的夜
晚。小妖抬起头向他笑了笑。那名侍者觉得这个女孩的笑很奇怪。
　　小妖在离开合欢酒店的时候被那名侍者叫住了。他追上她，手里
拿着那只火红色的打火机：小姐，您的打火机忘了。
　　小妖朝那只打火机望了望，然后向他笑着说：送给你了。情人节
快乐。
