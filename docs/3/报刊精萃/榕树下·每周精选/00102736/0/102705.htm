狂奔

○浙江　水无缘



今夜，我坐在大地的耳畔
月光束缚了我的肉体。

今夜，四周是静悄悄的
鹰隼在梦魇的头顶疾飞。

一只小兔子，从月亮上跳了下来
重演了古时的一幕
那株被撞疼了脚的歪脖子老槐树
摇了摇头
掉了一地的困惑。

今夜我是无语的
今夜我是沉思的
今夜我只是默不作声地
想一些事情

疯了的向日葵孤独地燃烧着
痛苦的空气扭曲了视线
我看见它飞了起来
风一样的，
狂扫尘世间。

今夜我是落寞的
落寞的象一座即将崩溃的泥塑
在沼泽的上方
缓缓陷落。
今夜我又是疯狂的啊
疯狂的象一个两眼通红的疯子
象披头散发的爱因斯坦
一把将时间揪住
远远的抛向身后。

今夜，
夜似乎凝固了
象一块黑色的巨冰。
今夜，似乎一切
都在等待。
只有那株歪脖子老槐树
忽然，
拨出了所有的根须
一路狂奔而去
站在了天堂的门口。
