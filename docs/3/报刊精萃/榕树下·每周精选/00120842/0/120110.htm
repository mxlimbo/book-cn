十二个眼神

○天津　沉默的橘子



三月——
还未填上内容的春天
是我荒芜了的眼神

七月——
忽乎吾将行
古人在读今人的心

二月——
我用手指画你的名字
雪用身体画季节的名字

六月——
有阳光，有空气，有水
足够我睁开眼
足够我呼吸
足够我哭泣

十一月——
黑夜里只有
梦的双翼掠过的声音
我聆听

五月——
不要碰
那是我的花

一月——
只顾回头寻一声呼唤
是谁趁机与我们擦肩而过

九月——
钟表应声而坠
时间的碎片散落一地

八月——
记忆中的天蓝色
覆盖了我最纯粹的声音

十月——
我捡到一片贝壳
上面写着
“不是今天”

四月——
来不及说出口的
都不存在了

十二月——
理智啊
不要吵闹
让我和幻想再多聊一会儿
