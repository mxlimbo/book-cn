私奔

○浙江　江南篱笆



私奔 在月黑风高之夜
他们商定在后花园
隔墙相见
三更的梆子声一响
一个香囊
被扔到了墙外

这次私奔蓄谋已久
计划几经修改
银子要攒足
干粮要齐备
今晚老爷祝寿
家丁都在前院赌钱

慢慢来
把梯子搁稳了
好兄弟在巷口把风
他是猎户 武艺高强
马车等在城外
再给你一个机会后悔

好吧走吧
把车帘拉开
私奔路上
穷山恶水
都是风景
