转糖情结

○湖南　土妖


　　小时候喜欢看“转糖”。
　　“转糖”的名声似乎一直就不大好。每次开班会，我们的那个美
丽年轻的，刚当上妈妈的班主任老师都会叮嘱：“同学们不要随便在
外面的小摊上买东西吃。你们看那个转糖，啧啧啧，好脏哦……”随
即作出一副厌恶的神色，好似那卖转糖的小贩就在她眼前似的。然而
每天放学，转糖摊前总会挤上一大堆细伢子。摊子本是摆在校门口的，
然后又挪到巷子口，生意反倒是好了许多。小时候的人都喜欢偷偷摸
摸的做老师反对的事情，那种心情会因为做了这类的“坏”事情而激
动很久，加上转糖本身也是很吸引人的，所以由偷偷摸摸到成了习惯，
放了学便跑到巷子口看转糖。
　　卖转糖的人好象总是在变，不知道是不是一家子，但摊子却总是
有一个的。行头很简单，一个小柜台，上面钉了块木板，用红漆划成
辐射状的许多份，每份都画个物，或鸡或鸭或狗的，最高级的是龙，
其次是凤，都是极品；再是一个炉和一口锅，刚好凑成一副担子。
　　转糖是近乎一种大众娱乐了，那种人情味是现在吃肯德鸡，麦当
劳无法比的，出了小钱，小贩便会从匣子里抽出一片中间有孔的竹签，
往画了东西的木板中央的轴上一套，然后由你转。一拨，竹签便转成
了一片白，边上的旁观者便合声喊：“龙，龙，龙……”出钱的那一
个也喊：“龙，龙，龙……”好似转到个龙是大家共有的啦！但往往
极难转到龙，最多是只洋咪咪（蜻蜓）和蝴蝶之类的，也有鸡，猴，
蛇什么的。然后小贩就用一个长勺从锅里挑出点儿熬化了的躺糖汁，
一极其纯熟的手法在石板上飞快的勾出一只蝴蝶什么的。勺里倾出的
一丝细糖线就能变成那很象模象样的活物，光是这份奇妙就足以抵得
上花的那几个小钱了，更别说还有转的时候那种博彩似的刺激。味道
还在其次，至今我还不知道那种象明矾一样的块状，却又是褐色，放
到锅里一热就变成稠稠甜香的糖汁的糖叫什么糖，没有白糖甜，粗糙
得很。反正也不是冲那个味儿，而是冲那个情趣，“转糖”摊前总是
热闹得。
　　再说小贩勾画好了个物，就从草把上抽竹签，贴在还热的糖画上，
再挑点儿热糖汁点在签与画之上，一会儿就冷透了，糖画也硬了，竹
签也粘牢了，然后用一块极薄的铁片把画从石板上小心的铲起来，一
立，递到你手里。一般的都不会急着吃而是举着，招摇一番，直到觉
得炫耀够了，才“咯嗦”几下咬碎，吞下肚，也不管上面沾了多少灰
尘什么的，图的就是这个乐。
　　有时也会转到“六团（tuo ）”“八团”，小贩就用竹签从糖锅
里捞出一团稠稠的糖汁，待凉了，变成个糖球，给你。“八团”比
“六团”大一号，至于为什么叫“六团”“八团”而没有“七团”，
委实不清楚。
　　还有一种高级点的，叫饴糖，味道有点象牛皮糖，韧性十足，也
甜得透一些，凉的也不会变硬，总是稠稠的。这种糖做不了画，只能
你特意张口点名要，小贩才宝贝的从柜子深处掏出个盆，挑一团给你。
因为比较贵，又没有糖画那么好玩的花样，所以除非是有钱又嘴馋，
一般是不买它的。
　　后来知道，小贩们是很有心计的，总是八把柜子放在路边的台阶
间，一头高一头低，高的一头木伴上画的是龙风之类费时做又耗糖的
东西，低的一头多是小虫和“六团八团”之类的小玩意。而那根用来
转的竹签也有奥秘，算作准星的一头套着个铁套子，重，所以转来转
去总是会停在低的这头。
　　知道内幕后，大家都好象发现了什么惊世大阴谋，觉得上了当，
小贩们的做法实在是大大的阴毒，都发誓再不买他们的糖了，然而熬
了几日终究是熬不过去了，又跑去看都说只看不买了。到最后还是受
不了诱惑，又纷纷的买了。小孩子的誓言是当不了真的，小，心志终
归是不牢靠。
　　有一次，不料碰上班主任抱着她的小孩也来看，我们本该四下逃
窜，如鸟兽散的，却都吓得迈不动脚。班主任竟也一脸讪讪的笑，分
明是说：“我屋里崽伢子总是吵着要，我也冒办法。”最后的结局是
皆大欢喜，班主任花钱买了一只龙一只凤，让她伢子举着玩了一会儿，
然后折下龙的一双爪子给他吃了，便一脸亲切的笑着把剩下的一只无
爪龙和一只完整的凤给了我们。大家心有余悸的分吃了，仍是怯怯的，
吃不出甜。只是后来的班会上，班主任绝少再提起“转糖”，却仍是
不忘叮嘱我们要讲卫生，不要乱吃东西，只是不再具体举例子而已。
　　再后来，“转糖”就随着我的童年一起消失了，也不知到底是什
么时候的事了。
