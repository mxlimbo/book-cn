中秋的月光

○宁夏　斜阳



　　他今晚是应该回来了。
　　你为他精心做了几样菜。此刻，你在等待。
　　你等了许多年，时间比你活得年月还要长，你一直在等。你走遍
了天涯海角，走过许多的时间，就是等他，要你心甘情愿的人。
　　什么时候，他的眼光开始游离，开始躲闪。你清楚的记得，在你
最快乐那个日子里，心以为得到了归宿。在你们新婚的那些日子，你
就已经发现了。
　　他还是回来了，没有表情的看你，是一种无奈的眼神，你知道。
可你一直渴望的就是他，你一挨近他，就想使自己融化。你从小就渴
望这个时候。
　　他告诉你，温暖是虚幻而不实的，这样的日子不能长久，你面前
的世界必须你独自对付。
　　餐桌上的蛋糕夸张着节日的气氛，可与屋子的两个沉默的人是那
样耀眼的刺激。看着餐桌上的一切，你忽然多了辛酸，而他的眼也有
了泪。
　　你知道他有了自己的世界，那是什么样的感情你不想弄清。你对
他是宽容的，因为你知道这个世界很大，人的心是没有边际的。他爱
不爱你这无关紧要，你关心的只是爱不爱他，当你把他的一切一切都
抛开时，你知道你需要的就是他。至于他如何去做，这无关紧要。你
关心的只是你的心。你需要的只是形式。你把形式当做内容，幸福只
是一种主观感受。
　　你站起来拉他到餐桌旁坐下。你伸出一只手，你的手微微的颤抖，
你想为他拭泪，也想摸摸他的脸，这样的一张脸在他不在你身边的时
候是怎样一个折磨人的思念。你不想去想。你只希望这只手不被拒绝。
　　你的心和他的心，遮得很严。
　　你知道，人的心留不住的。你可以变成他的心，离你远去，你几
乎每天都这样做。你也想过当初，想过后悔，但你得到了你自己的深
爱。自私的讲，你爱着自己爱的男人，你没有遗憾，你总这样安慰自
己，在日记里。
　　你等他的日子很平静，你理解他，你知道他的厌烦，不得不接受
的东西总是让人厌烦的。他的厌烦像蓝天一样晴朗。
　　他很自然，一点也不愧疚，他对你背弃你都那么坦然。
　　本来就没有什么可愧疚的。谁都没有损害谁。认为自己受了伤害
不过是人的偏见。因为他在伤害你的时候，同时也伤害了自己。你照
顾他，体贴他，你的心像把剑，让他的心也流血，不能愈合。他是那
么善良的一个人。总能看见他忧伤的泪，这样的时候你总是心痛的恨
不能奉献自己的所有去换他一张笑脸。
　　你已不是有意去争取他，不是用心去博取他的爱了。六年了，他
为你付出的太多，而他也知道你对他所有的一切，但他的爱就是这么
吝啬，他爱的天空你就从来没有照耀过，也许只是星星那样的光辉。
你还是不说自己的错，你自己就这么愿意做，你不假思索，自然地去
做，满足你固有的自私的爱心。
　　他不在的日子，你就找寻屋子里他的踪迹，想他的时候会不由自
主的摸他的衣服。衣服上有他身体的味儿，很好闻。你总是能想起他
看书的模样躺在书房的小床上。
　　你一点也不奇怪，生活就是这样，总能想部分安慰你。
　　他是个好心人，从不有意伤害人。他即使做了无数错事，也能使
人原谅他，因为他本来就是那么自然的人。今天，你接了个女孩的电
话，是他的网友打来的，为他的生日祝福，女孩声言很清脆，是很朝
气的那种。这个女孩他喜欢不喜欢你不知道，但你不难受，反而为他
高兴，想他寂寞、烦恼时总算有个倾听的人，想他喜欢的女孩一定不
错，当然自己也挺好，她未必能胜过你。在这一点你总是多了自信。
　　同事都说他不如你，各方面都不如。工作几年他一直默默的在单
位搞技术，而你因为业务出色，现在已经是管理着整个地区的资金调
度，很吃香的一种职位。你觉得奇怪。爱是生命隐秘的感觉，跟外在
条件没有什么关系，你知道那理智无法解释的感情难以熄灭，也就听
任他去。
　　他的出众其实你自己早知道的，但为了爱，你固执的一次次要他
放弃那些你以为风花雪月的东西。他也是努力的爱过你的，那样不犹
豫的几乎烧掉了曾所有从学校带回来的一切，你没有阻止，虽然你看
到他难舍的眼神。
　　你还是想念在一起他努力过的那些日子，那是你最幸福的时刻，
你喜欢摇着他的一只胳臂跟他散步，有时把脸贴在那只胳臂上，听他
轻轻唱着一首歌，你抬头望着他，他用大手把你的头搂紧，让你喘不
上气……
　　远了，都远了，你不敢再想，你除了在日记里倾诉，你永远都是
坚强的女人。
　　今天他好像要说什么，你阻止了他，你不想在这样的夜晚，在这
样的日子说不开心的话。
　　“月那么明亮，我们喝一杯，好么？”你对他说。
