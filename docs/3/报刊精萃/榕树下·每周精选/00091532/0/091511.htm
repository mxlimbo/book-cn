思念的子弹

○重庆　一兵

　
每夜，我的梦都走向南方
走入故园飘香的溪旁
走入湿漉漉的小城风景
哦，想家的日子越来越密

站岗时又望见
那天际的云
好像母亲新添的苍丝
任何一个远方的孩子
都会用感激报答
这用来丈量人生的
母亲纳制的鞋底

想家时奔上山坡
对着远方
吹故乡的那支口哨
然后以某棵树作基准
为来自南方的列车检阅

哦，我宁愿站成一支枪
每日每夜
都将子弹推向南方
