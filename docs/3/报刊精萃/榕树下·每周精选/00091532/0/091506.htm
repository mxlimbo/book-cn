老人

○广东　甘剑南

　

　　我走到老人的身旁注视着他的时候足足有十分钟。他没有反应，
一直很安详地端坐在那里，双眼微闭，头发稀落，静听风尘飘过的声
音。在老人面前的布袋里，摆放着十几把手工制作的二胡，漆了红漆，
工艺算得上是精细。在老人的手上，还握着一把二胡，他正在调弦试
音。
　　老人的周围，是新疆卖核桃葡萄干的、河南卖假玉石耳坠手镯的、
潮州卖廉价衣服的、四川人卖麻辣面商摊的叫喊声，合着的人群的匆
匆脚步，在街道的上空高低起伏。正是黄昏时分，夕阳从远处高高耸
立的群楼间隙扫过来，整个杂乱的街道便罩上了一种不可言喻的气息。
　　只有老人，是平静的。
　　“你是在卖琴吗？”我试探着问，声音很轻，生怕惊醒他沉思的
内心。
　　“嗯”，老人的眼皮只是微微颤动了一下，还是没有睁开。
　　“你这琴卖多少钱一把！”我试图和老人套上话。
　　“十元”，老人答话很简单，没有进一步为自己的商品进行必要
的推销。
　　“你能不能拉一曲给我听听呢？这样我才能鉴别二胡的好坏。”
我问老人。
　　老人依旧没有答话，只见他调整琴位，拨动琴弦，民间艺人瞎子
阿炳创作的《二泉吟》便从两手之间缓缓流出。老人指法娴熟，音律
控制得很好。二弦之间，手和弦的腾跃，是细腻悠长的腾跃，是世事
感怀的腾跃。我的目光和心灵，被老人的琴声裹挟着，无法逃避。一
种激越而苍凉的声音让我的耳朵灌满了泪水，直到漫过我迎风散乱的
头发和漂泊的心灵……
　　最终我从仅有的生活费中摸出十元买了老人的一把二胡，他在接
过钱时微微睁开了眼说了声“谢谢”。那一刻，我蓦地发现他眼角密
布丛生的皱纹，宛如岁月的沟壑。
　　以后的日子我为见工而奔碌，但还是去了几趟那地方，说不清是
一种什么样的情愫在牵引着我，不过再也没有见到老人。听旁边店铺
里一个卖衣服的江苏女孩讲，城管人员最近经常来查这些私自摆地摊
的，老人也就经常被驱赶走。“那老人从湖北孝感来，听他讲以前是
县文化馆的一名工作人员。他好像在这里没有亲人，孤苦伶仃的，一
天也卖不了几把琴。有时他就靠拉几支曲子，过路的行人也就顺手给
上个块儿八角的……”女孩显得很是同情地说。
　　良久，我惘然若失地站在那里。在我的周围，依旧是杂乱的车流，
匆挤的人群和高高的广告牌，这条街没什么大的变化，只有老人已不
在了。
　　……
　　那是1993年底，我在老东门的一幕。这么多年来，老人的一双眼
睛让我无法释怀。他飘浮于老东门上空的琴声，穿透了七年的时空。
　　新千年的阳光直直地射在了这片勃发的土地上时，东门已彻底改
变了模样，他已跻身于国内著名商业步行街行列。现在的东门不只拥
有五颜六色，还见证特区商业发展的波澜沧桑。我经常去逛东门，到
东门走一走，但已经无法辨认老人当年二胡卖艺的位置。眼前，只有
林立的现代商铺，人如过鲫，霓虹闪耀，映红特区夜的斑斓。
　　我终于明白了，每个人内心深处，都有一个属于自己的图景，对
它心存感念和追随。终其一生在寻找，寻找是为了铭记，记住某个地
方或某个人……我的图景原来是东门呵，一位老人合着旋律走入新东
门，也走入一段厚重而悠长的历史。
　　我的日子渐渐好了，东门也换新妆了，拉二胡的老人，你好吗？
