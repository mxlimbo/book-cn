九世 黄泉

○重庆　青蛙

　

　　　　　　　　　　　　　　九世

　　生灵无尽循环在世，如火般可消可长，又似水般可塑可溶。
　　有一种东西在生命中是无穷无尽的，象一根绳子联系着每一个时
间段，但是却很少有人能真正的抓住它。我们蔓延着，我们消亡着，
我们在一点一滴的积累、忘却；我们燃烧着，我们溶化着，我们在无
声无息的开始、结束……


　　　　　　　　　　　　　　黄泉

　　风栖息在无道无相的命中，犹如一抹胭脂一遍一遍的抚过岁月的
脸，渐渐地剥落，渐渐地堆叠成一道脆弱的墙挡住了青春的笑。花开
了，花瓣落了，眼睛睁开了，表情麻木了，当云飞烟绕的时候声音停
顿了。绝情、无心、长恨、离愁，迷茫的世界越来越混沌，在模糊中
颓废，在渺茫中灰飞到无边际的六道，轮回的衍变开始了。

　　水静静地流过沉默的双手，在十指间空荡荡的歌唱着：

　　你是我的叶子，你是我的花香，你是我生命中的注定，但是我却
失去
　　我是你的花蕾，我是你的美丽，我是你魂魄中的雕塑，但是你却
无形
　　血红色的凋零是你的残忍，黄昏中的混乱是我的纵情
　　我在你的眼中枯萎，你在我的命中死去
　　在你转过身的时候我将影子点燃，烧尽前世的缠绵，烧干今生的
眼泪
　　深蓝色的美丽是我的笑容，暗夜中的迷失是你的放浪
　　你在我的脸上沉淀，我在你的手中泯灭
　　当我抬起头的瞬间你让灵魂飞散，挥去今生的承诺，挥去来世的
相约
　　你是我的疼痛，你是我的重锁，你是我轮回中的错落
　　我是你的负担，我是你的沉疴，我是你九世中的点缀
　　你只是路过，你只是路过
　　我却多情，我却这么的多情

　　暗青色湮没了透明的眼泪，在分流到各个角落的时候才凝固在一
起，所以永远徘徊在选择的交叉点上，一次次的被冲撞着，一次次的
被抛弃着。多么珍贵的缝隙，但是这狭小的天地竟然容不下这颗凝固
的眼泪，因为被包容在里面的是暗青色中唯一的光亮，而那双手却一
直都没有将这美丽捧起来，只是麻木的被水流划动着，没有温暖的力
气将旋涡平息。
　　再也没有月亮了。
　　没有月亮的夜晚是封堵向往的围墙，笼罩着深沉的瞳孔，抹杀着
忧郁的唇线，不容浅色的轮廓在它的世界中游离，它残酷的笑了。其
实，在这个世界中有没有月亮都无所谓了。白天是昏沉的压抑的，晚
上同样是昏沉的压抑的，没有楼宇琼台，没有灯点灿烂，只有风，灰
色的风在旷野中肆意的狂奔。这里没有血肉，这里只有魂魄，这里没
有人影，这里只有鬼魅。关于前生今世的悲欢离合都在这里的荒原中
聚集着，最后被燃烧着，在这冰冷的大火四周膜拜着七情六欲，膜拜
着四相的印记和三生的夙愿，然而所有的灵魂却莫视着这神圣的朝拜，
因为一切已经结束了。
　　河水呜咽着。

　　在这条浑浊的河流中有多少是多情的飘逝，有多少是无情的流连，
只是所有的不甘心和所有的满足都在这里呻吟着。这里是没有方向的，
没有方向意味着迷失，绝情怒骂着痴心，善良诅咒着邪恶。原来将一
切混杂后的情景是如此的混乱，原来这样才能公平的在六道中筛选，
原来选择这些去处的都是空心的。混沌中飘扬着迷乱的旗帜，在每一
面旗帜上都镂刻着模糊的图腾，在这没有尘烟的无形世界中它们都是
狰狞的，在这狰狞的呵斥声中所有魂魄在魍魑魅魉间做着殊途的决策，
或是幻化，或是覆灭。

　　黄泉无道无返，没有来时路只生去时途。

　　她凄然一笑，他惊惧无声，象金针刺透指尖，深刻的疼痛却在痛
苦即将结束的时候才流出血来。第一滴朱红色的血滴入漠然的尘世，
第二滴暗红色的血滴进决绝的无情，第三滴紫色的血滴满整个黄泉，
万物湮灭，天地皆震。最后的一笑是她最美的一笑，飘荡在烟锁的重
楼之间，飘荡在茫茫的尘世之间，但是却怎么也不愿飘荡在来生的风
花雪月之中，生命凝固在这凄然的一笑中，什么都不重要了，什么都
不想要了。

　　天上、人间、地下同时旋转着。

　　旋转让生命重续着、变化着，只有那一笑的颜色凝固着，只有那
一笑的过程漫长着，看天河汇聚流散，看世间风云万变，看黄泉静谧
惨烈；等待着，等待能让这笑融化的人，等待着能让这笑重新塑造的
人。

　　所以，一切继续着

　　花开了又谢，风来了又去，人变化着，物变化着，只有情如雕塑
般屹立着，看歌舞升平中的嬉笑怒骂，看风烟乱世中的悲欢离合。叶
子终于落到了地面上，在雨水的侵蚀中慢慢地化成泥土，在这泥土的
深处隐约地歌唱着涟漪般的落魂之曲……
　　来抚摩我的容颜
　　虽然我的容颜已经融化在深暗的颜色中
　　来倾听我的声音
　　虽然我的声音已经溶化在透明的泪水中
　　来将我拥抱
　　拥抱在你温暖的怀中
　　来将我燃烧
　　燃烧在你深情的眼中
　　我的双手是你感觉到的冰冷
　　我的嘴角悬挂着绝世的美丽
　　我不是诱惑的深渊
　　我只是忘情的溪流
　　我不是痛苦的源泉
　　我只是生命的重塑
　　来吧
　　你是我平复的烈火
　　来吧
　　我是你融化的凝冰
　　来吧
　　我是你生命中最后的脚步
　　来吧
　　你是我魂魄中无忧的精灵
