曹雪芹的情结

○霜叶

　

　　在《红楼梦》一书中，我们可看到曹雪芹细腻地描写了大观园内
一班青年男女的爱恋情感，其中贾宝玉的情结，实际也显露了曹对情
爱追求的看法。男女之爱，他追求的是灵与肉的结合，鱼与熊掌不能
兼得之际，他舍肉求灵。
　　林黛玉、晴雯一类的女性，是曹理想情爱的精神追求；秦可卿、
袭人一类的女子，则是曹现实生活中肉体结合的本性。
　　林妹妹才高八斗，傲气傲骨，视功名利禄如粪土，对封建的婚姻
伦理道德敢干用行动去抗拒。宝玉对她的爱，在于心心相印，在于心
有灵犀一点通的心灵领会。他俩爱得纯洁，纤尘不染，完全在于感情
上的默契和沟通。晴雯的形象，则弥补黛玉的不足。林好使小性子，
身体较弱。晴雯则性格开朗，为人热情真率。不管是小姐或是丫头，
只要达到曹认可的情爱美，他一律持赞赏心态，一篇《芙蓉女儿诔》
感天动地。宝黛的爱情悲剧，引多少人泪流？宝晴的亲密关系，教多
少人欣赏？
　　人既有七情六欲，性爱是肉体的满足。女人，特别是那些身躯娇
妍者，有那个男子见了不动情？秦可卿生得袅娜纤巧，袭人则柔媚姣
俏。宝玉从不拒绝这“艳色的引诱”，一个在太虚境中与他作起了儿
女事来；一个则在宁府中与他云雨温存了一番。
　　有灵有肉的情爱，是宝玉的梦寐追求。
　　而《红楼梦》中的薛宝钗，则总给人是曹情爱中灵与肉的结合体
人物感觉。一则她具有林妹妹的才情，在“秋爽齐偶结海棠社，蘅芜
院夜拟菊花题”一回书中，曹通过探亲口中对薛、林两人题菊诗作了
这样的评价：潇湘妃子当居第二，言下之语薛为第一也。薛也具有秦
可卿、袭人的娇娜身子。书中对宝钗这样刻画：生得肌肤丰泽，雪白
胳膊，脸若银盆，眼同水杏，唇不点而含丹，眉不画而横翠。这样的
身躯直教宝玉羡慕不已。
　　当然，鱼与熊掌自古难以兼得，曹深明这一道理，最后他惟有取
灵弃肉，而使薛最终复归为一个施小惠而买人心的现实女子。没有了
“灵”的薛，宝玉弃她出走，也就必然了。
