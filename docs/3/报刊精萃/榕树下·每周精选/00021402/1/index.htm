【编辑的话】

　　这是个祝福的季节。
　　早晨起来看见一地的烟花，还有余温。上海在今年冬天有一场大雪，但是节日期间，感觉到春暖花开的气息。听见幸福和快乐的脚步，在不远的地方。洋溢着温柔和玫瑰的岁月，那时的我们最爱奔跑在草地上，爱听鸟儿的歌唱。玫瑰、百合、美人蕉和天堂鸟
　　问谁最爱这些礼物？
