跳鼠的故事

○晓雨

                     一、陌生的客人

    跳鼠，一种四不象的小动物，长着一双兔子般的耳朵，猪一样的
鼻子，牛蹄子似的脚和一条长长的大尾巴，走起路来一跳一跳的。两
只小小的前爪总是蜷缩在胸前，就像人的手一样，已经失去了其行走
的功能。两条后腿粗壮而有力，与粗壮的尾巴构成了它身体的最佳组
合，三足鼎立的特点是站时稳当，行时如飞，跳一步约半米左右。小
小的个头，与我们平常见到的老鼠差不多，但却没有老鼠那一副贼头
贼脑的模样，一双乌溜溜的大眼睛倒也显得十分可爱。
    初识跳鼠大概是八月份前后，记得那时西瓜已经上市了，那天我
们从安西县城买了好多的西瓜。晚上正吃西瓜时，那个小家伙一跳一
跳的进了帐篷，大大方方的到帐篷中央视察了一圈，又大摇大摆地跳
了出去。虽然我们互不相识，但它那副神气的样子，仿佛是在告诉我
们它才是这儿的主人，而我们不过是一群过客而已。但事实的确如此！
    那已是多年前的事了，当时我们工作在甘肃省安西县的一个叫做
白墩子的地方，任务是进行为期两年黄金堆浸试验。那儿距离县城约
四十多公里，周围是茫茫无际的戈壁滩，有生命的东西除了几棵稀稀
拉拉枯黄的骆驼草，就是那股小小泉水旁的一棵红柳树了。动物几乎
就没有，如果不是小跳鼠的出现，我坚定的认为那儿就没有动物，这
个观点虽然坚持了两个多月，甚至还打赌赢来一包红塔山香烟与半斤
新疆的莫合烟，但最终还是被那既招人喜爱又令人厌恶的小家伙给否
定了。
    跳鼠光驾临的那天晚上的确叫人兴奋，因为它使我们不再觉得孤
单，原来在这荒凉的土地上，还有另外一种生命存在。
    但是，第二天让人头疼的事情就发生了。
    因为它们实在太多了，不仅大方，而且贪得无厌，丝毫不理会我
们是否欢迎它们做客，大摇大摆的进了帐篷，这儿看看，那儿翻翻，
最不能令人容忍的是它吃肉，一夜之间就将食堂刚买来的肉吃了许多，
但是被它吃过的肉我们也无法再吃了，只好全部归它们所有。
    都是在那荒凉的地方求生，虽然艰苦一些，但我们是用自己的双
手劳动，才有肉吃，有饭吃，它们凭什么就那么不客气地坐享其成呢？
看来和平相处已不可能，只有搏斗才能保卫自己的劳动果实。


                       二、肉搏战

    其实我们在那儿很孤独，很希望有生命的东西与我们能和平共处，
但是小跳鼠却不愿这样。由于它们的到来对我们的生存带来了威胁，
我们不得不奋起反击，保卫我们的口粮。初次交战，我们就感到它们
的确不好对付！
    第一天晚上，是我和赖子一起去一试身手的。帐篷外面，有活蹦
乱跳的一大群跳鼠，我们手拿木棍，就冲了上去，但它们并不简单，
不但不怕我们，反而很有耐心的与我们斗智斗勇。我们追击的时候，
它们就拚命的跳，虽然只有一步之遥，但就是追不上。有时候快追上
了，当手中的棍子快要落到它们身上时，它们却往旁边一跳，然后站
在一旁，看你气得吹胡子瞪眼睛。我们停下不追了，它们也就停下不
跑了，有时还会反过来追你。那天晚上，我们费了九牛二虎之力，辛
苦了一个多小时，除了意外的打死了一只以外，毫无收获。
    毕竟我们还是打死了一只，大家得知这个消息后都跑出来了，想
看看这个小家伙到底是什么样子。然而大家拿着手电找到尸体时，竟
发现它被一群跳鼠围着，但不是默哀，而是你咬一口，他咬一口的撕
咬着尸体，不过几分钟的时间，尸体就变成了一副光秃秃的骨架了！
惊诧之余，我们才知道跳鼠这种貌似温顺的家伙原来是一种非常凶残
的食肉动物，对于同类的尸体也不会放过。看到这一幕情景，我仿佛
明白了它们所以能在这荒无人烟的不毛之地生存下去的原因了，那就
是不断的吞食同类，才能得以生息繁衍。
    不过使我们高兴的是发现它们只要被手电一照，就会停在原地不
动，虽然时间只有一分钟左右，但足以使我们有机会来战胜它们。
    第二天晚上，我们改变了进攻方式，两人相互配合，一人打着手
电，一人拿着棍子，在跳鼠被手电照到的那一瞬间，冲上去，对着跳
鼠狠狠的打上一棍子。这天晚上收获还真不小，总共打死了十多只，
但那仅仅是包围我们帐篷的跳鼠的的几十分之一或几百分之一。看来
还要继续总结经验，不断的改进方法，才能进一步扩大战果。
    赖子是我们从湖南请来的技术员，他是个很爱钻研的人，经过两
个晚上的战斗，他研究出了很多对我们有用的东西。他发现跳鼠所以
能跑得快，主要是那条粗壮的尾巴在起作用，如果将它的打断了，它
就寸步难行了，但是用木棍是不容易打到尾巴的，因为木棍无法拐弯，
用绳子可以，但绳子太软，使不出劲。
    第三天白天他就不停的寻找工具，终于被他找到了——那就是电
缆皮，因为是用橡胶制成的，既有重量，又有弹性。的确他的这个发
现使我们的工具得到了进一步的改进。
    第三天晚上可以说是战果辉煌，用赖子改进的工具，我们取得了
决定性的胜利，一夜之间打死了数十只。但是第四天早上那些尸体又
全部变成了光光的骨架，看来问题并不像我们想象的那样简单。


                         三、大决战

    我们不知道那些尸体是怎么一夜之间全部变骨架的，但是我们却
感到这些小家伙的确不简单，前仆后继的的队伍和宁死不屈的精神，
使我们又感到了危机，如果我们单凭电缆皮和手电，恐怕是无法与它
们相对抗的。
    也是一个偶然的机会，使我们发现了一种更为有效的办法，用这
个办法我们很快取得了决定性的胜利，当然这个功劳还是赖子的。
    第五天晚上，一只跳鼠的尾巴被打断后，赖子没有将它打死，而
是将它活捉了回来，他要进行一项研究。我们不知道他的葫芦里卖的
是什么药，但是他的确想到了用药。在那儿我们是没办法买到鼠药的，
即使能买到，鼠药能不能对付这种跳鼠还不知道，因为它们根本就不
吃粮食，只吃肉。赖子就是利用了这一点，很快解决了问题。
    那天晚上赖子将那只跳鼠放到帐篷里，默默地对着它坐了很长一
段时间，虽然那只跳鼠的尾巴被打断了，但是它仍不屈服，挣扎着挪
动它受伤的身体，却寸步难行。忽然，赖子象是想到了什么，从床上
跳了下来，拿出了试验用的氰化钠，倒出了一小点，喂进了跳鼠的口
中，约半分钟后，跳鼠浑身打颤，很快就伸直了四肢被毒死了。然后
他将那只毒死的尸体放到门口，熄了灯，等待奇迹的出现。不一会，
一群跳鼠就围上来撕咬尸体，可是凡是吃到了死鼠肉的，走不了几步
就全部倒了下来。
    第六天晚上，我们将十多只跳鼠的尾巴打断后活捉，然后给它们
喂上了氰化钠，再将它们全部扔到了帐篷外面……
    奇迹就这样出现了，第八天早上，我们在帐篷周围发现了许多尸
体，是有血有肉的尸体，而不是被啃光了肉的骨架，我们将它们全部
收拾起来，浇上了汽油，为它们举行了一个隆重的葬礼。从那天晚上
起，直到我们结束试验，再也没有见到成群结队的跳鼠了。但是我相
信，它们绝对不会全部消失的，因为它们有聪明的头脑，一定会找到
属于它们的有生息之地。
    这就是我难以忘怀的跳鼠的故事。离开那儿后的，我再也没有见
到过跳鼠，后来听说在我们试验场上建起了一个小型的黄金堆浸基地，
有人在离那儿约十多公里的地方，见到过成群的跳鼠，但是跳鼠却再
也没有光顾过那个基地。
    虽然它们未能与我们和平相处，但我不愿它们因为我们而失去生
存的权利。令我欣慰的是它们还生存在那片土地上，并没有因为我们
的而灭绝，并且知道了怎样生存：不与强大的敌人对立，和平的与所
有的生命共同拥有那贫瘠荒凉但又神奇的土地。
