网络与女性


○阿黎

    我是女性，我触碰了网络，网络给女性带来了什么呢？上网三年
的经历让我感受到很多很多……
    网络，它首先带给了我们女性――公平！
    在一个只需要良好地运用手指和脑袋的网络世界中，体力不再是
竞争的要素，所以在网络世界中，女性首先在这一点上得到了公平。
也许有人会小看这个因素，但就是这个因素决定了人类从母系社会进
入父系社会！几千年的人类历史就是受它影响而决定的！怎么可以轻
视它？网络带给我们的这一点点公平很可能预视着人类新纪元的到来！
    网络还使我们避免了一种人类已形成的心理定势，那就是在现实
中女人一站出来首先就要被人评品样貌而不是心智，男人却不这样。
一个男人皮肤黑些、嘴巴大些丝毫也不会影响人们对他事业能力的评
价，而女人长得丑会让人怀疑其工作能力，长得美又会被下意识里安
上胸大无脑之类的歧见。而在网络上，女性贴出贴子，发表文章，大
家只读到她的思想，直接品评她的心智，不受样貌的干扰。
    另外，在性别因素上网络女性得到更多的公平对待。比如我应聘
做兼职网页设计员或论坛主持时从没被问及性别，网上招聘这些职位
时也很少会写明要男性或女性的，而在现实中的招聘广告却有九成写
明性别。这说明了性别在网络中没有现实中那么重要。
    更让我高兴的是在现实中工作时我们会因为服饰、化妆等因素影
响上级或同事对你工作能力的评价以及被分配到任务的机会，有时不
化妆是失礼的、而有时化妆又是不应该，让女性要把大量的精力浪费
在斟酌这种事情上，网络却让我们能更专心于工作。在网上工作还有
一个更大的好处是不会遇到象碰撞一下身体或捏一下手之类讨厌的性
搔扰，嘻嘻。
    网络，它还带给我们女性――新的、更适合女性的信息渠道。

    一般来说女人不太喜欢看报，这好象是一个共病，不知是不是过
去的传统遗留造成的，好象看报纸是男人们的事情，我们常常可以看
到男人从一大堆报纸堆里爬起身来，而很少看到女人这样，但这个习
惯不可质疑地影响了女人得到的信息量。不过自从有了网络以后，那
么多的新闻站都把最重要的新闻放在最醒目的地方，凡是浏览公众网
站的人不论男女都自然而然地会看到一些重大新闻，过去女人那种两
耳不闻天下事的情形也就不攻自破了。
    而得到更多的信息其实对于女人是那样地重要，不但让她与丈夫
或男友之间多了许多的共同话题，而且眼界宽广了、知识丰富了、心
胸开朗了，而这些对于提高一个女人的素质修养是有极大好处的。
    网络，它更带给我们女性――充份参与社会的契机。
    网络有一大好处是让女性更充份地发表自己的观点。在网上，对
一条新闻、一个故事、一段评论，你都可以马上加以回应。也许有人
说这对于男人或女人不是一样吗？不，要知道一般来说，女性胆子比
男性小，行动的冲动也不如男性足，男人很可能会跑到邮局去寄出一
份意见书时女人最多是和好友谈上几句而已。但网络减少了跑邮局这
样的行动阻力，这就让没有男人那么具有行动冲动的女性有了充分发
表她们见解的机会。
    女人只有经常站出来说话而不是躲在一角旁观，她才能更主动地
进入这个社会，从而改变女性在社会中地位。
    网络，不但给我们女性带来了那么多好处，而且在网上女性还有
着许多优势。
    比如女性制作的网页总是非常优美，不论是颜色、每一个细小的
图标、又或是底图都充份展现了女性的爱美天份。浏览女站长的主页
简直就是一种美的享受，而男性站长们一般却把更多的精力放在实用
性上，比如提供软件下载、技术咨询之类，在美感上就比较少顾及。
所以常常可以发现他们的主页每一个页面都是单一的底色加上一些分
框，如此而已，除非正好有你想要的软件，否则他们很难留住人们的
视线，也很难让人记住他们。
    女性在建立网站上还有一个优势，就是情感细腻，兼且文笔优美。
她们用女性独有的动人文笔为你描绘了一个个在现实中看不到的奇丽
天地。而且一般来说女站长们建立的主页内容正是人们最需要的心理
读白，在平时的报纸上杂志上你是看不到这些东西的，而网络给了女
性发挥这一特长的机会。你打开一个女站长的主页就一定象是打开了
她的心，而打开一个男站长的网页一般来说看到一个技术或图片超市
的机会更多一些。
    而且女性站长会很好地体会每一位读者，然后给予最合适的回映
和勾通，体贴地关心着每一位留言的朋友。她们细致的文笔就象一只
细软的手在抚摸着你的心，是那么舒适，那么愉快。而这一切如果在
现实中去进行，会引来无数的误解和猜测，网络却给了女性这样好的
一个机会去充份展示她们的温柔细腻，不会给她引来不必要的麻烦。
结交异性挚友在网下可以说是不敢想象的事情。
    同时，网络这个新生事物给了女性一个自然地展现自己的机会。
在现实生活中女性如果尽力地展现自我极易引起非议，因为这有违我
们一向的传统道德观念。但在网络这个新生事物中这已成为一种大家
都能接受的合理方式，这对于女性来说真是一个福音。
    我上网已三年多了，上网的时间越长体会到的好处就越多。同时，
我也很好地利用了这些有利之处，在网上建立了自己的一番新事业。
    我从二年前开始制作自己的个人主页――梨香剑影集，在那里充
份地展现着自我，有“走遍天涯、书山我径、青藏路上、刀光剑影、
云裳影鬓”等几大主题。里面所有的文章都是我亲笔所写，每一个插
图都是我潜心制做，每一个小图标都是我精心挑选，每一段音乐都是
我灵魂的回响。
    自从它建立以后留言版上写满了网友们称赞和关心的话语，更有
网友专门为我的网站提诗做画，象“梨花香暮雨，剑气寒春风。诗情
流水意，琴心浮云踪”这样的留诗成为我心中永远的温暖。正因为在
这个主页中我充份展示了自已诗、书、文等方面的特长，于是结交了
许多在现实中没机会也确实从没有结交过的文友们，大家以文会友，
诗来赋往，真是从没有过的快乐，从没有过的知音佳感，网络给我带
来了许多的挚友和无比的快乐。
    由于我的主页制作得不错，被越来越多的人所认识了解，也有了
各报纸杂志前来约稿，还有记者来采访，更有大型网站来谈收购，还
有请我帮手策划建设新站或主持频道的。现在我仿佛看到了一片新的
天地展开在我的眼前，也许将来我真的会脱离朝九晚五的上班族生涯，
真正地在网中生活，这真是一个出人意料的结果。
    网络，它给了我们女性一个全新的世界，一个充份展示才能、发
挥特长的机会。我们应更好地珍惜这个机会，把握好自己的未来，在
网络中建立一片精彩无限的新天地！
