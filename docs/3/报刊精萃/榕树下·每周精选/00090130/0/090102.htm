当我们无奈面对死亡……

○上海　jency



    这几天，陆幼青的故事可以说牵动着每一个他认识与不认识的人
的心，只因为他在不幸要面对死亡时，拿出了比常人更多的勇气与胆
量。不错，我相信在这世上，每天都会有很多的人要无奈的去面对死
亡，陆幼青并不算特别，但能有这份坦然与从容，就已经是他的过人
之处了，当我们的生命已经到了要用几年，几个月，几小时甚至几秒
来计算时，我们能说自己会像他那样从容吗？我们能说我们也有他那
份与死神相约的勇气吗？
    对于死亡这个词，人类具有一种与生俱来的恐惧感，无论你是一
个平民百姓还是一个受人尊敬的领导人；无论你是身处平凡岗位还是
身居要职；无论你是一无所有还是有着令人羡慕的家庭，事业，也无
论你是一无是处还是有着赫赫战绩……当你知道自己将在不久的将来
要与这个美丽的世界，要与很多很多你舍不得的亲人告别时，我相信
很少有人会像陆幼青那样乐观与豁达，从容与坦然。毕竟人死不能复
生，况且还有很多很多的事要等着你去做，有很多很多的责任等着你
去扛。如果你是为人子女，那么你的父母还需要你的服侍；如果你是
为人父母，那么你的子女还需要你的照顾。如果你是一位普普通通的
职员那么你的公司还需要用你的智慧为其创利益；如果你是一位桃李
满天下的老师，那么你的学生还需要你的教育与指导；如果你是……
总之会有很多很多的理由让你放不下，但当死神已经无可避免的来到
你的身边时，当你已经在无奈面对死亡时，我们是不是也应该向陆幼
青学一学呢？
    曾经听说过许多癌症病人在得知自己的病情后寻死觅活的故事，
他们不是怨天尤人就是天天以泪洗面，有的甚至会动自杀的傻念头。
所以当我第一次听说陆幼青的故事时我有点不敢相信，我很难想象当
一个人的生命已到了要用天数甚至小时去计算时，还会有这份闲情逸
致去写日记，去感受痛苦，去记录这些只有他才体会得到的东西，但
陆幼青却确实让我相信了。以后的日子里，我天天作客榕树下，为的
是向自己证明乐观的人是不会这么容易死的，我甚至期待着奇迹的发
生。因为只要看到他的日记就证明他还好好的，乐观的生活在这个美
丽的地球上。
    当我们被身边越来越多的美好事物所吸引，当我们对这个美丽的
世界越来越留恋，当我们被身边越来越多的人所需要时，对于死亡也
许我们又更多了一份恐惧，所以我们有理由要好好珍惜自己的生命，
毕竟生命于我们每个人都只有一次。但当我们要无奈面对死亡时，陆
幼青就为我们做了一个很好的例子，那就是只要活着就要乐观豁达，
哪怕生命会随时随地消失，哪怕死神已与你定好时间，也要拿出百般
的勇气与胆量，直到生命终止的那一刻。我相信，即使以后他无缘于
这个世界，他也已在生命的最后一刻给他的妻女，给世人留下了一笔
宝贵的遗产，那就是他的乐观精神和他用心与生命写就的“死亡日记”。
