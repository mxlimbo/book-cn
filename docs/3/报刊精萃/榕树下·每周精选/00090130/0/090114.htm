殷墟

○北京　蜀乾尔

　

开启历史大门的声音
是朱红木轴和木板的私语
它曾在荒山里扎根、生长茂盛
在祖先的斧头下定型
    在文明里温暖定居

用钢筋水泥还原的都城
宫殿辉煌、道路更平整
铜鼎更坚定、也更冷竣
甲骨上的文字更清晰
更刻骨铭心

人挪活，树挪死
在汉白玉像前
唤一声盘庚
唤一声武丁
    唤一声淹没我们的
        人民

人躺着，化为尘土
铜鼎却站着
人睡着
梦想却醒着
