平民的快乐

○北京　素颜无痕


　　 
    说实话，我不是一个善于讲故事的人，但不久前发生的一件事却
让我有一种开口的冲动。那是一个发生在我身边的普通人的故事，甚
至可以说不是故事，因为没有惊天动地，也没有缠绵悱恻，所以，如
果您有耐心的话，就请沏上一杯清茶，蜷坐在沙发上，听我慢慢道来……

    她是我以前的同学，他是我现在同学以前的同学；两个人毕业后
都没有朋友，我们宿舍卧谈时一撮合，两个原本不认识的人就这样牵
到了一块儿，而且引出一段想不到的好姻缘。
    早就听现在的同学打了预防针：她这位老兄可是有些“丑”，但
人很好；俗话说“郎才女貌”，男的嘛，有才有德就行，相貌本不是
很重要的，但话说回来，倘若用“丑”来形容了，倒还真的得掂量掂
量──虽然我这位女同学也算不上漂亮，但终归是过得去的，而且以
前从来没有谈过恋爱，但她在饮食上很挑剔──不吃肉，不吃……，
这样的两个人，会彼此满意吗？然而结果是两人认识不到半年，就闪
电式的完婚了，而且听说颇为幸福（他们这次是补办几个月前的只领
了证的不足，请大家一起聚一下），带着一些疑惑和好奇，昨天，我
参加了他们的喜宴。在那里，我似乎找到了答案。
    一早，我们一行匆匆赶到他们家，虽然只有一居室，但不大的空
间，却温馨而洁净着。刚进门，女主人迎上来，笑容满面的她，一身
水蓝的套裙，中穗，手上一条漂亮的相思链；而且，让我眼前一亮的
是，一向疏于修饰的她也涂了口红，亮红的那种，自信而喜庆的颜色
──刹那间，都要认不出了！虽是许久没见，但相比以前那个怯怯的、
毫不扎眼的小姑娘，眼前的她还是让自己有些难以置信。彼时，她先
生正在厨房里忙活，见我们来了，忙出门张罗：“啊，是我们**的娘
家人来了──  欢迎欢迎！”桌上摆满了切好的西瓜、伊丽莎白，和
各式的糖果──看来是他的手艺咯──我悄悄打量了一下这个曾一直
只在谈话里的人物：不高的个子，黝黑的面庞，说实话五官的确长的
算不上好看，但那发自心底的笑容和一见如故的随和与亲切，还是让
把那个“丑”字早抛到了九霄云外。 
    一一等全了昔日的同学，大家来到早订好的酒店，席间，她先生
（他姓E ，我们姑且称他老E 吧）一直不停的奔前跑后，敬酒时，他
们一直手拉手，和善而幸福的笑着，不时悄悄的相互强彼此杯里的酒
换成啤酒和饮料（他们均不善饮──彼此说的），当然后来还是逃不
过大家的眼睛，在一阵哄闹声里，于是老E 又总是豪爽而慷慨的一一
代劳，微红的脸上，始终洋溢着掩不住的笑意。闪光灯的捕捉下，风
趣的老E 总是不时将手搭在同学的肩上，同学也小鸟伊人的甜蜜的笑
着，有些腼腆，却又幸福着。此刻，两个在外人眼里或许十分不扎眼
的人，看上去却是如此的和谐和美丽。
    爱情，可以滋润一个人，男人和女人，他们因为相爱而更加动人
和美丽。   
    后来，我们看到了他们去海南的照片，说是他们，其实绝大部分
是我同学的“个人专集”，用老E 的话说，是自己惟恐有损市容，还
是罢了吧──说这话时，老E 的脸上依旧是风趣而诚恳的笑，那样的
笑和话语自然的从口中流露出来，我想一个人要有怎样的爱心、自信
和豁达才能做到如此的宠辱不惊，谈笑风生呢？ 
    老E 是专门研究农作物的，单调而辛苦的工作。他却总是能在一
种轻松、风趣的氛围里将其中的辛劳化作流水轻轻道来。老E 一出差
就是几个月，而且大都是到农田里一窝就一天，去海南被他戏称为是
两个人去种地，有时在摄氏40多度的室外一蹲就是数个钟头，回到家
满身的泥土和汗水；老E 说**嫁给他他很知足，老E 还说**不爱吃荤
那他就尽力多买些素菜……说这些话的时候，老E 一直微笑着，丝毫
没有炫耀、抱怨和沧桑的味道，听他讲那些经历和勤苦，倒象是在诉
说另一个极不相干的人的故事似的，而同学也一直笑笑的听着，怜爱
而幸福着。 
    照片上，是金色的海滩，和碧蓝无垠的大海，古老而繁茂的热带
雨林千姿百态，最让我眼前闪亮的还是我的同学的每一张似曾相识而
又有些陌生的笑脸，从前，似乎很少见她那样灿烂的笑过，原本不漂
亮的她每一张照片上都光彩照人，尤其是那自然流露出的发自内心的
快乐在脸上的折射──灿烂，幸福，快乐，如此的具有感染力。
    同去的同学里还有一对，是属于“很上台面”的那种，男的颇有
些“款爷”味道，女的则贵妇一般，雍容和华丽着的，不由还回忆起
从前曾经参加的颇为隆重的婚礼，司仪，鲜花，婚纱，摄影，蛋糕，
音乐，一片歌舞升平的气势，相比之下，昨天的婚礼简直是简单再简
单了，但不知为何，平凡和普通的两个人，手挽手默默而幸福的笑着，
却让我更加的羡慕和感动。而原本再普通不过的两个人，此刻也那样
的美丽和亲切，感染着在场的每个人。
    许多时候，我们感慨为什么老天如此不公，没有赐予我们财富、
美丽和幸运？我们抱怨世界越来越冷漠，快乐如此难觅和企及。但更
多时候，我们在感叹和抱怨之余，又真正做了什么？做了多少呢？
    不知为何，脑子里就冒出“平民的快乐”这个字眼，用在这里可
能并不合适，因为其实我们大家都是平民而已，但不知为何，就是这
样被静静感动和启迪着，一时想不到更恰当的词语来形容给我的震撼
和感受。
    快乐、幸福，其实有时就这样的简单；没有阶级，不分贫富，关
键是要看你──有没有一颗发现和经营她的平凡心。

    ――好了，故事终于讲完了，夜幕也开始低垂下来，窗外传来小
孩子们的欢笑和嬉戏……那么，亲爱的你，是否也已打起行囊，准备
找寻属于你自己的快乐了呢？
