有风的日子

○江南梅

　

    有风的日子真好。

    你的消息云朵一样飘来。虽然没有问候没有祝福，但我知道你依
旧美好地存在。

    山群远远地沉默，天空水洗过一般。鸟儿都到哪儿去了呢？不挂
露珠的树梢，颤颤地晃动风的影子。

    这样的季节，风总是随雨而来，我昨夜的诗稿，至今还留有风雨
的痕迹。

    那么谁会是一抹阳光，烘晒我泪痕点点的心笺？

    “不要问我从哪里来，我的故乡在远方……”我相信风是唱着这
支歌走来的；我相信风的流浪只是为了寻找一段前世未了的情缘；花
朵都被风吹瘦了。

    谁是那个模仿新月柔弱如水的女子，谁是那个独倚栏杆对月吟风
的女子啊？

    所有的树都绿得像梦。连走过的风都绿了。

    而在寂寞的山中，依旧有杜鹃开得如火开得执着，开得杜鹃鸟也
忘了“不如归去”的叮咛。

    我还是折了几枝回来，插进案前的花瓶。我是要细细吟味那份自
开自谢自欣赏的寂寞情怀。

    与你执手看月的日子，都流水一般地去了。我独自守着那间梦中
的小屋子，拥菊看云，独自在西风里以芦花为帚，一点一点地打扫，
你遗落的每一丝屐痕。

    风去了又来。穿过秋天又穿过冬天的女人，裙裾飘荡，长发飞舞，
风中的我是如此美丽啊！

    有星子的夜晚，我依旧独坐阳台。

    风踮着脚尖走来走去，轻轻荡着夜的摇篮。花朵都睡去了，不眠
的是我春草般疯长的怀念。

    那支歌写得真好：“满天的星星都是我看你的眼。”我一边数星
星，一边寻找属于你的那一颗。但我怎样努力都找不到自己。

    我望得心都酸眼都涩了。

    真的要等一万年么？

    流年黯淡，岁月沧桑。

    想到多年以后，你在黄昏遇见我，像但丁于佛罗伦萨的街头遇见
心爱的贝德丽彩；想到你目送我风中远去的背影，眼泪像落英在风中
缤纷又缤纷；我是多么渴望衰老，渴望久无人至的心田，宁静地结满
青苔，渴望你握一把陈旧的往事，探访我遗世独居小屋；而那时，那
时我已不在……

    唯有风没有老去，唯有情怀不曾更改。
