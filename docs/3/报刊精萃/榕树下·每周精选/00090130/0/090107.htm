冒一辈子险给感情

○吉林　涤非


　
    人海之中，能够遇上她，这是一种缘分；在岁月之中，准备与她
共渡未来，这是一种冒险。
    有时候觉得两个人很不谐调，就像一根闲草和一棵古松一样，但
他们一旦步入婚堂，到可以平平和和地走完一生。在这一生中，虽偶
尔也会有险情，可那不过是平湖中的涟漪，更添迷人和秀美。有时候
我们看似很般配的一对，就像两株地下同根连、枝蔓共攀缠的青藤一
样，他们成为被人们祝福的新人之后，也未必就能共涉几多水，走到
彼岸，走到终点。
    婚姻是一段说不清、讲不完的古老童话，这童话的作者是每一位
已婚和末婚的成人，还有即将创造童话的少男少女。每一天都有成千
成万的人，喜结良缘。每天也同样有成千成万的人，解除姻缘。当然，
不能说所有的姻缘都将是幸福、美满的。也不敢说所有的离异都是不
幸和不负责任的。
    事有千百种，人有许多情。形势多变，社会复杂。每时每刻都会
有意想不到的事发生。这其中也包括人的内心深处，人的感情的变化。
尘海之中，找到了她，已然是一种幸运。但这却不能说是一段美满姻
缘的真正开始。真正的开始，是在每个人的感情深处：哪怕我们没遇
到任何人，一直捧着颗心，在等待奇迹。只要内心深处，懂得婚姻并
不都是花前月下的美好、海边沙滩的浪漫，也有阴霾蔽日、风雪交加
的季侯，也有风起潮涌、沙走石飞的海滩。婚姻是两个人感情的收容
所、是事业的驿站、是生命完结时的句点。浪子在这里停泊，不再言
四海为家；小女子在这里长大，学会了贤淑和宽容；还有婚姻的结晶，
那些充斥着世界欢笑声的孩子们，他们在这里得到了生命、得到了将
来改造世界、再造婚姻的全部基础。只有明白了这些，才可以试想一
下，自己有没有这份勇气和责任，去完成这种神圣然而又是冒险的使
命。
    好像有太多的人，沉迷于爱情的浪漫中不能自拔，一旦面对无情
的现实生活，茫然无助。他们不愿也不敢面对岁月留给他们的残痕，
想找回却知道永远也找不回当初的影子。他们心已死却又不死心，他
们在上演一幕悲剧而这部剧编、导、演都是他一个人。也好像有太多
的人，对爱情和婚姻看得很开，爱情和肉体，感情和金钱，分得清，
又分不清。概念在这里已经混乱，生存的某些价值也已经混乱。也许
他们很聪明，从开始就怕冒险，怕陷得太深伤得太重。也许他们很糊
涂，根本就不知道爱情和婚姻有多么美好，只知道享乐之中有一项叫
花钱买卖感情。
    既然芸芸众生，千奇百怪也就见怪不怪。人类古老的爱情故事已
经绵延了千万年，如今，不得不承认，你我皆在幸运之中。至少，我
们不用“父母之命，媒妁之言”，我们用自己的心去感悟、发现。至
少，我们不用从一而终，我们用自己的智慧去处理、抉断。
    毕竟，“云只开一个晴日，虹只驾一个黄昏，雪只落一个冬季”。
细细品味最细小的喜悦，回想和她的最初相逢和每一个破泣为笑、苦
尽甘来的时刻，这时候，你就该珍视这段看似简单却来之不易的一世
情缘。最美好的事，只能是一次。最爱的人，一生也只有一个。
    等爱的人，好像要等很久。轻率的婚姻，又有几个能幸福、长远。
孑然一身，也许孤独。两个人的世界，也未必不痛苦。为爱殉情不一
定高尚，为爱偷情不一定可耻。一切，都应该在准备冒这份险时想好。
如果最残酷的岁月的刑罚能够承受，那还有什么可以惧怕的了。
    别光步入婚堂时挽紧她，一直挽着她，走到暮色黄昏。
