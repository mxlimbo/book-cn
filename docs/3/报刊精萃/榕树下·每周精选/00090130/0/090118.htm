植物园里的寂静

○上海　张炽恒

　

十月之秋收敛起翅膀
它的羽毛从枝头落下

你听到那一种寂静里
有谁在说话  那话声如同

滴嗒的钟声  行走在
你的伤口上  滴滴如血

一片一片的火从树的周围仰视
那些敏捷而从容的剪影

它们只是些剪影
从你心上掠过  剪割你的思绪

那么悠长。你听不见
但知道那些拍翅的声音

是的你知道  周围潜伏着眼睛
它们是圆寂了的僧人  它们无语

从风中投来一瞥
那是一种僧人的目光

孩子们的声音远远地传来
那么小  那些跳跃的颜色

他们看不见你  你隐藏着
在树干、树梢或浆果里

在任何一片寂静的颜色中
透过风你向他们眺望
