感声线；他不愿去想难怪西娜咪咪CD封套上的影像为什么总做得这么
模模糊糊不食人间烟火一样；他不愿去想自己原来认为这世上惟一最
真、最自然的东西居然是最假、最伪的东西！
    值得我们歌颂的这个高科技时代一手炮制出的高科技优良产品，
以其非常卓越而成功的姿态彻底玩弄了一个大活人的真实情感。
    这就足以让咱们宽容的上帝在笑话人类一本正经严肃思考的间歇，
也笑一笑眼前这个蒋伍。

    蒋伍合衣缓缓将自己的身体放进浴缸里。水，透明而纯净的水，
从光滑明亮的金属制水龙头里一滴一滴的滴落下来，漫进蒋伍躺着的
浴缸里，一点一点淹没蒋伍的全身。
    蒋伍后来好像做了一个梦。他和一个穿橙色衬衣的人，互相拥抱
着，他们非常非常动情的在那儿亲吻着。吻了很久很久。两片嘴唇对
着嘴唇，极其温柔的摩挲着，好似要把自己熔化入对方身体里的血液
一样。那人的名字也许是叫西娜咪咪。或者是其他别的。
    这没什么不好。真的。也许机器才是对的。因为它其实也是人造
的。
    他们身旁一只黑色的鸟在那静静伫立着。    
    那鸟儿黑黑的眼瞳里闪烁过一轮水一样透明的湿润。
