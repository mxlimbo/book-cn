海的恋歌

○福建　瀛洲海客

  
比星还亮的眸
在我的额前点亮一座灯塔
夜色降临
思绪的野马
挣断缰绳
奔纵于梦的深处

微雨
无言落在双腮
一种渴望
在黄昏中等我
我是漂浮在地球上的海
风是扬起潮汐的鹰
掠过我平静的胸膛
把沉睡的灵魂惊醒
一声吟哦自咽喉飞出
溅满了雪白的浪花

你的手是轻游的桨
由浅及深划过波峰
春水澎湃
夏潮汹涌
来
我们一起谛听
血液的一次长鸣
脉管的一次呜咽
涛声悠远
你我是苍穹下
第一对相遇的爱人

我的呐喊
在胸腔中涌动
你以目光
以不息的温情
横溯我于热血之颠
孤独应声而逃
卸下行囊
我再一次看见
红云在你的发际燃烧

在这静谧的夜
敞开怀抱
解开心中千千结
波光艳影之中
把久已盛满的万种风情
依次倒空
天穹倒塌
我以躯体之礁竖起爱的旗帜
云过处
泪与水
汇成一场暴雨

白浪铺成的毯
在风雨中不安起伏
肢体的火焰拔得很高
一如蛇信
继续波涌
舔过我的脊梁
波涛的呻吟
暴露了夜的隐私

一切复归平静
明天
阳光依然要把水蒸发
你是透明的空气
依然会离我远走
在时间的路口
来也匆匆
去也匆匆

你的裙裾
仍将是海面的云霓
期待风做使者
掠过长空的霎那
吹散几片
深入我的肌肤
如你的热吻
印在我的心底
