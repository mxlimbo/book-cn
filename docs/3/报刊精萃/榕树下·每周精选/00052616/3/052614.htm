人中为什么在嘴巴的上面

○浙江　陈恩裕

  
    我一直都不大明白，中国人为什么把鼻子下面嘴巴上面的地方叫
作人中。中国的文化博大精深，不可能随心所欲把人体七分之一不到
的地方称之为人中，何况中医又是我们的国粹之一，这个稍有文化的
人便知晓的穴位，古人不会胡乱起一个名了事。查了一些书，或许是
因为资料不全，没有找到明确的答案。没有找到现成的答案，就只好
胡思乱想去想出一些缘由来，想出来的或许不是个正理，说出来让大
家一笑也无妨。
    人中谓之为人之中，细细想来有其道理。国术中说上三路，下三
路，这人中所处之地就在上三路和下三路之中，上面三个内外相通的
器官，下面三个内外相通的器官；上面三个器官外表都是明显的双孔，
下面三个器官外表都是单孔。这样说起来称嘴巴上面是人中也就有了
依据。说白了，上三路是眼、耳、鼻，下三路是嘴巴……这下三路除
了嘴巴，另两件说出来有伤观瞻，不过读者看我这样为难，想必也巳
经知道那是什么了。写到这里就感到毛泽东他老人家伟大，他在一部
著作里把上下三路简称为“眼耳鼻舌身”，一个“身”  字就把下三
路最说不出口的那二路名称给囊括了，真是伟大。
    人中在嘴上，先要分析一下嘴是怎么样一个玩意儿，嘴是人体中
最为重要又最不怎样的一个器官，俗话说：“民以食为天”，那这嘴
巴就是天门了，如果不是嘴巴源源不断地给我们输送食物，补充人体
需要的养份，我们就不可能快乐地生活甚至生存。即使在科学相对发
达的今天，人也不能光靠打吊针插管子过日子，而且嘴能辨五味——
甜酸苦辣辛，让我们在吃的中间得到享受，它又能说甜言蜜语，给他
人温情，它还能亲热抚慰，使人们感到温馨。然而它第一性的东西还
是吃和喝，吃了喝了然后才要拉和撒，才有温情和温馨。但是嘴并不
是个好东西，它是很喜欢越轨的，酒后出狂言，食足而思淫，这样一
来，人中下面的三口子常常一起犯错误。俗话又说：“病从口入”、
“祸从口出”，嘴巴实在是很不安全的器官。另两个器官我上面巳说
了，不便在文字上细说它们的优劣，反正大家在公共场所经常看到诸
如：“不准随地吐痰”、“不准随地大小便”的禁示，这样可以悟出
它们的劣迹，知道它们和嘴一样是容易犯错的劳什子了。
    上三路的眼耳鼻就要老实得多，而且颇受人们的重视，那三件真
的件件是宝，所以一有重创就被称之为残疾。眼观五色、鼻闻香臭、
耳听八音，给人的都是精神的享受，非常高尚，因而受人尊敬。即使
是那个品行并不怎么样的鼻子，我们也从未看到哪个公共场所出现过
“不准随地擤鼻涕”的公示，人们对精神劳动者实在有一种特别的偏
爱。眼耳鼻一般不犯错误，它只给人一种感观，它无法独立行动，所
以它们作的，一般说不上是犯罪。当然在政治不民主的社会它们也要
遭罪，如中国的文化大革命期间，看不该看的，听不该听的，甚至闻
不该闻的都会遭遇灾难，如看反动书籍、听敌台、闻资产阶级香风都
有罪。现在当然也有不该看的，但如只是看看，不调动下三路的积极
性，嘴巴不说，其它人中下面的东西也不活动，是不会有什么大罪的。
最近河南焦作“天堂”的那场大火，开头有人（大概是不负责任的人）
说是里面在看黄色录像，不过七十余人巳罹难，即使看了也不会说话，
幸存的几人若是聪明，即使看了也不会在媒体前说三道四，我一连用
两个“即使”是为了确切地表明这是假设。所以即使看了不该看的
（又是一个即使），只要不说不干，那抓进去的几个干部还是可以少
判几年徒刑的。看来我国的法律对眼耳鼻也没有什么办法，它们比较
的崇高和聪明。有罪的只能是嘴巴和它的弟兄们，这与上三路关系往
往不大。根据国际惯例来说就是：思想犯罪不是犯罪。
    我越想越觉得我们的老祖宗聪明，他们把人中定在嘴巴的上面，
将人分为如此上下两个一半，既理清了精神与物质的关系，又划清了
罪和非罪的界限，它让人们明白，上三路运用好了是智慧，下三路运
用多了要成邪恶。
    我们实在有必要知道人中为什么在嘴巴的上面，我们今后也一定
要管好人中下面的嘴巴以及嘴巴以下的东西。
