念奴娇

○重庆　龙在田

  
    寝室有老广，自诩普通话颇好，朗诵苏轼之《念奴娇・赤壁怀古》，
笑料百出。

《念驴叫・吃毕怀鼓》――“吃饱了撑着作驴叫”

    大江东去狼淘净，煎煮烘溜嫩物。骨垒西边，人道是三锅粥狼吃
毕。乱食传统，金刀白案，溅起千堆血，姜蒜油花，一时多少耗竭。
    ――“浪费作料”

    要想公鸡当咽，小巧出佳肴，用金银花，鱼鳝灌鸡，淡少盐，酱
卤烩肥腌毕。多情硬邀我，早盛好饭，肉生油闷，一顿还累九爷。
    ――“做工虽佳，白干一场”
