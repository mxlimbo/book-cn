母亲

○上海　flying_max

  
    我在广州出差的时候，一直想买点什么回来，同事说给母亲买了
一只镀白金的戒指，于是我也买了一只。25年多了，似乎我从来没给
母亲买过什么礼物。我不知道母亲的生日是哪一天，甚至填写档案的
时候，我不知道母亲确实的年龄，55吧，也许是54. 可是母亲一直都
把我当成她最得意的作品之一（当然那个之二是我唯一的弟弟）。
    据说我刚出世的时候，哭声嘹亮，头大肉厚，是个标准的大胖小
子，母亲也因此在邻居中地位升高，生了女儿的邻居，闲时都来抱我，
有外人在的时候，我通常都比较乖，没有闯祸，母亲抱我的时候，却
经常被我尿一身。到了晚上，我更加地不安定，没日没夜不知疲倦地
号啕大哭，父亲那时候正要考研，后来终于考砸了，母亲新做妈妈，
积极性比较高，抱着拍着哄着。我那时不懂事，不但尿炕，而且老哭，
奶糕真的端到嘴边，又不肯吃，明明不饿还乱哭，现在想想都害羞。
    后来我总算大了点，不尿炕也不乱哭了，上了幼儿园，又老是被
小朋友欺负，通常是被按在沙地上给人当马骑，弄了一鼻子沙，后来
鼻子老出血。那时候母亲很心疼，幼儿园阿姨又不大管，母亲就去找
打我的小朋友的家长交涉，每天又早早地来接我回家。那时候我比较
爱面子，老是怪母亲多事，有一个爱告状的母亲，似乎总让我抬不起
头来。
    再大了一点，有一天母亲挺着大肚子问我要弟弟还是要妹妹，我
吮着手指歪着脑袋想了想，说要弟弟，不久弟弟在我的批准下合法地
来到了这个世上。我不知道母亲为什么有勇气再生一个孩子，我那时
听说生孩子要开刀，开刀肯定比打针疼，我连打针都怕，生孩子我肯
定是不敢的，所以觉得母亲非常勇敢，母亲常让我看父亲的刀疤（是
父亲开盲肠炎留下的），说我是爸爸生的，弟弟才是她生的，我恍然
大悟，后来我一直崇拜爸爸，因为他“先生”。
    弟弟来到世上以后，我在家中的地位每况愈下，爸爸一直很宠弟
弟，每次我们兄弟打架，总是我挨骂。我以为弟弟才是妈妈生的，她
也会更疼弟弟，可是她每次都帮我，在我被罚跪不许吃饭的时候，每
次都是她去求情，取消禁令，而我到饭桌上的时候，桌上并不只剩下
残羹冷炙，我的饭碗里，总有母亲从弟弟筷下抢来的好菜。
    母亲替我夹菜的传统一直保留到我读高三，那时弟弟也初一了，
他的胃口极好，桌上的菜只要被他看上了无一能幸免，父亲照例把他
的那份荤菜省给弟弟，母亲的那份总是为我留着。我家的经济条件只
能算一般，一荤两素是经常的，有时我吃饱了，母亲还要我多吃些，
说我高三了，要多吃营养。她用筷子压着我的筷子，不让我把她的那
块排骨或者红烧肉退给她，平时我去上学前还老喜欢叮嘱我多穿衣服，
下雨天要关照带伞，穿套鞋。我那时就象同龄的孩子，倔强生硬，而
且最讨厌穿套鞋，嫌母亲罗嗦，经常在家里一个星期不说话。吃饭的
时候，匆匆地把饭菜往嘴里一拨，完事了就跑进自己房间，塞上耳机
关起房门，留下母亲呆呆地看着饭碗。学校开家长会的时候，母亲总
是拦着父亲，自己去参加。她总是穿着土里土气，一次同学说“你奶
奶来参加你的家长会了”，我赌气跑了出去，下次说什么也不让她去
了。我甚至不愿意和母亲一起出门，因为我越来越高，因为我嫌她给
我买的裤子太粗太大，因为她总想叮嘱我些什么，因为……我怕她丢
我的面子。
    我终于如愿考取了大学，母亲高兴得逢人就说，那时市里有一个
奖学金，考上大学的都有800 块钱，我认为那是我的钱，母亲却帮我
存银行了。按规矩孩子考上大学，要请办公室的同事吃糖，人家都买
了喜糖，高高兴兴地在办公室发一圈，母亲在家算了一算喜糖的预算，
毅然跑去上海，买了许多零装的大白兔奶糖和喜糖袋，晚上回来，她
和父亲点着蜡烛自己封袋。对母亲的小气我嗤之以鼻，幸好她自己拿
去办公室发，不管我的事，想到可以离开家去念大学，我一脑子的憧
憬。
    好不容易到了另一个城市读大学，还是没能彻底独立。在经济上，
我还得靠着父母，这让我浑身不舒坦，好象拿了他们的生活费，就有
义务听他们唠叨。母亲依然保持小气的一贯作风，给我买最便宜的衣
服裤子，总是大得可以套进去一个半我，还说这样天冷时候也能穿，
我以为即使是天冷，和天热时相比，我的体积也不会相差半个人之多。
可是我越解释，这样的衣服裤子越会从母亲每次来看我的袋子里冒出
来。母亲要我省点买衣服的开销，多吃点，因为大学的饭菜只能保持
肚子不饿，营养是没有的。我知道父母要供弟弟念中学，还得供我念
大学，所以我已经很省了，我不喜欢母亲偷偷多塞给我钱。每次回家，
看见弟弟还是那么狼吞虎咽地吃饭菜，我就会在母亲塞给我钱的时候
深深地内疚，我不知道以后我能为他们做些什么。母亲在电话里问得
最多的是每个月的生活费够不够，“够啦够啦，我要去自修了，长途
少打打啦”，我这么敷衍着。
    等到工作了，母亲又为我的个人问题担心，“还没女朋友啊，妈
妈单位里有个……”听到这里，我总会不耐烦地打断她，又要把什么
嫁不出去的同事的女儿介绍给我，烦死了，我又不是收容难民的。母
亲退休了还闲不住，应聘去做了教师，晚上她总比我晚回家，还常常
关照说，她会买好吃的回来，要我等她吃饭。吃完所谓的好吃的（一
般是小巧玲珑的炸小黄鱼，有时是瘦骨嶙峋的酱鸭），她就开始唠叨
学校里的事，如何帮学生找到好的单位实习啦，学生买了什么礼物送
她啦，家长如何感谢她啦，母亲做了几十年的会计都没现在做会计老
师开心过，一天到晚她的学生如何如何的，有一次甚至说她班上的女
学生如何漂亮，幸亏我及时打断了她的非分之想，要我去和十七八岁
的女中专生约会？打死我也不干。
    这次去广州，总算想起买点礼物给她，我在想，她第一句话是不
是会问这个戒指多少钱？幸好我把戒指递给她的时候，她什么也没说，
我想，这时候的我，在她的眼里才第一次懂事了吧，拍马屁的感觉真
不错。母亲使劲地把戒指往手上套着，终于悻悻地取下说，太小了。
“怎么可能，我套在我的小指上还大了一点点”，我的同事就是这么
给他母亲买的，因为我们根本不知道母亲手指的尺寸。母亲伸出她的
手来，我第一次注意到了她的手，又短又粗，好象十根胡萝卜，在我
的记忆中，好象一年四季她的手指都肿着，冬天肿得更厉害，我们家
所有的衣服几十年来都是她洗的，一刹那，我的鼻子酸了起来，我为
什么连母亲的生日也不知道？我为什么连母亲的年龄也不清楚？我怎
么做的25年的儿子？我总是嫌母亲唠叨，和弟弟一起喊母亲“老娘”；
我总是嫌母亲小气，拒绝她买的衣服和点心；我总是嫌母亲菜烧得不
好，吃她烧的菜我从来不添饭；我总是嫌母亲又老又粗又不会打扮，
不想我的朋友同事看见她。我怎么做儿子的？我怎么做儿子的？我一
遍又一遍地问自己。
    母亲终于还是戴不上那只戒指，她把戒指送给了我的外婆，她的
母亲。
