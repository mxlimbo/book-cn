我是一个懒惰的渔人

○浙江　江南篱笆


我是一个懒惰的渔人
三天打鱼 
两天晒网
在船上把诗集一页页撕碎
然后在岸上等待丰收

我种下石榴
小心翼翼地风干花瓣
把它制成鱼饵
这是你最喜欢的粮食
做鱼儿经常挨饿
哪怕花容月貌

你的记忆在黄河深处
冰封时开始发芽
而我 哪里去找一件棉袄
平安渡过冬天

我会找到魔瓶
请求被放逐
化为英俊无比的
一条鱼
洄游在入海口

我是一个懒惰的渔人
我从不修船补网
我被哪个朝代的爱情击中了
不由自主地流泪
