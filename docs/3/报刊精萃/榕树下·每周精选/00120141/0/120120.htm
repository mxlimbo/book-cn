坐在历史中吟唱

○甘肃　长恨


坐在历史中
坐在沉沉文化的氛围中畅想
指与页瞬间触吻
便有了千年的追溯与流逝
穿透时空
就像火光穿透黑暗
到复活的坟墓造访

高楼与繁华
以拆除的方式
还原成碧血黄沙的古战场
违禁的长发
回归为执戈战者的时尚

茹毛饮血刀耕火种的祖先
从白骨里站起并年轻成
婴孩的啼泣
我才明白
我的吉他和昭君的琴瑟弹奏的
其实是同一根弦响

数代帝王的宫阙与豪奢
只是烟云过眼
李白醉歌与舞剑的风姿
使唐朝终于透露出
斑驳的光亮

回头看自己的眼睛
才发现
现代文明睿智色彩的源头
依然是黄河母亲的涓涓泪滴
在刈割后大地的伤痕里
执著流淌
