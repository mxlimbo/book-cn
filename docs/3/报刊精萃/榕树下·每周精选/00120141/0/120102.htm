抚摸

○江苏　檀朗


穿越三座城市
我和你去抚摸一座大山
那里山杜鹃热烈交谈
我们爬上莲花峰，又登上天都峰

如果是在明朝
我们的扬州烽火狼烟
峰颠之上，我们象两只孤雁
迷惘于辽阔和浩冥
今天我们的城市莺歌燕舞并且灯红酒绿
而我们只是固执地在高处盘旋

山顶上的巨石
怎能洞悉他们的欲望、贪婪以及人们悲惨的身世
你开始把回声扔进山谷
石头说出的话语
被鸟衔走
然后又被弃掷在松枝上睡眠
直到来年的积雪将它们压醒

山是一张白纸，瀑布就是白纸上的浓墨
事实上它就这样烙进天空的记忆里
悲伤、孤独、绝望、毫无退路
我多么不愿如此陈述
那时我们只是瀑布之上两朵悠闲的白云

现在我强忍泪水
我要将你这披着风暴的野百合
种在这样的山谷和云间
我惊异你身后被破碎了的万物的幻觉：
你咬着手指看着雁群飞过头顶

而我从未指望这些鸟群
真能将你的亡灵运进天堂
又不让那些光芒白白流逝
