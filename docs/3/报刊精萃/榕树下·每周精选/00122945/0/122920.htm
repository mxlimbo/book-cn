与热情会合

○重庆　何容何易


你有坚实的兴安岭作砥柱
而我孤立长江中流
无依无靠 更无从怀想

你苍茫的林海里
东北虎啸聚山林的一晚
是我怵目惊心的日子
我的帆垂落桅断折魂无归
你温婉细致的呼兰河
飞雪消融冰凌解冻的日子
是我夙夜忧叹辗转难眠的时光
在那些日子
你凌厉的东北信风 常
挥师南下 将我隔阻
在你五彩斑斓的梦里
那些日子
你在北方躬耕田亩
而我在南方流离失所

南中国油漆剥落的门窗
锁住我空寂的身心
却锁不住你排空而至的诗句
我已开始后悔
那个铭心刻骨的日子
和七零年那个不可饶恕的年代
为什么不向你走近一些
为什么总是错过
那雪野 那原始森林

梅雨来临之前的季候
寒流早就已经退出了你的地域
抖开连缀你我的诗节
我看见雪融化开来
大豆播撒开来
生命中的杜鹃花生长开来
——你物产丰富的松嫩平原啊
我可能抵达你深邃的腹地里
那片坚硬的黑土？

——一颗心渴望北上
和热情会合
