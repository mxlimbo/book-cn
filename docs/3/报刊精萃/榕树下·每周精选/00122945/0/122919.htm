我看见整个秋天

○浙江　长歌当梦


我看见了整个秋天
透过我着火的一双眼睛
九月，苍鹰，藏着黄金的树林
以及整个秋天撞在辉煌的天空上
逐渐飞向我的诗章的回声
我看见整个秋天巨大的翅膀
看见它庄严地投入黑暗的
扑动的声响

我看见整个秋天
宛如一个末日的英雄
我看见夕阳，血一样绚烂的夕阳
我看见海的红褐色的披风
让我联想到永恒的弓
苍茫的西风已经搭在弦上
我想象临行的一箭
是带走了填满世界的忧伤
还是放飞了温暖的快乐

我看见粗犷的时间
秋天是春天的昨天
我看见一片片树叶
由死亡，衰老到年轻
在青绿的叹息下新生
我看见整个秋天的咳嗽
那重病的气息让我
眼眶湿润

我看见整个秋天
大地宽阔的胸膛
接纳我所有的伤痛
我看见了整个秋天
秋天看见了整个我
