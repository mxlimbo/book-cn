姐姐

○陕西　临渊羡鱼



姐姐 你在墙壁上的身影
被谁抱走
偷去温柔的人
你为何还不离去

姐姐 再给我做个泪水灯笼
你凋零的美丽
如同鲜花 相互热爱
停止吧 
为了我从此不再写你

姐姐 我们读不懂世界
这个笑过的没有人的地方
身后的泥土
把十指埋葬

采摘葵花的人
流着泪
归入天堂

姐姐 让我们把所有的悲伤
存入骆驼的眼眸
左眼为你 右眼为我
那是一对
最寂寥的泪水灯笼
