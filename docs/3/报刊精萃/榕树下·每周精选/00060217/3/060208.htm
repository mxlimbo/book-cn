江湖往事之刀光

○河北　跳舞

  
    刀光一闪。

    锐利的刀锋沿着一条奇异的弧线，划过夜色，划开我的喉咙，
削断我的血管，断绝我的呼吸。
    我感到喉头一凉，然后看到喷涌而出的血花，红色的血花在月光
照耀下格外的美丽。想不到自己的血也是如此的美丽。这是我最后的
念头。
    我死了。

    几年前，我初出江湖，年轻气盛，籍籍无名。带着一柄刀和一腔
热血，踏上了我的江湖路。教我刀法的师傅说过这是一条不归路，我
知道，可我不在乎，人生短短数十年，我已经寂寞的太久。我要成名！！
我要成名！
    当时，江湖上最有名的三把刀是秦风，花断和宋离离。他们是所
有用刀之人梦寐以求的对手，当然也是我的。只要击败他们，就马上
可以名扬天下，成为江湖上的新偶像！
    我决定挑战他们，我绝对相信我的刀比他们的快，这信心来自自
十三岁起连续十年从不间断的苦练。
    我第一个挑战的对象是秦风。

    秦风，男，四十一岁，出身江南刀法世家横刀山庄，惯用厚背砍
刀。二十一岁初露锋芒，斩杀江南大盗飞燕杨欢。两年后，击败来自
东瀛的刀法名家柳生，自此成名。五年前接任横刀山庄庄主，专心调
教弟子。
    这么样一个人，当然不会轻易接受我这个无名小卒的挑战。直到
我连伤他七大弟子，他才肯出来见我。
    秦风的外表很是温和端正，但他的刀势却异常的霸道凛厉。我差
一点就败在他的刀下，可惜他的年纪已大，刀法已过巅峰。而且他好
象并没有那种强烈的求胜欲望。而这种求胜欲对一个刀手来说，是至
关重要的。
    所以，他败了。那柄曾经纵横天下未尝一败的厚背砍刀，被我一
刀击飞，我随即停手，同时对自己这一刀异常满意。
    让我奇怪的是，秦风并没有我想象中沮丧，看着远处跌落的自己
的刀，他甚至有种解脱的表情。他居然还对我笑了笑。
    他不是一个真正的刀手，这是我的结论。

    争霸江湖的路，我已经走出了第一步。下一个我要挑战的，是花
断。
    花断，男，三十三岁，太行五虎之首，身世不明。十九岁成名至
今，身经大小七十余战，被创无数，而与他对战的高手，大多已被其
一刀斩断！
    挑战花断，并没有那么麻烦。他是一个疯狂的刀手。听到我是来
挑战的，他很干脆地马上抽刀应战。他的刀法也象他的人一样，简单、
直接、而且疯狂。也正因如此，他的刀法并不完美，甚至存在许多破
绽，可以致他死命的破绽。
    我只想成名，并不想杀人。可我没有办法。在花断疯狂的刀法面
前，我只有两个选择，杀了他或者被杀。或者说，我已经没有了选择。
    在他的又一个破绽出现时，我开始了反击，一刀割开了他的喉咙，
红色的鲜血喷散如花，透过血雾，我仿佛看到了花断的笑容。
    谢谢。这是他最后的话。

    尽管第一次杀人的感觉让我异常的不舒服。但后来慢慢的我就麻
木了。我击败秦风和斩杀花断的战绩已经传遍江湖，人人都知道最近
有一个如彗星般崛起的年轻刀手。我已经成名了。这结果就是开始有
人向我挑战。
    我已经记不清刚刚死在我刀下的，是这个月我遇到的第几个挑战
者了。他们和我一样的年轻，一样的自信。不同的是他们没有过和我
一样的刀法。于是，他们死了。而我还活着。
    我还活着吗？？还能活多久？难道现在这种杀人与被杀的生活就
是我当初的梦想吗？？曾经无数次在夜深人静的时候问自己这些问题，
答案是，我不知道。我只知道我已经不能回头。既然选择了这条江湖
路，就只有走下去，直到有一天被别人所杀为止。我甚至有点希望那
一天来得不要太晚。

    这一天来得很快。比我想象中要快。
    是一个有着凄清月色的夜晚，打发掉一个挑战我的刀手之后。无
聊的在路上闲逛，突然感觉到一种很强烈的杀气。我握紧了我的刀，
停住了脚步。竟然莫名地紧张，就象我第一次面对秦风时的感觉。我
已经很久没有体验到了。来的一定是高手，我想。
    是宋离离。三大名刀中的宋离离。他一向是个很神秘的人，也很
年轻，五年前初出江湖，击杀号称武当剑法第一的清云道人，而且在
随后的武当高手的报复行动中全身而退。被认为是三大名刀中最有前
途的人。
    本来他也是我的挑战对象。不过自从成名之后，每天面对的不是
吹捧者恶心的面孔，就是挑战者森寒的刀锋。我已经厌倦了这种生活，
甚至忘记了当初要败尽三大名刀的愿望。想不到宋离离竟然主动来找
我，在如此月夜。

    宋离离的目光冷酷如冰，他的声音也同样没有一点感情：“我等
了你好久，为什么不来找我？”
    “因为我已经厌倦了，为什么要去找你？”我说：“这种杀来杀
去的生活很有意思吗？”
    看着我，他的目光突然变得柔和，“其实我也早已厌倦了这种生
活，不过既然已经拿起了刀，就已经注定要将一生奉献给它。我们都
已无法回头。如果因为厌倦而放弃，就不配做一个刀手。真正的刀手，
就应该死于刀下。我一直在等待一个可以帮我结束这种生活的刀手，
那个人，会是你吗？？”
    听着他的话，我突然明白了为什么秦风败在我的刀下后，会露出
解脱的表情，而花断临死前会对我说谢谢。因为，他们都是真正的刀
手，厌倦了刀手生涯的刀手。
    我长出了一口气：“拔刀吧，我会用我的全力，这将是两个真正
的刀手之间的决斗。我们之中只能有一个人可以看到明天的阳光。”

    宋离离的刀法没有秦风的霸道凛厉，没有花断的迅速疯狂，却是
我见过的最诡异莫测的刀。每一刀都是从绝不可能的角度沿着奇异的
弧线砍出。做为刀手，遇到这样的对手真是畅快之至。相信宋离离也
一定有这样的感觉。

    可惜我们两个人之中只能有一个人活下来。活下来的那个，会是
我吗？？

    刀光一闪。
