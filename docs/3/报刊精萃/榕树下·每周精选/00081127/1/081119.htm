最后一个匈奴

○河南　匈奴


浩瀚的大漠风吹白沙四起
幽凄的古道淹没了先人的足迹
落日的余晖映照着一匹瘦马
孤独的少年如处子般伫立
手中擎着一面血红大旗

沧桑的古树发出无奈的叹息
相当年千军万马金戈铁蹄
现如今尘归埃土骨入烟泥
往昔的辉煌促成了今日的悲剧
灿烂的历史变成了昨天痛苦的回忆

如烟的丝雨萧索而凄迷
汹涌的风沙渐渐沉息
坚毅燃烧在少年的眼里
他始终坚信一个遥远的奇迹
匈奴定会横扫天下风云再起
