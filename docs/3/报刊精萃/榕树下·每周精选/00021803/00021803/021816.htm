奇数偶数和自然数

○玫瑰灰

    1 是奇数。
    1 是自然数。
    0 不是自然数，0 不是偶数。
    0 是……
    
    我从小就讨厌数字，我讨厌那些缠夹不清的东西，我甚至讨厌那
些阿拉伯数字的1 、2 、3 、4 ……天晓得我怎么会这样的。
    一直到5 岁我都不能顺利地从1 数到10，这使我的父母几乎以为
我的智力是有问题的，但是测智商的结果令他们欣慰，高达137 的智
商根本就是个天才。所有人都把这归咎为我的懒惰和狡猾。我忘不了
幼儿园老师看我的眼神：这是个狡猾的小东西，别看她是个小丫头片
子，其实小小年纪就会偷懒了……她看我的眼神象更像在看一个成年
的夙敌。
    从小学到高中到大学都没有老师喜欢我，尽管我的学习成绩是好
的，但是他们和我的幼儿园老师一样，不喜欢我这样的孩子，因为我
的想法是他们所不能理解的。我想，他们排斥一切他们所不能理解的
东西。这样决定我的生活肯定是他们眼里的非常规。毕业以后，我非
常规的选择了我的生存方式，我没有找工作，我决定用写字来养活我
自己，我想一个人过没有数字的生活。每次，我交电话费的时候，我
就更恨那些数字，我的钞票相对于帐单上的数字来说永远都是个绝对
的小数字。
    我和数字犯冲，我恨那些数字。
    
    白天，我窝在我的床上睡我的觉，太阳是别人的，不是我的，那
是我的黑夜；到了别人的黑夜的时候，我开始工作。
    我用我的电脑写我自己的字，我用我的电话费和网费到聊天室扮
演我小说里人物，在那里头，有比我的现实生活更真实的东西，而且，
那里没有数字。
    每到没有钱的时候我就疯狂地写字，把那些字换成可以供给我生
活的钞票。但是，由于我对数字的痛恨，我总是无法让的写字的速度
跟上我花钱的速度，那养成了我非常不好的消费习惯——有钱就疯狂
地花钱买东西；没有的时候就躲在我的六平方的小屋里写我的字，啃
最便宜的华丰方便面。我经常收到中国电信的催款单，一有钱我就交
电话费和网费，我想我还是中国电信的好用户，尽管我常在聊天室里
大骂这天杀的中国电信。
    现在，我曾经美丽的脸常常是浮肿着的，我没有规律的生活和固
定的收入，我的浑身都在可怕的浮肿。到我有钱的时候那些浮肿象就
像贫穷一样离我而去，到了没有钱的时候，他们又卷土重来。
    
    电脑是我的谋生工具。这是我在我电脑惨遭CIH 的时候突然意识
到的，没有电脑的日子，我没有办法写字，我习惯了用键盘打字，用
笔写，我已经忘了是怎么回事。
    那天，是他来修我电脑。我们在古旧肮脏黯淡的走道里认识。我
蓬头赤脚，打开门的时候，我曾经犹豫过一下，但还是让他进来了。
他进来，整个人背光站着，看不清面目，只有看见藏蓝色的夹克和底
下的牛仔裤，还有的就是亮得不怎么和谐的黑皮鞋。他提不起我交谈
的兴趣，所以我就蓬头垢面自顾自地睡我的觉，等我醒的时候，他已
经走了，而我的电脑又开始工作。电脑的边上留了一张他的名片，公
司的电话地址和他的名字。
    他没有向我收钱。
    半个月后，我收到了我的稿费，于是，我去他那里付钱，拿着他
的名片，那名片上只有一个名字，没有职务，我知道一个修电脑的小
人物是不需要写这些东西的。没有见到他，付了钱，回来了。
    电脑真是一样好东西，而能顺利地使用一台健康的电脑又是一件
好事情。那天晚上我的心情好得有点反常，所以我到聊天室准备奢侈
地花掉一点网费。
    悠漫故事的聊天室是我经常盘踞的地方，对于我来说悠漫给我的
虚拟世界远比我身边的嘈杂世界要真实得多。
    因为是很晚了，聊天室的人不多。我看了一下，很多新面孔，这
年头无聊的人是越来越多了。于是我用笑靥的名字进去了。
    在我后面进来的人的NICK很讨厌，是个数字，十分简单，也十分
招我讨厌。他的名字是——1 ；讨厌的数字变成了讨厌的名字。数字
的种种勾起了所有不愉快的回忆，本来很好的心情立即变得很糟糕，
于是，我把我的名字从笑靥变成了夜哭。
                     　　 
    夜哭：数字是讨厌的没有感情的东西。我恨数字！！
    1 ：是我让你不高兴了吗？
    我没有理他。
    夜哭：在世界上数字是最冷漠的东西。
    1 ：那什么是有感情的？
    夜哭：文字。
    1 ：为什么？仅仅因为文字是可以改变的？
    夜哭：没有原因，就是我喜欢而已。
    系统提示：用户1 改名为文字文字：我现在的名字你喜欢吗？
    系统提示：用户夜哭改名为笑靥文字：如果你是女人，如果，笑
靥：怎么？
    文字：我是说如果你是女人，那么你于文字和男人孰爱？
    笑靥：如果，是的，如果我是女人，我想我更爱文字。
    文字：？？
    笑靥：文字比男人更忠实，文字不会背叛我，在某些意义上来说，
男人和一条诲淫诲盗的蛇没有什么两样，他们让女人下地狱……
    我掉线了。
    这样也好，可以节省我的电话费。今晚我不想写任何东西。我把
键盘枕在我的脑后，看着黑糊糊的天花板，脑子里一片空白。硬硬的
键盘硌得我不舒服，使我无法入睡。其实我并不想睡觉，而我也不想
不睡觉。怎么了？我到底想要干什么？
    空气里有华丰方便面的辛辣味道，显示器上血红的www.yoomy.com
的字样在闪闪烁烁。
    一夜无眠。
    我家楼下的罗森便利店的生意好得惊人，每次我半夜来的时候我
都能看到这里的蓊郁的人气，在我有钱的时候，我到这里来买食物。
寂寞的时候我使劲地吃东西，吃很多东西。没有钱的时候，我就喝水，
喝很多水，然后不停地上厕所。那些日子我的脸肿得发亮，而且，疯
狂地发胖，脸上发了很多小痘痘，我是一个真正的丑八怪。

    第二次见他是我的打印机坏了。他来的时候总是在中午，和太阳
一起来到的我黑暗的家里。
    那是我的深夜，我对这样的造访非常的反感。我把门打开后直接
回到我所热爱的床上去了，但没有象我想象的那样立刻睡着，百无聊
赖的时候就观察着那个和阳光一起造访的男人：该不到三十岁的样子，
戴眼镜，老实而儒雅。还是永远的藏蓝色夹克和亮得过分的皮鞋。真
不知道他的鞋怎么会这么亮的，有必要吗？哎？他长什么样和我有什
么关系？我翻了个身把键盘挪到床上来，连上线，又到悠漫的聊天室，
我开始想念那个先叫1 ，后来改名叫文字的家伙。没有他。而平时他
好象永远都在线。
    我又开始想睡觉，缩在床上直犯迷糊。最后，我又一次极不争气
地到周公那里报了到。
    醒的时候是下午六点多，我喝了一大杯开水就开始工作。楼下人
家的吃饭声音，洗碗声音，说话声音不绝于耳。我踢蹋着拖鞋到公用
厨房洗我的衣服，洗我的思想，也洗我的生活。
    那个太阳下面来的男人影子在我脑海里挥之不去。带着的眼镜，
老实儒雅的脸。
    
    深夜。
    文字：哦。是你来了。
    笑靥：恩，是我。
    文字：知道吗？我可以想象到你的样子？
    笑靥：是吗？为什么？你怎么会？
    文字：是的，我可以，我能够。
    笑靥：恩。
    文字：在我想象里，对你的印象很色情。你永远穿睡衣，蓬着头，
还是光着脚的。
    笑靥：恩，那是你的想法。
    文字：你很慵懒，就像妩媚的猫或者是这一类的动物。
    笑靥：是的，我懒。我喜欢睡觉。
    文字：你这嗜睡的女人…………
    每夜和文字在悠漫的幽会成了我的一个指望，或者不如说是一根
救命稻草，在这样的日子里，如果没有文字这样还算正常的人和我交
谈的话，我也许早就疯了。就这样，我象嗜食鸦片的人一般地守侯在
悠漫的聊天室，等待着我的鸦片的到来。人多的时候我在边上静静地
窥伺着每一个过客，无论他变换了什么名字我都能一眼把他认出来；
没有人的时候我用笑靥的名字在聊天室守门，敬业得象像个重新上岗
的门卫。最后他来的时候，我总是会安静地等待他发现我的存在，在
他和别人打招呼的时候，我在推测他的神态和敲击键盘的手势。
                     　　 
    我的拖欠了很久的稿费终于来了，于是我又去付上次修打印机的
钱。我说过，尽管我不是一个好的理财者，但是谢天谢地，我总还算
是一个好顾客，我总能记得我有帐单没有付，至于帐单上有多少数字，
我因为不感兴趣，我记不住。我去的时候，选择了一个阴天的黄昏，
那样比较有黑夜的气质。
    到了那里，看见他从里面出来，和以前看的的不一样，衣冠楚楚
的样子，夹克和牛仔裤换成了藏蓝色的西服和深蓝色的衬衫。这样他
的亮亮皮鞋便很和谐了。他看见我，眼睛里有阳光的影子。他请我到
他的办公室。
    办公室很大，很亮，电脑开着，还在线上，在悠漫的聊天室，文
字也在而我不在。他走进来，坐在我的对面。这个人到底是干什么的？
我开始有些诧异。我就定定地看他，感觉有点冷。大概这里的空调开
得太冷了。
    门口美丽的秘书小姐敲门说：“许总，您夫人的电话。”
    “好的，转进来，”他说，“对不起，请梢等。”他一边和我说，
一边在键盘上飞快地打了几个字。
    文字：“我有电话，工作，大家再见。”
                     
    我浑身的血液流动一下子都有些涩滞…… 

    晚上的上海非常美丽，没有太阳的上海非常美丽，电影院在放一
部讲互联网美国电影《黑客帝国》，海报上有大大的源代码，是0 和
1.依旧是数字。
    原来我所热爱的没有数字的地方依旧是数字组成的。
    我像一个游魂一样在繁华喧闹的夜上海到处游走，最后一点的阳
光的温暖在我身上渐渐消失，我的手脚冰凉。
    你于文字和男人孰爱？
    爱文字，它不会背叛……
    而现在的问题是文字就是数字，冷漠的数字。
    他是1，他是奇数，是自然数。
    我是0，我不是奇数，不是自然数。
    我是什么？？？
