【编辑的话】

    停伫风中，你听到春天花开的声音吗。
    生命甜美的滋味，在季节中轮回，只要远方的呼唤没有停息，心就要远行。
    风景在漂泊中定格，爱情在无言中醇浓。
    某一刻，静坐在榕树下，旧日时光，温情缱绻……
