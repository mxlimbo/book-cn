我的钥匙生锈了

○湖南　杨六郎


隔夜的水如许清凉
热恋中的黄昏满是霞光
可我的钥匙生锈了
找不到地方把这种感觉珍藏

熬过冬天的种子依然饱满
长出新枝的老树不再苍凉
可我的钥匙生锈了
没办法打开向阳的百页窗

爱情来了如梅雨缠绵悠长
我的柔情也似潮水退了又涨
可我的钥匙生锈了
无法清洗积年的惆怅和迷茫

你的笑魇是我昨夜遗失的玫瑰
飘逸的衣袂胜过天边的霓裳
可我的钥匙生锈了
今夜
你无法成为我的新娘
