意大利札记

○北京　西颖


                           出门了

    这次的出国旅游，夹杂在单位分房子，与装修队无穷的理论，上
法语课，考试复习等诸多事项中，加之其本身的繁琐手续，出发前，
我已只知其苦，不知其乐了。
    遍寻《罗马假日》不得，却意外找到一盘名为《意大利假日》的
磁带，从百代公司六十年代黑胶唱片重新翻制：“缆车”钢索从山顶
一泻而下，“桑塔露西亚”低回宛转，反反复复地听，希望能酝酿出
一份感觉。
    我想只挎一个小背包，像言情小说里的旅欧情节。丈夫执意要穿
讲究的衣服。折衷之后拖了一个标准旅行箱。去机场时接连几辆出租
车都拒载，本想一出门把这几个月来的麻烦做个了结，唉！
    不胜其烦的感觉持续到匈航的飞机上。地面小姐坚决未给我们连
号的登机牌，在飞机上与人商量换坐位时，空中小姐更是不愿意理解
十几个时辰需要自己人的照顾，满脸鄙夷地前来制止，我板着脸用标
准英式英语声明自己应有的权利。我早已决定不为任何事情所动，等
我一顿攘扰后终于安顿下来，脸上马上浮现一股人造的微笑。
    铁鸟儿飞起，载着我们这许多人口腾空而起。丈夫是艺术工作者，
照例感叹一番，无法理解这一事实。我从不为这些事情惊讶，对现代
科学的力量熟视无睹。
    我戴上耳机，音乐频道有九个可供选择。我挑了一只听不出是何
种语言的民间谣曲，旋律忧伤，突然想起了李斯特，这个国家有多瑙
河流过，有吸血伯爵的传说，却以恁般面目示人。


                         露天博物馆

    漫长的旅途终于在罗马微蓝的暮色中结束。飞机逆地球的自转而
飞，飞过一个又一个彩霞漫天的黄昏，还是停在了傍晚。汽车沿途一
派异国风光，无奈时差像鼓锤一般把我的头当锣敲得生疼，从餐厅抱
头而逃。该晚是周日的意甲联赛直播，接待我们的一个甜美娇俏的意
大利小姐也很快消失了踪影。
    这位金发的美玲小姐让我想起大学时的一个美国外教，她俩的相
貌好象姐妹一般，美国外教的家庭笃信天主教。此时一琢磨，我已敢
肯定她会是一个意大利移民家庭的后代。我一向喜读各国历史，此行
刚下飞机，便有跨越遥远时空的细节供我琢磨，其乐无穷。
    从第二日开始，我们便卷入旋风般的急行军中。领队似一阵风把
我们撮了来去。罗马非一日建成，建成后又落下了两千年的尘土，却
要让我们一日看完。只好取一瓢饮，到一地就悄悄脱离人群，独自凭
栏。古斗兽场与秦始皇陵相差三百年，罗马帝国在我国历史亦被称为
大秦。这两处让人震撼的印象何其相似！我逡巡在圆形斗兽场可供进
入的二层环廊上，那天阳光时有时无，气温阴凉侵骨，密密相叠的拱
门偶尔投下斜射的阴影，绵延四合，层层相加，正是西洋宗教画里上
帝昭示神光普照的景象。那些拱门后面，常常出现远处的古典建筑，
和矗立其上的巨大青铜雕像，高低远近，整个城市浑然一个露天博物
馆。
    美玲小姐曾来中国学习中国文化。一路上她说过几次，中国也是
历史悠久的国家，可是也像意大利一样，最好的文物是在巴黎卢浮宫，
纽约大都会。言辞切然，悄悄在为她的国家解释，这一定是她心痛之
处。在罗马，每一个广场中央，都耸立着一根从埃及运回来的方尖碑。
上面象形文字宛然，是罗马帝国的征服者所喜爱的东西。它们与意大
利的广场的风格仿佛已经融为一体，人类对艺术美好的感觉竟是如此
相通。这些默默无言的方尖碑在这座露天博物馆里显示了历史潮流的
无情，此一时彼一时，强者自会在历史上留下痕迹。中国的建筑一直
是土木结构，自然不会给我们留下多少原迹。盛唐时期，文化、产品、
人种纷纷涌向中原大国的融和的轨迹，却可以丝缕不乱。


                         写生与购物

    冬天并不是欧洲旅游的旺季，走在大街上特征明显得可以让人一
眼认出的就是美国人和日本人。
    意大利的汽车几乎都小若甲壳虫，密密麻麻地挤在两三层高的欧
洲式小楼前。有几次，发现一辆车小到就是一辆摩托车外面包一层铁
皮外壳，我禁不住上前研究一番。一次旁边来了个白种人，也是一副
看稀奇的样子，不禁莞尔，这一定是个老美！
    在圆形斗兽场散步那天，回廊上散布着一些写生的中学生，衣着
打扮与讲究的欧洲人迥然不同，精致地嬉皮着，有几个人心思显然不
在画画上，聚在一处说话，声音倒是压得很低，没有追打笑闹，讲着
重重的卷舌音的英语。那些独自一人认真画画的孩子，坐在冰凉的碎
石地上，一笔一笔勾着那些拱门。有人凑在一边看，他们很沉着老练
的样子，丝毫不为所动，虽然那些画笔触很幼稚，并不像专业学校的
学生。
    虽然被欧洲人看作没有历史和文化。现在的美国毕竟是世界强国，
孩子们来到这个曾是他们起源的地方，安静地去了解各人能够领会的
东西，着实让人喜爱。
    另外一群给人深刻印象的日本中学生，则是在罗马西班牙广场上
的名品街上看见的。 GUCCI店外，有门警守门，一溜烟儿排出一条二
三十人的队伍来，在小街上蜿蜒。从关闭的玻璃门看进去，只有几个
人在挑选衣物。世界级名牌自然要维护名牌店的形象，认准这牌子的
人只得排队等待，店里店外几乎是清一色的日本小孩，实在巍为壮观。


                          惊鸿片羽

    罗马，比萨，佛罗伦萨和威尼斯，这一串名字听来已荡气回肠，
哪一地不该呆上半年，在古屋窄弄里细细流连，喝咖啡，吃冰淇淋，
喂鸽子！我却如旋风般一日半日就点水而飞。
    意大利虽有帝国的历史，却与希腊文化密不可分，曾有一段希腊
化的希腊文化时期。建筑、雕塑都有希腊遗风。在城市里遍布大大小
小的广场上，导游常指点广场边上某座楼上某层临场的某扇落地长窗，
一推开可以站在小圆弧形的阳台上，就就是某某教皇、某某城市统治
者讲话的地方。广场上犹是小黑石铺地，一条条小弄夹在矮楼之间，
来集会的人群自四面八方而来，又可各自散去。那几日正是意大利狂
欢节期间，装扮古怪的人，脸上浮现放浪形骸的神气，自由自在地信
步而行。有一晚是意大利铁路工人总罢工，我在一家小商店挑皮衣，
隔着一条弄堂听见广场上有喧嚷之声。意大利语节奏感极强，真是适
合发表演说，仿佛在唱一首歌。
    佛罗伦萨是文艺复兴的发源地，威尼斯则为兴盛之地。佛罗伦萨
在意大利语里听来恰是“翡冷翠”，在雁洛河畔像翡翠一样发着艳光。
老桥上金店密布，人群熙来攘往，正好容人细想但丁和比亚翠丝的相
遇，人海茫茫，擦肩错过，便成永恒。
    时光在翡冷翠仿佛留步，清冷的空气笼罩着灰紫的城市，是我想
留下来永远居住的地方。
    许多人盛赞威尼斯的美丽浪漫，在我眼里，它却具有一股英雄气
概，生机勃勃。这个城市建在先民一根根钉进海水的木桩上，他们逃
出了蛮族入侵的追缴，开始建城，而后从商，成立了强大的共和国。
城里到处是富庶华丽的装饰，圣马可方场三面长廊，全是商店餐馆，
得自威尼斯商人的真传。
    我们穿过狭窄只容人比肩而过的水巷，拍摄浮着倒影的石桥。刚
多拉从水面滑过，穿黑衣，草帽上扎着红绸带的船夫一路用美声唱法
唱着“桑塔露西亚”而过。这真是一个旅游的好地方，人人都这么快
乐。
    夜幕降临，我们经过自由大桥离开威尼斯内岛，直接向奥地利驶
去。
