红颜知己

○江西　若蓝


　　　　　　　　　　　　　
　　或许你的身边也有这样一个女子 她很想好好爱你正在默默爱你……

　　门开了，屋子里拉着窗帘，显得有些暗。可我还是看清楚了你蓬
乱的头发，满脸的胡渣。
　　“进来吧。”你抬了抬眼皮，声音沙哑，眼睛里布满了血丝。
　　这里真是乱的可以。迎面扑过来呛人的烟味。满地的烟头、报纸、
还有你的脏袜子。对于这样的场景我也是见怪不怪了。
　　我想你一定是又和她吵架了。
　　你拿着烟灰缸拧灭了手里的烟，还好，不至于太傻，还记得我讨
厌这种味道。
　　你斜靠在沙发上，脸上写满了颓废与疲惫。虽然不是第一次看你
消沉的样子，不过这次似乎更为严重。我在心里叹了口气，这天底下
唯有感情才可能把你折腾成这样。
　　走过去，拉开窗帘。阳光硬生生的闪了进来，刺在你身上，照的
你眯起了眼睛。推开窗，秋风灌进屋子。有点凉，也让人清醒。
　　我开始收拾这一屋的狼籍。等我拖好地，装完垃圾，再把脏袜子
洗好挂在阳台。你就一直坐在沙发上没动过窝。我在你身边坐了下来，
递给你一杯刚冲好的牛奶，你接过去狠狠的喝了一口（那姿势让我联
想到你喝酒的样子），险些被呛坏。总是这么不小心，你叫我说你什
么好呢。我递过温热的湿毛巾，拍着你的背，禁不住又在心里念叨。
看着你擦干净脸，做了个深呼吸，我知道你预备告诉我实情了。
　　“我和她分手了。”虽然不是第一次听你说这样的话，我还是有
些吃惊（她是你交往时间最长的女友）。我现在还清楚的记得七个月
前你神采飞扬的描述她的好，说她如何如何。太多的言语我也记不得
了，我唯一清楚的一句，也是你从来没有用在别的女人身上的那句：
“我知道，她就是我这辈子要找的人。”
　　我本以为……
　　你还在零零碎碎的说些什么，我听的不是很完全。大概是讲你们
如何认识，如何相恋再如何走到这一步。“我现在才发现，其实我错
了，我们都错了。我们一点都不了解对方，她更没办法理解我。”你
说。
　　我轻轻叹了口气，这次不是在心里，而且我相信你已经听到。
　　“你瞧瞧，连你都对我失望了不是，我看我是没得救了。哎！”
在别人眼里你是风趣的，睿智的，坚强的，甚至是洒脱的。唯有在我
的面前，你才会毫无顾虑的暴露你的软弱面。或许就因为我是你的红
颜知己，就因为在你的眼里我和其他的女人不同吧。到现在，我都不
知道，是该为此感的荣幸还是悲哀。
　　“振作点，我认识的你，可不是这样的。什么事都会过去。你要
找的人，要做的事，总有一天都会达到。相信我，你一定可以！”你
看着我，脸上开始有阳光的痕迹。“还好，还好，还没有到世界末日。
至少你还在安慰我，真是让我感动的痛苦流涕。”能开玩笑，就说明
你已经看开了。
　　“还是那句话说的好呀，女人如衣服，朋友似手足。”“我不是
女人吗？”“这个嘛……不一样的，呵呵。”我默然。
　　看着你刮干净胡须的明朗脸庞和你换上干净衣服的挺拔身影。我
笑了，我知道我笑的很柔，眼神里应该有溺爱的成分。我也不知道是
什么时候养成了这样的习惯，喜欢站在你的身边看着你微笑。
　　我说我们出去吃点什么吧。你说你饿了，大嚷着胃不能等。我想
起厨房里还有些面条，下了一碗，打上个荷包蛋，再洒上层青青翠翠
的小葱。你吃的很香，大赞我的厨艺有进步。我只是清楚，在没有米
饭的时候你才会选择面条。就象现在，你只能选择朋友来愈合你的伤
口。
　　你是真的累了，不一会儿就听到你均匀的呼吸。我轻轻的替你盖
上被子，象一个母亲对待孩子一样的疼爱，我还想吻一吻你的额头，
然后微笑着关门离开。那样的念头只是在脑海里一闪而过。想一想，
觉得不妥。只是带上门悄悄走开。
　　外面的夕阳很美，一大片火红色的云彩在天边燃烧。我漫无目的
的走。不想回家，也不知道该去那里。路过一家音像店，从里面飘出
一个极动听的清澈女声，我驻足。

　　“好想好好爱你 这一句话只能藏成秘密 关上窗外的雨 反复
触碰你爱过的痕迹 好想好好爱你 却没有权利再把你抱紧……”

　　歌词填的很好，我突然想起了你。其实我很清楚自己是爱你的，
我并不逃避这个事实。只是爱着的，并非都是合适的。换个角度，或
许可以成全我爱你的心。
　　我微笑着看夕阳看人流，明天又是新的一天。对于你，对于其他
很多人，都意味着一个新的开始。只是我还做着你的“红颜知己”。
还在你身边陪着你、看着你、默默爱着你。
　　只是你不会知道。
