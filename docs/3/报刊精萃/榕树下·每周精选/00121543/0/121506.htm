蝶恋花新说

○上海　右眼



　　一、玉蝴蝶

春天的掌心 交错 
她的气息纤织成网 
将你捕捉 

需一根刺 
若舟
横贯心湖 
你赤足跳上祭台 
只为否决 
关于殉情的疑惑 

从此
爱情只在春天
回来
并且出售


　　二、花心动

你并不是
我等待的那只
飞越沧海的蝴蝶
可你是
让我动心的
唯一那只

我的花纹
看见
你的翅膀
看见
我坐在花芯里面

一翕 万念起
千江有水
一动 万念灭
天上人间


　　三、蝶恋花

为一个瞬间
在茧里在土里
埋了整年

关于爱情
说是
飞鸟与鱼
曾经深爱过
