似水流年·女人三十

○姚娜(湖南)
　　　　　　 

    她的心是自由的，不属于任何人，包括她自己。

    她的心是什么颜色？她自己也不明白，但她知道，那颜色绝不是
赤、橙、黄、绿、青、蓝、紫中的任何一种。

    每一天，她必须早早地起来，风风火火地买菜，送孩子，然后独
坐窗前，用朦胧的眼眺望逐渐清晰的远方，她已经三十岁了，三十岁
的女人，还会有远方吗？

    远方，有人在树下唱歌，有人在海边吟诗，有人穿越荒野寻找太
阳……

    唉！三十岁的女人，再也不能称之为女孩，再也不能光着脚丫在
雨中奔跑，激起浪花串串，留下欢笑串串。三十岁的女人，在也不能
在受了委屈的时候惊天动地的哭，哭走烦忧，哭走恼怒。

    她，有了丈夫，有了小孩，有了公婆，有了一切琐屑的事物，脸
上已渐失少女的光泽，但是在每一个有月亮的晚上，她却仍会不由自
主地想起妈妈——那永远温暖的港湾。难怪每一次妈妈走的时候，她
都要送了一程又一程。而且仍旧像小时候那样牵着妈妈的手。船走了，
妈妈走了，她的心空空落落，也似那江水无语东流。

    三十岁的女人，意味着成熟，成熟是什么？是学会把清纯掩藏？
是学会把浪漫情怀抛洒？还是学会不动声色？她很困惑。

    家里来了丈夫的女性朋友，她热情款待，丈夫却说她不知嫉妒，
一定是爱得不够深。寒假暑假，学生们蜂拥而至，她与他们天南海北
任意遨游，丈夫又说她心太野，关心家庭不够。她差点在丈夫的言语
里迷失了方向。难道三十岁了，就该是一只鸟，天天呆在笼里，梳理
自己的羽毛？

    今年桃花开放的时候，她满了三十岁，为这，她伤心了好久。从
前，她喜欢站在高楼，望断天涯路；她喜欢独立江头，咀嚼黄昏的孤
独；她喜欢在琵琶弦上说相思；喜欢在深深庭院荡清秋；她会为一朵
白云的离去而悄悄落泪。她会为乡村的夜晚一声遥远的呼唤而遐思…
…这一切会因为三十岁的来临而成为往昔吗？她深深惋叹，那叹息，
丢在风里，凝成一株美丽的海棠，可卷帘人是否依旧？

    而今，她喜欢隔窗而望，把每一个来来去去的人看得清清楚楚，
明明白白，然后开始写诗，写散文，写小说，写得天昏地暗，写得山
清水秀。然后是日子像水一样清清澈澈地流过，她在这似水流年里看
到了自己的倒影：憔悴而略带忧伤。

    三十岁了。她忽然喜欢一个人呆在家里，那是很难得的时光。她
喜欢这样的孤独，一个人独自站在空寂的长廊上，看千万片树叶是如
何在阳光下泛着金光，听远处传来的隐隐约约的笛声。有时候，她还
听到了春天也有落叶的叹息。

    这时候，她觉得自己的心是一片广阔的原野，是一片汪洋的大海。
谁也看不清里面的花开花落，谁也弄不懂里面的潮气潮伏。那些像梦
一般的记忆啊！又轻轻的、轻轻的走过。

    和朋友们一起远足，爬上了那高高的山峰，即使雨连绵的下滂沱
的下也无怨无悔，那是她的少女时代。每一次演讲台上的潇洒自如，
每一次作文烂前的啧啧称叹……哦，那个快乐、热情而又忧郁的女孩！
每次每次都会有小鸟在她窗前唱着动听的只有她能懂的歌：谁是她生
命中的奇迹？谁又是她春天的回忆？

    那个在湖诗社写着点点心愁的女孩呢？那个在悠悠曲径寻觅美丽
涟漪的女孩呢？

    梦也会像长青藤一样蔓延到三十岁。三十岁的焦虑，三十岁的无
可奈何使梦染上了不知名的色彩，夜夜在月下徘徊。在这徘徊里，她
看见了缘分，缘分的天空那样湛蓝，缘分的世界里响起了一串风铃声。
她看见了两朵云，默契的走过一程后，一阵风来，便再也不能相伴。

    风铃声渐远，云儿渐淡，她看到了混乱的街道，拥挤的人群，听
到了“和了，和了”的声音，她看到了学生们忙着计算无理数、排列
组合。她听到了学生们忙着背诵advice，exam……他们的眼里不再有
激情，不再有清纯，不再有相聚与别离。

    许久不见的朋友再次相逢，几许欢喜，几许伤感，然而却再也无
法回到从前。时光的海就这样无情地横亘在面前。有谁，能够游过流
年的海？

    三十岁后，她有些明白，又有些不明白。

    “野渡无人舟自横”，她的心也许便似那舟。

    99年10月17日
