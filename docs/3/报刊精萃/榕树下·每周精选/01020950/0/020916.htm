云游黄昏

○商略(浙江)
　　　　　　 

初春的南方  
多雨。极阴冷  
窗下是晦暗的水流  
我听见。草籽的细细喘息  
还有云游僧人看孤雁坠地时的  
一声叹息  
 
暮钟响起  
昏鸦在槐枝上翘首  
梳双髻的宫女  
从宋朝的庭院  
步步金莲  
提一盏。荷叶花灯  
隐入一片青苔的故事  
 
只记得。双髻似螺  
寂寞时。听见什么在燃烧  
无晦无明。不生不灭  
还有恒河沙数的淡淡忧伤  
枯死的般若。游离的木鱼  
我喝着一碗酒。却是越来越清醒  
 
听见啊  
窗下有人吟唱。湿淋淋地  
寻寻觅觅。凄凄清清  
黄昏。顿时只剩下一付空壳
