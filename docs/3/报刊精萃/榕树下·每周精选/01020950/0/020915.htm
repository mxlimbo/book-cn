水泥厂

○江南篱笆(浙江)
　　　　　　 

水泥厂和火葬场  
在同一个方向  
它们常常同时冒烟  
不同的是  
一个是白的  
一个是黑的  
 
所以水泥厂的姑娘  
没人喜爱  
小城里流言说  
她们脸上  
有一层可疑的 油  
 
我那时读三年级  
我的邻居宣杰  
正和水泥厂的黄英谈恋爱  
她常来  
穿着华达呢  
用不标准的普通话  
叫宣杰的名字  
 
宣杰死于意外  
他在铁路边写生  
注意力太集中  
被火车撞了  
黄英沉默了几天  
又在另一幢楼下  
叫起了别人的名字  
 
我还小  
可我知道  
黄英不是无情的人  
宣杰死的很惨  
火葬场的工人  
失去了给他化妆的兴趣  
这时候 是黄英开的后门
