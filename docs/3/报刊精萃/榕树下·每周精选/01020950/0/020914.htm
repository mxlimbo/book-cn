○右眼(上海)
　　　　　　 

巨大的诗篇里面，我如此微不足道  
从一辆出租车里下来，触到坚硬的道路，然后  
被时间赶上一架飞机  
甚至来不及观察两旁的景物，我在它们  
之中生活了十八年，却未能  
接触到任何一个细节  
是如何发生和发展，直到它们死去  
我还活着；或者我死去，它们还活着  
 
我在一群人中间，表情和脚步划一  
不比他们痛苦，他们也不比我幸福  
我和几百个人同时来临世上  
不同时吃饭，不同时入睡，却有可能同时相爱  
不疾，不徐  
成群结队的我们漫不经心的生活  
享受，我们享受到共产主义的快感  
 
我爱的人他就在我身边，或者  
远在另一个城市，甚至他从没有出现过  
我置身于他的神秘阴影：出现或者消失  
这是某种臆想的狂喜  
猜想他从一个建筑物背后现身  
是一个阴雨连绵（不是晴空万里）  
他拍拍我的手（不是突如其来的接吻或拥抱）  
离开或者留下，我一定毫无怨言  
 
飞机火车驱赶了我们。我们  
像羊群从一个草场到下一个草场  
除了忽然中止  
我们无法感知自我的任何分裂和死亡  
我是生活的片段，生活是  
时间的某一个片段，而时间  
只能是不由分说，进入我身体内部的  
半个片段
