秀才和诗

○pozhenzi(四川)
　　　　　　 

    四个秀才去赶考。

    经过一座大山，山势崔巍，古木敝日。秀才们情有所动，于是各
吟一句，遂成一章。诗曰：远看大青山，近看青山大。

    我再仔细看，果然青且大。

    诗做完了，诗兴却未绝。恰好进山便遇上一座古庙，进得庙去，
发现门旁的一面壁上，留有许多过往骚人的遗墨。新墨旧墨交相混杂，
有的已漶漫若无，有的却森然惹眼。四个秀才看了看，想了想，也拿
出笔墨，找一处字迹消隐的地方，一人一句，将墨盖了上去。

    秀才们将诗留在了古庙的墙上，也就仿佛将诗留在了典籍里，心
中很惬意。于是趁兴，当夜便向考场赶去。

    考试据说很讲运气，四个秀才运气不好，等到发榜，名字全都排
在了孙三之后。秀才们只得有些丧气地走上了回乡路。

    又经过古庙。秀才们想起了不久前壁上题诗的辉煌，不免生出些
回味之意。于是进庙去看。

    诗还在，只是不知被谁用粗重的朱砂圈了起来。秀才们很奇怪，
好的坏的想了很多个为什么，还是不知道为什么。于是找来个和尚问。

    这一问，问得秀才们喜出望外。

    原来，就在秀才们离庙的第二天，老方丈发现了秀才们的诗。老
方丈参禅，他将庙门旁的这面墙辟为“随言壁”，就是希望它能成为
无言的禅悟交流地。这天早上，老方丈照例来到壁前看看，这样就看
见了秀才们的诗。

    老方丈看了，一哂，心中说：“此等秽物，也敢唐突上墙！”但
还未笑完，笑容却突然僵住了：这不是禅么？万物能归本真，去伪饰，
见山则山，见水则水，则禅理自出，禅机自显，禅意自归也。此诗质
真而形拙，是为又一禅意也！这一想，老方丈便亲自拿了朱砂，将诗
圈起来。眼下，正遍寻作者哩。

    秀才们虽不十分明了禅是什么玩艺儿，但诗作能得方丈赏识，心
中大有良马巧遇伯乐之感，发榜以来沮丧的晦气一扫而光。于是激动
地去拜会老方丈。

    老方丈的禅房果然是一方清凉天地。一尘不染，似有松风涌动。
知道眼前这四位就是壁上的禅悟者后，老方丈一下想起了禅史公案上
记载的那一幕幕精妙绝伦回肠荡气的禅论场面。多年来，老方丈深居
简出，淡言寡语，等的就是一次次这样的禅者之晤。

    “请问施主何处而来？”老方丈揭开了序幕。

    “哦，这个，我们从省城而来。”一个秀才说。

    “对，我们刚从省城……而归。”另一个秀才说。因一下想起了
落选之事，于是省掉了“赶考”二字。

    这回答似乎太普通了，并无禅意。不过，老方丈又一想，也许他
们是深入浅出罢。论禅却不求其玄，这倒更好了。于是又问：“施主
缘何而来？”

    “晚辈诗作得师父赏识，不胜欣喜，特来拜见。”一个秀才说。

    “对对，听说师父正在找我们，所以特来拜见，特来拜见。”其
余三个秀才说。

    怎么回事？老方丈觉得有些不大对劲儿。大巧若拙也不该离了质
朴之气吧？他抬眼定睛看了一下，看见了秀才们被兴奋、激动和诚惶
紧张弄得有些扭曲的脸。老方丈对自己当初的判断有了疑虑：他们是
否是习禅之人？他想他该试探一下。

    “施主平日常习禅否？”

    “喜欢喜欢。蝉在乡间是很多的，我们自小就和蝉打交道，夏天
常去树林里捉的。怎么？师父喜蝉？要不，我们给您捉几只来？”一
个秀才说。

    “蝉是个好物啊！读书人大都喜欢蝉的。连唐诗三百首里都有咏
蝉之作呢。”另一个秀才说。

    “但老衲所言非此蝉也。”老方丈打断了秀才的话。

    “哦，我知道师父所言何蚕了。此蚕乃古诗云春蚕到死丝方尽之
春蚕也。春蚕吐丝织茧为人衣，师父喜此蚕，实乃高洁心境之现啊！”
又一个秀才说。

    “呀，我家正事此事，师父既然喜欢，可即刻送一些幼蚕来。”
最后一个秀才说。

    老方丈彻底失望了，他决定结束这风马牛不相及的对话。他疲乏
地对秀才们说：“多谢施主好意了。老衲尚有禅课要做，施主请自便
吧。”

    秀才们并不知道老方丈在想什么，他们估计老方丈也许真是很忙
的，也就告辞了。谈话虽然短了点，但秀才们并不沮丧，他们心里溢
满了喜悦。至于赶考的事，那真的是运气啊。若是老方丈主考，他们
的名字会在孙三之后么？

    秀才们离开后，老方丈陷入了深深的惶惑中。喜悦已荡然无存了，
有的只是疑虑。老方丈感到自己几十年的修悟都是白费气力浪费时间。
秀才事件像一把利剑，一下就将几十年的修悟瓦解了，他感到自己倒
像是一位在迷津中迈步的盲人。

    老方丈急遽地消瘦了。两天后秀才们送来一簸幼蚕，更是雪上加
霜地将老方丈推向了绝望境地。看着自己的师父一天瘦比一天，老方
丈身旁的小和尚心急万分，不知所措。有一天，当老方丈又不思饮食
时，小和尚忍不住在门外向送饭的和尚低声抱怨了一句：“什么狗屎
禅，把人磨成这样。那些不知禅的人，难道就不活了？”

    这句话却被屋里的老方丈听到了。老方丈闻言一惊：对呀，殚精
竭虑，何悖于禅也！万物皆禅，人人都在禅中，不过有人知之，有人
不知罢了，岂可因人不知而虑禅之不存呢？

    老方丈全身一爽，忙叫小和尚端饭来，一气吃了个精光。

    老方丈打消了抹去“随言壁”上秀才们的诗的念头。随自然而然
之，老方丈决定让岁月风霜去消解它。他也悉心照料起秀才们送来的
幼蚕。

    这些无疑都帮了秀才们的大忙。因为秀才不识字的乡亲们到庙里
来时，都看见了红圈圈内的黑字，又都看见了庙里热闹的养蚕情景，
他们就知道了，秀才们果然没瞎吹牛。

    虽然也有一些识得字的，进庙来见了秀才们的诗和圈着诗的朱砂，
总要猜一猜为什么，并得出以朱砂圈之是要提醒后来者此等烂文字休
再上墙的结论，但这些都和秀才们无关了。秀才们在乡间过得很舒心。

    是啊，秀才们过得很舒心。
