亲爱的，你呀

○浙江　丁香女孩



　　曾经无数次在梦中设想着与你相遇，梦想着那个一生一世牵我手
的你一定要有着宽厚的肩膀，在我疲惫倦怠时，任我取暖，做我最坚
实的靠依。
　　梦中的你一定要有迷人的微笑，吹动我少女的悠悠心事，好让我
沐浴着你的笑颜送走夏的斜阳，揣一个秋的金黄走进冬的萧瑟里。
　　梦中的你有一双黑亮而温柔的眼睛，陪着我一起看朝霞满天，看
落日西沉，看雨后彩虹，看星光闪烁的夜空。看着我在下雪的天空飞
舞、飞舞，旋转出六角的花瓣化作护花的春泥。
　　梦中的你还要有一副动听的嗓音，会说好多好多的甜言蜜语来逗
我开心，凝视着我幸福的笑颜温柔地说：“你低眉浅笑时如云的秀发
悠悠滑落，直到我的心底叫我痴迷叫我欢喜。”
　　让我迷恋至深的你，可一定要有一颗善感诗意的心灵，会与我秉
烛窗下饮铭共读，携我戴月而行踏雪无痕；会欣赏我平淡微笑中滑落
的每一丝不经意的忧郁；会把夜空中流星雨的绚烂轨迹与我们忠贞浪
漫的爱情相系；会把每一滴敲窗的淅沥寒雨演奏成美妙而古典的音律；
会为我痴狂为我陶醉，为我凌波守望，为我至死不渝。
　　可是亲爱的你呀，你与我梦中的他有着太大的差距。你只会在小
憩的短暂空隙里，带给我出其不意的电波问候，略带羞色地说一句：
“没什么特别的事，只想听听你呼吸的声音。”让我在旁人羡煞的目
光中脸庞飞满幸福的红晕。你这个傻傻的男生，根本不会说任何的甜
言蜜语，只是在我一次次地拒绝之后，固执而深情地重复那一句：
“即使你不爱我，我也要一生一世保护你！”不善言辞的你呀，从没
有过什么地老天荒的承诺，但我能读懂你每一个的眼神里都写满了细
腻的关怀都用尽真心真情。
　　哦，笨拙的你呀，你只会随着我的快乐而欢喜随我的悲伤而哭泣；
在秋雨初至，瑟瑟寒风里，你温暖宽厚的双手会将我的苍白冰凉覆盖，
一再嘱咐我要添衣加被，好好将息；从不牵绊我为爱而孑然远行的你，
只会心疼地望着我，一再叮咛：“好好照顾自己，外面的世界纵然精
彩，若有一日你看倦看厌时也只需轻轻的将它关闭，记得我在这里等
你！”
　　哦，我一再说服自己根本不可能爱上你，可是为何每次见到你，
心中都会涌起丝丝柔柔的甜蜜。夜深人静之时，眼前浮现的都是你温
暖的身影。活泼好动的我，竟然会甘愿忍受漫漫寒夜的煎熬，用尽无
尽的耐心，重复着枯燥与单一，把对你的悠悠倾诉织进细细密密的针
线里。为了你，居然想试着改变自己，决心不再娇蛮任性不再一意孤
行。因为你的每一个淡淡的微笑每一片平常的话语都让我回味，倍感
温馨。
　　哦，我想我已经不可自拔爱上了你。偶尔想起过去的点点滴滴如
春风化做雨，也会有感动的泪水湿润我眼底。其实真正经得起风风雨
雨的爱情，又何须千言万语呢？反反复复，原来最好的一直在身边。
当我们相识、相知、相恋的爱的历程化为用心烹调的文字，剩下的便
只有沉淀后的醇香，那是美好的记忆呀！
　　哦，亲爱的，我终于不再逃避，就让我握住一缕在手心，不一定
最美丽，不一定最温馨，却是最最情深，最最心悸。那么，亲爱的你，
请为我披上圣洁的嫁衣，将长发高高盘起，从此我所有的温婉深情，
含蓄美丽全部只为你。让我们一起去迎接生活中的风风雨雨，相知相
守，携手走过漫漫的人生之旅。
