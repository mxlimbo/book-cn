我在今夜踏雨前来
只为欣赏你

（要说我也看不懂 谁又能明白）

灯火忽然辉煌 幕落之时来临
肃穆和我也会起立
和他们一道
热泪盈眶的鼓掌
（你在鞠躬谢幕）
