怀念

○重庆　剪不断

　
怀念那人的帆影时
我悄然立在窗前
怀念那些细碎的岁月
初春的风自南方来
如湖底柔长的水草了
游过额际
又乃是一尾蓝色的鱼

我转过身去
风景依然是
一种苦味的问候
更行更远
袅袅的探望烟霭般
已自生了
还将自灭吗


五月的回忆
黄昏里好象有什么
寂灭于微颤的草尖了
轻轻地悄悄地
如远去的那人
之背影
负走了一切
潮涨潮落的时辰

不肯晾晒思念
长路顺风的方向
没入荒原
匆匆吗
脚印，或深或浅
迟早都得凋谢
如你曾拒绝过的那些花瓣
