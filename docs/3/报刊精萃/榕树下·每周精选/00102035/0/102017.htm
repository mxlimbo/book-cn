午夜旅程

○他他蓝



　　在这个城市里我只带走了两条鱼。
　　很奇怪它们会在那里出现。繁华如锦灯如白昼的大商场中央，小
小的吧台一样的专柜。它们在密封着的彩色的玻璃瓶里游来游去。灯
光下小小的身躯颜色美丽。看上去寂寞。
　　小姐说瓶底的紫色颗粒可以提供氧气和养分，因此不用喂食。
　　我转头看他，他正饶游兴趣地看着这个彩色瓶子，一脸孩子气。
　　我轻轻地笑了，我买给你。

　　瓶子易碎，而我们要乘午夜的火车回家。他提了所有的包，我只
是抱着鱼缸。
　　卧铺车厢是空荡阴暗的。站台上有稀疏匆忙的人群，在白色灯光
下奔跑。铃声之后，火车开始摇晃着驶向寂静黑暗的铁轨深处。
　　短暂匆忙的旅程令人疲惫。我安静地坐在下铺，看着他把所有的
行李放在架上。昏暗灯下他的脸暧昧模糊，难以掩饰的困顿。
　　然后坐在我对面。开始抽烟。三五。英国烟的辛辣气味，在车厢
里绵绵地蔓延。
　　是我熟悉的气息。

　　睡吧。他开始往我的上铺爬。
　　睡在我的对面吧。我犹豫着问他。
　　他好奇地看着我。
　　这样我可以看见你，心里不会害怕。
　　我们在黑暗中沉默了一个瞬间。听见自己的呼吸缓慢。
　　他仍然爬了上去。
　　一直就不是愿意迁就我的孩子。我在昏暗光线里睁着眼。有点冷。
鱼在身边的小桌子上摇晃，不知道它们可不可以禁得起碰撞。

　　那个时候都的倔强又任性的。为他的短发，纯净眼神动容。在小
小的城竭力让自己的心不感到麻木，就算是疼痛。他让我心里温暖的
液体可以流动，这样地坚持很多年。
　　骄傲和猜疑像深壑横在中间。都是固执的人。各自与别人恋爱，
一再的分离提醒一再的爱情。直到一切成了习惯，在激烈背后平淡和
厌倦渐渐蔓延。
　　而我是容易沉溺的人。

　　在不知名的小站醒来了。窗外有稀疏的人声，在夜色中模糊的栅
栏和站牌。还是安静的。
　　我伏下身子仔细看鱼，它们还是很活泼。似乎不睡觉。
　　从车窗里看见自己花瓣一样寂静颓败的容颜，空洞得没有内容。
　　他还睡得很沉。微皱着眉头孩子一样无辜的表情。而醒着的他是
任性冷硬的。心里柔软的部分在疼痛。我的手指冰凉地掠过他的脸颊，
没有触感。沉默地观望，隔着不远的距离。
　　车在继续摇晃着潜行。终点在黑暗深处。

　　有一个短暂的梦。
　　我在彩色的美丽瓶子里游动，可以呼吸。光线迷离。可是听不见，
也不能发出声音。
　　在流光异彩的倒影中我看见自己的身躯，有梦幻般颜色，水滑过
去，像冰凉温柔的手指，或者亲吻。
　　我惊恐地用柔弱身躯撞击坚厚的玻璃。一下，又一下。感觉不到
疼痛，只是无尽的恐惧升上来，升上来。

　　女孩似笑非笑地看着我。眼神淡漠。
　　八年？你们怎么不恋爱？
　　他洁白柔软的手指轻轻蜷在他的手心里，浓密的长发垂下来，娇
慵地遮住一半美丽的脸庞。
　　恋爱？他的眼神水一般流动。然后笑。
　　如果相爱，哪里可以在一起这么久？
　　我微笑着。心里又温柔惆怅的东西涌动，到每一个缝隙里。
　　咖啡屋的窗户开着。一小片蓝的天空。阳光刺痛我的眼睛。所有
支离破碎心力交瘁的深情。
　　他有明亮幸福的笑容，为朋友，和爱的女孩。
　　八年这么久，所以不爱。
　　心里翻涌着潮水一样无声而绝望的快乐。

　　 虽然只是他的一段短暂恋情。

　　他的手指轻轻触摸我的头发。我醒来了。
　　火车还在黑暗中前进。不知怎么时候他下来坐在我身边。
　　快到了。起来吧。
　　车厢里竟然有很多人，都沉沉睡着，有起伏的鼾声。我的神智不
清醒，头发凌乱地呆坐在床边。不知道这个瞬间是什么时刻。
　　他把东西收拾好，揉揉我的短发，要我去洗脸。
　　车厢交接处灯光明亮刺目。我沿着窄窄走道摇晃着走过去。

　　我用冰凉的水淋湿了脸。镜中的女孩在强烈灯光下有苍白，明亮，
无可逃循的眼神。我抚摸自己的嘴唇，像灼热燃烧的蝴蝶栖息，没有
过亲吻，所以干涸。突然觉得累了。不能确定这劳顿旅程的目的。是
对他无可拒绝的逃避，还是对自己无法期求的沉溺呢？
　　注视自己。眼泪突然汹涌而出。我抬起手背遮住眼睛，不让自己
看见。
　　那一刻心里寂寞到丧失了语言。
　　不知道一个有多少平静自由的时光，可以给另一个人。

　　回去的时候，他在黑暗中用锐利的眼神看着我。鱼在他身边轻快
地游。空气中有潮湿阴凉的气息，就像他给我的土壤，使我如热带荼
靡腐败的植物，顽强又迅速地在他的手心里生长。如果割开，便是枯
萎。如果继续，便是溃烂。
　　车已经缓缓进站了。

　　在熟悉的站台下了车。我们在灯下有一些发愣。天空开始飘着纷
飞的小雨，寒冷。小站人很少，一个面无表情的男人在出站口勉强睁
着眼睛。天色灰蒙蒙，已是清晨了。
　　我披着他的外套，抱着和我们一样疲惫的鱼。出租车在熟悉的潮
湿街道上飞快滑行。黎明初醒的小城，虽寒冷却有温暖的感觉。
　　我轻轻把脸贴在他的肩上。

　　车在我家楼下停了。
　　我已经没有了告别的力气。把鱼放在他的手上，然后我说我要回
去睡了。
　　转过身便听见了玻璃在地上碎裂的声音。
　　在清晨模糊的光线里他的神色晦涩惶惑。彩色的碎片在他脚下闪
烁着美丽的光彩。鱼太小，消失在下过雨的地上。
　　我们沉默着。心里疼痛的裂口使我没有办法呼吸。
　　至少它们不用再以柔软身体撞击玻璃，至少它们可以自由呼吸天
然的氧气。
　　就算死亡，不用再沉溺。

　　午夜旅程之后，我决定要离开我的城市。
