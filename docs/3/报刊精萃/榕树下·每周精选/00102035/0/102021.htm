隐约隐情

○安徽　江淮猛男



　　　　　　　　　　　　儿子的问题

　　儿子尚小，倒很爱读课外书。读书当然不是坏事，但让人费神的
地方还是不少。眼下，书的概念谁都清楚，“儿童不宜”的东西，实
在让人防不胜防。每当儿子吵闹要书，我除了想到“银子”，便是这
难言之隐，生怕一不留心闹出什么乱子。
　　小心归小心，但失算的时候还是有的。那次，我满有把握地为他
买了本儿童杂志，不料儿子指着一则“幽默”，直叫一点都不好笑。
我想，出版商为了把书搞厚卖个好价，夹点“水货”也在情理之中。
顺眼朝儿子指的地方一瞧，倒着实吓了一跳。没想到儿童读物中的
“幽默”，居然闪着“黄色光芒”。好在让成人脸红的“幽默”，儿
童不懂。后来，再入书市，只得在童话中挑来捡去。安徒生、格林，
他们总不该害孩子罢？
　　此举看来不差，《格林童话》儿子看了几天，始终反映正常。但
这天，儿子冷不丁冒出一个问题，竟又让我一时语塞。
　　儿子问的是《三个懒汉》――这个老国王，临死前竟决定传位最
懒的儿子。最懒的人为什么可以当国王？儿子不明白，我亦无当国王
的实践，无现存的心得体会拿出来就是。但我猜想，既为一国之王，
身负一国重责，没点雄才伟略，不日夜操劳国家大事，当是亡国无疑。
老国王竟选最懒的儿子来继承自己的王位，高招的妙处，还真让人难
得其解。
　　但转念一想，若干年前“顶替”风大盛时，邻居王老将退休让贤，
也曾面临此类问题。而王老召开的家庭民主生活会，决议的结果，也
是让勤劳的大儿子自谋职业，让懒得“油瓶泼了都不扶”（其母语）
的小儿子，“顶替”上班吃皇粮。实践证明，王老的决策英明之至，
其子在学校教了一个月的算术后，懒兴大发，校方只得让其改教“劳
动技术”。没到两年，便又懒成了校长，比起几位一道进校，一直十
分勤勉，至今仍为教书匠的同事，真是大有成就。至于其他终日悠哉
悠哉的人，多能在社会上玩出正果的例子，着实并不鲜见。想必王老
和这些人，一定读过《格林童话》，受过老国王的启发。只是这其中
的奥秘，无法与儿子明说。否则，定然毒害下一代无疑。
　　看着儿子正等待答案，我只好告诉他：“老国王，当然是老糊涂
啦！”


　　　　　　　　　　　　反正不吃亏

　　中央某校的“函授研究生班”招生，在某市的录取比例，声言是
1.3 ∶1.录取后“每期学费××××元，面授×次……”然后，“发
给毕业证书，享受国民教育同等待遇”。录取率虽然不低，但报考者
多是不敢含糊，捧着“复习材料”大复其习，唯恐落榜。
　　朋友也报了名，却一如既往地该玩嘛还玩嘛，就是不肯捧本书复
点习。问其故，答曰：“闹那么紧张干什么？人家是公费，学费报销，
关键就在‘考’；我是‘私费’，关键是‘掏’。考上了赚一张文凭，
考不上省一万大洋，反正不吃亏！”真是高人高见，事实上，有关
“不吃亏”的事，确实还多着呢！这回，某单位的领导莫名其妙地收
到一份通知，称其“光荣入选《××世界名人大词典》”。会计依领
导之言寄去入选费、购书费，尔后“大典”寄来，领导几分高兴，四
处张扬。尽管他至多是个“县级名人”，但有“书”为证，自己口袋
里亦未少一分钱，想必也是“不吃亏”。
　　常见买高档烟酒的人，开口问店主：“有假的吗？”假的便宜，
也不是自己消费，所以店主也是如实相告，无需担心被“王海”给告
了。买假货可以降低办事的交易成本，何乐而不为？
　　名山胜水，游人如织，一位老先生手持罗盘，冲着一班游人中的
“班长”一拱手：“恭喜！恭喜！这些人中唯您贵相，两月后您必另
就高位！若是不实，您来扯断老夫的胡子。”“班长”果真是位领导，
他笑着掏出一张拾元递给老者，老先生死活推辞不肯收：“您是贵人，
老夫岂敢收您的钱！”马上有同行者抽出一张百元大钞，说：“老兄
有喜，就是小弟有喜！”老先生双手接过，目送这行人笑语而去……
　　留心生活便不难发现：行骗者多谈不上高明，受骗者多算不得愚
蠢。除了行骗者得钱，受骗者得益的“不吃亏”原因外，这种“双赢”
现象的背后，还应该有点什么！


　　　　　　　　　　我们准备做谁的祖先

　　谁是我们的祖先是明确的，但我们是谁的祖先就并不怎么简单和
明确了。
　　南明的兵部尚书阮大铖，其祖先是唐代镇南大将军阮枞江，其曾
祖父是明代著名抗倭将领、福建巡抚阮鹗。只因阮大铖为人奸诈，在
南明朝廷党同伐异、横行无忌，最终又叛明降清，进了《明史·奸臣
列传》。阮氏家族世代驻住安徽枞阳，但《明史》却声言为安徽怀宁。
《明史》总裁张廷玉是阮大铖的同乡，因厌恶阮的恶名，昧着良心来
了回“以权谋私”，将污水泼到邻县怀宁身上。今天枞阳的阮大铖的
后人，无不否认阮大铖是他们的祖先。他们宁愿让《明史》作证，也
不愿让“家谱”露底。
　　无独有偶。秦桧是南宋宰相，因“擅权阴毒”，用“莫须有”罪
名，杀害抗金名将岳飞，为后人痛恨，进的是《宋史·奸臣传》。蔡
京在宋徽宗时，官至尚书右仆射，后为太师。因以奢侈迎帝意，搜刮
民财，贿赂公行，同样为后人所不耻，也进了《宋史·奸臣传》，还
进了小说《水浒全传》。秦、蔡二人显赫宋朝，一个善造冤案，一个
善于贪赃，丑恶的水平和档次，都空前未有，属于当之无愧的“国家
级”贪官污吏。
　　秦、蔡二人的“出色表现”，同样害苦了他们的后人。福建莆田
的蔡姓，其祖先分别来自君谟和东京汴梁。来自东京汴梁的蔡姓，便
是蔡京的后代。他们害怕蔡京的名声污秽自己，便不肯承认是其后裔
子孙，谎称自己是君谟人。现在江苏无锡的秦姓人，确系秦桧的后代。
因为有秦桧这个老祖宗，他们只好谎称自己是淮海秦氏的后裔。
　　今天，踏着阮大铖们脚印前进的贪官污吏，依旧不少。其贪污腐
败的“实绩”，早已让昨天的贪官污吏们望尘莫及。以史推断，他们
的子孙，再有胆量，也是万万不敢认他们作祖先的。
　　所以，那些实在忘记了党纪国法的人，到了欲壑洞开的时候，至
少应该想一想：我们准备做谁的祖先？
