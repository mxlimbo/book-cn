不该有的伤害

○cowry

　 
　　昨天，带五岁的儿子逛街买东西，在商场里，我慢慢地挑选衣服，
儿子不耐烦了，几次三翻的拉我走，吵得我心烦，竟抬起脚十分厌恶
的气愤的踢了他一下。这一下深深的伤害了儿子，也重重的伤害了我
自己。
　　儿子当时那惊异的痛苦的屈辱的难以置信的眼神像刀子一样插在
我的心口，他好久没有说话，但是他的眼睛说了很多：这是爱我的妈
妈吗？这是我爱的我依靠的妈妈吗？为什么？为什么妈妈对我这样！
　　儿子呜咽着，难过的说：妈妈你踢我，你不爱我，你不爱我。
　　儿子以前淘气调皮闯祸惹人生气的事情多着呢，可是我没打过他，
这一次是为什么呢，只有我自己知道，因为我最近心情一直不好。于
是毫无道理的突然失去理智把火发泄到了幼小的儿子身上。
　　心无法平静下来，我不能原谅自己。不知该怎样挽回这个错误，
想起自己小时候的经历，总觉得对儿子的伤害是无法挽回的了。童年
的不快乐一直影响着我的性格和生活直到今天。我觉得妈妈不疼我，
甚至怀疑我不是妈妈生的，于是我自卑胆小不自信。
　　总觉得自己的童年是不快乐的。也许仅仅因为几次来自亲生母亲
的伤害。家里兄弟姐妹多，妈妈一面上班一面还要照顾家庭孩子，所
以繁重的生活压力把她的心磨砺的粗糙而坚硬，她顾不上关心孩子们
的内心世界，让我吃饱穿暖就不错了。小时候最恨的令我至今都不能
释怀的就是妈妈当着同学的面打我，一把笤帚都打断了，看样子我要
是不承认是我的错就会把我打死，于是只好“承认”是自己错了，实
际上妈妈说的事我一点都不知道，可她硬说是我干的，一点也不遗余
力的打只有十岁的我，后来我才知道，妈妈是跟奶奶怄气，结果把我
当了出气筒。这使我在心里与妈妈疏远了。
　　还有一次，我已经读高中了，一个很要好的女同学到家里来玩，
她送我一张画，是山口百惠和三浦友和的亲密合影，我非常喜欢就把
它贴在了墙上。可是妈妈进来了，根本不管我同学在，就敞开嗓子开
始数落我，不用心做功课，整天看些没用的闲书等等，说到激愤处，
伸手把那张画撕下来，三下两下撕了个粉碎……
　　妈妈从没把我当一个有独立人格的人，只当我是她的孩子，可以
随便她怎样。而我的心灵受到了多大的伤害是她从没想过的，也是她
无法想象的。我觉得自己很小就有了痛苦，这使我的额头有了与年龄
不相称的很深的明显的皱纹。然而这种来自亲生母亲的伤害让我能对
谁说呢？那种无助，那种孤独，只有埋在心底，并且使我有了一个决
心，等我做了母亲，我要全心的爱我的孩子，决不能让他的心灵受到
一丝伤害。如今我长大了，也做了母亲，我能够理解母亲当初抚养我
们的艰难和辛苦，从理智上我孝敬母亲，可是那些阴影还在心里拂不
去，偶尔触到了，我还是心痛，伤心。我告诫自己，幼小的心灵需要
的只是爱，宽容，依靠和安全，千万不能再犯母亲犯过的错误。
　　我常常用眼睛看着告诉儿子，告诉他，妈妈爱你，妈妈喜欢你，
你是妈妈的骄傲。每天下班我都先拥抱儿子，让儿子感到妈妈的爱，
来自妈妈的安全感和幸福感。而这些都是我的母亲不曾给予我的。
　　人是多么容易犯错误啊，哪怕是你一直小心的防备着。
　　晚上儿子就忘了，照例的心无芥蒂的开心的笑，跟我耍小聪明，
调皮。可是妈妈的心里却怀着深深的歉疚。这歉疚不会轻易拂去。
　　儿子，原谅妈妈！那是不该有的伤害。
