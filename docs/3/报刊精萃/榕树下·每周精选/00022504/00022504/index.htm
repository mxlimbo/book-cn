【编辑的话】

　　从一个异乡到另一个异乡，从一颗心灵到另一颗心灵。
　　无论时尚抑或传统，美的概念始终统一。
　　远瞻或者怀旧，放逐或者回归，爱意和本能是来自灵魂的渴望。
　　每一条走过的路，都有它不得不这样跋涉的理由。
　　每一条将要走上的征途，自有它不得不那样选择的方向。
