有时想起死亡

○woodman
　　　　　　 
  
我有时  
会想起死亡  
我想走的时候  
不告诉爹娘  
也不告诉  
任何一个姑娘  
 
把我的身体  
白的归白的  
黑的归黑  
红的就交给雨水  
 
选择黑夜  
做我的坟场  
允许棺木里透进一缕月光  
我最后站在悬崖边  
看见遍地粉碎了的安慰  
看见  
一根成不了材的木头  
终于烧成了炭  
成了灰  
 
这一生的阳光  
和我一起跃入深渊  
伴随的两滴泪水  
一颗还给出生  
一颗寄给天堂  
 
我有时会想起死亡  
想着  
把白天都变成了晚上  
想着天空中最后的太阳  
被一箭射下  
我的头颅上再也长不出  
一茬一茬的麦花
