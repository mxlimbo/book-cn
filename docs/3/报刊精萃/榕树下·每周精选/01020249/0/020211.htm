思念如潮

○无心无爱
　　　　　　 
  
    对不起，我太想你，所以才借助烟和咖啡冲淡这种思念。
    你送的香水百合每天都会开一点，散发着阵阵清香；你买的香草
牛奶巧克力，我每天都会吃一块，虽然我真的不爱吃这么甜的东西；
你爱听的《天空》我为你买到了，没有开封，我想等你回来我们一起
听，虽然我更喜欢赵传的《我是一只小小鸟》。
    朋友说，如果你不确定是爱情还是友谊，就去算算每天你想他的
时间，如果是偶然想起，或者根本想不起，那就铁定是友谊了；而如
果你每时每刻都在想他，就别再骗自己了。
    而我，不想再骗自己了。
    我想你身上淡淡的香气，想你温暖的大手，想你的眼，想你的唇，
想我们曾一起渡过的每一分钟。
    我只能靠这些回忆维持我敏感的生命。
    这种思念让我心痛，更让恐惧。我突然很怕，怕有一天你会消失
不见，怕我没有机会告诉你我的思念，怕你和别人说同样的话……
    一向洒脱的我变得如此放不开，这真是一个奇迹！
    人们总说，爱会创造奇迹，我信矣。
    我想你。
    好几百遍。
    不得救赎。
    你说你一辈子只爱一个人，那样的生命才最为完美。你说我是你
的初恋恋人，也是这一生唯一的恋人。
    这算是承诺吗？如果是，我希望它实现。
    这会不会是谎言？如果是，我希望你一辈子都别去拆穿它。
    思念让我的感冒久治不愈，我终于明白“为伊消得人憔悴”是真
的。
    你呢？在家过得好吗？有没有像我想你这么想我？有没有多吃点
妈做的小菜？有没有去书吧看那些小资情调特浓的书？有没有……
    唉，真想知道你现在在做什么。我后悔了，我应该自私地把你留
在我身边。要不应该放下矜持和你一起走。反正我不想这么过日子。
    我想你。真的。
    记得每次呼你你都不让我留电话号码，只打530 ，以前觉得挺逗，
也从来不呼，让你“天天天等”，现在想呼你都呼不着了。
    唉……
    你买的相思梅我一直随身带着，想你就吃一颗，可我发现那么大
一袋竟被我在这么短的时间里就吃剩了几颗。我决定明天再去买，然
后把袋儿都留着，回来让你报销！哼！总不能让我“赔了思念又费钱”
吧？：）
    行了，送你一首我正在听的歌，也算是一种温柔的警告吧——
“我可以永远笑着扮演你的配角，在你的背后自己煎熬，如果你不想
要，想退出要趁早，我没有非要一起到老。我可以不问感觉，继续为
爱讨好，冷眼地看着你的骄傲，若有情太难，想别恋要趁早，就算迷
恋你的拥抱，忘了就好……”
    虽然我爱你，但你不要太得意，要把握住幸福，否则，我会更爱
我自己。
    亲爱的，快点回来吧，我《想你想得要疯了》！（呵呵，其实也
没那么严重。）
    思念的夜于思念人之屋
