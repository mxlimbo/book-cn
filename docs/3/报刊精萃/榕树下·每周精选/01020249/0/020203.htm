除夕

○叶想
　　　　　　 
黑夜中有人轻声咳嗽  
这引发一场普遍的虚弱  
我一手抚心  
天空微微摇晃  
先有光，再有声音  
然后弥散开火药的气息  
这是除夕，爆竹声声  
山腹里传来岁月的回响  
 
时间是一种危险的病菌  
潜伏在黑沉沉的夜里  
零时的钟声惊动了它  
于是下起一场雨  
四条河流被风驱赶  
从指间向心脏汹涌而来  
今夜是除夕  
 
听风听雨  
又一年除夕  
我们被安排在苍茫的时空里  
唯一置身事外的  
竟是这一场烟火
