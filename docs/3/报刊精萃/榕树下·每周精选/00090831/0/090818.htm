阿寅表哥

○湖北　让布朗

　　　 


　　98年冬天，我在S 城S 大，为了毕业，做着一些没有什么意思的
文章。记得我费了九牛二虎之力，搞出了毕业论文的开题报告，大约
是关于知识分子人文精神的，却被导师以偏激为由否定了。虽然做这
题目，只是因为时髦，我的目的也不过是想来个惊彩的学术表演，我
还是非常沮丧。
　　父亲在此时来了电话，说阿寅表哥杀了人。
　　父亲说，“这个孽子啊。”
　　于是想起来，已经长久没见过表哥。
　　拿起一本《ELLE》，靠到J 的肩上去，我觉得心情安定了一点。
J 依然只顾兴致勃勃地，在网上和某个陌生的女人调情。当然也可能
是一个陌生的男人，谁知道呢。
　　表哥的故事，就那么遥远那么遥远地，漫上了心头。

　　阿寅表哥是三姑的孩子，属虎的，长我十岁。三姑生了五个表姐，
快四十了才盼到他这个老儿子。小时候每年都要回清江老家，因此也
很熟悉。他笑我是老鼠尾巴，因为我是鼠年年末生的。他说，他是老
虎头，“是头上那个王字。”
　　他一向生得瘦，手指节骨分明，不笑的时候，脸上的线条就锋利
起来。

　　三姑的家在清江镇清江边上，是古老的那种木头吊脚楼。沈从文
和黄永玉笔下的湘西有许多这样的屋子，细脚伶仃，线条很有魅力。
不过南方很多地方有，只要依山临水，不一定只在湘西。细细的光柱
穿过窗棂中落下来，映着三姑简朴的但收拾得一尘不染的家，屋里散
漫着陈年木头的柔和的腐朽气息。
　　姑父是个石匠，不过已经老了，做不了太多。三姑夏天卖凉粉，
平常糊火柴盒，做烛芯，把灯芯草卷在竹签子上，记得从前是做一百
支两分钱―――不知道现在该是什么价钱了。三姑可以快得象电影
《摩登时代》里的流水线，整天整天的做，看着笸箕里慢慢地满起来
一座小山。那些烛芯做成的通红的蜡烛，专为着每年的清明和七月半，
家家户户上坟祭祖。求神礼佛，也用它。
　　他们都是极好的人。谦和多礼，极其勤俭，多吃一块豆腐也觉得
罪过。

　　三姑和姑父很爱表哥，也很为这儿子骄傲。没读过什么书的表姐
们亦是如此。表哥那时候总是轻轻松松拿第一，一手字漂亮得和字帖
上的一样。父亲就爱拿他来说我，“看你阿寅表哥如何如何你又如何
如何”之类，也不管我还只是个小孩子。
　　父亲也爱他，象爱自己的儿子。
　　他们都认定了他要上大学的，正激动而又平静的等待这一天，那
心情宛若第一次做母亲的少妇，等待自己孩子的降生——做梦的时候，
都会不知不觉的微笑起来。
　　所以后来他们很长时间都不能相信命运给他们开的玩笑，他们一
遍一遍的对自己说不会是真的明天会好的或许是一个恶梦罢了。父亲
一直想弄明白到底怎么会变成这样，但他住在遥远的省城里，三姑又
除了哭之外什么都弄不懂。
　　表哥只是突然不再读书，交了些新朋友，每天在街头胡混。姑父
揍他，他也不叫痛，也不逃，冷冷的一声不出，不管打得他多狠。父
亲听到这个消息之后断定他无可救药的。后来看古书，才知道父亲的
道理，真正的孝子，应该“小杖则受，大杖则走”，打在儿身上，疼
在娘心上，父母打得重了就跑开，才是真心为父母着想。象表哥这样
忍心的人，“不成龙即成蛇”，将来若不是非常之辈，就必是“巨奸
大贼”了。
　　不久表哥就出走了。他拿走了他所能找到的所有的钱，在外面游
荡了一个多月，然后在一个早上乞丐一样的回来。他什么也不说，径
自到床上睡了。三姑在厨下呜咽着，打开煤炉的火眼烧水，又去找干
净的衣衫。姑父在床前站了一会儿，恨恨地想揍他的样子，但终于没
有揍，默默地出去了，以后也再也没有打过他。
　　大学当然没有考上，表哥日日闲荡，开始以勇力闻名。怕他不知
会变成什么样子，于是托了人情，送他去当兵，这是最后的办法，也
是最后的出路。

　　表哥走后的有一个夏天，回清江时带了弟弟去。弟弟回来得少，
什么都新奇，到处跑来跑去。有一忽儿没见着他，三姑就惊慌了：
“哎呀呀，去哪里了呢去哪里了呢―――”
　　她一会儿想是不是迷路了，一会儿想是不是跌到后山凹里了，或
者遇上了毒蛇。最后她认为弟弟一定玩水去了。方言玩水就是游泳的
意思，在三姑的想象里那就绝对要溺死的。
　　我们本来无所谓，弟弟也是大孩子了，不过就在附近哪里玩吧，
哪里管得那么多。可是三姑说了又说，说了又说，设想了无数的可能
的危险。终于我们也惊慌了，后来跑出去找，沿着清江喊来喊去。
　　这时弟弟却回来了，很不高兴我们的大惊小怪。
　　真不知道表哥怎样练出他的好水性。清江每年涨春水，水比往常
要宽要急，乡里一些青年，喜欢在这时赌赛横渡清江。因为常常有因
此葬身鱼腹的冤魂，这种事为所有有理性的成年人所反对。然而年青
人每年每年的成长起来，他们变老之后又会有新的一代，年复一年把
新鲜华美的生命付与这无意义的冒险。
　　阿寅表哥据说就参加过这种比赛。
　　三姑的头发都全白了。
　　这使她成为我所见过的最美丽最慈祥的老妇人。她没有年青时的
照片，但三姑怎么可能比现在银发如雪的样子更美呢。

　　表哥在部队里居然干得不错，据说很得了领导们的喜爱。他本是
绝顶聪明的人。他当的是汽车兵，后来转业，进了省城一家效益很好
的国营大厂开车。正是“马达一响黄金万两”好时候，何况又是拿公
家的车带私货。那一次父亲真的非常生气，因为表哥说，他没考上大
学并不后悔，不然毕业出来，拿那么一点钱，还不够他抽烟。即使读
得好，“象舅舅这样罢”，又有什么意思。
　　父亲当时正因为评职称的事焦头烂额，伤心之余，竟说不出话来。
在高校呆过的人都知道，评职称是一场怎样残酷的战争，尤其是对于
被文革耽误了的老家伙们。
　　多年后当表哥下了岗，在胡混瞎混时，父亲说，表哥当年的一个
同学―――也是父亲的一个学生―――当初成绩还不如他呢，现在已
经提拔了副县长了。副县长曾请父亲吃过一次海鲜，圆圆的脸，穿一
件上好的休闲夹克。
　　父亲愤愤的说，“竖子不足与谋！”
　　事情总是又好起来了，仿佛又回到了正轨。三姑开始憧憬抱孙子
的快乐，这快乐却来得很那么迟，使期待变成了一场漫长的折磨。表
哥快三十了才结的婚。这以前他不停的换女朋友，一度还和一个比他
大十来岁，开了个小酒店的，身世可疑的女人住在一起――这消息令
三姑几乎昏厥过去。他蓄长发，穿喇叭裤，整夜整夜的打牌，搓麻，
下歌厅舞厅；超高超重的载货，几日几夜不合眼的跑长途，不顾性命
赚的钱，也不顾性命的流水一样的花完。
　　仿佛日历上没有明天，迟了就要来不及了。父亲总是说，如果那
时积下一点，做一点事业的资本，该有多么好啊。
　　寄给三姑姑父的家用也有一回无一回，有段时候就干脆断了。父
亲骂他，表哥淡淡的说，反正给他们，他们也不用。一个钱看得铜锣
大。

　　当大家又一次要绝望了，表哥才把嫂子带回来。那正是电视上刚
刚疯狂播过《渴望》的时候，证明中国人还在惦念着传统。嫂子并不
象刘慧芳，薄嘴唇，尖下巴，细细的眉有点上挑，但是她很有礼貌，
笑起来也很甜。她收住了这浪子的心，后来又生了个胖儿子，三姑和
姑父已经满意得不知该说什么好。
　　至少看上去，一切回到了正轨。

　　三姑94年去世了，父亲曾带我回老家奔丧。她走得很突然，在某
一个早上一跤跌下去，就没有再起来。当时她拿着那把有那么粗糙恶
俗的鸳鸯戏水图的茶壶，用了一辈子了，还一点缺痕也没有，仿佛还
可以再用一辈子，但在那一刻也就粉碎了。
　　姑父已经病倒在床上，丧事有族里人帮忙操持。年青的道士穿着
简陋难看的长衫，起劲地吹奏着尖厉的唢呐，长衫下面露着质量很次
的牛仔裤和旅游鞋。这里的年青人百分之九十九到南方打工去了，运
气好的人回来就在那些破旧的吊脚楼中间栽上几幢钢筋水泥的新楼，
把这个镇子变得不伦不类；另百分之一的或者考学或者当兵，总之都
逃也似的离开了。不知怎的却剩了一个硕果在这里。
　　邻家的松堂叔是族里所谓“红白喜事理事会”的理事，知道一切
年青人所不知道，老年人例如父亲也未必知道的各种规距。他絮絮叨
叨的和父亲说，“入棺的时候，不要太伤心，千万不要把眼泪落到棺
材里，母家亲丁的眼泪，落到棺材里，那是很毒很毒的咒，阿寅一家
就算完了。切记切记啊。”
　　父亲吓了一跳，几乎再不敢流泪了。

　　吊唁的人很多，表哥表姐们就跪来拜去，迎来送往，没个消停的
时候。入棺，盖棺，也照例是哭，拜，拜，哭。表姐们泪水婆娑。她
们都很早就嫁出去了，而且都具有一样的贤淑忍让的美德和过早衰老
的容颜，所以我一直也没法分清楚她们。
　　然而表哥不哭。
　　他只是沉静的做他该做的事。甚至不必要那么拘礼的事，他也做
得一丝不苟。按乡俗，“天上的雷公，地上的舅公”，舅舅在家族中
本来就很受尊崇，在丧事上做为母家的代表，更是不同寻常，必须格
外礼待，而且有苛求的权力。父亲当然决不会为难任何人，是回家奔
丧的表哥，顽固的坚持按照接待舅舅的最古老的最尊贵的礼仪来办―
――从街口三步一叩的迎进父亲―――对父亲的反对置若罔闻。
　　他沉静地拜，叩头，一遍又一遍，专心致志，规范而完美，如痴
如梦的沉醉着这肢体的舞蹈，当表姐们的哭声如哀歌萦绕不去。
　　就是我也要下泪了。
　　父亲很不安，却不敢说什么。表哥的沉静令人恐惧。一如那垂天
垂地的白帏，障开了生的世界和死的世界。

　　嫂子抱着孩子，在邻家的院里，没事人似的，逗孩子阿宝玩。阿
宝笑嘻嘻的，胖得两腮鼓鼓，那眉眼分明是表哥，却又不是表哥。我
记得有一次看见海婴中年的照片，看见熟悉的鲁迅先生的容颜的陌生
版本，那种总觉得有什么不对的惘然是一样的。
　　街上浪荡着一些半大不大的孩子，穿着歪歪斜斜的西装，一脸是
无知的狡狯。
　　我想，我的故乡完结了。故乡不仅仅是空间的，也是时间的，是
地域，也是人文。那出旧戏已经下场，虽然还是这个舞台，但它的故
事不再与我相干。

　　张楚的专辑《孤独的人是可耻的》有这样的文案：“这是1994年
的春天，空气里有一种富裕的气氛，每个人似乎都站在一场洪流之中，
等待着来自欲望的冲击……”

　　关于表哥有了许多传言。那么大的厂子也没法不倒闭，表哥下了
岗。他离了婚。他没有要孩子，也几乎不去看他。他出了车祸，跛了
一条腿。又听说他开了马路店。他做过那么多年的司机，他知道他们
要什么。父亲气到没力气来生气。“我能说什么？我的外甥开妓院？”
　　嫂子后来还找过表哥，愿意原谅他所有的错，愿意一切都重来。
　　但是他拒绝了。
　　到98年初，他说要做生意，从本来就过得很艰难的表姐们那里挖
走了一大笔钱，然后彻底断了音讯。谁都不知道他在哪里。嫂子在给
表姐的最后的电话里说，她给阿宝改了姓，从今以后，“和你们家没
有任何联系，就算老东西死了，也别指望孙子来抱灵牌。”
　　垂老的姑父和大表姐住在一起，这些事和后来的一切，都不敢告
诉他。他依然为他那在省城里工作的儿子骄傲，愉快的谈论他几代单
传的娇孙。他的身体已经衰弱至极，然而生活那么美好啊，他是一定
一定要活下去的。

　　那个从前的表哥的同学，那个副县长，雇表哥杀了他的妻子。表
哥做得干净利落，没有留下一点线索。但死者的丈夫在警察面前神色
过于紧张而令人怀疑，他在外面乱搞的事颇有人知道，重要的是县里
的另一派也早就想搞他。经济上的问题已经暗中在查，何况又出了这
事。审了他两天，他崩溃了，供出了一切。后来他在狱中自杀，把头
往墙上撞，生生撞死。
　　表哥在逃。查出来他还在吸毒贩毒。他行事一向不计手段也不在
乎后果，这消息倒并不意外。
　　从此再没有他的消息。

　　99年的冬天特别的寒冷。下了大雪，把路口那棵木兰压垮了。我
记得它开花的样子，白白的，一大朵一大朵，有点傻气。J 并不在乎
什么木兰，也许他根本不知道那里有一棵木兰。他苦恼的是他竟想不
出什么新奇的点子来过千禧年。他每天和网友们探讨这件事，但这只
会使他对自己的想象力越来越绝望。
　　我在电话里对父亲说了我继续考博的决定。他有些失望，但还是
说，“好吧，只要你愿意。”
　　我知道他更愿意我结婚。也知道最后把读书读成了职业，读到除
了读书一无是处，其实是很没有出息的事。
　　我已经是一只生活在虚拟世界的软体动物。J 在这一点上算和我
有一点相同，所以我们也一致虚拟了婚姻。
　　父亲在电话里又说起姑父，“我不敢回去了，他见到每个人都逼
问，我的儿子呢，我的孙子呢，他们到哪里去了，为什么不来看我。
他知道一定出了事，已经瞒不下去了，可是快死的人，我们怎么对他
说。”

　　我们错在哪里呢？我们那么艰难走过的路，在哪里出了一点点，
一点点偏差？为什么这样对我们，为什么让我们在泥泞里绝望，一点
点幸福也不给我们。

　　新年近了，在食堂的门前，时常有些学生卖贺年片。有一天来了
个瘦瘦的男孩，地下辅了一摊子，他还在树上拉根绳子，一串串的挂
了好些，美丽的摇晃。我停住脚，他就笑嘻嘻的说：“嘿，怎么样，
买一张，给男朋友吧。”
　　有一张是日本卡通片《灌蓝高手》里的流川枫，我拿起来给他看，
他说这个两块，“真有眼光啊，这是我最好看的一张。”
　　我说：“一块五。”
　　“这个是活动的啊，看，拉一下，好玩吧——要么你拿这张吧，
这张是一块五。”他拿来另一张卡，一打开，啪的弹出个骷髅头。他
笑着一吐舌头，做个怪相，“哇，好酷，寄给他，吓他一下。”我也
忍不住笑了。
　　我买了那个骷髅头。把它揣在怀里，走进宿舍楼阴暗的楼道，我
的眼泪出来了。为那个似曾相识的笑容。

　　想起那不能忘记的儿时的事：
　　阿寅表哥轻轻地爬上阁楼，然后把笨拙的我拉上去。他把梯子也
拖上来，把隔板盖好，然后拿出他养的小洋老鼠给我看―――一身粉
白，气息咻咻，娇嫩而脆弱，通红的小眼睛在小笼子里看着我。
　　三姑的声音不知在哪里，闷闷的传来，“阿寅，阿寅，……到哪
里去了啊……”
　　表哥一声不响。昏暗的阁楼里看不见他的神情，我觉得他似乎在
那暗影里沉下去，沉下去，不断的沉下去，不由得心慌意乱。
　　仿佛那只咻咻的在笼中徘徊的小鼠。
　　三姑呼唤着，“阿寅，阿寅……”
　　窗外是清江荡漾的水声。
