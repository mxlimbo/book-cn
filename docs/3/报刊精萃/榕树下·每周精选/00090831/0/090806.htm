徐家汇：尖顶的教堂和我的成长

○上海　evian

　

　　半夜里还把自己挂在网上。和一个从外地到上海来念大学的人聊
天，问他是从哪里来的，他说如果你知道我的家乡就请你客。笑了，
的确是个遥远而又陌生的地方。然后，说着说着就讲到了徐家汇。他
说他从前念书的学校，我说我曾经每个礼拜都去的天主教堂。
　　我说看到教堂两个尖顶指着天空的样子，有点忧郁，却很安静，
豁然开朗的心情。站在门口的广场上，看一幢有些年纪的大房子，居
然就象是在看一幅画。那时候成天往徐家汇跑，在地铁下面的小店里
买一些好看或者不好看的东西，耗掉一点时间，然后买一个三明治和
冰激凌，吃完了就去教堂坐在长凳子上等弥撒开始。
　　其实好在徐家汇念书还是件幸福的事情吧，那样就能每天一早去
教堂做拉丁礼，听神甫说圣经里的道理，然后回去快快乐乐地上完一
天的课。可惜，偏偏就是不喜欢那学校，不喜欢得没有一点道理。当
然就一直没有办法去教堂里的拉丁礼，而为了考一个这个城市另一头
的学校，整整一年就没有去望弥撒。于是，它也开始变得有些陌生了，
成了一个有些遥远和陌生地方。
　　圣诞节的那天晚上，和一个很久不见的朋友约好去望圣诞弥撒。
她已经在一家医院里实习了，那个过节的礼拜正好在儿科病房。见面
的时候她说知道吗？原来圣诞节也会有小孩子离开这个世界。说话的
时候是立在门口的风里等侯，人太多了，事先发的票子都被枪光了。
有两个学生模样的人在跟门房间的老伯伯吵架，他们说教堂的门应该
是向所有的人敞开的，况且，我们是真的热爱主的。摇头，老伯伯把
那扇小小的门关上，不搭理他们。于是他们走了，用英文骂了很难听
的话。后来，一个好心的阿婆给了我们两张票，她说是给她的老先生
和女儿领的，可惜他们都来不了了，天太冷了，老先生的老毛病又犯
了啊。圣诞节的时候，真的也可以有那么多的不快乐的啊。
　　大家把教堂挤得满满的。那些平时用来忏悔的小格子里也站着人。
穿着有翅膀的白纱裙的小女孩擎着蜡烛和圣母像排着队从一边走过，
唱师班里都是些从附近的音乐学院找来的学生，他们的声音从上面飘
落下来，大家都跟着一起唱着那些好听的赞美诗，神甫在很远的地方
说，我们的圣主耶稣降生了。
　　想想耶稣也是幸福的吧。每一年都可以重新降生一次。而我们呢？
这些凡人，我们做任何事情的机会只有一次。错了就是错了，没有就
永远不会再来。
　　好象也就是在圣诞节以后，开始想要用功读书了。
　　不再给自己找一些乱七八糟的事情来做，有时候会想念那些教堂
里的音乐，教堂里的人，那些陌生而慈祥的面孔，和指向天空安静着
的尖顶。但还是很坚定的，一直没有去。
　　考试考好了的时候，终于还是去了一趟徐家汇。没有坐地铁，926
路开过华山路，看到那个一直不喜欢的学校，原来也是满漂亮的样子。
还有那教堂，走到门口，却没有进去，一直在后退，站得远远地，好
看到那尖顶沉默着的模样。
　　说过要去武大看樱花的打算。开信箱时却看到一段文字，“其实
那个校园也是有樱花的呀”。又笑。不知道那么多的吧。只是一种感
觉罢了。感觉到了，自己的成长。
