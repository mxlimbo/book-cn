激情谎言

○上海　艾艾

　
    激情是个永不褪色的谎言，燃烧在生活中心，或边缘。

    每个时代都有自己的激情主题：五十年代红色革命，六十年代上
山下乡，七十年代科技浪潮，八十年代改革开放。九十年代商业大潮
席卷社会，人们忽然从推动社会变革的崇高领域跌回到被动的个人生
活空间。英雄形象蜕变成人物卡通，使命者梦想被经济动物取代。日
常生活庸常琐碎乏善可陈，理性预期与人性退缩因果相连。九十年代
还有什么激情可言？时代似乎陷入一种惰性气体的灰色燃烧状态。

    激情的社会主题的消退却使一批从不露面的女性作家浮出海面。
一统天下的英雄人物坍塌了。女性作家从自我诉说起始娓娓道来这个
年代的个体化思潮。经过半个世纪的轮回，激情的性爱主题从历史边
缘回到人心边缘，生活边缘。人们重又捧起海上繁华都会述说凡人情
欲的张爱玲，十年一梦回读苏青。时代变迁，殖民时代上海令人厌疚
的昏色的煤气灯、滴水水门汀、嘈杂的有轨电车被时髦的服饰店、现
代射灯、矿化纯水、高速地铁取代。然后刨去短暂的社会英雄梦，生
不在此心愿旁鹜但求速效的殖民地人的心态根基还是那么顽固，劲道
还是那么发达；遭遇90年代从脚跟到脑心欲赶超欧美与世界接轨的上
海国际大都会，更加枝叶繁茂。城市物质文明越发达，人心越缩水；
私欲越膨胀，视野越短浅。野心和无奈连体，虚荣与颓废胞胎。在这
样的社会心理夹缝中焕发的女性作家群芽，她们梦想而澎湃的激情表
白，是否只是另一个玫瑰色的时尚谎言？？？

    人们说21世纪将是女性的时代。九十年代以来出现的新上海女性
小说家是引领世纪初时尚的一群。自恋与他恋，高层与低俗，美丽与
聪慧，市侩与优雅集于她们一身，象这个后殖民城市的风貌一样杂烩
表面美丽，内心零乱。卫慧的大胆小说上海宝贝和网络作家安妮宝贝
的" 告别薇安" 同时广受瞩目并不是十分令人不解的现象。

    卫慧作为时代女小说家的领头羊，是当代女性寻求成功之路的卓
越代表。有人说当代美丽女性只要拿起笔来自述，约略就可知名。更
何况卫慧兼具名牌大学的香水白领职业的内衣，反叛青年的口红新人
类的舞姿，还有雅皮士的口号。真的舞文弄墨起来可以说是走笔飞花，
有一身出色的行头，可以构成极诱人窥读的时尚展品。值得深究的是
卫慧并不是个被包装的展品。商业社会作家不被知名，无异于煤气中
毒。作为作家卫慧，她是一个具备极为深刻的商品意识头脑精明的巨
贾。虽然记者出身，卫慧没有遵循新闻采访主体不参与的采访原则，
而是凭借过人的机敏，深入一线连作卧底侦探，频频获取宝贵的原始
资料，并把它们细切力斩，满浇当代人偏爱为甚的性爱沙司，烹出红
红辣辣样给所有人印象深刻的仿西式大菜。从" 蝴蝶的尖叫" 看出她
那时刀功还不甚了了。商业社会作家的成功就是成名，作家地位的确
认就是畅销。" 上海宝贝" 的确收获满舱。对于卫慧一类的女性小说
家而言，激情的爱情主题是她们与之认同披之熨帖展示诱人的精神外
衣。注意到" 上海宝贝" 反叛人物虽然众多，人物行径怪异，但核心
人物在追求叛逆的人群中并不显得真的那么叛逆。新人类天天纯真的
爱并不能挽留倪可脱去世俗羁绊做一个自由天使。虽然她反叛，却无
法抵御马克的吸引力，那是表现为性取向的男性强势下财富权势的吸
引力，是作者凭借正常途径无法达到的社会状态。天天从一开始就指
出这一点。他说：" 你和我们不一样，你进取心很强。" 从这个意义
上来说，就象昆虫伪装保护色，作家的确是凭了反叛的外衣钻进城市
内部饱吃的蠕虫。虽然自信在闪烁，魅力在闪光，一个女知识分子脱
不出拼命钻营的老路。谁说二十一世纪将是女性的时代？！社会规则
并无改观，只是又多了一半公民畈依男权社会规则而已。也许这是象
行星围绕太阳一样不可动摇的宇宙法则。但愿宇宙只有一个太阳。

    与卫慧相比，安妮宝贝象是这个时代最后一个灵魂歌手，也许弦
已走音，声已嘶咽，人群减远，荒野上她一人默祷着十字架独歌；但
她还是要把她悠长缭远的微歌唱尽。安妮宝贝实是一个真正意义上的
边缘人，随时准备从这个失魂的社会的作灵魂脱身逃。虽然逃的无奈
逃的尴尬逃的趔趄一身棉布衬衣一双翻毛皮鞋，面容平凡眼神清亮，
激情打开身体与灵魂相遇。她跌跌撞撞的姿态也许会牵动人心的伤口，
那是逃不出去的人心结痂的所在。

    激情是个永不褪色的谎言，燃烧在生活中心，或边缘。红色的革
命的谎言，粉红色的性的谎言，燃烧在生活中心，或边缘。安妮宝贝
灰色的激情真言是褪了色的燃烧灰烬，她也不能真的点亮什么，只是
生活边缘一撮待散的忧思。社会心理夹缝中唤发的女作家群芽，也就
是如此而已。
