你说让玫瑰成灰

○陕西　烟子非

　

　　你说，当火已经烧起来的时候，就让它尽情地烧，即使玫瑰成灰，
也是一种凄美，让你我的生活更有滋味。
　　真是这样吗？这个下午，我就一直坐在电脑前，若无其事的剥着
新嫩的核桃，不停的重复着两个字，“真好”。我的眼睛始终停留在
你那些文字上。没有人，连我自己也不晓得，这两个字究竟是在夸那
些流油的核桃还是在夸你这些让人欲罢不能的文字。
　　开始翻看版上我们曾留下的所有文字。那首：
　　我的惊喜和快乐
　　总是这样容易满足
　　很想永远地离开你
　　听见你的声音我就再次软弱
　　想不到竟成了今日的写照。很多次了，想着还是远离你吧。最后，
我不得不重拾温柔，把微笑给你，把目光给你，把快乐给你，能把悲
伤给你吗？
　　你说，女孩，别以为我不爱你。我说，爱你，以眼泪、笑声及全
部的网络生命；爱你，以昨日的悲痛和今日的坚贞。夜里，我看见，
你用香烟点起一阵晨雾，我只好用口红涂去那片心痕。
　　我们都曾痛快的卸下眼泪，而思念像轻燕在雨中飞翔。谁说缘是
一张充满粘性的蜘蛛网在角落里斜斜张望，你我是来自不同地方的飞
虫，本形同陌路，无意间却都撞在了这张网上。
　　我却感到群虫辞归雁难翔，如是凄凉。
　　难道无可奈何的相识在这样的网中就意味着死亡或者永诀？
　　而相识是缘分，相识是美丽，相识是指间敲打的真诚，相识是屏
幕流动时留下的默契，是个美丽的童话故事还是蜘蛛打盹时留给我们
的幻梦？
　　我说，就让我们这样互相凝望，让时间在关怀里默默流淌……
　　真的能一切随缘说网络如斯，不过是惋惜、着迷、分离。真的能
无视鬓角的寂寞以及笔落的相思，一任“无可奈何花落去”？
　　隔着网，很想凝视你直到天亮，在越来越难以邂逅你的网络里，
遥远成空想。等待的心情永远焦灼成暴风雨来临前的凄凉。
　　宁愿断绝一切可以和你联系的电子通讯。断了这些根源，就可以
杜绝一些渴望，我总是自欺欺人的想。
　　你说，“迷失”曾经在你的字典里是一个没有生机的词语。你想
远远的飞去，抛开所有的回忆，抛开所有的烦恼，没有欢乐也没有悲
伤，就这么静静的飞，直至消失在苍茫。
　　我却分明看见你将放纵的欲望在烈火中熊熊燃烧，拼命的抽烟，
喝的烂醉；你想下笔千言，把思念密密麻麻写满整个明天……
　　我还能说什么呢？我只有死了，我的死，是酸死的。
