大漠遗言

○江西　沙迦

　
走进大漠
与边缘的生命做一次告别
手心有经搓揉而出的污垢。
在松散的零界上
插入我的双腿
在从狮子般的精力中
抽离灵魂
它虽在烦恼着怒吼
却只能任自己像沙般
随风卷走，
最后
被压挤在极限的狂乱。
沙尘已使我举口维艰
吼头似有炊烟缭绕，
不一会儿
就要燃烧！
火星会顺着枯燥的血管前行，
焚烧万灵之长的思维
将是如此简单！
势必我会成为死者
时间把我变成木乃伊，
灵魂或许在等待蒸发
咒语也必是我的忧郁。
当一切尚在演变时
三百年后的来者
你怎样回答我？
