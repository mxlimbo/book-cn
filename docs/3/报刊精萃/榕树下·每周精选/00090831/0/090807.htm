角斗士
　　　　　――心灵的家园

○情深缘浅

　


告诉我，哪里才是真正的家园。
请你告诉我。
号角在林中响起，是誓死的心在呼唤，凡人的
精英抛弃最后的理智在生与死的烈火中煎熬
上帝的眼睛是闭着的。他看不到这俗世里
殊死的争斗，只听到虔诚的信徒祈祷着出征的儿子
只听到儿童的啜泣，诉说着思念的父亲
可他也只是听着，也只能听着。
血在洁白的雪上绽放出朵朵生命的花，疯狂的
战杀已在这片片飞舞的雪花中落幕，胜利的欢呼随着
最后的一声嘶叫响起
这究竟是开始，还是结束。
呵，伟大的凡间的君主，你掌控着
凡人的生死，任凭着世人的悲喜在你的手掌之中
卑躬屈膝的活着。可是，这时间的巨轮也碾过了
你豪情万丈的雄心，碾去了你
不可一世的面容
唉，这迟暮老人的哀愁，最后的心愿
也无声无息地湮没在这丑恶无耻的谋杀之中
当年轻的黎明，垂着玫瑰红的手指，重现天际 
曙光女神传来了她无声地呼唤
正义的勇士挥舞着宝剑杀出重围。战马的呼啸带走了
他痛苦的影子。但愿永远放下染满无情的背叛者鲜血的利剑
带走我吧，带我回到我的家园。
美丽的原野上罪恶的火焰无情地吞噬掉
孩童的欢笑。贞洁的面容
在野蛮禽兽地无耻蹂躏里死去。
善良带不走邪恶，美丽带不走丑陋。
呵，至高无上的神明啊，你可曾
垂下过你高傲的眼神，体察这些在你眼中
微不足道地在痛苦轮回中挣扎的世人。
若是你看的到，为何又不伸出手，来拯救
他们的苦难呢。或者你也想他们经历你前世所受的
痛苦，亦或这本是一场生存的考验，最终
我们都将面对死亡。
死神向我们微笑着走来，我们也以笑脸相迎。
日换星移
肉体的痛苦难以掩盖心灵的创伤。孤独的勇士
再度挥起生与死的双刃剑。为了原野里
儿子的欢笑，为了榉树下妻子的期盼，为了
遵守承诺的誓言，为了拯救苦难的世人
一场惊心动魄的角斗正在上演
并非带箭的夫人，眼睛雪亮的女神，射出温柔的
羽箭，在我的心上，把我放倒终结，
亦非恼人的病痛，常见的杀手，以可恨的
糜耗，夺走了人的生命，从肢体之中，而是
对你的思盼，闪亮的眼眸，对你的聪颖和
温善的思盼，切断了我的命脉，夺走了我甜美的人生。
元老院冰凉地大理石的殿堂里回荡着的
独裁者的嘶叫，难以比拟圆形竞技场炙热的沙土上
复仇的火焰燃起的欢呼。我以我的剑在此起誓，
卑鄙无耻的举止无法阻拦誓死如归的心。
呵，胜利女神呵，请你睁开眼
看一看究竟谁才是你
忠诚的战士
古老而雄伟的竞技场被血色的玫瑰花瓣染红，伟大和渺小
同时倒在了尘土之上。不朽的灵魂迷茫的推开了
未曾开启的门。粉红色的砖在太阳下安详的躺着，茉莉花香
弥漫在夜空里，麦芒在微风中摇曳着身姿，掠过
浮云微笑的脸。
一个声音从天际传来
“我来晚了，让你们等的久了……”
