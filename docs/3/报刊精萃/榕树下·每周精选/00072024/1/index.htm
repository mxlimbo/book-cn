【编者的话】

    余墨留香，这一副长卷，
    掩藏着杨柳春风下的岸渚，
    和，杏花牧笛里的梦境。
    隐隐传来惊破霓裳的鼓声，
与，恍恍然缠绵曲折的箫音。
