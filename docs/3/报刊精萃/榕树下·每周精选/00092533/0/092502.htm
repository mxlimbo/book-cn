爱的定义!

○江苏　无心人1982

　
　　医生说：爱就像一场感冒，你可以抑制住自己的头痛，但你去却
掩饰不了眼泪和鼻涕。
　　社会学家说：恋人们不怕热，即使是四十度还要贴在一起，恋人
们不怕冷，即使大冷天还要逛马路，恋人们更不怕黑，哪里黑他们就
往哪里钻。
　　历史学家说：古代人的爱情是才子佳人，近代人是我们一起去死，
而现代的爱情观则是：不管你结没结婚，只要我爱。
　　厨师说：爱情像一颗洋葱头，你一片一片剥下去，总有一片让你
流泪，如果不是那样，就是一个茄子。
　　医生又说：高血压者不宜、心脏病者不宜，因为爱要奉献，不讲
索取。
　　文字学家说：我不明白为什么要把“爱”给简写了。因为原来的
“爱”字还有一颗心，不过现在还好，起码还有一个“友”字。
