秋天深处

○北京　蜀乾尔

 

　　秋天，是大自然恩赐给人类最美好的时光，走进它的深处，到处
是色彩斑斓的交响乐，你看：殷红的枫树叶热热烈烈得象燃烧的感情，
橙黄的银杏叶恬恬静静得象纯正的良心，苍绿的松柏叶深深沉沉得象
深邃的理性。处在红、黄、绿色彩和谐的自然，品味着情、意、理的
心灵，人生的意趣都集于其间了，其乐陶陶，其味融融。
　　我一直不明白，为什么天地间人与自然如此和谐的季节，却给人
们的是那么多伤痛和悲凉，凄清和哀惋的感觉。残花片片，而黯然至
于泣下；枯叶翻飞，而悲乎猿声于两岸；天际归鸿，而思念旧人于山
峰；寒蝉低吟，而伤痛别离于长亭；秋霜渐凝，对镜梳发，而叹华发
之易衰。秋风秋雨愁煞人，风过鹤鸣断肠声。秋天，在古代文人墨客
的眼里，红粉佳人的心中，是怎一个愁字了得呀。那沉重，那郁闷，
那哀叹，那悲声，使我们竟翻不动那一片片沉重的竹简，那一册册厚
重的典籍，那一叠叠凝重的诗笺。
　　在我们抛开书本，走出钢筋水泥的丛林，走向真正的田野的时候，
经过春的播种，夏的耕耘，季节也舞着大旗，挟着鼓点，吹着号角，
豪情满怀，气宇轩昂地走来了，走来了。我们在田野热烈相遇。
　　在秋风的指挥下，一望无际的水稻轰轰烈烈地掀起一阵阵金色的
波浪，万里棉田纯纯洁洁地排着一队队白色的方阵，满坡桔树汹汹涌
涌地膨胀着一串串橙色的爱情。初秋的太阳从高天上慈爱地挥着无数
条金色的长鞭，追赶着连绵不绝的群山向我们涌动着，涌动着。那些
紫红的，金黄的，乳白的山菊花为这涌动而亢奋起来。农人们在生命
中某些激动的血液的驱使下，高傲地站在他们田野，用灼热的镰刀深
深地收割着季节的馈赠。在那些极富韵律极富节奏的波浪低伏的时候，
他们高大而矫健的身躯便高贵地显现出来。如果仔细倾听，还可以真
真切切的听到他们汗珠滴入土地的清响，或者因了阳光的诱惑而蒸发
的白雾。
　　而田野终于裸露出来，袒示着她黝黑而丰腴的躯，那些麦桩，谷
根，或者一些庄稼的叶子，用它们的枯萎衬托着土地的高亢。远处的
山坡上，极无规则却又极富美感地堆着谷垛，积着麦垛，放着高粱秸。
它们紧紧密密地簇拥在一起，静静地等待着大自然最深切的问候。这
时黛青色的山中，偶尔有一栋两栋青灰瓦房，那些陈旧的斗檐下挂着
一串串火火红红的辣椒，或一吊吊金黄的玉米棒子，使它衰老的容颜
因而有了青春优美的光泽。
　　当天边的云群以急促的心情匆匆赶回家园的时候，秋风便以极大
的热情尾随其后追逐而来。漫天飞舞的树叶毫不掩饰它们对于风的感
动，用千姿百态的舞姿解释着风的运动。而树枝犹如支支琴弦，被风
这无形的巨手拔弄出天籁般的歌谣。风从山顶疾驰向山脚，又从山脚
飞奔过河流，然后与裸露的热土激情相捅。这时满山便响着阵阵的涛
声，清清的河水翻涌起纯白的浪花，而土地，敞开着宽广的胸怀，尽
情地吸纳着凉爽的气流。常常情不自禁地欢快呻吟。纷纷扬扬的树叶
在土地不经意之间，嘻嘻闹闹地积满了她的全身。秋风才停驻在一个
村口，深情地回眸它的杰作。只见的黄叶地，只见的红叶地，只见的
花瓣地。它沉醉地吸一口气，便触动了无数乡村诗人优雅的情绪。
　　枯树沉思的时候，只有漫山遍野的荒草微微地摇晃着，给人们一
种暗示。
　　走在齐腰深的荒草中，用脚尖感受着它曾经因憧憬而美丽的过程，
体味着它曾经因青绿而淡黄的柔软，竟感觉如在黄色的波浪里穿行，
前一排浪刚向你倾述着什么，后一排浪又急匆匆地过来给你补充。偶
尔一丝草叶轻拂你的脸庞，那种麻酥酥的触电感觉刹时直传入你的心
底，把我们微小的、世俗的愁闷，卑微、浮躁或者忧伤一扫而空。它
让我们从此无法忧郁。
　　我们登上高处，对某些古老的节日进行深深的祭奠，山凌绝顶，
方知浩渺无际；秋到深处，才知此卷难阅。这秋天，是人生中何等博
大的书啊，它是如此深邃而又如此宁静，它是如此浩阔而又如此亲近。
它包容了我们所有的忧伤，也涵盖了我们所有的欣慰，它昭示着我们
回到自然，回到过去，也鼓动着我们走向未来，走向美丽。
　　大漠风终于割断我们的情丝，它大把大把地撒着雪花，雪花纷纷
扬扬，秋天在深处悠悠扬扬。
　　――不用再回首，另一个秋季在不远处等着你和我。
