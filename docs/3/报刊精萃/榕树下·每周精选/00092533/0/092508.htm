目睹死亡

○上海　寄青


　　我清楚地的记得那个雨夜，那个一直被许多人认为是如何吉利的
日期――2000年8 月8 日，当时针再次的用临近12点来宣告它一日工
作的完结时，急促的电话铃声打破了这宁静的夜和这原本平和的一天。
　　我不知道表哥是如何克制住悲伤把这个噩耗告之父亲的，当时我
只是莫名地诧异着父亲接完电话后的反应。父亲先是坐在椅子上不停
地跺着脚，然后就趴倒在桌上号啕大哭起来，他哭得那么响，尤其是
在这样一个烟雨蒙蒙的入秋之夜。
　　“怎么了？怎么了？”母亲被父亲的哭声吓坏了，急忙摇着父亲
问个不停。
　　在我的印象里父亲是一个极传统的老派男人，男人流血不流泪。
我懂事以来，这是第二次看见父亲哭泣。第一次是奶奶去世，但那次
父亲只是一个躲在房里暗暗抽泣了一会儿，所以这样的哭法着实让我
粟然恐惧起来……
　　“良良死了！良良死了！……”父亲双手握拳捶着桌子，边捶边
叫了起来。他象是用了很大的力气在叫，尤其是那个“死”字，他叫
的特别清晰，特别响，不知道是在回应母亲的问话，还是妄图提高声
音来压制住内心早已澎湃失控悲痛。
　　“死”，这个字我曾在无数篇美丽的悲情小说中读到过，也曾在
无数部催人泪下的电影中感受过，然而这一次我觉得它离我那么近，
近得让我那砰砰跳动的心脏足以感到它可怕的威慑力。
　　良良是我的侄子，虽然小我一辈，而年龄只差我六岁。在他短暂
人生的十七个青春年华中，他一直是一个既听话懂事又斯文有礼的孩
子，并且一直以出类拔萃的学业成为我们家族的骄傲。
　　在那个雨夜，当我们全家感到医院救护病房的时候，看到的只有
良良那还上存温热的身体和唇角边仍在点滴流淌的鲜血。表姐和表姐
夫扑在他身上，撕心裂肺的呼喊着“良良回来呀！良良你回来呀！”
那声音在午夜医院的走廊里回旋着，一声，一声，声声不息。丧子之
痛，痛彻心骨。瘦小的表姐死命摇着良良那已一米八二的壮实的身躯，
她红红的眼里似乎已经流不出泪花，她只是喊，只是喊，不停地喊，
用尽全身的力气在喊，仿佛可以把良良的魂魄从另一个世界喊回来似
的。表姐夫的头始终靠在良良身上，凝聚着所有的悲愤干嚎着，等他
抬起头的时候，我看到他的衣裳染着殷红的血，我知道那是良良的，
更是他此刻伤痛的心里所流出的。周围的亲戚朋友们都在哭泣着，呜
咽着，没有人知道如何去劝阻这对夫妻的呼喊，他们只能一个劲地擦
拭着自己眼角不住流下的泪，擦拭着……
　　如果那个司机开车时注意力能集中点，如果她转弯时速度能放慢
点，那么也许就不会有这一场意外；如果她的公德心多点，如果她撞
人后能及时停车而不是一味只想着逃离，那么在我的表姐和表姐夫身
上就不会发生这样天崩地裂的悲剧。然而此刻这一切的如果都只能是
旁人美好愿望。那辆水泥搅拌车硕大的铁皮车身从良良17岁少年的身
体上飞驰而过，带走了他十七岁年轻的生命和青春灿烂的梦想。据当
时的目击者说，当车开过后良良还曾顽强的坐起来过，虽然嘴里大口
大口吐着血，仍喃喃地呼喊着企求帮助。但是由于司机的逃逸，在场
无一人敢把他送往医院，直到20分钟后才由交警处理的。良良是在送
往医院的路上死去的，他活的时候是被爱包围着的，想不到却是一个
人孤零零的上路。良良死的时候是睁着眼睛的，是要看清那个肇事逃
逸的司机，还是想要向周围的路人求救，又或者是想最后见一面他的
父母。
　　良良在他的日记里曾经写到“生命是可贵的，没有人应该一事无
成而离开这个世界。”我知道，尽管他也不断感到生活的无奈、竞争
的激烈、高考的压力，不过对于生命p生活，他的热情始终不减。然
而命运却用最无情的手段粉碎的这个男孩的梦，这个家庭的希望。
　　目睹死亡，我才真正感觉到了它的残忍。“死亡”远不象文学小
说或是电影里的这般轻描淡写。以前我常常想，死亡也许可以是解脱
和逃避这世间一切的烦忧的最好办法，可是我忘记了死亡也同样剥夺
了一切生的希望，自己的，亲人的。
　　当生命逝去时，它是幻变成海上七彩的泡沫，还是升入天空成为
一个不灭的灵魂。不管作家或是诗人曾用多么美丽的文字去诉说它，
但是当它真正撞入了你的生活时，你还是会出现浑身发冷、思绪混乱、
恐惧不堪、呼吸困难等等难受之极的症状，而这些症状不是可以只用
语言，文字，或者画面就可以感受到的。
　　所以不管生活多么的让你失望，死亡永远不该是可以被选择的路，
因为你不能安心的面对你死后父母所承受的心灵折磨，也同样不能坦
然面对另一个世界里的那些热爱生命却因意外枉死的英灵。
　　目睹死亡，让我们更珍惜生命。
