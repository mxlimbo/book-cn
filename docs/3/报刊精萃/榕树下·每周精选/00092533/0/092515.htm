我从你们那里拿走了许多……

○四川　弥赛亚

　
　 

在一只陶罐里取水
在一间仓库里取走粮食

向一名处女要去贞操
向一个盲人要去光明

问候一匹马，并且
向它要走温柔的草原
问侯一只峭壁上孤立的雄鹰
要走它的整个天空

背叛宗教，但我要来信仰
背叛妻子，但我要来忠贞

向一个坐在办公室的秘书
要来记录人生的档案
向一个四川籍的厨师致敬
要来他的舌头和味蕾

建造一座桥
由此我学会了沟通
破坏一座桥
从中我又获得了距离美

从农民那里要来麦粒
从工人那里要来钢铁
从商人那里要来贪婪
从妓女那里要来淫欲

向一个聋子要走音乐——
(吉它、贝斯、鼓和无数吟唱的人声)
向一个哑巴要走言语
(那是诗歌的语言啊！)

在大海里我学会包容
在土地里我学会生长
在黑夜里我学会沉思和忧伤
在白天
在白天我学会象你们一样的生活。
