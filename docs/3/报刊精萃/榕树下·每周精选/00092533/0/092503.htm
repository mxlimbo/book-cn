包月的女孩

○福建　郑林

　
　　没有人做过统计，全中国有多少这样的小镇。肮脏嘈杂的街道，
两边布满了俗气的小店，卖着平常不过的东西，各种车辆全无章法地
乱窜，排放着呛人的烟尘，不时有拉客的中巴，霸着半个街宽，大声
地吆喝着……
　　但这个小镇有点不同，她是坐落在海边的一个风景区，有漂亮的
海滨浴场，清澈的海水，还算干净的沙滩，更重要的是有南来北往的
游客，给小镇带来不尽的财源。
　　这个小镇就是福建沿海的东山岛。
　　我就是那些游客中的一个。连续三年来，我都像候鸟一样，从几
百公里外我居住的城市，来到小镇小居几日，体会一阵生活在陌生人
中的感觉。离小镇不远的海滨浴场，有漂亮的饭店和渡假村，但我更
愿意住在镇上，可以更真实地生活。这座街边的家庭式小旅馆，我已
是第二年入住了。旅馆只在街边占了窄窄的门脸，却有四层楼高，每
一层只有四个房间，两个标准间，两个单间，一楼和二楼都是老板家
用，三、四层租给房客。我到的时候，正是春末，小镇还没到旅游的
季节，老板带着我孤零零地走上楼，生意的萧条令她显得无精打彩。
时间已过了午后，三楼的一间标准房敞开着门，传出女子说话的声音。
我突然想起了什么，问老板娘：“我去年来的时候，这里也住着小姐。”
老板娘面无表情地说：“她们是包月的。”
　　由于四楼没客人，老板娘只肯把我安排在三楼，住在小姐们对面
的单间。我打量着简陋的房间，心里却有一种对未知生活的欣喜。我
作为一个普通的公司职员，唯一的好处，就是对物质条件不会太挑剔。
　　不知有没有人对旅游方式做过归纳，但我的爱好一定属于异数：
漫无目的地来到这个毫无牵挂的地方，只是在旅馆里睡懒觉，然后到
街上闲逛一番，随便找个小店吃饭，午后便带上一沓报纸去海边，看
累了就玩玩海水，或是看渔民结成一队在岸边收网，然后再吃饭，再
逛街，夜就深了，上床睡觉，一天就过去了。如此这般三四天过去，
就烦了，于是收拾东西打道回府，重做一个本份的上班族。
　　我的行为连见多识广的旅馆老板娘都感到诧异，她奇怪地看着我：
“先生你到底是来干什么的？”
　　我无法向她解释，甚至连“旅游”这两个字都羞于出口。虽然身
处陌生的地方，但还是无法逃避世俗的目光。
　　旅馆的楼顶，是一个很大的天台，站在上面，吹着烈烈海风的同
时，还可以远眺海岛的风光，但它在旅馆里最主要的功能，却是作为
旅客晒衣物的场地。午后，我洗完衣服，走上天台却发现几根铁线都
挂满了，旅馆的床单被套占了一根，另外三根却挂满了女孩子的衣裙
和内衣，甚至还有几件冬天的衣物。奇怪，旅馆里没住这么多人吧？
看着塑料桶里的湿衣服，我只好伸手把女孩子的衣物拉密些，好腾出
些空间。待我晒完衣服，一回头猛然发现身后竟站着个人，不由得吓
了一跳。是一个二十多岁的女孩，穿着白色的连衣裙，瘦瘦的，没有
化妆的脸在阳光下，显得苍白而憔悴。她手里也提着一桶衣物，默默
地站着。
　　我指着铁线上的衣服说：“是你的衣服吧，不好意思麻烦你了。”
　　她看了我一眼，表情有些木讷，嘴角似乎动了一下，但终于没有
说出声来。
　　我转身走下楼，并不在意她的反应，陌生人嘛。
　　这晚我睡得很迟，看完电视里最晚的新闻节目后，又躺在床上看
完白天买的所有报纸，似乎已是夜里二点了。窗外的大街上已是寂静
一片，隐约听得到远处大海的波浪声，旅馆里更是如死寂一般，只有
卫生间里马桶漏水的声音。这时楼下突然传来激烈的敲门声，把我的
睡意一下就打消了。似乎旅馆老板娘也睡了，许久才有人把门打开，
随后传来了踢沓的上楼脚步声，有两个女孩小声地嘀咕什么，然后我
的对门有了开门声。
　　第二天上午我起床时，已是十点多了。女孩们的房门紧闭着，还
放着门帘，上面贴着些小饰物，是和普通客人不同。我下楼看见老板
娘倚在柜台上昏昏欲睡，便抱怨道：“昨晚真吵，那些女的这么晚才
回来。”
　　老板娘有些吃力地用普通话说：“没办法，都是做生活啦，她们
开发廊的不做这么迟哪有钱赚呀。”
　　她们是开发廊的？这下我明白了昨天在天台上，那女孩憔悴的面
容。
　　东山岛的发廊，并不开在主干街上，而是散布在大街后的小巷里。
闽南沿海的民居，都是用石条筑成的两、三层小楼，窗子窄小，占地
面积不大，楼层的高度低于内地，便于防范台风。栖身在里面的发廊
阴暗而狭小，穿着暴露的女孩子暧昧地坐在门边，操着各种口音的普
通话，招唤着从门前走过的男人。
　　小镇的通用语言是闽南话，因此像我这样的人一开口，便立刻暴
露外地人的身份。到了傍晚时分，坐在门外闲聊的房东老人嘈杂的闽
南语，和“发廊妹”的普通话交织在一起，成了小巷里最有趣的景致。
由于旅游业的开发，使世代打渔为生的小镇居民，有了新的谋生手段，
经济也富裕许多，因此“发廊妹”除了做游客生意外，在旅游淡季仍
可赚当地人的钱。
　　我到小镇第三天时下起了雨，午后我蜗居在旅馆里昏睡，不知是
什么时候，被一阵隐隐约约的哭声弄醒了。我躺在床上盯着灰白的天
花板，辨别那嘤嘤的声音来自何处。不一会儿睡意就全消了，我点了
一支烟，打开房门，对面女孩们的房门紧闭着，谁在里面哭呢？这时
候她们应该在发廊里做生意呀？我拉开窗帘，窗外的雨已经停了，但
街上已是泥泞一片，我决定留在房里看电视。
　　这是一台陈旧的14寸彩电，除了国内的几个频道外，还能收到模
糊不清的台湾电视节目。这种百无聊赖打发时光的方式，是上班族生
活中的一种奢侈。
　　不知是什么时候，门外有了动静。从我半掩的门看出去，对面的
门开了，一个女孩轻轻地走出来，正是我在天台上遇到的那位，瘦瘦
的身子，依旧拎着塑料桶。我收回目光，心里有种说不出的疑惑。谁
料门外突然传来什么东西从楼梯上滚落的声音，我急忙跑到门外，只
见那女孩跌落在上天台的楼梯下，塑料桶里的衣服散落了一地。我大
声说：“你怎么啦？”
　　她却一声不吭，脸色白得吓人，挣扎着想站起来。
　　我伸手要帮她，却被她躲开，只好讪讪地呆站着。她默默地拾起
地上的衣服，转身回房里。
　　我愣了许久，决定找老板娘问个究竟。在一楼狭长的门厅里，老
板娘正和一个老太太在择青菜，我说了刚才的事，老板娘淡淡地说：
“她每天这个时候都要上天台晒衣服，身体不行了，有病，靠另两个
女孩养她。”
　　我说：“她就给另两个人洗衣服？”
　　“不是，”老板娘说：“她就洗自已的衣服，总共就那么几件衣
服，天天洗来洗去。你想她又不出门，哪要天天洗呢！”
　　老板娘话里的意思，那女孩脑子也许有问题。我明白了天台上多
晒的衣物，都是那女孩的。见老板娘无意再多说什么，我只好转身回
楼上。女孩的房门关着的，没有丝毫动静，联想到她们的职业，她身
上一定有一个俗套而悲伤的故事。可她为什么不回家呢？如此飘零异
乡，终不是长久之计……
　　由于白天睡多了，我预计晚上将很难入眠，因此想到街上多买些
报纸打发时光。但小镇的书报到得很迟，而且种类很少，令我很失望，
我只好到处瞎逛。夜里的海滩黝黑一片，只有海浪在不知疲倦地喧喧
嚣着，偶尔有忽远忽近的渔火，令人感到无尽的寂寞。这种氛围，最
勾起人们对生命终极意义的思考，我们为什么活着……这是一个心灵
最无法承受的问题，我逃也似地跑离海滩，回到灯火嘈杂的街上。
　　在穿过小巷的途中，我又看见那些密布在小巷里的发廊，来自异
乡的女孩们，强颜欢笑地招呼我：“先生，洗头按摩！”或是另一种
更暧昧的说法：“先生，进来说说话吧！”但我孤身旅人，怎敢接受
这种服务。不知住在我对门的女孩们的发廊，是否也在这里？她们靠
出卖欢笑和肉体满足自已的生活，还奉养着不幸的朋友，在拜金至上
的道德观盛行的年代，也算是一个亮点吧。我突然有了想见她们的念
头，于是在一间发廊前一犹豫，立刻就有一个小姐来拉我，窘得我急
忙快步离开。
　　第二天上午我特意留在旅馆里，等着对门的女孩们起床。大约到
了十点多，才看见她们开了门，一个长得较为丰满的女孩，举着套好
衣架的湿衣服，小心翼翼地去天台。我跟了上去，天台上正有昨晚我
晾的衣服。
　　上午的海风很大，天台上的衣服被吹得全挤在一起。那女孩正在
把晒衣铁线上衣服分开，我走上前说：“那两件是我的。”
　　她回过头看我一眼，说：“你是住我们对面的？”
　　我说：“是的。这里衣服干得很快。听你口音是外省的，是哪里
呀？”
　　她有些诧异地看着我，随后笑着说：“先生你问这么多干什么，
想跟我玩很容易，不过在旅馆不行，你可以去我们店里。”
　　我一怔，竟不知该说什么。她笑了笑，转身要走。我急忙追上她：
“小姐，我不是那个意思，我真的想认识你们，和你们一起的那女孩，
你们不想送她回家？”
　　她的脸色一变，玩世不恭的脸上，竟然有了一丝悲伤。好一会儿，
她才说：“送哪里？谁也不知道她家在哪里！她是被人骗到东山的，
在发廊没干几天就疯了，骗她的人看她身上没油水，早跑了。”
　　我说：“这样也不是办法呀，你们能养她一辈子？”
　　她说：“谁知道呢。只要我们在这里，她起码有地方住，有饭吃。”
说完，她转身走了。我呆呆地站在天台上，脑海里一片空白。海风吹
得我浑身发冷，我从来没像现在这样，感受到自已的渺小和无力。
　　我默默地走下天台，看见女孩的房门已关了，旅馆里又恢复了一
片寂静。这个世界对我们许多人来说，有太多的无奈和悲伤。这个可
怜的女孩，难道注定要飘零在异乡，最后不知所终！我仰头倒在床上，
点燃一支香烟，呆看着缕缕轻烟被风吹散。我能帮她做什么呢？直接
的帮助？我无法介入这些发廊女的生活。还有什么我能做的？寻求政
府机构的帮助？这样的事会有人管吗！况且我自已也不过是来自异乡
的旅人，还要回去做循规蹈矩的上班族。
　　心底一个声音冷冷地对我说：你无能为力……
　　我闭上眼睛，那女孩的影子在脑海中挥之不去，一股浓浓的悲哀
笼罩着我。突然，我从床上跃起，匆忙收拾起行李。既然无能解决，
那我只有逃避！我风也似地下楼，退房，然后出门登上一辆离开小镇
的中巴，逃离让我悲伤的地方。
　　中巴在街上转了几圈，大致装满了乘客后，终于要离开小镇了。
我远远看着我住过的那家旅馆，眼里渐渐模糊。我知道，明年我不会
再来这个小镇了，她带给我的悲伤也终将淡忘，只是丢失了同情心的
我，还算不算是一个善良的人……
