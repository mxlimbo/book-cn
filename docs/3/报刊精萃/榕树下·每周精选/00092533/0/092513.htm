驼铃

○史小凤

　
　　北京的春天，沙暴阵阵，人们还没有来的及体会到春意，夏天就
抢先登陆。污浊的气流中人们得忍耐好几个月的燥热的夏天。旱的太
久了，若能有一些清凉的雨该多么好。
　　燥热中碰到了一个老朋友，十多年未见面，一说起来话就收不住。
临别时他说，“今天是我许多年聊的最开心的一次，碰到你就像沙漠
里听到驼铃声”。看着我疑惑的样子，他满脸庄重，“我现在认识的
人越来越多，知心朋友越来越少；得到的东西越来越多，而值得珍惜
的又越来越少；知道的事情越来越多，能积累的知识却越来越少；看
不上我的人越来越多，我看的上的人也越来越少。面对那么多的诱惑，
孤独的感觉越来越浓，困惑也越来越多，对吗？”用现代人的眼光看，
他现在正是鸿星高照，很多的钱足以让有许多朋友而且到处受到欢迎。
这话从他嘴里流出，尖刻的有些惊心，沧桑的叫人诧异。这是一个产
生市场英雄而不产生自由思想者的时代，这位朋友此刻有点象跃马横
枪挑破西洋镜的堂吉可德，只是身影显得那样苍凉，或许是藏起了不
易觉察的东西。
　　很长的一段时间，我企图明晰孤独这个概念，面对丰富多彩的现
代世界，有这样的企图是很不合适的，首先没有能力对“人的孤独与
不孤独”评头论足，这曾经是心理学家的专利，没有一定的专业研究
谈这一类问题本来就不合适。再说“孤独”这个概念远非舆论一律，
不可能都持相同感受。但朋友居然说出了我一直找不到恰当语言表达
的见解，这些零乱与激情的见解，一遍又一地唤醒我对那些一晃而过
的碎片记忆。
　　在一些质地相当纯正的著作里，曾描述孤独有多么美，在那里，
孤独这个概念是含混的，大概是在精神的驻地，在人的内心，它已经
无法给弄的更准确了。它可能指一个人，当然无论是精神方面还是肉
体方面的状态。一个动物、一棵树都可以孤独，孤独是难以归类的结
果。这样的状态只能是被迫的，孤独往往是人与人心的通道被堵塞，
人要面对无数隐秘，它来自个各方面，也来自人本身，于是被嘲弄被
困扰的尴尬就始终相伴，人们都在不自觉的挣脱，说不出的恐慌使他
们失去了优雅。曾看过一部片子，镜头布满了飞速的人流，无数彼此
陌生的面孔匆匆相遇，又瞬间分离，城市的景象如潮水般的涨出镜头，
令人眩晕。这些画面总是让我感慨万千，每一个人都渴望理解和爱，
却又自我封闭，茫茫人海中无数你我擦肩而过，只要能坐下来，坦诚
交谈片刻，也许就会彼此了解，成为朋友。可是我们没有这样的机会，
没有这样的勇气，更没有这样的心态，谁也不知道这是为什么。
　　瑞典电影大师伯格曼在他的电影剧本自序中进行了一次准确聚焦：
“我们最后聚集到一个牢笼里，站在一起为自己的孤独哀鸣，既不相
互倾听，也意识不到我们正在相互窒息，每一个人都盯着对方的眼睛，
却否认对方的存在，我们在原地打转，如此地陷入自己的愁苦之中。”
我常想，如果一个人生活在人群中却没有人在乎，那才是真正的孤独。
而当真情普遍成了每一个人内心的梦想和永远不能说出的愿望，孤独
也就成为人们最软弱的部分了。
　　有人把这种感觉归为现代生活的竞争，认定是竞争把我们拖到了
孤独的边缘，为了生存的利益，人们必须面对竞争。的确，竞争手段
在推动高科技市场化的同时也改变了人的思维方式和情感世界。这时
无论是思想方式还是思维方式想保持一种康德式的纯粹性都将是艰难
而痛苦的。为了适应竞争在人身上我们发现了太多的变化。那种凝固
的，建立在传统基础上的束缚被打碎，一种新的独立感，创造性赋予
了人。在许多领域中个人被允许鼓励实现自己的目标。只要努力、勇
气、机遇和吃苦的精神具备，能否把握取胜的机会，是输是赢全凭自
己。尽管还有很多限制因素，但人可以藉以自己的努力，全力以赴地
得到想要得到的东西。为了走向成功，人们知道如何依靠自己，摒弃
了以往那种令人陶醉又使人受到约束的价值信条，想得到更多的念头，
把所有的智慧线都疏通开，所有的神经都紧张起来，过去想也不敢想
的事，现在已经做到了。人开始很实际地看待周围的一切，有了竞争
的催迫，追求发财和成功已成为人们最痴心的嗜好，在功利的河边，
宁可捞一根虚幻的稻草作为护身条符，也不能在岸边观望，座以待毙，
要么游过去，要么淹死在河里。在一个竞争的时代，社会关系和人际
关系已被市场规律所主宰，竞争者之间必然是建立在相互冷淡、漠视
的基础上，彼此都把对方视为实现自己目标的手段，都把对方作为自
己的工具来使用，那种不为名利，只因意气相投结下的友谊已寥若晨
星。最严重、最糟糕的是人格异化和工具化的悲剧开场了。人不仅出
卖商品，还得把自己当做某种商品出售。市场决定着人们的特殊的价
值，甚至决定着人们整个存在的价值，如果一个人所具的特质在市场
上卖不出去，那这个人也就分文不值，正象卖不出去的商品不具有价
值一样。即使一个人可能具有自己的使用价值，但人们也要对他的价
值作出判断，必须看这个人在市场信息上是否获得成功，离开了市场
是无法对一个人的价值作出相应的判断的。举例说，医生、职员、技
术人员，假如想使自己的商品或劳务能卖得出去，必须首先得具有某
种“人格”，这种“人格”应该是具有诱惑力的，有某种专项用途的。
此外他还得具备其它一些条件，他得精力充沛富于创造性，以及与他
这种地位相称的素质。在很多时候一个人的声望决定了他的价格，所
以声望比什么都重要。难怪现在一些本属寂寞的行当，不少人也开始
了浮躁的张扬，人们愈来愈多地借鉴于苦心孤诣的商业炒作，哗众取
宠的包装手法，虚张声势的广告策略，人格操守愈来愈多地让位于赤
裸裸的虚矫浮夸。
　　社会的物化使经济活动的目的就是为了获得物质利益，人自身也
成了经济活动的中心和目的，人所从事的一切都是为了自身，个人利
益是人的活动最强有力的的动力。人的自我保护意识增强的同时，人
也就越来越自私，想用更多的东西把自己遮蔽起来，这样使人总在不
满足的痛苦边缘挣扎，耗尽了人们力图满足需求的精力，可总是留下
缺口，忧心忡忡，危若累卵，整天为没有得到什么，错过什么，被剥
夺了什么而提心吊胆。竞争规律统治着所存的社会关系和人际关系，
处处渗透着互相利用，互相操场纵的精神，人们想要在事业上获得成
功，就必须击倒对方，这样使得人与人之间的关系己丧失了那种坦率
的，符合人性的特征。别看人们在社交场合，在庆祝会上碰杯时那么
多情，但在生存竞争的舞上台便会脱去文明的面具，露出原始的狠毒。
一种阴影总是飘浮在人们头顶，那就是害怕失败，失败已成了人们的
心理负担，整个一生都笼罩在不安的梦境之中。当然，除了那制定政
策的权力精英，但他们也存在难处，他们必须积极的，存远见的创造
出一个有用的权力网络，通过这张网络来分享信息与机会，并由此取
得名利，如果他被这张网络忽视，他的处境不一会比普通人强多少。
　　一些重要的因素帮助人们不让这些潜在的不安全明显地表现出来，
当然最主要的就是人们所拥有的财产，它支撑着人们最基本的自我，
人们越是感觉到自我还不满足，就越迫切需要去争取占有财产，如果
一个人一贫如洗，或者原来的财产丧失了，那在旁人看来他的“自我”
缺少了一个很主要组成部分，权力同样在支撑着人们的自我，有些人
的权力是由他所有的财产所带来的，诸如那些富翁、企业所有者之类，
多数人是由于在竞争中取得成功的产物，有了权力就获得统治他人的
能力，就被人尊重，甚至权力已成了取得更多财富的凭证，这两者巩
固了财产所奠定的基础，支撑着摇摇欲坠的个人自我。
　　人的倾向，无论是最美好的还是最丑恶的，都不是人们固定的和
生物学天性的一部分，而都是创造人类的那一社会过程的产物，人的
本性、欢乐和忧虑都是一种文化的产物，人的性格会随着事物的变迁
而发生多种不确定的变化。在过去的岁月里，人们过着秩序井然的固
定的生活，一个人很少有机会可以轻易地从一种环境到另一种环境中，
除了少数人外，人的经济和社会生活都被一些制度和政策所操纵，这
些制度和政策实际上控制了一切，没有一个活动领域能例外。但人在
这个环境中并不感到孤独，因为个体与他在社会中充当的角色是一致
的，社会的秩序被视为如同一种自然秩序，人在这一秩序中的地位是
确定的，安全感和相属感使人们的日子过的清新而明亮。
　　竞争激励了人们创造力和活力，同时也怂恿人们物的欲望，创造
力越大，生活中的诱惑就越大。就有越来越多的人尽力扩大创造力，
把自己变成挣钱的机器，消费的动物。黑格尔的“争取被承认的斗争”
变成挣钱的斗争，有钱就被承认，没有钱想被承认就难。当然，这种
诱惑并求不是对每个人都管用，也有许多例外。我曾遇到一个电子专
家，他是最早觉悟下海的人，拥有一个相当规模的电子公司，连一些
国字号的大公司对他也退舍三分。忽然有一天，他放弃了有用之技，
而受命于哲学，当我问他何故时，他说他的心太累，觉得世界上应当
有比做当一个挣钱的机器更有趣的事。一念之差，世上便少了一个有
用的工程师，多了一个无用的哲学家。对此，他既说不上后悔，也谈
不到兴奋，只是默默地做着自己愿意和乐意做的事情。现在他做的事
少有利益的算计，没有投资的犹豫，也没有功利主义和精神压迫，因
此，也没有那种失落那种怅然，可是他却远离了我们这个时代。
　　现代文明犹如一把双刃剑，它的一面劈开了人原始的混沌，使人
的智慧和能力都使得从前的人大为逊色，而让人走向更为繁芜丰厚的
世界。它的另一面却把人卸成几块，在世间同时扮演不同的角色。人
的每一种身份都给人带来一种新的体验，新的痛苦。它一方面让人眼
观六路，耳听八方，舌辨五味，鼻息通畅，社交、聚会、电脑、网络、
光缆、旅游填满了现代人的空间，已没有人重视知觉的奥秘，人仿佛
除了接受再没有选择，语言和图画携来的讯息堆积如山，现代传递技
术可让人蹲在一隅遥视世界，谬误与真理掺拌一起抛酒，人类像挨了
一场陨石雨，它损伤的是人的感知器官，失去了判断的基本能力。繁
忙的现代人即使大睁双目，还是拨不开无形的眼障，把握不住身边世
界的瞬息万变。人们幻想般地认为自己是世界的中枢，可实际上仍被
一种强烈的无足轻重和软弱无力感煎熬着，最终使人臣服，要不然诺
查丹玛斯们的预言怎么居然有那么多人信？另一方面它又将人世太多
的残酷和丑陋供人知觉。使得聪明、功利、忙碌的现代人，不能象比
他们“傻”的传统人过的幸福，过的悠闲自在。
　　在卡夫卡的《城堡》里人软弱与孤独就很明显，那个K 千方百计
地想与一个城堡中的神秘的居民接触，他满以为这些人会告诉他该如
何行动，并指出他在这个世界上的位置究竟在何处，他的一身都在疯
狂地作这样的努力，但是他失败了，最后孤苦伶仃地离开了这痤城堡，
卡夫卡描写的孤独与失落不是所有的常人都能领悟接受的，常人有上
帝赐于的经久不衰的感情，正是这种情感时时编织着温暖诱人的网把
人们联在一起，使人们觉得自己是一个族中的一员，不孤独。但这个
网有时会让人变得脆弱不堪全无远见，而且使功利的智慧锐减，往往
在竞争中作出不明智的举动。竞争推崇的是理性，是剔除了感情因素
的客观分析，是冷静到了冷酷的利害判断。但有关感情的事很难用理
智来定论，什么时候人能够将感情看得一文不值，纯粹用理智行事了，
人也就成为地道的现代人，也就不为其它事情所累，才有参与竞争的
基本条件。就算还不具备现代人的资格，也要识相些才是，要让感情
粗糙些，坚硬些，迟钝和浅薄些，不然跟不上时代的步伐，没法与人
竞争，到头来吃亏还是自己。由此，人们正变得世故起来，孤独的影
子也时时在眼前晃来晃去。
　　今天，我们已经习惯了孤独的生存，换言之，习惯了日复一日在
拼搏中让双耳渐渐失聪，让眼球终日充血，让心被扯出一根线，川流
不息地抽丝流血。我们在竞争中，以憔悴换回生存，忘了孤独也是自
己的权力。当我告别友人，行走在喧嚣、燥热街头，就像置身在茫茫
沙漠里，多想听到驼铃声。
