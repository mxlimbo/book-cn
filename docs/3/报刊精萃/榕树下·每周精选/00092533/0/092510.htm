上海碎片

○银铃飞舞


　　上海的里弄到处都是眼睛。
　　在进到永福路的一条小弄堂之前，经过一座咖啡石砖砌成的楼，
绿树掩映，围墙外挂着“市级保护建筑”的黄铜牌，门前有测绘队员
在比画着，不知道是要为它增添保护还是推倒重来，应该，不会象广
州那样辣手摧花吧？缓缓的在里弄里穿行，不经意瞥见圆形窗里风扇
灯昏黄地亮着，扇叶悠悠地转动，可知窗外走到了哪个年代？
　　老人在弄里悠闲地要着扇躺在竹椅上。据说再美丽的上海小姐在
早晨时也是穿着睡衣，头发蓬松，睡眼朦胧地踢着拖鞋在里弄里出来
买早餐。那一刻，是她们最最放松的时刻，不用担心自己的仪态，反
正嘛……大家都是一个样，我看你就是照镜子一般，不怕不怕。所以
说，早起看美人，假如这时候你仍能遇到一位穿睡袍的天仙，她，应
该无疑是里弄之花了！在我们还斤斤计较自己的睡衣能否出的厅堂时，
上海小姐们已经给了我们一个极好的榜样。
　　走进一栋西班牙黄色小楼里，静得可怕，我尝试着克服心中的恐
惧，沿着踩塌了边的楼梯旋转上去，终于敌不过对未知的害怕，不敢
再上，怕有鬼魅突然伸手出来，将我吞噬在上海无人的小楼里，眼看
四周魅气冉冉而起，逃也似的，我飞奔着离开了这个阴沉中透漏着丝
丝阳光的鬼城，来到猛烈的阳光下，方才清醒，方才镇静，它，确实
象极了魅力摄人的旧上海小姐，带点邪气，带点古雅，可是让人望而
却步。
　　我现在站在1412的里弄里迎风写着，途人经过，纷纷侧目。
　　夜里独自走在淮海路古雅现代穿插不齐的路上，迎着微凉的风，
感受这个城市的浮华，偶尔接过上海路人的目光，我笑笑，投下自己
信心十足的铃声，哈，给你奢华一时又何妨？怕就怕，今后要来硬拼
呢。那时再来告诉你谁是赢家吧。
　　在上海最大的DISCO 里起劲的舞着，终于找到了一家音乐合口味
的DISCO ，特别喜欢，到它另外开辟的小间舞厅里跳，那里的电子舞
曲是我的心头好，广州极少有这样的DISCO.不过，拘谨的上海人，一
离开了舞池，马上就恢复淑女绅士的模样，端端正正的坐着，再不动
一下子，真是让人好生奇怪。
　　在爱尔兰酒吧里，无数的外国人，我和朋友坐在二楼，可以看见
楼下演奏着的两个外国人，一男一女，在异国他乡给自己的同胞表演
自己的艺术，台边吃着东西的外国人，丝毫不留意自己的同胞，流落
他乡的对比，让人看了感叹不已。吧台边上坐着的上海女郎，时髦的
轻举一支细烟，和围在四周的外国人聊的起劲。在吞云吐雾的同时，
我看到了她红唇边上一丝倦意，是条件还没有谈好吗？还是厌倦了这
样的应酬？不得而知。
　　上海外滩的辉煌，在夜里更加旺盛的绽放着，熙熙攘攘的人群中，
我们一路逛来，狠不得站在和平饭店的顶楼，高呼“我要征服你，上
海！”。这是我当时最真切的感受。难怪乎，当初来到上海的人们，
总也充满了野心，在这个让人觉得靡丽到极点的城市，面对着如此灿
烂的夜景，任何一个想开创事业的人，恐怕都有拥有这一切的梦幻！
即使不能实现，那么让我们一起叫出来过把瘾吧，也是极其惬意的事
情。
　　在上海老街里胡乱晃荡着，突然发现一间古老的杂货店，老板在
门口看着书，毫不在意我的出现。于是我象个刚出县城的乡下人，这
里瞧瞧，那里看看，周璇的广告画，毛主席像章，老家具，银茶壶，
缺了口的花瓶，更多的是我叫不出名的物品，看上去好象蛮是识货的
人该懂的东西，我不懂，所以不识货。呀！竟然有黑胶唱片！这一刻
的惊喜，可是自抵达上海来的最兴奋的时刻！我赶紧叫老板进来，指
着黑胶唱片，要他放来听。老板看了看眼前这个任性的女孩，没办法，
唯有放下手中的书，撤了一个大黑匣子上面的杂物，翻开盖，就露出
了一台保存的好好的留声机。然后把唱片放好，一边摇着旁边的手柄，
一边轻轻的将唱针小心的放上唱片的轨道上，在结合的那一刻，咿咿
呀呀的歌声，伴随着黑胶唱片特有的沙哑声，磨砺过几十年一般，带
着二三十年代的怀旧绚丽，终于来到我耳边……
