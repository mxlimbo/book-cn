小劳魔鬼语录

○（广州）　小劳

　

　　恐龙：很多人以为这种动物已经灭亡，其实只不过是它们发现了
网络比现实世界更适合它们的生存，而全部迁到网络上罢了，这其实
也是一种进化的表现。

　　网友：与笔友，战友一样都是朋友的一种，但这种朋友的数量却
与你每月的的网费成正比。

　　电信局的网络服务：一种优秀的商业软件，有人称之为网络蚂蚁
和网络吸血鬼的有机结合，既有网络蚂蚁的蚂蚁特性（慢且脆弱），
亦有网络吸血鬼的吸血特性（吸干你的血汗钱）。

　　微软和比尔・盖茨：一个比起在日本东京地铁站放毒气的真理教
及其头子麻原章晃受到更多人咒骂的恐怖组织及其头子。

　　网恋：比起《聊斋》中的“狐恋”和“鬼恋”更为可怕的一种恋
爱方式，与“狐恋”和“鬼恋”一样会损耗恋爱者的精力和身体，更
要命的却是会吸取恋爱者的血汗钱。

　　MM：中文学名叫“美眉”（不是画眉）的珍稀动物，这种动物在
网络上的数量比大熊猫少得多，但男性网友对搜寻这种动物的兴趣却
远比搜寻大熊猫要大。

　　垃圾电子邮件：和现实中投进你邮箱的壮阳药广告相似，都是不
请自来，不同的是收取壮阳药广告不但不费你一毛钱，还可以废物利
用（比如卖给收购站或用来垫杯子），但垃圾电子邮件却还能让你为
中国的电信业发展多作金钱贡献。

　　IE5.0 ：一种高明的病毒，用免费的字眼来吸引你上当中毒，发
作时却能让你不得不亲手格式化自己硬盘。

　　讨论政治时事的BBS ：典型如网易虚拟社区的思想之剑，在这个
地方你对持异见者骂的越难听越起劲就表示你越爱国。

　　公款吃喝：这种消费方式虽然比较古老，但却是比银行降息，刺
激旅游业，增加大学学位等方法更能促进消费和增大内需，因为这种
消费方式的消费者往往出手更为大方。

　　广州的有线电视广播：一个名为生（省）线另一个名为死（市）
线的两个强盗发明的一种新型犯罪形态，两强盗除了姓名不同和经常
因发生分赃不均而发生冲突外，犯罪的手法却是一样。

　　色情行业：某些地方为发展相关的旅游业，医药事业而大力推行
的一种服务性行业，据说这种被称之为无烟工业的行业能带动内需，
刺激消费。

　　性别歧视：无论在现实中还是网络中都是十分严重，现实中往往
重男轻女，而在网络中却恰恰相反，男性（在聊天室）很可能不被重
视甚至被拒之於门外。

　　国产游戏：与中国足球和阿斗都为同义名词，在中国支持者甚众，
每次都能给予支持者新的希望，但希望过后却是更大的失望。

　　信息高速公路：是一个典型的豆腐渣工程，行车者每次都付出高
昂的过路费，所以又名高架（价）路，但却每次都不例外地塞车而且
很容易会出车祸。

　　OUT ：进化人（新人类）语言中的一句短语，翻译其中文意思为
“某人今天告诉你说最新开发的WINDOWS3.1在电脑城有售”。

　　翻版使用者：等同於古代的罗宾汉和水浒英雄，但是他们劫富济
贫却从不使用暴力，而是使用一种较为高明而有效的方法来对抗知识
霸权和知识垄断。

　　菜鸟：鸟类的一种，据统计这种鸟类的数量与麻雀相等，因为其
飞行的速度太慢而经常成为一个奸姓商人的盘中之菜。

　　手提电话：本来是一种防身武器，但现在却被一些妒妇用作防止
配偶找寻外遇和老防止手下偷懒的工具。

　　言论自由和新闻自由：宪法中规定的公民权利，但行使该权利有
一定的限制，该权利只能在网络中行使，而且行使该权利时公民有为
国家电信事业作出贡献的义务。

　　盲肠：一个可以令医生买楼买车和及令其女儿出国留学的人体器
官。

　　不在乎天长地久，只在乎曾经拥有：这其实是一种已经过时的恋
爱观念，而现在的恋爱观念是“不在乎天长地久，只在乎有车有楼。”
（用广州话读出可能会比较押韵点）。

　　CIH ：一种绝对不用]册的免费软件，用处是令杀毒软件商们笑
逐颜开，而该软件的用户却愁眉苦脸。

　　松糕鞋：时下众多人体增高方法中最流行的一种，特点是增高效
果好，立竿见影，但这种方法现只限於女性使用，副作用是使用时需
要经常跌打药的辅助。
　　　　　　　　　　　　　　　
　　被甩：某些倒楣的人经常遇到的事情之一，此类事情往往可以刺
激酒类的消费，但每当三杯落肚这些人就会成为哲学家，能说出如
“哈哈哈，看来她（他）的损失比我大，我只不过是失去了一个不爱
我的人，而她（他）却是失去了一个爱她（他）的人。”这样的经典
哲理名言。

　　IT商人：IT是英文中特指动物的第三人称，所以可以推论出卖天
价“瘟98”和不断催促我们升级电脑的IT行业商人都是禽兽。

　　电脑：电脑其实并非由美国人或是英国人发明的，而是民国时期
的一个刚用上电的中国人发明的，他的做法是用一条高压电线慢慢地
接到他的脑袋之中……

　　打字速度：是学校和家长考核学生电脑水平的惟一标准，这种考
核方法最大的优点就是能节省纸张，因为学习打字只需要买一本薄薄
的教学书就足够了。

　　阅兵的内幕：当江伯伯乘坐检阅车检阅完威风凛凛的海陆空三军
部队后，却发现因最近将军肚发福了而导致卡在检阅车的天窗中出不
来……

　　禽兽：一些无奈而且永远进化不了的动物，还在它们不会穿衣服
的时候就被人说它们因为不知羞耻所以是禽兽，但当它们懂得穿衣服
遮丑的时候却被人类骂作衣冠禽兽。

　　大虾：就像恐龙和MM一样，大虾和菜鸟是反义词，但四者的关系
就是大虾能分辨出恐龙和MM的区别，而且很懂得用一些专业得无人能
明白的专业用语来博得MM的欢心和菜鸟的崇拜。

　　Need for speed：这不是一个游戏的名字，而是一种感觉，当巴
士司机与其他竞争巴士抢客时，而你坐在车上你就会知道司机对这种
感觉的感受是多么的强烈。

　　一年一小便，三年一大便：市领导为了本市的工作尽心尽力，不
得不参加许多交际应酬而导致肾亏和便秘，医生建议用这种方法来治
疗。

　　呢称：一种网络上的猎食工具，网狼和恐龙通常会以一个很酷或
很优美的呢称做诱饵引MM或像小劳这样的纯情少男上钩。

　　MM与恐龙的区别：以“天使般面孔，魔鬼般的身材”来形容MM，
以“魔鬼般的面孔，天使般的身材”来形容恐龙。
　　　　　　　　　　　
　　Command & Conquer （命令与征服）：一些望子成龙，望女成凤
独生子女的家长常常会使用的一种法西斯式的教育方式。

　　希望工程：燃烧起了许多失学儿童的希望之火，但这些希望之火
常常都被一些经常用公款刺激消费的官员所扑灭。

　　畅销书：如《小劳魔鬼语录》这样的畅销书，你永远不可能在书
店找到，原因只有一个，就是这本书早就卖光了。

　　网瘾：比毒瘾更为恐怖，一天不上网就会令你四肢无力，心神恍
惚，而现时戒网的惟一有效方法是MODEM很巧地被雷击中……

　　商业步行街：特指一种社会现象，在现在这个失业下岗情况严重
的时候，很多本来很宽阔的马路都被小贩们琢炝耍是这条本来是车
行的马路就变成了商业步行街。

　　超频：有很多电脑DIY们（Do it yourself ）都以为自己是超频
高手，其实那些像周扒皮（《半夜鸡叫》中的地主）似的老才是真
正的超频高手，他们可以将员工每周40小时的工作时间超致每周70小
时，比起DIY 们更懂得“物尽其用”。
　　
　　乱码：乱码这一情况除了会出现在电脑之外，也经常会出现在那
些稍微欠缺文化的明星歌星的签名中。

　　电话：已故《围城》的作者钱钟书先生曾经说过“电话是吝啬人
的问候方法”，小劳虽然很喜欢钱钟书先生作品，但却对钱先生这句
话不以为然：寄张写满几万字的贺卡到美国给亲友所花不过几块钱，
但和美国的亲友谈几分锺长途电话所花的钱却远远不止这个数……

　　夜校的出勤率：当你读夜校一段时间后，你就会发现一个很有趣
的规律，夜校的出勤率与天气温度成正比，与降雨量成反比，与年级
数成反比，与邻近考试的天数成反比……

　　三维效果：3D游戏迷们倾家荡产地花钱买3D加速卡来增强这种效
果，同样很多女士们也会不惜性命和金钱地购买减肥丸来增强她们的
三维（围）效果。

　　屏幕保护程序：屏幕保护有两种，一种是电脑自动放出可以保护
电脑屏幕的动画，另一种则是有线电视公司在播放外地新闻时保护用
户的头脑免被侵害而手动发出的静态画面（彩条）或动态画面（海洋
生物知识）。

　　中国电影：如果说高雅艺术往往只有少数人懂得欣赏的话，那么
全国百姓都无法理解而只有那么几个中国的导演、编剧、演员能产生
共鸣、懂得欣赏的中国电影可算是最高雅的艺术。

　　相对论：爱因斯坦证明时间是可以延长和缩短的理论，例如上司
训话一个小时和你跟漂亮MM聊天一个小时的长度绝不相等……

　　灌水：确实是一种愚蠢的行为，灌水者还以为自己可以用垃圾文
章把BBS 灌满，但不知道与此同时自己的金钱却也正在灌入电信局那
永远灌不满的腰包中。

　　聊天室：一个狩猎的场所，色狼们布下陷阱等着MM上钩，而恐龙
们也咧开大嘴等着像小劳这样的纯情少男迷迷糊糊地@进去……

　　网络谣言：如果网络上出现了“中国电信局今天开始为中国公民
免费提供上网服务，不收上网费和电话费”的消息，那么一定是谣言……

　　网路塞车：和坐计程出租车时塞车的感觉相似，因为你不但会为
时间白白流失而担忧，更会为因此而流失的金钱而心痛。

　　靓仔、靓女：是销售服装的个体户对顾客的称呼语，所有的雄性
顾客称为“靓仔”所有的女性顾客称为“靓女”。
　　
　　离婚：当你改变不了你的妻子（丈夫），而你的妻子（丈夫）又
改变不了你的时候，你该更换一个妻子（丈夫）了……

　　婚姻的三部曲：新婚燕尔，夫妻相敬如宾；孩子出生后，夫妻相
见如冰；到了离婚的时候夫妻相见如兵。

　　心跳回忆：当你回想起刚才被十多个山东大汉手持大砍刀打劫追
了九条街而的脱逃时的感觉。
　　　　　　
　　从奴隶到将军：男人结婚前到结婚后的写照……

　　版主和版副的区别：有人认为是凶手和帮凶的分别，但实质上却
是凶手和替罪羔羊的分别……
　　　　
　　英文的规律：其实学习英文并不难，最重要是掌握其规律，最大
规律就是所有的英文都是只有二十六个字母而且都是从左到右排列的……

　　色情光碟贩卖者：这种行业的行家们往往隐藏在黑黑的小巷中，
突然弹到你的面前挥舞着一张无论是封面还是标题都十分“吸引”的
光碟，同时说着不咸不淡的普通话：“同志，影碟（因为普通话不正
所以读成淫碟（yin die） 要不要啊？”不过当你上当买回家后准备
一饱“眼福”之际，却会发现该碟原来是你儿童时代十分喜欢看的
《齐天大圣大闹天宫》……

　　婚约：承诺的一种，不过从最近日益增多的婚姻案件中就能反映
出越来越多人不信守这个承诺了。

　　电脑发烧友：就是那种认为出现了电脑、外卖快餐和洗衣机后，
婚姻制度亦将会随之而消亡的人。

　　电脑盲：就是那种一边把鼠标放在地下用脚踩，一边向电脑生产
商投诉说电脑的脚踏不如汽车脚踏方便的人。

　　网恋的神z感：当你和与你在网上约会了一年的网连对象在现实
中相会时，你可能会发现你的网恋对象原来是一条会打字的卷毛犬……　　

　　科技股：有如一个越吹越大的泡泡糖，尚未爆炸时叫做科技股，
在爆炸后则叫科幻股。

　　公司转营：在这个网络经济炽热的时代，许多软件公司纷纷转营
为网络公司。而一些游戏软件公司也发觉原来他们卖的广告远比他们
做的游戏要好得多，於是这些游戏软件公司也纷纷转营为广告公司。　

　　历史的必然性和偶然性：一个哲学理论，简单点举个例子：Windows
的崩溃历史是必然性，但至于说在一个星期内崩溃还是在一个月内崩
溃、是告诉你非法操作然后崩溃还是蓝屏后崩溃则是历史的偶然性。

　　上网赚钱：像现实中的非法传销和法轮功一样，一面告诉你前面
就是天堂，另一面却在前面布下通往地狱的陷阱……

　　电脑速度的瓶颈口：很多人认为硬盘和光驱是使电脑速度无法进
一步提升的瓶颈口，但实际上进行硬件升级所花费的金钱才是电脑真
正的瓶颈口。

　　政府机关的奔腾3 电脑：是识别官位的标I，是领导用来松弛神
经玩玩接龙、看看VCD 的娱乐工具。

　　金钱观：从今钱观就可以反映出人类的意识形态是不断随着时代
的进步而变化的，以前是“耶稣说金钱是万恶之源”，但现在金钱却
是衡量某人成功与否的标I。

　　改变历史进程：有人说蒸汽机的发明改变了人类历史进程；也有
人说电脑的发明改变了人类历史进程；但小劳说，只有那些电影和电
视剧的编导才会去刻意改编历史的进程。

　　保险公司经纪：只有在他们口中你才能体会到世人所要经受的劫
难是多么的多，而世人在经受这些劫难后又是多么的幸运――前提条
件是购买了他们保险。

　　日本青年社：绝对是个有名无实的组织，明明说是青年社，但从
社员到社长却全部都是糟老头。

　　落后的电子商务：当许多色情网站已经在网上把贩卖色情光碟的
生意做的如火如荼时，网络公司还在想办法解决网上交易的安全性问
题呢……

　　翻译软件：软件的一种，最独特之处就是它能把一篇很严肃的英
文祭文翻变成一篇很幽默的中文笑话。

　　隐形眼镜：我绝对相信这种东西会隐形，因为我从来未曾尝成功
过从眼镜盒和眼睛之外找回这种会隐形的东西。

　　节俭：这种美德并不需要学，也不需要培养，只要你在刚发薪的
那个星期花掉了薪水的3 ／4 时，你就会在下个月发薪之前学会节俭……

　　技术发达：如果说某城市的互联网技术发达，那就是说这个城市
搞互联网的人都会很发达。

　　热门话题：就如口香糖一样，一旦放入口中，人们都会死命地嚼
呀嚼，越嚼越有味，但其中的糖分全都溶掉的时候，就会只剩下一团
又没味又难看的烂胶而已。
