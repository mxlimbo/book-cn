一部让人落泪的喜剧

○上海　c和弦

　
　　就在两个小时之前，我根本不会相信一部喜剧也会让人唏嘘动容，
沉浸其中。而两个小时之后，当我静静地听完片尾曲的最后一个音符，
看完屏幕上的最后一行字幕，直到这张名为《美丽人生》的电影影碟
从影碟机里缓缓吐出时，却依然能清晰地感受到自己心绪的起伏。于
是，我开始明白，一部喜剧电影所能达到的最高境界是怎样的。
　　这部勇夺71届奥斯卡三大奖项（最佳男主角、最佳外语片、最佳
电影音乐）的《美丽人生》是继《邮差》之后，又一部获巨大成功的
意大利电影。影片描述了二战期间，犹太青年基多，来到小镇多斯坎
追求他的理想、爱情。在这里，他与小学老师朵拉坠入情网并共筑家
庭，几年后他们有了一个小孩。然而，残酷的现实很快打破了一家人
平静、幸福的生活。在法西斯主义政府的管制下，一家人被关入集中
营。面对着屠杀和摧残，天性幽默、乐观的父亲基多，为了让儿子保
有童年的快乐与天真，始终让孩子相信这只是一场游戏，一场竞赛。
直到他最终遇害，妻儿脱险，孩子仍以为这是一场游戏，而自己获得
了冠军。
　　这是一部完全意大利式的电影，也是一部纯粹以剧情和演技取胜
的电影。全片没有一处宏大的场面和夸张的音效特技，但扣人心弦的
剧情和演员精湛的表演使观者完全忽略了这些因素的存在。这也引发
了人们对于电影本质的进一步思考，即什么才是一部电影的精髓所在，
感观上获得的刺激与心灵上获得的“刺激”究竟哪一个更为重要？哪
一个才是未来电影的正确方向？意大利国宝级电影大师罗贝多贝里尼
在这部影片中自编自导自演，尽情挥洒着自己的才情。他的表演完全
承袭了传统的意大利式的喜剧风格。这种如孩童般天真、夸张的表现
方式也许会让看惯了“港式”“美式”搞笑的你有些不适应。但当我
看完全片后，再回想那些“星驰”“凯利”们的耍宝演出，反而觉得
他们有一种说不出的幼稚和笨拙。因为罗贝多贝里尼的表演是如此执
着而精到，在他所饰演的基多身上，自始至终洋溢着欢乐的气息，使
观者由不适应到适应，由会心到会意，直到最终自己的思维也变成意
大利式的了。至于这部电影的音乐，更是好到让人几乎不能感受到它
的存在。
　　很明显，罗贝多贝里尼想要在影片中表现的决不仅仅是基多这一
个人物的品格，而是以基多为代表，反映出整个意大利民族乐观、率
真和浪漫的个性。即使面对严酷的现实，仍不放弃对理想、对爱的追
求。在影片中，基多反复对孩子说：“孩子，这是一场游戏，赢的人
可以拿奖品。”这虽然是一时的搪塞之词，但在他的心中，何尝不是
把眼前的苦难也仅仅当成是人生道路上的一场游戏呢？
　　事实上，《美丽人生》所表现的题材与《辛德勒的名单》有些类
似。所不同的是，它给观众带来的震撼并不是以近乎真实的再现来完
成的，而是通过一种巨大的反差对比来实现的。一面是机枪和毒气室，
一面却是欢笑和游戏。即使面对战争与杀戮，主人公的心中仍有一片
属于他的美丽人生。在浓浓的夫妻之情、父子之情面前，表面凶悍的
纳粹内心却是如此脆弱和空洞。在崇高的精神和爱的面前，暴力又显
得如此无能为力。因为武器可以消灭肉体，却无法消灭精神和爱。基
多在集中营中大胆、幽默的表现，正是对法西斯统治者的极大嘲弄。
　　在众多的文艺片中，《美丽人生》是极为独特的一部。现今的影
片，为了打动人心，往往存在两种倾向：一种是过分煽情，刻意制造
浪漫，结果却显得矫揉造作，如《泰坦尼克》；另一种是过于沉重、
压抑，令观者心情郁闷，如《英国病人》。而《美丽人生》则独辟蹊
径，悲中有喜，笑里含泪，完满中伴着遗憾。影片中真正抒情的场面
不多，而每一个都如一段经典的旋律一般，让人难以释怀。其中所蕴
涵的能量更可以让你的泪水突然间夺眶而出。贝里尼出色的创造力和
驾驭全局的功力由此可见一斑。
　　据说，意大利教宗破例观看了这部电影，并赞誉有加。但这并不
重要，因为象这样好的影片是不可能被埋没的，正如璀璨的宝石即使
掩身于淤泥中仍掩盖不住它的光芒。在好莱坞商业片泛滥的今天，
《美丽人生》让我们领略了艺术片的巨大魅力，也令我相信——喜剧
同样可以让人泪流满面。
