卧虎乎？藏龙乎？

○上海　蔡骏



　　好久没有在电影院看电影了，好几年了吧，走出“燎原”的大门，
我跨过长寿路上的天桥，没什么人，晚上的风很大，掠过我的额头，
看着脚下来来往往的车流和两边的闪烁的灯光，忽然有了一种难以言
表的感觉。
　　若不是因为必须要看的原因，否则打死我也不会去看商业片的，
也许是李安导演的鼎鼎大名的作用吧。《卧虎藏龙》，一个非常俗气
的名字，仿佛是地摊上出售的某部拙劣的VCD.电影甫一开场，是从平
淡之中拉开序幕的，然后从盗剑开始情节变得紧凑，接着又是平缓，
再紧张，如此循环往复，直至结束。具体情节没什么说的，无非是传
统武侠片惯有的例子，一个大侠，一个红颜知己，然后又是一个不知
深浅的却出手惊人的小姑娘，章子怡饰演的这个人物虽然有些新意，
但依旧是换汤不换药而已。的确，一张一弛的节奏和扣人心弦的场面
还有那个故弄玄虚不大不小的悬念完全具有的现代武侠电影的所有元
素，也确实抓住了观众的心，但正是这些却把李安所要真正表达的东
西掩盖掉了。
　　那么李安在这一切表面的东西之下，究竟想要表达什么呢？在李
幕白与俞秀莲的对话中，一语道破了天机——人的心里才是真正的卧
虎藏龙。至此，才点明了电影的名字《卧虎藏龙》的意义，可是，电
影是否真正表达了李安导演心中想要表达的这种东西呢？众说纷纭，
我不敢下以定论，但究竟卧的是什么虎，藏的又是什么龙呢？可以肯
定的是，以李安的水平，他所谓的卧虎藏龙绝对不是表面所见的武功
或是正义与邪恶，而是一种深层次的象征意味，虎和龙就是一种欲望，
这绝非我们通常所说的人的七情六欲，而是埋藏于人的心灵深处的
“原欲”。提到这个词，熟悉电脑的朋友们一定不会陌生，近年来电
脑黑客猖獗，人们在研究这些人为非作歹的动机时常使用这个词，但
依然没有人能给“原欲”下以确切的定义和解释。当然，大导演李安
所要表达的“原欲”绝对不会来自于电脑黑客，甚至他心中根本就没
有这两个字，但是从电影中李安所深入刻画的某些细节我们还是能看
出一二的。
　　李安是台湾著名的文艺片大导演，他的《感情与理智》是华人导
演在世界影坛上的最高峰之一，他最擅长的是刻画人的内心世界，在
平淡的生活中人的情感。即便在《卧虎藏龙》这部典型的商业片里，
他依然努力地想要坚持自己的艺术追求，这是李安的可敬这处，但同
时也是李安的不幸。就象一个清纯的少女，一旦涂脂抹粉浓妆艳抹地
就失去了她的可爱，即便她的内心依然纯洁无暇。武侠片的特点决定
了李安永远都无法在这片土地上培育出他的树苗，再好的种子，到了
一片沙漠里也是要夭折的，这不知是李安的悲哀还是中国电影的悲哀。
所以，我们可以看到李安已经竭尽了全力，他的大牌演员也竭尽了全
力要贯彻他的意图，但无奈最终却是画虎不成反类犬。
　　但李安导演的确赋予了武侠电影以一种新意，虽然保持了令人眼
花缭乱的武打镜头和情感纠葛，但对于人物内心世界的刻画却沿用了
他的文艺片传统，既有狂乱，也有平静，动静结合，从李幕白与俞秀
莲不动声色的情感到玉娇龙与罗小虎任意放纵都是带有深刻的李安传
统的。只可惜，拙劣的情节和世俗的打斗镜头破坏了这一整体性。而
且，最后李幕白的死似乎也有牵强之意，难道一定要让他的死来感化
娇龙吗？难道短短的几分钟就能改变一个人吗？当然，剧情也不允许
李安在这方面施以重墨。
　　从各个人物来说，每个人的个性的确是表现了出来，对于李幕白
与俞秀莲，我总感到这两个人物还有深入挖掘的可能，最后以李幕白
的死和真情流露而戛然而止恐怕有些生硬。而玉娇龙这个性格复杂的
角色在个性方面也确实表现地不错，某些细节的处理也很见功力，但
以最后如此简单地感化让人觉得一切都可以用死来解决问题，但事实
真的如此吗？倒是剧中穿插的一段玉娇龙与罗小虎在新疆的生活令人
耳目一新，让人感到了一种野性的力量，加上那种“大漠孤烟直，长
河落日圆”的风光让人在视觉上得到了一种享受。也部分地揭示出了
李安所要表达的人心中的“卧虎藏龙”也包含了人性的自由。所以，
李安的虎与龙是善恶共存的，这也是符合哲理的，善与恶（当然，这
只是最简单的表达方式，世界上没有简单的善，也没有简单的恶）是
可以互相转化的，乃至于在人的心中轮回无止尽。
　　在电影特有的一些艺术因素上，除了武打镜头以外，李安还是保
持了他的特色，没有过多地运用摄影技巧，如主观镜头、空镜头、特
写镜头等很少，基本上还是生活流地表现，尤其是关于李幕白和俞秀
莲的镜头。至于最后一个镜头，也就是玉娇龙从武当绝顶上一跃而下，
离开了他的爱人，飞翔一般地自我了断，永别人生，的确有些出人意
料。那武当绝顶的悬崖峭壁，还有玉娇龙一身飘扬的白衣，在云雾缭
绕的空中如飞鸟般的镜头给人以最后的视觉冲击，不过更重要的是给
人以心灵的冲击。所有的人心里都会有一个问号，她为什么要求死，
尽管爱人就在身边。是受了李幕白之死的感化吗？是感到自己欠了李
幕白一条人命而心存愧疚吗？是厌倦了无奈的人生吗？是一种向道家
更高精神层次前进的解脱吗？李安导演没有给我们指明答案，他是给
了我们以思考的空间，类似于中国画里“留白”的高明技巧，没个人
都可以有自己的答案，答案在我们的心中，因为每个人的内心深处都
有“卧虎藏龙”。
　　虽然，李安导演在各方面都竭尽了全力，但也许这从一开始就是
一个错误，错误的时间，错误的地点，错误的故事。我们无奈地看到
文艺片向商业片俯首称臣了，于是，一个天才的导演也自跌了身价。
《卧虎藏龙》，李安导演，真的卧虎乎？藏龙乎？
