从梦中醒来……

○上海　风儿.

  
我的影子将消失于你的影子，
自从玄奥的爱绝望地将我占据
多少颗星在你肩上燃烧着跌落
点亮我执着而不朽的梦幻

而今，我的心沉睡在枯竭的河床
两岸轰鸣着无边的祈祷
我触摸一段隐蔽的命运
它的内核之光向着巨大的孤独沉没

在泪水中醒转，灵魂依然挂在
天空的枝头，淡无声息地飘摇
在存在的边缘低叹，一个碎了的影像
或一片轻狂的唏嘘

焦虑和恐怖幽灵般滑过灯光
阻断的意象排列奇特，纷至沓来
时间里有一种深深的漫长
一朵陌生的花在我身上开放

突然，一切记忆都变得闪亮
支离破碎的相思漫天垂落
你的声音坚定而亲切
穿过生疏的茫茫黑夜
