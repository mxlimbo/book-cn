○浙江　商略


那头青丝。是不是去年秋天里
沉郁不散的那片云？飘浮于无住
凝视时的满眼秋水。可也曾
一花一心事？一蝶一春梦？
听听啊。青灯燃尽时
窗下苔青的低吟。和孤雁长鸣的凄凉

冬天。以同样的茫然
倾听月光在水面破碎的声响
那深居着的。参不破的色空
在喃呒声里开不开得出
一颗不生不灭的莲花？

风。踩着你长长的影子而来
顿时的轻轻一偈。溅响了空空暮色
和一地落叶的竹林。你的影子
走进死寂和冰冷的月光里
想想啊。明天
谁能拾得那片片、瘦瘦弱弱的枯叶？
