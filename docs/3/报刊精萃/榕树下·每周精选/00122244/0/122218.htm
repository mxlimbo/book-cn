重逢

○四川　blackwindow.蓼蓝


　　叶子没想到会在这里遇到杨树。
　　叶子没想过会以这样的方式和他重逢。
　　星期天逛街购物是叶子不可或缺的节目。这天，清晨的阳光稀薄，
带着清新的凉意，洒在叶子的单薄的身上。她也同以往任何星期天一
样，拧着鼓鼓的一个包，口里吃着冰淇淋，坐在商场门前的凳子上喘
气。
　　叶子喘气的时候仍忘不了东张西望，这是她改不掉的毛病，不管
在什么地方什么时候，她都喜欢东看看西望望，好像有满心的喜悦与
欢愉要去发现。
　　叶子在望向她左边的时候，目光正好和一个男人的目光撞了个正
着。
　　好熟！叶子的心突突地跳了跳。
　　眼前的男人一身笔挺的西装，脸庞清瘦但硬朗，透着叶子素日里
所欣赏的干练与沉着。
　　好熟！叶子心里又一次肯定。有些什么东西从她的记忆里飞快地
跑过，眨眼就不在了，只留下一条浅浅的纹路，叶子沿着这条纹路上
朔，找寻，搜索，像猎狗似的逡巡，却最终也没嗅出什么来。疑虑更
深了，那些东西一会儿电似的溜出来，没等叶子看清楚，又闪电似的
缩回去了。叶子的大脑因而一片空白，却又分明感到乱七八糟的，闹
哄哄的纠缠作一团，无从打理。
　　叶子的心里有些紧张，怦怦的跳个不停。要不是冰淇淋化了，从
嘴边流过的冰凉感觉让她清醒过来，真不知她会不会就这样出神下去，
没有止境。
　　叶子手忙脚乱起来，一会儿找纸巾，一会儿包里的东西又快落下
来了，一会儿又望向那个陌生但又分明熟悉的男人。叶子乱作一团。
　　就在这时候，一个清丽的女子走到了男人面前。叶子看见始终看
着自己，嘴里向女子说了些什么，她便回过头来看了看叶子，目光有
点意味深长地，然后独自离开了。
　　这女子也好熟！叶子忽然有点糊涂了，好像在那儿见过，却又想
不起来，反倒想起了这么一句：长得挺好，可惜走路的姿态太不够优
雅！
　　真是女人的天性！叶子自嘲似的笑了笑自己，却看见那个男人径
直向自己走来。
　　近了！更近了！不能再近了！叶子的心再一次紧张起来，像被什
么拧着，升得老高，怎么也落不着地，在半空中晃呀晃的。
　　他走路的姿态，他终于清晰明朗的棱角，他含笑又紧张的眉眼……
叶子的大脑渐渐开阔起来，膨胀起来，却还是缺乏一个明确的开头。
　　　　　　　　　　　　　　　
　　叶子！停住了脚步的男人目光定定地俯视着慌乱的她，准确无误
地叫出她的名字。
　　是他！一个声音在叶子的心里惊叫起来。刹时刚才的那些闪电般
的东西都从她的大脑里有条不紊地一一掠过，稍停，好像在等待叶子
的检阅。
　　竟然是他，杨树！叶子先前因高度紧张而绷紧的弦一下松懈下来，
浑身软棉的没有一丝力气，杨树！她的语气也像田地里刚绽放的洁白
的棉花，蓬松而柔软。
　　果然是你！看得出杨树有点激动了，便显出一种手足无措的窘态
——这是以前在叶子面前惯有的神态，每每这个时候，叶子便觉得他
那种大男孩的羞涩与腼腆特别惹人怜爱，心里就洋溢着缕缕温情。
　　那温情此刻竟像潮涨潮落般地向叶子涌来，潮水攀过她柔软的身
躯，漫过她的眼帘向深处浸去，然后回流，又褪出来，叶子便觉得漫
天漫地都是潮起潮落了。
　　　　　　　　　　　　　　　
　　我们，到里面去坐坐吧？要树希冀地又略带犹豫地说，还是那种
不甚自信的犹豫，叶子一眼就看到了，于是说：你还是老样子！
　　两人相视一笑，叶子仿佛又回到了从前，看到了乡村小路上，两
个放歌的孩子并肩而行的身影，以及那天似乎永远不会落下的绚丽的
晚霞……
　　让我来吧！杨树接过叶子手中的包，向商场走去。他的背影宽厚
了许多，再也不是以前那个单薄的让人怜惜的少年了。叶子想，他的
确变了许多。
　　叶子满腹心事地慢慢地跟在杨树后面，商场里人流如织，她却视
而不见，眼里看到的全是一个梳着辫子的女孩子，背着书包，一蹦一
跳地跟在一个清清瘦瘦的男孩后面，他们的歌声随着风吹过麦浪此起
彼伏……
　　小心！叶子感觉一只手搂住了自己的腰，把自己从无边的往事中
忽的一下拔出来，由于没有准备而失去了平衡，一下踉跄起来，回过
神来，却发现原来上了自动楼梯，没有站稳，差点摔倒。
　　那只手把她扶正，很快地，又拿开了。
　　叶子没有在意这些，她心里想的是：他的声音比过去好听多了，
手也有力度多了。
　　想到这儿，叶子的嘴角便绣出了一朵小花，就像放学路上男孩摘
下结在女孩辫子上的那种，粉粉的，泛着野草的清香……
　　　　　　　　　　　　　　　
　　两人在三楼的咖啡厅坐定。此时正值上午，人不多，三三两两地
分布在角落里，都在切切地私语着，空气中弥漫着煮咖啡的焦苦香味，
透进人的心脾，沁着微微的心酸。这正是叶子所喜欢的环境。她有点
感激了，却没有说出来。
　　杨树选了个临窗的位子，从窗户向外望去，可以看到城市匆匆忙
忙川流不息地滚滚人潮。俯瞰都市的繁华与匆忙，让人倍感闲暇时的
可贵与可爱。这正是叶子喜爱的心境。叶子又一次被感动了，她低着
头，落了座，恰好看到窗外的阳光从的玻璃里折射过来，眼睛里便满
是七彩斑斓。
　　咖啡送上来了。杨树给叶子要了份甜点，精美而松软，他说过，
叶子的胃不好，不能吃硬的。
　　杨树给她的杯子里放了几块方糖，然后就一直搅拌着自己的咖啡，
叶子也和他一样搅着，勺子和杯子轻轻相碰发出的声音，细微而又敏
感地传递着微妙。
　　我们，有六年没见面了吧？终于，带着肯定却又盼望叶子肯定的
眼光，杨树凝视着叶子，身子微微向前倾了倾。似乎想努力地缩短这
六年的时间。
　　叶子迎着他的眼光，肯定地微笑着点了点头，但叶子又分明从自
己的笑里品出了咖啡的苦味，涩涩的，满口里钻。
　　杨树却在叶子的目光与微笑中低下了头，又沉默了。
　　以前你不是这样的，只要我肯定了，你是不会低下头的，你只会
仰着头，唱着歌或是扮个鬼脸，逗得我哈哈大笑，叶子想，你到底是
变了。
　　十九年了，叶子喃喃地说，声音细小得连她自己也不确信自己曾
说过。
　　杨树却听清了：十九年了！他重复着，刚才看到你的时候，我真
不敢认你，你变了许多。
　　说到最后一句时，叶子把杨树心里的怜爱与无奈听了个真切。叶
子想，你又何尝不是？
　　村里的杨树，还是老样子吧？该长高长大了许多吧？叶子忽然这
样问了一句，杨树似乎没想到她会问这个问题，怔了怔，还是略带眷
恋地说：应该是吧，我也好久没回去了，有两年了吧。
　　大家于是都有点怅然若失，没有再继续下去，但又彼此都感觉到
对方已经沉浸在回不去的童年与少年时代……
　　村里的那棵杨树，到两人出生时，有五岁了。两人出生时，正是
大雪纷飞万物凋零的时候，杨树却直直地硬挺着，那几片叶子打秋起
就挂在枝丫上了，在寒风中飘呀飘的，欲落未落。于是，杨家的大人
给他们的儿子取名叫杨树，叶家的大人给他们的女儿取名叫叶子。
　　待到两人可以一起玩耍时，便整日里的形影不离了。小学，初中，
两人要好得像杨树和叶子，不分彼此。每次遇上村里调皮的小伙伴，
甚至爱开玩笑的大人们，见了他俩，也说：哟，杨树叶子，杨树叶子，
放学啦……那时的杨树和叶子便抿着嘴直笑，快乐单纯得像两片在风
中纷飞的叶子，在西天灿烂的云霞中，在田野间散发着草香与泥土芬
芳的小路上，在小河里吱吱呀呀流转的水车上，自由地飞舞，放歌……
　　后来，初中毕业时，叶子考取了师范学校，离开了家乡，成了率
先飞出村子的叶子，而杨树，则念了高中。
　　叶子上师范时，常常回家看望父母，也看望杨树。
　　杨树基本上像个大人了，面对出落得越发窈窕的叶子，便有些不
自在，话也少了许多，但叶子却总能感受到来自他内心的某些小小的
颤动。这些小精灵总让叶子的心里无端地有点牵挂。
　　两人的见面不知从什么时候起有点尴尬，空气里总流动着未知的
紧张与喜悦，种种未知的情绪便在各自的心中滋生，发芽，抽叶，直
到后来，叶子全家从村里迁进城里，叶子从师范学校毕业，就再也没
有回过村子。再后来，听一位亲戚说杨树考取了吉林的一所大学，走
的头天晚上，他在叶子废弃的家里呆了很久；走的那天，他摘下了杨
树上的叶子，放进了自己的行囊……后来的后来，就犹如石沉大海，
再也没有彼此的音讯，直到今天！
　　时间过得可真快呀！好像这六年是在一夜之间冒出来似的！叶子
感慨万千地说，把陷入往事泥沼中不能自拔的自己用力地拉出来，太
用力了，以致有了生生的疼痛。
　　杨树也笑了笑，却笑出了满脸的沧海桑田，看得叶子的心隐隐作
疼，眼眶湿了，温热温热的，就像那西天不落的晚霞。
　　两人又零零碎碎地谈了些六年各自的琐事，双方的眼神都专注而
无助，只想竭力捕捉住对方每一个瞬间的微笑与忧伤，每一个不经意
的思念与留恋，每一声长长短短的唏嘘与叹息。
　　咖啡厅里一会安静，一会嘈杂，像走马灯似的换着客人，流走一
个个不同的欢聚或离别，带走各自或悲或喜的人生……咖啡冷了，又
热了；热了，又冷了，直到华灯初上，天边出现了点点星辰，两人才
从无边无际的往事中挣扎出来，柔和的灯光照着，那游丝般浮动的灰
尘渗在各自的眼里，一揉，一片晶莹。
　　末了，叶子终于问出了她早就知道答案却一直想问的问题：她是……
　　我妻子，结婚刚半年。她很像一个人！
　　叶子一下恍然大悟起来，难怪刚才看她也眼熟，原来，她有点像
那个人，那个在晚霞中自由地放歌地女孩。
　　叶子的心倏地被什么东西堵住了，郁闷地难受：是那硬挺的杨树，
那飘呀飘的总不落的叶子，是那吱吱呀呀流转的水车，还是那一大片
一大片被渲染过的云霞……
　　你……？杨树欲言又止，却掩饰不住的期待与失落。叶子仿佛又
看到了老房子里伫立良久的少年，在昏黄的灯光下，投影在冷清的四
壁，最后被收捡进那青青的杨树叶子里，带走了。
　　我还没有……叶子苦笑了一下，她感觉她已没法说一句完整的句
子了。她只得低下了头，没敢去看杨树的表情。
　　又是搅拌咖啡的声音，除此以外，似乎再无其它。
　　走吧！很晚了！杨树说这话的时候，叶子始终没有再看他的脸。
　　起身的时候，叶子想去拿包，杨树的手轻轻地覆盖在了她的手上，
拿开，些许的颤抖，然后，拧起了包。叶子跟在他宽厚的背影后面，
沉默着，走出了咖啡厅。
　　　　　　　　　　　　　　　
　　咖啡厅柔和昏暗的灯光依然亮着，似有若无的音乐穿梭在每个角
落，流淌成河，临窗的座位空无一人，唯有两只杯子相对着，沉默，
无语，中间隔着静默的河……
