台风——碧利斯

○重庆　四月风飞



今夜 南太平洋不太安静
鲨鱼四外逃散
大洋深底 惊涛骇浪
是碧利斯女神不高兴了
她曼妙的身姿
不愿再在海底隐匿
该出来走走了

江南 稻谷在田里入睡
据说 风云一号在宇宙寻视
想来 是没事的

1号气象员：南太平洋形成热带风暴
2号气象员：碧利斯台风向江南奔来
我平静地接收这个消息
该到季风的季节了

只可惜运河的小火轮是要系在港口里了
只可惜我远方的你母官又要电视讲话了

听说登陆了 是碧利斯
那个有着水蛇一样身材的女神来中国了
她要来江南采集未熟的稻子
——我父亲的生命

怎么办，我伤心的父亲
夜夜惦着他的稻子
无法入眠
