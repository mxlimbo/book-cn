一个修脚工的幸福人生

○简单的鱼

    修脚工，似乎是一个很暗淡的行业，一提起，兴许有人会嗤之以
鼻：“嘿，臭不可当的工作！”所以现在的年轻人极少愿意入这行，
哪怕是万不得以非得干这个，那也是随时就想找地方跳槽才好，能安
下心来踏踏实实学手艺的人少而又少。可是干了一辈子修脚工的老张
从来不为自己这行的前途渺茫担心：这世上总得有人吧，人就有脚，
保不准就得有脚病，那自然得有人拾掇吧，修脚这行就没不了！其实
这种道理本来就象一加一等于二那么简单，可是有的人却喜欢杞人忧
天的把事情想的灰暗，老张不懂那种所谓的灰调的时髦的人生理念，
更不可理解由此衍生出来的那些千奇百怪的生活方式。在他看来日子
只有一种过法——幸福的过法。
    幸福在很多人嘴里念起来好象是个沉重的奢望，可是在老张心里
就仿佛是个随常的物件，他的幸福无处不在，简单平常的象吃饭睡觉
一样。
    做了这些年修脚工，老张经过战争经过乱世，也经过改革开放商
品经济，可是他始终如一的做着自己的工作，一丝不苟。老张做人有
一套自己的标准，他相信有好手艺才能有好生意，他从来不为了一些
蝇头小利做些让自己跌份的事。他是个修脚的，可是老张的骨头是比
什么人都硬。这些年，老张自己开了个治脚病的小门诊，由于手艺好
每天的都是顾客盈门，按一般人的想法，这年头要是有好生意谁还不
赶紧多挣点票子呀，可是老张就不，不管会耽误多少挣钱的机会，他
每天还是晃晃悠悠的早上九点去诊所开门，下午五点准时关门。不管
是谁，来得晚了，明天请早，绝无例外。有一次，一位很有点来头的
领导生了脚病，要来看脚，却先派了个神气十足的秘书来打头阵，小
伙子一副狐假虎威的样子，满口官腔，老张偏偏就不买他的帐，到了
五点准时关门。秘书急了，赶忙抬出领导的官衔来压老张，寻思自己
一张嘴还不把这蔫老头吓趴下，可是老张神态自若的关上门，看都没
有看那秘书，径自回了家。有人说老张太不识时务，是顽固不化的老
脑筋，老张乐呵呵的置若罔闻，别人说什么管他呢！有的时候老张的
徒弟建议师傅周末别歇班，老张顶他一句：我也是人，为什么别人休
息不让我休息？
    可是老张也不是没有破例的时候，有一回他要关门的时候看见一
个男孩背了自己的奶奶来看病，男孩看上去比老张的孙子大不了几岁，
挺瘦弱，已经累的气喘如牛，老张看看老太太的脚疼的没发走路，二
话没说，马上把祖孙二人引进屋里，认真的帮老太太诊治了一番，回
到家都以经七点多了，老张说，那总不能看着人家孩子再背一回吧！
    老张这些年几乎从没有抱怨过什么，老伴生个女儿，他说挺好，
以后可以给娘当小棉袄呀；再生了个儿子，他说挺好，以后还可以传
个香灯，对得起祖宗。他从来没有挣过太多的钱，可是他认为自己钱
总是够花，在他的心中，过日子不需要什么太多的奢侈品，每天能有
一只烧鸡，二两白酒就是幸福生活了！所以老张这些年里每天饭桌上
准保有一只烧鸡，高兴也嘬上两口白酒，日子过的其乐无穷。记的第
一次到女儿家去，他给女婿提了三只烧鸡，把几个外孙女吃的直打饱
嗝。老张从来不在乎钱的问题，他大女婿是在大学里搞研究的，一次
偶尔提起很想买一套牛津大百科全书，得几千块，舍不得。没过几天，
老张就提着一大套百科全书和一只烧鸡给大女婿送去了。女婿满脸感
激，老张吃完烧鸡抹抹嘴，说：“钱这东西，生不带来死不带去，何
苦为它操心累断肠呀？”
    可能是老张爱帮人，所以他的朋友也特别多，想当年很多穷哥们
没了饭吃，老张就算只有一个烧饼也得掰给人家一半。到了后来，老
张也还是热心肠，谁家要是有了困难，他会倾其所有帮人家度过难关。
老伴老是说老张未免太大方，以后老了没个筹划，老张就说老伴净瞎
操心。
    老张教育孩子从来就没费过力气，他家没什么压人的规矩，孩子
们自己的道路自己选择。老张唯一的要求就是希望他们能靠自己的本
事混口饭吃就行。除了做人品格上大的毛病，老张基本不太管孩子，
而他的孩子也都在父亲的熏陶下有了一种简单而快乐的人生态度和一
颗善良自在的心。孩子们结婚以后，老张不象别的老人规定孩子什么
时候要回来，他也从来不打电话叫孩子们回来吃饭，可是不论谁在什
么时候回到家里来，总很快就能吃上可口的饭菜。老张家的孩子，不
管儿子女儿，回家的频率都很高。也许他们觉的，这个家才真的有家
的味儿！
    都说父母全是望子成龙，其实也不是。老张就是个例外，他觉的
要是都成了龙，那还不满天都是龙，和满地上都是人有什么区别？去
年他外孙女要考高中，全家人都希望她考省重点，离家又近升学率又
高，还没考就忙活着为她找门子，铺关系，可是小姑娘自己很有主意，
坚决要考一般的高中，她嫌省重点压力太大，不宜于自己的成长。大
家都不同意，只有老张举双手赞成，他说谁说非得在重点中学才能考
上大学？那得看孩子自己。没办法，老爷子发了话，父母叔姨也就只
好眼睁睁的看着孩子走进了普通高中的大门。不过一年下来，小姑娘
到是争气，在学校成绩名列前茅，乐的老张拿着成绩单合不上嘴！
    老张的日子其实真的是没什么写头，简单的就是一小段拆白人生。
他应对这世界的方法本就简单，又怎么会有那许多故事？可是老张的
生活里却有很多人可望而不可及的快乐和幸福。幸福其实只是一种衡
量标准，关键在于你怎样去看待，人说幸福人生只有一种模式，那也
许就是说幸福的感觉里其实都包含了满足。这个时候我才突然明白这
个修脚工幸福人生的真谛——知足常乐。老张从来没有说过自己是如
何的淡漠超脱，如何的不参与世人的争名夺利，他只是在真实的生活
里实践了怎样做一个快乐的人，怎样心满意足的走过自己的一生。
    活得累了的朋友，不妨停下来看看这个修脚工的幸福人生吧，以
后的日子究竟要怎么过，得看你自己了。
