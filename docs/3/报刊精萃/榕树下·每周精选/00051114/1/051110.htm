民族的血

○山东　雁秋

民族的血
是一首无尽的诗
低沉徘徊
吟咏着千古至今苍凉的岁月

民族的血
是一支永恒的歌
慷慨激昂
吟唱着万里无际壮美的山河

民族的血
是长城万里的烟尘
是关山难越的悲凉
是中原逐鹿的纷争
是乌衣巷口的斜阳

民族的血
映照出幽幽的紫禁
微微的雁荡
蜿蜒万里的长城呀
贯通南北的大运河

民族的血
是神州的脊梁
震撼着江河的横流呀
民族的血
是华夏的精魂
激荡着子孙儿女的心魄

民族的血
曾经  你奔腾激越
一泻千里
挟带着千年的璀璨
故都的繁华

民族的血
曾经  你耻辱忧伤
低吟彷徨
不堪回首——百年国耻
满目凄凉

五十年过去
弹指一挥间
你听——神州大地
进军的战鼓已经敲响
军号声声  旌旗飘扬
民族的血啊
饱含着华夏儿女的激情
充满着亿万人民的气魄
奔腾咆哮  慷慨激越
奏响了那迈向二十一世纪
无尽辉煌的胜利凯歌
