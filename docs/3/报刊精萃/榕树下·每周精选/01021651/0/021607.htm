纪念1320+往事随风

○右眼(上海)
　　　　　　 

   一 
 
春天在2月4日站立起来 
惊动了蛰伏一个冬季的 
所有欲望，站立起来 
 
瞬间，云就进入天空的身体 
偷看的小溪 
也替他们流着刺骨的眼泪 
 
 
   二 
 
而我巨大的石库门 
遥远的石库门 
层层叠叠的石库门 
呻吟微弱却清晰的石库门 
还在八百里外 
毫不知情地 
阻拦着时间退却的路 
 
你点上最后一根烟 
说：1320 
这只是火柴的名字 
但它把黑夜 
给点着了 
 
 
   三 
 
那些暗示分离的风 
在穿过我们支离破碎的身体后 
就再也不会回头张望了 
 
其实这一点 
我的诗歌早就知道 
我的诗歌 
还站在岸边的一朵腊梅花里 
孤零零的 
拥抱着湖水 
 
黄昏暗淡 
黑夜无言 
往事冰凉 
随风逝去
