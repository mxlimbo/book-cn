困惑

○菊开那夜(江苏)
　　　　　　 

我们到底何所烦忧  
涨破了诗篇  
固守昏沉  
面临灵魂的声嘶力竭  
 
谁拂试了明镜  
将禅一次次空灵  
谁拈起了娇蕊  
容颜暧昧  
我们注定要草草收场  
嘴一张就谋杀了思想  
 
仰天长啸  
所有的欲望都席地而坐  
密不透风，密不透风  
罗列着半生的破绽  
 
再不能背信弃义  
换取化险为夷  
 
告诉每一只困惑的眼  
我们除了束手无策  
别无良策
