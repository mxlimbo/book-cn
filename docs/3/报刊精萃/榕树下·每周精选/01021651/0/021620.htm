情人节之悲情篇：昨夜烟花

○英吉沙(其他城市或地区)
　　　　　　 

这婉约而又舒展身姿。是对谁展开？ 
让坠落的花瓣，游离了众人的目光 
多愁善感的妹妹，我是否应闭上眼睛 
才能发觉你所有的真实与美丽 
 
昨夜烟花。这曾经纯粹的光芒 
是一位隔着夜色眺望天空的少女 
在星光或明或暗的诱惑下 
怀揣着梦想，轻易地远离了家乡 
 
从黑暗中分辨光，从光中分辨出火 
走过的路艰辛而又狭长。激越的上升 
总连着疼痛的触觉。这种坚忍的开始 
正是为了璀灿的一瞬 
 
花开花落，谁的掌声里暗含了叹息 
而空中的舞者，依然把古典的神韵 
挥发得格外淋漓。年少的妹妹 
是否从此变得风般轻盈？ 
 
幻灭。那些身影是哪些人武器 
伤者的故事，是北风吹散的又一团烟雾 
 
无限光芒，魂牵魄系的烟花 
恍若从前的容颜，我爱吹口琴的妹妹 
在谁的梦里走远。少了琴声 
我在饮青梅酒时酩酊大醉 
 
我心事重重，所以注定 
黑夜往往都是从晕眩开始 
 
或许，天空就是更好的居所吧 
夜色深沉，一个化名英吉沙的游子 
想起刀锋。是离开，还是留下？ 
我看见半空烟花尽了
