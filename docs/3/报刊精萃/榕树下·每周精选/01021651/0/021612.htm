午夜棒冰

○丁小村(陕西)
　　　　　　 

    几个人从那灯火绚丽处走出来，已是午夜。钟楼上的大理石钟面
泛着蓝荧荧的光，仿佛一个消失了的梦境。大钟的指针在夜深处默默
走动，它发出的声响我们无法听到。就像一颗心在自己的胸膛中跳动，
但是太多的声响淹没了它微弱的步点。
    我们都有些醉意，走在街上脚步踉跄。霓虹亮处，街道两旁的电
线杆在走动。待我们定睛去看时，它们又一个个稳稳站在那里，如同
一排士兵。周东走过去，朝那根电线杆踢一脚。周东被电线杆顶得退
了三步，电线杆却依然站在那儿。周东说，妈的，还挺坚强的。赵南
说，同志们好，同志们辛苦了。那溜电线杆无声无响，赵南说，这些
蠢货，怎么不答话。我说，小子，他们压根儿就没把你放在眼里，钢
筋水泥，坚强的人儿，你算什么，一疙瘩肥肉，几根稀松的骨头。赵
南要过来跟我比试比试，我连忙退后三步，赵南是个胖子，我怕被那
堆肥肉压着。陈北在街中心做出一个花样滑冰动作，开始唱一只歌，
他吐字不清，唱来唱去，只能听清楚两句：

　　都给我立正站好 
　　听你爸爸唱歌 
 
　　我们听着觉得带劲，都跟着陈北唱： 
 
　　都给我立正站好 
　　听你爸爸唱歌 
 
    这时有一辆大卡车过来，轰隆隆地像一辆坦克开过来。车灯打得
很亮，晃得我们眼前一片暗黑。我们本能地往街边躲闪。只有陈北唱
得正得意，根本不理会那辆开过来的坦克。这辆大卡车只好熄了灯，
在我们面前停下来。陈北说，灯光师，灯光师，人都到哪儿去了。卡
车司机打开车窗，探出头来，但没有说话。我们知道是个外地人，我
们这座城市紧靠着国道，白天汽车得改道绕过街市，夜里过路的司机
就直走街上插过去。我们本不想去惹这个司机，但陈北这会儿正来劲
呢，我们可不想扫他的兴。我和周东赵南一起喊，开灯开灯。司机像
是糊涂了，没有反应，只是把头探出出车窗看热闹。我们又喊，灯光，
灯光。司机打开了灯。车灯像是一把板刀，雪亮锋利，陈北在灯光中
显得渺小。灯光给他了刺激，他在这把板刀上跳舞，像舞台上的一个
跳着踢踏舞的小丑。我们不再去理会司机和他的大车，我们都跳进灯
光中，像一群踢踏舞明星，我们一边跳一边唱：

　　一只耳光　打在我的脸上 
　　嘿　舒服　漂亮 
　　一把刀子　砍在我的心上 
　　嘿　舒服　漂亮 
　　一只蚊子　它趴在我的脸上 
　　嘿　痒痒　痒痒 
 
    我们跳得满头大汗，浑身酥软。我们走到街边，靠在梧桐树干上
喘气。汽车灯光一闪，轰隆隆像一辆坦克冲过去，它的尾灯在屁股上
微弱地亮着，一闪一闪的，像一个放得不痛快的屁，余音袅袅，断断
续续。

    这是一个冬天的晚上。在我们这座不南不北的城市里，冬天并没
有你想象的冷。我们几个人常常聚在一块儿玩。周东开了一家小小的
烧烤屋，我们常常到他那儿去吃烤肉喝啤酒。赵南在中心市场有一间
小小的时装屋，我们有时到那儿去看他们新进的衣服。那时装屋是赵
南跟他妹妹开的，他妹妹很凶，最讨厌我们这伙人去，我们为赵南感
到难过，仿佛赵南不是老板，而是他妹妹的打工仔。可赵南对他那个
凶恶的妹妹一点脾气也没有。陈北在电力局做电工，我们常常跟他一
块儿去混酒喝，他们常常为安一只电表对那些可怜的进城开店的农民
大宰特宰。我们乐得跟他一块儿上酒馆、去舞厅。我跟他们混到一块
儿纯属偶然，那年我中专毕业，上的税务学校，好不容易进了城关税
务所。有个中学同学给我介绍了周东，他那会儿在街上开了一家小店，
生意很差，我的同学让我把周东的税定低点儿。我给周东定了很低的
税，就这样跟他成了哥们，也认识了他的两个哥们赵南和陈北。这中
间只有我和陈北可以请他们吃不掏钱的饭、喝不掏钱的酒。他们有时
也请我们玩，赵南有次要把他的妹妹介绍给我作我的女朋友，可当场
被周东和赵南否定了。他们说，赵南别提你妹妹了，像个母老虎。这
使赵南很生气，很伤心。他说，我妹妹其实挺好的，我那店全凭她呢，
你们看我操过多少心？我觉得周东陈北有些不留情面，安慰赵南说，
做生意的，不凶点儿，行吗？你妹妹是个不错的女孩子，赵南你说她
看得上我吗？
    周东陈北见我这样，就说，那赵南你啥时把你妹妹带来让我们考
察考察。赵南说，那可不容易。周东陈北说，那赵南你放这个不荤不
素的屁干什么。赵南说，啥时我把她骗来，不过李西你自己要好好表
现表现。她可是最讨厌我们了。周东陈北说，不就是讨厌我们两个吗，
对李西可就不一样了。赵南说，那是。瞧你们两个，都是城市混混，
人家李西，好歹也多读过几天书，正儿八经的国家干部。周东陈北都
说，你赵南算什么，跟我们也差不多吧。
    赵南下了大工夫，想把他妹妹骗来跟我们聊聊。结果他妹妹一听
是我们，根本就不理会他，说什么也不来。这让赵南很失面子。周东
陈北也把赵南好一通奚落，弄得赵南觉得对不住我。要请我喝酒。对
不住我也就是对不住周东陈北，当然得一块儿喝了。赵南只好狠出了
一回血。喝酒时我们都有些沮丧，仿佛被谁抛弃了。赵南把他妹妹臭
了一通，我知道他是要安慰我，他说，我妹妹真是个嫁不出去的母老
虎，周东陈北你们说得不错，她配不上李西。周东陈北一听赵南这么
臭自己的妹妹，也不好意思起来，说，小姑娘嘛，当然有些架子，这
叫浪漫，你懂吗赵南。我说，是，是，赵南，我倒是越来越觉得你妹
妹不错。赵南听我们夸他妹妹，也高兴起来，跟我们每人碰了三大杯
啤酒。
    我们被冰凉的啤酒灌得透心凉，又被火暴热烈的四川火锅烧得满
身大汗。酒精在我们的身上慢慢游动起来，把我们的那点沮丧慢慢冲
淡了。后来我们对着那辆远去的汽车狂呼乱叫了一通，靠在街边的梧
桐树干上把我们胸膛里的酒气一口口往城市的夜空里倾倒。

    酒精慢慢地在我们的胸膛里升起来，变成一股淡淡的烟气，然后
我们感觉到那股烟气在我们的胸膛里燃烧起来，它的火焰是淡蓝色的，
像是煤气的火苗。这股淡蓝色的火苗舔着我们的周身的血管，我们觉
得身上温暖，嗓子发干。我们靠着梧桐树干，把那淡蓝色的火苗吐出
来，把夜空中冰凉的小水珠吞进喉咙。
　　真舒服啊！赵南说。
　　舒服，漂亮，周东陈北想起了我们刚才唱的歌。 
    我突然想起了一个问题，对他们三个说，你们说，你有没有最舒
服的时候？是什么时候？
    舒服这个词儿，大概是我们用得最多的一个词儿了，可是我们最
舒服的时候是什么时候呢？
　　他们觉得我提的这个问题有趣，都说，让我想想，你先说吧。
    我说，我小时候在农村，那会儿我们那儿不通公路，赶集都得走
路。有一次我爸爸挑着一担粮食去卖，我跟爸爸一块儿去。到了粮站，
我走得腿都硬了，一屁股坐在粮站的水泥院子里，腿一下子就软得像
根拿不起来的面条。那是个响晴的天，我觉得自己满身是汗，脸上像
是被一团火烧着燎着。我就想喝口冰凉的水，这时有个卖棒冰的到了
院子里，他敲着装棒冰的木箱子喊——棒冰，棒冰——我看着那箱子，
听着那喊声，觉得身上一下子凉爽起来，我转过头去找我爸爸，他到
粮仓前排队去了。我跑过去，觉得卖棒冰的像是追着我在喊叫。我跑
到爸爸身边，那里排着一长队等着验粮的人。我叫了声我爸爸，爸爸
转过头来看我时，我却把头拧到卖棒冰那边去了。我爸爸却又把头转
了回去。我又回头喊了声。这回我说，爸爸，我要棒冰。爸爸没理我。
我又叫了声。爸爸说，等一会吧。我就等着，那个队排的特别长，我
爸爸在最后。我就一直坐在屋檐下，望着那卖棒冰的，等着轮到我爸
爸卖粮。我坐在房檐底下，腿是软的，嗓子里冒着烟，眼睛有些生涩。
等到我醒来时，我爸爸已经卖了粮，取了钱。我被爸爸叫醒，第一个
动作，就是转过头去找卖棒冰的。可是卖棒冰的已经不再这里了。我
哇的一声哭了。我爸爸拉起我，我们小跑着去找卖棒冰的人。我们在
乡街上追到了他，他说，卖完了。我又放声大哭。我爸爸上去揭开那
木箱子，那里边有一只棒冰，已经化了一半的棒冰。我爸爸说，这不
是还有吗？卖棒冰的说，这根已经化了。我爸爸说，就卖这个。我拿
着那半根棒冰一下子含进嘴里。我觉得这凉凉的甜甜的棒冰是世界上
最让人舒服的东西。
    我说完了，他们都瞪着眼睛看着我，一个个大张着嘴，仿佛在等
着一只美丽的棒冰喂进去。
    周东说，没错，棒冰就是最让人舒服的东西。我上学时最不讨老
师喜欢。有一次，大中午，太阳很大，我在家里午睡，到学校里去迟
了，老师见我站在教室门外边，说，迟到了就别进来了，站外边去。
老师说的外边，就是房檐外边，那里太阳正照着。我只好又站回到太
阳底下。我站了好久，头都被太阳晒木了。这时围墙外边有卖棒冰的
在叫，棒冰，棒冰。我的心一下子飞到那木箱子里去了，我那会儿觉
得世界上再没有比棒冰更美好的东西了。好像过了很久才下课，老师
刚走，我就一溜烟跑出学校，买了一根棒冰。那会儿我觉得吃棒冰是
最舒服的事。
    说起棒冰，我也觉得它是世界上最好的东西。赵南说，从前我妈
妈就是卖棒冰的，她每天一清早去领棒冰。装满一大箱子，就架在自
行车后座上，满街推着卖。我放学时妈妈就在学校门外边卖。天气很
热，我看着别人吃棒冰，就想着问妈妈要棒冰。可是妈妈卖一只棒冰
才赚半分钱，我吃一只棒冰我妈妈得卖十几个才能赚回来。我妹妹小，
妈妈下午回家时会给她带一根化了一半的棒冰。我问她要，她有时会
给我吮两口。我妹妹为什么对我这么凶，就是因为她有棒冰啊。有时
妈妈卖得好卖得多，就会给我棒冰，我觉得那棒冰吃起来是最舒服的。

    陈北说，我现在就觉得世界上最舒服的事就是吃棒冰，我现在就
想吃棒冰。
　　是啊，这会儿我们多么需要棒冰。棒冰，世界上最让人舒服东西。
　　赵南说，大冬天，哪来的棒冰 。 
    再说了，现在哪有棒冰啊，说不定现在的小孩压根就不知道棒冰
是什么玩意儿。周东也说。
　　说不定会有的，陈北说。
    说不定也会有人像我们一样，会在一个醉酒的冬夜想起棒冰。我
这样想。
　　我现在真想吃根棒冰啊！周东说，可惜，现在没有。
　　我们为什么不去找找呢？我说。 
　　走，找棒冰去，我们现在有钱，可以卖一箱棒冰。 
 
　　拐角处有一家商店开着门，我们放开步子冲过去。 
　　店里有个女人，坐在椅子里看电视。 
　　我们一眼看到靠墙放着的冰柜。 
　　棒冰，棒冰。我们一起喊。 
    女人看到我们，立刻站起来，你们，干什么？她有些惊慌失措的
样子。
　　我们卖棒冰，我对她说。
　　女人眼睛里露出怀疑的，问，什么棒冰？没有啊。 
　　我们立刻都泻了气。没有，没有棒冰。我们转身往门外走。 
　　女人在我们身后对里屋出来的男人说，是一群醉鬼。 
　　周东说，妈的，这个混帐女人，竟然不知道什么叫棒冰。 
　　我们再去找去吧。 
 
    又一家店里，我们进去，店里的老太婆问我们，小伙子，你们买
烟吗？
　　我们说，不是，我们要棒冰。
　　老太婆说，没有棒冰，有冰淇淋。 
　　我们说，我们不要冰淇淋，我们要棒冰。 
    老太婆说，棒冰，棒冰，我都好多年没见到过棒冰了；我这里没
有棒冰，你们嫌贵的话，这里有狗头，有些化了，只要三毛钱一个。
　　要不是个老太婆的话，周东早就跳起来了。
    老太婆要我们买本地产的狗头雪糕，她当我们没钱。我们都笑了，
说，老太婆，我们不要那狗头，我们要棒冰，我们给你一块钱一只。
    老太婆说，我记得从前棒冰就五分钱一个，哪值一块钱啊；我这
里没有棒冰，有的话，我还是只要你们五分钱一个。

    我们走到另一家店，里边有三个男人坐在火炉边喝酒。我们想不
管有没有棒冰我们都得去问一下，没有的话，我们再也不找棒冰了。
我们找不到棒冰，我们在冬夜的街上已经转够了。
　　我们走进店里说，这里有没有棒冰。
　　一个正把酒杯举到嘴边的男人说，什么棒冰？没有。 
　　我们说，真的没有？ 
　　那些喝酒的男人有些不耐烦，说，没有就没有。 
　　周东说，真是奇怪，没有人知道棒冰。 
    赵南和陈北说，这世界上就没有人喜欢棒冰，没有人记得棒冰，
难道他们从前没吃过棒冰。
    喝酒的几个男人都放下酒杯，他们个个块头粗壮，看来他们嫌我
们多话了，那个最先跟我们说话的男人可能是店主，很是不高兴地说，
不买东西就算了，这里没有棒冰。
　　另一个男人说，哥们，喝多了就回去睡觉。
　　周东说，还没够呢，吃了棒冰再喝。 
    最后一个男人最粗壮，偏过头来看我们，说，我说哥儿几个有啥
事儿？
　　周东说，没事没事，就是想买棒冰。
　　那个男人说，我操，我看你们来事是不是？ 
　　我们几个都摆了阵势，我们这会儿心情正好着呢。 
　　那个男人说，我是刘小七，哥儿几个没事就别惹事。 
    刘小七我们都知道，我们并不想惹事，只是我们不能一听刘小七
就退。我们不再多说什么。
　　倒是那个店主不那么硬气，不跟我们吵，却去打电话。
　　我们怕他打电话叫人，我们可不想吃这亏，想要说点儿硬话开溜。 
    没想到他打的是110 ，警察的车仿佛一直跟着我们，没等我们撤
退，几个警察就下了车，从后面把我们包抄起来。
　　警察看到刘小七，再看我们几个，倒是有些糊涂。
    最后是那个店主，他好像认识警察，说，是这个兄弟多喝了点，
要跟小七来事。
　　警察不由分说把我们弄上车。
    周东他们没事人似的，反正也没什么大不了的。我可就觉得有些
晦气，好赖我也是个国家干部。弄到派出所去多难看。

    我们一下车，就有几个警察奔过来。其中一个冲我喊，李西，怎
么是你，这是咋回事儿？
    我一看，原来是我的中学同学沙锅。我说，我们买棒冰，买出麻
烦了。
　　沙锅跑前跑后，总算让我们没有了麻烦。
　　沙锅说，哥儿几个，到我屋里坐坐。 
　　幸亏是沙锅今晚值班，我心里想。 
    沙锅说，你们说棒冰，我也想起了棒冰。棒冰真是好东西。上中
学时，我们在一个周末到一个同学家去玩，因为是在山里，我们走路
去。我们走了好几座山，一个个又饥又渴，到处找有没有解饥解渴的
东西。可是除了满山的松树，什么也没有。我们就一气奔下山脚，那
儿有一条河，我们在河里灌了一肚子水，灌得肚子里呱呱响。可是这
解不了饿。我们就朝河对面望，那儿有一条公路。公路上什么也没有，
除了轰轰开过去的汽车和偶儿一两个骑车人。看到骑车的人，我突然
想到，我们可以去等卖棒冰的。可是我们三个人身上总共只有八分钱。
连两根棒冰也买不到。那就买一根吧，我说。于是我们脱了鞋，挽起
裤腿过河，上坡，最后我们终于爬上马路。我们坐在马路边等卖棒冰
的人。我们看着向西斜的太阳，心里念叨着那美丽的棒冰。后来终于
来了一个卖棒冰的，他本不想下车，可我们挡住了他的车子，他以为
我们要给他寻事呢。我们说，买棒冰。他说，几个？我们说，一个。
他怀疑地看着我们，觉得不可思议，说，就一个？我们说，四分钱你
卖不卖？他说，不卖。我们说，那就只买一个。他打开箱子说，有几
个化了些的，一毛钱卖给你们三个。我们说，那就买化了些的，买两
个，给你八分钱。他收了我们的八分钱，给了我们两只棒冰。我看着
他的箱子，看着用棉布盖着的棒冰，那会儿真想做个抢劫犯。后来我
们三个人一边走路，一边轮流着吮那两根棒冰，我们只想那棒冰永远
也吸不完，那真是世界上最好的东西啊。
    我们抽着沙锅发给我们的烟，觉得那烟甜丝丝的。原来不止我们，
还有人记着棒冰的。

    从沙锅那里走出来，我们一路说着棒冰，说着我们忘记了的棒冰。
可是现在到哪里去找棒冰呢？在这个城市，在这个冬夜，棒冰离我们
有多远啊。
    最后我们聚在赵南屋里，我们几个人说着棒冰。起来给我们开门
的赵南的妈妈说，你们说什么啊？
　　我们说，阿姨，我们想吃棒冰。
　　阿姨说，现在哪有棒冰啊，赵南你还记得棒冰吗？ 
　　赵南说，是李西提起的，我想起了小时侯你每天卖棒冰的事。 
　　阿姨说，现在谁还吃棒冰呢，现在没有棒冰了。 
　　我们说，可我们就想吃棒冰。
