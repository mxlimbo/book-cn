关于诗歌

○玄武玄武(山西)
　　　　　　 

    夜很静，比死亡还安静，比腐烂时肉体内部发生的剧变还强烈；
惟有在这样的夜晚，我才会偶尔想到诗歌。我写诗多年，越来越尴尬，
在我与现实的关系中，诗只是不断给我增添羞辱。如果我把自己比做
混球，那诗就是里面的一根骨头。有时候又想，诗像一个风情万种的
女人，而我呢，可能是一个色鬼。她媚笑着向我招手：过来呀过来嘛
这儿有个陷阱，你自己掉下去。我就两眼直勾勾地往前跑和往下掉。
你看我好贱。
    大家都明白，现在诗的地位，已从原来的淑女，变成歌厅整顿以
后的小姐，惶惶不可终日，不知道以后要怎样。一些纯文学刊物不再
发诗；报纸副刊大多彻底砍掉了诗栏。《诗歌报》已经停刊，国内仅
有的几家诗刊，又暮气逼人，成为满足老年诗人虚妄的成就感的场所。
诗人和自命诗人的家伙，个个丧魂落魄，因为多年呕心沥血做的事，
简直是在今天搭上弓箭射日头，没了一点用，还让人觉得好笑。一些
成名诗人茫茫然，担心将来文学史，会很轻松地抹去他们的名字，哪
怕是一百年的文学史。这的确残忍，但又有什么办法呢。我自己经常
陷入自卑，一来写不出好诗，二来写诗很傻。第二种自卑往往能抵消
第一种，因为即便写出好诗的人，也和我一样傻一样无聊。如此我就
有了安全感，幸灾乐祸的安全感。这心态很好玩，我估计其他人也略
有。
    那么谁来看诗呢。印象里离现在最近的两次年青人大规模读诗，
一次是汪国真，另一次是海子。两件事到最后都搞得很滥很荒唐。现
在年青人日衷于理智地玩感情，以及电脑游戏，他们最远的梦想，可
能也就只能到达寻找网上情人。四十五岁以上的人似乎不读现代诗，
包括中年作家。知识结构和自身文化观念，致使他们视现代诗为怪物，
说白了吧，就是读不懂。在八十年代，不是连老诗人都因看不懂朦胧
诗而发火么。政府官员有一些是文化盲，那些贪官找小姐都不一定有
档次。人们对这个世界产生的技术性变化穷于应付，谁顾得上看诗这
样感性的物事呀。写诗的人彼此阅读对方作品的也少，不少诗人转行
写小说，同样没人看。有一次看某文，说作家搞剧本是小妾心态，我
读着读着噗嗤一笑：搞剧本是小妾，那做小说，顶多就是个大老婆，
还有可能是前妻。说起来诗大概只有小朋友读了，也读不懂，当儿歌
背呀背，那是唐诗不是现代诗，床前明月光之类。
    我曾认为，诗是时间性的东西，就是要把一个人的青春转换为诗
行。如果这人的思维、精神和心态，维持不住青春的状态，这人的诗
歌生命也就玩完，事实上大多人青春期还没过就已经不行了。中国人
老得快呐。现在看来，要做到那些真的很难，尤其在这个时代，简直
不可能。这时代似乎没有青春，冷静理智地仿佛一加一等于二。它那
么精确，不允许梦想和模糊，连布罗茨基说的小于一那类的东西，都
不允许存在。
    这般妄下结论，一定会有人讥笑我像个弃妇，被时代抛弃或哀怨
生不逢时。所以接下来，我想做一些严肃的理论探讨。实际上以前，
我一直不敢想诗的前途，只是盲目地做下去；而现在，时间和经历给
了我力量，让我能够勉强正视内心最脆弱敏感的东西，比如诗歌这样
的软体动物，那隐秘的激情，原始冲动般的欲望。在我的电脑保护屏
上，写着一行字：我是谁。我要做什么。在电脑旁发呆太久，这话就
一晃一晃出来拷问我，刚才就是如此。我是谁这问题，我一直在诗歌
里寻找答案；而我要做什么，则是我对诗歌这种找寻手段的质疑。今
夜我在想，从胡适《尝试集》起，中国新诗发展近百年，收获可能并
不大，且多舛多难。白话诗的出现，始发韧者当记一等功；郭沫若的
《女神》一时间口气极大，很饱满，记二等功；徐志摩、戴望舒、朱
湘他们在诗艺上有想法，也记一功；接下来是李金发，很前卫敢牺牲，
同样记一功。但是总的说来，诗歌被毁得利害。先是国家大乱，丧权
辱国事儿不断，诗自身价值，不得不让位于民族主义式群众运动式的
爱国基调；然后拿诗歌来抗战，急功近利却莫能奈何；紧跟着建国，
又用诗歌来搞政治革命，来建设社会主义。这当儿诗歌始终是个工具，
充当投石器，传声筒，或者助火势的风箱。我觉得这些玩意，和真正
的诗、大诗和小诗，都距离遥远。中国新诗像个没长成的小孩儿，不
断让他干重活，结果呢，把孩子累着啦，再不能长大。
    有人会说，中国民众文化素质差，读不懂诗，尤其现代诗的那种
艰涩语言。这问题当然存在。但我认为，这可能是一个小问题。真正
的诗歌，应该是人性的东西，自由的东西，是人文精神，是对民族心
智的重建。一个真正诗人在文学上的成就，绝不仅仅是几首诗的诗艺，
还应当包括他个人的文学活动和社会活动对世界的推动。像梭罗，敢
于向美国政府提出《公民的不服从权力》，在约翰。布朗被杀害时向
政府愤怒地示威抗议，他并不写诗，但我认为他是诗人，或者像惠特
曼也不赖，是民族诗人，不仅在诗歌里喊出了时代的精神，还指明了
未来的美国，其方向和发展目标。这一点，我觉得他比任何一任美国
总统做得都多都好。而像朦胧诗的兴起和红极一时，我以为，最可贵
的一点也是真正人的东西凸现了出来，人脱离了人群，脱离了长期灾
难式的集体和农庄，非常新鲜地表现出个人化的思考和个人的欲望。
就此而言，有可能是所谓的伤痕文学望尘莫及的。至于技巧那都扯蛋。
只可惜在中国新诗中，这种东西太少太少了，小于一。诗歌式微，这
难道不是重要原因之一么。
    现代诗还有一个致命的硬伤，就是语言。我一直痛恨白话文和简
化字，认为它们是对语言的伤害。白话文那僵硬的句式语法，实质上
极理智的，更适宜逻辑分析更像手术刀，但与中文这种感性语言本质
上相悖。这种语法，对欧美线形文字，可能较合适，富弹性和节奏感，
有轻有重，联贯起来惝恍迷离像音乐；对我们古老的象形文字，则让
人难受，因为有那么多的废词儿，连词助词，夸张的形容词，虚假的
副词，假眉三道地排列在文章里。然后一个字一个字僵尸跳跃般蹦出
来，没有古文摇头晃脑平平仄仄地吟出来那样有韵味。句子总是很长，
要吟的话可能会出不上气。而且它离我们日常运用的语言有相当距离，
有可能还需要改革。想想看吧，一句文言诗如：大漠孤烟直。长河落
日圆。译成白话成了什么样子？还叫诗吗屁都不是。文言诗里那种意
象与意象之间的神秘关系，在白话里必须在语法中绕啊绕半天，最后
啥都没了。就程度而言，文言诗译成白话诗，和中文诗译成英文可能
半斤八两。谁都承认诗是语言的艺术，但我们能从文言诗里继承什么
语言风格呢，这断裂的语言实在可怕，只能学翻译体诗，结果就成了
现在诗坛上的绝大多数二半吊子诗体。
    一句话，文言诗是饱满的、感性的、物我交融的、意味深长的，
白话诗是干瘪的、推理的、个人中心的、晦涩难懂的。这些东西，在
目前的语言问题下解决不了。我自己不行，能行的我没发现，曾有一
个解决了一半，人就给死掉了。读文学史时，我对俄罗斯文学中一段
记忆特深，说俄罗斯文学肇始时，有一个诗人兼语言学家，他花费一
生时间精力，对俄罗斯语言进行清理整顿，从词汇到语法，使其变得
纯净优美。他选择自己作为牺牲，接下来才有了普希金。我曾经想向
他好好学习，可惜个人能力有限。后来又异想天开，试图采用方言口
语入诗，为此还采集民歌，俚语，又改编了一些民谣和传说。不过那
时候我的诗歌理想，是做雪莱而不是区区小彭斯，于是很快意志动摇
做别的。关于诗歌的语言问题，我还向几个成名诗人提及，他们不大
以为然。其实以为然又能如何呢，谁也做不到。现在的我已经不太关
心这问题，因为我不再写诗。难题留给我儿子那一代人去罢。
