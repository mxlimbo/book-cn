冰灯

○北方以北(其他城市或地区) 
　　　　　　 

春天已经走到楼下  
可我还是没躲过这场寒流  
我被掳劫  成为北方一座冰灯  
冰做的城堡囚禁了我温暖之心  
 
我没能躲过  
这场灭顶之灾  
城市积雪的路障已经排除  
听说南方人们都只穿衬衣  
而我清点昨夜残留的星光  
用流泪成冰不再清亮的眼睛  
 
梦中游来一只深海鱿鱼  
它的皮肤和我一样冰冷  
我拥抱着它孤独的触须  
它缠绕着我不眠的恐惧  
 
我的恐惧  是因为  
我忘记了我是冰 还是灯  
 
我和世界一眼望穿  
可疑的是我有没有思念阳光  
我们在深夜安息  成为情人  
又在每个清晨  彼此逃遁  
 
我并没有背弃季节  
树还在积聚  花在沉睡  
我是水凝固的品质  
陪伴着北方最后一次霜冻  
我和阳光遥遥相对  
 
我们遥望   
是因为我是冰  也是灯  
我是春天  最冷的泪痕
