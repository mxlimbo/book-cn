想我

○花儿开了(河南)
　　　　　　 

    我可以请求你今夜想我好么？
    不奢求你一生一世想我，不强求你来生前世想我；不乞求你错过
一个世纪想我，不征求你分分秒秒时时刻刻想我；不妄求你攫取生命
中的美丽想我，不力求你今夜枕着我名字入睡想我；不恳求你不需求
你想我。
    只要你一个夜晚想我好么？
    想我时你抛弃所有的杂念欲望痕迹灰尘，想我时你摈弃所有的意
识思想精神；想我时你毁弃所有的记忆往事相思，想我时你丢弃所有
身边的亲人朋友；想我时你放弃所有喧嚣沉渣猖狂放肆，想我时你废
弃所有不属于想念冲突；想我时你舍弃漠漠然冥顽不化的寂寞。
    去到女娲补天的洪炉中想我好么？我是那受日月精华一块会猛然
爆裂的玑石，在你想我的时间里，跃出一番欣忭欢悦的生命；去到生
命之花粲然开放的瞬间想我好么？我是那整个春山开放笑靥最灿烂的
郁金香，在你思念我的时间里，如醉如痴眷变难舍为真情感动的颤抖
盛开；去到杨柳岸晓风残月的湖面上想我好么？我是那清风拂水不黏
不滞吹皱一面湖水，在你怀念我的时间里，执手相看泪眼吹起断肠人
黯然离别的发梢；去到一支柔靡无骨哀艳缠绵曲子中想我好么？我是
那低徊婉转萦绕不去纠缠不已旋律，在你痴念我的时间里，四肢百骸
皮肤毛发都沸腾着欢乐海洋上的月光；去到一幅素淡国画中想我好么？
我是那流溢黑白中无法驱散历久弥鲜心感方知的境界，在你挂念我的
时间里，自然明净澄清无语痴痴呆呆笑看红尘；去到蝴蝶飞舞的花丛
中想我好么？我是那曼妙无比而优雅地轻柔飞舞的精灵，在你眷念我
的时间里，穿越时空横渡生命簌簌地拍动翅膀擦过溪水舞出淡淡的风
和影。
    在熙熙攘攘人来人往的街头想我，那里有我的浮躁和喧哗；去到
一杯苦涩咖啡中想我，那里有我的叹息和回味；在一些美丽的文字中
想我，那里有我的骨骼和元素；去到江河湖湖泊高山平原上想我，那
里有我的挚真淳朴和憨厚纯醇，在沙漠丛林中想我，那里有我的冷酷
和热烈；去到一瓶烈酒中想我，那里有我的哀恸和悲怆；在你对镜自
盼自艾自怨青春不在中想我，那里有我的关怀和殷勤；去到天下所有
事物中想我，那里有我变化成事物事物成我的天机。只要你想我，我
是无处不在，无处不有。
    只要你想我好么？
    想我的一举一动举手投足，想我的狂喜收敛猖狂放肆；想我的浪
漫潇洒风流倜傥温柔多情，想我的玩世不恭狂放不羁桀骜不驯；想我
的天真可爱童心未泯，想我的饱经沧桑物是人非；想我的含蓄蕴藉温
尔文雅，想我的下流无耻卑鄙阴险；想我的真情自然美中不足，想我
的阴谋诡计不怀好意；想我的莫名其妙心不在焉，想我的一切好么？
    只要你想我，就你一个人。
    想我好么？
    想我因为我们相遇相知相识相惬。我既与君相遇，金凤玉露一相
逢，便胜人间无数；我既与君相知，长命无绝衰，山无陵，江水为竭，
冬雷震震，夏雨雪，天地合，方敢不相思；我既与君相识，剪不断，
理还乱，衣带渐宽终不悔，为伊消得人憔悴；我既与君相惬，便要发
尽千般愿，要休且待青山烂，水面上秤錘浮，直待黄河彻底枯。白日
参辰现，北斗回南面，休即未能休，且待三更日头见。
    想我，因为我在想你。
