豪宅之间

○北京　达尔



　　这里是一片别墅区，红墙红瓦，有一个好听的名字——玫瑰花园。
　　初来入住，就去拜访小区物业管理经理。他很有派，坐在大板台
后面，动也不动，寒喧几句，见他的神态，就坐不住，诚惶诚恐间，
听他漫不经心的介绍，这里住着大导演张艺谋，大书法家庞中华……
还有些什么人物，就不知道了，但心里却颇为惊喜，没想到竟会与这
些名流为邻，也不枉在北京的这段时光了！
　　但这份惊喜稍纵即逝，同经理的神态一样，看门人对我们更加冷
若冰霜，我们的车来了，用于挡门的那根栏杆是懒得抬的，须得下车
点头哈腰一番，套上几句近乎，方才面无表情的将栏杆把手一压，那
栏杆就徐徐升起，不高不低地朝天撅着，很是威严，我们就从这威严
下面缓缓将车驶入，进门的那一瞬，还谦恭地向他招招手，然后悄声
骂一句：他妈的。我想咱也用上这国骂了，不过远不如他们骂的标准，
北京人的国骂可是运用的炉火纯青，其频率之高让人叹为观止。
　　我们租了三套别墅，东家是我们的总公司。小区内房子不多，大
大小小各式建筑不过二十多座。玫瑰花园里果真有玫瑰，还有草地，
还有冬青，小区的绿化工称之为黄杨，更多的是柳树，每当春日烟花
时节，四下里柳絮飘飘，“似花还似非花”。
　　入住没多久，楼西那块空地就开始起宅子，空地不大，甚至有点
南北狭长，也不知道是怎么个盖法。从夏天开始，直到秋天，楼前的
通道上堆满了建筑材料，那楼也在节节升高，等盖到二层，已渐见楼
的雏形，但猜不透怎么结顶。也不知房的主人是谁，偶尔见一辆红旗
V6型轿车开来，下来几个男男女女围着楼转转，指指点点。于是猜测，
不像老板，倒像明星。
　　说话间楼就结顶了，三层，因地方造势，虽说南北长点，但颇别
致，每层都有阳台，尤其是三楼，缩进去一个大大的阳台，尖尖的屋
顶，开着采光的天窗，看来如梦似幻。
　　接下来就是装修，主人开始来得频繁，工作之余望去，两个女的
穿得都很漂亮、高档，有时还来一个老年男人，一头的白发。看着看
着，那一家人就走到了我们的房内，且边进边喊老乡。一叙，果然是
老乡，山东的，再一叙，寿光的，与我竟然是邻县。女的与我年龄相
仿，一双大大的眼睛，又黑又亮，一看就是当明星的材料。这不是苏
红吗？果然是苏红，大家都兴奋起来，我于是想起十几年前的那首颇
为流行的歌——《小小的我》，真好听，那旋律顿时在我耳边响起，
只是小小的“我”已经不再小了！
　　那老人是苏红的爸爸，一个老军人，说起故乡来就滔滔不绝，我
们也愿意听，时不时还给他穿插上几句，多是家乡的变化，当然，最
大的变化还是家乡的蔬菜，绿色通道直达北京，每天都可以吃到新鲜
的。面对苏红却不大敢看她，甚而有些拘谨，可能她太漂亮，是大明
星吧！但她却是那么自然地跟我们说话，末了还邀请我们说，等房子
装修好了，请你们到我家作客，咱们开个音乐会，我弹钢琴，你们唱。
　　她的话让我们兴奋了好多天，此后不久，我又在电视里见到她了，
是MTV ，歌名《老朋友》。装修期间，她来得很少，他老公也来得很
少，常来的是他的爸爸和妹妹，中午就在我们的客房里休息，偶尔还
吃一顿我们食堂的饭菜，我们感到很开心。
　　小区的管理者对我们很不友好，夏天来到，刚刚调试好的空调一
开就断电，于是判定他们又在搞鬼。去配电室查看，开关都自动跳了，
合上后回来一试，三座楼的空调电不约而同地又断了，无奈之下，叫
上公司的许副总经理去看，到底是电气专家，一下就发现了门道，他
断定，开关上面没灰尘，说明是新的，30A ，根本容纳不了这些空调，
前一阵测试没问题，证明这些开关刚刚被人换过，旧开关肯定是60A 
的。心里有了底就去找电工，也就是那位物业经理。虽知是他给换了
开关，却不好戳穿，只是让他加几个千瓦的容量，但他却不为所动，
反而说我们电用得多了，别的住户就会受影响。许总倒真沉得住气，
一脸严肃，派头十足，反复强调咱们通融一下，给加几个千瓦，大有
一番打持久战的样子。最终电工妥协了，说可以换成60A 的开关，但
须由我们自己买，自己装。
　　出了门，许总说，怎么样？难为了你一通，还得让你感谢他，我
们这是住得谁的房子？怎么这么不理直气壮？倒像是他的，真是他妈
的！
　　这种他妈的事还真不少，一夜正准备入睡，忽听咚咚的砸门声，
开门一看，是两个门卫，气势汹汹地说，你们的车压了住户的黄杨，
住户告你们了，明天赔吧！
　　一夜没睡好，大清早就去看东邻的黄杨，果真有三株匍匐在地，
小心翼翼去扶，竟发现黄杨没断，顺了顺，又站了起来，高兴之余，
不禁对东邻老头很有点蔑视，正蔑视着，老头出门溜狗，与我见个正
着，跟他招呼道，对我们来客的车压了您家的黄杨深表歉意，我买盆
花赔您吧。
　　小区外边就是花市，不知怎么的，我选了一盆大大的、枝叶鲜翠
欲滴、正开着花的火鹤赔他，价格自然是不菲的，东邻老头分明想不
到我会赔他这么好的花，脸上的颜色就有点挂不住，而这正是我所需
要的。这么几株冬青，搞得纷纷扬扬，其实我们的冬青受害最大，有
两座楼道路拐弯处颇窄，那些冬青可怜巴巴，常常处在车轮子的威胁
与碾压之下，竟没有一个同情的，楼前的草与花也长得比邻人的差，
这交的物业管理费咋用不到我们的头上？
　　时常见到庞中华在散步，他那漂亮的硬笔书法一行一行印在我的
脑间，无论是楷、隶、行，那美丽的线条如水一般流淌着，把硬笔书
法一步步推向极致。坦然地讲，我并没有练过他的书法，在这之前，
我更沉醉于古人，像柳公权、颜真卿，但是我喜欢，在我原来工作的
那个单位，有那么些的人在练中华体，我喜欢他行书的那一撇、特别
是那一捺，那么舒展的一挑，把满腔的心情都放开了。让我们做得更
好，这菲利普的广告说得真好，那汉字是美丽的中华体啊！但我们却
从未同庞中华说过话，想来想去是我们在某种距离上离得太远，或是
现在已经过了练书法的年龄，或是计算机这时代的精灵，已经让我们
生疏了钢笔和墨水，在它的硬盘里，可以储存那么多的字体，只要你
喜欢，轻揿一下鼠标，便给你出来了。但我依旧景仰庞中华，由于我
们相邻，有那么几次找他的来客错敲了我的门，我便指出他的居处，
那美丽的字顿时浮现在眼前，如水一般流淌。
　　从没见过张艺谋，只是每日散步着从他的楼旁走过。这老谋子的
楼房已很显旧，门前积满了灰尘，偶尔窗子里透出灯光，知道主人的
到来。有一次几乎要见到他了，那是一辆充满力量的沙漠王吉普车，
我们静静地看着，期待着它的主人下车，但它依旧开着，并见房子的
卷帘门哗哗卷起，车开进去又哗哗的落下，随后是寂静。心中有一缕
淡淡的怅然，寂寂的走过那幢红色的小楼，草依旧、花依旧、树依旧，
月朦胧、鸟朦胧、人朦胧……
　　屈指一数，在这豪宅之间已生活工作了近三年，但我们离它很远，
我们是另一类。
