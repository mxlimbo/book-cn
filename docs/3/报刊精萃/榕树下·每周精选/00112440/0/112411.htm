将军的头

○叶想


 
　　“人道我乡世所无，既称世所无，天子胡不唤取守京都。”

　　　　　　　　 ――杜甫《花乡歌》


花惊定将军的头是低垂着的
失望于他率领的
这一支汉族骑兵队
大宛马掂量着他的意志
巴蜀山道是雾障着的

身体里突然涨潮起吐蕃武士的血液
花惊定将军一把抓住刀柄
然后又缓缓松开
在接战吐蕃侵略军之前
将军已经败于内心的征伐了

在驻防的小村子里
花惊定将军喝着闷酒
血战东川时他的声名
已随段子璋的头颅传送各州
这一次将军却失了武勇

犯军纪的战士被带到了
花惊定将军的面前
那受惊吓的青衣少女
受了更大的惊吓
因为那战士的头立即被悬在了树上

战士的头颅在树上俯视着
花惊定将军对少女的第三次拜访
嘴角有惨白色的嘲讽
将军觉得自己是更大的违犯者
因为他爱上了这个汉族少女

少女对将军的表白回答道：
“那被处死的人的罪过，
你若犯得更严重，
怕是要提着你的头颅来
换取我的爱情呢。”
少女说完去溪边洗衣裳
她是天真的

有着一半吐蕃血统的花惊定将军
终于被卷入了这场身不由己的战斗
因为猝不及防的袭击
将军把刀斩向同族武士昂扬的头颅

不该在这时候想起她来吧
花惊定将军居然在阵前拨转马头
然后紧随其后的吐蕃将军的大刀
砍去了他的头

花惊定将军是武艺超群的
在刀光闪过的那瞬间
回身的一刀也削去了
对手的脑袋

花惊定将军的俊美的头颅
被提在他自己的手中
温柔的靠在白马的勃颈
穿过村庄外的树林
来到了凌晨的水边

这一刻大唐代宗的春梦将醒
少女正来到水边
在洗血污的衣裳之前
她先在水面照自己的脸
然后她看见了对岸的将军

“对面不是花惊定将军吗？
是战败了吧，
头颅也被人家砍去了啊！”
将军却看不见她
眼睛里突然流出了血
然后流下了泪

这时候白马嘶鸣了一声
花惊定将军的头颅落在地上
接着他的身躯倒了下来
那白马扬起蹄子跑入树林
少女呆呆地听着那“得得”的蹄声
渐渐的听不见了
