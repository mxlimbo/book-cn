九寨沟，黄龙四日游

○北京　高非


　　游九寨沟是我多年的愿望。总算机会来了，10月31日我由成都踏
上了“九寨黄龙四日游”的旅途。
　　虽说是深秋时节，但为时不晚。九寨沟四季风景如画，各有千秋。
为了那些对九寨沟一直垂涎欲滴，欲游未遂的朋友有个说法，我这就
将我此行的经历如实供来。

　　费用：
　　10月底正值九寨沟的旅游由旺季转为淡季，成都各旅行社纷纷打
折降价。原来1200元（全包）左右的标准团，纷纷降至800 元左右，
而且还可以侃价。我不是侃价的高手，但也养成了开口就问“还可以
便宜吗？”的革命传统，只一个回合就以750 元敲定。同团的两个游
友得意的向我透露说他们是720 元。

　　旅途：
　　由成都市到九寨沟的路途是400 多公里山路。包括吃午饭，休息
在内要经历大概12个小时的旅途颠簸，晕车的朋友只好“闲人免入”
了。虽然全程号称是“国家二级公路”，但塌方，滑坡还是偶尔露狰
容，总使人担心是否会“壮志未酬身先死”。在快到九寨沟的盘山路
旁就看到了翻在路沟里的客车，幸好还没滚落到山坡下面去。但我们
的藏族司机师傅身怀绝技，胆大心细，“视死如归”，堪称川北公路
大虾，嘻嘻。一路上岷江在公路的两旁徘徊。江水一路急匆匆地奔腾
南下。它喘着排击的气息，泛着汗水般的浪花，义无返顾地和我们擦
肩而过。到了汶川县就进入了山区。两边的山上几乎寸木不生。大块
的巨石和风化岩似乎对我们这些闲人陌客很有意见，虎视眈眈，欲意
断之而后快。看着这荒山野岭实在无法想象这就是通往“童话世界”
的漫漫之路。导游先生代表当地的藏羌两族对我们的到来表示欢迎，
说谢谢我们肯把钱花在旅游上，对当地的经济复苏起了重大作用云云。
一路上当然免不了要“唱唱歌（上厕所）”。“歌厅”是导游预定的，
门票一律五角，当然其内的设施是一无所有，但把门人很敬业的，使
人充分体会到“感谢”我们的道理。
　　路经的松潘古城很是别有风味。老旧的城楼，沿街的小摊，面色
黑红古朴的藏民使人感到远离闹市的宁静和朴实。过了松潘县，两旁
的山上就开始逐见树木。牦牛闲散在山坡上，藏族小山村笼罩在袅袅
炊烟中，不时传来远处的鸡鸣狗吠，马群在牧人的吆喝下鱼贯而归……
这使我想起邓丽君的“弯弯的小河，青青的山川，宁静的小村庄……”。
她在灯红酒绿的花花世界怎么会创作出如此接近自然的歌词？

　　朵来寺：
　　朵来寺是离松潘县不远的一个小寺庙，本来不是我们要去的景点，
但由于其它较大点的寺院有“法事”
　　，于是导游只好饥不择食了。对佛教我是擀面杖吹火，一窍不通。
什么黄教，白教，黑教，花教的在我看都是宗教而已。小寺庙在落日
的余辉中显得神秘而庄重。寺庙的导游小姐一见面就拱手道“扎西得
勒”，然后带着游客进入光线昏暗的佛堂。一番讲解大乘，小乘，显
宗，密宗，顶礼膜拜后，最后当然是要游客交“随缘费”了。抬头看
着威严的佛像，低头看着面无表情的喇嘛，大多数人都象中了魔似的
纷纷“随缘”。不过我还是乘机溜了，呵呵。出了佛堂的下一个消费
是“上香”。导游小姐把“上香”的好处介绍的云山雾罩。什么积德，
还愿，轮回……一个老兄上了四柱香（130 元/ 柱），买了个“开光”
的佛像竟花去了1300元，不知道他的今生来世是不是都会有佛主照着
他了。我还是喜欢照相，找到两个藏族老太太手里拿着转经筒，坐在
墙角处晒太阳。暖洋洋的阳光照在老人历经沧桑的脸上，老人们眯着
眼睛好不得意。我拿起相机就要照，这时一个老太太笑眯眯地冲我伸
出一个手指头。哦，原来她的意思是一块钱。收费还不黑吗，我照了
一张，又和她们合影了一张。合影时又有个老太太硬是凑了进来，看
来她们还挺喜欢照相。当我收好相机，老太太这时向我伸出五个手指
头。咦？怎么由一个变成了五个了？哦，怨不得那个后来的老太太这
么喜欢照相，嘻嘻。看来藏族老人也都开放搞活了。在佛光普照的圣
殿旁手摇转经筒，口中也许念的是“佛主保佑发财”。

　　食宿：
　　九寨沟正在兴建和竣工了的星级宾馆不少，看样子四星级的也有
几家。但肯定和我们这个750 元包干的团队无缘了。司机在一个小客
栈门前停下，头发蓬乱的主人，无精打采地放下正在打的麻将牌，招
呼我们下车，安排房间，吃晚饭。房间的档次号称是“标准间”，但
基本上就是招待所。好在大家饥不择食，乏不挑宿。八菜一汤（实际
上只能说是八个盘子一个盆）在饥肠辘辘的团友们高昂的食欲下显得
很惨白，不到5 分钟就空空也了。老板只好又搞些青菜来，这回很耐
吃，那叫一个咸！卫生间里的热水是临时烧的，其它用具根本就没有
（注意要自己带）。“有热水喽，快洗澡！”服务员象报告特大喜讯
似的在走廊里高喊着，可惜我们不能拖着疲惫的身子去奔走相告了。
九寨沟镇虽位于山沟里，但由于有卫星接收装置和无线信号接收站，
电视和手机信号都很强，但这里10月底的气温很低，都在零度以下了。
大多数人都感到冷的不行，缩在被窝里哆嗦。天还不亮，大家就在猛
烈的砸门声中惊醒，“起床了，起床了！”，服务员很负责地用力砸
着每一个房间的门，就象是在叫犯人。呵呵，藏族同胞就是爽。

　　九寨沟：
　　九寨沟里有三条沟，Y 字型分布。主沟叫树正沟，其它两个是日
则沟和则查洼沟。目前沟内有站站停的旅游公共汽车，所以玩起来很
方便。
　　深秋的九寨沟显得格外动人，不愧被称为是“童话世界”。最神
奇的当然是九寨沟的水，不但清澈透底而且在不同光线强度的照射下
显现不同的颜色。尤其是五花海（这里的湖都叫海，也许是藏民对海
的向往）的湖底沉积了好多古老的树木，树干树枝上结满了白色的钙
华物。树叉枝枝桠桠触须般的向四周伸展。在阳光的照射下，水中的
树木以及它在湖底投下的黑影使它犹如匍匐在湖中的水怪，给人以无
限的遐想。
　　九寨沟的很多树木就生长在水里，兰绿色的水穿行在树林中，水
面上漂浮着红色和黄色的落叶随波而下，这称作是：树在水中生，水
在树间流。据说九寨沟的河流瀑布是西玛拉雅山脉落差最大的水系，
所以在沟内到处是湍急的河流，而诺日郎瀑布是国内跨度最大的瀑布。
　　火花海是以湖面上的倒影而著称，据说在午时，阳光的反射很是
强烈，如同点点火花闪烁。不过我是在下午4 时左右观看的，也很是
迷人。湖面倒映着山上如火如荼的枫叶，湖面的涟漪使水中的红色，
黄色摇曳，闪动着，就象是燃烧跳动的火焰。
　　芦苇海是金黄色的，当中有条弯弯曲曲碧绿的小河，远处有个小
湖泊，湖边有几匹白马在饮水，马和蓝天白云的倒影衬在湖面上，宁
静的仿佛一幅图画，世界在这里停止了呼吸。
　　水转经（用流水转动的经桶），水磨房是沟内藏民以往生活的见
证，现在虽已是残垣断壁，杂草丛生，但很有苍凉的美。老旧的木板
房，苍老的古树，踩上去吱吱哑哑作响的小木桥，湍急的水流拍打在
死气沉沉的水轮上，述说着它一去不复返的活力。
　　还有静海，五彩池，树正群海，珍珠滩瀑布等，也都是独一无二
的大自然的鬼斧神工，留给大家做悬念吧。不过日则沟尽头的原始森
林实在是没什么好看的，最好就不用去。藏民牵着马出租，他们说很
远其实只不过走路10分钟的路程，呵呵，看你舍得消费不？日则沟可
以就从天鹅海往下游玩。
　　我的那些团友好象就是对刻有地名的字碑情有独衷。每到一景必
兴致冲冲地排着队地和枯燥的字碑留影。刺眼的闪光灯把个字碑照的
体无完肤，痛不欲生。这可能就是“到此一游”的玩法。

　　黄龙：
　　四日游的第二个景点是黄龙风景区。黄龙距九寨沟大概有汽车4 
个小时的路途。途径的海拔最高处有4000多米。黄龙以雪山，钙化池
风景著称。也是美不胜收的去处。11月1 日的黄龙已经是冰天雪地
（白天还好）。沿着一条通往雪山峰底的小路蹒跚而上，气喘吁吁。
想体验高原反映吗？去爬黄龙吧呵呵。当然劳有所得，你可以看到融
雪的冰水涓涓而下，淌入一个个五彩的钙化池。钙化池的形状就象层
层的梯田，池里的水丝丝荡漾，色彩斑斓，池的边缘漾出的水流温柔
地流淌入下一个池中，仿佛她们在窃窃私语，交流着情感。如果说九
寨沟的海子是美丽的天仙，那黄龙的彩池就是大家闺秀的媳妇。嘻嘻。
爬黄龙可是个累人的活，我们这群四肢不勤，五体不壮的老爷少爷，
太太小姐们各个气喘吁吁。要不是有雪山峰底的黄龙寺，五彩池的吸
引，可能大多数人早就打道回府了。
　　五彩池是一个由好几十个钙化池组成的群体。从高处看下，各个
池水的颜色五彩斑斓，因而得名。巍巍雪山耸立在蓝天白云中，幽静
的山谷环抱着五彩池水，金黄色的黄龙寺屋顶隐现在郁郁葱葱的丛林
中……真是使人怀疑身处仙境，不知所措。
　　“蓝蓝的天空，翠绿的草原，奔驰的骏马……我爱你，我的家；
我爱你我的天堂”。不知什么时候司机在车上播放了这首藏族情歌，
听起来真是亲切。再见了九寨沟，黄龙；再见了我的天堂……

　　购物：
　　随团旅游导游当然要安排购物。无论在哪里旅游都有当地的特产。
藏羌阿巴地区的特产有牦牛肉，鹿茸，鹿鞭，中草药材，水晶，玉石
等。同团的大多数人是老广。原来听说他们最会保养，这一次算是领
会了。同房的老兄对鹿茸，鹿鞭之类的补品是情有独衷，鞭鞭角角的
买了一大堆。他用稀里哗啦的普通话和我大谈其补养之道。看着他那
红光满面的样子我也不得不深信不移。不过我还是怕补枉过正了，一
发不可收拾，呵呵。中药店是不打折的，有坐堂的“医师”免费诊断。
不过经他诊断的男游客是九男十虚（他自己也虚），于是乎处方都是
如出一辙。
　　松潘附近有个牦牛肉制品的定点商店，可以免费品尝，不过购买
的话价格不匪。在这个商店，游客的购买欲望最强，你推我挤，奋不
顾身。看来国人还是以食为天。不少游客发现了个品尝的捷径：用牙
签品尝效率太低，可以先以购买的名义装几大包，然后大饱口福，反
正付款是下一步。吃完了再去装几包，呵呵。有人后悔没带它几瓶啤
酒进来就地聚会。
　　以上就是我九寨沟，黄龙四日游的大概经历。希望大家前赴后继，
发扬光大，为振兴我国的旅游事业多付出金力。
