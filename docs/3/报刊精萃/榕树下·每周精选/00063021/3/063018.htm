最痴情的和最焦灼的

○广东　步兵


    段誉一个人孤独地走在大道上，看不见阳光，看不见方向，看不
见最后是否一定是死亡。他低着头，脸上露出古怪的笑。头上的那方
丝巾在风中飘舞，上面绣着一个“誉”字，非常醒目，然而又是那么
难以分辨。骑马从他身边经过的江湖侠士远远望见他就会心生警惕，
紧握腰刀，武功不高的甚至老早就“呛”地拔出寒光闪闪的长剑，严
阵以待。在这样一个荒郊野外，烈日朗朗，没有一丝风和一个人影的
环境下，像段誉这样一个古古怪怪的年轻人很可能就是一个武林大魔
头，随时都会向路人施以致命一击。

    我不知道段誉行走时有没有数着自己的脚步。我走在大街上的时
候周围虽然是高楼林立，人潮汹涌，但没有一个人愿意理我，甚至没
有一个人愿意用奇怪的眼光看我一眼。因此我只好低着头，看着牛仔
裤上的两个大洞。这两个大洞很多人都看不懂，但我却喜欢穿着它招
摇过市。日落在即，倦鸟归巢，我爬上一座废弃的高楼。楼顶虽然不
是一个抒怀或者涂鸦的好地方，但是我可以倾听来自远方的声音，听
见我的头发在风中猎猎作响。

    那时很可能有一支宝剑正斜斜地向段誉刺去。剑客右手持剑，左
手食指和中指回握，心中默念“斩字诀”，一股杀气从指间透向剑尖。
那剑来得又平又急，在空中发出一声啸叫，然后就如同游蛇一般直扑
段誉门面而来。剑客脸色凝重，气守丹田，心中或许有一战成名的希
冀，或许有鱼死网破的悲凉。然而段誉并没有动，他立在西风中的身
体和着骄阳，口中念念有词。剑客痛苦地闭上眼睛。在闭上眼睛的一
刹那他看见段誉似笑非笑的脸此刻突然焕发出一种烂漫的神采，然后
慢慢抬起头。

    此刻我发觉自己好象端起了一架望远镜，因为我是如此清晰地看
见一片海洋似的草坪。几只中年的胖天鹅在上面踱来踱去。这本不该
是如此丑陋的腿来来回回碾在上面的地方。因此我失望地叹了一口气。
然后我就看见一个女孩从林荫道上走过，她行走的时候面带微笑，虽
然隔很远，我也能感觉得到。她的手像在舞蹈，有意无意地在空气中
掠过。那只手是如此神奇，就像一折柳枝轻轻拂过水面，整个海洋都
清静了。我目送着她纤纤的身影走进浓荫之中。

    剑客硬生生地收住了剑。剑尖距离段誉的门面只有丝毫之遥，那
股劲风已将段誉耳边的发丝掠断了数根，在阳光下飘散开来。剑客庆
幸自己没有把招式用老，然后回剑入鞘，擦一把汗。这是一个年轻却
有一部虬髯的侠士，眼中有一股豪气也有几分忧郁。十年南柯终一梦，
空将愁思寄南雁。此次南下，是为游走江湖，还是为寻觅旧日同门的
小师妹？段誉仍然在笑，他的眼里带着几许迷茫和思念。这又是一个
为情所执的性情中人，剑客刹那间竟起了同病相怜之感。此时此地当
何为？对酒当歌矣。

    我在大街上像个闲人逛来逛去，在公共汽车里挤进挤出，看着那
些从车窗外呼啸而过的辉煌的建筑，我不动声色，面无表情，恬不知
耻地盯着一张又一张来自女性的漂亮的面孔，用目光作为梳子在那上
面细细梳理，好象是要熨平思绪，找出藏在深处的记忆。即使是那些
一回头就能让人恶心的脸，也不能打消我的行动。就这样一条又一条
街，一辆又一辆汽车。我不是变态佬，但是你也千万不要问我在做什
么，因为我也不知道自己在干什么。

    在破落的塞外酒馆，剑客和段誉围坐在切牛肉和酒坛前面，不声
不语，只听见女儿红破开空气，在一堆柔软的肉体里面打个转，然后
互相撞击发出的沉闷的空鸣声。剑客喝酒的时候眼睛透过没有窗纸的
窗户遥望远方，目光忧郁，一些女儿红顺着他的虬髯流下来，像一滴
滴情人的眼泪。而段誉低着头，眉头紧锁，双眼盯着面前的酒碗，好
象里面装的不是女儿红，而是那个在梦里出现过无数次的女孩子。店
老板悄立在里间，注视着这两个奇怪的人，竟无声地叹了口气。

    夏日午后炎炎的日光下，我静静地躺在那里睡觉。其实我并没有
睡着，因为我的眼睛还睁着，透过窗帘的缝隙看对面的墙壁。那墙壁
干干净净，但阳光打过来，白耀耀的，慢慢的就勾画出一副模糊的图
案，很像一个女孩子的脸。我翻了一个身，用手拍打着脑袋，想记起
这个女孩子是谁。但我实在想不起来。于是我放弃，并准备起身活动
一下。但是我又翻了一下身，却没能爬起来，只是手脚动了一下而已。
我这是怎么啦，我惊恐万分，汗水从脸上流下来。

    段誉并不想知道这个请他喝酒的大汉是谁，他只是在静静地想自
己的心事。但是剑客的自言自语还是不断飘入他的耳朵。我要去江南，
找回我的小师妹，我很久以前就想去江南了，我已经和她分开很久了。
剑客停顿了一下，仰头喝了一碗酒，然后转过脸来，问：你呢，你在
等待哪个女孩子？这时候段誉抬起头，看见剑客的眼睛像阳光射在宝
刀上那样反射出一股眩目的光芒。那光茫让段誉感到无比熟悉，他的
心里即刻有了一丝扯动，嘴里也不禁吐出那个熟悉的名字：语嫣。

    我又一次从街巷里穿出，爬上那个破旧的高楼，看见那片海洋般
的草坪。这时候我突然醒来了，原来是一个梦。我躺在无边的黑暗中，
仔细回味着这个奇怪的梦，突然记起那个墙壁上印着的女孩子的头像，
突然记起了那个从林荫道上走过的女孩，她的笑容灿烂，挥舞着的手
神奇，像在舞蹈。我还记起了我在大街上转来转去时的情景。现在我
在床上躺着，仍然弄不清楚自己在到底在寻找什么，只是悲哀的发现，
我可能再也见不到那个女孩子了。
