独游青羊宫之三笑三怔三叹

○编辑部 紫浣花

    在成都出差，闲来无事，遂独自去逛青羊宫。
    去的时候原没想到青羊宫香火如此鼎盛，本知青羊宫向来名噪四
方，然而刚刚走得几步还是着实吃了一惊。身边川流着身背黄袋的香
客，许多人显然是远道而来。处处可见虔诚捧香顶礼的膜拜者。有香
客从身旁匆匆掠过，听得两人对语：明天才是老君的生日啊？怎么今
天……但听答：观音生日才是当天办呢，道家得提前过……方与问者
一齐恍然。遂杂于香客中信步游来。
    青羊宫果然不愧为川西道教第一丛林。占地两万多平方米，殿宇
气势恢宏，法度严谨，层层递进。古有句云：“系剑阁之灵威，尽归
行在；簇峨嵋之秀气，半入都城。”可见昔日盛名。
    不过今日的青羊宫确实香烛鼎盛。殿前殿后人头攒动，行者拜者
祝者涌涌。空气中弥漫着香烛特有的浓郁而蒙憧的气息。红烛如泪溢
满香池，铜鼎中始终有黄纸盛燃如沸。有人烧大张大张红纸，我不知
道这是不是川西特有的祈祝方式，因为我在重庆未曾见过。升腾的热
气裹着片片纸屑仿佛电影中人工布景的雪花洒落，飘袅然漫空轻舞飞
扬。虔诚的香客衣襟肩头落满“梨花似雪”。殿前殿后时时响起钟鼓
之声。我脑海里突然莫名浮出一句宋词：“村南村北响缫车，簌簌衣
襟落枣花”。回视身周与这辞句全然风马牛的情景，一时不由得哑然
失笑。
    穿过第一殿，迎面便是青羊宫集道家建筑艺术大成的八卦亭。八
卦亭底座四方，顶为圆形。亭身为八角形，“乾、坤、震、巽、坎、
离、艮、兑”八卦各主一方；意味“天圆地方，阴阳相生，八卦组合
成万化。”绕亭一周不由得我拍手称妙：整亭以木石结构，四周无墙，
只以龟纹和云花木雕相隔，一下不施以一栓一楔，全部用枋，桷等堑
成方孔，斜穿直套，丝丝入扣。亭共有十六根柱子，外柱八根，每根
柱上栩栩如生盘绕着八条金龙，据说全亭共刻龙八十一条，象征老子
九九八十一变。我反复寻觅，竟只找出了八十条龙，未知最后一条龙
隐于何处，玩味一番，不由得再赞匠心之妙，此深得虚实幻化之味！
遂收了寻出隐龙之念，欲入亭中。却见迎面柱子上楹联，一时不由得
怔了。
    只见左边柱上：“不神以为神”；右边对应：“无极而太极”！
    反复品味，越想便越痴了。试想庄子拟逍遥，老聃讲无为时，何
曾有为神之念？不过是智者自思，愚者以之为神而已，以道家之狂真，
便敢直于此处说出，高！一时便想起了张三丰当面传太极之妙。同是
一道，悟者何人？！道家原是自有觉，只信者不觉而已！愚者还以不
神为神的时候，智者已至无极而太极之境。不由得深叹！
    留连半晌，又往后行。来至三清殿。此是青羊宫正殿，气势宏大，
殿宇庄严。正是声如鼎沸，往者涌涌。殿内殿外人头攒动。有众多香
客据地而息，殿中几无落脚之地。于是先自站在门外仰首读门首对联。
见殿左是：“柱下史兴，乾坤悠久而为祖；合佛教慈悲孔门忠恕树阙
先声”；殿右但书：“道德经括，人天治乱之大源；溯群仙统驭万类
生成归于太极。”淡淡又笑。此联倒也大气派，不失道家身份，合当
放在此处。儒道释三教道家敢自居首，万类溯宗均可归于太极之道。
    待进得正殿，将迎面的第一付对联收入眼底，便又是一怔：右柱：
“溯太极之形始惟以圣人开物成务；”左柱：“首皇动而立教嗟哉末
俗诧异称奇”；细思量，倍觉其中颇有玩味之处。此已至追索思之源；
教之源了。妙便在：“嗟哉末俗诧异称奇！”不由得左右看香客们或
满足或兴奋或疲累或麻木的神色，暗暗一叹。
    高大巍峨的正殿中三清塑像极是庄严肃穆，足可满足信道者们的
信念需求。殿前一递进还有四根柱子，稍作辩别后我分辩出左右两幅
对联。左边一对，上联是：过函谷而讲道何曾跨鹤谬谓登仙；下联是：
居柱下以传经叹美犹龙只缘知礼。右边一对：上联是：在天曰太极在
人曰性此之谓至宝；下联是：治世则聪明治身则诚是以为神灵！不由
得再拜顿首，对这拟联者的意境心悦诚服。道至高处不以自身为道，
而求点破人之悟道之心，此方为传真道。此两联已直揭道之本质矣，
直示信众：何为道？在心明性则为道！何为神灵？不过思者至诚求以
治世治身之根本，被以为神灵。换明白跟你说了：没有神，没有道，
把你自个弄清楚吧！妙在于此道家最庄严之处直自道破真道本性，此
道者之悟已至无所挂碍之境！何为破？何为空？此为破为空。惜乎满
堂求道心诚之人，有谁曾抬起头来略看一看，略思一思？且叹且慨，
一时不由得又痴了。
    信步游到殿后，一壁上绘一幅“八仙过海”图。在壁前稍作流连，
突然发现壁脚贴着一幅黄纸，细细看来，原来是一番关于道之高论。
曰：“……体道会道，主宰宇宙天地人奥妙。有志于道者，大道之行，
天下为公，人间可以作天堂，何愁身家国之治……”暗想：此言倒也
合庄子逍遥游之论，得道者可御道自在于天地之间。再看下文：“……
道味无穷，醒觉妙通。……至圣孔子得道后教人急切语：‘朝闻道，
夕可死矣’”一时瞪目！不由得哑然而笑。似乎便看见孔子他老人家
急切向人语的形状，倍觉欲笑不止。虽知孔子曾赞老聃如神龙难测，
却不料孔子此千古绝句被哪位道法中人竟作此解……又思量，天下之
大道原是一理：我思故我在。子非鱼，安知鱼之乐。吾非孔子，安知
孔子所闻所解之道？哈哈一笑，撇袖又行。
    转过几道殿堂，见一照壁，许多人闭上眼睛伸手向前尝试触摸照
壁正中镌一“寿”字，以祈得福。身后便是求签之处，至有排队相候
者。无语面对香客游者之虔诚，只左右看看，转身上了说法台。
    相传太上老君西出函谷，曾语关令尹喜：“予行道千日后，于成
都青羊肆寻吾。”此为青羊宫得名之由来。后老子果然在此现身升座
为尹喜讲经说法，故有斯台。比起其他地方的香火盛况，此地显然清
冷许多。然而小小殿堂前一幅对联却真真叫我痴掉：“放翁寻梅走马
锦城经此途；孔子备礼束侑谒斯台”. 忽然便想起浣花溪畔结茅而居
的诗圣，春日草堂细雨无声，晓看红湿，花重锦官不觉晓的意境。未
知走马锦城时的放翁是意气风发，还是犹自低叹：“愿得广厦千万间？”
而千年草堂前依旧有联：“暗水流花径江沌野堂争入眼；夕阳薰细草
朔云寒菊倍离忧”……越想越是怔怔痴痴了。圣传千秋的孔圣也备礼
束侑谒此台，而放翁却只曾途经此台，有未谒过？联中之虚却教我平
添许多想象的奔放。以放翁忧国忧民的热忱，如何教他可以放淡天下
不萦怀？想必是不曾谒过此台吧。或谒，也只于天苍流云中感慨民苦，
而不曾想像以庄子老子之脱身世外独自逍遥。那么，是独御道而风游
四海境界为高，还是胸怀天下广大博爱境界为高？千年梅香在那一瞬
似乎都翻卷重云破空而来。望尽风云，不由得悠悠长叹。
    独游青羊宫，有此三笑三怔三叹。
