【编辑的话】

    看这人间三月天
    爱这人间多情诗
    告诉我你曾经存在过
    在无数难眠的夜里
    看见一个陌生人
    他曾经的往事
    用诗歌盛载着
    在黑暗中，在榕树下
    让我们彼此相爱
