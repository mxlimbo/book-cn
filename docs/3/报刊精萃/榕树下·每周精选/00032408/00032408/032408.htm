汉城美女

○后庭花

    出了汉城机场，一路上我的眼睛就一直在捕捉美女的影子，来之
前我对韩国人没什么印象，一心想知道韩国女子长什么样。人说女人
欣赏女人，十有八九是想看看人家是怎么打扮的，这话不假，其实我
也是想知道时尚在韩国吹的是什么风。
    路上车水马龙的看不真切，第一印象是女孩子的头发，如同上海
一样，大多都染了漂亮的栗色，甚至是橙色，金黄色，浓密而微卷的
长发披在身后看起来极具风情，很多女子将长发挽起，前半部分式样
酷似三四十年代上海流行的S 头，用发夹松松将前面的头发向上侧微
微撑起，在一定程度上拉高了视感，也修正了脸型，感觉华丽，也有
人将前面的头发如此夹起，后面长发任其飘扬，效果相当不俗。
    发现了汉城美女穿的鞋之后，我惊讶无比。来之前便听人说起这
种鞋，说是只有韩国才流行，开始不以为然，见过才恍然大悟。小时
候看过童话书或动画片的女孩子大概都会记得那种骑着扫帚的巫婆，
她脚上穿的黑色的前面窄小有着长长弯钩的鞋便与我看到的韩国流行
鞋差不多，在乐天世界里我见着几个妈妈抱着小宝宝，脚上便蹬着如
此的黑鞋配以黑色长裤，不久便在大大小小的商场里看到，价格当然
不菲，穿着这样窄的鞋，头又尖又弯钩，跟又这样高，走在上海的街
头，回头率一定特高。话说回来，韩国的鞋做得特别漂亮，色彩与样
式非常丰富。在Doosan Tower等大卖场里，一整个楼面的鞋夺目而精
致，让人不得不心动，就是男鞋，大多式样在上海也是见不到的。当
时真是羡慕汉城女子，想要配什么衣服的鞋在这里都找得到？
    说完了头脚，不得不说说重点了吧，谈到服饰，实在是很难讲，
在最热闹的东大门及明洞，乍一看，人海仍是黑灰白的天下，细看又
各各不同，休闲的正统的，式样大都简洁，并没有上海女子的精致。
逛商场的时候我才深深感到了地球村的奥妙：一夜春风来，遍地桃花
开。四处都可以看得到小桃红或粉红，各式小上衣，针织衫，运动衫
及套装都有这种颜色，灿灿迟开，街上还不见它的影子，倒是同行的
三个模特在回去的那天分别换上了纷红毛皮小上衣、桃红镶珠片长围
巾及玫红与白色相间的连帽运动T 恤，当然都是这几天逛商店的成果，
我的行李中也多了一件桃红针织开衫。汉城的几个大商场都设有整整
两三个楼面卖女服，另设两个楼面卖流行的包饰杂货。各式各样风格
迥异的服饰鞋包要找到自己合适的搭配起来并不容易，难得的是除了
那种古怪的鞋子，我们几乎看不到有女孩子穿一样的衣服，连她们传
统的朝鲜裙也经过了改良，变得更个性化。
    汉城女子其实也没有什么特别的地方，同是亚裔，差别并不大，
但她们看上去很美，这种美并不单单是在气质，也并不同于日本女子
的仪态万方。身材的普遍清瘦是一个因素，雅致的妆面是另一个重要
的因素。初初觉得汉城美女脸庞的立体感很强，后来才发现是妆化的
好。阴影与高光搭配得相当合协，色调又配一，为使阴影不致于太突
冗，她们的脸部色泽没有特别白的，反而是一种深象牙色，使脸部轮
廓凹凸有致，妆面相当精致。
    种种因素的调和才能有一个美的整体，汉城的发达与物质的充裕，
使当地女子能有极好的条件妆扮自己，而作为一个整体，她们大多懂
得协调的意义。所以在我眼里，来来往往的汉城女子虽不出挑，然那
种雅淡的美丽足以摄人心。可惜的是韩国只有10－30％的女子在婚后
继续工作，把美丽禁锢在家里，红颜只为事郎君，未免太可惜了。心
下不平，暗自庆幸，还是上海女子幸福啊。
