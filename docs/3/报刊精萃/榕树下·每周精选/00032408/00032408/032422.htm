文字游戏

○聆雅山房

    闲日无事，修炼新版“绝代双骄”，见铁心兰花无缺月下赏向日
葵花诗，颇为稀奇，随手凑来二十八字相和，一笑。又，见月下太阳
花，愈加有趣，再凑“十六字令”若干，助其赏花雅兴矣。

                         月下太阳花
                              
              纤腰迎日舞窈窕，金蕊嫩瓣笑面娇。
              偏是专好月下见，香睡无凭闲梦遥。

                     又 十六字令之佳赏

          葵，轻袅翩跹盼日归，天涯远，凝恨对斜晖。
          松，寿考神灵赞木公。孤直影，高士自情钟。
          竹，玉立飘然落凤株。翩翩意，风采醉名姝。
          梅，倩影冰魂玉骨妃。南枝静，轻雪堕香微。
          兰，素裳幽芳弱质寒。知音少，寂寞隐空山。
          菊，未负前朝五柳题。东栏外，姿冷韵偏宜。
                              
                     又 十六字令之闲情

          风，吹絮摇花堕汉宫。逐飞燕，飘渺步惊虹。
          花，嫩脸芳肌妒馆娃。嫣然顾，杏面落飞霞。
          雪，剔透冰莹冷韵佳。仙踪杳，轻舞胜名葩。
          月，瑶殿琼阁忆广寒。青娥悔，惆怅翠眉弯。

    聆雅山房注：十六字令，本应押三平韵，惜竹菊雪月一干
字俱是仄声，故首字出韵，文字游戏，一笑而已。
