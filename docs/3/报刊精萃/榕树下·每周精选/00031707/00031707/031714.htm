演荀子……
    烟烟：就是，我见到他偷他皮夹子，补我们的加班费，快快，上台了，
开演了……
    荀况：我很伟大！
    烟烟：知道
    荀况：我讲究“礼”“法”。礼和法都是明君治国之利器。
    烟烟：两者有区别吗？
    荀况：没有，所谓“礼义者治之始也”，“法者，治之端也”
    烟烟：哦！
    荀况：不过要灵活运用！我只告诉你，别说出去呀！我虽然嘴上反对“
刑不上大夫，礼不下庶人”，心里我还是认为礼是约束上等人的，法是制裁
下等人的。
    烟烟：“由士以上则必以礼乐节之，众庶百姓则必以法数制之”是你说
的吧？我们后代有一大帮派，世人十之八九是帮中成员，你可谓是此帮开山
鼻祖。
    荀况：？？真有此事？哈哈，真开心，不知贤弟说的是哪一派？
    烟烟：两面派！
    荀况：你相信有鬼神吗？
    烟烟：烟烟信佛不讼经，唯铭其诲行事。
    荀况：我可是个无神论者，唯物主义者。不过这些真理我只对上等人说，
对下等人就不能说了！
    烟烟：为什么？
    荀况：你想下等人知道了没鬼神，就连鬼神都可以诋毁了，何况我们这
些人呢？造了我们的反，革了我们的命咋办？
    烟烟：聪明！
    荀况：比如祭祀活动，我就按上等人下等人区别对待。祭祀“其在君子，
以为人道也；其在百姓，以为鬼事也。”这样老百姓以为“鬼事”而遵守礼
教，对我们驱使他们有好处！
    烟烟：怎么还是觉得两面派？
    荀子：我们的社会，啊，就是你们所说的“封建社会”所遵之“礼法”
和“道”，是最高的思想境界，是“百王之无变”，掌握了“礼法”“道”
的君王们都“尽伦尽制”“足以为天下极”
    烟烟：怪不得清朝以后的最高领导人都必须自己是“伟大的理论家、哲
学家”有时再带一个“军事家”。不然无法超越“无变”和“天下极”了，
而你们的君王只要掌握，不需要做“立言者”
    荀况：孟轲的“性善论”好象被你乱批！好样的！我是提出“性恶论”
之鼻祖！“人之性恶，其善者伪也”
    烟烟：为什么是恶的呢？这让我想到了基督教的“在世赎罪”说！
    荀况：别提什么什么教！忘了我是无神论者啦？”饥而欲饱、寒而欲暖、
劳而欲休，此人之惰性也！”
    烟烟：饿了不吃，饱了再吃；冷了脱衣服，热了穿衣服；累时不休息，
不累时睡大觉的人就是没惰性了？
    荀况：现在人累了不敢休息，要为长辈代劳；饿了不敢吃，要让长辈先
吃。这种行为都是违背人的本性的，
    烟烟：反对！我不是这样的| ！你应该说是“违背我荀况本性的。我荀
况的本性是：父母累死我睡大觉，见了好吃的左右开弓抢着。”
    韩非子冲出来：导演！你评评理！烟烟对孔子说不找我们法家的麻烦，
他现在自己漏了“之戊”篇，就拿我老师凑数！
    导演：笨蛋韩非！烟烟说话算话，你的老师是莲儿扮演的！
