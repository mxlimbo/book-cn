恒殊词（一）

○恒殊
           苏幕遮·相思

暮云轻，花影聚。皓月孤圆，零落星星雨。
夜醉人稀苔自绿。悄送西风，渺渺离魂曲。
恨相逢，空寄取。烛泪横滴，寂寞眉前续。
低唱阳关生死许。秋水长天，肯把相思去？
99-8-28

            青玉案

西风万里清秋驻，燕争乱，黄蝶舞。
细竹轻舟谁与渡？一篙碧水，半蓑红露，流尽夕阳暮。
人际世事无常数，来去匆匆恨天妒。
泪断苍穹归雁处。静宵追忆，寂排筝柱，写入伯牙谱。
99-9-7

            醉花阴

飒飒青杨凭日瘦，羡煞纤纤柳。
蝉静蟀低吟，落叶知秋，帘卷西风透。
举头星月愁时候，无雨听残漏。
只恐梦中逢，醉醒空庭，寂冷一如旧。
