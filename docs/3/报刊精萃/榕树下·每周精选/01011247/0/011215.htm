人道寄奴曾住
　　　　　　——三论沈宏非 

○广州　林黑




　　旧笔记小说里，有讽刺江西派把人家前朝名家的诗句（知识产权？）
象抢衣衫一样哄抢了的，长衫秀士转瞬成为衣不遮体叫花子，现在该
叫“剥光猪”。沈宏非参透福克纳的教唆，对很多美好事物进行了野
蛮“掠夺”。可为了现代人的哥们儿义气特别是这一代的伦理观，我
又记起相反的一个细节：在《海狼》中，大卫·尼文扮演的英国人对
那主角——冒险的美国佬说：商团也感激你！引伸讲就是：不是你把
我们拖入了生死一线的绝地，而是给了我们一个机会和游戏的借口！
所有旁不相干的文化珍珠，在被沈宏非“抓”了壮丁之后，我想它们
如有知觉，应和大卫·尼文一样。

　　当年读日本人写怀念赵丹的文章，是那种女人的细致温婉，她由
最毫微的细节发出哀叹，感伤一代天娇形神枯稿，明丽英俊者哀乐中
年的落魄，发现了赵丹的鼻毛很长很乱，许久没剪过了。下次过境就
特意捎来一个。这个启示就是：生活中原来有那么多细腻精致的小事，
在人类的分工与享受中，原来有千奇百怪琳琅满目的门类。周恩来事
必亲躬每一件座垫、热水瓶之类的小事，也传为美德。无微不致，算
是南方文化的阴柔力量，于碎琐和细小处透露一种亲切。在今天，到
出处有品位超群的、阳光灿烂的、冷锋凛人的、牛逼轰轰的中产阶级， 
但论到细节论到耐看程度，不免发虚发飘。绅士化运动，得靠沈宏非
这类心细如发精于煅打、因为懂得所以虔诚的园艺师催谷和浇灌。

　　自戴安娜去后，美丽的女神似乎绝了种，我是说那种公认或较为
多数人认可的通行标准，哪怕仅仅是寄托我们审美趣味的栽体。只有
遥远的逝去了不可复生或替代的，才堪称经典。又如胡荣华的名字，
在媒体不那么泛滥时，他是少数人心中奇迹，更是一桩神秘，应属于
“僻典”——但王小波的书中，幽默写到了别的象棋大师如何给他
“下套儿”的传说或野史，温暖了一部分别有怀抱的读者。到今天，
频频现身于电视，神秘奇迹和百战百胜、无法逾越的胡荣华让位给口
才雄健、个性峥嵘、反上海男人类型的胡荣华，哪个更可爱更永恒？
空前普及往往也造成彻底消解，暴曝光率高真是双刃剑。陈白露说：
太阳升起来了……，但沈宏非绝不睡去。他一直在努力，要做个有故
事的人。他讲的故事总有似曾相识的不如意之感，酸风射眸，温情脉
脉，逝去的好时光、租来的东京繁华梦，总是气体总是白头宫女的演
绎。斜阳草树，寻常巷陌，人道寄奴曾住。沈宏非既是“道”的旁人，
又好似寄奴的老友。他毫不含糊打出一面怀旧旗帜，呼呼山响。比食
文化更宝贵的“脉”也早已断掉，因为地壳都变化了。圆明园还值得
重建么？

　　破落户心态，不是仅仅体现在冯巩演的八旗后裔那里：快讨饭了
还惦着吃“肉沫烧饼”。真正有韵味的破落户感情“场”，是那种落
花时节又逢君的欲言又止，抽刀断水水更流的颓唐。逝去的繁华与荣
耀，黄昏里的紫禁城。他不是经历者，但他写出经历者的现场感郑重
感，就是席正脍细、“钟铭鼎食”。在某种意义上，他是扮足了那分
欷嘘那分感伤，所谓把美好撕给大家看，把同光十三绝虚演给“后人”
听。在温雅和由浓转淡中他悄悄跟辜鸿铭在精神上接了头，敢于逆着
走出彼得堡、砸碎象牙塔的“平民化”洪流，维持“士子”的自尊。
每一位被华夏博大文化照耀过的人值得想想，我们不能象赵本山的戏
语那么苍白：昨天俺俩准备了一宿，今天俺俩来了，明天回去。

　　西北民谣：馍馍吃不到嘴里不算麦秋；《肉蒲团》也有语：饭僧
不饱，等于活埋（我手上是盗版，也拿不准确否这几字）。实用主义
者会更敏感沈宏非有虚张声势倾向，他在用文字造海市蜃楼、巴比仑
塔。他有牵强处、造作处、虚夸处。还乱弄民俗杜撰经典涂鸦历史，
更随时见把自己拆穿把自己浠释的痕迹，他有时引入的典故或“转合”
颇为冷僻。有时也会感觉他在一本正经地扯蛋，满怀沧桑有气无声唱
挽歌。正如现在的“骚”跟屈原时代的本义已经差了十万八千里，如
今提过年，还扯什么传说中的怪兽猛禽，就真有死钻故纸堆的恶癖了， 
企图以知识凌驾普通人，却不管这知识是馊的朽的。开始是玩票的心
态，后来成了看庙守灯的方丈；本来化腐朽为神奇，再进一步练功成
魔将游戏演成定点敲钟、晨昏诵经、许三观为连续卖血就多灌些井水、
奶水挤多了挤频了难免有尿的成分。恰因他是活动自然人，没什么封
号，我们分享着他的乏力和败笔，我们自由地去伪存真。在图表上，
涨停和跌停都是直线，有起伏才是同时代人的体温，犹如足球是圆的。
刘齐和王安忆不约而同地写出了装修后的新房里，闻到别人家的厨房
味道的尴尬事。尤其王安忆极尽其揣摩的能事，她嫁给了文字，超炼
成无性别的南海观世音。她把气味写得玲珑剔透、九流乱注，象电影
戏剧里越是没出场的人物越突出。——气味共享，是现代生存空间的
真谛，哪管你是尼姑心还是享乐主义信徒。

　　而文字，却是不能轻侮、不可战胜的。
