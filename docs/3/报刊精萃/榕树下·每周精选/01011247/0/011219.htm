中国历史：元兵曲

○江苏　柳谌
　　　　　 



习惯风沙的脸
来时也粗糙
去时也粗糙
边关明月与江南水乡
给了你不同的温柔
一样不曾改变你
粗犷的命运

一百年的时间到底能留下什么？
风尘仆仆，换了战马
世祖的遗诏尤未凝干
落户的儿郎又得长投别离书
远征的人群，收不住步伐
马背上驼着刚学会的礼孝信廉
一路呼啸
没留意绵绣的大旗边角
仍嵌有杂毛的兽皮

来去的白骨连成二胡的两弦
迎着朔风拉响 大漠叫魂曲
百年之后可有流传？
袤袤平原回音绝
就象一次经年的狩猎
不需记录
跟着水草前行
最终回到最初滋润的地方
谈不上成功，或者失败

最经典的一幕是大都近旁的守陵人
将锈了的战刀一一挖出
发给每一个匆匆过客
要求他们，割一段头发
一路掩埋
养一条草木丰盛的道路
让后世的牛羊能找回曾经的牧场
