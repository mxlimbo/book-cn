中国历史：先秦年代

○江苏　柳谌
　　　　　　 



那些时日，诸子尚小
喜欢逃课，乞讨生活
和信仰
没有前人，没有教条
利嘴给出，磨平青简
部分哲人，成了无齿之徒

杀戮不止，三江两湖
圈不到狄戎苗越
祖传城旗，朝秦暮楚
丁大点地，来回功名
什么霸主，什么英雄

美女倾城，屡禁不绝
纵容一曲胭脂红
祸起萧墙，连横难诀
南歌北调
强扭的瓜，不甜

过客刀客，孤独出手
人家有男初长成
奈何天意，奈何人意
一剑并不穿心，倒是断了
赵氏胡服燕长城
