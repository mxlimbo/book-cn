当时只道是寻常

○上海　talent_x

　　　　　　 


　　浣溪沙
　　谁念西风独自凉，萧萧黄叶闭疏窗。沉思往事立残阳。
　　被酒莫惊春睡重，赌书消得泼茶香。当时只道是寻常。

　　纳兰词凄清婉丽，蕴藉自然，小令为清一代之冠，特别是他的悼
亡词，更是此中翘楚。
　　纳兰性德与原配卢氏伉俪情深，但婚后仅三载卢氏即殁，纳兰悲
不能抑，写了二三十首悼亡词，这首是我个人比较喜欢的一首。
　　上片首句就把读者引入一个悲凉凄然的氛围中，“谁念”、“独
自”之语使读者眼前浮现出词人孤寂的形象，而且，当时已为秋季，
黄叶遍地，秋风瑟瑟，秋季本是持螯对菊的佳时，但是词人孤苦伶仃，
既没有伴侣，更没有心情。秋季又是容易引起愁思的季节，孤独的词
人唯有独立残阳下，回忆往事时光，他并非不知回忆带来的只有痛苦，
但是除了回忆，他还拥有什么呢？只有在回忆中，词人才能感受到自
己的存在，才能感受到妻子的那份浓浓情意。对他而言，回忆是唯一
的幸福，也是痛苦的折磨。
　　下片以“被酒”、“赌书”等韵事追忆夫妇间的恩爱生活。在阳
光灿烂的春季，尽情饮酒，酒醉后，就这样沉沉睡去又有何不可？
“赌书”用的是赵明诚、李清照夫妇的典，赵李夫妇常指书言某事载
某书某卷某行某页，以中否决胜负，为饮茶先后，纳兰用此典，表达
的是自己与卢氏志合趣同、诗情画意般的生活。而最妙的就是最后一
句：“当时只道是寻常”，这句话并非纳兰首用，黄东甫的《眼儿媚》
有“当时不道春无价，幽梦费重寻”之句，纳兰化用此句句意，表达
了对幸福失却的追悔，当时觉得这样的生活是很寻常的，要想再次与
她“被酒”、“赌书”，恐怕只有在梦中了，如果早知有今日的凄凉，
当时应该将那快乐的时光尽量延续，永铭心中，想必此时手心仿佛还
握着淡淡的幸福，那时的快乐太过清楚，才衬出现在的孤独。这句诗
看来平淡无奇，用词也很朴素，但正是这朴素与不加修饰，更显得词
人心中的悲苦与追悔，如果多读几遍，便会觉得那种缠绵的情致萦绕
于心，挥之不去。
　　人说同一题材的词多了便滥，纳兰的悼亡词数量虽多，但由于首
首出于真情，写尽其心境与深情，每一首都有不同的特色，几乎首首
是佳作，十分值得一读。
