雪

○江苏　woodman
　　　　　　 



　　一

我——
从不怀疑你的洁白
这种坚信
来自黄昏渐暗的天边

你一定要等天黑了才来
早有预感
随着我的烟火
忽明忽暗


　　二

眼角里的沟壑
总是装满温存的企图

黑夜里
谁和我就这么
默默的起舞


　　三

我是戈壁滩上打水漂的孩子
我有童年时爱流泪的眼睛
我们的拥抱始终隔着棉袄和口罩
你散落一地的轻盈与温柔
被谁的脚步踩痛


　　四

你答应我
不只出现在我的梦里
你没有答应我
不出现在别人的梦里

我醒来的时候
看见——
你已经在别处落户


　　五

我们的相约
只能存在于摄氏零下的温度
我照看了很久的炉火
居然烧着了泪珠

那些玩耍撒野的孩子呵
在你走了以后
才被冰冷刺痛


　　六

这一天早晨
我看太阳很舒服

习惯黑夜的眼睛
渐渐空洞
