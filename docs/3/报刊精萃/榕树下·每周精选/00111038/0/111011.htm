千分之一的法国

○辽宁　青青青


　　我看着旭站在那里，很奇怪他为什么不坐下来。我向旭笑笑，不
太安分的顺着台灯的线找着电源，想为手机充电。
　　突然，我的双手不那么自由了，由觉得整个人好似置于一种温暖
之中，原来旭把我抱在了怀中，脸埋在了我的长发中。不动声色的旭
抱住了毫无准备的我，轻轻的，静静的，空气中有一种温馨。
　　一直都没有理由的有种迷恋，这种迷恋是我不敢开口说出的迷恋。
喜欢被心怡的男子由背后抱住的感觉。生活中特别在乎什么东西时，
往往是说不出理由的，这份感觉，只有自己最清楚了。拥抱的方式我
最爱这一种。
　　旭把我轻轻的转过来，抱我在他的胸前，让我的唇碰到他的肩头，
我本能的推着他，却在也止不住泪水了，我的泪一滴滴滚落脸颊，好
象有一个世纪那么久的沉默之后，旭的唇印在了我的唇上。
　　温柔的旭，我不忍接受，由不想拒绝。我的手紧紧的抱着这个几
个小时前我还没有见过面的男人……
　　晴朗的天空，熟悉的马路，空气中有一点兴奋的味道。我要见的
这个人，左右着此时此刻天空的颜色，旭是我熟悉的陌生人。我的网
络情人。
　　一个女孩子，（我很少称自己是女孩子，可事实上我也真的就是
一个女孩子。）一个人走在网路上，真实的享受着网路的精彩，来不
及消化网络中的忧伤，其实，还不懂网络也只能是一张网，变不成一
条路。
　　繁忙的都市，匆匆的行人，每个人都有自己的故事，有些人是写
故事的，有写人则只是看。
　　真实的感觉里有来自旭的清爽的味道，站在旭的身旁，闹市中的
喧哗闯不进我们的世界。
　　咖啡，我们都点了法国，我喜欢法国留在口中的余香，旭却笑了
笑，意不在咖啡的说着，有浪漫的味道，浪漫？我却一点真实感也没
有。
　　听人说起过网恋的成功率只有千分之一，眼前这杯法国可能是我
一生中的千分之一杯咖啡，滴落在旭肩头的泪也只是千分之一滴伤心
的泪，这一吻是我千分之一的网络真情的释放。
　　旭问我为什么流泪，我不愿回答，因为我没有回答的勇气与信心。
我好怕，怕这温情仅仅是他千分之一的激情的瞬间。清清楚楚的知道
了已付出了真情，却不敢想是瞬间还是永恒。这种无奈与痛的感觉我
希望不要是千分之一的。
　　旭，今天你吻了我，带走了我真心倾吐的告白，那么明天呢？我
可不可以拥有你的明天呢？
　　我想我不用再想了。
　　真的，假的，对的，错的，都让它过去吧。为自己开一扇窗，贪
婪的吸一口真实的气息。
　　记得他们对自己的好，不要忘记自己所犯过我错，认认真真的去
生活。我，不会戒网的，就像戒不掉咖啡，也没有必要戒掉一份洒脱
与执著的心情一样，我现在最懂的是珍惜二字，这是我最后的网恋，
我的千分之一的爱情。
　　被人否认了九百九十九次的网恋，却还是有它千分之一的价值与
真实。我不一定就是那千分之一的幸运儿，可是在虚拟的空间中我把
真情作为代价投了进去，去换取千分之一的幸福。

　　我的寂寞，是你寂寞时不在你身旁。
