○上海　古风流香



以忧伤命题
上阕承担文字
下阕背负千钧的隐痛

那是一张忧时伤怀的明信片
典雅和凄美是千年前加盖邮戳
投进岁月的邮箱

我剪开历史的封口
躺在你修葺的文心字眼
触摸厚重的悸动
却无力为你解脱忧愁的绳索

千年前
那座文字与情感的庭院
你手托玉杯
喝出千古的愁绪
你放手纵剑 
狂野地宣泄一生的无奈
一剑刺伤文字的躯体
又一剑封住感情的喉咙

啊是千年前啊是千年前
你那最后失手的一剑
竟插在我怜惜你的心头
