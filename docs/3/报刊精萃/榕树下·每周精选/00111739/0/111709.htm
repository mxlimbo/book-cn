南方的米，北方的面

○wqc627



　　一种让北方人觉得精灵古怪的，在南方，尤其是苏锡常，沪宁杭
一带根深蒂固的说法是：由于北方人吃面食，所以自然满脑子糨糊，
所以就愚钝粗鄙懒惰，所以才越吃越糊涂。而且即便都是在南方，也
可以长江为界分出南北，比如苏锡常沪等地的人对江北人或苏北人就
不大看得起，认为他们粗鲁愚笨，甚至对贵为省会的“宁”地的人，
也就是南京人也看不惯，说南京人象“大萝卜”，也许苏北人和南京
人是南人中的北人，都带有一些北方人的特征吧？
　　似乎也可以理解，你瞧：米粒儿再多也可以数得过来，只要你有
足够的细心耐心和效率，而面粉粒儿呢？你数数试试，数不清！所以
就别费那门子劲了，加上水一搅和一揉就索性糊涂着吃吧。然而糊涂
的面粉一加上水，却成就了“筋道”的面团儿，北方人叫做“有劲儿！”
为什么？因为里面生成了高蛋白的韧性十足的面筋，你说做成什么吃
食吧，扛饿！北方人总觉得米饭吃不饱。
　　所以北方人就象“糊涂，筋道”的面团儿，迟滞木讷中却透着一
种坚忍和大度。而南方人呢？就象粒粒分明的可以数清的米饭，透着
一种精明和麻利，也脱不掉一些斤斤计较的小气。南方人“装”糊涂
也装不来，把米磨成数不清的米粉粒儿，那韧性可就差多了，粗一点
儿的可以做粉蒸肉（北方叫米粉肉），细一点儿的可以做成“河粉”，
“米线”之类，可我总觉得那些象是淀粉做的。
　　北方人也吃米，却偏偏喜欢把颗粒分明的米熬得一塌糊涂，做成
稠呼呼儿的，香喷喷的大米粥，这样才算好喝！可见北方人想“装”
精明也装不来，可南方人偏偏不肯，南方人爱把粥叫稀饭，那里面的
米粒依然是分得清的，米是米，汤是汤，这样南方人才觉得好吃。南
方人也吃面，可即使是“糊涂”的面，在南方人手里也变得精致起来，
南方人通常是把面食当点心来做的，比如皮薄薄的，一笼十只，一口
可以吃一只的小笼包子，就是南方人将面粉“颗粒化”的杰作。看来，
这“糊涂”也是装不来的。怪不得身为南方人的郑板桥慨叹：“难得
糊涂”啊。
　　当然当然，北方人也不总是“糊涂”，那还了得？也得换换口味，
于是逢年过节，或隔三差五儿馋了，也要全家人围坐在一起，有说有
笑，剁馅儿和馅儿，很隆重地，很热烈地，很当回事儿地“精致”一
回：包饺子！不过那可不是当点心，是当饭吃的。而且即便是“精致”
的饺子，那吃法儿也显出几丝豪气：包多少是按多少“盖顶儿”算的，
吃多少在家是论“盘儿”，在外边买喜欢论“斤两”算。赶上对口儿
的饺子，不吃到嗓子眼儿就不撂筷子，管他吃多少“只”，没数儿！
末了儿还得饶碗饺子汤，“原汤化原食”，到肚子里“糊涂”去！

　　也许南方人要学得大度豪爽些，第一课就是吃面食，喝稠呼呼儿
的大米粥，北方人要变得精明干练些，第一课就是吃粒粒分明的米饭，
喝米是米，汤是汤的稀饭。
