断尾巴的乌龟

○上海　树魁


　　它是我在江阴路花鸟市场上，用十四元钱买来的小金钱龟，买来
时还没有桃核大，看样子是刚从龟壳里爬出来没几天的小东西，当时
我是戴着眼镜，凑到玻璃缸前仔细地瞧，才瞧见了它，只见它也伸长
了脖子，侧着脑袋，用它一颗芝麻般大小乌黑油亮的眼睛望着我。看
它那神气，仿佛是在对我说，你看我多漂亮，细长的脖子，乌黑的身
体，背壳上还有几条金线，再看看我的尾巴，又细又长要多好看就有
多好看，它还故意将尾巴向我弯曲了过来，好像是在说，你见过这样
好看的尾巴吗？我看着这小东西，觉得它太可爱了，心里痒痒的，想
把它买下来。我问摊主，能否便宜一点，摊主不耐烦地回答我：“你
当它是什么？它是金钱龟……”，我不知道金钱龟长大以后会是什么
样，然而眼前的小东西却是那样可爱，使我舍不得离开它，于是我就
不再讨价还价，把它买了下来。与它同时买下的还有两只小巴西龟，
那是一种翠绿色的，眼睛上方有一条红眉的热带彩龟，也十分可爱。
　　回家以后，我用一只早已废弃的乳白色大灯罩，洗净以后，放进
一点清水，然后把三只小龟放进去，给它们安了新家，也许它们对这
个完全陌生的环境有点恐惧，所以沿着灯罩壁拚命往外爬，尤其是那
两只巴西龟，更是不安分，真象热锅上的蚂蚁急得团团转。那只小金
钱龟，虽然也想逃出这牢笼，可是它没有象巴西龟那样横冲直撞，而
沿着玻璃灯罩，慢悠悠地爬行，一边爬一边寻找逃生的合适地方，当
它自以为找到了一个可下手的地方后才抬起身子来，用两只后爪支撑
着自己，并伸出前爪往上爬，可是它怎么也不会知道，这玻璃罩的壁
是又高又滑，无论它怎样费尽心机也还是滑了下来，但是它还是不甘
心，又另外换了个地方照式照样试了试，这会倒好，不但没有爬出去，
反而仰面摔了个四脚朝天，我们全家人都围着玻璃罩，看着这既可爱
又可怜的小东西忍不住发笑，我们想看看它是怎样翻身的，所以没有
伸手去帮助它，只见它伸长了脖子，并向下弯曲，用头顶着玻璃罩的
底，费了九牛二虎之力，才翻过了身。然而它并不恢心，又重新换了
个地方，继续努力。就这样它换了好几个地方，试了好几回，才终于
明白无论它怎样努力都是徒劳无益，于是它才停了下来。不过虽然它
死心了，不再想逃了，但对它的主人却耿耿于怀，你瞧它趴在玻璃罩
里，偏着小脑袋，用它黑芝麻般的眼睛，气乎乎地盯着我们，细长脖
子在一张一缩地喘气，那神情仿佛是在说：“快放我出去，干吗把我
和两个讨厌的洋龟子放在一起。”
　　以前我从来没有饲养过乌龟，不知道拿什么东西来喂它们，于是
就去向别人讨教，有人告诉我说，乌龟喜欢吃瘦猪肉，我就弄了一点
瘦猪肉来，细细地切成象绿豆那样大小的肉丁，撒进玻璃罩里，果然
那两只巴西龟子，一看见肉，就好像三月不知肉味，立即张大了口直
扑过来，它们拼命地吞食着肉丁，不一会儿差不多都给它们报销了，
而小金钱龟却慢条斯理地爬过来，那样子真象是一个饱读经书的老夫
子，既斯文又傲慢，它爬到一块肉丁前，并不急于张口，而是用鼻子
在它周围嗅了又嗅，当它确定是一种美味佳肴时，才张开金口。说实
话我看着都为它着急，肉丁早已被两个洋龟子抢光了，再不吃，恐怕
连这一块也吃不到了，果然当它正要咬住肉丁时，突然被身旁的那只
巴西龟发现了，说时迟，那时快，那巴西龟划动四爪直冲过来，一口
就叼走了那块肉丁，天哪！面对这穷凶极恶的洋龟子，那金钱龟，不
但不敢冲上去抢，反而吓得直往后退。看到它这样窝囊，真是又气又
好笑，没想到连中国的乌龟都怕红眉绿甲的外国龟。看样子中外合住
是不行了，没办法，我只好又去买一个小鱼缸来，把两个洋龟子请了
出去。
　　从此以后小金钱龟有了它自己单独的天地，它可以自由自在地在
它的小天地里睡觉、玩耍、进餐，它生活得那样安分守己，根本没有
再想到过逃跑，它只是吃饱了睡，睡醒了伸伸懒腰，行动起来还是那
样慢条斯理。而相反的是那两只巴西龟却象南美洲的绿林汉，差不多
每天都把半缸子清水搅得浑浊一片，还时常用头撞那透明的鱼缸壁，
逃亡之心不死，除了它们全身翠绿的颜色和头上的那条红眉使人感觉
好看以外，其实也没什么惹人喜爱的地方了，而且食量大，拉的屎也
多。有时候它们也会侧着头，用眼睛瞧着你，不过那神情非常傲慢，
让人看了很不舒服，好像是在听它说：“瞧瞧我吧，我是地地道道的
进口货。”无论如何在我看来，不管你是进口的，还是国产的都一视
同仁，我都精心喂养。
　　可是好景不长，灾难终于降临了。那是在这一年的夏末，我发现
小乌龟的身上长着一种灰白色的絮状物。开始我不知道是怎么回事，
还以为黏上了什么脏东西，就找了把旧牙刷把白色的絮状物刷洗掉。
可是过了两三天，那东西又长出来了，而且小金钱龟身上特别多，头
上，背甲上都长满了。那条又细又长的黑尾巴上缠满了这种絮状物，
变成灰白色的了，而且看上去它的精神萎靡，总缩着头，静静地躺着，
连吃都不想吃，那样子实在可怜，让人看了心酸。巴西龟也不象从前
那样捣蛋了，它们的食欲大大减退，原来鲜艳的翠绿色，也失去了光
泽，变成枯黄色了。我这才明白原来它们得了一种病，可是这是一种
什么病呢，又怎么能治好这种病呢，使我伤透了脑筋。我想能不能通
过增加营养的办法来增强它们的抵抗力使疾病自愈，于是我就用新鲜
的虾仁喂它们，可是也没用，它们根本不想吃。本来是两天换一次清
水的，现在我天天换水，以保持水的纯净，可是也没用，眼见得它们
的病情一天比一恶化，可我就是束手无策。终于有一天早晨，我发现
一只巴西龟死了，四肢也已经僵硬，我忍痛把它丢进垃圾箱里，两天
以后另一只巴西龟也死了，现在只剩下小金钱龟了，不过它浑身长满
了白毛静静地躺在那里，也已奄奄一息了。它不再伸长脖子侧着头用
眼睛盯住你了，甚至连脑袋都不肯伸出来，往日那种仿佛饱读经书，
满腹锦囊的滑稽神情也不复成在了。看着它这种可怜的样子，我仿佛
有一种负疚感，平日它供你赏玩，让你欢欣，现在它命在旦夕，在向
你苦苦求救，你却束手策，冷眼旁观，人类的心难道真的如此狠毒吗？
我还想倘若我当初没有用十四元把它从龟贩子手里买来，也许它会被
一位富有养龟经验的人买走，那么它也就不会遭到如此厄运，而命运
偏偏使它落在我这样一个无能的家伙手里，这难道真是天意不可违？
我又想，那些被用来充当人类宠物的野生动物也实在太可怜，假如它
们没有被捉住，那么此时此刻它们又在什么地方呢？也许它们正在河
流旁，小溪边，或者在树林中，草丛里，和它们的父母在一起，和它
们的兄弟姐妹在一起，享受着天伦之乐，也许它们也不会碰到这样的
厄运，而现在面对这可怜的生命，除了向上帝祈祷，还有什么别的办
法呢？阿门！愿上帝保佑你，把你的灵魂带进天堂。
　　我足足有两天不忍心看一眼这个奄奄待毙的小生命，由它去听凭
上帝的安排，事实上我对它也不抱任何希望了，到了第三天，我想去
看一看这个小生命是否还活着，于是就走到玻璃罩前往里瞧，我不由
得愣住了，因为这小东西不但没有死，反而比前两天神气了，而且它
已经能趴着灯罩的壁站起来，象从前一样，伸长脖子，侧着脑袋，用
它那颗黑芝麻般的眼睛瞧着我，我好像听见它在说，“怎么样，你以
为我死了是吗？我还不想死呢，你看我不是活得好好的吗。”上帝保
佑，这真是奇迹，我高兴地叫了起来，并把它捧在自己的手心上，仔
细地打量着它，它身上虽然还有那些白毛，不过比先前好像退了很多，
这时我突然发现它又细又长的尾巴只剩下半截了，我惊讶得赶紧朝玻
璃罩里看，果然哪已经烂掉的半截尾巴还留在玻璃罩里。天哪，这算
什么呀？原先那条尾巴多漂亮，现在竟然只剩下这么短短的半截了，
断了尾巴的乌龟多丑陋啊，我叹了口气，把它轻轻地放回玻璃罩里。
　　小金钱龟虽然活过来了，而且也重新开始进食了，不久它身上的
白毛也消失了，重新变得乌黑油亮，甲背上的几条金线依然清晰可见，
但它已经不是我原先心目中的那个可爱的小精灵了，只因为它断了半
截尾巴，变得丑陋了，虽然我依旧每天为它换水，每天给它喂食，但
仿佛我是在尽一种义务罢了，我对它的态度已经冷漠了，尽管有时候
它仍然侧着脑袋盯着你看，象是对人类表示一种亲热，但是我连看都
不看它一眼。
　　人本来就是一种厌旧喜新的高级动物，何况对一只残疾的乌龟呢。
第二年春天我又到江阴路的花鸟市场买来了两只小金钱龟，我把它们
与哪只断尾巴的乌龟分开饲养，怕的是那只断尾巴的乌龟抢两只小龟
的食物，平时一有空，就逗两只小龟玩，给它们喂食，而对那只断尾
巴的乌龟，除了每天换一次水，喂一次食以外，几乎就记不起它来，
其实每当我给它喂食的时候，它总是表现得非常兴奋，只要我一走近
它，它就立刻伸长脖子，昂起头，两只滴溜溜的小眼睛望着我，然后
它得到的回报却是冷冰冰的一张脸和丢进水里也溅不起水花来的两块
瘦猪肉。可怜的小东西怎么会知道，自己为什么遭到冷遇，就这样它
在冷宫里熬过了一天又一天。
　　然而上帝是容不得人间有十全十美的东西的，因为他唯恐自己创
造的尤物，在人间遭受蹂躏，所以早早地把它召回天堂，过了不久，
那两只备受青睐的小金钱龟也相继被上帝召去了，最后又只剩下那只
断尾巴的乌龟，仿佛它经受过一次灾难以后有了免疫力，更使人惊奇
的是它居然越长越好看了，尤其是那背甲上的几条金线越来越清晰，
那龟壳原先是乌黑的，现在也渐渐地变成了古铜色，说实在的除了那
条断尾巴以外，我说不出有什么地方不惹人喜爱，真的多日不见，要
刮目相看了。再说一条断尾巴又有什么关系呢？俗话说一俊遮百丑，
更而况它有这样美的一副龟壳呢。既然金无足赤，人无完人，又何必
苛求一只乌龟十全十美呢？想到这里一种爱怜之心油然而生，同时也
深深感到内疚和羞愧，因为它使我看到了自己原来也是个虚伪的人，
想当初我对它百般爱怜，甚至为了怕它受洋龟子的欺凌，特地为它创
造了一个自由的小天地，这是为什么呢？还不是因为它长着一条又细
又长的好尾巴，而且讨人喜欢吗？自从它断了尾巴后，一下子就被打
入冷宫，蒙受冤屈。这又是为什么呢？还不是因为它变丑了，现在它
又重新得宠了，这又是为了什么呢？嗨，人啊人，你究竟是一种什么
样的动物呀？
　　现在那只断尾巴的金钱龟依然生活在它自己的那个小天地里，也
就是那个乳白色的玻璃灯罩，今年已经是第四个年头了，原先还没有
桃核大，现在却已经有成熟的桃子那么大了，它的龟壳越来越趋近古
铜色，背甲上的金线依然清清楚楚。去年它还结交了一个新朋友，一
只比它大一点的小黄龟，不过它俩的性格好像合不来，一个老喜欢钻
在水里，而另一个却更喜欢干燥的地方。
