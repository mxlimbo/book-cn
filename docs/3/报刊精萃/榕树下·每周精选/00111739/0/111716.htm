雪夜

○浙江　水无缘


悄无声息
十万雪骑夜袭了我的台州
我的城池
怀揣着细致的骨瓷的光芒
碎了一地

余温渐散
我是丢失了疆域的王
和南国的浪花作别
我做了风雪的臣虏

北国啊 北国
一踏上了你的冰河
我早已乐不思蜀
我骨子里的爱情
和骨子里的故乡
早已在记忆里苏醒
我要在你的石榴裙下
想一想过去的伙伴
想一想曾经暗恋的姑娘

雪的鞭子却不停地抽打着我
岁月的喉咙唱着衰老的歌
在今夜
我只是一个亡国的奴隶
坐在石头的牢房里
寻找美丽

窗外只剩下
最后一株梨花树
在飘荡
