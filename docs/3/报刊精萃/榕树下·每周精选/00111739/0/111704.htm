很想穿长衫

○天津　江南游子


　　我想穿件长衫。
　　我想穿件和屈原身上一样的长衫，衣作绣，锦为沿，绕衿谓裙，
一身瘦骨，在汩罗江边的风里飘摇。仰天长叹，吟一曲《离骚》，
“长太息以掩涕兮，哀民生之多艰”，这空前绝后的叹息，在时空里
久久回响。做孔子的弟子，听孟子讲经，浣纱溪边，听西施轻唱歌谣，
看那柔弱双肩如何承载家国梦想。那个年代的心灵，痛苦着，崇高而
又唯美。
　　或装扮成周公瑾那样的羽扇纶巾，临江凭栏，衣袂飘飘。“谈笑
间，樯橹灰飞烟灭”，因一声“既生瑜，何生亮”而吐血身亡，在最
辉煌的时刻死去，让后世无数人为之扼腕。那时候的人，直率得可爱，
宁可自承才不如人而气死，也不玩弄权谋、搬造谣言诋毁别人。
　　李太白的那件襦裙，也是我的最爱。白色前襟沾满酒渍，清风夜
里，举杯邀明月，神情迷离。盛唐的霓裳羽衣，轻慢柔美，成了中国
人永远的梦想。歌一曲“天生我材必有用”里，几多的豪迈；吟一句
“安能摧眉折腰事权贵，使我不得开心颜”里，几多的傲气：“行路
难，多歧路，今安在”，又是几多的辛酸。“人生得意须尽欢”，洒
脱情怀。繁华长安路上，死去多少梦想，也写就多少诗篇。从此后，
再也无处寻找这一份才气。
　　唐时的风，宋时的雨，挽一个柳永的发髻，着那件青衫，写一首
长词，千种风情，诉与谁听？管他功名利禄，“忍把浮名，换了浅斟
低唱”。且填词去，听花间柳巷传唱柳词。到沈园，为唐婉和一首
《钗头凤》，无论写在何处，是真情总该有人回应。或于香径徘徊处，
为晏殊那句“无可奈何花落去”，对“似曾相识燕归来”，这样的好
句子，一万年才得一句。会拿笔的人，都是文人雅士，酒肆山林，断
墙残垣，题的都华丽篇章，即便是山野村夫，竟也都会吟古风。
　　穿件鲁迅那样的长衫，垂直落下的前襟，容不下跪下的膝盖。脖
子上绕条白色围巾，挽着同学们的手，与先生一起呐喊。人们因愤怒
而血液燃烧，青春的脸上张扬着无私无畏，勇敢着的美丽。
　　要不穿件孔乙己那样汗渍斑斑的长衫，就算多么潦倒，难得是坚
守份读书人的矜持。
　　而当我每天必须西装革履，衣冠楚楚端坐在办公室里的时候，脑
海里飘过的尽是这千百年来的那一件件素色长衫。我知道，在这个早
已物化的世界，再也无处收藏一个读书人的清高。
