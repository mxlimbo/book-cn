属忆吉祥寺·樱

○上海　诺德


　
　　　　　　　　　　　　　　晚樱

　　一片枯黄且已经被焦灼过的樱花的叶瓣飘飘扬扬地从指缝中遗落
下来。我弯下腰拾起这片似曾熟悉的枯瓣，上面依稀可见的菁色的墨
迹已含糊，但很显然，我知道，当它还是鲜嫩的时候一定有过笔迹。
至于文字的内容，我想，已不再重要……因为写下的东西大多是记忆
的回誓，而不是预见。
　　初夏从来让我心情躁动，每早都有来自散落在各地朋友的问候，
然而现在除了阅读这些信笺外，我已经不能做其他事情了……
　　“今天只有一封信。”老邮递呼吸缓慢且声音沙哑。
　　信封背面渗透出植物焦烂后那暗黄的色斑，邮戳是千年三月的，
航空邮件，来自日本吉祥寺。
　　院子里那棵日本晚樱枝繁叶茂却早就错过了开花的季节……今年，
花其实并没有开过，当然也就不会再开了。
　　我问：“三月的信怎么拖到七月才……才给我送过来呢？”
　　“哦，哦，哦，那封信原本没有您地址的，只留下街道的署名，
门牌号码是‘晚樱’，您瞧。”他指给我看，“开始我没敢递过来，
希望有人询问，可是却久久没有音信，又想，您这条街坊也只有这一
棵樱树，它就在您的院子里……”
　　我颤抖的手哆嗦地在信唇撤开一条口子，生怕损坏一些尚不可知
却一定存在的“什么”。然而它还是在指缝间遗落了……飘飘扬扬……


　　　　　　　　　　　　　　二月

　　夏夜的星空中忽而会闪烁出一长条流星的划痕，耀眼而短暂……
正如眼泪从半空中晶莹地陨落一样，没有人知道它将带来如何的预兆，
是幸福还是悲哀！？
　　你我本该在今年的三月去吉祥寺看樱花的，那是一个曾经手握手
许下的誓言，但是还未等到花开的季节，守侯花开的人——你，已经
不愿意期待那花开一线美丽时刻的到来。
　　今夜你被他紧紧地搂着，微笑着，眼中充满着恍惚，就象在中学
时度过的许多迷茫的日子那样，你天真却也无神。
　　“民，这是我的男朋友……东大的同学……”
　　“呵呵呵……，恭喜你，也祝福你们！”
　　“初めまして……”
　　“初めまして！”
　　“淮海路上的灯火其实不比东京的差……你有觉得么？”
　　“现在的世界原本就没有多大的区别。不是么？”
　　“恩，是啊，上海已经算是很不错的了。”
　　内堂并不宽大的寿司店里只有我们三个人。情人节的早晨其实多
半情人还躺在床上陶醉在梦香里。我知道这是东大寒假的最后一天，
下午你们将要回日本，所以大家便赶早在这里就这么草草地过起了本
该双栖双飞的节日。
　　一如既往，我把一合精美的德芙递给你，然后只默默地看着，看
你多年来，我已经熟悉得不能再熟悉的吃巧克力的样子。然而在日本，
今天该是女孩给男孩送巧克力的日子，所以你地道地把一小块褐色的
但并不包裹任何光泽的“弹丸”递给了他。而我心里也突然涌出一阵
子的酸……却没有丝毫的嫉妒可言，当然也就更不敢有其他的奢望。
　　在隆冬清晨黯淡的红日升起的时候，我发现那晨日和日本——我
从小就并不怎么喜欢的国度——的旗帜仿佛相似的时候，你依偎在他
身上无限满足和你那张陶醉的韵红的脸蛋仿佛相似的时候……我已经
不能自己。所幸的是眼眶中没有多余的眼泪，因为它就要为你而流干
了。
　　我的单车飞驶在宽敞的大道上，在家门口的转弯处，只记得眼前
一团漆黑，醒来的时候，便只能靠读信笺打发时间了……


　　　　　　　　　　　　　　星空

　　躺在病榻上，我的思绪开始迁移……眼中又映出漫天星辰和那几
个曾经是无限快乐的幸福的夜。
　　虽然我曾拉着你纤细的肘踏在翻起的冰冷的江水中，不知去向，
你也曾撤着我的衣袖在暴风雨中摇曳，飘渺而恍惚，又当你倒进我胸
怀中温暖你瘦弱身体的时候，那一刻我便可以毫无思索地在你耳边，
轻轻地呼吸，并伴随着对你最无言语的爱。
　　那还是98年的盛夏，记得长江的洪水满过瘦西湖的护壁，鱼虾越
出池塘爬上你的足裸，我们的游轮漫无目的地漂浮在数十里的江面上
无法靠岸……身边的每一个人都穿着救生衣，随时准备着跌入江中。
不过惊恐却始终远离我们，一切都视死亡如无物般。因为，毕竟你我
的心此刻无所惧怕！
　　也就是这个夏夜，一样酷暑的七月，漫步在细雨中，你会搂着我
笑问：我们的分手不会是雨季！傻傻地让我为你唱“我是你夏天里的
冰棒，冬天里的太阳，阴天的大雨伞……”会……还会……
　　然而当一切寂静的时候，人们都睡熟的时候，夏虫淹死在洪水中
只能以无声哀悼的时候，你依然可以让我挽着，抱着，仰视星空。那
样一种自由和无赖只有记忆深处才可以感知和印刻的。
　　那时漫天的星辰没有多余的光亮，四野也没有杂音，隔几个小时
或则半个小时都可以看到流星。不多，但可以期待……
　　漆黑夜空中留下的短暂的辉煌和那一瞬光彩，就好比你脸上那不
常有的淡淡的微笑。不多，但至少可以期待……
　　　　　　　　　　　　　　　

　　　　　　　　　　　　 吉祥寺

　　远在日本的你的父母曾兴高采烈地告诉我，樱花是日本最最吉利
的花，它高大而纯洁，韵红但不娇艳，茂盛而不臃肿。我知道只有樱
花才能见证我们的爱情，所以在五年前，我把一棵你亲手选择的晚樱
带回了中国并把她种在我的门前。你则把另一棵我选择的早樱留在了
吉祥寺，在我们初识的那片长满菁色花朵的你父母的后院里，让她每
年都能开放出如你唇彩般韵红的花朵。
　　那年你才十六，个子低我五公分但也快一米七了，我还清楚的记
得你穿高根鞋后硬扯着嗓子吆喝着说我矮的样子，蛮横但不泼辣……
那时穿短裙的女孩真的很少，因为有人“喜欢”认为露出大腿那便是
最大的伤风败俗，但你还是坚持着不畏“寒冷”。我也不以为然，我
明白你穿短裙的娇媚是世间最青春的飘动……
　　你说你喜欢GTO ，但不喜欢鬼冢，因为GTO 的“老家”在吉祥寺。
我不喜欢GTO ，但我喜欢鬼冢，因为鬼冢的老情人同样在吉祥寺，我
希望我能成为鬼冢那样的男孩，刚强、矫健、风流而不下流，那样的
女孩则是你，美丽并兼顾着贤惠。
　　那年吉祥寺的夏出奇的不闷热，多雨多霞。当地的日本老人告诉
我们，在日本吉祥寺算是最有名的有情人终成眷属的地方，有那样的
天气则更是风调雨顺的象征。有这么一个传说，吉祥寺因为多雨则女
人的丈夫便不用出海。而那里街上的落叶就可以两个人踏着，发出沙
沙声响，有节奏有韵律，那就是他们幸福的滋味。起初我并不相信那
样的寓言，不过当我挽着你在彩霞底下也同样发出那样的沙沙声的时
候，我确信了，醉倒了，痴迷了。记得就是在一棵晚樱树下，我第一
次透过你的双唇感受来自你深处的气息，低微、清香、甘甜。而你幸
福的告诉我你是世界最骄傲的女孩，因为你不用抬头踮脚就可以让我
“公平”的吻你。
　　快乐是不言而喻的……夏天却也是多情短暂的。后来我回国了，
你我便许下那样的诺言，来年夏天樱花再开放的日子，就是相逢的时
刻。
　　

　　　　　　　　　　　　　　尾音

　　我在上海就那样等了你五年，每年都等待樱花开放。我曾写过的
无数张卡片也如石沉大海般寥无音迅。再后来就知道你在东大有了新
男友，今年的情人节你们会来上海。而后，我们吃料理，赠巧克力，
我也从此失去了站起来的“勇气”……
　　我拾起落到地上的那片樱花的花瓣，菁色的墨迹记载着永不熄灭
的爱情？我仍就不敢奢望。
　　“永远爱你的菁”；我打开信封开始默念这封迟到的告白。
　　“民，我十七岁那年，也就是我们相遇后的第一个樱花开放的日
子，父亲的会社破产了。只有我才能挽救我们的家庭，我父亲的会社，
所以我必须嫁给他……”后面就看不清楚了。
　　“我知道我爱你，但真的必须离开你……”断断续续的。
　　“但真的当我股起勇气离开那样的并没有爱情和亲情的家庭的时
候，我怀孕了，孩子是他的。我不知道如何把那样的消息告诉你，但
我真的还想最后一次看到你。所以我和他来了上海，我强装着幸福，
如果你细心，我的眼睛告诉你，我是爱你的，却也是迷惘的。”被樱
花的汁液遮去了几行字。
　　“当你收到我这封亲笔信的时候，我已经永远离开这个世界了，
用我对你最纯洁的爱来世偿还……爱你的菁”；纸滑出手心，落到树
下，缕缕阳光照在上面，字迹是菁色的，隐约的，匆忙的。
　　我拄起拐杖走到树下抬头把目光扫向树梢，我似乎看见你最美丽
的微笑……
