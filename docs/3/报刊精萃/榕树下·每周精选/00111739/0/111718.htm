一个象美玉一样的灿烂紫玉苍穹盖了下来，一阵阵奇妙的香气传来，
四周一片氤氲，两人只觉得神采大增，与此同时，在空旷的雪地上，

际的透明翡翠一样。

水晶植物一簇簇，一排排错落其间，一条条玲珑剔透的水晶尖柱在苍
穹下高高插立，还有一道道艳丽斑斓的彩虹在琼楼玉宇间纵横交错……

　　“它只是善者世界同达拉斯的一个链接点，并不是真正的善者世
界。”亚蒂兰斯指出。

塔在紫色的苍穹下巍然矗立，在平台的顶部，一把闪闪发亮，霞烟片
片的水晶神剑正倒插在一张张开的巨大水晶狮子口中。

那座金字塔冲去。

围着的平台广场。

从龙头剑柄射出，“啊”一声惨叫，高伯士被电光击倒。

　　“这剑有机关。”高伯士骂骂咧咧。
　　“这剑没有机关。”亚蒂兰斯也上来了“它是属于善的力量，只
有用善和爱，才可以将它拨出。”亚蒂兰斯伸出了双手“让我来吧！”
她闭上双眼，抓住剑柄轻轻一拨，“轰”一声，在震天动地的巨响中，
闪着耀眼光茫的水晶神剑被亚蒂兰斯拨了出来，神剑向上高高举起，
金光万道，四周的琼宫玉宇全披上了一层辉煌的金光。

　　“伟大的女神，你又创造了一个奇迹，你将会成为拯救宇宙最伟
大的神话，我，你最忠心的奴仆，只有一个卑微的请求，请允许我摸
一摸达摩克利斯神剑。”高伯士突然跪在亚蒂兰斯面前。

　　高伯士背对着他们举起熠熠生辉的水晶神剑仔细端详起来，他摸
了摸神剑，突然哈哈大笑起来。
　　“你干什么？”史杰连开始觉得势头不对。
　　高伯士背对着他们的笑声慢慢变成了阴森森，带着电音的笑声
“嘻嘻嘻”笑声开始变得邪恶起来“史杰连，有一件事我对你撒了谎，
我并没有拒绝同达提合作。”
　　高伯士猛得一下转过身来，他的样子全变了，变成了一个青脸尖
牙，双目闪着绿光的狰狞脸孔“只有白痴才会拒绝永生不死，我加入
了僵尸，没想到吧，白痴们，哈哈哈。”

来我都没见过你吃东西，怪不得在精灵的异次元空间你的棉被里全是
人手人脚，原来是它让你不自觉露出了本性。”
　　“可惜已经太迟了。”高伯士得意地笑了起来“你真的以为藏在
矿石中电颈装置就可以逃过电脑的追踪？你真的以为矿洞的那个爆炸
只是个意外？你真的以为在飞行器里碰到亚蒂兰斯只是凑巧？宇宙中
只有亚蒂兰斯至仁至善的心才可以拨出达摩克利斯之剑，但强迫她一
定不行，所以我跟你在矿洞上演了一场精彩越狱的好戏，让亚蒂兰斯
主动去拨这把神剑，这就是达提完美无缺的安排――B 计划，你太令
我失望了，到现在才发现这是圈套。”
　　“你这具腐朽的僵尸，看看你的样子，就跟你的心灵一样丑恶，
怪不得在异次元空间你的思想完全被精灵控制，你得到了永生，却失
去了自己最珍贵的东西――善，也就是生命存在的意义。”亚蒂兰斯
一针见血指出。

斯就是一刺。
　　“不”看着倒在血泊中的亚蒂兰斯，史杰连悲愤地向高伯士扑去，
但力大无穷的高伯士一脚便将他踢飞到十几米开外。

周围的水晶世界，水晶建筑也在狂风中大片大片地崩溃，“啊”史杰
连躺着的水晶地板也消失了，他凌空掉到了白茫茫的雪地上。
　　当他从雪地上爬起时，他发现自己又回到了南极的大冰原。
　　在大冰原的四周，几十架全副武装的金属飞行器将他团团围住，
大批大批的僵尸兵荷枪实弹地将四周围到水泄不通，十几个披着黑袍
的尖脸僵尸走了过来，为首的是身材高大的首领达提，十二个作为头
人的魔法僵尸全到齐了。
　　“高伯士，干得好，你是我的荣耀，我的骄傲，我宣布，你将成
为第十三个魔法头人。”达提笑着道。
　　“我有更好的提议，伟大的达提殿下。”高伯士阴阳怪气地冷笑
着回应。

　　“送你一张下地狱的免费单程票。”高伯士突然举起神剑对着达
提就是一劈，“吱”在惨叫声中，达提被劈成两半化作黑烟消失了。

尸兵同时向它们开火，八个魔法僵尸中弹倒地，只有四个化作蝙蝠飞
进了那艘魔法飞船。

流，光滑的机身灯光熠熠地腾空飞起，在达拉斯星，黑暗魔法无法施
展，所以魔法僵尸只能夺路狂逃。

给高伯士。

“轰”一声，远方的小黑点化作了一个火球。高伯士得意洋洋地把遥
控器扔掉“我喜欢这个节目。”

对那个军官封授，原来这场政变他蓄谋而久，他不喜欢头上有班魔法
僵尸，于是借着B 计划勾结这个军官利用达摩克利斯剑的威力发动政
变。

高气扬地望了望正挣扎着从地上爬起的史杰连哈哈大笑起来“姓史的
狗给我听着，放弃吧，奇迹不会发生，我已经赢了，神剑在我的手上，
我拥有宇宙中最伟大的力量，我就是传说中的神，识相的就趴在我的
脚下舔我的脚趾，也许我会大发慈悲，让你也做一只僵尸，哈哈哈。”

伯士举起了神剑，他邪恶的双眼露出了杀机。

就是一劈。

损，还没等高伯士反来，史杰连一记重拳竟将他直接打到腾空飞起。

亚蒂兰斯披着闪亮的霞光出现了。

到了史杰连的面前拾起了地上的达摩克利斯神剑。

回来了。”亚蒂兰斯把神剑高高举起“消失吧，黑暗魔法，在光明中
永远消失，返回属于自己的世界。”

全化作了一团团青烟，它们被光明溶成了一堆堆黑灰……它们失去了
在光明中存在的力量。

己燃着熊熊烈焰飞出了达拉斯的大气层，它象流星一样飞进了漆黑一
片的外太空，“轰”一声，它在太空中化作了一个火球。

一座废弃了的金属城堡里面，到处都是骷髅骨和锈迹斑斑的破烂金属
壁，一些地方还燃着熊熊烈焰，天哪，它简直不敢相信自己的眼睛，
那些火焰竟然全是黑的，黑色的大火正在四周熊熊燃烧。

从城堡外面的剑树林中扑了出来，漫山遍野地扑了过来，一个个黑色
的火球从它们的口中喷出，“轰轰轰”城堡到处都燃起了熊熊黑焰，
一些蜘蛛怪扑了进来，跟拿着各种武器的骷髅人展开了疯狂的搏杀……

漆黑一片的泥里……

成了一只巨形昆虫异形，它已经失去了记忆，它的心中只有残杀和仇
恨，“哇哇哇”在一阵惊天动地的怪叫声中，无数的丧尸僵尸异形怪

亲吻了史杰连一口“记住我的话，善是一切力量的源泉，只有善才拥
