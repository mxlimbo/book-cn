记忆犹新

○江苏　棠棣


我相信十七世纪，我死去
辗转于此
我埋藏在那儿，草草落葬
我出现在那条河谷的时间总共不过这么多
我落在我的死亡之中
是的，我还记得我的双亲
他们砥足而眠，希冀一处天空下的屋顶
现在，他们早已变成了马，在那里静静吃草
月光越过短墙
我想接摈事物 呈现出它们的魔力
它们把这称作现实中的预警

某幢房子，露湿的草和空中适宜的气温
我记忆犹新，它们联合起来，象一种力量
在我，则意味丧失，永远
我的泪水涟涟，象路人般犯傻
我爱过，刻骨铭心，现在已是真相大白
是她，她爱的不是我
穿透迷障，瞧，不过是一头披着张豹皮的公狸子
我终于说服了自己，答应让时间如飞而逝
