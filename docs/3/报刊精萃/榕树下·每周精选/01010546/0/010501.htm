爱你是我的习惯

○甘肃　轻衣侯



　　小张俩口子在别人眼里可是模范夫妻。他们住的那个家属区只要
有夫妻俩吵架，总有一方忿忿不平的拿小张家的幸福美满生活同自个
的悲惨命运做比较，声泪俱下的对自己的丈夫或妻子的恶行进行血泪
控诉：“你看人家小张俩口子过得日子，看看你这#**%￥#** ！！”
好像小张家是解放区的天自己却生活在万恶的旧社会里一样恨不得解
放！
　　可是幸福不幸福就像穿鞋一样，夹脚不夹脚只有自己知道。
　　这一天小张像以往一样在家帮妻子做晚饭，忙碌之间妻子看他神
色恍惚的就问他，“你怎么了？是不是病了？”小张躲开她直视过来
的眼睛的眼睛支支吾吾地说没事。妻子狐疑地望望他没有说话。
　　晚饭后小张坐在沙发上看电视，妻子坐在他旁边织着毛衣，孩子
今天很乖的写完作业洗漱完就上床睡觉，小张像以往一样陪在孩子床
边讲了故事，看着孩子睡了就关了灯轻轻掩了门走了出来。
　　“睡了？”妻子问他。
　　“睡了！”他对这一句已回答了上万次的话突然有些厌倦。
　　他坐了下去聚精会神地看起电视来。过了一段时间突然听到妻子
幽幽地说“你最近话越来越少了……”
　　小张刚打个哈欠正伸懒腰，听老婆这么一说怔了怔。
　　他轻微皱皱眉，扭过头去仔细地看着妻子，看到妻子的眼里隐约
有泪光。他的心一动，伸出手去抬起她的下巴轻轻地摇晃着，笑着说：
“你的小脑瓜又想到哪里去了？”
　　妻子拨开他的手低下头去深深地叹了一口气。两人谁也没再说话。
　　夜深了，小张又在做同一个梦了，梦到小美了。梦见他同小美在
跳舞，灯光音乐声中两人团团舞动，小美笑声像屋檐上的风铃一样声
声清扬悦耳。突然间小美杏眼瞪圆强逼他同他老婆离婚，看他左右为
难的模样小美声嘶力竭地说：“你不离开她我就死给你看！给你看！……”
说完就纵身往楼下跳去。
　　“小美！小美！！不要啊！！”小王紧闭着眼睛满头大汗地喃喃
的叫个不停。
　　夜色之中小张的妻子眼望着他，捂着嘴无声的哭泣者……
　　　　　　　　　　　　　　　
　　早上小张上班看见小美就远远地躲开，小美不屈不挠几次把他挤
在楼道里冷冷的望着他不说话，最后一次在楼道里小张又被小美挡住
小张无可奈何小声哀求她，直到答应下班后在老地方见面给她一个交
代后她才走开。
　　望着小美纤纤背影慢慢变小，小张禁不住重重发出一声叹息……
　　晚上九点，小张准时来到情深酒吧找了个偏僻角落坐了下来，点
了一颗烟无精打采地吸着抬头四处张望着，远远见小美穿着鹅黄色的
短裙精灵一样无声息地向他走来。“你早来了？！”小美温柔地问了
一句。他点点头勉强笑了笑小张叫了两杯咖啡，两人默默的喝着谁也
没说话。音乐轻柔舒缓空气中弥漫着花香，小美专注地望着他脉脉含
情。小张望着她的发边唇角他曾经轻薄的地方越来越不能抗拒她的诱
惑，“完了！完了！我注定会成了她的俘虏”。妻子孩子的影子在小
张的心里越来越淡去……
　　小张终于决定同妻子离婚了，离婚时妻子平静的问他：“你为了
这认识几天的女人就离开我和孩子，你这样做对我们公平吗？”小张
的回答是：“我和你从小就是邻居一同上学工作到结婚生子，我以为
我爱你，现在我才发现我爱你只是我的习惯，你觉得这样对我公平吗？”
　　小张同小美的婚姻并不像他们想像的那样美满，甚至小张还觉得
有一种说不出的遗憾。直到有一天两人没事一起打牌玩，小张输了小
美重重的刮了小张的鼻子一下，小张像起前妻，前妻知道他有鼻炎每
次他输了只是轻轻地摸摸他的鼻子，想到这小张滚下两滴热泪下来。
