天界网络搞笑版

○湖北　王锐



　　红尘网络方兴未艾。碧落和黄泉又如何呢？

　　宙斯很烦恼。因为网络的问题。维纳斯开了一个与美学有关系的
网站。竟然被人选为了天界十大色情网站之一。维纳斯很烦恼。宙斯
因为小蜜维纳斯的烦恼而烦恼。太阳神阿波罗和武器之神阿忒斯开个
购物网站在和如来手下弥勒的购物网站的竞争里破了产。宙斯为儿子
的烦恼而烦恼。更有甚者，宙斯一直看不起的那个自恋狂水仙花阿忒
罗堤斯开的玻璃网站竟然在一夜之间和印度那湿婆玻璃网站一样成了
天界来访量最大的网站。宙斯觉得应该和天界其他的最高统治者一起
开个会。

　　奥玎也很烦恼。他本来只统治北欧那里小小的天空。为了满足日
益增长的物质需求。他和手下七个打手一起开了个UO网站。叫和七神
做战。天界无聊的神仙们一时间趋之若骛。后来如来的手下斗战胜佛
孙悟空开俩个我们一起西游网站。那宙斯的手下也开了个特烙依战网，
一下子就把生意全抢了。看着日渐稀少的来访量。奥玎很烦恼。他决
定找天界其他的最高统治者谈谈。他上了OICQ找到了宙斯。两人一拍
既合。联名向在天界最高统治者论坛发了个帖子。号召大家在天界元
年的某一天在某个地方开个会。讨论一下天界网络发展的问题。一下
子点击率极高。

　　会议是在嫦娥姐姐开的起舞弄清影网站召开的。

　　天界有两个著名的新闻网站。一个是宙斯的信息之神赫眉斯开的。
还有一个是唐僧开的。这两个GEP 今天亲自出马。采访到会众神。宙
斯和奥玎来了，两个愁眉不展的神仙敷衍了一下在天界以罗里八琐出
名的唐僧。一心接受赫眉斯的采访。毕竟大家都是西方人，有亲切感。
唐僧不乐意了：大家都是神仙，拜托你给点面子好不好。奥玎从自己
肋下抽出一根长矛，指着唐僧：再说，再说我刺了你！宙斯挥了挥手
招来闪电一道，顿时把唐僧打的乌黑。袈裟已经乌黑的唐僧拿出一只
笔在小本上写着什么。宙斯凑上去看看差点没气死，唐僧写道：宙斯
发火，唐僧遭打。原因是这样的……

　　这时候如来和太上老君一起来了。如来有八百弟子，五百罗汉。
他自己做了一个佛学网站，不怕没有来访。还强迫手下弟子的网站做
了自己的子站。这样一来财源滚滚。太上老君开的道教网站因为是王
重阳和丘处机做GEP ，差点没做成自杀网站。好多人都因为练习了王
重阳的武功而走火入魔最后只好去了哈迪斯和阎罗的网站。行宫好老
君的弟子葛洪和孙思邈开的医学网站为他争了点气……

　　如来看着唐僧被打的落花流水，心里暗自愤怒。可又不能表现在
脸上，和宙斯奥玎打了个翘，口颂佛号：将网恋进行到底，施主为何
动无名之怒？宙斯看着如来气就不打一处来：你难道你觉得他象个苍
蝇吗？我一闪电打下来世界顿时安静了不少！奥玎在一边附和着：是
呀是呀。如来说：那你也要给我点面子吧。奥玎看着宙斯：对呀对呀，
大家都是神仙吗？宙斯看着奥玎墙头草两边倒，顿时怒发冲冠，也忘
记了来前和奥玎说好的合作协议。破口骂到：以前要我帮忙的时候说
我说什么你就做什么。现在看见如来，你就附和他说好呀好呀，拜托
做神仙有点原则好不好？

　　这时候湿婆爪牙雾爪的来了。奥玎顿时好象看见亲人一样的扑上
去抱着湿婆哭诉。湿婆一脸无辜的说：今天我还没掏耳朵呢？你说什
么呢你？？然后把奥玎的手打开：拜托大家都是神仙。不要性骚扰好
不好？？

　　这时候嫦娥姐姐出来了。宙斯那里见过东方美女。一见就倾心，
上去拿着嫦娥的手亲了又亲。那奥玎更不是等闲货色，上去就是一熊
抱。在嫦娥的脸上吻了一下。顿时如来和太上老君连说罪过。嫦娥娇
笑着把他们领进会场。才发现哈迪斯。阎罗。撒旦和上帝还有玉帝早
就到了场。

　　因为嫦娥开网站的钱是玉帝出的。所以大家就让玉帝当了主持人。
其实玉帝和嫦娥那档子事在天界早以不是什么秘密了。

　　这时候会议的召开者玉帝清了清喉咙：这次会议是为了让大家对
天界的网络问题有个清楚的认识。然后找到让大家都同意的方案来协
调天界网络存在的一些问题。
　　阎罗第一个跳出来愤怒的指着宙斯说：偶对你有意见。很有意见。
你们开的爱神网站开什么玩笑。竟然让我们的女鬼聂小倩和那人间的
书生宁采臣来了段阴阳恋。

　　宙斯跳了起来：什么？？你也不看看我女儿艺术女神缪斯在月老
那个瞎搞红线的网站里认识的吕洞宾怎么对我女儿的。弄的她天天没
心事工作。你看看现在人间的艺术那有什么美感。

　　玉帝说：那有什么？？我七女儿不是也在月老那认识了董永吗？？
他们现在做的鹊桥网站专门对那些夫妻两地分居的神仙对指导，教育
他们怎么找快乐呢？？

　　嫦娥说：是呀其实网络上爱情很不错呢？我的起舞弄清影里为李
白专门做了个文学专栏。美杜沙在里面就和张果老网恋上了。两人专
一着呢？？

　　太上老君说：是呀是呀。小李还写了个《第一次和女神仙亲密接
触》反响不错呢！

　　如来说：你还别说1 说这我就来气！我的弟子就因为看了这个开
始有了凡心。经过我再三教育女神仙是老虎。他们才收敛了点。可是
我那女徒弟陈妙常就因为看了这，已经和人私奔了！还闹出了思凡的
笑话！我KAO ！

　　唐僧在一边拿着小笔记下这样一段；佛主对《第一次和女神仙亲
密接触》说KAO ！

　　如来看见唐僧把他的话记了下来。顿时气急。抢过笔说：傻不傻
呀你？小心我那天用炸弹把你破网炸了。

　　撒旦说：现在我在人间都收不到灵魂了。凡人都不相信网恋。我
那天装一MM在天空界的聊天是和人聊天。竟然有无数男神仙象我表示
爱慕。这还是不是天堂呀？？神仙怎么可以有七情六欲呢？

　　玉帝说；神仙怎么不可以有呢？？你看我和嫦娥姐姐不就是吗？
说着就给嫦娥丢了个媚眼。

　　阎罗在底下嘀咕；什么呀！就你一个人可以！呕们都孤独着呢！

　　上帝说：我那天在一个文学论坛说我是上帝。竟然被人骂了！

　　顿时大家来了兴趣！上帝一看大家都看着他马上来了精神：被人
用什么板砖拍死了我！什么世界吗？？

　　如来说：我还好呵呵。一般要有人拍我，我就炸他呵呵

　　奥玎囔囔着：我说两句嘿。湿婆你哦也太过分了！我以前把自己
倒吊在天国的门口肋下插着七把长矛是要领悟死之国的文字1 你的塔
罗牌网站竟然叫我倒吊男！

　　湿婆泱泱也没说什么。就看见唐僧在那里埋头苦记：奥玎和湿婆
就版权问题纷争……

　　湿婆尖声尖气的说：那水仙花阿忒罗堤斯自恋也就罢了，还在自
己网站里播映同性恋游行，这明显是不公平竞争吗！凭什么你发达你
就你白阔？？我们印度人可是不少的哼~

　　就这样神仙们在那争吵着。到了最后也没什么后果。大家吃了顿
往母娘娘的蟠桃还是觉得天界的网络现在这样已经很不错了

　　第二天两大新闻网站公布会议内容。一时间来访问者无数……
