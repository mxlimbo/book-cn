桑给曲巴

○浙江　羌塘风


　 



　　—— 献给一位遥远的老人


我最后跃上马背的瞬间
你的笑容
象揉得最可口的糌粑
你说：“再见了，吃虫的。”

我想起给你虾皮的那个下午
你老花的眼神
在这些干瘪的尸首上
停留了无数的世纪

我感到
在你沾满酥油的手掌上
虾皮将大海复活
我的家乡
是你永恒的陌生

喜玛拉雅
南部的丛林
我是每一棵树的传奇
你离海很远
而神就住在你帐篷的隔壁

我手中摊出的
无法计数的生灵
并没有将你激怒
你用一声长啸
平息了让我不安的狼嚎

我在你的火堆边
席地而卧
青稞酒
燃活你一生的故事

现在
桑给曲巴
我又回到了海边
我周围的楼房
比你的森林还要稠密

在这个无法入眠的
深夜
你又在梦中将我唤醒
我在一张处长的任命书的背后
写下对你的怀念
（处长，就是你走五百里，到堆龙看到的那个县长）

今夜
你也许又喝醉了
听不到天上奔马般的飞雪了
明天你又推不开帐篷的门了
你的火堆还热着吧？
