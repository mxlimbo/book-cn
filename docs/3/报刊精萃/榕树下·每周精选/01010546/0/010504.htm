冻死骨（散文诗）

○nizhange



　　　　　　　　　　　　 关于政治

　　献给在世纪末的暴风雪中连最后的呼吸都被冻结的人吗？
　　那么诗人的伪善只是我心上的匕首。
　　剜出的文字是黝黑而眩目的，那些冥界的宝石，因过浓过厚的人
血而芬芳四溢。
　　难道我可以说：被摧毁被冻僵的人并不是人。
　　“如果我们要死，那绝不是猪的死亡。”休斯说，他黑色的皮肤
光芒四射。丘吉尔说，以老奸巨猾的狗的神情，和英勇。
　　“如果我要死，我要先划亮最后一根火柴。”那个小女孩说。火
柴里有天堂，天堂是明亮的，天使们的脸美丽得让人惊恐。这样的脸
在鹰的翅膀上，在镰刀斧子交错的火光里，在大卫之星的护佑中，在
奥本海默的灵感后面，消失于融化于红色的海洋。
　　有一种光等于亿万个太阳。
　　有一种寒冷至少可以冻僵一张脸。另一张，又一张，还有一张，
总有一张……
　　有一张脸被刻在时间的道路上，承载车辄，没有腿的脚，还有徘
徊往返的虚妄的重量。一张被冻僵的脸，没有表情，不会说话。那张
脸上闪烁着时代广场的礼花，庆祝这世界如旧的存在，和这存在的更
新。巨大的广告牌的投影里躲藏着无数张脸，能漂浮的，都升腾了；
被冻僵的，成为土地。
　　那广告是——DISCOVER CARD ：2001. 我的信用卡，可以买来食
物，书籍，CD，衣服，和文明社会的一切归属，再等多久我才能买到
酒呢，走进那些流光溢彩的BAR 和POOL？
　　然后变老。在镜子里独自宠爱皱纹的起伏和心的枯燥平坦，也许
可以独自驾车去天之涯，再因为无路而长歌当哭。
　　于是成人之前的厌倦使我步履维艰，暴风雪连车都淹没了呢，如
果我坐在那里就能看见窗外的一切消失，小小的机械王国输给雪，拉
着死人的手的是雪女，她们的指修长柔软，是一些缠绕于前尘旧事的
简单句子。
　　朱门的酒肉臭了，朱门是我们拥挤着投奔，热情地建设的门，这
门通向永恒呢。蚂蚁说：永恒就是我们的巢穴，水晶般晶莹剔透无可
指摘。美丽新世界里的我们，1984早己过去。可路上还有冻死骨。
　　也许这路本身就是冻死的骨。冻死的骨指引向朱门。我们的酒肉
需要谁付出代价？
　　我们又是谁刀俎上微微搏动的肉？


　　　　　　　　　　　　 关于艺术

　　我的血肉站在门前，肩上是鲜红的GAP 背包，几乎和我的人一样
大。微小的人和庞大的物一起走进大都会博物馆，就在昨天，上个世
纪的最后一天。我喜欢逆流而行，走回历史。
　　历史却只使我的脚疼痛。
　　疼痛的脚踏过木乃伊僵硬的影子，并没有惊醒普天之下最可怜悯
的狂妄和虚无。你好，空壳，用香料填满爱与恨腐朽后的空洞能保存
什么？你好，空壳，你的荣耀被囚禁在异域陌生的建筑里，被猎奇的
眼睛捕获。你好，空壳，我们也会和你一同睡，不过现在，我想坐下
来揉揉我疼痛的脚。
　　我还活着，所以有痛。但阿兹特克人的石像却早已死了，被血祭
的重量压入泥土，入墓三分。那些扭曲的脸，被死亡和恐惧冻僵的脸，
以神的名义展开的屠杀中哀嚎的千万张脸。他们渴望着回到泥土，他
们挖出自己的心脏狂欢于爱情般热烈的死亡。
　　而我们的祭祀是不动声色的，虽然一样致命，我们甚至毒杀了泥
土和空气，让家园也加入这场死之舞蹈。
　　这家园是母亲的身躯呢。母亲是我们被驱赶的神。如果所有的神
话都以弑母开始，那么那些冻僵的脸也没有母亲，这个世界是一个巨
大的孤儿。
　　我们是这个世界上无家可归的人。自戕的命运由路边的冻死骨拼
写，这些墙上不为人知的文字终将实现它的诅咒，如果没有母亲的宽
恕。
　　能够宽恕吗，母亲？
　　而我真是累了。
　　累得走回家乡，大都会里的苏州园林。沿着六臂的毗湿奴和梦境
般艳丽的浮世绘，回家。小桥流水，断肠人在天涯。我的舌头可以盘
旋卷曲成任何形状，却只有一朵隐密的花开放在唇齿的后面：清风明
月本无价，近水远山偕有情。
　　沧浪亭里的浮生早已不止六记了吧，皮市街上已没有旧书摊，风
尘翻不动那些柔软的笔记。纽约的天空下，芸娘们的灵魂有时候会和
莫迪格利阿尼被忧伤扭曲身子的裸女一起喝茶，抚慰加莱义民伤痕累
累的身躯。
　　而我终于坐下，想念初春时的柳色和樱花。吹梅笛怨，染柳烟寒，
在这世纪末的冬天，暴风雪在天边耿耿相待。浮生里可记得路边的冻
死骨？把美丽筑成牢笼的人能否看见路边的冻死骨？
　　这一切的美丽几乎要把我冻结。隔绝于世。在这富丽堂皇的一重
重殿堂里。
　　我们的历史。艺术和诗。强力与战争。
　　被冻僵的脸没有表情，更不会说话，没有历史，没有艺术和诗，
没有强力和战争。
　　它们一无所有，甚至没有自己。
　　如果我要死……


　　　　　　　　　　　　 新年快乐

　　那一夜，遥望着窗外的暴风雪，朋友不解地说：政府有提供SHELTER
的呀，他们为什么不去，反而在街头找死？
　　我也无法理解，我以为活着是最好的选择，除此之外别无选择。
也许死亡也是一种诉说。
　　这声音也在说：新年快乐？
　　会有新的一年吗？新的一年是不是那道朱门呢？新的一年里诗人
们还能心安理得地写诗吗？
　　有人告诉我：时间不是射线，时间是一张网，交叉小径的花园。
　　如果我要死，我要死在时间的外面，一切物理定律的外面，神的
手掌的外面，死得孤独而壮大，成为一张恐怖的脸，那被冻僵的逼视
永远被你们踩在脚下。
　　而你们无处可逃。
　　听啊，这是我的祝福呢：新年快乐！
