草或一种普遍孤独的题目

○北京　iamcown


　　草，有着各自随土而居的面孔和细微芜杂的肢体，不过在服用着
语言的流食以护理自身和外部关系的人们中间，依据一种恒定的便利，
当他们看见一些低矮而又无处不在的身躯，他们就不动声色地把它们
呼之为草。这种熟练的并不惊动人们内心的命名，是一场消化时空的
漫长战役，是应付生活的一技之长，在这泥地上扎根已久。草就是这
样的一切无所不在，可以随意处置的岁月和地点。
　　当我们看见草丛中的石块，或者是石块周围的草丛，我们无法说
出原先的秩序——是先有石块孤零零的额头的驾临，然后稚嫩的草籽
从周遭萌生；还是一面平静的草场，而几块粗暴的石头从逻辑的深处
砸下，在我们的观念之中强占了一席之地。
　　我们其实无法描摹出一株具体的草，因为草在草中，它们之间谁
和谁你终将无从分辨。作为想象中的个体，草是不存在的，这不能等
同于你拔出一株草并把它放在手中，草还在那啊！
　　草不以个体取胜，而是以手挽手攒密如林的生殖之墙作为屹立不
倒的根基。“野火烧不尽，春风吹又生”，偷梁换柱的诗人巧妙地给
出一个虚衔而俘获了完全不同的生命，正如死去的是父辈，而我们却
把英雄的光晕赠予了并不在场的儿孙。虽然一茬茬个体的极尽脆弱并
不能瓦解种族绵延的庞然怪异，然而烧成灰的草和新生的嫩草毕竟不
同，即使实际上它们也终将并无二致。不过，从中我们也深悉了我们
的用心，那就是我们需要偶像，然而我们不管他实际应该是谁。
　　草一样汹涌的战马和草一样茂盛的王国，涂抹着鲜亮的色彩，花
团锦绣，帜旗纷扰，从草中喷涌而出，活动于我们内心渴望的场所，
犹如一次尽兴的假日游行；它们又钻入草中。草拥抱了那些曾经斜眼
蔑视﹑相互钝砍的血肉，围剿了耻辱的残尸和干净的头骨，给它们消
失，仿佛一次草的清洗。草如此宽容，并因为宽容而弱小，普遍而无
可挽回地弱小。草仿佛是植物中的蚂蚁，它和气﹑谨小慎微地挪出空
地，又耐心地扯上幕布。草正是一座殿堂，汲取了神喻暗示的出发者
和在神邸前身心劳惫倾心低诉的归来者跪在了同一个地方。春天的野
草也终将被春天的野草沉没。草就是这样的一次一次再一次的覆盖，
刷白了一页书纸套写上另外的字体和段落，草因而得以遵循一个时间
之外的法则：重复重复再重复。因此，当草从你的脚下向远方蔓延吞
蚀直到眼睛的尽头，你也可以说它是在原地不动。

　　诸如此类自以聪明﹑骄矜夸耀﹑得意洋洋﹑故弄玄虚、不理不顾
的比喻和议论，无论如何我们还可以给出许多，但是现在，我却突然
想塞入完全无关的部分，至于理由，我确信如果我无法及时交出，最
后大家会替我想到，而且不止一个。不过基于长久以来的惰性和所受
教诲的根深蒂固，我还是不由自主一往无回地窜入了细琐求证的废旧
鸟巢，继续我偏执的巢中漫步，而最后无意惊起的一只鸟结论则使我
本身也感到了惊慌，这就是：作者根本无需为读者负责！因为对应地
也并没有什么特定的读者为那一位作者负责。写下和阅读是两条平行
的跑道，而我们常常面对的是只有一边选手在场的比赛，它们的记录
只能各自创造。当我们剥去两者合拍的封面，得到的是人，如果这些
暂时充当了一定角色的往来之客，对于人本身拥有的自由有足够的自
信，他们完全可以相互择选，犹如一次热恋抑或分手。但不幸，我随
即就摸到了一个足够大的漏洞，即在时间所允许的长度之内，恰恰正
是作者自己最没有选择权，除了幸运的恋人——情书的作者和那些威
权的伟人语录，又有谁能有幸指定他或它的阅读佳宾呢，谁能把他的
读者定位有效落实到岗位上呢？因此，我在上面满怀自信和逻辑严谨
的野心所抛出的这个合约，它的一方几乎是空的。你看，我其实什么
也没有说出。
　　幸好，在此肇事之初我总算已经攀附到了一个面子十足的理由，
原先它等同于即使最为彻底的革命也和过去的历史保持着大量暗中的
妥协，现在它则成为叙述能得以残喘拖长的要命的母奶。这剂母奶就
是：为了创新。虽然二十世纪末的创新已经发展成一间专门贩卖百年
旧货的老店，然而陈旧正是陈旧得以陈旧的理由，也就是说，它还仍
然风韵犹存。你看看，我已经为自己的意图妄为埋设好了理由的无赖。
理由就是为假想敌树立一个预设敌，就是在两壁悬崖之间放置一条河
流，使阅读看起来似乎能够渡过，不至于是一道让人望而却步的深渊。
而我则得以逃身事外，进入叙述的幸福的掩体。

　　这个中午的阳光优秀怡人，窗口外，在醒目的光线中爽爽傲立的
法国梧桐仿佛披挂上片片黄金碎甲的高大武士。美好的阳光美好到了
如同这时代的大多数杰出事物一样，表现着可分析、使人晕眩的数字
的精确和网络的紧密。在阳光下摊晒开我散漫无忌的感冒（生物体的
原始落后的主要证据之一），就相当于享受着一部舒适体贴的时尚健
身器具，而自然也就是一座最为现代的大健身房，了。我想人们现在
所谓的回归自然大致也就是这一小点意思，即在自然那里发现了又一
种新鲜的用途。实用自然，或自然机器化。神果然已经死无葬身之地。
　　我的目光或者说我的笔触从窗外收回，摸索到了书架上——其实
在这里书架是虚构的，充当着所谓的应有的铺垫——无论是已成后古
董的书架，还是换上了当代新装显得更便于进出的书架，一本本互不
相识的书，一张张字纸的粗暴的集合体，面容和睦地在一起挤，其中
都是已经死去的人物和故事，整个布置正犹如一块整饬而富于礼仪的
墓地。当然，也有占据着正在喧嚣的人物和路径的，不过那大致看来
已经等同于一个工具箱，与书架无关。但是谁又能说那些在街上自如
地安排着步脚的人们，那些无动于衷地穿过家具径直向床走去的人们，
不能够象书架里的书本一样平和相处，就象现实生活中实际的情况那
样呢？一排排气宇轩昂的高楼和在其中暗自搬运的人物，早经证明了
这一点。这种证明还将长期下去。

　　一回生，两回熟，第三回则群众彼此不知为何的默契反而要求作
者照着原来的胡作非为一路写去，也就是说原来的胡作非为也大抵成
就为了一份传统，但是由于我们尊奉创新，因此既然前面已有先例，
以下我们不妨故伎重演，但是拒不进入第三回。

　　 我说
　　 白昼和黑夜
　　 是两口同样漆黑的深井
　　 并肩耸立

　　（你说什么？)

　　 我说
　　 食物在嘴里的安居终于
　　 营养幸福

　　（你说什么？）

　　 我说我不说什么

　　（你说什么？）

　　（你说呀，说呀？你总得说些什么吧！）

　　这么长久以来，我们抛离主题随意漫游，我们快乐吗？现在，关
于草的美好记忆终于来到我的心中浮想联翩：

　　 草没有温度。
　　 草是乱哄哄的四处为家的幸福。
　　 草茎在阴影中欠伸了一下脚趾……
　　我把以上这些愉快的邂逅视为人生遭际中的不期而遇。你瞧，即
使经过了那么多的枝蔓横生，最后我们还是回到了草和草的叙述，虽
然此前我们竭力避免重复，以至于在叙述中恣意妄为；所以在这里我
们也应该可以对以前的所作所为保持心平气和。虽然这里面隐藏了一
个更加让我惊慌的暗示，那就是不论我们理想如何，最后都是一样，
草草收场，不过在此时此刻且让我们心平气和。
　　当然同时，与草无关的杂念也即刻掺和进来，它们的生命力也和
老草茎一样的坚韧，但是既然我们已经约定拒不进入第三回，那么以
下我们就只有滑向结局。结局就是切断飘忽的思绪而没有任何叹息。
　　回想最初，当我们写下这个神经兮兮的标题（它如果改成“草”
　　“草或一种辉煌的题目”或“草或一种普遍的题目”又或其它各
种字句又怎样呢），我们写下它，就象随手丢下一粒草籽，就是给出
了它肆无忌惮的生长力。我们看着它跌落地面，然而它的出生我们则
无从知晓，即我们无法确知它命中将如何开始，因为一开始，它就已
经开始了。这种情形无疑也符合现在我们对于结局的眺望。
　　那么我们究竟应该，如何，安顿一次叙述的结局呢？

　　当我们站在叙述的终点，是否就是站在了人生的终点。我打量着
这些松垮﹑缺乏足够睡眠的句子还不如一根草茎，它们犹如黑乌鸦停
留过的树枝，缕缕神秘的颤动早随鸦翅而去，它们努力的姿态再也等
不来东西，它们在我的逼视之下发窘。
　　让我们再一次回想最初，似乎一开始，我们就应该制作一个宏大
严肃的主题，诸如黄土河脐带﹑大海故乡﹑地球家园﹑宇宙的无极之
列；虽然我们的身高和手脚的伸展不过只在一两米之内，我们微渺的
情感也只在我们有限的视线之内发生，但这些大气﹑气派的立意显然
更切合伪个性时代健康条例的号召。不过我仍然为我的题目感到满意，
因为它足够大也足够小，这样说其实是说它大致可以似是而非。
　　我怀念第一个撒下草籽的人。
　　我想象寒冷，粗砺的史前，一根偏执的草茎哆嗦着昂起挑战整个
地球荒原的壮举。
　　撒下草籽最初是什么样的一种想象力和愿望，草茎四处勒紧了地
表。
　　草生处处，湿黑的泥地上恬然的青春和石隙中不懈的停留，似乎
说明着某种臆想的本能，而其实它们的存在和由来我们永远无法了然
于心。
　　不满是草随地生气勃勃的怒气，我喜欢看见雨水拔出草尖，刺一
样倔强掀开泥粒，草进一寸，路退一寸。虽然不久路上的人们又会使
路面稍稍拓宽。我不知道铺满路径的地面会是怎样，但我能想象出一
望无垠的草原。
　　一切远路在草中探头探脑，虽然它们之中有些远溯到了我们的视
线之外，但没人能肯定它们一定命中靶心。无数的远路们静静潜伏，
时睡时醒，因此我们又可以说每片草丛都是人所遗弃而又从未到过，
或叫历史。历史就是一车碎瓷一路抛撒。在丰茂的草丛，它们满地皆
是，它们在大风中在草根深处天上寒星般眨闪，我们只能就着史籍和
文物这些风偶然吹开的隙缝，短暂地窥视到它们意味深长而过时的眼
神。草其实是植物中的狼，路的宽窄是它和猎物之间对峙的距离。
　　“所有过去的事件，是一场和青草的争夺”，草和历史相伴想生，
草注视着每一栋伟大的建筑在最初奠下根基，然后留下了草中的废墟，
留下美。我喜欢古希腊和罗马那些留驻在亚琴海和地中海岛屿上的遗
址，昔日金碧辉煌的殿穹消褪了嘴边夸眩的油彩，它们的姿态也从而
剔除了原来总是化解不去的几何的僵。巨大的石柱和门廊已经分断成
一截截简洁的线条，坐对着海岛跌荡起伏的细草，它们已不再需要争
斗，它们坐在一起海风吹拂，晕黄的夕阳熨贴着和缓的斜坡大片大片
倾泻，背后是蔚蓝清晰的海。
　　但很奇怪，有许多例如象中国式的雕梁画栋的木质建筑在野草的
啃齿中却只能显得荒废迷离，一派颓圯，鬼狐出没，完全丧失了“天
人合一”的炊烟脉脉的温和。
　　“暮春三月，江南草长”“池塘生春草”“晨起理荒秽，草盛豆
苗稀”“青草池塘处处蛙”“独怜幽草涧边生”“浅草才能没马蹄”
“朱雀桥边野草花，乌衣巷口夕阳斜”“荒草凄凄鹦鹉洲”“长亭外，
古道边，荒草碧连天”“天苍苍，野茫茫，风吹草低见牛羊”“林暗
草惊风，将军夜引弓”“细草微风岸，星樯独夜舟”“一川烟草，满
城风絮”……清新？天然？幽怜？依依？苍茫？雄浑？力量？沉思以
及闲适等等？再没有比草在中国古诗中更丰富的生长，它们在我浅薄
的脑湖中飞雪般蜂拥而至，脚片纷纷。
　　然而当我试图从中挑拣出草和废墟和解的诗句时，我失败了。至
少没有象上面的诗句那样血液般自然地输送到了意识的门口，也许在
汉魏的作品里会有一点，但对于随手翻翻还看不到的东西，我就这样
偏激地想：也许它们本身真的是无足轻重。而且我更讨厌如果当我们
相遇，它们这样地对我介绍自己：咳——咳——咳，你好，我是古诗
——名句！所以，我的印象中便只有黍离之悲，只有“枯草衰杨，曾
为歌舞场”（注：这一句也是模模糊糊，查了书才清楚），就仿佛现
实中荒草深深长满了谁家的庭院。
　　“春天就是草长出来埋住了啤酒瓶的季节”，这才是我喜欢的诗
句。
　　然而草无所谓繁荣，没有香丘，只是一个个土坡的盘踞。“草草”
收场，这是多么紧张和深刻的表现得最好的诗——不是潦草的“草”，
而是绿颜色﹑有水﹑一掐就断的真正的草——我眼前迅猛涌过大面积
青草的野生部落，使我想起一种爆炸，它们盲目的疯长在我眼中映放
着那次毁灭性大爆炸的阴霾。
　　深夜的房间，我坐着，听暖气管在四壁汩汩地不断吞咽热水，仿
佛身体被遗落在一处语焉不详的水底。我想象野草拍打小手，带上它
们的小花和行李，密密麻麻抢过公路，跨过沟坎，跳过围墙，哗啦啦
冲进了屋子，争先恐后，草深及膝。自然又含有了童稚的乐趣。
　　如果我不想以天空作为屋顶，那是因为我想盛放别的东西。一切
皆草。所有饱含草汁的造物。草疯子。当上帝拔出一株草，把草扔掉，
用它根部的泥土开始造人，开始他的叙述，他的身旁一定斜靠着一个
小丑，丑角的冷冷旁观的眼神和嘲讽，和着自由的精神和苦草汁融入
我们的身体。我们的盲目和草有得一拼，而我注定要归于静默。上帝
的叙述能否象草一样自我增殖，我喜欢看到本文的叙述象草一样自我
增殖。我喜欢看到，它们各自沾带上了一点点的光的微芒，看到它们
象草一样，自顾自地从不同的方向抢占了后窗下偏僻的那一小块湿地。
　　赞曰：精心编织显而易见，热心揉合也势所应当，然而矛盾﹑歧
义﹑前后抵触、互相峥嵘等种种错谬至今则还是无法避免，形而上的
弊病一览无余。
　　对了，我们说的不就是草吗——草就是让你感到饶有兴趣而最终
却一无所得。
