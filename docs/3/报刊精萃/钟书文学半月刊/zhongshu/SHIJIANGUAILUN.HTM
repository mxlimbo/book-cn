时间怪论
黄笑天
    很久以来，在我的思想中总有这么一个疑惑：时间是什么？时间从何时开始，又将在何时结束？如果时间是在宇宙大爆炸后产生的，那么大爆炸前又是什么时候？如果时间将在宇宙坍塌后结束，那么坍塌后又是什么时候？一个个问题困扰着我，无论我怎么想都得不出一个合乎逻辑的结论。可有一点我是知道的，那就是我的时间是从我的出生开始，又将在我完成生命旅程后结束。
    自从我懂事那时起，我就总觉得身后有样什么东西在追赶着我，鞭笞着我不断前行，那就是时间，那个随着一长一短两根柄的旋转而不断流逝的东西――即使不转，也照样流逝。年轻的时候，总觉得时间太多又过得太慢，那是因为自己总认为时间离自己还遥远；步入中年后，人就会逐渐意识到时间的存在，但却总抱着一些不切实际的幻想，幻想时间会停止，而生命能永恒。然而，时间的脚步是不会停的，它不管你是否在意，都会紧追着你，时刻不停；人一旦迈入老年，就一定会十分注意时间的流逝，因为他们知道，时间离他们的距离已经很近，幻想是无用的，只有更快地跑，才能不被追上。但有些人会有不同的想法，有些人希望能再多享受享受，有些人希望使自己的人生更加丰富多彩。最终，任何一个人都将被时间追上，相同的是他们的肉体生命都完结了，不同的是，有些人的精神生命将延续。
    时间啊时间，你究竟是什么？或许，人生就是一场和时间的赛跑，明知跑不过他，却也一定要拼尽全力，实现自己的理想，而如果有人能赢得这场比赛，那他就超越了自我，领悟了人生的真谛，但这世上又有几人能够做到，或许会有，但或许永远也不会有。
