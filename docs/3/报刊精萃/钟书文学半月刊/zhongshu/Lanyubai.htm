蓝与白
　　记得君送我的第一个生日礼物是一个扎头发的蝴蝶结，那时我是短头发，我笑着对她说："我没头发可扎啊！"她揉着我的短发说："喏，这不是很多吗？"于是我俩一起放肆地哈哈大笑。那个蝴蝶结是蓝色的，我最喜欢的水蓝色，柔软的蓝色缎带闪着迷人的丝质光泽，蝴蝶结上没有任何装饰，没有俗气的小花，也没有仿真的塑料珠子，简单的可以，可是却非常别致，到后来我留了长发也一直舍不得戴，我把它放在玻璃柜里最显眼的地方。到现在我都一直很佩服君，她总能猜出我的心思，她知道什么时候的我需要鼓励，什么时候的我应该狠狠给几个"毛栗子"，就像妈妈一样（她本来就长得老成，有1.75米高）这大概就是了解对方吧.记得我问过她怎么知道我喜欢蓝色,她回答说："因为我喜欢白色，蓝与白的搭配是最和谐的，还有……你有好多东西都是蓝色的！"啊！原来是我自己暴露了。不过，后来我一直记得她那句"蓝与白是最和谐的搭配"。
　　我爱蓝，尤其是雨后天空的蓝色，洗过一样，仿佛与世界浑然一体。我爱蓝的自然， 因为最美的蓝色绝非人们可以调配出来，哪怕是最高明的画师，用最上乘的材料。最美的蓝是晶莹剔透的，是无暇的，是纯大自然的手笔透过这如水晶般剔透的蓝，我看到了真挚的良朋，她那没有一点刻意修饰的微笑，永远是这样至真，就像……就像那个简单而别致的蝴蝶结，闪着迷人的光泽！
　　她爱白，"因为它单纯，就像你。"她说。哇，好单纯的回答，我可以理解她为什么喜欢白了。
　　后来她去齐齐哈尔念医专，临走时我去送她，还送给她一条劳技课上编的围巾，白色的。我自以为很满意的杰作，她劈头就是一句："你织的？好丑哦！"我大叫："喂！这可是我的杰作啊，你就算安慰我，也该说好看吧！"她立刻做投降状连声恭维："好看，漂亮，美极了!没想到我们笨笨也会织围巾了。"于凭借身高的优势拍了拍我的头。都说离别是痛苦的，可我俩的离别却欢声笑语，望着远去的列车，我又想起了她的话："蓝与白，最和谐的搭配。"
　　蓝于白，我和你。
