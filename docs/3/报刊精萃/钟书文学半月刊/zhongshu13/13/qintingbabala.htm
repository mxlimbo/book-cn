内容标题：倾听巴巴拉

文/伊一

　　近日，从朋友处借来一张CD--巴巴拉·史萃珊演唱会的录音。细细品来，很是精彩，很是感人。

　　精彩的是她真实的歌声。巴巴拉·史萃珊从小在百老汇泡大，没有系统得受过声乐训练。可是，一生歌舞却将她的歌声锤炼得异常真实。未经雕琢，如古拙的玉璞；不曾筛滤，似涓细的山泉，一丝一缕的枯哑和黯然都被麦克风展露无遗，恰似夏末好友谈天说地时，忽逢一阵夹着痛苦低吟或无奈呜咽的凉风，袭淫人心，吹皱明丽的心湖，泛起粼粼的微薄，荡漾开去。但是，若只用枯哑和黯然来支撑她歌声的真实性，是决对不够的。组成她歌声的另一部分，便是她惊人的控制力。那时天然的控制，是夏日傍晚狂风暴雨后悬天而筑的七彩虹桥；那是极有风度的控制，是淡妆素裹的美人回眸莞尔，很简单， 很自然，却很到位。这就是她真实的歌声，无故作姿态，而绚烂无比。

　　感人的是她深情的演绎。巴巴拉的深情是从心底长出的一双冰凉的小手，直伸到你的身体里，触摸着你的心，让你浑身莫名一颤，继而揉动着，撞击着你的心，让你热血沸腾，不知不觉地，那双冰手融化了，和你融为了一体，也留给了你永恒的震撼。唱片里有一首歌叫《Father》，歌曲的开头是一阵呼唤：Papa, can you hear me? Papa, can you see me? Papa, can you find me? (爸爸，你能听见我吗？你能看见我吗？你能发现我吗？) 。看似简单的几句话，放在歌曲里，要唱出那股味道，靠的是功力，是感情。巴巴拉唱出了这种寻常而深蕴的情感，然我久久沉浸在一片悲凉和萧瑟的音符织成的迷离的世界中，久久地，去感受那本该属于我们全人类的未泯亲情……这就是她深情的演绎，无刻意追求，而动人心魄。

　　 我想，大凡艺术都可以分两类。王国维在他的《人间词话》中就把词分为二类："有造境，有写境。此乃理想与写实二派之由分也。"歌唱也一样。巴巴拉就属于"写实"类的歌唱家。她是伟大的。她的伟大在于她对歌唱，对艺术，对生活的忠实。她总是从生活中的细枝末节开始"着墨"，在所造的"合乎自然之境"里，"书写"人生真谛，"记录"处世哲理，"勾勒"灵魂轮廓。

　　 巴巴拉近日获得了个来美的"终身音乐成就奖"。此奖之于歌唱家，犹如西敏寺的墓碑之于诗人，是之高的荣誉。巴巴拉一生不求名利，但到了晚年，拿一个"终身音乐成就奖"，还是无愧的。
