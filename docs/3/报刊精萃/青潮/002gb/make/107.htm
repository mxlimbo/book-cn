情长路更长

嘻嘻鱼/文

　　流行音乐版只是这个无所不包的大BBS一个不起眼的角落。按多数虫虫的爱好，他们想讨论港台歌曲，而斑竹的口味比较另类，喜欢欧美风情。两者之间的矛盾一直没有得到调和，反而呈现越来越激化的趋势。于是虫虫们的帖子进不了精华区，而斑竹的长篇也往往得不到响应。这里就这么荒芜了下去。斑竹自己也十天半月才来打扫一次战场，幸好这个BBS家大业大浪费点没啥，管理员也没发现我的渎职行为，对的，就是我，我就是这个倒霉的斑竹。

　　又到了一周一度的"大扫除"时间，我漫不经心地点开我的版，准备对那几个老水龙头展开例行的砍杀。却发现这里又多了一个新名字——"玫玫"，看看资料，是个female。这家伙，倒的水还不少。而且全是我所憎恶的港台歌曲的名字！什么"最浪漫的事"啊，"等爱降落"啊，不就是帖帖那些歌词嘛！我看着看着气就不打一处来，我砍！才不管她是什么"美眉"还是"霉霉"呢。战场处理完毕，我擦擦大刀上的血迹，扬长而去。

　　本来这一走就该一周后再回去，但鬼使神差般的，第二天我又打开了这个网址，自己也不知道是出于什么目的。有几分意料中的，看见了那个"玫玫"的抗议文章。很长的一篇，几乎是声泪俱下地对我进行了控诉。鉴于她那些感性的废话太多，还有一串串触目惊心的感叹号和问号，这里就不让读者来接受第二次心灵拷问了，（至少当时我想到了这个词）大意是她把她辛辛苦苦写的歌词随感（哎呀，原来不只是copy的歌词，还有感想，我真混！）放心地帖在这里，也没想有什么人来看来回，更没想偷着灌水挣经验值，纯属小资产阶级妇女（嘿嘿，我加的）的自娱自乐行为。怎么就惹着版猪了（这是她在愤慨中打下的字，估计不能善良地认为是键误），让她积攒了这么多天的心情日记化为乌有（唉，我说这位女同志，讲话得实事求是吧，怎么就变成多天了，按我的删帖日程安排，也不过7天嘛。看你哭得那惨样，算了，这小处就不和你计较了）。最后，她也知道删掉了的帖泼出去的水，收也收不回来，只是仁慈地要求版猪（又是版猪！）写一份大小不少于2K的检讨书，体裁不限，务必情真意切感天动地，否则定要去管理员处讨个说法云云。我的妈呀，怎么秋菊打官司打到BBS上来了，看来哥们这次是栽定了。

　　这检讨书真是打小就没写过，（开玩笑！我这么听话懂事的三好儿童怎么会留下少年时期写检讨的悲惨记忆呢？最多不过是写了就忘而已嘛！）我在电脑前垂手而坐，严肃地盯着她的控诉书，突然想到是不是该焚香沐浴一下先，以净化肉体而净化灵魂达到下笔如有神的理想境界。自己打了自己一耳光（作作样子呢，别激动），龇龇牙，深深吸了一口气，展开"弹指神功"，屏幕上便"刷刷"地出现了这么一首……似乎也不能算诗的玩艺儿：

如果把整个杯子里的水都倒出来
也浇不灭我对音乐的喜爱
整个杯子里的水倒得出来吗
可以 所以 我喜爱音乐
如果把三根火柴一起擦亮
也点不燃我对庸俗歌词的热情
三根火柴可以一起擦亮吗
可以 所以 我对庸俗歌词依然没有热情
综上所述
热爱音乐的我，只有删除满是庸俗歌词的帖子
但这能成为一个我不尊重女士劳动成果的理由吗
不能 所以 请听我说：
曾经有一份字字珠玑的帖子摆在我的摊上，
我却没有去珍惜，
直到删除掉了才后悔莫及。如果上天再给我一次机会，
我会对那份帖子说：
"我看你！"如果一定要在这个动作前加一个期限的话，
我希望是——"一万年"！

　　虽然两个套子都已经是又老又滥，虽然我坚信就算是看一百个像鲁讯他老先生那么深刻的帖子也用不了一万年。但我还是深深呼出了一口气，按下了"发送"键。

　　第三天当然得去看看赎罪成果。她的回帖很简单："认罪态度不怎么样，方式还可以接受，现将死刑改为无期徒刑，劳改期间，请认真回玫玫女士发的每一份帖子。"最后是一个"：）"，可我直想把那括号扭过来成"：（"。我招谁惹谁了，非得把我捆这地啦？思来想去，不管动机如何，结果总是繁荣了我这地，其实还该我感谢她。哎，这世道。

　　然后就开始了我漫长的跟帖生涯，每天去版上收集她的帖子，再找些不痛不痒的诗句帖上去。她也还是老毛病，抄些歌词上去，再写一大段的感想。也许是我找的诗还差强她意吧，她总是又继续re我的跟帖。两个人就这么眉来眼去的，其中的猫腻很快被那几个老水龙头窥破，跟在后面起哄说清高的版主终是耐不住寂寞下凡来了。我不耐烦地把大刀挥了几下，也没把他们宣传桃色新闻的热情镇压下去。哎，不过就是个网络游戏嘛，犯得着这么认真么，爱谁谁去吧。网恋这么过时而庸俗的东西，本人没兴趣玩。

　　照例的一天，照例地看她的帖子。《情长路更长》，是一支很老的歌了。说实话，我对这些港台作词人的无病呻吟真是腻味透顶，每次都是瞟一眼标题，就接着看她的感想，看完了就ctrl+c，ctrl+v跟完了事。

　　今天，似乎有些不同。她写得很短，就一句话："每次听到这首歌，就想起当年和他夏夜漫步的情景，原来，真的是《情长路更长》"。没了，就这么没了。这是第一次见她的感想中出现她自己的感情生活。

　　心里突然不知道什么滋味，像是突然发现自己宠爱的女人原来其实是在向往另一个怀抱。什——么？她？——是我宠爱的女人？我为发现了自己心底的秘密而不禁一栗。
至于我是怎么进了社区的聊天室又与她同时开了对方的小窗，约好见面的时间地点之类的细节问题，我都记不太清楚了。事后回想其实好象是她先提出来的也没凭没据了。

　　只记得我见到她时心里一嘀咕，怎么这庸俗的网恋故事净发生在我身上了，约出来的网妹一个赛一个漂亮。（您可千万别嫉妒我这运气，谁叫本人也略有三分姿色呢？哎，大老爷们说这话，该打！）

　　她很瘦，长长的头发随意地披在身后，从这点可以看出她不是那种写字楼里中规中矩的白领丽人。她的神态有点拘谨，说话时脸还会稍稍一红。大概是因为第一次见网友的缘故吧，我是这么想的，她也是这么和我说的。

　　一边散着步，一边谈了很多。从她的庸俗歌词（当然她拒不接受这个形容词）到我乱帖的诗句（她居然能说出那些诗的作者和出处，当时让我心里暗暗傻眼），从五光十色的网络文学再到最近《读书》杂志那边的混战。我发现她对这些懂的居然一点不比我这自命不凡的自由撰稿人少，不禁同情了自己一把，人业余的都这样水平了，还要我这专业的干吗呢。

　　后来我问她怎么不去文坛上搅搅混水，她说她写的文字只能是一个个破碎的片段，就好象是在歌词里写的那些感想。有点儿念头就写点儿，终是成不了文章。
"只有我想写的时候我才写得出来，我没法和你比，可以逼着自己写。"

　　她说这句话的时候，已经走到我的前头去了，她边说边转过头来很羞涩而又很妩媚地一笑。我突然有种很奇怪的感觉，和着她的外表，神态给我的印象，她的闲散中藏着一份娇慵，一种不食人间烟火的娇慵。但这个念头只是一闪而过，因为我不由自主地吻住了她，她的身体在我的怀抱中倏地僵硬了一下，却又立刻变得柔软。脑子里什么也没有了，身体逐渐变得轻盈，轻到不存在，轻到可以挣脱一切的束缚，飞上蓝天，与白云星星为伴……

　　接下去的发展与市面上流行的庸俗网恋小说别无二致。我们一起去动物园喂猴子，去咖啡馆听音乐。累了就回到我的小窝自己做饭吃。当然，最经典的节目还是晚上一起去散步了，每当这时候，她总会哼那首《情长路更长》。夜很深了，我们还这么走着，沉默着不说话，只有相拥的肢体传统那一份默契与感动。

　　但令我不安的是，我对她的基本情况可是说是一无所知。我不知道她的联系方法，除了email和icq,我们总是在一次约会结束时约好下一次见面的时间地点，而如果她有事，她会打电话给我，再在网上作出新的约定。我不知道她在做什么工作，除了文学和音乐，她似乎什么都不谈及，而我又几乎可以断定她不能靠文学和音乐谋生，那么，她是靠什么来维持她的名牌衣服和香水，就是一个很大的问题。甚至我不知道她的名字，我一直叫她"玫玫"，也没有刻意去问过她的真名，她也没有主动和我说过，我不用打电话给她，真名的重要性就这么被忽略了。

　　这就是说，她可以随时消失在我的生活中，像一场新世纪的网络迷梦，而不是一场可以真实把握的爱情。这样的想法一旦深入，就几乎令我发疯。但我不知道如何去问她，因为每当我问及与这些有关的事，她总是迅速地转开话题，日复一日，我更加没有询问的机会。

　　尤其是最近的约会她老是要临时变动，在一起的时候也心神不安，稍稍时间一长就提出要回家了。而我每次送她回家的提议均被各种各样的理由驳回。

　　又一个下午，她又走了。我烦躁不安地在屋子里走来走去，把一只脏兮兮的毛毛熊踢着撒气。我觉得这样的关系肯定不对劲，可又无法和她一起来取得足够的证据。忽然，我的目光落在电脑旁，一只白色的手袋，她走的时候忘了拿了。

　　我盯着它，觉得它好象都比她实在。因为它是没有灵魂的，拥有了躯壳就拥有了它的一切。而她，对于我来说，还是那么飘飘渺渺不可掌握。

　　我跌坐在电脑椅中，拿起这只精致的手袋，以研究的目光注视着它。也许，它能提供一点我不知道的东西。

　　口红，镜子，面巾纸……每个女人都必备的玩艺儿和一个钱包。我有点失望而漫不经心地打开了钱包，想看看有没有她的身份证之类的可以告诉我她的真名。

　　刹那间我像被电击了一下，拿着钱包的手一抖，僵在原地。

　　一张照片，她，和一个男人，很亲密的姿势，绝不可能是兄妹或普通朋友关系。那男人看上去三十多岁，沉稳而内敛，她带着一贯的有点羞涩的笑，和一种娇慵。

　　一种不食人间烟火的娇慵。

　　闭上双眼，没有任何思想。直到感应到身旁的呼吸和体温。我有点迟疑地睁开眼，看着她，从未如此陌生过。

　　她退后两步坐到沙发上，很平静的语调，却掩饰不了的颤抖："我都说了吧，你愿听就听着，不愿听我就这么走了好了。"

　　我没有做什么表示，事实是，我脑子里是空白的，似乎已经丧失思考的能力。

　　"我大学毕业后就留在家乡，做一份很稳定的工作，男朋友百依百顺，我说要去散步他一定会陪我。我以为我就会这么嫁了，这么平静地生活了。可最后他说他要结婚了，新娘不是我。那地方我没法呆了，就独自来了这个城市。"

　　"谋生的困难远远超出我的想象，我这才发现我除了风花雪月什么都不会。总是连试用期都做不满就被炒鱿鱼，窘迫到了快要被饿死的程度。可我绝对不想回家，也不想让家里知道我怎么样了。"

　　"和所有的童话一样，他就这么出现了，但他并不是趁人之危。甚至他不和我同居。都是我要求的，否则我心里没法平衡。要是他没小孩的话，一定会离了婚和我结婚。"

　　"这么好一人，可惜我对他没火花。"

　　她自嘲地笑了笑。我拼命压抑住想哭的冲动。

　　"他们一家去欧洲玩了一个月，认识了你，我知道我不对可我不敢给你说，我知道我是在做一个梦，而只是在尽力拖延醒来的时间。"

　　她将手袋收拾好，站到我面前。

　　"谢谢你，给了我最后一次爱情。"

　　"留下来不可以吗？！"我站起来抓住她的肩头。

　　她轻轻推开我的手。

　　"我没有选择的自由，早就没有了。"

　　她迅速地在我唇上吻了一下，头也不回地走了。

　　我重坐回椅子上，盯着电脑屏幕，任夜色已浓得化不开。打开电脑，连上BBS，选中流行音乐版。

　　仍是有几分意料中的，看见她新的帖子在那里。仍是《情长路更长》。

　　"仍是情长路更长。"

　　我按下了"Del"。
