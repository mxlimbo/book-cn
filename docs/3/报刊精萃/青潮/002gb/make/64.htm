《莺啼序》

恋爱/文

因心抑抑，易梦窗原词之韵，
改押平声，另有多处变通。
四野薄雾轻笼，正烟雨江南。
花发竞，姹紫嫣红，夭桃李争妍。
鸟飞群，翱翔自若，欣慰日暖还轻寒。
念庭前旧燕，重来又卷珠帘。
风花潇洒，芳草天涯，三两自寻欢。
何须言、林涛松韵，更兼石上鸣泉。
望寒潭、应羡范公，揽纤腰、五湖舟扁。
天欲暮，云彩阳斜，何处乡关。
往事怎能如烟！纵空负、相思万般。
花含苞、犹待吐蕊，双双对对；
人断旧缘，影孤形单。
浮生若梦，浮缘如梦，
黄粱一枕梦易阑：梦短天长，梦窄春宽。
今夕何夕！月皎星灿风怡，可怜依人梦残。
炎凉世态、冷暖人情，此味亦倍谙。
乐未消、傲骨疏狂。
天降英才，苍生不负，断简休翻。
不学陶令、归去来兮，
男儿当建千秋业，更喜见、大千尽花颜。
自鸣鸾凤清音，七尺岂效、燕雀之喧？！
