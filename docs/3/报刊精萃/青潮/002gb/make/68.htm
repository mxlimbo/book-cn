可爱的女孩

梦江南/文

我可爱的女孩，什么时候你才会露出迷人的笑容
我在金秋的十月，收获黄色的希望
那荡漾的稻谷，和你含笑的眸子
绘织成远空的彩虹，在高耸的云端，象狂风一样激昂

我可爱的女孩，时光不是破碎的梦想
今宵，我们在火红的烛光里交错人生的杯盘
火样的人儿和火样的心情
那怕在风雨之中也摧折不了久违的期待

蓝天和白云在脚下，和天一样高瞻的心
在女孩的梦中留下支离的呼唤
可爱的女孩，你是最美丽的白莲
和明月同样皎洁的身躯引诱着我不眠的黑夜

我已经走不出女孩的陷阱，看吧，这是地狱的火焰
熊熊烈火锤炼着生命的呻吟，我还是一如既往
可爱的女孩，你说过要怎样奉献？
我有动人的眼睛，做你四月的星光

可爱的女孩，你已经开启了青春的大门
看着进出的人，我兴奋的神经开始麻痹
可爱的女孩，这个世界在你的眼中还有什么可以怜悯
我在你寂寞的背影里咀嚼着人生最后的一点依恋
