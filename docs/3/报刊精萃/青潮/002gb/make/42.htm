请给我一个不爱的理由


水晶宝贝/文

　　我养的鸟儿死了，从他身边醒来的那一刻，我想起了那只绿色的小鸟。它挥舞着翅膀在我身边叫唤着，我抬起手去抚摩它，却看到自己抹过一团空气。

　　我就在这样一个灰色的清晨醒来，带着对那只鸟儿的思念醒来。我以为我会看见窗外漂浮的云朵和明媚的阳光，却无意中发现空气中漂浮着粉尘的影子。它们离散地浮在空中，无助而张皇，我想把它们看得更清楚，却发现眼睛的乏力。我的眼睛就在这样一种浑浊的状态中感到隐隐生疼，我试图睁开它们，却感到一种无来由的窒息由脚底冒上来，于是我眯着眼，不再苛求让它们变得清晰。

　　为什么每个醒来的早晨，我都陷入这种莫名其妙的分裂状态。我捏了一下自己，感到一阵真实的疼痛。原来我真的醒着，我微笑着，

　　我了一眼旁边还在熟睡的他，那个赤裸着身体的男人，从隙缝里透出的细小灰尘包围着他，让他看起来朦胧而肮脏。于是我开始用指尖轻轻地划过他的肌肤，细细地把他身体上蒙着的灰尘一一拂去。我看着自己微微颤抖的指尖在那个结实的身体上游移着，带着某些不易见的柔情侵蚀他的身体，但他呼吸仍是有条不紊地进行着。

　　我不属于他梦里的世界，我在现实，与他隔了一个世界。

　　我知道，我不爱他。当每个早晨我清醒地从他身边苏醒过来的时候我就感到极度地厌恶。我精疲力竭，乏力地回想着前一个夜晚的事情，我便无法从容。我知道，我背叛了自己的感情，我的身体也背叛了我。

　　那些寂寞的夜晚，我们在昏暗的灯光下同床共眠，透过那一丝丝似乎能穿越肌肤的灯光，我们无声无息地触摸对方，然后在一片黑暗中我感觉到无数的小虫从我眼前穿越，让我与黑夜交织在一起，然后我象个孩子一般从容入睡。这是我记忆中的一切，朦胧而模糊。

　　我等待着他的苏醒，每个早上我都做着相同的事情。我从来不会披上睡衣去给他煮牛奶和烘面包，他总是笑我不是一个贤惠的女人。因为我总是吝啬于为不爱的男人做事，即使那些事情简单而乏味。

　　回想起那只心爱的鸟儿，那只我唯一亲近的宠物，我把它掩埋在那堆黑色的泥土下，用他和我的手，把它遗留在那个阴冷的地方。我跪在地上虔诚地挖着小坑，我看到自己的双手上沾满了那些柔软而潮湿的粉末，我抚摩着鸟儿那僵硬的身体，不忍放手。他在我的身边默默地站着，看着他的女人掩面而泣，却不发一言。

　　如果你心爱的东西突然间离你而去，你会怎样？

　　我突然转过头去看他，那张冷漠的脸上，没有一丝表情。

　　我会伤心，但只是一瞬间。

　　他冷冷地说出这句话，然后蹲下身子把我手上的鸟儿放进了那个挖好的小坑。一点一点的，我看见那只小鸟逐渐埋没在那堆黑色的泥土里。我摸着那个微微隆起的小山堆，感到一阵刺骨的寒意。

　　这就是跟我同居的男人，冷漠的男人。我不知道他为什么会深深地爱着我，他说我的身上有着一种懒洋洋的倦怠的病态美，我就象一个生病的孩子，等待着他的照顾。他喜欢这种感觉，他愿意把他最好的东西全奉于我。

　　可是他只是停留在我的表面，他喜欢着我忧郁的面容、高贵的举止，却忘记了深入我的内心。每个黑色的夜晚当我看着敞开的窗户扬起窗帘，外面那一片芭蕉的翠绿时，我总是有一种莫名的忧愁。我是个没有未来的孩子，他，从没有给我承诺过未来。

　　那只鸟儿是我从前最依赖的宠物。我厌恶猫猫狗狗的叫嚷声，只是喜欢在鸟笼前一站就是几个小时。我想我前世就是一只苍老的鸟儿，在鸟笼里郁郁过了一生。所以我想给这只小东西快乐，但它似乎是与我心灵相通的，我不快乐的时候，它同样耷拉着脑袋，没有生气的样子。

　　我静静地躺在他的身边，等待着他的苏醒。我一次又一次不断地对自己说，我要停止这种没有目标的生活，我希望得到一种更为健康的方式。我企求我有一具优美而健康的肉体，我迷恋着自己的身体，我希望它与自己的灵魂站在同一边，我不喜欢身体的背叛。

　　在我的等待下他醒来了。七点二十五分，一些没有任何意义的数字堆砌在一起，成了他与这个世界接轨的开始。他转过身子拥住了旁边的我，开始温柔地吻我。我睁着眼睛麻木地接受着这份温柔的情感，我的眼睛里空洞无物。

　　我不知道在这样的时候为什么我总是清醒的。舌尖的挑逗没有让我沉浸在兴奋的边缘，却让我愈加麻木。我清楚地感觉到自己舌头移动的轨迹，它只是跟随着那个男人，失却了自己的方向。于是我停下来，用手摆开他的脸庞。

　　我不爱你。对不起。

　　他吃惊地听着身体下的这个女人淡淡地说着这样的一句话，但瞬间又恢复了平静。我总是惊异于他的这种收放自如的情绪，他是个天才。

　　请给我一个理由，一个不爱的理由。

　　没有什么理由，我感觉自己的空洞，我只是个没有感情的孩子。

　　我们的脸离得很近。我逼视着他，希望从他脸上看到失望和悲伤的痕迹。我知道自己幼稚而固执，骨子里总是希望爱着自己的男人会为自己的离别而痛苦。但他没有，只是他的眼睫毛微微地颤抖了几下，带着一丝犹豫的影子。

　　我用冰凉的指尖扫过他那浓密而卷曲的睫毛，然后把手掌盖在他的眼睛上，深深地叹了一口气，我一定要离开你，真的，原谅我。

　　我前世是一只苍老的鸟儿，我只会在笼里忧郁地死去。我已经厌倦了这种颓败的生活，我要离开。

　　八点，我在他的目光中穿上衣裳，然后拿上自己漂亮的挎包走出了门口。我没有回头，我希望在关门的那一刹那，我把记忆永远丢在身后。

　　我是一只鸟儿，我开始过自己想过的生活。
