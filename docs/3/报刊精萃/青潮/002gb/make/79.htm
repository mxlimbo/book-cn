蝶恋花

昨夜梦里怜相顾，往事重忆，空惹泪无数。
百草千花寻常路，恰是相思肠断处。
泪眼迷离倚古树，满眼苍翠，不知君忆否。
情深难述谁人怜，悠悠梦里无寻处。

天雪
8.2日

花恋蝶

食盐/文

今朝镜前观面目，新茬又起，刀剃须无数。
粗茶淡饭不觉苦，正合叽里咕噜肚。
睡眼朦胧出旧屋，迎面滂沱，谁明我糊涂。
心里顿悟身外故，茫茫旅途天涯路。


蝶恋花，和食盐

天雪/文

茫茫旅途天涯路，行人匆匆，不见旧人渡。
粗茶淡饭不觉苦，他人美馐心却妒。
今朝镜前又观目，眼球红红，钞票不增数。
堪叹今生运不济，怎不如人多福禄。


蝶恋花

太匆匆/文

偶忆凋寒瑟瑟雨，恼尽春忧，泪眼还无语。
梦里依稀魂断去，西江画舫离人聚。
狂酒把欢人散处，醉掩西楼，何处是归路。
愁看人间花乱舞，纷纷飞尽黄昏树。
