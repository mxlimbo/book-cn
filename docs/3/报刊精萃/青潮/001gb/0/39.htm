《泣血的黎明》


　　每当黎明即将来临的时候，我都会被那发自内心深处巨大的痛楚痛的从睡梦中惊醒。事情虽然过去两年了，但是对于我来说似乎永远是发生在昨天....

　　98年夏天，我正在广东一家歌剧院工作。我是一名职业键盘乐器演奏员，由于职业的需要，在必要时也会搞音乐创作。有一次，我接到音乐总监下达的一项任务。为了配合一幕新编创的音乐舞台剧，需要我创作一首能够表达音乐舞台剧思想内容的音乐，我高兴的接下了这个任务。由于搞音乐舞台剧方面音乐的创作需要对这幕音乐舞台剧的主题和剧情有深刻的理解，我决定切身体验一下剧情。我决定学剧中的主人公一样，带着自己的恋人在黎明将临时去山中看日出。

　　我是在半年以前认识的"娜"！由于工作总是很忙，始终没有机会带她一起去玩，娜曾经开玩笑的说我不懂浪漫。现在有了这次"因公"体验剧情的好事，正好借此机会和娜浪漫一回。娜听了以后，快乐的光着雪白的小脚在地板上蹦蹦跳跳，还在我的脸上"盖"了一个红红的"戳"。

　　从我的住处到情人岭有4公里左右的路。情人岭北面临海；东西是巍巍耸立，连绵不绝的山脉；南面就是我们所居住的城市。情人岭树木四季常青，每当花开的季节，方圆几十里都香味浓郁，虽然不是什么名山大川，但也别有一番独特的情趣。上下山的路有两条：一条是山的西南向山顶走；另一条是从山的东南向山顶上走。由于要赶在黎明之前来到山顶，在半夜1：00我和娜就出发了。

　　我选择了西南方向上山的路，这条路相对来说要好走一些，是人工铺的青石板小路。青石小路上只有我们两个人，飒飒的山风和重叠的怪石挡不住娜的好奇心，她拉着我的手饶有兴趣的向上爬着。看着娜快乐的神情，我又高兴有内疚。哎！早就应该领她出来玩一玩，这

　　半年以来可真的把她闷坏了，我这个男朋友真的不是很称职啊！爬了将近3个小时的山路，我们终于来到了情人岭的顶峰——-迎神台，这时正好天色已经有些发灰了。

　　云雾还很浓，遮住了山上的景物。飘来的山风时而拨开云雾，让我们从云雾之间的缝隙看到了郁郁葱葱的树，鲜艳的山花，还有奇形怪状的岩石。娜兴奋的向山间高声大喊：我来了...！大山传来了绵绵的回音：我来了...我来了...。我忽然看到对面山峰上有两棵奇特的不知名的树，互相缠绕着，"娜，快看！那边山峰上有两棵相互偎依的树，象情人一样！""在哪呀？""看，那里"！可是娜还是没有看见。我说"在这个峰顶上也有"！娜疑惑的看着我，然后有些不信的四处乱看。我一下子抱住了娜，说："你看我们象不象情人树呀"！娜一下明白过来了，于是也紧紧的拥着我，我们开始忘情的接吻.......时间就在我们的吻中不知不觉的流过。

　　是一阵大大的山风将我们分了开来。奇迹出现了，风把云雾卷得翻滚成一团，就在这时红日已经在云雾中露出了半边脸！蒙蒙的雾气变成了红色，我们的脸也变成了红色。我迅速将背在身后的微型电子琴放在一块岩石上面，我疯狂的弹着这瞬间产生的灵感，如泣如诉的冥冥之音从我的指间飞向山谷！娜惊呆了，任凭一头秀发被风吹得凌乱的飘向空中。忽然，她伴着我的旋律高声吟唱"啊.....啊.....

　　啊..........."！当红日完全跳出云海之时，我忽然将琴高举过顶扔下悬崖！我疯狂的喊着"这把琴已经完成了它最高的艺术使命，再用它演奏任何乐曲都是对它的亵渎！哈哈哈哈...............！"

　　忽然，我听到娜惊恐的大叫"啊！"我回头一看，不觉立刻毛骨悚然，一只拳头大的蜘蛛爬在娜的腿上！我不顾一切的将这只浑身发绿的蜘蛛抓起来扔向悬崖，附身一看，"呀！"娜的腿上被蜘蛛咬了一个黄豆大的洞，洞的周围已经变成了黑紫色！看着娜痛苦的样子，我立刻扶她坐下，用嘴在她被咬的地方没命的吸着。呀，血水竟然是甜的，有毒！我脱下衣服扎住了娜的腿，想让毒气慢些上走，然后更是拼命的吸着毒液。这时，娜的神情已经有些恍惚，我感觉光吸毒已经不是办法了。于是，抱着她向山下狂奔，并高声呼喊"快来救人啊！快来人！"可是，哪里有人呀！不知跑了多久，我感觉娜的呼吸越来越弱，而我从嘴到喉咙似乎有火在烧，腿已经不听自己使唤了，终于我摔到了！我神智恍惚的抱起娜拼命的向前爬，我知道，只要我向前多爬一步，娜就多一份获救的机会。终于，我也晕了过去.....

　　当我醒来时，发现自己躺在医院的床上，手上还插着输液瓶。娜怎么样了？"娜！娜！"我高喊着拔掉手上的针头，疯狂的向外冲了出去。护士小姐被吓呆了，不敢阻拦。我一见到穿白大褂的就抓住不放，拼命的摇晃"娜在哪里？她怎么样了？！"后来，我被好几个医生一起按到床上，强迫性的注射了一针镇静剂，我又昏睡了过去。

　　等我再醒来时，从我的同事那里得知娜已经永远离开了我！"哈哈哈哈！啊哈哈哈哈！啊哈哈哈哈........呜呜呜呜......."我不知道自己是哭呢还是在笑！我2天2夜没有吃饭，面对着天花板发楞，

　　以后自己是怎么出的院，我自己也不知道！

　　出院后的第3天，我又买了一把微型电子琴，还是在午夜1：00时开始上山，还是在4：00时到达了那座山峰，还是在日出时将琴放到了那块岩石上。"娜.......！！"我的声音在山谷中回响。"我来了.........！我为你来弹这首曲子了，这首曲子里面流淌着你的血呀！！！！只有你才配听它，这也是我第二次和最后一次弹这首曲子了，你细心的听啊！！"。我任凭眼泪流淌在琴上，又一次疯狂的弹着这让我心碎的《泣 血 的 黎 明》！！！！曲终，琴翻滚着落向深渊，那琴声依然回荡在绵绵不绝的大山之间........

作者：富颉明（天雨）
