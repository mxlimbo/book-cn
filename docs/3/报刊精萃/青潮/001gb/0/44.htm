牵挂的心


杨会昌

　　那一天夜晚，甩甩手走出了她的房间，走在清凉空阔的街头，虽然有点儿孤单和忧伤，但仿佛释放了千万公斤的重负，以为从此没有思念，没有牵挂，更没有所谓责任的重压，灵魂的世界可以轻轻松松。经历了短暂的情感的纠葛，转眼又是潇潇洒洒的单身贵族，看来不仅注定要打一辈子的光棍，恐怕对所有女性的感觉都不会死灰复燃了。

　　可是没有牵挂的心总是空空荡荡的，何况没有牵挂只是一种自我的欺骗，仅仅过了一两天，我思念的潮水一浪高过一浪，想到她前天晚上喝酒时豪饮而醉，然后一个人躺倒在寒冷的房间，我的担忧和怜爱油然而生： 她酒醒了不，是不是生病住院了，她那几个结拜姐妹是否在看护着她……尽管我不能确定她是否在乎我一份情意的问候，尽管我甚至缺乏勇气，生怕遭到尴尬的冷遇，最终还是掏出了手机，给那位让我日夜牵肠挂肚、让我寝食不安的朋友挂了电话。

　　"M，你好么，酒醒了不？"她轻轻地说，"早醒了，只是头还有点疼痛。"

　　"为何不出来走走，干嘛一个人呆在家中呢？"

　　"刚刚停电了，一个人无事可做，正准备睡觉呢！"听到她如水一样的诉说，我的心快碎了。那低沉的话语，忧伤的口吻，让我心痛得掉下了眼泪。天哪，一个忧郁的女孩、一种寂寞孤独的倾诉啊，才八点多钟，象我们这些稿写作的夜猫子，就是到了十一、二点钟依然属于黄金时间呀，一个女孩由于挡不住寂寞，在夜幕垂临之时就要入梦了。

　　她不就是因为伤心难过喝醉了酒么，这有什么大不了的呢？尽管她是因为失去以前的恋人而借酒浇愁、失声痛哭，尽管她的一片真情不属于我，我也不能责怪她呀！我们不是说好了的，永远都只作知心的朋友吗？无论作为亲密无间的朋友，还是作为我私下钟爱的女孩，我都应该陪伴她，给她安慰，逗她开心，引领她走出寂寞的天地啊！我为什么就不能主动一点呢，为何有那么多的心理障碍？与她相知相处是我一直期盼的呀，没有了知心的朋友，离开了我灵魂深处爱慕已久的女孩，我满载牵挂的生命已经无法顺水漂舟了。

　　想一想，以前她是那么无忧无虑，快乐得象一只小鸟，我们相处得多么和谐、浪漫，不是恋人也胜似恋人了。相逢聚首的日子，我们端坐在三楼那冰窑一样寒冷的房间，我们的心被珍珠般的话语温暖着，朴素的话题一如家常的白话把人间的苦乐幻化成心河的浪花，一曲友情的高山流水缓缓流淌。每天，聆听她天真活泼地唱着"女孩的心事，男孩你别猜，别猜"，我的心感觉格外的快乐和幸福，这样的红颜知己简直就是红尘世界独一无二的女神。可是，才仅仅一两天的工夫，她已经低落得如此的消沉，前天告别时我都说了些什么呢，为何不大大方方地去面对她的一切。是的，我应该去看望她，不仅让我的牵挂找到思念的归属、让友情的天空彩霞飘飘，也要与她谈谈心，叙叙家常，再说一说我从事的保险工作、她从事的警坛上的风云，并且一定要告诉她："长长的日子大大的天，坚强一点吧，要知道你是一个穿制服的女孩呀，跟军人没有两样的！"最后，当她开始微笑的时候，我就把自己深埋心底的眷恋再度捂紧，叮嘱自己不要伤害她，只是象往常一样看一眼她酡红的脸庞，然后以快乐而又忧伤的复杂心情祝她晚安、与她告别！……

　　电脑写作稿（散文） ：云南省大理州 中国人民保险公司永平县支公司
