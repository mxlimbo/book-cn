走近桂河桥


杨会昌

　　从曼谷出发向西偏北方向行驶120多公里，我们来到了泰国风景秀丽的北碧府。穿过直属北碧的县城，再往北约四公里，越过了面积大约千余亩的"盟军坟场"（又称千人墓），高大雄伟的桂河大桥已经遥遥在望了，历史的沧桑和现实的壮美直落眼底。走近了桂河桥就象回到了悠远的昨天，一段扭曲的历史浮出了水面。

　　落阳夕照，彩霞满天，烘托着柔情的桂河水缓缓流动。看一幅真正江南水乡恬静的抒情，看杨柳依依烟花灿灿，青山隐隐绿树丛丛，一座举世闻名的大桥横贯宽阔的桂河，这就是桂河大桥呀，它的伟岸和精美确实令远方的客人痴迷！但是，抚摸着大桥斑驳的钢铁栏杆，脚踏着自历史深处延伸而来的铁轨，我们知道桂河大桥不仅仅是作为沟通的桥梁存在着的，它仿佛一位饱经风霜的老人，它是悠悠岁月的见证啊！

　　桂河桥是一座接驳铁路桥，加上引桥长约一公里，它始建于第二次世界大战。当年叫嚣着"大东亚共荣圈"的、穷凶极恶的日本鬼子，为了打通直达缅印的交通线，竟以惨无人道的残暴，强迫大批华人、泰人和盟军战俘修建自泰国万磅县途径桂河止于缅甸的铁路及接驳大桥。由于自然环境恶劣、施工工具简陋，加上日军丧心病狂的残酷驱使，致使数万名劳工和战俘在415公里长的铁道线上含冤死去，日军苦心经营的铁路也因此被人们称为"死亡铁路"。后来的人们，为了凭吊这些屈死的冤魂，在距离桂河不远的地方建起了"千人墓"，虽然只是近万座空空的坟茔，但历史的残酷留给我们的教训是深刻的。

　　如今，桂河大桥作为"死亡铁路"的一部分依然完好无缺，它一头连着泰国，一头通往缅甸，乘坐列车尚可饱览70多公里桂河流域内的妩媚山水。我们一行人没有时间沿通向远方的铁轨走回那个风雨飘摇的年代，只能在桂河边款款漫步，站在桂河桥头摄下一张永恒的留影，然后再用深情的眼眸极目远眺――，看到了锦绣的山川清丽而又婉约，看到了白帆飘荡在宁静的水面，触摸到了纯朴的人文景观蕴涵的厚重与温暖，感受到了和平的美好以及春天勃勃向上的热烈气氛。

　　那天，游览桂河大桥的人特别多，在熙熙攘攘的人流中，有中国人、法国人、德国人和瑞士人，但我不知道有没有日本人。如果有的话，不知亲临桂河的日本人有何感受？面对巍然肃穆的桂河大桥，是不是还会有人公然为历史翻案，象否认铁证如山的南京大屠杀一样，把一座默默无语的大桥也推倒、好让浩荡清洌的桂河水淹没他们的罪证？从许多日本人暧昧、含糊的态度上来看，从正义的东史郎的败诉和那些极右分子口口声声说什么要验证南京大屠杀的真伪来看，我相信有那么一些日本人做梦都这么想，而且他们要做的不仅仅是掩盖昨天的累累罪行，甚至还想重温所谓"大东亚共荣圈"的美梦，这不能不引起全世界爱好和平的人们、特别是曾经屡遭日本侵略蹂躏的亚洲各国人民的高度警惕！

　　好在历史是不能够篡改的，清晰的烙印早已把战争狂人钉死在耻辱柱上了，一座桂河大桥，一条死亡铁路，一片凄凉的坟墓，如同一本浑厚的象形文字记录下惨痛历史的每一个细节，任何人妄想撕毁昨天都是徒劳的。当然，我们需要做到的也不仅仅是让悲愤的记忆刻骨铭心，铭记昨天是为了开创美好的未来，但愿横跨历史的桂河大桥，以及那片弥漫着萧瑟风声的坟场，让我们懂得如何把握住今天，如何发愤图强，如何去制止那些狂妄的叫嚣！

　　电脑写作稿（散文） ：云南省大理州中国人民保险公司永平县支公司
