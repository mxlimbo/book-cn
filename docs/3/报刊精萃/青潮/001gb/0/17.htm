"无知者"未必无畏
——评王朔文化现象


"无知者"未必无畏，因为他并不是真无知。只有真知者才能无畏，因为他拥有真理。

海边小路/文

　　说到王朔，我很喜欢。这是一个有趣的作家，看他的作品不会象一般文学作品那样感觉沉闷，很搞笑，很解气。看过《千万别把我当人》后再看萤屏上的那些言不由衷的世界冠军，心里总是怪怪的，觉得崇高之下的一切有时显得特滑稽。鲁迅说，悲剧是将有价值的东西毁灭给人看。王朔却给人们显出了另一面：将貌似崇高、伟大、权威的东西解构得千疮百孔，演泽成类似于香港的周星弛式搞笑片，供人喷饭──这是一种反悲剧的创作方向，这也是现代社会多元化思潮在文学上的反映。

　　在二十世纪的中国文学中，有太多的独断论，有太多的伪君子，所以王朔的这种"真小人"风格才令人耳目一新。王朔是在八十年代后期西方后现代作品疯狂涌入和中国文学界倦于"寻根"、"反思"之际适时火起来的，它反映的是文人思考力的疲倦和民众对主流文学的厌倦。后现代文学的一大特点是反权威和无中心，解构一切成了它最大的使命。中国社会尽管还徘徊在现代工业社会的初期，但文学以其自身的超前性在中国某些工业化程度较高的大都市预告了中国社会的未来。王朔之后中国的后现代文人多如牛毛，随便一篇散文也来调侃几句崇高，"我们曾经很傻"成了青春期的最佳概括语。中国文人在经受了无数次自虐和他虐之后开始放弃充当精神牧师的愚蠢使命，由耻于言利向耻于言义"进化"，干脆脱下孔乙己们穿了几千年的破长衫，仰天大叫："我是流氓我怕谁？"甚至掀起轰轰烈烈的"窝里斗"，文墨战争层出不穷，各式文人如泼妇骂街，揭短的揭短，噱头的噱头，庸俗不但不是罪过，还成了一时之时尚，反正有钱赚则行，何必纠缠于敏感的思想问题累人累己？文人终于学乖了。

　　然而这一切都是相当表面的，是文人就有理想，有理想就有崇高。毛主席说："风物长宜放眼量，牢骚太多防肠断。"能"风物长宜"的是直面现实的政治家，而思想超前的文人们从来都是牢骚满腹的，只是时代不同了，表现的方式也不同而已。在这个把崇高当玩笑的时代，标榜崇高本身就是一种庸俗，因为崇高已经被太多卑鄙的东西所污染了。而像王朔这样在未写之前就宣言要赚读者的钱的做法，却因此而物以稀为贵，成了俗中之俗，雅中之雅了。但你说现在王朔就只爱钱吗？不，在没钱的时候最爱钱，像他这个号称中国最有钱的作家的作家，还有什么必要去为钱而写呢？即使为名去写也大可不必，像他这种"把无知当个性"的"无知者"，名声好不到哪里去，他树不了也从来不想树什么美名。我认为，王朔是个标榜"无知"的清醒者，他知道这个时代需要什么，他明白文人在这个时代该以什么面目出现为最好。因此可以说，他说自己无知，说自己庸俗，不过是一种策略。即使他以一副痞子的模样痛骂文人，但他自己何曾不是文人，一个洗不去抹不掉的文人。王朔以文人所用的武器来解构文人，这是工业社会和后现代文学所产生的悖论，但悖论的产生，并不能说明崇高的不存在。崇高不是一件衣服，说穿就穿说脱就脱，它是非常内在的东西，它是人类文化积淀的东西，存在于每个人的前意识中，不但是浸过太多充满人文理想的文学作品的文人，就是一般劳力者，何尝没有一个崇高之梦──金庸小说所推崇的"侠之大者"就是许多普通人慕之而未能至之的崇高。王朔的反崇高正应了一个深刻的命题：最激烈地批判社会规范的人，往往是最希望建立社会规范的。我们的时代正处在一个社会规范大崩溃、大转换的过程，正面地鼓崐吹什么变味的社会规范往往令人厌恶，只有用反规范的方法来颠覆旧规范，才能为期待着新规范产生的人们所接受。

　　王朔身上可以嗅出美国反叛式作家塞林格的味道，但他这种反叛无疑是中国式的，《麦田里的守望者》可以非常严肃地批判现实，而王朔却只能假"无知"之名来调侃现实。这多象魏晋时代的那些反叛社会却缅于酒池、游于清谈的务虚文人，自己固然"玩"得很过瘾，但对社会的批判深度却大大降低了。王朔不能像塞林格那样离群索居做得那么绝对，因此只能披上"俗人"的外衣，侃一些仿佛无聊而有趣的故事，赚众人一笑，赚一些钱。这是一个中国式的现实，中国式的悲剧。

　　孔子说："君子有三畏：畏天命，畏大人，畏圣人之言。"王朔可以尽情调侃这三种东西，也可以声言自己什么也不怕，但这又能如何？他取消得了权威的永存性吗？他崐真的能全然无畏吗？他如果真的能无畏于天下，又何必借助于"无知"之名？他不如说"真知者无畏"还来得干脆。大梦如觉，大醉若醒，王朔令人想起了将荒唐当真理的庄周。

　　谁也不能真正地无畏，除非他是疯子或哲学家。我也有孔子的三畏，因此我才喜欢王朔，喜欢多元化的自由而安全的文化。但我同样希望王朔所代表的多元化的自由而迷茫的时代的结束。多元化就是离开了思想的确定性去"无的放矢"，谁也没有确定的目标，谁也不知自己最终的方向是什么，于是"没有目的的目的"成了文人很爱说的一句话。中国的几千年历史中，权威的丧失是一件很痛苦的事，而权威的强化也是一件灭绝人性的事，这两个极端是中国文人永恒的苦。这个时代是权威正在一点一点地消亡的时代，文人的痛苦也正在一点一点地增加。最幸福的时代是权威如朝阳般上升的时代，汉唐就是这样，五十年代和八十年代初期也是这样；最迷茫最绝望的时代是权威如夕阳般下沉的时代，明清如此，现在也如此。而有一个时代是包括文人在内的所有社会成员都激发起空前的创造力去追逐太阳、追逐真理的时代，这是最充实的时代，这就是"百家争鸣，百花齐放"的时代，先秦如此，我们未来的几十年也或许如此。

　　现在是一个类似于魏晋的时代，只少了些血腥味，但思想文化上是一致的，都是在迷茫中艰难地思索，或在百思不得其解中干脆拒绝思索，转去把握最现实最直接的东西。海子是前者，王朔是后者。

　　"无知者"未必无畏，因为他并不是真无知。只有真知者才能无畏，因为他拥有真理。而真知者何在？谁也不知道，因为路才走了一半。
