本　性


　　　　　spider/文

　　　 　 我是一只笼中困兽
　　　　　楚楚可怜置身樊笼
　　　　　任荒原风沙一点一点吞噬美丽
　　　　　他说　我的双眸是幽幽深潭
　　　　　可别人说
　　　　　那是迷香四溢的陷阱
　　　　　千万不要靠近
　　　　　他说　我的柔柔话语似天籁般的歌
　　　　　可别人说
　　　　　所有的话都是谎言
　　　　　千万不要确信

　　　　　他是凡人　又怎能抵挡魔咒
　　　　　我的迷离眼神　浅浅笑意
　　　　　就可以让他溶解　痴迷
　　　　　终于　他不顾别人劝阻
　　　　　帮我挣脱桎梏
　　　　　重回放纵驰骋的大地

　　　　　释放出我久违的疯魔狂乱
　　　　　转瞬间　一切被淹没
　　　　　在我面前
　　　　　他的身体已没有禁区
　　　　　两片炽热红唇贴上他的肌肤
　　　　　烙上属于我的永久印记
　　　　　征服一个男人
　　　　　同征服一只猎物一样容易

　　　　　在他身体上
　　　　　我发现让自己快乐的秘密
　　　　　给他疯狂快感的同时
　　　　　我开始一点一点噬咬他的躯体
　　　　　鲜血涌出似朵朵绽开的花蕾
　　　　　甜腥的味道无疑是种兴奋剂
　　　　　给他千般万种柔情
　　　　　给他骇浪般的欢愉
　　　　　都只是用来消灭他痛楚的武器
　　　　　如饮甘露般贪婪吮吸着
　　　　　如此鲜嫩的生命
　　　　　直到最终一点一滴耗尽

　　　　　消失于夜幕之前
　　　　　我眼中闪动着温柔
　　　　　依然是那惹人怜爱的美丽
　　　　　轻吻那张苍白而可爱的脸
　　　　　他嘴角还有甜蜜笑意
　　　　　记住我吧
　　　　　兽的本性亦是索取
　　　　　回望夜空
　　　　　戏谑地笑着
　　　　　我看到 天使在哭泣
