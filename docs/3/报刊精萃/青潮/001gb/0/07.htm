菊 之 野


阿妍/文

　　一个和我一样爱好诗歌的朋友来看我，带来一束新鲜的菊花。

　　我喜欢菊花，它在我的诗中曾多次出现。看，那热烈的艳黄色、含蓄的淡紫色、温润、透明的淡绿色，三色而合的花束映亮了我的双眼。

　　手捧菊花，一片田野在面前悠悠地展开。

　　在那片田野上，我曾轻快如风地奔跑；在那片田野上，我的歌声多么清亮；在那片田野上，我第一次和野花、野草说话；在那片田野上，到处都可看到野菊花……。呵，那片田野！记录着我的童年，留有我最初、最纯真的梦。

　　有多少年呵！我已经远离了田野，远离了野菊花的清芳。我已多年不能奔跑，我的双腿被疾病羁绊。在十几平方的小屋，我和屋外的人一同，经历富有生机的春天、度过热浪袭人的夏季、走过云升叶落的深秋、送走雪花飘飞的寒冬。

　　而此刻，这束菊花正在静静地释放她们的思念。

　　她们还是当年的那些菊花吧！她们是否已经把我――曾经向它们倾吐我的爱的那个小姑娘认出！

　　手捧菊花，我的眼睛不知何时湿润了。我不由得低下头，仔细地看那细致、柔美、透着坚毅之气的花朵，看它们生机勃勃的叶片，看那挺直的花梗以及花瓣和叶子上晶亮的小水珠。

　　我竟然发现了一只极小的小黑虫静静地伏在一朵艳黄色的菊花上。我心中一动，童年的田野上不也有许多这样的小黑虫吗？现在，它或许也正在看着我？……

　　这个时候，我如何能忍住不细闻它们的香味？！当更深浓的菊香泌入心脾，我感到我已躺在那片田野上，身下与四周都是青青的草。

　　朋友端着一杯水走进我的房间，我立刻用我们习惯的诗的语言，给他描述我的感受。我让朋友闻那菊花的芬芳、他细看那只小黑虫透明的双翅……。我悠悠地告诉他，这束菊花使我重新回到了田野，而这只小黑虫却借着一朵花窥见了我的惊喜。

　　我看到朋友的眼睛也湿润了，我知道，此刻，这片田野也正在他的心头展开。

　　这片田野是心中的田野，它正穿越四面墙、冲出束缚着我的生活与情感的空间。

　　感谢朋友送的菊花，它为我开启了一片永恒的菊之野。无论时光如何流逝、无论生存的空间多么狭小，这片田野将永远开阔、充满生机。它将永远珍存在我心中，随时唤我徜徉其上，把这颗属于大自然的灵魂清洗与抚慰。
