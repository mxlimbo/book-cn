一个诗人的作家生涯

桂林晚报副刊部 刘春

　　1998年，是我心理状况最为动荡不安的一年，我突然产生许多不平，最主要的原因是，自己写了近十年诗歌，在包括《诗刊》《星星》等中国最著名的刊物发表过数百首诗歌，似乎没有预期的响动。对于一个写作者来说，还有什么比自己的作品遭受冷落更让人失望的呢？我思前想后，试图效仿海男韩东李大卫们，不当诗人当作家了。（是不是有点滑稽？要知道在此之前我已拥有两个不同级别的作家协会的红皮会员证哪。看来诗人与作家应该划清界限：诗人是写分行文字的人，作家是写分段文字——小说、散文——的人。诗人不属于作家的一部分。如此说来，诗人加入作家协会岂不是牛头不对马嘴？）

　　尽管那次转行不算成功，但直到今天我仍认为，当时的选择有充分的理由。我所在的省份并不缺乏优秀的诗歌人才，特别是 90年代，青年诗人发表的诗歌作品无论数量质量还是艺术观念都是80年代诗人无法比拟的。但是，这个地方没有任何部门或领导重视诗人（其实也怪不得别人，全中国都是如此），人们司空见惯的是一次次针对作家和影视剧本写作人员的笔会、培训、签约活动。最近的一次签约把范围扩大到唱歌的，弄得还挺隆重：将几个小姑娘送到北京，每月给一笔钱供她们吃住。

　　唱歌的与咱行业不同，暂不管它，单说作家签约。这一次被选中的三个家伙中，除了有在一个在《作家》杂志发表过作品（尽管只是一个数千字的小短篇），还勉强值得一提，另两个的入选无法不让人再一次想起那个被用滥了的词汇——大跌眼镜。看看吧，一个家伙的最高成就是在本省的小刊物发过三两个中篇；另一个更好笑，自费出版一本小说就顺利地把每月800元弄到手了——不要误会，我不是眼红人家的钱，这一点待会我会解释。意味深长的是，此次签约，有一个常喊我老师的诗歌爱好者差一点签上了。且慢为诗歌获得重视而高兴，此人能走到这一步是因为他与某相关人士是亲戚。不巧的是，在签约的前一天，这厮因多次抄袭别人的作品发表的丑闻曝光而被刷了下来。（不久后又因为再次抄袭被单位炒了鱿鱼。）

　　也许有人会这么反驳：人家拿政府的钱是有条件的——签约期间得在指定的刊物发表一定数量的作品。这我知道。但有一些我知道的事你不一定知道，比如一个省级文学刊物副主编酒喝多了这么放话：其实到和约结束时他们完不成任务，我们也会帮他们完成的。其实哪里用得着如此麻烦？前两年第一届签约的八人中就有两三个没完成任务，刚才那家刊物虽没帮他们擦屁股，却也没听说哪个部门叫他们退款！

　　说出这些话让我有点难为情，好像我患了甲亢而且病情不轻，好像我那么看重某些部门的经济支持，好像我那么虚荣那么功利那么希望一举成名天下知。不错，我承认都有那么一点点。靠自己的才华获取经济利益有什么值得脸红的呢？问题是，目前的这批签约者是否就是最佳人选？小说剧本嗓子都可以签，诗歌为什么不行？要知道中国自古以来就有"诗国"的美誉啊！

　　在这种环境下，几个搞小说的朋友要来扶贫了，一见面就装得十分够意思的样子，动员我加入他们的"阵营"；常聚会于酒桌上的某市级刊物主编也三番五次地开导：XX、XX出道时底子比你差得远，你看现在他们多牛逼。想想也是，要知道这些"名家"给我编的小报写稿还得频频劳神替他们改错别字的啊！一个在北方某大学中文系搞现当代文学研究的忘年交（这个词是不是有点恶心）也趁南下度假之机跑来鼓励。这个"把鲁迅郭沫若存进银行吃利息"（见李亚伟的诗歌《中文系》）的老知识分子教育人的方式比较特别——在一张纸上写满了密密麻麻的人名，然后很深沉很不容置疑地说（那味道像足了某个湖南出身的大人物）：你看看，这里面有哪个不是从诗歌转向小说并获得成功的？你同样可以试试嘛。我没看。不用看我也知道上面写的是哪些人。

　　虽然没看，但如此别致的激励方式让我蠢蠢欲动。阿Q说：和尚摸得我摸不得？那么我何尝不可以改动一下：韩东朱文海男李大卫张执浩转行写小说能行，甚至伊沙南野叶舟秦巴子都行，难道我不能试一试？当然我这样说并无贬低这些人的意思。朱文我没见过，文笔还马马虎虎，重要的是据说他对朋友挺够义气，使得"云南王"于坚免费替他大肆吹嘘，说什么"整个90年代进入我的阅读视野的只有朱文"之类的蠢话，不过于坚放心的人我基本上也放心，文笔马虎点也以后还可以锻炼嘛；张执浩的东西据说是小家子气了点，但人家那能把乡土诗改写成带牛奶味的小说的才华你不服不行；伊沙叶舟秦巴子也不便提，特别是吴胖子（伊沙姓吴）这个江湖大佬，拥趸遍野。对于莽汉我向来都多多少少给些面子（说得不好听是"提防着点"）。至于韩东我倒接触过，的确是个少见的朴素厚道好说话的主，朴素厚道得让你根本不相信眼前这乡巴佬会提笔写字，更别说写诗码小说这类高级运动了。1995年秋天的某个晚上韩东和李冯、鬼子等文坛新秀在桂林球迷酒家吃饭唱歌，一首《黄土高坡》高音上不去低音下不来的拙劣表现导致我拒绝了《他们》的约稿。这样的人能写小说而且能成为好小说家成为晚生代的代表人物，本身就是提高别人自信心的绝佳教材。

　　我开始写小说了。效率不错，仅花一个月就编了三个万余字的短篇。誊抄好稿子，心里产生一种快要晕倒的感觉，虚荣心也前所未有地膨胀，见一个文学朋友就拿稿子出来吹嘘，也不管对方有没有时间感不感兴趣；一有与别人交谈的机会就把话题往文学上扯，管他是不是文学爱好者。不仅自己四处推销，正在大学读书的女友也由于受多次暗示而变得乖巧起来，主动在本宿舍本班本楼层本校进行宣传。好戏还在后头，酒肉朋友罗十八读了其中一个叫《有病》的短篇，又屁颠屁颠地凌晨两点从八百里外打来长途电话，死命地煽风点火了一番，说什么"这个小说千万不能给小刊物，一定要给花城钟山作家大家"，"再弄几个《有病》出来广西三剑客不靠边站也不行"（好像我是能生分段文字的母鸡一样）之类，煽得我心旌摇曳。据朋友后来说，那段时间，我说话变得大大咧咧，走起路来大摇大摆，仿佛余华格非苏童北村已指日可追，何顿述平毕飞宇已不在话下，至于卫慧棉棉周洁茹之类刚冒出来的靠百分之一的天分百分之九十九的脸蛋吃饭的小女人就更不用提了。

　　我找了一家文印部把几个小说工工整整地打印下来，按照罗十八的建议（其实是自己的愿望），恭恭敬敬地贴上邮票，小心翼翼地将它们塞进邮筒，让他们飞向罗十八所列举的那几个神圣的地方。最初的几天甚至几个月，我充满自信地等待，等待自己像许多传奇人物那样一举成名，我甚至列举好了设宴庆祝时要请的客人名单。然而，随着时间的推移，我的信心渐渐减弱。半年后，我不得不调整战略，把目标对准《小说界》《东海》《芙蓉》这些第二梯队刊物。几个月的杳无音信把我再次抛入失望的深渊。

　　有几个晚上我脸红红地问自己：你是为内心写作还是为名利写作？政府认为你成名你就是名人了？写诗的难道永远不如写小说的？但是有些问题我固执己见：想靠写小说出名，没关系的不如有关系的；仅在语言上大胆的不如生活糜烂的（至于哪些作家可以入列我就不说了）；同是70年代出生，男的不如女的。

　　这几个小说后来还是发表了，我对此已没有半分兴奋，它们在读者之中也没有激起半点波澜（我怀疑根本就没几人读到它们）。我也没将它们发表的消息告诉任何人。

　　我的作家生涯随之结束。

　　2000年4月5日凌晨于桂林小香港遗址

　　

　　作者: 青年同志

　　标题: 小说和诗歌对比后想到的

　　内容:

　　刘春先生：

　　我作为一个晚辈，看了你关于目前文学中的诗歌和小说比较后的感想，我细细想了好长时间。

　　觉得你从一个诗人的角度谈了对诗的理解"特别是 90年代，青年诗人发表的诗歌作品无论数量质量还是艺术观念都是80年代诗人无法比拟的"。你的这个观点我大抵赞同，至少我觉得现在的文化气氛很好，许多年轻的文学爱好者的诗文都似乎不差于名家，这是思维自由社会的人的内在能力和感知的勃发，比如我曾经拿一些很脍炙人口的近代诗歌，甚至一些被《中国现代文学作品选》（南京大学出版社）和《中国现代文学名篇选读》（南开大学出版社）等等收入的。我看了中间的一些诗歌真的觉得我们的时代变化得真是快，那些年或者被认为是绝佳的今天怎么找不到多少味道呢？我一再都是承认是可能无法深入到诗人真正的内心世界，无法在他们的田园思维萌发。

　　然而当我看到一个《大学生》（好象是）杂志搞的一个关于诗歌在大学生心目中的调查发现，现在的大多数学生是对诗歌不屑的，有人甚至过分的评论"诗歌就是把主语、谓语、宾语放在三个口袋里，然后随便拼凑就是了"，当然这太极端，我们的思考和重视却总是在极端中展开的。

　　为什么现在的人，年轻人，这样一个特别富有诗意的季节的时候淡漠了诗歌！？

　　在这里引用马克思的观点，那就是经济上层建筑的作用使然了，这个追求物质快速回报的社会中，我们的视线不由自主对准了它——-市场，市场不是单单的菜篮子，而是我们选择的很多注脚，我们在比较时更多的把这个人可以带来的直接的物质利益放在了第一位，我们有着极多的开发区，他们对着有钱投资的老板们就怕掏出心肝了，因为他们的钱就在口袋里。

　　诗歌？这时候无疑是个羞物，她见不到市长的赞许的，也看不到企业家的RMB的。

　　诗歌的意境是需要人去慢慢揣摩的，然而我们疲惫的一天，为了生计都快倒下了，不快快睡觉或者和家人享受天伦，难道和你去钻研诗学？你要是说你看看诗集怎样，他会告诉你"你有神经啊！？"

　　写诗歌的人是不会有神经的，或者就是绝对孤独的神经，后者是你无法接触到的，他和你——一个物质利益为先的人——不会相逢一起的，不是一个道上的，走不到一个桥上去。

　　而小说，有人说正是"方便面"，吃了立即解饿，没有拐弯抹角的言辞，也没有晦涩的腔调。不需什么，虽然"吃"进马上可以拉掉，但是还是吸收了一些，且立杆见影。

　　这是没有办法制止和指责的，这是每个"独立人"的选择权利，也是社会筛选的逻辑。

　　爱诗歌的人，你们活着，为诗歌活着，你们是伟大的，你们寂寞中的伟大无形确实很凄凉，但这可能就是现实，不是诗歌。

　　怎么办？走小说之路？去过自己单居的生活？去苦闷？去彷徨？去发泄？去抗争？还是自杀？

　　没有现成的路，你自己去找吧！

　　就想我们的国家一样"摸着石头过河"！

　　

　　作者: 阿妍

　　标题: 与刘春交谈

　　内容: 与刘春交谈

　　本想称你刘春兄，但由于不知晓你的年龄，只好作罢。诗人似乎就该是情感的、愤怒的（——没有血液的奔涌如何成诗？）！也许这就是现实生活中许多诗人过得并不太好的一个原因。

　　恕我直言，从你的帖子中我能感到你内心的不平衡，我非常理解，但这样对你的创作会有影响，不管是诗歌还是你面对而为之犹豫的小说或者还有别的，保持一个良好的心态是必要的。诚然，写出作品都渴望发表、渴求相应的承认，诗人尤其如此——诗歌是极个人化的创作，正因为其极个人化，才更渴求有读诗之人的共鸣；诗人灵魂深处是孤独的，正因孤独才更渴望知音！而诗人又是最可以不孤独的，诗意的捕捉与品味本身就是给写作者写出一首诗后最好的回报，如果再有读者能进行阅读时的第二次创作，便收获更丰厚。如果一个诗人怀此平淡之心创作，那么未必不可以有发表的机会，而更重要的是在没有得到相应的看得见的回报的时候，它可帮我们抵御私心杂念的折磨。

　　如果苦苦追问为什么，可能没有结果，不如把"追问"的精力用于自我审视：我还有诗可写吗？我在重复自已吗？我对我现在的诗满意吗？……

　　而真想写小说，也没有不可，因为我希望我们不是为了写诗而去写诗，我们写诗、写小说、童话、儿童文学都是为了反映和进一步体味身处其中的"生活"。不管写什么，我们首先需要面对和处理的是自已、自我。

　　关于写作与自已与生活发表了个人浅见，只为增进沟通与了解、增进朴素而真挚的友情，愿刘兄能接纳这个现在还陌生的朋友！

　　青年同志，你的名字不太好写，所以总如此称呼你！在这个时代，你还能因"诗歌"而有这样的思考，真使我位感欣喜！！！

　　元亨利贞

　　2000.2.18

　　

　　作者: 海边小屋

　　标题: 愤怒

　　内容: 愤怒 　　

　　我感到愤怒
　　在钢筋水泥中我异常愤怒
　　在世俗的牢狱中我出离愤怒
　　我要冲破一切
　　我渴望毁灭一切

　　我的力量来自夸父
　　来自酒神和尼采
　　我无比愤怒
　　我要将世界的面貌复原

　　我的世界没有音乐
　　只有愤怒的号角
　　我要占领宇宙
　　我要狩猎未来

　　但现在我一无所有
　　我现在只能纵酒高歌
　　我愤怒地骑马奔腾于历史与未来

　　我如一阵飓风
　　我如一场地震
　　我要世界涌起最愤怒的海浪

　　在未来
　　我的名字将以最愤怒的形态铭刻在恒星的天宇

　　

　　作者: 海边小屋

　　标题: 天鹅——献给海子

　　内容:

　　天鹅

　　——献给海子

　　你是受伤的天鹅
　　你是扑火的飞蛾

　　只有在极痛苦的时候
　　我才能触摸你的美学意义
　　你如铺天瑞雪
　　秋收时人们才发现麦地是多么辉煌

　　每个时代总有许多天鹅忍受着弓箭与风暴
　　每个时代总有不少天鹅在绵绵春雨中退化成丑小鸭
　　有的天鹅许多年后仍是天鹅
　　有的天鹅莫名其妙羽褪翅折成为盘中餐
　　有的天鹅太累了降落在农庄吞食饲料
　　只有你，一只饥饿的、削瘦的、一无所有的高傲的天鹅
　　仍然去天山寻找未知的雪莲

　　

　　作者: 海边小屋

　　标题: 我不得不承认，许多诗人都有过浓的自恋情结和贵族倾向

　　内容: 以上三首诗都是献给海子的，但其实不如说是献给自恋的自己。

　　我知道时代思潮的主流已经走向宽容和平等的大众化观念，但某些诗人的自恋一时还改不了。没有救世主，也不需要救世主，但需要诗么，诗的出路在哪里？我不喜欢伊沙式的故作平民化的诗，但海子仿佛已经给人淡忘。

　　自恋其实就是幼稚，就是迷茫，因此我现在只能保持一种谦虚的姿态。
