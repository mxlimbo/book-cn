　　犹大："是我不好，不过她真的是刚来。"

　　瑞儿掩面痛哭。

　　犹大："瑞儿，别哭了好吗？"

　　瑞儿忍不住放声大哭起来。

　　犹大离开了我们的房间，瑞儿一直在等他回来。一个钟头过去了，他没来。他的好朋友告诉我他回家了，我说你把他叫来（忘了告诉你们，犹大在他自己的网吧里上网）。又一个小时过去了，犹大回来了，但是他叫人转告我他没心情。我不知道发生什么了。正要离开，聊天室里来了一个叫WWW的女孩，她叫住了我。

　　WWW："我是犹大的女朋友，我跟你说几句话好吗？"

　　瑞儿："我真的很想和你聊，可是寝室要关门了，有什么事你就说吧。"

　　WWW悲伤地说："我不是故意要看你们私聊的，我真没想到事情会闹成这样啊。"

　　我急着要走，也没来得及细想她的话，就道了个别匆匆离开了。过了一天，我电话他，才明白了那天晚上发生了什么。犹大当着很多人的面把键盘砸了，骂得很厉害，她要给我写道歉信，他都不把我的信箱告诉她。 我听了之后并没有胜利的感觉，只为自己由误会而生的一时冲动的愚蠢行为感到愧疚。无意之中又复杂了我们三个人之间的关系，给他们造成更深的裂痕。我无地自容。犹大答应我他会对她道歉，可是事情并没有就此平静下来。

　　我们很快把这不愉快抛在脑后，继续重复我们的快乐，天天都一样的日子过得很快，但我们彼此都很珍惜，也十分害怕失去对方。转眼间就放暑假了，我们见面的机会越来越多，然而对现实，对道德的恐惧总是如影随形，不断加重，压得我们喘不过气来。有一次我说："你说我们这样好吗？"他说："不好，名不正，言不顺，算幽会，还算偷情。"

　　更要命的是，他怕我长大了会离开他。我一提到我们是否该分开的问题他就冒冷汗。于是，我有感而发，写了个东东——永不言爱

　　是我卖乖 相识之初 急于表白 是我不该 缘起伊始 与你言爱

　　此时 你站在远远的暮霭 半点心不停摇摆 何处找寻往昔的尸骸啊 我的世界白雪皑皑

　　也许你怕跟不上我成长的节拍 也许你担心人性的败坏 是你不愿给谁造成伤害 还是惟恐自己被悲哀掩埋

　　可是那些我都不想去度猜 我怎能抛开对你的依赖 你的心是无底的大海 宠我宠得却象个小孩

　　只有你让我把心细细剪裁 只有你让我甘愿释放所有光彩 我已习惯在焦急中等待 等待你无期的到来

　　如果这份感情你无力承载 就把它收回吧 我们从此不再昧暧 哪怕 人生只是一场 感慨

　　2000年7月

　　*********************************

　　我把它发到他的信箱里，他说，看了，心里酸溜溜的。

　　就在这个月初，突然好几天都没有他的电话，我终于忍不住电话询问他。他说他心情不好，怕影响我，就没告诉我。我说你还有什么事不告诉我啊。他说他骂人了。

　　"什么？"

　　"我骂她了，当着很多人的面，把她骂哭了。"

　　"你怎么又这样啊？我不是跟你说过了吗！"

　　"是我不好，我去下载了个解密码的软件，谁知她拿去开了我的信箱，我发现的时候她正在看你给我的卡。"

　　"可是你也不该不给她留点面子啊！"

　　"也难怪她，以前我老在她面前夸你。你要知道，认识你之前，我只有她一个异性朋友，她对我可是放心得很，直到你的出现…………说者无心，听者有意。这样的事也不止两次了，我们吵过好多回了。"

　　我握着话筒直发呆，到底是谁该受到责备呢？天哪！我 失声痛哭。

　　"喂！！你怎么了 …………………………"

　　***************************************

　　一波未平，一波又起。两天之后的深夜，我被电话吵醒。

　　"喂，瑞儿啊？"

　　"恩。"

　　"明天我不上网了。 她一个人跑到H市去了，听她的同事说她的眼睛出问题了，可能就在医院附近吧！我现在就动身去把她找回来。"

　　"眼睛？严不严重啊？"

　　"不知道，还是配眼镜时老板发现的。"

　　"那你好好的把她给找回来啊！"

　　"好！我回来就给你电话？"

　　"好的。"

　　"想你了。"

　　"哦/"

　　我在极复杂的心情中煎熬了三天，却是音信全无。找到了没有啊？我鼓起勇气拨了他的手机，通了，传来一个悦耳的女声："喂？"我落荒而逃。

　　是的，他找到她了，我应该很高兴。他们在一起，她替他接电话，这很正常，不是吗？为什么我从来没想到过，为什么悲哀紧紧地揪住我的胸口？为什么热泪涌上我的心头？

　　不知道是做贼心虚，还是为了弥补什么，鬼使神差地，我在网上给他的手机发了两条短讯：

　　"你在干什么呢？电话也没人接？找到她没有啊？急死我了！"

　　"对了，你和她好好谈谈。 是我不好，你也有份！怎么可以对一个女孩子如此粗暴！我知道其实你很爱她，好好地爱你的女人，听见没有？？"

　　今天看来，这简直就是欲盖弥彰嘛！！可是当时我更希望她能看到我的话，能知道我很关心她，只是无法面对她而已。后来 事实证明我的目的达到了：犹大说，那时他不在，不过她把手机递给他的时候，表情出奇的平静，只说有个电话可能是瑞儿打来的。天真的我相信她和我有一样的感受——-提不起恨。

　　谢天谢地！他们都平安回来了。她的眼睛是弱视，还好没有什么大碍。但是她一直都回避此事了。

　　他曾给我写过一封信："terry....

　　不知道从哪儿说起了...

　　要不..咱..说你吧

　　说句很土其实很心里的话:真TMD高兴认识了你 我看我们相识的那天应该定为"幸运节" 全世界都该为之庆祝

　　我从你那儿得到的 比你想要给我的多得多 真的很谢谢你了 在如此虚幻的网络里能遇上如此的你 对我来说简直就TMD是个天大的讽刺 认识你之前 那些对于网事一本正经的人我总是嗜之以鼻的 直到你的出现 足以改写"虚幻网络"之反动言论

　　或许应了一句话"距离可以创造美丽 遥远可以铸就思念".....

　　你甚至不知道我啥样 如果我们近在咫尺 我们会有现在吗 有时倒是希望我们永远都这样保持着距离 现在这样已经很满足了我不奢望更多的 倒很怕会失去现在仅有的 ..."

　　其实，我也和你一样，不希望什么了。

　　我不知道接下来会发生什么，但我会勇敢地接受命运的安排。我也一度深感自己是可耻的，可我摆脱不了爱情的绳索。尽管这样，我还是希望每个人都幸福。如果可以，为什么不呢？

　　有一天，犹大对我说："我想去教堂。"

　　"去结婚吗？"

　　"不，去忏悔。"

　　"好啊！我们一起去！"

　　"哦？你先说说你要忏悔什么？"

　　亲爱的，我要跪倒在上帝的脚下，对他进行最虔诚的膜拜：

　　"仁慈而万能的主啊，您高高在上，俯视人间一切罪恶，您有拯救万物灵魂的胸怀。

　　所以，即使犹大和他的情人背叛了您，您也要宽恕您柔弱而无知的儿女。如果您可以给他美满的馀生，我情愿永世做你的唤仆，追随您的衣襟。

　　阿门！"
