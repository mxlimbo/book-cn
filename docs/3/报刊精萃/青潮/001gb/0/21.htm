侃球论道


海边小屋/文

　　足球是什么？一个有趣的游戏，一项充满活力的体育运动，一种正在形成的大众文化。

　　为什么要做球迷？因为足球比赛太精彩了。

　　做球迷要达到什么目的？欣赏美，享受快乐。

　　怎样才能从足球中得到快乐？——

　　一是发自本能地喜欢，因为足球本就是一项宣泄性的激情运动，人们从脚接触球的每一瞬都会感到生命力洋溢的快感。

　　二是精神寄托，因为足球比赛精彩得你非常容易产生幻觉，就象阅读一本情节动人的小说一样，你会将自己融入场上的球员中，他们进球，你会将他们的狂喜转移到自己身上，这样你将极具成就感。

　　三是审美愉悦，因为绿茵场上飞来飞去的皮球划出的每一道弧线都带有黄金分割线般的美感，而球员的飞铲、冲刺、劲射等都是一个个动态的塑像，比罗丹的雕塑更能表达视觉美的律动。

　　四是人生感悟，因为球场就是一个剧场，人生百象尽在其中，胜利与失败交相辉映，真实与虚伪同时存在，投机分子与老实人对比鲜明，一切都在黑衣法官（命运的象征）的一声哨中骤然定格——但不管有多少黑哨，球还得踢下去，我们的人生征程从中或许可得到某些启示。

　　五是哲学沉思，因为足球在今天已经演变成一种综合性的艺术门类和社会学研究的新课题，看足球可以明理，可以明白自己入球固然可喜，配合别人进球也足欣慰（集体主义精神）；可以明白球场上固然需要成熟的技战术，但水平相当时往往要靠拼劲和运气作裁判（必然律与偶然律的辩证统一、主观能动性的重要）；可以明白比赛前不管对手多么强大都要充满必胜的信念，但比赛后即使惨败也要保持自己的尊严（人类内在的生命力永不言败）……

　　总之，这五点依次递进，体现了球迷们看球的不同境界。而快乐，正是从这五种境界的迷醉中产生的，只有入境和忘我，才有足球的快乐。

　　这五种境界的快乐又各有区别。前两种境界层次较低，但又各有不同，本能地当球迷比较纯粹，快乐也比较持久、真挚而深刻，而纯精神寄托式的球迷往往纠缠于球队的成败之间或喜或悲，这对于战绩极差的中国足球的球迷来说就比较痛苦。后三种境界是超越性的，足球在此只是一个媒介，这种球迷醉翁之意不在球，而是透过足球看世界，可算是入乎其内，出乎其外，球场胜负能不系于心，理性并快乐着。

　　由一个很在乎中国足球能否"冲出亚洲"的球迷进化为一个只看世界杯和欧洲联赛的球迷，肯定有一个逐渐"看破红尘"的过程。这个过程颇为痛苦，但颇为值得。因为看球目的是寻找快乐，而非为几个不争气的球员在场上的不负责任的低级失误而大动肝火，足球胜也罢，输也罢，其本身都是足球，何必太过认真？然而现在大多数刚迷上足球不久的人显得过于狂热，一门心思要自己喜爱的球队赢球，一输就骂裁判，骂教练，京骂、粤骂满天飞，这在甲A甲B赛场屡见不鲜，但他们也总有一天会成熟起来的。

　　做一个带有淡淡的思和恬恬的乐的"得道"球迷，你会发现，足球很好，生活很好，一切本来都如此，何必庸人自扰？

　　做这样一个高境界的球迷很不容易，许多人想做但还没做到，我也是一样。
