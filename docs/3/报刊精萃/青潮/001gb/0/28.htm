以皓月煲酒，悟人生真味
――读林清玄散文精品《温一壶月光下酒》

李智红/文

　　林清玄是我最为推崇和喜爱的台湾当代著名散文家，是台湾文坛不可多得的，深受中华传统文化滋养与哺育的作家之一。尤其是在佛教文化的研究和运用上，更是炉火纯青，颇见功力和火候。字里行间所散发出的那股子浓郁的，传统文化和佛家思想的醇厚与芬芳，让我陶醉不已。正如他在《艺术水火》一文中的精到阐述："传统的精神正是一盆火，它必须不断地添加柴薪才能使它点燃下去，而壶中煮的是现代的水，现实代的中国艺术家正是装满了水的水壶，不管壶中装的是西方的乳酪或是中国的乌龙，使它煮好、煮香、煮开的，都必须是传统的火种才行。"

　　《温一壶月光下酒》是林清玄的散文创作进入集中喷发期的一部锋芒初露的力作。作家的散文创作经过了整整十年的磨练，已从豪气如虹，飞扬浪温向醇厚的沉思转变。透出了深厚而敏锐的洞察力和柔弱生刚强的审美品位，以及气清神定的宽厚胸襟。"将月光装在酒壶里，用文火一起温来喝，此中有真意，乃是酒仙的境界。"这此中的真意，便是人生的本味，便是生命的真谛。大美无言，人生的大美，人生的真味，尽在一壶酒中。那酒是用人生的月光，用思想的月光煲过的。那是何等的诗意，何等的旷达，何等的潇洒和超脱。他把东方美学的智慧融入宗教学的情怀，开创了泰文散文创作的一代新风。

　　在林清玄的《温一壶月光下酒》中，有许多篇什都是描写小人物，感悟平常事的。但林清玄的过人之处就在于能够从许多平常的人物，平常的生活琐事中，剥离出生活的本质，禅悟到人生的真味。

　　林清玄总是在以一颗平常心善待生活，他尊崇传统，崇尚一种简单朴素的生活方式。他在《合欢山印象》一文中说："我喜欢火炉，喜欢中国的东西，时时期盼未来的家中有个八仙桌、有个火炉，八仙桌要有古意，火炉要不熄，可以在炉旁读书写作，或把木炭轻轻地一块一块加在炉里，只希望有个小楼，不要公寓，这样即使在严寒的冬天，也能望见远方的青葱绿树，或者在园子里辟个小小莲花池，看田田莲叶撑一池生动的绿，只要有火炉和爱，我们就会不怕隆冬，不怕冰雪封冻了大地。"

　　在《温一壶月光下酒》一文中，我们则又看到了林清玄的飘逸和超脱。在林清玄看来，喝酒是有哲学的，"准备许多下酒菜，喝得杯盘狼藉是下乘的喝法；几粒花生米一盘豆腐干，和三五好友天南地北是中乘的喝法；一个人独斟自酌，举杯邀明月，对影成三人，是上乘的喝法。"，关于上乘的喝法，林清玄又另有一番见地："春天的时候可以面对满园怒放的杜鹃细饮五加皮；夏天的时候，在满树狂花中痛饮啤酒；秋日薄暮，用菊花煮竹叶青，人共海棠俱醉；冬寒时节则面对篱笆间的忍冬花，用腊梅温一壶大曲。这种种，就到了无物不可下酒的境界。"诗词自然也是林清玄上好的"下酒菜"，"喝淡酒的时候，宜读李清照；喝甜酒时，宜读柳永；喝烈酒则大歌东坡词。其他如辛弃疾，应饮高粱小口；读放翁，应大口喝大曲；读李后主，要用马祖老酒煮姜汁到出怨苦味时最好；至于陶渊明、李太白则浓淡皆宜，狂饮细品皆可。"林清玄好佛，但并不忌酒，佛家虽然讲究酒、色、财、气四大皆空，但他却觉得"喝酒到极处几可达佛家境界，试问，若能忍把浮名，换作浅酌低唱，即使天女来散花也不能着身，荣辱皆忘，前尘往事化作一缕轻烟，尽成因果，不正是佛家所谓苦修深修的境界吗？"我真佩服林清玄，能把喝酒的道行参悟到如此高妙，如此飘逸的份上，即使不是酒仙，也可看作是酒圣了。

　　林清玄是个参悟透了人生真味的作家，一个真正的，精神意义上的"隐者"。他认为"人间虽有生苦、有老苦、有病苦、有死苦、有爱别苦、有怨憎会苦、有所求不得苦、有五阴盛苦、有失去荣乐苦等诸苦，可到底有苦又乐，有香有臭，是一个多姿多彩的世界。……我的信念是，我们应该有肯定世间一切臭的污秽事物的气魄，因为再腐败的土地也会开出最美丽的莲花。如果莲花不出于污泥，而长在遍地天香的土地上，她的美丽也不会那么珍贵。我并不希望人世间都是庄严美丽的世界，也不期待能生活在众香国度，我只想渴的时候有一杯水喝，夜读的时候，有沉默清雅的桂花深香默默地飘来，就够了（《深香默默》）"

　　如何面与生活面对？林清玄这样语重心长地勉励我们："连石头都可以撞出火来，其他的还有什么可畏惧的呢？"

　　（《温一壶月光下酒》，台湾林清玄著，知识出版社1999年1月第一版。）
