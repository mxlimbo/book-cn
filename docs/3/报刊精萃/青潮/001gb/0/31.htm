醉在西双版纳（散文诗）

李智红/文

浓醉葫芦岛

　　一颗绿色的宝葫芦，罗梭江是藤，小孟伦是根。

　　举头埋首之间，尽是淋漓的翠绿。绿的混沌，绿的冥蒙，绿的酣畅，绿的阵营，绿的沼泽……女子陷进去，会纤秀成一棵绿色的槟榔；男人陷阱去，会茁壮成一棵伟岸挺拔的棕榈。绿的集合，三千多种珍稀植物在这里繁荣昌盛；绿的放纵，亿万树葱茏的生命在这里暮雨朝云。凝固的，是绿。奔涌的，也是绿。肥大的蕨类植物，依然沿袭着一种最古老也最坚韧的生存方式，依巨树滋长，借顽石抽根。缠了又缠的古藤，让我联想到一种割裂不开，扯拽不断，万缕千丝生死胶结的恋情。

　　在葫芦岛，每一片绿叶都是天籁，每一滴露珠都是音符。

　　在葫芦岛，一撮青苔就是一枚生命的徽记，一寸枝柯就是一种力量的延续。

　　依兰香的空气，菠萝蜜的雾岚，随意挥洒的万种花香，千寻鸟语……

　　被浓绿煎煮过，烧烤过，炖熬过的太阳，轻飘飘地浮游在一片稠绿的波涛之上，仿佛涨潮的翡翠上点染着一滴醒目的热血。

　　踏上葫芦岛，我们的灵魂就得承受一次绿色的捶打，我们的身心就得经历一场绿色的洗涤。

 

淡醉橄榄坝

　　好一颗耐嚼的"橄榄"。

　　橄榄味的山野，橄榄味的森林，橄榄味的小桥流水人家，橄榄味的风俗民情……

　　橄榄坝，一块充满悬念的土地。

　　庭院式的竹楼星子般洒落，糯米香的炊烟随处升起……

　　春天，自五百多个世纪以前迷途于这块灼热的沃土，便再也拔不出深深陷落进大地腹部的根须。

　　橄榄坝，你就是随手抓一把沙子，也能挤捏出滚烫的油花。

　　从不辜负这块沃土的，是傣家人的勤劳、善良、智慧、美丽。是累累压枝的硕果，是森森若云的橡胶，是昼夜涨潮的浓绿，是四季斑斓的花汛……

　　在橄榄坝的深情厚意中留恋得久了，你也就成了一棵枝繁叶茂的橄榄树。

　　橄榄坝，丰肥得丢一颗石头就会发芽，插一根扁担就会开花。难怪连豪气干云的澜沧江，也在这里蜿蜒不舍，步步回头。

 

魂醉傣家寨

　　依灵山而寨，择好水而居……

　　横看，是一幅散淡的水墨；纵看，是一幅简洁的素描；俯瞰，是一幅意境深邃的蜡染……

　　手挽手的棕榈，庇荫着每一条缠绵而又清幽的小径；肩挨肩的槟榔，高挑着丝绸般柔软的雾霭，羽毛般轻盈的烟岚。

　　佛塔，在散淡随缘中耸峙。

　　佛像，在大彻大悟中静穆。

　　古老的菩提树，枝枝叶叶都蕴藏着参不透的禅机。

　　在傣家寨，最教人心荡神驰的，是爱意氤氲的姑娘茶；最让人刻骨铭心的，是风情万种的孔雀舞。

　　一颗炽烈的太阳，始终鲜亮在傣家人清醇的竹筒酒里。

　　香竹饭的清香，随微风四处飘荡。

　　小伙子深情的葫芦笙，夜夜如溪水般潺潺流淌，直把阳台上那一个个纺线的少女，浇灌得春风满怀，月色融融。

　　在傣家寨，只要采撷到一滴相思豆的殷红，你的心便会潮起潮落。随意抚摸一把凤尾竹的枝节，你的手指便会长出忧郁而婉约的箫声。

　　在傣家寨，即使是一片平凡的树叶，也会让你浮想联翩，魂不守舍。

　　只要走进了傣家寨，你就扣开了真、善、美的大门。
