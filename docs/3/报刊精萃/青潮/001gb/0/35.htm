蚊　帐

题语:没有想到,在小小的蚊帐里,竟然还有不小的文章.一个活着的人的故事......

青年同志/文

　　这个夏季,你同每一个夏季一样,蜷缩进尼龙的蚊帐,你堵者每一个可以发现的漏口,它在寻探着每一丝可能的缝隙,你和蚊子,都很辛苦.

　　你是避免痛苦,它是寻找快乐.都为活着,并且更好.

　　一层浅浅的蚊帐,让里外欲活的更好的生命维系着一个脆弱的屏障,使得你们的生命都是如此的紧张.

　　帐子四四方方地照你的旨意撑着,你的活动的空间也被它轻易地控着.它给你天地,你给了它自己的自由.

　　在帐中你看着外面的动静,听着蚊子焦急的嗡鸣.你和你的朋友一样,甜甜的笼罩在帐子的宁静中,你们的视线隔着两层帐布的缝隙交错,你们的声音透着帐洞回响.一样的眼神,一样的料想,一样的愤怒,一样的不知所以.你们都是同样,唯一的遗憾,你们的手脚,再也无法在蚊帐间延伸,伸长,相拉,摇晃.

　　我知道你们都很无奈,你们被追杀进小小的天堂,可以悄悄的遐想"河岸上的金柳,是夕阳中的新娘".

　　小帐儿悠悠,你的心儿"飞着,飞着,春,夏,秋,冬,白天,黑夜......没有休止".

　　你只顾着把你的思绪送给了那个泉眼旁唱者山歌的美丽的姑娘.

　　哦,执着的蚊儿还是在你快乐而懈怠的关头捅开了你曾经满以为自保的小帐,它只需开个小窗.你未来得及对上那一句已快蹦出牙齿的山歌.

　　无论你如何躲藏,也敌不过小混蛋"向往"你那一贯的痴狂.你同你的兄弟曾经无数次欲放声呐喊,却最终都不由自主地选择了彷徨.

　　于是你对我哭泣.

　　我只好说: 　　

　　活该,你也无妨!

　　遮光避蚊不抵风
　　四四方方尚遗洞
　　喜怒冷看帐外事
　　却为嗡嗡始作蛹

　　补:赐予那些静默者,自安者等,如是.
