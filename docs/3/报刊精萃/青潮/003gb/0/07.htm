我的奶奶


作者：网命徒

　　 在我的亲人中，我和奶奶的感情最深。

　　我从三岁就跟着奶奶睡。很小的时候夜里去厕所，也要把奶奶叫醒，因为害怕必须奶奶陪着，为这奶奶没少骂我，而我那时尿又特多，每天夜里总要起来个两三次。再等我大些了，家里人还让我在奶奶屋里睡，意思是让我陪她说说话，照顾着点。可我正是贪睡的年纪，一宿不醒，奶奶夜里怎么样我又如何知道呢。倒是奶奶常常羡慕的说，唉，真眼馋你们这些年轻人啊，躺下就睡，睡着了象死猪。奶奶年纪大了，觉少，所以这样说。

　　奶奶是从旧社会过来的人，思想上多多少少保留着那时的影子。

　　我结婚后两年没要孩子，这把奶奶急得不行。每次见了我都神秘兮兮的叫到一边，故意压低嗓门问，"小子，过来我问你点事，你老婆，有影了吗？"，其实大家都听的见，因为奶奶耳朵背，说话声音就大。我当然知道她的意思，就说没有，还不想要呢。奶奶更是一脸的不理解，"咋了？没娃娃票（准生证）？"我说有。她更加迷惑了，说什么前院的二小子婚后没有娃娃票，两口子哭了半天。

　　等我老婆怀孕，奶奶又急不可奈的打听，是小子还是丫头，"照照（B超）了没有？"当我告诉她是丫头时，奶奶说好，丫头好，再说你们外面的人也不拘这个。但我知道，象她这样年纪的人，多少还是有一些重男轻女的思想的。但奶奶从来都说女孩好，似乎在安慰我。当我弟弟生了一个男孩后，奶奶高兴的自言自语，"说是男女一样，还就是不一样！"

　　奶奶现在是四世同堂了，逢年过节的，子孙们都去看望她。临走时，她总是亲自给每一个人收拾东西，给谁放什么她门儿清，绝对乱不了。一大帮子孙排着对从奶奶跟前走过，也从奶奶手里接过她送的东西。这几年，奶奶给我们带的东西总少不了窝窝头，就是用棒子面做的那种，是她自己做的。我们都说说奶奶还挺赶时髦，她怎么知道现在正兴这个。但每一次我都发现奶奶给我堂姐们的比给我们兄弟们的要多。等奶奶开始逐个给我们收拾东西时，我就故意站在她旁边，奶奶似乎明白了我的意思，就问，"是不是有点偏心？"我一脸痛苦的点点头，奶奶嗔我一眼，"姑娘嫁了再回来就是客人了，你懂什么！"

　　奶奶疼孩子，特别是对我们这些孙子辈和重孙辈们。每一次我回家，她总是停下手里的活计，搬个凳子坐在我面前，拽拽我的衣服问问什么料子的，穿这么少冷不冷，多少钱呀。当我说出了价钱，她马上板了脸说什么，这么贵啊，以后不好再买这么贵的衣服了，并反问我能穿一辈子？说完话了就这么光看着。有时候我们三口子回去，奶奶就满屋给我女儿找东西吃，能找到的不外是一个苹果或者几个桔子而已，当我女儿说不吃时，她就再去找，这回是一袋饼干，老婆在旁边拉拉女儿的手说，别吃，肯定都过期了。我回头瞪我老婆一眼，对奶奶说她们不爱吃，我吃。这时奶奶才坐下来，把她的重孙女揽在怀里，摸摸头攥攥胳臂，上下打量，不时的说，"瘦，身上没肉啊，你爸爸不让你吃饭？"然后又摁摁女儿的鞋，问挤不挤脚啊。奶奶有个憾事就是老记不住我女儿的名字，也叫不出。当我女儿叫她老奶奶时，她总是内疚的说，唉，你这个老奶奶有啥用啊，连你的名字也记不住，都怪你爸爸起的跷蹊古怪的名字。

　　奶奶还是个闲不住的人，经常的帮我的四叔收拾院子。麦忙时，奶奶竟踩着梯子爬上屋顶晒粮食，一家人都为此提心吊胆，她却象没事一样。四叔回来就大声呵斥奶奶，奶奶象做错了事的孩子，低着头，一言不发，灰溜溜的回自己的屋去了。十年前她的几个儿子就商量轮流给她做饭，她不肯，直到今天奶奶还是自己做饭洗衣服。

　　前几天我回去看她，她正戴着老花眼镜给自己做鞋。奶奶是小脚。市面上也有她穿的那样的鞋，但她总是自己做。我问她做这么多鞋干什么，她说趁着现在还做的动，多做几双，要不以后老了咋办。我问你现在还不算老呀？奶奶扶扶眼镜，一脸正经的说我老了么？看她那神情，似乎从来没考虑过这个问题。我连忙说，不老不老，您才86呢。
