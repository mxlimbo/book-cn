好姐妹

作者: 瑞儿125

　　珠儿最近在一个英文聊天室里新认识了个网友，谈得还算投机。这天，他把电话打到了珠儿的寝室里。正好其他人都上课去了，两人开始肆无忌惮地大放厥词：

　　"你是在读大二了？""是啊。"珠儿对这类问题感到麻木，她知道接下来该问什么了。"一米六七，五十三公斤，大眼睛，白皮肤，长发……好了 ，都告诉你了。""哇~ 大美女啊！那一定有很多男孩追你，有男朋友了吧？""没有！"珠儿没好气地一口否认了。"不会吧？为什么呢？""我，我怕人造谣……"珠儿这样说并不完全是找个理由来敷衍那庸俗的公式化的探问，前段时间是有件事让她够头疼的。

　　上个暑假到爷爷那里玩，大伯父大伯妈也在。许久不见，自是十分亲热。大伯妈是个又黑又瘦的女人，刻薄的脸上刮不下二两肉来。 她拉着珠儿的手，说长道短，平光眼镜后面小小的眼睛和唾沫横飞的刀子嘴堆积 出来的笑意怎么看也逃不脱"恶毒"二字。珠儿微笑着，倾听着，带着尊敬木然地表示同意。大伯妈突然把话题一转："五一节那七天你到那里去了？我们到处找你，你爸爸妈妈把你交给我们的啊！你说你这孩子，去那里也不给我打个电话，急死了人……"珠儿一楞，随即有点过意不去：父母在外地工作，叫自己放长假到大伯父家去，但长假难得，就和寝室的同学结伴到昆明去玩，却忘了还有这一茬呢。"啊呀呀！我说珠儿不会做那种事，是她自己编的还差不多"那女人一拍大腿，作恍然大悟状，还没忘了向这个镇定的女孩投去怀疑的一瞥。歉然一笑后，可爱的伯妈换上一副神秘的面孔，凑近珠儿的耳朵，压低了声音："莎莎说她有个在你们学校的同学告诉她，你在假期里跟你的男朋友私奔了。"说罢，她注意看了看珠儿的脸色，"我当时说珠儿那么乖，怎么会做这种让父母丢脸的事哦……"珠儿只觉得血管陡然扩大了几十倍，"让父母丢脸"却提醒了她：母亲是最好胜要强的，这女人千万别捅给她知道，让她生气。焦躁占据了珠儿的心，同时又感到好笑——我男朋友都没有，我跟谁私奔去啊，谁这么无聊开这种玩笑！珠儿不觉轻蔑地撇了撇嘴。"啊！那个人就是她自己……"一个可怕却又无法推翻的设想，让珠儿的脸变成死鱼的肚皮，一桩桩不愿回忆的往事又卷土重来，迷离了她的眼。

　　莎莎是五叔的女儿，圆圆的脸，和珠儿一样大大的眼，却更圆一些，动起脑筋来滴溜溜地转。虽说只比珠儿小三个月，可从小就赶在她屁股后面姐姐长姐姐短地叫着，姐妹俩甚是亲密。就象大多数的姐妹那样，珠儿是安静的，温柔的，很听长辈的话；莎莎是活泼的，大胆的，常显出叛逆的狡黠和自私的圆滑。两个小女孩在一起玩，常常是莎莎出主意，反而更象姐姐一样照顾，保护着柔弱的珠儿，自然，莎莎成了两人集团的主心骨。从小到大，珠儿一直为有这样的妹妹庆幸，还感觉跟自己妹妹是铁哥们，挺好的。没想到，后来她却改变了这种想法。

　　珠儿的学习一直挺优秀，小学毕业经过推荐考上了市重点，而莎莎所在的小学与那中学对口，所以两姐妹竟碰面了，从此生活在同一片狭小的天空下，有什么风吹草动的彼此都知道。由于学习勤奋，贫民窟出身的珠儿在年级里依然雄居榜首，天性聪明的莎莎总是漫不经心地，成绩平平，还隐约听说她和所谓的"黑社会"有什么关系。幸好不在一个班上，没怎么尴尬。但毕竟还是一家人，彼此的父母是兄弟，是妯娌，是叔嫂，聚到一起免不了互相攀比。现在想来，莎莎一定受了叔叔婶婶不少责骂，心里恨得牙痒痒。我还不知道，只是一味对她好，她定是以为自己在装好人了。唉，人心难料啊！这贫贱高低把我们生分了。"不对吧。还有件事你忘了？"有个声音从珠儿心底冒出。珠儿迷惑得很。"初三的事啊！"那声音见珠儿不象装糊涂，善意地提醒道。"哦，是这样，是这样的。"珠儿汗如雨下。

　　"后来我们升到初三，我和莎莎分到一个班上，我以为从此可以和她更亲近了，却不知她成天干些什么，想些什么。隔膜在悄悄地滋生，增厚，她在我眼里只剩下个灰白而模糊的影子。

　　那天，我跟寝室里的婧去打开水，我们正说着笑着，迎面走来一个男孩。'嗨，骨头！到那里去？'婧热情地打招呼。'呵。'他轻轻地一呼气，就算回答过了。我这才看清他的样子，窄窄的一张脸，大人气的脸，淡淡的眉，细长的眼挑入两鬓，若有若无的嘴。什么都淡，什么都冷，正如他的态度，拒人于千里之外，却又象化好了妆的戏子，即使凝然不动，也兀自咿咿呀呀地唱，浅笑着，暧昧着。他全身上下散发着灵异的气息，包括那白色运动服的背后飞舞着的两只妖艳的凤凰。他走远了。"他是田径队的？我怎么没见过他？""他你都不知道？我们年级的大帅哥哦！"婧有点夸张。我把才发的半期考试成绩单拿来看，我们年级有三百多人，从我开始往下数，眼睛都看花了，蓦然回首，他的大名剌剌地躺在最底端。我苦笑了一 下。自小就养成好强的性格，想要什么就努力去拿，毫不犹豫。我不得不承认，他吸引了我 。我从来没有见过他那样的男孩，漠然的，怪诞的，向任何人吐着芳香而甜蜜的毒汁。从那次偶然会面后，我便开始留意他的行踪，假装无意跟在他身后，听他说话的声音，躲得远远的看他抽烟的样子，甚至红着脸和他面对面走过。我得了一种病：一看见那对凤凰就会止不住心漏跳半拍，那红，黄，绿，白织成我生命中一段无可比拟的色彩。他的名字里有个'春'字，那段时间里我能把所有带'春'字的唐诗宋词倒背如流。也许我真的是疯了。渐渐地，他好象发觉了我的异样，碰到我时态度扭捏起来，脸竟微微发红。我更犹豫了，他会接受我吗？ 我是年级第一，是学习委员，是团支书，是老师的宠儿；他是最后一名，是黑社会的，躲在角落里懒散的人，只爱抽烟，玩电子游戏。我还以为我们也可以有传奇的爱情，终于鼓起勇气给他写了一封信，信中所云今天已不得而知，但也应该无异于《凤求凰》的琴音。

　　过了几天，中午，我正在教室里看书，莎莎却一反常态地走进教室。她拉着我的手："姐，我们出去说会儿话。"我茫然跟她来到寂无一人的操场上。她从兜里掏出一张纸，递给我。我的脑袋轰的一下，不知道自己身处何处——那纸正是我给他的信！她埋着头，严肃但迟疑不决：'是骨头给我的，我就是觉得这笔迹好熟悉，没想到是你。'她换了一口气，好象决心把一切都说出来，'拿回去吧，他现在是我的男朋友。'怎么会这样！我强作镇定，轻笑道：'不错，我 是喜欢他。'她扬起头，认真看了看我，说：'我把 他叫来了，你们聊聊吧。'果然，他从另一头走了过来，依然那么潇洒，脱俗，漫不经心。那令我朝思暮想的身影此时却给我带来了巨大的恐惧和愤怒，这算什么！示威吗？怜悯？还是显示她的宽宏大量，以及对姐姐的尊敬？我在心里冷笑，手却不住发抖。我和他打了个招呼，莎莎离开了。他毕竟太年轻，做作的坦然掩饰不了心里的局促。"莎莎说你是她姐姐？"我回答了。沉默。过了很久，我才问；'其实，我一直有个问题想问你……'我想问——你喜欢我吗？——可是现在看来多么可笑！'算了，也没什么。'' 没什么就别说了！'好象是想早点结束这样的尴尬，他粗暴地打断我的话。恍惚中我回到教室，她迎着我，我笑了，朝她点点头。第二天，骨头从门口经过，一个纸飞机'嗖'地降落在我的面前，我匆匆看了他一眼，他已吹着口哨走远了'我很喜欢你妹妹，我爱她，请你不要打搅我们'——纸飞机对我说。屈辱与我的自尊扭打着，把心撕成一片片。我飞奔出校门，搭上回家的班车，坐在靠窗的位子上，任泪水纵横。不知为什么，我小声唱着：'Bone ,bone,bone is mine.'是大大泡泡糖广告里那支歌的调子，很轻松，很快乐。

　　日子过得很快，要中考了。我由于惯性又投入学习中，同时也原谅了自己。模拟考试下来，我仍是第一。按班主任的新规定，我可以第一个选自己的座位和同桌。莎莎突然冒了出来，要求我帮她，做她的同桌，要不然她就考不上高中了。不知是处于歉疚，心虚，还是同情，我毅然放弃了优越的条件，搬到最后一排，与她并肩。我已失去了思考能力，尽管这并不聪明的举动带着大义凛然的成分，我的固执还是让疼爱我的班主任拂袖而去。

　　再后来，我们都上了高中；再再后来，我们分别上了两所不同的高校，素无瓜葛。

　　如今，她又回到我的生活中，以一个造谣者 的面目。"

　　珠儿想到这里，好象明白了，从头到尾。"原来是我错了，一直都是，我们还是好姐妹啊！"瞧！多聪明的结论！
