流浪的猫去做哲学家了

作者：青年同志

　　萨特曾经这样断言，世界上只有两种人：一种是痴爱猫的人，一种是厌恶猫的人。意外的是，我对猫的感觉是非常平淡的，正好交叉在他分类的之间。

　　假如我家从来没有过一只黑猫，我对猫的直面接触顶多只是邻居的一些花猫而已，它们喜欢半夜到我家来偷嘴，油漆的桌面上第二天总是梅花的尘土依附着。我对猫所以在此就没有特别好的感情，要是在夜晚听见响动，起床出来阻止的刹那，不但打不了它，桌子上的东西而且还会让它在紧张中碰坏。所以白天我见到猫的出动，总是给它一个突然一击，达到解气的目的。

　　万万没想到的是，我的父亲在出差的路上收养了一只黑猫，瘦小得很，而且是公猫，一般人是很少要的。这只猫来我家开始的时候一点脾气也没有，虽然是无锡市区捉来的，到了维扬城下的乡村里没有一点公子哥的傲气。那段时间，我们那里办理"农转非"的人不少，我戏称咱们家的猫是"非转农"了，希奇。我多在外面上学，很少在家中，母亲对猫多为厌恶，而父亲却偏爱得很，每天都贡上新鲜的鱼食。从前形影单薄的身板很快臃肿了许多，虽是公猫，可能是城市生活习惯的原因，它在开始到我家的那段时日里，从来不出家门，就向小家碧玉一般。守在闺中的日子里，做得最多的是讨好主人。盘着家里人的腿转，不过家里定了规矩，在午饭的时候，不管它怎么叫嚷，也绝不给它得逞。

　　父亲在家就是猫的最幸福的日子，这个时候，它一般都是受到最好的照料，而父亲却是经常出差的，所以在第一主人不在的期间，它无形中被冷落了很多。终于有一天，它不在恋家。在一些母猫的叫唤声下，看到外面的世界更大，胆子很快大了许多。晚上，似乎又怀念了城市的夜生活，只要外面一有"暗号"，马上窜出院门，投奔异性的关怀去了。居然由于此，它对生活的要求降低了，看见美餐也不是很兴奋了，仿佛夜晚的精神生活更加罗曼缔克。猫也有争风吃醋的，一个早晨，我无意中看见它躲在小屋里舔着尾巴，走近一看，原来是尾毛少了一撮，看来是和哪家公猫打架输了。看它一脸的苦样，别提是多么有趣了，这个时候，它便冲着我诉苦似的啼叫。我笑着心里暗想：强龙也压不过地头蛇啊，何况你是个孬小子。　

　　别提了，我家那猫还比较知趣，一段时间，什么诱惑都不再让它心动，重新守着门口只敢张望，直到伤口愈合......

　　就这样，反反复复，经过几年，它算是咱们家附近的老猫了，养猫的人大都知道，老猫是最不好玩的，特别是女孩子。所以我舅舅家的表姐过来看黑猫，直说是个"呆猫"，一点生气也没有，只知道睡觉以及吃鱼。懒猫毕露了。

　　最近一次回家，听说它已经出走了，而且还回来过一次，但还是走了。父亲显然有点若有所失的感觉，而让我意外的是，一直讨厌它的母亲也流露了几分遗憾。

　　看见不少人都说，喜欢猫的女孩大都是自私的，我不知道什么样的逻辑，但我认识醉完整的猫眯就是咱家的那个，彼得.梅尔用他独到的语言表达了《一个狗的生活宣言》(更改：《一只狗的生活意见》），创造了一个狗"仔仔"的奇迹。我不知道咱们家的那只懒猫是否也会那么智慧。假如它还会回来的话，或者找到了"猫生"的真谛，能否也给我们一个宣言？但愿，流浪的猫去做哲学家了。
