我给我家咪咪选择了安乐死


作者：飘雪

　　很早以前我写了篇祭奠我家猫的文章。因为那时候我很伤心，今天是我家猫来我家第6个年头的日子，我觉得我把这篇文章贴在这里比贴在我那里更合适，于是……

　　咪咪很瘦，因而眼睛显得出奇得大，她已经有3个多月只靠流食生活着。现在它很安静地躺在那里，好像是知道她就要死了。20分钟前我们给她注射了安乐死的针，她一点也没反抗，即便是针头进入她的体内时。突然，她奋力地抬起头来，眼光落在我母亲身上，很留恋地，然而无力地，终于永远地睡去了。

　　咪咪刚刚断奶就被我抱回了家。同所有高楼养猫的家庭一样，住在6楼的我们，从来不让她外出。咪咪对于外面的世界既渴望又恐惧，每天她吃饱喝足后，总是趴在窗台上，向外望着，然而真的抱她出去，她又在人的脚下打着剧烈的哆嗦，并且瘫软在地，不知所从。有时窗外有鸽子成群地飞过，咪咪马上很兴奋，将身体立起来，爪子向前伸着，像一张猫皮似的贴在玻璃窗上。又偶尔趁家里人忘记关上阳台门的疏忽，窜上阳台围栏，表演高楼平衡木。由于猫一贯以其灵活著称，所以我们谁也没有在意，只有母亲偶尔会说："这万一、万一……唉。"母亲声音很小，仿佛总说给她自己听的。

　　然而，"这万一"有一天真的成了事实。它让我觉得一辈子有愧于咪咪。

　　那天早晨，咪咪像往日一样突破阳台门的防线，跳到围栏上，她轻松自如地走着，我站在阳台门里欣赏着她。突然毫无预兆的，咪咪跳到了隔壁邻家的围栏，邻家的阳台是封好的，围栏的边很窄，我吓了一跳，不经意间手臂撞到阳台门上，门发出了很大的声音，咪咪受到了惊吓，一步没有踩稳，随着一声惨叫，摔下楼去。我冲上阳台，只听见"砰"地一声，咪咪重重地落在楼下的土地上。等父亲将她抱上楼时，咪咪的左半身被血污完全盖住了，全身抽搐不止，双眼鼓出很大，舌头伸得很长，只剩下出气的份了。母亲拿来云南白药和生脉饮冲剂强行给她灌了进去，她将大部分药水和着血沫喷了出来，但是药仍是起了效果，咪咪喘气的频率渐渐慢了下来，也不再吐血了。

　　接下来的日子我天天在医院里，看着咪咪接受治疗：打针、打点滴……咪咪的情况慢慢地稳定住了，但始终昏迷不醒。又过了一个星期，一直呼吸微弱的咪咪突然转醒了，并且有了意识。那时，我和母亲正在厨房做饭。咪咪突然摇摇晃晃，一路走一路"喵呜喵呜"的叫着，我和母亲一时竟然激动地哭了：总算一切的努力没有白费！接下来的日子，我们每天活在惊喜中：咪咪开始吃东西了、咪咪开始能小跑了、咪咪又会舔自己了、咪咪终于会排便了……我们为她的恢复叹服着、惊喜着，同时感叹着生命的顽强

　　从那以后，咪咪的消化系统变得很弱，吃的食物要极讲究：要半温状态、很细碎、很烂，不能是剩的，要新鲜。咪咪一次不能吃太多，而一天要吃多次。母亲从此就多了一份负担，每天侍侯咪咪吃饭，稍没做好，她就会吐。母亲常常对我说：别人说猫有九条命，这一摔大概只剩下半条命了。

　　这样子过了两年多，咪咪临死前的3个月开始不能进食。她总是很惨很惨地对着食物叫着，然而只要一吃就马上吐出去，只好用葡萄糖水维持着生命，每日缩在角落里，谁都不能碰，孤寂而痛苦地生存着。我们走了很多家宠物医院，大夫都说它的肠胃功能已经萎缩了，它只能这样拖着直到死，安乐死可能是最仁慈的解决办法。

　　我们反复思想斗争，看着咪咪日渐消瘦的身躯、古怪而苦痛地睁大的眼睛，最终为她选择了安乐死。

　　可怜的咪咪，在不足岁的生命里，痛苦却多于欢乐。
