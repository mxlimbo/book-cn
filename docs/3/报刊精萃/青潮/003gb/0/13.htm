BBS杂感


作者：天雪

　　内容: 作为真正网络社会中的人，聊天室、BBS、OICQ、ICQ是访问量最高的网络社交场所。BBS在思想交流方面远远超过其它三种，当然各有各的用途，它已成为网络中人最主要的思想交流场所。今天，我就说说这个BBS吧。

　　第一次接触BBS时，还不知道BBS的真正作用，因为那时初学会上网，是去年冬天11月份的事儿，只知道聊天室怎么进、怎么与人说话，并不知道除聊天室之外还有论坛这么个网上去处。接触BBS是因为上网没一个月，我就拥有了自己的个人主页，有主页必有留言板，可我并不知道留言板其实就是BBS，只知道每天有人给我留言，而我却很少回复或根本不回复，只看看都谁来看我了，大部份都是聊天室的朋友们，而我那时几乎天天都泡在聊天室，也就没回的必要。

　　真正接触BBS是去年刚过完年，聊天室的宏天在东网的情感天地发了一贴，说死说活要让我上去看看并给他回个贴，费了半天周折进去一看，论坛上到处都是长篇大论，还有那么多人在那儿发贴和回贴，心中一阵恐慌，尤如刘姥姥初进大观园般，哆哆嗦嗦总算给他回了两句话，还反复验证句子是否通顺后才敢发表上去。呵呵，回想最初上BBS的狼狈相，再看看今天发贴是如此轻松自如，心态是如此平静，有时着急发表（急于下网办事）未校对就发了上去，真是天壤之别。

　　现在细细算来，我上网的大部分时间都泡在了BBS上，也就有了很多感想。上网的人，尤其是爱泡在论坛的人，一般都是有些思想、有些文化基础的人，不论是好是坏都能写上几笔，由于在现实中人们之间的相互交流很难或说不如在网上交流这么方便和简捷、直观，抑或是网上这种交流的互动性给了人们更开放、更自由的空间，所以爱上了论坛，爱上了发贴子，发上去就希望有人看有人评，如果发上去后，贴子受到冷落，论谁心中也不会太好受。

　　我想说的是，每个发贴的人都希望有人关注你的贴子（尤其是那些上网久了的人），可你是否关注过别人的贴了呢？所谓交流，难道只让别人关注你，你不去关注别人；或别人关注了你，你才去关注别人吗？难道你就不能先学会关注别人吗？当然，上网费用太高，上网时间有限，网上你还有太多事务要处理要完成，你不可能关注每一个人的贴子，但你最起码可做到除你的贴子之外，对别人的贴子做到多一份的关注吧，尤其是对于那些新来的人，他们更需要被人关注。

　　生活中不容易做到的事，本以为在网上会容易的多，可我发现并不如此，"索取"和"付出"这一对生活中难以平衡的矛盾体，在网上同样难以平衡，每个人都希望自己的付出能得到平等或差不多的回报，谁都不愿意更多地或说是无为的付出。有些人付出了，没有得到他所期望的回报，也就寒了心不再付出；更有一些人连付出都没有，只想一味求回报，没人回贴，就觉得这里的人没人情味，不懂识他这块美玉……谁都不舍得付出，结果必是谁都没有了回报，人性中的真善美和温柔又都被残忍和冷酷所压制。当然，我说这番话有些过激，网上的善人善行和无功利的事件远远高于现实生活，BBS上也并非我说的那样，事实上一直有人在无回报地付出着，诗园中的红尘、okokok1、我是乎做得最好。

　　记得我还当文坛版主的时候，蓝天曾说我为什么总给别人回贴，有那时间为啥自己不多写一些，让别人给你回贴呢？我想了又思，思了又想，想了很多也思了很多，感慨颇多，那时就想写一篇类似此贴的文章，标题为《我更想作你们的读者》，由于某种原因一直未动笔。每个人都需要倾诉，如果大家都倾诉而没有听众的话，倾诉还有意义吗？内心的孤独又怎能释放出来呢？

　　我只希望：朋友们，当你们倾诉的时候，可否也当读者倾听别人的倾诉呢？互为读者，互相倾诉，那样岂不更好？！
