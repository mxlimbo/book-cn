感怀

（作者：莫人）

萧萧西风劲
那堪冷风寒月
妆罢归隐去
推窗拜天
彗彗北斗
夕然泛银
乍起酣雀
故国东流去
今宵良夜
喟周郎呕血
霸王刎别
往昔之
弃我者幌觅踪影
低首入画眉
慨而歌之
击楫中流
痛失昭华
君不见
八公山上
草木皆兵
空城之上
孔明赋琴
问我何求
叩我何忧
枉一世
年华如水流
空负一身
罢罢罢
我又何忧
我又何愁
江州泣下
无语泪流
吾辈忽起
豪兴逸飞
莽苍苍
大鹏击天
当世之时
英者
勇者
雄者
舍我其谁
昆仑巍巍
青峰直指
汪洋恣中流
泣者
歌者
狂者
痴者
奈若何
九华山上
卓而秀者！
