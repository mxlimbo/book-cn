老子的人生历程，
我们今天还需要倾听老子的声音吗？

——《老子隐迹》自序

作者：重构

　　自序：

　　在人类历史的长河中，最伟大的思想家寥寥无几，老子正是其中的佼佼者。

　　老子的五千言《道德经》，两千五百年来，不断地启迪着人类的智慧，无数学者都为其倾倒，从中汲取营养，继往开来，在茫茫的宇宙中，思索着人类的未来。

　　但是，老子的生平，史书记载极为简略，司马迁在《史记》中为老子作传，仅数百言而已。

　　因此，两千五百年来，老子的光辉形象，始终没能在文学作品中得以再现，这不能不说是人类的极大遗憾。

　　笔者在1991年秋天撰写《探寻圣人的足迹》（该书探讨了老子、"上帝"、释迦牟尼、孔子的思想，并翻译了《道德经》全文）时，便试图勾画老子的生平。1992年春天，观看了电视剧《孔子》之后，便萌生了以老子生平及思想为题材，创作一部文学作品的念头，并于1992年秋天动笔。

　　应当承认，这是一项非常困难的工作，当然也是一项非常有意义的工作。笔者依据的材料，主要是《道德经》和《史记》老子传，以及先秦诸子的简略记述。

　　既然考古学家可以依据史前人类的头骨复原史前人类的相貌，笔者相信根据老子的《道德经》，复原老子生平及思想发展脉络也是可以做到的。

　　有趣的是，笔者在创作《老子隐迹》这部小说时，只有一个极为粗糙的提纲，故事情节的展开，是在撰写过程中自然而然出现的，并且几乎保佑任何动作修改，一切都仿佛历历在目。或许老子的在天之灵，在恍兮惚兮中，传来了某种声音。

　　当然，这其实是一种机遇，适其时适其地适其人便会发生其事。

　　两千五百年过去了，老子遇到的问题，老子思索的问题，许多在今日依然存在。或许两千五百年太短了，人类还不足以解决这些问题；或许两千五百年太长了，人类应当能够解答这些问题了。

　　笔者相信，人类最伟大的思想家，他们的工作，实际上是在代表人类进行选择，即为人类选择一条新的发展道路；他们的声音，人类可能要经过许许多多年，才能够听清楚。但是，如果我们不肯倾听他们的声音，我们将永远听不明白，继续受制于基因与大脑之争。
