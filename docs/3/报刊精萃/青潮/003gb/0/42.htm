塞上别——昭君出塞


作者：应文坚

夕阳如血
将大漠赭黄的沙尘
侵染成血红
日久的枯骨
半隐半露于黄沙之中
以一种无奈的姿态
拌和着风沙席卷的啸声
唱着往昔命运的咏叹调
车辚辚,马萧萧
一组车队从繁盛的中原飘来
坐于车中的女子如烟如画
可是人们倾慕已久的昭君
中原不盛盛边戎
中原的女子源源不断地
将自己的青春和幸福
在干涸的大漠中
碾成粒粒尘埃,尘埃粒粒
夕阳残照
可与华夏和亲的女子
吐出的血一般猩红,一般鲜艳?
