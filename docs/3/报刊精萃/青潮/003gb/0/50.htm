二十一行诗二首——《对话》、《暴雨》

作者：麋鹿居士

对 话

如果一座城市，大得我无处栖身
面对陌生的道路和人，一张地图
如何把我引向宁静

如果我一分为二，精神的和肉体的
在四处都是方向的立交桥上
是否可以毫无牵挂地选择，背道而驰

如果精神痴迷于阳光默读的草坪
而肉体走向灯火迷乱的夜的深处
是否还可以清清白白地，合二为一

两只眼睛必将为没有露珠的早晨打湿
一只是因为丧失；一只是因为坚持
像两个对扩抗的主义相互敌视

而这只是一种假设，存在之中的虚拟
现在我是坐落于一片空地
四周是石头和人群，凝固审判的姿式

这就是我的世界吗？我的生活
陌生的道路写满了每一张熟悉的面孔
我一次次询问，却都陷入了哑语

呵，失语的时代！我也终将哑着
任内心的潮水朗诵汨罗江上的《天问》
任逃循的飞鸟，洒下带血的啼鸣

 

暴 雨

闪电反叛的巨爪撕裂天堂的大门
我所说的暴雨来自这个夏天
失眠的蚯蚓，最先带来反潮的信息

手持大哥大的经理联系着情感的业务
密雨网织，一见钟情的是屋檐下
两个避雨的离婚的金戒指

——快回家哟，放学的孩子们
茶馆门前，这场兴奋的雨
让卖伞的老阿婆懂得了市场经济

一双脚在速度中反复提升，躲避
一件连衣裙出卖了少女的秘密
搬家的蚂蚁，它所倾向的是一棵树的抗拒

老地方的乞丐幸灾乐祸
他没看见这场雨
他看见的，全是放大的——我

一群鸽子冲进此刻的雨中
我听见了它的"咕——咕——"声
雨雾中弥漫着失恋的情绪

来吧，呵来吧！更大更猛些
"可能的证明，可能的检验"
倾听的人，在一本潮湿的经卷中喃喃自语
