后记：

这是为唱和中青网新"扮主"莫人的历史咏叹"萧萧西风劲"而作。

历史的事件，全凭头脑记忆，个别地方可能有差异。

*D.C.-华盛顿的简称

*比利时森林大穿越*――德军将领古德里安以坦克兵团绕过马奇里防线，一举突破

*敦克尔克沙滩现奇迹――因希特勒希望英国屈服，没有着急围阡龟缩在海边的英军，致英军得以动员所有船支，一夜抢渡，逃回英伦

*巴巴罗沙开战端――巴巴罗沙是德军统帅部进攻苏联计划的代号

*伏尔加河畔鏖战急――这一点记不太清楚，斯大林格勒是靠伏尔加河还是第聂伯河？ 反正有史家评，拿下斯大林格勒的目的是切断河上的运输。其实，军临城下，这个目的已达到。但后来拿下或守住该城已成双方意志的象征。于是，一个个的德国兵团投入斯大林格勒，又一个个化为斋粉。

*长津湖，青川江――见"百战青锋"，宋时轮将军每向长眠在长津湖的英烈鞠躬，致意。

黄果树大瀑布与尼亚加拉大瀑布。本人曾去过位于贵州的黄果树大瀑布，在国内可称难得。为石灰岩岩溶所形成的景观。其上游河流可称水源充沛， 所以有这长年不断的飞瀑给游人以视觉的享受。该地区相对比较偏僻。 不象其他游览区人满为患。不过此一时彼一时，十几年后的今天不知又是如何了。我那位湖舟(胡皱)朋友还以黄果树大瀑布作为他的硕士毕业论文。年代久远，才想起来，他答辩我还在场。

尼亚加拉大瀑布则是位于美加的著名三大湖区。瀑布位于伊尔湖和安大略湖之间，水源充足，其上下游承结天成。这几个大湖钓鱼才是真正的乐趣所在。不象上游河道窄的黄果树大瀑布，这里上游河道平稳宽阔，只是地形的原因，到瀑布处水到瀑成，声势浩大，极其壮观。
