"虚伪"的雷锋精神

作者：飞鸿

　　内容: 如毛主席所说，世上没有无缘无故的爱，世上没有无缘无故的恨。同样世上也没有无缘无故的高尚与伟大。更没有无缘无故的先进人物。雷锋是与他的时代共生。

　　五六十年代是我国的新制度初步稳定时期。广大的人民在要大饼时得了西瓜很有些满足。更加热情地拥护我们的党，投入的进行新中国的建设。而在中国沈阳军区，一个小战士也积极地上进。他就是雷锋。

　　在此不得不搞清当时的社会大背景。众所周知，我国建国伊始就完全照搬了苏联的经济，教育，文化，军事等社会的方方面面。军事上要求"忠于革命忠于党"，作一个国家大机器上的零部件，能达到一般的要求。不是要求独立思考，更不谈个性。思想力求单一纯洁，对西方的东西视若洪水猛兽拒之门外。马列被当作绝对真理，用以"武装"战士的头脑。

　　再看雷锋，还得注意几个关键词：螺丝钉精神，好人好事，为人民服务，马列主义毛泽东思想。在著名的《雷锋日记》中就记载了学习马列主义毛泽东思想的事情。以及做社会主义建设中的一颗螺丝钉。也就是说，他有非常科学非常先进的理论为指导。按行为与意志的辨证法来看，在提倡高尚与无私付出的理论下，他才有化为具体的"好人好事"的行为。再加上自身的努力，逐渐为地方军民所熟悉。

　　同时那也是一个造神的的年代。《离开雷锋的日子》里叙述的：乔鞍山无意造成他的死亡。结果却以因公牺牲上报。毛主席一句"向雷锋同志学习"，名载新中国史册。

　　随着时代的前进，正如乔鞍山的经历的说明，遇到越来越多的坎坷和不平。他被社会当成一个另类。在九十年代，甚至有某个军区的战士因为学雷锋被人们当成疯子送进精神病院。

　　这又是为什么？中国有句老话："世易者时移，时易则备变"。在今天，思想的多元化和人们的信仰的破碎，以经济建设为中心摆脱了当初的束缚；功利化和世俗化成为普遍的现实。没有一个人敢说自己还信仰马列主义，并且完全按照毛泽东思想来要求自己。而，正是以它们为思想基础的雷锋精神理所当然的成为空壳。所谓的好人好事不过是千年以来一脉相承的社会道德和个人良心的泛化的表现。与真正的雷峰精神是毫不相干的，在一个丧失了实质的年代。大力鼓吹的只是虚伪的雷峰精神。^0^
