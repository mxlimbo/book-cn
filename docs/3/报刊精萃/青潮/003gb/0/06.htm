我码字儿


作者: 网命徒

　　码字儿一说缘于王朔，就象老舍先生说自己是写家一样，王朔说自己是码字儿的。听起来似乎很轻松，如我们说玩电脑，他就是码玩字儿。 　　

　　我也喜欢码字儿，特别是在电脑上感觉就更象了。一个个的方块字排起来，可不就是码吗，象一排排的方砖。我喜欢码字的历史很早，得从小学三年级说起，在一节语文课上，我的作文被老师在课堂上念了一半——恰好下课的铃声响了。就是这半篇作文，点燃了我对文学的热爱。我觉得我有写作的天赋，中国作协就是我将来的单位。 　　

　　自己的文章被老师在课堂上念给大家听，是我早期发表的一种形式，感觉真好，更要命的是一些女孩子似乎没听够，在下课后还来借去再看看，这种事情，就是木头人也该发两片绿芽了，何况是我，我能不猛受激励吗，所以说它贯穿了我整个的学生生涯，也就不足怪了。 　　

　　但我真正发表的东西很少，倒不是没投过稿，人民日报海外版都寄去过，就是没回音。我把自己发表的文章称"豆腐皮"，以有别于人们常说的"豆腐块"，乍一听好象是谦虚自己的水平浅，写的东西没有厚度，很表面化，其实我是不愿和他们一样人云亦云，我得特别，因为已经有人称我作家了，我，我，我是作家了呀——这就是那时候的我。 　　

　　随着年龄的增大，文学，逐渐的被我淡漠了，找一支笔也难，还码什么字。曾有那么一阵子，我喜欢对那些成天鼓捣诗啊文啊的人说，饿死你们狗日的文人！　　

　　想不到有了电脑，我又重新拾回了码字的感觉，文学的感觉了。特别是网上还有BBS这个好去处，好，我喜欢！凭着我写作文的水平，也热心的搀和起来，也就几天工夫，我就看到了自己的不足，网上真有能人啊！还去作协工作，就是给作协看大门，人家也不要我——我近视眼呀。好在我已经没那么多的功利心了，对别人和自己都能淡然待之，就是说，明知自己不入流也还是往里挤，不脸红。 　　

　　昨天一网友QQ我，说我的东东被收录在《青潮》里了，两篇。我赶紧下载了看，果然，不过是三篇（还有我用另一个名字的），我太高兴了。听说e-book正在走红，不想我也时髦一回。激动过后，我才发现我还是有一点点俗的，要不我怎么会这么得意？看来，文学是唯一能让我起功利心的东西了。
