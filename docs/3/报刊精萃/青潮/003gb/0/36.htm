忧郁的碎屑


作者：浪漫一生又何仿

　　似乎，我应是离忧郁好远的女孩子，亲人，朋友都只爱见我兴高彩烈的样子。那么，快乐的我一秒钟里，六十分之五十九都是开心的模样，剩余的那一刹那－－便是我深锁箱中的忧郁的碎屑。

　　忧郁，因为什么呢？只怕是那想起你时突如其来的孤寂所致吧。分住在不同的城市里，用不同牌子的电脑上网，使用不同的汉字输入法，成千上万的在线用户，偏偏，你选择了我。如果说从前我拥有过百分百的单纯乐趣，那么＜信仰＞过后，此般快乐已有一丝属于你了。这不算是爱情呵，爱情应是温暖的手，和煦的笑脸，这些我们都不能彼此分享，何况我的笑，为你而笑已被留在过去 ，在线的好友，难得有你了。

　　上网的最初，朋友说小心网恋吧，小心那个懂你心事的感性男子，不置可否的轻笑出声，网恋，多么可笑！现在，笑的人中少了我一个。仍是不看好网恋的，只是有故事告诉我，网 上也有生死两相许，愿意相信那全是真实发生过的，以便给自己找一个忧郁的好理由。你说过的，我是一个狡黠女子，事实不重要，聪慧的笑声里面，是因为用心。

　　朋友一个个送来离去的消息，先是你走了，又走了一个，曾经很放肆的玩笑，为你们封存 忧郁的碎屑里，只等时间来开启。只怕，飞出去的鸟儿只是要振翅高飞，已渐渐淡了想我的心迹，毕竟朋友丢了可以再找，前途只有一个呵！住在自己结的茧里，若要忽视季节的变换容易，不去想你离去的忧伤，好难，做完所有可以占去相信时间的事情，等我的，仍是独自一个人的忧郁，我不飞，飞不起也飞不动，无法跟上高飞鸟儿的步子，所以，我不说挽留，把祝福一遍遍说给你：走吧，飞去，把忧郁留下来，金色太阳属于你啊！

　　阳光照射进来，慌乱间打碎我封存忧郁的箱子，于是漫天飞舞着片片只属于我的忧郁的碎屑，光线里无处躲藏，只好轻轻的飞着，等人来安抚，来洞悉。
