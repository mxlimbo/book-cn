响 指


作者: 洛阳阿木

　　在一个清凉的早晨，我把一个女孩加入了好友。

　　响指。

　　她的名字让我想起了过去的一个年代，我们曾经背着黄书包，穿着黄军装，打着响指，吹着口哨，目不斜视，从女生面前走过……

　　任她们的笑声从背后响起。

　　我问：你怀旧？

　　她答：不，我现代。

　　我问：为什么要叫响指？

　　她答：我喜欢。

　　我问：喜欢打响指？

　　她答；不，喜欢这种声音。

　　我忽然有一种要见她的冲动。我为这种想法感到可笑。但我还是想见她，想看她打起响指，想听听那种声音。

　　她却拒绝见我。她说：有意思吗？

　　我却无法把这种冲动从心中赶走。每天静静在Q上看她的头像，我想，她会是一个什么样的女孩呢？

　　我就守在她常去的那个网吧门口，就象当年守在恋人家门口一样。

　　那个穿一身蓝色碎格子，披肩头发的是她吗？

　　那个戴着眼镜，手里拿着一本书，匆匆走过是她吗？

　　那个梳着两个小辫子，清纯模样的是她吗？

　　我没有上前，我知道她总会有一天来见我。

　　我说：不见我吗？

　　她嘻嘻笑笑：我见到你了呀！

　　我说：为什么不给我说话？

　　她说：说什么呢？

　　我沉默。说什么呢？

　　我说：我会等下去的，等到有一天，一个打着响指的女孩从我面前走过。

　　她笑了：随你！

　　我肯定那个戴眼镜的，常拿书的是她了。

　　她天天都要在饭后和晚上上网，正好符合她的上网时间。

　　她每天都要向我看几眼，然后进或者出那个网吧！

　　看着她来了，我就要上前打招呼了。

　　心忽然就跳起来了，就象当年见到我最喜欢的人从她家门里出来一样。

　　我还没把话说出，有人在后面拍了我肩膀一下。

　　我回头，看到一个和我差不多高的女孩。

　　她朝我笑笑，接着嘴角朝上一翘，顺手打了个响指。

　　黄色的短头发；涂着一种淡紫色，似乎反着光的口红；长长的眼睫毛；嚼着口香糖；若有若无的笑容。

　　她说：你好执著啊，要是你年轻十岁，我一定会嫁给你。

　　我忽然就笑了，然后慢慢点了一支烟。

　　这时好象有人喊她。她回头，给前边一群和她一样的少男少女抛了一个媚眼，然后说：说，请我吃什么？

　　我说：我为什么要请吃饭？

　　她说：你可以说请或不请，三分钟决定。

　　她要吃西餐。

　　她吃着汉堡包，喝着咖啡，细细致致，就象是在作一件艺术品。

　　我透过淡淡的烟雾，看她十六岁的面孔。

　　她抬头：拜托，你是我的叔叔呀，不要这样看我啊，我会受不了的。

　　我被烟呛了一下。

　　再打开Q，看她亮亮的头像跳动起来。

　　她说：嘻嘻，你好可爱啊！

　　我哈哈笑起来，然后把她的名字改了：

　　不是响指。
