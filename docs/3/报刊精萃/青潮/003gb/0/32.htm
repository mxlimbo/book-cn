艺术和流氓


作者: 十一郎

　　去农村写生的画家将穿戴整齐的姑娘统统画成裸体，被村民痛打，倒也是奇事。

　　他说是艺术，村民说是流氓。能将穿戴整齐的姑娘统统画成裸体的画家，只能是个疯子，还不够流氓的级别。

　　如果画人体就是流氓，那画家全是流氓。其实并不是。

　　当人体艺术风靡时，中国人民有的避而远之，也有人翻录至黑市高价售出，买者偷偷带回家中一饱眼福。究其行为，称其流氓极为勉强，仅仅心理不大正常而己。而翻录者，却是大大的流氓。

　　性，并不是一种文化、一种艺术。

　　许许多多自诩为作家的先生和小姐，隐姓埋名，孜孜不倦勤劳的编撰着一部部流行于世面的"通俗黄书"。可以这样去浪费纸墨，以前我并不知道。

　　大段的性情景描写夹在小说中，敢说、敢想、敢写、敢为天下先，敢说都是艺术？

　　有些作家的水平虽不敢恭维，但对这种情景的描写却灵光迸现，入木三分。

　　艺术是美的，但不是说人人都是艺术家，人体是美的，并不是人人都美。模特也分三六九等，如果画得是干巴巴的僵尸，你看了之后，估计三天不会近厨房一步。

　　把先天不全的东西拿出来示人，在作品中加些段子，倒大可不必。为什么进化了的人类还穿着衣服？是因为他们还有羞耻之心，又何苦在作品中将人赤裸。

　　好像作品中有了性情景就艺术了,没有就不艺术，就不叫文学，就不堪入目。谁这么想，就想错了。有些人就喜欢把东西写成街头的小报，也确实难能可贵。男女之情可写，男女之事可写，但不必写滥写烂了。

　　在艺术的幌子下寄生了一批营造伪艺术的人，人是艺术的流氓，艺术是流氓的艺术。

　　或许书里的性描写是极大的艺术，我浅薄无知，无法看出，但我相信，懂艺术的何其少。因此，我建议，在这类书上最好印上"十八岁以下儿童不宜"或"未婚者慎读"。都会有孩子的，何必为了一时口舌之快，害了他们。

　　你可以去做流氓，但孩子却是无辜的。

　　如为了炒作，为了言人所不言标榜自己，加入此类东西，便是心理的极端不正常、极端的不健全，有必要考虑哪个医院的技术水平。

　　否则，就要考虑进班房后的事情，我听说，耍流氓要进班房的。

　　先做人，再作文。

　　我是这样认为的。
