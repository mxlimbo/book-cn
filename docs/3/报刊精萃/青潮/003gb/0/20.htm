没有日子的思想

作者：张才

　　人们更容易记起在什么日子里做过什么，却不容易记住在什么日子里想起什么，于是才有了没有日子的思想，而正是那如此之多的没有日子的思想，更多更久地簇拥着你从生命的所有日子里走过……

　　水开了，热气在升腾、缭绕，几个世纪过去了，火车在升腾中奔跑，懒夫在缭绕中懒惰，空间仿佛已凝固了时间，形式似乎又凝固了内容。他们谁都没改变谁，谁也没想改变谁，只是在这一切都和时间毫不相干的世界里暗暗地悲伤着。我曾看过的一幅漫画：习惯于掘中国祖坟的人们，把孔老夫子描写成对弟子送来的火腿垂涎三尺。真没想到，这么多个世纪过去了，仍有那么多人还是和先贤一样，闻到肉香就垂涎，人类怎么就改不了犯馋的毛病！馋没有时代，没有时期，没有时间，人类无时无刻地不在馋着，却不知为什么要把最灵敏的鼻子进化到猫嘴的上边。让猫掌管着馋的权威，他躺在人类的怀里睡着，把眼球睡圆了又睡扁了再睡长了，到后来，便像现在这样，老鼠已不再是他唯一的天敌，人类的食物成为了他圆滚身体的支撑；再后来，也许还将有人像研究人类起源一样去研究猫的起源，只不过已弄不清会误差几个年代、几个时代，也弄不清几个世代后还会不会有猫。也许只有考古学家才知道是先有了气味还是先有了生物，是先有了猫科还是先有了人类。

　　不需问那是在春天还是秋天，风和日丽是最关键的，是乎热血总是在这种情况下更容易沸腾。那是一套牛车：车夫在车上忙乎着什么也不关键，两头健壮的公牛（包含驾椽的是一头公牛）和两头健康的母牛是这驾车的动力组成。他们亲近着，爱恋着，车夫在猛然间发生的剧烈抖动中摔倒，两头公牛争相雄壮地分别骑到了母牛身上，他们一个更比一个坚强地挺立着，仿佛忘记了自己平素的笨拙；他们一时更比一时猛烈地颤动着，任凭车夫用鞭子狠狠地抽打，直止完成他们这种本能的交和，甚至完全蔑视了车夫的存在、文明的存在，把这种酣畅无比的野合播种进宽敞无比的原野里，镶嵌到漫无天际的白云上。再文明的语言也掩盖不住这种原始的野蛮，再进步的文明也逃脱不了这类永恒的主题。于是，人们即漫骂那众多灿烂的山花，又赞美那众多灿烂的山花，在漫骂与赞美声中，山花还依旧故我地灿烂着、亮丽着，即使在那些冰天雪地的日子里，他也依然开在人类的心里。不知道是什么原因，谁也没问是从什么时候开始，是山花掩盖了人类思想的伤口，还是人类思想的伤口里绽开了山花。

　　温室里的花在绚烂着，娇艳着，以至模糊了年份季节和盛夏酷暑。把如此众多纯粹的天姿佳丽汇集到一起，却不需要任何像蜜蜂蝴蝶一样的昆虫去做他们爱和性的信使，自有丰收的美丽去渲染一切，浓郁的芳香去证明一切。我不知道也不需要我知道，疾驰如飞的马蹄下是否还有野花在摇曳着，缓慢移动的羊群上是否还有白云在倘佯着，拥有这满目密集的美丽竟又如此让你怀恋美丽。我曾尽情地呼吸过原始森林中那夹杂着陈腐落叶的花香，但那一切都是一任自由地清鲜着；我曾欣赏过一群野猪在那原始的花草上奔跑着，滚动着，任一片片花、一片片草在他们身下失去了鲜艳，但那一切都是极尽自然地芳香着。这种静静的欣赏不需要任何的评价，这种原始的欣赏也不需要任何的代价。不知是怎样一种轰轰烈烈的造物运动，让鲜花盛开在温室，让她的美丽有价，使她的娇艳堆金、郁香积玉。我不知道人们更关心的是花的芳香和美丽，还是花的芳香和美丽更关心自己的身价，至此，秀色成了盘中餐，芳香成了囊中物。世上只有买卖的公平不再有公平的买卖。

　　那是一个梦亦或是一类梦，不关是白天做的不还是晚上做的，不管是今年做的还是明年做的，不论是你做的还是我做的。控制不住的一失足从悬涯上跌落下来，并没伤一丝毫毛；威武无比的一失手你成为了断臂的武松，却只是出了一身冷汗，整夜的风只是刮在床上，整天的急只是着在心里，下雨却没淋湿仍然挂在室外的衣服，寒冷却没冻哭依旧醒睡在身边的幼儿。一片落叶飘摇着、盘旋着、飞舞着、歌唱着，从死亡的坟墓里生还，却还奏着死亡的曲调，唱着死亡的歌谣，拿着死亡的姿态……浩淼的水里升腾出一个绝代的才女，向我招着手、挤着眉、弄着眼……妻在睡梦中酣酣一笑，将手放心的放到了我的心上，洪水会意地消退了，露出了一片绿洲，狂风把落叶卷到了天的尽头。

　　人们总是最关心最初的起因和最终的结局，而不是具体的时间过程。搜寻──人们首先想到的是对象的方位，做客──人们更为关注的是位置的主次……唉！人一辈子能记清的何日何时何事必竟太少了，只有警察和法官才去关心那些本不属于他们的日子和时间，除此之外大多是属于每个人自己的遐想和梦想。
