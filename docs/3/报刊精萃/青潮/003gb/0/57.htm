未命名-2


作者: 明眸点墨

一个时期已经过去了
雪花飘落已成往事
无须步行窗外
灯光寂静得如同你的年龄
所有黑的发，花白的发
手指与爱的纤细轻柔
美丽正在院子里弹奏声音
唤醒些绿色
你依然幻想着安安静静的交谈
苍白如月光映衬的雪
而岁月失色，你的泪水
悄悄地流

那些离别的日子
暗自庆幸而又倍感失落
隔着夜晚望不见对岸的花
不自觉地想起阳光在骨胳里荡漾的感觉
那时你青春的脸庞满是笑意
所有诗句挺拨而笔直，写尽心声
后来一道道闪电刺破乌云
制造撕心裂肺的雨
而你意念坚决，并无丝毫苦痛
再后来，你终于找到自己的影子
你便象了找到拐杖的老人
坐在久远的年代，不再离开
