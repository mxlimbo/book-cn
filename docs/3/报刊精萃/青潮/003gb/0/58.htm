感觉


作者: 明眸点墨

旷野。星光下
你看着我
我看不见你
风，揉我满眼沙子

三维的思念，伤口
你的目光不易寻找
你信不信，今夜
我的手在木吉他
什么都没弹
我什么都没敢想

但还是被你发现
这我知道
连最后一层谎言
也被你无情戳穿
如今我赤裸
被你的目光深深击中

赤裸是寒冷的
忍着笑，眼泪便流出来
我知道
谎言，也可以是假的
你说，而我不再说
我只想躲到自己身后

瞎子的灯
照不亮我的路
今夜
我将与他撞个满怀
感觉
