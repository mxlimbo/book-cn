诗一首：下岗第七天


作者：麋鹿居士

下岗第一天

喘着粗气的母亲出院
医生说慢慢走啊，有钱再来

下岗第二天

工会主席领着摄像机和三个干事
慰问两斤白糖

下岗第三天

弟弟收到大学录取通知书
父亲去逝周年

下岗第四天

整上午排队到再就业办公室
默写自己的出生年月和籍贯

下岗第五天

一叠报纸。一把雨伞
两盒快餐

下岗第六天

彩气球漫天飞舞，一家大酒店
在晚间新闻里开张剪彩

下岗第七天

等了半个小时避开熟人
进了新开张的酒店没再出来
