狗屎

作者: 网命徒

　　狗屎是我们村的老光棍。据说生下来时奇丑，他的奶奶看了后一撅嘴，说咋象一堆狗屎呢。就这样，他就叫狗屎了。乡下牛粪狗屎的常见，所以叫起来也不算难听。狗屎长的丑，命也苦。三岁时父母相继去世，他就和奶奶过日子，最终，奶奶也熬不过他，他就成了孤家寡人了。四十岁时还没说上媳妇，这辈子也就没什么指望了。夏天一件大裤衩，赤着脚，四仰八叉的在树下歇晌。冬天则一件油渍渍的老棉袄，用捆麦子的绳子在腰间一扎，袖着手在北墙根晒太阳。

　　这一年，是不穿裤衩和棉袄的秋天，狗屎被队长指派去看坡。就是看着快成熟的庄稼不被人偷。这是个得罪人的差使，自然就落到狗屎头上。狗屎倒也尽职尽责，每天就在队里那块棒子地周围转悠。

　　一天，狗屎刚刚巡视到地的南头，就听棒子地里沙拉沙拉的有动静。狗屎大喜，原因是这么多天了还没逮着个把毛贼，似乎他天天在做无用功，没有成就感。狗屎猫着腰悄悄走过去，这一看不要紧，狗屎的眼再也挪不动了――一个女人正在那里方便，白白的屁股正好对着狗屎。

　　肯定是狗屎越来越粗重的喘息声惊动了那个女人，她回头看看，倒也没惊，笑骂道："臭狗屎，咋偷看人家拉屎？"说了还蹲着不动。此时的狗屎成了聋子，呆子，眼睛一直没离开那个白白的物件，嘴半张着……

　　突然，狗屎来了个狗吃屎的动作，扑了上去。

　　那女人也没怎么反抗，就让狗屎得逞了。虽然是第一次，狗屎却很熟。在歇息的空档，狗屎才看清，那女人竟是队长的老婆。他稍稍一楞，继而以更猛烈是速度进行了，仿佛不是泻欲而是在泻恨。

　　是的，他是在泻恨，在报复，报复队长。

　　有时大伙儿都在一起闲聊，狗屎要是走过去，队长就使劲嗅嗅鼻子，一本正经的说哪里有狗屎啊，我怎么闻到臭味了。大伙就笑，队长也跟着笑。狗屎也笑，但他心里恨得牙痒，也仅仅是骂骂队长的祖奶奶。这种时候很多，狗屎早就怀恨在心，今天机会终于来了，他能轻易罢手？

　　等狗屎终于"罢手"后，队长的老婆还迷着眼不肯起来，就那么看着狗屎，眼光柔柔的。狗屎不笨，他读出了那是满足和感激。狗屎这个气啊，本来是想报复人家的，却不想让她得了便宜，呸！狗屎照着女人就是一脚，还是那个屁股，刚才还那么喜人，这会怎么看怎么不顺眼。

　　骚货！狗屎狠狠的骂了一句，系上裤带走了。

　　后来这件事叫队长知道了，这都怨队长的老婆，在床上说漏了嘴。队长领着他的三个弟弟，在一个早晨把狗屎的一条腿打断了。两个月以后，狗屎再出来，就一瘸一拐的了。

　　天渐渐冷了，因为狗屎又穿上了老棉袄。

　　快到阳历年的时候，公社的敬老院建好了。狗屎虽还不老，但还是让队长争取了一个名额。一天早晨队长把狗屎的门拍的山响，"狗屎！狗屎！……"半天了里面还没动静，队长就一脚踹开了房门，狗屎一动不动，死了。

　　队长眼圈红了，骂道"这个臭狗屎，咋这么没福气！"，并把狗屎唯一的一把没有盖的暖瓶踢到了院子里。

　　接下来，队长自己掏钱给狗屎做了一身新棉衣，由队长的老婆还有另外几个女人给狗屎穿上，就拉去火化了。去火化的几个后生回来时嫌骨灰盒累人，就扔在了一个深沟里。再后来，队长扇了那几个年轻人每人一耳光，自己跑去抱回来埋在了村子西边的山坡上，那里有狗屎的父母还有他的奶奶。
