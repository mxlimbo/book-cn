我爱你之调侃版


作者：阿木

　　出场人物：

　　阿木：一个无业游民。

　　Ａ、Ｂ、Ｃ、Ｄ：化妆品店的营业员。

 

Ａ

　　天上落着小雪，还未到地上，就化成了雨。一片湿漉漉的。

　　阿木骑着单车，停在店门口，单脚支地。

　　阿木按了按车铃。

　　阿木：喂！

　　Ａ出来了。

　　阿木：我爱你！

　　Ａ：今天怎么了？怎么说这个了？看你，全身都湿了，冷不冷？

　　阿木：我是真的。

　　Ａ：看你都冻成啥了，进来暖和一下吧，看你，嘴唇都成紫的了！

　　阿木：我说是真的。

　　Ａ：我没说你是假的啊。进来坐会吧！

　　阿木：不用了，我得走了。

　　Ａ：那，你等等，我给你找个伞！

　　阿木：不用了，我走了。

　　Ａ：那我一会给你打电话！晚上见！

　　阿木：好的，我来接你！

 

Ｂ

　　天上落着小雪，还未到地上，就化成了雨。一片湿漉漉的。

　　阿木骑着单车，停在店门口，单脚支地。

　　阿木按了按车铃。

　　Ｂ出来了。

　　阿木：我爱你！

　　Ｂ：……

　　阿木：我是真的。

　　Ｂ：……

　　阿木：你怎么了？我，不，你……

　　Ｂ：我终于听到了，这么多年，我一直都在等，呜……

　　阿木：……别哭，我给你擦擦！

　　Ｂ：我是太高兴了，哈，泪都流下来了。我，也，爱，你　　！

　　阿木：我知道！

　　Ｂ：……等一会，我给你打电话，我们晚上去吃饭！

　　阿木：好的，我走了！

　　Ｂ：你真傻！一身都湿了，就是为了说这个？我给你找伞　　！

　　阿木：不要了，我得走了！

　　Ｂ：那好，等我电话！

　　阿木：好的。

 

Ｃ

　　天上落着小雪，还未到地上，就化成了雨。一片湿漉漉的。

　　阿木骑着单车，停在店门口，单脚支地。

　　阿木按了按车铃。

　　阿木：喂！

　　Ｃ出来了。

　　阿木：我爱你！

　　Ｃ：现在才说啊，我早就知道了！

　　阿木：不会吧，什么时候知道了？

　　Ｃ：在你看我的色迷迷的眼神里，我就知道了。

　　阿木：哈，我说是真的。

　　Ｃ：哼，太不浪漫了吧？这个时候说！

　　阿木：那要什么场合？

　　Ｃ：最起码啊，要坐在一个古色古香的桌子旁，地上是松软的地毯，一边音响还放着轻音乐，哇，那个时候说，还要同时递给我一个钻金戒指，说，嫁给我吧，哈哈！

　　阿木：你把我杀了吧，我只有买一块臭豆腐的钱了。

　　Ｃ：那算了，只有等你什么时候有钱了再说吧！

　　阿木：哈哈，我走了。

　　Ｃ：慢点啊，看你一身都湿了，早点回啊，别感冒了，一会来接我去吃饭啊！

　　阿木：好，走了！

 

Ｄ

　　天上落着小雪，还未到地上，就化成了雨。一片湿漉漉的。

　　阿木骑着单车，停在店门口，单脚支地。

　　阿木按了按车铃。

　　阿木：喂！

　　Ｄ出来了。

　　Ｄ：ＫＡＯ！你有病啊，这个时候来，不嫌冷！

　　阿木：我爱你！

　　Ｄ：去死，哪凉快去哪！

　　阿木：我说是真的。

　　Ｄ：拜托，老大，别说这么深沉的话，我会吓晕的！

　　阿木：别别，晕倒了我还得给你作人工呼吸，多麻烦！

　　Ｄ：去死！就会占人家便宜，说，晚上请我吃什么？

　　阿木：老大，放过我吧，我只有买一块臭豆腐的钱了。

　　Ｄ：啊？这么穷，把你卖了算了！

　　阿木：有人要吗？干活没力气，带孩子不会，哈哈！

　　Ｄ：我要啊，让你当司机，权当俺家卖了个毛驴，俺骑着毛驴来上班！嘿嘿，那才过瘾！

　　阿木：不会吧？

　　Ｄ：怎么不会？哈哈，回去自己弄一个草标吧！

　　阿木：算算，不和你说了，我得走了！

　　Ｄ：好好，一会我给你打电话，晚来一分钟，我一刀砍死你！

　　阿木：只怕你舍不得？

　　Ｄ：到时你就知道我舍不舍得了，哼哼！

　　阿木：好，你进去吧，我走了。

　　Ｄ：慢点啊！哈，真是有病，这么冷的天……
