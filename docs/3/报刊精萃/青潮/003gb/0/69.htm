情事 之一——要他还是要党？


作者: 小小胡杨

　　敏是我上中专时的辅导员，湖南常德人，同一专业，比我们高两届。她对我很好，但我们很久没联系了。不知道现在那个闪着大眼睛、有着灿烂笑容的敏姐过得怎样？

　　刚一入校的第一个晚自习，我至今还记得很清楚。来自十一个省份的同学能走到一起，谁能说不是一种缘份呢？在挨着教室门口的墙角处，站着一个清纯秀气的女孩，带一副眼镜，穿一件白衬衣，外套浅绿色的毛背心，中等个头，脸上保持着温和的笑容。后来知道她是我们的辅导员，负责解答我班的"入校新问题"，解决我们的生活中的琐事——一句话，以老带新，照顾我们吧。

　　敏说话和气，脾气很好，百问不烦，下了课以后经常去女生宿舍和我们沟通感情，我们一下子喜欢上了这个温柔大方的湘妹子（其实该是湘姐姐吧）。很快，有消息灵通者从老乡那里打探到原来敏还是俺们学校的学生会副主席哩！哇，人不可貌相耶，要知道，在工科学校里，女生能做到这个地步已经很了不起了。让大家感到"特美"的是，若有人问"哎，哪个班的？"答"工16呀""工16？""是呀，程小敏是我们辅导员！"，两问两答后，准是一句"噢（可是拉长音的！），知道啦！"。

　　敏对我们很尽心，怎奈俺们后来的表现实在是……一年级在广播操比赛中只拿到鼓励奖，综合排名也是倒着数靠前；二年级就弹劾掉了心软的班主任（把班级在年级落后的原因归罪于他了。人善被人欺呀，好心的班主任让我们一帮小崽子给治于死地了），短短两年，在学校是"臭名远扬"。唉，总之有点对不起敏的一片苦心。不过，说实话，我们还是对敏很有感情的啊。

　　更让俺们这一届别的班级感叹不已的最后一个"绝笔"是，俺班在敏毕业时，不仅欢送了她，也一起欢送了她的"心上人"。

　　这么说的确好象有点矛盾。敏是学生会副主席，但却喜欢上了同班的一个男生。本来她是不应该喜欢男生的。学生会副主席怎能带头犯错误！她是要配合学工办的老师去抓"逃犯"的呀。然而感情，却并不因为谁身上挂着个冷冰冰的不过也是热乎乎的头衔就同样被冰冻被蒸发。那个男孩是学校 体育特招生，篮球打得很棒，是校队的主力，潇洒是不用说的了，但是学习嘛，却是"上过白榜"。

　　以下是我对那天欢送会的一点记忆（五、六年了，难免有漏，发挥一点想象好啦）。
"讲讲你们的恋爱经历吧！"一堆十六七岁的孩子们闹哄着。敏在讲台下的一边站着，红透了脸。投篮手脸上是稍有尴尬的表情，顿了顿说："嗯，那时候吧，嗯，大家知道，我这学习嗯，是不怎么好。你们辅导员呢，嗯，那会是我们班的前三名，出于对后进同学的帮助，嗯，她吧经常帮我辅导，嗯，这个吧，后来吧，嗯，我们就，嗯……"班上爆发出一阵快乐的笑声。他又故作严肃状："不过呢，你们不要学我们啊，这样不好。"又是一阵笑声。

　　给我们表演个节目吧，我们撺掇着。

　　投篮手在众人的极力要求下，答应给我们唱一首歌。没有致词，又是顿了顿后，一个富有磁性的男音让闹哄哄的教室一下子安静下来。"你在他乡还吗？是否想过靠着我的双肩；你那不再熟悉的笑容，对我可以一种敷衍。手中握着你的照片，却感觉你很遥远！……你在他乡还好吗？……"投篮手的声音不很高亢，却将一种忧郁与别愁表达的淋漓尽致。一曲光头李进的《你在他乡还好吗》让我们欢呼鼓掌。

　　是啊，你在他乡还好吗？湖南常德的敏，陕西西安的篮球明星？

　　我们送给敏一个挺大的木制风铃留作纪念。那天的欢送会后，就很少见敏了，临近毕业，他们的事情很多。

　　但我们还是陆陆续续听到关于她的一些消息。有一些甚至是她离校后我们才知道的。

　　敏没能入党——在离校前最后一批发展名单上，没有她的名字。虽然她是学生会干部，虽然她学习很好，虽然她团结同学，表现不错——但是她恋爱了。"正因如此，她的影响就更坏"——学工办一位老师的原话。据说，学工党支部的老师曾经专门问过她一个问题，对这个问题的回答可以决定她能否入党。

　　"你能保证以后不嫁给他吗？"老书记的眼光似乎能刺透人心。

　　敏哭了。抽泣着没有说话。

　　没有结果。学校也许还是看在她其它各方面表现良好、为学校的学生工作做了很多的份儿，什么也没做——无奖无罚。

　　其实敏当时可以保证，先取得预备党员的资格，一年后的转正就是工厂的事儿了；而且，以后她能不能嫁给投球手，还是另一码事。但是，她只是以沉默应答。

　　据说，是她班一个广东籍的学生会干部作了"犹大"——他也要入党——但是名额有限。

　　投篮手的哥哥是学校所在县的公安局长。投篮手扬言不让"犹大"走出湖庄（学校所在的小镇）。但"犹大"是校运会上的短跑冠军，他比投篮手跑得快，第一次，给他逃脱了；毕业离校时，有关老师亲自驾车送"犹大"到西安车站，送他上了西安至广州的火车。

　　投篮手的恨未能发泄。

　　毕业后，敏的父母对她严加看管，不准她再到陕西去。

　　听说敏在看一台高精度磨床……

　　我没有再见过她。
