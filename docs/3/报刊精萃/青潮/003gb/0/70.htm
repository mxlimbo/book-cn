雨一直下


作者: 雨衣

　　雨一直下。

　　雄为苹举箸夹菜，苹说："谢谢。"雄的内心突然生出一丝悲意。

　　吃完饭。雄为苹拉开凳子，苹站起来，说："谢谢。"

　　雄从衣帽架上取下雨衣，给苹穿上，苹说："谢谢。"

　　雄推开饭店的门让苹先走出来，苹说："谢谢。"

　　雄提着苹的包，递给她的时候，苹说："谢谢。"

　　雄挡了一辆TAXI，打开车门，让苹上去，付了司机两倍于目的地的价钱，转过头对苹说："保重。"苹说："谢谢，别忘了我们吃饭时的话。"雄目光沉着、深邃，心情说不出的苍凉、悲壮。

　　雨一直下。

　　雄一个人孤独地走在街上，任凭风吹雨打。雄觉得这样心里好受一点。雄心里难受是因为苹。是因为今天苹说了那么多的"谢谢"。雄每听一次苹的谢谢，浑身就禁不住起鸡皮疙瘩。苹的每一句谢谢都像是向雄的依恋和热情泼上一盆冰冷的水。雄的心里冰窟一样。其实，雄早已预感到他与苹的结局，只是他一直相信真爱无敌，所以一再想要力挽狂澜。可是，今天苹的谢谢彻底打碎了雄的梦。雄的梦很美满，很绚丽，很壮观，那是他和苹曾经共同绘的蓝图，绣的美景，造的工程。然而，今天，这个梦破灭了。雄的心里突然之间涌出了一句苏东坡的名句："曾日月之几何，而江山之不可复识矣！"就禁不住泪如雨下。

　　雨一直下。

　　雄想起了他和苹的感情经历。

　　刚上大学时，雄壮志凌云，准备刻苦学习，将来考研究生。因为像雄这样一个来自农村的孩子如果不自己努力，不但将来没出路，而且对不起含辛茹苦的父母。像雄这样一踏入大学的门就知道明确自己的目标，坚韧不拔地走下去的人寥若晨星，而苹就是这样一颗星星。于是，在自习教室，在图书馆，在清晨的校园，在静谧的书店，经常出现雄忙碌的身影，苹朗朗的读书声。两颗星星的偶然相遇撞击出了雄和苹的爱情火花。从此，在人生的旅程里，在学习的道路上，多了两个相依相偎相濡以沫的伙伴。雄清清楚楚记得那一年的暑假，他和苹两个人忍着高温天气，骑自行车二十里，去代家教的情景。想起那时侯两个人的那份执着、纯真、朴实，雄禁不住哽咽了。记得当时，雄想给苹买一根雪糕解渴，苹说啥也不要。苹说要买就给你买一根吧，我不渴。雄再三再四要去，说这么热的天气，我一定要去。于是，雄买回来一根雪糕，雄让苹吃，苹说雄你吃吧。雄说，苹你吃，我专给你买的。苹说，你吃。雄说，你吃。后来，苹说，再不吃，就快化了。两个人才你一口我一口吃完那根凉爽、解渴的雪糕。然后，雄和苹心照不宣的哈哈大笑。一根雪糕才一块钱呀！可是当时的那份执着、纯真、朴实，让人想起来感觉生活真是美好。因为当时雄和苹两个人谁都知道挣点学费不容易。 两个人都知道家里把自己供到大学不容易，都知道为家里减轻负担。就这样，雄和苹走过了人生的黄金时代——大学时代。毕业后，雄被分到本市一所重点高中教学，苹经过努力进了报社。本来，雄和苹都在同一城市工作，应该可以是有情人终成眷属。谁知，半路上杀出了个程咬金——刚。刚是苹所在报社的一个老大哥，早毕业几年，已成为本地区知名记者，家在本市，有一套宽敞的住房。作为一个乡下来的女孩，留在城市多么不容易。于是就有人撮合苹和刚。当然，他们不知道苹有个男友——雄。可是，苹竟然答应了跟刚"接触"。于是，有了本文开头的那一幕。无论如何，雄都不愿相信苹是真心要和他断。可是不相信又能怎么样。苹真真切切地对雄说，雄，你也另找对象吧，最好是本市的，家庭条件好的。苹说，我们现在必须现实点。其实，最让雄寒心的是，苹竟然说了那么多的谢谢。

　　雨一直下。

　　雄想起他第一次请苹吃饭，苹说："谢谢。"那次，雄第一次拿到了自己的奖学金，首先想到了一直给他帮助的苹。雄为苹举箸夹菜，苹说："谢谢。"吃完饭，雄为苹拉开凳子，苹站起来说："谢谢。"雄打开小饭馆门，让苹先出，苹说："谢谢。"雄一直把苹送到女生公寓楼前，苹说："谢谢。"

　　雄怎么也记不起他和苹之间是什么时候不再使用"谢谢"的。今天，又听到了苹这么多的谢谢，雄的内心说不出的酸楚、难过。雄就纳闷，怎么同样是"谢谢"这两个字，感觉就不一样了呢？

　　雨一直下。
