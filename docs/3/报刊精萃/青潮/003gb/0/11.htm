也谈韩寒


作者: 南一

　　昨天看中央二台的《对话》栏目，有幸目睹韩寒，乍一看，还难接受，头发长到脖子，更确切地说是到衣领，发稍微卷起来，可后来的对话中发现，果真是个特立独行的人，谈锋甚健，会说话吧，善于诡辩，但又不着痕迹，有点举重若轻的味道，为我辈所不及。

　　但又狂了些，因为《三重门》无论如何我不会看第二遍，可笑韩寒很严肃地认为有些东西是永恒的。从中我也悟出了一些有趣的东西，当你回答不了或不想直面的时候，尽管可以说没有不之类的，保证有效，因为这时尴尬的是问话者。

　　众多人都强调韩寒只有一个，听多了不说他自己如何在心底骄傲，连我都别扭，怎么成了国宝级的了。其实这样的社会这样的文化氛围，再出一个甚至几十个韩寒都不足奇。

　　在座的一个男嘉宾可能对心理学颇有研究，他说韩寒的对话不按正常的逻辑，总之看来，他是在努力保持一种品牌，韩寒品牌。韩寒立即反驳说可以问问他的同学他平时就是这样的，言下之意不是刻意为之。整场对话韩寒给人的印象，从容而倔强，也有一点不以为然，这是唯一的一次，我看出了他内心的慌乱。

　　主持人过份的热情在开场时即遭到冷遇和反驳，我们都看得出她那种尴尬的笑容，而韩寒的前三板斧果然起到了作用，一时台下的观众不敢轻易地接过话筒，因为在机智而又不客气的韩寒面前，人们都有复杂的情绪，心里蔑视，嘴上害怕。

　　一位做了母亲的女观众首当其冲，她的问题其实本身就不是问题：你的人生积淀不够。（大意）背过《左传》的韩寒当然对有名的"濠水之辩"不陌生，他说，你怎么知道我的积淀不够呢。

　　这位女士也可算口齿伶俐的了，又转到OICQ和聊天室上，甚至不惜牺牲了自己的儿子，表面上看她胜了，因为她说OICQ是成人的地方，而韩寒去的是聊天室。这使对话变得象一场无味的辩论，胜利不值得鼓掌。

　　在以后的对话中，韩寒总是巧妙地用这句话来结尾，我们OICQ上见吧。这是鲁迅式的笔法。

　　不过遗憾的是我们的人太包容，正如主持人所说，还有的是害怕，没有锐利的辞锋，全场并没有出现针锋相对的局面，而这正是我希望看到的，没有争论就把问题折中，混淆一团。

　　这其中那位学者式的男嘉宾分析得很有见地，主持人问他想对韩寒说些什么，他说不想和他对话，因为他下意识地保持一种韩寒品牌，就象演员当久了不知戏和生活的区别。我认为，再怎样狂的韩寒，也不得不承认了。

　　看上去，韩寒还是很酷，书上的照片酷中带有一种学生式的可爱，可坐在场上的韩寒已经实实在在是不再属于可以简单介定范围的一个个体了，他正努力让人无法忽视他的存在，事实上他也做到了。
