杜尚时代


作者:许多

　从内心深处我是一个极懒的人。

——杜尚

　　让我设想一下这样的场面：一个人躲在阴暗的房间里已经很久了。他思考着。这样一个清晨，他终于醒来。然后，他迅速来到了他的画室。在《蒙娜丽莎》廉价的印刷品上随意加上了胡子和山羊须。最后他不无得意的取名为《L、H、O、O、Q》〈法文意思为她的屁股热烘烘》

　　这当然是我的臆想。事实上这已经是1919年的秋天了。他就是曾经使无数现代艺术家都骚动不安的马塞尔、杜尚。

　　杜尚1887年出生于卢昂附近的布兰维尔。父亲是公证人——典型的中产阶级。生有六个孩子。其中四个都成了著名的艺术家：雅克、维龙是画家。雷蒙德、杜尚、维龙是雕塑家。苏珊、杜尚和马塞尔、杜尚也是画家。

　　在杜尚传奇的一生中，几乎所有见过他的人都将被他吸引。他一生中没有艺术上的对立者，靠下棋来消磨时光〈不是普通的娱乐，而是达到法国国家专业队的水准〉出门旅游只带一个牙刷。可以说，杜尚的出现对于西方艺术史是石破天惊的。因为在他之前还从没有一个人放弃了绘画却又影响了那么多的画家。在他看来，生活本身就是一件艺术品。这在诗人佩兹的眼中显然成了"本世纪最优异的智慧的代表"了。

　　1908年至1910年，杜尚曾在朱利安美术学院学习。最初的画风深受塞尚的影响。我们可以看到他早期作品中残留的若干塞尚的影子：用色彩造型，砖块状的笔触。这样的目的正如他自己所说的是在于了解他们，是如何画这些画的。但在1911年前后，他就开始宣称要对传统绘画进行改革了。这在他的作品〈小妹妹〉中就已经闪出了火花。次年，便产生了决定性的作品〈正在下楼梯的裸体〉。这时他将自己思维的能力置身于任何有纲领的艺术运动之外，开始热衷于机器和活动中的形体。1919年，这幅画参加了纽约的军械库画展。这是怎样的一幅作品啊：连在一起的人体草图，他们或许正带着某种准确性，有节奏地，从旋转的楼梯上下来，走想生活~~~~~。正是这幅作品，使他在美国一举成名。美国人因此还送了他一个印象大师之类的称号。这时候，他年仅25岁。他的才华已经开始展现。他开始清楚的知道他要寻找一种全新的表现世界的手法以及无人所画的梦想正在这幅作品中重生。

　　1916年，杜尚署名穆特的另一件重要作品〈泉〉参加了纽约独立派画展。虽然这次画展本身就具有一定的前卫性。但〈泉〉的到来还是激怒了参加画展的权威们。这其实只是一个现成的男用瓷制尿斗颠倒过来钉在木版上，并用漆在斗壁上写着"R、MUTT、1917"的字样。然而，正是这件作品短暂的展出确立了"现成品艺术"的概念。由此，它的展出也让观众们明白这样的一个道理：艺术是主体在幻象上所得到的移情效果已经失去了作用。这使得人们不得不面对人工制品直接的出现在眼前。严格来说，作品〈泉〉其实是带有一种强烈的达达主义色彩的作品。但有讽刺的是，杜尚用这样的作品的初衷却是为了讥讽达达主义。一种雅皮的玩世不恭的幽默。他在〈泉〉当众销毁时辩解到：尿斗的外缘曲线以及倒立的构图与〈蒙娜丽莎〉是一致的，而其机理效果却与古希腊雕塑家普拉克西特列斯的作品有相似之处~~~~无论他怎么说，这件原作还是消失了。1927年，杜尚重新原样复制了一件，由纽约现代艺术博物馆收藏。他的这种行径，或许正验证了莱维德的话："在观念艺术中，观念是作品中最重要的。所有的打算和决定都是预先作出的。而创作只是随便敷衍罢了。"

　　同样的，在他的一生当中的绝大作品中，所有对于传统的追求，探索以及艺术创作所应有的艰辛都在他的几个"游戏"中颠覆了。他甚至传达了这样一个信息：从事艺术的不必绘画，雕塑也不必创作作品。因为，生活既是艺术。艺术是现成的。可以说，杜尚是不可模仿的，我们只能间接的领悟他的精神。

　　杜尚的时代是达达主义风头正劲的时代。他本意也并不曾想到要开创一个时代的到来。他只是用一种极端的破坏方式表达他对于传统的藐视。但不管怎样，西方美术史因为有了杜尚的出现而出现了生机。波普主义，装置艺术，观念，色域艺术，极少主义等等流派都或多或少受到他的影响倒是不争的事实。。杜尚的出现，确实改变了20世纪艺术的进程。20世纪60年代以后，杜尚已经被尊为"无作品大师"。而他作品当中的不可企及性和一次完成性却使得杜尚时代的其他艺术家们纷纷望而却步。这或许就是杜尚的魅力所在吧。
