另一种飞翔


作者：乔燕

　　当我看到进进在"云遮月"论坛里留给我的信件一般的温暖的文章时，我被深深地感动了！他给了我太多有力的鼓励，我由此再次感到我会也更可以更坚定地在这条路上走下去，无论最终能走进哪一块麦田。

　　同时我不能不想起那个送我调制解调器的朋友，若不是这样，我这个因行动不方便而在家养病的女孩，因为有着不能完全独立而导致的难以向家人提出更多愿望的个性的人，恐怕到现在还被拒绝在网络之外。

　　记得那时我们还没有见过，他是在电话中和我交谈之后，对我们共有的朋友说：乔燕过着好"原始"的生活！

　　我大声地笑了：在这个信息为粮的社会，乔燕过着好"原始"的生活！

　　我每天在一个安宁的世界里迎来黎明又送走黄昏，朋友们来看我时总爱说：在你这儿总能感到少有的世外桃源般的宁静与祥和，时间会无声无息地滑过，直到不得不走的时候，才要惊呼它在你这里竟快得好象我刚刚来到。

　　这几年来，我过着不与社会直接接触的生活，我只能通过心灵去感受自己、感受他人。这给我带来了很多局限——由寂静的个体生活方式决定的社会直接经验的缺乏；而同时，这样一种生存状态却为我拓展了另一片天地——还给了我一个人在成长的过程中日渐萎缩的心灵的翅膀。

　　当我不能用眼睛在人群中直接观察所在的这个社会时，我开始踏上用心中的眼睛向内审视人的"自我"的林间小径，它带我来到人的内在深处，我开始寻找我的归途。

　　我独自行走，能听到周围同伴们砰然有力的心跳，能感受到他们忙碌穿梭的身影，我能看到他们在时代中前行的脚步……

　　从一种角度来说，我是一个落伍者，当大多数的人们在人生的道路上按步就班地跨过一道又一道生活之桥的时候，我隔着一条汹涌的河把他们静静地观望。

　　而从另一种角度来说，我又是一个不断超越的行者，当深渊突然地横在身心尚显稚嫩的我的面前时，我几乎深深地坠落。我在深渊的边缘挣扎，我柔弱的身子失重般地下落，我本能地渴望抓住什么，当这种骨子里的不甘与渴望把我下坠的心灵一次次托起的时候，我终于停止了哭泣。

　　我开始回过头来追踪，沿着走过的每一个脚印。

　　我回到童年广袤的田野，在一朵嫩黄色的野菊花上发现了一只小黑虫窥视我的惊喜的眼睛……

　　我回到了飞舞着成群的蜻蜒的草丛上，在它们透明的薄翼上藏起了最初的想象……

　　我回到了那个名叫海灵的瘦女孩的身边，再次为她擦去凶暴的父亲打她时，她凝固在她脸上的晶莹……

　　我也回到了每一个晴朗的夏夜，父亲搬到门外的小竹床上，和两个妹妹并排挤在一起，一边凝望星空，一边静静地遥想他那讲不完的故事里可爱的拇指姑娘……

　　我再次走在宝丰县的土地上，让八岁到十二岁的童年的风带着我把那时蓝蓝的天重新仰望……

　　我在漂浮不定又不断变换的云朵里找到了那些飘远了的旧衣裳，他们是否还记得我——曾经轻轻包裹过的那个羞涩的小姑娘……

　　我还回到了我9岁时坐进过的教室，我看到在那个炎热的中午，她在连绵不绝的蝉鸣中因为锁不动那把笨重锈紧的铁锁而焦急害怕，却又在心中阻止自己就此走开的目光……

　　而此时，她的父亲已来到她身边坐在她面前，把用小刀刮下的铅芯的屑粉填入锁眼，一边看着她好奇的模样……

　　接着，她高兴地锁上教室的门，那把成为她个性成长中重要一环的锁，此时便再也不能把她一个小班长的责任心折磨……

　　拉着爸爸的手走在家家户户飘出饭香的路上，她悄悄闻着，心里又一次印上了偏偏不怕困难的倔犟……

　　……

　　我这颗成年的心在时空隧道里穿梭，这个迎面走来的小姑娘把她固有的纯洁再次馈赠给我。

　　我在她的身上找到了失去的自已，我不再为人生转折时背我而去的初恋悲伤。

　　我的脸上再次露出微笑，我开始能够尝试着以另一种心境观赏每一天都会升起的朝阳。

　　我也把微笑送给傍晚，把渴望留给隔着小屋在天边悄悄望着我的月亮。

　　我在一个下午被楼下一个小男孩大声呼唤妈妈的喊声吸引，我意识到这种只属于孩童的直起嗓子的呼喊早已离我远去，我不能自制地迅速写下一篇《呼唤》，这个根本不知有我这样一个姐姐存在的男孩永远不会知道，我真纯的童心曾被他注定要忘却的童声的呼喊拯救。

　　我要摘下多少颗闪亮的星星，悄悄放进每一个和他一样纯真的孩子们的手心。我能够凭此晶莹多少人的梦？

　　当那个我帮助过的和我生活在同一个城市的8岁女童在今天清晨和同学一起向我家走来时，她对爸爸说的"我想燕子姐姐了，我要去看她"的银铃一般的话在早晨的空气里使我们俩都获得了新鲜的生命。

　　我只有一双明亮的眼睛，当我在几年前看到报纸上号召人们志愿身后捐献角膜的倡议时，我才强烈地意识到爱心是多么需要指引。我由于激动而颤抖着手在《志愿身后捐献角膜书》上签下方方正正的中国字"乔燕"，可是仅此还远远不够。

　　我只有在又写下一首诗《心愿——为角膜盲患者朋友而作》之后，才让这颗心真正安宁：

一颗种子不该永远都是种子
当春天到了
你的眼睛再次长出蓝蓝的天空
那便是百年后
希望的旗帜在我的坟冢上
迎风飘扬

　　那个匆匆给我打来电话又匆匆挂掉的十二岁的小男孩，他那与年龄极不相称的沉重而又恐惧的声音——是否还在他人生的路上滞留，那本该给他温暖与抚育的母亲是否已从扭曲的欲望之火中觉醒？我在《烈焰中的小白杨》里那些愤怒的呐喊与含泪的呼唤，被我的知已在电话的另一端听见，他在我无法念下去的哭泣中又为何久久地无言？

　　其实我知道，这正是我能找到的一个出口，让我复苏的灵魂之土能够再次长出茂密的丛林，我在心与心的通道里捡起我能捉住的幸福，我在此时获得了早春那天边滚滚的雷声。

　　我在燕子归来的那一重天空里找回了我丢失的鸽群，我在我的文字中大声地朗诵：

 

这个时刻很安静
这样安静的时候　　我想起
久已不见的雪花　　那滚动在大地上的
鲜艳的笑声，　想起童年
夜幕里最亮的灯　　和
灯光里，　妈妈揉面的身影
她唤我回家的悠长悠长的喊声 　

对 ！就是在这个时刻
每个人都不再说话
不堪重负的嘴唇已然走出习惯了的生活
闭上双眼 深深隐藏的世界
凸现在心灵的角落
茫然的风四处游荡
它们带来了眼泪
带来了雨水
带来了震耳欲聋的雷声 　　

而这安静的时刻
什么开始荒芜
什么开始生长
什么开始升腾
又是什么正迅速地坠落 　　

谁正被悄悄打开
谁又回到久别的丛林
这个时刻，再坚硬的灵魂
也会被突然击中
我们注定要踏上远在天边的荒原
在风雨剥蚀的石洞前 仰望
漫天的星斗，在夜空下
对着一丛跳动的篝火出神

 

　　那斑驳的石洞前回响着我曾经迷茫的足音，而这团篝火正从早春燃到深秋，深蓝色的火苗把冬季的绝望无情地烧毁，令雨季泛潮的地面在温暖中飞升。

　　是谁在希望？希望每一个活着的人的心头重新长出树苗，让我们精神的家园因为有了绿色而再奏虫鸣。

　　我又在一个长长的夜里凭借语言唤出了一个崭新的黎明，我用热烈的心跳把它迎接，用洁净的眼眸等待下一个慢慢降临的夜色。

这是一个安静的时刻
这样安静的时刻
歌唱就是倾听
沉默就是诉说
谁是点燃岁月的隐士
谁又正汇成一条解冻的河 　　

也只有在这个时刻
我开始了另一种飞翔
当我的翅膀在空中无声地划过
我会真正感谢该感谢的人
原谅该原谅的生活

　　而这另一种飞翔中，是什么？将永恒地跟随我？！
