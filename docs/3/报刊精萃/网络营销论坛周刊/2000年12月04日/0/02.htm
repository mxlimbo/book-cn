树立全新营销理念--以消费者为核心


zlnike/文

　　面对越来越挑剔的消费者，选择项目总是难以符合需求，所售产品总是难以令人满意，于是一位学者发现，传统的营销观念必须赶快更新传统的营销理论讲的是产品，价格，分销和促销，实际上这是一种以厂商为中心的封闭式的营销观念，面对日益激烈的市场竞争和消费者的需求日趋多元化，这种以厂商自我为中心的营销观念越来越难以适应，于是美国的劳朋特提出了全新的营销的理念，这个理念与传统的营销理念有着质的区别，即经营的一切运作都是以消费者为核心。

　　不要考虑你的产品如何好，只考虑消费者的需求到底是什么。产品要想对路，必须对消费者的需求进行精细研究，或者说必须研究的十分具体，细微，要看清弄懂消费者到底是要什么样的产品，千万不能根据自我感觉，粗略的调查一番就开始经营。卖方市场时，经营者可以拿着商品叫喊“消费者请注意，这是我们生产的”，买方市场时，经营者只能是“注意消费者，看人家到底需要什么”

　　不要考虑你的产品价格如何适当，只考虑消费者究竟愿意出多少钱。传统的定价理论是，先计算研制和生产的成本，然后加上适当的利润来制定价格，然而，由于这种定价程序是由厂家自我为中心来进行的，因而在消费者中能否行的通自然是打问号，因为消费者他不管你的成本如何，所考虑的只是他所认为的此产品应值多少钱，他乐意承受多少钱，所以在定价上，首先要考虑你的目标消费群对这一产品愿意支付多少钱，然后，按照这个估算制定成本核算，进料，生产，销售都不允许超过这个估算范围。

　　不要考虑分销渠道如何畅通，只考虑让消费者怎样更方便。对于制造商来讲，不是仅凭你设立多少分销点，建立多大的网络就能提高市场占有率，而是应该首先考虑如何让消费者购买时更方便，快捷，安全，这就要求我们从以往的“要销量更大的”目标转向“让消费者更方便购买”上来，其实二者也并不矛盾，只是重点装移了，重点由“销量”转到“方便”而已，从而使分销的重心，由原来的更好卖转向消费者更好买，消费者购买方便了，产品好买了，销量也就更大了，这个方便是分销的实质之所在，方便顾客已成为世界的潮流，如自选超级市场，电视直销，电话购物，电话订餐和网上购物等，只是一些经营者还没有明确意识到这一点，因而没有能有意识的从消费者的方便入手，展开分销罢了。

　　不要考虑如何变换促销手段，只考虑如何同消费者最快，最及时的沟通。沟通的方式有多种多样，电话，信函，问卷，回访，座谈会，互联网，电视和广播咨询等，都可以形成沟通渠道，都可以利用，比如售后的电话服务，不间断的电话询问，实际上就是一种及时的沟通，他除了了解产品的使用状况，还可以收集消费者在使用过程中遇到的问题，及时反馈到设计生产部门，从而改进产品的使用品质，而且，这种沟通还可以增加企业和消费者之间的亲切感，消除隔阂，变成利益互补的共同体，从而保持稳定的消费群，这种沟通的作用，是直接的促销所无法比拟的。
