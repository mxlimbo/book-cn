大客户做品牌，小客户做接触


路思/文

　　作为一年150万元销量的小企业，因为经营状况比较正常，老板王磊决定打广告扩大知名度、扩大销量，这是事实，这是这个事件目标与出发点。解决问题基点在于：“大企业做品牌，小企业做销售；大品牌做广告、小品牌做接触”

　　一、 发生了什么事？What happened?

　　1、老板王磊要打广告扩大销量和知名度。
　　2、广告公司提出电视广告计划接这笔活接不到。
　　3、营销经理提出店面装修计划老板王磊左右不定。

　　二、 这事为什么发生？Why didthis happen?

　　1、 表面问题：老板王磊不知道下一步怎么办，拖几天可能放弃这个念头。
　　实际问题：针对老板王磊的问题，广告公司与营销经理的计划并没有能够打动老板，广告公司采用其他的策略威胁老板，这将更加不利于业务的进展。

　　2、 问题的本质：
　　广告公司计划――是根本不可行的，这点王磊很清楚，用30万元资金，包括创意、制作、媒体发布，整个思路对不对、创意对不对、丢下去就完了，不可能是一流的创意，也不可能是最好的发布时间，如果是密集投放时间很短。

　　营销经理计划――王磊一年多积累很多经验，能够进去的商场都有了，仅仅是把专柜装修的更漂亮，他的直觉是不能达到目的，因为在他的专柜周围有的是高级专柜，在这个方面比拼不是王磊现在的实力能及的。

　　三、 专业化分析：

　　1、消费者为什么会接受王磊提供的服装产品呢，也就是他的核心竞争力在哪里呢？应该是以下三个方面：一、产品因素――面料与款式，二是价格因素――消费者认为价格合理，三是通路因素，选择大商场做专柜，把小企业的营运与的企业的知名度和信任捆绑。

　　2、王磊为什么要打广告？王磊目前的思路，商场的运作成本很高，而且都是钢性的，如果能够提高每天的销量，运作成本就下来了。其实王磊这一年多都是在做销售，而不是做广告。这个思路是对的，小企业先做销售后做广告，大企业先做品牌再做销售。

　　3、 王磊的下一步怎么走，才能提高销量。其实整合营销中有一个重要概念，“接触式营销――在最容易接触消费者的地方接触消费者”，一般可分为以下三种情况，一是媒体接触，二是通路接触，三是目标消费群聚会接触。

　　四、 我们采取哪一条行动路线？ Which course of action should?

　　根据以上分析，有三种途径选择：

　　一是媒体接触，那么行动路线就是发布信息，提供产品带来的利益点和承诺，并且到柜台去有小礼物奉送；

　　二是通路接触，那么行动路线就是选择人群最多，最密集的商场，并在最好时机，展开各种信息传播活动，扩大产品与消费者接触面。

　　三是目标消费群聚集场所，这个没有具体产品不好说清楚，自己体会吧。

　　在以上分析的基础上，在探讨广告投放策略，就有些价值了。
