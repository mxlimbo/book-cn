网上超市营销

sundy/文

　　超市，在当今社会已逐渐被人们所喜欢，它以商品齐全、价位适中、直观方便受到许多消费者的青睐。随着市场经济的不断发展，许多传统的营销方法已被大多数经营者所掌握并适用到商品营销中。随着电脑的普及，上网者人数也日渐增多。据统计，广州、上海、北京等地已有50%以上拥有电脑的中青年人上网，网上超市营销应该成为一种时尚。

　　说到电子商务，我们所知道的最多的就是在网上读书出名的亚马逊，给人的感觉是，书籍是网上销售最为合适的商品。那么，其他门类的商品是否在网上也有销路呢？例如软件销售，传统的销售方法只有靠广告，文章以及产品资料给人大致的概念，而在网络上则可以用演示版下载试用图片、文字甚至声音全面地加以介绍，让人们全方位地了解该软件的特点，然后再选择购买。传统的电话、电视购物适合的商品，当然也可以通过网络销售出去，毕竟通过电话比通过网络介绍商品效果要差得多。

　　对于小批量生产、针对细分市场的产品，由于其市场推广、难度较大，搜寻用户成本较高，无法预测其市场容量，销售商不敢进货，生产商不敢生产，所以很难通过传统的销售渠道进行销售，如一些采用先进技术刚刚完成商品化的产品。以MP3播放机为代表，通过网络销售可以迅速进入市场，进而加速其推广过程。因网络具有与传统广播式媒体不同，对消费者针对性更强的特点，所以这类商品更适合在网络上销售。

　　服务也是一种商品。一些服务已经被固化成为可见的商品，如IP电话卡等等。网上销售在消费者与销售商之间相互信托的前提下，消费者可以直接授权销售商拆开商品，用最快的速度提供诸如用户号、密码等信息，消费者甚至不用见到真正的商品就可以使用，而另一些无法固化成有形商品的，如订报刊、订牛奶预约服务等，这些也都可以通过网上超市实现销售。

　　利用互联网进行电子商务活动是新生事物，网络技术的发展以及应用的普及，必然会给我们的生活和工作带来改变，只要消费者需求，就可以通过网上超市实现销售。随着新技术的不断发展和完善，网络销售这种新型营销模式定会成为新世纪的新的营销方法。
