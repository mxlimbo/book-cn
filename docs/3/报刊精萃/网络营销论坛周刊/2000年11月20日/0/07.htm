网络广告发展的瓶颈

--广告效果评价体系的建立

caoz.isme.net /caoz

　　1.网络广告市场现存问题

　　目前国内网络广告市场的竞争还远远没有达到一个有序和公正的环境，报价单外的高折扣率，甚至正式合同外的高贴送率都是行业内不标准，不规范的竞争手段。至于其他的统计数字骗局，为做营收数字而玩弄的财务游戏什么的，更是让那些不精此道的业务人员如坠入雾中，这些，都严重影响了网络广告市场的健康发展，也给整个网络广告服务行业的信誉带来了严重损害。

　　2.本文的讨论目标

　　本文在这里讨论的，倒并不是具体针对上述操作中的某种不健康的问题，而是这一切行为产生所引出的重要话题，网络广告的信誉评价，究竟有没有一个客观，公正的标准和尺度？有没有一个对广告效果进行真正客观公正分析的机构和第三方系统在操作和运营？很可惜，就目前而言，就国内而言，我们还只能看到无序和混乱。

　　其实目前越来越多的广告主已经意识到这样的原则，仅仅凭借一些印象数和点击数字，是很难对广告效果作出足够的评价，更多的参照标准的建立，已经摆到了日程上。以下根据本人经验，将其中一些评价尺度加以逐项说明。

　　3.效果评价体系的构成要素

　　来访效果评价可以分为来访者访问行为评价和来访者受众切合度评价两大方面。

　　访问行为评价要素包含如下方面:

　　广告来访用户的平均滞留时间--尽管从技术角度，http协议本身的局限并不能获取这一准确数据，但是通过一些相关的方法还是可以获取并不精确的参考数据，由于来访用户的平均滞留时间与广告主的站点设计及页面主题表现力有很大关系，因此并不能依此指标评价广告商的表现，但是作为参考，对于广告主对多个广告商的比较和广告主对自己多种广告牌效果的比较也都是很有益处的。

　　广告来访用户的平均页面请求--这也是对广告受众是否符合广告主站点定位的一个很好的参考数据，作为参考评价，广告主可以很明显看出自己多种投放策略中哪一种策略能够吸引到自己最需要的访客，也很容易对比看出不同广告商带来访客质量的差异。

　　广告来访用户的回访情况--对于CPC（Cost Per Click）计费而言，广告商很可能为了争取利益而通过各种虚假诱惑广告甚至强制弹出手段制造点击，这些点击往往造成了真实页面的调用并带来了访客，但是很多因为虚假成分进入广告主站点的访客不会再次访问，这样即便通过广告拥有了很大的pageview也是很低效的，广告来访用户的回访情况分析就能对这种行为产生很好的甄别，当然，该分析数据依然与广告主站点本身设计密切相关，但是通过对比不同广告牌来访用户的回访情况，对比搜索引擎来访用户的回访情况以及其他广告商来访用户回访情况数据，就可以很容易分辨究竟怎样的广告带来的来访才是真实有效的。

　　广告来访用户的行为率--广告来访用户，多少比例的人产生了广告主期望的行为，是最能体现广告主投资回报率的数据，一般情况下，根据广告主期望行为的诱惑力，实现代价，该数据会有很大的差别。一般而言，有奖调查问卷大体上正常来访行为率是三分之一到五分之一，交友及人才站点的来访用户注册行为率是十分之一到二十分之一，B-C电子商务网站的购买行为率国内是百分之一左右（这和网上支付能力，信誉以及产品类型、品种、价格都有很大关系），目前由于技术准备和市场都不成熟，国内按照CPA（Cost Per Sale）费的广告商还极少，但是对CPA的衡量应当也已经成为部分广告主选择广告商的主要依据。

　　以上评价尺度有一个共性，就是都与广告主本身站点或特定行为设计和表现力有关，因此单纯数据不能作为评价广告商广告能力的尺度，但是通过针对不同广告商，不同推广行为的数据对比，依然可以有效的分辨出不同的广告商广告价值并作出正确选择以节省宣传费用。

　　受众切合度评价包含如下方面

　　广告来访用户的平均年龄构成、平均职业构成、平均收入构成、地域构成等等都是广告主非常期望了解的数据，尽管不少广告商开始承诺了定向投放功能并提供其通过调查获得的受众数据，但是由于网上隐私权以及其他原因，实质上目前广告主并不能完全了解广告来访用户的真实个人资料，一切还都是广告商的单方面承诺而已。事实上，仅仅通过一个浏览行为，一个来访IP，你根本无法跟踪和了解来访者的任何个性资料，除非你根据IP地址去接入服务商查询拨号记录，但是除了公安部，没有人可以这样做的。就目前而言，通过Web页面发送的广告的所谓受众定向，除了地域可以通过IP实现以外，其余的还仅仅停留在一种概念的层次上，或者仅仅是根据一些粗糙的统计数字作出的简单推论上。对于广告主而言，更是无法掌控和把握这里面的广告商为谋求利益而设计的种种数字玄机。

　　受众切合度评价体系的建立面临两大障碍，第一是技术障碍，收集客户个性资料并对其浏览行为进行追踪并不是一件容易的事情，虽然目前有很多公司开始做这样的事情，但是由于样本容量的限制以及获取样本成本的限制，使得这种分析仅仅针对整体网络环境或行业有价值，还很难对单独特定广告效果进行精确统计分析。第二是法律障碍，由于网络隐私权益保护已经被越来越多的关注并开始进入相关法律条文，收集客户个性资料的成本和代价也就扶摇直上，这也制约了这种评价体系的建立。

　　4.结论

　　无论现实有怎样的困难，新技术和新思路总会不断拓展，解决方案总会产生，为此文，姑做抛砖引玉，希望广告同仁能依此打开思路，群策群力，共同建立一个公正，有序的竞争环境，来共同拓展市场，将中国的网络广告，推向新的层面。

名词解释：

　　广告主－－购买广告并发布的单位
　　广告商－－出售广告并承载发布的单位
　　广告来访用户－－通过网上广告进入广告主网站（而不是广告商网站）的浏览者
　　平均滞留时间－－在广告主站点（而不是广告商网站）的平均滞留的时间
　　平均页面请求次数－－单位访客单会话进程内在广告主站点内部平均调用的页面数。
　　行为率－－进入广告主站点的访客产生了广告主期望行为的比率。
回访情况－－访客重新访问广告主网站的情况。
