网络广告探讨


纸马/文

　　总而言之，网络广告和传统广告一样也是一种传播活动。

　　1、他是付费的使用媒介物（网络）得传播活动。
　　2、非个体的传播活动，面向大众。
　　3、是一种双向传播活动，选择权在受众。
　　4、一种可以重复的传播活动。
　　5、有目的的传播活动。
　　6、是一种进行广泛劝说的活动，与其他传播活动相比有更明确的对象。
　　7、和其他广告形式一样，也是一种促销手段。

　　网络广告包括：

　　1、广告主
　　2、费用
　　3、广告承载者（网站......）
　　4、受众，就是对象
　　5、广告信息（文字，BUTTON，BANNER......）

　　优势：

　　1、自主，方便，就目前而言，任何人都可以分布广告---垃圾邮件。
　　2、对象明确，从网站的定位可以知道其对象。
　　3、信息量大，易反馈（交互）。
　　4、低成本，制作费用低，更新快，发布成本也不高（至少目前是这样）。

　　类型：

　　1、垃圾邮件
　　2、BBS，新闻组
　　3、WEB
　　4、系统广播

　　现在已经有很多得分类广告站点象易趣就是，新浪的哪个二手设备也一样，各类的行业信息网，政府信息网也算是，这个算专业广告站点才是。

　　为什么厂商给网上商城产品的价格更优惠？

　　和网络广告没有关系。并非网站帮他做了广告。如果要做广告，一是厂商付费了，二是网站自愿（非连接到厂商主页，而是那个SALE的页面，其实也是自己的广告）。注意，是低价格，不是低于成本价（那是违法的）
　　你看到的价格是贩卖价。为什么价格不一样你应该更清楚。

　　以书为例，就算是5折，网站和出版社还是有利可图。
