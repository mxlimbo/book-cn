:: 特别推荐 ::
　　玫 瑰 [灵犀]
　　他坐车走了，他让花店里在他走后，包100朵白玫瑰送去给她，这是他这一生都欠她的，而他，所有的感情，都刻着一个叫玫瑰的名字。

:: 放飞心情 ::
　　携我归农 [寄怅亭主]
　　晨雾弥漫山野，风里有野百合的芳香。我和你就这样坐在晨雾里，坐在那低矮的草屋前。你睡着了吗？我低头望着你的脸，红润的象雾里的海棠。

　　两小无猜 [丁香]
　　那天，我觉得很快乐，仿佛又回到了小时候。真幸运，在成长的过程里，会有这样一个朋友陪我长大，会有这样一段两小无猜的时光留在记忆里！
