 玫 瑰 2000年11月20日　第二期
【作　品】玫瑰
【作　者】灵犀
【信　箱】cocowl@263.net
【授　权】独家授权
　　他还是无法适应这个城市，尽管来了有很多年，这个有着很多林立的大厦和街角咖啡店，以及来来往往衣着华丽而又行色匆匆的人群的城市，对他来说，这城市里发生的一切与他无关，这里依然是那么的陌生。每当做完一天的事情之后，从办公室开车回居住的地方，看着街上明亮和闪烁的街灯和绚丽的广告牌，竟然有一种身心疲累的感觉，但是他，才30岁，一切似乎才刚刚开始……然而他觉得自己老了。
　　每个周末，如果他不加班，会走路去不远的一家咖啡馆喝道地的蓝山咖啡。而且走过第一个街的拐角，他总是会放慢脚步。那是一个很小的花店，与其他花店不同是，那家门口的花桶里总是有很多的白玫瑰，洁白柔润的大朵大朵的花瓣，让他恍惚地想落下泪来，而每次花店里的老板总是冲他善意地微笑，也从不上前询问。
　　望着这个城市的天空，他，微微地在来去匆忙的人流中叹了口气，在陌生的这里，他依然无法忘记那个叫玫瑰的女孩。所有的记忆好象从未稍稍离开，深刻地仿佛一如当初。
　　认识她有好多年了，从小学开始，一直是同学，然后是初中……一直一直。从小的时候早晨开始叫她上学开始，他像兄长一样照顾她，他在学校，年年是三好学生，她却很皮，让他背了不少黑锅。每次想起她小时候的顽皮，在他心最柔软的地方，会有一丝甜蜜荡漾开来，久久的，直至嘴角有笑意起来。
　　慢慢地，他们都长大了。他依然很呵护她，仿佛已成了习惯，日子在他眼里还是和从前一样，但玫瑰开始有了变化。有很多人都说玫瑰长得越来越漂亮。总有一些男生搭讪上来跟他们一起走，故意说一些笑话让她笑得前仰后翻。
　　也不知道什么起，他念着玫瑰的名字，心里开始觉得温柔无比。她属于那种诙拮的女孩，聪明，漂亮，但最美在她的灵气，她的眼睛，若有所思的时候，让他的心颤抖。他固执地认为，只有他，才最能读懂她的美。但他，还是不露声色。他觉得，对她的好，她会觉得。而且，不知怎的，他有点怕。
　　时间过了好久，他去念了大学，而玫瑰工作了，每次放假回来，他总是带一些希奇古怪的小东西给她，他知道她喜欢什么，喜欢看她欣喜的样子。
　　4年后，他离开了可以有机会留下的城市，他选择了回来，在一家企业做技术方面的工作。他还是像以往一样，照顾她，不快乐的时候陪着他，也看到各色的男孩在她左右。玫瑰告诉他，虽然她不喜欢他们，但很喜欢被人追。她转过头来轻声问他，是不是觉得她很虚荣？他一直微笑，摇头不语。
　　有一次跟她一起走，路过一家花店，里面的花很多，其中有一只桶里有很多的白玫瑰，她看了很久，他问，是不是很喜欢？她轻轻说，你喜欢什么色的玫瑰？他心里一动，但还是说，男孩很少喜欢花的，女孩子才这样呢。她回过头，温柔地看了他一眼，说，我很喜欢白玫瑰，红玫瑰太张扬了。他说，我知道。他一直知道的，不是吗？她怔怔地，过了一会，转头看他，他看到她眼神若水一样清澈。刹那，他很想握着她温热的小手，然后对她说，让我送你白玫瑰好吗？但他，什么也没做，只是轻声说，走吧？
　　有一天，他在街上遇见玫瑰，身边有一个帅气的男孩，有很亮的眼睛。但他没怎么在意，他见过好多在她身边的男生，他一直以为，不会有人威胁到他在她心中的地位。
　　直到有一天，她在他寝室门口等他，她看上去有点迷惘，他问，怎么啦？她咬了一下嘴唇，楞了一会，眼睛看着对他说，她动心了，为那有着很亮眼睛的男生。他的心好象被什么东西重击了一下，他险些不能自已。他问，你确定？她低下头，说，从来没有一个男孩子像他这样追过我，让我觉得我对他很重要。我不知道这是不是我想要的，但爱情本来是冒险的，对吗？他不能回答，他已经溃不成军。只听见她继续说，可能我不是属于那种安分守己的人，我开始爱上这种感觉。我不喜欢平静，平淡如水的感情。那天她走了之后，他喝了好多酒，在他的本子上写了好多，玫瑰，其实最爱你的人是我。
　　但他知道，他永远的不会让她看到。
　　过了一个月，他辞职了，去了现在的这个城市，找了一份辛苦但待遇很好的工作，过了不久，他有了自己的车。除了感情，一切都很富裕。
　　有一天中午，很好的天气，他在大街上，手机响了，一看是她打来的，他按下OK键，她说，你什么时候回来？他说，怎么啦？她说，我下星期结婚，你来吧？他一楞，哦，那我回来，恭喜你了。
　　他在她结婚前天的那天回来了，她看见他的时候一脸欣喜，他一阵心酸涌来，但还是微笑着，像以前那样拍拍她的手。吃饭的时候，他低声问她，准备的怎么样了？还有花没去定好，等会去看，她回答。陪我去吧？她央求。他说，当然可以。
　　他们慢慢的走去那家花店，路上，她忽然说，从小到大，依赖你惯了，你不在，我一个人还真不习惯呢。他回答，快做别人妻子了，可不能像小姑娘那样任性了。她说，是呀，不过……不过什么？他问。她摇头说，不知道怎么的，我心里很乱，我是不是有患上了婚姻恐惧症？他转过头，看着她，你爱他吗？要听实话吗？她说，当然，他看着她。她迟疑了一下，回答，我不知道。他笑着说，傻丫头，胡思乱想什么？她也笑了，他说，对呀，这样才好，你要知道，开心很重要。
　　花店到了，在一个角落里，放着一些白玫瑰，不知怎的他心里一颤，觉得似曾相识。她走过去，蹲在那里看着好久不语。他走过去，站在她身后，说，我知道你喜欢，但结婚不能用白玫瑰的。喜事还是张扬一点的好。她奇怪的看了他一会，说，你怎么还记得我以前说过的话？他一冲口，你的，我什么都记得。她不语，然后很快的选好了花。走出那家花店，一路上，她没再说话，他问，怎么啦？她还是低头，他觉得不对了，他停下来，看着她，她慢慢抬起头，眼睛里有什么东西闪烁，然后轻声的说，有些东西，你不说，我怎么知道？他震了一下，低低的说，从小到大，我一直以为你知道的，她一下子泪下来了，哽咽着说，你现在怎么让我去面对明天？他心酸的不行，眼里也有潮湿的气息，我以为你不爱我。对不起，我真得不想弄成这样子，你的快乐，对我真的很重要。她泣不成声，他用手缓缓地擦去她的眼泪，过了好久她抬起头，你走后，我才知道你对我的意义，是不是要失去了才能知道错过的是什么？他搂着她，他的眼泪也大滴大滴露上她的肩上，泪弄湿了她的发丝，他在她耳边低声说：我爱你，玫瑰！对不起，是我做错。她摇摇头，不是，这太残忍了，乘我们没有崩溃前，你走吧，好吗？他点头，吻了一下她的嘴唇，一步一步的回头走远，冬天冷冷的街上没有行人。他忽然大声喊，玫瑰，我也喜欢白玫瑰。说完，他迅速的走过街的拐角，背靠在一堵墙上，他怕，他会支撑不住。
　　第二天，他坐车走了，他让花店里在他走后，包100朵白玫瑰送去给她，这是他这一生都欠她的，而他，所有的感情，都刻着一个叫玫瑰的名字。
