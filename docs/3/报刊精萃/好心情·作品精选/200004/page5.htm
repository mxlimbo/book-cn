 灵魂木偶 2000年12月10日　第四期
【作　品】灵魂木偶
【作　者】太阳
【信　箱】summer-snow@21cn.com
【授　权】授权刊载
　　舞台上，有只漂亮的木偶在各个角色的人群中穿梭，它想和人大打声招呼，亲口说声：“你好”。只是轻轻的一个微笑，但话语是他人的声音让耳朵听到。这就是做木偶的代价吧，是那么多的线将它轻轻牵住，还是永远的约束，如那蚕丝一般，却永远没有破茧而出的那一天。幕前，那一个招手、一次低头、一个轻声问候还是慢慢跺步，是否都有自己的思想。木偶不明白，为什么从没有一幕让自己自由发挥的表演。不，应该有自知之明，因为那样的结局是人类的歧视和肮脏的垃圾筒。乖乖的吧，也许又有一个美好的晚上。
　　帷幕落下，一切都将沉寂，木偶靠着窗。那是天空的颜色，没有太阳的星空是一抹梦幻，将它轻轻环绕，那是木偶最爱的颜色，要不然又怎么会有蓝色的眼泪。天空向它招手，告诉它广阔的心情；云儿慢慢漂浮，倾诉风的温柔；夜晚自由的鸟，欢唱高飞的感觉。
　　缓缓的，是那一抹烟还是木偶的灵魂，离开了这个躯体。看着飘渺的云彩穿过自己虚幻的灵魂。自由放荡，还是前世的梦仍然继续。平日的种种束缚，让小鸟想要翱翔天际。是那看不见的心弦，将他层层锁住，木偶的生命如此过活，自在的表情，愉悦的问候是梦中奢侈的幻想。
　　夜晚，深深静静，木偶的思想在逃出，木偶的灵魂自由放飞。世界任凭遨游，也许该逗弄世间的荒唐。还是嘲笑人类永无止尽的悲哀。让自己控制的手轻轻抬起，抚平今日的伤痛，快乐或许会再次回家。梦啊，继续吧，嘲弄吧，是夜间唯一放荡的灵魂，却是平日悲泣的木偶。
　　灰姑娘的时限，不需要钟声，一缕阳光射进窗户，什么都没有，只有木偶，还有门外匆匆的脚步。轻声说句“你好”吧，来衬托一下那颗刚出现的蓝色泪滴……沉寂……等待午夜时分的灵魂木偶……
