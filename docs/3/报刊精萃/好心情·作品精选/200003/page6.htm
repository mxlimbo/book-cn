 心 痕 2000年12月1日　第三期
【作　品】心 痕
【作　者】安琪
【授　权】授权刊载
　　有梦的日子，有所追求，有追求的日子挺苦，却是心甘情愿。
　　好久了，没有那收获的幸福，好久了，没有前进的苦味。
　　好想站在山的峰顶，感受那征服的快慰，却是总在这同一水平，让心变的平庸。
　　哦，原来心这么容易疲乏，原来最想遨游的心在岁月的沧桑中，也会没有了锐气。
　　哦，我明白了，为什么总是有那么多壮志难酬的遗憾；我懂了，为什么总是有那么多知音难觅的伤感。
　　可是，生活原本就是这么的折磨人的么？
　　那么我们少年时的雄心勃勃，以至付出的努力全都是孩子才会有的热血沸腾吗？
　　真的不相信，曾有的热情在有了能力的今天，反而要在岁月的磨损中无为地渡过。
　　那么我们现在还有什么呢？好了，让我们从新来想想吧。
　　就当作我们从新开始我们的生活好吗？
　　我知道，生命在经历了休憩之后，再从新找回自信乃至生活地无怨无悔，真的是很难的事儿，可是你是不会怕苦和累的吧！也或，单单是因了心中那不泯的梦想，也值的你去一试锋芒！
　　何况，你曾是那么优秀地站在讲台，将你的梦想与你的朋友分享，哦，那时的你，多么自信，多么出色，不仅只为了你的梦，还有那么多愿意和你一起追梦的同行者。能够过的充实并且可以帮助很多的人，是你当时引以为傲的事儿，那些往事刚刚过去，你离开了欣赏你，追随你的人，一个人面对无奈的生活，变的不再做梦的时候，你原谅了自己的逃避，可是你的心却不会时刻停止懊悔。原来你本来就不是一个可以容忍无为的人。
　　还好，沉迷的日子不多，你便觉悟了，这样，你还可以拥有青春，去为你不服输的心一起努力，为你的明天开创你想要的丰富的生活。
