:: 特别推荐 ::
　　曾发生过的爱情 [雪茸花]
　　当最后一缕斜阳从我身上移走，指间流出的文子已渐渐奏出的欢快的调子，唯有辉满怀期待的眼神，和宁渐行渐远的脚步缓缓划过心间……

:: 放飞心情 ::
　　离　别 [小傻]
　　校园里静静的安静了下来，他们走了，带着我们的期盼，带着我们的祝福他们走了，走上了将属于他们的真正的天空，我只能为她祝福……
　　觉醒春天 [苹果]
　　春天又是隐蔽的，藏在人的心灵，酝酿着美丽的芬芳。想想，这隐蔽的，不管是忧郁的、深沉的，还是热烈的、开朗的，也不管是舒心的还是痛苦的，打开看看，都有着金子般的珍贵。
　　幸　福 [蓝波]

:: 爱情魔力 ::
　　如歌的情感往事 [宇速飞翔]
　　总有太多不懂，缠绕着我；总有些许冷漠，凝聚眼眸，自己的心情还是需要自己去感受的。
　　寂寞梧桐 [JUDY]
　　火车呼哧呼哧地跑着。越片北行，天儿越冷。我的脸上已开始起鸡皮疙瘩。这样的天气，除了我这样的傻瓜，谁愿意满世界乱跑？
