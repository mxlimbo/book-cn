【杂文评论】

◇　金庸王朔之战后记　◇

·帝释天·

　　现代武侠之于文学，三十年一贯归于通俗。侠之大者金庸，现在既然有
了“20世纪文学大师”的称号，如今又要“探求一个灿烂的世纪”。那就不
能不把武侠提到理论的高度来论论了。。

　　心理学有“逆反心理“一说，作为心理现象，这一学说有其客观依据，
当深入探寻造成逆反心理的社会根源。都与其命令与强制有关，而命令和强
制剥夺自由。当社会存在命令与强制，逆反就是对命令与强制的反抗。造成
了反面对自由和神圣个体权力的捍卫。武侠小说也许就源于此总心理之上。
借助价值、道德、正义、爱、尊严、人性、美和良善。。等等。创造一个神
奇的文字世界。

　　金派武侠源于六，七十年代。。当年港，澳地区的休闲小说，在大陆慢
慢不再闭关自守后，流入大陆。对大陆不同阶段所造成的影向和思想意义上
的深远现在大家都见识到了。只要剥夺自由的命令和强制存在。武侠应该就
不会消亡。剥夺的越蛮横，大陆热爱武侠的人在无意识间就会越多，金庸借
助他的文化功底。在他的小说中不断论证人性的高尚和伟大。他的才能表现
于使武侠内涵从玄幻，文化垃圾到血淩浚的兽性搏杀变为一曲人性的凯歌。

　　还有影射。说到这不能不提提“笑傲江湖”，此书通篇偷窥、畸态、龌
龊、唯我独尊。。看了很难不把它当成是对文革的影射。六。七十年代。有
华人的地方就有极左势力，在专制体制下，批判者不得不把呻呤称为歌唱，
写为荒诞。避免无情打击。并让心领神会的读者暗中心有戚戚，而造成文字
的整体失去武侠的本意。这不是金庸的错，是统治者的罪恶，是时代造成的
。

　　我看的第一本小说是金庸写的，八三年我记得很清楚。当时我对现实充
满麻木。谢谢金庸，我第一次对化统文化的认识。对侠的向往。。是你给我
的。
　　但文学是什么，是对人类生存困境、人的本能、原欲、生命的玄秘和对
社会人生的文字表白。是纯洁精神的源泉。是人文、哲学、神性三位一体，
超拔、高迈的是而对生命的终极关怀精神。当思想述诸于文字，就与文字存
在互助互摄的关系，使作者获得自信、稳固、充实。读者在此中获得作者的
多重情感与之发生摩擦、盘诘、重审自我本身或在顷刻间产生复杂的种种情
绪，冷静、细密、求实地分析，探究经验，文化，历史的多重内涵，呈现于
文字的各种可能性。而这些金庸也许是因为世代的局限性。真真纯文学的文
字他没有留下。而只把武侠作为一种文学表达方式太狭义。

　　就此文体而言

　　一：谈到历史。。武侠没有给予历史充足的精神背景与文化关联，构不
成历史感与民族精神在文字中的重逢。
　　二：写作本身没有对语言进行关注，构不成对语言文字描写的新意和特
色。在表述上不存在进步。
　　三：在思想上更不存在意义可言，不过旧瓶装新酒。对传统文化进行金
氏注释。。而对传统文化的注释又如此薄弱，一笔带过。蜻蜓点水。对儒者
的弱智、道者的狡智、墨者的奇智、佛家的出世进行金氏注释。如玩耍一般
。如若金以后没人超越。。武侠仍只配称通俗二字。永不上大雅之堂。

　　给予金庸“20世纪文学大师”的称号，他受之有愧。对他与池田的对话
“探求一个灿烂的世纪”。没看过。就不说什么了。希望他配谈得上这个题
目。
　　对于大家对此次金，王事件行为的内在思维反思，上班了。留待下回分
解。
