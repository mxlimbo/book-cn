【杂文评论】

◇　也谈王朔金庸之争　◇

・Sieg・

　　本来是不想谈这个题目的，因为王朔已经在他的拥趸们的摇旗呐喊下向
金庸提出的挑战足以让他王朔自己再次成为受众关注的视点，在这样的背景
下，无论你写的什么都会被人轻而易举地归为护金或是保王，所以，三缄其
口当是最好的选择．

　　但还是谈谈吧，如果中国文学的道路上真的已到了缺失路标的时刻，那
么，多一个人介入这场混战所添的解构之乱应该是比他所带来的建构之秩要
少．如果那个人的脑子足够清醒的话．

　　现在假设我的脑子是足够清醒的．

　　当我刚听说这事时，我的第一个判断就是金庸的武侠小说是该批评，但
怎么着也轮不到他王朔来批评，五十步笑一百步也许还有可笑之处，可一百
步笑一百步那就叫荒唐．

　　王朔的小说我是在看完电影＂永失我爱＂及＂阳光灿烂的日子＂后立马
买的，四大本书在三个月后便送了人，－－－我一般总把看过的臭书送人，
当然朋友结婚时所送的书是个例外．

　　当时我的感觉是姜文他们的才气要比王朔高多了，能把一堆堆烂泥般的
文字整成如此鲜活水灵的电影语言，真是了不起．

　　王朔的小说总是充满对话的机智与行动的狡黠，但这些风格永远和他所
叙述的主题隔岸相望，当我见多了侃爷的嘴上工夫不过是言语层面的机锋，
痞子的乖张举止不过是应用语境的错置后，我对王朔小说的阅读态度仅比他
对金庸的好上那么一点：他是不读了，我是隔段而读，跳章而读．

　　的确我们以前所创作的诸多小说里，为了某种现世的理想或规范而把小
说的思想性给挤压在一个逼仄的环境里，然后对大陆的小说语言堵在言语资
源匮乏的政治语境之墙里进行残暴地强奸，以至被奸了十年后，大陆的小说
语言象受了刺激发了精神病似的，怎么也不敢从那可怕的政治之墙的阴影里
走出．

　　王朔式的小说在带领受伤害的大陆小说语言走出阴影的这一点上是功不
可没的，它把那如同埃及法老般阴沉的专制负壳给抛了出去，而且象抛一只
乌龟壳一般抛得干脆利落，于是，小说的言语开始有趣了起来，小说家们开
始懂得原来不用什么外在的理想的指引，小说本身也可以有存在的理由．

　　可是王朔毕竟不是摩西，他的小说带领着大陆的小说语言走出埃及，却
带头纵情于言语自己的金牛犊中，他不断地嘲笑那只乌龟壳是多么地愚不可
及，却忘了嘲笑的快乐并不能给小说带来任何将来的指引．

　　知道君特－格拉斯为什么能凭他那些跟生铁般僵硬的文本得奖么？知道
达里奥－福为什么能凭他那些小丑般恶俗的剧本得奖么？－－－只因为他们
都敢在蔑视权威的同时还在膜拜一个超越现世层面的十戒．

　　而从来就没有基督教文化传统的王朔及其追随者们一旦抛弃了乌龟壳就
变得比流氓无产者还无所顾忌，＂我是流氓我怕谁＂的宣称，即使不是王朔
想看到的，也是王朔带出来的．

　　可是，拒绝政治的负载并不等于可以拒绝一切负载，福柯巴特他们敢拒
绝负载是因为他们的前后还有拉卡托斯图尔尼埃，他们互为背靠把西方的解
构与建构统一在一个可以维持的动态平衡状态下，而我们却只把解构的一方
取来去对付那只可恶的大乌龟壳，却不想想娜拉出走以后该穿上什么．

　　于是八十年代后，王朔把大陆小说的语言带到了一个赤身裸体的境遇里
，在那里没有先验只有经验没有形而上只有形而下没有灵魂的飞升只有肉体
的困苦．

　　于是四大俗里金庸琼瑶们自然需热情得一拥而上，献棉被的献棉被扎头
绳的扎头绳，把大陆小说弄成了大陆俗说，你们看，社会主义文艺领域在改
革开放后是一片繁荣啊啊啊．

　　这能怪谁呢？正如大兴安岭大火过去后，首先大量长出的是低等且低经
济价值的植物．

　　法轮功不也是在共产主义之舟于腐败之海里挣扎时才能获得膨胀的空间
的么？

　　一旦一个资源空间得到开放，只要高秩序度的事件没有及时进入，低秩
序度的事件就会蜂拥而至．

　　所以，王朔小说和金庸小说能在大陆共同繁荣，这是一种和生物学里共
生差不多的现象，因此，按道理讲，我看王朔应该和金庸来个阿尔巴尼亚式
的拥抱比较合适，犯不着他去学曹丕逼自己的难兄难弟．

　　说到这里，我还得插些内容来圆自己的观点：那就是王朔没有做到的事
情，余华苏童格非叶兆言他们也没有做到．诚然，余华他们的文本比王朔的
更有层次性，而且在思考历史事件与生命意义方面要来得更严肃深刻，但是
，他们的文本仍旧停留在与现世层面同阶的水平上，这使人看完他们的作品
后只会在这个世界里更加无助地沉沦，而不会让人获得一种超越现世层面的
宗教性的提升．

　　在一个没有基督教文化或印度佛教文化等加持的社会空间里要让人们得
到超拔这个现世层面的体验是相当困难的，但是，如果对于人类来说，向往
纯粹向往终极向往永恒的确是一种本能冲动的话，那么，即使中国没有形而
上的宗教土壤，我们也应该嗅到期待这土壤出现的冲动嗅迹，哪怕土壤永不
出现，那我们至少也应该领会到这冲动嗅迹的存在．

　　而他们谁负载了发出这嗅迹的属于中国的文化负壳了呢？

　　在文学领域，我找来找去只找到王小波．那水一样的文字里浸着圣火的
种子，让王朔或金庸他们汗颜三九．

　　说完王朔说金庸，金庸对王朔发难的反应颇象电影＂新方世玉＂里的雷
老五，招人欺侮后正欲发作却忽将手一抹怒脸后露一亲切笑面道：＂我以德
服人＂．

　　金庸的武侠小说是我初一时出现的，我可入迷了，那时也不知王朔，金
庸也没现在有名，倒是梁羽生名气大得很．我看的第一部金庸小说是＂天龙
八部＂里的第四部，讲虚竹如何凭着蠢笨与诚实得到了功夫权势与美女，当
时我对权势不敢兴趣，但对他的功夫和梦姑却颇为着迷，晚上睡觉时就想着
明天上数学课时，窗外飞来天山童姥把我从教室里裹挟了去，带到一山洞里
用无上妙法一会儿就把数学啊语文啊英语啊以及其他许多我又恨又爱的知识
全塞进我脑子，另加一身比虚竹还高超的武功及长生不老之术，还配给我一
大堆漂亮的姊姊妹妹，让我快活得天天不用去上学．

　　一晃十多年后的去年这时候为了重温往日的快活我又买了金庸之盗版全
集，结果看得一点也不快活，于是也将之送了人，不快活的原因就是他的文
本叙述里有不可容忍的逻辑错误．为这事我还和ＡＧＩ联手在青年报ＢＢＳ
上恶辩了一番，把老挖他们辩到只能反复说金庸的东西就是好就是好为止．

　　金庸的作品如果被归为成人童话的话，我认为是相当合理的，因为这样
的归类可以使我们不必用学术语言资源去框出金庸在历史学上的各种失误，
因为金庸写这些小说的时候，就是以民间写作的姿态介入的，所以，用学术
语言资源去批评他的文本，可能会有欠公允，当然，若他或他的同盟胆敢宣
称他的作品能进入历史学的学术视野中，嘿嘿，左右来人哪，将他拿下！

　　如果他的作品是成人童话类，那么，全体中国人沉沦在童话世界里将是
一场可怕的悲剧：这不是一个和心理年龄相称的现象，不仅仅是对每个中国
人个体，也是对整个中国国民的群体．

　　一个民族的成熟，不是仅看它是否懂得人与人之间的仁恕之道，而是还
要看它是否有能力将物与物之间的关系正确映射到这个民族群体的集体思维
中，而一个整天沉溺于如同白日梦般的童话文本的民族，在今天这个科学技
术一日千里的境遇里，将会由于它始终持有童话的思维特征而被外族所消灭
，就象消灭一头善良的猪．

　　记住印第安民族和玛雅民族的衰亡吧，这两个沉浸在自己童话话语里的
无害者，将永远以受害者的形象立在同样无害的华夷之邦的面前．

　　也许武侠童话较之科学幻想对国人来说读起来更加亲切自然，而内功剑
术较之超弦探针对国人来说听起来也更加玄妙刺激，可是，如果国民性的某
个特点成了关系到一个民族将来的生存境遇的弱点的时候，我们是选择放弃
我们由传统而袭来的乐趣，还是选择接纳从异族而泊来的无趣？

　　金庸或法轮功还只是小童话，太平天国大跃进文化大革命等等才是大童
话，而在童话世界里的中国人又有几个能象顾准或遇罗克般保持成人的清醒
头脑呢？我们一次次从童话世界里出来，又一次次不自觉得回去，因为我们
太依赖我们的童话思维了，犹如吸毒者无法离开他们的海洛因．

　　当文化新左派们煞费苦心地为中国文化传统寻找自己的语言资源的时候
，我总担心他们是不是在寻找更大的童话题材．

　　扔掉吧，一个民族撑到现在，够可以的了，古印度文明古波斯文明古巴
比伦文明今安在？－－－是到了扔掉我们所喜欢的东西的时候了．

　　另：小乘佛学真的比不上大乘佛学吗？在这一点上我不同意ＡＧＩ的观
点，相反，我认为金庸的观点是有合理之处的．＂小乘佛法是’权’的妙法
，使得智力较低、悟性较差的人能够了解和接受，好好修行。＂是大乘派的
观点，你不能把大乘派的观点作为第三方观点来对小乘派进行批评，否则，
我拿小乘派的观点来冒充第三方观点来批评大乘派不就也同样可以了吗？另
外，小乘佛学的许多理论多在初转法轮时提出，许多观点都是基于客体事物
的实在性提出的，而大乘佛学则主要是在客体事物的虚幻性的立场上提出自
己的论题．在初转法轮时，以上两种对立的观点是并行出现的，而且一开始
并没有什么大乘小乘之分，当时只有佛教和数论，胜论，弥漫差派，耆那教
，吠檀多派等等在就以上两个观点及其他一些论题在争论不休，大乘是后来
作为异端从原始佛教里分离出去的，倒是它讲究入乡随俗权变宜人所以才在
中土及西藏开了个遍．所以，即便是大乘的理论后于小乘出现，我也看不出
前者有什么地方的观点要比后者来得高明，毕竟这些都是哲学理念上的差异
，根本就无法说是孰优熟劣，正如在亚里斯多德与柏拉图之间你难以说出谁
的理论更优秀一样．

　　在我看来，把佛教分为大乘小乘纯是只有历史学上的分类意义而没有什
么逻辑学上的分类意义，这种就好比把哲学划分为唯心主义和唯物主义两大
块一般，除了在历史学上我们记录下这么一种划分法外，从逻辑的角度来看
，这么划分纯是凭阶级立场在用主观意志瞎比划．

　　如果让我来划分佛学，我就按照某派的理论是否支持连续性和是否存在
存有性来划．象大乘里的唯识论就属于支持连续性（藏识）不支持存有性（
唯藏识）的，而小乘的一切有部就属于不支持连续性（点刹那）支持存有性
（点刹那是实存）的，而中观一派则是即不支持连续性也不支持存有性（八
不）．

　　如果硬是要在佛教领域把选个最复杂派别，我就会选择因明一派，那位
令人尊敬的法称，他的思想使对比之下的中国禅宗象一滩泥水一般令人厌恶
，所以，在中国，向来喜欢童话思维而拒绝复杂思考的中国士大夫们自然接
受了得意而忘言的其他各类派别，所以悟性我不敢说，可单就智力方面，我
看别说王维、苏轼了，就是他金庸自己在法称的著作面前，也是属于“智力
较低”的．

　　王朔金庸之争即便是放在世纪之末来看，也不过是一次可有可无的小事
件，然而，这里面隐含着的问题，却是让世纪末的中国人不得不直面的，我
们在八十年代左右喊的＂２０００年实现四个现代化＂会带着它天真而无邪
的童颜在２０００年到来的那一刻偷偷地死去，而我们国家的宣传喉舌也会
对这一死亡事件保持路人皆知的沉默．如果一个民族的体面不过是建立在故
作遗忘的儿童心态上，那么总有一天，王朔金庸刘晓庆赵薇冯小宁之后其他
的俗将会把这长不大的中国活活溺杀在，只适合成人居住的地球村里．

　　想想也真奇怪的，这么老的一个国度，却有着这么嫩的心灵，如果这世
界没有氏族间的竞争，这将是多么奇妙的一个景象啊．

1999.11.24.
