【诗词歌赋】

◇　阴影的深处　◇

・风儿・


我将坐回秋天的深处，某个夜晚
聆听大片的落叶向冬天进军
而春天，在冬的高处发芽，在半空
吐出淡黄色花瓣

湖水，飘上天空的宁静丝绸
我在水上画你的灵魂，用一滴水
一千只乌鸦抵挡不住你的突袭
微笑的灵魂漫过堤岸

夜晚和进化无关，和腐烂无关
和阴影深处的爱情无关，夜晚穿过
燃烧的夕阳，岁月金黄
夜栖息在花瓣上，被空气系住

这片安静的空气，歌声一般锁住我
仿佛某个浓烈的正午，忽然失去力气
阳光从时间的裂缝渗入，淹没了我
轻舞，闪烁，痴笑着隐退

如此多幻想，经过这片空气
仿佛我们依然静坐远方，背靠大海
常青藤从你的肩膀蔓延而来
覆盖了我的小小哀伤
