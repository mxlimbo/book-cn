◇　老人的教训、民俗研究和方言　◇

・samimi・

　　老人的教训在年轻人乍听来大抵是不经的，细细想来却也很能自圆其说。
这里面往往少有科学的成分，奇怪的是可以解释得通，而且对我们从科学的
书本上发现的一些疑问有莫大的帮助。有一些教训的本身也很有趣，多用俚
俗来说一些复杂的道理，掺杂些民俗的成分，若是将这些作为民俗研究的资
料，也是很不错的。

　　比如说，我们家乡有一句俗话是：“平平路摔死猪母”。这句话有一个
故老相传的典故。大概吾乡地小人稀，以前也没有在京城里做官的人，所以
没在各地流传开来，不能令普天之下共享。这里揭示的道理却不简单，劝大
家处世时要“眼观六路，耳听八方”，与“行路难”是同一个道理。还有一
句：“桐山的夜壶只养个嘴”。这一句里的“桐山”是与我们交界的福建省
福鼎县的一处地名，此地的夜壶有一特色，就是腹腔很小，壶嘴却大，那么
容量当然就很小了。此句用来讽喻那些自以为是的口若悬河的人是太好不过
了，与大家熟知的“墙上芦苇，头重脚轻根柢浅；山中竹笋，嘴尖皮厚腹中
空”的用法也是一样。大概也是因为早时桐山的夜壶不能在全国流行开来，
这句话也就“夭折”在邻近之地。

　　在我们本地的年轻人听来，这些话里意味深重，听着很能警醒过来。还
有很多这样的教训，如果能够整理出来，也是很妙的事儿。可是真正愿意去
做这件事的人，现在是越来越少了，民俗的研究大概大家觉得用途不是很大。
说起来，吾乡界临浙江福建交界，风俗中既有吴越之风又有闽江俗韵，可以
算是一个复杂的综合体。

　　这对真心要研究民俗的人来说可以算是绝好的研究对象，此间大有可探
究的东西吧――这是我外行人的揣度。就是我们的方言吧，既与闽南话有很
大出入，与永嘉话又不甚兼容，外地人听在耳中如闻鸟语。那时在杭州读书
时，几位北方的同学初听之下，以为我们是从日本留学归来的，颇闹了一回
笑话。

　　以前曾经见过一种单行本，是本地一位老先生自费整理印制的一个材料，
其中专门记录我们这里的方言，如“阿妗”、“佚佗”、“晚昏暝”、“暗
边”、“脚头窝”、“破脚骨”之类的词语，收录颇多，但是那时由于自己
年纪小，也不能去验证里面的记录是不是很完整。可惜现在这个单行本已找
不到了，那位老先生也过世很久，引为憾事。
