◇　帝国时代　◇

·老挖·


　　霏霏从城镇中心走出来，三个村民和一个侦察兵已经等着她了。眼前是
一片黑密的森林，几条幽静小路躲闪其间。霏霏命令骑兵侦察地形，三个村
民开始建造住房，她自己进入中心，合上电闸，织布机开始升级。这时有几
只白羊自己走到城镇附近咩咩叫着请求宰割。新的村民陆续从中心出来，在
霏霏的指派下宰羊、砍树、采草莓、造磨房和木材场。这是个黑暗时代，一
切都简陋，但一切都有条不紊，霏霏准确计算出升级到封建时代所需的食物
量、木材量等，制订出最恰当的工作计划，村民们按部就班地劳作着。

- = - = - = - = - = - = - = - = - = - = - = - = -

　　亮亮从城镇中心走出来，三个村民和一个侦察兵已经等着她了。眼前是
一片黑密的森林，几条幽静小路躲闪其间。亮亮跟着骑兵侦察地形去，她坐
在侦察兵的后面也即马屁股上面，听耳边风声呼呼，人一颠一颠蹦得老高，
手指东指西命令道：“看看这边！”、“去那边看看！”……白羊、野猪、
鹿群、狼、还有天上的鸟……“呀！”忽然她想起了什么，一溜烟回到城镇
中心，找见那把电闸，将它合上，织布机开始升级了。有三个村民站在原地
什么也没干。亮亮想起需要造一些住房，还要造磨房和木材场还有兵营，还
要砍树、种地、挖金子、开采矿石，以后还得造铁匠铺、市场、大学、修道
院、靶场、马厩、武器工厂、也许还要造船坞、警戒塔、城墙、城堡……等
等。她一跺脚：“什么事儿都要我来安排，你们就不能有点儿主动性吗！”
一探头看见一游客慢条斯理地走过，再一看原来是阿吉，心中大喜：“阿吉！
阿吉！快来帮帮忙啊！”又跑到电脑前点出发展简表一看，惊叫：“哇！霏
霏已经发展得这么快啦！”

- = - = - = - = - = - = - = - = - = - = - = - = -

　　豆子从城镇中心走出来，三个村民和一个侦察兵已经等着她了。眼前是
一片黑密的森林，几条幽静小路躲闪其间。豆子看得入迷，拿出画板正要动
笔：《森林静谧的黄昏》，不想侦察兵的马儿打了一个响嚼儿，仿佛提醒她
应该干正事儿要紧。豆子立即又被这马儿和马上骑士的英姿激发了灵感，立
刻速写一幅《风马啸啸图》，又吟诗一首：“风花雪月何时了／网事知多少
／……／关机、骑马、做运动／身体倍儿棒大家哈哈笑。”
　　豆子创作完毕，方满意进屋升级织布机以及命令村民造这造那去。任务
单上，行政长官的签名一会儿是“巧克力豆”一会儿是“人木”一会儿又是
“朱槿”或是“子虚先生”，弄得村民们以为他们正被一个智囊集体领导着。
　　事实上豆子也正是这样的意思：一切决策都是集体制订的，我们是极其
民主的！背后的意思则是：出了事儿也是集体扛着的，不关我事的。嘿嘿！

- = - = - = - = - = - = - = - = - = - = - = - = -

　　雪溪从城镇中心走出来，三个村民和一个侦察兵已经等着他了。眼前是
一片黑密的森林，几条幽静小路躲闪其间。雪溪还没走出大门之前就看见墙
上有一热狗型扳手，便顺手一扳，轰隆一下织布机开始做升级运动。那扳手
的设计的奥妙在于：假如那扳手不是酷似热狗或其他什么吃的，雪溪多半会
视而不见。
　　“人哪？人都到哪去啦？”虽然阳光柔和但还是让刚从屋里走出来的雪
溪眯了会儿眼，所以三个村民和一个侦察兵全都没被看见。等他看见了，便
猛一拍桌子，桌子没有就拍在大腿上，“哟！靠！你们他妈的都愣着干啥？
还不快干活去？”雪溪这几年领导当下来，桌子拍坏了几十张，那只手早练
成了“木沙掌”，可想而知他刚才拍了大腿后的感受。所以迫不及待地：
“赶紧给我弄套办公桌椅来！什么？木材不够？叫人去砍树哪！难道还要我
亲自去砍？村民不够？赶紧出人赶紧出人！什么？粮食没啦？种地呀！哦不
对，种地要木头。那谁，赶紧给我找羊去！什么？住房还没造？哇靠！”
……
　　“你，说你哪！拿去！”雪溪在安排村民种地之前，每人发了一张纸一
支笔，“有事儿没事儿的写两笔，咱们帝国报说不定就用得着！你们也保不
准谁谁谁就会成大文豪了！”完了后鞋子袜子全脱光，俩脚丫子往写字台上
一翘，开始掰着脚趾头合计着该吃点什么了。

- = - = - = - = - = - = - = - = - = - = - = - = -

　　风儿从城镇中心走出来，三个村民和一个侦察兵已经等着他了。眼前是
一片黑密的森林，几条幽静小路躲闪其间。“景色真美呀！”“你们要不要
听点音乐？”风儿把村民招集到身边，对他们说：“我这里有好多ＣＤ，有
贝多芬的李斯特的萧邦的瓦格纳的威尔第的马勒的……晚上睡觉前我给你们
放《安魂曲》吧！”“哦！对了，你们去挖点金子来吧，有了钱就可以去大
剧院听音乐会了！……”
　　城镇中心的墙上有一金色的四分音符形的扳手，风儿对此爱不释手，一
会儿扳上去一会儿扳下来，这使得中心内的织布机升级运动一会儿启动一会
儿停止，仔细一听，那轰隆轰隆的声音竟是按着《阿依达》大进行曲的节奏
来着。

- = - = - = - = - = - = - = - = - = - = - = - = -

　　老挖从城镇中心走出来，三个村民和一个侦察兵已经等着他了。眼前是
一片黑密的森林，几条幽静小路躲闪其间。老挖根本不叫人干活，进入中心
内的电脑面前，一脸坏笑地打下一串密码：回车HOW DO YOU TURN THIS ON
回车，呼啦一声一辆眼镜蛇车就停在了中心门口；回车MARCO回车，地形图
全看得见了；回车POLO回车，好家伙！雷达地图也全都看得见了！
　　老挖跳上眼镜蛇车，风驰电掣地游山玩水去。见到一金矿，老挖两眼放
光，跳下车亲自挖金子，一边挖一边自言自语：“金子！金子！都是我的！
都是我的！”
　　老挖还利用眼镜蛇车残忍地杀害了所有靠近他的人，以使他的作弊的卑
劣行径不至于暴露。
