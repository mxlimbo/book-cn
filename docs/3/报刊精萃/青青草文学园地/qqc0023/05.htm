【诗词歌赋】

◇　鬼又进城　◇

▓ 一块八


自带干粮
　零点的鬼
　　起身上路
　　　——顾城《鬼进城》

１．

——你踩碎了我的影子！

一睁眼就是两个眼洞融融的化了
满地眼珠子圆润着乱滚
穷鬼们紧捏铜钱
锃亮而防伪　　还炫耀地吭当一声
霎那间夜色如水
小心的鬼
　走路很快
自己放个屁
便吓化了
胆子大的说我操啊！
鼻涕一大把的　　
甩到同伴尸体
（行路的队伍人员逐减
　掉队的在路边草丛里啼哭
　直到人起炊烟的钟点）

２．

冥河边
鬼见了那自己倒影
生前的脸皮层层剥落　　哗啦做响
一回头便撞破了另一鬼的隔膜
说——“我真的不是故意的！”

３．

河边放哨的螺耸起肩——
——嘘！　
（生命应该会就此轮回）

鱼们兴奋着
把自己象年画般平展
等着拣鬼掉下的铜钱　　所谓穷鬼粗心
“牙都咳松了
头发竖起来
——刺穿鱼的肚皮！　呵呵……”
淌着冰水的鬼的对话
鱼听了都不高兴
勾肩搭背的
鳞也脱了下来　　拎着
露出结实的胸肌

４．

（鬼进城
很茫然
　 手握着手站成了个鸡蛋圈）

一个说一起上我家
音调如针
“——我老婆睡不早
　　　　一双嫩手　织毛衣呢！”
遥指西施岭的手被另一支手折断
鬼们哄笑着
叠成一团骨头

５．

其实
睡着的小巷都有鬼
门环一个个晃着
连声音也敲不出
没读过书的
　会拿了石块
　　在墙上写：
　　　“——打倒隔壁张三！”

６．

回家找老婆（昔日老婆）
的鬼
迷了路拐进县法院的传达室
飘出一阵长嚎
