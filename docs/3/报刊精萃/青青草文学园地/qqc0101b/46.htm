【特刊一号】

◇　关于网络作品与评论说几句　◇

·Meg·
 
　　“从邢育森出来有话说到现在，我想讨论的中心内容还是作品和评论如
何相遇的问题，是否存在有效的途径，使乐于并且善于评论的”看客“尽量
早地与好的作品相遇，并且起到推广与传播的作用，使广泛的读者在文学价
值的层面上看到某种真相——网络创作的水平与取访的成果。”


= - = - = - = - = - = - = - = - = - = - = - = - = - =


　　呵呵，“真相”？“水平和成果”？那不是都在BBS 上摆着呢吗？

　　难道网络把好作品淹没了？俺看正相反，偶尔有一篇好评论或好小说，
立马被捧成个宝，在各文学BBS 上飞来飞去，雷同的让人看腻为止。

　　手不利索怨袄袖，自个以为写的好却无人叫好，就抱怨网络淹没作品，
评论家或读者眼光不犀利，那真叫可笑。

　　网络上谁也别想霸占话语权，各个层次与品位的人都有评说的自由，相
对传统出版来说，是最容易得到承认和共鸣的，如果你的文字在网路上没被
重视，那最好掂量一下作品的分量。

　　现在怎么出现一种本末倒置的论调，某些作者为自己文章的出版欣喜，
认为获得了评论和读者的双重承认，不错，出版多了一个获取读者的途径，
也是对作者网上写作的肯定和总结。但是放在书店架子上的书和贴在BBS 上
的文章相比，似乎受到的冷落更多些。我看到书店里蒙尘的痞子蔡和其它网
络作家的书或网文集子，似乎过问和购买这些书的大多是上网的人，起码知
道这些作者名字的人，而一般不上网的读者很少会对这些奇怪的名字感兴趣。

　　现在不少出版社在炒“网络文学”这块招牌，所以竞相出书，出于新鲜
感会一时流行，但如果作品吸引不住读者（网上或书店里的），那出了有什
么意义。

　　还是网络上互动交流式的写与读最好。但要求作者自己要克服浮躁沉下
心来，不要急与索求掌声与喝彩。

　　有个电影学院的教授感叹：电影院里生意萧条，乏人问津，而到网上一
看，各大网站的影视论坛人气最旺，高手云集；写诗的也是，网上诗坛也很
热闹，但诗歌杂志却生存不下去；以前杂志上的征婚信息里，许多人都在自
我介绍里加一句“热爱文学”，似乎那也是时尚是品位，可是如今再也看不
见那样傻冒的附加值了，在功利世俗的商业社会里，文学的位置被挤在了角
落，倒是上了网以后，俺才发现原来还有这么多人对文学如此执着，文学论
坛也红红火火。

　　写字的人应该知道在网上更容易找到知音，而不是倒回到传统出版里去。
