【内容提要】
  • 本期选择的文章潜在的特性相对接近，在编者看来就是我们怎么感受生活，怎么表达？隔还是不隔？长期以来，中国当代散文形成些习惯，或下笔千言浮泛抒情；或故意艰险高深莫测；更可怕的是套话连篇。我们能不能发觉心底的真切感受？并用适当的文字表达出来？而且是真正独立和自由的思索？编者在这方面有点自私偏激的倾向。:-)
  • 一出《戏剧》上演了。人物是你熟悉、仅知或陌生的；然而，他们在作者的打量下，却显现出逾越外观更为真实可感的面目。他们不是戏剧化角色，而是在《戏剧》中演绎真实自己的人物。请他们逐一登场吧！从诗人——中学语文教师——少年——大力神——隐士——图书管理员——清洁工——上帝——铁匠——占星士——低年级女生——小鬼——幽灵——采撷者——信徒。
  • 这次从沙门文集里挑了两篇文章，完全是为了向人们介绍推荐沙门。感谢当初AGI把沙门介绍给我们；Sieg诗评别出心裁；土地测量员的［中国历史人物的解读］则是专业性的学术论文，爱好历史的朋友们可别错过哦。
  • 最后，还有几篇精彩故事和您分享，祝朋友们周末愉快！^_^
