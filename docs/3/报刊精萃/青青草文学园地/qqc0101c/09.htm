【杂文评论】

◇　读马原的《小说百窘》　◇

·邢育森·


　　我作为一个网络写手的困惑，从马原新书《两个男人》第一篇文字《小
说百窘》中也能感受到。昨夜无事，将之拿起，又胡乱翻翻。

马原问：

　　１１）成功小说家的标志是什么——获奖情况？读者多寡？同行认可程
度？专家评价或受批评界关注状况？

　　１２）哪类小说家堪称楷模——写出一时轰动的作品？不断写出轰动作
品？没写出轰动作品但平均质量很高并拥有一个稳定读者群的作家？当时受
冷落没读者而身后负盛名的作家？写出只供少数学者研究又无任何阅读快感
作品的作家？知识分子广为称道的作家？地道的畅销书作家？

　　这是其中的两个问题。马原一共提了１００个问题，当然有点凑数的嫌
疑，后面的一些问题就是在调侃了。在李一丁做的序中，说：“另外他（指
马原）的内心过于沉重，真诚多就比较累。”确实有点累，一口气先暴露了
自己那么多的困惑和不解，而不肯用一些现成或轻巧的答案来对付敷衍过去，
是个认真的老实人。

　　马原曾经一度被视为先锋探索小说的主要创作者，可是他现在离开了当
年评论家给他设置的坐席，如同他所一直向往的那样，没着没落，十三不靠，
流浪在文学创作的旷野上了。这些窘迫和迷惘，就是他在流浪的路途上，一
路用心思索过的追问吧。

　　在《十字街头》一文中，马原夸奖“田雁宁作雪米莉是大聪明”，然后
说了美国的畅销书作家诺曼。梅勒，甚至以海明威作品的发行量超过亿册来
作为例证，在《两种深刻》一文中，接着把大仲马拎起来说事，在《作家与
书或我的书目》中，一再强调霍桑的《红字》对他的巨大影响，在该文的结
尾，他把崇拜献给了约瑟夫。海勒。

　　读完所有马原讨论文学和小说的文字，我仍然觉得他是在困惑之中。在
某一篇章里，他似乎自圆其说，得到了一个看上去还行的结论，好象解决了
他的问题。而在另一篇里，他又沿着另外的道路说下去了。他在通俗作品和
纯文学作品之间东张西望，一会替这边辩护一下，一会又给那边解释两句，
就象一个要洗澡的人，披着毛巾，左边一个凉水池，右边一个热水池，伸手
探脚，就是打不定主意。

　　我们回到他的问题本身，一个成功的小说家的标志是什么？当然，我们
也可以用另一种偷巧的办法使这个问题根本就不成立，也就谈不上困惑，我
们可以说：什么成功不成功的，写你的就是了，成功与否都是外界的评价，
他们的事情，你一个写字的人关心这么多做什么？如果靠这样的反问，确实
可以让马原很羞愧的低头不语，但并没有解决问题，困扰着的，并不只是他
一个人。

　　从这里，我们又一次看出多元化的社会里，评判价值的迥异。出版商、
文学批评家、作家同行、普通读者，来界定什么是成功的小说家的标准肯定
是互相矛盾，难以一致的。所以问题的根本就是，作者可以自己没有明确的
先入为主的主张，但是他靠向哪一边，去无形中契合哪一边的评判标准？而
所有的苦恼，都源于作者一个完美主义或者理想主义的小念头，那就是最好
这些都能满足，人见人爱，看了都说好。所以才有这么多的举棋不定，张望
游移。

　　不要用想当然的简单祈使句来代替创作者复杂而微妙的心态，不是说：
去写吧，别胡思乱想，写好你的作品就是了。不是这样的一个命令和建议就
能解决创作者内心的思量过程。即使是拥有大量读者的创作者，也难免为同
行和评论者的轻视和忽略而暗暗苦恼；即使是得到一定赞扬和肯定的创作者，
也总是为没有广泛和真正意义上的阅读而悄悄神伤。

　　而且，创作者并不是一起笔就坚定不移的奔着某条道路而去的。他只是
在创作着，并没有强烈的意识，自己是在搞纯文学或者在搞通俗畅销文学，
他会问自己，为什么他要按照某个套路来写，他还是想随心所欲的表达他的
东西。问题是他的表达是否成立，在什么样的情况下被什么样的人阅读和接
受，这是一个互动的过程，他不可能不受到影响。

　　说回来，我们要评论和推荐好的作品。这个概念也是笼统和模糊的。什
么叫好的作品？评判的标准和原则是什么？sieg的《迷宫》和宁财神的《寻
找猪二》哪一个是好作品？都是，还是都不是？绕开这些问题，我只说我的
结论：要我，会推荐前者，而不是后者。

　　所以我的方法论很简单，就是看我们做事情的目的是什么，达到目的就
好，陷入意识形态上的争论无趣。对我来说，鼓吹推荐和评论，目的就是一
个：让那些容易被读者忽视的好作品，通过推荐和评论，出版和发行，引起
读者的注意和阅读。

　　象《第一次亲密接触》和《悟空传》这样的作品，网民们已经自发的进
行了广泛的转贴和传播，包括早两年的“网络书屋”阶段，吴过访谈的那一
批写手的作品在各个书屋的安营扎寨，这些都不用说了。可是那个阶段已经
过去了。

　　我们首先看到，个人主页性质的网络书屋们，实际上已经停止了对网络
写手作品的更新和追踪，创作者也不可能一写出新东西，就给各个地方都送
去一份，他只能发到某一个比较熟悉的论坛里去。这样一来，对更多的读者
来说，确实就失去了读到他们作品的机会。我们不可能要求读者把所有文学
论坛都转过一遍，放在收藏夹里，没事就过来四处瞅瞅，这样对读者的要求
也太高一些。就说能做到，请问，都有哪些好的文学论坛是应该光顾的呢？
谁能列出这样一个单子呢？即使列出这样一个单子，谁来，怎么，让读者看
到呢？

　　我们接着看到，那些写手们也在改变之中，他们的作品正在离《第一次
亲密接触》和《悟空传》这样的容易流行的风格远去，但是他们一样创作出
了好作品。而那种并不讨巧通俗的作品，很难获得网民自发的转贴和传播。
他们固然难逃寂寞，但总要给他们更多表现的机会。而且他们已经过了混脸
熟的阶段，不少论坛都是虎视眈眈的拍砖手，怀着逆反心理，等着他们露头
就狠狠把作者而不是把他们的作品拍死以逞快意。这种传播，只会增添不少
可看的闹剧，人们的注意力全被分散了。

　　所以，评论和推荐，就是要找一些这样容易被人随便浏览和避而不读的，
篇幅略长，风格艰涩的作品，用生动明快的评论，激发人们的阅读欲望。这
是对读者和网民而言。同时，对也来上网的编辑和出版商，这样的评论也使
他们有了解整个网络文学创作现状的可能，从而使一些在网上读起来不舒服，
但出成书就更好一些的作品获得真正意义上的阅读。

　　我们已经看到，象吴过这样的评论者和访谈者已经越来越少了。２００
０年，我们看到的，都是商业网站和出版公司在网络文学领域的运作，客观
全面的批评者缺席了。即使有一些反对声音，砖头也是拍向网站，拍向事件，
拍向写手，但就是拍不向作品本身。

　　又绕了一大圈，再回到马原的问题上，其实网络或多或少会解决他提出
的一些问题。首先互联网络与大众艺术已经是事实上的婚姻（俞白眉语），
这样一个好的平台，可以很快发掘出具有畅销潜力的大众通俗作品，网络会
继续捧红一个又一个的此种意义上的成功小说家。另外，网络也的确增加了
某一并不轰动的作品获得同行和批评界认可的机会，但这样的渠道目前并不
通畅，需要建立和完善。网络既然可以成就前者的成功小说家，也必然可以
成就后者的成功小说家，但现在还远远不够。

　　每个人都有自己的方式来解决问题。比如挂羊头卖狗肉。比如脚踩两只
船。比如开杂货铺。比如以偏盖全。比如骑墙。但这只是创作者的手段和技
巧，我们环顾网络文学，批评者仍然缺席在我们视野之外。我的写字人工作
坊，只是我的一个个人主页而已，很难承担太多沉重的责任。还是希望诸如
吴过的《原创导航台》，青青草的电子杂志，清韵的《网络新文学》、元辰
的〈批评与研究〉、林瑟的〈网络文学〉杂志等等，办得更好一些吧，影响
面更大一些吧。创作者可以不要求自己的作品必须是轰动的和畅销的，但一
份评论刊物或者传播媒介，还是应该有这样的雄心壮志和目的要求的吧。
