【内容提要】
  • 本期散文以怀旧为主。如果旧得有些年头，就成为怀古了。 假正经的《变形记／二百五版》有种古怪的黑色幽默感，一个人被迫异化 或者自动变形却没有结果，不知道是哪种更可悲。其实真正令人恐惧的， 是人失去了身份的确认，失去了ＩＤ。咆哮在青草这里推荐的《程序》， 大概也是这个意思吧？ 米莲《甘家口这个鬼地方》，收罗了一帮人的怀旧。当我们看着这些回忆 的时候，恍然想起一个国家、一个民族的正在发生和消逝的历史，有时间 的流逝、有哀伤的故事、有空间的变迁。 “介子”数品，有中郎兄弟的明洁雅妙，虽然历经风月销磨和革命的扫荡， 旗手的怒斥，喜欢写和喜欢看这类文字的，也始终没有绝迹。
  • 某些诗歌，笔者的心理现实常让人滞足难前：动人的主观意象缺乏客观化 的表现。这期选编的诗作，绝不会给你这种感觉：它们客观而直接地抵达 目的地——那里是笔者经过阅读与体会的山水而涉足的山巅——纵有他山 更高更远，此山正在脚下。人人可登高，却并非人登之处皆高；重要的是， 你能否望见前人未见的风景：立足于表现好个性，深入反映事物的共性。
  • 上周末的二○○一年特刊一号整理了近期草地上关于《文学地读与写》话 题的热烈讨论，刑博士认真写作和思考的精神难能可贵，他在『读马原的 《小说百窘》』中道出一个网络写手的困惑；Sieg则是执着我行我素的抽 象舞蹈顽固派，任砸不悔，高唱『在语言尽头的绽放』：）；相形之下凡 妮的写作平常心就显得气定神闲。这里特意推荐台湾专业作家傅月庵的写 作心得，与网络写手们分享。
  • 今儿是正月初四，中国人过年首先当然是享受各种美味小吃啦。讲究美食 的中国人即使喝粥也能说出一系列典故，从小无《下巴上的粥》就可见一 斑，还有徐剑的《食粥》、一锅粥的《涮锅子》，达摩在《闲聊》中则从 “吃”侃到网文创作……哎～怎么搭上去的？您老自己看吧。：）
  • 本期的《小说故事》里我们推荐两篇新朋友的佳作：李狗剩的《过年》、 tomliang的《小鸟》；南琛在小说《太监》中的写作手法越见成熟，我们 期待她的“下回分解”越来越精彩；心有些乱为大家送上贺岁新作《碧色 片断》，赶紧来一睹为快呀～～～
