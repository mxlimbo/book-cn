【杂文评论】

◇　戏说网络论坛雅、俗、痞、黑文学　◇

·陈容均·

　　象青青草论坛的元辰、空水等人玩的是雅文学，理解他们的作品得有一
定的文化修养－－上网的人其实都有一点－－所以更重要的是要有点耐心，
因为雅文学给你的不是凉快或火辣的感官刺激，而是细心品茶后的那种余韵
。

　　要看网络俗文学，最典型的地方是各类原创武侠文学站点，象什么《鹿
鼎后记》、《续鹿鼎后记》、《假如让俺当一天鹿鼎公》之类东东正源源不
断从各位当代韦小宝的笔下摇出来。当然，小三毛小琼瑶们也是不甘寂寞，
每天都有许多或海枯石烂或电光石火的爱情故事被她们克隆出来。

　　大陆痞文学的开拓者大概是王朔那厮，痞文学追求的是风趣幽默，使用
的手法是夸张渲染正话反说实话虚说，以讽刺假正经假道学假高深见长，严
格来说，台湾痞子蔡的《第一次亲密接触》还是以俗文学成分居多，最近在
各个论坛广为张贴的《一九九九黑通社网络颁奖典礼》（作者王小山）才是
如假包换的正宗痞文学作品。

　　黑文学是用来害人的，手法是无中生有捏造事实诽谤中伤他人，如果说
痞文学是在对人大声喝倒采，那么黑文学就是扔鸡蛋以至手榴弹，弄不好是
要打官司出人命的。

　　对于大陆的网民来说，区分一下痞文学和黑文学是很有必要的，在一个
较为集权封闭的社会里，痞文学和黑文学一样被看作应该严厉打击的异端，
“光辉形象”给扭曲变形那还了得！深受这种“正统”意识影响的文化人，
上了网挨了几块痞砖，急躁点的难免就要跳脚大骂了。

　　但是在一个民主开放的社会里，痞文学虽不登大雅之堂，却也光明正大
有一席立足之地，欧美的记者们常将本国的名人画成驴子、大象、毒蛇之类
的东西在报上公开发表，而名人通常都是“八风不动”一笑了之，人人有权
利表达自己的主观感觉，说你象猫象狗又怎么啦，没说你象猫狗混血产品就
够客气啦！

　　象金庸这次对王朔的回复我觉得是很恰当的，毕竟是香港社会出来的老
报人，王朔既然以痞子自居，他也就没资格和不肯入痞流的金庸对话！他想
要和金庸正经对话也是可以的，先升级一下，以俗流的姿势还差不多。

　　即使在最民主的社会里，黑文学的作者也是见不得光的，因为民主社会
还有严明的法制，一个人若是捏造事实广泛传播，那么由此产生的天文数字
的赔偿（所有相关的人都可以向你狮子大张口，声称蒙受了巨大的精神损害
，谁让你造谣来着？）会让他彻底玩完。

　　进入新的时代，需要研究新的游戏规则，从今往后，挨痞砖的人要学会
笑对痞文学，而玩痞文学的人则要学会将自己的作品严格和黑文学区分开来
。
