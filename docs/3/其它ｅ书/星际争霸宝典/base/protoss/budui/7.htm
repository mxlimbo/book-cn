简介
轰击甲虫对地面建筑和部队都有着极大的威胁。轰击甲虫的炸弹需要生产，15矿一个。炸弹的攻击方式是爆炸，即对一个半径范围内的部队进行打击。虽然不会对自己的部队造成伤害，担友军的部队就没那么幸运了，要小心。轰击甲虫的射程很远，可以对碉堡等防御工事进行打击而不会遭到还击。战争中期时，空投甲虫是十分有用的战术。要记住即时生产炸弹哦，快捷键是R。
