特殊能力
召唤
150150
消耗150的能量，可将远处部队召唤到仲裁者的身边。偷偷潜入敌人基地后，使用一个，保证可以打他个措手不及。
冰冻
150150
消耗100的能量，射程为9。可将地面一个3x3大小的矩阵内的部队冰封。冰封后的部队不能移动，攻击，但也不怕攻击。不过可以将敌人大部队冰封后强攻敌人基地，只要把基地毁掉，有兵也没用。
简介
自己不能隐形，但却可以为自军或友军的部队隐形，不过仲裁者与仲裁者之间是相互免疫的，不能互相提供隐形。仲裁者拥有微弱的攻击力，并拥有强大的魔法，虽然是建造时间最久，耗天然气最多的部队，但还是值得一造。
