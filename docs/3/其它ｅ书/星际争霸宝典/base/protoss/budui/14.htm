特殊能力
反馈
消耗50的能量，射程10。被击中的部队能量全部消失，并失去与能量一样多的HP，如果能量多于HP则会立即死亡。

200200
消耗150的能量，射程8。被击中的敌军部队会永久变为我方部队。通过召唤其他种族的农民，可使人口上限达到600，但你也要花更多的钱去修建各族的建筑。召唤来的部队如果拥有你还未研究的科技，则你可以直接获得这种科技而不用再去研究。

100100
消耗100的能量，射程10。被击中的生物部队会被定住一段时间，而你可以趁机攻击。
简介
由两个黑暗圣堂武士合体而成，拥有很强的等离子护罩，但HP只有可怜的25点。也就是说，如果碰上了Terran科学球的EMP振荡，将会死得很惨。黑暗执政官没有攻击能力，但拥有强大的魔法，是需要很高控制能力的单位。
