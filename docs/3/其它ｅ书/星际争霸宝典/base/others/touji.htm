坦克和建筑合体
   先将建筑飞起，坦克等在旁边。然后将建筑物落下，在下落过程中将坦克移到建筑物下落的地方，并用O变形。建筑物下落后就会和坦克合在一起。
不会攻击的飞龙
   大家可能要奇怪了,不会攻击的飞龙对我来说有啥意思?呵呵,这个办法是用在对手身上的.是让他的飞龙不会攻击.具体办法是你用神族,造出海盗船(CORSAIR,就是母巢之战中新增的那种飞机),并且开发DISTRIBUTION WEB(封锁),冲到对方的基地里.如果对方此时开始孵化飞龙或自杀飞机(他是否在这样干就要靠你自己猜了),不要犹豫,立即用DISTRIBUTION WEB封在他的蛋上,然后撤退.过一会,他的飞龙或自杀飞机孵化出来了,表面上没有任何异样.但是不能攻击!(用臭蛋孵化出来的,哈哈)然后你可以想象一下,当他的飞龙准备来攻击你时,飞龙会象白痴一样原地打转.他应该会拼命按攻击键,结果还是不行,再按,再按....他会很气愤的认为他的星际争霸游戏坏了,需要重装一遍!
   感谢DRONE.RTG为我们提供这个有趣的BUG.
主基地移到碉堡上
   用shift+s连续两次将基地向地堡滑动即可把主基地移到碉堡上。
人口无限
   使用小狗变飞龙的Bug后有时程序会出问题，人口限制会消失，我曾经造出过装满一张128X128地图的口水龙。
全民变飞龙
   最近发现居然连虫族基地也可以变飞龙，据说按小狗变飞龙的方法虫族所有东西都可以变飞龙。
小狗变飞龙
   先研究出变蜘蛛，然后准备好一队口水龙和一队小狗（不一定要满十二只），编好号（假定分别为1，2）。按住Alt并不断切换1，2，1，2……同时另一只手用鼠标狂点变蜘蛛的图标。成功后小狗就会变成蛋，取消掉就变成飞龙了，不取消则变成蜘蛛，不过要花钱。另外，如果升级了小狗的速度再变，则变的飞龙速度也和小狗一样快（比普通的飞龙快多了）；如果小狗升级了攻击速度再变，则变的飞龙攻击会变为不间断攻击。此BUG只在1.04版有效。
隐形飞龙
   用Protoss，召唤一只口水龙过来。然后原理和小狗变飞龙一样，只不过用Protoss的侦察机（隐形的那种）来变。此BUG只在1.04版有效。
Terran主基地平移
   先将主基地升起，按住Shift+S，然后选一个地方降落。当快降到地面时，用右键点想要滑到的地方。用这种方法可以将主基地移到很贴近矿的地方，大大加快采矿速度。此BUG只在1.04版有效。

虫族无敌工兵
   在游戏进行中选一只蝎子和一只工兵，利用蝎子的酸暴喷工兵，等工兵的血减至刚好一滴的时候，赶紧让其盖任何一种建筑物，过一会儿再消除，这时工兵的血会变为0，无敌工兵就此产生了。此BUG只在1.00版有效。

大和架坦

要有大和巡洋舰和坦克，坦克要研究出攻城模式，地图上还要有水。 在水边把坦克转换成攻城模式，要一个支架落在水中，然后变成坦克模式。 让大和从旁边开过时把坦克转换成攻城模式，让坦克的支架落在大和的身上。 成功后大和就会背着坦克漫天飞了。(只对1.04版有效)

异形生物

对着满地爬的异形生物连点100下，它就会像原子弹一样爆炸，不过这只是玩玩的，并不伤血。

Protoss无敌大法

第一步:造出狂战士(八个以上) 第二步:造出电兵(两个以上) 第三步:升级电兵的电能 第四步:狂战士编1队,电兵编2队 第五步:两个电兵和体时的(R)一瞬间,迅速切换到狂战士。 结果：狂战士无敌。 (对1.05版有效)成功率只有5%

农民远程造建筑

此ＢＵＧ由ET.Anna发现。转载自ＥＴ星际战队

１、 确保你是虫族或人类，先拿人类做列子。如果发现对方是神族，当你探路 农民到他家时，地图是亮的，返回后地图边暗（如果想造在别人建筑上面必须别 人的地图是暗淡的），但你可以看见他原先的建筑。

２、这时选择2个农民，让他们到 12：00或5：00的位置（the hunter map 只有这2个位置成功率最高），你会发现12：00左边有1个 成3角形的矿区，5：00左边也有相同的矿区（3块矿组成），让1个农民上去采凹进 的那块矿，然后马上按"s".再让另1个农民堵在他上面（也就是不让原来农民出来）

３、这时你选择原来的农民，然后造个基地在神族基地上面，你会发现神族基地上突然 冒出了你的基地，然后那个农民会"飞"过去造，这时神族无法采矿，如果他攻击你， 你可以等基地血快没了取消掉，然后马上再造1个上去，这样神族永远无法采矿，而 你不过是每过1段时间损失75元而已。 不仅仅是基地，用如上方法，你可以在任何可以造的地方，造任何东西，你会发现 农民尚未到达目标，那里已经凭空出现了你要造的建筑. 虫族的方法也同上。(可以用来rush)比如2v2,2 zerg 每人飞1个地堡到另1家zerg 家里.....他会迷茫的发现，自己家里凭空冒出2个碉堡...然后....
