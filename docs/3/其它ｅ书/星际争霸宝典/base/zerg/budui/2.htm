简介
原始的Zerg小型部队，一只幼虫可孵出两只小狗，价钱便宜。Zerg玩家快攻的重要部队。虽然很弱，但由于可以迅速造出一大群，以多打少也不吃亏。特别是升了代谢推进和肾上腺后，效果更加可怕。后期也可作为吸引对方炮火的部队。用来攻击Protoss的圣堂武士是一个很好的选择。
