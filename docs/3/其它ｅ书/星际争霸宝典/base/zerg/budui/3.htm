简介
Zerg的主力部队，价钱便宜，建造速度快，空地同时攻击。可以在短时间内造出大量的口水龙来把对手灭掉。口水龙攻击方式为爆炸，对小型部队攻击力只有50%，对中型为75%，大型则为100%。口水龙也是Zerg空投的最佳选择。现在很多人都是靠不断的造口水龙来淹没对手。
