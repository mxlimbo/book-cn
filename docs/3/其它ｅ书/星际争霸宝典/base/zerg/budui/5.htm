简介
用自己的身体撞击敌空中部队，与之同归于尽，攻击力达到惊人的110。与Zergling一样，一个幼虫可孵出两个自杀飞机。由于生命力低下且不能对地攻击，使用时要非常的秘密和小心。对付大批量的空中部队时，自杀飞机的效果也不大--还没靠近就被干掉了。用几个自杀飞机在基地附近巡逻，可有效的防止敌人空降。另外，如果一次不能杀死敌人，就不要使用自杀飞机。敌人的部队完全可以通过回复能量罩或修理将HP加回来，你的自杀飞机就浪费了。下面是杀死不同的敌空中部队所需的自杀飞机数：
隐形飞机--2个 运输船--2个 科学球--2个 巡洋舰--6个（一般靠近前会死一个) 探测者--1个 穿梭机--2个 侦察机--3个 海盗船--2个 航母--5个 仲裁者--4个 宿主--2个 飞龙--2个 空中卫士--2个 吞噬者--3个
