简介
Zerg族的房子，拥有天生的反隐形能力。由于宿主可以移动，使得Zerg玩家的侦察可以更早展开。由于宿主很脆弱，要小心敌人的宿主屠杀战术（如用Protoss的海盗船）。另外，对付Terran的隐形飞机时记住多带上几只宿主。
