简介
由口水龙进化而来这些生物被作为蜂巢积聚地和边远殖民地的保卫部队。潜伏者必须埋到地下才能用它的尖刺攻击，攻击模式为飞溅攻击，用他来对付人类机枪兵和虫族小狗简直是易如反掌。唯一的弱点是，在地面上时他们一点攻击力都没有，而且挖洞也需要一些时间。要小心的是，潜伏者的地刺会误伤自己的盟友。
