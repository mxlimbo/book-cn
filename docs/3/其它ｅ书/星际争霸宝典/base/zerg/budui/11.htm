简介
由飞龙进化而来，是Zerg最强力的空军兵种。吞噬者的攻击方式为爆炸，对小型部队只有50%的攻击力，对中型部队则为75%，而对大型为100%。可怕的是，它吐出的酸液会使敌人的攻击间隔增加八分之一，而且这种效果是可以累积的，最多可以累积九次。想象这时敌人的攻击间隔会有多长吧。此外，每粘上一次酸液，敌人受到攻击时还会受到1的额外伤害。酸液的另一个用途是使被击中部队不能隐形。可惜的是吞噬者不可以对地攻击。
