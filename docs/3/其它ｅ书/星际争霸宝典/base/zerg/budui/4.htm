简介
Zerg的基本空中部队。飞龙的攻击有反弹效果，当进攻一群敌人时，第一下攻击有9的攻击力，然后就会反弹。第一次反弹有1/3的攻击力，第二次则有1/9的攻击力。只能反弹两次。由于飞龙是小型部队，在敌人空军没出现前，口水龙和巨型机器人等防空部队只能对它造成50%的伤害，要充分利用这一优点。由于一群飞龙会重叠在一起，所以对付区域攻击的部队如海盗船或瓦尔基里护卫舰时会比较吃亏。另外，当你可以升级成空中卫士或吞噬者时，最好也要留一些飞龙作为护卫甚至诱饵。这样可以更好保护高级部队。
