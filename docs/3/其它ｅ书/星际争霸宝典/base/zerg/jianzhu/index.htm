Zerg建筑详细资料：
Hatchery（孵化场）
  进化项目：

挖地
花费资源：300
建造时间：120
生命值：1250
装甲类型：重
装甲：1
建造快捷键：H
基础建筑，雄蜂采来的矿和天然气都要送回这里。Zerg部队都由在这里产生的幼虫进化。还会产生紫色的有机物（Zerg的其他建筑都要在这种有机物上建造），但自己本身并不要求在这种有机物上建造。。

生命值：750
装甲类型：重
装甲：1
建造快捷键：E
