简介
特工的攻击方式为震荡，对小型部队有100%的攻击力，而对中型只有50%，对大型只有25%。由于特工攻击力不高，攻击间隔又长，不宜作为作战兵种。好好运用它的特殊能力吧。移植眼是一定要研究的，对放核弹帮助很大。
