简介
坦克的攻击方式为爆炸，对小型部队只有50%的攻击力，对中型部队则为75%，而对大型为100%。这种攻击方式可以攻击一片范围，不过会炸到自己的部队。围攻技术是一定要研究的科技，不过要注意的是坦克变形后是不能装上运输船的。
