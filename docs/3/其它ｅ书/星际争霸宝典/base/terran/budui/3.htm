简介
火焰兵的攻击方式为震荡，对小型部队有100%的攻击力，而对中型只有50%，对大型只有25%。所以最好用火焰兵去对付叉叉兵或小狗等小型部队，如果用来对付口水龙或龙骑士等大中型部队就不爽了。将火焰兵放到碉堡中是对付叉叉兵和小狗的最好方法。
