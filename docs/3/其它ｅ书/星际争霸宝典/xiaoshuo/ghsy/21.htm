光辉岁月（21）
作者：新唐

　

第二十一章 尼米兹星港

离海德拉行星大约200万公里的一片虚空中, 有一个巨大的人造钢铁平台, 这个大约300平方公里的平台上除了一个小区中有一些封闭的屋子外, 其他地方就是一览无遗的用便宜的钢结构框架制成的平面. 这块巨大的平面上有一条条的星船停泊长轨, 以及一些折叠起来的用来在无重力状态下起重卸货用的云梯. 这个人造平台就是著名的尼米兹虫洞东出口星际中转港.

星际船只在这个港口出发可以通过尼米兹虫洞瞬间在位于特伦斯星系的西出口平台溢出, 进入特伦斯的联邦军大型军事基地, 反过来, 海德拉行星所在的星域也可以在有事情的时候很快得到特伦斯基地的支持, 特伦斯基地是地球联邦军事委员会所委派维持东银河地区的重要军港, 连Protoss人也干脆就把人类这个种族称做Tarrans.

平日里这个冷清的中转港除了每天一班的金矿船经过外, 大部分时间都是很空闲的, 但自从两天前驻守这一星域的特伦斯301游击师团遭到Protoss袭击后溃退下来, 平静的小港突然变得热闹非凡, 整个港口周围充满了大大小小的战舰和战机, 各个战舰上的灯光在黑暗的太空中星星点点地闪烁着, 将真正的星星们的光芒全遮盖了

在平台左边的泊位上, 熙熙攘攘的SCV机器人在光滑的金属地板上到处奔跑进行着修理工作, 大批在前次战斗中受损的战舰和飞机有条不紊地鱼贯入港, 进行修复和零件更换工作. 而在右边的泊位上, 大量的难民船和各种民用飞行器则乱哄哄地挤成一堆, 争先恐后地想抢先排队进入虫洞, 尽快逃入安全的特伦斯星系, 争斗时有发生, 甚至有个别民用船之间因争吵而互相开火, 仅仅是今天就有三艘民用逃生船被这样击毁, 37人死于非命.

这种战败溃逃的沮丧场面让所有的联邦军人都感到了脸红, 他们无法去责怪这些仓惶逃亡的平民, 谁都有权珍惜自己的生命, 你又该说什么好呢, 很多军官站在战舰窗前看着看着就流下了眼泪, 有些人实在看不下去了, 就干脆走到另一边的窗口去看增援的战舰进港的情况.

中转港的下方有个仅供军事用途的紧急出入口, 此刻来自特伦斯星系的地球联邦第二星际远征舰队和铱星特遣舰队的大型战舰群正在一艘一艘地在虚空中凭空出现, 然后慢慢滑向一边集结. 很多301游击师团的军官看到不断出现的大型战列舰时, 都禁不住高兴地大声喝起彩来, 毕竟这个星系已经很久没看到过这么多的大型战舰了.

对于已经很长时间没有经过大型战争的人类联邦来说, 来自黑星帝国的Protoss大军让每个军人既兴奋又害怕, 从已经公开的资料来看, Protoss的军事力量和科技水平超过了历史上人类所面对过的所有异星生物, 而海德拉行星的前哨战也显示了异星人的战术水平相当不俗. 不过作为军人, 他们的生命他们的价值不正是在战争中获得永生的吗?

地球联邦第二星际远征舰队的旗舰伊斯坦布号超级战列舰进港了. 缓慢而巨大的舰身无声地滑过大队巨舰让出来的中央通道, 被星光照耀下投射下来的阴影象一副巨大的被子慢慢铺过一边几个悬停的战列舰,使得那些本来已经黝黑的舰身变得更加朦胧. 各舰的舰长纷纷站到各自舰桥上的巨大玻璃窗前向旗舰举手致敬礼.

伊斯坦布号火控系统操作员尼克上校完成了通过时空隧道后的设备监测工作, 松弛地靠在宽大的真皮座椅上, 开始如常一样默默地观看着周围的一切. 在非战斗时间里, 他的工作是最少的, 这个时候舰上其他的各组船员仍在繁忙地报告或跑动着, 身后第二舰队总指挥台上的第二远征舰队司令哈里森元帅正在仔细地点算着兵力, 他的几个参谋和副将则手忙脚乱地指挥布置着舰队队形. 窗外的一部分巨舰开始移动起来, 转到各自的指定的战位上.

尼克上校望着窗外由战舰群的舰宽指示灯形成的数列长长的象机场跑道灯一样的壮观景象, 这些无穷无尽的长灯一直延伸到目不可及的地方, 与背景上的亿万星辰融为一体, 展示着人类巨大的武装威慑力量, 以及人类与宇宙抗争, 抵御异星侵略者的强烈愿望. 尼克下意识地摸抚着操作盘上熟悉的按钮和操纵杆, 清晰的金属质感使他心中涌起一股自豪感, 他希望能够尽快使用这些威力强大的杀戮武器, 看着不可一世的异星人在他的炮口下化作灿烂的火光.

身穿笔挺的上校制服的尼克看上去象个文静的书生, 事实上他也不是一个喜欢血腥的人, 他在地球上加利福尼亚海边做救生员的时候, 就经常主动要求值夜班, 原因就是喜欢没事的时候仰望深邃宏大的星空, 直到有一次去佛罗里达人类宇航史遗址的博物馆参观后, 才下了决心加入了联邦军队. 他其实是个浪漫的人.

舰队出发命令的下达, 令尼克集中起精神. 一眼看去, 窗外的各式舰只的尾部纷纷冒起了红光, 那是中子引擎的开动引起的. 尼克很佩服舰队的队列指挥官, 同速前进的数千只舰船让人有种从未移动的感觉, 除非你正好看到后方迅速远离的尼米兹星港.

尼克看到火红色的海德拉行星在前方越变越大的时候, 已经是三天以后了. 前方的驱逐舰群已经和异星人的战舰接上了火, 远处空对空导弹的硝烟和不时亮起的舰船爆炸的火光让人真切地感觉到战争正在眼皮底下发生. 尼克注意到己方的舰队队形开始改变, 所有舰只开始呈散兵线慢慢排开, 但向前急行的速度并没有减缓.

几年来的从军生涯使尼克知道身后一头白发的哈里森五星上将是个身先士卒的猛将, 伊斯坦布号旗舰从来都不会远远地缩在后面. 两个小时后, 尼克就发现自己的战列舰已经开到了战斗第一线, 他的全副身心集中到眼前的控制台上. 现在就是火控系统操作员显示价值的时候了. Now the show time! 尼克觉得心里很安宁, 控制台上的复杂按钮已经成了他身体的一部分.

拥有24门威力强大的万向ATS激光炮的伊斯坦布号超级战列舰第一次齐射就将正前方急速穿行的三艘异星人的海盗船Corsair打得粉碎, 爆炸引起的绚烂是蔚蓝的, 蓝得有点炽白, 周围的星光顿时黯然失色, 伊斯坦布号的指挥舱也被这种毁灭的焰火照得通亮. 尼克没有丝毫的停顿, 快速旋转的自动目标跟踪仪指出了下一批目标, 他熟练地按下射击钮, 24道耀眼的激光束形成的瞬间弹道就像催命的死神再次无情地扑向敌舰.

火控操作员尼克和导航员克利斯多的出色配合加上强大的武力和坚硬的盔甲, 使得这艘本来应该呆在后方指挥的旗舰成了最前线的空中堡垒, 四周的舰只在总司令的鼓舞下奋勇向前, 整个天空中几乎全是战列舰凶猛的激光炮射线和远程空对空导弹的烟幕, 为数不多的异星人战舰群被迫后退, 潜入低空依托地面武装抵抗.

尼克和他的同事们被胜利的激情感染着, 兴奋地看着异星人的战舰逐渐消失在云层中, 从云层的空隙里, 可以清晰地看到地面上满地离子炮塔. 这些东西对于眼前强大的人类武装来说并不构成很大的恐惧. 尼克和其他人一起在座位上焦急地等待着哈里森五星上将发出追击的命令. 他们并不知道哈里森心中的恐惧.

这里的战斗刚开始就让哈里森感觉到有点不对劲, 对手的实力太弱会让每一个出色的将领都产生本能的恐惧, 本该出现的异星主力舰队并没有露面. 就在这时从尼米兹星港传来了铱星特遣舰队受攻击的消息. 一边读着那份不长的紧急战报, 哈里森厚厚的嘴唇开始哆嗦, 他那黑人特有的黝黑而圆胖的脸开始涨得通红.
