光辉岁月（16）
作者：新唐

第十六章 复仇

前地球联邦军少将比得洛夫斯基的葬礼在离克达姆行星驻军总部二百公里处的一个山谷里举行. 出殡的那天, 灰暗的天空淅淅沥沥地下着毛毛小雨, 春天有点冷冽的寒风吹佛着棺木上覆盖着的国旗, 鲜红色的国旗边角随着忽大忽小的风势频繁地翻动着.

程仁中将随着其余两百多名高级军官在悠长凄凉的军乐声中庄严地向已故的同僚行军礼致敬, 这是他恢复军职以后参加的第一次活动. 军乐声停止后, 行礼完毕. 程仁放下右手, 低沉着眉毛, 聆听着军中牧师为老将军作最后的祷告. 他从眼角里看到浑身戴着黑纱, 只露出苍白的脸庞的琪琪那娃中校, 比得洛夫斯基的独生女儿, 也是程仁中学时的同班同学.

琪琪那娃的脸上除了已经干枯的泪痕和新沾上的一些雨丝外, 没有任何表情, 她只是木然地注视着被修复的父亲的遗体和面容, 脑子里一片空白. 牧师毫无韵律地读着圣经的语调被隔离在她的听觉以外, 她只听见毛毛细雨落地时发出的轻微的沙沙声, 她在心里默默地数着雨点.

漫长的祷告结束了, 军乐声再度响起, 雄壮地奏起军歌<<我的征途是星辰大海>>, 所有的军官用整齐划一的动作双手手将胸前的军帽端正地戴上, 然后举起右手行出征礼. 其余人士包括家属也都站了起来, 肃穆而悲伤地看着棺木下方的火箭点火升空, 在两架战斗机的护送下消失在谜蒙的雨幕中.

这是联邦高级军官的星葬仪式, 比得洛夫斯基的棺木飞出大气层, 在两架战斗机护送10万公里后, 将继续孤独地在无声的星空中飞行三年, 最后飞向半人马星座附近的一个黑洞, 被宇宙里最强大的力量所吞噬, 归于无形.

葬礼结束后, 一些和死者家属相熟的人走上前去安慰他们. 程仁犹豫着是否要上去和琪琪那娃打个招呼. 其实这已经是一个军中公开的秘密了, 他的弟弟正是害死比得洛夫斯基的凶手. 鉴于比得洛夫斯基的死去和资料的丢失, 联邦政府暂时没有办法对程义采取更进一步的行动.

联邦法律严格保护私人的财产并尊重隐私权. 程义购买的那个岛屿也包括陆地上方100公里纵深的领空以及岛的周围300公里的海域, 业主可以拒绝任何人的访问. 而如果使用武力硬闯而在私人领地发生战斗的话, 闯入者被伤害是活该且会反被业主起诉, 而守卫私人领地的业主方有人伤亡的话, 则闯入者立刻就犯下了谋杀罪.

由于联邦政府目前没有任何证据可以申请对程义的趣香园的搜查令, 使得法律上和军事上的行动这两条路被堵死, 程义暂时处在安全的地位. 对趣香园的卫星侦察也不能发现任何不合法的地方, 而程义的私人战斗机群也从这一事件开始日夜不停地在空中广泛侦察搜索任何非法闯入者的踪迹.

程仁最终还是走上去和老同学琪琪那娃打了个招呼. 琪琪那娃无神的眼睛看到程仁的时候开始露出恶毒的怨恨, 令程仁禁不住打了个冷战. 他只好打起精神厚着脸皮胡乱说了几句安慰的话就慌慌张张转身企图溜开. 琪琪那娃的戴着黑丝绒手套的手抓住了他的手, 程仁的心一下子凉到了冰点, 他预感到只要他一回头, 无数人包括他一家的命运就会从此改变.

琪琪那娃深知程仁的性格, 她在读书的时候就开始暗恋这个沉默的东方男孩, 数年的校园生涯, 她无时不刻地在观察着程仁, 有时候甚至不顾少女的羞涩, 主动约程仁在坐夏夜的草坪上看星星. 从他仰望星空时的不平常的神情到看完星空后出现的深邃的眼神, 她就知道这个看似性格软弱的男孩有着广阔的胸怀.

当程仁生硬而礼貌的身躯慢慢转过来的时候, 琪琪那娃又一次看到了那久违了的深邃的眼神. 她低下了头, 用几乎听不见的轻声说了一句: "明天陪我去趣香园." 程仁像是已经早已想好了似的立刻就点了点头. 他离开黑纱披身的琪琪那娃的时候, 暗暗擦去一滴流出眼角的泪水.

第二天清晨, 一架没有武装的小型民用飞机从克达姆行星驻军基地起飞, 经过四个小时的航程, 进入趣香园的控制海域, 受到两架巡逻的战斗机拦截后, 程仁要求访问的信息传到岛上. 仍然睡得迷迷糊糊的程义被这个消息惊的从床上跳起来, 而更令他惊讶的是, 这架民用飞机在发出信息后不顾拦截, 继续向岛上的停机坪飞过来并开始下降.

程义纵有天大的胆子也不敢下令向他的哥哥开火, 只好穿上衣服带着忐忑不安的心情去迎接程仁. 当他看到随机下来的琪琪那娃和凯丽的时候, 心里感觉到恐怕今天不会有什么愉快的事情. 他勉强装出高兴的心情领着不怀好意的客人们进了一栋豪华的红顶小屋里.

程义首先以祝贺的语气打破沉默, 他们兄弟俩已经有一年多没有见面了, 上一次见面是程仁被人从褐灵星上抬着回家的时候, 现在程义看到哥哥又回复了以前雄赳赳的军官形象, 他是真心的感到高兴, 毕竟他的家里只有他和哥哥以及年老的父亲三个正式成员.

说话的同时他也暧昧地扫了一眼程仁身边的凯丽, 不过这段时间和美女们泡得太多, 凯丽的美色在他的眼里已经根本不值一提了. 程仁也真诚地笑着寒暄, 他注意到程义胖了很多, 脸上的肥肉一圈一圈的, 明显的眼袋出现了下垂, 可能是传说中的淫荡生活所至吧, 他对他的弟弟可是太了解了. 这个时候他脑海里出现了凯丽提醒的危险信号, 不过他返回去的信息是让凯丽压制情绪.

程义开始转过头去问候琪琪那娃, 一边虚伪地对比得洛夫斯基的去世表示深切的同情, 一边表示愿意出力帮助调查事件的真相. 程义的是口才非常了得的, 无比诚挚的表情让旁观者都觉得动容, 感到了这位英气俊俏才华四溢的年轻人毫无保留的爱心. 不过这里只有程仁知道, 琪琪那娃是当年学校里话剧团的主角演员.

琪琪那娃露出楚楚可怜的样子掩面抽泣起来, 作为主人, 程义连忙非常大度地走上前去, 递上手巾以示安慰, 刚才的扫描显示来访的客人们身上没有携带任何武器, 这导致了他犯下了一个非常低级的错误. 变故就在这时发生了. 外表孱弱的琪琪那娃是受过严格军事训练的中级军官, 徒手对付程义这种手无镈鸡之力的花花公子绰绰有余. 她突然反手扭住程义的手腕, 一下子就将程义压制到沙发上, 手里亮出一根比钛金属还硬的有机碳酸硅头簪逼住程义的太阳穴.

在场所有的人都被吓住了, 当今的医学技术可以修复几乎人体的任何部位, 却无法修复人脑的损伤, 琪琪那娃的头簪如果刺了进去, 也就意味着程义的死亡. 几个冲进来的女卫兵手持步枪不知所措的呆在原地, 程义的眼光里露出临死前的恐惧. 他看到哥哥程仁正紧紧地握住凯丽蠢蠢欲动的手, 慢慢地按照琪琪那娃的威胁指示和其他卫兵一道退到二十米外.

琪琪那娃的眼睛里闪动着复仇的火焰. 她喝令程义交出比得洛夫斯基失落的资料和他杀害比得洛夫斯基的战斗影像. 比得洛夫斯基收集的对趣香园不利的资料已经被销毁, 而那场暗杀战斗的影像资料却因技术原因被保留在岛上的资料库里. 可怜的程义是第一次遭到这种近身的暴力恐吓, 他早已吓破了胆, 并不知道琪琪那娃实际上是在瞎蒙.

那场可耻的战斗资料被调了出来, 当琪琪那娃看到父亲的座架被猛烈的导弹击得粉碎而冒出大量的火光的时候, 早已泪流满面, 几乎不能自己. 程义的信息系统连上了星际互连网, 这场暗杀行动的全部资料在网上公开地曝了光, 并传送了一份给克达姆行星总督府.

当一切都做完了的时候, 程义在持续的寂静中感到了死亡的来临. 琪琪那娃的胸脯急剧的上下起伏着, 将愤怒的双眼紧紧盯在程义的双眼前方5厘米处, 她畅快地看着仇人恐惧的眸子, 她准备亲自结束仇人的性命. 随着一声凄厉的惨叫, 程仁的眼前一黑, 大片热腾腾的鲜血像喷泉一样喷涌向空中, 将他的浑身染得透红, 一滴一滴顺着他的衣角流向地上花斑斑的地毯上, 圆形的血渍迅速地扩大.
