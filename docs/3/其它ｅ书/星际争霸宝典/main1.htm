关于《星际争霸宝典》

黑瞳工作室 WONK

《星际争霸宝典》是利用业余时间整理编辑的一部关于目前最流行的即时战略游戏《星际争霸》及其资料片《母巢之战》作战资料的指导手册。编这部手册的目的一方面是因为本工作室的成员水平不高而兴趣却蛮大，为了便于查找相关资料，于是就有了编写这部指导手册的冲动。其二，我们现在能够查阅到最新、最全的星际资料是在网上，但为了一些很小的问题却又要上网找遍网站，应该不是一个明智的选择。有了这部指导手册，你用一张软盘就可以装着它南征北战了。而且我们用了CHM格式进行编译，使查阅相对来说轻松得多。以下是这部手册的基本结构和说明。

本手册包含五个部分：

一、星际争霸传说篇：这部分收录了星际争霸单人游戏的剧情攻略和国内外以星际争霸为题材撰写的精彩小说。

二、星际争霸基础篇：详尽的星际争霸各种族的建筑、作战单位、魔法科技的资料。另外还有基本操作教学、地图编辑器使用祥解、投机取巧等。通过它，你可以从细微的地方了解你控制的作战单位，取长补短，必定有所收获。

三、星际争霸战术篇：星际争霸的战术千变万化，不可能有永远不败的战术。但通过掌握一些基本的战术思想对初学者和高手来说都是必须的。

四、星际争霸战报篇：战报是我们学习先进的作战思想、借鉴高手战斗经验、迅速提高水平的有效途径。好的战报会使人回味无穷。我们选择了国内和国外水平较高的几十场比赛供大家学习借鉴。在编辑整理中我们考虑到容量的问题，删除了战报中的图片。但好的战报你只需通过文字一样可以了解到其中的精华。

五、星际争霸联网篇：这部分是逐步教会大家如何上网战斗以及网上战斗的各方面应该掌握的东东。还介绍了国内的一些战队及基本情况。
