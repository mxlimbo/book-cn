毕业生就业大盘趋好　研究生入京数量放开

　　据了解，2000年全国普通高等学校共有北京生源本、专科毕业生2万余人，比1999年增加500多人。北京生源毕业生中，本科层次的占65．4％，专科层次占34．6％，农科1％，林科0．3％，医科6．8％，师范7．7％，文科10．1％，理科10％，财经20．7％，政法3．3％，体育0．6％，艺术2．8％。预计2000年高新技术企业对毕业生的需求量会继续保持增长势头；随着贯彻落实扩大内需的方针政策，国家综合运用积极的财政政策、货币及其他宏观调控手段刺激拉动经济的增长，以及一批项目开工建设，都会间接拉动相关单位对毕业生的需求。与此同时，北京市用人单位，特别是高新技术企业对毕业生的质量和素质也会有更高要求，毕业生就业竞争随之会更加激烈。

　　具体到对毕业生的就业政策，在引进接收非北京生源毕业生方面，北京将加大力度，在不降低接收条件的前提下，对于毕业研究生和获得省部级优秀毕业生称号的本科生进京工作将不作数量限制，对于重点国有企业、高新技术企业以及远郊区县引进接收非北京生源毕业生将采取倾斜政策。同时，为适应市场经济发展的需要、适应劳动人事制度改革的需要，为毕业生和用人单位创造一个公开、平等、竞争、择优的环境，北京市人事局将出台和调整相关政策，加强和完善对毕业生就业工作的管理。
