2001年考研竞争将更激烈

　　进入90年代以来，特别是自1995年开始，全国报考硕士研究生的人数逐年递增。1994年全国有11.2万人报考，1995年增加近40%，达到15.5万人；1996年继续攀升32%，达20.4万人；1997年，24.2万人；1998年，27.4万人；1999年，31.9万人，2000年全国硕士研究生报考总人数再创新高，达到39.2万人。与此相应的，全国硕士研究生招生人数也在逐年增加，自1995年至2000年，全国招生录取人数分别为4.3万、4.6万、5.1万、5.8万、6.5万和8.5万（注：此处"招生"包括国家计划、定向、自筹经费、委培等）。

　　由此可以看出，报考人数与招生人数一直保持同向增长，但相比报考人数的扶摇直上，招生规模的增加就显得比较有限。这直接导致了考录比由1995年的3.6：1下降为2000年的4.6：1。虽然2000年扩招幅度较大（30%），但考虑到新增部分中只有20%属于国家公费，且免试推荐名额等也在增加，因此，考研竞争越来越激烈。

　　据有关部门预测，2001年考研报考人数还将保持相当的增幅，招生人数继续扩大，总体竞争强度将进一步加大。其主要依据是：

　●2001年全国应届本科毕业生（97级本科生）总量比2000年还将有较大增加，就业压力进一步增强，找一个满意的工作越来越难。由于国有企业调整，政府机构改革分流等因素的直接影响，社会能够提供的就业机会相对减少，而高校本科毕业生又在逐年增加。高校毕业生供需比由1995年的供不应求，发展到1999年供过于求的局面。北京地区，这表现为毕业生待分配和回原籍参与二次分配人数的急剧上升。1993年，北京地区待分配人数为300，二次分配人数为1000，两者合计不过1300；而到1999年，找不到工作的总人数已上升到近9000。（注："待分配"指分配结束后仍未找到工作；"二次分配"指外地生源在学校毕分办未能落实就业，回原籍参加再次分配。两者之和即为毕业分配结束后全部未能就业的人数。）

　●社会总体人才需求的学历层次越来越高。特别是政府机关、大型国有企业、知名外企等，学历要求起码是硕士。在北京、上海、深圳等大城市，甚至出现了"人才高消费"的倾向。要想找到满意的工作，获得满意的薪水，高学历是必备的硬件。

　　因此，2001年考研将持续升温，竞争更为激烈。
