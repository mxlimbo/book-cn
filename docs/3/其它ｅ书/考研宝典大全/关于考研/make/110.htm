2001年读研是否自费

　　我国研究生考试及招生将有重大改革，研究生免费读研将成为历史，今后的趋势是自费读研。

　　2000年适当扩大推荐免试工作，明年教育部再给予全国50所高校推荐权，增加研究生免试的比例。同时招生管理办法也将进行改革。推行两段考试：考生报名后先参加国家统考，取得资格后再参加专业考试。

　　另：据说自费后平均每年交5000左右。

　　现在各个高校扩招部分一般交费额为：7000-9000元/年，三年共24000-30000元，一次性交清。

　　估计今后自费应该参照这个标准办理

　　但据我估计，今年应该是过渡阶段，不同学校将会有一些自己的相应规定，收费会成为主流，但不会是全部学校。而据周远清部长所说国家会有所考虑，并且至今国家未出台收费的通知。应该还是会有许多不收费的招生专业可选择的。

　　计划内招考的学生，不会收费。委培生和自筹经费生，仍然会收费。但各校以专业情况会采取不同的收费办法。有的学校拿大头，导师出一点、学校出一点、个人出一点。经济管理类的收费略多一些。
