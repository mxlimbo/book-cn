我国将试办公共管理硕士(MPA)专业学位教育

　　华声报讯中国即将试办公共管理硕士(MPA)专业学位教育。复旦、上海交大、华东师大和同济大学日前被国务院学位委员会批准为首批试点单位。

　　据《文汇报》报道，公共管理硕士是中国新设置的培养高层次应用型人才的专业学位，其目标是为政府部门和非政府公共机构培养具有现代公共管理理论和公共政策素养、掌握先进分析方法及技术、精通某一具体政策领域的专业化管理者和政策分析者。公共管理硕士专业学位的招生对象主要是为国民教育序列大学本科毕业获得学士学位、有4年以上实际工作经历的政府部门及非政府公共机构的工作人员，入学考试采取全国联考、统一命题、统一阅卷、统一录取办法。试点期间主要采取半脱产和在职兼读的教学方式，学习年限一般为4年。
