厦门大学1998年研究生入学考试试题
财政综合卷

　　招生专业：财政学　　考试课程：综合考试

　　研究方向：各方向

一、填空题，每题1分，共25分

1.税收伴随着生产力的进步、______的丰富、生产资料所有制的更替和______的变化而发展演变。

2.我国现阶段的税负转嫁有两种形态，一种是与固定或官方价格相联系的______的税负转嫁，另一种是与市场自由价格相联系的______的税负转嫁。

3.一般纳税人因销售退回或折让而退还给购买方的增值税额，应从当期______中扣除，因进货退出或折让而收回的增值税额，应从当期的______扣除。

4.我国企业所得税的应纳税所得额是指纳税人每一纳税年度的______减去______后的余额。

5.纳税人销售不动产，应当向______主管税务机关申报缴纳营业税。

6.资金的时间价值是企业______和______必须考虑的重要因素和标准。

7.短期资金需要量的预测方法主要有______法、______法、______法和______法；其中最具灵活性的一种预测方法是______法，它适用于______的企业。

8.货币执行支付手段职能，使得商品流通过程发生了变化，______关系转化为______关系，______活动由此产生。

9.付息的产生是以货币资本______和______的分离为条件。

10.开发银行是指专门为经济开发提供______贷款的银行。

11.货币的交易需求产生于收入与支出的______，而预防性需求则产生于收入与支出的______。

二、名词解释，每题2.5分，共15分

1.税收效率

2.查验征收

3.风险报酬率

4.资金成本

5.存款保险制度

6.期权交易

三、简答题，每题5分，共30分

1.什么叫税收公平？

2.什么是混合销售？对混合销售征税有何规定？

3.企业采用债券筹资方式有何利弊？

4.为了实现目标利润，企业应采取的一般途径有哪些？

5.试比较商业银行和商人银行的异同。

6.存款货币的创造需要具备哪些条件？为什么？试简述其基本原理。

四、计算题，第一题10分，其余每题5分，共30分

1．某企业在投资决策时，拟采用一次性付款的方式购置一台新设备，价值100，000元。按照直线法计提折旧，使用年限为10年，报废时无残值。该设备期望每年能产生12，000元的税前利润，假设所得税率为30%。

（1）计算该投资项目的回收期（税后）（2）计算该投资项目的会计报酬率（税后）

2．某股份有限公司发行3年期公司债券，面值为1000元，按6%利率每年付给投资人利息60元，如果市场利率为5%，则该公司债券的市价可能是多少？

3．某公司以价值50，000元的商品为抵押，向某商业银行申请2个月期的抵押贷款。银行按商品价值的80%提供贷款，年利率为8%，采取利息先付方式。求利息额和借款人实得的贷款额。

4．一般纳税人连欣百货商场某纳税期零售收入162200元，其中食用植物油等农产品销售收入45200元；本纳税期可抵扣的进项税额为17400元，请计算该纳税人本纳税期应纳增值税税额。

5．某纳税人1996年度月工资2000元，为某加工厂提供业务咨询取得收入5000元，因作品出版取得稿酬收入2000元，在甲国取得利息收入折合人民币4000元，在国外已缴纳预提税900元，在乙国因转让专利权取得收入6000元，已缴纳个人所得税900元，计算其全年应向中国政府交纳的个人所得税税额。
