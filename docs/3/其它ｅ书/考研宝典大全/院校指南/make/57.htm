重庆大学

博士研究生招生简章

一、培养目标：培养德、智、体全面发展，在本门学科上掌握坚实宽广的基础理论和系统深入的专门知识，具有独立从事科学研究工作的能力，在科学和专门技术上做出创造性成果的高级科学专门人才。

二、学习年限：脱产生３年，在职生可再延长１年。

三、报考条件：

１、拥护中国共产党的领导，愿意为社会主义现代化建设服务，品德良好，遵纪守法；

２、已获得硕士学位的在职人员，应届硕士毕业生（最迟须在入学前取得硕士学位）；获得学士学位后连续工作满六年（从获得学士学位到博士生入学之日），并达到与硕士毕业生同等学力的人员；

３、身体健康状况符合规定的体检标准；

４、报考国家计划内博士生的年龄不超过４５周岁，报考委托培养或自筹经费的可适当放宽；

５、有两名与报考学科有关的副教授（或相当职称）以上的专家推荐（报名时发专家推荐书）。

６、有考生人事档案所在单位人事部门开具的同意报考的证明。

现役军人报考博士生，按中国人民解放军总政治部的有关规定办理。

四、报名时间、地点：

１、报名时间：2000年1月3日～2000年1月31日

２、报名地点：重庆市重庆大学研究生招生办公室，外地考生可以函报。

五、报名手续：

１、凡符合报考条件者，持所在单位人事部门出具的同意报考的介绍信，应届硕士生持所在培养单位研究生管理部门出具的介绍信，原属定向或委培的应届硕士生还应持原定向或委培单位人事部门出具的注明在读博士期间是否继续定向或委培的同意报考的介绍信，交验工作证（学生证）、身份证、学位证书（原件，函报者附复印件亦可，考试时带原件验证），交纳报名考试费１００元（含初试及复试费），同一底片一寸免冠近照三张，即可向重庆大学研究生招生办公室报名，领取空白报名表格一套（报考攻读博士学位研究生登记表一份；专家推荐书二份；体格检查表一份；政治审查表一份；准考证一份；报考时请注明报考专业、导师及选考科目名称。

２、考生须按所发的表格上的要求及时填写，并交所在单位签署意见，定向或委培博士生须在报名登记表中单位意见一栏内注明。加盖人事部门公章后，于2000年3月6日前将下列材料寄回我办。

(1).报考攻读博士学位研究生登记表；2).专家推荐书（二份）；3).硕士课程学习成绩单（复印件亦可，但须盖公章）硕士学位论文全文和评议书；4).硕士学位证书复印件（应届毕业硕士生必须在入学前补交）；5).体格检查表（经县级以上医院检查合格有效）；6).政治审查表；6).准考证（加盖我校钢印才生效）。

*同等学力考生不交3)、4)两项材料，但应开列已经学习过的硕士课程及由有硕士学位授予权的高等学校出具的不少于六门所学硕士学位课程的成绩单；送交在公开发行的刊物上发表的与硕士学位论文水平相当的学术论文全文（附刊载刊物原件）。

３、经审查合格的考生，由我校研招办寄发准考证。　

六、考　试

　　１、考试方式：考试分初试和复试，初试考试科目为政治理论课（应届毕业硕士生和获得硕士学位者，可以申请免试）；外国语（含听力）和两门业务课，每门考试时间为３小时，复试科目为外国语及专业课，同等学力者除参加以上考试外，还需加试两门与所报考的学科、专业有关的硕士学位主干课程（笔试）。

　　２、考试地点：重庆大学，具体地点待发准考证时通知。

　　３、考试时间：2000年4月15日～2000年4月17日（17日复试）。

七、录　取：在初试、复试成绩均合格的情况下，坚持“德、智、体全面衡量，择优录取，保证质量”的原则。

八、报考费用：考生报考的一切费用由本人自理。经济确有困难者可向所在单位申请补助。

九、学习期间待遇：按国家教委和财政部有关文件的规定办理。

十、专业目录中所公布的招生人数含国家招生计划、委托培养、自筹经费招生计划。
