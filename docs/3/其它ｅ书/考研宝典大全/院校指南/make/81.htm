国防科技大学

2001年在职人员单独考试的规定

　　为进一步扩大招收有实践经验的在职人员，满足用人单位对高层次专门人才的需求，我校组织单独考试，欢迎符合条件的在职人员报考。具体规定如下：

一、报考条件

　　除《考生须知》中规定的报考条件外，还须满足：

1、1997年9月1日前本科毕业，在本专业或相近专业从事技术或技术管理工作，业务优秀，已发表过研究论文（技术报告）或已经是业务骨干的在职人员。

2、有两名与报考学科、专业相关的高级技术职务专家(副教授或高工以上)的书面推荐。推荐书中应包括被推荐人的政治思想、道德品质、工作表现以及工作能力和成绩等方面的内容，并加盖专家所在单位公章。

二、录取类别

　　对参加单独考试的考生进行录取时，军人考生可录取为计划内定向或计划外委托培养硕士生，地方考生只能录取为计划外委托培养硕士生。

三、考试科目

　　单独考试的考试科目为五门（均为笔试）：

（1）英语。初试时仅考基础英语，专业英语在复试中考核。科目代码为111；

（2）政治理论。70%马克思主义哲学，30%时事政策。科目代码为210；

（3）高等数学。要求与樊映川编高等数学内容相当，不含线性代数和概率论。科目代码为310。

　　另二门为各学科、专业所要求的专业课考试科目（4）（5）。

四、报名和考试

1、报名时间：2000年10月20日-11月10日

　　考试时间：与全国统考时间一致。

2、报名地点和手续：凡报考单独考试的在职人员，需持所在工作单位介绍信、本人工作证(身份证)复印件、大学本科毕业证原件及其复印件等证明材料到我校研招处报名，领取整套报名表格。由于采用计算机自动阅读考生报名卡，因此，一般不接收函报，外地考生可委托别人办理，特别困难的也可由我处代办。

　　所有报考材料务必于11月20日前寄送我处以便送湖南省教育厅考试中心审查。报名登记表单位意见栏内请注明“为╳╳单位定向培养或委托培养”。审查后符合参加单独考试条件者发给单独考试准考证，参加单独考试，不符合参加单独考试条件但符合参加全国统考者发统考准考证，参加全国统一考试。

3、考试地点：所有准许参加单独考试的考生均在我校考试。考试前一天考生到我校研招处报到、了解有关考试安排。
