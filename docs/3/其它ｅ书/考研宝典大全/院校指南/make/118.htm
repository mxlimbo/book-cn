安徽大学

硕士研究生招生简章

一、学校概况：　　

　　安徽大学是安徽省属重点综合大学，是国家面向21世纪重点建设的百所大学之一。学校位于全国四大科教基地之一的合肥市，校本部占地800亩，校舍建筑面积29万多平方米。校园景色秀丽、环境优美，是合肥市“安全文明校园”和全国绿化400佳单位之一。

　　安徽大学创建于1928年，重建于1958年。经过40多年艰苦创业，学校已成为一所兼具文、理、工、哲、经、法、史、管等多种学科、在国内外具有一定影响的综合性大学。

二、2001年我校29个学科、专业计划招收攻读硕士学位研究生210名；各专业另可招收委托培养和自筹经费硕士生80名。

三、我校法律史和电路与系统两专业具有单独考试权，考试科目名称与同专业的统考科目相同（英语、政治和三门业务课均由我校组织命题）。报名时间：2000年10月14日至11月14日，考生直接向我校研招办报名（外地考生可函报）；

四、同等学力考生报名时必须提交以下原始材料：　　
　　（1）大学英语四级证书，　

　　（2）以第一作者公开发表的学术论文（有CN刊号）；

五、我校备有2001年招收攻读硕士学位研究生专业目录单行本，公布指导教师名单和考试科目指定参考教材，考生可来信索取；

六、我校每年暑假期间（集中时间）及9月至12月（晚上和双休日）举办研究生考前辅导班，有意者可与我校研究生招生办公室联系；

七、为帮助考生复习迎考，我校提供历年考研试卷供参考，每套15元，款到即寄。购试题时请注明专业和研究方向。
