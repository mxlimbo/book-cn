西南交通大学

博士研究生招生简章

一、培养目标　培养德、智、体全面发展，在本门学科上掌握坚实宽广的基础理论和系统深入的专门知识，具有独立从事科学研究工作的能力，在科学和专门技术上能做出创造性成果的高级人才。

二、报考条件

１、拥护中国共产党的领导，品德良好，遵纪守法，决心为社会主义现代化建设服务。

２、已获得硕士学位的在职人员；应届硕士毕业生（最迟须在录取前能够取得硕士学位）。

３、身体健康，年龄一般不超过45岁。

４、有两名与报考学科有关的副教授（或相当职称）以上职称的专家推荐。

５、在校硕士生中少数政治思想好，硕士课程学习成绩优秀，修满学分，且通过阶段 性科研工作，显示其具有较强的科研能力和发展潜力，确有培养为博士生前途的，由本人 提出申请，导师推荐，经院（系、所）审核，学校主管部门批准后，可作为提前攻读博士 生报考。

三、报名、考试日期：

1999年第一次博士生入学考试安排如下：

报名时间:1998年9月20日至10月10日

考试时间:1998年10月24、25日

1999年第二次博士生入学考试安排如下:

报名时间:1999年4月20日至5月10日

考试时间:1999年5月22、23日

四、报名地点： 西南交通大学研究生招生办公室，邮政编码：610031 电话： ( 028 )7600296;联系人：张胜娥

五、报名手续：

１、考生须凭单位人事部门介绍信（介绍信上注明报考类别：定向、委培、 统分字样）和学位证书(复印件)，应届毕业的定向培养硕士生须出具原定向单位介绍信到我校报名（外地考生可函报，函报日期以邮戳日期为准），交报名考试费100元。报名时发给考生有关表格，考生按表格要求用钢笔填写，同时准备一寸正面免冠近照三张（分别贴在“ 报考登记表”、“准考证”和“体检表”的指定位置）。并加盖单位公章后按以下规定日期寄回所有报考材料： 第一次在1998年10月15日前;第二次在1999年5月15日前提交以下材料: （１）报考博士研究生登记表 （２）体格检查表（自行在县级以上医院检查） （３）专家推荐书（２份） （４）硕士课程学习成绩单；（５）硕士学位论文和评议书(复印件)， 应届毕业的硕士生必须在录取前补交硕士学 位证书(复印件)。

２、以上材料齐全并审查合格者发给《准考证》（考试前在研究生招办自取）。

六、考试地点： 西南交通大学

七、考试科目和方式： １、考试科目：外语、政治理论课、基础课、专业课。 ２、考试方式：闭卷笔试；基础课和专业课在报名时应注明所选考试科目名称。 ３、政治理论课。 应届毕业硕士生和获得硕士学位的研究生，在校期间考试成绩合格 者可以申请免试。

八、复试时间： 初试结束后在10天之内进行复试。

九、录取原则： 根据考试成绩，德、智、体全面衡量，择优录取的原则，在同等条件 下，注意录取“委托培养”的在职人员；对极少数确有发明创造或有较高学术建树的高级 职称的考生，采取优先录取的政策。

十、学习期间待遇： 毕业后回原单位工作的定向和委托培养的博士生不转工资、人事 关系，享受原工作单位福利待遇。非定向脱产博士生按国家有关规定发给普通奖学金和优 秀奖学金。

十一、毕业分配： 博士研究生毕业后，根据国家需要和学以致用的原则，由国家统一 分配（定向和委培生除外）。

十二、英语考前复习班： 为帮助考生复习，我校举办英语考前复习班， 第一次复习 时间：1998年10月13日至23日; 第二次复习时间:1999年5月11日至21日。复习地点在西南 交通大学内，复习费每生200元(含资料费),考生可在报考博士生时提出参加复习班申请。
