东南大学

2001年硕士生招生简章

一、我校2001年有有90个学科、专业(含医学三级学科)招收攻读硕士学位研究生。拟招收国家计划硕士生（非定向生、定向生）和委托培养、自筹经费硕士生1200名。本目录中的招生人数含上述各类招生数，并包含部分走读生。委托培养、自筹经费硕士生须按录取当年国家计划硕士生的国家拨款标准缴纳培养经费。

二、原南京铁道医学院已并入东南大学，成立东南大学医学院（院、系、所包括基础医学部、临床医学部、公共卫生学院）。

三、报考美术学、工商管理硕士（MBA）专业和参加单独考试的考生必须在我校报名、考试，外地考生可函报（需提前15天），并在东南大学招生主页（http://xsc.seu.edu.cn/zsb/）“网上报名”栏目注册。其他考生在当地省级招办指定的时间和地点报名、考试。

四、报考条件：符合下列条件者可以报名参加国家组织的全国统一考试：

1、拥护中国共产党的领导，愿为社会主义现代化建设服务，品德良好，遵纪守法。

2、国家承认学历的应届本科毕业生或往届本科毕业生；有国家承认的研究生学历的在职人员或已获硕士学位的在职人员只能报考为原单位委托培养的硕士生；同等学力报考条件见教育部有关规定。

3、年龄不超过40周岁（1961年9月后出生）。

4、身体健康状况符合规定的体检标准。

5、普通高等学校应届本科毕业生须经所在学校推荐，在职人员须经本人所在单位人事部门同意，其他人员由人事档案所在单位开具证明。

五、我校2001年仍为部分在职人员组织单独考试。招生对象为：符合单独考试条件且为与我校签订了全面合作协议的单位所推荐的优秀在职人员，以及与我校已有培养联系的优秀在职人员，且为原单位委托培养。详见单独考试招生简章。我校2001年继续招收工商管理硕士（MBA）研究生，一般为原单位委托培养。考生需参加全国MBA联考。详见工商管理硕士（MBA）研究生招生简章。

六、凡考试科目可“任选一组”的专业，考生须正确选考外语语种和考试科目，否则由我校指定。考生报名时应在信息卡上填涂选定科目组中每门考试科目的代码，应特别注意科目代码第一位为考试单元号。　

七、我校部分专业在两个以上不同的院、系、所同时招生，考生应注意选择并准确填涂报考的院、系、所名称及专业的名称和代码，否则由我校指定。　　

八、“建筑设计”、“城市规划设计”和“创作与设计”等科目，考试时间为一天，从8：30至17：00（考生午餐不得离开考场）。“素描与色彩画”考生应自带图板、文具和一号图纸若干张，请考场为考生准备绘图桌一张。考试时间为4小时，其中“素描”考2小时，“色彩画”考2小时。　

九、对于同等学力考生，在取得复试资格后，还须参加我校组织的本科阶段主干课的加试，其中笔试不少于两门，加试科目及要求请直接与报考的院（系、所）联系。

十、为帮助考生考前复习，我校于2000年9月中旬到考试前举办英语、政治、数学及MBA考前辅导班。报名时间：2000年9月10～15日。招生办公室恕不代购教材和往年试题。有关专业参考书、教材、往年试题、专业课考前辅导班以及专业、导师的详细情况请直接与报考的院、系、所联系。联系电话号码见各院、系、所招生目录。

十一、国家计划内考生录取后，一般均能享受普通奖学金。硕士研究生入学后，还可申请一、二等的优秀奖学金，一等1000元/年，二等500元/年。第一学年优秀奖学金的评定与入学统考成绩挂钩，对推荐免试生优秀奖学金的评定与本科阶段学习成绩挂钩，择优考虑。第二学年根据第一学年的成绩和表现，实行滚动竞争，重新评定。学校每年还设有企业和校友奖学金，奖励学习成绩优秀和科研成果显著的在校研究生。对违反校纪、触犯法律或学习不认真者，将停发普通奖学金并须交纳部分或全部培养费。

十二、我校已开通招生语音信箱，有关考场安排、考试成绩、录取名单等姓名查询，本地考生可拨打电话16810286，外地考生可拨打电话025-16810286
