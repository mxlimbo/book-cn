华北电力大学

硕士研究生招生专业目录

一、我校今年招收国家计划内硕士生76名，根据按需培养的原则，适当招收计划外委托培养硕士生和自筹经费生，全校各专业招生总规模200名左右。我校设有中华电力基金、四方公司、融商集团奖学金，用于奖励优秀研究生。四方公司和融商集团设立优秀研究生新生入学奖，以奖励考研成绩优秀的考生。

二、2001年硕士生招生报名及入学考试时间由国家教育部统一规定，报名时间为2000年11月10日至14日，考试时间另行通知。北京地区考生在我校研究生招生办公室报名，外省市考生在当地报名，报名和入学考试须在同一地点。报名者须由本人人事档案所在单位出具证明。在职人员持单位介绍信，应届生持学校教务部门介绍信，其他人员持人才交流中心或街道办事处介绍信。

三、学习成绩优秀、科研能力强的德、智、体全面发展的优秀应届本科毕业生，经有关学校（经国家教育部批准有应届本科毕业生推荐免试权的学校）推荐，可于2000年9月至10月向我校研究生招生办公室直接联系，申请免试入学攻读硕士学位。经审查同意后，正式办理推荐免试报名手续。

四、同等学历报考硕士研究生的考生按教育部有关规定条件办理。同等学历考生不允许跨学科报考，在通过初试后，复试时必须加试二门所报院（系）本科主干课程。

五、凡专业前带“*”者，均有博士学位授予权。
