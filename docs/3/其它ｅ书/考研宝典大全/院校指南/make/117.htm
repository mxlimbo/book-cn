中国科技大学

2001年招生简章

(1)中国科学技术大学2001年招收攻读硕士学位研究生950名，计划外委托培养硕士学位研究生100-160名。

(2)考生在登记表中如实填写工作单位并加盖单位人事部门公章，附本人最后学历证书的复印件。本校原则上不接收大专毕业生报考，少量优秀的大专毕业生（工作两年以上）需提供由教育部门出具的本科课程进修成绩单以及通过国家英语四级考试的证书。成教毕业生也必须提供四级英语证书。

(3)填表时务必写清学科专业名称、研究方向及考试科目编号。

(4)工科：数学(一)、(二)两型(教育部命题)。理科：高等数学(甲)、(乙)两型(本校命题)。

(5)中国科学技术大学有单独命题考试权，只招收原单位委托培养硕士生。参加单独考试的考生必须在报名截止前直接向我校研究生院招生办公室提出书面申请，履行报名手续。考试地点在中国科学技术大学。

(6)我校有MBA专业学位授予权，2001年招收130名。考生需直接到我校研究生院招生办公室报名。考试地点在中国科技大学。
