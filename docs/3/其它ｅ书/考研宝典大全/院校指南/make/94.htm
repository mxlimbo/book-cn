吉林大学

博士研究生招生简章

一、培养目标：

　　培养德、智、体全面发展，掌握本学科坚实宽广的基础理论和系统深入的专门知识，具有独立从事科学研究工作的能力，能在科学研究上做出创造性成果的高级专门人才。

二、学习年限：全脱产三年。

三、报考条件：

1.掌握马克思列宁主义、毛泽东思想基本原理，坚持四项基本原则，具有良好的道德品质，遵纪守法，决心为社会主义现代化建设勤奋学习，努力攀登科学高峰。

2.应届毕业的硕士生(最迟在录取前能够取得硕士学位)，已获得硕士学位的在职人员。

3.身体健康状况符合规定的体检标准、年龄一般不超过四十五周岁。

4.要有两名与本学科有关的副教授或相当职称以上的专家推荐。

四、报名时间：2000年3月6日 至 3月17日

五、报名地点：

　长春市吉林大学研究生招生办公室(外地考生可函报)。

六、报名手续：

1.符合报考条件的考生，需持硕士学位证书、毕业证书原件和单位人事部门同意报考的介绍信，交报名费100元(外地考生另付30元邮资)，领取有关表格(函报者由我校招生办寄送表格)；

2.考生报名后，需在4月1日前寄送以下材料：
(1)报考博士生登记表；
(2)两份专家推荐书；
(3)硕士课程学习成绩单(复印件)，硕士学位论文和评议书(应届生可缓交)；
(4)硕士学位证书、毕业证书(复印件)或证明书(在校期间考试合格的应届毕业硕士生报名时需开具证明书,必须在录取前补交学位证书复印件)；
(5)政治审查表；
(6)县级以上医院的体格检查表。
经审查合格的考生发给准考证。
报考登记表、体格检查表和准考证均须贴有近期免冠照片，寄送材料时请用挂号。考生报考的一切费用均由本人自理。

七、考试时间：2000年5月6日至5月7日

　　考试地点：吉林大学(具体地点发准考证时通知)，准考证一律自取。

八、考试和加试：

　　考试包括初试(笔试)和复试。初试科目见招生专业目录，外国语(含听力10％)，业务课两门。每门课初试时间为三小时，初试后择优进行复试。

　　跨学科报考考生需加试一门综合课，覆盖面应为所报考专业硕士研究生必修课两门，加试时间5月13日至5月14日。

九、录取：

　　根据考试成绩和硕士学位论文等材料，德、智、体全面衡量，择优录取。

十、毕业生就业：

　　在保证国家需要的前提下，贯彻学以致用的原则，由国家安排就业(定向培养和委托培养博士生除外)。

十一、有关事项：

1．招生专业目录上公布的招生数为计划内非定向、定向名额，不含计划外委托培养、自筹经费。
2．凡在报名时采用不正当手段、弄虚作假者，无论在招生工作中的哪一个阶段被发现，一律取消其报考、考试或录取资格。

热烈欢迎有志青年踊跃报考我校博士研究生！
