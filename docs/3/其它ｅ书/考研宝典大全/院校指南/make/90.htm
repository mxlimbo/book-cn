2001年硕士研究生招生简章

　　一、2001年我校计划招收国家计划、委托培养和自筹经费生约320名。

　　二、各学科、专业面向全国招生。所列各专业均接收定向、委培或自筹经费硕士研究生。欢迎高校、科研单位及国家重点建设单位，推荐本单位优秀在职人员报考定向、委托培养硕士研究生，同等条件，优先录取。

　　三、考试科目中有任选科目的，考生必须在报考登记表指定栏内注明。如不填写，则由我校指定。
