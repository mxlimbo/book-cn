兰州大学

2001年法律硕士专业招生简章

　　法律硕士专业学位是具有特定法律职业背景的专业性学位，主要培养立法、司法、行政执法与法律监督、法律服务以及行政管理、经济管理和社会管理等方面需要的高层次、宽基础的应用型和复合型法律人才和管理人才。

　　一、招生规模　60人

　　二、报考条件

　　1．拥护中国共产党的领导，愿为社会主义现代化建设服务，品德良好，遵纪守法。

　　2．国家承认学历的应届本科毕业生或往届大学本科毕业学历的人员；获得国家承认的大专毕业学历后经两年或两年以上（从大专毕业到入学当年的九月），达到与大学本科毕业生同等学力的人员；国家承认研究生学历的在职人员或已获硕士学位的在职人员，可以再次报考为原单位委托培养的法律硕士专业学位研究生。

　　3．所学专业为非法学专业。

　　4．年龄一般不超过40周岁。

　　5．身体健康状况符合规定的体检标准。

　　三、报名时间与地点　　报名时间：2000年11月初　　　　　　　　

　　报名地点：各省、市、区报名点

　　高等学校应届本科毕业生须持所在院校的介绍信、身份证；在职人员须持所在单位人事部门的介绍信（其他人员持人事档案所在单位出具的介绍信）、身份证和毕业证原件（并备交一份复印件）；到指定报名点办理报名手续。

　　四、初试与复试

　　初试时间：2001年1月中旬左右。

单位名称及代码:　兰州大学　10730

专业名称及代码:　法律硕士　030180

考试科目及代码:101英语　201政治理论（文科）　398刑法　498民法　598综合考试

　　其中英语、政治理论采用全国统考试题，刑法、民法、综合考试由“全国法律硕士专业学位教育指导委员会”命题。联考科目的考试大纲由“全国法律硕士专业学位教育指导委员会”编写。各科考试时间为三小时，考试方式为笔试。

　　复试时间：2001年4月左右　　　　　　　

　　复试地点：兰州大学法律系

　　初试成绩符合教育部规定的复试基本要求的考生方可参加复试。符合复试基本要求但因招生规模限制无法复试的考生，可以转到第二志愿“法律硕士联考”招生单位。

　　五、录取与培养

　　“法律硕士联考”考生一律按计划外委托培养或自筹经费培养录取。被录取的在职人员学习期间不转人事档案、工资和粮户关系，毕业后回原单位工作。

　　本地学员利用双休日及晚上不脱产学习，外地学员每学期集中一个月左右的时间学习。按照培养计划规定，完成课程学习和必修环节，成绩合格，通过毕业（学位）论文答辩，德育、体育合格者，授予硕士学位，颁发硕士学位证书和硕士研究生毕业证书。学习年限三年。培养费9000元/年。

　　六、考前辅导

　　1.辅导班：请与法律系办公室（兰州大学一分部）联系。　电话：0931——8913767

　　2.辅导材料：请与研究生院（办公楼319房间）联系。　　电话：0931——8912127
