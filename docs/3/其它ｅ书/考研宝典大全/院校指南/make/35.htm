复旦大学

2001年硕士学位招生简章

一.自2000年4月起，复旦大学与上海医科大学合并，成立新的复旦大学。

二.我校2001年招收硕士生规模为1850名（包括单独考试及M.B.A联考生），其中国家计划非定向及定向培养1152名，自筹经费和委托培养698名。

三.我校法律硕士专业学位研究生只招收非法学专业毕业生。其中在职人员只能报考非全日制走读委托培养硕士生，其他人员只能报考全日制自筹经费培养硕士生。

四.录取为我校数学、物理、化学、生物等一级学科有关专业的研究生，在攻读硕士学位期间成绩优异者，可提前攻博或硕博连读。

五.我校欢迎优秀理工科本科毕业生报考各专业硕士生,同时鼓励跨学科报考。

六.报名参加我校单独考试和M.B.A联考者须在2000年11月10日至14日来我校报名，招生简章另行制订。

七.我校医学院（原上海医科大学）招收硕士研究生说明：　

（一）医学院所属系（所、医院）代码为"101"至"119"，各专业只列出计划内招生人数。

（二）同等学力者报考医学院各专业除符合教育部有关条件外，同时须具备下列条件：　　

1.获CET四级证书；　　

2.近三年以第一作者在国家级杂志上发表论文或已成为单位的业务骨干（需有详尽的推荐材料）；　　

3.报名时须出具大学本科课程学习成绩单原件。　

（三）医学院所有科研型博士点均招收硕-博连读生，优秀硕士生两年后可转入博士阶段。"临床医疗技能训练与研究"方向均招收医学博士（临床医学）研究生。学制五年，以培养临床技能训练为主，同时完成规定的学位课程，并进行科研能力和教学能力的培养。采用分阶段连续培养、中期考核筛选、择优进入第二阶段的培养方式。非博士学位授权点学科只能培养临床医学硕士生，未转入第二阶段者亦作为临床医学硕士生培养，学制三年。　　

（四）医学院设下列各类综合考试科目。所有考生原则上考原毕业专业的综合试题，应在报名登记表封面及报名卡片的相应栏内注明考试科目及其代码，否则由我校指定。　　

1.医学、高护、法医专业毕业考生的综合考试（代码为306）内容：生理学、生物化学、病理学、外科学、内科学。　　

2.中医专业毕业考生的综合考试（代码为307）内容：中医基础理论、中医诊断学、中药学、方剂学、中医内科学。　　

3.卫生专业毕业考生的综合考试（代码为352）内容：环境卫生学、劳动卫生学、流行病学、卫生统计学、营养与食品卫生学。　　

4.药学系、化学系、生物化学专业毕业考生的综合考试（代码为353）内容：有机化学、无机化学、高等数学、物理化学、分析化学。　　

5.数学专业毕业考生的综合考试（代码为354）内容：数学分析、线性代数、概率论、常微分方程、复变函数。　　

6.生物系毕业考生的综合考试（代码为355）内容：细胞生物学、动物学、组织胚胎学、微生物学、进化论。　　

7.医学检验专业毕业考生的综合考试（代码为356）内容：生理学、有机化学、微生物学、实验诊断学。　　

8.生物医学工程专业毕业考生的综合考试（代码为357）内容：模拟电子电路、数字逻辑电路、微处理机。　　

9.卫生管理专业毕业考生的综合考试（代码为358）内容：预防医学、管理学基础、医院管理学、组织行为学。　

（五）医学院只招收委托培养的单独考试硕士生，考试科目与相应的招生专业相同。报名时间及地点详见第六条。

（六）为培养德才兼备的应用型公共卫生高级管理人才，医学院本年度招收15名定向或委托培养应用型公共卫生管理硕士学位研究生。培养年限为2～3年。研究生完成学位课程后,一般回原单位完成学位论文,由我校导师为主,与原单位具有副高级以上职称的老师共同进行指导。　　

1、报考条件：必须是在我国卫生管理机构、卫生防疫机构、疾病防治机构等部门从事公共卫生工作两年以上的本科毕业生或同等学力者。大学本科毕业工作四年以上的优秀在职人员可参加我校组织的研究生入学单独考试（报名时间、地点同上）。2、报名、考试时间:与全国统考相同。凡参加全国统考者,在当地报名点报名、考试。　　

3、入学考试科目：英语（101或111（单））、政治(理科)（202或212（单））、综合考试、466卫生统计学、专业课(可在956卫生事业管理学、945流行病学、948劳动卫生学、949环境卫生学、950营养与食品卫生学、952儿少卫生学中任选一门)。

（七）考生报名时，请务必填写指导教师姓名、学科专业和研究方向，否则由我校指定。　

（八）医学院所属各专业参考书目未予列出。
