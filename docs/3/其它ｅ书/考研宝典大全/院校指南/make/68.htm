深圳大学

2001年硕士研究生招生专业目录

　　深圳大学是于1983年创办的我国第一所特区大学，是一所集文、理、工、商、管、法、师范于一体的综合大学，是经国务院学位委员会批准的硕士学位授权单位。学校师资力量雄厚，教学、科研设施齐备。校园坐落在风景秀丽的深圳后海湾，环境优美，具备良好的学习、工作和生活条件。

　　2001年我校有政治经济学、金融学、国际法学、文艺学、行政管理、通信与信息系统、信号与信息处理、建筑设计及其理论、结构工程等9个专业，共24个研究方向面向全国招生。计划招收硕士研究生86名，其中包括委托培养研究生和自筹经费研究生51名（招生专业目录上公布的招生数仅供参考，以教育部最后下达的招生数为准）。

　　欢迎广大考生报考深圳大学硕士研究生！

一、报考条件

　　1.考生必须坚持四项基本原则，遵纪守法，品德良好，决心为社会主义现代化建设服务，年龄不超过40周岁，身体健康；

　　2.国家承认学历的应届大学本科毕业生必须经所在学校推荐方可报考；

　　3.国家承认学历的往届大学本科毕业生须经人事档案所在单位同意方可报考；

　　4.同等学力考生须同时具备下列条件方可办理报考手续：

　　①.国家承认学历的大专毕业生，工作满2年或2年以上;

　　②.以全国自学考试或在高校进修形式，完成8门以上与报考专业相同或相近的本科主干课程学习，并有成绩证明;

　　③.英语通过国家大学英语四级水平考试;

　　④.同等学力考生不允许跨专业报考硕士生。同等学力考生复试时还须加试所报考专业的两门本科主干课程（笔试）。

　　5.有国家承认的研究生学历的在职人员或已获得硕士学位的在职人员，可以再次报考硕士研究生，但仅可报考为原工作单位的委托培养硕士研究生。

二、我校各专业均可接收获得学校推荐免试资格的应届本科生。

三、报考办法

　　1.报名手续：

　　①.普通高校应届本科毕业生必须持所在学校推荐信及身份证;在职人员必须持所在单位人事部门同意报考的介绍信及本人毕业证书和身份证;其他人员由人事档案所在单位出具同意报考的证明及身份证和相关证明材料。于规定时间到当地省、市招生办指定的报名点办理报名手续。

　　②.深圳地区报名点：深圳市招生办公室（深圳市人民北路深中街16号Tel:2294044）

　　2.报名及考试时间：

　　报名时间：2000年11月

　　考试时间：2001年1月

　　以上报名、考试日期如有变动以教育部公布的时间为准。　

　　3.考试办法：考试科目共五门，其中政治、英语、数学（一）、数学（四）为全国统考科目，其它专业课的考试科目由我校自行命题。各科考试时间（建筑设计除外）均为3小时，各科满分均为100分。

　　4.考试地点：由当地报名点安排。

四、注意事项：

　　1.考生报名所提交的所有材料必须真实可靠。凡弄虚作假者，一经发现查实，一律取消报考、录取资格。

　　2.凡任选考试科目，考生必须在报名登记表中注明所选科目，否则将由我办指定。

　　3.凡需考生本人填写的通讯地址栏，一定要清楚、准确填写，如因考生填写原因而造成投递失误，其责任由考生自负。

　　4.考生请在报名材料中附上两个标准信封并准确、详细地填写本人的邮政编码及通信地址，贴足邮票，以便准确、及时地邮寄准考证和初试成绩。

　　5.根据有关规定，报考委托培养的考生，在录取前均需与我校及委托单位签订三方共同遵守的协议书。应届毕业生或在职人员被录取为委培生，在学习期间的培养费、生活费、医疗费及其它福利待遇均由委托单位支付，学生毕业后回委托单位工作。
