青岛大学

2001年硕士研究生招生简章

　　一、报考我校医学院各专业考生可直接与青岛大学研究生处医学部联系（地址：青岛市登州路38号，266021，电话：0532-3834684，联系人：张辉）。

　　二、大专毕业生以同等学力报考我校，只能报考与本人所学专业相同或相近的专业，并必须自学或进修过大学本科相应专业4-5门主干课程，报名时须附上本科自考成绩单复印件或高校教务处的进修成绩证明。

　　三、我校综合考试的内容如下：

　　文学综合：古代文学、外国文学、文学理论

　　口腔综合：含口腔颌面外科学、口腔内科学、口腔修复学、口腔正畸学、口腔组织病理学。

　　基础综合：含生理学、生物化学、病理学、免疫学、生物（遗传）学。

　　检验综合：含血液学和血液学检验、临床基础检验学、临床生物化学和生物化学检验、免疫学和免疫学检验、微生物学和微生物学检验。

　　四、为准确及时地给考生投寄准考证等，请考生在报名时填写三个有本人详细通讯地址和邮政编码的5号信封，并与报名材料同时寄我校研招办。

　　五、我校在招生专业目录后附有业务课考试科目参考书。所列参考书我处不负责代售。
