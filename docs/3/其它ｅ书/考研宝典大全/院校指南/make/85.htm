东南大学

博士研究生招生简章

一、培养目标

　　培养德智体全面发展，在本门学科上掌握坚实宽广的基础理论和系统深入的专门知识，具有独立从事科学研究的工作能力，在本门学科上能做出创造性成果的高级科学专门人才。

二、学习年限

三年（在职博士生可延长一年）。

三、报考条件

１、坚持四项基本原则，品德良好，遵纪守法，勤奋学习，热爱祖国，愿为社会主义现代化建设服务。

２、已获得硕士学位的在职人员、应届毕业的硕士生（最迟在录取前能获得硕士学位者）；同等学力者报考博士生应先征求博士生指导教师的意见，经所在系的系主任批准后，报研究生院审批。

３、身体健康，年龄一般不超过４５岁。

４、有两名与本门学科有关的副教授（或相当职称）以上的专家推荐。

四、报名日期

1、春季入学的报名日期为1999年8月20日至9月4日。

2、秋季入学的报名日期为2000年2月21日至3月4日。

3、双休日不办理报名手续。

五、报名手续

１、交报名费壹佰元整。

２、凭单位人事部门介绍信和硕士学位证书（应届硕士生凭学校研究生主管部门的介绍信）到我校报名（函报以邮戳日期为准，报名费同时汇出），领取空白表格，准备一寸正面免冠近照三张，自己贴到相应的表格上。

３、填写考生情况登记卡随即交验（函报考生在收到此卡后，当天填好寄回）。填卡时应将所报考的研究方向、考试科目的代码、出生年月日、籍贯（填至市、县）等项填写清楚。

４、考生征得所在单位同意后，春季入学的于1999年9月24日前，秋季入学的于2000年3月26日前递交下列材料：

　　（１）报考博士生登记表；

　　（２）两位与本门学科有关的副教授（或相当职称）以上的专家推荐书；

　　（３）硕士课程学习成绩单；

　　（４）硕士学位证书（复印件）、硕士学位论文摘要和评议书的复印件（应届毕业的硕士生允许推迟到录取前补交）；

　　（５）由县级以上医院出具的体格检查表（必须做肝功能检查）；

　　（６）政治审查表；

　　（７）同等学力者不交(3)、(4)两项材料，但应递交已经学习过的硕士课程的成绩证明(必须由高校研究生主管部门出具)和寄送在公开发行刊物上发表的与硕士学位论文相当的学术论文全文及科研成果材料。

5、经审查合格后可发给准考证，所有考生应于考试前一周内来校领取准考证。

6、考生考试时应持准考证、身份证、工作证及硕士学位证（应届硕士研究生持学生证）以便核查，否则不得参加考试。

六、考试日期

１、春季入学的为1999年10月23日至10月25日

２、秋季入学的为2000年4月24日至4月26日

七、考试地点 东南大学

八、考试科目

1、马克思主义理论（硕士毕业生可免考）。

2、外国语（英语为听力15%，词与语法30%，阅读理解40%，写作 15%）。

3、业务课二门（基础理论、专业知识, 考生必须将选定的考试科目的名称及代码在报名登记表封面上写清楚）。

4、面试：面试方式以口试为主（亦可笔试并对专业英语进行适当考核），面试时间、地点另行通知。

5、同等学力的考生，必须加试两门硕士学位课程，考试的科目、方式与时间另定。

九、关于委托培养

　　报考委托代培或定向培养的博士生，应由考生所在单位在考生报名时向我校提出书面要求，录取后双方签订委培或定向合同，毕业后仍回原单位工作。我校对校外考生招收委托培养博士生，一般不招定向培养博士生。

十、报考费用

　　考生的一切报考费用，原则上均由本人自理，经济确有困难者，可向所在单位申请补助。

十一、学习期间待遇

　　按国家有关规定办理。另我校已试行助教博士生制度，博士生在校期间一般应兼任助教、助管或助研工作，完成三助工作的可得到相应酬金，学习优秀的还可获得优秀奖学金。

十二、有下列情况之一者不予报考

１、攻读硕士学位期间有考试作弊行为者；

２、凡受过刑事、行政处分及党纪处分者。
