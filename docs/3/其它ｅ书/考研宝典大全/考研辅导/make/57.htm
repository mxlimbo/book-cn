陈文登：考研专家对你说

　　十八个字将是２０００年数学命题的特点

　　２０００年的数学试题会有什么样的特点？考生们应该注意什么？陈文登教授关切地指出：２０００年考研数学试题的命题特点可以用十八个字来概括，即"信息量多，题量大，知识面宽，综合性强，难度大"。信息量多即考试范围广；知识面宽指不但有数学知识，还会涉及到物理、力学、几何等学科的内容；综合性强不光有高数方面的知识，还有线性代数、概率及数理统计多方面的知识；综合性强了，学生会很难驾驭，所以说难度就大了。

　　根据２０００年命题的特点，陈文登教授强调：

　　一、根据题量大的特点，应掌握一些题型的快速解法，掌握一些常用的变量替换和辅助函数的做法。要熟记重要的定理和公式，并能熟练运用，在考试时减少追忆的时间。

　　二、针对综合性强、难度大的特点，应该采取的相应措施是：重点掌握常用的解题方法和技巧。多看点具有综合题解方面的书，这样可以启迪思维。对一些具有重要意义的综合性较强的题，不仅要理解，还要熟记它的解题方法，必要时背下来。

　　至于如何应付信息量大、知识面广的特点，那就要看考生的基本功了。

　　沉着应战，最大限度地发挥自己

　　数学考试不光是考数学知识，也是对考生综合素质的考核，比如心理应战能力，分析问题、解决问题的能力，难题化简的能力等等。有不少考生平时学得很好，基本功很扎实，但考试时把握不住"战机"，所以考不出理想的成绩。还有的同学考前乱"投医"，结果本来已经掌握的东西却被搅成一团麻。如何做好考前的准备和如何在考场上最大限度地发挥好自己是每个考生应重视的问题。对此陈文登教授特别提醒考生要注意以下方面。

　　一、临近考试前不要天天抱着类似《考前冲刺》之类的书看，这些书里没有解题的方法和技巧，只有一些模拟题，而那些题都是以往考过的题，一般来讲，每年的试题不可能重复考，做这些题对考研没有太大的作用。应该掌握的是数学中的一些重要的题型的解法，因为题型是有限的，而试题是无限的。搞题海战术对学生只能是一种误导，其结果事倍功半。

　　二、你所看的参考书中如果某一个例题的解法特别繁琐，而且再没有其他简单的解题方法（而不是作者故意繁琐化），这种题就不会考，别在这种题上浪费时间。

　　三、对于基础不太好的同学，不要认为上了辅导班，习题会做了就掉以轻心。数学和其他课不同，其他课通过一段时间的学习获得提高就可以了，而数学要经常练习，即"曲不离口，拳不离手"，尤其对一些基础性的运算要非常熟练。因为任何的解题方法和技巧都建立在内容熟悉的基础上，只有非常熟悉基本理论，解题技巧才有发挥的余地。

　　四、临考前不要再看新的参考书了，把原来的参考书吃透了就行了。因为每本书都有自己的解题方法和思路，吃透一本就行了。要好好总结一下过去参加各种考试的经验，这很重要。

　　五、考试时思想一定要放松，情绪要平静下来，尤其是当见到一些平时没见过的题时，千万别乱了方寸。对于比较有把握的题，不做便罢，要做就要完全做对，千万不要东做一点西做一点影响得分。对于非常陌生的题，如果有会的部分一定要写出来，在考试时应"分分计较，每分必争"。如果某题你做出的结果很复杂，就应该马上否定自己，重新再做一遍。

　　六、建议同学们考试时做题程序为：填空题，计算题，选择题，最后再做证明题。当然，如果证明题是你所熟知的类型也可先做。对于社会上有的单位打出某某老师押题百分之多少，同学们根本不要去信，都是胡说八道。陈文登教授认为，考前押题是对学生的不负责任的态度，是拿学生的前途作赌注。作为老师，对学生的求学应持严肃的、负责任的态度。

　　迎战２０００年的考研，自然是件压力很大的事，但对于有备而战的考生来说，成功的自信大于一切。在此陈文登教授预祝考研的同学心想事成，考研成功！
