2000年“政治经济学原理”试题评点

　　2000年马克思主义政治经济学原理由原资本上义基本原理过渡而来，考试内容也带有过渡性，以两门课程中相联系、相连接的内容为主，但带有明显的政治经济学课程的文科6道题12，理科科7道题18分（其中2道题为文、理科通用考题）。试题的分布、集中在前五章，即第一章4道题，第二章2道题，第三章2道题、第四章3道题，第五章2道题。第六、七两章没有出题。

　　试题的题型，主要是三种：单项选择题（7道）、多项选择题（4道）和辨析题（1道）。此外，理科还有论述题（1道）。单项选择题，两种类型，一种是考查对基本概念（或理论观点）的认知。例如，文科第5题考的利润慨念，文科第6题、理科第7题（通用）考的国家垄断资本主义产生和发展的根本原因。这类题，形式规范，答案稳定，难度小，只要复习了，记住了，不难得分（相比来说，文科第五题更容易些）。 另一种是考查两个概念（或理论观点）之间关系的准确理解和掌握。例如，文科第4题、理科第4题（通用）考的使用价值和交换价值的关系，理科第5题考的劳动二重性原理和资本主义生产过程二重性的关系，理科第6题考的区分劳动力和劳动与揭示资本主义工资本质的关系。这类题，考核选择关系，带有政治经济学研究方法的色彩，有个理解问题，就增加了难度，得分率低一些。这是值得考生注意的。

　　4道多项选择题，其中理科的2道，第20题考查价值规律的实现问题（这是2000年新增加的知识点），第21题考查产业资本循环过程中采取的三种职能形式（这个知识点1999年理科第6题刚考过），比较来说，难度较小，容易得分。文科的2道题，第19题货币的支付手段职能和第20题借贷资本的特点，则更难一些，得分就低。这两个知识点，多年来一直没出过题。货币支付手段职能问题，辅导材料上一般会写的，但考生复习得没那么细。借贷资本的特点，是政治经济学专业课性质的试题，理论上有深度，辅导材料一般写得不多，考生理解不好，蒙题而丢分，得分率可能是最低的。多项选择题是多年来得分最少的题型，看来2000年也不例外。这种题型要求对某个基本概念或理论观点做到全面地、准确地理解和掌握。考生在复习时应充分注意。

　　辨析题是文科第31题"股票价格是股票价值的货币表现",应该说不难。考生只要答上什么是股票和什么是股票价格这两个要点，就可以得及格分，至于为什么没有价值的股票会有价格，这第三个要点就难一些，考生一般不容易得分，理科分数比文科多，所以，往年出论述题的机会就多于文科。

　　2000年理科第33题论述题出的社会资本再生产原理，联系我国当前经济生活中存在的结构性矛盾来分析。这是一道规范的政治经济学的试题，按理说，基本原理部分并不难，联系实际部分的答案也是理论性的，要求也不高，但是，实际得分并不高，可能是因为有关社会资本再生产的知识点，多年来作为马克思主义原理课一直没出过题，一些理科院校因授课时数限制甚至都部没讲这个问题，考生没有予以足够重视而疏于复习。其次，答案的要点多，考生答不全。此外，一些考生平时不注意时事政治学习，对我国当前经济改革与发展的实际情况了解不多，答题答不到点子上而丢分。

　　总的来说， 2000年马克思主义政治经济学原理部分试题的难度，在于出了几道多年来一直没考过的知识点的内容，又带有政治经济学专业课的色彩。这是考试大纲这部分内容向政治经济学过渡后出现的新情况。2001年考试大纲的这部分内容将继续向政治经济学过渡，这一点应引起考生复习时的充分注意。
