张锦芯：早动手早准备满怀信心通过考试

　　从近年来发展形势预测，2000年报考硕士研究生的人数还将大幅度增加，1995年比1994年增加了一倍，但录取的比例却有明显下降。1995年却是五个考生中录取１人，与研竞争会越来越激烈。

　　硕士研究生英语入学考试是一种水平测试、选拔测试，必将择优录取。1999年英语的分数线是５１分，而1999年英语的分线是５０分（不同专业还提高到52分或57分）。

　　这个数字仅反映了99年的题比98 年的题略难了一些。实际上考试中心在98年的综合小结中曾指出自1996年来英语考试试题的难度相对稳定，也就是说命题组在出题时有一把基本的尺子，即考试大纲中提到的，达到大学本科毕业生能达到的及格或及格以上的水平。

　　应该说近年来出的考题确在相当的难度，当然不是说一年的考题中所有的题型都难，而是这次这几部分题型难，下次就可能是那几部分题型难；在一促题型中，如阅读理解部分，总有１－２篇短文偏难。

　　几年来在编书和办班的过和中，我们对历年的试题进行了详尽的分析和题解，清楚地看到，考研是测试考生综 合运用语言的能力的，也就是对考生读、写、译等能力的测试（由于种种原因，听说能无法测试），这样能力的获得取决于考生的基础（语法和语言基础）、词汇量、阅读量（包括较广泛的知识面）和语感。

　　为了帮助广大考生认清形势，早做准备，我们建议：

　　应届生 ：１、复习四、六级已学过的语法知识；２、按考试大纲要求去阅读，量要大、南要广，提高阅读理解能力；
在职生：

　　１、掌握大纲上的词汇和词组；２、复习四、六级的语法知识；３、每天１－２篇短文，逐步加大阅读量，提高阅读理解能力；４、从学习范文中练习写短文，由易及难，逐步学会写短文，关键还在勤练；５、选一些短文进行翻译练习。在考生复习一段时间后，最好能参加一期辅导班，也要选 择 一、二本好的模拟试题集，进一步复习巩固。如果不能参加辅导班，也要选择几本好的辅导材料（复习指导和模拟试题集）进行自学。
