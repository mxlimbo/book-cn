如何复习公共基础课？

　　公共课的复习是很重要的。一般来说，考研的人总是先复习英语。英语的复习是自始至终不可丢的，只是各个阶段复习侧重点不同罢了。开始的时候是重点突破，中间一般来说则是全面复习，做综合试题，最后冲刺阶段越来越紧迫，则又把注意力转移到自己的薄弱环节。尤其应该注意的是，愈到考试临近，有的同学对英语的复习愈加丧失信心，有的甚至产生放弃英语复习的念头，这样做是不对的，一定要坚持不辍，每天不须花太多时间，只需一个小时即可。

　　我的政治成绩还算可以，所以在这里说点儿经验还不太惭愧：复习政治最重要的是要学会融会贯通，这里既包括政治课内部各个组成部分之间的融会贯通，又包括与政治课外一些课程贯通。我自己是考法律系经济法专业的，在考政治时，我觉得我自己是多次运用了专业课上的一些知识点的。当然，这与各人的专业、知识面的宽广有一定关系。总之，要做到融会贯通，关键在于理解，切不可单纯死记硬背。

(小叶，北京大学法律系经济法专业)

　　考研对公共课要求很高，尤其国家就每门公共课规定了最低分数线，因此对每门课都不能掉以轻心。英语和数学不可能一蹴而就，必须提早进行复习。英语的词汇量要求不大，六级水平词汇就差不多了，关键是在于每个词的精度和深度，同时也要有一定的阅读量和阅读速度。数学的解题速度一定要提高，在把基本原理搞清以后，应多做一些题，因为考研题目的量较大，有的题不是不懂，而是没有时间做。政治的辅导书铺天盖地，我的经验是：以国家教委学生司出的那本书为核心，再选一两本互补的辅导书，理解加记忆，对考政治(文)的同学来说，理解更为重要，因为能直接从书里找到答案的题目很少，大部分题目需要自己的理解和判断。最后要提醒的一点是，在对各门课都给予了高度重视的基础上，要争取分数最大化。政治和英语很难得高分，而数学要得高分相对来说更为现实。
