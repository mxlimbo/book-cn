复习方法论——考研技巧

复习经验与答题技巧

　　我报考的是中国人民大学经济学院的国民经济专业。在国民经济专业课考题中，数学是一张试卷，政治经济学和西方经济学也是一张试卷。政治经济学部分考试题型为名词解释、简答题、论述题；西方经济学部分是名词解释、判断题、简述题、论述题（这是以1998年入学试题为例）。综合几年的考题可以发现，其出题范围，可以说是覆盖全书所有的内容，几乎在教材中的每一章都出过题。总体上看，整张试卷都很注重对考生基本概念、基本原理的测试，同时还考察考生答论述题时的逻辑、思维是否清晰，因此答题时应当扎扎实实将每一个知识点用严密的逻辑串起，切忌为了凑字数乱答或重复用句。

　　由于专业课考试重点题型是名词解释、简答题、论述题，所以我总结出用论点、论据、论证形象地说明三种题型的答题方法。名词解释基本上没有发挥的余地，只要提出论点即可。比如生产函数、总供给与总需求、引致需求、分税制等，书本上都有现成的答案，必要时能画图解释一番更好。但注意，对名词内涵和外延都要解释清楚，如这样一道题："我国M1层次的货币"，有的同学仅答了："M1＝M0＋企业活期存款＋机关团体存款＋农村存款＋其他存款"这个层次。就失分不少。

　　简答题的答题方式是"论点＋论据"，就是不只是按逻辑列出知识论点，还要列出可靠的论据。如在"产业资本顺利循环的条件是什么？"这样一道题中，谁要只答"时间上的继起性和空间上的并存性是产业资本顺利循环的条件"不失分才怪。

　　论述题是整张试卷的压轴题，所占分值也较大，大都在15分～20分之间。考生在答论述题时逻辑上一定要严密、清晰。最简单有效的方式就是用一、二、三、四，A、B、C、D把各答题要点用逻辑这根红线串起来。在答题前，应该在草稿纸上列一下提纲，以免想到哪儿写到哪儿，给人杂乱无章的感觉，得高分也就无望了。

　　经济学院研究生入学考试历届专业课考题可到经济学院办公室买到。考生应提前购买历届考题，因为专业课参考书目如果有所变动，那里会有所显示。

　　

面试与录取　　
　　接到分数成绩通知单之后，如果你的总成绩在330分以上，并且各单科成绩在55分以上，就应该与院办公室联系，因为有些相对较低的成绩可能能上自费；或者虽然你已上线但高分者较多，可以转到其他专业或其他学校。多联系，多询问，不会有坏处的。

　　人大招生工作很严格，一般情况是按分数从高到低录取，大都是等额复试，很少出现参加复试之后却没有被录取的情况。如果分数很高，基本上可以高枕无忧了，成绩相对较低但又过分数线的，可能碰巧有自费名额，多联系，机会就会来的。

　　在复试时，一般由复试老师提几个问题。在复试前作准备时，可看些与本专业有关的近期发生的经济事件的理论研究文章，若回答不出，虚心向复试老师请教，他们会告诉你的。这样，最后一关也就过了。祝你成功！

　　朱金周中国人民大学经济学院

　　

复习方法　　
　　若仔细观察一下两年来考生的成绩，尤其是录取考生的成绩，大家会发现：专业课的成绩基本上相差不大，差距主要体现在数学、英语和政治三科上。大家都知道，近几年来考研的题目越来越难，对于文科学生（包括本科学经济的学生），甚至对于某些工科毕业的学生而言，报考经济类的研究生，数学是最大的困难。我认识的几个1999年的考生都是因为数学的失利而失败的。所以我首先提醒有志报考北京大学中国研究中心的同学（尤其是文科同学），必须在数学上多花力气，如果时间允许，最好报名参加一个考研数学辅导班。题目必须多做（而且最好多做些难题），即使快临近考试了，每天也要抽点时间练练数学题。惟有多练，才能见识到各种题型，才能找到解答某种题型的最快捷有效的办法。

　　对于专业课，我的体会是：由于专业课内容复杂、知识点较多，因此复习专业课最有效的办法是先将教材（宏观和微观各选一本）精读两遍，然后抛开书本，将本书中的知识点用自己的话写出来，整理成精悍的笔记，然后再翻阅其他参考书，将新的知识点补上。这样做，一方面加深了对理论知识的理解，另一方面也便于以后的复习。

　　

关于考试题　　
　　专业课考试一般有3种题型：一、判断对错并说明理由；二、选择并说明理由；三、解答题。北大中国研究中心出题的最大特点是：每一道题一般都对应着书本上一个具体的知识点（一般很少有一题考多个知识点的情况），您只要想到这个知识点，并用它来清楚地解释某个现象，或进行具体的推导，一般都能全取该题分数。如果您没能找到相应的知识点，那么无论怎么发挥也得不了分。判卷老师最忌大而无当的论述，因此你要牢牢地掌握书上的每个知识点，最好知道该知识点可以用来解释何种现象。考试时，或用文字，或用图表，或用数学公式将理论表达出来，万万不可天马行空，乱答一气。

　　

面试　　
　　从1999年起，面试变得非常重要起来。首先实行差额面试，达到一定考分的考生都有面试资格。今年录取了25人，却有近40人参加面试。更为重要的是，面试总分为300分，录取时将面试成绩加到考试分数上去，然后按分数的的高低进行录取。

　　考试分数下来之后，你如果在面试名单上，就应该多往中心跑一跑，让老师有机会了解你。即使面试后你没有被录取，也应该多到各处跑跑，说不定就能转到别处去念研究生。如果你能得到中心某位老师的推荐，那就更有希望了。
