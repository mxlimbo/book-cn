法律硕士前景

　　法硕发展前途是很大的，我就从以后从事律师业来说，法硕直接面对case与现实十分切合，中国向法制化发展，使律师的数量和质量都需要进一步的提高。

　　其次，社会分工的发展和法律涉及的面的增加，使复合型的法硕有了更广阔的发展空间。

　　再次，目前我国律师从业人员中高学历者还是比较缺乏，法硕在这方面无疑有优势。最后，与传统的法学硕士相比，法硕学生可能更具学习的动力，因为法硕直接面向市场，必须充实自己，才能不辜负昂贵的学费，而且法硕生可选的学校更多，从一个名校的法硕毕业出来，肯定比一般学校的法学硕士更有机会。所以考法硕为了以后不后悔，尽量报名校，学费以后是可以赚回来的。
