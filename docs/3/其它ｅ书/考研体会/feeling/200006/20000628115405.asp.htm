第七章 第三次考研
　　1999年8月27日，一个值得我永远铭记的日子。
我的命运从此有了一个巨大的转折，当我一步一步的从这一天走到现在，当我从容的接受那一点一点降临到我身上并且需要我用毕生心力去珍惜去维护的机遇的时候，我只是平静的在想：如果真有上帝，那麽，他对任何人都是公平的。
　　就在这一天，我遇到了我未来的新娘，她就象上帝派来抚慰我心灵的天使，轻飘飘的又是突然的降临到了我的身边。她，就是我的小英。
　　上火车前广州下着大雨，从混乱恐怖的广州火车站挣扎到直达西安的火车上时，我已经是狼狈不堪。天很热，我穿着短裤，更索性脱掉T 恤，大口大口的喘着气。卧铺车厢里满都是形形色色的人，突然有人在后面轻声喊我，回头看是个女孩，要我帮忙放一下行李，行李架上已经挤的满满的，没办法，我就把安放在上面的我的包拿下来塞到床下，把她的行李放到空出的位子上，然后我就坐了下来。再转过脸来看时，女孩已经是坐在正对着的车窗旁边了，她侧着身子，一付气定神闲若有所思的样子，我觉得应该跟她聊几句了，因为我忽然发现，她真的非常漂亮！
　　跟她讲话的时候，我已经规规矩矩的穿好了T恤，我尽量的表现的很随意很绅士，就这样，我认识了这个家在广东、现在正上西北大学旅游系三年级的女孩。
漫长的旅途，竟因为有了她在旁边而变的无比的轻松愉快了起来。
　　我是上铺，她就在我下面的中铺，起初我还只是“以貌取人”，对她也只是所有男人此情此景下都会表现出来的那种态度，可那麽一瞬间，凌晨时分，当我从铺边梯子上爬下的时候，看见她熟睡中惹人怜爱的面容，微曦的光线照着她长长的生动的睫毛，那一刻，我突然产生了一种强烈的想要庇护她关爱她的愿望，并且莫名其妙的被自己的这种愿望给感动了。
　　尽管我知道，此行西安，压力非常大，放弃了不错的薪水放弃了那麽多即得的利益，就是为了成功的考上研究生，所以，我不能再有其它的非份之想。但是，我还是不自禁的幻想：是不是老天真的恩赐了这次机会给我？难道电影小说里面的那种浪漫的机缘真让我碰着了？不过我还是抵御住了“诱惑”，心想，就当是一个美丽的梦吧，还是做正经事情为好。
　　可是没想到，她竟然主动要我的电话和地址！我自然是当仁不让的要了她的电话，心里暗说，这可不怨我，是她主动的，再浪费这个机缘可就没道理了。
　　到了西安，分手后的当天晚上，我给她打了电话，从此以后，在追求这份浪漫情感的征途中，我变的不可自拔也根本不想自拔了，因为没过多久，我就惊奇的发现，再也不轻言爱情的我，却已经无可救药的爱上了她。
　　后来她告诉我说，当我在火车上善良的为她摆放行李，一路上倾劲心力的关照她的时候，她就有点喜欢我了，当她回校后有点六神无主可接到我打给她的第一个电话的时候却兴奋喜悦的时候，她就知道，一段浪漫的西安爱情故事终于渐渐的开始了。
　　直到我们第一次去看电影，电影已经开始后当我知道明天就是她的生日，匆忙出去，在黑夜的大街上为她买到一盒大大的米旗蛋糕，又匆匆的赶回到影院她的身边；直到那次在肯德基，我突然收住笑，郑重而局促的递给她一张纸，纸上写着：做我的女朋友，好吗？直到中秋节那天，我在花店里定了一大束玫瑰，连带一盒广式月饼，让花店的人送到她的宿舍；直到一个月后，在西大美丽的校园里，在月色的笼罩下我第一次的吻了她，我们终于越来越深的相爱了，我们在更多的了解，更多的欣赏中，在轻松快乐的相处过程中，在共同抵御一些烦恼，趋散一些忧愁的漫步中，逐渐的培植和维护了唯属于我们两人的成熟而健康的爱情。
　　后来有一次，她送给我一瓶亲手叠制的幸运星，我情愿相信，我能够比较顺利的考上交大，能够非常幸运的和她共守未来的一份承诺，全部的理由，都在那3 65颗凝聚了她深深爱意和祝福的幸运星当中了。
　　因为有了这份爱，我原以为单调枯燥的苦日子变的甜蜜了起来，我原以为几尽悲壮的考研生涯也变的浪漫可爱了起来。
　　我是在一种非常塌实非常健康的状态中走过这复习考研的半年时间的。
　　从九月开始，我正式复习。住在原来单位的单身宿舍里，那里环境很好，离城远，清净省心。每天我都有规律的看书学习，吃饭就去单位食堂，刚开始还因为老碰到过去的同事而不太自在，时间长了，脸皮也厚了。大家都知道，我不远千里的辞职回来，就是想一圆旧梦，那麽多人经常问我的一句就是：复习的怎麽样了？我只当是对自己的鞭策：这麽多双研究都盯着呢，可别再放空炮闹笑话了。
　　我在上海交大没有认识的人，上海工作的同学也不能给我提供更多的信息，于是我通过交大的BBS来发些求助的帖子，幸运的是，就这样，我结识了计算机系的一个研究生，她也是北航的。后来她给我寄来了历年的专业考题，向我介绍计算机系的导师，并且引荐我跟导师勾通，及时给我提供考研的信息，我真不知该怎麽谢她，正因为能够随时了解学校的一些招生动向，虽然远在西安，虽然没有接受什麽专门的辅导，我也能比较从容的进行复习。
　　真想尽可能详细的讲述一下我复习的过程。因为我最终成功了，所以，本来不怎麽显眼的东西，讲出来后，都成了宝贵的经验。
　　先讲讲英语吧，我学的是最典型的聋哑英语，想当年在北航，考了两次六级都没过，第一次是36，根本就是闹着玩，第二次，索性不做听力，全都选C ，省下时间做其他的，最终也只考了53，幸亏研究生考试没有听力！98年那次考研，我的英语破天荒考了72分，颇得意了一阵子，对自己的学习方法愈发肯定了：英语考试，就是个词汇和阅读！我的词汇量向来都挺大的，光凭吃老本也差不多够了，加上这两年来大量的阅读计算机原版书籍，疯狂的考各种计算机认证，啃的也都是英文材料，所以，对待英语，我最感触深刻的就是：大量的阅读！第二次考研，我总共读了大约5 00篇应试文章，而这次，我更是平均每天都读它几篇，读的多了，作文、翻译等等也都跟着好了起来。当然，除了阅读，做一定数量的模拟题也是必要的，从开始复习起，我大概是每隔两天做一套，所谓熟能生巧，大概就是这个道理吧。
再看数学，我一直以为，自己大学时数学很好，所以考研也就不在话下，可经历了98年49分那次噩梦（虽然上线了），我强烈的意识到，光凭一个单元一个单元的复习，题做的再多也没用，到了考场，白花花的一片，跟本就无从下手一样，所以，多做些模拟题，进行综合性的训练是非常必要的！复习过程中，我根据不同的阶段，规定不同的复习内容，每个阶段末，都做模拟题进行实战仿真。我就是这样稳扎稳打的复习数学的。
　　对于政治，我无话可说，考了三次研，只有政治成绩是每况愈下，我也搞不懂，费了不少劲，自以为也融会贯通了理解了，可“中央精神”还是那样走位飘忽，宛若凌波微步，搞的我晕头转向。
　　最后，不得不讲讲让我几度凋零几度遗恨的专业课！这次我考的是数据结构和微机原理。数据结构我没学过，复习的时候，只知道交大用的是严蔚敏的那本经典教材，手头只有几份交大前些年的考题。可以这麽说，考研的五门考试中，数据结构上我花费的心血最多，也最担心。不晓得课本被我翻看了多少遍，只要书店里有相关的习题集什麽的，我都如狼似虎的买来一读，仔细的琢磨交大的考题精神，仔细的推敲每一道试题。我从来没有象对数据结构那样虔诚的对待别的学问。跟数据结构比起来，微机原理就相对简单的多，我基础很好，又特意去西工大听他们的专业课辅导，临考时，已经是非常的踌躇满志了。
　　就这样，我有条不紊的复习着，几乎每个星期总有那麽半天，要麽我去找小英，要麽她来看我，我们一起逛街，一起看电影，一起畅想将来在上海我们的美好的生活。
　　11月报名前，为了开介绍信，我好不容易把档案从原来单位提出来（辞职去深圳后，我的档案一直放在原单位），放到省人才中心，被原单位刮去七千多块。报完名，拿到了准考证，一直等到2 000年元月22日，我第三次“光荣”的走进研究生考试的考场，决定我命运的最后一战终于开始了。
　　一切都进展的很顺利，考完了英语、数学，考完了政治、微机，最后一门，数据结构，我惊讶的发现，今年的考题跟往年大不一样！我几乎跟傻了似的，楞了半天没缓过劲来，回过神稍冷静了，我竭尽脑力完成了答卷，粗算了一下，也就是及格上下的水平。不晓得怎样走出考场的，前几日考试顺畅所带来的好心情一下子就没了，我真的害怕再次失败了。
　　不过等冷静下来了，我想，既然题型大变，只要没有太多黑箱操作，那对大家来说都应该是公平的，我觉得难，大家也应该有相似的感受，反正我其他几门挺好的，而且考计算机系的人都考同一门数据结构，我应该没有太多劣势。这样一想，我坦然了许多，快过年了，我还要留一份思念给回家了的小英，还要尽快找到新的工作，还要寄希望于新的一年。
　　小英说，她是我永远的幸运星，说我一定会考上的，说我们的未来将是多麽美丽多麽灿烂的景象。
　　她的话就象康太克治感冒那样的灵验，只要早一句，晚一句。
　　我想我成功了，因为有了她，我似乎就真的拥有了一切。
　　春节过后，我把简历发到网上，也主动出击，去了许多公司，最后确定留在了一家开发网络安全产品的小公司里。
　　其实有许多公司都对我感兴趣，有一家公司老总非常赏识我，甚至让我去做他一人之下众人之上的技术总监，可以持股，诱惑着实不小，可最终我有了自己的选择。之所以如此，完全是因为，我已经决定要向网络安全方面发展，我报考的导师就是搞网络安全的，在入学之前的半年时间里，我要做好充分的准备。
　　就这样，我再就业了，虽然薪水不是很高，也可以满足了，一方面学习新的东西，另一方面，可以有更多的时间陪在小英的身边。
　　剩下的，就是等待考试的结果，等待最后的好消息了。
