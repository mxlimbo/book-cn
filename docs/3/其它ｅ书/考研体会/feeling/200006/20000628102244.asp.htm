纵然失败，我无悔于我的选择

http://www.kaoyan.com 2000年06月28日10:22:44 考研热线

作者 jungle
--------------
2000年考研终于接近尾声了，谜底开始浮出水面，如同研渡寒潭所言，几家欢喜几家愁。不幸的是我属于后者，“金榜题名，倒数；洞房花烛，隔壁”。我想，论坛里肯定也有和我一样，虽然全部上线，但因排名靠后而无法被录取者。
悲乎？于我而言，遗憾是有的，但我并不悲伤，这次的考研经历对我是一个美好的回忆，我尝试过了考研的艰辛，知道成绩后的紧张，无法被录取时的彷徨。现在坐在计算机前打这篇贴子的我，心情是平静的，我祝福那些金榜题名的考友，特别是研渡寒潭，希望你能如愿以偿，也谢谢你把我看作一位知已朋友，想起在那些等待分数线的日子里，有一位考上北大的战友和我互相鼓励，使我的这次回忆增色不少。
昨天，我看见longants的贴子，写了一篇长长的贴子想安慰他一下，没想到马上就
写完了，因为误操作，写的东西全部都丢了。这就象我的这次考研经历，挣扎到了最后，还是以失败而告终。我很想对longants说，既使他三次都没考中，而且这次是因为专业课而中箭落马，也没必要有那么沉重的心情。比起那些在麻将桌旁．酒馆里高谈阔论消耗掉的时间，一次哪怕不成功的考研也比之有意义了许多。

我们总在计算，考研让我们失去了多少宝贵的时间，投入了多少的金钱和精力。其实，我觉得，人的时间和精力是无穷的，你不在这方面消耗它，就会在另外一个方面消磨掉它, 相比较起来，考研对很多在职生而言，比浪费在一些生活中无聊琐事上有意义了上千万倍。当然，考研并不是我们到达金字塔上层唯一的途径，但是对于很多在职考生而言，考研是最好的选择。成功了，它可以改变目前了无生气的生活；失败了，至少它还能使我们想起曾经有过的理想和热情，使我们觉得我们还能坚持对人生的一种态度，还没有让生活使我们麻木。

我是作为一个失败者来谈论这次考研，我很想劝劝那些和我一样失败的战友，不论是没上线，还是上线没被录取的，不要再把精力投入那些钻牛角尖的问题上去了，不要再挖空心思的想自己失败是否存在人为的因素。当然，对于那些因为黑幕而失去入学机会的战友，我也深表同情和对现实不公平的不满。但是，就象农夫的贴子中说的一样，“多找自己的原因“。还记得我到川大研招办询问我能否被录取时，研招办的老师说，今年这个专业除去保送生，招得还没去年多，我是没有希望了。当时，我很不心甘地问老师，难道我就这么完了吗？那位老师说了一句，那你为什么不考第一名呢？是啊，为什么我们没考第一名呢，如果我们有实力进入前三名，就可以保证百分之一百的入学，不用去想黑幕．是否送礼等等烦心的问题。如果打算再考的朋友，不要再沉溺于这次考研的得失了，现在就用全部的精力准备下轮的战斗，争取考第一名吧！我祝福所有人，也为所有为理想而奋斗过的人喝彩！

再来谈谈现实的情况,我考了两次,觉得专业课的名堂确实很大,我第一年,当时写信给导师时,老师教我到学校去一次,我认为专业书看了五六遍, 历年试题都倒背如流,很想通过自己的实力来证明一切,结果我太天真,两门都只考了70多分,今年我早作安排,国庆节到学校去了一次,与导师见了面, 理所当然带了东西,老师的说法让我心里有货,今年我对专业课只不过没事翻一翻,元旦到学校去了,一门课的题目几乎搞定,当时搞得我有些激动, 又有些悲哀,(这就是中国的国情,一边反腐倡廉,一边花天酒地,所有事情都是由人决定)另外一门就是他出题,他倒是一点没透, 不过我从他的做法,心中已踏实不少,考完后,立马打电话告诉准考证号,成绩下来他出的那门80多,另一门90,可惜我的数学一没过,总分344 ,我现在再和他联系,怎么也联系不上,可能他怕我找他,有些同学看了可能有些气愤,可这就是实际,我考取的同学没有一个没搞到考试范围就上考场的, (包括一些前10名的高校)在中国这块人治的地方,要想得到承认,就必须面对人,所以说,我认为没什么可报怨的,就算到国外,处理好人与人的关系也同样重要.
其实从心里我也很反感这些,否则我就不会考研,不如在单位混混,将来搞个一官半职,但在中国哪里不一样呢!我不知道,只是觉得心中有个地方应该不是现在的样子, 所以我不满意现实,通过考研来实现我的理想,我也知道考研没有这么大的力量,但只要有变化就行.
