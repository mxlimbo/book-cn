我们的梦想在何方？-----对BETTER的回复

http://www.kaoyan.com 2000年06月28日10:02:36 考研论坛

作者：huizi2000
――――――――――
你起了一个充满希望的名字BETTER。谁不希望自己的明天更美好呢？
我和你一样考完数学的一瞬间我就知道自己希望不大了。明知明天还有两们
专业课考试，那晚我却怎麽也睡不着。谁都知道梦想被击碎时的痛苦。
成绩出来了。我果然数学差一分，可惜了我专业课90以上的高分。
是有许多人不考研一样过的很快乐，但这些人不一定是你我。
要不要再考？我自己也在困惑。“不想再失败了”只是一个推脱的借口。
因为我当时也是这样对别人说的。关键在于我们考研的目的是甚麽？
我们值不值付出这莫大的代价去考研？我也是在职的，我深深明白在职考研
所付出的代价。我有一个同学，大学毕业时别人都有一个好工作，月薪都有几千
圆，而学习成绩十分优秀的她却分到成都邮校教书。于是她冲破重重阻力考研成功
她使我想起了一句话：知识改变命运。
那麽我想问你：你为甚麽考研？其实这句话我也在不停的问我自己。
我又是为甚麽考研？不过我看你写的文字，感觉你是会再考的了。
而我还没决定。一方面我不甘心一分就击碎的上研的梦想。另一方面我又找到了
一份较好的工作（考完研后我辞职了），它给我以我自己没有想过的高薪
（7000-8000RMB/Mon）。可是却工作很辛苦。又有好友拉我去自己开公司创业。
人生走甚麽样的道路？我不知道。我对同学说：我找不到自己的方向。
茫茫人海，我的位置在那里？！！！我上研的同学不知道我在说甚麽。因为
他们不能感到我一个人浪迹在北京的辛酸和苦痛。虽然才毕业一年多，可我感到
想一个世纪那样漫长。常看《平凡的世界》，我感觉我和孙少平一样都是打工的。
我不过多一个北京户口。孙少平肉体痛苦，可精神幸福；我也许肉体并不痛苦，可
我精神痛苦。有时候精神的痛苦比肉体的痛苦更可怕。
佛说“人生是苦的，苦的根源在于人的欲望。”
我们如果没有梦想，我们也就不会有这莫大的痛苦。也就不会有这辛酸的考研史。
我现在也在问我自己：我的梦想在何方？因为这决定了我是否想考研。
