我的选择

http://www.kaoyan.com 2000年06月28日09:49:24 考研论坛

作者：璞玉
————————
走出校门已有近两年了。时间飞逝得让人难以相信。
我的人生经历很平常、很顺利，包括升学、工作，甚至一个人在沿海城市闯荡时，由于在一个福利待遇不错的国有金融单位，我的生活依然平静快乐。有人说“简单就是快乐”，但我以为只有历经沧桑的人才有资格说这种话。更何况简单的人并不能一直在复杂的世界中快乐地生存。所以为了争取生存的主动、为了打破这种平静、为了……，我辞去了工作——考研！
之所以辞职，是因为我对自己的克制力没有信心。我很佩服昨日W2在《路不能越走越窄》一文中描述的那种定力。太温馨、舒适的生活对我是一种难以抵御的诱惑。所以我给自己断了退路。
从去年10月开始，我象个鼹鼠一样呆在窝里，一心一头啃啮着书本，除了考研，外面世界的阳光灿烂和暴风骤雨都和我无关。当考研结束、春节过去，我一边急不可耐地想去呼吸外面的空气，一边又忐忑不安地等待着成绩。结果是我痛苦的历程绝没有结束。其实失败是意料之中的，我知道想成功不会这么简单。W 2说不要把青春投入在一件事上，这没错。但一旦决定要做的事就要做好，否则尝试再多，都将一事无成。更何况我已付出了相当的代价。所以现在我的眼里只有考研——尽管我还没有品尝过爱情的甜美、感受过创业的激情。我现在面前的路也因此只有一条，不管它是宽还是窄，我只知道，“走过去前面又会是一个天！”
