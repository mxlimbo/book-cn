考研,一条不归路吗?

http://www.kaoyan.com 2000年06月28日11:12:06 考研论坛

作者：kaoyan.com网友jenethy2000
那日同学突然冒了一句,考研一条不归路,你可要当心.是呀!下了n次决心,动摇了n次.现在终于决定考了.考吧,给自己一个机会.拼一下,要么延续大学生活, 要么终止学校生涯.

还是希望继续读书,从小到大一直读书,没读傻也没读的聪明,只不过为了养活自己,报答父母.到是渐渐少了一份灵气.随着年龄的增长,烦恼随年龄n 倍增长.要考虑的事太多.

考研为了今后更好的发展.作为计算机专业的学生,现在找工作好象还行.可我还是想多学些,研究生的头衔对我实在是一个诱惑.虽然那样会失去很多, 但我还是选择了它.怎么样都是个活法,大不了终善其身.我也无所谓了.到是苦了父母.这么多年来的操劳.为了我他们太辛苦了.让他们等太久, 实在是我不忍看到的.我一日不出来工作,对他们来说,一日就是一份操心.我也没办法,真的.如果明年研究生收费上学,对我又是一个不小的打击.

算了,有些东西我也不去想太多了,来点阿Q精神吧!

暑假去上恩波考研复习班,拼一下吧!但原明年的今天,我能如愿!
