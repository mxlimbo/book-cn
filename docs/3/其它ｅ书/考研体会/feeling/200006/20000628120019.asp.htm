我为什么要考研

http://www.kaoyan.com 2000年06月28日12:00:19 考研论坛

作者：kaoyan.com网友yan2001
1997年，我毕业于山东青岛一工科院校，因为财力及其他原因，我回到了家乡--河南某市。
没有找到一个与专业相关的好单位，我就这样没有任何目的地回到了家乡。但就在回到家的第二天，奇迹出现了----我哥告诉我：报社领导让你明天去报到，后天上班。当时我正在舅舅家玩，还以为他和我开玩笑。但事实正如他所说，我真地万分幸运地进了报社，做起了记者！熟悉行业内情的人都知道，报社可不随便就能进的地方，有许多副地级的关系还等着进不去呢！我没花一分钱、没托其他任何人，只凭着我哥- ---一个通讯员与报社领导的关系，我进去了，我心里一直非常感激领导。
刚进报社那阵子，因为没有学过新闻，领导让我做校对工作，好在我文字功底还好，工作也很认真，做得很好。其间我抽空大补新闻专业知识。
1997年11月，我如愿以偿地做了一名光荣的人民记者。开始了也许是我终生的职业生涯。社会新闻，主个既有趣又具挑战性的领域曾令我日夜难眠- ---线索是一个大难题。慢慢地熟悉了，靠我尖锐的笔锋和踏实的作风，赢得了同事和社会的认可。但渐渐地我发现，批评稿子太难发了，同事、朋友们都会在你辛辛苦苦半天以后对你说- ---算了吧，自己 人，别发了。更有甚者，领导在你完全不知道的时候，已经把你的稿子撤下来了，因为当事人直接找到了他们。
如果说损失一部分心血还可以忍受的话，报社的那种死气沉沉的气氛更让我心凉----一张党报只有十几个写稿，任务是一加再加。我曾经一个人挑起社会新闻每周4 ＊半版的任务，当时不觉着累，但后来当领导要求每人每月完成二十多篇稿子时，我怎么也想不通了----每个工作日发表一篇稿子，硬性的，不说要不要休息，新闻也不是那样写出来呀！再加上越来越感觉到领导对下属除了工作任务，其他一概不关心的时候，我失望了。
有时候自己在想：我做记者为了什么？不像有些人图个脸熟，到哪儿办事都能过；有些人想通过关系赚些钱。也许我太嫩，总想着能用自己手中的笔为社会、为劳苦大众谋点什么，让社会少些不公，让恶人多受惩罚。但理想和现实差别太大，我觉得在这样一种氛围下，又怎么能对得起记者的称号？
虽然类本地，报社的工资不算低，加上时间上自由些，又有社会地位，很多人都非常羡慕我的工作，但我还是下定了决心----考研！离开这个让人难受的地方，寻求新的发展。
人大新闻学院是我国新闻行业泰山北斗，是我入新闻行以来最想去的地方。我想，不考刚罢，要考就考最好！免得将来出来还是受人看不起- ---知名度小的学校出来的学生的确如此----这是国国情。
为了我们的将来，为了我们自身价值的发挥，兄弟们，我们考研吧！！！
