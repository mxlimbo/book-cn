考研的初衷很简单,只是不喜欢当时的工作和环境,想重新回到曾匆匆逃走,头都不想回一下的学校.想重新拿起
书本,学自己想要的,做自己喜欢的.可却不曾料到这对于我来说只是一条不归路的起点.以后的几年里考研成了我无法摆脱掉的梦魇,每次走出考场, 我都诅咒发誓再也不考了,可每当春天来临,万物复苏的时候,我的心便
又无法遏制地蠢蠢欲动起来.一次次的失败愈发使我感到自己是否真的走错了路,轻易放弃自己原本安逸的生活和工作,是否真的有些白痴? 真的应了那就老话'心比天高,命比纸薄'?时时感到自己象是一只钻进迷宫的耗子,既看不到出路,又忘记了来路.考吧?真的有些迷茫了.不考呢? 自己的理想和所有的信心仿佛都丢在了这条不归路上,再去做什么都提不起信心,重重的挫败感会深深的盘踞在我日后的生活中,挥之不散.
我想自己是那种没有理智,不懂得该在适当的时候放手的人.我渴望成功,渴望找回在一次次失败中溜走的自信!
失去它我将一无所有!!
