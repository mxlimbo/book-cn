考研总动员

http://www.kaoyan.com 2000年06月28日12:12:40 考研论坛

扬剑
"东风吹，战鼓擂……"，五六十年代报刊中常见的这句套头话，往往用来形容人们大干快干革命事业的热情。今天，把这句话搬用到我和我的同学们身上，竟也是出奇的贴切。因为，我们同样在争分夺秒、众志成城地进行一场考研的攻坚战。

刚刚跨入大四的门槛，全班大多数同学便心领神会、不约而同地树起了考研的战旗，且大有红旗插遍全中国之势。这期间，每个人都乐此不疲地向各招生院校发信，询问报考及相关事宜。卷入这支浩浩荡荡考研队伍当中的人，那些平时努力的同学并不惹人注目，他们报考自是在人意料之中。最令人称奇的，是班里的几位" 神仙"们也不能免俗。同窗近四年了，这些仁兄分别得封"睡仙"、"食仙"、"牌仙"绰号，盖因他们在相关领域有着"突出成就"。现在，甚至连神仙们都入了世，由此可见这场" 战争"的白热化程度。据截止目前的最新统计，在我班32位同学中，有考研意向的竟达28位之多。剩下的几位分别是委培生和体育特招生，他们旱涝保收，没有就业压力。

为什么考研这么热？尤其我所在的这个班里如此高温，其实用不着多问，这不是"和尚头上的虱子--明摆着"的事情嘛。可找的理由太多了，千条万条，有一条最关键，我们都图一个更好的发展。想想呆的学校，虽说是重点，但排在了榜上的老末。看看学的专业，名字挺动听（说实话，要不是它，我也不会当了一次大头鬼），却空而无物，华而不实。总而言之，当然从中毕业的学生就业不会好。考研，此时对我们来说就是一个改变现状，改变命运的绝佳选择！

值得庆幸，在"革命"形势风起云涌之时，同门前辈们当仁不让地为我们播撒火种。他们的切身体验，不管是成功的还是失败的，每一段经历都能让我们体味良久，深深思索。他们这样谆谆告诫，对待考研一定要早下手，早做准备，最忌犹犹豫豫，东挑西选，到头来虚耗时间，耽误大事。就我个人感受，这个时候听取学长们的经验谈，还是相当管用，至少在一些方面不会走弯路。比如，我接受了一位老兄的劝告，调低了自己原来的目标，保证高命中率。

自打响考研第一枪以后，班里出现了一个有趣的现象，同学走到一块儿，三句话就离不开有关考研的话题。有细心者作了这样的统计：交谈中出现频率最高的一句话是" 你准备报哪个学校和专业？"；另一句"你和学校研招办及导师联系了吗？"居次席；还有一句"你的考试指定教材都找齐了吗？"排在了第三名。有人于是戏称，我们都得了" 考研综合症"。相信，随着进入考研的第二阶段，新的话题将会不断出现，像报哪个考研辅导班，用哪本考研辅导教材等必然成为我们考研一族新的关注焦点。

考研，到底是一件幸事，或是一个悲哀呢？作为千千万万为此奋斗的学子中的一分子，我，有时会萌发出上面的念头……。也许，没有人能够说得清。但正如有先行者说的那样，考研是属于年轻岁月的一种攀登。那么，时不我待，就让我们大家动员起来， 将考研进行到底， 共同为了美好的明天去拼搏！
