路真的很长

http://www.kaoyan.com 2000年06月28日11:59:43 考研论坛

作者：kaoyan.com网友raymon
　　当我为了我的理想而不顾一切放弃了我的工作的时候，是多么的慷慨激昂，＂这条路已经有很多人成功的走过，我为什么不能走呢？＂我反反复复的问自己，问昔日好友，当然回答都是不尽相同的，友人说＂成功的理由是一样的，但失败的理由确是因人而异的．＂我走的路对吗？我的选择正确吗？我会成功吗？＂我真的很迷茫！
　　路真的很长！当我安静的校园的角落里去为我的未来而努力的时候，当我看到校园里的大学生快乐的生活着！在这个完全陌生的环境里，没有人认识我，没有人注意到我这个寂寞的考研人．有时觉得自己很酷！但太寂寞了，有一种失落感，有时真的很怀恋我的大学时代，如果上天让我在再次回到我的大学时代的话，我不会在因为那些无聊的事去浪费我的青春，荒废我的未来．不会让现在的我在为那时的稚嫩的行为而感到后悔．
　　路真的很长！复习已经有一段时间了，但是复习并不是想象的那样顺利，首先数学已经有几年没有碰过了，复习起来并不顺手，时常被卡，英语和政治都还罢了，有时真的想放弃，但后路已断，只能向前，还好有我们的考研论坛，每次想放弃的时候，我都会来看看战友们的帖子，再次鼓起勇气，在此我非常感谢各位战友给我的帮助．
　　路真的很长！我不知道在这里是不是有很多和我一样的寂寞的考研者，为考研而一怒之下断了后路，而后躲在一个寂寞的角落里为着自己的梦想而奋斗的．让我们携起手来，让我们不在孤独，不在寂寞．坚持到底，为着共同的理想而奋斗．谢谢大家．
