理智与情感－考研后记(二)

http://www.kaoyan.com 2000年06月28日12:13:26 考研论坛

作者：kaoyan.com网友火狐2000
　　　　
　　99年7月23日我回到了成都，回到了我梦中期待了无数次的地方。
　　记得回来那天是一个很好的天气，天高气爽，蓝蓝的天，并不是很热。8次还每进成都的时候，我就趴在车窗上看个不停。那就是梦中的川西平原啊，绿绿的油菜，金黄色的油菜花，在早晨6 .7点钟的朝阳中，显得是那么美丽。我贪婪的看着。这美丽的景色与进川前所见的秦岭景色迥然不同。从火车车窗望出去，一望无际的川西平原是一片充满生机和活力的绿色。金黄色的朝阳慷慨的撒在这片刚刚醒过来的土地上。
　　这就是成都啊，是我生活了四年的地方啊，而今我又回来了。
　　蓦然之间，一个影子又在我心中跳了出来。我回到了成都，我将如何面对她？
　　这两年以来，她对我应该说还可以，但是这真的就是一个好的开始的信号吗？
　　如果只是朋友，我肯定会毫不犹豫的马上去找她，但是连我自己的心里都清楚，我所期望的决不仅仅是朋友啊。
　　她曾不止一次的对我说：我们是最好的朋友。在信中，她还给我写过这样的句子：无论风雨岁月如何变换，我们的眼中始终有彩虹。
　　回忆和现实始终在我认识她的过程中交织这，即使我理智上清楚，我的追逐是永远没有结果的虚幻，我还是醉心与她给我的每一次惊喜，其实那些根本就算不上什么惊喜，我只是按自己的期望给它加上一些美好的色彩，希望会有见到彩虹的那一天。
　　事后我回忆起来，我才深深的为我自己悲哀：爱一个人，既然肯吃这么多苦，那是一种什么样的力量啊。
可惜，这种爱，在那个成都女孩那里又为我带来了什么呢？
　　抛开这些，应该说我回到成都，回到母校，还是很愉快的。23号那天，是我以前的两个同寝室同学来接我的。
　　老同学见面，分外高兴，短短的两年，我们都没有改变的太多，岁月的风霜再我们的身上并没有留下太多的痕迹。我们都还年轻，而年轻就是最大的资本，我们都有很好的未来。
　　坐在同学开来的车上，我们离开了火车站。车开在一环路上，我看着路旁熟悉而有陌生的街道。店铺。楼房，我知道这又是一段生活了。
　　我觉得那天的明媚天气，可能预示着我的成都考研之路最终是平坦的，虽然出了一点问题，最终我还是得偿所愿，得到了一个光明的结局。
　　我住进了同学的宿舍，研究生的宿舍条件应该说一般，连电话都没有，我惊诧于我们学校给高素质人才所创造的条件，再今天很多本科生都有了电话和电视的时候，这里居然打一个电话都这么困难
　　学校只是后来装了20部电话，而整个学校可有一万多名同学啊。
　　可惜那个时候，我对自己母校的种种弊端尚未认识到，就象我上没有认识到那一段感情最终给我到来痛苦远远多于欢乐。
　　一个人，只有他用自己的理智，客观的分析问题，敢于直面自己的痛苦的时候，他才可以说走向成熟了。
回到学校，我马上开始了复习。暑假里人很少，很是情静正适合复习。我先是再系楼里复习，过了没几天我就到了图书馆。
　　记得第一天到图书馆，我的心情很复杂，一种很复杂的感觉充斥这我的头脑。就是在这里，就是在图书馆，我认识了她，从此，也开始了一场爱痛交织的长跑。
　　那个95年小雨飘飞的夜晚，有在我的脑中渐渐清晰....

　　你不能这样！我在提醒我自己。我清楚一旦陷进这种感伤的情绪中，我将彻底完了。不要说考研，连平静的生活也将不可得。我太清楚我自己大学得后两年都是如何度过得了，我实在不想再那样庸庸碌碌，混混噩垩。
　　幸好我终于控制住了自己，我的心情渐渐平静了下来，不管我周围都是什么样的人，我的学习效率都极高，心情也很愉快。
　　现在想起来，再单位呆了两年，才知道自己是如何的不足，才知道有一个良好的学习环境是多么的不容易，大概我平和的心境和这有很大的关系。

　　如果一个人，从没有经历过失去，旧不会知道拥有的可贵。就象那些始终呆在学校里我的同学一样，虽然身为研究生，有这良好的学习环境，就不会觉得这有多可贵，多值得珍惜。
　　其实细细想来，世界上的很多事，莫不如此，如果我们对每件事都能倍加珍惜，每件时都能尽到自己的最大努力，我想当我们离开这个世界的时候，我们大概就不会有太多的遗憾了。

　　可是旧因为我们的惰性和劣根性，我们常常阴差阳错，我们常常后悔莫及，我们常常要让自己强迫忘记前尘往事。
　　幸好，99年考研的这段时间，我市尽了自己的最大努力，即使后来受了感情困扰，我依然在朋友的关心和帮助下，挺了过来，而且获得了挑战困难的巨大勇气。

　　这也是我的一笔莫大财富。
　　刚开始复习的时候，我倒没觉得有多困难，一切都进行的很顺利。我的英语一直很好，所以我对英语一直不害怕，事实证明我的直觉是对的。我一直牢牢的抓单词和阅读，我知道这两样抓住，英语旧差不多了，象写作和语法也极依赖单词和阅读。
　　开始的时候，时间都花在了英语和数学上。数学仍了三四年，检起来很是痛苦，我高数。概率。线代一起上，先检教材看。高数还可以，以前学的可以，线代和概率比较痛苦，尤其是线形代数，以前等于没学，好多东西如醍醐灌顶一样，豁然明白了，这才领略到数学的美感。

　　这并不是瞎说，在复习的过程中，我的确很多时候都觉得数学的重要性，也觉得它太有用了，这得益于我的两年工作生涯，它让我找到了学习的方向。
　　这也说明和实际结合起来的学习才能有动力和兴趣。
　　可是那段时间我的心情虽然很好，我的胃却很疼。
　　胃是96年的夏天开始疼的，不知不觉就疼了。我以前吃饭很快，有时早上也不吃早饭，我想胃肯定就是那样坏了把。
　　我的胃疼，我的头脑却异常清醒，知道我不应该去找她，自从我六月份给她写了一封信之后，我就从她 的世界消失了。
　　我真的是没有办法。我是回成都，她也在这座城市里，里我的学校不过骑自行车20分钟的时间。如果我告诉她 我回到成都，我就会去找她，如果去找她，我就完了。尽管临离开天津的时候，她让我给她打一个传呼，尽管我觉的99年的上半年是我最了解她时期，我已经漫漫了解她了，尽管她的信里出现了以前从没有过的温柔，尽管五一我在青岛完的时候，还在关心着她，还给她买了一只兔子，并且我也把那只翡翠的绿色兔子带到成都来了。
　　尽管，我知道，我很爱她，很爱她。
