为爸爸祈祷--爸爸用生命换我考研

http://www.kaoyan.com 2000年06月28日12:11:14 考研论坛

作者：kaoyan.com网友bertmail
　　爸爸是个真正的男人.爸爸用自己的毅力养着5口人的家庭
　　还有爷爷奶奶
　　我在外上大学,爸爸节衣缩食,给爷爷奶奶买可口的饭食,可是爸爸却吃他们剩下的,剩得少了,就多买几个馒头,我知道,他有时候吃不饱. .. ...
　　为了挣些入井费,爸爸天天下井,爸爸是普通的煤炭工人
　　就这样,病魔还是不肯放过这个好爸爸
　　我知道,这一切的一切,只是因为他吃不好,还有为了在外上学的儿子能吃饱穿暖,能安心读书.那一天,我回到了家乡,爸爸要手术了,是我去签字的. 医生讲了列出的可能的危险,我怕极了,可是我有勇气,我签了.
　　在爸爸进手术室之前,爸爸看着窗外好长时间......
　　一万多的手术费全是借来的,当时家里只有几十圆,是全家的生活费.
　　是我推着手术车把爸爸送进手术室的.爸爸抓我的手不肯松开,我知道,也许,这就是最后......
　　妈妈,叔叔,妹妹都哭了
　　我咬着牙没有哭,我知道,爸爸要我坚强!手术进行了6个小时,我为爸爸祈祷了360分钟,期间我给一个同学打了电话,控制不住自己就哭了. 想起来妈妈给医生跪下, 想起来爸爸进手术室前的表情,想起来和爸爸走过的路......
　　爸爸从手术室出来了面色苍白,嘴里鼻子里都是管子,呼吸急促,混身发冷,紧握着拳头,还有就--
　　爸爸坚强的脸上挂着两行未干的眼泪......
　　就是那两行眼泪,深深地印在我的心里
　　我再也忍不住了,跑到一边的楼梯上,痛痛地哭了,这是我生命中最刻骨铭心的哭,往常的爸爸在心中,可是现在的爸爸......真的55555 5555 555555
　　回到病房,看见爸爸的胸腔管还在滴血,爸爸呼吸还很急促,还没有苏醒过来
　　我又走了
　　是爸爸知道我学习紧张赶我走的,我在火车上望着故乡,又哭了.
　　爸爸要休养半年才会恢复,着半年,他还在为我们的生计操劳,妈妈有时候憋不住告诉我:爸爸吃饭时有时还是吃一口噎一口.还不到半年, 爸爸就坚强地要去上班, 给我们挣钱,供我上学
　　我要考研,是我的实力,也是爸爸要我做的,爸爸说:只要你能上,绝不让你因为经济困难而辍学.
　　每当爸爸打电话给我时,总问到吃得好不好,还说家里都好,其实我知道,仍旧是节衣缩食,还债,供我上学
　　我要做个争气的儿子,不要让爸爸失望
　　因为我知道,在我的每一步路上.都是爸爸用生命换来的.
　　......
　　真的,我不能再说下去了.
　　最后,还是祝爸爸平安,健康.
　　"爸爸,儿子会争气,会学你做个真正的男子汗的"
