其实，几乎每个想考研的人都有这样那样的顾虑，担心这，害怕那，这些都是我们思想上的障碍，都是我们前进道路上的绊脚石。朋友，真的想考研吗？那就下定决心，排除万难！
世界上毕竟没有十全十美的东西，我们应该考虑清楚什么才是我们真正想去做的，什么才是对我们最重要的，患得患失最终只能原地踏步。
“如果你已决定稳稳当当这样下去，那么你就已作出了一个基本的打算：你打算不再发展。”
“如果你不能采取行动，你以后生活中对命运的抱怨会对你的心情和健康产生不良影响。”
我也才刚参加工作不到一年，虽然这样过一辈子也还算
可以，但我不愿这样一成不变的下去，我还年轻，我还有追求！我认为，在确定了我们的追求后，真正地去采取行动，朝着这个方向努力，我们才会真正地感到快乐！
让我们共同努力，去实现我们的梦想吧！
