jindian:考研是需要韧心的，我非常认同这种执著，如果能
始终用这种执著来面对我们生活和工作中的种种挑战，
我相信是无往不胜的。
我也是刚毕业的，到一家石油公司做推销员的工作，
我自嘲自己是“卖油郎”，这份工作可以说非常
不适合我，因为我是个喜欢静下心做事情的人。我跑了三个多月一点头绪也没有，
说实在的，这三个月我是一直在做自己的思想工作，
我想过很多辞职去干别的的想法，可是种种原因我必须坚持下去，这种坚持对我来说是非常痛苦的，
在单位里仍旧是个什么都不懂的新人，是别人照顾的
对象，这对于我是无法忍受的。今年我们科长给我
定了指标，不算高也不算低，于是我真正地开始
了那种推销员的工作，有不断奔波的疲惫，有被人
一口回绝的痛苦，当然有我跑到第一笔业务的狂喜和
被人认同的微妙感觉。
可是，我知道我不太可能一直做这份工作，我的心情
一直没有工作的快乐感，但是我知道我如果没有完成
指标而放弃这份工作，我今后就始终有这份阴影，
我今后的工作中面对挑战时就会故伎重演，现在我
就在凭着这个信念工作，我相信我一定不会输，
我也祝愿所有选择考研特别是辞职考研的朋友们能够
成功！因为我们都是在拼搏的人！
拼搏着是美丽的！！！
