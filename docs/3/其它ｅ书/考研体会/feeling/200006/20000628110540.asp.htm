别犹豫，时间又在飞快地跑了！

http://www.kaoyan.com 2000年06月28日11:05:40 考研论坛

作者：kaoyan.com网友别后悔
我也是一个1976年生的MM，刚满24，毕业两年，不同的是我从一个三流学校的专科毕业。其实，我从不忍提起我的高中时代，因为，当年脆弱的我经受不了一定点的挫折，老师的误解和专制竞让我一蹶不振，在高中三年里，我从未认真学习过，并以弄差学习成绩为任务，呵呵，很可笑吧，居然有人拿自己的前途开玩笑！我本来是一个跳级生，跳级后居然也拿年级第一，从不参加数学竞赛的辅导班，却能在竞赛里拿第一，可是后来，我却参加了三次高考。第一次，本来是要考美术，但心里无法割舍对数学和物理的爱而考了理科。因为从很小的时候起别人就认为我只要愿意完全可以当个画家，也完全可以当个科学家，可惜，当画家不是我的理想，我从小一心只想当个科学家（只是在以后才明白，科学家不是那么轻易就能当上的），在那昏暗的三年里，我却从未认真地学过，哈，哪有爱而不去学的？我就是这样一个人，真是昏了头，况且，这一年我的父亲患了重病，家里一团糟，根本无心读书。第二年，家庭状况一直不好，我的父亲那时仍未恢复健康，我对未来其实很模糊，情绪也不稳定，学习还未进状态，又落榜了。其实，我知道，外界都不是主要原因，应该怪自己的脆弱，我想也许没人能知道那几年我灵魂的游离，那几年可以说是我的大脑一片空白。第三年，我开始认认真真得读了，也许老天故意捉弄我，让我在那一年里大病小病不断，根本不能坚持正常学习，在高考的三天里我带病上场，最后，考的却是一个三流学校的专科！录取通知书来的那天，我大哭一场。从此痛恨考试！不过，幸运的是，我居然发现我很爱我所学的专业：计算机。因为痛恨考试，我的学业成绩并不突出，但某些东西我认认真真地钻研着，而且我的动手能力也越来越好。毕业后，我来到了现在的国企单位，也算混得不错，但我的专业在工作中还用得不多，而且层次交浅，我的工作主要是搞工业控制，所以专业在很大程度上变成了业余爱好。我知道这不是我的理想所在，我应该还能更上一层楼，我应该学到更多的东西。在我本命年到来的时候，我才蓦然惊醒，原来岁月已经毫不留情地走了，而我距我的理想、梦想却越来越远！我的同学中有的已快研究生毕业，有的已结婚生子，而我？还是不能再等待了，如果现在不努力，那失去的更加不会回来！我太爱计算机了，如果我这一辈子不能从事自己喜欢职业，自己想过的生活，我将郁郁终老！我知道，在2 4岁的这一年里，还有很多其他美好的东西，但是对于我而言，理想才是第一重要的！想想看，人生还有几十年，为了今后那么长的日子，舍弃一些又怕什么呢？看了叶颜和黄海风的帖子，才发现自己并不孤独啊，原来还有那么多和我一样心中怀有理想的同龄人！不要再犹豫了，一犹豫时间又飞快地跑了！
