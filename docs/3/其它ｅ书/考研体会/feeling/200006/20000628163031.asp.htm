公务员，研究生，如何选择？

http://www.kaoyan.com 2000年06月28日16:30:31 考研论坛

各位网友：
我95年大学毕业，由于学的是机电一体化专业，被分到一个国营企业，在那里度过了两年半的痛苦岁月，其间，考过一次研，但由于准备不充分，差几分。也是在这期间，我决定改行搞计算机，并于9 6年考过了程序员。9 7年，一次偶然的机会，一个在省城区政府的原工厂同事让我到他单位帮忙搞计算机网络，至此以后，我经常到他们单位上修电脑，调网络。9 7年底，我被掉到了现工作单位，从事计算机网络管理和政府网站的制作。开始，还是很新鲜，干劲十足，但久以后觉得没多大意思，并且经常要做很多杂事，如：发传真、搞工作安排表、搞会务（不知道机关为什么有那么多会）、写标语等等。更为重要的是，我渐渐发现向我这样高技术的，既没后台，又不会搞人际关系，在机关很难有出头之日。
98年底，我又燃起考研梦，但这次我改报了管理信息系统，没有报计算机是因为计算机考的专业科对我来说比较难，竞争也很激烈。可能是老天故意捉弄我，考试分数下来英语差一分，我四处奔走，结果还是以失望告终。
99年，我在机关的工作有了些进展，成立了信息中心，单位领导也开始关心计算机、上网。由于我工作比较勤奋，同事们和一些领导对我还比较好。但考研的梦一直在我头脑中缠绕，也有人劝我读个研究生班算了，但看到我原来的一些同学和同事都在读研，我心里很不甘心，于是，9 9年底，我又报了统考，学校和专业没变。
2000年，我的工作又有了新的机遇，我们区里准备建大学科技园，我被借调到科技园信息部，负责科技园创业楼的网络建设和管理。而正好这时，考研的分数下来，我考了3 83分，很可能被中南大学管理系录取。这时候，我既高兴又忧虑。为了两全其美，我想读在职的，学校同意了，说要单位同意并签协议。我马上找单位，不知是我太心急还是命中注定，无意中我得罪的我的主管领导，并且他是一个很固执的人，我多方劝说，他还是不同意，而没他的同意，我就签不了协议。（后来想了想，其实他以前就对我有成见，只是我没注意）于是，我面临选择：是脱产去学习，还是放弃研究生，好好工作？两者对
我来说都来之不易。工作，当一名公务员，稳定、待遇还可以、社会地位较高、工作压力不大，是很多人羡慕的职业，就算我研究生毕业后未必能找到这样的工作了。研究生，是我多年的梦想，我当然想读，但要我放弃工作，一方面年龄比较大（2 8了）毕业就30了，还要去找工作；另一方面，我不是一个敢闯敢干的人，要再去打工，从头
开始，我有点心虚。
而我必须选择。
希望大家看了我的这篇说明后能理解我的苦衷，提些意见和建议，谢谢。
beenjoy
