读研的痛苦----个医科基础类研究生的悲哀

http://www.kaoyan.com 2000年07月02日10:06:08 考研论坛

读研似乎现在是很热潮，可是读研中的酸甜苦辣又有谁知道？
能够作为一个研究生目前好象是值得好多人羡慕，尤其是对那些渴望读研的人来说已经够幸运的，还叫什么痛苦？可是读研所要付出的代价又有多大，却无人理会。
作为一个医科研究生所要担心的事情如果罗列一下可以举出很多很多，如：
1，学习的压力：通过了考研，考试似乎不在话下，可实际上进入学校以后，各种考试同样也能压的你喘不过气来。
2，实验的压力：对于医科研究生来说，尤其是对基础学科的研究生，实验的压力简直令人难以想象，你要随时担心今天的实验是否成功，花了老板的钱没有理想的结果怎么交差，明天的实验又会怎样，三年研究生的生活能让你老五年。
3，分配的压力：三年研究生读完会到什么单位，在找到工作之前你绝对不会知道，而对学习基础的医科生来说，只有两条路一是苦读GRE和TOFEL准备出国扒洋分；二是继续读博士逃避找工作，三年后仍是出国扒洋分，如果这两条路你都不行的话，一个不太美妙的结局就会等着你，你只好找一家高校混日子，拿着可怜的一点工资等待学校哪一天开恩分给你一间住房，再等哪一天LUCK来了，让你有机会出国扒洋分，改善一下生活，然后回来做教授。而实际上你只能幻想总有一天能有自己的课题，自己的实验室，但如果你不出国这也只是一个幻想。
4，经济上的压力：每个月你只有2、300元的收入，，连自己吃饭都是问题，更不要说还要负担书费、交通费及学校的各项不合理收费如高昂的住宿费（这一项就要去掉你每月收入的一半），如果你还有家庭，假如他也在读研，或者你有孩子，每个月经济压力就会让你彻底崩溃。读研的人很多年龄都不会太小，有很多都会碰到上述的情况。我也是其中的一员。我的妻子在读硕士，我正要读博士我俩加在一块的收入每月不到RMB700元，都近三十岁的人还要向父母伸手要钱，真是觉得无地自容。幸运的是我们还没有孩子，虽然我们的年龄都已不小，但是我们根本不敢要。有人也许会说可以做兼职补贴吗，先不说找不找的到兼职工作，就是有一份兼职工作，沉重的学习负担和实验工作也会让你两头都顾不上，结果说不定你连一样工作都做不好，而学位比面子要重要的的多。（这可和某些学习计算机的人不可相比）幸好我们的家庭条件尚过得去，我们也工作过尚有点积蓄，能让我们在学习期间填饱肚子，否则这书是读不下去了。
这只是我个人所碰到的一些事情，但我的同学中（当然大部分人都是基础学科的研究生）象我这样的不在少数，我只是将我们共同的一些经历写出来，希望给狂热考研的人一些心理准备，尤其是给那些准备读基础类学科的人提个醒，也希望有人能理解我们。
