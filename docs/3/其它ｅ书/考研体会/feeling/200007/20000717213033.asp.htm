考研让爱情走开

http://www.kaoyan.com 2000年07月17日21:30:33 考研论坛

作者：kaoyan.com网友qbbluesky

记得有句话。战争让女人走开
我要说。考研让爱情走开
我不否认有人由于有爱情的滋润。或是受到伤害后发奋考研
但是对于大多数人而言。谈恋爱会影响考研。
第一。恋爱中的人如果不能天天在一起。就会花一些时间在想念对方上。如果是同学可以在一起学习也会分心。我有个同学。在图书馆自习时特地不和女友坐在一起。
第二。我认为世界上最难把握最微妙的就是人的感情。
恋爱中的人尤其如此。有谁能保证爱情的路上一帆风顺出了问题会影响心情影响效率。
第三。我认为一个人在一段时间内。能把一件事情办好就不错了。鱼和熊掌不是不能兼得。但是能做到的人是少之又少。
当然。我不是让正在谈恋爱的人都分手。这只是我个人的一些观点仅供参考。
