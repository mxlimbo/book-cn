放弃考研，在最后的一刻―― 一位法硕追求者的心路历程

http://www.kaoyan.com 2001年01月15日21:41:23 kaoyan.com网友 俞

　　真的有太多的话要说，却迟迟不知如何下笔，HORACE发最后一帖时的心情想来也是如此吧，这应该会是我在这块我所挚爱的菜地上的最后一帖了，在我心目中这是一块多么神圣，光荣的土地啊，她只属于那些为理想奋斗不息，坚持不懈的强者，而我只能带着满身的失落和孤寂痛楚的心屈服于严酷的现实，尽心地去当一名人民教师，我已不再有资格成为这里的一员了，尽管这里的一切是多么的令我留恋，尽管我已习惯一上网就迫不及待地在IE的地址栏中输入WWW.KAOYAN.COM，我无数次地感动于一篇篇深刻优美的文章，我更珍惜与一帮未曾谋面为共同理想而奋斗的网友们的友谊，HORACE，白菜，小林，木木，还猪格格，翔宇，阿飞，还有很多，也许你们根本就不知道我，但是在我心中，你们是最纯最纯的朋友。我很少在菜地灌水，因为自惭形秽于自己拙劣的文笔，但今天我一定要写些什么，这三天对我来说实在太过难熬，我有太多太多的理由去说服别人，但我终究过不了自己这一关。
　 有太多的人说考研太苦，太累，我不在乎，从大二开始起，我就崇拜李敖和李阳，强迫自己过一种特立独行的生活，有这样一段谚语：种下行动的种子，你会收获习惯，种下习惯的种子，你收获的是命运！一段日子后，我就养成了以苦为乐的生活习惯，达到了享受辛苦的境界，人的改变更多的是一种思维方式上的革新。我是以一种苦中作乐的心态来考研的，我的终极目标永远是自我的完善和提高，我受日本作家阿部次郎的著作“三太郎的日记”影响很深，终生的“自我修养主义”是我最大的追求。从去年的年初就开始备考，暑假单身去北京上任汝芬的政治班，受益颇多。我很欣赏西安交大政治的辅导模式，每一位老师都有丰富的文史哲知识，听后的收获绝对在考研之外。事实上见到任老师前，一直以为他是LADY，没想到是个神气的小老头，我对他是极为欣赏的，不知为何菜地的政治辅导老师评选中将其排除在外，IT IS UNFAIR！北京之行有两大遗憾，一是没见到慕名以久的HORACE，当时他实在太忙，另一个是未上长城，这个遗憾是我自己留给自己的，我郑重地对自己说：不到长城非好汉，不入北大（人大）非俞//！！想想当时是多么地意气奋发，不过当时我有足够的理由自信，我辅修了近两年的法学（我们的教材是法律出版社的，比法硕的教材不知难多少，特别是梁慧星的民法）我也自感学的不错，我是英语专业的，英语应该有优势，政治我觉的不难，现在想来是大二，大三时的泡图书馆起的作用，政治除了考死记硬背之外，更重要的是一个人的思维方式，分析解决问题的方法，这在不定项选择中最为明显，而这一点靠辅导书是学不到的。我最喜欢做不定项，好比猫捉老鼠的游戏，十分过瘾。龙十一，任汝芬的模题我的不定项错误率控制在10%-15%。一切都在良好的运行着，尽管我对人大，北大还有一种潜意识上的敬畏，所以我多次让HORACE帮我驱除这种消极意识。我几乎为这次考研付出了一切，无休止的逃课，在外住宿，我放弃了普通话水平测试，英语专业八级的辅导，试讲的准备等等。长时间的复习也只能用苦中有乐来形容，法硕的教材我做了三遍笔记，政治背了两遍，英语的词汇表我一个一个单词查字典，老石的阅读也做了近一半。时间流逝至10月底，当我犹豫于自己是报北大还是人大时，当我对自己越来越有自信的时候，甚至当我在学习之余已开始憧憬于研究生的美妙时光时，我忽略最多的家庭却给了我致命的一击。
　 我是家里的独生子，父母都是双职工，也都无例外地下岗，他们很宠我，他们对我没有什么要求，对我则几乎有求必应，我对他们只有索取没有回报，我只想到自己的考研父母一定会支持，我考上他们一定很高兴，暑假前的备考完全是自发的，在外住宿的钱也是自己节省下来的（事实上辅修法学的2000块钱也是我省的），暑假要去北京前才告诉他们考研的事，当时他们表示出一些犹豫我已经感到很不可思议（我原本以为他们会很惊喜），但因为我当时的极大决心，他们同意了。之后我一心备考，父母也没说什么，我们表面上达成了一致。直到10月27号，我的生日前夕，我在网上看到了一篇文章，是一位女孩的考研经历，她家境不好，于是她毅然放弃大四考研计划，而是找了分工作，两年之后用自己的钱去考研读研。那一刻，我忽然发现自己竟然连家里的经济情况都不了解的情况下就决定考法硕是一种多么自私的行为，我为自己感到羞耻。生日那晚，我郑重的向父母提出了要求：我已经22周岁了，我有权知道家里的经济情况。那一晚，我无法入眠，这个生日没有鲜花和蛋糕，但却是如此的刻骨铭心（我的20岁生日是另一种的刻骨铭心）。我发现一夜之间自己长大了许多，成熟了许多，我告诉自己不能再这样自私下去了。从那时起，我的考研不再执著，从那时起，在我的内心中，今年的考研已是一次注定的没有结果的尝试，从那时起，我开始患上了以前一直难以理解的成功恐惧症，我甚至觉得自己有些变态，那时复习备考已成为一种习惯，我就故意放纵自己，很普遍的情况是很好的完成了一些学习任务，立即有一种莫名的恐惧涌上心头，于是立即恋恋不舍地扔下书，去上网，玩游戏，看电视，泡图书馆，就这样循环往复，直到12月。我还是自己报名了，填了人大，我不知道是什么驱使我去报名的，我只想全过程地经历完考研，毕竟过去的一年我付出的太多太多。12月，我终于将好不容易养成的考研生活模式击的粉碎，我开始将目光转向择业，我做推荐材料，跑学校，准备试讲，尽管每天我还会惯性的翻翻书，12月27号，我去本市一所最好的职高试讲，得了优秀，受到学生老师的一致好评，28号，我在协约书上签了字，我不知道是不是造化弄人，当我右手拿着签好的协议书回寝室时，左手则收到了刚寄到的人大准考证。那一天，我一整天都在笑，我强迫自己表现的开心，但又有谁知道我的心在流血，又有谁知道半夜酒醒后的我泪眼朦胧，半睡半醒间仿佛看到人大离我远去了，我奋力去追，却越追越远。
　 新世纪到了，我尽力收拾自己残破的心境，让自己以全新的姿态去迎接一个崭新的开始，这也算是我自己给自己找个理由来逃避吧，但我仍然在看书，我依然决心进考场，哪怕只为了证明自己。但老天却连这样一个小小的愿望也不让我实现，1月7日，职高要我过去，1月12日-1月18日要我代讲一个星期的课，我能说什么，领导说大学有事的话我去给你请假，而家里的事能请假3天吗，考研更是不能提，因为领导当初录用我时就很担心我跳槽，我是男生，英语口语又挺好。我彻底屈服了，之后的一周我过的极为痛苦，脑子里总装着考研，法硕，人大。12日，我去职高讲课，一天上下午四节课，我拖着疲惫的身躯去看了本市的考研考场，又是一个令人心痛的巧合，考场也是一所职高，而且和我上课的职高隔江对望。12日晚，我不偏不倚在那天晚上做了这样一个梦：13日，我去了考场，我目送着其他几位同学进了考场，这时忽然出现了许多知道我考研的人，他们问我为何还不进考场，我说不考了，这时他们反映各异，有的惋惜，有的愤怒，有的惊奇，我想逃避，我想找个地方发泄，对，我要去海边，那里没有其他人，但是我一开始进了一个村，迎面过来一个学生，恭敬地叫了我一声俞老师，我继续逃，终于听到了水声，我看到海了，我再也忍不住了，我用尽全力地对着大海，对着天空大喊，直至声嘶力竭，喊累了，我靠着岩石坐下来，所有的委屈和痛苦一下子涌了上来，我抛开一切抱头痛哭，似乎要把这一年多来所有的苦，所有的痛，所有的累，所有的屈都哭出来，猛然间，我醒了，我竟然哭醒了，天呢，梦回子夜，惟有泪千行！！去年的此时我是死也想不到会这样睁着双眼迎来2001年1月13日，这神圣的一天。13日早上8点整，我羡慕地目送着一位位考研同仁带着各种表情步入考场，我只是远远的站着，衷心地为他们祈福，8点30分，考试铃声响起，那一刹那，我有一股强烈的冲动冲入考场，但我还是收住了脚步，将人大的准考证郑重地放入口袋，随手拿起高一教科书，留恋地望了考场最后一眼，向着江对岸骑去，那里还有一群可爱的孩子在等待着我，只是此时双眼早已泪如泉涌。
　　 今天是14号，各位考友正与民法，刑法“恶斗”吧，而我这个可耻的逃兵也只有在这个时间来菜地做祈祷，让自己破碎的心好受些，明天此时，这里一定会很热闹，只是这种收获后的喜悦，热闹不属于我这样一个失败者，眼泪已不知不觉流了下来，外面原本晴朗的天竟然飞起了雪花，是到了该说再见的时候了，别了，我挚爱的菜地，别了，我亲爱的法硕兄弟们，别了，真诚可敬的HORACE，白菜，小林，别了，红火的法硕论坛，别了，我一生难忘的考研岁月！！！！！！！我真诚地祝福你们，为理想而坚持到底的你们――成功！知道吗，我本来多想接过HORACE的班，为大家做些什么，只因为我们是一家人，相亲相爱的一家人！
　　 说声再见，希望不是永别，也许两年后，也许更久以后，我会回来，当我自己能承担自己的时候，当我再一次决心攀登自己人生长城的时候。终有一天，我会再去北京，我不会忘记自己对自己的承诺：不到长城非好汉！！！　　　　　　　THE END
