我在考研路上（三）━导师与我

http://www.kaoyan.com 2000年08月01日09:02:19 考研论坛

作者：kaoyan.com网友liyan


有的同学肯定现在还在为到底是否联系导师而犯愁，可我没有听没人的意见（事先联系导师不好），而按照自己的情况与老师见了面。在此一些经验仅供大家参考。
我是在一个偶然的机会认识了我的导师，因为我所要报考的专业不是我的本科专业，而是我校外语学院的某语系专业，并且我校外语学院有下列规定，不许非外语学院学生旁听其专业课，不许非外语学学生进入外语楼。因此我考研的难度是相当大的。我校大三学生的外语课已经停课，像我这种打算学专业外语的人如果没有外语课上真是一件很痛苦的事，所以我找机会一直在和9 8届的同学一起上公共外语课，虽然不是我所想听的内容，但是我想一定会有所得的。没想到，误打正着。我所要报考专业的导师正在带这批学生，因此经过几次与老师的主动交谈，和老师混得很熟。但是我一点我很清楚，与导师关系再好，也没什么用。（这有用没用当然是指真正考研的时候了）老师不一定参与出题，更不一定参与阅卷。但是有一点，和老师接触是非常非常有必要的，因为老师不仅可以提供给你及时的考研信息，而且也会在你需要指点的时候帮你一把，我觉得大学老师都是退很热心的，而且都很爱惜人材（那是真正的教授，而不是像有些人是披着人皮的禽兽）。所以在老师身边的几个月里收获颇丰。
我曾记得有的同学和我说，“考研吗，其实很简单，搞定导师也就什么都搞定了。“我觉得他是进入了一种误区，因为考研说到底还是一种实力的较量，能否取胜，主要取决于你自己。不要把外界因素考虑得太多，那样最后还会害了你。所以每走一步都需谨慎对待，这是我的一点经验之谈，望能与大家切磋。
