考研与八股

http://www.kaoyan.com 2000年08月08日11:11:37 考研论坛

作者：kaoyan.com网友chengcheng

前几天， 读书之余看了《读书》上的一篇文章，文章内容是讲科举与八股文的。
当然科举是什么不用我在这儿多说话，但是我看了感触很深。文章说科举令许多读书人疯狂至此，废尽心力，为了一举成名，在古代，有百岁的秀才来应试。而看不起科举的施耐庵后来也已“贡生”的名义为荣。在古代，读书人，并没有多大出路，　只有科举一条路。而在现代，许多年轻人没有出路，也只有考研一条路了。考研到了现在，已不是一种考试了。在许多人心中，考研成了一种人生寄托，　一种奋斗目标。考研已被异化。
考研与科举，　现代与古代，　到底是进步与退步呢？有时，我觉得现在考研之风胜行，　也是中国现在人才机制不行的一种表现。于是许多人没有出路，只有考研这样华山一条路。
象我，也是如此吧。不善于溜须拍马，不会踩人爬上去，而在我们单位，向来是凭关系。可是我又不甘心一辈子就这么当个小职员，希望换一个好点的地方，希望到一个自己能认真做事，发挥自己能力的地方，这个愿望是卑微的，也是渺芒的。但现今也只有考研这条路了。
每天晚上坐在灯前，奋力苦读。有时，面对那些课本，我不禁想，这与读八股有什么区别？写这篇文章也是对自己的嘲讽吧。
