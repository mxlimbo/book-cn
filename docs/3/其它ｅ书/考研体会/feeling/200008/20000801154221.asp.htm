阿飞的故事——献给考研加油站

http://www.kaoyan.com 2000年08月01日15:42:21 考研论坛

作者：kaoyan.com网友xieding

阿飞是我的呢称。

初到这个网站，是阿飞的第一次上网。那时，应该是去年底，阿飞刚刚买了猫，刚刚决定考研。阿飞毕业工作两年了，他觉得好象什么都没变，自己没变，朋友没变，事业没变。两年的生活对于阿飞来说，是个空心的梦，看上去繁华一片，骨子里充满堕落与颓废。因为，阿飞是在娱乐场所干了两年。这些生活跟阿飞的初衷是违背的，它脱离了阿飞的生活原则，他原是充满生气和向往的。

99年底，阿飞辞了工作。他独自坐在空空的二室一厅里，觉得伤心而自责。这套房子是阿飞的父母为他买的，所有家具，所有生活用品，甚至女人，阿飞觉得父母都买好了。阿飞的薪水从来不够用，虽然他很节约，可他太爱交朋友了，这让他花钱。每当钱不够的时候，阿飞总是能得到他父母的帮助，这让他痛苦，可实在。所以，阿飞决定考研。尽管这两个字阿飞从来就没想过会出现在他的身上。

春节后，阿飞把房子租了出去，把电脑卖了，把家具给了朋友，把他所熟悉的一切都抛弃，只身一人到了武汉。阿飞有个朋友在华工上班，他准备到那儿安心复习。阿飞心里其实是甜蜜的，他想他或许找到了出口，虽然朋友都说他荒唐，他自己也觉得不可思议，可他就这样来到了武汉。他是满怀壮志来的，下火车的时候他还觉得一切都太快，可他马上否定了自己的胡思乱想，他是来寻找出口的，他想，无论如何，他都要找到出口再回去。

两个星期后，阿飞开始记日记，在日记里，阿飞又开始伤感，他对自己简直失望透顶，没有进入学习状态，没有按照他想象的那样去生活。阿飞开始第二次上网。这让他有一种虚伪的满足。然后，开始第三次，第四次，···到第N 次。可是，阿飞的学习仍是没有进展，他开始怀疑自己，他不知道自己在做什么？不知道为了什么而做，他寻找什么？出口吗？还是另一种入口？

其实，阿飞一开始就没有意志力，他就是那种经不住诱惑的人，他喜欢热闹，可又安于平静。一旦生活泛起微微波澜，阿飞又喜欢遐想联翩。他觉得自己是莫里哀笔下的那个什么人物，整天就知道胡思乱想。明明这一切都是自作自受，可他就这么简单地任其所为。阿飞想自己真是无可救药了，刚来武汉的时候，自己还是一张白纸，认真点画，还可以出名的。可现在这张白纸已经被阿飞自己弄丢了。他不喜欢说迷失自我，他觉得他一直都知道自己在做什么，就是不知为了什么？

五一，阿飞回了一趟重庆。看见朋友们过着井然有序的生活，他觉得自己终究是堕落了。以前有人这样说他，他不承认。可这次，阿飞像透过重重的障碍，看见了自己的最深处。在漆黑的夜里，阿飞抱着一本英语单词书默默地哭了。在这种时刻，阿飞觉得往事应该像过电影一样从眼前掠过，可没有，阿飞只是伤心，他竭力想去回忆点什么，可什么也没想起。至少，他是哭了，这令他自己有点安慰，仿佛还没有泯灭良心似的，仿佛，他还可以重来，因为阿飞意识到自己到底该要什么了。

从重庆又回到武汉的第一天，阿飞在网上认识了斑竹，TOMSUN和另一个网友。这些人其实阿飞是熟悉的，只是没有认真说过话而已。在单一而枯燥的复习生活中，他们成了阿飞唯一的朋友，这是真话，他从心眼里感谢他们。在炎热而寂寞的武汉，阿飞觉得自己连过客都不是，不曾拥有什么，也不曾被谁拥有，在独自喝酒的时候，阿飞想起这些人，总是暖暖的，尽管他们看来仍是这么不真实。可就是这些不真实的朋友，给了阿飞学习的勇气和一种友谊。这些都是阿飞渴望的，他渴望生活充满激情，就像阿飞一直钟情于N OKIA一样。

可，明天。阿飞就要走了。

阿飞再一次坐到这里的时候，心就开始隐隐痛了起来。窗外仍是武汉炙热的太阳和被风吹扬的树荫。阿飞写到这里，鼻子就开始酸起来，这有点令他感到羞耻，他觉得他怎么能哭呢？要哭也应该是考研考上了或者没考上。这儿哭，一切都不对，一切都不对。

阿飞转过头去窗外，看见白煌煌的一片。阳光透过树枝，照在窗台上，那些树叶在阿飞的眼里脉络清晰，像阿飞的生命，阿飞的生活，阿飞的心里。都是清晰可见的。这个下午，阿飞仿佛看见了他那不算美丽但还差强人意的未来。

阿飞就是我，我叫xieding.

阿飞
