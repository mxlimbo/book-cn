投桃报李--我的考研路程

http://www.kaoyan.com 2000年08月17日14:34:58 考研论坛

作者：kaoyan.com网友小昭.

九月份就要去北京读研了,说起来,我也是个考研战争里的成功者.
在我复习考试的整个过程里,从网络上得到了很多帮助.本来是自己孤军奋战,上网之后才知道有那么多和我一样志同道合的人.各个考研站点给了我莫大的鼓励
我一直想总结一下自己的经验,从而回报给需要这些信息的人们.直到现在,才真正动笔.
我会唠唠叨叨的叙述我的经历的想法,希望对您有启发.

第一次准备考研----半路夭折

我93年考上大学,当时的想法是97年本科毕业,再上3年研究生,2000年正好硕士毕业.多么圆满.可惜,我的蓝图虽然画得不错,但从没认真设计过怎么让它一步步变成现实. 直到了大三暑假才开始复习,那时我回家学了一个署期的英语,效果挺好.9月回到学校,才发现别人的数学都复习了一遍或者两遍了.我顿时慌了手脚. 觉得来不及了,信心没了一大半.勉勉强强学到国庆节,基于以下原因,终于决心放弃考研:

第一,我没办法处理好复习和大四课程的关系.我从小是个乖乖女,听话习惯了.不敢翘课,去上课呢,又有没办法忽视老师讲课声音的打扰.看着其他同学如入无人之境的复习, 我心里慌张,矛盾的什么都干不成.

第二,大三一整年我学习的太辛苦.那一年,专业课很繁重,我又选了辅修专业,又考六级.很少有休息时间.觉得学习真是学烦了.

第三,信心不够.我开始惧怕数学.(要考数学一)死心的认为再努力复习也来不及了.

第四,我了解的信息太少,觉得六神无主.不知该买什么书,怎样进行复习.我太闭塞了.

决定放弃之后的日子过得很轻松.虽然朋友劝过我,我也不听.收获是专业课学得不错,辅修专业也过了.当然,到了考研成绩出来时,看着别人的喜怒哀乐, 我是真后悔过的.如果时间可以倒流,我定要坚持到底.

我的经验总结:复习应该在五月份就开始.特别是要考数一的和跨专业的朋友.

第二次考研----惨遭失败

97年毕业,工作之后的一年只知道玩,没出息极了.到98年夏天又想考.我的动力来自哪里呢?
我的大学比较有实力,同学们大多很出色,在各行各业混的都不错.在几次同学聚会之后,我立志决不能比别人干的差,这是最主要的动力让我不断鞭策自己.

我还有一个奇怪的理由:我一向认为我英语不错.(其实顶多算不差,连'好'都不敢当).我想要一个经常用英语的职位.正好当时工作也不是太合心意. 要辞职呢?自己也有点怕,父母也不支持,只好选择折衷的办法:考研.算曲线救国吧.

失败原因 (各位网友,万万不能重蹈覆辙!!)

第一:没创造良好的复习条件
单位规定不让考研,只好边上班边复习.还不能让同事们知道.白天看看专业课,晚上绝大部分时间复习数学.我最不明智的地方是我还住在集体宿舍. 这样躲躲藏藏的,晚上复习时间只有一两个小时.要是自复习开始就出去租房,就有成功可能.

第二,当时我使用别的单位开的介绍信去报的名.十一月份的时间浪费在报名上了.您也许觉得不可思议,可是,作为一个在职考生,看似简单的报名真的让我煞费心机.

第二,当时打算考同济大学.同济的两门专业课里包括了四门课程,其一我还没学过,余下的也是大学里几乎最难的三门课程.而我只能上班空闲时才能看看书.

第四,我不明智地参加了一个考研复习班.复习数学和政治.政治班没有太多的成效.数学班呢?当时我连大纲上最基本的知识点都没弄明白, 上课听讲时,根本不知道老师在讲什么,只能机械的记笔记.通常是:上节课的内容我还没搞懂,下节课又开始了.而且,上课占据了我宝贵的周末时间. 那时,周末两天可是我唯一能全心全意学习的时间啊.到最后我的数学只浏览过一遍内容.竟然从没做过任何练习题!! 再者,授课老师的水平也值得考虑.如果参加考研班,您一定要已经掌握了所有的知识点.在这种情况下跟着老师复习才有用.

就这样,在考研班结束后两周我就奔赴考场了,一直以来我都在工作,连考前看考场都是请人帮我去看的.

考完第一天的英语和政治,我没什么感觉.分数出来后,这两科都刚好考到99年分数线
第二天考数学,拿到试卷的那一刻我就知道我完蛋了,有把握的题不超过30分.这些题做完后我就没事干了,
看着别人奋笔疾书,我心里好难过啊.　然后我就考虑,专业课考试考不考了?我想,这么辛苦,都走到了这一步,一定要考完,看看我能惨到什么地步.我绝不是半途而废的人.
全考完之后,我虽料到了我的下场,还是忍不住要做白日梦,想象万一万一考上的情景.要是考上了,就太对不起在学校全天复习的同学了.当然了, 白日梦终究是白日梦.我去年的总分比分数线少了一百多分呢.(不好意思,真丢人)

虽然是这么令人羞愧的分数,我却没受到太大的打击.
在复习中,我体会到了学习的乐趣,很喜欢那种苦苦思索后的豁然开朗的感觉,好像我刚刚进入复习状态.
经过这次考试,我的大脑好像刚刚苏醒.不利用一下太可惜了.
我决定再考一次,最后一次.我不能忍受一次又一次的折磨.我会好好安排.我知道了我的弱点:时间太少了!!

第三次考研----目的达到了

这次考试,虽然最终是成功了,但我觉得有很多地方可以做得更好.

知道成绩已是四月份了,我并没有开始复习,其实休息一下是应该的,可我休息的时间太长了.
到五月份,还没决定是辞职孤注一掷还是象上次那样考.我很为难.
同时我挺欣赏的一个男子请求我做他的女朋友.我很矛盾.

怎么办呢?幸运的是我得到爱人的同时又得到他的支持.不幸的是,谈恋爱是很费心神的事情.我正式开始学习的日期是十一月三日,从五月开始的六个月黄金时间被我荒废了, 真是好了伤疤忘了痛.失败的教训全被我给吃回去了!万幸的是这段时间为我以后的全心全意的学习做了一切准备.

1.我联系好了一个安静的独居的住处,可以保证一定的时间.复习资料也准备齐全.

2.我拜访了一些领导,得到了人事上允许秘密参加考试的同意.报名的后顾之忧没有了.很多在职的考生都会遇到这个问题.有一个宗旨:规章制度是死的, 人是活的.凡是有人参与的行为,必定有活动的可能性.

3.我有了比较稳定的恋爱心态,不象一开始那么起伏不定了.能有很平静的态度.

4.这半年,我学习很少,但并不是不学.我数学看了一部分,英语一直在学.

5.我定了报考的方向.我是在职考,时间总不能和在校生比.专业课也没有竞争性.我觉得应该报研究所,没有他们自己本科生的竞争,专业课也要科目尽量少. 我选了一个符合这两个条件的地方.

正式开始复习的第一天,我就开始了倒计时,只有八十多天复习时间.我大致分了三个阶段,
11月,集中复习数学,十二月把所有的科目复习一遍,一月考前复习.
这段日子,我早上起床6:45,乘车7:00早饭7:30上班7:50下午我随身带着一本小的英语词典,在车上的时间我会背单词.上班很忙.但一有空闲, 我会上网看看考研信息,看看专业课,抓紧每一刻.下班5:45(常会加班)六点多回宿舍,买晚饭吃或者煮方便面.晚上基本可以学6小时,到12点, 准时睡觉.

到了12月,我上下班路上看的内容改成了实事政治.其他一切照旧.从无间断.
这两个月的复习很顺利,我谢绝一切娱乐活动.每天生活很规律,很充实.特别是周末.一想到整整两天可以埋头苦读,不用跑来跑去,真是高兴极了. 学习计划执行的还不错.1999年的节日还特别多,国庆五十年,澳门回归,千禧年.还记得12月31日晚上我自己学习时,觉得学不进去,也不愿放松自己, 就在担心地球会爆炸,我就这样凄凉的死去之中迷迷糊糊的睡着了.
元旦之前,我向公司提出要提前休为期20天的春节探亲假,当然是和领导先打好招呼的,被批准.这样从元旦到考试我又可以全天复习了.

考前的这一段时间,我的状态很好,又培养出了每天听半小时评书的习惯,每隔四五天出门买东西,放放风.

我有一个比较好的学习习惯就是喜欢做计划,我通常会想好,在一段日子里想达到什么目的,然后分块.留出一两天的空白.我一般会计划的很满, 这样就常常完成不了.但我不会因为没完成就失去成就感,因为我知道自己定的内容太多.这样才能鞭策自己提高效率,拼命达到目的,万一不行, 还可自我安慰一下.计划的时日用完之后,预留的那一两天,就可以用来解决剩余的少许内容,还可复习一下,解决一下难题.
每一天,我也这样安排.我学累了就会计划一下.

以下是我各科复习的一点心得

英语:突击复习是没用的.重要的是单词量.最好把大纲里的单词表对照一本好点的辞典背一遍.了解词的意思,及在句中的用法.时常读点文章. 我有一个朋友,把大学精度和泛读当成阅读理解来复习,效果也很好.熟读唐诗三百首,不会做诗也会吟.只要有好的语感,不论唐诗还是英文都是一样的. 去年我买了一本考研的单词祥解,错误太多.

政治:紧扣教委的大纲,每个知识点都要复习到.我去年用郭政平的书(包括时事部分),知识点虽全,但不详细.听说王晓峰的书较祥.当然国家教委的书也很好. 做到这一点之后,不论出什么问题,你都有话可说.时事政治,郭政平去年编得不好,(我的个人感觉).有一道实事的多选题我在它的册子里和1 2月后的补充材料里都没见过.好几种实事政治书里都没有.这道题就是:下届世贸总干事选了泰国总理和澳大利亚总理.
但是,我一个平时较关心时事的朋友就知道答案.看来,要养成习惯.

关于猜题:我北京的朋友告诉我过英语作文和政治的猜题.都是出自那些名家,特别是政治.当年的热点列了很多很多.最后冲刺阶段我花了很多功夫在这上面, 考试的时候才发现还是基本知识最重要.

关于往年的真题,一定要仔细研究.考过的知识点一般不会再考了,主要是研究正确的答题思路.
比如英语的阅读理解,政治的辨析,论述题.要培养自己的思维,了解自己的不足.

这两门需要背诵的科目,加上数学公式,回忆法很好用.
背完一部分后过一会儿,你回想都背过什么.在坐车比较挤的时候,也可以回忆.很有效的.
数学:我用的是陈文灯的书,我觉得这本书非常好.我看了三四遍.每一遍都看得很细.第一遍:主要是看和记忆,遇到看不懂得地方,就用别针别起来, 不研究太多细节.因为这时候知识不全面,不能融会贯通.经过这一遍,对知识点有个概括的掌握.第二遍:看得最仔细.这一遍会发现原来一看就懂得地方或许很深奥, 原来理解不了的基本全能看懂了.这一遍过后对数学就不存在知识方面的问题了,差的是技巧.第三遍:仔细研究书中例题.研究解题思想.每一部分结束后, 我都会再看看上部分的例题,试着自己做.对自己薄弱的地方,再看书,再练习.配套的有本习题集,可以参考.
复习中,发觉自己的解题能力太差了.都怪我,上大学时常常抄作业.悔恨啊!!
我做了历年研考的每一套题,特别是近年的几套,我自己先做,再看答案,在分析我的不足在哪.研究正确答案是怎样从题目一步步推导得出的. 很有帮助.

专业课:我复习的不多也不太好.也是看往年的试题,研究答题风格.如果你比较方便就能够联系到导师,应该请他给你一些帮助.不方便就算了, 不是太必要.

一月份时,为了培养考试的感觉,我严格按照考试的时间进行了多次模拟考.真正的考试不过又是一次模拟而已.到考试的那天,激动的有点没睡好.

考英语,发挥得不太好,时间安排不合理,最后只留了10分钟写作文.

政治:卷子一发下来,我就觉得最后二十天背的"最可能考的"东西都没用处.只好胡乱发挥,把试卷也写了个满满当当.

数学:考试时间最紧张一门课.先易后难.要掌握好答题节奏.难题慢慢分析.有一道大题我觉得不难,可就是不会做.看来平时练习太少了.其实我有一点思路, 可是后来又把写出来的划掉了.最后没时间再写.太可惜了.

考这三门的时候,我周围坚持下来的人逐日减少.因为报同一个地方的考生都坐在一起,我的信心也逐日加强.考最后一门专业课,我早交卷一个小时, 实在没什么可写的了.那时天上飘着小雪,刺骨的寒风吹着,我觉得很轻松,虽然当时的我,头发乱糟糟的,人脏兮兮的,身心疲惫不堪.我觉得我可以考上. 其实我本可以考得更好.之后,有人问起我万一考不上的打算,我就说去自杀.那时已经钻进了死胡同,也可以这么说,我拥有在了解自己的基础上的充分的信心. (自杀的话,千万不必当真.改善自己处境的途径有很多,可别那么死心眼儿)

当天下午我就去上班了,累死了.
最耗人心神的是等成绩单.以及在单位处理关系.若上学时考多么轻松,学校会为你处理一切.就是这时,我才后悔当年的退却.但是,工作后考也有好处. 经过几年的社会经验和业务经历,你会知道自己感兴趣的方向,知道自己适合做什么.
会清醒的看待自己,不再盲目.在失去过读书机会后,再回到教室里你会非常珍惜.你站在一个较高的位置看待今后三年的生活,收获会更多.

去复试的时候我发现了一个问题,我觉得我当初应该选择一个更有难度的地方,人都是这样吧,得到之后就觉得所追求的东西没有想象的那么珍贵, 那么好.我做事情对稳妥要求太多,挑战就少了些.内心里,我非常相当清华的学生,这是我的梦想,高考和考研并不是没有可能来实现.可是我怕万一. 不知以后还有没有机会了.
所以,您在选择院校的时候一定要多方面考虑,千万不要抱着有学上就行了的想法.

终于写完了!!从来没写过这么长的东西.多么希望发表后能引起轰动.;-)
