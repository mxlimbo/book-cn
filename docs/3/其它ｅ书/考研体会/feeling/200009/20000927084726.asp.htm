每年都可以KY而且有时不是第一也是成功

http://www.kaoyan.com 2000年09月27日08:47:26 考研论坛

作者：kaoyan.com网友黄海风

屏幕上再次出现的是一张成熟男人的脸，上面好象有一点点的沧桑。
这个男人叫熊倪。
不是汉城比洛加尼斯跳得更好却没有得冠军的那个神奇的小男孩，不是那是在亚特兰大意器风发的俊朗少年，这就是熊倪，一个十二年一直在不断求索的英雄。在中国跳水的群星中，熊倪只能算其中的优秀者，而不是天才，天才只有伏明霞和孙淑伟——那个在九十年代初（是不是很多年以前了）突然轻松超越熊倪的少年。
熊倪在汉城向世界宣布他的存在的时候，李那和桑雪还没上又儿园吧，汉城和悉尼仿佛隔了不知多少个时代，那个时候上初一的我正在对有理数的加减法不屑一顾，（现在有时我在灯下有一搭没一搭地做着题，想着不太可能的K Y，）我现在怎么也不明白那时会有这么多的傲。很多金排选手其实正在被慢慢忘记，1988和今天好象隔着千山万水，我在ELONG上读着生于70年代的感伤与共鸣，熊倪却和那个时候一样新鲜，但我看到那张成熟男人的脸，才真切地感到时间是流动地，流过了千山万水，用了八年熊倪从跳台银排走到跳板金排，过了十二年熊倪更加星光灿灿，用了八年北京才迎来了又一次的申奥。。。。。。
写到这里我感到一阵说不出的幸福
我们年年都可以考研（只是要辞职，我不敢）考程序员考注册会计师（我对此没什么兴趣），我们一年有两次机会可以参加自考（单位不准考研，自学点东东的自由还是有的），可以参加上海市高级口译证书，B EC从I到III，我们一年可以有四次考托的机会，我们随时可以去微软认证！！！！！！！。。。。。要的只是执着
不要再象暗夜里迷途的人
