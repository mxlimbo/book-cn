格瓦拉与我何干

　　“革命”是什么？为什么要“革命”？要“革”谁的“命”？在如今这个浮躁盲从的时代中，在这个物欲横流的社会里，来谈论这样一个陌生而遥远的命题，多少会让我们感到唐突、不解和一点点好奇。格瓦拉是谁？什么人在演格瓦拉？为什么要演格瓦拉？如果你不是置身于这样一批洋溢着革命激情的话剧人中间，你是不会觉得革命和格瓦拉跟你有什么关系。

　　坐在话剧《切·格瓦拉》的演出现场，无论你是一个善良的老实人，还是叱咤风云的商业巨子；无论你是对社会心怀鬼胎不怀好意，还是天天都能扼住命运咽喉的强者，你都不会对眼前追光下所展示的这个格瓦拉的精神世界无动于衷。

　　格瓦拉是古巴民族英雄，多年前在那个遥远的国家搞革命，后来死在玻利维亚的丛林里。多年后，格瓦拉成了一个国际偶像派革命人物，而且走上了如今的北京舞台。实际上创作者们是希望通过格瓦拉来反思一下，我们还要不要纯净的理想，要不要单纯的执着，要不要对当今的社会用革命的眼光重新审视。创作者们是不想让观众踏踏实实坐在那儿玩味所谓艺术感觉的，这个戏台词的高速节奏、演员们夸张的表演、剧中语言的步步紧逼，对现实社会的嬉笑怒骂，都让观众难以保持平常心态，也让最冷静的人难说与自己没关系。比如说社会腐败、市侩心态、商业法则、道义准则、人生理想、世界大同、人性与反人性、革命与反革命等等，以及我们躲也躲不过去的“．COM”。在观看这个戏中，我们会笑、会愤怒、会自嘲、会崇高、会悲壮，当然也会惴惴不安。我们痛恨自私、丑恶、不公，但我们又何尝不是自私、丑恶和不公中的一员。怎么办？先革自己人性的命吧。

　　这个戏好，但问题也不能不说，否则也怪对不住格瓦拉的。比如说节奏、段落转呈、灯光音响都还有点糙。不过这个糙倒也不是特别有碍大局，有一哥们儿说了：“革命”嘛，不糙点成吗。

杭　程
