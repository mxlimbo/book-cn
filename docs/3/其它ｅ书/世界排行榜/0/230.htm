吕蓓卡·韦斯特最愿意目睹的5个历史事件

　　吕蓓卡敏捷的才思、充满机智的语言和深刻的心理分析使她蜚声世界，被公认为当今最有才华的作家之一。她最令世人瞩目的成就是1945年及1946年对纽伦堡审判所作的出色报道。她也是一名受欢迎的小说家和评论家。作家安东尼·韦斯特就是她和H.G.威尔斯所生的儿子。她最为著名的小说有《黑羊羔和灰猎鹰》、《叛逆的意义》和《火药车》。

　　1．科尔特斯到达阿兹特克市，受到阿兹特克皇帝蒙特兹马的友好接待
　　阿兹特克一定是座美丽的城市，比任何欧洲的城市更美，此外，蒙特兹马和阿兹特克人民的举止和礼仪是那么文明和优雅。这是一座真正的城市，可他们要牺牲人民，这种事决不应发生。

　　2．克里斯托弗·雷恩第一次看见圣保罗教堂的圆顶（1697）
　　他以前从未见过这类圆形屋顶。这位英国建筑师虽然知道圆顶建筑是怎么回事，该如何设计，而且本人还亲自设计过一个，可自己却从未亲眼一睹，因此这一时刻一定是很不寻常的。

　　3．在西西里岛西那摩斯的监狱里（奔尼伯罗撒战争中）的希腊战俘被告知如能背诵任何一部优秀的希腊戏剧中的10行台词便能获得释放（公元前413）
　　这是表现出真正文明的一个生动的场面。想想当时的情景：既背诵优美的诗歌又得到自由。

　　4．拿破仑踏上圣·海仑娜岛（1815．10．15）
　　我不能容忍这个家伙以及他所做的任何事。

　　5．第一个复活节的早晨（29）
　　原因当然无需再讲。
