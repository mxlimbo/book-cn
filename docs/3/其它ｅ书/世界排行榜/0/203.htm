贾德・马默博士提出的10名他所愿意进行分析的人士

　　马默博士现任南加州大学精神病学教授，他于1933年在哥伦比业大学获医学博士，曾任美国精神病学学会和美国心理分析学会主席，除了写过大量研究文章外，他还发表了多部专著，其中包括：《现代心理分析：新的方向和视角》（1968），《变迁中的精神病学》（1973），《精神病医生和他们的病人：对全国私营诊所的调查》（1975），《同性恋行为：当代的重新估价》（1979）。
　　据马默说：要列举10名我所愿意分析的历史人物对我来说确实是一个困难的选择，我最初的想法是选10名杰出的伟人，因为对他们的心理活动的观察会很有帮助，很令人兴奋。例如像耶稣、爱因斯坦、毕加索、莎士比亚、贝多芬、爱迪生和潘恩等。但是我后来还是决定不这样做，我想心理治疗并非医生或病人可以轻松享受的娱乐，它是专门服务于那些因精神上或感情上的矛盾而急需这种治疗、并可能通过这种治疗而得到实际帮助的人们的。因此我现在所挑选的10名历史人物并不是在他们各自的领域内最伟大或最重要的，而是一些具有杰出才能，然而因感情上的冲突和困难过早地去世或在生活中曾遭受巨大痛苦的人们。我没有选黑人或其他人种的人，并非出于种族偏见，因为我对这些种族中的历史人物缺乏了解，因此无法作出恰当的选择。我也意识到我所挑选的人中大部分是作家和艺术家，并且没有从某些领域内挑选任何一个作出巨大成就的杰出人士。我想这部分反映了我自己的兴趣，同样也由于挑选的人数不可能太多。

　　1．爱米丽・勃朗特（1818-1848）
　　这位小说《呼啸山庄》的作者也许是勃朗特姐妹中最具才华的一员。她表面上沉默寡言，感情淡漠，可在她的作品中她表现出埋藏在心底的强烈感情波澜。她30岁时死于肺结核，生前没能实现自己的宿愿：出版她唯一的一部小说和为数不多的精彩诗篇。

　　2．哈特・克莱因（1899-1932）
　　克莱因是位很有前途的美国诗人，32岁自杀身亡。他从小生长在一个父母离异的家庭，不幸的生活使他染上了酗酒的习惯，他常因内心的冲突和矛盾而深感痛苦。

　　3．爱米莉・狄更生（1830-1886）
　　这位杰出的美国诗人，幼年因父母感情疏远，从小就养成腼腆和胆怯的脾气。在她30多岁时，她几乎成了同外界断绝联系的隐士，很少再作任何诗歌创作。

　　4．文森特・梵高（1853-1890）
　　这位举世无双的荷兰艺术家曾有一种强烈的孤独和不适的感觉，这导致他最后的精神崩溃，37岁时他开枪自杀。

　　5．弗兰茨・卡夫卡（1883-1924）
　　卡夫卡是位天才的捷克小说家，擅长刻画人物内心的绝望、恐惧和孤独。可他从小就处于一种恋母和仇父情结之中。他对自己这种不正常的恋情一直很感内疚，然而却又无法摆脱。由于长期感情压抑，他的肺结核病日趋加剧，并在他40岁时夺走了他的生命。

　　6．三岛由纪夫
　　这个才华横溢和喜好炫耀的日本作家，从小由精神不太正常的祖母和专横跋扈的父亲扶养，他对罪和忿怒的深刻感受十分明显地表现在他强烈的军国主义思想和对流血、死亡、自杀等行为的沉迷之中。他在45岁时期腹自杀。

　　7．埃德瓦特・蒙克（1863-1944）
　　这名出色的挪威艺术家是二十世纪表现主义绘画的先驱。他5岁时丧母，少年时又先后失去姐姐、父亲和哥哥。死亡和苦难的幽灵伴随了他的一生，并影响了他大部分的作品。

　　8．埃德加・爱伦坡（1809-1849）
　　爱伦坡既是浪漫主义诗人，又是侦探小说的先驱。他特别擅长描写那些令人毛骨悚然或变幻莫测的情节和场面。他成年后一直饮酒过度，最后导致过早的夭折。

　　9．奥古斯特・斯特林堡（1849-1912）
　　这位瑞典最伟大的现代作家，童年时曾遭受贫困和精神的伤害。成年后他经历了一连串不幸的婚姻，并经受了由此产生的精神打击和感情的波折。

　　10．迪伦・托马斯（1914-1953）
　　托马斯是威尔士的诗坛奇才，他感情上屡次遭受的打击以及毫无节制地饮酒，断送了自己杰出的文学才能。33岁时他经历一段情绪极其低落的时期，6年后，仍然是酗酒使他过早地结束了生命。
