鲍比·费希尔眼中10名最杰出的棋手

　　鲍比·费希尔是国际象棋史上最伟大的棋手之一。他14岁获美国象棋锦标赛冠军。15岁被授于象棋大师称号。因此也是最年轻的棋王和最年轻的大师。鲍比不仅棋艺精湛，而且擅用心理战术，1972年他在冰岛雷克雅未克击败了波里斯·斯帕斯基，登上世界冠军宝座。他是第一个获此殊荣的美国人。
