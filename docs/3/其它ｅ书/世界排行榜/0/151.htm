伊里・纳斯塔思眼中的10名网球史上最佳男球星

　　这名人称网坛顽童的罗马尼亚球星虽技艺超群，却是个颇有争议的人物。他曾获得不计其数的金牌，其中有美国室内锦标赛（1970）、美国公开赛（1972）、百事可乐大满贯（1976）和挑战杯赛（1977和1978）等冠军。1969年罗马尼亚授于他运动大师称号。
