6位最令人不堪忍受的演员

　　1．哈吉·阿里
　　哈吉·阿里在本世纪初是个闻名遐迩的人物，他的表演包括吞下一些不可思议的东西：西瓜子、假珠宝、硬币、桃核——以及任何观众要求他吞下的东西。虽然这种表演毫无情趣可言，却能给人留下深刻印象，他的演出场场爆满。阿里曾表演过这样一个杰作：助手先在台上搭起一个小小的金属城堡，阿里则喝下一加仑水后再喝下一品脱煤油。然后他通过一个鼓形滚筒的压力，使腹中的煤油喷出来，在舞台上形成一道6英尺的煤油弧，并将城堡点燃，当火焰直冲屋顶时，他再用同样方法让水喷射出来，将火扑灭。

　　2．切莉姐妹
　　奥斯卡·哈默斯坦为了填补财政上的空缺，便邀请了切莉四姐妹伊丽莉白、艾菲、吉茜和爱迪到纽约演出。他自称：“我要让观众开开眼界，看看最糟糕的表演是怎么回事。"1896年11月16日，切莉姐妹在奥林匹克剧院正式登台，三姐妹穿着火红的衣服，戴着火红的帽子，穿着火红的鞋袜在舞台上蹦蹦跳跳，而吉茜则在一旁，一边敲打着低音鼓，一边唱着：
　　　　樱桃成熟了！（切莉在英语中为樱桃之意——译注）
　　　　樱桃成熟了！
　　　　切莉三姐妹，
　　　　来到舞台上！纽约市的观众坐在那里，惊讶得目瞪口呆，然而中西部的观众却没有那么有涵养，他们毫不迟疑地将垃圾和烂番茄朝四姐妹扔去。后来切莉姐妹不得不在台前设一道铁丝网，以挡住愤怒的观众投来的雨点般的脏物。

　　3．伦那德·科兹
　　科兹是十九世纪的英国人，他想入非非地意欲以改写莎士比亚剧本成名。他改写的《罗密欧与朱丽叶》给人留下了难忘的印象。他在剧中扮演男主角罗密欧，在舞台上异想天开地想用一把鹤嘴锄扒开新娘朱丽叶的棺材。他的服饰也难以置信：头戴插着羽毛的帽子，身披星星点点的斗篷，穿着一条异常宽大的马裤——平时他也是这副打扮。他在全英国各地被人哄笑、辱骂，有人甚至威胁说要绞死他，但他还是顽强地演出。在一次演出中，有几位观众因为哄笑过度，腹部痉挛，不得不到医院就医。1848年，科兹在一次车祸中丧失，终年74岁。

　　4．佐田·哈特曼
　　哈特曼自称是个日本人和德国人的混血儿，一个天才的发明家，他于本世纪初向纽约的观众们呈上一台"芬芳的协奏曲"。哈特曼用一台蓄电池发动的电扇，将一团团芳香扑鼻的烟雾朝观众席吹去，他自己则用蹩脚的英语向观众解释说，每一种香味代表着一个国家。不过他往往只介绍到英国（玫瑰香味）和德国（紫罗兰香味）就被观众赶下台去了。

　　5．佛罗伦丝·福斯特·詹金丝
　　1943年的一次出租车车祸，使这位詹金丝小姐能用颤音唱出高音。她在高兴之余，立刻撤回对出租车公司的起诉，其而且还送那位惊魂未定的司机一盒进口雪茄烟——世界上最糟糕的女歌剧演员就这样登场了。詹金丝小姐多年来一直是音乐会和歌剧中的笑柄，批评家们绞尽脑汁用模棱两可的话来嘲弄她。罗伯特·劳伦斯在《星期六评论》上写道：“她唱得最动听时，让人觉得有一只未被驯服的大鸟在鸣叫。"其中最尖刻的要数《新闻周刊》上的一段话：“詹金丝小姐唱到高音时，让人觉得她背脊底部的什么地方有着持续的疼痛。"然而尽管如此，詹金丝的舞台生涯一直持续到70岁。1944年10月5日，她在莫扎特的《魔笛》中扮演天使，身披五彩头巾，头上别着一枚镶着珠宝的梳子，插着一枝玫瑰。她手持花篮，将里面的花朵撒向观众，最后竟然将花篮也随手抛出。

　　6．爱尔娃·米勒太太
　　爱尔娃是在堪萨斯州长大的，她深信，只要刻苦训练，必能成为一个出色的歌唱家。可是她的亲朋好友却与她意见相左。不过爱尔娃并不理会他们的反对意见，她先是参加了学校的合唱队，后来又参加教堂的唱诗班，甚至还在加利福尼亚波莫那学院专修声乐。但是尽管如此，她的声音却让人觉得黎明时垃圾箱里一只蟑螂在爬行。60年代，年逾五十、身为家庭主妇的爱尔娃雄心依然不减当年。她出资灌了几张唱片，其中包括她最爱唱的"为了小鸭子的缘故"。她说服了当地一家迪斯科夜总会的管理员，播放她的一张唱片"闹市"。没想到她一举走红，这张唱片在3个星期内售出250，000张拷贝，并使她荣获"堪萨斯摇滚之鸟"的美称。接着她又在多次电视节目中露面。《时代》杂志称：“说得仁慈一些，她的节奏完全是自由形式……她有一种狡猾的才能，会在两拍之间巧妙地着落，从而产生出一种完全崭新的效果，而且无人能够仿效。”
