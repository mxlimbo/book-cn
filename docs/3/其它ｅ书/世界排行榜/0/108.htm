H.L.门肯眼中10名最令人乏味的作家

　　门肯（1880-1956）系本世纪最杰出的记者和编辑之一。在二三十年代，他是美国最主要的反传统的文人。他对当时流行的传统文学大加鞭挞，而对于诸如德莱塞·安得生及辛克莱·刘易斯等现代作家则大加推崇。他著有多部散文集、评论文集以及其他学术著作。其中包括他的长达6卷的《偏见集》、《评民主》及《美国语言》等。
