朱莉娅·蔡尔德眼中的10本美国最佳烹调书

　　（当然除了她自己写的书外！）

　　朱莉娅是美国获埃米奖的电视节目"法国厨师"中的明星，她精湛的烹调艺术和充满活力的风格令千千万万个电视观众倾倒。她自己也写过数本非常畅销的烹调书，其中包括她同西蒙·贝克和路易西特·伯索尔共同编写的《掌握法国烹饪艺术》（共两卷）及一本介绍她自己最喜欢的一些菜肴的烹饪方法的书：《来自朱莉娅·蔡尔德的厨房》。
　　1．《意大利传统烹饪集》　作者：玛塞拉·哈兹安
　　2．《吃的艺术》　作者：M·F·K·费希尔
　　3．《拉鲁斯美食大全》　作者：拉鲁斯
　　4．《技术》　作者：雅克一佩齐
　　5．《墨西哥烹饪》和续集　作者：戴安娜·肯尼迪
　　6．《美国烹饪》和《优秀烹饪的理论和实践》　作者：詹姆斯．尔德比
　　7．《中国烹饪之秘诀》　作者：郭艾琳
　　8．《香肠、鹅肝酱馅饼和其他熟食制作的技巧》　作者：简·戈里格森
　　9．《烹饪之乐趣》　作者：艾玛·尤鲍尔和马里恩·贝克
　　10．《法国外省之烹饪》　作者：伊丽莎白·戴卫
