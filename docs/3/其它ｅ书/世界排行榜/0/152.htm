玛蒂娜·纳芙拉蒂诺娃眼中的8名网球史上最佳女球星

　　这名1974年仍被网球杂志称为新手的捷克运动员，在弗吉尼亚斯利姆斯联盟赛中连续几个赛季成绩突出，并在1979年第二次获得温布尔顿网球赛单打冠军（她曾于1976和1979年同比莉·简·金合作参加双打）。1975年她为了能在自己所愿意的"任何地方和任何时间"打网球，逃到美国。
