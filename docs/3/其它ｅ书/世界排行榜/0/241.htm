10起惊人的巧合

　　1．格林贝利希尔
　　1911年在一个叫格林贝利希尔的地方谋杀埃特蒙德爵士的罪犯在伦敦被处以绞刑。这3个罪犯的名字正好分别为格林、贝利和希尔。

　　2．杀人出租车
　　百慕大兄弟两人先后被同一个司机驾驶的同一辆出租车在同一条大街上压死。当时兄弟两人都骑着同一辆机动两用车，而且出租汽车上坐的竟是同一个乘客，两次事件发生在相隔1年的同一天。埃斯基恩・劳伦斯和他的兄弟内维尔死去时都才17岁。

　　3．二十三
　　1932年，一个外号叫"荷兰人"、名字叫舒尔茨的酒贩子在纽约23号街派人杀害了一个23岁名叫文森特・考尔的男子。舒尔茨本人也于1935年10日23日被人杀死，凶手的名字叫查理・沃克曼，被判无期徒刑。结果在监狱呆了23年后被释放。

　　4．野鹅
　　1974年11月一天，诺伊尔・麦凯布一家正在他们座落在英国得贝市金斯顿大街上的家中欣赏弗兰基・莱恩所唱的歌剧《野鹅》的录音，突然一只加拿大鹅打碎了玻璃从窗子跳进了他们的卧室。

　　5．空中落婴
　　1975年春天在底特律市一个婴儿从14层高的楼房摔下，正好落到正在行走的约瑟夫・费格洛克的身上。1年后又发生了同样的事，费格洛克和这两个婴儿均没死。

　　6．密码
　　1944年6月6日――盟军在诺曼底登陆前的几个月，伦敦电信报上刊登的拼字游戏中包含了许多有关盟军行动的高度机密的密码字，如：Omaha，Utah，Muberry，Neptune，　　　　以及有关诺曼底登陆本身的字符Overland。这套谜字是由一个小学校长设计的，可他本人对密码一无所知。

　　7．倒霉的路
　　1974年弗兰克・克拉特沃西在一次晚宴结束后驱车返回位于萨默塞特市沃希福特路的家，途中不幸翻车。1小时后，从同一个晚宴出来的他的孪生兄弟杰克在同一条路上也翻了车。

　　8．死亡的时刻
　　1975年1月28日清晨3时，在英国列斯特市他姐姐家中度假的查尔斯・戴维斯突然死去。当他姐姐打电话到查尔斯在里兹的家，准备把这个不幸的消息告诉他的家人时，却得知查尔斯的妻子也是在同一天清晨3时突然去世。

　　6．特殊的感谢
　　4岁时，罗杰・劳塞在麻省塞勒姆的海滩被一名叫爱丽丝・布莱斯的妇女救起。9年后，也就是1974年，在同一个地方，罗杰划动一只竹排进入水面，把一名男子从水中救起。该男子原来就是爱丽丝的丈夫。

　　10．孪生兄弟
　　俄亥俄州利马市的詹姆士・卢斯和俄亥俄州达顿市的詹姆士・斯皮林格是一对孪生兄弟。但是他们出世不久就被分开。他们各自的养父虽然互不相识，却同时给他们取名詹姆士。这两个詹姆士都娶了个叫琳达的女子，而后来都同妻子离婚。两人都给自己的第一个儿子取名为艾伦。两人都喜爱到弗罗里达海滩度假，两人都是6英尺高，180磅重，具有同样的爱好，都曾受过警官的训练。1979年他们第一次相遇，当时年龄都是39岁。
