九死一生：10件难以置信的事例

　　1．最后的抵抗
　　尤里西斯·格兰特总统的儿子，西点军校毕业生费德里克·格兰特曾在军队服役。1874年他娶了妻，两年后格兰特将军奉命将队伍开进黑山地区，搜寻并歼灭了由人称"疯马"的酋长所率领的苏族印第安部落。1876年6月25日，在蒙大那州小比格霍恩河附近的山谷里，卡斯特将军的266个AE？兵遭到了伏击，面对"疯马"手下3500名印第安战士的攻击，卡斯特进行了最后的浴血抵抗，在仅仅二三个小时内，他本人和手下的所有人马都被斩尽杀绝，但是里面却没有总统的儿子费德里克·格兰特。原来他并没随部队出发。就在部队开拔的前一二天，由于他妻子分娩在即，且是第一胎，因而格兰特获准回家照顾妻子。格兰特后来曾出任驻奥地利大使，芝加哥警察局长，并在美国同西班牙的战争中升任将军。他死于1912年，享年72岁。

　　2．礼服
　　1919年30岁的俄国电子工程师弗拉基米尔·兹沃利金移居美国，他先为西屋电器公司工作，后来又转入美国无线电公司。由于发明了电视发送管及电视接受器，他被公认为现代电视之父。二次大战刚爆发时，他正好在德国柏林办事，而要从柏林返回纽约，他必须先到伦敦，然后坐船穿越大西洋到美国。他购买了从伦敦出发的"雅典娜"号客轮的船票。"可我无意中把自己的礼服丢在了黎巴嫩。"在同作家布鲁斯·费尔顿的谈话中，他回忆道：“考虑到进入船上一等餐厅用餐时，衣着过分随便会有失体面，因此我决定先上街买一套礼服。回来后搭乘下一班轮船。"就这样兹沃利金没乘上"雅典娜"号。1939年9月4日"雅典娜"号在驶至爱尔兰海岸时被纳粹德国的U形潜艇发出的水雷击中，造成128名乘客丧生，其中有28名美国人。此时这第29个美国人正在伦敦购买新的礼服。

　　3．受害者
　　1924年5月21日芝加哥有两个年轻人策划进行一项重大的犯罪活动。其中一个19岁，名叫内森·利奥波德，是芝加哥大学毕业生。其父为一个制箱厂老板，百万富翁。另一个年仅18岁，叫理查德A·洛布，是密执安大学毕业生，其父为西尔斯·罗普克公司副总裁。几年前，利奥波德提出同洛布建立同性恋关系，洛布表示同意，条件是利奥波德必须协助他杀死一个人。该阴谋包括绑架、枪杀以及索取绑票。他们考虑了3个对象，都不甚满意，最后确定了第四个。他就是年轻的阿曼达·多伊奇——西尔斯·罗普克公司财产的继承人之一。他们准备在他离开哈佛预科学校时把他绑架。洛布和利奥波德把汽车停在校外，可阿曼达并没出来，他们就把洛布的表弟罗伯特"鲍比"弗兰克斯拉上车子，兜了一圈，然后用一个凿子朝他头部猛扎四下。后来人们在一个阴沟里发现了弗兰克斯的尸体，阴沟附近还发现利奥波德的眼镜。杀人犯很快被抓住，洛布坦白了自己的罪行。至于那个本来会被杀害的多伊奇怎么会没出现？原来那天他正好生病，没上学。由于躲避了这场灾难，多伊奇后来在事业上十分成功，他成了好莱坞制片人。

　　4．幸福的情人节
　　1929年2月14日，盗匪头目"臭虫"莫兰同他手下7个匪徒约定到位于芝加哥北克拉克大街卡瑟奇公司的仓库察看一批从加拿大运抵的烈性酒。莫兰临出发前因有事耽误了时间，就在这时，他手下的7人——弗兰克，彼得，戈森伯格，亚当海尔，阿尔·温申克，詹姆士·克拉克，约翰·梅和莱恩哈特·施维然在仓库中等待他们的头目。突然，一辆卡迪莱克牌的旅行车停在外面，车上跳下4个人，其中两人身着警服，他们冲进仓库。两人手持散弹枪，另外两人端着机关枪，强迫这7个人面对墙壁排成一行。然后开枪把他们全部杀死。这4个人是阿尔·卡波尼手下的匪徒，他们同莫兰是死对头。事情一干完后，他们立即离开现场，5分钟后，"臭虫"莫兰到达仓库，十分幸运地躲过了这次情人节大屠杀。

　　5．波士顿的庆祝会
　　1942年11月28日早晨，波士顿大学鹰队（橄榄球）的一名行政人员在椰林夜总会预定席位，准备当晚为全队举行庆功会，拥有杰出的四分卫埃迪·多尔蒂及后卫迈克·霍洛伏克的波士顿大学队是全美第一劲旅。它在8场比赛中保持全胜，共获249分，而它的对手只得19分。因此波士顿大学队只需再击败一个弱队圣十字架队便可问鼎冠军宝座。圣十字架队目前的战绩是四败一平，那晚，就在椰林夜总会的面积达900平方英尺的地下室里，挂在天花板上的装饰织物突然莫名片妙地着了火（消防警至今仍未查清火源），夜总会顿时变成了可怕的地狱，仅12分钟内，490个人死去或奄奄一息，有一个房间叫泰勒斯厅，里边的人无一生还，全部化为灰烬。经最后核实，死者中包括前牛仔电影明星伯克·琼斯。但没有一个人是波士顿大学橄榄球队的队员。因为那天下午，战绩辉煌的波士顿大学队表现严重失常，竟被处于劣势的圣十字架队打得土崩瓦解，狼狈不堪，结果以12比55的悬殊比分输了这场球，爆出了近年来体育比赛的最大冷门。垂头丧气的波士顿大学队队员无兴参加什么庆祝会，就在庆祝会临开前，他们取消了在椰林夜总会预订的包房和席位，早早地上床蒙头睡大觉了。

　　6．次要的目标
　　二次大战时，第一颗原子弹的首先目标应该是日本的广岛还是小仓？起初美国空军将小仓这个拥有40万人口的工业城市定为首选目标，但有报道说该市附近有一个专门关押美国战俘的集中营，这使得空军的官员们犹疑不决，最后他们决定首攻目标改为广岛市，小仓则作为第二目标。1945年8月6日，空军保罗·蒂贝茨上校驾驶空中堡垒在广岛上空投下了一颗原子弹。造成全市255，000人中10万人死亡或失踪，68，000人受伤。按照原定计划，3天后将投下另一颗原子弹，这次决定不管有无美国战俘营，首要目标仍是小仓，长期则是替补目标。那天查尔斯·斯威尼驾着另一空中堡垒飞向目的地小仓。可这时指挥人员告诉他，该市上空气候很糟，他只好改变航向，飞往长崎。在拥有28万人的长崎扔下了一颗威力更大的钚原子弹。共有36，000人死亡，6万人受伤。幸运的是小仓的40万居民未遭劫难，他们安然无恙地活了下来，并在原子科学的时代得到了巨大发展。今天小仓已是北九州市的一个部分。

　　7．迟到
　　1953年5月29日，34岁的新西兰养蜂人埃特蒙德·希拉里和尼泊尔向导坦辛·诺凯成为历史上最先登上世界屋脊29，000英尺高的珠穆朗玛峰顶峰的人。希拉里由于创造了战胜高山空气稀薄和狂风暴雨的奇迹，一时间他成了英雄人物，并获得皇室授于的爵位。7年后，他的运气使他战胜了死神，1960年12月6日，由于打算二星期后重返尼泊尔，他预定了一张联合航空公司DC-8从芝加哥飞往纽约的班机。就在同一天，一架环球航空公司的超级星座飞机在俄亥俄州起飞也同时飞往纽约。当时纽约的气候很糟，大雾弥漫，阴雨不断，这两架飞机在纽约上空相撞，结果造成134名乘客和机务人员死亡。可是埃特蒙德却不在机上，由于他迟到机场，没能赶上联合航空公司的班机。然而他的妻子的运气并不很佳，15年后在珠穆朗玛峰附近的一次空难中不幸身亡。

　　8．医生的命令
　　1958年的一天，伊丽莎白·泰勒准备陪同迈克·托德前往纽约弗利爱斯俱乐部参加一个托德举行的盛大宴会。就在临行前两天，泰勒突然气管炎发作，托德打算动身时，泰勒的体温高达华氏102°，医生禁止她外出，当时的气候很糟，没人愿意坐他的私人飞机同他一起去纽约。柯克·道格拉斯，乔·刘易斯以及乔瑟夫．梅齐威克兹都婉言拒绝了他的邀请。最后剧作家阿特·利恩表示愿意同行，因为他想在路上修改一下一部电影剧本。可没想到的是，飞机在新墨西哥坠毁，机上的人全部遇难。

　　9．摇滚乐团的危机
　　一个名叫"蟋蟀"的摇滚乐团刚完成爱达华·克利尔湖城的演出，准备前往明尼苏达州的莫尔海德市进行下一轮的表演。由于演员们都很疲劳，乐团领队巴迪．赫利就在附近的梅森市包租了一架飞机，准备登机的有赫利，韦伦·吉宁斯（一个演奏乡村音乐的大胡子德克萨斯人）以及吉它手汤米·阿尔索普。那天是1959年2月3日，这架包租的N3749N号飞机正要起飞时，一个叫理查森的地方司法官请求吉宁斯把机位让给他，因为他患了感冒，身体不适。几乎在同时，一个叫里奇·瓦伦斯的人也向汤米·阿尔索普询问是否可以将位子让给他。结果吉宁斯和阿尔索普都没登机。飞机起飞后不久失事。机上的人全部遇难，而韦伦·吉宁斯和汤米·阿尔索普则幸免于难。

　　10．血洗好莱坞
　　1969年8月8日晚，小说家杰西·科辛斯基准备飞往洛杉矶同他的朋友沃伊泰克·弗里考斯基会面，然后两人同去好莱坞参加在女演员莎龙·泰特家举行的晚宴。那晚有一些未受邀请的人也闯入泰特家。这一群吸毒后行为疯狂的匪徒（查尔斯·华生，苏娅·阿特金斯，帕特里夏·克伦温克尔，莱斯利·范·豪登）在他们崇拜的头目查尔斯·曼森——一个罪犯和无业游民的唆使下，用手枪和匕首杀死了有孕在身的莎龙·泰特以及杰伊·塞勃林、阿比·盖尔·福尔杰、斯蒂文·派伦特和弗里考斯基。本应参加晚宴的杰西·科辛斯基因为路上延误了时间而没能出席，原因是法国航空公司的工作人员把他的行李送错了地方——送到了纽约而不是洛杉矶。他后来在一本名叫《初次约会》的书中叙述了这千钧一发的事件。
