威廉·韦斯特摩兰将军最愿意目睹的5个历史事件

　　这名1936年毕业于西点军校的军人第二次世界大战时曾在欧洲和北非战场作战。他还参加过朝鲜战争。1964年他被授予将军军衔，并出任驻越南美军司令。1968年尼克松总统任命他为陆军参谋长。他一直在此岗位上工作到1972年退休。

　　1．耶稣被钉十字架，复活和升天（29）
　　这一连串圣经上记载的事件对本星球的文明产生了巨大的影响。

　　2．汉尼拔翻越阿尔卑斯山（前218）
　　迦太基军队经过同风暴、大雪和野蛮人的顽强搏斗，带着满载货物的象队，成功地翻过阿尔卑斯山脉——这是毅力战胜艰难的一个最佳实例。

　　3．罗伯特·李在阿波马托斯向尤利西斯·格兰特投降
　　值此，这个年轻的国家成为一个团结的典范。

　　4．泰坦尼克号客轮沉没（1912．4．14-15）
　　很可能这是历史上因人为的疏忽而造成的最大的灾难——使人重新明白自然的力量可以摧毁人类的创造和人类自我陶醉的成就感。

　　5．奥维尔和威尔伯·赖特在北加州所作的人类首次飞机飞行（1903．12．17）
　　这是一件使交通运输和战争发生革命性变化的大事，同样也是使人类踏上探索外层空间的征途的历史性事件。
