9位被部下杀害的指挥官

　　1．约翰·菲尼斯上校（？—1857）　英国人
　　1857年5月10日，在印度的密拉特，约翰·菲尼斯上校，——当时他是第11英军土著团的指挥官——得到报告，他的部下已经占领了市政广场，并且有反叛的迹象。他骑上马，飞速赶去阻止这一疯狂举动。但那些印度士兵已下决心反抗英国人的统治，他们朝菲尼斯上校开火，立刻将他打死了。作为导火线，这点燃了印度民族大起义的火焰。

　　2．戈里科夫舰长（？—1905）　俄国人
　　1905年6月13日，俄国黑海舰队铁甲巡洋舰"波将金”号上的水兵，因不满生活待遇而发生哗变。戈里科夫舰长和其他一些军官在激战中被水兵打死，他的尸体被扔进大海。次年舰上水兵从敖德萨被引渡回国，许多人被判处死刑。

　　3．古斯塔夫二世（1594—1632）　瑞典国王
　　1632年，在三十年战争的吕岑战役中，古斯塔夫·阿多夫国王正率领着自己的骑兵朝神圣罗马帝国的军队冲去时，被人从背后枪杀。究竟是谁将他杀害的，至今依然是个谜。但是许多权威的历史学家都坚持说，古斯塔夫要么死于意外事故，要么死于混进自己军队里的奸细之手。

　　4—5．理查德·哈伦中尉和托马斯·迪罗中尉（？—1971）
　　 美国人
　　1971年3月16日早晨，在越南边河美军基地，有人朝军官宿舍的窗内扔了两枚手榴弹，哈伦中尉和迪罗中尉被当场炸死。不久，军士贝利·史密斯被逮捕，并受到军事法庭的审讯，他被判有罪，但后来又宣布他无罪。真正的凶手始终没能找到。

　　6．托马斯·杰克逊将军（1824—1863）　美国南部同盟将军
　　　1863年5月2日晚，在切恩斯洛维尔战役期间，杰克逊将军去北军阵地附近察看，希望能找到北军的薄弱之处。但他在回来的路上，被南军士兵开枪击中，因为人们还以为他是北军的骑兵。8天之后，他因伤势过重死去。

　　7．拉什克维奇上尉（？—1917）　俄国人
　　1917年3月12日在彼得格勒，沃林斯基团的士兵们拒绝向大街上的示威群众开火，因此将他们的指挥官拉什克维奇上尉打死了。这是俄国革命的前奏曲，因为沃林斯基团在枪杀指挥官之后便加入了革命的阵营，成为俄国军事史上第一支哗变的陆军部队。

　　8．戴维·马库斯上尉（1901—1948）　美国人
　　1948年，美军军官戴维·马库斯上尉辞去了他在五角大楼的职务，加入了刚成立的以色列军队。1948年6月10日夜晚，他在由特拉维夫通往耶路撒冷的一条道路上被以色列军队的一名哨兵打死了。那名哨兵误以为他是阿拉伯人，因为他正好去附近的田地里小解，他觉得有些冷，就披了一条白色的被单。

　　9．纳狄尔·沙赫（1688—1747）　波斯国王
　　他从一个土耳其的部落民变成一个英勇善战的将军，后来又成为波斯帝国的国王。作为一个征服者，纳狄尔·沙赫将阿富汗、印度西北部和蒙古都划入了自己的版图。1747年，他被自己的随从谋杀了。他的死得到了民众的欢呼，因为他的统治太残暴了。
