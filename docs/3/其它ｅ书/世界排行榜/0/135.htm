威尔特·张伯伦眼中10名职业篮球史上的最佳球星

　　威尔特·张伯伦的超群球艺在美国职业篮坛统治了整整14年。这个身高7.1英尺的中锋创立了全国篮球协会得分和运球两项最高纪录。他还是篮球史上唯一在一场比赛中得分100分的运动员。1972年他带领洛杉矶湖人队在全国篮球协会锦标赛中夺冠，第二年他离开篮坛，改打排球，效力于国际排球协会。
