10件被拍卖的奇物

　　1．乔治·华盛顿的洗衣帐单
　　一张注明1789年属于乔治·华盛顿的洗衣帐单以1100美元的价格被拍卖。据说华盛顿曾雇用一个名叫玛丽·弗斯的女人替他洗衣服，肥皂由玛丽自备。这张帐单上有华盛顿本人及他手下两个上校军官的签名。

　　2．斯威登伯格的颅骨
　　1977年3月6日瑞典哲学家伊马纽·斯威登伯格的颅骨在伦敦索斯比拍卖行拍卖，价格为2850美元。这名瑞典人死于1772年。颅骨的买主是瑞典皇家科学院的一位教授。他购买的目的是将此同存放在别处的此人的身躯骨胳重新连接起来。

　　3．黑尔太太的诗稿及羊皮
　　约瑟法·黑尔的著名儿童诗歌《玛丽有个小羊羔》的亲笔手稿于1975年在纽约索斯比拍卖行出售。与此同时被拍卖的还有一块小羊身上的毛皮，正是这块毛迫使黑尔产生了创作这首诗的灵感，另外还有玛丽·梯勒的手稿。黑尔的这首诗就是为她而写的，还有一封黑尔太太的儿子，霍雷肖写的信。在信中他提出这首诗是他母亲的作品；还有一封玛丽·梯勒写的信，信中对这块毛皮进行了鉴定并提到这首诗是如何产生的，还有一本玛丽·梯勒和范妮·迪肯森合写的书，书里记载了关于玛丽和她羊羔的真实故事。该诗共有3页长，上面有签名，并注明日期是1865年1月23日，地点费城。这是唯一已知的黑尔太太的手稿，估价为5000至10000美元。结果没卖掉。

　　4．伦敦桥
　　伦敦桥于1968年被麦克柯洛石油公司以2，460，000美元的价格买下。桥被分解成10，000块石块，装船运抵亚利桑那州的哈瓦索湖城。新的伦敦桥在哈瓦索湖的一条支流上重新建起，工程于1973年结束。

　　5．180，000个死猫
　　1890年，英国利物浦一家名叫列文顿的公司出售位于比尼哈桑附近的一个埋葬地里挖掘出的180，000只埃及木乃伊猫，售价为每吨3．15先令。

　　6．摄影者
　　已知最早拍摄一个摄影师在摄影的照片于1977年3月在伦敦南凯辛顿的克里斯蒂斯拍卖行出售。这张3×4时银版术照片显示杰佩士·豪格正在摄影室拍照。时间约在1842年至1843年之间。1973年以9000美元出售的爱伦坡的像片曾创下银版术照片的最高价，而这次豪格的照片则以9860美元成交。

　　7．厕所
　　一座建于十九世纪的橡木单座位厕所于1975年在新英格兰一个拍卖行售出。为何有人会花140元买一座废弃的旧厕所呢？买主说，因为这座房子的木头很适合做油画的框架。

　　8．糖纸
　　1975年11月美国邮票拍卖行以90美元售出一张1908年的可口可乐口香糖的包装纸。这张长约2．5时的红色糖纸上印有白色的字：“可口可乐口香糖，留兰香味。”

　　9．两个脑袋
　　1976年克里斯蒂拍卖行售出一个南美吉瓦洛部落人的头颅。这具已经缩小的头颅售价为9350美元。1977年克里斯蒂拍卖行又以20，400美元卖掉一具保存完好的毛利人的头颅。这两具头颅原均为胡普的收藏品。

　　10．朱迪的假睫毛
　　1979年，已故演员朱迪·加兰的丈夫在加州的贝佛里山举行了一次朱迪个人物品的拍卖，总共获得250，000美元，其中包括以125美元的价钱售出的假睫毛。
