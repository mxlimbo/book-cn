6种鸟灭绝的日子

　　1．拉布拉多野鸭，1872年12月12日
　　这是栖居在北美洲的一种野鸭，毛色为黑白相间，个头很小。人们一直以为这种野鸭具有很强的适应能力，它们的灭绝始终是个不解之谜。这种野鸭在加拿大东部孵出，但却要飞到费城度夏。滥捕滥杀也许是它们衰亡的原因，1872年，当最后一只拉布拉多野鸭在长岛随着一声枪响坠地之后，再也没有人见到过它们的踪影。

　　2．瓜达洛普岛的卡拉卡拉鹰，1914年12月1日
　　这种鹰毛色为棕黄，头部为黑色，翼上长着灰色条纹，最后看见活着的卡拉卡拉鹰的是R·H·贝克，他随即举枪将它击落。这种鹰常以山羊为食，所以牧羊人对此恨之入骨，用投毒、枪击的方式使其灭绝。

　　3．旅鸽，1914年9月1日
　　旅鸽呈灰棕色，其数量曾一度非常之多，当一群旅鸽飞过天际时，只见黑压压的一片，遮天蔽日。1810年，有人对一群旅鸽作过估计，认为共有2，230，272，000只。然而定居者的滥杀和森林的大面积毁灭使得旅鸽数目急剧减少。1869年，在一次围捕中，就有7，500，000只旅鸽落网。1909年，有人悬赏1500美元给一对活旅鸽的饲养者，可是没人领赏。最后一只旅鸽玛莎于1914年在辛辛那提动物园中孤苦伶仃地死去。

　　4．卡罗林那长尾鹦鹉，1918年2月21日
　　这种鹦鹉长着绿色、黄色的羽毛，尾巴长长的，十分漂亮，它们一度是美国东西部森林中常见的鸟。但是由于它们会损毁大片庄稼，所以农民们就对此滥加捕杀。最后一只卡罗林那长尾鹦鹉为雄性，名叫印卡斯，于1918年死于辛辛那提动物园。据管理人员说，它是因悲伤而死的，因为它那30年来相依为命的伴侣"珍妮夫人"，于前一年的夏天死去。

　　5．黑松鸡，1932年3月11日
　　黑松鸡是美国东海岸草原松鸡的亲属，一度曾是波士顿附近常见的野禽之一。但是由于大量捕杀，其数量急剧下降。自1928年12月以来，只有一只黑松鸡孤零零地住在玛莎葡萄园。1932年初的严冬使这只8岁的黑松鸡离开尘世。

　　6．尤勒捕蝇鸟，1955年9月26日
　　这种捕蝇鸟十分罕见，现只保存下两只标本，它们身长821英寸，毛色为橄榄绿和黄色。据詹姆士·邦德（这是一位研究加勒比地区鸟类的权威，而不是弗莱明笔下的007）估计，这种鸟是在1955年那场"珍妮"飓风中全部灭绝的。
