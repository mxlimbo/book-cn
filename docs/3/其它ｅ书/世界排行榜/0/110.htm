6位写过一部小说的名人

　　1．莎拉·伯纳特的唯一一部小说——《云中》——1878年
　　背景：奥斯卡·王尔德称她为"天仙般的莎拉"，她不仅是法国戏剧界最红的明星和丑闻最集中的怪人之一，还是一位多才多艺的艺术家。在她昙花一现的生涯中，莎拉从事过许多种艺术创作：绘画、雕刻，撰写回忆录——她还写过一部短而迷人的小说《云中》。
　　概要：《云中》是以莎拉1877年乘热气球飞上天的一段经历为基础的，同行的有她的挚友，画家乔治·克拉朗和气球驾驶员路易斯·高达。气球上还有一把椅子，这是莎拉最喜欢的东西，整部小说就是从椅子的角度来描述的，椅子也成了一个人物。他们4个在一个可爱的下午乘坐气球开始了一场小小的历险。远处出现了一个监狱，他们就对此发表了一通充满哲理的谈话；他们对可能来临的坏天气表示担忧。椅子也觉得忐忑不安，唯恐人们为了减轻气球的重量，把它扔出去。但是女主人公多娜·索尔——就是莎拉，太喜欢这把椅子，舍不得扔掉它。正在这时，气球降落在一个法国村庄，村民们被这群不速之客弄得瞠目结舌，而他们则度过了一个愉快的下午。

　　2．温斯顿·丘吉尔的唯一一部小说《萨弗劳拉》——1900年
　　背景：丘吉尔在写《萨弗劳拉》时才23岁，那是1898年，他在印度服役。小说最初80页写得挺顺手，但接着问题便接踵而来，他只得靠增加小说中的人物及提高情节的复杂性来解决。但是丘吉尔觉得写小说十分枯燥，他宁愿去当士兵和战地记者。这部小说完成后，丘吉尔来到南非，参加布尔战争。他被布尔人俘虏过，但又逃了出来。他回到英国时，成了一名英雄。1900年他被选入英国国会，同年，他的小说正式出版，并获得了3500美元。
　　概要：丘吉尔自己是这样概括《萨弗劳拉》的：“一位开明的领导人推翻了一个专制的政府，然而自己也被一场社会主义革命所吞没。"故事的地点是北非小国劳利尼亚，它由一个独裁者安东尼·莫拉勒统治。萨弗劳拉是反对党领袖，他年轻、富有，并且深得民心。在一次反对党集会上，萨弗劳拉宣称，推翻独裁统治不能靠人民起义，应该靠一种策略。莫拉勒听说此事后，就让美丽的妻子卢茜尔去与萨弗劳拉接近，试图弄清他的计策。然而他俩相爱了。不久，国内发生动乱，莫拉勒被一位外国人刺杀。萨弗劳拉劝说群众平静下来，并担任了该国领导人。但是原海军部长来到港口，威胁说除非萨弗劳拉离开该国，不然他就命令舰队炮轰城市。萨弗劳拉同意了，他和卢茜尔一起离开了劳利尼亚。多年之后，劳利尼亚人民要求他回国，他重新回到劳利尼亚，并把这个国家变得和平而繁荣。

　　3．珍妮·哈洛的唯一一部小说——《今日今夜》——1934年
　　背景：珍妮·哈洛是位电影明星，扮演过喜剧及性感的角色，但她最大的抱负是想成为一个作家。《今日今夜》的全部情节是在梦中出现的，她觉得有个强烈的欲望要把这个梦写出来。她找到朋友卡利·威尔逊——作家兼米高梅影片公司制片人，向他求助。他们两人共同写下了小说的基本情节。然而这部小说在珍妮·哈洛生前没有出版。她去世后，她的母亲因为财政困难，就把小说卖给了米高梅影片公司，但是该公司并未将它搬上银幕。直至1965年，小说的手稿易了好几位买主，才最终得以出版。
　　概要：《今日今夜》叙述的是一对有钱、漂亮、幸福的夫妇的故事。在喧闹的二十世纪，朱迪·兰斯多和彼得·兰斯多却过着浪漫的"社会"生活。一天，彼得不幸在一次车祸中双目失明，接着他们那个世界急剧崩溃。朱迪向丈夫隐瞒了他们已破产的消息，她与彼得的好朋友之间的旧情也死灰复燃。朱迪为了赚钱，不得不去夜总会扮演裸女高迪娃的角色。为了使丈夫确信自己是在白天参加慈善工作，她改变了彼得的时间感觉：让他在晚上吃早餐——这就是书名的含义。最后他得知了真情，他俩的婚姻差一点破裂，不过朱迪最终还是回到了彼得身边。

　　4．H·L·亨特唯一一部小说——《阿尔帕卡》——1960年
　　背景：亨特是世界上最富有的人之一：在他最后的40年生涯中，他每周收入超过100万美元。亨特认为加尔文·柯立芝总统是美国最后一位成功的总统，他花了无数金钱在报刊杂志和广播电台上，竭力宣传反共思想。《阿尔帕卡》就是宣传这种思想的一部作品。
　　概要：年轻、英俊的胡安·阿查拉住在一个小国阿尔帕卡，它由一个独裁者统治。胡安的双亲把他送往欧洲，希望他能遇见一些聪明而可敬的人，他们能帮助他为阿尔帕卡写一部新宪法。旅途中，胡安遇到美丽的阿尔帕卡女子玛拉，后者前去巴黎参加一部歌剧的首场演出。不久玛拉一举成为世界明星，但她旋又放弃了歌坛生涯。胡安写成了新宪法，他和玛拉结了婚，两人一起回国向人民宣传这部理想的宪法。

　　5．本尼托·墨索里尼唯一一部小说——《主教的情妇》——1909年
　　背景：1909年，这位意大利未来的独裁者正在奥地利统治下的意大利特兰迪诺地区做工会工作。他的职责之一便是协助编辑一份社会主义报纸。墨索里尼负责为报纸撰写连载小说，《主教的情妇》就是这样写成的，它于1929年正式在英国出版。
　　概要：小说写的是1648年特兰迪诺地区的大主教马德卢索及其情妇芭蒂茜拉的故事。芭蒂茜拉的父亲是大主教的密友和顾问。农民和教士都痛恨芭蒂茜拉和她的父亲，强烈要求放逐他俩，他们认为这样一来，大主教就会由一位专横的暴君变成仁慈的精神领袖。事实上大主教马德卢索根本不要这份权势，他只想和芭蒂茜拉结婚，可是梵蒂冈决不答应。最后芭蒂茜拉被政敌毒死，马德卢索一人过着孤独而不幸的生活。

　　6．哈里·李森纳唯一一部小说——《告诉我女人是什么》——1946年
　　背景：哈里·李森纳长期以来一直是电视的新闻节目播音员，他还主持哥伦比亚广播公司的"60分钟"节目。要说他会写爱情小说，真是难以置信。但事实正是如此，对于22岁的年轻人来说，这部小说写得异常老练——当时正值第二次世界大战期间，他在美国陆军服役。这部小说写得十分坦率，富有自传色彩。小说中充满机智、幽默而玩世不恭的对话，远远超过了同类作品。它的美妙与动人令人为他放弃了小说家的生涯而倍感惋惜。
　　概要：这是第二次世界大战期间的一段初恋故事。威尔逊是一位略微玩世不恭的小镇记者。他生活中已接触不少女人，但从未真正堕入爱河。后来他遇见了一位美丽、聪明、性格复杂的女子玛丽丝，他俩相爱，随即成婚。但不久威尔逊便发现，她为另一男子而怀孕，虽然孩子流产，但疑虑却笼罩在他心头：他怀疑玛丽丝在利用他。后来威尔逊应征入伍，他俩被迫分居，军旅生活使他成熟起来，使得他俩的离异成为定局。不久玛丽丝写信给威尔逊，称自己另有新欢，并要求离婚。但是当他请假回家时，她又改变初衷，恳求他回到自己身旁。可是甜蜜的爱情已经变酸，他意识到任何拖延的企图都是徒劳无益的，这时他永远离开了她。
