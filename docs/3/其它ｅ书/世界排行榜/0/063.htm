懂人话的10只动物

　　1．长毛猎狗阿列奇诺（1960）
　　伊丽莎白·曼是著名德国作家托马斯·曼的女儿，她是一位驯兽师。她曾训练了一条长毛猎狗阿列奇诺用打字机记录下她口述的单词，这条狗是用鼻子按按键的。它在1963年7月10月开始接受语言训练，6个月以后就能打20个单词。它的本领已经超过了单纯的听写练习，有一天伊丽莎白问它想去哪里，它打出了"car（汽车）"这个词。

　　2．蒙古狗班杰（1965-1978）
　　麻萨诸塞州的白茜·马库丝太太养了一条出色的蒙古狗，名叫班杰。它出世后9星期就会唱歌。它最拿手的歌是"雨水不停地洒在我头上"。这条狗曾在"沙利文表演赛"和"今夜表演赛"上出足风头。

　　3．蒙古狗克莉丝（1949？-1959）
　　克莉丝是一条产自格林威治的蒙古狗，它能用爪子来拼写单词：爪子抓一下，代表字母A，抓两下，字母B，依次类推。有一回，它竟然成功地预测了一次赛马的胜者，来自杜克大学的一群灵学专家专门对它进行了研究。经过一系列测试，证明克莉丝的确具有特异功能，它在预测自己的死期时，只相差1天。

　　4．聪明的汉斯（1890）
　　汉斯是埃尔伯费尔特地方的一匹俄国马，据说它会回答问题，辨认乐器，还会做算术。19世纪末，一个科学家委员会专门对它进行了研究，他们认为，这骑马会对主人卡尔·克拉尔下意识的微弱手势作出反应。有好几首流行歌曲以聪明的汉斯为题，孩子们也以它来命名自己的玩具马。

　　5．鹦鹉杰科（1827-1854）
　　杰科是一只非洲灰鹦鹉，它住在萨尔兹堡。就像当地许多鹦鹉一样，它也会说德语。然而，与众不同的是，看来它理解自己说的话。当主人独自离家时，它说：“上帝与你同在。”但是当主人与别人一起离家时，它会说：“上帝与你们大家同在。”

　　6．奇迹太太（1926-1957）
　　奇迹太太是一骑马的名字，它能用鼻子在一架巨大的打字机上拼出要说的话，这架打字机的按键是橡皮做的，它曾预测美国将卷入第二次世界大战；并且，和公众舆论相反，它正确地预言杜鲁门将再次当选。1951年，它指引警察找到了一位失踪两年的孩子的尸体，从而成了报界的头条新闻。著名灵学家莱茵博士在二十年代对奇迹太太作了研究后声称：这骑马具有特异功能。

　　7．海豚莉茜（1950？-1960）
　　莉茜是一条雌海豚，它于1960年4月16日死于著名神经生理学家约翰·李利的水池中。李利博士曾进行一系列人与海豚交际的实验。他觉得海豚有着比人类更高形式的意识；它们的语言是建立在一种与人类不同、但异常合理的逻辑之上。1968年，李利把所有养在池中的海豚都放走了，他说："我没有权利为了个人的方便把这些海豚囚禁在集中营里。”录音机曾录下莉茜的一句话，译成人类语言就是"现在是6点了"，或者是"这是个诡计"。

　　8．白马摩洛哥（1600）
　　在莎士比亚时代，一匹名叫摩洛哥的白马，令英国和法国的公众目瞪口呆：它会掷骰子、回答问题、猜年龄——它是靠敲打蹄子来表示的。据说它的主人因行施巫术而被判处死刑，当摩洛哥得知这一消息之后，它就跪倒在法官跟前，法官最后赦免了它的主人。

　　9．会讲话的乌龟（1978）
　　1978年夏天，乌干达的官员一再否认下述传言：一只会讲话的乌龟在首都堪帕拉郊外挨家挨户拜访。但是成千上万乌干达平民却说这是真事。维多利亚湖畔金加镇的警察曾报告说，那只乌龟来到警察局要求见局长。报告中未提乌龟为何事上门，我们只知道乌干达当时的独裁者阿明得知此事后大动肝火。他威胁说，谁敢再提此事，格杀勿论。

　　10．大猩猩瓦肖（1965）
　　它把敌人称作"肮脏的杰克"，还会自己造词：如称西瓜为"喝的水果"，天鹅为"水鸟"。它还能理解一些句法的雏形。这只名叫瓦肖的大猩猩是靠手势来进行交流的。当它1岁时，心理学家爱伦和彼特丽丝·加德纳教它使用手势语（一种聋哑人使用的语言），现在它住在俄克拉荷马的大猩猩保护区，正在教其他猩猩使用手势语。
