历史上15位最有影响的科学家

　　1．艾萨克·牛顿（1642-1727）　英国数学家和哲学家
　　他在天文学、光学、机械学和物理学方面的工作，为现代科学理论奠定了基础。

　　2．蔡伦（公元50？-118）　中国官吏
　　纸的发明者。

　　3．毕升（？-1051）　中国发明家
　　发明活字印刷。

　　4．阿尔伯特·爱因斯坦（1879-1955）　德国出生的美国科学家提出相对论。

　　5．路易·巴斯德（1822-1895）　法国化学家
　　细菌学创始人。

　　6．加利略（1564-1642）　意大利天文学家
　　为科学方法的发展作出重大贡献。

　　7．查尔斯·达尔文（1809-1882）　英国博物学家
　　进化论的提倡者。

　　8．欧几里德（前300）　希腊哲学家
　　奠定了平面几何的理论基础。

　　9．尼古拉斯·哥白尼（1473-1543）　波兰天文学家
　　现代天文学奠基人，提出地球及其他行星绕日运转。

　　10．詹姆士·瓦特（1736-1819）　苏格兰工程师
　　蒸汽机发明者。

　　11．米歇尔·法拉第（1791-1867）　英国化学家和物理学家
　　发现电磁变换原理。

　　12．詹姆士·克拉克·马克斯韦尔(1831-1879)苏格兰物理学家
　　提出光的电磁理论公式。

　　13．奥维尔·莱特（1871-1948）　美国发明家

　　14．维尔勃·莱特（1867-1912）　美国发明家
　　制造出第一架成功的飞机。

　　15．安东尼·劳伦特·拉瓦锡（1743-1794）　法国化学家
　　现代化学的奠基人。
