5个打牌时死去的人

　　1．约翰·贝内特（香水代理商）
　　贝内特先生和他妻子默特尔住在堪萨斯城一套非常时髦的公寓内。在1929年夏季的一个不幸的下午，贝内特夫妇邀请霍夫曼夫妇一起打桥牌。霍夫曼太太事后回忆道：“当时打着打着，贝内特夫妇开始互相埋怨对方，而且语言越来越尖刻。"后来有一副牌，贝内特先生叫错了一张牌，他本不该开叫，却叫了1梅花，霍夫曼先生叫2方块。这时轮到贝内特太太，由于她急于想成约，就叫了4梅花。出牌时，贝内特先生又出错了一张牌，他妻子骂了他一句，接着两人就吵了起来。争吵中贝内特伸手打了一记默特尔的耳光，忿怒的默特尔骂他是流氓，贝内特更加恼火，他威胁着说要离家而去。默特尔一边请霍夫曼夫妇回去，一边怒气冲冲地冲进她母亲的卧室，还没等霍夫曼夫妇出门，她就抓起一把自动枪跑回来，对准丈夫"砰！砰！"开了两枪，贝内特当场死去。有趣的是，贝内特要是在吊王牌前先做成一套黑桃长牌的话，那晚也许不致于丢命。

　　2．詹姆士·布特勒·希考克（枪手）
　　1876年6月当外号为"粗野的比尔"的希考克进入南达科他州人称"死亡森林"的黄金峡谷时，他有一个预感，就是自己也许永远无法活着出来。8月2日那天，他同3个朋友在一家酒吧打牌，一时间笑声朗朗，大家玩得很开心。希考克每次都是坐在背朝墙的那个位子，但是那天下午查理·利奇跟他开玩笑，故意占了他的位子，而且不让给他。这时杰克·麦考尔走进了酒吧，他曾在那天上午同希考克打过牌，而且输给了希考克。麦考尔一进门就拔出一把直径为0．45的自动枪，对准希考克的后脑就是一枪，希考克当场死去。此时他手上正握着一对A和一对8，从此人们就把这副牌称作"死亡之牌"，他死时脸上还带着微笑，那年他才39岁。

　　3．阿尔·乔尔逊（演员）
　　1950年10月23日，乔尔逊同朋友在旧金山圣弗兰西斯旅馆打牌对，突然心脏病发作，当场死去，享年64岁。

　　4．巴士特·基尔顿（喜剧演员）
　　1966年1月31日下午70岁的基尔顿在好莱坞家中同人打牌时突然发病，抢救无效，于第二天早晨去世。

　　5．阿诺德·罗思坦（赌徒）
　　这位曾操纵1919年世界棒球系列赛的奇人，在纽约市公园中心饭店汉普·麦克马纳斯包租的套房内赌牌时，突然受到枪击，腹部中弹。他曾在前一场赌博中欠下麦克马纳斯32万元。此时他不顾疼痛挣扎着冲出大楼，立即让人把他抬进医院，他于1928年11月4日停止呼吸，临死前他始终不肯说出开枪杀他的凶手。
