历史上10位玛丽·麦卡锡最愿意成为的人

　　“很可能是美国历史上最聪明的女子"，一个评论家说道。麦卡锡毕业于瓦萨学院，1966年她发表了小说《那一群人》，描述了瓦萨学院1933年毕业的7名女子的生活。该书给她的母校瓦萨学院带来了很坏的名誉。
　　1．托马斯·杰弗逊
　　2．拜伦勋爵
　　3．卡图卢斯
　　4．伊丽莎白一世
　　5．布鲁因莱斯基
　　6．圣奥克斯丁
　　7．狄德罗
　　8．托马斯·纳什
　　9．克利斯蒂·戴齐塞恩
　　10．阿基坦的埃莱诺
　　注：卡图卢斯（前84？-前54）是罗马抒情诗人，描写自己不幸的爱情的自传性诗篇特别出名。布鲁因莱斯基（1377-1446）是意大利文艺复兴初期的杰出的建筑师。法国作家狄德罗（1713-1784），编写了巨著《百科全书》，该书收列了各种实用知识和18世纪的政治思想，托马斯·纳什（1567-1601），是伊丽莎白时代的作家和剧作家，他曾一度同克里斯托佛·马娄合作。克里斯蒂·戴齐塞恩（1363？-1431）曾写过大量的爱情诗歌，一部法国查理五世的传记以及几部歌颂妇女的作品。
