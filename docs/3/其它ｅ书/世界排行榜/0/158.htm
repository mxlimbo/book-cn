10位难以相处的婆婆或丈母娘

　　1．瓦丽德·努尔·巴努太后（？-1583）
　　她是奥斯曼帝国苏丹"傻瓜"赛利姆的寡妻，在她暮年，她千方百计离间儿子姆拉德二世皇帝和他最钟爱的皇后，威尼斯女子萨菲·芭福。然而萨菲最终挫败了婆婆的阴谋，1595年姆拉德去世后，萨菲使用大胆而微妙的策略，使她的亲生儿子在姆拉德的103位儿子中脱颖而出，成为奥斯曼皇帝穆罕默德三世。

　　2．玛丽·瓦索（1674-1767）
　　当著名哲学家卢梭终于娶了出身贫寒的女仆赛莱丝·瓦索为平时（他俩于1745年相遇，于1768年最后成婚），他同时也必须接受那位贪婪成性的丈母娘。卢梭当时主要靠抄写乐谱为生，那点微薄的收入中的大部分都掉进了丈母娘玛丽的腰包。伏尔泰于1764年写了一本小册子，极大地损害了卢梭的声誉。伏尔泰声称卢梭是个毫无心肝的人，他的丈母娘是被他折磨而死的。然而事实并非如此，卢梭一直善待这位岳母，她活到93岁高龄才离开人世。

　　3．阿格妮丝·斯特恩（？-1759）
　　英国作家劳伦斯·斯特恩被他那位既咄咄逼人、又贪得无厌的母亲弄得焦头烂额，后者老是向他有钱的妻子要钱。而他的妻子伊丽莎白·斯特恩本人也是一位既刻薄、又爱争吵的人，常被婆婆的无理要求弄得大发脾气。斯特恩对母亲的行为大不以为然，甚至当她因流浪而被捕入狱时，他也不愿出钱保释。

　　4．朱丽安·玛丽公主（1729-1796）
　　这位守寡的公主是柔弱而半疯狂的丹麦国王克利斯蒂安七世的继母。王后卡洛琳·玛蒂尔达是英国国王乔治三世的妹妹，她有一个情夫，即国王的医生约翰·斯特卢恩西伯爵。朱丽安·玛丽将这一切都看在眼里。当卡洛琳王后和斯特卢恩西着手在丹麦推进民主改革时，这位王太后发动了一场宫廷政变，王后被捕，她的情夫被砍了脑袋。卡洛琳王后遂被放逐到德国的汉诺威，并于1775年她仅24岁时去世。

　　5．慈禧（1835-1908）
　　慈禧是中国清朝咸丰皇帝的妃子，很少有人胆敢冒犯她的权势。咸丰死后，慈禧的亲生儿子同治继位，但不久他便死于天花。同治的皇后是位精明能干的女人，很可能会摄理朝政，然而她却在慈禧的撺掇下吸了过量的鸦片。她的死亡使慈禧能够按她的意愿选择一位新皇帝。

　　6．弗朗茜丝·丘吉尔（？-1899）
　　伦道夫·丘吉尔勋爵的妻子，英国首相温斯顿·丘吉尔的母亲吉妮·丘吉尔总是无法取悦自己的婆婆弗朗茵丝·丘吉尔。也许是因为这位媳妇比婆婆——她是位公爵的女儿—-更加可爱、端庄。吉妮曾这样概括婆媳间的关系：“我俩彼此间永远彬彬有礼，但这更像是一座随时都会喷发的火山。”

　　7．萨拉·迪拉诺·罗斯福（1854-1941）
　　整整3年中，萨拉一直竭力反对她钟爱的儿子富兰克林·罗斯福（后来的美国总统）与艾莲诺之间的婚姻。不过两人结婚后，艾莲诺对婆婆还是很尊敬的。可是后来富兰克林·罗斯福因病致残，婆媳间为他的未来生涯爆发了争执：萨拉要儿子退休，而艾莲诺要丈夫投身政界。艾莲诺的密友卡洛琳·菲利普这样写道：“这位富有魅力、仪态万方、可蔼可亲的老太太内心深处隐藏着对媳妇的妒忌，这种妒忌有时会以最粗鲁的形式迸发出来。”

　　8．玛琪·盖茨·华莱士（1862-1952）
　　她是一位难以驾驭的女人，自从她女儿嫁给哈里·杜鲁门为妻后，她就一直与女儿女婿住在一起，直至33年后她以90岁的高龄去世。她从来就看不起杜鲁门，即便他后来当上了美国总统后依然如此。她曾当着杜鲁门的面说：要是换个别人也许能成为更好的总统，这"别人"大概指的是"好人杜威。"不过杜鲁门也真是个好女婿，他闭着嘴一声不吭。

　　9．玛丽，英国国王乔治五世的王后（1867-1953）
　　当国王爱德华八世宁可抛弃江山，执意与"我所爱的女人"——曾两度离婚的美国人瓦莉丝·辛普逊结婚时，英国王族拒绝承认这位新娘。第二次世界大战结束后，温莎公爵夫人瓦莉丝写信给她的婆婆玛丽，希望能改善两人之间的关系。玛丽的回答是冷冰冰的沉默。

　　10．莉丽安·卡特（1898-　）
　　在吉米·卡特投身海军之后，他的妻子罗莎琳不愿回到乔治亚州的老家，因为，正如卡特后来写道：“我们因结婚而获得的自由很可能受到亲属的遏制或支配，特别是她的母亲或我的母亲。"莉丽安曾抱怨说，卡特更关心罗莎琳，而不太关心她。有一回她对记者说：“我爱罗莎琳，不过她有一点……孤僻。”
