9位在科学实验中受伤或死亡的人

　　1．加利略（1564-1642）　意大利天文学家
　　加利略改进的望远镜打开了人类观察宇宙的视野，但是望远镜也毁了他原已十分衰弱的视力。他长时期用望远镜观察太阳，这使他的视网膜受到不可弥补的损害，这也可能是最终令他失明的原因。

　　2．卡尔·威尔海姆·希尔（1742-1786）　瑞典化学家
　　希尔发现了数量令人难以置信的元素和化合物，但是他养成了一个危险的习惯：嗅或尝他所发现的物质。他曾品尝过氰酸的滋味而大难不死。然而他毕竟在劫难逃，死亡时的朕兆与水银中毒极为相似。

　　3．让·弗朗索瓦·彼拉特·德·洛西埃（1756-1785）　法国物理学家
　　洛西埃是一位雄心勃勃的物理学家和化学家，他一心一意想成为法兰西科学院院士。他曾发明了矿工使用的防毒面具，但是这未能使他当院士的抱负如愿。1783年9月，蒙哥尔佛兄弟在巴黎进行气球飞行实验，洛西埃认为这是出人头地的大好机会。洛西埃自告奋勇成为气球上的第一个人，并于11月21日进行了首次自由放飞。他希望自己能成为第一个乘气球横渡英吉利海峡的人，但是．．布朗歇和约翰·JP杰夫利博士击败了他。可是他并不气馁，于1785年6月15日试图飞越英吉利海峡。他自己设计了绳索和吊篮，并使用两个气球。一个气球中充氢气以增强提升力，另一个气球中充热空气，以方便升降。不幸的是，在1700英尺的高空，加热空气的火焰点燃了极易燃烧的氢气，洛西尔及其助手坠地而死。

　　4．亨弗利·戴维爵士（1778-1829）　英国化学家
　　这位用电解法分析化学物质的创始人，可以说是一直在自取灭亡。年轻的戴维在当药店学徒时就被老板撵走，因为他老是引起爆炸。后来他从事化学研究，养成了吸进各种气体的习惯。正是这一习惯使他偶然发现了氧化氮的麻醉作用，但是这习惯也数次令他几乎丧生。化学物质的毒性使他在生命最后的20年中成为残废。1812年的一次氯化氮爆炸严重地损坏了他的双目，使他从此难以进行化学研究。

　　5．大卫·布鲁斯特（1781-1868）　苏格兰物理学家
　　布鲁斯特是万花筒的发明者，他主要从事光学和光谱研究，这一领域理所当然需要良好的视力。不幸的是，1831年的一次化学物品爆炸使他几乎失明。后来他虽然恢复了部分视力，但是眼疾在他余生中一直折磨着他。

　　6．米歇尔·法拉第（1791-1867）　英国化学家和物理学家
　　戴维博士的眼睛在1812年受伤，但这却导致了一个意想不到的结果：法拉第成了他的秘书，后来又成为他的受护人和竞争对手。法拉第改进了戴维的电解技术，在电磁领域作出诸多发现。然而法拉第似乎也继承了他的保护人的不幸：他与戴维一起工作时，曾因氯化氮爆炸而损坏双眼，而且还饱受化学品慢性中毒之害。

　　7．罗伯特·威廉·本生（1811-1899）　德国化学家
　　本生气初致力于有机化学研究，不过很快就改变了研究领域。他两度因砷中毒几近死亡。1843年，他因氰化砷爆炸失去了右眼。此后他转向无机化学，并创立了光谱学，这门科学导致许多元素的发现。实验室中一种常用的灯——"本生灯"就是以他名字命名的。

　　8．伊丽莎白·弗列希曼·阿夏姆（1859-1905）　美国X光技师
　　阿夏姆是美国第二位X光的受害者，也是第一位女性受害者。（光的第一个受害者是克莱伦斯·麦迪逊·达利，爱X迪生的助手，死于1904年。）阿夏姆自1897年开始在她姐夫开办的诊所里任X光技师。整整7年里她一直操纵X光机，而没有采取任何保护措施。她不仅在实验与工作时使自己暴露在X射线里，而且还时常将X光对准自己，向患者证明这种射线是无害的。1904年，她得了严重的皮肤癌，并迅速扩散到全身，阿夏姆于1905年做了手臂截肢，并于同年晚些时候去世。

　　9．玛丽·斯克罗多斯基·居里（1867-1934）　波兰出生的法国化学家
　　放射性毒害最著名的受害者当然要数玛丽·居里了，她曾两度获得诺贝尔奖。居里夫人和她的丈夫彼埃尔（1906年死于车祸）于1898年发现镭。从此，居里夫人就将毕生精力投入放射性研究以及射线的治疗作用。由于她接受了过量的射线，结果患了白血病，于1934年在痛苦中去世。
