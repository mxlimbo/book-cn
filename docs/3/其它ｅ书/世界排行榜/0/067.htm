12位著名的爱猫者

　　1．利蒙德·钱德勒（1888-1959）　美国作家
　　三十年代，钱德勒养了一只黑色的波斯猫，名叫塔基。他经常对它说话，仿佛它是一个人。他有时把塔基称作秘书，因为这只猫时常坐在他的手稿上打盹。

　　2．温斯顿·丘吉尔爵士（1874-1965）　英国政治家
　　丘吉尔晚年，得到一只姜黄色的小猫，名叫约克。丘吉尔喜欢和小猫一起进餐，他时常让仆人把约克找来，这样午餐就能开始了。约克对年迈的主人是个莫大的安慰，它常常睡在丘吉尔的床上。

　　3．科莱特（1873-1954）　法国作家
　　科莱特住在法国南部，她家里总养着不下10只猫，其中还不算常来"串门"的邻居家的猫。她以猫为主题写过好几篇故事，其中包括著名的《猫》。她经常对人说，是这些猫教会了她如何自我控制，以及如何进行"一种高尚的欺骗行为"。

　　4．撒缪尔·约翰逊（1709-1784）　英国词典编纂家
　　约翰逊博士的传记作者詹姆士·鲍斯韦尔写道：“我怎么也忘不了他［约翰逊］对那只名叫'豪猪'的猫的宠幸。他经常出外为它买牡蛎，唯恐仆人们会嫌麻烦而对'豪猪'产生怨恨之心。”

　　5．罗伯特·E·李（1807-1870）　美国将军
　　李将军的家书中时常提到猫。在墨西哥战争期间，他曾去信给女儿，要她送只猫来作伴。

　　6．利奥七世（1760-1829）　意大利教皇
　　米塞托出生在梵蒂冈，由教皇养大，它是教皇的宠物，时常踡身躲在教皇那件长袍里。教皇死后，法国作家夏多布里昂收养了这只猫，据说，这只猫的毛皮至今仍保存在法国的一个博物馆里。

　　7．穆罕默德（570-632）　阿拉伯宗教领袖
　　这位伊斯兰教的创始人对猫另眼相看。有一次在祈祷时，他为了不打搅猫，挥剑砍去了长袍的一只袖子。

　　8．彼特拉克（1304-1374）　意大利诗人
　　彼特拉克晚年的伴侣是一只猫，他死后，这只猫也被杀死，并制成木乃伊。如今，它依然保存在一个壁龛中，壁龛上饰有一只大理石制成的猫，还刻着据说是这位诗人的亲笔题词：“仅次于劳拉。"劳拉是彼特拉克一直钟爱的女子，可是她究竟是谁，历史学家始终未能确定。

　　9．埃德加·爱伦·坡（1809-1849）　美国作家
　　这位作家写过一篇著名的恐怖故事：《黑猫》。不过，在现实生活中，爱伦·坡却十分喜爱猫。他和妻子弗吉尼娅无论到什么地方，总要带着那只名叫卡塔琳娜的猫。当弗吉尼娅得了肺病即将死去时，那只猫紧紧地依偎在她身旁，尽可能给她一点温暖。

　　10．黎希留大主款（1585-1642）　法国政治与宗教领袖
　　这位主教大人专门辟了一间猫室，每天他的仆人阿贝尔和特桑弟尔将白色的鸡肉片送入那里喂猫。黎希留死后，遗嘱里还给两位仆人和那些猫留下一笔年金。

　　11．阿尔伯特·史威泽（1875-1965）　法国医生，传教士
　　史威泽医生十分爱猫，对于虐待猫的人恨之入骨。他是个左撇子，但他却时常用右手开处方，为的是不惊动那只躺在左臂上睡觉的猫。

　　12．南安普顿伯爵第三（1573-1624）　英国冒险家
　　伊丽莎白一世统治末期，这位伯爵被关进了伦敦塔里。他的猫从烟囱里爬进囚室与他作伴。伯爵后来被詹姆士一世释放。为了纪念那只猫，他让人给他画肖像时，以黑白猫作为背景。
