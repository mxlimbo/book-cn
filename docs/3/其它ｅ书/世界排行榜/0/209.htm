8个巨型干酪

　　1．重17．5吨的切达干酪
　　这个以"金色巨人"的名称而闻名于世的全球最大干酪长14．5英尺，宽6．5英尺，高6英尺。1964年由丹麦斯蒂夫奶酪公司为美国威斯康星奶酪基金会制造。总共耗费了183吨牛奶——相当于16，000头母牛1天的产量。运输是通过一部特制的名为"奶酪机动车"的牵引拖车进行的，当时它被运抵纽约世界博览会内的威斯康星展馆。存放在一只四周是玻璃的冷冻箱内。1965年人们把它切成两磅左右的小块，1968年又拿出向公众展示，并以每包3美元的价格出售，1978年在威斯康星奶酪制造协会召开的集会上，剩下的两块以每块200美元的价格被拍卖掉。

　　2．重6吨的切达干酪
　　1937年7月12日一批纽约州北部的奶酪师们在W·L·基而西的指导下开始制作这块超大型奶酪。经过烘烤以及不停地翻动、整整花了7个星期才使它熟透。制作这个奶酪总共用了6，000头母牛产的奶，它被运到西那库斯城、放在纽约州博览会上展出。

　　3．重4吨的切达干酪
　　这只由加拿大奶酪师制作重达8000磅的干酪在1883年的多伦多博览会上展出时引起了轰动。著名殡葬诗人詹姆斯·麦金太尔为之赋诗一言：
　　　　你，巨人般的乳酪，
　　　　安详、无声地躺在我们的面前，
　　　　晚风轻轻地吹拂你美丽的身躯，
　　　　没有一只苍蝇敢停下把你叮咬。

　　4．重1400磅的切达干酪
　　这个约四分之三吨重的巨大干酪是纽约州一名奶酪师特地为安德鲁·杰克逊总统制作的。它被放置在白宫的前厅，用了将近两年的时间才使它变得熟透。在美国前总统华盛顿生日的那一天，杰克逊敞开大门，让华盛顿全城的居民进来享用。据目击者说，当时白宫周围方圆半哩内都能闻到这干酪的诱人香味。干酪在两小时内就被分享完毕，只剩下很小的一块算是留给总统品尝。

　　5．重1200磅的切希尔干酪
　　1801年，托马斯·杰佛逊总统收到了麻省伯克夏一个叫做切希尔的小镇送来的一只特大干酪。这只名为"超民主，反联邦"的切希尔干酪，通过雪橇、轮船和马车几次转运才运抵华盛顿州。切希尔居民赠送此物的目的是赞扬杰佛逊总统反对联邦主义集团的胜利。这次赠送运动的发起者是一个牧师，名叫约翰·利兰德，他利用全国民众对这件事情的注意，大力宣传他所信奉的教会，杰克逊很受感动，他本人也向利兰德的教会捐赠了200元。

　　6．重1100磅的切达干酪
　　这是1840年送给维多利亚女王的结婚礼物。而对这个直径长90英尺、重量达半吨的庞然大物，女王感到有点惊讶和不安，不知如何处置才好。幸好干酪的制造者们要求先借用一下，打算把它运到各地作巡回展览，女王欣然表示同意。可后来当他们准备把这只已沾满灰尘和污垢的巨大干酪重新送回王宫时，女王拒绝接收。接着围绕着干酪究竟该放置何处发生了长时间的争执，最后决定存入英国档案馆，在档案馆里这个巨大的干酪也就莫名片妙地变得越来越小、直至完全消失。

　　7．重1000磅的鲁尼干酪
　　这是个鲜为人知的古代奇迹。根据公元77年普林尼斯著的《自然史》的记载，这个以意大利北方古城命名的干酪是在意大利中部著名的卡拉拉大理石矿附近一个今天叫做塔斯卡尼的地方制造的。鲁尼干酪很可能是牛奶和羊奶混合而成。因此既带有切达奶酪的味道又带有派米山奶酪的口味。

　　8．重1000磅的切达干酪
　　该奶酪从新西兰运抵伦敦，横跨半个地球，创路程之最。它在1924年温布利博览会上引起了轰动。
