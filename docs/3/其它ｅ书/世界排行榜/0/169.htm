11个涉及性丑闻的宗教人物

　　1．德尔斐的一个普西亚（前七世纪）　希腊女预言家
　　在德尔斐神谕的早期，一些年轻美貌的处女被选做普西亚，做普西亚的人必须发誓终生禁欲。普西亚就是采天地之灵气并能作出预言的女祭司。大约在公元前七世纪，一天，一个叫伊克筮德的希腊人走进德尔斐庙，向普西亚提问，可他没想到普西亚竟是如此美貌的女子，顿时情欲大发，无法控制，他把普西亚按倒在地，并当场强奸了她。这事激怒了德尔斐人，后来他们只任命50多岁、相貌一般的女人担任普西亚。

　　2．约翰七世（938？－964）意大利教皇
　　这个年仅18岁的教皇不断盗窃教会的财产来满足他日益增长的赌博嗜好，他还通过雇用一批暴徒来维持他的统治。但他荒淫无耻的生活导致了他最后的倒台。他有过许多情妇，数目之多以至批评者说他把神圣的殿堂变成了妓院。有人说甚至看到他就在圣彼得大教堂里强奸前来进香的信徒。公元964年5月的一天，他同一个情妇通奸，当场被情妇的丈夫抓住，此人忿怒无比，他对这个掌有通向天堂钥匙的教皇毫无畏惧，把他狠狠地揍了一顿，3天后这个教皇既没作忏悔，也没接受神礼就离开了人间。

　　3．亚历山大六世（1431-1503）　意大利教皇
　　亚历山大生于西班牙的瓦伦西亚，幼年时名为罗得利戈・波吉亚。1492年8月11日他通过贿赂，收买了部分主教，从而在选举中得到足够的选票，登上教皇宝座。当时的亚历山大已有7个私生子，其中4个是他长期的情妇卡塔奈所生。当他62岁时，他又在众多的情妇中增加了一个。这个名叫吉奥莉亚・法妮斯的女子年仅16岁。当时他们之间的关系在教会内外都只是公开的秘密。

　　4．赫尔德里克・茨温利（1484-1531）　瑞士宗教改革领袖
　　茨温利在1506年至1515年担任瑞士格拉勒斯地区主教期间，曾犯过他自己所称作的"禁欲不彻底的过失"。事实上他同教会里好几个女人有性爱关系。为了抑制自己的情欲，他曾发誓决不同女人过多接触。后来当他的一个女友自豪地向村民们透露她同主教有性关系时，茨温利不得不向他的上司递交忏悔书。在他同罗马天主教决裂时，梵蒂冈为了把他弄臭便将此信公布于众。但这则丑闻并没能将茨温利搞垮，他仍然成为政治和宗教领袖。

　　5．约翰・汉弗莱・诺伊斯（1811-1886）　至善主义派牧师
　　由于公理会教会拒绝任命他为牧师，约翰・汉弗莱・诺伊斯自己单独创建了至善派教会，该教会的信念是上帝要求人类实行完美的爱并分享这种爱。1847年当警察当局获悉这种爱和爱的分享原来就是互换妻子时，他们就以通奸罪逮捕了诺伊斯和他的一些追随者。尽管此事成了一件大丑闻，诺伊斯后来仍然在纽约的奥尼达地区成立了一个名叫"奥尼达地区"的组织。在该组织中，他和300个信徒之间实行"交叉婚姻"，也就是说，组织里的任何一个男人和任何一个女人都可以过夫妻生活，诺伊斯这种自由性爱的原则不断遭到社会上的抨击，可他这个组织却推持了将近30年。

　　6．亨利・皮切（1813-1887）　公理会牧师
　　1874年宗教领袖和社会改革家亨利・皮切因犯通奸罪而受到起诉和审判。该丑闻当时震动了整个美国。向皮切牧师提出指控的是他的布鲁克林公理会教会中的一个叫西奥多・蒂尔顿的成员，他指责皮切同他的妻子发生关系。从此后提供的证据显示，皮切是在教会办公室对蒂尔顿的妻子进行诱奸的。他说上帝希望他俩发生关系。为时4个月的审判经过52次投票，结果使陪审团大为沮丧，因为皮切仍旧回到他的教会，只是名誉稍有损害。

　　7．霍雷肖・阿尔杰（1832-1899）　唯一神教牧师和作家
　　1864年霍雷肖・阿尔杰担任了麻省布鲁斯特教区的牧师。他年轻、精力充沛，因此很受大家喜爱，他为教区的男孩们组织游戏和各种各样的娱乐活动，但在他担任牧师15个月后，教会的成员们发觉他对女孩子一点不感兴趣，而当时也有谣传说他对男孩子特别偏爱。教会的领导决定调查此事，后来果然发现他同至少2个以上的教区居民的儿子有同性恋关系。阿尔杰承认了他的鸡奸后，牧师的职务遂被撤销。他离开教区，到纽约定居。行为从此他开始了写作的生涯。他后来成为美国最成功的作家之一，特别是以他在小说中塑造的"邋遢鬼迪克"和"游荡者费尔"等人物而出名。

　　8．迪万神父（1877-1965）　和迫使命运动创建人
　　根据警察局的记录，这个黑人宗教领袖的真名是乔治・贝克。他命令成千上万个和迫使命运动的信徒，无论是否结婚，都必须禁欲。但他自己却卷入了一系列的性丑闻中去。1931年他因同一女人睡觉而在纽约长岛被警方逮捕。另外据报道他还诱奸了好几个他的信徒，他对她们说：“我是在让你们把性欲充分表现出来，这样就可以帮你们根除性欲。"1946年当69岁的贝克娶了年仅22岁的加拿大白人妇女为平时，进一步激怒了美国的白人公众。

　　9．艾梅・麦克弗森（1890-1944）　福音主义传教士
　　这位对舞台艺术很有研究的福音主义传教士创建了国际弗斯奎尔福音教会。1926年5月她在加州桑塔莫尼卡海滩突然失踪，警方和公众都以为她是下海游泳时被淹死了。但是1个多月后，她又在墨西哥的一个边境小镇重新出现，她说自己是被绑架走的。不久后才知道她原来是在加州卡迈尔地区的一个蜜月别墅同一个名叫肯尼思・奥尼斯顿的已婚男子同居了10天。令人惊讶的是艾梅竟然安然地度过了这则丑闻所引起的风暴，而且这位福音传教士的名声还越来越响。

　　10．普里・哈吉斯（1925-　）　基督教十字军的创建者
　　哈吉斯曾任右翼通俗广播节目的福传教士，并在奥克拉荷马州土耳沙市创立了美国基督教学院。他一贯大声疾呼反对电视台播放性的节目，并布道反对色情文化。他还指责说摇滚音乐是共产主义者的发明。可1976年时代杂志披露哈吉斯不但同学院里的女学生，还同男学生有性关系。他的这种双重性恋的秘密直到一对夫妻在新婚之夜互相得知对方都与哈吉斯发生过性关系时才真正得以暴露。面对别人的指控，这位原教旨主义者只是把自己的性行为归咎为自己的"基因和染色体"，他目前仍在领导着基督教十字军。

　　11．小克劳迪厄斯・弗米利耶（1929-　）　圣公会牧师
　　以"兄弟"这个名字著称的弗米利耶曾管理过博伊斯农场，这是设在田纳西州温切斯特附近的一家收容所，专门收容有困难或麻烦的青少年。根据揭发他曾同好几个受他看护的少年发生性关系，其中有些甚至不满11岁。据说他还邀请一些有钱的朋友前来同农场的男孩进行淫乱活动，有时他把这些活动拍成电影，然后把电影卖给该收容所的一些赞助人和倡导人，在这些买主中，有的甚至远在沙特阿拉伯居住。最后一个15岁的少年向当局揭发了整个情况，弗米利耶随后遭到逮捕和审讯，并被判处25年至40年的徒刑。目前他仍在上诉，要求改变判决。
