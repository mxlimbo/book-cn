历史上8艘最令人胆寒的战舰

　　1．瑞典战舰"瓦萨号"（1628）
　　瑞典海军觉得有必要建造一艘巨型战舰，它有两层甲板，配备64门大炮。这艘超级战舰于1628年8月2日第一次试航。当它缓缓驶离斯德哥尔摩港时，舰上全体水兵向前来观看的瑞典国王致敬。这时，一阵狂风吹来，舰身立刻倾斜。接着"瓦萨号"又慢慢恢复AE？衡，但随即再一次朝右舷倾斜。在众目睽睽之下，"瓦萨号"的下层甲板慢慢进水，最后竟然沉入水底，50多名水手落水淹死。该舰打捞工作自1956年开始，到1961年打捞完毕。现在人们可以在斯德哥尔摩港重睹它往日的风采。

　　2．法国潜水艇"布隆湟号"（1863）
　　这艘早期潜水艇是在法国的洛歇港制造的，艇身长140英尺，重420吨。潜水艇靠压缩空气驱动，艇上配备了鱼雷。当它浮在水面上时，一切都很好，但一旦沉入水下，问题便接踵而来。"布隆湟号"在水下无法保持平衡，艇身在下沉过程中剧烈晃动，令全艇官兵惊恐万状。经过7次不成功的试验之后，法国海军当局宣布，为了保证水兵们的生命安全，这艘潜水艇立即退役。

　　3．美国南北战争期间南军潜水艇"亨利号"（1864）
　　这是一艘锅炉改装的潜水艇，长60英尺，外形就像一支雪茄。它靠人力驱动，8个健壮的汉子摇动连接推进AE？的曲柄，就能产生足够的动力。这艘潜水艇不啻是一个死亡陷阱，包括发明人HL・・亨利在内的10多人，先后在试航过程中丧生。1864年2月17日，"亨利号"攻击了北军战舰"豪萨托尼克号"，并将之击沉。然而，胜利者"亨利号"竟然与敌舰双双沉入海底。

　　4．英国军舰"船长号"（1870）
　　“船长号"由海军中尉库柏・科尔斯设计，是一艘远洋军舰。舰上火器包括2门双筒12寂谒。军舰由钢铁制成，吃水线附近还包有铁甲。然而甲板过低是"船长号"的致命伤，它于1870年9月6日在西班牙菲尼斯特海岬外遇到暴风雨，舰身立刻倾覆，连同其设计者本人一起葬身鱼腹，舰上500余人中只有18人生还。

　　5．美国巡洋舰"维苏威号"（1888）
　　“维苏威号"由爱德蒙・扎林斯基制造，这位扎林斯基意欲将成千上万磅炸药投向敌舰，然而设计者犯了一个不可饶恕的错误：舰上的火炮竟然不能自由转动，若要瞄准敌舰，就必须令整个舰身转动。因此，"维苏威号"连保护自己的能力都没有，更别说进攻敌舰了，所以它不久便悄然退役。

　　6．美国铁甲舰"卡他J号"（1893）
　　为了节省弹药，美国政府中一些人异想天开地设计了"卡他J号"铁甲舰，当它高速向敌人大军舰冲去时，坚固无比的船头能将敌舰撞沉。然而下水之后，人们才发现"卡他J号"根本派不上用场，它的航速太低，调头也很不灵便，加之它的自我保护能力为零。最后，英国海军部只得忍痛割爱，把它用作"实验靶A号"，在它下水后的第16年，一阵猛烈的炮火，无情地把它摧毁在拉帕哈诺克水域。

　　7．英国巡洋舰"无敌号"（1907）
　　这艘战舰的命运是对其舰名的一大讽刺。"无敌号"是由约翰・费什勋爵设计的，他曾说过：“速度是最好的自卫。"为了获得25节的高速，舰身改用薄铁甲。事实上，这艘巡洋舰毫无用处：薄薄的铁甲根本挡不住敌军炮火的袭击，舰上炮塔的转动齿轮也有缺陷，8门火炮一门也不能自由瞄准。这一点在1916年日德兰海战中得到了证实。当时，"无敌号"和德国巡洋舰"德芙琳格"号遭遇，对方的第一阵炮火便令"无敌号"爆炸，并裂成两截，立刻沉没。

　　8．英国轻型巡洋舰"愤怒号"（1916）
　　“愤怒号"上装有2门18伎诰兜幕鹋冢对于这样一艘轻型战舰而言，这对火炮实在是一场灾难。只要它们一开始射击，舰身的震动就会使铆钉四处乱飞。因此，"愤怒号"根本无法作战，加之它的铁甲太薄，在海上遇到风浪时，舰身就会有多处凹陷。第一次世界大战结束后，这艘巡洋舰被改装成了一艘航空母舰。
