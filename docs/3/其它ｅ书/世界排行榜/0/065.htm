8种吃人的动物

　　1．熊
　　北美黑熊，虽然个头不如棕熊那样巨大，但却常常攻击人类。1963年，阿拉斯加的越桔浆果年景很糟，饥饿的黑熊就开始攻击人类，并杀死1人。

　　2．鳄鱼
　　河流入海口的鳄鱼是世界上最凶猛的食人者，每年要杀死2000余人。1945年2月19日，它们造成了世界上最大的一起动物攻击人类事件。当时英军包围了一支日本军队，日本士兵——其中大多数人负了伤——退到了孟加拉湾附近的一处沼泽。机关枪声和血腥味引来了几百条鳄鱼，整个晚上，英国人听见沼泽地里一片惨叫声。第二天早晨，只有20名日本人还活着。

　　3．大王乌贼
　　大王乌贼是软体动物中最高等的种类，它们的眼睛酷似人类眼睛。它有10条手臂，全身展开时长度达65英尺。人们经常将它们和大章鱼混为一谈，其实章鱼只有在受到威胁时才会攻击人类，而大王乌贼却是一种食肉猛兽。1941年3月25日，英国轮船"布列塔尼亚号"在大西洋上沉没，正当十几位幸存者抓住一艘救生艇时，水里出现一只大王乌贼，它伸出长长的手臂，缠住1人的身体，将他拖进水底。雄性大王乌贼在交配后偶尔会将雌乌贼吃掉。

　　4．豹
　　豹是最危险的捕猎对象，它奔跑的速度快，而且行动诡秘，不易被人察觉。印度中部经常有豹袭击农舍伤人的事情发生。其中有一只叫做"帕那瓦食人者"的豹，至少杀死过100多人。这只豹于1910年被吉姆·科贝特打死。次年，科贝特又打死了钦帕瓦特一只食人的雌虎。

　　5．狮子
　　狮子和老虎一样，通常并不袭击人。狮子攻击人时一般是成群结队出动的，不过也有单身狮子吃人的报道。1943年10月，一只狮子在赞比亚的卡萨马地区被人打死，在此之前它吃掉了40个人。

　　6．虎鱼
　　虽然还未证实过虎鱼杀人的报道，但是南非东北部河网区的观察人员经常听说，土著在虎鱼出没的水中洗澡时，手指、脚趾或身上的肉被虎鱼咬掉。虎鱼有好几种，其中个头最大的长24英寸。它们的上下颚短而有力，牙齿尖而锐利，血腥味对它们具有极大的吸引力。一群虎鱼能在5分钟内将一条受伤鳄鱼身上的肉全部撕光，不过，平时虎鱼的行动却是懒懒散散的。

　　7．鲨鱼
　　鲨鱼共有200至250种，其中只有18种对人类有危险。最著名的食人鲨鱼有大白鲨、虎鲨、白鳍鲨、恒河鲨和鳍头鲨。最出名的鲨鱼攻击人类事件发生在1916年7月12日，12岁的莱斯特·史蒂维尔在新泽西州的马他瓦河中洗澡，这时一条大白鲨袭击了他，把孩子以及前来搭救的人都咬死了。在此后的16天中，在新泽西沿海区域10英里范围内，又有4人被鲨鱼咬死。两天以后，在马他瓦河口外4英里外，这条大白鲨落入了渔网。

　　8．老虎
　　从1903年至1911年的8年间，尼泊尔的喜马拉雅山中有只老虎一连杀死了438人，它被称作钦帕瓦特的食人者。老虎一般不以人类作为捕食对象，除非它老了，病了，或者爱上了人肉的滋味。
