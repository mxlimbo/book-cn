10个著名人物被捕时他们正在干什么

　　1．哈维·贝雷（1889—？　）
　　贝雷是银行抢劫犯，1930年，他一次抢劫了银行200万美元——这是有史以来最大的抢劫案。当联邦调查局的特工人员前去逮捕他时，他正在堪萨斯城的米逊高尔夫球场打高尔夫球——和他在一起的是他抢劫银行的同伙：汤米·霍尔顿、弗朗西斯·基廷和弗兰克·纳什——这时是1932年。

　　2·梅内姆·贝京（1913—　）
　　1940年9月20日，未来的以色列总理贝京——当时他是犹太复国主义团体的领导人之一——正和他的棋子在立陶宛的家里下象棋，苏联警察破门而入，并且将他逮捕。他站起身说：“这盘棋我认输了。”

　　3．亚历山大·杜布切克（1921—　）
　　1968年8月20日，杜布切克（捷共前总书记）正和其他几位改革派人士在布拉格开会，突然有消息说苏联军队已入侵捷克。第二天清晨刚过4时30分，苏联伞兵进入了捷共中央大厦并逮捕了杜布切克，当时他正在接一个电话。苏联士兵从他手中抢走话筒并扯断了电话线。

　　4．尼娜·豪斯顿（1919—　）
　　1947年，俄亥俄州的托莱多市，尼娜·豪斯顿在自己的汽车驾驶座上睡着了，她的汽车正在修车场等待修理。警察突然拘捕了她，因为警察发现了汽车后座上用报纸裹着的一具尸体，后来证实那是被她谋杀的她丈夫的尸体。

　　5．詹姆斯·尼尔·雷（1928—　）
　　1952年在芝加哥，詹姆斯·厄尔·雷——16年后被宣判为杀害马丁·路德·金的凶手——抢了一个出租汽车司机11．09美元，被警察追赶。他逃进一条死胡同，情急间翻墙进入一家私人住宅的花园，但当他从一扇打开的窗跳进屋子时，正好摔进了盛满热水的浴缸里，警察很容易地找着并拘捕了他。

　　6．麦克西米伦·罗伯斯庇尔（1758—1794）
　　1794年7月27日，正当这位法国雅各宾专政的首脑罗伯斯庇尔在出席会议，并要求将革命的敌人全都逮捕起来时，他自己则因被判叛国罪遭到逮捕，第二天便被送上了断头台。这也就是有名的热月政变。

　　7．朱利叶斯·罗森堡（1918—1953）
　　1950年7月17日，朱利叶斯·罗森堡正和他的妻子埃塞尔·罗森堡一起收听无线电中"孤独的流浪者"一档节目时，联邦调查局的特工人员前来逮捕了朱利叶斯，其罪名是充当苏联克格勃间谍。埃塞尔·罗森堡在此后一个月差几天时也被捕了。

　　8．德·萨得侯爵（1740—1814）
　　1801年，警察因撰写黄色小说和引诱女性卖淫的罪名前去逮捕德·萨得侯爵时，他正在自己的出版商尼古拉斯·马塞的办公室里商谈出版事宜。事实上，正是马塞通知警方他们要抓的人正在自己的办公室里。

　　9．约翰·西德隆（1910—1938）
　　西德隆在1937年绑架并杀害了一个富有的贺卡制造商查尔斯·罗斯。1938年，正当他在加利福尼亚圣安尼塔赛马场押10美元的赌注时，被联邦调查局的特工抓获。

　　10．奥斯卡·王尔德（1854—1900）
　　1895年，这位赫赫有名的大作家正和他的朋友阿尔弗雷德·道格拉斯勋爵在伦敦的考德根饭店喝酒时，被伦敦警探以同性恋的罪名逮捕。
