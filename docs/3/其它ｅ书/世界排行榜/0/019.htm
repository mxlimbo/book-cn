8次美国历史上最肮脏卑鄙的竞选丑闻

　　1．格罗弗·克利夫兰与詹姆斯·布莱恩竞选美国总统（1884）
　　布莱恩的支持者们发现克利夫兰有一个私生子，是几年前和一个名叫玛丽娅·哈尔平的漂亮寡妇生的，后者当时与好几位政治家有着密切关系。于是共和党人抓住这件事朝民主党竞选人克利夫兰头上大泼赃水，他们四处散发传单，传单上画着一个孩子正大声喊道：“我也选克利夫兰当总统！"旁边还有一首儿歌：“妈妈，妈妈，我的爸爸在哪里？快带我到白宫去吧，快，快，快！"但是这一做法却起了一个非常糟糕的反作用。克利夫兰并没有否认这段故事，而是开诚布公地将事情的经过全都讲了出来，并且承认自己是孩子的父亲。他的这番真诚的表白使他反而赢得了公众的信任，将局面又扭转了过来，最后竟然是克利夫兰入主白宫。

　　2．罗伯特·雷诺兹与卡梅伦·莫里森竞选美国参议员（1932）
　　为了把莫里森参议员挤出参议院，雷诺兹跑遍北卡罗来那州，攻击他的对手由于个人嗜好而具有共产主义倾向。在对农村的选民演讲时，雷诺兹总是拿着一瓶鱼子酱说：“大家瞧啊，卡梅伦最喜欢吃的就是鱼子酱，红色俄罗斯的鱼子酱！这样一瓶鱼子酱要两美元啊！对于一个竟然不喜欢吃我们古老的北卡罗来那州母鸡下的鸡蛋的人，你们能喜欢他吗？"当然，农民们自然喜欢吃当地母鸡下的蛋，因此也就喜欢雷诺兹了！

　　3．托马斯·潘德加斯特与联合党在堪萨斯城的一次竞选中（1934）
　　堪萨斯城的民主党人托马斯·潘德加斯特在一个投票的日子里，带着一大群头戴面罩的人坐在黑色的无牌照汽车里，冲进了联合党人的总部。有许多人被橡皮子弹击伤，其中包括《堪萨斯晨星报》的一名记者，另外还有4人死亡。当选举结束时，潘德加斯特支持的候选人获得了极大的胜利。由于潘德加斯特牢牢地控制着堪萨斯城的安全事务，所以他有办法让人们都去投他支持的候选人的票，而当时他支持的人正是哈里·杜鲁门。杜鲁门1934年成为美国参议员。

　　4．厄普顿·辛克莱与弗兰克·梅里姆竞选加利福尼亚州长职位（1934）
　　为了击败有"丑闻揭发派"作家之称的辛克莱，梅里姆的竞选班子拍了一些所谓的"记录片"。其中有一个记者向人们询问究竟愿意选谁当州长，回答的是一个站在破茅屋跟前的老太婆，她说：“我支持梅里姆，虽然我只有这么一间破茅屋，但是我决不能再把它也交出来与人共享了。"但是旁边有一个贼眉鼠眼的人接口说：“我支持辛克莱，既然他的理论在俄罗斯行得通，为什么就不能用在加利福尼亚呢？"结果，辛克莱在竞选中一败涂地。为了这次竞选，共和党人竟花去了1000万美元，这个纪录在加州的竞选中还从未被打破过。

　　5．尤金·塔梅吉与詹姆斯·卡米切尔竞选乔治亚州长职位（1946）
　　塔梅吉具有强烈的种族主义倾向，为了反对具有民主思想的詹姆斯·卡米切尔，他就雇用了一些很像卡米切尔的人，让他们驾驶着汽车，在全州境内兜来兜去，在每辆汽车上还都坐着两个漫画式的黑人。这两个黑人的任务就是把雪茄烟头到处乱扔，假如有人指责他们，他们就自我介绍说他们是支持卡米切尔的人，因为卡米切尔关心黑人。他们甚至把烟头扔在加油站里，使里面的工人都恐慌不已。最后，塔梅吉在竞选中大获全胜。但是非常可惜，他并没能登上州长宝座，因为他在1946年的12月就因肾衰竭而去世了。

　　6．林登·约翰逊与科克·史蒂文森竞选美国参议员（1948）
　　在当时参议员的初选中，约翰逊的对手是得克萨斯州州长史蒂文森。初期的迹象表明，众议员约翰逊不可能在这次选举中取胜。然而在6天之后，事情却发生了戏剧性的变化，在得克萨斯的边境小城艾利斯，有203个人在最后一分钟投出了自己的选票，其中202人是选约翰逊的。当史蒂文森为此提出抗议时，最高法院的法官雨果·布莱克仲裁说选举结果有效。约翰逊以87票的优势取得了初选胜利。为了这次的战绩，专栏作家德鲁·皮尔逊给约翰逊起了"常胜将军林登"这样一个绰号。在1977年7月30日，当时主持艾利斯城选举的法官路易斯·萨拉斯承认说，他和南部得克萨斯的政治大老板乔治·帕尔（他在1975年遇刺身亡）操纵了那次选举。而约翰逊此时已是离任的美国总统了。

　　7．克劳德·佩珀与乔治·斯梅瑟竞选美国参议员（1950）
　　当时"麦卡锡主义"正蔓延全美，佛罗里达出生的克劳德·佩珀素有开明派发言人之称，因此也被列入了强硬派的黑名单。乔治·斯梅瑟以此展开了一场典型的右翼攻势——他称自己的对手是"红色的佩珀"——同时还利用各种宣传手段揭露佩珀的"个人罪恶"。他将佩珀描绘成一个"性格不健全的人"，说他的妹妹是"同性恋者"，而弟弟是"悲剧性人物"。由于那些农村里的人们根本听不懂这些话的具体含义，他们只是吓坏了，因此佩珀遭到了惨败。

　　8．米勒德·泰汀斯与约翰·巴特勒竞选美国参议员（1950）
　　泰汀斯是来自马里兰州的老资格的参议员，他受命调查约瑟夫·麦卡锡参议员关于共产主义正渗透美国国务院的报告。泰汀斯的结论是否定的，他指出麦卡锡的指控只不过是一个骗局，“也许还是这个自由国家历史上最骇人听闻的一场大骗局。"威斯康星州麦卡锡的支持者们为了报复，决心无论如何也要阻止泰汀斯连任参议员，而设法让巴特勒在竞选中取胜。他们挖空心思地将美国共产党领导人厄尔·布劳德的照片和泰汀斯的照片并排贴在一AE？，而后者的眼睛正注视着前者。这样的一组照片总共张贴了30万份，由于当时人们对共产主义普遍存在着恐慌情绪，因此最后的选举结果使这位有着24年参议院生涯的泰汀斯只得告老还乡。
