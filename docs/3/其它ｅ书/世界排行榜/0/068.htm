10位著名的恨猫与惧猫者

　　1．詹姆士·鲍斯韦尔（1740-1795）　苏格兰作家
　　鲍斯韦尔是以撰写撒缪尔·约翰逊的传记而出名的，他十分憎恨猫。但是由于约翰逊博士很喜欢猫，所以鲍斯韦尔在他面前从来不敢流露出半点对猫的憎恶。

　　2．约翰尼斯·勃拉姆斯（1833-1897）　德国作曲家
　　勃拉姆斯似乎对猫有不共戴天之仇，他平日没事就站在窗口，用弓箭袭击邻居家的猫——他对这项活动很入迷。

　　3．乔治·路易斯·德·布丰（1707-1788）　法国博物学家
　　这位著名的博物学家对狗大加赞扬，但认为猫毫无用处。他宣称：“猫具有天生的恶意，和一种怪癖，这种恶意和怪癖随年龄增长而增长。"他还说：“它们只会模仿社会习俗，从来没有自己的行为方式。”

　　4．德怀特·艾森豪威尔（1890-1969）　美国总统
　　1978年，这位总统的孙子大卫·艾森豪威尔在华盛顿国家档案局举办的一次演讲中宣称，他的爷爷"艾克"总统脾气十分可怕。后者住在盖茨堡时，电视机旁总是放着一支猎枪，用来射击飞进屋子的乌鸦。他还下命令，凡在周围地带一见到猫就格杀勿论。

　　5．亨利三世（1551-1589）　法国国王
　　亨利是位狂热的罗马天主教徒，当他迫害法国新教徒时，就像狮子一样勇猛；可是只要猫一出现，这位国王就像一只小鸡一样胆战心惊。如果一只猫走近他，他会吓晕过去。

　　6．拿破仑·波拿巴（1769-1821）　法国皇帝
　　在一次战役中，有位副官走过拿破仑的卧室，听见他在里面大声叫救命。大吃一惊的副官立刻冲进卧室，只见他的统帅半裸着身子，全身大汗淋漓，挥剑朝墙上挂着的布幔刺去。副官定睛一看，原来那里蹲着一只小猫。

　　7．弗利德里克·斯莱·罗伯茨（1832-1914）　英国军事家
　　由于罗伯茨在征服阿富汗的战争以及后来的布尔战争中取得辉煌胜利，他被封为伯爵，并任命为英国军队的总司令。然而，当他和猫遭遇时，他总是屈居下风，每次罗伯茨伯爵看见猫，就会紧张得喘不过气来。

　　8．彼埃尔·德·伦萨（1524-1585）　法国诗人
　　伦萨以写十四行诗出名，他是法国国王查尔斯九世的宫廷诗人。然而对于猫，他却写不出任何美妙的词句。他是这样写的：
　　　　活在世上的任何人，
　　　　不会比我更痛恨猫。
　　　　我憎恨猫眼，猫脸，
　　　　一见它就避之不及。

　　9．珀西·彼希·雪莱（1792-1822）　英国诗人
　　雪莱和本杰明·富兰克林一样，也喜欢做电的实验。他曾不止一次在雷暴来临之际放风筝，但是他总在拴风筝的金属丝下绑一只猫，至于猫的命运如何就不得而知了。

　　10．诺亚·韦伯斯特（1758-1843）　美国词典编纂家
　　韦伯斯特在词典中对猫颇有微词，他认为，家猫是"一种狡猾的动物，被激怒时会变得十分恶毒。”
