11种有影响的电脑及其程序

　　1．分析机（1835-1869）
　　由于缺乏政府资助，英国数学家查尔斯·巴比奇未能建造起自己发明的分析机，它的设计堪称现代电脑的先驱。假如这台分析机真能问世的话，那么它能够使用输入机器的打孔卡器，进行算术运算，还能在储存库中储存信息。具有数学头脑、才华横溢的拉弗雷丝夫人——英国著名诗人拜伦的女儿，曾为那台未出世的分析机编制过几个问题，她可算世界上第一位程序设计师。

　　2．玛克一号（1944）
　　世界上第一台自动数字计算机是哈佛大学的霍华德·H·埃肯于1937年设计的，由国际商用机器公司于1944年制造成功。这台被命名为"玛克一号”的计算机主要用于计算弹道数据，它每秒钟能进行3次加法，速度比人快20倍。从今天的眼光来看，玛克一号的计算速度慢得令人难以置信。这台计算机是个庞然大物，它长51英尺，高8英尺，共有750，000个部件。

　　3．电子数字计算机（埃涅克）（1945-1946）
　　玛克一号问世后不到两年，电子计算机的计算速度就提高了数千倍，1秒钟内能进行5，000次加法，第一台这类“电子数字计算机"被命名为"埃涅克"。它依然是个庞然大物，共有18，000个电子真空管，重30吨，占地15，000平方英尺。埃涅克是宾夕法尼亚州立大学制造的，由物理学家约翰·莫奇利和工程师普利斯珀·埃科特共同设计。他们俩后来共同创办了自己的电脑公司。据说，埃涅克首次起动时，整个费城的电灯都为之黯然。

　　4．万能自动计算机（尤尼瓦克一号）（1951）
　　万能自动计算机能通过数字储存输入的信息，它于1951年呈交美国政府以进行统计工作。它使200，000个小时的工作量减至28，000小时。1963年，这台计算机光荣退休。在此之前它共进行了73，000小时的工作，如今它陈列在史密森研究会的博物馆里。

　　5．计算机（1961）π
　　在1940年出版的《数学与想象》一书中，爱德华·卡斯纳和詹姆士·纽曼声称：“即便就今天而言，要将π的数值计算到小数点后1，000位，也要花10年时间。"21年后，国际商用机器公司数据中心的一台电脑用了8小时43分钟，将π的值计算到小数点后100，265位。它在一天8小时上班时间里，完成了100位数学家计算10年的工作量。

　　6．跳棋电脑（1962）
　　一台IBM704型电脑输入下棋程序后，于1955年轻而易举地击败了它的操纵者撒缪尔斯。1962年，一台IBM7014型电脑击败了西洋跳棋世界冠军罗伯特·尼雷。事后尼雷说：“在残局方面，我自1954年以来从未遇到过对手，我从那时起，直到今天才输第二盘棋。”

　　7．麦克哈克四号（1967）
　　由麻省理工学院学生理查德·格林布拉特设计的电脑程序在一次巡回表演中首次在与真人对弈国际象平时取得胜利。与以后的程序相比，这一外号叫麦克哈克的程序还显得较为粗糙，然而它已经能够打败许多优秀国际象棋棋手。1976年，由大卫·斯莱特和拉里·阿特金设计的电脑程序在加利福尼亚举行的保罗·梅森巡回比赛中战胜所有对手。1978年，一台家庭微型电脑在电脑象棋比赛中赢得冠军。

　　8．登月电脑失灵（1969）
　　当"阿波罗"靠近月球时，登月舱上的电脑突然发出警告：它们已经超负荷。宇航员尼尔·阿姆斯特朗和爱德温·阿尔德林立刻接管了电脑的工作，直接与休斯敦的控制中心联系，进行登月准备。事后查明，是雷达系统的干扰使电脑中的电流紊乱。

　　9．写小说的电脑（1973）
　　按谢尔顿·克莱因设计的程序，一台电脑成功地写出一部2，100词的侦探小说，不过它不及真人写得好。下面我们就来看看其中的一个片断：“詹姆士邀请了布丝丽夫人。詹姆士喜欢布丝丽夫人。布丝丽夫人喜欢詹姆士。布丝丽夫人和詹姆士同住一家旅馆。布丝丽夫人在詹姆士身边。詹姆士充满激情地抚摸布丝丽夫人。詹姆士是布丝丽夫人的情夫。玛丽安跟在他俩身后看到了他俩的恋情。玛丽安十分妒忌。”

　　10．被"杀死"的IBM2740型电脑（1976-1977）
　　据《电脑世界》1977年8月号中德雷克·伦德尔撰写的一篇文章披露，一群自称"尤尼塔共产主义战斗队"的武装恐怖分子曾在意大利策划过10次针对电脑的袭击。第一次袭击发生于1976年5月，15名枪手闯进罗马的一家税务局，用鸡尾酒酒杯砸碎了8台IBM2740型电脑终端机。

　　11．电脑协助偷窃102，000，000美元（1978）
　　在有史以来最大一宗银行偷窃案中，电脑专家马克·利夫金利用一台电脑将洛杉矶太平洋保险银行的102，000，000美元转入一家瑞士银行的存户名下。然而这位专家大获成功后得意忘形，将内情透露给一位商人，后者立即报告了联邦调查局。利夫金于1978年11月5日被捕。在保释期间，他又企图进行第二次数额为50，000，000美元的电脑偷窃勾当，事情败露后他重新被捕。1979年3月，他被判处8年徒刑，那台电脑未受起诉。
