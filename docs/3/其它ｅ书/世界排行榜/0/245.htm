10位看到自己讣告的人

　　1．汉纳・斯内尔
　　汉纳的丈夫离家出走参了军。为了找到丈夫，汉纳男扮女装，也报名参了军。她参加过几次战斗，后来腹部受伤，在施行手术时，医生发现了她的真实性别。一时间她名声大扬，成了一名英勇的女战士。离开部队后，她进入酒吧工作。1779年12月10日她56岁生日时，无意中在翻阅一本名为《绅士》的杂志时，发现里面竟刊登了她的讣告，讣告说汉纳是在瓦立郡的希思兰市去世的。也许汉纳有点相信迷信，因为当她读到有关自己的死亡消息时，她突然感到脑子里闪现出一个阴影。此后她的精神状态越来越差。1789年被送入医院治疗。在医院里她变得精神很不正常，并于1792年去世。

　　2．丹尼尔・布恩
　　这位伟大的美国边民退休后在密苏里州定居。1818年一家美国东部的报纸刊登消息说布恩已在一次狩猎中丧命。人们是在一个鹿群经常出没的地方发现他的尸体的。当时他的姿势是跪着，他的枪靠在一个树桩边，在100码以外的地方有一只死鹿。这则消息很快就传遍了全国。当丹尼尔看到这则关于自己死亡的讣告时，便哈哈大笑起来。其实此时的布恩已经年迈体弱，虽然铺设陷阱之类的事还能干，但追猎动物已力不从心了。两年后，布恩死去，终年86岁。他的真正讣告写得很精彩，它出现在拜伦的《唐璜》中的一段7行的诗句中。

　　3．简・埃伦伯勒女士
　　她是历史上最美丽最富有性感的女子。她的名字叫简・迪格比。17岁时她嫁给英国掌玺大臣埃伦伯勒勋爵。但后来又同奥地利王子私奔。在她丰富多彩的一生中，她曾是小说家巴尔扎克、巴伐利亚国王路德维希和他的儿子以及希腊国王奥托的情妇。她最后一次维持了将近26年的婚姻是同叙利亚沙漠迈兹拉勃部落的酋长的结合。他叫迈特吉尔，他知识渊博，是个贝多因人。埃伦伯勒夫人在一次沙漠旅行回来后发觉人们都在说她已死了，原来这消息出自一家巴黎出版的报纸。该报于1873年3月，用显著的版面刊登了埃伦伯勒夫人死亡的讣告。讣告说：“一个充分利用――或是滥用――自己婚姻的贵族夫人最近逝世。30年前她离开了第一个丈夫，同施瓦曾伯格伯爵私奔。后来她到意大利，在那里又结过六次婚。"这个在欧洲各地的报纸都加以转载的讣告称她的最后一位丈夫为"骆驼骑手"。巴黎这家刊物的以后一期刊登了一篇赞美埃伦伯勒夫人的文章。此文出自她的朋友伊莎贝尔・柏顿的手。伊莎贝尔是阿拉伯人柏登之妻，平时十分势利和傲慢。柏顿夫人还说根据记录她有权发表有关埃伦伯勒夫人的生气和作品。埃伦伯勒夫人得知后惊讶万分，她拼命地写信给报社否认自己已死，否认自己曾同意让柏顿写书。埃伦伯勒于1881年8月死于痢疾，比讣告上所说的日期整整多活了8年。

　　4．詹姆士・布特勒・希科克
　　希科克是个富有传奇色彩的人物，曾任中西部城市的行政长官。他每天都要阅读密苏里州的一份大报"民主主义者"，1873年3月的一天，他突然发现报上登了他去世的消息：“一个得克萨斯人之所以干掉希科克是因他的哥哥死于这个快枪手的枪口。"希科克对此很感不安，他立即提笔给报社写信：“我要纠正你报12日刊登的一则错误消息，以前没有，今后也不会有一个得克萨斯人把希科克干掉。此致！・・JB希科克。"编辑们把他的信在报上登出，并写了一段按语。"今天我们很高兴给你们登出希科克的声明，绝大部分的读者也一定很高兴地得知他仍活着。不过我们愿意告诉希科克，如果哪一天他真的死了，我们还会尽可能地给他发表一份像样的讣告，当然这一天也许还要若干年才到来。"3年后，希科克在一次打牌时，遭人暗杀。

　　5．阿尔弗雷德・诺贝尔
　　这名后来成为百万富翁的火药发明者，朴素心情忧郁，但却很有理想。当他的哥哥路德维希于1888年5月12日死于心脏病时，一家法国的主要报纸误以为是阿尔弗雷德去世，并刊登了一则讣告。在讣告中阿尔弗雷德・诺贝尔被称为"死亡的商人"。看到这则讣告，诺贝尔感到非常震惊。震惊的原因主要并不是人们无中生有地宣布他死去，而是他自己最终竟被人视为死亡的商人。这段对他一生的简单总结的文字完全没能反映出他对人类所抱有的希望，对人民的热爱和他的慷慨。他感到有必要修正人们对他的错误看法。他之所以后来建立诺贝尔奖金，这也是其中的一个原因。诺贝尔奖金是奖励那些在和平、文学和科学方面作出最大成就的人们。

　　6．P・T・巴纳姆
　　美国杰出的剧团管理人巴纳姆80岁时疾病缠身，他知道死期已近，在病床上对一个朋友说：“要是能看到自己死后人们将如何来写我的讣告"他将非常高兴。这个朋友就把他的心愿告诉了纽约市太阳晚报的一个编辑。1891年3月24日，当巴纳姆打开当天的报纸时，看到上面写道：“伟大和独一无二的巴纳姆，他想看到自己的讣告"。以下便是讣告的前言部分：“巴纳姆一生中几乎拥有一切，包括骏马和珍尼・琳达。因此没有理由不让他得到他最后一件想要的东西。因此我们简单扼要地总结了他的一生，就作为一旦伟大和独一无二的巴纳姆的生命被夺走时，本报为其所作的讣告"。巴纳姆的讣告一共占了4个纵栏，还配有木刻的他本人当时的画像，41岁时的画像，他母亲的画像，第一亡妻和瑞典歌唱家珍尼・琳达的画像。两星期后，巴纳姆去世。

　　7．利奥波德・撒切-马索奇
　　这个警察局长的儿子，在加里西亚出生，在奥地利度过了青少年时期。他有受虐癖。他的第一个情人安娜・库图威兹经常用木棍打他，而且还要他另找男人同她寻欢作乐。他的第二个情人范妮・普斯顿同他签订一份协议。规定范妮每天打他时必须身穿裘皮外衣。范妮出色地履行了这个协议。把他当仆人一样对待。当他同一个名叫温达的女子相遇并结婚时，他已成为一名著名的作家。在婚后的15年时间里，温达每天用一根带有钉子的皮鞭抽打他，对待他就像对待奴隶一般。最终温达还是离开了他。他又娶了一个出身低微的德国女子，名叫赫尔达・梅斯特。此时的撒切-马索奇已经开始患有精神分裂症。他甚至有一次要勒死自己的妻子。1895年梅斯特悄悄地把他送入疯人院，并向外界宣布他已死去。报纸登出讣告，热情地赞扬了这个富有才华的作家。撒切-马索奇偶尔头脑清醒时，也曾从报上得知有关自己死去的消息。他真正死去的时间是1905年，也就是在讣告发表的10年后。由于撒切-马索奇生前具有这种病态的表现，精神病学家就用他的名字Masoch创造出一个新词Masochism，意思是：一种以受异性虐待为快的病态色情狂。

　　8．马克・吐温
　　1897年，由于一个亲人去世，马克・吐温心情十分悲痛，他停止在公共场合露面，开始过隐居的生活。可就在这时，他突然发觉外界盛传他已死去的消息。原来是一家喜欢制造耸人听闻消息的美国报纸在头版头条刊登了这一消息。说他是在伦敦去世的，死时身无分文。接着一个全国性的新闻组织便派一名记者去马克・吐温家了解真情。没想到马克・吐温竟然还活着。面对瞠目结舌的记者马克・吐温发布了正式声明，他说：“詹姆士・罗斯・克莱门斯是我的堂兄，二三个星期前他在伦敦病倒，现在已经恢复，外界传说我在伦敦去世之事可能是从我堂兄得病之事中编造出来的，这样的报道未免同事实相差太远。"马克・吐温于1910年去世。

　　9．伯特兰・罗素
　　本世纪30年代罗素曾访问过北京，在北京他突然生病。由于一些日本记者一直想见他，但始终没被同意，他们就认为罗素死了，并在报上登了这则消息。很快消息传遍全世界。罗素写道：“能让我看到自己的讣告，实在是太有趣了。这是我一直希望实现的、但又知道不能实现的愿望。"一家教会的报纸只用一个句子作为他去世的讣告：“教会对罗特兰・罗素之死感到松了一口气，这一举动也许是可以得到上帝的宽恕的。"罗素看后决定自己动手为自己写一篇讣告，他写道："他的一生尽管令人难以捉摸，但一贯具有一种与时代相背的东西，这东西能使人想起十九世纪初叶的那些贵族反叛者。……他是一个死亡的时代的最后一个幸存者。"他让泰晤士报于1962年登出，因为他预计他将在那一年离开人世。但泰晤士报直到1970年罗素真正去世之日才刊登了这其他自己写就的讣告。

　　10．爱德华・里肯巴克
　　这位赛车驾驶员后来从军驾驶战斗机，在第一次世界大战中他战功卓著，曾先后击落26架敌机，成为美国空军王牌飞行员。在和平时其他担任汽车公司和航空公司的管理人员，第二次世界大战开始后，他主动要求为美国执行战争任务。1942年10月在一次侦察飞行中，他的B-17飞机坠入太平洋。军队立即进行非常严密的搜寻，但没有任何结果。美国各地的报纸都报道了他阵亡的消息。但在一个月后的11月13日，搜寻人员在太平洋找到了他和另外7名战士，他们在一只木排上漂泊了23天。里肯巴克归来时，迎接他的是一大堆讣告。纽约每日新闻上刊登的是一幅漫画，上面画了一只黑色的花圈在水上漂浮，下面写道：“再见了，爱迪。"纽约人杂志上的标题是：“赛车路的尽头。"看到这些报道，里肯巴克乐得咧开了嘴。他用钢笔在上面打了个×，说：“不，见鬼去吧！”

　　注：以上10个只是编辑们最喜欢的例子，事实上还有许多名人活着时看到自己的讣告的事例。例如美国总统托马斯・杰弗逊，魔术师哈里・霍迪尼，舞蹈家约瑟芬・贝克，歌唱家珍妮特・麦克唐纳，小说家厄内斯特・海明威，驻外记者埃德加・斯诺。此外还有许多人虽然没有看到他们的讣告，但却听到了不少有关他们自己死去的谣传。类似例子在近代的有歌唱家保罗・麦卡特尼。在几张甲壳虫唱其中，有曾暗示他已死去的歌词。还有女演员贝特・戴维斯，她的律师曾告诉她，有关她死去的消息已传遍纽约，可戴维斯听后只是说：“我才不管呢。"还有印度年迈的不同政见者纳拉扬，1979年5月他仍健在时听到德赛首相发表的赞美他的悼词。
