林·拉德纳眼中的5位最佳电影编剧

　　拉德纳是40年代美国主要的电影剧本创作者之一。1942年他创作的《当今妇女》荣获奥斯卡最佳剧本奖。作为因政治倾向而受到指控的"好莱坞10人"中的一员，他拒绝到议会非美活动委员会处作证。他被监禁达9个月之久，并被电影行业列入黑名单。虽然他并没停止创作，但直到1965年，他的名字才出现在他的剧本《辛辛那提少年》上。1970年他又一次荣获奥斯卡最佳剧本奖，获奖剧本是《M.A.SH》。

　　5．查理·卓别林
