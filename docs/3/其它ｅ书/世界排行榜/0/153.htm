20支拥有最多不友好球迷的职业球队

　　宾夕法尼亚州的股票经纪人和前大学教授鲍普·麦克马洪曾对一些职业球员进行调查，了解他们心目中哪些职业队的球迷最不友好。在172个被调查者中间包括棒球明星期特·罗斯和马克·菲德里奇；全国篮球协会的朱丽斯·欧文和皮特·马拉维奇；全国橄榄球联盟的弗兰·塔肯顿和O.J·辛普森以及全国冰球联盟的托尼·埃斯波西托和吉尔·佩劳尔特。
