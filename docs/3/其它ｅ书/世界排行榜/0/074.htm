本尼·古德曼眼中二十世纪最受欢迎的7首歌曲

　　这位世界著名的"摇滚乐之王"于30年代和40年代大力推广了爵士乐并开创了大乐队的时代。他作为乐队指挥和单簧管演奏家曾与爵士大师吉思·克鲁巴，拉恩内尔·汉普顿和哈利·詹姆士等人同台演出。1962年他带领乐队到俄罗斯进行了该国历史上第一次的爵士乐巡回演出。俄国首脑尼基塔·赫鲁晓夫出席了首场演出。古德曼现在居住在纽约市，偶尔会在夜总会露面。
　　1．《星尘》（1929），由卡迈克尔和派利希演唱。
　　2．《唱，唱，唱》（1936），由刘易斯·普利马演唱。
　　3．《身和心》（1930），由乔尼·格林，埃德·海曼·罗伯特·索尔和弗兰克·伊顿演唱。
　　4．《请来小丑》（1973），由斯蒂芬·桑德海姆演唱。
　　5．《开始跳比律舞》（1935），由科尔·波特演唱。
　　6．《你是我生命的阳光》（1973），由斯蒂维·旺德演唱。
　　7．《亚历山大的格拉泰姆乐乐队》（1911），由欧文·柏林演唱。
