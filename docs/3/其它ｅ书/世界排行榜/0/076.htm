爱伦·勒纳眼中当代10名最佳歌曲作者

　　作为勒纳连锁商店创始人的儿子，爱伦对家族的事业毫无兴趣，他曾编制过广告宣传手册和广播节目，以后，他开始从事歌词的创作。他的第一个作品音乐剧"波列盖杜恩"大获成功，为此他获得了纽约戏剧论坛奖，接着他所创作的《窈窕淑女》、《卡米芬特》和《琪琪》也接连获得成功，1951年他所创作的《一个美国人在巴黎》荣获奥斯卡最佳剧本奖。以下是他选出的最佳歌词作家，顺序按字母排列。
　　1．伯特·巴卡拉克
　　2．利奥纳多·伯恩斯坦
　　3、4．约翰·坎德和弗雷德·埃伯
　　5．伯顿·莱恩
　　6．理查德·罗杰斯
　　7．保罗·西蒙
　　8．斯蒂芬·桑德海姆
　　9．米尔·斯泰因
　　10．吉米·威伯
