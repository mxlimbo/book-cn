马迪·沃特斯眼中历史上10首最佳布鲁斯歌曲

　　马迪·沃特斯原名为麦利利·摩根菲尔德，从小生长在密西西比河三角洲地区。他很早就开始作曲，唱歌和演奏布鲁斯歌曲。移居芝加哥后，他于1941年第一次灌制唱片，在50年代和60年代，他创作的歌曲中有十多首被列入排行榜的第10名，其中有《我是你的男人》、《请向我求爱》等。从此他的歌曲一直成为"滚石"和其他演唱组的演唱歌曲。目前他居住在伊利诺斯州的拉格兰奇市。
　　1．罗伯特·约翰逊创作的《十字路口》。
　　2．罗伯特·约翰逊创作的《行走的布鲁斯》。
　　3．比格·马西奥创作的《烦恼的生活》。
　　4．利罗伊·卡尔创作的《多么长，多么久》。
　　5．麦金利·摩根菲尔德（马迪·沃特斯）创作的《真想回家》。
　　6．麦金利·摩根菲尔德（马迪·沃特斯）创作的《长途电话》。
　　7．坦巴·雷德创作的《当你有麻烦的时候》。
　　8．孟菲斯·米尼创作的《大黄蜂》。
　　9．桑·豪斯创作的《布道布鲁斯》。
　　10．利罗伊·卡尔创作的《监狱布鲁斯》。
