10种最有可能成功的小型企业

　　为了弄清在今天的经济状况下哪种企业最可能成功或失败，"货币"杂志调查了26个银行家、商校教授、小型企业管理局的官员以及退休经理人员服务团的成员，他们被要求从81类小型企业中列出最可能成功或最可能失败的企业。（这里所列出的数字都是扣除工资后的税前利润）下面是爱德华特·亨利调查后所作的研究报告中的统计。

　　1．建筑材料商店
　　由于住房建设的发展和自己动手修理房屋的人日益增多，建材的销售额自1973年以来增长很快，而且估计今后也将继续增长。

　　2．汽车轮胎和配件商店
　　今天，越来越多的人自己动手修理汽车，这种局面造成了这类商品销售额的增加。

　　3．酒类商店
　　一个勤快的零售商预计能从每1元钱的营业额中获得3分多的利润。

　　4．体育和娱乐俱乐部
　　由于对运动感兴趣的人日益增多，该行业的利益也日益增大。一个俱乐部初建时需要一笔较大的投资，但开办后，从每100万的会员费中可获利26，000元。

　　5．殡仪馆和火葬场
　　小型的殡仪馆和火葬场每做10万元的生意，可从中得净利9，200元。虽然目前死亡率在不断下降，但死亡总人数仍保持在每年约190万人左右，而且随着人口的老龄化，死亡人数还会增加。

　　6．种子和园林物资供应商店
　　利率约和体育俱乐部相同。

　　7．体育用品制造厂
　　户外运动的不断普及使得对自行车和自行车零件的需求增长8％，对球类运动器材和滑冰及滑雪的需求增长19％。

　　8．工程·实验室和科研设备
　　从1980年至1983年，这些行业估计每年将平均增长5％，这主要是因为飞机生产的扩大以及航天飞机的使用和发展。

　　9．五金商店
　　该行业得益于日益发展的"自己动手"运动，高额的获利率意味着五金行业为了得到一定的利润，并不一定要实现很大的营业额。估计每65万元的销售额中可得到利润数约为3万元。

　　10．办公室用品和设备供应商
　　该行业明显不受经济衰退的影响，因为任何一个公司都需要办公用品，因而利润十分可观，每100万营业额可获利4万元。
