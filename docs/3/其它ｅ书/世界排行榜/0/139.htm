亨利・阿姆斯特朗眼中10位拳击史上最佳非重量级拳击手

　　亨利・阿姆斯特朗是世界上唯一同时拥有3项世界冠军称号的拳击手。1937年他获得次轻量级冠军。1938年又获次中量级和轻量级的桂冠。他被公众视为当代最伟大的拳击手
