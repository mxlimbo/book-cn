6位著名人物但你不会知道他们是黑人或有黑人血统

　　1．汉尼拔（前247—前183）
　　他是迦太基的统帅，人们总是形容他有着黝黑的皮肤和蜷曲的头发胡须。当时，迦太基大多是闪族人和非洲黑人的混血种人。

　　2．约翰·詹姆斯·奥杜邦（1785—1851）
　　奥杜邦这位著名的艺术家出生在海地，他是一位法国船长和当地一位黑人女仆的儿子。他的母亲在他出生后不久就死了，奥杜邦被带到法国，由他的继母抚养成人。

　　3．普希金（1799—1837）
　　人们都知道他是最伟大的俄罗斯诗人，但他确实有八分之一的黑人血统。他的曾祖父是一位黑人俄国将军，名叫阿伯拉姆·汉尼拔，他在8岁时被人从非洲抓去，作为送给彼得大帝的礼物。

　　4．大仲马（1802—1870）
　　这位《三个火枪手》和《基督山恩仇记》的作者是帕里特利侯爵和一个圣多明各女黑奴的私生子，他母亲名叫玛丽·仲马。

　　5．小仲马（1824—1895）
　　他是大仲马的儿子，因此有四分之一的黑人血统。1852年他出版了著名的《茶花女》。

　　6．彼得·乌斯蒂诺夫（1921—　）
　　他是一个极有才华的制仆人。他的祖母有一半埃塞俄比亚黑人血统。乌斯蒂诺夫在《我的亲人》一书中写道：“我的祖母名叫玛吉莱娜，她是在埃塞俄比亚正与英国军队作战时，出生在一个帐篷中的。……我祖母最小的妹妹直到最近还在海尔·塞拉西的宫廷里当宫女。”
