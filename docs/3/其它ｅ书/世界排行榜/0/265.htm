18条奇特的法则

　　许多读者要求再提供一些影响我们日常生活的一些非法律条文的法则。我们从阿瑟・布劳里克所著的《莫菲法则第二卷――再读为什么事情会不对头》一书中选出以下18条。

　　1．奥立利的厨房法则
　　厨房是不可能打扫干净的。

　　2．利伯曼法则
　　人人都说谎，但这无妨，因为没有人听。

　　3．但尼斯顿的法则
　　道德是对它自身的惩罚。

　　4．戈尔德法则
　　合脚的鞋子是难看的。

　　5．现代科学简易指南
　　如果对象是绿色的或蠕动的，那就是生物学。
　　如果对象是散发臭气的，那就是化学。
　　如果对象行不通的，那就是物理学。

　　6．康威法则
　　在任何一个组织内总有一个人清楚地知道该组织是如何运行的。
　　这个人应被解雇。

　　7．格林的辩论法则
　　如果你不知自己在说什么，那么任何事情都有可能是对的。

　　8．斯图尔特的反作用法则
　　要得到宽恕比得到许可更容易。

　　9．历史第一规律
　　历史本身从不重复，只是历史学家们互相重复。

　　10．劳斯特法则
　　紧闭的嘴吐不出诗句。

　　11．奥立佛的方位法则
　　无论你到哪里，你就在那里。

　　12．林奇的法则
　　情况困难时，人人都离去。

　　13．格利姆的成功公式
　　成功的秘诀是真诚，一旦你能伪装成真诚，那你就成功了。

　　14．梅森的法则
　　一旦你能用灵魂换取某件东西，你的灵魂就是多余的了。

　　15．香肠的原则
　　爱吃香肠的人绝不应观看香肠的制作过程。

　　16．哈里森的假设
　　对每一个行为总会有平等和相反的评论。

　　17．汉龙的法则
　　决不要把任何可以解释成愚蠢的行为说成是恶意的举动。

　　18．缪尔法则
　　当我们挑选出一件独立存在的事时，我们总会发现它实际同宇宙的万物都有联系。
