《平民年鉴》刊出的13件趣事

　　1．跳蚤杀手
　　瑞典女王克里斯蒂娜（1626-1689）非常讨厌跳蚤。她决定向它们宣战，准备亲手将住宅中的每一只跳蚤消灭掉。为此她让人特制了一只小型轰击器――长度仅为4英寸，并装入微型炮弹。每当她看见跳蚤，就将炮口对准它，开炮射击。该轰击器目前被存放在斯德哥尔摩军火工厂展览厅，供大家参观。

　　2．凡尔赛颅骨
　　1919年6月由战败国德国和获胜的协约国共同签署了凡尔赛和约，从而宣告第一次世界大战正式结束。根据条约，德国必须放弃阿尔萨斯――洛林和所有的殖民地，允许协约国在莱因兰驻军，向战胜国支付战争赔款，还有一项是要求德国交出一个人的颅骨的条款。在这个重大的国际条约中的第八部分第二节的第246条竟出现如此奇怪的规定：“在本条约生效后的6个月内……德国必须向英国女王陛下的政府交出德国在其东非保护国所拿走的姆克瓦瓦苏丹的颅骨。"姆克瓦瓦苏丹曾是德国控制的坦干伊喀图瓦赫部落的首领，他领导人民起义抗击德国统治者，结果起义失败，他被德军抓住并杀害。他的颅骨被当作礼品送到德国。瓦赫部落的人民一直想要回这只赋予魔力的头颅，因此他们一直要求英国政府在凡尔赛条约中提及此事。几年后德国方面坚持说颅骨已经归还坦干伊喀，如果事实确实如此的话，只能说明该颅骨已经失踪。

　　3．超级性别隔离
　　在维多利亚时代的全盛时期，英国出版了一本介绍如何处世行事的书，名叫《高夫夫人礼仪大全》。书中说在任何一个书架中由男性作家写的书绝不能同女性作家写的书放在一起，除非这两个作家正好是一对夫妻。

　　4．超级父亲
　　它应该成为现代生儿育女父亲之冠。这头名叫莱格普果的荷兰良种牛，通过人工授精，在3年零4个月的时间里共生了15，000头小牛。它于1848年因一次意外事故死去。

　　5．美国摩天大楼
　　1941年6月，美国总统弗兰克林・罗斯福指示商业部长杰西・琼斯买下世界上第一高楼――纽约的帝国大厦。罗斯福打算将此改为美国政府的办公大楼。这座高102层、耗用钢材60，000吨、能容纳15，000人的建筑的产权主要是属于大都会人生保险公司和一个由民主党全国委员会主席约翰・拉斯科布控制的公司。他们开价4500万美元。琼斯感到价钱太贵而没有买下。事后琼斯承认，为此罗斯福总统一直对他耿耿于怀。

　　6．女性斗牛士
　　据西班牙斗牛史权威巴纳比・康拉德说，在历史上的斗牛传奇中有一则关于一名女性斗牛士的故事最为有趣。她的名字叫玛丽亚・萨洛姆，但人们都称她利维特。1880年她出生于西班牙的吉恩，本世纪初她首次进入斗牛场。由于她表现不凡，深受观众的欢迎。但在1908年，西班牙政府宣布女子斗牛既不道德也不合法，因此予以取缔。考虑到斗牛的资格将被取消，里维特突然向公众宣布自己并非女子，而是一名男士，名叫奥古斯丁・罗德里戈茨，他还当场脱下假发和假乳房，说自己不属于政府禁止之列，仍旧能同其他男子一样继续斗牛。但是观众对他的起局大为恼火，许多人甚至想私下揍他。罗德里戈茨不得不离开马德里，最后他穷困潦倒，死于马约卡市。

　　7．说密码者
　　1918年正是第一次世界大战西部战场的关键时刻。美国陆军通讯兵部队感到情况十分不妙，因为美军务战区之间，许多通过电话传递的命令的密码正被德军截获并破译。在他们研究如何解决这个问题时，通讯兵上尉E.W.霍纳提出就他所知美国印第安查克图族人的语言是世界上会说的人最少的语言。如果能在美国远征军中找到查克图族人的话，那么通讯部队就能拥有一种无法破译的密码。最后霍纳在部队中找到8个查克图人，他们被调入第141步兵团4连，然后再被派到西线各前沿阵地。他们互相之间全部用查克图语传递秘密命令，德军的密码破译人员顿时感到一筹莫展。接着这些天的战争开始朝有利于盟军的方向发展。这种用印第安语传递消息的做法在第二次世界大战中产生了更为明显的效果。部署在太平洋战区的美国海军陆战队招募了一些纳伐鹤人。该印第安部落总共只有5，000人，已知全世界总共只有28个非纳伐鹤人（人类学家和传教士）懂这种语言，他们中没有一个人是德国人或日本人。大战初期约有30个纳伐鹤人从事无线电情报传递和接收，到大战结束时已上升至420人。

　　8．猪耳朵做的钱包
　　当1978年88岁的华莱士・莫雷在缅因州戈勒姆去世时，美国失去了近代史上的一名不为人所知的天才。此人曾用一只猪的耳朵制成一只"丝织"钱包。这位麻省理工学院的毕业生说他这样做是为了显示美国的科学水平。这只钱包至今仍被存放在史密斯博物馆作永久性展出。

　　9．杰出的军犬
　　当西班牙探险者和征服者胡安・莱昂（1460？-1521）在波多利各镇压了当地人民的反抗后，取得了对该地区的控制，并出任总督时，在他军队的功臣中有一个名叫贝兹利洛的军犬在战争中表现尤为突出。据西班牙编年历的作者海利拉说，这条猎犬像人一样能辨别当地居民中哪些是和平的，哪些是参加反抗的。正因如此，当地的印第安人宁可同不带这条军犬的100个西班牙人打仗，也不愿同带着这条军犬的10个西班牙人交锋。它的主人是一个步兵，同主人一样，贝兹利洛也获得不少战利品，包括黄金、奴隶和其他很多东西。后来莱昂出发到比米尼岛上去寻找青春泉（结果于1513年发现了佛罗里达）时，决定把贝兹利洛带上，并命令部队发给它工资和每天的食物津贴。

　　10．空中飞牛
　　爱荷华州一头名叫福恩的根西乳牛至今仍保持母牛飞天的最佳纪录。1962年5月9日，福恩做了第一次空中飞行，当一阵突然刮来的飓风把它吹到空中，它轻松地飞了一阵，然后安全地在半哩路外邻居家的田野里降落。它遇到邻居家的公牛，同它性交后，十分悠闲地返回主人家。后来还生了头小牛。5年后，1967年1月的一天，它正在外边吃草时，突然又刮起一阵飓风，它再次升天，正好一辆客车开过，里边的乘客目睹这一情景，都惊讶得瞠目结舌。福恩从汽车的另一侧上空飞过，又一次安全地四脚落地。从此以后，它就再也没有飞天的机会了，因为每当气象站有风暴的预报，主人就把它赶入牲口棚内。经过一段乏味的退休生活后，它于1978年7月25日在爱荷华州斯考特县去世，享年25岁。

　　11．美洲的一夫多妻制
　　除了美国的摩门教徒曾实行过一夫多妻制外，十九世纪在整个北美、中美和南美洲只有过一个国家正式实行过这一制度。这就是巴拉圭。1865年至1870年这5年对巴拉圭来说是腥风血雨和充满灾难的5年，它同巴西、乌拉圭和阿根廷三国的联盟交战，结果全国100万人口中有一半死于战争。青年男性中约有三分之二丧命。战争结束后，巴拉圭人口性别的比例失调，成年女子有19万人，而成年男子则只有29，000人，再加上结婚费用昂贵，结果全国有一半儿童是非婚生儿童。为了弥补人口的不足，政府制定法律，实行一夫多妻制，允许一个男子可以娶多个女子为妻。不久由于这个制度的实行以及其他一些原因，巴拉圭的人口得到了迅速的增长，今天她已拥有300万人民。

　　12．可爱的猪
　　许多动物学家认为猪是人类饲养的动物中最受虐待的和丑化的动物。人们认为猪既胖又懒，而且最脏。其实这并非事实。猪之所以像现在这样是人为的，如果让猪生活在大自然的环境里，它会吃得更少，性格也更活跃，而且是属于最爱干净的动物之一。一个专家说，无论是野猪或家猪，都很"瘦、干净和活泼"。它们是很挑剔的动物，他们讨厌肮脏的环境和泔脚饲料，是人故意让它们在泥浆中生活，让它们拚命吃，拚命睡，变得出奇的懒惰――为的是把它们养胖好卖大价钱。

　　13．好得无法相信
　　1821年土耳其奥斯曼帝国的军队占领了希腊。有一次希腊人奋起反抗，土耳其军队被迫躲进雅典郊外的宏伟和神圣的卫城内。从卫城的最高点，土耳其士兵向下面的希腊人开火，突然间土耳其人弹药用完，如果得不到补充，希腊人很快就能攻上来。他们想出一个办法，就是将神殿的柱子敲下来，设法从地基中挖出铅心来，据说当时为了使地基牢固，接缝处使用了铅，一旦得到铅，土耳其人就可将它做成子弹，当土军的这一打算传到城下的希腊人耳朵里时，他们都感到恐惧万分，因为对他们来说，卫城是一个神圣纪念碑，土耳其人为了得到弹药补充竟然打算破坏它，这简直不可思议。因此为了保住卫城，阻止土军的亵渎行为，希腊便将大部分自己的弹药送给土军，战斗又重新打响。该故事是一名读者提供的，他叫杰里・桑基。他记得是以前自己的一名老师讲给自己听的。因而我们无法确认它的真实性。但如果是真的话，那无疑是历史上的一大奇闻。
