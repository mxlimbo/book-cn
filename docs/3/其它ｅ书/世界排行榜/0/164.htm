10名反对子女婚姻的父母

　　1．埃德华·巴雷特，伊丽莎白·巴雷特·勃朗宁的父亲。
　　由于患有性神经方面的疾病，巴雷特粗暴地反对子女结婚成家。可当时备受疾病折磨的伊丽莎白已同诗人罗伯特·勃朗宁产生了恋情。1846年两人便离家私奔。他们来到意大利定居，并在那里一直生活到伊丽莎白去世。

　　2．卡梅伦-福尔科尼特夫人，露西尔·卡梅伦·约翰逊的母亲
　　正当黑人职业拳击手约翰逊处于他运动生涯的顶峰时，他结识了19岁的露西尔小姐，并开始向她求婚。露西尔的母亲对此十分恼火。她向法院起诉，指控约翰逊犯有拐骗罪。尽管1912年约翰逊同露西尔正式结婚。但第二年约翰逊还是被带上法庭，并被判决犯有多项罪名。为了避免坐牢，约翰逊同妻子逃离美国，前往加拿大，随后又转到欧洲定居。1924年他们两人宣布离婚。

　　3．琼·芬利，波利·芬利·克罗克特的母亲
　　1806年波利的母亲写道，边民大卫·克罗克特"朝我看时，他的眼光凶狠得就像一把砍肉的屠刀。"她想让女儿嫁给另一个追求她的男子。虽然她曾开玩笑时说过大卫是她未来的女婿，可当大卫真的提出要娶波利为平时，芬利夫人差点把大卫赶出门外。不过，最后芬利夫人还是同意了他们的婚事，并主持了婚礼。

　　4-5．杰茜·鲁特·格兰特和海纳·辛普森·格兰特，尤利西斯·格兰特的父母
　　格兰特出身于一个生活泼困的农夫家庭。而朱丽叶·登特的父母则很富裕，他们在密苏里州拥有一座收入丰厚的大庄园。格兰特父母对儿子同一个奴隶主女儿结婚深感不安，因此1848年他们拒绝出席儿子的婚礼。不过尽管如此，当儿媳婚后来到他们家时，格兰特夫妇仍表示欢迎，并以礼相待。在婚后的最初几年内，格兰特和朱丽叶在经济上还不得不依靠朱丽叶父母的帮助。

　　6．南希·李，玛格丽特·李·休斯顿的母亲
　　南希对自己20岁的女儿同46岁的前得克萨斯州州长结婚大为震惊。休斯顿曾结过婚，而且嗜酒；玛格丽特则是一名狂热的教徒，她想帮他戒酒以拯救他的灵魂。1840年，在南希的勉强同意下，两人举行了婚礼。婚后他们共生了8个孩子。

　　7．利奥波特·莫扎特，沃夫根·莫扎特的父亲
　　作为自己天才的儿子的艺术经纪人，老莫扎特坚决反对任何会转移莫扎特对音乐事业追求的事情。因此1782年当莫扎特同康斯坦丝订婚时，老莫扎特采取了完全敌对的态度。可莫扎特本人决意要完成这门婚事，另一方面，他又不想同父亲过分对立。后来在他的恳求下，父亲终于勉强表示同意。对此，莫扎特感到欣喜不已。

　　8．尤金·奥尼尔，乌娜·奥尼尔·卓别林的父亲
　　这名剧作家的朋友们猜测奥尼尔之所以反对卓别林同自己女儿结婚是因为卓别林同奥尼尔妻子的前夫关系非常密切。也许更重要的原因是卓别林已3次离婚，而且还受到女演员琼·巴利的指控。巴利声称卓别林是其私生子的父亲。1943年，女儿乌娜同卓别林结婚。从此，奥尼尔便再也不理乌娜，而且也不再提起她的名字。然而卓别林同乌娜尽管年龄相差很大（卓别林54岁，乌娜18岁），两人的婚姻却一直维持到卓别林去世，时间长达34年。

　　9．蒂莫西·雪莱爵士，珀西·雪莱的父亲
　　雪莱爵士曾为儿子精心安排了一条从政的道路和其他的一切。可当反叛精神很强的雪莱擅自要同一名"门第不等"的女子结婚时，雪莱爵士大为恼火。父子间的关系急剧恶化。1811年19岁的雪莱同商人的女儿哈丽雅特·威斯特布鲁克私奔。忿怒的父亲立即同儿子断绝一切关系，并切断他的经济来源。后来经济来源又得以恢复，但雪莱同他父亲再也没能彻底和好。

　　10．费利德里希·威克，克拉拉·威克·舒曼的母亲"想同一个流浪者结婚，这简直是太荒谬了！"威克咆哮道。她非常担心这样的婚姻会毁了女儿成为一名钢琴演奏家的前途。尽管她极力反对并粗暴地将两人分开达6年之久。但克拉拉同舒曼之间的爱情始终如一。1840年，两人终于正式订婚。3年后威克终于回心转意，并同这对年轻的夫妇重归于好。
