历史上10位德怀特·麦克唐纳最愿意成为的人

　　麦克唐纳以其"精深的学问、高雅的气味和尖锐的批评"而著称。在过去的50年里，他一直仔细地观察通俗文化和政治的发展。开始曾为《幸福》杂志工作，后来又替《党派评论》、《纽约人》和《乡绅》等杂志撰写稿件。到40年代，他出版了自己的刊物"政治"。他写的书包括《同美国格格不入》和《银幕上的德怀特·麦克唐纳》。
　　1．托马斯·杰弗逊
　　2．马克斯·比尔博姆
　　3．荷马（包括《伊利亚特》的作者以及《奥德赛》的作者——很可能是一个名为娜西卡的女性）
　　4．威廉·莎士比亚
　　5．霍纳斯·瓦格纳
　　6．罗莎·卢森堡
　　7．福尔摩斯
　　8．耶稣
　　9．威灵顿公爵
　　10．拜伦勋爵
　　注：马克思·比尔博姆（1872-1956）是英国批评家、散文作家和漫画家。霍纳斯·瓦格纳（1874-1955）是历史上最佳全能型棒球运动员之一。罗莎·卢森堡（1870-1919）是德国革命家，曾帮助建立波兰社会主义政党和德国共产党。
