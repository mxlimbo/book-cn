14种对人体健康有害的食物添加剂

　　1．人造色素
　　蓝色1号-用于饮料、糖果、烘烤食品，测试结果不佳。蓝色2号-用于宠物食品、糖果、饮料，未经充分测试。柑桔红2号-某种佛罗里达桔片，可能致癌。绿色3号-用于糖果、饮料，测试结果不佳。红色3号-用于水果鸡尾酒、糖果、烘烤食品，可能致癌。红色40号-用于软饮料、宠物食品、香肠，对老鼠进行试验证明致癌。黄色5号-用于果子冻、宠物食品、糖果，可能致癌或过敏性反应。

　　2．经过溴处理的植物油
　　乳化剂。用于软饮料，可能将有毒物质滞留于人身。

　　3．丁基羟基苯甲醚
　　抗氧化剂。用于干玉米片食品、制糕饼的油、口香糖、油炸土豆片，可能造成过敏性反应。

　　4．丁基羟基甲苯
　　抗氧化剂。用于干玉米片、口香糖、油炸土豆片，可能造成过敏性反应和肝肿大。

　　5．咖啡因
　　兴奋剂。加入软饮料，存在于咖啡、可可和茶叶中，可能造成流产、先天性生理缺陷、失眠和神经紧张。

　　6．谷氨酸单钠
　　味精。用于汤、各种酱汁、炖品。会导致"中国餐馆综合征"（在前臂和颈后部产生烧灼感；胸部发闷；头痛），实验证明，它会损害幼鼠的脑细胞。

　　7．磷酸，磷酸盐
　　多种用途。加入烘烤食品、奶酪、腌熏肉、干玉米片、可乐饮料、加工成粉末的食品。大量使用会产生一种因饮食营养失衡而产生的骨头疾病。

　　8．丙酸
　　抗氧剂。用于植物油、肉制品和口香糖。未经充分科学试验。此物并非必须使用。

　　9．奎宁
　　调味。用于健身饮料、奎宁水、苦柠檬。未经充分的测试，可能与先天性生理缺陷有关。

　　10．亚硫酸钠
　　防腐，漂白。用于水果切片、葡萄酒、葡萄汁、脱水马铃薯，对维生素B有破坏作用。

　　11．氯化钠（盐）
　　调味，防腐。大量用于加工食品。适量使用是必要的，但过量使用会导致高血压及促使心脏及肾脏的病变。

　　12．硝酸钠、亚硝酸钠
　　媒染剂。用于腌熏肉制品，产生致癌物亚硝胺。

　　13．蔗糖、葡萄糖、玉米糖浆
　　增甜剂。用于软饮料、烘烤食品和其他许多食品。无营养价值，增加不必要的热量，促使龋齿的产生。

　　14．二氧化硫
　　防腐、漂白。用于水果切片、葡萄酒、葡萄汁、脱水马铃薯。有高度毒性，破坏维生素A。
　　被联邦食品和管理局认定是安全的。
