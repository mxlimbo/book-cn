美国军人最喜欢的25种食品

　　1973年和1974年有关部门对美国陆、海、空三军及海军陆战队中3890个军人进行有关食物爱好的调查。以下25种是378种食物中最受欢迎的。
　　1．牛奶
　　2．烤牛排
　　3．鸡蛋
　　4．玉米
　　5．桔子汁
　　6．草莓油酥饼
　　7．法式煎马铃薯
　　8．炸鸡
　　9．冰淇淋
　　10．牛奶和冰淇淋的混合饮料
　　11．熏肉
　　12．肉酱面
　　13．肉圆面
　　14．啤酒
　　15．熏肉、莴苣和西洋柿三明治
　　16．牛奶和冰淇淋混合饮料
　　17．奶油玉米
　　18．桃
　　19．桔子
　　20．法式炸虾
　　21．苹果（新鲜）
　　22．意式馅饼
　　23．火腿
　　24．西瓜
　　25．桔子
　　表示在调查中两次提出，目的是了解军人在第二次被问到该项食物时是否会有一个不同的评价。
