约翰·韦恩眼中纳5名最佳男女演员

　　约翰·韦恩（1907-1979）在电影中扮演超级英雄达50年之久。最初他只是在廉价的西部影片中担任一般的角色。1939年他在影片《关山飞渡》中的表演赢得了评论界的一致赞扬，从此他确立了自己在好莱坞的明星地位，他先后演过的电影多达几十部，其中有《红河》、《硫磺岛的沙滩》和《绿色贝蕾帽》等，1969年他被授予奥斯卡奖，表彰他在影片《真正的勇气》中的杰出表演。
　　1．斯潘塞·特雷西
　　2．伊丽莎白·泰勒
　　3．凯瑟琳·赫本
　　4．劳伦斯·奥立佛
　　5．莱昂内尔·巴里莫
