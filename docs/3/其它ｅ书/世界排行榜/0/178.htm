25种奇特的工作

　　1．琴肚装配
　　装配钢琴的内部或腹部的零件。

　　2．骨头粉碎
　　操纵碎骨机，粉碎后的动物骨头将用来制造胶水。

　　3．胸衣熨烫
　　专门熨烫男女上衣的胸口部分。

　　4．取脑髓
　　将动物脑袋放在屠场的桌上或挂在钩子上，打开颅骨，然后剔出脑髓。

　　5．鞋底磨光
　　将鞋底的前面部分磨光擦亮。

　　6．小鸡性别辨认
　　插入一只灯管，检查小鸡的性别，然后将雌雄分开。日本有一个大学专门设立了鉴别小鸡性别的学位。

　　7．涂料喷洒
　　在制革过程中，将一种涂料喷洒在经过鞣制的皮革上。

　　8．车厢挂接
　　在火车停车场内挂接或脱御车厢。干此工作的人在开动车厢时使用手刹车来控制速度。

　　9．判断蛋味
　　将蛋打碎。闻气味以检查是否变质。

　　10．卷发
　　烫发前专门用手指给人卷发。

　　11．指针安装
　　在钟和表的装配中，专门安装指针。

　　12．鞋底凿槽
　　在制鞋过程中，操纵机器将鞋底切割成符合脚底的形状。

　　13．烫猪头
　　在屠宰场，用烙铁烫除猪头上的毛。

　　14．布匹检查
　　在纺织厂里，通过卷动码布机检查布AE？的质量。

　　15．糖料调制
　　在糖果制作中，调配糖果的原料。

　　16．标记制作
　　通过在纸衬上排列并裱贴字母、标识语和数字来制作标记或指示牌。

　　17．母本维修
　　修理金属母本唱片，主要是清理音纹中的灰尘和金属屑。唱平生产必须通过金属唱片母本。

　　18．清理工业垃圾
　　清除厂区、工地或矿山周围的垃圾和瓦砾。

　　19．制作珠宝盒内托架
　　向机器送入纸板和织物，机器将纸板和织物粘贴在一起，然后将之加工成珠宝盒内的托架。

　　20．缝合裤裆
　　操纵缝合裤裆的机器。

　　21．落线筒
　　纺织厂内专门从粗纺机上取下线筒的工作。

　　22．橡胶带卷合
　　在轮胎制造中，操纵一台将橡胶带卷合在一起的机器。

　　23．牛仔监督者
　　监督牛仔。

　　24．使鞋面皮革起皱
　　操纵起皱机，使鞋面皮革产生细小的皱纹。

　　25．木棒颈部绕线
　　在高尔夫棒的生产过程中，操纵一台机器，将尼尤线绕在木棒的颈部。
