11起重大的盗墓事件

　　1．居鲁士大帝（公元前600？-前529）
　　这名中东的征服者被安葬在一个高35英寸的石灰石坟墓内，在他死后200年时，亚历山大大帝手下的军官在公元前330年进入这座坟墓、发现了里面的黄金棺木。6年后，当亚历山大本人来到时，坟墓已被破坏，里边的财宝也已被盗走。他严厉地惩罚了守墓的卫兵，并将一个有牵连的军官处死。他亲自命令将居鲁士的遗体放回坟内，并用盖有他本人印章的封条封闭了出入口。以后该墓曾多次被盗。今天这座白色巨坟仍耸立在伊朗部派帕萨尔加德城，但里边的财物则早已被洗劫一空。

　　2．威廉一世（1027-1087）　英国国王
　　威廉一世豪华葬礼是在法国的圣斯蒂芬教堂举行的。当主教们把这名征服者的死去多日的尸体放入一只狭长石棺时，他的肚皮突然破裂，顿时散发出一种无法忍受的臭味。参加葬礼的人们纷纷逃离，致使葬礼无法继续进行下去。1562年盗墓者闯入威廉雄伟的大墓，将里边的财物洗劫一空，只剩下一根威廉的股骨。80年后这根股骨又被葬入另一个墓穴。但在法国革命期间，即使这根小小骨头也难逃劫数。威廉的第一座大墓如今仍在教堂的旁边，但墓内已空无一物。

　　3．劳伦斯·斯特恩（1713-1768）　英国小说家
　　这位小说《项狄传》的作者死后被安放在伦敦圣乔治墓地（今日伦敦海德公园幼儿园的所在地），人们都知道该墓地是盗尸者们袭击的主要目标。英国一些医学院校供解剖用的尸体很多是来自圣乔治墓地。就在斯特恩被安葬后没几天，他的尸体就出现在剑桥大学的解剖台上，当时在场的一个斯特恩的朋友见到后，顿时惊叫起来。后来斯特恩的尸体得到重新安葬。1969年斯特恩财产托管会从他的坟墓中找到了他的颅骨，将之移葬到北约克郡斯特恩旧居的附近。

　　4．黑鹰（1767-1838）　塞克部落酋长
　　1832年黑鹰战争中，塞克部落失败。结果土地被剥夺，部落被征服，连他们的酋长，这位英勇顽强的斗士也惨死在牢房。他的尸体被埋葬在一只低矮的坟墓内。9个月后，当地一个叫詹姆士·特纳的医生将他尸体盗出，并携尸逃到伊利诺利。他把尸体身上的肉全部割下，准备将他的骷髅公开展览。当爱荷华州长得知此事，坚持要特纳把尸体送回爱荷华。可州长拿到尸体后又送给了特纳的一个朋友。此人一直把骷髅放在自己的办公室内，时间长达13年之久。据说后来黑魔的妻子前来观看时，表情简直像个好奇的儿童。可惜1853年的一场大火把房子连同黑鹰的骨头全部烧成了灰烬。

　　5．约翰·斯考特·哈里森（1804-1878）　美国众议员
　　十九世纪盗尸之风严重地困扰着中西部各州。而神圣的医学院校则是这些尸体的最终买主。这名父亲和儿子分别是美国总统的众议员死后被埋葬在位于俄亥俄州北本特他的家庭墓地内。参加葬礼的人突然发现哈里森新近去世的侄孙戴文的坟墓可能被盗。进一步调查后发现坟内的尸体已经失窃。哈里森的儿子约翰当即同一些人一起进行追查。一天后，年轻的约翰在辛辛那提的俄亥俄医学院的停尸间不仅发现戴文的尸体，而且还发现自己父亲的尸体。原来就在他们忙于寻找戴文尸体时，盗贼就乘机搬起了刚刚落葬的哈里森的遗体。后来两具尸体都放回了原处。非法向医院提供尸体的人也被逮捕。有关这个案件的广泛新闻报道导致了法律条款的某些改变，这些改变对改善医生在公众中的形象起到了积极的作用。后来，哈里森的遗体被重新安葬在辛辛那提的斯普林园林公墓。

　　6．坐着的公牛（1834-1890）　苏族酋长
　　这个印第安首领被埋葬在北达科利他州耶茨堡的波斯特公墓。整整50年没有受到惊动。后来南达科他州莫伯里奇（酋长的出生地）一个做旅游生意的商人开始组织人马寻找他的遗体。由于北达科他州当局拒绝给予合作，来自莫伯里奇的一些盗墓者于1954年一天夜里进入墓地，挖出他的遗骸。他们把遗骸放入一只精美的盒子内。今天在"坐着的公牛"公园内建有他的坟墓，墓边矗立着一个巨大的大理石座基，座基上安放着一尊重3吨的酋长半身花岗岩雕像。为了防止被盗，他们将棺材葬入一个混凝土的结构内。

　　7．亨利·贝当（1856-1951）　法国军人
　　这名第一次世界大战中的英雄在第二次世界大战中却成了维希傀儡政权的元首。大战后，他因叛国罪而被判处无期徒刑。后来他死于法国大西洋边的约岛，他的尸体也被葬在那里。1973年2月一个右翼政治组织将他的棺木挖出，他们坚持说蓬皮杜总统答应把他重新安葬在军人公墓中的烈士墓地。同一星期警察在巴黎附近一个车库内停放着的汽车里发现了贝当的棺材。后来当局为贝当举行了隆重的重葬仪式，总统还送了一只花圈——但安葬的地点仍是约岛。

　　8．弗朗西斯科·维勒（1877-1923）　墨西哥革命领袖
　　维勒遇刺身亡3年后，一个墨西哥陆军上尉打开墓穴，盗走了他的颅骨。对于他为何要肢解尸体，并且只盗走维勒的颅骨这个疑问，当时有一种较为普遍的说法，说是因为美国政府为了证实这个墨西哥革命者确已死亡，愿意悬赏5万美元给任何能提供维勒死亡证据的人。但这一说法至今尚未得到证实。而且颅骨也未找到。1976年，原埋葬在墨西哥北部派拉尔小镇的维勒尸体被重新安葬在墨西哥城革命纪念馆的地下室内。

　　9．查理·卓别林（1889-1977）　英国电影演员、导演、制片人
　　1977年12月27日"查理"卓别林被安葬在瑞士他家附近的一个乡村公墓内。2个月后的一天早晨，当村民们醒来时，发现卓别林的墓穴已被挖了个洞。盗墓者通过电话要求家属用60万美元赎回卓别林的尸体。但此要求遭到了卓别林遗孀的拒绝。警察根据打电话的地点进行搜查，终于抓住了作案的两个东欧难民——一个波兰人和一个保加利亚人。他们将这只重300磅的棺材埋在了村外10哩远的玉米地里。1978年5月23日，查理的遗体被安放在村公墓内一个特制的混凝土墓穴里。该案的主谋罗曼·华达斯被判1至4年半的徒刑。同谋加尔乔·加内夫被判18日徒刑，缓期执行。

　　10．迈克尔·托德（1909-1958）　美国电影制片商
　　托德死于飞机失事，他的尸体被葬在伊利诺斯州森林公园内的瓦尔德海姆公墓内。1977年6月，托德的妻子伊丽莎白·泰勒前往凭吊后的2天后，托德的坟墓被盗。他的棺材被撬开、尸体被搬走。警察根据3天后盗墓者打来的电话这条线索找到墓地附近的一片树林里，发现有一只橡皮袋，袋里放着托德的尸体。有人猜测，这些未被抓住的盗贼原以为棺材内有价值昂贵的手饰，但托德的儿子则认为这种猜测没根据。后来根据家人的要求，托德的尸体被埋葬在一个秘密的地点。

　　11．艾伦·沃茨（1915-1973）　英国哲学家
　　这名受人欢迎的禅宗和东方哲学的倡导者死后被火化。他的大部份骨灰埋在加州密尔合谷的禅宗中心。他的妻子简得到3块骨头碎片，她将这些碎片放入一只镶嵌绿色松石的铜制西藏匣子内，然后把它当作奉祀物置放在家中。1975年8月的一天，一个盗贼将此偷走，同时还偷走了一台彩色电视机。沃茨太太发出警告说该匣子受到神的保护，任何人偷了它都会遭殃，但究竟谁是盗贼，以及诅咒是否验灵至今仍无人知晓。
