历史上10位马塞尔·马索最愿意成为的人

　　法国著名的滑稽演员马塞尔·马索首次成名是在1947年。当时他塑造了一个名叫"比齐"、面孔涂满白色的小丑。这个苦恼的形象是他以狄更斯小说《远大前程》的主人公为原型而创造的。1949年马索建立了自己的公司，他到世界各地巡回演出，其中包括多次到美国献艺，他还拍过许多短篇和一部长篇。
　　1．耶稣·基督
　　2．摩西
　　3．米凯朗琪罗
　　4．利奥纳多·达·芬奇
　　5．悉达多（佛陀）
　　6．莫扎特
　　7．亚里士多德
　　8．莎士比亚
　　9．爱因斯坦
　　10．托尔斯泰
