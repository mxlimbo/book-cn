10种历史上最具影响力的汽车

　　1．奔驰（1885）
　　奥托·本茨不仅第一次成功地制造出了内燃式汽车发动机，而且还制造出了第一辆供公开销售的汽车。

　　2．杜里尔（1892-1896）
　　这是第一辆在美国诞生的汽车，1895年它获得了汽车大赛的第一名。

　　3．戴迪恩（1899-1900）
　　它有一个很小的但速度很高的单汽缸引擎，极其受人欢迎，在许多国家同时生产。美国的戴迪恩汽车制造公司于1908年又制造出装有V－8型引擎的汽车。

　　4．奥德斯摩比尔（1900-1905）
　　它的速度极高，是第一种大批量生产的单AE？缸汽车。

　　5．福特T型（1908-1927）
　　这是有史以来影响最大的汽车，它改变了人们的生活方式，不仅是美国人的，而且是全世界人们的生活方式。

　　6．卡迪莱克（1912-1915）
　　它采用了自动点火装置，并且成功地安装了V－8型发动机，从而得以大批量生产。

　　7．帕卡德（1915-1932）
　　它的性能优越，造价昂贵，很多年来一直是深受上层阶级欢迎的豪华型汽车。

　　8．埃塞克斯（1922）
　　它是第一种价格低廉的汽车，为当时的汽车工业注入了新的生机。

　　9．大众（1936-　）
　　大众汽车用它的小型车身赢得了两百多万的用户。它有一种空气冷却的发动机，设计简单实用，在战后创造了销售方面的纪录。

　　10．福特（1965-1966）
　　这是一种新型的个人化的小型汽车，一问世就受到了公众的欢迎，并且使许多厂商紧随其后。
