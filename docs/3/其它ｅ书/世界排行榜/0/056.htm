7个将自己装在容器里冲下尼亚加拉瀑布的人

　　1．安妮・埃德森・泰勒（1901）
　　一个密执安州的女教师（1858-1924？）。她希望用这一冒险行动来为学校集资，当她将自己装进一个412英尺的木桶里，从瀑布上冲下来后，并没受什么伤，但她也没能获得多少资助。

　　2．博比・利奇（1911）
　　博比・利奇（1842-1920）是躲在一只铁制锅炉里从上面冲下瀑布的，他的脸部和小腿都受了伤，毕竟他当时已是69岁高龄了。后来，他是由于踩了一块被人们扔在地上的橘皮跌倒身亡的。

　　3．查尔斯・斯蒂芬斯（1920）
　　他是一个58岁的英国理发师。事先，他用铁环将自己的手脚都固定在木桶里，结果，木桶在冲下爆布时被撞碎了，人们只找到了他的一条手臂。

　　4．吉恩・卢西尔（1928）
　　吉恩・卢西尔出生于1893年，是多伦多一个马戏团的演员。他用铁和橡皮制作了一只大圆桶，以保证自己的生命安全。当时有大约10万人前来观看他的冒险――虽然他获得了成功，但人们并没有给他太多的掌声。

　　5．乔治・斯塔赛克斯（1930）
　　他是一个自成一派的神秘主义者，而且还是一本名为《人类各个时代的神秘面纱》的书的作者，虽然这本书并未出版。1930年他46岁，住在布法罗市，为了写一本有关自己经历的书，他便乘坐一只木桶从尼亚加拉瀑布上冲了下去。没料到他竟被卡在瀑布下面的礁石中达14个小时之久。

　　6．威廉・赫尔（1951）
　　威廉・赫尔（1913年出生）是一个加拿大人，因无执照打猎而受过监禁，他希望用这次冒险行动为自己争得荣誉。他在一只木桶里放了14只汽车轮胎和厚厚的棉垫，以为能万无一失。共有20万人来观看，其中也包括他的母亲。但是，他没能活着回到岸边。

　　7．威廉・菲茨杰拉德（1961）
　　他是出生于1931年的一个美国黑人，虽然他的冲流工具和吉恩・卢西尔的一模一样，但他的运气却远不如他，菲茨杰拉德葬身在大片布的轰鸣声中。
