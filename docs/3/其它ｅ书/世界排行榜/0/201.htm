以意大利人姓名命名的5个人体部位

　　1．科蒂器官
　　内耳的听觉器官，以意大利人阿尔方索·科蒂的姓命名。（1822-1878）

　　2．尤斯塔皮奥管
　　从中耳到咽喉部的通道。其作用是平衡耳内的压力。以意大利解剖学家巴托洛米欧·尤斯塔皮奥的姓命名。（1524？-1574）

　　3．法洛普奥管（输卵管）
　　联接卵巢和子宫的两根管道。女性的卵子通过该管道排入子宫。以意大利解剖学家加布利尔·法洛普奥的姓命名。（1523-1562）

　　4．鲁菲尼颗粒
　　对温度作出反应的感觉神经末梢以解剖学家安吉罗·鲁菲尼的姓命名。（1864-1929）

　　5．塞托里细胞
　　睾丸内给精液输送营养的细胞。以组织学家恩利科·塞托里的姓命名。（1842-1910）
