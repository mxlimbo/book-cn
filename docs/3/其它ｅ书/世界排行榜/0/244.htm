乌里·盖勒眼中历史上最伟大的10名具有心灵感应的人

　　原以色利列兵乌里·盖勒认为他的心灵感应能力来自遥远的胡发星球，从1969年开始他专门从事心灵感应能力的表演。他经常在世界各地的电视台和夜总会表演他的能力。美国、英国及法国的一些科学机构都对他的这种超凡感应能力进行过研究。
　　1．乔治·古尔捷耶夫
　　2．格里高利·拉斯普丁
　　3．尼古拉·特斯拉
　　4．弗兰兹·安通·迈斯穆
　　5．诺斯特勒达莫斯
　　6．艾里娜·彼得罗夫娜·布拉优斯基
　　7．埃德加·凯斯
　　8．丹尼尔·邓格勒斯·霍姆
　　9．尼娜·库拉吉娜
　　10．伊琳娜·休斯
　　乔治·古尔捷耶夫（1872-1949）是一名俄国神秘主义者，他教授他的追随者们如何超越自己的智力和体力的局限。出生于克罗地亚的电子学先驱尼古拉·特斯拉（1857-1943）发明交流发动机以及用于收音机及电视机中的特斯拉线圈。丹尼尔·邓格勒斯·霍姆（1833-1886）系苏格兰人，是一名受欢迎的心灵感应师，他能升空，能说自己不懂的语言，曾为帕拉特·勃朗宁、拿破仑三世及其他一些名人表演过他的特殊技能。尼娜·库拉吉娜于20年代后期出生于列宁格勒，她曾向一些科学家演示在自己的皮肤上变出烧灼的痕迹，并表演对无生命物体产生影响的能力。
