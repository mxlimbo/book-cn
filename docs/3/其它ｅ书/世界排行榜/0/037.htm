导致妇女和儿童死伤的6次海难

　　1．"凤凰号"　密执安湖（1847）
　　该汽船往返于五大湖区的布法罗市和芝加哥市之间，船上的乘客是初来美国开始新生活的荷兰移民。当票船航行到密执安湖心时，锅炉房突然爆炸起火，火势熊熊，很快便席卷了全船。船员争先恐后冲向"凤凰号"仅有的两艘救生艇，他们登上后才向为数不多的几位乘客示意，让他们一起逃生。当救生艇离船而去时，剩下的乘客，其中包括不少妇女儿童，从"凤凰号"上跳下，意欲攀上救生艇的船舷，却被水手们将手掰开，溺水而死。这场灾难中丧生者达190至250人。

　　2．"北极号"　加拿大纽芬兰岛附近（1854）
　　美国的"北极号"客轮在大雾中与另一艘船相撞后，开始下沉。船长詹姆士·鲁斯命令水手首先让乘客登上救生艇——这是海上的一条不成文法律。然而，当水手们看到海浪涌上甲板时，他们只顾自己逃生，把船长的命令当作耳边风。看到这一情形，船上的男乘客也纷纷跳下沉船，只留下束手无策的船长和一群妇女、儿童。全船共有435人，其中只有61个水手和24个男乘客生还。

　　3．"晨星号"　伊利湖（1868）
　　在"晨星号"驶离克利夫兰，朝底特律破浪而去的两小时后，它正面撞上了一条货轮"科特伦特号"。两条船顿时嵌在一处，无法脱离，随即慢慢下沉。"晨星号"上头等舱的51名乘客，以及一些移民——其中包括妇女和儿童，登上了两只小小的救生艇，不过在此之前，船上的水手早就捷足先登。死亡人数23至32人。

　　4．"佛罗里达号"和"共和国号"　南将各脱岛附近（1909）
　　在大雾中，一艘载着意大利移民的小船正驶往纽约，不料一头撞上白星航运公司的"共和国号"，后者很快下沉。船员们先将妇女儿童送往暂时安全的"佛罗里达号"。但是当前来救援的"波罗的号"赶来时，"佛罗里达号"上的船员先让"共和国号"上的头等舱乘客登上救生艇，而把许多意大利妇女、儿童撇在一边。意大利人大声抗议，但是无济于事，特权阶层占了上风。这次事故无人丧生。

　　5．"莫罗城堡"号　新泽西附近（1934）
　　该船堪称瓦德航运公司的旗舰，当它从加勒比海中的忘忧岛返航时，突然起火，时值初秋的黎明之前。水手们没有唤醒船上的乘客，抢先登上救生艇逃命。第一条救生艇上只载着3个乘客，第二条救生艇上坐的全是船员，全船318名乘客中，有86人丧生。

　　6．"海伦娜号"　亚德里亚海（1971）
　　凌晨6点，猛烈的爆炸声震撼了这条希腊渡轮，当时船上1，100名观光游客还穿着睡衣。事后生还者指控船员们只顾自己逃命，全然不管妇女和儿童的尖叫。调查表明，该渡船严重超载，乘客比法定人数多182人。渡船不堪负担以至造成24人死亡。
