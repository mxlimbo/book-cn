历史上6条严重的裂缝

　　1．自由钟的裂缝
　　约翰·马歇尔1755年生于弗吉尼亚，就在他出生的两年前，1753年6月在宾州费城的一座建筑物上挂起了一个大钟，该钟重一吨，底部圆周长12英尺，传说1776年7月8日当美国的独立宣言公开宣读时，该钟响起了宏亮的祝贺声，从此人们便称之为自由钟。1801年马歇尔被任命为美国大法官，1835年马歇尔去世时，自由钟的金属壁上突然出现了一条垂直的裂缝，因而当它为大法官敲响丧钟时，声音也变得十分沉闷。

　　2．南福克大坝的裂缝
　　1889年春末，宾州西部大雨滂沱，科纳莫河上南福克大坝的库区内水位暴涨，在坝下离匹兹堡55哩远的一条狭长的谷地内有一座居住着2，209个居民的小镇，此刻他们丝毫没意识到末日即将来临。5月31日在空前的水压下，大坝终于裂开，并很快全部倒塌。顷刻约翰小镇被30英尺高、总量达7，800万吨的汹涌洪水全部吞没。

　　3．将加州切开的裂缝
　　这实际上是地壳的断裂，这条圣安得利亚斯断层从加州的西北角一直延伸至加州湾，总长达600哩。1906年5月18日沿断层270哩的范围内，地壳突然活动，造成旧金山的大地震。断面水平抖动高达21英尺，整个城市被大火摧毁，由于水管全部破裂，因而无法扑灭大火。

　　4．奥威尔·赖特的螺旋桨上的裂缝
　　正当哥哥威尔伯·赖特在欧洲作飞行表演时，弟弟奥威尔·赖特在弗吉尼亚米艾堡陆军基地作试验飞行。1908年9月17日奥威尔驾驶飞机上天，坐在机舱内的还有托马斯·塞尔弗里奇中尉，突然机上两个螺旋桨中的一个产生了一条细小的裂缝，飞机立即从75英尺的高度坠落。奥威尔幸免于难，但腿部、臂部和肋部都受重伤，而塞尔弗里奇中尉由于头部重伤，不久便成为航空史上第一个因飞机失事死亡者。

　　5．造成美国打谷机号船沉没的裂缝
　　1963年4月10日，美国核潜艇在麻省科德角海区沉入8，400英尺深的海底。同年8月美国一艘深水潜艇发现一段标有打谷机号潜艇船壳号码的铜管，随即海军有关部门进行了调查，结论是潜艇的管道出了问题。一个海洋学家认为，由于当时风浪过大，造成海洋流速异常，潜艇不得不潜入很深的海底，结果承受的水压超过标准，很可能是在船体或管道产生了裂缝，然后使整个船只崩溃。

　　6．几乎杀死上将的裂缝
　　美国海军上将理查德·伯德三次率舰出征南极，没有损失一人，在他于1934年3月28日出发的第二次探险期间，他曾在南极保林气象预报站单独生活了4个月。一次他不小心踩到冰面上被雪盖住的一条长长的裂缝，并跌了下去，幸好下面的一条深沟只有3英尺宽，他自己设法爬了出来，他低头再朝下看，发觉在这翠绿色的冰墙下面，这条越往下越开阔的深沟竟有几百英尺深。
