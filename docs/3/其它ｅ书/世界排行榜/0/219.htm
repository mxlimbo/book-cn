历史上10位爱米莉·哈恩最愿意成为的人

　　作为威斯康星大学采矿工程系的第一名女学生，毕业后她曾先后从事过中学英语教学、大学讲师、剧本创作、新闻采访等工作。她还是一名著有20本书的作家。她最近出版的两本书是：反映美国妇女争取平等权利的《曾受崇拜》和D·H劳伦斯的传记《洛伦佐》。
　　1．希罗多德
　　2．阿天拉·贝恩
　　3．费德南·麦哲伦
　　4．英国查理二世
　　5．李太白
　　6．罗伯特·默恩斯·耶基斯博士
　　7．玛丽·蒙塔克夫人
　　8．多萝西·奥斯本
　　9．马嘎尔尼勋爵
　　10．理查德·伯顿爵士
　　注：阿夫拉·贝恩（1640-1689）是英国小说家、剧作家和诗人，她成为英国第一位靠写作为生的女性。李太白（？-762）是"八大醉仙"中的一位，被公认为中国最伟大的诗人。罗伯特·默恩斯·耶基斯博士（1876-1956）系美国心理学家，在推进比较动物心理学研究方面作出了巨大的贡献。玛丽·蒙塔克夫人（1689-1762）系英国诗人和书信作家，尤以其机智的语言和反传统的风格著名。多萝西·奥斯本（1627-1695）是英国政治家之气。她写给坦普尔的信在语言上堪称英国之最。乔治·玛戛尔尼勋爵（1737-1806）是英国首任驻北京的外交代表。
