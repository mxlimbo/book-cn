10次著名人物间的会见当时其中的一方还未出名

　　1．1789年在纽约，乔治·华盛顿和华盛顿·欧文见面
　　当时华盛顿总统去百老汇的一家商店参观，欧文家的一个仆人发现了他，急忙带着6岁的欧文来到总统跟前。总统问欧文叫什么名字，还抚摸了一下他的头。当时谁也不会想到，欧文长大后会成为一名著名的作家，还写了一本《华盛顿传》。

　　2．1860年在波士顿，拉尔夫·沃尔多·爱默生为年轻的温德尔·霍姆斯看一篇习作
　　年轻的霍姆斯在大学里写了一篇有关柏拉图的批评性的文章，将它带给爱默生看，因为爱默生是他们家的老朋友，霍姆斯很尊崇他。57岁的大作家虽然手头有很多工作，但还是仔细地读了这篇文章，并且给了未来最高法院的首席法官这样的忠告：“假如你是朝一位国王射击，就必须一枪将他击毙。"霍姆斯将他的这篇文章扔进了垃圾箱。

　　3．1868年在法国，史文朋遇见了莫泊桑
　　后来成为法国大作家的莫泊桑当时只有18岁，他在海滩上看见一个像是快要淹死的人，想去帮助他，这人正是性情古怪的英国大诗人史文朋。莫泊桑把自己介绍给了史文朋，史文朋便邀请他去自己的别墅吃晚餐。但是，餐桌上的那道主菜差点没把莫泊桑吓昏过去：盘子里装的是一只大得出奇的烤猴子！

　　4．1887年在华盛顿，格罗弗·克利夫兰接见了富兰克林·罗斯福
　　当时5岁的罗斯福跟着父母去首都参观，他被带到白宫去见总统。根据罗斯福后来的回忆，克利夫兰俯视着这位将来的接班人说：“我的小朋友，我要给你一个奇异的祝愿：你也许会成为世界上的任何一种人，但永远也不会成为美国总统。”

　　5．1894年在加利福尼亚，本杰明·哈里逊因为不付25美分而和赫伯特·胡佛相遇
　　当时刚卸任的美国总统哈里逊在斯坦福大学担任客座教授，他没有买入场券就溜进球场看棒球比赛。学生会的领导人、19岁的胡佛正负责球场秩序，尽管他非常清楚这个棒球迷正是哈里逊总统，但他还是抓住他，并且十分客气地要他付出25美分。当然，哈里逊立刻掏出一枚硬币给了这位未来的总统。

　　6．1895年在美国印第安纳波利斯，刘·华莱士提醒布思·塔金顿在将处女作付印时要三思而行
　　华莱士这位著名的作家，在去世前10年时，曾去家乡访问，在那里他遇见了塔金顿。塔金顿此时正竭力设法使自己的第一本书付印。华莱士告诫他说：“我第一部小说的出版，使我的律师生涯终结了。因为，当商人或农民组成的陪审团只要一听到小说这个词，就会立刻皱起眉头，我宁愿穿着马戏团小丑的服装出现在法庭上，也不愿戴着小说家的头衔。”但是尽管塔金顿听了这番话连连点头，但他后来还是写了许多畅销小说，其中包括《非凡的安伯森》和《十七岁》。

　　7．1905年在华盛顿，西奥多·罗斯福派人叫来了埃德温·鲁宾逊
　　鲁宾逊的诗集《夜幕下的孩子》给罗斯福总统留下了深刻的印象，于是他派人叫来了这位年轻的作者。鲁宾逊当时的生活十分窘迫，总统向他提供了纽约税务局的一份工作，并给了这样的忠告：“我想你最好首先考虑你的诗，其次再考虑你的关税事务。"这个职位使鲁宾逊在整整4年时间内得以继续创作诗歌。

　　8．1910年在纽约，萨拉·伯恩哈特在舞台侧幕旁遇见了莉莲·盖希
　　莉莲·盖希成名之前，曾在萨拉·伯恩哈特主演的一部戏中担任舞蹈演员的角色。当她们在侧幕旁等待上场时，这位红遍欧美的法国女演员抚摸着年轻的莉莲那漂亮的鬈发，用法语说了许多赞美的话。可惜莉莲对法语一窍不通。

　　9．1921年在美国俄亥俄州的阿克伦，哈维·法尔斯通给了温德尔·威尔基一个忠告
　　29岁的威尔基在为法尔斯通的一家轮胎公司工作了好几年之后，决定辞职，去开设律师事务所。法尔斯通，这位53岁的老板在向年轻律师告别时直言不讳地说：“我很喜欢你，年轻人，但你不会有什么太大的成就的。”“为什么？"威尔基问。"因为你是一个民主党人，而任何一个民主党人都不会有很大成就的。"19年后，威尔基竞选美国总统——作为一个共和党人。

　　10．1951年在美国内布拉斯加州的林肯市，约翰尼·卡森第
　　一次遇见迪克·卡维特
　　卡维特当时刚10岁出头，他偷偷溜进后台。卡森当时已经非常出名，经常在电视里露面。卡森听说这个孩子也会表演魔术，就十分高兴地在当天晚上的表演中将这个小同行介绍给了观众，这使得卡维特兴奋不已。
