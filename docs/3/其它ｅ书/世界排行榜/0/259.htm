历史上12个“第二”

　　1．第二个教皇
　　继圣彼得之后，历史上第二个教皇是圣利努斯，人们称他为殉教者，但对他的情况知之甚少。只知道他在位时间是公元67年至79年，可能是个犹太人，出生在意大利的塔斯卡尼。

　　2．第二次环球航行
　　在麦哲伦作第一次环球航行的60年后，英国人弗朗西斯·德雷克乘坐一艘长75英尺的金鹿号帆船又一次绕地球航行一周。这次环球航行并非平淡无奇，他们曾抢夺了一艘装满财宝的西班牙船，在巴塔哥西亚镇压了部分船员的叛变，还有一次船只不幸撞上暗礁，1580年德雷克回到目的地英国。

　　3．第二次气球飞行
　　1783年9月19日，一只羊、一只鸭和一只公鸡成了第一批作品球飞行的动物。法国国王路易十六和玛丽亚·安托瓦内特在凡尔赛宫亲眼目睹了这3个动物被放入吊在一个热气球下面的篮子里。他们在空中飞行了8分钟，然后安全地降落。1783年11月21日，法国作了第二次气球飞行，但这次在布洛涅森林别墅的园林内升空的热气球装载的却是两名法国人，他们在空中自由地飞行了25分钟。

　　4．第二次赴南极探险
　　1911年阿蒙森到达南极的几个月后，由罗伯特·斯考特上尉率领的一支厄运探险队也到了南极，可这一次就在他们离开食品补给站11哩远的地方遭到了连续9天的暴风雪，结果全部因饥饿和寒冷而死去。他们留下了令人心酸的告别信和大量记述这次探险活动的日记。

　　5．美国第二次被选入国会的女子
　　1920年出生在印第安人的领地内，父母均为传教士的玛丽罗伯逊被奥克拉荷马州的选民们选为第67届议会的众议员。任期为两年——1921年3月4日至1923年3月3日。玛丽曾任中学教师和督学，第一次世界大战期间她在该州的莫斯考奇市组织了一个供餐机构，专门向经过该市的成千上万士兵提供免费饮食，从而大大提高了她的知名度。来自蒙太那州的共和党人珍尼特·拉斯金是第一个进入国会的女子，她是个和平主义者，她担任众议员的时间是1917年至1919年。

　　6．第二个横渡英吉利海峡的女子
　　第一个作次壮举的女子名叫格特鲁德·埃德利。在不到一个月的时间里，于1926年8月26日，另一名美国妇女米尔·科森夫人从法国出发，用了15小时30分的时间游过了波涛汹涌的英吉利海峡，到达英国。她的速度比埃德利慢了59分钟。

　　7．第二个美国亿万富翁
　　第一个是约翰·洛克菲洛。第二个便是亨利·福特。当他仅是个失业的机修工时，便萌发了一个念头，即设计一种后来叫做T型的汽车。他把金钱看成是"传送带的一个部分"。福特发明的装配线，以及他付给每个工人空前的5美元一天的做法对工业世界产生了革命性的影响。当一个记者问他，作为一个亿万富翁有什么想法时，他回答是：“哼！狗屁不如。”

　　8．卡波思犯罪集团的第二个领袖
　　人称"执法杀手"的弗兰克·尼蒂最初仅是卡波思组织中一名枪手，后来被提拔为财产管理人。在卡波思锒铛入狱后，他便同卡波思的弟弟拉尔夫共同领导这个庞大的集团，尼蒂后因逃税的罪名被判徒刑。当1943年司法机关准备以敲榨罪而第二次将他投入监狱时，他开枪自杀了。

　　9．第二个投原子弹的人
　　根据杜鲁门总统的命令，保罗·蒂贝茨上校在日本广岛投下第一颗原子弹，接着于1945年8月9日，25岁的美国空军少校查尔斯·斯威尼驾驶着B-29轰炸机在长崎上空扔下了一个外号叫"胖子"、外型像只西瓜的钚原子弹。结果36，000多人被杀，当斯威尼看到原子弹爆炸后升起的蘑菇云时，他曾责问自己，"我们干了些什么啊？"可不久他便毫无内疚地感到自己只是执行了任务。

　　10．第二个进入美国当代二大职业棒球联盟的黑人
　　在杰基·罗宾逊第一次冲破种族壁垒后，拉里·道比于1947年7月加盟克利夫兰印第安队。成为历史上第二个进入二大职业棒联的黑人，也是第一个进入美国联盟的黑人。他以0．301的平均成绩帮助克利夫兰队赢得1948年的三角锦旗，并且连续6年成了美国全明星队成员。

　　11．第二个登上珠峰的登山队
　　第二个登上珠峰的登山队是瑞士队，成员是厄内斯特·雷斯和弗里兹·卢克辛格。他们是1951年5月18日登上29，028英尺高的峰顶。3年前埃德蒙特·希拉里爵士和坦辛·诺凯首次登上珠峰。

　　12．第二个登月者
　　在尼尔·阿姆斯特朗踏上月球19分钟后，他的月球舱内的同伴埃德温·阿尔特林也登上了月球，时间是1969年7月21日。阿尔特林原是一名热爱体育事业的运动员，此次他随同登月的任务是帮助阿姆斯特朗收集样品。他在重力很小的月球表面，高兴地又蹦又跳。重返登月舱时，他一下子就跳到了最高一级悬梯。他当时的年龄是39岁。
