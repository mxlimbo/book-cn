鲍勃·库西眼中的9个最佳篮球明星

　　鲍勃·库西被专家称为"篮球史上最杰出的后卫"，他出色的投篮和传球技艺使波士顿的凯尔特队在50年代末和60年代初称霸美国篮坛。他本人连续9个赛季在全国篮球协会获传球得分第一名（传出的球使队员直接得分），并连续13年被选入全明星队。在他帮助波士顿队第五次蝉联冠军后，他于1963年退役，然后在一所大学担任教练。
