萨切尔・佩奇眼中的10名最优秀投手

　　1948年，当两大职棒联盟中的克利夫兰队同佩奇签订参赛合同时，他已年满42岁。在过去的20年里这个传奇式人物一直效力于黑人联盟，他的投球技术无与伦比。1934年他曾连续获胜21次并赢得62个对方无分局。这样他在该赛季创造了赢31场输4场的纪录。据说在他整个棒球生涯中他曾投出300个对方无分球，其中55次为无安打。1965年59岁的佩奇开始为堪萨斯队效力，他是高级职棒各队中年龄最大的球员。

　　2．山姆・斯里特
　　3．唐・纽科姆
　　4．凯特费希・亨科
　　5．山迪・库费克斯
　　6．希尔顿・史密斯
　　7．布立特・罗根
　　8．鲍勃・莱蒙
　　9．艾里・雷诺
　　10．威利・福斯特
　　注：佩奇曾为黑人联盟中的堪萨斯城帝王队打过几个赛季，斯蒂里特・史密斯和罗根也曾在该队担任投手。史密斯常常在后半段比赛代替佩奇出场，他的全经历纪录为161场胜和22场输。福斯特为芝加哥的美国"巨人"队做投手，1926年一年赢26场球。
