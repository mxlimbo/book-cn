威廉・夏勒最愿意目睹的5个历史事件

　　夏勒是当代世界著名的历史学家，驻外记者和新闻评论员。二次大战期间，他为哥伦比亚广播公司和宇宙新闻社提供大量文件资料，报道了纳粹的兴起和灭亡。1941年后他写了许多畅销书，如《柏林日记》、《第三帝国的兴亡》以及《二十世纪之旅》。

　　1．甘地之死（1948．1．30）
　　甘地去世时我的朋友文森特・希恩和埃德加・斯诺都在他的身边，而我却不在。我认识甘地更早，他是我所认识的最伟大的人物，也是对我的思想影响最大的一个伟人。

　　2．苏格拉底之死（前399）
　　柏拉图曾对此作过令人难忘的叙述，那情景常在我脑海出现。

　　3．阿道夫・希特勒之死（1945．4．30）
　　作为一名记者我曾在他处于权力顶峰时在柏林采访过他。我也很愿意亲眼看到他在一切都丧失后是如何地消失的。

　　4．在奥林匹亚举行的奥林匹克运动会
　　最好是苏格拉底从雅典徒步前往观看的那一次奥运会。我曾报道过1928年和1936年在欧洲举行的现代奥运会。运动员们表现出色，可惜观众中没有一个像苏格拉底那样能传递智慧的哲学家。

　　5．田纳西州达顿市的一次荒唐审判（1925．6）
　　特别是克拉伦斯・达罗责问威廉・詹姆斯有关圣经内容的那一天。
