10部电视台最常放映的电影

　　《电影指南》杂志的编辑对全国各地的电视节目导演进行了一次书面调查，叫他们列举出10部在电视台最常放映、最受欢迎的电影。
　　1．《卡萨布兰卡》（1943年，汉佛莱·博加特和英格丽·褒曼主演）
　　2．《国王金孔》（1933年，布鲁斯·卡勃特和费·雷主演）
　　3．《一夜风流》（1934年，克拉克·盖布尔和克劳德特·科尔伯特主演）
　　4．《马耳他之鹰》（1941年，汉佛莱·博加特和玛丽·阿斯特主演）
　　5．《侠盗罗宾汉》（1938年，埃多尔·弗林主演）
　　6．《非洲皇后》（1951年，汉佛莱·博加特和凯瑟林·赫本主演）
　　7．《鸟》（1963年，罗德·泰勒和蒂齐·赫德里恩主演）
　　8．《公民凯恩》（1941年，奥森·韦尔斯主演）
　　9．《34号街的奇迹》（1947年，毛利恩·梅勒和埃德蒙特·格温主演）
　　10．《温柔地爱我吧》（1962年，爱尔维斯·普雷斯利主演）
