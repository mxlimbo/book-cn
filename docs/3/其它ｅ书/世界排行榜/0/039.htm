16位种植（或命令种植）大麻的名人

　　1．夏伊克・阿兹―沙瓦伊・海伊达（1150―1221）
　　海伊达是波斯神秘教会――苏菲派的创始人。据信，是他发现了大麻的功能，并在库拉森的王宫里精心培育这种植物。这位君王去世后，人们依照他的命令，在他的陵墓四周都种上了大麻。至今，苏菲派教徒仍去他的陵墓朝圣。

　　2．穆罕默德・拉尔・乌德・丁（1483―1530）
　　他是印度莫卧儿皇朝的第一位皇帝，很喜欢掺有大麻的甜食，并在德里附近种植大麻。他的孙子阿喀巴大帝（1542―1605）扩展了帝国的疆域，并吞了阿富汗和孟加拉，在整个印度北方有计划地栽种大麻和罂粟――鸦片的原料。

　　3．加西亚・达・奥塔（1490―1570）
　　奥塔是果阿的一位葡萄牙医生，他在当地种植了大麻和许多其他药物，并在一本题为《印度药物种类》的书中探讨了它们的用途。该书第三版是在印度出版的，是探险时代药物学家的必读书。

　　4．弗朗索瓦・拉伯雷（1501―1566）
　　拉伯雷是《高康大》的作者，著名法国作家，他父亲从祖上继承了一个大麻种植园，拉伯雷就是在这个种植园里长大的。在拉伯雷的另一部小说《庞大固埃》的第三卷中，作者详尽地描写了大麻的培植方法，这表明他对大麻有着丰富的知识。

　　5．列奥那德・弗切斯（1501―1566）
　　弗切斯是德国植物学家，他种植大麻及其他药物，并请艺术家把这些植物维妙维肖地临摹下来，作为他的著作《植物种族史》的插图，他为大麻确定的拉丁文名称Cannabissativa直到今天还被科学家沿用。

　　6．詹姆士一世（1566―1625）
　　这位英国国王命令本国人民栽培大麻和黄麻，并且向美洲大陆詹姆斯城每一位殖民者提供这两种植物的种子。1619年，美洲大陆的第一个殖民地政府――弗吉尼亚议会通过了一项法案，要求每家每户从下一个季节开始都必须种植大麻，这是美国历史上第一个大麻法案。

　　7．威廉・巴德（1674―1744）
　　1737年，这位弗吉尼亚庄园主为了大量获得英国政府为种植，大麻而提供的奖励，在他那片面积为18万英亩的土地上全部播下了大麻种子，事后巴德写道：“在这里的气候条件下，大麻长得很好但是这里的劳动力要比穆斯克维昂贵，运价也更高，所以我们没有赚到多少钱。”

　　8．卡洛尔・林奈（1707―1778）
　　瑞典人林奈是现代植物学之父，他首创了动植物的分类法。林奈在自己的窗台上栽种大麻，为的是研究大麻雌雄花授粉的机制。

　　9．乔治・华盛顿（1732―1799）
　　这位美国第一任总统从世界各地进口大麻种子，并于1765至1795年在维农山庄的私人葡萄园里大量栽种。他希望美国能建立自己的大麻工业，以此与英国、俄国和意大利竞争。

　　10．托马斯・杰弗逊（1743―1826）
　　杰弗逊是美国总统中最多才多艺的人，他于1811年在蒙提塞罗种了1英亩大麻，他写过一本有关培植大麻的小册子，并于1815年发明了一种大麻粉碎机。

　　11．亨利・克雷（1777―1852）
　　克雷是肯塔基的参议员，他于1810年敦促美国政府通过一项法案，要求美国海军购买本国大麻产品，而不要向外国进口。这项法案使得大麻种植业成为肯塔基州赢利最多的产业，这一状况一直持续到南北战争。

　　12．罗伯特・克里斯蒂逊爵士（1797―1882）
　　克里斯蒂逊博士是苏格兰毒物学家，不列颠医学会会长，他在爱丁堡医学院他父亲的植物园里种植大麻。他对植物中提取药物的杰出研究是现代药物学家的经典。

　　13．西奥多尔・罗斯福（1858―1919）
　　这位美国总统于1904年在华盛顿特区建立了一所联邦"毒物种植园"，其原址就在如今的五角大楼附近。那里种植了大麻、罂粟、古柯等作物，这样美国就无需向外国进口这些植物的制品了。

　　14．亨利・福特（1863―1947）
　　这位汽车大王曾为了"实验目的"种植过大麻，他在大麻地外围了一道篱笆。原先以为这块地上的大麻在他去世后已经毁掉去，但他的家人后来发现那里的大麻变成了野生植物。

　　15．林登・贝尼斯・约翰逊（1908―1973）
　　约翰逊在出任美国总统期间，曾于1968年下令在密西西比州大学的农场种植大麻，该农场是植物、化学和药物栽培的中心，如今它已经成为美国官方药物研究最重要的原料供应地之一。

　　16．托马斯・列提希（1941―　）
　　列提希是电视系列片《拉赛》中的童星，后来他于1972年被捕，理由是他在圣路易城外一处农庄里栽培大麻。
