历史上11位吉拉尔德·福特最愿意成为的人

　　这位前密执安大学橄榄球队的明星中锋，从1948年被选入众议院后开始了他不断取得成功的政治生涯。1973年阿格纽辞职后，他被尼克松选为副总统。当尼克松本人在1974年8月8日辞职后，福特成为美国第38届总统。但在1976年的大选中被吉米·卡特击败。
　　1．乔治·华盛顿
　　2．亚伯拉罕·林肯
　　3．托马斯·杰佛逊
　　4．本杰明·弗兰克林
　　5．温斯顿·丘吉尔
　　6．亚历山大·汉密尔顿
　　7．道格拉斯·麦克阿瑟
　　8．马哈德马·甘地
　　9．马可·波罗
　　10．托马斯·爱迪生
　　11．卢·格里克
　　福特指出：“我所说的自己愿意成为这些人是指如果处在他们当时的历史时期的话。”
