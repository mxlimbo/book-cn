历史上10位C.P.斯诺最愿意成为的人

　　斯诺是著名的作家、科学家和政府公务员，他在担任英国政府的科学顾问前曾在剑桥大学从事20年时间的分子物理学研究。他的著作包括长达11卷的《陌生人和兄弟》（1935-1970）、《两种文化和科学革命》（1959）以及《现实主义者》（1978）。
　　1．卢克莱修（罗马哲学家及诗人）
　　2．西德尼·史密斯
　　3．本杰明·富兰克林
　　4．奥古斯多斯·凯撒
　　5．梅尔本
　　6．康熙
　　7．佩雷斯·加尔多斯
　　8．歌德
　　9．阿基米德
　　10．维洛内塞
　　注：卢克莱修（前96？-前55）是罗马诗人和哲学家，曾著有长诗《论事物的本质》，该诗提出原子结构的规律可以解释人的大脑和灵魂的存放。西德尼·史密斯（1771-1845）是英国僧侣和散文作家，以他的智慧、政治敏锐和反美主义主张而出名。梅尔本勋爵（1779-1848），是英国首相和维多利亚女王十分信赖的顾问。康熙（1654-1722）系中国皇帝，在他统治期间，曾大力提倡学识和艺术。贝尼托·佩雷斯·加尔多斯（1843-1920），西班牙小说家，其文学成就被许多评论家认为仅次于塞万提斯。保罗·维洛内塞（1528-1588）是意大利装饰画画家。大型的历史和宗教场面的绘画使他蜚声世界。
