15个相信占星术的名人

　　1．弗兰西斯·培根（1561-1626）　宝瓶座
　　在法国皇室占星术家的影响下，科学家培根本人也从事起占星术，他提倡"净化"占星术科学，但在晚年时他放弃了这种观测星座的做法。

　　2．查尔斯·戴高乐（1890-1900）　人马座
　　第二次世界大战前，当戴高乐仅是一个法军上尉军官时，他所做的星占中有一个预言他将成为法兰西的统治者。在他的一生中，他曾数次请教占星术家和手相师。

　　3．伊丽莎白女王一世（1533-1603）　室女座
　　伊丽莎白有一个专门为她算命的占星家，这个名叫约翰·迪的博士，占星时总是用一只水晶球。为了奖励他对女王的服务，女王后来任命他为伦敦圣保罗大教堂主管。

　　4．伊恩·弗莱明（1908-1964）　双子座
　　这位詹姆士·邦德（007）的创造者在公共场合谴责这种神秘的学说。但早在第二次世界大战他服务于情报机构时就开始让人替他占星算命。弗莱明很关心自己的死期，但占星家们都没告诉他。

　　5．伽里略·伽里雷（1564-1642）　宝瓶座
　　伽里略也从事占星术，但他的预测总是不准。1609年他为托斯卡尼大公爵制作了一份星相图，他预言公爵会长寿，可几星期后公爵就死去了。

　　6．卡尔·荣格（1875-1961）　狮子座
　　卡尔·荣格发现占星术对于研究他的精神病病人很有帮助，为了证实他的理论，他仔细研究了483对夫妇的星相。

　　7．约翰尼斯·开普勒（1571-1630）　摩羯座
　　作为鲁道夫二世的帝国数学家，开普勒的职责包括为神圣的罗马帝国制作天宫算命图。他认为从事占星活动同他从事的其他那些科学性更强的研究并无矛盾，并说：“占星学……是大自然给予天文学的一个附属学科。”

　　8．马丁·路德（1483-1546）　天蝎座
　　在约翰尼斯·利希滕伯格所著的一本占星术书的前言中，路德写道：占星学是对无神论者的一个警告。路德经常同他的神学朋友菲利普·梅兰克森讨论占星的学问。路德死后，一个罗马教廷的占星家根据星相推测他死亡的确切日期。

　　9．玛丽·安托瓦内特（1755-1793）　天蝎座
　　这位奥地利出生的法国王后在凡尔赛宫廷拥有一个占星家，在安托瓦内特出生时，占星师曾预言会有灾祸，因此她以后每次生日都不举行庆祝活动。

　　10．萨默塞特·毛姆（1874-1965）　宝瓶座
　　毛姆在他法国里维耶拉的家中招待占星家，并请他们算命。算命后，毛姆除了请他们吃一餐饭外，没有付一分钱，这使得占星师们非常恼火。

　　11．亨利·米勒（1891-）　摩羯座
　　米勒在他的作品中多次列举了占星术具有象征主义的事例。在西德尼·奥马尔所著《亨利·米勒：他的天体世界》的前言中，米勒指出占星学表明"宇宙有一种律动的模式。”

　　12．J．P．摩根（1837-1913）　白羊座
　　摩根每个月都要就生意上的事务请教他的占星师伊万杰琳·亚当斯——据说是约稍·昆西·亚当斯的后裔。除了摩根·亚当斯小姐的顾客还包括国王爱德华七世，恩利科·卡鲁索和玛丽·匹克福德。

　　13．西奥多·罗斯福（1858-1919）　天蝎座
　　罗斯福是占星学的忠实信徒，他把自己的生辰算命图放在一个棋盘上，然后每天对它加以研究。对占星术的术语也很熟悉，他最关心自己的所谓月亮同火星相克的星相。

　　14．伊迪丝·西特韦尔（1887-1964）　室女座
　　这名英国诗人虽然请占星师替她算命，可她从不把自己的真实的生日告诉他们，她把自己分别说成是属于好几种星座，因此人们怀疑她是否得到过正确的星占。

　　15．西尔维斯特·斯泰隆（1947-）　巨蟹座
　　斯泰隆的母亲是个占星家，她曾预言斯泰隆会成为一个作家。斯泰隆非常相信占星术，为了使自己将来的孩子有一个好的星相，他还同妻子塞奇商量好怀孕的日期。有一个占星师曾让斯泰隆特别当心那些重型的机械，不久以后，他在一家健身房刚锻炼好身体，一个重300磅的器械倒在他的身上，压伤了他的一块肌肉。
