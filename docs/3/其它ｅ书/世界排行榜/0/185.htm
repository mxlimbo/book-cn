9个过去曾经是或现在依然是牙医的名人

　　1．乔治・比尔斯（1825-1903）
　　1867年他第一个制定了一整套曲棍球比赛的规则。

　　2．埃德加・布坎南（1902-1979）
　　这位美国性格演员曾在1963年至1969连续放映的电视连续剧《衬裙》中担任主角。

　　3．赞恩・格雷（1875-1939）
　　这位美国小说家在进入牙医学校前，曾是一名"流动拔牙师"，专门到各地小城镇给人拔牙看病，与此同时他还在巴尔的摩的一个棒球队打球。1986年他在纽约建立了自己第一个牙科诊疗所，但到1904年他AE？医从文，全身心地投入文学创作。他的最受欢迎的西部小说是《紫艾灌丛中的AE？士们》。

　　4．道克・霍利迪（1851-1887）
　　霍利迪是西部时间一个臭名昭著的枪手，他最初在佐治亚州的亚特兰大行医，后来到格利芬。他患有肺结核，1873年他将诊所迁至得州的达拉斯，希望南方的气候会有助于他自己疾病的治疗。霍利迪长AE？咳嗽，因而吓跑了许多病人，1875年他第一次被逮捕，从此他的行医生涯就此结束。

　　5．爱伦・琼斯（1907-　）
　　这位美国歌星曾在《歌剧院之夜》，《水上舞台》及《当乔尼回家时》等影其中担任角色。

　　6．马龙・卢米斯（1826-1886）
　　米卢斯是无线电报的先驱，1868年他把世界上第一个无线信号发到18英哩以外的地方，比马可尼发明电报整整早27年。1872年他获得了发明专利，但由于缺乏资金而没能对发明作进一步的改进。

　　7．卡利・米德尔考夫
　　职业高尔夫运动员。米德尔考夫1949年赢得美国公开赛冠军，1955年获得马士达高尔夫锦标赛的冠军。

　　8．保尔・里维尔（1735-1318）
　　1775年他骑马从查尔斯镇飞速赶到莱克星顿，向美国殖民军通报英军即将来临的消息。这一功绩被诗人朗费罗写入他的著名诗篇《保尔・里维尔的功绩》。独立革命前他在波士顿行医时，曾于1768年在当地一家报纸上刊登广告，招徕病人。独立革命后，他弃医从商，投入银器制造业。

　　9．托马斯・韦尔奇
　　他发明了不经发酵的瓶装葡萄汁并建立韦尔奇葡萄汁公司。
