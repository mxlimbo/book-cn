6位不用（或很少用）标点符号的作家

　　1．杰西·安德泽约斯基（1909-　）
　　他是一位多产的波兰作家，早期作品深受天主教影响，50年代初他转向共产主义，但3年后又放弃了这一信仰。1962年始，他开始了新的写作尝试，即不用标点符号。这一年他的小说《天堂之门》问世，16年后该小说被译成英语，在伦敦出版。小说描写十字军东征背后的种种动机，整部小说为一个句子，前40000个词没有用一个标点符号。

　　2．纪洛姆·阿波里耐（1880-1918）
　　他是一位意大利军官的非婚生子，出生后被父母遗弃，他是在法国利维拉省的教会学校中长大的。后来他到了巴黎，当过教师、教士、捉刀代笔人，还写过色情小说。最后他成了文学批评家、小说家和诗人。他是未来派和立体派的创始人，并创造了"超现实主义"这个术语。他发现了原始派画家亨利·卢索，毕加肃、布拉克、马提西、弗拉明克和杜菲等画家也都是他的好友。1913年他的诗集《醇酒集》使他一举成名，这部作品激怒了许多批评家，因为里面没有标点符号。对此，阿波里耐自己是这样解释的：“我摈弃标点符号，因为我觉得它们毫无用处，事实也确实如此，诗的节奏和诗行就是真正的标点，除此之外，什么也不需要了。”

　　3．提摩西·德克斯特（1747-1806）
　　德克斯特在从事写作之前做过商人，并因此发了大财。他在麻萨诸塞州的纽卡斯尔买了一幢很大的邸宅，门前放着40个真人大小的木雕塑像，其中有尼尔逊、亚当、夏娃、华盛顿、路易十四，还有他自己。在邸宅中他供养了一位妻子，一个星相学家，一个大块头宫廷小丑，一位非洲公主的女管家，还有一位充当桂冠诗人的鱼贩子。1802年德克斯特开始写作，出版了一本24页的自传体哲学小册子，题为《智者的困惑》。在这本书中，德克斯特竭力为自己的生活方式辩护，他描写了自己发财的过程，并暗示他可以成为美国的好皇帝。这本书的一个显著特点是它只包含一个句子，或者说不包含任何句子。书中没有任何标点符号，从语法上说，这个句子没有开头也没有结束。它就象时间和空间一样，来自无限，并归于无限。人们对这一"文学瑰宝"的反应使德克斯特意识到一部没有标点符号的书实在算不上一本书。在这本书的第二版中，他增加了一页，这一页中有整整13行全是标点符号—-一行又一行的分号、逗号、问号、句号。德克斯特说：读者们可以"随心所欲"地自行在他书中的字里行间"撒点胡椒和盐"。

　　4．约翰·多布逊牧师（1794-1847）
　　多布逊牧师以数学见长，也从事文学写作，但他是以憎恨标点符号而出名的。1815年，他的两卷本《几何原理》出版了，全书除了每段末尾用了一个句号外，没有任何标点符号。不过这本书的完整性被出版商偷偷地破坏了，他在第一页中加上了一个分号、一个冒号、一个逗号和一个句号。

　　5．哥楚特·斯泰因（1874-1946）
　　这位宾夕法尼亚出生的先锋派女作家后来成了巴黎最著名的美籍作家之一。毕加索和海明威都是她的好友，她的沙龙和艺术收藏品是举世闻名的。她的地位是由《三个女人的一生》和《艾丽斯·B·托克拉斯的自传》奠定的。她的风格——简单化与重复——独具匠心，她自认这种风格在詹姆士·乔伊斯之上。但最突出的是她藐视标点符号。斯泰因认为“句号有着自己的生命"，所以她只用句号。有时她会不留神写下一个逗号，不过她十分厌恶逗号，认为它"具有奴性"，使得作品读起来过份容易。她认为问号和感叹号"具有鲜明的背叛性"。斯泰因最出名的字行是：“玫瑰就是玫瑰就是玫瑰就是玫瑰。"她终生未婚，与一位叫艾丽斯·B·托克拉斯的女子过着同性恋生活。

　　6．E·卡明斯
　　卡明斯是波士顿人，他既是诗人，又是画家，他在文坛赢得一个令人不安的外号：“可怕的孩童"。他随心所欲地使用标点符号、语法和排版，最初令评论家"大为迷惑"，他们指责他"想入非非"，不过，后来他成了一个出名的诗人。卡明斯憎恶句号和逗号，但他喜欢括弧和连接号。他有些诗中没有一个标点，这种奇特的风格吸引了一些模仿者，不过他们缺乏卡明斯诗中独特的韵味。
