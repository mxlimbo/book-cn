安德鲁・韦思眼中美国历史上8件最佳的绘画作品

　　当韦思被问及为何他的作品能名闻世界时，他回答道，“这是因为我画的东西正好反映了生活的基本特征：天空，大地，朋友及其他熟悉的事物。"他的油画和水彩画中栩栩如生的人物和优美的风景使他跻身于当代世界上最杰出的画家行列。目前他同妻子贝茜居住在她宾州的老家。
　　1．温斯洛・霍姆的《猎狐》（1893）
　　2．费兹・莱恩的《河谷》（1863）
　　3．查尔斯・皮尔的《乔治・华盛顿上校》（1772）
　　4．托马斯・伊金的《狩猎》（1874）
　　5．阿尔伯特・赖德的《跑道》（1910）
　　6．格兰特・伍德的《革命的女儿》（1932）
　　7．埃德伍特・希克斯的《诺亚方舟》（1846）
　　8．乔治・宾厄姆的《期货商访问密苏里》（1845）
