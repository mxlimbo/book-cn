二十世纪发生的12起艺术骚乱事件

　　1．伯爵夫人参加哑剧《埃及之梦》的首场演出
　　1907年1月3日，巴黎。
　　这个哑剧是以法老时代的埃及为背景。剧中女主角的扮演者是歌舞厅女演员科莉特，男主角则正好由科莉特的朋友和情人，伯爵夫人莫尼扮演。当剧中的男女几经曲折终于重逢时，两人紧紧地拥抱，并长时间地接吻，这时，全身几乎一丝不挂的科莉特表现出超乎寻常的激情。在场的观众，包括伯爵本人都看得目瞪口呆。当帷幕降落时，愤怒的观众大叫大闹，向舞台乱扔东西，整个剧场乱作一团。

　　2．阿诺尔德·舍恩伯格的首场演出
　　1912年的舍恩伯格虽然尚未创立出他的12音调体系，但他的作品已经发展成一种新型的音乐，特点是风格严肃，形式简洁，并且曲调忧郁，不加装饰和渲染。但是他的这种不同于传统的音乐作品，特别是1912年公演的《卢耐尔》引起了一片反对。人们批评，辱骂，甚至发出歇斯底里的嘲笑。一个批评家说：“如果这算是音乐，我要恳求上帝别让我再听音乐了。"甚至在几年后，仍有人指责这部作品。当一个人因袭击他人而受到指控时，出庭作证的医生说，此人是因为几年前听了舍恩伯格的令人头痛的音乐而变得神经失常的。

　　3．纽约国际现代艺术展
　　1913年2-3月，纽约。
　　1913年在纽约举办了一次规模空前的艺术作品展，参加展出的有来自美国和欧洲各国的现代画共计1600多件。其中有毕加索、马蒂斯和杜尚等欧洲大师的作品及斯隆、马林和AE？伦德加斯特等美国画家的作品。由于美国观众一时还未能适应现代派艺术在表现视觉艺术时的大胆风格，因此不少人表现出强烈的不满和敌对情绪，指责声、嘲笑声不绝于耳，甚至有一批激动的观众扬言要撕毁杜尚的作品《走下楼梯的裸女》。尽管如此，这次展览仍然获得不少的支持者，取得圆满的成功。

　　4．斯特拉文斯基的乐曲《春天的礼仪》和尼金斯基的芭蕾的首场演出
　　1913年5月29日，巴黎香榭利大剧院。
　　那晚的音乐是一种具有全新风格的音乐，以致一些观众认为这是音乐的无政府主义。那晚的舞蹈，尼金斯基表演得那么大胆，以致许多上流社会的芭蕾爱好者们认为超越了传统的道德和审美的界限。这两者，音乐和舞蹈加在一起就使得大部分观众骚动起来。起先剧场内不时地响起口哨声和喝倒彩的声音，一些支持者们则要求人们保持安静，接着过道里出现了斗殴，警察将肇事者拉出剧场，没过一会，大规模的骚乱再次爆发，并一直延续到演出结束。几年后，作曲家兼指挥皮埃尔·波莱兹说这部音乐作品是"现代音乐的奠基石"。

　　5．弗厘兹·克莱斯勒的小提琴独奏会
　　1913年12月，康奈尔大学。
　　这名奥地利著名音乐家，第一次世界大战期间曾在奥军服役，并受过伤。1919年他前往美国作巡回演出。可当时的美国人民战争情绪仍很高涨，因此他的演出受到强烈的反对，其中有好几次演出就因抗议组织的干预而被迫取消。那年12月，也就是第一次世界大战停战1年多以后，他到康奈尔大学举行独奏音乐会。演出中一批美国军团的退伍军人撞入剧场，将电源切断，场内一片混乱。但克莱斯勒仍继续演奏，在黑暗中，他坚持演奏了整整40分钟。剧场内捣乱活动仍不断升级，最后不得不把警察叫来维持秩序。

　　6．达达主义派表演
　　1920年5月26日，巴黎。
　　达达主义是以反对资产阶级的价值观、挑动观众进行骚乱性的抗议活动而闻名。他们全力以赴地举办这次演出。舞台上，演员们身穿各种奇特的服装，吟诵他们的诗歌，发表宣言，展示绘画作品。布莱顿在他的脑袋的两边各挂一支手枪，埃鲁阿特打扮得象个芭蕾舞女演员，其他一些人则在头顶上套着玻璃瓶或漏斗。所有这些装束和表演的内容都着意抨击资产阶级的艺术、哲学、伦理及其他一切资产阶级所视为神圣的东西，许多人说这次演出将巴黎达达主义的反艺术运动推向了高潮。可场内的观众对这样的演出十分厌恶，他们忿怒地大声咒骂，并不断地向台上的演员扔西红柿、鸡蛋和牛排。当然达达主义者仍声称这次演出大获成功。

　　7．乔治·安塞尔的钢琴演出
　　1923年，布达佩斯。
　　作为一名反浪漫主义、反表现主义的先锋派作曲家，安塞尔常常不得不面对许多充满敌意和歇斯底里的观众。为了安全起见，他每次演出都随身带一把手枪。他在布达佩斯作首场演出时，观众们无法理解和接受他那些刺耳和生疏的音符，剧场内爆发一阵阵的骚动。第二天演出时，安塞尔决心要让观众耐心地听完他的演奏，他命令剧场工作人员把所有的门都关上，然后在众目睽睽之下，他从口袋里掏出手枪，并把它放在钢琴板上，结果那场演出顺利地进行到结束。

　　8．西恩·奥卡西的《犁和星星》的第2场演出
　　1926年2月11日，都柏林阿贝剧场。
　　这部由奥卡西创作的舞台剧反映的是1916年复活节发生在爱尔兰的一场反抗美国政府的起义斗争。在演到第二场时，演员们把一面国旗竖在台上，以此当作一家酒店，可观众中一些爱国者对此大为不满，认为这是对在这场自由爱尔兰斗争中牺牲的烈士们的莫大侮辱。剧场内立刻骚动起来，观众中持有不同意见的人们开始互相拳打脚踢。这时剧场的经理威廉·耶茨立即把警察叫来维持秩序。事后，有人扬言要炸毁这座剧场。以后该剧每次上演时，总有警察到场，因此除了偶尔爆炸过几只臭气炸弹外，没有再发生任何其他的捣乱活动。

　　9．巴纽尔和达利的电影《我们的时代》的初次上演
　　1930年12月，巴黎。
　　这是一部充斥着暴力和色情镜头的超现实主义影片。该片反映了人类的虚伪和邪恶，有力地讽刺了资产阶级社会的传统和体制。当然资产阶级社会对此十分不满，一家报纸称这部影片是"淫秽、令人厌恶和毫无价值"，另一家报纸则声称"国家、家庭和宗教都被涂上了污泥"。一家极右的报纸甚至发表文章，煽动青年闹事。紧接着一批极右思想的青年对放映这部电影的剧场连续6天发起袭击，结果造成价值12万法郎的损失。最后由于反对的声浪日益高涨，这部电影不得不停止公演。停演的时间长达35年之久。

　　10．李维拉的壁画揭幕
　　1948年6月，墨西哥城，德普拉多旅馆。
　　1948年6月李维拉完成了他画在德普拉多旅馆餐厅墙上的一幅壁画。画中墨西哥著名的历史学家拉米莱兹正捧着一本打开的书，书上清晰地写着"上帝并不存在"这几个字。当马丁尼兹大主教得知后，他拒绝给这幢政府管辖的旅馆举行宗教仪式。一批愤怒的青年冲进旅馆的餐厅，用小刀将画上的这几个字刮掉，李维拉又用钢笔重新写上。当地的青年扬言，只要这几个字一出现，他们就要把它们涂掉。旅馆不得不把壁画盖上。尽管这场争执尚未有明确的结论，李维拉本人却遭受到很大的不幸。例如一次他去看电影时，电影院不让他进去，而且没过多久，他的家也遭受到暴徒的破坏。最后，一名不愿披露姓名的牧师总算为旅馆举行了宗教仪式。

　　11．伦尼·布鲁斯的演讲
　　1962年，伦敦。
　　布鲁斯曾同伦敦一家专供自由人士演讲的卡莱巴酒吧签订合同，由他每天前往发表演说。这位毫无约束的社会讽刺家就性、色情文化和吸毒等一系列社会现象发表他自己的看法。听众们对他那些异端邪说十分不满，有的人大声叫喊，有的人中途退场，有的则以拳相见。在场的俄国诗人叶甫图申科和英国剧作家也感到听不下去，混乱中一个怒气冲冲的女演员对准酒吧经理的鼻子猛击了一拳。不久布鲁斯被英国政府宣布为不受欢迎的人、并被遣送出国。后来他还被指控犯有淫秽罪并受到起诉。

　　12．未经批准的现代画展
　　1974年9月15日，莫斯科。
　　在苏联，除了正统的社会现实主义作品外，其他任何流派的艺术品都不得展出。一批艺术家们只好躲到莫斯科郊外，在一块泥泞的田野里搭起临时画廊，展出他们大胆创新的绘画。当苏联警方得知后，便开动推土机和高压水车，撞入展览场地，吓得参加画展的观众和画家们四处逃窜，混在人群中的便衣警察乘机把画作扔到地上用脚踩或放火焚烧。在场的30名外国外交官目睹了艺术家们遭到殴打，记者们遭到推拉。连美国参赞威勒姆斯也被粗暴地挤来挤去。军警们还为自己的行为辩护，说开推土机是为了在此修建一座"文化公园"，两个星期后，为了安抚美国和世界舆论，苏联政府总算同意正式举行一次同样的画展。
