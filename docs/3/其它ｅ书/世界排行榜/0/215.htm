由于饮酒而发生的12起著名的事件

　　1．基德上尉的绞刑（1701）
　　基德上尉因犯谋杀和海盗罪被判处死刑。1701年5月23日他被推上伦敦的绞刑架。当时一大批基德的崇拜者拥入刑场大声为他唱赞歌。一个喝得酩酊大醉的刽子手正准备给基德套上绞索时，看到这个场面紧张得几乎站不住脚。基德刚被吊起，突然套在脖子上的绞索断了。基德立即摔落在泥地上，第二次再把他吊起，总算成功。但第二天出版的报纸严厉地批评了行政当局把绞刑执行得如此糟糕。

　　2．波士顿茶叶党（1773）
　　1773年12月6日50个殖民地居民和通信委员会的成员在麻省波士顿一个叫本杰明·伊兹的出版商的家中聚会。他们打算在那晚把停泊在波士顿港口的3条船上的茶叶全部销毁，以此作为对英国政府在美国征税的抗议。出发前，为了此行，伊兹拿出一只大碗，在碗里倒满他自己酿制的烈性朗姆酒，这些爱国者们就一碗又一碗地喝了起来，速度之快，使得在一旁负责斟酒的伊兹的儿子彼得忙得不亦乐乎。下午6点过后，大部分人都变得吵吵闹闹、情绪激动，其中有几个人甚至走路的步子也踩不稳，跌跌撞撞。接着他们离开伊兹家，前往格里芬码头。船就停在那里。在以后的3个小时里，当他们上船将茶叶箱往海里扔时，脑子才开始清醒起来，这时有的人感到肚子十分难受。

　　3．奈特·特纳的叛乱（1831）
　　1831年8月21日夜晚，黑奴先知奈特·特纳在佛吉尼亚州的南安普顿发动了一次叛乱，结果造成了50个白人和120多个黑人死亡。虽然特纳向来滴酒不沾，但他手下6个追随者那晚痛饮了苹果白兰地，还吃了大块烤肉。当他们攻下第一个庄园时，先打开酒窖，痛饮烈性的苹果酒，然后把白人全家杀死。从深夜一直到第二天，他们不断地袭击庄园，杀死白人，夺走马匹、枪支和白兰地酒，到中午特纳的队伍已扩大到将近60人，可其中许多人由于酒喝得太多，不时地从马上摔下来。当特纳赶到一支先遣部队时，发现手下人正在庄园的酒窖内酗酒。这时一批白人军队正赶来，特纳马上把人马集合起来，勇敢地同白人搏斗。结果把白人打得狼狈逃窜。但是第二天形势急转直下，很多新加入造反队的人都纷纷逃走，只剩下特纳和总共20个人马，面对3600多个全副武装的白人民兵，结果这场叛乱被镇压了下去。

　　4．安德鲁·约翰逊副总统的就职仪式（1865）
　　1864年作为林肯的竞选伙伴，约翰逊不停地到全国各地进行竞选活动。后来终因疲劳过度染上霍乱而躺下。1865年3月4日是举行他副总统就职仪式的日子，那天早晨他醒来时，感到自己无力下床。为了提起精神，他喝下了威士忌"药"酒。由于身体过分虚弱，因此他立刻醉了。当他开始醉熏熏地宣读就职讲稿时，结结巴巴，言语不清，并有数处读错，接着约翰逊又激动地脱稿讲了一阵，最后，工作人员打断了他，让他立即宣誓就职，然后最高法院的大法官把他带了下去。

　　5．林肯被杀（1865）
　　1865年4月14日下午3点，演员约翰·威尔基斯·布思在华盛顿区柯克任德酒吧开始饮酒。4点钟，他来到迪利酒店要了一瓶白兰地。两小时后他又到福特戏院隔壁的塔尔塔伏尔酒吧喝威士忌。谋杀的准备工作做完后，他于9时30分又重新回到塔尔塔伏尔酒吧。此时，总统的侍从查尔斯·福布斯，马车夫弗兰西斯·伯恩斯以及贴身保镖、贪杯的约翰·派克也在这个酒吧喝酒。10时15分派克继续喝酒——此时的总统周围无人保卫。布思离开酒店，进入隔壁的戏院，并开枪打死了林肯。布思的同谋乔治，阿兹劳特，本来打算刺杀安德鲁·约翰逊。由于喝得酩酊大醉，再加上心情害怕，不得不放弃了谋杀计划。

　　6．小比格霍思河战役（1876）
　　直到今天人们对当年在小比格霍思河战役中美国第七骑兵团的官兵们酒后吃败仗的情况仍有不同的说法。据说卡斯特的副司令马科斯·里诺将军随身带着一桶半加仑的威士忌酒。可能是4天前到达鲁斯巴德河时，官兵们从一艘装载威士忌酒的轮船上弄到不少酒，据参加这场战斗的印第安老兵们说，在卡斯特士兵的尸体上都发现带酒壶，而且壶里的酒都已喝掉一半。当时的情况是：卡斯特被击败的第二个晚上，里诺的军队已被包围，而这个酒徒指挥官此时已喝得酩酊大醉。

　　7．撕毁洛斯恩布多思条约（1886）
　　1886年3月27日，美国陆军乔治·克鲁克将军同阿柏支族印第安首领吉罗尼姆在墨西哥萨诺拉的洛斯恩布多思达成停战协议。根据协议，吉罗尼姆和手下人将向美军投降，然后由美军将他们遣送到亚利桑那州的圣卡洛斯保留地。可不幸就在协议签署的那天晚上，一个瑞士出生的美国酒贩向阿柏支人出售了一大批威士忌酒和麦斯克尔酒。当这些印第安勇士们喝得酩酊大醉时，他们就反对执行这个城下之盟。夜晚，激动和兴奋的吉罗尼姆宣布决不向白人投降，并拒绝执行协议。随后他在20个勇士的陪同下撞进茫茫的黑夜，同白人们又一次进行血战，5个月后他被白人抓住。

　　8．沙利文-基尔莱恩之战
　　1889年7月8日，当时无手套拳击冠军约翰·沙利文同杰克·基尔莱恩在密西西比的里奇堡举行一场冠军争夺战。这两人都是嗜酒如命，起初沙利文的老板不许沙利文饮酒，但交手43个回合后，为了克服疲劳和华氏112°的高温引起的不适，他又表示同意沙利文喝一点掺威士忌的冷茶。不幸酒刚下肚，他就感到非常难受，立刻在场上呕吐了起来。在交手到56个回合时，多次挨打的基尔莱恩也喝了一些波旁酒，这是一个叫威廉的计时员送给他的，以后每一个回合结束时，威廉就给他喝上几口，等到第75个回合基尔莱恩认输时，他已喝完整整一瓶。

　　9．埃纳河的第三次战役（1918）
　　1918年5月，由埃里希·鲁登道夫率领的德军部队到达巴黎37英里外位于马恩河边的蒂耶里堡，该部队准备投入第三次埃纳河战役。他们一路上占领了法国多个盛产香槟的省份，那儿的地窖都贮满了酒桶。这些数年滴酒未沾的士兵们，眼看巴黎即将被占，便放肆地喝啤酒来。整个部队甚至连军事警察们也加入了狂饮乱喝的行列。5月30日清晨，在费斯米思镇的街道上，横七竖八地躺满了醉得不省人事的士兵，以致于军车无法通过。德军中这次大规模酗酒大大延缓了部队的前进速度，有的地区甚至完全停滞不前。结果法军和美军赢得了宝贵的时间，重新建立防线，发起反攻，使得鲁登道夫的进攻计划以失败告终。以后战事的发展证明这是第一次世界大战中德军的最后一个可能获胜的机会。

　　10．嗜酒者互诫协会（1935）
　　该协会是由纽约一个名叫比尔·W（嗜酒者互诫协会的成员，一般只用姓的打头字母）的股票经纪人所创建。他本人原来也是一名嗜酒者，只是在经历了一次精神的打击后，他决定不再饮酒，他还成功地帮助一名叫鲍勃的医生戒掉了喝酒的习惯，比尔一次出差到俄亥俄州的阿克龙，遇见鲍勃医生，两人便谈及了各自的嗜酒经历，比尔向他介绍了自己是如何成功地把酒戒掉。此时的鲍勃酒精中毒已经很深，且终日喝得醉熏熏，头脑昏昏沉沉。在比尔的帮助下，他于1935年6月10日过了最后一次酒瘾，第二天他们便组建了现在称为"嗜酒者互诫协会"的组织。从此他们没有再喝过一滴酒，比尔活到1971年，鲍勃则于1950年去世。这个以嗜酒者互相帮助诫酒为宗旨的组织如今已拥有数百万个成员。

　　11．电影《小山雀》的摄制（1940）
　　同他拍所有其他电影一样，费尔茨在拍《小山雀》的整个过程中，总是略带醉意地走进摄影棚。每天早晨2至4杯马提尼酒落肚，费尔茨带上一只盛满马提尼酒的鸡尾酒瓶子前往宇宙制片厂。在略带醉意的情况下，他的喜剧性表演似乎更为出色。在拍摄《小山雀》期间，他每天要喝掉两瓶金酒，但费尔茨这类酒后的兴奋表演常使他的搭档梅韦斯特感到恼火。特别是有一次，费尔茨一时激动，对梅韦斯特表现出过分的亲昵，他甚至伸手拧了一下梅韦斯特丰满的身躯，并称呼她为"我心爱的小母马"，弄得梅韦斯特哭笑不得。费尔茨虽然常常吃过午饭要打个盹来醒醒酒，但他从未因为喝酒而在影其中表演失常。

　　12．创作小说《装有发条的桔子》
　　英国作者安东尼·伯吉斯写的这部小说尽管也有对未来世界的描写，但主要情节则是以作者本人的亲身经历为基础的。例如：小说中写到的强奸的一个场面就是以第二次世界大战中作者的妻子受侮辱的事实为素材。那件事件还造成作者未出世的孩子惨死在娘胎里。作者在创作这部小说时，情绪无比激动，他不得不一再喝酒来使自己平静下来。他承认道：“为了能叙述这些令我痛苦不堪的事实，我只能借助于酒，我是在醉意中创作出这部小说的。”
