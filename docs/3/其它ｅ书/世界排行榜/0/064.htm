8次不寻常的海豚事件

　　1．海豚当护士
　　1954年10月3日，一群科学家正在佛罗里达附近水域进行炸药的水下爆炸试验，在爆炸点不远有一群鼻子如同瓶子一般的海豚。其中有一条被爆炸震晕了，以45°的倾斜在水里漂浮着。这时，另外两条海豚游了过来，扶着它两边的鳍，使它浮在水面。整群的海豚都围在这条受伤的海豚周围，直至它恢复知觉后，这些海豚才四散游开。

　　2．海豚——游戏伴侣
　　1955年，新西兰奥波诺尼港出现了一条1岁的雌海豚，在此后的整整一年里，这条海豚一直和大人、孩子一起游戏，它甚至让孩子们坐在它的背上。城里的市民们称它"奥波"，并且通过一项法律，禁止在港口内捕杀海豚。然而就在法律生效的那天，人们在海滩上发现了奥波的尸体。显然，那是渔民们用炸药炸鱼时，把奥波震昏了，随后它被潮水推上海滩窒息而死。奥波诺尼的市民们埋葬了它，并在坟墓上撒满鲜花。

　　3．海豚——保镖
　　有一位叫伊冯·弗拉迪丝拉维奇的南非妇女，于1971年6月在一次游艇爆炸事故中掉进印度洋。不多时，3条海豚朝她游来，显然在保护她不受鲨鱼的攻击。后来，她随波漂到一个航标处，并爬上了航标，这时海豚才离开她。据救援人员说，她在水中漂了200英里，在整个过程中，海豚一直守卫着她。

　　4．海豚——富有同情心的一族
　　根据苏联塔斯社的一项报道，1977年9月，一艘苏联渔船在黑海遇到一群海豚，它们围着渔船游个不停，直到船员们起锚跟随它们行驶。这群海豚将渔民带到一个浮标处，那里有一条小海豚不慎卡在上面无法脱身。当渔民将这条海豚放回水里后，那群海豚又将渔船引回原先的地方。

　　5．海豚当向导
　　1978年5月28日，4位渔民在南非达森岛附近海域，因大雾弥漫而迷失了方向。4条海豚出现了，它们围着渔船游，直到渔船改变方向——这真是千钧一发，因为前面就是一片暗礁，它们突出在水下，水面上根本看不见。这几条海豚推搡着渔船，直至它行驶到安全的水域。到了那里，它们拦住渔船，船员们就抛了锚，这时海豚才离开。当迷雾散去之后，渔民发现，这里正是他们早先出发时的海湾。

　　6．海豚——保护神
　　苏联塔斯社报道，1978年8月14日，渔船"尼弗尔科依号"上的水手听见水下有海狮发出的救命叫声。原来这只海狮被一群凶猛的鲸鱼团团围住了。几分钟之后，一群海豚出现了，这些鲸鱼就离开了。当海豚离去时，鲸鱼又转身朝海狮冲去。这时海豚也迅速调转身子，飞出水面，跃过鲸鱼头顶，在海狮四周围成一圈，海狮得救了。

　　7．海豚——好向导
　　1978年9月30日，50条鲸鱼眼看着就要在新西兰的奥克兰港附近冲滩。市政官员千方百计想把它们重新引入海中，在这紧急关头，一位官员想出了一条妙计。他驾驶一艘汽船将经过港口的一群海豚引到鲸鱼冲滩处。海豚一到那里，情形立刻有了转机：鲸鱼在海豚的指引下，乖乖地朝大海游去。人们最后见到它们时，它们离海岸已有15英里。由于海豚及时赶到，这群鲸鱼得救了。

　　8．海豚患者
　　据塔斯社报道，1979年1月，一条海豚试图跳上一艘苏联渔船，但是接连几次都未能成功，后来迷惑不解的船员帮助它上了船，这时才发现，它身上有一道长长的口子，鲜血直淌。船上的医生立刻对伤口进行了处理，在此过程中，海豚躺着一动不动，治疗完毕后，船员们又把海豚重新放回海里。
