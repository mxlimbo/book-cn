5个著名同性恋者的5个真正的妻子

　　1．保尔·魏尔兰太太（马蒂尔德·莫泰）
　　马蒂尔德·莫泰16岁时同25岁的诗人保尔·魏尔兰相遇。1870年他们在巴黎结婚，当时马蒂尔德年仅17岁，她看上去还像个孩子，结婚对她来说似乎就是真的玩做家庭主妇的游戏。新婚的丈夫则期待着终生的幸福生活，可是到1884年，当经过修改后的法律允许离婚时，同丈夫长期分居的马蒂尔德成为第一批要求摆脱婚姻枷锁的妇女。他们之间究竟发生了什么？天真的、出生于市民家庭的马蒂尔德婚后不久便发现丈夫具有汹酒和动手打人的恶习，他常常莫名片妙地在外过夜，这既让她感到轻松自在，又使她产生一种担忧，就在他们唯一的孩子乔治出生前一个星期，魏尔兰将马蒂尔德从床上打到地上，在乔治3个月时，他甚至把儿子往墙上扔。一次，魏尔兰曾试图烧掉妻子的头发，结果没成功。他变得不修边幅，有时甚至整个星期不洗澡，不换衣服，这时他在恋爱——是他一生中第一次真正的恋爱。他爱上了一个名叫阿瑟·兰波的农村少年。兰波年仅17岁，但在诗歌创作方面具有才华，由于他给于魏尔兰一种从未感受过的肉体快乐，便使后者对他百依百顺。魏尔兰让他住到自己家里，兰波不仅弄得魏尔兰神魂颠倒，还偷窃了马蒂尔德家中珍藏多年的象牙十字架。1872年魏尔兰丢下了妻子和孩子，同兰波私奔，但这两个衣服肮脏、蓬头垢面的同性恋人之间的关系也远非美妙，不久兰波另求新欢，魏尔兰妒火中烧，拔枪朝他射去。兰波虽然中弹，但伤势不重。魏尔兰被判处2年徒刑。兰波这个15岁就才华横溢的青年，此刻年仅20岁，却已才华耗尽。他放弃了诗歌创作也抛弃了同性恋的恶习，他当上了一个旅行商人，在埃塞俄比亚同当地一个妇女同居。
　　返回法国后，他试图找一个妻子，可一场大病让他失去了右腿，1891年，年仅33岁的兰波去世。魏尔兰那个可怜的儿子乔治后来曾当过一段时间大巴黎市的邮政局长，因饮酒过度，于1926年去世。至于马蒂尔德，33岁时她重新结婚，嫁给了一个建筑承包商，并生了两个孩子，后来这第二个丈夫同她离婚。魏尔兰出狱后精神稍有恢复，便继续进行诗歌创作。1896年，52岁的魏尔兰在一个富有同情心的朋友名叫克朗兹的妓女的寓所里去世。

　　2．彼得·伊里奇·柴可夫斯基太太（安东尼娜·伊凡诺夫娜·米留可娃）
　　安东尼娜·伊凡诺夫娜·米留可娃患有一种近似情欲狂的精神变态疾病，1877年她28岁时爱上了她的音乐教师。37岁的彼得·伊里奇·柴可夫斯基。可柴可夫斯基当时十分痛苦，因为他有同性恋倾向（他曾疯狂地爱着他姐姐的儿子"鲍普"），他很恨自己，也很害怕别人会发现自己的这种毛病。他曾一再劝安东尼娜放弃这个念头，特别是因为自己偏爱男性，无法满足女性的需要，再加上对方又是性欲狂，两者结合必定是悲剧。可伊凡诺夫娜·米留可娃仍坚持要嫁给他，直到她用自杀相威胁时，柴可夫斯基不得不屈服。可这宗从一开始就不完美的婚姻使这位作曲家经历了恶梦般的遭遇。柴可夫斯基14岁丧母，可他依旧病态般地迷恋着对母亲的回忆，他曾同歌剧演员迪斯利·阿托特有过一段短暂的、然而结局很糟的恋情。之后，他又受到纳达茜特·梅克女士的爱慕，梅克女士是个寡妇，比他大10岁，常同他通信，并在经济上给他资助，可她从不要求同他会面。柴可夫斯基十分陶醉于这种天使般纯洁的友情，同安东尼娜结婚后，面对妻子病态的狂热情欲，柴可夫斯基痛苦万分，在一个寒冷的夜晚，他甚至走入一条齐腰深的河里，祈祷冰冷的河水会给自己带来肺炎，夺走自己的生命，可结果他既没得肺炎也没冻死，他只好逃到哥哥阿纳托尔家避难，在那里他整整昏迷了两天两夜。面对丈夫的抛弃，安东尼娜在同一连串男人的交往中也得到了安慰。她后来生了一连串的孩子，数目不详。1917年她死于疯人院，比柴可夫斯基多活了整整24年。柴可夫斯基死于霍乱，当时霍乱流行，他不顾一切地喝了受污染的水，结果染上了疾病。

　　3．奥斯加·王尔德太太（康斯坦丝·芬埃德）
　　1881年，在一次晚会上他俩一见钟情，从此便堕入情网。两年后，他们在英国派丁顿的圣詹姆士教堂举行婚礼。在婚前恋爱阶段，王尔德为了从伦敦赶到都柏林同心上人匆匆地耽上几个小时，常常任意打乱自己外出讲课的计划。"我们每隔两天互相通一次电报"，王尔德沾沾自喜地说道。王尔德给她的信常常使她欣喜若狂，她写道：“我的整个生命就是你的，你可以利用它，就像利用自己的生命一样。"但日后王尔德给这位平静美丽的爱尔兰姑娘所带来的只是悲剧。刚结婚时一切都很美好、蜜月是在巴黎度过的，记得他们首次同房后的第二天早晨，王尔德正好同一个朋友在巴黎的街头散步，他突然停下，买了鲜花，并写上美丽的词句让人送到旅馆里新娘的床边，他还眉飞色舞地向朋友们详细叙述了前一天夜里他们作爱的经过。在伦敦，他们搬进了一座精致的房屋，为了使王尔德高兴，康斯坦丝特意穿上了同室内布置相配的古典服装，当王尔德在客人面前讲一些有关婚姻的俏皮话如："在婚姻生活中，三人成伴，两人不欢"以及"婚姻是恋爱的坟墓"等时，她也总报以附合的微笑。1885年她生了个儿子，取名西里尔。1886年第二个儿子出世，取名维维安。康斯坦丝第一次听到有关王尔德在外鬼混的消息是传说他迷上了一个女演员，还经常去一个高级妓女的寓所。她痛苦地容忍了这一切，就如她容忍了丈夫对她笃信宗教的讽刺和挖苦，然而她无法容忍的是王尔德竟然同年轻的阿尔弗雷德·道格拉斯勋爵发展同性恋的关系。这两个人公开地炫耀他们的恋情，毫无疑问，道格拉斯的父亲昆斯伯里侯爵对此大为恼火，他指责王尔德犯鸡奸罪，可王尔德无视朋友的劝告，错误地认为他的名声足以使他抵挡一切，竟然以诽谤罪反向侯爵提出起诉。但让王尔德惊讶的是调查结果显示道格拉斯父亲的指控证据充足，因而他自己反遭逮捕，并被判处两年苦役。王尔德的丑闻公开后，康斯坦丝为了避开公众的注意，就把她自己和孩子的姓都改掉，换了一个荷兰古老家族的姓。西里尔和维维安都被送到瑞士受教育，她自己也离开英国去欧洲旅行。当王尔德母亲去世时，善良的康斯坦丝虽然自己十分痛苦，仍特意从意大利的热那亚赶到里丁监狱，把这消息告诉狱中的王尔德。但比这更让王尔德感到痛苦的是两个儿子都离他而去，他曾写道：“法院认定我不适宜同孩子一平生活，这对我来说实在太可怕了。"出狱后，他曾同阿尔弗雷德·道格拉斯一起在意大利度过了一个短暂的假期，然后他们就永远地分手了。1898年康斯坦丝在热那亚接受了一次脊椎手术，不久就在一家私人医院去世。王尔德活到1900年，他一直住在巴黎艺术街上一座破旧的旅社，今天这个地方已是阔佬们居住的一家豪华饭店，饭店的大门上挂着一块刻有王尔德名字的金属牌。他死后被安葬在巴黎的拉雪兹公墓，同那些艺术大师们放在一起，如：阿培拉德，巴尔扎克，科莱特·肖邦，伯恩哈特等，在他的墓边安放着他自己的雕像，是由雅克·爱泼斯坦制作的，王尔德要是知道这一切，他会高兴的。

　　4．W·H·奥登太太　　　 （爱利卡·曼）
　　1935年诺贝尔文学奖获得者托马斯·曼的女儿爱利卡·曼同英国杰出的诗人魏斯坦·奥登结婚。这宗双方都同意的婚姻很不完美，却又令人满意。结婚的那天他们俩是第一次见面，爱利卡30岁时同一个德国演员离婚，奥登比爱利卡小两岁，是个谨慎的同性恋者，他俩的结合是奥登过去的情人和现在的朋友克里斯托弗·伊舍任德促成的，爱利卡当时从德国逃出，属政治难民，她需要英国护照以便能继续从事写作和教学工作。假如她能嫁给英国人，那么根据法律她将自动获得这张急需的护照。奥登是个热心人，伊舍任德向他提AE？此事，他就同意了。在给伊舍任德的回电中他只用了二个字：“乐意！"这件婚姻使他们之间产生了深厚的友情。但是，没有什么事实可以说明他们俩曾在一起耽过一个夜晚。爱利卡除了给父亲做一些必不可少的编辑工作外，常常周游列国。她曾到埃及从事战地新闻报道，到德国采访纽伦堡审判，还到美国、欧洲和中东讲学。奥登则继续从事写作，他的作其中包括多篇伤感的爱情诗，献给伊舍任德——他早先在柏林的男性酒吧及后来在上海充满情欲和东方色彩的男性澡堂中的伴侣。爱利卡直到1969年患脑肿瘤去世为止，一直是W·H·奥登的妻子，奥登后来移居美国，入了美国籍，他晚年又回英国居住，1973年他在维也纳去世。

　　5．三岛由纪夫夫人（杉山洋子）
　　三岛由纪夫和杉山洋子结婚时，虽然被调查的妇女中有50％认为自己宁可自杀也不嫁给这个日本的一流作家。但19岁的洋子却对自己从众多的候选人中被选为三岛的妻子而感到兴奋不已。三岛择妻的条件苛刻，其中包括身高不能超过他本人（他身高为5.2口尺），要圆脸漂亮，要爱他本人而不是他的名声，要能照顾家庭孝敬公平，要能不打扰他的工作等等。1958年6月1日，在他们第一次见面的两个月后，就正式结婚了。婚礼以传统的方式进行，蜜月旅行结束后，他们回到东京，不久洋子便面临十分严峻的事实：丈夫是个同性恋者（这使女人感到痛苦，但作为一个日本妇女，对待这类事比一个西方女人要平静得多）并且是亲法西斯主义分子和一个叫做"X之会"的、由80个人组成的秘密武装首领。对她来说更为糟糕的是偏偏的忌妒心很重，她总是试图在儿子的生活中占据比儿媳更为重要的位置。但杉山洋子并不是那种听其命运摆布的女人，她曾成功地劝说丈夫不再在家里举办那种男性聚会，丈夫那些搞举重的朋友常在聚会上把衣服脱得只剩一条三角裤，然后抹上油，在照相机前摆弄各种姿势（1963年当她看到一本书上刊登他丈夫的一组裸体照时，她感到非常不安）。她生了两个孩子，但偏偏仍旧不喜欢她，常用讥讽的口气称她为"新娘"。1970年，始终崇拜战争天皇的三岛号召发动了一场军事政变，他的号召无人响应，他决定抓住这次机会向天皇尽忠——破腹自尽。他曾把破腹自杀比作"最后一次手淫"。三岛在他的追随者面前，用双手将一把军刀插入自己的左腹，然后又拉到右腹。根据他自己的请求，一个刽子手立即走上前将他的脑袋砍下。洋子是在乘坐一辆出租汽车时听到这个消息的。尸体运回家后，洋子按照他在遗嘱中提出的要求，给他穿上"X之会"的制服，并在胸前安放一把军刀。第二天就将他火化了。他死时45岁，仍有一个愿望没能实现，就是生前他曾希望能带孩子去参观迪斯尼乐园。他母亲同以往一样，仍旧责怪儿媳，认为三岛要是娶一个更好的女人，绝不会自杀。1979年在东京一家百货公司举办了三岛书籍展览，其中包括一尊同三岛本人一样高的裸体铜像。展览得到了杉山洋子的同意。总共吸引了20，000多个观众，其中70％是男性。
