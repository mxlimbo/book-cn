威廉·斯波尔丁眼中8次证据充足的飞碟（UFO）实例

　　斯波尔丁是世界著名的飞碟专家，目前正在亚利桑那州凤凰城一家宇航公司任质量控制工程师。在过去的20年间，他一直积极从事飞碟研究。从60年代中期开始，他担任地面飞碟观察公司西部主任。该公司专门研究有关飞碟的资料。下列的大部分资料是斯波尔丁根据信息自由法获得的。

　　1．麦克明维尔的照片
　　保罗·特伦特是一名居住在奥勒冈州麦克明维尔的农夫，他拥有两张历史上最清晰的飞碟照片。1950年有人看到远处有一个圆盘状的物体。特伦特立即取出照相机拍下两张照片。当时是大白天，因而前后的物体都很清楚。从照片上看有两个物体可能是飞碟，要么是悬在电线上方的一个较小的物体，要么就是在很远距离外一个很大的物体。照片虽经国内一些最好的照相专家的鉴定，他们说该照片是他们所分析过拍得最好的照片之一。但直到用电脑程序进行分析后才对它的真实性加以了肯定。电脑对它分析的方式同分析卫星拍下的照片一样，结论是照片是真实的。高分辨力的图像和物体的尺寸证明该物体体积很大，且当时处于很远的距离外。特伦特先生拍下的这张照片是经得起科学检验的为数不多的飞碟照片之一。

　　2．显示出受智能控制的飞碟
　　1952年7月14日傍晚，一架泛美航空公司的飞机正在接近弗吉尼亚州的诺福克市，这时飞机驾驶员突然发现在城市的东方有一片红色的亮光，很快这亮光就分裂成6个明亮的物体，形成一个带状队形，正低低地朝飞机的方向飞来。这些飞碟呈现出火红的颜色。"它们的边缘清晰，轮廓分明，呈圆形。"纳什机长说道。"顷刻间，我们可以看到它们形成了一个梯队——一个逐渐升高并略有倾斜的队形。其中为首的一个飞碟飞得较低，后面则每一个依次比前一个飞得略高一点。当这队飞行物处于飞机的正下方时，它们突然一起侧转身体，调头朝相反方向飞去。这时目击者看见了物体侧面，估计约有15英尺厚，同物体直径的比例约为1∶6。正当这6个飞行物飞离目击者时，另外两个飞行物从后面飞来加入它们已形成的队列，并迅速离去。第二天早晨空军调查人员向驾驶员询问了约两个小时，调查人员说他们还得到7份来自地面观察人员的报告，所有的报告情况都互相符合。飞机驾驶员推算这些物体的飞行速度约超过每小时10，000英里。这些飞行物的动作系数和在形成队形时表现出来的灵活机动性和精确性使得空军人员不得不将此视为一次飞碟事件。

　　3．在华盛顿特区的发现
　　1952年7月连续两个周末，飞碟闯入华盛顿特区上空。正在飞行的飞机驾驶员用肉眼看见了这些飞行灵活、行为古怪的物体。雷达荧幕清楚地显示它们的方位，3个雷达系统共标出12个物体，它们以每小时几百、甚至几千英里的速度飞行。操作人员对雷达进行了检验，发现一切正常。后来，在公众的压力下，美国空军宣布雷达上出现的目标并非飞碟，是由于气温的骤变造成地面的亮光折射至大片层的缘故。这是空军部门第一次笨拙地参加此类调查，也是日后成百次旨在AE？息公众情绪而作出的不合逻辑的解释的前奏。

　　4．梅赫的照片之谜
　　1952年7月末，一个叫拉尔夫·梅赫的海军陆战队员在佛罗里达的迈阿密市摄下了一组令人惊讶的飞行物的照片。有好几个人同时看见这个火红椭圆形物体。梅赫设法拍下了将近50个镜头。后来由于地面的建筑挡住了梅赫的视线，他无法继续拍摄，他向军事部门报告了此事，并把胶片交给他们分析。交出照片前，他做了一件很聪明的事，就是把这一组16毫米的胶片的前7张剪下，藏了起来。几年过去了，军事部门一直未将照片还给梅赫。1957年末一些记者曾问其他有关那次的发现，不久后中央情报局又找到他，向他询问当时的情况以及为何这组照其中有一段已遗失。又过了几年，地面飞碟观察公司用计算机技术分析了梅赫所留下的几张胶片，结论是照其中确实是飞碟。要是所有的照片都能让他们分析，该有多好啊！与此同时，地面飞碟观察公司向中央情报局提出了起诉。梅赫的胶片事件是该起诉的主要一部分。虽然至今失去的胶片仍未弄回，但中央情报局发布了许多文件，证实飞碟的存在。从政府手里弄回这些胶片大概也只是时间问题了。

　　5．一次雷达和目击者同时看到的飞碟事件
　　1952年12月6日黎明，一架美国空军B-29飞机正在墨西哥湾上空作训练飞行，突然机上的辅助探测仪上显示出雷达探测到的目标：几个快速飞行的物体。当机上雷达又转一圈时，该物体竟飞了13英里！经过计算，机组人员惊讶地发现它的时速起码有5000英里。机长立即开动主雷达器，经调整后，又跟上了飞行物，第三个雷达又打开，找到了同样的目标。机长命令所有的人注意它们的行踪。突然几个蓝白色的光团从边上闪过，飞行员们惊讶不已。接着连续几分钟，不仅所有3个雷达而且肉眼也看到了一组组的神奇飞行物。机组人员们都紧张地注视着它们不断地接近。这些飞行物速度惊人，飞行员们连内心祈祷一下也来不及。突然探测仪上跳出一个巨大的物体，在荧屏上足足有半英尺大小。一些较小的飞行物仍以5000哩的速度向前飞行，同那个巨大的物体融合在一起，融合后的物体突然加快速度，达到时速9000哩。当飞机降落时，情报部门的官员已等候在机场，他们向每个机组人员询问了情况。没有什么事实可以否定他们的发现，经检测，雷达的运行也完全正常。该事件在项目蓝皮书上得以记载，至今仍被视为未被解释之谜。

　　6．B-47事件
　　1957年9月25日，一架美国空军B-47飞机在墨西哥湾和南方中部各州上空作训练飞行时，遇见一个不明飞行物。该物体积大如一座粮仓，散发出火红的光芒。飞机的速度变换多次，飞行物始终同飞机同速飞行。肉眼看到的情况同空中及地面的雷达定位情况相符。飞行物很明显在发射出某种类型的高强度的电磁能。有关人士向机组人员询问了情况后，政府施加了压力，因此有关该事件的记录没被披露。B-47飞机装有抗电子干扰装置，神秘的飞行物围着飞机转圈，似乎是在同飞机"玩耍"，机上的先进电子设备测量出飞行物发出的信号，它在一个狭窄的微波区内辐射出大量的电磁信号。看上去飞行物是用微波辐射作为它的推进系统。这种系统能减弱或完全排除地球的重力。难怪美国政府极力不让公众了解这类事实，因为毫无疑问这表明这些飞行物具有极大的空中优势，也暴露了我们军事体系中真正的弱点。

　　7．1957年一个巨大不明飞行物造成的惊慌
　　1957年11月发生的一次惊慌是在第二个俄国载狗卫星进入轨道前几个小时开始的。地点是德克萨斯州莱伏尔兰德市的郊外，大约在午夜前一个小时，第一个在那晚目击一连串神秘事件发生的人打电话给政府有关部门，告诉他们他看见了一个巨大的不明飞行物。"我开始看见田野里发出奇怪的闪光，接着该物体飞离田野，加快速度朝我冲来。当它接近我时，我驾驶的卡车的灯突然全部熄灭，引擎也突然熄火。我跳下车。扑倒在地，刹时间我感觉到一阵强烈的气浪，这怪物就在汽车上方轰隆一下飞过，它发出的声音响如雷鸣，震得汽车不停地晃动。这物体约有200英尺长，发出一种可怕的红光。"在以后的3个小时里，又有9个人报告也遇见了似乎是同样的物体。在6起报告中，目击者的汽车在接近飞行物时都感受到强烈的电磁效应。虽然政府所作的解释十分牵强，但这一系列普通居民遇见不明飞行物的事例很有力地证明了这类现象是确实存在的，因而也使政府的态度发生了改变，不再像以前那样对这类有关飞行物的报道只是嗤之以鼻。

　　8．伊朗事件
　　1976年9月，也就是美国空军项目蓝皮书结束将近6年后，在伊朗德黑兰上空发生了一起4架喷气机同一个不明飞行物相遇的事件。当时，先是约有数百人目击一个奇怪、明亮的物体，伊朗的雷达也跟踪到一个不明的目标，两架最先进的喷气机升空前去拦截和辨认。当第一架飞机飞抵同它相距25海里的位置时，飞机同地面的联系突然中断，与此同时第二架飞机也很快地同它靠近。驾驶员聚精会神地注视着对方，突然这物体在它的下面抛出一个较小的物体，飞机同地面的联系也中断了。这较小的飞行物朝飞机迎面飞来，驾驶员立即按动空对空导弹键钮，可导弹系统失灵，驾驶员只好又将飞机强行俯冲，刹时间飞行物在飞机的座舱上面"嗖"地一声掠过，差一点将飞机击中。这时这飞行物又回到那个大飞行物边上，同它巧妙地连接了起来，然后飞离了这个地区。根据信息自由法，美国政府只得公开了有关该事件的资料，因此政府任何试图否认这是一次飞碟事件的说法都是无法成立的。
