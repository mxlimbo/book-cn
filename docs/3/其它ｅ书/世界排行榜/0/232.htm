玛丽·雷诺最愿意目睹的5个历史事件

　　玛丽·雷诺的真名是玛丽·查兰，出生于英国。1939年他发表了第一部小说《爱的目的》一举成功。第二次世界大战爆发后，她停止创作，从事护理工作。战争结束她移居南非开普敦市，并在那里居住至今。她最近的著作有《天堂之火》及《波斯少年》。
　　1．耶稣复活（29）。
　　2．五世纪时处于完好状态的雅典卫城。城内耸立着镶嵌铜边的大理石雕塑以及依旧清晰可见的艺术大师们的绘画作品。
　　3．伊苏斯战役前（公元前333）亚历山大对士兵发表的演说。他那令人惊讶的口才一定是发挥了无穷的号召力。
　　4．伊尼戈·琼斯的宫廷假面剧。华丽的场面。
　　5．能在拜伦的继承人令人痛心地焚毁他的"回忆录"前读到这部作品。
　　注：伊尼戈·琼斯是英国第一位伟大的建筑设计师。假面剧是一种表达某种寓言的戏剧，由戴假面具的演员表演。此剧起源于意大利佛罗伦萨的麦第奇宫廷。当时年轻的琼斯正好在佛罗伦萨学习艺术。1605年他将此剧搬至英国并为此剧绘制舞台布景，设计服装和特别效果。他在这方面的艺术才能受到了广泛的赞誉。当时假面剧的剧本是由本·约翰逊创作。
