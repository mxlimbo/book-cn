7位将别人或自己写的东西实实在在地吃了下去的著名人物

　　1．曼涅里克二世
　　他是非洲历史上最伟大的统治者之一，是现代埃塞俄比亚的缔造者，出生于1844年。他曾被自己的敌人狄奥多尔二世囚禁了整整10年，最后越狱逃走。他于1865年宣布自己为绍阿王国国王，并且以武力征服了邻近的王国，建立起强大的埃塞俄比亚帝国，自己当了皇帝。当意大利企图吞并埃塞俄比亚时，他率军在阿杜瓦战役中大败意军，迫使意签订了著名的《亚的斯亚贝巴条约》，使埃塞俄比亚成为一个独立而且强大的国家。只有一件事是鲜为人知的，曼涅里克二世只要一有病痛，就要吃几篇《圣经》书页，他坚持说这样就能使自己恢复健康。1913年12月的一天，他觉得自己非常不舒服，因此将《圣经》中的《帝王记》篇全都撕了下来，并咽下肚子。但是很可惜，这次他一吃完竟然就死去了。由此看来，即使是最好的书，假如我们将它生吞活剥地吃下去，其结果也许适得其反。

　　2．菲利普·安德里亚斯·奥登伯格
　　他是十七世纪德国著名的法律学者和历史学者，还是一位极有声望的政治家。由于他写的一本小册子得罪了当局，便遭到逮捕。对他的判决是让他把自己的这本小册子吃下去。法官对他说，他同时还要遭鞭打，直到他把最后一页吃下去为止。于是，他不得不以尽可能快的速度将整本书都咽下了肚子。奥登伯格后来死于1678年。

　　3-4．两位教皇的使节
　　在1370年，教皇派了两名使节去见伯纳波·维斯康蒂，通知他已被革出教门。教皇的命令是写在一卷羊皮纸上的，上面还有铅印和绸带。伯纳波·维斯康蒂恼怒万分，将两名使节逮了起来。他要他们将羊皮纸、铅印和绸带都吃下去，然后才将他们释放。

　　5．西奥多·里恩金
　　他是一个丹麦作家，当时丹麦正受瑞典统治。1644年，里恩金用拉丁文写了一本书，指责瑞典给丹麦带来了无穷的灾难。瑞典人立刻将他投入监狱。几年之后，里恩金的健康状况越来越糟，这时瑞典人让他作出选择：要么砍掉他的脑袋，要么他把自己写的那本书吃下去。里恩金当然选择了保留脑袋——他将自己的书煮成糊，然后喝了下去。

　　6．恩斯特·托勒
　　托勒1893年出生于普鲁士，在法国完成了法律学业。第一次世界大战爆发后，他加入德国军队，在战场上负伤，战后成为一名活跃的和平主义者。他因所谓叛国罪被监禁，出狱后他参加了推翻巴伐利亚王朝的战斗，后来又一次遭到逮捕，并在监狱中呆了5年。在被监禁的日子里，他写了许多著作，如《破坏机器者》，《这就是生活》，还有自传《我是一个德国人》和《铁窗中》。出狱后，他目睹希特勒的上升，在去苏联和美国旅行之后，他回德国作了大量反对法西斯主义的演讲。1933年，他被纳粹逮捕。据他自述：“简直是惨无人道，他们竟强迫我将最近写的手稿和书全都吞了下去。"后来，他被驱逐，独自一人来到纽约（他的妻子离开他另外嫁人了）。他非常贫困，因为他将自己所有的钱都捐给了西班牙内战中的难民。最后，他于1939年5月22日在自己的寓所中自杀了。

　　7．艾萨克·沃尔玛
　　艾萨克是许多本小册子的作者，他讽刺了萨克斯·梅宁根公爵的各种丑行。公爵有一天请他赴晚宴，据弗雷德里克·玛文叙述：“公爵没有为他上一道菜肴，而把他的每一本书都搁在他的盘子里，下令让他统统吃下去。"这时是1910年。
