历史上10位布鲁斯·梅茨利希博士最愿意成为的人

　　这位麻省理工学院的历史教授和《交叉历史》杂志的副编辑在学术上是一名杰出的创造者，他写的《寻找尼克松》一书在1972年出版时，被纽约时报的书评专栏称为是历史心理学这个新领域的发展中具有极其重要意义的作品。他近年来发表的作品还有《詹姆士和约翰·斯图特·米尔》（1975）以及《基辛格》（1976）。
