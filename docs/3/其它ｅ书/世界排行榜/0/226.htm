历史上10位布里吉德・布罗菲最愿意成为的人

　　作为一个小说家的女儿，布里菲本人也是一个知名的小说家、散文作家、剧作家以及评论家。她的第一部获奖小说是《哈肯费勒的猿猴》，其他的作品还有莫扎特和奥比里、比尔兹利的心理分析或传记，以及一本广播文学集锦《不要永远不忘》。
　　1．亚里士多德
　　2．提香
　　3．安德鲁・马韦尔
　　4．托马斯・特莱恩（一个素食者：“生命对所有的野生动物也一样的珍贵，它们也同人一样要求延续自己的生命，一样不愿中断自己的生命。"）
　　5．乔治・费德里克・亨德尔
　　6．巴尔瑟沙、纽曼
　　7．亲王殿下的狗（在一只亚历山大教皇送给威尔士亲王弗雷德里克的狗的项圈上写道：“我是亲王殿下的狗，请告诉我，先生，你是谁的狗？"）
　　8．约翰・纳什
　　9．肖伯纳
　　10．科莱特夫人的一只猫
　　注：安德鲁・马韦尔（1621-1678）是十七世纪最杰出的诗人之一，著有《花园》和《致他羞怯的情妇》。托马斯・特莱恩（1634-1703）是一名英国的苦行僧，曾著有数部有关饮食及卫生的作品。巴尔瑟沙・纽曼（1687-1753）是巴罗克时期德国的多才多艺的建筑师，曾设计过桥梁、公共建筑、宫殿和100多座教堂。约翰・纳什（1752-1835）是英国的建筑师和城市规划专家，曾规划伦敦的里真茨公园和里真茨大街。
