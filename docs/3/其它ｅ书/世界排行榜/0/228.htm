阿巴·埃班最愿意目睹的5个历史事件

　　埃班是以色列最受尊敬的外交家之一，1949年他作为以色列首任大使出使联合国，并于1950年至1959年期间兼任驻美国的大使。从1966年至1974年他出任外交部长时的一系列杰出外交手腕使以色列安然度过1967年的危机。1959年他被选入以色列议会，至今他仍在为之工作。他发表的著作包括《我的国家和阿巴·埃班一部自传》。
　　1．犹太人从巴比伦返回重建犹太教堂，波斯国王居鲁斯允许犹太人回归，犹太人花了几十年时间重建了这座教堂。这样一来，既使犹太教得以保存，又使基督教以后得以诞生。
　　2．雅典同斯巴达战争结束后伯里克利斯在雅典阵亡将士葬礼上的演说（前431）。
　　3．在费城签署美国独立宣言（1776．7．4）。
　　4．贝多芬第九交响乐（合唱）的第一次正式演奏（1824）。
　　伦敦爱乐乐团付给贝多芬100英镑，要求获准组织一场义演以演奏该曲。贝多芬急需这笔钱。
　　5．温斯顿·丘吉尔在众议院的演讲（1940．5．13）。"我所能奉献的只有热血，辛劳，汗水和眼泪。”
