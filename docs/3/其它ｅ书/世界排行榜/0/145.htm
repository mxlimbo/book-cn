10名冰球史上最优秀的职业球员

　　世界排行榜第一卷刊登了由公民荣誉厅主任比尔・斯克罗德选编的10名最优秀的职业冰球手名单，该名单在冰球迷中引起很大的争议，因而加拿大运动杂志专栏作家德鲁・斯奈德邀请一批读者挑选10名最佳冰球手，在300份答卷中，戈迪・豪，鲍比・奥尔鲍比・赫尔（斯罗克德名单中的前三名）以及盖伊・拉夫勒被一致选中。

　　1．戈迪・豪
　　1946年至1971年在底特律红翼队打右边锋，他创下进球最多、得分最多、传球得分最多3项纪录。1973年至1975年他效力于休斯顿空气队。

　　2．鲍比・奥尔
　　1960-1976年在波士顿布鲁因斯队打后卫。荣获1970、1971及1972年最佳运动员称号并连续8年（1968-1975）获得詹姆斯・诺利斯最佳后卫奖杯。

　　3．盖伊・拉夫勒
　　1971年至今在蒙特利尔加拿大人队担任右边锋和中锋。

　　4．勃比・赫尔
　　1957-1972年效力于芝加哥黑鹰队，出任左边锋。7次名列全国冰球联盟赛得分第一名，12次被送入全明星队。1972年担任威尼佩格喷气机队的教练兼运动员。

　　5．豪伊・莫伦茨
　　先后出任蒙特利尔加拿大人队（1923-1934，1936-1937）、芝加哥黑鹰队（1934-1935）和纽约园林看守人队（1935-1936）的前锋。豪伊以闪电般的速度而闻名，倍受球迷的欢迎。他32岁去世。

　　6．特里・奈查克
　　先后为红翼队（1950-1955，1957-1964，1968-1969）、布鲁因斯队（1955-1957）、多伦多枫叶队（1964-1967）、洛杉矶国王队以及园林看守人队（1969-1970）担任守门员。进入球坛不久便被选入全明星队，仍保持总共103场不失一球的最高纪录。41岁时因同队员打架受伤而死。

　　7．"火箭"毛莱斯・理查德
　　1942-1960年在加拿大人队打右边锋，是唯一在赛季（1945）中进球达50个的球员。1961年被选入冰球荣誉厅。

　　8．道格・哈维
　　曾在加拿大人队（1947-1961）、园林看守人队（1961-1964）、红翼队（1966-1967）、圣路易斯蓝队（1968-1969）出任后卫。在园林看守人队（1961-1962）担任教练，7次荣获詹姆斯・诺利斯奖杯。1973年被选入冰球荣誉厅。

　　9．琼・贝利沃
　　1957-1971年在加拿大人队打中锋，是继戈比・豪之后唯一得分总数超过1000的球员。

　　10．"旋风"弗雷德・泰勒
　　曾任渥太华议员队（1908-1909）、伦佛鲁百万富翁队（1910-1911）及温哥华百万富翁队的后卫、中锋（1913-1921），1923年只参加了一场比赛，1945年被选入荣誉厅。
