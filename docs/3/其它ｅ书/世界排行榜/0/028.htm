10起最不公正的判决

　　1．I．D．科森
　　1944年8月，11岁的科森和他的家庭成为第二次世界大战期间在美国这片土地上的第一批平民伤亡人员。正当科森和父亲、叔叔及2个年幼的堂弟在佛罗里达埃格林空军基地以北的小农庄里时，美军的轰炸机由于意外而将炸弹扔在了他们的房子上。这些人中，只有科森一人侥幸活了下来，但他的脊椎被弹片弄断了，使他腰部以下完全瘫痪。联邦政府急于平息此事，对他的伤残赔偿了3750美元，5年后又判决追加15000美元。但是，从此之后他再也没得到过政府的任何资助。科森始终坐在轮椅上，他的生活费是由一个社会救济组织提供的，每月167美元。

　　2．斯蒂芬·丹尼逊
　　1925年，当他16岁时，在纽约的一家商店里偷了1盒价值5美元的糖果，为此他被送进州的少年教养所。2年之后他又被送进州监狱，在那里，由于他的一些小错误，又被判追加刑期。但他后来似乎被监狱管理人员遗忘了，在那里他一直呆到1959年才被释放。为了5美元的一盒糖果，他竟然被监禁了34年。

　　3．阿尔弗雷德·德雷福斯
　　1894年，法国政府的情报部门发现，军队的高级参谋人员中有人向德国出卖情报。阿尔弗雷德·德雷福斯上尉立刻被捕了，原因是那份情报的署名是D，他姓氏的第一个字母——还因为他是参谋人员中唯一的犹太人。他被军事法庭判处终身监禁，在魔鬼岛服刑。1899年，那个真正的出卖情报者马里·埃斯特汉斯少校坦白了自己的罪行。但法国军方拒绝承认自己的错误，仍然让无辜的德雷福斯在魔鬼岛上服刑。到了1904年，他才重获自由。1906年，法庭重新审理了他的案件，为他恢复了名誉，并授予他荣誉勋章，军衔升至上校。

　　4．利奥·弗兰克
　　1913年在乔治亚州的亚特兰大，利奥·弗兰克，一个铅笔工厂的犹太厂主被指控犯有强奸谋杀罪。死者是他厂里的一个13岁的女孩。有证据显示，是这家厂的看门人吉米·康利而不是弗兰克谋杀了那个女孩。但主要由于弗兰克是个犹太人，他便被判有罪，并被判处死刑。1915年，在死刑执行前又被改判为终身监禁。不久，40个中产阶级的白人冲进监狱，将弗兰克带到马里塔小镇，用绳子把他绞死了。

　　5．伦纳德·汉金斯
　　1933年，汉金斯因涉嫌在明尼苏达州的明尼阿波利斯抢劫银行并枪杀2名警察和1个路人而被判有罪。3年之后，联邦调查局逮捕了一个名叫杰斯·道尔的人，他供认是他犯了那AE？抢劫谋杀罪。联邦调查局通知明尼阿波利斯警察局，汉金斯完全是无辜的，但是当局却不肯释放汉金斯，因为联邦调查局不愿意让他们看道尔的宗卷。结果，汉金斯又在监狱里呆了15年，直到1951年才被释放。

　　6．朱利叶斯·克劳斯
　　1931年，18岁的朱利叶斯·克劳斯和另一个人一同被逮捕了，罪名是抢劫俄亥俄州的一家商店并枪杀了一名店员。4年后，那另一个人临死前坦白说，和他一起犯下那桩罪行的并不是克劳斯，而是一个名叫柯蒂斯·库梅尔的人。但是当局对此无动于衷，既没再作调查也没去追捕库梅尔。后来，克劳斯在服刑9年之后越狱逃跑了，他找着了库梅尔，并说服他和自己一起去向当局投案。结果，库梅尔被判有罪，但法庭却拒绝重新审理克劳斯的案件。于是克劳斯又不得不继续呆在监狱里，直到1951年才被释放，前后整整20年。

　　7．塞缪尔·穆德
　　1865年4月14日深夜，塞缪尔·穆德医生替谋杀林肯总统的凶手约翰·布斯接好了断了的腿。10天后，穆德被捕。1865年5月10日，他和另外7名嫌疑犯被送上一个军事法庭，对他们的指控是"合谋杀害亚伯拉罕·林肯总统"。虽然这一"合谋"并无证据，但穆德医生却依然被判有罪，送往佛罗里达海湾的加登基岛服终身苦役。在以后的4年，他努力地为岛上的黄热病患者治疗。1869年，安德鲁·约翰逊总统特赦了塞缪尔·穆德，但当他回到马里兰的家中时，已经"身心俱毁……被疾病折磨着，再也没能恢复健康。”

　　8．詹姆斯·蒙哥马利
　　1923年在伊利诺斯的沃克根，一个62岁的白人老姑娘玛米·斯诺指控詹姆斯·蒙哥马利强奸了她，后者是一个26岁的黑人。经过20分钟的审讯，蒙哥马利就被宣判有罪，因为检查官是一名三K党人，他将一份医院证明藏匿了起来，上面说斯诺小姐依然是一位处女。为了这个莫须有的罪行，蒙哥马利在狱中度过了26年，直到1949年才被释放。

　　9．加布里埃尔·拉塞尔
　　1968年在法国马赛，加布里埃尔·拉塞尔，这位31岁的女教师，和她的学生、16岁的克里斯廷·罗西成为一对情人。这样一件事竟然触怒了当局，罗西被送进少年教养所，后来又被送进一家精神病院，接受"睡眠疗法"；加布里埃尔被逮捕了，其罪名是"败坏学校风纪"，并被投入监狱。由于无法忍受折磨，她于1969年自杀身亡。

　　10．斯科特波罗的男孩们
　　1931年在亚拉巴马州的斯科特波罗，9个黑人青年——海伍德·帕特逊、克拉伦斯·诺里斯、查利·威姆斯、安德鲁·赖特和劳勒·赖特兄弟、奥泽·鲍威尔、威利·罗伯逊、奥伦·蒙哥马利以及尤金·威廉斯——一起被捕了，其罪名是轮奸了2位白人妇女，当她们正坐着一辆货车穿过亚拉巴马北部时。尽管医院的报告证实这2位妇女并没有遭受强奸的痕迹，但这9个黑人青年还是都被认定有罪，其中8人被判处死刑，只有1个13岁的被告被判终身监禁。此后，曾进行过多次重新审理，分别在1933年、1936年和1937年，其中只有4人被释放，虽然这时那2位妇女中的一位已承认她们并未遭受强奸。在1943年、1946年以及1950年，又有3人分别被释放了。而第八个人——海伍德·帕特逊于1948越狱逃跑，在密执安州因触犯刑律而再次被捕，1952年死于狱中。至于最后那个克拉伦斯·诺里斯，他于1946年被保释出狱，随后就逃走了，直到1976年才得到赦免。因此，结束这一案件整整花了45年时间。
