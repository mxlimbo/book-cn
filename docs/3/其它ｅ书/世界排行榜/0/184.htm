浪费纳税人的钱：14个金羊毛奖得主

　　1975年3月，威斯康星州参议员威廉姆·疲劳克斯迈尔创立了金羊毛奖，奖给那些政府开支和浪费行为中数额最大、最荒谬和最具有讽刺意义的实例。以下是一些该奖的得主。1．众议院
　　1975年众议院拨款130万元将19座电梯改为自动式，另一方面却仍然雇人操纵电梯。而参议院则早已开了雇佣工人操纵自动电梯的先例。可以毫不夸张地说，它已开始建造一幢耗资1亿2千万元的办公楼，这幢工期已耽误了4年，投资也已超过154％的大楼里将设有一个仅供议员享用、并能鸟瞰城市景色的餐厅。可议员们目前已经拥有两个餐厅。

　　2．联邦航空管理局
　　耗资57，800美元调查美国航空公司见习空中小姐的体形，对每位空中小姐必须提出79个测量数据，包括小腿的尺寸、臀部的大小和鼻子的高低。据说设计安全器材时需要这些数据，但人们不禁要问为什么这笔开支要纳税人承担，而不是航空公司或飞机制造商承担。

　　3．海军部
　　1974年9月，一个叫做台儿霍克的私人民间组织举行了一次聚会，为此，海军部出动64架飞机将1，334名官员送到拉斯维加的希尔顿旅馆，共动用资金191，000美元，并耗费347，000加仑的汽油。另外，海军另外还花费537，000美元重新装修美国副总统纳尔逊·洛克菲勒的临时官邸。其中包括窗帘26，400元，银器21，200元以及瓷器10，400元。

　　4．美国联邦能源署主席弗兰克·扎布
　　1975年扎布先生花费25，000元及将近19，000加仑的燃料周游全国，给经济界或公民团体做报告，宣传保存能源的必要性，然而他起码有一半行程是乘坐空军的豪华飞机，这种飞机1小时烧掉的汽油足够让每个美国旗车司机行驶整整1年。

　　5．空军部
　　空军维持着一支叫做第89军事空运联队的机队。该队拥有23架超一流喷气机，每架价值6，100万美元。它的唯一任务是运送政府高级官员。尽管该队所到之处均有商业航班运行，却仍要花费每年约600万元纳税人的钱作1，000次左右的航行。航行中约有87％是运送国防部官员。另一件有关空军浪费的典型事例是在1978-1979年，它们花费31，000元进行一次调查，决定男性航空兵是否允许在穿军装时使用雨伞。

　　6．国家航空和宇航局
　　虽然该局从月球上取下的岩石已被存放在一座价值870万美元完全符合标准的钢筋混凝土建筑里，该局却仍然要申请280万美元为它们建造一座新居。

　　7．陆军工兵部队
　　达到联邦政府中超支的最高纪录，1976年9月，在178个正在施工的主要民用建筑工程中，约有83个超过预算至少达100％。超支最多的项目是得州古柏湖防洪工程，以及新奥尔良至路易斯安娜州维尼斯之间的抗飓风工程，超支额分别为325％及1，027％。

　　8．司法局
　　花费27，000美元调查犯人为何想越狱。此外该局还耗资200万美元研制了一种新型的配有多角度后视镜的警察巡逻车。可不幸的是地方警察局根本无力购买这种昂贵的车子。

　　9．美国邮政局
　　花费340万美元纳税人的钱开展一场宣传运动，鼓励美国人多写信，而实际情况是各类邮件包括一等邮件都在亏本运行。更为离奇的是该局还耗资775，000美元调查该项宣传运动的效果。

　　10．联邦储蓄保险公司
　　在1976年3月至1977年5月，罗伯特·巴纳德担任该公司主席期间，他一再动用公司的车辆和司机送他太太和孩子去医院、博物馆或去海滨度假。该公司还为巴纳德加入弗吉尼亚一个私人网球俱乐部支付会员费。

　　11．全国心理健康研究所
　　花钱调查秘鲁一家妓院里人的行为和社会关系。该项调查是该所一个研究人员价值9万7千美元的研究费用的一部分，具有讽刺意义的是该研究员曾写过一本叫做《学术小动作的书》。在书中他指出"并不是所有的学科都享有同样榨取政府或基金会资金的机会。"他还劝说道：“假如你的研究需要你出国，那么你用美国人的收入，到一个生活费用低的国家，便可以过上一种如同东方君王一般的生活。要是在国外呆满18个月或更多一点时间，你还可以获得一笔2万美元左右的免税收入。"这位研究员自己在秘鲁果然住满了18个月，全部开支当然是来自纳税人。

　　12．国防部
　　在国防部庞大的预算中有一笔鲜为人知的项目叫做"紧急和特别开支"，其中包括一笔每年120万美元的招待和社交费。1977年至1978年在这项目下支出的有：为限制战略武器会谈顾问委员会成员的太太们举行的一次茶话会（107美元），接待来访的墨西哥空军司令，包括安排他去迪斯尼乐园游览（3，284．19美元）；给美国空军独立节高尔夫球锦标赛的获胜者颁发奖章。此外，一个官员举行了一个欢送会，花费了约176美元，几天后，就是这个官员又为他的朋友举行一次晚会，费用为271．8美元。毫无疑问，这两项开支全部由纳税人承担。

　　13．国家公路交通安全署
　　花费120，126美元研制一种车身很低的摩托车。见过设计的人都认为这种车子不行。一家公司在签订了生产合同后，经过研究发现该车问题严重，要求取消合同，安全署坚持要公司投产，结果车子出厂后证明完全是个失败。令人惊讶的是安全署居然又开始研制另一种同样无用的低车身摩托车。

　　14．住房及城市发展部和经济发展署
　　两个部门共动用279，000美元的资金在密执安森林中建造了一个社区中心，该中心地处偏僻，不通公路，以致后来某一天，当中心的房子倒坍时，竟然一连数天无人知晓。在这笔开支中，住房及城市发展部花了100，000美元建造了中心的房子，经济发展署负责修建公路，可因经费不足，只得中途停工，因此中心无法使用，整整闲置了3年。后来一个机警的护林人突然发现房子已经倒坍并立即报告了当地的邮政局长。
　　一枚节约的奖章被授于斯密索尼协会，该机构建造了航空和航天博物馆，工程按时完工，于1976年6月1日提前4天开馆，工程造价比10年前作的预算还降低了500，000元。此外，整个建筑还增加了4个在原设计中没有列入的项目：一个大型剧场、一个天文馆、一个画廊以及一些方便残疾人参观的设施。
