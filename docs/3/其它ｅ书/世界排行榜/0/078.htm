利奥纳德・费瑟眼中的全明星爵士乐队

　　费瑟是一位很有成就的作曲家，他曾为康特・贝茜和其他一些美国和欧洲的流行歌曲演唱家配乐。他著有多部著作，其中包括《从阿姆斯特朗到迈尔斯》，《七十年代爵士乐百科全书》等。他目前担任洛杉矶时报的爵士乐编辑。他被公认为全美最有权威性的爵士乐评论家之一。
　　领队、作曲：杜克・艾林顿。
　　小号：路易斯・阿姆斯特朗，比克斯・贝德比克，路伊・爱尔德里奇，迪齐・盖莱斯普，迈尔斯・戴维斯。
　　长号：杰克・蒂加登，J．C．希金博特姆、J．J．约翰逊，比尔・沃特罗斯。
　　中音萨克斯号：本尼・卡特或乔尼・霍杰斯、查理・派克。
　　高音萨克斯号：科尔曼・霍金斯或莱斯特・扬、约翰・科尔特莱恩。
　　低音萨克斯号：哈利・卡尼。
　　钢琴：阿特・泰特姆。
　　吉他：查理・克里斯廷。
　　低音提琴：查理・明戈斯。
　　鼓：西德・凯德里特。
　　演唱：比利・哈立德。
　　客座独奏演员：本尼・古德曼，单簧管，乔・维纳蒂，小提琴，莱恩内尔・汉普顿或米尔特・杰克逊，电振打击琴。
　　赫巴特・劳斯，长笛。
　　康特・贝西，钢琴。
　　利奥纳德・费瑟还说明道，“虽然艾林顿是作为领队和作曲被选入全明星队，但他的钢琴演奏水平同泰特姆也不相上下。此外，阿姆斯特朗和蒂加登也同样能在声乐方面同哈立德一比高低。
　　“我感到很难在本尼・卡特和乔尼・霍杰斯之间或在科尔曼・霍金斯和莱斯特・扬之间作出比较，他们都是出类拔萃的天才，因此在这种情况下我只能把两人同时选上，尽管有时只需一人即可，如电振打击琴那样。”
　　“爱挑剔的人也许会指出在小号的位置上选入了比克斯・贝德比克，而他实际上是个短号手。这两种乐器的声音其实是非常接近，就连一些短号和小号手们也区别不出。”
　　“如果再一次阅读这份名单，你会发现许多爵士乐伟人如埃拉・费兹拉尔德、雷德・诺伏等没被选上，我必须强调的是作这样的选择本身就是很不恰当的，不过它有一个好处：即让我们想起在那么短暂的一段时间里竟然涌现出如此众多的爵士乐大师。
