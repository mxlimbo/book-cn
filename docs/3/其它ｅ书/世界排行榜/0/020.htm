9次政治方面的重大失言

　　1．“美国对第三次世界大战可以作出许多贡献。"（1975年，罗纳德·里根所说，他本来应该说"第三世界"。他的这个错误一共重复了9次。）

　　2．"谢谢你，证据州长。"（理查德·尼克松1974年对华盛顿州州长丹·埃文斯说。证据—evidence，埃文斯—Evans，发音接近。或许尼克松由于水门事件，太经常地想到"证据"一词了。）

　　3．"这是一个令人怀疑的总统。"（尼克松总统1974年在水门事件被揭露后所说，他应该说"令人怀疑的先例（Precedent）"，却说成了"令人怀疑的总统（President）"。

　　4．"我希望撤销对阿格纽副总统的指控，并宣布他有罪。”（约翰·康纳利1973年在为副总统辩护时，将"宣布他无罪"说成了"宣布他有罪。"）

　　5．"我的心和你们的心一样黑。"（民主党人马里奥·普罗卡西诺1969年竞选纽约市市长时，对一群黑人这样说。）

　　6．"弗兰克·奥康纳正在你们中逐渐壮大，就像癌一样。”（马里奥·普罗卡西诺1965年在支持他的党内同伴竞选纽约市市长时这样说，他后面那句"像癌一样"不知是怎么冒出来的。）

　　7．"让我们一劳永逸地将问题解决。那些警察被派去并不是为了制造混乱，而是为了维持混乱。"（芝加哥市长理查德·戴利1968年在一次集会上为警察的行为辩护时这么说。也许他应该说"维持秩序"。）

　　8．"忍受痛苦的权利也是自由经济的快乐之一。"（艾森豪威尔的顾问霍华德·派尔在1956年总统竞选时所说的话。）

　　9．"警察完全有能力和罪犯比个高低。"（约翰·海伦作为纽约市长在1922年针对犯罪率高所作的评论。）
