11本曾遭到13个或13个以上的出版社拒绝的畅销书

　　1．西奥多・盖泽尔的《莫尔贝利大街》（纽约：先锋出版社，1937）
　　盖泽尔的这本书曾遭到23家出版公司的拒绝。一天，他从一家出版公司出来，手持被退回的书稿和插图，朝回家的方向走去。当他行至麦迪逊广场时，突然遇见了一名大学时的同班同学。此人正好是先锋出版社的儿童书籍编辑。他看了盖泽尔的书后表现出极大的兴趣，并当场同盖泽尔签订出版合同。该书出版后大为畅销，曾先后印刷20多次。

　　2．詹姆士・乔伊斯的《都柏林人》（伦敦：格兰特・理查兹出版社，1914）
　　本书收集了乔伊斯创作的15篇短篇故事。在这些故事中作者用十分暗淡的色调描绘了都柏林市民的生活。由于他拒绝对书中的内容作任何修改，总共有22家出版商拒绝接受此书。最后，格兰特・理查兹出版社同意接受，并于1914年出版。乔伊斯在写给贝内特・舍夫的信中说，"当这本书最后出版时，一些人将书堆在街头焚烧，这实际上是又一种迫害异教徒的行为。"不久以后，该书受到了广大读者的高度赞扬，被称为是一部天才的作品。

　　3．理查德・霍克的《M．A．S．H》（纽约：威廉・莫罗出版公司，1968）

　　霍克在他的信心十足的代理人的帮助下，花了整整7年时间完成了这部富有幽默色彩的战争小说，结果却遭到21家出版公司的拒绝。最后，他联系的第22家公司――莫罗出版公司――总算答应出版。当该书被拍成电影后，它的知名度大为提高，便立刻成了一本畅销书。此外，霍克又完成了几部续集。1972年电视台播放了一部以本书的人物为基础的大型电视连续剧。

　　4．查尔斯・肖的《只有上帝知道》（纽约：皇冠出版社，1952）
　　在3年的时间里，肖的这本反映美国在南太平洋的海军的幽默小说遭到了几乎所有的澳大利亚出版商以及20家英国出版公司的拒绝。当这名澳大利亚作家找到一名美国代理人后，他的书很快地就找到了买主――皇冠出版社。不久书的平装版和海外版便投入了市场。1957年，该书被拍成了电影。

　　5．索尔・海尔达尔的《康-提基号》（芝加哥：兰德・麦克奈里公司，1950）
　　在兰德・麦克奈里公司决定出版这本描述作者本人乘坐木排横渡太平洋的惊险经历的书以前，约有20家公司认为该书毫无出版价值。结果这本书于1951和1952年连续两年被列入非小说类书籍前10名的排行榜。而且将此书改编成的纪录性影片还获奥斯卡金像奖。该书的总销售量达到数百万册。

　　6．理查德・巴赫的《海鸥》（纽约：麦克米兰公司，1970）
　　巴赫这则长约1万字、描写一只海鸥的动人故事曾遭到18家出版商的拒绝。后来麦克米兰公司接受了该书。刚出版时，该书仅悄悄地卖掉7500册。不久，销量开始大增，并被AE？为当月的畅销书。单在艾冯地区就售出100万册。接着又发行了海外版。到1975年为止，在美国总共售出700万册。1973年该书被拍成电影。

　　7．理查德・布莱克莫尔的《洛纳・杜恩》（伦敦：桑普森・洛公司，1869）
　　1869年，当桑普森・洛公司出版的这套长达3卷的书刚刚问世时，市场上无人问津。这次出版的失败当然使曾经拒绝出版该书的18家出版社更感到拒之有理。可两年后，当桑普森・洛公司将该书的3卷合成1卷，并再次出版时，竟获得了意想不到的成功。1874年，美国哈普公司未办理版权手续就在美国出版此书，结果大为畅销，竟然成为当年的头号畅销书。从此，该书便一版再版，供不应求。此外，该书还曾三度被搬上银幕。

　　8．派德里克・丹尼斯的《梅姆姨妈》（纽约：先锋出版社，1955）
　　这部描写一名自由女性的长篇小说曾被17家出版社拒之门外。当第18家出版社――先锋出版社――将它推向市场时，竟然一炮打响，极为畅销。不久该书便被搬上舞台，在百老汇风风火火地演了很长一段时间。接着此书又被拍成电影，由罗莎林・拉塞尔主演，同样大获成功。10年后又有人将此改成一部音乐剧，在百老汇上演时又吸引了大批观众。到目前为止，该书的总销售量已超过200万册。

　　9．欧文・斯通的《渴望生活》（伦敦和纽约：朗曼斯格林公司，1934）
　　这部反映艺术大师梵高的一生的小说刚一出版就大获成功。读者争相购买。可在出版前，竟有17家出版社先后拒绝出版此书。1956年本书被拍成电影，由著名影星柯克・道格拉斯主演。至今该书已发行了70多个版本，共2400万册。

　　10．劳伦斯・彼得的《彼得原则》（纽约：威廉・莫罗公司，
　　1969）
　　总共约有16家出版社曾先后拒绝接受此书。可当莫里在报上发表了一评论述同一主题的文章后，这些出版社却又手持合同纷至沓来。最后由莫罗公司买下该书的版权。到目前为止，此书已售出600万册。并为彼得以后写就的《彼得计划》、《彼得语录》和《彼得的人民》等书的出版铺平了道路。

　　11．弗兰克・赫伯特的《沙丘》（宾州拉得诺市：切尔顿公司，1965）
　　赫伯特这部大型科幻小说曾先后被13家出版社退稿，他们对该书的评论是"节奏太慢"、"篇幅太长"、"内容陈旧"、“缺乏悬念"以及"情节混乱，让人感到厌烦"等。但在赫伯特和他的代理人锲而不舍的努力下，终于找到了一家愿意出版该书的公司。该书出版后曾获两项科幻小说的最高奖。并创下了销售1000万册的纪录。
　　注：如果以上事例仍不足以说明专家的看法未必正确的话，我们不妨在这里再举一例，完全可以使那些未来的作家们不必灰心。1977年，自由作家查克・罗斯将一部刚打印完毕的杰西・柯辛斯基的著作《台阶》的稿子分别送交14家出版社和13个代理人。结果他们都没认出该书的作者系全美书籍奖获得者，大名鼎鼎的柯辛斯基之大作，而且也无一例外地拒绝出版此书。
