10位逃脱了惩罚的著名人物

　　1．B・特拉文（1882？―1969；1890？―1969；1894？―1969；1901？―1969）
　　这个自称为B・特拉文的人是世界上最了不起的逃亡者和变色龙，也是最出色的劳工阶级的小说家。他著有30本书，其中包括极为出名的《西拉・马德的财宝》。他至少有4次逃脱了政府的追捕。他开始是在德国作为一个演员起家的，是个无政府主义者，使用莱特・马鲁特的名字。1919年他在慕尼黑被送上法庭，罪名是叛国罪，最后被判处死刑，但是他逃跑了。他逃到莫斯科，跟随斯大林战斗；接着他又逃到法国。此后不久，他又玩起"死亡之船"的把戏，为获得美国保险公司的赔偿而将船只故意弄沉（他1934年一本最畅销的小说就是《死亡之船：一个美国水手的故事》）。1923年，他来到墨西哥，与沃伯勒一起组织了一场反对墨西哥联邦政府的暴动。他当时的名字是艾尔・鲁比奥，很快他就被宣布为整个国家的头号敌人。他在墨西哥四处躲藏，最后竟成功地摇身一变为小说家B・特拉文。除此之外，他还是一个享有盛名的摄影家，作为摄影家，他的名字是特拉文・托斯温。他又是一个受尊敬的考古学家。而在墨西哥的契亚帕斯印第安人中，他又成了一名伟大的医生。《生活杂志》曾以5千美元的代价，向人们征集有关这个捉摸不定的神秘人物的照片或资料。到了五十年代，他不再躲藏，而是结了婚，并定居在墨西哥城。他成功地抹去了自己早平生活的所有记录。在临去世前，他焚毁了各种私人文件和信件，但他的小说却十分清晰地向人们展示了这个自由斗士的传奇经历。

　　2．基姆・菲尔比（1912―　）
　　菲尔比出生在英国，原名哈罗德・艾德里安・菲尔比。1939年，他进入英国秘密情报局后，由于才华出众，很快就成为一名出色的间谍。二次大战期间，升任对苏情报处处长。在1949年，他成为英美情报交流系统的首脑人物。罗斯福、丘吉尔、杜鲁门等人都曾给予他高度评价，人们相信，不久他就将成为英国情报局的第一号人物。然而，没人知道菲尔比早在1933年就接受了苏联情报机关的训练，受雇当一名苏联间谍。1951年，英国秘密情报局发现他们的两名间谍，伯吉斯和麦克莱恩是双重间谍，同时在为俄国人工作。但是，他们两人在被逮捕之前，就成功地逃到了莫斯科。肯定有人向他们透露了他们受到怀疑的情报，菲尔比便是有嫌疑人员名单上的第一个。英国人花了10年时间想证明菲尔比是一个双重间谍，但一直未能成功。直到1961年，当菲尔比作为驻外记者在贝鲁特时（他于1955年离开情报机关），一名苏联间谍乔治・布莱克被捕，并且作证说，菲尔比一直在为苏联工作。但是在1963年1月23日，菲尔比在赴一个晚宴的途中，成功地摆脱了跟踪他的特工人员。然后，他乔装成一个阿拉伯人，步行了整整300英里，穿过叙利亚抵达土耳其。6个月之后，他在莫斯科重新露面。现在他与自己的苏联妻子尼娜一起过着平静的生活，他们住在苏联的某个地方，――这是历史上最成功的双重间谍。

　　3．冈特・沃尔拉夫（1942―　）
　　沃尔拉夫出生在德国的科隆，是世界上最优秀的新闻记者。他写了许多本书和无数文章，作为一名劳动阶级的斗士，他总是不遗余力地揭露各种丑恶的活动和交易：德国新纳粹不断地窃取政府权力；政府对人民实行的秘密监视措施；商业活动中心卑劣行径，等等。在4年内，他曾成功地扮演了各种角色：移民工人，不法之徒，精神病患者，土耳疲劳工，钢铁工人，邮局职员。在1976年，他扮成在波恩政府机构的代表，和葡萄牙的保守派将军安东尼奥・斯宾诺拉达成一项秘密的武器交易。1977年，他进入德国一份最大的报纸――右翼的《图平时代报》，撰写了许多反劳工的文章。在另一些场合，他又以巴伐利亚大主教随从的身份出现，或者是警方的密探、工厂厂主。他曾多次入狱，法庭也曾多次下令禁销他的书籍。他是一个真正的千面人，是欧洲最有影响力的记者，如今他也许住在瑞典，也许仍在德国过着地下生活。

　　4．哈里・霍迪尼（1874―1926）
　　霍迪尼不止一次地死里逃生，事实上几乎没有东西能将他禁锢。在手腕和脚腕都被绑住的情况下，他仍能从一个沉入水底的被锁住的大铁箱中逃脱。他也曾经从保险柜中、银行金库中和监狱地牢中逃出来，他确实是最伟大的逃跑表演艺术家。他身体的每个部位都灵巧异常，能用一只脚的脚趾将绳子打结或将绳结解开。在他死后50年，许多他玩过的把戏依然是个秘密。有人以为他甚至能从死神那儿逃脱，用一些神秘的方法与活在世上的人进行联系，他的几位密友对此坚信不疑。

　　5．弗朗西斯科・维拉（1877―1923）
　　维拉是一位具有传奇色彩的墨西哥革命者。当美国入侵墨西哥时，他率领军队痛击美国人。1916年3月9日，威尔逊总统命令潘兴将军追踪围剿维拉的军队，负责这次军事行动的是乔治・巴顿。但维拉却奇迹般地从重围中逃脱了出来。在墨西哥革命期间，维拉逃避了许多次追捕，但是在1923年7月20日，当他退役后在杜兰戈的农场里过着平静的生活时，却受到伏击，在汽车中饮弹身亡。

　　6．亨利・查利埃，"帕皮隆"（1907―1973）
　　他是少数几个能从法国魔鬼岛上的监狱里逃走的人之一。1931年，他被判犯有谋杀罪，化名帕皮隆的查利埃必须终身服苦役。他和另外几百个囚犯一起坐船押往法属几内亚的圣劳伦特，在那里他策划越狱，但最后又被抓住了。接着他被关进了魔鬼岛上的监狱。3年零9个月后，他成功地越狱逃跑了。他靠着绑在一齐的两口袋椰子，穿过鲨鱼出没的海域，游到了法国本土。1970年，他的故事成为一本畅销小说，法国政府撤销了最初对他的指控。

　　7．D．B．库珀（1926―　）
　　对于这位现代传奇人物我们知道得很少。1971年，D．B。库珀提着一只公文箱在洛杉矶登上了一架客机，他突然宣布说自己的箱子里装着一枚炸弹，要求政府给他20万美元。正是他的这次行动，才使美国的辞典中出现了"飞机劫持者”（Skyjacker）这一词汇。他让飞机在西雅图降落，在那里，他得到了20万美元和4顶降落伞。他放走了全部乘客，但依然扣留着机组人员，并下令让飞机向南飞行。联邦调查局的特工人员守候在西海岸的每个机场，所有的监视系统都紧盯着这架飞机。最后，他让飞机飞往华盛顿州降落，而他自己则顺利地跳伞逃走了。有官员宣称库珀已经在密林中丧生，但却拿不出任何证据。有一点很少为人所知，D．B．库珀曾在一次投资中丧失了所有的钱，也许正是为此，他才精心策划了这次劫机行动。

　　8．罗纳德・比格斯（1930―　）
　　他是英国最大的现代恶棍，看上去他过着一种极有魅力的生活。1963年8月8日，他和同伙一起抢劫了一辆皇家邮车，得到了730万美元，创下当时的世界纪录。在以后的1年里，伦敦警察破了这一抢劫案，比格斯一伙被送进了监狱。然而这笔钱中的大部分却神秘地失踪了。比格斯策划了一次成功的越狱行动，从英国看守最严的监狱里逃了出来。他来到巴西，与人同居并使其怀孕――根据巴西的法律――他成为巴西公民，不再会遭受引渡的危险。他的英国旗子也允许他和那个巴西女人结婚。在1978年，一个称为"性手枪"的摇滚乐团体将比格斯列为"1978年的红歌星"。接着，他又在一些影其中担任角色。天哪，全能的罗纳德・比格斯！

　　9．理查德・尼克松（1913―　）
　　尼克松出生于1913年1月9日，现在我们将他列入这份排行榜，是因为这位美国第三十七任总统在他的任期内发生了许多政治风波，其中包括臭名昭著的水门事件。最后，由于福特总统的赦免，他才幸运地躲过了法律的制裁。当然，由于所有这一切"政绩"，人们一定会在很久很久以后依然记得这位美国总统的。

　　10．无名氏
　　除了上述这些逃跑者以外，肯定还有不少最成功的逃跑者是我们尚不知道的。他们读到以上这些时，一定会露出会心的微笑，因为在他们眼中，以上这几位只能算是业余爱好者。
