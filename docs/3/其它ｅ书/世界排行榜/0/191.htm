10种最可能失败的企业

　　1．洗衣房和干洗店
　　1972年以来，家庭洗衣设备的改进以及合成织物的广泛使用使得整个洗衣行业发展缓慢，一些历史较长、声誉较好的洗衣店通过提供特别服务才得以站稳脚跟。

　　2．旧车经销行
　　银行一般不愿向旧车经销商或旧车的买主贷款，因为风险很大。估计该行业将面临衰退。

　　3．加油站
　　剧烈的竞争和微薄的利润使得这个昔日十分赚钱的行业变得令人生畏。一个加油站每年要做到180万元的营业额，才能挣到3万元钱的利润。

　　4．地区性货运公司
　　雇佣有工会作靠山的卡车司机成本昂贵，再加上政府制定的各项规定十分严格，使得这个行业充满风险，估计今后的发展将日趋缓慢。

　　5．餐厅
　　开设餐厅很有成功的希望，且利润不薄，约3．5％。但是在任何一个餐厅取得成功的同时，大概有十几家餐厅因缺乏管理知识而遭到失败。

　　6．婴儿服装店
　　目前婴儿人数不多，零售服装店，特别是单独开设的店正面临剧烈的竞争。

　　7．面包房
　　超级市场开设面包部使得独立的面包房日子艰难，一些面包房通过提供特别的服务才得以生存。

　　8．机械商店
　　全美总共约有5000多家这类商店，因而竞争激烈。每10万元的销售额中约能产生3，300元的税前利润。

　　9．肉店和食品店
　　此类商店如不增设诸如送货上门之类的服务项目的话，今后的处境将十分艰难。

　　10．汽车洗衣行
　　由于职工流动频繁，同行竞争激烈以及投资较大，很少有企业家对这个行业感兴趣。
