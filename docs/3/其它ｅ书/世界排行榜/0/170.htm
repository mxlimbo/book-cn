8个曾经带过（或可能带过）贞洁带的女人

　　贞洁带是男人用来防止自己妻子不忠的一种穿戴物，它被系在腰部，中间部分还连有一根从两条大腿之间穿过的带子，带子上垫着一块板把生殖器和肛门盖住，板上留着洞，正好可让大小便排泄出去。贞洁带上还装有一个锁，男人掌握着钥匙，他把女人的贞洁带锁好，然后带上钥匙可以完全放心地外出干活。这样一来在家的妻子无法同其他男人相好。不过假如这个男人是铁匠大概就例外了。有一则笑话：一次一个十字军士兵即将出征讨伐异国教徒，临行前他把自己妻子贞洁带的钥匙交给一个很亲密的朋友，可他离家还没多远就看到这个朋友急匆匆地赶上来，并大声叫道：“你给错了钥匙！”

　　1．凯瑟琳・亨利埃塔（1579-1633）
　　这个身材瘦小、举止优雅、很有心机的女侯爵，年仅20岁时就同法国国王亨利六世发生了大片大落的感情纠葛，亨利虽然爱她，但并没有给她所最想得到的东西――婚姻（他娶了玛丽・麦得西斯为妻）。他可能曾强迫年轻的凯瑟琳戴上贞洁带，国家图书馆收藏的一张蚀刻画可以说明这一点。根据十九世纪法国作家P.G.J.尼尔的解释，画中那个戴着贞洁带的裸体女郎就是凯瑟琳，那个接受凯瑟琳交出钥匙的男人就是亨利六世，躲在床背后的男人是凯瑟琳的情夫，他用一袋钱币从女佣那里换取了贞洁带钥匙的复制品，为了强调这个意思，画家还在画面上画了个小丑正在设法不让蜜蜂从篮子里逃出来，也不让一只猫看一只老鼠。画的下方写着以下的字：“一个拿着锁住妻子的钥匙却又戴绿帽子的丈夫。”

　　2．一个无名女子的尸体（16世纪末或17世纪初）
　　1889年一个名叫帕金格的慕尼黑文物收藏家来到奥地利的一个省城，当时那里正好挖掘出一座十五世纪的教堂，在教堂院子的一个角落里有一只铅制的棺材，打开棺盖，发觉里边有一个女人的尸体――估计她死时年纪还很轻（因她的牙齿仍完好），而且可能是个贵族（因为她身着价格昂贵的锦缎衣服），她那略带红色的头发梳成了辫子，两个戴着手套的手臂交叉放在胸前，掀开外衣，在她的盆骨处套着一副贞洁带――一只皮包的铁箍，前面连着一块带有锯齿状空隙的板和后面一块肛门板，板上也有一个洞。帕金格收藏了这副贞洁带，那个女人又被重新埋入土中。这女人究竟是谁？为什么被埋藏时还要套上贞洁带，难道她丈夫的忌妒心重到竟然妻子死了还不放心的地步，还怕她在地狱里会不忠吗？

　　3．奥地利的安妮（1601-1666）
　　在巴黎克勒尼博物馆内展出的5698号展品是法国路易十六的母亲、路易十三的妻子-奥地利的安妮所制作的。这副贞洁带的腰环是外面包着丝绒织物的金属做的，前面一块带有齿形空隙的弯形板是象牙做的，安妮同路易结婚时，两人都只有14岁，安妮是个金发美女，皮肤细嫩，可直到结婚4年后路易才同她睡觉。当路易对异性开始产生兴趣后，他的情欲十分高涨，且妒忌心很重，因此他就让安妮套上贞洁带。

　　4．B・女士　　（十八世纪）
　　法国作家伏尔泰22岁时，在他全身裸露的情妇身上发现了贞洁带。伏尔泰称这情妇为B・女士。伏尔泰的诗《锁》就是为她而写。诗的开头写道：“我胜利了，爱能支配一切，我正在接近那幸福的瞬间，这是我的幸福时刻，同样也是你的幸福时刻，可是一个暴君出现，他要破坏我们的幸福，他就是你的丈夫，他是性爱监狱的看守，他锁住了你充满魅力的自由神殿，他破坏了我们的激情，他用可恶的钥匙夺走了我们的欢乐。"这首诗接着叙述了冥府之神气卢托如何用金工之神维尔肯所制的贞洁环给皮罗舍蒂娜套上。

　　5．玛丽・拉琼（十八世纪）
　　所有我们知道有关玛丽和诱奸她的那个狡猾的AE？埃尔・伯赫的情况都是来自一个名叫弗里迪的律师1750年所写的一本薄薄的书，书中记载了弗里迪在奈姆法庭的审判中对伯赫的谴责。据说伯赫以娶玛丽为妻的许诺诱奸了年轻无知的玛丽，伯赫是个忌妒心极强的家伙，他设法让玛丽戴上在生殖器部位开口的贞洁环，并用锁锁上，为了保证绝对可靠，他又在贞洁环的接口涂上封蜡，在蜡上刻了他的印章，当他外出办事，他就把钥匙和印章带走，即使在他们的孩子出世后，他还强迫她用贞洁环。后来她明白他决不会娶她为妻，就向法院提出起诉，甚至在审判时伯赫仍随身带着钥匙和印章。弗里迪的发言充满了激情，对这个诱奸者进行了严厉的谴责，并对使用贞洁环这类行为进行了严厉的批判。他说：“他先是囚禁了这位年轻女子的心，接着又用铁皮锁住了她的身躯，他对待她简直比对待一个奴隶更为残忍，有什么样的行为能比给一个女子套上枷锁更为野蛮？比奴役她的身躯更野蛮？比用锁把她锁上更残忍？这种行为只有忌妒心最强的佛罗伦萨人才会仿效。"这件案子的结局不详。

　　6．奥迪娜・安西芬蒂（二十世纪初）
　　意大利雕塑家设计了一种贞洁环――一条四边缝入金属带的裤子，并配上锁扣，他设法让妻子穿上，只是在中午和下午8时妻子洗澡才让她脱下。一次他妻子同朋友高梅莉一起外出吃饭，突然她感到要大便，可她穿着这裤子无法大便，她便立即往家里跑，从丈夫那里拿到了钥匙，在女友高梅莉的帮助下，打开贞洁环的锁，然后冲进厕所，出来后再套上环，把锁锁上，将钥匙交给安西芬蒂，安西芬蒂对妻子的这种残忍行为后来终于使他遭到逮捕和起诉，但最终的结果并不清楚。

　　7．琼・帕拉特的妻子（二十世纪初）
　　多年来，帕拉特这个"使人痛苦的药店老板"一直被怀疑犯有虐待妻子的罪行，但始终没有证据，直到1910年的一天警方在帕拉特的寓所里发现他的妻子被链条锁在床边，脱掉她的外衣还发现她的身体被一件锁子甲锁住。帕拉特随即被逮捕。审判时，帕拉特极力为自己的行为辩护，他说自己主要是出于对妻子的行为不放心，此外他说这件锁子甲很长，并不妨碍她妻子弹钢琴。事发后，当时巴黎和伦敦报纸纷纷刊登该则消息，引起很大的轰动。3月5日巴黎一家报纸登载了一幅漫画，画的是帕拉特正拖着锁子甲锁住的妻子往前走。漫画下面的说明是：“这就是让妻子忠诚并牢牢依附于你的方法。"帕拉特最后被投入监狱。

　　8．亨利・利特里的妻子（二十世纪）
　　无论丈夫亨利如何打她，利特里太太总是不断地同别的男人发生关系。她曾经在3个月的时间里同3个男人相好，亨利的职业是面包师，但他对中世纪的历史很感兴趣，他从一本有关十字军的历史书中受到了启发，书中写道：一个十字军士兵出征前给他妻子套上贞洁环。亨利就到巴黎的克拉尼博物馆仔细观看了一只展出的贞洁环，回家后，他请一个骨科医生用金属和丝绒布为他妻子制作了一只类似的装置，在给她穿戴时，妻子反复叮嘱亨利"无论如何不要把钥匙丢了"。不久后，一个旧情人前来看她，把她衣服脱掉发现了这个贞洁环，他向警方报告了此事。当法院第一次传唤亨利时，他只以为是开玩笑，可第二次传唤时，他去了，于1934年1月21日他正式在巴黎出庭接受乔杜伊法官的审讯。他被指控犯有虐待罪，可当利特里太太在法庭上说她自己无法对丈夫忠诚后，法官对利特里作了很轻的判决：3个月的徒刑缓期执行，以及罚款50法郎。
