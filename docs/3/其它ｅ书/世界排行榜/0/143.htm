乔治·布兰达眼中10名橄榄球史上的最佳踢手

　　乔治·布兰达在全国橄榄球联盟所属的球队中踢球的时间长达26年，他的成绩为美国职业橄榄球史写下了光辉的一页。1949年他初次闯入芝加哥熊队，在整整20年后，他以最后一分钟球触底线得分和直接射门得分的超人技艺，为奥克兰袭击者队创造了常胜不败的奇迹。1975年47岁的布兰达离开球坛，他总共得分2，002分，成为全国橄榄球联盟有史以来的第一号得分手。

　　2．金诺·卡佩莱蒂——波士顿爱国者队
　　3．弗雷德·考克斯——明尼苏达维京人队
　　4．吉姆·特纳——丹佛布伦科斯队
　　5．简·斯特纳拉德——堪萨斯城首领队
　　6．吉姆·巴肯——圣路易斯红队
　　7．马克·莫斯利——华盛顿红皮肤队
　　8．汤姆·登普西——布法罗鸟嘴队
　　9．唐·科克罗夫特——克利夫兰布朗斯队
