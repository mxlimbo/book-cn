7位虽患中风仍取得事业成功的人

　　1．乔治·弗雷德里克·亨德尔（1685-1759）　德国-英国作曲家
　　1737年亨德尔52岁时中风。5年后他创作了《弥赛亚》，以后仍继续作曲，直到去世。

　　2．沃尔特·惠特曼（1819-1892）　美国诗人
　　1858年他第一次中风，身体完全恢复后，他对1855年第一次出版的《草叶集》进行了增补和修改。1873年第二次中风使惠特曼半身瘫痪，但他继续修订这本诗集，直到1881年才定稿。最后修订本在他去世的那年得以出版。

　　3．路易斯·巴斯德（1822-1895）　法国化学和微生物家
　　1868年的中风使他终身偏瘫。然而正是在他中风后，巴斯德通过大量的免疫研究，于1881年研制出炭疽病的疫苗，并在1885年研制出狂犬病疫苗。

　　4．温斯顿·丘吉尔（1874-1965）　英国首相和政治家
　　1949年第一次中风，1953年第二次中风，可他从1951年再度出任首相后，一直在此职工作到1955年辞职才停止。辞职后他又活跃在议院的讲坛，直至1959年。丘吉尔第二次中风后的主要成绩有：在他的政府的领导下，1955年英国研制出第一颗氢弹。以及1956年至1958年发表了他的4卷巨著《英语国家人民史》。

　　5．德怀特·艾森豪威尔（1890-1969）　美国总统
　　1955年中风后，他很快就完全康复。第二年他再次被选为总统，并完成这一任期。退休后他仍活跃在共和党的政坛，并写了3本书。艾森豪威尔是高尔夫球迷，1968年，也就是他中风后第13年和去世前的一年，他在一次球赛中打入了他那年的唯一的一洞。

　　6．埃伦·科比（1913-）　美国演员
　　1976年的一次中风，几乎使她丧失生命。但她在电视连续剧"沃尔顿一家"中塑造的沃尔顿奶奶的形象并未因此而永远消失。相反，编导把剧情改成了沃尔顿奶奶被送进了医院。这样在1978年，局部瘫痪和语言有困难的科比又重新出现在"沃尔顿一家"的电视剧中。

　　7．帕特里夏·尼尔（1926-）　美国演员
　　1965年尼尔受到一连串中风的打击，几近丧命。虽然第一次中风时，她正身怀六甲，可幸运的是她竟然顺利地产出了孩子。1968年她重上银幕。由于她在电影《玫瑰主题》中的出色表演，从而获得奥斯卡最佳女演员奖提名。
