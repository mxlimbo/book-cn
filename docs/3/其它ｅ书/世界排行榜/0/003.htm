12位著名人士他们改动了自己的出生日期

　　1．约翰·詹姆斯·奥杜邦　美国画家和自然主义作家（1785年4月26日—1851年1月27日）
　　约翰·奥杜邦喜欢隐瞒自己的出生日期。虽然事实上他出生于1785年4月26日，但他在海地的官方登记证上却写了出生于1780年5月4日，出生在新奥尔良。

　　2．汉弗莱·博加特　美国电影演员（1899年1月23日—1957年1月14日）
　　瓦纳兄弟电影公司宣称博加特是在1899年圣诞节出生的，其目的是使他的背景更具浪漫色彩。

　　3．詹姆斯·卡格内　美国电影演员（1899年7月17日—　）
　　电影公司的宣传部门把卡格内的出生日期后移到1904年，为的是更符合他那张娃娃脸。

　　4．乔治·科恩　美国作曲家和演员（1878年7月3日—1942年11月5日）
　　科恩的父亲将儿子的出生日期向后挪了1天，因为7月4日正是美国的国庆。

　　5．莫德·戈恩　爱尔兰爱国主义者（1865年12月20日—1953年4月27日）
　　这位爱国主义者鼓励并帮助威廉·巴特勒·耶茨创作了许多著名的诗篇。她说自己是出生于1866年——纯粹是出于尴尬无奈。她的父母是在她出生的前一天结婚的。

　　6．劳伦斯·哈维　英国演员（1928年10月1日—1973年11月25日）
　　哈维对自己的年龄说了谎，他说自己是1927年出生的，为的是在14岁那年进入皇家海军。

　　7．洛克·赫德森　美国电影演员（1925年11月17日—　）
　　在赫德森的早期电影生涯中，他的经纪人帮他把年龄增加了2岁，说成是1923年出生的，为的是让他去扮演一些更成熟的角色。

　　8．安德鲁·杰克逊　美国总统（1755年—1845年6月8日）
　　杰克逊更可能是于1755年在由爱尔兰驶往美国的船上出生的，而不是像许多传记上所称的是于1767年在南卡罗来纳州出生的。当然，如果杰克逊承认这一点的话，那他就无法成为总统了（总统必须出生于美国本土）。

　　9．维克托·麦克莱格伦　英国演员（1886年12月11日—1959年11月7日）
　　他给自己增添了4岁，说是出生于1882年。这样，他就成了伦敦救护队的一员，参加了英布战争。

　　10．洛拉·蒙蒂斯　爱尔兰舞蹈家，巴伐利亚路德维希一世的情妇（1818年—1861年1月17日）
　　她自称是西班牙舞蹈家，并且说是1823年出生在西班牙的塞维利亚，然而实际上她出生于1818年，出生地是爱尔兰的利默里克。

　　11．珀尔·怀特　美国默AE？女演员（1897年3月4日—1938年8月4日）
　　她一直显得非常年轻，然而她给自己的年龄加了8岁，将自己说成是1889年出生的，因为她不希望总是扮演孩子的角色。

　　12．奥斯卡·王尔德　爱尔兰剧作家和诗人（1854年10月16日—1900年11月30日）
　　王尔德将自己的出生日期改为1856年，仅仅是为了满足自己的虚荣心。
