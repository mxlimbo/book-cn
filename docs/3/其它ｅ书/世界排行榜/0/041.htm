天上掉下的25样东西

　　1．干草雨
　　1977年7月3日，英国的德维兹镇上空气来一团由干草聚成的乌云。下午喝茶时分，这团乌云来到了镇中心的上空后，立刻下起了"干草雨"，大把大把的干草从天上掉下来。除了这团"乌云"外，天上万里晴空，微风轻拂，气温是26°C。

　　2．金色的雨
　　1971年末，澳大利亚悉尼市郊外下起了一阵金黄色的雨，当时的卫生部长雅各先生宣称这是蜜蜂的分泌物，其主要成份是未消化的花蜜。不过，当时该地区未曾报道过发现大群蜜蜂，而且谁也无法解释为什么它们要把这种分泌物洒在悉尼。

　　3．黑卵
　　1786年5月5日，在一场持继5个月的干旱之后，海地首都太子港突然下起了一阵黑卵雨，有几颗黑卵因为掉在水里，所以没有砸破。第二天，里面的小生物破壳而出。这种动物蜕了一层又一层皮，样子挺像蝌蚪。

　　4．肉雨
　　著名的肯塔基"肉雨"发生在南部的巴斯县，时间是1876年3月3日。那天爱伦·克劳奇太太正在院子里煮场，突然晴朗的天空中撒下一阵肉雨，肉片的大小犹如大的雪花。有两位先生气尝了这种肉片的滋味后说，它们既不是羊肉也不是鹿肉。科学家们检查了肉片之后宣称，它们是马——或者是人的婴儿的肺部组织；另一些肉片经检验后被确认为软骨和肌肉组织。本地人对此事的解释为，一群兀鹰正从天上飞过，由于某种原因，它们一起把吃下的食物吐了出来。

　　5．一块重达3902磅的石块
　　有史以来世界上最大的陨石雨发生于1976年3月8日，当时成千上万块陨石降落在中国的吉林省，其中有100多块重逾100磅，其中最重的1块竟达3902磅，它比其他地方发现的最重陨石块重1000多磅。

　　6．钱币
　　1976年10月8日，一架轻型飞机嗡嗡地飞过罗马维尼西亚广场。突然，从飞机上撒下大把大把的500里拉、1000里拉和10000里拉的纸币，使广场里人们惊讶不已。这位发疯的财神爷究竟是谁，人们一直不得而知。

　　7．煤灰雨
　　1969年，一阵细细的煤灰雨降落在伦敦郊外的克兰福德公园，使公园看护人员大为恼火。然而据大伦敦市议会官方报告所称，这种"煤灰"实际上是一种黑色真菌的孢子。世界上只有一处地方发现过这种真菌——新西兰。

　　8．人类粪便
　　1978年4月23日，美国田纳西州利普瑞城的一所废弃的学校附近，突然有一块重达25磅的绿色冰块从天而降。联邦航空委员会宣称，这是从一架飞机上掉下的人粪便，该飞机的厕所显然漏了。不幸的是，这种自天而降的污物并不罕见，科罗拉多州的丹佛市便饱尝这种惊吓：至少有两户人家的屋顶挨过这种"绿色炸弹"的轰炸。据说还有一位不幸的农民，他在发现这种绿色炸弹究竟为何物之前曾经舔了它一口。

　　9．500只鸟
　　1977年11月末，大约500只死了的和濒死的画眉、鸽子突然降落在加利福尼亚的圣路易斯市街道上，没有发现附近有谁使用过杀虫剂，也没有人对这一现象作出解释。

　　10．火焰
　　1869年5月30日夜晚，德国格莱芬伯格及附近村庄的居民目睹天降大火，随之而来的是一阵震耳欲聋的霹雳。据当时在屋外的目击者报道，火焰的形状及颜色均与通常的闪电不同；他们还说，在几秒钟时间里，他们觉得自己被火焰团团包围，并感到没有空气可呼吸的窒息感。

　　11．白色的絮状物体
　　1977年10月11日，在加利福尼亚州的旧金山海湾，突然有一大片白花花的絮状物自天而降，其中最大的一片长约20英尺。飞行员确定它们是从4000英尺的高空落下来的，有人认为这与蜘蛛的迁徙有关，可是附近没有发现蜘蛛的踪影。

　　12．发光的绿雪
　　1953年，加利福尼亚夏斯塔山附近下了一场闪闪发光的绿雪。弥尔顿·莫耶夫妇报告说，他们一接触到这种雪后立刻觉得奇痒，随后手上、臂上和脸部都长出了"化脓、奇痒的疹疱"。原子能委员会否认这次绿雪与不久前在内华达州进行的核爆炸有任何联系。

　　13．神秘的文件
　　1973年7月25日，纽约州阿尔伯尼市《时代联盟报》的编辑报道了有关鲍勃·希尔的不寻常事件。希尔是北格林布什镇电台老板，那天下午4时15分，他正走出电台大楼外倒垃圾。这时他发现有一些"飞旋的片状物"从比电台300英尺天线更高的地方飘然而下。他紧紧追随其中的两件白色物体，直至它们降落在一堆干草上。原来这是两捆公式及其附图。从字面上看，这些公式解释了"正常化的灭绝"，以及"大卫格林斯坦的不完全方位"。官方未对这一事件作出任何公开解释。

　　14．豆子雨
　　巴西约奥帕苏亚市的朗歇·萨尔瓦多·塔吉诺报告说，1971年初，帕拉伊巴州他的庄园上空降落了一场豆子雨。据当地农业部门解释，一场暴风把非洲西部某处的大量豆子刮上了天，然后在巴西东北部撒落下来。塔吉诺煮了一些这种豆子，但据他说，它们硬得没法吃。

　　15．银币雨
　　1940年6月17日，苏联高尔基市附近突然降下一阵"银币雨"，银币总值数千卢布。官方解释是：一次山崩使一个地下宝库重见天日，当时正好刮来一阵龙卷风，把它们统统吹上天空，然后再撒在高尔基市。然而奇怪的是，这场银币雨中竟没有夹杂其他物品的碎片，这一点未获任何解释。

　　16．蘑菇状物体
　　1963年7月30日早晨，墨西哥市的交通曾一度受阻，因为当时从天上降下成千上万灰色的蘑菇状物体，它们从晴空万里的天上飘然至地。数千名目击者将这些物体形容为"巨大的蜘蛛网"、"棉花球"、"泡沫"，众说纷纭，莫衷一是。降落到地面之后，这些东西很快就分解了。

　　17．天降人尸
　　1978年9月25日一个星期一的早晨，玛丽·福勒和她8岁的儿子正坐在一辆停靠路边的汽车内，突然只听得"哗啦"一声，一具人的尸体突然穿破汽车的挡风玻璃砸了进来。这具尸体是从太平洋西南航空公司的一架班机中掉出来的，这架喷气客机在同另一架小型飞机相撞后不久便爆炸起火，这是美国历史上一次罕见的空难事故。汽车里的母子二人只受了轻伤。

　　18．蛤蟆
　　天降青蛙、蛤蟆，虽说不上司空见惯，但也不是绝无仅有，世界各地均有报道。其中最著名的一次要数法国的拉莱村。那是1794年的一个炎热的下午，突然倾盆大雨自天而降，当时正与奥地利人作战的150名法国士兵不得不放弃战壕，以免被大水淹没。在持续30分钟的大雨中，成千上万只小蛤蟆，以及蝌蚪，随雨点一起自天而降，落地后便朝四面八方蹦去。雨停之后，士兵们发现他们的帽沿及折缝处藏着不少蛤蟆。

　　19．橡树叶
　　1889年10月末，苏格兰登弗莱地方的教区牧师莱特先生被眼前的景像吓了一跳。起先他以为那是一群鸟，徐徐地降落在地上。他走近一看，才发现那是橡树叶子，这些树叶覆盖了1英里宽、2英里长的一片土地。最近的橡树林是在8英里之外，而且这批树叶中并未夹杂别的树的叶子。

　　20．洋苏木种子
　　1897年8月的一天，太阳即将下山，意大利马萨拉塔地方的上空突然布满血红色的细粒状云层。1小时之后，这些云层崩溃了，从天上降下无数种子。不久，地上便覆盖了一层半英寸厚的树种。不少种子已经发芽，它们全都是洋苏木的种子，这种植物主要见于中东和亚洲。当时降下的是清一色的洋苏木种子，未夹杂任何其他东西的碎片。

　　21．鱼雨
　　1974年2月，澳大利亚北方省的基拉内车站附近，突然有150尾银光闪闪的鲈鱼状的鱼从天而降。"鱼雨"并非十分罕见的自然现象，"官方"解释是：龙卷风造成一种虹吸现象，将海水和水中的鱼一起吸入空中，然后再将它们携往远处，撒落下来。

　　22．冰块
　　1965年2月，一块重达50磅的大冰块穿透了美国犹太州伍兹克洛斯市一家炼油厂的屋顶。弗兰克·爱德华在一本题为《稀奇之最》的书中，报道了一位德国杜塞尔多夫木工的不幸遭遇。他当时正在屋顶上工作，不料一块6英尺长、6英寸粗的冰柱从天而降，正好击中他，把他当场砸死。这是1951年的事。

　　23．桃雨
　　1961年7月12日，路易斯安那州歇莱佛堡附近突然乌云密布，随即便从天上降下了成千上万只未熟的桃子。

　　24．致命的白色粉末
　　1976年7月10日，意大利色维索市的居民们听见附近伊克密萨化工厂方向传来一阵响亮的呼啸声，人人都吓得大惊失色。随着呼啸声而来的是一团厚厚的、灰色的云朵，它朝着这个城镇滚滚涌来，并随即降下了一阵白色粉末。粉末粘在所有的物体上，它发出一股可怕的气味。10天之后，该地居民才得知，这种粉末中含有二氧芑，这是一种剧毒物质，它比砒霜和番木鳖硷更加致命。可是这已为时过晚，二氧芑的毒性已开始发作。该地区的所有人员全部撤出，用铁丝网围了起来，并宣布为污染区。该地的动物全部死绝，婴儿和孩子的皮肤上长出黑色疱疹，许多老年人死于肝病。该悲剧的严重后果目前尚不能得出明确结论。

　　25．太空垃圾
　　1962年，一块直径6英寸、重21磅的金属物体砰然降落在威斯康辛州马尼托沃克镇的一条大街上，钻入地面数英寸之深。后来这一物体被确认为苏联第四颗人造卫星的一部分，该卫星于1960年5月15日发射上天。自从1959年以来，先后有6000余块人造卫星和宇宙飞船的碎片从轨道里飞出来，其中有许多落到地球表面。1979年7月11日，77吨重的美国太空实验室脱离轨道后散落在南印度洋和西部澳大利亚一带。落到地面的最大残片是1吨重的水箱。
