10个日后在其他领域内更为出名的原橄榄球运动员

　　1．乔尼・麦克・布朗（1904-1974）
　　这名亚拉巴马州全美队的中卫于1926年代表第一支南方的橄榄球"浪潮队"前往罗斯博尔参加比赛，在同华盛顿队的一场关键比赛中他发挥出色，使"浪潮队"终于以20比19取胜。布朗非凡的表现引起了好莱坞明星发掘者们的注意。1926年他让布朗拍摄了第一部电影《军号声》。从此他开始了长达40年的电影生涯。总共在300部片子中担任角色。特别是在30年代，他以一个在同坏蛋的打斗中从不丢失帽子的英俊牛仔的形象而蜚声美国影坛。

　　2．厄斯金・考德威尔（1903-　）
　　考德威尔在自传中说道：“当他在厄斯金大学读一年级时，一些高年级学生一再动员他参加校榄橄球队。他发现自己一旦摸到了橄榄球就深深地爱上了它。考德威尔后来写了许多短篇故事和小说，其中包括《烟草路》和《上帝的小块土地》等作品，他用南方家乡的一些贫穷的人和黑人为原型，为自己的小说塑造了不少栩栩如生的人物。

　　3．德怀特・戴维・艾森豪威尔（1890-1969）
　　无论从何种角度来看，这位人称"堪萨斯旋风"的陆军橄榄球队后卫注定会成为一个非常杰出的橄榄球运动员。可是在第二个赛季中，一次意外的腿伤使得他不得不放弃这个他很有可能出人头地的职业。然而艾森豪威尔的前途还存在于其他方面。二次世界大战期间，他担任了西欧盟军总司令，后来又出任第34届美国总统。他对橄榄球的热情从未减退，在他的第二任总统期间，他成为第一个全国橄榄球基金会颁发的金勋章奖获得者。

　　4．吉拉尔德・福特（1913-　）
　　1931年至1935年福特在密执安州的橄榄球队担任中锋，表现出色。离队后，他拒绝了芝加哥熊队和底特律狮队让他转为职业球员的邀请，进入耶鲁大学读法律。读书期间，他还担任校拳击队和橄榄球队的教练。1941年他加入海军，退役后他进入政界，先是当选为众议员，后来是副总统，最后在1974年尼克松下台时，他出任美国总统。他的精湛球艺曾受到广泛的赞扬。1959年体育画报创刊25周年之际，将他AE？为全美优秀橄榄球运动员。

　　5．克利斯・克里斯托弗森（1936-　）
　　克利斯托弗森在波莫那学院读书期间（1954-1957）参加院橄榄球队。由于他在历次比赛中的杰出表现，他获得了校方授于的罗得奖学金，这使得他有机会在毕业后到英国牛津大学继续深造。然而他真正感兴趣的并不是橄榄球，而是唱歌和歌曲创作。70年代初期，他成为一名出色的西部乡村音乐歌手。与此同时，他还从事表演艺术，他主演的一系列电影使他名声大增，其中包括《爱丽斯不再住在这里》，《巨星问世》以及一部反映职业橄榄球运动的电影，名叫《粗野》。

　　6．理查德・尼克松　（1913-　）
　　在任总统期间，尼克松有一个不太好的名声，即他对橄榄球运动的兴趣几近狂热，在一些重大的比赛前，他常常直接打电话给那些职业球星，对他们指手画脚，告诉他们该怎么打，不该怎么打。然而尼克松本人的球艺却不敢恭维。他在威蒂学院（1930-1934）读一年级时，参加了一次初级橄榄球队的选拔赛，由于总共只有11个候选人，他很快就被选中。在以后的3年时间里，他在队里主要是担任一名替补衡撞手，因此比赛时常常坐冷板凳，不过他的教练称赞他是个“很坚强"的青年。如果说球艺一般的话，那么他在赛场上的“斗志却是非常顽强"。跟他原来同队的一名队员后来回忆道，尼克松由于求胜心切，几乎在每场比赛中都越位。

　　7．弗农・帕林顿（1871-1929）
　　这名优秀的美国学者是在十九世纪90年代初期在哈佛就读时第一次接触到这项"新型"的运动，后来他在埃波里亚学院教授英语期间，加入了学院非正式的橄榄球队。1897年他前往奥克拉荷马大学任教，在该校期间，他除了上课还兼任校橄榄球队主教练。他运用在哈佛学到的技术，把该队带成一支战绩彪炳的劲旅。该队连续3年的赛绩为9胜2负1平。1900年开始他潜心从事学习研究并写出巨著《美国思想的主流》，该书使他荣获1928年普利莱最佳历史著作奖。

　　8・弗雷德里克・雷明顿（1861-1909）
　　雷明顿在耶鲁读书时，是一名优秀的全能运动员，可他特别钟爱橄榄球。据说在1879年同普林斯顿大学的一场比赛前，为了使比赛看上去"非常紧张和激烈"，他特意从附近的屠宰场里弄来一些牛血涂在自己的球衣上。除了运动外，在大学期间，他还显示出超凡的艺术天才。后来他创作的那些优秀的美国西部写实画，使得他蜚声海内外。

　　9．保罗・罗伯逊（1898-1976）
　　这位才华横溢的艺术家第一次引起公众的注意是他在大学期间（1915-1918），由于他作为全美橄榄球队成员所表现出的精湛技艺以及作为进入全美优秀大学生行列的第三名黑人所获得的荣誉，他曾先后效力于哈蒙德、阿克伦以及密耳瓦基等职业橄榄球队。后来他离开球场，进入歌坛和演艺界，成为国际知名的歌唱家和演员。由于他同情共产党，因此在当时的美国是个很有争议的人物。然而他杰出的艺术成就使他继续光芒四射，继续受到大众的欢迎。在他73岁生日时，在卡内基大厅专门为他举办了一次盛大的庆祝晚会。

　　10．邦伦・怀特（1917-　）
　　作为本世纪上半叶美国最杰出的榄橄球运动员中的一员，科罗拉多州的"超级球星"怀特不仅被公认为全美最佳后卫，而且在为AE？兹堡海盗队（1939-1940）和底特律狮队（1940-1941）打球期间是第一个获得5位数年薪（15，800元）的职业选手。离开球坛后，他从事法律工作并效力于民主党，由于他在党内的出色工作，肯尼迪总统于1961年任命他为司法部副部长，并于第二年任命他为美国最高法院法官。
