12条有着奇特街名的美国街道

　　1．田纳西州查塔努加的灰姑娘街
　　它使我们想起童话世界里那个穿着水晶鞋的可爱的小姑娘。

　　2．缅因州波特兰市的菲森德恩街
　　它是为了纪念十九世纪缅因州的一位参议员威廉・皮特・菲森德恩而命名的，这人现在已不出名了，但该市有3条街分别以他的姓名命名：威廉街、皮特街和菲森德恩街。

　　3．马里兰州巴尔的摩市的机身大街
　　这条街是马丁制造公司修建的，由于这家公司是为轰炸机制造各种部件的，因此除了机身大街外，还有好几条他们修建的路被命名为活塞街、螺旋桨街、内燃机街，等等。

　　4．夏威夷州火奴鲁鲁的裤裆街
　　听起来带着点色情味，这正是典型的夏威夷古老的风格

　　5．加利福尼亚州圣迭戈市的哈维特路
　　这条街街名读起来很像"随你的便"――（haveit――yourway――），到了这里你可以自行其事了。

　　6．怀俄明州夏延市的懒惰的Y街
　　实际上"懒惰的Y"是一种牛的名称。

　　7．衣阿华州曼宁市的无名氏街
　　在给街道命名时，东西向的街用了第一、第二、第三……而南北向的街则用了姑娘的名字，如梅、苏、安妮等，最后大概想不出什么好听的名字了，所以干脆把它叫做无名氏街。

　　8．印第安那州汉诺威市的她街
　　由于这条街两侧都是一些妇女团体，所以就被人称作了"她街"，现在由于汉诺威大学的扩大，它也被包容进了大学内。

　　9．俄亥俄州托莱多市的第326街
　　想来托莱多市一定很大，但是纽约也只有第264街，芝加哥是第146街，而洛杉矶是第142街，看来它们都比不过托莱多的第326街。不过事实上托莱多只是个小城市，之所以将这一条街命名为326街，只是为了在街名数字上成为美国第一。

　　10．得克萨斯州奥斯汀市的2又1/2街
　　在得克萨斯，任何事情都在向大的方向发展，唯有这个州首府奥斯汀的这条街，却是用一个小小的分数来命名的。

　　11．明尼苏达州法利波特市的时间和地点街
　　根据传说，当年修建这条路的老板每天临收工时都会叮嘱筑路工人：明天在同一时间和同一地点上工。后来人们就索性把这条路说成"时间和地点街"。

　　12．加利福尼亚州里弗赛德市的王街
　　这是一个中国人的姓，这位王先生当年住在这条街的一幢房子里，他是个性情乖僻的隐士。
