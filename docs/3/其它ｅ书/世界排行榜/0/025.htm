12个美国军人出身的杀人犯

　　1．埃尔默・伯克　"扳机伯克"（1917―1958）
　　第二次世界大战期间，"扳机伯克"曾去意大利作战，他曾只身一人攻占德国人的一个机枪阵地。战后他成为一名职业杀手，6次成为谋杀嫌疑犯。当他枪杀一名酒吧侍者罪名成立后，在纽约州的新新监狱坐上了电椅。

　　2．迪安・艾伦・科勒（1940―1973）
　　他参军后在得克萨斯的胡特堡服役，但一年后就被军队开除了。几年后，他来到休斯顿，开了一家糖果店。1970年―1973年之间，他对至少27个少年进行了性虐待，并且残酷地将他们杀害。而他的下场则是被自己的两个10多岁的从犯杀死。

　　3．艾伯特・德萨尔沃　"波士顿的刽子手"（1931―1973）
　　17岁时他就开始服役，成为一名武装警察，到1956年退役。从1962年7月到1964年1月，他在波士顿谋杀了13名女子。1967年他被判处终身监禁，但是6年后他在监狱里被人用刀捅死了。

　　4．约翰・戴蒙德（1896―1931）
　　第一次世界大战期间他在美国军队服役，1919年因开小差被判了1年监禁。释放后，他成为纽约市的一名恶棍，曾5次由于谋杀嫌疑被捕，最后被他的同伙杀死。

　　5．约翰・德林杰（1903―1934）
　　年轻时曾因盗窃汽车被警察局拘捕，1923年加入美国海军，在"犹他"号舰上服役。1年后，德林杰在波士顿港跳水逃跑。他至少对10个人的死亡负有责任，被宣布为美国的头号通缉要犯。

　　6．欧内斯特・英格尼托（1925―　）
　　第二次世界大战爆发后他参加了美国陆军，1946年因在利文沃斯袭击2名军官而被开除军籍。1950年，他开枪射击自己的妻子和其他7个亲属。他的妻子幸存下来，但亲属中却有5人死去。被捕后，他被鉴定患有精神病，被判终身监禁。

　　7．爱德华・利昂斯基（1919―1942）
　　他是在美军驻澳大利亚基地服役的，1942年，他扼死了3个年轻的墨尔本女人。利昂斯基承认自己是"杰克博士和海德先生"（史蒂文森的一本小说名），他杀死了那些"想发出声音"的女人。最后他被美国军事法庭判处死刑。

　　8．杰弗里・麦克唐纳博士（1944―　）
　　1970年2月17日清晨，一伙刚吸过毒的歹徒冲进了北卡罗来纳州布拉格堡麦克唐纳博士的寓所，他们毒打了博士一顿，又用刀刺死了他26岁的妻子科莱特和2个女儿。麦克唐纳博士――美国军队的1名军医――对军方的调查人员就是这样说的。最后，军方同意他荣誉退役。但是他的岳父阿尔弗雷德・卡萨布，一个新泽西州的冷冻鸡蛋商人，在阅读了军方转来的调查报告后，认定他的女婿，这个被人称为"极富魅力的男人"正是杀害他女儿和2个外孙女的凶手。为此，他私自作了大量的调查。9年半后，麦克唐纳被带上法庭，并宣判有罪，被判处终身监禁。

　　9．卡尔・潘兹兰（1891―1930）
　　1907年，潘兹兰在蒙大拿州海伦娜市的一家酒馆喝酒时，结识了一个招募新兵的军官，由此参加了美国陆军。没过多久，他就被送上军事法庭，罪名是盗窃联邦财产，为此他被判3年监禁。释放之后，他从一个盗窃犯变成一个同性恋强奸杀人犯，在二十年代中他谋杀了21个人。

　　10．杰克・斯莱德（1824―1864）
　　他是在美墨战争期间加入美国陆军的。退役后他成为西部最出名的职业杀手，他的名声之大甚至掩盖了他的牺牲者的数字。马克・吐温在1861年曾见过斯莱德，他描写道："如此彬彬有礼、和蔼可亲的一个人……你很难将他和他那罪恶历史联系在一起。"1864年，他在蒙大拿被处以极刑。

　　11．霍华德・安鲁（1921―　）
　　他是美军坦克部队的机枪射手，曾在意大利和比利时作战，为此还获得过好几枚勋章，并在战后光荣退役。但是在1949年，他站在新泽西卡姆登市的大街上，在短短12分钟内枪杀了13个无辜者。

　　12．查尔斯・惠特曼（1941―1966）
　　查尔斯・惠特曼1959年成为一名海军陆战队士兵，1964年退役，在此期间他被同伴们称作"神枪手"。1966年7月31日晚，他杀害了自己的妻子和母亲。第二天早晨，他带着一包炸药走进奥斯汀的得克萨斯大学，在警察赶到之前，他引爆自杀，同时炸死15人，重伤31人。
