美国军人最不喜欢的25种食品

　　下面列出的是最不受军人喜欢的25种食物。它们的排列顺序是根据它们不受欢迎的程度，因此第一号酸乳就是军人最最不受欢迎的食物。
　　1．酸乳
　　2．脱脂乳
　　3．煎风草根
　　4．低热苏打水
　　5．甘蓝泥
　　6．法式煎胡萝卜
　　7．干梅汁
　　8．皮梅甫（罐装）
　　9．法式炸花菜
　　10．奶油洋葱
　　11．腰果色拉
　　12．烘南瓜
　　13．焖猪蹄
　　14．无花果（罐装）
　　15．胡萝卜、葡萄干和芹菜色拉
　　16．烤三明治
　　17．烧特瑞克＊
　　18．腌甜菜和洋葱色拉
　　19．葡萄干馅饼
　　20．鸡蛋汤
　　21．豌豆汤
　　22．猪肝和洋葱
　　23．奶油艾莫尔
　　24．猪肠
　　25．酸酪食品
　　＊　表示不存在的食物，提出的目的是想了解军人对不存在的食物的反应。结果对这些食物的反应竟然比某些存在的食物还更佳。值得提出的是第三种不存在的食物竟然排在最后40名内，处于芽甘蓝和油炸秋葵荚中间。
