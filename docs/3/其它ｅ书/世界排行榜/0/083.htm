海伦・海斯眼中的10名最佳舞台女性角色

　　这名当代的杰出的演员，9岁就开始在百老汇登台演出，她曾在一系列著名的舞台剧中担任角色，如《苏格兰的玛丽》、《维多利亚・里贾纳》和《韦斯特利的树林》等戏。由于她在电影《航空港》（1969）及《麦迪伦・克劳德特的罪孽》（1932）中的出色表演，她获得了奥斯卡金像奖。她儿子詹姆士・麦克阿瑟也是著名电视演员。
　　1．《卡密拉》中的玛格丽特・高蒂雅。
　　2．《玻璃动物园》中的阿曼达・温菲尔德。
　　3．《皆大欢喜》中的罗莎琳。
　　4．《第十二夜》中的维奥拉。
　　5．《我心中的佩格》中的佩格。
　　6．《米迪亚》中的米迪亚。
　　7．《圣人乔恩》中的乔恩。
　　8．《罗密欧和朱丽叶》中的朱丽叶。
　　9．《雨》中的萨迪・汤普森。
　　10．《进入黑夜的长行》中的玛丽・泰龙。
