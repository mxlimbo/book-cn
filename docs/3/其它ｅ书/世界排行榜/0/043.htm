10次生死攸关的日蚀和月蚀

　　1．公元前2640年——中国天文学家
　　古代中国人对日蚀有着自己的看法：他们认为那是恶龙吞吃了太阳。他们能够预知日蚀的发生，当这一时刻即将来临之际，人们敲锣打鼓，发出阵阵喧闹，意欲将恶龙赶走，恢复天地间的和谐。黄帝手下就设有两名专门预报日蚀的官吏，他们的职责除了测算日蚀的准确时间外，还包括组织人们进行驱赶恶龙的仪式。然而有一次，他们的预报不十分精确，在人们还未组织起来之前，恶龙就开始吞吃了太阳。眼看太阳就要被恶龙吃掉了，可是，由于某种运气——也许是因为那天恶龙已经吃饱了，所以它把吞下的太阳重新吐了出来，使它重放光明。但是黄帝对此毫不姑息，他彻底调查了这次事件，发现原来那两位天文学家因酗酒误报了日蚀。黄帝大怒——这种人命关天的事岂容如此疏忽，于是那两位官吏立刻被斩首示众。

　　2．公元前586年——吕底亚人和米堤亚人
　　吕底亚人（他们是希腊斯巴达人的盟友）和米堤亚人（他们由波斯居鲁士大帝统治）进行了一场旷日持久的战争。公元前586年——战争的第6个年头，这两个国家之间又爆发了一场恶战。当时正值5月28日白天，那天发生了日蚀。这次日蚀很早以前就被希腊数学家泰利斯测算出来了。据历史学家希罗多德记载：“正当两军鏖战犹酣时，白昼突然变成黑夜……吕底亚人和米堤亚人看到这一情景，立刻停止了战斗，极想达成和平协议。"双方都将这次日蚀视为上苍对这场战争的不满，故而这场持续6年的战争就此结束，数万生灵得以免遭涂炭。

　　3．公元前413年——雅典人的错误
　　由尼西亚斯率领的雅典大军和由杰利普斯率领的斯巴达军队，在西西里岛上进行了一场持久战。尼西亚斯和雅典人受到敌人攻击后，撤退至西拉库斯，那里有一支70艘船只组成的舰队在接应他们。尼西亚斯已经同意乘船离开西西里岛，可是在8月27日晚，天上出现了月蚀。祭司们立刻对尼西亚斯说：这是上天要求雅典人不要撤退，而应坚守阵地。尼西亚斯听从了他们的劝告，结果雅典人被杀得落花流水，70余艘战舰全部摧毁，这是伯罗奔尼撒战争的转折点。

　　4．公元840年——一位统治者被吓死
　　他就是查理曼大帝的第三个儿子，名叫虔诚者路易斯，因为他笃信基督教。他于公元813年加冕，成为神圣罗马皇帝，称路易斯一世。他撵走了父亲的成群妻妾，将道德败坏的姐妹们如数打入修道院，改革教会，并救济穷人。他还将部分国土分封给4位儿子中的3个。当他正打算封地给第四个儿子时，那3个儿子推翻了他的统治，并把他监禁起来，长达1年之久。但是由于他深得民心，所以路易斯一世又重登宝座。当一个儿子违反协定进攻萨克逊尼时，他率兵击退了这次入侵。据史书记载，正当路易斯班师回朝之际，他病倒在莱茵河上的一个小岛，不久便去世。然而还有另一种说法：这位路易斯一世是被5月5日的一次日蚀吓死的。他死后，帝国的国运日渐衰败。

　　5．公元1032年——上天的救援
　　图斯库伦的乔治伯爵是意大利最有权势的领主之一。他决心要控制教会，并不失时机地运用自己的私人军队和财富，使他年方14的孙子泰奥菲拉克特当选为教皇奔尼迪克特九世。这位年幼的教皇既自负又残忍，终日躲在拉特朗宫中，唯恐罗马人把他逐下教皇宝座。后来，有一次重要的节庆来临，教皇必须亲自在公众前露面。教皇的政敌策划了一个在圣彼得的巴西里加宫行刺他的方案。他们怕身藏兵器容易被人发现，所以打算携带几节绳子。具体计划是先由一些人制造骚乱，转移卫士的注意，然后再冲向教皇，用绳子将他勒死，并且趁混乱之际逃之夭夭。当这一天来临时，教皇的敌人混入公众中，正打算执行计划。这时，据高尔·格拉伯记载：“早晨6点左右，天上发生了日蚀，它一直持续到8时，所有人的脸都苍白得像死人，周围的一切东西都像是浸透了黄色和桔红色。"在这漆黑暗与混乱中，密谋者们犹豫了，他们的计划被教皇识破，他立刻回到宫里。这位教皇在一生中曾断断续续4次登上基督世界的宝座，直到1048年7月才被废黜，并在一处修道院里终其一生。

　　6．1504年——魔法师哥伦布
　　哥伦布第四次去新世界时，带了两条损坏严重的船只，它们因超载严重，吃水太深，已十分危险。他不得不在牙买加附近停泊整整一年，假如得不到食物补充，两条船上的人都得饿死。但是牙买加土著执意不向白人出售食物。正当哥伦布一筹莫展之际，有一天他随手翻开一本《扎库多年鉴》，得知几天之后，也就是2月29日，会有一次月蚀。哥伦布灵机一动，在那天找来了土著的酋长，对他们说，如果再不给他食物，他就命月亮无光。这话遭到酋长奚落，然而就在那天晚上，月蚀如启发生，土著个个惊恐万状。哥伦布向他们说，只要提供食物，他就使月亮重新发光。土著连连答应，不多时月蚀结束，月面重新放光，土著酋长忙命大家把食物送到这位魔法师跟前。就这样，月蚀拯救了哥伦布和两船人的性命，使他们免于饿死。

　　7．1764年——冠军的诞生
　　这一年的一次日蚀过程中，英国坎伯兰公爵的马厩里出生了一匹栗色小马驹，它理所当然被命名为"日蚀"。它注定要成为历史上最出名的马。当公爵出售自己豢养的全部马匹时，"日蚀"以75几尼的价格落到了一位绵羊商人手里。在它4岁那年，"日蚀"首次参加赛马比赛，在4英里赛跑中一举夺魁。这骑马给著名赌客丹尼斯·欧克利留下了深刻印象，他不惜花了650几尼，买下了"日蚀"的一半所有权。后来"日蚀"接连赢得11次赛马冠军，欧克利又以1000几尼的价钱把这一半所有权脱手。"日蚀"简直所向披靡，当时有一位赛马迷这样写道：“日蚀从来不会输，从来不需要别人鞭打催促，也不需要马刺鼓劲，它从来不会被竞争对手的速度与力量吓倒——它永远一往无前，远远把对手抛在后面！"在长达两年的赛马生涯中，"日蚀"共赢得26项冠军，两年后它功成退役。"日蚀"死于1789年2月27日，它的一只蹄子被嵌上黄金，制成酒杯，陈列在纽马克特的赛马俱乐部，至今仍在。

　　8．1806年——印第安人的先知开口了
　　他的原名是拉拉威西卡（意为"响亮的声音"），但是后来人们都称他"坦斯克瓦塔瓦（"敞开的大门"）。他宣称自己有幻视，是个魔法师和预言家。他和兄弟一起，向密西西比河东岸的印第安部落游说，要他们联合起来，反对白人殖民者，摆脱白人习俗，恢复印第安人的传统。当时印第安那领地的总督威廉·哈里逊将军对坦斯克瓦塔瓦的影响感到十分不安，就写了一封信，派人送往印第安部落。信中写道："这位竟敢以造物主名义说话的家伙究竟是什么人？如果他真是一位先知，那么就让他令太阳停住、令月亮改变方向，令河流停止奔流，令死人从坟墓里爬出来。"坦斯克瓦塔瓦接受了挑战，他宣布1806年6月16日中午将有一次日蚀，届时，他将命令太阳失去光辉，黑暗降临尘世。日蚀果然如启发生，印第安人欢庆这位先知的胜利。然而另一位先知却铸下大错，他劝说印第安武士进攻哈里逊将军的部队，说他们不会受到子弹的伤害。结果这场进攻惨遭失败，四分之一英勇的武士死于枪口。1840年这位哈里逊将军当选美国总统，而那位先知则隐退至加拿大，靠英国人的年金度日，最后他死于堪萨斯州。

　　9．1879年——祖鲁大屠杀
　　一来是由于英国人和祖鲁人之间对特兰斯瓦尔有着领土争端，二来是由于英国人极想削弱祖鲁酋长色特瓦约的势力，所以他们就对祖鲁国发动了战争：切尔姆斯福德勋爵率领了一支13000人的大军向色特瓦约的40000人的军队发动进攻。1月22日，一支英国部队在伊桑德尔瓦纳山上扎营。祖鲁人包围了这座山，并对英国军队发AE？进攻。英国人和他们的盟军，内塔尔的卡弗尔人四面受敌。正当战斗进行到最激烈的时候，下午一点零二分，天上出现日蚀。英国人起先以为他们可以利用祖鲁人的迷信来使他们退兵，可是这种念头大错特错了。祖鲁人说：“太阳变得很暗很暗，就像黑夜一样，”而祖鲁人喜欢在黑暗中作战。这次日蚀鼓励了他们一鼓足气击溃英国入侵者。3小时之后，英国人的营寨被荡为平地——1300名英国人和内塔尔的卡弗尔人全部丧生，头颅被砍下，肚子被剖开，为的是让他们的灵魂可以释放出来。几个月以后，英国人又派来大批增援部队，在乌隆迪一举击溃祖鲁人，并控制了祖鲁国，最终于1887年并吞了这个国家。

　　10．鲍里斯上尉的妙计
　　在非洲腹地，比属刚里旁边有一个独立的国家，名叫曼格贝图，全国有156万黑人，由耶姆比奥苏丹统治，这是个食人者的国家。2月18日清晨，一位比利时军官，上尉阿尔伯特·鲍里斯和20名士兵误入该国国境，被食人者俘虏。他们被押在死囚牢里，等着受死，这时鲍里斯上尉偶然翻阅了一下揣在口袋里的年鉴，得知那天晚上8时将有一次月蚀。鲍里斯上尉想出了一条计策，他命令看守向苏丹递送一张纸条，上面写着：“如果你敢杀害我的士兵，我就要杀死月亮，月亮的死亡就是你的末日！"当天晚上，苏丹来到牢房，要看看鲍里斯如何杀死月亮。这位上尉等到8时正，就指着月亮说："你的死期到了！"没过多久，月蚀开始了，月面渐渐昏暗无光。苏丹大惊失色，连忙跪下，要求上尉饶命。并说，只要上尉不要杀死月亮，他就将所有被俘者如数释放。鲍里斯要求苏丹承认比利时国王对他们国家的统治，苏丹一口答应。这时，上尉朝天空一挥手，月蚀结束了，月面重放光芒。鲍里斯上尉和他的士兵全都获释，4000平方英里的曼格贝图国及其属地成了比属刚果的一部分。
