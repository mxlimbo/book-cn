　　1977年罗克斯韦尔公司收到一笔113，400，000元的退款。这意味着普通的纳税人又要多承担一笔税款。美国钢铁公司也收到113，400，000元。那些大石油公司支付的税率仅为11％，而那些最大的银行的税率则更低，仅为7．1％。尽管法律规定公司的税率为48％，实际支付的平均税率为17．2％。只相当于一个年收入26150元的四口之家的应交税率。
