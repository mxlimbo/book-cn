6个完全不必存在的政府机构或职位

　　1．小企业管理局
　　事实上，只有百分之一的美国小企业得到过这个管理局的帮助。它在历史上却总是丑闻不断——卷入政治风波或经营管理混乱。即使是曾经得到过它帮助的那2，400家小企业，现在也只剩下30家了，而管理局每年的花费却高达17亿美元。

　　2．美国州际商务委员会
　　它唯一的作用似乎只是对来往于州际间的货运卡车增收税款并给它们制定各种各样的规章制度，使它们不能按时起程、抵达，或单程空驶，既浪费资金又浪费能源。更糟糕的是，委员会制定的州际间商务运费高得使许多卡车主望而生畏。这个委员会每年的经费需要6600万美元。

　　3．联邦赤字管理委员会
　　由于华盛顿的联邦赤字一年高于一年，政府便增设了这样一个委员会，来协助政府设法减少赤字。然而到1980年，联邦赤字非但没有减少，还猛增到830亿美元。

　　4．联邦政府的电影制作
　　谁也不知道美国政府每年究竟花费多少钱用于制作电影，但是自从第一次世界大战以来，政府大约已制作了10万部影片，从牙齿保健直到大豆的营养价值。据统计，关于牙齿的影片就有585部，其中至少有12部是教人们如何刷牙的。在美国联邦政府的健康、福利、教育和环境保护部门都有完全独立的制片机构和设备。众议院的一次调查证实，有45个政府部门同时在拍摄自己的电影，他们至少雇用了1461个雇员。联邦政府每年花在电影制作上的费用约为1亿5千万美元。

　　5．军方的仆人
　　五角大楼的500位将军总是把许多军方服务人员当成自己的仆人——马伕，私人秘书，厨师，侍者，司机，信差，打杂的，修剪草坪的，甚至还有管家。从这些将军目前的收入状况看，他们完全有能力自己来雇用仆人，就像一位老资格的房屋管理委员会主席在一次大发雷霆时所说的：“他们自己也可以去草坪上刈草呀！"目前这些军方仆人每年需要政府支付500万美元的薪水。

　　6．全国教育研究会
　　参议院的一个小组在对该研究会的机构设置、经费使用等进行调查之后，向参议院作证说：“取消这个研究会，而让教育部的下属机构负责教育研究工作，也许是最明智的选择了。"建立教育研究会的目的是为了促进教育制度和教育方法的改革更新，但它实际上似乎更注重研究教师们的政治倾向，或者大学目前的状况等问题，与建立它的初衷毫无关系。参议院的结论无疑是完全正确的：“该研究会对全国的教育体制改革和教育方法改进完全不起作用。"但是该研究会依然存在，每年的花费是8千万美元。
