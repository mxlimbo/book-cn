11位在40岁以后还生孩子的著名妇女

　　1．斯维特拉娜·阿丽路伊娃（1926-　）　斯大林的女儿
　　她于1970年嫁给建筑师威廉·威斯利·彼得斯。他俩的女儿生于1971年5月，该年斯维特拉娜已经45岁。

　　2．露茜儿·保尔（1911-　）　美国女演员和喜剧演员
　　她的第二个孩子黛茜·阿纳茨生于1953年，当时她已经42岁了。

　　3．伊丽莎白·巴列特·布朗宁（1806-1861）　英国诗人
　　她的独生子罗伯特·维特曼·布朗宁出生于1849年，当时她已是43岁。

　　4．伊冯·布拉斯威特·伯克（1932-　）　美国参议员
　　她的第一个孩子奥特姆·洛克森生于1973年，当时她是41岁。

　　5．罗莎琳·史密斯·卡特（1927-）　美国第一夫人
　　卡特太太的第四个孩子，也是她第一个女儿阿密，诞生于1967年10月19日，两个月前卡特太太过了40岁生日。

　　6．奥黛丽·赫本（1929-　）　美国女演员
　　她的第二个孩子路卡·多蒂生于1970年。当时赫本已经42岁。

　　7．罗丝·菲茨杰拉德·肯尼迪（1890-　）　美国总统约翰·菲茨
　　杰拉德·肯尼迪的母亲
　　她的幼子爱德华生于1932年，当时她已42岁。

　　8．爱丽丝·罗斯福·朗沃斯（1884-　）　美国总统西奥多尔·罗斯福的女儿
　　爱丽丝在与尼古拉斯·朗沃斯结婚18年后，于1925年她41岁时，生下一个女儿保罗琳娜。她声称为女儿取这个名字是为了纪念圣徒保罗。这位圣徒毕生不遗余力地提倡克己的美德。

　　9．玛莎·米歇尔（1918-1976）　美国总检察长约翰·米歇尔的妻子
　　她和约翰的独生女也叫玛莎，她诞生于1961年，当时米歇尔太太已经43岁。

　　10．玛格丽特·菲特勒·洛克菲勒（1926-　）　美国副总统尼尔逊·洛克菲勒的妻子
　　玛克·洛克菲勒是玛格丽特和尼尔逊所生的第二个孩子，也是洛克菲勒家的第6个孩子。他于1961年出世，当时他母亲已经41岁。

　　11．格洛亚娅·王德比尔特（1924-）　美国画家
　　她的第四个孩子安德森诞生于1967年，届年她已44岁。
