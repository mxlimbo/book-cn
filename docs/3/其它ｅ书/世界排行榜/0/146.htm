乔尼·朗登眼中的10名赛马史上最佳职业赛马师

　　乔尼在40年赛马生涯中总共获得高达2400万元的收入，特别是1943年他骑"弗里特伯爵"赛马取得三连冠的好成绩，在赛马总成绩上他仅次于威利·休梅克。1958年被选入赛马荣誉厅。1966年退出赛马场，从事驯马和养马工作。

　　4．戈登·理查兹　　　　9．达雷尔·麦克哈格
　　5．拉菲特·齐凯　　　　10．唐米德
