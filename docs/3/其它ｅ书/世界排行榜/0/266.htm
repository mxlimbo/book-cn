17个可能同野兽一起生活过的孩子

　　1-2．罗密勒斯和利麦斯（前八世纪）
　　据说这对孪生兄弟刚刚出世不久便被他们的叔叔扔到荒野，结果由一只狼把它们养大。过了若干年，一个牧羊人把他们救出。公元前753年他们创建了罗马市。学者们一直认为有关他们童年的经历是虚构的，但最近的研究表明，他们曾和动物一起生活过，因此有关他们俩的传说很可能有一定的真实性。

　　3．海赛的狼孩（1344）
　　1344年德国海赛王国的猎人抓到一个约7至12岁之间生活在荒野中的男孩。狼群给他找来食物，并挖洞让他睡，孩子用四肢在地上爬行，而且能一下跳很长的距离，人们把它当作怪物，而且强其他吃煮熟的食物。他在返回文明社会后不久便死去了。

　　4．立陶宛的熊孩（1661）
　　1661年在立陶宛的森林里，一群猎人发现一个同一群熊生活在一起的小男孩。尽管他用嘴咬，用手抓，表示反抗，但猎人们仍把他带走，孩子被带到波兰的华沙，并被取名为约瑟夫，可他仍然吃生肉和野草。后来约瑟夫学会了一些词汇，还在一家贵族家中当了仆人，但他始终没能摆脱像熊一样吼叫的习惯。

　　5．爱尔兰的羊孩（1672）
　　1672年，在爱尔兰南部的一座山上，一个男孩掉入了猎人们设的陷阱。自从他逃离父母后，就一直同一群羊一起生活，他虽然只吃野草和干草，却长得健康壮实。他被带到荷兰阿姆斯特丹由尼古拉斯·塔尔普医生照看，他一直没能学会说话，而且一直像羊那样发出叫声。

　　6．弗劳马克熊孩（1767）
　　1767年在匈牙利弗劳马克村附近的山上，当两位猎人开枪将一只熊打死时，当时同熊在一起的一个女孩向猎人发起攻击。这个身体健壮、个子高大的18岁女孩婴儿时就同熊在一起生活，她被带入人间后，由于拒绝穿衣服，只吃生肉和树皮，被送入卡普芬市的精神病院。

　　7．阿维龙的野孩（1800）
　　十九世纪几个猎人在法兰西阿维龙森林中抓到了一个自幼就单独在森林里生活的17岁男孩。人们给他取了个名字"维克托"他回归文明社会后生活并不愉快，因而他不时想逃跑。他起初还经常嗥叫、咬牙。但后来逐渐地适应了同人类一起生活。直到他40岁死去时，他只学会3个词。

　　8．迪那·桑尼查（1867）
　　1867年一队猎人在印度的米尼波里附近的山洞里发现同狼一起生活的男孩。他们把这个大约7岁的孩子带到阿格拉地区的塞坎德勒，还给他起了个名字迪那·桑尼查。可是这孩子不肯穿衣，而且经常在骨头上磨牙。他在孤儿院生活了28年，仍不会说话。1895年他死于肺结核，他学会的人类的一种习惯——抽烟——使他的疾病更加恶化。

　　9．威廉·米尔丁（1883）
　　一个很有趣但未能完全证实的故事是有关英国斯特里特汉姆第14代子爵威廉·米尔丁的生气（一个权威认为这个孩子的名字叫威廉·拉塞尔），1868年在西非海岸的一次海难使得这个当时年仅11岁的米尔丁同猩猩一起生活了整整15年才被发现并带回英国。埃德加·赖斯·伯勒斯可能就是受到米尔丁的身世的启发而创作出他最为著名的人物——人猿泰山。

　　10-11．阿尔玛和卡玛拉
　　1920年10月辛格神父在印度的密特那波村附近抓住了两个同狼群生活在一起的女孩。一个大约3岁，另一个约5岁。辛格称她们为阿尔玛和卡玛拉。她们除了偶尔发出嗥叫声外，平时一点不出声。她们用四肢爬行，喜欢吃生肉，回到文明社会一年后阿尔玛死了。卡玛拉于1929年死去，死前她只学会45个词。

　　12．泰桑西多
　　1933年12月，一位樵夫在萨尔瓦多的阿华查潘省的原始森林中抓住了一个约5岁左右的男孩。这个外号为泰桑西多的男孩从婴儿开始就单独生活。仅用野果和生鱼果腹。新闻记者厄尼·皮尔见到他后报道说，他刚来到人类社会时靠发出嗥叫声来同人类交际，而且经常咬人，向人发动袭击。最后他学会说话，并适应了人类的生活。

　　13．卡查山的豹孩（1938）
　　1938年一名英国运动员在印度北卡查山发现了一个8岁的男孩同一只母豹和它的一群小豹一同生活。孩子被送回父母那里，他是5年前被这只母豹叼走的。他虽然几乎看不见东西，但能利用他特别发达的嗅觉辨认人和物体。

　　14．叙利亚瞪羚孩（1946）
　　1946年一天一群坐着吉普的伊拉克士兵在叙利亚沙漠追赶一个同瞪羚一起生活的12岁男孩。他像瞪羚一样向前蹦跳，跑得比任何人都快。带到文明社会后，由于缺乏理解以及受到粗暴的对待，他多次逃跑，最后主人把他脚后跟的肌腱切断，从而他变成了残废。

　　15．撒哈拉瞪羚孩
　　1960年9月，巴斯克诗人琼·克劳德·阿曼在西撒哈拉沙漠区发现一个大约8岁的男孩同一群瞪羚一起生活。阿曼对他观察了整整2个月，他估计这孩子可能是撒哈拉游牧部落摩尔族中的一个孤儿。这孩子用四肢行走，吃草，刨根，似乎已被瞪羚群体完全接受。阿曼发觉他活得很愉快，因此决定不把他带走。

　　16．布隆迪的猴孩（1974）
　　1976年春季约翰内斯堡的《时代》杂志刊登了一则有关发现一个猴孩的故事。1974年一些布隆迪士兵报告他们抓住了一个同一群猴子生活在一起的男孩。医生们估计这孩子是在内战时丧失父母，后被这些猴子养大的。尽管人们千方百计地使他适应文明社会，但他总是摆脱不了猴子的习性和行为方式。同人交际时，喉咙里不断发出咕噜声，只吃水果和蔬菜。当哈伦·莱恩博士从他的老师、著名的行为心理学家斯金纳处了解到此事后，便同另一个心理学家皮拉德博士一同前往布隆迪，对该孩子进行研究。通过对他过去历史的认真调查，弄清了他曾在孤儿院生活过的基本情况，他们根据调查和分析的结果认为这男孩的古怪行为产生于某种孤僻心理，并非因为同猴子一起生活的缘故。

　　17．巴斯克尔（1974）
　　这个男孩被一个男子带到印度旁遮普邦萨坦齐市的天主教教会。该男子声称他是在一个森林里发现了此男孩同狼一平生活。男孩约3至4岁，全身长满毛发，在他的肘部、手心和膝部都长有骨痂。据约瑟夫神父说，这个被称为"巴斯克尔"的男孩5个月后便学会直立，在2年不到的时间里，便学会在教会附近干些杂事。他通过手势同人交谈。约瑟夫神父说他后来不再抓食活鸡，但一闻到血的味道仍会朝散发出气味的方向走去。巴斯克尔究竟是否同狼一起生活过至今尚未得到证实。
