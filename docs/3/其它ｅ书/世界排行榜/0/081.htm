伊萨克·斯特恩眼中的10名历史上最佳小提琴手

　　斯特恩生于俄国，在美国接受音乐教育，1931年他11岁时首次登台同旧金山交响乐团同台演出。40年代初，20多岁的斯特恩就已受到广泛的赞扬，从那时期，他就一直在美国和欧洲各地巡回演出。所到之处，掌声雷动。直到今天人们对他的热情仍依然如故。现在他是全国艺术委员会的成员，并连续多年担任纽约卡内基中心主任。
