10个著名的改变性别者

1．温迪·卡洛斯（原是沃尔特·卡洛斯，1939）　美国音乐家
　　卡洛斯以设计莫式声音复合装置而著名。他的第一张唱片集"通电的单身汉"在60年代后期售出100多万张。虽然卡洛斯事业上十分成功，可他个人生活却十分糟糕，他对自己长就一个男子的模样很感苦恼，1969年5月，他开始以一个女人的方式生活，他最后一次作为沃尔特露面是在1970年的一次演出。1971年他同斯坦利·康布里克合作为一部电影配完乐后，便完全消失。1年后他接受变性手术，在以后的7年时间里，她实际上过隐居生活。1979年情人节那天，她正式把名字改为温迪，并在同年5月的"花花公子"杂志中公开了她的真实情况。

2．卡纳莉·康恩（原是但尼·奥康纳，1949）美国歌唱家和新闻记者
　　奥康纳2岁时就觉得自己好像是女孩，而且经常穿他姐姐的衣服。由于他很想证明自己是男孩，他踢足球，同女友约会，甚至试图参加海军陆战队。21岁时在他成为丈夫和父亲1年后，他又决定重新确定自己的性别。由于康恩既具有男性又具有女性的生理特点，他还同一个男朋友同居，他通过打临工挣到足够钱于1971年接受了变性手术。她后来成为一个有成就的歌唱家和抒情诗人。至今她还担任好莱坞的记者和电视谈话节目的主持人。

3．葆拉·格劳斯曼（原是保尔·格劳斯曼，1919）　小学教师
　　1971年担任了31年之久的小学教师格劳斯曼改变了自己的性别，她以一个女子的身份重返新泽西州赛达希尔小学。学校董事会宣布将她解雇，理由是她的出现将在学生中产生"反面影响"。随之引起法律诉讼一直持续到1976年，当时美国最高法院拒绝受理此案，但新泽西州高级法院则判定格劳斯曼应该得到赔赏，因为学校董事会的顽固立场使格劳斯曼失去了做教师的资格，与此同时，格劳斯曼仍维持同自己妻子的婚姻关系，并积极扶养3个女儿，她依然保持自己长期来形成的对摩托车、南北战争和手枪三样东西的浓厚兴趣。

4．唐·兰格利·豪尔（原是高登·兰格利·豪尔，1929）美国作家
　　戴姆·玛格丽特·拉塞福非正式领养的儿子豪尔于1962年迁至南加州的查理斯顿，在那儿居住不久，他对古董的鉴赏能力得到当地的赞赏。他写的几本书也使他名声日增，可他有一些出名的怪脾气，例如他为自己宠爱的小动物初次公开露面而举办专场招待会。他所写的书主要是一些妇女的传记。他很喜欢把自己当成他笔下那些女性。1966年，他开始接受变性治疗，为期两年的治疗结束后，道恩嫁给一个22岁的黑人机修工。养母玛格利特对事情的进展很满意，但道恩的外婆却抱怨道：“我实在不希望道恩嫁给一个浸礼会教徒。”

5．德博拉·哈丁（原是奥斯丁·哈丁，1933）　美国大学教师
　　哈丁后来回忆道：“在学校时，男孩子在教室的一边，女孩子在另一边。我的目标是到女孩子那一边。"可实际上要作这样的调整是很困难的，他终于在五年级被学校开除。1953年，他加入海军，希望能被派往国外，最好是到沙漠地区，然后再悄悄地前往丹麦接受变性手术。但他整个服役期间始终耽在美国本土，后来他虽然结了婚并生了一个女儿，他仍旧在美国各地寻找能给他施行这类手术的医生。他有好几次甚至想自己动手阉割。1970年他终于实现了自己的愿望，在摩洛哥卡萨布兰卡动了手术。在过去的10年里，哈丁曾在电台和电视台出现过多次，还在各大学讲授有关变性的课程。

6．南希·亨特（原名不详，1927年）　美国记者
　　亨特曾结婚两次，生了3个孩子，可直到47岁那年他才不得不正视自己是两性人的事实。他曾参过军，读过大学。耶鲁大学英语系毕业后，他在"芝加哥论坛报"从事新闻特写，干得很出色。

7．克莉斯汀·乔根森（原是乔治·乔根森，1926）　美国军人
　　1953年2月13日当克莉斯汀走出纽约的国际机场时，突然发现有关她的新闻报道铺天盖地，有的报纸甚至刊登如下的通栏标题"离家男大兵、归来金发女"。克莉斯汀·乔根森是美国第一个受到广泛报道的、成功的变性人。乔根森过去总觉得自己生理上有问题，当他由于体重太轻（98磅）而被军队除名时，他便查阅有关医学资料，了解到男性和女性的身体都存在着雌雄激素。为了掌握更多有关的知识，他参加了一个医疗技术培训班，而且不经医生同意开始给自己服雌性激素。1950年他飞抵丹麦，由克里斯蒂安·汉堡医生给他施行了变性手术。回到美国后，面目一新的克莉斯汀利用自己意想不到的知名度，开始从事演员和夜总会招待员的职业。

8．简·莫里斯（原是詹姆斯·莫里斯，1927）　英国新闻记者
　　这个曾被称为"英国报界最有闯劲的人物之一"的莫里斯回忆起3岁时当他坐在母亲弹奏的钢琴下时，第一次感到自己应该是个女孩子才好。青年时代的莫里斯曾当过7年兵，还进入牛津大学读书，大学毕业后他开始做一个流动记者。尽管他也深深地恋爱过，并结了婚，生养了5个孩子，他始终对性生活不大感兴趣，他称之为"有点乏味"。然而当他26岁登上世界屋脊珠穆朗玛峰并因此而出名时，他体验了从未有过的兴奋和激动。以后的10年是他转性的过渡时期，他过着一种女性的生活，并于1972年在卡萨布兰卡正式接受手术。1960年，他还是男性时，写过一本杰出的书《威尼斯》，赞扬这座意大利城市，可后来变成女性后，她却写文章攻击这座城市。

9．祖德·巴顿（朱迪丝·巴顿，1940）　美国邮递员
　　祖德是已知为数很少的女性变男性的人中的一个。她16岁前一直是个像男孩一样的顽皮女孩，因此很少有人敢欺侮她。她把头发理成男式平头，举动同男孩十分相像，以致学校的教师劝她去看心理门诊。大学时学校向她提供4年的奖学金，条件是女学生必须穿裙子，可是巴顿宁肯没有奖学金，也不愿脱掉男式服装。大学毕业找工作时，她居住在一个中西部城市，也因她不穿女装而只给她安排在工厂做工。后来她听说了有关斯坦福大学开设的变性课程，并于1972年和1973年两年中接受了4次手术。"外科手术确实是个奇迹，”巴顿说道，"我一生中第一次感到自己是个完整的人"。目前他在邮局干活，另一方面还在读研究生课程，他打算当一名临床心理学专家。

10．雷内·理直兹（原是理查德·拉斯金特，1935）美国 网球运动员
　　大部分两性人总把自己看作是那种显现不明的性别，理查兹也是如此。1975年这位事业上很成功的眼科医生和网球运动员接受了变性手术，手术后她移居加利福尼亚州。当雷内获得在加州举办的第一届网球锦标赛冠军后，一个爱打听的记者了解到这个身高6英尺2英寸的运动员的历史，并在全国电视新闻中透露了这一情况，因此在她参加的下一个大赛中，比赛的组织者决定，除了25名选手被淘汰外，剩下的选手都必须接受性激素检查，这在网球史上尚属首次。理查兹对此提出诉讼。1977年纽约最高法院判定"医学上无可辩驳的证据证明此人确系女性"，理查兹打赢了官司。现在理查兹已不再行医，而是将她全部时间投入职业网球运动中去。
