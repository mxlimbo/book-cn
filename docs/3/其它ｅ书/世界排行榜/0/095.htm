8位曾经演过恐怖电影的著名影星

　　1．詹姆士·阿尼斯（《那个东西》，1951）
　　在该其中阿尼斯扮演一个来自外星的怪物——一个具有人脑的胡萝卜。他袭击了位于北极的一个空军基地。导演霍华德·霍克斯利用这个怪物在影片中编织了许多悬念，他采用怪物突然展示其正面的镜头，使观众感到毛骨悚然。最后作者巧妙地安排了一个结局，即由空军把这怪物烧死。此后，阿尼斯继续在大型电视连续剧《硝烟》中扮演主角迈特·迪龙。

　　2．汉佛莱·博加特（《X医生的复活》，1939）
　　在同华纳兄弟公司签订的合同期间，博加特被迫在一部恐怖其中担任主角。这是他的第一部，也是唯一的一部恐怖片。这部名叫《X医生的复活》的影片是几年前拍摄的一部默片的续剧。博加特扮演莫里斯·X医生——一名专杀幼儿的杀人狂。事发后，他被判处死刑，并送上了电椅。但死后的莫里斯阴魂不散，又得以复活。这时的莫里斯具有一种令人可怕的习惯，即他总是给别人抽血，不管别人是否愿意。当然最后正义终于战胜了邪恶，一名处于危险境地的漂亮姑娘被救了出来。X医生再次被送入坟墓，而且再也未能重返人间。

　　3．查尔斯·布朗森（《蜡像馆》，1953）
　　在影片中布朗森扮演一个长相丑陋的聋哑人，他是蜡像馆主人文森特·普莱斯的助理，一场大火把蜡像馆化为灰烬，并烧毁了普莱斯的双手。普莱斯变得精神错乱，由于他的双手无法制作精美的蜡像，普莱斯便在被他杀害的人身上涂蜡，以此作为蜡像，正当他们要向下一个目标——费莉斯·柯克下毒手时，执法人员及时赶到把这名女士救出。普莱斯也受到了法律的制裁。

　　4．迈克尔·兰顿（《我是一个少年狼人》，1957）
　　这是兰顿拍摄的第一部影片，在影片中他扮演一个青面獠牙、外形似狼的怪物，这个怪物是一个疯狂的医生将秘密储存的血清注入一个无辜的少年后变成的。拍完该片后，兰顿在影片《幸运》中扮演乔·卡特莱特。

　　5．斯蒂夫·麦奎因（《无用者》，1958）
　　影片中麦奎因扮演一个小镇上的普通少年，当他报告说看到外形如海蜇般的巨形怪物正在吃人时，人们都认为他在故意捣乱，因此没人理睬。这样，这只怪物就毫无抵挡地沿路把人一个接一个地吞食下去。但在这关键时刻，这个少年不计恩怨、挺身而出，设法营救镇上的人们。最后他同消防队员们一起把这怪物赶入一家餐馆，向它身上喷射冷却剂，使它很快结成冰块，然后运到北极。

　　6．杰克·尼科尔森（《乌鸦》，1963）
　　尼科尔森拍的第一部影片就是一部喜剧性恐怖片，同时参加该片演出的还有文特·布莱斯、波里斯·卡洛夫以及彼得·洛尔等。影片《乌鸦》讲的是魔法师克莱文医生同贝德洛医生一同前往邪恶的斯卡拉巴斯医生处寻找克莱文的AE？子莉奥诺。同行的还有克莱文的女儿和贝德洛的儿子（由尼科尔森扮演），两人坠入爱河。在莉奥诺的唆使下，斯卡拉巴斯把贝德洛变成了一只乌鸦，并提出要同克莱文一比高低，看谁的法术更强。结果克莱文取胜。不过最后他还是同意斯卡拉巴斯把对自己不忠的妻子带去。

　　7．唐纳尔德·苏瑟兰德（《行尸走肉的城堡》，1964）
　　一群流浪艺人在德莱戈伯爵（由克里斯多佛·李扮演）的城堡停留，奇怪的是接连几天好几个艺人被杀，尸体被做成了木乃伊。当地的警长（由苏瑟兰德扮演）想抓几个值得怀疑的流浪艺人了结此案。可艺人们不答应，他们同警长展开了搏斗。在混乱中，人们发现了德莱戈伯爵收藏的大量木乃伊。正当警长要逮捕伯爵时，伯爵不慎跌入了他自己设计的杀人机关。

　　8．罗伯特·沃恩（《洞穴少年》，后改名为《冲出黑暗》，1958）
　　该片是导演罗杰·科尔曼在50年代拍摄的，当时正兴起一个拍摄少年恐怖片的热潮。沃恩在影片中担任主角。这是一部典型的洞穴人同魔鬼斗争的故事，但结局很让人感到意外。沃恩不顾一些见多识广的人们的反对，前往河对岸的世界里去探险。他在那里看到了非常可怕的情形：由于原子战争，人类正处于文明的尽头，而不是文明的开端。
