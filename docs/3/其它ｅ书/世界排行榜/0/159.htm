20位有非婚生子的著名人物

　　1．英格丽·褒曼（1915-　）
　　英格丽·褒曼与意大利导演罗伯托·洛赛里尼的关系令美国人震惊。当她离开了结婚12载的丈夫前去意大利见洛赛里尼时，从好莱坞到华盛顿都爆发出一气愤怒的斥责声。参议员爱德温·约翰逊发表了长达1小时的激烈演说，称这位女演员是"自由爱情的崇拜者"，"是一股强大的邪恶影响"，“是好莱坞堕落的倡导者"。他们两人之间的儿子罗伯提诺于1950年出世，又引AE？一片谩骂声。

　　2．马龙·白兰度（1924-　）
　　1961年，正在拍平时的著名演员马龙·白兰度遇上一位19岁的餐馆洗碟女工塔莉塔。他俩的罗曼史使塔莉塔于1962年生了个儿子，取名为特霍图。1970年，他俩在未婚状况中又生了一个女儿，取名塔莉塔。

　　3．本万努托·切利尼（1500-1571）
　　1558年，切利尼尽管已经和情妇生有3个孩子：康丝坦莎、乔万尼以及一个未曾取名的夭折的儿子，他还是决定成为一名教士。后来这位意大利雕刻家尽管从教士的誓言中解脱出来，不过在他与情妇比埃拉合法结婚之前，他俩又生了3个孩子：乔万尼、伊丽莎白和丽普拉塔。

　　4．塞万提斯（1547-1616）
　　1584年，正当塞万提斯要和卡塔琳娜·帕拉索瓦成婚之际，他的情妇安娜·洛雅丝为他生了一个女儿：伊莎贝尔。这件丑闻于1605年旧事重提，那一年这位著名作家因涉及一宗贵族谋杀案而被捕入狱。

　　5．保罗·塞尚（1839-1906）
　　保罗·塞尚这位法国画家是靠父亲的微薄补贴而度日的。1860年，他使模特儿霍坦丝·菲吉成为他的情妇。这是塞尚一生中唯一一段恋情，其结果使他们于1872年有了一位孩子，名叫保罗。

　　6．赫南多·科尔特斯（1485-1547）
　　赫南多·科尔特斯是西班牙冒险家，他和他的翻译、顾问兼情妇多娜·玛丽娜——一位印第安贵妇人于1520年生有一子，取名为唐·马丁。后来，唐·马丁从父亲手中继承了100名奴隶和墨西哥的地产，西班牙王室封他为圣詹姆士骑士。

　　7．凯瑟琳·丹妮芙（1943-）
　　1970年，女演员凯瑟琳·丹妮芙和丈夫、摄影家大卫·贝利离婚后，生了她的第一个孩子，一个儿子，名叫克利斯蒂安。孩子的父亲是电影导演罗杰·瓦蒂姆。1972年，丹妮芙和电影演员马赛罗·马斯特洛安尼成为一位名叫切娅拉的女儿的双亲。

　　8．笛卡尔（1596-1650）
　　虽然这位法国哲学家从未结过婚，但是他与一位女仆于1635年生了一个女儿。这位名叫弗朗茜妮的女儿5岁时就不幸夭折，令笛卡尔悲痛万分。

　　9．大仲马（亚历山大·仲马）（1802-1870）
　　大仲马先后有许多情妇，其中一位是女裁缝玛丽·拉贝，他俩于1824年生有一子，取名亚历山大。后来大仲马又和女演员贝尔·克莱尔莎梅生有一女，取名玛丽。这位伟大的法国作家的能力并未随年高而衰退，他于1851年与另一情妇生了次子亨利·波依尔，并于1860年生了次女米茵拉·科第尔。

　　10．本杰明·富兰克林（1706-1790）
　　1731年就在富兰克林和德波拉·丽德婚后不久，他将一个刚出世的儿子威廉带回家中。虽然富兰克林从未透露这孩子的母亲是谁，但有证据表明，她不是个女仆就是个妓女。威廉被富兰克林家族收留后成了一位律师，并成为新泽西州州长。

　　11．伽利略（1564-1642）
　　意大利天文学家和物理学家伽利略和情妇玛丽娜·甘芭共同生活了10年，他俩共有3个孩子：弗吉妮亚，生于1600年；莉维亚，生于1601年；温桑奇奥，生于1606年。两位女儿后来都成了修女，儿子则在意大利的佛罗伦萨当律师。

　　12．歌德（1749-1832）
　　著名德国作家歌德和才华横溢的夏洛特·冯·斯泰因之间的恋情持续了10年之久，随后他又和比他小16岁的女子克莉丝蒂安同居。后者为他生了5个孩子，但其中只有儿子奥古斯特长大成人。最后歌德终于1806年与克莉丝蒂安正式成婚。

　　13．瓦伦·哈定（1865-1923）
　　1919年的参议员、后来的美国总统瓦伦·哈定和他的情妇南·布列顿生下一个女儿，名叫伊丽莎白·安妮。如今哈定的独生女已成为亨利·布利普希太太，她是位家庭主妇，住在南加利福尼亚。

　　14．乔治·哈里森（1943-　）
　　哈里森是英国音乐家，他和妻子结婚后8年，妻子帕蒂·波伊德离开了他，而投进他的密友、吉他演奏家埃立克·克莱普顿的怀抱。同年，哈里森认识了奥丽维亚·阿莉亚丝，她是一位有墨西哥血统的美国女子，在洛杉矶哈里逊开办的唱片公司当秘书。1978年，两人生了一个女儿，取名为达尼，意为"有钱人"。女儿出生后不久两人便正式结婚。

　　15．阿道夫·希特勒（1889-1945）
　　虽然不少接近希特勒的人都相信，他直到去世一直是位童男，但德国历史学家和传记作家维尔纳·梅瑟却宣称，希特勒于1918年第一次世界大战期间在法国威弗林村驻扎时，曾和一位法国农家姑娘有过18个月的恋情。后来，希特勒在野战医院中得知那位名叫夏洛特的姑娘生了一个儿子，她给他取名让·马利。1940年德国入侵法国后，希特勒命令盖世太保找到让·马利的下落，并让他在法国警署中身居高位。如今，希特勒的这个儿子是一个失业铁路工人，他住在法国的圣昆丁。

　　16．托马斯·杰弗逊（1743-1826）
　　从1790年至1805年，在杰弗逊任国务卿、副总统和总统期间，他和弗吉尼亚庄园中的女奴萨莉·赫明斯共生有5个儿子和2个女儿，他们是：汤姆、贝弗利、埃斯顿、艾迪、麦迪逊，两个女儿的名字一样，都叫哈丽埃特。其中有5个长大成人，按照当时的法律，他们都是杰弗逊的奴隶，不过杰弗逊最后使他们获得了自由。

　　17．本娜迪特·德芙琳·麦克阿里斯基（1947-　）
　　1971年，这位激进的罗马天主教教徒在任国会议员期间，曾生了一个女儿，名叫洛伊森。本娜迪特于1973年嫁给米歇尔·麦克阿里斯基，她从未透露过那位女儿的父亲是谁。

　　18．诺尔曼·梅勒（1923-　）
　　诺尔曼·梅勒是美国作家，他先后与6个女人生过8个孩子，他曾说：“我无法做爱而不生孩子。"当他与第四个妻子比芙莉离异后，如今和模特儿诺丽丝·邱奇以及他俩于1978年生的儿子约翰·布法罗住在一起。

　　19．伊凡·屠格涅夫（1818-1883）
　　1842年，当时屠格涅夫在文学方面尚未成名，他和他母亲的女仆佩拉亚平生了一个女儿。那女儿后来改名为波琳耐特，由屠格涅夫一直钟情的女演员波琳·维亚多特抚养。

　　20．洛奇·德·维加（1562-1635）
　　德·维加结过两次婚，不过这都没有妨碍他的恋情，这段浪漫史使这位作家于1599年有了一个儿子菲南多，孩子的母亲名叫佩莉莎。后来德·维加又和著名西班牙女演员米茜拉·德·路扬生了好几个子女，其中包括女儿玛西拉和儿子洛奇·费利克斯。
