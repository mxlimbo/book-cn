8条改变了人们生活的广告

　　1．一条出售鞋子的广告
　　美国南北战争期间的葛底斯堡战役之所以会打响，只是由于报纸上刊登的一条广告。在1863年6月底，南部联军的詹姆斯·佩蒂格将军正率领部下在宾西法尼亚州急行军，他们都已经衣装不整，大部分人的鞋子都穿坏了，只能光着脚行军。在路上，佩蒂格将军正巧看到一张葛底斯堡出版的报纸，上面有一条不起眼的广告，说在葛底斯堡的一家鞋店里有上好的靴子供出售。于是，佩蒂格将军立刻下令改道前往葛底斯堡——虽然这样他们得多走9英里路——将那家商店里的靴子全都买来。但是当他们朝葛底斯堡进军途中，被北军发现了，于是爆发了一场遭遇战。其后，双方投入了愈来愈多的兵力，使这场战役成为南北战争期间双方伤亡最为惨重的一场战役。最后，北军获得了胜利。但是，从一种特殊的角度来看，双方在这场战役中其实都是失败者。因为北军伤14497人，阵亡3070人，被俘和失踪是5434人；而南军伤12706人，阵亡2592人，被俘和失踪5150人。

　　2．一条招聘美国海军部长的广告
　　在1802年，"海军部长"还并不是一个很重要的职位，因为美国海军当时一共只有3艘小舰艇。当第一任海军部长本杰明·斯托达特结束自己的任期（1798-1801）后，托马斯·杰弗逊总统找不到适当的人选来担任这一职务。最后，在无奈的情况下，杰弗逊总统在报上刊登了一条招聘广告，希望通过此法找到一位有能力和资历来担任这个职务的人。很快，巴尔的摩市的市议员罗伯特·史密斯（1757-1842）前来应征，结果他被录用了。他的任期从1802年到1805年，但几个月之后，他又同时代理了司法部长一职，最后，作为海军部长他一直干到1809年。然后，他又当上了美国国务卿，只是由于和麦迪逊总统意见不合才辞去了这一职务。

　　3．一条广告创造出一个"恶魔”
　　威廉·皮拉特出生在伦敦，家庭希望他能成为一名外交官，而他自己却想当一个演员。从皇家学院毕业后，他移居加拿大，在那里当了一个农场工人。1910年当他23岁时，一天在无意中看到旧报纸上的一条招工广告，美国西雅图附近的一家剧院需要人手。于是他赶到西雅图，见到了剧院老板凯利。两个月之后，当他正在农场里砍木头时，收到了凯利的一封短信，说他已被录用。皮拉特将斧子砸在木头里，头也不回地离开了农场。他开始了自己的演剧生涯，并为自己取了个艺名：鲍里斯·卡罗夫。后来，他成为好莱坞的大明星，并在《弗兰康斯坦》中扮演了世界上最出色的恶魔的角色。

　　4．一条广告使我们看到了《马耳他之鹰》
　　在1916年晚些时候，22岁的高中辍学学生塞缪尔·戴西尔·哈梅特从报纸上看到一条广告，这改变了他的整个生活，在此之前，他曾在货轮上当过搬运工，也曾在铁路上干过活。那条广告是国家警察署招募警探的，哈梅特立刻前去报名，不久，他成为旧金山警察局的一名侦探。在8年之中，他侦破了许多后来变得很有名的案件，例如尼基·阿斯坦走私案，法蒂·阿布·克雷的强奸案等，他提升得很快，并积累了大量探案经验。从31岁起，他便开始创作侦探小说，所谓硬派侦探小说就是由他开了先河。他的《红色的收获》、《该诅咒的戴恩》都畅销一时。后来，他便写了《马耳他之鹰》以及《玻璃天空》、《削瘦的人》等著名侦探小说。他所创造的警探萨姆·斯佩德成为一个家喻户晓的人物。他死于1961年。

　　5．一条广告引发了一场有关猴子的审判
　　田纳西州通过了一道"布特勒法案"，禁止在任何公共学校和大学里教授达尔文的进化论。这项新法案是由一些三K党人提出来的，他们支持原教旨主义者的观点，只允许"人是上帝创造的；第一个女人是由亚当的肋骨所变成的"一说。纽约的人权同盟认为这条法律是不能容忍的，它违反了学术自由的原则，使传播科学成为犯罪。于是，人权同盟在《新闻报》上刊登了一条广告，宣称说：“谁如果由于在学校里教授达尔文学说而受到伤害和损失，我们将负责赔偿。"结果，在田纳西州的代顿市，一位教自然科学的教师约翰·托马斯·斯科普斯真的又重新开始在课堂上讲述达尔文的进化论了。不久，他遭到起诉。1925年7月，著名的"斯科普斯与猴子案件"的审判便开始了。人权同盟为被告请来了美国最著名的律师克拉伦斯·达罗；而担任起诉的检查官是美国前国务卿、并三次被提名为总统候选人的威廉·詹宁斯·布赖恩。新闻媒介涌入代顿市，11天中共有数百万字的新闻向公众披露这一案件的进展情况。最后，斯科普斯被判有罪，罚款100美元。但通过这次审判，布赖恩和那些坚持原教旨主义观点的人被公众视为傻瓜，因此从道义的角度看，胜诉的还是斯科普斯和纽约人权同盟。

　　6．一条广告拯救了一个囚犯的生命
　　1962年的5月2日，一条富有戏剧性的广告出现在旧金山的《巡视者》报上：“我不希望看到我的丈夫因为一桩并不是他犯的罪行而被送上电椅，因此我愿意为任何一个人提供10年服务，无论是厨娘、女仆还是园丁，只要他能为我的丈夫洗清罪责。"结果，当时旧金山最出名的律师文森特·哈勒南读到这条广告后便与写这条广告的格拉迪丝·基德联系，原来这位女士的丈夫罗伯特·李·基德正被控谋杀了一位古董商人。作为证据，警方在古董商的店里找到了一把留有基德指纹和血迹的古代宝剑。在案件审理中，哈勒南律师证明了那位遇害的古董商并不是被这把宝剑刺死的，况且剑上只有基德一个人的血迹。至于基德的指纹和血迹，律师解释说，那是因为基德和他的一位朋友曾去那家古董商店，他们俩开玩笑似地挥剑比划，不小心将手给弄破了，才有了血迹和指纹。陪审团在经过11个小时的讨论后，判定基德无罪。当然，哈勒南律师没有接受基德太太向他提供的10年服务。

　　7．一条广告使一个海军军官最后入主白宫
　　1946年，在第12届国会众议员选举中，加利福尼亚选区的杰里·沃里斯作为民主党的候选人参加竞选，他是一个富有而且善良的人。当时共和党找不出一个可以与之抗衡的候选人，于是便组织了一个"百人委员会"，来物色合适的候选人。他们在报上刊登了一条广告，其中有这样一段：“我们寻求一位众议员候选人，他可以没有任何政治经历，但必须有勇气去击败老资格的众议员。他可以是任何一个年轻人，但最好是曾在军队服役，而且受过良好教育的。"有8个人看见广告后前来应征，但委员会经过面试拒绝了他们。这时，一所大学的校长在看到广告后向百人委员会提议说，有一位年轻的律师，原先是海军军官，名叫理查德·尼克松的年轻人也许是比较合适的人选。委员会约见了尼克松，询问他是不是共和党人，尼克松回答说自己在1946年总统选举时投了托马斯·杜威（共和党人）一票。委员会经过辩论，以55对22普通过让尼克松成为共和党的众议员候选人。开始时，形势完全对沃里斯有利，因为他曾是罗斯福总统的好友，而且在加州深得人心，甚至连尼克松本人也很崇敬他。但是尼克松并没有因此而手软，正像西奥多·怀特后来所说，"从手段的恶毒和不负责任的角度看，任何一次竞选都及不上尼克松首次进行的众议员竞选。"尼克松称沃里斯是共产主义的推销员，以及诸如此类的种种污蔑。结果，尼克松以64784票对49431票击败了沃里斯，当上了众议员。他就是从这里踏上了政治之路，最后走进了白宫。但是最后他又因为"水门事件"而被迫辞去总统职务，这一点似乎从一开始就注定了。

　　8．一条自愿当雇佣兵的广告
　　丹尼尔·格尔哈特是从越南战场回国的退役军人，1975年当他33岁时失了业。他有妻子和4个孩子，但他没有足够的钱来维持生活和支付医药费。由于他唯一擅长的技能是作战，所以他于1976年1月在《士兵的命运》杂志上登了一条广告，说自己愿意成为任何政府或个人的雇佣兵。几个月之后，他被安哥拉的游击队雇用了。但他到非洲仅仅3天，就在一次战斗中被古巴人抓去了。他受了审并被判处死刑。当时的美国总统福特要求将他赦免，但却没有结果。1976年7月格尔哈特被处死了，抛下他的家庭和3万美元的欠债。
