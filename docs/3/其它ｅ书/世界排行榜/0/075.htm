13首出自意想不到的作曲家之手的热门歌曲

　　1．家、甜蜜的家（1823）
　　歌词作家和流浪演员约翰・霍华德・佩恩从未有过一个永久的住处。他曾说：“我常去巴黎、柏林、伦敦和其他城市，听到人们演唱或用手风琴演奏'家，甜蜜的家'。可我在那儿既无钱，又无栖身之地。”

　　2．家中的老人（1851）
　　当斯蒂芬・福斯特刚写好这首后来被人称为"斯瓦尼河"的歌时，他从未听说过斯瓦尼这个名称。他只是想在歌中加进几条南方河流的名称，有人提出用"雅祖"和"盆地"，可福斯特认为这些名称太俗。后来他哥哥查阅了地图册，向他推荐了位于佛罗里达州的苏瓦尼河。福斯特就把它改成“斯瓦尼河"，并写入了这首歌中。

　　3．迪克西（1860）
　　“迪克"是南北战争期间南方联军最常唱的歌曲。一些将军坚持要部队唱这首歌以提高士气。写这首歌的却是一个出生在北方俄亥俄州的诗人。名叫但・艾米特。他是在纽约的一间寓所居住期间创作了这首歌曲。

　　4．共和国战歌（1862）
　　这是南北战争中北方联军最爱唱的爱国歌曲。由一个名叫威廉・斯特菲的南方布道歌曲作者作的曲。抒情诗人朱丽亚・豪作的词。

　　5．黄金中的银线
　　这首被售出200万张唱片的歌曲的作者是哈特・丹克斯。他在一次偶然的机会中读到了刊登在一份威斯康星州的农场杂志上的一首诗，该诗描写一对幸福地欢度晚年的夫妇。丹克斯读后深受鼓舞，他便为该诗谱写了曲调。可是就在该歌曲公开发表一年后，丹克斯同自己的妻子离了婚。

　　6．我是多么的渴（1891）
　　这首深受酒鬼们喜爱的歌曲却是出自两名行为端正、生活检点的英国绅士――爱德华特・里姆鲍尔特和费利普・道德利奇。前者还是一名著名的学者。最初他们把此歌写成一首宗教歌曲，名叫"幸福的日子。"但是没过多久，那些酒馆的常客们便改动了歌词，加进了许多粗俗的词句。最后把歌名也改掉，变成"我是多么的渴"。

　　7．我在新罕布什尔的老家（1898）
　　这首歌颂幸福、宁静和舒适的农村生活的歌曲曾售出100万张唱片，一次居住在纽约东十四街一幢破旧的房子里的两名歌词作家收到了一份房租催款单，他们便在该催款单上涂涂改改地创作了这首歌。这两名作者的名字是：哈利・蒂尔泽和安德鲁・斯特林。

　　8．喂，我的宝贝（1899）
　　“喂，我的宝贝"是歌曲作家乔・霍华德所作的爱情歌曲中最有代表性的一首。他作的歌曲有"爱的梦幻曲"，"爱我吧"和"让我做你的最后一个情人"。他的歌曲似乎着重歌颂爱情的永恒和持久。可他本人却先后结婚9次。

　　9．带我去看球赛（1908）
　　在美国，无论是大人还是小孩，几乎无人不知这首歌曲的歌词。它被称作是全国最受欢迎的娱乐性歌曲，几乎每一本出版的歌曲集都将此歌列入。但该歌的作者阿尔伯特・蒂尔泽直到该歌发表整整20年后才第一次观看了棒球比赛，该歌的作曲是杰克・诺沃思。

　　10．把你的烦恼装入你的背包，微笑、微笑、微笑（1915）
　　在第一次世界大战期间，英军中举行最佳歌曲竞赛，奖励那些能振奋军人士气的歌曲，结果这首由英军陆军上士费利克斯・鲍威尔作曲的歌首次获奖。通俗音乐权威西格蒙特称该歌是"最具乐观主义色彩的歌曲"。但鲍威尔本人却于1942年自杀。该歌的歌词作者是乔治・阿瑟夫。

　　11．清晨的卡罗莱纳（1922）
　　该歌一发表便大获成功，而且至今仍受大众的喜欢。它曾被用作不下三部电影的插曲，并由第一流的歌手演唱。具有讽刺意义的是这首表现南北卡罗莱纳州美丽风光的歌的曲调作者沃尔特・唐纳尔森本人却从未到过这个州。该歌的歌词作者为戈斯・卡恩。

　　12．我的蓝色的天空（1927）
　　该歌灌制的各种唱片共售出1200万张，均由歌唱家奥斯汀演唱。这首歌颂婚姻幸福的歌曲曲调竟是由一名独身主义者所作。也就是沃尔特・唐纳尔森。歌词作者是乔治・怀廷。

　　13．全员游戏（1958）
　　这首曾为米高梅公司录制的摇滚歌曲是46年前由美国副总统查尔斯・道斯创作。当时他在卡尔文・柯立芝手下服役。最初该歌取名为《A大调之曲》。这首具有半古典色彩的曲调于1951年突然走红，1958年连续六星期雄踞排行榜榜首。到60年代该歌的唱片已售出100多万张。
