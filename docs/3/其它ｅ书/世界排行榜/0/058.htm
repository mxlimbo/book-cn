7句出处常被人弄错的名言

　　1．不一定每个憎恶孩子和狗的人都是坏人。
　　这不是W·C·菲尔兹说的，而是利奥·罗斯坦说的（是在一次晚餐上，他向客人们介绍菲尔兹时说：“不一定每个憎恶孩子和狗的人都是坏人。"）。

　　2．年轻人，到西部去！
　　这不是贺拉斯·格里利说的，而是约翰·索尔说的（他是在1851年为《特别快车》杂志撰文时写了这句话）。

　　3．每个人都谈论天气，但谁对天气都无能为力！
　　这不是马克·吐温说的，而是查尔斯·达德利·沃纳说的（他是报纸编辑，此语是他在1897年8月24日写在报纸上的）。

　　4．适者生存。
　　首先说这句话的并不是达尔文，而是赫伯特·斯宾塞（他是在《生物学原理》以及更早一些的著作中这样说的）。

　　5．统治最少的政府是最好的政府。
　　这不是托马斯·杰弗逊说的，而是亨利·戴维·梭罗说的（他是在《不唯命是从的公民》一文中写下这句话的）。

　　6．洁净整齐与上帝之神性已相距不远了。
　　最先说这句话的并不是《圣经》，而是约翰·韦斯利说的（在他的《论穿着》一书中）。

　　7．千里之行，始于足下。
　　这不是孔子说的，而是老子说的（见《老子》64章）。
