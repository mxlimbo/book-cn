8个令人惊讶的奥运会选手

　　1．詹姆士・康纳利
　　著名的海洋小说家，1896年雅典奥运会美国三级跳运动员。

　　2．A．C・吉尔伯特
　　发明家，1908年伦敦撑杆跳运动员。

　　3．尼罗
　　罗马帝国的皇帝，公元66年在希腊奥运会上参加马车比赛。

　　4．米什莱恩・奥斯特梅耶
　　钢琴演奏家，1948年伦敦奥运会法国铅球，铁饼，跳高运动员。

　　5．乔治・巴顿
　　二次大战时的美国将军，1912年斯特哥尔摩奥运会美国五项全能运动员。

　　6．保罗・伊格那亚奥・迪里凡尔伯爵
　　墨索里尼的财政部长，1920年安特卫普奥运会意大利击剑运动员。

　　7．本杰明・斯波克医生
　　儿科医生、作家、政治活动家，1924年巴黎奥运会划船运动员。

　　8．瓦拉斯达蒂斯
　　阿美尼亚国王，在公元390年的奥运会上参加拳击比赛。
