鲁道夫·瓦伦蒂诺谈关于一个完美女性的10项气质

　　鲁道夫·瓦伦蒂诺曾在《酋长》、《血与沙》以及《鹰》等爱情巨片中担任主演。他被许多影迷称作为20年代银幕中最伟大的情人。他死于1926年，当时在世界各地的影迷中引起了歇斯底里的反应。在他的葬礼中，一些影迷大哭大叫，有的甚至当场自杀。每年在他的逝世纪念日，总有成千上万个忠实的影迷聚集在他的墓前向他致意。
