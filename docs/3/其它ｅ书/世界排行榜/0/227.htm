历史上10位罗伯特·佩恩最愿意成为的人

　　佩恩生在英国，住在美国纽约。他是个大旅行家，曾在中国居住多年。他的广泛兴趣使他创作出多部有关亚洲、近东、古希腊罗马以及西班牙内战的书籍。然而，真正使他出名的则是他写的有关诸如达·芬奇、甘地、希特勒及斯大林等名人的传记。
