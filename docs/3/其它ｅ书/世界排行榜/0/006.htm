6位未能实现计划的天才

　　1．理查德・瓦格纳（1813―1883）　德国作曲家、文学家
　　在1856年，瓦格纳就开始着手于歌剧《胜利者》的创作。这部歌剧是以佛教徒的生活为背景的，描写一位少女爱上了一个佛教徒。少女前世曾是一位婆罗门的公主，鄙视所有其他种姓的人。在现世中，一位佛教徒帮助她抛弃了成见，以平等的方式与人相处。瓦格纳的大部分作品都流露出"超人"和种族主义的思想，但他的这部作品却批驳了种族或等级概念。在他生命的最后阶段，仍竭尽全力地想完成这部作品，但是他发现很难用音乐来表现它的主题，最后，这个充满宽容和高尚情感的爱情故事，是用文学形式表现出来的。

　　2．托马斯・爱迪生（1847―1931）　美国发明家
　　在1890年，作家乔治・拉斯洛普――作家霍桑的女婿――向爱迪生提议，合作创作一部科幻小说，其中的主人公就是爱迪生本人。爱迪生当时已经读过儒勒・凡尔纳的科幻小说，很是欣赏，因此就同意了拉斯洛普的提议。当拉斯洛普开始勾勒小说的框架时，一位出版商也对此表示了极大的兴趣，出资赞助小说的创作。小说当时定命为《进步》。爱迪生为小说写了一些笔记，其中包括他对二十世纪中叶的一些推测：化学、光学方面的进步；大量的塑料产品；用飞机进行运输和交战；去火星旅行。他还推测在培养了十一代之后，猩猩就会讲英语了。然而，爱迪生写小说的兴趣很快就被许多实际工作，例如选矿机械的设计所取代了。最后，他完全中止了与拉斯洛普的合作，为自己的文学生涯画了个句号。

　　3．约瑟夫・康拉德（1857―1924）　英国小说家
　　康拉德的传记作者和最亲密的朋友理查德・柯尔记述说，康拉德在他生命的最后几年中，脑子里总是萦绕着一部新小说的最初几章。小说要描述的是发生在某个东欧国家的事。他非常生动形象地向朋友描绘了一些场景，使听的人仿佛身临AE？境。据柯尔说：“'在一个大雪纷飞的冬日之夜，士兵们都聚集在一座灯火辉煌的王宫外面，他们围着篝火，在静静地等待着。王宫里，正在召开一个重大的会议，一个国家的命运将被决定……'当康拉德向我描绘这一场景时，我几乎能感觉到空气中正凝固着紧张的气氛，我还能感觉到那堆堆篝火的热量。”

　　4．阿瑟・柯南道尔（1859―1930）　英国作家
　　在他死前不久，曾设计出福尔摩斯故事的一个新情节。人们从他的手稿中看到了它：谋杀者踩着高跷去作案。但他未能完成这部作品。后来，这个主题被G.K．切斯特顿成功地写了出来，而他也是独立地想出这一情节的。

　　5．德彪西（1862―1918）　法国作曲家
　　德彪西有一部著名的歌剧《佩利亚斯和梅丽桑德》，将人类的虚幻境界和忧郁无望的处境表现得淋漓尽致。在此之后，他又开始构思另一部歌剧《乌舍庄园的末日》，是根据美国作家爱伦・坡的同名短篇小说改编的。它的主题表现了充满恐怖气氛的变态心理和行为，这很像德彪西当时的精神状态。到了1908年，他和都市歌剧团签订了合同，让其演出这部歌剧。当时，他已完成了该剧大部分的歌曲以及音乐的总体构思。但后来，他对这部歌剧的创作不再感兴趣，直至最终完全将它放弃了。另外，德彪西还放弃了另一部很有希望成功的歌剧，那是完全能和瓦格纳的《特里斯坦和依索尔德》相片美的另一部《特里斯坦》。最后，德彪西还染上了吸毒的恶习。

　　6．弗兰克・劳埃德・赖特（1869―1959）　美国建筑师
　　赖特最后的设计之一，也是他最大胆的设计，是一座528层楼的伊利诺斯大厦，大约有1英里（合1600米）高，这简直可以说是空间时代的创举。赖特完成了这座大厦的模型，有26英尺高。假如大厦真的建成，将比现在世界最高的建筑――芝加哥的希尔斯高塔（443米）还要高将近4倍！按照设计，这座锥形的建筑是用最轻质墙体构筑起来的，它的外部还有许多部用原子能驱动的电梯，能够在任何情况下高速运行。赖特计划在1957年建成大厦，造价约为1亿美元，其中的办公室可供13万名工作人员使用。赖特认为，要完全满足纽约市在办公室空间方面的需求，大概需要造上10座如此规模的大厦。他还认为，这样一来，就能在城市中开辟更多的公园绿地，有助于改善城市的居住环境。居然从建筑学的角度讲，赖特计划也许是可行的，但是投资者、建筑工程师、承包商和消防官员却从各自的角度发现了许多根本无法战胜的困难。
