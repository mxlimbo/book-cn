历史上6位罗纳德·里根最愿意成为的人

　　原电影明星里根在他担任电影演员公会主席期间，产生了对政治的兴趣。他从民主党转为共和党的支持者。1966年他轻而易举地当选为加州州长，并于1970年再度当选。里根于1968年开始争夺共和党的总统提名。1976年他差一点击败在职竞选的吉拉尔德·福特。1980年他当选美国总统。
　　1．亚当（要能看到世界的创始实在是太妙了）
　　2．科尔特斯
　　3．巴尔沃亚
　　4．刘易斯和／或克拉克
　　5．朱尼佩罗·塞拉神父
　　6．任何首先跨过大片原进入西部的人，我真羡慕那些最先看到这片人迹未至的新天地的人。
