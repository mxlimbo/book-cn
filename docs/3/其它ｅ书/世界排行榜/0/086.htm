诺顿·西蒙眼中历史上的11件最佳绘画作品

　　这名退休实业家是美国的首富之一，他也是一名极为出色的艺术收藏家，他收藏的艺术品堪称世界之最。其中包括许多世界大师的名作。早在1954年他就开始对收藏艺术品产生了兴趣，当时他曾购买了一幅高更、一幅波纳德和一幅毕加索的作品。
　　1．弗朗西斯科·泽巴伦的静物写生《柠檬，桔子和一朵玫瑰》（1633）
　　2．拉菲尔的《圣母，圣子和书》（1504）
　　3．伦勃朗的《艺术家儿子之像》（1653-1654）
　　4．梵高的《一个农民的画像》（1888）
　　5．巴萨诺的《逃入埃及》（1540）
　　6．乔凡尼·保罗的《圣母画像》（1427）
　　7．毕加索的《带书的女人》（1932）
　　8．塞尚的《花其中的郁金香》（1890-1892）
　　9－10．利比的《圣本尼迪克特和阿波罗神》（1483）和《圣保罗和弗雷迪阿诺》
　　11．德加的《洗衣妇》（1884）
　　西蒙补充说道：“碰巧的是，所有这些画都收藏在诺顿·西蒙博物馆（位于加州巴塞得那市，向公众开放）。”
