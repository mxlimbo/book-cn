后　记

　　在韦小宝面前我们是否有种自卑与不满足感？

　　在韦小宝身上我们是否看到了自我与他人的另一面？

　　我们不知道自己身上潜藏着多少智慧与能量，不知道自己能否如韦小宝一样纵横捭阖，把原本平庸得近乎卑贱的生命推向人生的极致，并演绎一场智慧与狡猾，正义与邪恶的游戏，把一生中的亮点与暗点都交给他人去评判。

　　在韦小宝的身上，我们的世界观是否变了？我们突然发现这个世界不是英雄俊杰所能主宰的，也不是按所谓的大儒侠士的设想而美好……

　　世界应该是有点邪恶、有点自卑、有点不知所以的糊涂小市民的。

　　在韦小宝身上，我们发现世界并不如那些社会学家和道德家所标榜的那么简单，那么容易操纵。

　　市井小民顽强的生命力，以及智慧与邪恶，再加人生经验的老练，蕴含了许多颠扑不破的真理，你不懂，我不懂，唯一个大字不识的小痞懂，真乃一人精！
