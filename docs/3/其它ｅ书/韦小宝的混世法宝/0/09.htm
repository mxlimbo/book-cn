第八章　不动声色的厚黑学

　　在我们的现实生活中，从电视到报刊的新闻媒介连篇累牍，大张旗鼓地渲染着成功者的形象。

　　我们不禁要问，他们的出身与我们都是一样的，无论知识结构还是到家庭出身，都与我们相差无几，他们成功了，我们为什么不能成功？

　　成功的人成功的感觉是一样的，可他们在成功中所采用的手段不是一样的！

　　但有些东西是成功的人必须要掌握的。

　　比如厚黑学。

　　厚黑学是一种极具实用价值的学说，它既可用来实现美好的目标，也可用于达到邪恶的目的。

　　从表面上看，厚黑学常常令人吃惊和反感，因为它对于罪犯如同对于圣人一样易被利用。

　　厚脸——盾牌！

　　黑心——利矛！

　　每个人都有自己的脸皮观念，这关系到自己的尊严和地位。

　　但有的人认为，面皮厚对别人的责难和非议无动于衷者才是了不起的人。这就演绎出了厚脸皮这一观念：

　　保护自己的自尊心免受别人恶言恶语的伤害，不理睬他人的风言冷语，善于运用厚脸来保护自己。

　　每一个人在实现自己的目标过程中，都需要脸皮厚才能把把怀疑撒在一边，不对自己的能力、动机心存疑惑，不怀疑自己的价值。自己在自己的眼里是尽善尽美的人。

　　人世间有一种脸皮厚的人由于极其自信而把信心灌输给他人，他们将他视为成功者，让他放手大干，获得成功。

　　在我们的现实生活中，厚脸皮者不见得非要独断专行，或者咄咄逼人，反之，生活中的厚脸皮者多是一些卑躬屈膝，唯唯诺诺的人。

　　这种人脸皮特厚，你打他的左脸还会把右脸给你打，这种人能随机应变，善于处事，且能置他人的所想所思于不顾。

　　韦小宝就是后者，他位拜伯爵，官封征远大将军，但他从没有咄咄逼人过，犹如生活在夹缝中的小草，威力很大，你饿不死他，砸不死他，他的生命力是那么顽强。

　　顽强得那些一等一的英雄大汉死的死、伤的伤，而他还安逸地活着。

　　韦小宝身上那件刀枪不入的背心就是一个很好的暗示，也是一个很好的寓言。

　　韦小宝的脸也有同样的承受能力，可见脸皮之厚。

　　黑心就犹如我们生活中的手术大夫一样，不对病人实施麻醉就实施手术，置病人的嚎啕和恳求于不顾。麻利而果断地切开肌肤是为了保证手术成功，他必须极端冷酷无情，对自己给患者造成的痛苦麻木不仁无动于衷。

　　心黑者大都目光不短浅，没有不必要的同情心。他不问代价，把注意力全部集中在目标上。

　　就犹如韦小宝杀瑞栋一般，一口水喷向瑞栋眼中，跟着身子前纵扑入他怀中，左手搂住他的头……一把短剑从他胸口直刺而下，直至小腹，剖了一道长长的口子。

　　心黑者有胆量面对失败，要想采取卓有成效的行动总要冒着失败的风险。

　　厚脸、黑心是同一事物的两个方面。某人具有厚脸的力量，能够不理睬众人的责备和非议，这种力量同时也是黑心的源泉。

　　正如运用黑心不可能不运用厚脸一样，运用厚脸亦不可能不运用黑心。

　　厚脸、黑心最粗野的状态是不带道德色彩的，它只关心如何达到他的目的。

　　换一句话说，为争取胜利，厚脸、黑心的人不惜一切代价。

　　厚脸、黑心犹如矛与盾一般，它是进攻与防守的武器，它本身不具有邪恶的属性。然而，人们可以运用厚脸、黑心，不惜一切代价，纯粹追求私利。

　　有一些人直观上擅于将厚脸、黑心作为实现他们自私自利动机的工具。对这些人来说，为了获胜，没有不能付出的代价，只要能够赢得胜利，他们可以不惜损害别人的利益。

　　我们无法把韦小宝划分到那一个层次，但他的厚、黑观，有其独特之处。这种独特与其说是理智的，还不如说是感性的。

　　这建立在感性上的东西，在实际生活中，可能更加实用，纸上谈兵的事与我们毕竟是有距离的。

有泪无泪就哭

　　眼泪是人体内的一种分泌物，人在高兴、悲伤和打哈欠时都会与鼻涕一起流出来。

　　鼻涕的重要性被人忽略，可眼泪的地位却在日益上升。

　　刘备用眼泪哭出了天下，可谓功劳之大。

　　现在天下太平，可眼泪并没有因此而退出舞台，反而使用更广泛了。眼泪也因此在逐渐失去它本来的意义。

　　因为今天的人心越来越强硬，越来越麻木，对生活的激情越来越淡漠……

　　眼泪就这样干涸了！

　　现在流泪的人是那些看言情小说的人与爱看电视剧的人，他们的眼睛还清亮，可悲可叹的是他们只为虚假的故事流泪，眼泪有什么珍贵性呢？

　　哭泣一旦失去她的纯洁性，再去看某一个人流泪的表演就索然无味了，同时还有几分受骗的感觉。

　　我们想到了韦小宝的哭。

　　这是一种脸皮厚的哭，也只有脸皮厚的人才哭得这么真诚、这么生动。

　　我认识一位画家，这画家有位老师，二人平生关系并不怎么样，说白了这位画家的老师并不怎么看重他，这位老师死了，画家比每个人都哭得伤心。为什么呢？很简单，他不过是在强调这位老师在自己心目中多么重要，以证明自己在老师心中多么重要。

　　反正死无对证，就哭他个昏天黑地吧！

　　这画家的脸皮经泪水一浸泡自然就变厚了。

　　在今天，没了真情只有脸皮厚才哭得出来，才哭得这么投入。

　　无论是真哭、假哭，还是大哭，或捂着嘴捏着鼻子哭，为了达到自己的目的而哭，那是需要勇气的。

　　在得胜山韦小宝和茅十八遭到盐枭的跟踪，茅十八又身负重伤，随时可能招来大队人马，怎么办？

　　韦小宝的哭发挥了作用，只见忽听得小孩（韦小宝）哭叫，一怔之下立时停步转身，只听得他大声哭叫："你怎么死了？"不由得又惊又喜。一人道："这恶贼死了？"另一个人道："他受伤很重，挨不住了。这小鬼如此哭泣，自然是死了。"……只听那小孩兀自捶胸顿足，放声号陶，一面叫道："老兄，你怎么忽然死了？那些贩私盐的追来，我怎抵挡得了？"

　　这是韦小宝的诡智，也是他的痞子精神。

　　这两个跟踪的盐枭上了韦小宝的当，自然是丢命了。

　　韦小宝很善于哭，他说："我妈要打我，鞭子还没上身，我已哭得死去活来，她鞭打时自然不会重了。"

　　这是韦小宝在那种坑蒙拐骗的环境中练出来的本事。

　　韦小宝最精彩的一哭是在五台山上，他与皇帝一块，皇帝见不着他当和尚的父亲。让韦小宝去叫方丈来打开门。

　　韦小宝也有了一番发挥：

　　韦小宝心想："若要本寺方丈来叫开了门，倒有威逼迫老皇爷之意，倒还是软求的好。"双手在胸口猛捶数下，跟着也大哭起来，一面干号，一面叫道："我在这世上是个没爹没娘的孤儿，孤苦伶仃的，没人疼我。做人还有什么乐趣？不如一头撞死倒还干净。"

　　假哭是他熟习的拿手本事，叫得几声眼泪便倾泻而出，哭得悲切异常。

　　韦小宝哭得这么旁若无人，可见也是一种勇气，可见脸皮够厚。

　　在我们生活中有这种拉下脸来说哭就哭，而且哭得没有一点破绽的人吗？

　　韦小宝的厚脸、心黑透着一种智慧。

　　心灵的智慧是人类生命之根，智慧只有一种，而智慧的运用则是无穷无尽的。

有腿，不妨多跪

　　厚脸、黑心斗士最大的本事是不动情感，处事泰然，敢于拼搏，能够摆脱伴随失败的情感的左右。

　　他的每一个举动都是为他的目标而服务的。

　　韦小宝的哭便是一例。

　　在上天赋予我们这具肉体时，他可能没想到有这么大的功能，有这样多的作用。

　　如我们的腿是用来走路的，有关节是便于屈伸，便于我们上坐下蹲，便于我们爬坡下坎。但双膝一弯却演绎出了更多的人生含义。

　　跪是对自己的自信心的一种考验，也是对尊严的一次检验。

　　那些自以为高人一等的人是不屑一跪的，是耻于一跪的，他们认为下跪是很丢面子的事儿。

　　有人在权贵面前下跪，在金钱面前下跪，可没人在朋友面前下跪。

　　韦小宝就爱在朋友面前下跪，一跪便成了莫逆之交、生死之交。

　　这一跪比起那些在危险面前下跪者要值。

　　韦小宝最生动的一跪在陈近南面前——

　　韦小宝微微仰头向他瞧去，见这人神色和蔼，但目光如电，直射过来，不由得吃了一惊，双膝一屈，便即拜倒。

　　这也许是韦小宝一生最真诚的一跪，最没有目的一跪，最谦虚的一跪。

　　韦小宝在这一跪之间便泄露出了内心状态，此人的威严震撼了他。

　　韦小宝心目中没有英雄，只有此人能穿透他的厚脸皮。

　　下跪有损自己的尊严，下跪者要有十分的勇气，同时要脸皮十分的厚才跪得真切，跪得不做作，才能使人产生十分的信任感。

　　下跪是一种最好的拍马屁方式，能使对方产生十分的自尊感，能轻易地麻痹对方。

　　下跪者同时是顽强的，这里面蕴含着栎树的坚硬与小草的柔顺。

　　在狂风中，小草弯曲，参天栎树连根拔起，可是不管风多大也不能把弯倒在地的小草连根拔起。

该出手时就出手

　　手是人最忠实的仆人，不管是给予还是索取，是邪恶还是善良，它都是按照着指令，准确无误照章办事。

　　韦小宝的撒石灰、捏卵蛋、斩脚背都是手所为，而且手一点不软。

　　如韦小宝杀瑞栋、往海老公背上插刀子。

　　再看——

　　匕首锋利无比，板壁不过一寸来厚，匕首刺去，如刀豆腐，跟着插入了喇嘛后心。那喇嘛大叫一声，身子软重，靠着板壁慢慢坐倒。韦小宝听得叫声，知己得手，走到第二名喇嘛后，又是一匕首刺出。转眼之间，如此连杀四人。匕首刃短，刺入后心之后并不从前胸穿出，每名喇嘛中剑坐倒，房中余人均不知他们如何身死。

　　可谓极端黑心，极端无情。

　　有一些人的脸上老是挂着甜蜜的笑容，渴望别人把他认可为好人。绝大多数人一眼就看穿他们的笑容只不过是皮笑肉不笑。这些人不过是红漆马桶，徒有其表，内心极端软弱。

　　厚黑者不会自认公正善良，不会渴望心满意足，也不会谋求他人赞同，在行动时，迅捷、果断、不受感情左右；在退让时，泰然自若，依然故我，任凭他人品头论足。

　　这便是一位地地道道、名符其实的厚脸、黑心实践家。

　　韦小宝便如此。

　　韦小宝的手不但能出人意料地取他人之命，而且还极会索钱。

　　为了他美丽的情人，他设了一个陷阱让郑克塽陷进去，郑克塽挨了一通揍，韦小宝又出面当好人，让郑克塽欠下了他一万两银子，这一万两银翻来翻去，郑克塽欠下了韦小宝三百八十万两整。

　　韦小宝砍断郑克塽的指头逼他写下欠条，郑克塽一贫如洗。

　　韦小宝一掷千金毫不吝啬，可该出手时决不手软，可见心之黑。

无所不能的嘴

　　几千年来，实践厚脸、黑心的中国大师们运用了五花八门、名目繁多的手法，制造诱"饵"驱使对方按照他们不可告人的计划行动。

　　在制造诱"饵"的过程中，有的使用金钱，有的使用权力，有的使用美女，这样的事例屡见不鲜。使用语言者也不并少，但韦小宝却用得奇特。

　　韦小宝的嘴可以什么都说，无话不说，但有时候他不说，一句话不说。

　　如韦小宝与康熙一道力擒权臣鳌拜，真正恢复了他的权力！"见王公大臣的神色对自己忽然不同，对自己恭顺敬畏得多。康熙直到此刻方知为君之乐，又向韦小宝瞧了一眼，见他缩在一角一言不发，心想：'这小子不多说话，难得。'"

　　韦小宝这样的表现，甚是出人意料，可又在意料之中。

　　韦小宝语言的诱"饵"是天下无敌，充分地显示了他语言的攻击性。

　　韦小宝语言的特点是：

　　不骂你便捧你。

　　许多英雄侠士用手中的剑去征服他的对手，韦小宝便用一张嘴去征服对手。

　　他第一次与江湖侠士的较量，便是对那帮盐贩子的一通臭骂，其刻毒凶狠，想不出是出自一位少年之口。

　　那些盐贩子不是茅十八的对手，被他打得落花流水，可这宫府出一千两银子的赏钱？四处捉拿江湖大盗却又不敌韦小宝一张嘴。

　　这韦小宝嘴上要义气有义气，要故事有故事，要奉承有奉承……

　　韦小宝的嘴可以说是四处出击，有横扫千军之势。

　　如韦小宝护送公主赴云南，赐婚平西王世子吴应熊。韦小宝所面对的是连皇帝也不怎么放在眼里，时刻想造反的吴三桂。

　　"鞑子皇帝对吴三桂如此宠幸，一时是扳他不倒的。不过这实是大好机会。吴三挂这奸贼不造反，咱们要激得他造反，激不成功，就冤枉他造反……"

　　吴三桂老奸巨猾，手中又握有重兵，见多识广，有很好的心理素质，韦小宝如何让自己这张嘴去派上用场呢？

　　韦小宝与吴三桂一见面，酒过三巡后，韦小宝的那张嘴便活动开了——

　　"王爷，在北京时，常听人说你要造反……今日来到贵府，才知那些人都是胡说八道……是啊，我想你造反，也不过是想做皇帝。可是皇上的宫殿没你华丽，衣服没你漂亮。皇上的饭食向来是我一手经办，惭愧得紧，也没你王府的美味。你做平西王可比皇上舒服多哪，又何必去做皇帝？待我回到北京就跟皇上说，平西王是决计不反的，就是请你做皇帝，您老人家也万万不干。"

　　一时之间，大厅上一片寂静，百官停杯不饮，怔怔地听着他不伦不类的一番话，心中都怦怦乱跳。吴三桂更是脸上一阵红、一阵白，不知如可回答才是……

　　这番话可见韦小宝的嘴的厉害，吴三桂何许人也，也在这番话下尴尬万分。

　　这是韦小宝的本事。
