周公解梦



是吉兆，表示不久将生。若梦见老鹰在空中飞，则可能得到宝物。梦到鸽，将会出人头地有所作为。


老虎是凶兆，表示身旁的人将会生病，或遇火灾。


梦到马，象征幸运。会因外来的因素而发生令人幸喜的事。



是吉利的梦。将成为继成人或富翁。

梦见小狗，表示会和邻居、同事、学校的朋友为了小事发生口角。还有梦见被狗咬，则表示将于朋友发生争夺，可能会有被欺诈杀害危险,是破费大笔钱财的前兆。


暗示你最近诸事不顺，交友运也大衰，朋友对你口是心非，为了利益还暗中算计，利用你，堕入爱河的男子也常常为一些小事发生口角。

梦见天鹅在蔚蓝的湖面上游荡，表示你的初恋不成功，但因为很美丽，给你留下了终生难忘的印象，至今心有余痛。梦见天鹅在空中飞翔，暗示快乐的日子就要来临。

狗熊象征你事业上的障碍，如果梦中将熊杀死，你会克服工作中的障碍，取得预想的成绩。梦见黑熊向你袭击，工作中会出现很难克服的障碍，有时也预示会有疾病降临。
