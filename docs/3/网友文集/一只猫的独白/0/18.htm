你能陪我多久

你说，我陪你……
那么，你能陪我多久，
一晚，两晚，还是一辈子？

你说，我会一直陪着你，
那么，一直有多长？
是不是只是到天亮？

这个世界上，还有谁有勇气陪伴一个人一辈子？

如果你注定要离去，那么不要再说
就像不知道什么是温暖的人不会害怕寒冷
没有过陪伴的灵魂不知什么是孤独

如果你执意要留下
那么告诉我那将是多久
因为我知道自己
对诱惑的抗拒注定失败。
也将从此无法逃开
给我一个期限
让我在寒流之前能把自己保护好

这个世界上，还有谁有勇气等待一个人一辈子？

你说，要陪我
那么我问，你能陪我多久？
