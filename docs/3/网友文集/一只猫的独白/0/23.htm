无话可说

（一）

摘下墨镜的瞬间
发现阳光倾泻了一地
只是在极短的刹那
感到眼睛的刺痛
然而
只是刹那而已

这是一个没有谎言的欺骗

太久的躲在墨镜后面
竟以为世界真的是清凉一片

突然不懂为何要拒绝阳光
也不明白为何要欺骗自己


（二）

人总是在一番轰轰烈烈之后宣布自己长大
然后在一切归于平淡乏味时发现开始苍老

慢慢终于明白

承诺是用来安慰 而不是用来实现的
等待是用来考验 而不是用来希望的
沉默是用来欺骗 而不是用来逃避的
爱情是用来过日子 而不是用来痛苦的

只是不懂为何年轻的时候
总是那么努力的说服自己

相信承诺
相信等待
相信沉默
相信痛苦的爱情也终有甜蜜的日子


（三）

阳光灿烂的日子里我微笑
我说看这世界是多么美好
阴霾密布的日子里我快乐
我说看那乌云后就是阳光

可是我怕黑夜

当夜来临
当我躺在床上
当最后的光亮熄灭
浓郁的黑暗便沉沉的压来

喘不过气

想呐喊 却失去声线
想哭泣 却失去力量

死一般沉寂的黑暗中我总是想起那句话

不在沉默中爆发
就在沉默中死亡


（四）

想笑的时候发现心里是在流泪
想哭的时候却发现脸上挤出笑容

如果爱到极至是一种负担
那么痛到极至便是一种解脱

我已说的太多
如果我曾有所保留
那么
在多情的面前
就不会显出痴情的可笑

然而这个世界里没有如果
该发生的已经发生
该结束的也已经结束

我也终于

无话可说…………
