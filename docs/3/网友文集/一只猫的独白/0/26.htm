月夜中秋

如果
我曾学会淡漠节日
那么
在川流的人群中
在一如往常的阳光里
我就不会打乱习惯的脚步
依然麻木的行走
时间便会在陌生里消弭无形

如果
我曾学会遗忘月光
那么
在月圆的夜晚
在一如往常的静谧里
我就不会在秋风中凝望
自家乡蹒跚前来的月盘
照亮黑夜似载满亲人的目光

如果
我曾学会湮灭回忆
那么
在昏黄的灯光下
在一如往常的音乐里
我就不会翻起埋藏许久的往事
任一页页不堪回首的记忆
如潮般汹涌而至

然而
在时间的锤炼中
我却终究一无所成
于是看到倾洒的月色
在酒杯中反射出温柔的磷光
身后孤独的灯光里
一缕音乐如烟轻扬

月华淡淡晕开
模糊于稀薄的云层
泪水
终于滴落无声
