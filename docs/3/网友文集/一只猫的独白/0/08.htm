忽然

　　几乎是习惯性的坐在电脑前面，有些麻木的点来点去。

　　虽然比平时多了奥运快讯可以看，可看过之后依然是无所事事的茫然。

　　一根电缆的连接或许缩短了空间的距离，却永远不能改变时间的差异。

　　忽然之间感到浓郁的厌倦，随着自心脏涌出的血液慢慢流淌，散布到全身直至点击着鼠标的指尖。

　　忽然之间停止了所有的动作，听到一种咕咚咕咚的声音，似从幽深的空谷传来，一声声沉闷的撞向我的身体，然后渐渐远去。

　　忽然之间想关掉所有的灯，所有的光源，想置身于最沉重的黑暗。蜷缩在令人窒息的寂静中感受自己依然温暖的存在。

　　忽然之间，想要毫无保留的爱一回——如昙花一瞬的绝美，倾尽所有的伤一回——如飞蛾扑火的悲壮。

　　忽然之间，只想找一个理由，让我可以释放掉所有积攒的力量痛哭一场。

　　然而所有的感受只在瞬间发生，也在瞬间结束。

　　时间继续着，生活继续着，不留一丝痕迹……
