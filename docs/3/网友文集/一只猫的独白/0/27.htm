在我的世界里...

在我的世界里
不要说爱的热烈
如苦涩咖啡中的香浓
爱的太深郁结在心间
便成了一堵厚重的痛
所以我宁要一杯清茶
爱那一缕幽香埋藏在淡淡滋味中

在我的世界里
不要告诉我什么是未来
我只是一株微小的蒲公英
离开了故土 便只能飘散在风中
既然
从起飞的刹那
就已无法决定坠落的方向
那么让我可以在飞翔中舞的美些，再美些

在我的世界里
不要试图教会我认真
生命原是永不醒来的梦
何必要计较梦境中的剧情
我只想在一旁冷眼旁观
看我哭我笑我痴 就如同
看一部冗长的电影在上演

在我的世界里
不要诧异于
我的无知 我的轻松
我的伤感 我的漠然
我虽深知现实的存在
却仍要固执的保留
这个属于我自己的世界

所以
请不要打扰我
当我在我的世界...
