昨夜情绪

　　和他们告别后，将近零点的时候，我一个人坐在地铁站里等车。很是疲惫。

　　地铁站上仍然有喧嚷的人群，对于他们，夜才刚刚开始。闪烁的霓红，一个个巨大的广告招牌，在来来去去的地铁后面忽隐忽现。

　　我戴上耳机，把外界的声音屏蔽，只让许美静淡淡倾诉般的嗓音在耳边响起。然后看依然热闹的周围无声的动着。

　　隐隐感到喉咙有些疼，大概是在火车上说话太多了，我想。

　　刚刚和三个好友出游回来。是一个靠近边境的湖，火车大约是三个小时的行程。大家都是趁着上学和打工的间隙出来放松一下。是难得的出远门。

　　那是一个很静谧的湖，尽管有很多的游人。

　　然而我们并没有在大湖边停留，而是走进山的纵深处，因为听说那里有个更使人痴迷，使人留连的的"仙境"。

　　"仙境"就在无数的转弯后突然出现，让人猝不及防的被眼前的美呆住。那是一个很小的湖，湖水是柔和却幽深的绿色，像一块不小心掉落人间的玉石。四周除了来路的密林外，都是依水直起的青山峭壁，却只在对面的山脚下留了一小片青青草坡。草坡上一个小小的木屋，与湖边的船坞一起形成了各色绿中绝美的点缀。山顶依稀还有些残雪，与云雾相接，天空此刻便只剩了头顶很小的一片。不知为何，觉得一直最爱的天蓝色在这里竟显的如此浅薄。

　　我们爬到靠近岸边的一块大石上坐下，看阳光从背后山顶射进来，只落在极少的一处，在湖面上形成一条翠绿的光带。近处的水清澈见底，可以看到指头大小的鱼在水底吐的泡泡慢慢升到水面便消失。竟然还有五只手心大的绒绒的小鸭子从我们坐的石头前很努力很快活的划过，游向远处等着它们的鸭爸鸭妈。

　　我们望着对面草坡上的小屋，想象着住在那里会是什么样子。

　　后来很辛苦的竟顺着一条不易觉察的小路真的走到了那个草坡上面，一路上石子树根，悬崖陡壁，却极其值得，因为每一个角度的转换都是另一副同样完美的图画。草坡上开着些野花，带着湿润的香气。湖在青山中便如碧玉沉香。

　　很静的美，用同样静的心去感觉。我们从城市中走出，谁都无法抗拒心灵与自然和谐时产生的巨大震撼。

　　然而这种震撼却最快的消失，一旦离开了它赖以生存的自然。

　　回来的火车上四人快活的与疲劳对战。大家兴致勃勃的谈着各种话题，似乎很快忘了刚刚的美景。火车在夜色中穿行，车窗在黑的背景下便成了极清晰的镜子。我看自己略带疲惫却同样快活的脸，不知此刻想要的是究竟是那安静的寂寞，还是这热闹的陪伴。

　　在地铁站口他们嘱咐我一定要让男友来接。我说放心，然后和他们灿烂的告别。

　　我等的地铁还有一段时间才来，我拨通电话，很坚持的说，你明天还考试，千万不要来接我。放下电话却是很深的落寞，有些后悔为何没有自私一点。

　　我坐在人群中，音乐把我与周围的世界隔离开。喧嚣中的安静，便显的格外孤独。在太熟悉的环境里我发现我已记不起白天的感觉。包围我的只有此刻的寂寞。

　　我以为我是喜欢安静的。我在零点时分孤独的等待载我回家的地铁。却很是疲惫。

　　地铁来的时候，许美静正开始一首新歌，很无奈的音乐很清净的嗓音伴着地铁的进站声驶进耳朵。 "结果还是无所谓 让梦想 继续的颓废 流言城市 霓红灯火 怎么能淹没 挣扎和寂寞……"

　　站起身来的一瞬，眼睛竟突然潮湿。然而只是一瞬而已。

　　那只是一种情绪，与现实无关，有些莫名其妙，所以没有让它继续……
