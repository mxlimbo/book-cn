随想三首

星斗

一直盼望
和你看满天星斗
你会告诉我
哪一颗 曾是你凝望我的眼睛

直到今夜
我才猛然明白
原来 我也是那繁星中微小的一颗
循着亘古不变的轨迹
照亮夜空里最清冷的角落

而任我怎样的期盼
我们也只能
在苍茫星河中远远相看

昙花

我守在安静的夜里
等侯你悄然绽放
然后 在你最绚丽的一刻
将你轻轻摘下 小心的夹进我珍藏着青春的书页

我禁锢了你的美丽
可是为何
仍禁不住 心里的丝丝忧伤

暗示

如果 你不曾爱过我
那么 那些细致的温柔
是否只是我的错觉

如果 我不曾在意你
那么 我想我就可以
轻轻放下所有的记忆

可是 却依然希望
如果 你在无意中又把我记起
那么 请一定一定
在我夜的梦里
给我一点温柔的暗示
