往事如歌

岁月会流逝，记忆在淡漠，我却永远记得，那些不再唱起的歌，只因我们曾共同谱写。

——之一——

如果你一定会远去
离开我驻足的地方
那么让我
为你送别 好吗
我会在你我相拥的河边
看着你慢慢流淌
直到泪水带走所有的悲伤
直到秋风吹散无尽的过往
让我心爱的收藏
在风中
飘成一首美丽的歌

——之二——

握在手心的
是你送的贝壳
你说要我记得
我们曾看潮起潮落

海潮往复中
浪花与沙滩依依不舍
就像你我
在相遇与告别中
难以割舍的交错

我将贝壳放在耳边
听见依然遥远的涛声
像是一首
永不息止的歌
