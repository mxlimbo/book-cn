只愿梦未醒

总在清晨时分
发现泪沾双眼
却记不起
那是怎样的一个梦境
让我在心痛里告别夜的黑暗

一时间竟恨起了窗外熹微的晨光
是如此执拗的照亮我心底的空旷
可我知道
一定曾有个美丽的世界
被我遗失在醒来的床前

在记忆的空白里我渐渐绝望
直到看见一只鸟儿飞过蓝天
猛然想起
在我夜的世界里
是有过那样一片沁心的湛蓝
蓝色在黑幕里如宝石般闪亮
我于醉人的波光中沉迷的飞翔

当所有的灿烂
伴着夜色消褪
你便在我的梦境里
悄无声息的出现

柔软而冰冷的唇
吻过我无助的双眼
你的呢喃响在耳畔
——
我就是你深爱的蓝
我会永远陪在你的身边

我紧握住你的双手
想把所有的温柔记在心间
只是你的身影依然模糊
我在期待里睁开双眼
却在醒来的刹那
只看到手心里空白一片

终于想起了你最后的低语
——
不要醒来
不要醒来
那样
在你的世界里
我就会越走越远
