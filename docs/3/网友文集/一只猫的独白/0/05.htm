想要颓靡

　　气温突然冷了下来，毫无征兆的。我连毛衣还没来得及从箱子里拿出，行人中已有人穿了棉外套，我奇怪自己竟一直都没准备好面对将至的寒秋。

　　早上一进了公司，就开始下起大雨，没有带雨伞，却很放心的对朋友说，看这雨，下不了多久的。说这话时我忘了已不是夏天，雨的样子虽然看上去并无多大改变，却不再可以率性的来去匆匆。天不是很阴沉，雨却一直没停，即使渐渐无力，也仍顽强的倾泄着。我心里纳闷，为何天一变凉，就会凭空多出这些个绵延不止的水来。久了，感觉心里也滴出了水。

　　想来这座城市里从来就没停过雨，一年四季，除了冬天无休止的大雪，其他时间便几乎都是湿的。只是同样的水，为何秋的却特别绵长。

　　昨天一个同学无精打采的过来，很无奈的说夏天过去了，接下来的日子该怎么度过，只剩一个凄清冷秋还有漫漫长冬。我很同情的对他说，那你只能期待来年春天了，心里暗想大约只有冬眠才是最好的解决方法。我却是期待冬天的，因为爱着放眼望去的一片苍茫白色，甚至喜欢白色里那股彻彻底底的冷。我想我一定是个极端的人，不然怎么会同时喜欢着夏的酷热和冬的至寒，大概只有这样的温度才可以刺激我保持清醒。春的柔软会让我忘乎所以，沉迷于冬天过后的温暖。而秋，又太过萧瑟，湿着粘在路面上的落叶，看久了心就会跟着一点点的灰暗起来。可是爱与不爱，一年四季总是不停止的走着，就像心情，不可能永远保持在让自己舒适的状态。

　　现在是秋了，夏冬之间有些凋残的季节。我不知道该做些什么，或许就这样跟着落叶一起颓废直至腐烂，然后安慰自己说这是为了新生。

　　唯一的节日是下周二的仲秋。可是在那个普通的工作日里，会有什么是不同的吗？不会的，在这个没有农历的国度，那遥远家乡的节日已失去了概念。

　　其实这本该是一个用来整理记忆的季节，只是我不愿去回忆，那会使秋天格外显得潮湿冗长。过去的总是过去了，不管是否想挽留，该来的已经在路上，不管是否期待着。

　　秋风起了，雨还下着，继续颓靡，直至冬季。
