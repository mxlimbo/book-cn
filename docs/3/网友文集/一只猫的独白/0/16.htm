满足

我在黑暗中点起一盏橘黄的灯
泡上一杯清茶
看暖暖的香气化做极细极柔的水雾在灯光中冉冉上升
窗外 已有些许泛白
我知道
夜 正开始慢慢褪去

我安静的等待
等待所有的黑暗在阳光中一点一点的散尽
然后拉开窗帘
就会看到那个醒来的 因色彩而美丽的世界

黎明是如此的寂静
寂静中听的见自己的心声
我有些惊异的发现
心里竟有久违的恬静和柔和在丝丝蔓延

可是 为什么不呢
我听到熟睡的恋人在身旁均匀酣畅的呼吸
我看到亲人的思念正穿越茫茫空间向我热切的飞来
就连昔日冰冷的电脑荧屏
今夜 我却也触摸到了温暖的关怀
我不知还有什么理由
能让我拒绝 在如此寂静的黎明
享受这美丽得近乎奢侈的 幸福

夜尽了
天亮了
我拉开窗帘
便看到了炫目的色彩
蓝天白云 是我不变的最爱
还有绿叶 在风中轻声摇摆

我虽深知自己贪婪的本性
可此时此刻
竟真的再也想不出
这样美丽的世界里 有什么
是我 还没有得到的……
