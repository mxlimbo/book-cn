暮色苍茫

　　我坐在教室宽大的窗台上，向外望去。已是黄昏，很浓郁的灰蓝天空在我身边蔓延。

　　这是今天的最后一节课，推门进来时，里面的安静让我深深后悔在校园咖啡里贪恋了几分钟悠闲。我很感激这个宽大的白色窗台，因为它使晚来的学生不必席地而坐。然而坐在这样一个方便的位置上，我忍不住要把脸侧过去，看向窗外。

　　这是在四层楼上，从这里看下去的感觉就像幼年时趴在六楼的阳台向下张望，觉得自己是在世界的另一端，审视着漠然无知的人群。人群越来越稀少，我看不到他们的面容表情，每个人都同样的步履匆匆。对面食堂的灯光在越来越稠的天色里温暖无比。但是我看到最后一批人从里面走出，我知道，那光也就要熄灭了。是回家的时候了，人们在这时候都要回到一片属于自己的灯光里。

　　一大群鸟儿从远出飞来，落在我看不到的地方，周围突然寂静下来。也许天空在夜晚的寂寞就始于这一刻鸟儿归巢。远出的树枝和屋顶开始模糊，那些渐渐融化在蓝黑色里的轮廓，让我心生错觉，感到朝思暮想的家乡就隐藏在不远处越来越深的黑暗里。

　　我把脸贴在玻璃上，有些凉，衬出教室里的温暖。终于看不见了窗外，取而代之的是灯光反射在玻璃上用心听讲的同学和我最喜欢的女讲师。我对着玻璃里的老师歉意的笑笑，正过头来。

　　身后是寂寞冰冷的黑暗，眼前是温暖宁静的明亮，坐在窗台上，像是在两个世界中间，我感到一股不知名的温情在我身体两侧流转。

　　突然有个预感袭来，在遥远的某天，我会在不知何处的角落，坐在窗前，思念起这个暮色里黑暗与明亮的世界，就好像现在的我，思念着一直深爱的那个远方的城市一样。
