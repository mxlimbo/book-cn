两 岸

相隔是宽广的海峡
但两岸并不遥远
切开记忆的闸门
仿佛近在眼前

相互依恋的波浪
不息的翻卷
深情交融的思念
一年又一年

同顶一片蓝天白云
总会有阴霾的日子
共享的中秋圆月里
留下几分无奈的缺憾

每当岸边的鸥鸟醒了
声声的谛鸣仿佛在呼唤
寻那久别的摆渡人
为何一去不再复返

这流动不羁的生命
不停的用情感去企盼
共建一座连心桥
将理想化作重逢的夙愿

相隔是宽广的海峡
两岸并不遥远
在某一年的春天后
长虹贯通天堑
