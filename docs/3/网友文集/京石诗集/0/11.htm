星 座

一

遥远的天际
隐藏着一段美丽的故事
忽远忽近 寻寻觅觅
思念的露珠
在阳光的照耀下沉淀些许斑迹

而今我知道这情景的起终点
不单是轮回在童年的梦里
一曲缘尽的歌绕粱在十字架的沉苛
一颗无言的心流淌冰凉的血滴

春夏秋冬 日复一日
无奈是炽热的风 更是冷骤的雨
离愁是残叶飘零的结局
如同一篇序言的再度翻版
掩卷坠满无望的忧郁

于是星座的边缘坍塌了
光芒灭了 天堂鸟飞走了 月光隐去了
没有了爱的灵魂
也没有了真实的日子

二

天堂最终是蓝的
湖水最终是绿的
血最终是热的
爱最终永恒
日月不变的交替辉映
缘的复苏是涅盤再生的激情
星座不是幻灭的影子
她是时隐时现的明灯

三

轮回是有的
星座如是说
