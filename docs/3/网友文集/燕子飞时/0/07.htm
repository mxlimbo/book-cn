五官情感超极秀

　　一、比如耳朵

　　隔着整张脸的山川河流，比起其它五官，耳朵更象是网恋，相恋不相见。 感知着彼此的存在，每天问候一句："你还好吗？"，假想中的耳鬓厮磨，总有一种错觉，他（她）是近在咫尺的。当然，也许从某种意义上来讲是近在咫尺的，比如两个耳朵我们量一下它们的直接距离，也不过就是一只手就量得过来的宽度而已，说到见面却是难上加难。即使见了面，也必定是痛彻心肺的结局，然后退回原位，一切尽在不言中。

　　在这个社会我们变得狠心且世故，看着两只耳朵谈恋爱，我们体会着那份新奇，那份朦胧，却轻易不感动。

　　二、比如眼睛

　　眼睛注定是脸上最动人的一道风景，就好象情人对于千篇一律的生活总是一针强心剂。

　　两只眼睛是永远也走不到一起的，它们隔着一道鼻梁理性的存在，就好象情人之间的约束。可是为了看清前方的道路，我们的大脑却安排了它们必须交会。 交织的目光越缠绵，摆在前面的事实就越清晰，暗地里情投意合，表面上看着却是毫不相干的两个个体。

　　即使有一方放弃或是死去，另一方也一样会好好活着，辨认起方向来依然毫不含糊。

　　三、比如鼻子

　　鼻子怎么看怎么都象是自恋狂，一副自我陶醉的样子。它大模大样的居于脸的中央地带，你可以认为它不自量力。看着它的样子，你也可以说它上窄下宽，敦敦实实。实在看它烦，也可以说它上不着天，下不着地，看着轻浮。它还是老样子，自恋依旧。

　　自恋是一种情结，不是一种错误。在这个物质极大富有，理念极度宽松的时代，英雄更是不问出处，亦不用枪林弹雨拼杀，功成名就后无须给自己舔伤口的时间，别人的赞美慢慢转化成自身的血液，你自己不爱上你自己，你又对得起谁？又或者干脆问一句，自恋需要理由吗？自己都不爱自己，让别人怎么爱你？

　　这样的人注定情感丰富，但见别人热泪盈眶之时，自己也不免酸酸的，跟着流出些杂质来，就好象眼睛流泪的时候鼻子也会流出点鼻涕，也许比较容易受人影响。

　　四、比如嘴

　　嘴是看上去最象夫妻的，不管是以双唇还是以唇齿论。

　　夫妻的亲密程度就如同唇齿相依，虽然有时牙齿咬了嘴唇，有时嘴唇封闭了牙齿的呼吸，但是那种紧紧依靠的状态却是改变不了的。有时一些外力可以打破这种状态，那么后果就是必然会变得丑陋无比。或许有修补的可能，但是修补过后的感觉却无法和最初的匹配相提并论，偶尔有勉强说得过去的，代价也是十分昂贵。

　　五、比如眉毛

　　眉毛就如同人生的好友知已。看上去简简单单的，如两条并行的线，无任何花俏的外表，也不会象其它的五官，各有辅助手段。只是默默的长久不变的躲在边缘地带，发挥着自己的作用，只有在愁绪缠绕，皱紧眉头的时候，两条眉毛才会比较亲密，保持着最近的距离，危难之中，如雪中送炭。而烦恼消除后，又马上恢复到正常状态，又是两条并行的线，只余下心照不宣。

　　也许它不是万能的，但是没有它是万万不能的。

　　这样看来我们的五官也同样能诠释我们的情感，无法不感叹这一脸风流。
