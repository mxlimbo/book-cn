城里的耗子

　　我是城里的一只耗子，说的文雅点就是老鼠。

　　我在城里有巴掌那么大块地儿，没事儿来玩吧。这是我常对乡下亲戚说的话，语气骄傲得很。不过那都是从前的事儿了，现在城里车多人多，空气污染很严重，而且精神压力很大，所以有的时候就去渡渡假，接触接触大自然，去哪？当然是去乡下，我那些亲戚的气色看上去好极了。

　　我很小很小的时候，我的祖父就告诉我，在外面看到花花绿绿而且闻起来很香的东西，可千万不要吃啊，那是毒药，会毒死鼠的。而我父亲告诉我同类东西的时候，却告诉我那是毒品，是鼠界的海洛因，吃了可是会上瘾的，千万不能沾。

　　当我问到为什么和祖父的答案不同的时候，父亲用很不屑的口气说我祖父是老脑筋，别听他的。没想到我父亲这么大岁数，逆反心理还这么强。

　　有一天，我和我女朋友小娜散步的时候，居然真的看到这样一种东西放在那，五颜六色的很好看，而且我居然又听到一种关于这种东西的答案，小娜告诉我，她听她妈妈说这种东西叫做老鼠药，从前是真的会毒死人的，可是现在人类社会假货横行，打假都成一种职业啦，老鼠药当然也全是假的咯，居委会的大妈发多少包都是药不死鼠的。虽然我听着有很多名词不太懂，什么假货横行啊，什么居委会的大妈啊，可是在小娜面前我是不能丢脸的，而且小娜的妈妈的姐姐的丈夫的小姨从前被人当做宠物养过，所以小娜一家是见过世面的。

　　我用坚信不疑的眼光肯定了小娜的话的真实性，然后壮着胆子走到前面闻了闻，很香甜的味道，为了逞能，我舔了一大口，"味道还不错。"我建议小娜也来尝尝。我们吃得很饱后，找了一个地方晒太阳，我的心里很后怕，万一真的毒死我们怎么办？后来一想，和小娜死在一起，也算死而无憾了。想着想着，我就进入了一种晕晕乎乎的状态，象做梦一样，好多美好的事情。然后就觉得很渴，我们耗子向来是很怕水的，可是那天却感觉到水是那样一种甘甜的滋味。以后我和小娜经常一起疯了似的找这种食物，我爸爸的话没错，这是毒品，会上瘾。

　　听说现在很流行一种叫电脑的东西。在试验室上班的远亲白鼠回来探亲的时候，说那上面有一种东西叫鼠标，尾巴很长。我很想让他带我去见见世面，可是后来再也没见他回来过了，听说是因公殉了职，家里得了一大比抚恤金。后来我在电视的广告上终于见到了电脑的样子。没怎么细看，只发现电脑旁边的ＭＭ很漂亮，我说ＭＭ的时候，小娜很困惑，我在心里暗暗的笑，网络时代最怕什么？最怕网盲。

　　很偶然的一个机会走到一个房间里，房间的桌子上摆着的那样东西和我在脑海中电脑的印象对上了号。我费了九牛二虎之力爬上了电脑桌。特地去看了一下那种据说和我沾亲带故的东西。真让我失望，如果那个拖着长尾巴的东西就是的话，那他可真难看，又不会动，没有亮晶晶的眼睛，没有亮闪闪的皮毛，光秃秃，真为我们鼠类抹黑。我气得踢了它一脚，就听它"咔嗒"一声，吓得我落荒而逃。

　　就在我马上就要逃出房间的时候，一条裙子闪进我眼里，然后就是一声尖叫，我猜想这是他们人类求救的信号，我被这样的叫声弄晕了头，四处乱撞，就看那个花裙子跑来跑去，我也开始大声的呼救。

　　眼光闪过墙角的时候，我心里一揪，墙角的小洞里两个耗子脑袋挤挤挨挨的，我认出来那是我的两个死对头，出了名的古惑鼠，没事儿专找我的麻烦，这会儿正在那里看着我的热闹，居然还悠闲的吃着名牌爆米花。我横下一条心，就是赌这一口气，我也得逃出去，那一刻，我的心里没有爸爸，没有妈妈，甚至没有小娜。

　　终于逃到安全地带的时候，我松了一口气，晕了过去。

　　再醒来的时候，我躺在那捡来的快餐盒搭起来的小床上，旁边围着一大堆鼠，他们的嘴一张一合，似乎是在说什么，可是我什么也听不见，静悄悄的。

　　看了好多医生，都说我的器官一点没有损坏，推测是一种心理障碍防碍了我的听力。罪魁祸首大概就是裙子。

　　让我可能康复的条件有两个：

　　一、城市没有裙子。可是城市里是不可能没有裙子的。

　　二、我永远不出去散步。象我这样一只个性活泼的耗子原则就是不自由，毋宁死。

　　于是，在这喧哗热闹的世界上，我聋了。
