爱神难道是僵尸变的？

　　晚上和朋友出去玩，朋友之一突然心血来潮，出口不凡，他说："夜幕降临了，恋爱的空气真浓啊！"言外之意就是，大街上到处都是成双成对游荡着的年轻男女。另一个朋友的回答极其另类，大意就是谈了恋爱的人喜黑，喜静，呀，怎么居然和僵尸差不多。几个人都为了这个回答狂笑不已，引得路人侧目。

　　回到家里，仔细一想，还真发现两种事物有异曲同工之妙。

　　僵尸的出动时间正是夜晚，或觅食，或游玩，是一种专过夜生活的东东。而沉浸在爱河中的男女也是颇为喜欢在天黑下来的时候散散步，说说情话的。在模模糊糊的对视中，可以表达出不能在光天化日下表达的情感，月光下的情人，自是让人别有一番滋味在心头。

　　僵尸的喜欢一种相对的静谧。哪怕是无可奈何的群居，也要给自己创造一个相对独处的环境，有谁看见过两个僵尸一个墓穴来着？而恋爱中的男女更是不喜被人打扰，情到深处，恨不得全世界只剩下自己与对方两个人才好。

　　僵尸是一种执着的东西，觅到心仪的食物，一口咬下去，不吸干绝不放手。而一个人喜欢上另外一个人的时候，自然也是穷追不舍的，充分发挥着一不怕苦，二不怕累的八字方针，直到无任何甜蜜可言。

　　僵尸又有着拿得起放得下的高贵品质，这一个血干了，毫不犹豫的丢掉，马上就去寻觅下一个目标，重复着从前的一切。而两个曾经相爱的人，发现对方不再有吸引力的时候，自然会根据合则聚不合则散的原则分手，然后找寻下一个恋爱点，停留在原地打转的人极少极少。

　　僵尸是一个相对来讲语言比较简单的东东，基本上就只会"呵~~~呵~~~~"，别无其他语言表达方法，眼神却是灵活生动，顾盼出自己的喜怒哀乐。而两个人一旦相爱，语言也就变得贫乏起来，无非是"傻样"、"我爱你"什么的，万变不离其中，最让人心动的就是一双会说话的眼晴。

　　综上所述，让人不免怀疑，爱神这长着小翅膀的可爱东西，难道是僵尸变的？要不，就是同宗同系？一身冷汗……
