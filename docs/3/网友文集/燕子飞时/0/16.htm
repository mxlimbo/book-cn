甲方乙方

　　月黑风高。

　　电脑甲："我叫不紧张。"

　　电脑乙："呵呵，又不是相亲，你紧张什么？"

　　电脑甲："我就是要找个媳妇，你怎么知道我不相亲？嘿嘿！"

　　电脑乙："我叫雪衣豆沙。"

　　那个时候，满聊天室都是电脑，挤挤挨挨的，弄得聊天室象蒸笼似的。不紧张这个名字引发了好多电脑记忆中储存的赵本山。电脑甲推开挡在身前的一个个障碍，接收到电脑乙的名字，感觉甜得发腻，心里一阵震荡，差点死机。

　　又一个月黑风高。

　　电脑甲："我爱吃白菜叶。"

　　电脑乙："我爱吃白菜帮。"

　　电脑甲："我爱吃汤里的菜。"

　　电脑乙："我爱喝菜里的汤。"

　　"绝配，绝配。"满聊天室的电脑在尖叫，各种各样的名字，眼花缭乱，影影绰绰，电脑甲居然看见电脑乙脸红了那么一下，电脑甲心里一乐，刷出两个大字"媳妇"，红艳艳的，象新婚时剪出的窗花。

　　电脑甲："我们这里快下雪了。"

　　电脑乙："真的吗？我也喜欢雪呢，真白呀，就象……就象雪衣豆沙的雪衣。"

　　电脑甲："馋丫头，你就象传说中那个馋媳妇，形容什么都用吃的东西，嘿嘿！"

　　电脑乙："嘻嘻，给我唱首歌好吗？……，在电话里。"

　　那部顽强地承受着电脑没日没夜的折磨的电话，老而不旧，上面落满了爱情的痕。

　　电脑甲："我也没娶。"

　　电脑甲："你也没嫁。"

　　电脑甲："我现在经常吃雪衣豆沙，真甜啊！"

　　过了很久，电脑甲又说："下雪了。"可是电脑乙不说话，因为聊天室里没有电脑乙了，渴望看雪的雪衣豆沙去了一个几乎不怎么下雪的地方，据说那里有爱情的滋润，可以让人貌美如花。电脑甲一下一下的划着窗上的冰凌，凉意透过触摸，有点短路。

　　那个时候，电脑乙正在书店里闲逛，儿童读物里童话的世界，一幅小小的插图。一栋小小的雪房子，嵌在冰天雪地里，透出屋里晕黄温暖的光。高温的天气里，电脑乙流着汗也流着泪，湿了的雪衣豆沙应该是一种糊状物了吧，看不出原形了，他还会认出我吗？电脑乙慢慢的想，手里紧紧攥着那本童话，就象攥着几许清凉。

　　总是月黑风高，在那个老地方，电脑乙藏在角落里，看了电脑甲一夜。电脑甲什么都不说，临告别的时候，刷出三个大字："郁闷牙！！"深紫色的，很绝望。

　　电脑甲的电话已经看不见爱情的痕了，铃响的时候，灰尘四起，呛出电脑甲一脸的泪。

　　电脑乙绕过大半个国土，踏上北方的土地的时候，深深吸了一口清冷的空气。电脑甲远远的站着，一眼就看到电脑乙的红色绒线帽。便跑过去，"我叫不紧张。"电脑乙"扑哧"一声乐了，露出两排可爱的牙。太开心了吧，开心得想哭。两簇泪花即将绽放的时候，手机响了。

　　"嗯！我到了。"

　　"一切都好。"

　　"办完公事儿就回去了。"

　　"好了，BYE-BYE！"

　　在饭店里点菜的时候，电脑甲先点了一个雪衣豆沙，服务生乐了，"很少有先生喜欢吃这种甜品的。"电脑甲笑笑，又点了一个醋溜白菜，点了一个汤，没管服务生不屑的眼光，电脑乙说够吃了。

　　点了最便宜的菜，却开了一瓶很好的酒。两个人卿卿我我的，一个吃掉了所有的白菜叶和汤里的菜，一个吃掉了所有的白菜帮和菜里的汤。雪衣豆沙却一口没动，两个人争论着，电脑甲说其实雪衣豆沙的雪衣有一点点淡淡的黄，电脑乙却说是洁白洁白的。

　　冬日里的天使漫天飞舞，电脑乙欣喜地追逐着飘扬的雪花，仔细看着落到手上的雪花，晶莹惕透，而后化成清水，顺着指缝间流走了。电脑乙闭着双眼，默默执著的站着，象是冬日里空守着荒地的稻草人。

　　电脑甲："你在干什么？"

　　电脑乙："嘘……雪花从我的肩头滑落，温柔，多象一双手。"

　　电脑甲走到电脑乙身边，温暖着她冰冷的面颊，然后告诉她："北方的女人，从来不在寒风中哭，那样不但伤心，而且还很伤皮肤。"于是，电脑乙把脸埋在电脑甲温暖的胸膛，泪湿衣衫。很多年以后，静静的雪夜，电脑甲都还能感觉到胸前的凉意。

　　电脑甲藏在聊天室里想，这会儿电脑乙该是逛着街吧，穿着素色的裙。好多电脑在耳边叽叽喳喳，电脑甲在屏幕上一行行地刷着自己的感情，不呆滞，但是很麻木。

　　无人喝采，那些字呈雪融状，映在浅蓝色的背景上，象滑过晴空的雪花，纯美，而不真实。
