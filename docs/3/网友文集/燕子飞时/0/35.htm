写字楼之宝黛相会

　　"两个大石狮子，三间兽头大门……打扮齐整的老妈子，小轿晃晃悠悠……抄手游廊，紫檀屏风，雕梁画栋。黛玉下了轿缓缓的跟着小丫头们走着，听见一个软软的声音："林姑娘到了。"黛玉看见了银发的外祖母，祖孙俩一见面，不由得哭个不住……"

　　黛芊突然醒了过来，不由自主的哽咽了一下，将眼睛完全睁开的时候，发现离自己很近很近的地方停着一张脸，大惊，"啊……"叫到一半儿，才发现是每天中午给自己公司送外卖的云国。云国看着黛芊泪痕隐约的脸，满眼都是研究的问号和一种抑制不住的浓厚的兴趣，每天中午来送外卖，都看见黛芊趴在办公桌上午睡，脸上的表情总是睡得很紧张，直到他的外卖送来，她才开始睡眼朦胧的吃午饭，若有所思发着呆。

　　"你没事儿吧？"云国温和的看着黛芊。

　　"哦……没事儿，谢谢！"黛芊勉强笑了笑，接过自己那份午餐食无甘味的塞起饭来。

　　黛芊很奇怪自己刚才做过的梦，《红楼梦》看过八百六十遍了，应该是黛玉初进贾府的那一段没错，在梦里黛芊一会儿好象是黛玉，一会儿又好象高高的悬在空中，只不过是一个看客。黛芊离家比较远，所以每天中午就索性在办公室里解决午餐和午睡的问题。她自己知道自己趴着睡觉的时候，总是会做些不太好的梦，比如说今天就是哭醒的。可是，今天的梦还是太离谱了。怎么会做起这个来了？吃过饭，黛芊拿出自己的随身听，决定听些音乐，不去想那个破梦了，必竟它只是个梦而已，"蒙娜丽莎她是谁，她是否也曾为爱争论错与对……""吁……"听着林志炫她长长的舒了一口气。

　　"三个美丽的姐妹，一个肌肤微丰，腮凝新荔，温柔沉默，可亲；一个削肩细腰，长挑身材，顾盼神飞，见之忘俗；一个身未长足，尚小，可爱；一模一样的装束。……黛玉和众人说着些无关紧要的话，药丸，往事……不见其人，先闻其声，在一阵笑声过后，黛玉觉得眼前一亮，只见眼前这个人，彩绣辉煌，恍若神仙妃子，八宝攒珠髻，朝阳五凤挂珠钗，赤金盘离璎珞圈，金丝豆绿宫绦，比目玫瑰佩。黛玉知此人必与众人不同……熙凤和黛玉寒喧着，黛玉便又哭了…………"

　　黛芊又一次在中午的阳光灿烂中哭哭泣泣的醒来，联想起昨天中午的梦，开始奇怪起来，这梦居然还可以续着做？突然有一种想和别人说点什么的想法，正巧看见云国拎着外卖走过来。

　　"你平时爱做梦吗？"黛芊懒洋洋的半眯着眼睛看着云国。

　　"嗯……不是经常。"

　　"你说，做梦有续上的吗？我的意思是说，比如昨天做了一个梦，今天再做梦的时候，发现和昨天的梦居然是连着的？"

　　"没遇到过？你呢？"

　　"……"

　　说了半天，黛芊觉得累了，也饿了，就拿过外卖吃起饭来，她说话的时候，云国几乎是沉默着的，不过，黛芊觉得这也没什么，反正她也就是想找人说说，说出来就舒服了，并不在乎有没有什么安慰什么的。只是很感激云国听她说了这么多，云国走的时候，她破天荒地说了一声"再见"。云国回过头来，笑了笑，走了。

　　"坐着翠幄青轴车，黛玉与刑夫人同坐。……下了车，厢庑游廊，小巧别致……许多盛妆丽服的姬妾丫环。……告辞……荣府……耳房钻山，四通八达，轩昂壮丽，大匾"荣禧堂"……青绿古铜鼎，待漏随朝墨龙大画，……进了东房门，陈设精美……吃茶……见王夫人，黛玉与王夫人寒喧着，自王夫人口中听得"混世魔王"四字……用饭，只听"宝玉来了"……"

　　这次，黛芊是被云国弄醒的，云国手中拿着一盒补脑液，轻轻敲着黛芊的胳膊，"试试这个。"黛芊接过药，放在办公桌上，"谢谢，多少钱？""嘻嘻，一万块，给我吧。要是说钱你就把药还给我吧，我可是把你当朋友才给你拿药来的。我一个朋友从前睡眠也不好的，多梦，吃了这个药好了许多了，也许对你有好处。""那……那谢谢你了。可是我觉得我这个不是什么睡眠不好，太莫明其妙了……"黛芊笑笑，边吃着午饭，边又讲起这个连续梦来，云国也觉得很奇怪，说着说着，两个人的话题海阔天空起来，不知不觉的黛芊一看表，居然聊到了快上班的时间，云国笑着说："回去被老板骂，我可要找你算帐，呵呵，走了，再见！"黛芊望着云国的背影，不免有点惋惜，谈吐如此不凡，为什么偏偏要做一个送外卖的呢？

　　之后的每一个中午，黛芊都做着和最后一次同样的梦，总是到"宝玉来了"就不能够继续下去了，这种可遇而不可求的事情，怎样努力都是白费。黛芊感觉自己正在被折磨着。要不是云国每天中午可以陪着说说话，这个梦非得把黛芊折磨疯了不可。有的时候，晚上云国也约她出来，除了中午云国给黛芊送外卖来的那一瞬间，其他的时候黛芊根本感觉不到云国是一个送外卖的。有的时候，黛芊觉得云国的学识比起她这个大学里的高材生来绝对是有过之而无不及。

　　傍晚，华灯初上，闪闪烁烁的映照着街上人们或快乐，或疲惫的脸。黛芊愁眉苦脸的被罩在五颜六色的灯光下，随着走动变幻着颜色，偶尔偏过头看着云国，黛芊从来没想过自己的男朋友会是个送外卖的，不过想想，送外卖的也没什么不好，爱上了他，看他的方方面面就都是好的。

　　"我听说某某寺里的签求起来很灵的，要不，这个星期日我陪你去看看？呵呵，太迷信了呵，可是你总也不好，索性死马当活马医吧！"云国哄着黛芊。

　　"啊？求签？哈哈，只在电视里看过呢？"

　　"嘘……心诚则灵！"云国一本正经的样子逗乐了黛芊，于是，两个人约好星期日一起去某某寺求个签来看看。

　　山上的空气很好，清新宜人，黛芊的心情也好了许多，一路上玩着，不知不觉到了山顶的某某寺，黛芊本想开几句玩笑，可是被寺中的肃穆和善男信女的虔诚给感染着，心里也有了一种神圣的感觉。

　　到了签房，房中的僧人双手合十，"哪位求签？""我。""那么请男施主回避一下，在这里扰乱女施主的心田，求签会不准的。"云国退出门外，望着。黛芊回过头来，将心静了静，轻轻晃动签盒，一根竹签跳了出来。黛芊将签拾起来，只见签的正面写着。

　　"一个是阆苑仙葩，一个是美玉无瑕。若说没奇缘，今生偏又遇着他；若说有奇缘，如何心事终虚化？一个枉自嗟呀，一个空劳牵挂。一个是水中月，一个是镜中花。想眼中能有多少泪珠儿，怎经得秋流到冬，春流到夏！"

　　签的背面写着：只等有缘人，佳音宝中求。上上签。

　　黛芊看了看，知道是《红楼梦》中的《枉凝眉》，可是不知道为什么这样凄婉的一首歌会是上上签。疑惑地将签给僧人递了过去。

　　僧人看了半天，脸上露出笑容："女施主，恭喜你，喜事将近。施主前几世中有一世曾经是林黛玉，因故未能与宝玉结合，于是，上天有好生之德，特赐了一段姻缘，让你能在后世的轮回中和宝玉之后世重逢，再续前缘，只是有一个条件，他的名字中必须有个"宝"才能算是有缘人，如果你要嫁的人名字中无"宝"，那么只好又要多等一世了，不过，施主的签显示，有缘人已到，且字中有"宝"，只不过天机不可泄露，请施主一人心中有数既可，切勿和他人谈起，切记切记。恭喜施主了，善哉善哉！"

　　黛芊听得一头雾水，只是听明白了，自己的前世中曾经做过林黛玉，然后，现在得找个名字中带"宝"字的人嫁才算皆大欢喜，上上签？

　　云国看见她走了出来，便急急的问："怎样，怎样？"黛芊苦笑，摇了摇头，云国忙安慰她："没关系，我们本来就是来试试的，怎样都无所谓，看看，还有没有好的药，要不，你这梦就做着，还可以每天给我讲故事呢？"拉起黛芊的手，发现炎炎的夏日，黛芊的手却冰凉。黛芊突然哭了起来，云国轻轻拥着黛芊，"傻丫，哭什么？让梦折磨成这样？也许过一段时间就好了呢？别哭了"黛芊心里却在想，什么你的名字里没有"宝"字呢？黛芊想起了公司里一直缠着她的魏宝民，不由得打了个冷战，不会是他吧？惊吓中，哭声便渐渐歇了。

　　梦还是一遍一遍的重复着，还是看不到宝玉的脸。黛芊试过不趴着睡，不在中午睡，可是都没有用，梦境依旧还是重来，虽说这不能算是恶梦，可是成年累月的做同一个梦的折磨却是让人无法忍受的。黛芊愈发的瘦了，只有和云国在一起的时候，她才能笑得开心些，其它的时候，总是默默的。黛芊也试过正眼看看魏宝民，看有没有什么可取之处，必竟签上显示他算是有缘的，可是，却怎样看也看不出任何感觉。于是，她横下一条心来，什么前世今生的，我只嫁我喜欢的，没有"宝"字就没有罢！我怎么能为了一个签去毁了我一生的幸福呢？自此，豁然开朗。

　　一日，和云国走在街上，一人喊"宝儿"，云国一回头，原来是云国的一个亲戚，聊了几句便道别了。可这一声"宝儿"听在黛芊耳中却如春雷般，黛芊一把扯住云国："刚才他叫你什么？啊？""宝儿啊，嘻嘻，忘了告诉你了，我的小名叫宝儿，现在长大了，爸妈都很少这么叫我了，亏他还记得。"黛芊突然间的热泪盈眶，吓了云国一跳，黛芊蹦跳着，唱着歌，泪还没擦就笑了起来，云国以为她在笑他的小名儿，也跟着傻傻的笑了起来。

　　"宝玉来了！黛玉心想，这个宝玉，不知生来是怎样一个惫懒人物，不见也罢了。心中想着，已进来了一位年轻的公子……束发嵌宝紫金冠，二龙抢珠金抹额；二色金百蝶穿花大红箭袖，五彩丝攒花结长穗宫终，石青起花八团倭缎排穗褂，青缎粉底上朝靴……面若中秋之月，色如春晓之花，眉目如画……佩一美玉。黛玉一惊，怎似在哪见过一般，何等眼熟到如此！宝玉也在打量着黛玉，两弯似蹙非蹙笼烟眉，一双似泣非泣含露目，闲静时如姣花照水，行动处似弱柳扶风。'这个妹妹我见过'……'颦颦'两字极妙，极适妹妹的……美玉无瑕"

　　"是你，云国，云国"黛芊醒了过来，梦终于续下去了，而更让黛芊欣喜若狂的是，梦中终得一见的宝玉，那么眼熟，原来就是云国嘛。黛芊舒服舒服的伸了一个懒腰，回头一看，门口站着云国，手中仍旧提着外卖，不过看包装似乎不是他打工那家的外卖，今天也没穿店服，稍带有一点闪光感觉的圆领丝质 T恤，与皮肤的颜色融合的很好，将西装的严肃凭添了几分神秘的光彩，那一份帅气被发挥得淋漓尽致。我几生几世的守候啊，黛芊轻轻的叹着。

　　云国慢慢走了过来，口袋里装着穿硕士学位服的照片，怜爱的看着黛芊，心想：关于外卖的毕业论文终于完成了，硕士学位也终于拿到手了，呵呵，体验送外卖的生活到此结束，可怜这傻丫头还一直以为我是一个送外卖的呢？
