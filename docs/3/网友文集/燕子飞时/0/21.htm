穿起回忆晒太阳

　　我没想到我们现在就已经可以这样频繁的怀旧了，可是，很多曾经属于我们的东西确实已经成为了过去。不会再重来的一切以一种固定的姿态留在回忆中，在我们各自不大却明亮的角落里发着熠熠的光，沉寂，却永不消失。

　　三毛，一个已然沉淀了一段时间的名字，她的灿烂在上个世纪化为永恒，在人生这样一个没有硝烟的战场，她带着一贯的传奇色彩突然退出，众说纷纭，可我始终希望她在那一刻是无怨无悔的，最后映在她脑海中的影像也许就是那平沙万里的撒哈拉吧。

　　初识三毛，是用一种排斥的心理。因为当时最要好的朋友十分的不喜欢三毛，受她的影响，三毛在我的心里被刻画成了一个疯子，不会享受生活，天生喜欢四处乱跑的疯子，在她屡次摇头否定中，我想买一本三毛的书了解一下的计划也屡次被粉碎，居然被她蒙蔽着"同仇敌忾"了长达一年之久。

　　当属于三毛的"撒哈拉"那样别致地轻轻在我面前展开，我沉醉于对那种浪漫，那种勇敢，那种逆境中的智慧的欣赏，于是，我深深地庆幸于我偶然的发现。那样的一片沙漠海，三毛依靠着爱情和坚强白手成家，于是就有了棺材变成的家俱，于是就有了旧汽车轮胎装饰成的温馨小座位，三毛凭借着无数个不可思议的创造，在这样一个没有花朵的荒原开出了春天，千千万万个女人骨子里可望不可及的浪漫，让勇敢的三毛发挥的淋漓尽致。

　　眼泪中体会出来的微笑有着更加强烈的感染力，想不喜欢这样的三毛有多难啊！

　　三毛的文字如同她的演讲，不愠不火的情绪，不紧不慢的语速，那样平和，不经意间那一段岁月便如同一条河，静静流过心房，留下的痕迹添满了对往昔的怀念，每一个人自己的感想大概也融在其中变成了一种向往。

　　可是却总有一些人喜欢不择手段的扯着名人的尾巴，希望这样别人能够分一点注意力给他。关注三毛的人中，这么做的也不乏其人，甚至在她死后，居然还有一个非常无聊的人据说是顺着三毛书上写的地方一个不漏的访查了一遍，自认为证据确凿的开始揭露三毛作品所谓的漏洞。

　　可笑。其实，荷西是不是存在有什么重要呢？三毛和荷西的故事的可信度的高低又有什么重要呢？三毛书中的经历是不是真的又有什么重要的？重要的是一种对生活的态度。无论作品中的经历是否真实，反映在作品中的生活态度却是实实在在属于作者的。我喜欢三毛那种对理想的执着，对爱情的追求，对生活充满希望的生活态度，那种浪漫带着唯美的色彩姗姗走来的时候，立刻去感受这种赏心悦目是多么美好的事情。何必去追究这种色彩是蜡笔画出来的还是其它的什么画出来的？那是一种多傻的作法。于是，我笑过之后，对三毛的挚爱半分未减。

　　生是一种开始，无论多么美化了的死亡都只是对于生的一种结束。我宁愿相信三毛是被自己的执着累坏了，累倒了。那么就让她好好休息吧。给予她生前不能够拥有的宁静，也许就是爱她的人目前能够给予她的最大的怀念。

　　一页页的翻过了三毛的生活，点点滴滴的记录着热爱生命的旅程，我好象能够望见那长长的裙，长长的发，在阳光下飞舞，以无比绝妙的姿态晾晒着一串串长长的回忆。

　　执起那份美丽，轻轻穿在我的回忆里，艳阳天，也晾出那么长长的一串爱意………，晒太阳。
