想起今天有个节

　　从小就对军人很有好感，妈妈总是说，我小的时候很怕生，从不肯让不熟悉的人抱，只有一个例外，如果是解放军叔叔，那我就不哭，还乖得很，小手摸着军帽上的星，笑得很好看。

　　心有千千结，不知道我这个情结从何而来，不过可以肯定绝对不是来自于什么"拥军拥属"这类口号的鼓舞，这类口号对于我来讲实在是没有什么具体的印象。我想我这完全是一种天生的，很自发的情感。

　　爸爸还在的时候，我还很小。爸爸有时会带我上班，做为一名市政府的干部，不办公事的时候，他却从来不坐配给自己的车。所以，每次都是抱着我在市政府走进走出的。

　　那个时候，我最爱看的就是在大门站岗的警卫，身着军装，英姿勃勃的站得笔直，每次我都会看得目不转睛，走过去了也要转回头看，扭得脖子酸酸的，直到看不见了为止。时间长了，警卫们也都知道有这样一个小姑娘喜欢看穿军装的叔叔，换了岗也都喜欢逗我玩，"妞妞长大了说不定要嫁个军人哦！"

　　门口的警卫换了又换，我也看了又看。留在记忆中的并不是任何一张具体的脸，想起的只是那一身庄严的橄榄绿，让人感到无比的安全、踏实。我想最初的情结会不会缘自于这里？

　　后来，爸爸就不在了，我的习惯却保留了下来，再后来长大了，就不太好意思明目张胆的直盯着人家看了。

　　上学的时候，老山那里仗打得正凶，每时每刻都被无数个英雄的故事感动的一塌糊涂，那个时候，我就知道了有这样一群穿着军装的人在开满老山兰的老山前线保卫着祖国的大门，小小的我，不知道该怎样表达这样一种情感，好在有老师的教导，知道这是一种神圣的使命，于是，小小的心灵真的就升起一种神圣的向往和钦佩来。

　　人说现在的兵，兵痞多。痞就一定不是好兵吗？年复一年，中国这么大的地方，不是这个森林大火，就是那个地区大水，这个时候，冲上去的是谁呢？也许有人会说，那是没办法违抗命令。他们是没办法违抗命令，那他们去了之后不那么卖命行吗？行！可是他们却勇往直前，不亚于在真正的战场，每当看新闻的时候，看着大火中他们的脸，被熏得黑黑的；大水中奔波着的他们，脏瘦脏瘦，虚弱，却没有一丝的懈怠；看着看着，我就会哭起来，这种泪是忍不住的，没有任何做作的成份，很单纯的一种感动，这样的人民子弟兵，绝对无愧于那一身的神圣，这样的人民子弟兵，我喜欢的理直气壮。

　　有时和朋友走在街上，遇到有军人走过，了解我的朋友总会开我的玩笑："瞧，你喜欢的兵哥哥！"

　　"嘻嘻，对呀，我就喜欢！"我回答的快又直接。

　　于是，想起今天有个节，八一建军节。
