双双嫁给电脑的手

　　我嫁给了电脑，在我们相识的最初。从这个角度来看，我似乎属于电脑比属于我的主人还要多些。

　　说实话，我的清晨和黄昏还是很幸福的。因为我美丽的主人，和我拎着的美丽的提包被人注意着，倾慕着，我也格外努力的白暂、优美，最宝贵的一点，这段时间我优雅且安静，随着我主人的行走轻轻摆动，春风不如我得意，秋阳不如我灿烂，夏雨不如我洒脱，冬雪不如我轻盈。公车上有意无意的碰触，就算是我最痛苦的事了。

　　你开始对我流露羡慕的眼光了，我知道。但是你一定要坚持听完，再直抒你的胸臆，世上什么都有得卖，就是后悔药没得卖。当然，如果你一定要羡慕下去，我也会珍惜，因为下一分钟，我就能猜得出来你的变化，你的眼睛会出卖你。

　　我的痛苦起始于包办婚姻。我的主人在看上电脑的那一刻，就一厢情愿的认为那就是我最好的归宿，而且还嫁一赠一，陪嫁了她的一对眼睛。从此后，我便和电脑耳厮鬓磨，日夜不分。眼睛还可以左顾右盼的溜溜号，我却时时刻刻为了键盘和鼠标这两件家务操劳着，就这样堕入了水深火热的生活，呜呼哀哉！

　　你眼中的同情能滴出水来，我可不需要。不过，我也不奇怪，大多数人听到这里，和你反应是一样的，所以我也不怪你，只是你还是什么也别说，让我说下去。

　　我主人每天要进行大量的文字处理工作，既然把我嫁给了电脑，我的作用就是作好她的代言人，我也只好每天不辞辛劳地在键盘上跳来蹦去，跌跌撞撞。只有主人转用鼠标的那一会儿，我们中的九个部分都可以歇一歇，只有食指，因为生的地理位置很重要，所以我们只能趴在那里很悲哀看着她喘着粗气继续蹦跶，一刻不得闲。十指连心，所以我想我的主人心情也不爽，笑容节省到只留给"上帝"。

　　灰尘无孔不入，工作一段时间我们就都灰头土脸的了，我的主人这个时候到是很善解人意，马上把我们丢在水里洗涮洗涮，个个唇红齿白的，就精神饱满了，可是有一点我很不喜欢，就是每次洗完都要把我们涂啊涂的，滑腻腻，香喷喷，害得我只好屏住呼吸，差点背过气去，这个问题我私下里怂恿过电脑，适当的抗议一下，可是他说他也无能为力，尽管这种东西他也要分担。

　　我是唠叨了点，可你也别一付要睡着的样子吧，更痛苦的还在后面呢，知道我为什么最珍惜清晨和黄昏吗？一言难尽，简单的说，就是只有这两个时段我是自由的，和我主人一样娴雅的，这两个时刻的我很休闲，再确切点说，我放风了，幸福就被关在我主人的门槛外，槛内就是我主人的家，也就是我另外一个牢房。

　　夜深人静的时候，我又被拽到电脑身边，看着电脑也一脸的疲惫，我也不好再发脾气，只好蔫头搭脑的晒月亮，听着耳畔一声声"猫"的惨叫，就知道主人又爬网上去了，我就大大的松了一口气，哪怕在网上聊天，也要比白天轻松得多，于是眯缝起双眼，睡意弥漫，满桌子淌得都是九个手指的口水和右手食指的眼泪。
