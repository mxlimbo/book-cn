有一种柔美在边陲

　　一种淡淡的怅然，若有所失，又带着那么一点点的不甘心。每当看到报刊杂志有涉及东北三省的内容，却只是提及黑龙江和辽宁，而没有任何关于吉林的只言片语的时候，我的心里就是这样一种感觉。

　　一个城市，一个地域最动人的一个侧面当然就是城市中的许多女人，她们的装束、谈吐、生活方式各方面都一五一十的反映出一个城市的风情。来过东北的人大都对哈尔滨女人的热烈，大连女人的时尚有着强烈的印象，且津津乐道。吉林却始终不在话题之内，似乎吉林的女人都人间蒸发掉了。仔细想想，其实也不奇怪。

　　哈尔滨有"冰城"、"小巴黎"的美誉，且有太阳岛这样一个美丽的地方，来东北的人岂有不去之理？大连的服装节也是颇为引人注目的，且城市被花草装点的动人无比，带着强烈的时尚感在重多的北方城市中独树一帜，自然醒目，是决不可放过的游览之地。于是，酝酿出两地女孩子的出众特点，又占着地理优势，自然由南来北往之客将赞美之辞带向祖国的各个角落。

　　而吉林省却始终象一个陪衬，不尴不尬的卡在另外两省中间，不免有些讪讪的。吉林的女孩不美吗？当然不是。一个省最重要的体现在省会，吉林的省会长长春有多所大学座落，但却还是很难营造出上佳的氛围，而且似乎越是高科技的地方，就越是装腔作势的厉害，尽管有电影节，有伪皇宫，却没有办法酝酿出出色的韵味，我在长春四年，长春不缺漂亮的女孩子，和大连和哈尔滨的女孩子相比，缺的也许只是那种可以让人心底有所震憾的特点吧。而来吉林的人大都只来过长春，于是，整个吉林就这样被下定论了，就这样被迫的默默无闻着了。

　　其实，女人就象是海底的鱼，根据不同的海水深浅，而有着不同的特点。海的纵深处，虽没有浅海的繁华与斑斓，但是那份宁静、详和、柔顺却是独有的特点。

　　在吉林的东部纵深处，祖国的边陲——延边朝鲜族自治州就生活着这样一种柔美的"鱼"。地域不大，女孩子却极有特点。

　　这里的女孩子最大的一个特点就是温柔，当你第一次来的时候，下了车，最先接触到的大概就是女司机了，这里女性的司机比较多，妆容精致的做在司机的座位上，那份温柔自然而然的从言行举止中表达出来，温婉可人，无任何造作成份，体帖，可亲。尤其是那说话的语气语调，温软动听，不知不觉的攀谈中到了目的地，旅途的劳乏顿时会减轻不少。

　　受韩国影响较深的装扮方式，轻松、随意中一份份美丽飞扬在城市的大街小巷，姣好的象牙色皮肤，润泽的一张脸，五官精致，带着一种难得的宁静，不张扬，不拔扈。

　　这样的一种美是兼有北国的豪气和南国的雅致的。她们可以大口豪爽的喝酒，也可以秀气的坐在陶吧，安之若愫，心静如水。这样矛盾的统一，使她们身上散发出一种柔柔的光，虽不夺目却倍感温馨。

　　这里的女孩子不但夏天喜欢吃冰淇淋，冰天雪地的冬日也喜欢吃冰淇淋，坐在温暖如春的房中，看着窗外漫天飘雪吃着冷饮的女孩子该是多么惹人疼爱。

　　裙角永远是一份永恒不变的风景，春夏秋冬，虽是四季分明，爱美的女孩子们就也有四季分明的裙装，将这城市打扮得愈发迷人，一种别样的情怀就这样装点着祖国的边疆。

　　不要吝啬你的脚步，请将它迈向没有尽头的深处，都说北京女孩大气，上海女孩洋气，重庆女孩爽气，广州女孩忙碌，哈尔滨女孩热烈，昆明女孩靓丽，西安女孩清雅。同样的时尚，不同的女人。做为一个中国女孩，我为自己是这样千娇百媚的一份子而自豪。做为一个中国吉林延边的女孩，我为自己是这样丝毫不差于其它地域的女孩，且安然自若的一份子而骄傲。

　　我们的祖国自然是百花齐放的，只是无论如何，不要忘记祖国的边陲有这样一份柔美，轻舞飞扬。
