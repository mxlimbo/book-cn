守望天使——罗大佑

　　看了一篇有关朝鲜的文章，我的感动来自于这种对于朝鲜美好的记忆，做为一名朝鲜族的女孩，对于朝鲜，除了国家和国家之间的友好情感外，自然而然的带着一种民族情结。相同的语言，相同的风俗习惯，我知道隔着一条鸭绿江的，是我一脉相承的兄弟姐妹。

　　七彩缎的只及胸口的短上衣，红艳艳的长纱裙，是我小的时候最常穿的一种民族服饰，明艳的色彩中，渲染着我的欢笑，童年也就在记忆中五彩缤纷的闪烁。慢慢长大，受环境的影响，民族服装渐渐变成了节日或家族活动里才穿的服饰。外祖父的寿宴上，我穿着我最喜爱的一套民族盛装，洁白的主色系，领口绣着淡粉色的小巧的花瓣，袖口和裙角渐下渐深的粉色，同样绣着五彩的小花瓣，白纱下是白缎子的衬里，长长的快要拖地的裙微微地篷开着，漂亮的衣结下飘着长长的飘带。盈盈叩拜的时候，不禁沉醉于这种服装带来的隆重气氛，寿宴过后，将它重归衣橱的时候，开始盼望着下次绽放的光彩。

　　家里没有录音机的时候，音乐来自于一台唱机，一撂撂的唱盘放在柜中，是朝鲜的歌曲和延边歌曲，朝鲜的歌曲通常是厚重的黑唱盘，表明它质地优良，封皮上是年轻的姑娘，身着熟悉的服装，在秋千上荡漾。一首一首的跟着哼唱，我从来没有把它当作过外国歌曲。就象来家中作客的朝鲜亲属，相同的语言带来的亲切感，让我意识不到他们其实是外国人。

　　亲戚的亲戚家住乡村，村外的河，是一条支流，河的对岸就是朝鲜。一次假期去玩，沿着河边儿闲逛，窄窄的河床，窄到可以清楚的看到河对岸正在嬉闹的小孩的脸。一望无际的绿油油的稻田，隔着稻田远远的，后面是一幢幢墙壁洁白的民房。我展开着和作者同样的想象。当我手中挥舞的手帕得到回应的时候，我激动的双颊发红。

　　最近，关于朝鲜的一些消息令人恐怖的漫延着，我却始终不能相信我们曾经援助，并与之并肩作战的国家会变成传言中的那个样子。正巧前一段时间，姨夫与他的好友一行去朝鲜旅游，姨夫的掌中宝帮我录回了朝鲜的美丽。去的时候，姨夫带了许多的糖果与口红。这是最实用的礼物，在车上将口红和糖果赠给售票员和小孩子的时候，她们眼中的欢喜并不妨碍她们不卑不亢的致谢。在山中游玩的时候，每隔一段路就会有一个失物箱，不论是游客还是本地人，见到路上遗失的物品，都会送到失物箱。同样，谁丢了东西，就可以沿着山路的失物箱中找寻，十有八九都可以找得到。于是，我很高兴的知道，朝鲜虽然相对来讲暂时是贫穷的，但是一样保持着她的尊严，秩序井然，她只是需要时间飞跃。到了城区，合理的城市规划展现给我们一个美丽的城市——平壤。优美的配乐声中，跟随着镜头，在这个朴实美丽的城市生活着的人民，同世界上其它国家千千万万的人民一样，勤勤恳恳的忙碌着，一脸的平和。让我没有理由相信，他们就是传言中所说的生活在水深火热中的群体。

　　看着三八线附近的游览区里给游客从井中汲水喝的朝鲜战士，他们脸上的诚恳让你如沐春风。井水便更加甘甜的入口，缓缓滋润着心田。

　　自然而然的想起了这些事情，我记忆中的朝鲜也同样那样美好，而且希望她会永远美好的存在下去。对于我来讲，这不仅仅是一个美好的祝愿，它还包含着一种期望，期望着那里的亲属生活能够更上一层楼，期望着那和我说着同样语言的国家的繁荣昌盛，能够与我们并肩屹立在亚洲的东方，生生不息……
