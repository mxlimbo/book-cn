日　子

　　还是在春暖花开的时候
　　我就知道,你在听我吹笛子了
　　
　　那时候我们过的是那样的日子
　　
　　我们没有种很多花
　　而是种很多菜
　　我们去集市上卖菜
　　我们看着很多菜
　　还看着很多人
　　
　　闲的时候,你写诗
　　我就在河边柳树下吹笛子
　　骗黄昏来
　　黄昏之后月亮长大了
　　月亮和你托着腮听
　　
　　后来日子也长大了
　　我们不必蹲在集市上
　　没着没落地东张西望了
　　
　　你还在写诗,
　　写那些能给穷人安慰的诗
　　再骗穷人的钱
　　然后是秋天来了吗
　　我不再欺骗黄昏
　　你再听不到我吹笛子了
