送你一匹马，送你一匹马

　　送你一匹马：男，天涯网友，济南电台
　　冰冰：女，天涯网友，海南人

　　那天是黄昏，这样的清秋，你，冰冰和我离开海边，走在沿海边的一条路上，去见雨人。我们都不会忘记，路，是那么长，深深浅浅的，高高低低的绿的颜色，零星有淡淡粉色如紫金绣帖颜色的小花串串心不在焉地点缀。晚风里，送来花香，有一点点凉，就是季节交替时候那种空气里转变的震动，有一丝怅然――很清爽的怅然，不浓的，就似那若有若无的花香。

　　还记得当时，我穿着件淡蓝色裙子，中间高大的你白色的T恤，冰冰温婉可人的一袭紫色纱裙。

　　传说中有阿拉伯人必喝的三道茶。

　　第一道苦若生命，第二道甜似爱情，第三道淡如微风当时我想，我们的友谊，就象这茶，淡如微风，淡淡氤氲在飘散渐远的花香中。

　　雨人笑问：每个人都送了一匹马了吗？你神色飞扬，一脸豪气：送了！！

　　是的，马儿，你送了我们每个人一匹马，那，就是你的快乐，如此健康的快乐，这快乐感染着我和我们，让我们在你策鞭打马回济南的时候，我们对我们所得到的礼物，是如此沉醉。

　　亲爱的送你一匹马，我得到了你送我的一匹马。那是一匹有生命才懂得去爱的马。若干日子后的今天，我也要送你一匹马，因为，我要这神马载你过隙，直到天涯中的我们再次能看到你在奔驰的骏马上，头发飘动，神采飞扬的样子。
