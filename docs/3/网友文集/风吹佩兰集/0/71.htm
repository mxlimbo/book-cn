无法悲伤

　　下了车，我突然迎着夜风闯到了马路中间。

　　我飘似的在马路中间。

　　满大街灯红酒绿，车水马龙，人来人往。

　　我看见若干辆车在朝着我的方向横冲直撞。

　　有几辆觅食的TAXI，一辆累累硕果的垃圾车，有一辆小昌河。

　　混乱。

　　我在激动，还有兴奋。反正我迎着猎猎的夜风飘似的冲到了马路中间……

　　我衣裳起舞，长发纷飞。

　　我突然停驻，四顾茫然。

　　我期待着期待着什么期待着什么和什么什么什么的什么

　　可是没有发生什么

　　很混乱的秩序突然在我面前镇静下来

　　车纷纷停驻。

　　衣鬓纷飞的我在马路中央。

　　后来我灰溜溜地走了

　　灰溜溜得连个骂我的人都没有

　　胖胖后来就付了车钱出来了

　　我站在路边灯火阑珊的地方等他。

　　我张了张嘴巴 可是我没说

　　其实我想说：我想死去，我真的好想现在就立刻死去。

　　下午胖胖带我转来转去看房子。

　　三年前年轻的他已经买了房子给我。

　　现在他看上了那个三倍大的。

　　他在办公室兴致勃勃地望着我，笑咪咪地说："等将来，我们再买靠海边的最漂亮的房子。"

　　斯域说胖胖把你宠坏了。你周围的人都在宠你你不觉得么。他噼里啪啦地列举了一大堆人名。后来他说，我也是一个。

　　迎着夜风，趁胖胖付车钱先行下车的我，想独自拥抱那车流。

　　突然间，我很想泪流满面。

　　其实我很想对老天说：求求你，让我可以现在死去。实在实在在实在是因为我的生命，已了无遗憾。

　　我朵朵寸寸的幸福，早已丰盈欲滴。我总是盼望在这幸福轰然怒放的时候，可以满足我能孑然死去。

　　就象一段爱情。请在心犀相通欲语泪先流的时候告别。

　　就象一种幸福。请在洒了又满了无遗憾的时候让我离去。

　　离别，它总能斩断缠绵，因为它成就完美。

　　我记得从前闲谈的时候天骄说他曾经怕死。

　　这个念头让我羡慕了好久。

　　要怎样的代价才能让人惧畏死亡！

　　我好羡慕他，他竟然可以有那么多期待与盼望，让他怕死。

　　可是，为什么我的幸福总是丰盈欲滴，我对生活说，哦够了够了够了足够了

　　可是生活仍然不住地给我太多太多。

　　我想试图悲伤。可是我无法悲伤。
