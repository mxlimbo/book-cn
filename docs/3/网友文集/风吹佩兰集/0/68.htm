偶尔忧郁

　　小时候喜欢记日记。

　　现在也喜欢把东西先落到纸上，再放到电脑。

　　好象真的那么自恋。

　　我喜欢游动于我笔尖的每一个字。

　　真的觉得，笔尖划过的时候，每个线条都变得，有灵性。

　　一拆一编，一拆一编……仿佛给本子织了一件素色的衣。

　　美丽的文字，美丽的线条让我情不自禁地想吻吻它……

　　比如说现在。

　　入夜的时候，嘴唇总是如此柔软

　　纸是如此地平绵温暖而芳香。

　　不动声色的芳香，有着墨水淡淡的墨香

　　香得自持。

　　把头埋在文字里，心，突然安宁。

　　就象秋后暖意正好的维多利亚时代的麦场。

　　就象阿尔玛.德台玛的《94级荫凉》

　　那种颇有文化质感的乡村情调，好象很符合我们这样偶尔忧郁的人们。

　　远远近近的层次，树林，麦场，草丛，温暖，明净而安详。

　　这样的环境是无法不使人们因可能的避世而感到陶然的，而白衣男子臂下的书籍与文字总使人感到自持、尊重、与高贵。

　　其实，总会有时偶尔忧郁，因为我们的敏感而优秀。听说上帝造我们的时候，就顺手把模具打破了

　　因此，世界上就再无一人与我们一致，我们，是独一无二的，自己。

　　顺着自己的眼睛往下看——，你的鼻梁，你的身体，你的柔软的臂，
你的灵巧的手，你的用于行动的腿和脚……，只有自己对自己如此的忠心耿耿，不离不弃，做好自己，别对不起自己，只可以，偶尔忧郁。 ：）
