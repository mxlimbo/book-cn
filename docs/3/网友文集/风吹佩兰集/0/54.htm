伊

　　伊：上海女子

　　是的，我对伊人还是蛮有兴趣的。原因之一是醉拍春衫惜旧香的吹捧。

　　秀庐那帮子以亲耐的老椰为首的一批老中青同志，委实酸得可以。

　　我常常想象相忘江湖、伊之流的同志对我的狗P文章一定是一边小心地皱眉，一边轻轻地扇着鼻前的空气看的。这还算是抬举了我自己，我指的是如果他们有兴趣看我的破东烂西的一眼的话。

　　这依然无法阻挡我对伊同志的兴趣。

　　而伊的回答是我此次赴沪之行最令人兴奋和幸福的卖点之一。

　　她说："我不想见你。"

　　我按捺不住心头的狂喜，但依然假模假式地娇滴滴地说："哎呀，真的吗，你真的不想见我吗，很多女孩子都很想见我耶~~~~~"

　　当时我听到我肉麻的声音从同济实验室中袅袅地飘出来的时候，我忍不住大笑着想吐出来。

　　"是的。"伊认真的说。

　　没有一次见网友的经历比她的这一句回答更让我兴奋不已。

　　不为别的，为了她的能坚守自己，不妥协。

　　我喜欢。
