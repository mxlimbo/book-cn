玫瑰灰

　　玫瑰灰，天涯网友，网络作家，上海女孩

　　灰灰是个很不错的女孩子。

　　不错等于：认真，善良，负责。

　　我喜欢做事认真，为人善良，承担责任的人。

　　人有了这三点，他的人生肯定光明了一大半。如果再加上意志和毅力，那就亮堂透了。不过和灰灰见了两面，还不知道她透没透，等她老了看看再说。

　　灰灰邀请我去戏剧沙龙去看一个网络有关的先锋戏剧的彩排。

　　我快乐地在黄昏时分，溜了过去。

　　我先是深深地迷恋了一下戏剧沙龙庭院幽深清新与曼妙。

　　然后就听到两个男女演员在场子里发情般的对白。

　　暗的灯光，舞台上有个二层的可爬上爬下的铁架子。

　　那对狗男女又跪又滚煽情之至地表达网络聊天室的对白。

　　俺亲自捡了个最靠前的位置，以便查看女演员有几颗雀斑男演员有几道抬头纹。

　　我知道灰灰就在场子里和我一起看戏，而我不知道哪一个是。

　　灰灰曾经有一张悄悄给俺看的她老人家亲自吹嘘了很长时间的照片，而至今俺仍对照片上五官找不着感觉的她耿耿于怀。

　　当男主角和女主角在当晚生日PARTY大闹大嚷之后，女人那样楚楚地立着，男人比画上来一支牙刷，满场子莫名其妙地在嘴巴里捅过来，捅过去。女人注视着他。

　　"哎呀哎呀哎呀哎呀……哎呀…………"那么好听的音乐起了

　　女人旋到他身边，静静地，斜着美丽的眼睛忘着他，替他抓住那牙刷在他嘴巴里刷过来，刷过去。

　　俺的眼泪亲自掉下来，而心里还是放不下找到灰灰的这个事，就一脸祥林嫂表情地回头四处张望——

　　于是我看到了，有着异常美丽的鼻子的灰灰美女，正在和她亲爱的水月居士，甜蜜的相拥与对视……
