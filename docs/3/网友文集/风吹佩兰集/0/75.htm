只想做个女人

　　我的那些所谓的狗P"才华"只是给大伙儿看着消遣的。抛开我那些"仿高深"、"拟渊博"、"超高贵"、"伪大雅"，在别人眼中，只希望自己是一个很普通的女子。

　　不去关心输赢，知道自己也是个很容易调整好心态的家伙。愿意在和你的交往中，你高兴，我也高兴，生活的质量也在于乐趣，我愿意因为我简简单单的存在，使你眼前一片清新。

　　相信你不会介意我上窜下跳的调侃，更多的时候，我只是希望你听到我的胡说八道，不感觉寂寞。

　　我愿意给你我的温柔，就象一把蓬松舒适的降落伞，在哪怕飞机坠毁的时候，让你的心灵依然感到安全与宁静。我愿意看到你在我面前的发泄，哪怕暴跳如雷，哪怕心如刀割，哪怕垂头丧气。。

　　我知道我也许帮不上什么，可是我愿意倾听你的心情，明媚的彩虹，总在倾盆大雨之后。

　　我愿意赞美你和鼓励你，尽管你会说我是拍马P，假道学。可是你应该知道，我所有的赞美都是最由衷的，所有的鼓励都是最真的，我也愿意你能以你更多的进步，来让我对自己的预言有更多的成就感：-） 。

　　我愿意你做你最真的自己，你不喜欢什么就不用去做，我不愿意你为别人的鸡毛蒜皮的狗P看法，改变自己太多。喜欢吸烟，就不用戒，不喜欢天天洗澡，就不去洗。开开心心，做你自己。

　　不愿意你累，尤其是心灵。我愿意我的温柔是一只电熨斗，不假声色地熨平你所有的焦虑与忧伤。我愿意尽力去理解你，每个人都有自己的空间，这我知道，我会感谢你对我的善意的谎言，那也是因为你对我的在意。当看到你开开心心的时候，我也愿意躲在我的一隅之地，安安静静开始用心做自己的事情。

　　我也愿意，在你心中，我只是个女子，普通又朴素的女子。

　　我愿意在身体不好，汗如雨下的时候，有一个膝盖可以让我蜷缩着，舒服地躺下。我愿意在虚弱无助的时候，读出你眼中切切的焦灼与关心。我可能会不好意思麻烦你，而和你忘不了地客套，而你的坚持仍然会让我有着另外的感动。我也许也会在寂寞的时候想起打电话"骚扰"你，我也愿意听到你明媚的声音无一丝阴霾。也许我也会发一些没来由的小脾气，小性子，那是因为我愿意从你的介意中得到你在意的讯息，尽管，不是为了爱情。

　　也许在那么秋凉的夜里，我会希望疲惫的你陪我去看月亮，做些稀里糊涂的傻事情让你的心好烦，也许我也会吃些没来由的醋，说一些酸不拉几的话，夹枪带棒的挖苦与讽刺，让你无可奈何，左右为难，那可能真的与感情无关，只因为，我是个普通的女子。愿意衣着美丽，当你走过，我长成一棵正开花的树，我愿意我满树的芳香沉浸着你每一个想象。我愿意在见你之前有足够时间礼貌的打扮，我愿意我小小的绰约施舍给回忆，一个最耐人寻味的梦。只因为，我是个普通的女子。

　　只因为，我只希望，在你眼中，我只是个有女儿性情的普通女子。
