谈余华《在细雨中呼喊》中同性恋式的友谊

　　余华是个喜欢描写死亡的作者，他笔下的每一次死亡，就象他沉着地抡起偌大的生锈的铁锤，重重地向读者心头砸去，不知是否是因为情节真的需要，还是做为男性作者的一种表达自我发达的"胸肌"的方式。无论如何，《在细雨中呼喊》中描写的孙光林与苏宇的一段同性恋式的友谊，顿时象在微风细雨中，一个初恋的少年看到了他年轻的恋人飘散的头发一样，使琐碎而阴暗的故事，变得神采飞扬。

　　孙光林是踏着松软的泥土和迎着迎风起舞的青草中，逐渐长大的农村孩子。家庭的贫穷、龌龊、扭曲与歧视，他从小就已习惯于一个人默默地坐在池塘边，思索。他有一个叫孙广才的父亲，那是个彻头彻尾的无赖。

　　就在他十二岁的时候，一家苏姓的城里人搬来他的村庄居住。苏家的两个孩子穿着商店里买来的衣裤。孙家兄弟，则是光着上身，象阳光下黑黝黝的三条泥鳅。

　　孙光林独坐在池塘边默默无语的时候，他神往的目光便长久地落在苏家那文明，和谐，温柔，友爱的气氛中。苏家成了他的永远的向往。苏家的长子，就是苏宇。

　　苏宇是个纤弱，文静，总是在路边走路的大光林两岁的男孩子，余华心情模糊地写到苏宇由于长高，裤管高高吊起，露出的一段脚踝，似乎，这一段脚踝曾长久地吸引过他的目光。光林从没有勇气和苏宇说一句话，他自卑，只是已自己的姿态独自在校园里。

　　光林永远也忘不了这个极易害羞的苏宇，第一次和他讲话时满脸通红的样子。

　　那是学期快结束的时候，苏宇走向了在路的另一边的光林，说："孙光林，你还认得我么。"

　　光林的泪水几乎要夺眶而出，他轻轻点点头， 说道："你是苏宇。"

　　从此他们放了学不由自主地走在一起，可是，他们开始彼此为在校门口就这么快分手而感到不安了。苏宇开始送光林回家，知道看到光林走上回家的小桥，才轻轻挥挥手，走向另一条路。

　　在光林心目中，苏宇是那么美好的一个朋友。

　　然后发生了一件事情，应该说是每个少年都会遇到的生理上的事情。光林惶恐不安，又无法自控，一方面他痛恨自己，在白天陷入深深的自责中，一方面在夜晚他又无法抗拒。

　　他迷惑，因为没有父母来指导他，说这是很正常的事，他于是恨自己肮脏，他不去理苏宇，因为，他觉得自己已经配不上这么美好的朋友。

　　苏宇也惶惑了，对于光林的闪避，这时候开始，毫不知情的苏宇有了另外一个朋友，高大威武开朗的郑亮。

　　直到光林鼓起勇气对苏宇说这件事，苏宇没有一丝惊恐，而是很认真地解释给他。

　　一个微春的夜晚，那是光林难以忘记的时刻，苏宇走在中间，他的手一直搭在光林身上，高大的郑亮走在另一侧，他们谈笑风声，就在这个时候苏宇安静地提起了这个话题，他们三个开始悄悄议论起来，郑亮态度落落大方地说：晚上睡不着觉的时候，这么一下子很灵。

　　郑亮的神态使光林想起几天以前还在进行的自我折磨，此时，由于苏宇的默默帮助，使他的心情是如此的轻松自在。

　　再后来，苏宇入狱了，因为青春萌动的他在一个春天的夜里，袭击了一个少妇。

　　当他出狱时，光林怕苏宇受伤害，绝口不问不提任何这件事情。

　　而苏宇以为光林嫌弃他。当光林和苏宇郑亮到了一起的时候，他这位朋友的声调还是那么羞怯，但对话题的选择已不象过去那么谨慎，他直截了当地告诉光林和郑亮，他抱住那个少妇时的感受，苏宇说这话的时候脸上流露出明显的失望，那一瞬间，他发现女性身体和他想象的相去甚远。 他说："和我平常抱住郑亮肩膀差不多。"

　　苏宇的目光当时犀利地望着光林，而光林则是慌乱地扭过头去。不知为何，苏宇这句话深深地刺伤了他，正是苏宇这句话，让他对郑亮产生了嫉妒。

　　直到后来，两个人解除了误会。一个有月光的晚上，苏宇的手如光林期待的那样放在了光林的肩上，光林听到苏宇的声音极其柔顺地飘到他耳中："我劳教时，总担心你看不起我。"

　　他们在河边的石阶上坐下来，河水在身边潺潺流淌，悄无声息地坐了很久，苏宇说："我有句话要告诉你。"

　　苏宇没有立刻往下说，而是仰起了脸，去看那斑斓的夜空，月亮正象一片云彩缓缓地飘去……

　　苏宇继续说： "就是前几天告诉你的，我抱住女人时的感受——"苏宇的脸在黑暗里模糊不清，但声音十分明朗。

　　"其实不是抱住郑亮的肩膀，是抱住你的肩膀。我当时就这样想。"

　　月光下，可以看到苏宇生动的微笑。苏宇的羞怯与微笑，在那个月光时隐时现的夜晚，给光林带来长久的温暖。。

　　苏宇在19岁时脑血管破裂而死

　　光林想起了一个夜晚，苏宇摸黑到农村找他，为了告诉光林他要到化肥厂上班的事。

　　光林在高中毕业后，考上了大学，却再也无法象苏宇参加工作来告诉他那样，去告诉苏宇。
