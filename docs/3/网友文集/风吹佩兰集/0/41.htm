很认真的卓锐

　　卓锐：天涯著名网友，天涯社区技术主创建人

　　（想当初撞到天涯，心中第一个名人就是卓锐。他的大名在社区里如煮沸的米汤水在我心中绵延不绝。没办法，谁让这位是天涯系统的"老板"呢。）

　　卓锐肥头大耳小眼睛，不驮背，腿很直，走起路的样子很挺拔，即使是摇摇晃晃的时候，也很挺拔。

　　卓锐也很认真。

　　早在电话里聊天的时候，谈到他回老家的那次，他说：回家我爸妈拼命让我吃东西，我都胖了。他总怕我不相信他真的胖了，就叹息着说：不骗你，海口那么热的天气，我回海口的时候，已经胖得连原来的裤子都穿不进去了。

　　卓锐来青岛第二天，也就是与我的第一次见面，我领他爬墙。那一座短墙。

　　胖胖的卓锐决心不爬上去，要跳上去。

　　第一次：远离五米开外，他起跑，加速，冲锋——起跳！…… 马上就成功的千钧一发时刻！他掉下来了。

　　第二次：远离十米开外，卓老板往手心里吐了口XX，显然下了决心……起跑，加速，冲锋—— 起跳！…… 马上就成功的千钧一发时刻！他又掉下来了。

　　第三次：他很认真地爬上去了。

　　卓锐特爱记录，常备小本本，亮眼睛过客麦55及青岛帮等人七嘴八舌地说，他就认真地七七八八地记。可是，有一个时候他说什么也不肯写一个字……

　　临别时，我和段誉两个张罗着向他要礼物。卓锐鼓捣了半天，摸出一本他随身带的《未来时速》给段誉。又抠呲了半天，摸出一本贴身带的《迷你简明字典》来给我，这要真是英语字典我也就要了，可是这的的确确是本汉字字典！我学了二十多年的中国话，要这个啥用？！切！见我不愿意，卓锐眨巴眨巴小眼睛沉思着看看我，神情特幽怨，最后只好又在包包的深处抠出了一本刚买的《财富对话》。我和段誉于是逼他给我们签名留念。他S活不肯，拼命抵抗。僵持半晌，他沉痛地说：那好，等我练练。卓锐就找出记录我们意见的那个小本本，在上面用钢笔一笔一划地写。写不到一个半字，他又S命捂住，说什么也不肯给我们看，也说什么也不同意给我们签名。哪里可以饶他！结果…..结果……结果怎么样呢？

　　卓锐说：那我按手印吧，行吗？

　　在咨询过该按哪个指头后，只见卓锐非常认真地把蓝墨水涂到食指上，又非常认真地用嘴巴把食头吹得半干，非常仔细地在书上按了下去，然后写了个日期：2000，1，2。
