纳兰纳兰纳兰

　　纳兰：天涯著名网友，天涯社区筹划人之一，青岛女孩。


　　——评价一个人是不是真的浪漫，要看他是否可以安忍于痛苦中，有一些人是可以靠痛苦滋润的，既不会伤了别人，也不会伤了自己，这才是真正的浪漫者。（摘）

　　和纳兰通电话，她的第一句是："哦？你真的是女孩啊！"她的声音里有浓浓的童音，有很多伊能静刚出道时的《十九岁最后一天》的天真。

　　距离仿佛使我们没有了距离

　　我们决计见面，说不清的我便有好多关于见纳兰的可爱的想法。

　　下班前，由于和锦湖的约好见面，那家伙的没完没了真想让我咬断他的喉管儿！

　　阿弥陀佛，终于可以踏上美丽的路程，和纳兰见面。

　　春风得意马蹄疾！飞车上我，笑容是如此的红润

　　娇小的纳兰安静地埋在厚厚的沙发里，我立刻把她CATCH到

　　我冲她招手："嗨！纳兰！"

　　她笑开来，亮晶晶的眼睛眯在一起，象猫一样生动

　　纳兰的第一句话是： "你的眼睛红红的，一眼就知道上网多了，熬夜熬的……"

　　她嘻嘻笑着说。

　　我差点吐血！

　　不过哪！给堂堂斑竹大人这个印象，嘻嘻，文盲说啥话来着？咱是文盲咱好学？咱刚来，咱好学呗，说不定这小妞儿一心软，赏我几个红脸啥的

　　纳兰安静地在我面前，她的肌肤如瓷，眼睛很亮很亮，纳兰的眉毛是我见过女孩子中最漂亮的一种，轻轻松松我想到了李若彤演的小龙女

　　纳兰的耳朵上各有至少两个耳洞，别挂着银色的耳环。

　　纳兰笑的时候，脸上的笑容和光彩，使我仿佛可以听到最细最致的古瓷微微碰撞的清音，她的银色耳环便如檐前的风铃，在凉意的晚风中

　　告别的时候，我望着她娇小的身影隐在TAXI中，我发动了我的机车我想：我们都是浪漫的人
