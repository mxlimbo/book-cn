也谈网婚

　　求婚，是对一个女人最隆重的赞美。

　　说女人不喜欢婚姻，那是假的。

　　我赞成婚姻，也赞成网婚。

　　婚姻给人的感情找到一种视觉的永恒，尽管离多散多，但毕竟是往天长地久这方面做了努力。

　　网络使现实生活中所不可以的情感混乱具有了合理性。一般是IRC可以开多少个窗口，就可以泡多少了帅哥或小蜜，私聊更是网恋者的温床，哈哈。

　　网恋好玩么？据说网恋是刚上网者喜欢玩的游戏。

　　人总是能在愚弄别人中找到快感。何况得到别人的青睐和宠爱证明自己的魅力。

　　现实中的恋爱以婚姻为目的，看上去很沉重。

　　网络中的恋爱倒是可以尽情大搞，看上去很美。

　　可是刺激之后是什么呢。

　　无休止的空虚。

　　得不到的空虚，得到也空虚。

　　为了恋爱而恋爱，找不到清楚的路，等于没做功，没意义。

　　上网为了什么呢，为了放松。

　　放松的意义，也在于得到。

　　而无休止的空虚让人感到的却是失去的时候，我们需要的是网恋的激情，还是相互了解的稳定的友情网路历程的意义在于什么？

　　记得恩雅说：

　　我不想回忆

　　回忆是旧的，日子是新的

　　我不会煽情

　　煽情是假的，牵手是真的

　　我不能停留

　　停留是短的，行走是长的

　　虚假的东西，没有用，实实在在的学习和提高才最重要婚姻使人的关系在彼此激情的追逐后趋于稳定，和平才是发展的基础吗，嘿嘿。

　　所以我赞成婚姻，也赞成基于婚姻那种态度严肃的网婚。

　　好象公布了两个人的关系的同时，你会肯定地对自己说：哦，我需要的，就是这个人了。　　

　　而其实，今天的恋爱更象酒。现代人更喜欢喝啤酒。冰箱里冻一下，新鲜扎啤。越新鲜越好，用一海大的杯子，放着强劲的迪斯科，灯光闪烁，痛快酣畅。谁高兴放些软软的音乐，点些朦胧的灯光，用些细巧的杯子，说些甜糯的情话，抿些古旧的红酒的那种婚姻呢。

　　谁有这样的自信，说：你是我要的。我爱你。我要跟你一辈子。谁会这样置自己于死地。现代人都学会圆滑灵巧，立足于不败之地：哦。是吗。对。有道理。可能吧。让我想想。明天再说。 从不表态，以免被别人抓住痛脚。所以啊。两个人的相近相亲，就更成一个可望不可及的梦。

　　更见网婚弥足珍贵，哈哈。

　　至于网络与现实，我的概念里应该是分得很清楚才是理智。你在现实的世界里生活，在虚拟的世界里游走汲取和提高，再把所学所得用于现实。
