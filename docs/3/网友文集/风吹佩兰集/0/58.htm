爱你，跟随你

　　听了一首很普通的歌。一句歌词，却记了很久，甚至凌晨，
　　想到从梦中惊醒。拥衾而坐，细细地想，秋凉渐渗。
　　那句歌词是：……用一辈子跟随你。
　　
　　依稀往梦似曾见，心内波澜现……
　　抛开世事断愁怨，相伴到天边……
　　不知为何，这首孩提时会唱的歌萦绕。想必现在的新新人类
　　已是对它全然陌生。这是射雕第一部的歌。翁美玲拍的那个版本。
　　逐草四方沙漠苍茫，哪恨雪霜扑面；射雕引弓塞外奔驰，笑傲此生无厌倦。天苍苍，野茫茫，万般变幻。。。。
　　那么女人呢？
　　冷风吹，野茫茫，藤树相连；猛风沙，天苍苍，藤树两缠绵。
　　应知爱意似流水，斩不断理还乱……
　　无论天涯海角，无论世事如棋，相伴到天边。
　　……

　　也想起ESPN的频道放的一个关于火山的片子。一对夫妇，丈夫喜欢火山。妻子陪伴他研究。从美国的夏威夷，到印尼附近的爪哇岛；从太平洋的深海火山群岛，到日本的富士山。。最后，他们的生命一同消逝，在一次措不及防的火山喷发中。电视的画面，是火山在薄发！整座，不，不只是整座，而是整个的山系，山体慢慢变红，在山口，且是众多的山口，火焰喷薄而出，震慑天下的灾难，在视觉效果总是如此宏伟，火红的岩浆，以不可思议的速度与方式，沉着地滚动，滚动。。。火红的山体憔悴下去，黑红的岩浆怒吼着，铺张开来，它甚至于高傲而迷乱地扑向大海，义无反顾。海浪，滔天，怎样的壮美。想那夫妇，在大自然做最愤怒宣泻的时候，能一并含笑于火山之中，丈夫无怨无悔，他死于他的最爱；妻子无怨无悔：跟随你，因为爱你。

　　我也想到陶然亭。那个在北京，很想去的地方。与君一醉一陶然，固然惬意。而此时，却因为心里种好的那圃白玫瑰园，要去送给在那里安眠的，石评梅。石评梅只活了26岁。死于悲伤过度。在高君宇去世后不久。无须再解释，爱你，所以跟随你。
