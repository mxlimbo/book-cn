下雪了

　　突然感动起来。被自己感动。居然想起了《森林大帝》。

　　雷欧与莱娅。

　　有一册的第一个画页，浓密的森林，黯然的夜色，当中是一轮大大的明月，右下角是两个小狮子头。

　　上的是六年级。那一年的教室，那一年的那本图画书上，趴着两个小脑袋。

　　一个对另一个说：你说，这是什么意思？

　　另一个在想：该怎么说呢，表达什么深深思念之情？

　　这个就是我，我当时拼命在搜肠刮肚，想什么语文课文里边的段落大意中心思想。

　　我那个同桌说：天黑了，它的意思是天黑了。

　　看森林大帝，就象11岁的我也在森林里甜蜜地奔跑，当然还有我的那个12岁的同桌。

　　下雪了，就象回到了十几年前，我的童话，和我爱的，永远的森林大帝。
