最完美的网络爱情

　　想，最完美的网络爱情，该是这样的……

　　他们在网络上相识，该是在聊天室。

　　他们在询问过了年龄与性别后，发现彼此很相当……

　　他们慢慢地聊起彼此的生活，和对生活的感受……

　　他们开着小小的玩笑，却发现小小的玩笑能激发出那么多生活之外的，灵感。

　　他们开始是有节制地，很礼节地，说白白。然后各自有一丝莫名其妙的，怅惘。

　　然后他们有些不约而同地，在第二天的时候，可能是上班的时候，偷偷上网溜一眼，其实只为了看看那个带来一丝怅惘的名字。可是，没有。

　　于是，他们又艰难地，期盼的地，焦灼地，等待，等待他们曾经相遇的那个时间，那个时间是唯一的，和最后的希望……

　　剧情1 他们没有见到，而且，从那以后，再也没有见过……也很完美。

　　剧情2 他们欣喜地发现，他们又见面了

　　他们在尽力掩饰彼此的喜悦，故作矜持地再彼此问好……

　　缓缓流动的，是一种别样的心情……

　　慢慢地，他们……

　　有一天，他们如果可以有机会见面，比如说是四天。

　　这个来看那个，或者那个来看这个。

　　第一天：他们在酒店的门口见了，一个笑着，直视着另一个的眼睛。

　　这个垂下眼帘

　　夜里去散步的时候，风冷，慌慌乱乱地，一个挽住了另一个的手。

　　第二天：他们手牵着手四处玩，在一个特定得，合适得不能再合适的场合，合适得不能再合适的理由，合适得不能再合适的心情，有了一次令两人都终生难忘的轻轻拥抱。

　　第三天：在那个的办公室，等那个加班后的下班，离开办公室的时候，要熄灯。刚熄的时候，两个在黑暗中迟疑了一下，就吻到一起了。

　　第四天：这个要走了，在酒店里打电话给那个，本是想着要约那个去找件喜欢的礼物送那个。当那个来到酒店的时候，他们轻轻地牵手，微微笑着相拥，调皮地吻，后来他们都紧张起来，两个在秘密的甜蜜里战栗。最后他们似乎都在犹豫，都在挣扎，都在放弃，都在崩溃，溃不成军的时候，他们深情地做爱。

　　自此，永无消息，永不相见。
