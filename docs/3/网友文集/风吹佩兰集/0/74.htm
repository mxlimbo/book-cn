杂　思

　　流水一样的音乐，我的好心情就是这样得来的。

　　没有这样的一个时候，我是如此地想得到一个可以写字的本子。

　　来到麦当劳，逡巡半刻，笑意盎然，对服务生说： "我要一杯热水。"

　　于是我得到了一杯精致的免费的热水，找到了一张大桌子铺我的纸，我的笔，我满怀的圣诞节来临的，笑意。

　　哦，好象说多了，呵呵。我得到本子本不是为说这些的。

　　可是心里的忍俊不禁似乎让我想说更多，呵呵

　　比如说。我喜欢看美女写美文。

　　看到黑丫的所谓大陆三毛的形象，又看到她写的什么和她的情人在一起，把她抱起来，简直是快要呕出来，，抱得动么，呵呵（拜托她不要来天涯闲逛，否则她会掐S我的，嘿嘿）

　　我知道我说这话肯定要挺着挨砸，实在是美女虽多，才女确少，换过来大概也成立。

　　这话是不该当众说的，如果不想挨骂，呵呵，我想表达的，只是一种我此时此刻的真实的想法。

　　我爱美女，嘻嘻，爱就是爱吗。

　　人都喜欢快乐。

　　快感，就是快乐的感觉。

　　美感，就是美丽产生的快感。

　　美好的东西产生美感。

　　美女是美好的东西。

　　所以，美女让我快乐。

　　几个服务生以不明显的游移在我身边晃。

　　我想，他们可能在想：这伙计在这儿霸占我们最大的桌子胡呲也就罢了，还要我们搭上免费的热水和成本过五毛钱的杯子。煞是可恶，，嘻嘻。

　　本想深情款款再阐述美文令我心折，美女写出的美文更是如何地让我心折，其实就是吗。美貌的女子的心路历程是有一定不同的，其着眼点与表达的方式也是绝不尽相同的。也许也是由于对自己未知的这个内心世界充满好奇吧，嘻嘻，当我临街遇到另我心折的美人，多希望她是什么王安忆黄爱东西陈染程黧眉什么的。

　　什么逻辑？！我呸！
