这世界你最坚强我最善良

　　嗨你，我爱的人。

　　赶在七夕后一天写这个文因为，

　　昨天我实在是幸福得没有时间。

　　接到你手中的玫瑰的时候，我在飞。

　　突然这意象象梦想成真里面的油彩，嗨你

　　充盈的幸福抹得我哪都是，哪又都不是。

　　模模糊糊我好象站在天堂与人间寂寥的山口，因等待而顾盼生辉。我好象看到你从这边走来，带着你的坚强，嗨你，又好象看到你从那边向我微笑着走来，带着你的爱情，笑容灿若春花。

　　每一年每一天我们都在不断地说，哦我爱你

　　甚至我们有着类似聊天室中那种时尚的暗号来说我爱你

　　哪怕是在汹涌的人潮中滚滚的风尘里，我们相视一笑，就早已心领神会，谢谢你。

　　我也曾逼问你你爱我什么，常常是在光艳照人的时候才这样诱惑地说。哪个女人不喜欢自己漂亮哪个女人不喜欢男人夸自己美？我也是的，所以我选择最自信的时候问你说，亲爱的，你爱我什么？

　　你眯了很长时间的眼睛陷入了很长时间笑容暧昧的思考，而我在焦灼地等你说：哦你真美，或者与你思考的时间成正比的轰轰烈烈的赞美，可是，你后来只是简单地说：我爱你，因为你善良。

　　我当时很失望你知道么，这么普通的评价，这么普通的特点，哦那这世界上值得爱的人真是太多了……

　　昨天我望着那束玫瑰，我突然明白了你的心。再美的花，也有凋谢的时候，就象青春。你因爱我的善良，因此将做准备照料我的一生。

　　刚见戴放说：谁有这样的自信，说：你是我要的。我爱你。我要跟你一辈子。谁会这样置自己于死地。

　　尽管你只爱我的善良，我也不知爱你什么，嘻嘻，可是我说我爱你。

　　于是我很想置自己于死地，忧伤地对你说：你是我要的。我爱你。我要跟你一辈子。

　　抛开繁华，和你在一起，然后在你的怀抱里闭紧眼睛被你摇着然后哼哼唧唧逼着你唱：这世界我最坚强你最善良。
