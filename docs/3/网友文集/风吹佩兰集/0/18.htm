爱情解构——给最亲爱的佩兰阿姨

作者：素素
发表日期：2000-01-11 00:31:00

　　暗lian佩兰其实是很个人的事，不值得在此宣扬。

　　——但是我不得不含羞带臊地说，兰，人家好喜欢你（脸很红，一如霜打的茄子）。

　　 "她在寂寞的暗夜在风中寂寥地轻行"

　　"在一个夜，月黄昏的夜色，"

　　——原谅我，兰，对于你的想象，只能基于夜晚。为了驱散你的倩影，我已经枯竭到只能把你想象成一只扑火的大粉蛾子。

　　难道你还能无动于衷？

　　——我生活在焦灼的等待之中，时光一点一滴的流逝都有可能使我的心破碎。所以我开始作哀怨状地提示你，这不是威胁，但也差不太多。

　　所有颜色因为那一笔轻轻的一颤而有了生命，神采都茵茵熠熠着活泼而流动起来，不是流动，是飘浮！

　　——飘浮，这个词多么适合你，一种轻盈飘渺、蜻蜓点水的感觉，许久以来，我是多么期待你眼光第二次爱抚地略过我的躯体啊。

　　神啊，请多给我一些时间。

　　柔弱无骨好象淡淡一触就会跌落，，却从远方又回旋了回来……月黄昏……香呐！

　　——想念你，在每个想念你的夜晚，我只能独自入眠，为了能够使自己认为你在身边，我的房间里一直充斥着你那个牌子的香水味道，只要一闭眼……

　　对不起，我马上回来，键盘脏了。

　　并且号召更多的有志之士来暗lian佩兰，并且为推广一个完美无缺纯洁无暇光芒万丈名垂青史的的佩兰做出自己心甘情愿和力所能及的贡献……

　　——我的爱，我不能够得到你，其他人也不能。你是如此冷漠如此清渺，以至于每一次你都能置身事外，受尽爱情煎熬的我哟，一定要让这种情绪扩散到每一处崇拜多情男子的地方，让每一位还未陷入爱河的人儿了解到我所受尽的苦楚。

　　兰，知道我的苦心吗？但愿你知道。

　　而佩兰又是如此的完美，甚至为躲在深山里呼喊一下她的名字也感觉到是对她的亵渎……主啊……饶恕我吧……她又那么完美的几乎只能暗lian……阿门……南无阿弥佗佛……菩萨保佑……

　　——喔，兰，我词穷到只能让佛祖来证明我的痴情，虽然拜欢喜佛的时候好象不能说南无阿弥佗佛的，可是我不管，你哪里知道一个在爱里面的人是会如此的语无伦次呀。

　　喔，兰，我想无需累诉，你亦知道我的。我们的爱，从这一贴开始，我们的爱，从这一天开始，我们的爱，从我殚精竭力费尽心机为你造势的那一刻开始。

　　你答应过我，当天涯民众全部都因为我的描述而爱上你的那天，就是你能够开始爱我的那天，我知道，它即将来临。
