鬼鬼和兰兰的早餐

爱情与关于爱情的自然法则

写在前面

　　话说西班牙最后一头怨牛-牛鬼鬼小朋友以一篇《风吹佩兰，美名美奂》，俘获天涯小草风吹佩兰草心，并将其草纳入房中，从此过上了贫下中农疯牛派传人牛鬼鬼与婉约派鼻祖佩兰兰的幸福生活……

 

　　鬼鬼一骨碌爬起来，用爪子钳着兰兰的耳朵："起床啦！"

　　兰兰不动。

　　鬼鬼洗澡回来，伸着湿漉漉的头，用额头顶兰兰："懒虫！起床啦！"

　　兰兰惺忪着眼，咂吧咂吧嘴儿，伸出两个指头："再……十分钟……"

　　过了十五分钟，鬼鬼用粗糙的舌头舔开兰兰的眼睛，把套在毛茸茸的爪子上的超薄拱型雷达表给放在兰兰的眼皮子底下：兰兰瞥了一小眼儿：七点十五分。

　　兰兰只好一骨碌爬起来，寻思，不能说话不算数，信贷吗，言而无信就贷不出来了。

　　鬼鬼去做早饭，兰兰去洗澡。放兰兰喜欢鬼鬼也喜欢的无印良品。

　　 "坐在你的身边是种满足的体验，看你看的画面，过你过的时间。"光良品冠在快乐地哼唧。

　　兰兰裹着浴巾，头发湿漉漉地出来，若有所思着，站在整面的水银镜子前。

　　鬼鬼把早餐摆好在凉黄玉的餐桌上。鬼鬼屈着鼻子，檐外的风铃玎玲做响："兰兰，吃饭啦！"

　　兰兰呆呆地转过头，一脸迷蒙地往餐桌的方向走，

　　喃喃地说：爱情……与……关于爱情的自然法则。

　　鬼鬼烧了银耳莲子粥，烙了菠菜淡油饼，烤的紫菜片，拌了豆腐皮，烘得热忽忽的面包，水果蜂蜜和咖啡。

　　 "吃！"鬼鬼挥舞着筷子。

　　 "什么是爱情和关于爱情的自然法则？"兰兰慢慢搅动银耳和莲子。

　　 "爱情就是水到渠成！爱情就是守株待兔！爱情就是专心致志！就是勇往直前！就是乘风破浪！"鬼鬼夹了块烤紫菜，塞到兰兰的嘴巴里："今天比较听话，起床起的早，奖励一块紫菜！"鬼鬼笑咪咪状。

　　 "呜~呜~"兰兰噎了半日，憋出一句："那爱情的法则是什么？"

　　 "爱情的法则，，，"鬼鬼去夹豆腐皮给兰兰吃，"我们每天可以不吃肉，却不能不吃豆腐，知道吗？这个有营养……"

　　 "呜~呜！"兰兰继续生噎，并瞪眼。

　　 "爱情的法则是，，，就象拉风箱，你见过风箱吗？我在农村耕地的时候见过……"鬼鬼抬起小牛头，眨巴眨巴他长着大外双眼皮的大眼睛深情地望着兰兰。

　　 "呜~呜~，我~见过~"

　　 "爱情的法则就象拉风箱，你拉得慢拉，锅里的火呀，就灭拉，你要是拉的快拉，那火就往风箱里窜，发风箱就烧坏拉……再吃片紫菜！看我烤得多好吃！"鬼鬼又夹了一片紫菜塞到兰兰的嘴巴里。

　　 "呜~呜~"

　　 "所以要不紧不慢地拉，爱情也要不紧不慢地爱。其实，爱情也没什么特别的法则，大自然的法则也就是爱情的法则。。"鬼鬼笑咪咪问："紫菜到底好不好吃呀，这个在韩国特受欢迎啊。"

　　 "呜~呜~"兰兰边吃边想着鬼鬼说的话。

　　 "喜欢吃肉吗？"鬼鬼瞅着拌豆腐皮剩下的调料琢磨着。

　　 "现在不喜欢。"兰兰老老实实答。

　　 "那什么时候喜欢？"鬼鬼笑咪咪问。

　　 "饿的时候喜欢。"兰兰笑咪咪答。

　　 "我想把熟肉冻在冷冻柜里。"鬼鬼认真地思索。

　　 "啊？！"兰兰大吃一惊，寻思：把熟肉冻在冷冻柜里倒是从没听说过……

　　 "等你想吃的时候，我就拿出来，把它切好，拌拌给你吃……"鬼鬼一本正经地说："我一定要把我的兰兰饲养得白白胖胖的，呵呵呵呵。"鬼鬼把蹄子在雪白的餐巾上蹭了蹭，心满意足地笑起来。

　　……兰兰梦才醒似的，抬起头，定定地望着纯洁美丽可爱英俊潇洒憨厚智慧善良温柔的鬼鬼，"嘤"地一声，扑向鬼鬼的怀里。

　　无印良品在唱："春也来了，花也开了，微风都沉醉，虽然我不说话，却也早已万语千言~"

　　兰兰和鬼鬼合唱："分分秒秒显得清澈又珍贵，只有你才能给我这种感觉~~~不管心有多疲倦，梦想还有多远，只要有你在身边我也无所谓~~~"

　　……

爱情与关于爱情的自然法则（续）

　　唱完了歌，鬼鬼去刷碗，兰兰去化妆。鬼鬼面向观众说：爱情需要每天劳动多一些，让自己累，抛弃柴米油盐的琐碎，

　　兰兰化完状，鬼鬼赶着他的牛车送兰兰上班去了。在电梯口，鬼鬼和兰兰拥吻了一下，兰兰甜蜜蜜地走进了无数@_@艳羡的女同事和无数*)_(*忌妒的男同事中，一声拜拜消失在电梯中。——鬼鬼面向观众说：爱情需要每天甜蜜多一些，给自己看，也给别人看。

　　送好兰兰，鬼鬼赶着牛车，来到了八佰伴，给兰兰挑了一支紫色的美宝莲，又挑了一件绣着猫猫的丝制睡衣。——鬼鬼面向观众说：爱情需要每天关怀多一些，让自己感动，更让她感动。

　　出了八佰伴，鬼鬼赶着牛车来到音像书店，买了本《音像世界》，买了所有的无印良品的CD。鬼鬼面向观众说：爱情需要每天学习多一些，培养和她一样的兴趣，缩小心灵之间的差距。

　　出了音像书店，鬼鬼赶着牛车来到车行，把"奔牛50拖拉车"升级成"奔牛摩尔跑车"。鬼鬼面向观众说：爱情需要每天进步多一些，给她新鲜的感觉，给她狂野的滋味。

　　出了车行，鬼鬼赶着牛车来到酒吧，吃两块点心，喝一盏咖啡，听取助手汇报股票的牛势。鬼鬼面向观众说：爱情需要每天赚钱多一些，给自己物质基础，给她上层建筑。

　　出了酒吧，鬼鬼赶着牛车来到网球场，打了两场球。鬼鬼面向观众说：爱情需要每天锻炼多一些，给自己强壮健康，给她幸福保障。

　　出了网球场，鬼鬼赶牛着车来到菜场，买了一堆瓜果蔬菜回家，在厨房忙碌起来。鬼鬼面向观众说：爱情需要每天菜肴多一些，拴住她的胃，她就不会飞。

　　听到门铃，鬼鬼系着围裙跑出来，给刚从男同事的"便车"（？）上下来的兰兰开门。鬼鬼面向观众说：爱情需要每天宽松多一些，无伤大雅吃豆腐，夫妻恩爱才幸福。

　　鬼鬼和兰兰吃过晚饭，一边和兰兰看看电视，打打毛衣，读读报纸，一边慢慢聊。鬼鬼面向观众说：爱情需要每天交流多一些，小两口多拉拉家常，婚外情才不会彷徨。

　　夜深了，鬼鬼和兰兰……鬼鬼面向观众说：还看什么看？都走都走，我要关灯了！

THE END
