　　席间，夜。

　　亮眼睛，江湖过客，卓锐，麦乌乌，小飞，及青岛诸位同仁在座。

　　推杯换盏，谈笑风生，觥俦交错，浓浓的暖意。

　　酒越要越多，过客的脸挂着笑，越来越僵硬的笑，也是依然很甜蜜的笑，桃红样的，蜜如春花。

　　亮眼睛的眼睛，越来越明亮，启明星一样，在幽幽的夜幕中悠扬地闪烁，洞人心弦。

　　卓锐的两只手绞在一处，缩着肩膀，低低地笑。浓浓的剑眉下，黑黑的眼睛，很聚光。

　　玉面书生小飞神色飞扬，把酒言欢，一面喜滋滋地望着身边佳人……

　　麦乌乌一双似喜非喜含情目低垂，两条似蹙非蹙笼烟眉微皱……

　　若有所思。而且一直若有所思……

　　宝宝面若桃花，眼睛鼻子咪到一块，摇摇地举着杯，直冲过客高叫："贩子大哥~ ~ 哦不，翅膀大哥~ ~"

　　说时迟，那是快，麦乌乌听到宝宝乱叫出的"贩子"两字，立刻，怔住了……

　　她迷茫而慌乱地抬起眉梢，神色凄迷。

　　俄而……

　　想到这里是青岛，竟不是在北京……

　　麦乌乌幽幽地轻叹一声，那叹息如一片洁白的天鹅的绒羽，轻到甚至没人可以察觉……

　　就是在这个时候，手机响了，是乌乌的。

　　乌乌慵懒地摸到手机，有些心不在焉……

　　当她看到来电显示时……

　　 "腾"地一下，两朵红云直上面颊！

　　毕竟是亮眼睛最关心好友，诡笑着问："谁的？"

　　麦乌乌一脸娇羞："贩子的……"
