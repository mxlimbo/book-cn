关于信任、写作、情感、网络及其他

　　草庵居士版主您好！

　　借宝地就我的观点，和您商榷。

　　1． 关于信任

　　这个问题我不关心。言即行，行必果，那是道德的陈腐观念。没有人可以约束我们，哪怕用自己的诺言。最重要的是，不可以逃避自己的内心。信任不不信任，被信任和不被信任，都是自己的事。即便明年的这个时候你不兑现，那也很正常。哪怕只有一个解释：老子我不愿意了。我也会为你叫好。而，如果你违背了自己的内心，为了君子一言的虚荣，那才叫可耻。

　　2． 关于写作

　　我写字，不为了名留青史，不为了教导别人，不为了获奖。我相信其他爱文字的，大家都一样。写它，是为了纯粹自己的灵魂，写它，是为了读自己文章的人。把自己对世界的感觉和发现提炼出来。那是属于我对世界理解的一种哲学再让读到的人去感悟，去理解，从而再产生属于他们的，更新，更美，更有力的思想。在现代的社会，价值已经面临着可怕的沦丧，物质主义腐蚀着人性中最珍贵的部分，人的个性在不断地丧失。在这个时代，写作是为了维护人性所做的最后努力。这种努力，使我们保留了这世界人类的最后尊严。

　　我珍惜这种尊严。

　　3．关于另类情感

　　比如说类似恨的东西。这是一种畸形情感。人有了这种东西，心灵也会变得畸形。没有什么比这更痛苦，就是发现自己心灵上的阴影。这种自己给自己带来的永远无法晾晒干净的东西，是对自己最大的嘲弄。报复也一样。深不齿。

　　4．关于草庵居士所说的"把天涯兼并"我觉得这种想法，不是一个成熟的男人该想的。第一、 就象世界必然要有多种国家存在，美国可以选择当世界警察，却不能使地球仪上都画个大大的USA。可口可乐可以占领市场的大部分分额，可是，它依然不能阻止我喜欢喝妙士。所以，我觉得，如果"天涯"只是个其他所有网站的代名词，我觉得您这话说得很幼稚。我只是从客观上说话，因为我欣赏客观成熟和理性。第二、 如果"天涯"只是代表天涯。那么请问：这种咬牙切齿从何而来？是曾被羞辱的仇恨？是艳羡和妒忌？我不曾在天涯拿过一个子儿的MONEY，我只能说，从这个角度想，您这是属于我在第三点上面简单阐述的另类情感。

　　5、关于网络就象我们概念中未来的地球村。平等、民主、博爱是我们对人类未来的感觉。关于网络，我认为提出未来的网络村，才是理性。我不喜欢某人一统天下的英雄主义，也不欣赏不经过思想的语无伦次。我希望我们共同拥有最璀璨的星空，最新鲜的空气，最民主的声音。

有奖征文――版主就职演说

　　各国总统履新，均有就职演说，余虽百姓，但就职版主，仍应仿效之，以应世界民主之潮流，并利各位网友之监督。

　　很荣幸能成为《文学鉴赏》的新版主，这是我从没有想到的事情。我出生在天津，生活在美国，自叹中文基础薄弱。难敌故乡之文学高手，随心有惶恐，但，终有兴趣接下此任。

　　余在美国生活多年，从事金融行业。偶有文字小作，仅为票友之性质。工作每日繁忙，但仍心系故土。很高兴保定热线的朋友能告诉并推荐给我这个天津的网站。能让我飘泊的心有一个寄托。

　　然，既做版主，就应有所行动，故，余思前想后，总希望能为天津的网站有所贡献，为每日拼命工作但仍发扬中国传统，热爱文学之网友能尽一己之力：

　　1． 自今日至2001年6月30日为止，在此其间在《文学鉴赏》发表的文章均有资格 参加评选。并将以各位网友之投票为准。

　　2． 获奖之前三名，余个人出姿三万美元，做为奖励。用于得奖之网友访问美国之费用，访问期为30天。旅程：美国全境。余本人将全程陪同，以示敬意，并保证造访美国总统及相关之政府和学术机构。

　　3． 获奖之前4-10名之网友，每人可获赠本人私人捐献之最新型笔记本电脑一部，数字像机一部。彩色喷墨打印机一台。

　　4． 在本栏发表之票选优秀文章，余私人出资结集在美国或中国出版发行。所获之利由各文章之作者集体决定支配。

　　5． 每年出版一部网友名录，费用由余私人承担。凡在本版发表文章之网友均可免费获赠，

　　6． 本人将尽心尽力，服务各位网友。余之观点：文不在多，精则灵。无论何种观点，何种文章，余皆欢迎。散文，小说，杂文，政论，感想各色文章，只要有情有色，但不色情，就全部接纳。

　　7． 本版主亦将尽力每月为各位网友推荐网友之好文章。并望各位网友大力支持。

　　8． 广开言论，自由发表意见。言无不尽，不尽不欢，并望各位多多指教。余将利用余之工作之便利，为本网站寻找投资，各位网友亦必要尽力支持。多多光临。

　　有人问余之真实姓名。余三十多年，行不更名，做不改姓。FIGI就是本人之真实姓名。有疑惑者，请致电美国MDV查询。

　　此文发之唐突，未及与另一版主商量。如各位有任何意见，请多回帖子，嘻笑怒骂，任君挑选。余皆受之。
