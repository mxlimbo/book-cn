TRY

　　真没办法，独自的时候，不讲点故事，还真舍不得。

　　年轻的时候，阿醒遇到过一个人，遇到就嫁了。

　　嫁了还问人家：爱情是什么？我没恋爱过。

　　人家说：你当初干吗来着？

　　阿醒说：没干吗啊。

　　人家说：你和你老公不好吗？

　　阿醒说：挺好啊。

　　有一年春天，阿醒在网络上遇着个人。

　　他说他是男的。那他就是男的罢。

　　他们俩相处得挺好，虽然有时候唧唧歪歪的。

　　后来那男的说咱俩见一面吧，我从国外回来，时间不多，你来我的城市吧。

　　见一面就见一面。正好她出差，就去了。

　　他敲门，她去开门，还真是个男的。

　　他望着她笑。

　　她也望着他笑。

　　傻呵呵地笑了很久。

　　他们微笑着拥抱，心事重重地接吻，好脾气地做爱。

　　他轻声说：我得回去了。

　　她说再见。

　　说得太快的"再见"，男的怔了一下。

　　远远地见她在昏黄的床灯底下笑魇如花，塑料花。

　　男人扭头说OK，却又没立刻走，面着黑觑觑的壁站定。在黑暗里影子很深很长。然后他走了。

　　再上网见面的时候，又已经分别在大洋两岸。

　　男人说：你好。

　　女人说：乱搞过了，我们的故事OVER了。

　　男人说：你丫说什么。

　　女人说：心灵的纠缠都是通过FUCK来FUCK去来得到彻底解脱。

　　男人说：你爱我吗？

　　女人说：我们结束了。

　　男人说：说你爱我。

　　女人说：你爱我吗？

　　男人说：我先问你的。

　　女人说：说你爱我。

　　男人说：我爱你，你爱我吗？

　　女人说：哈，你上当了。后悔吗？

　　男人说：我从不后悔。

　　男人去睡了，一睡就是好几天。

　　阿醒每天上线N遍。

　　N天后，男人出现了。

　　他说：我在美国的INTERNET上杀出一条血路，为了找你。

　　她说：啊你。

　　他说：出了点车祸，可是知道最接近死亡是什么样子了，算了都不说了，没别的就是挺想见你。

　　屏幕这头她的眼泪就象断了线的珠子。

　　她泣不成声，敲出三个字：我爱你。

　　他在那边似乎是笑了，或许是别的什么，后来他说这三个字象锤子一样，砸在他的心头。

　　她也薄薄地笑了，薄得另人心碎。

　　突然她说：哦，我说爱了，你干吗呢，在如释重负呢吧。

　　他说：阿醒，其实我在流泪啊。

　　爱，多好啊。世界上的每样东西都因此变得非常地ATTRACTIVE，虽然不见他。

　　深秋来的时候，有天晚上流连在花圃里，大丽花开得层层叠叠的，堆积如山的花瓣奢华得象梦的宫殿。一只绿豆蝇怕冷，颇为享受地把头扎到花蕊里，夜光下露出绿荧荧，圆溜溜的尾巴，笨拙得要命，可爱得要人羡慕。

　　呀，她快乐地想，多好呀。人没有恋爱可该怎么活呀。

　　她怀着巨大的幸福小心翼翼地守口如瓶，仿佛这幸福振翅就会飞。

　　后来的日子就在网络上一成不变地过下去了。

　　男人说：不敢写那些题材的东西，怕写不好。

　　阿醒说：呀，还有什么值得怕的东西吗？这个世界上，我只怕两样……

　　男人说：饿？渴？

　　阿醒说：不，第一，怕先生死在我的前面；第二，怕你死在我的前面。没了。

　　良久，男人说：真自私啊你，

　　阿醒在这边亮晶晶地望着屏幕，喃喃地对自己说：这回我算知道荷西死，三毛为什么不要活了。

　　男人在那边发狠地敲了一行字：我明天就出车祸！

　　阿醒慌得乱了手脚，说：不不，别说这样的话，就算救救我，求你了。

　　男人在给阿醒的信里说：有人问我，是不是在和你网恋，被我很认真很严肃地回绝了。阿醒，你知道吗，我很严肃地告诉你：我们真的不是在网恋。

　　我们不是在网恋？我们没在网恋那是在做什么？她想，有点糊涂。

　　我们是在做什么？是什么让我对你如此牵肠挂肚，胜过亲人？！

　　她回信就写了一句：那我们是什么？

　　阿醒和阿小说：我不想再网恋了，烦了。

　　阿小说：NNND，你进去了。

　　阿醒说：你说对了。我进去了。我输了。输了，我就该先出局了。

　　收到了他的回信，亦很简短，他说：阿醒，我们没有在网恋，我们是在恋爱。

　　她望着这样一行字，再度湿润。

　　他说你什么时候来美国呢？

　　她说明年年底再说罢。

　　他说：呵，

　　然后轻轻敲了一行字：那时候你也许见到李卫东偕夫人了。

　　不是说，伤痛的结尾可以成就完美的故事吗？

　　阿醒说：我必须离开了。

　　阿小说：那就做得彻底，别留一点痕迹。

　　阿醒说：再不见可以，我舍不得删那些信。

　　阿小说：MMMMD。

　　阿醒说：我删我删。

　　阿小说：你做不到，我赌一块钱。

　　阿醒说：你赢了。我是做不到。听说婚姻是爱情的坟墓是吗？

　　阿小说：是。

　　阿醒说：为什么我们的爱情却总是将要死无葬身之地呢。

　　后来这些日子就过去了。她就真的在网络上消失了。

　　时间总会帮你将一切遗忘。

　　就比如这个时候，她带着女儿坐在避风塘精致的桌椅前，一面低低头，用着生滚烧鸭粥，一面用筷子拨开荷叶，珍珠鸡挟着荷香的味道就精致地扑来，音乐梦呓一样地响：

　　 TRY……

　　 TRY TO LEAVE

　　 TRY TO FORGET

　　 BUT I REALLY DON'T KNOW WHY……

　　一刹那――

　　记忆汹涌，时空倒转――

　　女儿美美地吃着，她低低地暗惊，想：

　　呀！年轻的时候，我还曾经爱过一个人呢
