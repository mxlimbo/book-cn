关于假新闻

　　俺从小就不爱看新闻，爱看动画片。

　　三岁看老，落下这病根，现在我也不爱看新闻，爱看MTV。

　　所以按理说假新闻和俺是不大沾边的。

　　何况俺主意很正，对各类宣传百毒不侵，只占便宜不吃亏，任而天花乱坠。

　　可是有一天我客户跟着假新闻蹑手蹑脚地来找我来了。

　　我是做航空的。

　　客户哼哼说：我昨天买了张机票。

　　我说：对，你昨天买了张机票。

　　客户说：你凭凭凭什么要我五百七？

　　我说：青岛到北京标准票价就五百七。

　　客户大手潇洒地一挥，变出一张报纸：

　　"你丫想蒙我，报纸上说了！为了群众利益，各大航空公司联营，现在机票打折了。"

　　我一看，报纸上赫然印着：航空公司为了广大群众利益，实行联营，票价由航空公司自主控制，降低销售云云。

　　NND，我立马致电各航空公司，质问机票打折为啥不通知俺。各航空云里雾里状，后来说了：啊是，联营就是几个大航空联合起来，统一口径，坚决不打折，团队也最低九折。

　　还好，当我把这事告诉客户的时候，他暴躁的声音只给房盖儿掀开一角，周围欣喜若狂地围了好事者一堆，这半天的生意是甭做了。

　　还有一回，报纸上登出了小飞机可以由航空公司自行制定票价。嘿，又是查无此事。

　　第三回是，新闻说七月一日，废除手开机票，执行打印机票。每个接到手开机票的客户都胆战心惊，死求着我们务必给他们换成打印机票，而实际情况，现在手开机票仍然执行，估计能到年底？得，我也不说了，说出来怕又是一条假新闻。

　　若干经历，现在俺也好象人民日报本报评论员一样NB，痛心疾首地对客户说：这这这都什么世道啊，现在新闻机构灰常灰常不负责任，得着点风，就捉影，给他点阳光，他就灿烂。

　　客户见我深受其害，且大义凛然不象是装出来的，皆心悦诚服，我自己寻思着报纸的假新闻给咱衬托得这么有权威，干脆俺也办报纸算了，嘿嘿。

　　其实痛恨假新闻。

　　从听众或观众的角度来说，人类本质中最宝贵的信任之堤，渐渐倾陷。

　　对新闻机构的真诚已将被人沦入耻笑。每天必买的三毛钱报纸的唯一价值，除了如何生活小窍门就是广告，哦，广告也是吹得多，实的少，还好，每个人都知道广告是个卖笑的婊子，而总把那些板着面孔硬撑的新闻，当做名门淑女。该说大部分的名门淑女该是端庄有致，仪态万方，高贵可人的，而假新闻就象是让这些淑女混迹于婊子中春光乍泄了那么几把，那么婊子和淑女，五十步和一百步，还有什么太大的区别？据说人总是在淑女面前出卖自己的尊严和情感，在婊子面前释放自己的欲望和金钱。假新闻，也必玩火者自焚。当真实，都变成了所谓的，当新闻已然没有了尊严，当媒体已失去了大众对他的信心和情感，它的未来，只想在那些减肥秘籍，和秋天怎么酽萝卜，和张爱玲胡兰成之类的前朝旧事、鸡毛蒜皮里徘徊吗。

　　而做为媒体，我了解市场经济，你需要找到一些卖点。真正的崛起，是由那些智慧、合作、勤力。你不勤奋，没远见，不敏锐，也不配做新闻。你想为了市场，为了每个月保住那干巴巴的几钱银子，你想被领导赏识，制造噱头，无中生有，断章取义，用欺骗来追求发行量，

　　那有意义么，要么，就是你没有职业水准，要么你就是没有职业道德，普天下认真做文化的人有的是，你怎么就那么贱！

　　而这种信任危机，是真正最可怕的。

　　如果你有一次欺骗我，我就可以再不信任你所说的话。我不会配合你，也不会帮助你，因你凌辱我对你曾有的真诚。面对这些，假新闻，你还有什么话可说呢。
