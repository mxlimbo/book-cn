　　秋雨的右手的小手指上有一枚银色的指环，在昏黑的夜中，暗暗发光。

　　言归正传。

　　说心里话，如果能被人在天涯中写出小记的人，都是可圈可点的人。

　　（寒君、野火、我认为的纳兰、翅膀和萧秋雨。）

　　秋雨正是这样的一种人。

　　抛开开玩笑不说。

　　我眼中的秋雨，是位身材高大魁梧，举止儒雅斯文的大男孩。我叹佩秋雨的丰富敏感与渊博，更叹服于来自他身上的一种亲和力。无论长幼高低，秋雨总是很耐心地帮助别人走向大家的圈子。就象我初到天涯的时候，举目无亲（因为谁也不认识，介绍我来的人也早不来了），四处乱撞，在聊天室里，总是秋雨温和耐心地陪我慢慢聊。到后来，我才发现，秋雨不是因为某个人才这样做，而完全是一种习惯，对每个初来乍到的人，他都是细心地打招呼，细心地找到一些有趣的话题，引起大家说话的兴趣，非常照顾别人的想法。我想，这大概也是秋雨能把舞文弄墨这个版做得很好的原因吧，至少，我很喜欢，甚至纳兰也说，打开电脑，先来舞文弄墨。

　　秋雨的朋友随处都是（大家也是有目共睹），我想，正是他的这种亲和力感染着每个"沦落"天涯的人。

　　对了，秋雨还问呢 ：我家的那块园子怎么样啦？？我告诉他：茂盛着哪，就等你回来给大家红脸哪！

　　百忙之中不忘天涯舞文弄墨，谢谢萧秋雨。
