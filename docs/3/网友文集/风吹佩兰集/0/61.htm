感谢网络

　　这两天周末。天隐隐似有小雪。在温暖的家里，一面听从天籁村下载下来的蔡依林孙燕姿的最新音乐边做打扫，干干净净之后上网看东西、写东西，OICQ开着，有一句没一句地和在线的朋友聊天。如果说上网大概是一年前罢。偶尔来到BBS，且一人不识。小心翼翼地看帖子，跟帖子，小心翼翼地看人家聊天，再自己换个若干名字性别去胡扯。

　　那天和网易的铁嘴阿良在OICQ里谈起来，商量给我在网易做文集的事。我问咆哮怎么认得我的呢，阿良说，佩兰你是个名人啊。我倒，我怎么就是个名人呢，我一般只在天涯社区玩啊。阿良说：谁都知道天涯是个水平很高的社区，风吹佩兰是天涯社区的名人啊。呵呵，风吹佩兰就在上网的一年之间成了名人了，上网之前，佩兰是什么样子的呢？

　　上大学的时候，忙于寻找一些生活中所难了解的奇迹，反而对文学音乐美术比较疏落，除了学校组织的一些这类的艺术比赛。因为那些东西都是很有基础了，平时练的倒是很少。

　　那时候记得的只是些高中时代的小虎队伊能静之类的歌，晚会表演节目也是临时抱佛脚去学些新歌，无从谈起爱好。那时候，写文章也从不去投稿，除了一些命题作文，应老师约，发在学校的校报上而已。平时就呼呼去了。毕业后更不用说，闲来逛逛书店，买买期刊。打打球。和一般人没什么两样。

　　可是总会有一种女人，她们总是要认为自己是不同的和优秀的。如果不是这样，她们就会象玫瑰一样枯萎。比如说我 ：）。

　　然后我就上网突然发现BBS这样一个好玩的地方，随便讲话就可以扔上去，拿不到稿费，亦不必负什么责任。很多繁琐的心情一旦变为文字，就如丝路花雨，滴滴晶莹。

　　都是喜欢文字的人啊。在网络的各个版块中居然发现那么多的同好。能诗的，善文的，爱乐的。每写一篇文字，都会得到这些来自朋友们的关系和建议，真是开心。而在现实生活中，你能在周围找到几个懂填词做赋，能诗善文的呢。甚至这些网人在现实的生活里，绝大多数都是理科出身，所叙述的也大都是理科生的文科爱情，哈哈，这就是网络，网络总会让你找到知音。

　　大约是网络作文的这种心态罢。无欲无求，不为名利的。写出的东西总是清新，虽然青涩。写了一些东西之后，就遇到了些报社来约稿，或者是约着出书，虽然还得故做谦虚地承认这是赶上了网络泡沫经济的大好时机，但写文章的女孩子谁不想轻轻松松出一本书，做为岁月和青春的印记呢。

　　其实来自网络，最让我感激的莫过于认识一群对酸的狐朋狗友。假期来临的时候，总要各个城市窜窜，见那些素未谋面的网友，而且我得说，见过的网友从来没有让我失望过。有做计算机的，也有专业的作家，或记者，编辑，公务员，学者等等，有趣的是生活中的问题总是很容易找到这些朋友出谋划策，问题就非常轻松地迎刃而解。小到电脑的问题，网友暖暖在QQ里指点就OK了，有时候公司的管理问题，学MBA的哥们瞎子就帮着总结了，平时休闲娱乐，有个网友叫BATZ的和我达成共识，每天都给我提供一首非常好听的歌的下载地址和歌词；还有就是打架都有很好的帮手，记得从前有一次在中文在线，遇到个南河先生，瞅他指手画脚的不甚顺眼，便挖苦了他一通，这伙计开始和我掉书袋，出了一堆题目让我解释，都是些魏晋南北朝时的名词。恰好一个网友就是北大就是魏晋南北朝的博士，哈哈，这位网友一面在电话里给我讲，我一面得意非凡地往电脑上敲，最后南河先生拱手认输，我笑得不行，却也在这之中，学到了不少东西。

　　抬头望望窗外的天空，阳光下，天空湛蓝得厉害。每一天都很可爱。写着美丽的文字，哼着快乐的歌，谈着若有若无的恋爱，做着轻松的有未来的工作，过着精致的生活，感谢网络。
