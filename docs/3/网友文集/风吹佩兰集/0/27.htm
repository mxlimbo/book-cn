狠心的张建把佩兰给丢了

　　张建是这样认识佩兰同学的。

　　张建同学在网络上写帖子，佩兰同学就忙不乐得P颠P颠地跟帖子，那个叫崇拜呀。

　　张建同学是个在纽约的文学青年，还出书。文字那个叫细腻，秀美，还湿润润的。总之FANS无数。

　　其实张建同学是不会理睬佩兰同学的。

　　不是一个档次的，他心想。不是对手的人怎么会做朋友怎么会做知己或者红颜知己？长得美也不成，何况佩兰同学的照片看上去是个小眼睛，最要命的还是飞机场，瘦弱不堪。

　　可是她心无旁骛忠心耿耿地看他所有的帖子，并认真的分析，探索，并独自回味，回的帖子也象自说自话，傻了吧唧的，不过张建同学偷偷看着很受用。

　　有一天，瞎子同学和天骄同学来了天涯社区。

　　啊！那是怎样大大大大的两展大红旗呀！比起来，张建同学象根野地里的小葱。张建一百零八个红颜知己都离开他的时候，佩兰还在张建的帖子后面努力耕耘。

　　张建：佩兰弟弟，你好

　　佩兰：小建姐姐，你好

　　张建：辛苦辛苦 （怜惜状）

　　佩兰：心甘情愿 （痴迷状）

　　寂寞的张建同学，就和傻了吧唧的佩兰同学开始的心电交流。

　　不过佩兰同学很来电，张建同学很快觉得这个傻瓜很没劲。

　　张建：天涯没意思。还不如网易。

　　佩兰：有什么有意思的吗？

　　张建：恩，网易可以养鸡。挺逗的。可惜没空，我从没养活过。

　　佩兰：我帮你养。

　　佩兰同学跟着张建同学P颠P颠地去网易养鸡。

　　张建的鸡。取名叫佩兰。一岁。

　　张建利索地把张建的密码给了佩兰，为方便她养鸡，然后扬长而去，寻找新的传奇去了。

　　佩兰在绝大数的时间里，用张建的密码登陆网易去养那只叫佩兰的鸡。带佩兰鸡去散步，去逛街，吃虫子，宴会，还带佩兰鸡去开会。

　　佩兰：咱鸡两岁了

　　张建：哦，我从没养到过两岁。

　　佩兰：我带它做很多事情，增加经验值，它终于长了一岁了。

　　张建：MD养个鸡还这么不容易，什么时候才能把它养大啊。

　　佩兰：一般鸡的寿命都设定到二十岁。

　　张建：要它死可是个漫长的过程。

　　佩兰：哦，可不是。

　　张建同学也会给佩兰同学打电话。

　　黑夜那头的他也不说什么。白日这头的佩兰同学轻声细气地随便说点关于那只叫佩兰的小鸡，它胖了，今天带它去晒太阳了，它今天吃了一个橘子，三条毛毛虫，一杯咖啡。

　　张建在她轻声细气的诉说里，在电话的那一头点燃一支烟，他其实听也没听，就是听到她细细的声音，觉得塌实而放松。当郁闷都没了，他说，哦，我挂了。就挂了。

　　快乐的时候，他想不到她。

　　郁闷的时候，他才想听听那只名字叫佩兰的小鸡的故事。

　　他们也写信。

　　她把那只小鸡的生活写了好几页，写得缠绵细致。最后对他说，好好的，我一直在你身边。

　　他有时候回，有时候不回。回的时候她快乐，不回的时候她盼得悲伤。

　　佩兰同学温柔地和暖暖同学谈到她在养一只小鸡，和张建同学一起养的。

　　暖暖：你在泡他吧。

　　佩兰：我没有我没有 （很认真）

　　暖暖：他在泡你吧。

　　佩兰：不会吧。1？你这么一说……（仔细思考状）

　　暖暖：哈哈

　　佩兰：不会吧。他说我不值得骗，一骗我就信。

　　风风雨雨。

　　那只叫佩兰的小鸡长到十六岁了。

　　佩兰：今天用你的名字登陆，又有个女孩子向我狂打招呼。我告诉她我不是你。叫檬檬

　　张建：哈哈，崇拜我的女孩子很多的。

　　佩兰：哦

　　佩兰：咱们的鸡，十六岁了，快到二十岁了

　　张建：哦，

　　佩兰：咱们把它放了吧，省得养到二十岁，就死了。

　　张建同学没回答，突然断线了。

　　春去秋来。一转眼秋天就来了。

　　青岛刮了几场风，下了几场雨，满街的落叶萧瑟地舞。

　　小鸡二十岁了。

　　佩兰想：它真的应该死了。再养好比我还大了。

　　可是小鸡没死。原来当初领养的时候，忘了设定死亡期限了。

　　不知为什么，佩兰松了一口气。张建也不知在忙什么。

　　在QQ上见了，佩兰幽幽地对张建说：我们忘了设定小鸡的死亡期限了。

　　张建说：哦。

　　佩兰说：那，小建，我们，还是，把，它，放 了 吧。

　　张建说：放了就放了罢。

　　佩兰用张建的密码登陆网易，去最后看那只叫佩兰的小鸡。

　　那小鸡精神矍铄，神采熠熠。

　　佩兰突然有些万念俱灰，她恹恹地，手，还有点抖。

　　她点那个放生。

　　她曾设想过千百个结局。可是都没准。

　　她想着这只二十岁的小鸡，叫佩兰的小鸡。

　　这二十年的日子。让她柔肠寸断。

　　系统在她点了放生之后，"叮"地响了一声，出现了一句话：

　　 "狠心的张建把佩兰给丢掉了……"

　　她大吃一惊。望着这样一行字，，她泪如雨下。地球这面的白天就象下大雨。

　　——似乎这里就该是结尾了。

　　但其实是这样的：她含笑着在QQ上对张建说，你看这系统，它该改改，放生是件好事情啊，它该出现"佩兰终于得到自由啦"什么的，可是它却提示"狠心的张建把佩兰给丢掉了。"

　　张建没言语，良久，他轻轻地对佩兰说："我们，能，把它，再，拾回来吗，佩兰？ "

THE END
