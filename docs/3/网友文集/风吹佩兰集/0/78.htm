网上网下

　　昨晚上从九点开始睡，睡到早上七点半，足有十个半小时，真舒服。

　　对网络有点厌倦了，毫无意义的堆砌和个人生活的大量BE OCCUPIED。

　　甚至成就了一种网络综合症，逢人必嘻嘻嘻，嘿嘿嘿，呵呵呵。

　　上了网的时候，本来就什么事都没有，还把在线名单翻了一遍又一遍，一遍又一遍。哪怕，并不会和这些期待的名字说上一句话，或发个 MESSAGE，却依然盼望奇迹在下一秒出现。

　　也把社区聊天室翻了一遍又一遍，看看有没有人在，和都有哪些人在。

　　天涯聊天室也是自由地频繁出入，仔细查看每个人的IP，看看有没有曾相识的和似曾相识的。

　　每一秒，每一分，都在期待着下一秒，和下一分。

　　说些不咸不淡的话，做些不三不四的事儿。

　　我真的有些厌倦了

　　上网的日子曾经是如此激情与美好

　　而不上网的日子，细细数过，更别是一番宁静与曼妙。

　　便想起了那几日电脑坏掉的日子

　　当清晨的第一缕阳光洒进我的房间，揉揉眼睛，便从睡梦中醒来，随手旋开收音机，早间新闻伴着清新的音乐流出来，流出来，悦耳的声响，流淌着整个公寓。

　　便叠被子到整整齐齐，便打扫房间到一尘不染，同时烧着我的红枣莲子百合粥。洗漱后，到楼下的糕饼店去买清晨第一炉，刚烤出的面包，鲜香松软，或者煎个蛋，或者做一杯醇厚的咖啡，或者一只新鲜的水果。在如此新鲜明媚的早晨，享受着没有网络而早起的生活。

　　晨雾似乎还未褪去，路边的植物似乎被笼上一层蒙蒙的轻纱，深粉色的茉莉花和淡粉色的牵牛花，那样星星点点的，那样柔弱的，那样含笑着的，那样不事声张的，从藩篱后探出头来，象在问候着我这位久未谋面的早起的人。

　　脚步变得从容，甚至下班时分。

　　不再急急地往家赶，因回家也上不了网。

　　穿过黄昏的露天集市，绿的油菜，白的萝卜，红的西红柿，毛茸茸的桃子，黄澄澄的梨。一溜弯是刚出海的鱿鱼，螃蟹，蛤蜊，小黄鱼。。面包房甜香甜香的味道，烤鸡店油香油香的味道，初秋熙来攘往的人群中，女孩子脂粉味在绿色的空气里暗香浮动。。买主们心不在焉的问价声，小贩们半真半假的换价声，踟躇着的TAXI寻找客人时犹疑地转着方向的声音，一切显得如此的市井与，真实。

　　真实得，我都惶惑起来。

　　贪婪地查视着这久别未享的人间烟火……

　　似乎自己，已远离尘世，隐居到网络这座仙山已许久，偶尔回到这里，连海鲜的淡淡的咸腥味也令我如此激动。

　　买上个烧饼，提一条冬瓜，捎两根香菜，拎上二块钱的蛤蜊。想起家中还有久已未动的干白干红，遂呼尔将出赏美酒，呼是传呼的呼，：-）

　　便有三两好友来与我同销万古愁。 ：-）

　　酒到微醺，风生谈笑，摆上一盘棋，两人对弈，余者观看，并深以 "两人玩棋，多嘴是驴"为不然。

　　一面是胡搅蛮缠，一面是自作聪明，一面是穷凶极恶，输者一掀棋盘，遂满堂大笑。

　　夜深，送好友一一至车站，沿途绝不放过不打烊的书店，一头钻进去，直到店主乒乒乓乓的拉窗帘，哗哗啦啦地扫地板，才又袖了手，一脸赖皮相地笑吟吟离开。

　　独自归来，夜风悄悄拂面，似乎清醒了许多。月明星稀，街上的行人渐散，马路上凌乱着这日的塑料袋，在等待明日凌晨环卫工人的扫把。一切的一切，显得如此的不以为然。

　　多了我一个就多了，少了我一个也没什么，平凡的世界依然不动声色地如此平凡。

　　也就象我的网络仙山，多了我一个就多了，少了我一个也没什么，地球依然在转，网络依然在转。

　　这么说起来，好象我也在发个告别的帖子似的。我倒是真的不想这么走，要真的想走了，发个帖子，结果肯定又换个名字留下来，不然也会挤在天涯聊天室里去寻觅着一些故人。其实那样做更多的可能是喜欢看到一些狐朋狗友们悲悲嘁嘁的回贴而产生的一种心理满足。

　　其实，我也喜欢这种满足，每个人在心里对自己都是没把握的，每个人心里都会有一处最柔软的地方。我也会很好奇和很在乎，还有谁在对我关心和注意。。

　　如果谈到走，心里最不舍的是谁呢？

　　偶尔出了公司去办事，走在繁华的街上，我依然在出神地想着这个问题。我发现，我竟然真的很舍不得那些冲我要巧克力，冰淇淋，金利来西服，大彩电的我乱认的弟弟妹妹们。

　　想起他们在我面前，又淘气又撒娇，快活得蹦蹦跳跳，嘻嘻哈哈，这种纯真的感情真让人留恋和不舍。
