流浪客与子夜

　　流浪客：北京新华社，通晓法语、英语、西班牙语等，天涯著名网友
　　子夜：北京新华社，通晓西班牙语、法语、英语，天涯体育聚焦版主

　　流浪客和子夜为夫妻，他们家的小流浪客，天涯宝贝，绝对精品。

　　那样，我得说：流浪客是标准意义上的美男子。

　　流浪客本人的相貌，该比照片上更标致得真实。

　　做演员不合适，恐怕要被演员这个职业的这两个字儿给糟蹋了。做他的流浪客，是最棒。

　　什么玉树临风、什么面如冠玉、什么潘安，什么什么还有什么，用在他身上都不为过。

　　恰好，他也有一位非常般配的太太子夜。

　　子夜身段苗条，柳眉杏眼，瑶鼻樱唇，长发飘飘，这也算是一般啦，我不想写太多这些女孩子中很长见的美貌，就象我比较喜欢写流浪客这样在男孩子中比较不寻常的美貌。

　　子夜风趣幽默，很娇弱的外形，隐隐有男孩子的性格。

　　当年流浪客大耍笔头子写去古巴的事，是做为子夜的随行家属身份去的。你看人家子夜不声不响的，流浪客反倒美得底儿朝天状，，哈哈哈，，建议大家去读一读子夜的文章，比流浪客那狗P不通的臭水平要强多了，呵呵呵，当然最棒最棒的，是他们共同的作品——他们家的小流浪客！
