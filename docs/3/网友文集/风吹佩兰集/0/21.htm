　　提到淑女，已是感觉十分陌生了。最初的印象，无非源于些老旧的文艺作品里的描述。现在早不看了，自然也就难有什么深刻印象了。事实上这并不是我一个人的问题。随便上街找一堆男人，无论年纪，问是否喜欢淑女型的女孩子，回答中九成以上是肯定的。不过追问一句淑女的形象上，回答就会有些分歧了。要是问到判断淑女的具体标准，估计就能问倒一片。如不死心，再继续追问有谁确实见过淑女的话，恐怕就有全军覆没的危险了。原因很简单，严格来讲，这世界上压根就没有什么淑女，绝对没有。有也是假的。所谓的淑女，无非是些大男子主义者的臆造之物，或者是在家中受尽黄脸婆欺凌的弱丈夫的幻想。由此衍生出来的，不过是一群伪淑女罢了。文学作品里倒是有很多关于淑女的刻画。可文学作品不骗人还有谁骗人？相信这世上真有小龙女的人该去测智商。某些时候，某人也许会说："某女孩子很淑女。"之类的话。说这话不是别有用心，就是让鬼迷心窍了。这种话的潜台词往往是"这娘们有点淑女的样子，可惜是假的。"一语中的，凡是淑女一定是假的，如真保换。这话听起来有点让那人尤其是男人失望。可事实没法改变，就象当初达尔文说人类是猿变的一样。事实再令人不悦，你也得接受。否则到时候上了那些伪淑女的当，你可真没地方喊怨去。总之，假的就是假的。虎骨是好东西，可街边地摊卖的虎骨再象真的，你敢卖吗？
