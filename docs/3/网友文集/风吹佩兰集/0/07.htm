伤心地铁

　　抛开那盟约 和你四目相接
　　想必你的眼里 应该隐瞒着更多的细节
　　他想必狂野 让你对我坚心拒绝
　　他会陪你过今夜
　　他也许 就在 这列车的某一节
　　凭一种 啊 男人的直觉
　　去承受这份残缺
　　当缘起和缘灭
　　我们的过去已不能重写
　　我失去了全部的世界
　　在这伤心的地铁
　　这么伤心欲绝
　　当列车停止 再第五街
　　冬天的纽约 冷的这么直接
　　象是你的拒绝
　　它千真万确 让人心淌血
　　他令你狂野
　　你们爱得轰轰烈烈
　　他在等你过今夜
　　我知道 他在列车的某一节
　　凭一种 啊 男人的直觉
　　去承受这份残缺
　　当缘起和缘灭
　　我们的过去已不能重写
　　我独自在陌生的世界
　　在这伤心的地铁
　　这么伤心欲绝
　　当列车停止在第五街
　　那一夜 那心里深深的雪
　　那男人的直觉 那伤心的地铁
　　
　　这是喜欢的无印良品的一首歌《伤心地铁》
　　为什么喜欢呢，我喜欢这种深深孤独的 心境
　　孤独到绝望 并且，这种在异国他乡的孤独，更彻底
　　当孤独已到最深处，其实就什么也不是了
　　其实也就不算什么了，甚至 感到一种单纯的喜悦
　　对未来 好象又明白了一些。
　　
　　每个人，都把心划分很多界限，也是为了安全
　　所谓 KEEP IN DISTANCE
　　距离是我们定的，不是别人。
　　所谓物以类聚，人以群分
　　总会我们与一些朋友互相吸引，彼此欣赏，而走到一起
　　那只是时间的问题，对不对 而且
　　现在"类"和"群"的标准，取决于我们的气质与能力
　　我想是不受时间和空间限制的，真的
　　无论欧美，无论朝代，就象，我们有时候读小说
　　我们还是可以被深深打动，人性是相同的，真的。
　　
　　这个世界也给了我们顺其自然的原则
　　很多东西往往是：我们还没求，它就来了
　　比如说，你 和 我
　　总有些东西不断的走，不断的来
　　所以就LET IT BE，TAKE IT EASY
　　孤独的感觉，也一样适用
　　
　　即使它来了
　　就象《伤心地铁》这么美丽而忧伤的人生体验
　　何乐而不为呢
