自然的心

你已经过去，你不会再次降临
那些烟蒂，灼烧着记忆的甬门
深夜花园的灌木，石子刺痛脚
露凉，风弯曲着身子亲我的脸

森林精灵的聚会，绿色的歌声
快乐成为难以理解的迷惑歌颂
徜徉的瓦尔登湖，天堂的酒瓶
天使丢弃了翅膀，它忘了贞洁

从容，我的出现愿你无所动容
叹息着蜿蜒过你那眼神和身体
那件烟薰过的T恤反复的摇荡
我剪碎了梦，粘附那憔悴舞蹈

我曾是华丽的歌者，客串自然
自从梦想出生，就用爱的灵魂
反复吟唱一首首属于鱼儿的歌
还把歌词精雕细刻在你家檐下
燕子衔泥的时候吓得妈呀一声

看到你笑了，那是我想给你的
我也会再给别人，我爱的精灵
天池鲜花盛开我在清澈地招手
自然的微笑和忧伤，为了自然
