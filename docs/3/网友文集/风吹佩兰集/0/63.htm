机场冰咖啡

　　他妈的。我在想。

　　一杯冰咖啡。

　　一个简单的包。

　　一个憔悴而忧伤的我。

　　拿出手机，想礼节一下。和各位说他妈的去你妈的哥们我走了。CONNECTING……

　　我惶惑而忧伤，我想计划着我嘴巴里该吐出的人模狗样的话：哎~~~~你好~~~~大恩不言谢呀~~~~

　　真他妈的俗，关上手机。

　　我索性坐成一尊Elegance的会忧伤地思索的雕像。可是我听到我心头的冰山在汩汩地融化，波涛汹涌，转瞬间似已溃不成军。我很喜欢眼前的这杯冰咖啡，我看到杯身和杯口的水露露，象幽曲寒绵的毛玻璃。这是我喜欢的一种意象。我沉迷于 它的这种冰冷和暧昧，是的，当感情变得暧昧，其实是心在逐渐坚硬，逐渐冰冷……

　　我依然相信我的心事，自与你别后将不再有涟漪

　　我依然相信有一种爱只一瞬却已胜千年咖啡很苦，就象是喝着蓄谋已久的泪水。真TMD酸。

　　怎样的一种心情和一种意象，竟然能让我这种无情至极的女人，轻轻的颤抖。

　　我遥望着远方，从浦东机场的这间 Coffee Shop。

　　我的远方没有远方。

　　你为什么不说出来呢你为什么不说出来呢你为什么不说出来呢？

　　我知道你舍不出你自己。所以你的将永远是种暧昧的意象，模糊而毫无价值，就象做爱的女人装出来的呻吟，嗲而虚伪。

　　痛苦因无法痛苦。

　　这种文章是没办法在天涯贴出来的。因为他们那群狗日的比关心我的 更关心我的私生活，靠。在那里我TMD象在镁光灯照射的舞台上，我的冷静理智放浪形骸与熟透着的风情无处可避地是一种直指内心的表演，可是，很多的时候你知道，就象现在我愿意试着哽咽住喉咙堵住的感觉，躲在这样一家寂寞幽暗的Coffee Shop 放松我的惆怅。

　　补充一下，文中出现的那首诗曾经奉献个过二百来个帅哥。

　　钦此。
