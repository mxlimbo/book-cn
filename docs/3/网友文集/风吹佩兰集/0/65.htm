玫瑰赠与侠士，美酒赠与佳人

　　去看那个叫娟的美丽女人。她有着偌大的房子，如宫殿，且装修考究。只一个人住。一个独身女人。

　　她过生日，我去祝贺，只有我一个。偌大的房子遂有些寂寞。 娟在典雅明净的厨房里忙碌，不紧不慢的，一举手，一投足却是别有韵味。女人在厨房里，所流露的风情，总是别样的。有的是一种那些不懂下厨房的女人所永远得不到的气质。何况此时，是一个美丽端庄而自信的女子。

　　娟的所有房间，都是独具匠心，所有的摆设，都是别出新裁。 她象一朵寂寞百合，安然地在这少人涉入的幽居，绽放。

　　她的前夫，先是一个普通职员，后来下海，也没什么成绩。倒是娟，自学且拿到了学位，在一家收入颇丰的外企，养活着两个人。可是，离婚了，原因，却不在她。

　　"我被人踹了，"娟笑吟吟："也是我对他不够关心，他爱上别人了。虽然，那个女人当时看好的是他的钱――我给他的钱。"

　　她的笑容漾开来。我也哈哈大笑着，两个人笑得弯了腰，我笑喘着说："神经病。"

　　太太比那女人强得多，有魅力且有钱，男人却宁愿选择没钱没魅力的，为什么？

　　可能人的快乐，也来自对自己的一种肯定。也因为没有得到关心吧，也因为娟太成功，他在太太面前反而找不到自信。他也肯定知道那个女人不如他的太太，可是，她的崇拜能给他快乐的感觉，能找准自己的感觉。他宁愿离开美丽而风情的娟。对他来说，这很重要，其实他爱的，其实不是那女人，而是，他重新找到的自己。

　　"我曾想尽力挽救我的婚姻，"她执了斟了半杯的酒也我对撞， "我就在那个公司辞职，回到离家近一点的地方上班。并且和他说；'我可以忘记一切，只是你不可以再这样。'可是，同样的事情又发生了，我们就离婚了，不久，那女人离了婚，他们，结婚了。"

　　娟淡淡笑了起来，神情中却是说不出的淡淡的忧伤。

　　"尽管他是个不怎么成功且伤了我的感情还很窝囊的男人，可是离开之后，我发现我最爱的，还是他。"杯中美酒一饮而尽。

　　"一个人过，也蛮好啊。"我们换了几种不同样子的酒来喝，娟的面色已潮红，笑吟吟地，如云的长发随便地挽着。

　　"其实，做事情努力些，不错。除了成就事业，也可以忘记痛苦。"我笑着点点头。想着，又想着，又细细想起来。

　　"下次也给你带酒。"我离开时，笑着说。

　　她笑呵呵地送我到街，说："这世道真是变了，美酒送给佳人，玫瑰要赠与侠士了！"

　　两个人遂当街地大笑，清脆的笑声被晚风送了好远。
