龙非池中物 挟雷欲上天 （小记贩子）

　　光盘贩子：天涯著名网友，北京中科院博士 　

　　贩子大哥问我：佩兰怎么一下午也没喝水？

　　我说：你帮我倒点吧。

　　贩子大哥去饮水机那找出一次性的套在一起的杯子，拔出一个，　　注满清水。当他转过身的时候，他的表情竟然很慌乱，把水放到我面前的时候，更是心事重重，手便哆嗦了一下，水轻轻地溅出来，几滴洒到软蓝色的牛仔裤上。

　　他谈起从前见紫冰的时候，有一段尴尬的事，他说：就象给你倒水一样尴尬。哦？我问：给我倒水很尴尬么？贩子大哥心怀鬼胎地说：我很害怕写小记的人。烟客曾经写过贩子小记，

　　嘿嘿，他威胁道：然后我发起了捣烟运动，并取得阶段性的胜利，嘿嘿，所以，我给你倒水这个情节不要写上去呀！

　　喝光了第一杯水，贩子大哥顺带着去帮我续第二杯，并老谋深算地说，要连敬我三杯。

　　席间，贩子大哥先给大家满酒，满到我这的时候，他如释重负，说：齐了。

　　我便给大家布茶，补酒。终于，看见贩子大哥长出了一口气，说：成！你也给我倒了第四杯了，补回来了，我不吃亏了。

　　顿时，贩子大哥笑魇如花。

　　谈到网络，贩子大哥说：佩兰，你来得晚，有些东西想和你说说，其实网络也是很伤人的，不象现实的世界，前面倒了，后面绕过去。网络世界的感情，就象是飞蛾扑火，前仆后继，所以要注意保护自己。

　　谈到女人，贩子大哥含笑着，说女人有天生的适应家庭的能力。他的太太是北大的班花系花兼狗P才女，从小娇生惯养，昏后仅半年就驾轻就熟，尽管工资比贩子大哥还高，在家务上，却从不用贩子大哥操一点心，幸福生活，悠哉乐哉。
