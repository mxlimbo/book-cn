夜读紫式部《源氏物语》

　　关了电脑，已是凌晨。倚着厚厚的趴狗和枕头，旋起台灯，幽粉色的光不动声色地氤氲着。便听到有丝丝风雨敲窗的声音，象夜的叹息。找到一支透明黄的荧光笔，笔尖扁得正好，夜读紫式部《源氏物语》。

　　便读到雨夜品评的一段。评的，其实是女人。这里却也有了一段延伸。

　　左马头说："……不论男女，凡下品之人，稍有一知半解，便尽量在人前夸耀，真是可厌。一个女子潜心钻研三史，五经等深奥的学问，反而没有情趣。我并不是说做女子的不应特地钻研学问，只要是略有才能的人，耳闻目见，也自然会学得许多知识。。。。

　　其实，这左马头也算不上是个什么高人，他所说的自然也不必当做什么蓝本。只是孔子那句老话是：三人行，必有我师。看到这，心里倒是有些触动。。

　　想到自己，对知识，其实就是个三脚猫，蜻蜓点水。

　　有时候，总想问别人：你认为我是怎么样的人呢？其实，有谁在心里不知道自己是个什么样的人呢，有谁能比自己更清楚？问的目的，更多的是想了解自己掩饰的自己虚空虚无虚伪的能力和道行有多成功罢了。谈到掩饰，如左马头说的"便尽量在人前夸耀"是不是一种掩饰呢，是的，它是。满足自己的虚荣，掩饰自己的空泛，想粉饰别人的感觉，陶醉和欺骗自己。人生也有涯，而知也无涯。看到其他网友们的渊博与虚怀若谷，也就象每次到书店的感觉，觉得该学的真是太多，哪里该留给自己买弄浅薄的时间呢。。也谈到历史，说句心里话，我从来对历史哲学不感冒的，也无兴趣。其中一个原因，就是潜意识中还没真正考虑到需要它。向左马头所说：只要是略有才能的人，耳闻目见，也自然会学得许多知识。。

　　是可以学到，但也不必提"只要是略有才能了"，只要是有思维的人通过耳闻目见，都会学到许多知识的。我也想：其实，历史就是巨人的肩膀，是人类耳闻目见的浓缩精华本。其实，读懂它，才能明白得更多一些。

　　又想谈到难得糊涂的问题，对于郑板桥这种24K精品版，纵然可以权当认为他对很多问题大彻大悟了。因此他可以以傲视上下五千年的翩然姿态，萧飒一声长吟：难得糊涂。而泱泱古国，潮起潮落，世间如他者又有几人呢。

　　于更多的绝大多数如我的普通人：该是人生福在快乐，人生贵在明白。

　　想来想去，心里似乎也万籁俱寂一些，打算以后也放下心来读一读什么《前汉书》《后汉书》，什么《春秋》什么《札记》，也多明白明白。
