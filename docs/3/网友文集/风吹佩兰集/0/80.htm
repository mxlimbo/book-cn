那些乍然而惊的相遇

　　那些乍然而惊的相遇呵
　　
　　我许久地，在这里等了你许久的
　　我知道你不相信那些美丽的童话
　　我在还怀疑关于开花的树或其它
　　
　　我还没有在乎我的衣衫是否美丽
　　是否吸引你是否能够施舍给你梦
　　是否我曾经可以做那掌中的女子
　　华清新浴的女子红拂夜奔的女子
　　或其它。
　　
　　于你我是漫天的飞花漫天的细雨
　　甚至是夜里长着翅膀的风或叹息
　　是你惑的迷惑和手心清楚的纹理
　　是宿醉隔夜再续的红酒和残凉呢
　　
　　这些都不重要，亲爱的
　　我会是一切的一切似曾亘古不变
　　我就悄悄地微微笑着以你看不见的角度
　　或尘或冥想或梦里，只为你能给我的
　　
　　那些乍然而惊的相遇……
