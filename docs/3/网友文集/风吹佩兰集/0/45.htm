亮眼睛和过客

　　作者：风吹佩兰
　　亮眼睛：北京中粮
　　过客：某公司董事

　　亮眼睛与过客为夫妻，均为天涯著名网友

　　我还是想再回味一遍亮眼睛和过客。

　　客观地说，亮眼睛不是标准意义上的美女，以我做同性之间版主色咪咪的眼光来看。

　　可是以我做舞文弄墨版主的极其审视的眼光来看，"大美女"三个字非亮眼睛莫属。

　　她这网名取得真是好，眼睛就是闭着，也能觉察到光芒来。

　　她不靓，但很亮。

　　这是一位极其聪慧的女子，就象日光，照得人很亮。她是透明的，而实际，又有着赤橙黄绿青蓝紫丰富而缤呈的内涵。

　　无疑，她是极优秀的，待人，待事，待物，对待生活。是的，你会觉得和她在一起是完全的舒适，这种舒适来源于她对生活的各个方面的一种真心实意的纯真，和满足。

　　过客和亮眼睛在一起的时候，他的笑容有着不易察觉的舒展，真不知道这样的琴瑟合鸣的福分，是曾经修了几千年才能修到。
