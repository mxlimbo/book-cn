老 歌

　　哲二不承认自己是一个悲观的人，因为每件事情他都把结果想得很美好，最后落得一个失望。

　　‘假如我是一个悲观主义者，失望一次下次还会这样么，可是我不仅下次这样，下下次还这样，下下下次，，，下下下下次，都这样。’

　　哲二说，‘所以可见我是一个乐观主义者。’

　　我服了他的下下下下次，如果他不是一个偏执狂，那才真见了鬼。

　　哲二把我那里的旧磁带都收集到了一起。

　　‘我要做一个老歌的拼盘，你帮我找人出，肯定赚钱。’

　　哲二想挣钱想得有些发疯。

　　可是我偏偏是一个不折不扣的悲观主义者，‘磁带是说出就能出的吗？就算出了卖得动吗，你看看现在有多少新磁带，能卖出去的又有多少？’

　　哲二对我的话毫不理会，一盆冷水还回来，浇得我一个透心凉，‘我不管，我这盘里面都是经典老歌。’

　　我一点儿不以为然，经典的东西多了，再说人家要听，买原版的多好，要你这个拼盘干吗？！

　　哲二把他的拼盘小样拿来了，里面有邓丽君，蔡晴，凤飞飞，罗大佑，陈百强，刘文正，齐秦。。。正是一个大杂烩。

　　我说：这真是一个毫无个性的大拼盘。

　　按照以往的经验，我断定这次将是哲二的下下下下下次，但是他的这个忙我必须要帮。

　　我找了一个音像公司的人和他谈。

　　哲二对人家说：你先听听，听听就知道我这东西有多好了。

　　然后就死乞百赖把人家按倒在沙发里听那个小样。

　　音像公司的人听完出来后跟我说：快让你这个朋友死了这条心吧。

　　这下子我踏实了，当初我说什么来的？根本就是不可行的事嘛，你以为这世界上的事都这么好办哪，不给你点儿教训你还不明白呢。

　　没想到我说完这番话，哲二居然不为所动，‘我知道问题出在哪了！’他说。

　　然后的几个月，我都找不到哲二的人，不知道他跑到哪里去了，直到有一天我在一个地摊上看到哲二的那盘拼盘《老歌》，印制的极粗糙。

　　看地摊的人嚷道：十块钱三盘了啊！ 我掏出十块钱买了三盘，另两盘是毛宁和尹相杰的。

　　一回家我就给哲二打电话。

　　‘喂，我看到你的那个拼盘了，被盗版了！’ 电话那边是哲二懒洋洋的声音：‘你在说什么啊。’

　　我把买磁带的经过说了一遍。

　　哲二在电话里笑了起来：‘没有盗版，那就是我的正版，只不过是找做盗版的人做的，还不错吧。’

　　不错个屁！我心想，这件事怎么会发展成这样。

　　哲二在电话里还说：‘对了，哪天哥们请你吃饭吧，毕竟那母带还是从你那里划拉来的呢。

　　当然了，最后发行的还不如我那小样呢，不过我当初不就是为了挣钱嘛，也满足了大众怀旧的情怀，一举两得，还什么事都没费。’

　　哲二最后说：你瞧，其实世界上的事就是这么简单。

　　挂了电话，我坐在椅子里愣了半晌，然后把十块钱三盘的磁带一股脑地从楼上扔了下去。

　　99/10/15中午
