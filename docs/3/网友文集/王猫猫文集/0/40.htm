路过中关村

　　我揣着一包瓜子和哲二去海淀图书城。 路过中关村的时候，哲二让我在门口等他，他去办点儿事。我站在门口有一面大镜子的楼门前，旁边有一个女人在卖计算机的零部件。这个中午的阳光有些灼人，我穿的毛衣外套似乎多余了。

　　突然听见踢哩趿拉一阵脚步声，然后一个穿制服的人把我喝住：

　　'干什么的？'

　　'恩？没干什么。'我一愣。

　　'没干什么？没干什么你站这儿干什么？！'制服很凶的问。我朝周围看了看，发现卖零件的女人不见了，而她的零件还在，在我的脚底下。

　　'我随便站站。'我的口气突然有些发虚。

　　'随便站站？'制服挑衅地看着我。'那就请你随便跟我走一趟吧。'这事可不能随便，我明白。

　　'这些东西不是我的！'我说。可是制服显然已经不相信我了，'不是你的你站在这儿干什么，那么多地方你不站，你偏偏站在一个不属于你的东西的旁边，这怎么解释？''是啊，是啊，我为什么要站在这呢？你说的很有道理，'有汗顺着脑 门往下淌，'这件事我也不明白。' '会明白的，一切事情。'制服这样总结道。 制服和我一唱一和地聊了起来，周围围着的人也同我们一起聊，我想起 我揣着的瓜子，就拿了出来。

　　'吃，吃，大家吃啊。' 可是制服义正严词地制止住了我，'不要妄想贿赂我们。' 后来有明白人替我说话，说的确不是我的，是一个女人的，那个女人跑 了。 虽然这样，制服还是对我进行了好一番说服教育。 就这样，在这个明媚灼人的下午即将过去的时候，事情终于算明白了。 '以后不要随便站！'制服带着零件走的时候对我说道，'站错队不是 好玩的！' 我捣蒜般点着头，原来不能随便站啊。

　　这就是在这个秋天的温暖的下午，在中关村发生的一件小事。
