眼中的王猫猫

Pass

　　我第一次见到王猫猫的时候她正在屁颠屁颠地过马路，一辆大型的公共汽车在后边紧紧地追着她，王猫猫一点也不紧张，她躲过那车时的姿势就象一个训练有素的间谍，"真他妈的悬啊。"王猫猫摸着吓白了的小脸对我说。

　　我一下子就对面前这个都市小女子产生了好感，在现在这个年代不是每个人都能如此准确地表达出自己的思想，而且刚见面她就和我说这么知心的话，显然是准备和我做一个长期的朋友的，为了表示友好，我就把兜里的烟盒掏出来撕了在上边写了名字当名片递给她，这是一个很庄重的场合，庄重的场合都得搞点书面的东西，王猫猫严肃地接过烟盒纸看了一眼，很爱惜地丢掉了，当时街上的风肯定比烟台海滩的还大，因为她一松手那纸片就撒腿而去没有一点若即若离的意思，我追了四条街才在一个捡破烂的正要下手前抢了回来，那个捡破烂的嫉妒地看着我，我知道他也很想拥有这个纸片，于是我就又撕了一片写上名字塞给他，王猫猫被我的善良深深地感动了，"你丫有病吧。"她说，这么短的时间她已经开始关心我的身体了，这是多么大的鼓励啊。

　　我和王猫猫坐在一家美国人开的快餐店里，很快切入了正题，"你不能把所有的鸡翅都吃了。"我崇拜地对自己的偶像说。"我没有，我给你留了一块。"王猫猫谦虚地把第十二块鸡翅塞到嘴里，她吃东西的时候目光深沉象正在吃食的小老虎，我伸手去取那块属于我的鸡翅时看见她的目光我吓了一跳没敢拿闷头品着手里的蔬菜沙拉，吃完饭之后王猫猫就显得更加亲切了，她用纸巾特优雅特矜持地擦着嘴边的油，我这才发现她没有胡子，也没有鼻涕，人和人的素质就是不一样啊，上次见到的MIKKO就两样东西都有。

　　街上很冷，人们弓着腰努力把自己假装成虾米，他们以为这样风就不会吹他们了，真是幼稚，也有人戴着口罩装成特务，我们的王猫猫就不这样，她总是挺胸抬头从来不四处乱看就算有帅哥经过也顶多飞个媚眼，王猫猫在我眼中是一个很美的女子，不但看上去很美，看下去也很美，她写东西时经常把自己幻想成毛毛虫，我很佩服，我只能把自己想象成桌子腿什么的一点也不美丽生动，这方面我还有一个朋友宁财神也挺厉害，他能把自己想象成一个正在发情期的螳螂，于是在他的字里行间我们经常能闻到一股很特殊的味道。

　　王猫猫是一个很有审美观点的人，她只穿两个牌子的衣服，一个是"2021"，另一个也是"2021"，她看人也很准，"倍儿瓷实。"她背后是这样评价她的领导雪溪的，非常客观，王猫猫说这句话的时候正坐在她的办公室里，她的脚放在桌子上，我发现她的脚长的非常美，于是我就很认真地观察她的脚，我觉得她的脚和毛虫的脚在数目上有很大的区别，因为毛虫没有脚，在办公室里的王猫猫终于流露出都市女子特有的寂寞感，"电话怎么还不响，"她若有所思地说，"是该装个电话了。"她又深谋远虑地说。

　　晚上我们去了歌厅，这是我第一次和三陪小姐以外的女性到那么神圣庄严的地方去，王猫猫一进歌厅就表现出与众不同的大家风范，她把麦克风系在腰上谁也不给爱谁谁，当然这也反应了她对歌唱这门艺术的热爱，后来另一个小朋友MIKKO哭了，因为他举了很多次手王猫猫也没把麦克风给她，MIKKO当时就说了一句很深沉很绝望到现在我也没能完全理解的话，"打倒王猫猫。"他说。

　　王猫猫还是一个心地非常善良的人，对于一些思想有问题的同志她也很注意帮助他们改造自己共同进步，有一次我在聊天室里就亲眼见到她帮助一个叫作爱屋及污的落后青年，"你他妈的别招我，找抽啊。"王猫猫温和地对他说。

　　这就是我见到的王猫猫，一个清纯可爱的网上女作家。
