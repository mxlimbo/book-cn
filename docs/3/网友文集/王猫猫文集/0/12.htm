你是一个坏东西

　　早晨拉开窗帘，发现外面蒙蒙地下起了小雨。 趿着拖鞋走出来，迎面碰见邻居的女儿上楼，她照例是不理我的。小女孩子的骄傲。 下了楼，点燃一根烟，站在楼洞口看雨。 实在没什么好看的，门口有稀疏的几棵灌木和歪歪的绿草，然后就被一幢幢的楼 房挡住了视线，唯一欣慰的是人很少，不象平时的嘈杂。 邻居太太穿一身纯棉碎花衣裤买了早点回来，油条的油腻冲淡了一些空气里的清香。

　　'早啊，看雨呢？'邻居太太还打了一罐豆浆。 '恩，买早点去了？'我也明知故问。 邻居的女儿又跑下楼：'妈我出去一趟。' '干吗去啊，外面下雨呢，早饭不吃啦，大周末的瞎跑什么。。。'母亲不满了。 女儿皱着眉跑过我身边：'真烦。' 我突然就想起罗大佑的歌来：

　　'假如你先生来自鹿港小镇 请问你是否看见我的爹娘 我家就住在妈祖庙的后面 卖着香火的那家小杂货店 假如你先生来自鹿港小镇 请问你是否看见我的爱人 想当年我离家时她已十八 有一颗善良的心和一卷长发' 香烟快吸完的时候，我的脚板儿隐隐有些发凉。 爬回我的小窝，躺在我的小床上，一时又思绪万千。

　　突然听到镜子里有人叫我：

　　'王猫！王猫！'

　　我仍旧懒得起身：'干吗啊？'

　　'你是一个坏东西！'镜子里的人说着，还咯咯笑了起来。

　　'坏就坏吧，这年头坏人好当。'

　　'那你痛苦吗？'镜子里的人似乎严肃了些。

　　'我痛不痛苦关你P事。'

　　'有时候痛苦会让人堕落，当你不能忍受的时候。'

　　'那么就让我堕落好了，又关你什么事。'

　　'人类堕落谁最高兴？想想看。'

　　'谁最高兴又关我什么事。'

　　'唉~~，'镜子里发出一声长长的叹息。

　　我昏昏沉沉就这样睡去，梦里徘徊着大佑的歌声：

　　'台北不是我的家 我的家乡没有霓虹灯鹿港的街道 鹿港的渔村 妈祖庙里烧香的人们台北不是我的家 我的家乡没有霓虹灯鹿港的清晨 鹿港的黄昏 徘徊在文明里的人们'
