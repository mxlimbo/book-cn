也来开两分钟研讨会

林迷糊

　　一会儿还要见老板，简单说两句吧。

　　紫兄说猫猫的文字在“不想说什么”时最佳，我以为是对的。但我个人的感觉是，在猫猫不少“不想说什么”的文章里，缺一点点气氛。

　　比如有一个故事：有一个欧洲女子，年轻时被打纳粹的美国大兵强奸，生下一个儿子，她一直非常不喜欢这个儿子，尽管这个儿子非常希望得到她的爱，最后，当儿子生活潦倒时，母亲终于有点过意不去，去看望儿子，并表现出十几年来少有的温情时，儿子的反应是，。。。猫猫给想想？

　　这是我几年以前看的一部欧洲影片(记得是比利时的)，影片最成功的地方，在于其气氛的培养，到了最后，当母亲出发去看儿子的时候，每一个人都觉得，不论发生什么，都是意料之外，情理之中。换句话说，到了这会儿，不论编剧想说什么都可以，都让人揪心。

　　猫猫淡色的笔法，也许是自己的特色，但有时候，会让人觉得，怎么说，到喉不到肺：)，原因可能就是淡色之中缺一点点对比，或者一点点反差。我个人觉得，也许应该注重一下渲染气氛的训练，有了不用，和没有，写出来的东西恐怕是不同的。

　　也是姑妄言之，姑妄听之吧。
