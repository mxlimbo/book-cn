一场戏

　　零点的时候，电视里正在演一场戏，他们在一起吃饭，那个叫拉宾的男人对着给他倒酒的侍女的胸脯凝视。我喝了口酒，酒很辣，尝起来像辣椒。

　　他们跳上汽车跑远了。

　　路的两边是绿树，女人对拉宾说：

　　"我们一起走吧，反正我们都是一个人，我丈夫外出了；而你的妻子，她此刻正在睡觉。"

　　拉宾："我们不顺路。"

　　女人："他们说你是把好手，可是你要证明给我看。"

　　拉宾："我知道你需要什么。。。你需要找个人抽你一顿。"

　　女人："我家里面有皮鞭。"

　　女人离开了，她不是一个年轻的女人，当然，拉宾也不年轻，"我起码有四十岁了。"他对我说。

　　"我明白。"我喝了第二口酒。"那么，她罢休了吗？"

　　"当然没有，"拉宾说，"你知道，有一种女人，她们是永远都不肯罢休的。你不知道什么时候惹了她们，她们突然就变成一条恶狼，恶狠狠的扑向你，把你撕成碎片。"

　　那次拉宾渡一个可爱的柔弱的女人过河，她非常虚弱，在船上，她站立不稳。拉宾扶着她，她大叫起来，"你侮辱了我。"她抽泣着，"我要控告你。以猥亵罪。"

　　"真是太妙了。"我笑着说。

　　拉宾把他的脸藏在灯光的阴影里，"她们有时候是显得很幼稚的。我们应该允许人有丧心病狂的时候。"

　　那个时候两个女人联手在法庭上演戏。

　　拉宾问柔弱的女人："请问在发生受控事件时，您在做什么？"

　　柔弱女人："我在大叫。"

　　拉宾："您为什么不反抗呢？"

　　柔弱女人："我的双手被你抓住了。"

　　拉宾："这么说我长了三只手吗？我在用第三只手猥亵您。"

　　我没有说话。

　　拉宾也不说话。

　　风把窗帘吹得一飘一飘的，有一小部分阴影落在拉宾的脸上，使他看上去显得有些邪恶。他的裤腿被风吹得一鼓一鼓的，于是我说：

　　"冷吗？我把窗户关上好了。"

　　"好吧。"他像个老年人一样答应我，然后听从我的摆布。

　　我拿了一床棉被盖在他的身上，他看上去真的很老了，"现在是什么时候了？"他问我。

　　"是二零零零年的一月，凌晨。"我站在窗口看着外面回答他。

　　窗外的寒风里树影一齐摇动着，有些微的雪屑被刮下来，落在地面上，还带去一些尘埃，那些尘埃，你知道，它们总有好几千万年了，虽然看上去还是那么的新鲜。这个世界上有很多东西是你所不知道的，你根本不会想到，有那么多的事情你根本不知道，它们天天在你的眼前。

　　此刻的拉宾，静静的坐在躺椅里睡着了。他没有等到黎明的到来。

我像个傻子守护在他的身边，看他把这场戏演完，之后坐起来抽烟。我甚至点了一只蜡烛来驱散烟雾，有人说那很灵验，他们对你说：

　　"很灵的，只要你试一试。"

　　我本来是要叫醒拉宾的，他睡得太沉了。可是我点了蜡，为的是驱散烟雾。这延误了我叫醒拉宾的时间，这一次，我想到要喝些酒，酒还是很辣。

　　他们说：

　　"那么抽一支烟吧，烟可以让它不那么辣。你为什么不试试呢？"

　　我再次点燃香烟，当然，还有蜡烛。

　　我想，黎明前我一定要叫醒拉宾，以便让他回到他的世界里去。他不喜欢这里，我看得出来，人们总是喜欢呆在生他们的地方，他们称那里为故乡。在故乡里，人们似乎可以为所欲为，他们熟悉那里的一切，然后他们在那里养成他们所特有的癖好，比如赌博。

　　既然说到赌博，我不得不多一句嘴，我觉得，赌博是最明智的，我所知道的越南人喜欢的俄罗斯轮盘赌，就是放一颗子弹在枪膛里，一边是太阳穴，一边是一辈子享受不尽的金钱，假如你幸运，钱就是你的；假如你不走运，你只好就等下辈子。

　　直接把欲望兑换成生命是很需要勇气的。

　　天真的亮起来了。

　　"拉宾。"我叫。

　　拉宾没有动，他的两条腿僵直的伸开，毫无生气。

　　我走近他，替他把被子拉上去，遮住了他的脸。

　　2000年1月31日
