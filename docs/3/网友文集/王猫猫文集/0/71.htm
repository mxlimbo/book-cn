我爱猫猫

俞白眉

　　网络上许多人知道的一个著名女作家王猫猫是我的偶像。

　　不过说这个也没有用。第一，王猫猫是很多人的偶像；第二，我的偶像也不止她老人家一个。既然并非一对一的关系，而是多对多的关系，那么我对她的一点非分之想只好打住了——群雄并起，白眉自知功力浅薄缺乏竞争力的，只好把一点痴情深深埋在内心深处不敢面对——留着独自在亮着台灯数伤痕时对自己叹气罢。也只叹得一叹，不觉就泪流满面，不能自持了。

　　初，网络文学还混沌那阵子，天下大乱，斜刺里杀出来了一个王猫猫，悍勇非常，文笔老辣，下手时一般轻易不带留活口的。白眉作文，非得五六千字才勉强把话说利索的，到了王猫猫大侠女那里，一两千字的哲二系列小段，不露声色地就杀人于无形了。这个在武侠小说里叫飞花摘叶亦可伤人。 白眉心中暗表：网上文字，毕竟写心情描情绪者居多——若小女子作家，更是如此。或者，风花雪月，轻舞飞扬，不甜不酸的初恋涩果；或者，百无聊赖，颓废苦闷，不尴不尬的一夜迷情；又或者，放荡形骸，游戏人生，不清不楚的堕落故事。无非把一点有限的人生经验放大夸张，作者往往写得忘情，且写且哭，且哭且写，恨不得逮谁跟谁嚷嚷"我还纯情着呢"或者"我已经不会再纯情了"或者"我已经不知道什么是纯情了"。三种姿态，形状各异，实则本出一脉。

　　若往开了说，又何止网上，又何止女性。传统媒体热卖中的拿自己黑白艺术照当封面的所谓七十年代女作家群，网络文学中号称经典的《第一次亲密接触》，可不都是把自己关浴室里一人照镜子顾影自怜感春悲秋照出来的？整个大场子浮躁如此，都忙着小资呢，咱们还真不好意思提更高要求了。

　　所以好象猫猫女士这样的就显着稀罕了。一样的如花似玉，一样的感情细腻，人家可是从一开始就琢磨着言之有物，透过现象看本质啦。睁开眼睛看世界的文字，总是要比闭着眼睛光寻思自己身体构造的文字可爱。

　　白眉这么说的一个明证是王猫猫的幽默。本来我有一个偏见，以为幽默是这样一个东西：只有你比别人更清醒，它才不至于沦为贫嘴。我的另一个偏见是女人终究不可能是清醒的。我的这两个偏见造成了我的第三个偏见：贫嘴的女人偶然我们可以见到，幽默的女人则从来没有。要感谢王猫猫，她让我一下子就把后面两个偏见都纠正过来了——否则，对很多女权主义者来说我将是一个固执己见满肚子大男人主义的混蛋。那样的话，只会对我自己的婚姻前途不利。

　　在网络上大放厥词是一种快乐。但是一定要有挨砸的准备——挨砸的时候就不很爽。俞白眉大放厥词以后，已经做好了挨砸和还击的准备了。比方说，可能有些对号入座的会比较生气：网络写作本来是卡拉OK，自娱自乐的，无关功利，你自夸你想夸的就是了，何苦打击这么一大片爱好者的积极性呢？关于这种批评白眉想，是这样——假如卡拉OK歌厅里允许大家爱怎么唱就怎么唱，也就应该允许一边被聒躁得不耐烦了的茶客评议说：这个唱得好听一些，那个的嗓子嘛，呵呵，和公鸭子其实也没什么不同。
