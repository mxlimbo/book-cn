时间概念

　　‘人生其实就是堵车。’电视里的男人说道。

　　那次我迟到了将近一个小时，因为堵车。

　　哲二说：你这人怎么一点儿时间概念都没有？

　　我说：路上堵车来的。

　　哲二说：那你不会早点儿出来啊！

　　我突然恍然大悟，我是可以早点儿出来的，为什么我没早点儿出来呢？

　　因为我没有把会堵车的因素考虑在里面，也就是说，我考虑问题还不很全面，我还应该想到，如果没有堵车，可是表出了问题让我看错时间了怎么办，或者路上遇到熟人聊了会天怎么办，或者我突然拉肚子了怎么办，当然还有很多种其他的可能性让我晚到。

　　在所有的这些可能性里，假如哲二没有等我，她先走了，我们就会错过这次见面。

　　没准这就是我们最后的一次见面。

　　人生究竟有多少种可能？ 就在早一点儿与晚一点儿之间。

　　假如那天我来得很早，情况也许是这样的：

　　‘王猫，我爱你！’哲二的红唇送上来。

　　然后汽车载着她奔向远方。 我泪眼朦胧地看见她的一小块红头巾在风中飘扬。

　　或者我来晚了：

　　‘王猫，我恨你！’哲二的五指山送上来。

　　然后她哭着跑向密云，也许是廊坊。

　　我无辜地捂着红彤彤的脸蛋眺望。

　　再或者。

　　‘亲爱的猫，明天我要嫁给他了，你，就不能有点儿表示吗？’哲二低头抚弄发梢，发现上面净是分叉的。 ‘我，让我，再深情的给你一――枪！’ 我掏出手枪气急败坏地朝哲二扫射。

　　哲二如花的容颜在我面前慢慢暗淡，褪色，哲二倒在血泊中。

　　‘天啊，为什么会是这样？！’我仰天长啸，‘既生猫，何生二！！’

　　为什么任何一种结局都是不幸？！

　　不！

　　我和哲二结婚了。

　　我们在一间小屋里。

　　‘亲爱的猫，我太爱你了，哪怕你的脚有点儿臭。’

　　‘我也是啊，我的二，这世界上只有一个男人知道你有狐臭，那就是――我，我不允许别人知道！’

　　时间‘嗖’地过去了一半。

　　‘臭袜子你自己洗啊。’

　　‘早知道你狐臭我还不如娶一狐狸呢，起码还是一国家二级保护动物。’

　　时间‘嗖’‘嗖’‘嗖’地过去。

　　满脸皱纹的哲二无奈地看着满脸车道沟的王猫。

　　‘熨斗在哪里？’

　　‘在抽屉里。’

　　‘让我帮你把五线谱熨平。’ 都是时间惹的祸。

　　所以，你现在知道了为什么悲剧比正剧受欢迎了吧，那是因为里面的主角还很年轻。

　　时间比不幸重要。

　　‘你怎么又迟到了，什么时间概念？！’哲二揪着我的耳朵。

　　这个时候，我的朋友们，我正年轻。
