一段日子

　　我的这个情人，很早以前我们就认识了，认识的时候，就有种"恨不相逢未嫁时"的感觉， 我匆匆忙忙的连现任男友都没顾上处置，就一头栽进了汹涌的爱河。 等到栽进去了，才知道爱河原来可以这么汹涌啊，它一浪接着一浪，让你没机会喘气。突 然喘了一口气：我这儿干吗哪？！ 走了，出来了，觉醒了。 那么多河，我为啥非要在这一条河里淹死呢。 对于我突然冷淡的退出，情人表示理解，同时我们保持了良好的朋友关系。 有一天我看一位名作家的小说的时候，突然想起情人来，他就象我手里这本小说：看过很 多次了，看的时候不忍撒手，等看完了，有一段时间再也不想看，全忘了，过一阵子一看， 挺好，又当新的看了一遍，又被感动一回。

　　于是我飞快的呼他，他飞快的回了电话，我们飞快的见了面，突然间缩短的距离让我们彼此都不适应。

　　"还是有距离好。"他说。

　　"现在都知道相见不如怀念不是吗。"我郁郁的喝着咖啡。"让你活在我的怀念里 吧。" 他不置可否。

　　草率的冲动毁了我，毁了很多回。这次也一样，我回家痛苦的揪了半天头发：不该见他，真的不该！　我在桌子前静静的坐下来，拿出那本小说继续读。夏日的夕阳默默照着我的头顶，透过纱窗，和着一股烫人的青草香，很多年很多个夏日都是这样的，象一个剪影，我坐在窗前，等候。等候我的父亲来把我带走。等候一个富贵多情充足的模子把我带走。然后把我的心掏出来放在书里夹好，不带它走，它是个麻烦的东西。父亲要永远年轻，永远和母亲相爱，幸福地和我一起住，还有妹妹。我的心在另一个地方生活，也许幸福，也许不幸，这要看作者怎么写了，但是一定要曲折，有爱有恨有苍凉，这样，我等着。等着孙悟空把他的分身术教我。等着福尔摩斯来找我。等着一段惊心动魄的日子。妹妹说：你的脑袋越来越大了，这些日子。是吗，一定是手脚老不动的缘故，光锻炼脑子了。

　　朦朦胧胧的，我还想有个他，吃饭的时候，他可以和我闲聊：

　　"北约真凶！"

　　"要是瓦尔特活着就好了。"

　　睡觉的时候，他搂着我：你的后腰为啥有三个小坑？

　　压马路的时候，他拉着我，我被动的东张西望。

　　这个暑假在九月份结束，这段日子在九月份失踪。

　　冬天教室的火炉旁偶尔映出一个火红的脸膛，那是另一个故事。

4/14/99
