道德的标准

　　阴雨绵绵的下午，玉帝把王猫叫到身边。

　　'王猫啊，你来我这里有多久了？'

　　'玉帝老儿啊，我已经来了有十多天了。'

　　'那么我现在要交给你一个任务，你去把世界上最好的人领到我这里来。'

　　'好的。'

　　王猫领旨飞身下了凡尘。

　　不多时，王猫领了一个人来到玉帝面前。

　　'玉帝老儿，我把人带来了。'

　　'哦？'，玉帝微眯起双眼，'待我看来。。哇！'只见那人：

　　蜡黄的皮肤红红的眼，

　　黑色的牙齿泛金光，

　　干柴似的身板浑身针眼儿。

　　'这，，这是什么人？！'

　　'这就是您要的世界上最好的人啊。'

　　'完了，这个世界该毁掉！'玉帝长叹一声，靠在藤椅上流下泪来。

　　王猫看了很奇怪，'玉帝老儿，你怎么了，你看，这个烟鬼酒鬼加色鬼把世上能享受的东西都享受了，集了人类精华于一身啊，您看见他，就知道人类是多么会享乐了，这样的人类您不喜欢吗？'

　　玉帝只是说：'妖孽啊，妖孽。'

　　王猫不理玉帝，接着说：'那您把我这趟的差旅费给报了吧，来回一共是……'

　　王母娘娘实在看不下去了，大叫：

　　'王猫，还不下去！'

　　'一会叫我上来一会又叫我下去，烦不烦。'王猫嘟囔着和那个鬼一起走了。

　　王母娘娘安慰玉帝道：

　　'你别生气了，它不懂事，其实人类中好人多得很呢。'

　　玉帝笑了笑：'我当然知道了，可是好人还知道自己是好人吗？！'

　　雨还稀稀沥沥下着，整整下了九九八十一天.
