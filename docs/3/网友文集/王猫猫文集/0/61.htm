同学会

　　一个清冷的早晨，我在睡梦中接到哲二的电话：喂，周日来参加同学会。

　　'哦，好。'我正要打算挂电话。

　　哲二很失望的问我：你怎么一点儿也不惊喜？

　　然后不等我回答又说：当然了，这是你一贯的特色。

　　我哭笑不得地拿着电话听他一个人在电话的那一边自言自语。

　　'你知道我为什么激动吗？你的初恋情人，你一直单恋而没恋上，甚至人家都不知道的那个——于妍，她也来，你不想见见她么？！'

　　这小子说得比我还热烈，真不明白要来的那个是我的单恋情人还是他的！

　　哲二很响的把电话挂了。

　　而我的肩膀这时候已经被冷风吹得生疼。

　　一张圆圆的娃娃脸立在我的面前，梳着一个马尾巴，她很厉害地训斥我：

　　叫你交作业为什么不交？全组就等你了你知道吗？你拖了全组的后腿！

　　我并没有分辨，她不知道，我的作业被老师提前要去了，我没拖谁的后腿，还是前腿。

　　这个厉害的女组长是我少年时的一个梦想。

　　在那些个并不明媚的上午，我托着腮帮子在课堂上看着于妍的背影发傻。

　　她从不和邻桌的男同学说话，这让我十分欣慰。

　　但是，她也从来不跟我说话，或者说，不跟我好好说话。

　　她对我说过的仅有的几句话都是祈使句。

　　初中毕业后就再也没有见过她。

　　一晃快二十年了。

　　而她一直也不知道曾经有一个男生那样的喜欢过她，只是因为她的学习成绩特别的好。

　　周日一早我就和哲二等在说好的宾馆里了。

　　陆陆续续来了很多人，大部分人我还能叫出名字。

　　于妍那天一出场就给了我一个惊奇，她胖得我根本认不出来了，披散着头发，穿了一条红裙子，她应该知道红颜色在她这个年纪穿是需要慎重的。

　　哲二看了看我，'怎么样？还去相认吗？' 然后他又不等我回答就跑过去揪住于妍：

　　'嘿，我这儿有个人想见你。'

　　他把于妍带到我的面前，于妍叫道：王猫！我记性不错吧！

　　我笑了笑，说：真想你。

　　于妍爆发出一阵大笑，还戳了我肩膀一下：你怎么还跟原来一样啊。

　　然后她就拿着一杯冰凉的可乐消失在嘈杂的人群里。

　　整个同学会都在交谈，交换名片。

　　我听到无数个啤酒杯碰撞在一起的声音，我想我喝醉了。 天黑了。

　　宾馆里响起悠扬的小提琴声。

　　一个白衣服的女孩站在大厅的边缘静静拉着琴，独自一人沉醉在她自己的世界里。

　　于妍冲我走过来，旁边还有几个和她体态相仿的女同学。

　　'我要走了，你还没把你的电话留给我呢。'

　　我拿出一张纸条写我的电话。

　　于妍问：没有名片吗？

　　我说：没有。

　　她冲我笑了笑，接过纸条，'再见啦。'她说。

　　我突然说道：等一等。

　　于妍转过身来，那几个女同学也回过头看着我。

　　'还有一件很重要的事，'我说，'我憋了快二十年了。'

　　我很认真的看着她：'我必须告诉你，我喜欢过你。'

　　小提琴声在这个时候嘎然而止。

　　正象宾馆玻璃外的那一抹余晖一样消失殆尽了。

　　99/10/21
