向王猫猫同志学习

宁财神

　　君不见论坛之水天上来，直灌到海不复回。君不见网上红颜一大把，朝侃风花暮呲雪。怎么着您才能找着一位既细腻又睿智但是绝对不小资而且时不时她还感性一把写两首好诗的可心人儿呢？难那！做人难，做女人更难，做会写字的女人难上加难，而在网络上写字的女人怎么呲，也就是一老大难。放眼望去，满世界诚恳执着的病入膏肓、整屏幕催人泪下的文字瓦斯，看多了腻，看久了烦，都说网上女文豪一抓一把，可我怎么就老也找不着看着还算顺眼的呢？真逼急了我们可不可以就明知山有虎偏向虎山行、咱们也灌饱一肚子燕京瓶啤来上一局孤身犯险寻找网络上最后一个女文豪呢？其实没那么豪壮，我也就是想借着名人的效应蹭几道文学的饭局，说白了，就是腰里揣副牌我逮谁跟谁来。说我诌，我就诌，闲来无事我溜舌头，瞧着不错吼声好，觉着不对您就骂老喽(lou1)。瞧着脸儿熟的，顺竿儿就上，这不，整一屏幕有一半儿都被这名字给占了——王猫猫，没跑儿，就它了，让咱们这就收拾收拾开始学习。

　　学习伟大的网络文学先行者王猫猫同志。学习它在中国网络文学革命准备时期，以坚定的灌水路线，同网络上其他热衷于风花雪月的小资青年作了尖锐的斗争。它在这一场斗争中，是中国网络文学灌水派的鲜明旗帜。学习它在网络文学蒸蒸日上处处开花的新时期，领导灌水大军推翻旧思想、建立新文体的丰功伟绩。学习它在中网新空气乃至后来的网络各大论坛艰苦奋斗的时期，把小说杂文和诗融为一体的丰功伟绩。它在文学创作方面教给我们许多有益的东西。现在的网络文坛，除了一小撮反动分子以外，都是王猫猫同志革命事业的继承者。事物总是发展的。一九九九年的革命，即网络文学混脸儿熟的革命，到今天，不过是大半年，网络文坛的面目完全改变了。再过大半年，也就是二零零零年，二十一世纪，网络文坛的面目就更要巨变。

　　网坛将变得空前繁荣空前牛逼，届时不但要出现象张德培和阿加西这样的风流人物，也将出现象格拉芙这样的优秀妇女同志，依我看，也就只有王猫猫同志能当此重任。而作为王猫猫同志的忠实追随者，我们在捧猫拍猫学习猫的工作上非常不利，这使我们感到惭愧。王猫猫同志是一个谦虚的人，我看过它多篇著作，感到它有一种自发的智慧和气魄。从它对于网友的意见每贴必回、网友的来信每封必复中，知道它是很虚心的。它全心全意为了改造网坛现状而耗费了毕生的精力，真是鞠躬尽瘁死而后已。当然，象很多站在正面指导时代潮流的伟大文坛巨匠大都有它们的缺点一样，王猫猫同志也有它的缺点方面，这是要从历史条件加以说明，使人理解，不可以苛求于前人的。

　　下面，让我们先听听王猫猫同志的自述：

　　《答小无——我的光荣历史》

　　往事不要再提，人生已多风雨，纵然记忆挥不去，爱与恨都还在心底。

　　98年一个明媚的夏天，王猫还象小猴一样到处乱窜，螃蟹，新鲜的那个，跑来告诉我：来新空气啊，很好玩。 '什么空气？这个名字很好嘛。'

　　王猫进去看了一眼：

　　'花前月下'、'豆蔻年华'、'他乡明月'，恩，都不适合我啊，算了，将就去'朝花夕拾'吧。

　　那时候的朝花夕拾里除了别理我就是烫母猫，王猫衡量了一下，觉得两个人都不好惹，还是免开尊口为好，就偷偷贴了几篇很正经的文章。

　　然后呢，别理我就跟了几贴膏药在后面，让我乐不可支，也知道了朝花夕拾是转贴版，自创的文章要贴到别处去。

　　王猫睁大眼睛一看，别理我那小子在哪里都有他的影子，我本着'人敬我一尺，我敬人一丈'的原则，追着他的影子贴了几贴，眼看就到一千贴了。

　　哪里想到这就是'灌水'啊。

　　等到有一天王猫再上新空气，怎么也贴不了东西，一问，才知道炸弹那个老家伙把新空气封了，因为灌水的人太多，要整治一番。

　　整治的后果就是大家都能贴东东了，王猫还不能贴。

　　王猫于是又起了个名字'我是王猫猫'，还有一个'王狗狗'。

　　继续灌。

　　烫母猫这只多事的老猫怕我连累它，连连劝我不要灌水了，还让我给炸弹那老贼写了一封道歉信。

　　唉，王猫年轻没有经验，就真的写了一封，如今落了口实在人家手里，兀自懊悔不已。

　　再后来，当时和我并肩作战的几个灌水大王都走掉了，只有我还坚守阵地，终于功夫不负有心人，这块骨头让我啃下来了！

　　眼看着啃骨头的蚂蚁越来越多，王猫渐生归隐之心。

　　从中得见，王猫猫同志就是在如此艰难困苦的条件下开始它的灌水生涯的，面对着无良网管的封坛，面对着亲密战友的叛变，面对着网络上千千万万灌水爱好者的同时发难，王猫猫同志抱着必胜的信念，坚持下去、破釜沉舟、惊天一击，如它所诉"这块骨头让我啃下来了"，它终于取得阶段性的胜利。这不但是王猫猫的胜利，也是我们这些王猫猫爱好者的胜利，同时是网络文坛灌水派系的伟大胜利。胜利成果来之不易，光说不练的话，您根本无法了解王猫猫同志所做出的丰功伟绩，好，就让我们把话掰开了说，一篇篇拎出来，血淋淋、泼辣辣道给您听：

　　王猫猫同志的部分著作（由于历史悠远，作品繁多，我们一时无法集齐，见谅）车轮//我的哲学生涯//结局//圆明园的日子//老歌//一段日子//假象//你如何还能这样温柔//在老汤家的一个晚上//郊区夜游//猫猫轶事//高三的小夜曲//鱼我所欲也//红人和黑人//你是一个坏东西//和寂寞对话//wheredid you sleep last night//生活的变迁//梦一——一八//日子//诱惑//一江春水向东流//本我的外衣//乱猫1/2//现在就开始回忆//想象//真假王猫//地狱之门//回忆回忆再回忆//警察的哲学//红蜻蜓//道德的标准//火星叔叔马丁和王猫//买卖//画皮//船//一个虔诚朝拜者的一生//遭遇上网……

　　王猫猫同志的作品文风独特，让人分不太清楚哪篇是小说哪篇是杂文，而且其笔力苍劲，思绪流畅，常以诗入文，以文化诗，以小故事讲大心情，以世俗之眼观世事之象。它很好地把握住世俗之"象"上的一个"意"，从网络文学盛行至今，网络上很少出现过如此文章。举例说罢：

★《鱼我所欲也》，文中无一句不实，把两条小鱼的爱情生活娓娓道来，联缀这些生活细节的"实"之后，产生了一种无法描述的"意"，且见文中最后一句："我一直在等他，等窗前的家雀叫，等我身上的旧鱼鳞褪了又换了新的"，王猫猫的文章气长且绵，别人若照此写法，只有气短、滥和酸。这其中，需要有着曼妙瑰丽的想象力，和浪漫无端的诗才。写作的人多重自身，对"实相"和"幻相"有多种同情，一身而有多身的相同之情，王猫猫有的，而且不少。

★《Where Did You Sleep Last Night》，听grunge的女子，想必也不是什么善茬儿。在暗夜里倾听歌谣的愤怒者，拾起一根树枝，拼命抽打路边的小草，慨叹着"除了音乐、一切都没有"。人乐合一，全无造作之气，不同于往常的孤影自怜，有的只是秋士寥落、萧骚寂寞，不停咆哮着"my gal"的歌者，和在一旁倾听着、思考着的作者，两根线路顺势而上，在藐视月亮那一刹那，擦出一个火花。同样是长讴婉转、灵气逼人，那么我们是不是可以说，乐坛有kurt，网坛有猫猫呢？那个英年早逝的主儿，啊呸！祝我们王猫猫同志永远年轻，永远健康！！

★《我的哲学生涯》，看完这篇文章，你不得不佩服王猫猫抖包袱的本领，在网上写作的女性，还有个叫作老实巴交的，也是一个抖包袱的高手，它们两个共同的特质就是不动声色地把故事推向高潮，然后飘然而退，留着你在原地傻笑。关于老实巴交，我过两天也会好好学习它，这次就不多提了。说回到王猫猫，它的灌水功夫由来已久，基本上天天一篇，看着跟日记似的，长短不一，各有各精彩，最神的是，它每篇文章的题目都会加上俩双引号，算作"王记"？而这双引号，除了是一种个人风格外，还有一个"把故事包裹起来，让你开贴时层层打开"的意思，有点象传说中的神灯。鼠标点去，每篇都有新感觉，新惊喜。而这种惊喜，往往来自于它独特的叙事风格，王猫猫文字的真实意义在于逼近生活本身，致力于把我们带入颠簸、漂浮和悸动的心境，以及与此相关的整个情绪氛围、生活自身的节奏、色彩和旋律，可以用胡塞尔的一句话来形容它和它的文章——"走向事情本身"。我们可以从王猫猫的文字里，从它的叙事风格中，重新找回萨缪尔。贝克特的基本发现——横跨于生活形式与审美形式之间！

★《圆明园的日子》，这篇文章让我想起来早些年带着一帮人去画家村讨债的景象，不知道那天被殴打致残的身影里有没有王猫猫同志，若有，且让我先给您陪个不是。这篇文章的文体很有点象罗兰巴特的《恋人絮语》，它不是一种全面回忆的纪实体小说，它展示的是局部的、破碎的甚至是隐喻的思维方式，王猫突然消失，卫青换妻，墙上那幅熟悉、但是怎么也想不起来是谁的画儿，是非常精彩的意象。这也正是先锋小说多数在走的道路，摆脱了欧洲十八世纪末的浪漫余韵，接近二十世纪爱略特以后的距离意识。但是王猫猫的语言并不先锋，它笔法流畅，文中常有警句、断句和富有节奏感的诗句，与正文产生表面歧义的句子等等，整篇文章由这些小的片断，隐约的变化和一根也许连作者自己都看不清楚的主线贯穿着。王猫猫用自己的语言方式去完成了文本对于外在的、异质性和边缘性之物的辐射。它的文章有某种简短性的美感，人物和情节互相依附，互相利用，从而各自扩大自己的表意势力，作者偏爱那些突然显现的火花，它们和正文纠缠不休，也常常以调侃和顽皮的方式戏弄和瓦解正文的严肃性，"这幅画里狰狞的成分冲淡了色情的成分，假如那个作者是想让我们在一开始感到有些色情的话"。它的文章充满智慧，又富于情趣，既是启蒙的，也是反思的。用句爱谁谁的诗说：" 背倚天堂，正有重重美景豁于眼前……"

　　雷蒙德。威廉斯说："文化观念是针对我们共同生活的环境中一个普遍而且是主要的改变而产生的一种普遍反应"。从某种意义上说，网络文学目前对于所谓的传统文学唯一构成威胁力的就是技术层面，快速迅捷，开放性，包容性，互动性，而除此之外，它对于传统文学的权力结构几乎构不成任何挑战。网络作者以满足为目的的写作，或是情感宣泄、自娱自乐等明显特征正在逐渐消失，取而代之的是功利性写作，积极向传统媒体靠近，开设媒体专栏，写一些连自己都看不下去的文字……曾有人提出过如此的设想：网络作家在网站上开设特殊的小说站点，读者可以通过付费的方式得到进入该站的口令，而作者亦可得到赞助商的资助，这样，网络小说作者就可以通过这种手段养活自己。可是按照现在的发展趋势，这个基本上等于痴人说梦。如果说把网络作为传统文学培养地，积极向主流媒体输送养分的话，那么，网络文学的出现，本身就个悲剧，相当于实验室里束手待毙的小白鼠。在网络的时代，作为学科的文学艺术完全有可能以一种新的方式继续存在，并自觉地运用网络所形成的新文学艺术循环。而当下的文学艺术体制的权力构造中出版商和批评家们对批评话语的垄断，是否能接受直接来自网络的挑战呢？而这，是否能构造出一种新旧的截然断裂和对立呢？苦苦追寻正确答案的结果就是，我们呼唤一个领导者，一个拯救者，一个能够万花从中过、片叶不沾身的思考者。在这个意义上，现今能保持严谨的、无功利心的创作态度，保持天马行空的想象，保持旺盛的创作激情、保持一些网络作者应该具有的忧患意识的人，就是王猫猫，也只有王猫猫，它将在不久的将来，从天而降，带着满脑袋的光环和三个月不洗澡的异味，把还在继续负隅顽抗的我们，拯救出苦海，为我们创作出一个高度个人化的、真正多声部的复调空间。

　　几个人打架是斗殴，全国人打架叫革命；几个人写作叫文学创作，全国人写作那就是文学复兴，最后，伴随着慷慨激昂的义勇军进行曲，让我们振臂高呼：让我们踏着王猫猫同志的足迹走遍千山万水，道尽人世沧桑，让我们响应王猫猫同志的号召，使网络文学繁荣起来，兴盛起来，牛逼起来！！

　　伟大的革命家、思想家、哲学家、诗人、寓言家、小说家、散文家、灌水学家、美女王猫猫同志万岁……万岁……万万岁！！
