命 运

　　有一条红色的带子从远方飘过来，我伸手把它抓住，这是命运。我知道。

　　这个时候他们不停的对我说：住嘴。

　　我却忍不住不说。

　　这是命运。我知道。

　　我在命运之篮里放了一只蜡烛，它没有照亮我的道路，只是把我的篮子烧了一个大窟窿。

　　我从窟窿里掉了出来，平躺在马路的边缘，有汽车带着尖锐的呼啸声从我的耳边疾驶而过，我只有爬起来走掉。

　　最最不幸的是我撞上了一个小偷，他一眼看中了我，偷去了我身上全部的家当，我成了一个透明人，那时候，我灰心极了，满眼看过去的，都是灰色的人墙。小偷拿了我的家当并不走远，而是躲在商店的角落里检查，把我的东西一点点翻出来看，先是我的衣服，那是一件穿了很久的满是补丁的长衫。

　　'天哪，现在还有人穿这种衣服！'小偷很气愤，忍不住仰天长叹。

　　我这个透明人站在那里看他，他毫无知觉，继续拿出一只被虫子啃过的苹果，然后大声的叹了口气，惹得路人都纷纷向着他望。

　　'我真想大哭一场。'他这么说着，就把虫子从苹果里剔了出来，接着，抓住苹果在衣服上面使劲蹭了蹭，'吭哧''吭哧'的啃了起来。

　　我不由得同情起他来。老实说，这不是我的错，我不知道这一切是怎么发生的，假如事前有人告诉我，我或许会准备好一件好一点儿的衣服和一筐没有被虫子蛀过的苹果。

　　陌生的面孔上沾了一些苹果的残渣，我知道，下面的东西只能让他更加失望。

　　小偷索性靠着墙根坐在了地上，两条腿叉开，两只手胡乱的翻检着偷来的东西，嘴里时常发出些长短不一的叹气声。

　　大厅里的灯光渐渐明亮，小偷坐的地方周围已经聚集了一群观望者，他们起先在窃窃私语，后来声音终于大了起来。

　　'可怜的人。'他们这么对小偷说。

　　我的眼泪终于不争气的滚落了，慢慢滴下来，没有象先前那样滴在我的衣服上，只是在空气中划了一条不太直的线，然后消失在水泥地板上了。

　　我在惋惜，不知道是为了他，还是为了我。

　　放在最后面的是，是我的良心，小偷瞪着眼睛瞧了一会儿，就把它举起来给众人看：'喂，快帮我看看，这个是个什么东西？'

　　大家看了半天，都说不出个所以然。

　　小偷一下子惊觉：'不是个好东西。'

　　然后，他'啪'的一下把那良心扔得远远的了，'幸好我发现的早。'他嘟囔着。

　　哪里知道，那良心在附近象枚炸弹一样'轰'的炸开了，众人四散着逃开，小偷也急急忙忙收拾起他的行李，飞也似的跑掉了。

　　透明人一下子觉得没有了意思，跑到街上在原地躺倒，指望着命运之篮从这里经过时，会偶然的想起她来，把她重新拎到篮子里面去呢。
