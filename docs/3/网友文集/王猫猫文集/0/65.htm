一切从零开始

　　哲二坐在椅子里闭目养神，耳朵在听于妍他们说话。

　　烛光渐渐亮了起来，哲二睁开眼睛。

　　于妍把蜡烛推向他。

　　'干吗？'

　　'照照。'

　　'照什么？'哲二有点儿纳闷。

　　'照照你不长胡子的下巴。' 其他人都笑了起来。

　　于妍却神情倨傲的望着他，看上去甚至有些严肃的样子。

　　哲二木讷地摸摸下巴，早上刚刚才刮干净的，现在摸起来很光滑，好象真的不曾长过胡子一样。

　　他去看于妍时，于妍早已经把眼光转向别处，说起别的来了。

　　于妍的脸躲在蜡影的后面，有一点闪烁，让人琢磨不定。

　　哲二突然回过味来，于妍凭什么这么对自己这么放肆？不长胡子什么意思？明明就是说我是太监嘛。

　　哲二想要骂，可是又觉得有点儿晚，不骂又不甘心，心想：死丫头，看我怎么整你。

　　时间在这个时候突然静止。

　　所有的人象化石一样被定格在那间小小的昏黄的小屋里。

　　一百年后在博物馆里，哲二参观到一百年前的那天，他们一伙人在酒吧里聚会的场景。

　　他的脸上还残留着一些愤愤不平的神情，在烛光的映照下，竟然显得有些狰狞，像照镜子一样，哲二照到了自己当年的模样。

　　哲二扭脸去看于妍： 于妍的脸看上去却很妩媚，让他一下子就想起了她当时的表情——严肃的，不苟言笑的。 仿佛偷偷看到了别人的秘密似的，哲二觉得有点儿心虚，这个人现在已经是自己的老婆了，看看有什么关系呢？哲二这样劝慰自己。

　　于是哲二偏要目不转睛的盯着于妍，象要从里面看出什么来一样。

　　于妍从蜡像里走了出来，冷冷说道：骗子。

　　哲二吓了一跳，猛然间发现自己走神了。

　　哲二走出博物馆，阳光刺辣辣的照在脑袋顶上，哲二觉得自己的脑袋活象一只煮熟的鸡蛋。

　　街上的行人三三俩俩的走着，没有人注意到哲二。

　　只有一个卖报纸的挥舞着手里的《晨报》冲哲二说：《晨报》《晨报》，今天的《晨报》。

　　哲二不耐烦地挥挥手，突然发现报上有一个他熟悉的人影——那是一个好看的女人的身影，和'她'很像，但是肯定不是'她'。

　　'她'那个时候还要好看。

　　在哲二的记忆里，哲二已经把'她'细细的浏览过了，细腻的肌肤和柔滑的感觉，'她'还不知道？！ 这让哲二觉得不可信。

　　女人都是假正经！ 哲二深信。

　　哲二的心还停留在那个喧闹的酒吧，木质的桌椅一圈圈散开来，把他们包围，把他的心包围。

　　他沉默不语，怕一开口，那颗不听话的心从嘴里蹦出来。他用酒堵着嘴，叫自己不能说，哪里知道酒精麻痹了他的神经，他缓慢的，把一切讲给于妍听。

　　活着就是痛苦。

　　哲二总结道。哲二的生命里有两个女人，然而都是不属于他的，或者曾经部分的属于过他，在哲二的主观世界里属于过他。

　　于妍认为哲二卑鄙地占有了自己。

　　于妍从激烈的争执变得渐渐冷漠。

　　哲二认为自己喜欢的就应该能得到，通过努力，通过各种努力，哲二的字典里没有'道德'。

　　有时候也不全是痛苦。哲二又总结道。

　　假如你摸到了一张好牌，这张好牌也许就决定了你的一生。哲二觉得自己就摸到过好牌，那是很久以前了，他心花怒放地把这张牌捏了好久，舍不得打出去，生怕好运气就此跑掉了。

　　可是他拥有一切男人的缺点。

　　于妍拥有一张随时准备蔑视一切男人缺点的脸。 她总是匆匆忙忙的，匆匆忙忙的来，匆匆忙忙的去。 她象一阵风似的在屋子里一阵乱刮，然后屋子就干净了。

　　哲二瞪着眼睛，对一切视而不见。

　　千万别把女人当个事儿。哲二告诫自己，实际上他也是这么做的，这样她们才会喜欢你。

　　对着逐渐冷漠的于妍，哲二自我分析着。

　　然而于妍一去再也没有回头。

　　哲二觉得自己留着的那块白手帕上的血不仅不荣耀，反而更象一个污点，时刻提醒他，他曾经拥有的是一个空洞的谎言。

　　百年前的天空和现在的一样，灰色的。太阳已经老得不能再老了，它总是在早晨迟迟到来，下午又早早就回去休息了。 连太阳都在偷懒。

　　哲二躲开那探头探脑的太阳，跑到阴凉的地方坐着，他晒太阳晒得太久了，不由得有些憎恨。

　　那张好牌在于妍的冷笑中彻底被毁掉了。

　　'你真以为我是处女吗？'于妍望着被酒精染红了脸的哲二。

　　哲二的确不曾怀疑过。

　　于妍用大头针轻佻的把手指扎破，血滴在桌子上。

　　那时候，哲二知道，这张好牌完蛋了。

　　一切都变得毫无意义。

　　哲二费尽心机得到了于妍，得到的，都是假的。

　　这件事让哲二怀疑起自己来。

　　在享受了过程之后，突然间发现那是一个错误的目的地。

　　哲二不知道是不是应该继续享受，他并不在乎于妍是不是处女，可是问题是于妍骗了他。 在他骗了于妍之后，才知道，自己其实也被骗了。

　　这滋味还是不好受。

　　由此哲二开始怀疑其他的一些事，是不是自己是一个彻头彻尾的打着胜利微笑旗帜的失败者。

　　哲二在阴凉地儿里想得有些躁热，就起身想要回家了。

　　这个时候，哲二猛地趔趄了一下，被一块石头绊了一跤。

　　趴在地上的哲二没有急着站起来，而是任由泥土沾在脸上，身上，被蹭破了的腿上。

　　他的脑子里在想一个问题：摔倒了站起来就可以了！

　　身上的土掸掸就可以了。

　　蹭破的腿过几天就会长好。

　　　摔倒的哲二在地上认真的想明白了这个问题，他站了起来。

　　他要去追逐那个真正属于他的'她'。

　　以前的那个哲二还在地上趴着，站起来的这个人，是一个新人。

　　太阳这个时候突然探出头来，重重的叹了一口气，又躲到乌云里面去了。
