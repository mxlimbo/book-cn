征 服

　　'就这样被你征服，脱下我所有衣服。。。'

　　歌者的脚下满是鲜花，歌者走在鲜花里，'我视鲜花为粪土。'

　　哲二几乎乐得喘不过气来：太晕了，这个歌手！

　　于妍在旁边诧异地看了他一眼，继续低下头来看《心有千千结》。

　　琼瑶把这个缠绵悱恻的爱情故事交给了她，她就放不下了。

　　于妍有着一对大大的眼睛，长长的睫毛遮下来，象一幕屏风，正是'犹抱琵琶半遮面。'

　　哲二当初就是看中了这双眼睛，只是这双眼睛而已。

　　哲二突然回头对于妍说：太晚了，你该回家了。

　　又是这句。

　　于妍觉得委屈，哲二不象个男人，他从来不留自己过夜，哪怕完事后于妍累了，他也要把于妍揪起来，让她穿好衣服回家。

　　'为什么？'有一次于妍问。

　　'不为什么，我不习惯两个人睡，睡不着。'

　　'早晚都要习惯的。'于妍嘟囔着。

　　哲二摇头，'回去吧。'

　　于妍没有从哲二那里得到过琼瑶式的承诺，这让她很不安心。

　　'我们现在这样不是挺好吗。'每当于妍说到结婚的问题时，哲二都这么回答。

　　然而于妍需要婚姻做堡垒，去寻找那样一种具有普遍意义的踏实的感觉，一般女人所具有的，不分美丑，不分高低贵贱的，哲二对她说：这个世界上没有谁能给你安全感。

　　于妍放下了手里的琼瑶，她忽然悟到了一些东西，一些她早该悟到的东西。

　　她站了起来，'我回去了。' 哲二照例是不送她的。

　　我们相互交换了一些温暖。于妍想，只是现在他的温暖已经温暖不了我了。

　　我要不要再去找一些温暖？

　　我要不要把我的安全感交给别人去把握？ 快乐的按钮在别人身上。

　　于妍开始哭泣了。

　　夜里哲二破例来了一个电话：你没事吧？我看你走的时候不太高兴似的。

　　于妍说：没有，没有不高兴。

　　哲二说：那就睡吧，别瞎想，我爱你。

　　于妍答应了一声。

　　夜风把窗户吹开，冲进屋一股冷气。

　　于妍再也没有去过哲二的那件房子，仿佛达成了默契似的，哲二也再没来找过她，没有电话，没有任何消息。

　　很久以后，于妍逛街的时候碰见了哲二，哲二带着一个女孩，比于妍年轻，比于妍眼睛大，于妍愣了一下，哲二冲她笑了笑。 然后他们就错过了。

　　于妍听见那女孩问哲二：谁啊？ 哲二说：一朋友。

　　于妍就笑了。
