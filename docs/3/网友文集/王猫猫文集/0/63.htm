相 亲

　　那天哲二突然打电话来对我说：我要相亲去了。

　　那口气不像是去相亲，倒很像是出国或者分别什么的，弄得我也有点儿惶惶然了。

　　哲二以前也交过几个女朋友，但是都很短暂，交朋友的时候，我就见不到哲二了，等和女朋友吹了，他才又在我那间十二平米的小屋里出现了。

　　所以这次我也很怕他去了之后肉包子打狗，就剩我一个孤零零的没人理了。

　　好在这次时间很短，没过几天，他就出现在我昏暗的屋子里，忧心忡忡的样子。

　　'怎么样了？'我换了个舒服的姿势坐下，准备听听他相亲的故事。

　　他点了根烟，把自己笼罩在烟雾之中，慢慢讲起来。

　　下面就是哲二讲的哲二的故事。

　　哲二那天穿了一身休闲西服，看上去十分潇洒。

　　女方很痛快，都没叫介绍人出场，直接就约他在公园门口见面。

　　到了公园门口，女的已经来了，并且一眼就认出他来了，'嘿，哲二吧。' 哲二点点头。

　　女的就说：去公园转转吧。

　　然后就跑到售票处买票去了。

　　哲二顿时对女人有了一个好印象。

　　现在这样的女人可不多了，哲二当时想，本来今天哲二打算带女的去吃一顿海鲜，再打打保龄什么的，现在看来，去普通馆子吃一顿就行了。

　　女人的身世很不好，年幼丧父，她是家里的老二，上面一个姐姐，下面一个弟弟，那个时候过的很难，姐姐很早就工作了，为了供她和弟弟念书。

　　哲二听了很有同病相怜的感觉，自己要找的正是这样的一个女人啊，哲二知道只有有过这样经历的女人，才会更珍惜生活，珍惜生活中的一点一滴。

　　哲二也是一个苦孩子，父母都在农村，父亲瘫痪在床，他一个人考到北京来，已经让父母欠了一屁股债，而他现在所在的研究所又不景气，他还要供下面的弟弟妹妹上学。

　　相同的经历，让两个人的心靠得近了。

　　那天吃完饭后，两个人还没聊够，就又回到公园坐了一会儿。

　　哲二觉得很久没有这么畅快的聊过天了。

　　'你就不是一个好听众。'这时候哲二跳出来对我说。

　　总之那天哲二的感觉特别的好，他送女人回家的时候，看见女人在回头冲他招手时依依不舍的神情。

　　所以后来介绍人告诉他没戏了的时候，他就觉得特别意外。

　　介绍人说的大意是，女方说她受的苦太多了，她不能让她的下半辈子还是在这样的一个环境里，一辈子都摆脱不了，她虽然很喜欢哲二，但是为了将来着想，只好忍痛作了这个决定，希望哲二不要生气云云。

　　'这就是现实。' 哲二抬起头看着我，'我一点儿也不生她的气，相反，我希望她能找到一个真正对她好，能满足她的要求的人。'，带进来一些灰尘。

　　'我对我自己都不能负责，又怎么能为别人负责呢？'

　　哲二说完这句话就拍拍屁股走了，带着一脸的沉思。

　　我重新在椅子上坐好，望着外面的夕阳，发起呆来。

　　99/10/15下午14：46
