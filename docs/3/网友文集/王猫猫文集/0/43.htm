我的网友mikko

　　我的网友mikko是一个人见人爱的风流倜傥，英俊潇洒，聪明机智，幽默风趣的小伙。呸，我这儿不是找骂呢吗！诸位，原谅我瞎说一回，我实在是见了mikko绷不住了，大家非叫我写他，我偏不写，最后，我还是写了，这都是因为我的网友mikko，他实在是太优秀了。

　　话说那一天的晚上天气晴朗万里无云，我的好朋友mikko打来电话死活要请我吃饭，而我呢，死活都不肯去，最后mikko使出了他的美人计，我终于决定将计就计，且去看看他有什么招数。

　　在一片热闹的街头，我兴奋的见到了獐头鼠目的mikko戴着一缺角的眼镜正贼眉鼠眼的四下乱踅摸，我一个箭步窜上前去：'有花生米吗？'

　　mikko猛回头，急促的喘着粗气瞪着我：'要油炸的还是五香的？'

　　然后不等我回答，就一把握住我的手：'王猫同志，我可找到你啦。'

　　mikko把我拉到一个灯火通明的烤鸭子店，边走边说：'同志们都等着急啦。'

　　我看了看那油腻的店堂，心里很疑惑：'不是说有三文鱼吗？'

　　mikko谨慎的看了我一眼，低声道：'咱们都是贫农，不兴吃那个，到时候你可不要露出马脚啊。'

　　走进包间，果然见到了xx， xx和xx，因为不便用真名，只好暂时用xx代替。

　　当下xx们对我的到来致以了热烈的掌声。

　　席间，xx们对这家烤鸭店感到了极大的不满，它竟然没有熊掌和鱼翅，这让除了mikko外的同志们十分扫兴，并且烤鸭的小饼也太厚太硬，吃起来略有一点点象树皮，很不卫生。

　　所以同志们当场表决，由mikko同志赔偿大家的精神损失和物质损失，但是鉴于mikko同志的表现良好，可以考虑缓期执行。

　　之后，我们转移了会议地点，去了——酒吧。

　　那里人影撞撞，红男绿女游来荡去，尤其是有一个穿着短裙的美女从mikko面前款款走过，看得mikko声泪俱下，'早怎么不带我来这啊？'

　　当时我就严厉的批评了mikko的这种小资情调，并且正确指出：他的这种思想是危险的，是可耻的，如果不赶快打消，是要犯错误的，犯了错误再后悔是来不及的。所以我就奉劝mikko同志要及早的悬崖勒马，浪子回头。

　　mikko眼泪汪汪的打断了我的话：'你丫有病啊，我看姑娘犯法啊，我喜欢。'

　　mikko同志就这样在众目睽睽之下赤裸裸的暴露了自己的身份，一点儿也不顾及身边的同志们。

　　再后来，mikko和xx们都点上了烟囱，我迷茫的朝四周望去，只看到七根大烟囱，而我，则象白雪公主伫立在七个小矮人当中。

　　孤独啊，孤独感紧紧的抓住了我，我彷徨，我踌躇，我犹疑。

　　凌晨，烟囱们在我的提议下散去了。在这个晴朗的夜空，我终于看到了曙光。

　　我向mikko和 xx们挥手致意：同志们，再见了，要时刻记住，世界上还有三分之二的人民处在水深火热之中啊。

　　mikko和xx们对我的话深表赞同：'你丫吃拧了吧。'

　　最后，让我们用毛主席他老人家的一段精彩语录来作为我这篇文章的结尾吧。

　　'天苍苍，野茫茫，风吹草低见牛羊。'
