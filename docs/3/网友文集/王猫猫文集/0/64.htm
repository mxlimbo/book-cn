一个无聊的下午

　　阴霾的天空布满了乌云，一场大雨就要来了。

　　我怀揣一本诗集钻进出租车准备去看哲二。

　　哲二的家在西北三环的边缘，两间土灰砖房，他一个人住着。

　　刚一进哲二家的门，雨就淅沥哗啦下起来了。

　　'嘿，我来的真是时候。'我边在门廊下蹭鞋上的泥边说。

　　哲二把手插在裤兜里看我蹭，一付睡意朦胧的样子。

　　屋子里的光线很暗，因为下雨的缘故就更加暗，主人不开灯，我也懒得说，这暗淡的光线给屋里的家具涂上了一层深灰色，在我的感觉里变的不洁起来。

　　我落坐了很久哲二都不开口说话，仿佛他天生就不爱说话似的。

　　我咳嗽了一声，问道：有茶吗？

　　哲二如梦方醒，说道：我这就去烧水。

　　他踢踏着拖鞋出了屋，我听见外屋有接水声，开火声。。。

　　我站起来到门口看着他，'没水了？不喝了吧。'

　　'没事，很快的，你进屋坐着吧。'

　　我百无聊赖的在屋子里走来走去，目光落在书架上，书架里的新书不多，书也很杂，从技术书到小说，杂志，甚至还搀杂着几本中学课本，我看到每本书上都积了一层灰尘，就懒的去碰它们了。

　　我坐回到椅子里，眼睛四处张望着，看到在我旁边的书桌上摆着一个相框，一个廉价的木质相框，里面有两个人在笑着，笑得很开心。

　　其中的一个毫无疑问是哲二，另一个人我不认识。

　　他们俩肩膀挨着肩膀并排站在一座桥上，那个我不认识的人还把手搭在哲二的肩上。

　　不知道是照相技术的原因，还是相机本身就不好，那张照片看上去非常模糊，没有层次感，好象还有些逆光。 但是两个人是笑的非常开心的，仿佛他们看到了什么特别滑稽的东西，或者有其他什么让他们开心的事情在眼前发生了。

　　'喝茶吧。'哲二已经把茶端到了我的面前。

　　'恩，'我接过茶，问他：'照片里这人是谁啊？'

　　哲二漫不经心地瞟了一眼照片，回答说：一个不认识的人。

　　'啊？'我惊讶的叫了一声。

　　哲二看了我一眼，'奇怪是吗？其实也不奇怪，那天我心情不好，就一个人去北海玩，碰到这个人他也是一个人，就一起玩起来，还照了相。'

　　哲二说到这里，把双手枕在脑后靠在椅子背上，'现在，我连他叫什么都忘了，要是没这张照片，早想不起来了。'
