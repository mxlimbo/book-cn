戏 票

　　有时候人老想要不自觉的还原记忆，明知道那是不可能，也还是喜欢自个跟自个较劲儿。

　　我坐在哲二空洞的屋子里，想着三个月前这屋子还象个宫殿一样的辉煌，现在一切都没有了，只有哲二坐在藤椅里挖着鼻孔。

　　‘你就不想想办法么？’我问他。

　　‘想？想什么办法？’他的食指依旧在鼻孔里。

　　‘我活得好好的还要我怎么样？’ 我很诧异，‘你不想东山再起？’

　　‘哎呀。’这回，哲二把食指拿出来在椅子上蹭了蹭，又用中指和食指去揪鼻头，‘我曾经辉煌过，不是吗？那就够了。’

　　我对他这样的泰然自若的对待挫折很不服气，‘怎么？被打败了？再也不想爬起来了？！’

　　哲二根本就象没听到我的问话似的不理睬我，他尽量的把头向藤椅后面靠着，一只脚搭在暖气管上，另一只脚无目的的在地上拖来拖去，两只手平放在略微隆起的小腹上面，嘴里有调无字的哼起歌来。

　　真失败。我对着自己说。

　　我自己是那种对于曾经拥有过的东西一直耿耿于怀的人，假若失去了，就总是不自觉的检讨总结一番，看看有没有可能重新得到。

　　那个冬天狂风大作，我把好不容易弄到的戏票丢了，眼泪汪汪的搜寻着每一个角落，手被冻僵了，脸也冻的通红。 哲二拉着我说：不要找了，快走吧，戏要开演了。

　　可是没有票怎么看戏呢？ 哲二用他刚刚才抹过鼻涕的手拉起我狂奔。

　　戏院门口人山人海。

　　任哲二怎么苦苦哀求，检票员也丝毫不为所动。

　　哲二转过头来看着我，从衣兜里摸出他自己的票：你先去看吧，我再想办法。

　　‘什么办法？’我问。

　　‘你别管了，反正我有办法，你先进去，一会儿我就进去找你。’哲二推着我进了戏院。

　　‘你真有办法？’我不放心的问。

　　‘恩。’哲二用眼神示意我小点儿声。 我狠狠的瞟了一眼那个检票员。

　　直到戏快结束了，哲二才摸到我身边来。

　　‘好看吗？’他问我。

　　‘你怎么才来，马上就完了。’

　　‘我又不象你那么爱看。’哲二的食指又去找他的鼻孔了，‘好看吗？’

　　‘恩，’我点了点头。

　　‘特好看。’

　　‘那就行。’哲二停止了挖鼻孔，双手在膝盖上来回的搓。

　　‘冷吗？’我问。

　　‘有点儿。’ 我用双手帮哲二搓他的膝盖。

　　哲二一脸幸福的看着散场的人群对我说：咱们也走吧。 回去的路上，我絮絮叨叨的检讨自己丢了戏票的过错。

　　哲二不耐烦的制止住我说：你别罗嗦了好不好，知道自己拥有什么就行了，别老是想着失去了什么！

　　寒风吹歪了我的帽子，我没有伸手去扶。

　　那一年，我上初二。

　　此刻，哲二已经跑进厨房端了一碗方便面出来，‘你吃不吃？’他用脚踢了踢我的腿。

　　看着哲二一脸兴奋的样子，我一把夺过他的面碗，‘我吃这碗，你自己再去泡一碗来。’

　　你个小兔崽子。哲二扭身进了厨房，带进来一股清香四溢的味道。
