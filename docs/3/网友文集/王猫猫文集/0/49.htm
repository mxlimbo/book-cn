错 位

（一）

　　早晨哲二来找我：‘我觉得有些不对劲。’

　　他说不好到底哪里不对，‘我想我是患了妄想症了。’

　　‘出什么事了吗？’我问。 他摇了摇头，‘星期天我的女朋友来家里玩，你也一起来吧，给她壮壮胆子，我怕她会害羞。’

　　‘好的。’ 哲二不安地在椅子里扭动着身体，似乎有什么话要说。

　　我静静地等着。

　　哲二开口了：‘我的女朋友，她是一个好女孩。’

　　我笑了，‘那是当然。’

　　哲二从后门出去了，我看着他的背影逐渐消失在我的视野里。

（二）

　　这是我第三次来哲二家，不知道为什么，我觉得哲二的母亲并不怎么好客，我从没见过哲二的父亲，哲二说他早就不在了，他母亲一个人独居惯了，这也是我来得少的原因。哲二的母亲看上去要比哲二大很多，哲二是她唯一的孩子，可是我并不认为她很溺爱哲二，这是好品质，我知道很多独生子被娇惯得不成样子，他们自私，霸道，尖刻，然而哲二例外。

　　哲二把他的女朋友于妍介绍给我。 那是一个娇小，文静的女孩子，不善言语，我相信哲二的眼光，于妍会是一个好妻子。

　　席间，哲二的母亲不断地问于妍话，我看到于妍涨红着脸，一一认真地回答着。

　　哲二的母亲：你现在住在哪里啊？

　　于妍：住在公司的宿舍里。

　　哲二的母亲：你父母在北京吗？

　　于妍：不，他们都在湖南乡下。

　　哲二的母亲：家里有几个兄弟姊妹啊？

　　于妍：我是老四，下面还有一个弟弟一个妹妹。

　　哲二的母亲笑得有些高傲：‘这么多的孩子管得过来吗？’

　　于妍回答说：管不过来，爸妈常说当初少生几个就好了，生得越多，操的心越多。

　　哲二的母亲若有所思的点了点头，又对我说：王猫，你自己吃啊，都不是外人。

　　来，尝尝我做的这个甜酒酿。

　　说着，就动手给我舀，我忙接过去说：伯母我自己来吧。我不大爱吃甜食。

　　于妍忽然兴奋地说：是甜的吗？我来一碗。

　　哲二替于妍补充道：她就爱吃甜的，吃起来没命。

　　幸好我不爱吃甜的，不然非为这个争起来不可。

　　于妍不好意思的笑了。 我觉得这天的气氛真是融洽极了。

（三）

　　我所租的房子是部委的，这个月要统一装修，房主要我暂时搬出去住一两周。

　　那阵子我正在写一个长篇小说，这些琐事搅乱了我的思路。

　　我到处去找朋友借房子，我知道，两周是不可能租到房子的，酒店又太贵，只有去借。

　　哲二听说了马上打电话给我说：我有房子，是妈妈给我准备的新房，本来妈说先让于妍住在那里的，说一个女孩住宿舍不方便，现在先借你算了。

　　我说：那怎么好意思。

　　哲二说：没关系，于妍住宿舍也住惯了。只是我觉得有点儿不对劲。

　　‘怎么不对？’我问。

　　‘也没有什么。’哲二欲言又止。

　　‘到底怎么了，你这么吞吞吐吐的？’我疑惑起来。

　　‘是我妈妈。’哲二还是不肯说似的。

　　‘你妈妈怎么了？’我问，‘你一口气说完了行不行？！’

　　‘我妈妈她原来其实不赞成我谈女朋友。。。要知道，她不是我亲妈，，，这事我只跟你说，我是她领养的，我并没有什么父亲，从来就没有过。。。她一直指望我养她，不要别人，，现在她的态度忽然变了，，，还给我弄了一套新房，，，我想的太多了大概，，，最近我的精神不太好，，，其实要说这儿也没有什么，，，算了，改天再跟你说吧，下班我把新房的钥匙给你送去。’

　　哲二就这么颠三倒四地把电话挂了。

（四）

　　哲二的新房被布置得十分漂亮。

　　乳白色的绸窗帘，白色的缎子被罩，奶白色的低柜，白色的真皮沙发。。

　　‘不错啊，你这里。’我对哲二说。

　　‘恩，都是我妈弄的。’他的脸上有一些异样的光采。

　　‘有福啊。’我感叹道。

　　我平时邋遢惯了，走进这么干净的房间，几乎不敢坐下来。

　　我说：我要回去拿我自己的被子，别给你们盖脏了。

　　哲二心不在焉地说：恩，那我也走了。

　　我们一起出了门。

　　我回去从我租的房间里拿了一床床单和被子，然后又回到新房里，开始小心的铺床。

　　突然间响起开门声，我回头一看，是哲二的母亲。 她看见我，显然吓了一跳，‘你怎么在这儿？’ 我解释说因为房子在装修，暂时在这里借住几天，实在不好意思。 老人家听了似乎很不高兴，‘这屋子里耗子多，我拿了些药来。’

　　她把耗子药放到厨房碗架子的高处，‘放在这里小孩子够不着，不要当好吃的吃了才好。’

　　临走，哲二的母亲审视地看着我：你要住几天？

　　我说：大概要两个星期。

（五）

　　在哲二的新房里，我终于能够静下心来写我的长篇。

　　就在两个星期的最后一个星期六，房主如期地通知我可以搬回去住了，这时候，我的长篇还差一个结尾。

　　为了这个结尾我整整一宿没有睡好，我起身去厨房泡了一杯莲子汤，觉得实在是太苦了，我只在碗架子下面找到一罐红糖，将就着放了一点儿。然而没过多久我就觉得腹痛难忍，恍惚间上了一辆计程车，‘去海淀医院。’

　　然后我就人事不知了。

　　直到第二天一早在医院里清醒过来。

　　我突然间意识到我犯了一个错误，当哲二对我说他母亲不是亲的的时候。

　　于妍说：是甜的吗？我来一碗。

　　我说：我不爱吃甜的。

　　哲二说：我也不爱吃甜的。

　　哲二母亲说：这屋子里耗子多，我拿了些药来。

　　我拼命地从病床上爬起来，给哲二打了个电话：于妍在哪里？

　　哲二：她感冒了，刚从我这儿走，妈妈让她熬点姜糖水，大概是去新房了。

　　等到我和哲二赶到新房的时候，看见桌子上残余的半盏姜糖水，于妍静静地躺在雪白的床单上，脸扭曲得变了形。

　　我跑进厨房，看见碗架上层的装耗子药的纸包破了一个洞，而那个洞正对着放红糖的罐子，和我猜想的一样。 ‘赶快送医院。’我对几近疯狂的哲二吼道。

　　哲二哭着说：怎么会这样？怎么会这样？

（六）

　　北京的秋天最短，路上满是金黄的落叶。

　　在哲二的婚礼上，我看到于妍象一个仙女那样美丽动人，我敢说，那是我见过的最美的新娘。

　　我拉着哲二的手说：请接受我的祝福。 哲二笑了笑。 我又说：远离你的母亲，不要为此而内疚。

　　哲二的眼神暗淡下来，于妍马上注意到了，问我：你说什么？

　　我摇了摇头，走开了。

　　那次事故后，于妍就聋了。
