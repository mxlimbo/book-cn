我眼中的Sieg

　　Sieg在网上的全名是Siegfried Shiva，但是大家习惯叫他Sieg。

　　常来青青草bbs网站的人都知道，Sieg是一个哲学家，不过他自己不认同这种说法，我也不认同，有时候我想，Sieg是一个杂家，文的武的都成，同时兼备文学和科学的大脑。

　　想要注意Sieg并不难，因为他的东西已经慢慢成为了网上的一种精神，看到他的同时，你就不能鄙视网络文学了。但我并不是要说他的东西有多么好，对于一部作品，每个人有每个人的看法，他的东西对于多数人来说晦涩生硬难懂，他也不是写给大多数人看的，主要的是大多数人也不会去看。但是，这里面要不要包括我呢？

　　我就看不懂Sieg先生的东西，当然这不是我要写这篇文章为他标榜的原因。原因是什么呢？

　　也许是因为这一次的上海之行吧。

　　Sieg是一个地道的上海人，这很出乎我的意料，在我的意识里，对于上海人有着一种固执的偏见，总觉得他们狭隘，小气，肤浅，任何一种高尚或者接近高尚的思想都与他们无缘。

　　第一次见面是在上海商城的剧院里，Sieg穿着黑色的长风衣，如同佐罗旋风般出现在我眼前，他在与他那张瘦脸不相称的大眼睛里写满期待：

　　"你愿意来我家里听瓦格纳吗？"

　　"来我家。。。"

　　"。。。听瓦格纳。。。"

　　"你愿意吗？"

　　瓦格纳是谁？

　　我是在尼采的书里听到瓦格纳这个名字的，他不是这个世纪的人，他属于和尼采在一起的德意志共和国，对于他，我完全的不了解，只知道尼采已经把他屏弃了很久。

　　这个人又是谁？他为什么在这个时间，这个地点对着所有头一次见面的人发出这样的邀请？

　　我的脑袋里涨满了问号。

　　第二次见面是在一个昏暗阴郁的酒吧里，三十几个陌生或不陌生的人围坐在一长溜木质桌子旁，Sieg坐在我的左边，而他的左边是那个曾经在网上把周星驰解剖了的俞白眉先生，他正引逗Sieg说出那一大长串我从未听说过的外国人名：图尔尼埃，莫里亚诺。。。

　　而Sieg居然是好斗的，他活象一只公鸡，气急败坏的给俞白眉解释他的观点，他似乎不能明白这个世界上为什么有这么多什么都不懂的人。这一点似乎暴露了他上海人的本性，但是似乎又符合大多数艺术家的做派。

　　这个时候的我是很迷惑的，在Sieg望着远处什么都没有的某一点的清澈见底的目光里，我依稀仿佛看见了一扇大门，它若隐若现，我很想向它走去，但是只看到闪现在那门边的一圈光彩，就已经令我眩目，我又如何能走得进呢？

　　这是一个充斥着欲望的时代，只要你想，一切都不晚。没有人会封锁你，除了你自己。

　　Sieg坐在那里继续着他旁若无人的谈话，时而重重的喘一口气，手指神经质的敲着桌子的边缘，"正是因为生活在此处，所以艺术才在别处。"在他庞杂的语言里我只听见这么一句。

　　第三次是在Sieg曾经盛情邀请我们的他的家里。

　　这是一间不足十八平米的房间，而房间里却挤着十八个人。那一晚，瓦格纳的唐豪瑟在满床的烟草和棉袜的混和气味中轰然响起。狂热的Sieg在群体的茫然中败退给了中国的黄色笑话，那穿戴整齐的老头独自在电视机里自吟自唱，和所有的背景音乐一样，不具任何穿透力。

　　那一晚与其说我为瓦格纳悲哀，倒不如说我在为Sieg悲哀。因为我是一个庸俗的色情笑话大王，而在这个领域里，Sieg是个白痴。他被所有过时的笑话感染，我觉得，那一刻，也许是他要放弃艺术而选择生活的契机。

　　我带着些许的满足离开了那间小屋，留下满室的狼籍。

　　这之后我再也没有见过Sieg，也许在将来的某一天，他会突然跑到我身边，说道：

　　"我真正发现了生活，它不在此处，它和艺术一样，也在别处。"
