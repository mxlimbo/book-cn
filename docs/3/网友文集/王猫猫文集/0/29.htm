假 象

　　顺着脏兮兮的街道，我溜达到阿囡的家。

　　她没在。

　　我于是坐在门洞口等她，低着头玩自己的手指头。

　　这样过了好久，阿囡还没回来，快下班了，来来往往的人多起来，一个女人提着书包上楼了，过一会儿，提着菜篮又下楼，看了我好几眼。

　　我不高兴坐在这里给他们看，就站起来瞎转。

　　家委会门口有很多人在取牛奶，排着长队，一边聊天，我也跑过去排队，听他们聊。等到快排到我的时候，我就溜啦，又跑去队尾排着。

　　我听见他们说：阿囡前一阵子病了，和别人打架来的。

　　我不禁为阿囡担心，阿囡那么瘦小，怎么打的过呢。

　　果然，阿囡被打的很惨，因为对方联合了好几个人来打她。早知道我早就来了。

　　渐渐的，取奶的人也都走光了。

　　我又孤零零的站在那里等着。

　　一个瘦小的身影向我靠过来：喂，你在等我吗？

　　不是，我在等阿囡。

　　我就是阿囡啊。

　　不对，阿囡瘦瘦的长一张娃娃脸。

　　我不是也瘦瘦的吗，也长了一张娃娃脸。

　　我不耐烦了，"不是你！总之不是你！我等的不是你！"

　　女孩子受伤似的跑掉了。

　　我继续着我的等待。

　　99。4。15下午17：00
