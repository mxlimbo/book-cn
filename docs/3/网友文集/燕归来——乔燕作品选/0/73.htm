鼓励――育儿的良方

　　一位精通家电维修的朋友告诉我，小时候他和哥哥都喜欢拆东西。一次家里的闹钟铃不响了，他就把闹钟拆开想看看怎以回事，鼓捣了半天，正着急时母亲走了过来，他以为母亲一定会吵他，可母亲却说装不好没关系，只当做实验吧，说不定你还真能修好呢！他精神为之一振，后来不仅把闹钟 装好了，铃也响了。以后家里东本坏了他俩都自告奋勇地修，遇到难题时连吃饭也暗暗琢磨。他说他兄弟俩的手艺离不开当初母亲的鼓励。

　　朋友很幸运，他母亲懂得孩子需要鼓励的道理。对孩子来说外界的一切都是新鲜的，他们渴望尝试、渴望探索。如果他们的好奇心与探索的欲望得到保护和鼓励，他们就敢于去做未经历的事而不 怕失败，他们能从尝试的过和程中获得经验和乐趣，明白自已有什么能力、没做好的地方如何改进、哪些地方还需要学习。这些感觉对孩子意义重大，他们需要借此认识自已、巩固自信、培养对学习的兴趣和持之以恒、百折不挠的精神。

　　可惜的是，许多父母并不了解这一点，认为孩子致力这动那只是为了玩久，是淘气的表现，对孩子想要做的事横加干涉，剥夺了孩子了解自已和探索外部世界的机会。孩子缺乏探索的体验和实践的经验，不太了解自已的能力，便不容易形成自信、果敢、坚毅的性格，缺少创造性思维，遇到难题容易采取消极的应对方式。

　　儿童期是一个人性格形成的重要时期，愿为人父母者能充分运用鼓励这一育儿的良方，给孩子营造一个宽松、健康的成长空间。
