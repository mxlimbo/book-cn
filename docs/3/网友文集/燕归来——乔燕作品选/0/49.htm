听诗的时刻

　　此刻，听得见自已的呼吸、窗外远处女人呼唤孩子回家的悠长的喊声。泪水正从我脸上悄悄地滑落，他在电话的另一端耐心地等，等我的哭泣告一段落。

　　啊！朋友的一首诗！

　　他说这首诗能如此地触动你，比得到什么样的赞扬都更让他欣慰。

　　诗歌拯救人的想象！

　　当日子成为一种重复，生活在人们心头刻满伤痕；当笑容不再只是喜悦的流露；当阳光、清风、雨水不再能给我们快乐；还有什么比诗更能唤醒我们沉睡的心灵？我们眼睛被遮住，看不见树的手臂正被楼群吞噬；耳朵被塞住，听不到虫鸣正由歌唱变为控诉；我们在黑夜独行，却又将每一位擎起心灯的问路人抛在身后……。当这一切已拦住去路，还有什么比诗歌更能在瞬间撞开尘封的心门，引领我们踏上一条本能拥有，却又丢失已久的回归之路？！

　　诗歌还原我们的生活。有诗陪伴，便能在子夜看到黎明、在正午听到月亮的歌声、在小河边觉察远山的心跳、在花丛中发现蜜蜂的爱情！天空将听懂我的倾诉，风儿将了解我的企盼，雪花将回应我的呼唤，春天将在我的心里落户……。
听诗的时候，拥塞的心胸得到清理，失聪的双耳恢复听力，迷茫的眼睛重新看清方向，走了调的语言自已找回韵律。

　　听诗的时候，一股清泉悄悄润入龟裂的心田，心底好似有一根弦在微微颤动，那是不是与生俱来的生命的灵性正被轻轻敲打？是不是它们正在被诗歌浸润的土壤里自由自在地穿行？

　　听诗的时刻是宁静的时刻，我晶莹的泪便是诗歌开出的花朵！
