你从哪里来

你从哪里来 你一来
便来了春风，来了春雨
你是清晨第一滴露珠
谁听到你在草叶上轻轻地笑

你从哪里来 你一来
便有整个城市的牡丹花一齐盛开
你把这一切抛在一旁
只把她的一根手指握在手掌

墙上浮动着光亮
辟出一块洁净的领土
颤抖的心该如何把你捧起
听凭你映亮每一个长夜
你从哪里来 有谁--
认得你途经的路
谁能猜出你在孕育一个怎样的故事
一只小鸟落在窗台上
它给你的故事带来了怎样的序章

你一来 便来了春风，
来了春雨，卷走了所有的忧郁
你从哪里来啊
你嘹亮的哭声让我想起这个问题
你纯洁的笑又让我不知不觉得把它
--彻底忘记

　2000年4月23日
