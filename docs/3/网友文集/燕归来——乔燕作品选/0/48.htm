仍能唱出的最爱

　　总觉得音乐是有魔力的。记得还在荥阳车站时，每逢休息便坐着火车，晃晃悠悠在来往于洛阳到郑州的这一段铁路线上。除了偶尔坐慢车人少些，平时车内总是很拥挤。冬天，车窗紧闭，吸着各种版子的香烟的男人们和个别妖艳的经唇女子吐出的烟在车厢内弥久不散。下了车身上便带上了烟的印记，久久不去。这些味道追随着我，钻进大衣，毛衣，手套，也钻进我随身听所释放出的音乐里。

　　在火车上听音乐的感觉也象这烟所呈现出的弥久不散，不过这不散的是心底那一丝温暖的迷离。记得那时最爱听粤语歌，觉得那样的发音最能传达出心底迷惑又新鲜的情绪。反复播放的《千千阙歌》是至今仍能唱出的最爱。

　　　　　　来日纵是千千阙歌，飘于远方我路上，
　　　　　　来日纵是千千晚星亮过今晚这月亮，
　　　　　　都比不起这宵美丽，亦绝不可使我更欣赏，
　　　　　　啊......留下今晚共我唱！

　　　　　　来日纵是千千阙歌......

　　岁月悠悠，许多往事已在风中模糊了容颜，淡淡地散于水流般的琐碎日子，唯有这音乐保存了最初的感觉，点点滴滴地提醒着那曾经逝去的岁月。
