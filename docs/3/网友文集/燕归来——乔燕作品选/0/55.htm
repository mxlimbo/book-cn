夜半铃声

　　睡梦中隐约听到电话铃声，迷迷糊糊翻身时，它又响了一次。

　　这么晚了会是谁？我恍惚想起以前也有几次夜半铃响，好不容易摸索着接起放在枕头里边的电话，要找的却是陌生人，这一次别又是打错的吧！我不打算接了，可电话铃并没有停下来的意思。夜深人静，要是再不接，父母也要被吵醒了！

　　我只得与电话那头的"夜猫子"打招呼，而里面的声音却清醒而又急切："乔燕吗？是我呀！哎，我做了一件不该做的事，懊悔死了！我不能跟父母说，也不能跟同事说，只好给你打电话了！"噢，是位老朋友，他在流动单位工作，现在外地一城市，每隔一段时间会记得问候一下，五一节那天才打过一个电话。我忙问怎么了，他像一个受了委屈的孩子一五一十地把做的错事告诉了我。

　　原来，他闲来无事去打游艺机，没想到把钱全输了--"那可是半台电脑的钱呢，我怎么这么犯晕！这好象不是我该办的事呀！到现在我还没睡着，我直恨自已。"我能想象他的烦恼--他是个开朗、积极的男孩儿，没事喜欢看书、练书法、或者与同事一起出去吃点东西、逛逛街，他一时难以接受这件事中自已的行为。他的话语里透着真心的痛惜与懊悔，我安慰他："只身在外地难免会寂寞、无聊的时候，谁又没有一时糊涂过呢？！损失的钱可以再挣回来，在这儿摔了一跤以后不在这儿摔就行了，别太难过了！其实，这时你能想到给我打电话我挺高兴的！"……

　　现代生活中，我们的要求并不多，许多时候，我们只需要一双能倾听的耳朵、一颗能理解的心灵、几句温暖而真诚的劝慰。放下电话，我暗暗地想：以后的深夜再响起电话铃，我一定要及时地接听，因为此时接起的很可能是一份信任与友情。
