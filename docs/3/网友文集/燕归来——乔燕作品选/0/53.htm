痛苦中的保尔

　　保尔也有痛苦。保尔的痛苦与很多人的痛苦不一样。

　　如果你看过近期中央台播出的电视连续剧《钢铁是怎样炼成的》，你一定记得这一幕：当保尔听到医生说他的病不可能恢复时，保尔不相信也不能接受这是真的。他绝望地站在波涛起伏的海边，手枪就在保尔手中，它就指在保尔的太阳穴上。

　　谁能理解保尔的痛苦？理解他并不是因为再也不能健步如飞、再也不能欣赏大自然中一切美好的事物、再也不能仔细端详他所爱的人们的面庞而痛苦，而是因为他觉得自已不能再像同伴们那样更好地工作、更好地创造、更好地奉献，自已的生命已没有意义、已失去存在的价值？

　　保尔早已把自已的生命与人类最伟大的事业紧紧地联系在一起，一旦他认为自已不能再为此奉献什么时，"活着与死了还有什么区别"便成为一个痛苦的疑问，他内心腾起了想结束自已生命的痛苦的火焰。

　　一个生物的"人"的生命是渺小的，一个精神的"人"的生命是强大的，难忘的岁月在保尔的脑海清晰地流动，成群的海鸥在空中自由地啼鸣。乌黑的手枪一触即发，但最终，它被保尔从自已额边永远地移开。

　　英雄和我们一样珍爱对于一个人仅有一次的生命，在生与死交叉的路口，在裹挟着痛苦的洗礼中，英雄的保尔迈上了新的追寻之路。这路是恒久的，这路是永存的。

　　让我们重温：……人的一生应该这样度过……！
