你是那大狗，我是这小毛毛虫！

　　这两天一共收到了五封小朋友的来信，前天妈妈从楼下的信箱拿前三封信上来的时候，一进门就兴高采烈地说：大丰收了！

　　是啊，妈妈真了解我，这是我的丰收，孩子们的信总是那么纯真直率，他们不会说什么虚话，这在成人的世界时是难得的清新。

　　这不，一个叫祥祥的六年级的小朋友在信中附了一副画，薄薄的半透明的画纸上一只可爱的大脑袋卡通狗清晰地呈现在我眼前，那大狗向前平伸出一只手，手指的前面一只黄色的小卡通虫子微微咧着嘴乐，几簇头发向四边分出，一看就是一只活泼快乐的毛毛虫。那只狗比一个摊开的手掌还要大一点，而那个黄色的小毛毛虫只和一个大人的大拇指甲盖差不多大。

　　再看右边，一行黄色的字竖着写下来："画给您的画"，每个字都和那个小毛毛虫一样大。而右下角那三行字是蓝色的，写着："乔燕姐姐，希望你能像小狗，我能像小毛虫一样，你用你的手帮助我克服生活学习上的重重困难！"我开心极了，不禁笑出了声。

　　这就是孩子们的心，纯洁透明，没有世俗的污染。他们不会有过多的琐思，他们的情感不会被"把燕子姐姐比作狗是否好听"的思维模式阻滞。我沉浸在这纯净的快乐里，我感到此刻我的心灵是多么轻快。他们的率真、他们对我的信任和爱象雨水冲洗树叶上的灰尘一般，默默净化着我的心灵，他们用那特有的品质给我的心灵插上了一双轻盈的翅膀，带我去久别的蓝天再次飞翔！

　　我想，一定会有一天，我会来到他们面前，用我的手把他们揽入怀中，摸摸他们蕴藏着多少真爱的小脑袋，感受我的心与他们的心一起纯洁地跳动！
