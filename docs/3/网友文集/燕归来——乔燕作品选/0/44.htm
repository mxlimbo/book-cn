呼 唤

　　我在灯下翻看朋友的诗集，窗外传来一个男孩儿喊妈妈的声音。那喊声极大，用足了力气。我不由得想起小的时候，自已也常这样大声地喊。

　　我八岁到十二岁在宝丰县度过，那时住的是平房。吃过晚饭，我们小孩子们就都跑出来玩儿。先是砸沙包，后来天黑看不清沙包了，我们便就着昏黄的路灯捉迷藏。那时作业一会儿就作完了，我们的时间很充裕，拿妈妈的话讲，我们姐妹仨是可着劲儿地疯跑，总要妈妈扯着嗓子喊我们回去吃饭。我们玩兴未尽，听到妈妈的呼唤，也总是可着嗓子喊：知道了，就回来！然后过好一会儿才磨磨蹭蹭地回家。那时，我并没有注意到妈妈的喊声和自已的喊声。

　　十二岁时，我们一家又回到洛阳--有我最初的生长印迹的城市。住上了楼房，再加上年龄的缘故，我不再那么疯跑了。放了学便赶紧做功课，闲下来看看电视、翻翻课外书，或者偶尔找找附近的同学。头脑里开始有了属于这个年龄的小秘密，一个野丫头变成了一个文静的少女。不知从哪天起，嗓音在悄悄变化，有时忘了带什么东西在楼下喊五楼家里的妈妈，我也不再扯着嗓子--既使扯着嗓子也是又尖又细，那很容易引来几个不同楼层的女人同时推开窗子。

　　就这样，童年那种无拘无束、肆无忌惮的呼喊渐渐飘远了。

　　窗外，男孩仍在喊他的妈妈。听他声音，应该还不满十二岁。也只有不满十二岁的声音才具有这样的穿透力，才会这样地什么都不必顾忌。

　　我不知道这个男孩儿喊妈妈做什么，是要去上晚自习而忘了带东西？还是因为想去同学家玩一会儿而向妈妈请示？抑或是不知道妈妈在谁家串门，而自已又忘了带钥匙。但有一点可以肯定的是，他不知道自已的喊声在我听来是多么动人；他不会去注意这样喊妈妈时，心中的颤动；他不会想到这样的喊声会一年比一年少、多年之后这样的喊声会离他而去；他更不会想到当他不再拥有这样的喊声时，他很可能像我此刻这样被另一个孩子的喊声吸引。

　　——这是多么有意思的事。远处，一个陌生的男孩儿喊妈妈的声音让我跌入时空的隧道。此刻，似乎伸手便可触到童年，而这种贴近的感觉和意识分明又在同时告诉我：我与这一切是多么遥远。

　　男孩儿的喊声仍在继续，这每一声都像喊在我的心上。怎么他的妈妈还不答应？她不在家吗？他做错事惹妈妈生气，正被她用沉默惩罚？我忍不住替这个男孩儿着急。我能感到他已把音量开到极限，而这种因到了极限而略带沙哑的质地是多么地真实、迷人。我在这样的喊声中下潜，我知道这样的下潜不会有底。
下潜中，被时光封锁的门正悄悄打开，我丝毫没有去想这喊声会从哪一声开始结束，虽然这喊声是必然要结束的。

　　刚才那一声呼唤之后，我再没有迎来他的下一声呼唤，而我的身体持续下落，我在这下落中失去重量。

　　我在时空隧道中穿行，能感到身体被打碎的痛快，我隐约觉得正被一种光吸收，仿佛在无限地缩小，又好像在无限地膨胀……。

　　一个普通的孩子的喊声把我带进了一种并不普通的升腾。

　　今晚，我能否找到一条通路，把一个感恩的微笑留在他无邪的枕边？我能否摘到一把又一把的星星，放在所有像他一样的孩子们的手心，晶莹他们纯洁的梦？

　　而此刻，我总觉得应该感谢谁。
　
　　——可我应该感谢谁呢？
