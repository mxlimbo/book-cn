这是谁家的猫

　　周六，《曲苑杂坛》为观众奉献了一场外国马戏团的精彩表演。

　　高高的圆形铁丝网内，一位漂亮的俄罗斯女子一会儿抱抱狮子的头，一会儿亲亲老虎的嘴，有一次甚至扳开老虎的嘴，让半个头地老虎嘴里停留了片刻。她脸上是甜美的笑容、身后是飘飘的长发，手中的小棒在半空中划出一道道优美的弧，狮子、老虎们便心领神会--它们一会儿一个挨一个放倒自已威武的身躯，躺成一排；一会儿小心地把四只健壮的脚放在两根木头上，并且优雅地走过去；一会儿又跃过一排排将近一米高的架子，架子上熊熊的火焰更增添了场面的壮观……。

　　狮子与老虎们那么听话，那么乖顺，我真有点怀疑：这狮子还是狮子老虎还是老虎吗？我恍若梦中，脑海里闪着在《动物世界》、《人与自然》等节目中看到过的情景--孤傲的狮子在原野上奔跑，身躯在半空中自由地舒展；色彩斑斓的老虎在山泉边垂下头颅，水中映出它美丽的倒影--心中不禁默默感叹人的无所不能。

　　节目演到一半，有人敲门，是楼下的一位老奶奶来借钳子。母亲去拿钳子的时候，老奶奶瞟了一眼电视里正被那女子搂着的老虎，惊奇地说："哟，这是谁家的猫？长这么大个儿！看人家这猫养的！"

　　我哑然，一种悲壮感油然而生。
