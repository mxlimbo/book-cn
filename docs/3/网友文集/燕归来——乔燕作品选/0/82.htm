关于"论河南人"专题的说明

　　"论河南人"是《云遮月期刊》的时事专题，属于期刊的机动栏目，应时而生。

　　专题收录了云遮月网友在中青在线的"青年话题"论坛关于"河南人"问题所进行的辩论发言，以及那里的网友的相关发言。记录了此次辩论的发生发展过程中起过衔接作用的关键发言与回答，比如辩论如何开端，由一个主要问题引发出两个以上问题的原因以及此后的各方观点。

　　私下里和本地朋友谈起过就此事的辩论，有人认为这样的辩论没有结果因而没有什么意义。的确，为没有结果的事情而忙碌是有些傻，但辩论之于其它谈话的特殊性在于：于辩论的过程中寻找对方的表达内容与表达方式上的问题、进而辩驳对方观点阐述已方观点，并在这一过程中体会发现与表达的意义。这也是历来的辩论赛的赛题均选那些尚处于争议中而无定论的话题的重要原因。

　　既然是这样，那么记录这一辩论的过程，整理双方的主要观点与表达，将辩论双方的观点与论据以及此次辩论的发展与延伸客观地展现便是一种有价值的记录。

　　而身为河南人的辩手，与参与讨论的朋友们一起冷静地反观自身，了解他人，深入沟通，共同完善，已有意无意地成为一个人不间断地成长中客观存在的一环。

　　在这场辩论中，许多朋友表现出良好的语言组织与表达能力，也有些朋友失去了平衡健康的心态，发出了不理智的声音——这一现象在论辩双方均有存在，而这种心态下所做的发言当然是不在《云遮月期刊》收录之列的。

　　这些文章以时间与事件的发生发展顺序为线整理，结构完整的作为文章收录，内容及字数较少的作为帖子收录，附带收录与此有间接关系的趣文，如何大拿网友所编的网友辞典一类。
