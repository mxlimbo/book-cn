花与花瓶

　　这是个未婚男子，他真心爱着的女友突然跟一位比他更有钱的男人走了，一同不见了的除了熟悉的身影，还有他放在柜子里的几千元钱。他很痛苦，他告诉我他实在想不通为什么会是这样。我很理解他的心情，此刻，令他痛苦的已不仅仅是不能再拥有所爱的人，更重要的是他正被他所爱的人用最意想不到的方式否定--否定他的爱，否定他当初的选择。

　　当爱情受挫，女人与男人都会痛苦，但这两种痛苦却很不相同。失恋的女人为男人的变心而哭泣、伤心，她们的注意力往往集中于感受自已的哀伤与孤寂；而对失恋的男人来说，他们痛苦的核心在于更容易陷入"为什么会是这样"的追问，由于这种追问不会有令他们满意的答案，他们必然难以摆脱痛苦，而把怀疑与困惑指向自身，使一个人精神结构中最重要的成份--自尊与自信遭到沉重的打击。也正因为这种不同，女人的伤口容易随着时间与新的爱情的产生而愈合；而被那种追问折磨过的男人，既使在多年以后，或者已经投入了新的爱情，想起往事时，心的伤口仍难免隐隐作痛。

　　如今，这位朋友就面临这个问题，我在想如何把他的思路从那种无谓的追问中引开。

　　我问：假若这件事发生在结婚之后会怎么样？他毫不犹豫地说那样他会更痛苦。我接着问：一个已经成年了的人会仅仅因为结没结婚而前后判若两人吗？他说不会，因为一个人的性情、品质、习惯一旦形成便很难改变。说这话时，他的语气沉静多了。

　　我说：如果婚姻是一束鲜花，人们都希望自已的鲜花能安全地走完它生命旅程。你的婚姻之花尚未开放，而你总要结婚的，恋爱的过程就是在为自已未来的婚姻之花挑选一只合适的花瓶，不同的人可能选择也不同。他若有所悟地说：是啊，有的人选择好看的，有的人选择坚固的，我属于后者，因为我希望我的鲜花能开到生命的终点。

　　"那你的花瓶怎么样呢？"我这样问时，他似乎已经意识到了这个问题，沉默了一会儿，说：现在想来，它好像并不适合我，尽管我那么喜欢它，而且已经爱了那么久；她现在离开总比成为我的妻子后离开更好，这么一想，我感到心里好受多了。

　　我也感到很欣慰，因为这表明他正从"为什么会是这样"的追问转向一种思考--她是否适合自已、能否与自已共塑理想的婚姻。

　　如果一只花瓶有裂纹，它就会不断渗水。假若我们知道它有裂纹，就一定会换掉这只花瓶。可怕的是，花瓶有裂纹我们却不知道，还满怀欣喜地把鲜花郑重地插入其中，等到有一天，水已经渗完，我们才发现这一切，再去换瓶、添水，可能鲜花早已因缺水而夭折。

　　婚姻的鲜花需要一只坚固而且没有裂纹的花瓶，如果我们选的花瓶是有裂纹的，那么在插入婚姻之花之前发现裂纹是多么幸运，因为我们还来得及换掉这只花瓶，不管它有多美--是如玻璃般晶莹剔透、还是如瓷瓶般华美润泽--这与婚姻之花的生命相比，都太不重要了。

　　我相信他不会再沉浸于失去这只花瓶的痛苦，因为他已经明白这只花瓶无法保养他珍爱的花，他会继续寻找，他一定会如愿--找到一只更好的花瓶，呵护他的婚姻之花开到终点！
