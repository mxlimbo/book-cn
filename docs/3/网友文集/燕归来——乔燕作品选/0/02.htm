作者简介

　　乔燕：（网名）阿妍

　　河南洛阳人，生于1973年秋。

　　洛阳铁路分局洛阳北站职工。自1993年养病以来，走过曲折的心理路程，乐观坚强，学习不辍。

　　1996年，北京师范大学继续教育学院中文系毕业，1998年，四川师范大学与四川广汉速记秘书学院心理学函授毕业。并于1998年始主持心理热线至今。

　　2000年夏上网，建议和管理"云遮月"论坛，编辑《云遮月》电子期刊，与《云遮月》时事增刊——《悼念亡灵》与《论河南人》。现又开设"心灵氧吧论坛"，关注心理健康。

　　由于电子书篇幅原因，本书未收录我的大量儿童文学作品，以后将择机再行出版。
