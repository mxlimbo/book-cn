别做这样的父母

　　许多父母对学习的理解很狭隘，认为只有与课程有关、为了提高分数而进行的学习才叫学习，其它的活动都属于学习之外的事。

　　记者曾做过一个实验，给一个家庭300元钱，孩子可以选自已喜欢的东西，而购买的决定权在妈妈手中。一个男孩儿每次挑选玩具时，都向妈妈说明选择该玩具的理由，如：变形金钢可组合出各种不同的形体、这支枪能发出多种声音……可是妈妈一次次生硬地拒绝了他，理由是"这跟学习无关，现在以学习为主，以后再买！"。为了使妈妈同意买一个玩具，男孩儿不断降低着自已的要求，后来他选了一种更便宜的拼装性玩具，简直是低声下气地说：这个是拼的，能开发智力，妈妈给我买一个吧！可妈妈还是说等他考双百再买。最后，小男孩得到了妈妈为他选的书包、文具盒和一套高档精美的文具套装，可他却是紧绷着脸离开商场的。

　　我不是要鼓励家长多给孩子买玩具、少买学习用品，而是觉得这位母亲的观念有问题。她认为玩具只是玩的，对学习--当然是她所谓的学习--没有什么帮助。其实不然。玩儿是孩子的天性，他们在玩儿中能获得许多书本上学不到的东西。比如：把变形金钢组合出各种形体、用拼装性玩具造出自已的建筑等活动不仅锻炼了动手能力，而且是开发智力与创造力、调动想象力与空间感、进一步发展大脑感觉统合能力的好机会。这些能力能帮助孩子更好地理解与吸收在课堂上所学的知识。摆弄玩具时，一次次失败与创新也是在锻炼孩子的耐心与毅力，为他克服学习与生活中的困难积累经验，只可惜，这些学习的机会因为带着"玩儿"的色彩而被他妈妈弃置一旁。

　　母亲一次次拒绝了孩子，并说要买也要等到"考双百"。这时，买玩具成了一种交换、"好好学习"则成了获得玩具的代价。孩子会由此感觉到父母只关心他的学习、自已愉快与否根本是不重要的，这会伤害孩子对父母的感情；而得不到玩具的沮丧、万一考不到双百便会使玩具更加遥遥无期的烦闷更容易使他生出对学习的厌倦。

　　不知这位母亲是否会注意到孩子的失落，这样的情况在生活中是多么常见。在此，我想提醒每一位为人父母者：在与孩子起了冲突或者感到孩子厌倦学习时请先咽下怒火、放下举起了的手，反省一下自已的观念与行为是否存在类似的问题！不要再做这样的父母--当孩子烦恼时自已还感到委屈，请让孩子感觉到你在平等面对他--尽管钱是你挣的、他的个子还不如你高！
