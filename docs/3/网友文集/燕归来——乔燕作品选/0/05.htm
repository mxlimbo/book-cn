云遮月


　　月有几年没出门了，她真说不清。

　　几个要好的朋友曾问过月，每天在家呆着，你急不急？

　　月总是笑笑，习惯了就不急了。

　　真的不急？

　　真的！

　　别是不想告诉我们，不想让我们为你担心吧？！

　　不是的，每天看看书、听听音乐、写点儿喜欢的东西，时间过得蛮快的。

　　那就好，我们知道你挺坚强的，可我们不希望你太压抑自已！

　　不会的，那次我心里莫明其妙地难受，给红打电话，电话一通我就哭了，看，没压抑自已吧！

　　是，没压抑。这样就好，想哭就哭。

　　现在月的朋友们基本不再问这个问题了，他们都知道月挺好的，很乐观，也很充实，不然月就不是月了，他们也就不会象现在这样喜欢做她的朋友。

　　那些还在问月你每天在家急不急的人不了解月。

　　月真的不急，急又有什么用？她还常感叹时间不够用呢。

　　月也不常哭了，最初的一年多也就是二十岁到二十一岁，月已经哭够了。

　　现在她只想笑。

　　微笑。

　　真心地微笑。

　　她相信真心的微笑能给她新的生活。

　　朋友还问过另一个问题，月，你想不想出去呢？

　　想。怎么会不想呢？

　　月不知道这些年自已居住的这个城市有了多大的变化，她不急于知道这个。
　　她想出去是有别的愿望。

　　好几次下雨的时候，月跟朋友们诉说，好想在雨中走走，以前就喜欢下雨天打伞散步，雨特小时还喜欢淋雨。朋友说，月你真有诗意。

　　月笑着问，是吗？

　　其实月的诗意很谈也很自然。月写诗但并不奢望成为诗人。

　　从朋友那儿，月听说过一个新建的公园，草坪开阔。许多人爱去，尤其是在中秋的傍晚。

　　朋友说下个中秋带你去吧！

　　月想自已以后一定会去那儿的草坪坐坐。不，现在提倡环保，坐在草坪与草坪间的走道上也好。

　　月想起自已十八岁时有一张照片，穿了件很素的裙子，白底儿上散落着极淡的小点点。就是坐在草地上照的。可惜不知丢在谁那儿，或许早就没有了。

　　早就没有了的不只是照片，还有月一次次的纯真的情感。

　　月也觉得很有意思，人们说中国的足球是屡战屡败，而自已的情感是屡走屡来。它们有着同样的特点，那就是月从没有刻意要求过，它们都是自已来的，而且来势汹涌。只是，时间证明它们来是为了成为下一次走。

　　只不过，它们不是有意这样，它们走时往往也带着或多或少的伤痕。

　　月淡淡地笑笑，对自已说，怪谁呢？都怪你自已还相信它，还对它那么认真。

　　这些话月也对朋友讲过，常常附带一句，其实我知道我没有爱的资格。朋友们极不爱听，每个人都激动而又认真地纠正月说你不能这么说，你有爱的资格！他们说这话时甚至有点儿义愤填膺。

　　月这时总是感到很温暖。她说，哦我想说的是我现在没有爱的能力，还把握不好自已的爱情。

　　朋友们知道月指的是客观原因。月每次心里受伤都是因为客观原因。

　　月渐渐明白了自已身边是可以有一个人的，只是客观原因对这个人要求很高。

　　月始终是相信以后能自已出去的，这已经作为信念在心里扎根。而这个人也是应该相信这一点并且现在就相信，只是，他还要知道月可能永远都不能自已出去可能永远都像现在这样并且在知道之后心里仍然不沉重不压抑不失望。

　　也就是说，他不应该是只带着月一定能好的企盼才来到她身边的，他应该有力量把月的现在当作最好状态全盘接受下来。

　　就像那天月问水我要是以后好不了怎么办时水回答的，好不了就好不了呗，你还是你我还是我。

　　水的话让月感到水确实还是水。其实月早就想过并且放开了这个问题，她问水这个问题不是因为害怕，而是因为水一直在等月的回答。

　　水是外省人，在与月相邻的城市工作。他是到这个城市时偶然认识月的。

　　非常偶然。

　　每隔一段时间水都给月打电话，问月你这次能给我一个让我高兴的答复吗。

　　水不小了，月不想他被自已耽误。

　　于是月总回答没有让你高兴的答复，你在你们那找一个好的吧！这时水总说
　　我不跟你说这个了并把话题岔开。到了下次电话他再问，就像以前没听到过拒绝似的。

　　月于是觉得水这人挺有意思，他是个不会让自已烦恼的人。

　　就这样月坚持不懈地拒绝了一年，水坚持不懈地问了一年。月心里想水一定是相信总有一天他能打动我。

　　水的电话总是不急不躁。今年春节万家团聚的日子，水语气深长地说他可以把房子买在月所在的城市并请个保姆，而他一有空就会回到月身边。就是从那一天开始月觉得自已在电话里对水说话的语调柔和多了。

　　月问了一个她以前从没问过的问题。水，假如，我是说假如我们一起过后半生，你期望我以什么样的状态出现在你面前、你对我有什么要求？

　　水认真地说，我只希望你每天高高兴兴的，做自已想做的事，当我这一段工作很累，你能问候我说你这几天辛苦了，要保重身体，我就心满意足了。

　　月忽然有了一种特别的感觉，和以前别人给她的感觉有些不同。

　　月想起水说过好多次。月，谁说你不能出去，我可以推你出去，只需要有个轮椅就行。月说楼太高。水说你那点体重算什么呀，只要你同意，不需要你自已上下楼。

　　月这时心里就热乎乎的。别人也说过类似的话，可和水说的感觉不一样。

　　月心里乱了。

　　月想，坏了，以前心里不乱的。

　　月知道自已心里一乱就要坏事。

　　月在电话里再也无法像以前那样平平静静地就说出拒绝的话了。

　　月不知道怎么办好了。

　　月以前是知道该怎么办的。

　　月这会儿独自微笑，她在心里和自已说话，月有这个本事，因为她总是安安静静地和自已在一起。她想起家人说过这个小区旁边有个小花园。搬家过来时，
　　月坐在车上见过，那景色印在月脑子里，只要一去想，就好象在眼前了。

　　水带月去的就是那儿。

　　水抱月下的楼，他喜欢月像个小孩儿似的兴奋地笑。

　　水让月坐在花坛的水泥櫈上，自已坐她旁边。

　　花与树在月光里静默，户外的空气把他们包围。

　　月说这儿真美，我好久没这样开心了，真想呆在这儿不走。

　　水说那我以后还带你来。

　　月亮皎洁，圆圆的。

　　水说月你就象这月亮。

　　　　是吗？
　
　　　　是的！

　　一片云缓缓移动，把月亮遮了一点。

　　月说可是月亮也有缺的时候。

　　水说缺了还会圆，缺圆圆缺，都还是这一轮月亮。

　　云彩把月亮全遮住了。周围花树的轮廓暗了一点儿。

　　水说月你看你就像这月亮被云彩遮住了。只是遮住了，云彩会走的，你怎么样都还是你。

　　月静静地听着，抬头看夜空。

　　水让月靠在自已肩膀上，轻轻握起月的手。月，我就是那个能等到遮月的云彩飘走的人。

　　月看着重新明亮起来的月亮，轻轻说，明天我们还来看月亮！
