第五十部

　　三月十二日。礼拜一。二十一点十一分，Carlton，晴朗。

纤云弄巧，
飞星传恨，
银汉迢迢暗度，
金风玉露一相逢，
便胜却、人间无数

——鹊桥仙，秦观。

　　我并没有想到过居然会来两个人，所以只订了一间标准间。
　　Linda陪着月亮和我到了房间，然后跟月亮用手语交谈了一会。
　　"我想，我该走了，这间屋子对于三个人显然太挤了，何况，我还得为我今晚的住处忙活呢。"说完，Linda冲月亮揶揄地眨了眨眼睛。

　　月亮好象有些害羞。

　　从头到尾我一句话也没有说。

　　我只是看着月亮。

　　怎么可能看得够啊！

　　我看得竟有些呆了，以致于没有注意到Linda的离开。

　　门，轻轻地开了，又轻轻的闭了。
　　掩去了红尘千尺，引来了心香万瓣。

　　房间里静悄悄的，月亮和我的视线纠结作一处，视线里面有风，有雨，有雷，有电，有山，有海，有乱花狂絮，有晨露晓月，有整整的一个世界。

　　两个人的世界。

　　不知什么时候月亮靠到了我的怀中，仰了脸儿看我。

　　我也看月亮，近在眼前的月亮。

　　那眉，那眼。
　　远山黛，水波横。迷离如月。
　　那唇。
　　落花满天，碎红无数。温润若玉。
　　月色撩人。

　　已等了亿万年了。

　　为那一吻。

　　我们的初吻。

　　原来吻一个人是这样的滋味。

　　如新酿酒般甘甜，如醉茉莉般芬芳，如五月风般慵懒，如三春雨般轻柔。
　　仿佛前生相识，今生再见，赴千年前那个约，解万载前那个结。
　　冤家！
　　便从此醉，自向来痴。
　　醉便如何？痴又何妨？愿就死了在这月色之中，再不肯重堕轮回。

　　齿边的唇愈来愈热，口中的舌愈来愈甜。

　　衣物如化蝶的蛹般褪落，赤裸的肢体偎依着，交叉错杂，绞曲缠绕，如常青藤对榉树之爱。

　　远古的封印随风飘逝，地心的熔浆破岩而出。
　　有火磅礴。
　　九天十地，皆尽通赤。
　　知火明，知火灭，知火浓淡，知火遍身。
　　心受喜，心受乐，何必求解脱！
　　这地之上，这天之下，便只一个女人，便只一个男人，便两个赤裸裸的灵魂相拥。
　　金风玉露一相逢，
　　便胜却、人间无数。
　　浑忘今夕何年，今宵何处。
