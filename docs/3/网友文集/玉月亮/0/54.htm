第五十四部

　　"为什么你让Linda冒充月亮？"

　　"因为，我想晓得你是不是知道——我——是——谁。"

　　"恩？"

　　"还记得你给我讲过白朗宁夫人的故事吗？"

　　"记得。"

　　"我从来没有看过她的诗，听你讲的故事以后，我就去买了一本她的书。"

　　"恩。"

　　"我看了她的诗。其中有一首，永远会记在我心里头。"

　　"哪一首？"

　　"恩，是那一首，里面写了有这样的话，"如果你一心要爱我，那就别为了什么，请只是为了爱才爱我。

　　别这么讲：'我爱她，为了她的一笑，她的模样，她那柔语的声气；为了她这感触正好合我的心意，那天里，的确给我带来满怀的喜悦和舒畅。'亲爱的，这些好处都不能持常，会因你而变，而这样唱出的爱曲也将这样哑寂。

　　别爱我只因为你又怜又惜地给我揩干了泪腮，一个人会忘了哭泣，久受你温柔的慰安——却因此失了你的爱。

　　爱我，请只是为了那爱的意念，那你就能继续地爱，爱我如深海。"" "我也记得。那恰好是十四行诗集的第十四首。"

　　"看了以后，月亮就问自己，那个叫瑜的男孩是不是也这样的爱月亮呢？"

　　"是的。"

　　"你知道吗？这世界上，也许会有三千个人听到过月亮这个名字，大概又会有一千个人见到过月亮漂亮的脸，其中可能有五百个人知道月亮的家在哪里，也许还会有三百个人知道月亮的电话号码——那里面，说不定还会有一百个人对月亮说，我爱你。"

　　"恩？"

　　"可这世上从来就没一个人知道 月亮 是 谁！"

　　"恩。"

　　"假使有一天月亮消散在空中了，而另外的一个女孩取了月亮这个名字，住在月亮的家，用着月亮的电话，不会有人发觉有什么不同的。太阳照样升起，人们照样奔忙，那一百个人仍然会对那个女孩说，我爱你。没有人知道，月亮其实已经死了。因为，其实根本就没有人 看 到 过 月 亮。"

　　"恩。"

　　"所以，月亮想，如果真的会有那一个人的话，他该知道月亮是谁的，他该看得到月亮的。他爱月亮，只是因为他爱月亮。"

　　"恩。"

　　"你说你爱月亮，我却不知道你爱的是不是月亮。可我必须得知道！"

　　"你是对的。"

　　"可我很害怕。怕你把Linda当作了我。"

　　"就差一点点。"

　　"你怎么发现Linda不是我的？"

　　"我——不——知——道，可——我——知——道。"

　　"恩。我明白了。"

　　"如果我把Linda当作你的话，你还会见我吗？"
