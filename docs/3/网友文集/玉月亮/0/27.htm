第二十七部

　　十一月十一日。礼拜一。八点零六分，YAHOO聊天室。微云。

哦，上帝，让我看见——哪怕是一刻也好——我们深爱着的灵魂，让他们披露，他们究竟属于哪一类，住在什么地方。

——丁尼生《毛黛》（1855）

　　"瑜！给你一个很大很烫的吻，恩恩。"

　　"恩，一个不够，还要。"

　　"恩，给你一千六百个吻，从你的耳朵一直亲到脚，有一个地方我会亲的特别多。"

　　"哇，你都快把我点着了。"

　　"你想要我吗？"

　　"连脚趾头都在想，我简直能把你吃下去，骨头都不放过。"

　　"来吃我呀。"

　　"今天的晚饭，我是在一个中国餐馆吃的。"

　　"味道怎么样？"

　　"忘了。我坐在那儿看里面的中国人，中国男人。我在想你。"

　　"恩。可惜我这儿没有夏威夷餐馆。"

　　"我在想，也许我应该试试跟一个中国男人做爱。"

　　"这可不是个好主意。"

　　"你会妒嫉吗？"

　　"我会妒嫉得发疯的。"

　　"你有什么疯可发呢？你有你的生活，我有我的生活。虽然我很喜欢你，你也喜欢我，我仍然只不过是你在网上的女朋友罢了，我在现实中跟别的男人睡觉，又有什么关系呢——你总不会疯到要我一辈子不跟男人睡觉罢？"

　　"我——不谈这个了好吗？我不喜欢这个话题。"

　　"为什么不喜欢呢？你不是很喜欢性吗？"

　　"那不一样。"

　　"有什么不一样呢？你以前不也经常和你不爱甚或是不喜欢的女人上床吗？你不是也一样能从中得到乐趣吗？你不会告诉我说，就因为有了我，你就永远再不干女人了罢！"

　　"住嘴！"

　　我知道月亮说的每一个字都是对的，所有的这些，我自己其实也知道得清清楚楚，只是我不能去想，不愿去想。我只希望现在的这个样子，能多拖一天也是好的。可现在，我无法再拖了。

　　我知道月亮在逼我，在逼我说一句话。

　　我可以说这句话，因为现在我终于知道了，那不是一句假话。

　　可说了又如何呢！说了就可以改变那些或迟或早要来临的事吗？说了就可以把太平洋和大西洋的水都烧干吗？说了就可以让玉月亮的耳朵听到我的情话吗？当一个人找不到道理来支持自己的时候，这个人就只有咆哮了。

　　这是我和月亮第一次争吵。

　　我们都说了许多可以扎进对方骨头里面的话，因为我们彼此已经十分了解了。

　　"我不会再跟你聊天了。天天关着办公室的门说三个钟头的废话，我简直是有病。"

　　"你以为我喜欢熬夜啊？我已经三个月没有在晚上睡过觉了——就在这听你满口胡言。是该结束了。明天我就会找一个男人，跟他上床，然后好好的睡一晚上觉。"

　　"用不着等到明天，现在就去找吧！你很性感，就算是在半夜也找得到男人的，你想要怎么干就能怎么干。再见！祝你玩得愉快！"

　　"我会的。再见！"

　　第二天的早晨，我们还是在网上见面了。比往常的时间要早一个钟头。

　　然后我们互相说了好多对不起的话，月亮哭了。

　　于是，一切又回复正常了，所有的事就好象没有发生似的，我们跟以前一样的谈话，调情，一切又跟以前一样了。

　　月亮有一次问我，"你爱我吗？"

　　我沉默了好久，回答，"不。"

　　我知道我们之间发生的那次争吵，不过是海啸来临前的暴雨罢了。

　　要想在真正的海啸来时活下来，还是不要说爱的好。

　　月亮也沉默了好久，说，"这样也好。"
