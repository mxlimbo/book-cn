第三十二部

　　一月三日。礼拜五。二十点二十八分，尼斯。很冷。

现在他能了解山灵西列诺斯的智慧了，他满怀厌世的情绪。

――尼采，悲剧的诞生。

　　我醒来的时候，太阳已经落山了。房间里面黑得象地狱，我发现自己躺在一堆空酒瓶和许多焦黑色或黄色的烟头的中间，我的身上沾满了恶心的呕吐物，黏呼呼的。我头痛欲裂，我挣扎着爬起身来，试图打开阳台的门，第三次才打开。

　　天空，一轮如玉般皎洁的月亮照在当空，是圆的。我死盯着月亮，我催眠般的一遍遍告诉自己，月亮是黑的，月亮是污浊的，我所看见的，只是幻象而已。可是月亮顽固地发着光，我没有办法改变她的颜色。我没有发现我的指甲已经刺入了我的掌心。

　　我回到房间，打开了通讯簿。已经很久没有打开过了，上面甚至积了一层薄薄的灰。我翻了几页，拨了一个号码。

　　"Isa？"

　　"是我。你是？"

　　"我是瑜。"

　　"哇！是你啊！我还以为你死了呢，怎么半年都没给我打电话？"

　　"对不起，我一直很忙，又常出差，在这里的时间不多。你呢？一直还好吗？"

　　"还是那个样子，上学，或者跟讨厌的朋友出去喝酒，无聊死了。"

　　"今天晚上有安排吗？"

　　"没有什么特别的。"

　　"可以请你喝杯咖啡吗？"

　　"当然可以。去哪里？"

　　"尼斯花市。Diable(邪恶)。"

　　"好啊。十点钟Diable门口见？"

　　"十点钟Diable门口见。"
