第四十六部

　　"我——我也不知道。"

　　"不对，一定有原因的——你不是那种女人。"

　　月亮沉默了一会。

　　"你太了解我了。"

　　"远远不够——如果你根本没去机场，也许我还能猜得出是为什么，可我就算想破了脑袋，也想不通你为什么要站在那里看我八个钟头，却不出来见我——告诉我，我得知道。"

　　月亮又沉默了一会。

　　"好罢。"

　　我没有说话，只是等待。

　　"有好多好多原因。"

　　"恩。"

　　"我想我很害怕，我还没有准备好，你就来到我身边了。我觉得手也没地方摆，脚也没地方放，好象天也变了，地也改了，一切都变得乱糟糟的，什么都认不出来。"

　　"恩。"

　　"我觉得原来我们在网上的日子好极了，我从来没那么快活过，天天打电话给我的男人说我爱他。我怕见到你以后什么都变了，什么都没了。"

　　"那现在你怎么又想见我了呢？"

　　"你已经来了，我即使不见你，还是什么都变了，什么都没了。"

　　"恩，既迈了这一步，就再也回不去。"

　　"你还记得我说过我是个不认输的女人吗？从来没有人能够打败我的。你说要见我，我说不要，如果我去见你，那就是我输了，我恨透那种被打败的感觉了——可就算是现在，还是你赢了，因为明天我会见到你的——无论如何总还是我选择了时间和地点。"

　　"我一直晓得你是个很骄傲的女人，可今天才知道你傲到了骨子里。"

　　"还有，我想好好看看你，仔仔细细地看你，看进你的心窝儿里面去，看你是不是跟我心里头想着的那个人一个样。"

　　"结果呢？"

　　"你坐在那儿一动不动，光是抽烟，连饭都没有吃，我怎么看得出来呢？我觉得我傻死了，就好象看了一整天泥巴人一样。"

　　微笑不知什么时候又回来了，"连一点点都没看出来吗？"

　　"恩，你长得很难看，比我以前的男朋友们难看多了。"

　　"真遗憾，还有呢？"

　　"我多看你一眼，就觉得多爱你一点——我看到你把烟凑到唇边的时候，恨不得我自己就是那根烟。前些时候，我已经在纳闷我小小的身子里面怎么能装那么多的爱，现在我才晓得，那时候不过才将将淹了个底罢了。"

　　"还有吗？"

　　"恩，还有个原因，不过我现在还不能告诉你——我以后一定会告诉你的，好吗？"

　　"好罢，反正我听到的原因也尽够了。"

　　"你能原谅我吗？"

　　"现在是这个样子的，那就应该是这个样子的。"

　　"恩，我得走了，去机场。明天下午七点半，等我。"

　　"我会的。"

　　"吻你，明天见！"

　　"吻你，明天见！"

　　我下了线，站起身打开阳台的们，云层缝里，透出了几丝阳光。

　　四十分钟以后，我接到了一个电话。是月亮在机场。

　　月亮大概说了一百遍我爱你。

　　我是沉默的，因为并没有Internet，不管我说什么，都是一样的。月亮听不到。

　　我第一次真正意识到，月亮是个聋子。

　　我等不及明天了。

　　无论是真实的绝望，还是真实的极乐，都比现在好。
