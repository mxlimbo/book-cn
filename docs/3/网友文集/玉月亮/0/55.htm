第五十五部

　　沉默。

　　"我想，我还是会见你的――我太爱你了。可那 不 一 样。"

　　"是的，那不一样。"

　　"你现在心里头在想什么？"

　　"你呢？"

　　"我――我觉得很――英语里面我不知到底是什么字，在中文里面，我们叫它幸福――XING FU。"

　　"幸福？幸福是什么？"

　　"让我想一想，英文里头，有几个字跟它的意思差不多――Blissfulness，Felicity，恩，还有Blessedness――可它们都不是我觉着的那个意思，总是差了那
么一点点。"

　　"幸福――这个字看上去挺漂亮的。"

　　"你呢？"

　　"不知道。比开心甜一点，比快乐美一点，比喜悦厚一点，比满足大一点――也许，就是你说的那个字罢――幸福。"

　　"恩，那就叫它幸福罢。"

　　"恩，还有一个问题。"

　　"什么？"

　　"明天搬到我家里面住好吗？这里太贵了。"

　　"家？"

　　"对，家。我一个人的时候，那里不过是一间公寓，我们两个人的时候，那里就是家了。"

　　"我做饭给你吃好不好？"

　　"好啊！――你饿了没？"

　　"啊！你一说我才想起来――饿死啦！一天一夜没吃东西了！咕咕。"

　　衣衫一点点把美丽遮了起来，而另一种美则如凤凰般从火中而升，绚烂夺目。

　　而当月亮坐在妆台镜前梳她的长发时，这世界上便同时有了两个美的奇迹。

　　月亮的发，就那样的一点点缠绕着我的心，化到了每一根血管里，从此流遍了我的身。

　　月细风尖垂柳渡，梦魂长在分襟处。
