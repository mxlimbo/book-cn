第三十八部

　　"好了，我讲完了。我不晓得你信不信我，可我说的，每一个字都是真的。现在该我听你说了。"

　　"月亮，我回来，只是想跟你说一句话。"

　　"你不想要我了，是吗？你连一句话都不相信我，是吗？"

　　"月亮，我爱你！"

　　"你说什么？"

　　"我回来就只想对你说这一句话，我爱你！"

　　"你——你真的爱我？"

　　"是的，我爱你！我爱你很久了。"

　　"可你为什么不告诉我？为什么？！"

　　"因为我害怕。你住在世界的另一边，你告诉我你是个聋子。

　　我不知道你长得什么样子，我也不知道你在生活中是到底怎么样的一个人。

　　我唯一所知道的是我真的爱你，可我不知道这爱是不是够强——强到能使我对一个万里之外的女子保持忠诚，强到可以让我和一个聋子生活在一起，强到可以让我可以接受你有你自己的生活方式。

　　我不敢说，我怕我说了，却做不到——那会伤害你，我更害怕那会伤害我自己。

　　我是个很自私的人，我只想保护我自己，所以我想，还是不要说爱的好。"

　　"我也害怕，怕极了。每一天我都想，也许明天你就对我腻味了，也许明天我就见不到你了，也许明天你就会发现在你眼前活生生的女人更有趣些了，也许明天你就不会再来跟我说话儿了。"

　　"我不敢说爱你，可又舍不得离开你，我只好选择了逃避——我想只和你保持一种网上的情人关系，少一点我会受不了，多一点我都不想要。

　　可到了昨天，我才明白，我躲不了的。

　　我或者全心全意地去爱你，或者彻彻底底地忘记你，其实并没有第三条路好走。

　　可我根本就忘不了你！那除非割了我的心，那除非煎了我的骨。

　　我只有全心全意地去爱你，我只有爱你，无论爱你要付出什么样的代价。"

　　"对不起，我昨天一定伤你很深。"

　　"说对不起的应该是我，我昨天说的话才真的伤人。"

　　"昨天你离开我以后，都做了些什么？"

　　我迟疑了，我不晓得我是不是应该说真话。我知道真实是伤人的，可我不能欺骗月亮。不能。

　　"昨天我离开你以后，找了一个女人。"

　　"你——你——""真的对不起，你骂我罢，你说什么都可以，求求你骂我罢。"

　　"你，你跟那女人做得好吗？"

　　"一塌糊涂，只有两分钟，我根本做不下去——可是，我毕竟还是做了。我只有请你宽恕。

　　我请求你宽恕我。你无论做什么都可以，无论你做什么都是我的错。"

　　一阵可怕的沉默，我觉得浑身发冷，我有点后悔我说了真话。
