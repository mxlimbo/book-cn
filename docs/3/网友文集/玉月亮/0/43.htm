第四十三部

　　三月八日。礼拜五。十二点四十五分，尼斯机场。雨。

前路漫长而黑暗，一切都尚未注定。

——电影台词，终结者二。

　　我没有带多少行李，只是一些随身的衣物，都装在了一只黑色背包里面，还带了一只布老虎，准备送给月亮做礼物。

　　我在候机厅给月亮打了个电话，没有人接。我试了几次，也就不再坚持。

　　虽然并不饿，我还是买了个三明治，然后找了个位置坐下，慢慢地咀嚼着，三明治有些硬，好象不是今天做的。面前是很大很高的玻璃围墙，玻璃是透明的，却没有阳光照进来，因为今天是难得的雨天——在蓝色海岸，一年里倒有三百四十天是阳光明媚的——候机厅里面便因此充满了阴郁的光，一切都是灰蒙蒙的照得人的脸都快活不起来。候机的人不是很多，所以又显得有些冷清，人们零零散散地坐着，不少人在抽烟。

　　从玻璃墙望出去，是机场的跑道，因为雨，跑道变成了黑色，湿湿的，几架飞机跟玩具似的摆在那里——有时我真不敢相信我居然敢把我的生命完全托付给这纸一般脆弱的东西——铁疙瘩怎么就能在天上飞呢？再远些，是海。因为雨，岸边的海水泛着惨白的混浊。风不大，微小的浪拍着岸，有许多泡沫状的水花。

　　只有远处的海水还是蓝得透明，蓝得可以净化疲劳的心。

　　遥远的存在总是美的。

　　我买了几包烟，回到座位上抽烟，消磨等待的时间。我想起来了，好象我的一生都是在等待。

　　这可能是为什么我要抽烟。

　　我忽然觉得肩头有一点凉，伸手去摸了摸，是水。哪儿来的水呢？我抬头向上看去，正好看见一滴水珠从高高的候机厅天花板坠落，很慢很慢的，水珠在空中拉长，直到再也不能连在一起，就断作了两段，还是很慢的，落在了我的肩上，一点声音都没有，也没有溅起来，它只是匆匆的在我肩上的衣服上扩散，溶进了已有的一片阴湿。

　　然后我就觉得更冷了一点。

　　我的心突然梗了一下。无数的疑问象洪水一样冲进我的脑海。

　　我真的准备好了吗？我真的该见月亮吗？我真能见到月亮吗？如果见不到，又怎么办呢？我没有办法回答自己。

　　我觉得原本满满的信心象在阳光下的雪堆一样迅速融化——我甚至可以听到它融化的声音。

　　我觉得自己就象一个只有一个筹码的赌徒，却不加思考的把这唯一的赌注押上了赌台，我好象看见了庄家嘲讽的笑。

　　庄家有个名字叫做命运，他开的赌场从来是吃人不吐骨头的。

　　可我们不得不赌，因为我们除了押下筹码外一无选择。

　　总算可以登机了。

　　我抽了一份Financial Times，找到了我的座位，重重地坐了下去，我努力让自己的身体在位子里陷得更深些，似乎这样会安全些，可我还是觉得有什么东西在咬噬我的背，我没办法坐得安稳。有几次，我几乎就想冲出机舱，一直冲回家，冲回到我的电脑前，让自己再次在Cyber甜蜜的梦中沉睡。

　　飞机开始在跑道上滑动，机轮的摩擦发出很刺耳的声音，即使在机舱里面也听得到。

　　我如释重负——我终于可以不必在见与不见之间徘徊了。

　　我已在路上，回头已不可能。
