第四十九部

　　月亮慢慢地向我走过来，她的脸色越来越苍白，简直要哭出来的样子。她一直看着我，仿佛想从我这里得到一点勇气，一点鼓励。可我哪里又有勇气给她呢，我自己也是在害怕，怕得要命。

　　月亮走到围绳前一步的地方停了下来，再也不肯动了似的。

　　我们一动不动的对视着，仔仔细细地看着对方，这么近，这么近，再没有任何阻碍在我们之间了。

　　时间凝固了。

　　只知当时明月在，在我身边。

　　我们的目光越来越贪婪了，越来越不能自抑了，几乎是同时的，我们伸出了手。

　　都是右手。

　　我终于触摸到月亮了，真的。

　　我感到月亮的指尖在我手中颤抖，或者，那不过是我的手在抖。

　　我手足无措，可我得做些什么。

　　我突然伸出双臂，一把将月亮揽了起来，把她从围绳上面抱了过来。

　　抱得那么紧。我太害怕这又是一个梦了。

　　月亮轻轻地对我说，"我，累，坏了。"

　　然后月亮就闭上了眼睛，把头埋在了我的怀里。

　　我轻轻的抚着月亮如水的长发。

　　我也闭上了眼睛，我心中充满了感激。

　　那时是你用一块红布，蒙住我双眼也蒙住了天，你问我看见了什么，我说我看见了幸福。

　　我感觉，你不是铁，却象铁一样强和烈，我感觉，你身上有血，因为你的手是热乎乎。

　　不知道过了多久，我再次睁开双眼的时候，围观的人大都已经散去了，只有先前的那个女子仍然站在一边，还有几个机场的工作人员远远的在指手划脚，不知嘀咕着些什么。

　　我低下头看月亮的脸，正好看见她的眼光从下面看上来。

　　我再次陷了进去，那深不可测的温柔。

　　我们象石头人似的看啊看啊。

　　这次是月亮先醒过来的。

　　她挣扎着从我怀中脱身出来，冲着边上的那个女子做了几个手势，是哑语。那个女子也做了几个手势，她们似乎在交谈。然后那个女子对我说，"您好，我是Linda，是月亮的朋友。月亮说她要来法国，正好我一直也想来尼斯旅游，就陪月亮来了。我懂哑语，这样顺便也可以帮你们俩做翻译。"

　　完全没有经过任何的思索，我立刻拒绝了，"非常谢谢您的好意。不过，我想我跟月亮能交谈的。"

　　我取出了Palm，"有些话，我必须直接对月亮说，希望您能理解我――我希望可以同月亮永远在一起，您不可能帮助我们一辈子的――也许一开始会很困难，但我们会有办法的。"

　　Linda有些愕然的样子，接着她对月亮又作了几个手势，似乎是在告诉月亮我所说的。

　　月亮笑了。

　　月亮笑的时候，眼睛弯弯的，就跟月芽儿一样。

　　我把Palm举到月亮面前，写道，"月亮，我们去旅馆罢。"

　　月亮掂起脚在我脸颊上轻轻啄了一下，算是回答。

　　这是从月亮上掉下来的第一个吻。

　　这个不算数的，我暗暗的想。
