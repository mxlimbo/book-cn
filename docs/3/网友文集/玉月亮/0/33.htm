第三十三部

　　已经有近七个月没有见过Isa了，她还是很性感。

　　Isa穿了一件火红的吊带裙，和一件黑色开司米开襟短外套。裙子很薄，我可以清楚的看到她乳房骄傲的轮廓和两点小小的凸起，金色的发披在肩上。她站在Diable门口的灯光中，冲我笑，灯光似乎都被吸引到她身上了。几个小伙子经过，冲着她吹口哨。

　　我盯着Isa的眼睛走了过去，一直走到我的唇可以感到她唇的湿润，我们吻了一会儿。然后，我揽了Isa的腰，我们走进了Diable。

　　Diable是尼斯一间很有名的酒吧，每晚，里面都会有Live Music和很多的年轻人。他们不知道该去哪里，所以就聚在了一起。大家围在一起的迷茫，总是比一个人的迷茫好一点的。

　　你可以在里面发现任何颜色的头发，各种形状的鼻环以及五光十色的刺青。

　　我以前并不是很明白为什么许多人会喜欢那种有点邪恶的感觉。

　　我现在知道了。

　　乐手在唱Prince的歌，紫色。歌词热得可以把酒吧里年轻的心烤化，许多男孩和女孩在座位上肆无忌惮的扭动着，拥抱着，品尝青春的味道。

　　我的手伸进了Isa的裙底，她穿的是很薄的莱丝内衣，我猜跟以前一样，还是香奈尔的。内衣的质地非常滑润，柔软的翼状物舒展着，我触着，摸着，抚着，慢慢的，我的指尖感到了一点点湿润。

　　Isa笑着拿拳头砸我的肩，骂着："死东西，把手拿开。"

　　她并不坚决，所以我的手还是停留在原来的地方。我的另一只手，或者夹着一只烟，或者举着一杯酒。

　　我喝的是不加冰的Whisky，大杯。

　　Isa喝的是尼斯的Bastis混合酒，浅黄色的液体。有时，我会抿一点在嘴里，然后喂进Isa的唇间，我们的舌在吸吮着。

　　我们不怎么说话，就算是说话，在Prince尖利的吉它声中，也听不见。我们就跟聋子一样。

　　当我的胃再也不能忍受的时候，我去卫生间，呕吐，然后我会很快的回到座位上，接着喝。

　　酒似乎越来越淡了。

　　这一次，我没有醉，虽然我很想。

　　可能是午夜了罢，我把嘴紧贴了Isa的耳朵冲她吼，"我们去你的公寓罢。我想让你快乐。"
