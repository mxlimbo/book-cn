第三十六部

　　一月三日。礼拜六。七点零三分，办公室。曙白。

我怎样才能让你晓得，我对你的爱，比你感到的还要多一点？请宽恕我，我真不知我做了些什么。

请宽恕我，我无法让自己停止爱你。

请不要拒绝我，那痛苦会刺穿我的身躯。

请相信我，我所说的都是真的——我无法让自己停止爱你。

　　——请宽恕我，Bryan Adams。

　　清晨的太阳，在地平线上小心地露出了半个脑袋，好象还没睡醒的样子。

　　我在这半梦半醒红色的光中发疯一样的开着车，以致于走错了路，绕了很大很远的一个圈子。

　　我的思绪中，只剩下了一个念头：我必须去见月亮。

　　我必须告诉月亮，我爱她。

　　我突然意识到一件事——我怎样才能见得到月亮呢？我和月亮所有的联系，只有Internet罢了。我没有月亮的地址，我没有月亮的电话，我甚至不知道月亮的真名是不是月亮，什么都没有。

　　月亮在周末上不上网，我根本就不知道。

　　尤其是在昨天我如此残酷地污辱了她以后。

　　我的公寓里面，是没有Internet联结的。

　　而在蓝色海岸，你几乎可以找到任何东西，除了网吧。

　　唯一我所知道可以上网的地方，是我的办公室。

　　但，今天是礼拜六。

　　我是不能进入研究所的，保安警察不会允许我进去，虽然他们笑容可掬，彬彬有礼，可在他们的字典里面并没有通融这个词。

　　我第一次后悔我没有申请Internet联结，后悔得头发都要掉下来。

　　可我必须见到月亮，我不能等到礼拜一，绝对不能。

　　我唯一的选择是进我的办公室，现在，马上。

　　我绕到研究所后面，翻墙而入。

　　我不是很清楚我被发现的后果，也许，我会因此失去工作和名誉，也许，会比这更糟——警卫是有武器的。可现在，已经顾不得这许多了。

　　我只有在心中祷告我不会被抓住。

　　周末的研究所里，静悄悄的，所有的人都去享受周末的闲暇了，似乎只有我一个人在。我的状态显然不是很正常。

　　我轻手轻脚地打开了我办公室的门，没有开灯。接着我坐到了电脑前，输入我的密码。屏幕一点点亮了起来。

　　我不知道月亮会不会在那里，也许，她现在正偎倚在一个男子的身边熟睡，就如她昨天说的那样。

　　我忐忑不安。

　　我的手在抖。

　　我接通了MESSENGER。
