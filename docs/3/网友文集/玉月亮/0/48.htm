第四十八部

　　三月十二日。礼拜一。十九点四十分，尼斯机场，晴朗。

记得小苹初见，两重心字罗衣，琵琶弦上说相思，当时明月在，曾照彩云归。

――临江仙，晏几道。

　　我瞪着那个女子，我想，我那时的表情一定很狰狞可怖。那个女子肯定是被吓到了，她畏缩地向后小步退着。一些旅客不晓得发生了什么事，便围了过来。

　　这时，我听到了月亮。

　　"瑜，我，在，这里。"

　　是月亮的声音！我向发出声音的方向看去。

　　是在出口前最后一个拐角。

　　这世界上怎可能有如此之美的女子！月亮个子不高，一头鬈曲的黑得发亮的长发如瀑布般一直搭到腰际，头上唯一的饰物是一朵红色的小花，插在发际稍前一点的地方。

　　月亮的肤色是黝黝的，脸色却有些苍白。象海一样深的褐色眸子，上下都长了有长长的睫毛，向中间微微拢着，而从那双眸子里面射出的是怎样的一种目光啊！是深情，是忧郁，是无法抑制的激动，是徘徊彷徨的忐忑。她的目光就象雷电一样击中了我的灵魂，轰得我忘记了一切的所有。

　　她的眉毛很长，几乎是平平的在双眼上掠过，那是上天蘸了最重的墨在她的面容上画下的。

　　鼻梁只好用秀丽来形容了，边沿有点儿微微上翘。

　　但又能找到什么言语来形容那一对唇呢？红红的，，鲜润的唇就如朵含苞待放的玫瑰，美丽的唇形在整张让人会在梦中思念的面容上描了一笔重重的红色，火一般的红色。

　　她的体形是完美的，所有一个女人可以有的曲线，她一样都不缺。她只是穿着白色的夹克和套头衫，下面则是一条半旧的蓝牛仔裤，可即使是这样，她的性感仍然一直逼得我喘不过气来。

　　我一直以为美丽和性感是不能共存的，就如水与火一般，一种特质会自然而然的削弱另外一种。

　　我这时才晓得我谬误到了何等地步。

　　这已经不是人间应有的美。

　　因为，她就是月亮。
