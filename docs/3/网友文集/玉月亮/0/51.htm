第五十一部

　　三月十三日。礼拜二。十一点零七分，Carlton，晴朗。

如果你一心要爱我，那就别为了什么，请只是为了爱才爱我。

爱我，请只是为了那爱的意念，那你就能继续地爱，爱我如深海。

——白朗宁夫人《葡萄牙人十四行诗集》第十四首。

　　昨夜雨疏风狂，痴意云密水长。

　　仿佛明天就是世界的末日。

　　我们疯狂地眷绻着，无休无止地索取和付出着彼此身体里的最后一丝精力。

　　将性命拼却，由伊肆意怜。

　　几线阳光，不屈不挠地从窗帘狭窄的缝隙间挤了进来，在地面上画着细长的明亮带子，扭曲出奇异的线条，其他不那么幸运的光线，则透过了窗帘厚厚的红色纤维，把房间染作了暖暖的暗红色。

　　月亮把脸埋在我的胸膛上，我的手指则在月亮的发间慢慢的游走。

　　没有人做声，房间里面安静得就好象根本没有人一样。

　　我们筋疲力尽，心满意足。

　　我们一点点轻微地移动着肢体，让每一处肌肤带来并给予触感，若即若离，似续似断，有时，我们之间唯一的接触不过是两人的小指尖，可我们一秒钟也没有分开过。

　　我似乎可以感到有一细缕流质般的东西从月亮的指尖流入我的身体，然后它在我体内慢慢舒展扩张，无处不在沁入了我每一丝细微的肌纤和每一片细小的骨节，最后慢慢地与我融为一体。

　　想起了以前不知哪里读到话，"将两个泥人，一起打碎，便你中有我，我中有你。"

　　原来是这么一回事。

　　爱，竟然是一种实体的存在？好久，好久以后。

　　月亮说了一句话，"我，好，累。我，想，睡觉。"

　　我没有回答也无法回答，只是把月亮的脑袋往我的腹部挪了挪，好让她枕得舒服一点。

　　一开始，每隔一两分钟，月亮会受惊似的猛然睁开眼睛看我，好象担心我眨眼功夫就会消失似的，后来，月亮睁开眼睛的间隔越来越长，差不多半个钟头以后，她的呼吸终于平稳了。

　　我看着梦乡中的月亮，我看着她的嘴角一点点往上翘，最后，梦中的月亮绽出了一个灿烂的笑容。我也笑了。

　　后来，我也睡着了。

　　再次睁开眼睛的时候，又已经是夜了。月亮还在甜甜地睡着，我小心翼翼地挪动身体，不想惊醒月亮的梦。

　　在洗手间的镜中，我看着自己。没有开灯。镜子幽黯到通至永恒，脸孔溶在黑色中，依稀可辨。我伸出手指，抚摸着镜中的眼，突然，我发现眼里有针尖大小的一个亮点，我让视线顺着深远的隧道流进去，流近那点亮。

　　终于看清楚了，是月亮。

　　占了我的眼，驻了我的心。

　　无边的暗夜里，我一直如盲人般摸索前进，我知我在，我不知面前为何。我前进，后退，奔波，挣扎，我看不见原因，我看不见结果，没有人看到过我，我则漠然注视黑暗。我是一个符号，一串数字，使用着并被使用着，卑猥如尘。

　　可今夜有微光。

　　也许微光终究还是会熄灭，但总已是看见些什么了。

　　天边有月，眼中有月。
