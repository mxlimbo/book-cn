第三十五部

　　一月三日。礼拜六。五点三十七分，尼斯。惊涛拍岸。

我就要回到老地方，我就要走在老路上，我明知我已离不开你，花房，姑娘。

花房姑娘——崔健。

　　我顺着尼斯和嘎纳之间的海边道往回开。

　　风很大，吹得我手中的方向盘在发抖。

　　没边没际的海岸线，和路平行着向无穷远处延伸，一直刺进了夜最深的地方。呼啸的风声中，浪头一个接着一个自杀般地扑上岸来，在岸边岩石上撞得粉身碎骨。连大地都似乎要裂开了。

　　在这凌晨前最黑暗的时候，我还是能看到雪白的浪花在月色中飞舞，无数的晶莹散漫在空中，就象我现在的心绪一样。

　　没有尽头的路上，只有我一个人，一辆车。天地间似乎只剩下了我一个。

　　不，天上还有月亮。

　　月亮斜躺在天上看着我，有点忧伤的样子。

　　我打开了车上的音响。涛声和嚎叫声混作了一处。

　　我听见了我的心在跳。

　　"我独自走过你身旁，也没有话要对你讲。
　　我不敢抬头看着，你的，脸庞。
　　你问我要去向何方，我指着大海的方向。
　　你带我走进你的花房，我无法逃脱花的迷香，我不知不觉忘记了，方向。
　　你说我世上最坚强，我说你世上最善良。
　　我不知不觉已和花儿，一样。
　　我想要回到老地方，我想要走在老路上，这时我才知我已离不开你，花房，姑娘。
　　我就要回到老地方，我就要走在老路上，我明知我已离不开你，花房，姑娘。"

　　我停下车，站到了水边，冬天里最刺骨的风吹在了我的脸庞，吹在了我的胸膛，我扯开了我的衣裳，好让风扎我扎得更猛烈些。我的衣服已被沁得湿透了，紧贴了在身上，我每一根骨头都在这深冬的寒冷中战栗，我每一个毛孔都在这冰冷的海水中紧缩。

　　我让我所有的神经都去触摸这我一生中最强烈的感觉。

　　这时我才知我已离不开你，月亮。

　　我终于知道了什么是爱。

　　我不管了，月亮是圣女也好，是婊子也罢，她是聋子也好，是哑巴也罢，她近在眼前也好，远在天边也罢，她是众人宠爱的公主也好，她是被所有人遗弃的破烂也罢，她爱我也好，她不爱我也罢，就算月亮每天晚上都有一个男人在她的床上，我也不在乎了。

　　我要的就是你，月亮。

　　我明知我已离不开你，月亮。

　　我得给她说，我爱你。

　　现在。
