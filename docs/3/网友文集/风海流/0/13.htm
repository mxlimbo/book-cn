二 世纪乐章

　　题记——我并不懂音乐

　　不知不觉，这二十一世纪真的敲响了我的门，她并不像徐志摩那样礼貌合斯文，"轻轻的来"到康桥，而是带着重重的步伐，嚷着无序的乐章，要闯进来。

　　在白猫黑猫的努力下，经济建设这个中心使人民的生活水平提高了，遗憾的是总设计师邓伯伯还是走了，就在这世纪末。正因为他老人家一句"一国两制"，使不少有识之士眼镜大跌，撒切尔夫人更是摔了一跤。香港澳门都在欢乐的乐声中回到祖国的怀抱。那盛大的阅兵式，雄壮的歌声响彻长安街。

　　但胡长清和成克杰这些现代和坤的出现，使轻松的步伐变的沉重；李洪志带着他那骗人骗财的法轮，又给轻松的头脑加上了紧箍咒；张子强与张君的猖狂，使矫健的步伐带上了血迹。刑场上的枪声如此刺耳，亡命之徒的枪声更加骇人。

　　当然，我们忘不了许海峰与那神奇的枪，体操王子李宁的健硕，忘不了邓亚萍与萨翁的握手，伏明霞与熊倪自信的笑，忘不了葛菲顾俊的深情拥抱，王涛在胜利一刻振臂高呼。激昂的乐章由掌声和欢呼声编写，由国歌演奏。

　　但球迷们无法接受二十年未能治愈的"恐韩症"，未能踏上世界杯的征程；大家也想不通吴艳艳与禁药的丑闻，在泳池边殴打裁判的山东"四剑客"；球场上大家不满的声音——假球，无奈的呐喊——黑哨。声音如断了弦的琴——嘶哑。

　　想起了弗莱明绿色的灵芝——青霉素，还有白白胖胖的多利羊，筑起因特网的李伯纳斯，在太空漫步的加加林，在月球上奔奔跳跳的阿姆斯特朗；更忘不了李四光与大庆的石油，"杂交水稻之父"袁隆平，邓稼先的"两弹一星"。原子弹爆炸的巨响如此惊天动地。

　　但成功的欢呼又让我回到沉思。东京的沙林毒气，广岛和长崎的人间地狱，库尔斯克号上那118位官兵，网络上的色情与暴力，这正如飞机坠毁，火箭爆炸，被炮火炸得支离破碎的南联盟，爆炸声如此恐怖，哭声如此悲凉。

　　我手捧余秋雨的《文化苦旅》左右端详，在钱钟书的《围城》中转了又转，走过徐志摩的"康桥"，穿过韩寒的《三重门》，在大榕树下为《死亡日记》的作者陆幼青祈祷；我聆听巴金激流、爱情、抗战三部曲，回味鲁迅熟悉的《南腔北调集》，多么和谐，动听。

　　而所谓"新新人类"的卫慧，她那一本标榜"用器官写作"的污染世风的《上海宝贝》，竟然被炒的神乎其神，还摆上了热销书柜，想不到这种腐朽的文化垃圾，在文明的世纪之交，居然有立足之地。这垃圾不但让文化界变得臭气漫天，还招徕臭味相投的苍蝇，发出恼人的嗡嗡声。

　　二十一世纪来了，带着无章的噪声闯进我的门，但我期待她进门后，不要制造那噪声，唯留下一篇激昂、催人奋进的新世纪乐章。
