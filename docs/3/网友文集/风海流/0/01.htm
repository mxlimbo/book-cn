《风海流》自序

　　高一时，认识了Mr. Ming ，他有一句"吃饱了撑"的名言。的确，在那时我真的吃饱了撑，面对只有地理一科会考的悠闲，像猪一样，吃了睡，醒了吃。但无心插柳，地理还优了，因此不得不佩服自己的小天才。

　　我对地理的"风海流"一词有莫名的好感，或许音节受我耳称来。在假期，我没吃饱，不过也撑，看了第一部外国名著《鲁宾孙漂流记》。看完之后，鄙人觉得笛先生的小说显然离题了。鲁宾孙的确漂流了一阵子，但他在船上行使时不算漂流，在美洲经商时不算漂流，在孤岛苦闷的27年不算漂流；只有他抓住烂船板，漂流到孤岛上才算漂流，这不过几天的漂流，连一章也没占够，何以为漂流记呢？

　　哲人反驳我，鲁宾孙的确只漂流了一阵，但他的一生都在漂泊，带着挑战的漂泊，那27年的孤岛历程，只是他人生漂流的停留过最大，最有感触的驿站而已。

　　27年只是一个驿站，我从坠地到现在不过18年，我有资格把书名改为《风海流》吗？我向哲人请教。

　　哲人说，可以，人的一生都是漂泊的，只是你与鲁宾孙漂流在不同时间，不同地点，不同的浪花中而已。鲁宾孙一生在大西洋漂流来漂流去，有过真正的漂流与抽象的漂流；你也有过，但为抽象的，抽象比真正的有着更多的感触。你漂泊在人潮中，跟着时代的风而漂流，你随着知识的风漂流向人生的重要驿站——大学。但在你前面有一狭窄的海峡——高考海峡，海峡将挤去好多人。祝你一路顺风，别被挤掉。

　　哲人走了，化作一阵风，把我吹进了人潮。我被人浪卷了进去，我在随风漂流、挣扎。

　　说到在人潮中漂泊，我只写了一篇《头是圆的》，文笔略显清高，我最欣赏的系列是《圆……》，可这样的文笔在高考，老师顶多给30分。我怕了，怕混不到要饭吃的一张文凭。于是忙写了一些合老师口味的文章，我不愿被高考海峡挤掉。第一篇《遥望星空》可谓典型的应试作文，完完整整的编故事，为老师而作。还有《用不了的五元钱》使人觉得我老实得可爱。其实那并不是我，放在文集中有点穿西装打领带穿皮鞋打篮球一样，长得像乔丹也看不舒服。

　　为了舒畅一下，与同过海峡的对手搞好关系，于是写了使人心情舒畅的《我……》，但自嘲中，戈尔布什曝光率过高，影响了整体画面效果。其实还想加上几篇《我X》，但渐有江郎才尽的感觉，于是住手。

　　我现在与大家一样，在人生的路上，在人潮中漂流，在漂流中摸爬滚打，在摸爬滚打中愈来愈接近高考海峡。听过了海峡的人说，海峡宽得很；听没过去的人说，那里比炮烙之刑的铜柱更恐怖。我更怕了，怕别人轻松地过去了，留下几个人（包括我）来受炮烙。我也同时担心，自己各科成绩愈来愈"肥"，海峡相对我来说愈来愈窄。

　　但我别无他路，因为从我坠地的一天起，就注定要卷入社会，卷入人潮，注定要随风漂流。

上杉奇也 2001年1月26日于广州
