二 圆圆的头

　　人头也是圆的，开始我不知道，直到那一次到了天河体育中心，一睹张惠妹的"妹"力之后才发现。

　　人群一个圈一个圈的围着中心舞台，这有点像封建等级制，前排的人用钱换来看张惠妹的圆，而我与朋友也用钱换来了最后一排的票，也看到了圆，不过是前排有钱人的圆。自那以后，我发誓当我没有给别人看我的圆的资本时，决不去看秀凑热闹，反正到处可以看到圆。

　　高中生压力过大，不少人以上街为降压的指标工作，于是也常有人邀我去，我却没有这兴致，婉拒道："没空，你另找他人陪你看圆吧！"广州市区人口多达400多万，但并不见到圆山圆海，因为都集中到上下九路、北京路和天河去了。那几个地方的商业街，旺过阿斯旺，圆潮汹涌，如当年红卫兵串连，不过当年毛主席眼皮下的圆是千篇一律的绿色，而且朴实程度可以上吉尼斯世界纪录。而现在上下九路、北京路和天河的圆可以变得个五颜六色来，红色的黄色的绿色的……而款式也多种多样，有卷如蜻蜓点水搞起的水波衍射，有陈佩斯似的凸镜，有发线如瀑布直冲地下的等等，怕那飞瀑之下，必有什么什么，到时上下九路、北京路和天河会多出几个深渊。

　　如果身高不济，建议大家别去那地方，否则你会被人潮淹没。在那里的行道之人，特别是女性，身高都好比郑海霞。若一WNBA教练经过，必慨叹中国教练不会选运动苗子，不过他很快收回了自己的话，方才发现那群高人都有一双比电视塔还高的松高鞋。女生比男生更爱上街，就是因为这电视塔，校规不许穿电视塔回校，无法一展身材，于是便带上电视塔上街孤芳自赏。星期一我见一女生一跛一拐上学来，便问其故，答曰："冰封三尺，非一日之寒，未穿惯那'电视塔'咯！"

　　几年前，广东才子伦文叙出生地状元坊又成了圆圆之地，我慕名来过一次后，便发誓再也不来了。男的如做秀，比谢霆锋穿得还夸张，老广州见这一幕也痛心疾首，从原来的文明之地变到现在的用广州话形容就是"鬼五马六"。男男女女口中都叼根烟，如电影中的黑帮讲数。听说这晚上性暴力不少，这也难怪，女的喷的一身香水比九里香还甚，穿得比麦当娜海火辣，比张惠妹还少，还不使色鬼欲火棼身？近来又听说那有人用艾滋病针头扎人，实在骇人，要是这让九泉之下的伦相公知道，状元爷必火冒三丈。

　　小学时候有人以一台386为自豪，六年级时486还刚刚问世，初一，某人加重以一台奔腾75成为全班人嫉妒的火力点。不久后，又一后起之秀以奔腾133勇当急先锋，后来接踵而来的MMX166，MMX200，一直发展到现在的奔腾四代，电脑的普及不比文化大革命时人口增长慢。由于受盖茨巨额身家的诱惑，不少人也对IP神话有了莫大的兴趣，电脑城的圆也大有后来居上之势，由于不少人爱上了3D游戏，又恨自己过早把现在早已不中用的MMX166请回了家，升级又尚无足够储备，只好对奔腾4叹气"想说爱你不容易"，于是把爱放在廉价的3D加速卡，一块名为Voodoo Benshie（巫毒女妖）的加速卡大受欢迎，由于名称如够前卫，还有代表性，于是班上的男生把那些潮流派代表的女生统称为"Voodoo Benshie"。

　　上高中后我又发现了多个圆集散地，那就是名师开办的补习班，由于在《再谈"减负"》中也谈过了，因此也不去看那里的圆了。

　　我们班曾有人断言"哪里人头涌涌，哪里就有Voodoo Benshie"。看似觉得这句话很有道理，不过自那一次从火车站出站口出来后，我才发现圆头涌涌的火车站找不到Benshie。火车站可谓是圆来圆去，圆流动性最大的地方。这里的圆比上下九路、北京路和天河的圆更有特色，有的像鲁迅说的标致的富士山，有的像一大堆草，最可爱的要算是像诺贝尔刚从实验室出来刹那的圆，炸开来的。这里的人不单头圆，语言更圆，可以把外来人的盘缠骗得分文不剩。但有的人不跟你圆，方过魔方，只要你稍有不逊之举，或外貌长得像猪八戒，不顺眼，那方人就如关羽托世，怒火烧到面耳皆赤，一群衣冠不正的弟兄上来把你拳脚一顿后扬长而去。火车站另一大风景为出站处，检票人认衣再认票，只要你身着入流衣，像个小康家庭成员，不用票也放你出去；若你身着如外来妹或外来兄，检票认定厉声吓你"把车票拿出来。"如见带有小孩的，便如拔萝卜似的把小孩拉到1.1米高的量身处。开始外来人不知何故，以为广东人待客如宾，量体裁衣给送给小孩，殊不知这小孩高出那线一点点时，那检票的便用不生不熟的普通话大骂，外来人不解，以为超高没得衣服送，几欲先走，但被一肥硕保安逮住，咒骂一顿，后来方知量高是要她的小孩补票而已。而全国各处上车时1.2米以下免票，出站时全变1.1米了，由于补票要加手续费，比车票贵了几十圆，从中的差额，就是政治课说的"剩余价值"。由于出站外来人多，加上出站口如鼠洞，几十个车厢的人挤得水泄不通，把几天在列车上尚未爆发出来的"味力"全挤出来，成为看圆的最美的风景区，色香味俱全，加上铁路员工的苦心经营，效益好过不少中外合资企业。只见一铁路员工领导用舌头舔一舔手指，数钱不已。我开始明白钱为何用"圆"做单位了，因为赚来的钱，都是从圆里赚回来的剩余价值。

　　中国人圆十二亿，如果这全真变成外汇储备，那么再来几个索罗斯，中国也不屑一顾，可惜这十二亿圆是人头数，俗语说"人多乱，龙多旱，鸡多了不下蛋，媳妇多了婆婆做饭。"不过我想加一句话，"人少了，圆哪里赚？"
