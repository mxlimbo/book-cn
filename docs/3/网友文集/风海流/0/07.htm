二 我等……

　　我等，无奈地等待死亡，说不定马上，说不定是明天，或者100年后，我等自己的青春到来，同时衰老又降临到我的头上。

　　我等梦中的那个妹妹与我牵手，但我怕等来的是一个姐姐，甚至是......老太婆。我等，等大学的录取通知书，等本科的文凭，而等来的却是高中肆业证书，还有拿假文凭的歧视。

　　我等，等贪官一个个被拉去打靶，等天上可以再见艳阳天，但我等来的只是朱镕基总理脸上越来越多的无奈的皱纹，等来的是一片片乌云。

　　我等，等新世纪的第一抹阳光，等到的却是一场暴雪。等股市的彪升，却等到连续二十日的跌停。我等戈尔接克林顿的班，却等到布什的呆笑，我用MMX166来下载，结果等到死机，我等网上情人的E-Mail，却等到了一句"just a joke"。等到见面那一天，方知对象为男性。

　　我等什么？等水中花，等镜中月，等物理老师说的虚像，等数学老师说的分母为零。

　　我不但等到花儿也谢了，我......也谢了！！！
