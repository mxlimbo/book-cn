二 再道"精"字

　　 精华，是一件事物中作用最大或最引人注目的地方，"精"字更是精华中之精华。俗语说："三个臭皮匠，顶一个阿亮"，这可是对阿亮的不尊重啊！三个人做一个人的工作，既多吃了两份粮，又多占两个床位，还多出两份薪水。只用一个阿亮，可以精简机构，这对于个人，对于集体，对于国家，"精"字可谓有着举足轻重的重要意义。

　　 1+1+1=3如果高考的数学有这么简单就好了，那么题海战术就失去了存在的意义，做1+1+1道练习题，是不可能等于做三道高考题的意义的。因为高考题是经过精雕细琢出来的，重在于精。

　　 "精"字是在可能的情况下，尽可能的减少人手，历史说明，并不一定人多好办事，并不一定能够人多力量大，火烧赤壁，淝水之战，退避三舍等著名的以少胜多的战役是很好的例子，这就是因为胜方的统帅用兵在于精。俗语说"人多乱，龙多旱，鸡多了不下蛋，媳妇多了没人做饭"也生动的说明了这一点，加之三个和尚的故事，更证明了人员在于精而不在于多。

　　前两年，我国裁军100万，这是好事，这并不意味着我们开始犯了生于安乐，对战争放松警惕的错误，这却是有力的证明我国的国防科技水平又上了一个新的台阶，国防开始趋向于精，这不但减轻了国家的军费开支，还解放了生产力，使更多人投入到经济建设的第一线中。

　　想当年，全国各地所谓村村来炼钢铁，虽然在数字上的确是炼了不少铁，但炼出来的废铁却占了很大比例，这明显犯了一味求多不求精的错误；现在东北一个鞍山，上海一个宝山，四川一个攀枝花都以技术密集型的方法建立了冶钢工业基地，炼出来的钢不但质量高，数字更比村村炼铁时多得多，"精"字在其中发挥了重要作用。

　　刘禹锡在《陋室铭》中曰"山不在高，有仙则名，水不在深，有龙则灵"当中"仙"于"龙"就是山和水的精华所在。回顾历史，为何刘备要痴心不改，苦口婆心，非要三顾茅庐请诸葛亮出山呢？何不在街上拉几个臭皮匠呢？因为刘备也知道1+1+1永远大不过诸葛亮一个人的才能。

　　"精"字易写不易做，我还不知道这篇文章有否违背"精"字的原则，就让我们好好的去看看自己的"精"字吧！
