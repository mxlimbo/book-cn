一 遥望星空

　　我相信只要立志考大学的人都晓得这是什么题目，某某年上海的语文高考作文题，听说难得考生哭哭啼啼。在高一时我不畏老师的"恐吓"，一心挑战难度，因此写了第一篇《遥望星空》，由于考题要求写记叙文，而我的编故事水平比中国国家队的还差，因此文章写得不是甚爽，出离了我的风格，于是在此再勾一笔，不过又出离了题目，这是杂文。

——题记

　　星空，我想到了愿望。98年12月份听天文学家说将会有一次大型的狮子座流星雨。于是不少友人想借此机会许个惊天动地的愿望，便邀我去看。其实我本想看一看这转瞬即逝，在万里之外的小光点怎么可以为人达成心愿。不过当时正是初三时，父母厉声喝道："上不了高中，就……"他们每次讲到这里，就没有说下去，留有余地。中国人做事就有爱留余地的坏习惯，不是死而后已，所以才出现了背水一战的典故——把桥毁了，只能进攻，没有退路。也正是因为父母的话留有余地，不知他们要把我怎么怎么样，于是不敢去看。

　　不过心理还算平衡，听友人说看了也白看，整夜天上铺着一层比青藏高原还厚的云层，看到的点点粒粒，也不知道是飞机还是萤火虫。

　　后来还真有人告诉我愿望达成了，听罢有点莫名的不快，但还故作替她高兴的样子问什么愿望。她说是考上高中。听后，心情突然舒畅若仙，不禁笑在心头。虽然是达成了愿望，不过连我在的桂花中学也考不上。技术监督员单从这一点便可推出，那流星定然是假冒伪劣的。

　　不过也有点欣慰，其实许愿的人很少许这种愿。在我看来，有兴致在凌晨顶着睡意遥望星空的人大多数属百无聊赖那一类。百无聊赖者最想过节，因为可以出去"蒲"。但气得他们冒烟的是中国的节日，为了继承毛主席说的"自己动手，丰衣足食"，中国节日少过梵蒂冈的面积。这也因为中国正处于社会主义初级阶段，生产力低下，我们只有以勤补拙。一旦有什么风吹草动，百无聊赖者便领导前卫，勇当急先锋，比蒋介石"剿共"还积极。就拿流星雨来说，百无聊赖者便可借遥望星空为题，与猪朋狗友大做文章。百无聊赖者的愿望就如百无聊赖者一样，都十分平庸，任何人只要努力，都可以实现（庸人除外）。诸如发达、找个女友、得到XXX的签名等等，举不胜举。不过最多人许的愿都如星星的颜色——金。他们都希望不劳而获，腰缠万贯。

　　提到金色的星，我又想起了先前《圆》提过的Voodoo Benshie。Benshie也属于百无聊赖者，酷爱追星，她们坐在演唱会场最后一排遥望明星。明星也因为有众多百无聊赖者遥望，于是才捷足先登"发达"的愿望。如张惠妹在天河秀一秀，进帐已是天文数字，也许正因为他们的收入数字与天文有关，大家才俗称其为明星。

　　歌星并不是搞个天文数字的佼佼者，尽管郭富城一个"祝你百事可乐"就卖了四百万，但仍不敌成龙在好莱坞卷起的《龙旋风》。其实也很想为翻版CD抱不平，正是为了有天文数字的收入，正版CD才贵过山珍海味，是一般的百无聊赖者买不起，于是才钟情于翻版。看见某些艺人在镜头前眼带泪花呼吁"别买翻版"，我便知这一定是个好演员，演技一流。

　　现在也有老人家为明星吵得不可开交，那就是布拉特和约翰松，为的是球星的转会问题。路易斯·菲戈的转会费比天文数字还天文。不过皇家马德里的球迷却异常兴奋，对新球会主席赞口不绝，愿花巨资投入球会建设。可他们好像不知道，这买菲戈的天文数字还不是从他们身上赚来的。作为球迷，由于被中国队伤透了心，也不得不遥望星空，看外国的球星。其实看球星也是许愿，希望球星帮助球队夺冠，希望看到好球。所以中国没有球星，他们没有帮球迷达成愿望的能力。

　　这几年，不少球迷不在满足于遥望星空，因此有人开始摘星。别以为我说中国足坛最无聊的摘牌会，我是说赌波。上回Tiger Cup上，印尼后卫为在世人面前一展其进攻才华，便以精准的努射把球打进自家大门，成为足坛佳话。开始只以为东南亚遭受金融风暴后，民不聊生，为养家糊口，那球员才做出如此惊天动地的事情。后来见香港和大陆有人豪放过李白，不惜千金换假球。还有俱乐部为夺个不值一提的杯子或升降级问题，也不择手段搞了个甲A甲B，不，应该是假A假B颇有名气的"秋交会"。河南建业的球迷无奈之下以对联表示不满——"三陪小姐，假情假意假温柔。中国足协，黑球黑哨黑手党。"看来中国足球的改革，职业化的进程大有胎死腹中的征兆。球员工资不比"星"的天文数字少多少，球却愈踢愈臭，愈打愈假。球迷也开始清醒了，铁杆也不铁了"多几个钱实际，买张球票花费又受气。"球迷便出离假A。遥望西天的球星。

　　现在天上的星星比旧时候少了，其实并没有少，看不到而已。而我们所谓的遥望星空，只不过是在漆黑的夜空中，找一点两点光来安慰自己罢了。
