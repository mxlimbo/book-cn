三 圆圆的缘

　　在《笑我自己》中提到，有个名叫蒋薇薇的女孩，是我的第一缘，不过只是好感而已。时间会冲淡记忆，幼儿园的快乐我忘记了，却还记得幼儿园的同桌，因为缘是圆，没有出口可以逃离。

　　后来，我缘来无运，有时觉得自己像温兆伦在《一生何求》所唱的一句"寻遍了却偏失去，未盼却在手"所以一直痛并快乐着，既无女友关怀，又无女友烦恼。虽没有爱情的经历，但与几位女生也有着不浅的交往。

　　有一女生，我称之为L姐，她也只允许我一个人这么叫她。L姐身高一米八三，不过要穿上十厘米的"电视塔"（松高鞋），但这173也吓倒了不少男生。"L姐"这昵称又两层含义，因为众人尊称其为"Lion"这并非她是lion座这么简单，还因为她武艺高强，不亚于杨紫琼，常将男生来个特训（特殊无人道的教训）而得名，我也常被其特训，由于她口才不济，不如我滔滔不绝地"论它妈的"便对我施以拳脚。也许不打不相识，现在我们可是一对欢喜冤家。另一高见由我提出，L姐身高可以在WNBA占一小前锋位置，可用英国著名出版社LONGMAN来形容, 只不过她是LONGWOMAN罢了，既然LONGWOMAM长似火车，中国人又懒惰，因此简称为"L姐"。 L姐嗓门可与董文华媲美，连年参加卡拉OK，连年抱一"牛肉干"回来，"牛肉干"上写的全是第一名。后来她不参加了，用金庸的话说就是"无敌英雄最寂寞"。 L姐常以为自己说话头头是道，便在别人讨论代沸点时，铿锵中不乏气质的一声，"其实，我觉得……"大家频频点头，如哈巴狗见主人时摇那尾巴。可我每次听其总结后都发觉不少比臭氧层空洞还大的漏洞，不过为了不受皮肉之苦，我也点头了。

　　DREAM是我小学的，但不同班，初中时又被拼在了一起。升高中的一步，她迈到了广师附中。早在小学时就认识她了，是那黑黑白白的点点做了沟通的桥梁，这黑黑白白的点点就市常昊与马晓春最拿手的家伙。其实围棋也可以悟出不少人生哲理，就如钱钟书的《围城》一般，笑声中充满无奈，还教会了我要学会放弃。不过当年在小学围棋班，我可没有过无奈何放弃，我把这一切留给了我的对手。几次把同桌吃得一子不剩。正因为我与DREAM在班上棋艺鹤立鸡群，于是老师叫我俩对弈。两盘下来，我全胜。DREAM之前未逢敌手，却不料连输两盘，差点哭了。从那时开始，我便听明白了一首歌《男人不应该让女人流泪》，男生应该让一让女生，否则她们的泪可以汇成另一个太平洋。上中学后，我发现DREAM很坚强，从不为困难而伤心流涕，很多事她看得很开。当时传闻她早恋了，这事传到某些被封建思想毒害到极点的家长耳中，被这些张口漏风的口瞎说一套，把DREAM当"飞女"唱。这些家长的口不比周幽王烽火戏诸侯的烽火传得慢。某天晚饭，妈突然吐言："你少点和那人下围棋啦！听说她是'飞女'。"听罢，我如置身于土耳其地震之中，还差点引起海啸！本想立刻替DREAM平反，但中国当前是"妈妈只生一个好"，孩子与家长之比是1：2，出于少数服从多数，我只好又学哈巴狗摇尾巴似的点头应道："哦！"此后我的确少了找DREAM 下棋，不是因为妈的一句话，而是因为初三。

　　还有一只FISH与我是老友，FISH与我住得很近，交往很频繁。她像不少女生，都爱看书，不过更有性格，听说一本名著4天OK。效率比李洪志治病还高，不过效果也像李洪志治病，老不见效。见她刚打开《文化苦旅》的扉页，过目自序，便有另寻新欢，又借来两个山庄《呼啸山庄》和《荒凉山庄》，看罢开头与结局，又觉得没有激情，再把莫泊桑的《漂亮朋友》弄到手。她说"只看头看尾，是饱读诗书的捷径。"这使我想到豆腐渣，那设计师定是只设计头设计尾，忘了中间，从而走了建设的捷径，不过倒塌也会走捷径而来。

　　最近，DREAM的一封信中挟着一首诗，听说是她班上一个男孩因失恋而作的，是写得颇感性，于是抄了下来：

　　 缘来是圆

　　 缘是圆
　　 是一个耐人寻味的圆
　　 缘是圆
　　 我在绕着她转
　　 缘是圆
　　 是一个走不完的圆
　　 即使天涯海角
　　 我们仍能相见
　　 缘来是圆

　　也许缘就是一个圆，如一个呼啦圈，套在不少少年的腰上。要用心地摇，有技巧地摇它，圆才不会跌下来；若你停下来东张西望一下，圆就跌下来。转了又跌，跌了又转，不少人从这转转跌跌的游戏中转向了成熟，但也有人从这转转跌跌的游戏中跌落了深渊。我两样也不是，我还没机会玩这游戏呢！
