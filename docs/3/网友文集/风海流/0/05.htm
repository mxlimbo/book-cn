四 遥望星空

　　 到了月饼市场大战的收官阶段，虽如今夜花好月圆，但在佳节思亲时，千里共婵娟之夜，我仍然躲上宿舍，为高考而努力。

　　几个舍友家在市区，便挤上公车图个人圆月圆。平时校园在饭后的嘈杂，今天显得格外宁静。宿舍中只剩下了我和阿超。他是本校为数不多的农村生，在3 + X = 高考的方程中，他的X = 化学，超被尊称为稀有气体。原因有二，其一，成绩彪炳，是学校今年北大、清华的希望。其二，性格内向，永远相信沉默是金，最外层电子为8，化学性质非常稳定。

　　同阿超的交往中，除了请教问题，便艰于从记忆中找到其他事了。因为他的成绩总不出前三甲，我几乎对他五体投地，但唯独不明他床头的一句格言――"我宁可做一颗暗星，也不做恼人的月亮。"为此，我不知抓破头皮多少次了。

　　八月十五的明月已经挂在窗外，又挑起了我的神经，多想问他"为什么？"但又怕万一这是爱因斯坦爷爷之流说的名言，在显得自己孤陋寡闻之余，还要面子上过意不去。

　　"恼人的月亮出来了，去散步吗，奇？"在阳台晾衣服的超以破纪录的方式邀我。

　　"好哇！"我答得很爽，但"恼人"一词又一次让我回到了那句格言的覃思。

　　躺在草坪上，看夏夜的星空，是那么美，那么遥远，我再也按耐不住了："为什么你要做暗星也不做明月呢？为什么你要说 '恼人的月亮'呢？"

　　"看看这美丽的星空，比那恼人的月亮美多了，星空是无限的，而月亮只是一个小丑。我的祖祖辈辈都躲在繁华的都市外，面朝黄土背朝天。而我就成了家中第一个大学生的希望，姐姐在外地打工，父母在家乡种地，都是为了我。不少人羡慕我，可以在省重点读书，但这只是我沾了师兄师姐的光罢了，是他们的优异成绩使我被别人羡慕。"

　　"那和月亮星星有何相干？"

　　"月亮只是一个爱钻出来沾太阳光的小丑，只要光被挡住，月亮就是一个黑影，而且阴晴圆缺变幻无常。星星在一年三百六十五日都会在夜空中守候，虽然它们的光远远不是月光的对手，但它们是靠自身发光而不是沾别人的光，我做人就是要做一个像星星一样，发自己光的人，决不做那些从后台钻出来沾光的小丑。我要用自己发的光报答父母，姐姐。"

　　夏夜的星空，好美丽，八月十五的星空，但大家都在为千古流传下来的主角――月亮叫好时，月亮早已悄悄进入了我的盲点。遥望着一片美丽的星空，静静地，我和超都在静静地寻找，寻找那一颗代表自己人生的星星，哪怕，哪怕那只是一颗非常不起眼的暗星。
