作者简介

　　我——上杉奇也，非日本人，在《风海流》中的《笑我自己》中已经阐明我的笔名的来历。奇也现在只是在校高中生，就读于广州市桂花中学。

　　文集《风海流》是我的处女作。在高中之前，我对文学创作毫无兴趣，更爱看看足球。因此《风海流》中关于足球的题材有点过分。自从认识白岩松和崔永元后，我渐渐发现语言的魅力。在新概念作文比赛中，又认识到韩寒的风趣与文字的乾坤。略读过余秋雨的《文化苦旅》，深沉的感慨感染了我动笔的欲望。再读过钱钟书《写在人生的边上》后，又见自己知识的孤陋，从《围城》的讽刺声中，感到自己语言的乏味，为了可以更好的把文字摆好，我便开始博览群书。在E书中找到众多书目，无奈学习苦闷，压力过大，时间有限，无法精读。

　　 《风海流》写于我高中二年级时，其中大部分为作于2001年的农历新年前。当中各文章文风不一，有讽刺的，也有沉思的。但就个人而言，我更爱写讽刺的文章。张爱玲说过"如果没有幽默天才，千万别说笑话"，我也不知道有否方这样的错误。不少人说我的文章有点像韩寒的文风，这是抬举了在下，可能努力多几个春秋，我才有机会与韩某人相提并论。

　　我其实写文章的题材太窄，找不到更多的突破口，加上自己不太会编故事，记叙文写得很一般。所以《风海流》中的讽刺性散文与议论文还算不错,记叙文就......(我的脸有点红了)

奇也个人简历

　　1983年在广州出生，父母皆为医务工作者。
　　1996年毕业于广州桂花岗小学，成绩一般，升学到普通中学桂花中学。
　　1999年因填报自愿有误，以高过桂花中学50多分的成绩升读原校高中（尚未毕业）。
　　1999年高考作文改革，话题作文开导我写作灵感，无奈未有满腹经纶，作文十分平平。
　　2000年听李忠文（广东作文讲学代表人物）老师讲作文，对字里行间的乾坤兴趣激增。
　　1999~2001年三次错过《萌芽》的新概念作文比赛，于是想以自己写书来发泄。
　　2001年春节前开始涂鸦，少时几日一篇，多时有一日几篇，并把一些自己以前写的有意思的文章与写得还可以的文章收集在一起，编成处女作《风海流》。
　　2001年，新的开始，E书时空要帮我处电子书啦！多谢E书时空。
