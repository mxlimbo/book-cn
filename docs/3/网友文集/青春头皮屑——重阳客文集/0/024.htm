昨 是 今 非

　　我正在太空研究生命悬浮颗粒的感应现象，从地球飞来的蝴蝶带来了两条消息。一是今天是地球上的公元2000年6月1日21时13分，此外更重要的是，那深蓝的带刺玫瑰刚刚通过审核，即将进入国家生物保护区。

　　玫瑰是自己主动申请的。我尊重它的决定。

　　太空的空气的沁脾舒畅，没有风却有一种果酸一样的味道。这把蝴蝶迷住了，它决定留下陪我几光年。

　　第三餐时，(这里不分昼夜早中晚餐，我用一二三餐作为一个循环摄取食物，属于未根除地球人的陋习劣根的无聊行为。)我请蝴蝶喝冷藏已久的醉生梦死酒。

　　蝴蝶醉了，说了不少地球上的往事。它好象回了梁祝里，痛快极了。醒来后却绝口不提梁祝。我问现在是不是夏天，蝴蝶说是的。想起自己许久没回地球了。在太空更需要面对的是黑洞和真空。地球太遥远了。一下子我感到了这遥远，却又开始怀疑自己为什么问蝴蝶那个问题。

　　关于地球的记忆中，花开是不分春秋冬夏的，可花谢总免不了是在夏天。每一次凋谢都有不一样。第一次的时候玫瑰提前暗示我，可直到它凋谢以后好几个月我才领悟到；这一次它为了选择付出了四个夏天的代价，我依旧没能在它凋射之前对它说出我的梦。

　　关于这个梦已积年成久，给它掸去了灰尘以后我发现它已经有了三个破洞。在地球时空里我认为它是最完美最辉煌富丽堂皇的永恒，但事实是现在的它一破了。

　　我开始用介质分解器研究三个破洞，其中一个是在太空上产生的，其余两个是在地球上二万光年以前有的。它们之中又尤其以太空上的破的最大。当我尝试与它对话时，他只说了一句话：太空和地球是不同的，可都能让我们破洞产生。

　　破洞十分寡言，在以后的五千光年我不断努力与它交流，它反复仍是那句话。直到有一天我忽然失眠。

　　那一夜，循着旧路我来到关于地球的梦境，我见到了破洞，它主动向我走过来，说：玫瑰恐怕再也见不到了。这句话引发我又不停地思考，三千五百光年以来我仍然想不出它的涵意。破洞又保持着它一贯的沉默风格，毫不理会我的疑惑。

　　我决心回到地球。飞碟已经发动不起来了，我只好请求破片牺牲一些碎片做启动剂。

　　终于开始进入大气层，蝴蝶远远地飞来接我。它告诉我，玫瑰燃烧了半片花蕊赐与我最后的机会。它把时间定住了，直到我来。血染红了花刺。

　　今天是2000年6月一日，夏天。我开始明白，玫瑰也有眼泪，可以冰冻几万个光年的等待。
