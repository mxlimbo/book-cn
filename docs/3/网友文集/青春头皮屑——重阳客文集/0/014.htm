　　四个多钟头之后，我们来到了号称人间天堂的姑苏城，江南水乡，风景如诗如画，说不尽的风流韵味。江山如此多娇，引五位美眉尽折腰。苏州园林精致典雅，玲珑剔透，实在是一种女性化的十二分纤细的唯美。阿姐阿妹们大家都玩得很开心，一个劲儿地赶着摩登脚步走马观光，拙改园、狮子林、虎丘、寒山寺，虽观者云集却丝毫不沾纷扰吵闹，在匆匆步伐之中我心中奇妙哉地有一种静态的沉淀溶化姑苏山水的独特艺术气质里面。这座三千高龄的古城历经沧桑，勃勃生机的活力中有一种内在的深沉稳重。

　　虽然有幸与诸多青春美少女同游小桥流水之间，心里却因现实而悲哀沉重。只怪兜里的RMB Very Very Poor。古人有言"兵马未动，粮草先行"，如今兵马早行，秋风萧瑟，人丁单薄兼之粮草不足，我真是犯了甜言蜜语三十六计之兵家大忌，想献殷勤都办不到，这一份痛苦又谁能够体会？！只恨自己太冲动，已经和尚追着尼姑跑了，看来只有一条黑道走到尽了。逮着了机会倒束手束脚不能发挥，我好恨丫！补贴了钞票又未能阴谋得逞。失败！真失败！太失败！

　　深深夜色中，我取出这几日的"随军家书"一五封沉甸甸的信。每封都份量很重，说着一大堆狂热草率语无伦次的文字。这是我一连五日的胡言乱语，发自肺腑，出自真心，情真意切而极具穿透力。写的时候情不可抑得感动了自己，不知道，亲爱的小妹，这五封没有地址的信，能否寄到你的心里？在苏州一直没有单独相处的机会让我交出自己的真爱在你的心上称量。每当伴你多一刻，就会爱你多一份；在没有你的每过一秒，我的思念会加深一分。

　　冷静务实的理性已被感性狂热分子彻底俘虏，我被一股莫名其妙的力量牵引着去一些找不着北地干活。电话连打两次后，小妹就老是说正要出门，一大筐事等着做。越过万水千山巧遇她，却只得到Busy Busy Busy Busy Busy的回答败兴撤退。我沉寂了。诱惑还要支配一切，在独自内心强烈挣扎一个月之后，我已成伤心一片海。镜里朱颜都变尽，只有痴心难灭。再也不能这样一半是友情一半是爱情左右为难地活下去了。摊牌吧！无论输赢，无论惊喜或迎头痛击。即使求爱不成，也强过此际慢漫长夜，抱情无门空自怨，生命的充实是因为爱情（痛苦）的折磨？我自仰首狂笑问青天，天无语。

　　我趋步向前，尽量温和地看着她："我有几句话要跟你说，我们到前面草坪上坐会儿，好不好？"

　　在单趟的人生旅程，我只是一名过客，仅此而已。为了成长而渴望，为了相思而疯狂，为了失落而孤傲自赏，为了年轻感伤成熟回望，为了过去大哭大笑过又曾经哭笑不得过，为了生存而拼命伪装，在每一个驿站的交叉口一步步彷徨复彷徨……

　　活着的意义在于不知道活着的全部滋味。

　　活着将会发生什么？如果一切存在都已有了指定的过程和归宿，那每个生命仅仅是一条流水线。模棱两可的东西可为人津津乐道，类似的开始其结果往往都不类似，亲身体验才能在现实感受中消除。

　　失恋的歌有千千万万，听到的感受有万万千千。爱情传说对于我只是传说，对于她，也许是真实的，也许根本不存在于她的童话空间。

　　无味的生存是一种应付，对于一切都只是应付，只不过形式不同而已，敷衍麻木自己和别人。悲欢喜乐，只是一种从众的情绪的需要和结果，在心灵的谷底却没有感到存在，这样活着的人生真谛，只是麻木不仁？现代人的特征难道是迷失自我表里不一？

　　清晨往往是美好的，美好人生又在哪里？

　　我不知道。

　　我怀疑，也许从来没有谁能真正明白生命的全部真谛。
