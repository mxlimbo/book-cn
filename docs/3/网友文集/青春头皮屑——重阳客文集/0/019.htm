文革一幕·寒冷季节

　　我们可以忘记历史，但不能否定过去。 　　　　　　　　　　　　　　　　　　　　　　--题记

　　现在，无产阶级文化大革命运动正以排山倒海之势，雷霆万钧之力，向一切牛鬼蛇神和反革命分子展开了猛烈冲击！一切陈腐落后的渣滓都绝对逃不出灭亡的命运！

　　1967年的冬天，全国人民满怀对党对毛主席无限忠诚和无限热爱和对反党反社会主义黑帮无比痛恨和无比愤怒的高昂革命热情，迎来了轰轰烈烈的文化大革命的高潮！

　　历史将永远铭记这一伟大而神圣的时刻！

　　无产阶级革命造反精神万岁！

　　毛主席万岁！万万岁！

　　鱼梅村的高音喇叭一早便传出召开斗争四类分子大会的振奋消息。一个最忠诚于伟大领袖和革命导师毛主席的红卫兵战士的盛大节日来到了！鱼梅村牛鬼蛇神的丧钟敲响了！

　　广播里传来了激动人心的革命歌曲：

　　"马克思主义的道理，

　　千头万绪，

　　归根结底，

　　就是一句，

　　造反有理！"

　　村民们陆陆续续走向设在村小学操场的会场。迎面墙上贴着最最敬爱的领袖毛主席的巨幅画像，他老人家脸上露出上帝般的微笑。

　　村里红色牌坊刷写的毛主席语录在初冬的阳光下格外刺眼的鲜红。

　　县城专程派了一位"井冈山"红卫兵代表前来领导工作。用她自己的话来说，要向一切反革命开炮，把革命的圣火熊熊燃烧的铺天盖地。她严肃批评了鱼梅村的革命工作，认为鱼梅村的革命还不够深入细致，没有把隐藏在人民内部的反革命分子揪出来。看来这一点她极为不满。

　　这次大会斗争的主要对象是王幼知他爸王可白，罪名是宣扬封建糟粕，顶礼膜拜"四旧"历史垃圾，妄图与无产阶级文化大革命背道而驰。--因为这个文盲农民床头的小木箱里有本宋版《三字经》。

　　王幼知脸色发白地跪在地上，红紫僵硬的手不知道该放什么地方。他今年14岁。去年考初中，数学卷子有几大题是求解多边形长度，别人都拿了直尺量好了答案，他忘带了，结果没考上。这时"革命"的高潮开始了，学校变了批斗场，学生批斗老师。他爸便让他老实在家呆着。

　　冬天的风寒，刺骨。

　　县里来的红卫兵代表好像一点也不冷。她浓眉大眼，乌油油的辫子，像《人民日报》上刊登的给毛主席佩戴红卫兵袖章的女学生，穿一身没有领章帽徽的绿军装，一排亮皑皑的毛主席像章金光闪闪，宽大的武装带紧紧的扎在腰间。显得英姿飒爽，眉宇间似乎透着一股正义之气。

　　王幼知偷偷地看了她一眼，"如果我有个这样好看的姐该多好啊！"他想。

　　王幼知他爸被剪了阴阳头，所谓阴阳头，是红卫兵革命小将的-大光辉发明，使人看其头便预知该人死路一条。他爸现在的模样非常滑稽可笑。

　　王可白挂了牌子在前边跪着一动不动，王幼知和他妈在后排下跪。

　　高音喇叭放着：

　　"不记阶级苦，牢记血泪仇，生产队里开大会，诉苦把冤伸。"

　　一声吆喝："把牛鬼蛇神统统押上来！"

　　斗争会开始了。

　　"乡亲们，我代表井冈山红卫兵司令部向你们表示问候，并向伟大的革命导师和领袖致以无产阶级文化大革命的最崇高敬礼！"女红卫兵的声音悦耳动听，"今天，我们要开一个团结的大会，胜利的大会，同仇敌忾的大会，令反动分子不寒而栗的大会！在我们伟大毛主席的伟大号召下，现在全国的革命形势一片大好，不是小好而是大好！但是在我们身边隐藏好多的阶级敌人，我们要把他们打翻在地，永世不得翻身！党中央毛主席是我们的靠山，毛泽东思想是我们行动的最高指示！无产阶级革命造反精神万岁！"她唰地站起来，高高地举起红宝书，像神圣地举着革命命运的火炬，庄严的脸上红光闪闪："无产阶级专政万岁！"

　　下面人群稀稀落落地应和着："无产阶级专政万岁！"

　　"打倒一切牛鬼蛇神！"

　　"打倒一切牛鬼蛇神！"

　　"毛主席万岁万岁万万岁！"

　　"毛主席万岁万岁万万岁！"

　　"革命有理造反无罪！"

　　"革命有理造反无罪！"

　　女红卫兵成了真理的化身、法律的体现、时代的宠儿，宛如一位光明的战争女神，给大地带来光明、带来火种、带来希望。

　　她解下腰间的武装带，大步走上前，狠狠抽向王可白的阴阳脑袋。英勇无畏的革命小用沉重有力的鞭打将牛鬼蛇神一下子打倒在地。

　　血流了一地。

　　王幼知他妈嘶哑的尖叫一声昏倒过去。

　　诺大的会场一时间寂静得死一般的恐怖。

　　女红卫兵使劲踢了王可白两脚，他渐渐醒过来。

　　"你不交代？"

　　"我有罪。我认罪。罪都是我一个人犯的。不关她们娘俩儿的事。要打就打我吧！"

　　"我有罪，我认罪。……"牛鬼蛇神声音越说越低，渐渐有了呜咽的声音。

　　"哼！一切反动的东西都不会自动地退出历史舞台，一切牛鬼蛇神都要进行垂死挣扎！"混合着阴沉的鞭打清脆响亮地响着。

　　继续是狠狠的两脚，顿了一会儿，又是两脚。牛鬼蛇神睁开了眼。

　　女红卫兵举起武装带指着王幼知，"你是当狗崽子，还是划清界线？"

　　"我，我……"王幼知小脸结结巴巴说不出话，脸色更显苍白。眼前的姐姐像村头杀猪的李屠夫家的大丫头似的满身血腥。

　　"快说！"声音威严提像庄严法庭宣判。

　　"我，我……划清……清……界限。"王幼知脑子里只想："我不要流血！我怕血！不要！"

　　"很好，"正义女神满意的点着头，"那你来打倒这个反革命分子！"

　　王幼知被一只革命化的手拖着站起来，浑身哆嗦，嘴唇发紫，像纸糊的童子。

　　"快打，背叛了反动家庭才是革命阵营的一员！否则，--必将和反革命同样的下场！"

　　会场里悄悄的议论声，嘘声不断。

　　王幼知战战克克地接过武装带，迟疑地、慢慢地举起来，突然在半空中定住了。

　　眼前正义女神注视他的目光像无数的针刺在身上，这张脸一下子变成另一张脸，然后又变成另外一张脸，他猛地什么也看不到了，"扑"的一声倒在地上。

　　正义女神揪住王幼知他妈的头发，让她站起来再坐下去，又站起来。他妈撑不住，眼泪鼻涕一起流下，不住磕头求饶。革命女将毅然踩住她的腰，用锁链痛快地抽打着软绵绵的屁股。他妈吱呀吱呀地叫了一阵子，终于昏倒过去。

　　会场一阵轻微的骚动，有人压着嗓子说，"这丫头真够厉害的！""哎，厉害。""无产阶级专政到底咋回事呢，他二爷？"

　　这时王可白身上传来一种臭味，牛鬼蛇神把屎尿拉在裤子里了。革命女将的将手潇洒一挥，命令把他丢进大队的粪池去。

　　多么火热的斗争场面！多么卓有成效的批斗！多么英勇多么顽强的大无畏的红卫兵代表！任何牛鬼蛇神反动分子，在无产阶级文化大革命面前作垂死挣扎，都无异于螳螂挡道，终究会像秋风扫落叶般地被革命战士打倒在地，永世不得翻身！

　　造反有理，革命无罪！

　　无产阶级文化大革命万岁万岁万万岁！

　　饶舌多嘴的老太太嘀咕道，"作孽，作孽！"

　　第二天从粪池打捞上一具尸体，恶臭冲天，见了的人都不住呕吐。

　　寒风吹在空空荡荡的操场，地上湿润润的，王幼知醒过来，脑子里仍是浮现恐怖的鲜血，他天真而怀疑地想，"革命烈士的鲜血也是那样红吗？"

　　他轻轻爬起来，没惊动旁边瘫倒在地上的他妈。

　　他出去看了牛鬼蛇神的尸体，没哭。又问了女红卫兵离去的方向，一路追赶着。当走到半路他实在没力气了，一屁股坐下。喘了两口粗气，猛地一下子站起来，使出浑身的力气朝着前方一字一顿的粗着嗓子喊着："我、日、你、娘、的！"
