　　与此同时，勾践在苏州度过了一生中最黑暗最不幸的时光，每次在基层体验生活总是上下为难。第一天下文件要增产增收为国家多贡献一份食粮，第二天检查团来考查带走了不少地方土特产，第三天说今年粮食大丰收要平衡物价抑制通货膨胀下调收购价50%，第四天说国家财政透支失暂时打白条，第五天要学习最新指标，第六天又要抓几个阶级斗争典型作批判大会，第七天下达任务人人写一份忆苦思甜的新社会就是好来就是好的总结报告，第七天，要全国人口普查重新办理户口簿粮油本身份证每人交费一百美金，第九天要庆祝国庆大典排演歌舞......

　　勾践每天忙得头昏眼花，感到自己政治才华被完全压抑，得不到半点发挥，还要不时耕田喂马背政治经济学原理，骂道老子好歹也是克莱登大学留学博士，至少也得照顾高级知识分子给个优惠搞个免考什么的，政策整天这么样真他妈的淡出鸟来了。但勾践同志毕竟是具有党的优良传统作风的革命者，想的这些话都跟着大便排出体外，从来不从嘴里冒出一字半句不忠不孝不满的话。并且还不定期地在席梦思上加了层烟叶子，请人用巧克力做个苦胆，卧薪尝胆，弄得老婆说他嘴里全是酸不溜秋的味道，身上是烟灰缸的杂啐呛的人难受。

　　于是传开来说勾践每晚抽一根大烟叶子，读共产主义著作。大伙十分感动，把他树立为榜样。夫差听到这消息特地把他招到府上，让他吃了一顿鸿门雁后走出门不慎被德国狼狗咬了一口......从此以后只有做太监的份儿，勾践连打了八十三篇出差业务报告要求回越地把共产主义思想全面推广开来，夫差只当看笑话全扔炉里烤火了。第八十四份报告勾践说记得越国有个秘密宝藏地带，头愿意发动群众把它找出来献给夫差，夫差也正好猫玩耗子玩腻了，西施又在一旁甜言秘语了几句。于是大笔一挥：同意。望与群众密切团结，切实费彻党的方针政策!勾践终于得以回家。

　　回越以后，勾践见人口稀少，大力提倡多生多育，尤其提出重女轻男，从贵州云南菲律宾进口不少年轻姑娘，大伙都不知他葫芦卖的什么药。又过三年，勾践贴出布告通告联合国，说，越国缺少劳动力欢迎各国男士出国来越国定居，这里风景优美四季如春，气候温和物价低廉国家福利特别好，吴国青年男子一大半移居到越国。同时又设计推出一系列昏君淫夫夫差的八卦秘闻，全国各地四处蔓延，每家每户都恨死双重变态精神分裂的夫差了。接着出版一本《越国的奋斗》号召部队一举攻垮夫差老巢。上下活动，终于飞机坦克大炮冲锋枪一齐发动，胜利地开进了只剩几千人口的苏州。

　　夫差正躺在床上数着西施左边眉毛有多少根，匆匆跑进一人报告说勾践攻陷城地了。夫差大惊，要起身又觉四肢无力，便问，国防部长伍委员呢?报告又说，伍委员长一生为革命而贡献了青春，贡献了生命，上个月爱滋病复发，已经去马克思他老人家那里了。夫差大叹一声啊呀!说话间冲进一彪人马，夫差低头叹气：勾践你来了。

　　只听得一声大喊昏君你睁大眼睛看看我是谁!夫差一看，原来是范蠡。

　　西施立刻扑进范蠡怀里:亲爱的老公，我盼星星盼月亮终于盼到你啦!范蠡对随从说，你们几个赶紧护送我老婆回家，如有懈怠，立刻砍头! 见西施走远了，逼夫差吃下一颗移魂丸，这药令人成植物人状态。又一声报道勾践来了。

　　勾践穿一身范哲思定身设计的成熟男人秋装系列，神色自若地走到夫差身边。说，哥们几年不见，你怎么还没死啊，你以为我真去挖宝给你啊，你个猪脑子，历史现在已经证明你必将淹没在人民群众的汪洋大海之中，现在我代表人民对你进行判决!见夫差神色呆滞却不求饶，说这时候你都不求饶，我KAO!I服了YOU!又看夫差的背微微动了一下，便说，你想求饶啊?你说清楚不就行了吗？你不说我怎么知道你想求饶呢?

　　虽然你很有诚意的看着我，可是你还要跟我说你想求饶的!你想求饶的话我会同意的，你想求饶我当然不会饶你啦！不可能你要求饶我不饶你，说不求饶我偏要饶你，大家讲道理嘛!我不是真的想求饶吧?难道你真的想求饶吗?......现在我数三下，你要说清楚你到底求不求饶......

　　周围的人大吐白沫，勾践问范蠡习不习惯?范蠡说给我点时间我吐啊吐啊就习惯了。

　　勾践本想把夫差抓回去做三级片男主角，谁知夫差呆如木鸡再没点当年的下流风采，便安排他自杀而亡。热热闹闹打道回府。

　　却说西施被一行人送到一艘豪华游轮，不久一架飞机落在上面，走出范蠡。两人于是相忘于江湖，在落山鸡安家落户，范蠡潜心写作了《我和第一美人不得不说隐私》《我和第一美人见不得人的隐私》《我和一第一美人羞于启齿的生活》等一系列百万字作品，久居名人畅销书排行榜冠军。一日看到勾践在《越人日报》上发表言论说要提倡传统道德，把孔教定为国教，要求领导队伍年轻化，向老一辈革命家范蠡同志学习公而忘私未到法定退休年龄就主动让出位置的大家风范。范蠡掰手指想想勾践已连任六十三届委员长，不禁释然一笑，拥着西施又进入真实的梦乡。
