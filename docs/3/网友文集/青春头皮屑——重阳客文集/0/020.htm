无题

　　来是空言去绝踪，月斜楼上五更钟。
　　梦为远别啼难唤，书被催成墨未浓。
　　蜡照半笼金翡翠，麝熏微度绣芙蓉。
　　刘郎已恨蓬山远，更隔蓬山一万重。

　　来是空言去绝踪，月斜楼上五更钟。

　　碧天如水夜云轻，幽梦空记省。静夜思，心难平。枯坐小楼，万般飞絮不招自来，落入无情流水渐远去。八月终南山，花香凉意浅浅拂掠，七七香花般清幽幽的魂灵忽忽悠悠地作别凡尘，永远地离去了，天下间从此只有玉豀生一人孤零零的漂泊凡尘，徘徊复徘徊。

　　晓镜但愁云鬓改，夜吟应觉月光寒。在梦境我却忘言，尘满面，鬓如霜，你也无语，小轩窗，正惆怅，沉默相对，无语话凄凉。七七，可记得，我远行千里，祸福未知，你送问归期，前途惘然两相叹息，你背身过去低语巴山夜雨涨秋池，未曾别离已盼归期。相见时难别亦难，形影相隔奈何堪！
　　
　　远书归梦两悠悠，只有空床敌素秋。月上五更疏欲断，一树碧无情，空言绝踪催断肠。此生何在？庄生梦蝶。秋风寒冽，落叶人何在，寒云路几层。世界微尘里，吾宁爱与憎？！


　　梦为远别啼难唤，书被催成墨未浓。


　　七夕，满天星光，洞房红烛高照。你红妆粉饰，眉黛如画，明眸流盼，红喜金光相映羞颜……七夕七夕，今夕何夕！七七、七七，可堪相约月钩时？你终是远去，生死相隔阴阳难通，只见梦中惊喜不晓梦醒凄凉。方才良梦，泪痕犹存，斯人已消。可恨玉豀生命中潦倒，处处碰壁，抑郁不得意而孤寒独老，更连累你和我苦辛漂泊，红颜成梦，芳香辗落化尘灰，憔悴皱起玉瓷容，盈盈笑语追忆肝肠断。生亦何欢，死亦何苦？勿念世人，忧患实多！我魂魄已碎，仕途之心早断，腐鼠之徒再不挂念于耿耿，红尘过客，匆匆百年之后幻作一抔黄土。我踱着步，踩在无边无尽的虚空和幻灭的苦闷上。烦躁零碎的碰撞搅得我心神不宁，肉体和精神被残忍撕裂的疼痛抵消了失落的空洞。我气喘吁吁，可谁会相信李义山竟会文思枯竭握笔难下？！

　　痴情再深，誓言任坚，全不抵渺无影迹的飞鸿一瞬、沧海无情。梦非人废，更梆沉沉，灯下管笔无来由。何苦违心催诗？七七已经不在，祭文无论如何华美绝伦，凄凉至情，却无半点意义，仅仅是我这个世俗之人的无聊之举。可笑我如此颓丧，可怜自己无能如斯！百无一用是书生，志比天高，命若纸薄！呵呵呵！


　　蜡照半笼金翡翠，麝熏微度绣芙蓉。


　　烛光照着金色屏风，丹青描画的翡翠鸟翩翩欲飞，麝香熏过的幽香微微透过芙蓉帷帐，七七安睡在床上，抿嘴含笑，长长的睫毛温柔无限……我痴痴地凝视着、凝视着。骤然一阵风起，七七消失得不留痕迹，无处追寻。一切淡淡地来，却不能好好地去，我握住的是苍凉的虚空--良辰美景奈何天，梦里不知身是客，又见芳华逐尽流年，自哀自怜空殇情。心断新半酒，看我穷酸秀才消愁又几千。

　　身无彩凤双飞翼，残宵犹得梦依稀，此情可待成追忆，只是当时已惘然。为了功名前程，我踏上茫茫仕途，你从未埋怨，从未阻拦。我在异乡广阔而喧嚣的浮躁中奔波，你总是一如既往地默默支持我，独自承受孤寂和相思。我却放浪行骸陪伴王公达官驻连于青楼红院，吟诗作谜，以乐慰乐。涂鸦什么"直道相思了无益，未妨惆怅是轻狂"！惆怅本非刻骨，轻狂却堂皇有理。才名未就，轻浮先生。回首还望，已然追悔无用。


　　刘郎已恨蓬山远，更隔蓬山一万重。


　　蓬山之难难于上青天，兼程将行，此去无多路，天长地久有时尽，此行绵绵无绝期。欲问孤鸿向何处，不知身世自悠悠。

　　我自以为决心十分坚定，等及走上这条不归路，各种思绪纠葛缠绕心间。宁可让我焚身化为碎片、堕入永无天日的幽灵所在，现在也只有惩罚可以使我心安；但是我半点听不到你的召唤，连仅有的梦也被失眠剥夺干静，渐渐连你的样子也开始在记忆里模糊，只剩下你散发的幽幽淡香，叮当叮当的款款脚步声，艰难拼凑一个残缺的影子。

　　前汉之刘晨，即今世之玉豀生；天台山之仙缘，转长安城之缘起；蓬山之无可奈，梦中人共徘徊。湘波如泪色蓼蓼，楚厉迷魂逐恨遥。天荒地变心虽折，若比蓬山意未多！

　　原有空空，已知空空。欲别空空，终为空空。

　　空来空往尘土中，做得不空闲误空。无空方得空，有空须悟空。

　　何是何非，无是无非，事事非非，飞飞是是。空有我，我即空，空无我，方真空，难得糊涂，我却已忘言。


　　（本篇为剪刀+胶水，混合剽窃+拼凑的试验残次品。）
