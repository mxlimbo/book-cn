乱弹四大天王

　　十来年前，大概在九一九二年的光景，香港有几个有闲阶级在电台天意调侃，选秀选出四位所谓"天王"，尤其小白脸居多，媒体随之跟风，吹吹捧捧起风落雨，煞是热闹。

　　因为天生的自然环境，恶劣不堪得只能当实力派的张学友上气接下气连唱几出舞台剧之后告别单身贵族，跟在老婆屁股后面专心致志带BB，情歌唱腔转成催眠曲调了，其他三位花边绯闻不断兴风起浪的天王清一色是成熟型小白脸。（另注：虽说像刘德华也老大不小了，皱纹深得铺土就能种豆生花，但人家就能用脂粉填抹得平平的，宣传攻势从15岁到35岁"一个都不手软"，全面轰炸。床头贴着大大鹰钩鼻的，纯情女学生大有人在，不讲人家是白脸也太蔑视老偶象的用心良苦了。）

　　至于黎明离富城二位本来就是凭脸蛋混饭吃的标准小白脸，闪光灯下泡上十多年也不容易。黎明从变声走高调低层次入行不懈努力向上，兼之化妆有术保养美容卓有成效，这哥们的广告又唾沫横飞无孔不入，从牛机到摩托车到矿泉水到椰子汁逮着什么拍什么，鼓了腰包又添名气，一个石头砸两个鸟儿，是精明的主儿。

　　富城这娃儿一亮声就是独门独户的太监唱腔，不过既然泰国人妖能红遍五大洲四大洋，小郭这把嗓子也不愁没市场。况且舞男基础练就的花哨功夫十人扎实，蹦蹦跳跳也能捧个活力四射什么的。劲舞劲歌经常一口气喘不下去看得人提着嗓子为他急啊，也教人不忍心打击落水狗。这孩子倒了争气，原本一自生自灭的路子硬是挺了过来没趴下，令人叹服林平之一号人物练就葵花宝曲也血雨腥风的苦水沧大的。不能因为挑逗女同志的录相带就给灭到万劫不复了。慈悲为怀，提携后进青年嘛。总得让落后分子也感受一下民主大家庭的温暖。

　　天王现象是港台俗文化的娃儿，和故事相传的"十景"心态。都牵扯到广大人民群众的劣根性，偶有几位专家学者嫉恶疾呼，却成螳螂拦道、寡不及众。即是四大天王退出娱乐界，新的四大天王肯定又会冒出来，还有什么四小天王、十大明星、四大美女之流。倒也一片混乱之中显得人才济济精英辈出，颇有歌舞升平的味道。很能点缀一些社会名流大众代表的高雅品位。
