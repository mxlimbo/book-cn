爱

爱
期待
却无奈
彼此伤害
魂飘云天外
许你一个未来
双手将天空打开
轻轻擦拭你的尘埃
罗兰花在我心头盛开
纯洁不能永远与我同在
也许要把灵魂一点点出卖
该怎样付出把你的一生负载
星光诱惑了青春真情已经不在
就让魔鬼嘲笑一个无能者的失败
忘记你快活地死去会有多美妙可爱
成功的意义就是愉快地接受失败
没有了躯体的灵魂在那里存在
感情已令我的痛苦无限超载
灼人的温柔哪里可以出卖
就像影子提醒你的存在
折磨永远不把手松开
神经飘着年华尘埃
别让天堂门打开
我软弱的未来
你消失窗外
宁愿受害
无可奈
等待
爱
