清晨窗户的玻璃

　　我静静坐在窗前
　　空气中漂浮着放荡和慌张
　　玻璃上有张忧郁的脸庞
　　这张脸在想念一位遥远的姑娘
　　
　　记忆的灰尘呛的他不断难过
　　在她面前无论历史或者未来都不免受伤
　　
　　我安详地注视玻璃中凝想她的目光
　　一边前进一边隐藏
　　
　　岁月将要流走过往
　　我笨拙地尝试学习解脱
　　
　　让风吹来的地方就是那个方向
　　令我对整个世界蔑视放荡
　　然后他倏地消失在晨雾里
　　混入无知的无聊的无奈的波浪
　　 --这该诅咒的阴谋的海洋
　　
　　清晨的阳光唤起紫罗兰苏醒的芳香
　　玻璃上的脸慢慢上升
　　庄严地看着胸口不断沉降
　　渐渐死亡，安详
