城 市 独 白

　　城市在夜空下喧嚣着，在霓虹灯的光亮与混着汽车尾气的黑暗之间，我穿梭而行。"你说这城市很脏，我觉得你挺有思想。"空气中漂浮着张楚的吟唱。这城市张扬人的欲望，却压抑曾经和未来的梦想。许多路横在前方，已经选择了不止一个方向，在风化了的哀伤和袅袅漂起来的沉寂中，我发觉自已和城市一样彷徨。纵使灯红酒绿，走出夜总会仍不禁心伤。

　　所拥有的仅仅剩下单身的理想--我已不追求最美好的故事，在路上我已经遗失磨殆了软弱的执着的纯洁，在被污染的时刻自己也涂抹了黑暗，正如眼前灯光背影中隐藏的黑暗。生活，只要求给一个封闭的空间，任意自由。

　　或许太过沉闷了，我几近忘却阳光透过窗帘，扰醒我的梦也带着一分温暖。但我已习惯夜色，光明在前，只看注视它身后的阴沉。人生是一出悲剧，这不是由我来安排的。"对于无法抗拒的生死轮替，我们只须漫不经心。"

　　但世界是足够的广阔，生命是足够的丰富，欢乐永远都不消失--并不是只有我的失落，也会有我在失落中仍藉望的她的灿烂，温柔似春风，生动如彩霞。

　　冲一杯浓浓的咖啡，品味深夜。苦涩的味道尝久了，会有一丝的甜蜜。
