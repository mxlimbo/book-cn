重庆森林

　　重庆森林，挪威森林，王家卫让人想起村上春树。一样几乎是絮絮叨叨地描绘一些琐碎的物品.整个篇幅都象一种个人的独白.意识流是不好懂的.而王家卫把它搬到了屏幕上,这种搬迁并不让人觉得厌倦,迷蒙,你不知不觉地跟着他的思绪在走,你永远也不会明白下一秒钟会发生些什么事情.个体在这里彼此独立,人和人之间,永远都相距那0.01公尺,而正是这0.01公尺拉开了我们的距离.我们彼此相望,却无法触摸.就停留在这个距离好了,它让我们安全.生活的表面充满了浮丽的掠影,而深层次的都是碎片.有人一片片地想把碎片拼成美丽的图案,快拼好的时候,它却 "当啷"一声掉在地上.所有的人都在自言自语,人与人之间隔着这让我们安全的距离遥遥相望.

　　晃动的时间让我们不安.223说:沙丁鱼会过期,凤梨会过期,所有的东西都会过期.我不知道这世界上有没有东西是不过期的.我们希冀时间可以在这里凝固.那些瞬间,我们彼此相爱,彼此拥有,彼此呼吸着同一种呼吸,那些瞬间就如包装起来的罐头,它也是有保质期的.过了期,就只能为人们所摒弃.223一口气吃下了30个过期的凤梨罐头,想去爱自己失恋后看见的第一个女人.另一个623对布娃娃说话,对牙膏说话,对毛巾说话,对空的啤酒瓶说话,他们喋喋不休,可是无人倾听,我们都在失语.那种失语,是无人倾听的失语.这种状态缠绕着每一个你身边匆匆走过的路人.他们行色匆匆,他们无法交谈,只能自说自话.

　　阿菲则是一如她唱片中的形象,另类乖巧得让人生疼.她爱623,梦游一般地去为他做各种各样琐碎的事情.鱼儿在游泳,这种飘浮的爱就如鱼儿自由地游来游去.而当623最终转过身来的时刻,她却因恐惧而逃跑.逃跑是安全的.家卫的人物都很爱自己,他们对未知的可能的伤害都采取了断然拒绝的态度.这种拒绝,毋宁是划了一个圆圈,想让自己安安全全地活在圆里面.她自由地在空中飞翔,有一天,当她落地,回到男人的怀抱,那一刻,她微微笑着,笑得令人生疼.聪明的家卫没有再做下去,落地的飞鸟在人群中总会有快乐和悲伤,她会给伤害,也会给温柔地疼爱.我们无需去想下一步的故事,生命由断裂的碎片构成.感谢家卫,他在这片森林里讲述了无数破碎的离弃,忧伤,孤独,失语,错位,最后,不知道他是不是因为自己都在为这片破碎而流血,他想了想,让那飞翔的鸟儿又飞回,雨停了,她该沐浴爱的阳光.
