且听风吟


　　没有十全十美的文章，如同没有彻头彻尾的绝望。村上春树在这篇小说里以一个28男人的身份来叙述20岁时发生的事情。“直言不讳是很困难的事，甚至越是想直言不讳，直率的言语越是遁入黑暗的深处。”失语是他首先提出的命题。我们都在尽能力地诉说，想尽力地找到对应的诉说终点，把语言的桥梁架到那里，从此我们的生命也有了落处。“我”在14岁以前患着不可救药的失语症，是个沉默寡言的少年。精神病医生说：文明就是传达。需要表达，传达之事一旦失去，文明既寿终正寝。写书的哈特费尔德生性抑郁，没有朋友，母亲是他唯一的依赖。而“我”从前的一个女友则无声无息地把自己挂在了树林里,没有理由.她从来没与我讨论过自杀的话题,语言在这里已完全成为多余,寻找一种合适的表达方式成为文明的悖论。

　　孤独是第二个悖论.玩世不恭的鼠终日泡在酒吧里,写着幻想中的小说.“我”在酒吧的地板上发现一个醉倒在地的少女,于是送她回家.少女出身于单亲家庭，独自居住.有了身孕后独自打胎,没有人知道那父亲是谁,为何离去.送“我”加利福利亚唱片的少女神经衰弱,进了精神病院.至于“我”自己,则在凌晨时分钻到厨房里看坏了的冰箱.孤独与失语往往是并发的.善于在电台,屏幕上进行各种各样的表达的人们走到黑暗中,睁着怀疑和冷漠的眼睛,喃喃无言.爱情是一种疼痛,忘却是一种疼痛,怀疑,疏离,绝望,都是一种疼痛.那个法语系的女孩子无声无息的死亡让“我”陷入了一种不知所措的茫然境地.这种离开是一种绝望到极致疼痛到极致的选择,可是,“我”永远也不会明白她为什么会作如此选择.这是一个永恒的疑问,“她”让我对自身及世界产生了完全的怀疑.唱片店里的女孩与我有短暂的邂逅,“我们”言语不多,却能从怀疑走向信任.她带给我抚平伤痛的希望,夏日的气息,海潮的清香,遥远的汽笛,女孩肌体的感触,洗发香波的气味,傍晚的和风,缥缈的憧憬,以及夏日的梦境.可是,“这一切宛如一度揉过的复写纸,无不同原来有着少许然而是无可挽回的差异.”一样地哭泣,叹息,一样地渴望着死亡,暗夜里两人拥抱,想从彼此获取一种温暖,可以抗拒窗外的寒冷.性是一种本能,聪明的村上在这里没有选择性.两人拥抱,仅仅是希望一点温暖和信心.疼痛仍在那里,死亡的阴影仍在那里,“我们”只是两只孤独的兔子，寂寂地等待着天明.成长的路上充满了伤痛与别离.加利福利亚少女的不知影踪,法语系女孩把自己挂在了树林里,卖唱片的姑娘在人的长河与时间的洪流中无影无踪,“我”徒坐在海边,想哭却哭不出来.“假如所有出色的女孩,全是加利福利亚的.”歌声响起的时候,她们年轻的面容因为别离而永恒存在,美轮美奂，活着的,存在的人都在岁月中渐老,只有她们永煲青春,花开不败,这种选择,原来也没什么不好.
