想起了一些故事传说


传说中
西施以自己的美貌为武器
征服了一代君王
无盐也有超人的智慧与勇气
也让她流传千古

女子 美丽和不美丽的
都有千载传唱的故事
美人的笑容可经倾国倾城
范蠢仍然承受了思念的苦楚
因为他是
聪明 有才华 富有牺牲精神
丑女的才智让她做了国母
君王身侧
却是无数宫娥 言笑晏晏 观之忘情

千百年来人们无法忘记你们的故事
其实 如果可以选择
你会情愿只是名浣纱的女子
为他织一条可以拂去风尘的汗巾
或者 山野里的农妇
粗茶淡饭 闻每一缕黄昏的炊烟

写诗的女子想着那些无尽的期待
神情有些恍惚
所有的传说原来都只有一个主题
所有的诗篇都只有一句：
上耶，我欲与君相知，长命无绝衰！
