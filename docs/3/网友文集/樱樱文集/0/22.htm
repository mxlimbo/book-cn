樱子的歌唱


机器轰鸣声中樱子在歌唱
机器轰鸣声中樱子呆呆地写着诗行

麦田里 稻花香
樱子的头上长着透明的翅膀
樱子的笑容在飞翔
春天里樱子在歌唱
一寸一寸用心在歌唱
秋天来了樱子呆呆地写着诗行

黑夜来临时樱子有些忧伤
忧伤的孩子仍然在歌唱
忧伤的孩子亲吻着自己 亲吻那无尽的月光
黑夜来临时星星是樱子的情人
星星有时纯洁地微笑 有时又有些放荡

太阳出来时樱子有些疯狂
疯狂的女子在大声歌唱
疯狂的女子眷恋着温暖 眷恋那火辣辣的阳光
太阳出来时樱子的眷恋也绝望
绝望的樱子吐出自己的心祭奠自己的热爱
没心的孩子微笑着沐浴阳光

机器轰鸣时樱子在歌唱
机器轰鸣时孩子的歌声在流浪

荒野里 一地清凉
