镜里的天堂


　　她的家里有一面镜子。那是一面青铜做的，很古老的镜子。母亲说，在她的曾爷爷的曾爷爷的更曾的时代，这面镜子就已经存在于家中的某个角落里，直到如今。

　　虽然古旧，岁月却不曾在镜面上留下伤痕.暗夜里,它仍旧闪着隐晦的光.镜嵌在一个木质框架里,木框上雕刻有许多镂空的图案,镜的正上面有一个小小的蛇头人身模样的画像,身子是一个裸体女人,而头做成了阴阴的蛇形,披散着满头的长发.这面镜子一直放在阁楼上.那天,她上楼去找一个母亲旧时用过的滕箱,找开箱子,就看见了它.箱底已让虫给蛀过,镜仍是完好无缺的.她不自禁地把镜子拿了出来,凝视着那个半人半蛇的图案.她向母亲讨那面镜子.母亲说,这面镜子不知什么时候就在她的家中了,从前,奶奶曾经告诉她,这是面很奇异的镜子,奶奶说它可能会给人带来好运,但也可能是噩运,母亲就把它收了起来,如同奶奶曾经做过的那样.她并不相信这些好运噩运的故事,她只是想要那面镜子.镜子最后终于到了她的梳妆台上.那天夜里,她迷迷糊糊地打了个砘，梦见她迷路了,走到一个很陌生的地方,那儿有大海,沙滩,岸边盛开着各色奇丽的花朵,天边有大朵大朵的黑云,海上,黑色的浪潮涌动,黑色的鸥鸟,黑色的云和浪,只有沙滩闪着银白的光茫.她头晕目眩,独自坐在岸边的岩石上.那是个冷清而美丽的世界,她惶然四顾,看不见一个人影.恐惧攫取了她的心.这时,远处走来一个老人.他穿着黑色的长衣,对她很和善地微笑.她问老人这是什么地方.老人说:这地方没有名字,这里所有的风景都是人类的梦境,所以,它非常美丽,在这里,什么样奇怪的事情都可能发生.老人说在这里,她可以四处行走,看看周遭的景物,前提是她不能用手触摸任何东西.老人的面容忽然变淡.她醒了.睁开眼,看见窗外头顶上半轮银色的月亮,月光正照在她裸露在外面的手臂上.屋里有什么东西闪闪发亮,吸引着她的眼睛.她转过来,看见了摆在梳妆台上的那面镜子.镜子闪着一缕幽幽的冷光.她不由自主地起床,站在镜子前,看着月亮清辉下自己的脸.镜里的她面容却忽然变得模糊,镜面也变得模糊,她不再看见自己,镜里的风景在不断变换,她有些诧异,镜子不断地变大,变小,时而又绽放出大朵的红花,一些人影纷纷呈现,透过镜面,她看见一个全然不同世界的风景.她看见一座城市,那是一座夜里的城市,秃鹰在上空盘旋,整个城市给一层浓浓的黑雾笼罩着,她走在街道上,身旁的人都匆匆忙忙地奔走着.而在各个饭店,宾馆和酒吧里,人们举杯相劝,笑里藏刀,觥酬交错的喧哗充斥着整个城市.而在各个角落里,男人们紧紧拥抱着女人,说着美丽动听的语言,他们亲吻,做爱,在大汗淋漓的喘息声中他们的尖叫声划破了夜晚.她急急忙忙地逃离那座城市.路上有很多人与她同路,他们互相交谈,她可以听见他们在谈论物价,孩子,房子,升学等话题.他们说:钱总是不够用,需要去借钱买房.孩子升学需要送礼,而工厂快倒闭了,那些没有倒闭的,也面临下岗的威胁.他们叹气.有人在打麻将,麻将的哗哗声在夜里流动.他们的叹息声凝成了城市上空的黑云,驱之不去.她继续走着,这时候,她看见很多的孩子,他们都很瘦弱,穿着洗得发白的不干净的衣服.有些孩子告诉她他们准备去做零工,挣点学费钱,有些则拖着长长的鼻涕跟在父母的后面,看到有人过来,就跑上去拉着那人的衣角乞讨，他们跪在别人的面前,露出可怜兮兮的笑容.有些孩子到了人贩的手中,他们某个器官被割去给远处某个医院里躺着的富人,有的被卖到很远的地方,将再也不能与父母相见.夜很长,远处,隐隐传来狼的嚎叫.她很惊惶地走着,匆匆地奔跑和逃离.不知道过了多久,耳边的喧嚣声忽然沉寂下来.她发现自己已走到了海边.这时,凌晨的柔光正暖暖地将世界照亮..那是一片完全不同的世界,很纯净,是她在路上所没能见过的一种宁静和安祥.晨曦透过层层的云雾,温柔地投射在无际的海面.白色的沙滩上绽放出无数美丽的向日葵，它们自由地生长,沙鸥快乐地在海面上盘旋和歌唱.银色的浪花拍打着海岸,奏响了一首动听的晨曲.她回过头.忽然间,她发现来时的路已无可寻找,仿佛那一条路,那条有火山,地震,孩子的哭泣,老人的叹息,那条充溢着鲜血与黑暗的路已随黑夜隐去,太阳就要出来,她就要看见阿波罗驾着他的战车重新出现.而阳光终于如约而至.它慢慢地爬升着,海上鲜血淋漓,阳光正在痛苦地喘息,想挣脱黑暗的拥吻,一点一点地诞生在这片土地上.她看见沙丘上的鱼儿开始舞蹈，张开嘴,“欧欧”地吐气,呼吸,金色的鳞片在阳光下闪闪发亮.它跳跃而下的姿势是死亡,空气中弥漫着新生的气息,死亡的气息.阳光和沙滩在热烈地恋爱着,岩石是紧紧拥抱的情人,她见证了他们拥吻的深沉和欢乐的笑声.空气中还有岁月的味道,向日葵流下欢乐的泪水,她情不自禁也跟着流下眼泪,每一滴泪水的落下,都变成化石,凝固在脚下的沙滩上.她想:噢，这就是传说中的天堂了.她几乎不再记得那一夜的奔忙,惊惶,那片黑暗仿佛已沉入某个夜晚的噩梦.而这是个安静的和平的世界,这是个安全的,属于甜梦的世界.她在海边支起画架,开始描述这个世界.她画得很快.每天,在太阳落山时,她也坠入沉沉的梦里.而她会随太阳一起苏醒，在浪花的歌唱声中开始第二天的作画.身旁的风景是不停变换着的.有时,沙滩会变成一片鲜绿的草原,上面盛开着各色奇丽的花朵,它们在阳光下热烈地交配,羚羊和兔子在草原上欢快地奔跑,风从远方吹来,带来满天的杨花飞舞.牧羊人的歌声也在远处悠悠地飘荡.有时,她会看见一些有着洁白羽翼的安琪儿,他们飞翔在蓝天白云中,背上的翅膀轻盈地沐浴在阳光中,每一根羽毛都闪闪发亮.她很忠诚地画着,她从来不吃任何东西,她根本就不需要吃喝拉撒这些尘世的需要.她的皮肤现在变得透明而光滑,和阳光磨擦着,温暖而润泽.她画大海里的孤舟，上面的帆饱吸了海风,正精神抖数地准备远航.她画鱼的舞蹈,画它们飞跃到空中的热吻,她画那些蛙声,嘲声,浪涛声,花儿的笑声,小草吱吱喳喳的说话声,她画岩石的沉默,它们体内总蕴藏着一个久远的故事,这些故事让它们化成了凝固的坚强.她画阳光,太阳是有时是黑色的,有时又是血一样的鲜红.她肆意地泼洒着浓墨，她什么也不用思想,这个世界太过于美丽,她要把它们都留下来,留在自己的画布上.她记得从前的那个梦,梦里的风景就是这片大海.出于敬畏和恐惧,她从来不去触摸这里的任何东西.有时,小鸟会停留在她的头上叽叽喳喳地啼叫,海风也吹拂她的发梢,几滴露珠亲吻她的面庞,变成一粒粒的珍珠留在发间.她只要想起那个夜里走过的无尽长路,就越发执著地要为这个世界作画.如果有一天离开这里,我也可以打开画布,看见我曾经来过这样一个唤为天堂的世界,她想.日子慢慢地过去.有一天,她正在作画的时候,从沙滩深处走过来一个青年.她抬头看他.青年站在离她不远的地方,他注视着她,目光里有几许熟悉,却又是分明的陌生.他微微地笑着,不知怎的,她想对他笑,她觉得他的笑容很温暖.忽然,她的心脏感到疼痛.那眼神,照亮了她内心里一片很隐蔽的角落,那里曾是黑夜漆漆,连她自己也不知道自己究竟生长了些什么.而于刹那间,所有的景物都为之黯然失色，只有那道目光,仿佛洞穿了她的前世和今生.她仿佛早就认识他,知道他.她仿佛走了很远的长路来到这里,就是为了等待他的到来.她迟迟艾艾地站起来."Hi,你是在这里吗?"她听到他的声音.他沐浴着阳光,神采飞扬.而她却透过他身上的阳光,窥见了他身上隐藏的黑暗,在那里,曾有过挣扎的热情,欲念,她看见他的野心勃勃地跳动,是的,他从尘世中走来,旧世界的影子映着他的心灵.她仿佛生来就知道他曾有过的苦痛与创伤.他不过是个自私自利的青年人,有着很多梦想,有些梦碎了,有些梦又拼合.他一路跌跌撞撞,无意中闯进了这个世界.他也许并不知道她生来就是为了在这里等待,他可能也是从那个城市中走来,而他只是一个路人,终将离开.这个世界,如此美丽的世界,天堂般的世界,不过是个梦影罢了.他将离开,而她自己,除了画出的这些梦的影子,还可以留住什么?她可以往何处去?那一瞬间,她心里转过千百个念头.而他只是看着她,微微地笑.他向她走来.他仿佛看清了她燃烧着的痛楚,相逢的喜悦,等待的漫长.她等了太久,走过漫漫长夜，见证过一路的尘世忧愁和哀伤，那些流血,暴力,瘟疫,战争,旱灾,洪涝;那些谋杀,强奸,贪污,行贿,抢劫,那些流言蜚语，中伤,嫉妒,贪婪;那些的愤怒，麻木,冷漠和嘲讽的情感填满了整个黑沉沉的夜晚.她好不容易终于到达了这里,也等待到了他的到来.一种巨大的幸福和忧伤感淹没了她.他向她走来.他伸出手,他的眼角有些疲惫的皱纹,她想他需要好好地休息..她会请求他不要离开,他从她的画中走出来,她再也不需要作画,她只要真实地拥有这从天而降的幸福.他伸出手.她看着他,心里一阵眩惑。

　　她终于伸出她的手.晨风中,她伸手的姿势有些可笑.她看着他的脸.他在微笑.她快触摸到他的手指了.而她什么也没有触摸到.忽然间,整个世界都在颤抖,幻成一片光与影的碎片满天飞舞,她头晕目眩,只听见一片“当啷”的声音.有些东西在破碎,那瞬间,她感到一阵撕裂般的疼痛.她好不容易再睁开眼睛,镜子已化为碎片,四处散落在梳妆台上.那个半人半蛇的雕像落到了地上,蛇头已和身子分离,蛇的眼睛鼓鼓地看着它,两道鲜血从那里面流出来.月亮映着她惊愕的面容.
