他们说我们终会相遇


你相信吗

他们说：在活着的年代里
孩子热爱阳光 孩子歌唱理想
孩子有圣洁的瞳孔
孩子们穿着五彩缤纷的衣裳

在活着的年代里
我路过这些孩子
他们让我泪满盈眶

活着的年代
我用泪水和世界说“hello"
那一刻
哭声嘹亮
黑暗 鲜血 生命在疼痛中诞生
母亲轻了 世界重了
我是一根肋骨
我要寻找故乡

活着的年代
人影绰绰 鬼影绰绰
高山 河流 大漠 流血的太阳
我从金沙江上来
沿江而下
一生的路
漫漫长长

我活得很长
这长长的一生
我爱上了一个影子
忧郁的目光 不合时宜的长相
在人群中踉跄
有时候
一只狗
都可以朝他“汪汪”

不，也许在活着的年代里
他衣着光鲜 容颜娇艳
他微笑 痴笑 假笑
人们说：那是一个好青年
上帝都嫉妒得疯狂

我活得很长
这一生
风吹皱了我的青春 雨压弯了我的脊梁
有一天我是一名老妇人
悲哀地走地城市中
悲哀地将往事回想

我不能与你相遇了吗？
我责问苍天
我满面皱纹 白发苍苍

我走出活着的年代
有一天我已经死亡
他们说我们终会相遇
你终将来将我拜访

有一天 有一天我的墓头长满青草
有一天 残阳 孤松 远处 一只寂廖的狼
在这幅美丽的图画里
你的灵魂穿过墓茔
一阵狂风
吹得画面吱吱作响

有一天 我们终会相遇
有一天 我已死去百年

他们说：女人就是无聊 充满幻想
