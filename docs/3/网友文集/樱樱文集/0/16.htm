朋友

　　一个闷热，没有阳光的下午。

　　窗外的梧桐静静地伫立着，一丝风轻轻地吹过。寻呼台小姐说：对不起，小姐，上午没有你的信息。

　　放下电话，转过头，朋友站在我身后：我早就知道不会有。

　　我看着她，她美丽的长发披散腰后，很美，而现在，这种美丽，有点刺痛我的眼睛。我没吭声。

　　事实上我无话可说，整整一上午，她眼见我傻乎乎地拨电话时，就在身后淡淡地不屑地笑着看我，那时，我们还在文殊院，她喜欢这种地方，早年就曾努力地让我去明白一些事物的不可测性。而愚顽的我总是拒绝接受。坐在那只老乌龟的头上，我对她说：千年乌龟万年鳖。她就笑。那时，我心里有些快活，就对她笑，看她四处地瞧着那些老佛爷，我就打呵欠，她说：人不可做亏欠人的事儿。她相信因果报应，前生今世，而我打着呵欠说：我没有追求，我来这儿只是为了一餐斋饭。于是我傻傻的坐在老乌龟的头上，瞧她去摸那佛象的头，她说要祈求好运。我却睡眼蒙胧。

　　然后我们坐在那儿喝茶，淡淡地聊一些天，深的浅的过去的末来的打算，梦想，我没有谈很多。听着她说起想离开那个偏僻的小城市，前天我就对她说：我只有一句话，四年的青春已经过去了。你还在想，考虑，反反复复地思考，一个女人的青春岁月中，能有几个四年？

　　其实是我已经有些厌倦了这个话题，四年前我就开始听她说同样的话语，而四年后，生命中太多的往事都如风吹去时，仍然在重复着同样的话题，四年里春花开过夏雨洒过秋月哭过冬雷也阵过，我一次次地听着同样的话语时，早已经有些厌倦了。去年中秋时，她也在这儿，那时，我就说：Action speaks louder than any voice.今年，我连这个都懒得再讲了。我想自己是个相当的没有心的人。她将我当成最好的朋友，也许我走过的路激起她对自己的挑战命运的欲望，而我，却有些厌倦了。。。

　　我们在文殊院的饭堂里等了整整半个小时，才等到开饭。去买牌时，我好奇地瞧着菜单：回锅肉，辣子鸡丁。就问：咦？这儿也有肉卖？人家笑着答我：哪里是的，这都是豆腐。哦。。我恍然大悟地点头，人欺骗自己到这个份上，也真够意思了。原来老佛爷也是有着凡尘欲望的，不过是想着大鱼大肉，吃着豆腐白菜，然后告诉自己：这就是鱼肉。忽然想老佛爷如果想女人的时候，是不是就会对着菩萨多跪一会儿，磕几个响头，痴痴地念几句经？或者是恨恨地瞧那些非同性香客一眼，想着她们不过是一具臭皮囊徒裹一堆污血浓肉尔？

　　我这样想着这些大不恭的念头时，脸上就露出了一种奇怪的神情。她注意到了。嘿，你又在想什么？答：我想吃老佛爷的豆腐。她用不算愕然的愕然目光注视我。多年朋友成亲人，我在什么情况下会说什么话，哪怕是常常是最神经质最无聊的甚至是最不淑女味儿的言语，她都只能这样瞧一眼我。如同我看着她披散满头如丝般的秀发，穿着长裙婀娜多姿地走在街上时，也明白她不过是有着和我一样的念头，不过脸皮较厚的我常常会就夸张地把它表达出来而已。我不是淑女，下辈子都不是。这让我想笑。。。。

　　从文殊院回来时，已是中午，继续打传呼，其实如果不是因为她，我早就放弃了，其实我这时只是想要一个答案，虽然我知道这答案一定会是什么什么的。三年前她对我说：那个人不昨的，不过是你把他夸大了。后来我离开了那个不昨的的人，或者是，不昨的的人离开了我。这都无从深究了。而她，三年前我第一眼看见那个男人，也觉得那是个不昨的的人。我却没有吭声。只是听着她常常地对我讲那个人的没心没肺的故事，三年后了，她还在讲，讲起那个男人她就绝望。甚至不肯对人承认自己有了男友，却离不开那个人，只不过是自己不能支撑自己的心和命运。心需要支撑。心需要安慰，心需要同情和温暖，我在支撑不下去的时候我喜欢自虐，而她在支撑不下去的时候喜欢有一个男人的肩头，即使这个男人她一点也不喜欢。所以自虐到今天的我变得有点没有女人味儿，而依赖到今天的她却是最有女人味儿的女人。

　　电话铃声终于响起。是他，关于这一上午空拨的电话他肯定会有令人满意的解释，放下电话我对她说：抱歉，你的东西拿不到了。这时她就不屑地看着我，我想她在笑我没名堂的举动，前天她就曾告诉我我现在的行为和三年前无甚分别，而傻乎乎的我还举例证明我不是，证明我现在是长大了的女人，不再是那个纯情得捏出水的小女孩儿，小女孩儿可以犯错误，成年女人却会尽量避免，否则还不如一头买块豆腐撞死算了。

　　她出去了，我又拿起电话来查台，得出那个结果时，我看见她已回来，把结局看得一清二楚，这时我心中忽然无名火起，可是却不是因为她，而是因为自己，我和她真是天生一对，从来都把彼此的缺点错误瞧得再明白不过，我感觉到悲哀。

　　我对她说我要走了。不去车站送她了。她好象并不惊讶，虽然肯定会十分失望，去年她走时我就坚持不去送她，理由是我自己来来去去，都从来是一个人就这样地走过，我自己孤孤单单地走过了春夏秋冬，就恶毒地不肯去让朋友也感觉到一点温暖，我记得我曾找出过无数的理由证明我十分忙，脱不了身，而今年，我连理由都懒得再去找，我只是说：我想走了，对不起，我不能送你了。。

　　其实我是想说：送走你，送走你之后，一个人回来，孤单的回来，我不想一个人回来，我不想感觉寂寞。孤独的人是可耻的，我想我常常是可耻得出奇。

　　没有你的日子，我会感觉到寂寞，只为我知你是真心爱我的朋友。

　　没有你的日子，我知道你也会寂寞，只为我是最明白你可以支撑你的友人。

　　可是我帮不了你，我亲爱的朋友，你所有的问题，你所有的困惑，矛盾，痛苦，我都帮不了你，而你也帮不了我，我所有的痴傻无奈，你也只有瞧着的份儿罢。我们都已经长大了。

　　你曾对我说：我喜欢你那时的大眼睛，很亮，很明，很可爱的笑脸。

　　拖着重重的脚步，走在来来往往的人群中，我的影子已经不见了。。。消失在人流中，你再也见不着。

　　风吹我面，我慢慢地走着。忽然间，泪流不止。。。。。。
