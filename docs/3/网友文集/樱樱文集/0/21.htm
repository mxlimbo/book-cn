不曾相遇


　　夜里，骑车经过那小小的牛奶摊。就看见那姑娘在笑了。

　　很秀气的女孩子。笑得很好看。拖着疲惫的身子，天天地经过这个牛奶摊，看见她的笑容，就想忍不住地停下来买她的东西。

　　她把酸奶用干干净净的小袋装起来，递给我。然后看见我自行车里的大包的书。

　　“姐姐，你上夜课啊？“不太标准的普通话，却软软地好听。

　　“不。”我说，仍然很疲倦。日日地在夜已黑尽时，穿越半个城市。这时候，我总是很疲惫的感觉。总想可以找个地方坐下，喝一杯暖暖的水。我开始迷上了喝酸奶，喜欢那种酸酸的感觉，骑在车上，一点点地啜着那乳白的汁液，有时候精神会很恍惚。

　　“你是干嘛的？”“我就是卖牛奶的啊。”我讶然，自己问出了个很弱智的问题。而女孩长得很乖很甜。莫名的我有种心疼的感觉，这寂静的夜，昏黄的路灯下，寂寞的卖牛奶的女孩，孤单地日日停留摊前喝牛奶的女子。这个城市很大，没有人会注意这个角落里淡淡的对白，然后把背影淡淡地隐在黑暗里。

　　下午，坐在电脑前，有些眼花缭乱。头已满满地为一些莫名其妙的词眼所塞满。他们在笑，那个女孩在笑，象个小孩子一样和男人在说话。我开始写一封信。

　　我说：“我是个不太会懂得如何与男人相处的人，有时我真觉得我是不太懂得该如何和男人相处了。哪怕是和上司还是其他的人。他们都很时髦啊，人生活着，就该是这样的啊。。。。而我是乡下人，从开始就是，再也没有改变过。我没法变得时髦起来。真没有办法。

　　我的朋友，在这点上你与我一样的失败着，也许我们将会同样地被驱逐出神的家园。我们已经被放逐了，我曾经觉得我会很坚强，现在我觉得我一点也不是，放弃生命的意念今年一直缠绕着我。有时候我想我是不是可以放弃你和深爱我的父母亲走掉，只是来生，如果有来生，我一定会得到更严重的惩罚。如果不是因为害怕这个。我真想走啊。。。

　　我常常是这样在想着的。”后来我把信发出去。我坐在窗前，看百合窗折叠着层层叠叠，如同人生中那些层层叠叠的记忆一样。我独自住在楼里，偶尔，电话响起，有关或者无关，陌生的话语，偶尔，看见某道熟悉的风景，仍然是看不清楚，都浮现在一层雾气里。这时候，我会哼一些歌，哼着哼着，对每一个走过我座位的人微笑。

　　仍然固定地在那儿买牛奶。其实我并不喜欢喝牛奶，只是想停在那儿，和那个很乖很甜的女孩说两句话。然后拿起牛奶，转身离去。如同其实我并不喜欢恋爱，只是无意识地走到了你面前，问一声“你好”，然后消失在黑夜里。生活，层层叠叠，我拨开那扇百页窗，只模糊地看见了楼外蚁般的人群涌动。

　　现在我坐在一扇透明的窗户旁边。外面的阳光好极了，透过灰色的天空，肌肤都很明亮。我抬起头，眼中有雾气升起。

　　很温暖。
