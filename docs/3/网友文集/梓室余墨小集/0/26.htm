印度――新兴的软件超级大国

　　新加坡和印度两国企业联合在班加罗尔（Bangalore）郊外投资兴建的印度第一座世界级资讯科技园，将由正在印度访问的吴作栋总理正式主持开幕。新加坡财团、印度达达集团和卡纳塔克州政府一共投资2亿4000万元发展这个占地27公顷的世界级资讯科技园，并且负起兴建和管理这座资讯科技园的工作。

　　这是新加坡企业一项新的富于战略意义的投资，因为印度正在成为世界软件业的超级大国。

　　过去五六年里，世界上软件业增长最快的国家是哪一个？就是印度！这些年来，印度一直保持50%以上的增长率，而同期世界软件业增长率仅为20%。印度软件业的崛起，堪称全球信息产业发展历程中的一个独特现象，其在国内外取得的成就和信誉为世界所瞩目，特别是印度的软件出口更具优势。

　　印度60%的软件出口销往美国，23%销往欧洲。日本、澳洲和西亚的市场也在扩大，尤其日本市场发展势头良好。南非，拉美和中国的市场也在开拓之中。

　　在近10年里，印度的软件出口额增长三十多倍。软件业已成为印度增长最快和最重要的出口部门，而且印度企业界并不满足53%的年增长率，他们的目标是100%，并称只要基础设施和高素质人力资源有保证，每年翻番不成问题。

　　著名的顾问公司麦金塞（McKinsey）上个月预测，印度软件业和相关服务业到2008年营业额将达到870亿美元，其中出口占500亿美元。更有专家预测，印度目前每年值40亿美元的软件出口额，将在今后数年内突破1000亿美元大关。

　　世界软件业几大巨头也瞄准了印度，纷纷在印度投资。自从微软在海德拉巴德（Hyderabad）设立了其美国以外的第一座软件开发中心后，该地区已成功地吸引了大量外资，很快成为印度重要的软件中心。欧洲的巴安（Baan）公司已经宣布，其80%的全球研究工作将在海德拉巴德进行。电脑视景（ComputerVision）公司则把80%的全球研发工作放在普那（Pune）开展。网威（Novell）也在班加罗尔建立了自己的研发中心；甲骨文集团（Oracle）的网络电脑（NetworkComputer）操作系统也是在班加罗尔写就的；其他如鲁申（Lucent）、美国电话与电报（AT&T）、土坯（Adobe）公司、美国波音公司，英国宇航局、英国电信等，都开始把他们的一部分软件开发工作转移到印度。国际商业机器（IBM）、甲骨文集团、SGI与微软都在印度的有关院校投了巨资。

　　印度政府大力扶持软件业的发展，总理瓦杰帕伊认为一定要确保印度全力投入信息技术革命的全球竞赛，让印度成为软件超级大国。印度各邦政府的干劲也很大，纷纷推出自己的信息城计划。现在印度软件业最发达的城市是孟买、班加罗尔、德里、普那和加尔各答。

　　印度是世界上仅次于美国的第二个最大的讲英语的国家，拥有许多从英国留学归来的高科技人员。软件行业在印度有较大的普及，具有深厚的基础。其编程人员工资只相当于发达国家的八分之一至五分之一，加上发达国家电脑人才不足，所以跨国公司纷纷将业务委托给印度的软件公司。印度的软件公司拥有现代化的技术和雄厚的实力，足以处理大的软件项目，能提供质优价廉的软件，并确保及时交货。

　　尤为可贵的是，印度在软件领域开展了大量的研发工作，拥有自己的知识产权，这是软件企业或国家软件工业成败的关键。印度软件开发人员在全球市场发挥着重要的作用，很有希望成为全球的技术领袖。世界银行把印度和爱尔兰、以色列、新加坡、菲律宾、中国、匈牙利、墨西哥等7个竞争对手作了比较后指出：在提供软件和服务方面，印度居这些国家榜首。

　　经过近年的高速发展，阿拉伯海沿岸的班家罗尔已成为世界闻名的印度"硅谷"。这次新加坡财团的合作者印度达达集团是印度最大的软件公司，30多年来，它一直是印度软件业的领头羊，在全球有57个办事机构，拥有1万多名优秀的软件工程师，不论在专有系统还是开放系统上都以高效著称。

　　新加坡财团和印度达达集团所持的股权各为47.28%；印度卡纳塔克州政府的工业发展局则持有5.44%的股权。除了作为印度吸引外资的磁体，班加罗尔资讯科技园也希望扮演外国投资者开拓印度庞大的资讯科技人才和资源以及商业机会的枢纽。吴作栋总理曾指出，班加罗尔的资讯科技园是新加坡和印度两国企业建立策略性联盟关系的榜样，可以作为印度发展新式工业园的模式。

　　印度的软件业已成为世界软件市场的一支生力军，它很有可能发展成为本世纪世界的软件开发中心。新加坡完全应该利用自身的有利条件积极参与印度软件业的发展。

　　注：原载新加坡《联合早报》2000年1月21日。
