我所了解的社会主义市场经济体制

　　我所了解的社会主义市场经济体制，是中国在特定历史条件下创造的一种充满矛盾的体制。

　　比如，众所周知，现在中国的失业问题相当严重。按理说应该学习其它国家的经验，鼓励创业精神，让中小型私营企业有个宽松的发展环境，国家还应该对某些企业进行扶持，以此来消化就业人口，促进经济发展。可是据我观察，中小型企业在中国的日子是很难过的。本来申请注册一个企业就是很难的，成立后又被税务部门课以重税。有些税的设计极不合理，企业如果不走旁门左道根本就无法生存。而税务部门的贪污腐化又极其严重。另外，贷款的取得如果通过正当渠道也极难以得到，办事又要通过层层盘剥，这一切都成为企业发展的障碍。

　　由此我们可看到一个社会资源无法有效配置的怪圈，国有机构的冗员在社会上无法为足够的中小型私人企业吸纳，所以国有机构裁员也困难重重，而政府的有限资金又继续投入一些无效益乃至亏本的大中型国有企业，生产买不掉的产品，银行的坏帐也进一步增加，如此恶性循环。

　　今天的西方宏观经济学理论和微观经济学理论还很难充分解释中国当前的经济面对的问题，虽然有些学者如经济学家杨晓凯等作了一些有益的尝试，提出了一些初步的理论。在中国做生意者对中国的错综复杂的商业环境、政治、文化一定要有清醒的认识，照搬别处的做法是行不通的，这可的确是中国特色，不可不察。

　　注：原载多维社多维观点2000年10月26日
