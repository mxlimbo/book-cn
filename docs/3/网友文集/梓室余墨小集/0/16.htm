澳洲为和平作出新贡献

　　这次联合国维持和平部队在处理东帝汶危机上，发挥了重要的作用。维持和平部队的行动如此快速有效，澳洲的努力功不可没。澳洲在和平部队中扮演重要而积极的角色，使东帝汶失控的局面迅速得到安定。看来澳大利亚对亚太地区事务所起的作用值得加以探讨。

　　长期以来，澳大利亚奉行和平外交政策，在维护地区的安全与稳定和世界和平方面发挥着积极的作用，它在诸如裁军，难民，人权，反对核试验，废除化学武器，保护生态环境，反对毒品走私等问题上所持的观点和立场，体现了这个国家热爱和平的愿望，为国际社会所承认。对联合国维护和平的所有重大行动，澳洲政府提供了大量的人力、物力和财力方面的援助。另外，国家每年海外援助计划的专款在10亿澳元以上，用于帮助发展中国家和受援国发展经济和社会进步。

　　在难民问题上，澳大利亚政府持有传统的人道主义立场。由于澳大利亚所做的巨大努力，它长久以来被世界上很多受迫害、被逐出家园的人视为避难的场所，世界各地的难民不断涌入该国。六十年代末，越南排华，许多华裔难民漂流于大洋之上，当时连中国政府都不愿插手此事，最后还是澳洲和其它一些国家收留了他们。

　　澳大利亚政府曾反对南非白人政府的种族歧视政策，响应国际社会对南非实行经济制裁的行动。它支持中东各国为谋求公正、持久与和平解决争端所做的努力，欢迎南北朝鲜对话解决统一问题。它主张柬埔寨人民在没有外来干涉的情况下自己解决争端。主张和平解决索马里，南斯拉夫内战。澳洲反对大国在亚太地区争夺霸权，支持核不扩散条约，倡导建立南太平洋无核区，发起成立亚太经合会议。

　　澳大利亚还积极参加联合国维持和平行动，曾派遣军队和观察员赴中东，索马里，柬埔寨，南斯拉夫，伊拉克，东帝汶等国家和地区执行联合国的和平使命。此外，澳大利亚还经常作为国与国之间冲突、争执的调解人和中间人，化解矛盾。澳大利亚的前外交部长加里斯・ 埃文斯因为首倡举行柬埔寨和平谈判并提出一揽子解决柬危机的方案而被提名为1992年诺贝尔和平奖的候选人之一。

　　澳洲是一个一半具有亚太国家的色彩，一半处於西方阵营的国家。从经济实力上讲，澳大利亚同美日欧相比稍逊一筹，但和发展中国家相比却又强大得多。从地理位置上讲，澳大利亚是个中型国家，偏安一隅，与世无争，传统上少有与外国为敌的记录。从资源上说，澳大利亚地大物博，人口不多，没有对外扩张和侵略的动机，不会引起对霸权主义的联想。从政治上看，澳大利亚实行完善的议会民主制度，社会稳定，而且澳大利亚是一个由外来移民组成的国家，来自世界各地的移民五方杂处，兼收并蓄。这些特点都使澳洲在当今动荡不安的世界上处于一种特殊的位置，得以担任协调各方势力的角色。在多极化的世界中，大国与小国，发达国家和不发达国家，富国与穷国之间的确也需要一个中间地带作为尖锐矛盾的缓冲。而澳大利亚正是一个大多数人都能接受的调解人，世界各国对这个远离其他大陆的国家普遍有一种好感和希望。

　　这次东帝汶的维和行动，是历来国际间共识程度最高，决策速度最快的一次。在短短几天时间内让派驻和平部队完成所有必要的决策程序，和过去在卢旺达及科索沃等种族屠杀事件相比较，实不可同日而语。部分亚洲国家也一改过去不干预的政策，愿意在联合国和印尼同意下参与和平部队。连传统上反对和平部队的俄罗斯和中国代表都表示支持。这充分说明了澳大利亚在世界维持和平事业中所具有的影响力。

　　澳大利亚和世界各国一样在调整着自己的位置，力求在变化莫测、多极化的国际舞台上尽可能地扮演积极角色，同时维护自己的国家利益。东帝汶维和行动的成功让澳洲在维护世界和平事业上再立新功，也奠定了今后澳洲更积极地参预亚太事务的基础。从亚太政治经济局势来看，澳洲的参与将为区域带来新的稳定因素和平衡点。澳洲应该注意是，如何努力维持自身传统的中间人、调解人角色，坚持不偏不倚的和平立场，当好世界的和平使者。
