海外华文发展的强心剂

　　中文可说是一种国际性的语言。中文和汉字很早就走向海外，日本、朝鲜和越南都有很长的历史采用汉字的作为自己的文字。从而这些地区相当程度上受到中国古代文化的影响，形成所谓的"东亚儒家文化圈"。

　　近现代华文的对外流传是由于大量的、持续的华人移民。华文的使用早已不局限于大陆本土。除了台湾、香港、澳门，南洋――新加坡、马来西亚、文莱和印尼，聚居了大量的华裔，虽然大多数已经加入居住所在地的国籍，但他们中很大一部分人口仍然使用华文，以华文作为他们的母语。他们在当地开办的华校，有时也吸收少量的外族人来念书。这些所谓的海外地区，台、港、澳、新、马，都发展出本地化的汉语文学，当然这种文学与大陆的文学有著千丝万缕的联系。

　　近几十年大量的大陆新移民又为海外华文的传播带来新的气象。这些新移民受过较好的教育，有较强的文化需求，在世界许多地方建立了华文的报纸和杂志等，并建立起海外华文文学的消费市场。

　　另外两个因素也对华文的海外传播很有利。一个是中国政治和经济地位的上升，另一个是互联网的发展。

　　华文走出中国，不仅获得了天空，也赢得了新的大地。这次高行健获奖，无疑对海外华文的发展是一针强心剂。提升了汉语文学在世界上的地位，影响深远。

　　新加坡的《联合早报》是一张亲大陆的华文报，标榜用亚洲人的视点看世界，是唯一允许在大陆发行的海外报纸。这次对高行健获奖却持完全肯定的态度，批评大陆当局的做法。因为他们深知诺贝尔奖对当地华文地位的提升深具意义。

　　我个人认为大陆当局此次对诺贝尔奖和高行健的抨击是不合时宜的。因为国家之间的竞争，除了政治、经济之外，文化竞争也是必不可少的一环。文武之道，一张一驰。没有强大的文化力量哪能建立大国？

　　笔者认为大陆对汉字现在实际已经失去垄断权。所以大陆当局应该拥有全局性、世界性的眼光，对高行健获奖作一理性、客观、策略性的评估，而不是仅仅作出简单、粗暴、无理的指责，徒然沦为世人的笑柄。这次外交部和作协至少犯了两个重大失误，一是事先对高获奖估计不足，二是事后回应不当。周总理说，外交要做到"有理，有利，有节"。外交部和作协的同志如果不能作到深思熟虑，不能为领导人当好参谋，就应该下岗，回家卖红薯。

　　注：原载多维社多维观点2000年10月21日
