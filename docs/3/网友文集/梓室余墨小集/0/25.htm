网络联接"亚非拉"

　　因特网的发展正成为全球经济增长的新动力。为了在２１世纪以充分的实力对抗来自美欧日等发达国家的强大技术冲击，亚洲、拉美和非洲等許多发展中国家已经将发展网络经济作为重要的战略目标。资料显示，全球因特网用戶约２．６亿，发展中国家仅占２５％。欧盟成员国加上挪威、瑞士、日本、美国、加拿大、新西兰和澳大利亚的人口数量仅占全球人口２０％弱，而其网络用户的人数却占全球的７０％以上。发达国家的电子商务具有相当规模，而发展中国家普遍处于刚刚起步阶段。网络经济的发展将成为发展中国家在新世纪实现经济稳定发展的重要决定因素。

　　占世界人口一半以上的亚洲国家近年来积极追赶网络经济发展的潮流.目前除了香港，新加坡,台湾和韩国等新兴工业化国家（地区）的网络技术已接近欧美之外，亚洲许多发展中国家也已在这一领域初具规模。

　　与此同时，网络经济的浪潮也延伸到了拉美地区，并正在以惊人的速度发展。在非洲的一些国家，也正在积极引进外资和培养人才，以发展本国的网络经济。１０個非洲国家正同法国阿尔卡特公司合作，计划建立一个可将南部、西部和中部非洲以及葡萄牙连接起來的网络。

　　可以预见，不久的将来网络的发展将把亚非拉的发展中国家联接在一起。网络技术的进步和网络经济的兴起对"亚非拉"的社会和文化会有何种影响？又对全球流行的网络文化有何种影响？值得关注和研究。

　　注：原载香港《明报月刊》２０００年10月。
