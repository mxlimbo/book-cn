电子书带来出版业革命

　　对爱书人来说，书是自己的影子，走哪跟哪。电子书的出现将大大减轻爱书人携带书籍的负担。网上出版在今年已得到了很大的进展。被人们誉为文明经典的绝大部分书籍，现在几乎都在互联网上四处游动，任何人只要愿意下载，并作小小的一点改换格式的工作，都能够将它们变成电子图书。从莎士比亚到马克吐温和马基雅弗利等的名著，都可以在互联网上找到。例如宾夕法尼亚大学的"在线图书网页"(http://digital.library.upenn.edu/books)就是全面的免费资源，可提供一万多本电子图书。中文的电子书也不少，不久前香港博学堂(www.chinesebooks.com)就提供鲁迅的作品供人免费下载。

　　电子书带来的好处是显而易见的。在网上出版电子书，降低了出版周期，作者完成的作品通过网络几乎立刻可到读者手中，取消了印刷和库存带来的成本。相对于印刷品，读者要付出的代价是很低的，书价只是传统书市价的零头。网友也可以只下载有兴趣的单元而不必购置整本书。需求量不大的学术书也能出版了，绝版书也有机会重见天日。一些工具类的书，用电子出版可以不断更新内容，就象软件一样通过网络升级，得到最新的资讯。

　　电子图书装置——姑且称之为"阅读器"（e-book），是一种像书一样的计算机，可以显示从不同来源下载的文本。随著诸如Pocket　PC这样的手持式系统变得越来越普及，以及屏幕技术的继续提高，电子图书和传统书本在方便性上的差距将会逐渐缩小。以后爱书人出门只要装备一台 "阅读器"即可，不要背大堆厚厚的图书了。

　　电子书市场逐渐受到商家重视，传统出版公司开始以电子形式重版书藉。一些新创的公司纷纷出现，专门为数字化市场出版。有人预测，电子书的普及率将与数十年前纸本书的普及率相差无几。六十年前，当纸本书打进出版业市场时，数百万读者即使买不起昂贵的精装书，也可以在意想不到的地方，以意外低廉的价格买到想看的书。

　　电脑网路与出版业高科技两者结合，将把未来阅读方式带往难以想像的境地，这是上一世代绝对无法料想的事。希望电子书的发展能给广大读者带来福音，即使不富有，也能象台湾的李敖一样坐拥书城。

　　注：原载香港《明报月刊》２０００年9月。
