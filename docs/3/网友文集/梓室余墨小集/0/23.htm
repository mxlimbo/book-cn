虚拟社群 商机无限

　　当今世界，商业竞争日益激烈，大型跨国企业几乎将触角伸向世界的每一个角落。人们不禁要问，我们还能像发现新大陆、开拓新西部的先辈一样开发我们自己的商业空间吗？

　　答案是肯定的，社会和科技的发展为我们提供了新的机会，它就是最近报章的热门话题——网上虚拟社群(virtual community)。

　　俗话说："物以类聚，人以群分。"马斯洛也提出，相互交往和被人尊重是人类的基本需要，这些都说明了人的社会性。经济活动和市场正是人类社会交往的直接结果。人们相互交换利益，取长补短，社群里由此产生了形形色色的商业活动。而网上虚拟社群同样也提供了一种新型的商业空间，蕴涵无限商机。

　　早在1970年代初期，科学家们已经开始利用网际网络来分享资料、合作研究和交换信息了。当时科学家们基本上舍弃了具体的校园，而在网际网络上组成互动式的研究社群，这也就是虚拟社群的滥觞。

　　今天网络的规模不知比当年大了多少倍，通讯成本也不断降低。围绕着大量的布告栏服务站和讨论组，已经兴起了一个广阔的地下文化，也创造了许许多多的虚拟社群。这些网站把人们连结起来，克服了金钱、时间和地理的限制，快捷、便利地满足了人们幻想、兴趣、人际关系、交易四大互动需求。可以预见，随着网络的高速发展，虚拟社群的商业发展无可限量。

　　虚拟社群出现至今，如果以网络的时间标准来衡量，简直可说是有永恒那么久，但其潜能至今为什么还没有被人们所充分重视和开发呢？事实是，对于网络，大多数人更多关注于其通讯功能，没有多少人意识到所有通讯网络的真正价值都是建立和加强人与人之间的联系。历史上电话的使用也出现过类似的情况。电话的发明者贝尔(Alexander Graham Bell)当年以为他发明的这个新工具的最大价值是报道新闻和广播交响乐，直到后来别人把无法辩驳的证据摆在他的眼前，他才不得不承认电话的最大用途是人与人之间的通讯。

　　不少专家认为，虚拟社群的兴起将会大幅度改变今天的人类社会和商业景观，虚拟社群会成为网络上最富竞争力的商业活动形态。现在的网络科技已经提供了一个技术成熟的用户基础，在这样的环境中建立虚拟社群可以收到事半功倍的效果。

　　相对整个网络架构上累积的投资，每一个企业建立自己的虚拟社群所需要的投资递增实在微不足道，从这一层面来看，虚拟社群组织者在网际网络上建立社群，简直可以说是占尽了前人种树后人乘凉的便宜。

　　最近不断有好消息传来。例如，英国一位17岁的少年在年初投资150英镑推出一个以犹太人社群为对象的网站，现在其身家估计达500万英镑，成为英国最年轻网络富豪。新加坡方面，交通及资讯科技部宣布将和各行各业的团体商会合作，在今后的两年中推动10至20个电子商业社区发展。教育部和国家图书馆管理局设立了学生虚拟社群(Student Virtual Community)试验计划。新加坡国立大学的几位研究生也正在着手设立一个以新加坡中国人为服务对象的商业虚拟社群网络。

　　展望虚拟社群的发展前景，令我们难掩心中的兴奋。治理虚拟社群需要一套全新的思维模式和管理方法。虚拟社群的出现破坏了传统的规模优势，推动了白手起家的浪潮，进而威胁今天世界上最大公司享有的霸权。

　　公司若能了解、顺应和塑造这种新的经营方式释放出的力量，将会创造巨大的财富。凡是因循守旧、不图改革的公司，将尝到自毁长城的苦果。另外，"速度是上帝，时间是魔鬼"，出手快的人将坐享持久的优势，应抓紧时机，先下手为强。

　　注：原载新加坡《联合早报》1999年10月11日。
