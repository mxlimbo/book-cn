现代化之矛盾

　　西方现代化社会的每场技术革命都带来一波未平一波又起的矛盾和危机，不但自然生态和能源领域，其政治、价值、伦理、人性和社会都承受巨大压力与冲击。像英国工业革命和法国大革命，随之而来是海外殖民战争，一直发展到欧洲两次世界大战。可见现代化带来的风险与恶果，几同世界末日。

　　现代文明所达到的成千上万的蔚为奇观的成就，如此出色地为我们工作，丰富我们的生活，同样也可以耗尽、缩减和摧毁我们的生活，并且经常如此。很多现代的创造物不但没有服务人们，反而奴役人们；不但没有帮助人们突出他们的身分，反而淹没他们的身分。几乎每一项发明或发现——从原子的分裂到DNA（脱氧核糖核酸）的发现，到电视和电脑——都可以掉转过来对付我们和伤害我们。例如，如今以一次空袭来全面摧毁一个大都市，要比第一次世界大战时容易得多了。

　　在现今这个电视时代，希特勒或斯大林这类狂人要糟塌整个民族也比以前轻易得多了。人类历史上可有过像我们现在这样拥有在短短数十年间改变地球气候、耗尽地球矿物资源或其丰富的动物群和植物群的能力？现在恐怖分子手中拥有的毁灭潜力又比二十世纪初大了多少？

　　电子技术与数码信息的时代来临了，不管人们乐不乐意接受它，它给当今世界的生产方式与生活方式带来深刻的变化。网络软件硬件与网络服务业，在1998年度就为美国带来3010亿美元的好处，130万个就业机会，而且短短几年内又有数倍增长，而间接的好处就无法统计了，对推行美式文化之成效是显而易见的。试问，谁能拒绝如此的诱惑？

　　可是，对于本来就高度机械化和城市化的发达社会而言，人无疑成了不断适应技术与机器的奴隶；本来，除了住宅之外，影剧院、邮局、银行、超市、医院、图书馆、地铁、车厢、加油站也是我们城市的社区(Community)，人们还算是有处可去。不久的将来影剧院、邮局、银行、图书馆和售货员被电脑取代是大势所趋，剩下只有医院和无人服务的超市与加油站了。社区功能的缺损对人性与人际是一种威胁；人口密集的现代化城市里，人们难摆脱的除了紧张生活与工作的压力，还有强烈的孤独感和戒备心，人与人之间保持着一种刺猬式的关系。

　　快捷便利的生活带来的是生理肥胖,病变等后果；财富与地位的悬殊是造成社会怨恨的根源。尤其在物质发达丰富的社会里，消费和占有所体现出的是社会承认和自我实现的价值，并不是纯粹出于生理与日常的需要。

　　专家和教授告诉我们，在以创新为常态的知识社会中，稳定或力求保持现状，将视为企业的反常行为。未来的企业和组织要装备"数位神经系统"，全速竞争，否则就要灭亡。在所谓的"学习型组织"中，人们必须拼命学习商业竞争所需的知识（learn），然后再把这些知识全部忘掉（unlearn），因为世界变化太快，旧知识会成为掌握新知识的障碍。看来，在这场游戏中，人们不但要耗尽自己的体力，也要耗干自己的脑汁。

　　以幸福与道德的理想标准无法衡量现代化的价值性。真实的情况是，现代化的程度不是幸福的程度。但是我们似乎没有回头路可走。只有做梦的人才会相信可以通过以这样或那样的方式截断文明的进程来达成解决。

　　不错，现代化不是无懈可击的，人们批判它的弊端往往都很正确，但不可能创建一个完全没有矛盾的现代化，西方人现在不能办到这一点，未来的我们也同样是做不到的。


　　注：原载新加坡《联合早报》2000年8月1日，略有改动。
