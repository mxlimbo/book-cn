政治让位给艺术

　　任何人现在如果路过新加坡河畔的禧街（HILL STREE），都会为一个五颜六色的建筑立面所吸引。这是原禧街警察局大楼，经整修后，焕然一新，新加坡新闻与艺术部即将进驻这里。

　　原警察局大楼始建于1932年，现为受保护建筑。这次整修工作保留了古典式的立面，扩大了建筑空间，中庭也罩上了透明玻璃顶。尤为有特色的是建筑师从天上的彩虹得到灵感，把所有的百叶窗漆成各种颜色，增添了欢乐气氛，更为符合现在的用途。建筑底层和中庭计划开放给艺术商店和公众艺术活动。

　　除此以外，新加坡国会也已经迁出旧的国会大厦,由新闻及艺术部接管。旧国会大厦建筑，将作为文化遗产中心，里面设有表演场地、电影室、黑箱等。另一项计划，是把位于奎因街的公教中学旧校舍，改为附属于美术馆的国家画廊，供本地艺术家展示作品。

　　艺术发展在新加坡得到越来越多的重视。为推动文化艺术的发展，政府将继续提供更多的基础设施，艺术活动的空间也不断得到改善和扩大，艺术工作者会有更多的表演与展览场地。新加坡的目标是成为世界一流的艺术城市。

　　注：原载香港《明报月刊》２０００年7月。
