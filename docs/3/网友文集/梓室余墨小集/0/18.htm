《绿色香港摄影展》提倡环保

　　香港的城市风情为世人熟知，然而这个繁华的现代都市更也蕴藏着大自然和人文历史的瑰宝，这就鲜为人知了。

　　首次在香港以外举行的《绿色香港摄影展》，6月22日－6月29日在新加坡新达城展出，呈献的是香港较少为人知的大自然景色，和逐渐被遗忘的乡村情怀。

　　香港一共有260个外岛，岛上孕育2000种植物和超过440种鸟类。事实上，香港的面积40% 已列为自然保护区，还有23个"郊野区"。摄影展即捕捉其中6个区域迥异的天然景色。

　　摄影展的作品却全部出自澳洲人艾思滔（Edward Stokes，52岁）之手。他是作家，也是摄影家，对于环保不余遗力。自1993年定居香江以来，他便用摄影机捕捉当地自然风光，希望借着照片提高人们对环境的意识。

　　对于城市的环保概念，艾思滔认为："香港地小人多，环保的出发点必须以人为本，保留大自然美好的一面，使紧张的都市生活变得更美好。因此我们必须较实际地看待环保，使更多人因此受惠，甚至将它与旅游挂钩。"

　　正因如此，他的作品除了捕捉香港的地理环境之外，他还去探访香港数百年前的历史轨迹，使照片更具时间和空间上的力度。

　　他是应香港驻新加坡经济贸易办事处邀请，首次来新加坡举行摄影展。展出的下一站是马来西亚吉隆坡。

　　注：原载香港《明报月刊》２０００年8月。
