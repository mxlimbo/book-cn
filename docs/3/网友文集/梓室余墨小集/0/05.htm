中国市场的竞争环境与国际市场相比有哪些异同?

　　中国市场的竞争环境与国际市场相比，相同的是中国现在也和其它国家一样走市场经济的道路，不同的方面很多，这里我想举几点加以说明。

　　1.中国市场现阶段是非完全的市场，或者说是双轨制。因此企业需要从事大量的行政行为。例如摩托诺拉公司为了让中国邮电部采用其系统，曾把邮电部的有关人员全部送到美国培训，实际上是公费旅游。所以在中国的企业一定要高度重视行政行为，说这是企业的生命线也不为过。

　　2.中国的市场是很不成熟的市场。表现在法制还不健全，商业数据缺乏以及不准确。此外，消费者的消费心理也很不成熟，比如一餐饭吃掉几万元，崇拜洋货等，所以如何摸准顾客的心理是一个问题。

　　3.中国文化的独特性。"在过去的30年中，中国人已经使自己成为国际事务中的一个重要的角色。他们被看做是压服不了的、神秘的，的确，除了在中国，任何其他国家的人都不可能真正了解中国人；在不少人的印象中，中国人是根本无法理解的矛盾体。"美国人史密斯在《中国人的性格》中写到。中国人经历了长期的闭关锁国，现在虽然走向开放，对外交流还是很少。中国的历史很长，沉淀下来的东西很多。由于社会发展的环境不同，现在大陆的华人、香港人、台湾人、新加坡华人之间都能看到文化心理上的差异，这从文化发展来说是好事。可是如果是经商，就不能因为懂华语就以为具备了去中国经商的条件。

　　注：原载多维社多维观点2000年10月27日
