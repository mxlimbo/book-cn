新编成语词典

　　本词典内容纯属搞笑，不适合学龄儿童独立阅读，敬请家长陪同指导。
　　
　　 [狐假虎威]
　　
　　 释义：指模仿、效尤或抄袭别人的创意的行为。
　　
　　 典故：美籍华人杨致远创办“雅虎”公司，成为互联网第一家专门的系统的搜索引擎网站。国人张朝阳效尤之，创办“搜狐”公司，竟然也大获成功，果然是狐狸沾了老虎光，于是，后人就把这种模仿或者效尤别人的行为叫做“狐假虎威”了。
　　
　　 例句：上海美女作家卫慧写《上海宝贝》大赚特赚，别人见了眼红，纷纷推出《北京宝贝》、《黑龙江宝贝》，这真是狐假虎威。：）
　　
　　 [金字招牌]
　　
　　 释义：指借别人互相炒作互相利用的行为。
　　
　　 典故：2000年初，香港著名报人、武侠小说家金庸以一元钱的价钱把自己得意名作《笑傲江湖》的版权卖给中央电视台，这是金庸首次把版权卖给内地影视界。中央电视台正想借这块“金”字招牌来巩固自己在内地影视界的老大地位，而金庸也有意把这块行将褪色的“金”字招牌在内地多挂几块，多挂几年。于是 两家情投意合，共偕连理。
　　
　　 例句：王菲和小弟弟谢霆锋谈恋爱，互擦金字招牌，效果不错，老姜弥辣，小姜益红。
　　
　　 [王（朔）婆卖瓜]
　　
　　 释义：指不甘寂寞，不顾名声，豁出去，跳着上的行为。
　　
　　 典故：从99年底起，王婆的十八代孙子王朔一直新闻不断，先是出了本新作《看上去很美》，结果被批评家批得一无是处。然后发文单挑金庸，再挑着一系列文化名人开涮，颠覆权威，哄动一时，几乎天天上娱乐头条。尔后在新千年趁着余威连续出版了两本“文化评论”，引来洛阳纸贵。在人们视线稍稍离开的时候，他又干了一件出人意料的事：触网，开办“文化在中国”网站，自贬为打工一族。看来，老王头的确不愧是王婆后人，一下子就把国人的眼球给吸引过去了。
　　
　　 例句：湖南大贼张君大案刚刚告破，邓建国就请来音乐人张卫宁以这个案件拍电视剧，果然是王婆卖瓜炒作老手，深谙其道啊。
　　
　　 [叹为观止]
　　
　　 释义：指不守信用的行为。
　　
　　 典故：文化大师余秋雨先生，先前三番两次，信誓旦旦说不堪盗版折磨，从此封笔不著书，孰料新千年之际，还是出了本《千年一叹》的文化书，也许这真是他最后一本书了，“叹”也将为观止了。不过，问题是，一位甚富名望的文化大师，如此出尔反尔，姑勿论其如何狡辩，这一次他真的让人失望了，不论其行为本身还是此书本身。
　　
　　 例句：本以为女歌星杨某某已经告别歌坛安心作人家商人妇，谁知道叹为观止，最近又出了新专辑。
　　
　　 [扬眉吐气]
　　
　　 释义：指飞扬跋扈不可一世妄自尊大的行为。
　　
　　 典故：在影视圈小有名气的男明星王志文同志，有一回接受记者采访，记者临走给他一张名片。王同志正嚼着大大泡泡糖，为了不污染环境污染花花草草和小朋友起见，于是眉头一扬，很潇洒地把泡泡糖一口气吐在名片上，正好被反回来的女记者看到，气得大哭。
　　
　　 例句：歌星苏某某真是扬眉吐气，不给歌迷签名不止还让人打歌迷。
