爱情趣解之金山词霸迷你版

　　中午酣睡醒来，打开电脑，百般无聊，忽发奇想：不知love换了头尾字母会变成怎样的意思呢？于是打开金山词霸2000迷你版，逐一查询，取其词形相近相似者，发现出来的单词虽千奇百怪，似乎差天共地，但细想之下居然也颇有禅家佛偈味道哩，嘻！原来这金山词霸“外表冷漠，内心狂热”，实一情圣也，为碌碌众生揭示了人类爱情真谛。乃摘录如下，并尝试破解：

　　◎bovine n.牛的,迟钝的——都说恋爱是麻木的，你看，不是说“迟钝”吗？至于“牛的”者，大男人主义者是也。

　　◎cove n.小海湾——有什么浪漫得过，在雨后黄昏，与爱人倚坐于小海湾边，共看天际星宿浮沉？另外还有“爱情象个小海湾，波浪不大，但时时潮起潮落”之意。

　　◎dove n.鸽子，斑鸠——小情人吵架，夫妻翻脸，都想和平解决，如果是情意真切，爱情是可以得到和平稳定的，鸽子正是和平的象征。另作“放鸽子”之解：你约好她，她失约，即谓之“放你鸽子”是也。

　　◎govern vt.统治，治理；支配——呵呵，这不是典型的“气管炎”（妻管严）么？碰上河东狮吼，悍妇执政，统治家庭，治理丈夫，支配经济大权，这皆是做丈夫的最不幸之事:-(

　　◎hovel n.茅舍,肮脏的小屋——有时候，情人们最大的梦想就是能有茅舍一间，结庐河边，一亩田，安然度过一生，即是最幸福之事，也足矣。君看金庸小说中，最甜蜜的誓言和许诺中，都不能没有“茅舍”这个东西，比如张无忌和赵敏，杨过和小龙女，最后的归宿都是茅舍和小屋。

　　◎hover vi.徘徊；傍徨；翱翔——有多少爱情可以没有经过徘徊、彷徨的阶段呢？

　　◎move vi. &vt.搬动；移动——记得郑中基有首歌叫《搬屋》，写爱情尽头，老郑回到旧居收拾东西走人，看着昔日爱巢，如今物是人非，触景生情悲从中来，“搬出这天地，决心跟恋爱分离”，此之谓也。爱情已经搬移。

　　◎movie n.电影；电影院——爱情不能没有电影，电影不能缺乏爱情。我爱电影，但更爱你。

　　◎novel n.小说——琼瑶阿姨的小说写尽爱情百般变化，但，终究不过是小说，现实就是现实，不能等同。所以，现实的爱情可不是小说情节，没有启承转合，只能踏实追求。

　　◎ooze慢慢地流渗出；勇气消失（解释语焉不详，岂词霸之bug耶？）

　　——别别别！您别轻易说放弃，爱情就只离您一步之遥！要追求爱情，千万别让勇气慢慢消失。（另查有“淤泥，软泥之意，正是正是，爱情是泥泞小道）

　　◎poverty n.贫穷，贫困——所谓“贫贱夫妻百事哀”，物质贫困的爱情，有多少是可以维持幸福的呢？

　　◎rove v.流浪,漂泊——爱过，伤害过，然后可以离开，可以再次漂泊。没有什么可以留住我的心，我惯于流浪，我属于漂泊。

　　◎rose n.玫瑰——哪个女人能够抵挡男人的鲜花攻势呢？玫瑰似乎是每一场爱情不可或缺的点缀。而爱情本身就上玫瑰色的。

　　◎sovereign n.君主adj.统治的——嘻！又是一个气管炎，难道男人和女人之间的爱情真的是一场战争么？“就这样被你征服，喝下你藏好的毒……”

　　◎wave vi.挥手——挥手自兹去，天下没有不散的宴席，缘分已去，爱到尽头，就要分开，“人世间最痛苦的事情莫过于此”。但是，曾经真心相爱的情人们啊，即使分手，也要微笑着为对方挥手祝福，然后在泪水迷蒙中离开……

　　——嘿！小小迷你词霸，真是阅尽人间情爱真髓呀！


2000，5，13。夏午。
