流行文化二十年

　　让我们来简单检阅一下中国流行文化的二十年，看看我们是怎么长大的吧。

■八十年代--伤痕一代、苏醒一代、激情一代

　　中国刚从十年浩劫中惊醒，中国人身上满是伤痕累累，带着伤疼，小心翼翼地摸索中前进。

●文学：

　　中国刚从十年浩劫中惊醒，中国人身上满是伤痕累累，带着伤疼，小心翼翼地摸索中前进，于是，这时期流行伤痕、流行反省，王蒙、刘心武等人的伤痕小说引起了过去的一代人的哀思；不可忽视的是《曼娜回忆录》、《第二次握手》这些不折不扣的"地下文学"，以另一种方式和姿态表达着压抑已久的青年人对压制的反抗。

　　同时，以香港为代表的港台海外文化随改革开放而流入，结果，中国人一下子为这些光怪陆离的流行文化迷惑了眼睛，金庸古龙的武侠、琼瑶岑凯伦三毛的曲折言情使多少人废寝忘食茶饭不思？中国人如饥似渴地吸收，虽然有些消化不良，但它有"开眼看世界"的意思，使中国人的思想日趋开放大胆，其解放思想的作用，实在不可忽视。

●流行音乐：

　　一袭黑衣一副黑超，罗大佑以反叛斗士浪漫骑士的姿态自海峡彼岸杀来，与那时激情洋溢、充满了浪漫叛逆的中国青年人一拍即合，终于做了一个二十年青春梦，同时的参与制造激情的还有我们国产的崔健同志，自豪地宣告我们一无所有。

　　那时候还有一个小女子邓丽君操着靡靡之音来试图迷惑青年人，结果她得逞了，无论当时舆论如何围剿，通过掉粉的磁带，她的歌声响彻大江南北的大街小巷，回旋在每一个青年男女的梦境当中；在广东一代影响更大的是许冠杰，他的歌浅白易懂，唱出底层小人物的心声，于是大行其道，被誉为一代"歌神"。同时，香港还有张国荣、谭咏麟、梅艳芳等俊男美女，使得大江南北，有卡式收录机处便有港台流行歌，反正，"靡靡之音"的论断是再也恫吓不了年轻一代了。

●影视：

　　那时候电影文化还处于春风吹又生的苏醒状态，众多明星大多是以乡镇广场露天电影的形式走向千家万户的。八十年代无疑是刘晓庆和陈冲的天下，前者凭借《芙蓉镇》、《火烧圆明园》等至今仍为人称道的出色演出，一举征服了观众，尽管她的私生活在八九十年代被炒得沸沸扬扬，但是她最初的美丽不可抹杀；"小花"陈冲的惊艳美丽与刘竟相辉映，更凭《末代皇帝》杀入好来坞，为中国人争光。

　　若论八十年代的电影英雄，一百个中国人可能会举起二百只手异口同声的说：是周润发！不错的，他是一个神话，他那黑衣墨镜、一丝不苟贴贴服服的头发，他那无比优雅的举枪姿势，他的一举手一投足，我不知道，对于那时候的中国人影响有多大！几乎是毫不费吹灰之力，他就征服了全中国的年轻男女，甚至有女孩说，就是他误了她终身--她痴痴地等了发哥近十年，还抱着不切实际的幻想。如今，周润发已成了一个符号。

　　电视方面，那时候中国人三餐不饱，电视这魔盒子简直就是奢侈得不能再奢侈饿娱乐了。在沿海靠近港台的广东，电视最先兴起，那时候有电视的人家，每天晚上准是里三层外三层围得泄水不通。

　　"昏睡百年，国人渐已醒……万里长城永不倒，千里黄河水滔滔……"相信今日上了年纪的朋友对这首歌可谓耳熟能详了，那是霍元甲，拳震中华，脚踢俄国力士，打出了多少中国人的恶气！

●政治人物：

　　对政治人物的崇拜，更是中国人特色之一。八十年代，毫无疑问是邓小平的年代，天安门游行队伍中"小平您好"的大字，喊出多少中国人的心声，他们渴望着这位三起三落的传奇人物能把多灾多难的中国带出历史的泥潭，逐步走向富强繁荣。

●体育：

　　八十年代对体育明星的崇拜，颇类似于对霍元甲的崇拜，都是寄望于英雄式的人物能打败洋人，为炎黄子孙争口气。于是，创造"五连冠"奇迹的女排姑娘们、在奥运连拿三金的"体操王子"李宁，都是中国人的英雄。

■九十年代--变化的年代、流氓的年代

　　不是我不明白，是这世界变得太快！从火热革命式的八十年代激情梦中醒来，一身冷汗！

●文学：

　　不是我不明白，是这世界变得太快！从火热革命式的八十年代激情梦中醒来，一身冷汗！九十年代，是一个铜臭味最浓的年代，出国热、下海潮，一个一个接踵而来，社会的经济、人际变化剧烈，文人们再也不能无视于窗外的烦嚣，于是一个一个走出了书房，操起了营生的计活，比如张贤亮等人。

　　于是，九十年代初期的文坛，透着浓烈的商品味。于是，王朔们应运而生，贩卖着机灵，贩卖着什么都不吝的姿态，在九十年代着着实实火了他奶奶的一把，丫一副流氓嘴脸，软硬不吃，硬是把一帮穷酸文人气得眼红得，那劲就甭提了。

　　王朔式市井笑话听多了，中国人笑得也累了，于是想歇一歇下来，那已经是九十年代中期了，这时候江苏有个叫苏童的家伙，另类的文笔，摇晃的镜头，正好记录了这一代人骚动不安的心，也带起了一股先锋文学的热潮，弄潮儿有鲁羊、邱华栋等人。

　　冷不丁的，一个三十年代旧上海的传奇又一次奇迹般复活：被夏志清誉为现代文学最惊奇的两大收获之一的张爱玲"咸鱼翻身"，在九十年代奇迹般火红了一把，她的文集和研究丛书一套一套地出，改编的电视电影也一套一套地拍，其实，这也契合九十年代怀旧、末世纪心态的潮流了，不奇怪不奇怪。

　　九十年代另一奇迹就是王小波了。笑里藏刀、绵里藏针，一不小心又刺痛了中国人的痛处，汩汩鲜血就这样流了出来如英雄，有网人形容他的文字是圣水泡着的文字。可是当他唠唠叨叨时大家都保持沉默，可是当他永远保持沉默时，大家都唠唠叨叨起来。

●流行音乐：

　　菌男衰哥+商业机器包装=四大天王赚钱机器的成功运作。只是不知道，是我们年轻人的浮躁浅薄成就了他们，还是他们的浮躁浅薄造就了我们这一代追星一族，反正，是彻底造就了疯狂的一代，他们可以把生死置之度外，只为见偶像一面。

　　王菲在九十年代的流行乐坛可说是一道最独特最耀眼的风景线。这个不梳头发谓之最新发型、出唱片不做宣传、对传媒冷若冰霜、嫁了个穷光蛋艺术家老公最后不欢而散，现在以三十多岁"高龄"吃嫩草的香港歌星，从她走上舞台开始，话题就从来不断，喜欢她的，谓之"另类"，不喜欢，骂她"做作"，反正，她就是那么牛逼地火红起来了，在这个疯狂的变化的年代。

●影视：

　　如果说八十年代中国大陆影视界是刘晓庆的天下，那么，九十年代绝对是巩俐的天下（严格算起，她应该是从八十年代末红起，但大红于九十年代），从张艺谋手中精心雕刻的艺术品到嫁为商人妇，再到后来北大入学、补钙广告风波，她的话题从来不断，但人们更喜欢争论这样一个问题，离开了老谋子，她就没有了艺术生命力了吗？

　　八九十年代港台有"双周一龙"之说，"双周"，是指周润发和周星驰，这个时期，周润发继续他的英雄本色，甚至冲击好来坞，但人们更喜欢的是他八十年代末的辉煌。这时期，是周星驰的电影世界！相对于发哥，周星星同学的星途并非一帆风顺，先是六年潜伏于电视台做儿童节目等闲散位置，后来凭借《他来自江湖》、《盖世豪侠》等电视剧，创造出了独特的"无厘头"表演方式，并以此成功登陆电影大屏幕，从此青云直上，颠倒众生，有星爷的地方，必有轰然大笑……直至今日，星爷在网上还大红大紫，不能不说这是一个奇迹！

　　"一龙"，就是成龙了。与上述两位相比，他的运气是太好了。从七八十年代接李小龙的班起，他的星运从来不减，至今电影数量也许已经以百计了吧。八十年代已经相当走红，到了九十年代更是花开灿烂，电影越来越精细，特技越来越出色，并成为成功打入好来坞的中国第一人。与他一样的人物还有李连杰和杨紫琼。

　　电视方面，九十年代的中国，电视唱主角。中国家家户户都有了电视，于是，这免费的午餐豪不迟疑抢了电影的饭碗，一个一个电视明星风头甚至盖过了电影明星。那时一部《渴望》的"好人一生平安"唱出了多少中国人的心声！刘慧芳是那时中国人的大众情人。

●政治人物：

　　真是变得太快了，我们没来得及明白怎么回事，伟大领袖毛泽东居然又一次在中华大地上到处阳光灿烂一片，车头上、人们的墙上，到处是领袖的微笑；有关领袖的一切如老毛选、老袖章、老革命歌，竟然奇迹般火红起来，毛主席又一次被中国人请上了神坛，在变化的九十年代，这事儿既奇怪又合理。

●体育：

　　当年"东亚病夫""睡狮"的称号，是彻底被打破了，中国体育在九十年代灿烂开花结果，奥运上佳绩屡创，一个又一个体育明星也被打造出来，邓亚萍、李小双……但是，屡次兵败的中国男足，始终走不出亚洲，最有热情的中国球迷的耐性正接受挑战，相反，中国女足的铿锵玫瑰已经在全世界迎风傲放。

■2000年--飘一代

　　飘一代是一个注定，是一个宿命，在九十年代疯狂变化的年代成长，在麦当劳的奶水喂养大的孩子，注定要"飘"。这是一个飘的年代。

●文学：

　　飘一代是一个注定，是一个宿命，在九十年代疯狂变化的年代成长，在麦当劳的奶水喂养大的孩子，注定要"飘"。这是一个飘的年代。新千年伊始，一场关于美女作家的争论在网络上打响，其争论的焦点是，"美女作家"到底是"美女+作家"还是"美+女作家"，这给中国文坛在新世纪的大秀拉开序幕，紧接着，王朔跳出来，余杰跳出来，余秋雨跳出来……个个粉墨登场，从不同角度铨释了"飘"定义。

　　飘出来还有痞子蔡，还有网络文学及众多网络写手。

●流行音乐：

　　2000年的夏天，中国各大城市都被一片紫色所包围，紧接着又是一片红色，而背景下的主角，都居然是同一个乳臭未干的十八岁小子，谢霆锋。他的出现，对流行乐坛几乎是摧枯拉朽的破坏！从此，歌星们再也不需要笑脸迎人而可以板起脸孔，对养活他的东家可以毫无顾及大大地抨击一番！他是全国各大报刊娱乐版的头条常客，他拉着王菲阿姨的手的亲热照片满世界都是！他的出现，自是符合飘一代的特征，一切带着年轻的叛逆，他的走红，是那么必然。

●影视：

　　世纪末，现代鸳鸯蝴蝶派老同志琼瑶阿姨忽然死水微澜，凭借一部宫廷+搞笑的戏说电视剧《还珠格格》竟然再一次横扫了大江南北！其盛况实在不亚于当年的《渴望》，但是有一点是《渴望》无法企及的，那就是，这部电视剧提拔了的新人数量！几乎剧中有角色的人物，都被《还》剧狠狠地带红了一把，演技稚嫩的黄毛丫头赵薇、在台湾半红不黑的林心如苏有朋、新人周杰范冰冰、老同志李明启等等，竟然仅仅就被这一部电视剧带挈而青云直上鸡犬升天，历经年多了这帮家伙还霸占着电视机唱主角，这实在是一大奇迹也。

●政治人物：

　　忽然的，在北京街头，在香港街头，在年轻人的T恤上都不约而同出现了一个大胡子，在网上，连篇累牍地有纪念他的文字；今年夏天的北京舞台上，丫也在台上大摇大摆。丫就是切·格瓦拉，这位穷人的朋友，革命斗士，居然在新千年的中华大地复活了！是一种暗示，还是一种表白？

●体育：

　　写这篇文章的时候，悉尼2000年奥运已经接近尾声，中国队暂时以26金一人之下万人之上位列第二，这明确地告诉我们，新千年，将是中国体育腾飞的千年，是体育明星后来居上风头压过影视明星的千年，过几年，中国大概就会出像美国的迈克·乔丹这样的体育大明星了。

2000年9月30日凌晨1时半于躲进小楼
