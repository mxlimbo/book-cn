韩寒兄弟，你还能走多远

亲爱的韩寒兄弟冒号

　　先给你讲两个故事。

　　其一，老经典了，《伤仲永》，初中语文课本里头的文章，不管你如何鄙夷现行语文教育体制，如何把你的语文老师贬得一文不值，估计初中的时候，你胆子还比较小，上语文课还能听讲，这一课文里面的内容，相信你已经读过，这里不再赘述。这里仅套套它的中心思想吧：一个本来就很普通的人，硬是让神童的称号所累，最后堕落，真变回普通人了。

　　其二，小弟家中发生的真人真事。我的表弟，今年才4岁多的小屁孩，还没读书，很是精巧伶俐，大约是港台那边的电视剧看多了，小小年纪就经常语出惊人，比如，有回吃饭的时候，他吃着只着，忽然对他爸爸说了一句：“爸爸，你爱你老婆吗？”当时我们饭喷一桌，两当事人面面相觑。有段时间我们提起这孩子都赞口不绝：“这孩子特聪明，将来有出息啊。”于是在我们的溺爱纵容下，孩子闹得越来越厉害，真什么都敢说什么都敢做了，丁点大孩子什么不好学，学一套油腔滑调？于是我们家都开始烦他了，他那些惊人之语，不再是饭后谈资开心笑话了。

　　故事说完，韩寒兄弟你聪明人，我虽然说得不聪明，但是你应该知道我说什么了吧。

　　没错，你是挺有才华很稍带有点个性的人，作为同龄人，我是妒忌加眼红的，小小年纪在社会上如此出名，也是不容易。可是，你可千万别因此而真把自己当回事了，大家关注你，那是因为大家一来觉得这么点大的孩子，居然能写出这么老成的东西，先别管是否是装腔作势，都觉得不容易，加上你挺及时和聪明的自我炒作，大家更加好奇了，于是加以关注。--你必须清醒这一点，买你帐读你书的，我估计大部分还是学生，而在“成人”读者里面，大多是抱着好奇心态的。

　　所以，你的书一时好卖，你别骄傲，时髦的说法是“泡沫经济”，虚假繁荣，没多少人会对你实际的才华感兴趣，没人会成立“韩学会”，只是，大伙觉得你这小屁孩说起话来跟小大人私的，不容易。因此，你千万别骄傲，别因此而拿自己当回事，一旦大家对你兴趣消失了，你的书卖不动了，大学都瞧你不上眼了，你江郎才尽了，你只好选择当乞丐了，你没得选择。这不，我在网络上就看见你抱怨《零下三度》被编辑改得面目全非，你说“这不代表韩寒”，嘿，你还真得意了，可是，没看出来吗，大人们对你的兴趣开始消失了哦。

　　你的遽然走红，我看本来就是机遇的事，你不值得如此骄傲的。新世纪的中国文坛，卫慧王朔一干人搅得烟尘滚滚好不热闹，加上网络商人的适时炒作，给你们这帮人一个出头的好机会，所以，你作为一个聪明人，你应该明白凭一些小聪明混饭吃，并不是长久之计。文化还是要学一点的，前辈们的经验你应该借鉴，将来再来搭救知识结构的不合理，那就太迟了。老实学点东西吧，哗众取宠一时得意，终究不是正路。

　　韩寒兄弟，你走好了，你是个聪明孩子，静下心来，你应该会清楚自己下一步该怎么走。

　　匆匆走笔，辞尚达意，亲爱的，就此搁笔了。此致那个敬礼。
