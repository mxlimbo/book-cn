没有季节的城市

――广州印象

　　广州可真是一座没有季节的城市。在这里，你完全感觉不到季节的更替甚或季节的存在。郁达夫用“浑浑噩噩”来形容在这里过秋的感觉。我以为，广州的四季都可以用这个词形容：当你尚未感受完全春雨恩泽，毒辣的夏日艳阳已经毫不犹豫飘然而至。秋天和冬天，更是一早私下交易，感觉再好也没能察觉他们的到来，一切了无痕迹，无懈可击。

　　广州有着暴发户的嘴脸。沾着沿海城市的光，这里很早就先富起来，“东南西北中，发财到广东”，广东人生意头脑之精明灵活天下闻名，广货风行四海。腰包鼓了之后，和大多暴发户一样，也开始讲究排场和形象了，为了将自己打扮得更体面更有文化，广州人竭力挖出自己的族谱，比如几年前的二千多年城庆，就是为了证明自己历史悠久系出名门，但是，再多的粉饰，先天不足，底蕴微薄，广州文化氛围依然不浓，虽几经努力，终难成气候，流行音乐人的大举北移，便是明证之一。

　　广州人尤其是广州女人，最能体现广州人的精明。在体态上，广州女人不比东北大妞的丰盈厚实，广州女人一般以小巧玲珑见长，在这盛世，也许看上去并不美，所以每年“美在花城”选美大会全是外来妹天下，真正的花城小姐倒没几个。在言谈举止上，更不像京妞们大大咧咧冲口而出的豪爽劲，广州女人说话总要琢磨再三。打扮上，广州女人更比不上“上海宝贝”们善于打扮得花枝招展：广州烟花繁华之地，却非时装中心便是一证，广州女人精打细算，不会将钱花在打扮上。所以广州女人务实，贤惠，强悍。走在西关街道上，你随处可见广州女人勤劳的身影，在大小商铺里她们往往是掌事人，雷厉风行，指挥若定，男人也只是她们的配角。广州女人在家庭里任劳任怨，大小事务一揽在身。当然，广州女人叫外地人最顶不顺的是她们往往世故排外，动辄就会给你白眼。

　　和许多发达的老城市一样，广州亦是繁华与破旧并存的城市，在你惊叹于天河新城崭新先进的高楼大厦同时，你也会惊诧于西关老城的破落残旧。或许正是因为新旧并存，广州的人际关系也往往是十分冷淡，新旧城区的广州原住民和外来户格格不入，就是两个圈子，圈子边缘往往没有温情。

　　而我，就在这个没有季节的城市生活了近二十年，说不上对她有深厚感情，但是，这里始终是我的家，家里虽然寒酸，毕竟是我的家，我没有资格厌恶或唾弃她。广州最打动我的，是西关黑巷里昏黄的灯光，是下班后人来人往的天桥上的斜阳，是珠江渡轮上橙黄的落日，是冬夜里酒吧随意的爵士和随意的交谈……

　　没有季节的城市。
