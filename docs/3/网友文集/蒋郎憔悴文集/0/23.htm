网络谣言与你的生活息息相关

　　前新加坡总理李光耀曾经说过，新兴的网络媒体是极重要的战略阵地，对国家安全、社会人心影响巨大，因此要严格控制网络上的意识形态，否则一旦失守，后果不堪设想。虽然他这一说有杞人忧天的意思，但是毫不夸张，网络是现代战争的重要阵地。现在有学者就提出，网络其实也是一个战场，无论军事还是经济。

　　他们何来此说？对我们的生活又有如何息息相关的影响呢？下面就谈谈我的一些浅薄的看法吧。

　　网络有个重要的特点就是它的自由性和交互性。只要你有一台电脑，一只MODEM，无论性别、国籍、身份、学历，只要你上了网，你就有说话的权利，你可以网络上提供的各大免费的社区和论坛BBS上贴帖子，人人均可畅所欲言，这里无所谓的语言霸权，这里人人平等。网络这种平等的语言权力，它的优势自是不言而喻的，它使得人和人之间的距离前所未有的拉近，也使得人人都有自由的发言权，你可以针对任何一件事情发表自己的看法，极大地调动了人的思考积极性和参与性，这是好事情。

　　可是另一方面，我们也不能忽视这种自由性和交互性的带来的种种不利因素，正如一则著名的网络笑话所说的，“在网络上没人知道你是一条狗”，躲藏在网络背后的芸芸众生，没有了面对面的尴尬和法律的顾忌，可以为所欲为，于是，网络上也成为了谣言丛生的地方，并且经过网络的迅速传播，影响会极大，其危害性也是极大的。

　　这些谣言，有的是一些网站为了增加点击率，为了吸引网民眼球而故意为之的，比如一些文化、娱乐明星的花边新闻小道消息，由于法律一定程度上的监控，这类谣言还是有所顾忌的，因此占的比例比较小，更多的谣言是一些个别网民的善意的玩笑或者恶意的中伤，这类谣言通常与你的生活息息相关：深圳一个女网民经常半夜收到神秘的电话和大量的电子邮件，来电来mail者语言下流污秽粗俗，这个女网民忍无可忍报了警，经警方侦查得知，原来是这个女网民的追随者因爱不成，一恨之下在网络论坛上写了一个黄色的帖子，说“征求性伴侣”云云，落款留了该女网民的电话和e-mail，帖子在网络上广泛流传，于是有了上述事情发生。看，网络谣言其实与你的生活息息相关，你随时都有可能成为网络谣言的中伤对象，为你添上不必要的麻烦，这真是亟待法律的完善的监管。

　　小而言之，网络谣言会给你添麻烦；大而言之，一旦通过网络传播某种不良言论或者不利思想，在网络上占领了一个国家或者民族的意识形态，这其实是一个阵地，一个战场，甚至可以说是一场没有硝烟的战争，一旦失守，这个国家和民族就岌岌可危了。比起传统纸墨上的意识战斗，网络更有无可比拟的优势，它的威力也将千万倍，无怪乎李光耀总统如此重视据守网络阵地，他那一说真是极有远见，因此，我们可以看到，现在每一个发展互联网的国家，都开始越来越重视网络的监守，因为事关你的生活，更事关你的国家和民族的安危。

2000，9月26夜。
