当东施遇到卫慧

　　“故西施病心而颦其里，其里之丑人见而美之，归亦捧心而颦其里。其里之富人见之，坚闭门而不出；贫人见之，挈妻子而去之走。彼知颦之美而不知颦之所以美。”

——《庄子·天运》

　　自从春秋战国那个好事之徒庄周在其著作中将此等市井笑话夸大其辞渲染一番之后，东施小姐的形象从此一落千丈，不但落了个千古笑柄，而且，直接严重地影响到她原先开的小酒吧的经济效益，村里的男人们远远见了她就远远躲开，而女人们则窃窃交口取笑，谁还敢光临她？

　　东施心里愤愤不平，同是女人同样有鼻子有眼的，凭啥西施那贱人就皱眉头就比俺好看？不就是丫靠那点儿小道消息、风流韵事出名么？庄周这招人不待见的小报记者实在是可恶，若见了他，非“要把他的X撕烂”不可！

　　几千年下来，与西子捧心相反但几乎齐名，“东施效颦”也成了世间最恶心的事情的代名词。

　　忍辱苟且偷安的东施在经受几千年来世人耻笑之后，在那么阳光灿烂生活美好的一天，当她在网上看到那两美女作家卫慧和棉棉的一系列骂街帖子之后，她顿时灵光一闪、茅塞顿开！她一边读着两人的帖子一边哈哈大笑：“俺从此可以翻身出头也！”次日，众多可爱复可怜的中国网民们便都在网易主页上看到了这样一个帖子：《两美女：老娘抽你丫的！》这份署名东施的帖子，开篇劈头就写道：“卫慧、棉棉，你们改悔吧！”然后时而笔触锋利时而循循善诱，从自己一个被帮闲文人小报记者庄周污蔑诽谤而身心俱毁的惨痛亲身经历，对被广大网民集体精神强奸的两美女寄予老前辈式和革命战友般的关心和同情；同时又对两美女作家通篇呻吟、床第交欢的色情文字给予迎头痛击，谓之“你两愣青的初体验可把我家小宝给带坏啦”。总之这篇文字既同情又批评，既抑又扬，既体现了一个饱受传媒炮轰的的老同志对后辈的关怀同情，也表达了为人父母者对这类床第文字的深恶痛绝。帖子最后说：“你两愣青嫩着呢，也敢来丢人现眼？俺不想说别的了，一切在俺的新作《像东施那样疯狂》有交代。”帖子刚一贴出果然马上招来一千多人“电击”，三千多人的回帖，还上了精华版和网易首页，后来又经好事之徒到处转贴，再经各大传统报刊杂志转载，一时间成了网络上下的热门话题，有的.com还专门设立了“东施VS卫慧”的专题，不但提供BBS让已经分化成东施派和卫慧派的网民开骂对战，还专门请来了各界文化名人代表撰文分析。著名老作家陈老刀即撰文曰：“此乃世纪之交文化界一大盛事也，想不到隐匿多年的中国最早的妇女解放运动代表、优秀战士东施女士居然能站出来为已经风化堕落得不像样子的中国文坛说话，实乃一幸事也！”与此同时，网民们开始关注东施文章中提到的《像东施那样疯狂》一书。为满足广大网民的好奇心，著名搜索网站鸭壶专门请来了已经身价百倍、稿费高涨的东施女士来到聊天室作客。这是部分谈话内容：

　　东施；大家好！我已经来到活动现场了，欢迎大家对有关问题进行提问。

　　欲望手枪：东施偶像好！大家都比较关注，您写那个《两美女：老娘抽你丫的！》到底有何居心？是不是想踩着别人就上？

　　东施：俺吃的是中国米，流的是中国汗，有什么招人不待见的居心丫？俺是实在看不惯70年代的孩子堕落呀，痛心疾首之下就写那个帖子，没什么别的所谓居心。

　　内蒙古宝贝：偶像！您的新作《像东施那样疯狂》何时跟大伙见面？内容是关于什么？

　　东施：（笑）到时您不就知道咯！马上就会写完了:-)好孩子糖糖：偶像呀，俺特想给您老写个信儿，有邮箱么？

　　东施：可以。missdongshi@yayaho.com

　　自从著名“新生代网络作家”东施在鸭壶聊天室露面并透露《像东施那样疯狂》即将写毕之后，东施家里从深夜到深夜，一天24小时几乎不断地有海内外各大出版社打来电话要求取得版权，美国一家出版社居然开出了100万美元的天价，立马创下了中国五千年文人第一稿费的记录！但最后，为表示高风亮节崇高人格，东施舍远求近、舍高就低，把40万字的书稿交付给西藏一家文艺出版社出版，列为该出版社“纸老虎丛书”之一。

　　另一项文坛记录又被东施刷新了！从交稿到付印到发行，最后只用了不到三天的超短时间！东施的处女作，《像东施那样疯狂》正式出版了！

　　这部小说印刷极尽精美，封面是由著名摄影师精心为东施拍摄的写真玉照，只见东施双手捧心，眉头紧皱，赫然是千古美人形象“西子捧心”的翻版，但今番不同往日也，这个用新新人类的话来说“帅呆了酷极了”的形象，成了21世纪最流行最IN的打扮，成了经典，引来了街上无数少年男女的模仿——东施这回是彻底替自己报了昔日一箭之仇：谁还敢说东施效颦？东施今日引领了潮流，重新阐释了“美”的概念！就是西施在生，也非得活活羡慕死不可！东施得意地解气地想。

　　在这部40万言、由某著名骂嘴题词并欣然写序的小说中，东施彻底把自己给写进小说中去，她为广大网民和百姓深情回忆了当年自己是如何疯狂地创造出中国最早的行为艺术“捧心”的过程，却为西施不问自取抢夺了自己的艺术劳动成果，还为此背负了“效颦”的恶名至今。小说同时还穿插了自己和西施、范蠡、勾践错综复杂的四角关系，而这种关系是建立在东施对国家民族的忧患意识之上的。小说“以最洁净的文字”为读者描绘了最纯洁干净的性爱，呻吟，“让卫慧之流看到我笔下的性交之后立马惭愧得一头撞墙去吧”，东施这样得意非凡地写道。

　　这部小说一经出版，立时掀起了抢购热潮，一时间洛阳纸贵，全国上下大小书店书摊排起了长龙，一版再版还供不应求，一度断货；各大书库网站这几个月的浏览人数直线上升，银子书屋的服务器甚至而因此瘫痪；但与此同时，文化界也兴起了“东施现象”的大讨论，有人喻之这是继卫慧棉棉之后的另一次新世纪的文坛大事。不甘寂寞的陈老刀跳出来对这部小说进行评论，也博得广大读者喜欢，借此春风也再次火了一把……

　　看看东施是如何成名的吧，各位文人们！
