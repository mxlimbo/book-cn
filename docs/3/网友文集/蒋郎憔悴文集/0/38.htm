【网事新编】之〖白蛇笑传〗

自序

　　今夜肯定心绪不宁，乃戴上headphone听西洋之Dancing Music以佐兴伴我重写古老传说之网络新篇，玩弄古人物于键盘之下也。纪念上网一周年并献给昔日一同并肩作战在论坛灌水的菜鸟菜虫们。

　　是为序。憔悴志于最后之五一假期，夜，心不宁。告别一个故事。

2000年5月6日夜

　　1000年后，因为愚民迷信，将西湖雷峰塔放倒，不但令誉满天下的西湖十景之一“雷峰夕照”从此消失人间，而且也将困囚塔下千余年的白娘子得以逃出生天，重化人形，遁入人间，隐姓埋名，从此不过问三界之事，乃专心修炼。

　　忽忽又百年，人间已暗换新天，人类步入网络时代，千年等一回。

　　此时悬挂在白娘子头上的头衔可多了，诸如“中国妇女联合会理事”啦――想当年她曾经为了追求真挚爱情与封建势力代表法海老儿有过激烈斗争呢，端的是妇女解放运动的先驱前辈，此位置当非她莫属了；又如“中国蛇学会会长”――这个名堂怎么看都应该属于她了吧。……

　　尽管如此多的职位在身，白娘子还是感到不满足呀。她心里还有个结。话说那天她到市场买菜，那个卖瓜的王婆笑嘻嘻地递给她一张名片，她凑近一看，见写着“中国首家线上卖瓜公司，www.wangpo.com公司CEO王婆女士”，呵！好家伙，想不到网络大潮兴起，这王婆也赶潮流来了！

　　白娘子又是艳羡又是惭愧，网络时代都来临了，可自个还置身事外，对电脑对网络都还一知半解呢，眼看着十二亿人民争上网，还剩自己在发展……不行，这样下去迟早要被淘汰，没准要下岗了呢，要赶紧补上这一课，咱也要赶赶潮流，争当跨世纪的白蛇！

　　于是在接下来的两个月，白娘子买了无数本“一日通”“两日晓”等等电脑参考书，上了60天的IT补习课，终日在“http”“www”“java”“flash”间流连，终于让她修炼成功。

　　去年十二月，乘着电讯资费下调，白娘子前脚从电脑称拎回一只“56K”的小猫，后脚踹进点心局开了Chinanet 163的帐号，回家又迫不及待按照说明书安装小猫，对瘟酒吧进行网络设置……忙得满头大汗，最后，点击“连接”，小猫咭咭呱呱一路高叫着……突然，冷不防弹出个错误信息条：“系统没有安装猫凳，请检查设置”！把白娘子急得就想露出本相把这只该死的小猫儿给一口吞了……经过再三检查，还是不行，最后白娘子只好打了昔日的好姐妹青蛇小青的手机求助。

　　小青现在可风光了，当年被法海带回西天佛祖的七宝池一番忏悔，佛祖佛心仁厚，不忍伤害，就命她去后山修炼。在后山有一台老式的苹果机――没办法，虽说出家人不问世事，可是对于人间的最新科技，和尚们还是不敢脱节的。凭着小青的聪明能巧，居然无师自通把苹果机钻研通透，终日编些打字游戏的小程序，也自得其乐。后来随着佛祖添购了286、486……直到现在的Pentium III、K 7，她的爱机也从漆黑的DOS时代过渡到缤纷的Win时代，她的电脑知识也是出神入化，稳坐天界第一把交椅，俨然是电脑专家了，去年4月26号，玉帝老儿家里那台Pentium III中了CIH病毒，幸好及时请来了她，才修复了感染的硬盘，挽救了天庭的重要机密档案呢！

　　且说白娘子拨通了小青的手机，不多时只见一阵青风，小青按下云头跳到白娘子家中，笑道：“白姐姐敢情也步入网络时代啦？”白娘子不好意思一笑，道：“好妹妹别取笑姐姐了，快帮忙收服这只该死的孽畜！”小青便来到白娘子新添置的那台K7 600前，左摸摸右看看，不一会便查出根原――却是白娘子心急大意之下把电话线给插反了，因笑道：“姐姐您别急嘛，要制服这小猫可不能用急。”说得白娘子便要去打这淘气的小青了。

　　于是在小青从旁悉心指导之下，终于连上线，首先登陆到王婆那个wangpo.con，当见到王婆在首页那张经过photoshop处理过的大头照片之时，白娘子忍不住自豪高呼：“呵，看！我白娘子如今也跻身网络时代啦！”接下来白娘子手中鼠标不停游移，把在报纸上、车厢皮上经常看到的网站诸如新浪呀、网易呀，去了个遍，先还是兴奋新奇不已的，到后来面对如此庞杂浩如烟海的信息，她真花多眼乱目不暇给呢，于是犯愁了，对小青道：“好妹妹，这么多的资料，叫姐姐两只眼睛怎么看得过来呀？”小青运筹帷幄胸有成竹高屋建瓴――只见她飞快打下了yahoo.com、sohu.com等两个网址，然后对白娘子说：“姐姐，您想找什么资料，您只管对我说好了。”白娘子想了想，道：“对呀！你帮姐姐找找我们的同类――蛇的资料吧！”“好！”小青在空白处打下“蛇&研究”的字样，按下“搜索”按钮，不一会而，屏幕上出现了一大屏全是关于蛇及其研究的专业网址，呵！足足有上百个呢！

　　白娘子感慨道：“呀！想不道这人类比我们蛇还了解蛇呢！”于是按着网址逐一浏览。

　　一上午下来，白娘子还真收获良多呢，除了找到了大量的蛇研究资料，还“当搂”下载了好几百K的软件资料。小青不禁问道：“姐姐可真能融洽到网络新生活呢！”白娘子感叹道：“是呀，以前你我要找个人或找点修炼的资料，要日夜不停腾云驾雾往来四方八面，身心疲惫，但是你看现在，姐姐只要点一下鼠标便会不请自来，比我们的法术还快捷方便呢！”又道：“不过，看了半天，我也觉得好多网站看了等于没看，重复建设严重，你看这几个网站，资料基本上都一样，互相抄袭，浪费了我的时间呢。还有这一大堆综合网站，全是千人一面，全是以免费邮件免费主页来招揽访客，一点特色都没有，我来一次以后就再也不想上了！”听得小青只竖起大拇指啧啧称赞道：“想不到姐姐此等网络上美名曰‘菜鸟’的还有如此精辟见解呢！”因见白娘子也看得累了，便神秘兮兮道：“姐姐，待妹妹带你到一个很好玩的地方……”说罢键入网易社区BBS的网址，化名“青青”便遁了进去，首先来到“爱情故事”论坛，只听小青道：“这可是网络上有名的灌水胜地哩，要套狼泡妞，这可是好地方。你只要去别处复制粘贴几个帖子来，美眉哥哥们都要对你崇拜备至呢。”当时只见该论坛内十多人火热朝天得正灌得起劲，白娘子定睛一看，全是名叫“美美”“眉眉”的女孩子们，鸟语花香地正围着一个ID叫“许仙”的翩翩少年频送秋波呢！那许仙在众多美眉围攻之下，应付裕如，不时口占一绝，出口成章，或妙语连珠大呲李白苏轼唐宋名篇，或指点顾城海子李敖后现代主义经典朦胧诗派，端的是口沫横飞天花乱坠，几秒钟功夫又发一帖，如入无人之境，引得美眉尖叫此起彼伏芳心大乱……

　　看到昔日意中人如此放浪，白娘子妒火中烧，便要抢过来化名进去大声呵斥这负心薄幸的情郎一顿，小青连忙按止下她，手执网虫宁财神之著述《聊天室套郎不完全手册》，附在白娘子耳边如此这般地吩咐一番，直乐地白娘子眉笑眼开。

　　当下小青搬来她的镇山之宝，美国原装手提电脑一台，用手机连接上网来到爱情故事论坛，化名“独孤求败”进去，白娘子则化名为“东方不败”进去。

　　小青的独孤求败和白娘子的东方不败两人一进去，就用小青传授的copy&paste大法，来回切换于“唐诗三百首”和论坛版块之间，将那一句句唐诗绝句往论坛上就贴，没几分钟功夫，满满一版全是她两人的灌水文章了，直引得当时论坛内抗议之声四起，美眉红颜失色：“你们这么肆意灌水难道不怕大妈封了你们的post权么？”其间有个ID名“法海”的和尚更是拍案而起怒发冲冠：“论坛难道就真没王法么？你们可知道俺可是这儿的斑竹，你们若再灌水，非封杀你俩ID不可！！”……一时间神憎鬼厌天人共怨。

　　那许仙果然被她俩的灌水搅得兴致全无，心生退意，潇洒一句道：“众位美眉，哥哥我先走一步了，他日有缘，我们再共砸砖论战之乐！”话毕拂袖飘然而去。

　　白娘子和小青二人见他中计，心内大乐――此乃重演当年西湖上两人运用法力奏起风雨以阻许仙之船好借故登临相识之故伎也――遂退出论坛，进行计划的第二步，由那小青用最新最先进的黑客工具侦察许仙的去向。却见那许仙并不退出社区，乃径直来到另一冷清的论坛“西湖往事”孤芳自赏地发些小帖，正自寂寞难奈呢！

　　小青二人见机不可失，连忙化了新的ID“青青”和“素素”进去，白娘子的“素素”楚楚动人地发出一贴道：“先生何故如此冷清？方才见先生文才卓然，小女子也曾学得几分皮毛，也斗胆向先生请教一二。”许仙不知是计，见她帖子措辞也甚为文雅，一时也感到兴趣，道：“好呀，我们就互相学习一下。”然后自我介绍：“小生乃北大中文研究生也，专攻中国古典文学，祖上也曾是显赫人物……”两人就此搭上了。从远古春秋战国之《离骚》、老庄，到近来红遍网络的《第一次亲密接触》，从现实中的曹雪芹鲁迅到网络上的痞子蔡、宁财神、安妮宝贝，许仙滔滔不绝连呲两三个小时不带重样；而在小青在旁手捧一本《中国文学大辞典》的坚强后盾支持之下，在来回切换窗口复制粘贴之下，白娘子也开始了和这个网络才子长达两个小时的精神交流。两人一见如故相逢恨晚，最后还彼此交换了e-mail地址和icq号码，许仙还给了他的文学主页地址邀请白娘子到访指点，两人十分投契，恋恋不舍，许仙深情说道：“啊！谢谢上天赐予我美丽的美眉和这段此生难忘的网恋……”正自陶醉，冷不防一声呵斥道：“你两个妖孽要藏身哪里害人！”却是那个爱情故事的斑竹法海和尚！

　　白娘子和小青两人都大吃一惊，以为自己的蛇身暴露，谁知那法海道：“你们瞒谁都可以，瞒我老鸟法海却不行！你两个不是刚才那两个什么‘独孤求败’‘东方不败’两个灌水王么？却化了名躲在此次欺瞒许仙硕士，俺第一个不答应！”两人都嘘了口气，原来是被识破了伎俩！但白娘子奇怪了，那法海却又怎的如此神通看破两人身份的呢？连蒙在鼓里的许仙也辩白道：“法海斑竹误会了，这两个才气横溢的美眉又怎会是方才那两个灌水的无聊网民呢？”只听法海斑竹嘿嘿一笑，施然道：“这个也无甚技巧可言，要瞒菜鸟有余，却瞒不了我法海！――你瞧她两人的IP地址，不是和刚才灌水那两人一样么？许硕士切不可被她们迷惑！呔，兀那无聊客，方才水漫我爱情故事论坛，看我不收拾你！”小青向来自诩电脑专家，却也疏忽了这一节，其实她也知道破解之法的：便是使用匿名上站。现在后悔也来不及了，对白娘子道：“姐姐，风紧扯乎，咱们赶紧鞋底抹油吧！”
