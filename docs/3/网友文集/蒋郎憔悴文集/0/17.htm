北京琐碎

　　2000年的夏天，我有机会到北京一家网络公司做暑期兼职，在北京呆了一个多月。第一次远行，第一次来到我梦中的京城，经历了很多，感受了很多，于是有以下的琐碎片段……

一、北京第一夜

　　那天北京大雨，空气中不觉透着点寒意，同事说：“过立秋了，夏天算是过去了。”我心头猛然一惊，呀，这么快夏天就过去了，可是在我这个除了工作还是工作的暑假，我真感觉不出它的飘然而逝，心里不觉有些可惜，但是不遗憾，在这么一个特别的暑假，平生第一次，在异地孤身而过，遇到很到事，认识很多人，爱过，痛过，快活过，孤单过，但是，这都是将来回忆里最浓重的一笔啊，我在这个暑假收获很多很多……

　　往时的暑假，我大都是在家抱着书本听着音乐握着笔杆想着女孩而过的，但是今年，我决定让自己过得不一般。也许，是逃避什么，或者是寻觅什么，在广州这个空城，我感到了寂寞，我想我该换个地方呼吸，于是，我就来到了北京。

　　承蒙网易公司的咆哮先生抬举和帮忙，我得以顺利地在7月中来到了北京这个城市。

　　说起来一切都是老天关照，为了让第一次出远门的我不感寂寞，又安排了一个很灵气的小女孩与我同行。于是就和这个自称乏味的女孩，坐24小时火车，怀着无限向往往首都直奔。

　　我还记得我们一路上出奇地融洽，在那孤寂的旅途上，说说笑笑，每到一站，我们就在列车意见本上留言签到，写些逗趣的话语或即使感受，如今想来，竟是有种前尘往事的感觉了，可是，实际上也才是一个月前的事情，可是如今女孩先我一步离开北京，又去另一个城市寻找梦想去了。

　　火车于7月11日晚上7点准时进驻北京站，怀着新奇和兴奋，带着梦想和失落，我踏上了这片曾经演义无数传奇和斗争的热土上，在西站广场，望着夕阳下那的北京城，我心里真有些疑在梦里，是的，我确实离开了生我长我的城市，来到了异地，一切于我都是新鲜的。

　　马上给家里打电话报平安，接着又给咆哮先生打电话，问明了地址，我挤上了公车，正式开始了我的北京之夏。

　　我在车上，努力把疲倦的眼睛往外探，想好好地把北京城的一切都带到回忆里去。马路两旁灯火璀璨，高楼大厦霓虹灯把北京夜色打扮得分外妖娆。啊那是毛主席纪念堂啊，我心里浮现出有关它的资料来。北京夜色，一点不让广州专美，她繁华之中比广州更多几分从容，或者说自信吧。

　　这，就是我的北京第一印象。

二、人物印象

　　先说说我的头儿咆哮的印象吧。我跟他的接触，始于四月讨伐卫慧棉棉的时候，我写了个游戏文字《笑傲江湖名人版》什么的贴论坛上面，然后咆哮来信跟我说不错，希望以后能多写些这类好玩的文字贴上面，于是就有了以后巩俐试题白蛇笑传什么的，我跟咆哮的接触也多了。

　　在信里，在网上，感觉上咆哮是个活泼喜动的家伙，没想到第一次跟他通电话，竟然是个文质彬彬温文有礼的书生，再在北京见他，个子不算太高，戴着眼镜，有些老成的样子，温文尔雅的，跟网上的感觉真是有点距离啊，这以后见了不少网人，感觉也是如此。

　　咆哮对人是很真挚的，我来第一晚就请我吃饭洗尘了，这之后，我还就在他家里住下，给他添了不少麻烦了。

　　小薛，也就是猫兄了，完全还是个学生的样子，是个典型的江南书生，温柔细腻的，说话轻声细语，但可别小看他，他的笔锋犀利，又名野花，在四月的美女讨伐战、五月鲁迅保卫战中板砖横飞，所向披靡，决非恭维。然而让我奇怪的是，板砖狠辣的另一反面，他居然还能写出柔情似水的言情文字，倒叫我大跌眼镜了。

　　铁嘴阿良，网络上的写手，人很憨厚的样子，对人说话都很温和的，可是，他拍起砖来也是砖不长眼的角色。但是，他给我讲他在北京的遭遇，他带我去他原先住的潮湿肮脏的地下室参观的时候，才知道他还有深沉的一面。

　　由于是在文化频道的关系，我经常有机会碰到很多久仰已久的人物。他们的文字跟他们的生人是有距离甚至完全不同感觉的。

　　那天早上，正在网上浏览，忽然一前一后走来两高贵美丽的女子，说是来找咆哮的，我连忙招呼她们。只见这两女子一个穿着黑色衣服，雍容华贵，另一个一袭粉红套装，是个现代感十足的白领女人。

　　后来咆哮回来了，向我介绍道那黑衣女子就是网络上著名的写手黑可可，那粉红女子就是另一写手芭蕉，我吃了一惊，呵呵，想不到闻名已久的网络大腕就在眼前了……

　　至于另外一些写手，宁财神、俞白眉、刑育森、MIKKO等等，在一天晚上，我们几个去三里屯一个网络写手聚会的酒吧里有幸见过，我在一旁暗自观察，觉得他们太年轻了，跟他们文字的深沉老练实在是迥然不同的。

　　另外当然还要感谢热心的焦中华妹妹，年纪轻轻还是高中生的她，小小年纪就周游列国，发表文章无数，出了一本《酷女孩走天下》的散文集，实在是才华横溢，后生可畏。难得她很是热心，帮我联系住处什么的，这段友情，这分真诚，我是永记在心的。


三、网友聚会

　　我的网友中，大多都是北京的，所以来北京，我还能跟他们见见面。

　　记得把八月初的一个周末，我们三个人，我、猫和阿良去了大兴见乱弹论坛的网友。

　　乘坐个把钟头的车，终于抵达大兴。乱弹这群朋友away、海市蜃楼、快刀手就活生生地出现我眼前了，个个都是虎背熊腰的粗爷们，跟他们在网上细腻温柔的形象和文字果然是不同的。

　　我们接着喝酒，聊天，大谈网事。是夜，共聚于北京街头大排档。away大哥把他人生最浪漫的一幕彻底展现给俺们看了……（省略文字若干K）

　　这之前，我还有幸能和我的偶像、某社区砸砖巨手光盘贩子同志会面，贩子老哥可是历史研究的行家里手，社区众望所归的老大哥了，难得和我们这两个年轻人无拘无束地聊天说笑，关心备至，实在让我们感动不已。

　　前几天，在网上结交的一个广东的诗人蓝猫过来北京休假，约我见面，除了和我同来的女孩之外，我见的第二个岭南女子了。猫猫和我一见如故，相谈甚欢，虽匆匆来去，惊鸿一瞥，寥寥数语，但是网上神交已久的了，第一次见面仍是我美好的回忆之一，我会小心细致地收藏起来。


四、怀念猫


　　猫姓薛，我管他叫小薛，但是，猫这个名字对于他来说，真是再恰当不过了：小薛既能写像猫一样温驯、柔情似水的文字，也能写像猫爪子一样犀利的砸文，以猫命名真不为过。

　　有时候——特别是吃饭的时候，我看着小薛一边吃饭一边激扬文字的模样，我不由得就想起了鲁迅先生来，而其实他俩真的非常相象，两人都是江南书生，两人都是能酸能砸。但是其实猫更像柔石多一点，有点迂腐，但是执着；有点率性，但是性情。

　　初到北京的那天晚上，咆哮请吃饭洗尘，猫同往，猫一边吃一边与咆哮讨论宇宙物理问题，我在旁边无从插嘴，而猫圆瞪着双眼，高谈阔论，口沫横飞，一盘炸得麻辣香脆的鸡翅膀搞定之后，猫关于宇宙大爆炸的问题也谈得差不多了，就喵一声抹了抹嘴，物质文明、精神文明双丰收。

　　此后这样的场面也经常出现在吃饭的当儿，甚至吃饭的路上，猫一边走一边思考问题，完全没有留意身边车来车往，然后迫不及待要把最新的思考心得向我们汇报，这些问题天马行空涉及甚广，从文学艺术到IT近况，无所不谈，充分显示猫知识渊博。有时候思想火花迸出来拉都拉不住，猫竟然在公司里大谈老板小丁的事，可见猫真性情也。

　　我在这些争论中自然无太大置喙之地，然而显然是得益不浅的，我的好些帖子的灵感往往就是来自于中午餐厅里吃饭时候谈话中得到的启发。

　　猫之率性可爱在下面一件事里可以看到。那一晚，老板小丁在网站上和网友聊天，猫和我童心大发，化了名上去捣乱，插科打诨，谁知道那天晚上小丁竟然下来14层巡视，猫有些心虚，以为小丁是冲着他来的——因为他是利用公司的局域网聊天的，小丁如有心追查，轻而易举就可以查到肇事者是谁的。猫紧张了，跑过来向我说，完了完了，这下要炒鱿鱼了……我肚子里笑坏了，但是我却真的担心，猫和我这两顽童上去捣蛋，真的很有可能会被处分。

　　那晚我陪着猫散了一把步，安慰他不要担心。猫却一本正经地告诉我，唉，我也不想干了，反正很快我就会离开这公司。——我没想到，这一天来得这么快。猫这么快就离开了。

　　今夏北京的阳光特别毒辣，我和猫在这样的阳光里，一起下饭馆吃饭，一起去大兴和网友聚会，一起去雍和宫里“蹭导游”，一起去什刹海的胡同里闲逛，一起在公司疯狂泡网，一起去长安大戏院游玩……如今回想起来，那么遥远那么贴近，可是，如今，猫也离开网易离开北京了，今夏北京的故事，暂且划上了句号。然而，猫，还有一大群同事的可爱笑容，历历在目。

　　有时候发现猫和我有很多相同之处，猫率性敏感且执着，有些不合时宜，我也是。猫有他的理想，比如他要写很满意的小说，要办自己的网站，要写自己的东西，我也是，我也不会妥协。所以吧，我和猫才这么投缘，虽然我们话不多。

　　听说猫要回江南，回到自己的地方，要写小说，要办网站，我虽然远在千里无法为他饯行，但是，通过网络，我遥祝他一切顺利，我在网络这一头等着看猫的好戏。

　　而今夏北京，和猫和咆哮和阿良的那些日日夜夜，点点滴滴，虽然极其短暂，却是我的黄金时代，永远的镌刻着那些着最美丽最善良的片断。


五、北京之夏

　　回忆仍在继续，我即将回到我自己的城市。这，终究是别人的城市，我们只是过客而已。我不知道下一次是何时再能重游北京，但是，这一回，我想永远不能忘怀。
