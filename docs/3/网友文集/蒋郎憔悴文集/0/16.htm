成为校园作家之后——从郁秀谈起


　　就说现在这文坛（或者说文化市场更准确）也越来越像时装界了，一会流行美女作家一会网络小说，而现在风头正猛的是什么？你到书店畅销书架去瞧瞧便知端倪：从销量20多万册的《三重门》、《零下一度》到《太阳鸟》、《黑眼睛》（听说是一个12岁孩子写的一本小说），似乎可以下一个定论，这一季流行校园文学。

　　26岁的深圳姑娘郁秀小姐，16岁出版的处女作《花季·雨季》风靡全国，一版再版，获得无数奖项，虽然只此一本，但已备受赞赏，小小年纪就在全国范围内成名成家。然而3年之后她26岁，捧出了第二本著作《太阳鸟》结果又回怎样呢？我相信凭着《花季·雨季》的余威，它照样能畅销，但有《花季·雨季》成功在前，人们一定会将两者互相比较。

　　当年读罢《花季》，曾长呼一口气，心想这本书必定能风靡全国，还在晚报上写过一篇书评隆重向同龄朋友推荐。当时我的书评的中心是夸奖这部书很能够把握住当下（深圳特区）当时（改革开放之后）校园生活的新情况，并能够很写实地用活泼的文字将之表现出来，因此深受同龄人的喜爱。《花季》有些借鉴《红楼梦》的意思（郁秀最喜欢的书就是《红楼梦》），就是善于从日常琐碎中提取趣味，材料虽然显得零碎，但是通过文笔清新淡雅从容写出就能够充分表现出生活的本质。三年后再读《太阳鸟》又如何呢？我认为没有脱离《花季》的影子，从文字、结构、表现手法皆没有新意，几乎就是《花季》的再版了。等了三年，结果等到的是《花季》的翻版，不禁些微有些失望。

　　我在想，如果郁秀决意走作家之路，那她脱离出校园作家的影子之后的第一部著作应该要怎样写呢？《太阳鸟》决不是答案。这个问题对于天分极高如郁秀者，我想恐怕都是难以一时回答的。这个问题再提升一点，这样问：校园作家成名之后，后面的路该怎么走呢？

　　首先，校园作家面对的困难有这些：其一，来自外部的压力。这些校园作家小小年纪凭借一两部作品成名成家，人们的目光都关注在他身上了，校园作家们不免会有些惊慌失措；其二，来自自身的压力。他们仅仅凭借一两部作品成名，这终究带有很大的偶然性，或者只是一时初生牛犊之勇，并不代表自己真正的才华，这就为后面的突破埋了羁绊。

　　就是来自内外的矛盾，很多成名的校园作家，在一两部作品之后就销声匿迹了或者江郎才尽了，这样的例子很多啊，我印象中就有90年的时候我读过一个段姓少年作家的《红苹果季节》，这个段少年当时颇受刘召棠等老前辈的看好，但是现在似乎已经消失了，很是可惜。

　　当然，成功的例子也很多，当代大家王蒙十多岁就写了轰动全国的《青春万岁》，几十年过去了，王蒙先生不断学习不断突破，现在已是中国文坛的中流砥柱了。但是王蒙先生的成长进步，是和他没有受到上述内外两个原因影响有很大关系的。《青春万岁》从写作到出版再到出名，期间经历很漫长的时间，他并非一夜成名的，写成之后因为别的原因并没有能够顺利出版，免除了无必要的扰扰，使他能安心地继续学习成长；更重要的是他自身的所具有的文学创作天赋，使得他能保持旺盛的创作能力，因此这也是现在一些校园作家所不能比拟的。

　　那么，现在这些成名之后的校园作家，后面的路该怎么走？很多前辈都提出了自己的看法，我认为都有道理。比如著名网络写手王小山在其专栏文章中提到，成名之后"让他们生活吧"，我认为这有两层意思：第一是，请不要骚扰他们，让他们安心的生活吧；第二是，校园作家们，你们的底子还很薄，多些观察生活体验生活，以求让自己笔下更有分量吧。

　　这是后路之一，即请校园作家们再回到生活中去，细心体验生活，增广见识和丰富阅历。第二，我认为，校园作家们都请安心下来吧，暂时收一收虚荣心，不要为他们甜言蜜语的捧场话而骄傲。这很重要。因为只有将浮躁的心平静下来，才能将思想也平静下来，安心的学习，潜心修炼，那么，才有可能有质的飞跃。拿韩寒来说，无可否认，他是天分极高的一个人，从他文笔的老练就可知道，学识也达到同龄人无法企及的一个程度，但是，就请稍稍收敛一下浮躁的心吧，稍稍收敛一下无谓的狂妄傲气吧，锋芒太露，只会给自己添麻烦，心思浮躁，这只会不利于思考，写出的文字当然也无可避免的虚浮空洞。

　　总之，用主席的话祝福所有成名或不成名的校园作家，"广阔天地，大有作为"，天长地远，有你们翱翔的空间和时间，所以，成功的不必妄自尊大，而应重新进入生活；失败的不必妄自菲薄，潜心学习，用心生活，将来必能找到属于自己的正确的位置。
