南郭先生：你为什么还不忏悔？

　　（本故事纯属搞笑，如有雷同，实属巧合，请勿对号入座！）

　　话说我们的南郭先生在齐宣王那支三百人的鼓乐队里滥竽充数，因着老齐年纪老大，乐队人数众多，一时也觉察不出来，南郭先生在乐队里也着实混了好一段日子。后来老齐驾崩，他儿子小齐齐缗王喜静不喜闹，听音乐不像他老子那样讲究排场铺张，却只喜欢独吹独奏，我们的南郭秒先生眼看要捉襟见肘原形毕露了，好在他还有最后一招：三十六计走为上计！

　　他连夜卷席走人，回到乡下，自称“光荣下岗”，“多留机会给别的年轻人”，见人就吹宫里见闻，却也受到乡人尊敬；他凭着在老齐那里骗来的俸禄，一时也生活无忧。

　　忽忽三年过去，静极思动不甘寂寞的南郭先生因为坐吃山空，已经是囊中羞涩了，就琢磨着找点事干，但是由于年轻时只念着追美眉，荒废了学业，身无一技，却叫他去哪里找工作呀？正犯愁，这日他在收音机里听到西洋传过来的“爵士”音乐，是个喉咙生了个大肿瘤的家伙Louis Armstrong吹的小号，乱七八糟不知所云，旁边那个老爱故作深沉状的DJ还自我陶醉似的赞叹不已，谓之“天籁之音”云云。

　　“就这个跟驴叫似的东西也敢叫‘天籁之音’？”南郭一边听一边发牢骚，说着说着忽然灵机一动！“这可不是老天爷叫我翻身的绝好机会么？哈哈！”顿时计上心来。

　　当下南郭先生立即就把藏在箱底多年，已经发霉发臭铺满灰尘的“竽”给翻出来，吹去铺积有寸的灰尘，南郭先生便仿着收音机里的“爵士乐”吹了起来，一时吹得天花乱坠鬼泣神嚎，风云为之色变！住楼上肥婶再也忍受不住，当时一个大花盆就仍下来：“南郭孙子，你丫是想自杀还是想谋杀？”南郭却闻言大喜，肥婶的反应正是他想达到的最高境界哩！不禁欢呼一声，叹道：“等着瞧，明儿老子就敢把贝多芬那孙子给挑了呢！”于是收拾了细软就奔皇宫而来，凭着一张三寸不烂之舌加上“金钱外交”，南郭终于混进宫中，又回到了工作岗位啦！

　　次日，齐缗王照例一个个地传乐手们进宫挨个演奏。终于轮到南郭先生了，只见他不慌不忙地捧着竽行进殿来，对齐缗王道：“大王！请准奴才为大王演奏西洋引进的爵士乐，为大王解闷！”一句话引得这小齐好奇心大起，点头道：“好呀，寡人也想听听何谓爵士乐。”南郭先生心内大乐，脸上却倍儿严肃，也不急着吹奏，道：“大王，这音乐非有高尚品位人氏不能欣赏，大王请细心聆听奴才的演奏。”才吹了起来。他这“爵士乐”全无章法可言，胡吹乱奏，听得殿上众人只想掩起耳朵，却见那齐缗王怕人笑话他品味低俗，不但没有露出丝毫厌恶，而且还如痴如醉地频频点头，抚掌道：“果然是好音乐！好！”心里却琢磨着如何赶快把着家伙打发走，等南郭先生一吹完，立即就封了他为“人民的音乐家”，还赏赐了金银珠宝无数。

　　南郭先生在宫里用音乐倾倒齐缗王的事不迳而走，一时在王宫内外引起了轰动，南郭先生由此声名大振！他府邸外日夜挤满不远千里慕名而来的各大唱片公司老板，纷纷要求合作出版唱片事宜。

　　经过慎重考虑，南郭先生选择了著名的暴利金唱片公司，顺利出版了他的处女大碟――《爵士苦旅》。在这张被乐评人誉为“实力足够问鼎葛林美”的唱片中，南郭先生用随心所欲想到哪吹哪的爵士音乐形式，淋漓尽致地表现了知识分子对中华文化的忧思和反省。这张唱片刚出版立即被抢购一空，迅速地走红齐国，一时间有井水处便可闻驴叫之声。

　　南郭先生成了新闻人物，成了新新人类心目中的偶像、白马王子！他的豪宅外大批手持唱片待签名的少年男女们，几乎挤破门庭！南郭先生不得不花数十万美金从美国延请了五个经过西点学校严格训练的大个子老外作为保镖以保人身安全。

　　俗话说“人怕出名猪怕壮”，南郭先生的平地一声迅速窜红着实引来了不少人眼红。围绕着他的官司和诽闻从来不少，比如那天他跟个美眉呆在一房里交流音乐心得，却不慎被狗崽队的同志们偷拍下来，制成录影带摆街上出售，气得他就想请保镖们把那报馆给砸了！后来转念一想，灵光一闪！这可不是个出名的好机会么？当下他便写了长长一个启示登报，立即成了各大报纸的头版头条：“我国著名音乐大师南郭先生发表退隐声明，自谓由于被某报娱乐记者偷拍，身心俱累，严重影响了他的音乐事业，情不得已甘愿‘封嘴’引退！”这消息犹如平地惊雷，立即在全国上下引来轩然大波！

　　那边“退隐”风波尚未平息，这边厢忽然有个自称昔日齐宣王时期曾经跟南郭先生共过事的“金笛书生”余鱼同在报纸上发表了一篇名为《南郭先生：你为什么不忏悔？》的文章，在这篇洋洋洒洒三万余言的长文内，余书生以南郭先生昔日同事的身份，揭露了南郭先生当年在宣王任内不懂装内行、滥竽充数的丑行，文章罗列了大量的证据，气势咄咄逼人，文章最后责问：“南郭先生，你为什么不忏悔？你作为公众人物，百姓偶像，你应该为你昔日所作所为进行检讨和忏悔，但是，你为何还想掩饰？你为何还不忏悔？”这无疑是更大一颗重磅炸弹，南郭先生看后耳朵“嗡”的一声站立不稳，便要昏厥过去：这余鱼同正是当年和他睡上下床的兄弟呢，那年在宣王混吃的时候，他曾经抢了余鱼同同志的一包巧克力，没想到这小子居然记仇至今积怨如此！良久才反应过来，但他终究是个见过大场面的老江湖，立即平静下来，又一乐：“哟！俺正想写部自传来个临别秋波呢，这招人不待见的臭鱼正好替俺炒作了一把哩！”南郭先生花了一晚上写好了回应文章《对小错我从不追究》登报上，文章以爵士式游离笔法抨击讽刺了余鱼同之流的借名人出名的心理：“小余弟弟求的不就是让我写回忆录时能为他添上一笔吗？”并说“我对往事无怨无悔，大错没有，小错不究，有何忏悔之处？不信请看我的新作，一切自有说明”。

　　几天后，南郭先生退隐前最后一部作品，30万字的自传《一嗟三叹》出版了！在这不部印刷精美的书中，近1/3的篇幅是南郭先生在各大旅游经典留下的仙踪玉照近百幅以及他和美眉们大胆露骨的“剖士”；在文字部分，南郭先生为他亲爱的听众知音们神情回忆了这一年来的丰功伟绩，对于余鱼同提及的敏感往事，他作了太极式的推磨处理：“我认为我这年来的成绩才是我回忆录的重点。”这部临别秋波之作一经出版，立即又是洛阳纸贵的抢购，正当南郭先生洋洋得意正要坐收上百万的版税之时，麻烦又来了：先是爱卫办的同志投诉他的自传封面太过不雅，有伤风化，影响市容；接着是一群好咬文嚼字的老头集体写信给报社给他挑了近万个错别字……

　　正闹得不可开交之际，齐缗王来旨要他进宫听他的告别演出。南郭先生身心俱累，无精打采地吹着乐器，看着齐缗王似笑非笑的样子十分熟悉，迷迷糊糊中只见一人指着他鼻子大骂：“你这狗奴才真够大胆！骗了老子还想骗我儿子……”――这不是前朝大王齐宣王么！“要不是你丫的驴叫音乐楞是把寡人从地府里吵醒，寡人还不晓得你就是以前那个在寡人手下混饭吃的小东西哩！来人哪，把这欺君的狗奴才推出去斩了！”南郭先生吓得P滚尿流，当时就在殿上昏倒，送往医院经抢救无效，一命呜呼了！
