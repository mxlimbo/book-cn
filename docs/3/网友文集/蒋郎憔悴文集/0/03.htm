年华如斯（下）

　　六、我的太太有外遇

　　发现我的太太有外遇，是意料之中，但是仍然教我悲伤不已。我素知道她是不安于屈于我小小一个报馆编辑怀中的。

　　有一日，我在家中收拾衣物，发现她的衣袋中有一条花花绿绿的领带，上面纹着方格。领带上隐约有一唇印，那一刻，我顿时什么都了然。是的，她到底背叛了我，这让我厌恶、寒心，同时我的尊严毫无保留，被她尽情践踏。我恨她。虽然如此，我只能忍气吞声。被妻子戴绿帽，如果传了出去，我非但不能在这里住下去，就是在报界我也是再也混不下去的。如果我日益壮大的读者群知道他们的慕云大侠家里竟然是如此狼狈，相信他们会毫不迟疑地甩了我的报纸不看。

　　终于，我也知道“他”是谁了。他竟然就是我们同屋那个苏太太的丈夫陈先生！因为有天早上我和陈匆匆擦身而过，我彼此消失在视野的一瞬间，我看到了他的领带，花花绿绿，上面纹着方格。

　　我为我发现的这个真相而懊恼，那一整天，我浑浑噩噩，坐在报馆里，一个字也写不下去。

　　原来如此！这可真是近水楼台先得月啊，可是我的太太也太水性扬花了吧，才搬来半年就和邻居好上了！我问自己，我分明是已不爱她了，为何还耿耿在怀？写罢《江湖情幻录》第三回“惩恶锄奸”之后，我蓦然省悟，原来我们文人对于这个问题都有同一习性：小器。

　　苏太太似乎还不知道他丈夫有外遇的事情。这时候，似乎所有的疑问都解开了，何以我经常在楼下的面当经常能见到苏太太的身影了，原来陈先生经常不回家，冷落了她！

　　有一天，我从在走廊里和苏太太碰面。

　　“周先生你好，我前天没存好你们的报纸，错过了周先生的小说，周先生能借我看看么？”我暗暗吃了一惊，想不到这个弱质的女子居然会看武侠小说，于是点头，说：“好啊……原来苏太太也喜欢武侠小说的，我现在就回屋拿来给你。”她跟在我身后，说：“是啊，闲着没事，都爱看的。我一直追看你的《江湖情幻录》，秦玄在这一回该去找他的仇人报仇了吧？”我一边拿出报纸递给她一边说：“是啊，写完报仇，这个故事就结束了。”她有些惋惜的样子：“这么快就结束啦？”我说：“对啊，我的同事金庸梁羽生他们长篇的写得非常好，我长篇的功力比不得他们，但是中篇的故事我很拿手，可以和他们比一比的。我以后也会照这样写一系列中篇的小说。再说，太长了，人家也没有耐性看，是这样吗？”她点点头：“是啊，老这么长，我等急了就不看啦，周先生的策略很对。”我说：“谢谢。”她低头看了一下报纸，说：“你写秦玄杀了那个通奸的颜氏了？”我说：“怎么了？”她摇头说：“不喜欢这样。”我问：“为什么？”她笑笑没答，然后忽然无缘无故地问：“周先生今晚有空吗？我想请周先生吃饭。”我愕然。我们平时没有什么来往的，她竟然开口说要请我吃饭？我正要问有什么事，她抢着说：“我有些东西不明白，要问周先生的。”我“哦”了一声。

　　七、我们其实都知道

　　晚上，我和苏太太来到楼下的餐厅。我们随便点了几个菜，我低头只顾吃。苏太太胃口本不太好，现在有了心思，根本没有动筷子，我连说：“苏太太也吃一些吧，虽然说是苏太太花钱请的，可也要对得起自己的银子啊。”我开了个玩笑。

　　果然她噗嗤一声笑了，说：“那好啊，我不会亏待自己的。”说着也开始吃了一点。

　　“秦玄还在行侠江湖呀，什么时候让他去报仇呢？”她没话找话。

　　“快了。”我也没话找话。

　　“改天能上你们报馆看看吗？金先生、梁先生当然还有周先生都是我崇拜的人呢。”“行啊，你什么时候上来我们都是欢迎的。不过金先生前些天已经递了辞职信，他说要自己开办一家报馆，叫什么‘明报’。”“那多可惜呀，以后他全力投入办报，没功夫写武侠小说了。”“那倒不一定，查先生是个不同一般的人，他既能办报纸还能同时兼顾写小说呢，他才华横溢，我相信有一天他一定能闯出自己一番事业的。”“那你呢？周先生有打算效法金先生出来独立门户，另开一派？”“没有……”“哦？那为什么呀？”“呵呵，人和人注定有所不同的，有人以建立功名为快乐，有人的则是无拘无束为人生快事。”“呵……周先生可真像你笔下的人物，个个都超然世外，我真担心周先生有一天会看破红尘，去少林寺出家当和尚呢。”苏太太很俏皮地说，我无意看见她，灯下通红的小脸红朴朴地，越发动人。

　　“恩，周先生和太太感情如何呢？”她的话锋突然一转，多少让我听得出这话的预谋。

　　“不算太好……”我在这样单纯的女子面前，无法欺骗。

　　“恩，假如周先生的太太有了外遇，周先生会生气吗？”我没答，我只是奇怪的看着她，她的眼睛里有异样闪动，这瞒不了我，我反问：“如果换了是苏太太的先生有了外遇呢？”我分明地看到她眼里的幽怨：“有又能如何呢？我无法阻止他，可是，他……他竟然是跟周先生的太太在一起呀！我真是无可奈何……”她的声音快要哭了。

　　我叹口气：“原来我们都知道啊，我原以为只有我一个人知道。”她没话，只是自怜自伤地苦笑，说：“我们走吧周先生。”我结了帐，和她并肩走出去。一九六二年冬天的寒风立刻包围而来，我们都不禁打了个寒战。

　　街上人迹稀少，这年的冬天少有的冷，只是不知道那是天气的缘故呢还是缘由心生呢，我问她：“苏……丽珍你冷吗？”她望着我的眼睛，对我这个称呼有些意外，然后笑笑：“不冷。你呢？”我说：“我好冷……”她突然冒出一句：“那请周先生握着我的手吧，我的手暖。”我迟疑地望着她，情况尴尬，我拉也不是，不拉也不是，倒是她大方地递手过来让我牵着。那只手很暖很暖，暖得让我妒忌地发狠劲，我用力地握，握得她几乎要夺回去，我才松开。

　　“不知道他们是怎样开始的呢？”八、《江湖情幻录》大结局我坐在灯下，点燃了一支香烟，让香烟的烟雾飘荡在整个空间。香烟的味道总是让人虚浮不安的心能够得以很快安静下来。

　　我开始写《江湖情幻录》的大结局。这部小说有点虎头蛇尾，我承认那是因为我这段时间一直心绪不安，我无法安下心来继续写这个故事。我打算辞去《商报》的职位，和写马经狗经的简老八一起过去新加坡一家华人报纸那里干。于是，我必须把这个故事在这个星期内结束。

　　为了逃避开世间的烦扰，好让我的处女作有个完美的结尾，我来到了离岛这间临海旅馆2046号房里写故事的结尾。

　　其实故事还没开始，就要结尾了。

　　……秦玄来到了四川蜀中。他终于打听到仇人谢九亭的下落。这日来到集市上。突然间听到前面人声嘈杂，隐约伴随一阵清脆银铃声传来，一把清脆的女声娇叱道：“不想找死的快闪开！”接着路旁一阵慌乱，人们纷纷躲避在旁。

　　一只全身毛皮闪真油光的小驴急促向这边奔来，驴子上坐着一个貌美如花的妙龄女郎，正气喘喘的挥着皮鞭赶着驴子，似乎急着赶路，只见她粉腮红朴朴，香汗淋漓，越发美艳不可方物。

　　突然间，她看见了路中心还有一个不怕死的站着，不禁大怒，喝道：“让开！”驴子转眼已来到那人面前，那人丝毫无退让之意，眼看驴子要撞上去了，女郎也丝毫无勒止之意，就在那千钧一发之间，那人身形一变，蓦地里顺地倒下，手里的一剑物事向驴足削去，只听得驴子一声惨嘶，前足居然被那人手中的物件平平削去，那驴子惨嘶着委顿在地，女郎“哎呀”一声四平八稳被放了下来，跌倒在地，情状狼狈已极。

　　这时街上群众都围观过来，指手画脚，对这个趾高气扬的女郎平头品足，幸灾乐祸。女郎哪曾受过这样的委屈，急得就要大哭起来了，忽然一人刮着脸皮唱道：“大姑娘，不只羞，跌在地，眼泪滑溜溜！”正是那个削了她的驴子双足的人，却也是个和她年纪相仿的少年而已，穿着邋遢肮脏，发着臭味。

　　女郎找着了发泄对象，也不哭了，气得杏目圆瞪，道：“你知道我爹是谁么？你敢得罪我？四川你肯定是走不出去的了！哼！”“管你是谁，哈哈，反正我看大姑娘流眼泪是蛮不错的事嘛。”秦玄就是看不惯这样娇生惯养的女孩的娇纵，有意气气她。

　　果然，那女郎越发气恼，一声不吭，唰地拔出长剑，一道寒气直逼众人，围观人群见要动手了，纷纷闪让开去。

　　秦玄得势不饶人，继续刮脸笑道：“不害羞，大姑娘急了要动手，小伙子急了要动口。”说罢作势要向女郎亲去。女郎大怒之下，乱砍乱刺，全无章法可言，但是每到那眼看要到肉的一瞬间，秦玄总是能轻巧闪过，女郎知道这下遇到强手了，也不甘示弱，渐渐使出了看家本领。

　　秦玄与她来回数十回合，眼看也闹够了，弓指一弹，女郎的宝剑立时被震飞掉地，秦玄一抄，横握在手，厉声道：“闹够了！大姑娘一个，骑着驴子满大街跑，作威作福，也不知羞！”说罢，狠狠把剑扔在地，大步而去，头也不回。

　　那女郎眼看着少年得意非凡的样子，说也奇怪，怨恨反而渐渐减弱，只是还在回味刚才秦玄那厉声一喝的样子，那是她此生未曾见过的，只觉得心头一阵阵受用，甚至还希望这少年多骂她两句哩。

　　……晌午，秦玄在路旁茶亭里喝茶。远远便看见那女郎一拐一拐的向这边走来，心里暗自好笑，心想今天把她教训得也够了吧，这样的官家小姐，娇生惯养，也是在需要受点苦。

　　正想着，女郎也已看见了他，“哼”一声，在另一张桌子上坐下来，秦玄童心又起，唱道：“大姑娘，白衣裳，黑脸蛋，一脸晦气不打一处上！”女郎也不顶嘴，恶狠狠瞧他一眼道：“呆会我爹爹来了，你便要吃苦头！”秦玄笑道：“丫头你芳龄几何呀？还爹爹长爹爹短的，真不害羞！”女郎一时也找不到话驳他，自己理屈，正要发作，忽然路上来了一伙人，一看就是那些流氓地痞，招摇过市走了过来，为首一人一眼就看见那女郎，顿时色心大起：“哇！哥们，这娘们格老子到美，逗得俺心都软啦，乖乖不得了！”其余众人也看到了，也都色迷心窍起来，纷纷过来搭讪。

　　女郎气得脸都白了，猛地拔出宝剑，那些地痞仗着人多势众却也不怕，一闪开，又成包围之势，嬉笑不已，这个说句脏话，那个道句秽语，只气得女郎挥着剑乱戳，那些地痞原也是有些功夫底子，一时也奈不了他们何。

　　秦玄在一旁看着视若无睹，自斟自饮，看着他们像耍猴子一样得意，那女郎在众地痞包围之下，一时冲不出，急得唬人话也叫不出来了，只看这秦玄这边，可怜兮兮，只望他搭救。

　　秦玄眼看他们也玩得差不多，才懒洋洋站起来，喝道：“你们几个龟孙子给爷爷格老子闪开！”那几个地痞猛给人一喝，先是一惊，但眼看是个懵懂少年，一点惧意也无，有几个掠起衣袖过来，恶道：“你这厮找死不是？”话还没说完，说话人还没来得及搞清什么回事便已飞倒在地。

　　余人见这少年出手惊人，知道不是寻常之辈，嘴里吆喝着纷纷窜逃而去。

　　女郎见这少年神威，先前不快大去，对这少年充满了好感，但嘴仍硬：“哼，谁让你救我的。”但是她眼神里的爱慕之意却是逃不过秦玄眼睛，秦玄轻笑一声，正要说话讽刺，突然先前那几个地痞去而重回，奇怪的是人人都血流满面，或被人削去耳朵或被人斩去鼻子，十分狼狈，人人面如死灰，来到女郎面前，全体倒地嘭嘭磕头不迭，口称：“谢姑娘、不，谢仙姑饶命哪！……”这大出秦玄意料之外，不只他们到底发生了什么变故，连女郎也颇觉意外，突然她望见远处一人，全都明白了，喜出望外大声叫嚷：“爹爹！……”便奔跑而去。

　　秦玄顺着她跑的方向望去，只见路中心立着一个人，黄衣黄袍，头戴秀士方巾帽，也是黄色，这人长须飘飘，面容甚是俊美神丰，但是眉宇间却掩盖不了那一股杀气，让远在数丈开外的秦玄也忍不住打了个寒战，他知道，此生遇到的最厉害的对手来了。

　　那人一步步向秦玄这边走来，二话不说，手掌一人一印，转眼间，这七八个活生生的地痞已脑浆涂地变成死人了！

　　那人眼角也不抬，自有一股逼人而来的贵族之气，只望定秦玄，冷冷道：“你知道得罪我宝贝女儿有什么下场的啦？你也是条汉子，自己解决吧。”女郎拉着父亲衣袖，满是一副撒娇的架势，哼一声道：“我告诉过你嘛，我爹爹可不是好惹的哦，你可知道我爹爹是谁？他就是‘半蜀王’谢九亭啊！意思就是四川半块天下是我爹爹的！”得意之极。

　　秦玄听见“谢九亭”三字，便犹如晴天里一声闷雷，直冲心口，果真是惊愕地望着谢九亭久久无语。

　　父女俩都道这少年听到这名字吓坏了，谢九亭冷笑一声，挥手便要拍去，这回竟是女郎叫止道：“爹爹不要――”谢九亭回头看她，笑道：“丫头，敢情你看上了着小子了？呵呵，我道整个四川没你看得上眼的呢。”女郎也不害羞躲避，川女自古豪放，这些儿女私事不象世间女子含羞答答，只微笑地望定秦玄，仿佛望着自己的猎物。

　　秦玄只看了谢九亭一眼，就知道以自己现在的功力，十年内是无论如何和赶不上的了，只是，师父的丧身之仇，日夜没齿不忘，让他如何能够白白看着仇人在眼前走过？

　　他懊恼且迟疑，无意间眼神与女郎对接，那一瞬间，他已作了此生最大的一个决定，他决定也向女郎报以微笑，笑里有挑衅、有痛苦、有怜悯，一时间百感交集……

　　小说就这样结局了。

　　我的《江湖情幻录》就在秦玄这一微笑之间结束了，结束显得仓促，可是我不这么觉得。我知道，这样的残缺的结局对于一部武侠小说来说，简直就是败笔，明天刊登出去，读者肯定会来信骂我马虎，可是，我就愿意这么结尾。

　　这也许是我第一部也是最后一部武侠小说了。

　　我曾经说过，只有那些心中存有激情、存有纯真、存有梦想的人，才能写出完美的武侠小说，可是，我现在已经失去了这些，失落在这个编号2046的房间，失落在1962年的冬天，我，再也没有能力写武侠小说了。
