序　言

　　我十七岁就写诗了，但我却不是诗歌的狂热痴迷者，相反更多的是音乐的狂热追求者。我是抱着对诗歌怀疑的态度写诗的，我相信诗歌可以更多样化些，而不仅限于抒情或叙事。

　　这本诗集里的三十多首诗是我写诗四年来的一个成果。在这四年里我写过不少诗，最后剩下的只有这些风格并不太统一的”精华”。我不相信诗歌可以大批量生产，写诗是一个长期精炼的过程，有些诗在初稿完成后大约过了一、两年才找到了它最后完成的方式，对于我来说那就像作曲家作曲一样。我必须说，音乐对我的影响很大，如果没有音乐，我将不能理解语言的种种奇妙的变化组合。
