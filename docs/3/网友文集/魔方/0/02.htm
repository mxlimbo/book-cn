残旧


翻开一本边际发黄的旧书
一九九零年出版
两块六角五分

而现在是一九九九年
这种价钱只够买两瓶汽水
打一次公共电话

九年，足以令封面残旧
一个女人的脸蛋贬值
我想起那时简朴的生活
如今已变得奢侈
那时我还是个十岁的孩子
欣赏大自然
所有的思想与我无关

直至这本书被贴上标签加上图章
在图书馆陈列如一只标本生物
不再被人注视
在文字的浅滩搁置
隐匿在沙砾深处

多年来只有两个人借阅过它
我是第三个

1999年
