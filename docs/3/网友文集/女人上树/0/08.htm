穰草

　　郴州幸自绕郴山，
　　 为谁流下潇湘去？

　　 -秦观・踏莎行


　　 1．


　　公元1998年10月18日的夜晚刚刚开始，我坐在一栋写字楼的一张桌前，对一个词进行想象或者研究，这个词就是："穰草"。在我想象的路上，我总会侧过头来，这已经成为我的习惯了。就在我稍稍移眼的一瞬中，我看见无数的灯火在窗外闪耀，在亲切而又温暖地闪耀。

　　我感觉到自己有点异样，像被什么轻轻托举起来，之后，自己向那边飞了过去，很快地，我又飞了回来，如一只蜜蜂重新叮上一枝花朵上。

　　没有任何东西阻碍我在这个词语中的跋涉，前进，或者说穿行。

　　在这里，我倍觉幸福，黄昏的霞光披了一身的幸福。我昂了昂头，没有任何东西阻碍得了，没有。黑暗中没有人存在，只有巷语，车流声，还有属于自己的呼吸。"穰草"从读音里分离出来，慢慢的游离，游离，变成一个具体的所在，瞧，它鲜明的堆在那儿，很坚挺，蓬勃有力。你能不能说这一切已经在虚构之外呢？


　　 2．


　　我听见了一种声音。很绵长，细腻，似吟哦，又仿佛什么人在叹息。我想，它从哪里来呢？那么如此这般，夹杂一种迷人的草香，还有草雾漫漫的味道。

　　我双目注视着，"穰草"词语开始闪动起来，这就有点意思了。

　　我首先看见我自己，液晶晶的裸体从草垛上滚了下来。一遍一遍的滚了下来，犹如一根陈木从山包上滚落，夹裹着风与木制造出来的一种声音。似乎是夏天的夜晚，隐隐发蓝，在无限远处无数星星是散落的镍币，小时候，我对金钱没有感受，只是待到长成修长的十八岁的时候，它才成为钱币，才成为活着的钱币。可以触摸的物质。十八岁和钱币把我固定在那个夏夜。

　　在我家村头的老槐树向西有一个草堆，那是一个叫凤霞的年轻的寡妇家的，我每次放学路过，她总会与我相逢，手腰之间一个穰草，步伐轻盈。她是带穰草回家的。这个年轻的寡妇总是报以一笑。因为这灿烂的美丽一笑，我想，我那时最美丽的造句，就是"我变成了一个穰草"。"穰草"和"我"在那个美丽寡妇的手腰间，不断交替。一会儿是我，一会儿是穰草。

　　我怎么会这么清晰的感觉到那女人手指在我的身上呢？

　　我最初的想象生活就是那个叫凤霞的寡妇怎么样来对付我，她肯定很平静，慢慢的解开中间的草腰绳子，再一小撮一小撮的放入火中。肯定如此，像对付穰草一捆。

　　

　　 3．


　　当我再次转首时，在窗户玻璃上出现了我的脸，这么看过去，我的脸不仅仅贴在玻璃上，还贴在夜晚的灯火上。灯火与黑夜要使我的脸分崩离析，然而，玻璃又在固定住了我。这种悖反的想象还是使我得到了一种安全感。但是我的心还是跳得厉害。很危险。我甚或担心自己在这个夜晚窒息，心脏骤停，届时，"穰草"必然会渐渐萎顿下去，成为生命中的一个菌，或者其他什么，鲜为人知。

　　现在是哔卜哔卜的火声。我知道那是凤霞灶堂里的火焰在笑，在哗动。犹豫而又彷徨。门或者窗一直未被我敲响。我一直激烈的想象她的睡姿，这个叫凤霞的年轻的寡妇美丽的睡姿。这样秘密的夜晚又很多个，对一个女人美丽的睡眠的猜想，使我的想象力长了很长，有了力度和深度。我应该感谢那一段生活，那无数个夜晚。

　　向你致敬，夜晚，夜晚中白棉花一样的身体。


　　 4．


　　在我十八岁时有两件事值得纪念，一件是下面长出琶，另一件是拥有一张面值一元的纸币，二者都是无意间的事情。有一天，放学归来，扫完地，喂完鸡，搬出澡盆。下面我就要洗澡了，这几乎是我的夏天晚课。我坐在燥盆里，澡盆里的水不深，很浅，刚好把屁股埋下去，水随着手一划之后，轻轻漾动了起来，这时候我似乎有了一种坐船的感觉，盆不在水上，而水在盆内，这是很奇怪的。我开始寻找，这种感觉从哪儿来的，我开始发现在我的叉开得裆部有一团黑色的东西，像一簇水藻。原来，感觉来自这儿。对那个年轻的寡妇的想象大大损伤了我的智力。起初，我还以为什么东西落进洗澡水里，一个黑绒毛的小鸡或者其他什么。然而，当我的手去捉的时候，我捉住的不是小鸡，而是我自己。我一下子惊呆了，原来有一样什么东西悄悄的赋予了我。青春从来不和人商量就秘密无意间来到你的体内，眼光中。这是我此后明白的道理。当时，我被自己吓住了，我清楚的记得自己从澡盆里跨出，赤脚飞奔进屋不知所措的形象。
此后，我在一面镜子里第一次发现唇髭上的胡须，一根又一根，即细又长，即密又黑，那还是一面有点破损的镜子，像一个三角形挂在南墙上，我的面孔在这个三角之中被反复的固定端详着。
穰草堆一天一天的矮下去，我看见凤霞，我还看见她手腰间的穰草。

　　为了能接近她，我为自己编造了一个理由，有一天，我随手拿了一个白色的蛇皮口袋开始了拾废铁的生涯。整个暑假里，人们隐约发现我在凤霞屋后的一片竹林里来回逡巡。除了竹叶，鸡屎，砖屑，并没有什么废铁。然而每天傍晚时分，我躲在竹林隐秘之处所见到的，是凤霞上完厕所系裤带的样子，那给我留下了很美好的印象。我至今还能想起那一张满足又安宁的被一抹夕光所照耀的俏脸。我母亲看见我那拿了一个白色空蛇皮口袋出去，同样也是一个白色蛇皮口袋进门。她只是责备的看了看我，她没有说什么。我在母亲的目光中，一低头和白色蛇皮口袋一起窜进了屋内。出于某种证明，我在第二天黄昏回家时，白色的蛇皮口袋里装满了破锅，烂铁之类，它们在我的背上快乐的碰撞着，碰得我心花都有点怒放了。在走到那个穰草堆跟前时，我和凤霞再次相遇了。我背着一个鼓嚷嚷的白色蛇皮口袋向她走去，她空着干净的两手走了过来。她向我一笑，继而问我，背的是什么？是钱。我快乐的答道。


　　 5．


　　第二天，我去了集镇。这是我第一次出远门，身上还背着快乐的废铜烂铁。集镇上的人很多，但我无心观看。我在搜寻一个肯花钱买去我身上这么重的东西的人。我几乎累坏了，不过，我被一种隐秘的希望支撑着。那是在经过村头那个穰草堆时产生的。这天的天气不太好，下过一阵雨，好在又停了。路面是雨水的反光，我赤着脚就出发了。雨后的太阳照得穰草堆幽幽发亮，草堆腰间部位有一朵小花引起了我的遐思。我边想边往集镇走。我放下蛇皮口袋，我确实累坏了。一个麻脸的卖苹果的老头显然被口袋重重落地的声响吓了一跳，他说，买废铁的，往前直走。我还向别处张望，并没有听清那老头的话，怕我挡住他的生意么？他还有力的挥了挥手。我背起蛇皮口袋向那张微微跳动的麻脸投去了最后一瞥，跟着就上路了。

　　 这条集镇大道我走了整整一刻钟，一个钟头之后，我才在一家叫"日升商店"的屋后找到了废品收购站，找到了那个人。他站在一张破桌子前。头顶上废品收购站几个红漆大字歪歪斜斜的在闪光。他站在那里的形象在生锈残铁的废物所构成的背景前，显然很凶悍有力。我在接过钱的时候--面值一元的纸上面还有一点污迹和折痕。那个人的手背上手臂上的毛很茂盛，我的视线牢牢地粘在上面，沾满了汗迹和锈污的上面。就这样，无意间，在雨后的清新的阳光里，我拥有了一元钱，这是我自己的一元钱。我第一次见识金钱的感受，真是难以言表。我母亲说我"愿意怎么花就怎么花"。

　　我在路过穰草跟前所起的隐秘的愿望并没有实现，因为我走遍了集镇大道也没有搜寻到一朵花，一朵用布绒制作的花。那一元钱的纸在我的手心隐隐发汗。应该承认，我对花的认识少得可怜，我对这种朴素，色彩十分鲜艳的布绒花的热爱，是穿过我们村庄的货郎担架上获得的，它插在绷直的绳缝里，微微在颤。


　　 6．


　　尽管如此，我仍然很快活的走在回家的路上。我朴素的想：既然有这样的东西存在，它总将会出现。当我靠近那堆穰草堆时，我愉快的心情稍微暗淡了一下。我继续往前走，但心里总有点遗憾和愧意。因此，我的快活也就慢慢的平息了下去，它使我获得了正常的呼吸，还有行走。她来了，我想做错了事一样的眼神看了一眼向这边过来的凤霞，年轻美丽的寡妇凤霞。我有点为难起来，我很想告诉她我准备为她买一朵布绒花，那种色彩鲜艳朴素的花，简单而又不朽的花。可是除了一个空空如也的白蛇皮口袋，什么也没有。我拿什么问候你呢。我窘得脸发红，站下来回答她的问话，却有点语无伦次了。"我……我去……镇上……上了。"她伸手捋了捋额前一绺刘海，还是笑了一下。随着草堆轻轻一动，我知道草堆又矮了下来。

　　她夹着一个穰草走了不到五步，想起了一件什么事似的停了下来。她说。

　　"卫国，晚上过来帮我做烧饼，啊？"

　　无疑，这是一件美差。

　　说真的，那时候，我真有点不知所措，我只是轻轻的嗯了一声算是回答。我镇定了一下，不过很快的，我的快活心情重新又飞了回来，还带回来另外的物质。我几乎蹦着跳着出现在家门口的，我母亲一见到我一扔下蛇皮口袋，就首先告诉了我要晚上去帮凤霞做烧饼的事，原来，她已经和我母亲说过了。然后问我今天卖铁几分钱一斤。我说五分钱。还平静的竖了一下巴掌。我有点不动声色的样子，使我自己很满意。

　　天一擦黑，我吃罢晚饭一推饭碗就过去了。在去的路上有一直拉长身子的黑猫吓了我一跳以外，其他没有什么。我老远看见凤霞的灯在厨房里闪着，我怎么看，都像只在笑的眼睛。
我一撮一撮的将穰草送进灶堂。透过灶洞，凤霞的脸在灯火的照拂下那么红润，美丽，又那么安详。
我总是不留神把火烧的大了，旺了。她总是说，小一点，小一点。压住。我立即用火叉在燃着的穰草上重重的压上。


　　 7．


　　1998年9月到10月间，我始终被一种莫名的感伤所笼罩着。当我走过城市广场的时候，心里总会涌上来一股泉水般的哀痛。眼前的她的影像久久不去，在我的生活不远处，她仍然活着。她用干净的双手从穰草堆里拽出一捆穰草带回家，不停的在那条路上走着，来来回回。是的，我几乎不敢想象她骑车在桥上落水的景象。我无数次站立起来，看见虚空的一个点上，一个叫凤霞的年轻的美丽寡妇在溺亡，我又无助的无数次的坐了下去。随着她的逝去，我的那个美妙的夜晚还存在么？她的尸体在离桥不远的水花生里发现的，人们耐心的找到了她，她的自行车。人们将她的自行车从水中捞了上来，那是一辆崭新的凤凰车，车轮在空中转动了起来，水滴到河里，桥上。转动的车轮在阳光里使人头晕目眩。

　　我不知道我们是如何到达东房的，东房的床上一床猩红色的绸被散发出一股诱人的气息。三门橱的穿衣镜有一个蓝色的布帘子，挡住了我们慌乱的向床头靠近的影像。很快，她脱掉了衣服，在她脱去最后一件小汗衫的时候，她上竖的胳膊根部露出的腋毛使我有点迷茫，我看见她的高翘的乳房随着衣服的除去轻快的那么跳动了一下。那么跳动了一下。这使我获得了一种甜蜜的印象。我坐在那儿不动，就像一捆穰草面对天空、村庄一样望着她美妙的躯体。

　　她伸过来那双干净的手，像解草绳腰一样解开我的裤带，我在自己的喘息声中注视着，她和我的想象中稍稍不同，吹气若兰，鼻息如风。她的手有点颤抖，远不像想象中平静，舒缓。她的脸上弥漫着一种活泼的色彩。她慢慢的恢复了平静，一撮一撮的将我送往一种火中。忽然间，我猛地说，我们到穰草上去，到穰草上去。她显然被我这一句话搞糊涂了，她的那双干净而又修长的手停了下来，停顿在空中，我的胸膛的上空。


　　 1998．10．16．
附后记：
　　读完我这篇小说的一个朋友向我打听那一元钱的下落，我在阳光的街头对她说，或者在那个美丽的寡妇的床头，或者并不存在。另外，我想起了那个夏天两件无意间生成的事（详见本文第4小节），实质上，在无意间完成了了人类的某种关联。这是我针对这个故事引起的最远的一个思考。
