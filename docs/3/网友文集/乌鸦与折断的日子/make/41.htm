李敖和《北京法源寺》

　　正如愤怒的，滔滔向前的潮水，一再地掀起了狂风巨浪，惊天地，泣鬼神，一路挟裹着所有能够带走的一切，无论是亭台楼阁还是废墟残垣，泥沙巨石，还是生命与污垢，呐喊和嚎叫……我又感受到了李敖所特有的那种激情，那种悲怆，那种凄美壮观的侠骨柔情。而李敖就是李敖嘛！还没有第二人。也许正如他所讲的，真正理解他的，只有他自己而已！这是清醒的悲哀抑或是难得糊涂的无奈和酸楚？谁知道呢？

　　对于戊戌变法（还有鸦片战争，抗日战争等等都应该，但是却没有更波澜壮阔的的作品来纪录展示，实是我们中国人的一大憾事！那种悲怆，那种沉重，灾难深重的中国人啊！）等诸如此类重大的题材，这些可歌可泣的英勇不屈的志士仁人，这些思考过，痛苦过，抗争过，也失败过的英雄们！他们慷慨激昂横眉冷对千夫所指，他们义无反顾走向断头台――血债是要用同物来偿还的！（他们当然知道，但是世人不知道啊！）他们有大气魄，有大眼光和大手笔，他们用新的思想来呼唤和呐喊，来鼓噪煽动，他们以自己的生命为代价与旧势力，与那个破败腐朽了的，垂死挣扎着的女人和集团，与狞笑着的死神和人间地狱拚搏着，抗争着，绝望着，希望着。他们视死如归，他们"笑谈渴饮匈奴血"无怨无悔，他们用自己的血肉之驱铺就了的，不过是一条仍然崎岖不平，依旧危险荒芜寂寞孤苦的路，但那毕竟就是一条路，使后来者可以继续前行或者另外披荆斩棘开出新的路径来，为了那只在火焰中歌唱着的凤凰，为了一个民族和国家的重新崛起……

　　我赞叹，我欣慰。为这部作品，为有那些可敬爱的人们！

　　这样的题材的确是只有李敖这样有思想和魄力和亲身经历的始终战斗着的人来表达，用他一如既往的宽广的视角，那铮铮的铁骨（和英雄惜英雄的所见略同，铜豌豆的骨气和象征与自喻）和力透纸背的理性的表达，至少是现存的写重大历史题材的作品中最好的。我以为。

　　我自足，也不无忧虑和遗憾。

　　这部波澜壮阔的作品也并非没有缺点，而且是大缺憾。大概是良莠各半的比例，那就是这部作品的情节未免松散做作（确实像个卖弄风骚的絮絮叨叨的瘪嘴老太婆），像一个恶劣的裁缝剪制的蹩脚衣物，布料虽然是上好的，但是穿在身上怎么瞅都不大顺眼，不大协调，不大舒服。虽然或某一个角度看是很突出的，很优秀的，很精绝的。

　　人物塑造未免空泛，苍白，没有明与暗的对比，没有人性的深层次的挖掘（除了纯粹思想方面的），没有诸多矛盾（不仅是社会的，还有人性自身所无法避免也无法回避的那些矛盾）的对立和激化，和消解。他们有点太完美了，太纯粹了，太多地表现他们直面人生和死亡的悲壮，却忽略了人性中不可能回避的那些狭隘的，自私的，男女的，甚至是世俗的爱和恨，怨和愁，除了在"变法"与"革命"上的短视和保守方面的揭示，其他的呢？甚至连点"欲望"的填充都没有，要知道，爱和恨是不可能分开来谈和做的（而且对于这些仁人志士来说，完全超脱了世俗人正不择手段享受和报复着的爱与恨，是不可能的。只有写出这些来，才会更加波澜壮阔感人肺腑，也更真实一些，他们不可能不具备这些东西和经历，即使是一部分也好，但是我没有看见。），他们也不可能。因而便只有千人一面，千人一语（好像就是你李敖一个人在不停地"夸夸其谈"，在分解装扮自身，现身说法设坛祭祀一样。虽然谈论的思想很精深博大，有觉悟和感召力，但毕竟让人觉得那"只是你的思想"而已――要知道，他们还没有那么先进，他们的局限就是他们根本不可能超越他们那个时代的种种限制！），他们的经历，他们的性格，他们无法回避的自身的种种矛盾，那种矛盾后不得不去决断的犹疑和最终的做出痛苦决策的心理，以及他们面对自身无法避免的悲剧性命运时的无奈与落寞，与凄凉，与豪迈，都没有很好地表现和把握，展开，铺排。有些牵强，有些支离破碎，有些叫人无法理解和相信。实为最大的失败之处也！

　　虽然我相信你李敖，你的见解和思想。但是作品是另外一回事。

　　尤其是语言缺少必要的变化和修饰，这也许是李敖写小说的最大的障碍么？至少我觉得现在是这样。

　　不可以，也不应该叫每一个出场的人都用一个腔调谈论，这是他李敖没有想到呢还是做不到呢？不管如何，有事实在。大家可以看书去。康有为，谭嗣同，梁启超等等人的语言都没有区别出来。这一点在谭嗣同与大刀王五他们交往时候的描写中尤其明显，他们（王五们）的侠义之情与一般的武侠小说中的所谓"侠客"的说词简直没有什么两样，平板，苍白，甚至是死气沉沉，有血无肉，有教条的言语而无人的真实的形体再现，无法感觉，也无法去触摸，只有模模糊糊的影子晃来晃去股。

　　但是，他可是有血有肉有见地有豪气和侠骨柔肠的，轰轰烈烈活过爱过恨过的"大刀王五"啊！

　　谭嗣同的性格界定得太教条了，他，以及康有为（他总是一个模模糊糊的影子似的人物，文章开头时候还是一个性格鲜明，有激情和思想的跃跃欲试的人，但是后来就被淡化了，如那"堕水成离"的"依然的落花"，只是一朵落花而已！随风，随水漂去了！一直没有找到自己的根。）等，都不可能超越他们的时代的限制，然而他竟然对时局，对人事和前途看得那么清晰，那么准确，那么的自信――甚至是对死都那么坚信（他从来没有动摇过么？也许是吧！但是，其实这对于谁来说都要有一个渐变的过程，至少是要做到叫人可以相信的程度。当然这不在于自己有多少慷慨激昂的表白，像谭嗣同那样一而再再而三地解释给大家听，就不太合乎谭嗣同的个性。我以为。他是那种说出一部分，然后做给人看的真豪杰和真的汉子！

　　所以他才会有那样一首感人至深的诗留存万世！

　　还有，这个决定也不是一下子就可以决定的，他的心理变化路径呢？历程呢？他真的那么坚信么？还是在内心中独自默默承受着，痛苦着也快慰着――能够为了理想抛弃自己生命的大义凛然的人可是不多的！因为生命是如此的可贵和值得珍惜，是只有一次而没有第二次的――这也许就是他们对于佛教有研究和看法，同时也揉进自己一生旅程中去的一个原因吧！但是这一切我都没有看见，除了总是一模一样的慷慨激昂的论调（多了反倒叫人麻木了，漠然视之了）。我倒是愿意相信，他谭嗣同一直一直地博览群书，为了国计民生苦思冥想（甚至是从佛教中去寻求，去发现，去失望和痛苦），四处奔波，广交天下友，就是想要为自己的判断和决定找到最真实最可信（至少是对于自己而言的）的理由和例证，他不是先知，也不会自诩为圣人（或者是如什么"康圣人"之类的古怪叫法，在这方面只有鲁迅是最清醒的，同时当然也就是不被人理解的最深重的，自斟自饮的苦痛和寂寞！），这其实就是他的清醒的，真正可以叫做人和伟人的区别与可贵之处。也是他与别人不可能一样的地方，他的伟大与可敬爱也在这里了。

　　关于梁启超，李敖说是他比较喜欢的一个人（康有为是自比？！），他不固守自己的思想，总是能够为自己增添新鲜的血液，因为他不是为了自己个人的欲望或者得失而计算和斤斤计较的人，他能够以民族和国家为根本，为指针，从而调整自己的方向。因而他为了真理并不回避过去的自己，甚至不惜伤害自己老师的感情！但是，同样的偏失也存在于他身上，有点不丰满，有点松散和苍白，不真实可信。

　　总之是，这些称得上是主要人物的，都有些雷同，单一，过于完美。整部作品不是一面完好的多棱镜，折射出灿烂的七彩的阳光给世人。顶多有一两色夺目的光辉。

　　其他人的塑造还是可以的。往往是着墨很少的人却有了一些深长的余味。

　　另外，正如在李敖的一篇杂文里说的那样，中国传统文人由于思想和认识的局限，不大可能发动自下而上的变革和反抗（由于与自己所受的训教和传统文化相悖离，想是都不敢想的，何况是逾越雷池一步呢！），他们只是信奉一切归于皇上一人，一切变革也可以由皇帝一个家伙来颁布实施（不管他是否手握生杀大权或者实实在在的权力来贯彻执行），他相信这一点，相信只要皇帝愿意，（或者叫他愿意）就可以了。他们也只是为这一点而努力做着"踏踏实实"的准备，但是，他（他们）都错了，这其实才是他（他们）悲剧的根源。当然，他谭嗣同也不是没有怀疑，只是他们还是太相信。

　　这一点李敖看到了，也写出来了，但是还是缺少脉络，缺少这中间的悲苦和彷徨，缺少人性的深层次的展示。还没有到高潮就草草结束了，总是叫我感到欠缺和遗憾。书中《楔子――神秘的棺材》所表现出来的余味没能继续深化，那种有些残酷的东西没能始终贯一，所以叫人悲哀得不那么沉重，至少我是这样的。

　　但是，也许李敖写此书的目的之一，不过是为了表达自己的见解和思想罢了，就这一点来说，他做到了，而且很完整。

　　我是先谈《北京法源寺》，再来谈李敖的。那还是去年，偶然在网上看见了他的文章，我才知道有李敖这样一个怪怪的人，看了他的作品后我连声叫绝，坐立不安了。真是苍天有眼，降下这样一个顶天立地（虽然他那么瘦小枯干但却那么精神矍铄精力充沛精神不死！）的人，他思想上天马行空不拘不束常有惊世骇俗的真知灼见，行为上放荡不羁任人指骂叫嚣诬蔑依旧我行我素我口对我心我笔直我言，壮哉！抑或暗自庆幸哉？他当然不是完人，但他绝对不是某些标榜"正人君子"，实为古旧衣钵的卫道士眼中一个很大的"坏蛋"。他是真理的发现者和维护者，是一个伟大的思想着同时也表达着人，他是一个智者同时也是部分的被"欲望"左右着，"放荡"着的拜伦和歌德（但是，正如叔本华白天骂女人，晚上找女人一样，人性中的某些方面是人不可能超越的。但这才叫做真实的人生！）他骂国民党时，竟然把脏话发挥得淋漓尽致入骨三分！简直是敢言他人连看都不敢看，何况正视的那真理！这才是真声音，是真思想的表达和呐喊，是负重历史的痛苦沉思和无奈回顾，是新时代的喉舌和鼓噪者，是先知也是勇往直前身先士卒的勇士，是混乱社会独一无二的反叛者，和无时无刻不在抗争着呐喊着的斗士！是大闹天宫对抗天庭指责玉帝的那个泼猴子……

　　但是，这个人也就是李敖呀！

　　我所深深尊敬和爱着的李敖！

　　

　　我还是用这句话来做这篇文章的结束语吧！

　　"以为倘要完全的书，天下可读的书怕要绝无，倘要完全的人，天下配活的人也就有限。每一本书，从每一个人看来，有是处，也有错处，在现今的时候是一定难免的。"这是鲁迅说过的话，我以为是对的，就拿来说明我的看法。

　　我不相信"他们"是"完全的"和"配活"的有限的人们，至少在作品中不是这样，当然也包括这部并不"完全"的书。但是我还是要说出一些看法来，让大家明了，虽然也仅仅是一家之言。

　　正如康有为他们无法超越那个黑暗的时代一样，也许李敖也无法超越自己，至少在某些方面是这样的。

　　这才是真的世界抑或是真实存在的人，是真的李敖，是不是呢？

　　还是让大家来说吧！

　　我就说这些.

　　

　　2000年2月11日
