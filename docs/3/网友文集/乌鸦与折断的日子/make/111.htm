回眸，只是为了更真实地面对！(二)

——一周新闻回顾

　　题记："信息和高科技依旧在勾勒着前途未卜的世界，它们都在诱惑着我们，正如历史本身，叫我们在有限的时空内寻找着，赞叹着，那伟大而悲壮的，抑或是平淡而五味的人和事件终于凝固成一个个短短的瞬间，成了一件件等待被挖掘的残砖断瓦，或者是永远被尘封着的棺木和简帛，那软弱的字句如同鲜血的颜色，被时间和记忆消磨着，这无法不叫你悲哀和困惑……然而，我们回眸张望，只是为了更真实的面对！"

　　腐败！腐败！到处可以看见腐败的阴影！人和权力的，归根到底是权力的腐败！如这满目苍凉的秋色，不容你有回避的可能！

　　我们的世界怎么了？为什么会这样一再尖锐地刺痛我们的心灵和肉体？叫我们不得不变得麻木和习以为常？或者出离于愤怒而无话可说？

　　生命在继续，思考在继续。不！这不应该是我们的现实，我们仍然要来关注，来探讨，来评价和改变！

　　如果我们能够！

　　一方面是经济的飞速发展映照出一篇的光明和希望，一方面却是我们各种制度下面的腐败在滋生蔓延，如洪水猛兽般鱼肉着百姓，如旺盛的野草，"割去一茬又长出一茬"。

　　我止不住要问：

　　为什么会这样？

　　连李鹏都不得不站出来承认"三峡工程确实发生了腐败"，"问题很复杂"，但是"这种现象将会得到克服"的。

　　是的，这种种现象都会被"克服的"！我们有理由这样相信和希望！这样来改变！　

　　又是公安，总是公安的话题引起我们的关注。毕竟是他们在守护着我们的安宁！

　　对于"公安机关与企业手拉手"的事情我们大多耳熟能详，从酒店到黑社会，到山东某公安高级领导的"通风报信"。但是对于公安介入移动公司，帮忙讨债，我们还是第一次听说。

　　那么谁对谁非？如何来看待和改变？在9月6日的"行为事件"中有振聋发聩的评论。

　　任何一个稍稍关注社会的人都不会忘记，那因运政人员耽误时间而母子双亡的恶性事件。对此我们应该用怎样的道德来衡量和评价那作为人的生命的价值和权利？法律又是如何来规定的？你可以参看《孕妇即将分娩　执法者拒不放行——秉公执法还是严重违法？》的讨论，虽然仍就是沉痛的叫你无法面对，无法挽回。

　　《收容站滥抓无辜搞创收 遣送者层层批发逼死命五条》又是五条无辜的人命案子，"收容遣送"到底该如何收容？如何遣送？我想，有各种条例和规章制度在，有法律在！他们如此草菅人命并非是因为无知，而是缺乏人性！对此我无话可说了，因为我们的一条命是那么的贱，被"100至200元"地买卖着，像批发的商品！当有人不堪忍受"手铐和警械"的虐待，在集体逃跑时"酿成了5人死亡、4人失踪的恶性事件"时，在不足一页"情况说明"中，我们的领导，我们的父母官这样"理直气壮"地回答：

　　"没有任何消息，我们也不负任何责任！"

　　呜呼！你说我们还有什么话可说呢？除了愤怒和悲伤！

　　除了腐败，那《常德"9·1"运钞车大劫案》七死四伤的惨状谁能忍住悲痛的眼泪？谁能忍住悲痛的哭诉：

　　仅仅因为并不属于自己的钱，你们为什么要这么凶暴残忍地对待这些无辜的人？就可以剥夺他们年轻而美好的生命？

　　为什么?为什么？谁来回答？

　　这么多事件，充塞着我们已经拥挤混乱地的世界，改变着我们的生活，影响着我们的情绪和思想，对于"权力的腐败"和"腐败的权力"的世界和事件，我们如何看和改变？"权力变异是导致腐败的根源"，但是权力为什么会变异？"极端的权力导致极端的腐败"，那么我们的监督体制是否健全？我们的老百姓的呼声谁来理睬？

　　《温江村民七问村官》是一个很好的开始，那些话是我们憋在心中多年的了，可是回答呢？空洞苍白得像例行公文，避重就轻闪烁其辞！这便是我们因为"文化程度不高"而认为开出"１３７万巨额白条""并不奇怪"的村干部！对此有人说《减负先减官》，尤其是《批文作交易　百余万落腰包……》这样的计委干部必须要让他们"栽"得惨重！而《用人无责——官场腐败的制度源头》也许更是一个很好的视角，就象是一把切开我们政治体制的刀子，把那缺陷和遗憾摆到台面上来了！

　　大家怎么看和想？

　　是的，大家到底应该怎么看和想呢？怎么去做？

　　转移目光，让我们来关注其他。

　　《行为天下》栏目直接目击历史真实，"联合国迎来最盛大千禧元首峰会"是不能不来关注的话题，虽然还没有更多热点来震撼我们的世界，但是一些可爱的花絮却充满了脉脉温情，叫你唏嘘，也叫你忍俊不禁的发出会心的微笑，因为那"联合国内记者大睡"、 "手铐告急"与"社会主义万岁!"的图片报道会引你更深入地思考和面对这个世界！ 　　

　　《与行为对话》隆重推出关于历史的回顾，有并不相同的《检讨书两则》来检讨和对比历史。也有老诗人、老作家邵燕祥的文字《自传及交代材料提纲》帮你拨开迷朦的雾障，来直面历史的真实。面对那个极权的时代，面对那些被扭曲和压榨着的人的悲苦和无奈，我们能说什么呢？我们只能希望那些历史永远变成历史，而不再重演！

　　我们多灾多难的民族已经无法承受这样的打击了！

　　因为，刚刚掀开黑幕的一个角落，便已经够触目惊心的了！

　　这些是太沉重的记忆，都已经变成往事，不堪回首却又必须回顾！

　　因为，回眸，是为了更真实地面对！

　　2000年9月8日
