天凉好个秋

　　天气是渐渐地凉了，而秋色却并未高远得那么叫人兴奋异常。落花是缤纷的，黄叶斑斑驳驳，飞舞旋转如翩跹的蝴蝶乱你的眼睛。红红的太阳悬在清亮亮的天的深处，总有种莫名的凄凉和惆怅的味道。云朵是堆积起来的，像纠缠在一起的巨大的海藻，疯狂地刺激着你的想象。例外的只能是田野，黄澄澄的稻子静止成浩瀚的海洋，给人以欣喜和安慰。鸟雀却是机灵欢快的，围着呆立不动的稻草人威吓着，唧唧喳喳品味着深秋的成熟的果子。几个农人叉腰而立，那沧桑的面容是一幅幅惊心动魄的素描，深得可以淹没你的世界。

　　然而，这样的境界也并不总是与我有缘，我已经多么年不见了。如实说，这些不过是我脑海中诗意的想象罢了。

　　今年的秋天总是没完没了地下雨，哩哩啦啦的小雨，大雨，大暴雨，偶尔强劲尖利的风旋转着，劫掠着光秃秃的枝桠上仅存的几片黄叶，夹杂着雨水的冲刷，愈发显得清冷，悲怆，叫你满腹莫名的滋味，像那屋檐下冲刷得伤痕累累的青石，自然而生动地裸露着长满苔藓的面容。

　　我一天天龟缩在阴森森的屋子里，闻着潮湿的叫人压抑而又感到空虚的气息，独自观望着外面浸泡在水中的湿淋淋的世界。如同看见日渐苍老的自己，渐渐被暮色淹没。

　　然而，那叫我如此牵肠挂肚的，那美丽的、高远的、清凉的秋天在哪儿呢？那隐没在太阳明亮的光辉和雾霭中的群山在哪里呢？他一声声呼唤着我的名字的童年的玩伴在哪里呢？

　　老了！一声叹息！便如同这被浸泡着的秋色，变得沉甸甸的了！

　　梦回故乡，依稀可以闻见浓郁得化不开的那秋色，那乡愁，那蓝得天高地远的穹庐下，忙忙碌碌的人们哼唱着低沉而欢快的歌子。这个声如甘冽的山泉水，打湿我的寂寞！

　　我寂寞。因为我没有再见到那个透明的秋天，那少不更事然而踌躇满志的孩子，奔跑在开满鲜花的山野。除了在伤残的记忆里，在无甚可想的想象中……

　　我忽然记起那悲凉的诗句，"茅屋为秋风所破歌"，然而它表达不了我心中的悲愤与无奈，我没有那一间可以遮风避雨的茅屋，没有，我不过是一个寄人篱下的过客，匆匆走过仓惶的岁月，走进注定中的死亡。

　　我反抗，然而我被这"秋风秋雨愁杀人"的寂寞围裹着。因着这寂寞，我沉默无语……

　　2000年8月9日
