自由撰稿人的心声

　　作为一名自由撰稿人，我时刻都将面临许多待以解决的问题。与编辑的沟通，以使得我的文章更附合要求和顺利的发表；努力去了解读者的需要，让他们喜欢我写的东西；思考读者对我提出的意见以及如何找到更多吸引人的话题，这样我在他们眼里总是新鲜的。无论是那一种情况，我都要在这许多的选择中确定我的方向。我当然可以停止我的脚步，但我明白，作为一名自由撰稿人，这是我应该做的，虽然这不是必须要由我来做，但从内心我愿意与很多人分享对不同事物的看法，我得告诉他们，我是这样想的。

我写我能看到的

　　对某一件事情的看法我不能埋藏在头脑里很久，因为我总忍不住要把它写出来。这可能是能成为自由撰稿人的动力，不断的尝试去做，不断的接受难以接受的事实，不断的告诉读者他们该小心什么。

　　我写我能看到的，这很重要。我无法去虚构一个情景来感动读者们，他们比我更能理性的看待他们能看到的所有作品。我试图建立的那种思维模式可能读者们对它毫无兴趣，而我却深陷其中。写作的灵感来至于我的观察及敏锐的感觉能力，我能体会到第六感对我所起的作用。尽管那些事物看起是平淡无奇的，但我总能发现其中的闪光点，我为读者们作着挖掘的工作。

　　jelly曾经在我的留言板上提醒我，你得继续写点新东西了，她期待我的大作。那段日子我为什么会停滞不前呢，我是否感觉到脑袋空了，灵感没有了吗？那双敲键盘的手好久没有打出一个字。我只是看着电脑屏幕，我只是想我该怎么走下去。作为自由撰稿人总会有消沉的时候，这种消沉会因为你的怀疑而更加能影响你，我对自己的怀疑曾经达到了极点，我怀疑自己真能写吗，我怀疑自己过去写出来的东西，真是我写的吗？我深处于自己的陷阱当中，我试图把那些想法简单化，我试图确定自己是可以胜任自由撰稿人这个称号的，而不是更多的把自己打入怀疑的仓库，我要关上仓库的大门。最终我做到了，我把怀疑抛在了脑后，而把自信带入了我的生活，我能感到自己散发出的能量。编辑采用我的文章，读者赞扬我文章写的真不错。机会一下全都撞在了我的身上，此时此刻我才感觉到，自信是很重要的灵丹妙药，特别是对自由撰稿人来说，面对来自你自己，编辑，读者的压力时，自信显得尤为的重要。它能让我继续走下去，无论这其中我能遇到什么，它都将提醒我，我写我能看到的。

我眼中的编辑

　　"很抱歉，这篇文章我们无法采用"，这是我听到最多的一句话，来自于编辑对我的最后承诺。而在那时，我会感到自己是否该休息了，因为我花了不少时间写出来的文章竟然被轻而易举的否定了。还好，我没有那么做，因为我换了一家媒体，那篇文章就发表了。我也能了解，什么是相互的选择，过去我只是认为只有编辑能选择着我。

　　编辑是自由撰稿人最喜欢的人，他们能让你的作品与读者们见面，而不是把它放在抽屉里。在我所经历过的投稿过程中，我总在担心编辑是否能领会我作品的意思，他是否会曲解我的中心思想，这种担心有段时间让我寝食难安。我虽然知道可以在投稿件之前再多修改我那篇作品，但这种感觉在把稿件投出去的瞬间就产生了，我是在担心编辑那边吗？担心有时候是对的，编辑并没有你想象中那么能抓住我作品的意思，他们有时候是以他们自己的眼光来看待我的作品。我总在想，编辑过去在我眼中就好像一个标准，一个鉴别你文章好坏的标准。而现在我却认为，编辑更多只是以他们自己的要求去看我的作品罢了，比如编辑喜欢什么，你却在文章中没有满足，那么编辑会提醒你，那里还差点什么。有一个现象能更能说明这个问题，同样一篇稿件，一位编辑告诉我，我的作品有些话不够简洁，而且语言风格有待改进。当我把这篇文章给另外一位编辑看时，他却告诉我，我的作品非常不错，他决定采用了。看到没有，这就是编辑，一个给你灰心，一个给你信心。你相信谁的。编辑总会是我们的朋友，他们需要我的出现，我有时候会给他们不断的惊喜。是的，编辑需要那种能时刻保持新鲜的自由撰稿人。

读者，你好

　　我感到幸运，有了媒体的存在，我的观点变得好像很权威了。当我一篇文章变为铅字的时候，我能感觉到读者的眼光及心里的变化，他们会暗自评论一篇文章是否值得去读，或者他们想指着我的鼻子告诉我，你写的太差劲了。对我来说，任何的赞扬及意见都是好的，它们都给我前进的动力，赞扬给我信心，意见给我机会来改进自己。

　　我愿意与我的读者朋友交流我和他们的想法，虽然他们无法用键盘敲出他们想说的话在媒体上发表，但他们对某一件事物的看法的确值得我去学习及思考。我从没有认为我会是比读者更厉害的人，虽然我在媒体上有一席之地。直到今天，我都认为读者是我写下去的动力，没有读者我的作品将毫无意义。他们说的每一句话对我来说都意味着他们想让我带给他们更多有用的东西，也就是说他们需要我。我感到幸运，能遇到他们，能和他们作双向的沟通，我和他们能开诚布公的说出自己的想法，赞扬和批评，接受和解释。从中我能感觉到，什么是我的价值，什么是自由撰稿人的价值。

　　当我一篇作品发表的时候，我能感觉到那一双双犀利的眼睛在看着我写的一切。每到这个时候，我都感觉到一丝的紧张。而每次收到读者的来信，我那悬而未决的心总算放下了，原来一切都没有想象中那么坏，我喜欢那种被承认的感觉。读者会是愿意帮助我的人，他们不是想让我难堪，他们只想让我告诉他们更多他们想知道的东西。

走的更远

　　时间总会超出你的想象，我们身边的许多事物在变化着。你所熟悉的编辑开始消失，你必须重新开始与新的编辑建立良好的关系。读者开始变得挑剔，你不可能拿那些连你自己都怀疑的文章去敷衍他们，而且更多的人和你一样，怀着美好的愿望加入到这个自由撰稿人行列里来。一切变得开始没有过去那么简单了，你得时刻警惕那种显而易见的变化。

　　作为自由撰稿人，我们都希望自己的文章变为铅字，这是否会令人感到兴奋，但在这之前，我知道我被拒绝了无数次。是什么信心让我坚持下来了，我想除了我相信自己外，更重要的是，我能感觉到，读者需要我。我并没有过搞估计自己，只是我总能给读者他们想知道的。我能带领他们从一个地方游历到另一个地方，每一个地方他们都能获得很多东西，我知道，他们下次还想跟着我去探险。

　　作为一名自由撰稿人，我能感受到那一份喜悦。我总能出其不意的告诉读者一些我想说的话，而我也有这个机会。有时候，我不停的追问自己，这种日子是否会持续相当长一段时间，还是很快就会结束，什么会让自己永保活跃的思维，什么会让我更加珍稀眼前的一切。我一直在探询自己所经历的创作道路上每一个步伐是否走对了，虽然我已经走到了今天，但这并不意味着我能继续走下去。我有这种念头，并不是我有开始怀疑自己的念头，而只是我告诉各位，作为自由撰稿人他们所经受的压力，有时侯担心的念头能让自己更想抓住眼前的一前，这也预示着，只有不断的听取来自于编辑和读者的声音，我才能走的更远。

结尾

　　来到文章的结尾，我开始想如何收尾了。在过去的作品当中，对于这个问题我总感到结尾十分困难，虽然结尾意味着我终于写完了，但没有好的结尾却会让文章变得毫无生机。我是否该直接说再见呢，不，我得感谢王黎明编辑，如果没有她，我就没有这个机会，告诉你，我早想告诉你们的话。祝好运！
