（三）

　　六月，正是热闹的季节，清晨幽香的气息另我几乎要醉了。美极了的朝霞好似在燃烧。心情格外好。但我知道这份心情是来自与他的存在。背上书包走出家门，脑海里全是鸿。我突发奇想一定要给他打个传呼。鸿很快回了电话。这天早上我们都没去上课，我们相约9：00在网上见。双手飞快地在键盘上穿梭，他仿佛就端坐在我面前，让我禁锢不住情怀飞得很远……。那天我们说了好多，我们用一种特殊的方式来宣泄心声。压抑在彼此心里的话一行一行闪现在屏幕上。顷刻无语，突然间，屏幕闪动，一句话跳到我的眼前"我想我是真的爱上你了！"简直不敢相信自己的眼睛。我呆呆的盯着屏幕。感觉空气在不断的升华凝结。我该怎么办？我爱他吗？说不清这是什么样的感觉，这是爱吗？其实我们都在痛苦而执着的系着这份感觉，却不敢承认这就是爱了！我们一次次游戈在爱的边缘。我们好迷茫，好胆怯，不知是对还是错。不敢去想，因为它缘于网；不敢表露，因为谁也无法预知结果；不敢迎接，因为怕它是一种涩涩的酸楚，是一杯冰冷的水；不敢面对，因为选择逃避比选择面队更容易。在真情面前我们都难以逃避。而现实又使我们难以从容，那种不曾预料的美好感觉 实在让我欲罢不能。于是，我选择了沉默。悄悄地把那一天藏在了心里：6月13日。
