（五）

　　我们的恋爱是与众不同的，我们在见不着的两个世界里进行着。很久以来我们一直幻想着能在一起的日子：牵手在淅淅沥沥的雨中；一起去看属于我们的星星……这样的日子哪怕只有一天，我都足够！我们一直不敢轻言放弃，我们的心在遥远的空间为了相见的那一眸而靠近，而等待。我们相约第二个约定就在火车站，还说好那天我会背一个漂亮的白色背包在站台上等他来接我。

　　这一天终于到了，这年国庆，我义无返顾的踏上了北上的列车。在列车出发的那一刻我才发现原来生命里的东西是如此难以抗拒，原来自己爱得如此执着。平生第一次有这么大的胆量和勇气。我背着渴望和期盼；激动与不安，在一天一夜的行程中我没能留心延途的山色，我不知道自己是睡是醒，是梦是想。脑海中不停的在放映鸿的模样，放映下车的那一刻。虽然我一直认为缘早缘迟，邂逅是必然的，爱也是冥冥中注定的，但自己还是慌了手脚。

　　列车晚点八分钟到站。下车前再照照镜子，披肩长发配一顶白色淑女帽，清纯可爱。绝对没问题。我对他会不会来不存在一点怀疑。远处，一个男孩，他的胸前怀抱着一大束玫瑰花，好象是一团火。他就是鸿，我敢肯定。只有他总会在不经意间留给我一份惊喜。他正急着向车厢里张望。六个月的朝思慕想啊！现在我和他就近在咫尺，可是我却没有叫他，我躲开了他的视线。鸿，他真的不是我心目中的白马王子。我虽称不上是漂亮佳人，但我心目中的他应该是高大的阳光男孩。但眼前的鸿他不高而且很瘦，带着一副眼镜。我在慢慢的失望。这时他从我身边走过，募地，我心一颤。是啊！就是眼前这个男孩，曾多少个夜里因思念他而彻夜难眠，为他落泪。曾深信鸿就是解开我的发辫，为我披上嫁衣的人。回忆我们的一幕幕，我始终被幸福包围着，他的爱已深入我的骨髓，无时不在。我爱的是他的人，不是他的外表。我……，我到底在干什么？就在鸿要从我视线中消失的片刻我喊出了他的名字。我不后悔我的决定。鸿转身，他接触到了我的眼睛，好灼人！我们相视了许久，他的表情我已找不出词来形容。只记得他的第一句话："送给你，特意为你选的。"还是那个声音清晰，阳光，还有那种磁性，那么熟悉。

　　"谢谢，好漂亮！"

　　"跟我还说谢谢吗？"

　　 我们相视而笑。我心里就像阳光一样那么暖。那时我认识到了我的爱没有变，依旧蔓延。其实幸福有时离你很远，有是却很近，它就徘徊在一念之间。

　　我们打车回到住处，这是他的学校。是一所不大但很有气氛的大学。在把那束玫瑰精心安置好后我们跑上了宿舍的天台。远方的夕阳已落到了地平线上，好美！夜提前来了。夜色第一次变得这样柔和。他温暖的手令我第一次有了温馨，踏实的感觉。一切都是梦幻般的，甜蜜的，心跳的。世界要轻轻的消失了，只剩下我和鸿还有两颗跳动的心。此刻我们凝眸而视，他用埋藏已久的情感拥我入怀，好久，好紧。

　　"终于等到这一天了，我们等了好久！希望这不是做梦！"他呢喃于我的耳际。

　　天际中几颗星星荧荧点点。"多美的夜啊！有星星。"我顺势坐在他的腿上，一双眼睛闪烁着。他抬头望望自语："我们的幻想实现了，这是属于我们的星星！"说完他笑了，我也笑了。这是我第一次亲眼看到他笑，带着一种男人的倔强和满足。我把头埋进他的怀里，他抚弄着我的长发，轻吻我的唇，我羞涩的脸...不要打扰我们，我们在一个真实的梦中不愿醒来。

　　和他在一起的时间里他把最深的爱都赋予了我，六天当中我被他的爱一层层包围着。他总是说要好好把握我在他身边的每一天。我无数次被他感动，被爱感动。我终于明白，去爱一颗爱你的心吧！我爱身边这个真实的鸿，愿意陪他携手人生。我衷心希望能抓住那双可以安抚我，激励我的温情的手，和他风雨同路，不管以后的道路是怎样的泥泞，我也不会皱眉，不会回头。用一生......
