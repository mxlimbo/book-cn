（七）

　　我们有缘，只是我们的爱情像源头水尾，有太多的峰回路转。我一直在固执的等待一个人，因为我心中有爱，这爱，早已许诺给他；这爱，只愿从他手中接过；也唯有这爱，也许生来就是留给他一个人的。别忘了，我们有第三个约定。只是我心中期盼，从掌到心的这段路短一点，再短一点......

　　虽然我不曾带走我的红玫瑰，但那飘零的花瓣早已落满了我的心怀。阳光中，它在憧憬，它在等待。

　　我甩甩头，又给自己加了一份信心。然后，把这片花瓣夹在了自己的日记本中。

2000年12月6日完

　　我们的故事还在继续，一天天我们用爱在生活，为爱而奋斗。我们要将爱情进行到底！双手合实，为爱祈祷！
