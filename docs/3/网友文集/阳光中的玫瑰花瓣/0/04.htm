（四）

　　其实爱本身就难以驾驭，除非自己能看得清。爱不可求，但能求。爱确实能求，又怕你不敢求。

　　真是出乎我的意料，在我生日那天，意外的收到了一个包裹，是鸿送给我的礼物。我的心高兴得几乎要跳出来了。鸿就是鸿，他总能给我一份惊喜，这就是他不同于别人的地方。我无论如何也猜不到他送给我的礼物如此特别：是一盘磁带。

　　封面写着："送给我心爱的女孩，虽然只是短短60分钟的录音，可却是我要对你表白的一颗心。从6月1日起到6月18日止整整18天录完，无论你喜欢不喜欢，我只想让你把它保存，如果想我的时候，可以拿出来听听。"

鸿 19日

　　我一口气听完了整盘磁带，眼睛已完全被眼泪所占拒。这时，电话铃响了，我光着脚丫跑过去抓起话筒，没错，是他。那天我第一次在电话里哭了。那一刻我终于明白了那份缠绕了我很久而说不清得感觉原来就是爱！我无药可救的被爱俘虏了。鸿无语，直到放电话时，他轻轻地说："宝贝，生日快乐！一定会有一天我会让你感觉到我的温暖，我的爱！"从那天开始，就注定我要和无奈与等待做伴。没有真实的他在我身边，也许我什么也得不到。有的只是思念的煎熬。可我真的没有理由骗自己。我会如约的守侯，等着他来轻轻的叩门，为了爱，也为了我们的承诺。

　　时间是积累的痛苦，而我就坐在痛苦的山尖上，即使爱情对我一点也不仁慈，我也不会放弃期待。既然选择了爱他也就是选择了距离。只要不在别人的欢笑里流着自己伤心的泪，不在别人的快乐里看见自己揪心的寂寞……

　　也许正是为此，在以后的日日夜夜中我们在各自的心里细细蜜密的编织着梦境。将思念化为星星和动力去照亮彼此的天空。
