（二）

　　真没想到，爱情居然可以这样开始。认识他是奇缘，靠的只是一丝网线。这一切不是在每个人的生命中都可以经历。虽连自己都无法接受，但我还是陷入了，认真了；的确爱了，也的确付出了。

　　大2，每天下了自习后，我和好朋友总爱蹬着脚踏车去我们想去的地方。林阴路上载满了我们的欢笑。平静的生活使我们总幻想着一份浪漫与激情。那是在花一般的三月，我们带着满心的好奇第一次走进了网吧，走进了网的世界。从那天起我和网络成了好朋友。每每闲暇时就会坐在点脑前随心点击，双击鼠标，再双击。而缘份就像一程不用买票的车行，如此神奇，在未知的时刻，未知的地点以出奇不意的方式不期而遇了。鸿就这样出现了。

　　"HI，你好！你的名字很美，交个朋友好吗？我是学美术的。上大一。"天呀，好直率的男孩，自然，随心。莫名的好感由然而生。

　　"是吗？那好啊！"我很乐意的接受了。我们聊了很久，像老朋似的无所不谈。言谈中仿佛感觉我们似曾相识，一点也不拘束。不用说，从那天起他走进了我的世界，而以后的日子开始变得好精彩！那一天我们有了第一次相约，在下午5：30我会给他打传呼，一言为定！现在回想，如果我们没有那个约定，那么我和他也许将会形同陌路，以后的事情也就不会发生了。

　　那天是4月10日。我的网名叫小雪。

　　当他的声音第一次通过电话传到我耳朵的那一刻，我预感到这个男孩会走进我的生活。颇具磁性的声音，清晰，阳光还有那浓郁的东北口音。既熟悉又陌生。冥冥中我的心就像不安分的季节一样有一种莫名的悸动。我们都在无意中走进对方。

　　"告诉我地址好吗？我给你写信。"

　　"好吧！我也会给你写的！""是吗？一定？"

　　"一定！"我也不知道为什么就这样答应了。也许是老天在安排吧！

　　三天后，我收到了一封漂亮的信，信封像海一样蓝。是他的信。他的文笔好极了。让人觉得一种温馨的感觉在悄悄四溢。鸿是那种很开朗，自然的男孩，而且还喜欢浪漫。每当我握着他的信时，就会干到无比幸福。他给我的总是一个绚丽的梦，朦胧而醉人。说实话，他的这种性格我好喜欢。对于他我已有了一种深深的依恋。也许是因为我们缘于网络，而网络带给我们的又不仅是真诚，还有一丝神秘。所以我们不断的发E-MAIL，写信，偶尔也会通电话。默默中有了一份然于心的牵挂，每天欲枕着他的名字入眠。在有他的日子里，觉得生命不再孤单，生活有了期待。我在极力克制自己的情感，释放自己的理智。但都不管用了。我知道我要陷进去了……。时间一天天飞转，我们都在怯生生的编织着这美丽的邂逅。
