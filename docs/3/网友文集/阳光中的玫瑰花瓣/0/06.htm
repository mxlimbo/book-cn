（六）

　　时间在我和鸿身上总那么吝啬。我要走了，临走那天，外面起了很大的雾。我收拾好行李，最后环视一下四周，窗前那束火红的玫瑰依然盛开着。转身的片刻，鸿拉着我的手说："你要等我，四年大学毕业后我去接你回我们的家，要有信心，相信我！我们一言为定！"这是我们的第三个约定，透过他载满真诚的双眸，我知道这个约定最美最美，也最长最长。

　　鸿送我上了火车，我们默默无语，人群中我才发现和他能在一起是多么难得的际遇，我才体会到了什么叫离别。爱一个人不能和他相守是意见多么痛苦的事啊！我看得出鸿的无奈，他拉着我的手久久不肯放开。只剩下五分钟车就要开了。鸿最后一次吻了我的脸，这一次是含着泪。当他红着眼睛站在车窗外时，我的心抽搐般的疼，他把手放在车窗上，我们掌心相对，我知道此时已无需再说什么，任何语言都是那么苍白无力。空间的裂痕在蔓延，离别的现实吞噬了一切。

　　一声汽笛鸣响，车开动了，缓缓的。我就要走了吗？不，我的欢笑还这儿，初恋还在这儿，还有那一束火一样的玫瑰也在这儿。我要把心留在这儿，如果上帝还关爱到我，我要再次踏上这片土地。突然，鸿好象想起了什么，他飞快地从口袋中掏出一支烟轻吻了一下，车已加速了，我紧紧的握在了手中，就像握住了他的心。

　　远了，更远了。再见了，鸿。再见又须何时才见？车轮碾过我的视线，他的身影已远，但没有消失，他一直像一盏阿拉伯神灯一样站在远方，站在我的生命中。耳边响起了这首歌：让我再一次握你的手，让我再一次亲吻你的脸，顺着我的脸滑下的是我的泪，在我胸中跳动的是他的心...。而此时，我已无法再触摸到他。我们彼此在离别中沉默，在沉默中孤独，在孤独中泪水涟涟。我的脸发烫，心乱成一团...
