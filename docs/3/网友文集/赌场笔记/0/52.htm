第二十一章

　　中午吃饭的时候，我本来想将刘萍讲的故事原本原样地说给魏老八听，但仔细掂量了一下，还是算了。我怕挫伤了他的自尊心，因为"义气"二字对于他来说，重于金钱，甚至重于生命，我怕他根本受不了这种打击。眼前就有"一指禅"这条毒蛇，这已经严重地摧毁了他的信心，让他猝不及防地不断修正自己的航标，如果再加上"花脸"，我估计他要哀鸣了，他过去的一切生活理念都会摧枯拉朽般地坍塌了。如果走了大半辈子的人突然有一天发现，他过去的生活全是失败，全是欺骗，这绝对会加速他的死亡，让他快马加鞭地走向生命的终点。阳光和希望存在于生命之中，生活才会变得美好。这使我想起了一个小故事，发生在八零年代，说有一个假老练，走到一家电影院前，抬头看见一幅巨大的电影广告，那是一部即将上映的法国新片，他旁若无人地大声念道：《总统铁事》。偏偏遇到旁边一个专门爱挑剔别人错别字的人，他走上前来纠正道："同志，那个字不念'铁'，而是念'轶'。"假老练哪里会服气，两个人在街上大声吵嚷起来。假老练提议道："我们打个赌，只赌一块钱，找一个有学问的人问一下，他说谁输谁就输。"恰巧对面走过来一个白发苍苍的老者，一看就是教授级的人物。两个人忙上前问个究竟，老者对那个爱挑剔别人错别字的人说："你读错了，赔别人一块钱。"假老练拿着一块钱高高兴兴地走了，那人还脸红脖子粗地准备与老者争辩，这时老者语重心长地对那人说："给他一块钱，让他糊涂一辈子！"

　　我准备给魏老八一块钱，让他也糊涂一辈子。因为人生难得糊涂！

　　深夜十点四十五分，我和魏老八来到了那家赌场所在的酒店的底搂咖啡厅。那是我们和"福建帮"约好的谈判地点。魏老八本来是要独身闯龙潭的，他对我说："一个人去和一百个人去没有任何区别，因为要单凭我们要与他们抗衡是根本不可能的，再说身上不可能允许你佩带武器，所以说没有任何必要去很多人冒这个危险。我一个人足够了，雷鸣打了我一枪，我必须把这一枪找回来。"

　　我对魏老八说："两个人总比一个人强，再说刘琳是因为我才被雷鸣打死的。如果不是她推了我一把，挨枪子儿得很可能就是我，所以说我必须去！！！"

　　魏老八拗不过我，只好答应了。哪想到按平了我这边，刘萍那边却闹麻了，她眼泪婆娑地非要跟着我和魏老八一起去。魏老八这次没有心软，坚决不答应。他表情严峻地劝说道："你妹妹就是这种倔脾气，高矮扭到费，结果把命都搭进去了。我早就说过，这不是玩逮猫儿，不是打耍耍仗，更不是游山玩水，图个新鲜，这是要流血的，弄不好把这条老命都要挫脱。如果你要替你妹妹报仇，有的是机会和时间，今天弄不好连雷鸣的鬼影子都看不到一个，报个卵仇？"

　　几句话说得刘萍蔫了下去，刚才的气鼓气胀、杏眼圆睁的样子顿时被一个可怜兮兮的、梨花落泪的形象所替代。我们最后决定：我和魏老八去见"福建帮"老大，疯子和刘萍在家原地待命。

　　咖啡厅里没有几个客人，安静极了，安静得有点神秘，甚至有点恐怖。我明显地感到了一股冰冷彻骨的阴气迎面扑来，我不由自主地打了一个寒战。这间咖啡厅就像一座尘封千年的古墓，幽暗而肃穆。那张长长的深色的吧台仿佛像一口硕大的涂满香料的灵棺，吧台上方昏暗的射灯像古墓中的长明灯一样，忽闪忽闪的磷火似的泛着贼光，而站在吧台里面的几个服务生，痴呆似的僵立在那里，像在墓中站立多年的殉葬品。

　　这间咖啡厅的气氛太让人压抑恐惧了，也许只有我们才有这样的感觉，没准别人还认为这是个谈情说爱的好地方嘞！不过这并没有什么大惊小怪的，有的人就是喜欢到坟地里抒情，缅怀古人，展望未来。我以前的一个朋友就有这方面的癖好，今天西安兵马俑，明日武侯祠刘备墓，他喜欢的旅游地点都是古人的墓穴。我怀疑他爷爷以前可能是个盗墓贼，不然的话他为什么有这方面的基因？最后，他与相恋多年的女人——一个四十多岁的风韵犹存的寡妇——双双自杀在北京十三陵一座距地面九层楼深的古墓里，算是了却了他一生的夙愿，因为他崇拜古代帝王的灵殿……

　　两个小伙子把我们带到了靠近角落的一个包间，进门前按规矩把我们全身搜了个遍，其中一个用很生硬的普通话说："你们先在包间里等一会儿。"说完就掩上门，悄无声息地走了。

　　魏老八点上一根儿烟，很悠闲地吸了一口，然后缓缓地、漫不经心地向空中吐着烟圈儿，一副胸有成竹的样子。我问魏老八："你认为今天的谈判会有什么结果？"

　　魏老八说："结果可能会出人意料。"

　　"为啥子？"我迫不及待地问。

　　"我估计雷鸣还是用的在劳改队里学的那一套：欺上瞒下。那个时候他在队里骄横跋扈，今天洗涮这个人的脑壳，明天修理那个人的脑筋，底下的人敢怒而不敢言，都认为他的背后肯定有人扎起。连我都一时半会儿地被他蒙在鼓里，但狭细想来他的行为又是那么地不合情理。比如：他要求每个新犯必须向他上贡，家庭富裕时常有汇款的，必须分一半给他，没有汇款的就强占人家的新皮鞋新毛衣，如果不答应，他就夹人家的'毛驹'，给人家小鞋穿，无中生有地说人家有严重的逃跑思想，并威胁说，他要向政府干部汇报。其实这些行为都是政府干部们严令禁止的，脑壳再昏，也不可能允许雷鸣这样胡作非为。只可惜一般的犯人根本没有勇气去揭发检举他，他一天到黑有事没事就站在干警面前汇报自己的改造动态，而且说到放松处，他会及时地仰天大笑，像个演员精心演绎着一部精品电视剧，他的演技给人造成的假象是，他与干警们的关系是多么地亲密。雷鸣是有文化有头脑的，他极为成功地利用了'狐假虎威'这句成语。

　　"我当时越想越觉得不对劲，也越来越不服雷鸣这口气，我决定避开一般的工区干部，直接找中队的最高领导人——指导员去反映他的问题。因为一般的工区干部做不了主不说，还没准是和他穿一个连裆裤的，这曾经是有过先例的。结果不出我之所料，指导员一听，拍案大怒，当场命令武警仔细搜查了雷鸣。说出来有点吓人，从他的箱子里共搜出两千多元人民币，全是他虾子吃了人家新犯的汇款。还搜出当时在成都非常流行的'高尔夫'西服十件，'金兔'毛衣十六件，是他估到新犯上的贡。雷鸣当场被指导员贬到小煤窖搓炭花儿去了，半年都没有翻身。"

　　"你的意思是说，'福建帮'对雷鸣的行为也是蒙在鼓里的？他在利用'狐假虎威'，欺上瞒下？"

　　"据我分析，很可能是这样。所以我直接找他的顶头上司问个究竟。"

　　魏老八的分析有一定的道理，雷鸣毕竟是在劳改队里学的这一套，也是在那座染缸里成长起来的，他永远磨灭不了在那里烙下的印记，为了生存，他很可能将他的"劳改经"——"狐假虎威"搬出来。

　　"你的分析有什么具体的根据吗？"我问魏老八。
