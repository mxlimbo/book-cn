第十八章

　　我万万没有想到张楚的这句歌词变成了刘琳生命的绝唱，她那发音不准的歌声曾遭到我无数次的讽刺打击，我甚至曾让她彻底断了唱歌的念头，因为痛苦的歌声足以把人逼得神经错乱。我对她说："唱歌唱左了，完全是耳朵的问题，你首先听拐了，所以才会唱拐。你每天在我面前晃来晃去的，连我都受了一些影响，我现在已经发现我都有点拐了。"　　　　　　　　　　　　　　

　　她不以为然地笑了笑，继续旁若无人地高歌，害得我屁滚尿流，狼狈逃窜。如今这些美好的与不太美好的所有回忆，都在那无情的枪声中烟消云散了。

　　刘琳是死在我的怀里的，看到她逐渐变硬变凉的身体，我欲哭无泪，因为我根本无法接受这个残酷的现实。昨天夜里我们还缱绻在春情荡漾的温柔乡里，我们几乎说了一夜的情话，絮絮叨叨地重复着激情的话语，在她一遍又一遍的泪水中，我的手演变了各种方式的柔情蜜意，在她一次又一次喜极的尖叫声中，我发现了一条通往爱情的捷径。因为爱，她的皮肤开始柔软，她的指甲开始尖利，她的牙齿叼住了我的肩膀，像一头负隅顽抗的母兽拼命撕咬着征服了它的天敌，我的背部开始流血……

　　而现在躺在我怀里的刘琳已经悄然地睡去了。没有了唱左的歌声，没有了缠着我学"千术"时的娇语，我甚至想轻轻地唤醒她，让她重新活灵活现地站在我面前，我一定用我一生的精力去爱她，去呵护她，去宠溺她。

　　把她推进火化炉的那一瞬间，我禁不住留下了眼泪。这也是我这辈子第一次为一个女人流泪，所以我格外地珍惜。我会把她永远地镶进我的记忆，在我心中寂寞的角落里，我会永远永远为她留下一个空格，让她安详地睡在那里，让她静静地聆听我为她搏动的声音，但愿这个年轻的声音变成一首安眠曲，永远伴随着她，让她安息。

　　我把那张《姐姐》的CD碟装进了她的口袋，愿它变成一只美丽的火凤凰，陪伴着刘琳一同在熊熊的烈火中涅超生。那张CD碟是我和刘琳一同来珠海买的，它也变成了我和她唯一的纪念物。我没有来得及和她合张影，没有来得及留下任何可以回味的痕迹，但她的音容笑貌已埋进了我的脑海，留下了永不磨灭的印记，就让她在我的心中与我合张影吧！

　　愿我们互相永远不要忘记！！！

　　魏老八只受了点轻伤，无关大碍，几天以后就出院了。这种枪战场面他大概司空见惯了，根本没有受到什么惊吓，只是为刘琳的意外罹难而长呼短吁，嗟叹不已。他睁着充满血丝的眼睛咬牙切齿地说："雷鸣那个杉木烂贼，不报此仇，我们就枉自变成了一个男人。"

　　报仇，已经变成了我心中的痛点。以前我臆想的种种打败雷鸣的恶毒念头，与这次划破天穹的枪声比起来，真是小巫见大巫了。与雷鸣相比，我们实在是太善良了，我们就像一个初出茅庐的半大小子，最狠毒的念头也就是恶作剧似的往对方的脖子里塞几只蚂蚁，我们不想致对方以死地，因为我们还没来得及产生杀人的念头。

　　魏老八告诉我，雷鸣的心狠手辣他是早就领教过的，他一点都没有感到奇怪，如果这次他一点动静都没有，你反而要多加小心了，因为他有你根本意想不到的狼子野心。

　　魏老八说："有一次我们中队逃跑了一个犯人，中队干警让我们这两个最红的'红毛犯人'协助干部抓逃。我至今还记得那是一个风雨交加的深夜，格老子因为走得太急，我身上只穿了一件薄薄的囚服，走在稀洼洼的山路上，冷得我全身打抖。天黑惨了，几乎伸手不见五指，不准打手电，不准吸烟，不准说话，全凭平时的记忆，一点一点地在陡峭的山壁上摸索着前进。就这样，我和雷鸣被我们的管教带着，花了三个小时的时间，才走到了埋伏地点，那是一个凄凉恐怖的垭口，是翻越邛崃山到成都的唯一一条小路。一边是原始森林，一边是万丈深渊，我们就在垭口的一个烂窝棚里驻扎了下来，守株待兔似的看那个逃跑犯会不会从这里偷偷翻越邛崃山。像我们这样的抓逃小组起码有十几个，分布在各个关口要塞，堵住了任何一个逃跑路线，加上狱政科出动的追捕小分队，可以说是布下了天罗地网，任何人想要轻易逃出去，恐怕是天方夜谭。

　　"管教带我们出来抓逃的原因有两个：第一，逃跑的是一个新犯，连管教都没来得及熟悉，而我们却几乎天天和他在一起，就算他化了装，他的长相，他的声音，很难逃过我们的眼睛和耳朵；第二，这是管教给你的一次立功赎罪的最好机会，如果你亲手抓住了那个逃犯，你肯定立功减刑，要知道全中队几百号人，独独选中你去抓逃，这是管教对你的最大的信任与奖励。

　　"我们的管教是个和蔼可亲的人，平时对我们可好了，连重话都没有说过一句，尤其是对雷鸣，真可谓是仁至义尽。因为他觉得雷鸣是个很聪明的人，又会唱歌，又会作曲，还玩得一手好乐器，像这样的人在监狱里浪费青春真是太可惜了。

　　"那天夜里，没有任何动静，那个逃跑犯大概去翻越二郎山了，我们完全扑错了方向。失望之余，我疲惫得靠在稻草上几乎睡着了，但成群成堆的蚊子围着我的脑壳嗡嗡叫着，咬得我满头血包，抠都抠不赢。但奇怪的是，蚊子却不去咬雷鸣，他安安静静地靠在那里，闭目养神，悠闲得不得了，这真是叫我羡慕惨了。现在想来，只有一种解释，他的血里肯定有毒。

　　"要到黎明的时候，老管教要出去小解，这本是个很普通的事情，但他上了点岁数，加上路途劳累，又一夜没合眼，眼一花、腿一软，一头栽进了一个不大不小的硝坑里。我迷迷糊糊听见外边一声'哎哟'，赶忙提了一根青冈木棒冲了出去，我以为那个逃跑的憋极了想冲关，可出去一看，鬼影子都没有一个。我顺着'哎哟哎哟'的呻吟声找了过去，才知道老管教卡在了硝坑的中间，情况非常的危急，如果掉进那个深不见底的硝坑，老管教就没命了。我急得全身一下子都被冷汗打湿了，连忙叫雷鸣出来帮我把老管教从坑口拉出来。雷鸣出来倒是出来了，可他站在那里一动也不动，我一下毛了，大声骂道：'你狗日的快点儿，老管教缩下去了，找都找不到了。快点把皮带解下来，我们接在一起，看够得到他不？'他不但不解皮带，还一下子把我拉到一边，悄悄说：'干脆丢他妈一坨大的广儿石，给他仗进去，把他干掉算了。翻过这座山，就是邛崃县，赶一趟班车，哈哈儿就拢成都，难道你不向往自由吗？'听了他这番话，我都不晓得咋个形容我当时的心情了，反正有点毛骨悚然。这个老管教可是像慈父一样的对待他呀，他竟然用这种方式报答人家的恩情，他竟然起了这种歹毒之心。老管教终于被我拉上来了，他的伤还真不轻，盆骨骨折，右手臂骨裂，满脸的鲜血、泥土、青苔，我把老管教一步一步地背回了中队。从此我和雷鸣的隔阂更大了，我从没向那个老管教提起过这件事，害怕他伤心，我只是提醒他要小心雷鸣这个人，这反而招致了老管教对我的一番教育，他语重心长地对我说：'要注意团结啊！'从此我不再在老管教面前提雷鸣这两个字，他自己的路他自己走！"

　　听完了这个故事，我只有一个念头：让雷鸣这个人活在世上，就是犯罪！！
