第十二章

　　芦山县实在太小了，两条几百米长的街道成"十字型"交叉，这就是整个县城。我们一队人马像无头苍蝇一样在街上逛上几圈，没有发现任何蛛丝马迹。

　　已经是傍晚时分，还没有任何动静。按理说，格桑的目的并不是抓住刘萍，而是我和魏老八，我们才是那场赌局的罪魁祸首。他已经知道了我的手机号码，他应该主动与我们联系。我攥着手机的手已经冒汗，仿佛那块黑黑的塑料和刘萍的性命紧紧地相连着。

　　18点34分，手机终于响了，是格桑的声音："喂!杜老板，你的大理石生意好吗?"

　　在这种时候他还没忘了揶揄我几句，他已经杀了花脸，还绑架了刘萍，我们之间已经没有任何调和的余地。我强压住怒火说："格桑，这件事本来就与她无关，你不要把我俩的恩怨强加在一个女人身上。如果你是个有脾气的男人，你就直接找我!"

　　"我就是在找你呀!而且我找得很辛苦。"

　　"你在什么地方?"

　　"我在宝兴，你到了以后直接打我的手机，电话号码在你的机子上。"格桑说完就挂上了电话。

　　格桑像是一条狡猾的狐狸，他知道我们绝对不会置刘萍的性命而不顾的，所以他坦然悠荡地跟我们玩起了捉迷藏。他知道离雅安越近，他就越没有势力。

　　魏老八显然已经不耐烦了，他向三十几个弟兄下着命令："到了宝兴，只要一看见那个老，少废话，乱枪射死!"

　　宝兴县位于芦山的北部，距离芦山大约40公里。我们浩浩荡荡地上路了。

　　天已经黑透了，车灯早已打开，前面的路越来越狭窄。刚刚驶离芦山10多公里，一座黑黝黝的高山迎而扑来，山石像煤炭一样泛着黑光，阴森恐怖。由于道路不熟，疯子已放慢了车速。我打开车窗朝外一看，顿时倒吸了一口冷气，道路的右边是幽深的峡谷，从谷底传来轰隆隆的湍急的水声；左边却是笔直的山崖，陡峭峻险，像鬼斧劈过似的。

　　一种无名的恐惧笼罩着车内，前方就像一个无底的黑洞随时准备吞噬我们。

　　魏老八压低声音说："疯子把车开慢点。武器全都准备好，以防不测。大家眼睛放亮点，不要中了他的埋伏。"

　　我的心似乎都停止了跳动，握枪的手也渗出了汗水。我的眼睛一眨不眨地盯着前方，不一会儿，我的眼球开始酸痛了。一只孤独的乌鸦在我们的头顶飞翔着，"呱呱"地叫个不停。虽然它是一种不祥的鸟，但以前我并不怎么讨厌它，但是此时此刻，我真的开始对它厌恶了。

　　为了缓解车内紧张的气氛，我问魏老八："你讨厌乌鸦吗?"

　　"不讨厌。"魏老八答道。

　　"为什么?"

　　"在劳改队时，我拿它打过牙祭。"

　　"乌鸦也能吃?"我吃惊地问。

　　"怎么不能吃?跟鹌鹑的味道差不多，鲜惨了。"魏老八咂着嘴唇说道，仿佛他还在晕那个让他永世难忘的乌鸦味道。

　　一听魏老八说起鹌鹑，我们立即想到了那炸的又焦又黄又酥的小鸟，全车人开始往肚里吞口水。这时我们才想起，晚饭还没吃，肚皮早就饿扁了。

　　我们继续在崎岖的山路上匍行，一会儿爬到了山顶，一会儿又跌入了山谷。晚上10点，我们才驶到离宝兴还有8公里的灵关镇。

　　终于看到了灯光，终于看到了房舍，终于看到了人烟。我们仿佛从地狱中刚刚爬出来一样，重新又回到了人间。

　　魏老八说："我们先找个馆子，炒它几斤回锅肉，杀它几只鸡，再整几条花鲢，把肚皮喂饱了再说。"

　　8辆小汽车，三十多号人一起涌进镇子，竟引得全镇的老少男女围观，他们把我们当成了稀有动物。

　　找了一家宽敞点的饭馆，我们一哄而进，像饿红了眼的土匪，东翻西操的，惊得老板张大嘴巴，半晌说不出话来。

　　魏老八说："老板，赶快泡茶来!"

　　"嗷，"老板这才回过神来，一头栽进屋里，叫醒睡梦中的老婆、儿女，全家齐动员，大打翻身仗。泡茶的泡茶、杀鸡的杀鸡、宰鱼的宰鱼，热闹非凡。

　　魏老八对我说："马上给格桑打电话，说我们已到了灵关，问他在什么地方?"

　　我拨了格桑的手机，但他的电话处于关机状态。

　　魏老八说："?不张我们嗦?"

　　我又拨了一遍，还是关着机，看样子他把我们调到灵关就不甩视我们了。

　　热腾腾的饭菜一样一样地陆续端了上来，管它味道好不好，先把肚皮盔圆了再说。其实无论是回锅肉还是鸡鱼，都是打死了卖盐的――咸死了。魏老八边吃边说："有条件就讲究，没条件就将就，反正我这个肚皮是个渣子凼凼，啥子都装得。"

　　饭馆老板是个40多岁的高大汉子，一脸的络腮胡，笑起来眼睛眯成一条线，非常可爱慈祥。他见我们狼吞虎咽的样子，笑着说："科学家就是辛苦，这么晚了还没干夜饭。"

　　一听这句话，三十几人全部停止嘴巴的咀嚼，表情僵木地望着老板。

　　老板继续说："我的眼睛还是比较雪亮的，你们瞒不过我，我知道你们是啥子人。"

　　魏老八饶有兴趣地问道："那你说我们是啥子人?"

　　"你们是到宝兴大熊猫保护区研究大熊猫的科学家，而你……"老板一指魏老八，"是他们的领导。"

　　我"扑"地喷出满口的米饭，魏老八也被逗乐了。

　　老板显然知道他猜错了，他仍不服气地说："允许我再猜一遍。"

　　魏老八说："你猜对了，我给你200块钱。"

　　老板两眼顿时放出惊喜的光芒，说："你们是山上的公安，是抓逃跑的霉和尚的。"

　　"什么霉和尚?"我问魏老八。

　　魏老八小声对我说："这里的人管犯人叫霉和尚。"

　　老板还想猜下去，魏老八忙打断他："不要乱猜了，我们是地质勘探队的，你赶快到镇上的旅馆打个招呼，说我们有三十多人今晚要住这里。如果床位不够，就多找一家。"

　　老板笑颠颠地跑了。

　　我又给格桑打电话，还是关机。魏老八说："看样子，今晚他是不会开机了。再说，随便他干啥，我们就驻扎在灵关不走了，哪个跟他打游击战，要钱还是要命，干脆点!"

　　这一夜，我躺在冰冷的被子里，辗转反侧，根本无法入睡。刘萍缠着我讲笑话的样子一直浮现在我眼前，不知道她现在受着怎样的罪?也不知是死是活?"

　　清晨6点，我的手机就响了，我猛地翻起身，抓起了电话："喂!是格桑吗?"

　　"你们这么大批人马驻扎在灵关，想把我吃了?"电话那头格桑恨恨地说道。

　　"格桑，我不想捉迷藏，你现在在哪里?这件事情你想怎么解决?"我毫不示弱地问道

　　"我现在在硗碛，你一个人坐车过来，不准带武器，否则后果自负，我要用刘萍换你。"

　　"好!我答应你!"我坚定地说，"不过，我要听刘萍说话。"

　　过了一会儿，电话那头传来刘萍虚弱的哭声："我是刘萍，快来救……"

　　刘萍还没说完，格桑就抢进来大声说："快点来!否则她就没命了!"电话挂断了。

　　"硗碛在什么地方?"我问魏老八。

　　"离这里起码60多公里，在夹金山脚下，就是红军翻过的那座雪山，是个藏民区。"

　　"看样子，只有我一个人去一趟。"我说。

　　魏老八说："太危险了，人生地不熟，又是单枪匹马。"

　　我说："那不可能不管刘萍吧?我们出的千，让一个女人扛着?"

　　魏老八想了想说："我估计他绑架你为人质，主要还是为了他输的50万元，并不真想要谁的命。这样，你对他说，钱我们已经带来了，要解决什么问题，必须到灵关来。"

　　我把身上的枪拿出来交给了魏老八，他从上衣口袋里拿出两叶剃须刀片，认真仔细地藏在我衬衣的袖口夹层里，说："这就是你惟一的武器。叫疯子开车送你，我在灵关等你的消息。"

　　上车前，我回头望了望魏老八他们，三十多人寂静无声地肃立在风中，男人之间的告别是不需要语言的。
