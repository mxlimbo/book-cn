第二十章

　　我没想到刘萍哭了整整一夜，从隔壁传来她断断续续的低沉的抽泣声，几乎要把我的精神彻底毁灭。我整夜睁着困涩的双眼，竖着狼犬一样的尖耳，在寂静的黑暗中极力捕捉着来自墙壁那边的哪怕一丁点儿声响。对我来说，刘萍的呜咽声是我这一生中所遇到的最大的折磨，因为它在每时每秒地提醒着我，我已永远地失去了刘琳。

　　第二天，刘萍的眼睛红肿得像两个熟透了的李子，见我走了进来，连忙转过脸去，假装收拾床铺叠被子。其实床上的被褥枕头整整齐齐的，昨晚她连床都没挨，硬是坐在沙发上屯了一夜。我叫她去吃早饭，她轻轻地摇了摇头，说："我现在什么都不想吃，只想安静地坐一会儿。"

　　我不想再去强求她，她大概还没有从失去妹妹的悲怆中彻底解脱出来。看到我要离开房间，她连忙问道："能不能陪我坐一会儿？"

　　我反身坐在了沙发上。她垂着头，轻轻地叹了一口气，问我："你很爱我妹妹吗？"

　　我就知道她要问我这个问题，我点了点头，说："是的，我很爱她，但遗憾的是，这个爱对于我们来说来得太晚太晚了。"

　　她睁着美丽而谨慎的大眼睛，久久地凝视着我，好像我撒了一个弥天大谎。这反倒使我惴惴不安起来，我甚至怀疑我是否真的爱过刘琳，怀疑我是否欺骗了自己。爱是一种冲动，它本身就不应该掺杂着过多的理智进去，也不应该受任何人为地干扰。我和刘琳同居一室，竟能奇迹般地相安无事，理智战胜了肉欲，现在想来真是匪夷所思，如果还有这种机会，我不会轻易放过她的。只可惜，这种机会对于我和刘琳来说只有一次。

　　久久地，我们在这令人窒息的沉默中度过了整整一个小时。我的思绪飞得很远很远，我想到了我与刘萍初识的那个夜晚，也想到了与刘琳永别的那天早晨，我想到了与她们姊妹一次比一次热烈的爱情，更想到了我那个秘密的爱情墓地，我不由自主地朝桌上的骨灰盒望去，那里长眠着我积聚了一生的激情。

　　也许这种沉默太令人苦涩不堪了，刘萍率先打破了寂静，她说："我妹妹本来是很乖的，只可惜生长在我们这种家庭。"

　　"她已经把你们家里的一切都告诉我了。"

　　"她告诉你她杀'花脸'的事了吗？"刘萍仰起头来问我。

　　"'花脸'？！"我吃惊地问道，"就是在雅安被格桑杀死的'花脸'？"

　　"是的，就是那个'花脸'。但他不是格桑杀死的，是刘琳干的。"

　　"到底是咋个回事？"我急切地问道。

　　"唉！"刘萍叹了口气，说："说来话长啊！我很早就认识了'花脸'，那个时候我和妹妹刚被我父亲从河南领了回来，父亲还是一如既往地赌、赌、赌，我们的生活根本没有着落，吃了上顿没下顿。我开始在一个砖厂打工，每个月赚个百八十块钱补贴家用。就在那个砖厂门口，我认识了'花脸'，他每天蹲在那里，靠砸'娃儿牌'维持生活，也叫砸'人人宝'，靠一颗色子，三个茶杯，经过他眼花缭乱地折腾过后，你猜中色子在哪个茶杯里，你就算赢钱了。不过这完全是一个骗局，'花脸'总能控制你的输赢，不然的话他每天蹲在哪儿干啥？还不如找个烂茶馆喝点烂茶。看到他每天都有几十块钱的进账，我当时真是羡慕死了，要知道他每天赚的钱等于我半个月的工资。我每天蹲在那里聚精会神地看他表演，想拜他为师，跟他学技术。有一天，'花脸'问我：'妹儿嘞，砸点钱赌赌运气噻！'我说：'我想跟你学这门手艺。'他听后笑了，说：'我还从来没有听说过一个女娃儿蹲在马路边上干这种事。这样吧，如果你真想拜我为师，我教你打假麻将吧！'就这样，'花脸'成了我的师傅，我跟他学了一整套打勾手麻将的技巧，我们两个长期流窜于雅安的各种茶馆，杀遍雅安无敌手。那个时候，女人会打麻将的很少，会这种当时认为最先进的千术的，更是找不出几人。哪象现在，打麻将瘾大巴嚓的多数是女人，而且多数的老麻将都晓得'洗盒子'、'砌墩子'、'龙摆尾'什么的，全部打醒了，而且越打越简单。以前还讲究断幺、无字、平、缺、将，现在斗拢就和牌，牌运大过牌技，越打越没有打头。慢慢地我们的收入越来越少，雅安本来就很小，识破我们这点伎俩的人越来越多，我们慢慢地失去了市场。后来他认识了格桑，专门给格桑跑腿，我们的来往基本上就中断了。有一天，他兴冲冲地跑来对我说：'我请来了M市赌王，专门收拾一下格桑。'我问他：'你不是整天和格桑在一起吗？'他说：'他虾子有点不落教，老子要整他的枉儿。'他让我给你们作掩护，反正格桑根本不认识我，'花脸'让我装成你们包得'二奶'，专门去胀格桑的眼睛，扰乱他的判断。我知道他把我当成奖品奖赏给你们了。以前他也打过我的主意，只因他自己太丑，有那个色心没那个色胆。开始我百般地不愿意，我成什么人了？我又不是'鸡'，但当时我正好没有抓拿，再说陪你们又不是陪睡觉，就勉强答应了，哪个晓得与你……后来你们用'倒挂金钟'赢了格桑以后，格桑就回到马尔康搬款去了，十天以后他来到雅安找你们报仇，哪里还有你们的踪影？他感到这场赌局后面一定隐藏着巨大的阴谋，他千方百计地找到了'花脸'，他许诺二十万，叫'花脸'说出这场赌局的真相。'花脸'哪里经得住这种考验，稀里糊涂地就把你们彻底出卖了。这一下可被格桑抓到了把柄，逼着'花脸'把你们引出来，不然就拿'花脸'试问，因为'花脸'也是那场赌局的主要成员呀！'花脸'开始死不答应，就算格桑把他五马分尸，他绝对是宁死不屈。格桑看硬的不行，就来了一招软的，格桑许诺，赢了你们以后，他俩二一添作五，如果现钱没有二十万，格桑负责补齐。其实开始我也不知道内幕，格桑绑架我时，我也以为是真的。其实这全是'花脸'的馊主意，他把我当成诱饵抛了出去，同时放出风去，说他被格桑杀死了，所有这些苦肉计只有一个目的，就是把你们引到雅安，连他的弟兄们都蒙在鼓里。他知道魏老八是非常讲义气的，他不可能坐视不管。他们找了一个千术高手，叫什么'一指禅'的，准备与你们决战。他们认为他们铁定赢了，而且还是非常体面地赢，因为格桑是个非常爱面子的人，他不想在江湖上给人家留下话柄，他还特意从雅安请来黑社会老大'潮虫'作赌证，为的就是能在江湖上扬名。哪想到结局却出乎他们的预料，你们不但赢了那个'一指禅'，还引得他们双方火并，损失极为惨重。

　　"格桑根本不是人，本来是假绑架，我只是个诱饵，但格桑也将我当作那场赌局的一员干将，他把输了五十万元的一切怨恨都发泄在了我的身上，他们折腾了我整整一夜……"

　　说到这里，刘萍的眼圈一下子潮湿了，她无助地摇着头，仿佛再次尝试着拂去那段不快的回忆，她喘了口气继续说道："后来我妹妹刘琳从'花脸'的一个弟兄那里知道了事情的真相，她恨不得扒了他的皮，抽了他的筋。有天夜里，'花脸'从一个歌厅嫖了婆娘出来，被我妹妹一直跟到了家门口，她使出全身的力气，一刀插进了'花脸'后心，只剩下半个刀柄……"

　　听完刘萍的叙述，我不禁毛骨悚然，真是江湖险恶，人心叵测啊！我对刘萍说："杀死你妹妹的就是'一指禅'的哥哥，他叫雷鸣。"
