第四章

　　哈尔滨人身高足有1.80米，敦敦犊犊的，手臂特别长，活像一头大猩猩。尤其他的脸非常恐怖，又长又扁，鼻孔像两个黑洞肆无忌惮地朝前敞开着，深不见底。两只又大又圆的眼睛朝前暴突，像镶嵌在眼眶外的两只硕大的玻璃球，给人的感觉是，只要你用手指拇轻轻一刨，眼珠就会滚落下来，你还得找个盘子赶忙接到。他有40岁的样子，眼角的鱼尾纹像菊花一样盛开着，他虽然穿着一件质地很不错的西服，但整体看上去有点农豁农豁的，如果再腰上拴一根草绳，绝对像一个东北农村食堂的老采购。

　　他从服务生的手中拿过一副崭新的扑克牌，剥开封口，将牌"哗"地抹成扇子形，用尖尖的嗓音说："请验牌！"

　　这种东北味十足、阴阳怪气的声音立刻让我联想到小品演员赵本山在"泻痢停"中的广告词："看疗效！"不过，在这个莽锉锉的野物嘴里冒出这种太监式的声音着实让我惊了一跳。

　　魏老八戴上特制的眼镜仔细查验了扑克牌的背面，在确定没有暗记后，剔出了大小王，以及所有梅、方、红、黑的10、J、Q、K。

　　哈尔滨人诡秘地笑了笑，开始洗牌。全屋的人都仿佛屏住了呼吸，所有的视线都集中在哈尔滨人纤长的手指上。他洗牌的动作可以用"飞快"两个字来形容，娴熟而规范，没有任何马脚。我知道有一种"假洗"，就是用一根小指拇尖插进牌里，卡住自己背的张子，不管用什么样的洗法，不管洗多少遍，始终跟没洗一样。不过，会这种"假洗"而不露任何破绽的，已经是"软牌"中的高手了。不排除哈尔滨人使用这种手法，但这种手法最怕切牌，你只要连切几次，他事先储存的"连张"全部打烂。

　　哈尔滨人表演了洗牌绝技后，将牌"叭"地按在桌中间，做了一个"请"的姿式，示意魏老八切牌。魏老八也不是吃素的，这种场合早已司空见惯了。他知道经过如此眼花缭乱的洗牌后，表面上的七八张牌早已被洗牌人暗记在心了，如果洗牌技术没练到这种境界，那还出来操啥子江湖？魏老八不会像平常人那样，端起一叠牌上下交换或者插入牌中，因为那几张"连牌"始终存在。他只拿起最上面的第1张与第4张分别切入牌中，最大限度地打乱洗牌人的记忆 ，因为第1张牌是记忆最深的，而第4张牌以后则是人的记忆"盲点"，很容易出现记忆紊乱。不过这只能是最大限度地破坏记忆，对于一个顶尖级高手来说，超强的记忆是最基本的素质。如果你的记忆力很差，你干脆不要参与赌博，你只能娱乐加点"小刺激"，浅尝辄止即可。

　　哈尔滨人似乎根本不在乎魏老八的切牌方法，他像用漏勺捞虾一样，抄起纸牌，在空中划了一道很美的弧线，然后像天女散花一样"刷刷刷"地单手用大拇指将牌"弹"出来，还是从最底下滑出来的。

　　魏老八摸出200元投石问路。哈尔滨人一怔，他没想到魏老八下这么小的注。

　　"气胸"在一旁讥笑道："我简直不相信自己的眼睛。"

　　魏老八厉声说道："你不要在那里扎扎哇哇的，该出手时我会出手的。"

　　端了点子后，魏老八该拿第三铺牌，展开一看：逗起334，剩下的6点和9点加起来是半砣（5点）。而哈尔滨人则是逗起145，剩下的两张牌加起来是8点，魏老八输。

　　魏老八将手中的牌往桌上一丢，直视着哈尔滨人，冷笑着问道："原来是江湖上流传很神的'弹花手'嗦？"

　　"弹花手"浑身一颤，一时语塞，僵在那里。"气胸"则气焰嚣张地插嘴道："你不管是啥子手，这里面是经过数学公式计算的，除非你在台面上抓到我。"

　　魏老八点燃一根烟，不慌不忙地揶揄道："你小学没毕业，还数学公式？我看你是狗撵摩托，不懂科学。我晓得这位老兄的'弹花手'功夫很到家，无论你怎么切牌，他都晓得哪铺牌是几点，而且，想要哪铺牌，他都可以通过端点子得到自己想要的牌。不过，我还是不想赌，你们去其他地方表演算了，M市这个地方，水是有点烫的哟！"

　　"虚了嗦？""气胸"继续挑衅着。

　　魏老八气的大骂道："你啷个咬到鸡巴横起奔（扯）呢？跟你说，你们这种'弹花手'早就过时了，你啷个不研究一下流行趋势呢？如果你要赌，百分之百地输。"

　　"气胸"嬉皮笑脸地说："那我咬到不奔（扯）嘛！"

　　既然"气胸"这样提劲打靶，只有我出马用"空手道"灭他们的威风了。"弹花手"再凶，你毕竟只有5张牌，而我用"空手道"偷牌、换牌，等于是用6张牌对付"弹花手"的5张牌，而且，你还猜不准我的点数，造成你下注失误。我抓起发给我的那5张牌，早已偷在袖口里的一张牌顺势滑出，我在6张牌中迅速判断出最佳组合，得出最大点数，然后在不经意地倒牌中将那张多余的牌换进袖口。纸牌在我熟练的手法中换进换出，不时擦出轻微的"沙沙"声，这声音太美妙了。而这种美妙的音乐只能由我和魏老八共享。"气胸"瞪大眼睛紧盯着我的手，想从中发现点什么，但他哪里知道，这种空手换牌速度只能用高速摄像机拍摄下来后才能看清楚。"弹花手"的眼睛恨不得吊在我的手腕上，因为他发觉我的牌都不是他事先知道的点数。开始他以为是他自己计算失误，但接二连三地输牌，他终于沉不住气了，马上清点牌的张数，一共36张，没有任何出入，他哪里知道在他决定数牌前，我袖口里的牌早缩进桌面的牌堆里去了。我不可能盘盘出千，真真假假才能迷惑对手。"弹花手"知道他终于遇到高手了，在他输了8000元后，将牌一丢，投降了。

　　走出茶坊时，已是深夜，清凉的空气迎面扑来。魏老八气咻咻地说："格老子数学公式？'气胸'那点水水我又不是不晓得。以前在劳改队时，人家问他喜欢看啥子电视，他说：'我最喜欢电视择制片亨特儿。'他把'译制片'说成'择制片'，天生一个假老练。今天又跟我吊儿啷当的，屁话超过文化，懂音乐打体育，扯把子撕卡子……"

　　在魏老八一连串的骂骂咧咧声中我把刘琳送回了宾馆，然后回家蒙头大睡。我没有想到魏老八当晚带了十几个弟兄摸进了"气胸"住的望江楼大厦……
