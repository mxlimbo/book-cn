第八章

　　第一次看见张放，就给我一个很好的印象。他高高的个头，一身很讲究的西装配上一双鳄鱼皮鞋，显得精明能干，智慧超人。他少了魏老八与金老九身上的流气，多了一分成熟稳重的个人魅力，他的眼睛像好来坞明星理察·基尔一样，眯成一条缝很随意地观察着周围的一切。

　　他就是魏老八说的与他一起贩海狸鼠的那位朋友，不过现在他早就对这桩只有农民才干的买卖失去了兴趣，与魏老八一道下了赌海。

　　这一天，我们一行人——魏老八、金老九、张放和我——驱车前往位于M市城西的云天大厦，我们是去和这座大厦的总经理谈判，准备将这赌场的场址设在这里。

　　这座雄伟壮丽的大厦是1995年才兴建的，分客房部、餐饮部、娱乐部三大部分，是M市比较高档的吃喝玩乐的场所。总经理姓张，叫张开国。魏老八讲，1983年严打时，张开国和他被关押在M市看守所同一间监舍。因为看守所内一星期才打一次牙祭，所以回锅肉成了最值钱的东西。一件毛衣，一双新皮鞋只能换一份只有七八块肥肉的回锅肉。魏老八由于名气和拳头在那间监舍里当上了牢头狱霸，他硬性规定每次打牙祭时，每个人必须向他进贡，最起码一个人贡献出两片肥肉，而张开国天生一副挨打板儿，他的那一份肉每次都被魏老八全部强占，一片不剩，只留下几根葱和回锅肉渣渣。有一次张开国实在忍不下去了，看见别的人嚼得油水直飚，他忽然放声大哭起来，边哭边喊："八哥，我求求你，你就给我剩两砣嘛!我都有半年没拈过'闪闪'（肥肉）喽!"

　　魏老八看他可怜，就夹给他两片最肥的肉片。张开国眼睛一瞪，脖子一伸，肉味都没尝到就吞进肚子里去了。由于长期无油水，当夜张开国拉了一晚上稀。

　　"俱往矣，数风流人物，还看今朝。"张开国出狱后，凭着他的三寸不烂之舌，在M市非法集资，又通过关系在银行贷了款，盖了这座大厦当上了总经理，真是一步登天，成了M市有头有脸的人物。

　　本来他想从此大干一场，哪知由于他的知识水平太低，毫无管理经验，听不得旁人劝，又喜欢钻牛角尖，一意孤行。在他眼里天老大，他老二，除他之外，别人都有"劣根"。可以这么说，自从1995年开张起，他就没有赚钱，身背着国家的债，他怕个卵?

　　他自认为他的云天大厦是三星级，在总服务台上挂满了世界各地的时间，有巴黎、伦敦、纽约、莫斯科，甚至还有法兰克福，他难道不知道M市在一年内也很难看到一个外国人的鬼影？在三楼的夜总会他请来了电声乐队，模特表演队、舞蹈队，但仅仅一个月，便偃旗息鼓了，因为他连工资都发不出，惟一还算是嫌钱的就是二楼的餐饮部。因为环境不错，一些婚宴、单位请客都喜欢在这里，但就这点赢利还不够交银行的贷款利息呐，加上他疯狂地爱上了他儿子的钢琴老师，更加无暇他的生意。那个少女涉世不深，刚从一所艺专毕业，分配到一所小学教音乐，平时收几个学生，教教钢琴基本指法，骗骗家长的钱。由于她被贫穷吓怕了，立刻被张开国的富有、风度、豁达所吸引，精心编了一个圈套，去等张开国上钩。女人要想勾引男人是不用任何人指导的，因为她们有这种本能，张开国已步入了中年，没想到一个妙龄女郎主动投怀送抱，自然没有理由拒绝，他不忍心伤一个少女的自尊，他明白他应该百般去呵护她，给她所渴望的一切。"谁说老来情无份，桃花偏照夕阳红"。只可怜了张开国的结发老妻，觅死觅活地与他大闹了一场，最终喝农药自杀了。

　　张开国正愁没结果，这一下正中下怀，洒下几颗鳄鱼眼泪之后，他与那个钢琴老师公开同居了。每日两人手拉着手，像向M市人民示威一样在街头逛来荡去，尽享这份迟来的爱。也许是阴间的妻子死不瞑目，偏要到阳间来骚扰骚扰他们。有一天夜晚，张开国与他的新爱不知不觉逛到公园，明媚的月光下，二人春情大发，相互搂抱着在一棵桉树下狂吻，不曾想灌木丛中跳出一个赤膊大汉，用杀猪刀架在张开国的脖子上，将他结结实实地绑在树上，抢走了他的手机、BP机，手表、金戒指和若干现金，并当着他的面强奸了那个钢琴老师。张开国害怕那个歹徒进一步伤害他的女人，急得直喊："别反抗，顺到他!"

　　那个歹徒是外地的一个杀人通缉犯，刚好逃到M市，正愁没个着落，碰巧张开国送上门来，一年过后，那个杀人犯被敲了沙罐，在他的犯罪事实中有在M市的这一幕，这个案子被人写成通讯发表在《××都市报》上，只是隐去了张的名字，而在M市，没有不透风的墙，谁都知道通讯中写的"一对热恋的男女"指的就是张开国和他的新欢，没有一个人同情他，都说是他死去的妻子对他的报复，活该!

　　从此，在M市街头很少能看到张开国牵着钢琴老师的手逛来逛去了，他只有在黑暗中悄悄地安抚那位容易受伤的女人。

　　一走进张的办公室，他完全没有要两砣回锅肉时的表情，而是像正要丢两砣肥肉似的。他穿着雪白的衬衣，打着一根大花领带，但领带打的太短了，在腰带以上几寸，倒像一条红领巾扎在课外辅导员的脖子上。

　　他一边用牙签剔着牙齿一边问："张放，啥子事?"

　　"就是电话里说的，赌场的事。"

　　"你准备开在哪里?"

　　"三楼的夜总会里。"

　　张开国沉默了一下，问："租金怎么算?"

　　"你开个价?"张放问。

　　"每个月三万!"张开国真敢开价。

　　没想到张放说："好!我答应你。"

　　张开国显然后悔了，他想他真该说五万块一个月，他将身子向前探着，说："水、电费还要另算。"

　　张放一下子火了，说："张开国，我是看得起你，才正儿八经地找你谈生意，啥子烂地方，要三万块一个月，我答应了你，就是想撇脱点，耿直点，水电费才好多?你也太小家把式了。"

　　我们忿忿走出云天大厦，魏老八说："他是啥子人，我最清楚，偷葱葱蒜苗犯的罪。"

　　张放说："他是怕开赌场有风险，想把价钱撬高点，没想到我答应的这么爽快，他没想到他把夜总会承包出去了，就是出了什么事也和他没关系，再说，我带来的赌客都是大款，今后吃、住都在云天，给他带来多少效益，太瓜了!"

　　这时张放接到一个电话，他听完后，关上机，兴高彩烈地说："问题解决了!"

　　"怎么解决了?"我们催促他快讲。

　　"我有一个朋友叫胡光，你们都认识的，他在河边开了一个休闲山庄，自从去年被公安踩扁以后，一直生意不好，他准备租给我。"

　　"价钱如何?"

　　"当然非常优惠，你们想想，云天在市中心，危险程度高，休闲山庄在乡下，面积又大，又有吃的又有住的，一个月才收我一万元。"

　　"太好了!"大家都兴奋异常。

　　魏老八说："所以说，朋友多了路好走。以后来了客人能住在山庄的就住，住不下的建议人们住在M市宾馆或者市委招待所，那里条件不错，价钱又便宜，绝对禁止他们去住云天大厦。"

　　接下来的几天真是把我们忙昏了，金老九到珠海去买全套赌具，包括洗牌机、发牌器、下注铃、赔码铲以及各种筹码；我和魏老八到家具厂订做了一张2米×4米的赌桌，按张放提供的图样，让美工在桌上绘出庄家、闲家以及和牌的位置，还有三十张供赌客用的高脚转动沙发，当然，茶几、茶杯必不可少。
