第五章

　　与"弹花手"的这场赌局，本是千术技艺的切磋，是一部跌跌宕起伏的情节剧。但由于时间的差距而失去了落幕的悬念，失去了扣人心弦的魅力，蓦然变得索然无味。但幕后的"气胸"却把这部情节剧引入了歧途，无端生出许多花边新闻。他就像一个智商很低的三流导演喧宾夺主地表现自己，挤眉弄眼地鼓到你笑，让你全身像爬满了毛毛虫，害得你屁滚尿流地从电影院里逃出来。看电影《好汉三条半》时我就有这种感觉，没想到又遇到一个井底之蛙"气胸"。他根本不像从"赌博之乡"隆昌出来的，倒像一个山村二流子只学了一点小魔术，便直接参加中国魔术大赛，而当众宣布："中国没有魔术，我就是魔术。"

　　"气胸"将他在劳改队里的恩恩怨怨直接在魏老八面前展示出来，而且还是在魏老八的地盘上，这使得魏老八万分恼怒。所以当晚他越想越气，便带着"疯子"他们几个跑到"望江楼"，想找"气胸"说个聊斋，能搁平更好，实在搁不平就只有撕破脸皮，大家摆开阵势大干一场，然后将"气胸"彻底赶出M市。

　　"气胸"虽然没什么文化，但他不属于"识字不多，吃得热和"那一类的。"死吃哈（傻）胀哈脓包"不属于他，他的智商并不低。他就像一只精明的耗子，肆无忌惮地闹得你全家鸡飞狗跳，等你咬牙切齿地准备好毒药时，他却知错似的销声匿迹了。

　　魏老八到达"望江楼"时是深夜一点，扯着哈欠的女服务员张着大嘴对魏老八说："他们……刚……刚才……才走……"

　　"走了好久？"魏老八问。

　　女服务员嘴唇叭叽着，眼泪水都扯出来了，她不耐烦地答道："我不晓得！"

　　"疯子"见她嘴巴有点狡，又是个潲水婆娘，就把手伸进腰里做掏刀的姿势，边掏边吓她："我一刀捅死你……"

　　女服务员吓得一屁股坐在地上，裤裆一热，挤了点惊尿，结结巴巴地答道："走……走了一个……个小时……"

　　一伙人失望地撤出"望江楼"，有人提议上高速公路撵一下，魏老八气鼓气胀地说："都一个小时了，他就是坐着'抱鸡婆'，也已经拢了内江，撵鬼大爷啊？"

　　第二天，我一觉睡到中午，雅安之行不仅让我疲惫不堪，连心都受到了牵累。我没有躲开尘世的喧扰，反而坠入更加凌乱的世界，我心里的负担加重了，尤其是刘琳这个小太妹，让我心乱如麻，神情恍惚。因为在她身上有太多太多刘萍的影子，她就像幽灵一样缠住了我的灵魂。

　　我来到宾馆叫她吃午饭，一推开门，见她仍慵懒地依在床上，枣红色的头发垂下来正好掩住那张年轻俏丽的脸庞，发育成熟的后背温柔地透过薄薄的内衣映入我的眼帘，我不禁有点想入非非，但同时又因为这种近似荒唐的念头而羞愧地无地自容。房间里乱作一团，沙发上放着皱巴巴的衣裤，烟灰缸里堆满了烟蒂，看来昨夜她没有睡好。

　　她看见我傻站在那里，冲我嫣然一笑，说："紧到矗在那里，找个地方坐到噻。"

　　我将沙发上皱巴巴的衣裤一古脑丢在床上，对她说："赶快去冲个澡，我们去吃午饭。"

　　她光着身子，"咚咚咚"地跑进卫生间，这与我和刘萍相识的那晚何其相似。那时刘萍也是三下五去二扒光衣服，风风火火地冲进卫生间，留给我一个曲线优美让人神智迷乱的背影。而此时此地却物是人非了，真是人生如梦，转眼就恍若隔世。

　　楼下的音像商店正高声放着张宇的《一个人的天荒地老》："……不痛的人不懂煎熬，曾经幸福的痛苦的该你的该我的到此一笔勾销……"张宇苍凉哀怨的歌声正好可以送给远方的刘萍。

　　刘琳在卫生局故意把水搞得"哗啦哗啦"响，她一边洗澡一边哼着小调，透出万分的轻狂。这种轻狂挑逗得我足以窒息。永恒的折磨、疯狂的希望彻底将我包围了，这种不知名的想象逼得我只想逃离。

　　我拨通了魏老八的手机，让刘琳在卫生间里自娱自乐去吧。

　　魏老八在电话里描述了昨晚在"望江楼"的那一幕，接着问道："今天中午到哪里吃？"

　　"就是我不晓得，干脆到南门吃'烧鸡公'！"

　　"要得要得。"魏老八笑着说，"我看你都变成骚鸡公了。你这会儿在哪里？"

　　"在宾馆。"我压低声音说，"她在洗澡。"

　　"将就一起洗个鸳鸯澡噻，我发现她有点喜欢你哦。"

　　"要不得，她是刘萍的妹妹。"

　　"这有啥子嘛？我经常搞这些台子，顺藤摸瓜。但我摸到的都是烂丝瓜，你运气还好，摸到根嫩黄瓜。"

　　一阵无聊的调侃后，我挂上了电话。刘琳这时湿漉漉地从卫生间里溜了出来，最打眼的就是一副精致绣花的乳罩，兜着一对沉甸甸丰满的乳房。我立即慌了神，眼睛都不晓得往哪里搁。我连忙转过身去，假装若无其事地吹着口哨。

　　刘琳边穿衣服边说："昨天晚上你打牌的时候简直帅呆了，我已经有点喜欢你了。我真的想拜你为师，你把你的技术哪怕只教我一点点，我就心满意足了。"

　　"我昨天在车上已经跟你说……"

　　我话还未说完，刘琳就打断了我："不要装疯迷窍，啥子学不得？你喜不喜欢我姐？"

　　我点了点头。

　　"对喽！因为我是她妹，所以你要教我。"

　　"这跟喜不喜欢你姐有啥子关系？关键是……"

　　"那你喜不喜欢我？"刘琳突然冒出这么一句。

　　我没想到她会这么直杠杠地问我，难道这个小太妹看穿了我心中那个肮脏的角落？我沉吟了一阵，摇了摇头。刘琳睁大吃惊的眼睛又追问了一句，我还是坚决地有点痛苦地摇了摇头。我以为刘琳最多不依不饶地撒一会儿娇，那想到她咧开嘴"哇"地哭了起来，边哭边说："那我这么远跟你来干啥子？我以为你会像喜欢我姐那样喜欢我，那想到你要绷起，我太……太瓜了。你以为你是周星驰？"

　　她这么一哭，吓得我连忙劝她。哪想到她一甩手冲出门去，气冲冲地跑了。我以为她只是年龄小，耍点娃儿脾气，没想到直到天黑，她也没有回来。

　　这时，我心头才有点慌了。
