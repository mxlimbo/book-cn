第十九章

　　几天以后，刘琳的姐姐刘萍乘飞机赶到了珠海。是我在电话里告诉了她发生在珠海的一切，发生了这么大的事，我不可能瞒着她，再说也没有必要瞒着她。她没有来得及参加刘琳的葬礼，因为我打了几天的电话，她家里一个人都没有。经过几天的努力，电话终于打通了，我猜测她在得知刘琳死去的消息后，首先会伤心的大哭，然后把我骂得狗血淋头。但是我的猜测完全错了，她在电话那头表现得出奇的平静，就像她听到刘琳只是弄丢了一百块钱似的，毫不在乎。我倒反而为刘琳打抱不平起来，她们毕竟是骨肉相连的同胞姐妹呀！

　　我莫名其妙地在电话里冲她发起火来，竹筒倒豆子一样，稀里哗啦向她发起了牢骚。闷在心中排解不开的所有怨愤、悲泣、忧郁、失落，一古脑儿地向刘萍倾泻过去，管她接受不接受，抛过去就万事大吉。我就像一个迷了一年的路，终于找到家门的痴呆儿，傻傻地咧嘴笑着，挥舞着树枝一样的手臂，嘴角上流淌着飞舞的涎液，向自己的家人狂奔而去。家人已经陌生了，他们不耐烦地像赶苍蝇一样百般阻挠我诉说我一年来的遭遇，我的话匣子根本就没有关上的意思，也根本没有注意家人冷若冰霜的面孔。

　　刘萍被我的牢牢叨叨彻底惹毛了，电话那头她冷冷地说："你有什么资格对我指手划脚？"

　　这句话立即把我问愣了，马上闭上了鸟嘴，一时半会儿还真不知道如何回答。我突然冒出来一句杀着："我已经跟她结婚了。"说完这句冒冒失失的话，连我自己都被我的这句话吓到了，我像一座铸了八百年的铜像，满身绿色锈斑，一动不动地矗立在那里，脸上挂着不尴不尬的微笑。

　　电话那头的刘萍也绝对惊呆了，听筒里沙沙地响着，像萧飒的寒风，刘萍躲藏在风沙的背后，暂时消失得无影无踪了。我连对着听筒"喂喂"了几声，刘萍才像刚从阴府冒出来一样，冷冷地说："我马上坐飞机过来！"

　　这是自从那场"雅安赌局"后，刘萍对我说的最多的一次话，也是她说的最正常的一次话。看来她已经完全地恢复了，为她高兴的同时，也为我们两个的有缘无份而深深地遗憾着，她妹妹的出现已经把我对她的仅存的一点爱化为了灰烬。我连在灰烬中扒来扒去，哪怕能寻找到代表我们过去的一小块木炭的心情都没有了。因为我和刘琳最初也是最后的那夜疯狂，使我对刘萍彻底失去了感觉。人的一生面临着无数个十字路口，只要你很随意地选择了其中之一，你就永远地没有了退路。什么"悬崖勒马"，什么"亡羊补牢"，什么"脱胎换骨"，什么"金盆洗手"，全是人们自己安慰自己的借口，不是你没有了退路，而是你已经丧失了退回去的心境。

　　站在机场外，看到那架从成都飞来的波音757像一只归家的老鹰在珠海的上空盘旋徘徊着，它在寻找着它熟悉的巢穴。

　　跟刘萍相见的一刹那，我竟有一种恍若隔世的感觉，她就像我童年的梦中曾经玩耍过的地方，熟稔亲切，陌生而遥远。一件淡黄色的翻领Ｔ恤，一条洗得泛白的"Lee"牌牛仔裤，一双普普通通的"Playboy"波鞋，恬静素雅的装扮衬托着她那黑色瀑布一般的长发，乖巧动人的鼻梁，使她的全身笼罩着一丝"文化"的气息，这和她的妹妹刘琳有着不可更改的天渊之别，除了长相极为相似以外，你根本找不出她们俩有着千丝万挂的血缘关系，她们像生长在两个完全不同的家庭环境里：一个是大学教授的千金，谈吐典雅大方，举止端庄而不失诙谐，惹得教授的学生们各自揣着各自的求爱信，蜂拥而至；而另一个则是个老街娃的独女，满嘴污言秽语，走路一拽三摇，举止放浪形骸，惹得全街的"骚鸡公"半夜起来打更扯鸣，翻着墙头吹口哨。教授的女儿自有她暗自以身相许的意中人，而这个"街女"非我莫属，我喜欢。

　　突然间，我发现我和刘萍根本无法沟通了，她应该不属于我们这个世界。她对她的妹妹冷若冰霜的态度把我激怒了，我甚至不想跟她多说一句话。她见到我时，劈头盖脸就问："谁把她杀死的？"

　　我不想告诉她雷鸣的名字，告诉她了也一点不起任何作用。

　　但她仍不依不饶地问我："是谁把她杀死的？"

　　我还是没有回答她。

　　当在酒店里见到放在桌上的刘琳的骨灰盒时，她却一改刚才的冷漠，不顾一切地扑了上去。她把骨灰盒紧紧地抱在怀里，生怕她从她的臂膀间不经意地悄悄溜走，她把脸轻轻地贴在上面，害怕吵醒沉睡中的刘琳似的，她开始"嘤嘤"地哭了起来，柔弱的肩膀微微地抖动着，一缕被泪水浸湿的头发随着她的哭声而荡来荡去。

　　我可能错怪她了，刘琳毕竟是她的血脉相连的亲妹妹，她没有理由不悲切，没有理由不伤心落泪。我想走过去安慰安慰她，但却不知从哪里下手：抚摸她柔顺的长发，还是递过去洁白的纸巾？是说些让她节哀的话，还是把她从骨灰盒上吆喝起来？怎么都不合适，还是不要去轻易惊动她，让她安安静静为她的妹妹送行吧！

　　我悄悄地从房间里退了出去。

　　魏老八正在房间里斜在沙发上看电视，看到我脸色不太好看地走了进来，忙问："人已经接回来了？"

　　我点了点头，顺势从冰箱里拿了一罐可乐，"咕咚咕咚"地直往嗓子眼儿里灌，然后问："雷鸣那边有没有消息？"

　　魏老八答道："明天晚上，我约好跟他们的老大谈判。"

　　"老大？"

　　"是的，'福建帮'的老大。"

　　"谈什么？"我迫不及待地问道。

　　"我让他交出雷鸣。"

　　"水仔知道吗？"

　　魏老八摇了摇头，说："我不想再惊动水仔他们，这次来珠海已经够麻烦他的了，我实在不想因为我们私人的恩怨惹得他们再和'福建帮'杀来杀去的。"

　　"他们的老大答应了吗？"

　　"答应了，晚上十一点。"

　　又是一场没有硝烟的战斗即将拉开那重重的帷幕，这次我们面对的不是温文尔雅的牌局，不是玩弄心计的尔虞我诈，没有施展千术时的镇定自若，没有大胜而归时的沾沾自喜。这次我们直面的是一位声名显赫，"威震江湖"的帮会老大，他集金钱、地位、狡诈、毒辣、阴险于一身，与他硬碰，简直就是飞蛾扑火，不自量力。但我们已经无法退缩了，因为身后是一辈子的负疚在等待着你，它就像一片硕大的沼泽地，里面有数以亿计的蚂蟥虫豸，当你陷入那片烂泥而挣扎不起时，他们就会争先恐后地蠕动过来，用吸盘吸住你的肌体，狼吞虎咽地喝干你的鲜血，直到你化为一滩泥水，终生与蚂蟥共舞。

　　是直面怒视死亡，还是退回去享受一辈子的折磨，我们别无选择地将我们的生命押在了"死亡"上了。
