第十二章

　　第二天凌晨三点，我们已经潜伏在那家酒店的对面．我和魏老八、疯子躲在街东头一辆黑色的轿车里，而水仔带着它的兄弟们则藏在街西头的一辆红色的轿车里．我们的人已经把这条街包围得严严实实．魏老八不停地抽着烟，黑暗中那红红的烟头一闪一灭的，照亮了两颗猎豹一样的双眸．而我也眼睛一眨不眨地死盯着酒店的大门，生怕漏掉哪怕一只蟑螂．

　　天气又闷又热，远处的海面上不时传来隆隆的雷声，车内的空调已经开的最大了，但粘乎乎的汗水仍止不住地顺着脊背往下淌着．我忍不住打开了ＣＤ机，把音量调得最小，欣赏起张楚的《姐姐》，这是下午我和刘琳到街上的一家音像店买的："这个冬天雪还不下／站在路上眼睛不眨／我的心跳还很温柔／你该表扬我说今天很听话／我的衣服有些大了／你说我看起来挺嘎．．．．．．"

　　凌晨四点，酒店的大门仍然没有任何动静，街上的行人渐渐地少了，有几个喝醉酒的少年踉踉跄跄地相互搀扶着，嘴里哼着任贤齐的《伤心太平洋》，有一个还蹲在路边大口大口地呕吐着，惹来了他的朋友大声地狂笑．

　　这时水仔打来电话，说："赌场马上就要散了，你们盯好酒店的大门，一旦发现目标，马上电话联系，选好位置，将他网住．如果他们的人发现了，立即火力伺候．"

　　正说话间，一群一群的人从酒店的大门涌了出来，我们立刻紧张起来，我全身的汗毛孔全都张开了。魏老八说："眼睛放亮点儿，别让这家伙溜掉了。"

　　在熙熙攘攘的人堆里，我发现了那个让我们等了一夜的他。魏老八马上打电话通知水仔："人已经找到，人已经找到．上身穿了一件白色的Ｔ恤衫，下身穿了一条黑色的牛仔裤，身高一米七十左右。"

　　水仔答道："我已经看到了，八哥，你放心吧！你们可以先回酒店了，把人抓到后，我立刻给你送来．"

　　我们熬了一整夜要绑架的这个人，就是魏老八曾经放他一条生路，尔后又与我们在雅安大赌一场的"一指禅"。昨天晚上我在赌场见到他的时候，只能用四个字来形容我：目瞪口呆．当时我根本无法相信自己的眼睛，我没有想到在遥远的珠海能见到"一指禅"，真是冤家路窄，人生何处不相逢啊！昨天晚上我把这个消息告诉魏老八的时候，他倒没有大惊小怪，只是微微地笑着说："看来老天爷都要帮我，他在那里还是干他的老本行吗？"

　　"是的，"我答道，"他是那家赌场的金牌发牌员．"

　　一个大胆的计划在魏老八的脑海里像浮尸一样飘了上来，他马上找来水仔，说："把那个发牌员绑架了．"

　　"然后．．．．．．"水仔问道．

　　"然后就放虎归山，让他在那家赌场当卧底．"

　　"这样做太危险了，"我插嘴道，"我们现在根本不晓得'一指禅'脑壳里面是怎么想的，万一他又像上次在雅安那样，帮格桑打我们的'倒挂金钟'，那我们就真正的死得早了，有多少钱就输多少钱．"

　　"我倒不这么认为，虽然我们在J市的赌场上抓住了他和他二爸精心策划的'千局'，又让他的二爸尸沉湖底，但毕竟我们给了他一条生路，这等于给了他第二次生命，我相信这次他一定帮我．"

　　看来这是唯一的办法，也是打败雷鸣的唯一途径．

　　我和魏老八并没有立刻回酒店，而是一直跟在'一指禅'的身后，他似乎察觉了一点儿什么，像耗子一样顺着墙边哧溜哧溜地向前蹿着，刚一拐过街角，就幽灵一样钻进了一辆出租车，说时迟那时快，一辆红色的轿车尖叫着冲过去，拦住了出租车的去路，车轮与路面摩擦出一大股浓浓的烟雾，水仔的兄弟们嚎叫着围了过去，一把将车内的'一指禅'提了出来．然后四个膀大腰圆的小伙子抓起'一指禅'的四肢，硬朝红色的轿车塞去．'一指禅'呜呜地低声呻吟着，无助地蹬踏着四肢，他像一只还未发育成熟的仔鸡无奈地被一群"饿痨虾"急唠唠地送进了厨房，陪伴他的是一大堆嫩绿的青椒．

　　他开始以为遇到了一泼当地的'棒老二'，但仔细打量一下这泼人的打扮，气质，又不像是一般的土匪，他隐隐约约地感到这不是普通的绑架，而是有更深一层的含义．他消瘦的脸颊更加苍白了，额头上渗出一颗一颗豆大的汗珠。长期在赌场上熬夜，已经使他的身体非常虚弱了，再加上又惊又怕，他几乎已经虚脱了。

　　一进酒店的房门，他一眼就看见魏老八正襟危坐地坐在沙发上，他一切全明白了。他感到整个房间都旋转起来，他无力地扶着墙壁，几乎没有了呼吸。魏老八在J市放了他一条生路，但毕竟抚养他成人的二爸被胡疤杀害了，而且正是魏老八们揭穿了他们的千术，才导致了这场人间悲剧。从那以后，他再也没有见到他亲爱的二爸，他真想重返J市，找到那个孤岛，然后跳进冰冷的湖水，一直向幽深的湖底游去，在那里长眠着他日夜思念的二爸,只是他不知道能不能寻找到那颗思维慎密的头颅．．．．．．

　　他有气无力地对魏老八说："随便你咋个处理，栽在你手里我心甘情愿了。"

　　魏老八说："我救过你一回，所以我不打算再救你了。不过．．．．．．"

　　"一指禅"像看到了一颗救命的稻草，就像在J市的那艘快艇上一样，他慢慢地抬起了头，睁着那双迷惘的眼睛望着魏老八。

　　"你和那个雷鸣到底是什么关系？他知道不知道你的'一指禅'？"魏老八问道。

　　"我只是个替他打工的，我不可能轻易透露我的牌技，只是因为大家都是四川人，所以他对我比较照顾。"

　　"我想让你在他的赌场当一个卧底。"魏老八咄咄逼人地盯着"一指禅"。

　　"一指禅"脸上的汗水"刷"地淌了下来，他嘴里嗫嚅着，半天没有说一句话。

　　魏老八说："我们俩过去的恩恩怨怨从现在起一笔勾销，我们重新开始，做一个好搭档。其实我很看得起你这个人，这也就是我为什么在J市放你的原因。"

　　"一指禅"听了魏老八的这一席话，苍白的脸上竟然泛起了一丝红晕，他慢慢地抬起了头，脸上荡漾着内容诡秘的笑容，他点了点头，说："其实那场雅安赌局，我已经输得心服口服了，当时我的心里相当复杂，你既是我的仇人，同时又是我的救命恩人，我想杀你，也想找个机会报答你。没有想到世界竟然如此渺小，我们没有相遇在北京、上海，却在小小的珠海相逢了，看来我们的'缘'还没有散尽。我答应帮你，但帮完这次过后，我们最好各奔东西，我不想和你做朋友，也不想和你做仇人，就当成天涯沦落陌生人吧，相别何必曾相识。"

　　魏老八笑着说："果然痛快！走！我请你喝早茶去。"

　　又是一场"倒挂金钟"的精美赌局，赌局的设置机关已被魏老八和"一指禅"悄悄地启动了，只是不知雷鸣有没有第六感官的超前反应。掘开的墓洞已经敞开，只等雷鸣乖乖地躺在那儿，然后对着洞口像《地道战》中的汤司令那样，咬牙切齿地喊道："给我放毒、灌水，把他们全都闷死在里面！"但我心里非常明白，这是一个非常冒险的赌局，魏老八把我们的一切当作赌注，连换成筹码的时间都没有，就这样一古脑儿的甩在了赌桌上。其实这正是一个赌徒的'本色'，他根本不考虑等待他的究竟是什么，在他觉得该出手时他会毫不犹豫地大打出手，他是个亡命徒，但他却相当惜命，魏老八经常对我说："你不要看我没得啥子文化，我这条命还是条皇帝命，不然我为什么有'三妻六妾七十二妃'？但我妈老汉儿在做我的时候，可能门外正在打雷，所以闪了一火，把我投胎投到一个劳改犯身上。我这条命是很值钱的，你算一下我这辈子要找好多钱？要不是这样，我早到土里头去割红苕藤藤去了。"
