第一章

　　汽车驶进雅安城区的时候已是傍晚时分，美丽的雨城被五颜六色的霓虹灯装扮得绚丽多姿。我跳下汽车，大口呼吸着雨城清新而湿润的空气，气味微甜、沁人心脾。熟悉的楼宇，熟悉的街道，我想起了那场惊心动魄的"雨城赌局"。

　　按照以前刘萍告诉我的地址，我很快就找到了她的家。这是一个典雅别致的住宅小区，林荫小道弯弯曲曲，一座设计新颖的亭榭，四周簇拥着密密匝匝的矮丛林，清澈的水池中央耸立着披满青苔的假山，整个环境显得优雅而恬静。

　　她的家住在B幢3单元4楼右手第一间，我小心翼翼地按响了门铃。

　　门徐徐拉开了，刘萍亭亭玉立地站在我面前。我几乎不敢相信自己的眼睛，这难道是我每每为她牵肠挂肚、魂牵梦萦的刘萍吗？枣红色的头发，紫青色的睫毛，烟黑色的口红，高耸的胸部撑起一件露脐的短毛衣，下身是一条宽大的国防绿军裤，活脱脱的一个小太妹。

　　"你找哪个？"她像陌生人一样冷冰冰地问了我一句。

　　"我……我……"我张口结舌，还没从极度惊愕中清醒过来。

　　"你是不是找错庙门了？"她边说边想关上房门。

　　"喂，喂！"我一下子急慌了，"我就是找你噻！"

　　"我啷个认不到你哎？"她努起嘴唇，极不耐烦地"砰"地关上门。

　　"刘萍！"我大叫了一声。

　　她"哗"地拉开门，霍地冲到我面前，双手叉腰，机关枪一样朝我开了火，"你这个瓜眼镜，四个眼睛都没看清楚我是哪个，你好生盯一下，刘萍是我姐，我是她妹，本小姐大名刘——琳！！！"

　　"刘琳？"刘萍的妹妹？我怎么没听说过。就在我不知所措地张大嘴巴木戳戳地站在那里时，她已经一阵风似地旋回屋里，顺便丢下一句："进来噻！害怕我咬你一口嗦？"

　　这哪里是什么女人，简直是一个小疯子。我尴尬地迈进门去，好像随时等待这个女疯子的修理。

　　她点上一支烟，一屁股歪在沙发上，翘起二郎腿，斜着眼睛，说："赶快报上姓名！"

　　这个刘琳实在没有家教，看上去漂漂亮亮的，说起话来一身匪气，就像是在社会上"飘起"的。

　　我没好气地说："赶快把你姐喊出来，就说龙兵来找她。"

　　"啥子？"刘琳一下子从沙发上弹了起来，吃惊地走到我面前，上下把我扫视个够，脸上也变得灿烂起来。

　　"耶？你是龙兵嗦？看你戴个眼镜瘟戳戳的，原来是面带猪相，心头嘹亮。听我姐经常说起你，说你打牌有点凶！"

　　刘萍经常提起我？听到这句话，我心头温暖了许多。这时，里屋的门"吱"地开了，刘萍面色苍白地倚在门框上。她幽幽的眼睛无神地望着我，有气无力地说："谢谢你还记得我，你回去吧！我不可能再和你在一起了。"说完回身掩上了门。

　　我面有难色地进也不是退也不是，刘琳却冲我做着鬼脸。她一把将我拉坐在沙发上，说："你先坐着休息会儿，我给你泡杯茶。"

　　我仔细端详了一下刘琳，间直和刘萍是一个模子刻出来的，只是刘琳的打扮太前卫了一些。尤其刘琳的笑脸与以前那个活泼可爱的刘萍一模一样，这不免勾起我无尽的哀伤。那场"雅安赌局"得到的与失去的一样多。

　　刘琳把茶放在我面前的茶几上，说："我姐姐自从那件事以后几乎和精神病人差不多了，整天不开腔不出气的，闷起脑壳傻笑，说起来也够可怜的。有时摆起你，她的情绪还稍稍好点，像个正常人。但是过不了一会儿，她就恢复原状了。"

　　"你们家就只有你们两姊妹？"我问。

　　刘琳杏眼圆睁，说："未必我没有妈老汉儿？我妈倒早死了，但我爸是个名气犯，是我们这条街的老麻将，他娃娃耍得鲜，刚刚邀邀约约地出去，看明天早上回来得到不？"

　　这个没教养的刘琳，性子有点暴躁，而且满口"社会"语言，没大没小，刚刚还静如处子，转眼间就动如脱兔，她是不是也和刘萍一样操过几天"扁卦"？

　　走出刘萍的家门，我有一种碰了一鼻子灰的感觉，我开始后悔来雅安了。本想离开那个纷纷扰扰的世界，哪想到又掉入另一个更加混沌污浊的泥淖。雅安的天空又淅淅沥沥地飘起了雨丝，我的思绪也像那细长的雨丝一样绵绵不断地滴在地上……

　　在雅安宾馆松软的床上一觉睡到天亮，一阵急促的门铃声将我吵醒。打开门一看，原来是刘琳，也不知她是如何找到这儿的。她一阵风似地冲进屋来，劈头盖脸地就说："赶快穿衣服，帮我去报仇！"

　　"报啥子仇？"我一脸疑惑。

　　"有几个虾子长期在麻将桌上医我，把我和得瓜兮了。今天你要帮我和转来！"

　　我一听原来是这种"耍耍麻将"，连忙摇头说："不行！不行！我的麻将技术管不到火，再说你每盘都和又能赢多少钱？"

　　刘琳恼羞成怒，指着我的鼻子尖，说："你到底帮不帮？"

　　"不帮！"我咬紧牙关回答道。

　　我以为刘琳会一跺脚地给气跑了，哪想到她马上堆着媚笑，拉住我的胳膊，哀哀地求我："就帮我一次嘛！你会出千未必怕他几爷子？"

　　我在她身上发现了刘萍缠着我讲笑话时的影子。无奈之下，只有冒充麻将高手，被她生拉活扯地骗到麻将桌上。结果可想而知，不一会儿，我就输了几百块钱。

　　在回宾馆的路上，刘琳斜着眼睛，冲我唠叨个不停："原来是个假打嗦？我还以为是个赌圣，现在的'李伯清'满街遍野都是！"

　　本来我的"空手道"只能应用于"软牌"，与麻将这种"硬牌"根本不相通。为了证明我不是李伯清，我被迫给刘琳表演了我的绝技"空手道"。这一下麻烦惹大了，刘琳彻底服我了，高矮要跟我学几招，要我收她为徒，这怎么可能？梁姐反复告诫我，没有她的同意，我是不能收授门徒的，师傅的告诫我不可能违背。

　　正在这时，魏老八打来电话，让我火速赶到M市，说有一个外省人在M市耍"软牌"，杀遍全城无敌手，必须由我代表M市向他挑战。

　　我刚刚从泥泞中跃起，将身上洗净。此时此刻，我又身不由己地返回泥泞……

　　我的头发已被雨水淋湿，曾经共度的良宵也被风吹干。爱过，就是美丽的，所以我们决不后悔，我毅然踏上了征程……
