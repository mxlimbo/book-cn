第十四章

　　珠海的夜晚很美丽，珠海的海滩却一点没有任何新意。被太阳晒暖的沙子，带着咸味的海风，悬在夜空的圆月。我和刘琳在沙滩上已经逛了无数个回合，身后留下了一串串杂乱无章的足迹，我们终于在一个凹洼的地方坐了下来。刘琳的情绪好像很低沉，她低垂的头，一缕染过的头发轻悠悠地耷拉下来，在她的鼻尖蹭来蹭去的。她那忧伤的脖颈上有一层细细的绒毛，在月光的照射下透出一片朦胧的白色，我差不多要去吻她了。我知道如果吻下去的话，我和她之间的内容就会发生天翻地覆的变化，她就会是我的女人，就会一天到头地跟在我的身后缠着我。我有很多有关女人的故事，我可以翻着花样地给你讲上十天十夜，这次和雷鸣赌过之后，我准备静下心来，写它一本新《十日谈》，把我在女人方面的心得体会、离奇经历编纂成一本宏伟巨著，掀起中国新的一轮隐私揭秘大战，引得那些喜欢专门偷窥别人私生活的人们疯狂抢购，哪怕长上一百颗针眼。我还应该在西南书城签名售书，向笨拙的男人传授经验，向寡居的女人'仗'去秋波。我必须用两个很生涩偏僻的字来当作我的笔名，比如用"孑孓"，让文化稍低的读者想要认识我，必须先买一本《新华字典》。我还必须在脸上蒙上一块黑布，以防被我害过的女人组成粉红兵团，到书城来掀我的摊摊。

　　我问刘琳："你好象有什么心事？"

　　刘琳叹了口气，说："我想我的姐姐．"说完又把那首唱烂了的《姐姐》翻了出来，她唱歌的声音像低声的呜咽："姐姐我看到你眼中的泪水/你想忘掉那侮辱你的男人到底是谁／他们告诉我女人很温柔很爱流泪／说这很美／哦！姐姐！我想回家．．．．．．"

　　又是那个阴魂不散的刘萍，她像罩在我和刘琳之间的一团阴霾，久久地拂之不去，我曾经尝试用刘琳替代刘萍，有几次差点成功，刘萍已经远离了我的记忆，变成了遥远的一团浮云，我期待着我和刘琳之间来一场飓风，彻底将刘萍吹得无影无踪。然而每次到了关键时刻，刘萍就像鬼魂似的飘然而至，悬挂在我和刘琳中间，让我的激情、亢奋顿时变得索然无味。

　　去他妈的刘琳、刘萍！我决定不再对她们想入非非。我跟着刘琳干吼了起来："我的爹他总在喝酒是个混球／在死之前他不会再伤心不再动拳头／他坐在楼梯上面已经苍老／已不是对手／哦！姐姐！带我回家／牵着我的手／我有些困了．．．．．．"

　　我变了调的嗓音像一只迷了路的秃狼．

　　魏老八他们并没有去夜总会找小姐，他们是去找水仔商量对策。半夜回来的时候，魏老八带回了新消息。他对我说："雷鸣最近一直在澳门，刚才'一指禅'打来电话，说雷鸣大概明天就回到珠海，叫我们做好战斗前的准备，随时等他的电话．"

　　我莫名其妙地紧张起来，这对经历过无数大场面的我来说，实在是不应该的。冲锋号已经吹响，临到跃出战壕时却忽然想起了拉稀，像卓别林在《兵营生活》中那样，等他爬出战壕，战友们已消失得无影无踪了，他误入了敌人的编队，跟着人家越走越远，等到双方惊谔地发现军服不同时，他哪里还跑的脱？我倒不是害怕自己误入雷鸣的方阵，而是担心自己这种心理状态，容易迷失自己。

　　刘琳也开始失眠，躺在床上翻来覆去的，她这种不安的情绪直接影响到睡在沙发上的我，我辗转反侧，感觉芒刺在背。我俩就像地下工作者那样，扮成假夫妻，让魏老八们误以为我们睡得非常甜蜜，非常安稳，他以为我早把刘琳搞定了，这给了我那狗屁虚荣心极大的满足，我趾高气扬地揽着刘琳在魏老八面前晃过去、晃过来，好像炫耀着我的战绩。刘琳也好像默认了一样，小鸟依人地倚在我的肩头，俨然嫁鸡随鸡、嫁狗随狗了。这种默契让我久久地激动不已，我认为刘琳在我面前成功地将一个女人的"情"和"欲"合二为一了。有时她是"情"的化身，有时她是"欲"的魔体，而更多的时候她是二者的结合。就这样，我们俩不明不白地厮混在一起，彼此都有一种无言的默契，"就这样被你征服，喝下你藏好的毒，我的心情已落幕，我的爱恨已入土…….."那英的歌提醒我，在陷入了感情的泥淖后，哪怕对方为你准备了一碗砒霜，你也得硬着头皮将它当成一杯浓香的咖啡，苦笑着把它喝下去。

　　"一指禅"就为雷鸣准备了一大碗纯度极高的砒霜，这是他在苦思冥想了几天几夜后，躲在地下室里精心调制的，随时准备端到雷鸣面前，用特大的瓢羹一口一口给他喂下去，然后亲眼看着他七窍流血，在地上扳来扳去。欣赏死亡，这是将死亡置之度外的人才有的悠闲心境。

　　第三天下午，"一指禅"向魏老八发来消息："今天晚上，雷鸣将有一场大赌局，对手是澳门的一个大亨，在江湖上颇有点名气，连'十四Ｋ'都尊敬他。我是这场赌局的发牌员，这可是一个千载难逢的好机会。你们这个时候闯进来，向雷鸣直接挑战，碍于澳门大亨在场，他不得不答应。'福建帮'开的场子不敢应战，传到江湖上是要被人耻笑的。但如果没有那个澳门大亨，情况就不好说了，多半不会应战，因为你们是有备而来的，他不敢冒这个风险。记住！赌局大约是在晚上九点钟开始。切记！切记！！！"

　　魏老八立刻向我们布置任务，他表情严峻地说："刘琳，你要记住！你是我们的赌后，你只要随心所欲地发挥就算完成了任务，有'一指禅'帮我们，不会有好大的问题。枪，我估计根本带不进去，不过有水仔他们在外面接应，一旦干起来，他们可以随时冲进来。我要让雷鸣尝尝我的厉害！"

　　夜幕降临了，离出发的时间越来越近，这时我反倒镇静了下来，权且把它当成一场普普通通的赌局吧！但我心中隐隐约约总有一种不详的预感，我觉得"一指禅"是不可信任的，他那犹犹豫豫、游离隐晦的目光包含着许多说不清的含义，在雅安，他可以帮格桑，在珠海，他完全有理由帮雷鸣。这是一个深不见底的陷阱，引诱着我们走入黑暗，他那碗熬制已久的砒霜会不会是为我们准备的？

　　水仔提了两提包钱，拿给魏老八作为今晚的赌资，作为一个相交不久的朋友，能做到这一点真可谓仁至义尽了。他对魏老八说："我不能陪你到赌场去，他们都认识我，一见面非干起来不可，我只能当你的大后方。你放心！赌场上有我们的人，一旦发生什么意外，我立刻带人冲进去。你就专心致致地赌你的钱吧！好！有什么情况再电话联系。"

　　水仔走后，魏老八不禁仰天喟叹："有这样的好朋友，我没有理由不赢。"

　　是的，天地人和我们全占了，我们没有任何理由不赢，现在只看"一指禅"的表演了。在这种大型的赌场上，你休想展露一点千术，只能靠发牌员施展"倒挂金钟"。"一指禅"会不会临场倒戈呢？我的心里七上八下的，实在有点提心吊胆。这场赌局只能有５０％赢的把握，甚至更少。"一指禅"成了我们的救命赌注，我们把他押在了牌桌上。

　　冒险，正是赌博的精髓，我们准备险中求胜。"一指禅"，你发牌吧！
