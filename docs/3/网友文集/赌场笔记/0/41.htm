第十章

　　从茶楼走出来后，才发现天色已黑，和平路上灯火辉煌，深蓝色的夜空星斗阑干，交相辉映，构成了一幅绚丽多彩的画卷。我们早已饥肠辘辘，连忙坐三轮赶到马巷子，这是全城闻名的火锅店区。这里早已人声鼎沸，热闹非凡。我们挑了一个干净点的桌子，一口气点了十几种菜。魏老八还破例要了一瓶"雪花"啤酒，我知道他不是为了赢的那4万块钱，而是14年来第一次在雷鸣面前争回了一点面子。

　　魏老八端起酒杯对刘琳说："为我们的赌后干杯！"

　　刘琳不好意思地说："啥子赌后哦？我心子尖尖都在颤抖。"

　　我说："第一次上战场，能有你这样的表现就很不错喽！"

　　得到我的褒奖，刘琳的脸顿时绯红，睫毛一眨一眨的，鼻子尖竟渗出细密的汗珠。

　　魏老八呷了一口酒，说："虽然今天只赢了4万块钱，但比我赢40万还要高兴。可我总觉得这里面有点不对劲。"

　　"有啥子不对劲？"我问。

　　"雷鸣不是这么瓜的人，我觉得他好像在'喂'我们，而且他一个劲地喝茶……"

　　"那是由于他紧张吧？"

　　"他在珠海操了这么多年，早就是老鬼了，我觉得他像在演戏，好像出千前的铺垫，让你习惯他的某一个习惯动作。我也说不出个点点，总之我有这种预感。"

　　正说话间，邻桌的两个飞女银瓶乍裂般的划拳声惊了我们一跳，她们两只手张开着，尖声尖气地喊到："两只小蜜蜂啊，飞在花丛中啊……"话音还未落，右手这边两个半大小伙子，打起光胴胴，全身被酒烧得通红，他们震天动地、莽声莽气地吼道："红红的太阳蓝蓝的天，绿油油的稻田一望无边，乡下的农民爱看三级片。好多钱的三级片？五角钱的三级片……"

　　这一粗一细的划拳声，此起彼伏，抑扬顿挫，几乎我们的耳膜振破。魏老八不满地摆着脑壳，揶揄道："他们可能要下蛋。"

　　在回宾馆的路上，刘琳一直斜靠在我的肩头，她可能喝麻了，旁若无人地哑着嗓子唱着那首张楚的《姐姐》。进门以后，我像扛麻袋一样将她丢在床上，她却一个鲤鱼打挺一把揽住我的腰，嘴里含含糊糊地嘟囔着："别丢下我！"

　　我不知道我们是如何吻在一起的，也许是酒精的作用吧。她满脸泪水，全身像火炭一样，透过微弱的灯光我看见罩在她身上的一层亮亮的雾气，光滑的膝头不耐烦地蹭来蹭去，最深处的甜美膨胀成一种热烘烘的颤抖，我的全身都快痉挛了。她温暖柔软的舌尖给了我巨大的勇气，我想要抛掉我们随身携带的物品，但刘琳却阻止了我，她坐起身来对我说："我觉得我对不起我姐。"

　　我狼狈地满脸羞愧地从宾馆逃了出来，像一只丧家的一野狗，我和她之间始终有一个刘萍……

　　第二天的赌局如约进行。雷鸣开始猛烈地反击，他加快了思维过程，加大了赌注，也更加频繁地一口接一口地喝着茶水。我们明显感到来自雷鸣的阵阵寒气，就像古墓中的阴风。

　　我和魏老八始终紧盯着雷鸣的双手，生怕一不留神被他钻了空子。然而他的手气出奇地顺，下多大的注，或跟不跟，他都掌握的恰到好处，就像他早已看穿我们的底牌一样。这真让我们匪夷所思了，牌是崭新的，也是仔细验过的，大家都穿着短袖上衣，纸牌也没有离开桌面，这个昨天立下的规矩，大家都在认真地遵守，应该没有出千的可能。刘琳的手气一点不坏，但当你牌好的时候，他却知趣地不跟了，你牌坏的时候，他却咄咄逼人地扑了上来。雷鸣就像是一只老奸巨猾的千年灵狐，他耐心地在你的巢穴外徘徊着，随时准备给你致命的一击。

　　刘琳已经抵挡不住了，魏老八准备给她换换手气，但赌局已经到了结束的阶段，纵然你有特异功能，能翻天覆地也已无济于事，魏老八上去也只是多撑几个回合而已。

　　赌局一清盘，我们干输了18万元。其中的10万元可是借成都傅老大的"水"钱啊，超过20天就翻番。我们垂头丧气地从茶楼走了出来，像输光了家产的穷光蛋。

　　魏老八一直紧蹙着眉头，自言自语道："这里面一定有啥子'汪二麻子'，我一直有这种感觉，并不是不能输这10万元，而是输得莫名其妙。"

　　刘琳像做错了事的孩子，紧紧拉着我的胳膊，一声不吭地垂着头。魏老八安慰她说："不关你的事，可能我们全被雷鸣烫了。"

　　我突然想起了我的千术老师梁姐，马上毫不犹豫地拨通了她的手机，在我们一筹莫展的时刻，我需要她的帮助。

　　电话那头"沙沙"作响，隐约传来梁姐熟悉的声音："喂！哪个？噢，是龙兵嗦？我现在在泰国的清迈，这里有一场东南亚的赌博大赛，我是特邀嘉宾，你有啥子技术上的问题嗦？"

　　我将与雷鸣的赌局大致描述了一下，梁姐问："你们验过牌没得？"

　　"验过。"

　　"那他有啥子多余的动作？"

　　"没有发觉。只是一点比较奇怪，他边打牌边一个劲地喝茶。"

　　"喝茶？……"梁姐在电话里那头思忖着，"那你把那副扑克找来再用化学眼镜检验一下。"

　　我的脑子"嗡"地一响，糟了，是不是牌上有问题？我们像离弦的箭一样，三步并作两步朝清和茶楼奔去。我们找到发牌的那个胖服务员一问，她说："跟你们一起打牌的那个人把牌刚刚拿走。"

　　"好久的事情。"我焦急地问。

　　"大约半个小时前。"

　　也就是说，雷鸣将唯一能让我们抓住把柄的证据悄悄带走了，这更说明了这副扑克上有猫腻。我又挂通了梁姐的电话，把这边发生的事一摆，梁姐说："问题就出在那副牌上，开始验牌时的确很正常。但雷鸣的手指上涂了隐形药液，他在喝茶时偷偷将手指打湿，然后涂在扑克的背面，边打边涂，几个回合下来，扑克牌就都涂满了他点下的暗记，肉眼是看不到的，而他戴有隐形眼镜，你们的底牌他一清二楚，格老子，你们不输谁输？"

　　"这种千术叫啥子名字？"我问。

　　"'踏雪留痕'，是一个澳门的千王独创的。"

　　魏老八的脸变成了猪肝色，久闯赌坛的人被雷鸣彻底戏耍了。他咬牙切齿地说："跑的脱，马脑壳，挖地三尺，也要把他挖出来。"
