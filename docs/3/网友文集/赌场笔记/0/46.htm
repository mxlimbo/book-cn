第十五章

　　今夜的珠海竟然如此妩媚，灯火辉煌的海滨路宛如一条绵亘不断的火龙，把珠海点缀得更加妖娆动人。三三两两的恋人躲在街心花园的灌木丛中卿卿我我地低声私语着，热烈地讨论着他们的爱情。

　　十点整，我们踏上了通往赌场的电梯。电梯门关严的一霎那，我有一种通向死亡的悲壮，这跟以往我奔赴赌局的感觉是完全不同的。我记忆中的赌局带给我的都是轻松、愉快，我面对的是一群不谙赌术的"兔儿"，他们不是把钱一个劲儿地往赌桌上押，就是缩手缩脚地瞻前顾后、痛失良机，他们不知道在大起大落、翻江倒海的同时，我那只罪恶的"千手"正灵巧地运送着纸牌，在不知不觉中，那只手如鱼得水地恣意戏谑着人们的脑筋，我享受着这种轻松赌局所带来的致命快感，在这种快感中，我幸福得快要晕眩过去。而这次所面对得雷鸣却是个长期在赌桌上混的高手，他谙谂牌中的一切，从他熟练的掌握"踏雪留痕"便可以管中窥豹，况且他的身后有鼎鼎大名的"福建帮"，弄不好就是一场血战，那就失去了我们来珠海的根本意义。

　　水仔从澳门的回力球赌场调来了几十个弟兄，随时准备血洗"福建帮"，彻底削弱他们在珠海的势力，他们多年的积怨有可能就在今晚爆发，我甚至已经闻到了污血的腥味。

　　刚一走出电梯，几个膀大腰圆的小伙子拦住了我们的去路，他们个个脸青面黑、凹眼凸颧，典型的福建人，其中一个矮挫挫的操着一口生硬的广东话拖声拖调地问道："乜事啊（有什么事）？"

　　他肯定把我们误认成了当地人，我这个假广东马上迎上前去，答道："我啲系雷仔嘅朋友（我们是雷仔的朋友）。"反正都不是真资格的广东人，叮呤哐啷地操起半生不熟的广东话，脸都不红一下。哥俩比□□——一个毬样。他们照例搜了搜我们几个的身，唯独刘琳不好搜，只好放我们进去。而恰恰在刘琳的腰间插着两把上满子弹的Spec Ops·45，刘琳相当暴露的打扮，让觊觎她的男人们将她忽略了。所以说，女人是男人的天敌，一点都没错。　　　　　　　　　

　　一跨进赌场，一番喧嚣嘈杂、生意兴隆的场面迎面扑来，赌客们都把一生的注意力全聚焦在千变万化的牌桌上了，他们的眼睛快要滴出血来，谁也没有注意到我们这几个远方来的、眼睛同样要滴出血来的不速之客。倘若这时有人大喊："台风来了！！！"绝对没有一个人理你。按四川话说，就是："张你捞球！"除非你怪声怪气地喊："楼下有人裸奔！"看吼得动他们不？这时，一个"马仔"模样的小伙子迎上前来，满脸堆笑地问道："各位老板，发财！发财！"遂既压低声调说："他们在贵客厅。"

　　看来他肯定是水仔的人了。我们顺着他的目光看到了贵客厅的大门，看来必须立即行动！免得夜长梦多，我们朝贵客厅疾步冲了过去。

　　贵客厅门口站岗的小伙子刚想拦住我们，我已经抢先推开了大门，我一眼就看见了雷鸣和一个五十几岁的、白发苍苍的老人对坐在一张豪华的赌桌前。而"一指禅"则站立在桌子中央，正低头认认真真地洗牌。赌桌后面相隔三四米的地方则站立着几个保镖模样的人。与此同时，雷鸣也看见了我们，他惊讶地张大了嘴巴，脸上的一块肌肉轻微地抽搐着，他压跟儿就没有想到我们能来珠海找他，而且还能找到他。他知道他在M市赢的那十万块钱有点烫手了，他的"踏雪留痕"难道留下了蛛丝马迹？魏老八难道已经抓住了他的"漏眼儿"？此时他甚至怀疑到了"气胸"，那个见财忘义的隆昌小人。但"气胸"懂个铲铲？他知道什么"踏雪留痕"？他只晓得站在边边上数票儿，坐在饭桌前拈闪闪。　　　　　　　　　　　　　　　

　　站岗的那个小伙儿也跟着冲了进来，他结结巴巴、神色惊慌地问雷鸣："雷……雷……哥，伲度……"

　　"冇事。"雷鸣故作镇静地向他摆了摆手，示意让他退下。

　　那个老者气宇轩昂，清癯美髯，风度不凡。他肯定意识到雷鸣有麻烦了，但他仍一丝不动、稳如泰山，而且瘦削的脸上还洋溢着并不在意的微笑。正如水仔所介绍的那样，再大的的风雨他都见过，再高的风浪他都遇过。眼前的一切在他看来简直就是和风习习、溪水潺潺，小菜都没有一碟。魏老八走上前去，毕恭毕敬地对老者说："实在不好意思，打扰您赌牌了。"

　　老者答道："没有关系。你找雷鸣有事吗？"

　　老者的嗓音特别好听，像远山孤庙里的钟声，深邃而悠扬。

　　"我跟雷鸣有一场私人赌局还没有结束。"魏老八边答边直盯着雷鸣，盯得雷鸣像全身长满了追魂大疮，从头到脚地不自在。

　　"好啊！你们可以继续，我无所谓，"老者非常大度地说，"我喜欢赌，更喜欢看人家赌。常言说，陪嫖看赌，人生最大痛苦，我不但不痛苦，还有些爽嘞！"

　　"这样更好，"魏老八说"我不会耽误你很久，我只需要一个小时。"说完转身对雷鸣说："你不会不敢应战吧？"

　　这真有点洗雷鸣的脑壳。赌界的人最忌讳仓促出战，来者不善，善者不来。从天远地远的M市到珠海来找你，这本身就不寻常，里面一定蕴藏着万般杀机。出千的人赌"奸"不赌"运"，他不可能霸王硬上弓。如果不应战吧，又害怕那个澳门老头耻笑，传到江湖上要遗臭万年。雷鸣的老大不在珠海，不过这倒不成什么问题，全赌场都是自己的弟兄，未必虚你几爷子？可雷鸣是全靠自己的赌技才操到今天这一步，如果因为自己的私怨砸了这个场子，雷鸣实在担当不起。他沉思了一会儿，出言谨慎地问魏老八："你准备啷个赌？"

　　魏老八已经把雷鸣逼到了死角里，垂死挣扎也好，负隅顽抗也好，估到你赌。他慢条斯理地说："我觉得在各种赌法里，'百家乐'是最为公正的，今天看看我们哪个挂红？"

　　雷鸣摇摇头说："'百家乐'是比较公正，但太过于简单了。两三张牌比大小，等于猜谜子，那不如我们两个'试圈儿'，石头、剪刀、布更简单，谁赢了谁把钱拿走。"

　　没想到雷鸣有这么多的过场，魏老八问："那你说啷个赌？"

　　"我认为'梭哈'比较能表现个人的牌技，我们不如比一比各人的拿手好戏。"

　　"我答应你。拿副新扑克出来吧！"

　　"慢着！"雷鸣阴阴地笑着，转身对那位澳门老者说："能麻烦你帮我们发一下牌吗？"然后又问魏老八说："你同意吗？"

　　这真是我们意料不到的事，摆着一个现成的发牌员不用，而去麻烦一个事不相干的老人。难道雷鸣已经发觉"一指禅"是我们的卧底？我浑身的鸡皮籽籽都冒起来了，这完全打乱了我们精心策划的布局。看来雷鸣真是狗□□抹香油——又尖又滑。而我们就象骑上了一只吊睛大虫的脊背，上去容易下来难。事到如今只好硬着头皮往上撞，魏老八的脸变得铁青，他的嘴唇已经乌黑了。

　　雷鸣说："如果你同意，我先上一趟洗手间。"

　　我知道他要偷偷往手指上涂抹隐形药液，准备施展他的"踏雪留痕"了。
