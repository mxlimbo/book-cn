第三章

　　听了如此沉甸甸的故事，我的心脏根本负载不起，我的额头也渗出了密匝匝的汗珠。我气喘吁吁地问刘琳："故事讲完了？"

　　"讲完了。"刘琳答道。

　　我长长地吁出一口闷气，我不想再听这种悲惨的故事。人的一生，如草木一秋，转眼成云烟，如果被这种人间惨事压在心头，人生更彻底失去了意义。我尽快地拂去心中的郁闷，脸上使劲挤出幸福的微笑。刘琳惊异地盯着我不自然的表情，问："你是在哭，还是在笑？"

　　我尴尬地笑着，问："后来你妈怎样了？"话一出口，连我自己都万分惊讶。我知道我的心里还在惦记着这个故事的结局。

　　"病死在狱中了。"刘琳小声答道。

　　这也许是刘琳心中永远的痛，我偏偏残忍地揭开它，我为我的愚蠢后悔不迭。我命令刘琳："赶快说点高兴的事！"

　　刘琳说："这应该是你的本事，我听我姐说，你非常幽默，也非常擅长讲笑话，我命令你马上摆一个。"

　　我实在佩服这姊妹俩的韧性，小小的年纪遭受了那么多不幸，仍能很乐观地面对着个世界。不过，此时此刻讲笑话，是不是更加残忍？但现在我必须这样在做，为了稀释刚才那个"家史"所带来的沉重。我的脑子飞转着，立即想起一个："有一个农村青年，准备到广东去打工，他爹依依不舍地把他送到村口，语重心长地叮嘱道：'儿啊，到了那边千万别跟那种女人干那种事，你一干那种事就会染上那种病，你一染上那种病，你媳妇就会染上那种病，你媳妇一染上那种病，你爹俺就要染上那种病，你爹一染上那种病，你娘就会染上那种病，你娘一染上那种病，村长就会染上那种病，村长一染上那种病，全村就会染上那种病哩！'"

　　刘琳早就笑得上起不接下气，而我却像个"冷面杀星"一点都笑不出来，我残忍地剖开了她的伤口，然后为了她一口脆皮巧克力冰淇淋……

　　一抹血红的夕阳挂在西边的天空，傍晚时分，我们到达了M市。

　　魏老八一看到刘琳，先是一愣，然后把我拖到一边，问："啷个把她带来了？"

　　我知道他一定把她错认成刘萍了，我说："你狭细盯一下，那个不是刘萍，是她妹儿。"

　　魏老八眨了眨眼，笑着说："吔？你龟儿子在吃串串香嗦？"

　　我忙不迭地摆着脑壳："千万不要乱开腔，算起来她应该是我的小姨妹。"

　　魏老八忙正色到："现在谈点正事。有个外省人，听说是哈尔滨的，在M市夸下海口，说他的'扯马古'，杀遍天下无敌手。有几个不信那个教的去碰了一下，全部洗白。这两天M市都闹昂了。我本来也不想管，但人在江湖，身不由己。第一，我不想让外边的人耻笑M市。第二，我们两个的存货也不多了，包包头是干的，心头就是慌的。没有钱，啥子事都是空了吹。"

　　"他用的啥子手法？"我问。

　　"我也没来得及去看，就等你回来才好办事。"

　　一听说又要上赌桌，我就像是注射了一针兴奋剂，雅安之行埋在我心头的阴霾顿时烟消云散。我们胡乱找了个馆子，三下五除二喂饱了肚皮。我在M市宾馆给刘琳订了一个房间，催她回去休息，但她高矮扭到我不放，非要跟我去开开眼界。无奈之下，只有带着这个"跟屁虫"。魏老八也打电话叫来"疯子"，以防不测。

　　夜幕早早垂下了。在"海韵茶坊"的包间里。我们和那个哈尔滨人面对面地坐在了一起。

　　我猜想哈尔滨人不会是单打独斗，果然不出我之所料，他身边足有5个保镖。其中一个一看见魏老八就惊叫起来："八哥，原来是你嗦？"

　　魏老八一愣神，马上不卑不亢地回敬了一句："原来是'气胸'嗦？"

　　被称为"气胸"的人马上把烟散过来，皮笑肉不笑地说："请八哥多多关照！"

　　"气胸"长得又干又瘦，头发像稻草一样乱糟糟的，眼睛滴点小，眼皮神经质地一眨一眨的，像是对谁使着眼色。他的左手臂纹了一个"恨"字，右手臂却纹了一个"爱"字，代表着两个截然相反、内容炯异的故事。这两段故事让他如此刻骨铭心，以至于让它们变成一辈子的印迹，时刻提醒着"气胸"铭记心头。

　　魏老八告诉我，"气胸"就是他在劳改队里为了花脸与他打过架的"隆昌帮"的一员干将。听了这番介绍，我万分惊讶，难道劳改队里结的"梁子"回到社会上就若无其事地化为乌有了？

　　魏老八对"气胸"说："过去的是就让它过去吧！毕竟我们是同一战壕的，现在大家都出来了，俗话说，不打不成交。我们今后就是朋友，多个朋友多条路嘛。"

　　魏老八这几句话不温不火，等于给"气胸"下了矮桩，没想到"气胸"脸色倏地一变，冷笑一声道："在你的地盘我咋个敢惹你？我这次不是来扯筋的，我纯粹是以赌会友。"

　　魏老八一听这话，脸色立即阴了下来，说："既然以赌会友，我的原则是：朋友之间不犯杀。所以我不赌。"说罢便站了起来。

　　"气胸"把上半身探过来，双手撑着桌面，问了一句："我们真的是朋友吗？"

　　魏老八一听这话，知道"气胸"来者不善，本来站起来的身体又重新落了座。他斜了"气胸"两眼，语气很重地说："那干脆我们两个打一架，这样比较撇脱（干脆）些，免得你今后鸡叫鹅叫，还赌个卵牌？"

　　"气胸"说："我不是虚火你，现在又不是动不动就甩锭子的年代。我早就听说你号称'M市赌王'，你有没有屁眼跟我们赌？"

　　魏老八顿时怒发冲冠，一拳砸在茶几上，震得几个茶杯浑身乱颤。他强压着嗓音问："咋个赌？"

　　"长庄马古。也就是我们的人长期坐庄，你可以洗牌、切牌，然后由庄家发出4铺牌，端点子要大小，按东西南北顺序拿牌。拿牌前先下注，'马古'庄家不赔双。不知八哥敢不敢赌？"

　　魏老八悄悄告诉我，这是流行于隆昌一带的"霸王庄马古"，用我的"空手道"足以对付。

　　"你们派哪个出战？"魏老八问。

　　"气胸"回头一使眼色，那个一直闷着脑壳的哈尔滨人慢悠悠地站了起来。
