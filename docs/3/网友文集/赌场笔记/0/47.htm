第十六章

　　雷鸣不知道我们已经看穿了他的"踏雪留痕"，他仍在准备出千前的一切技术细节。我们则像守候多时的猎人，蹲在一棵歪斜的老泡桐树下，嚼着隔夜的干粮，忍耐着蚊叮虫咬，专等他登台表演。一旦他的"踏雪留痕"露出一丁点儿蛛丝马迹，我们立刻将拉满弓的弩箭向他的心脏射去，箭头上涂满了我们在森林里熬制的"追魂蛊惑透视眼"，足以将他麻翻在地，让他在腐败糜烂的枯叶上抽搐痉挛，口喷五尺长的涎液，最后长啸一声，悲惨地与潮湿的泥土融为一体，变成一堆没有标记的坟茔。山顶上的秃鹫笑了，它准备在我们走后开始饕餮这顿丰富多彩的大餐……

　　其实这只是雷鸣进洗手间的时候，浮现在我脑海里的恶毒念头。虽然这种念头有点小气，但小气专门对付小人，小气得让小人都要怒发冲冠。

　　看来我们的"赌后"今晚根本派不上用场，缺少了"一指禅"，刘琳想亲手把那十万元赢回来的夙愿，看来肯定泡汤了。不过幸亏她不光是一个看客，她把我们的武器顺利地带了进来。现在看来，这一点是顶顶重要的，种种迹象表明，今夜的赌局不会简简单单。从雷鸣弃用"一指禅"就能够嗅到今夜非同一般的味道，那味道如同你身边刚刚驶过一辆装满氨水的农用汽车，催人泪下，被迫让你感动不已。

　　雷鸣重新回到了赌桌旁，他信心十足、容光焕发的样子与刚才的惊慌沮丧判若两人，看来一切技术工作他都准备得很充分。

　　晚上十一点半，赌局正式开始。

　　这是一场闻不到硝烟的战斗，没有枪炮声，没有匍匐前进，没有铁丝网前的闪躲腾挪，但其紧张惊险的程度一点也不亚于一场大型遭遇战。交战的双方小心翼翼地撕开了纸牌的塑料包装，像正要剪断一颗新型子母雷的保险，使劲屏住呼吸，尽量别让手指有一丝的颤抖。他们各自戴上红外线透视镜，开始仔细地检查纸牌的正反面。我忽然想恶意地笑出声来，因为魏老八和雷鸣的样子简直就是一幅活生生的《老猴翻虱图》，那么的一丝不苟，那么的专心致志，那么的兴趣盎然。

　　澳门老人开始洗牌，他洗牌的手法相当熟练，搓、插、抽、叠，令人眼花缭乱。扑克牌在他灵巧的手上听话得像美丽的花蝴蝶一样上下翻飞，翩翩起舞，我禁不住惊呼：此洗牌法非一日之功也！我琢磨这个老头儿恐怕在他吃奶的时候就开始练此硬功了。他不厌其烦地起码洗了八百遍，颇有点卖弄的意思，而且在开始发牌时，他还意犹未尽地咂着嘴巴，得意洋洋地瞟了魏老八一眼，他多想把这副牌一直洗到东方的天空出现鱼肚白啊！那才叫过瘾。不过发牌比洗牌似乎更具吸引力，因为他发出的每一张牌都紧系着魏老八和雷鸣的命运。别人的命运完全掌握在自己的手里，他摇身一变，变成了主宰尘世命运的上帝。如果不是他的身份、地位、年龄，"一指禅"早该下课。

　　发给魏老八的底牌是一张梅花Ｊ，跟着是一张黑桃８，该魏老八说话。他试探性地往牌桌上甩过去五千元，没想到雷鸣根本不跟。雷鸣的右手边放着一杯刚沏好的热茶，那是他的"踏雪留痕"的辅助工具，喝茶时用以偷偷打湿他那根涂了隐形药液的手指。如果他施展了蜻蜓点水似的绝技，我戴在眼中的隐形眼镜立即可以识别出来，我可以在牌桌上当场让他的千术穿梆。按照赌规，发生这种情况，亮在桌面上的钱全归我们所有，那少说也有几十万啊！钱，我们有了；"踏雪留痕"，我们给他破了；在精神、技术、金钱上，我们全面击溃了雷鸣，一棍子把他打闷，让他几辈子缓不过一口气。

　　但非常奇怪的是，牌都已经赌了好几个回合了，雷鸣尿都没尿那个茶杯一眼，仔细算下来，他还略有点盈利。难道他认为根本不用"踏雪留痕"就可以彻底击溃我们？他有这个把握？把握来自哪里？那一定有更强有力的技术在后面支撑着，不然的话，以雷鸣的奸诈，他不可能打这种纯粹撞大运的"硬牌"，而且这儿还是他的地盘。

　　更强有力的技术？我暗暗忖度着，开始向四周观察。

　　这间贵客豪赌厅足有一百平米，装潢得典雅大气，华丽的巨型吊灯，深红色的天鹅绒窗布，沉坠而庄严，四周的墙壁上挂着几幅叫不出名的人物油画，把这间庞大的豪赌厅装扮得如同古罗马教皇的秘密宫殿。在那花花绿绿的墙角旮旯儿里是否暗藏着几只潜望镜？镜头后面的眼睛是否可以清清楚楚地看见魏老八手上的牌，然后再通过特殊发射器传到雷鸣的耳朵里？我开始盯住靠得我们最近的这幅画，画上的这个外国老儿正瞪着一双翡翠绿的猫眼聚精会神地望着我们，两撇翘起的八字胡仿佛还忽闪忽闪的。我站在他和魏老八之间，挡住了他的视线，而且还虎视眈眈地与他对峙着。但此时意外发生了，我的隐形眼镜马上就要掉了出来了，我必须马上把它塞进去，而且绝对不能当着雷鸣的面，否则我们的狩猎计划全部泡汤。我立即朝门外走去，想找一间洗手间解决问题。

　　所幸的是，洗手间里空无一人，我想刚把隐形眼镜放进扒开的眼皮里，门外嘻嘻哈哈走进来两个人，我急忙一转身躲进了身后的厕室，轻轻地将门虚掩上。进来的是两个福建仔，唱歌似的用闽南语激烈地争论着，好像在比试谁把尿屙得更高些，高得是否能打断高压线。这两人的尿特别骚，绝对是长期熬夜火重肾虚的结果，而且哩哩拉拉地放长线，紧到屙不完。就在我忍无可忍掩鼻冲出去的时候，又一个憋尿的人走了进来，而且操着一口怪眉日眼的普通话。在这种场合，不是闽南语就是粤语，怎么能冒出来个操普通话的？我稳着没动，仔细听了一下，这人的普通话还是四川人说的椒盐，这可真让我纳闷儿了。难道"福建帮"里除了雷鸣，还有四川人？我顺着虚掩着的厕门向外望去，不禁大吃一惊，我差点儿失声叫了起来。

　　他是隆昌的"气胸"！！！

　　我的脑子飞速地转动着，我不敢肯定"气胸"不现身的具体原因，但我总觉得我们一定陷入了一个不知名的陷阱，它像一个阴险的风口，呼啸着将我们吸吮了进去，我们好像根本没有转身的余地，没有呼喊救命的能力，因为口中灌满了噎人的黄沙。正在这时，我腰间的传呼猛烈地震动起来，我感到噩梦已经降临了。传呼上是一行叫我心惊胆跳的字句："赶快退出！我们在门口接应。'一指禅'已经出卖你们！老头不是澳门大亨。水仔。"

　　我三步并作两步地向贵客厅冲去，撞开那扇大门一看，一幅更加让我吃惊的场面呈现在我面前：刘琳平端着那把Spec Ops・45，枪管紧抵着雷鸣的脑袋，而魏老八的枪则顶在那个老头的裆部。魏老八压低声音，严厉地向雷鸣命令道："送我们出去！"

　　我和疯子迅速地将赌桌上的钱一古脑儿地全部装进了我们带来的两个大黑提包里，在众目睽睽之下，押着雷鸣和那个老头一步一步朝电梯移去。无数个枪管瞄准了我们，谁也不敢轻易扣动扳机，这个时候谁要是不小心放一个坐地响屁，谁的命就得搭进去。电梯一到底楼，我们的人黑压压的一片，早已伸出了无数枝大小不一的枪筒接应我们，我们坐进水仔的汽车，风驰电掣般地驶离了那家酒店……
