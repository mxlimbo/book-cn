第十一章

　　胡疤像刚刚清除了身上的一个恶瘤一样，兴高采烈地走上楼来。他把一个用旧报纸包的东西放在了魏老八面前，说："这次多亏你们帮忙，那两个杂皮还想在我这里臊堂子，他们不晓得我的东西有好烫。这里是10万元，一点小意思，就算我老兄的一点谢意。"

　　魏老八摆摆手说："是朋友我才帮你这个忙，既然是帮忙，我就不会收你的钱。你也不要多心，以后兴许还有我求你帮忙的地方。"

　　胡疤说："有啥子问题，只管开声腔就是了。"

　　魏老八问："那两个人你准备咋个处理?"

　　"先让他们家里人拿50万元来取人，然后撕票，丢在湖里喂鱼。"胡疤咬牙切齿地答道。

　　"这样吧!"魏老八向前欠了欠腰，说："既然你老兄想感谢我，我有一个小小的要求，不知能不能得到满足?"

　　"尽管说!"胡疤催促道。

　　"我想带走那个发牌员，由我来处理他。另外那个随便你怎么处理，你看如何?"

　　"这个……"胡疤迟疑着。

　　"如果不方便……"

　　"不不不!没有关系!"胡疤连声说道，"我答应你。"

　　魏老八笑了，说："麻烦胡老兄给我们准备一艘快艇。"

　　天色已渐渐发亮了，湖面上罩着一层白白的雾气。摩托快艇如离弦的箭一样，轰鸣着向湖中射去。清晨的冷风迎面扑来，吹醒了我们一夜未眠、疲惫不堪的脸。发牌员被双手反铐着，兀自缩在角落里瑟瑟发抖，寒风吹散了他的头发，衣领也被吹得倒竖起来，正好遮住了那张恐惧失望的面孔。

　　上了岸后，魏老八打开了发牌员的手铐，问他："你叫什么名字?哪里人?在哪里学的千术?"

　　发牌员只是惊恐地摇着头，一言不发。

　　魏老八说："老弟，我并不想砸你的饭碗，因为我们都是吃这碗饭的。再说练成你今天这步，也确实不容易，我很欣赏你。"

　　发牌员慢慢地抬起了头，他仿佛看到了一根救命的稻草。

　　魏老八接着说："但是为了朋友，我必须拆穿你，因为我没有选择。现在趁着天还没大亮，你赶快跑路吧!胡疤肯定知道我会放你，天亮以后，火车站、汽车站全是他的人，到时候你就没命了。记住，以后千万别在J市出现，胡疤是这里的地头蛇，他不会轻饶你的。因为你打了他的'倒挂'。"魏老八说完从包里摸出1000元钱塞给了发牌员，并催促他道："闲话少说，快跑吧!"

　　发牌员拿着钱，向魏老八鞠了一躬，转身飞也似地跑了。

　　在回M市的车上，我问魏老八，为什么放了他?

　　魏老八说："得饶人处且饶人。况且他和我们都是干这行的，也算是一种人才。说不定他与你是师从同门呢!我觉得他好像有什么难言之隐，说不定那个假成都赌客知道他的底细，他是迫于无奈才打了胡疤的'倒挂'。唉!设身处地地想一想，如果换成你，你该怎么办?"

　　魏老八是讲义气的，从他这番话就可以窥见他的处世哲学。人人都喜欢金钱，但厚厚的金钱摆在你面前，只为了"义"字而不动心的，在这个"一切向钱看"的社会里太少见了。魏老八并不是不稀罕金钱，该是他拿的钱，少了他一分钱他都会跟你拼命。但朋友的情谊他始终把它放在第一位，他说，那种情是根本无法用金钱买到的。赌博赌成了视金钱为草芥，魏老八是第一人。这时我才知道，为什么他能在赌桌上镇定自若，胜不骄，败不馁，从来都是淡淡一笑，泰然处之，那时的金钱在他眼里真正成了不值一文的草纸。能做到这一点的人太少太少了。赢钱后的魏老八可以到成都太平洋百货买6000多元的杜彭打火机，8000多元的杜彭皮带；输钱后的魏老八可以一个人蹲在厕所撕扯着自己的头发，痛哭流涕，但是在赌桌上你看到的永远是微笑的魏老八，他能暂时把喜和悲统统埋在心底。在赌桌上，我见过有的人暂时赢了10多万，就又挽袖口又松领带，鼻孔冲着天，连燃着的烟卷也垂直起来，一副洋洋得意地样子，嘴里大声说着："照这样赢下去，明天我就去买一辆桑塔纳2000"。我也看到有的人刚输了一点钱就惊风火扯地怨这怨那，嘴里日妈捣娘不干不净地骂个不停，结果越扳越输，死得梆硬。赌桌上的人生百态，犹如一部社会大学的教科书，包罗万家，永远没有结尾。

　　日子就这样平平淡淡地过着，而在这平淡中我却忙碌着。我和魏老八走南闯北，继续我们的赌博生涯，我的"空手道"已经到了出神入化的地步，手到牌到，一把一把的金钱揣进了我们的腰包，我们的积蓄迅速增长着，甚至是迅速膨胀着。

　　这些日子，我的身份是变化无常的。为了不引起对方的怀疑，我有时装扮成事业有成的大老板，装运河沙的土老板，卖假烟的小老板；有时装扮成城府颇深的工程师，吃了家长钱的中学教员，为了适应这些角色，我购置了许多衣服――我们称之为"工作服"，我的大衣柜里琳琅满目，让你目不暇接。为了到偏远的农村去杀那些横霸乡里的暴发户，我甚至准备了一双红军穿过的草鞋，装扮成身上揣了点钱的卖货郎。在城市与乡村的每一个角落里，我和魏老八像两个幽灵一样到处游荡着。

　　魏老八说："我们再干两年就彻底收手，然后办个实体，做点正当的生意，一是洗钱，二是在社会上也有一个身份。其实我已经开始厌倦这种生活了。"

　　也许这就是我俩共同追求的目标吧!

　　但是平淡的生活并不平静，因为我已经深深地陷进了黑色的漩涡。

　　这天中午，我的手机骤然铃声大作，我一看来电显示，是个很陌生的但又有点眼熟的区号。一种不祥的预兆立即笼罩着我，我拿起了电话："喂!"

　　一个遥远的、空旷的、恐怖的女人的声音从电话里传来："快来救我!我是刘萍。"

　　电话"嗒"地一声挂断了。我呆在那里，半晌说不出话来。我开始以为谁在开玩笑，但翻开电话簿一查，那个电话就是从雅安打来的，只不过是6字头的，大概是雅安的郊县。我马上找到魏老八，将这些情况一说，魏老八的脸色立刻变得铁青，我第一次看到他的脸色变得这么快，颇有点骇人。他立即给雅安的花脸打电话，但对方一直占线。我们开始焦灼不安起来，不知道雅安那边到底发生了什么。

　　魏老八从腰间拿出一把六四式手枪，递给我说："揣在身上，以防万一。"他又回家拿了两杆来福枪，藏在汽车后座下面。然后我们直奔银行，取了50万元现金。我问："带那么多钱干什么?"

　　"如果非要用钱赎人，难道我们见死不救?"

　　疯子接到魏老八的电话，也立即带了两个弟兄赶到了。魏老八问："家伙带齐没有?

　　疯子一撩上衣，露出腰间卡的七八枚手榴弹。魏老八一挥手："赶快出发!"

　　我们5人立即钻进汽车，由疯子驾驶，飞也似地冲上了高速公路。魏老八又开始拨花脸的手机，这一次终于拨通了："喂?花脸吗?啥子?死了?啊!刘萍刚才打来电话，是6字头的，是芦山嗦?你们先别要动，我们马上赶到!"

　　魏老八挂上电话，转身对我说："花脸被格桑杀了，现在在医院，刚刚死，刘萍也被格桑绑架了，花脸的二三十个弟兄到处在找。看来，格桑是逼我们现身啊!"

　　我意识到了问题非常非常的严重，花脸惨遭杀害，刘萍又下落不明。

　　我们风尘仆仆地一到雅安，就与花脸的弟兄们会合了。二话不说，我们又分乘8辆出租车浩浩荡荡朝芦山县驶去。

　　沿着川藏公路，一到飞仙关立即向右拐，一条宽阔平坦的水泥公路展现在我们面前。离芦山越来越近了，魏老八的脸越来越难看，我问他："不舒服?还是为花脸的死?"

　　魏老八摇了摇头，沉吟了许久，才吐出一句："我以前就在这里劳改。"

　　我浑身不禁打了一个寒噤。望着身后长长的车队，再看看前方空旷的公路，我们是不是正在驶向监狱的大门呢?
