第十一章

　　珠海国际机场。

　　在夜幕的掩护下，我们一行四人――魏老八、疯子、刘琳和我――悄然步下飞机舷梯，我们每人脸上的神色凝重得快要挤出水来。

　　　　十万元！我们被雷鸣用＂踏雪留痕＂赢走整整十万元，而且这十万元每天还有五千元的利息。其实这比抢我们还让我们恼火，因为我们也是出老千的，而且百战百胜，从未失蹄。如今却被那个忘恩负义的小人把我们戏耍了，这说明我们的智慧，技术在某一瞬间被雷鸣玩于股掌之间了，这真是我们的奇耻大辱。此时的雷鸣一定躲在一个旮旯儿里得意忘形地狂笑不已，笑得上气不接下气，大小便失禁。

　　 临上飞机的时候，我压根就不想带刘琳一起走。她哭丧着脸，嘴唇翘起老高，一直跟在我的屁股后面。我对她说："又不是去游山玩水，欣赏祖国的大好河山，而且还有一定的危险性，万一出点拐，我啷个向你姐姐交代？"

　　她胀红着脸，眼里含着泪水，说："钱是我输的，我一定要把它赢转来，再说我多多少少会一点武功，我也可以像疯子那样给你们当保镖噻。"

　　她不提武功还好，一提起她那点三脚猫功夫，我们都笑了起来。我问她："你会不会'猴子扳疙蔸'？"

　　话还未落音，只见刘琳鬼影一闪，"唰"地已蹿到了我的背后，随后我的身子早已腾到了半空中，我"妈"一声重重地摔在地上，脑子里"嗡"地一下，只听见魏老八好似在遥远的地方边笑边喊："把她带到！把她带到！！她有点像杨紫琼！"

　　我艰难地从地上爬了起来，装成若无其事的样子问道："你这一招叫啥子名字？"

　　刘琳满不在乎地答道："老头钻被窝。"

　　我们从机场大厅一走出来，就看见两个衣着光鲜的小伙子笑容可掬地向魏老八招着手。魏老八小声对我说："是我去年在广州混的时候结识的朋友，'潮洲帮'的。"

　　我们彼此寒喧了一番，然后迅速钻进两辆黑色的轿车，风驰电掣般地朝市区驶去。车上，魏老八对开车的小伙说："马上通知'水仔'，叫他立刻来见我，我有重要的事情需要和他商量。"

　　我们被安排在珠海最豪华的银都酒店，但此时的我们哪有心情去欣赏窗外美丽的夜景，魏老八说："但愿我们没有枉自跑一趟，但愿那个狗日的雷鸣回到了珠海。"

　　约摸过了一个小时，门铃响了，魏老八拉开门一看，正是他的朋友水仔，跟在水仔身后的还有四五个脸青面黑的人。水仔身材不高，但相当J实，而且眼睛黑而透亮，黝黑的皮肤像黑豹一样紧扎而光滑。大家落座后，水仔笑眯眯地对魏老八说："我知道你是无事不登三宝殿，有什么需要兄弟帮忙的尽管开口，我一定全力以赴。"

　　魏老八说："我需要你在珠海帮我搜一个人，据说是在澳门、珠海这一带的赌界上混。还有，你马上帮我找三把火力强点的家伙，揣在身上心里头要踏实些。"

　　水仔听到这里，脸色顿时严峻起来，他说："找一个人倒是容易，只是不知道他是在哪条道上混的。珠海这里有很多地下赌场，大部分控制在'潮洲帮'，'福建帮'，还有就是对面'十四K'的，如果他真在赌场上混，就看他是在哪条'盘口'，不过我相信，在珠海找一个人是再轻易不过了。他叫什么名字？"

　　"雷鸣。"

　　接下来的几天真是度日如年，这仿佛是我一生中最最难熬的时光，我无暇带着刘琳到景色迷人的迎宾大道上哪怕逛上一圈，整日窝在宾馆里心情烦闷的盯着电视机的荧光屏。魏老八也是一根香烟接着一根香烟地抽着，不愿意多说一句话。这可能是他赌博生涯中输得最惨的一次，他宁肯输给一个叫化子，也不愿输给雷鸣，因为对手是他的不共戴天的仇人。雷鸣像一条饥饿的蚂蝗，吸光了魏老八血管里的鲜血，同时也舔干了他的精髓。魏老八就如同一个干枯的木乃伊，静悄悄地倒卧在雷鸣为他挖掘的墓穴里。他仿佛听到雷鸣正趴在墓穴的入口处，张狂而得意地狞笑着，他知道雷鸣准备封堵坟墓了，并且为他树起一块歪斜的墓碑。

　　刘琳也乖了许多，不再像以前那样咋咋哇哇、飞叉叉的。她好像突然长大了许多，俨然一个文文静静的大姑娘，她像做错了事似的，睁着迷惘的大眼，久久地盯着窗外，没有了肮脏的'社会'语言，没有了无邪的笑靥。我试图给她讲个笑话，让她重返无忧无虑的过去，她摆摆手，拒绝了。

　　魏老八最终打破了这种令人窒息的气氛，他大声武气地说："其实我们才输了好多钱？才十万块！我们这一辈子有多少个十万块在等着我们去赚啊！没得好大个男女关系。走！！今天晚上我们找个酒廊去'蹦迪'，顺便看薅得到一个婆娘不?"

　　晚上我们刚想出门，水仔就风风火火地闯了进来，魏老八问："有消息了？"

　　水仔大口地往嘴里灌了一罐'可口可乐'，说："人已经找到了，就在珠海。他在水湾头那边一个大酒店里开了一家赌场，不过那家赌场是属于'福建帮'老大的，这说明雷鸣是'福建帮'罩着的。"

　　一听到这个消息，魏老八顿时放下心来，他握着水仔的手说："谢谢你了，兄弟！我知道你们和'福建帮'的水火不相容，我不想让你们因为我的事又像去年子那样杀得天昏地暗。"

　　水仔一听，说："八哥在说客气话了，去年我们同生死共患难的日子，你难道忘记了？难道你把我当作外人了？闲话就少说了，你八哥的事就是我的事，只要你一句话，是要他的胳膊还是要他的大腿？"

　　魏老八说："哪里需要那么火爆？我只想和他再赌一把，彻底消灭他的'踏雪留痕'，人活在世上，不蒸馒头也要蒸口气啊！"

　　水仔说："如果冲进他的赌场和他硬赌，他肯定找理由推拖。不如先偷偷摸摸到那家酒店观察观察地形，然后再想办法逼他赌。"

　　我对魏老八说："干脆我跟水哥他们走一趟，我认识雷鸣，而他对我却不是很熟。"

　　水哥说："还是算了吧，万一被雷鸣认出来，就会打草惊蛇了。"

　　魏老八笑着说道："你放心吧，他经常化装成鬼眉日眼的样子，到处招摇撞骗，有时连我都没认出来，差点儿糊他龟儿一耳屎。"

　　一想到马上要见那个狗日的雷鸣，我的心竟然狂跳起来，我没有机会施展我的'空手道'，就莫名其妙地让他的'踏雪留痕'打得落花流水，这是我自跟梁姐学习千术以来的第一次失败，所以才让我羞愧难当，耿耿于怀。

　　晚上十二点，水哥底下的一个'马仔'把我带到了那座酒店的楼下，赌场在24层，我不由自主地仰面望了上去。

　　雷鸣！我到你的老窝来了！

　　一走出24层的电梯门口，我被两个小伙子拦住了去路。其中一个用蹩脚的广东话问我："h边个？"

　　我答道："唔h边个，我赌钱好唔好？"

　　他们用疑狐的眼神上下打量了我半天，然后仔细地将我的全身搜了个遍，最后还是将我放了进去。

　　赌场里人声鼎沸，大多数人都围在'百家乐'的赌桌旁。我大摇大摆地在人群中穿来穿去，反正没有一个人认识我。我大概瞅了一下，没有发现雷鸣。魏老八临走时反复交代，一定要速去速回，免得生出意外，但此时意外真的发生了，我发现了一张非常熟悉的面孔。认出他的一刹那，我的头发被惊得一根一根地竖了起来……
