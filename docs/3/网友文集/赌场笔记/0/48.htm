第十七章

　　在回银都酒店的路上，魏老八迫不及待地问水仔："到底是怎么回事？"

　　水仔答道："我们的人刚刚探到消息，'一指禅'把你们出卖了，于是雷鸣精心策划了这场赌局。他先让'一指禅'放出假信息，说他不在珠海，然后又安排了与那个假澳门大亨的赌局，诱你上钩。其实那个老头就是雷鸣和'一指禅'的千术老师，外号'瘦鬼'，是澳门的千王。由他发牌，你哪能不输？还有，你知道'一指禅'叫什么名字吗？"

　　"不清楚。"魏老八摇了摇头。

　　"雷笛。"

　　"雷笛？！"魏老八惊叫了一声。

　　"他和雷鸣是同母异父的兄弟。"

　　"真的？！"连我都大声惊呼了起来，世上的事竟有这般离奇！

　　"还有更让你们意想不到的，"水仔继续说道："你知道他们是怎么样的同母异父吗？"　

　　我们全体人猛摇头。

　　"'一指禅'的二爸实际上就是他的生身父亲。"

　　"哦！我晓得了。"魏老八似乎大彻大悟，"'一指禅'的二爸强奸了他嫂子，也就是'一指禅'的妈而生下的'一指禅'。"

　　大家都为魏老八的这番话而哄堂大笑。魏老八说："我还指望他作卧底，卧个铲铲？今天差点儿遭洗白，亏了我们还没收了他几十万元赌款，也不枉自跑一趟珠海，兄弟们有伙食开了。反正他不仁，我就可以不义。格老子哪天逮到'一指禅'要好生修理他一下。"

　　水仔说："还用等到哪天？'一指禅'早被我抓到了，不然的话，我为什么知道他叫雷笛，他妈偷他二爸？"

　　听到这里，魏老八缄默不语了，他眼里的凶光被街上的霓虹灯映得透亮，他的胸脯急速地起伏着。陷入雷鸣和"一指禅"的骗局，而且险些手爪爪都被烧卷，这是他绝对不能容忍的。他肯定有点儿恼羞成怒了。

　　一回到银都酒店，"一指禅"就被押了进来。看到他披头散发、魂飞魄散的样子，我不禁起了一丁点儿的隐恻之心，但瞬间就被刚才我们在赌场上那惊险的一幕冲得无影无踪了。他那苍白的面孔比以前更加惨不忍睹，甚至白中带绿，有点像鬼片中千年吸血僵尸的造型，而且能随时溶化成一滩又稠又粘的绿色液体，伸出无数只柔软的、涎答答的触角千方百计地想缠住你，让你毛骨悚然。他的衣领被扯得歪在一边，嘴角还有结痂的血迹，腿也一跛一跛的，显然他曾进行了最激烈的反抗，只因寡不敌众，才被生擒活捉。他那最无神的眼睛横撇撇地乜视着魏老八，目光中竟然射出一束骇人的凶光，这是以往的"一指禅"根本不存在的。他知道他已被逼到了一个悬崖的边上，没有了任何退路。身后是万丈深渊，身前是咄咄逼人的魏老八，他只有迎上前去作最后的垂死挣扎。

　　他颇不耐烦地问魏老八："要杀要砍？痛快点儿！"

　　魏老八一下子笑了起来："真是他妈的怪事，吃屎的把屙屎的估到了，我都没着急，你倒不耐烦了。"

　　"一指禅"说："我不想你又用啥子义气来软化我，因为那根本不起任何作用。"

　　这几句话一下子把魏老八惹毛了，这真是他妈的活生生的、如今现在耳目下的《农夫与蛇》。魏老八羞愧得恨不得铲自己几个最响亮的大耳屎，叫自己记住，这个世界上哪会有"义气"二字？"革命不是请客吃饭，不是做文章，不是绘花绣画，不能那样雅致，那样从容不迫，文质彬彬，那样温良恭俭让……"这是魏老八在读幼儿园时背诵过的语录，真是放之四海皆准的真理。魏老八哪里是一盏省油的灯？他最不能容忍的就是欺骗，因为那是对他人格的一种嘲弄。如果"一指禅"敞开天窗说亮话，告诉魏老八他与雷鸣的关系，魏老八绝对不会为难他，爹亲娘亲，不如五指连心。但"一指禅"没有这样做，他反向魏老八抛出了一颗又肥又大的诱饵，因为即使不为他同地不同天的哥哥，他也要为他的生身父亲报仇。而我们还一直蒙在鼓里，还得意洋洋地勇赴赌场，传出去一定笑掉万多人的大牙。魏老八咬牙切齿地说："好！我成全你！"说完向疯子一使眼色。

　　疯子知道魏老八眼中的含义，他从腰里抽出一把锃亮的、一面是锋刃一面是锯齿的真资格的瑞士军刀，几个人立刻扑上去按翻了"一指禅"，我知道马上要对"一指禅"用刑了，刘琳赶快把脸别了过去。随着疯子的军刀在空中划出一道孤寒的弧线，"一指禅"惨叫起来，像被拖往屠宰场的约克夏白皮猪，他右手的五根指拇被疯子齐斩斩地剁了下来……

　　这次刘琳的表现堪称优秀，她的胆略，她的不怯场给我们留下了非常深刻印象。而且正是由于她的身手敏捷，我们才得以安全地从赌场退出。魏老八说："回到四川后，干脆把她娶了，这个婆娘做老婆、做情人都非常地优秀，今后她绝对是你的一把好帮手。"我也这么想，应该彻底跟她摊牌了，只是不知道刘琳的心里是如何盘算的。

　　当晚，我热烈而温柔地吻了她，在我喃喃地在她的耳边稀里胡涂地堆砌着爱情造句时，刘萍没有不合时宜地打扰我们，她知道我和她妹妹的爱情篇章已经不可避免地翻开了最辉煌的一页。此时的刘琳早已是泪流满面，她紧紧地揽住我的脖子，傻傻地说："从看见你的第一眼起，我就认定你是我的男人。"

　　这话听起来有点耳熟，象是南茜在大学第一眼看见里根时说的。不过从刘琳的嘴里吐出来，我听起来非常地入耳。看来天下的女人，不分中外，不分贵贱，对男人的感觉全都一样。

　　第二天与水仔热烈告别，此次珠海之旅多亏了他的倾力相助。他对魏老八说："雷鸣绝对不会善罢甘休，他还会卷土重来的。'福建帮'也一定鼎力助他一臂之力，你们一定小心加小心！"

　　魏老八说："让他们到四川来找我！不过你们也要提防着点儿，如果知道是你们帮我，又是一场火拚。"

　　"我们时时刻刻都警惕着呢，没有你们这件事，我们也不会和平相处的。"

　　我们说说笑笑地走上电梯，簇簇拥拥地来到了大厅。刘琳一直紧挽着我，将头轻轻地靠在我的肩上，我的右臂碰到了她那丰满的、弹力十足的胸部，不禁让我想起了昨夜的缠绵悱恻，她全身湿漉漉的在我的怀中扭曲呻吟……我有些冲动了，想就在大厅里偷偷地吻她一下，当我转过头来看她时，正与她含情脉脉的双眸交织在一起。她似乎看出了我的企图，马上羞涩地推了我一下，就在这时，只听见水仔大叫了一声："爬下！"话音未落，"啪啪啪"几声枪响，大厅里顿时乱作一团。魏老八全身一震，一个倒栽葱斜着倒了下去，我急忙回头看刘琳，她的头已软软地垂在胸前，我一把抱住她，将她的脸捧起来一看，鲜血"汩汩"地从她的嘴里大口大口地涌了出来。她拚尽全身的力气，死命地抓住我的头发，一字一句地说："带……我……回……家……！！！"
