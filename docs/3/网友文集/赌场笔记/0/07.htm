第六章

　　真是天无绝人之路，晚上，我和魏老八在街边上擦皮鞋，坐在魏老八旁边的一个戴眼镜的小伙子也在擦皮鞋。魏老八无意中一转头，正和那个小伙子脸对着脸，四目一对光，魏老八"咦"地一声："哎!薛凯，你往哪里跑?"

　　那个叫薛凯的马上没有了刚才的神气，浑身哆嗦、结结巴巴地说："八哥，我……没……有跑。"

　　"老子找了你两年，你躲到哪里去了?"魏老八声音提高了一倍。

　　"我一直在郑州做生意。"

　　魏老八从上衣口袋的电话簿中取出一张皱巴巴的纸条，递到薛凯面前，说："你欠我的十万块钱呢?"

　　"我现在没有现金，要有也是在郑州……"薛凯脸上冒出冷汗来。

　　"管你啥子州?就是美国的加利福尼亚州我也要去拿!好久走?"

　　"我……今天……才回M市。"

　　"成都到郑州的火车几点钟开?"魏老八问。

　　"晚……上11点过……"薛凯战战兢兢地答道。

　　魏老八抬腕看了看手表，说："还来得急，你马上打个电话回家，就说过几天再回来，我们马上打的去成都。"

　　薛凯无奈，只好点点头。

　　魏老八又说："来回的车票我全包，如果到了郑州，拿不到钱怎么办?"

　　薛凯扶了扶鼻梁上的眼镜，说："你砍掉我的手。"

　　高速公路上，一辆深蓝色的桑塔纳飞驰电掣般地朝成都驶去，车上坐着魏老八、薛凯和我。

　　车上静极了，薛凯像泄了气的皮球缩成一团，这是我第一次随魏老八去收赌账，说不出是兴奋还是紧张，感觉新鲜极了。

　　魏老八用手机给威远一个朋友打了电话。那个朋友告诉了一个传呼号，是一个在成都火车站倒卖火车票的，据称搞火车票没有任何问题。

　　一个小时过后，我们来到了成都火车站，我按那个传呼号打了个传呼，传呼马上回了：

　　"哪个"对方问。

　　"我是威远猪脑壳介绍的。"我答道。

　　"啥子事?"对方问。

　　"我们要今晚3张去郑州的卧铺。"

　　"我马上过来。"对方挂了电话。

　　不一会儿，一辆红色的夏利停在我们跟前，从车上走下三个其貌不扬的人，其中一个矮子问我："你们是猪脑壳的朋友？"

　　我们忙点点头。

　　"这么晚了，卧铺不好搞，只有3张硬座，如果等一天，明天准有。"

　　我说："硬座也可以，上车再想办法搞卧铺，多少钱一张?"

　　矮子说："朋友介绍的，原价。"

　　我们付了钱，千谢万谢与他们道了别。这时离开车时间只有35分钟了，我们匆匆走入候车大厅。

　　列车一启动，魏老八就说："一定要搞到卧铺，在火车上熬夜最恼火。"

　　薛凯说："我的一个朋友认识这趟车的餐车长，不如我们去试一下。"

　　魏老八说："可以，不过一会儿等龙兵去联系，他普通话要好一点。这趟车是郑州的，害怕他们听到四川话反感，到时事情更不好办。"

　　薛凯不服气地说："还是我去，我在郑州全是说普通话，再说我与这趟车经常打交道。"

　　魏老八一想也是，就同意了。我们一行3人来到了餐车上，一个胖女人马上站起身来大声说："怎么，想买餐车座位?十块钱一位。"

　　薛凯说："我们找餐车长。"

　　他的普通话还凑合。

　　约摸等了30分钟，一个睡眼惺忪，又高又胖的男人摇摇晃晃走了进来，高声叫道："谁找我?谁找我?"

　　薛凯马上诚惶诚恐地迎上去，可能是由于紧张，他这次用的是椒盐的不能再椒盐的川味普通话，说道："等于是错(昨)天我给我的隔壁子打了个电话，他说你在这趟车上当炊哥，板眼儿有点长……"

　　那个餐车上眯着眼睛，根本没听懂薛凯在说些什么，他马上打断他："什么事?什么事?"

　　"等于是我们想困觉，想补张卧铺。"

　　"什么?你用四川话说，我还能听懂点。"

　　"卧铺!"薛凯马上变成四川话。

　　"几张?"

　　"3张。"

　　"等着。"胖餐车长转身走了。

　　一切竟这么顺利，我们都不敢相信，同时我和魏老八也被薛凯的普通话逗得前仰后合。看样子这个薛凯是个宝，我们的旅途不会太寂寞。

　　薛凯比我更像是一介书生，白皙的皮肤，身体单薄，身材不高，瘦削的脸颊架着一副老式的黑框眼镜，没肉的鼻梁坚挺着透出几分刚毅。他是山西大学中文系毕业的，按说生长在农村的他人材一般，又没有过硬的人事关系，学艺又不是很精，能分配到一间不错的中学教书，也算是比较好的归宿了。可薛凯命带桃花，一个年轻貌美的哑女看上了他，薛凯开始并没有往心里去。在大学时，他在爱情上碰过一鼻子灰，所以他收敛起那颗夜夜萌动的春心，一头埋进了工作。可世上男女之间的感情偏偏很奇怪，你越是不经意这段感情，它反而离你越来越近，不知不觉中你已尝到了它的芬芳。哑女几乎天天来学校找薛凯，这件"离奇绯闻"在学校已闹得沸沸扬扬，哑女一点也没自卑感，因为她父亲是M市某乡乡长。乡长的千金虽然是哑巴，但身材相貌却是一流的，这也许是老天还算公平的一面。在一个大雨滂沱的夜晚，在薛凯的单身宿舍里，这对男女无法用语言，而是用手比划着钻进了被窝。

　　在一阵仓促忙乱的做爱之后，窘迫的薛凯发现哑女心满意足地偎在他干瘪的胸膛前睡着了。他忽然预感到：他的命运将发生革命性的变化。

　　这种变化在他结婚后2年达到了顶峰，薛凯当上了M市团委书记。

　　可就在他青云直上的时候，他染上了赌博，先是打点小麻将，然后是10点半，然后是小马古，慢慢地赌博就像海洛因完完全全侵蚀了他的肢体和灵魂，而最不应该的是他在赌桌上认识了魏老八，他并不知道魏是赌坛高手，况且身边的金老九又是个千王。由于青云直上养成的桀骜不驯这个时候害了他，他大肆挪用公款，欠下了几十万元的赌债。

　　他害怕东窗事发后不免有牢狱之灾，又害怕魏上门讨账，他终于精神崩溃了，怀揣着疼爱他的丈母娘给他的存折，告别了泪水涟涟的哑妻，离别了生他养他学习战斗过的M市。

　　列车飞驰着，义无反顾地奔向北方大地，此时的薛凯反而没有了任何心理压力，他躺在卧铺上蒙头大睡。

　　我对魏老八说："他哪里睡的着?是不是在盘算什么?"

　　魏老八笑了笑，说："随便他盘算啥子，如果在郑州拿不到钱，你听到的，他说砍他的手。他的手都可以不要，未必我还怕个鸟?"

　　"我看他不像个傻子，精明得很。"

　　"我只认钱!"魏老八坚定地说，"不过，时刻观察他，别让他中途下车跑了，虽然跑得了和尚跑不了庙。但我俩划不来，又费马达又费电。不过，我想他是真的在睡觉不敢有什么鬼点子，他敢耍我?他自己晓得后果的。他这种人比较能吃苦，毕竟以前是农民，我保证他又能吃又能睡，所以这几年在外面挣了钱，不然怎么敢答应还债?"

　　"我觉得他有点可怜。"我说。

　　魏老八马上摇着头，说："千万别让这种假像迷住了双眼，我上当上多了。干我们这行就是要心狠手辣，我的确也是这样做的，但在实际操作中会遇到很多问题。有些人嘴巴硬得很，等输了钱你到他家里讨债，他婆娘儿女，八十岁的老母全跪在你面前哭诉，喊着：'叔叔，饶了我们全家吧!'我的心一下就会软下来，一般我都会将赌债减免一半，或者干脆全免。但遇到当官的，我决不会心慈手软，因为我当了一回瓜娃子。有个烟厂的科长欠了我二万多块，有一天，我把他堵在家里，家里人又哭又闹，他老母亲气得住院，小孩又要动手术，他婆娘说赌博已经把他家害得一贫如洗了。我相信了他家人的话，只收了二千元。可是没过多久，我看见他又买手机又买摩托，还养了个情妇，我简直气惨了，又不好反悔，怕丢了面子。看到他洋洋得意的样子，我发誓，不整这些贪官整哪个?难道整穷人?所以薛凯这件事我决不会心软，他老丈人家有得是钱。"

　　正说到这儿，薛凯从卧铺上爬了下来，一边打着哈欠，一边问道："你们吃午饭没有?"

　　我说："早就没得卖了，只能买到方便面。"

　　"可以，可以，我要三碗康师傅。"

　　我和魏老八相视一笑，果然筋骨人吃得、累得。

　　一碗面下肚，薛凯的脸色红润起来，他发现坐在我们对面的是一个非常漂亮的姑娘，顿时挺起了腰杆，来了精神。他故意大声说："有人说我像优客李林中的林隆璇，你们看像不像?"

　　那姑娘本来在看小说，听到这句话，果然抬起很美的眼睛打量着薛凯，薛凯愈发得意起来。

　　魏老八问："哪个林隆璇?没听到讲过。"

　　"台湾的著名歌星，又会唱歌又会弹钢琴，又会作曲，我最喜欢他那首《认错》，唱得好听得很。"

　　说着竟哼了起来："I don't believe……是我放弃了你，只为了一个没有理由的决定……"

　　"别唱了，我是不是该安静地走开?"魏老八打断了薛凯，"你像林隆璇，人家还说我像杨子荣呢。"

　　那个姑娘忍不住"扑哧"地笑了起来，不一会儿，薛凯就和那个姑娘热火朝天地聊了起来，越聊越近，大有相见恨晚的样子。等我和魏老八上了厕所回来，薛凯已经和那个姑娘手拉着手，含情脉脉地对唱起《选择》："你选择了我，我选择了你，这是我们的选择……"

　　魏老八也笑了，对我说："他简直是个诈骗犯的料。"

　　"具备这种素质的人毕竟很少。"我附和道。

　　薛凯哪里理会我们的闲言碎语，正拿出钢笔给那个姑娘留通讯地址。

　　魏老八拿起一块卤鸡翅说："管你选择哪个?我先啃一支'变心的翅膀'。"
