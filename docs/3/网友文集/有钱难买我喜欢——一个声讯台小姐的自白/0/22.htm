第二十一章 不管黑猫、白猫，能抓到老鼠就是好猫


　　泮溪酒家的十二路站牌下等了半个小时，我们还是坐冷巴回了宿舍。公交公司提供的冷气巴士我们不是不想乘坐，而是高出一倍的票价不能承受。

　　打开冰箱，取些凉爽的冰块加进可乐，我痛饮一气。南南听到我说话的声音走出房间，"乐乐，来得早不如来得巧。你爸妈刚离开，留了封信。喏，写字台那，你看吧。"

　　我拆信，坐在沙发上筛糠着读完信件，泪潸潸而下。

　　"怎么了？乐乐，哭什么？有事说出来，大家可以帮助你。"阿言问，她正切土豆碎块用来煎鸡蛋吃，主持人下夜班的早饭都吃这个。

　　噙着眼泪，我颜面无存地说，"哪有逼自己的亲生女儿卖淫的父母啊？"

　　阿言几个轮流阅读信的内容，惊叹不止，起初她们对我长期不回家不理解，也不确定有我所描述的父亲。现在事实在眼前，由不得她们不信。

　　爸爸是这样写的：梦玫孩子，根据目前家庭困境及必须扭转的趋势，我告诫你有必要转变自己思想的标准，达到解决全家生活的问题，为立足深圳努力。我们来圳一是因为居家惨遭破产的劫难，含愤来深圳是为了避难。二是以你和你两个妹妹的前程为主，希望你与梦帆攀上高枝移民国外，彻底摆脱农村。三是实现定居南国，达到光宗耀祖的目的。达到富裕的地步一直是我们共同努力和必须要达到的宏观理想！出来好出，回去难。家乡已回不去，来圳又没赚到钱，现在无有立足之地。你妈的商店无钱无精力继续营业，即使开店赚钱也不多，已到了山穷水尽的地步。就现状讲，全家迁来深圳的人，无一个象我们家这样困难，事实正是如此，几乎面临死的地步。为了扶持家庭走向平稳的生活，为了给将来的幸福奠定基础，爸爸特地把梦帆从小餐厅调到安华大酒店工作。在那里可以结识"大人物"、高层次的福贵之人，继而帮助我们发家致富。现在梦帆正滋长心眼，后悔荒废过的时间和机遇，正在设法拯救家庭，挽回就要履没的家庭。

　　你虽蒙爸多次教导，给以很大程度的希望，也未能够理解自立创业的意义，更没担起一定要由你担负的责任。至今我们已来深圳两周年，你未给家庭添设资金，又失去多次难得的机遇，落得自己无有长进，真是痛心啊！到目前为止，你仍未关心家庭一点，连自己也未能找到安顿或能够帮助解决困难的外力支援。不能再这样，应该猛醒。

　　如若真的再混下去，那即是在慢性自杀。如果父母亲灭亡在深圳市，你有愧自己应尽的责任。看看别人来深圳，有的一至两月就过上了好日子，买房亦买车，我们却仍是讨饭郎！深圳的确是个好地方，只要有本事就可以完善远景，能够实现夙愿和来深圳的全盘计划。不去利用追逐你的男人有害于自己。象这样的话，就不如呆在家别出来。况且老家的人也无时无刻不向前发展，为什么你不如人家有远见？有些别人寻觅不到的机遇你却任意挥霍。

　　你怎能对得起自己做人的人格？怎样维持现有的家？怎样去见家乡亲友？又怎样破产被逼出走的，难道你都已忘的一干二净？爸爸说过你心胸小，脑壳也小，你不应该跟爸爸为仇作对！你没有理由这样对爸爸。我们是苦难深重的家庭，我们是被血渗透的家庭。一定要记住我们必须实现定居南国的目的。如若你有良心有报负，就应该醒悟，急忙跳出迷茫的圈子，彻底砸碎自己被禁锢及不发达的脑壳。你妈落后的思想也正进步，又在怨恨你不把握机会，你倒不如没有文化的妇人。来深圳已两年多，也该长些心眼了吧？爸爸希望你转变成一个既聪颖又灵活的具有现实精神的深圳小姐，能够大刀阔斧地创业。寻找已抛弃的东西，赶快为了灾难又摇摆的全家作出你应要做出的一切贡献。你应作到一个姐姐的表率，第一个动力资源。梦帆接着也会相继尽责。说句实在话，能够混出个人样来，自己及全家高高在上，这才是真正的路子及本领。这种思想是别人无法比拟的。

　　希望你能够就地成材。从哪一点讲，爹妈的路子都是绝对正确的。邓小平爷爷一贯讲，"不管黑猫、白猫，能抓到老鼠就是好猫。"这句话千真万确，我们老百姓同样遵用。一齐抓经济，谁先富起来，谁就是英雄！深圳就是这么一个现实条件。再不成长，悔之晚矣！近期，你从未关心家中生活，我们度日如年啊！后面如何生活下去？还要供养梦梦上学，沉重担子以后怎样担付？你怎么一点也不考虑？你临时的热线职业也不是固定的理想工作，不长远规划是不行的，还需再寻找称心稳妥的正式职业。爸和你妈考虑，希望你和你妹都具有现实的金钱思想，创造一个崭新的家庭。我们还将在条件成熟后衣锦还乡。把信看上二十遍，定可以改变你的原来观念。要寻找机遇超跃普通人，变贫穷成大款，有远赴国外的追求。爸爸亲笔。

　　"他让你读二十遍，我看一眼都气得手脚冰冷。你这自吹自擂有文化、懂情理的父亲用亲情的鞭子逼你上梁山，逼你去卖淫。乐乐，不要妥协，为了这样的家长葬送一生的幸福不值得。"南南同情地抖着信拿给莫儿。

　　"你见到没有？每一页的题头还特别重申了生死、离别、争气、安身尤其重要！光阴似箭，失去的太多啦，应奋力夺回来。机遇就在眼前，莫失了良机。清醒头脑抓紧立足，安家定乾坤，切记，切记！我觉得你父亲真是无耻之尤，家庭的责任和担子怎么能压在一个柔顺女孩的身上？他堂堂一个男人，为何不能负责？又不是老得不能动了，为什么把担子强压到女儿身上？"阿言气愤地丢掉手中的菜，抚着我的背安慰我。

　　"叫我怎么办？我无论躲到任何地方，他都会不惜一切代价找到我，威逼我！"我哭得脸色转青，一张脸上麻麻的疼。

　　南南瞅着我心碎难过的样子，机灵地说，"乐乐，我有办法。你花钱另外买个户口，就可以不受父亲的控制，以后谈恋爱结婚完全自主。"

　　听到南南的建议，我有点希望地抬起头说，"这样也可以，可是我想和他断绝父女关系，只有这样他才没有权利约束我的行为。"

　　她们称是，同意我的想法。我打通114查法律咨询热线，查号台小姐给了我一个手机号。拨通后那个律师告诉我说，亲生父女的关系是没有办法解除的，即使法律上宣布双方不再有关系，但血缘上他仍旧是我的父亲。这种案例在中国没有鲜明的解决方式。

　　我决定买个新户口，这事我得找常孚帮忙。为了户口我约常孚见面，他在人民医院入口处等我。我绕到常孚身后轻拍矮小的他的肩膀，我希望见到他的脸不要像见到他的身高那样失望。常孚转身面对我，我不自然地说了声，"嗨！"他满意地打量我，显然我比照片吸引他。我勉强地笑了笑，常孚的声音温柔、悦耳，人的样子实在不敢夸奖。

　　他不自然地避开我，因为我企图挎他的胳膊，他畏惧熟人见到。常孚急急走在前面带路，我们去新安酒店，他说新安酒店的烤鸭做得有名气。我辛酸地尾随他。电梯里，他量量我的身高说，"和我挺般配。"我心里怪生气地想，他才一米六八，我踏上高跟鞋差点高过他。不自量力。要不是为了户口，我费事睬你。

　　我们开间房吃饭，点了六个菜一汤：烤鸭、马蹄烙、四只乳鸽、西芹烧虾球、蜜汁吊烧白鳝、宫保鸡丁、两盅冬虫草炖水鸭汤。我吃烤鸭不喜欢往小煎饼里夹大葱，浓重的葱味令人口气不清新。瞅着我吃东西，他的一盒三五烟快见底了，我抢过烟盒自己叨了根，常孚帮我点燃。"你胃口不好吗？怎么不动筷？"我吐出烟圈，微笑着问常孚。

　　听我提意见，他别别扭扭地拿起筷子，觉得不自然又放下。我看不过眼夹起一块马蹄烙饼喂他。常孚受宠若惊地囫囵吞咽下去，我让他喝了口碧泉柠檬荼。

　　"你多吃些，我知道你们没什么荤菜打牙祭，你瞧你瘦得快成林黛玉了。以后想吃好的，就找我，我带你出来解馋。"他言简意赅地说。

　　"请我吃饭的人多得是。可家里的事闹心，所以人渐消瘦。所以，我想独自弄个户口，和家里人断绝关系。"我轻描淡写地引入正题，户口的事是见他的主要目标。

　　"那么买个广东省附近的户口吧！我认识的一个朋友可以办。河源、紫金那边的户口三四千块就买到了。深圳关外的指标得等买房子的人往外卖，市内根本买不到，只能这样了。"常孚的目光游离不定，闪烁着我捉摸不透的光芒。

　　"越快越好，"我说，"我很想变成另外的身份，没有家庭的困惑，没有亲情的麻烦。近来我的压力太大，我的忍耐到了极限，你不帮我谁帮我呢？你也清楚我轻易不接受别人的馈赠，我脸皮薄，不愿意欠别人的人情，只能找你帮忙，你是我的一切。"我连哄带骗。他没爱过谁，对我的爱令我说什么他信什么。

　　"你想改个什么名字？"他同情地盯着我的眼睛。

　　我欣然讲出早已定夺的新名字，"我想换成凌子这个名字，它的意思是凌波轻云的天之骄子。你觉得好吗？"他赞赏地点头，烟不离手。我摸他焦黄的手指，邀功说，"我又报了四门自考课程，去年考的八门顺利过关。你难道不夸夸我吗？"

　　他一点也没意外，平静地回答我，"夸什么？我认为你具有这样的天赋。在我心中，你是完美的。"

　　我温热的唇贴向他，他手足无措地坐着，眼看我就吻到他，服务员推门进来了。他条件反射地推开我，正襟而坐。脸色暗红，眼神慌乱。我自我解嘲地拿张纸巾，径直去洗手间。

　　我一脸委屈地坐回椅子，等他就刚才的举动道歉。他捏起我的手，放下三千块港币，我惊愕地抬头，"我收你的钱不合适，这种方式的赠予令我不安。你不是已经帮助我了吗？"我迫使红晕飞满脸颊，以表我的坦诚。我巴望他再坚持，那样的话我可笑纳了。

　　"不是，"他温情的眼睛俯视着港币，"再没有谁比我更了解你，你可爱、善良，女性的天然美在你身上汇集，我为拥有你的爱而陶醉。这钱是情人节的礼物，我想与其买东西不如给钱，这样你就可以买自己喜欢的东西了。你得接受我的一片心意。"他见我默不作声，打开我的皮包把钱放进暗层。

　　因为他善解人意，我快活主动地上前吻他。常孚经不住我的甜吻，惊涛骇浪地回吻，喃喃道，"知道吗？我怕见到你冲动，出门前先释放掉才敢来见你。"

　　我耸耸肩膀，沉闷着想，这个男人的性爱缺斤少两，有条件弥补他又放弃。难道真不能直率地纵情吗？"要知道我是爱你的，如果你想要，我会给你！"我半欺骗地带有点试探地表白。

　　"不，"他疼爱地抱我，动也不动，"我不想伤害你。既然不能娶你，我当然不能占有你，我们可以柏拉图式地恋爱，直到永远。如有来世，我会在黄泉路上等你，我们做下一世的夫妻。"

　　男人爱上女人后，应该具体从财物的方面表情达意。两片唇加一张舌头轻易地推出空话，好听却不实用。爱我你就应该买房、送钻戒、配备手提电话、开信用卡、送股票，这些才是爱我的证据。我卑鄙地说，"好，我会忠实地守约，奈何桥上我一定等你。"口口声声爱我爱到骨头里，那你怎么不买房给我？你也不是没有能力。哪怕买在关外也可以啊。我需要实际地解决问题。

　　老婆大人致电召唤他回转家门。他速速买单，没吃完的菜打包，我提回去给同事们分享。我独自上了的士，沉不住气地取出钱数。三十张百元面额的港币可能令司机眼馋，他居心险恶地取片香口胶递给我，"小姐，吃块香口胶吧！我天天都吃。"

　　我愕然，警戒地说，"不用！谢谢你。"司机不合乎常理的举动吓醒了遐想的我，我收钱入包。做出有急事要办理的模样，让司机停车。司机的车开远我才放声大笑起来，这司机九成是财迷心窍。三千块港币兑换成人民币，我到招商银行申请了张一卡通。

　　尚没拿到改弦更张的新身份证，我就以凌子的身份自居。CALL台机主姓名改换成"凌子"，同时翻找电话本通知要好常联系的朋友，我旧颜换新颜。面对朋友的不解，我随口编织种种谎言煞有其事地认真周旋。王八蛋骗你，我确实很痛苦。名字身份可以改，然而我的骨髓敲有秦梦玫这三个字。我是秦继洋的女儿，逃脱不了。实际，买个独立的户口未使我的灵魂有几许轻松，我明白我只是多了层新面具罢了。
