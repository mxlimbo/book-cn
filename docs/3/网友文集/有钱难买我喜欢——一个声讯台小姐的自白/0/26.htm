第二十五章 一对多是多姿多彩的热线电话种类


　　星期五我值白班，九点接班。紧接打进的听众不开口讲话，我想着华东南，心湖荡漾着春意，一扫平日的不耐烦，甜蜜得引导听众说，"请问您是哪位？怎么称呼您呢？我是乐乐，您打通的电话是热线电话，我们可以随意地聊聊天，说说话。您如果不想说话请按键表示好吗？您如果是先生请按一下键，是小姐请按二下键。如果您还是不想说话，我可以念诗给您听。我们这里有散文和脑筋急转弯，您如果不想听诗，我可以唱儿歌和流行歌曲┄┄"我言语动听内容万变也没打动听众的铁石心肠，他不屈不挠地听我唱独角戏，没有丁点反应。当我准备真的念报纸时，他天经地义地挂线了。唱这通话时间长达五十五分钟的单簧累得我直喘粗气，灌进了两杯开水。

　　放下电话，同事找我谈话，我百媚妖娆地冲人家笑，笑着笑着捂起嘴，脸色恰似三月桃花红。222的主持人妹妹总结我的行为是恋爱综合症，她说，乐乐谈恋爱，全世界的人都分享了爱的滋润。

　　润迅传呼台跳槽的芦苇是广东清远人，身高一百六十公分，微胖，长相不靓，人格厚道，没有太多的主心骨，凡事经不住别人劝告，思想随着别人改变。她看到别人的牙签都觉得比自己买的优质。

　　我哈哈大笑，自告奋勇地说教他讲外语。

　　"你教我外语？我告诉你，我英语四级呢！"他极不认输，挂电话时还妄想纠正我，"人难做还不如难做人呀！"

　　早上九点钟下班，我们C班全体开会。公司开会不占用当班时间，怕妨碍接电话。公司经理黄英主持，助理张严和班长555线的主持人欧阳维持纪律。我见到黄英有种犯人见到警察的恐惧，她样貌相当严肃。

　　这间不是专门的会议室，充其量只是公司的大客厅，连着走廊的洗手间。拂过的风夹带着洗手间的味道，看着大家皱起眉头掩鼻，阳光悄悄嘀咕，"这会何时结束┄┄""我们三令五申地强调禁止与听众谈论其它主持人的话题，总有人违反规定。我重复宣布一次，如有再犯绝不姑息养奸。现在当着大家的面，我念念电话里主持人慎谈的话题，"她厉害的双眼扫射参预会议的全体主持人，清清嗓子说，"一、本台她人的外貌、家庭、婚姻状况；二、本人的个人资料，如真实姓名、婚姻状况、工资待遇；三、本月个人话务量的情况，工资发放情况；四、涉及家庭关系，伦理道德及特殊行业的信息（如股票，非熟悉行业等）。除了以上四条，我希望我们公司的主持人遵守邮电局的六不准规定，有技巧地聊天，别谈低级趣味的内容┄┄"大家心里不以为然。黄英说的怪好听，严格遵守肯定导致我们揭不开锅，围绕规定轻轻打擦边球是难免的事实。

　　男士线的章晨光和夏天涛涛不绝，传授自己成功主持热线节目的经验之谈。经理提醒新人提出工作疑问，趁老主持人在场好予以解答。没人踊跃开口，黄英环视全体，点白天的名发言。白天腼腆地说，"其实也没什么，渐渐地适应了。只不过好多新听众提出的关于我们热线是做什么的类似问题不好回答，听众对于我的解答不满意，想请教这个问题。"

　　《夏日男女情》栏目的主持人石头有个呆头呆脑男性化的艺名。她不慌不忙地解说，"这个问题可以根据听众的性格自由发挥，尽可能地幽默，轻松。譬如我们的工作是把悲伤留给自己，把快乐带给你┄┄"助理张严对石头的解答挺满意，他强调另外一个屡见不鲜的弊端，"最近有听众投诉反映，我们的主持人深更半夜CALL人家打声讯电话，引起了听众家人的误会。这个问题希望大家以后注意，不是不准你们CALL听众，必须把握分寸，别令听众反感┄┄"老员工轮流发言，黄英的眼睛扫到谁，谁马上发言。她是威不在声的那种女人。老油条式主持人背后称她叶赫那拉在世。

　　"最好是周末或者平时留台给听众，祝他快乐，工作顺利。听众才不会弄清楚我们希望他们打热线时才CALL他们┄┄"同事们各抒己见，好的点子纷至沓来。

　　我没有比较额外的表现，她的眼睛瞅我已有一阵子，看样今天再不表现是难过这关了。我鼓了鼓劲说，"我觉得最好不要CALL听众，如果听众长时间不打电话给我们，才可以留个温馨友好的台给听众，最好加上祝全家快乐之类的话。我从前的一天晚上接到一个很不开心的听众电话，他和我彻夜攀谈才休息。我下班时CALL他留台说，今天的太阳很好，请把你潮湿的心拿出来晾晒。他很高兴，以后成为了我的忠实听众┄┄"我的发言激起同事的同感，众说纷纭。

　　黄英权威地说，"大家应该学习采谰乐乐的观点。对待听众不要治丝益棼，理丝不找丝头，会越理越乱。处理听众这个问题一定要考虑清楚，没有计划地行事，只会坏事。"我怀疑她管理热线的策略中融入了《孙子兵法》的精华。

　　大家相对点头，无声地同意黄英的说法。

　　"黄经理，听众百般挑剔，我们应该怎么办？"又一位新同事粉墨登场。热线人员变动频繁，二个月下来，公司员工的面貌新换一半，做到一年以上的几乎成了妖精。

　　"顾客就是上帝，无论他怎样挑剔，我们都要包容。就算听众骂出脏话，你尽可以把电话摆在桌台上，暂时别听。等几分钟再拿起来听，不就可以了。"雪莲是元老级的主持。她也是妖精，一个其丑无比的妖精。

　　"关于这点，我想说几句，"我坐直身子，坦荡地说，"我不知道大家有没注意美国的连锁超市沃尔玛。沃尔玛超市的墙壁上贴着两句话，上面写着：请注意一、顾客永远是对的。二、如有问题请参照第一条。看到这两句话对我工作的帮助很大。听众打进电话来，只要你想着听众是送钱来的，无论他怎样不礼貌，我们的心情都可以平衡。"

　　我语出惊人，四座皆服，同事们奋袂而起为我鼓掌。听众耳朵里的主持人如沐春风，得意洋洋。真实生活中，我们生活清贫，我们私自约见听众基于工作闭塞，认识听众便是活着的意义。我们需要成家、立业，需要有自己的深圳房产。听众是我们的支点。

　　班长欧阳说，"是啊，听众不讲话也不要紧，反正我们接起电话有一毛五分钱的进帐。积少成多，月底不就是一笔不小的财富吗？"

　　提到钱，会议的气氛振作起来，工作研讨会议持续了二个来小时。
