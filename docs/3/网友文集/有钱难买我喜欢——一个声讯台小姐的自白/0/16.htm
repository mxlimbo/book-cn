第十五章 热线主持人的生活岂止是无奈


　　老听众李远钢出差回来打通106说，"我很高兴能够在异地他乡认识你，并且受到你的信任。毕业参加工作三年，过去好象浑浑噩噩，而今天，你给了我这份好心情，我不知道这是否缘份，但我一定会好好珍惜这份好心情。在此我真的谢谢你，乐乐。"

　　"你现在还好吗？好几天都没有跟你通电话，不知道你在不在意我们上次的通话？你要我告诉你，我的童年及我生活中的一些事，我答应了你，希望你给我时间和机会考虑，有好多事我不想提起。但为你我一定会打开这本比小说、电视剧的内容还丰富的书。可以寄来你的玉照吗？我们能够成为那种朋友吗？"听众陈珂十真挚地表白。

　　我呸，自大狂。他哪里知道，我想听他的故事是因为我没话题找话说，诱惑他多打些电话，他还以为我真的钟情与他。

　　李亦寄特快专递给我，里面有好吃的果脯还附带信一封：乐乐，昨天晚上，打完电话去睡觉。躺在床上，翻来覆去睡不着，满脑子都是乐乐两个字，眼前仿佛都是你飘然的倩影。我可以从你的说话想象出你的音容笑貌，我猜你一定非常漂亮。就这样想着你，好不容易熬到四点钟，我才睡去。今天早晨醒来，我跌跌撞撞地去东莞虎门，忙碌一整天回到住的地方。我独自呆坐，四周是冷清的静，好象连钟都不走了似的，一切都是那样的悄无声息。总觉得自己还有件事没有做，知道吗？乐乐，我成天想着你，好想跟你通电话。你说长途打不进来，我不信一定要试试。我冲出房门，跑到街上找电话亭，真是活见鬼，电话亭都关门了。天下着雨，我一口气跑过三条街，终于找到一间商店。拨你的热线号码，电脑提示音说我的号码是错的。我再打，同样的回音，一连打十多分钟，同样的回音重复不断。老板也生气了，骂我真笨，连个电话号码都打不通┄┄我想见你，等我回到深圳，咱们出来见见面好吗？我想送你一只戒指。

　　傻蛋，我们的声讯电话只限本市区域通话，出了深圳根本打不进来。我心里骂着他，拿果脯分给同事吃。我对李亦付出捏造的关心造成其错觉，他以为我对他有点情人意思。其实这样对我们的工作太有利了，每个听众都把主持人看作女朋友来牵挂，电话业务不就源源不绝了吗。

　　"妈的，这女孩真丢人，败坏我们热线的整体形象，"莫儿摔掉电话后骂着谁。

　　"怎么了？是谁惹我们莫儿动气？你向来不生气，肯定有人激怒了你。"我扶住隔音板，上身前伸到100台问莫儿。小雨走后，我刻意促进了同她的关系。人在集体生活中，不能没有朋友。

　　"103新来的张小宇去见一个听众，非要听众请她吃饭。人家听众从洗手间出来，她已经走了。买单时听众发现公文包里的钱包没了，手机也找不到。不是张小宇还有谁会拿？现在听众打电话过来投诉┄┄"莫儿说完，走到客厅CALL听众。现在电话不多，为了话务量我们主动呼听众打热线电话进来。

　　这个张小宇也真笨，你想要钱可以跟听众好好商量嘛，最多是撒撒娇，卖弄一下风骚喽，人家不见得不会给，何必偷呢。

　　我问103台下班的同事，文文为何没来上班。同事忙不迭地说，"她前天偷南天大厦的自行车，被关到银湖收容所去了。昨天她打电话过来公司求我们拿钱赎她，这种丢脸的事谁愿意干┄┄"这倒好，又有人折腾进去了。

　　近来进线频率降低，郑总和温情晚上请我们管理人员喝荼，顺便召开公司首脑会议。我们看着老总的脸色叫东西吃，净拣便宜的点；油条、马蹄糕、麻团、青菜、白粥、豆浆。

　　如今公司每条单线做二万分的成绩，核算的结果是亏损。温情一锤定音地说，"最近主持人见听众的情况恶劣，是话务量降低的主要原因。你们各个分台开会重申，再发生类似情况，公司不会坐视不管┄┄"基于公司效益不好的因素，郑总决定取消底薪制。每分钟提成一毛六分钱，免除台长职务，三班各评出一名值班经理来管理日常工作。我吊儿郎当的工作态度当然评不上值班经理。莫儿人稳办事牢靠，成为我们A班的领导人。

　　散会后，几名台长嘀咕说，"整天亏损亏损，你们的新房子从哪里来？"

　　"是啊，他们买的房子是我们主持人用口水换来。"樱子不甘落后地加上一句话。

　　我跟阿言想，艺术培训中心学吹洞萧的课程无法继续下去，因为二百块的学费我们已经力不从心。

　　紧接其上，郑总相应制定了许多新条款：连续两个月总话务量排末尾的主持人要自动离职。热线主持人辞职离开公司，当月薪水必须扣押一个月。

　　昨天离职的云朵得知被扣发一个月工资，她气愤之余将公司的几部电话机拔下拿走。郑总开会公开指责云朵的行为实属偷盗，郑总大义凛然地说，"我若是与她一般见识，根本没有她的活路。几部电话机值不了多少钱，便宜她了。鸡鸣狗盗的人，其它单位更容不下她。"

　　我不可告人地想，我要是她绝对不偷电话机，我去劳动局黑你一状，你的声讯台准倒闭。整人须往死里整，打架的怕拼命的，拼命的怕不要命的，不要命的就怕我这种阴险的。

　　打进的电话少了以后，我们上班就聊天吹牛，开辩论赛以打发时间。

　　"姐儿们，猜个字迹吧！乐乐吹箫打一个字、一个成语、一种行为，你们猜猜吧！谁猜中，我请她去国贸的旋转餐厅喝早荼。"102的主持人秦沛沛说。同事们竞相猜测，答案全不对。沛沛得意万分，"告诉你们，打一个字是咬，成语是吞吞吐吐，行为嘛，"她嘿嘿地笑，卖关子不讲。

　　身为管理者的莫儿说，"你们不要讲黄色段子。听众听到就惨了！"她皮肤上坚持不懈地长脓胞，正抹着药水。

　　沛沛说，"唉呀，班长。不怕的，现在话务量这么低，我们对听众说这些，话务量会有所提高的。现在哪还有人看黑白电视，来点色彩大家皆大欢喜，多好！"沛沛是已婚妇女，乳房长在肚脐上方四指高的地方，她天生如此。一个怪女人。

　　"沛沛，到底是什么？哪种行为呢？"东方吃着芝麻糊问，芝麻香钻进每个人的鼻孔。

　　"告诉你们，是口交！"沛沛淫荡地笑着，她下半身晃个不停，椅子吱吱地响个不停。我们骂她流氓，这时102的电话响了。"喂，你好！"沛沛以丹田之气发声，迎接听众。

　　"小姐，你会唱十八摸吗？"听众问沛沛，他提的问题正好是沛沛的专长。 "我不会唱，可我会做。你想不想试试？我用泰式按摩法给你推油，从脑门到口，到胸┄┄"沛沛调戏着听众，嘴巴边讲边用手由脸摸到屁股，说话声音低了下去。

　　阿言问我，"你怎么喜欢黑色的衣服？黑虽然使人气质高贵，可感觉压抑┄┄"我浏览手中听众的来信，分神解释，"我认为黑色可以掩盖一切暇疵，我甚至希望自己的肤色变黑。非洲人种多棒，脸上长痘痘也看不出来。"

　　现在的城市人不知犯哪门子毛病，年轻人乌鸦鸦的一片漆黑，净穿灰色、黑色、白色、棕色的衣服；上点年纪的妇女却大红大紫又大绿，整个阴阳颠倒。

　　沛沛又加入同事们的聊天，她说，"乐乐，听说你在宿舍，睡衣里面是光板身体，从来不穿内衣是不是？"

　　啊！我心内一惊，连这情况她都摸清了。我搪塞说，"哪个龟孙王八蛋告诉你的？胡说八道。只有一次我的内衣没干，所以没穿。"无论是谁讲的，这事如假包换。我喜欢夹紧双腿的微妙。

　　我听到沛沛好象又和谁小声耳语，刚想坐起来问个究竟，我的电话有听众打响。"乐乐，快接电话。听众说他喜欢你这种类型，打到106去了。"

　　"喂，你好！"我没弄明白沛沛的意思，匆忙地接起电话，含糊地说。

　　"小姐，不喜欢穿内裤的那个主持人是你吗？"听众流里流气地问。

　　"是啊！怎么了？这叫个人喜恶你懂不懂？"我窘得气急败坏，口不择言道。

　　"我想干你！没有内裤多省事，我轻轻一用力┄┄"听众的秽语比三级还三级。

　　我豁出去地骂，"你干我？我还操你妈呢！"

　　"行啊，只要我妈没意见。可我替你为难，你拿什么家伙操她？"他喜欢讲脏话的女孩，这是种意外的刺激。听见女人讲脏话，他有生理性的快感。

　　我冲着话筒那头大叫，"你他妈的日后生孩子没屁眼，不得好死！"砰地挂了电话，这样的钱我宁愿不赚，一分钱才一毛六。"沛沛，你太不够意思了，怎么可以让听众听见这些话？"我怒形于色地走到沛沛台上冲着她发火。

　　沛沛惺惺作态地说，"好乐乐，别生气。轮休时到我家来，我请你吃饭好吧！刚才是电话没挂好，听众自己有耳福听见的，不能全怪我。亲爱的乐乐，生气会长皱纹的，别这样好吗？我家有好多老公从香港带回来的毛片，你到我家随便看还不行吗？"

　　我瞟瞟她，捏起台上的西瓜子嗑。别的同事也起哄，"沛沛，我们也要看毛片┄┄""莫儿，你知道吗？小雅和听众聊黄色内容，电话里的听众手淫┄┄"沛沛诌笑着告密，她唯恐天下太平，总趟混水。

　　莫儿刚想仔细询问情况，100一对多的电话响，莫儿接起来。听众不说话，她唾液星子满天地飞，听众还是收线了。接着102的电话响，沛沛抓起电话，听众闷着倾听二分钟收线。103响起，新主持倩倩接起电话，听众马上挂断。104又响，我跑过去，"我来接，我来接。我断定是一个听众打的，我来对付他。"拿起电话，我就说，"我知道你又想挂电话，不过我不怕你。因为我会先你一步。"说完我挂了电话，满屋子同事怅快地笑。105的电话响了，我推开阿言接起电话说，"嗨嗨，没想到吧！还是我，气死你喽！"听众马上收钱，打到106。我的电话是自己掏钱买的子母机，我拿起子机去客厅倒水，嘴巴说着，"我叫乐乐，你斗不过我的。山不转水转，咱们山水有相逢在106，你就到此为止吧！咱们掏净心肝聊聊。你是哪座山的？跑哪边码头？我想领教领教。"

　　话务间的主持人对我挑大拇指，这样的癞利头乐乐剃得最好。

　　听众不答腔，我又说，"您是男。我拿颈上的人头打赌，如果对的话，请按一下键，假使不是，请按二下键。"

　　他阴森地说，"不错，我是男的。"他那边没有丁点声音，他的呼吸声仿佛也是静止的。

　　我倒要看看你葫芦里面卖的是什么药。执拗的脾气上来，我及时行骗，做了声讯这行不会骗人也不行。"您真有性格，肯定是重感情的人物，是出类拔萃响当当的男子汉┄┄"我奉承话三千句，道给他听。

　　听众说，"你敢出来见我吗？我听说你们声讯台的小姐个个牛头马面，声音好听的不如声音平常的漂亮。你的声音这么动听，肯定貌如钟离春┄┄"他有意识激怒我，把我比作千古有名的丑女钟离春。我无所谓地说，"有缘份就可以见面。今天接到你的电话是缘，至于份呢？"

　　"份？粪不是丢进马桶里了吗？休再罗哩吧嗦，你敢不敢真面目示人？废话少说。"他粗鲁地说，很不耐烦。

　　"好，你开宝马车来接我，我就见你。而且我要你请我到香格里拉大酒店吃西餐，办得到的话万事好商量。办不到你家姑奶奶我不买你的帐。"我大唱高调，想逼退他。

　　"行，这个周末我在你们楼下的停车场等。我派人接你。咱们君子一言，驷马难追。记住，别耍我，不然有你的好看。"我提醒他公司地址。他说，"不用了，你们那个鸟地方瞒不了我，这是我的地头。"他挂断电话，我后怕起来，这人来势汹涌止不定是什么厉害的角色。

　　"乐乐，快来。我们打升级，你来刚好凑够四家。"沛沛推开话务间的门对我说。

　　"那有什么好玩的？我们玩打电话才有意思。"我把电话放回106台的机座，坏笑着说。乱打电话是我的拿手好戏。

　　"怎么玩？"沛沛叫出几个没电话的主持人，济济一堂地拥在客厅。我写下几个临时编的手机号码，让沛沛拨。一个女的听电话，沛沛是整人天才，她变态地冲人家喊，"去死吧！"这部电话是分机，我们不担心机主回拨。

　　莫儿打个138字头的手机号码，是宏亮的东北口音接听。莫儿柔软地说，"还不回家吃饭，老在外边浪什么？快回家，要不人家今晚不让你上床。"

　　东北人丈二摸不着头脑，"你是谁啊？我哪个老婆，我听不出声音来┄┄"他一个劲追问，我拔掉电话线，又插上，指使阿言拨130零的手机，"我爱你！"在我的示意下，电话通后大家齐心合力地呐喊。

　　我和刘流去沛沛家，她住深港花园。老公是做走私生意的香港人，个把礼拜回来一次。沛沛不爱他，看在钱的份上婚姻还算完整。沛沛婚前的五个男朋友都没满足她，香港老公搞得她欲仙欲死，在床上腾云驾雾。沛沛拿拖鞋给我们换，她说，"我不知道爱不爱他。一见到他我就想做爱，反正我离不开他和钱。"

　　四房二厅的房间贴着浅蓝色的壁纸，墙裙上钉有钢化玻璃，落地窗吊着四重窗纱，远处的青山绿翠隐约可见。王牌大屏幕彩电，健伍家庭影院3500A。豪华水晶吊灯光线温柔可亲，玻璃餐桌配有六把三足鼎立的血红色椅子。四拉门的冰箱下层放着超级市场里价格昂贵的进口水果，西梅、加州提子、桃驳李、奇异果、黑布林、山竹。上层是令我涎水滴流的柳橙汁、爵士朗姆汽酒、梅酒、百利甜酒、果汁香槟、舒波乐啤酒、星格柠檬苏打酒伴侣。音响是舶来品，我翻译不出英文商标，怕她笑我没见识，就没问。

　　沛沛的卧室有一张圆形的大床，床上方的屋顶嵌有硕大的镜子，两只玻璃衣柜立在屋角，其中一只挂有她结婚时的粉色婚纱。衣服五颜六色非常漂亮。我爱不释手地比试婚纱，我问沛沛，"你们的床为何没有棱角？睡起来没有边界多别扭。"

　　沛沛邪恶地笑，手摩挲着床说，"我老公喜欢这种床，可以随便睡。我们做爱时方便。屋顶的镜子是增添情趣。你想我们做爱时可以从镜子里面看到正在燃烧的肉体，这多刺激啊！"

　　我想，以后结婚，我要订做一张这样的大床。

　　好家伙，"这里面都是什么啊？"我眼睛睁得很大，NANG着鼻儿问带我参观书房的沛沛，她脸上是幸福的家庭主妇笑容。
