第二十二章 我们的收入降低到热线历史最差劲的水平线下


　　那段时间，电话少得可怜，我们的收入降低到热线历史最差劲的水平线下，白天的话务量竟然出现了零的突破，公司有随时倒闭的迹象。

　　阿言的男朋友工作调动去了上海，她乐不可支地随迁而去。南南走了，她去嘉年华跳舞，抠到某某集团的一个分公司的部门经理，两人闪电式结婚了。据说孩子七个月后准出世。还有些老员工随时随地准备走人。留下来的主持人们无组织、无纪律地见听众。郑总和温情似乎丧失了全部信心，对种种劣迹不闻不问。

　　莫儿的脸痊愈，搽了粉的她有七分姿色。我伙同莫儿晤见总想图点什么的听众。肝脑涂地的听众出钱陪我们玩世界之窗、锦锈中华、欢乐谷、未来时代、小人国、香蜜湖、东方神曲，醉生梦死地到了99年２月份。

　　我和莫儿发誓泡遍深圳所有的酒吧，这个愿望有的放矢地被实践。深圳的第一间酒吧威雀、凤凰台夜总会、世纪会、夜猫、JJ、DJ、FACE、群星、太阳城广场、骑仕吧，柏宁是我们玩腻的地方。金色时代、人气最旺的龙胜吧、德堡啤酒城、世纪宫夜总会、公牛吧、金殿、辉煌、花花公子、金枫叶、纽约、还有地王的硬石、毕打奥、圣堡罗、黑巴德的自酿啤酒城、君悦、麦迪逊┄┄这些地方海赚了我们带去的听众的钱。

　　新年濒临，四分五裂的街道无不悬灯挂彩，大红烫金的字幅、门联惹眼地张贴在显眼的地方。象征吉祥如意的利是花和桔树一盆盆被人买走。各大超市里的收银员点钱点得手指长趼子。

　　没钱过年的滋味是苦辣的。所以，我们主持人的日子不好过。除夕的晚上，莫儿的听众章宾请荷包谫陋的我俩吃台湾花园的元太祖蒙古烤肉分店的自助火锅。我俩几乎乐癫了。

　　蒙古烤肉店里大哗的男女老少脸上带笑，新年新气象的除夕晚上富裕的生活是沸腾的。我羡慕他们，甚至有股冲动，抢光他们的钱夹子。我从莫儿眼中观察出她亦有同感。

　　穷人的肚里没有油水，专门饿上一天的肚子转眼间填饱，吃饱喝足的之后，当然可以撇开听众了。我对莫儿耳鬓厮擦一番，卸磨杀驴的莫儿征求章宾的意见，"我们想回宿舍看春节联欢晚会。"

　　坐在对面的听众章宾鬼祟地偷打小九九，他的钱不能白花，他下面行动的步骤是看电影，黑黑的包厢里顺便沾点女人的味道。他拒不服从。伶牙利齿的我说，"一年就一次，可不能放过，该睁眼时一定得睁眼。"

　　听众无可奈何地买单，他不悦地招来的士请我们上车，说了声再见扬长而去。我、莫儿意外地对视，莫儿低声说，"他不送我们，的士费得自己出，多不值！"

　　我灵机一动，对司机说，"对不起，我们要到前面买点东西，现在下车。请多包涵。"

　　大胡须司机扭头冲我们蔑视地瞅着，囔囔道，"有没搞错，没有钱别坐车！"

　　我和莫儿下了车，步行回宿舍。99的春节风和日丽，携家带眷的游客悠闲地拍照留念，收罗景色。我瞅着他们摆着各种各样的姿势，烦躁不安。我认为风流、轻佻地摆弄姿势太麻烦。大剧院斜对面的邓小平画像前许多人谝能地跨进花坛里边，栽种的花朵多半糜烂被践踏得惨不可睹。我埋怨地怒视那些缺少五讲四美三热爱品德的人。

　　同事们玩的玩，值班的值班，宿舍里空空如也。一览无余的两个傻B打开电视机，"铿锵"声声，电视台文艺联欢晚会的安塞腰鼓猛烈地敲响。莫儿吃大白兔奶糖，我剥光糖纸用开水溶化糖块，沿着窗户根倒。我想吸引蚂蚁或者爬虫，看看它们，也许心里会舒服点。莫儿心疼地说，"你看你，这糖是一分钱一分货买回来的。你不重视，我还在乎讲电话的劳碌呢！劳驾，别忘记，公司扣来扣去，我们一分钟才赚一毛四分几。"

　　我笑嘻嘻地上前搂住她，"买个红灯笼，怎么样？"蒙胧的夜空没有遮拦物，烟花不可捉摸地灿烂后消失。我俩勾肩搭背地下楼买了只巨大的红灯笼，挂上客厅的吊扇叶片。灯笼大，走来行去难免碰到草绿色的长缨。我说，"莫儿，我们把它挑在窗户棂上吧？这样多碍事呀！"

　　"这间房这么冷清，就得红色吉利的东西来渲染。弄到窗户棂上象什么样子嘛？又不是家里死人挑灯笼？"莫儿刷着厚厚的绿色指甲油。

　　我俩傻瓜般地抢着灯笼，一会她挂回来，一会我挑去窗棂，折腾得四肢疲软。哎！真不想说春节的破烂事。莫儿坚持看完了联欢晚会，我十一点钟便颓唐地蒙头睡觉。半梦半醒之际，我好像哭了。我想妈妈，虽然她不好。

　　初一交早班。公司的利是分八毛、八块、八十块装在红包内，放在纸箱子里让大家抽。按人头每位主持人都有，各凭运气摸奖。莫儿得到仅有三名的八十块利是。讲定下班以后去向西村吃麻辣烫，她请客。因为我抽得了仅有八块钱的红包。那天，我狠宰她一餐，她酸溜溜地说，逮着机会，她肯定不放过我。

　　正月初六，常孚打电话给我，深陷情网的他说，"乐乐，我很想见你。你可知我的心？我日日夜夜地牵挂你┄┄我给你准备了一个大大的利是。"

　　他深中肯綮，一听这话我乐了，等的就是这红包。我揣摸常孚承诺的大大的利是究意有多大？有没有一千啊？口气这么重，没有一千也有八百。我应约去安顺和大酒楼见他。

　　见了我，他放自己的舌进我口内，山崩地裂地吻了十分钟，才取出红包。这红包视觉上看起来腰大气粗，大得可以塞进一千块百元面值的人民币。假装上洗手间，我偷偷查实，气不打一处来。二百块打发要饭的还行，拿来糊弄我？我啐。追溯根源，对于别人的承诺，不要当真为好。

　　我故意点名贵菜式：红烧官燕盏、鸡骨炒煲生鱼、木瓜炖鲍翅、生灼游水虾、明炉石斑鱼、竹丝鸡炖水鱼。不替你花掉三千块我就投河自尽！他为我倒稠稠的椰奶汁，嘴巴蹭着我的脸皮经验地说，"你呀，要多吃点燕窝和水果，它们对皮肤的延缓衰老有奇效。我老婆天天吃这些，她三十三的肤质靓过你。我这样说你别往心里去，我是为你好，关心你！"

　　我恼他。坐着说话不明白站着的腰疼，你老婆有条件我可没有。你老婆有你卖命赚钱买燕窝、雪蛤王吃，我哪有啊？你老婆靓过我，为何还要与我动机不纯地坐在这里。男人真是没好东西！服务员端上锃亮堂皇的有着王室风范的银质餐具，左边的纤手背置腰后，稍哈上身尊敬地为我们打开盖子。菜肴冉冉地升起热乎气，我脸朝着气体蒸，这样能够令皮肤汗毛孔伸缩透气。

　　我大智若愚的德性有别样妖骚的美，常孚掀起我的衣服吮得滋滋响。服务员呆板地守卫包房门外，没人叫，她应该不会进来。他抱我，他西裤的前门拉链处硬如卵石，我不挣扎，我不迎合，我恨他小气。

　　"不行，我受不了！"常孚放开我，他进入包房里的洗手间。

　　几分钟后，"放掉了？"我问他。

　　"嗯！不放掉的话，我怕伤害你。"他疲累地说，表情是"五个打一个"的舒服。

　　正月初十，一切迹象表明温情公司的气数将尽。十二个小时的工作周期，居然只有几十分钟至上百分钟的通话量，有的继续爆零分。以前公司每日每人的话务量是四百分左右，相比之下现在经营惨淡。原因多不胜数，许多单位申请声讯呼出限制甚至使用充值卡IC电话机，能打通热线的少之甚少。主持人怪杰百出，八仙过海地各显神通，挂机的情况尤其恶劣。

　　所谓挂机即听众拨通主持人的热线号码，把电话放在旁边，等到一定时间挂断，再重拨过来。声讯行内还有个专用名词叫做蹦机。蹦机就是打通马上挂断，再重打再收线再重打，反复重拨同个号码。有时候一分钟之内打通三十次，主持人就会每分钟之内得到三十倍的收入。这种行为如被发现，热线行业开会做出通报批评，一般情况下极少出现。

　　"小连，求你了，帮我挂几个小时吧！好吗？你帮帮我啊，要不我月底领不到工资，你知道我们靠话务量吃饭。拜托，求你了，好吗？"主持人长虹蔑己人格地求听众挂机。

　　"好啊！行是行，但我有个小条件，同时有个浓缩的小请求，如果咱们谈得好，我给你挂。挂多少个小时我都答应你。"听众连某男人得意洋洋地摆架子。

　　"行，你尽管说！"

　　"你吻我一下，你还得答应跟我见面。"

　　长虹勉为其难地说，"可以，但你千万别告诉别人啊！"她吻了一下空气，听众哈哈笑，帮她挂机。

　　耳聪目明的长虹听见听众使用另一部电话按键，打通同事某某的电话说，"你们台的那个长虹为了求我挂机，在电话里亲我┄┄"长虹气得直跺脚。

　　某某绘声绘色地唱＜两只老虎＞，听众戏弄她说，"你们这里全是女老虎！"

　　如此流气的听众多如牛毛，他们分别在各主持人面前讨好，谁有隐私一旦为他们窥探，热线的同行顺便都晓得了。他们叫做"流窜作案犯"。

　　晚上，我们赶向公司值夜班。隔着远远的一段距离，发现同事们群情激愤地砸公司的门。我凑近房门，上面贴着张字条，有句话-公司经营亏损，即日起倒闭。我脑袋凉丝丝地冒冷气，"别闹了，他们肯定早有打算。不然不会昨天发完工资，今天宣布关门。吵没用的，老板有良心，上个月的工资不是发给我们了吗？"我劝道。

　　"我们发的是上个月的工资。公司有三十个人，每个人仍然有二百块押金被扣，这是一笔不小的数目。我们的钱是怎么来的，别人不知道你还不清楚吗？我建议找劳动局、找市政府、找行管办，我们告他非法经营。做了这么长时间，公司不跟大家签订劳动佣工合同，我们的损失有多大啊！"北京人佼佼翘着卷舌说。

　　"要告你们告，我不参预。"我悄悄拉莫儿，"我们不还住宿舍吗？我们可得悠着点。"

　　莫儿竭蹶地随我离开，她忧心忡忡地说，"宿舍我们也不能长住，你还有个家回，我怎么办？"她眼里雾气弥漫，比较恍惚。

　　"着急也没用啊，我们回宿舍商量吧！以后找其他同事合伙租赁房子也行。天无绝人之路，办法不都是人想出来的嘛？"我心情灰暗地劝她也劝我。

　　十二路车上，一靠唱歌讨钱的壮年男乞丐手拿破木棒，为乘客献艺。莫儿说，"别人年纪大了不能赚钱做乞丐，他年轻力壮，还出来讨钱真是不要脸。"

　　"哎，你没见报纸有关乞丐收入的专题报道吗？这些乞丐每个工作日赚到的钱超过八十块钱，比我们的收入高多啦。我要是舍得下脸皮我也去讨钱算了。"

　　宿舍的门洞开，老房东监视着同事们收拾行李。"怎么回事？不给我们住了吗？"莫儿焦急失色地问。

　　"你们老总已经欠了我一个月的房租。上个月你们居住的是他当初交纳的房租抵押金，这个月你们不搬的话我没钱收，只好请你们另觅住处。"房东给我看租房合同，话说得仿佛他做事做得已经情至义尽。

　　我怎么办？只有回家。我打电话到安华城大酒店，找中餐部的妹妹，"梦帆，我是姐姐。咱们家的情况怎么样了？"

　　梦帆一听是我，立刻哭鼻子，"咱妈可想你了，生病也没钱治疗。爷爷前段时间去世，又不够钱回家奔丧。家里每况愈差，我压根不知道怎么办才好。"

　　泪没有防线地滚滚流下，"我现在马上回家，你别担心。"梦帆告诉我家里的新地址，我把东西一古脑装进蛇皮袋，乘出租车回家。

　　福星路的大门村，我找到了开百货店的家。妈妈见到我格外惊喜，她不像梦帆形容的病情危急，眼神还是那样的严厉，皱纹还是那么的多。爸爸以恫吓的语气怒视着我，"你还回来啊？你心里还有这个家吗？是不是又混不下去？"

　　我放下东西，吃一只菠萝冰雪条。爸爸洞若观火地说，"近期，你爷爷又病故，已于前日殡葬。咱们非但不能凑合几百元办丧事，连回乡的路费都没有。怎么办？必须采取措施力揽狂澜，安排家庭脱离绝境！"

　　我不以为然，"你自己怎么不挣？你的私生女儿为什么要我养？"

　　"你是长女，爸希望你能起顶梁作用。你的责任是创造条件，保证家庭将来的幸福。否则万劫缠身。"他喝着金威啤酒，大声的叹气。

　　妈妈切菜准备做饭，她的声音飘出厨房，"我们希望你找个香港人结婚。年纪大点无所谓，只要有钱就没问题。让有钱的香港人帮家里买套房子，户口迁过来，跳离苦难。上个月有人提亲，过几天，你可以和人家见面。"

　　我慢慢接受了他们的理论，爷爷溘逝的消息不能打垮我，我是陷入了命运的凹道。我总不能任由他们渐渐风烛残年，孤苦无依地等死。

　　很快，年纪大的香港人与我见面。我们坐在福华路的品上品咖啡厅里谈话。看看我身边这个苍老有钱的香港人吧，经过植种的头发不真实地贴着脑门，无法伸直的双腿。他担心我的美貌年轻，似狐狸狡猾地说，"我给你们家买完房子，万一你不愿意再跟我好怎么办？那我的损失可太大啦，三十几万的房屋不是小数目。"

　　他想得细致入微。她年轻漂亮，陪伴着我老人家能守住寂寞吗？我的性能力已经微弱，我满足不了她。她如果红杏出墙我无法抑止，这些必须考虑的问题恐怕得不到妥善的解决。他异常清醒，智叟一样地审视着对面的女孩，目光炯炯。

　　我气哽了，嫁给他以后红杏出墙，是毫无疑问的可能。然而，钱是致命憾缺，先搞定他，将来会有办法料理。我非常伪善地保证，"我绝对可以做到对你对婚姻忠实，只要你给予我家里提出条件的成全。"

　　他详细地打听我的工作、健康、处女与否、怀孕与否、家庭安排、学历、性要求频率、个人心理卫生的情况。我尽量耐心地回答任何内容的提问，要知道这次相亲的结果太重要了。

　　分手后回到家里，二个香港人带着谈好价钱的妓女买我们店里的避孕套和毛巾、牙刷、牙膏。妈妈五块钱一盒进货的避孕套卖了十五块钱，她笑逐颜开地找赎。我鄙夷地绕过他们，忽地想起，南南说她有一次经过公司写字楼，上面突然有人扔下一只使用完的避孕套，里面盛有新鲜精液。

　　妈妈追问下来，我实况转播了香港人的状况。妈妈挥着手臂擦试冰箱，她声声锵铿，"别嫌人家老，只要有钱就成。爹有娘有不如自己有，只要有了钱，你可以找小白脸，他那么老也管不了你。现在当务之急就是弄到房子，懂吗？梦玫，你可得长心性，别再犯傻。什么是感情？有钱就有感情，就是这么回事。你也二十多啦，趁着眼前的光景多赚些钱，以后想做什么做什么┄┄"我决定打个电话给常孚，最好是他买房给我，我必感激他一生一世，来生愿做牛做马衔环以报。我打他手机，结结巴巴地开口，"是我！我爸妈想把我卖给香港老头，你能不能拿钱出来供套房给我家？"

　　"不行，我想送女儿到法国学跳芭蕾舞，这需要很多钱。这事我帮不了你。"

　　我呆呆地坐着，手提电话的信号中断，电话筒发出刺耳的叫声。男人果真靠不住。

　　尽管我委屈求全地出售良心，结果仍然令人心力交悴，香港老头转告媒人说我太漂亮，他不放心，为免生事所以弃权。我转眼成了家里吃闲饭的人，爸爸的脸色比较近似于晚期癌症病人的难看。我得找工作，得贴补家用。
