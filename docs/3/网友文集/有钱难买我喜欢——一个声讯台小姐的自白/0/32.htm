第三十一章 我一对一的主持工作极不成功


　　天转冷，我晚上常睡不着，暖不热被窝。我诸事不顺，一对一的主持工作极不成功，我意志消沉，几乎要放弃了。范爱民则像棉被一样温暖，我伤心落魄时总是他陪我。他一直觉得我和华东南已经分手，但我爱华东南，我不爱他。

　　又有新同事，我有感而发，"电视上看到经理介绍新同事时老员工会鼓掌欢迎，为什么这种场面在热线行业没有呢？"

　　陈红奚落说，"因为我们这里换员工太勤，鼓不过来掌┄┄"她话音刚落，电话响了。听众要求陈红唱歌，"我在马路边，拣到一分钱，把它交到警察叔叔手里边，叔叔拿着钱对我把头点，我高兴地说了声叔叔再见┄┄"现在马路上丢失的一毛钱有无数人踩过，却没人认为它值得耗费时间拣，我看歌词应该改成，我在马路边拣到一块钱。

　　"乐乐，你觉得浪漫可靠吗？值得捕捉吗？"女听众KK傻乎乎的问我。去康宁那天我带着她，她吓得逃到洗手间嚎啕大哭。

　　"感情只是一种感觉，浪漫的氛围是人创造的，假如不后悔，就去做追求吧！"我相对而言。

　　"我想我是个挺蠢的女孩，我忘不掉他。"

　　"别傻了，KK。也许他根本不爱你。他既然出了国，只能忘记他！"KK大学毕业后供师兄男友继续攻读硕士学位，没想到男友考托福出国三个月就为了取得绿卡提出跟她分手，将要娶当地华侨的遗孀。

　　"不，爱不需要任何条件对不对？我打国际长途电话给他，我要个分手的理由。他告诉我，他连面包都没得吃，哪里敢奢望拥有玫瑰。"

　　"也不是啊！人家穷得几枚硬币叮当响，我连硬币都没有。我一样追求爱情┄┄你别再找他。一段感情当你为它所累的时候，是欲示着分离。他的理由意味他必须放弃。"和KK说着不老的感情话题，我看着华东南的相片泪串串滴落。

　　她泣不成声，"分手的理由纯粹是借口，在学校里我没嫌弃他穷┄┄如果不是互联网，如果不是一网情深BBS和你们热线台，真的不敢想像我怎样走过这样的一段泥泞路。我认识了好多一网情深的网友，他们对我很好，劝我忘记昨天。现在我拼命复习考托福，还不是为了让自己坚强地站起来，让他后悔。"

　　当一个女性为情所困，她什么也不会做只是哭。电话线里长久地沉默着，我不知说什么，用随身听播放那首郑中基的《爱是如何伤了你我的心》。

　　她心死如灰地说，"听说他快结婚了，也不知道应该送什么礼物。也许更好的办法是什么也不送，躲得远远的。我告诉他我也要去美国，他说大家以后不要故意在容易撞到的地方出现。他做得到，我做不到。我想知道他过得究竟好不好。得不到我只好祝福，或者那个遗孀可以给他一切┄┄"她鼻道阻塞，发音不清。

　　思想混乱的KK知道做任何事情也于事无补。即使KK知道他爱自己，但为了现实，他只能选择另外一个女人。网友也说他根本不爱KK，但KK知道他爱。只是爱得苍白无力，爱得懦弱，爱得无奈，爱得令人伤心，爱得令人心碎。

　　痛并快乐地活着，这就是生活的本质。

　　天天听这些陈谷子烂芝麻的话主持人非常疲倦，我们被听众当作垃圾筒，听众倒完废旧物资拍拍屁股扬长离去，我们还要处理消化这些垃圾。

　　马晴晴把一本书举过头顶问我们，"你们知道英语教材是谁编的吗？是朱元璋。"

　　我们起哄，"明明是许国璋英语你偏偏说成朱元璋，还英语高材生呢？好在你没当老师，不然肯定误人子弟。"

　　棋子指着马晴晴的脸笑，"这还不算什么，她那天告诉听众说澳门有零点一四公里那么大，人家听众纠正说是二十多公里┄┄""这算什么？我告诉你们，我决定撰写声讯体裁的小说，我会一举成名的。我现在是拿听众做试验、找灵感，所以才胡言胡语。你们这些吹毛求疵的门外汉不懂┄┄"亢奋的她嘴角明显有激动的白色唾液沫。

　　我踔厉风发地看着马晴晴，一针见血地指出她工作的不足，"你对听众太坦诚，听众制造扣人心弦的紧张让你为他们着急为他们担心，事实上什么都没发生┄┄"马晴晴气愤地撕了些纸巾走出话务间，她生气的时候有点尿频。

　　中午，张严拿起盒饭发放，刚巧撞见陈红躲在房间换裤子，张严指着她骂，"你他妈的什么东西，这是办公室你知不知道？在这里换裤子是对大家的不尊重，你给我滚出去！"

　　"你跟谁他妈的？你他妈的你！换衣服怎么不行了？厕所脏成那样怎么换啊？我怎么不尊重大家啦┄┄"陈红提着裤子硬着头皮尖声说，她有些亏心。可当着大家的面骂她，太没面子。

　　张严骂骂咧咧地冲向陈红，陈红口里的唾液不甘示弱地喷出，颇有斗殴经验的当胸二拳头，陈红倒地了，他仍跺她的小腹。我们扯走张严。男人打女人真不象话。陈红也不上班，拎起手袋下了楼。下班后，张严回家必经楼下的汽修厂，陈红的男朋友擒块红砖当顶砸到了他的天灵盖上。我再也没有见到张严，听说他成了医院的常客。陈红的男朋友有钱，药费不算什么。

　　班主任通知培训班的学员考试日期。考试那天我乱了阵脚，我根本没好好背老师划的重点内容。试卷发下来，我慌得很，邮电局的六不准规定答不出，同桌紫绛悄声告诉我，我自由发挥地答了一通。聪明的同学去洗手间翻查资料后继续回考场答试卷。我忍俊不禁，不料想人长大作弊的手法还是没有长大。

　　以知识为基础的经济叫做什么经济我不会答。

　　1、你认为做为一个合格的心理咨询员应具备的条件是什么？2、根据口才的基本要素，你觉得应该怎样答询求助电话？3、如何处理无声电话？4、试用点、线、面之间的关系阐述成功应具备的条件。5、电话咨询中有一位二十几岁的女孩子很喜欢她的老板，可老板是个有妇之夫，她明知这样做很烦恼却挣脱不出你会怎样劝解她？这五题简答题我发动脑细胞完全靠临场发挥。

　　监考老师欲看我的试卷，我压住试卷不好意思给他看，"老师对不起，我忙着另外的事情没来得及背诵讲义。"

　　老师有点失望，不太相信的笑着说，"凌子，你应该是考得最好的一个。我对你的期望那么大┄┄"老师说文科考试多靠自由发挥，只要不空题就能得分，我急忙填上空白的地方。

　　关于电话信息咨询中常见条例所持的基本观点我正确答出了：1、倾听、概括、释义、答疑，适当指导，2、根据年龄段导引人生价值观，3、咨询中经验性的咨询、工作压力、人际关系、情感及健康。这题我得了满分10分。不久，我拿到了广东省心理学会、广东省心理卫生协会和心理咨询专业委员会签发的培训证书。

　　夜班，性激素分泌旺盛的听众问"有永远的爱吗？所谓天长地久的爱难道不是习惯吗？性是种行为意识吗？这种行为意识包括许多种，例如接吻、拥抱、拉手。那么拥抱和性行为一样吗？"

　　"我不知道。"我工作得非常吃力，对听众的电话感到不耐烦，唯独下班的时候高兴，我觉得自己干不下去了。

　　"我面对所爱的人有欲望，怎么办？"

　　"调节，不要为痛苦活着！"我用自己的论调影响他。

　　"你所指的调节是自慰或者手淫吗？"他的这个问题点中我的死穴，我的自慰次数频繁。

　　"请你打160查询性咨询电话，我们这不谈性┄┄"我打发他挂电话。我打开班主任推荐的费英秋的《素质的透视与测评》飞快地摘抄。我自己的问题亦没得到妥善的解决，我怎么帮助听众。

　　下了夜班又没完没了的开会。

　　"热线主持人要站在听众的角度看问题，语气中恳、语调平和、语言要美、语汇丰富、吐字清晰、情绪饱满、热情大方，温文高雅┄┄具备这些条件你才有资格和听众对话。"黄英敲着桌子大发雷霆。

　　昨天有听众投诉到180，说白雪情感抚慰电话的主持人辛儿辱骂听众。她的口头语多，听众特别反感。她的下场是卷铺盖走人。

　　考试通过，班主任组织我们到培训中心参加联谊活动。声讯界的主持人可以自由带朋友参加。我和范爱民坐了一会后转去旧金山喝酒。

　　来得多，旧金山的经理认识我，送了我一张八折卡。我点了炸薯条、香蕉船、水果沙律、滨海羊肉串烧、热柠檬荼。范爱民要了雷博士龙舌兰和红色金门桥。我让服务员拿来一瓶蕃茄酱，用不锈钢叉子挑着薯条沾蕃茄酱吃，我爱吃酸的。

　　范爱民知道通过买单表现对我的关心，我很感激他。可他像是被妈妈改良得没有个性的男人，我说咋做就咋做，男人怎么能够这样？我等着他牵我的手，他如一潭任春风吹拂的死水，怎么也荡不起波纹。

　　"二千年的上半年，假若我没找到愿意娶我的人，那我嫁给你好吗？"我期望的看着范爱民说。他着装一贯的板板正正，不苟灰尘。他手夹玻璃酒杯的颈，缓慢有条理地晃悠。

　　"好的。"他简短地说。他讲话总是迅猛出口，然后全身而退，静如处子。

　　坐了一会，我提议买单。范爱民想打折，我说，"别，一二百元打什么折，费事。"

　　我们拦的士去书城买三毛的小说。公共汽车的喇叭里成天广播的东南亚最大的购书中心居然只有一本三毛的《万水千山总是情》，再也找不到多余的，范爱民说三毛的小说现在没人看了。

　　他还是买二瓶鲜奶，送我上夜班。我看不懂他想些什么。

　　 华东南夜里打电话给我，我睡意立消，"是不是喝多了？"他天天玩到很晚才回宿舍，打电话给我常是凌晨一二点左右。

　　"宝宝，今天我没出去，躺在床上睡着了，连晚饭都没吃。觉得口渴醒来，想起你值夜班，所以打电话给你。"华东南嗡声嗡气的说。

　　"我想你了。"我拉紧毛毯。天冷，上夜班同事们都披盖厚重衣物。

　　"我刚刚做梦，梦到咱俩下地铁。我手上抱着一个孩子，不知道是不是我儿子，反正梦里面我至始至终抱着他。你很着急地说手机丢了。我帮你找，可怎么也找不到。从后面追上来一个年青乞丐，他拿着一部手机问是不是你丢的。你接过来看正是你丢失的那部，你很高兴地欢腾跳跃，突然亲了乞丐一下。乞丐愣在那里，摸着脸。我抱着孩子转身走了，然后就醒了！"华东南用一种伤感的语气叙说他的梦境。

　　"那证明咱俩会有一个孩子，而且是个男孩！"我开心的说，把脸靠在棉坐垫上，心莫名激动着。

　　他好象怕事实如此，否定说，"不，梦是反的，说明咱俩没有孩子。"

　　"我已经是你的人了，如果我真的怀了你的孩子，你打算怎么办？"他没吭声，我没有指望地说，"我知道肯定是凉拌，弄死孩子是吗？"

　　"才不会呢，我没有这么狠心。如果真的有了孩子，我们就生他下来，其实我挺喜欢孩子。孙丽可能生不了，她和旧男友打过三次胎，她还有严重的妇科病，医生说她挺危险┄┄"爱别人痛苦，被人爱幸福，被人爱同时爱别人美满，可我和华东南相爱并不美满。

　　我犹豫片刻，"亲爱的，我们俩在一起后，你和孙丽有做爱吗？"

　　"没有，真的。我们已经半年多没有了，她没有什么欲望。你信吗？"华东南问。我发现他精神茫然，提到孙丽和以后他都这样。

　　"我信啊！"我附合说，底气不足。回顾做爱时他的狂热我有点相信。

　　"其实我不知道未来怎样，我挺没用的。你嫁给我说不定会后悔┄┄"他不开心的讲。

　　"放心吧，即使你一无所有，即使天底下没人愿意要你，她抛弃了你我也会要你。我会陪着你。只有你抛弃我，没有我不要你的┄┄"我的温柔纤细无遗。

　　"孙丽也这样说过┄┄"他的话让我面如灰烬，怔在那里。

　　早上下班，艾然有事先走了。我左顾右盼地等候九路车，开来一辆冷巴，我不情愿地上了车。我不爱坐冷巴，因为冷巴贵一倍钱。一位年迈老人上车，我准备起身，没轮到我表现，已经有人让座给老人，我抿唇暗笑。好人还是比坏人多。

　　一个轮休司机和新调来的驾驶员说话，说到冷巴车将要替换普通巴士的时候，我冷不丁地插话，"你们九路车是怎么回事？这么冻的天居然全部发不开空调的冷气巴士。不开空调也就算了，干嘛还照常收费？"

　　"这个我们不清楚，司机只管开车。票价是市政府定的，你有意见可以打市长专线反映，跟我们说没用。"轮休司机舔着嘴唇说。驾驶员通过倒车镜别有用心地望了我一眼。

　　"打电话反映了几次，可这种情况还出现。你们把普巴与冷巴相结合轮流发车是不是？也让乘车的人有消费选择。我们不坐随叫随停的中巴，拼命挤巴士还不是为了省钱？可你们硬发冷巴，我们不坐也得坐，这不是强奸民意吗？"我气头上，强奸民意四个字讲得全车人都听得见。

　　两个司机还解释，左侧一个肥头大耳的中年人头也没抬地说，"吵什么吵？想反映去市政府，别扰乱公共秩序┄┄"我和司机齐齐住口。我肚子里骂他，我想他要是不再出声我就憋住不骂他。

　　谁知他又加上一句，"吵了一早上。"

　　我立刻尖锐的反击，"我反映问题怎么了？这事关系大家的利益。"

　　"要反映你找市长啊，别吵大家的清静。"他加高声音说。

　　"我从最基层反映不对吗？你讲公共秩序吗？刚才那个阿婆上车后站在你身边，你装作没看见，以为我不知道。恬不知耻的说我┄┄"我一脸瞧不起，声声指出他的致命点。他没再讲话。到216总站我下了车，他出其不意地对我吐了口痰，"啊，呸"。看看，这种人怎么不死呢？东南西北死的净是好人，如今坏人长寿。

　　我梳着湿湿的头发交给梦梦一块钱，吩咐她买两个北方馒头，她乖张地伸出小手，"我帮你买也行，你得给我一毛钱的小费。"妈妈开的百货店需要送外卖，不重的货物都由梦梦送去，送一次付一毛钱小费。她放进蜡光纸折叠的存钱罐，天天晚上睡觉前象个守财奴地数一遍。

　　我哑然失笑，按着她的头说，"你攒多少小费了？"她拿来存钱罐一古脑往书桌上倒，好大一堆一毛的硬币。

　　"有十二块五毛钱！"她神色十足。

　　行，比我强。我二十多了还没有值得自豪的存款。我苦楚地看着存折上的二千元钱辛酸不已。这是我紧衣缩食两个月的结果。我决定买一部586电脑，价格不算贵，差不多一千八百元。早在温情热线时，我认识了广东人东仔，东仔是观澜本地人，他上个月开了间网吧，专门承接装机、维修电脑之类的生意。我换好衣服，传呼他过来市内帮我装配电脑。我第一次有欲望见他，毕竟是有求于他。

　　他果然貌不惊人，侃起电脑网络却头头是道。为了照顾我的钱，全新的586电脑他收了我一千五。其实我很希望他别收钱，白送得了。这年轻的小广东不会做人，每个月交六百块雷打不动的声讯费，区区一千来块还计较。

　　东仔稍稍讲解了电脑的有关知识，我迫不及待的开始逐步实习。九六年，我参加过皇岗村的电脑培训班，好歹有点基础。我很快成为了痴迷的网虫。我经常泡互联网，KK告诉我的BBS上日日可见我的ID乐乐（LINZI），我动不动就呸聊天室的网友，他们评价我的口水特别多。我去吧板发表了许多随心所欲的文章，全跟华东南有关。人寂寞时总会想办法找节目娱乐，我则借网络分散自己的注意力。

　　同事们听我说网上可以认识数不清的素质超人的白领男人，她们想见识见识，休息的时候来我家。我教她们上网的基本步骤和操作程序。网络的新奇感远胜热线的枯燥无味，她们目不转睛，连上几个小时舍不得下线，把对听众的热情转移到了网友身上。后来，她们干脆泡网吧，上班迟到的情况日趋严重。独树一帜的马晴晴托我在ＢＢＳ的跳蚤市场买了部二手486，正式起草关于深圳声讯行业主持人喜怒哀乐的描写。她说自己是将爆出内幕，以便全世界的人认识热线主持人。

　　我往电脑前一坐，几个小时的光阴便不翼而飞，用96340拨号网费加电话费每个月至少四百元，交时电话费心疼如割。过了那阵子，我依然乐此不疲地坐到电脑旁，手指劈哩叭啦地敲字。我跟听众谈的聊的也全是BBS一网情深，我爱一网情深这四个字。
