　　事情离奇。那名女听众拨打夏天的热线已有三个月的时间，她是某集团公司的会计总监，常利用集团的几十部电话给夏天挂机。她坦白布公地讲自己的不幸婚史，一个单身女人的艰难拼搏使夏天流泪。夏天心太软，怕伤害她就应邀见了女人。女人开着公司的凌志车常接夏天下班，几次小恩小惠之后，带夏天到新买的第四套房子度周末。进入房间，女人调了多种鸡尾酒给夏天喝。夏天醉到将要酩酊时女人宽了夏天的衣，也解了他的皮带。法式大床上，女人采取女上男下的方式强奸了处男夏天。这个女人变态地怀孕了，非他不嫁。

　　想像夏天用他那不标准的江西普通话喊救命，我头晕脑胀。

　　下班前，公司发放工资存折。我有一千六百五十八元。在公司话务量的排行榜上位居探花，九千多分；状元是欧阳，一万三千多分；榜眼是阳光，八千多分。

　　下班的第一件事是取钱。从建设银行出来，我直接跑到万佳买送给华东南的礼物。我买了两包底裤、附送光盘的《看上去很美》，还找到齐秦九八年开西藏演唱会的碟。邮电代办所的工作人员说影碟不能寄，必须有文化局的证明。于是我去花店订了十一朵红玫瑰，配上满天星、情人草、勿忘我，连同买的东西装进漂亮的包装袋，花店的人马上按照地址送去了。

　　我这人不存财，有点小钱非得花光，才能够睡着觉。我进柏林名店买了条自由鸟的白色休闲裤，一件粉红色的短袖T恤。逢华强北路的百丽鞋分店打折酬宾，我花二百八十元买了双乳白色凉鞋。听说国外已有知名模特跌死于这种松糕底鞋。

　　回到家我给了妈妈五百元生活费，她乐不可支地数了二遍。
