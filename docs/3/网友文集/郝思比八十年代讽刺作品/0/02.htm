无名英雄


　　随着社会主义精神文明建设的广泛深入开展，好人好事如雨后春笋般层出不穷。无名英雄也如夜间的朗朗繁星，令人目不暇给。大前天，报上"新风赞"一栏里刊登了一桩这样的事迹，看后真感人肺腑，摧人泪下。

　　市手表厂职工维君星期天正好歇班，在家里既无事可做，又无朋友相伴，只好独自一人到公园来玩。他把船划到湖中央，仰面朝天，悠然自得地抽起烟来。

　　正在这时，一只小船乘风破浪地向他冲来。维君从眼角里瞥见，船上坐着一对夫妇和一个儿童。维君并不着急。这种颇具惊险情趣的游戏他已干得不干了－－把船划到对方船前时，再来个急转弯，使对方大吃一惊，自己却从中得到无穷乐趣。所以他这样不慌不忙地抽烟自有他的道理。果然正如他所料，对方的小船快到他船前时灵巧地掉转了方向。由于船急转弯时产生过度的倾斜，坐在船舷上的小孩不能与离心力抗衡，就一头钻入湖中。维君以为这孩子要练习游泳，所以当时并没有在意。当看到孩子父母呆若木鸡的样子时，他才知一场事故就在眼前。救死扶伤是革命的人道主义。他未加犹豫就跃入湖中。

　　我们这位英雄谙习水性，毫不费力地就把孩子救了上来。孩子父母本该先打听英雄的姓名、住址、工作单位，继而感恩载德才是。但看到孩子鼓着肚子，像一个喝多了啤酒的醉汉，他们自然就先照顾孩子去了。我们这位英雄就这样被冷落到一边。维君刚刚从水里上来，心嗵嗵只跳，又对救护工作一窍不通，只好转身走出公园。

　　刚刚上岸，他本想对孩子的父母做番自我介绍。但如果那样去做，岂不显得自己道德水准过于低下？无名英雄从来都是不透露自己的姓名，便悄悄离去的。

　　回家的路上他想："如果当时有一位熟人在场该有多好！自己悄悄离去，既可显出品质的高尚，又可当回无名英雄。可惜呀，想象毕竟不能代替现实。"正想着，迎面走来一位同事和他的女友。

　　"维兄，这么早就回家呀！"

　　"嗯......"他摸着头发，一时不知如何回答。心想：这位同事看到我身上湿漉漉的，一定会问起缘由，那时我就装作不好意思的样子把事情原原本本地告诉他，那么"无名英雄"的光荣称号也就......但没等他再说什么，那位同事已拉着女友匆匆忙忙地走了。

　　他摸摸头，还有点湿润；又摸摸衣服，都快干了。他顿时又气又恼，一股无名之火不知从何而来。心想，一路上若遇不到熟人，或遇到熟人他们又不过问，那对夫妇又忘恩负义不去打听他的姓名，他的壮举岂不就此埋没？他真想返回公园找那对夫妇。当想到"无名英雄"的真正内涵，他就打消了这个念头。会到家里，他一头扎到床上，不吃不喝，像得了什么大病。第二天，还真地发起了高烧。父母为他找来了医生，但什么药都无济于事。

　　当厂领导得知他得了重病前来看望时，他吞吞吐吐地说："或许是救那个落水儿童时着凉了......"

　　厂领导听后大吃一惊："原来报上寻找的那位'无名英雄'正是你呀！为什么不早点和厂领导说呢？我们厂为有你这样的职工感到骄傲呀！"

　　第二天，报上就以"今日无名英雄"的大字标题刊登了他的事迹，并配发了照片。他的病也奇迹般地好了，接着就上班去了。
