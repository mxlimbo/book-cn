便宜货


　　待业青年小刘在家里呆着没事可干，三个月前让父亲托人到工商局办理了个体营业执照，来到青年街集贸市场经营各式服装，一直买卖兴隆，每天都能售出十几件衣服，获纯利四五十元。以此看来，一周后他就可以把商场那台进口冰箱弄回家了。但近一周来却行情不佳，几十种款式新颖的服装高高悬在这里，愣是无人问津。小刘经营还比较遵纪守法，从没有强拉顾客的习惯，因此只得眼巴巴地望着狭窄的人行道上的湍湍人流，内心独自兴叹。心想：自己在商场一无亲戚，二无熟人，唯一的那台名牌冰箱肯定要被别人弄走了。

　　这时从东边走来一位姑娘：黑黝黝的脸膛，爆炸式的头发；至于穿着，虽谈不上入时，却也能明显看出是由上乘料子做成。但姑娘肥硕的臀部和笨拙的步态，使小刘一眼便看出她从乡下而来。他暗自思忖：农村姑娘都打扮得这样漂亮，还是党的富民政策好啊！

　　姑娘在小摊前站住了。小刘忙客气地问道："师傅，您要什么？"

　　"你也真是！"姑娘嗓门既高又粗。"我又不是男的，你叫我什么师傅？你见电影、电视上什么工厂里哪个师傅不是男的？即便我是男的，你就不看着我年轻？"

　　"好吧。您要什么？"小刘忍住笑说。心想，这姑娘怪厉害的。

　　"让我看看吧。"姑娘摇晃着身子说。她瞧了一会儿，说了声"没好的"，就转身走了。小刘只有暗暗责备自己："眼看着一条肥鱼就要上钩，却将它放跑了。"

　　小刘正无精打采地斜在柜台上抽烟，姑娘又一溜烟地兜了回来。

　　"唉，我转了一圈，发现只有你这件衣服还差不离儿。这么大的地方就发现不了一件合适的衣服，你说怪不怪？嗨！你这件多少钱？"

　　小刘听了姑娘的话，心里一亮。他看看衣服，又看看姑娘，于是随口开了个价格："按进价给你，六十元整。"

　　"什么？你说什么？你当我不知道？我的一个亲戚就是卖这玩艺的。我说你呀，你也不看看自己在骗谁？"姑娘出言不逊，好像早已意识到要敲她的竹杠。

　　小刘第一次要这么高的价钱，而面对的又是一个不知客气为何物的村姑，因此脸上不免感到热辣辣的。心想：别看这姑娘土里土气，可见识还不算少。早知如此，实在不该要这么高的价钱。

　　"你才骗不了我哩！"姑娘见小刘不吱声，又接下去说，看你这衣裳是什么料子呀，最多也不值五十五元！"她显出十分老练的样子，把价格向下狠狠地杀了五元。

　　"是......就是五十五元。你怎么对这件衣服的价钱知道得这么清楚？这可是进价啊......"小刘不知为何说话竟有点吞吐。

　　"比这大得多的国营商场我都转过不知多少次了，你还能骗了我？别以为你们城里人聪明，我们比你们一点也不傻！给你五十五元算了！"姑娘如数地扔下钱，扭动屁股走了。她昂首挺胸，嘴里好像还哼着什么曲子。

　　"赚了她三十！"小刘对着姑娘的背影做了个鬼脸。姑娘好像听到了这句话，就回顾了一眼。但她那十分自信的表情说明，她绝不相信是在说她。她坚信那有点冒傻气的小伙子绝没占她的一点便宜。
