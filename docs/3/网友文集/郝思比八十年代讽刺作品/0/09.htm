我的班长


　　以前的每个星期日对她来说都是快乐的一天。她要么到其它院校串串同学，要么和同乡们玩玩扑克，再要么就干脆躺在床上读一本言情小说。总而言之，星期日她很少呆在教室。

　　外院校的同学邀她前去玩耍，她已答应人家，说无论刮风下雨都要去。可这类的事情我们常常遇到：计划内的事情往往被一件骤从天降的事情代替。现在的情况正是这样。可怎么办呢？如果失约，一定会受到她们的责怪。她只好写封短信，说明情况。然而当她提起笔时，却不由得在纸上写下了如下的文字：

　　"谁若仅凭这些文字就认为我是那少有的好心人、即将被追认的共产党员、我们的好班长的走卒，或说我这个未涉爱河的姑娘和他之间有什么罗曼史，那么我敢说：他即使不是专营造谣中伤的买卖，至少也是个胡乱猜疑之徒！

　　"我所以给学校广播站写那篇题为《我们的好班长》的广播稿，我所以给校报写那篇《我们身边的英雄》的悼念文章，是因为我们那可敬的班长的所做所为着实令顽石点头、铁人流泣。

　　"大家知道，我本不是一个性情柔弱的姑娘，可是很多次，我却被我们的班长感动得潸然泪下。

　　"那次我发高烧，几日粒米不进。当时同学们正准备期终考试，很少顾及我的病情，就连那几个平素和我要好的女同学也包括在内。是我们的班长，对我象对亲妹妹一样关心；是我们的班长，对我象对小学生似地无微不至地关怀；是我们的班长，对我象对孩子般细致入微地照顾。他安慰我，他体贴我......更令人感动的是：在期终考试的前一天晚上，我高烧四十度时，其他同学都到教室复习功课，他却一直在床边陪着我......"

　　写到这里，这位冰清玉洁的姑娘停笔思忖起来。她认为在这篇文章里，应把自己的肺腑之言和盘托出，保留一星半点儿都是罪过。自己对他的认识，对他的想法......如果有必要，对他的感情也应写入其中。虽然她自知写作水平有限，可仍然觉得：如果对班长的牺牲抱以冷漠的态度而不洒一掬同情之泪，自己就有负债累累之感。她重新提起笔来。

　　"可现在我们敬爱的班长与世长辞了－－他为了救一儿童的生命而勇拦惊马壮烈千秋了！"

　　写到这里，笔开始在她手里颤抖，她如丧考妣地失声痛哭起来。抬头环视教室内别无他人，就愈发觉得此地是一个以泪相悼的好地方。自从认识了他的伟大，她已把他当成了偶像来崇拜。

　　她继续写了下去：

　　"鲁迅先生长歌以当泣来悼念刘和珍君，我现在也用这种办法来悼念我们伟大的班长（我丝毫不觉得伟大有夸大其词之嫌）。"

　　写完这段文字，她忽然感到自己的感情变得莫名其妙起来。这种奇怪的东西从心中冉冉升起，通过蒸发到了脑海，然后由大脑神经末梢传到了右手。"能把自己最隐秘的感情写上去吗？能当众公开心底的秘密吗？我不能写上去！ 我一定不要写上去！别人会说我有神经病的！" －－她虽然这样警告着自己，可手中的笔却象失舵的船一样失去了控制。

　　"要说我对他有什么私人感情的话，那也是现在才有的。这完全是由他的伟大引起的，此外别无它因。

　　"我已经深深爱上了一个死人，爱的程度连我自己都难以置信。别人一定会象嘲弄一个呆子一样嘲弄我。难道一个死人就不值得爱吗？难道......"

　　这位小姑娘想向读者提出一连串的提问，以便让读者知道：她爱他并非偶然，而是情理之中的事情；她爱他并非咄咄怪事，而是天经地义。她希望读者完全不必大惊小怪。但如何才能使读者哑口无言，和她统一认识呢？

　　翻了翻写下的稿子，还不到三页。她本打算写七八页、甚至十几页的－－至于到底写多少，象弄不清自己有几根头发一样，她也弄不清。反正事情是这样的：如果看她的稿子，得半小时的时间；如果广播她的稿子，得一顿饭的工夫。一言以蔽之：韩信将兵，多多益善。凡是涌上脑际的对烈士的赞美辞，她都来者不拒；废话也得视其"废的程度"加以考虑。她搜索枯肠，约莫过了二十分钟，还是没写出一个字来。

　　她忽然想起班长常常以日记自励。在他的日记中，一定能够发现英雄思想深处那熠熠发光的东西。在同学的帮助下，她很快找到了烈士的日记。啊！一股无法抑制的喜悦顿时涌上心头。她想，自己从中一定可以发现他那高尚的思想经历了怎样一个"钢铁是怎样炼成的" 过程。她迫不及待地打开了日记。令她失望的是，烈士牺牲前刚刚换过日记本，上面仅有他的最后三篇日记：

　　四月十五日

　　明天就要考试了，同学们都忙于复习。吃过晚饭，我到宿舍看她。

　　"这几天你一直对我精心地照顾，我对你真......"她泪流满面地说。

　　"我认为，男女同学之间除了爱情之外，还有友情存在。为什么男女之间就不该有纯洁的友谊呢？"

　　"可不是，家里的亲哥也没对我这样好过......"

　　"我觉得你太可爱了。我没有妹妹，你就象我的妹妹一样，可你知道我已有女友。这也正是我们的友谊所在啊！"

　　她听后哽咽了半晌。

　　四月十六日

　　我本来以为会很顺利地答完考卷，谁料到今天的考题竟这样难以对付！多亏优越的地理位置救了我的命啊！

　　四月二十六日

　　这次入党我的竞争对手可真不少！我承认，在某些方面我不如他们。这就要求我必须在其他方面胜过他们！

　　如果一辆受惊的马车正向一个在路上玩耍的儿童冲去，那我就毫不犹豫地去当第二个欧阳海！－－只不过"同曲异工" 而已。欧阳海为了什么，我不知道。但我是为了入党！那个傻丫头生病时被我的假言假行还感动得直哭呢，她又怎么知道我的目的呢！

　　至此，这位小女生一下子看清了英雄真面目。然而，还有很多人继续把班长当英雄崇拜呢！
