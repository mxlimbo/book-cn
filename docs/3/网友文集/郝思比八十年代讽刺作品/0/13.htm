助人为乐的姑娘


　　暑假的某日，烈日当空，天气酷热。我独自一人踏上了石家庄到北京的列车。虽然车内的旅客嘴巴对着嘴巴互相进行人工呼吸，当由于我天生脑瓜子聪明，提前两天买票，还是谋到了一席之位。

　　车到保定站，周围的几个人纷纷下车。然而一群男女已在车下前蔟后拥，早做好了争座的准备。

　　一位老者来到我面前。只见他满头银霜，步履蹒跚；手中拄一拐杖，肩上挎一褡裢。一看便知他从乡下而来。他问我身边的座位是否有人。心想：和这样大把年纪的一个人坐在一块，别人不把我们当做父子才怪呢！我说那人马上就来。老头无奈，只好呆在一边为我站岗放哨。半小时刚过，老头就有招架不住之势：双腿瑟瑟直发抖，上身摇摆似筛糠。见此，我忙拾起一张过期报纸，煞有介事地看了起来。

　　"老大爷，您坐这儿吧。"一个女性甜甜地说。

　　虽然老头再三推辞，姑娘还是将座位无私地让给了他。周围的人们自然对她投以既敬佩又赞叹的目光。我也顺便向她瞟了一眼。我第一次被一个姑娘的容貌吸引住了！没见过仙女的人对她的丰姿实在难以想象。我暗自后悔：我为什么那么早地去谈女朋友呢？

　　看到她站在走廊中间，一股对她的同情油然袭上我的心头。我的舌头象被鬼使神差，在口中完成了一系列动作：

　　"姑娘，您坐这儿吧。这里没人。"

　　"谢谢！"仅此一句，我便觉得一股暖流窜遍我的全身。女朋友吻我时也没这种感觉。

　　"你刚才看什么来？"姑娘口中发出的每一个字都令我心花怒放。

　　"......一张报纸，上面警告人们要警惕小偷......"

　　话一脱口，我就感到后悔。我真想再解释一句："我可没怀疑你......"但是，一言既出，就驷马难追了。

　　"是啊，如今小偷猖獗，我们可要提高警惕啊！害人之心不可有，防人之心不可无嘛。"

　　多么有见识的一位女性！单就她看来，"女人头发长，见识短"这句话实在应该清理出中国语言。上车时女友还嘱咐我要千万注意身边的东西，真是杞人忧天！

　　这时，只见姑娘双目交睫，昏昏欲睡。车快到北京了，还不能和姑娘多搭几句话。若白白错过这大好时机，只能后悔终生。我正想着，姑娘的头开始随着车厢的颠簸，有节奏地向我的肩上倾斜。她每向我靠拢一次，我的口中就象被人放进一勺蜜糖。此种甜美，前所未经，无以言表。说来也怪，刚才我还想着中外的一打电影名星，刹时把她们忘得一干二净了。只有和她在一起，我才真正体会到了"光阴似箭"这句话的真正含义。爱因斯坦的相对论也在此得到了极其完美的体现。我只觉得眨了一次眼，呼了一口气，北京站就到了。无名的惆怅顿时向我袭来。

　　姑娘刚对我说完再见，我就不由得和姑娘一同站了起来，又一同离开坐位、一同走下火车、一同穿过地道、一同挤出站门。仅此不算，我还失控地随姑娘走出了五百多米。总之，姑娘象美丽的女妖一样迷住了我的心窍。

　　"来回跟着一个姑娘不免有失大雅吧？"姑娘虽然面带讥讽，但又不失温和的说道。

　　可不是！我，一个已有女友的潇洒少年，怎么能这样......要不是姑娘的一句提醒， 不知道我要在迷魂汤里泡到什么年月呢。

　　刚要离开姑娘，忽然发现自己和姑娘被什么东西拴在了一起。我心中不禁一阵高兴。虽然这时我处境尴尬，但世上还有什么能比和这样漂亮的姑娘拴在一起更令人陶醉的呢！

　　"你看上去文质彬彬，怎么好意思对一个陌生姑娘无礼呢？"姑娘仍然没有动怒。

　　"我不知......我实在不知......怎么回事......"我讷讷地回答。随后我惊叫了一声："这不是我的链子吗？"

　　"当然是你的链子，要不怎么能拴住我？"

　　"这是我钱包上的链子呀！"姑娘一直对我温和，我也没好意思和她动怒。

　　"不管是你钱包上的链子，还是你皮包上的链子，你实在不该调戏一个陌生姑娘。"

　　我忙伸手去摸口袋中的钱包，早不翼而飞了！然而链子的一端仍紧紧地拴在我的腰带上。

　　"我怀疑你偷了我的钱包！"我已对姑娘毫不客气了，猛地把钱包从姑娘的口袋里拉了出来。

　　"要知道诬陷好人是有罪的......"姑娘捂住双眼哭了起来。刚下车的众旅客围拢过来。那老头自然也在其中。"老大爷，您看他偷我的钱包......"姑娘哭声更高了，

　　哭得那样真真切切，以致于我也开始怀疑自己是否弄错了。

　　"我早看出他不是个好人！"老头气愤地说。

　　"抓住他，揍这个流氓！"众人齐声喊道。

　　一位乘警拨开人群，走到我面前说："你的钱包里有多少钱？如果说对了，钱是你的；如果不对，你趁早还给姑娘。否则，小心把你。......"

　　后面的话我没听清。但不管怎样，我总算在黑暗中看到了光明，在痛苦中看到了希望。世上还有什么人能比我们的警察同志更会秉公办事呢？

　　"五百元整！"我的语气当然理直气壮。

　　警察连续把钱数了三遍，无论如何也不够五百元。这时我忽然想起在火车上买面包花了九角。但后来无论我怎样解释，也无人相信我的话，更无人为我辩护。警察同志非把我押到派出所不可。亏我连连认错，并保证下不为例，警察同志才高抬了贵手。又亏我天生双腿灵便，才逃脱了人们的一阵老拳。

　　我真幸运！
