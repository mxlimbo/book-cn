惊人的相同之处


　　女孩马尔曼和男孩吴尔希有许多惊人的相同之处：两人的名字里都有一个"尔"字，今年都二十二岁，身高都一米七二，左耳后面都有颗黑痣；都是在十年前失去了父母，家里都有一个未成年的妹妹；在同校、同系、同班共同学习了四年，上学期间同一天恋爱，恋爱期间同一天开始接吻；毕业后被分配到同一个单位工作，工作后又经常一同上街、一同到集体食堂买饭、一同到影院看电影。所不同的是，他们不一个性别；因为没有结婚，所以也经常不一同睡觉。不过他们俩人已经集资购置了一套家具、一台冰箱、一台彩电、一台空调，准备一同的民政部门一同办理结婚手续。

　　一天晚上他们又一同约好对方，在同一时间到了同一地点。他们一同张开双臂，同时抱住了对方。这样约么过了一刻钟时间，每个人都和对方一样，谁也没说什么。他们又不约而同地松开双臂，彼此对着淡淡地一笑，然后异口同声地说："你好像心里有事？"他们都得到了同样的回答："不要瞎猜疑了。"他们又同时问对方："那么刚才吻我时，为什么你没有闭上眼睛呢？"彼此都默不作声。这时恰巧一只飞禽从空中飞过拉下两滴屎来，落到他们头上每人头上一滴。

　　又过了一刻钟，他们心里一同感到应该回去。当走到门口时，又一同停住了脚步，并一同笑着问对方道："你到底有什么事？"彼此都没回答。马尔曼心想：我心里的确有事。吴尔希心里也想：我心里的确有事。马尔曼又想：但我不能直接告诉他。吴尔希也又想：但我不能直接告诉她。他们一同伸出右手，摸了一下对方的左脸，然后回到了各自的宿舍。

　　他们刚要上床睡觉，同时在自己的上衣口袋里发现了一张纸条，上面写着同样的文字：我们分手吧！
