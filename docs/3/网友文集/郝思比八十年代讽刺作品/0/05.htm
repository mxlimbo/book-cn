闪电式恋爱


　　如今的小青年们把恋爱的种类分得五花八门。不过，若单从进展速度上加以区分，也不外乎以下几类：马拉松式、渐进式、闪电式。人们都说闪电式恋爱投资少、收益高。有一位年轻人用自己的切身实践检验了这句话的真伪。

　　赵州是新疆工学院的英语教师，二十多岁。本来一米七九的高个儿头，又刚在一所理发店修了头发，样子要多帅就有多帅。如果电影导演曾见到他，非把他强行拉走，压给他一个重要角色不可。可坐了六天的硬座火车，腰酸背痛，满身疲惫，浑身象抽了筋似的。一下火车，他找到写有"英语视听速训班"的牌子，非让人家把他单独送到旅馆不可。

　　一到住处，他就歪倒在床上。顿时，鼾声雷动，直到次日上课时才被人们叫醒。

　　虽然过了整整十二小时的觉瘾，但他仍没睡够。他惺忪着眼睛坐在教室里。尽管那老头是一个知名教授，一堂课下来他还是不知讲了些什么。快下课了，他抬起下垂的眼皮往前瞧去。呀！这一瞧不要紧，引起了下面不长不短的故事。

　　就在他的课桌前面是一位女生－－不管她以前是什么校长啊、教导主任啊或是一般教师，只要她来到这里，尽管称她学生无妨。姑娘的一头披肩发当时就把他迷住了。赵州又擦擦双眼，伸长脖子细细瞧了瞧从姑娘的发缝中露出的一块肌肤，那简直是白玉琢成的。姑娘并没注意，也没回头。他望着姑娘的背影，想象着姑娘的俏丽模样：瓜子儿脸，含情脉脉的双眼，具有柔和线条的鼻子......他觉得，无论自己如何施展想象力，也难以把姑娘的美丽在大脑中描绘出来。

　　下课铃终于响了。他象间谍追踪目标一样目不转睛地盯着姑娘。就在姑娘出门转身的一刹那，他有幸看到了姑娘的面庞：两只眼睛非常冷峻，典型的有棱有角的鼻子......总之，完全是他想象之外的一种模样。

　　姑娘的仪态举止当然无可挑剔。但如果我们都不违背心中的审美标准，这姑娘的确称不上什么美丽。如果硬说她美丽，那是在照顾她的情绪。尤其是她那只鼻子，实在对不住观众：从两眉中间斜着向外流下的线条刚过眼睛，就向下一垂，又狠狠地向里勾去。但我们的赵州认为。她美就美在这只鼻子上。

　　姑娘为人极为倨傲，说话时总是昂首挺胸，好像唯恐别人看不见她鼻子下边那个小钩儿。开始几天，赵州很绝望，因为姑娘丝毫没有和他接近的意思。如果他不假思索，匆匆向姑娘射出爱的一箭，姑娘肯定会不慌不忙地向旁边一躲，或用手指轻轻一拨，让它向垃圾箱里飞去。

　　赵州终于打听到了姑娘的来龙去脉。她叫郎漫，济南市某中学英语教师。姑娘学历不高，专科毕业。这使赵州出乎意外地狂喜，因为他本人是名牌大学毕业。他不由得增强了追求目标的信心。

　　他使出了虎心豹子胆把那封信送给了姑娘。

　　第一个星期日赵州和郎漫来到了海滨第一浴场。

　　路上他问郎漫："你会游泳吗？"

　　"去年刚学的。"

　　赵州心想：在做技术指导时，可以触摸她那富有弹性的肌体了。

　　使他大失所望的是：这个游泳"初学者"比他还谙习水性，就好像受过专门训练。他没有获得一次自己所想象的"一手拉住她的臂，另一手托住她的腰"那样的机会。不一会儿，他就装作身体不适上了岸。

　　他们在靠近浴场的一家饭店里进了餐，花去了他上百元。然而他觉得非常值得。

　　不到一周，赵州的钱包里就出现了赤字。他给家里写了一封要求寄钱的信。

　　每天只有两小时的课，其余的时间赵州总是和她一块度过。这天他们到了公园，他无心观景，就买来罐头、啤酒和郎漫坐在小树林中的石板上喝了起来。一瓶啤酒下肚，他的情欲开始高涨起来。借着酒劲儿，他把双手搭在郎漫肩上，郎漫既没反对也没激动。当他张开双臂拥抱她时，郎漫面带愠色地说：

　　"难道这就是你爱我的方式吗？为什么楼楼抱抱才叫爱呢？"

　　赵州顿时感到羞愧万分。他仿佛现在才真正了解姑娘。她所以对他不够热乎，完全是因为郎漫爱得更高雅一些。她生在闹市而毫不轻浮，这不禁使赵州对她爱得更深了。

　　第二周下来，他又开销了二百多元，还得向家里要钱。虽然这样下来对他一个穷教师来说紧张了一些，但这又有什么可惜的？为了最可爱的人嘛。

　　不管姑娘有怎样的一套恋爱理论，二十天的学习班快要结束了，赵州却连姑娘的一吻也没得到，这实在令他心寒。他决定：学习班的最后一天郑重向她提出这个要求，而且不管她答应与否，都要付诸行动。

　　郎漫答应说今早七点在旅馆门口等他，并随他到家里一趟，见见未来的公婆。快八点了，还不见郎漫的影子。赵州顿时心急如焚。他忙回到学习班一打听，才知郎漫昨天就办好了手续，提前一天回到了济南。
