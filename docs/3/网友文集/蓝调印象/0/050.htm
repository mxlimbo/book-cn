谁 才 快 乐

　　小时便得知，对面楼住着一个疯子。

　　很戏剧性的开头，似乎应该引出些许故事。可十几年风平浪静的也过去了。疯子是个女的，有时我在厨房做饭能看到她趴在窗台上晒太阳，白白胖胖的，头发很长，漠然的看着四周。家里人一般不让我看她，她与人目光接触时，好似没有看到任何东西般的平静。我对她微笑示意，她依然漠然高深的望着我，然后一会便消失在了密密的窗帘后。

　　十几年应该是波澜无惊的，但小时的一件记忆出卖了她，我原是恨过她的。

　　那是一个夏天闷热的黄昏，我去找冉冉玩。走到前面那栋楼的时候，忽然听到有人"啊"的一声尖叫。我快步跑过去，看见两个中年妇女呆呆的站在楼洞草坪前，顺着她们的目光看过去，有一只猫躺在草坪上。在她们语无伦次的话语中，我得知，是那个疯子从窗台上把这只猫扔了下来。"不是扔，是掼了下来，"另一个妇女反驳到，"不然这只猫摔不死。"她们叫我："小女孩，你去把不猫抱过来让我们看看还有救吗？"当年我年纪很小，爱猫如命，不懂畏惧死亡。跑进草坪我小心翼翼的把它抱出来。那真是一只很小的猫，它还没有来得及长大呢，黄白相间，娇小可爱。奇怪的是它身上一点点伤痕都没有，只是变的好软好软，连头都直不起来，我托着它的脖子，它用明亮温驯的大眼睛看着我们，我能感觉到手里的温热，它很慢很慢的呼吸着。心里忽然第一次有了这样酸楚的感觉，我们眼睁睁的看着它越来越慢的呼吸，眼睛渐渐暗淡，然后一动不动，停止了….那两个中年妇女气愤的说那种人根本不配养动物，另一个说她是疯子她知道什么，前一个就说疯子就该关进医院…..在她们激烈的争论中，我哀伤的捧着这只小猫，不知道如何磁能挽救它的性命，没有人告诉我我该怎么办……后来我和冉冉亲手把它埋了，做了一个漂亮的小墓。晚上很晚了都睡不着，和家人哭诉为什么不把这个疯子关进医院里去？家人安慰我说，她不是每天都疯的。

　　我想我还是恨过她很久的，我是一个爱猫如命的人。当那只猫用它温驯明亮的眼睛看着我时，我甚至看不到谴责和忧伤，这更让我难过，我发誓这一生我也不会去做任何虐待任何动物。

　　在幼时惊天动地的猫咪事件过去后，以后的十几里年波澜无惊，疯子依然在天气好的时候趴在窗台上晒太阳，大部分的时候消失在那层厚厚的窗帘后。我对她的恨意已转化为淡淡的怜悯。自己都可以为她找出很多的理由为什么不去医院，因为她大部分时间是平静的，因为她什么多不会做，因为她不会威胁到别人，因为他在家里更自在，因为，因为她的父母爱她，不想离开她……亦或她爱她的父母，不想离开他们…我再在窗台看到她都会给她一个微笑，她依然漠然高深，冷淡的看着我们。人在慢慢的长大中开始变的包容和仁慈，人的天性显了出来，或许她的世界比一般人要来的快乐和平和，但是，我很同情她。

　　文章到这里也该完了，有一天上网到凌晨，为了传一个很大的文件，我耗着没睡，一边和网友聊天，毫无防备的，忽然窗外炸起了女人尖利的嚎叫，我头皮也紧接着炸了起来，"腾的"从椅子上跳起来拉开窗帘，判断声音的来源，不是马路上的，声音来自对面楼。对面楼群漆黑一片，忽然有一家灯拉亮了，有人凄厉的惨叫有人大声劝阻喝止着夹杂成一片。那个熟悉的窗口告诉我，是疯子家。仔细听，是疯子在放声大喊："我不快乐啊---我不快乐---我不快乐-------……"她的母亲在大声劝慰她，父亲则在喝止，安慰渐渐也变成了嚎哭，乱成一团。周围有灯不断拉亮，睡梦中的人们纷纷惊醒咒骂着，忽然听到有个男人大声喝到："大半夜的喊什么！他妈的，这世界谁快乐！"周围一下静了下来，疯子的喊声停止了，有人低低的啜泣，灯又一盏一盏的熄灭。我呆站着窗口，这世界真奇怪呢，连我看来平和安详的疯子都说自己不快乐…….我们每个自诩正常人的，又有几个敢喊出：我不快乐？ 在别人的目光里哪个活的不是战战兢兢，辛苦倍至。本不快乐的，又要诫与自己要惜福惜福，应该快乐点，本来快乐的，在别人眼里又怕显得轻狂招致嫉恨，又摆出一副无所谓的样子。装来装去，自己都快不知道快乐是什么。

　　第二天依然晴朗，昨夜的尖叫让人疑是一场噩梦。窗外阳光明媚，鸟语花香，我站在阳台上拉开窗户把手伸出去试一试温度，我看到了疯子，她依然安详的趴在窗台上，似乎昨夜什么都没发生过。我迟疑了一下把伸出去的手摇摆两下，对她招手示意。她漠然的看着我，我只好厚着脸皮吐吐舌头把手缩了回来。在关窗户的那一瞬间，我又抬头看了她一眼--------是我看错了么？她微微的挑起嘴角，给了一个极淡极淡的微笑…….
