相 见 不 如 怀 念


　　站在空空荡荡的车站前我恼怒的几乎要哭出来。该死的班车又早发了，我只出来的晚了那么几分钟，只是晚了几分钟啊！全都是为了给同事包红包，下班一家人热热闹闹的去喝酒庆祝，我推辞掉了，一向不喜欢这种场合，晚上又和然然说好去滑冰的。只是包了个红包，就晚了…站在车站边有些落寡和无可奈何，一辆辆'的'在面前减速，我挥手让它们走。怎么打？送出红包身上只剩一张10元的钞，原以为坐班车回去，打车回去这么远的里程起码要20元以上......一辆'的'在我面前嘎的停下，我只好勉强的问："回生活区多少钱？"司机隔着玻璃微笑着说："5元好么？"5元，我呆了一下，5元？！
　　
　　打开车门，我坐了进去。

　　司机是一个年轻的男子，歪过头来说话的时候，我看到他耳上带着一个小小的银色耳环，在夕阳里一映闪闪发光。我看看他有点意外，说这么便宜是回头车吧，他点头说对极了。用如此便宜的价格坐回去让我很不安，对他说今天班车发早了，也许下一站也有人没坐上，可以一起打，他笑了笑不置可否。到了下一站果然还有两个人没坐上车，好象是一对情侣。他不说话，我打开车窗问他们是否回生活区，他们很愉快的上来，我也很高兴，这样平摊价格对司机也算合算。而他在一边却不说话，隐隐的笑着看我做这一切，不擅长与人打交道的我仅仅说合伙做车一起出钱就涨红了脸。对他的不语让我有点恼，他为什么一直都在隐隐的笑？这有什么好笑的！我在心里暗暗发誓以后再也不干这种事了。司机问他们去哪里，情侣们报了地名，他又问我去哪里，我头也不抬的说去旱冰场~他看看我，然后静静的开车。

　　低头间似乎还能感到他的视线，我夸大动作的去翻包找书，低头间看到他锃亮的皮鞋，不禁有点点迷惑，小心的细细看上去，笔直的黑色西裤，洁白的衬衣端正的系着袖扣.....好象从没见过这么整洁的出租车司机呢，真奇怪。可是，关我什么事~耸耸肩我翻开在班上看到的页角，专心的读我的《散文》。刚刚沉浸到荷塘月色中....落单的荷花，花瓣在逆光处薄如蝉翼，却还能带着一点透明的粉紫.......他扭过头来说了一句话，我没在意，他提高声音又说了一遍，我迷茫的看看他，他说："坐车看书对眼睛不好。""哦....."我合死书，心里怪怪的。天,这么怪的一天，碰上这么古怪的一个司机。顾客坐车看书关你什么事？我望着窗外，路程已经有一半了吧，天色渐渐暗下来，后坐的情侣窃窃私语，女孩掩嘴呵呵的笑，我忽然想起来自己还没有吃晚饭......司机腾出一只手放了一盘磁带，很吵很吵的RUCK，我把头歪远趴在窗外，他很快把磁带抽了出来。正在这个时候电话响了，车上四个人，三个人忙着翻自己的电话，只有我没动，我只有传呼。想不到电话竟然是司机的，他低低的对电话那边的人说："很快回去...有人坐车 …….呵呵….对....车很好.......我很快会回去….."。放下电话他找了另外一盘磁带推进去，似曾相识的前奏渐渐响起。我不禁微笑，他说："知道谁的？""BEYOND。"他颔首，轻轻的问我："喜欢么？""喜欢啊。"话音落下，歌慢慢响起来，居然是那首最喜欢的《喜欢你》，歌在车里回荡着，遥远的天边燃起了云，殷红的一片，他的声音如此轻，温柔的象是对情人间的低语，点点的暧昧....."细雨带风湿透黄昏的街道.... 抹去双眼无辜的仰望 ....是那伤感的记忆 .....再次返起心里无数的思恋 ....往片刻欢笑永挂在面上....愿你此刻可会知 .....是我忠心地说声 ....喜欢你...那双眼动人...笑声更迷人......"

　　我忽然有点不安，他根本不象一个司机，穿着如此整洁，带着浪琴手表，手机用最新出的9980，我打20元以上的车资他用商量的口气问我5元好么。我下意识的再看看自己，简单的粉色T血，靛蓝色的牛仔裤，斜背着一个方方的包，头发梳成的两个小辫现在已经散了吧，一脸黄昏的倦倦庸懒的神情……他何以会注意倒我呢？

　　他跟着音乐轻轻哼唱起来，我把书塞进书包，他看了一眼模糊的封面说："喜欢看散文？"
　　"对，"我点头.
　　他歪歪头："我喜欢沈从文。"
　　我笑："好象现在很少人看他们那个时代的书了。"
　　"现在人看余秋雨，看余华…..不过沈从文有一篇你一定要看。"
　　"什么？"
　　"《边城》，写的很美。"
　　"《边城》…..？"我喃喃到。"
　　对，尤其是结尾……写的很妙……."
　　"呀！"我忽然看到窗外旱冰场正飞速的掠过，他笑笑说："真对不起，开过了。"他分明是成心的！我咬了咬嘴唇不再说话。他回头说："先送你们，指指路吧。"后面的那对情侣很快到了目的地，付钱下车。他慢慢的开往旱冰场，"一下班就去滑冰？"
　　"恩…"
　　"没吃饭吧？"
　　我抬头看他，他的眼睛很明亮很明亮的望着我，在那一刻我竟然恍惚了…..心里低低的问：你是谁呢？你究竟是谁呢？为何要这样对我？......传呼的声音滴滴响起，我忙低头去看，是然然呼我。透过车窗左看右看的找电话亭，已经快到旱冰场了。他默默的把手机递过来，我摇头，他笑，却并不把手放回去，执着的放在我面前。接过来用，还带着他手的微微余温。是然然在街上给我打的，她打公话急急的说："夏夏我不能去了，阿珞他们加班，我这里有急事，我们改天吧…..""可是我已经到了呀！….."…..放下电话我有些发呆，他问："男朋友？""不，我的朋友们，他们有事不来了。"他停了车，窗外分明已经到了，他低低的说："一个人还要去么？……""去呀，一个人也要玩的好！"给了他一个阳光灿烂的微笑，我转身下车。（请让我微笑着下车吧，让这出剧目完整的谢幕.....）

　　打开车门放下钱，舒口气，我回头对他说："谢谢你。"（想说的，其实是谢谢你对我的关照….）"能告诉我你的电话么？"他不关车门，黑暗里我能感到他略微有点焦急，我微微的楞了一下，莞而："刚才你在电话里说你会很快回去的…"他笑又扬眉："我只问一个电话！那好吧，传呼号也可以，我知道你有….."我笑出声，他也笑出声,一副赖皮的孩子般的表情.我慢慢的说给他听，他认真的输入到电话里，心里忽然划过这样一句话：你怎么可以就这样把你的号码随便告诉一个陌生人？你怎么可以这样轻易的被他打动？这不是你的风格呵~难道就因为你对他有好感？就因为他很温和，就因为他也喜欢BEYNED，就因为他看沈从文……在吐出最后一个字的时候，我把传呼的最后一个号码改了，改的面不改色，却有微微伤感。他小心的把号码存好，把我的钱写上一串数字再递给我，"是我的电话，夏夏，我们会再见。"我看着车慢慢的驶远，转身把钱买了冰场的票，我永远不会给自己任何受诱惑的机会。即使一个人也要玩的开心呀！他不了解女人，我永远不会主动给他打电话，永远不会………

很久以后，我看了《边城》，结尾说："到了冬天，那个塌了的白塔又重新修好了。那个在月光下唱歌，使翠翠在睡梦里为歌声把灵魂轻轻浮起来的年轻人，还不曾会到茶峒来。

　　这个人也许永远不回来了，也许明天就回来！"

　　他说的一点没错，很妙的结尾。

　　我们也就这么结尾了么？我的司机？很多次很多次，我又困在了车站或者别处，我是一个粗心大意的人，我期待过你的到来，可是你象一个传奇，再也没有出现。

　　你给我打过传呼么？
　　你期待过我的电话么？
　　你想过我找过我思念过我么？
　　你现在知道我告诉你的是一个错误的号码了么？
　　你是怎么发现那个穿粉色T血困在车站一脸迷惑的女孩？
　　深夜里,有时我回想起那坐在车上的一分一秒，你一直在隐隐的笑，你在笑什么呢？

　　有时在街上，我会下意识的在出租车流中找你，找那个整洁的，带着耳钉，嘴角有着隐隐微笑的男人……..。

　　这个城市里永远有着有疾无终的爱情，相见永远不如怀念…………
