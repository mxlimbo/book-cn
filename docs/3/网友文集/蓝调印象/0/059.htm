喜欢 -- 樱桃小丸子

　　
　　好象在很久很久以前了，流行收集贴画（大概是在小学的时候吧：），我买的贴画里面也兼带着有一些樱桃小丸子的贴纸。只喜欢漂亮图片的自己看到这样简单的画面禁不住奇怪，这样的画居然也会有人喜欢也有出版…..#￥%……%%%-…….

　　最近在书友会里发现进了很多漫画，也有《樱桃小丸子》，我不由笑笑笑…..，久违了呀，连封面都是很童稚的样子呢` 于是抽出来借，老板记录的时候，在漫画书和我之间来回看看看…..，我面不改色的拿起书出门。其实在小学就开始看漫画了，绝对有足够长的漫龄，而且上班了以后也一直在看，很多朋友拿起我手里的漫画书都大惊小怪，你这么大了居然还在看漫画啊………>: 其实有的漫画里情节很好,并不压于小说,有的独白和诗句也很经典,有些也渗透着人生的道理吧…….只是看人怎么看了。

　　看《樱桃小丸子》的时候是先在看铁凝的《大浴女》，被尹小跳的爱情搞的一塌糊涂，就顺手拿起《樱桃小丸子》来换换脑筋。打开漫画，整个人好象一下跌入了一个粉红色带有水果清香的儿童世界里。散漫的爸爸啊，爱唠叨的妈妈啊，宠爱自己要命的爷爷了…….还有淅沥糊涂的小丸子………这些情节根本就太熟悉了嘛，假期到最后一天还没有做完作业了，在学校里被男生欺负啊，和好朋友吵架伤心又和好了………..似乎能从小丸子身上看到自己呢~ :")

　　用大人的角度来画小孩的漫画应该是比较困难的吧~可是故事情节安排的好极了,一点没有给人矫揉造作的感觉,幼稚不精美的画面看多了,也觉得蛮可爱的,能从小丸子细细的眉毛挑起和弯下看出她的不愉快和烦恼。虽然她一点都不优秀，甚至还有点笨笨的，不过这样才最真实最可爱啊！做出各种各样的糗事，总有着无数的快乐和烦恼，永远在小学三年级里………...有一次小丸子收到不幸循环信，她大为紧张的和爷爷商量怎么办，爸爸看见了说：那都是骗人的，谁相信谁就是大傻瓜。爷爷说：啊……我相信了，我真是个傻瓜…….小丸子一脸认真的说：我也相信了，我也你一样是傻瓜，真好…… 简直令人绝倒！~：D 还有很多类似的情节了，也有感人的，小丸子和好朋友小玉吵架，两人事后各自忏悔，然后在黄昏的神社里相遇，说着自己的不对，拥抱在一起。一向惜墨如金的作者，给她们画上黄昏美丽的晕黄，她们在神社的树下埋下玻璃瓶，里面放着彼此想对对方说的话，然后牵着手回家。童年时的友谊也是这样好啊，也想起自己和小时候的好朋友，在学校的樱花树下计划着把将来的愿望放在铁盒子里，然后埋在校园。可惜，我们没她们那么幸运……..我们当时坐在树上觉得隐藏的很好了，结果被一个眼尖的体育老师发现，把我们哄了下来……..呵呵，想起来比较糗。还曾经和然用很多很多土把有浅沟的路面铺平，自以为干了件大好事。结果第二天下雨，都成了泥巴冲的到处都是，路过的行人怨声载道，说谁在路上弄了这么多土？！………..小学的吵架了赌气了，被老师训了，考试没考好了，当时对于自己天塌般的苦恼，现在回忆起来也是丝丝甜蜜。一点点小事就可以升入天堂也可以打入地狱，只有小时候才会的吧~ 看完樱桃小丸子

　　翻出自己以前收集的贴画，居然还保留了几张她的贴画，看到她傻傻的可爱的小笑脸，真高兴。生活一如儿童般简单的时候竟是如此快乐........
