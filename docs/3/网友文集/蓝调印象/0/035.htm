没有无瑕疵的爱


在你来之前，亲爱的，那碗水也许倾洒出了一些，但是我愿意把剩下的全部都给你

――题记

　　也许一直都爱说自己是追求完美主义者，所以给周围的人造成了一种假象，要的东西必是十全十美的。

　　一天，QQ里的朋友忽然蹦出一句话："你在等一段没有瑕疵的爱吗？为什么还不去爱？" 不，我不是在等。

　　正因为我太了解爱情了，所以我知道爱情是不可能没有瑕疵的。越是生活在真空里，越容易被感染。这是一个没有完美的世界，我们可以去追寻完美，但是我们要的完美往往不是真的十全十美。每一段爱情都无可避免会有伤痕，如果你以为爱情容不下一点瑕疵，那么你大概一辈子也找不到爱情。

　　你所爱的那个人不可能从来没有爱过什么人，深爱你的那个人，不可能从他出生下来第一个就爱上你。我们都不可能保护着饱满柔软的那颗心直到自己真正爱的那个人来到。在你来之前，亲爱的，那碗水也许倾洒出了一些，但是我愿意把剩下的全部都给你。

　　我愿用我的爱包容下你以前的伤痕，我也希望你用你的爱包容下我为爱而受的伤，爱因为双向的包容才变的完美。

　　C ，写这段话给你看，我没有站在孤岛里，不论是否有人爱着，无论是否爱着别人，我都深深感激，深深幸福，因为我已如此富有，爱过，被爱过。即使现在我停驻在这里，哦，那是我在为我的爱热身。
