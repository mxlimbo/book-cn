只有那么一点点想你了 亲爱的

爱情是株植物，种下的时候谁能想到它会发出什么样的芽，开出什么色彩的花？

————题记

　　今天忽然发现我的瑞士军刀坏了,它裂了一个口子,我检视着,是在什么时候坏的呢?曾经以为它永远都不会坏........不,是你告诉我的,是你给我灌输了这种思想,你说它不会坏,永远都不会坏.......就象我们的爱情....这个世界上有什么是能肯定一辈子的呢? 当时你说要买,我看着标价$后面跟着的数字连连摇头,说没有必要为了买一把刀花近1000元.你微笑,说:"有的东西是一辈子只拥有一个的,要就要最好的,比如一把好刀,再比如,"你拥着我轻轻说,"一个好女朋友....."我摇头,你大笑.后来还是买了两把。我不想要功能那么多的,只挑了一把小巧玲珑的带在钥匙上。有很多人问我是真的吗?我说是,他们说天啊,花那么贵的钱只为了买一把刀子~~~我说:"有的东西是一辈子只拥有一个的,要就要最好的啊~ "看看我的口气多象你,可惜,你听不到我说这些话了....

　　我们都曾经以为自己很了解这个世界,曾经以为自己能掌握自己的一切,那些年的我只穿黑白灰色的衣服,看纯文学的东西,只听轻音乐,看国外的影碟,冰激凌固定吃那一种牌子....我不知道你如何容忍我的,我们很任性,也许是年轻吧~那时还没有学会包容别人,....你让着我,但是勉强着,因为你也不大.....

　　我们曾计划过未来要怎样布置我们的家,计划过那每间屋子,每一张床,甚至为窗帘的颜色争论,我说要那种凉凉的冰蓝色,你说一定要温馨的黄,最后我们决定在夏天里挂冰蓝,在冬天里挂鹅黄.那春秋天呢?你热切的问着我,好象我们明天就要结婚,窗帘即将挂上。

　　 最后的最后,我们还是各奔东西......呆的太久已经不是爱情了吧,只是日久生出的情.....也许我现在的想你，也是那岁月流失下来的---感情?我不知道,反正我不象从前一样了,什么事都要给自己一个肯定的回答,什么事都要有头有尾,我累了,所以我要离开,就这样。

　　你一定要我给个答案,你问我是不是不爱你了?我不知道如何回答,我想你,想知道你过的好不好,是不是仍然在早晨定很响的闹钟,家里没人是不是还去吃泡面,抽屉里是不是仍然乱乱的放着一大堆CD,枕头边的照相框里又换了哪个女孩的照片.......我想知道你过的好不好,不好,我会难过,很难过很难过....但是我仍然要离开你,亲爱的.....

　　我知道你恨我,但是我没有办法,我不想在你的爱里窒息死去......爱太多太重,我承担不起,我不想把我年轻的时光全部泡在恋爱上,你说我自私也好,我想做太多的事情,惟独,亲爱的，惟独不想再谈恋爱.....我爱不起~我太自私........我不知道这算不算自私,瞧,我又辩解了,好,是自私,没错的,我很自私,我想实现自己的梦想,我看到那梦的曦光,它是那样的诱惑着我,让我情愿为它去付出一切......我知道你不喜欢我说这些,也许你有一点点厌倦.你说过,你的梦想就是娶我这样的一个女孩做妻子.我知道你期望我说嫁一个象你这样的男孩做丈夫,可惜我很冷静,我知道我们说的不是某部上演的三流电视剧的对白.

　　我想让你和我一起向前走的时候,你滞留在原地不动,你问我为什么不爱你了,我看着身后远远的你,看着前面不知的终点,亲爱的,你为什么不和我一起走,要让我如此为难,爱不是借口,也不是最好的问题和答案啊!你为什么一定要问我爱不爱呢?如果我不爱,那些年轻的疯狂的岁月去哪里了呢?......如果我爱,为何又如此坚决的离开你,明知道你在我背后醉生梦死,让我心如刀割,我还是要向前走.想到我背后你爱的怨恨的目光,我的步伐就几乎沉重的无法移动。你说我根本爱你爱的不上心，有许多的女孩子，她们都比我对你要好~ 是的，也许吧~爱情是株植物，种下的时候谁能想到它会发出什么样的芽，开出什么色彩的花？

　　你还是倾倒于对你好的女孩子们那一边去了，我没法怪你，也曾经在夏日炎炎的中午大街小巷的给你找你爱的CD，也曾经在深夜打的去药店给你买药，也曾经在凌晨的寒冬里泡电话,也曾经一起一趟一趟的做着没有目的的公车，只为你，亲爱的，可是，爱走了，是不是谁也留不住？纵然我们很爱很爱很爱过...........

　　我仍然不知道离自己的梦还有多远，仍然漂着，居无定所，仍然象那个我说过的,不落地的蒲公英，飘飘荡荡........我开始喜欢这种感觉，虽然我变了，真的，一样的眉,一样的眼,却不会再那么肯定，那么自信，那么骄傲，那么矜持，这个社会终于还是改变了些什么给我。我想，也许我是在期待什么人，但是，我绝不会相信他能给我带来一切了~王子的故事已经结束了，我们不再看童话，我也知道那每个结束在婚礼上的故事是最佳的结局，那才是童话。年轻的妈妈总是告诉女儿要嫁给一个王子，却没有告诉她怎么成为一个公主，和她到底是不是一个公主......

　　在全世界最最冷的那天,下班,我穿着羽绒服瑟瑟的走过街道,来来往往的人群中,远远的看到一个人穿着毛衣站着发呆,我想天啊,那是谁?他居然穿的那么少！仔细一看,居然是你,我们远远的打招呼,挥手,甚至看不清楚对方脸上的表情.从那以后我就没有停止的想过你,想你为什么那天要穿那么少,想你感冒发烧生病了吗..........

　　倘若时光倒回,我仍然还是要离开你的,可这不并不防碍我想你,有时我自己也想不明白爱情是什么了,我想做一个翩然飞舞的小蝶,纵然爱情是黄金做的网,我仍然想要无价的自由,我变了,我承认这个世界上没有绝对的,没有完全纯净的,没有不变化的,亲爱的,你看到现在的我一定不相信这是我,瞧,我变了,如同当年我们相信的那个世界,它也变了呵......我不再穿黑灰的衣服,不再那么矜持而又绝对。今天的我穿着白色的裤子,兰色的外套,淡紫色的毛衣,手里拿着粉红的随身听，留短短的头发,听着过去曾经被我们嗤之以鼻的流行歌曲,你相信那是我吗?坐在公车上，车缓缓滑过街道，望着窗外倾洒着阳光的街,手里握着那裂了缝的瑞士军刀，我不仅微笑,一再微笑.......而这一切,都没什么,亲爱的,我只是有一点点想你了.........

与 2月凌晨 音乐兰中
