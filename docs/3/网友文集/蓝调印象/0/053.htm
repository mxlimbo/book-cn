死党论…….

　　死党是什么呢？

　　和你有着过命的交情？从小玩到大的伙伴？最最最……要好的朋友？…….

　　这些都似乎不够足以表达你们之间的感情

　　可以肯定的是 在你的身边有那么几个她或他围绕着…….

　　死党就是当你买了旱冰鞋他会比你还欣喜若狂的人

　　死党是当他买了新赛车你会和自己一样的骑着满大街溜达

　　死党是他可以当着别人面数落你的不是而不许别人说你半句的人

　　死党是你生病了他会一边为你张罗着拿药打针一边骂你活该不小心

　　死党是拿你的CD 磁带 漫画….成自己的一样共享 并且兼容

　　死党是出去玩你可以理直气壮的告诉他你本月资金告急由他打理一切的人

　　死党是你看着他在爱里和了散散了别人无所谓而你会为之心痛的人

　　死党是他失恋了会拎着酒瓶子第一个来找你喝酒的人

　　死党是除了贴身物品（此物品包括情人丈夫妻子….）均舍得给他的人

　　死党是有时什么都不用说只需一个眼神打过去就可以明白一切的人

　　死党也是当你体会了某种枯涩辛酸的东西舍不得让他也去尝试的人（这种东西例如爱情~）

　　死党是相隔多年打通电话仍会又喊又叫的人

　　死党是万水千山也隔不断的………..
