不良嗜好

　　偷偷的去了CHAT，忽然有人敲开了我的小窗，问："听说你有三个杯子？""什么？！！！"慌乱中打出："我的三个被子?!!"象是被人踩了尾巴....赶紧冲去看帖子，好啊，原来是陶然这个丫头揭了底..*-……%￥#..

　　横横，其实我的杯子还不止三个呢~ 那天和陶陶聊天说起的只是一个保守数字。 >:)

　　仔细想起来,我好象真的有很多"不良嗜好"也~ 仅仅杯子吧,收集了起码有10个以上,都是那种漂漂亮亮的陶瓷杯,带着把手,不同的颜色，画有温馨的图案。很多已经用纸包好珍藏起来，只留下最最喜欢的三个常用。象帖子里写的，那个白色有着小熊图案的，用来喝茶，喜欢看菊花在琥珀色的茶水里轻舞飞扬。黑色的八角杯，画着圣诞老人的图案，最喜欢用它来冲咖啡。冬天里暖暖的捧在手里，咖啡的颜色在黑色的杯子里反衬的更深，冒着袅袅的热气，单单摆在那里，杯子上银须红袍的圣诞老人就很有节日的气氛~ 那个杯壁外面是白色，杯壁里面为蓝色的杯子最为奇怪，我有时都不仅想，颜色真是神奇的东西，仅在一个小小的杯子上就可以搭配的异常和谐。只把冷水倒进这个杯子里它的颜色都会变深，没有水的地方杯壁是蔚蓝，有水的地方变成深蓝。我喜欢用它来喝牛奶，蓝白搭配的，悦目之极。死党阿周来玩的时候看我摆弄杯子，说，"你真是怪嗜好呀，要那么多杯子干吗？" 我一边整理一边笑眯眯的说："将来我有家了，要在厨房漂漂亮亮的挂一排，让每个来玩的朋友，有他们自己喜欢固定用的杯子。"阿周听了大为感兴趣，立刻动手挑她将来用的。只见贪心的她，左手拿着刻有HAPPYDAY的杯子，右手攥着画有海盗地图的杯子，左右犹豫不定。我慷慨的一挥手说："好吧，到时候两个都留给你专用！~"阿周当即捧着杯子欢呼雀跃一番，我不禁大叫："小心我的杯子呀！现在摔碎了你将来就没法用了*-……%￥-……%￥#.........."

　　我还喜欢收藏手链，项链耳环戒指都兴趣不浓，只偏爱手链一样。每天里不同的心情和服饰来搭配着带。穿那条长到脚踝的白色长裙时，会带那副景泰蓝的手镯，叮当作响中，步步生风。穿大毛衣的时候喜欢带杜马陶饰，袖子长覆手背，左右抬手时才露手腕上细细串联的彩色陶珠。喜欢一条用井绿色线串起秋香玉的手链，精致的如同旗袍上的盘扣，古典的系在手腕上，几乎没有衣服可以配它。最终找到一件翻领系腰的同色衬衣，配宽摆的白色长裙，麻质布鞋，恍惚中自己已穿过那悠长的时光，成为古代里一戴着秋香绿玉的女子。

　　还有着许多其他的细碎的小小的嗜好。例如买书包皮，一定要纯白的打印纸，再为上者就是透明洒有细碎小花的包装纸。包好摆放到书架上，也可以清晰的看到书名。这样细致的活常常交给泽兰做。自己的态度有时象一面镜子，往往会感染给别人。泽兰和我一样，成为彻头彻尾的完美主义者 :)有着几百本的藏书，她给我包过的仅仅只有十几本，没有合适的包装纸，我还没说她自己都不会去包。但她包装过的那十几本，每本都是经典。张晓风的散文集，我们商议着用一种浅浅的细碎的粉红，正如晓风文章里的轻轻柔柔。舒婷的诗集则用的淡紫，蒙蒙的雪青，清新雅致，还未翻阅心里便起了暖意。

　　敏感的神经让我们更多的感受到人生的细微末节，上天给了我们更多的审器官，让我们自己也可以成为美的一面，创作出美。

　　一个人独自值班的时候，氧气和高高来陪我。我把细格棉布铺在办公桌移到窗口，苹果去皮削块，配上超市买来的色拉，拌成沙拉倒入陶盘，泡上咖啡，摆放好银色刀叉。两人进来时不禁又惊又笑："我们以为你一个人可怜希希的周末值班，你却能把办公室变茶会......"
　　
　　我喜欢生活过的有滋有味
　　
　　我喜欢收集那些漂亮的图册卡片

　　喜欢做菜，喜欢用色彩鲜艳的餐具搭配

　　喜欢大大小小的玩具熊

　　喜欢在房间里插满鲜花
　　
　　喜欢做这些带给我满心的喜悦

　　我喜欢生活里的细致优雅 喜欢尽善尽美

　　它如同五彩的缎锦 我给它镶上细致妥切的花边

　　生活亦在细微中体现

　　我享受我的每一个小小的"不良嗜好"............
