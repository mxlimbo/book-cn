其实哪有那么好....

　　曾经有那么一个阶段，疯狂的到处寻找白光的歌。说起白光，一定绝大部分人都不知道她是谁，说起最早的黑白电影《马路天使》里唱歌的金嗓子周旋，也许能知道，白光，就是和周旋一个时代的人。

　　刚开始听伊能静唱白光的歌是在伊那张翻唱旧上海老歌的怀旧专辑里，每首曲子都那么嗲嗲伢伢，加上伊能静特殊的童音，散发出旧上海的精致委婉。最最喜欢的就是那首《如果没有你》，很简单很简单的歌词却唱出天荒地老此情不断的感觉。暗想这首歌的原唱者该是怎样的精细曼妙的一个女子，该是怎样的小巧委婉.....该有怎样的故事发生在她身上，她会有怎样辛酸的经历，用细细的嗓音唱出那种婉转心驰的曲子......

　　后来在亦舒的书里看到这首歌原来是由白光唱的。白光两个字让我激动的不得了，白光，呵，好名字啊！大拙大愚的味道，白色的光.....俗话说想象里的玫瑰总比现实里的玫瑰香，我的假想以给她一层一层的涂上了暗香袅袅无比曼妙.....然后开始到处去查到处去问，很多的人甚至不知道白光是谁，有的人连周旋都不知道。我这个人最大的好处绝不轻易死心，只要发现一个新的音乐网站就无一例外的敲进"白光"两个字搜索查询一番。

　　最后终于在一个音乐网站找到了一堆老歌，周旋的，白光的，里面夹杂着那首《如果没有你》，我手忙脚乱的同时点了它们下载，想想又赶紧把其他的全部暂停，窗口也不开了，眼巴巴的看着下载进度20% 30%的跳.......到99%的时候真有点手足无措，是不是该先焚香沐浴再洗耳静听呢？.....小心的点开WINMAP，慢慢的戴上耳机，缓缓闭上眼睛......不到三十秒我就扔掉了耳机，不可置信的看着屏幕....上帝啊！为什么以前从没有人告诉过我------白光的嗓音是这样的粗！！！什么精致委婉啊，什么曼妙精细啊！我简直没有有勇气再听下去了，也不忍心再听下去，我怕再听下去把伊能静留给我的好印象都抹没了~~~~我甚至怀疑自己是不是下错了......

　　后来在朋友那里借到了徐晓凤的唱片，当年被美誉为小白光的她，也是那么粗粗的嗓音......在心里再次肯定了，原来白光就是那样的风格......。心里不免隐隐失望着，其实是大大的失望！早知如此，还不如不让我听到，白玫瑰也不会变成沾在衣服上的米粒，红玫瑰也不会变成墙上的蚊子血......

　　我们都在想象里把事物幻想的过为美丽了吧，以为会伤心欲绝的，发生时也不过如此，以为会欣喜若狂的，经历了也还好，盼望以久的，得到了还会有隐隐的失望，不如不得到。其实这个世界哪有我们想的那么好........
