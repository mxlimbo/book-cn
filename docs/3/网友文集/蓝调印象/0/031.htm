灵魂的栖息地


　　据说，这个城市写日记的人越来越少，人们越来越没空和自己的心灵对话....在追逐些什么呢？有时连自己都不能回答。只知道别人都在往前走啊，我也不能落后，我也要走，也要走，要快点走。生命就成了一条不分昼夜的河，义无返顾的向前流逝。

　　文凭，职称，奖金，股票....我们忙着所有人们忙碌的东西，如同被一个巨大的魔杖指挥着。没有时间停下来，我们只有喧嚣的白天和浮躁的夜，生活让我们麻木，让我们开始匆匆过往。我们是这样的匆匆的，匆匆的，在这个世界上奔跑着，没有时间停下脚步对它欣赏和赞叹，更没有时间对自己的心灵进行休憩和回顾。心灵的那片土地上荒芜着，杂草丛生。我们也希望有一片青绿色的草地杜绝一切的浮躁和喧嚣，我们也希望能在沥沥春雨中超然独步，可是我们被生命的琐屑吞没了。我们总以为有一天会，有一天能........恕不知生命是一列从生到死不停歇的火车，在奔向目的地时我们还在醉生梦死。

　　我们没能停下脚步来看一看，追逐之外的是什么，这个世界除了追逐还有什么。我相信这个世界上还有真正的道义，真理，爱情，诗歌和梦想，它们没有被真正的遗忘。但人们越来越沉浸感官的享受中，它们是否已蒙上了灰尘？我们的视野是否已变的狭窄？我们的激情是否已经冷却？我们追求好的东西，追求高质量的生活，我们要吃的好，穿的好，用的好，那么，我们身子里面的呢？这个金镶玉砌的身子里面呢？我们是不是该在那里捍卫起一个人的灵魂，品格，志气.......捍卫起我们最后的，灵魂的栖息地.......
