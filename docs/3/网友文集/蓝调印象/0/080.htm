最爱梧桐道


　　原来回家的路上是两排桃花，后来换成了法国梧桐，居民们曾经抗议过，后来，最终慢慢的接纳了这些美丽的梧桐，包括也曾经强烈抗议的自己。

　　每天下班，阳光都会细碎的从树间照射下来洒在路上，踩着这些光斑，一步一个轻盈
的回家。雨天，雨水瑟瑟的打在叶子上，不喜欢打伞时就从这棵树跑到那棵树，在它们之间玩着小孩的游戏。

　　秋天是最美的，整个梧桐道都在阳光里慢慢浸成柠檬黄，映着湛蓝的天空，迎面吹着凉爽的微风，好似整个人也在它们中变的清澈透明，一颗心，也就剔透起来。

　　黄昏的时候去溜狗，走在楼后长长的梧桐道里，一路金黄，看不到头，每个路人都仿若走进了一副长长的画卷里，不知道我和雪儿是画卷的哪一笔。

　　晚上曾经夜归，和要好的朋友挽着手闲闲的走过，夜很静，一丝风都没有，每片树叶都仿佛凝固在空气中，在路灯明亮的照射下，每一片都好似一副妙夺天工的精品画，我们甚至都不忍大声说话，怕惊醒了这象梦一般美丽的夜晚，轻轻的谈论着我们的将来，和一些不着边际的梦想。走着，走着，有树叶轻轻悠悠的飘落下来，落在我们的风衣上，再慢慢的飘落在地面上。整个路面星星点点的布满了晕黄的落叶，让人不忍踏步，这是每年一次的奢侈啊！
