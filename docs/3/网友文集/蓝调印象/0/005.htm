爱你 哪怕千疮百孔

　　把刚刚买的新手套给陶陶看，她连说好看好看又仔细看了看说："两个手套花纹一样，可是上面的颜色有一点点不一样啊.....""老板说这才特别嘛..."她呵呵的笑了说："你上当了，小傻瓜，那不是一套的！"我也笑，象一个小傻瓜一样笑了。买的时候老板说你看颜色有一点点不一样很特别的呢..... 我何尝不知道那是她在哄我，肯定是成套包装的时候包错了。可是我很喜欢那副手套，它和我想要的几乎一模一样，用纯羊毛细细的白线织成，手背中间是灰色的花纹，好似朵朵蝴蝶连成的。白色的手套很难找，这副难得如此得我心，纵然有点小小的瑕疵又怕什么呢？

　　死党告诉我，她的一个朋友有个习惯，买东西的时候绝对不要最后一件，理由是最后一件都是被别人挑剩下的。所以陪这个朋友买衣服特别累，纵然有一大堆新货还要在里面翻覆的挑挑拣拣，非要最最完美无缺的那件。时常看到一件合适的了，样品？不要！最后一件？不要！处理？不要！

　　我就不同了，如果是最后一件，那么我会觉得自己好太幸运！稍有瑕疵也不要紧-----因为我是真的喜欢它，连同它的缺点也一起爱。世上哪有那么多十全十美，如果是真的爱，那么我不在乎你是不是有缺点，因为我是那么幸运，让我能遇到你。

　　养的狗狗白雪很得来往网友的宠爱，因为它很乖，从来不冲客人汪汪叫，一律亲热的跟前跟后当成一家人，很久不来的朋友还能得到它热情的作揖拥抱，他们都说你的狗好乖好乖哦~ 可是白雪很爱玩，每天早上三四点就开始趴门叫我们起床带它出去玩，一天三次不止。它又爱生病，我要抱着它去很远的地方看病，它还经常吐在地板上，要家人收拾，家里人烦了说把它送出去。可是每次都只是说说而已，看着它乖乖的样子谁舍得。每天下班回来它都会亲热的摇头摆尾的过来和你打招呼，出门几天回来它见你更是上扑下跳嘴里呜呜着，一刻不离开，告诉你它有多么想你多么惦记你。就这样吧，我们容忍了它，心甘情愿的为它洗澡，带它看病，一遍一遍的收拾着它弄脏的客厅，吐过的地板，因为我们是这样的爱它，连带它的缺点，爱是最有包容性的~

　　衣服是这样，宠物是这样，人也是这样啊！你是这样的爱他，哪怕他粗心大意，哪怕他忘了你的生日，你也会原谅他，因为他是无意的。你是那么的宠她，哪怕她和朋友去吃饭而没有赴你的约会，你都会为她体谅，因为她很久没有见她的姐妹们了，需要聊聊天，谈谈心。

　　真爱，就会包容，只要我爱，哪怕千疮百孔。
