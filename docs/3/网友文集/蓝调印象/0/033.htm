流浪的小孩

　　一个人值班的晚上,独守长夜,拿着随身听无聊的点在自动搜索电台.坐在窗台仰望星空,好夜如风,繁星满天,忽然随身听的的滴滴声顿住了,搜索到的电台放出的竟然是伊能静的流浪的小孩.....

　　".....流浪的小孩,少年多挥霍,.....梦也渐渐被现实带走......"

　　有多久没有听过这首歌了呢?曾经非常非常的喜欢过她,纯真的眼神,软软的童音,喜欢她的唱腔,喜欢她自己填写的词.并且希望自己也能做一个流浪的小孩,天大地大,一个人去流浪,走出一片自己的天下.

　　后来,上了初中开始喜欢三毛,,小学的时候还在她的死引起的轩然大波中似懂非懂,中学就捧着她的作品默默而泣了.非常非常羡慕她的经历,并期盼自己长大也能一个一个城市的流浪,一个一个国家的游历,在海上的一个小小的岛屿上安家.少年时对未来的期望是那样的纯真无邪,总以为未来能象自己安排的那样顺其自然,那样想当然,却,也不过是美梦一场.....

　　再后来,曾经非常非常的渴望过爱情,渴望惊心动魄的故事.那时最最喜欢的莫过于金庸笔下的杨过,率性认真,真真的性情中人.希望能和期盼的他长相厮守,相守到老.......而生活也不过是这样轻描淡写的过着,不动声色的看着我们哭,看着我们笑,看者我们相守又分离.在别人眼里平平淡淡的两个人,相爱又分离,却是自己惊心动魄的故事......就是这样爱了爱了,恨了恨了,好了好了,分离分离了.......

　　爱过,恨过,受过伤,相守再分离,梦想渐渐破灭,成为对现实的一种嘲笑.在自己的自嘲总一次一次长大,在欢笑中,在眼泪中,在心碎中长大.对过去渐渐淡忘,听凭时间如同流水般哗哗的流淌而去,极淡极淡的悲哀,却已是彻底绝望彻底心碎后无奈的叹喟.....

　　甚至不再有那么多期盼,甚至包括爱情.看神雕侠侣里小龙女那一句:"冥冥中,自有天意." 不由长叹,冥冥中,自有天意.....知道自己无法再做第二次小龙女,无法给出第二次那么绝对那么强烈的感情.期盼的人在天意中渐渐淡去,刻骨铭心的爱过,不会再有第二次,人生真正的爱情只有那么一次.......只有那么一次,却难相守,却难相忘.......

　　然后就这样,风平水静,然后就这样,不盼,不喜,不怒,不忧......

　　而在这样一个深夜里,仰望满天繁星,潸然泪下,我几乎忘了,在很久很久以前,我曾经许诺过,我几乎忘了,自己曾许诺过,我要做一个流浪的小孩.......
