我只要一个阳台

　　家里装修的时候曾私心窃想过，如果把阳台给我一个人就好了…….

　　去陶然家淘CD的时候，一进去就感叹：你家太大了…….然后再感叹：你家实在太大了…..她说：你要说上几遍啊，我说：你听的后一遍一定是回音，因为你家实在太大了……

　　还有一次去同事的新房，也是空空荡荡的大阳台，群山环绕，有一种身在此山中的气氛……我说：你怎么不装饰装饰啊，她说不是刮了瓷又装了塑钢的窗户了吗？还怎么装饰？哦没装好太太的晾衣架……..我看看屋里的豪华家具叹气：是，是无法再装修了…….

于是梦想，如果，如果！我有那么一间群山环绕的阳台，我不会封闭它……
我要在阳台上放上黑铁的桌子，几张椅子，用白铁的水壶装满大束的干花。
我要在天气好的时候铺上斜纹的桌布，享受阳光照耀的早餐。
我要在周么的黄昏，召集好友，围在一起喝茶聊天。
我要在傍晚点燃蜡烛，坐在椅子里弹琴，数着星星的升起和落幕。
我要在家中来人的时候，携带着小说逃避进这个世界，闲闲的在月光下阅读与畅想
我要在每个夜晚与他携手相依，品茶论诗。

如果我注定生活在一个水泥森林里，无可避免的要封闭我的阳台。
我会把墙壁涂成十月天空的颜色
我会用蓝白相间的窗帘
我会在围台放一圈贝壳
我会在墙壁打上书桌堆满图书
我要在里面放一个有着柔和光芒的灯
我要在地面铺上竹席放一张窄窄的床
我要从屋顶挂起吊瓶插上香水百合
我要封闭这个都市的喧嚣和浮躁
我要给自己一个独初的宁静的空间
我想要一个可以让心灵超然独步的地方
我要门口挂上一个牌子：关着门的时候请别打扰我，如果门开着，请帮我把它关上…

　　这是怎样的一个奢侈的梦呢…….在现在这个空间每平米用千元来计算的时代，连梦似乎都隐灭了…..

　　我只要一个阳台。
