不见不散

　　今天中央6放了《不见不散》，尽管三天后有一场比较重要的考试，但是还是任性的看完。第一次看是在9几年我忘了，一个人去的，买了一大捧爆米花，安然的做在一堆情侣中。他们两个两个人笑的声音都比我大，我就故意极为不淑女的吃的也格外大声。当看到刘元和李清吵架红，他一个人上了飞机。镜头切换到刘元老年一个人在养老院孤独的写回忆录，画外音响起："...我把我的爱人丢了..."眼泪就止不住哗锝掉下来，为刘元为李清为自己，那句话简直就是为自己说的，和我当时的心境那么象....坐在一对对情侣中还故做潇洒，我把我的爱人丢了.....

　　后来想不到还是被葛优那小子给骗了！！！ >:) 剧种终于喜剧收场又止不住和大家一起破涕为笑.今天在家中再看起来真是感慨万千,身边倒是有个他----不过是老爸!呵呵,老爸和我一样上了葛优的当,看到刘元写回忆录的时候笑叹:"没缘分呐~"心里不由坷噔一下子,还是老人看的开.无论当年再执着再心痛,再放不下的,过去了回忆起来也不过长叹一声没缘,谁也不能总揪心着昨日的事,不过今天了吧? 遂释然.
