乌龙玫瑰

　　今年夏天忽然开始喜欢喝起了乌龙。

　　去茶庄买茶的时候，老板也笑咪咪的说，今年的乌龙格外的好卖呢~

　　乌龙泡开会在杯子里打开的大大的叶子，很舒展悠闲的漂着，透过玻璃杯甚至能看到它大大的槠色叶子上的纹路。想起和陶然STAN在茶馆喝茶的时候，陶然喝的玫瑰香片惊呼好香，我奇怪，自己泡的香片为什么是苦的呢？陶然说如果只有玫瑰应该就会香了。想起来了自己翻出香片单单把玫瑰挑出来放在杯中，转身放好香片的瞬间杯中已是香气四溢。两个深粉色的花苞改天换地，香气流连间，杯中活色生香。玫瑰就是玫瑰，走到哪里亦是精彩，哪怕泡成茶。乌龙淡淡的香气里渗进玫瑰，颜色也活泼起来，小缀一口，齿颊留香。

　　泡出以后甚是得意，有朋友来玩就谝谝，说，来喝喝乌龙玫瑰。他们说没有这道茶啊，我说有，我自己发明的啊。

　　人生随处都可以创造出美丽，包括给自己泡一杯小小的，芳香的乌龙玫瑰。
