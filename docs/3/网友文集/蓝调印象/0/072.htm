优 雅 的 变 老

　　在杂志上看到Michelle Pfeiffer米雪尔.菲弗的采访。注意起她，是从看她演的《纯真年代》《蝙蝠侠》《危险关系》开始的。她有一双猫的眼睛，能塑造出各种个性迥异的角色。在二十多年里她被连续的评为最美丽的女人，在摄影师眼中她永远都是"离镜头越近，就越散发出美丽光芒"的女人。

　　采访里记者问她：作为全世界最美的女人之一，对中年和老年的自己有什么希望。 她说：虽然脸上已有了几条皱纹，我对自己还是满意的。我可不想天天化着浓浓的妆，像个蜡人。我真正关心和希望的是，我能优雅地慢慢变老。看到这里，轻轻地吁了一口气，在别的明星大谈怎么防老，怎么抗皱，怎么用化妆品时，Michelle Pfeiffer说：希望能优雅地慢慢变老。为了这句话，爱她。

　　没有一个女人是不怕老的吧，老了预知着你将会变的动作迟缓，皮肤松弛，唠唠叨叨，不可抑制的肥胖，笨拙……连曾是世界上最美丽的女人伊丽莎白.泰勒也不例外。所有的化妆品都在喋喋不休的告诉我们怎么防止老化，怎么避免皱纹的产生。科学家残酷的指出，想阻止人的衰老那是不可能的。看过亦舒的一本书，很有意思，名字已经忘了，说主人公想恢复青春的愿望实现，身体倒回倒十七岁，自己却又无法适应了。人的相貌年轻了，心是老的，又能如何？

　　一次去朋友家玩，下楼梯的时候正逢邻居领着孩子走进楼洞，映着斜阳，邻居穿着月牙白色的旗袍，镶朱红色边，一边笑着和孩子说话，一边抬手掖掖鬓间的发丝，映在阳光中的她，一举一动都极有韵味。眼角已泛起了细细的皱纹，孩子也快和她一半高，可她无声的笑着冲我们点头，那种风度，气质无可形容。让我们两个小女生在她背后啧啧乱叹，找不出词来描述。心里模模糊糊的想，待三十岁以后也要同这个邻居一样吧，这样的美丽，这样的优雅。

　　放假出去玩，回来后死党给我展示她的新套装和6分的高根鞋。看到我买的平底靴子她痛心疾首，说你怎么还这么小孩气啊….. 我说：我要在适当的年龄穿属于自己钟爱的衣服，我会在25岁以后再穿套装，30岁以后穿旗袍…….心里忽然浮起了Michelle Pfeiffer，不仅微笑，轻声告诉自己：我们怕老，只是不要让老把我们吓怕了，这原本就是无可避免的不是么？我们的心可以不老，它可以保持着年轻时的热情和活力，多彩的阅历让我们变的丰富，充满韵味，哪怕外貌一天天的老化，我们可以和索非亚罗.兰一样每时每刻都风情万钟，和Michelle Pfeiffer一样优雅的，慢慢的变老………….
