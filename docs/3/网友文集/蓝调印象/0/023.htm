化腐朽为神奇



　　今天和陶陶到TOP那里去学琴，回来的时候陶陶感慨的说TOP很厉害啊，很平常的练习曲到他手里都能弹的那么好听，我说是啊是啊。平时大家学琴练习的时候，都说练习曲目弹出来不好听，可是吉他到了TOP手里却似改天换地，弹出来那么美，简直是一种享受。陶陶说，TOP有化腐朽为神奇的力量。

　　早一些的时候写过一篇帖子，是有关烟火屏保的，那是虫子送给我的。后来他发E-M给我说这个程序是他看来就一个养眼的作用,想不到我却能带来灵感写出美文....... 心里不禁美美的笑，我也算不算化腐朽为神奇了呢？把平凡的字组合在一起，希望能赋予魔力.......

　　想起来小时候妈妈喜欢把剩下的布头拼起来给我做裙子，让布块安排的恰倒好处，颜色搭配协调，拼的好看精彩得到别人夸奖，她就很得意说自己能化腐朽为神奇呢~

　　似乎身边的人都有这个魔力呀，我经常看泽兰用普通的包装纸包东西，包出来后那么优雅大方，常常把书拜托给她包，简直成为一种艺术品。听朋友唱歌，本来听过原唱觉得很一般般的歌，从他们嘴里唱出来却比最初听的更为优美动听，反而会又喜欢上那首歌。也在别人家里见过用易拉罐，酒瓶做的花瓶，插具，摆设。可以把用垃圾做成艺术品，家具，摆设，用废弃品再装饰房子，真是化腐朽为神奇。他们可以想到，做到，真的好了不起。每个人的角度理解都不同呢~ 我想，我们每个人一定都有一面都可以做着化腐朽为神奇的力量，不要说你没有，那你曾经又没有把"爱"这个字,化腐朽为神奇过呢?.......

　　珍惜身边的一点一滴,更珍惜自己,也许你随手可以点燃它们,带来更为绚目灿烂的一面-------我把这称做:化腐朽为神奇
