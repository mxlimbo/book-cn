活着

　　有些话说过了会忘，可是还是要说；

　　有些事做了没用，可还是要做；

　　有些诺言许下了不一定会实现；我们还是要庄严的许下；

　　有些人爱过了会分手会别离，我们还是义无返顾的去爱。

　　人生中，有哪些事是可以都一一实现，都能一一改变的？

　　人生中，有哪些事都能一一如我们所愿的呢？

　　明知，死亡就在那一端，宁静安详的等待着我们，我们依然快快乐乐的生活着，做着不可能实现的梦，说着会忘的话，许着不一定会实现的诺言，爱着也许会失去的人。

　　我们爱着，微笑着，生活着，这就够了，人生原是一条转不到头的路，醒不了的梦，永不断的爱。
