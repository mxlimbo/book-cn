幸福的定义

　　 我时常在想，每个人对幸福可能都有不同的定义吧.....

　　 对我而言：

　　 幸福是清晨的阳光洒满了要走的路

　　 幸福是傍晚走在空旷的校园里放起喜爱的歌
　　
　　 幸福是意外的收到了死党良心发现寄的贺卡

　　 幸福是冬夜和好友通宵烤火聊天

　　 幸福是在书店突然发现了要买的书而口袋里恰好钱够

　　 幸福是在最意外的地方碰到意外的人

　　 幸福是被人记起并想念

　　 幸福是在一个人值班的长夜里有朋远道而来陪伴

　　 幸福是在寒冷的晚上有好书相伴

　　 幸福是当熟稔的音像店主诉说一盘好的CD微笑着告诉他我有

　　 幸福是黄昏带着狗去散步

　　 幸福是午后在微风吹起的窗帘边弹吉他

　　 幸福是家里没人把音响开到最大听音乐

　　 幸福是终于写出自己想要的那种感觉

　　 幸福是和自己爱的人在一起的每时每刻

　　 幸福是和那个人同时拨对方的电话

　　 幸福是每个夜晚在台灯下静静的听这音乐打着稿子

　　幸福是珍惜身边的每一个人因为相逢是缘

　　幸福是认真的活在每一天 因为不易
　　 ...................

PS：也许还有别的，请你帮我补上吧~ ^_*
