用 半 张 床 种 花

　　在<读者>上看到一个很有趣的小插图,画一个人用他床的另一半种满了花，他正在为花浇水………墙上的钟表指向12:14分了,我们假设那是晚上12:14分吧,要上床睡觉了,床的另一边还放着拖鞋,睡觉前,给自己身边躺着的另一半浇浇水…….清晨醒来,惺忪的睁开眼,看到是会一朵含苞预放的娇羞的小花吗？那在花间醒来的心情将会是怎样的舒馨和温柔……在《缤纷》上看到介绍家居，有个人硬是把泥土搬到二楼的阳台上，只为了从卧室里醒来时，第一眼睁开就能看到窗外盛开的鲜花。

　　 画中的浇花人细看似乎不太快乐,如果我是作者,我一定把他画的眉开眼笑,看啊,这个世界上最美丽的花园就开在我的身旁。我同它们一起入睡和清醒，我看着它们抽芽，打苞，聆听它们之间那花的窃窃私语，我能听到它们绽放前的娇羞的心跳，我能看着它们舒展着枝叶在风中微微颤抖， 在夜间它们用娇柔的花瓣轻轻抚过我………..我会在床畔种上白色丁香，黄色百合，粉红玫瑰，种上木兰，水仙，山茶，天竺菊，还有那大朵大朵的波斯菊……《花仙子》的歌印象深深的曾这样唱到啊：…….大波斯菊花当作帽子戴头上，小小蒲公英在我身边连飞舞……

　　其实，睡在花边的我们又何尝不是一朵花呢？ 用半张床种花，另一半的我们原也是花，人世间的花，无论叫什么名字，都会一样的纯净 美丽 芬芳。生命象花一样热情洋溢的绽放着，五彩缤纷，我们抽出希望与热情的花苞，带着开朗芬芳的心悄悄酝酿，………..我们都是这园中最美丽的那一朵。如那句台词：Everybody is beauty……..
