衣 香 诺 诺

　　相信每个女人都有她自己的购衣经。一向不喜欢追流行，所以从来不刻意的去买牌子的东西，讨厌那种穿一个样式的T血，喝一个牌子的可乐，看同一部电影，听同一张CD，真怕会变的如同一个模子里刻出来的人。

　　99年的的冬天正在忙着买大衣。平时不爱逛商店，只有要买东西时才想起来去看。那时这里正流行中空棉薄薄的长大衣，又以黑色为主色，常见有裹着黑色中空棉大衣头发盘的高高的女子行色匆匆。喜欢，却不想扎堆，成为黑色乌鸦中普通的一只。在一个北京服饰的店里看到一件纯白色的中空棉长大衣，样子简约大方，只是腰身稍肥，可是白色的大衣在冬天太难伺候。不忍，放下了。后来圣诞节的时候看到它挂在墙上打折，看来大家都存了一样的念头，好看是好看，白色的衣服在冬天毕竟不好换洗啊。过年前，又去那个店，到处都在狂打折，大衣已经标出了不可思议的低价，看看，仍没买。转了一圈回来，店员正在给它挂一个新的价牌，已经降的无可再低了。穿上试试，周围的朋友连连称赞好看好看，只是稍嫌单调。我对店员说：不要再挂上去了，我买下了。

　　开始精心的为它寻找搭配，有时一件衣服单单好看，要想穿出彩必须有经典的搭配，或是丝巾，或是手袋，皮包，帽子，鞋……。

　　不喜欢在大的商店逛，它们想做的面面俱到，单个的方面就做不出味道。也不喜欢去专卖买东西，一件外套穿着好看高兴，碰上一个人和你穿的一样，两个人穿的和你一样，三个人和你穿的一样，这个高兴恐怕就要打折了。最喜欢的是那种独家经营的小店，老板都是极有品位的女子，从远方进到的货源，一般一个样式只有一两件，店内布置的也别具一格。在"依依"淘出了一个棉麻质地的帽子，原白色，它搭配的一件黑色大衣上的烟灰色围巾也一并买下。围巾似乎有点鹿绒的面料，很柔和的溶进了一些豹纹。戴上帽子系上围巾以后，大衣的味道就出来了。老板递给我一双和围巾同色的羊绒手套，绝配。穿着出去时有人回头看，在书店里有人在一边窃语：看，多象日本女孩………..心中的小有得意肯定是有了……。

　　年后陪朋友逛街，她买毛衣，我仍想买一件长大衣，来和白色的大衣换着穿。在一个大型的文化市场转，里面有很多很多家风格迥异的小店，转到最后一家都累的走不动了。黑色的长大衣仍是主流，一路试了几件，有合身的，但不合意。买衣服的时候常有这种感觉，一件衣服，一眼看中，穿上合适无比，又极其出彩，那件衣服就是你的，跑不了。朋友对着镜子试一件浅紫色的毛衣，我从镜子里看到她身后一件大衣的背面，烟灰色，极其卡腰，大大的裙式。我走过去把模特转过来，正面一个大翻领，有点象海军服，双排扣，简单的两个兜，羊绒面料，做工精细。心里一动，让老板把大衣取下来试穿。老板说不用试，穿不上，都在这里放了一冬天了，袖子和腰太瘦，有人试过，扣子都系不上。我固执的让老板取下，正好那天外套里穿的是纯毛的紧身黑色毛衣。系上扣子，恰恰好好，肥瘦多一分穿不上，少一分撑不起来，简直就象是为我量身度造的，老板都笑了，我也笑，不敢大笑，怕扣子会开，呵呵~ 朋友在一边旁敲侧击，过完年，春天一到大衣也就没有销路了，难得有能穿上的人老板还不卖的便宜点……..老板干脆取出进价单让我们看，最低价卖给了我们。回家后，家人一看说呵，这不是列宁式么，又流行回来了。

　　我穿穿脱脱试了好几次，把双排扣拆下重新缝制到合适位置，这次就可以大笑了。穿在身上上班时斜挎一个大大的休闲包，包也是在一个专门卖包的小店买的，一反流行休闲包的繁琐，干干净净的淡灰色，表面柔滑象是琼皮。大衣的口袋里正好放下一个超薄的随身听，拉出长长的耳机听。那个时候正在留长发，披着风大时吹起来不雅，低低的在耳边扎两个小刷子。冬日的黄昏，常常这样妆扮着听着歌，一路走过长长的街道，和拥攘的人群擦肩而过。和朋友自嘲一件这么正统的大衣被我搭配成这样，她说，不啊，正因为这么搭配才出味道，一种懒散随意的休闲，低调的精心。又是得意。

　　2000年以后是个性化的年代，追随潮流累而不菲，凌驾与流行之上，我们完全可以做人群中那个最特殊的开开心心的自己。
