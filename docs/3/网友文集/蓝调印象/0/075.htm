在 水 之 湄


　　在水之湄，晓风的文章中说，"湄"是那水跟岸之间，亦水亦岸亦草的地方，在古文中，"湄"本身就有"水边""水草相交"的意思，一注横如眼波的水上浅浅青青温温柔柔如眉一般。

　　初见这个字，便爱上了它，取做自己的名字，有几分阴柔，缠绵不清的，亦情亦爱亦恨亦愁亦哀亦泪亦喜..........亦这缠绵的尘世，离去时丝丝相连的眷恋，丝丝相连处便是湄，那湄亦怨亦喜。

　　每个人细看了，原都是系舟水湄的舟子，想系住那一番爱恨情愁，即使是铁打的汉，心中也有那清柔的一面，只是深藏在心底罢了。每个人丝丝的眷恋给了谁呢？你的舟系在何处？是否系在了那亦水亦岸亦草的地方，是否亦情亦爱亦恨亦愁？说不清的湄.....

　　解开舟，我想做个泊舟水湄间的女子，那些倦倦淡淡的心事啊，我亦是个过客，不是可驻岸的归人。月亦如千年前的清朗，水亦如多年前的脉脉，那舟桨打起的是几寸相思，几声叹喟，爱恨情愁都生在了一起，纠缠出湄，亦连亦断，亦有情亦无情。倦了，看那烟波袅袅，谁在扶琴弄萧，悠悠我心。

　　亦岸亦水，今世，我要的就是这份缠连。
