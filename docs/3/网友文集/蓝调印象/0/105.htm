爱情转了一道弯


　　三水，阿薇和琪琪是好朋友，许多年，许多年的好朋友。在学校时，阿薇和琪琪都爱欺负三水，因为他脾气好，总摆出一副任你们欺负无可奈何的样子。后来，阿薇说其实三水很聪明的，他只是在让着我们罢了。琪琪想三水是一个很能包容人的人，他的绅士气是有生俱来的，这不经意间淡淡散发出来的，恰恰也是琪琪和阿薇最欣赏的。回想起小时她们如何在三水的袜子上偷偷染涂料，愚人节骗他去校长办公室，背后贴纸条......都是他们聚会时最经典的笑话。三水每次都静静的听她们说，任两个女孩在他面前笑弯了腰，才淡淡的说："那时我不过是在哄着你们玩........"两个女孩不由的哇的一声，又笑又骂的找他算帐。三个人嚷成一团，琪琪总是最先退阵，在一边欣赏着三水和阿薇争来争去的。在别人面前总是一副大人样的三水，也只有和她们在一起才最象个小孩。面对伶牙俐齿的阿薇，三水每次也只有举手投降的份儿，连连说："好好好，我不说了。""早知道不说了还要说！"阿薇总是得意洋洋的在抢白一句，然后挎起琪琪的手商量着去哪哪儿玩。三水只有苦笑摇头跟着的份儿。

　　不知是谁说过的，三角是最稳定的关系，也许正是因为这样，才没有别人对此议论什么。好朋友，真真正正的好朋友。如果出去有一个人有事不在，那么那两个人也浑身不自在。琪琪常和三水一起等阿薇，不知为什么，她和三水在一起却没有象阿薇那样热闹。原本不是一个爱绕嘴的人吧，只是有时爱开开玩笑，吵不过时琪琪就叫阿薇替她骂三水。但是，她也喜欢和三水静静的说话。身边的朋友来了又走，最稳定的不过是自己从小玩到大的这些好朋友，所以三个人都格外珍惜这段感情。有时赌赌小气也不过一阵子的事，这种朋友吵不散了。

　　三水喜欢阿薇，这是三个人心照不宣的事，但没有人戳透。这种事何必说透呢，这样也很好啊，三水不急，阿薇也不表态。有时三水和阿薇说着说着就把琪琪忘了，琪琪在一边和他们说话没有人听见，她心里有一种很异样的感觉，两个人这么好干脆做情侣好了，可又不做。她知道阿薇是不会答应三水，因为阿薇曾经说过三水能是最好的朋友，却不能成为丈夫的那种人。也许是因为太了解了已经没有了那种感觉，可阿薇说她觉的三水有时很小。那是因为从小一起长大的缘故吧，琪琪想，我们看三水是把他从小到大叠加到一起的，而别人看到的只是现在的他。这样也好啊，每个人之间都很了解，一个人说了一句什么话，哪怕是沉默，旁边的两个人都能查觉到他快乐不快乐。只是有时琪琪觉得三水对阿薇的关心更盛过于自己，可那又有什么奇怪的呢？每个人对自己喜欢的人都这样。尽管但是，她还是希望三个人之间有的是最纯洁最无邪的友情。在这个嘈杂的世上，要防备的 ，要追逐的太多太多，希望在疲倦时在无奈时，有那么一两个人可以陪你对酒当歌，共饮共醉，同怀共感，无论顺逆，贫富，相处如初，天长地久的走下去，十年，二十年，三十年后还是如此，他们还能这样快快乐乐的在一起回想当年。

　　后来有一次大家出去，琪琪碰上一个熟人和他站在路边说话。三水等的不耐，看到那个男孩对琪琪的态度十分亲昵，随口问阿薇是谁，"是她的男朋友啊！"阿薇说。三水大惊。琪琪走过来时看到三水一脸惊鄂，问他怎么了？三水居然有几分恍惚没有听到琪琪的问话。在那一瞬间，琪琪感觉到三水受伤了，那么这个受伤是因为作为好朋友的受伤，还是因为别的呢？三水不会给她答案，琪琪知道，她心里幽幽的想：三水你心里装了几分阿薇，又装了几分我呢？难道你就没有一丝一毫的喜欢我么............这种心理是不正常的吧，在这寂寞的世上有了这两个好朋友还有什么不知足的呢？琪琪并不想失去其中任何一个，更不想两个都失去。可是为什么会这样呢？自己还有一个玩的很好的男孩，可是他有了女朋友也没有什么啊？难道我喜欢三水吗？可是答案却又和阿薇一样，三水是最好的朋友，但决不能做男朋友。可是那种若有若无的感觉是什么？不愿再去想了。
　　于是又这么过了一年，两年........三个人还是在一起玩，过圣诞节，过情人节，三个人一起过。别人看着猜不透，连自己也懒得去猜了。"男朋友呢？"三水问琪琪，"分手了，"琪琪淡淡的说。看上去很文静的自己，其实很爱动，可是处久了呢，其实还是一个很静的人。自己也搞不懂自己，琪琪不知如何向三水解释，"我们不是一种类型的人。"人就象湖水一样吧，一层一层的，有的人是一汪深潭，有的人只不过是一个浅浅的水坑。有的人每一层都有不同的颜色，散发不同的光彩，在最深最深的的湖底藏有瑰宝，见到的人就不愿再离开。而湖水的主人也会把他挽留，那瑰宝原是一颗晶莹剔透的心啊！那个人看到了这颗心却也伤心的离开了，没有带走它........为什么和他分手，琪琪自己也不知道，只是觉得越来越疲倦，不想再继续下去。"是他提出的吗？"三水小心翼翼的问，琪琪抬头看看三水却已满眼是泪。

　　不知如何解释，三个人现在在一起渐渐的却越来越感觉遥远，踏入社会越来越久，从小玩到大的兴趣爱好开始有了不同的改变，慢慢的，想法，生活态度，观念，理想，志趣.......都开始转变。在学校里和刚出门那时的感觉不一样了。校三个人的聚会次数开始减少，三水也有了恍惚和沉默，向琪琪诉苦："我见了阿薇不知说什么好.........."难道我们再也不能回到从前了吗？琪琪想，我们终于长大了，开始为自己的将来，前途，甚至还有婚姻做打算，可是我们的友情却没有一起长大。我们真的不能回到从前了吗？那些快乐的在阳光下开玩笑逗嘴的日子？那些纯真自然的日子？

　　聚会的日子阿薇却没有来，三水说："我们走走吧。"两个人顺着长长的人行道走，琪琪问："你有没有站在楼上好好看过这里的夜，虽然比不上大都市，但是也很美的。""你去过这里最高的建筑么？我们去那看看，夜会更美的。""可以吗？"琪琪有几分兴奋，可是时间有些晚了，而且那里曾经发生过自杀的事件，是不允许人随便上去的吧？可是看着三水的眼睛，琪琪觉得有好多事都是自己这样犹犹豫豫拖延了，自己不是也曾经向上去过吗？总想有一天，有机会再，就这样拖延下去........"好！我去！"
　　三水对升电梯的人说："越高越好。"那个人看了看他们，按了最顶层。"是不是看我们不象殉情的呢？"琪琪小声对三水说，三水也不由自主的笑了。到了最顶层果然非同凡想，仿佛置身与灯海之中。刚才走过的路成了缎带，汽车成了微型的模型。琪琪都一一指给三水看："人们都说香港的夜景很漂亮，但是我还是喜欢这里的，虽然它比不上香港。.....很奇怪吧！因为我是在这里长大的，香港再漂亮也不是我的家啊！我看着这里的建筑一点点建起，熟悉这里的一草一木。这里有我的家，我的学校，我的工作，我的朋友..........尽管它不是很美，..........但已经成为我生命中的一部分了............"

　　 "对它有感情了是么？"

　　 "是呀。"

　　 "今天你说的话很多。"

　　 "我心情好了才会多说话。"

　　 "你说从这儿跳下去会有什么的感觉呢......"

　　 "你怎么了？！"

　　 "没怎么！我只是想有一天活的没意思了，就从这里跳下去。"

　　 ".........."。

　　 "你那时会伤心吗？"

　　 "......为什么要这么说？........"

　　 "我有话想对你说........."

　　 "不要说了。"

　　 "你很聪明，知道我要说什么。"

　　 "..........."

　　 "我很喜欢你，真的很喜欢你。"

　　三水说完一笑，不等答案转身就走。"等等，"琪琪追上去，"你不是喜欢阿薇的吗？！"三水看着她， "我也以为我一直喜欢阿薇的，我不知道是怎么回事，...........我真的不知道........"

　　谁能告诉我这是怎么回事？谁能告诉我这是为什么？

　　这一晚是琪琪一个人走回去的，没有阿薇，没有三水，难道注定要两个人都失去吗？
　　
　　爱情走近了变成友情，友情走近了却变成爱情，爱情在什么时候转了一道弯？
