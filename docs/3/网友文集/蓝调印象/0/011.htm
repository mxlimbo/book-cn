春天里的野蔷薇

　　一个平平常常值班的下午，从通勤车上下来朝班上走去。无意中看见路边靠着墙的一丛丛绿色植物上点缀着星星点点的小白点。那是些什么东西呢，上次来时好象也看到过，看看时间还早，于是信步走过去。走近了才发现，天啊，那竟是一大丛一大丛的野蔷薇！

　　 它们在黄昏的照射下娇艳的开着，散发着清冽的香气，象梦一样安详自在的开着花,在黄昏金色的阳光里舒展着娇柔的花瓣。有微风拂来，花朵轻轻摇动，在晶莹剔透的白色花瓣底部，有极淡极淡的粉红浅浅晕上去。花蕊娇嫩的浅黄色，和白色的花瓣搭配的是那样清新自然，让你不得不赞叹大自然的细致入微，只是那么小小的一朵花，都被它设计的如此精心而美丽。绿叶间还有许多含苞未放的花苞，星星点点的分散着，象夜幕里点点的繁星。站在花丛中的感觉是如此的恬静和美好，整个人都被淡淡的香气笼罩着，我简直无法相信这就是我深深喜爱的蔷薇花啊！它们就长在我天天上下班的路边，而我却从未发现。

　　犹豫再三，终于摘下几枝，驿外断桥边，寂寞开无主，何不一路与我携香做伴？我把它们带到了班上，同事们都惊叹到如此美丽的蔷薇，却不知就采于大家天天上下班的路边。生活中有多少美丽被我们错过了呢？在阿尔卑斯山中竖起的牌子上，写的即不是禁止乱丢东西，也不是其他的什么广告和指令，那上面写的是：“慢慢走，欣赏啊！”提醒匆匆上山的登山者们不要忘记路边美丽的风景。这句话也许应该竖在所有的地方，提醒每一个匆匆赶路的人。我们不能只永远忙于赶路，而忽视了这么美丽的野蔷薇，不是吗？
