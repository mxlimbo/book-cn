那年那月

　　那年那月，我们在谈恋爱。

　　避开所有的人，我们在这个城市的边缘流浪。磨破很多双鞋子，说很多不着边际的话。最后我们发现了那家银行。

　　那是一个星星很多的晚上，我们坐在银行门口的大理石台阶上。你抱着膝盖对我讲你的故事，你的梦想。我托着腮静静的听你讲，恍若满天的星辰都在默默聆听。大理石冰凉温润，我把冒汗的手掌贴在上面，四处弥漫着花的香气。你温柔的凝视着我，我看到一千朵玫瑰在你眼中绽放，天上的星星都在款款的唱着情歌。

　　你轻轻对我说："喜欢你。""什么，"我随即又明白了你在说什么，脸上浮起了淡淡的红晕。你握起我的手，放在唇边轻轻一吻，拉起我向回家的路走去。

　　我们挽手走过街道，有人时，把手藏在身后，不松开。你送我回家，一直看着我上楼，不肯走。我叫你走，你只冲我静静的笑，我打开门时看见你的烟还在楼下忽明忽灭。

　　再约会时，你对我说老地方，我们都会去那儿，那儿有我们的星星，有我们的歌，有我们的梦，我们的天堂。

　　那儿属于我们，如同你属于我心的一个角落，不更改，不褪色。它温柔的存在着，在我的心中存在着，那是你对我说喜欢我的地方啊……如同玫瑰永远芬芳，爱永远存在。有一天我会远离这座城市，如同你。有一天这里会被搬迁，会被拆除，会被推毁，夷为一片平地，建起更高的楼房，更多的建筑，走过更多的人。没有人知道，在某一年某一个夏天，一个温柔的夏夜，有一个男孩对一个女孩轻轻的说过："喜欢你……" 。

　　亲爱的，你还记得么，那年那月，我们在谈恋爱。
