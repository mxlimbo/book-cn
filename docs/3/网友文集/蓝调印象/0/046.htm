人生中最重要的

　　这些天有些烦烦的，也许是到岁末了吧~天总是阴阴的，不下雪又不明朗，暧昧的阴着。没什么好书看，没什么好碟，没什么新CD，没有什么可以让人振作起来的东西。其实这些也都是借口吧！它们不过是平凡日子中的一些组成部分。只是我，又进入低潮期了.... 还盼望什么呢？还以为生命中会有奇迹么？还以为会有一个接一个的惊喜么？

　　在图书馆借到了张欣的〈岁月无敌〉，方佩临死前给千姿写了一封信：
　　"........ 孩子，妈妈尤其要提醒你的是，女人最大的敌人并不是贫穷和默默无闻，尽管这两点会让你深深地感到人生的乏味和无聊，但更到的敌人却是时间和岁月。当风华一过去，你定会知道踏实恬静的心态是笔怎样的财富........"

　　看的时候，心里一动。

　　很早很早以前看美国拍的连续剧〈超人〉。有一次歹徒绑架了超人的女同事凯斯琳，问她："你知道人生最想要的是什么吗？"凯斯琳脱口而出："恬静的心态！"歹徒咬牙切齿的说："笨蛋，是钱！除了钱还是钱！"当时年少轻狂，我颇为赞同凯斯琳的话，因为那时年少，寝食无忧，视金钱如粪土。上了学，上了班，工作以后慢慢接触社会，我不得不承认有时钱的确很重要，但是如果让我选择，我仍选择那份恬静的心境，而这恰恰是因为我接触了社会。我在很努力的，很努力的积攒这笔财富。有时它比无形的东西更为重要。能陪我荣辱与共，大起大落。

　　很早的时候在另外一个BBS上认识一个女孩，她写风清丽，为人总是淡淡的，很恬静的样子，鸿雁往来我们成了网上要好的笔友。他说她没有电脑，所有的东西只能用班上的电脑写，又不想让同事看到，用网名发帖子上去到是最保险的方法。我听她的话语里有隐隐的遗憾，多希望自己有这样的能力送她一台电脑，可以让她有更多充裕的时间写自己想写的的东西。相比之下我能有自己的一台电脑是多么的幸福。后来有一阵我的电脑硬盘显卡都坏了，很长时间没有去BBS，她问我怎么不写东西了，我说电脑坏了没心情，她笑了说，如同以往一样淡淡的说：是不是我没有电脑少了这些麻烦呢？不过无论如何都不要让自己停下笔来啊！
我忽然觉得她其实比我富有的多......

　　无形的东西比有形的东西难抓住吧！象一个铸铁师一样我小心的锻造着自己的心境，我不怕风霜侵袭，风华早晚要逝去，年少时的熙熙攘攘也终归宁静，我更想要一颗踏实恬静的心。　　

　　在这个阴天的下午，听着一张有关海洋的CD，喝茶，练钢笔字，写下了这些文字，这是2000年的倒数第三天。
