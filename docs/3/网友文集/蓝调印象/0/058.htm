昔日的歌-----送别依依

　　那天你忽然弹唱起<送别>,问我听过吗?在熟悉而又模糊的前奏里我恍惚着,听过,我听过的.......

　　其实我又何至听过呢?8年前的一个秋天里,曾经一遍一遍的唱起它,在空旷的操场上回荡着.....长亭外,古道边,芳草碧连天........

　　是一次校园的歌唱比赛吧,结果的输赢已经记不清楚,可那凄凄歌声,那在校园里回荡的场景,在心里如同海水般涌来,仿若自己还是那个13岁的小女骇,和全班的同学整齐的站列在讲台上,用稚嫩的声音唱出离别的悲伤与哀愁,那种无法形容的剔透和澄净,是无法用语言形容出来的啊!

　　那些旧日的时光,13岁羞祛的自己,那容易被泪水打湿的年纪,梦想和阳光一同灿烂的日子,带着无限希望和憧憬的日记啊......韶光逝,留无计,今日却分离.......

　　曾经以为自己坚持的东西也就在这些年默默改变了吧.....曾经以为一定能实现的目标也就无声的放弃了吧......往日的一些死党也都不知去向了吧.....天之涯,地之角,知交半零落,聚虽好,别离悲,世事堪玩味.......

　　只有音乐啊,能让你回到从前,唤醒记忆,把你认为曾经已经忘记的了东西轻轻拾起,温柔的拂去时光的留尘,那些少年清愁,也就淡淡的化做了悠扬的吉他,飘荡在长长的岁月里.

　　谢谢你........
