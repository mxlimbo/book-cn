不出声的电话

　　机械的打着字，当着东西，在CHAT里有一句没一句的说着话，忽然间，电话叮零零的响起来......有10秒种的恍惚，接着一个个WED蹦出无法连接，CHAT里大大的显示着--现在脱机工作，我才醒悟到：掉线了....

　　这就是电话三向连通的好处，总是能被一些顽固的人给打掉线，一遍一遍不停的打进来，线路争抢，然后他的坚持，我的坚持，然后要么他放弃，要么我掉线，一个朋友说他曾经给我连续拨电话拨了20几次才打进来，那么这个顽固的人是谁呢？我拿起话筒轻轻的问："喂？........"话筒那边一阵沉默........潜意识里的还在等着那边的人一大堆的抱怨，什么打了半天没人接啊，可是那边竟然是一片沉默，"喂喂？"我以为他没有听见，可是电话那边还是固执的沉默，是谁呢？我陷入沉思，也沉默起来，10秒，20秒，30秒....我甚至听不到那边的呼吸声，静的仿佛没有人存在，这边的secret garden奏着哀伤的调子，那被寂静压下来的沉重让人无法忍耐，："喂喂？"......."哪位？请说话好吗？"........"如果你再不说话我就扣掉了？"............啪嗒，我终于还是扣掉了电话，重新拨号，上线，找站点，重新进入CHAT。
　　　　
　　"怎么掉线了？"聊天的人问。
　　"有电话打进来了......"
　　"谁？"
　　"不知道，对方不说话......"
　　"哈哈，谁那么傻？"
　　"傻？"
　　"对啊！打电话还不肯说话！"
　　"你认为这样很傻吗？ 那么我也曾经做过这样的傻瓜......"

　　是的，我也做过，当默默的想念一个人的时候，默默的拨打他的电话，在他："喂？"了之后再无声无息的挂掉，只为了听听他的声音，总不肯去按重拨键，喜欢一遍一遍拨打那5位数字，只为了喜欢拨打他的电话的那种感觉，即使没有人接，也喜欢长长的拨号，听电话那边传来嘟-----嘟-----的声音，有那么一次忽然被人接起，仓促的不知道说什么好，电话那边传来急急的："喂喂..."不知道应该怎么办，只好一味的沉默，他说："谁？.....你怎么不说话？光我一个人说话多没意思？"天，他还是那么幽默！我无声的笑，然后眼泪滑落在话筒上，再默默的扣掉，........那是一个让人心碎的夏天。

　　嗳，也许我们应该感谢电话，它有时是拨动一个人的思念的好东西，在你不想让他知道你想他的时候，你可以默默拨号，再默默扣掉，那电话在深夜叮零零的响起后又归附沉寂，你可知道，那是一个人的心弦被默默的拨动？在你看电视，看书，吃饭，睡觉，上网.......的时候，有那么一个人在疯狂的思念着你？而他又不敢让你知道.........
