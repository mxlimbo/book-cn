《小丸子 烟火 青春 及其他.......》


　　记得在看小丸子的漫画时，有那么一段话：

　　小丸子问爷爷："青春有那么好么？恋爱，梦想及希望每天都是粉红色的？"

　　爷爷说："没有这回事呀！青春有很多辛酸，痛苦的事....内心的迷途，因为年轻人的冲劲达到顶峰...."

　　小丸子："我只会在挑选糖果时才会有迷惘呀！这是青春的开始么？"

　　爷爷："并不是这么简单的迷惘呀！青春是更加想深一层，青春并不是件乐事，失恋，对将来的不安，朋友的背叛，及其诸多......."

　　听完爷爷的话小丸子告诉好朋友小玉说："从13-14岁开始有苦恼，汗水，恋爱，失恋，读书，背叛，友情，而且还有乐事。这些便交织成青春了！"
　　
　　每次看这段话我都会笑，笑里很辛酸.....

　　朋友送我一个屏幕保护程序是烟火的，很漂亮。有时写稿子累了，倚在椅子上，看着屏幕黑下来，看着一团团瑰丽的烟火姹紫嫣红的演绎着绚丽与繁华，看着一切繁华平静后，屏幕再陷入长久的黑暗中.......
　　
　　文章里我常常喜欢用烟火来形容爱情，刹那的辉煌，照亮天空，然后徐徐坠下余生都是灰烬.....。其实青春又何尝不是呢？绚丽的时候可以耀眼，可以不知疲倦，可以拼尽一生尽君一日欢.....毕业的时候一圈人玩的很疯，深夜去放烟火，支架子在草坪上嘻嘻哈哈的烤烧烤，在朋友租的小房子里吃火锅，从下午吃到深夜，啤酒喝完了就传着喝白酒，无论男孩女孩一人一大口，那时想醉就醉不了，我们击掌而歌，畅谈长夜....更早以前，和同宿舍的一起去爬门摘槐花，翘课看电影，熄灯后点着蜡烛唱歌，在床上打着手电看小说看到窗晓发白.....那些日子都去哪了呢？

　　开始容易疲倦，总想抱怨，想起抽屉里躺着的毕业留言册，书架里和死党在海边嬉戏的照片，封在盒子里的小纸条，压在手绢底层的情书，毕业前烧毁的那一摞摞信，学校后院那棵挂满槐花的老槐树，那郑重许下过的誓言，曾经的梦想和希翼......心里总有一种涨涨的暖暖酸酸的东西，那些人呢？曾经说我们永远不分开，曾经说过要经常联络，曾经要娶我要爱我要有一个世界上最最温暖的家的人呢？
　　
　　一直一直很喜欢那篇《让岁月白发苍苍去吧》的结尾：深夜，我独自一人坐在灯下，披挂着这尘间俯仰皆是的寂寞。我知道，黑暗已经攻陷这世间的每一个角落，时间正围绕着额头穿过胸膛一泻千里疾行而去.....我摊开纸，提笔写到：让我从一开始就带上深深的悲欢........让岁月白发苍苍去吧。...........

　　如果，如果，白发苍苍的只是岁月，那该有多好.........
