错过的那一拍.....

　　我是不是应该早点察觉到呢？今年的四季竟少了一个节拍的迎合！

　　那是我烈夏后宁静致远的初秋啊........

　　初秋原是落入凡间的精灵吧，芭蕾舞鞋上足尖轻轻的一点，裙椐摇摆，天地间便已悄悄的转换了颜色。

　　春有花的灿烂，夏有阳的热烈，秋有收的丰盛，冬有雪的晶莹，而我却独独钟情于初秋。

　　承接过盛夏的骄阳转化丝丝清凉，温柔而含蓄，似有形却无痕，人影飘过，暗香浮动，只留给我那一街梧桐清清浅浅的柠檬黄怅想......

　　是因为出生在这个季节么？用那稚嫩的眼睛深深的凝视过的第一方天空，那是属于初秋晶莹剔透的蓝啊，可以盛开出爱和梦想的颜色.....

　　就要这样错过了么？10月初还在阳光明媚的海边徜徉，回来时却已是一片萧杀，阴雨连绵，落叶缤纷，屋里阴冷袭人，那四季乐章里最最婉转的一拍呵，我竟然没有踏上.........
