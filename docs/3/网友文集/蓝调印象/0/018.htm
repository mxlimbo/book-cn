给我一个愿望………

　　下班等车的时候，同事对我说了一句话，我带着耳机听不到她在说什么，她的眼圈忽然红了，看着我……。我摘下耳机，她低声说："我好朋友的丈夫死了，车祸…….你见过她的，是莉………"我恍惚的看着她，莉？是的，莉，我见过她……..我看到过她和同事在一起逛街，她曾笑意盈盈的和我打招呼，她有着明亮的眼睛和亲切的笑容……….."她的孩子才两个月…."同事哽声。我伸手默然的握住她冰冷的手，不知如何表示。"她对象才27岁，多好的年纪啊……..她以后可怎么办？她孩子怎么办？在她最需要照顾的时候……….."同事把头掉到了一边，我不知道说什么好，我怕我一张口，眼泪也会掉下来……..班车来了，我默默的上车，再回头透过玻璃窗看她，她对我疲倦的摆了摆手。

　　心情就这样沉重起来，低头，才发现耳机还别在纽扣上，戴上耳机，歌，居然已经翻面放到了《恋曲1990》："乌溜溜的黑眼珠和你的笑脸……."歌声滑过，心里一片怅然………住校的时候，，每天晚上熄灯了，我们还在说话。那时年轻的我们在争论，如果真的有神存在，只给你一个愿望，你会许下什么样的希望。我们决定从老大开始，宿舍六个人一个接一个不停顿的说下去，中间不许有间隔的时间。老大快快的说："我要做这个世界上最幸福的人！"几乎与此同时，老二也说了一句："我要做世界上最幸福的人！"

　　宿舍里静了10秒钟大家忽然一起呵呵的大笑起来，老三笑笑的嚷着说："我只要做第二个最幸福的人就好了，你们两去争第一个吧！"老四也接着说："那我做第三个幸福的人好了！"老五在黑暗里叹息，"我想让神给我一朵传说里的七色花，因为人的一生太漫长了，有太多太多的愿望想去实现…….."老五的话让我们陷入了寂静。我关掉自己照明写日记的手电，长长的思索，有人轻轻的问："六儿，你的愿望是什么……."我沉默片刻说："我要祈求神，让它保佑我周围的人都是快乐幸福的……""为什么呢？""因为，"我躺在床上微笑，"周围的人都是幸福快乐的了，我置身于其中，还能不快乐吗？"黑暗里有人说："如果真有神呐，还是让它实现六儿的愿望吧！我们都算是她周围的人啊….."宿舍里的人都笑了。

　　第二次许愿，是第一次爱一个人的时候，在深夜默默的祈祷，如果真的有神，那么请撤去我以前所有的愿望吧！什么赌气了，争吵了，成绩了，幸福了，都不要再算。我只求神让他爱我啊……….我甚至不敢奢求他爱我比我爱他多，只求他爱我和我爱他一样就心满意足…………我真的那么以为，有深爱的他在身边就足够了，一生就这一个愿望就足够了，真的足够了，我不敢奢望太多呵。

　　可是我错了，错了…………死亡一次又一次的阻击着我。它带走我的亲人，带走我的好友，带着我为他叹息的陌生人………说什么生命是可以终止的契约，爱可以跨越生死……哪怕我深爱的他和我莫不相识，我也要他好好活着啊！！！如果他爱我，纵然深爱着我，生死相隔，又有何用？！只会更添苦痛…….做世界上最幸福的人又有何用？！哪怕我见不到我的朋友们，但我知道同他们呼吸在同一片天空下，在同一片星光照耀着，心里都会有默默的惜福。活着本身就是一种最好的形式啊…….活着就不会绝望，活着就表示着有一线希望，有爱你的，你爱的，和你在一起的；有你的朋友，你的亲人，在你身边，你可以看到他们，触到他们，感受到他们，纵然见不到，你是知道，你是满足的，因为他们也活在这个世界上，纵然不幸福不快乐，但是他们也会和我们一样祈求着，祈求有快乐幸福的那一天，活着就是最大的愿望啊！如果有神啊，请你原谅我，原谅我是这样的易变和奢望，原谅人类是这样的脆弱和不堪一击。我祈求你撤去我以前的所有希望，请你保佑我身边的人，和那些我叫不上名字的人们啊！保佑他们一生的平安，请不要那样轻易的带走一个人！无论他是好人还是坏人，因为每个人身后都站着那么几个爱他的，关心他的，会为他落泪的人啊…..请不要轻易的带走他们……不要轻易的带走一个27岁的父亲，留下他只有两个月的女儿，和那个有着亲切笑容，明亮眸子的妻子………….终于还是落泪了，在罗大佑的歌里，在公车上………………..
