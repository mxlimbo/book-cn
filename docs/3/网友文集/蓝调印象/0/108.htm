冰 场 恋 人

　　我是个对自己很没信心的人,很多事情都以为自己做不了。

　　很久以前非常着迷于滑冰，看着别人轻松自在的穿着滚轴滑出去，悠然自得的转身，然后转圈，倒滑，跳跃转身……整个人简直佩服的五体投地。而自己是个正滑着战战兢兢还需要两个人扶的家伙。时常想自己如此的胆小如鼠这辈子大概也学不会滑冰了吧…..呜呼……..

　　住校的时候宿舍老大借给我旱冰鞋偷偷去练，我问她有什么技巧么，她咬牙切齿的告诉我：不怕摔！我看着她惨笑…..我既胆小又怕摔…….其实胆小就是因为怕摔嘛……..她说那我可以教你点别的，我满怀希望的问她教什么。她说：摔到了能迅速的站起来方法..。！・#￥-*%……-…….

　　偷偷找了学校最偏僻的小篮球场，晚上一个人慢慢练习。第二天拥拥攘攘的教室走廊里有人大声喊我："六儿--！"我回头，是隔壁宿舍的人。她笑逐言开的大声喊到："昨天我看到你在小篮球场滑冰了，摔了好多交，疼不疼啊？….."刹那间众多目光集中在我脸上，我面色惨白的看着她勉强微笑（看过漫画没有？漫画里主人公内心复杂哭笑不得时常有这种表情）……上帝啊，我怎么会认识这种人……..

　　这里刚开室内旱冰场的时候，人潮拥挤，10￥一张的票人人乐而不疲。也大有人约，不会滑，小男生扶你的手那是理所应当的事。在大马路上你走路有人主动来邀请你握你的手肯定被骂做神经病，旱冰场里不会。可本小姐拒绝，我发誓起码要学会正滑不用人扶的时候再进去。讨厌的人死也不愿伸出我的手，十几岁时的自己就那么娇矜。然后和朋友们哗的一大帮的去玩，看到别人不用人扶可以滑的流畅自如，心里无比崇拜，一个人战战兢兢的滑下一圈便浑身冷汗。看到有人倒滑如风般忽忽而过，想自己永远不可能有这一天了……555555555~~~~~~~

　　虽然自己学东西很慢，但是有一点，我从来不想轻言放弃。有机会就去滑，虽然每次都会浑身冷汗，如站悬崖峭壁凛冽………虽然进度极其缓慢缓慢缓慢……….但，也看的出有所进步-----我终于不用人扶了。

　　后来认识他，和他在一起，我管他叫Z 。圈里的人一起去滑冰，我这才发现他滑的很好，他的每一样都能做的很好，好到十全十美。可是，他知道我喜欢滑却从来没说教过我………。他不肯带我滑，也不是很仔细的教我，陪了一会就自己去场里转了。别人都在快乐的滑着，旱冰场上人很多，我不敢上场，在栏杆边上蹭着，扬着头找他，他在人群中很快乐笑着，若隐若现的。朋友看到说：咦，Z滑的那么好你怎么不叫他教你？我笑笑说让他自己玩吧~ 掉过头来眼泪就掉了下来，好容易碰到一个愿意把手交到他手里的人，他却不珍惜…… 一个人练倒滑的时候他忽然滑了过来说：不对不对！你怎么这么笨啊，你看看别人滑的多好！在那一刻，我清晰的看到他，这个人绝对不是陪我过一生的人。他不教我，亦无耐心陪伴爱，他对他所爱的人就是这样的么？如果真是这样亦怕我也无法忍受一生吧？在我最需要的时候他不会陪在我身边，他需要的是一个和他一样光彩夺目十全十美的人。可他却没有耐心等着毛毛虫变成蝴蝶。最后我们还是分开了，一如我所预料。

　　然后有一阵我几乎泡在了旱冰场，每周都去。总有人教我，我依然很笨，只是每个人都比他耐心仔细。深夜会无端的掉泪。我的滑冰技术飞涨，在此期间跌肿过手腕，肩膀，膝盖……扭伤过脚腕，最厉害的那次手腕肿的一个月不能碰键盘。到后来我宁肯跌断脚，也不愿摔到手，手是要用来弹吉他和敲键盘的。

　　我幻想我有一天是可以扬眉吐气的，我经常在旱冰场上盼望着能碰到Z，他总有一天会来吧，用概率来算总会碰到的，我想。那一天我要在Z面前做一个超级难度的动作让他目瞪口呆。等待中，我终于学会了转圈，在滑行中忽然停住，跳跃变换方向…….SK一直教我怎么跳起腾空旋转360度，我不敢，一直不敢，我还是那个胆小的我啊，我清楚自己的战战兢兢，我每次都会出冷汗，尽管我现在装的那么自信那么镇定自如。SK一直笑我胆小，他是我在旱冰场上认识的，我开始一直很躲他，整个人又沮丧又颓废，尽管他是那样温和有礼………..。后来我才知道他居然是学校上一届，一个系的，我居然都忘了，真是丢人丢到家了…….SK对我的好，对我的喜爱，身边的每个人似乎都心知肚明。有一次SK低头给我系鞋带的时候对我说："夏夏，我会等着你忘记那个人来爱我。""哪个人？"我扬头四处张望着没心没肺的说。SK对我只是包容的温和的笑。

　　我有练过跳跃旋转360度的，其实是想给SK一个惊喜，我几乎都做成功过。可是每次快成功的时候我总会害怕，耳边响起那句：…..你怎么这么笨啊！…… 我在这里这么拼命的是给谁证明呢？证明我不笨，证明我的爱从来没有错过……. 在旱冰场上一个人就怔怔的掉下泪来。

　　终于有一天，我在冰场上看到了Z。那天我正和SK拌了嘴，他坐着不滑，我一个人闷闷不乐的下场滑着一圈一圈乱转，忽然在前方看到了Z和另一个女孩拉手滑着。他们滑的颇为不顺，女孩有点磕磕绊绊，可Z带的很快，还让她在外圈滑。我摇头又笑，他还是老样子啊，有人会和我一样后悔吧~ 前方出现了一块很大的空地，我略微加速超到了他们前面，微微的转了一个弯，最佳的速度，最佳的角度，最佳的位置，最佳的人选，我只迟疑了一下，腾空跳起，旋转---------我能看到自己的头发成扇形洒开，白色衬衣在空气中优美的旋转起它的衣摆，包括脖子上那小小的方巾亦优雅的飘舞起，我能感觉到那旋转的快感，腾空跳跃的轻盈，战胜自己胆怯的快意，我能看到周围人艳羡的目光，我甚至能看到Z看着我微微张大了嘴巴，我成功了！ 我甚至成功圆满的做出了这个动作！我曾很多次幻想着自己会在Z面前扬眉吐气，会滑的比他还好，会让他知道，我不是他所说的那么笨……….可是，在我腾空跳起的那一瞬间我就知道那一切都不重要了，什么Z，什么你怎么这么笨啊都离我远去了，我为我自己而跳跃的，我为那个手把手，肯耐心教我的男孩做出的这个动作，感谢他，让我做的如此美丽和欢欣。Z在我面前已变的模糊，我穿过人群，穿过栏杆滑到笑意吟吟的SK面前，伸出了我的手。

　　其实很多事我们都做的来，说永远做不到的，只是我们没有去做。
　　比方，忘掉一个人。
