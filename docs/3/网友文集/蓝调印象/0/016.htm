返潮季节


　　每年春秋天的时候，总有这样一个季节穿插。室外暖意融融，而室内却阴寒难耐。如果窗外也是阴翳寒冷也久算了，偏偏窗外却一片阳光明媚，在屋里越坐越冷，阴冷的能钻进你的骨子里去。一个东北人就曾经向我抱怨说：“我们东北那都是干冷干冷的，不象你们这里怎么是阴冷阴冷的....”

　　最难受的还是住校的时候，窗外明明是阳光明媚的好天气，室内偏偏阴冷袭人。晚上睡觉没有暖气，床冰凉冰凉的，被子也感觉潮潮的，整个人都蜷在被子里打哆嗦，半天也暖和不过来。我们把所有的厚衣服都翻出来盖在被子上，抱怨这个见了鬼的季节。一个舍友说她有亲戚在哈尔滨，那个地方更冷，晚上睡觉连腰带都盖在被子上，能暖和一缕是一缕.....全宿舍的人都缩在被子里逗的呵呵直笑。冻的睡不着都在聊天，轮流讲笑话，鬼故事总是毁誉参半，有人赞同有人抗议，被子渐渐被体温暖热了人才慢慢睡去，朦胧中还能听到其他舍友低低的说话声。

　　离开学校的第一个秋天好不习惯，总似乎还能闻到宿舍里空气中那潮潮的味道，怀念舍友讲的那些让人爆笑的笑话，低低的聊天声.......。在家里上网时，在OICQ上一碰头都问你们那里冷不冷啊，都说冷啊冷啊，裹着毯子穿着厚外套的，就剩下手没法穿衣服了。我眼睁睁的看着自己的手在键盘上隐隐冻成青紫色，我打字问一个网友：“将来我们的手会不会出毛病？”他斩钉截铁的说：“不会！胳膊会！”我好奇问为什么啊？他说：“因为胳膊老悬空着啊~”我不禁失笑...有点点象住校时的感觉。

　　在里程里一点点品尽人生的各种滋味，发现心情居然也是有这个季节的，在别人辉煌灿烂时，也许你正斯人独憔悴，仿佛剧中的主人公刚刚失去了挚爱的亲人，而周围的人正欢天喜地的庆祝着什么，繁华锦绣，炮声连天，笑语盈盈，美酒芬芳，这些都成了无声的画面，一幕一幕闪过....。花天走地的全是内心的凄苦，外界的欢乐，阳光，笑脸，都成了你可望不可及的暖.....“总是无缘无故难过起来，但是音乐正在响，笑话正精彩，怎么好意思一个人走开......”

　　别人告诉我，这个季节叫返潮。我知道，返潮以后会有明媚的春，和灿烂的秋，它并不会太长，幸好，幸好.......。
