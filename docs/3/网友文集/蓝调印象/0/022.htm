好东西 慢慢来

　　中央八在放《开创世纪》，广州的朋友告诉我香港无线电视台已经放过了，周围朋友也告诉我中央八刚刚放过一个《家族风云》很类似的。可是我还是执迷不悟的看看看看看....。现在对冬天的冷风过敏嘛，只好呆在家里看电视，从晚上九点半一直演到十一点半，漫漫长夜也就熬过去大半。知道是一共有100集的时候简直吓了一大跳，天啊，要看到什么时候，那其不是每天晚上都要耗在这里？！慢慢的看下去又开始安心，太好了，这么长，能看很久很久，让我每天都有期待，让我每天都不会失望。夜幕深沉的时候我准备好厚厚的毯子，软软的垫子，热热的饮料，遥控器放在触手可及处，夜，就又开始了.....。

　　看一本好书的时候也是，简简单单的开头，慢慢看下去，情节开始错综迷离，人物关系盘根错杂，整个故事开始波澜壮阔起来，这时，我就会偷偷去看看后面页数还有多少，我真不希望看完，最好能一直一直这样看下去......

　　好的电影，好的书，好的故事，总是一续再续，因为我们不希望它们结束。电视台一天一集慢慢的放，我们一天一集慢慢的看，每天都会有期待。如果上天给我一个爱人，我亦不希望在一天内就把爱花完爱的翻天覆地电闪雷鸣。我要他每天都能看到我的微笑，我要他每天都会带着期待，我要爱情走的更沉稳更甜蜜更长久。

　　新闻组里有人发帖子问谁要《开创世纪》的完整的剧本？跟贴说我要我要.....，收到了把它稳稳的寸在OE里。不小心漏掉的情节，会去补看剧本，可是乖乖的绝对不会去看结尾，我要好东西，慢慢来------ ：）
