请在那盏路灯下等我......


　　每天,我都会在这里下班,深冬的黄昏,暮色四合,华灯初起,和着下班的人群,和同事打着招呼,明天见了,明天见,然后,各自匆匆的奔向自己的家.
　　
　　这时候的我,总喜欢东张西望,看看在街头等人的情侣,看看接孩子放学的大人,买菜回家作饭的同事.....这是我最钟爱的一个画面,它让我强烈的感觉到了回家气氛....热热闹闹的街道上有挽着朋友的,挽着对象的,挽着孩子的,脸上带着笑的,说着话的,频频看表的,.....
于是整个人,就有那么一点点的落寞......
　　
　　多希望有那么一个人,在路灯下是等着我的........
　　
　　一个人走过长长的街道,一个人穿过人群,走过黄昏,走过喧嚣,走过落寞,....在走过了四季之后,在所有的心情沉淀了以后,在看着周围的一切热热闹闹的有了自己的归属了以后, 我希望有那么一个人在为我等候.人都是怕寂寞的动物不是吗?我渴望你能陪伴我,但又不希望离的太近,我希望我们在一起,又请求你给我留下独处的空间,我们都是一只只竖起要自由的刺,又渴望爱与温暖的刺猬不是吗......

　　可是就在这个时候啊,穿过嘈杂,穿过黄昏,穿过了落寞,我多么希望有那么一个人能在那盏路灯下等我,人们奔向的不是各自的家吗?可以让身体休憩的,可以让灵魂滋养的,可以让心不再疲惫的,.....多希望你就在那盏路灯下为我等候着,风轻云淡的微笑着,告诉我,轻轻的告诉我,那颗心不用在流浪了......
　　
　　来,牵着我的手

　　　　　　让我们穿过人群 穿过嘈杂

　　　　　　　　　　 穿过黄昏 穿过落寞

　　 说,走,让-----我们------回-------家-------去-------............
