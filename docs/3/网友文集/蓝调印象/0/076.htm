在得失之间

　　将近一个月的装修终于在这个秋天的末尾彻底结束,到处乱七八糟的,我坐在木地板上,围着一大堆的书籍报纸杂志发愁,在想如何将它们各归其位.从一个旧旧的诗本里掉出几张纸条,翻开看看,你相信吗? 那竟是你以前写给我的纸条:

　　"....小人精,立即还书!!!...."

　　"六儿,快写些反映校园生活的稿子,10天之内要用!....."

　　"你做主吧,文学社聚会...."

　　............

　　3,4年的时间竟然没有把它们变的晕黄,我茫然若失的坐着,黄昏的阳光温暖的照进来,这些年,这些字条,它们在那里呢?就在这个诗本里静静的夹着吗? 这些年我们在做什么呢?上班,下班,交新的朋友,吃饭,聚会,过生日,谈几场恋爱,然后再一一失恋,考不同的试,取不同的级,学不同的东西,间杂着争吵,和好,再争吵,然后变的行同陌路......

　　有些伤感而又迷离的东西在这个秋日的黄昏悄悄的笼罩了我,曾经以为已经长大了的自己已经坚强了的自己不会在多愁善感了的自己居然为了这些纸条泪光盈盈........

　　好怀念那些纯净如水的时光,那些不知忧愁为何物的我们,终日写诗的自己,曾经以为生活就是那样的纯净美好,不用面对复杂的人际关系,不用面对这个嘈杂的社会,不用担心工资奖金调动福利待遇平等以及出人头地,还有一系列你认为不公平却要遵守的规则,如今面对它我只能一再一再的微笑,如同面对你.....

　　应该怎么办?还是这样下去?

　　他们问起我,自己也茫然的不知道怎么回答,决不原谅决不原谅,在心里默念1000遍的话语在这几张纸条里竟然默默软化,是昔日时光里的美好回忆,是这个秋日温馨的阳光,还是对时间穿梭的叹息?还记得你说过到最后的最后你要一个人孤单的离开这个城市,我许诺,我一定会去送你---离开......说这么多又有什么意思呢,你说我如同生活在梦里,其实你才是如同戏剧里的角色,这样很好,任何一个人生交到我们手里,我们都能叫它变的与众不同精彩万分,这样的结局是我们要的吗?

在得失之间
我们从来无从取舍
总有些不能忘记 又不能放弃的心愿
而心里默默排练好的角色
却常常 无法按照原来的剧本上演
但愿 从今以后
能回到最初 真实的友谊

　　　　　　　　　　　　明日最佳排挡　　


后记：

　　16日整理东西，用铅笔在一张旧旧的报纸上涂鸦下这些话，17日夜打字到电脑上决定
把它发出来，18日值班的深夜电话狂响，是你在外面的街上打来的.......

　　这算是朋友的默契吗？象是共鸣的琴弦，在拨动此根，彼根也会振动。如果不是因为家里线路的问题，这篇帖子就已经在17号的深夜发出，登在柳泉那个小小的角落里，不知道你还会用什么样的心情拨动这个号码。

　　打来的时候正关着灯听音乐，躺在椅子拼起的床上半睡半醒，电话响起来的时候有10秒钟的恍惚，以为是装置设备出问题了，在音乐里听不清你的声音便快快的说："我不知道你是谁请你等一下我去把音乐关掉。"关掉音乐跑到电话前你低低的说："原来你真的忘了我了，忘了一个在记忆里死去的人........."我沉默着赤脚站在地板上，丝丝的凉气渗透上来，仍然固执的站着不肯说话，你轻轻的笑："你掉线了吗？为什么不说话？....."于是我就笑了，淡淡的在这个寒冷的深夜里，向着电话那边看不到的你微笑着......

　　你说要向过去的一切一切告别了,所以要打个招呼问候一声，我说你怎么这么讨厌自己活在戏剧性里就非要把别人也拽到你的剧本里你深夜不睡就让别人也不得安宁........于是就象从前一样吵吵闹闹，却不过是已行同陌路。我叫你去回家，去上班，去好好对朋友，要你去面对现实，你却不肯向这一切妥协，你说:给我一个理由............我的理由真的很空----因为我们活着一天就要负一天的责任，对家人负责，对亲人负责，对朋友负责，对爱自己的人负责。你说活的太累，抬头看看这个世界上谁活的不累呢？谁不是一脸的倦尘？

　　不知道说了多久的话，我说要去睡了，你说好BYEBYE ，心里低低的浮起那句诗：我欲成眠君且去，明朝有意抱琴来。可惜我们没有明朝，琴弦已断，旧歌难闻......

　　打完电话时整个人已经冻透，换来的是后半夜翻江倒海的胃疼，想不到住校时落下的病根现在又会复发，无法入睡，坐在桌子上围着被子静静的听歌，想起那些住校胃疼的深夜总是拿着厚厚的一摞书抵着，倒抽着冷气故做轻松的和宿舍老二低低的聊天，那些日子也就那么过去了。从来不敢去回忆，总怕往事太伤情，甚至任性的说过人们不能总活在过去回忆是毒药的话语，现在，就让我忘情的回想过去吧，回想那些洒满阳光的快乐的纯净的日子.......

　　今夜仿若格外的漫长，漫漫无边的黑色把我笼罩，下一首歌是《寂静的声音》啊，我知道那低迷飘缓的旋律又将响起，同是一个寂寞的人会用他暗哑的嗓子低低吟唱：黑夜，你好吗我的老朋友，我又来与你聊天了.............
