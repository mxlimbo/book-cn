心里的门


　　我想每个人心里都有很多门，很多扇很多扇，不同的人，手里拿着不同的钥匙。每个人心里都有那么一扇小门，关着自己心里最温柔，最易感的一面。有的人拿着钥匙，正好的钥匙，打开了这扇门。于是那扇门的主人，向他展示了自己也有很温柔很脆弱的一面。有的人也拿着那把钥匙，他也许打开了门，却没有走进去。最恨的是，有的人拿的钥匙根本不是，还要乱试。有时，门也许会自己打开，也许是一句话，一个眼神，一首歌，一个午夜，那扇锁住温柔的门悄然而开了 有时，也许，在某一个特定的环境，对某一个特定的人，打开了，而他却没有珍惜。于是，那扇门又关上了，比以前关的更紧，更严。人们说起他，哦，他是多么的冷酷。可谁有知道呢？他心里的那扇门，锁住了无限温柔。
