相思秋季

　　当初秋刚刚地落下足尖，当天边的第一颗寒星刚刚升起，当夕阳那抹余辉温柔的滑过天际，你心里柔柔的浮起轻雾，秋天已经悄悄地，悄悄地偷走了你的心。

　　有多久没有注意过晚霞，有多久没有凝视草尖的那颗露水，你心里葱茏的季节，走在阳光象金子般洒落的黄昏里，渐渐晕黄。楼顶飞过的鸽群晾出了白色的翅，阳光镀在每一个人脸上，染上每一朵野花的芳香。蔚蓝的天空，金黄的梧桐叶，灿烂的阳光，滑过天空的鸽群，还有那更远的你看不到的山间，漫天飘舞的蒲公英，飞扬缤纷的芦苇，更远更远的地方，落着雪花。每一个凄清的夜，星星更高，更远，在梦里和你遥遥相望，说着谁也不知到的呓语。

　　你可曾听说过，每片落叶飘坠都带着一个愿望，那是与你相约啊，明年的秋天再见。

　　别离后，相思竟成了一棵没有年轮的树，永不老去……
