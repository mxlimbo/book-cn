冬天 啊 冬天 ！

　　我知道冬天又到了，我不喜欢冬天的，甚至可以说有点讨厌。

　　以前小时候上学喜欢冬天。没有一个孩子不喜欢雪的，冬天会下雪，很大很大的雪，可以不用做操，不上体育课，全校的学生都在操场上打雪仗，堆雪人。冬天会有长长的寒假，可以不用早起，舒舒服服的赖在被窝里，不用温习功课。冬天过新年，枕头底下会有红纸包的压岁钱，有新鞋新衣服新帽子新的一年，可以去亲戚朋友邻居家里串门吃糖------

　　而现在我长大了，不再有假期，山东这个地方一年四季分明，夏天可以热到将近40摄食度，冬天照样冷的要穿羽绒服，每次下雪的时候，我们的上班车要在路上跑2，3个钟头，班车上的人都在跺脚，为什么要跺脚呢？因为路滑雪大车开的很慢，时间太长实在太冷了脚都冻的麻木掉，全车的人一起在跺脚....。更糟的时候车开不动了我们还要下去走路，走很远才能到班上，下班的时候同样再辛苦的走回去。学生可以放假，我们不可以，因为我们是大人。冬天又不能出去玩，室外太冷，每年过冬还要穿的厚厚的笨笨的，没有新衣服，现在的我要给别人红包，每过一年又老一岁...........这些还都不足以成为理由，最让我不能忍受的是每年冬天我都会生病，无论我穿的比别人多，还是预防的比别人好，每年入冬都会先感冒，然后高烧，然后低烧不退，运气好的时候不引发胃炎，可是我一般运气都差，今年已经得了两次急性胃炎。现在又多了一样，我对冬天的冷风过敏，不能出门，否则会诱发起疹，很痒很痒.....。虽然防过敏药能控制住一点，可是又不敢多吃，怕会降低抵抗力.........如果是你，你会不会讨厌冬天呢？

　　上学的时候并不爱生病，我也不挑食，不抽烟不酗酒，喜欢运动打球滑冰，没有一切坏习惯。刚感冒的时候也不吃药，想喝点热水抗过去，可是别人抗过去了我会转为低烧，打掉瓶都退不了。到最后医生都不敢给我开点滴了，她说你不能再打了，再打就超量了。 坐在一圈打点滴的孩子老人中间，耳鸣不止的我简直悲哀的不想再过下一个冬天了......

　　讨厌透了冬天后，我开始乱七八糟的想，我为什么不能喜欢它，象以前一样喜欢它？在我们很不高兴的时候，有时会假装自己很开心让自己高兴起来。好比有时我们不喜欢吃某样东西不喜欢看某个节目，有时假装自己很喜欢吃很喜欢看的。假装假装着，慢慢的，开始真的开心起来，真的喜欢吃那样东西，它模样怪怪的，可是味道很好，那个节目虽然有点假，但很有趣。我开始假装自己很喜欢冬天，冬天好啊，冬天很冷，大家都要穿上厚厚的衣服，每个人看上去都胖胖的,笨笨的,象波比熊。冬天的时候班上会分很多苹果，我都会买了沙拉酱做沙拉苹果给大家吃。我们甚至可以偷偷的用烤箱烤地瓜吃，唯一困绕的是不让香味传的那么远让头儿闻到 ：P 至少今年还没有下雪，温度还没有降到最低，我的感冒啊发烧啊胃炎啊都得过去了。同事告诉我她冬天更惨，每年她过冬手脚都会裂，治了很久都治不好..“那怎么办啊？很痛的吧~”我关心的问她，她笑笑说“那有什么啊，一年才一次！”我也很信心十足，是啊，一年才一次嘛~ 冬天还有很多振奋人心的节日没过呢~平安夜啊，圣诞节啊，元旦啊，新年啊，情人节啊....年底一般都会有大笔的年终奖金，可以用来犒劳自己，买很多很多的好东西。在外地工作上学的朋友也都回来了，带来外地新鲜的事物礼物话语，在这个冷冷的冬天里带来热乎乎的友情。冬天又可以看到卖冰糖葫芦的，干干净净的放在玻璃柜子里，还有满街飘香的桂花糖炒板栗，热乎乎的香气四溢的捧在手里。冬天可以有更充足的理由让自己吃巧克力，补充热量嘛~

　　我真清醒虽然坐班车漫长寒冷，但毕竟还不住在东北 ：）

　　我真庆幸自己虽然爱感冒发烧，但毕竟还没有严重到住院 ：）

　　我真庆幸自己虽然有胃炎，但毕竟还没有更严重，朋友说医生让她做胃镜她自此一个月都不想吃东西 ：）

　　我真庆幸自己虽然对冷空气过敏，但还没有手脚开裂，急性荨麻疹来的快去的快没有给我留下任何疤痕，我可以坐在屋子里电脑边吃朋友带来的热乎乎的糖炒栗子 ：）

　　我真庆幸自己开始一点一点的喜欢起冬天，虽然它带给我无限烦恼，但是我爱上了它的缺点，这是不是表示着我会爱它爱的更牢固一些呢？凡是事物都有正反两面啊，有不喜欢的，也有喜欢的，况且还有那句老话撑着呢：冬天到了，春天还远吗？
