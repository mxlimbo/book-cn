美好的......

　　朋友和我讲，他今天在街上看到一个极美的女子，只是一个背影，窈窈窕窕，纤步袅娜，长发如云，他禁不住追上去，看到那女孩的正面却大失所望，悲叹，早知如此，还不如不去看她的正面，也好在心中魂牵梦系一番。

　　上中学的时，进办公室交作业本一个新老师迎面出来，惊艳，想，我们中学居然有这么这么漂亮的老师啊.....后来她教我们英文，在课堂上天天看她，怎么看也没初次漂亮了，死党们也纷纷抱怨说我夸大，其实她真的很漂亮，只不过天天见就习惯了。

　　在外地的朋友回来，陪她在街上闲逛，坐下来喝可乐的时候，我看到当年她痴恋大我们好几届的他从商店走出。我小声说："你看，是他啊......"她低头浅浅的笑，缀着可乐却并不回头："他怎么样了？""据说结婚了，挺幸福，只是胖的有点走形，呵呵，不仔细看，还真认不出来他....."她也笑，仍不回头，"不想看看他吗？"她摇头低声说："只要他在我记忆里永远那么年轻，永远那么美就好了，现在见不见他都不重要。"我心里一动.......

　　许多年前，我还是个学生，很晚了，在看书。忽然听到楼下有人大声的喊叫女孩的名字，对面的楼就有人推开窗户说："你回去吧，我不会下去的！"是女孩的声音。楼下的男孩叫声仍是不断："你下来，你下来....."我好奇的掀开窗帘悄悄去看，楼下婆娑的树影中，只能看到一个模糊的白衬衣的影子在晃动。叫声慢慢沉寂了下来，轻轻的，楼下响起了悠长的口哨声，不知道吹的什么曲子，百转千洄，荡气回肠，那时的我哪还有心思看书，爬在窗台上静观其变，楼下固执的白衬衣影子不肯离去，对面楼的女孩也不肯出来。在安静的夜里，哀伤的口哨声，象水波一样缓缓荡开，让每个听到的人都为之心动....我想我从没听过那么美，那么柔肠寸断的曲子，溶进了一个男孩年轻的爱，年轻的期盼，憧憬和心碎...... 我似乎是听着口哨声睡着的，醒来的清晨，一如既往的晴朗，窗外无人，更无悠长的口哨声，昨夜竟如同一梦，散的无影无踪。自己也慢慢学会了吹，但怎么也寻觅不到那首曲子，在电视上看过专业口哨的表演，但总也比不上心中的那首凄美，那一夜已成为绝唱.....

　　讲给姐姐听，她一歪头笑了，说："有一次，你刚把小钢琴拿出来，信手在上面弹了一首曲子试音。我在屋里看书，听到你弹的优美异常，不敢去问你弹的什么曲子，甚至不敢翻动书页，生怕你弹到一半就不弹了....""哪首哪首？"我禁不住兴致勃勃的问。"就是你经常弹的那首《乱世佳人》啊~！""啊？那不是我经常在弹的吗？""是啊~"姐姐微笑："就是因为你现在经常弹，听多了，再听起来怎么也没第一次听那么美了。"我笑，那夜的口哨再无耳闻，这对我来说又何尝不是一种幸运，它在我心中已成天籁。

　　逝去的美丽，在心中留下淡淡的影子，往往比他们清晰的时候更为瑰丽.......
