谣言

　　就在今天下午里.......忽然很想见到你.......,于是打了很多个电话,找了很多个借口,只是为了去见你.........。

　　11月的天空还依稀晴朗，在我打字的窗口能看到有鸽群飞过，为什么不呢~于是就去了.....自己真是一个如此性急的人啊！

　　走走停停，熙熙攘攘的人群，蓝色的天空，纷杂的目光，凌乱的建筑，心不在焉的自己，多希望能这样不经意的走过大街小巷，不经意的走过人群，不经意的走进你的视线里........

　　可惜见到的你是如此的平淡，就象今天的天气，不冷不热......不冷不热.......脑子里就飘飘的想起《红胶囊》里的一些话：我们大部分的时候，多太在意别人怎么看我们，都太害怕不别别人喜欢，所以，就象受咀般的不感表达心里的想法...... 有时候我们似乎感动了但.....其实我们并不是一真的能了解生命里的这个那个,只是从别人身上学习,跟着大家的做法似懂非懂的爱着,本本分分的过着........

　　我想我回家比呆在这里好吧~呆在我的蓝色小屋里,打打字,写写稿子,看看鸽群飞过,不也是挺好的吗?

　　于是我对你说我要走了,你看我又是一个如此随便的人,晃手里的车钥匙离开,然后就溶于这个黄昏里,路边的印象店里在放黄磊的歌呢~:

　　想冷淡转身而去 不再管这世界 走开 走开 走开 那么浮躁的情绪 让人心情好坏　　我们究竟为何存在 人们从何时开始无法真心对待 你猜 你猜 你猜 多少隐忍的悲哀 在倔强的姿态 听到谣言如风吹去吹来 翻腾在人海 谁知道风从哪里来 今天是和等精彩 明天就忘怀 哪怕有人受伤害 听到谣言如风吹去吹来 翻腾在人海 谁知道风从哪里来 也许把黑说成白 恨也说是爱 这就是我们的年代.........

PS:在凌晨后下线后乱乱敲打的,自己看了看却忍不住笑了,也许是真的困了,不知所云,于是违反了自己一向精心修改整理才小心发的习惯,就任性几回吧~

　　极不甘愿的人生旅程,我们都很难逃开;每当遭逢生命中不堪的时刻,总有这样的感觉......但是,反正不可能回头了,就勇敢的前进吧~ :)
