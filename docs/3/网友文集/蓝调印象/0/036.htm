美 丽 香 片

　　忘了在哪篇帖子里写过，我是只一个喝咖啡的人，很少很少喝-----茶 ：）

　　说起茶，总有这样的印象，有高高的檀色桌椅，镂空的木制窗户里洒进细碎的阳光，紫沙的茶壶.....旁边坐着的应该是一个珠环翠绕，抚琴弄萧的小姐，或者是一个长袍飘飘，吟诗做画，眉目俊朗的男子....茶不应该是一个出现在这个爱情都象即食面的时代吧~

　　朋友看了我的帖子，几日后，诡秘的笑着送了我一包香片，说你会被打动的。说这是一种香片，其实全部是由花组成的，有玫瑰，菊花，茉莉，决明子，和有一些我叫不上名字来的小花，美丽的象被做成干花一样摆放着，用玻璃纸精致的包装好。用开水冲泡的时候尤其漂亮，微微晕黄的菊花，深粉色的玫瑰，洁白的茉莉.....透过透明的玻璃杯，看着它们荡漾在如同琥珀般的茶水里，陶醉的小缀一口.....唔，真是又苦又涩......用难以下咽来形容它真是一点也不夸张~ 放到晚上玩电脑的时候，懒得起身再去倒水，忍忍喝掉，后来再续了几次水，一边打着字，一边无意识的喝下去，忽然觉得入口不是那么苦涩了....有着淡淡的疏离的花香，似有若无的苦，清淡的草药味.....低头看看茶杯，如同干花般的漂亮香片却都没有了看相，玫瑰泡成苍白，菊花快没有了颜色，茉莉早已散了花瓣........可茶偏偏这个时候才有了味道，才入了口...... ：） 生命中总有这种玩笑吧~ 美奂美论的香片难以下咽，偏偏等它泡的色泽尽失的时候才渐入佳境。如同他们说的，可爱又美丽的女孩可遇不可求，想尽善尽美，连一杯小小的香片都不会让你如愿呢
