惆怅旧欢如梦

　　黄昏时候，天空开始零星的飘起雨点，小说里总是在这个时候开始有故事发生的，现实里也总是如此。

　　我踩着一点一点落在地上的雨滴，用Disman听着蔡琴的歌。喜欢她唱的那首《缺口》，喜欢她在歌曲开头前轻声的哼唱，她的声音给我一种旧时光温醇如红酒的感觉。在这种气氛尤佳的时候常常会跳出来一个两个扫兴的人，我抬头看到前面那对亲热的情侣，他们的左右手不惜余力的全部缠绕在对方的腰上，用一种奇怪的姿势向前横行着。如果他们大方的挽着手走路一定会比现在舒服些，也比较适合这条落雨梧桐的街道上的气氛，我想想，再看看他们夸张的动作，忍不住一个人偷偷笑起来。再抬头看他们时，对面走过来的人也在看他们，脸上拼命抑制着一个正要浮现在脸上不礼貌的大笑。我微微怔了一下，这个和我具有同样幽默感的人居然是你…….。

　　早该处世不惊了不是么，这个世界很小，我们住在一个城市，一共上了两所同样的学校，在第一所学校时，你快毕业了，你发现我，你祈祷我能再见到我。果然，我们又相遇在另一所学校。他们说这叫有缘，我不相信缘，我只相信巧合。就象今天，不过是巧合而已。在这条路上上班下班走了有八百遍了，巧的是当我们都在笑那对做作的情侣时我们碰到了。你收回目光，视线撞到了我身上，你惊讶的看到我，然后一怔，我已经微微点头走了过去。
我永远不再是那个十七岁的女孩子了，会为我们的一次相遇，一次争吵而翻江倒海。分分和和这么多年，我足够能坦然的面对这一切，包括我们漠然的擦身而过，生命里没有不能承受的，只是你没有经历而已。

　　感谢天，你还是老样子，没有人能和你一样把白T血蓝牛仔裤穿的如此出彩。记得我说过你穿白T血蓝牛仔裤好看，你时常穿着这一身在我面前晃，然后小声告诉我："其实白T血很难洗。"我笑不可抑。我也几乎是老样子，只是又长高了两公分，二十岁以后我还以为自己不会再长个。为你剪的齐腰长发现在又慢慢长了，他们说明年的冬天就可以长到腰了，我又可以梳自己最喜欢的那种两个长长的麻花辫了，你常说那样的我就象一个印第安女孩。
你看，过去的点点滴滴我都能象起来，虽然不经常想，但是它们都能牢牢的保存在我的记忆里。张小娴的文章里说：你应该庆幸自己过去的爱人还是那样英俊潇洒，那毕竟是你爱过的人。是的，我们其实都该庆幸，不是么？

　　做作的情侣走过去了，你也走过去了，雨滴沙沙的打在梧桐叶上，蔡琴在唱："……生命不断的转弯/起起落落变成习惯/爱情象是考验/从不沉默永远……."多好的词，爱情象是考验，从不沉默永远。他们说你和那个女孩子最终还是分开了，告诉我的朋友看着我目光闪烁，我替他把话说出："你想问我会不会和他和好是么？告诉你，不会。"他不解："为什么？当年你们曾那么好…"当年？我低头笑，当眼泪掉尽以后我习惯用微笑面对一切。我听到自己的声音很遥远的说："旧欢如梦…。"

　　一生中能刻骨铭心的爱过几次呢？如瑞德对斯佳丽说的："如果我的爱用钱来比喻，我的爱是1000块钱，过去我把每一分都花在你身上，而你不要。现在你再来问我要爱，对不起，亲爱的，我一分钱也没有了。" 是的，我的爱也业已消失怠尽，如同瑞德说的1000￥，我每一分的憧憬，每一分的热情，每一分的狂热，每一分的思念，每一分的渴望，每一分的哀愁，每一分的爱------亲爱的，我都给了你。我知道，你也所剩无几，我们都扬着头似毫发未伤的活着，甚至比别人活的更好，没有人知道我们内心的破碎不堪……..

　　我试图去爱过别人，如同你的试图，我可以用温情去爱一个人，可以承受一个人对我的爱，可是他们总向我索取太多，他们要这颗心，哪怕它破碎不堪。我剩下的还有什么呢？你耗尽了我的爱，我只剩下了这颗破碎不堪的心，和心里那一小小点点的梦想，我要靠着它们支撑下去，不会再给任何人。

　　我时常想，也许我爱的不是你，而是我们曾经走过的那段岁月。十七岁的时候我告诉你我只会做蛋炒饭，你拍着手说太好了你最爱吃的就是蛋炒饭。现在我已经做的一手好菜了，你知道么？可是我只愿意给自己做饭吃。你请我跳第一只舞时，我紧张的直踩你的脚，你连声说着不要紧不要着急却又踩在了我脚上。现在我连快三都跟的上了，可是我再也不会与你共舞……..我爱的是你是我还是那段岁月，它如此的刻骨铭心，让我心甘情愿的耗尽了全部的爱………….

　　在这个世界让我们必须坚强，哪怕你心里在哀嚎在滴血，人前你还是要笑脸相迎，不让别人闻的到一点点腐烂的气息。即使受伤，也把自己变成小小的兽，在属于自己的洞穴里舔舐伤口。久而久之我们会忘记痛，偶尔会想起对方，嗨，你还好么？我现在一天要上8个小时的班，2个小时在路上，下班以后每天有3，4个小时泡在电脑上。我睡的很晚，夜很长…….
我有快乐的理由，但是我没有再爱一个人的理由，我已经没有那1000块钱了……..

　　现实里，我们永远站在走廊中，幽深而绵长，我们千心万苦的去打开一扇门，我们往往发现在那门后还有一扇门，打开它，门后还有门，永远都是一道一道的门，而我现在放弃了，我知道那扇门后放的是什么，也知道开门的代价………

　　"……….有一点遗憾/幸福没有答案/什么不能计算/谁能够抚平悲哀/不必再去感叹/要笑着把眼泪擦干/夜晚是个难关/寂寞需要勇敢………….."

　　回头看看，一街空寂的落叶，我知道你刚才笑什么，你在笑那对情侣，当时的我们潇洒的不用挽手，爱就已洋溢在了脸上……..只是，一切以惆怅旧欢如梦………….
