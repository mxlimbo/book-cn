另一张脸

　　有时想，上帝很不公平，他给了每一个人不同的脸，有美的，有丑的。有美的一张脸的人似乎活的要容易一些，因为人们一向对美的事物寄予更多的同情和帮助。而有丑的一张脸的人，要付出更多的艰辛和努力去得到别人的肯定。可是呢~上帝也公平，他给了每一个人同样的心灵，从你出生的那一天起它们都是一模一样的，然后完全由你自己来决定它的美丑。是的，我们不能决定自己的脸，但是可以决定自己心灵的美丑。它是人们看不到的另一张脸，由你自己来决定它的卑微和伟大，脆弱和坚韧，宽容和吝啬。

　　我的朋友曾给我讲过一个童话叫骷髅城。传说那里面的人住的都是骷髅一样的人，他们没有脸全都长的一模一样，人们不会因为一个人长的漂亮些难看些来评价他对待他，写童话的人也许认为这样才是真正的公平和平等。童话毕竟是美丽的，谁不曾有过这样的向往，人们就不会因为美丽而眩目的东西迷住了眼睛，忽视了内在。

　　天长日久时光流逝，你的脸开始慢慢衰老，起皱，另一张脸渐渐浮现出来，象是你本来的面目开始慢慢呈现。有的人有孩子般纯洁的目光，有的人却有了怨恨恶毒的眉角，有的人宁静的微笑，有的却带着刻薄的嘴角。那是你心灵的脸啊！这些年你象一个画家一样，一直在涂涂画画着自己的脸，只是你自己不知道。有的人看到了，有的人却被一些表面的东西所迷惑了，他们更在乎的外面的那一张脸。而你心灵的脸永远在内心深处，不怕时光流逝，岁月侵蚀，长青不老.......
