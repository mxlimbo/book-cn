《爱情》 你怎么说？

　　终于终于，当到了这首歌，莫文蔚的《爱情》......
　　
　　这首歌太老了，以至于在很多站点只有她的新转辑，没有这首歌的MP3，也怪我，想换声卡磕磕绊绊一年多才换上。然后，开始在各个音乐站点寻觅这首歌。一年多的时间没有声卡，也漠视了许许多多的音乐站点，只在文学的站点上打转。当我试着从记忆里搜索起音乐的站点，寻寻觅觅，寻寻觅觅，终于，音箱里响起《爱情》纯净的前奏。一连串碎玉倾洒般的吉他，伴着细碎的钢琴，我心里一阵欣然，莫文蔚独特的声音响起来：......若不是因为爱着你 怎么会夜深还没睡意 每个念头都关与你 我想你 想你 好想你........

　　当年的自己痴痴的守着电视看完了这首MTV，曾经深深被它打动，并且深深相信爱情是那样的纯真和天荒地老至此不渝。磕磕绊绊的走过来的青春时光，受过伤，也伤害过别人，再也不是当年稚然的自己，再次听起这首歌，歌声却还是那样的纯真和执着........
　　
".......爱是折磨人的东西 却又舍不得这样放弃 不停揣测你的心理 可有我姓名 爱是我唯一的秘密 让人心碎却又着迷 无论是用什么言语 只会 只会思念你......."

　　当我们得到过，失去过的时候，不仅有迷惑，什么才是永恒的？

　　是如传说中说的，我们原本是石头，爱情赋予我们亮丽的色彩，于是我们才有了瞬间的辉煌......

　　 还是我们应该实际客观起来，爱情原本是神话，只有男女因共同生活培养出来的默契命运与共的感情.........

　　那种才是我们要的？那种才是我们寻找的？

　　是否，又是否，我们在爱里受伤以后，不应该因为爱的痛而失去那颗承担的心？

　　如同林清玄说的：要寻找到生命最内在的本质，是不能有任何借口的。当我们还有借口的时候，本质就不会显露出来.......

　　庆幸的是，《爱情》这首歌，还如同我以前听到的一样纯真甜美，即使人不同了，心不同了，年年岁岁花相似啊............
