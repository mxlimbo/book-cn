死 后

　　好象活着的人都很忌讳说死后的事情，尤其是在病中。

　　当阳光灿烂，秋风爽朗时，一个人静静的想想倒也并不可怕。

　　和他在一起时，他曾说过，如果有一天我先他而去，他会把我的骨灰放在一个小小的皮囊里随身带着，然后去西藏当喇嘛。按说应该被他感动的淅沥哗啦一番，可是我却没有，非常好奇的问他为何一定要去西藏当喇嘛。他楞了楞，不好意思的说："我喜欢西藏啊……."我抚掌大笑。

　　很多人死后喜欢把自己葬在墓地里，一年一度的总会有家人来看自己。墓地修的越来越豪华占地越来越大…..其实真正的想念存在心里便可，为何一定要做出一种仪式一种姿态？花上万元的陵墓和黄土一坯里埋的还不是一样，臭皮囊一张枯骨一堆。

　　本想死后把自己的骨灰撒倒那些我想去而最终没有去过的地方，可又觉得太麻烦，撒着麻烦，撒的人也麻烦。再说那些地方亦是我不熟悉的，纵然我渴望去，纵然美奂美仑，也不一定喜欢，到不如自己成长的地方来的有感情。还是把骨灰埋在那棵楼口的芙蓉树下吧，它已经深深的刻进了自己的名字，朝朝夕夕就象我生命中的一部分。我可在树下安详的回忆起我的童年，我的少年……..在阳光里跳过的方格，和死党坐在树下的石凳上长聊，早晨拿着面包匆匆的赶车，和他在楼口分别时的恋恋……

　　有一个朋友曾经对我讲过，传说，那些在海面上飞翔的海鸥都是死后的水手变成的，他们一生都漂泊在海上，死后也是如此的眷恋，它们在海上嬉戏飞翔着，慢慢的等着他们的同伴也变成海鸥。

　　我不会让我爱的他去做喇嘛，我知道他也不会去的。虽然不在一起了，我知道，若我死后埋在可棵芙蓉树下，我一定能听到我年少时和他的窃窃耳语，听到我们的欢笑，争执，分手……….年老的我会不会在树下为我年轻时犯下的错辗转反侧呢…..

　　如果我在海边出生，如果我是一个水手，我也一定渴望变成一直海鸥吧~无论生前还是死后，我们都渴望在自己深爱的地方存在着，哪怕不是永远……..。
