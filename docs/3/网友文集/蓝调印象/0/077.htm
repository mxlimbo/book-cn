怎样对待生活

　　昨天晚上，我们聊了那么久，你向我长长的叹息，什么都不顺心，什么都不如意。刚刚踏入社会的你对工作不满意，家人不支持你，朋友不理解你，你抱怨你的一切是那样的不顺心不如意。

　　你让我想起了我的一个朋友，她并不幸福，父亲早年去世，她和弟弟完全靠母亲微薄的工资生活。家庭条件差，弟弟又年幼不懂事，让她和母亲费尽了心血。可是，我从来没有见她抱怨过什么，她曾给我讲过小时侯的生活。那时她爸爸在一家工厂上班，工资高，全家又住在靠近工厂的一家小村庄里，种着地，还养着鸡，生活已完全自给自足，丰衣足食。上班余暇爸爸带领着全家下河摸鱼，上山打野兔。春天来得时候，院子里的槐树开花了，香飘满院，妈妈做一大摞一大摞的槐花饼，连同自己腌制的直冒油的咸鸡蛋吃都吃不完。班上同事们常常来玩，一家人热闹非凡。她爸爸去世时，刚满三十岁，正当年轻力壮之即，只因为一场疾病。她常常叹息的倒是她的弟弟，说当时他年纪太小，对这些事已经都不大记得了。她给我看过照片，她的父亲十分英俊。她说自己一点都不后悔，因为度过了一个完美到极点的童年，她印象中的父亲永远年轻潇洒。她对我说过的一句话让我终身难忘：生活都是以豁达的姿态对待每一个人的，有时候觉的自己受伤了，只是因为我们太在意自己了。

　　为此，我敬佩她一辈子，敬佩一个困难打不倒，用豁达乐观的姿态对待生活的人。

　　朋友，你有没有想过，生活都是公平的，它给予你了此，就不在予于彼，从来没有十全十美，不然又怎么称之为生活？你怎么可能要求它处处称心，处处如意？能改变的只有你的态度，看你怎样对待生活，你可以乐观的对待它，也可以悲观的对待它，而生活对待你是不会变的。我们并不否认有一些人确实运气好，你用尽力气才做到的事，而他却不费吹灰之力的得到了。但也许在将来，同样一件事把他打倒了，你却依然伫立，因为你经历过，磨练过。这不是没有可能，因为好运气不可能一直跟着你，任何人也不可能帮你一辈子。我想我们都还是太年轻了，在我们身边有很多很多的幸福我们忽略了它，却去注意了别人的幸福。

　　乐观起来吧，my friend ，生活就象一面镜子，当你对它微笑时，它一定也不会忘记对你，露出一个阳光灿烂的笑容.........
