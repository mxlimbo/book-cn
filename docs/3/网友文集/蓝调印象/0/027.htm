寂寞之旅

　　说起寂寞，总想起那么一首小诗" 翻开书页/寂寞/合上书页/寂寞 "想这个作者也是个寂寞过的人吧！有什么能这样无声无息悄悄的弥漫上来，重重的压在心上，给你的脸涂上一层落寞。

　　有时，寂寞是无人陪伴。与你在一起的只有你的影子。没有人和你说话，没有人倾听你的心事，哪怕有那么一双默默凝视你的眼睛呢！

　　更可怕的寂寞是，当你置身与人群中，却没有那份归属感。在混乱的谈话中，你找不到落脚点，在思想交流间，没有迸出令人兴奋的火花。没有同行者，没有人分享你的精神，你的想法。你在说，可没有人听的懂你的话，或是几句应付的回答，或是麻木的表情，这只会让你更难受。你停止了徒劳的对话，默默无言。

　　你想同人分享，当你领略了一本好书，一部好的影片，一首美秒的乐曲。你想让人同你一起分享那些微妙之处，分享你灵魂深出莫大的喜悦。可是没有，没有一个人。

　　在你，继续孤单的向前走时，没有人与你同行。你怕一个人，怕没有侣伴，怕与后面的人的那段遥远的距离，但你更怕你遥望顶峰的那种感觉。你义无返顾的向前走，尽管这一路，注定了艰辛与孤独，你别无选择。

　　只有寂寞，与你一路如影相随。
