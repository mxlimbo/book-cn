写作时的心情……..


　　拿着笔涂写，在电脑前敲打着按键，编织着自己的心情，编织着故事，那种心情快乐的无以比拟。看着自己笔下的人物哭哭笑笑，生生死死，有时甚至连自己都已不知道结局。随着主人公欢喜，悲伤，快乐，失落。有时故事写完了，我却感到感伤。

　　有时，真不知自己日日夜夜敲下键盘的是些什么。难道这些就是我人生的全部意义，难道这些就是我全部的梦想和希望吗？！写作真的是一件很孤独的事。没有人在你身边，没有人关心你，没有人在乎你。自己成了一个晶莹透明的肥皂泡，也许下一秒钟就会消失。想找一个人同你一起分享写完的快乐也没有。一个人在室内来回的踱步，你甚至想让全世界听见你的呼喊"我写完了！我写完了！"可是，你却忘了，根本没有人在乎你。你烦躁不安，你想得到评价，如同女人穿上新衣不能照镜子一般痛苦。你想知道它的好坏，你需要得到肯定。有时，自己是不能肯定自己的，不是吗？写作真的是一件很寂寞很痛苦的事，可是它辛酸的过程和甜蜜的结尾我都一一深爱.............
