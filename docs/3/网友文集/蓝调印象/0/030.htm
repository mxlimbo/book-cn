蓝色的思念


　　悄悄的,悄悄的涂满天空,一如我对你，悄悄的思念。淡淡的抹上了几丝忧郁的色彩,为那个已经知道却不承认的答案。

　　整个天空都是我的,我在上面涂满了你的名字。

　　许多许多晴朗的日子里,我如期而至,固执的等待。

　　原因太多,我只知道一个,是你厌倦了吧……

　　这已足够。

　　不想让你知道,我的点点思念任它蚀心腐骨,你的影像越来越模糊,却更加难以忘记。

　　我想，我一生也飞不到那晴朗的天空里与你相逢吧！

　　但我会向往。

　　如若早早知道，我仍会选择你，仍会长长的思念。
