ONE 的回忆

　　99年为你的ONE格外温暖着,你还记得这一切吗?我会在这里等着你,在心里最隐然的角落里藏着对你的小小爱恋….

　　那是你在情人节送我的香水,在满街弥漫着玫瑰的时候,它显得格外乖巧的讨人喜欢,惊喜的我问你如何知道我用的是这种香水.你略为得意的说:"到柜台前一种一种闻的啊,觉得这种和你身上的味道最象,你用的是这种吧,喂,你?......."我早已深深的埋在你的怀里,小声的一遍一遍的重复着"谢谢,谢谢......"

　　谢谢你没有同那些满街送玫瑰的情人一样,一束玫瑰简单的打发了这个珍贵的节日,谢谢你肯在香水的柜台前为我一个一个的挑选,珍贵的不是礼物而是你做这些事情的心啊,我竟然成为了一个如此容易满足的小女人,在爱里的人是比较容易满足的,不是吗?包括现在已经离开我的你,我依然深深感激.那淡蓝色的瓶子依旧深藏在我心的深处,散发着淡淡馨香.亲爱的你现在身在何方呢?下一个情人节你将和谁一起度过?99年为你的ONE格外温暖着,你还记得这一切吗?我会在这里等着你,在心里最隐然的角落里藏着对你的小小爱恋,温柔的为你淡淡微笑,即使你走远,走的我看不见,即使你没有在意,在意我对你的微笑,即使你不知道,不知道我为你做下的这一切,我依然感谢你,谢谢你陪伴我度过青春最美好的时光,谢谢你给我这美好的遐想,谢谢你如此温柔的对待我,甚至温柔的离开我........那对你的微笑,在尘世淡淡依存,它是我心里没有季节的莲花,如同ONE一般芳香四溢.....
