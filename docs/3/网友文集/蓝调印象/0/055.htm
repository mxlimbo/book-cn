我要告诉那个不会笑我的人


　　记得很清楚，是在经历了一件很糗的事后，别人问起，我用玩笑的口吻来讲述，别人都呵呵笑，我也自嘲着自己，只有自嘲才是解围最好的办法吧，可是当他听说的时候，他没有笑，他说他很后悔，后悔当时不能在身边陪着我，当时的我一定窘迫极了。他愿意陪我一起承担那份难堪，他愿意同我一起来分担，再或者如果他陪着我事情就不会那样了！

　　春游前去学校的车上，和一个上届男孩坐在一起，他絮絮的问我带了什么东西，有没有忘带的。我不耐烦的背了一遍假想中的清单，连身份证笔记本都背出来了他还有什么不满意呢？他小声提醒我："你忘了带口香糖，住在旅馆里不方便刷牙的时候你可以吃口香糖。"我呆呆的看着他，其实我包里装着，我只是想快些结束和他的对话，而他却站在我的位置为我想到了。

　　如果你的家人爱人说你遇到什么事情总不和他们商量，那一定是因为遇到问题时，他们总是先给出指责和责怪，而不是肩膀和双手来为你分担。生病时我不愿意告诉家人，宁可自己去医院打针，因为我怕他们责怪我，我有好好照顾自己，我也不想生病，这时我最需要的是安静和照顾，所以我宁可选择一个人。如果让我选择陪伴，我要那个真正关心我而不是用关心杀死我的人。

　　有些话讲出来，身边的人会笑你，而有个人会认真倾听你的话，他认为你能做到。做一些事情，不可以什么都告诉周围的人，只有真正爱你的那个人，才不会笑你。

　　当我再次伤心难过的时候，我真想时光倒流回到那个十七岁的夏天，有一个男孩永远对我张开大大的手臂，他不会笑我，他会和我一起来承担。
