尴尬人生

　　好象在我们生活中总有很多很多尴尬的事情发生......

　　在宿舍里过生日时所有人给我都点歌,感动的豪言壮语发誓说一定要学会唱,在大家一片沉默中广播里放出一首英文歌和一首日文歌......

　　住校早晨来不及吃饭咬着苹果去上课,学生会长和学生科长守在门口亲切的对我说:"这位女同学你还没有吃饱啊?".....

　　刚学会骑车拐弯时插在一对情侣中摔到,女孩亲切微笑:"我男朋友没那么漂亮吧?"

　　买了一袋巧克力准备独享,路遇邻家小孩,欲从中取出一块,邻居笑言:"你别给他一袋子啊,把他宠坏了......".....

　　开会偷看小说,历经2小时45分钟后看完,邻座做手势要求传递,被发言人领导看见,误以为在作笔记,一边表扬一边要求讲讲体会,尴尬万分站起脑子只有小说情节回荡.....

　　深夜打稿晚起不及洗刷,直去门口便利商店买面包,路遇旧情人携盛装新女友,欲低头混过,被人跟上,看着凌乱头发苍白色脸偏偏追问过的好不好.......

　　早晨去快餐店吃饭带免费粥走被告之要交两毛钱,无奈穿的运动装没带钱,理直气壮的问姐姐有没有两毛钱,却当场被服务员笑成一片(笑话我没两毛钱),虽然平生最恨人炫耀名牌,但是也好歹一身阿迪达司运动装纽巴轮的休闲鞋,他们怎么会认为我是来混免费粥喝的呢?!....

　　从小学就被老师怀疑作文不是自己写的,百般辩解到一味沉默,上班以后宣传科连连电话追问是否是始发稿,虽理直气壮但回想小时侯好象没有进步,怎么总被人怀疑呢~~~~~~~


　　唉........人生尴尬人生尴尬啊~~~~~
