间 之 稧

　　从温暖的客厅回来，自己的房间冰凉冰凉，我夹着《中国旅游》掂着脚尖走在木地板上，DISCMAN放在地板上用脚就可以按开，一边开台灯，一边点香薰，想想又回厨房去热了一杯烫烫的牛奶。回屋里时香橙的味道已经可以淡淡的闻到，音乐肆意流淌着，是莫文蔚的十二楼，很喜欢她唱慢歌的声音。

　　打开OE有朋友的信笺，他说："....原始海洋是地球收缩冷却时400年的大雨形成的。其实天气也不重要，重要的是境遇控制人的心情....。"一个人笑了好久....，他是这样解释阴天的，因为我总是阴天时心情不好。

　　蜡烛的火苗总是在晃，歪头看看，哦，原来窗户没有关紧，漏出一道窄窄的缝。说是密封的窗户，可是每天我关上了，晚上总会漏出一道窄窄的缝，有时打字手很冷，总觉得有风。

　　再密封性好的东西，也会有缝隙是么？

　　再严密的东西也会有道间隙是么？
　　
　　你的心可有缝隙？象那封闭的很好的塑钢窗户总会漏出一道窄窄的逢。那里在宣泄着什么遗漏着什么呢？象冷风一样不易察觉的情感么？或是有风时才感觉到？

　　这些年不就是在捕捉着那些缝隙么？一点点的细微的感受，心情，尘世的美......成为可触动的冷风，再如同那晃动的烛苗一样燃烧，散发出淡淡馨香....
　　
　　我只想将过去深深埋葬，郁郁藏香。

　　那些缝隙让我想起生命中的某段时间，有时感觉时间什么都没有给我们留下，只有那些小小的缝隙，象是时光在心上的划痕。有冷风吹过的时候，刺痛呼啸而过。象情感的某个场景，心灵的某次创伤，忘了了，间隙仍在。

　　朋友说:"如果我能选择遗忘，我倒真是希望能忘记一切。好让所有关于音乐，关于文学，关于爱情的幸福再完完全全地重来一次... " 我也想啊，可是那些缝隙呢？我已经快把过去忘掉了吧，只故意记得那些愉快的，快乐的，可是悲伤已与喜悦相染，希望已同回忆相间，我无法将它们一一割舍........

　　这世上没有一件是完美的，比如一颗看似密封的心，那上面早已布满了小小的缝隙。
