　　当即,女孩手里的冰激凌呈90度直角栽到了地上,嘴巴足足张了有30秒钟没有合上.她小心翼翼的劝氧气,这个名字是否不雅?是否不太符合她说的什么什么什妙不可言动听悦耳的...........可氧气主意已定,可怜的吉娃娃呀.......

　　不过可怜的氧气只养了三天半的宝贝大头便被人把狗偷走了,她为此把眼睛都哭红了,发誓说就此再不养狗,这还没有忘了告诉女孩她回家以后把狗的名字改了,女孩欣慰,忙问改成什么名了?氧气垂头丧气的说:改为叫毛头.......换汤不换药嘛~~~~~~真是的,女孩又忽然想起来什么说毛头这个名字好熟啊,怎么总好象在哪里见过的,啊!我知道了,是一个网友的网名呀!!!氧气一副我爱叫就叫我怕谁的表情斜视着女孩说:那怎么了?谁规定我家狗不许叫这个名字了?! 没怎么没怎么.....谁敢说有什么!女孩微笑,嘿嘿嘿嘿~

　　写到这里象燕七的氧气似乎写完了,但总有点意犹未尽的感觉,就把我准备在主页上介绍新音乐写的有关我和氧气的一点贴上来了:

　　飞跃小提琴------------（以小提琴，吉他，曼陀铃，电吉他，加以敲击，演奏出段段优美乐章，乐器配合及运用之精巧简直神乎其技，百听不厌。）刚开始喜欢听NEW AGE时的风格是清丽略带哀婉的调子，氧气借我的CD听来听去，最后实在忍无可忍说你怎么老听这些让人听了心里难受的曲子！我则说无法忍受她那种吵吵闹闹的风格,她见说不过我就非要自己去买一张，说实在无法容忍我的风格了~然后，我们就到天籁去觅CD了 ：）她和那个爱微笑的小姐说："我想要一张欢快的，但又不是很吵的，要有节奏感的，曲子要悦耳的,百听不厌的…………….."小姐微笑着听完,就一秒钟也不犹豫的把《飞跃小提琴》拿出来了，一放到CD机里那昂劲的开头把我们强烈的吸引住了，说天呐，天呐，这是小提琴吗？这是小提琴吗？！出店门的时候我们一人买了一张。

　　长润公园-------------这张不折不扣的是赖的死党氧气的，她买了以后我一听倾情，里面的音乐非常轻快悦耳，最喜欢里面有一首曲子有乐器模仿的轻轻的鼾声，一听到它,我总能想起迪斯尼里造型可爱的狮子在呼呼大睡的样子~我让她卖给我，她偏不肯。我再去买却怎么也买不到了，我知道她是想故意逗我开口求她~ 哼,我就不开口,我借借借借她个天老地荒,我---不--------还--------了~~~~~~ :D
