谁说要非爱他不可？

　　一天和朋友聊天，他们说有一首黄品源的歌，中间有莫文蔚大段大段的R&B，非常有味道，当下和朋友说好借她的唱片。
　　
　　一个人去音像店，本来想寻觅辛晓琪的老歌《深情难了》却意外的发现了一张莫文蔚的精选的CD，很多很多的慢歌，有一些没听过，本来是想自己刻录一张的，这下看到喜出望外。买回家去一首一首听，听到最后竟是一个男声先幽怨的唱起来："离开你 是傻是对是错 是看破 是软弱 这结果是爱是恨 ....."然后莫文蔚淡淡的说唱："从女性的观点让我明白的说 无论你是挖心掏肺 呼天抢地或是热情如火 不过白白惹人讨厌 让你嫌你罗嗦 恨不得没跟你认识过......"一个人听的在深夜不禁笑出来，好一个没心没肺的小女子！朋友说的也许就是这个吧？听听男声，果然是黄品源的嗓音。去翻翻歌词，还是是李宗盛写的，想想又笑，除了他谁还能写的出来？真想为他拍手喝彩，写的真是精彩！

　　那段："你讲也讲不听 听又听不懂 懂也不会做 你做又做不好...."唱到这里莫文蔚轻轻的呵一声，带着点点的不耐，听着不禁会意一笑。很好的网友COCO时常向我诉苦，说下届有个小师弟喜欢她，无论风天雨天都在女生宿舍痴痴等待。其他宿舍的女生唏嘘的不得了，劝她和他出去。COCO苦恼，本来并不讨厌他，可是也绝对谈不上喜欢他啊....我告诉她万万不可和师弟出去，如果不爱他就不要给他任何希望，COCO喃喃："可别人都说我好残忍......"

　　也许该把这首歌给COCO听听------凡是中国的女孩都好象陷入这样一个境地：等着人人来追，等着人来爱，等着别人对自己好，等着时间过去，等着自己感动，坐等爱情发生.......等啊，等啊等下去，等到的真不知道对他的是感动+时间=爱情，还是自己真正爱上的。如果哪个女子胆敢对自己那么好的人不喜欢，那她真是不知好歹！残忍！ 冷酷！........ 可是，有没有人问过她到底爱的是不是那个人？就算他把全世界都放在盘中呈现给她，可是不喜欢还是不喜欢啊！你强求我的喜欢，难道你就不残忍么？
　　
　　仔细想想，如果这个人客观的摆在你面前，你到底喜欢他几分？

　　他是否是你真正喜欢的那个人？

　　难道为了不伤你的心，我就要对自己狠心？狠心放弃我心目中的爱？

　　谁规定的，你对我好，我就要答应你，无论爱不爱只为了你对我的好，对我的这份真心？

　　爱不是这个样子的吧？

　　DISMAN里莫文蔚还在唱：".....最终日子还得往下过 你可以说我冷漠 或是怪我刻薄 我倒想等着看你没我到底能不能活....."说得好，谁说要非爱他不可？
