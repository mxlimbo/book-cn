年轻时的爱

　　她很年轻，刚刚二十岁，漂亮，圈子里的人称她为艺术家，从小学画的，获过奖当然没的说，小小年纪就生活在颂扬之中，但是她一直不满意自己，不知为什么。

　　他们那一圈的都是玩艺术的朋友，年纪比她大，一般都是宠着她让着她的，年轻嘛，漂亮嘛，当然得宠。一次吃饭时候一个歌手却狠狠的说了她.....那时大家泡在酒吧里，酒吧也是自己人开的，墙上挂着她的新画，每个人都说好，当然喽，她临摹的功夫是一流的，微妙微肖。那个歌手却说：这些画是个人都能画出来...她愤怒极了："你是人，你画的出来么？！"歌手说：你只得其形未得其神，临摹谁都可以做到，只要舍得花时间，这样画下去只是个画匠！一个艺术家要有自己的特点要会创造要........"她根本没有听完，一怒之下跑出酒吧，一路跑着，委屈极了，因为从来没有人这样说过她，而画匠这个词对她又是多么的敏感。朋友这边有追出来的，酒吧主人在那边打圆场说歌手不该这么说她。朋友追来说别哭了别哭了那个人就那样的臭脾气.....她一摸自己的脸原来全都是泪，朋友说其实你画的挺好的嘛，隔行如同隔山他懂什么...。就是，他懂什么，她越想越气，我早晚要证明给你看！

　　那一年几个画画的朋友去了国外，她也跟着走了，父母不愿她去，说国内都画不好出国家就画的好了？她却是憋着一口气去的，她要证明给他看她不会是个画匠。

　　去了荷里活，又辗转去了旧金山，最后到纽约。到了纽约已经是出国的第6个年头了，在纽约的她已经不在画画，做电脑，又做了2年，终于回国。国内的电脑市场大，她做生意做的很好，旧时的朋友来找她吃饭，说那么久了回来怎么不联络啊，她说太忙了联络不上啊，相约的是去那个作为活动据点的的老酒吧。
　　
　　酒吧几乎保持了老样子，一贯的风格，老板还是他们的朋友。一圈的人散了几个，大部分也还在，有的转行了，有的还是老样子。意外的是，她发现了8年前她画的画...... 还是挂在老地方，没有变。一圈人嘈杂的说着话，谈笑着，酒吧里热热闹闹的放着老歌，大家似乎忽略了今天的主角。她站在那里好似被8年前的自己给击中了，那时的自己多锋芒必露，多盛气凌人，那么幼稚......

　　她闲闲的问起那个歌手，大家都说他现在可发了，她楞，说怎么发了？酒吧的老板说他早已经不做歌手了，在幕后做制作人，他写的歌唱片公司都抢着要呢~ 她心里却暗暗的想多可惜呀，记得当年他唱歌唱很好的，他爱穿白衣服，晚上人少的时候就坐到酒吧的钢琴前闭着眼睛自弹自唱，有时唱自己写的歌，大家都放低了声音听他弹听他唱，连星星都会沉醉的歌........也许碰不上他是好事呢~ 在荷里活挣扎了4年，最后一个很有名的老画家看她的画时说：孩子，你画的是很好，但是没有创意，没有新东西，一味的临摹下去一辈子只能是个画匠。现在做点别的还来得及.....她没有愤怒也没有委屈，几年连打带爬的国外生活让她明白了不少，她的恐惧她的迷惑她的包袱都解开了.....幼时的一件件往事涌上心头，那少年班的老师看她照着黑板上画的那么好夸奖她了，周围邻居亲戚老师的赞扬了，她的迷迷糊糊，雄心壮志.....她能用最好的技巧画出一幅画，而画里没有灵魂，电脑打印机可以比她做的更好更完美.....她很快改行做了电脑。

　　现在，如果现在他再对她说那番话，她一定能心平气和的接受，一定会客观的接受了.....。她的目光又落在了那幅画上，酒吧老板顺着目光看过去笑，说还是他让保留着的呢~ 老板继续笑着说："从你一气之下跑出去那天，他就请我一直挂着，经常站着看不说话，画再也没有再摘下来过......."他让挂的？！她心里嗡的一下，那暖暖的酸酸的，带着几分苦涩的味道涨满了心扉。是他么？ 是他么？ 当他闭着眼睛弹琴唱歌的时候，她总是痴痴的望着他，而他弹完睁开眼睛的时候总会第一个看向她，穿过长长的钢琴幽暗的酒吧，深深的，深深的凝视她。如果不是喜欢他又怎么会那样在乎他说的话呢？

　　她掩饰的笑笑，低头喝咖啡，眼泪打落在那香浓的卡其诺里，年轻时候的爱，为什么那么傻......
