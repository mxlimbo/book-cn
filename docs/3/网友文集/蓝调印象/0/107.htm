班车爱人---给这个城市熟悉的陌生人

　　他们天天一起坐车，坐早班车。每天7：20分奔向这辆6340班车，周一到周五，从不停歇。不过天天如此的不只他们两个，还有将近五十个人。可是不知道从哪天她发现了他，或是说他发现了她，更或许他或她都在更早更早以前发现了彼此。她喜欢他穿那件玄色T恤淡色牛仔，在衬衣西裤的男士中他象一个淡色的青影。这正是他的风格，淡淡的，不经意的距离。他更换的T恤都是淡色的，有一种蓝，不是女孩那种娇嫩的粉蓝，而是秋天10月天空的蓝，不耀眼，一种沉稳的儒雅，她爱极这种蓝，并且发现这种蓝只有穿在他身上才合适，这是属于他的蓝，象他这个人。

　　每天一起坐车，走向车站时都在默默的寻觅着这道影子，淡青色的影子。他总是默默的站在一个地方，这个车站没有他认识的人，她从未见过他与别人说过话，他的脸上也总是一副淡然和冷漠，这是所有坐车人的神情。有一个朋友和她开过玩笑说，为什么现在所有的漂亮女孩在公共场合里总是一副冷漠的样子，而且好象是越漂亮的越冷漠，要把人拒之千里之外。她笑着说这也是种防备啊！"那你呢？"朋友狡猾的问，"你为什么总笑着，总是带着淡淡的微笑？""因为，"她摆出一副小心翼翼的样子说："我并不漂亮啊！"朋友大笑。这是她的理由吗？不完全是吧。在这个车站如果她的朋友看见她是不会相信的，因为在这个车站她从来不笑。在路上无论怎样愉快的走向车站，到达后她就会变的漠然，因为，面对着五十张同样漠然的脸。

　　这是现代人的特点，无论你走向哪一辆班车，哪一辆地铁，哪一辆火车，在陌生的人群中，视线接触的都是一张张漠然的脸庞。无论你和他多么熟悉，在他还没认识你时，那脸庞上的漠然就是最好的防备的武器。防备什么呢？人类的本能，你如果是个坏人呢？如果你是想加害他呢？如果你是想利用他呢？如果你会给他带来麻烦呢？所以漠然是最好的表情。我不想认识你，我需要安静，所以你也别来跟我说话。这些是这个面具下的吗？她也不完全明白，只是有时感到窒息。她在别的地方看见过他们，这些一起和她坐了两年车的人。在宾馆，商店，书店，在路上，甚至在菜场，他们同他们的丈夫妻子朋友同事在一起，议论着，说着，带着微笑，或者愤怒。表情丰富极了，可是他们回到车站上，每个人都成了一些没有表情的面具，彬彬有礼。

　　在她的目光滑向人群时，她看见了他，同样淡漠的人，但是他的眼睛里还有一些没有湮灭的东西，她相信她看到了，压抑在这个面具下的热情和生命原本的张扬。那么他和她是一类人么？她小心注视着他，在视线接触的一刹那间，他们又若无其事的把目光移开。她想让自己相信他的目光是在看着她的，哪怕只有一秒钟，是为她的。但是她又怀疑，他是那么的漠然，即使那天天准时的6340在7：25分还没有发出，人群开始骚动时，他还是站在那里，若无其事的淡然着。他真象凡尔纳笔下的斐利亚.福克，那个八十天环游地球的人，沉稳冷静。也不完全象，起码福克不会嚼口香糖。她想想就偷偷的笑，那天她在车上看见他在嚼口香糖，这让她很开心，因为她也在吃。早上起的太晚了，这全怪她凌晨才睡，只是为了看完那本书。为了掩饰一下没有洗刷的痕迹她只好嚼口香糖了。那么他呢？好象很少有早晨就吃口香糖的人吧，她当时心里有一种冲动很想去问问他，当然这是不实际的。他在车这头嚼，她在车那头嚼，她很为他们有这点默契而高兴。但是他为什么看上去这么淡然，这么都无所谓.........

　　一天天就这么过去了，每天做车上班下班，走那么几条马路，乘那一辆班车，面对不变的那些面孔，好象今天和明天已经没有了什么区别。可是她渴望每天能看见那一张面庞，并期望有一天它能发生些许的变化。她不知道怎么表达，她也根本不会去表达，这是女孩的矜持和一种防备。如果他只是和其他人一样呢？一样的漠然和惯性的过这一天一天的生活......

　　有一段日子她被借调了，好几个星期没有从这里坐车。她原本以为自己会很想他，可是她却 没有，依然平静的踏着轨迹过她的生活。有时她很奇怪为什么从没在别的场合见过他。后来想这个城市这么大，她并不知道他住在哪里，叫什么名字，干什么工作，甚至不知道他有没有结过婚，他对她来说其实也只不过是个--熟悉的陌生人。再回去坐车时他见到她，也没有流露出什么，还是那样淡淡的。她甚至痛恨起他的淡然的神色！可是令她恐惑的是她回去明明知道没有什么变化，可是看到那些漠然而熟悉的人群，她有一种异样的感觉，她回来了，回到这些人群中......这是怎么了？她不喜欢他，或是喜欢？她觉得自己好象被这个城市快吞噬了。晚上，她在台灯下仔仔细细的看自己的脸，她算不上一个容貌出众的女孩子，但是眉清目秀，奇怪的是她长的不象一般的汉族人，整个人有一种奇特的气质。现在她的脸也蒙上了一层黯淡的光么？那种无所谓的，平凡庸常的，现实的.....她把镜子啪的倒扣过去，关上台灯。整个人坐在黑暗里，何时看过的诗也就慢慢的一点一点随着黑夜涌了上来。

　　初夏的早晨和他们开了个玩笑，当他和她慢慢的走向车站时，6340早发了，他们一同气喘吁吁的跑上车。人很多，只能堵在门口了，他和她挨的很近的站着。她闻到自己涂的浅蓝色ONE的清香，过热的空气让它的香味开始散发。那么他也闻到了么？也许我该换一种再清淡一点的香水。他离她站的那么近，她甚至能感到他身上散发的热气。可是他，没有回头。

　　下班后的黄昏竟然下起了雨，她接受了一个顺路同学好意的伞，是她多年的同学，就要结婚了。祝福后她积极的陪他讨论起结婚的事项，在这时他们后面超过了一对人，是他，她看到那件熟悉的蓝色T恤。而他旁边的那个女孩，正挎着他的胳膊亲热的说着什么，他们又慢慢的消失在雨丝中......"嗨嗨！我问你呢.....""什么？"她茫然的看看身边的同学，心里象渗进了这个季节冰冷的雨水。"我问你打算什么时候结婚啊，好请我们吃喜糖。""我？"她轻轻的笑了，平静的回答到："我自己也不知道。"

　　她不会知道，他走在他们后面时听到她正在和那个人说话，他第一次听到她的声音是那样的清柔，却说出这样的话语："我认为蜜月去苏杭比较好......""张家界和九寨沟不好吗？""随你啊，反正是都你做主！......"他们在讨论去哪里度蜜月！！！他默默的走过，和他的妹妹。他没有听到妹妹在他耳边喋喋的说着什么，只是很想再回头，看看那个长着一副异族人面孔的清秀女孩，找了一个怎样的男友。难道他不如她那个身边的他么？他还要再这样漠然下去么？他多么想把他热烈的眼神投向她。看着她美丽却淡然的脸，他经常猜想她是一个怎样的女孩呢？他见过她在路上歪头看着嬉戏的麻雀微笑，温柔而灿烂，让他为之心动。走向车站的时候她却已没有了表情，一样的淡漠。他多么恨人们的这份漠然！他宁肯希望大家见面说说笑笑，打打闹闹的枪座。他想和见到的每一个人微笑大声的打招呼，尤其，是和她......当她的目光滑向人群顺便捎带上他时，那是一种怎样忧郁的眼神啊！他相信只有他能看出那是一种哀愁。他希望她的目光能在他的身上停留，他该......他根本不知道该怎样。如果在学校里他也许会冲她说一声，嗨！可是在这里吗？在这些淡漠的人群中？她会不会觉的他是个怪物？他会不会冒犯了她，她会不会摆出一副更冷漠的样子看着他，象是在看着空气？他不知道他为什么面对她有这么多的怯懦。他也许早该换个车站做车了，离他住的地方已经设了一个车站，妹妹经常问他为什么舍近求远，他今天很想告诉妹妹为什么，想指给她看，看这就是我喜欢的那女孩，你看看她是不是有一种很特别的气质？.....可是这一切已经太晚了....她就要结婚了。他以后再也不会去那个车站坐车了，他对她来说也只不过是个陌生人。如果再见到她，他一定会大大方方的对她说：嗨！并且向她祝福，哪怕是含着眼泪的祝福......在雨丝中他们相错而过，雨继续在这个城市沥沥的下着，象是笼罩了一切。
