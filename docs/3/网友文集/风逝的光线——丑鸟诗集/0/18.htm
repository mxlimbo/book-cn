落雪，或者孕育春的眼睛

1、


从另一个世界的语言里
匆匆抖落的诗句
把上帝多余的表情铺陈于地

象许多曾经丰满的日子
从矗立寒风中
苍苍的十指上
飘然坠落

2、

这些令人目眩的翻腾
从早晨，到日暮
直到两鬓斑白的冬
的深处

象许多难于在掌心安眠
的尾音一样
不可预见，也无法影响

3、

雪落下来
让树的颈圈光亮起来
套着匆匆而过
的岁月
套着过期的心情

然后
一闪而过

4、

请留下我
留下悬挂在树梢的歌声

让我用温热的心脏
催促冬麦的生长

在一片纯洁的土地上
孕育春的眼睛


2000-11-28
