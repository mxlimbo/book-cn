三月殇﹒ 怀海子（二）

所有的三月都生长黑色的翅膀
所有的春天都在今夜静默
嘘 我不说

一粒麦子在你的血管里游动
摄取清澈的希望
或者守侯生命的绝唱

你只用一只眼睛来烧灼黑暗
另一只留给自己和未来
代替月亮的镰
收割闪亮的标题和
跳跃的诗行

你从不忘记草原
以及第一株青草
从不忘记北方那一夜的屋顶
和春天里盘旋的鹰

于是你不停地追赶太阳
或者在悲壮的黄昏里
拥抱死亡

在生命的调色板上
只有白色无情地
碾过三月的村庄

而对于在三月之后启程的后来者
只能在春天的静默里
倾听你撞击生命
的回音

2000-03-26凌晨
