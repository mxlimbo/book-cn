攀缘生命的悬崖

献给魂归雪山的勇士们

顺着风的方向
我用灵魂眺望

在比远方更远的地方
缓缓陨落一轮太阳
一轮勇士的血凝固
的太阳

我看到凛冽的雪峰
象肃穆的白烛
无声地燃烧

我看到一扇门
看到生命之旅
的最后一个驿站

看到匆匆的旅者
以不同的姿态凋零或
绽放

我看到一串串跋涉的脚印
累积成生命
的悬崖

而每一步艰险的跨越
就象一次冰河的坼裂

只有历经坼裂的痛楚
才能从这里
提炼完美
走向生命之巅

我看到一株青草
以柔韧的臂膊
起吊生命的重量

我看到大地的面颊上
奔涌不息的泪水
为灵魂的再生
恣意流淌


2000-05-23
