第一百三十七回

　　秦汪冉若无其事的续问道，"贾兄，你看看，你看看，这多不好意思。咱们何苦如此这般搞下去呢？弄得血乎拉叽的，大家面子上都不好看。还是借一步说话，您看如何？"

　　秦汪冉这一刀出手，三人已经明白，己方三人不要说是跟面前这魔鬼般的陌生人拼命，就是想逃命都大成问题。那商贾模样之人努力使自己的语声平静道，"尊驾下手如此狠法，不留半点余地，小老儿忘性大，实不知是什么时候得罪的尊驾，结下的梁子？要是确是小老儿以前瞎了眼，不小心得罪了尊驾，小老儿告罪先了，就不知道还有没有转圜的余地？"

　　秦汪冉摇头道，"第一条，我跟三位是初见，以前绝没有什么梁子，这一条上贾兄只管放心，第二条，我现在下的手绝不算狠毒，这位贤兄不管怎么说，好歹还站着是不？我找三位，只不过想问一件事，只要贾兄给在下一个满意的答复，在下立刻便放过三位。现在，贾兄，咱们借一步说话如何？"

　　到了这地步，形势比人强，不答应又能如何？那锦袍大汉扯了块衣角草草裹了断腕，三人在前，秦汪冉押后，四人走到了道旁林中无人处，阳光从树木枝叶缝间透了进来，照得各人脸上一道道的明暗不定。待到三人站定，秦汪冉并不立刻开口发问，只是背着手缓缓的围了三人转了一圈，那意思倒好象一只恶狼围着羊群团团打转相似，这时不光那锦袍大汉满头大汗，连那商贾模样之人和那书生也自汗湿重襟。

　　秦汪冉转到了那商贾模样之人面前，盯了他眼睛轻声道，"在下在茶馆中听得三位畅谈甚欢，好象是有一笔大买卖要做，可巧在下也是个生意人，听到有这样好的生意可做，便不由得跟着动了心。只可惜在下的耳朵不是太灵，许多事情听得不是特别明白，只好请了三位到这里，想烦三位给在下讲个明白清楚。就不知三位肯不肯成全在下这个小小的不情之请？"

　　三人这时才明白祸从口出，自己几个人一时扯得高兴，却无意间为自己惹下了天大的麻烦。

　　那商贾模样之人苦笑着道，"尊驾有什么要问的还就请讲，小老儿但凡是知道一二的，定然不敢隐瞒。"

　　秦汪冉笑道，"在下这里先多谢了。在下方才听三位讲到，今晚上三位有些朋友想造访晚晴山庄，不知是怎么一回事啊？"

　　那商贾模样之人沉默片刻，才长叹道，"要小老儿讲了这件事出来，不是不可以，不过尊驾得先给小老儿一句话，否则，小老儿干脆现在就交了这条老命给尊驾得了。"

　　秦汪冉立时会意，笑道，"这件事我是打墙缝里面听来的，肯定跟三位挂不上干系。贾兄还请放一百个心在肚子里头。"

　　那商贾模样之人道，"不是小老儿信不过尊驾，但这事关系到小老儿兄弟三条命，尊驾还请赌个誓最好。"

　　秦汪冉道，"这样也好。在下要是事后伤了三位一根小指头，或要是哪一日告诉第五个人我是从三位这里知道消息的，天打雷劈而死。"

　　那商贾模样之人道，"离土三尺有神灵，尊驾可别拿这等事寻开心。"

　　秦汪冉正色道，"我自然醒得——我犯不着自己给自己找麻烦是不是？现在我咒也赌了，贾兄，该您开口了。"

　　那商贾模样之人咬了咬牙，道，"咱们哥三个，是接了郭老大的帖子，来扬州是要动一动晚晴山庄这座金山。"

　　秦汪冉点头道，"这个在下倒也是猜到了三分，就不知郭老大是哪一路的神仙，你们要动晚晴山庄，又想要怎么个动法？"

　　那商贾模样之人道，"尊驾想来也听说过'酒色财气四大寇，遮天盖日乱关西'？"

　　秦汪冉笑道，"听说过，听说过，如雷灌耳得很啊。"

　　那商贾模样之人道，"郭老大便是酒色财气四大寇里面的老大，大名唤作郭睐任的就是。这一回，酒色财气四人均至，却是要遮扬州的天，盖晚晴山庄的日。"

　　秦汪冉暗自心中点头，知道自己刚才猜得不错，正是这四个人王来了。

　　那商贾模样之人续道，"这一遭，来的其他高人还有十二燕子连环坞中的雨燕坞坞主七星映月赵冷哲，云燕坞坞主托塔手林孝义，白衣长发会白山黑水双杀，孙祥，孙和兄弟，冀北大豪九节锁魂周云。"

　　秦汪冉微微惊异，前面四人的名字他在茶馆时便已经知道，至于这九节锁魂周云，在茶馆中三人没有提起，秦汪冉却是现在才得以知晓。九节锁魂周云的名头，秦汪冉以前倒也听说过一二，周云使的是一条九节钢鞭，以是上得了个九节锁魂的名号，在冀北地面上，乃是头一名独行大盗。

　　秦汪冉颌首道，"颇是有些大人物啊。只不知这许多高人动晚晴山庄，有没有个计划什么的？"

　　那商贾模样之人涩涩的道，"计划确是有一个。无非是里应外和，设伏狙杀之类的。"

　　秦汪冉笑着问道，"贾兄能不能说得稍微详细些？"

　　那商贾模样之人长叹一声，艰声道，"详情小老儿知道的也不多——不是我敢唬弄尊驾，小老儿兄弟三人，到底不是这场戏的主角——小老儿只是晓得，晚晴山庄的护院里头，已经有一个是我们的人，今儿个傍晚时候，他自会在水井中下药，先自放倒一批晚晴山庄看家的狗腿子。另外我们这一边约来的第一流好手，加起来共有九人，而郭老大已经打探清楚，晚晴山庄现下，除了庄主独孤鸿外，硬把子共是五个人，连他们的起居习惯，郭老大现下都也已经摸得一清二楚。动手时段四爷自己伺候其中一个，至于剩下的四个硬手，我们这边两个高手对付一个，便定可狙杀这五个硬把子于迅雷不及掩耳之间。这以后大夥儿再合力生擒独孤鸿兄妹，逼问晚晴山庄的财富所在。"

　　秦汪冉笑眯眯的问道，"算盘珠子打的噼啪响啊！晚晴山庄看起来的确是在劫难逃了。至于这九位高人现在何处，贾兄想来也是晓得的？"

　　那商贾模样之人道，"郭老大他们兄弟四人是落脚在城东的香影廊，九节锁魂周云是住在瘦西湖边上的巫山楼。至于孙家兄弟和燕子连环坞的人住在哪里，小老儿是真的不知道了。"

　　秦汪冉拍了拍那商贾模样之人的肩头，和声道，"贾兄，此话当真？好朋友面前可不兴作假。"

　　那商贾模样之人被拍得全身一颤，哀声道，"当真！当真！——小老儿肚子里面知道的那一点东西尽都吐给了尊驾，小老儿是真的再也不知道别的什么了。"

　　秦汪冉微笑问道，"贾兄不妨赶紧仔细再想想，就没有忘了什么细枝末节的？现在说出来还不算迟。千万别以后再想起来，那时若要后悔，可嫌晚了些啊。"语气中威胁之意，浓得简直要让三人窒息。

　　那商贾模样之人只觉得唾沫都是苦的，哀声道，"小老儿不敢有半句虚言，确实已经是知无不言，言无不尽了。尊驾还请依了前诺，高抬贵手，放我们兄弟三人这一马。"

　　秦汪冉知道对方再也不敢藏私，笑道，"那是自然，不过我现在不能就放你们走路，好在一晚上的时候也不甚长，就委屈三位在这里一宿了。"那三人还没有反应过来秦汪冉话中含义，秦汪冉已经出指如风，在三人腰间章门穴上各点了一指，三人眼睁睁的瞪着秦汪冉，一一软倒。那商贾模样之人大叫道，"你，你，你想干什么？"
