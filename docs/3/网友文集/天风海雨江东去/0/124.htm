第一百二十四回

　　西窗剪烛下，把酒语沧桑

　　岳飞断然摇头道，"这可决计不成。我平时没时候管教云儿，他现下已经是够无法无天的了。汪冉，你可千万不能再助长他的气焰。"

　　秦汪冉微微一笑，道，"少年人若是不无法无天，那反倒奇了。相比之下，我看云儿纯朴率真，其实还算是老实的。"

　　岳云闻言，胸中不禁大生生我身者岳鹏举，知我心者秦汪冉之感，挺胸抬头，拿眼角偷偷撇向岳飞，看他有何反应。

　　岳飞一扬眉，却道，"云儿到底老实还是不老实，归根结底还是我说了算。知子莫如父，汪冉，这一条上，你就不必再与我争辩了。"这一句话，倒是把作老子的威风抖得十足。

　　秦汪冉轻笑一声，不再说话。

　　李铁枪看到岳云失望神色，赶紧道，"云儿，别发愁。明儿个我传你六招精妙无匹，所向无敌的锤法，用心练好了以后只管打你老爹的脑袋出气就是了。"

　　岳云冲李铁枪一瞪眼，道，"练好了以后我先拿你试锤！"

　　李铁枪双手一摊，一副痛心疾首之状，道，"怎么又是恩将仇报！而今果然是好人难做啊。"

　　众人说笑闲谈间，夜色已被染得渐浓，一大块墨玉般的天际上，连一抹儿云都没有，但见月升中天，满天繁星点点，明灭不定。早春时候的晚风依旧是冰寒的很，外面也不知是不是帐篷的一角，被吹得'扑拉拉'的乱响。

　　酒过数巡，岳飞起身道，"各位远来辛苦，今晚还请早些休息罢。这官衙左边侧院里面有数间客房，众位可以在那里歇息落脚。"

　　岳飞手中举了一盏黄橙橙的油灯，在前头领路，带了诸人左一弯，右一绕，顺了走廊而行。走廊栏杆上多有刀剑砍削留下的深痕和斑斑点点绛紫色的血迹，想来前些时候岳飞自金人手中夺回建康之时，也曾在此有过血战。那血迹到底是金人留下的，还是宋人留下的，现在又有谁能够分得清楚？不过一会儿以后，众人便到了官衙左边侧院，出现在众人眼前的，是深深的一个天井，里面锁了几株经年梧桐，落叶遍地，静悄悄的颇是详和，四周围了一圈白墙红窗的小房间。

　　岳云取来钥匙，把房门一一开了。入眼只见都是一色的单进客房，里面陈设虽然并不豪华，但干净的很，桌椅盆架，被褥床铺什么的倒也齐全。诸人各自选了房间，便一一同岳飞父子道别。

　　蒙慎行一只脚已经进了房门，突然间想起了什么，回头问道，"鹏举，你却住在哪里？"

　　岳飞笑道，"入城时蒙帮主可看见许多门板在街上？那里面便有两张是我们父子的。"

　　蒙慎行奇道，"明明摆了这许多客房在这里，白白的空着也是浪费，你父子怎么不来这里住呢？"

　　岳飞笑道，"这里的客房虽然不少，但无论如何也是容不下四千兵丁的。所以……"岳飞笑笑，打住了话头。

　　蒙慎行道，"难怪你麾下将士用命。除了你日间所言的厚赏严罚之外，就这一条与士兵同甘苦，又有几个将军能够做到？"

　　岳飞淡淡的道，"我的手下是人，我也是人，与他们吃住一起，本来就是自然而然的事，又有什么做到做不到的。"

　　蒙慎行长叹一声，道，"鹏举，你实非常人。今后你若能得机遇，必能成就惊天动地的大事。好了，今夜已迟，明日见时咱们再聊。"

　　岳飞笑道，"蒙帮主，晚安。"

　　诸人纷纷走进房间，秦汪冉却不移动脚步，仍是独立在门外，衫角被风吹得微微抖动，身影更形单薄。待众人都已经进屋，秦汪冉斜倚在门框处，懒懒的问道，"鹏举，还有酒吗？"

　　岳飞微喟道，"那是自然。"转身对岳云道，"你先去睡，我跟你秦叔叔聊会天后就来。"

　　岳云虽然不情不愿，却也只好给秦汪冉道了晚安后便自离去。 到了这时，天井中就只剩下秦汪冉与岳飞两人。秦汪冉忧郁如远山的眼眸与岳飞明亮如晨星的目光无语相接，两人嘴角都浮了一线淡淡的笑容，那里面包含的到底是惺惺相惜，还是知音会心，也只有他们自己心中明白清楚了。岳飞向前迈了半步，将右手伸了出来，秦汪冉微微迟疑间，也伸了手出去，这两只或许是世间最有力，最坚强的右手，便紧紧握在了一起。
