第一百十三回

　　朝霞满天，染尽山麓。众人伏马狂奔，身上也被初升的太阳映得红了。明日的太阳升起时，众人是不是还能够有机会看到，这时又有谁能知道呢？

　　忽然间众人面前出现了一个三岔路口，赵得胜叫道，"就是它了！走最左面那条！"前几日阴雨连绵，小路上泥泞得很，马匹奔跑的速度立时变慢。遇到积水，众人更不得不下马牵了马匹而行。

　　这三岔路口其实倒是帮了众人一个大忙，尾追的金兵到了三岔路口，迟疑了好是一会，最后才决定分兵三路追去，离众人愈发的远了。

　　小路尽头，又是一个三岔路口。赵得胜闭了眼想了一想，才断定是走中间那一条路。这却是一条山路，弯弯曲曲的盘山绕岭，走了半个时辰才尽，出现在众人面前的是一片一望无际的平原，两条大路，一条向东，一条向南，众人沿了向南那一条路行去。倒霉的是后面金兵，一再分兵后，这时跟在众人后面的只剩下了不到一百人。带队将官再追一阵，自己觉得很是有些不妥，便只好悻悻然的回转了去。

　　众人在大路上再走了有大半个时辰，就到了一条河边。这一条河，河面宽阔的很，水却不算太深，水草丰盛，河水尽被水草染作了浅绿色。几只懒洋洋的野鸭在河中心悠闲游弋，偶尔伸嘴在水里捕捉小鱼，颇是不明白为什么岸上这些两条腿和四条腿的动物要这般匆忙行路。

　　韩世忠道，"这条河八成就是小淮河了。往前面再走约摸十数里地，想来便该是懒鸭滩。现在咱们多半是已经甩掉了追兵，不如就在这里休息片刻，好好商量一下怎样通过懒鸭滩如何？"

　　诸人身上这时几乎没有不带伤的，一夜亡命狂奔后更都疲惫异常，闻言都道，"那好，便依韩将军所言。"

　　众人各自下了马，牵了马走近河边。江南春来早，河岸边上已经是绿草茵茵，走在上面，脚底下软绵绵的就好象是地毯一般，让人恨不得就躺了在上面再不起来。天上白云数朵，地上河水缓缓而流，新柳倒垂，枝曳近水，花儿间插羞涩吐蕊，群鸭凫水，再好的丹青圣手，又怎能描画这早春美景万一？回想片刻前生死搏杀，又怎么能不让人生出人生如梦的感慨？算起来人生不过弹指间，但蜗角功名，蝇头微利，世人纷争无休，所为又何苦来由？

　　众人或坐或蹲，或者撕下衣襟包扎伤口，一时间静悄悄的没有一点声息。李铁枪更四脚朝天的躺倒在地，眼望蓝天，突然毫无原因的笑出了声来，也不知道他心中想到了些什么美事。

　　终于蒙慎行打破沉寂道，"韩将军，懒鸭滩的地形到底是什么样子？"

　　韩世忠在地上捡起一小块石子，向草地上划去，口中道，"懒鸭滩前前后后差不多有两里地，这一边是小山丘，另外一边就是这小淮河。中间更有半里地地方颇是狭窄，只能够容三匹马并行。那小山丘乃是土山，虽然不甚高，却连绵百里，颇是宽广，山上树木茂盛。如果我是金人，定然会在这半里地设伏。两头用弓箭射住阵脚，中间以骑兵冲击，再在两头稍远处派两支兵，作为第二层截杀之用。如果带兵的金将善于用兵的话，更会在去路前面设下路障，防止咱们一鼓作气的冲了过去。"

　　秦汪冉凑了过来，道，"那么，咱们就不能如方才那般硬闯过关了？"

　　韩世忠道，"硬闯总是可以的，但能不能闯过去，那就难讲的很了。怕就怕咱们的马匹被弓箭射伤，或者被路障堵住，一旦停了下来，就难免被围，那时候再要杀出，可就是极渺茫的事了。"

　　秦汪冉问道，"过这懒鸭滩，就没有别的路了？"

　　韩世忠摇头道，"除非咱们弃马，翻山而行。但这土山上根本就没有路，咱们只好披荆斩棘的自己寻出一条路来，会走到哪里，只有天知道。"惯走山路的人都晓得，七弯八绕，歧途处处的山路之中，极易迷路。而在原本没有路的山中行路，除非是当地打柴的樵夫，迷路简直就是理所当然的事。

　　蒙慎行接道，"那也总比硬闯的好些。咱们只要辨清了方向，最不济多走些冤枉路，牛头山还是早晚能到的。"

　　许大马棒这时也拢近了来，闻言插口道，"蒙帮主，若在山中迷了路，可不是多走些冤枉路就完了。我当年在长白山上迷过一次路，整整在山里面转了三日三夜，最后却又转回了原来的地方。要不是手下人找了来，当时真不知道如何收场。那时我手中还有一张简略的地图在手，犹自如此，现在咱们对这座山一无所知，嘿嘿，许某倒是宁可试一试硬闯这懒鸭滩。再说，懒鸭滩有没有金兵埋伏，也是可有可无的事，咱们说不定吉星高照，顺顺当当什么事都没有的就过了懒鸭滩也未可知。"

　　蒙七接道，"难道就没有第三个法子？"

　　秦汪冉试探问道，"咱们可不可以走水路？"
