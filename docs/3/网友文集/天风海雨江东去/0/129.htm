第一百二十九回

　　独孤鸿又笑道，"眉儿，你道是谁帮你捡回了镯子？这位秦先生可是非同小可。他便是江湖上威名赫赫，无人不知，无人不晓的天下第一高手，冥天血龙秦汪冉！"

　　秦汪冉笑道，"江湖上的朋友推爱过甚，其实秦某不过是个无拳无勇的凡夫俗子罢了。"

　　独孤鸿笑道，"秦兄，坐，坐，坐！"说着自己先坐在了独孤眉身边。

　　秦汪冉再次道谢，才坐在了矮几另一边，只觉得拘禁异常，就让他一个人去独挑山西连云十八寨，恐怕也还自在些。

　　独孤鸿又取出一只酒杯，为秦汪冉满了一杯酒，酒色碧绿，清香扑鼻，秦汪冉竟是生平所未见。

　　秦汪冉心中急速搜索，想哪一家姓独孤的有这等排场气派，却想不到有哪家高官有姓独孤的，突然一个念头扑进脑海里面，秦汪冉脱口问道，

　　"独孤兄莫非是杨柳岸的人？"
