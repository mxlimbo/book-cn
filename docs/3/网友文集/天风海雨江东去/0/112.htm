第一百十二回

　　许大马棒提气大叫道，"准备换马！"秦汪冉第一个明白过来，大声喊道，"好主意！与前面金兵马队相遇时，大夥儿准备再次抢马！"接着挥手拍了在马臀上，马速加快。这时天东一线灿烂的光芒露出，已经是破晓时分！

　　粘罕，查雄麾下近千骑，蹄声隆隆，不一时从大路远处另一端出现了，烟尘中白光闪闪，金兵都已经把兵刃掣在了手上，挥舞着冲将过来！

　　古时作战，最惊心动魄的就是两支骑兵全速冲锋相遇之时，那就好似两头红了眼睛的公牛迎头撞击一样，或许说是两群野牛相互冲击才更合适贴切些。而所谓'狭路相逢勇者胜'，再没有更好的注解给这句话了。只不过当下这一回，两边的人数相差太也悬殊，众人十余匹马，便给对面金兵马队作早点也稍嫌不足。许大马棒嘶声大叫道，"大夥儿都站了到马背上，准备跳了过去！"

　　一瞬间两下里中间地带只剩下三十余丈，二十余丈，十余丈，九丈，八丈，七丈……对面金兵盔甲上的接缝都瞧得清清楚楚了。秦汪冉一马当先，瘦瘦的身躯如钉子般钉在马背上，衣衫被烈风鼓得尽向后飘去，血龙宝刃也已经撤在手中，斜举过顶，刀锋在风中居然被激发出了尖锐的啸音！秦汪冉双目如鹰，蓄势待发！六丈，五丈，四丈……

　　待到两边中间只有不到两丈远近，秦汪冉腾身跃起，血龙宝刃化作了一团血雾，罩向了对面当头的几名金兵，紧接着兵刃相接的'叮当'声连成了一线，血肉横飞间，秦汪冉已经倒坐了在对面一匹马上，周围五六骑，都已经失了乘者。秦汪冉左臂红湿了一片，想是也中了敌人一击。

　　蒙慎行和李铁枪几乎是同时飞出，蒙慎行身在空中，左右掌凌空击出，掌力如同无形的兵刃砍了出去，两名金兵身子猛的一震，便如断线风筝般向后激飞。劈空掌练到可以五步外伤人的，天下间除了蒙慎行外，只怕再也不能找出第二个了。

　　李铁枪这一次倒颇为乖巧，腾身前先扯断了两个脚镫在手，身子只一凌空，便运了真力掷出脚镫，正中两名金兵胸前，深陷入骨，两名金兵口中鲜血狂喷，眼见不能活了。李铁枪轻轻落下，右手如猿臂般急探而回，已经抢了一杆樱枪在手。三人举手投足间杀了十余人，十余匹战马背上也就因此空了出来。许大马棒长鞭挥舞，如人使臂般卷了韩世忠几人到对面马上，众人刹那间都转了方向而行。蒙七高高跃起，落在马背时同时再发出了一把细草闲风斜雨针，打头几名金兵纷纷落马，蒙七这一次出手，更伤了几匹战马眼睛，突然间眼前一片漆黑的战马狂跳了起来，横冲直撞，后面金兵只好暂且勒马，绕了而行。

　　众人快马加鞭，向南而行，这时撤离拇麾下骑兵正好赶到。这时再无他法，只好硬了头皮迎了上去。李铁枪一枪在手，精神大振，大呼声中当先冲出，第一个冲进金兵阵中。那一杆樱枪在李铁枪手中，简直就不再是单单一杆樱枪，枪头红樱抖成了碗大的枪花，枪头却隐去不见，待到能够看见时，便定是正在从某人的心口拔出。真叫是杀出一条血路，李铁枪打头，蒙慎行在左，秦汪冉在右，蒙七了凡居中策应，许大马棒押后，鲜血绽放，惨呼连连，众人就如同一把尖刀，生生从五百金兵中间冲了过去，在铜墙铁壁般敌阵中劈开一条窄径。

　　什么叫做乱军之中？武林高手过招，讲究的是料敌机先，见招拆招，就算是在以寡敌众时候，也靠了轻功身法拉开同敌人的距离，决不会使自己处在同时四面受敌的局面。但乱军之中，四面八方都是敌人，人人几乎都是依了野兽般的本能狂砍乱磔，哪里又有什么招数可以去拆了？又怎么去拉开同敌人的距离？更何况金兵都身披盔甲，护了要害，极难一斩就伤其性命，而你若不能一招杀敌，对方就算受了伤，也照样可以继续对你攻击。再说既然是骑马，马是活的，你想让它向东一尺，马却未必如此听话，可能向东一尺三寸，或是只向东七寸，你砍向敌人的一刀便可能为此落在了空处。而一刀落空，可能就要为此付出惨重代价。千军万马中取上将首级，那只不过是神话罢了。这时的出手就只在乎三个字，一个是'快'，一个是'狠'，一个是'动'。快是要在敌人将动未动时就先出手，更要能随机应变，狠是要能够一招取敌人性命，决不给对方留一丝一毫的生机，动是决计不能让马停顿，一旦停了下来，想再要摆脱纠缠，那就难得多了。蒙慎行几人算得上是天下间最了得的几个武林高手了，饶是如此，五百铁骑也给了他们颇沉重的压力。刀光剑影中间，众人努力向前。

　　众人冲过以后，金兵伏尸不下三四十人，受伤的更不晓得有几个。李铁枪的左肋左肩皆已受伤，血染衣襟，蒙慎行的大腿上也被重重砍了一刀，许大马棒的长鞭只剩下了不到三尺长短，蒙七的肩胛上亦被后面金兵射了一箭。韩世忠虽然得诸人照应，但金兵认出了他是何人，立功心切，不畏生死而向他招呼的极多，最后还是被一枪刺在了右臂上，险些挑断了筋骨。金兵也自甚为不幸，原来堵截和追杀的马队奔得都正急，两下里撞在了一起，乱作一团，落马的有数十人，马蹄蹬踏间，更有数人枉自送了性命。待到重新整理队形完毕，韩世忠诸人已经奔出了数百丈开外了，出了金兵的视线。两边金兵并作了一处，再度追了上去。
