第一百零九回

　　赵得胜闻言瞪大了眼睛向前望去，却哪里看得到什么火光了。他却不知道李铁枪这双眼睛是多少年来在暗室里面盯香火，在悬崖上数对面树叶练出来的，就是天上的老鹰，怕也不如他望得远。

　　蒙七问道，"怎么办？"

　　蒙慎行微一沉吟，道，"正好。咱们缺少坐骑，到牛头山路程甚远，我正为这件事发愁呢。难得金兵好心，给咱们大老远的送坐骑来，怎么好意思不要。汪冉，铁枪，北儿，你三个随我来。"

　　许大马棒插口道，"蒙帮主怎么忘了许某？难道许某的功夫就这么不济事吗。"

　　蒙慎行一笑，道，"那就也偏劳许大堂主了。"

　　韩世忠等余下几人，便先在路边树丛中藏了起来，蒙慎行五人，展开身法，悄没声的向前掩了过去。

　　李铁枪看得不错，前面远处的火光，正是撤离拇手下的五百精骑所点的火把。撤离拇与韩常分手以后，就来到了黄天荡东首的大路上守候。

　　到距离金兵不过二十余丈远近时候，蒙慎行向身后做了一个手势，要大夥儿上路边树顶而行。五个人各自施展轻功，跃上了树梢，这时轻功高下，丝毫取巧不得，立时分了出来。

　　秦汪冉幼年时迭逢名师，可谓是'科班出身'，所学的轻功无一不是极品中的极品。在朱家尖船上救险，使的是'一叶知秋'，其长处在于及远，方才收拢芦苇，用的是'分光散影'，那是在斗室中进退趋避的无上妙法，现下上树，使的却是'梯云纵'。梯云纵一向被认作是由武当派祖师爷张三丰所创，乃是武当不传绝学。其实天下武功都有一个本源，然后再慢慢的发展完善而成，哪里有从石头里面一下子蹦出来的？梯云纵在一代大师张三丰手中得成大成那是不假，但其实早在五代十国的时候，江湖上就已经有了这一门功夫的雏形，其要义是在于如何在空中换气借力。比方说，你原来本领，一跃可达一丈高低，但如果会得梯云纵，在空中又有可以借力处，就可以接连的一丈一丈跃上去，中间看上去却好象只是一跃。而其他轻功，即使也能借力续行，中间的停顿却极明显，高手相争，生死只在一线，这霎那的停顿可就是你死我活的分别了。只见秦汪冉身子只一起，便一口气到了树顶，姿势如鹰击长空，好看已极。

　　李铁枪的功夫却全是今天从东边学一招，明日在西面偷一招的胡乱拼凑而来，差不多算是打一架，明白一条武功上的道理。他能够有今日的名声，实在是不晓得打了多少恶仗才可以成功。李铁枪的轻功极其朴实，只见他提气一跃，身体轻飘飘的向上直飞，到了差不多树中地方，紧接着他身体向后微仰，居然沿了树身直冲了上去，速度快极。这对顶尖好手来讲，本算不上有什么了不起，但奇的是李铁枪在树身上这么猛的一溜跑上，竟然没有丝毫声响发出，当真就好象是一只猫儿上树一般。这却不知道是李铁枪从那一家的功夫里面偷学来的了。

　　蒙慎行轻功平平，若勉力上跃，几个起落也可以到顶，但是不是能够控得不发出声响，那就难讲的很了。蒙慎行上树，用的却是手指上的功夫，他只不过轻轻一跃有五六尺高低，接着便伸右手向树身上插去，五指如破腐朽，深陷入木，右手接着再向下猛然发力，身子又窜上了三四尺光景，这么样两只手先后交替，也是极快的就到了树梢。

　　许大马棒上树的法子却颇出乎诸人意料，他是先从腰间解下了一条极长的马鞭，那马鞭似乎是用牛皮浸透了油熬炼成的。许大马棒右手一挥间，那条马鞭便如同一条灵蛇般飞上了半空，寻了一处枝叶紧紧缠了上去。妙处却在许大马棒如此猛力挥鞭，从头到尾却静悄悄的全无声息。接着许大马棒腕上用力，一个身子就好象上钩的鱼一样弹了起来，在空中划了一个大大的弧线上了树。这一手玩的实在是漂亮的很，连蒙慎行也看得在肚子里面暗暗喝彩。

　　蒙七的功夫却同蒙慎行截然不同，他身子跃起以后，并不是直直的向上而升，而是斜飞而出。别人是上一棵树，蒙七却上的是两棵。只见蒙七的一个身子在两棵相邻的大树间来回穿梭，只要一接触其中一棵树的树身，便向另外一颗反弹而去，就这么折转而上，直达树端。这一手功夫有个名堂叫作'草上飞'，相传乃是唐时红线所创，武林中流传甚广，但能够使得如蒙七这般行云流水的，江湖中也是寥寥可数。

　　诸人都上了树，接着便在树顶上向金兵所在的地方行去。初时行得甚快，到距离金兵只有四五丈时，各人屏声静气，慢慢的靠向前去。再过一小会，树下金兵脸上的胡须都在火把亮光中看的清清楚楚。金兵阵前一员大将，骑了一匹黑油油的高头大马，身披重铠，怀中抱了一对金装锏，正是撤离拇。
