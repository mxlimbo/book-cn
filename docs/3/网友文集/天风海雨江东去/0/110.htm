第一百一十回

　　蒙慎行向秦汪冉努了努嘴，又指了一指撤离拇，意思是让秦汪冉招呼撤离拇。秦汪冉微微点头，算是应了。众人这时几乎已经在金兵的头顶之上，下面金兵的呼吸之声，清晰可闻。

　　蒙慎行慢慢的举起了右手，众人都盯着他的动作，只等蒙慎行右手挥下，就一起跃下抢马。

　　蒙慎行终于重重的把右手挥下，一道红光闪起，秦汪冉第一个冲了下去，血龙宝刃，就象从最深的地狱底部升起索命的阴鬼，舞成了一朵艳的让人伤心的红花，铰向了撤离拇的头顶。凡能看到秦汪冉这一刀的，就再也不会奇怪为何秦汪冉会得了个绰号叫'冥天血龙'。

　　蒙慎行也从树上扑下，与秦汪冉只是身前身后，运起混元乾坤掌力，右掌瞬间已变得鲜红如血，左掌却变得青白如玉，双掌一左一右，劈向了两名金兵。掌力到处，两名金兵被震得飞出丈外，人在空中便已自鲜血狂喷，待落到地面，早已经没有了气息。蒙七与李铁枪分别在秦汪冉左右扑下，蒙七身在半空，手一扬间，'哧哧'破空声响起，还看不真切是什么暗器，数名金兵便已经应声滚落马背。正在蒙七背后的许大马棒看到，心中一惊，蒙七所发的，正是'细草闲风斜雨针'！这种天下暗器中歹毒第一的斜雨针从黄工直死后已经绝迹多年，怎么今日会出现在这少年人手中？

　　李铁枪手中没有武器，他身在半空，就已急速旋转了起来，靠近地面时，就好象是一个大陀螺将要落地一般。一团白影中忽然腿影飞起，两名金兵中腿落马，落地以后兀自不知道是被什么击中。

　　许大马棒的兵刃却是他从不离身的旱烟袋，那旱烟袋黑黝黝的看上去毫不起眼，居然通体是精铁打就，许大马棒身未着地，旱烟袋上好大一个烟锅早已经敲了在一名金兵头颅上，白花花的脑浆崩出去了好远，惨不忍睹。

　　撤离拇反应也算得上一等一的了，红光一起，他便向前猛然伏身，脚下同时催马，左锏'苏秦背剑'，向背上挡去，右锏'倒打老君炉'，向身后打去，这两招不求伤敌，只求能够逼开身后来袭的敌人。可惜这一切都不能延长他的性命一分一毫。秦汪冉身在空中，撤离拇出招看得清清楚楚，气息陡沉间，下落之势快了数倍，下落方向也由撤离拇的马后变为了撤离拇的战马右侧，撤离拇的右锏便堪堪在秦汪冉头上数寸处扫过，刀光一敛，血龙宝刃改铰为刺，刀尖紧紧贴着撤离拇的左锏扎入了撤离拇的后肋，层层重铠竟不能阻刀势分毫，刀锋只一入肉，便又急剧旋转了起来，鲜血如狂花怒放，血龙宝刃带着血肉从撤离拇前腰飞了出来，撤离拇半个身子就在这惨烈至极的一刀下被斩得不知去向。这时秦汪冉正好落地，伸左手扣在马肚带上只一扯，牛皮所制的马肚带应手断绝，秦汪冉紧接着左手猛的一震，撤离拇已经失去生命的躯体便向左侧跌了开去，秦汪冉更借了这一震之力，腾身上马。所有这一切都发生了在电光火石之间，真的就如是从虚无冥天中而来的催命无常正在勾魂拘魄一般。

　　这时旁边的大队金兵才如梦方醒，各自催马围向蒙慎行诸人。此刻便正所谓分秒必争，若是一旦被围将了起来，就算是你武功通天，不付出相当代价，也休想脱身。蒙七再次扬手，细草闲风斜雨针二度出手，当先两名金兵中针落马。蒙慎行和李铁枪都跃上马背，催马向前，李铁枪经过蒙七身边，伸手一提，便把蒙七也扯上了马，但二人都已经来不及去抓其他失了主人的马匹的缰绳了。想不到许大马棒在这等危急时候，却显出另外一种天下独步的绝技。
