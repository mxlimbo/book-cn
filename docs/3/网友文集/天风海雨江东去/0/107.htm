第一百零七回

　　走了有差不多半顿饭功夫，远处传来人喊马嘶之声，接着一片红光亮起。韩世忠所料不错，金兵果然开始放起火来。所谓燎原野火，势不可当，但草原上面着火，虽然扑天盖地，过后千里焦土，但若单论火势，又怎么能和比人头还高的芦苇荡着火相比？'劈劈剥剥'声中，几处起火的所在迅速扩大，连到了一起，烈焰接天。火，就象有生命的巨兽一般吞噬着面前的一切，更把吞噬的物事变成了自己的一部分，蠕动着向前滚去，越来越快。一开始是火借风势，后来火竟似乎比风跑得还快，熊熊烈火裹着粗粗的浓烟，在芦苇荡上左冲右突，奔腾无忌！火起以后，韩世忠一行人行路却容易了许多。原来火势冲天，远远的照过来，就好象点了盏极大的明灯，与白日间所差无几，众人找路方便了不少。众人向前急奔，希望能够在烈火赶到前出了黄天荡。蒙慎行负了梁夫人，李铁枪拉了赵得胜，许大马棒伸手扶在了韩世忠腰间，提气施展轻功，众人跑得极快。一开始众人离起火处是越来越远，但才半盏茶功夫，火势蔓延的速度已经快过了众人奔跑之速，而且似乎更要追了上来！若只是秦汪冉和李铁枪二人，两人轻功卓绝，想来应当能够在烈火卷来前出了这一片芦苇荡，但现在众人中有数人武功低微，需他人带了而行，那便万万逃不过了。

　　秦汪冉这时晓得，若众人只是这般奔跑，出不了黄天荡，就要被烈火追上，定要想个法子挡住烈火才好。但这烈火吞天噬地，又有什么东西能够挡得它住？他边行边心中急想，突然间脑际灵光一现，想到了一个法子。接着秦汪冉便停了下来。李铁枪眼角撇见，急叫道，"三哥，你怎么还不快走！"

　　秦汪冉大声喊道，"你们先行！我要挡住野火！"
