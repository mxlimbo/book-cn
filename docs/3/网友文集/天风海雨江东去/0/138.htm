第一百三十八回

　　秦汪冉温温和和的道，"贾兄还请放心，杀人灭口的事在下虽然也曾做过几回，但在下既然说过了不伤各位，便决计不会食言。不过今晚上却容不得三位回去报信，点了三位的穴道，咱们彼此都放心些。在下的点穴功夫虽然平平，但也有些蹊跷处，若三位强自运气冲穴，便难保会不会有个什么三长两短的。这穴道一十八个时辰以后自解，那时晚晴山庄的事差不多也该有个了断了，那时候三位要到哪里去，便与在下没什么相干了。"

　　说话间，秦汪冉弯腰到那书生身边，伸手拍开他身上穴道，但旋即又封了他腿上以及左臂的几处穴道，只留他一只右手可以活动，秦汪冉续道，"这扬州城外，按理说不该有野狼什么的，不过为三位的安全起见，还是小心点的好，贤兄功夫不错，一只右手，防身想来该是尽够了。"

　　那商贾模样之人躺在地上，恨恨的道，"我兄弟三人这次栽是栽到了家，以后也不敢想找尊驾要回这个场子，只是尊驾能不能将高姓大名见示，让我们兄弟栽也栽个明白？"

　　秦汪冉正向大道上走去，闻言回头笑道，"在下哪里有什么高姓大名了，小可凑巧也姓贾，排行老大，熟人都叫我贾老大，贾兄也这么招呼我就成。"

　　那自称贾老二之人听到秦汪冉这句话，气得几乎背了过去，只是苦于身不由己，就是想动一动手指头也不能够，只能眼睁睁的瞪着秦汪冉的背影消失在树丛中。 ……

　　秦汪冉道，"至于这三人到底是谁，鹏举，我既已经发誓守密，便不便告诉你了。"

　　岳飞问道，"你接下来便去了晚晴山庄？"

　　秦汪冉摇头道，"不是。我行事的作风你最清楚不过，该能猜得出我下一步会去哪里。"

　　岳飞略一沉思，道，"我知道了，你定然是去了……"
