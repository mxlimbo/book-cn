第一百三十回

　　这扬州杨柳岸，晚晴山庄说起来可颇有些来头。晚晴山庄首创自唐末一代奇人独孤云天的手中，一直是颇神秘的一股势力。晚晴山庄当年乃是以漕运和盐业为起家之根本，是天下间数一数二的富商大贾。杨柳岸里面，从没有出过一个真正意义上的"江湖人"，但却一直同江湖上似即似离，有着千丝万缕的联系，更以其高深莫测的家传武功威镇江南武林，其中尤以一式神秘已极的'山抹微云'，号称天下六大绝招之一。五代十国，群雄纷起，这班唱罢那班登场，也不知流尽了多少英雄血，砍落了多少好汉头，多少荣华富贵盛极一时，却难传三代。可晚晴山庄居然能在这乱世中一直屹立不倒，单就这么一份不倒翁的本事，就够叫人咋摸品味的了。入宋以来，赵氏夺天下于孤儿寡母手中，对自己的臣下当然信不太过，对行商和江湖上的势力也限制颇多 ，晚晴山庄的势力自然也有些不如以前，但仍不失为一股无人胆敢轻辱的力量。而若是单以财势计，除了皇帝老儿和那些个封疆大吏不算，北李记，南杨柳，算得上是当时最富有的两个家族了。

　　独孤鸿微微一笑，道，"小弟不才，腆居晚晴山庄第三十七代的主人，仗了先人余荫，过个隐居富家翁的日子，和秦兄赤手空拳打下的大名可不能相比。"

　　秦汪冉道，"我那也不过是虚名罢了，江湖上多有言过其实之事，秦某的实际同传说的大有出入，独孤兄言过了。"

　　独孤眉睁着大大的眼睛，突然插口问道，"人家都说你手下送了六百余人的性命，乃是江湖上杀人最多的一个高手，那难道是真的吗？"

　　秦汪冉心中暗暗叹了口气，他平生快意恩仇，饭恩必偿，睚眦必报，取人命如草芥，这时却平生第一遭后悔自己双手染血太多，小心翼翼的答道，"六百人是绝对没有的，不过，不过……"

　　独孤眉无邪的眼光就似乎是在拷打秦汪冉般，秦汪冉艰难续道，"这个，这个，秦某这双手下，一二百亡魂倒确是有的……或许还要多些……"秦

　　汪冉接着赶紧道，"正所谓人在江湖，身不由己。行走江湖的，恩怨情仇逼迫之下，想要手不沾血，那是极难做到的事。但秦某与人交手之时，向来颇留余地，若不是别人逼我到万不得已的时候，秦某也是决不轻伤人命的。"

　　独孤眉好奇的看着秦汪冉脸上的那一道长长的刀疤，又问道，"你武功那么高，谁又有这么大的本事，能在你脸上砍了这么狠的一刀呢？"

　　独孤鸿岔道，"眉儿，别这么没规矩，人家的往事，哪里有你这么乱问的。秦兄，先父母早故，我这妹子是我一手带大的，平日间晚晴山庄里人人都让着她，硬是被宠坏了，秦兄还望莫怪。"

　　秦汪冉苦笑道，"这刀疤也没有什么大不了的。我也不是天生就是一等一的好手，功夫浅时，自然就难免被别人伤了。好在我运气不坏，没有在那时送了性命。"

　　独孤眉并不松口，接着问道，"你还是没有告诉我，是谁砍了你这一刀呢！"
