第一百三十二回

　　此景……

　　此时……

　　秦汪冉说到这里，微喟道，"鹏举，你是知道的，我自幼漂泊，先父故去时，梁山上的长辈亲朋也几乎都凋落殆尽，朝廷要斩尽杀绝我这梁山余孽，明教的人也要找我讨回帮源洞的血债，江湖之上，举眼全是仇家，遇到你之前，连一个可以信任之人都没有。一个人刀头舔血，挣扎出一条生路，别说成家，连交朋友的心都是极淡。我又自知相貌甚丑，从不敢想有哪一个女子会愿伴我一生。打从做了那没本钱的勾当，挥金如土的日子也颇过了一段，'帷飘白玉堂，簟卷碧牙床'，算是绝色的女人，我睡过的也不是一个两个，差不多也快'赢得青楼薄幸名'了，但那一日我见到小眉，不知为何，却，却……"秦汪冉语声断绝，说不出却什么来。

　　岳飞叹道，"但凡是人，谁又能一生不动心的？汪冉，你的感受，我是晓得的――有些事，的的确确是无可理喻的……"

　　秦汪冉苦笑道，"可笑的是，我这只癞蛤蟆，也不晓得自己撒泡尿照照自己的模样，当时居然起了个心，想能够不时的见她一面。于是当独孤鸿邀我第二日去杨柳岸晚晴山庄时，我居然不假思索的就答应了下来……"

　　岳飞道，"就换作是我，那也肯定是要答应的……"

　　秦汪冉感激的望了岳飞一眼，续道……

　　此时……

　　此景……

　　第二日却是晴天，秦汪冉日才过午，也没吃响午饭，就离了住处，向杨柳岸行去。虽然独孤鸿是请他去吃晚饭，但一个人怎么待得住在房间里？秦汪冉顺了护城河，一路迤逦缓缓而行。路是一条狭狭的煤屑路，脚踏上去，便发出一种摩擦的微音，舒服的很。有十多里路，真乃五步一楼，十步一阁，一处是金粉楼台，一处是竹篱茅舍，一处是桃柳争妍，一处是桑麻遍野。路全是弯弯绕绕的，望不远，却总给人带来惊喜，一忽儿一座桥扑进眼睛里面来了，一忽儿又是一座亭子冷不盯的跳了出来――明明前面空空荡荡的什么都没有，忽然一个弯儿，就'柳暗花明'了。真正数不尽"三十六家花酒店，七十二座营弦楼"。秦汪冉心中满满装着莫名的喜悦，踏在这无边的春色上走着，走着……总不能这么样走一下午吧，眼看离城已远，行人已稀，虽然秦汪冉的腿并没有丝毫倦意，肚子却实有些饿了，于是找了个路边的茶馆坐了下来，茶馆的名字换作绿杨村，却不知是哪个落第秀才取的，竟然还雅致的很。茶馆破旧不堪，也不知使了多少年的幌子，挂在绿杨树上，随风飘展。
