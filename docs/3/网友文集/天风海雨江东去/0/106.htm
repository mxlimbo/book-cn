第一百零六回

　　荆棘烈火路，腥风血雨行

　　世忠诸人把小船靠了边，大夥儿匆匆跳上岸来。这时候众人心中稍定，才得功夫辨认方位，发觉眼前密密麻麻的全是芦苇荡。秦汪冉心中苦笑，原来众人到的地方，正是日间来时曾到过的观鱼堤。蒙慎行低声问道，"现在咱们身在何处？咱们又该往哪里去？"韩世忠四下里看了一看，应道，"这里似乎是观鱼堤。此处芦苇茂盛浓密，躲在中间，黑夜里敌人一时间决计发现不了咱们。但兀术甚能用兵，想必定然会派一支兵候在岸上搜索逃脱的宋兵。待到日出，这里就藏不住了。而从这里往北近百里，都在金将孛堇太一所帅兵马控制之下，中间更有一大段路全是无遮无拦的平地，地势平坦开阔，一览无余，难以隐藏。而在平原之上，骑兵掩杀过来，断无生路。若是向南边去，左近除了鹏举在牛头山上的一支兵马，再没有宋军了。咱们现下，也只好去投奔鹏举，等见到了鹏举后，再另做打算。不过兀术定然也会想到咱们往那边去，进而派兵在中间要道上截杀，这一路上能不能逃过金兵的围杀，就得看咱们的运道如何了。"

　　秦汪冉问道，"韩将军，不知道去牛头山一路之上，有没有什么必经之处？"

　　韩世忠叹道，"不错是有一处，那地方叫作懒鸭滩。这懒鸭滩有差不多两里地的样子，一边是连绵的小山丘，一边是小淮河河水，依山背水，可以埋伏兵马。若兀术晓得那处所在，定然会派骑兵先赶到那里去等咱们。不过只要过了懒鸭滩，往后可走的路便有多条，就不那么容易追踪堵截了。"

　　蒙七问道，"韩将军，那咱们是现在立刻动身呢，还是等到天亮再说？"

　　韩世忠叹道，"这中间各有利弊。夜里走路，咱们又没有火把，这一带道路错综复杂，就怕找不到路。但等在这里，又怕天明时金兵已经把这一带重重包围封锁了起来。唉呀！"

　　蒙七被韩世忠一声惊呼吓了一跳，问道，"怎么？"

　　韩世忠道，"我刚才忘了一件极重要的事，只怕兀术会想到放火烧这一片芦苇荡！"

　　秦汪冉道，"若是这样，那咱们就当如何？"

　　梁夫人接道，"世忠说得不错，兀术定然会想到纵火逼咱们出来。事不宜迟，依妾身之见，咱们还是立刻向牛头山去的好。"

　　蒙慎行道，"那便这么着罢。咱们中间，有谁识得去牛头山的路的？"

　　这一句话问出，众人你看看我，我看看你，不由得面面相觑。韩世忠涩声道，"我一向率领水师，从镇江到黄天荡，带兵走的尽是水路。我与红玉，都是在地图上晓得的去牛头山的大概方向道路，若是在天亮时候，不难寻出路来，可这黑夜之中……方才我说夜里怕找不到路，就是为此。"而许大马棒，李铁枪等几人里面，除了秦汪冉曾经来过一趟黄天荡以外，个个都是平生第一遭到这个所在，那就更连问都不用问了。

　　赵得胜自从上了小船，除了指点众人划桨时略略说了几句话外，一直只是沉默，这时突然开口道，"从黄天荡到牛头山这一段路，我小时候倒是走过的！现在倒还记得一些路。"

　　众人听到这句话，就好象在漆黑深夜里面乍见明灯，纷纷围了上来问道，"那便怎样？"

　　可惜赵得胜第二句话却是，"但这观鱼堤周围，我也是第一次来。咱们只要离了观鱼堤，上到大路，我差不多就能摸出方向来。"

　　许大马棒心急，听到这里忍不住骂道，"他奶奶的，你这小子说话怎么这么大喘气！一句进，一句出的！"

　　秦汪冉截道，"那总也比没有人认得路强的多了！这黄天荡边的一段路，咱们就慢慢摸着走吧。"

　　众人计议得定，便自出发。现在众人晓得的，便只有牛头山是在南方，借天上北极星辨定了方向，众人只管向南边而行。幸好众人之中，秦汪冉与李铁枪都练就了夜中视物的本领，两人走在前面，分开芦苇，深一脚，浅一脚的向前趟去。风吹芦苇，沙沙的极是怵人，偶尔众人惊动了正在睡觉的夜鸟野鸭，野鸭'呱呱'的大叫着扑楞楞振翅飞起，众人的心头就一阵紧缩。
