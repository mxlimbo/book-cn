第一百十五回

　　突然间负责侦察的两名金兵急匆匆的跑向张本，道，"将军，那几个宋人来了！"

　　张本问道，"他们离这里还有多远？是步行还是骑马？"

　　其中一个金兵气急败坏的道，"只有一里来路了！既不是骑马，更不是步行！"

　　张本诧异道，"怎么？"

　　那金兵道，"他们是乘了筏子顺河而来的！"

　　张本猛的自拍前额，叫道，"糟了！我怎么就没有想到这一着！"赶忙回身大声道，"所有弓箭手，快随我来！"

　　待到张本率了一百来名弓箭手到得河边，正好看见韩世忠诸人的筏子漂了过来。许大马棒看见金兵，道，"果然有金兵！幸好咱们没有硬闯了来！"

　　金兵这时可以做的，只有张弓搭箭，向筏子上众人射来。但两下里相距既远，又有秦汪冉和李铁枪两人护着，这些许弓箭自然伤不到诸人分毫。张本立刻下令埋伏的骑兵在岸上追赶，但越过自己挖的壕沟先浪费了不少时间不讲，不过三里路过后，大路与河岸已经是分道扬镳，沿岸尽是松软湿土，马跑在上面，或者是泥足深陷，或者是一步一打滑，更有人仰马翻摔倒在地的。眼见着筏子远去，张本气得在岸上只是跺脚，却已经是无计可施。

　　众人随波逐流，不觉间数十里地已经过去，约摸后面金兵再也无法跟得上来，众人便撑了筏子到岸上，重新上马而行。不过这几匹马自打生下来就没有游过这么长时间的泳，这时已经疲惫至极。众人就不去催马，信手由缰，听了马儿在那里慢慢而行。昨夜以来，头一回觉得心里轻松了些。

　　兀术大破宋军水师，大军终于可以渡到了江北，多日间的郁结一旦而解，心中大快。十万大军渡江，也是件颇壮观的事，兀术坐了在帅船头上一边欣赏，一边盘算下一步该当如何行事。

　　'现下我这一支兵仗打得也尽多了，所获亦是甚丰，北归休整，运还辎重乃是当务之急。过江以后，第一件事是与孛堇太一会兵，接手他押运而来的粮草，下一步就是率部北归。北归有两条路可以选择，走东路当过楚州，楚州前些日子仍在宋军手中，听说挞懒正在围攻这一座城池，也不晓得他得手也未。走西路则经过陕西，娄室元帅正在陕西攻城略地，倒也可以顺便去助他一臂之力。但无论走东路西路，都不妨先去镇江走一遭，上一次到镇江败在韩世忠水师手下，这般富庶的一个城镇就在眼前，如果不得染指，实在有些可惜，不如先洗劫了镇江再图北归。'

　　心中算盘打定，兀术嘴角不由得露出一丝浅笑。

　　已是正午时分，兀术在帅船上设酒，邀了黑峰，柳如是共饮。

　　柳如是昨晚回舱以后一夜不能入睡，一会儿想到自己多年追求的荣华富贵终于近在眼前，兴奋的很，一会儿又觉得自己的所作所为大是不妥，又颇烦恼懊悔。就这么翻来覆去，迷迷糊糊的，柳如是一直折腾到天明也没有想出个所以然来，起身后便一个人立在船尾，呆呆的望了湖水发怔，思潮滚滚，心中有万语千言想要说了出来，却能与谁诉说？种种事端，千头万绪，孰是孰非，又有谁能告诉自己 ？看着湖水上偶然泛起的一圈圈水波慢慢扩大，扩大，终于消失了在新漾起的水纹里面，再也看不见了。柳如是惘然若失，脑中一片空白。待坐到席间，一开始柳如是只是强打精神，假作笑语风生。今日兀术摆下的这桌酒，颇有些庆功酒的味道在里面。在这一役中，除了那一位神龙见首不见尾的算出月蚀的无名高人以外，柳如是出谋划策，实在是得居头功。是以兀术频频举杯向柳如是敬酒，言语中极尽笼络接纳之意，许以心腹相交。不过几盏酒下肚，柳如是心中的烦恼就丢到了九霄云外，强颜欢笑慢慢变作了真正的开怀大笑。功名利禄，锦绣前程，就在伸手可及的眼前，还有什么可以去烦心的？人生得意当尽欢，莫使金樽空对月，今朝有酒今朝醉，明日愁来且自愁，却哪里有时间去想得了那么多了？

　　兀术转而面向黑峰笑道，"黑峰老弟，今日你立功也不在小。待到小王见到娄室元帅，定当向他力保老弟。将来老弟就是能得裂土封候，那也是不会让小王吃惊的事。"

　　黑峰双手施礼道，"末将多谢王爷提拔。"

　　兀术摆手道，"哪里的话！"言罢大笑。

　　突然一人匆匆走近兀术，施礼道，"报王爷，韩常，粘罕，查雄求见。"

　　兀术道，"知道了。让他们过来就是了。"

　　待到韩常等三人在帅船船头出现，兀术便觉得有些不对，原来三人看上去神色沮丧，不象是带来了好消息的模样。但兀术仍是起身迎去，笑道，"你们可是给我带来了韩世忠的人头下酒？"

　　韩常走在当头，紧走几步，单膝跪下，道，"末将等无能，有辱将命，还请王爷降罪。"

　　兀术摆手道，"没找到就没找到，这么大一片地方，要抓几个人，本来就不是十拿九稳的事，没有找到韩世忠等人也是正常不过的事。韩将军何必这么大惊小怪？"

　　韩常起身道，"回禀王爷，末将等确实不是没有找到韩世忠诸人，而是遇到了他们后却生生被他们逃了出去！"

　　兀术听到这里才脸色一沉，道，"这就是你们的不对了。韩世忠那条小船上不过十来个人的样子，你们几个人手下近两千人，没有遇上他们倒也情有可原，怎么遇上了还能让他们逃了出去！"

　　粘罕在一旁吞吞吐吐的道，"王爷，他们，他们还不单单是逃了出去！"

　　兀术的脸色这时已经是颇不好看，问道，"那么，你们还有什么好消息没有告诉我的？一并说了出来，也省的我去乱猜！"

　　粘罕不敢回答，眼光看了向韩常。韩常咬紧牙关，终于大声道，"回王爷。韩世忠诸人不但逃了出去，更还夺了十余匹马，杀死了六十余名咱们的精兵，连撤离拇将军也丧生在他们手中！"

　　兀术听到这里，手上用力，'格'的一声，手中瓷杯被他生生捏碎，杯中美酒沥沥洒了一地，瓷杯碎片刺入兀术手中，鲜血渗出，兀术却恍若不知，沉声问道，"撤离拇是怎么死的？"
