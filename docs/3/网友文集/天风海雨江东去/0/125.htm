第一百二十五回

　　岳飞冲秦汪冉眨了眨眼，低声道，"前些时候一个朋友送了我六瓶顶好的三锅头陈年山西汾酒，我一直收藏至今，刚才没有舍得拿出来，现在正好咱们哥俩偷偷的喝。"

　　秦汪冉微微耸肩道，"那好得很啊。"

　　岳飞道，"随我来。"言罢转身而行，秦汪冉便在后面不即不离的跟着。不多时后，两人却又回到了岳飞帅帐之中，待秦汪冉跟进以后，岳飞把帐篷掩了，随手把油灯放在地上，接着便走向书架。

　　岳飞言中的六瓶宝贝汾酒，却原来摆在书架之上，藏了在一排兵法书的后面。孙子太公，白石吴起，参差而列，权充护酒的大将。

　　秦汪冉看见，微微一笑道，"鹏举，你对兵家典籍可有些不敬啊。"

　　这时岳飞两手，各拎了三瓶酒，闻言转头笑道，"这些前辈当年，恐怕也是爱酒的。如今让他们伴酒而眠，岂不也是一件美事？弄不好在他们心中，这也是求之不得的好差事呢！"

　　秦汪冉并不接口，在黑暗中沉默着……

　　岳飞将酒放了在桌上，把桌上几根残烛点了，伸手捻灭了油灯，再从桌边抽斗里翻出了一把牛角柄解手小刀，先起开了一瓶酒，递给了秦汪冉，接着又为自己起了一瓶酒。

　　秦汪冉接过酒瓶，并不立刻就饮，而先是把鼻子凑了上去深深了吸了口气，闭了一会眼，才缓缓吐气道，"果然是好酒。酒香辛而不辣，烈而不冲，醇得很。"

　　岳飞缓缓道，"咱们兄弟二人，也颇有些时候没在一起这么喝酒了罢？"言语间右手一翻，掌心中变戏法般的现出两只青花瓷酒杯，接着其中一只酒杯竟然缓缓的浮了起来，又缓缓的平飞向秦汪冉，就好似有一根无形的丝线在空中拴着那只酒杯般。单凭这手'千里同心'的精纯内力，江湖上就难找出几个人办得到的，而这寥寥几个人中，能做到象岳飞施展时这般信手拈来，行若无事的，恐怕就更绝无仅有了。

　　秦汪冉却只当作没看见岳飞露的这手惊人已极的功夫，伸左手接了酒杯，先满满给自己斟了一杯，举杯道，"先干为敬。"接着仰头喝了，只觉得一股醇厚如玉的热流缓缓顺了喉头而下，香冲鼻际。秦汪冉不由得再赞道，"好酒！"

　　岳飞微笑着也斟满了手中杯，一饮而尽，闭目道，"当然是好酒，足以浇化胸中块垒，忘却世间烦恼。"

　　秦汪冉拉过身边一张椅子，把大半个身子滑了进去，缓缓的又斟了杯酒，也闭上双眼，将杯中酒慢慢抿了下去，轻声道，"酒确是好酒，但有些胸中的块垒，就是美酒成河，也化它不去，有些世间的烦恼，就是身在醉乡，它也常在心头……奈何……"语声如梦如幻，似是在对岳飞而语，又似只不过是自言自语。

　　岳飞看了秦汪冉疲惫的面容，叹道，"汪冉，你到底遇上了些什么事故，何必自苦一至于斯！"
