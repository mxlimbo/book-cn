第一百十八回

　　沉默良久，梁夫人才开口对韩世忠道，"妾身当时得受封安国夫人的时候，曾经有幸蒙召进见，入谢太后。太后她老人家为人体下，仁慈得很，对着妾身，那时就已经是多加宠眷了。后来将军平定苗贼 ，居功至伟，皇上亲书了"忠勇"二字，悬揭旗上，颁赐将军，一时圣宠无二。

　　妾身那时随着将军同至建康，又入谒了太后数次，极蒙褒宠。前些时候听说皇上已经回到越州，又去虔州迎还了太后。皇上对太后是孝顺的很的，想来太后说的话，在皇上耳中也该是极有分量。不如妾身这里拟一封密奏给太后，写得看上去呢，象是在弹劾将军，但其实里面却实是在请求赦免将军。太后是个软心肠的好人，想来会顾念前功，在皇上面前说些好话，豁免了将军这场新败之罪，那也未可知。"

　　韩世忠半响无语，最后道："现如今也只有用这个办法了。不过我也得再上一封奏章，自劾己罪，里面表述我忠心耿耿，决无二心之意。两下里一起使劲，或许能够让皇上继续信任于我。"

　　蒙慎行听得云天雾罩，全然没有搞明白韩世忠夫妇到底在弄些什么名堂，怎么天底下还有老婆参劾丈夫的道理？但明明眼见韩世忠夫妇情爱甚笃，料想他们总是有他们的道理的，蒙慎行也就不开口再问。

　　韩世忠看见蒙慎行神色，苦笑解嘲道，"让蒙帮主见笑了。我身在官场，许多事做得让自己都恶心。但又能有什么法子呢？俗话说得好，常在河边走，怎能不湿鞋？要想保住头顶上这顶乌纱帽，保住手中这一点点兵权，不怕蒙帮主笑话，更无耻的事世忠以前都做过。有时我真的想……"话语到此，突然中断，韩世忠到底还是不敢把牢骚说出口来。

　　蒙慎行似是明白了些韩世忠的烦恼，又似乎是什么都不明白，也只好说，"韩将军，家家都有难念的经，又有谁没做过违心的事？只要大主意没拿错，那便差不到哪里去。"

　　韩世忠苦笑道，"只要大主意没拿错，小处错些不打紧？这话说起来容易得很，可当真做起来了，什么是大，什么又是小？这十数年来，我是真尝到积重难返的滋味了。有些事，你一开始总以为自己定能把持得住，可慢慢的就身不由己，发现全不是原来想的那么回事。咱们的大好河山，说不定就会丧了在这些不打紧的小错里面。"

　　韩世忠这一番话说得愈发吞吞吐吐，含糊不清，蒙慎行想赞同，不知何从赞同起，想反驳，也不知何从反驳起，只好道，"我是个山野粗人，大道理懂得的是不多的。只是以前听人说过，只要文官不贪财，武官不怕死，当官的再为老百姓多操心作主些，这天下就是铜浇铁铸的，任谁也夺它不去。难道这十个字做来就这么难吗？"

　　梁夫人听得扑哧一笑，道，"这十个字何止是难啊，几百上千年来，就怕没有几个人做得到。蒙帮主，您是丐帮的大头脑，我且问你，你帮中的手下，为了什么才跟着你打天下？"

　　蒙慎行道，"这里面的原由就多了去了。有些是穷的没有饭吃，进了丐帮，为的是有碗饱饭吃，有的是慕丐帮天下第一大帮的名声，想借了丐帮的力量在江湖上混出牌号来的，也有的是想借丐帮的势力保护来做生意的，还有……我也讲不完这么许多。"

　　梁夫人肃容道，"蒙帮主，你讲了这么多，在妾身看来，却只有两个字罢了。这两个字不是别个，便是'名利'。只是想有碗饭吃的也好，还是想成了什么大侠英雄也罢，是想发财做老爷的也好，还是想名垂千古的也罢，所为或高或低，或尊或贱，其间虽然千差万别，却总逃不出'名利'二字。说起来人生富贵功名，不过是身外之物，但自古及今，那一个是看得破的？我这样讲，可不是说名利是样坏东西——求名利其实是件极好的事，若人人都不求名利，只怕大家早就都要饿死了。"

　　蒙慎行听得暗暗点头，并不开口，静静的等着梁夫人下文。

　　梁夫人顿了一顿，续道，"但天底下的好事却也少有不会变成坏事的。你想，寻常人辛苦一生一世，就算是每日都拼了命的干活，那也不过勉勉强强的糊自己一张嘴罢了，又能挣到多少名利？于是有些聪明人便琢磨了起来，看看哪里有求名利的便捷法门。慢慢的各人就就发现了各人的法子，这正是八仙过海，各显神通的事。江湖中的这类人，便是强盗蟊贼了，整日价出生入死，过得是刀头上舔血的日子，也无非就为了比寻常人能多挣些名利罢了。不过江湖中毕竟人心纯朴，想要什么，就真刀真枪的去抢去干，相比总还好一些。那官场之中，却大不一样，说起来跟妓院娼寮里面倒有些相象，里面追的也不过是'名利'二字，但数目规模却大的多，手段法子也巧妙的多。表面上或巧笑接送，或道貌岸然，或忠心耿耿，或两袖清风，或白鹤古琴，隐居山林，说什么'处则不失为真儒，出则可以为王佐，'实底下却无所不用其极，就强盗妓女怕也比他们善良清白些。但天底下总共就那么点东西，哪里有这么多的'名利'去分给大家了，你有了我便没有，我有了你便拿不着，所以才会去争去抢去害人，就算以致于头破血流，你死我活，那也没什么稀奇。名利之所在，才是官场中人要作官的真实目的，哪里是什么'为国为民'了。文官不贪财，武官不怕死，那就是要官场中人弃了作官的根本，蒙帮主，你倒说说看，这事该是容易还是不容易？"

　　蒙慎行迟疑道，"可我听说孔夫子，孟夫子都是做过官的，象他们这样的大圣人去做官，难道也是为了名利不成？"

　　梁夫人道，"名利二字是从古及今人人都想要的，就贤如孔孟，恐怕也不能例外。孔子为什么讲究'言扬行举'？那是因为他老人家生在春秋时候，那时用'言扬行举'做官，故孔夫子只讲得个'言寡尤，行寡悔，禄在其中'，那禄不是名利是什么？孟夫子是战国时候的人，那时候讲究以游说做官，所以孟夫子历说齐梁，做过那两国的官。你想想看，那时周天子尚在，那是正儿八经的皇家正统，若不是为了名利，孟夫子为何不去辅佐王室？"

　　蒙慎行沉吟了一会，道，"梁夫人，那若照你这么讲，官场之中岂不是就连一个好人也没有了？"

　　梁夫人摇头道，"好人这个词本来就虚的很。什么叫做好人？什么又叫作恶人？那金兀术在咱们眼中，乃是坏到不能再坏的大恶人，但在金国的老百姓眼里，只怕他却是个顶天立地，万家生佛的英雄豪杰 ！这不过是闲话，咱们也不必去在这些词语上的些微区别认真计较。官场之中，自然也有蒙帮主所说的那种好人——他们爱名利也是爱的，但却有自己的原则，决不会为了名利出卖这些东西，也就是所谓'贫贱不能移，威武不能屈，富贵不能淫'了。能够做到这一步的，那就是难得的好人了。"

　　梁夫人语峰一转，续道，"不过即使是好人，也并不是就真能够一尘不染的如神仙一样。比方说吧，世忠说起来大概还不算是太坏的一个官。但世忠身处官场其中，如果不同上下左右虚与委蛇，说得难听些就是同流合污，转眼就会被踢出了这个圈子，那就什么事也不要想做了。妾身这里掉一句文，'政柄操之自上'，无权无势的人，任你怎么样，也做不了多少事。只有手握大权了的人，才能正正经经的做些实事，反过来说呢，若他要作恶，那作起来也厉害的多。而也只有在上面掌权的人清了，下面的人才能清。但掌权却不是那么容易的，世忠劳苦功高，为朝廷南征北战，舍生忘死，打了多少恶仗，到如今才不过是区区一个节度使，手下兵不过万。要是就跟那些人翻脸，那就连节度使也做不成，又谈得上什么为老百姓操心作主了？不过蒙帮主还请放心，我们夫妇俩虽然算不上是什么青天大老爷的好官，但总还有一颗良心在肚子里面，有时不得已做了些狗皮倒灶的事，但失大节的事总是不会去做的。"

　　韩世忠打断道，"红玉，你胡说八道些什么呢。说这些混话，就不怕让蒙帮主听了笑话？"

　　蒙慎行'荷荷'干笑两声，道，"天色已晚了，还是早些休息的好些。明日还有好些路要赶呢。"

　　月儿俏然独挂树梢，星光有如媚眼，闪烁不定，树林里面静悄悄的，偶尔悉悉索索的似是有松鼠在树上跑来跑去，也不知它们半夜三更的不睡觉在忙些什么。众人披衣而卧，披星戴月而眠。李铁枪咭里咕噜的说些谁也听不懂的梦话，不时还来上一下拳打脚踢，显然是精力太过茂盛，就睡觉时也不肯稍微老实些。
