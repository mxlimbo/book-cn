第一百二十七回

　　秦汪冉又停了口，浅浅啜了口酒，举杯齐眉，将酒杯在掌中缓缓旋转把玩着，目光有些惘然地停留在杯沿之上，似乎在努力回忆着些什么，再过了片刻，才自言自语般地续道，"那一日的上午，淅淅沥沥的下了些牛毛细雨，弄得地上半干半湿的。下午时虽然雨已经停了，但天空依旧雾茫茫的，倒似乎雨还是在下着。那时也不知是为了什么缘故，我去到了扬州，真是忘了我当时为何要去扬州了……到了扬州倒也罢了，千不该，万不该，我不该在那一日的下午去了二十四桥得月楼，就是当年凌天第一次遇见小怜的得月楼。"

　　秦汪冉紧闭的双目前，似乎又出现了那时的情景……此时……此景…….

　　一切都是灰色的，灰色的天空，灰色的愁云，灰色的远山，灰色的近水，灰色的垂柳，灰色的河流……灰色的人群打着灰色的油纸伞，匆匆走在灰色的街道上……灰色的房屋，灰色的围墙，灰色的二十四桥，灰色的得月楼愁眉苦脸的蹲在了灰色的河水边……秦汪冉着了一身灰色的长衫，一个人立在桥头，望了灰色的天际，一时间脑子里面空空荡荡的，仿佛就是一尊灰色的雕像静静的竖立在那里，连时间都似乎被染成了灰色，迟滞不前……当那条灰蒙蒙的乌篷船划进秦汪冉视线时，秦汪冉其实并没有怎么留神，船舱外，只有一个披了一身灰色蓑衣的老艄公立在那里，不时的无力的摇一下桨。桨声埃萘，那条灰蒙蒙的乌篷船如蜗牛般慢慢的在河面上前行，离秦汪冉所立的桥头越来越近…….

　　其实秦汪冉是先听到她的。那声音柔柔的，却又清脆的很，就好象是在清晨起床时迷迷糊糊中失手打碎了一只琉璃碗相似，能直钻进你脑子里面去。秦汪冉听到船舱里面传出一声，"你快把它还我！"听到那个'你'字时，秦汪冉好象是被人在灰色的梦里猛然推了一把，听到那个'快'字时，秦汪冉的视线不由得转向了那条乌篷船，听到那个'把'字时，秦汪冉心中突然觉得怪怪的，就似乎在什么地方多年以前听到过这个声音般，听到那个'它'字时，秦汪冉心中突然起了个念头，想去看看那说话的人到底是谁，听到那个'还'字时，秦汪冉才明白过来说话那人是身处船舱之中，自己是看不到的，听到那个'我'字时，秦汪冉觉得心中似乎多了些什么东西，又似乎少了些什么东西，虚虚的上不着天，下不着地。语声终于消逝沉寂时，秦汪冉心中竟然觉得就好似是丢了件什么心爱的物事般，有些惘然。

　　紧接着'当琅'一声脆响，一只圆圆的不知什么物事从乌篷船的船舱帘子里面滚了出来，在船板上滴溜溜的乱转，接着便向船舷外滚去，眼看就要落入水中。

　　秦汪冉当时几乎没有任何念头在脑子里面，身子一腾间便向桥下掠去，其势快逾闪电。身在半空中时，秦汪冉便已经看清那圆圆的物事其实是一只淡青色的汉玉镯子，上面似乎还点缀了几点血红的颜色。待到秦汪冉身近船板时，那镯子已经离了船板，向灰色的河水中落去。秦汪冉长吸一口气，猛然挥臂，身形借力，在空中硬生生的转向，也向河水中落去，紧接着右臂暴长间抓住了船舷，左腿踢出，堪堪在那镯子行将落水时挑中了它。那镯子被这一脚踢得直飞到了半空之中，这时秦汪冉右手猛然发力一拉，身子被这一拉之力震得也向上腾去，秦汪冉身在半空，伸左手轻轻接了那镯子，转身轻轻落回了到甲板之上，正好见到船舱的帘子被掀了起来，一个人从里头钻了出来。首先进入秦汪冉眼帘的是那人的面容，秦汪冉就觉得自己胸口被绝顶高手以十成内力打了一掌开碑手相似，猛然一窘，又好象是一首妙手天成的诗，事前全不打招呼的贸贸然撞进了苦吟人的心头……

　　 '妙手写徽真

　　 水翦双眸点绛唇

　　 疑是昔年窥宋玉

　　 东邻

　　 只露墙头一半身 '

　　登徒子当年乍然见到的东邻大概也不过就是这样了吧。头发是乌黑如墨的，梳了两条长长粗粗的辫子，直落到合握可掌的腰际――什么叫三千烦恼丝，其实真正烦恼的，大概远不仅仅是烦恼丝的主人吧。大大的，弯弯的一双剪水眸子上，上下都长了有长长的睫毛，向中间微微拢着，呵护着黑水银般清澈的瞳仁，眼光从里面射出来，竟有些朦朦胧胧的。眉毛很淡很长，轻描淡写的在眼睛上面画过，几乎是平平的。鼻梁只好用秀丽来形容了，边沿有点儿微微上翘，为她平添了一丝俏皮。但又能找到什么言语来形容那一张嘴呢？红红的，鲜润的嘴就如同一朵含苞待放的玫瑰，完美的唇形在整张让人会在梦中思念的面容上描了一笔重重的红色――不错，红色，火一般的红色。跟在这张让秦汪冉目瞪口呆的面容后出现的是一整个少女。只见她着了件火烧般红的七镶七滚的丝绵袄子，镶滚之外，下摆与大襟上还闪烁着水银盘的梅花，袖上另钉着名唤"阑干"的丝质花边，宽约七寸，挖空镂出些花草花样，腰身如柳。手上戴了一双深红的绒线手套，下面着了一条也是火般红的细褶杂花裙，在早春的风中被吹拂得流动如水，脚下套了一双紫羔皮靴，又是火一般的红！

　　这女子的出现，当真便好似在无边无际的灰色天地中，突然划过了一道鲜红的闪电！

　　这少女低着头冲出了船舱，原没料到甲板会有生人，猛然抬眼间看见秦汪冉，骇得低声惊呼一声，紧接着抬了右手掩住了嘴，回头就又钻进了船舱之中。

　　接着船舱里面传出了一声低沉的男子声音，问道，"怎么了，谁吓到了你？"秦汪冉心中不知为何，听到这男子的声音后，竟然生出了隐隐的敌意。

　　语声中帘子又被掀开，出现的是一张中年男子的面孔，看上去差不多三十二三年纪，头上戴了顶雪白的簪缨银翅帽,身着银白色的长袍，腰间系着条碧玉带，面如美玉，目似明星，实是生得一表人才。秦汪冉看见，心中的莫名的不快平空又深了几分。那男子见到秦汪冉，也是微微感到意外，脸上亦露出了些疑惑和不快之意，沉声问道，"不知尊驾是何方高人？又不知尊驾不速而至，到小可船上有何高干？"

　　秦汪冉不愿跟他多说话，把手中镯子一扬，递到那男子手中，道，"我看见这只镯子要落入水中，便多管闲事捡了它起来。现在既然物归原主，我这不速之客也就该告辞了。"

　　这时船头那艄公才走到船舱前，道，"东家，刚才这位相公倒真的是下来救这只镯子的。东家您刚才没看见，这位相公的功夫那真叫高，从桥上'唰'的一下子就飞了下来，在空中还会拐弯，说起来就真给天上的神仙在腾云驾雾似的。"

　　听到艄公这几句话，那中年男子的面色转和，道，"哦！原来却是如此。方才倒是我言语无礼了，尊驾还望莫怪。尊驾救起舍妹心爱之物，使之不至落水，小可无以为谢，便请入舱小酌几杯如何？"
