第一百十六回

　　查雄道，"撤离拇殒身之时，我们几个都不在现场，但据当时在场的兵丁回报说，撤离拇将军是死在了一个使刀的宋人的偷袭之下！"

　　黑峰听得，猛然起身问道，"那是不是一柄红得象血的刀？"

　　查雄一楞道，"正是！黑峰将军，你怎么晓得？"

　　黑峰咬牙切齿道，"因为我也曾被这柄刀架在了脖子上过！早知今日，当时我就该把他乱箭射杀在朱家尖！"

　　兀术问道，"韩世忠诸人里面，还有些什么人？"

　　查雄答道，"据兵丁回报，韩世忠一行人里面，还有一个使长马鞭的，一个使樱枪使得极好的，两个会魔法的……"

　　兀术怒道，"胡说八道！真是越说越不成话了，这天底下哪里有什么魔法了！"

　　韩常道，"末将当时在黄天荡西面，未能与韩世忠诸人相遇，只能听兵丁描述来测度当时情景。想来那两人中，一个是使的极其微小的暗器，因为暗器形体极其细微，来无影，去无踪，士兵不明其所以然，才认作了魔法。而另外一人，则是练就了极罕见的凌厉无匹的劈空掌，可以在五步外伤人性命。这等武功甚是罕见，宋人中能够使得出来的屈指可数。"

　　兀术眉头拧作了一团，问道，"这几人是什么来历，你们心中可有数吗？"

　　韩常摇了摇头，并不开口。

　　黑峰插口道，"王爷，末将心中倒是有些猜测，就不知道对也不对。"

　　兀术转身面对黑峰，道，"什么猜测，说了出来。"

　　黑峰道，"末将在朱家尖时，曾经围住过一群中原武人，其中有两人在中原名声甚响，一个唤作秦汪冉，外号叫作冥天血龙，末将曾与他交手两招，他使的是一柄血也般红的宝刀。另外一人，乃是丐帮帮主，陆地龙王蒙慎行，此人当年曾经在万人兵营中孤身刺杀辽国大将。听说这蒙慎行练的乃是混元乾坤掌力，论掌力在宋人中可称第一。所以末将猜想，护了韩世忠逃走的人里面，这两人都在其中。"

　　兀术冷笑道，"韩世忠过了这一关，也未必就真的能逃走了。张本的千人马队在懒鸭滩正以逸待劳，等了他们去，他们就算是三头六臂，只要是人，久战之后，必会疲乏，我就不相信他们还能闯得过懒鸭滩去！"

　　兀术这时已经控制住了自己的情绪，接着道，"韩常，你们几个也不必过分自责。我原料想跟随韩世忠的几人，不过是他水师中的兵丁罢了，没有想到平空会钻出来这么些好手。说起来是我料敌有误，其中罪疚多在于我。你们也劳累一夜了，入座一起喝两杯解解乏罢。"

　　韩常三人称谢入座，众人继续喝酒，但比起方才的时候，气氛显然低落了许多。同样的酒，喝到嘴里面的味道却大不相同。

　　张本是在下午时候才返回本寨的，带回来的消息自然不会让兀术太过高兴。但事已如此，夫复何言。兀术即刻下令，全军过江后与孛堇太一会兵。大军浩浩荡荡，向北方而行，暂且按下不提。

　　韩世忠一行人继续赶路，到得傍晚时分，已经近了牛头山。远远的望过去，当真好象是两只大牛角立在天边一般。牛头山座落于建康与黄天荡中间，仿佛是太初时造物犯的一个大大的错误般孤零零的兀立在那里。牛头山分东西两座山峰，两峰森然并峙，列嶂如屏。方圆数百里里面，就只有它一座石山，显得格外突兀不群。山上巨石嶙峋，巧怪巉岩，层出不穷，上面光秃秃的几乎寸草不生，只有寥寥几株极顽强的荆棘在石缝中挣扎求生，徒剩焦黄枯萎的枝干，并无花叶，与左近万物繁荣的景象交相对照，越发显得萧索。众人经由磴道盘行萦曲，扪石而上。既而山绝路隔，继之以木栈，倚石排空，周环曲折，如蜀道之难跻攀。正所谓'黄埃散漫风萧索，云栈萦纡登剑阁，剑阁峥嵘而崔嵬！一夫当关漫漫道，万夫莫开重重途。'

　　众人再向前行，见石上斑斑点点，尽是深紫色的凝结成块的血迹，想来不久以前，这里曾经有一场惊心动魄的喋血厮杀。山中静悄悄的没有声息，偶尔孤鸟哀啼，反倒更添了几分寂静。

　　秦汪冉道，"我前些年，曾经来过这里一回。若是在四川遇到这等山，也不足为奇，但在江南，这等石山的确是少见的很。这座山险是极险了，却不甚大，若鹏举驻兵在此，咱们找到他应该颇是容易。"

　　可是山路已经近穷，不要说人，连个鬼影子都没有遇上，这牛头山竟然好象是一座空山！眼看前面出山的峡口已经可以望见，蒙慎行不由得嘟囔道，"我们过这座山，根本就没有人来管。要是金兵来，恐怕也就这么平平安安的过去了。"

　　蒙七突然手指上空，道，"爹，你看，那是什么？"
