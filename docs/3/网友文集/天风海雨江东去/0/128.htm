第一百二十八回

　　秦汪冉听到'舍妹'二字，不知为何，心中对面前这男子的反感居然立时散得一干二净。待到这男子邀他同饮时，若是平时，秦汪冉断然不会接受这等一面之交的邀请，可现下听到'小酌'二字，心里头竟然却是一百二十个愿意。

　　秦汪冉口中踌躇道，"这，萍水相逢，实在不好意思叨扰。"口气却颇松动。

　　那男子笑道，"天下人哪里有不是萍水相逢的，叨扰二字怎么用得上？只是舱中简陋，酒味寡薄，用以招待贵客，难免有些怠慢了。"

　　秦汪冉笑道，"兄台既这么说，秦某若再推辞，反倒显得有些做作了。如此只好恭敬不如从命了。"

　　那男子扬眉道，"这么说尊驾贵姓秦喽，小可却乃是复姓，上独下孤，单名一个鸿字。"

　　秦汪冉笑道，"孤鸿海上来，池潢不敢顾 。志存高远，不屑与凡鸟为伍，兄台的大名高雅的很啊！秦某却是双名，前汪后冉，汪乃是汪洋大海的汪，冉乃是冉冉东升的那一个冉。"

　　独孤鸿正在伸手撩起帘子，闻言回头异道，"秦兄莫不是江湖上人称冥天血龙的那一个？"

　　秦汪冉微笑道，"不敢正是区区。"

　　独孤鸿将帘子揭了起来，道，"想不到会同天下第一高手这般相遇，实在是小可的容幸，秦兄还请进舱。"

　　秦汪冉谢了，低头钻进船舱，抬眼看去，不由得大吃一惊。

　　秦汪冉怎么也想不到这平平凡凡的一条乌篷船的船舱里面，竟然会布置的别有洞天！

　　只见船舱四壁，张着刻丝弹墨的幔子，两侧窗口，挂了金丝藤红漆的竹帘，上面描着漓江山水。船板上的毯子厚厚的，软软的，走在上面，似乎脚也要陷了进去，毯子上却绣了有薜荔藤萝的花样，精致的很，叫人几乎不忍心踩在其上。舱门远端，摆了一张红木矮几，四只几脚，雕成了盘龙游凤的形状。矮几上摆了只绿玉果盘，里面纷乱摆了些紫橙橙的葡萄，倒好象葡萄也是玉雕的般。一只高脚银烛台摆在果盘边，烛光幽暗，再旁边有暗银酒壶立在旁边，两只黑沉沉的祁连老山玉刻就的夜光杯置在几角。矮几一侧，闲闲的坐了只玉鸭薰炉，燃的好象是瑞脑香。

　　 酒香与炉香混在了一处，闻着让人只想躺了下来。

　　 矮几前并没有椅子，只是厚厚的叠了数层垫子，那少女就正盘了一条腿斜坐在矮几旁边，右腕上正套了秦汪冉刚刚捡起过的那只镯子。满舱幽暗中，似乎最亮的就是她一双闪烁的美目，而这一双美目，正带了几分吃惊，几分羞涩的望在秦汪冉身上。秦汪冉一时间只觉得自己处在了一处自己绝不该在的所在，手脚都不知该放在何处。

　　幸好独孤鸿这时也进了船舱，笑道，"秦兄，这是小弟相依为命的妹子，也是单名，唤作一个眉字。"

　　独孤眉，独孤眉，独孤眉……秦汪冉在心中咀嚼着这诗一般的名字，知道自己今生今世，是再也忘不掉这三个字了。微微的一躬身，秦汪冉道，

　　"幸会。"
