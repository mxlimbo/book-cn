第一百十四回

　　韩世忠应道，"能走水路，当然好得很，可是这里前不着村，后不着店的到哪儿找船去？"

　　梁夫人在一旁一直只是倾听，并不说话，这时突然开口道，"没有船，咱们何不自己扎筏子？"

　　赵得胜听到，接道，"我以前倒是扎过筏子，扎筏子不要别的，一是大段木材，二是麻绳。木材咱们可以伐树，却不知道到哪里能够找到麻绳。"

　　李铁枪一直躺了在一边漫不经心的听着，这时插嘴道，"得胜，那麻绳是不是就用来把树干捆了在一起？"

　　赵得胜道，"正是。怎么，铁枪，你有办法找到麻绳？"

　　李铁枪道，"麻绳倒是没有，不过想把树干弄在一起，也未必就非得用麻绳吧。咱们要过的又不是汪洋大海，我想只要在树干上开些孔，再用木片塞了进去，两边压结实了，凑乎着过懒鸭滩应该可以。"

　　秦汪冉道，"好主意。咱们何不现在就动手制作筏子？"

　　河边多的是笔直的杨树，众人没有斧头，唯一的利器就是秦汪冉腰间的血龙宝刃和李铁枪夺来的樱枪，于是血龙宝刃当仁不让的又充作了砍树的家伙。这一日也算血龙宝刃霉星高照，先落魄到砍芦苇，现下又沦落到砍伐树木，这样发展下去，不知道血龙宝刃最后会惨到什么地步。若是宝物通灵，真不晓得它会想些什么在心里。

　　秦汪冉拖了李铁枪起身，两个人开始伐木。他们两人伐木，看起来倒也有趣。血龙宝刃虽然锋利无匹，但刀身极薄，若是不分青红皂白的猛砍，只怕砍不倒两颗树就要断折。只见秦汪冉围着树干一阵急转，血龙宝刃的刀锋只不过是切了在树干之上，这纯是利用了血龙宝刃的锋刃，却不妨事，与其说秦汪冉是在伐木，不如讲他是在割树。不过如此伐木，伐一棵树差不多要转二十几个圈子，换了别人，就晕也晕倒在那里了。

　　李铁枪用樱枪伐木则别有一功，那樱枪虽然长大，但实实在在的是用铁制的不过是头上半尺长短的一个枪头。李铁枪抖枪如花，'夺夺夺'声中只管向树干上刺去。妙在他使力极匀，枪头垂直的刺入树身，又垂直的拔出，决不至于弄折了枪头。李铁枪边出枪边绕树干而行，一圈下来，树身上已然是整整齐齐的一圈枪刺出的小孔，深入三寸，便是拿墨盒刁斗来量恐怕也未必有这么正准，看上去就好象给树干挂了一根项链相似。接着李铁枪猛然跃起，在半空中旋身出腿，踢在树干中段，树身晃了几晃，终于缓缓的倒了下来。

　　两人忙了有小半个时辰，小淮河沿岸倒下了三十余颗杨树，就是做再大的筏子也尽够了，两人这才停手。蒙慎行和蒙七这时过来帮忙，把伐倒的树木聚拢了来。许大马棒先只是坐在一边闲看，最后也自过来相助。三十余颗杨树平平的垒作上下两层，看上去已经是个筏子模样，现在就要看李铁枪的法子是否真能奏功了。

　　李铁枪如刚才伐树般炮制，以枪尖在树身上各处开了许多小孔，更让临近的树干上面的小孔正正对上。秦汪冉的血龙宝刃反正已经数次蒙尘，'虱子多了不怕咬'，也就不在乎再充当一回削木条的角色。刀花绽放间，一根根光滑笔直的木条出现在秦汪冉手中。蒙慎行的混元乾坤掌力，如斧如凿，硬生生的把一根根木条拍进了树干之中。这三人以后如果金盆洗手，退出江湖，倒是蛮可以联手去做木匠，生意想起来定然不会太坏。蒙七和许大马棒就一人一头，帮着抬起树干两端，以便蒙慎行下手。

　　顿饭功夫以后，一只极大而且极难看的筏子总算完工了。筏子上枝叶茂密，甚至还有几只鸟窝在上面。众人直起腰来仔细端详面前的杰作，忍不住相对菀尔。

　　这筏子虽然宽大，但无论如何是容不下近十匹马在上面的。蒙七灵机一动，道，"马好象是会游泳的，咱们何不让马拉筏子？"许大马棒闻言不由得笑道，"兄弟，马会游泳倒是不假，但它那一点游泳本领，在水中浮起身来将将就就，就再多驮了个人也怕要沉了下去，让它拉筏子是万万不能的。不过你这主意倒也不是完全没有用处，咱们可以把马匹拴了在筏子上，让它们自己跟着游来，到咱们上岸时候便还能有马可骑了。"

　　剩下的最后一件事是找撑筏子的撑杆。毛竹杆子是没有地方寻去了，李铁枪只好又弄倒了一棵树，秦汪冉的血龙宝刃只好又把树干劈开，削作了六根撑杆。

　　许大马棒将马套了在筏子前面，吁了马拖了筏子下水，筏子摩擦地面，吱拉作响，众人心都提到了嗓子眼上，生怕这费了多少心血气力制成的筏子未等下水便寿终正寝，李铁枪乃是始作俑者，比别人就更多了几分担心。总算这筏子争气，到底是入了水中，完完整整的居然还是一只筏子。

　　这时候终于万事俱备，李铁枪乐呵呵走到筏子前，一个大弯腰，道，"小子的摆渡生意今日头回开张，这就请各位客官上筏。"

　　梁夫人笑道，"我们可没有摆渡钱与你，你这亏本生意是做定了。"

　　众人玩笑声中上了筏子，蒙慎行，秦汪冉，李铁枪，许大马棒，蒙七，了凡各自持了一根撑杆，用力撑去，筏子离了岸，向河中心飘去。众人是顺流而下，所以几根撑杆除了偶尔矫正方向外，并没有什么太多的用武之地。几匹马扑腾腾的跟在后面凫水，心下颇是疑惑，为何放了好好的平坦大道不走，却要莫名其妙的逼它们在水里面瞎折腾。

　　虽然两岸没有猿声啼不住，众人的筏子还是走得极快，十数里地转眼即逝，懒鸭滩就在前面不远处了。

　　张本的千人马队，早已经到了懒鸭滩多时。张本处事小心多虑，所以才被派了驻守本寨。他到了懒鸭滩，第一件事是布置弓箭手，然后是分派马队于四处准备截杀，中间更伏下五十名精兵手持长矛狙击，最后派了百人去大路前端挖掘壕沟陷马坑。天罗地网已经布下，就等韩世忠诸人来投。
