第一百十九回

　　兀术多一刻也不肯再在黄天荡停留，连夜行军，至夜分时已经与与孛堇太一会兵一处。十余万金兵安营扎寨，埋锅造饭，火光连绵数里，蔚然壮观，夜色之下，就如一条硕大无朋的金蛇般蜿蜒躺在了大江之侧。

　　兀术大帐之中，人头济济，坐满了金军将领。兀术正襟危坐大帐当中，韩常侍立于侧，黑峰，柳如是乃是客卿身份，便挨了兀术而坐，其他各人都依官位高低，各自入座。

　　待到众将到齐，兀术开口问道，"孛堇太一将军，近日我被困黄天荡，消息断绝，不知各处战局已经到了何等地步。你这里有什么新消息没有？捡要紧的讲些与我听。"

　　孛堇太一起身回道，"回王爷，各处来的消息甚多，其中极重要的也有数条。第一则是建康被一个叫做岳飞的宋将夺去，幸好前时守兵，大都被我派兵接引救回，但还是有近千兵丁失了影踪，生死不知，到镇江的咽喉要道，亦就此断绝。第二条乃是挞懒将军处来的军报，其中道，'现我军略地山东，已近全功。唯向北而进时，围攻楚州，宋守将赵立，异常顽强，抵死而战，城池一时不得而下。闻南朝已委派刘光世，调集淮南诸镇，往援楚州。若其援到，恐更难得手。请乘便夹击。'但我在此接应王爷，实不敢分兵去援。第三条是娄室元帅在陕西，与张浚相持不下，城池得而复失，失而复得的有好几处。现下张浚更檄文召集熙河经略刘锡，秦凤经略孙偓，泾原经略刘锜，环庆经略赵哲，并及统制吴玠，共五路大兵，号称四十万人，马七万匹。一旦集结完毕，张浚便欲以此与娄室元帅决一大战。第四条乃是圣上有钦命给王爷，上嘱王爷，若一旦脱困，便即速引军助挞懒将军和娄室元帅征战克敌。"

　　兀术听到'岳飞'二字时，眉头不由得皱了一皱，心中大骂，"姓岳的这厮，用兵怎敢这等猖狂大胆！他料我不肯再行牛头山险地，居然给我唱起了空城计！实是欺我太甚。建康既然失守，要去镇江，便不是一件轻而易举的事了。但镇江近在眼前，不拿下来委实有些可惜。"待得听到挞懒之事时，兀术先是心中一宽，心道，"山东终于得手了。"但随即听到楚州消息，却甚诧异，楚州城并不是什么了不得的大城，城防也定然不会太过坚固，挞懒是大金国里数一数二的良将，怎么会一时取它不下？不过这是小事，无关大局，兀术倒也并不把这事放在心上。

　　待到听到娄室之事时，兀术脸色凝重了起来，知道这是牵动全局的大事，心中计算道，"我若与娄室合兵，当有近二十万甲兵，挞懒的六万兵马则可用为接应预备。以二十万对四十万，要求全胜，可真得好好想一想了。"

　　孛堇太一陈诉已毕，兀术闭上双目，陷入了沉思之中。

　　"镇江看起来只好放弃了，虽然我众他寡，但建康有险可依，姓岳的这厮又鬼的很，想起来没有数十日，决然拿不下来，再加上征途往返，弄不好会陷在里面两三个月。而在这当口，两三个月实在是耽搁不起。牛头山和建康这两笔账，留到以后再慢慢与他算去。至于楚州，挞懒既已经围了城，与他一起夹击便没有多大意思，攻城时兵力无法展开，人马就再多，也派不上什么大用场。但楚州已经被困多日，想来城中存粮就还有也定是有限的很。楚州饷道，必经六合。若我去六合县屯兵，设南北两屯，断他粮道，待楚州粮尽后，不难一举而摧。过了楚州，一路尽在我大金国控制之下，那时就只需分少许兵马，就可以运还辎重。这一段屯兵的时间，正好又可以休整兵马。然后我当引兵趋向陕西，与娄室会师，再同宋军决一死战。"

　　心中思虑得定，兀术猛然睁开双眼，眸子中精光四射，沉声道，"我现在急欲北归，运还辎重，赵立若许我假道，我也没工夫去击他。文牍何在！立刻替我修一封文书，要赵立献城假道，否则我当往去夹攻。写毕后以六百里加急送到楚州去。"

　　兀术接着站了起来，大声道，"众将听命！"
