临界的闲适

　　下班很早，看了一天电脑，眼睛累得很，抬头看天，竟然有点不能适应明亮的蓝。忽然想起很久没有在傍晚散步，便一个人逛进了白鹭洲公园。

　　这个公园在南京城南，不无例外是古人的私宅，安静而优雅。公园里人很少，这个时候，柴米夫妻们正忙于锅碗瓢盆，公园里只有几个悠闲的老人，零零散散的走在湖畔，林间。

　　有洲必有水，园内有一汪清水，波光滟潋。我坐在桥栏上，尽收眼底的是满园苍翠，一湖夕阳。由于安静，我能听到水流的声音，舒缓而从容；我也能听到植物的枝叶摇曳的声音，轻柔而精致。这个园子里竟似有无数甜美的精灵在飞舞，淘气地拨弄出天籁，却又怕惊着我，而故意放慢了脚步，压低了声音。

　　天空的蓝由明亮转灰，夕阳的红不再耀眼，由厚实渐渐稀释。难以想象自然的力量，可以将那么深沉的红和那么纯粹的蓝过度得如此自然，毫不矫情毫不生硬，彷佛他们本来就是一体的。

　　晚风带着初秋的微凉，却很温柔，生怕惊了这傍晚的静谧。在这繁华都市的一角，所有的生灵都用柔软的触须和思惟在交流，彷佛婴儿的手指轻触我的眉心，我有醺然的震撼。

　　夜色悄然而至，公园里的人渐渐多了起来。离开公园，都市已经霓虹闪烁，我心里却藏了个秘密，只有那些小精灵才能会心的秘密——这临界的闲适让我忍不住想叹息，想高歌，却也怕惊扰了这凡尘的热闹。
