别人的生活

（一）

　　写一个你不了解的人是困难的。他说。

　　并且，这个人生活的环境你一无所知，你从来没有来过武汉。他继续警告我。

　　是的。我说，我从来没有到过武汉。

　　你想写我？他开始引诱我。

　　想。我上钩了。

　　这意味着你必须先了解我。他循循善诱。

　　以你的方式？我问。

　　是的，以我的方式。

（二）

　　让我想想我们如何开始。

　　你没有来过武汉，那我们就从南京讲起吧，你在南京是吗？

　　是的。

　　你一定知道梅花山。

　　嗯，每年我们都去看梅花，这几年那里还举办了大型的梅花节盛典。

　　你知不知道山上有个洞？

　　不知道，我老实地回答。但是我相信每座山上都有山洞，只是有的出名有的不出名。

　　对，我说的就是那种不出名的山洞。

　　好，我脑子里已经有那个山洞了，在梅花山上。

　　真是个聪明的女孩。

　　那个洞在背阴的一面，不容易发现，因为洞的周围没有象样的路，却有很多树。

　　那你怎么会知道那个洞的？我插嘴。

　　我想找个别人发现不了的地方，所以就专找没有路的地方钻。

　　你在逃亡？

　　不，我跟一个女孩在一起，确切地说，是偷情。

　　你真幽默，我笑。

　　那个女孩跟你现在差不多大。

　　她漂亮吗？

　　漂亮，不过也许没有你漂亮。我微笑起来，像当年那个漂亮的女孩一样心满意足。

　　你们在山洞里干什么？我问，很傻。

　　接吻，抚摸。但是没有做爱。

　　哦。我说。也许是春天，女孩柔软地伏在他怀里，长发遮住了半边脸。洞外的树沙沙作响，是风吹过。

　　你想不想知道我们在那里待了多久？他打断我的想象。

　　我没有作声。用爱因斯坦的理论想就知道多久都不算久。

　　两天一夜。他说。

　　你们带干粮？

　　没有。

　　莫文蔚说：爱情真伟大。

　　后来呢？

　　我离开南京，回武汉了。

　　你能不能告诉我你们为什么要在一个山洞里呆那么久？

　　因为相爱，而且无处可去。

　　哦。

　　我毕业时为了她去南京，在光华门外租了一个平房，和她同居。后来她爸找人把我打了一顿，还把房子里的东西都砸了。

　　她跟你去武汉了？

　　没有。

　　那是个冬天。他说。

（三）

　　这么说，你对南京还是很熟悉的。

　　不。我在一个小公司上班，其余时间就跟玲待在租来的屋子里。

　　很相爱？

　　是的，我为她背井离乡。

　　她父母为什么不让你们在一起？

　　因为我穷。我的月薪只有八百，我父母都是农民。

　　她是南京人？

　　是的，她爸爸是个小官，但是很势利。

　　她可以跟你同居，为什么不能跟你私奔回武汉？

　　我回答不了，因为如果是我，什么都可以做到。

　　她爸爸找人砸了你的家，你就放弃了？

　　没有，直到她亲口对我说："对不起，我们在一起不会幸福的"之后，我才放弃。

　　后来呢？

　　我回了武汉，继续念研究生，两年后提前毕业考进了中行，从事技术工作。半年后跳到一家外资企业，月薪翻了几十倍。我同时在武汉好几个大学任教。

　　你只是想证明，你不会一直穷下去是吗？

　　开始是。

　　现在呢？

　　我发现钱对我再也没有意义了。

　　那个女孩呢？她怎么样了？

　　嫁人了，嫁给一个不大不小的官。他似乎笑了笑，带点嘲弄。

　　幸福吗？

　　她老公前年因为经济问题被抓起来了。

　　啊，她很不幸。

　　我倒觉得那男人抓起来对她而言是件好事。

　　为什么？

　　他打她，经常性。

　　你怎么知道？

　　那边的同学说的。我知道玲很困难，让一个同学给她送了两万块钱，说是同学们凑的，没说是我给的。

　　她父母呢？

　　知道我现在很有钱，托人捎口信说如果我愿意，让女儿跟那男的离婚，嫁给我。

　　你答应了？

　　没有。

　　你不爱她？

　　爱。

　　那为什么？

　　你知不知道很多感情是不能回头的。

　　你们再也没有见过面？

　　见过一次。我因为业务的原因经常飞南京。有一次，同学瞒着我安排她见我，想撮合我们。

　　在哪里？

　　我当时住在金陵饭店。都没有认出她来。她很――憔悴。

　　你是因为这个原因不想要她的吗？

　　不是，我很心疼她，但是回不到从前了。

　　你们――你们――

　　你是想问我们有没有发生性行为吧？

　　我被噎住了，脸发烫。

　　没有。他说，轻描淡写。

（四）

　　你之后爱上别人了吗？

　　也许有也许没有，我不知道。但我没有那样的热情了，为一个人一无所有背井离乡。

　　你会结婚吗？

　　会。我身边有三个女人，一个医生，一个律师还有一个报社的编辑。我想我会娶她们中的一个，但是还没拿定主意。

　　你真是个幸福的男人，居然有女人等着你挑，而且听起来档次都比较不错。我调侃他。

　　可我宁愿有个女人是我愿意不顾一切去争取的。

　　像玲那样的？

　　他没有回答。

　　如果让你选择，你愿意在南京逃亡似地生活还是在武汉如此优越舒适地生活？

　　人生是没法假设的。如果当时我留在南京，她也坚持下来了，我也无法肯定我们就能幸福。

　　你已经对爱情失去信心了，你不相信爱情了！

　　是不相信爱可以无所凭借。

　　你是说钱很重要。

　　并不尽然，但也是个因素。

　　没有激情的生活是死水一潭。

　　激情也会归于平淡的，有些问题只是考虑得早或迟的问题。

　　假设你是玲的父亲，你会接受当年的你吗？

　　不好说。至少我不会找人打这个青年一顿，砸了他的家。他笑了，全无芥蒂。

　　你恨他们吗？

　　开始恨过，现在只是同情，如果他们当初跟我现在这么有钱，就不会做得那么绝了。

　　你很豁达。

　　只是真正走出了，能以旁观者的眼光来看了。

　　你说宁愿有一个女人等你争取是不是说明你没有激情只是因为没有碰到你爱的人？

　　除非亲自遭遇了你所说的爱情，这之前我无法回答。

　　我要把你的故事写下来。

　　有什么意义呢？

　　生活又有多大意义呢？就是这样进行，经过，感受而已。我不企图挖掘什么，更不渲染什么。

　　那么，去写吧。

　　我已经写完了。

　　你打算用什么题目？

　　不知道。我常常为文章的题目困惑，我总是写完后草草加上去，有时辞不达意。

　　无题？

　　适用我的每一篇文章，不过我从来没有用过。

　　那就叫别人的生活吧。
