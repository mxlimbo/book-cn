何以解忧

　　人无远虑，必有近忧，然则，何以解忧？古人云：唯有杜康。言下之意，一醉解千愁，却也有一说，抽刀断水水更流，举杯消愁愁更愁。可见忧愁一事，最难排解。

　　不善饮酒，每每举杯，沾唇即止，非为矜持，实不胜酒力，忧愁来袭，便以茶代酒，排遣忧思。

　　找一僻静处所，以纯水烹茶，看茶叶如秋花翻飞沉淀，尽涤杂乱思绪，渐趋平和。入口微苦，而细品甘醇，想人生若此，品得忧愁亦是一快，便于茶中求宁静，当可解忧。

　　茶之解忧，取其静。

　　忧愁两字都关心事，所谓心底无私天地宽，若从宽处着点，读书亦是解忧之道。综览古今，学贯中西，自然胸有丘壑，虚怀若谷。所谓得意说儒家，失意说道家，此说虽有偏颇，但不妨一试。本土道学讲逍遥，若得逍遥，自然去忧。故而忧愁来袭，不妨一卷在手，悟得大道自然，何忧之有？

　　茶或书都是风雅之物，若去雅求俗，亦可去忧。

　　身处闹市，忧心重重，君欲解忧，不妨畅怀。呼朋唤友，小聚鲸吞，说欲说之事，骂想骂之人，畅快淋漓，又能果腹，也算畅快。

　　又或私处一室，披头散发，恣意哭笑，至口渴喉干，精疲力尽，畅饮一番， 酣眠一宿，发泄殆尽，自然平和。

　　解忧之道，各有千秋。或远游，或听歌，或闲逛，或购物，或倾诉，但取所需，各得其所。
