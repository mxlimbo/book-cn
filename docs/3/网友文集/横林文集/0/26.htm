Windy

　　Windy跟我的交情已经有十几年了，我们是初中同学。后来我考上了一所比她好的高中，就不能经常见面了。

　　由于高中比她的好，我就比她放松，她就比我用功，所以，考大学时她考了一所比我好的大学。

　　我们一直通信。她的字漂亮但是很草。

　　她曾经坐了一夜火车到我的大学来看我，由于不适应北方的天气，脸上都退了皮才回到江南。不过，从她在黄河边拍的照片上却看不出那份狼狈。她乐颠颠地把所有照片都洗了两张，寄给我一份。

　　后来，她被那所比我好的大学保送了研究生，我就选择了她所在的城市工作。

　　虽然在一个城市，见面却少，又不通信，反而没有我也在念书时那么热络。但是，心里总想着，毕竟距离近了，也就不计较别的了。

　　过了一年，中秋。她没有说要跟我一起过，我也自找节目了。跟男朋友在烛光里情意融融地把西餐吃到一半，接到她的电话，叫我到某某饭店去，说要让我见一个人。

　　我去了。

　　包间里的灯光很温馨，她的脸淡淡地红，很美。她要我见的是个男孩，很清秀。她说想跳舞，我们就去跳舞了。

　　我滑进舞池的时候，看到那个男孩牵着她的手坐在角落里，她的笑容很甜蜜。

　　我不喜欢那个男孩！我坦白地告诉她。她讶异地看着我，我不知道自己是怎么了，那一晚，我没有注意月亮，但是我想哭。

　　还是跟以前一样生活，很少见面，很少联系。

　　她决定考GRE，我没有反对，淡淡地说："我来了，你又要走。"

　　一年后，她接到了好几所大学的全奖通知。她第一个征询我的意见，我沉默。她终于选择了跟那个男孩在同一个城市的那所学校，尽管并不适合她，我没有说话。

　　一个午夜，两点，她打电话给我，哭了很久。想见我。

　　我去了。

　　她说不能自拔。

　　我没有流泪，只是残酷地告诉她："他不适合你！"他是个自我的人，没有柔情，没有爱意，他有的只是跟他前途息息相关的策划。

　　他们闹过一次分手。她先让步了。

　　她来我这里的时候，总是一个人。她说那个男孩怕看见我，我能看穿他，让他无处躲避。Windy忽然看着我，忧伤地说："你跟他谈谈好吗？他说你像离开他的那个女孩，那个女孩曾经是最懂他的。"我看着她很久很久，她已不是原来的她，我想抱着她哭，可我哭不出来。

　　我做了，按照她的说法跟那个男孩谈了。尽管这是徒劳，对他们之间的关系而言。我知道，她爱他，不可更改。

　　他们一起去美国了。临行我避开了跟她的见面。

　　我呼吸这个城市的空气，我对自己说："这是生活的本来面目。"

　　她偶而给我发email，很短。我回复，说着一些无关痛痒的话。

　　我的白天是她的黑夜，时差刚好十二个小时。我计算过。

　　那天，北京时间上午十点，她打来电话，没话找话的说着她的情况，我轻轻叹气，她终于不可遏制的痛哭。

　　我说："你该知道这结果的。"

　　"可是，我真的已经习惯了。"她无助的说，我想象她徒劳想抓住什么的眼神。

　　"如果爱成为习惯，你不会哭；如果伤心成为习惯，你又何必哭？"我说。

　　"我每次都想得很清楚的，我很明白的，可是我却无能为力。"

　　我第一次为她的爱流了泪，我忽然发现那么多次我想哭哭不出的原因都只是因为——无能为力。

　　这一次，我哭了，可我同样无能为力……
