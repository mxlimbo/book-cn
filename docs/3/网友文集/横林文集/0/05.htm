雨夜听筝

　　午夜。

　　忽然下雨。

　　听到雨打在窗上的声音。

　　我没有开灯，把音量调到最低，欣赏古筝。

　　每个音符都象落在心头的雨滴，轻轻漾开去，蔓延成一种意境。

　　闭起眼，似有淡淡的檀香在缭绕，袅袅的弥漫一室的温柔，渐渐散开去，散开去……散到林间，几枝修竹懒散的斜在窗前，雨滴顺竹叶滴落窗台。

　　屋里定有云鬓微乱的女子轻抚古筝，有一搭，没一搭；她的眼神定已穿过雨丝，象檀香一样飘入林中便化为雨雾，笼湿润的心事。

　　不敢掀起窗帘，我知道外面没有修竹，只有蒙在雨雾里的幽幽路灯，带城市颓废的冷漠……
