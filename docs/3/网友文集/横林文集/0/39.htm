拥你入眠

（一）

　　房雨的每个细胞都在反复回味刚才的兴奋。小波的脸艳若桃李，她咬着他耳朵无助喘息的那一刻，房雨几乎要晕厥，迷乱地唤着她的名字，彷佛墯入地狱，又好象飞上天堂。

　　尤物。小波是他的尤物。他轻轻地叹息。转过目光，她在他身边睡得那般安静，好象什么也没有发生，脸在幽幽的灯光里有点虚幻，睫毛的阴影在房雨看来带着某种冷漠不可接近的意味。

　　他忽然觉到一阵空虚，这感觉使他迫切地想把小波纳入怀里，紧紧搂着她，在她的气息里入睡。可是他没有这么做。

　　小波说:："房雨，你不要这样抱着我，我不舒服。"这是第一次，他们做爱后，小波对他说的。他放开她，她就睡着了。可是房雨睡不着，有什么不对？难道相爱的人不该拥在一起入眠吗？他想，也许那天是个例外。

　　第二次，房雨搂住小波的时候，她没有反抗，但是显得很不安，一会儿扭一下身子，一会儿发出一点怪怪的声音，像梦呓又像抱怨，房雨放开她，她立刻又安静了。

　　第三次，房雨搂住她时，她一动不动，柔软光滑的躯体在他怀里乖顺得像只小猫，他感觉到一种充实的幸福。没多久，她忽然甩开他，腾地坐了起来，有点歇斯底里地冲他嚷："我跟你说过，不要抱着我睡觉，我睡不着！"房雨惊呆了，有一千个理由觉得小波的做法不妥，却一句话也说不出。

　　而小波在做爱的时候依旧那般狂野，使房雨相信她对他的爱深如大海。

　　也许只是习惯吧。房雨想。决定接纳小波的这个习惯。

　　在她睡熟的时候，他就这样凝视她，在浓浓的爱意中感受空虚。

（二）

　　小波不吃早餐，但是喝牛奶。这大概就是她肤如凝脂的原因，房雨想。

　　小波一边喝牛奶一边偏着头对房雨说："Tommy，我今晚不过来了，小雪出差回来。"小雪是她的室友。房雨沉默了一会，问："你什么时候搬过来这里住？"

　　小波笑："我们现在这样不好吗？"

　　"不好，我要每天看到你。"他不理解小波为何始终不愿意搬来与他同住，而非要每个月花一笔钱在一个离公司更远的地方租房子。

　　小波笑笑，一边穿外套一边吻他，说："我走了，回头给你电话。"

　　门砰地关上了，房雨又有那种空虚的感觉了，好象怎么也抓不住小波，她总是轻而易举就让他在幸福边缘落空。

　　或者这是小波的方式。房雨想。

　　是的，小波的很多方式都是他理解不了的，比如再冷的冬天她都要在水杯里加冰，她喝咖啡不加糖，她睡觉一定要开一盏灯，她从不把房雨介绍给她的朋友，也从不接受他的礼物。

　　小波总是在他们的爱情里设置距离，有时不落痕迹，有时用心良苦。

（三）

　　房雨打电话给钟平："我想向小波求婚。"

　　"你头脑发热啊？"钟平带几分调侃。

　　房雨越来越不明白，为什么人们都那么反感结婚，究竟是人们怎么了？还是婚姻怎么了？结婚有什么不好？

　　"结婚有什么好？"小波说。

　　"我爱你，你也爱我，我们就应该生活在一起，这不需要理由。"房雨的声音有点高，他发觉自己被一种古怪的情绪控制了太久。

　　"我们现在不就生活在一起吗？"小波说，往纯水里又加了两块冰。

　　"你在恐惧什么？爱情？还是婚姻？"房雨忍无可忍地追问。

　　小波看着他，看了很久，久得房雨足以平静自己的情绪，才慢慢地说："我不想改变现状，就这样。我也不认为结婚可以使我们更幸福。"

　　可是为什么会这样呢？房雨苦恼地想。

（四）

　　房雨是个传统的男人。

　　他念书时爱过一个女生，但是因为分配而分手了。后来就认识了小波。他喜欢她的冷静，她的狂野，甚至是某些不可理喻的冷漠。

　　他想娶她，并不是想困住她，而是想给她最完整的爱，从情感到责任。他认为婚姻是对爱情最完美的诠释，可是小波不接受。

　　他知道自己是小波唯一的男人，他也知道小波为他放弃了出国的机会，可是他不能理解为什么他们之间总有一些阴影，来自不同的方式和思维。

　　小波安静地躺在他身边，呼吸匀称。他又想搂住她了，事实上这个念头从来都在他体内疯长，他知道小波会拒绝，可他决定不放弃。

　　小波在他怀里不安了，他不放手，抱着她，把她帖在自己胸口。他分明感觉她的身体开始僵硬，带着她固有的执拗和倔强。他不放手。

　　小波开始挣扎，嘴里不满地抱怨："房雨，你让不让我睡觉？你抱着我我难过！"一层淡淡的苦涩泛上心头，房雨觉得胸腔里一阵酸楚，眼眶发热。

　　小波气愤地坐在床上，看着房雨，看到他眼睛湿润，她不动声色。

（五）

　　离开小波已经半年。

　　一个人睡在宽大的双人床上，房雨仍会想起小波，不是翻云覆雨的快乐，而是他把小波静静搂在胸前的感觉，那时她温柔，乖顺，让他心碎。

（六）

　　房雨在随意按遥控转换电台的时候看到了一则紧急通告，因施工原因，洛水小区的电缆遭到破坏，居民用电全被切断。

　　房雨心里一凛，想起小波就住在那个区。小波睡觉都要开着灯，她是如此害怕黑暗，如果小区停电，而小雪又刚好出差……

　　房雨拨小波的电话，没人接，手机也关了，他心里发寒，似乎看到她缩成译一团，泪流满面。

　　他只想见到她！这个念头如此迫切，他连睡衣都没有换，就跑下了楼。

　　整个洛水小区一片黑暗。

　　小波没有换锁，他轻而易举就打开了门。房间里一片漆黑。他点亮打火机，轻轻唤着小波的名字。

　　一阵压抑的啜泣渐渐变成了肆无忌惮的痛哭。

　　房雨看到小波蜷缩在床的角落里，可怜楚楚地看着他。他一阵心疼，把她搂进怀里。

　　她的身体那般柔软，房雨发现她瘦了，相识三年，第一次感觉她如此脆弱而可怜。她不让他走，要他留下来。

　　黑暗中，房雨轻轻地碰了一下小波，那个温暖的馨香的小小躯体顺势滚入他怀里。

　　那一晚，房雨彻夜未眠，拥着她的柔软也拥着她的温顺。而小波在他怀里睡得安静而甜蜜。
