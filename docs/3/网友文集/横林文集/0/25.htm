阿　宝

　　公司内部网出现了一个人气很旺的玩家，大名丁老怪。从丁老怪满口粗话，一付流氓样里可以推断出这是个男孩子，也许是带着厚厚镜片的程序员，也许是留着长长头发的美编。

　　丁老怪经常在内部网的BBS上灌水掐架，进了聊天室说黄段子泡女孩子，一副死皮赖脸相。

　　有一次他在OUTLOOK里发了一个邮件给全公司的头号美人，要约她吃饭。美人找到公司的电脑高手，娇滴滴地说："我不管，你给我把他找出来啦！"

　　高手推了推镜片，霹霹啪啪敲着键盘，总算从千百个人的IP中把这个色胆包天的家伙揪了出来，居然是阿宝。

　　阿宝不漂亮，胖呼呼的脸上长满小痘痘， 皮肤黑，笑起来牙齿一点不整齐，在美女如云的公司实在是很不起眼。但是她刚到公司就成了焦点人物，因为她那别出心裁的打扮。

　　阿宝敢于穿桃红上衣配宝蓝裙子；十个手指甲一半涂成鲜红，一半涂得碧绿；她用几十种颜色的皮筋把头发捆得结结实实，两个辫子就像牛角一样顶在脑袋上。

　　被识破女儿身的阿宝上内部网的兴趣大减，很快居然在内部网自杀了。

　　到了夏天，阿宝不仅鲜艳，而且性感，胸口低得让人心猿意马。

　　我生日，阿宝孤身一人参加我们的聚会，结束时，特地挤过人群跟几个比较有型的男孩握了手，包括她第一次见到的我的男友，一边握手，一边念叨"帅哥啊。"第二天回公司，阿宝主动问我："你男朋友说我是花痴了吗？"我哭笑不得，她却哈哈大笑。

　　阿宝忽然就有了个男朋友，好象就是从楼上掉下来的。每到休息时间就跑到我们楼层，坐在阿宝身边窃窃私语，亲密得周围的人都不好意思不回避。

　　听说这还是一段网恋。阿宝在南京人气最旺的西祠网站招摇撞骗的时候，碰到了这个男扮女装的家伙，便像模象样调戏起对方。

　　见面的时候，真相大白，两人居然还在同一个公司，于是立刻开始了这段感情。考虑到如此浪漫的事情发生在公司也是职员的骄傲，大家对阿宝及其男友的种种举止也就从咋舌到欣赏了。

　　阿宝闪电式地谈完恋爱，就开始准备结婚了。我们一致认为，结婚这么庸俗的事不该发生在如此另类前卫年仅二十二的阿宝身上，但是她执意要嫁人，并振振有辞曰过了这村就没这寨了。

　　于是阿宝就结婚了。

　　结婚后的阿宝改变发型，长发披肩起来，还穿起了深色衣服，不露胳膊不露腿，我们起哄："哎，你老公也太不解风情了。"阿宝轻轻一笑，居然连牙齿也没有露。

　　更可怪的是，阿宝进出公司再也不打车了，居然买了张月票。有一次我在超市看到她，她正在跟老公嘀咕："还印花商品呢，比我们家楼下那个熟食店的香肠还贵一块八！"

　　确定自己没有认错人后，我忽然有点怀念起那个丁老怪。
