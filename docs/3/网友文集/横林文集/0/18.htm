美人楼轶事

　　念书时，我们六号楼被誉为"美人楼"，因为这里住着一朵校花，四朵系花，班花数目则无从考究。

　　住在"美人楼"的好处自然是可以日日饱餐秀色。美人一多，故事也多，虽然陋室没有电视，却也能隔三岔五看些偶像情景剧。

　　仲夏夜，空气里漂浮着洗发水和香水的味道，整个美人楼都蠢蠢欲动，红唇乌发撩得夏夜气喘吁吁，莺歌艳舞引得楼前的帅哥望眼欲穿。然后便看美人们一个个飘然出楼，给她的王子嫣然一笑，王子揽起纤纤细腰，双双飘入教学楼图书馆或飘出校园。这是一道夜夜美丽的风景，倒也不希罕。

　　我喜欢看的是那些苦着脸的帅哥，大多是惹得红颜一怒后前来负荆请罪的。所谓最难消受美人恩，若是把"恩"字改作"怒"字或者更恰当。帅哥先是陪着笑脸在阿姨那里一个个电话往美人房间打，打得阿姨烦了，便只有坐在花坛边抽烟，边抽边瞅着那个亮着灯光的小窗子，虽有罗密欧的一腔激情，可惜窗口迟迟不见朱丽叶深情应和。殊不知，此时正是美人考验王子的最佳时刻，若是帅哥此时垂头丧气地撤退，那后果不堪设想，日后再想要美人开恩，恐怕更是难于上青天了。聪明的帅哥此时会拦住一个出楼或进楼的单身女子，目光恳切，语气急切的说："同学，你帮帮我好吗？"只要表情控制得当，语气处理到位，那么，这个女生十有八九会去敲响美人的门："ＸＸＸ，楼下有人找！"此时，美人大多觉得面子挣够了，便千呼万唤始出来，姗姗走到帅哥跟前，冷冷问："你找我什么事？"接下来，我虽听不清台词了，结果却大多是喜剧。

　　当然也有些郭芙式的美人，非要让帅哥在楼前等到六号楼熄灯，统计完几千个行人几十辆轿车的往来后才肯露面的，大多只是在窗口探一下头，扔下金子般的几个字："你先回去吧，有事明天说。"潜台词便是"我原谅你了。"此时帅哥欢呼雀跃，百米冲刺回男生楼，刚好可以赶上熄灯关门（学校考虑比较周到，男生楼总比女生楼晚熄灯五分钟，因此成全了很多帅哥的苦肉计）。

　　这些戏路大多耳熟能详，却无新意。要说吸引人的却还是那些"我猜中了前头，却猜不到结局"的手笔

　　那时，帅哥已经用尽方法把美人骗下了楼，美人不无例外冷冷地问："找我干什么？"帅哥不开口，痴痴地看着美人，目光专注，眼神痛苦，看了几个世纪那么久（这是一种修辞手法），美人忽抬头幽幽地看着帅哥，吹气如兰道："你还看着我干吗？你这样看着我，也不能减轻我的痛苦！"顿时，整个世界都因这句美丽的台词安静了下来，我只恨自己没有带笔记本，只好拼命默记背诵这经典。更不能料到的是，帅哥忽然冷冷一笑："既然如此，我无话可说！"无话可说的帅哥果然不说话了，转身大踏步离开，放开喉咙大声唱道："来也匆匆，去也匆匆，恨不能相逢；爱也匆匆，恨也匆匆，一切都成空。"余音袅袅里，我大为叹服，显然这是以诗情画意与意味深长为风格的代表作。

　　而我印象最深的却是以壮烈见长的一出。站在我的寝室窗口，可以看到整个二号男生楼。那天我就是坐在窗口，看到男生楼顶楼的露台上站着一个人，姿势很悲壮，大有："别过来，过来我就跳下去！" 的架式。我近视，看不清他的表情，但是听到就在窗外不远处有个清脆的女声在喊："你跳啊，你干吗不跳？"我大惊失色，赶紧戴上眼镜趴上窗台看发出这个画外音的主角――美人，果然是美人，长发披肩，红唇如血，神态自若，体态婀娜。围观的人越来越多，那个壮烈的男主人公却忽然转身离开了栏杆，身影消失在我视线外，围观的人立刻发出一阵失望的嘘声，宣告着这场别出心裁的苦肉计的失败。

　　住了四年美人楼，看了无数出青春剧，无论夸张也好，搞笑也好，真真切切，实实在在，笑过以后留下些许的感伤，感伤这易逝的青春，感伤这再不能回头的大学时光。
