算　帐

　　我跟老公从大学时代起就养成一个好习惯，但凡支出一定入帐，这六七年倒也记了满满三本帐了。

　　周六老公心血来潮，说要计算一下工作以来的总支出。我立刻端出计算器和笔，老公咳嗽一声，表情严肃地开始了计算工作。

　　我一边啃苹果一遍听着老公按计算器的声音，时不时提醒一声："你轻点，当心手指按破了。"老公看我吃吃地笑，脸色更加严肃，声音低沉地说："过来！你报数字，我累加。"

　　我听话地拿过帐本开始协作。才报几个数字，我就明白了老公脸色发黑的原因。说来也怪，我俩的饭都是单位管的，但每天的支出都要三位数以上。不累加不知道，一合算每个月的支出都要到四五千的样子。

　　说实在的，我有素面朝天的好习惯，对于那些时尚杂志上的化妆品服装从来不感兴趣；而老公从来不必买烟买酒，这方面的支出等于零。按常规，我们这样的男人和女人搭伙过日子实在是应该只有收入没有支出才合理，可现实情况是我们两个算帐算得脸色铁青，手脚冰冷。

　　随着时间的流逝和数字的增长，我渐渐有点麻木了，嘿嘿笑着说："看你按计算器那样，好象跟它有仇似的，这钱可不是它花的。"老公抬头看了我一眼，这个帅哥算帐算得眼睛都发红了，我着实有点舍不得起来，非常诚恳地问："你没什么吧？"他叹了口气，非常满足地说："我很幸福，这些钱竟然不是我们欠别人的。"这幸福迅速感染了我，怎么想都觉得那钱花得有理。

　　三个小时后，老公端端正正地在最后写上了一个可怕的六位数。我精疲力尽地往沙发上一躺，觉得大功告成，而老公还在把本子翻来翻去琢磨着什么。果然是经济学专业的高材生，不久他就把曲线图给画了出来，并且神情庄重地总结陈词道："从曲线图来看，我们的支出并无一定的规律，但是月平均保持在四千至五千之间。其中，以2000年1月支出最高，为一万两千多，而支出最少的一个月是1999年6月，总支出两千五百多，括号，此月我出差十六天……"

　　他的声音慢慢地模糊起来，我很快就在沙发上睡了过去。醒过来的时候，老公还在翻帐本，看来我睡得不久，便皱着眉头诉苦："就刚才那一会我好象做了个恶梦，心情别扭着呢，就想不起来是什么梦了。"他笑道："肯定是算帐算的。"

　　"你还在看什么？"我凑过去。老公的脸色已经恢复了血色，还带着一种温柔平静的笑意："看看帐本好象回到了每一个走过的日子。你看，这一天是我的生日，你给我买了一件自由鸟的外套，我们还请了同事一起在紫琅阁吃了晚饭，又去唱歌。"

　　我把头靠在他的肩膀上，指着另一页的一个数字说："这一天我们去领结婚证了，你看看你，那天什么礼物也没有买给我。""我为什么要买礼物给你？我整个人都给你了嘛。"老公大言不惭道。

　　又翻过一页，那上面清晰地记录着我给外婆买的一套衣服。那时，外婆正病重住在急救病房，我匆匆回去了几个小时，那天刚好降温，我怕外婆冷，便在医院附近的商场买了一套保暖内衣放在外婆的病床前，还没有等外婆醒来就回了南京，谁曾料那是我此生最后一次为她买东西呢？在整理遗物的时候，姨妈告诉我，外婆最爱我买的那套内衣，竟自穿了离开这个世界。而此时，她又在哪里？我买的衣服还能给她带来温暖带去我的思念吗？

　　老公轻轻擦去我的眼泪，又把帐本翻过另一页。那是我们出去旅游的一笔记录，虽然只有简单的一个数字，但是，那些湖光山色却洋洋洒洒的逼入回忆来。

　　在所有的支出中有一部份不少的数字是老公每次回我家时给我父母的。这一笔笔记录从五百到五千，从不间断。老公从来不像别的女婿一样，提着礼品去我家，他说那是做给别人看的，但是就算在我们两个经济不宽裕的时候，他也一定要给我妈妈一些钱，我妈虽不缺这些，但我们的心意却是她所珍视的。我想我爱老公的很大原因就是他的这种为人风格吧。

　　刚开始记帐的时候并没有想到这小小的帐本会附上这么多记忆，算帐时才发现，这帐本记载的最重要内容竟是岁月！
