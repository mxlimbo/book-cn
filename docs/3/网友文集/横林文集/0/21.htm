临摹老公

　　我熟读《孙子兵法》，活用《三十六计》，终于不落痕迹地把一个叫做孙兵的男人骗来做男朋友。

　　此人身高体重能达国标，外表风度堪比明星，就是眼睛没我大，于是在我嘴里美其名曰"绿豆眼"。后来嫌三个字比他名字还多一字，就简化为："豆眼"，再后来，干脆俐落一个字解决问题："豆"。叫得顺口了，正式场合也就脱口而出。终于惹来问话："孙兵，你女朋友叫你什么？"该同志微微一笑，从容答道："我的英文名字！"

　　生日那天，办公桌上堆满鲜花，看都不用看，我就知道他孙兵绝不会有如此创意混迹其中，自打认识以来，他连片叶子也没送过给我，对他，我绝不抱有任何幻想。不过那天他倒是打了个电话来，言简意赅地说："我们结婚吧？"我说："好！"他说："那你去人事部门开个独身证明。"我回答同上，一个"好"字。

　　几天后，我们在民政局领了大红结婚证，我不免有点忿忿："求婚连束鲜花都没有！也没有说上一百个理由。"孙先生悠然道："别内疚，我这人不喜欢女孩给我送花，再说，娶你是我自愿的，你也别费劲编什么理由了，都是走形式，咱不搞这套。" 更有甚者，听他给朋友们打电话时说到娶我一事的语气居然大有"为民除害"之意。我咬牙切齿想到一句千古名言："君子报仇，十年不晚！"要知道，这句话从我决定开始追他那天起就一直支撑着我的意志。现在结婚了，报仇的机会终于来临了，我露出森森白牙奸笑。

　　我早听说过，控制一个男人的胃是控制他全部的绝佳捷径，于是，一为人妇就雄心勃勃开始了控制老公的重大战役。但是我做过两三顿饭后，他就在两个人的家庭会议上宣布了"不开火不做饭"政策，一脸坚决。我慷慨质疑道："敢问，你能不能给我个不做饭的理由？""从全局考虑，是不希望你劳累，不希望你的青春美色在油烟里泯灭！"我当然不相信他的这番外交辞令，但我是个知趣的人，我相信他的潜台词是说我做的菜味道不怎么样。第一场战役我就在无声无息中溃败。反过来想想，不做就不做吧，我也乐得清闲，于是跟着他吃食堂，吃馆子，不亦乐乎。

　　老公取缔了我从胃起步的控制行动后，加固在洗衣领域对我的监督。由于我从小对晒衣服，收衣服深恶痛绝，明确分工的时候，我的责任只到在洗衣机的协助下完成衣服的清洗为止，老公负责晾衣服与收衣服事宜。当时一疏忽，没有将叠衣服纳入分配计划。

　　老公收完衣服说："我来叠，你收进柜子，如何？"我一衡量，满口答应。他叠了两条牛仔裤后，要求我把其余衣服全部用衣架撑起挂在衣柜里，我立刻深感上当，开始扯皮。他也干脆，把叠好的衣服全抖开，往沙发上一扔。我心里琢磨："反正客厅是他的领地，他看电视的时候，看不过去自然会叠。"于是若无其事地去玩电脑。趁着倒水的工夫，我瞥了一眼沙发上的衣服，不禁佩服起老公来，想必他看着那堆衣服也心烦，居然用一条美丽的毛毯把沙发罩了个严严实实，心安理得地在那里看电视。

　　我开始发现报仇的难度，此人战斗力非常强。

　　家里的信用卡和存折一概都是老公掌管，然后按时给我发放零用钱。这财政大权是我一时头脑发昏拱手相让的，我承认当时完全忘记了"物质是基础"的真理。老公半推半就领了财政大权，开始还很谦虚，不久我就发现自己的战斗力随着财政控制权的消失而日渐减弱。

　　恋爱的时候，一跟他有争执，我就拿钱出气，出门大吃大喝，看电影，泡酒吧，打个车满城转。现在好了，一旦发生争执，我照旧使出杀手锏――离家出走，但是每次总是潇洒地出了门才发现钱包里的人民币连吃顿饭都不够！

　　老公知道我没几个钱，想跑出南京城的能耐都打折扣，索性气定神闲在家等着我走投无路的时候丢人现眼地回家。跑了两次我也就学了乖，不跑了，省得灰溜溜回家的时候更叫敌人暗地里笑破了肚子。

　　出走战术不再有效，我沦落到相信肥皂剧的台词："眼泪是女人最强大的武器。"第一次使用武器，老公果然手足无措，我越哭越有趣，为了给他留下一个意味深长的印象，索性躺到了被子里，来掩护自己马上要笑出来的表情。我这个人，脑袋一沾枕头，站着也能睡过去。醒来时，居然听到了厨房的炒菜声，原来这次是真的镇住了老公，他居然在开火做饭讨好我。

　　等我吃惊地发现老公的厨艺可以拿厨师等级证书的时候，简直想吐苦水。我真不知道，他还有多少没有使出来的杀手锏，可我自己已经快黔驴计穷了，这场仗还怎么打下去？

　　老公如此狡猾，令聪明女子苦折腰。

　　看他那副吃定我的样子，我终于感觉到大江东去的落寞，报仇的雄心壮志也落花流水飘零去了。
