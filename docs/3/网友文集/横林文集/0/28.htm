小林子其人

　　苏林是老公的同事，人称小林子。此人长相一般，却有一口雪白牙齿，笑起来大嘴一咧，颇有广告气质。

　　小林子很讲究生活质量，买只帽子买双鞋都要千儿八百，只可惜到了他身上却硬是显不出名牌的味，他却自得其乐。偶而缺钱也会嘀咕几句："妈的，一千块钱的东西穿我身上都比不过小孙身上两三百的。"此人口中的小孙，正是鄙人的老公孙青同志。牢骚归牢骚，买起东西来，小林子照样款得很。

　　小林子跟孙青属于同乡兼同事，很是哥们情深，因此隔三岔五来我家蹭饭也便顺理成章了。无论我做菜手艺多差，小林子都吃得津津有味，末了总不忘记来一句："叨唠贤伉俪了。"

　　小林子没城府，想什么就说什么。慷慨激昂地痛斥处长，针碥部门。说完举起茶杯咕嘟咕嘟喝茶，喝个底朝天便又露出一口白牙，笑得爽朗。孙青有时劝他说话三思，大多是被小林子一句："我他妈就这德行"顶回来。

　　孙青不多话，做事周到漂亮，又长得一表人才，搁哪都撑场面，是各部门抢手的优秀青年。就在他被最高领导借调去办公的时候，小林子被人狠狠踩了一脚，据说是处长给他穿了小鞋，原因不言而喻。机关里利害重重，又岂是小林子那样无心无肺的人能应付的？

　　那晚他来我家吃饭时安静得很，孙青便陪他默默喝酒，喝了一瓶又一瓶，他眼睛红红地看着孙青，笑了笑："我要是能像哥们你这样说话长个心眼，是不是会好点？"孙青说："是。"

　　小林子有一阵不来我家，孙青说他在谈恋爱。他打电话来时我便开他玩笑，他在电话那头哈哈大笑，我想象他咧着大嘴，牙齿雪白的样子，心里充满了温暖的感觉。

　　我特意做了一桌菜，让小林子把女朋友带来玩。女孩很漂亮，眉眼嘴角的那股子世故却让我心里发寒。小林子抽个空在我耳边悄悄说："我一直想找个跟你一样漂亮聪明的，这个虽然差不多，却少了些什么，说不出，你看得出吗？"我笑笑，说不出所以然。晚上孙青对我说："这个女孩太精明，她看上的只是小林子的条件。"

　　小林子真的失恋了，据说是他自己提出分手的。他瘦了很多，用一种我从来没有听到过的口气说："她有另外一个男人！"孙青拍拍他的肩膀，他笑了笑，也重重拍了拍孙青的肩膀，然后又露出那口雪白的牙齿大声道："你小子他妈的就是比我强，我还真找不到你老婆这么好的！"

　　生活恢复了原来的面貌，小林子笑起来的时候依然广告气质十足。
