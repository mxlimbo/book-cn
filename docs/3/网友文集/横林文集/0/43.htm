拇指姑娘

　　平时我上网聊天不喜欢和女人交谈，理由是受不了她们的娇柔造作；当然我本身也是个女人，并且是个自认一点不做作的女人。

　　拇指姑娘第一次碰到我时，毫无例外地对我说："你的名字真美！"我毫无例外地回答："谢谢。"通常，这句话后，我就不怎么会和对方说第二句话了，但是，那天也怪，聊天室竟然非常冷清，所以当她接着问我是女孩还是男孩时，我又答了一句："女孩。"

　　使我惊讶的是，她接下来就对我说："我也是女孩，叫曾燕，是宁波ＸＸ单位的。"

　　我愣了好久，才问出一句："你第一次上网呀？"

　　"不是啊。"她的信息很快。

　　"你总是一见到对方就告诉别人你的情况吗？"我更好奇了。

　　"有时会晚一点再说的。"她非常认真地回答。

　　我的天哪，我感觉碰到了一个白痴，说实话，就是在网上跟我厮混了一年半载的人都不会知道我姓什么的，我觉得在这个虚幻的世界实在没有必要知道对方太多。

　　"你干什么一见面就告诉我这些，我又没问你。"我决定刁难她一下。

　　"不知道，我一见到你的名字就很喜欢。"她说。

　　我忽然冒出一个念头，她是个男的，她说的那些都是假的！可是我又不能确定，于是我冒出了一句我自己也想不到的话："我想马上给你打电话！"

　　"真的？"她用了个惊喜的表情，跟着屏幕上就跳出一串电话号码。我立刻拨了过去，接电话的是一个非常甜美的声音："梧桐，是你吗？我就是拇指呀。"
"什么？喂？我找曾燕。"我继续试探她。

　　"哦，我就是。"她有点失望，"您哪位？"

　　我忽然笑了出来，我感觉自己多少有点神经质，我轻轻地说："拇指，是我，梧桐。"

　　"啊，太好了！"她立刻开心起来，丝毫没有介意我的试探，"我就猜是你嘛，要不哪会有这么好听的声音呢？嘻嘻。"我忽然有点惭愧起来，她是那么单纯，不设防的一个女孩，我却没来由地猜测她，就从这一点来看，我比一个娇柔造作的女人更讨厌。

　　拇指每天都会到网上来找我，看到她，我的心情就会象秋日的天空一样纯净，她所有的感情和语言都是那么心无城府。

　　有一次有个叫柳的家伙在网上跟我纠缠不休，他说："你既然取了这么个风雅的名字，定然是个才女啦？"我不理会，他又接着说："我来出个上联你对一下。天若有情天亦老！"当时我心情很烦躁，想臭他几句，可是我还没有开口，拇指就对所有人大声说了一句："柳若有情柳亦精。"我忍不住冲着屏幕大笑，屏幕上又跳出了一条拇指的私人信息："姐姐，你要是心情不爽我陪你找人打架！这个家伙挑起了我战斗的欲望！"我微笑地说："算啦，我猜那个柳树精的绿脸一定气得更绿了！再说你来了，我的心情已经很好了。"这是一句真心话，我发现自己真的越来越喜欢她。

　　有一阵我为某些感情困扰着，终日郁郁寡欢，甚至对着屏幕掉眼泪。但是我从来没有对拇指说过，在我心中，她是个小女孩，我从来不愿意自己的烦恼困扰她，我只是一个人承受着。可是有一天拇指却非常认真地问我："姐姐，你说过，哭和掉眼泪是不一样的，就象难受和痛苦不一样是一个道理，你能告诉我是什么不一样吗？"

　　我回答她："难受是因为有欲望，不甘心，但是痛苦是忍耐，是承受；哭可以放肆地发泄，但是掉眼泪却只能压抑地忍受。"

　　"你现在在掉眼泪，在痛苦是吗？"她问，我能想到她那轻柔的眼神。眼泪忽然就盈满了，唉，她是个多么敏锐，纤细的女孩呀，她还是发现了在我苦心经营的平静下，隐藏的波涛汹涌。那一晚，我们谈到很晚，我说了很多，忽然就觉得自己轻松了，她温柔地说："姐姐，我想我了解你们的感觉，不管你相信不相信。因为如果我是男人的话，也会爱上姐姐的。"

　　我忽然感到她是那么的贴心，我说："我相信你是了解的。"她用一种非常温馨的粉红色打了一句话："姐姐，我爱你！" 温暖一下子溢满了我。

　　我觉得幸福，因为认识了拇指姑娘――这个我在网上唯一的女朋友。
