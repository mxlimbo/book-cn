信

　　下班前，在传达室拿到一封信，是大学同学写来的。我已很久没有收到信了，更让我感动的是，这封信长达三页，端端正正的钢笔楷书。我反反复复的读了好几遍——忽然发现，收到一封信的感觉远比接到十个电话来的亲切。

　　其实以前我是一个很喜欢写信的人。那是从初中开始的。

　　班上有个女孩，她妈妈在她很小时就去世了。她很沉默，不合群。一个偶然的机会，我在纸条上给她写了一段友好的话；第二天，她回了一封长长的信给我。之后，我们几乎每天都会给对方写一封信，写信，读信，成了彼此生活中的一种乐趣。那些日子，我写信的热情是远远高于写作业的。回家总是赶紧把作业赶完，然后，在夜晚最安静最美丽的时候，听着音乐，娓娓的在信里表达自己。

　　现在想起来，那时的我们真是不可思议，明明就坐在一个教室里，而且，平日里也是形影不离的，却还能每天写一封长达千字的信给对方。翻看那些信，写的也无非就是些对人对事的看法，虽然没有多少意义，但是我后来作文总能拿高分大约也是缘与此的。

　　念高中时，邂逅了一个在其它城市读大学的男孩，带着少年的青涩与激情，我们以一周两封信的频率交流了近三年。他的字很漂亮，文笔清新怡人，每次收到他的信都象节日，那个晚上，我一定要用整晚的时间来构思回信的。在那些紧张而枯燥的日子里，写信与收信成了生活中唯一的亮点。

　　上大学不久，我就坠入情网。我的那个他少言寡语，寒假暑假与我通信也是三言两语，连扯两句天气的技巧都没有，通电话反而可以让我的失望少点，所以我的写信热情也渐渐冷却了。

　　大学毕业后，与朋友们的联系是以网络和电话为主的，如果有人问我要地址，我会不加思考的报出一串我常用的email地址。去年冬天，心血来潮的给十几个大学同学每人发了一封贺卡，没几天却统统因邮资不足退了回来。原来久不写信，连邮资从五毛涨到八毛都不知道——书信在我的生活中成了一件稀罕的事。

　　可是，今天当我收到这封信的时候，那些久违的感动却一一涌上心头，我似乎能看到写信的人在温馨的灯光中凝思微笑的样子，那么遥远却又那么贴近。

　　我想，我也该提起笔，给朋友们去信了……
