邻　居

　　因为拆迁，借了套房子住。

　　搬来的第一天，要用剪刀，一时找不到，我便按响邻居的门铃，隔着门，有个老人问什么事，我说明意图，对方不开门，我只好站在门口，像个不受欢迎的推销员。

　　隔了几分钟，老人又出现在门口，还是不开门，只从防盗门上一块剪破了绿沙的洞口把剪刀递给我。

　　回家对老公说："我真服了这里的小区文化，借把剪刀像黑社会接头。"

　　老公还剪刀的时候，照例是从绿沙洞口递进去的。

　　日出而作，日落而息。我们跟周围的人并无半点干系，渐渐习惯了这种漠然。

　　楼下装修，一早就乒乒乓乓，影响休息。我甚为恼火，总想干涉，但又存个疑团，为何隔壁的那对老人保持沈默呢？他们就在那家的正上方，应该更难以忍受才对啊。我便也忍着。

　　一日早起，碰到邻居夫妇下楼，老太太小声嘀咕："吵得觉都睡不着。"老先生温柔地说："他们装修要赶时间，而且不用开闹钟就提醒我们起来晨练也不错啊。"老太太听后莞尔一笑。

　　有时我和先生晚上回家会碰到邻居老夫妇也回来，他们一进门就用钥匙把防盗门锁好几圈，我和先生相视一笑，真是一对谨慎的老人。

　　和先生出外旅游几天，回到家里，才开灯几分钟，忽灯光全熄，出门看看，别处都很正常。先生按响邻居的门铃，老先生隔着门问什么事，先生说灯忽然熄了，借个电筒看看是电闸还是保险丝的问题。老人转身进了房间，我们以为他会从绿沙洞里给我们递把电筒的，可他居然打开了那道锁了很多圈的防盗门。

　　老人带我们到二楼检查了电闸后，说十有八九是保险丝断了。他回家拿了根保险丝，又要爬高给我们换，我和先生坚持不肯，老人却说："你们小年轻没有弄过这个，帮我打着电筒就好。"

　　灯光再度亮起的时候，老人带着电筒回到了家里，依旧把防盗门锁了好多圈。

　　下班回家再看到邻居破了绿沙的防盗门时，居然有种温馨的感觉。
