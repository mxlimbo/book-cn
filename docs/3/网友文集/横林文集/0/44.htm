庐山恋

（一）

　　因为没有坐过船，所以选择从水路去庐山。

　　放舟大江自古以来便是一种很潇洒的方式，我虽无缘独坐扁舟，可是船行江上，照样可以体验那番逸兴风发。

　　时已入秋，船行风疾，甲板上的人不并不多。茫茫江面，水天一色，时不时涌现一带绿林，仿佛少女脖间缠绕的丝巾，飘逸灵秀；而船头船尾的浪花被卷得如同白木耳般翻飞，饶有情致。

　　船行至半夜，忽然听到很大的水声，我以为下雨，披衣上甲板，却原来是风起浪涌，水浪在静夜里分外响亮。江面一片苍茫，偶而可见几点流星般穿织的江上灯影，仿佛银河星座洒落，映得这江上的夜分外孤寂，我心里亦涌上一阵落寞，想这大江滚滚千年，大约看寻常了这种孤旅寂寞，又岂在乎多我一个不眠的旅人在此兴叹呢？

（二）

　　船至九江，已是次日上午。

　　接我的是个年轻人，高高的个子，淡褐色的皮肤，笑起来很亲切，牙齿洁白而整齐。他说他复姓欧阳，单名飞。

　　看我形容憔悴又有些感冒，他以为我晕船，便安排我先在宾馆休息。我在甲板上站了半夜，受了些凉，头也有点痛，洗了个澡就睡了下去，一觉睡到天黑，被一阵电话铃里惊醒，欧阳飞在电话里笑:"你不是来庐山睡觉的吧？"我有点不好意思，匆匆梳洗下楼，他在大厅等我，穿著一件休闲毛衣，眼睛含着笑意。

　　睡了一整天，我恢复了很多，吃过晚饭，便想在九江逛逛，但是怕影响欧阳的安排，便婉转地说："我自己转转行了，到时找不到宾馆，就打你手机。"欧阳也便答应了。

　　天空中瓢着零星的小雨，我在烟水亭附近的甘棠湖边散步，这亭原是白居易贬谪江州司马时经常盘亘的浸月亭，后周敦颐之子以"山头水色薄烟笼"的诗情命名为烟水亭。夜晚无月，当然无从寻觅古人"别时茫茫江浸月"的情致，但是"闲思往事似前身"的感觉却尤为分明。似乎从一个世界遁入另一个世界，一切都可以重新开始，一切都不必计较。这种幻觉让我没来由地想喝点酒。

　　走进一个不知名的酒吧，找了一个黑暗的角落坐下来。酒吧里人不多，爵士乐很浪漫。我慵懒地坐在椅子里，审视出入酒吧的人。

　　于是我就看到了欧阳飞，他从门外走进来的时候，我心里轻轻一动，仿佛某根神经被牵动了一下。我并没有招呼他，只是安静地看着他坐下，看着他一个人在那里喝着东西，看着他点了支烟，他的脸便在烟雾里有点朦胧了。

　　他发现我的时候，显然很吃惊，愣了一会，才端着杯子走过来，站在我跟前，想说话，却终于只是笑了笑，我微笑地请他坐下："九江真小。"他看看我，向我举了举杯。欧阳的话不多，声音很温柔，和着音乐，恍如在梦中。

　　十点多的时候，他把我送到宾馆，约好第二天接我的时间就离开了。

　　九江的夜晚有点冷，但是空气清凌凌的透明。

（三）

　　庐山的南山盘山公路全长二十四公里，正是"一山飞峙大江边，跃上葱茏四百旋。"中所言的"四百旋"所指。欧阳飞笑着叫我数数是否有四百多个弯，数了几十弯，窗外的风景便始如画卷般展开了。群山蜿蜒，幽谷深坳，云锁雾罩，奇妙莫测。

　　小车顺着盘山公路像蜜蜂跳Ｓ舞一样左拐右扭，我有点晕车，由于山势增高，耳朵里似灌入水一样，听觉不灵敏起来，也就没有心思看风景了。我气血翻涌，几乎就要吐的时候，车终于在庐山大门口停了下来。

　　在我的要求下，他陪同我往山上步行。抬头，已能看到"云中山镇牯岭"了，依山而建的房屋，高低错落，雅致独特，被云絮重裹，衬着满山绿树，那般隐约，那般神秘，真个天上宫阙，人间仙境。

　　我回头想对欧阳发表一番议论，但见他目光闪亮，微笑地看着我，刚到嘴边的优美词汇就没了影。他了解地说："秋天的牯岭是最美的，万紫千松一点红。"经他一说，我果然留意到青松和红瓦屋面相映成趣，真是天街云市。

　　落塌云中宾馆后，稍事休息，欧阳便开始带我游览庐山。

（四）

　　游览路线是从花径开始的，据说这里曾是一千多年前白居易梦遇桃花仙子后，穿林寻趣，所偶遇的桃花林原址，当年诗人挥笔写下了"人间四月芳菲尽，山寺桃花始盛开。常恨春归无觅处，不知转入此中来"的佳句。此际虽然不见桃花人面，但是当年的风流曲折在欧阳温柔低沉的叙述里倒也别有情趣。

　　因为季节的原因，游人很少，我们漫步行走，低语谈笑，转眼就到了天桥，一虚一实两道峡谷对峙兀立，鬼斧神工。欧阳告诉我遇上云海的时候，银浪翻滚，恍如蓬莱仙桥。只可惜当日天晴云高，不能见此佳境。

　　依着天桥便是锦绣谷，石壁挺拔，层峦叠嶂，峰回路转，步游其间，仿佛身在画中。

　　移步又到仙人洞，佛手崖，果然仙崖翠壁，清幽雅然。一路听着那些美丽的传说，在欧阳的指点下，湖光山色都灵动了起来，听得我不知身在何处，几番疑是仙境。

　　我一路的拍摄景物，欧阳笑问怎么不留几张自己的影，我认真地说："只怕玷了这份灵气呢。"他轻描淡写地说："在我看来，你才是一道不俗的风景。"我心里轻轻一动，侧目看他，他若无其事，笑意盈盈。

　　下午我有点累，欧阳便陪我在宾馆附近散步闲谈。

　　林木森森，云雾蕴集，我们坐在林间的石椅上倾听风声，欧阳微笑地凝视我，眼神清亮，这情景何似在人间？他忽然问我："你可知道仙人洞本是吕洞宾与他的妻子隐居的地方？"我摇头，他笑到："两口子在洞里相敬如宾，不是吕洞宾吗？"我会意轻笑，羡慕起那种神仙眷侣的逍遥来，不禁轻轻叹息："只可惜这生活只在传说里有。""那也未必，你若住在牯岭镇，有个爱你的人相依相伴，不也就成了神仙？"欧阳笑道。我苦笑了一下，脑子里迅速划过了办公室的键盘声和都市夜晚的灯红酒绿，那个世界虽然已经遥远却不可逃避。

　　傍晚时分，天气阴沉，我们闲逛到了龙首崖，此地又名舍身崖，传说以前天池寺有个小沙弥和村姑相恋，苦于戒律，不能成婚。有日在此谈情，忽见浓云密布，以为天降飞云助渡幸福彼岸，于是双双跳入谷中，痴心感动黎山老母，将二人救往仙境。因为暗含舍脱凡身之意，后来成为佛教信徒舍身成佛之处。

　　龙首崖傲骨嶙峋，跃出欲飞，险象异常，四周上下壁立千仞，雾气缭绕，惊心动魄。我和欧阳坐在崖前沉默着，我不知道生命在大自然面前有多少尊严，但是想起当年信徒舍身的壮举，却让我有泫然的悸动，或者，只有融入自然的生命才是真正永恒的生命。欧阳的眼神有点迷离的停留在对面的悬崖上，他轻轻的说："跟你在一起，我刚刚有点看清庐山。"我不敢看他，只是静静地坐着。

（五）

　　庐山上手机收不到信息，我没法接到邢冰打来的电话，只好给他打过去，他的声音在我听来竟然有点陌生，我有点害怕这样的感觉，他问我想不想他的时候，我知道自己回答的有点勉强，事实上，到了九江，我就很少想起他，我想是庐山的美丽诱惑了我。

　　我倚在床上看书，电话响了，是欧阳飞。他问我在做什么，我说看书，他便沉默着，我能听到他呼吸的声音，时间彷佛凝固了一般，我想我们都在期待着什么，这期待那么长久，让我几乎窒息。

　　"我……"

　　"我……"

　　我们异口同声的说了一个"我"字，又异口同声笑了起来。这么一笑，倒没有那么尴尬了，他轻轻地说："我想见你。"我心跳若狂，我想我等的也许是这句话，只是他说出来后我却无所适从了。我的沉默让他有点局促了："我想你也许累了，算了。"我说不出话来，他却没有放下电话，就静静地听着彼此的呼吸，过了不知多久，他叹了口气，挂上了电话，我若有所失，心里忽然就空落起来。打开窗，晚风凉得很，我有点瑟缩。在这海拔一千米的山巅，我茫然如同一个迷路的小孩。

　　那晚我做了个恶梦，从龙首崖上的松树上传来奇怪的笑声，我惊恐中找不到欧阳，那棵树忽然裂开，天空响起惊雷，我掩着耳朵，天旋地转中失足落入了悬崖。

　　我满身冷汗的醒过来，心有余悸。拿起电话想拨给欧阳，几番迟疑，终于没有按下那数字。洗了个澡，不敢再睡，心神不定的看书一直到天亮，才迷迷糊糊睡着了一会。

　　欧阳看起来也没有休息好，脸色很差，他问我："睡得好吗？"我说不好，他盯着我的眼睛问为什么，我闭着嘴不回答，有点固执的倔强。

（六）

　　这是沉闷的一天，我根本无心看风景。跟着他不知走了多少路，他也很少说话，满山的风景忽然就黯然失色了，看到沿途的行人兴致盎然，我觉得自己快要在这一千米海拔的高空窒息。

　　我百无聊赖地坐在一块大山石上不肯再走，他一言不发的坐在另一块石头上，一根接一根抽烟，我的心一点一点收缩，到后来痛得不知所以然。眼泪渐渐弥漫上来，我静静地流泪，他背对着我，可是我流泪的时候，他却感应到什么似的回过了头，他的眼神那么痛楚，他坐到我身边，很轻很艰难地说："对不起。"他的脸那么年轻，年轻得让人心疼，只是我的心已没了方向。我不需要道歉，于他于我而言，都只是山的精灵蛊惑了年轻的心。

　　晚上我坚持要到网吧发email，他带我到牯岭镇唯一的网吧，我当着他的面给邢冰发了mail，告诉邢冰我想他。

　　往回走的时候，他一言不发，我用尽力气用一种理智得自己都害怕的声音说："我想明天把三叠泉，五老峰，含鄱口都玩完，后天转一下秀峰和白鹿洞书院就回去。""为什么？"他死盯着我，"为什么要安排的这么紧张？"我故意笑的很随意："没什么啊，只是我觉得这几天把时间浪费得太厉害了，我想早点回去，还有很多事要办呢。"他不说话，难堪的沉默。他忽然苦笑地说："浪费？你觉得这几天很浪费吗？"他的声音听来不再温柔，带点苦涩的刺痛，我不能接口，残忍地沉默着，我想迅速逃离庐山，逃回我的现实，这个念头让我清醒得像一片刀刃。

　　晚上欧阳按响了我的门铃，我把窗打开，在冷风里，他的酒气清醒了很多，我给他泡了杯茶，兀自坐在床头看书。我没有抬头，但我知道他一直都在看着我。我神色不变，但是心里波涛汹涌，我每一分钟都以为自己会崩溃，但是每一分钟我都很镇静。

　　不知道过了多久，他站起来，非常清晰地说："我走了，你好好休息，明天我们按你的计划安排行程。"他礼貌地离开了我的房间，我瘫痪似地倒在了床上，久久地不能动，泪水下来的时候，我的人已经空洞得没有了思想。

（七）

　　早上下起了雨，便如同我那阴暗的心情，愁云密布。

　　我们一早就出发了，在含鄱口走马看花一番，就往三叠泉去，他不带感情简单地向我解释着景点，不像受朋友之托招待我而像是某个旅行社的导游，我很佩服自己能神色不动地承受下来。

　　去三叠泉的那条小道很美丽，两边都是峭壁悬崖，铺石的小路上绿树成茵，路边的峡谷里怪石嶙峋，流水淙淙。也许是因为我的不动声色，欧阳也尽量的调整着自己，我们很有默契地调整着气氛。

　　行到一处，只见对面悬崖上伸出一棵俏生生的红枫来，姿态婀娜，其下一块巨石，上面汩汩流出几道雪白的浪，倒像一个小瀑布，衬着四周碧绿的山色，分外幽雅。我们不由都驻足欣赏，我说要拍下来，他便伸出手要牵我过去，我犹豫了一下，终于把手伸给了他，他的手大而温暖，他紧紧握住我的手，看了我一眼："你的手怎么这么冷？"我没有回答他，只微笑了一下。在他的扶持下，我爬到最危险的地方取了那个景。我不忍离去，就坐在附近欣赏，他坐在我旁边，淡淡地说："人若能跟这树一般就好了，喜爱她便可以肆无忌惮地欣赏她，热爱她，你说呢？"我答非所问的说："或者只是因为不经意发现，才觉得她美丽。"

　　当他再次牵我的手回到小路上，我们之间那份难堪已经渐渐稀释，走过这不亚于泰山十八盘的三叠泉之行，我终于看到了曲折玲珑的三叠泉。

　　这泉从半山涌出，就象苍天开了个小口，仿佛九天散洒，水泻银河，顺着山势，沿途三叠，总落差有一百五十多米，蔚为壮观。如云喷薄，如雪飘铺，若珠滚抛，若玉碎溅。

　　我拍了很多照片，而欧阳只是坐在一块石头上静静地看着我到处跑，到处取景，他的嘴角有一丝淡淡的笑，我恍然觉得这几天他长大了很多，比起当天在码头接我的那个年轻的男孩子，他身上忽然多出一种带点沧桑的成熟来。这个念头从心里划过的时候，我感觉到一种甜蜜而痛楚牵动。

（八）

　　因为下雨，我们放弃了五老峰的计划，早早回了宾馆。

　　按照行程，次日我就要下山了，所以入夜后，我一个人便离开宾馆，想好好逛一遍牯岭镇，这个天上人间留下了太多我的心情。

　　夜风很冷，一年有两百天雾天的庐山又下起了蒙蒙小雨。在庐山的这几天，我已经习惯了这种雾蒙蒙雨蒙蒙的天气，悠然自得地走在雨里。我拒绝去想及任何人任何事，只想任心灵一味的空灵，只是我又怎能做到呢？

　　欧阳站在马路对面。

　　我看到他的时候，他也看到了我，我们之间隔着一条并不宽的马路。

　　他的目光越过川流的车凝视我。天空瓢雨，我因为寒冷而轻轻发抖。

　　他向我走来，越过那并不宽的马路，刻骨的酸楚淹没了我，泪水盈满我的眼睛。我知道我所在的海拔已经距离天空越来越近，我知道我的泪水已经距离他的眼睛越来越近，我亦知道我的梦境距离现实越来越近。

　　在他的目光越过川流的车凝视我之前，我从来没有发现自己潜在的寂寞已如此深沉，在他走近我的身边之前，我从来不知道这种刻骨的酸楚是宿命。

　　他在微笑，笑得有种精灵般的神秘，他发丝上的水珠在灯光下闪亮，他的声音低沉而温柔："我想你是喜欢庐山的。"我又怎能否认，我只好轻轻点头，一点头我的泪水便滑落了，泪水落在地面会有声音吗？那声音是否如心碎的声音那么清脆？

　　他轻轻捧起我的手，轻轻放到唇边，他的气息热热的吹到我的手上，他轻轻吻了一下我的手，抬起头，眼睛明亮，在这无星无月的夜晚，在这海拔一千米的天上人间，我轻轻闭上了眼睛，因为幸福。

（九）

　　我跟着欧阳去看秀峰，在"飞流直下三千尺"瀑布前我轻轻许下心愿，如果可以，来世我愿意做一棵树，我愿意选择庐山生长，因为这里有欧阳，我想他会在千棵万棵树中认出我来，或者，那棵三叠泉路上的红枫就是我的前世，才会让他驻足不忍离去。

　　我要离开九江的那晚，欧阳带我到我们偶遇的酒吧，坐在当天我坐的座位上，欧阳凝视着我，一言不发，看了那么久，看的时间都凝固了，他说："不是因为庐山，你明白吗？"我点点头，重复着："不是因为庐山。"

　　汽笛响起，江涛汹涌，我知道自己在一点一点离开庐山，离开欧阳，也许一生都不再相见，也许来生还在庐山相恋。
