秋雨情思

　　入秋以来，天就像漏了一样，不停下雨，而这雨也分明没有秋雨的萧瑟，竟一味如春雨般缠绵。

　　平日在公司倒也不在乎外面的天气，拉下百叶窗，窗里窗外是两个世界。偶而把孔雀竹羽放到窗台上透透气，这植物喜雨，看她探出脑袋舒枝展叶倒也有趣。而秋风夹着凉沁沁的雨丝直往身上扑来，竟是那般无孔不入，直觉得那阵清凉透过嘴唇浸到四肢百骸，这一刻，才感觉到秋天已经来临。

　　碰到休息日在家，便百无聊赖起来。白天昏天黑地地睡觉，晚上华灯初上，看看雨不大，便穿着防雨的风衣出去透气。宽阔的马路浸了雨水，映着灯光，别样绮丽；而路灯光下的雨丝纷纷扬扬，抬头望去，像无数淘气的精灵，直往你眼睛里飞来，这一点灯光下竟也幻出了万千瑰丽世界。

　　夜来风雨声竟大了起来，能听到敲窗的声音，这点方显出秋雨打芭蕉的萧瑟来，再不似春雨润物细无声那般温婉细碎。

　　写作的时候，只有雨声伴随键盘的敲击，益发显得夜的静谧。我有时会放点音乐，大致是蔡琴的那些抒情作品，悠悠扬扬，缠绵缭绕。而这雨大约是无法得知我的快乐的，我便独自偷享从文字中找到的喧嚣的世界。

　　夜深的时候，沏一杯茶，坐在黑暗里。一曲终了，安静得只听到心跳和呼吸，我如此贴近自己，贴近记忆，所有的快乐和悲伤都飘飘扬扬地远去，只在那份感应自己的宁静中放松。

　　有雨的夜容易怀旧。我没有经历失败的爱情，也想不起那些伤感的情人，我只想起那些陪我走过花季一程又一程的朋友。

　　总是特别怀念大学时代经历的那些雨夜。那时候，我们多么年轻，可以围着一盏轻轻摇曳的蜡烛，说一晚上的心事。外面的雨时急时缓，而我们的心情如此简单，快乐的时候畅怀大笑，伤心的时候默默流泪。

　　那些快乐的时光永不再回了。

　　而这雨毕竟年年都在下，这心情也年年都可以从记忆里翻出来玩味。有时，我想，一年四季如此轮转，也叫我们不容易忘却那些远在天边，近在记忆的老友吧。
