如果这是我们相爱的最好距离

　　我已经记不起那一晚怎么会到这个网上来闲逛的了——这是一个局域网，国家某机关的全国专网。

　　那是一个很普通的聊天室，不用注册，不用密码，一切都显得很随意，事后我想，如果需要注册的话，也许我就懒得进去了。

　　随便取了个ID，点了"enter"。

　　聊天室里人并不多，我很随意地问了一句："谁愿意聊聊？"

　　一条深蓝色的信息跳了出来——

　　原野微微笑地说：秋雨梧桐，大家都很闲。

　　那么随意，我忍不住微微一笑，半躺在椅子里，开始与他交谈。他无非问我是哪个省份哪个部门的，我说我不是他们系统的，他却没有追问我怎么能上这个局域网。他似乎很淡然，说话很短，但是一直保持着那个微笑的表情。

　　时间很快过去了，我说要走了，他淡淡地问："明天能见到你吗？"我很随意地回了一句："也许吧！"

　　"我等你。"他用了一个句号，我猜想他的表情一定和我一样随意。

　　我连续一周到这个聊天室来，原野每天都在，有时早，有时晚，一来就和我打招呼。我已经知道他是二连浩特的，他告诉我一个月之后调到青岛，离开草原。

　　他的淡然似乎不是没有理由的，他爱看佛学的书，爱填词，在我的印象中他简短到"不知道"都只说前两个字。

　　这期间有个珠海的朋克似乎对我的名字一见钟情，总是找我聊天，我想他是个浪漫主义彻骨的人，听说他成天都在网上疯狂找我。于是我设了一个留言中心，对那些经常泡聊天室的人说，假如看到朋克，让他给我留言吧。

　　这是一个很随意的留言中心，不用密码，不设限制，谁都可以闯近来，谁都可以给我留言。比起互连网，我觉得这里真的很简单，很轻松，似乎处处都不用设防，反正都是一个系统的，当初设计者一定是这么想的吧。

　　原野有时会给我留言，话很少，大多是些简单的问候。我好象渐渐习惯每天晚上去聊天室待一阵了，无论时间长短，就和他随便地说上几句话。淡淡的，象他曾经用过的一个ID，是悠然。

　　我们谈了些什么我已经记不清楚，只是有一次说到要出家，他说去昆仑，我问他吃什么，他说每天都会为我采雪莲的，我问他还下不下山，他说永不，我说会很寂寞，他说不会，因为有你，当时我好久都没有能说得出话来。第二天他告诉我说梦见为我采雪莲了，只是雪莲采到了，我却不见了……我想就是在他做了这样一个梦之后吧，我开始明白自己其实每天是为了他而来的。

　　二十天后，我的留言中心忽然出现了朋克的信息，终于有人告诉了他我的留言处，他说最近澳门回归，所以很忙，即使上网也只是为了找我，从别人口中我得知他是个年轻有为的骨干领导，平时事务很忙。之后，每天他都给我留言。

　　不知道是哪一天，我刚到聊天室，原野就告诉我："昨天你走后朋克来找你了。""我知道，看到留言了"我很淡地回答他。这一晚他情绪不太好，据他说是喝多了，我想也是，他要离开二连，朋友们都会为他饯行的吧。我没有和他过多计较，早早地就离开了。

　　第二天很巧，我刚碰到原野，朋克就来了，可能是因为一个系统的吧，人们都出奇地热情，几种不同的颜色由几个不同的ID打出来同一句话："秋雨梧桐，找你的人来了。"我没有忽略这其中有一条信息来自原野，他用了一种叫做"深蓝忧郁"的颜色。朋克一见到我就告诉我马上要开会，只是来看看我，叫我等他，我说你开会去吧，我很快要走，他说反正会是他在主持，我想劝他离开，原野忽然对我说："我走了！"朋克调侃的地问我："为什么我一来，原野就要走？"我说不知道，其实我是知道的。我打了句私语，让原野等我三分钟。我终于把小宠遣去开会了，然后我就和原野激活了netmeeting，我修改了设置，是为了不让别人发现我在这里。

　　在netmeeting中，只有我们两个人，谁也没有说话，可是我能感觉他在那端的屏幕和我一样的沮丧。过了很久，他忽然说他很感激我，我不知道怎么回答，我说我们以后不要去聊天室了，用netmeeting吧，他说好。

　　接下来的几天，我们都在netmeeting里聊天。我感觉话题越来越不着边际了，可是自己越来越不受控制地每天都要来见到他，哪怕说着心不在焉的话。知道他在屏幕那端，我有一种说不出的感觉，似乎，很开心，又有点忧伤。

　　知道他生病那天也是他接到调令的那天，那时他已经连续打了几天的吊针，我说你不该来，他说没有办法，想你！眼泪忽然就掉了下来，我说不出话来，这是我第一次对着屏幕流泪，为一个叫做原野的ID。我说不能这样，你该做些别的，他说："但无非是上网，无非是想你……"我忽然间又无言以对了，我已感觉到我们之间的一种无奈来了，我第二次在屏幕前流泪，有心痛的感觉。

　　我想该戒了。

　　但是我每天还是会去那个网，不管斗争多久，我还是会接受他的呼叫或者去呼叫他。

　　直到有一天，他喝了很多酒。他见到我劈头第一句话就是："我不想再上网！"我似乎一直在等他说这句话，但是我还是过了很久才艰难地说出："好的"这两个字。

　　"我很受伤！"他说。

　　"我不是故意的。"我的泪轻轻滑落了，对他而言，我只是一场狂风暴雨，带凄厉的美，呼啸而过，让他猝不及防；对我而言，他是个过客，只是过客，风过可以不留痕，他却不是风。

　　他退出了会议。可是没有退出netmeeting。我盯着屏幕，不知道过了多久，他又呼了我，屏幕一片空白，很久，很久，他打出一句话："忘了说再见。"

　　我知道他想说的是——不要再见。

　　我很艰难地打出一句话："什么时候再见？"

　　我知道错，可是我还是打了这句话。

　　"不知。"他依旧那么简短。

　　沉默。

　　"你还想见我？"他忽然问。

　　"是的。"我干脆地打出了这两个字，本来我以为自己会犹豫很久的。

　　"我怕你，真的怕你！"他说。

　　"不要！"我的泪又来了，我看不清屏幕，但是我能想象他的表情。

　　"因为我爱你！"他终于说。然后就退出了会议，接着关闭了netmeeting。

　　我坐在椅子里，屏幕上空荡荡的，空荡荡的……

　　我想这是我们相爱的最好距离！

　　窗外飘起了秋雨，梧桐叶飘到了窗台上，我忽然想起他说的某句话，好象很远，很远，他说——我会去海边看海鸥，不是为你，是为爱。那时我告诉他我爱海鸥，我想做海鸥，飞过海，飞过忧伤，飞过一切……
