虎牙――致命吸引力

　　我喜欢长虎牙的男人，这是根植在审美潜意识里的。我不止一次为小说中的男主人公安上一对对虎牙，尽管这对人物塑造是几近可笑的画蛇添足之举，却乐此不疲。

　　我谈过几次恋爱，都无疾而终，有的从此陌路，有的成了泡吧喝酒聊天的朋友，萧苇是后者。

　　我喜欢旅游，有钱的时候就坐飞机坐火车去远方，没钱的时候就打电话给萧苇，借他的车兜风。萧苇基本有求必应，他是个三十五岁的黄金单身汉，拥有汽车洋房。我有时会开玩笑地说："真后悔没有嫁给你，否则就不用成天为生活奔波了。"萧苇笑得很温暖："现在也来得及啊。""好马不吃回头草。"我一笑置之。

　　我有时怀疑自己有问题，我总在幻想一个长着虎牙的男人，那个男人吻我的时候我会晕眩，我把这感觉写到小说里。现实中我的几次接吻都味同嚼蜡，我发誓不碰到一个长虎牙的男人就拒绝接吻。跟萧苇恋爱（或者确切地说是交往）的时候，正赶上我已发过毒誓，所以他只是拖了几回我的手。

　　年纪不算小了，独自参加了几次同学的婚礼后，就有人热心地问我对恋爱对象的要求，我认真地说："长一对虎牙吧！"他们于是哈哈一笑："小雪，你越来越幽默了。"

　　后来我就真的碰到了一个长虎牙的男人。我是在萧苇的办公室看到凌风的，他正在接受萧苇的面试。我饶有兴致地看着他不紧不慢回答那些可笑的问题，萧苇当场跟他签了约。

　　三天后我打电话给萧苇，不是借车，是借凌风，这个长虎牙的男人。凌风三搞两搞就把我那台故意格式化的电脑重装好了。然后我请他吃饭。

　　凌风爱笑，一笑就露出虎牙，这使我对他充满了某些不可遏制的情感。几次约会后，凌风吻了我的额头，我想爱情降临了，我感动得要流泪。

　　我储备了整个青春，等待一个长虎牙的男人来吻我。

　　凌风是工科研究生，他总能从我的小说中找出疑点一二三，为此，我尽量避免跟他谈及我的小说。可是我的创作必须有听众，我依旧去泡吧，跟那些朋友们谈小说，萧苇是其中一个，他听我说话的时候总是微笑着，眼神在烟雾里忽明忽暗，有时也问问我跟凌风的进展，我笑而不答。

　　我已经找到一个长虎牙的男人，这个男人英俊，聪明，精干，并且温柔。而我心里却弥漫隐隐的雾，隔着雾，我不知道是否应该让他来吻我的唇，或者我是怕这个吻让我无法进入幻想以及自己小说中的那个境界。

　　萧苇笑我："你这部小说的主人公居然没有长虎牙。""是啊。"我懒懒地回答，心里却有说不出的滋味。我跟凌风已经持续半年了，我不敢分析除了虎牙，他对我的致命吸引力在哪里。

　　我终于跟凌风分手了，还是那种无疾而终。

　　我约萧苇喝酒，他问我："看了你的新作，男主人公戴了一副眼镜，你能告诉我度数吗？"

　　我百无聊赖地晃着杯子："无所谓。这有必要定义吗？"

　　"我有个建议。"萧苇眼睛清亮地看着我，"四百度刚好。"

　　"why?"我心不在焉地问。

　　"因为我的眼镜就是四百度。"他声音很低，他的脸距离我不过几分距离，在我还没有回过神来的时候，他吻住了我。

　　三个月后，我嫁给了萧苇，这个没长虎牙的男人。
