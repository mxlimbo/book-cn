老人·海鸥

　　那一年，去看海，是冬天。

　　坐在海边的礁石上，风很大，斜阳很艳。

　　离我不远处有个老人，向海面上拋洒食物喂海鸥。那些洁白的鸟儿围着他翻飞，飞出柔美的曲线；夕阳的光星星点点的撒在碧蓝的海面上，也将老人的背影勾勒出一种凝然的意境。

　　老人回头看到我，冲我笑笑，招手示意我过去。接过他递给我的一袋食物，我会意地轻轻往海面上撒去；海鸥也会意地向我飞来，在我身边起起落落，欢快地唱着。老人微笑地看着我，那么平静，那么安详。波光点点的海面，夕阳红得醉人……

　　当海鸥都已远去，夕阳也已落下，我问老人："您经常来喂海鸥？"他点点头，打了个手势，我忽然明白，他是个聋哑老人。

　　我看着他，想起海鸥围着他翻飞时，那个被阳光镀了一层光辉的背影。

　　后来我把这件事告诉恋人，我说希望下一世能做海鸥；他温柔地说："你若是海鸥，我定是那不说话的老人！"
