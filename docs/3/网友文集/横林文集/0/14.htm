末班车心情

（一）

　　很庆幸，赶上了末班车。人并不多。

　　发车时间已到，司机却并无开车的意思。我有点烦躁起来。

　　大约过了五分钟，我终于忍不住了，非常礼貌的对司机说："先生，对不起，车票上好象是三十发车……"司机还没开口，邻座一个老太太笑嘻嘻地说："小姑娘，不要急，这是末班车啊，很多人都在赶呢，等几分钟，误不了事的。"

　　老太太头发花白，一脸平和的笑。

　　我忽然就想起当年在学校念书的时候，每天都要六点下课，而回家的末班车是六点一刻，那时，司机也是这样等着我的。当我匆匆忙忙的赶到时，我也记得那些经常同车的人习惯说："好了，来了，可以走了。"面对如此简单的一份关切，当时我是怀着怎样一份感激呢？

　　思想间，一个小伙子风风火火地爬上了车，满脸汗水，但是带着轻松的欣喜。他虽没有说话，可我感觉他心里定然抱着感激，感激车晚点了，只是他也许并不知道，这一车的人都是怀着一种体贴的心情在等着他的。

　　邻座的老太太还是带着从容的笑，她看看我，轻轻地点头致意，我会意一笑，这份默契，让我感受到月圆般的丰美。

（二）

　　车子在高速公路上奔驰，暮色渐渐深重起来，公路两边的植物和房屋都笼在淡淡的雾一样的光线里。

　　远处有一缕烟雾在袅袅飞升，我知那是某个工厂的烟囱。这本是并不浪漫的景致，于这极度放松的旅途，我却愿意放纵自己的想象去体会这一切的情趣，我或可把它想成炊烟。很久之前有首歌唱："又见炊烟升起，暮色照大地"，这曾是我以为很美的一种境界，炊烟在我看来最是有人间烟火味的。

　　记得有一年傍晚坐汽车去苏北，那时见到了真正的炊烟，我便因此羡慕起了那份田园惬意来。

　　每一缕炊烟下，都有一个家，也许并不富裕，但是，那炊烟分明在告诉世人，女主人是如此潜心地爱着这个平凡的家，也许丈夫酒足饭饱和儿女衣食无忧就是她的最高理想了，如此朴素如此单纯，却又涵盖了生命中最朴素的本质。有炊烟冒起的家是让人羡慕的，那份平和那份满足，已在飘起的意境里写尽。

（三）

　　夜色已深，我给妈妈打了个电话。妈妈第一句就是问我晚饭吃了没有。怕她担心，便说随便吃了些，妈妈说，已经为我准备了最爱吃的菜，会计算好时间给我热一下的。

　　放下电话，心里塞满了柔软的感觉。

　　我很小就在外面念书了，每次回家都是那么匆忙。我习惯赶末班车回家，而那时妈妈总是在等我。最晚的一次因为车子晚点，妈妈等到凌晨两点，而我走进家门的时候，满满一桌子的菜却是热的。

　　印象中，每次回家，都是先看到家里温暖的灯光，然后看到妈妈站在门口的身影，不管多晚不管多冷。不同的是，妈妈的身影越来越瘦，而那灯光越来越深的刻在记忆里，在我孤独和痛苦的时候，总是想到这灯光，这身影，而想起时总有泫然的悸动。
