无关爱情

　　有时我想，何必问是不是爱？爱又如何，不爱又如何？哭过，笑过，有些东西就已圆满――

　　那时，学校后面有一座寺院，香火很盛。不知什么原因，我有一阵不爱念书，经常吃过饭就跑到寺院里，到处转转。我喜欢闻那缭绕的香味，看一个老和尚在小屋子里写毛笔字，我常能一看就是一个下午。

　　那一段时间，我经常逃课，考试也不及格，就那样过了一个冬天。

　　春天来的时候，我还是不合群，沉浸在一种现在也无法理解的迷惘中。

　　春游那一天，我离开了大队伍，坐在草地上，眯着眼看小草上的虫，看了很久很久，直到有一个很好听的声音在我头顶响起："嗨，干吗呢？"我抬头看看他，阳光很烈，我看不清他的脸，个子高高的，穿一件青色的大毛衣，我没有理会他，继续看我的小虫。他坐在我旁边，一言不发，过了很久，我忍不住问："你在看什么？"他笑着问："你呢？"我没有回答，抬头看看天，很蓝，再看看他，他的笑容很好看，我冲他一笑："我在发呆！""爱上一种无所事事的感觉。"他轻轻地说，"好象自己跟自己分离，轻飘飘的，是不是？"我吃惊地看着他，他说的就是我整个冬天的感觉，那个我弥漫在寺院的香气中，我的心离我好远……

　　我越来越少去那个寺院了，但是我给他写很长很长的信，他也回我很长很长的信。他说，总在夜深人静的时候，点一支烟给我写信，我似乎能在信纸上闻到淡淡的烟草香，最后凝聚在我身体的某个部份，弥漫成一种温馨的感觉。寺院的香味不再熟悉，我不太逃课了，也不再动不动考不及格了。

　　一年后，我已经习惯每周收到他两封信了，他说读我的信像在读他自己的思想；我也在他的信中找回一种熟悉到不能再熟悉的感觉。可就在这时，他的信断了，整整半年没有消息，我从焦急，愤怒，到最后的无所谓，也花了整整半年的时间……

　　高考结束那一天，没有想象中的狂喜，我很平静，收拾完东西，一个人静静地离开校园。在传达室门口站了几分钟，还是忍不住进去了，有他的一封信，就是那天到的，很短，只有三个字：我想你。

　　我忽然有点明白了。抬头看天，云淡风轻，在那个夏天，有种少见的清淡。我折起信，放进书包，到学校后面的寺院转了转，出来时，刚好赶上末班车。

　　然后就开始了我的大学生活，偶而还是与他通信，通电话――感觉淡淡的。

　　有一天我回寝室时，室友告诉我有好几个他的电话，熄灯睡觉后，他的电话又来了，声音有点哑，低低地问我："发生什么事了吗？我为什么有心痛的感觉？牵牵扯扯的……"我沉默了一阵，终于淡淡地说："我恋爱了！"他很久没有说话，我也不记得最后是怎么道别的了。

　　很长时间没有他的消息，我想，也许真的结束了。

　　但是他的信还是来了，问我快不快乐，又说有了一个女朋友，清纯如水，最后一句是――一如你当年的模样，而我知道她永不会是你……

　　那一晚我喝了点酒，然后坐在桌前，静静看我们这四年来的信件。可能是酒精的作用吧，我半夜醒来，不可抑制的痛哭出声。

　　又是两年，我们还是偶而通信，通电话。他说我变了好多，跟他第一次见到的那个爱看小草小虫的女孩完全不同了，我说，你给我的信上好象还有淡淡的烟草香，跟当年寺院中的香气搅在一起，让我有时忘了自己是谁。
