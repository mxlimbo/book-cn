一滴泪

　　若冰第一次见到方潇，是钟洛的生日。卿卿说方潇是钟洛最好的朋友。

　　方潇很瘦，长长的头发，看起来有几分忧郁。那晚他喝了很多酒，一直在笑，后来趴到了吧台上，若冰却分明看到他眼角有一滴泪，清亮。

　　若冰固执地想起那滴泪，也固执地想见方潇。

　　方潇已经不太记得她，电话里的声音漫不经心。她约他喝茶，他没有拒绝。

　　他还是那么瘦，已是深秋了，却还穿著衬衫，是那种藏青中透着紫红的面料。他的眼神里没了第一次见面时的千回百转，从容，带点无所谓的淡然。

　　若冰凝视他，却说不出话来，他并不在乎她的沉默，两个人便这样默默地坐着，在幽幽的灯光下。

　　他们便经常一起喝茶了，不怎么说话，他想心事，而她想他。他有时会笑着说："你不会觉得我很沉闷吗？""不会。" 她微笑地摇头。

　　若冰知道是因为那滴泪，她固执的相信，那滴泪引她走入他的生命，要走很久很久，也走很远很远。

　　好几个秋天过去了，卿卿终于嫁给钟洛了。若冰是伴娘，方潇是伴郎。

　　那一晚，方潇送若冰回家。她醒来的时候，他说："你昨晚醉了，一直在笑，只是眼角有一滴泪。"

　　她轻轻的说："我第一次看到你，你也醉了，一直在笑，你的眼角也有一滴泪。"

　　方潇看着她，看了很久很久，轻轻的问："那么那滴泪呢？"

　　"你昨天看到了。" 若冰幽幽地说。

　　她摊开手，掌心有一张纸条，很旧很旧，上面端端正正写着"相见却无从睁眼，心中漆黑一片。潇"

　　这是六年前，方潇夹在一本书中的，书是卿卿借给他的。

　　若冰淡淡地说："那本书是我的。"
