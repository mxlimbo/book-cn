最浪漫的事

　　我病了。感冒引起发烧，浑身无力。

　　老公手足无措："我们去医院！"我不想动，便对他说："吃药就可以。你给我拿点药。""哦！"老公应了一声，飞快跑到专门放药的抽屉前，我只听到一阵乱翻。他忽然又一个剑步跨到床前，拿起电话就拨，过了一会，听到他在低声嘟囔："怎么没人？"我问他："你给谁打电话啊？""我妈，问问她吃什么药。"他的样子内疚得很，大约是因为没有电话没通。我又好气又好笑："不用打了，把抽屉搬过来，我自己知道。"

　　我拿到药，他把抽屉放好后，又端端正正坐到了床前。我叹口气，无奈地对他说："你给我倒杯水好不好？""哦！"他立刻跑到客厅从饮水机里给我倒了一杯凉水过来，我知道自己少说了一个热水的热字，但是看他一脸诚恳，双手奉茶的样子，也不忍怪他。

　　昏睡中感觉冷，便对老公说："给我换条厚点的被子吧，出身汗就可以退烧的。"老公立刻从柜子里翻出了最厚的被子。

　　一整晚，我都处于似醒非醒的状态，热得一身一身出汗。每次我一把被子蹬掉，就感觉他轻手轻脚为我盖上了。天亮时，我才沉沉睡去。

　　醒来，老公满脸阳光地注视着我，神秘地问我："你在家生病的时候，吃什么？"我想了想："稀饭。"他笑嘻嘻地转身出去，一会儿就端了一碗稀饭进来："是这样的吗？"我讶异地看看稀饭，再看看他，他笑得很是得意："我早上打电话问你妈妈后给你煮的。"

　　我忽然就说不出话来了。

　　我记得曾有人说过，爱一个人，就把他当作你的父亲，儿子，兄弟，朋友，情人来爱。而老公现在看着我的眼神，像一个父亲，悉心照顾着女儿；像一个儿子，因为做了某些尝试，期待一句赞扬的话；像一个情人，舍不得漏掉我一个表情……

　　想起刚结婚时，有的只是小时过家家的兴奋；而此时，他在床头喂我稀饭，我居然有相濡以沫的感觉。

　　他看着我,忽然很认真地冒出一句："你要是老了，走不动了，我也这么喂你，好不好？"我噗嗤一笑："我老了，你也老了呀。"

　　我心里分明想起了一句歌词："我能想到最浪漫的事，就是和你一起慢慢变老，老得哪儿也去不了，你还依然把我当成手心里的宝。"
