雪

　　是冷遇　
　　将她培育成冰清玉洁的花朵 　　
　　从此，一个温暖的预言 　　
　　流传人间

——题记

　　花期更替，经过了春兰秋菊的喧哗，在长长的难耐的寂寞之后，有一种剔透的花瓣飘着淡淡的哲学意味从天而降，牵初生的舞步作风景——这是一个不泯的片段，关于芳华永驻的故事。

　　我们实在算不得智者，只是在每一个冬季，只要用心，就能听见许多静的足，纤细的足在窗前行走，许多轻的声，透明的声在空中低吟，轻描淡写却生动无比。

　　雪已经落下来了！

　　这些善良的孩子是一贴贴良药，吐放灵秀入髓的馨香，经久不散。这是我觉不到寒冷的秘密吗？

　　花凉如水，那些刻骨铭心的如诗情怀，千言万语的少年浪漫，纷纷以蝴蝶的姿态飘落。

　　踏雪履香，我问，花儿选择冬天盛开，需要多少勇气？因寒冷而孤独，因孤独而绽放，因绽放而美丽。我不明不暗的寂寞被这雪白的美丽照亮，又渐升一种温暖——一冬的磨难都披你不尽的温柔啊！

　　这是一个洁白的童话，也可以写作一阙清瘦的小词，按你的方式理解雪吧，——天空依旧是天空，雪依旧是雪，而你已不再是你，啜一口雪的芬芳，如饮薄酒一杯，滤尽陈年的荒凉。

　　你看你看，岁月的边缘已挂满水灵灵的往事。
