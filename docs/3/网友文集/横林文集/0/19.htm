楼上有一个人要往下跳

　　寝室有个北京女孩洋洋以纠正我的普通话发音为乐。

　　我说："吃饭啦。"她说："不是雌饭，是吃――饭。"还把上唇翘得高高的，我白她一眼说："当心牙着凉。"小雨和洛东就笑起来。

　　我们四个人形影不离，说好不谈恋爱不脱离组织，以小集体形象纵横校园。

　　洋洋习惯晚睡，她说对着三个睡着的人比看恐怖片更刺激，原因是小雨磨牙，洛东抠墙，我用日语说梦话。我反驳："我不会日语。"她说："反正不是普通话。"

　　上完晚自习感到饥饿，四个人就结伙到小吃街上觅食，时间已晚，为了赶在熄灯关门前回来，我们一路狂奔。到校门口时，胖呼呼的洋洋终于不支，弯腰喘粗气道："歇，歇，歇会。"大家都停下来，我一屁股坐在花坛边仰头深呼吸。校门口有一栋商用楼，我抬头时看到楼顶上有个人影，飘飘忽忽仿佛要离开栏杆，我一个哆嗦厉声喊道："楼上有一个人要往下跳！"叫的太急，家乡话就出来，另外三个人愣了几秒就大声笑了起来，洋洋说："你又说日语了。"我连忙用普通话翻译了那句话，她们看看楼上，鬼影子都没有，洛东说是我眼花了，准是方便袋之类的物事。

　　三个人逼着我重新说一遍"日语"，我只好说："雷（楼）让（上）有夜（一）个宁（人）要忙（往）户（下）头跳。"一说完她们又开始大笑，笑得气都喘不过来，我说了十几年的话一入他们的耳竟然比卓别林的戏剧还有煽动力。

　　洋洋从此不再纠正我的普通话，她们没事就缠着我学苏南话，还说："就从楼上有一个人要往下跳学起。"对于其它语系而言，吴语难度本来就大，而她们又偏偏从一个长句子开始，当然更加困难，学了好几次都古怪得让我忍俊不住。

　　在她们还没有学会这句话的时候，我有了男朋友，率先脱离了四人帮小集体。

　　我终日风花雪月，晚上熄灯才回来，回来就挂着耳机写日记，写完就睡觉，跟她们交流越来越少。我很快发现她们也各自散了，小雨也恋爱了，洛东成天泡图书馆，洋洋忙忙碌碌地搞起播音工作。

　　快毕业的时候，我回寝室的时间更晚了。有一次，都快熄灯了，寝室还是我一个，十一点之后，她们三个一起回来了，笑声大得一楼道都能听到。我说："你们干什么去了？""吃火锅。"小雨说。我心里忽然一阵不舒服，她们居然没有通知我，好象就没有把我当这个寝室的成员，这样想着的时候，跟她们的距离又远了几分。

　　几天后，我整理东西，却发现了一张纸条："小雪，今晚我们寝室聚会，下午五点在亲亲火锅城见，你一定要来啊，我们定的是四个人的份。"纸条压在我的文具盒下，但是我没有注意。

　　我愣住了，是我忽略了什么吗？那一晚，她们那么开心，笑得那么夸张，还喝了酒，是真的快乐吗？快乐后面是不是有沉沉的失落和不满？

　　离校那天下雨，我一个人走出检票处时，洋洋，小雨和洛东已经在月台上，她们用力地朝我挥手，洋洋说："怕搞不到站台票，我们就早点混进来了，想送你。"

　　我鼻子一酸，眼睛就湿润了。小雨说："总会再见面的，别难过。"说这个话的时候，她自己却流泪了。洋洋说："还想跟你学苏南话呢，就学了一句。楼上有一个宁要往下头跳。"听到这不伦不类的发音，我流着泪笑出了声，她们也笑，眼泪却不受控制的流，不知是谁先抱住了谁，四个人就紧紧抱在一起了，互相为对方擦着泪，而泪又不听话地涌出来。

　　四年来，我们第一次抱在一起，却是分离的时刻，想起来多少有点伤感，想青春何尝不是如此，等你察觉，已经一纵即逝，永不再回。
