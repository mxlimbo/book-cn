思念不是一种病

　　等你在穿山越岭的另一边，
　　我在孤独的路上没有尽头。
　　时常感觉你在耳后的呼吸，
　　却不曾感觉你在心口的鼻息。

　　这个午后，我反复听齐秦的这首<<思念是一种病>>。我把这段时间留给我的一个网友，他叫寸夕。

　　认识他之前我很少听齐秦的歌，他第一次到我的聊天室时，我却刚好在听这首歌。此后，连续三天，每次他来时，我的MP3总是刚放到这首歌，这使我相信，冥冥中，他和这首歌有着某种牵系。

　　他从来都用一种表情——平静，可是在他的平静下，却有着一种天生的霸道。我记得他第一次到我的聊天室时，我刚要去吃饭，他平静地说："等一等，我翻遍了所有的名字，才找到你！去吃饭之前，先对我说句话，我会等你。"我本来可以置之不理的，可是我竟然没有立刻离去，我画了一个"^&^"，他问我是什么，我说是个笑脸，可以想象，皱起鼻子地笑，或者吐着舌头在笑，他平静地说："你去吧，我等你"

　　他每天都在固定的时间来看我，用他那个平静的表情与我交谈。有一次我们同时都在听齐秦的歌，整屏都是歌词，后来我回头去看，这些不同的歌的词交错在一起，有种异样的美。

　　他不太合群，身高一米八五，长的很帅，在现实中是个偶象般的人物，又是电台的DJ，使他总生活在聚光灯下，但是，他的内心却有一种平静的孤独，使他远离喧嚣，远离追逐。

　　他跟我无所不谈。有一次，我对他说："我都看了你那么久平静的脸了，你笑一个，好不好？"他迟疑了很久，最后在屏幕上出现了一个微笑的表情，后面的那句话却很委屈——你叫我笑，我就笑了，我从来不是这样的。

　　我没有再逗他，我能感觉他平静下率性的孩子气，这很象我，霸道，善感却又有点孩子气的任性。

　　我们曾经谈到青年文摘上那篇《很爱很爱你》，他说他就这个故事做过一档节目，很多听众都打进电话。他整理了这档节目，并且和编辑部联系，得到了这篇文章作者的email地址，将这档节目和一些听众的信发给了作者。我心里有一种很特别的感觉，与他之间的距离竟然那么近，近得可以呼吸到一种叫做欣赏的气息。

　　他容易动感情，对生活中的细节，从来都有着敏锐的触觉。上班时，看到两个聋哑人在用手语交谈，他做了一档夜话节目，我问是什么主题，他说，希望相爱的两个人争吵时就象这两个聋哑人一样，那么就没有伤心的话语了。当时我好久好久都说不出话来，沉浸在他那句话的意境中，后来他打了一个^&^，我从心里轻轻的唤了一声寸夕，我想我的呼唤他会听见，因为屏幕上出现了两个字——梧桐……

　　我说过，美丽的东西总是不长久的，缘若圆，便不远。事实上也是这样，因为某些无法更改的原因，他告诉我，他会离开一段时间。

　　在他离开前的三天，我们约好，每天晚上都通电话，在很深很深的夜里。

　　他的声音从电话那端传来，很好听，很感性，他跟我谈了很久，一直到凌晨。后来，他对着话筒轻轻唱起了歌，那么深情，那么动人……或许是因为感动，或许是因为离愁，我轻轻地流下了泪。他说他很开心，无论怎么走，无论走多远，都有个心愿，要我生活地快乐，生活在那个调皮的微笑中。

　　最后一天，却因为他没有挂好电话，我一直拨到一点也没有能拨进去，而他偏又没有我的号码。第二天，他还是上了网，他说等了我一晚上，我说电话没有挂好，可能是天意，他说："你再说一句话吧，让我一直记在心里。"我毫不犹豫的打出了一行字——电话线路可以不通，但是心灵的线路永远为你畅通！

　　在新年钟声敲响时，我接到的第一个电话就是他的，他的声音还是那么好听那么动人，这时候，我的耳边又响起了齐秦的《思念是一种病》，我想我会思念你的，寸夕，但是，不会成病，因我知道，你会在我的祝福中，和我一样生活得很快乐。
