阴　天

　　连续几天艳阳高照，忽然碰上了个阴天。虽是阴天，却不郁闷，是那种绝不会下雨的阴天。

　　心情不高不低，不明不暗；坐在临窗的茶座，可以看到都市的街道。没有班驳的树影，满街的梧桐绿得很纯粹；天是灰白的，灰白而不迷蒙。

　　可以什么都想，也可以什么都不想，是一份平滑的闲适。

　　所有的快乐都在晴天，而所有的忧伤都在雨天——阴天似乎很少成为记忆的背景，留给它的大都是些如水的情节，很淡，不留痕迹。

　　于是选择在阴天给朋友打电话，语气轻柔，没有喧嚣生活的浮躁；选择在阴天化很淡的妆，施施然漫步行人很少的街头，幻想自己走在绿荫葱郁的山路。

　　阴天是一个内敛含蓄的情人，不善表达，却有脉脉的眼神，使我只想轻轻依入他怀里，闭上眼，呼吸一种叫做从容的风情。
