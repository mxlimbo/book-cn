你好，打更人

散步的鱼(10月23日11:30)

　　我是一个打更人。

　　我是一个打更人。现在我就开始打更。你看见我异常坚难轻松的践踏自己的影子前行。它在梆声灯影里忽短忽长它是那么难看。你几乎听不到什么梆声。是的，这里没有梆子也没有更时。这只是一个故事的城。

　　我很老了也跛了所以我打更。我没有子嗣没有亲眷也没有妻子。我是一只孤单的老狗。秋渐深了我开始咳嗽。这几乎省却我敲梆子的气力。我的咳嗽是那么准时而稳定。就象我年轻时候的手。有时我会被自己突兀尖锐的笑声吓一跳。没人会相信我年轻过。我真的老的象只狗了。那些年轻的人们都消失了。

　　顶上是南去的过客雁亡亡的去声。周遭是人家的狗渐次伏的隐吠。很夜很夜那些回还的音。都不忍闻。可我是打更人。打更人将习惯一切事物。

　　街角有几个模糊的人影。我的眼睛几年前就昏花的厉害了。那是一些游魂。我咳着经过的时候。它们绊倒了我。当我抖抖索索爬起来的时候它们已经四下散了。打更人将习惯一切事物。这座城就象一个迷宫。永无休止的变幻图案。我记不清有多少年了。你可以想像一下打更人在鬼魅流影里穿行的声场。我们都没能步出这座迷宫之城。云彩被风扯吹丝丝缕缕。那是一些流言。没有天日。这里是亘定的白夜。每晚我会遇见她。

　　她一定被诅咒过。

　　很多年了。那是一支流水的行板。我老的已经没有了叙述故事的渴望。你可以允许打更人蜷在廊影里作暂憩他的咳声已经长久的打扰梆声。这将使时空发生紊乱。现在我们可以看到那个女人了。

　　不要轻掷美好的颂言给她和表达她的身体任何定格都将失去真实。歌声魅影。我担心她随时会唱出一些歌来。那时我也许会进入另一座迷宫之城。我已经惧厌了任何类似的假设。看着她罢。女人。

　　他们发誓要走出去。他们发誓相爱用尽这一生。

　　嘘。你不要走近。那些土馒头是一堆堆的波浪涌伏。他们已经被淹没了。他从一树坟头生长吸汲死人骨血而阴繁的花枝上撷了一枝花。他把花插在她的鬓角。黑鸦鸦的发上抿了一支燃烧也似的花。他轻轻的吻了她一下。美的邪异。

　　他们没能走出去。他们都迷失了。自己和爱人。

　　我很倦了。倦的不想再前行。他说。

　　来呀。爱人。我好像已经能看到出口了。来牵挽情人的手吧。她说。

　　他抚摸到了她的裙裾。他随她走了一阵便放开了他潮气的手指。

　　他又跟随她走了几步便躺在路边的尘土里。他立刻睡着了。

　　她毫无识觉。

　　黑鸦鸦的发上那枝花依然快活地燃烧着。

　　他和她在这个城市里消失了。再也没走出来。

　　好了。打更人的胸腔已回复平稳的呼吸。那些痛痒的气流不再冲击他的肺叶。它们不再不安份的快乐跳跃。秋天的咳疾已被压缓。我们将继续。在空旷阴霾的城市里打更。长长长长的路有没有尽头。

　　你为什么永远盯视自己的脚。

　　我要走得坚实。

　　你频频回头有什么牵引你的目光。

　　我要知道我走的路。

　　你为什么不看你每晚要遇见的女人。

　　…………………………。

　　梆声。

　　我已经太老了。老的无法回答任何问题。

　　禁声。

　　女人来到我身边。

　　她的面容在我眼中只是一片发亮的云彩。我的眼睛几年前就昏花的厉害了。我无法开口。也许我应该咳嗽。女人伸出手来。女人的手放在打更人的脸上。那真的是一双美丽的手同样我也不想收集语言来描述它。它很美丽。我想这已经够了。然后我们看到女人吻了打更人。对。就是这样。很轻很温柔。象情人的手在抚弄你的心弦。我们都会有一种想哭出来的欲望。你们真的看到两滴泪了。它们跟随着它们在坠落然后它们很快不见了。它们也许会滴入地心。

　　好了。现在。让我们离开吧。不要回头。不要惊扰。

　　他们发誓要走出去。他们发誓相爱用尽这一生。

　　他们现在在一起。

　　我们要开始遗忘这个城市。
