时空风情画对话

散步的鱼(9月8日19:40)

　　这是个唾面自干的世界。

　　所有的日子也并非云淡风淡。

　　在泛滥如水的日子里。

　　是选择逃避还是冲撞其间？

　　很多时间在打开的聊天室里闲坐，充当一次真正意义上的过客和看客的角色。没有戏份的独角戏。发疯似的运用各种聊天表情，自嘲着不是正确浪费时间的方式。

　　寄托。投注某种事物寄托？虚假的心理安慰，不自觉的。已是不少了。

　　诗人在狭长的地带说道：在那里，一枚针用净水缝着时间……………。

　　沉侵在优裕的感伤里。那是一些破败陈旧的语言。

　　某个概念中的荒原。番僧立于高地。土黄袍子张如鸟冀，在风中噼啪作响。

　　没有文字去诠释.回避且作空间幕布的风情画罢。但不能。云彩流的那么快。一一游走,从番僧头上.这个枯瘦的人(？)用一种什么力量震撼时空并得以长久立于此贫瘠的心灵之原。使者苍鹰只能用锐利如刀剪的眸子和飘落的羽毛来作答。

　　而我们不明白。

　　这是缭乱低沉的圣乐从地缝中漏出.我的龟裂的心灵之原，那些散落规则的奇形石块，你们在讲述些什么被尘土封缄的故事。

　　番僧没有眼珠.他的眼睛隐藏在他每一枯皱的皮肤下。从他蠕动的嘴唇里你听不到什么启示录。颂经声和流云一般快.那是围绕时空的一种回旋。万事旋进又旋出。而番是不要膜拜的。

　　看见风洞口的那棵树么？它不代表什么意义。

　　一棵树。一棵风洞口的树。

　　它也是番僧。
