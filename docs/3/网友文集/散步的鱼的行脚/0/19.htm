阳光照耀我的破衣裳

散步的鱼(10月23日11:30)

　　阳光照耀我的破衣裳。

　　五子才哼了一句就被老婆可爱的小脚丫温柔的踹了一踹。

　　起来。做饭。送儿子上学。

　　五子本来堆了一脸笑的。现在却是凝猪油似的固定在脸上了。五子觉得空气有点僵。他奶奶的。五子悻悻的套上裤子。每天早上跟奔命似的。五子看看身边的老婆。老婆还在半醒半梦中。五子觉得今天老婆好象提前结束青春的尾巴了。脸皮松弛毛孔粗大。躺在那儿就跟块大木头。五子想女人的脸色就跟那墙纸一个色儿。女人睡衣的腋下破了个洞。于是五子仅有的那点冲动也破洞而出全泄没了。五子想起曾在先生网站里看过一篇文章。上面步步为营循循善诱的教导妻子怎么博取先生的欢心。有点言传身教的味道。五子暗暗点头。当时文章后有张跟贴。一个男人意犹未尽的说，真应该叫我老婆来看看。五子当时有点可怜那男人。现在五子可怜自己。五子捅捅睡姿不雅的女人。女人没点女人的样子。五子说。女人咕咙了句什么。又翻身睡过去了。

　　五子把儿子送到学校。然后火烧屁股似的赶去公司。老远的看到主任吴痕挺着肚子站在秘书陈世间的桌旁共看电脑上的一份文件。五子瞅吴痕没在意溜到自己的位上坐下。还没坐定。陈秘就咯咯的笑了起来倒把五子吓了一跳。奶奶的。五子在心里骂了一句。偷眼看吴痕和陈秘边看文件边调笑。吴痕倒是有本事。三言两语就把个陈秘挑逗的花枝乱颤娇谑不断。五子心里酸酸的，小心喽，别把脸上的半斤粉抖下来。五子想陈秘脸上的妆层层剥下来是什么样子想的正失神还没开始剥陈秘的衣裳的时候。忽然听到吴痕叫，五子。吴痕在陈秘的桌子敲了半天的手指。最后敲出一句。下次别迟到了。

　　中午休息的时候五子打开邮箱收发Email。没有灯花的信。五子感到有点失落。这女人真会吊人胃口。五子想着就有点兴奋起来。陈秘刚吃过午餐回来正拿面化妆镜补妆。陈秘的妆是随时随地需要补。

　　有时在五子眼里看来别有一番风情。陈秘这会儿正刷唇红呢。五子就把陈秘想成了灯花。五子有了冲动忙给灯花写信。在信里五子把灯花小妖精叫了无数遍最后还意思意思的把自己的那点冲动隐晦曲折的写了进去。五子自觉热血沸腾余情未了。正待再附一小首情诗上去。上班时间却到了。最后总算急急添了一句，吻你的眉。香存我唇。不用说，这灵感是自陈秘身上得来。

　　灯花是五子的网上情人。一次五子去了常去的聊天室。谁知那天聊天室里女人特别的少。差点就成了单性乐园。男人们开始从女人的感情侃到女人的身体五子插在里面妙语如珠舌长三寸左右逢源将场面抖的比大家争美眉还要热闹。这时就进来一个叫灯花的人。开始在屏上刷征情人和条件。当时男人们以为又是哪位哥们穷开心谁也没理会。五子就上去打了两个字：应征。也是当作玩意儿的意思。谁知那位却是真正的美眉。后来五子就领会了网恋的滋味。灯花纯属妖精型女人。常常几句话就把五子摆弄晕了。五子说，我的妻子是妇产科医院的一名冷血，多疑，充满不耐烦情绪的护士。她从青春期就直接进入了更年期。灯花就嘻嘻的说，你的情人灯花能给你所有你妻子不能给你的东西。五子听了心就嘭嘭不争气的血脉旺盛起来。结果那一夜老婆对五子的服务就很满意。

　　现在五子半死不活的回到家里。那句阳光照耀我的破衣裳的歌儿是半句也哼不上来了。应付了晚饭。应付了老婆的唠叨和老婆妇产科医院里鸡零狗碎的狗屁事。应付了儿子的功课。应付了家务劳动。最后最后。五子终于可以和灯花小调一下闲情了。现在五子就美滋滋的坐在电脑前搜肠刮肚穷凶极恶的向灯花献媚。两人云山雾罩你来我往你进我退的玩着文字游戏。一个字。

　　爽。灯花是个很水的小女人。很容易就让五子有男人的自豪感。

　　有时五子也会有点使君有妇的憾感。这感觉就象五子常哼的歌阳光照耀我的破衣裳唱过了也就算了。五子毕竟不是那些缅于情爱幻像的小男人。五子明白灯花本来就是自己的一个性幻象。只不过恰到好处的显现在某个特定的时候。

　　灯花也许不过是个寂寞的女人。五子也许不过是个寂寞的男人。

　　网络成全了他们。五子在那天晚上忽然明白了。五子说，灯花我爱你。灯花在网络的另一端沉默了一瞬。然后灯花说五子我也爱你。就象老鼠爱大米。

　　阳光照耀我的破衣裳。

　　真正的阳光。真正的阳光照耀在我们身上。

　　阳光照耀我的破衣裳。

　　五子才哼了一句就被老婆可爱的小脚丫温柔的踹了一踹。

　　起来。做饭。送儿子上学。

　　五子呆了一呆。

　　五子眼睛就有点湿了。

　　五子抱住女人说老婆我爱你。

　　女人就娇娇的点了五子的脑门子说，傻样儿。

　　阳光照耀我的破衣裳。

　　阳光照耀我们的破衣裳。
