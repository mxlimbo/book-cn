风淡云淡的日子之并非网恋

散步的鱼 (07/03/2000)

　　上网很久了，网事风云一片片的从屏前飘过。

　　聊天也很久了，网友也如风云一片片的从屏前走过。

　　没有停驻。

　　就那么在屏幕前，风淡云淡的日子一一溜走。

　　然而还是有故事的。小小的不算故事的故事。无聊的坐在机前。聊天人的话语一行行的从眼前划过。我找不到想聊的对话人。我不想和任何人对话。闲着，就那么闲坐着看人聊。

　　有人来对话了，淡淡的应对过去。也不知与那人说了几句话心里竟然有一些触动。后来我发现仅仅只聊了三句。也许是因为夜。也许是因为，谁又能知道是因为什么？只是有点不同，那人仿佛同我一样。一样的感觉。于是聊了开去。谁也没问对方的姓氏及其它。只是讲着此时感应的句子。一句句的深怕没有明天似的说了下去。我知道此时是危险的。这样突然的情感呵。它来的如此不明也会去的如此不甘。只是抓住此时吧。只要此时。那人也是如此的对应啊，明天，我们都知道这一时的相契是多么不易。也许明日我们就会忘却今夜这一时的深深的动情。我们是多么聪明呵。什么也不问什么也不来不及问的只是讲下去，急急的，切切的讲下去讲下去。这一时是多么短暂而我们将要付出的感动将会是多么长啊。我们感动着对方也为对方感动着。分的那一一刻终是来了。我们都知道它终是会来的。只是那么一句句恋恋的抢着。对将逝的作着徒劳的挽留。结局终是来了。线路如此轻易的解决了这一问题：异常离开。

　　这是一个小小的不算故事的故事。

　　在风淡云淡的日子的背后。

　　我想，之所以恋恋不忘的是：这是一幕没有结局和开始的网路相遇。我们将永远不知对方是谁而将长久为此感动。

　　我想，这也是我们所希冀的结局。
