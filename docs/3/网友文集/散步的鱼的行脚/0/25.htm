风淡云淡的日子:鱼网情网和散步的鱼

散步的鱼 (07/12/2000)

　　这依旧是风淡云淡的日子里的故事。
　　我是那只散步的鱼。
　　象许许多多的故事中的一样：那个人。那个你要遇见的人。总是在你最不经意间遇见他。

　　就是这样。
　　我遇见了他。
　　那个聊天室开始在眼中变得模糊而感伤。
　　总是在夜晚进入。
　　不会去找人对话的。只是为了要看那人的名字。
　　你总是这样调皮而又处处留情吗，在网上？那人问我。
　　那日，那人把深入我心的名字换作了：鱼网加情网。
　　是想网住你。那人说。

　　<诗经>上有一首诗。我在心里默默的念给那人听：生死契阔,与子相悦。执子之手，与子偕老。这是不对当时情境的，然它却是萦绕于心。太过悲哀的一首诗。我们无语以对。
　　交谈亦是很少的。
　　他知有我。我知有他。
　　我们是真正的网上恋人。我知道。我们永不会去寻找对方抑或探问对方的讯息。这是永远的爱然却是多么悲哀的爱呀。那人说，我愿几十年后，仍能在网上看见你的名字。
　　不知如何去应对。我低低答，是。
　　是太过聪明抑或太过愚笨？也许我们都不愿去追问。
　　在这个虚拟的世界里。

　　只是在每个静夜中默默等待。
　　等待。
　　鱼网情网和散步的鱼。
　　相遇。
