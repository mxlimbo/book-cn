被鱼淹没

散步的鱼(9月27日11:0)

　　一位女孩义正言辞的指责我：你盗用我的名字。
　　我的反应很不如人意。
　　我是狼狈的退出聊天室的。
　　虽然背后没人指着我萎葸的后影说，切~~~~又是一条鱼。
　　
　　不知什么时候起。
　　网上渐渐泛滥了鱼。
　　
　　可以理解鱼们都是一些敏感动人的女性。当然也不乏雄伟的男人穿插其间。
　　这时也许会吓一跳，男人也可以是鱼吗。
　　
　　作为一只鱼整日游洇在网里。不知为什么那么多的鱼甘心甘愿的被网住。
　　由此我坚定鱼是女人。
　　鱼是一种苍白美丽的动物。
　　
　　悠游并挑逗着。我拧着纤细丰腴的腰尾我没有思想。所以你不要对任何一种鱼抱有绮思。
　　散步的鱼不属于任何一种情态。
　　
　　我刻意的做鱼。恣意卖弄着清韵清欢。
　　云淡风淡里有一些老片子发黄的美丽。
　　花开始漫天飞扬。
　　男人和女人披着湿漉的长发在花中穿插过影。
　　
　　可是鱼们总有那么一些动人的小伎俩。女人在性幻像里爱把自己扮成小鱼。女人也许会对她的情人说，你是我子宫里的一尾鱼。
　　原始的欲念成就了鱼。我们渴念器官如鱼儿欢跃的本象。
　　
　　轻易的洞穿了神明以及我们的信仰。
　　所以我们很轻易的做回了鱼，漾在温润的欲望里啃啮和倾听地心深处隐隐而动听的悲鸣。
　　惊爱鱼的闪着幽蓝发亮着的小白牙齿。
　　
　　我是一只鱼。
