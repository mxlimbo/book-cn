风淡云淡的日子：踮着脚的花

散步的鱼 (07/04/2000)

　　当我还是小孩子的时候
　　你拍着我的肩要我快快长大
　　当我不再是小孩子的时候
　　你被我的青春惊吓
　　青春鸟儿飞过来呀
　　我变成了一朵踮着脚的花
　　我说
　　等着我呀，变成你的那个她

　　那人的名字。叫，阿。

　　是的，他叫阿。阿总是拍拍我的肩：小鱼儿，怎么还没长大？然后他就走啦。

　　阿不知道我已为他开成了一朵踮着脚的花。

　　阿的世界是广漠的。世界外有一朵踮着脚的花。

　　我不知道阿为什么拒我于他的世界外。而我总是费力的想伸展入他的眼界内。阿。你有没有看见，看见一朵踮着脚的花。我总是在阿静寂的眼瞳里看见一支小小的可怜的踮着脚的花。阿摸摸我的头。不再说话。

　　阿总会被一个又一个的女孩捕捉。但阿总能逃离一次又一次的情劫。我说，阿。看着我吧。阿用被惊吓的眼神看着我。阿走了。

　　阿说过，有一个她。在他心里。

　　我想变成那个她。

　　虫们不知躲在哪里喁喁情话，最亮的星子已经害羞啦。我说，世界已经如此如此的薄弱了。不要再悬心自己的重量吧。

　　阿，选择一个夏日私语吧。等着我呀，变成你的那个她。

　　阿，记住某年某月某天某人的话：等着我呀，变成你的那个她。
