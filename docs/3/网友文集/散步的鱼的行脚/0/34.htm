这个时代是后镜里公路的风景

散步的鱼(9月8日19:40)

　　这个时代是后镜里公路的风景。一切都发生的那么快。

　　再动魄不过的人物风云，也如风云般一一从眼前游走。

　　悲喜着自己的悲喜。人生着自己的人生。不说如戏，本就是戏剧人生。

　　天空被高楼分割的破碎支离。都市里的丛林恐怕上帝也会无能为力罢。

　　这里找不到自已的戏份。

　　在淡漠的精致眉眼下。情感着游戏的情感。深恋着游戏的深恋。

　　颓废是被踏在脚下的名词。人们在上面彻夜狂欢。

　　世界宽敞。

　　这里是夜夜爆发惨笑的房间。
