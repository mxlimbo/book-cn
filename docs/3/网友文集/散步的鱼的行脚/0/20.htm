想哭就到我怀里哭吧

散步的鱼(10月23日11:29)

　　记得当时年纪小。你爱谈天我爱笑。有一回坐在桃树下。我们不知怎么睡着了。梦里花落知多少。

　　她采撷一支月季花。被躲在花心儿里的虫子咬着了。她举着手指吓呆了。

　　他走过去看到她瘪瘪的表情。好啦。他说。她的嘴巴已经扭得象只鱼了。想哭就到我怀里哭吧。他皱着眉说。她这才哇啦哇啦的哭起来。顺手还抹把鼻涕搽在他的新衣服上。

　　最后她脸上带着眼泪冲的泥巴道道在他怀里睡着了。

　　她的成绩很遭糕。老师就常训她。同伴们都走了她还在罚站。他帮她收拾好文具。抚平被女老师揉皱皱的功课薄。你呀。他走到她身边拉她坐回座位上。她忍了半天的洪水终于在他怀里爆发了。

　　他在那天夜里向女老师的窗户扔了半块砖头。

　　他有了一个小鸟依依的女友。她也有了可以小鸟依依的男友。一天晚上他送他的女友回家。在后门口遇见穿着月白裙子倚在桃树下的她。她的眼睛在月亮下一闪一闪的象沾水的星星。你怎么了。他的衣服又遭秧了。她的男友当大兵去了。

　　她走的时候他唤住她。他指指湿了半边的衣袖对红眼睛兔子的她说，明天可以帮我洗衣服吗。

　　他结婚了。她也结婚了。

　　他们回家看望父母的时候他们的孩子就会一起在院子里玩泥巴。

　　她和爱人下岗了。她开了一间小餐馆。他从那里经过。几个小流氓在里面起哄调戏她。他进去三拳两脚把他们压服了。他帮她收拾打烂的摊子。她的背有些佝偻了。他抿抿散乱在她额前的发。她扎在他怀里抖的象片叶子。他的心也被她淋湿了。

　　他瞒着爱人省了些钱塞给她的他。

　　他的老伴去世了。孩子们也都走了。她有时也会上来陪陪他。

　　她的老伴去世了。孩子们也都走了。他有时也会上来陪陪她。

　　有一天他对她说。我们在一起吧。

　　她看着他。她又开始下雨了。

　　他的孩子不同意她。

　　她的孩子也不同意他。

　　他们说。你们都已经年纪大了。

　　他的眼睛就很寂寞了。人老了。老了有一天也就去了。

　　她倒是再也没哭过了。她经常可以听到他在她耳边说。想哭就到我怀里哭吧。

　　有人在我们耳边说。想哭就到我怀里哭吧。
