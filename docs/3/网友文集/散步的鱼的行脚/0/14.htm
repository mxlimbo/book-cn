我不是人（二）　计程车惊艳

散步的鱼(11月6日15:45)

　　朋友老乐是个计程车司机。

　　因为听人说计程车的鬼故事有很多。特别是在夜间开车做生意时很容易碰鬼。所以在一次聚会里大家闹着老乐讲来听。老乐这回却一反常态避而不谈逼急了就讲个荦笑话应付过去。要知道老乐一向是以诙谐闻名于众。这样的躲闪开始大家以为是这小子又一挑兴的手段。可是后来看下去实在不象。在大家的轮番轰炸后。老乐终于给大家讲了一件事。

　　大家知道和平路那个桥洞吧。有个开车的哥们曾告诉我有时开车从那里经过会看到一个穿红衣的女人站在桥洞边。车开过去后就不见了。有时会看见一个婴儿包裹的好好的在地上放着。可是停下车去看。地上什么也没有。当时我听着就觉得这哥们是喝高了在那儿吓我呢。谁都知道和平路是我常跑的道儿。一天最少也跑它几十趟。别说女人和小孩了，连点风吹草动都没见过。不过不能说这哥们的话对我没产生影响。有时从那桥洞经过我也会左右看看加快车速超过去。我自己也觉得好笑。后来再见了那哥们我狠狠克了他一顿。

　　那天早上下了一场雨。应该是五点吧。水泥路面积了一滩滩的水渍。秋天了。有点凉。本来这么早不会有什么客人。可是转了两圈后有人拦车了。是个年轻的女孩。

　　我不知道其它的哥们是不是和我想的一样。反正我挺喜欢拉年轻漂亮的女孩子。她们往车里一坐。空气都不一样了。我开着车车上的女孩就想想她的心事或是看看窗外也有补补妆的。有时还会聊上几句轻松的话题。总之那种空气让人觉得舒服。这是个安静的女孩。我突然有了想和她说话的欲望。我说小姐你早。她没有回答。我从后视镜里看到她似乎心不在焉。我笑笑。也许是在想她男友吧。女孩在和平路下车。是在那桥洞边上。我经过桥洞回头看。她好象在等人。

　　后来一连几天都遇见女孩。当然我总是刻意在某个时间发车。女孩很瘦皮肤有些黑。举止间都怯怯。她一直穿着一条白睡袍似的裙子。纤尘不沾。这是难以抵御早上轻寒的微风的。不冷么。但是我想女孩都是要漂亮的。你在等人？女孩久久才回答，是。女孩的脸上慢慢生动了一朵微笑。转瞬既逝。我有点呆。我常常从后视镜里偷偷窥视女孩。后来我把件事想象成约会。每天早上女孩在等我。这个秋天的奇遇是属于我的。

　　讲到这儿老乐是怎么也不肯讲下去了。无论大伙怎么威逼利诱。老乐只是闷头喝酒。我注意到老乐的手有点微微发抖。老乐往口里泼了一杯酒后忽然哭起来了。她的脸。老乐说。这家伙哭起来可真难看。

　　那天女孩上了车就开始咿咿哑哑的自言自语。声音太轻。大意是我一定要等。我要回去了。后来女孩只是重复，我要回去了我要回去了。老乐注意到女孩的脸色非常难看。白腊腊的。女孩在桥洞口下了车。老乐悄悄的跟住了女孩。女孩在桥洞里来回徘徊着。轻盈的身体和雪白的裙在黑黑的桥洞里就象一个美丽而悲伤的幽灵。

　　时间一分分的流逝。女孩的背影也越来越绝望。最后女孩就象被折断的花枝在桥洞里蹲了下来。老乐强压着怜惜之心没有走上去。这时来了一辆车。老乐的表情就象有谁猛然掐住了他的咽喉。喉间只发出吱唔的卡卡声。车从女孩的身体辗过。没有血没有残破的肢体也没有呻吟。有风从桥口吹来。老乐才觉到自己在FF发抖。女孩从地上起来走向老乐。轻轻的脚步却是砸在老乐的心上。老乐挪动不了。只觉得背上的汗水在一道道的流下来。老乐只能眼睁睁的看着。心里翻来覆去的说我流汗了我流汗了。女孩的脸渐渐变了。老乐最后只看到一个骷髅。骷髅上蒙着一张面皮。那面皮仿佛还随着呼吸微动。接着居然发出声音来。一股浓重的腐肉臭。他为什么不来为什么不来。这是老乐最后听到的声音。老乐咕咚一头载倒在地上了。

　　他为什么不来？

　　现在老乐很少跑早车了。老乐现在跑平安路。
