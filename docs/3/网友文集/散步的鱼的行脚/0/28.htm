你知道是谁在背后打量你？

散步的鱼(9月8日19:51)

　　鱼在一根头发上打了三个结。

　　这根头发后来被绕在黑的喉结下的第一颗纽扣上。

　　是蛊术吗。黑勉强的挑起一点气氛。很勉强.木屋的烛焰和长发如巫的鱼有一种诡密的色彩。鱼的眼光没有表情穿越了黑的身体盯在无垠的什么地方。鱼就是这样。黑想，鱼使我不寒而栗。黑开始打量头顶上木屋的构造。他发现陈年发黑的椽木上已出现了白蚁的痕迹。黑在不知所措的时候常用打量头顶的东西来掩饰自己。鱼显然知道黑的毛病。然后鱼的眼中就飞快的掠过一种幽怨的表情。

　　嘿，蒙马。鱼拉拉黑的头发。黑呻吟了一声捂住眼睛，别讲什么蒙马的事。我不是蒙马。我是这个城市的人。我他妈的在这个监牢出生也他妈的在这个监牢进火葬场。听见了吗？黑把一个很圆的烟圈喷在了鱼的鼻尖上。我走了，黑逃出木屋。鱼没有追出来。黑知道鱼一定是在编结蛇发女怪样的头发一边在想蒙马。黑觉得好象丢了什么在鱼那儿。我想不起来。黑拍拍突起的脑壳，这些天我不正常。不，是鱼不正常。黑想天下那么多漂亮的女孩。自己何苦老是来找这个古怪的鱼呢。黑总是想得很开并且为这一点骄傲。黑吹起了口哨踏着积水走了几步又忽然停了下来，我他妈的想不起我掉了什么在鱼那儿。

　　黑在游荡了半个市区后总感觉脖子下面有一块皮肤上灼热难当。黑终于发现那是一根十分长的打了三个结的头发。黑怪叫一声，哪，去你的。黑把那根头发扔进路边的阴沟里。果然，那块皮肤不再有被灼热的感觉。黑很高兴。黑决定找长三他们打通宵麻将。

　　黑在长三家门口遇见了倚门而立的长三的妻子。长三妻子探出头那种狐疑的目光分割凝固了黑骤然堆在脸上的笑。黑在心里暗自骂了一句脏话。但长三的女人突然张开难看的嘴唇尖叫了一声讨厌，长三给你们这帮狐朋狗友带坏了成天不回家昨天发工资一分钱没拿回来你说你们把长三藏哪个老鼠洞去了，我要去问问长三不要这个家了不要他儿子啦我知道他嫌我胖他不想想用脚趾头想想我是怎么替他操持家的长三这个狗敢不要我我扒了他的皮踏得他生还过来。

　　黑苍皇逃窜。黑想怪不得长三一副提不起来的死样子不过这也不能怪长三，他老婆也太胖太难看了。

　　鱼坐在窗前盯着黑的背景在一排排矮房子后消失。鱼解开长发开始编结。鱼每次在黑走后都是如此。鱼是那种廋不露骨的女孩，鱼是苗人。鱼终年穿着黑色衣裙。鱼是在一次敲错门事件中结识黑的。后来鱼说，推门进来时阳光托着黑那些光真亮呀剌痛了我的眼睛。和阿咪达讲的一样。阿咪达说我要遇见一个人他披一身阳光来带我走啦。

　　阿咪达说过，那人叫蒙马。

　　鱼有时会陷入一种神思恍惚的状态。这时女孩会张开廋廋的手臂迎接一种虚无的东西并且喃喃呓语，蒙马还是小时候就来了。鱼的眼睛里有一圈狂热的光环，蒙马骑在马上他真小我也真小我在山上挖草药蒙马骑马来了来了把我带走啦。鱼细心的编结好最后一根发辫.女孩身上有一股青草甜美的气息。鱼的头发总会被编结成很多黑人种的发辫。这些发辫在鱼身上显现出了一种奇特的异族的美。

　　城市的雨季因难熬而漫长。黑在雨季里百无聊奈象一头被关在笼子里的小兽。朋友们都不知道跑到哪里去躲避这个长得没有尽头的雨季了。唯一知道下落的是长三。长三终于有一天被他妻子从麻将桌上抓回家。长三揉着布满血丝的眼睛手里始终抓着一张麻将牌。长三的妻子在恶声恶气回答黑关于长三的询问时难看的嘴里吐出的脏字并没有妨碍黑看见躲躲闪闪藏在门背后的长三。后来黑说。我看见长三的眼睛就知道长三是完了。长三有一双羊眼睛。当他固执的盯着你看时乞怜的目光让人鄙薄不已。黑想，世上那么多的眼睛长三你怎么就长了一双羊眼睛呢。

　　或许是因为雨的缘故。黑想雨总是使他的心情处于一种极端烦燥不安的状态。重要的是你不知道这种烦燥来自何处。雨季开始的时候,黑重新出现在鱼的木屋。我知道你会回来的蒙马不喜欢城市的雨。鱼在这个雨季的首尾出落的美丽异常。雨水洗去了女孩身上来自家乡的气息。鱼和鱼的木屋使黑更加烦燥不安。因为这个原因，黑频繁的出入于鱼的木屋。

　　黑在夜里总会被一只鸟惊醒。黑看见一只被雨水淋透的鸟儿冲破雨帘跌在他的脚下。他怎么也无法弯下腰去拾捡鸟儿。鸟儿细碎的叫声就象是一个女孩在哭泣。黑想不起来鸟儿身上那股熟悉的气息来自何处。然后黑就被惊醒了。

　　故事到这儿也许就完结了。

　　世上已有太多没有结局的故事了。

　　因为，你知道是谁在背后打量你呢？
