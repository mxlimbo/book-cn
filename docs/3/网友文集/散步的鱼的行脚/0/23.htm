云淡风淡的日子：圆舞

散步的鱼(9月8日19:48)

　　你是落入凡间的星子，墨墨雨夜里。阿说，太张狂。

　　阿费力的在我的眸子里泅泳。我知道。

　　在某些夜晚我喜欢长时间伫立在镜前。

　　因为我觉得我很美丽。一种纤细的美与残酷的美相替出现。这时我会用一具恐怖的面具遮住面容。待恐怖的面相定型后，面具将被缓缓移开。

　　他们想像不到丑陋的面具后会是这么一张美丽的脸。阿说。

　　阿捂住眼睛长长呻吟一声。

　　捂住眼睛的阿脸上有一种幽怨的小男孩的表情。你深知这种表情多么引诱女孩们的怜爱之心。

　　回想起来，阿和我的相遇无疑是被蒙上了宿命的色彩的。

　　一如黑与白之间存在数千个深深浅浅的灰色。

　　我不喜欢那个聊天室的但我还是进去了。进去就遇见了阿。

　　说到这儿我想起了扑火的蛾子。她知道投身入火的结局但她终究还是入涅槃之境。和阿对话。有一场小小的美丽的火灾发生了，在我的心里。我不知是该额手称庆还是感伤一叹。

　　一切一如传说的再版。平淡的对答。然，是不一样的。

　　我有一种惊心动魄的美。眼里撒落前世的尘灰。

　　有一些什么在向我们悄悄移近。在世界的静夜里。

　　我是一个女巫。就那么随意的叩开了阿的三道心门。却全然不解那手式已沾染了注定的烙痕。

　　命定的有一些人要用一生去诠释一个玩笑或是表演一种舞蹈。

　　我和阿无疑也在此列。那是圆舞。自有着舞的游戏规则。相对的舞者必须不停的旋舞。直至最后时分疲竭的相遇。两者历经了无数的圆。圆舞。他们把它称作圆舞。作为生命的舞者，应不堪于所有类似于此的美丽而残酷的嘲笑。讽剌。我说。莫大的讽剌。然而又是莫大的成功。生命的骗局，你总甘心涉入。

　　我看见阿躲在思念的空气里迟迟不出。

　　入夜出门。

　　有一种奇特的了解。对于阿。我对阿一无所知却仍然能够懂得他。因为懂得，阿说，所以慈悲。你不懂某些事，但绝对懂某些情境。

　　在某个时间停止流动的空间。

　　有人在我耳边唏嘘：我的，鱼儿。

　　他是阿。

　　我只有这样一个名字。在世上。

　　我告诉你们了。

　　我是鱼儿。阿的女孩。

　　我对阿的故事无动于衷。我深信我们不过是在完成一种仪式。

　　仪式里阿在那么一个时空交错的地方忽然现身于我眼前并试探我的惊喜。

　　阿捂住眼睛呻吟一声，鱼。

　　阿会这样做的。

　　事实上阿就是这么一个混蛋。

　　时间是一头咻咻喘息着的兽。浮欢也已是声竭力嘶了罢。

　　阿。你们都听见了。阿的女孩就是这样呼唤他的。

　　最后最后。

　　世界也许需要一次近似死亡的歇息。

　　于是我们所有云淡风淡的故事都结局在云淡风淡的喜悦里。

　　我们都是舞者。
