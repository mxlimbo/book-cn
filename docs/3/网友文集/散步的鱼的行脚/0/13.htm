我不是人（一）　心魔

散步的鱼(11月6日15:45)

　　她在一幢房子里关了五个月。

　　偶尔倦怠的看看窗子外的天空。一种灰凝的蓝。很少去室外。

　　庭院里有一只瓦缸积着一些青苔。她有许多鱼。有时额抵了圈沿向里张望。水中是模糊的人面还有模糊的鱼影。她从不为鱼换水。水是一种窒息的惨绿。

　　手中多了一把犀角小梳。姑母带了朋友来看她。角梳里带了几丝红纹。不会是犀的血呢。姑母和朋友在一旁细碎的聊着天偶尔也带了爱怜的眼神看她。可喜欢？

　　姑母轻声问。她用梳子缓缓划下长发。喜欢。她说。为姑母端来茶点她依旧倚在缸沿边观鱼。也不过是喜欢。喜欢代表什么呢。水中两尾鱼摆了硕肥的尾翅追逐。

　　那女孩子怎样了。

　　怎样。也不过是等日子罢了。

　　哦。唉。真是做孽。

　　她母亲去医院守夜。夜半还听见她房里有动静。

　　她没守在里面？

　　不敢。说是里面房间小。挤。

　　…………………

　　说是听见那女孩子说，你向外让一让，让一让。

　　姑母向身后张了一张，啊哟哟，惊煞人了。房里不是没有人，她对谁说呢。

　　就是呀，还不是那…………

　　这女孩子完了。怎么招上这邪魔。

　　眼神都不对了。见了她哥哥硬要往床上拉呢。见男人就把衣服揭起来，还向人笑。她父亲气呀，心脏病都发了。

　　啧啧，请个半仙呀。我知道有个张仙儿很灵光。

　　请了，不管用。说是那邪太凶。制不住。真是。

　　真是。

　　姑母她们在说些什么呢。她是漫不经心的。任何事。把世界都挡在她之外了。可是，那个女孩子遇见了什么呢。一个男人？邪魔？姑母说那女孩子在变瘦在慢慢死去。死有什么可怕的。她扯了一片叶子扔在鱼缸里。鱼群都逐逐围上来了。

　　夜半她醒来。刚刚的激潮还未过。面上晕红她有些气吁。她记得她的身体曾一上一下起伏的送迎。她好象变成了一尾鱼，游过肌肤的水波都是一种欲望的突剌。周身都是懒松却畅快无比。从未历经过。她觉得兴奋莫名。抚着自己发凉却灼烧的身体她慢慢的把手伸入了身体的隐密之处。她要再次品尝。

　　你瘦了。姑母说。她微笑。姑母摸上了她的面庞。象一只狭长的桃子却渐渐失了水色。你有了黑眼圈。姑母从女孩的身形嗅到了什么。你遇到了什么孩子。告诉姑母他是谁。谁把你变成了女人。

　　天井里有一只积了斑驳青苔的瓦缸。里面有许多鱼在游。它们很快乐。

　　姑母请了法师来作法。经幡。香炉灰。桃木剑。黑狗血。她的小小庭院遭遇了一场小小劫难。法师点了一串纸钱走来走去口中喃喃。最后抓了一把空气放在一只小泥坛里带走了。姑母安详地张望着。她在姑母的眼中看到了与她类似的欲望。夜来了。姑母终是没有留下来陪她。你的鱼缸里有一只鱼带了籽，可能要有小鱼出来了你当心点。姑母说。她走过去从鱼缸里抓出那只鱼。她的手握的很紧。一只鱼尾在激巨的摆动。后来有什么东西顺着她的手腕滴下来。她的手中没有动静了。

　　你的肚子！

　　她低头爱抚着腹部。你看到了。她笑。他就在我身体里。我很高兴我的爱人将循着我的身体来到这世界。

　　你们都听见了姑母的尖叫。

　　她步入黑暗中的房间。

　　爱人，我来了。

　　她的手指缓缓向身体的最隐密处伸去。

　　爱人，我来了。
