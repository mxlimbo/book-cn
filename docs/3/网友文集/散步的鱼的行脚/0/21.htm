另一种夜晚

散步的鱼(9月8日19:40)

　　很夜很夜了。清清玄玄。

　　夜深闻私语，月落如金盆。

　　无月。只有几枚星子。

　　亦无人。私语我自知。

　　那也是浅浅的夜了，夏的。归来。对过的窗内闲闲的散出一些音，纱的窗里是黄昏的灯。一种异的情调，一个人埋头拨着怀中的木吉它，身前摊一本曲谱。一向是讨厌吉它的。音质浮燥而浅，以为。但从那人指端流出的不知掺和了什么的也不知什么名目的音却叫人动情。即使盲音如我，亦知那人，年青的，并无十分的技巧。但却使无心的归人如我，动了心。在那窗前呆立些时，才折身走入家门。倚身在门后，有细细的喜悦。

　　数年前的事了。忆里的一朵剌青。时时浮出的。如今亦有一把吉它。闲时拨弄，不复那夜那时的情怀。只是有时猜想，会不会亦有闲人如我，立于窗下听琴？但我知没有。只是想想罢了。

　　旧言说，抓住此刹那，便为永恒。也曾用心细细把刹那变为文字变为永恒，但终究怀旧之心永恒而字纸刹那。生命中遗失错过太多太多，抓住一个刹那，即使永恒，那便如何。

　　只是有时，在特别的静夜里，回想一些刹那的永恒。那是美得令人心痛的。
