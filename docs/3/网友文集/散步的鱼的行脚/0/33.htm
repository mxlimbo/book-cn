第四类接触

散步的鱼(9月8日19:40)

　　似朋友似情人。

　　不是朋友不是情人。

　　第四类接触。

　　在网络的虚拟世界里。

　　情感游戏。

　　我还是那只朴实的散步的鱼。终日昏昏醉梦间的在网里散步和游走。

　　间或遇见我想遇见的和刻意去遇见的人和事物。

　　听起来很是无聊。事实也是如此。然，也不尽然的。并非无故事。总是一些云淡风淡的日子里的故事罢。

　　桑是聊天室里很受欢迎的男人。当然是对女人来说的。风趣带点儿贵族气带点儿野气再沾上点儿诗气。总之，凡是能让女人心动的都有那么一点儿吧。鱼除了散步之外当然也没闲着，在尽可能允许的范围内也谋略过试图吸引桑的注意。接触总是接触过的。也就是如鱼轻啄水草式的，非是情人非是朋友也似情人也似朋友。在云淡风淡的日子里也只有这种云淡风淡的方式才相配罢。也只有在这种虚拟世界里才能产生和感受这种第四类接触的情感游戏。于是云淡风淡的日子开始美丽着云淡风淡的美丽。风情着云淡风淡的风情。

　　听起来很炫。然，故事也就止于此了。

　　没有其它。这只是云淡风淡的日子里的一个如鱼一样普通而又朴实的故事罢了。

　　我依然是那只散步的鱼。终日昏昏醉梦间的在网里散步和游走。

　　间或遇见我想遇见的和刻意去遇见的人和事物。

　　在所有云淡和风淡的日子里。
