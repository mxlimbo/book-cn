致春儿：愚兄之处可为家！

春妹少年展风华
云游四海逸天涯
我乘锦霞逐伶去
大洋彼岸逢秀颊
豪叙京城旧友事
默语往日书云雅
切恨如是谋面晚
白宫草坪羞无花
归途小栖洋中岛
独踱海天思梦暇
无奈越海再聆语
不解忧念更牵挂
冬日喜得雨燕归
夜游古都哙肴佳
寅时尽兴意未倦
相敬无言屋檐下
无奈候鸟知时返
此后冬驻再无夏
近闻春妹归故里
愚兄之处可为家
