阿熙的故事——答绿野仙踪

　　我可是唱着欢乐颂走来的，决不是为男孩子的事情和谁不征战。再说我也不属于男孩子堆里的啦！没啥可争的，俺！

　　没错！有许多很好的中国大陆的女孩，漂亮聪明，敢于挑战生活。她们魅力四溅，风采怡人。在世界的各个角落你都能遇到，我走过30多个国家。住过工作和生活过的地方有五六个，但我一直保持着一种直觉，就是一眼就能看出来中国大陆出身的女孩。哪怕是在几百米之外。

　　每每当我在离开中过的土地的时候，总是保持着一种意识。那就是远离中国人，去看去理解异国的文化和人情。我总是独自旅行，独自生活。用心去读这个烂漫多彩的世界。因此不敢说对那些嫁给鬼子的中国女孩有太多的了解。一点小故事，不知道是否能够找到一点她们的感觉。

　　《中国女孩的故事－－阿熙》

　　阿熙可以成为一个出色的外交官的。她有过人的直觉和用心对话的能力。出众的身材和用不尽的精力和热情。在一个驻华外交官的卸职的招待会上，她走过来告诉我她终于得到签证赴欧洲啦！祝贺之后，我感谢她多年来对我的业务支持和帮助，并告诉她我将在半年后去那个国度工作三个月。我将愿意尽可能在那里给他必要的帮助。包括免费住在我的沙发上，如果不在意的话。在我说到最后一句时，她收起了职业式的笑容，说了一句：我在意。她踏着对美好外来的十足信心的脚步，回头走啦！

　　………

　　她来啦！还是那般神采飞扬，但是你用眼角看她时，会发现一脸的倦容。我签完了她拿来的文件，抬起头正视她的风采祝她签证延期成功。她果然获准了新的签证，因此护照上也被写上了汤先生的雇员的字样。当然，这不是真的，我并不需要这一雇员，可是我不能说慌，这是信用社会的基本法则。因此她在我的办公室上一周的班，然后请她辞职，并把通知移民局的文书放到秘书该看到，但又一定被忽视的地方。

　　 ………

　　欧洲人其实也是碎嘴子，一个中国女孩仅仅一周工作时间就能给人留下那么多的印象？实在是难以让人理解。我总是可以听到关于她的近况，重来不是我主动问的，当然也没有人主动的和我说。下午茶加上中国式的谈话和旁听的技巧让我几乎是对她无所不知。毕竟曾经是我把她带到这个办公室里来的。

　　 ………

　　圣诞节是个家人相居的日子，长长的假期。没有人再去理会我这个外国人。只有出去旅行啦！临行前，也就是圣诞夜她应邀敲响了我的门，一顿久违了的中国饭过后。她拿出了礼物，一个德国人作的闹钟。并讲给我办公室里的那一周我无理迟到的滑稽故事。让我笑个不停。

　　………

　　我拿出了我给她准备的礼物，一个信封装的一把钥匙和房子的租约。那是我租下的一个小房子，已经付了租金，本想退掉大房子省点钱，可又改了主意，舍不得住惯了的地方。这份礼物是很难被隔着那个漂亮的信封被猜出来的。信封上写着：送给卖卖火柴的小女孩。不管是啥样的女孩，面对礼物永远是要激动的。当她打开信看到她的礼物时当然是激动。她问我能让她哭一会儿吗？我说，不能！！你那办公室里著名的灿烂笑容要是在我这里丢失啦。估计我会被这个国家驱逐出境的。她笑了，……。

　　那夜，她睡在我的沙发上。醒来时我已经跨过了两个国度，开始了我的旅行。

　　………

　　她带着她的两岁的儿子和老公来看我。中国人和欧洲人生的混血儿就是很漂亮，那儿子，那老公都好的无可挑剔。老公送我一个烟斗，并告诉我她护照还没有换新的，还可以看到汤先生雇员的字样。当然，我知道那签证早过期啦！是以本可以换该国护照的婚姻理由，在延续着中国护照上的拘留签证。
