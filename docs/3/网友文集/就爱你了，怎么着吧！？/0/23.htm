就爱你啦，怎么着吧

　　写小说，人家写的那么过瘾。拿别人的事，讲给别人听，还根别人要钱。到我这就不是那么会事儿了，跟死一回似的，我真真的没这瘾。不到非得生一把死一把不行的时候，决不写这些。

　　今儿，死的可真有点大发了，三个女人一个孩子把我从车里架出来还是抬出来的，我到现在也没搞清楚。如何处置我，他们在商量，还征求我的意见。我实在是死了，眼睛实在是一点也睁不开。魂没散。我敢保证一点也没散。

　　我听的清清楚楚，"还想吐吗？胃不舒服？"其实那里还能找得着胃。"上谁哪去？我姐姐家还是我爸爸那里？"我指了一下老爷子，可话还是说不出来，只能在心里说。"他那里是九楼。"我当然知道那里是九楼，早晨就是从那里走的。我也当然知道她是什么意思，可只有他是一家之主。不去他那儿，去哪儿？我还知道这所有的人中数她最男人，敢做敢为。可又敢说把我接回家去？也怪！没人说把我送回家去。好像人人都知道我没家似的？每天睡在垃圾箱里？早晨夹着大哥大上班。可我好像没求谁接纳我啊！真有点让人伤感，就为不知道如何处置我是好。可人死一把容易吗？我愿意死吗？

　　说真的我愿意死，就是死不了。人说吃二十片定就有危险，我六小时前我一次吃了二十一片，剩下六片刚才也吃了。现在神采迥然。我从不信这东西可以夺命，我的实证结果是它能够安魂。对我极合适。最让我欣赏的是："二十片以上有生命危险。"特有刺激，我把它推荐给天下所有的熊男人共享，此物特相思。

　　我从不爱惜生命，有时一天吃六顿饭。有时就吃一顿，有甚至粒米不进。我实在是觉得，心、肝、脾、肺还有腰腿筋骨什么的，都是身外之物。常找不着脾了什么的，可我也不急，总觉得那些物件没什么用。我就受不了，没神儿，丢了魂之类的事儿。我真爱这东西？一到后半夜就来神。我自己有时也搞不清我是人，还就是个魂儿。人们都说女人是汪水，我觉得我顶多也就是点气？ 可能连气都算不上。所以我比女人还女人，也就是熊男人。

　　车门开了，不知道是到了哪儿。没睁眼，直觉被呼得掀了起来，像是要扔进火化炉，感觉很好玩。烧了吧！熊人！！我自语着。

　　"他笑了！看哪！！他笑了！！！ "

　　是的我笑了？ 笑的好开心？ 可还是睁不开眼。大家都笑了。

　　晚上先吃了那二十一片药，然后拨通了她的电话，就一句像样的话。

　　"就爱你了，怎么着吧！"
