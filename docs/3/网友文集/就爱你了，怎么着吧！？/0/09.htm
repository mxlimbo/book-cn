无题 -爱人 老汤批

天也高，风似逍遥，云在风中笑

海也远，雾似飘渺，花在雾中笑

我，我望着天，天似知道

你，你扶着海，海把你抱

向往在身边环绕，努力得到

自由在冲我微笑，张臂拥抱

好 云！

妙 花！！

豪 问！！！

够 浪 漫。

够 幻 想……

够 痛 快……

也无题 -锦瑟 老汤批

天再高,不任我逍遥

海便远,不及你缥缈

我的心事,天不明了

你的思绪,任海拥抱

我的自由要比天骄傲

放肆的海洋不配向我咆哮

不 胜 寒？

好

奥 妙

好 一 天 骄！

有 胆 气！！

再也无题 -老汤 老汤批

天不高，洽足我展袖逸逍遥

海甚小，可够你汲来伺墨宝

我情意，书尽天际仍未了

你心性，可奈往复于砚揉磨如潮

唤天骄，再掘汪洋水一槽

顿首问，不知您奈得？耐不消？！
