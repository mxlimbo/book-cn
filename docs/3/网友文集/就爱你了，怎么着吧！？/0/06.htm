《纯粹》- 和锦瑟的仿古旧作

纯粹 - 锦瑟

纯粹 -老汤

天
摇荡我的心意

地
包裹我的呼吸

风
拂乱了我的思绪

雨
冲洗我褪色的记忆

封锁了心
锁不住逃离
捆绑了双翼
绑不住背弃

那浮云万里
仍有我飞翔的空隙
那疆土无垠
仍有我跋涉的足迹
风再急
不能将我的信念连根拔起
雨再厉
不能浸透日月尘封的往昔

自然
任你变幻无穷尽
我
仍是这个纯粹的自己

天憾情意
地扼生息
风绞思绪
雨荡梦呓
枷锁心靡
捆缚叛逆
云载吾情
踏游千里
豪风无奈
骄雨泣急
天地之间
谁能胜己
