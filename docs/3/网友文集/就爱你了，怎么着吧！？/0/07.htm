挥霍温柔 - KIKI

挥霍温柔 - 老汤

既然上天赐我温柔
就请让我尽情挥霍
直到只剩结冰的表情
才算学会不再泪流

既然美丽再无寄托
就请让我尽情挥霍
直到镜中的脸孔都变模糊
才算学会了解脱

只是我还不懂
相恋的人们为何要不停争斗
难道只有放逐天际尝遍了苦头
才能体会真正的温柔

如果真要我懂
就先冻结我的笑容
藏起眼底的彩虹
让灿烂的星空
阴云密布

我是真的不懂
也不愿装懂
为何一定说回味曾经的拥有
才是幸福

放肆
因不忍泪流
任我
挥霍温柔

上 苍 赐 我
柔 情 似 火
我 却 期 待
如 冰 冷 漠

无 奈 挥 霍
泪 扑 炙 热
直 至 憔 悴
灵 肉 解 脱

依 旧 困 惑
征 战 为 何
伤 迹 累 累
爱 方 有 魄？

审 慎 斟 酌
音 容 如 漠
彩 虹 一 炬
繁 星 剧 落

问 天 何 为
我 亦 奈 何
难 舍 温 情
又 求 淡 漠

洒 尽 心 雨
抛 干 爱 诺
天 地 与 我
宛 如 幽 歌
