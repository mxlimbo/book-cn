《太阳的烦恼》-和锦瑟《月亮的下巴》

月亮的下巴 (锦瑟)

太阳的烦恼 (老汤)

我在数日前抛给你的一个微笑
你送还我同样的回报
可我还是那样的年少啊
怎么你却变得苍老

我在多年前抛给你的一个微笑
你曾是笑弯了腰
我枯索乏味的想像
因了你而纵怀逍遥

我在上一个世纪抛给你的一个微笑
你含蓄的点头看到
放任我游荡的思绪
在你的云层中幻无飘渺

我在前一个轮回中抛给你的一个微笑
你撩给我眨眼的心跳
于是我化作你近旁的一颗星
伴你出没隐现的暮暮朝朝

呵,我在今天给你同样的微笑啊
你的下巴沉沉甸甸的苍老
我问你黑夜中挂了太久的感觉
你说那是岁月的味道

锦瑟
2000年4月24日午

收到你昨日的微笑 
轻扬眉宇只是为了礼貌
漠然中再度被幽思烦恼 
为何你依旧清澈且无味道 

曾记得你旧日的微笑
如雨后的山花无限妖娆 
谁能抵御那美丽的诱惑
思绪在花间缠绵缭绕

梦中见到你的微笑 
如痴的我不知如何是好 
几分憨傻几分嬉闹 
醒来只能无奈把头摇

街边那个女孩象你一样对我微笑 
她向呆傻的我手中塞了一页广告 
我将它折成飞机 
让它带走你灿烂和我的烦恼 

今天面对你依旧如初的微笑 
我已不再被你的清纯倾倒 
你问时光都锤炼了你什么 
我说去体验久酿醇香的玄妙

老汤
2000年4月25日
