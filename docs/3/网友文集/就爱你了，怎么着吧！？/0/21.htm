皮蛋豆腐――杂食篇

--应邀为绒而傻女系列DEMO

　　PD飞了半个地球来找我，因为她陷入了深深的麻烦。爱上了一个不要娶她的人，其实她也压根没想嫁给他。所以把我搞的稀里糊涂。我咋也搞不明白她到底为啥如此忧伤。

　　好久没吃中国饭啦，这时候千万不要吃大饭店。就去找地道的小吃，才过瘾！其实我几乎天天在那里吃，也吃这东西，叫皮蛋豆腐。可我是吃完了皮蛋吃豆腐，要不就吃完了豆腐吃皮蛋。可她的吃法儿就不同啦。她把皮蛋和豆腐用筷子绞碎，搅合到一起，成为一种糊状的东西再吃。她要我尝她搅合的作品，我的第一反应是好恶心啊！

　　她其实一点胃口也没有，加上时差的关系和满头的烦恼及满腹的忧伤，能喝点啥就不错啦，吃是不行啦。我们在试图讨论一些问题得以关键的细节，探讨着事务原本的逻辑。可她就是说不出来真正的想要啥，可要是挖掘不出这东西。快乐和幸福，从何而论啊！？……。她在不停的搅弄着皮蛋和豆腐，黑与白越来越不清楚啦。这时候我才发现盘子角上我熟悉的葱末是要搅合进去的。原来我以为那只是点缀，搅合进去在黑白之间平添了无限的烂漫，真的很好看。

　　该作结论啦。尽管那是间昼夜服务的小吃店，可总不能讨论到天亮啊！其实问题已经说清楚啦。不想嫁，也无意娶。多明确啊。因此，这不是烦恼的来源，来源是看不见摸不着带来的无名之火！那咋办？把他弄过去陪读？男孩来作陪读？！！有点坑人，不是？你是牛市，涨！人家现在也是牛着呐。要是这一折腾还不给人家搞的跌停板，联打三个跌停板都收不住。那时候，也简单啦。砸手里啦！没得抛，非嫁人家不成啦！人家理由好简单，就爱你啦。咋着吧？您！……。得！！劲啦，不是！？老问题，到底要啥？情感！没错！！除啦撕大地(Study)用尽智慧，情感也得消耗啊！？可这不就在消耗着呐吗？不害痞(Happy)？！那是因为没学会享受痛苦(enjoysad)！并从中获得美感！慢慢练吧！……。

　　皮蛋和豆腐，被搅合成了青色，小葱依旧跃然。她气色好多啦，挖了一大勺她的作品放到嘴里后就大叫："真真的好吃耶！"。是的，从那一脸的灿烂中已经看到啦！

　　是吗？！！其实好简单啊！？就象歌里唱的，没有黑也没有白，没有恨哪来的爱！？就是说俩字儿－－搅合！！就齐啦！

　　……。
