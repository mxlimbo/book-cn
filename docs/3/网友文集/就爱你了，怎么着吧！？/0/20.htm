北京新发现

　　北京，还是老样子！ 但也有新发现:)

　　ivy还是在东三环上班，在世界上有名的外国大公司里跳来跳去。Hook.走了，流落到英国，他自己总是要更正说是Scotland。后来又到了瑞典，过的不错！他说满街的金发碧眼的北欧女看着就高兴。

　　HOOK.走了，又来了个张小坏，小帅哥。玩帅大王！天天都是花团锦簇，芳草如云的！妄称网上无美女。我一糊涂就把ivy.介绍给小坏认识了！想震慑一下他。结果，小坏就摔着长发，跟连环画上的风云雄霸天下上的人物似的就出场了！那阵势要是没有帅呆了半个城，也得酷毙了一条街。ivy.乐坏了，从此有了一个好好可爱的小弟弟。……。再后来我也都不知道了！就是，时不常的半夜俩人从酒吧打来电话。要老汤去喝酒。老汤开车，只好咖啡了。然后把这俩宝贝一一送回家。

　　意识中老是把小坏当成Hook.，看来Hook.在我心中的印象是的位置是没人可以代替了！记得Hook.见女孩子就接巴，女孩子越漂亮他就越接巴。最严重的就是老马真把ivy.带来时Hook.竟然堆到桌子底下去啦。但是唯有一个女孩Hook.老腻腻的躺人家怀里，一点不害羞。大家称那女孩是Hook.二姨，就是白白净净喜眉喜眼的Demi.也叫大米。直到上周我才知道了一个秘密，Hook.二姨，大米姑娘才二十二岁。可她竟然如此落落大方好一个成熟的职业女性的样子。惊得我嘴张的有盘子那么大！
