那种美丽让我渴望,让我恐惧――回致爽爽

爽爽：

　　你说"突然想起《克莱默夫妇》，她难过时用小臂紧捂胃部"其实，该是他。

　　没错，霍夫曼本人就是巨蟹座的。巨蟹座的男人的特有的性格，构成了压力和郁闷的处事态度，而这也构成了胃不适的医学成因。因此，巨蟹座的男人多胃疾。统计和数学是真正好的一个工具。如果算命先生说"幼年有过大病差点没死对吧？"。这是你看着被算命的那人的天庭，也就是额头上两眉间的上方，一定是深陷的。因为小时幼年正是发育那块骨头的时候。内在的和外在的是相关的，这是我要说的。霍夫曼演好那部戏，也是他太理解角色了。LA的一个和他廷熟的朋友告诉我，霍夫曼也是巨蟹座。我比霍夫曼的表现要直接，我是直接把手掌按在胃上。因为男人的衬衣正好是左襟压在右襟上钉口子的。恰好可以在两扣子之间按到胃。我有时，就是这个样子的傻笑，据说样子很可怕！

　　真想去看看你家的那些马，我想不起来哪匹马应该是你的，他们好象都是男人的马。还有你家的那一大群狗狗，也看不出来那一个是你的。就象那里没有女孩子似的，联你胃疼的方式都按男人的路子来！还脱蹬骑乘吗？你真的比男孩还勇敢。

　　一切都是有统计分析意义的，可以在已经获得的信息的基础上。用数学方法，去织补出未有图象。你熟悉我家花园里的蔷薇和香椿该不亚与我了解你家的大狗狗们和健壮的马匹。望着它们扬起的烟尘，我完全能够想象出你有多美丽。完全不同的女孩，在驾驭着兽性的忠诚中飞扬着灿烂的神采。那种美丽让我渴望，让我恐惧！

　　必须更正两点：1、你没有结过婚就已经成为哲学家了，这已经证明德国人说错了。2、我不是你的爸爸，如兄如父式的爱不适合你。懦弱的女孩才如此期待。你的自信和勇敢给了你无限的魅力。你能征服一切你所爱的人性与兽性。如此待我仅仅是无奈的情感。决不是你真实的天性。

　　我不再反对你去德国了。别忘了沿着易北河和莱茵河走走，那里有我的足迹，更重要的是那里是日耳曼文化和高卢文化的发祥地。相信凝重的德国风光中你会感悟到许多哲理，那会让你的美丽罩上神秘光彩。

　　也许我们永远都不曾谋面，尽管你我对彼此身边的一切都那么熟悉。如你所说，我在圈内你在外。数学上你圈外的意义是无限的空间，任你主宰。

　　感谢上帝创造如此神奇！

小爷儿，汤
