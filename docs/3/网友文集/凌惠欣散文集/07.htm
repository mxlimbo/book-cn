梦回水乡

　　子夜，随风飘落的雨淅淅沥沥地敲打着窗棱，还有一颗游子的心。

　　有树叶凋零的声音，将湿漉漉的心灵扣响，沉淀了原始的呼唤，呼唤归途。

　　卸下夏日的疲惫，我返回水墨江南。那依水而居的崇山峻岭，绿茫茫一片。而触手可及的映山红，满山遍野地倾洒着醉人的芳香。

　　微风浮动的水面，有浮萍在轻轻地摇曳，晃动着一生的绿。出水芙蓉则毫不羞涩地绽放着，从夏荷的莲叶中走出来的渔家姑娘，在层层碧波中荡漾着笑容。有清脆的歌声划过水面，蜻蜓也在荷尖舞蹈。

　　水边的村庄啊，你是否也在静夜里聆听远方游子的呼唤。

　　夜，渐渐成模糊的轮廓。

　　早起的采茶姑娘，身披月的华衣踩着月光的影子，晚归的牧童，伴着晚霞坐在牛背上笑声阵阵。水田里的蛙鸣取代了虫子交头接耳的声音，岸上休息的人群唧唧喳喳地聊着千百年不变的收成。

　　为生活而奔波的绝不仅仅是，城市拥挤的人流。

　　执着于那片土地的人们，同样在自己的脚下努力耕种着新的收获。

　　故乡的那弯月浮上来了，月光浅浅地照在我的身上。我又感觉到了那熟悉的乡音，还有淡淡的泥土的芬芳。
