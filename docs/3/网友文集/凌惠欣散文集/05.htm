　　红月亮系列（序）

　　这是一种纯洁。

　　在云淡风轻的晚上，你婀娜而至，轻洒朦胧的余辉。因而灿烂了的月光河，在茫茫夜色里沉淀了孩童天真的笑容和对未来的遐想。

　　这是一段岁月。

　　在你圆睁的双眸里，倾泻了岁月流动的音符，铮铮淙淙。我看见飘逸的嫦娥拿着花篮，抛洒着缤纷的梦想。在水样流动的希望里，我们走过畅想中的时光。

　　这是一种柔情。

　　那远行的游子吹起悠扬的箫声，孤独的旅人借听萧声，也露出安详的微笑。这微笑又温暖了月光河畔濯洗乌发的牧羊姑娘,她黑亮的眼睛在柔美的月色下朦胧成心上人守候已久的红月亮。

也酿成了甜蜜的酒儿

放开自己的情怀吧
倾听夜莺的歌唱

也已哭红自己的明眸

红月亮
缥缈美丽的红月亮

生活的路上
不再布满荆棘
未来的希望
还在飘荡的温柔里回荡

结束一段过去
重新开始新的历程

　　月色正浓，却是分别的时刻。几个朋友前来送行，说的话儿，不过是些陈年的往事。絮絮叨叨，欲离难走。何时再有如此的明月，何时再见这些多年的老友。不知道，只是明白，有聚有分，哭罢，喊罢也会离别。

　　一起走过的日子，几多忧愁，几多欢乐，已经放在了红红的月亮的心里，挂在了遥远的夜空。

　　分不清谁在说话了，也没有了必要。就是话语，回荡在空空的四周。

　　一直觉得红月亮是浪漫多情而美丽的，所以我用淡淡的笔触描写了发生在夜晚的一些小故事。常常想，这个世界其实是美丽的，只要我们每个人多一颗宽容的心。
