面对面

（二）

　　我常常做同样的梦，梦见自己在某一个地方看见了某些恐怖或恶心的东西，于是就拼命地跑拼命地跑，边跑边回头看，看自己是不是离那个地方越来越远了，以至于整夜都在观察中度过。我想这是人的一个通病，你总想忘却的某些东西会在不经意间成为一个参照，让你总是把同类的事情与之相比较，如果你不去摆脱它，它就会一直扎在你心里直到肉体死亡。我把这一看法对学过心理学的同事小咬说的时候，他告诉我这在心理学上通常被称为"强迫症"。
