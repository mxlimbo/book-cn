面对面

（一）

　　我相信很多上网的人并不是十分有钱，恰恰相反，他们上网的原因和我一样，穷极无聊——穷极了并且无聊。

　　虽说目前中国上网资费拜敬爱的中国点心所赐是世界上比较高的，但相对于其他的娱乐来说还算便宜，绝对要比泡牛便宜，何况，在网上也可以泡牛。

　　

　　提问！
　　回答！
　　钱和爱情的关系是什么？
　　我们没有钱，所以柏拉图！
　　回答正确！加一百网时！
　　耶！！！！！
　　（呱唧呱唧呱唧呱唧……）

　　上网的时候，常常会让我想起小时候的一首儿歌：

　　 两个小伙伴丫
　　 正在打电话丫
　　 喂喂喂
　　 你在哪儿丫？
　　 喂喂喂
　　 我在这儿丫！

　　后来我在Q上把它唱给玫听，可她马上一句话砸了过来："不要用丫！让我以为是在骂人！（嘭地一记直拳！）"

　　呵呵，对了，刚刚想起来，玫的大学是在北京念的，和蕾一样。

　　我一直不明白的一件事就是为什么玫会在这个出版社工作——她念的是法律，专长是平面设计，却回到天津在这家并不景气的出版社总编办当一个小跑儿。

　　第一次看见玫的时候，是第一次在全社大会上申报选题的那个黄昏。她隔着大桌子坐在我对面窗下，长长的头发披在肩上，瘦瘦的人缩在沙发的一角，像一只猫。那个时候夕阳的光照在海河平静的水面和对岸高耸的远洋大厦的玻璃墙上反射到会议室里，是一种潮湿的怀旧颜色。这使我想起了蕾，多年以前的一个黄昏，蕾也是这样披散着一头长发静静地坐在我对面。

　　于是我的生活有了一个简单的划分：白天我为看蕾活着，晚上我为上网活着；白天我的眼睛活着，晚上我的手指活着。
