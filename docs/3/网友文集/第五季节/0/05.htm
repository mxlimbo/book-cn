第五季节

（四）

　　后来我分别问在场的每一个人，可他们都不记得那个奇异的晚上，异口同声地坚持说那天我们什么也没有做，而是一起上了一整个晚上的自习，甚至连我们所坐的位置都描述得毫无二致。

　　这个借口更让我怀疑这是个阴谋。

　　幕布在变幻不定的灯光中拉开，乙用梦一样的声音呻吟着说，如果你为现实而痛苦，欢迎进入我们的迷幻世界。

　　我们用手中的乐器把世界拆得支离破碎，然后又随意地组合了一番，然后抛下愕然的观众满足地跑回后台。

　　当一个校园歌手忧伤的歌声响起的时候，我点起了一支烟沉默起来。不远处一双美丽的眼睛一眨不眨地看着我。假的，我用夹着烟的手指指舞台说。

　　你们呢？

　　也是假的，都是假的，虚伪的。

　　那你为什么还要唱？

　　我是骗子。

　　有什么意义？

　　什么什么意义？

　　你到底在骗谁，典型的中国人？

　　我不知道该说什么了，或许人的一生就是一个追求——得到——失去——再追求的过程，我只想追求美，可真正的美总是可望而不可及的，除此之外，我想不出人还有什么活着的理由。

　　这时演出的组织者满脸嘲讽地把两张破旧的纸币塞进我衣兜里。散场以后，我们用它们换来了很多很多酒，然后我们在空旷的舞台边上坐成了一排，开始喝呀喝呀。

　　喝呀喝呀……

　　最后一个节目施放的烟雾沉淀在我们周围，在我眼中天棚上亮着的惟一的小灯使我们就像波德莱尔诗中的某些意象。这时奇异的事情发生了，我听见女孩在问我：蕾是谁？

　　我记得自己很久没有对任何人说起过蕾了，或许是乙对她说的？

　　这时我又听见她的声音：不是乙，是你自己。

　　我这才惊讶地发现她的嘴唇根本就没有动！

　　不要害怕，她"说"，其实很久以前，我们就是这样交流的——只要我们想什么，别人就会知道，我们那时候都是透明的。可后来我们发明了衣服，我们有越来越多害怕让别人知道的东西，我们就又发明了语言。

　　我释然了，开始和乙丙丁一起回忆中学的日子。乙说我给蕾塞纸条时像一只菜鸟。丙说那时的天蓝得可爱。丁说那时我们纯净得像太空水。我说我们好象已经有一个世纪没有这么愉快过了。

　　她说十年以后我的记忆里将出现一个第五季节，我们会在雪花中伴着蝉声翩然起舞，在萧瑟的秋风中品味一朵雪莲绽放的味道，而我们嗤之以鼻的民谣将是我混乱繁忙的小办公室里惟一的清风。

　　请为我一舞。我用意念打开一盏暗红的面灯说。

　　我已经为你跳过一次了，至少现在不会再跳了。

　　什么时候？

　　也许将来的某个时候，在你开始自己的第五季节的时候。

　　将来？我说，我对将来可没抱什么希望，我觉得自己像是一条正在接受训练的狗，训练得好就能多得些骨头，更可悲的是我会渐渐把这当成惟一的追求和乐趣，于是我的一生将这样"快乐而充实"地走过。就像蕾，她最终还是选择了现实，这无可非议，在现实面前我们无能为力。但最可悲的是，我们将清醒地看着自己一步步走进去然后迷失。

　　你的生活仿佛只有过去，你自己就像一部照相机一样，其实过去的事已经过去，未来的事还没有发生，我们应该珍惜现在。

　　她在猩红的灯光中缓缓站起身。

　　不！别离开我！

　　我的堤防彻底崩溃，甚至分辨不出自己的声音：别走！我空虚极了！我害怕极了！

　　我拼命地挣扎，可身上的每一个关节都好像生了锈，发出刺耳的吱嘎声，我的眼眶同样干涩，泪水没有流出便已经被吸干。
