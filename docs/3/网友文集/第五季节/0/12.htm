面对面

（五）

　　如果国家富强，鬼才愿意做乞丐！

　　 ——周星星《武状元苏乞儿》

　　得到管理员大叔第一次亲密接触的MM是秀秀。

　　我想网络聊天对很多人来说最大的吸引力并不在于其他，而是一种欢乐，一种在现实中得不到的完全放纵的欢乐。比如秀秀来的时候，我们正在逗楞游游，方法是全部化装成游游的亲戚，比如"游游的哥"、"游游的姐"、"游游的男人"、"游游的大舅"、"游游的大舅妈"……

　　这时候，秀秀对大家说：你好，谁可以陪我聊聊？

　　没有人回答。她又连续问了几个人：你好，有空吗？当问到我头上时，我雀跃着说：有有有，有空！她很快地说：我们聊聊好吗？我说：不好。她问：为什么？我说：因为我不想。她说：我的心情很糟糕，想找个人聊聊。我说：换个花样吧，真没创意！她说：你总是这样对一个女孩子吗？我说：谁知道你是不是呢？她很惊讶地说：难道看名字还看不出来吗？我说：是啊，就是因为你的名字太女人了。她说：那好吧，再见。

　　于是我们继续胡闹，过了一会儿也觉得无聊了。当时钟敲响12点时，大家纷纷作鸟兽散。我看了看名单，秀秀还在。于是对她说：你还没走？她说：没。我说：那怎么不说话？私聊？她说：没有。我说：:::@@::: 你真有钱！她问：为什么？我说：银子哗哗地从荷包里流着，你却不说话。她笑笑：真的只是想找个人多的地方，看看大家热热闹闹的也好。

　　我开始有点相信她是个女人了，凭感觉。于是问：好像情绪真的不好啊？她说：我们又不认识，为什么要骗你？我哈哈大笑：这年头什么人没有？！她说：难道网络真的没有真实吗？

　　这时候我突然掉线，抬头看看窗外，天已经蒙蒙亮了，便软手软脚地爬上了床。闭上眼睛的时候，想起秀秀最后说的那句话：难道网络真的没有真实吗？

　　常听人说，只要断了线就和虚拟世界完全切断了联系而回到平凡的世界，因此觉得失落和伤感，网络世界脆弱得只靠一根细细的电话线来支撑。我自认是一个非常理智的人，在网络上摸爬滚打的时间也不短了，可从来就没有这个感觉。网络世界其实并不脆弱，大家到这里来只是想以另一个身份在另一个世界里开始另一种活法。这个没有国界不分种族的世界是绝对自由的，因此也没有什么秩序可言，甚至老祖宗流传下来的道德伦理在这里也不灵了，一切有如原始社会。如果某一天忽然觉得活得失败，完全可以换一个名字从头再来，就这么简单——也许你以前臭名昭著，可换一个名字马上就变成个大善人，你更可以一边做坏人一边做好人，做坏人不用怕被判刑，做好人也不用怕成烈士，反正一切都是虚拟的，这在现实社会里是绝对不可能的事情。说穿了，网络就是一个成人游戏罢了，什么真实不真实，执著于这个问题实在是幼稚。

　　有人说，在不久的将来人人都会有两个身份证，一个是现实社会的，一个是网络社会的。错了，网络社会不需要身份证。

　　向所有执著地妄图在虚幻的网络上找到真实感情的网民们发出以最大分贝的嘲笑！
