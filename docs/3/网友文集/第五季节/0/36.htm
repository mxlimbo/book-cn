面对面

（二十九）

　　"在带上这个金箍之前，你还有什么话想说？"

　　"曾经有一份真诚的感情摆在我的面前，但是我没有珍惜。等到了失去的时候才后悔莫及，尘世间最痛苦的事莫过于此。如果上天可以给我一个机会再来一次的话，我会跟那个女孩子说，我爱她。如果非要把这份爱加上一个期限，我希望是，一万年。"

　　 ――周星星《大话西游》

　　打开光驱，拿出盘收到盒子里――盒子里是我收集的所有的周星星电影――这是装进行囊的最后一件行李。在任何场合我都从不掩饰对周星驰的欣赏，是他教会了我把视线集中到一点以改变以往对事物的看法，是他教会了我用欢笑去面对痛苦，我一直利用一切机会对别人说，周星驰和金庸一样影响了整整一代人。

　　我把蕾的照片和思念留在了家里。

　　应该跟朋友们道个别吧。我拨号上了网，在论坛上发了一个贴。

　　老金致所有的朋友：

　　我就要启程去米国了，到了那边估计短期内不可能上网了，所以在这里向大家道个别。我会在明天中午12点整在天津市XX道的XX饭店订好酒席，请天津的朋友一起来聚一聚。北京的朋友们，由于时间有限就不能去了，如果愿意过来的同样欢迎。我会叫人拿一朵黄玫瑰（左手）和一本《人之初》（右手）在门口等着。

　　外地的朋友们，虫子、海鸥、蒙面、绛紫、小小……我会记住你们的！希望再来爱线的时候还能看见大家都开开心心地在一起！

　　最令我感动的是，第二天，同志们早早地就陆续赶来，游游、hhh、酷鱼、麻杆羊、悦儿、小米……我委托游游和hhh在门口各拿玫瑰和《人之初》迎宾。不久，北京的朋友也风尘仆仆地杀来，开着宝马的警察、天女和家有喜事姐妹、兜兜、乐歌……大包间里顿时热闹非凡。香槟酒开，我高喊："死游！偷窥！来吧！不用再等了！"

　　这时，听见游游在门口高喊："同志们，又一位老战友到，当当当当――秀秀！"

　　包间的门被打开，一个长发的女孩儿拿着一束花站在门口。

　　竟然是玫！

　　正愣着，楼道里又传来hhh的叫声："同志们，再看是谁来啦？当当当当――WINDFLOWER小花！"

　　我看见又一个长发的女孩儿也拿着一束花走到门口。

　　赫然是蕾！！

　　我咚……
