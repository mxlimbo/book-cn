面对面

（二十）

　　后来将近半年时间里，大家果然都没有再演出，听说好几所学校都下了命令，在可容纳500观众以上的场地演出必须要校长亲自签字。我们乐队的人除了被那次演出折腾得心力交瘁以外，也都在忙着应付考试。已经大四了，我的四级还是没有过，只好每天夹着书本到处找教室自习，可这时我才发现自己根本就不知道在什么时候哪个教室有座位――三年里我连图书馆的大门都没有进过，甚至连阅览证都已经丢了。我常常转遍了整个教学楼才终于找到座位，可刚坐下不久就发现人家晚上是有课的，不得已只好跑到敬业广场或新开湖边借着路灯念英语，而此时总会看到三三两两的学生抱着吉他无忧无虑地唱着歌，一如我们当年。

　　这时才发现整整三年时光，我一直没有真正拥有过这所美丽的学校，而当真正意识到这一点的时候却已经开始嗅到毕业前的伤感气息了。当我的同学们满足地拿到了各种证书的时候，我在拼命地念着英语；当他们开始享受大四清闲的时光时，我还在！拼命地念着英语；当他们纷纷找到志同道合或志不同道不合的异性出双入对的时候，我还在！还在！！拼命地念着英语；当他们隔三差五地到小饭馆共谋一醉或在学校各处拍照留念的时候，我还在！还在！！还在！！！拼命地念着英语。

　　（啦啦啦~~~~~~~~是我是我还是我……）

　　可我欠下的债实在太多了。

　　蕾还用和以前一样的话鼓励着我，每次都是问我念得怎么样了、有什么困难没有，还说哪本哪本书比较好，如果我买不到她可以给我寄来。她的六级早已PASS了，正在准备考托福，双学位的学分也差不多修满了，她还说等考完了托付就去考驾照。有时她也劝我出去走走，说"当你看到外面的天空，才知道世界是多么广阔，我们像一只只被关在井里十多年的小青蛙，现在终于爬到了井口。"

　　（是啊，她矫健地爬到了井口，而我这只癞蛤蟆还没找到井壁呢！）

　　在中秋节前的信里，蕾告诉了我她手机号码和EMAIL地址，然后埋怨说"你们学校真是的，宿舍里连电话都没有"。这是一年以来她给我写的最长的信，语气也温柔了很多，而此前很长一段时间里，她的语气都不像我女朋友而像是我的姐姐。

　　她说："我现在找了一份兼职，给一家化妆品公司做推销员，嘻嘻，这样可以免费用他们的化妆品，卖出一件可以拿一份提成，我倒不在乎这些钱啦，只是想让自己锻炼一下，以前的我太软弱了，所以才老让你欺负（嘟着嘴说，嘻嘻，跟你学的）这样我白天学习，晚上工作，一天下来又累又充实。

　　今天我感冒了，知道为什么吗？昨天晚上我没有去推销，因为下雨了，虽然不大，可忽然就不想出去了，一个人泡了杯咖啡坐在窗前傻傻地看水从窗上流下来。快十点的时候，我跑到了操场上，整个操场空空的，泥土被雨水弄得软软的，走在上面好舒服！我一边走一边想了很多很多。后来雨越下越大，我忽然觉得很冷很孤独，忽然想，如果这时有谁给我撑起一把伞，我一定会感动死了，没准当时就会嫁给他！

　　再过几天就要中秋节了，我想我又要一个人过了，不知为什么，每年中秋都会非常想家，也许我还会自己跑到操场上去。最近北京老是下雨，讨厌死了，旦愿那天不要下雨，我想看月亮。"

　　我给她回信说："你做化妆品推销员太合适了，我们家蕾这么漂亮，不化装都能迷死人，嘿嘿（色迷迷地）……如果很累就先歇几天吧，一场秋雨一场凉嘛，你身体这么弱，一定要先把病养好，要是再着了凉转成肺病（呸呸呸，大吉大利！），我可不想找个林黛玉做老婆，呵呵。中秋节快到了么？小心哪，也许我会给你一个惊喜。"

　　这时候我们已经开始找工作了。两天以来我连续被几家单位拒绝，理由都是一样一样的――"我们对你的经历和能力很欣赏，可是你的英语好像不太好，这样吧，等你通过了四级也许我们会高薪聘请你，谢谢你对我们公司的信任。"

　　不久电台放话说招聘编辑，于是我报了名。在顺利通过了几轮考试以后，最初的一百多人只剩了十个，而这十个人中有六个可以留下，当然还不排除有扩招的可能。最后一关是面试，我神情自若地回答着一个个考官的提问，心中把握十足――凭本少厚厚的简历，应聘娱乐频道编辑是再合适不过了。

　　最后一个考官翻了翻我的简历，然后很感兴趣地问："你以前搞过乐队啊？"

　　"对。"

　　"叫什么名字？"

　　"我搞过三支乐队，不过最成功的是最后那支，叫文艺复兴，我们的原创歌曲还在咱们电台的`环球音乐公告牌`中播过。"我一脸谦虚地说。

　　"听说上次天大搞了一次演出？"

　　"对，我参与发起的，我们乐队也参加了。"

　　"我没有问题了，谢谢你！麻烦你请下一位同学进来。"

　　最终我没有接到通知书，后来听录取的同学说，问我问题的就是那次被公安局传过的那位广播电视局副局长。然后，我不得不参加了两次招聘会，投递了无数份简历等回信。

　　蕾有一点说得很对，当天大给每个宿舍都装了电视的时候，我们宿舍还没有装电话，为了找工作我只好又借钱买了一个传呼机，可有了传呼才发现身边没有电话。那段时间楼下小卖部和电话亭一天到晚都排满了毕业生，有敢通过电话谈情说爱者、说话超过三分种者、没事瞎聊者一概杀无赦。只是这一天，当我排队等着给一单位回电话时，忽然听见前面打电话的人说"祝中秋快乐"才忽然想起今天是中秋节了！

　　天哪！

　　我蹬上了车跑到了滨江道，用最快的时间浏览了一下几个柜台，终于发现了一条羊绒围巾，是蓝色和浅驮色格格的，还绣着SNOOPY，标价80元。我从所有的口袋里翻出自己的全部家产，一共是98元7角。于是掂着钱愣在那里，是要围巾还是去北京？要围巾就去不了北京，去北京就要不了围巾。最后我决定还是要围巾――拿着98元7角去了北京又能干什么？让蕾掏钱请我吃饭？

　　剩余的18元7角刚好够寄围巾的，从邮局出来的时候天已经完全黑了，这时我的传呼忽然响了，液晶屏幕上显示的是蕾的手机号码。我苦笑：老大，用什么给她回电话请问？

　　回到宿舍，迎面碰见小张，他搂着女朋友正等在门外。他们说知道我会寂寞，所以请我去喝酒，西门他们已经在那边等我们了。我们一边喝酒一边回顾乐队走过的历程，西门告诉我，有一天他遇到一个二年级搞乐队的学生问他："最近都在传说你们被禁演了是吗？"这消息多少令我们觉得有几分悲壮。后来想想又觉得可笑，谁也没禁我们，自己怕了才是真的。喝了半天酒，传呼又响了，这次是留言，我用饭馆的免费电话打通了传呼台，小姐用腻腻的声音说："一位叫蕾的小姐留言说，她正在操场看月亮，今天月亮很圆，祝您中秋愉快。"

　　接完电话以后我又喝了很多酒，然后对他们说，我想演出。他们说也想演出，于是我们借着酒力说了很多要重振雄风之类的豪言壮语，最后我对西门说："找个地方吧，哪儿都行，只要让我演一次。答应我，咱们排一首《喜欢你》。"

　　他们把我扶到宿舍楼门口时我坚持不让他们再送，自己趔趔趄趄地往回走，在厕所里碰上了另一群刚喝完酒的，大家因为抢地方吐就打了起来。从派出所出来的时候，酒已经醒了大半，在路上抬头看着硕大浑圆的月亮，心里觉得空得难受。回到宿舍时其他人早已呼呼睡去，我拿出信纸，手哆哆嗦嗦地给蕾写信："十分抱歉，今天没有去，我给你买了一条围巾，有SNOOPY的，我想你一定喜欢。天凉了，出门别忘了围上它，我能想像得到，我们家蕾带上她的样子一定像只小熊一样可爱，呵呵。今天晚上朋友请我喝酒，我收到了你的留言。我也看见了，今天的月亮真的很圆。我看见月亮的时候就想起了你，后来就喝多了，还和人打了一架，现在脸还疼呢。不早了，我也该睡了，脑袋昏昏的，浑身像散了架一样，你也早睡吧。"
