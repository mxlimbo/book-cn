面对面

（十八）

　　经过大三暑假的闭关，我们连续参加了几次天津地下摇滚演出，凭借几首原创音乐与"钢混"一起成为当年天津地下摇滚里学生乐队的代表。出了名，演出的机会也越来越多了，尤其是过了11月的时候，几乎每周都要有一次演出，我们的足迹几乎踏遍了全市所有的高校，但这种演出都是不给钱的。舞厅那边试演了几个地方却颗粒无收，原因是我们乐队人太多（后来又找了我一个吹萨克斯的师弟，编制扩大为六人）而且早已不屑于排练流行乐甚至BEYOND，再有就是一般舞厅或酒吧都要求天天工作且通常要干到后半夜。由于一直没有找到挣钱的办法，每月可怜的生活费还都要用于还债，我的经济状况变得极窘迫，又不能让家里知道自己在学校不好好学习却天天搞乐队还欠了一屁股债，只好勒紧裤腰带过日子，连烟也只能抽最便宜的"凤凰"或"恒大"。

　　我当然没有时间和金钱再去北京，所以每次都写信对蕾说自己用一小部分精力来经营文学社，其他的都投入到了考四级上——实际上我已经第三次没有过了。题外话，这又让我想起了AB的那个小说《AABB》，他说自己的考试都是靠偷试卷混过去的，而对我来说根本不用，文学课我从来不愁，语言课嘛，事先好好求求老师再加上考试抄抄一般也能过了，只有四级——喂！老兄，你是怎么通过四级的？别告诉我也是偷试卷哦！

　　蕾每次都很温柔地鼓励我要好好念英语，"必要的时候无论是文学社还是其他什么重要的事情都要放弃，当然不包括我！"可我此刻已经完全沉浸在那种被崇拜的快感中了，大三快结束的时候，几乎每个想搞乐队的新生都会来拜见我们——爽歪了！

　　我们的演出终于出事了。大三那年的万圣节前的一个星期，我们酝酿了很久的一次规模最大的摇滚演出的准备工作终于进入了最后冲刺阶段，这是一次以电台庆祝"点播音乐十周年"为名义的演出，届时电台做摇滚节目的DJ将为我们做主持人，一个电台的朋友在"环球音乐公告牌"里滚动播出这个消息，天大的"求实礼堂"也预定好了，当时所有著名的地下乐队都已做好了准备。演出前两天，我们忙着印刷和散发传单，上面有所有乐队的节目单、宣传语和联系电话，几乎所有高校的海报栏里都贴满了我们的巨幅海报，每个宿舍都发了传单。

　　可就在演出的前一天，我忽然接到了一个通知说让去一趟学校派出所。一进去就看见另外几个操办演出的人都在，还有电台的那个朋友，一个警察拿着手里的海报和宣传单问我们："你们知道在干什么吗？"

　　我看见在一些比较激进的话下面用红笔都划了道道。警察狠狠地教训了我们一顿，中心思想是问我们有没有想过高校搞"这样的东西"会造成什么后果。然后他限我们在24小时之内把节目单上所有原创歌曲的歌词都交给他们审查。还好我们的歌曲并没有什么太出格的东西，最后只有一支校外乐队的一首名叫《下岗歌》的歌被禁演，他还叮嘱我们一定要维护好现场的秩序，如果出现喊反动口号或打架以及有人受伤等情况，就让我们"后果自负"。

　　我想至少有一点他说的是对的，尽管是在高校，但在任何一个可以容纳两千人的礼堂里演摇滚乐都不能保证没有人扔东西、砸东西，没有人打架、没有人受伤，于是也打消了本来就不坚定的接蕾回来看演出的念头。电台那边的人是不会过来了，听说广播电视局的一个副局长也被叫到了警察局训了一顿，列在海报上的DJ都被扣发了当月奖金，而我在电台的那位负责宣传的朋友也被炒了鱿鱼。

　　那天晚上我们发动了50多号人高马大的男生带上牌儿分散在会场里维护治安，派出所也派了警察坐在会场后面。在演出前我从后门出去绕到前门看了一眼，夜色里礼堂门口不大的广场上黑压压地挤满了人，后来据统计，除了各乐队带进去的几百个朋友以外一共卖出了1700多张门票。

　　在看演出时你也许会羡慕台上的演员，可你不知道，在演出之前，通常我们一早就要来到会场试音、彩排，中午往往顾不上吃饭，不到观众进场都不会下台——谁都担心演出中出什么纰漏。而我们更是连续操劳了好几天，又加上一番惊吓的折腾，到上台以前早就筋疲力尽了，说句实话，身上除了头发哪儿都是硬的。在后台，我们只能一根接一根地抽烟以消除紧张，更主要的是驱赶一阵赛过一阵的睡意。不演出的早已烂泥一样地塌在椅子里，眼睛直勾勾地听着台上和观众的反应，惟恐出什么差错。轮到我们时，我们都木木地站在刺眼的灯光里，没有欢乐，没有激情，任台下掌声嘘声响成一片，只是例行公事般地一首一首地唱着早已烂熟的歌——无论再好的歌，在唱过百十遍以后都不会再令你有什么感觉，况且又是在那样一个场合下。

　　但在唱到最后一首的时候，我还是很留恋地四处看了看，相信登过台的朋友都知道，在强烈的聚光灯下，你看不清台下的什么，只有一片黑压压的人头。但我能看见一双双高举的晃动的手，也看到了星星点点的打火机。当唱到高潮的时候，我听见下面整齐的声音冲上来，回授音箱里我们自己的声音变得小而又小，我们只是凭着感觉在弹在唱，反正谁也不会在乎是否弹错了和弦，我们都只能听见巨大的鼓声和低沉的贝司。

　　虽然观众再三要求，那支乐队最终也没有唱《下岗歌》，直到结束除了有个别人扔了几个塑料饮料瓶但马上被制止外什么事也没有出。当观众陆陆续续散去以后，我们终于长出了一口气，然后一个个塌在舞台上散乱的乐器和音箱中。不久，大灯一盏盏地灭了，只剩两盏暗红色的面灯吱吱地亮着。我像一滩烂泥半挂在一个音箱上，天花板上横竖交叉的铁架子挂满了各种灯具，发出冰冷的钢铁色，但我知道它们还是滚烫的，谁能想到在不久以前那里还回荡着爆炸一般的轰鸣呢？

　　蕾，她现在在做什么？我吃力地看了看手表，快11点了，想必她已经结束了一天的学习洗完了澡穿着雪白的内衣躺在柔软的被子里看书。蕾对我说过，她把自己的小床收拾得特别温馨，她最喜欢躲在温暖的被子里一边晾着头发一边在挂满SNOOPY的小台灯下看席慕容。我想，她的头发一定还湿漉漉的，散发着淡淡的清香。而我此刻就像一条被抽去了脊梁骨的蛇，躺在阴暗舞台肮脏的地毯上，一个星期奔忙却没有洗过澡，浑身瘙痒还散发着一股酸臭的汗味儿。这时，礼堂看门人的声音从遥远的黑暗里声音嗡嗡地荡过来："快走吧，要关门儿了关门儿了门儿了儿了……"我听见小张的声音也嗡嗡地荡了回去："马上马上上上……"

　　拎着沉重的琴盒走到门口的时候，我回头看了一眼，心里清楚地预感到，这种演出生涯也许要告一段落了。这个晚上无疑是最成功的，我相信是高校演出空前也是绝后的一次，可对我本人来说，一直以来期盼的一次演出却是在最没有激情的状态下完成了。

　　礼堂大门缓缓地关上了，我又回到了现实。
