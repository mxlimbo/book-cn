面对面

（十四）

　　那年暑假的时候，蕾说要和我好好地谈一谈。她在信上说："我觉得我们之间的距离越来越远了，也许当初的感动只是一时的，我们终归要面对现实。"

　　那个下午我一直坐立不安，倒不是有什么危机感，而是在考虑我们之间的感情到底是什么。好像从前的一切都经历得太富于戏剧性了，似乎这种情节只有在小说里才能看到，这么长时间以来，我真的没有感觉到自己有哪一次是真真切切地摸到了爱情。

　　夏天是扼杀思想的季节。你应该知道凌乱而肮脏的宿舍里的那种恼人的热总是有如附骨之蛆让你烦躁抑郁。无论是坐着还是躺着，汗都不停地流出来，像无数的蚂蚁在皮肤上放肆地爬行。躺在床上，蚊蚊们嘤嘤歌唱着，用一对一对饥渴的眼睛看着你，肮脏的蚊帐却又怎么也理不顺热呼呼地死贴在身上，小吊扇嗡嗡哭泣着拼命工作也无法使人感觉到一丝的凉爽。

　　看看表，该去接蕾的火车了。我打开柜子，从一团团的衣服里挑出最干净的一件，抖落上面的蟑螂穿在身上。走在阳光里，毫无见情人前的喜悦，只觉得烦躁！想想还要去挤公共汽车，更觉得烦躁！烦躁！！汗哗啦哗啦地流着，柏油路被晒得发粘，脚在湿漉漉的鞋窠里也感觉到烫，烦躁！烦躁！！烦躁！！！这时忽然看见小高低着头迎面走来。

　　"喂！"我喊了她一声。

　　她抬起头，两眼迷茫像看陌生人一样地看着我，过了半天忽然扑到我怀里，哇地一声哭了出来。我晕头转向地问她怎么了，可她只是哭，于是也只好任由她。我们就这样在明晃晃的太阳底下站着，不知是不是被晒的，我只觉得脑袋昏昏沉沉浑身发烧。怀里那个柔软的身体颤抖着（我还从来没有抱过蕾，啊啊啊！这可是我的处女抱啊！），这时便忽然想起那个听见她们翻云覆雨的夜晚。

　　（靠！怎么老是这么龌龊？！）

　　老天有眼，还好她终于停住了哭声，我便拉着她跑到附近礼堂后面的阴凉里。她哭完了，情绪好像好了很多，双眼红红的，神态有些木。我连忙问她到底出了什么事，她就一五一十地对我说，自从乐队解散以后，她就天天和云涛住在一起，有时候云涛把她带回家，让他妈妈睡到小屋，然后就在大屋里拉着她要办事，"我不同意，他就打。"

　　小高哽咽着说："我真的不是那种女孩儿。"

　　"后来呢？"我问。

　　后来她发现怀孕了，可云涛已经和她分手了，她刚才看到我时正是刚打完了胎。

　　（这都是什么事儿啊！！！！）

　　可我也没什么好说的，只有搂着她的肩膀说些空洞的安慰话。把小高送回宿舍以后，才发现天已经黑了。"靠！"我骂了一声，赶忙打了个车奔向车站，一路上心情糟糕到了极点。到了车站，远远地看见蕾守着一个大提包抱着肩膀低着头坐在广场花坛的边上，一只脚在来来回回地踢着地上的什么东西。

　　她什么也没有说，只是疲惫地看着我把提包放到车上。路上，我对她解释了迟到的原因，她也没说什么，只是一直愣愣地望着窗外。我只好把她送到提前安排好的女生宿舍楼下，看着同学把她领了上去。

　　第二天，我们又来到那家咖啡馆门口，谁也没说什么，各自喝了杯冷饮天就黑了。她提议说出去走走。我们就沿着海河慢慢地走。一路上看到几乎在每棵大树的阴影下面都有男女在窃窃私语。我看着看着忽然一句诗脱口而出："更阅尽人间春色"！

　　蕾轻轻拧了我一下："看就看，瞎叫什么！"

　　我捂着胳膊呲牙咧嘴地看着她，她一脸无辜地抬起手把被风吹散的头发拢到一起，河边的风的确很大，迎面吹得她的T恤紧紧贴在身上，一股冲动让我一把把她拦腰抱住。

　　"你干什么？！"她压抑着声音惊叫。

　　（呵呵！还是自己的女朋友感觉好！）我陶醉在美妙的感觉里，什么也不想说。她挣扎了几下也不再动了，脸热热的小声在我耳边说："讨厌啊，别人都看见了！"

　　我就这样拦腰把她抱到了一棵大树后面，低下头去就吻了她（啊啊啊！这可是我的初吻哪！），她也由抗拒慢慢地放松了牙齿，我的一只手也开始慢慢地活动起来，这时忽然想起了《大话西游》，星星说："真是光阴似剑，才一转眼就讲到了重点。在断崖边上，我不顾一切地摸你，你也不顾一切地摸我……"于是忍不住笑了。

　　她用白晰的小手抹着嘴红着脸问："你又笑什么？"

　　（这可不能让你知道！^四^）

　　我连忙解释说："我忽然明白了一句俗话——夫妻没有隔夜的仇。"

　　"要死了你！"她脸更红了，使劲打我："你怎么那么色啊！"

　　（糟糕糟糕！）

　　还好我又灵机一动："不是不是啊，呵呵，其实我是想起了崔健的《一块红布》：那天是你用一块红布，蒙住我双眼也蒙住了天，你问我看见了什么，我说我看见了幸福。啦啦啦，我看见了幸福……"

　　"傻啊你，还啦啦啦！"她也笑了。

　　"春天在哪里丫，春天在哪里，春天在海河边的大树下, 春天可不在小朋友的眼睛里呀眼、睛、里！"

　　（本来以为很难的问题，居然这么容易就解决了，真是佩服我自己啊！呵呵！^o^）

　　后来她还是忍不住问我为什么在给她的那么短的信里还老是小高小高地说个没完，我就对她说："除了小高只能说云涛了，我天天跟你说另一个男人，不是秀豆就是玻璃。"

　　（天才！真是天才！只有天才才能想出这样的答案！）

　　就这么简单，问题解决了。我真的觉得自己摸到了爱情，爱情其实什么也不是，就是一男一女高高兴兴地在一起泡着。整个暑假，每天傍晚我都陪着她散步，看夕阳。可就是这样她还不满足，连大白天也让我陪她去逛滨江道、和平路、大胡同、温州城……但一次次被我无情地拒绝了，理由是第一，白天太热；第二，白天真的热；第三，白天实在是太热了！

　　开学的时候，她依依不舍地走了，临上火车的时候再三嘱咐我不要再搞乐队，要先努力把四级过了，然后争取在毕业以前把六级也过了，"这样即使不考研也能找到好工作"。还有就是就算要搞乐队也不要再找女人（我告诉了她小高的事）。第三是每周至少要写一封信，字数不能少于1000字。

　　（我咚……）

　　我对她的约法三章是：一，要时刻想我；二，不要和其他男同学有过分亲密的接触；三，不要一边想我一边还和其他男同学有过分亲密的接触！

　　我记得火车开的时候，她的小手使劲地按着车窗直让我担心玻璃会被按破，还有就是她的另一只小手一直在抹着眼泪。

　　而我的一只手在挥舞，另一只手在不停地搓着脖子上的泥儿（这也不能怪我，她带了整整两大包衣服和零食，而车站的人又挤成了糨糊）。

　　难怪连小白脸儿贾宝玉都说"女人是水做的，男人是泥做的"，诚不我欺！
