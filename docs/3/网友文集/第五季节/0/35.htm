面对面

（二十八）

　　在工作了整整两年以后，我终于决定出国了。这个念头的产生完全起于一次偶然。那时有一位在加州大学任教的华裔教授在我社出版他的中文版回忆录，责编是我。我看得很仔细，从行文上也帮他润色了不少。暑假的时候，老教授回国来参加一个学术会时顺便来天津看稿子的情况，我们见了一面，他对我看的稿子非常满意。

　　回家以后妈妈忽然对我说："如果老先生能把你带走就好了。"

　　想什么呢？我又不是他什么人……对啊！又不用他给我花钱，为什么不试试呢？于是抱着试试看的心态给老先生写了封信，信中把自己的全部经历都如实讲了，并表达了希望能拜在老先生门下刻苦读书的强烈愿望。没想到他在回信里对我表示全力支持，并许诺只要托福能考上580就给我写推荐信。他对我说，其实国内的等级考试和托福考试是有很大不同的，托福重的是实用，是词汇量，所以并不很难，而国内的等级考试却是连外国人都不一定能考好的，比如说选择题，其实几种说法在生活中都是对的，可我们却一定要求要选择一个。因此，他鼓励我说："不要害怕，勇敢地去考。"

　　于是我请了两个月病假上了一个新东方，又苦念了些单词，居然也考了个590。不久以后推荐信和录取通知书都来了，有了推荐信，使馆也没有太为难我，签证还算顺利地办了下来，剩下的就是辞职然后等开学了。

　　米国啊米国，亲爱的学位！米~~~~~~~~国！啦啦啦~~~~~~~~~
