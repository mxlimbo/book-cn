第五季节

（五）

　　终于快毕业了，有着白鹭一样细长的小腿的女孩终于也要远离这座城市了，就像只度过了冬天的候鸟。我无法挽留她，每当我从幻想中清醒过来开始实际的时候，我知道我们也许就会像我和蕾一样，我开始明白这个世上有很多事你是不能去做的――不是不想，不是不敢，而是不能，因为这是现实。

　　可悲的是，这样的时候越来越多了，我知道这意味着我将失去什么，可是我无法挽回自己的纯洁和童真。

　　我们从早晨就开始喝酒了，边喝边互相翻看着彼此的毕业证书，经过了几个月的奔波，我们各自找到了满意或不满意的工作，回头看看几年的大学生活，竟仿佛已如隔世，仅仅几个月前的事却变得无比模糊。我开始怀疑是艰苦的奔波和接踵而来的解脱的轻松使自己有些健忘还是从前真的只是一场荒诞的梦。或许是酒精的作用，我已经有些晕了，我们脸上都不自觉地挂着傻笑和口涎毫无感觉地干杯，直至我的头像个顶在桔梗上的铅球，我吃力地把筷子伸向桌上最后剩的半条鱼，发现它和与它裸露在外的骨头同样颜色的眼睛正死死地盯着我。

　　于是一种莫名的伤感油然而生，仿佛有很多事情一下子冒出来，可我麻木的大脑却无法捕捉住它们。

　　我痛苦地抛下乙丙丁走进这个午后明媚的阳光里。

　　湖畔有两个人抱着木琴在低声吟唱着清新的民谣。忽然有什么东西落进了我的眼睛，我用指尖把它挑出来，原来是一颗红色的小沙砾。我轻轻地把它弹落尘埃，这时平地里忽然起了一阵风，等风过去以后，我看见一个满脸成熟的中年人双眼红肿着沿着满是梧桐的小路走来。他看见我，表情有些惊异，然后走到我面前抛给我一支烟，然后坐到我身旁。

　　喝酒了？他问。

　　我点头。

　　我也是。他说。

　　我说我很想唱歌，于是就唱了：

　　这个夏天的雨下得十分正常
　　鱼儿浮出水面在空中漂荡
　　街上人来人往却没有声响
　　某个角落里蝉的叫声荒唐
　　这个夜晚我有些感觉紧张
　　我看见你在我的梦中翱翔
　　梦的湖水里睡着蓝色太阳
　　梦中自己的欲望愤怒成长
　　这个夏天的雨下得十分正常
　　洗去多年尘封看见我的模样
　　街上人来人往却没有声响
　　某个角落里我的歌声感伤
　　夏天 有茂密的思想
　　夏天 最感伤
　　夏天 不能停止思想
　　夏天 最凄凉

　　歌唱完了，我吐了个标准的烟圈故作神秘地冲他笑笑说，他们已经记不起自己本来的模样了。

　　他问，那我们本来是什么模样？

　　我看看身边的湖随便地说是鱼，然后煞有介事地举起他的手让他看那些细密整齐的鳞片，这让我有一种过家家的感觉。

　　正当我想笑的时候，我看见他的眼里泛起一层神秘的光彩，然后他指着湖水向我描述说有一个蓝色的太阳正不停地摇摆，它的光芒渐渐散发出来浸染了整个湖，然后继续扩散而且光线越来越强烈，就像X光一样穿透了一切。他闭上眼睛说感到眼睑里一片舒适的蔚蓝。

　　有东西开始一点点离我而去，他闭着眼睛张开双臂说，我渐渐变得轻如鸿毛。
