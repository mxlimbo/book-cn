面对面

（六）

　　上到小学高年级的时候，我不行了，个子越长越慢，也瘦了下来。于是就慢慢地学会了跟在其他孩子后面哄哄，我想我性格中懦弱的一面就是从那个时候开始养成的。

　　那个时候我们开始都听一个叫"宝忠"的孩子的话了，因为在某个没课的下午，我们一起过了铁路郊区去玩（我们住在城郊的结合部），结果遇到了一帮农村孩子，他们兵不血刃地抢走了我的弹弓，有人跑回去叫人时在路上遇到了宝忠，他从草地里捡了一根粗大的树枝冲向对方，不仅抢回了弹弓，还抢到了大家的尊敬。

　　现在回想起来觉得也不是因为宝忠讨厌女生我就也讨厌女生，准确地说，一是因为大家都表现得讨厌女生，二是我一见某些女生就想欺负之，而对其他女生是没有什么感觉的，只是不搭理她们就罢了。

　　有一个黄昏，我和宝忠在放学路上看见蕾和另一个女生有说有笑地在前面走，宝忠从她们身边跑过去，突然大叫了一声，吓得她们也惊叫了一声楞在原地，我也跑到她们后面，双手各搂住一个头使劲往一起一碰，听见"咚"的一声。等我跑走的时候，听见身后哭声震耳欲聋。

　　我踩着宝忠的足迹跑进一个胡同，却被他迎面一个大耳光："你小子怎么这么狠？"

　　记得我当时脸上火辣辣的，却表情愤怒："我讨厌女生！"

　　我的这一举动并没有得到宝忠的好感。事实上，一直到小学毕业我也没有真正得到他的好感，也就没有真正被其他同学接受。到五年级的时候，我索性把精力放在了学习上，后来竟然考上了家门口的重点中学――四十五中，而宝忠他们都上了普通中学，从那儿以后，再在游戏厅、录象厅、台球厅或路上见到的时候，我们谁也不理谁了。

　　我想是由于那个下午的关系，蕾和我已经走上了完全相反的两个方向，由于心中的愧疚和一切举动的徒劳，我很快就不再往她铅笔盒里放毛毛虫和蚯蚓了。大概在五六年级的时候，街头出现了很多游戏机，我的业余时间几乎都放在了那上面，有时候坐在操场边上看夕阳，心中也隐约感到了一丝寂寞。

　　蕾虽然爱哭，但学习成绩一直很好，所以也理所当然地考上了四十五，只是我们不再在同一个班。
