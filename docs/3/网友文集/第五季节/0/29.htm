面对面

（二十二）

　　蕾回信了。

　　"中秋节的晚上，我一个人孤零零地坐在操场边的看台上看月亮，而这个时候某人却在一杯接一杯地喝酒，我记得他答应过我不再喝酒的。中秋节的晚上，我一个人孤零零地从操场往回走，而这个时候一个醉鬼正和另一帮醉鬼打架，我最讨厌有人打架而旁边一大群人围着看的场面了。中秋节的晚上，我躺在温暖的床上静静地看席慕容的时候，一个鼻青脸肿的人正在派出所被警察训斥。这就是你要给我的惊喜吗？！

　　你的围巾我看到了，的确很可爱，可你知道吗？我需要的不是一条围巾，我是一个女孩子，我也有虚荣心，我需要的是一个真真切切的男朋友，他能拉着我的手陪我逛街，他能温柔地给我围上毛茸茸的围巾，然后在我耳边说，我们家蕾真像一只小熊一样可爱！我想那时候我会幸福得流泪的。我想我的要求并不高，可你能做到吗？

　　很快就要放寒假了，我要留在北京找工作，不能回家过年了，你自己好好的，还是先把四级过了吧，什么也不要想了，这么多年来，我自己一个人也习惯了。"

　　寒假过后得知四级考试仍没有过，虽然经过努力水平有所提高，可考试的难度也一次比一次大，身边所有的同学都在为我鼓劲儿，为了我，他们不在宿舍里打牌，我在念书的时候他们不大声说笑，甚至当楼道里有人大声喧哗的时候，他们还会善意地出去让对方体谅一下我。于是整层楼都知道了，在快要离开这座宿舍楼的学生里，还有一个至今仍在为四级考试而拼命的人。

　　（啦啦啦~~~~~~~~~是我是我还是我！）

　　幸运的是三月份的时候终于找到了工作，本市一家不太景气的出版社要我去他们属下一个曾经辉煌如今夕阳不红的杂志社当编辑，他们看中了我的文笔和搞文学社的经历，还有我的诚实――我在面试的时候就已告诉他们我至今没有通过四级，但还有最后一次考试机会，如果不通过，就没有学位证和毕业证。

　　找到工作后我马上给蕾写了封信，但快一个月了她也没有回。直到春暖花开的四月的某一天早晨，我接到了蕾的传呼，连衣服都没有换就拎着秋裤趿着拖鞋噼哩啪啦地跑下了楼，足足排了半个小时的队后终于等到了电话，可拨通了以后听见对方说："对不起，您拨叫的用户已关机，请稍后再拨。Sorry……"

　　过了一个春节，我还清了所有的债后手里还剩了一点钱，于是在经过了一番激烈的思想斗争以后踏上了开往北京的火车。到了她们学校的时候已经是中午时分了，楼下的阿姨用传呼器喊了半天也没见她下来，只好找到一个电话拼命地拨打她的手机。终于在黄昏时分她开了机，声音听上去有些陌生："刚下课吗？"我说："是啊。"蕾说："吃晚饭了没有？""没有，你在哪儿呢？""我刚出去回来，正往宿舍走呢。你怎么想起给我打电话来了？""我想给你一个惊喜嘛！""哦，呵呵。"（可她的声音好像并不怎么喜）

　　我放下了电话跑回她们宿舍楼下，正看见蕾低着头把手机装到包包里，然后她一抬头就看见了满脸堆笑的我。（恩，这个表情还可以，只是惊的成分太多了，喜的少了点）

　　晚饭的时候，蕾一边用筷子拨拉着碗里的米粒一边问我："工作就这么定了？"

　　"目前也没有什么更好的，你呢？"

　　"我想去深圳，在等最后面试的通知，不过北京有一家公司，就是我做化妆品推销的那家，已经决定要我了，虽然待遇不算太好，但可以解决北京户口。"

　　"能有多差？"

　　"见习期三个月，转正以后也就是3000多。"

　　（5555~~~~~~本少见习期一年，转正以后也就1000多……）

　　"英语怎么样了？"蕾问。

　　"正在拼命念，这次比较有把握了。四级一过拿到毕业证就行了，海阔天空任我闯荡！"

　　"哦？"蕾抬起头看着我，"有什么打算？"

　　"考研！"我的话像一枚枚钉子，一个字一个字地钉在桌上，"我、一、定、会、考、到、北、京！"

　　"快吃吧，口水都掉碗里了。"蕾终于跟我开了第一个玩笑。

　　吃过了晚饭，我们坐在操场的看台上，操场上正有一队中国学生和一队八国联军在打比赛，蕾指着一个踢前锋的黑头发的中国学生说："这是我同学。"又指着一个踢后卫的黄胡子的外国学生说："这是我家教的学生。"

　　然后她问我："你说他们谁能赢？"
