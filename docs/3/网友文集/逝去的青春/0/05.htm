五

　　我不怎么注重平时考试的成绩，不过分数都写在老师的脸上。如果你看到是阳光明媚的晴天，那么你肯定的得了不错的分数；如果是平静如水，那么还算差强人意；如果是阴云密布，那你就惨了，准是哪门功课出了纰漏。对每个学生都有不同的脸孔，要是让班主任去演戏，定能的OSCAR奖，真是造化弄人，居然这个种子落到了校园里，化着园丁照料我们这些狗尾草，真是天才的陨落。

　　有一件小事不能不提一下，兄弟班级为了分座位闹翻了天，据说是为明显的不公。他们也是，这时间没有民主的事情多了呢，怎么不见怪不怪呢？人家送礼的同学也不容易啊，也是有本钱和感情投资的啊，分好点的位置也是有情可原的嘛！

　　班级里拱猪成风，每晚必拱。开始输的人用鼻子拱牌，钻桌子，喝凉水，玩的时间长了，也就不新鲜啦。于是有人提议，若是有人再输，就在走廊里喊“我是猪”，时间长了，又换了方法，让两个人一起去喊，第一个人喊:“我是猪”，第二个人喊“我才是猪”。可是没想到提议的那个人连输了两把，于是他对着走廊喊:“我是猪，我真的是猪”。其余众人已经笑得肚子痛啦。

　　有时更绝，输的人要给自己班里的女生打电话，很诚恳地说：“你知道吗，有三个字我一直想对你说，却难于启齿，可是我怕再不说就没有机会了，我。。。我。。。我是猪！”

　　有时就更恶劣了，赢的人指定一个人（一般是个不知情的彪型大汉或者楼长或者某女生），然后输了的人要揪住他（她）的领子恶狠狠地说：“你是猪！” 还有，输了的人要跑到贴着某某病广告的电线杆前大喊：“我的病有治了！”

　　后来大家就打电话找女生，开口很诚挚地说：“你知道吗，我欺骗了你，其实我是一头猪。”一次，某君班主任打电话祝贺其获三好学生，刚开口：“XXX，你知道吗？”该女生就极不耐烦的说：“我知道，你是猪，你的的确确、彻彻底底是一头蠢猪！”
