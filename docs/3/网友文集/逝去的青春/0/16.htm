后记

　　不知不觉间，客居这个江南的城市已有三年。千余个白昼与黑夜已默默地离我而去。在这深秋的夜晚，有瑟瑟的金风，有萧萧的落木，还有江南特有的温婉的闲愁。乳白的灯光漂白了四壁，浓浓的，热热的绿茶却温暖了我的落寞的心情。

　　寒窗十年，喜耶忧耶，抑惑是无喜无忧，如今已不重要。流年似水，逝去不再，所剩只是一个两袖清风的书生，一个踯躅彷徨的成年人。蝴蝶梦中漫游了这么多年，终于到了醒来的时刻。揉揉惺忪的睡眼，原来世情并不像期待中的那么美好，红尘之中并不欢迎我这个踌躇满志的青年。人生风雨的洗礼，上苍给他的挫折，这些仿佛是告诉他，花谢月缺，你已并非往日的少年。 当某些事情发生在昨天，或者是昨天的昨天。在着半睡半醒的时刻，仿佛也不是很遥远，可是就是隔着一层薄薄的面纱，若即若离，已经是时空之隔。红楼隔雨相望冷，珠箔飘灯夜未归，隔雨相望往日的点点滴滴，才恍然觉得那一转身离开的东西，去让我用一辈子的时间去忘记。我伸出手妄图去挽留她们，她们却一笑一颤的从我手边离去，留下了雨后的清新与花谢的残香。于是我有想把他们请进我的日记中存储起来，把我的蝴蝶梦夹进我的诗集中，让她们在将来的有一天，如春草在春风拂过时，在心底里发芽。

　　昨夜雨疏，风雨声中落花知多少，梦里落花知多少？缤纷的落英让我梦回少年的旧时光，梦回日暮时的乡关，梦回我的白发鬓鬓的老母。小桥，流水，老树，夕阳。母亲为一少年扣好胸前的纽扣，拂去身上的灰尘，然后目送少年渐行渐远，化为夕阳与大地相接的远方里的一个黑点，化为她心中永远不会休止的思念。少年的步伐总是很轻快，很坚定，然而不经意间却让落日的暮色洒满了双眼。

　　 野菊花轻轻摇摇头

　　 我没有家

　　 我的家在天涯——"　 也记得曾经有春风沉醉的晚上，夜色朦胧之中有一对小儿女，对着那徐徐的清风，对着那迷茫的夜色，吐露青春萌动的心声。也曾一般年少，几许痴狂；也曾为不明白的初恋付出最真诚的相思。也有山盟海誓，也有海枯石烂，可是到了长大以后，却发现这些原来都是南柯一梦。所不同只是但见青丝雪白头。"世间前世非今世，镜里今年老去年"，只是河边的垂柳依然青青，春风依然依依。

　　 世纪的雨霏雪霏，让我心醉却不堪憔悴。

　　 淡淡的雨，

　　 淡淡的泪，
