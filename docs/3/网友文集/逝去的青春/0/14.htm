十四

　　皖中地区重点中学联考开始了，语文、数学、外语、物理、化学无门考了两天半。没有紧张，也没有激动，我们早已如老僧坐定，所谓的宠辱不惊。不过任然是做不出题目还是骂出题人是他妈的白痴，分明是和我过不去嘛！我就这点没有复习到，他居然就考到了，怎么能不让人恼火！记得以前有个学兄在试卷上写了首打油诗："孙子出题实在难，爷爷做不不出实在烦，早知读书这么难，不如回家去种田"，结果可想而知，他是回家种田去了。考试结束，名次升升降降没有什么太大的变化，唯一不爽的一件事，就是这次作文。作文依然是 材料作文，材料大致的寓意是要学会坚强。结果有1/3的同学失去了他们的亲人，1/3的受病痛的折磨，1/3的要么是失恋了，要么是其他什么痛苦的事情，一时间在学校里传为佳话。

　　书琴的成绩上升的好快，差点闯进三名了，我问小蔡用什么法子把琴书调教的这么出色，小蔡微笑不语，好像他是佛主拈花私的。后来根据小道消息说是因为借的东风之便，因为和几大高手相邻而坐.

　　灿烂的笑容挂在琴书的脸上，愈加显得的可人。琴书的父母都是知识分子，平时对她要求很严格，估计是每次考试后前都要要求她考到多少名的，所以平时看着修长的琴书背着沉重的书包，低着头默默从走廊走到教室里，然后一言不发的爬在书本上看书，都替她感到可怜。看到琴书有解放的一 天，怎么能不为她高兴呢！而且看到亲属脖子上面挂着她老娘买给她作为礼物的项链，把她的白白嫩嫩的脖子衬托的更加晶莹剔透，让每个为她单相思的男孩子们都不能自已了。

　　刚好送走皖中联考，紧跟着就是市联考，这次联考的题目因为是我们学校自己出的，所以就相对的难了好多，肯完一门考试都要死数以万记的脑细胞。本来这次考试的目的就是让我们在最后的关头戒骄戒躁，保持努力的心态而不要放松，知道一山更比一山高，不能功亏一篑。数学考试最后一题我刚开始动笔，下课铃声已经响起了，心里听居丧的，虽然是久经沙场，打击还是满大的，瞥了一眼琴书的试卷，比我更惨，还有大片未开垦的荒地在她试卷上。走出考场时候，我有意看了琴书一眼，发现她眼睛红红的，有低下了头，而且是好象有哭的样子。我有点不忍心，走过去安慰她，她也不理我，知识紧锁着她那长长的秀眉，渐渐的远去了，我只看到她那柔弱的肩膀背着的那个讨厌的大书包，消失在喧嚣的尘埃里面。

　　中午吃饭的时候和德利谈起琴书，我说她挺可怜的，天天在高压下是怎么过的。德利不以为然的哼了声"干卿底事？"，然后就继续喝他的酒了，好象是心情不好的样子，我估计他也和我差不多，肯定是考试不理想，没有人可发泄气愤的，就朝我发泄，我也懒得和他计较，随他在那里喝闷酒，我只是在一边慢慢的品茶，无意见竟想起我的子君。也不知道她考试怎么样，这次考试应该是和我们试卷是一样的吧，如果她能考到我的分数，大概考一般的本科的学府大概是没有什么问题了吧。不过这也说不定，谁能说自己一定能考的上呢？所谓天时，地利，人和，诸葛亮都说"谋事在人，成事在天"，何况我们一芥书生！
