十三

　　清晨起来，发现瓦片上有层浅浅的霜花。昨夜的西风，又凋零了多少碧树呢？

　　刚跑到教室，洁娜从她那鼓囔囔的书包里掏出几件白色的毛线衣，我一看就知道是子君的手笔。子君也知道我就喜欢两中颜色，白色或者是黑色。子君是托付她们门卫阿姨的孙女就是洁娜带给我的。立马就脱下我的破西装，套上毛衣，从心底里面感到温暖，这冬日的寒冷有算得了什么呢？一直在幻想，爱情有可歌可泣的壮举，也许如一守歌水所唱的那样，平平淡淡才是真啊。身体温暖起来，头脑也灵活了许多，五星级的题目都搞的定了。

　　语文加强班上，学《长恨歌》，我们语文老师在那里摇头晃脑的吟唱："侍儿扶起娇无力——"，若祥捅了我一下说"你看贵妃是不是刚刚和皇帝办完事？"

　　 "I think so!"

　　不过我们语文老师心里明白就是不说出来，好象他是德高望重的正人君子似的，其实不然，我私下里思忖他不过是在眼掩饰他的好色而已。因为有一次我和他一块下楼梯的时候，他就盯着前面的一个女生猛看，看的差点他那1000多度的眼镜都掉下来了，而那个女生只不过是穿了一件透明的短裙而已。我知道这样说老师未免太人渣了一点，而且极易惹来麻烦，有诽谤他人之嫌疑，保不准会被告上法庭。这是有前车之鉴的，听说浙大就有个研究生只为了一时尽兴，调侃了他们导师几句，结果就被起诉了，而且还丢了学位。" 避席畏闻文字狱，著书都为稻粮谋"，文字狱者古今有之，中外有之，可谓知识分子的悲哀。虽然现在的知识分子不在处于娼妓之下了，可是倘若遇到为人民办事的公仆之类，你有怎么说得清楚！

　　外面雪下的正紧，纷纷扬扬的把大地都穿了一身银装素裹，教室里面没有暖气，冷到升不出手。第年级的同学都回去过年了，只有我们这群迷途的羔羊还在这里做着前途无量的美梦。鞭炮声在学校的四面响起，对我们来说无疑是四面的楚歌，让我们军心大动，无心恋战。就像是远在边关打仗的士兵突然听到辽东小妇的羌笛奏出的《出塞》。于是便有好事者，出来组织造反。一篇战斗的檄文在年级里传来传去，终于在补了几天可以后，突然有一天，所有的学生都在寝室里睡觉。我们众志成城，取得了胜利，收拾包裹回家的路上，真是感慨万千。

　　一个多月没有见到子君，刚一到家我就要往她家跑，被我妈骂"花喜鹊，尾巴长，娶了媳妇忘了娘"，我也不管了。子君比我先到家好几天，她们没有补多少天的课。我到子君家里名誉上是为子君补课，因为子君中间为了头疼病缺了一段时间。正子君旁边过的真是神仙日子，有吃有喝，有人陪着聊天、下棋。不过和子君下棋我赢的少，输的多,我要让她输了，她定会不依不饶的纠缠，如果明显让着她，她也不高兴，这让与不让之间实在是难以把握，就如佛家说的取舍吧。

　　我喜欢吃子君做的菜，子君的厨艺相当的不错，吃的我是乐不思蜀。我更喜欢坐在旁边一边听音乐，一边看子君认真做菜的样子。子君闲我笨，不让我插手，我也乐得清闲。和子君斗嘴，恼了就跑过去要揍她，乘机搂住子君的腰不肯走，直到大婶到厨房里来。一个寒假过是神仙的日子，出了子君偶尔头疼，真是一点烦恼都没有。惹得我妈跟大婶说："我生的儿子真是替你生的，在你家呆的时间比自己家还长！我就闹不明白，我怎么生了个这么不孝顺的儿子！！"

　　我喜欢冬天，喜欢这个温暖的冬天。其实每个季节都有可爱之处，就看我们的心境如何，像今年这个冬天，虽然也下了大雪，冷的缩在房间里面不敢出去，可是丝毫都不影响我的心情。我依然每天高高兴兴的喝茶，高高兴兴的和子君斗嘴嬉闹。当纷纷扬扬的雪花漫天飞舞时，我们也忍不住起了童心，到外面去打雪仗或者是堆雪人。冬日天短，这个冬季也显得好短，翻翻日历到了开学的时间，我心里还是老大的不乐意，就像是小时候，晚上听爷爷讲故事而不愿意上床睡觉而赖着不走一样。
