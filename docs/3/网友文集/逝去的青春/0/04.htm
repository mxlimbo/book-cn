四

　　期中考试到了，同窗们个个都像押往刑场的死囚，病恹恹的没有一点的生气。晚自修以后还有人留在教室里不肯走，妄图补回平时浪费的时光。寝室里面也听不到笑话，听不到吹牛了。早晨起来一个个眼睛红红的，像输钱的赌徒一样。

　　 "读书是为了父母亲面子的问题，
　　 成绩能证明老师猜题的能力。
　　 你在压力里苟延残喘的样子，
　　 仿佛这样的你永远没有抗议的权利。
　　 课本里教的和现实中的学的成了一种彼此矛盾的对立。
　　 学历是不是最终教育的目的？
　　 谁在操作这个竞争的游戏？"

　　郑智化先生的《补习街》听了这么多年，真是我们生活的写照。气氛真是憋闷到了极点,约阿德去喝酒，他的我室友，我们混的还不错。小酒馆里面弥漫着淡淡的酒香，还有哀而不伤的情歌。我们的旁边一桌几个高年级的同学在大声的喧哗，他们如此的嚣张是为资格很老，其中不乏有八年抗战的老油条。我们伟大的祖国什么都要将资格，看看我们的领导们，哪个不是垂垂老矣的老人家呢？他们是我们教育制度的牺牲品，是江湖人士，是所谓的古惑者。我和阿德默默的喝酒，突然阿德猛的拉了我一下，我回头一看，原来在我的身后，有个家伙正拿着刀追另一个。好险，如果不是阿德反应快，我岂不是成了无辜的牺牲品。遇到这种晦气的事，哪里再有喝酒的兴致。

　　 考试的成绩下来了，几家欢乐几家愁。有人打算去喝酒庆祝，有人打算去喝酒消愁。有一点颇让班主任不爽，我们几个不听话的家伙居然还是佼佼者，得意门生却让我们老班丢脸了。总评下来，我班的平均成绩在年级八个班中处于下中等的位置。也是预料中的事情，我们的物理老师实在是太差，可是他是我们班主任的班主任，所以没法罢他的课。老头子经常为一道物理题耽误一节课，我们真替他着急， 暗地里为他捏了把旱，又不好提醒他，只好哈欠连天的熬到下课。我们是老头子最后一界学生了，班主 任希望老头子能光荣的退休，让我们给他个面子。提到面子问题，我们都无计可施，面子问题是我们国人最重要的问题，我们的圆明园可以给人家烧掉，但是我们的祖坟却不能给人掘了。，是同样一个道理

　　不过好在我们的英语老师特厉害，而且满漂亮的。看到漂亮的老师上课，精神都容易集中起来。我想每个男生都希望把我们所有的老头子老太太都下课，全部换了漂亮的女老师。说不定我们的老校长也同意我的意见呢。三班的学生揭竿而起把他们的外语老师下课了，虽然取得了一点胜利，代价也很大，带头的几个同学都受到校方的警告。革命的道路是曲折的，前途是光明的，人间正道真实沧桑的很。

　　我没有兴趣参加此类的事情，事不关己，高高挂起嘛，明哲保身是我一贯的作风。记得路易十六曾经说过："我死后哪怕洪水滔天"。这样说来显得太自私，可是这也是没有办法的事。学生运动是有前车可鉴的，比如某年的那些事情，我不是学历史，所以也不怎么记得了，只是好象发生过吧。课业一天天的加重起来，一本薄薄的书越学越厚，补充的资料像垃圾堆那样高。我们在题海里面遨游，在题海里面挥霍我们的大好的青春。

　　一不小心，马上到春节了。天降瑞雪，校园里银装素裹的。松针上的冰晶在阳光的照耀下发出绚丽 夺目的五彩之光。好久都没有见到子君了。因为马上有要终考，我怕打扰她的考试，也不敢去找她。中午的阳光射透玻璃，撒在我的信纸上，我懒懒的拿起笔，用我心底的话和子君述说我朦胧的心事。对她往日的关怀表示谢意。想到了从前，渐渐的我不知道自己在说什么了。

　　 几天后，子君来看我，也不见有什么表示，只是淡淡的告诉我家里的一些无关的事情。转告我，母亲的话，家里一切安好，让我安心读书之类。一个学期没有理发，头发有长起来，都遮住眼睛了。送子君上车的时候，她说她自己回去不让我送了，我猜她是怕我忙，也就没有坚持。他从车窗里面伸出手扫了下我的头发，有埋怨的眼光盯了我一下，然后对我说："考试考好点，看你头发这么长——"公共汽车渐渐的远去了，突然不经意间，我竟发现子君的回眸。那温婉的一瞥，我知道应该是我这一年最大的收获，同时我也知道我已经深深的爱上了她。漂泊了这么多年，竟有回到了从前，竟有从终点回到了起点。

　　春节的气氛好浓。鸡灶节过后，家家门前都是满地的鞭炮的纸屑。老秀才年纪大了，不能再写春联，我成了法定的接班人，顿时身价倍增起来。整天忙的不亦乐乎，从早上一直能写到晚上，比公仆们整天开会还忙。给子君家里写春联的那天，他家高朋满座，午饭的时候竟没有我坐的地方。理所当然的该在厨房里面和子君一起吃。我很开心，虽然从前不知道和她一起吃过多少次，这次的感觉却大为不同，因为我们之间的心有灵犀的默契。

　　 短短的寒假，除了和子君一起做点功课，就是和老同学去喝酒，日子也过的满快的。返校的时候，总觉得别扭，感觉没有玩够。不知道同志们对学校有什么看法，反正我是深恶痛绝之，虽然我的成绩也是前十名。我一直认为学校是对人性的一中摧残。题山题海，吾生亦有涯，而知亦无涯，以有涯追逐无涯，殆矣！！！
