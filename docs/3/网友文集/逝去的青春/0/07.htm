七

　　刚到学校就听说小彩被女孩子泡了，当了倒插门的难朋友。那女孩是二班的，颇有风尘侠女的风范。就是有点过于前卫，流言蜚语比大明星还多，非一般的男生可以接受的。我们都替小蔡惋惜，可是他自己喜欢，这点让班上所有暗恋小蔡的女孩子们恨的牙痒痒，如果可能的话，她们都愿意和小蔡的女朋友决斗。有天喝酒的时候，有人问小蔡为什么会喜欢现在的女朋友，小蔡睁大了眼睛问："爱一个人需要理由的吗？"搞得我们面面相觑，无言以对。感情的事情确实难以琢磨，不然的话怎么会有那么多的英雄们，泪洒情关，

　　 "无怨无悔我走，走不尽天涯路，什么时候流血不流泪——"

　　小蔡已经连年总评第一了，三好学生，优秀学生干部都有了。我却被班主任扬弃了，道不是为了将来高考的时候加分问题而苦恼，而是面子问题。班上六十多人我得了四十多票，和小蔡差不多，竟然没有当上三好。后来反思了一下，大概是因为我参加的集体活动太少了，有小蔡罩着我，我经常缺席，不管是上课还是活动。有一次还和班主任顶了几句嘴， "你到哪里去了，开班会你不知道么？"他忿忿地说。"知道""知道你还去看杂志，我就是搞不懂杂志有什么好看的？"然后，我感觉已经和他没什么可说的了，我就保持沉默，可是掩饰的不好，眼神里流露出不屑的神色，所以就得罪了他，以后就没有对我垂青过。 三好学生的事情并没有对我影响太大，不久来了年级足球联赛就把我的全部注意力吸引过去了。我们 班踢球的人很多，所以我们班球队实力相当强，循环赛中所向披靡，无人能挡，很顺利的打到了决赛。决赛的对手是二班，他们和我们一样一场不败，不过都是小胜，而我们都是一5：1或者是4：0这样的大比分胜利。比赛前天晚上，队员们都是踌躇满志，报着必胜的信念。大声吟唱"不破楼兰终不还"第二天下午，天空阴沉沉的，好象有下雨的样子。场上的阵形依然是老3.4.3的进攻阵形，阿胜，小波，小蔡是前锋，我在左中场.开场半个小时打的还算流畅，我们控制了场上的局面，节奏都是有我们来定的。，配合传球都很稳妥，就是临门一脚的感觉太差，几次打到门框上。运气的糟糕把我们的气势全部化解了，我们越打越急噪，在没有角度的情况下也不在配合，就是对着对方的球门乱射，在半场快结束的时候，也总算蒙进去一球。中场休息的时候，我们的队员都气的直骂娘B。下半场开始以后，我们的猛功的节奏渐渐的慢了下来，可能是体力不支了。场上的局面也渐渐被动起来，好在我们的后卫还是很顽强，坚决顶住了对方的进攻。只是我们的球门频频告急，我们球员也是越来越急，连章法都没了，小蔡气的在大叫，也没人理他。

　　最后10分钟了，我们还是1：0领先。对方的球员也开始沉不住气了，他们的前锋老窦在过中场没有多远的地方开了个大脚，我认为是传球的，谁知道可能是顺风，居然在空中画了个弧线飘悠悠的飘到球门了，我们的守门员还没有回去，球进了。1：1，我们全线防守，11个球员全部缩回自己半场，一直坚持到了最后一分钟。正当我和小蔡说晚上到那里喝酒的时候，裁判却告诉我们还有3分钟补时，我的心凉了半截。果然我们没有顶住最后的3分钟，我也不知道最后一球是怎么进的，反正我们是输了，因为我看到2班的球员在欢呼在跳跃，在堆人，在 脱衣服。

　　酒很苦也很辣，特别是失败的酒。十来个兄弟也不知道喝了多少酒，因为我们连酒瓶都书不清楚了。为了安慰我们女生，我受命与危难只秋，撰文一篇，总结我们失败的教训，并发誓以后一定勤加练球，不再辜负女生的厚望。此文天地为之含悲，风云为之变色，我敢说其言辞之悲痛，语气之诚恳，感情之充沛，一定不比为我们校长写的悼词差，如果我们校长大人仙逝有人给他写悼词的话。女同学也确实被我们诚恳的告白所感动，有几个女孩子居然哭了起来。我们的班主任也顺应民意，和我们同仇敌忾，让我们花悲痛为力量，继续努力，争取下次取得胜利。几天后是子君的生日。暑假时她的生日并没有到，我母亲也并没有记错。子君很了解我当时的心思，所以替我找了个借口罢了。我到礼品店去挑了几件女孩子喜欢的小东东给她作礼物。价格并不高，不然她要埋怨我浪费银子了。在送她回去的路上，我搂着她的肩膀，子君的馨香让我心醉，让我痴迷，让我浮想联翩，也让月色出奇的朦胧。我要到子君寝室去，她不然，我耍赖，她要生气了，我只好作罢。不过子君突然回过头来给我一个浅浅的印在我心上的吻。我也生平第一次接触到女孩子不同于我们之出，好柔软的身体，好馨香的怀抱。有子君伴我，我此生还需要什么，只要她能永远的幸福，我愿用我一生的时光去换使我痴迷的那一刻，让时空不再流动，让那一刻化为永恒吧！！！
