谁家杨柳堪系马？

还有什麽希望留住呢,
在这样的年纪?
是年少轻狂时曾对他许下的盟誓,
还是与她初初相遇时的忐忑不安,
或者是现在对你依然如旧,温柔似水的思念?

如果说总要留住一些痕迹,
那无非是温柔的风,多愁的雨,
情生意动的春,以及多事离别的秋..
那些懵懵懂懂的爱恋,
莫名其妙的忧愁,
那些似曾相识的感动,
怅然若失的心情,
在生命的转弯处,都被抛的无影无踪.
在也不能回头追寻.

那麽关于你和我,
关于你和我之间的故事,
要如何才能保留呢?
时间是一匹马,
即使有杨柳青青般柔软的心,
又何堪系住时间的马?
