我是一条感冒的鱼

阳光耀目、无意间打了一个喷嚏。
朋友笑道"你感冒了!?"
轻笑摇头、我宁愿相信这是你思念我的缘故。
是你穿越时空、向我传递你的心情。
在如烟尘世、你我本是一条孤独的鱼,
因为偶然误入网中、挣扎弹跳,
却忘不了那缠绕如丝的温柔感觉,
而成为留恋温柔、执着如斯的鱼。
一日为友、终生结缘。
虽然分隔茫茫人海、你还是用如此隐晦的方法,
小心的试探我的心、传递你的心情。
周围阳光流动、人影恍惚。
一个想念的喷嚏化散成万千颗粒,
在空气中传散、消逸,
每一个都映着你的名字和你的思念....
于是、我收拾心情,
揣想着你的方向,
全身全意的等待下一个喷嚏。
今夜、我是一条感冒的鱼。
