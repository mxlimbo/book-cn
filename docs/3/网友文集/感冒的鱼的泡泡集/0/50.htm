筑一面墙,守住一份永恒

在以前的家里,有一面空白的墙.于是在上面贴满了许多卡片和朋友的相片. 墙也因此而绚烂起来.有时看着那些相片——有的人微笑着,有的人漠视着,有的人搭着以前的我的肩,看着我生活.寂寞,烦躁的心也就会莫名的平静下来.

而现在,再也没有这样的一面墙了,卡片和相片也被尘封在心的角落之中,不敢想也不敢触碰——有多久没有看到他们的笑容了,即便是那印在相纸上永恒的微笑? 他们又是否象我此刻想念他们般想念着远方的我?或者是否也经历了风风雨雨,同我一样哭过笑过呢? 又有多久没有从另一个角度冷静的看看自己,就象看相片中以前的我,到底在追求些什么, 又得到和失去了什么呢?

收到远方朋的来信.熟悉的字体写着熟悉的字句,[狡兔三窟,东奔西顾.衣莫如新,人不如故.] 尤其是最后一句[人不如故]又怎不催人泪下?!泪水冲开了记忆的门,洗去多年的尘埃,才发现他们的笑竟仍是那麽生动,那麽清晰.于是相信这世界上还有真诚和永恒的东西.

真的好想筑一面墙,一面象以前那样的墙.隔开风风雨雨,隔开世事人情,
在我心里, 来守住一份永恒.

是的,筑一面墙,守住一份永恒.
