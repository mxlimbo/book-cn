冬　眠

一辈子最美丽的春天的后面是一辈子最寂寞的夏天.
一辈子最寂寞的夏天的后面是一辈子最冰凉的秋天.
心事辗转,象变换不定的冬.
在心经历春夏秋冬之后,
又错入一场长长的冬眠...
醒来时,那一年再美的春天也只是存在遥远的记忆中.
随着更多更美好的春天到来,渐渐淡忘..

不要总以为我躲在过去,
其实一个悲伤的心.怎麽会有我这样的快乐.
心中有雨,又怎会有晴朗的笑容?
现在的我有一颗从容不迫的心,
一路行来, 不急不徐,
看尽美丽的春,明快的夏. 多事的秋,温暖的冬..
