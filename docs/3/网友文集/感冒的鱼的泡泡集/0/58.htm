以　为

这是一座美丽的城、
我漫无目的的走过一条条街道、
想像着你如何生活。

你经常走的路我去了。
你每天搭的渡轮我摆过了、
那一间你曾说起过的咖啡店、
我也伫了很久。
呼吸着你每天呼吸的空气、
看你所看到的景物、
试图让自己离你更近。

还以为离开你其实无妨、
还以为所有的事情都能风吹云散。
还以为可以挣脱那些日以继夜的思念。
还以为我可以忘记你的一切。
可是啊、
有关你的点点滴滴、
有关你的只言片语、
还是将我牢牢捆绑。

恍惚中、
阳光下的城市好像燃烧了起来、
烧啊烧的,不知怎的竟烧的心好痛.
也许再多的记忆也是徒劳无功、
也许这是一座只属于你的城。
