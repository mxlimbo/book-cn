看　星

早上起来送报,看到满天的星星,
就象一群久违的朋友.在这样一个紧张,繁忙的都市中,
很少有机会抬头看看天,虽然它们仍然出现,
但人们却容易忽略了它的存在.

上一次看星是什么时候已经记不得了,
但却清晰的记得那一年的夏天,
和你相约在山上看星.

星星布满了整个天空,
照亮了夜色,也照亮了你和我...
也许如此,
才会对你说了一些我原以为这一生都不会说出的话.
而你却说你就象这满天的繁星,
看似很近,仿佛一伸手就可以触碰到,
但实际上距离好远.
别再迷恋美丽虚幻的景象,去追求一些实际的东西吧!
那一夜我的心碎得就象天上零零乱乱的星.......

再次看星,
少了一些心痛,却多了一分平静,多了一分思念.
不知在远方的你的身边,
是否有了陪你看星的人了吗?
