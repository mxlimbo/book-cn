小虾米的狂想曲

如果我长大了。
我会在星光如屑的夜晚,
踏过长长的街,寻找这座城市中最美丽的屋檐。
轻轻的敲你的窗,将你从梦中唤醒。
牵你的手在奶油色的月光下翩翩起舞。
看巧克力色的屋檐下闪出一盏一盏明亮的窗。
象不肯休息的精灵的眼。

或者与你悄悄的穿行寂寞的城,
从一个陌生的车站,走到另一个陌生的站,
看灯火璀璨的月台,风中飞舞的报纸,
和蜷缩在角落的猫。
小心的拉紧惟一的温暖。

或者在寂静的海滩,听海浪幽幽的叹息,
看他的背影离去后,
再点燃一支小小烟火,把一切藕断丝连,
一切有关他的故事燃烧成灰...

我还希望长大的时候。
也背上一个简单的行囊。
从一座寂寞城流浪到另一座城。
在你经常歇脚的那个小店,
点上一杯你常喝的叫做寂寞的那种酒。
写一张简单的明信片,
留下一个灿烂的微笑.
寻找着你的气息.

我希望能够有一次相知的喜,
如影随形的念,刻骨铭心的痛,
以及缠绵悱恻的恋.
