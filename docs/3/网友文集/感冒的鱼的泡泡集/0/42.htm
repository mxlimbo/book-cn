相见欢

你走了？
你选择了静静的离去，没有和我说一声再见？
我想也许这样的告别对我们来说是最好的吧？
聊天室里人声鼎沸，你的名字还在网上，
我却只看到了不应该属于你的那片空白...
我猜想你是不愿见到含泪的道别，难舍的祝福，和离去的身影。
在网上充斥着我想你，我爱你的时代，
你只是淡淡的问候，浅浅的道别，
但是我却从你的巧笑嫣然中，听出了你心情的些微变化。
从花散如雨的笑，到率性认真的怒，
从微柔难转的愁，到倾流泄尽的喜...
是的，我明白，
我明白你那不为人所知的心事。
明白你的喜欢，心怡，眷恋，痴迷，难舍..
多少时候以为自己还是心止若水，
多少时候以为自己还是飘渺云烟，
但你在不知觉中竟成为温柔的牵绊。
这一次，你将以怎样的心情想起我？
会不会，能不能，可不可以，
再与我执手以对，泪眼相看？
