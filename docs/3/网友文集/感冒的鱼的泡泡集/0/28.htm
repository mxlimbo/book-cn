歌/心情

对我来说歌就象一本日记，
记载着我的喜怒哀乐。
有的歌会让我想起一个人，
有的却又让我忆起一段心事。
有时哼的歌一闪神就换了首，
有时却让我在刹那间，恍然如时光交错，
陷入 迷蒙之中。
那时的心爱，心喜，心痛，心苦，心酸，又在心头涌起。

有的人，有的事只会从眼前如浮云掠过，
有的人，有的事却从此轻轻闯入我的内心世界，
惹出欢喜悲伤。

只有歌，才能让它们在此起彼落时，交换成心事。
现在的你在哼着那首歌，
又会想起那个人，那段事呢？
