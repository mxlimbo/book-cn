希腊乘船记

　　心里曾经反复的挣扎斗争，到底是否要把这一段经历写出来。在朋友眼里我是一个遍游世界的旅游家，好象什么都知道，在旅游时也都是化险为夷，哈哈我也因此还可以挣到一些咨询费。如果写出来会不会自断财路？不过想想为了大家还是决定说出来。反正羊毛出在羊身上，我以后把咨询费提高就可以把损失夺回来的....

　　第1次到希腊旅游，决定要乘船周游各岛。在雅典的一家小旅行社里买了船票，象我这样的贫乏旅游者当然是买的最便宜的了。第二天早早坐车到了港口，看到了我的那艘船。也许是我在国内乘船的经验很少，最大的游船就是去南方时坐的从苏州到杭州的，所以各位不要笑话我当时那种惊讶的心情，那么大的一艘美丽豪华的游艇，哈哈赶快上去看看。拿出我的票问守在舱口的乘员我的座位在那里，他迅速的瞟了一眼，指着一个小小的旋梯"上面，上面！"。经过一等舱，酒吧，餐厅，二等舱怎么还没有见到我的地方啊？咦不知不觉时我已经爬到了最上面。原来如此，在船的最上面是宽阔的甲板，有一排排的长椅，我的座位是在这里的吗？果然是最便宜的船票，要睡在甲板上啊，回头看时只有零零散散的几个游客，倒是可以看到美丽的景色。哎呀不对，我坐的船是夜船，7点出发第2天早上到达MYKONOS岛，那么岂不是连景色都看不到了？...

　　但是当船驶出外海，海鸥在船尾上下盘旋，远处雅典白色的城市斜映在被晚霞染红的爱琴海上，衬着游船拖着长长的白色的尾迹时，我还是被深深的感动了。只是可惜身边没有美酒佳肴，打开带的面包，就着矿泉水为眼前希腊的美景干杯！..在天色渐渐暗下，解决了肚子问题后，尤其是外海寒冷的海风吹在我的单薄的身上时，我首先要考虑我如何能够打发这漫长的一夜。别的人都是怎么过的呢？回头看看怎么刚才的那几个人好象又少了几个。想想那些乘坐头等舱的人现在肯定开着大宴会的吧？把我所有的衣服都拿了出来穿上——其实也只是多套了一件衬衫。坚持！看看我带的小说打发时间。不知道过了多少时间，天色暗的连书都看不了时，我无意间回头瞧瞧，咦？怎么刚才的那些人一个都不见了？大家都到那里去了？拿起我的书包四处寻找，走下旋梯，穿过餐厅，哈哈！！我真的是个大傻瓜！原来在餐厅的后面有一排排的沙发座椅，前面带着电视，虽然放的是希腊的电影，但是这里才是我的座位。有空调，自动售货机，座位也软绵绵的，象飞机上的座位，特别是对刚刚坐完几个钟头硬木椅的屁股来说不知道要舒服多少倍了。不过各位不要笑我，因为船上的所有标记都是用的希腊文，我怎么能够明白？唉看来老老实实的做人，言听计从也是要吃亏的，所以我得出的经验就是相信自己的屁股也不要相信希腊船员的话。哈哈!

　　在MYKONOS岛上住了几天，因为岛太小了，抬头不见低头见，没有几天游客就都互相认识了。晚上在餐厅都是结帮结伙，特别是我们这样的单身游客，每天都象是开派对。最后一天晚上，又是一大群人聚在老地方，几个人喝的意气风发，把店里所有的WINE都喝了一遍，对于每天，每顿饭都做好了计算的贫乏旅游者来说这样的狂欢是难得的更是痛苦的。我想大家明天酒醒以后都会为自己的钱包痛哭吧。回到我的HOTEL已是半夜2点了，别的游客还在醉生梦死，只是我明天要搭乘8点的船到SANTORINI岛，我买的指定时间的船票要便宜好多的。也许是刚才的亢奋一时不能消退，看着窗外明亮的星空竟不得入睡，看看表已经快5点了，明天7点就要起来赶到港口，实在是没有能够起来的十足把握，但是睡魔这时袭来，不知不觉中睡去了。"哎呀！！"再睁眼是已经是天光放亮了。看看表8：30了，天哪竟然睡过了！虽然一个鲤鱼打挺翻身起来也不能赶上那班船了。怎么办？脑子转的飞快也是想不出办法。只能凉拌了－到旅游社去买下一班船的票，这张票只好作废了。

　　满脸沮丧的敲开门"有什么事可以帮助的吗？"对不起，没有能够赶上船！旅行社的小姐要过船票简单的看了看，"那么你想换乘那一班的船走？"咦？看来我的这张船票似乎还能够使用。"下一班的船是几点的？下午2点，你坐吗？看到我点头她接着说那么你的这张船票的优惠就没有了，你要再加付优惠的钱。啊原来如此，那也是没有办法的啊。多少钱？50美金！"哇～哇！！天哪要50美金,在日本也要拼死拼活的打工好久哪而且对于我们这种旅游者来说，每天1美元2美元都要计算考虑的50美金真的是巨额数字了！并且给我精神上难以估量的打击。

　　垂头丧气的出了门，心里还计算我的损失的时候，身后传来了招呼声。是在岛上认识的从欧洲来的单身游客，"你不是今天早上就要离开吗？"是啊是啊，只是....我把自己睡过觉而错过船的事向他说了，特别是我损失50美金，倒博得他的同情，拍拍我的肩膀，走吧我请你吃中午饭！哈是在MYKONOS的最后一顿饭了，又有他这么热情，那么就不客气啦！MYKONOS真的是一个美丽的小岛，随着时间的变换展现不同的风景。和朋友坐在海边，吹着习习海风享受着美酒，聊着有趣的话题....不知道是什么时候他突然问我"那么你换到几点的船了？2点！可是现在已经1点30分了？！！什么？我下意识的跳了起来！那就是说...我又赶不上了？我到这个时候才真正觉得自己是个天大的傻瓜。不管怎样2点的船赶不上了是明白的事实，是事实就要接受，这就是人生啊。但是，我还要到那家旅行社去吗？太丢脸了...唉没办法，丢脸也是人生的一部分啊，反正丢脸的事也不少了，加油干吧，小伙子！自己给自己打着气...

　　"你又错过了船？！"她左右晃动着头，脸上充满了不可救药的神情，标准的欧洲人的作风，表情丰富而且夸张。"你是打算坐下班船走，还是想在这里定居啊？"是几点啊？"晚上10点！"哇那么到了SANTORINI岛都几点了啊，我正在沉思，她高昂的声音压了过来"坐还是不坐？！"啊..啊坐，坐！"那么你在这里写上你的名字，因为是深夜的船有深夜优惠，所以退还给你40美金！"啊？？真的？！哈哈哈哈我亲爱的40美金又回到了我的身边，没有赶上船真的是太幸福啦，我脸上的笑容象开了花似的无法掩饰，弄的她也忍禁不俊！"这回千万别晚了！9点就到，如果再错过了你就把40美金交回来！"哈哈为了我堂堂男子汉的面子我绝对不会迟到了！我在心里对着蓝天发了誓...

　　再见吧，MYKONOS岛。坐在开往SANTORINI岛的船上，看着灯火渐渐远去，40美金是我的了！所以我得到的经验就是拉下老脸会有意想不到的好处。哈哈。轻车熟路的找到了座位后和衣睡下，还抑制不了幸福的微笑，却完全忘记了前后还是损失了10美金....
