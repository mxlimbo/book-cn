魔窟――重庆大厦

　　2月有机会去香港旅游，问朋友在香港有什么便宜的旅社，他哈哈笑着说当然是重庆大厦了！在亚洲各国都有一些近似于传说的有名的HOTEL。这里所说的并不是那些豪华高级的HOTEL，而是贫乏旅游者之间很早以来，并且是被广大接受认可的便宜旅社，重庆大厦就是其中一个。

　　从全世界各地涌来的贫乏旅游者，首先会到重庆大厦来看看房间，它的地点也是在香港最繁华的尖沙嘴的正中。如果你看过王菲主演的"重庆森林"你可能就会对它有了初步的印象，（哈哈，我是没有看过〕它不是一个旅社的名字，正确的说是一座杂居大厦。奇奇怪怪的印度人的商店，快要倒闭的小旅行社，含泪甩卖的杂货店..还有一些你都不知道是作什么的玄妙的店铺。重庆大厦有16层，还分为A，B，C3栋，一不留心就会迷失，不过在这种鱼龙混杂的危险的大厦里，每个人好象都是虎目圆睁，提起十二分精神的。入口处有2台电梯，我试着想看出它的年代，最后放弃了，只是我可以肯定它比我的年纪还要大。挤进去5个人就开始响起警告声，所以这里什么时候都象是清早的通勤电车，也许是香港最繁忙的两部电梯吧。夹杂在印度人，欧洲人，黑人，白人之间，耳边充斥着各种语言，空气中飘浮着奇怪的味道，紧紧的保护着我的背包，等待....终于到我了，警告声已然响起，但是人们还是熟视无睹的向里涌来。最后一个人轻车熟路的将身体缩进来，警告声突然就停止了，哈哈，果然不是凡人啊。电梯晃悠悠慢吞吞的向上爬行，哇没有见过这么慢的电梯了，和几个印度人挤在一起，咖喱饭的气味直向鼻孔里钻。好象过了好几个世纪，终于停了下来，我庆幸自己没有随着电梯坠落，而且我坚信它坠落的速度一定比上来的快！

　　这里面有Travellres hosel,chungkig house等有名的招待所。我所要找的就是travellres hosel旅社。一下电梯，迎面而来的就是一张公告。"发生火情迅速到屋顶避难！"我手里拿的旅游书中也特别强调，在重庆大厦投宿的人们第一件是就是要看好地形，认清逃走路线！哇看来是凶多吉少啊。房间不出我的想象，脏乱不说，出乎我的想象的是如此窄小，我和店主进来后就转不开身了。房间的墙纸不知被谁撕开，随着微风扭动着向我献媚，而且这个房间还没有窗户，嗨监狱里还有个窗户的啊，咦那么风是从那里吹来的呢？我抬头一看，上面有几根奇怪的20英寸左右的大管子，穿出房间，微风，啊不是，是冷风从洞中传来。一晚要港币150元。我的天啊，在物价昂贵的欧洲也没有那么贵的房间啊，要不说香港是仅次于日本的地价昂贵呢。但是这里还有电视，虽然是和电梯差不多年龄，但是还坚持工作，誓不下岗。打开看看只有两个台，而且十分模糊。和有的人一样，空有地位没有实在的东西，我想只是因为有电视才可以要高一点的价格吧。如果你要是愿意发奋一下，再加上30港币就可以睡到稍微宽敞一点的房间，还有浴盆。

　　晚上躺在床上不得入睡，想起临走是朋友给我讲的一些发生在重庆大厦的事情。有多少旅行者在这里"阴沟里翻了船"，早上醒来发现所有的东西被洗劫一空，不知道是不是用了"五鼓鸡鸣还魂香"？但是有的只是从隔壁的房间爬了过来，趁你熟睡中从容作案，哈哈堪称胆大妄为。看看我的房间只有一个入口，那么还是比较安全的了。起码不会顺窗子进来。但是请各位读者不要忘记我之前所提"注意火情！"，如果真的着了火，那么我不就是瓮中捉..??不对，是关门打...??怎么成语都不会用了？应该是龙陷浅滩啊。在这种大厦里发生火灾真的是无可幸免的。记得看到以前的报道，说重庆大厦真的发生过一次火灾，只是火不算太大，不过有个旅行者从窗口逃脱时不慎摔下而死。现在的我充分明白他的心理，如果那是他换成我的话，摔死也是毫不奇怪的啊。

　　在香港去拜会了我以后要去的公司，上司问我住在那里时，我笑着说是重庆大厦，一晚只有150港币。他坚持不信，怎么香港还有那么便宜的旅社的吗？旁边的香港人却差异的看我真正的为我担起心来。看来在电影"重庆森林"之后，对于香港人的印象那里就是犯罪者的巢穴，罪恶之城了。他说要看看我住的地方，哈哈于是就带他到了重庆大厦。门口仍然聚集着那一群印度人，远处卅卅两两的是一群警察，他吃惊的说从来没有见到过这种阵式，哈哈为了不让他魂飞魄散，就让他在下面等，我回去去取行李。当我再回来时，却找不到他了，难到是被抓走了，还是被绑了票？四下寻找，却见他钻进挥泪甩卖的杂货店里大购其物，一边念念有辞"太便宜了，太便宜了！"哈哈现在他又不怕了？？

　　以前香港有九龙城，里面也有好多旅社，但是从九龙城消灭以后，作为高举便宜旅社大旗的重庆大厦，也就名正言顺的坐上了头把交椅。最近听说它要内部改装，把HOTEL装修，电梯换装，但是我想它的复杂奇怪的内部构造是改变不了的吧？在称作"魔窟"的九龙城被消灭以后，新一代的"魔窟"的重庆大厦会不会有那一天也被消失呢？
