爱情如刃

夜凉如水，
远处传来酒吧的琴声和喧闹声，
这里只有你和我。
听你的鞋跟敲打在青石道上，
回音在幽长的街道里纠缠叹息，
在风中逸散的你的发就象我理不清的细碎心情..

漂洋过海的来看你，
尽管万千心事交错变换，时光聚敛收散，
终于还是来到了你的身边。
却是和你的最后的一夜。

回程的机票放在行囊中，
如果你说，我就会放弃，
只是你是否看出了我眼中的盈盈期盼…？
假如不说，
又是不是就可以假装忘记那即将要面临的分别？
于是我们象以往那样快乐的笑着，谈着，看着街边的橱窗。

冰凉的月光映照在你的如花笑靥，
我的心却象是寂寥的街角。
好吧,是分手的时候了，轻轻的握你的手，
今夜我是最出色的戏子，
努力的微笑，诚恳的祝福，
然后转过身来，大步离去，
感觉你的余温一丝丝，一点点的从我的掌心，手尖滑去，
遗落在风中。

我不能回头，不敢回头，
那样你就不会看到我不及拭去的泪...
爱情如刃，
我感觉到它切开我的灵魂，我的思想。
伤痛中不能分辨你是微笑，抑或含泪...
