迷　路

曾经在那样一座温柔小城，
遗失了我的心爱，
于是停留下来，
执著的寻找收集关于你的消息。

那细碎青阶上的屐痕，
转角街边的花香暗影，
抑或是吹落满天黄昏的晓风，
都是你每一种幽微细致的表情。
而入夜的霓虹灯火，浮光掠影般，
象一幕幕无声的电影，
璀璨辉煌又旋即淡去，
如你每一个令我心动的容颜，
在记忆中逐渐模糊远去.....

因为你不在我的身边，
所以迷失了方向。
那样的一座柔美小城。
我在街头穿梭寻找，
终究迷路。
