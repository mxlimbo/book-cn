思念无因

思念一个人是在什麽时候呢?

是在清晨,被微风温柔的唤醒,
看阳光透过摇曳的窗帘把明亮,阴暗变换成朦胧...
或是看天上的流云消散,流动,再汇聚成朵,
就象那个遥远的夏日午后...
是在下班以后拖着疲惫的身体,
依靠在空无一人的末班电车中, 看霓虹流掠过眼底...
是看到留言电话中闪烁的灯光,却听不到一句问候,
或是满怀期待的打开信箱, 只看见自己仓皇失措的表情...
是看雨丝缠缠绵绵,交错缠绕,润了黑夜...
是听到那首熟悉的你最爱听的歌, 将往事细细道来...
是偶然路过那片曾经与你同坐过的小店,看烛影摇曳,
或是徘徊在街角, 错身而过的似曾相识的一瞥....

思念有弦,
我以心为琴, 让时间轻柔弹拨.
