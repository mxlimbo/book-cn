希腊租车记

　　在希腊可以经常看见一些"伤痕累累"的人，而且奇怪的是他们的伤都集中在半边身子，原来在希腊的小岛上主要的交通工具是摩托车。虽然有Bus(有的岛上只有骡子，任何用汽油的交通工具都不能使用),但是又慢又少。在斯尼欧卡看有名的夕阳西下，最后的Bus是7：30，可夏天的希腊的日落是9：00。那你是看还是不看？这时如果有一部摩托车的话，就可以随心所欲了。而且在岛上迎着爱琴海的海风，衣袂飘飘，自由自在的飞驰，何乐而不为之但问题也就是出在这里。希腊岛上的交通规则很少，不对，简直就是没有，除了靠右行。不过有时我也在左边开。如果没有车，你甚至可以倒着开到海滩，不过我还没有见到这么明目张胆公然挑衅的。

　　在岛上没有交通警，没有红绿灯，没有交通堵塞。连别的国家所必须的保护头盔都免了。所以岛上的居民有的连自行车都没有碰过，就敢开着摩托车上路了，并且胆大勇猛。时而见到单手扶把，一手提着一袋瓜果梨桃，腰中扣着绳索捆绑着身后的货物，背后挂着几条法国长面包，象直叉青天的长剑，左突右转，如入无人之境...有些象北京清晨在滚滚车流中呼啸而去的中学生!但练到这种水平的毕竟少而有少，所以要到可以傲视天下的地步时，身上的伤痕就少不了了。在这样的一个国家里，你难道不想尝试一下无法无天，疯狂飚车的乐趣吗？

　　头一次到希腊，深感Bus的不方便，于是到了Mykonos岛时，决定去借一辆车。到那时为止我的自行车的水平敢说傲视群雄，汽车的水平倒也略通一二，唯独摩托车是初出娘胎头一回。在岛上有许多租车的店铺，而且大家的价钱都几乎一样，倒也没有什么选择的需要。随便选了一家。店员迎了上来，双手一合，先来了热情的拥抱，我却极为担心我嘎嘎作响的肋骨。他的手式丰富，不停的在空中挥舞，不知情的人看来倒象是在挥赶蚊虫。他旋风般转到我的面前，打着响指摊开一本色彩丰富的画报，不停的说"这里有宝马，奔驰，最适合在海滨路兜风，还有各种跑车.."等等，我只是想借一辆摩托车，他难不成把我当成了日本人？"只要摩托车？"他用手拍打着头，我的心随之一阵阵抽紧，"那你的选择的就少了！我只有两种车型！"没关系，我只想要能开动的，并且最便宜的。他的手画着圆圈，"来吧！"引我到一排车前，看我挑了一辆。"ok?! 那麽请给我你的驾照！"啊？坏了，忘了有驾照的事了，只好说忘带了。他又开始用手拍打头，我的目光随着一上一下，"那可不行，不能借给你！"难道不能宽容些吗?"不可以的。"想不到他还如此固执就象希腊的骡子。唉！大败而归。但是我企是如此轻易罢休的人吗？

　　再找一家，这次是个表情丰富的人，他的眉毛一时不停的颤动，形成他的喜怒哀乐。同样的他也向我要了驾照，我此刻早有对策。什么?!我也故作极度惊讶状，又现学现卖刚才的人的动作，双手一摊，这种摩托车难道还要驾照吗?!在我们日本(对不起，并不是我卖国求荣，只是为了以后作铺垫)象这样的摩托车是不要驾照的！你看看，你的车yamaha.honda全是日本制的，在日本没有自行车的，小学生都骑它上学!(谎话越说越大)他迟疑不决的问那你会开吗？好！等的就是你这一句。当然，还用问?!其实这种车很好开，右手是油门，脚下是刹闸，剩下的就是自行车的技术了！翻身上驹，双脚一踹蹬，蹬磕飞虎颤，不是啦，是小腿肚，档把回来时打到了我，也不敢回头看他的脸色，绝尘而起，到也游刃有余。最后来了个"拉带",看他的心痛的样子，眉毛也微微打晃，不过看来我通过了第一次的试验。好吧带我到屋内，递给他我的护照，"咦，你不是日本人那"那里人不是一样，你知道中国什么多吗？"人多啦！"啊，那是那是，不对，我的意思是中国的自行车多，日本人的自行车的技术都是中国传过去的，所以我的技术没有问题了。(又开始胡吹)我刚才那招你见过吗？"那是没有见过，但你还是少使的好!"我只是为了证明我的水平，当然不会再用了。终于经过他的考核，我可以堂堂的无照驾驶了..哈哈，"精诚所至，金石为开"的老话，还是极为正确的！

　　第二年，又到了希腊，又到了Mykonos,又到了那家店。一进门就看见了他，还记得我吗？他的眉毛快速的舞动，你又来了？哈哈，他居然还记得我，看样子无照驾驶的毕竟是少数，倒也令人记忆深刻。"你小子居然骗我，说日本不要驾照。"咦他怎么知道了？"我后来问了别的日本人，说也要驾照的。"原来如此，不过我的水平不错吧，也没有事！"那到也是，今天怎么又来了？"当然是来借车啊，不过我忘了带驾照。"啊，又没有带？"没有关系吧，我们已经是朋友了，不是吗？哈哈。我笑着狠狠的给了他一拳，"那倒是!"他也回了我一掌。

　　一会儿功夫，我又已狂飚在乡间小路上，却也不去想他临别时嘟嘟囔囔，我想他也许在说吃一堑，长一智。交友要慎重之类的话吧！
