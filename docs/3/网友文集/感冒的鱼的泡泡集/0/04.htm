苏联航空

　　没有想到我到现在为止，利用最多的航空公司竟然是苏联民航！

　　去年夏天第一次去希腊，打电话给旅行社，一般来说价钱都是在12万日元左右，我问那位小姐，"那麽最便宜的是那一家呢？"回答就是苏联民航，只要8万日元。但是必须在莫斯科停住一个晚上。因为转乘去雅典的飞机是第2天早上，虽然如此第2天到达雅典也是10：00左右，并不耽误多少宝贵的时间。

　　我因为是独身旅行，所以要订单人房间，价格8000日元。拿到机票的时候，看到HOTEL的预约证，精美别致，封面是莫斯科国家芭蕾舞团的写真。哈哈，我还没有住过这麽高级的HOTEL呢，8000元大钞，在日本也能住上3－4星的酒店了，物价低廉的俄罗斯许不成是5星级了！

　　饱经风霜的苏制老式客机停靠在离侯机室最远的角落里。成田机场是以付钱多少来决定停靠位置的。也有些太现实了吧！看到飞机的时候不禁有些嘀咕，难道要座这飞机飞9个小时？！唉都上了贼船了，后悔也来不及了。

　　真纳闷，俄国的胖子应该不少，为什么座位却那麽狭小？我身高185公分，只好把双腿折叠起来塞入座位中。这下可好，甭想出来了。幸好我的屎尿还算少。飞过西伯利亚，机体开始旋回、窗外一片片森林，有的民家竟然建在森林当中，真不知他们如何回家！降落在莫斯科国际机场，顺利做完换乘手续，等待去HOTEL。谁知这一等竟然用去了5个小时。据说先前有一批巴基斯坦的团因为误机，加上电脑故障，所有人都在莫斯科停住。轮到我们当然费时间。倒也闲来无事，认识了一些朋友。好不容易到我们了，一位身着军装的苏军，腰别手枪，带我们一群乌合之众，大包小包，拖泥带水的下得楼来。"没开玩笑吧，这么小的一辆车，难怪花时间！"一次只能挤进5个人，再加上行李，快成罐头了。

　　开出机场，进入所谓的高速公路，眼前一片黑暗，原来他们不开路灯的。车辆也少的可怜。"这就是苏联那！"东转西绕的，拐进一条小路，且慢！放眼望去，没有灯火辉煌，没有高楼大厦。。难道我的5星级HOTEL就在其中吗？！是的，又拐过几个路口，有一座4层小楼。"到了！"下车一看，门口有实枪荷弹的红军，黑暗中只见两眼炯炯放淫光（SORRY 应该是银光〕、把守着个个通路要道。进入苏联需要签证，我们属于过境，但决对不能迈出HOTEL，呸，难道还怕我们黑在苏联不成？！

　　在机场的几个小时，早就把机内食消化完毕，现在只想大吃一顿。下到一楼，餐厅早已歇业，没办法只有一个小卖部了、东西倒有一些，只是，只是这里只收美元！！而且奇贵无比。搜尽衣兜只有5＄，买一瓶COLA和一块饼干，要了4＄，整个是抢钱啊！太没有人性了！回到房间什么都没有，连热水都没有！！8000元那，也要打工8小时，竟然被骗的血本无回！5月的莫斯科的一家偏僻的，简陋的HOTEL里，哆嗦着冲了凉水浴，喝了冰凉的COLA，睡去了。。。我是要去旅行哪，还以为是来受罪！

　　清早电话铃声响起，是预约铃，洗涮的时候又是铃声大作，"bus is coming,get down,now!""啊？不是6：00才来吗？我还未收拾完呢。""NO!get down now!!!this bus is last!!"慌忙的冲下楼来，大骗子！那有车的影子！一会儿功夫又冲下几位，全是狼狈不堪。这时前台扔过来一包东西，"早餐！"喝倒没有料到！打开一看俄国黑面包，酸奶，奶酪，巧克力，矿泉水。。很丰盛，但是很难吃。留下矿泉水，其他的－扔掉！

　　从回忆中清醒过来，为什么，为什么我又选择了苏航？！唉又是贪图一时便宜。还立下誓言再也不座苏航呢。不过也曾发誓决不一个人再去希腊呢！原来我是这样的性格的人啊，可要提高警惕了！进入机舱，依然轻车熟路的叠好双腿。你要问我在飞机上干什么？当然是睡觉了，即没有报纸，又没有电影。昨晚一夜未睡，就是作好准备工作。雨越来越大，台风到了吧？！我可把命又交到俄国人手里了。据说苏航的飞行员都是原空军出身，退役后转入民航。胆大勇猛倒不必怀疑，只是冷静心细值得推敲！实际上每次座苏航，安全着陆时机舱内都会响起如雷掌声！起初以为是为驾驶员的出色表现鼓掌，但发现不光乘客，连空中小姐也兴高采烈的鼓掌后，我想也许大家是为了一次重生而欢呼吧？
