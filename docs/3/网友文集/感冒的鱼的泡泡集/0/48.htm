许愿池

那是你最喜欢的地方.
在看得到小小港湾的山坡,
有一泓清澈的池水,
映着天上的流云散开又汇聚成朵,

你说那是你的许愿池.
试试看很灵验的啊 。
不忍拒绝你央求的眼神,
于是背转过身,面对你的盈盈笑靥,
虔诚的许愿,投下一枚硬币.
看它在阳光下旋身翻转,
轻轻巧巧的闪过一道美丽的弧线,
落入池中,激起小小的涟漪....

你问我许了什么心愿,
我笑而不答,
只是痴痴的看着你眼底流溢的笑意。

经年以后,再次来到池边.
清澈的池水只映出仓皇失措的我的脸.
竟然,竟然只有我一个人.

看静静躺在池底的硬币,
想你如水般清凉笑颜.
还有那个静静的夏日午后的小小心愿.
记得要让你永远快乐幸福,
却忘记要把你留在我的身边.
