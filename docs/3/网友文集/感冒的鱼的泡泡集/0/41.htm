我喜欢这样的日子

大雨的日子、
穿一件旧衣服、
舒适、体贴。
在临窗的地方找一个柔软的角落把自己窝进去。

选一张干净的唱片、让音乐随雨滴弹跳纠缠、
透过窗还可以看到屋沿下的雨滴、
飘来散去又化成千丝万缕。

找一本书、厚厚的书。
头上的一盏灯、射出一晕光、
暖暖的把我包住。

就在我触手可及的地方、
你同样专心的看书。
但当我望着你的时候、
总能够看到你温柔的眼波和浅浅的笑靥。

手边有一杯清茶、
是你的、也是我的。
把书放在伸开的腿上、
我用左手翻页、
而我的右手、
与你的左手、
轻轻相牵。
