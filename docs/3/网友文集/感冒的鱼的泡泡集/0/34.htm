那一夜北京有雨

那一夜向你发出邀请，去寻访这座古老城市的雨夜。
看着我期盼的眼神，你不忍拒绝，
于是和我慢慢穿越静静睡去的城市。

路灯下旋绕盘恒的细密雨丝，
缠绵不绝，湿了你的肩和发。
宽阔笔直的长街漆黑如镜，
映的霓虹如水波荡漾流溢。
偶有车象风一样掠过我们身边，
红色的尾灯无声的闪入黑暗中，淡去，
象今生永不能遗忘的爱恋，
常自寂静的夜梦里，轻巧滑过...

花去身上最后几枚硬币，
错过最后一班地铁，和你漫步在街头。
有一种想和你紧紧依偎的冲动，
似乎除非这样，谁都不能感觉到谁，
但你只是轻悄的拉过我的手....

雨住了，
洒水车寂寞的驶过街头，
月光将长街洗刷的更加晶莹清凉。
你轻快的跳过一个个闪亮的水洼，
因为紧紧的抓住我的手，
好象我也和你飞翔在这落雨的街头。

转过头想寻找你的双眸，
因为逆光，看不清楚，
于是我闭上双眼，
用心感受你每一个幽微细致的表情...
雨会停，月光会淡去，
与你终究也是会散的，
但我会记得的，
这城市不为人知的静谧清新和你再不曾给我的温柔...
