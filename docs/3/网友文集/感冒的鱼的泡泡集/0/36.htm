惹上寂寞

又是个静静的夜。

回到家里，拿起电话才发现找不到打给你的理由?
窗外阴雨连绵，对面屋顶的灯光也被映的朦朦胧胧，
收音机里的歌手悲伤的唱着她的心事，
电话却始终没有响起...

真是个寂静的夜啊!

当我想念你的时候，你又在想到了谁呢？
夜晚总是会做梦,有时哭泣，有时微笑，
连自己都以为曾经忘却的事，有时却又回到梦中?
也许这才是真正的我自己。
回到现实世界，有谁能看清我？
有谁能察觉在我笑容背后所隐藏的一缕寂寞？
有时不明白为什么人不能活的随心任性？
悲伤时哭泣，快乐时微笑，
让自己有颗孩子般纯洁的心?

所以我想念，想念你那孩子般纯洁的笑容，
在这冷冷的，静静的夜里。
夜深了，寂寞象这绵绵细雨，
无声无息的由心底将我淹没。
年轻的心第一次，惹上寂寞，
竟失措的，一夜难枕.
