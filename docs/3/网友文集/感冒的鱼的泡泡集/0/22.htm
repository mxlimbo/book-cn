猜　心

轻轻的揽过你的手，
柔柔的执在我的掌心，
感觉到你的纤纤素手在微微的颤动，
仿佛是我微荡的心湖。

你笑着问"如果有一天我们老去，那么你还会记得我吗…？"
我微笑，然后细细的顺着你的修长的手指抚去，
一丝丝的融化你指尖的那一抹冰凉，
化为淡淡的温柔，
象缠绕你眉间的一缕长发，
你知道，
此时这握在你我手中的温柔就是日后我们相见的誓盟...

缓慢的，小心的，
你的手指在我掌心移动，
我看到你眼底的那一霎狡狭的笑意，
我用心感受你冰凉的指尖从我的掌心滑过，
在我的心湖划出一波波涟漪，
和着一种甜蜜的痛楚，
扩大，交叠，消散，慢慢的将我拥住。

我知道你写的是那个字。
但是，我不能说出来。
它是一个诡秘的咒语，
如果不说还可以相安无事，
倘若说了，这就是我要找的，
蓬然一声，灰飞烟灭。
于是，在你的指尖和我的掌心，
继续着这个猜心的游戏。
