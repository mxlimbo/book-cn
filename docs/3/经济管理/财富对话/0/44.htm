柳传志

Liu　Chuanzhi

　　柳传志，联想集团有限公司董事局主席兼总裁、中华全国工商业联合会副主席。1967年进入国防科工委十院四所从事科学研究工作，1970年调入中科院计算所从事计算机外部设备研究工作。

　　联想集团（位于中国北京市）是中国最大的计算机产业集团。集团发展的联想品牌微机1998年产销78万台，为中国计算机市场占有率第一，亚太地区第三名。现自行研制开发的微机主板和附加卡已大量出口海外，年销量达到300万块，成为世界五大板块制造商之一，1998年销售收入达176亿元人民币。
