周明臣

Zhou　Mingchen

　　周明臣，1992年任中国粮油食品进出口（集团）有限公司董事长兼总裁。1970年起先后任中国五金矿产进出口总公司副处长、处长、副总经理，中国仪器进出口总公司总经理。

　　中国粮油食品进出口（集团）有限公司，自1952年成立后，已逐步发展成为集贸易、实业、金融、信息服务和科研为一体的大型企业集团，排名《财富》杂志1999年世界500强企业第362位。截止1998年底，公司海内外资产总额达47.7亿美元。
