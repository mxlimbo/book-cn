美亚在线：抗战到底

　　时间最后指向了2000年10月6日：

　　上海"美亚在线"（www.cnnews.com）在这一天收到CNN（美国有线电视新闻网）的律师Christopher B. Roblyer于当天发出的电子邮件，其主要要求和意见如下：

　　一、美亚在线已经注册了CNNEWS.COM的域名，并在此域名下，提供与CNN具有竞争性的中文新闻内容。

　　二、美亚在线注册并使用的www.cnnews.com侵犯了CNN商标的独家使用权。

　　三、从CNN的利益出发要求美亚在线立刻采取以下措施：(1)停止使用www.cnnews.com的域名.(2)将www.cnnews.com的域名转交给CNN.(同时，将Network Solutions公司即NSI的域名更改协议的复印件随函交给CNN.)(3)同意在将来的不再注册及使用含有或部分含有CNN标志的注册商标，服务商标，产品标志，公司名称，域名等等.面对是全球最大也是最盈利的新闻机构之一CNN的嚣张气焰，美亚在线迅速予以还击。在美亚在线给CNN及其律师的第一封复信中，以强硬的态度拒绝对方的"通牒"，并表示，如果要打官司，他们也将聘请国际知名律师奉陪到底。在信中他们仔细阐述了"CNNEWS"的含义。注册域名中"CNNEWS"的正确解释应为"China Network News"，即"中国网络新闻"之意的缩写。

　　好样的，美亚！

　　美国人吃了一惊，看来中国人也不是好欺负的。看来美国在线的经验也行不通了。在10月16日CNN律师的第二封来信里，美国人玩起了缓兵之计：Would you please send me your reply in Enlish if at all possible.Otherwise, it will take time to have it translated。（如果可能的话，请您发一份英文版的回复信给我，否则翻译起来将花费时间。）

　　美国人心虚了。因为根据《统一域名争议解决政策》的规则：

　　一、CNN不具备提起域名争议的条件

　　根据《统一域名争议解决政策》第4(a)条款的规定，提起域名争议必须同时具备以下条件：

　　(1)被提起争议的域名与投诉人所持有的商标或服务标记相同或具有误导性的相似；且

　　(2)你方(指域名持有人，下同)对该域名本身并不享有正当的权利或合法的利益；且

　　(3)你方对域名的注册和使用均为恶意。

　　投诉人在域名争议解决程序过程中对于以上三种情形的同时存在，应负举证责任。

　　应当指出，以上三个条件必须同时具备，CNN才可以作为投诉人向争议解决机构提起域名争议。然而，事实恰恰相反，以上三个条件都不具备的CNN根本不具备提起域名争议的条件！

　　首先，cnnews.com这一域名与CNN商标既不相同又不相似

　　其次，中国方面对cnnews.com这一域名享有正当的权利

　　众所周知，在互联网域名方面，中国的代码是CN。那么，cnnews就是CN加上NEWS，也就是"中国人的新闻"的意思。

　　第三，中国方面对于cnnews.com域名的注册并无恶意

　　根据ICANN的《统一域名争议解决政策》第4(b)条规定，"恶意注册和使用行为"是指这样几类行为：

　　(1)有证据证明，你方注册或获得域名的主要目的是为了向商标或服务标记的所有者或所有者的竞争者出售、出租或其他任何形式转让域名，以期从中获取扣除了与你方持有域名的相关费用之后的额外收益；或者

　　(2)根据你方自身的行为，即可以证明：你方注册或获得域名的目的是为了阻止商标和服务标记的持有人通过一定形式的域名在互联网络上反映其商标；或者

　　(3)你方注册域名的主要目的是破坏竞争对手的正常业务；或者

　　(4)以牟取商业利益的目的出发，你方故意以连接源、赞助者或联接者的形式造成与投诉人所持有的商品或服务标记间的混淆，从而诱使互联网络用户访问你方的网站或者其他联机地址，以及订购你方的产品或服务。

　　很明显，中国方面注册和使用cnnews.com域名并没有以上所列的恶意，我们的目的很单纯，cnnews――中国新闻，与CNN何干？

　　由此可见，CNN的所作所为完全是无理取闹般的威吓之举。它这种看到美国在线"成功"的利用中国企业的无知强取豪夺中国企业倾注了巨大心血和价值的域名后所采取的行经，实在是无耻之尤。

　　可是，CNN光打雷不下雨，多少让人觉得奇怪。事情过去那么长时间了，CNN就只有两封律师信，其他的动作好象一点都看不到了。NAF(全美仲裁论坛)的仲裁主管助理Kristen E. Porter在给笔者的来信中谈到：至今为止，他也没有发现有任何CNN关于对“CNNEWS”的起诉。并且，通过对《统一域名争议解决办法程序规则》的分析，CNN如果诉讼的话，完全占不住脚。另外，身为国家级的互联网络信息中心，负责管理和运行国家顶级域名CN的CNNIC，其域名http://www.cnnic.net.cn也是同此事件有关联。如果CNN告倒了CNNEWS，就算放过了CNNIC，CNNIC也会脸上无光，没了面子。因此，CNNEWS定会招来诸多关注。但笔者认为，其中也许还有更深的意义。至今为止，CNN也没有向任何一个域名仲裁机构递交了起诉书。这也足以证明了美亚的反击政策是非常正确的。如果不予以CNN坚决的回击，说不定美亚会变成又一个金智塔。

　　国内的网友在输入网址http://www.cnn.com的时候，相信一般来说浏览器会告诉你“该页无法显示”。不用猜疑，不是CNN的服务器当了机，而是由于前段时间CNN发布对中国政府的反动言论而招致封锁。这不由得使笔者联想到CNN选择有北京深厚背景的“美亚在线”作为起诉对象，是否有其另外的想法呢？其中的玄妙不得而知。

　　这场关于域名的斗争才只是一个开始，我们还将面临更多的关于互联网知识产权的斗争。不仅是域名，还有软件名、技术专利等等更广泛的领域。五十五年前，我们迎来了抗日战争的胜利；五十五年后的今天，我们何时才能迎来域名斗争的胜利呢？
