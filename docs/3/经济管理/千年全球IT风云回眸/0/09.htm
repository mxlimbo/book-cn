网易上市：门户站点的生死线？

　　自上周一开始，国内知名的门户站点——网易（ｎｅｔｅａｓｅ．ｃｏｍ）已经开始其全球路演。如果一切顺利，预计６月底７月初，网易就将以“ＮＴＥＳ”的代码在纳斯达克正式交易。至此，网易将成为继中华网和新浪网之后，又一家在纳斯达克挂牌的中国概念网络股。

　　在这三家门户型站点为成功上市弹冠相庆、享用资本盛宴之时，许多尚未上市、但是模式与它们相近、并且同样以纳斯达克为目标的网站，是否还有在资本市场中分一杯羹的机会？风险投资家是否还对他们怀有足够信心？甚至这些网站是否还可以生存下去？任何时候的衡量标准都是时机。

　　美国投资银行美林证券是此次网易上市的主承销商，由于受证券市场相关法规的制约，目前各方尚处在禁言期内，美林证券北京首席代表何宁先生不愿对网易上市作出过多评论，他只是表示，虽然目前全球股票市场的高科技板块还没有彻底从三个月以前的震荡中恢复过来，但是，他个人不认为网易选择这个时机上市是“逆流而动”。现在纳斯达克股指维持在近４０００点左右，已经比最严峻的时候要强许多。何宁先生称，网易现在面临的时机自然无法与５０００多点的高涨时期相比，但是，起码处于整个市场逐渐复苏的阶段。

　　但是，也有分析人士指出，如果拿已经在纳斯达克上市的两家中国概念门户网站——中华网和新浪为例，网易的市值不会比这两家高。

　　中华网的股价自去年夏天到今年三月上升了１２００％，其间，从去年１２月以后还分别进行了两次拆股。伴随着三月份以后的股市动荡，它的股价下跌超过了一半。即便如此，去年秋天买入中华网股票的投资人都因为持有这只股票获利甚丰。投资界认为，中华网业绩不错的原因是，它是第一家在海外上市的中国概念网络股，而且一度是中国乃至亚洲网站在股市上的代言人。

　　新浪上市三个月以来，演绎了一场与险恶的市场氛围的战争，并且成为这场战争中为数不多的胜利者，其股价从挂牌时的１７美元上涨到现在的３４美元左右。据美国资深财经专栏作家大卫·库拉帕卡评述，新浪可能最有资格成为“中国雅虎”，因为其向网民提供的信息真实可靠。最重要的是，就目前形式看，新浪在中国市场已经打败了雅虎。

　　因此，分析人士称，网易及有望随网易之后上市的网络公司并不具备以上两家网站的“不可复制性”价值。根据一般的投资常规来看，虽然市场对于每只股票的偏好程度并不完全由上市顺序决定，但是，除非新加入竞争的网站有足够的、惟一的特性，投资人惯常是不会考虑改变其投资组合的。

　　在这种情况下，市场的资金容量又是一定的，供求关系决定越晚进入者，获得投资人青睐的机会、以及所要投入的成本就越高。这样，不仅仅是网易，未来其他有可能上市的网站也许都很难再重复中华网和新浪的胜利了。

　　不改变商业模式只有死掉

　　当媒体还在争论门户和电子商务网站究竟谁能赚钱的时候，一些“．ｃｏｍ”已明白及早抽身的道理，悄无声息的不再提“门户”，而是倡导出一个又一个新名词。

　　几乎没有人再像以往那样奉ＩＣＰ和ＩＳＰ为神明，取而代之的是ＡＳＰ（ＡｐｐｌｉｃａｔｉｏｎｓｅｒｖｉｃｅＰｒｏｖｉｄｅｒ应用服务提供商），ＰＳＰ（ＰｌａｔｆｏｒｍＳｅｒｖｉｃｅＰｒｏｖｉｄｅｒ平台服务提供商）。甚至，有些“Ｏ级”人物还准备转移战场，把无法再进行下去的网上模式搬到网下，依靠网上已经形成的品牌效应，做生活中的有形产品，也就是说，如果你将来看到以某某网站名称为商标的服装、运动用品或者任何你生活中需要的东西，千万不要惊讶。

　　另外一个势头就是转向无线互联。ＷＡＰ所到之处，不仅仅是那些移动设备生产厂商为之鼓噪，“．ｃｏｍ”甚至成了主角。

　　搜狐也许称得上是迄今仍旧最坚定地高扛门户大旗的“仅存硕果”，如果说已经上市的新浪、挂牌在即的网易令搜狐真有种“起个大早，赶个晚集”的尴尬，那么，搜狐无疑把无线应用当作一个“新经济增长点”。搜狐目前已经开始与中国电信、中国移动和中国联通合作，在全国９个省市开通了通过ＷＡＰ上网的服务。

　　ｅ龙公司已经在几个月前与Ｍａｉｌ．ｃｏｍ换股完成购并，此举令ｅ龙避免了“挤破头争上市”的辛苦，即便如此，其董事长唐越也没有清闲下来——因为没有了“现在活着的压力，还存在能否继续活下去的压力”。唐越一直密切关注着无线应用的动向，不排除通过资本运作，进入这个领域的可能。

　　而一些新生代网站由于已经不具备中华网、新浪和网易那样的时机，所以在进入互联网产业时，就已经绕开了门户的圈子，直奔ＷＡＰ。比如，掌中万维网的首席执行官肖湘阳曾经是摩托罗拉部门经理，现在的业务也是提供支持各种无线终端产品的无线互联解决方案。

　　被收购——长痛不如短痛

　　最新一条关于收购的消息是，国内一个拍卖网站已经被另外一个起家于传统拍卖行的网站收购，其实，不止一家网站明白自己撑不到去纳斯达克的那一天，尽早被收购是一个化长痛为短痛的良方。中雇网的创始人就曾经表示，开始做这个人才招聘网站时，就明白融资上市是一个比经营管理更令人挠头的事，虽然被收购的感觉未免“有些怪怪的”，但是，这已经是对创始人投资的最好回报了。

　　思科一度成为世界市值最高的公司，其首席执行官钱伯斯就是一个擅长收购的高手。其实，对于已经上市、获得足够发展资金的公司而言，收购也是一条更快强占市场的途径。中华网通过投资参股的形式，已经在多家网站拥有股份；新浪的首席执行官王志东公开表示：“新浪虽然仍旧把主要精力放在发展内容上，但是，不排除收购合适对象的选择”；而率先在香港创业板上市的“Ｔｏｍ．ｃｏｍ”也认为，现在的市场虽然表现低迷，却是收购的好时机。

　　互联网市场里，第一名是胜利者，第二名有生存的余地，其后的就会被外界遗忘。对于那些知名度不太高的小网站来说，独立上市的机会微乎其微，通过收购成为一个大网站的某个频道，的确是目前所能找到的最好出路。

　　已经上市的公司操作收购，可以采取现金加股权置换的形式进行，这种方法能够减轻收购公司的现金压力，被收购公司的员工又可以保持一部分已经许诺的股票期权，此外，中国加入ＷＴＯ之后，现存的种种对于外资进入电信领域的限制都有松动的迹象，一旦国外的网站们名正言顺地裹携美元涌入国内，这些土生土长的“．ｃｏｍ”们受到的冲击可想而知。因此，收购对于已经上市的公司来说，也是一个“花小钱，办大事”的方法。
