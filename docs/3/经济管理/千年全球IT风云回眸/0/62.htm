网络震撼大片之国内版(一)

　　对于国内IT业而言，2000年是一个有些迷茫的年份。自诩为弄潮儿的中国IT人第一次发现自己的方向感有些模糊；判断力有些不够用。虽然整个行业仍然在迅速成长，然而NASDAQ的地震却让大洋这边的淘金者们有些不知所措……于是，在命运的十字路口，中国的IT人用仅存的一点理性与激情，选择着属于自己的未来之路。

　　从TOM.COM的横空出世，到国内三大网站登陆NASDAQ，到国内网站的裁员风暴，到传统企业大规模进军网络业……所有这一切都证明，2000年对于中国IT业而言，绝非一个平静无事的年份。然而与以往不同，我们再也无法像以前那样对一个事件的前景影响做出客观的评价，因为一切似乎都变得与以往大大不同了……

　　方姜吴之争

　　领衔主演：方兴东、姜奇平、吴敬琏

　　友情客串：中国IT评论界和经济学界知名的和不知名的辩手们

　　3月底，中国IT评论界发生了一场空前激烈的“泡沫与制度之辩”，论战的双方分别是IT评论新锐人物方兴东、姜奇平与著名经济学家吴敬琏。这场辩论实际上是传统经济理论与网络时代新的经济观的一次碰撞。尽管短时间内还无法说清谁对谁错，但是这场论战清楚地表明，以网络为代表的新经济中的许多现象已经真正吸引了传统经济学家的注意力。新经济对传统经济渗透和变革的步伐，正在逐渐加快。

　　而就在这次论战发生之后不久，就赶上了NASDAQ网络股暴跌的事件。也可算是为这次辩论加上了一个很好的注脚。

　　震撼强度★★★★☆

　　

　　裁！裁！裁！

　　领衔主演：TOM.COM、人人网、中华网、硅谷动力

　　友情客串：香港苹果网站、索易、千龙新闻网等进入下半年，有关网站裁员的消息就成了许多媒体最敏感的话题。

　　7月31日，一向“勇猛”的TOM.COM宣布裁员80人，占其员工总数的16%；8月14日，有新闻集团作为后盾，实力强劲的人人网宣布大幅裁员4成，共计102人；8月20日，隶属于中华网旗下的上市公司香港网（hongkong.com）宣布裁员47人；紧接8月26日，中华网宣布，在内地裁员48人；8月31日，TOM.com的旅游网站又爆裁员新闻，宣布裁减员工80%；10月31日，香港苹果网站继前次裁员百人之后，再次大规模裁减员工约100名；11月4日，硅谷动力大规模裁员的消息传出：网站将裁员20%，并且有一大批高层管理人士将会丢掉饭碗……

　　原来被认为是“金领一族”的网站员工忽然发现自己引以为自豪的饭碗正面临前所未有的危机。在风传了很久之后，网站的裁员风暴终于在2000年的下半年浮出了水面。只不过许多人对裁员的公司里面竟然还包括了握有大把现金的TOM和中华网深感意外。在投资人越来越看重现时盈利能力的情况下，在“传统”企业中最常见的“减员增效”的把戏终于被拿到了知识精英们的面前。网络公司一打一打地往里招人的美好景象成为历史了。有消息表明，2000年下半年刮起的网站裁员风暴其真正规模要远大于我们目前所看到的部分。许多网站由于怕引起传媒的注意而对裁员的动作采取了相对低调的处理方式。不过，对于那些已经风雨飘零的小网站来说，裁员显然还有更现实的理由――“裁”掉所有的员工，然后倒闭。

　　震撼强度★★★★★

　　

　　TOM哥哥的故事

　　领衔主演：TOM.COM

　　友情客串：163.net、鲨威体坛网站、羊城报业、昆明风驰等

　　在李嘉诚旗下的TOM.COM横空出世之前，TOM这个名字能让我们想起的大概是那只一直有些傻的、经常被老鼠耍的笨猫。然而李嘉诚很快用实际行动向世界宣布，他的这只“TOM”并不是什么温顺的小猫，而是一只凶悍无比、胃口奇大的大白鲨。

　　3月1日，TOM.COM借助于网络股的热潮以及李氏家族的良好声誉，在香港创业板成功上市。该股在认购期内获得近700倍的高额认购，公开发售股份冻结资金高达509亿元，全香港约有三十余万人参与认购。上市首日，股价暴涨近5倍，可以说是把网络股当时的风光表现得淋漓尽致。

　　有了股市资本的支持，TOM自然要有一番作为，于是TOM.COM看上了163电子邮局的传闻大约在6月底就已经出现了，但是当时业界人士对此多不以为然。然而到了7月份，这件事情已经变成了公开的秘密，当时最乐观的估计是163.net可以卖到7亿元港币以上。结果到了8月中旬，当TOM正式公布要收购163.net时，此事早已经不算什么新闻了。

　　就在收购163.net刚刚敲定的同时，TOM.COM对于鲨威体坛网站与羊城报业相关股份的收购也浮出了水面。8月14日，TOM.COM宣布，将以1.5亿元收购内地体育资讯网站鲨威体坛全部股权，并以2.36亿元的资产置换内地运动广告及大型体育活动经营公司羊城报业七成股权。不过，这时的TOM自己也开始面临越来越大的压力。在8月份新一季的财务报告中显示，TOM在上市第二季度的经营亏损额高达1.485亿港元，是去年同期的40多倍。

　　但是TOM.COM依然不愿回头。在新一轮融资成功之后，10月11日，TOM再次使出收购的招数，以发行2.78亿元新股的方式收购中国内地最大型户外广告媒体公司之一昆明风驰明星信息产业股份有限公司(“风驰”) 49%或以上的股权。11月21日，TOM又宣布，已与明报企业有限公司（下称「明报」，旗下两间附属机构签署谅解备忘录，收购明报全资拥有的中文时事杂志《亚洲周刊》的五成股权。

　　所有人都在担心TOM近乎疯狂的并购活动会得到一个怎样的结果，但执着的TOM，李嘉诚的TOM依然我行我素。看来，TOM的未来究竟会如何，只能由时间来做出回答。

　　震撼强度★★★★☆
　　

　　摇啊摇，摇到NASDAQ

　　领衔主演：新浪、网易、搜狐

　　友情客串：中国概念股

　　三大门户做了很久的NASDAQ上市梦，在2000年终于实现了。然而这究竟是一出喜剧抑或是悲剧？只怕是没有人能说得清楚。

　　4月13日，尽管纳斯达克上空已是阴云密布，新浪首日上市却得了个开门红，其挂牌发行量为400万股，发行价为每股17美元，共集资约6,800万美元。上市当日，其股价就上涨了22%。虽然比当初中华网上市首日的表现相差很远，但是考虑到NASDAQ网络股一个个“尸横遍野”的惨状，这个成绩已经足可以让新浪的人们弹冠相庆了。

　　新浪暂时的好运并没有帮助那些“走在后面的兄弟”。6月30日，网易在美国纳斯达克股票市场刚一上市就流血不止。该股开盘价是15.5美元，开盘后一路下跌，收市时，该公司股票价格已经变成了12.125美元，下跌了21.77%，上市当日就跌破发行价。面对此种惨淡光景，已经由公司CEO变为CTO的网易最大股东丁磊用“不以物喜、不以己悲”形容自己当时的心境――只是不知道丁磊是真的有古人的那份豁达，还是在故作镇静？

　　7月12日，三大门户的最后一位成员搜狐“姗姗来迟”，在美国纳斯达克市场上市交易，由于搜狐公司降低了上市发行价格，挂牌当天并未发生大幅跌破发行价格的情况，上市当天，以13.0325美元开盘，以13美元报收。张朝阳终于把搜狐的“爪印”小心翼翼的按在了NASDAQ这个大舞台的地板上，尽管这时的搜狐显得有些弱不禁风。

　　搜狐上市5天之后（7月17日），中国概念股开始进入了有史以来最为黑暗和沮丧的时期。在纳斯达克上市的“中国概念股”在大盘暴跌的情况下紧跟大盘一路走低，至7月28日，搜狐和网易双双跌破五元大关，向垃圾股的悬崖滑落。到年底，搜狐和网易的股价已经跌落到了3美元以下，新浪也已经跌到不足4美元。当初抱着淘金梦“不远万里”到NASDAQ上市的丁磊张朝阳们，看到这副景象，这时恐怕连哭的力气都没有了。例如张朝阳就幽幽地表示，自己可能已经“不幸”跌出了中国富豪前五十名的行列了。试想一下，当年的知识英雄拿着个计算器每天算着自己财富又缩水了多少，那种感觉恐怕也是“别有一番滋味在心头”吧……

　　震撼强度★★★★★

　　
　　网络民族抗战

　　领衔主演：姜伟、雅虎

　　友情客串：全国各新闻媒体及爱国网虫

　　去年美帝国主义轰炸我大使馆后,红客这一个新名词流行起来了。在今年台湾大选期间，两岸的黑客也相互过招了好一阵，无疑，网络会是表达保护民族尊严意愿的一个新的重要场所。

　　8月份重庆中学生姜伟发现雅虎在其网页中将中国的香港、台湾地区列为独立的国家后，向雅虎发去了一封电子邮件，要求雅虎就此道歉并立即改正其错误，之后全国各媒体纷纷报道了这一消息。迫于舆论压力，雅虎中国区负责人致电姜伟表示，已经对错误进行了更正，就用不着道歉了。但雅虎仅仅更正了其在中国大陆网站的错误，而在全球的22个其它语种网站仍然存在这种有辱中国国家尊严的谬误，雅虎的负责人扬言，其它语种的网站不受中国法律制约。

　　红客是一种民族抗争，这位重庆少年也是。在网络将越来越主导话语权的今天，网络涉及民族尊严的斗争会更多更激烈。

　　震撼强度★★★☆☆
　　

　　域名风暴

　　领衔主演：中国互联网络信息中心（CNNIC）、美国网络解决方案公司(NSI)和美国的IDNS

　　友情客串：全体有志网民均可参与

　　11月7日中文域名之战终于拉开帷幕，主要竞争者是：中国互联网络信息中心(CNNIC)、美国网络解决方案公司(NSI)和美国的IDNS。目前各方所采用的技术不同，接受的域名注册方式也不同，都想争取主人翁的地位，因此造成一定的混乱。域名之争实际上是以前域名抢注的延续和扩大，是不同国家、不同企业之间的利益之争。由于英文顶级域名资源已经枯竭，其他语种域名所蕴涵的商机无穷。

　　震撼强度★★★★☆
