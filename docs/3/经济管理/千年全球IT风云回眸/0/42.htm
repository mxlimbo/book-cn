五角大楼：沉浮在黑客攻击的旋涡

　　五角大楼的首席信息官称，1999年针对美国国防部的电子攻击为22000次，2000年的头7个月已经发生了14000次。

　　国防部的一位助理秘书亚瑟・马尼表示，大部分的攻击没有构成危害或危害很小，但有些为外国工作的黑客曾攻入了国防部的非机密的计算机系统，并下载了大量的资料。但据国防部的官员表示，还没有黑客能攻破机密的计算机系统。

　　由于1998年安装了可以探测攻击的设备，美国国防部在1999年首次具备了精确统计攻击次数的能力。1999年，它共侦探到22144次攻击，大约3%（600起）造成了系统的临时关闭或其他损坏，另有1%（约200起）侵入了非机密的计算机系统。今年的攻击次数则上升了大约10%，而造成危害的比例则与去年相似。

　　在接受采访时马尼预计，未来的攻击次数将“有增无减”。他说，大部分造成损害的攻击是由软件中的漏洞引起的，这些软件分别来自微软、Netscape和Lotus等公司。

　　他还指出，尽管五角大楼在产品的设计阶段就与软件公司进行了密切合作，但要使数百万行来自印度、爱尔兰、以色列和其他国家的软件中不存在漏洞几乎是不可能的。他说，“我们根本搞不清楚这些软件是在哪个国家编写的。”

　　一位要求不透露姓名的五角大楼的官员表示，这些漏洞中有很多是无意的，但有些则似乎是由软件的开发人员故意留下来的，另外还有一些本来是专供系统管理人员使用的，但被黑客发现了，并被用来对系统进行攻击。因此，五角大楼基本上不在敏感的系统上使用现成的商品软件。

　　马尼表示，五角大楼的电脑安全任务非常艰巨，整个国防部有大约10000套计算机系统和150万台个人电脑，其中大约有2000套系统在运行着关健任务软件，这意味着它们必须不间断地运行以处理各种任务。他说，“我们必须全天候地对企图对我们进行军事冒险的企图、计划进行调查。”他指出，五角大楼已经发现有些国家在从非机密的计算机系统上下载资料。

　　8月份，国会曾在2001年度的国防预算中额外拨出了1亿6300万美元用于解决计算机系统的安全问题，但有关专家表示，这笔资金对于解决五角大楼的计算机安全问题来说是“杯水车薪”。

　　马尼说，引起五角大楼对其计算机安全问题关注的事件发生在1998年，当时，加利福尼亚的几名年青人在一名以色列人的指导下，利用Sun公司软件中的一个漏洞对五角大楼的几个机构的计算机系统进行了攻击，他们的目的是试图对以色列发动军事进攻，并可能联合对国防部的信息设施发动进攻。

　　军用计算机系统的管理员曾针对加利福尼亚黑客案中暴露出来的系统弱点发出过警告，但没有受到足够的重视。
