博库状告TOM.COM侵权

　　近一段时间以来，中国网络版权官司频繁，博库与TOM.COM两大网络巨头也未能幸免。　　

　　因为版权问题，素以保护版权闻名的电子图书专业网站博库已正式状告TOM.COM网站。据6日从博库公司获得的消息，北京市第二中级人民法院已经正式受理此案。　

　　博库公司称，TOM.com的子公司www.cn.tom.com擅自在其上登载了博库公司独家签约的电子版权作品《我们干点什么吧》和《长袖善舞》（周洁茹著）。根据著作权法，博库公司认为被告未经允许登载上述两个作品，侵犯了自己的著作专有使用权，因此将TOM.com与其子公司www.cn.tom.com告上法庭。
