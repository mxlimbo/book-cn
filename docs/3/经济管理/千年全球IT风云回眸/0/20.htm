润讯再败：美国在线全面入侵

　　被称为“金智塔第二”的深圳润讯公司却是在上演“非暴力合作运动”。在笔者的上篇文章《中国必败—如果CNN起诉的话…》里提到AOL又将拥有www.myicq.com和www.myicq.net的润讯公司告上WIPO，当时的案子进行状态是延迟再审。而如今，这件案子已经落下帷幕，但是WIPO的官方还迟迟未曾公布仲裁结果书。

　　2000年8月底：

　　穷凶极恶的美国在线又向WIPO递交了域名争议起诉书，要求收回深圳润迅集团所注册的两个域名：myicq.com和myicq.net。而作为被告的深圳润迅公司除了极少数人知情外，其它的人几乎是一无所知。这已经反映出他们对该事件的重视程度。在记者电话联系到此集团旗下之一的互联网公司的一位副总经理后，这位副总称自己是在网上看到这一消息才知道这件事的。

　　根据《统一域名争议解决办法程序规则》：

　　第4.（c）条规定："争议解决机构依第2（a）条完成其向被投诉人送达投诉文件责任之日即为域名争议行政解决程序开始之日。"第5、(a)条规定："被投诉人应在行政程序开始之日起20日内向争议解决机构提交答辩文件。"根据这两个条款，"域名争议解决程序"开始之日应该是WIPO将投诉书送达润讯公司之日。从这一天开始，润讯公司应该是有20天的提交答辩状的时间。而根据现有资料来看，显然润讯公司没有做出应有的任何反应，也许根本没有想过去应诉。

　　此后，润迅公司就域名事件不久前做出了说明，在说明中，深圳润迅公司指出：1999年润迅代理ICP业务，为此注册了myicq．com和myicq．net两个域名。2000年5月，润迅收到AOL的律师信，要求无偿转让上述两个域名。因为该域名为润迅合法注册，因此润迅拒绝了AOL的要求。其后润迅的台湾合作方要求终止合作，不再开展ICP业务，双方签订终止代理协议，润迅也终止了有关业务。

　　2000年9月13日，润迅收到WIPO的信函，通知润迅，投诉方的律师要求中止关于域名侵权的仲裁程序；并进一步说明，根据WIPO的规则，如果投诉方在一个月内不再提出恢复仲裁，此仲裁程序将自动终止。

　　在收到该通知后，润迅立即向WIPO仲裁调解中心负责此案的人员发出电子邮件，请求对方说明应如何应诉及提供相关材料。但时至今日，润迅方面并没有收到任何回复。

　　但是据笔者向国外消息灵通人士打听所得的结果是，AOL并没有放弃争夺域名的打算。该案仍在进一步的进行中。况且，"润迅立即向WIPO仲裁调解中心负责此案的人员发出电子邮件，请求对方说明应如何应诉及提供相关材料"这种情形简直令人难受之极。只要登陆上WIPO的网站，就可以找到相应的所有有关应诉的资料。可悲，可叹！这使得我们对润讯所公布的所谓"情况"表示怀疑。笔者甚至怀疑它是否在应有的应诉期限内回应过WIPO？

　　润讯危矣！

　　"哀其不幸，怒其不争"。
