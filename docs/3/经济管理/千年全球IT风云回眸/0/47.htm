圣诞节病毒早早登场

　　一种被称为“Navidad”（西班牙语“圣诞节”）的电子邮件病毒现已“登场亮相”。据电脑专家称，在“财富500强”企业中，至少有10家已受到过感染。幸好这种病毒的性情还算“温和”，尽管会给人带来烦恼，对电脑系统却不具有“杀伤力”。　　

　　该病毒专门感染微软的Outlook电子邮件程序。如果用户向一台受感染的电脑发送邮件，便会收到一封以该病毒为附件的回复邮件，其文件名为“Navidad.exe”。病毒文件运行后，会在桌面上留下一个象一只蓝眼睛的图标，并出现一条西班牙语信息：切勿按此钮。如果打开该图标，将出现另一条信息：FelizNavidad，非常不幸，你没有经受住诱惑，将失去你的电脑。不过，用户可手工关闭病毒程序。

　　电脑安全公司McAfee.com的反病毒研究员AprilGoostree称，尽管这种病毒的危害性不算严重，但如果系统内运行的程序占据了较多的资源，该病毒会使邮件程序“死掉”，它还会向系统内所有的邮件地址发送回复邮件。　　

　　Goostree说，McAfee.com的加利福尼亚分公司在过去的四天内已发现100多个该病毒的样本，受感染者中至少包括10家“财富500强”企业。大部分邮件都是用西班牙语写的，因此Goostree推测这种病毒可能来源于南美洲。McAfee.com拒绝透露哪家公司受到了感染，不过据“联合新闻”报道，其中有ExxonMobil和Intel。

　　数家反病毒公司已针对该病毒升级了自己的软件。McAfee.com建议用户在电脑中安装反病毒程序，而且每月至少升级一次，此外，在打开可疑的电子邮件附件之前，最好使用杀毒软件预先进行检查。　

　　过去也出现过若干种以节日为主题的病毒。去年出现的一种病毒会在圣诞节那天对硬盘进行格式化，该病毒传播非常迅速，当专家们引起警觉时，它已经登陆了三个大洲。在节日来临之际，Navidad无疑给人们敲响了一记警钟。
