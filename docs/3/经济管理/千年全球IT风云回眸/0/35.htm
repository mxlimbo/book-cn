160万美元挥金如土 终至狼狈倒闭

　　一家新创公司耗资160万美元在超级杯橄榄球大赛大打广告，但因钞票烧完，最后只好狼狈倒闭。

　　Epidemic Marketing公司表示，本周将宣布倒闭，旗下60名员工也得自寻出路。这家公司就像最近的一般网络公司一样，用尽首次募集到的760万美元资金后，最后只有关门大吉一条路。

　　其他一些最近关门的公司，包括网上运动装备零售商Boo.com、网络玩具零售商Toysmart和ToyTime、流行服饰网络公司Violet.com、工手艺网站CraftShop.com，以及儿童用品网上零售商Red Rocket。他们全都是因为财务发生问题而倒闭。

　　Epidemic公司会计经理James Wallen表示，该公司上周与一家加州公司商讨并购事宜，但目前还积欠员工二个礼拜的薪水。

　　Wallen说：“当（加州）公司取消收购计划时，我们除了关闭外，别无他法。每个人都很失望，但没有人感到惊讶。”位于丹佛的Epidemic公司成立于去年9月，他们利用外发电子邮件，告诉客户上他们网站就有钱可以拿。

　　该公司的营销策略是像病毒般的刺激营销市场。他们在超级杯打的广告内容是，一名男子进入洗手间，就会得到一笔赏金。这支广告是暗示Epidemic会付钱给登门访客，只要他做一些每天都会做的事。

　　Wallen表示，这支广告并未吸引一般客户的高度注意，但却帮他们招揽到重要的事业伙伴。事实上，许多公司也开始怀疑，广告是否能真为他们增加营收。
