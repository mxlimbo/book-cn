数字解说2000电子商务年(三)

　　“鼠标”与“水泥”的结合

　　2000年电子商务的发展由于大批传统企业的参与而更为务实。“鼠标”与“水泥”的结合，使传统企业的资源得到重新整合，加快了企业电子商务化的进程。从企业物流体系、网络基础建设、企业内部网及外部网建设的企业电子商务应用系统建设等方面的情况来看，不同行业具备不同的电子商务发展潜力。

　　■网络基础建设

　　加速电子商务

　　物流体系是企业发展电子商务市场的基础之一。在企业对企业电子商务（B2B）

　　中，高度自动化的物流系统是必不可少的；在企业对消费者电子商务（B2C）中，高效、高覆盖率的配送网络将使企业拥有营销优势。据“2000年中国六个行业企业电子商务市场研究报告”显示：在所调查的电子及通信行业、家电行业、汽车行业、医药行业、日用消费品行业和工业设备行业中，由于传统营销渠道和产品的差异，六个行业的企业在物流配送资源上有较大差异。其中，家电企业拥有物流中心和配送网络的比例最高，电子及通信行业企业的这一资源比例最低。日用消费品行业由于中小企业较多，拥有物流中心资源的企业比例也不高，但对于配送网络的建设却较为重视。

　　从六个行业网络基础设施建设情况来看，电子及通信行业、汽车行业、家电行业企业整体水平较高，近两年企业在此项建设方面的平均投入较大。从六个行业企业关键业务上网情况来看，家电、医药、日用消费品行业企业在外部业务环节（如采购、销售环节）的网络化建设程度上要高于其内部管理环节建设（如人事管理、财务管理等），表现出更为活跃的市场营销机制，也更适应电子商务时代“客户驱动”型经营模式的发展趋势。

　　由于工业企业通常通过供应链实现产品从原材料到成品销售的价值过程，因此，供应链中企业间的商务关系十分密切，供应链上信息的共享程度和企业供应链管理水平直接影响到企业的生产经营效益和整个供应链的经济效益。在网络经济条件下，供应链管理将使企业通过电子商务技术完成从订货到销售的过程，将商品供应和商品需求有机地联系起来，在准确的时间和地点、以恰当的价格，把商品从生产商手中转移到零售商手中。EDI方式作为发展时间较长、应用较为广泛的企业对企业在线商务模式，目前仍占据企业间电子贸易额的大部分，而且还将在一段时期内继续占主导地位。但互联网EDI技术的出现，使这种原先主要应用于行业大型企业的电子商务模式也能够给中小企业带来效益。从六个行业不同规模的企业情况来看，各行业都有一部分中小企业建立了供应链管理系统，并通过互联网上开放的数据交换方式，与同行业大型企业甚至不同行业的企业间实现信息共享和协同商务。比如：汽车行业、工业设备行业的供应链间协同商务表现十分活跃，中小企业的参与程度较高，企业间电子商务发展将更为迅速。　　

　　■电子商务

　　为中小企业提供新市场

　　尽管大部分中小企业在资金、专业人才、基础设施等方面缺乏独立构建电子商务应用系统的充分条件，但随着电子商务应用服务提供商（ASP）、互联网数据中心（IDC）等专业化电子商务服务形式的出现，“外包服务”已成为解决中小企业上网和应用电子商务的捷径。
　　
　　对于营销资源有限的中小企业来说，互联网的出现——特别是电子商务的兴起，为他们提供了新的市场机遇、低成本的营销和宣传渠道。对六个行业中不同规模企业的调查比较发现，大部分行业的中小企业比大型企业更注重网上销售业务的开发，以获得对企业营销活动的直接支持。此外，中小企业在与合作伙伴进行网上合作等方面比大型企业更积极活跃。下图是电子及通信行业不同规模企业网上内容与服务提供的情况，从中可以看出，中小企业除了利用网上渠道进行企业及产品的宣传以外，在网上开发销售业务的企业比例超过了大型企业。

　　2000年电子商务在中国仍处于发展的初期阶段。调查中一个普遍的现象是：尽管各行业建立企业内部网和有独立域名的企业外部网的企业比例都超过了50％，但真正实现网上销售收入或从事网上采购业务的企业比例却仍比较低。六个行业中除电子及通信行业已开展网上销售业务的企业比例占这一行业被调查企业总数的35％以外，其他五个行业中已开展网上销售业务的企业比例都低于20％。从企业过去一年中实现的网上销售收入情况来看，目前企业对企业电子商务（B2B）中大型企业占据主导地位。但仅以企业网上销售收入的绝对数字，并不能准确反映出不同规模企业参与电子商务的程度和需求发展潜力。比如，在家电、医药、日用消费品、工业设备行业中，中小企业网上销售收入在企业总销售收入中所占的比重都超过了同行业的大型企业。在这些行业中，电子商务对中小企业经营所产生的实际效益要大于大型企业。
　　
　　从各行业企业网上生产性采购额占企业总采购额的比重来看，除电子及通信行业以外，其他五个行业的此项数据都以中小企业为高。中小企业将是电子商务中最为活跃的力量，也是不可忽视的重要角色。

　　■企业选择网络渠道

　　带有区域性

　　企业选择网络渠道主要用于宣传和购销活动。从调查的结果看，六个行业企业开展网上销售业务的渠道目前仍集中在企业自己的网站上。由于目前大型电子商务平台仍不完善，行业性网站大部分仍以信息服务为主，加上网络及网上支付的安全问题等原因，使得企业在开展网上销售业务活动中对公共电子商务平台的选择仍十分有限。但在选择公共电子商务渠道中，大部分行业的企业都更倾向于垂直门户网站和专业电子商务平台。　　

　　调查中还发现，不同行业企业在选择宣传和发布信息的网络渠道时有明显的行业倾向性，对于提供信息和服务的各类行业网站都给予了普遍关注。大型门户站点、知名的电子商务网站、大型贸易性服务网站以及包括人才服务在内的专业服务性站点仍是大部分行业企业选择最多的宣传和发布渠道。而不同地区的企业在选择宣传和发布的网络渠道时，也带有明显的区域特点，即对于本地区的网站也较为重视。

　　来自一些权威研究机构对网站受众的调研资料也表明：除了一些知名度高的网站以外，国内不同地区的网民经常使用的都是本地的网站。这也是中国的互联网和电子商务初期发展阶段的一个特点。但区域对企业的影响要比对个人用户小得多，从互联网经济的特点和未来发展趋势来看，摆脱区域性市场和区域性经营意识的限制，也是传统企业向电子商务企业转型过程中重要的一步。
