2000年中国网络十大案件(一)

　　随着网络普及与发展，现实生活的形形色色案件也在因特网上有所体现，即将过去的2000年我国与网络相关的各类案件时有发生，其中以下十大案件可能最具有代表性。

　　一、上海个人主页案“网络色情犯罪案”

　　3月，上海警方破获首宗利用国际互联网传播色情图片的“网络色情犯罪案”，据悉当事人所创建的一个个人主页，数月中浏览人次高达百万，扩散量惊人。

　　公安部协查通知上海徐汇警方，称上海某网站上一个免费个人主页含有色情图片，警方随即展开调查。发现该网站隶属上海某电信综合开发公司的一个子公司，这个免费个人主页上有100多兆信息，其中包括33幅色情图片及与境外黄色淫秽网站的超级链接。

　　公安部门最终在数以万计的指令中找到了色情主页的制作者，并从LOG文件的IP地址中发现该址为陕西西安某科技有限公司所使用。该公司一名河南籍职员尚某见到上海来的警方人员后投案自首，承认自己是想利用浏览次数多来争取广告。

　　二、网页设计员传播淫秽文章案

　　利用国际互联网复制淫秽文章并通过电子邮件广为传播的网页设计人员吴某，今年6月被佛山市检察机关批准逮捕。

　　吴某是广东省三水市人，今年27岁，毕业于武汉某重点大学，在佛山市一家电脑城从事网页设计与维护工作。今年5月，吴某以假名在广州新飞网站登记了一个名为“成人文学库”的个人主页，然后从国际互联网上复制了近400篇淫秽文章，公开给“网友”阅读，同时散发了近4000封电子邮件广为宣传，吸引他人登录，仅一个月左右，就有5000多人次访问了他的个人主页，造成了极坏的影响。

　　5月31日，民警在佛山市某电脑城一间电脑店铺里将吴某抓获，当场缴获了储存有大量淫秽信息和文章的电脑设备。吴某对自己的违法犯罪事实供认不讳。

　　三、中国网上“蒙面人”诈骗案

　　6月1日，雅宝竞价交易网客户中心接到山东泰安一位王姓的用户反映：其通过雅宝网汇款1650元购买了一部诺基亚8810手机，但是款汇出后就与卖主再也联系不上，对方寻呼不回电、手机不开机，发E－mail也如石沉大海。

　　6月10日前后，接到雅宝公司建议的山东泰安王振宇、内蒙包头肖力等5位受骗者先后向张家口市警方报案。6月12日，雅宝网收到受害者反馈信息：张家口市警方已查到欺诈者的真实身份，网上“蒙面人”的真实姓名叫刘建，男，网上注册的姓名刘福全是其父亲。

　　刘建的网上资料说明，其行为是一种事先预谋好并经过精心准备的诈骗行为，其在网上提供的交易信息多为“特别好用的××手机，手机去年3月购买或仅用过20天，因想换机子所以贱卖……我的发货方式是特快专递，你的钱一汇到我就发货，如果你看到机子质量有问题见货后3天给我发回来，我退款。”等等。市场上卖到三四千元的V998等手机，其开价只有2000元左右。据悉，刘建诈骗信息的访问者一般都高达一百多人。

　　四、所谓“上海智狐机械设备有限公司”网络虚假广告

　　6月，上海市工商管理部门已查获一起网络虚假广告案，据了解，目前此案在上海尚属首例。

　　今年4月的一天，上海市奉贤县工商分局接到群众投诉，称互联网上的上海智狐机械设备有限公司发布虚假广告。工商执法人员立即上网找到该公司网址，对其内容进行下载，采集证据。经工商局一再查找，证明该公司是子虚乌有。原来，上海市××招商办的两位工作人员张某和吴某于今年１月份在网上申请了一个顶级域名并自制网页，杜撰了一个所谓的“上海智狐机械设备有限公司”，他们在网页上发布虚假广告，自称该公司是“本市最大的食品机械生产企业，是上海市重合同守信用单位，产品多次获得市优、部优、国家级新产品、国家级星火产品、获上海市星火产品一等奖等奖项”。这些虚假的内容吸引了全国40多家企业，点击率高达560多次。据张某和吴某交待，他们之所以这样做，其实只是为了把客户介绍给当地的厂家，从中获取业务中介费。

　　五、福建网络诈骗团伙案

　　7月福建泉州警方摧毁一个网络诈骗团伙。这个诈骗团伙利用互联网向全国各地的客户发送电子邮件，声称某公司经销低价的货物，并开出优越的成交条件，将客户骗到一地后，犯罪嫌疑人“热情”接待，使用掉包手法进行诈骗。

　　据报道，7月3日，上海客商林某根据互联网上的信息，携带15万元到晋江市欲购买电脑配件，所带的钱全部被犯罪分子骗走。7月7日，山东青岛客商谭某从互联网上获得信息后，到晋江购买塑料造粒时被骗走15万元。晋江市警方接到这些客户报案后，立即进行布控侦查，抓获4名犯罪嫌疑人，警方追回赃款46万元人民币。

　　据4名犯罪嫌疑人交代，今年3月份以来他们伙同另外3人，策划利用互联网进行诈骗。该诈骗团伙严格分工，有3个人常驻广州、厦门，利用国际互联网向全国各地的客户发送电子邮件进行诈骗。
