2000年IT业颁奖典礼

　　编者按：在本年度里，IT业内的发展可说是风起云涌，好消息、坏消息更是一个接着一个，现在我们就不妨一起来回顾今年的精彩新闻片段，看看它们都能在读者心目中获得什么样的“奖项”。

事　件　篇

　　贺岁片奖

　　8848是一个贺岁片专业户，去年年末,8848空降“海龟",谭智浮出水面,王峻涛淡出权力中心,而今年又是一本大戏上演:“8848分家",分出的8848做B2C及相关业务，由老榕领路；老公司则主做B2B，由谭智掌鞭。有趣的是,每次8848闹新闻,事先都已经有传言闹得沸沸扬扬了,虽然套路总是相同,可新闻效果实在不错,所以我们授予8848专业贺岁片奖。

　　柳暗花明奖

　　年初的时候MP3.com霉运高照，被全球五大唱片商告上法庭，诉其侵权及盗版，一时间人们纷纷开始讨论MP3.com究竟还有多少时间好活了。但是，“山穷水尽疑无路，柳暗花明又一村”，MP3.com居然能奇迹般地和五大唱片商全部达成和解协议，更有意思的是唱片公司虽然在法庭上把MP3.com当作唱片业的天敌一般，但私底下却十分看好MP3.com，和解时就提出了获得MP3.com股权的要求。看样子，MP3.com日子会越过越好了，而且它的经营模式也很可能成为一种新的音乐唱片销售模式。

　　最赚钱职业

　　年末,中国互联网信息中心CNNIC推出了中文通用域名(中国/.公司/.网络)注册,NSI正式推出.com/.net/.org的全球通用中文域名注册,虽然他们都宣称自己的域名系统都非常先进,然而,据ICCAN的报告,目前多语言域名系统还很不完善,并建议暂缓推出多语言域名注册。可是,对于域名注册商们来说,赚钱才是最重要的,至于多域名注册是否早产了,多标准是否会扰乱市场,他们就没空想这么多了。

　　现代聊斋奖

　　瞧这网络，狐狸一上市就成了精，也开始吃人了。9月14日，搜狐宣布以440万股搜狐股票并购ChinaRen，按照当时的股票价格，市值相当于3000万美元。由于两家公司都在CNNIC排名前10位，这次并购被人们看作是一个影响中国门户网站格局的大动作。但是，狐狸毕竟是狐狸，要吃人还得看自己撑不撑得下，到现在为止，Sohu还没真正把ChinaRen吞下去。

　　学雷锋奖

　　“e国1小时”这个口号一在北京推出，就引发了一系列的争论。有人称e国是自残式的电子商务，有人替e国算计卖一件东西赔多少钱，还有人半夜订货买一罐可乐来考验它。可是e国却依然坚持实践着他们所理解的电子商务:北京三环以内保证1小时送到，建配送中心、雇配送小工、开e国门户店，等等。至于是赚是亏，个中滋味，恐怕就只有e国自己知道了，但是，这种全心全意为客户的思想，毫无疑问是可以获得我们所颁发的学雷锋奖的：)

　　最佳广告策划奖

　　为了吸引广大网民的注意力，网站们想尽了办法，最多使用的招数自然就是做广告了。于是，八仙过海，各显神通，有横刀立马舍我其谁的大气之作—“网聚人的力量”，有充满暗示的—“上我一次，终身难忘”，也有让人莫名其妙的模仿之作—“你XX了吗？”，不过，最具轰动效应的还应该算是e龙的那句广告词“你(生)活腻(味)了吗？”令不少人看了以后觉得很别扭，一别扭就记住了e龙。由于这则广告词的争议更是为e龙做了无数的免费宣传，从而非常“成功”地达到甚至超过了预期的目的，所以，最佳广告策划奖e龙当仁不让。

　　惹火烧身奖

　　每年两度的CNNIC评选结果都会吸引国内大小网站关注和众多网民的眼球。因为每届的CNNIC的评选结果都是网民认定网站价值（至少是使用价值）的标准，于是，就像一年一度甲A的升降级一样，为了一个唯一的结果，在各种目的的逼迫下，“假球”、“黑哨”也就不可避免地出现了……

　　今年的CNNIC评选过程，不仅令业内人士满地找眼镜碎片，更是令众多网民捧腹，先是有嫌疑说某大网站在上海高校拉选票；继而又出现新浪等大网站拒绝认证这个结果，并声称自己网站的价值不是由CNNIC评选所决定的……到了最后，搞得CNNIC的官员也是尴尬不已，看，惹火烧身了不是……

　　大跌眼镜奖

　　在NASDAQ股市大幅跳水、IT业界人心惶惶，许多网站都开始出现经营困难之时，有人非常“及时”地推出了网站拍卖会，新闻媒体对此大肆炒作，有的还声称此次拍卖象征着中国网络界“诸侯争霸”时代进入了“统一六国”的时代。然而9月29日，在北京中国对外友协友谊馆里，中国首次网站拍卖会从开槌到结束却只持续了1个多小时，拍卖会当场只成交了一家网站和一个域名，令那些看好此次拍卖会的人士大跌眼镜。

　　败家子奖

　　5月中旬,欧洲最著名的电子商务网站之一boo.com宣布倒闭。在此之前公司的融资高达1亿美元，这家公司在最风光的时候市值曾高达4.5亿美元，但如今却只能以区区30多万美元的价格出售给美国的一家时装网站fashionmall.com。由于毫无节制的花费和商业模式的过于超前，Boo.com的投资商们面对这么一个败家子，当然是宁可它倒闭，也不愿继续那种肉包子打狗式的投资了。

　　最佳跳水池奖

　　在悉尼奥运会上，世界游泳好手们屡破世界记录，原因据说是游泳池设计先进的缘故。而今年，当中国的三大门户网站新浪、网易、搜狐纷纷登陆NASDAQ时，NASDAQ股市却屡创新低，11月底跌破了2600点，跌至15个月以来的最低点。市场上的股票狂跌不止，一个跌得比一个惨，导致现在那些手握NASDAQ股票的高级白领们见面打招呼的开场白都从“又赚了多少？”变成了“又跌了多少？”

　　因此，鉴于在NASDAQ上市的股票都纷纷大幅跳水，并且跌无止境的态势，我们决定授予NASDAQ市场最佳跳水池奖的称号。

　　最佳新人奖

　　正当网络不被人们看好,都在讨论网络是否泡沫过多的时候,国内最大的IT商联想,今年却开始大举进军网络了,成立“神州数码”和“FM365”网站,还收购了著名证券网站赢时通,一副财大气粗的样子,好像一点都不怕自己可能会被呛水,真是初生牛犊不怕虎。有鉴于此,我们决定授予联想最佳新人奖,至于它是否可以做出个名堂来则有待考察了。

人　物　篇

　　最佳剪辑奖

　　授予满舟，在他写了《黑客攻击防范秘技》后，顿时被媒体炒作得沸沸扬扬，称之为“少年黑客”、“最年轻的CEO”等等，后来又被某名牌大学以特殊人才的名义特招入学。可后来人们发现这位“少年天才”名不副实，《黑客攻击防范秘技》里有大量内容是抄袭来的，并且没有得到原作者的同意，当然此时他已是名利双收了。鉴于此种情况，授予他最佳剪辑奖是再合适不过的了。

　　最佳网络文学奖

　　“最后一个生日将来了，我要走了，人生总有那么多的节骨眼当然受不了，我实在不敢在这样的时刻稍加逗留，更不敢深入地去想一些事，因为，心，会碎成片片。”摘自陆幼青的《死亡日记》最后一篇（10月23日）。

　　陆幼青先生的脖子、前胸等部位已经布满了大大小小的肿瘤，生命已经早就开始倒计时了，陆先生开始了写日记，并以此作为自己人生的最后演出。借助网络的力量，他的《死亡日记》感动了大量的读者，虽然他的《死亡日记》并不仅仅是网络文学，但由于最初是在网络上发表的，所以我们把年度的最佳网络文学奖授予陆幼青先生。

　　大彻大悟奖

　　微软的比尔·盖茨，在做了那么多年的微软掌舵人后，在今年微软正处于官司缠身的窘境时，比尔·盖茨却突然宣布自己决定让位，去改做首席技术官了，此招乍一看还以为比尔·盖茨已经丧失信心了。其实不然，他这招是暗渡陈仓的妙计，辞职以后，他就开始面壁思考微软的下一代产品了，经过数月的思索，比尔·盖茨终于大彻大悟，确定了微软的终极目标：微软·Net，此招一出，不但可以使微软从官司中解脱出来，还可以继续保持新世纪的软件霸主地位，由此可见比尔·盖茨的厉害，鉴于比尔·盖茨是经过面壁思考大彻大悟想出来的微软.net，所以我们授予他大彻大悟奖。

　　最受尊崇“大虾”奖

　　这个奖项也和“五大高手西湖论剑”有关，但这个奖既不是给丁磊的也不是给马云的，是给金庸先生的。除了张朝阳没有看过金庸先生的武侠小说外，其他几位都是金庸先生的崇拜者，阿里巴巴的马云更是了得，他招聘员工居然还要问应聘者是否看过金庸先生的武侠小说，因为他认为从金庸先生的武侠小说里可以学到很多为人之道和经营之道。既然中国网络精英们对金庸先生如此崇拜，那么，把最受尊崇“大虾”奖授予金庸先生应该是没有异议的。
