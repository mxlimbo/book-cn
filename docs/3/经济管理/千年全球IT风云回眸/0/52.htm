2000年IT业最吸引人的10大事件(二)

　　6.电信公司重组惊心动魄。对一些电信公司来说，在投标欧洲的UMTS的巨额费用就像石沉大海，没有引起投资者及债券商的任何兴趣。上月英国电信集团的第二季度的收益比上年少了一半，不过这主要是购买许可证花去了一大笔钱。可是就在同一天，它宣布重组。随后AT&T集团和Worldcom集团也作出了类似的决定，进行大范围地重组，以便挽救日益下跌的股价。而Worldcom的总裁兼CEO Bernard Ebbers则度日如年，在今年六月，他只好放弃对Sprint集团的收购。

　　7.美国在线和时代华纳的新旧媒体联姻。在今年很早的时候，可追溯到今年一月，大型公司之间的合并就成了一种时尚。新的互联网公司气吞山河，要一口吃掉像时代华纳这样的传统大公司。今年一月，美国在线准备通过换股的方式，实现3500亿美元的合并交易。合并后，美国在线的股东将拥有合并后的公司的55%的股权。欧盟批准了这个合并，上周两家公司向美国联邦贸易委员会呈交了最后的让步，本周四已获批准。两家公司的总裁急切希望能在今年底完成合并。但自从他们宣布合并后，在整个新旧媒体中都发生了翻天覆地的变化。

　　8.L&H从欧洲的IT天使变质为不良少年。多年来，比利时的L&H一直以其领先的语音识别技术产品而成为欧洲IT业的骄傲。也是欧洲技术变革的楷模。今年三月，它雄心勃勃地以5.92亿美元收购美国的对手－－Dragon（龙）系统公司。这也是L&H的恶梦的开始。今年八月，《华尔街时报》的报道称，L&H的韩国公司的财务有违规行为。一石激起千层浪，9月份，美国证券交易委员会就对其财务制度进行调查。而11月份，L&H公司的合伙缔造者Jo Lernout和Pol Hauspie从联合主席及管理位置上退了下来。而在11月底，L&H公司的主席及CEO就要求破产保护，并称未发现违规的财务报告或有问题的财务交易，并认为这是保护债权人及自身利益的唯一希望。为上，公司在比利时的资产遭到法庭的冻结。
　　
　　9.微软在反垄断案中频繁失利。当任何人都认为微软案不可能很快了结的时候（也许在我们有生之年都不能看到结果），但美国司法部却在美国地区法庭的第一回合里重重地打击了微软。今年四月，杰克逊法官首次作出决定，拿出事实裁决书，并认定微软通过反竞争的方式来保持垄断，并试图垄断整个浏览器市场。微软的这种行径违反了《谢尔曼反垄断法案》的规定。今年六月，杰克逊更使出毒招，判决把微软公司一分为二。完全限制微软的反竞争的行为。微软立即上诉，现在双方已经进入了第二回合的较量。
　　
　　10.拒绝服务（DOS）的攻击更是家喻户晓。今年二月，黑客们使用了并非一流的，但很具攻击性的DoS软件发动攻击。在黑客们的不停攻击下，大牌的互联网公司，如亚马逊、EBay、CNN和Yahoo及其它公司纷纷瘫痪在地。黑客们不是以进入网站的系统为主要攻击目标，而是向网站发送成千上万的信息，阻塞互联网的通道，最终导致无法访问这些网站。不过黑客发动攻击的电脑大多是被他们植入了特洛伊木马及zombie（蛇神）软件，在用户不知情的情况下发动的攻击。发动DoS攻击主要是那些整天都通过宽带接入互联网的电脑。一加拿大16岁的少年在对CNN发动DoS攻击时被捕，不过他只是一个小虾，真正的元凶可能永远都没法捉到。但DoS的攻击的威力仍然让人们担忧，任何人都可使用它，并可能导致在线商务停滞不前。
