　　最近比较烦　　　　　　　　

　　领衔主演：新浪、搜狐、e龙、今夜网

　　友情客串：其他在龙年里惹上官司的网站们

　　几乎在同一个时间，国内两家最大的门户网站新浪、搜狐均被诉侵权，不过这两件网络官司都是发生在新浪、搜狐上市时间，炒作性质比较大，最后都有头无尾了。e龙发表了一个声明称，其上海站点的内容被搜狐原封不动地照搬到其上海都市频道，表示要诉诸于法律，其后就没有下文了。9月18日在北京市第二法院开庭的今夜网告新浪侵权一案，当场新浪辩护律师反戈一击称今夜网侵权，这件官司到今都没有判决结果，可能是私下和解了吧。

　　一场雾里看花的故事

　　领衔主演：中国电信及国家有关部门

　　友情客串：全国热爱上网的虫子们

　　9月18日，一场涉及大范围电信资费价格调整的价格听证会在北京如期举行。

　　在这之中，手机单向收费的问题得到了空前的关注，然而这方面的实施却并不顺利。直到11月21日，在有关各部委的组织下，手机实行“移动间单向收费”方案终于议定，并已经上报国家有关部门审批。

　　移动单项收费方案的主要内容包括移动电话之间通话由主叫方付费，资费标准维持现行的每分钟0.4元，漫游收费将区分省内、省际，省内漫游是每分钟0.5元，跨省漫游是每分钟0.6元，移动电话和固定电话之间通话仍实行双向收费，固定电话付费的资费标准仍执行现行标准，而移动电话拨打固定电话移动电话方付费的资费标准将由原来的每分钟0.40元降为每分钟0.30元。

　　11月23日，关于中国政府拟实行手机“移动间单向收费”的报道，引发在香港上市的中国移动(0941)和中国联通(0762)股价连日大挫。仅仅是未实施的单向收费“传闻”就引发了金融市场上的地震，可见该问题影响之大。但近日国家有关人士表示，近年内不会实行手机单向收费。

　　而另一方面，对于其他电信资费进行大范围下调的信息已经明确发出。11月19日，信息产业部政策法规司司长刘彩表示，“国际电话费、互联网接入及出租线路等多项电信资费将成倍下调。”

　　电信资费成倍下调的方案已上报国务院，估计批准通过的时间不会太长。成倍下调的项目包括国际电话收费、互联网接入、出租线路等多项，比如互联网接入要由每分钟7分降到2分。

　　不过，他同时也明确指出：“市话费是不会降的。”至此，关于电信资费的种种算是已经水落石出。

　　企鹅传奇

　　领衔主演：腾讯公司和那只可爱的小企鹅

　　友情客串：TOM.COM以及泡在QQ上的少男少女

　　2000年夏天的两件事却让腾讯成了大家关注的焦点：一是市场传闻香港著名网企TOM.com欲高价收购OICQ。二是腾讯在新版本的OICQ上放置广告条，由此在用户群中引发了不小的争议。

　　不知不觉间，OICQ已经发展到了后来者必须仰视的规模。其三千万的名义用户数（账号数量）和数十万的同时在线人数足以让ICQ这样的国际化产品在国内黯然失色，其国内即时通讯市场“王者”的地位无人可以动摇。俗话说树大招风，出了名的腾讯也招来了各种各样的议论。例如，在广告条推出时，就有舆论对其“急功近利”的做法提出“质疑”。不过腾讯对此的态度基本上是“你说你的，我做我的”，一副爱理不理的态度。原因是那个小小的广告条一年可以为腾讯带来数千万元的收入——在巨额利润的诱惑下，一点小小的批评就显得微不足道了。

　　年底，OICQ的新一代产品OICQ2000的第一个测试版总算是推出了。在推出几个测试版本之后，正式版很快就登上了各大软件下载网站的最热门软件排行榜。与以往的版本想比，这个新版不但增加了对Skin的支持，加入了隐身的功能，还内置了个人信息工具“资讯通”。和一个以IE为内核的浏览器Tencent Explorer。从中不难看出，腾讯公司不甘于仅仅在即时消息领域称王，还有把触角伸向其他诸多领域的雄心。

　　当然，OICQ的发展也引发了一些其他方面的问题。例如许多青少年热衷于OICQ上的网恋之类，影响了学习和生活。不过这些问题腾讯本身基本上是“无能为力”的，倒是不久前的另一件事值得一提，就是关于OICQ消息群发软件OICQAD的种种议论。当初腾讯没有在OICQ中加入群发功能就是为了防止有人滥用，许多对OICQ核心技术比较熟悉的程序员也有意控制该功能的使用与传播。然而最后还是有人开发了这样一个令广大OICQ用户头疼不已的玩艺。据传，腾讯很可能在年底之前对其采取某种行动，不排除起诉作者的可能性。

　　秀与黑

　　领衔主演：海信、黑客黑妹、当当网上书店

　　友情客串：众多作秀运动的积极参与者与注视者

　　黑客是让每一个网站经营者头疼的名词，今年二月份黑客大规模侵袭雅虎、美国在线等大型网站。据估计造成的间接损失近二十亿美元，有分析家称黑客将成为新经济的杀手。但今年国内有一个很有趣的现象，那就是争着把黑客拉到自己身上来自我炒作，于是当当书店自爆家丑称,来自8848的IP黑掉了它的网站，海信做防火墙软件，向全球黑客叫板。没想到有位“黑妹”一把火，把海信老家给端了。

　　将网络管理进行到底

　　领衔主演：信息产业部、国务院新闻办

　　友情客串：全国大大小小的网站

　　可以说，个人主页培育中国整整一代网络人，现在多少做网络的是从个人主页开始的，如Ｃａｒｂｏｙ，高春晖等人。但国务院在９月底通过的《互联网内容服务管理办法》将彻底终止个人主页时代，按照条例规定，互联网信息的发布必须先要经过登记注册。

　　信息产业部正试图把互联网管理走向规范化，可以肯定的是，个人主页已经走完了它在９７年的最高峰时期。

　　本年让业界盼望已久的《互联网信息服务管理办法》于９月２５日正式施行，办法明确规定，信息服务提供者不得制作、复制、发布、传播九类信息。紧接着在１１月６日，信息产业部颁布了《互联网电子公告服务管理规定》，同时国务院新闻办和信息产业部联合颁布了《互联网站从事登载新闻业务管理暂行规定》，核心思想是加强上信息的管理，促使网上新闻业务和ＢＢＳ服务走向有序的良性竞争。这些法规的颁布让人感觉到，上层对互联网的有关立法的逐步重视。

　　牵　手

　　领衔主演：搜狐、ChinaRen、雅虎、台湾奇摩、联想、赢时通

　　友情客串：高唱着单身情歌待“嫁”而沽的网站

　　所谓“大鱼吃小鱼，小鱼吃虾米”。在竞争激烈的网络业，吃掉别人或者被人家给吃掉都不算什么让人惊奇的事情。从资本回报的角度来讲，只要能获得理想的回报，网络公司宁愿被有实力的企业吃掉，也好过前途不明朗的情况下苦撑危局。

　　9月14日，搜狐公司发布消息称，该公司于当日正式与在线服务社区网站ChinaRen.com达成收购协议。这次收购行动总价值约为3000万美元，合并后的新网站在访问量上将一举超过新浪，成为中国最大的门户网站。

　　这是一起至今为止仍然争议多多的收购案。因为许多人认为被收购者在网站内容上要大大强于收购者，并且看起来这是两家不懂得如何赚钱的网站的合并，没有理由认为它们只要合在一起就会懂得赚钱之道——两个“零”相加，结果再怎么样也无法等于“一”。

　　11月9日，美国网络业巨头YAHOO!公司在台北召开新闻发布会，宣布以1.46亿美元的价格收购台湾地区最大的门户网站奇摩站。

　　业内人士认为，此一购并案将是台湾地区大型网络公司合并趋势和风潮的开端，台湾地区门户网站的市场及排名，也将因此一购并案重新洗牌。除了上述两个网络公司之间的购并之外，那些非网络公司涉足网络业所进行的购并案就更加引人注目了。

　　非常评选

　　领衔主演：CNNIC及知名IT网站

　　友情客串：新闻媒体及全国网民

　　关于网站在CNNIC十佳的评选中作弊的传闻一直都存在，在2000年中的这次评选中集中暴露出来而已。面对漏洞百出的评选，网民和媒体纷纷提出质疑，更有网站内部人士大暴作弊内幕，使得CNNIC的威信一落千丈。

　　5月29日，网易发出的电子邮件揭开了一系列闹剧的序幕。许多使用263等免费电子邮箱的用户，收到了一封名为“丁磊致网友的信”。在这封邮件中，丁磊号召网民们投网易一票。这封“非常时期”的“非常邮件”立刻激起千层浪。

　　通过互联网拉选票是最方便快捷的，但是更容易留下痕迹，让人抓到把柄。所以一些网站采用了线下拉票的方法。投票前的一段时间，网站员工的朋友、亲属，甚至朋友的朋友、亲属的亲属也都会收到一些表格，表格上要求填写姓名、身份证号码、家庭住址、电话等栏目。一些高校的学生宿舍中也出现了临时“雇佣兵”的身影……

　　6月12日，著名IT网站Chinabyte披露了一封读者的来信，信中披露了有些网站如何编写投票程序以在CNNIC评选过程中进行作弊的过程。据其透漏，在CNNIC的投票中进行舞弊，在许多著名网站那里已经是公开的秘密了。

　　6月13日，对“十佳网站投票舞弊事件”一直保持沉默的中国互联网络信息中心(CNNIC)，终于“开口说话”，表示CNNIC已“注意到在此次活动中出现了一些不正常的现象”，并且要求“有关网站立即停止一切针对此次评选的不正当行为”。

　　CNNIC的网站排名最后在尴尬中姗姗出场。而网民对网站乃至整个CNNIC评选的信任危机使CNNIC后来表示今后将暂停“十佳网站评选活动。
