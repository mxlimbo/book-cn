TOM.com3.7亿港元收购163.net

　　【ChinaByte综合消息】北京青年报消息：TOM.COM于9月4日宣布，他们已与飞华签署协议，收购飞华旗下一间持有实质操控163.net之权利的全资附属公司全部已发行股本，代价为4800万美元(约3亿7440万港元)，其中4500万美元(约3亿5100万港元)将以每股5.69港元之价格发行6168万六千股股份(相等于3亿5099万3340港元)及现金6663港元支付；及余下的300万美元(约2340万港元)则以偿还若干股东贷款的方式支付。

　　另外TOM.COM也宣布与羊城报业及shawei.com(鲨威体坛)分别签署合作协议。

　　根据协议，TOM.COM同意收购羊城报业70%股本权益，总代价为2亿3600万元，将以每股5.3元的价格发行4464万股股份(相等于2亿3659万2000元)；及现金8000元支付。

　　此外，TOM.COM收购鲨威体坛全部已发行股本及转让贷款，总代价为1亿5500万元，将以每股5.5元之价格发行2405万股股份(相等于1亿3227万5000元)，及支付现金1万3006.3元作为鲨威体坛股份之代价；及以每股5.5之价格发行412万8000股股份(相等于2270万零4000元)，及支付现金7693.7元作为雇员代价。
