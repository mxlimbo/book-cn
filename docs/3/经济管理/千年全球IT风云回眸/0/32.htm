Boo.com是这样倒闭的

　　英国互联网公司—博(Boo.com)的命运让人着实地领会到了什么叫朝生暮死，什么叫花无千日红。这家创建仅两年，实际运营才6个月的网上服装零售公司，17日因耗尽资金而宣布倒闭，成为欧洲第一家破产的互联网公司。22日，该公司由于无人问津，找不到新的买家而被毕马威公司着手清算。而在去年，美国《财富》杂志还将其评为该年度12家最酷的公司之一。

　　BOO.com敲响丧钟

　　融资盛况空前遥想博公司创立时，也是红红火火，雄姿英发。网站的主要创立者是两位瑞士名模喀秋莎·丽达（女）和欧内斯特·马姆斯坦（男），他们拥有公司40%的股份。当他们两人的玉照在伦敦的报刊大肆张贴时，引发了无数崇拜者的热情。可以说，博公司的出现正迎合了英国网络公司的时尚—明星加盟，美人主理。正由于此，公司最初的融资情况非常喜人。投资者都是国际大公司，有法国贝纳德·阿诺尔集团、贝纳通家族、美国J.P.摩根集团、高盛集团等，轻轻松松筹集到总值达1.35亿美元的资金，博公司也因此被誉为欧洲资金最雄厚的互联网公司。

　　由于Boo.com的目标定位是全球市场，雄心勃勃地打算为18个国家的顾客量体裁衣，销售服装，因此其前期的全球宣传攻势颇为猛烈，花费巨大。为符合时尚的格调，他们的办公室也在租金最贵的伦敦金融中心区而不是在一般网络公司聚集的低租金区域；高级经理去世界各地看时装展时坐头等舱、住五星级酒店。公司规模也迅速扩大，本来预计只有40名员工，但很快便在世界各大城市如伦敦、纽约、巴黎等地设立办公室，并雇佣了450名员工，超出预期10倍以上。

　　经营问题多多

　　尽管前期宣传轰轰烈烈，但在实际运作过程中，博公司却遇到了接二连三的问题。

　　复杂的网络技术、网上立体服装展示尽管显得十分高贵，但要实际操作起来却不那么容易。而且在伦敦的数据中心建立和创办起多国语言的网站，遇到的问题要比设想的多得多。因此，各种各样的测试和意想不到的问题推迟了Boo.com出台近五个月，直到去年11月份Boo.com才正式露面。

　　在网站正式开通后，经营状况也极其不如人意，许多问题已暴露出来。如网站的设计过于超前而实际可供选择的时装款式并不多等。更重要的是，该网站所出售的名牌服装价格与高档时装店的价位毫无二致，对消费者缺乏吸引力。因此到去年第四季度，公司启动资金已花掉一半，而去年12月的营业额只有6万英镑，连公司在全球各地的450名员工的薪水也无力支付。公司被迫缩减规模，裁减了大约1000个职位，但已无济于事。至今年五月，公司资金已基本告罄，尽管投资者又注资3000万美元，但仍无法避免倒闭的厄运。

　　入不敷出倒闭

　　博公司在17日宣布倒闭时发表声明说，“我们在业务有起色时不得不把公司进行清算（今年2月份的网上收入已达657000美元），所有的努力都付诸东流，我们非常失望。”创办者欧内斯特·马姆斯坦说，博公司主要是毁在花销太大。他在接受《金融时报》记者采访时说：“我们希望每一件事都做得完美，我最大的错误就是缺少一个能够控制财务的伙伴。”博公司的一名经理承认，公司的失败在于全球市场推广和广告开销过大，短短2年时间吞噬了其1.35亿美元的投资。公司的债权人大多是广告代理，这些广告代理从博公司手中至少挣了2500万美元。

　　实际上，博公司就是一家典型的只花不挣，入不敷出的互联网公司。它采用的经营方式投资大、回报小，完全有悖商业原则。

　　由于博公司的教训，世界时装和体育服装行业的名牌企业对那些有财务赤字的小互联网零售商供货时也显得小心谨慎。PentlandGroup集团在今年早些时候已经决定不再给博公司供货。在博公司倒闭前，有供货商认为，该公司的一些做法将会拖垮一些优秀企业。

　　除了经营策略的失败，博公司内部管理不善，人员稳定性差也是其致命伤。今年三月份，越来越多的顾客对博公司的配送服务及其他问题表示不满，但博公司依然我行我素，几乎没有花时间去修改商业计划或做技术上的改进，只是空口承诺而已。

　　上个月，该公司有三名经理跳槽，其中包括一名财务总监。有的员工已经好几个月领不到工资，有的一个月只领到800英镑（合1194美元）。为此，员工们非常气愤，认为遭到了剥削。公司的倒闭，将导致300人失业，这些员工的安置也都将是棘手的问题。

　　步后尘者紧跟紧随着博公司倒闭的步伐，才开张3个月的另一家英国互联网公司Netimperative.com本月22日也宣布即将关门，因为其资金已经快消耗光了。

　　跟博公司一样，Netimperative也曾雄心勃勃，在欧洲举行的全球互联网大会上曾扬言要赶超博公司。而现在它惟一赶超博公司的是倒闭的速度。据Netimperative公司的CEO披露，目前该公司的董事会已开始资产清算。

　　由于欧洲互联网公司接二连三的倒闭，欧洲互联网投资的脚步已明显放慢。Netimperative.com公司种子基金的投资者阿齐姆·爱资哈尔认为，互联网公司的倒闭也未尝不是好事，这种筛选和淘汰有助于把麦子从麦壳里剥离出来，使投资者辨别良莠，淘到真正的金子。也有人认为，对于风险投资来说失败是必然的，如果一起步就成功，风险也就无从谈起。

　　一项由Pricewaterhouse Cooper上周提供的最新研究表明，四分之一的英国互联网公司将在６个月之内花完他们的钱。博公司的倒闭预示着欧洲互联网公司的丧钟已经敲响，而全球互联网公司的倒闭潮是否会因此而引发，实难预料。
