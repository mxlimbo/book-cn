“ILOVEYOU”病毒已有五个变种

　　“ILOVEYOU”病毒出现24小时之内，网上已经出现了五个该病毒的变种，其中“Mother's Day”变种最为厉害。“Mother's Day”病毒邮件的标题是“Mother's Day Order Confirmation”，邮件内容告知收件人其信用卡已被订购的母亲节礼物扣除326.92美元，并请收件人确认附件中的发票。而一旦附件文件被运行，这个病毒除象“ILOVEYOU”一样会自动发送病毒邮件外，还会删除系统中的所有BAT和INI文件，使系统可能彻底瘫痪。
