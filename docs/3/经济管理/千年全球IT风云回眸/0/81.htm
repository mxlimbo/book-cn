2001：改变人们生活的五大技术

　　蓝牙技术

　　我们在日常生活中正在不停地增加新的设备，如移动电话、便携式电脑、可收发信息的寻呼机、MP3播放器及数码相机。但我们得用大量的线去把它们连接起来。但如果你用数码相机拍了一张照片，并把它存到你的掌上电脑，且删除一些不好的阴影，并通过你的移动电话通过电子邮件传给你的朋友。所有这一切都无需缆线连接即可完成。这种无线就为蓝牙技术。它能在10米的范围内把所有的设备均连接起来。并通过无线进行通讯。这项技术开始于1994年。装有这项技术的设备将于明年初正式面市。摩托罗拉已经向人们展示了一种能作手机和无线调制解调器的手机。估计到2002年，有这种技术的设备将占主流。

　　有浏览器的视频游戏机

　　也许人们的家中或办公室里并没有PS2，但他们仍在玩各种各样的游戏，他们所需要的仅仅是连接到网络里的电脑。这就是为什么有浏览器的游戏在男女老幼中如此流行的原因。对游戏迷而言，只要你的PS能联上互联网上，则你就可随时随地地玩它了。

　　Transmeta（全美达）公司的克鲁索处理器

　　使用全美达公司的克鲁索处理器，不仅可以延长电脑的寿命，还可降低电脑的耗电，这还可把这种芯片移置到更轻、更小的手提电脑上去。


　　数码录影机

　　对海量硬驱动器来说，数码电视记录器（DVR）是很必要的。它最终会完全取代你的VCR。


　　有Friskit搜索引擎的流式Napster

　　在Napster网站，你是否有过极想听某首歌，但却没有时间或足够的硬盘来从存贮从Napster处下载来的MP3的这样经历呢？Friskit是一种可把音乐编码为流式的搜索引擎。它能立即满足你的这种需要。
