2000年全球最佳CEO(二)


　　“股市就是如此，适者生存。”Ollila说，他一下提高了声调。“每一天你都必须有出色的表现，你应该在短期计划与长期计划之间寻找一个平衡点……这是唯一的态度，只有这样才能避免不必要的失眠。”面对激烈的市场竞争，Ollila永远都当仁不让。“我认为，在应对即将到来的挑战（即以软件应用与服务）面前，我们比其它任何公司都有优势。”他宣称。“我认为，远东的公司并没有优势可言。优势在于我们——因为我们很清楚市场需要什么。”从1992年以来，诺基亚在重大的策略和产品开发中傲视群雄，拿了15个第一。其中包括，第一台运用于全球通讯网络的数字电话，第一台专门为亚洲客户设计的移动电话（具有全息显示和亚洲语言显示的界面），第一台可由客户自行选择外壳的移动电话，第一台拥有短信息交流功能的移动电话。

　　仅九月份一个月，诺基亚便成功地将WAP商务电话投放市场、在巴黎近郊推出可支持客户及企业用户移动因特网服务的第三代集成系统、与电信巨头摩托罗拉公司和爱立信公司联手开发一种通用标准（该标准使人们在世界任何地方收取当地信息，开展电子商务）。（目前的移动定位系统曾使“定位服务”成为可能，但是，该系统不具备兼容性。）

　　每当诺基亚公司宣布一种新产品或新服务时，使用的都是“集体名词”。在整个诺基亚公司，许多人，包括Ollila本人，名必称团队。就连座落于芬兰湾畔的公司总部也是各行各业人士齐心协力的典范——内部装潢设计师Iiris Ulin的天然素材和芬兰建筑师Pekka Helin的全透明玻璃-钢筋建筑风格浑然一体，相得益彰。

　　Ollila“有能力创建一个以自己为核心的团队，一个懂得如何集思广益，群策群力，然后朝着一个既定的目标努力的团队。”曾荣膺99年最佳CEO的称号瑞士ABB公司总裁兼CEO Gran Lindahl这样评价Ollila。他还注意到了Ollila的一种与众不同的关键性策略，“随着，他就会负起责任，说‘好吧，现在我们开始着手吧！’”Ollila相信，在诺基亚公司，每个员工都享有自己的权利，每个人，即公司60166名员工都可以向他直抒胸臆。“这对于发挥个人与组织的创新能力有着积极的意义。”他说。“但是与此同时，我们也是一个非常务实的组织。”他强调说。“我们不会一个劲得钻牛角尖。事实上，我们所做的任何一件事情都有一个时限问题。每个人都应当负起责任，然后说：‘是的，就是这样。我们就应该这么做。’”他坚持说。“否则，讨论问题时，大家可能兴致勃勃——但实际上，并没有任何人搭你的茬，买你的账，没有人愿意将它贯彻始终。我们可以让每个人各抒己见。但是，有时也需要有人承担起责任，然后奋力向前冲刺。”事实上，Ollila在诺基亚的成就之一就是，许多人都可能把握机会，并发挥自己的特长，将其贯彻始终。

　　现年31岁的Johanna Tikka就是一个典型的例子。在墨西哥市的诺基亚公司办公室里，她一半的时间用于协助诺基亚公司墨西哥公司总经理Sampsa Lahtinen工作，另一半的时间则用于网页设计，往来信函处理。她是于1994年加盟诺基亚公司的。前三年，她在芬兰北部的Oulu研发中心工作。在其期间，她自学了西班牙语。1997年，她调往墨西哥为诺基亚公司当地销售与营销部工作。

　　“在这样的公司里，你完全有机会证明自身的能力。任何一个人只要想获得发展的机会，想尽快成熟起来，我们都愿意为他们提供一个发展的平台。”Ollila说。“事实上，这位小姐对公司的任何一件事情都全身心投入。”他指的是Tikka。“这就是我们的宗旨。创建一个能够吸引年轻人的平台，创建一种特殊的工作环境。成功固然是不可或缺的一个组成部分。但是，最基本的是要有一个发展的平台。”我们不仅应该了解作为CEO的Ollila，而且应该了解作为普通人的Ollila。“他对多元文化问题相当敏感——对于芬兰公司或同事应该怎么措辞，对于旧金山海湾地区的年轻人又应该怎么措辞，对于中国或日本的商业人士又应该怎么措辞，他都了如指掌。”ABB公司的Lindahl说。“他对文化特别敏感。他懂得如何深入到客户的文化中去，当然还懂得如何深入到全球各地所有员工的文化中去。”但是，事业并非Ollila生活的全部。从他的朋友Lindahl言谈中，我们不难发现，Ollila兴趣广泛，注重家庭生活。“对于从事这种职业的人士来说，恋家确实不容易。”在离赫尔辛基北部150公司处的地方，Ollila和他的家人拥有一个“避暑小屋”，在那里，他们可以忙里偷闲，享受短暂的乐趣。

　　对于像自己这样家庭与事业并重的人，Ollila是这样评价的。“我们必须在事业与家庭之间找到一个平衡点，否则你怎么能处理好人与人之间的关系，怎么能成为一个优秀的经理人呢？”
