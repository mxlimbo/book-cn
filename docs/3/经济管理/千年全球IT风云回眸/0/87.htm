新世纪通信发展十大预测(一)

　　因为通信业的发展，人们不仅仅缩短了彼此间的距离，工作和生活质量也有了极大的提高。可以说，人们从没像今天这样随时享受着通信的巨变带来的诸多便利。相信下个世纪我们会有更多的惊喜。

　　有资料显示，进入本世纪的最后一年，中国的固定通信网与移动通信网的规模分别位居世界第二位和第三位。而下个世纪我国的通信业将经历从量变到质变的飞跃，信息技术将成为最活跃、最热门的技术，信息产业也将成为最具活力的朝阳产业。

　　通信业的魅力不仅仅在于它创造了发展速度的奇迹，更在于任何一种新的猜想一旦在人脑中产生，就会迅速变为现实，而且时间的间隔越来越短。

　　在新旧世纪交替之时，11位通信行业的资深记者，大胆预测了新世纪第一年我国通信业可能发生的重大改变，希望能揭示出一些未来通信业发展的趋势，以及下世纪在人们生活中可能因通信技术的发展而带来的变迁。

　　之一　3G之争将初见分晓

　　到明年年底，近年全球通信市场最大规模、也是竞争最激烈的一场市场比拼将进入最后的决战，并有可能初见分晓，这就是3G(第三代移动通信)标准之争。3G作为宽带移动通信，以每秒2兆的传输速率，将手机变为集语音、图像、数据传输等诸多应用之大成于一体的未来通信终端。

　　目前摆在移动通信运营商面前可选择的第三代移动通信标准有三：由欧洲几大厂商支持的WCDMA，由美国支持的CDMA2000，还有一个就是有中国参与制定的TD-SCDMA。移动通信运营商最终会选择何种3G标准，将影响到未来若干年内全球移动通信发展的大方向，更会关乎未来。

　　按照各方的说法，三种标准各有特色、各有其优势。WCDMA可以通过GPRS等中间技术，将现在广泛采用的GSM向3G平滑过渡；CDMA2000能以最小的投入实现现有窄带CDMA向3G的过渡；而TD-SCDMA则可实现由GSM直接向3G的平滑过渡。

　　虽然都不错，但在单个通信运营商面前，选择却只有一个——每个移动通信运营商只会采用一种3G标准搭建其网络，并投入商业运营。

　　明年年初，日本的DoCoMo将第一个推出3G的商用试验网，它选择了WCDMA。而其它标准也将拼命寻找机会，去实现其“归宿”。远的不说，单说中国市场，明年将是三大标准逐鹿之年。虽然明年国内移动通信的运营商在数量上有可能增加，但是三个标准都被采用的机会却不大。也就是说，三个标准之中，很有可能有一个将不被看好，因而被迫失去广阔的中国市场。因为无论是经济性，还是将来的市场竞争，甚至是终端的发展，过多的制式标准都弊多利少。

　　WCDMA已抢了市场先机，CD?MA2000无疑会在其发源地美国先站住脚，TD-SCDMA如要立足，则必须先在国内被采用，这也是其在抓紧时间抢进度的主要原因。

　　无论怎样，到了明年年底，3G几大标准在中国市场的命运，将初见端倪。

　　话又说回来，3G前景虽美好，不过离着真正普及商用还得有几年。明年真正会和用户见面的，会是“二代半”，即GPRS等过渡技术。明年GPRS手机将是市场新热点。

　　之二　手机产品走向两极化

　　今年中国移动通信迅猛发展的奇迹在明年将继续上演：手机用户将可能突破1亿户。巨头们将展开拉锯战：摩托罗拉、诺基亚、爱立信、西门子市场格局可能重新洗牌；第二军团厮杀更为激烈：阿尔卡特、飞利浦、三菱、索尼以及其他日韩品牌继续混战；国产品牌继续推进：康佳、TCL、波导可能胜出。但无论如何，明年4000万部的需求量，将使每个厂商获得大发展。

　　明年手机市场产品将走向两极化：高端引领趋势，低端走向普及。

　　随着中国移动和中国联通GPRS网的开通和普及，移动数据能力极大提高，GPRS、WAP、SMS等移动业务更好用，另一方面，移动梦网等计划的深入将促使移动互联应用更丰富。主导手机厂商将推出符合移动互联应用的高端GPRS手机。

　　明年3000万的手机用户增长量，将使手机大众消费时代真正到来。手机市场下限不断下沿，经济实用、外观优秀、功能丰富的低端手机将成为市场主流，并将成为决定手机厂商市场占有率的关键。手机厂商都将推出针对这一市场的丰富新品，实用的功能和美丽的外观将是这一市场产品的共同特点。

　　之三　运营商瞄准宽带网

　　目前各大运营商的高速骨干网络已具规模，包括明年将要加入到运营市场中来的铁通和广电，带宽从2.5G到40G不等，如此充裕的带宽资源促使明年运营商的注意力一定是放在宽带多业务接入网的建设上，而传统的窄带话音接入网在新世纪里将成为古董。

　　在经济发达城市，基本上能够实现光纤到小区、大楼和路边。这将包含两种潜力巨大的市场：商业用户和个人用户。商业用户将能够享受到10M—100M的带宽和相应的一站式服务，加上与之配套的增值服务如IDC、ASP的成熟，B-B商业模式的大规模应用已是时机。数量激增的个人用户不但可以充分体会高达1.5M带宽的上网乐趣，更可以随意选择多种上网方式和服务，如ADSL、HFC、cablemodem等等。预计这部分人群将占国内上网人数的20%左右。

　　设施水平的提高将促进国内网络经济的进一步繁荣，也使得中国股将成为低落的纳斯达克救市的希望。只是，新经济可能还无暇顾及到经济欠发达地区。

　　之四　消费者真正做“上帝”

　　如果预测明年通信领域将会发生哪些变革？我想明年移动互联或许能走出“曲高和寡”的尴尬局面，随着GPRS网络的开通，人们利用无线终端上网将不再是“纸上谈兵”，况且到那时人们的概念也将会有所转变，那就是手机上网、无线终端上网不一定是移动互联的核心，而能提供随时随地的信息服务将成为移动互联的发展重点。许多想像中的服务将会成为现实。

　　还希望蓝牙技术从炒作走向成熟的应用，百米到十米内“无线束缚”，这将是多么惬意的事情，蓝牙耳机将使你开车接听电话再容易不过，不必担心警察会伸手让你靠边；蓝牙装置将使你装修时不必费心考虑家电、音响、空调等的电线如何布置而不会破坏居室整体氛围。

　　我想随着中国加入WTO成为事实，竞争的态势将让消费者真正体会“上帝”的滋味：不光是资费的下调，更重要的是服务的彻底改变。

　　之五　国产手机要翻身

　　2001年将是国产手机的关键一年。国产手机采取何种策略，才能在实力强大的“洋品牌”的压力下突破重围？需要这些国产厂商的高度智慧。

　　国产厂商们自己说，他们希望能走彩电业的老路。先以“贴牌”方式打入该产业，树立品牌，再逐渐掌握核心技术，以求全面突破。但相对于彩电业，手机的技术含量更高，市场变化更快，市场淘汰力度也更大。

　　什么是国产手机的突破口呢？应该而且只能是“价格”。尽管现在看来有些不可能：生产规模小、元器件要受制于人，当然没有条件发动“价格战”。但把利润做“薄”，迫使洋手机放弃低端市场的竞争而转向高技术含量、高利润的高端手机市场，才是国产手机的机会和出路所在。

　　包括信息产业部在内的国家有关部门，对国产手机的关爱态度也是不言而喻的。今年国家从手机的入网费中拨出14亿元，支持国产手机业务的发展。而国家连续5年从固定电话初装费中提取5%，也是支持国产手机的研究和产业化的专项经费。

　　有消息说，在今年11月举行的中央电视台2001年广告招标会上，有9家生产国产手机的厂商参加了这一广告竞标大战，明年国人又可以在电视上看到国产手机的广告大战了。
