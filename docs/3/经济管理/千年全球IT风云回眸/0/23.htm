今夜网被侵权，新浪“献身”说法

　　备受业界及媒体关注的“今夜网”诉“新浪网”侵权和不正当竞争案将于9月18日上午9：00在北京市第二中级人民法院正式开庭。这是近来网络侵权纠纷案件中的一件“大案”，必定更为引人关注。

　　近一段时间，网络侵权纠纷有愈演愈烈之势，业界普遍认为，网络著作权问题的不断产生，网络著作权纠纷的逐渐增多，无不表明现有的著作权法已经滞后于技术的发展，这种情况不但使原创者的网络著作权得不到有效的保护，同时也造成部分网站在做内容的时候如履薄冰，限制了网站的发展。互联网在中国虽然还属低级阶段，但壮大发展肯定是大势所趋。设立网络的法律法规，规范网上行为，对网上转载方式加以限制，这些都是必需的，但怎样兼顾著作权人、公众、网站三者的利益，在资源共享、保护当事人利益、满足公众需求之间，在有利于互联网的长远发展的基础上，找到一个各方都能接受的平衡点，这是最值得关注的。

　　中国知识产权局局长姜颖不久前表示，目前网上侵犯知识产权现象时有发生，世界各国都面临如何保护网上知识产权的问题，中国也将根据国情研究制订有关网上知识产权保护方面的法律。（OYO）
