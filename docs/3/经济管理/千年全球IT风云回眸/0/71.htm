2000年中国网络9个招牌

　　2000年，网站处在资本旋涡的中心，就象前几年一样。只不过那时是热闹和赶潮的中心，而今是被风吹雨打的中心。2000年是网站的苦命年。过去轰轰烈烈是为了大干快上，而今革命的首要问题是如何求生存。那些赶上热潮、拿到投资的网站很幸运，也很悲惨。因为，热潮使每一个投资者和创业者都受到感染，而今场子摆开了，局面也打出了，可是后续的发展资源却突然断了。整个互联网产业进入窒息状态，似乎大家都要靠“憋住气”比生存。毕竟，活着是硬道理。

　　新浪网

　　最青睐的网站。从58元到5元，不管资本市场如何折腾，都不能否认一个事实：新浪网是用户心目中的最佳网站。与其他毛里毛糙的新兴公司，新浪还是有一定的企业文化和团队合作。同时，也是资本运作与业务运作最体现国际化的中国门户。2000年，是新浪网从边缘媒体成为主流媒体的转折点，而且借助互联网在中国的涨势，成为中国第一媒体的时间也不会太远。但是，2001年面对国际巨头，新浪网是购并别人还是被别人购并，是个悬念。

　　中华网

　　最有钱的网站。公司做得好，不如上市上得好，尤其是上得早！这是互联网游戏的基本规则。而在中国诠释这个规则的最佳案例就是几大门户的上市命运。作为中国最早上市的门户，中华网无疑是最大的财主。在这样的市场环境下，有钱才有硬道理。中华能不能利用资金优势，构建自己真正的市场优势，除了天助之外，还要看他们自己的策略和运作能力。

　　e国

　　最卖力的网站。北京三环以内保证1小时送到，建配送中心、雇配送小工、开e国门户店，有人将e国的电子商务讽刺为“折磨电子商务”。但是，我们不得不承认，正是e国，让老百姓第一次建立起中国电子商务的“满意度”和“忠诚度”，尽管付出了更多的成本和血汗。但是，比起把大把的钱投入市场推广，e国的所作所为是值得业界的借鉴和用户的敬意。作为新生事物，在中国电子商务起步阶段，必然有大量的付出和开拓。要嘲笑互联网的“雷锋”，也不应该嘲笑那些真正努力做事的人们。

　　8848

　　最折腾的网站。除了几家门户之外，认为2000年上市是铁板一块的只有8848了。因此，无论是此起彼伏的上市传言，还是内部争论，以及最后的“一分为二”，8848都是2000年的最大焦点之一。可是，再折腾也折腾不过市场。2000年对于8848实在是黑色的一年。2001年，出身为B2C的8848以B2B的面貌出现，媒体会不会接受？用户会不会接受？资本市场会不会接受？实在凶多吉少。

　　Tom.com

　　最泡沫的网站。3月1日，一无所有的Tom．com在香港创业板上市，发行当日股价涨幅达五倍。之前，超额认购近2000倍，打破了历史纪录。达到了互联网资源转移最疯狂的顶点，可能也是最后的疯狂。现在，Tom．com的策略是利用资本市场高潮时融入的资本，在低潮期进行大规模购并，并向综合媒体的模式发展。但这种回归传统的做法能否顶住股价的不断下滑，是长期的挑战。

　　Chinese.com

　　最烧钱的网站。托普集团携国内股市圈得的6亿元资金，为中国互联网的普及添柴加火。巨额广告投入，远离实际却大而无当的商业模式，Chinese．com的出手使FM365也要黯然失色。天然地，Chinese就是个极具价值的品牌，但广告轰炸以后，Chinese依然响亮，依然还是原来的品牌外壳，因为谁也不知道Chinese．com是做什么的，我们也没有必要知道。

　　FM365

　　最迷茫的网站。从起初“赶超新浪”，到最近的联想电脑“附件”，从铺天盖地的路牌广告，到静悄悄的坚守。“自力更生”的产品思路和目空一切的自我感觉，FM365的战略调整过程是传统企业进军互联网的典型代表。也是必要的代价。究竟是“谁让我如此心动”？当然不是谢霆峰。

　　Eachnet.com

　　最幸运的网站。B2C被资本市场落井下石，网上拍卖在中国更像一阵热风，借助热潮而起的拍卖网站，享受了短暂的风光，更享受了无尽的痛楚。但是，在资本市场的冬季，国内拍卖网站易趣网，却“顶风作案”，于9月份，成功融入2050万美元，堪称奇迹。易趣网的结局有些特殊，但是也说明了一个道理：只有成功的公司，没有过时的商业模式。而且不管市场如何变幻，都要对自己的商业模式意志坚定。商业模式的随风摇摆是创业的第一大忌。

　　阿里巴巴

　　最令人操心的网站。是新的领导企业，开创商业模式的一代新风。国外的有AOL、Netscape、Yahoo、Amazon、eBay、CMGI、Ariba等，国内的有搜狐、新浪、8848等。虽然，阿里巴巴被《福布斯》评为全球最佳B2B站点之列。虽然，有世界贸易组织前任总干事萨瑟兰加盟顾问委员会。但是阿里巴巴的处境却很微妙：它是B2B的代表，却又是B2B遭遇质疑的首要目标。无数人忍不住替它操心：阿里巴巴什么时候能够让芝麻开门？当然，最让人羡慕的还是，在股市风暴之前，从软银搞定了2000万美元。
