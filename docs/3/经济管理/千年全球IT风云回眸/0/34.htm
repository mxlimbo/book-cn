美国迪士尼玩具网站倒闭

　　中新网波士顿5月23日消息：由和路迪士尼当大股东的Toysmart．com由于不敌eToys和Toys－rus．com等竞争对手，现正结束线上教育玩具业务。负责监督迪士尼互联网业务运作的Go．com公司经发言人说：“线上玩具市场的竞争，激烈得令人难以置信。”　

　　分析家说，鉴于线上玩具公司的数目去年大幅增加，Toysmart．com这次结业，是意料之中的。

　　外电报道称，Toysmart．com不是唯一倒闭的网上玩具公司。尼克尔洛德翁公司经营的Redrocket．com已经在五月五日结业。近数星期，KBToys．com裁掉三十位员工，连总裁都开除了。

　　业内人士认为，Toys－mart失败原因在于商品类别太单一（只出售教学用具），对于网上零售业来说，必须保持丰富的产品门类，并且力求与众不同，才可突围而出。

　　Toysmart．com的总裁洛德说，公司一百二十名员工，大部分在本星期遣散，少数员工会留下直至把未完的工作处理完毕为止。他说，Toysmart．com结业，一半是因为金融市场，另一半是因为玩具业和电邮的竞争本质。

　　迪士尼去年投资约二千万美元设立该网，当时承诺再提供廿五万美元经网上和其它方式推广。迪士尼?Toys－m art．com六成股权，在Toysmart．com的五人董事会中，可以占其中三个席位。Toysmart．com总共售出了七万五千件货物，包括教育玩具和小孩子的家具玩具。
