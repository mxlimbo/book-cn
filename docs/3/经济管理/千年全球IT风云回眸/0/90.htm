2001年互联网发展前景

　　互联网及依托互联网的电子商务在2001年的发展前景如何？美国洋基集团（the Yankee Group.）日前发表了一份名为《2001年互联网市场战略预测》的研究报告，对互联网在2001年的发展前景进行了预测。

　　报告主要指出了2001年的四项趋势：1.女性网民将成为互联网用户的主体；2001年将有更多的互联网公司倒闭；政府机构将出台更多旨在保护消费者网上隐私权的法规；在线广告收入的增长速度将呈现出不断下降的势头。

　　女性互联网用户上升

　　洋基集团在这份研究报告中指出，在今后一年中，女性将在互联网消费群中占据主导地位，女性网民的数量将超过男性。

　　吸引女性的注意力（以及她们的钱包），对于那些希望在大动荡时期幸存的网上零售企业来说至关重要。Greenfield在线公司上周发表的一份研究报告显示，女性网民购物的比例要高于男性，在2000年第三季度，女性在线购物的支出额就高于男性2%。

　　政府机构加强对消费者隐私权的保护

　　基于消费者隐私保护组织和消费者的呼声越来越高，在明年年底之前，政府机构将会出台更多旨在保护消费者网上隐私权的法规。

　　如果政府机构介入消费者网上隐私权的保护，最大的输家将是那些必须承担额外成本的企业，以及那些依靠出售消费者私人信息牟利的企业。这样，那些无法保护消费者网上隐私权的企业将面临代价高昂的司法诉讼。

　　免费ISP风光不再

　　洋基集团在报告中指出，尽管消费者需求在不断上升，但是，由于广告收入不足以弥补支出，免费ISP将难以生存，除非它们能想出更多赚钱的路子。另外，由于各大ISP逐步开始向消费者提供品牌化的网络接入服务，今后市场上将会出现一些基于收费模式的私营ISP。

　　在线广告市场发生巨大变化

　　由于在线广告收入的增长速度持续下降，在线广告代理商将进行大规模的合并重组。其结果是，市场上的网上广告代理商的数量将更少，规模将更大，实力也将更强——它们将具备向客户提供全方位服务的能力。

　　在线广告代理商将在明年推出一个关键服务——测量在线广告和离线广告的效果。

　　企业门户网风起云涌

　　洋基集团表示，ICP今后将会加强同企业间的合作，开发定制化的内容管理应用程序。这一情况已有所体现：雅虎公司在上周四宣布，它将和安达信咨询公司合作，向客户提供定制化的企业门户解决方案。在今年早些时候，这两家企业曾一起为北卡罗来纳州设计过一个定制化的门户网站；IBM在企业门户领域也相当积极。它已经和Verity公司达成合作协议，将其企业门户解决方案和Verity公司的信息恢复应用程序结合到一起。

　　电子商业多媒体化

　　洋基集团最后预测，汇流媒体将从一种单纯的娱乐应用程序，转变成为一种用于提高企业生产效率的工具。发生这种转变的主要原因，是宽带互联网接入的应用日趋普及，人们开始可以大规模传输商业应用类多媒体资料，如远程教育，远程电信会议，以及满足各种特定需要的数据信息等。它们都无法通过电视或广播等传统媒体进行低成本传输。
