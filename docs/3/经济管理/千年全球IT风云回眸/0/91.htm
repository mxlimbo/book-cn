二○○一年中国网络八大预测

　　预测1：资本

　　预测2001年WTO问题解决，国际资本有力带动国内IT产业的发展。同时，以国内创业板带动的国内资本市场开始构建自己的风险投资生态环境，摆脱完全依赖国外的被动局面。

　　预测2：股市

　　互联网产业主要由风险投资推动，股市永远是投资者的第一出口。因此无论状况如何，2001年我们还是要继续紧盯Nasdaq。会有更多的公司争取国内外上市，数量一定会超过2000年。

　　预测3：领头羊

　　纯互联网公司一马当先的美好时光已经永远过去，互联网的大潮将由更多方面的力量共同推动。作为中国最具资源、最应该领袖群伦的联想，能不能“吃一堑长一智”，在互联网方面真正有所表现，依然是我们关注的焦点。但是，互联网彻底改变中国IT产业生态的事实却无可质疑。

　　谁是未来的领袖，2001年应该初露端倪。

　　预测4：电子商务

　　电子商务依然是互联网最激动的领域，一旦互联网回暖，电子商务肯定是首当其冲。中国门户时代已经结束，只有整合、购并的几场热闹。如果没有电子商务的高潮，中国互联网的发展必然是残缺的。因此，2001年，我还是将很大的砝码加诸在电子商务上。只要B2C市场继续增长，只要网上消费客户满意度继续提升，只要网站不断改善服务质量和模式，资本市场必然还会重新捧起B2C这只“臭脚”。

　　预测5：无线互联

　　如果说2001年还不是无线互联网年，那么至少应该是宽带互联网年。因为，无论是宽带互联网的基础网络还是接入网络，都已经整装待发。只要商业模式有所突破，应该进入规模化实用阶段。尤其是宽带互联网的收入模式不错，在饱受资本市场打击下，对注重眼前收入的投资者应该更加秀色可餐。

　　预测6：鼠标＋水泥

　　互联网还是整个经济，乃至整个国家的热门和焦点，无论是传统企业还是新兴企业，2001年的互联网动作应该更大。毕竟，在2000万到5000万网民之间，是投资最佳时机，也是产业最热闹时机，谁也不会简单错过这个机会。对人才的需求不会因为调整而发生实质性的逆转。2001年，人才与资金还是这个高速发展的产业最饥渴的资源。

　　预测7：软件技术

　　互联网的到来，我们必须切实认识到不可逆转的产业价值转移，认识到软件业商业模式的根本性的转变。认识到是服务，而不是产品，将成为软件的核心价值。Linux的主流化是软件业变革的第一信号。微软．net计划的出台，虽然对微软自身是凶多吉少。但对于整个软件业全面转型的确是标志性事件，软件业的乱世即将展开。

　　预测8：商业模式

　　商业模式是互联网产业的最大特色，也是持续兴奋和刺激的根源，自然要不段演化。只不过过去的商业模式注重求新，今后的商业模式更注重实际收入。股票大跌，美国的互联网公司也是巨大压力，但对于他们的全球化却是利大于弊，因为大大降低了成本。可以断言，2001年将是美国互联网大举全球化的一年。至少对中国，门户、B2B软件等领域的国际化竞争应该全面开演。国内的兄弟们要作好思想准备。

　　总之，2001年，应该是有好戏的一年。
