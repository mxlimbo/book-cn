联想以3540万美元收购赢时通40%股权

据彭博社香港消息，联想控股公司宣布将以价值3540万美元的现金和股票购买赢时通40%的股份，从而成为赢时通的最大股东。赢时通被认为是中国最大的网上证券交易商。
最近一直致力于开拓互联网服务项目的联想表示，收购赢时通是为了利用中国证券市场难得的发展契机。到目前为止，上证指数和深证指数在今年年内的升幅位于全球前四位。联想表示，它将以每股9.2美元的价格认购540万股赢时通的可转换优先股，成为赢时通最大的一个股东。而联想将向赢时通支付1609万美元的现金，并将发行大约110万股新股给赢时通。联想发行的这110万股新股，市场价值在1309万美元左右。同时，联想将向赢时通提供价值620万美元的设备。

联想执行董事杨元庆称，通过投资赢时通，联想将能更快速地获得技术、经验、市场的资源，并在内容方面的得到加强。

由于多元化发展战略获得了投资者的认同，自今年年初以来，在香港上市的联想股票价格已经上升了90%。
