新世纪网络业大预言

　　互联网界所谓的dot-com公司正在大洗牌，各界对于电子商务近期内发展的预测也是莫衷一是。虽说大家都不否认全面e化是必然的趋势，但是对于眼前这一两年内的发展，反而没人敢拍胸脯保证。

　　一份由美国波士顿的谘询顾问公司Yankee Group所做的调查研究就对近几年内互联网的发展走向提出大胆预测。该报告首先认为，政府在即将来临的21世纪能否有效地保护网络使用者的隐私与安全，将对于网上销售增长有重大影响。此外，该报告也认为，前阵子倒闭裁员盛行的女性网站未来仍有利可图。该公司表示：“2001年女性上网消费人口将大幅增加，从而取代男性，支配网络销售市场。”

　　然而以女性为主的网上销售策略却也广受批评。尽管上述报告与另一份由Greenfield Online公布的报告均支持女性网站的商机前景─Greenfield Online报告指称2000年第三季女性消费者网上消费额较男性高出2%——但是多数业界人士仍认为，现阶段将行销对象限为女性，无异是在竞争激烈的网络销售市场上自绝生路。

　　此外，业界也普遍预期免费上网服务（ISP）将成明日黄花。事实上，市场上稍具规模的免费ISP，包括着名的Alta Vista，以及华人熟悉的北美新浪网（SINA）在内，都已经宣布终止服务。新浪网在寄发使用者的终止服务通知中明白表示，终止原因在于提供服务的ISP赞助厂商已无钱可烧。目前北美市场还在苦撑的免费ISP，也仅有NetZero较具知名度─但也不保证能撑多久。

　　Yankee Group也预期，持续衰退的网络广告市场增长率将促使广告代理商出现联合。未来将只有较大规模的广告商有能力提供客户较具弹性空间的广告配套服务。而该报告也对于当今正处于“度小月”阶段的入口与内容网站提出建言：唯有与大企业合作，推出针对客户需求量身打造的网站，才是维持竞争力的终南捷径。雅虎与安达信顾问公司（Anderson Consulting）甫宣布合作提供的个别化客户服务网站解决方案，就是最好的例子。
