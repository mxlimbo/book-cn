2000年谁是赢家？

　　当然你心里肯定也有自己的想法，比如我就觉得至少该把黑客们列为本年度的赢家吧，他们今年可是干出了不少惊天动地的大事。输家我想到的是那个财大气粗的沙特王子，今年在网络上，他可栽了不少。看看下面这些人、事，和你有没有共鸣？

　　赢家

　　David Boies律师应该算个大赢家。

　　现年59岁的Boies在今年可谓是大出风头。今年世人瞩目的三件大事中，他都是力挽狂澜的得力干将。首先，作为国家司法部的检查官，他在地方法庭对微软垄断一案初战告捷立下赫赫战功；紧接着，作为Nasper的律师，他又将已经成为众矢之的的Napster救出苦海；而在扣人心悬的总统选举中，他又作为戈尔的得力干将，使他在佛州获得反击的机会。这一系列的成功使Boies简直快变成超人了。

　　2000年的无线设备也是春风得意。

　　从无处不在的手机到让人着迷的“CrackBerry”，它们都是那样让人无法抗拒。我们无法想象如果没有了它们生活将会变得怎样。无线上网技术成了即将过去的一年中最烫的话题。

　　Oracle公司的首席执行官Larry Ellison今儿个也高兴。

　　看着老对手微软在反垄断案中老是脱不了身，他可以暗暗窃喜，更令他大快人心的是现在他持有的Oracle股票价值已经超过了比尔.盖茨的财富。（其实Oracle的股票也跌得厉害，只是没有微软厉害而已。他们两个的资产都今年都缩水上百亿美元。）

　　Napster的创始人Shawn Fanning发明的音乐互换技术震动了整个唱片工业。

　　从一个普通的大学肄业生突然变成全球注目的焦点，Napster也从一个18岁的小伙子某天的突发奇想变成一个一种现象，一个极具商业价值的模式。Fanning无疑也是赢家之一。在Fanning最风光的时候，同一天同德国媒体巨人贝塔斯曼签定合同，而且同时在《时代》和《新闻周刊》露面。

　　思科的总裁兼首席执行官钱伯斯（John Chambers）

　　在科技股市跌声一片的时候，钱伯斯带领着思科在2000年实现连续10个季度持续增长，同时占地688英亩的新厂区获得批准，思科的前景迷人，而他自己也有人给他戴上了“网络之王”的帽子，你说，他能不乐吗？

　　输家

　　Pet.com的首席执行官Junie Wainwright今年是欲哭无泪

　　曾经名噪一时的宠物商店Pet.com终于低不过今年网络界的凄风苦雨，于11月宣布关门。作为首席执行官的Wainwright只能表示“相当难过。”

　　.com公司的雇员也是2000年的输家，他们获得的股票期权变得一文不值，同时他们又面临着失业的危机，没有比这更糟糕的了。

　　Priceline.com的首席执行官Walker曾经是1999年福布斯杂志的封面人物——“新时代的爱迪生”，但是在股市的大崩溃后，这位‘爱迪生’可能需要一些新的创意了。

　　2000年对比尔.盖茨来说也是倒霉的一年。

　　反托拉斯法案使微软被一分为二，盖茨也从CEO的宝座上退了下来，“今天你要去哪儿”（微软主页上的经典问候）？还是回家睡觉吧！

　　网民们隐私也在今年黑客与准黑客中输得一败涂地。

　　从FBI的“食肉动物”到医院、信用卡公司无孔不入的黑客，你的名字、E_mail、消费习惯、喜爱的站点、病历、信用卡等等。你已经没有隐私了。

　　电子书：Stephen King的小说被搬上了E_book，我们可以通过屏幕来看故事了，但是好象响应的人不多，甚至像微软这种公司也没有动静。有什么理由让我们离开我们信赖的笔和纸而去迎接那个冰冷的多媒体魔鬼呢？
