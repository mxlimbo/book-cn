岁末点评互联网(一)

　　新世纪的最后几年间，互联网在世界范围的大发展，成为一道热闹非凡的风景线。这条四通八达的全球信息高速路，使人类获取知识、交流信息更加便捷，也使越来越多的人把“上网”当作生活中的重要内容。专家们预言，互联网对人类社会的政治经济以及文化发展，将产生越来越大的影响。

　　在２０００年，我国的互联网业的发展保持翻番的态势。中国互联网络信息中心的专家近日估计，在１９９９年上网用户８９０万、上网计算机３５０万台、互联网站点１．５万多个的基础上，到今年底，全国的上网用户有望超过２０００万，上网计算机将有８００多万台，互联网站点数将突破３万个。

　　网站：十字路口寻支点

　　自１９９４年我国正式联入国际互联网后，近年网络业发展强劲，曾有一段时间几乎每天都有新网站诞生。但是，不少网站经营者忽视了我国潜在市场巨大、现实需求不足的“网情”，表现出追逐上市、热衷炒作等急功近利的浮躁心理。现在，网站有的调整经营战略，有的改弦易辙，有的作价待沽，曾经“触网”的投资者也表现得异常冷静。用一家大型网站负责人的话说，现在众多网站处在选择突破的十字路口上。

　　需求、用户、市场是网络业发展的前提。在互联网规模最大的美国，互联网事业的快速发展依赖于雄厚的经济实力，和经济持续增长带来的巨大需求。与发达国家相比，我国在互联网用户数、普及率、主机数、网络规模等方面均有差距。发达国家用户普及率在３０％以上，而我国只有１．４％，互联网要真正形成产业规模，还有相当距离。

　　尽管我国上网人数不断增加，可网民的大多数对网上购物心存疑虑。由于企业信用、银行汇兑、物流配送等外部环境还不完善，电子商务一时还难以推广应用和形成规模。业内专家指出，网站要在拓宽服务领域、推出个性化、专业化的信息产品中探索现实的赢利模式。目前，国内网站正在进行电子商务的探索，并开始与传统产业的衔接，一些网站已经取得可喜的进展。

　　据国家有关部门透露，十多个相关部门已经在会商对策，将对发展电子商务进行综合部署。１９９９年，全国２００多个电子商务网站总交易额约为２亿元。随着专业网站的兴起，企业间的网上商务逐步占主导地位，预计到２００２年，电子商务总交易额将达到１００亿元的水平。
