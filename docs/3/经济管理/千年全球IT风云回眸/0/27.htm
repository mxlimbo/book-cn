Lycos“下嫁”Terra　夹缝中以图生存

　　长期生活在Yahoo!和AOL等互联网巨人阴影中的美国老牌.com公司Lycos，美国西部时间５月１６日宣布以125亿美元出售给Terra Networks，这一举措是在夹缝中艰难生存的Lycos为自己的未来选择的新的出路，也是美国门户站点第一次被来自美国以外的公司所收购。

　　Terra是一家来自西班牙的网络服务提供商，在Terra股票不会跌破20%的假设下，给Lycos的出价每股97.55美元。

　　欧洲媒体巨人Bertelsmann将从合并后的新公司购买长达５年、总价值超过10亿美元的广告和服务，并将为该公司优先提供站点内容。

　　合并后的新公司命名为Terra Lycos，总部将设在美国马萨诸塞州的沃尔瑟姆，Lycos首席执行官Bob Davis和首席财务总监Ted Philip保持原位。由Terra Networks通讯公司母公司Telefonica董事长Juan Villalonga出任新公司董事长，合并后的新公司在全球37个国家设有办事处，其中包括北美、拉美、亚洲一些增长迅速的市场。如果得到股东批准，该协议将在今年第三季度完成。

　　Lycos公司CEO戴维斯说，合并后的公司不在仅仅是互联网商业游戏的一个强有力的玩家，而是在全球市场都拥有巨大能量的发电站，双方的这个协议将改变整个行业。

　　Villalonga补充说，两家公司的联合创建了一个新的媒体巨人，它的能量是任何其它互联网公司或媒体公司都无法比拟的。

　　尽管由于看好此项交易，Lycos股票价格暴涨，但它们仍在已宣布的收购价以下顺利交易。

　　一位股票分析家说，现在想预测出股票价格差还为时过早，当年时代华纳与美国在线宣布合并时就存在价格差，但在几天内就顺利成交了。

　　他说，让市场知晓这一消息需要一些时间，人们在做决定前也需要仔细分析协议条款。他相信专门从事套汇交易的投资者一开始会持观望态度。“他们将是缩短差距的人。套汇投资者卖空Terra Networks，大量持有Lycos股票，将最终填补差额。但投资者下水前总希望能够看到前景。”尽管门户市场的震荡已成定势，此项协议实际上抬高了全球互联网竞争的门槛。

　　Terra Lycos将同时拥有Telefonica下属一家无线合资公司49%的股份。

　　交易完成之前，Telefonica将向Terra Networks支付20亿美元的优惠认股权。这样，Terra Lycos拥有的现金将达到30亿美元，成为世界上资本化程度最高的互联网公司。

　　Terra Lycos董事会将拥有14名成员，包括Villalonga和10名其它Terra Networks任命成员。Davis将加入公司下属媒体公司Telefonica Media的董事会。

　　Lazard Freres公司担任Terra Networks的财务顾问，Credit Suisse First Boston为Lycos财务顾问。

　　Terra和Lycos称他们预计2000年收入约为5亿美元。合并后两家公司拥有约5000万用户，每天页面浏览量达到1.75亿次。

　　去年两家公司均亏损。Lycos在1.355亿美元销售中共亏损5200万美元，Terra 7850万欧元销售损失1.73亿欧元。

　　激烈的竞争西班牙语和葡萄牙语市场是一个值得涉足的领域。分析家预计，仅在拉丁美洲，到2003年，网络用户总数将翻七倍。

　　这一巨大的市场潜力激起了互联网公司的激烈竞争，纷纷抢滩拉美市场。

　　除了拉美主要媒体公司，许多小型站点，如纽约的StarMedia Network和Argentine El Sitio也意欲在该市场中站有一席之地。世界上最大的西班牙语媒体集团，西班牙主要广播公司Grupo Televisa正准备启动自己的网站，Esmas.com. 这一趋势甚至吸引了一些拉美明星的支持。今年年初，Julio Iglesias和拉美电视明星Don Francisco就慷慨解囊，投资了一家新网络公司。

　　而这些公司面临着来自微软、雅虎、美国在线等美国网络巨人的越来越激烈的竞争。

　　美国在线计划把其拉美分公司做上市，打算在整个拉美地区开展基于订购的在线服务，启动多家门户站点。10月，微软宣布将与墨西哥最大的电话公司联手，共同开发美洲人的西班牙语门户站点。

　　美国在线拉美分公司与巴西最大的网络提供商之前一场激烈的战役已经在该地区打响。

　　上个月，为赢得该地区70万巴西消费者，美国在线提供了免费互联网接入。为了与之竞争，拥有该国众多杂志独家权的Universo公司不得不大幅降价。

　　在夹缝中生存尽管Lycos在受众群上排名第五大站点，许多分析家怀疑该公司是否有能力缩短与美国在线、雅虎和微软等行业领袖之间的差距。

　　一些人认为Lycos的出局对自己是一件好事。

　　分析家说，雅虎、美国在线和微软的成功使其它综合网站很难再有立足之地。许多其它“二级”站点已经决定重新调整方向，转而面向更具体的消费者群体。比如，Disney的Go.com将自己重塑为娱乐站点，Excite@Home也举起了宽带内容这面大旗。

　　比较起来，为了能够从领先网络综合站点中能够脱颖而出，Lycos一直在苦苦挣扎。

　　Lycos一直在寻找合作伙伴。去年，公司试图与Barry Diller的美国网络公司合并，但由于没有得到股东的支持，谈判很快搁浅。

　　一些分析家认为与国际公司合并对于Lycos来说是一个明智的选择，这样可以很其很快从对手中脱离出来。

　　Terra与Lycos公司的合并将创立一个规模可观的大型国际化网络公司。Terra Networks经营着面向西班牙语和葡萄牙语区的互联网公司，西班牙最大的电话公司拥有公司66%的股权。

　　一些分析家指出，该协议仍存在一定风险。西班牙语互联网市场仍在发展建设之中，可能不会立即获得收益。拉美网络市场是最不成熟的，到2004年，该市场将占全球广告市场的不到5%。他还警告说，Terra Networks的ISP用户只有200万，而其中有一半为免费上网用户。

　　根据1999年12月Forrester Research公司的分析结果，该年度，雅虎、美国在线和微软的浏览量总和占所有网络浏览总量的15%，剩下所有站点总和占5%。而这一差距在逐年拉大，2000年，大型网站将占17%，小型站点占4%，GC 2004年，该数字将变为，大型占20%，小站点1%。

　　与两年前的门户站点之争相比，过去的对手们现在开始纷纷另辟蹊径。

　　Excite.com宣布将集中力量开发宽带服务，而此举有可能是以牺牲其窄带门户站点作为代价。

　　AltaVista也从门户之争中迈开，回到搜索引擎这一老本行。本月初，AltaVista启动了旨在与Google.com一争高低的Raging搜索引擎服务。

　　与USA Networks合并破灭之后，Lycos也追赶着行业合并热潮，开始不断收购小公司，增加了财务站点，Quote.com，MP3公司Sonique，网络社区Valent和在线游戏站点Gamesville。

　　然而所有这一切都没能给站点访问量带来突破性的增长。有此看来，想跟上步伐并不难，难的是能够做全军的统帅。
