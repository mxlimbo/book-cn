数字解说2000电子商务年(二)

　　从吸引"眼球"到吸引"钱袋"

　　如果说1999年人们在研究"网民"群体，那么到2000年，一个新的消费群体已初见端倪――这就是电子商务消费群体。据"2000年中国电子商务个人消费市场研究报告"显示：在所调查的三个城市3532个网民中已有30％的人参与过电子商务活动，这部分人不仅是每周上网时间在6小时以上的网民，而且在过去一年中以消费者身份不同程度地参与过网上采购、网上拍卖、网上竞价、网上教育、网上订票、网上炒股等电子商务活动，以他们为代表的电子商务个人消费群体已经逐步形成。

　　年轻人更热衷电子商务

　　在这一群体当中，24岁以下的学生群体目前所占比例最高，达到51．5％。尽管这类人群大部分没有固定收入，但由于热衷互联网，追求时尚潮流，因此成为电子商务活动中最活跃的人群。其次一类高比例的电子商务消费群体是24岁至30岁之间的上班族，占到群体总数的40．3％。这类人群具备一定的经济实力，对新事物易于接受，而且热衷于时尚型的消费，不仅是使用互联网的高比例人群，也是目前电子商务个人消费市场上活跃的消费群体。虽然目前上述两类消费群体还不是社会的高收入、高消费群，但未来三五年后，他们将成为最具消费力的人群，而这期间中国的电子商务必将有极大的发展。可以想见，这些年轻的群体将成为未来电子商务消费市场的中坚力量。同时，这两类主要消费群体对于电子商务所持的积极态度和较高的认知程度，也将有力地拉动企业对消费者电子商务（B2C）市场的增长。

　　从目前电子商务消费群体的收入结构上看，大部分人的平均月收入在1000元至4000元之间，但平均月收入在1000元以下的电子商务个人消费者也占到21％的比例。可见，收入水平对电子商务消费行为的影响并非是主要的，电子商务正在越来越广泛地被各个不同的消费层所接受。

　　此外，目前电子商务个人消费群体还呈现出有高等教育背景的比例高、公司职员的比例较高和男性比例明显高于女性等特征。如果为最热衷电子商务的个人消费者勾勒一幅轮廓，他应该是一位年龄在18到30岁之间的男性，有高等教育背景，在公司就职且平均月收入在1000到4000元之间。

　　从吸引"眼球"到吸引"钱袋"

　　如同我们真实世界中的消费群体一样，电子商务消费群体在网上也有各自不同的行为特点和消费习惯。深入研究这些消费特点，细分不同的消费群体，对商家制定有效的电子商务发展和实施策略，把握商机，争取市场份额，有着极其重要的价值和意义。

　　ICP打造"门户"的时期，如何吸引"眼球"一度成为全球性的热门话题。而进入电子商务时代，特别是电子商务步入理性发展的阶段，"商务运作"被重新正视。同时电子商务的经营已不只属于互联网公司和风险投资商们，传统企业――特别是一批优秀企业的介入，不仅带来了丰富的产品和服务，更为电子商务领域注入了科学化、规范化的管理模式和企业经营意识，"商务"气氛越来越浓，新的商机与新的竞争也将伴随而至。仅仅吸引"眼球"已不能适应这种商务发展的需要，企业经营更关注的是如何吸引消费者。如何促使快速增长中的网民群体成为电子商务的最终消费者，如何使这些电子商务消费者成为自己的电子市场中的忠实顾客，如何整合网上、网下的资源使企业在新经济中建立新的竞争优势……这些问题已经摆在21世纪几乎每一个企业经营者的面前。通过"2000年中国电子商务个人消费市场研究报告"中对三城市电子商务个人消费群体的网上消费和行为特征的调查分析结果，可以得到解答上述问题中重要参考依据。同时，这一调查研究行为的本身也为电子商务的经营决策者们提供了科学的、可操作的决策手段。

　　研究结果显示：不同职业的电子商务消费者上网的高峰时间有所不同。比如：每周一至周五的工作或上课时间，公司职员上网比例最高，达到61％；国家机关人员上网比例其次，达到59％；而学生上网的比例仅为39％。在周一至周五的业余时间，学生上网比例明显提高，达到60％，而上班族人员上网的比例基本与一周工作时间的上网比例相同。周末和节日期间几乎是所有电子商务消费者上网的高峰时间，其中，学生上网比例高达78％，上班族人员上网比例也比平时增加8％左右。对于个体和自由职业者，由于其工作时间弹性较大，因此节假日的上网比例与平时的上网率基本相同。除了消费者在每周不同时间段的上网高峰呈现一定规律外，在每天不同的时间里在网上采购的商品也已呈现出一些有趣的特点。比如：我们从一些电子商务网站了解到，北京地区的网上消费者通常在白天上网选购CD产品，而在晚间上网选购图书。估计消费者白天选购CD主要是为下班后欣赏，而夜晚安静、自由的环境，更易激发人们的读书热情。

　　准确掌握网民们不同的上网习惯，详细了解消费者的网上采购行为，对商家制定有效的产品和营销策略是十分必要的。比如，商家可以针对消费者在每天不同时间集中采购不同商品的特点，有针对性地选择不同商品的宣传或促销广告在网上的投放时间。另一方面，对于那些为在下班后欣赏CD而在白天进行网上采购的消费者，商家在送货服务等方面应实现及时响应。尽管互联网使人们可以在任何时间（anytime）、任何地点（anywhere）找寻自己的需求，但消费行为的实现仍然受到目的、情绪、购买力等因素的影响。在传统商务中消费者行为、需求的研究已为企业普遍重视，而电子商务跨地区、跨行业等特点，使得不同消费群体不仅在网上消费行为、需求等方面与网下消费有明显差异，而且将在国家／地区文化、语言、消费结构、消费理念等方面呈现更大的差异性。

　　广告和促销怎么讨好消费者

　　广告和促销这两种传统商务中的营销宣传手段，由于互联网的出现而更为丰富多样。那么，目前流行的网络广告和网络促销形式哪些更吸引网上消费者的注意呢？从三城市电子商务消费者的偏好情况来看，电子邮件的宣传形式最受消费者喜爱，其次是网上的旗帜广告（banner）；而网页上的边栏广告只吸引约15％的电子商务消费者注意。同时，不同年龄的电子商务消费者对于不同内容的网络广告其偏好程度也有所差异。可以看到，"有奖促销"内容的网上广告最吸引电子商务消费者的注意力，其次是宣传娱乐活动的广告和发布新闻动态的广告信息。从网上促销方式来看，"网上商品打折"的形式对81％的网上消费者有吸引力，此外"网上购物抽奖"、"网上购物送礼品"等传统的促销方式在网上销售中对消费者也具有较大的吸引力。

　　由于电子商务消费群体中大部分是年轻人，而且选择网上渠道选购商品和服务的目的、动因与传统渠道中有较大差异，因此，网上促销更容易引起消费者的注意。调查的结果，只有2．5％的电子商务消费者对网上促销不感兴趣。可以说，互联网正在成为企业进行产品宣传促销、发布信息的有效渠道。但是对于不同的商品应该采用怎样的营销组合？不同商品的网络广告促销吸引哪类网上消费群体？……这些都需要企业根据不同的市场定位深入调查研究。

　　2000年流行的网上商品和服务

　　2000年互联网上的商品、服务和信息内容有了极大的丰富，网上的商务形式也不断创新。仅从消费者网上采购商品的集中程度来看，图书可谓流行榜排名第一的商品，所调查的三城市电子商务消费者当中，57％的人在这一年中在网上购买过书籍。其次，计算机软件、音像制品、计算机硬件及外设产品、小家电产品也保持着20％以上的购买率。从上图中我们还可以看到更多商品的网上个人消费情况。一些知名度较高的电子商务网站和大型综合网站成为电子商务消费者经常光顾的网上购物场所。

　　除了网上购物的电子商务消费形式外，网上教育、网上银行、网上炒股、网上保险、网上医疗等电子商务模式也在这一年中逐步发展起来。从调查的情况看，目前接受网上教育的电子商务消费者比例最高，达到65．6％。网上教育在传播各类知识的同时，也在培育着网络人才和网络文化，这也许是人们为适应21世纪网络化社会的生存竞争所做出的最有价值的投资吧。
