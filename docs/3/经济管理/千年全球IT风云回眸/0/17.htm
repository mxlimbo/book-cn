雅虎收购我国台湾最大网站奇摩

　　据我国台湾媒体报道，今日下午雅虎宣布，将购并我国台湾的龙头网站奇摩站。雅虎创办人杨致远今日下午亲自主持了新闻发布会会。这一购并将会以股权交换方式进行，据传奇摩站换取雅虎股权，奇摩站最大股东精业证实将以股权交换方式进行，但是并不愿说明交换比例。

　　我国台湾最大门户网站的奇摩、蕃薯藤都曾积极准备在美Nasdaq上市，但都在大市不佳下的情况下向后推延。

　　精业新任发言人副总经理沈桢林指出，目前精业持有33%精诚资讯股权，再由精诚资讯持有73.69%的奇摩站股权，以此推算，精业间接持有奇摩站约24.3%股权，比上半年的33%降低了些，主要原因为奇摩站上半年办理1500万美元现金增资，由外面法人参与认购，使持股比例降低。

　　业界人士分析认为：此购并案一旦完成，将彻底改写我国台湾的网络发展史，我国台湾网络业界先前尚未出现世界级业者称霸的局面。

　　据悉在我国台湾门户网站中奇摩站被公认为龙头老大。另外雅虎原本在台湾地区也名列3到4名左右，因此二者相加，领先地位将更明显。

　　我国台湾媒体报道，已经成功在Nasdaq上市的新浪网也曾经对奇摩站表达购并的意愿，但最后因为价格未谈拢而作罢。
