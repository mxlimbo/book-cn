世纪驱动倒闭 10万用户数据危在旦夕

　　世纪驱动(http://www.21drive.com)是国内首家专业提供网络硬盘服务的商业网站，据该站介绍到目前为止已有近10万的注册用户使用该站提供的网络硬盘服务，虽没有大规模的广告宣传，该网站却在广大网民中享有一定的知名度。然而今天登陆该站时却看到了这样的一个通告：

世纪驱动网站重要通告(紧急！！)

各位世纪驱动网站的新老用户

　　你们好！

　　今天我们怀着极其难过的心情向各位宣布，世纪驱动由于短时间内没有足够的运营资本进入，将从2000年9月29日18时起无限期宽(应为“关”——编者注)闭。请各位朋友尽快告知您的朋友，告知全国近10万的世纪驱动用户，尽快处理其在服务器上的个人数据！

　　世纪驱动网站是由深圳的一个互联网创业团队于今天四月份在深圳开通的专为网民提供网络存储服务的专业网站，自开通以来得到了各大媒体的关注，新浪、中华网等国内一大批知名网站都对其进行了大篇幅的报道，经过近半年的发展到现在已有近10万的注册用户，每天访问量近50万PAGEVIEWS。但由于缺乏足够的资金进入，无法为这服务提供更好的设备和更宽的带宽。为了保证用户数据的安全，特决定关闭本网站，请各位用户在对我们表示理解的同时尽快清理个人目录的所有文件。

　　最后，世纪驱动工作组的全体成员：Tobee,aniko,thunder衷心感谢各位对我们的服务所给予的宽容和理解，咱们后会有期！

　　如有任何问题，请尽快联系:tobee@cmmail.com

　　世纪驱动

　　2000年9月27日

　　据知情人士介绍，该网站的倒闭是情理中的事。因为网络存储需要很多的很多的硬件设备投入和租用比较宽的通信带宽，需要很多的资金投入。但由于该网站一直没有找到合适的投资方介入，设备和带宽都无法得到升级，特别是最近很多用户反映该网站速度特别慢而且长期没有维护，这可能是它要关闭的一个预兆吧。
