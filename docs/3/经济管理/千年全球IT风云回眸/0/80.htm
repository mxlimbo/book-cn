2000年中国网络十大案件(二)

　　六、网上泡制“性爱公社”案

　　7月12日，珠海市公安局计算机安全监察部门在工作中发现，《珠海视窗》的“留言板”中有淫秽内容，于是循线追踪找到了涉嫌留言的本市专线用户“珠海××国际保健品有限公司”。侦查中，警方从该公司负责平面及网站设计的员工黄×的计算机上发现了制作淫秽留言的证据，同时还发现了制作淫秽色情网站的证据。在大量的证据面前，黄×对其发表淫秽留言及制作色情网站的违法事实供认不讳。

　　从今年年初开始，黄×使用工作用计算机，通过公司的专线联网，利用国外免费主页空间，先后制作了“性爱公社”、“天天情色网”两个色情网站，并且利用国外色情网站的广告条谋取非法利益。同时，又在《珠海视窗》“留言板”中发表两条有淫秽内容的留言。

　　七、金洪恩电脑与惠斯特科技电子商务不正当竞争案

　　7月，我国第一起因电子商务而引发的不正当竞争案近日在北京市海淀区人民法院进行了当庭宣判，原告北京金洪恩电脑有限公司胜诉。

　　原告北京金洪恩电脑有限公司与被告北京惠斯特科技开发中心均为财务软件开发商。1997年11月，北京金洪恩电脑有限公司自主开发研制，并由清华大学出版社出版了对股市炒作进行全面指导的软件产品《股神》，其主要负责人称其《股神》产品投放市场后已占有了相当的市场份额。并于1999年4月8日取得了商标注册证。被告北京惠斯特科技开发中心在1999年4月生产出版了名为《股市经典》的软件产品，在该商品包装上用的是“股神2000”的名称，即产品内装光盘表面印制的名称仍为《股市经典》，对产品上机运行，其Readme.txt文件和光盘内容的文件名称也是“股市经典”。原告北京金洪恩电脑有限公司认为被告北京惠斯特科技开发中心的行为使消费者误认为《股神2000》是《股神》产品的升级换代产品，在市场上造成了混淆，损害了原告的合法利益，北京市海淀区人民法院最后的判决是，被告侵权成立。

　　八、北京某公司电子商务走私案

　　8月北京海关走私犯罪侦查分局密切配合，日前成功侦破一起涉嫌低报价格，骗取国家关税，牟取暴利的走私案，犯罪嫌疑人田某被抓获，查获走私赃物集成电路后封装条带，案值570万元人民币。

　　海关工作人员在开箱例行检查一批从德国进口的集成电路时，发现其进口货物的实际价值与报关单价格不相符，引起警觉。北京海关走私侦查分局接通报后马上行动，以走私货物为突破口，顺藤摸瓜，很快某公司经理田某就进入了侦查员的视线，7月18日，走私犯罪嫌疑人田某被抓获。

　　确凿的证据面前，田某交代了走私犯罪的全过程。去年，他经同学引荐，与北京某公司合作，担任挂名业务经理，代理中介业务。今年年初，当得知某公司需从国外进口一批集成电路后封装条带，用于IC卡智能模块封装的信息。田某通过关系揽下了这笔生意。在金钱利益的诱惑下，他胆大妄为，在洽谈生意中寻价、订货、发货均使用电子邮件往来，并将原始文件删除，无法在计算机系统内查到，企图销毁罪证。他采取欺骗手段，低报价格，用同一手法，先后四次偷逃关税76万元。最后一次，7月11日，将原本每1000个集成电路后封装条带价值64马克的货物变成了5.5马克，比实际价格低报了90%。

　　九、网上书店旌旗席殊出售禁书案

　　8月，网上书店旌旗席殊，因为违反出售禁书规例，而要面对北京市新闻出版局彻查。

　　据报道，旌旗席殊董事长席殊表示，此次事件相信是与早前曾出售一本禁书《沉沦的圣殿》有关，导致北京市新闻出版局彻查。他强调书店财务状况没有问题，但承认出售禁书及被彻查，肯定会为该公司带来不利影响。

　　十、网上对骂网下动刀 武汉一网吧发生凶杀案

　　12月武汉市中山大道永安网吧凶杀案，经江汉警方缜密侦查告破。11日，程业强等4名凶手已归案。主犯朱凡、谢俊的父母望子投案自首。

　　9日凌晨，刘仁俊在永安网吧网上聊天时，认识了网友程业强，双方在网上发生口角谩骂。刘的网级高于程，将程踢出网络。程大为光火，告知同伴替其“报仇”。

　　朱凡、谢俊等5名网友协力要在网上同刘比个高下。刘觉得程很不礼貌，便在网上问其人在何处？程回答：“爷们就在永安网吧，等你过来送死。”刘便约上朋友找程算账。双方相见，矛盾升级，发生扭打，刘掏刀刺伤程背部。

　　程的同伙朱凡等人上前拦住刘的去路，混战中，朱凡抽刀将刘连刺三刀，而后6人逃之夭夭。

　　刘仁俊因失血过多，送往医院后死亡。

　　武汉警方接警后，很快将此案告破。经审查，程业强等6人，初中毕业后在家闲玩，为寻求精神刺激，通过网上聊天结识异性朋友。一年多来，他们每人勾上的女网友都在20人以上。
