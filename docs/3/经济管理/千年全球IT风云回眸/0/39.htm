Adobe公司网站遭黑客“劫持”

　　美国时间星期二，一位黑客从Adobe.com网站的主人那里“劫持”了该网站，造成这家大型软件企业的网站在星期三的大部分时间里都无法正常使用其网络服务器和电子邮件系统。　　

　　Adobe系统公司负责信息系统的副总裁杰拉德·鲁特尔星期四证实有一名至今仍未查出身份的黑客当时有能力私自更改adobe.com网站的域名记录。该黑客似乎骗过了网络解决方案公司，把adobe.com网站的域名记录转移到了位于中国的ICANN授权注册商Paycenter公司。此外他还修改了该网站的域名联络信息，服务器名称和网址也被修改了。　　

　　域名服务器的修改导致星期三一早访问www.adobe.com网站的用户被引导到了Paycenter网站首页。此外，鲁特尔还说Adobe公司雇员在星期三的大部分时间里无法收到来自外部的电子邮件。那些发过来的邮件都得到了“未知主页”的答复。但目前还不清楚这些电子邮件是否被转发到了Paycenter网站或第三者手上。据鲁特尔说，Adobe公司星期三上午与网络解决方案公司和Paycenter网站共同合作更正了被篡改的域名记录。这两家公司星期四对我们提出的采访要求没有给予答复。

　　星期三adobe.com网址给出的联络人是中国厦门的希尔·李，同时附有一个mail.com网站上的电子邮件地址。李还被列为厦门的Macroscape电脑网络公司的网站管理员。

　　李在星期四给InternetNews网站发了一份电子邮件，否认他“黑”掉了Adobe告诉的域名记录，说他自己是被人误解了。李没有做出详细的解释，但是根据他的ICQ档案，李使用的绰号是“老黑”(Oldblack)，他今年24岁，他在他的个人兴趣中列上了“作黑客”。

　　直到星期四上午，adobe.com网站恢复了正常。这次adobe.com网站被黑客劫持是最近一系列与网络解决方案公司有关的黑客时间之一。今年6月，internet.com网站的用户被引导到了蒙特利尔一家网站，最近发生的其他黑客时间的受害者包括nike.com网站和exodus.net网站。
