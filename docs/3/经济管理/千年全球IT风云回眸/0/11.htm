低调狐步舞:搜狐上市述评

　　七月份是美国夏季假期的开始。这个时候，华尔街的动作总是比较减缓平滑，今年也不例外。搜狐今天（七月十二日）上市，跟市场动作非常合拍。该公司开价每股13美元。上市后头两个钟头，纳斯达克指数为4058，上升了2.57%，搜狐股票最高是13.78美元、最低12.625美元，走势非常平滑。收盘的时候，搜狐每股是13美元，保持了开市价，成交额是2,563,300股；纳斯达克指数为4099.59，上升了3.62%；道琼斯指数为10783.76，上升了0.53%。搜狐公司计划上市460万股（普通股），连同今年年初发行的优先股，一共期望获得资金八千多万美元，今天在纳斯达克融资五千多万美元。

　　美国有句俗语说：“没有消息就是好消息”（No news is a good news）。五月份和六月份，美国经济出现减缓现象，到了七月，投资者们等待联邦储蓄再次提高利率的消息，而这次利率变动则很可能是克林顿当局最后一个较大动作的经济政策。然而，在七月十一日美国州长会议上，格林斯潘表示，近期内不考虑提高利率。这等于没有告诉市场任何消息，对股票市场来说，自然是个好消息。十一日的市场走向是，格林斯潘讲话前略呈下滑、讲话后就持续平滑缓慢地上升。雅虎第二季度报告说明获利加倍，结果，今天一开市，该公司的股票就开始平滑上升，带动多数网络公司和纳斯达克指数缓缓上升。与此同时，中国网络公司网易、新浪和中华网的股票也一路平滑上升。在纽约股票市场，由于主要公司第二季度收入和利润普遍见好，道琼斯指数也呈现一路平滑上升。

　　在这样的气氛下上市，搜狐公司股票走向自然是平滑而没有大起大落。这两三个月来，为了平安上市，搜狐不但做了公司改组，而且一直保持低调：避免更多地在新闻媒体抛头露面，不但在国内如此，就是在美国开市的时候，该公司也只是做了简短的标价说明。二月完成上市登记以后，赶上三、四月市场大幅度下跌，该公司几次推迟了上市时间和变动了价格，将最初的每股13到16美元提高到每股16到19美元；七月十日再次在市场登记，准备十一日开盘，后来又决定推迟一天。因为市场波动而推迟上市和走狐步窥测时机，是正常现象，然而，一再推迟就会说明自己狐疑太重而导致投资者的信心下降。不过，在众说纷纭和前者上市出师不利的时候提高自己的价格，大都是自信的表现。搜狐公司用提高价格的作法当赌注，平衡了一再推迟上市可能造成的市场心理的负面作用。

　　今天开市的时候，该公司恢复了初衷，每股开价13美元。这样做，说明该公司看到今天市场见好，却仍然决定保持低调上市的态度。同时，该公司挑选中午上市，更是煞费苦心来捕捉时机的做法。在华尔街，中午是“联络”(Networking)时间，各投资公司利用这个时间分析上午交易、交换想法和作必要的协调。如果市场走势平滑上升，那么，即便出师不利，也不会下跌到流血程度。今天的市场恰恰处于这种状况。因而，搜狐公司来回游步到这个时刻上市，加上低调低价，几乎是保证了公司上市不至于跌破开市价格。

　　今天收盘的时候，网易公司每股12.5美元，上升了4.17%；新浪网公司每股28.8125美元，上升了13.55%；中华网每股21.625美元，上升了35.69%。与美国同行竞争者相比，雅虎每股124.9375美元，上升了18.42%；美国在线（AOL）每股57.5美元，上升了2.68%。上市的四个中国网络公司都没有创造最初媒体舆论所描绘的奇迹。按照这里分析公司的意见，搜狐公司股票收盘和开盘的价格没有变化的原因是：中国网络工业市场很不成熟，跟同行相比，该公司大同小异，没有特别显著的独特和吸引人的地方。因而，尽管搜狐公司采取了低调狐步舞上市的方法，也只是达到了更符合实情的融资目的，并没有创造发财神话。

　　上市的原意不是圈钱，而是变成公共公司(go public)，即任何人都可以通过购买股票而成为股东的公司，公司由此获得发展需要的流动资金。为此，在美国股票市场有一条最基本的上市规则：决定上市以后，公司就要处处表现出自己是一个公共公司，而不是上市圈钱的个人公司。说白了，公司要时时刻刻证明自己在为所有股东的利益着想和操作，而不是把股东投资的钱看作是原公司所有的个人财富。如果做不到这一点，至少说明公司领导层对市场、公司和业务都没有信心，因而，过了上市的初热以后，就很可能失去投资者的兴趣。

　　一些网络公司获得的资金够用几年，却也要上市。对这样的公司，华尔街投资者并不看好。原因：通过股票获得的资金是流动资金，股票是一种债务，在不缺钱的时候把公司的债务和流动资金猛地提高到上百上千万甚至上亿美元，是在降低公司利润比率，因而很明显，公司没有良好的经营计划，甚至可能是为上市而上市、把上市看作个人圈钱的把戏。

　　华尔街的投资者们不是傻瓜。动人的圈钱故事可以永远讲下去，他们却只愿意花钱听一次。利润和如何获得利润的具体营业计划，讲起来不那么动人，却是华尔街投资者们永远听不腻的东西。

　　三十年前，几乎没有非美国公司在美国股票市场上市。据汤普森金融交易公司报告，到2000年，非美国公司在美国股票市场上市的数量已经相当可观，欧洲公司最多，占全部的40%左右，其中最引人注目的是德国的集成电路公司Infineon Technologies AG (IFX.N)；亚洲公司占30%左右，其中最引人注目的是中国联通公司ChinaUnicom (CHU.N)，六月中旬，该公司曾在美国股票市场吸引了42亿美元；拉丁美洲公司占2.2%，据《纽约时报》报导，墨西哥和巴西股票市场试图跟美国市场竞争，因而吸引了绝大多数的拉美上市公司。美国掌握全世界股票市场资本总数的45%，日本和英国共掌握25%到30%。就可见将来看，美国股票市场仍然是获得发展资金的最佳选择，但不是唯一的选择。

　　目前，美国投资金融界对中国的网络公司总的来说并不特别看好，与此同时，绝大多数人都看到，中国很快将成为网络用户最多的市场。产生这种似乎互相矛盾的现象的基本原因，是中国的网络公司普遍没有获得利润的具体营业计划。换句话说，象任何企业一样，网络工业企业要想获得投资者的青睐，不在于用什么商业模式，而在于有没有在可见将来获得利润的具体计划。商业模式是获得利润的类型，不是利润、也不是获得利润的营业计划，而投资者是为利润而投资、是因为对利润计划有信心而打开腰包的。因此，如果中国网络公司要获得美国投资金融界的投资兴趣，就不要在商业模式上大作文章，而要根据市场具体需要拿出具体可行的利润计划。
