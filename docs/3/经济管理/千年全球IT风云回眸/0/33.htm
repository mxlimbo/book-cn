英又一家互联网公司倒闭

　　周一，Net Imperative（网络需要公司），专业的在线通讯及互联网信息提供商，继上周boo.com倒闭后，成为英国互联网史上又一家倒闭的公司。

　　据拥有公司28.5％的股份的Durlacher投资商讲，在今晨，公司召开了董事会，公司决定开始进行清算。作为互联网泡沫的主要投资者，Durlacher曾投资57万英磅给Net Imperative公司，但它目前的股票价值只相当于最初投资的1.9％的价值。

　　消息出来后，使得Durlacher的股票再度创29周以新低，达到每股68便士，这与它在二月中旬时的每股455.5便士相比，下跌了85％。后来略有回升。上周五，它的股价为77.5便士，它的价格/收益比为19.7。公司的整个市值为3.67亿英磅。

　　上周Boo.com因缺乏资金的再度注入，成为英国首家倒闭的互联网网站。目前公司正在清算，及寻找下家。
