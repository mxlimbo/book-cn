2000年中国网络9个大侠

　　2000年底，美国《福布斯》杂志公布年度中国大陆富豪排名榜，50个人中IT业的英雄格外醒目。但如今，如果你说中国互联网托起了一批数字英雄，许多人可能要满地找牙。谁是真的英雄有待大家评说。

　　李泽楷

　　互联网领域的成功故事个个不同。如果说，孙正义的成功是靠自己的先知先觉，那么李泽楷的成功就完全是依靠他老爸的资源优势，迅速暴富。但是，“小超人”的崛起的确是亚洲互联网发展的重大推动，也是整个世界对亚洲互联网刮目相看的主要原因。可以毫不夸张地说，由于李泽楷的出现，亚洲互联网从此进入了一个新时代。当然，当困难到来之际，代表最高奇迹的李泽楷也不得不承受最大的压力。如今，在许多互联网“英雄”寻钱无路的时刻，李泽楷动辄有数十亿美元的进项令人惊羡。但是，巨大的成功也是巨大的风险，作为亚洲互联网界的第二号人物，李泽楷必须把处于舆论中心、负债额高达400亿的电讯盈科牢牢辄住。为了大家，也为了自己。

　　孙正义

　　在寒风中，孙正义无疑是中国互联网最大的一个打气筒。这位“互联网帝王”已经投资609家互联网公司，投资近25亿美元，现已从30个上市公司中套现出30亿，在资本市场上还有250亿美元的股票。还有420多家互联网企业正在等着IPO。孙正义代表着互联网成功的顶峰，无人企及。但是，互联网股市的调整，他的软银和他本人也无法逃避地卷入旋涡。2000年，他在国内频频亮相，放出豪言：“未来几年内，我们将在中国投资200家以上的互联网公司。”“我对中国的互联网市场有百分之百的信心。股市的起伏跌宕是常有的事，关键要看清产业的未来。”对孙正义来说，他在美国创造的奇迹，在亚洲也可以拷贝。当然，现在互联网界，只有钱是硬道理。人们都翘首以待。

　　张朝阳

　　英雄也有气短的时候，至少现在的张朝阳如此：抛头露面的时间少了，话语中的语气也变了。但是如果没有张朝阳，中国互联网就少了什么。因为，迄今为止他是中国互联网界最重要、也最鲜明的一个符号，具有独特的象征意义。经历了风风雨雨，面临着风起云涌，张朝阳旗帜鲜明的“个人主义”已经四面楚歌，他身上独特的棱棱角角已经被磨砺得差不多了。因此，可以夸张点说，看张朝阳的变化就可以折射中国互联网的变化，看张朝阳走过的路，就可以理解中国互联网走过的路。难怪，也有人提出，如果说张朝阳真的代表了一个时代，那么如果他真从代表符号中退场，才真正表示中国互联网有了新的飞跃。

　　丁磊

　　互联网给美国创造的奇迹，不是技术，也不是应用，更不是产业。而是通过互联网，使美国百万富翁（社会资源的代表）的平均年龄从50多岁，降到了30多岁。相当于整个国家年轻了20岁，其社会影响和社会变化远远超越了互联网本身。以丁磊的年轻和财富来昭示互联网的巨大机会，无疑是最佳的广告，和最生动的题材。但是，万万没有想到，是丁磊率先开了中国概念股“流血上市”的先例，反而成了所有媒体嘲讽与抨击的把柄。互联网代表的变数有时实在让人哭笑不得。当然，这不是丁磊的个人过错，而是时势的必然。也是年轻的中国互联网必须经历的一场新的锤炼。因此，明年我们还要继续看丁磊，继续看丁磊背后还能崛起多少更年轻的“网络英雄”。这是中国互联网最有意义的“指标”。

　　马云

　　马云无论是组织“西湖论剑”，还是登陆“福布斯”，或者“中国互联网之父”的帽子，马云的最大心愿就是能够跻身“中国网络英雄”第一阵营。对此，北京媒体微词颇多，但是平心而言，其他“网络英雄”占尽天时地利，尽得北京的近水楼台之便，只有他处在“偏远”的杭州。如果我们能够以更加平和的心态看待阿里巴巴的崛起与发展，那么马云的确是中国互联网的一个奇迹。无论是马云代表的“B2B”概念还是“地方互联网”的象征，2001年都是人们值得关注的。

　　杨元庆

　　与其他“赤条条、无牵挂”投身网络的英雄们相比，杨元庆必须背负最沉重的传统包袱。但是，号称“IBM＋AOL”的互联网策略，也使他同时背起了中国互联网最重的担子。从FM365的推出，到一年内“实施100家、影响1000家”这样豪言壮语的电子商务战役，以及大比例、高价位收购赢时通，都没有成为成功的典范，而更多的是反面案例。看来，这位把PC卖得像白酒一样畅销的天才，在互联网里还要继续埋头耕耘。毕竟，做互联网与卖产品还是两回事，要融汇贯通，杨元庆就不得不具备“跨栏”素质。

　　王志东

　　新浪的成功无疑是一个团队的成功，但王志东还是核心。2000年4月，新浪网领先于中国的三大门户，逆势而上，率先登上纳斯达克，成为中国互联网的强心剂。虽然股票价格从58美元的高点，一直滑落到5元左右。但是2000年，新浪的确实实在在获取了实地，“第一媒体”的势头开始初露峥嵘。与其他英雄相比，话语不多是他的优点，也是缺点。比如“年底大多数网站要倒闭”这样的论断，实在不如“沉默是金”。这不，Yahoo、Lycos也都来了，谁死谁活，还有折腾。毕竟，把中国互联网市场轻易地留给少数网站，显然是不可能的美梦。

　　王峻涛

　　无论作为第一代股民，还是第一代网民；无论是外表，还是骨子深处，王峻涛里里外外都是一件活脱脱的“B2C”产品。要他转型成B2B，实在是可怕的折磨，只能把他埋没。1999年年底，王峻涛的算盘打得很好：2000年8848上完市，功成名就，就早日退休。但是人算不如天算，从数次上市的传言，到“淡出权力中心”的争论，再到8848分家的事实。他的退休计划彻底泡汤了，而且不知会推迟到猴年马月。回归B2C，为他的未来增添了新的变数。回归B2C，为他的未来增添了新的变数，但也是他回到了本源。2001年，如果B2C在葬礼中复活，王峻涛无疑又是最大的赢家。

　　王（先+先）

　　寒风中，号称“神秘人物”的王先先出现了，一下子成为业界最炙手可热的人物。因为他代表Tom．com（许多人称它为“他妈的坑”），在国内频频出手收购。在这个资源（尤其是资金）饥渴的产业里，王先先一下子被许多人赋予了“救世主”的期望。他的思路非常明确：首先是将Tom．com的尾巴“．com”去掉，变成一个实实在在的综合性媒体公司，初步完成“产业化”；其次，从香港这块互联网的弹丸之地突围，将实力全面渗透到内地市场。这个策略无疑十分正确。但购并并不难，难的是购并之后的整合与管理。这才是聪明、精明甚至带点狡黠的王先先所面临的最大困难和挑战。有道是“谋事在人，成事在天”。要从投资家转变为企业家，王先先还有很长的路。
