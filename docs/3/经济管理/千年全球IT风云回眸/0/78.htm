岁末点评互联网(二)

　　网民：需求日益个性化

　　文字、图片、声音、视像等在网上快速传输，互联网使“天涯若比邻”变成现实。网上会议、网上商场、网上教育等与人们工作、生活密切相关的服务项目，为人们在家办公、购物、学习、交流提供了可能。网上内容的快、多、活及互动性，电子邮件、搜索引擎、软件下载等带来了挡不住的“诱惑”，吸引着更多的人上网。

　　我国网民人数今年将达到约２０００万人，其中７５％以上是１８岁到３０岁的青年人，２５％是女性，８０％多的网民是大专以上学历。兴趣广泛的网民希望在网上获取信息、学习知识，希望上网速度快、费用低，给自己带来更多的方便、实惠。一位从天津大学毕业后工作一年多的网民说，刚开始上网时好奇、着迷，坐到电脑前不愿离开，鼠标无目的地乱点。现在上网是发电子邮件、看新闻、找专业信息。

　　随着生活水平的提高，我国家庭拥有计算机的数量将大量增加，习惯于上网获取信息的人会越来越多。不少网民都经历过“触网”时的痴迷到“用网”时的平常心，惜时惜金让他们学会控制手中的鼠标。为此，互联网提供的服务也将追求质量，日趋专业化，生活化。

　　在互联网事业高速发展中，一大批政府网站、媒体网站、社团网站、教育网站、商务网站已经初具规模，众多新建立的网站中，法律服务网站、医药类网站、家庭服务网站、女性网站、网上书店等排出长长的“菜单”，等待着网民的多样化需求。

　　信息：开渠疏浚做规范

　　把互联网称为信息的“海洋”恰如其分。但是，有关调查表明，绝大多数网民对网上信息的质量不满意，认为真假难辨，可信度不高。据中国互联网络信息中心统计，有近９０％的网民表示，对电子邮件中的“垃圾”会毫不犹豫地删除。

　　网上信息良莠不齐，这是一个不争的事实。由于管理不到位，一些网站对所传播的信息不负责任，甚至在互联网上发布、传播有害信息的问题日渐突出，这些使网上信息的准确性、严肃性大大降低，严重妨碍了互联网事业的健康发展，甚至对社会文化的发展带来消极影响。在我国，由于网民大多是年轻人，因此，加强国际互联网和国内网站的管理，努力为青少年健康成长营造良好的文化氛围尤为重要。

　　在年初的“两会”上，许多代表、委员们强调，信息产业作为新兴市场，也是最为开放的市场，内涵丰富、涉及面广。要在博采众长的基础上，趋利避害，规范管理，促其快速健康发展。１０月，国务院向全国人大常委会提交议案，提请审议关于维护网络安全和信息安全的决定草案。这个草案中，列举了利用互联网的１５种行为，明确规定“构成犯罪的，依照刑法有关规定追究刑事责任。”在政府有关部门的主持下，《互联网信息服务管理办法》、《互联网站从事登载新闻业务管理暂行规定》、《互联网电子公告服务管理规定》等先后出台。

　　我国的互联网事业要健康发展，一方面需要有关部门加强管理，另一方面要求有关经营者严格自律，履行自己的社会责任。目前，我国的互联网业正由初期的盲目、无序向理性、有序发展，政府的“兴利除弊、促进发展”的产业政策和“优胜劣汰”的市场法则，将共同规范网络业步入健康发展的轨道。
