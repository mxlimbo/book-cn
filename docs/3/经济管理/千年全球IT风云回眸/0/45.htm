爱虫”依旧兴风作浪

　　据报道，曾于五个月前袭击网络的“爱虫”病毒今日仍在兴风作浪，上周五，很多用户受到更具破坏力的新一代“爱虫”袭击，并深受其害。

　　近几个月来，已经出现了近50种“爱虫”病毒的变种。这次新出现的变异体有条引人注目的主题语："美国总统与FBI机密"，这种病毒惹麻烦的本事也因此倍增。据反病毒公司McAfee.com的官员透露，已经有大约30家组织声称自己被这种名为VBS/Loveletter.bj的讨厌程序纠缠而难以脱身。

　　和最初的“爱虫”相似，这种病毒是通过微软的Outlook软件传播的，可以从受害用户的计算机中删除图象文件。Gullatto称，微软已对其Outlook软件作了改进，加入了能帮助各公司防止病毒入侵的修补程序；只是有些公司仍未能安装此程序，使其系统完全处于不设防状态。

　　McAfee.com和另一家美国反病毒软件商Sophos均声称曾在自己的网站上发过帖子，提醒人们注意这种病毒。Gullatto称，美国和欧洲的用户都遇到了类似的麻烦。
