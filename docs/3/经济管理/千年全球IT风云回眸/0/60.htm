岁末聚焦互联网

　　热闹波动中，我国的互联网业即将走过迅猛发展的2000年。每年100％的发展速度，使互联网这个高科技的产物走进了寻常百姓的家庭。　　

　　互联网这条四通八达的全球信息高速路，吸引越来越多的人“网”上行。据中国互联网络信息中心的专家日前预测，到今年年底全国的互联网站点数将突破3万个，上网用户超过2000万，专线上网和拨号上网的计算机800多万台。　　

　　与上述情况不同的是，现在不断传来网站裁员、倒闭、被兼并的消息，年初风风火火投奔网络的人才又纷纷回归“传统”产业。　　

　　网站：十字路口寻支点

　　记者日前走进了京城一家坐落在豪华写字楼内的网站，发现西装革履的网站负责人并不轻松，他坦率地告诉记者，我现在每天想的问题是，对内按时给员工开支，对外去融资找钱，使网站度过这个难熬的冬天。谈到网站的前途，他乐观自信地说，我国的网络业肯定要大发展，网站“钱”景广阔，但网站现处在经营困难的冬天，投资还未见到效益，下一步，打算在应用和商业运作上做文章。在采访中了解到，现在，大多数网站目前的经营状况不乐观，徘徊在选择突破的十字路口。　　

　　现在多数网站是赔钱赚吆喝，表面看来，是今年纳斯达克股指的低迷和网络股的暴跌、风险投资人的谨慎花钱导致的。实质是网站自身发展缺少动力、核心、目标，网站经营者忽视了我国潜在市场巨大与现实需求不足的“网情”。一些网站开办之初，追逐上市，热衷炒作，攀比宣传，一些急功近利的短期行为是网站陷入困境的现实原因。如今，网站有的调整经营战略，有的改弦易辙，有的作价待沽。　　

　　融资、扩大宣传是企业发展的手段，决不是目标和归宿。互联网是工具不是一般产品，需求、用户、市场是网络业发展的前提。美国互联网的快速发展是建立在美国经济持续增长的基础之上。与发达国家相比，我国在用户数、普及率、主机数、网络规模等方面均有差距，发达国家用户普及率在30％以上，而我国只有1．4％。

　　互联网像电视、电话那样进入家庭，还有一段距离。　　

　　尽管上网人数不断增加，可多数还停留在免费上网浏览阶段。尽管电子商务一喊再喊，但由于企业信用、银行汇兑、物品配送等外部环境还不完善，难以推广应用。业内专家指出，网站要在与传统产业衔接、拓宽服务领域、推出个性化、专业化的信息产品中探索现实的赢利模式：在传统产品营销领域，互联网作为工具大有作为。世界上成功的网站非常重视优质服务和多样化的经营。美国的“雅虎”网站为方便用户，率先开发了网上搜索工具，此后又建立了中文、日文等十多种文字的搜索站点，开辟电影、体育等网站，使其用户数不断扩大。　　

　　财富的兑现关键看公司是否赢利，而非名气。　　

　　网民：需求日益个性化

　　值得注意的是，今天的网民心态、鉴别使用信息的水平、上网目的、需求正在发生变化。用户希望网上内容从不同角度给自己带来方便、实惠，甚至利益，希望速度快、费用低。　　

　　天津大学毕业后参加工作一年多的关延涛告诉记者，刚开始上网时好奇、着迷，到电脑前不愿离开，鼠标无目的地乱点。现在上网是发电子邮件、看新闻、找专业信息。遗憾的是，网上信息不少，但重复较多，专业知识信息少得可怜，应用不方便，上网时间比以前少了。　

　　随机采访中了解到，上网半年以上的人大都经历“触网”时的入迷到“用网”时的平常心，惜时惜金让他们越发珍惜手中的鼠标。网上信息产品不断满足用户变化的需求，才能留住老用户，发展新用户，提高点击率。　　

　　“变用户为客户”，一些大网站已开始新一年的战略调整。　　

　　信息：良莠并存待精选

　　如果说互联网是“路”，那么信息是“车”，现在网上是车辆拥挤，空车不少。

　　打开电脑，拨号上网，五花八门的信息映入你的眼帘。覆盖全球的各路信息确实给用户带来方便，但生活节奏快的现代人需要更快捷、实用的信息服务，前几年是把各种信息装到网络的篮子里即可，现在要装的是精品信息、分类信息、鲜活信息，总之信息要有使用价值。　　

　　绝大部分的上网人对网上信息的质量不甚满意，认为真假难分，良莠并存，可信度不高。有人戏言，网上“垃圾”成堆，“大字报”随时飞。网上信息数量不少，质量不高；一般信息多，实用信息少；静态信息多，动态信息少；二手信息多，原创信息少。更甚的是传播色情、暴力、谣言。长此以往，使网民对网上内容半信半疑，影响网站的点击率。　　

　　信息的权威性、准确性、严肃性，新华社、人民日报等传媒网站做得不错，广州的一位政府工作人员对新华社记者说：如今假新闻不少，看到比较要紧的消息，我会立即上新华网搜索，如果新华网没有，就姑妄听之。

　　上网费用偏高制约着用户的上网时间，中国互联网信息中心今年上半年的调查显示，上网用户每月的上网费在100至200元，而占全球互联网用户一半以上的美国标准上网价格为每月20美元，用户无限上网。国内网站在加强管理、降低成本方面仍需努力。　　

　　网站传播信息，要研究网民的心态、需求和经济承受能力。根据需求开发适用的信息产品，节省用户网上查寻时间，他们欢迎信息之车直达用户之门。网络商之间的竞争，最终会变成价格、质量、服务的竞争。首都在线已经降低上网费，并开通城市间漫游的全国服务专线。　　

　　我国的互联网业经过市场波动起伏的历练过程，由初期的盲目、无序向理性、有序发展。政府的“兴利除弊、促进发展”的产业政策和“优胜劣汰”的市场法则将共同规范网络业步入健康发展的轨道。有机构预测，在未来几年内，中国的网民数将跃居世界前5位，当用户超过3000万时，网络业会有质的飞跃。　　

　　冷冷的冬天必将过去，暖暖的春天一定会来。
