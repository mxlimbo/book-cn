2000年全球最佳CEO(一)

　　当许多高科技产业的经理人正忙于营造一个连线世界时，芬兰诺基亚公司的董事长兼CEO Jorma Ollila却独辟奇径，致力于开创一个无线世界，即无论何时何地，都可以通过无线方式将人、文字、数据连接在一起。Jorma Ollila称此举为“创建移动信息社会”。

　　“移动信息社会的作用在于帮助人类充分利用有限的时间，提高生活质量。它不仅有助于人类提高效率，而且有助于人类取得更辉煌的成就，取得更巨大的成功。它为每一个提供的超常发挥的契机。”他说。

　　对于这位前Citigroup公司的银行家而言，这就是他平生最大的夙愿。事实上，作为世界最大的移动电话制造公司、1999年岁入达199.2亿美元（197.7亿欧元）的诺基亚公司的首席执行官，Ollila八年来一直信奉并致力于倡导一种文化：那就是，敢于冒险。

　　“如果在整个事业生涯中，你从未曾体会过失败的滋味，那只能说明你个人的才智还没有得到充分、合理的发挥。”Ollila颇为自信地说。在诺基亚公司有一种显而易见的文化，那就是，容忍错误、鼓励员工学习和发展。对于Ollila这位政治学、经济学和工程学硕士而言，每天工作就是保持和宏扬这种文化。

　　比如，他十分关注人事任命。“每一次人事任命都是向员工发送一种信号：那就是，我们鼓励这一行为！我们想注入团队的正是这种个性、行为和经验。”Ollila说。“这是最强的信号。”他肯定地说。

　　再者，Ollila在诺基亚公司内部推行轮岗制。四名高级经理人必须经常性地轮换职位。1992年以前，Ollila曾任诺基亚移动电话公司总裁。1992年后，升任集团公司CEO。此后，公司任何一名高级经理人的岗位几乎都发生了变化。比如，诺基亚公司CFO（首席财务官）Olli-Pekka Kallasvuo在1997年至1998年之间曾任诺基亚公司美国分公司执行副总，后回到赫尔辛基西郊的Espoo任公司首席财务官，负责协调与投资者之间的关系，收购业务及地产业。

　　“我们的团队是一支年轻的团队。‘灵活’是致胜的关键。”Ollila强调说。

　　对于现年50岁的Ollila来说，亦是如此。他的CEO生涯大致可以分为三个阶段。第一个阶段，即在他担任CEO的前四年。当时，有线电缆业和轮胎制造业已经有了相当规模，但是，这些成熟的业务并不是诺基亚公司的核心业心。为了发展电信业，就必须当机立断、忍痛割爱。“这是极其棘手的问题。发展的潜力确实很大，但是，与此同时，我们必须花费更多的精力来重组其它产业。”他回忆说。

　　1996年夏天，大规模的资产重组完成了。Ollila将自己从1996年起到现在的CEO职务描绘成是“是一个让你感觉良好的核心产业中的一支百分之百致力于组织发展、变革、扩大客户群的团队领导。”现在他的CEO工作重心又发生了变化。随着市场竞争的日趋白炽化和移动通讯技术的进步，Ollila说，他准备更新诺基亚公司的软件，因为这将是一个大有作为的领域。“我们现在已经是一家软件公司，而且将来也会朝着这个方向继续发展。”Ollila说。下列这组数字足以证实Ollila所说的这番话。诺基亚公司2万名研发人员中有过半（约为60%）是软件工程师。十年前，软件工程师仅为研发人员的约33%。

　　“我们公司正在经历一场真正的、深刻的、根本性的变革，我们研发部门对软件内容的更新就是最好的例证。”Ollila说。“如果你认为这仅仅是一种制造业，你所做的就是生产移动电话的话，那你就大错特错了。倘若你真的持这种观点，那么，你是注定要失败的。”四年来，Ollila几乎是屡战屡胜，捷报频传。1999年，诺基亚运作利润高达39.4亿美元，比1998年增长57%。净销售额为199.3亿美元，增长48%。每股收益（EPS）增至2.26美元，增幅51%。2000年前九个月，诺基亚公司的财务财务状况仍然十分喜人。运作利润达到了34.5亿美元（40.5亿欧元），增长56%。净销售额达179.7亿美元，增长57%。每股收益增加48美分，涨幅58%。

　　但是，今年诺基亚公司也遭受了一些挫折。比如，Ollila于7月27日预测，较之2000年第二季度，第三季度每股收益（EPS）将低于18.9美分，诺基亚的股票价格下跌25%。事实上，第三季度的每股收益为16.2美分。

　　一个月前，英国《经济学》杂志一篇题为“大赌博：无线产业”称：“现在，Ollila先生以及他所领导的齐心协力的团队的智慧正在经受最大的考验，这一考验比90年代初期以来任何时候都严峻。”事实上，诺基亚的最大挑战来自日本。
