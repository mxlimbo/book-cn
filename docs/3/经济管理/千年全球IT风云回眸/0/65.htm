2000年中国互联网七大经典并购

　　并购上市作为资本运作的两大手段在互联网的发展中上演着一幕幕触目惊心的巨作，而今NASDAQ的萧条黯淡有目共睹，特别是在搜狐拿到了最后一张入场券以后，贴着CN字样的网络股成了那斯达克的禁品，于是并购成了风险投资商的最爱，在此笔者挑选了七个颇具代表性的并购事件与大家共享。

　　1：多来米，吃虾米长大的鲸鱼。

　　——个人网站第一次被推上了商业舞台。

　　99年6月，叶仁浩执掌下的多来米中文网myrice.com，400万一口气吞下了国内16家知名个人网站，其中不乏有华军软件园，黄金书屋中国足球网，等互联网上可圈可点的精品，通过并购-整合-包装-推出的连环模式多来米一路杀进了中文门户网站的前列，取得了CNNIC优秀网站排名15的佳绩，个人网站被第一次推上了商业化运作的舞台而叶仁浩作为一个传奇人物也频频见诸报头，多来米这个寓意中国人信息时代精神食粮的后起之秀一下子火了起来，时至今日，面对资本市场的低迷，一向充满幻想的叶仁浩也颇为无奈，包括“华军的出走”“多来米的裁人”，“IPO的泡影”直到最近传言lycos要收购多来米想不到以并购起家自己也逃脱不了同样的命运。

　　2：163电子邮局，“国有资产”还在流失吗？

　　——短时间的二次易主，骇人听闻。

　　163.net作为国内免费电子邮局的先驱，2000年终于跑到了它的尽头资本的要求无情的斩断了它的国有“情结”，5000万转手深圳新飞网倒也不足为奇，互联网的生存和发展本来一开始就很难逃脱风险投资的左右，然而短短数月又巧妙的转入了TOM的旗下，之间夹杂着一个神秘人物王先先更是让人匪夷所思，“一身兼二职”也成了史无前例的创造，现在的TOM钱亏的比股票上的缩水还要多，依此现状163.net的收购不得不一推再推，真不知道中国第一家电子邮局的结果如何。

　　3：联想闪电入主赢时通

　　——门户与垂直网站的碰撞。

　　2000年对联想来说应该是最辉煌的一年，柳传志终于把联想从传统PC商带进了互联网的新天地，FM365的推出直到神州数码的跨世纪行动不难看出联想在向全球财富500强的目标挺进，而贯于其间，闪电入主赢时通无疑是经典之笔，6月23日，联想以3537万美元（合计近3亿人民币）正式入股中国最大的证券及移动电子商务平台提供商YESTOCK。COM，从而走完了继FM365 以后终结FN365的第二步棋，YESTOCK和LEGEND的联姻可以说是建立在双赢互利的基础上的，高科技股的低迷打碎了赢时通独立上市的美梦，特别是内部股东的变更使它难以继续支付巨额的运作费用，相反在这一点上，联想给了它需要的一切，包括联想自己的声誉和威严，同时赢时通也成了联想抛砖引玉的高招后面的棋局是怎样，FM365，神州数码，还是VC商的介入，我们期待着。

　　4：搜狐并购中国人——同样作品牌，同样是网络公司，1+1>2吗？

　　2000年的网站并购多的简直让人难以悉数罗列出来，其中“SOHU。COM”和”CHINAREN。COM“的“握手言和”无疑是最具争议的一个。同样都是靠品牌起家，牺牲那个放弃谁，尽管张朝阳一再声称一段时间内两个品牌将共存然而大家都清楚同一个公司不可能花钱来作两个品牌，如何在磨合中淡化中国人的品牌融入到搜狐的黄色文化中显然是并购的关键，不过资本市场倒没有因为中国人的加入而掉头，甚者搜狐的股票一度跌破3美元的防线，中国人在搜狐手里，是杀手锏还是一个更大的包袱，是资源互补互利双赢还是两败俱伤只能用事实来说明。

　　5：广东证券发威，盛润化名中天网——传统企业终于向互联网开了炮。

　　网络经济的产生，人们曾经用“注意力”“眼球”来形容它，描述它，赞美它。COM万能论一时间被炒的天昏地暗，终于NASDAQ对高科技股下了禁令国内互联网的规范管理更是第一个拉出网上证券来开了刀，“非券商禁止开展网上交易”的最新规定无形之中将纯粹的网上证券网站的E-trade梦想打入了地狱相反这也成了传统券商迅速挤进互联网的绝好时机，看来还是政府给了传统券商一个奋起直追的机会，当然广东证券对盛润2000的收购给传统证券交易商提供了一个非常值得借鉴的触网模式，而盛润2000也歪打正着的进入了一个新的领域——证券ASP服务提供商，得以有更大的发展空间和一张比原来更漂亮的脸蛋无论从哪方面看，这次并购都是几近完美的。

　　6：携程并购现代运通——鼠标+水泥的典范。

　　互联网的兴起，带动着全球所有高科技公司股票飞涨，速度之快令人瞠目然而随着新经济的产生，泡末成分是不可避免的，而最终的破灭又是必然的之间除了风险投资商在大把大把的点钞票外，摔得最疼的还是网络公司自己痛定思痛以之余，不免想起自己以前曾经嗤之以鼻的“老古董们”不得不感叹互联网毕竟只是新时代的工具而已，哪怕它带来的是革命性的变化抛除传统经济的支持互联网更多的只是概念和激情，于是传统与纯网络的结合成为风险投资商为NASDAQ预定的另一道菜，而最先品尝到它的甜头的应该是下面的携程旅游网（CTRIP。COM），10月16日，携程网宣布收购现代运通商务旅游服务有限公司，在此之前，现代运通是国内最大的电话订房中心在酒店，宾馆及其他相关服务业中也颇具实力，成熟的销售渠道众多的合作伙伴更是携程网垂涎已久的，运通的加入俨然成了CTRIP在高科技股中风雨飘零的定身线，相信最新一轮1200万VC的注入足以说明了一切。

　　7：虎吞奇摩，台湾门户重新洗牌——国际网络公司的介入，竞争激烈之余，中国互联网市场将趋于成熟。

　　11月9日，全球最大的网络公司雅虎YAHOO宣布发行255万股全部购并台湾奇摩网所有股份和股权，折合1。6亿美元，雅虎一声不响的吞下奇摩的确让人始料不及，然而又在情理之中，随着互联网的发展亚太地区早已成为国外网络公司心怡的圣地，而对中国互联网的市场更是觊觎已久并购奇摩可以迅速占领台湾地区的市场，进而加快开拓全球中文网站的步伐也必将引起台湾地区网站并购的热潮，其成功之处在于奇摩近40%的本地市场占有率和雅虎国际化运作实现了很好的互补，然而毕竟奇摩网是本地色彩很浓的网站，人们能不能对雅虎-奇摩的品牌认可还是个问题，再者由于两岸文化差异的影响，奇摩这个跳板能不能把雅虎载到大陆的互联网空间也很值得考虑。
