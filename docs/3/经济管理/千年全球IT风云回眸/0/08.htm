“新浪”成为获准海外上市的“黑马”


　　4月据《北京晨报》报道，日前，一直对海外上市保持低调的新浪网爆出冷门，新浪已获得信息产业部和证监会的批准，允许其在海外上市。新浪网极有可能成为今年在美国纳斯达克上市的第一家中国互联网服务提供商。

　　据报道，新浪首席执行官王志东近日将前往美国，但他并未透露在美上市的明确情况。

　　报道说，新浪高层对此事采取了慎而又慎的态度，但新浪内部人士透露，王志东不日将赴美，届时此事将进一步明了。另有媒体报道，新浪首席执行官王志东表示：“我们一直在就海外上市与信息产业部和证监会进行沟通，目前一切进行得挺顺利，但要做的事情还很多，还会有一些不确定的因素。”

　　今年以来，有关中国互联网公司将在美国纳斯达克上市的传闻日盛一日，网易、搜狐、８８４８均成为被议论的热点网站。惟有新浪似乎身处局外，处变不惊。可能谁都没想到，不声不响最为低调的新浪，上市进程却是最快的。
