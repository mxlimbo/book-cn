2000年中国互联网官司素描

　　“哎，我当时在法庭上真的想哭。他们的律师回避版面内容不谈，只是抓住我们告状的动机不放，说我们想靠官司炒作成名……我感到很悲哀，为互联网，也为法律。”

　　坐在北京华彬大厦穹形的玻璃大厅里，阮小姐讲着她在法庭上的遭遇，她端咖啡杯的手因为情绪激动而颤抖，看上去她显得非常疲惫。26岁的阮小姐平生第一次遇到官司，她所在的今夜网状告的是中国第一门户网站——新浪网。

　　以小搏大：胜算几何？

　　今夜网诉新浪网、E龙诉搜狐

　　当王志东刚刚把新浪送上纳斯达克，就不得不面对新浪的第一个网络官司。9月18日百网公司的今夜网状告新浪侵犯著作权，索赔额是10万元。

　　这是新浪头一次上公堂，类似这样的问题以前碰到不少，律师函也接过很多，但都能以协商解决，百网公司却拒不接受任何协商。新浪不承认侵犯了今夜网的著作权，因为今夜网的页面很难说具有独创性，如果说页面相似就是侵犯著作权的话，那“新浪”至少可以告五六十家网站。随后，新浪反过来把百网公司告了，原因是今夜网直接盗用了很多新浪的东西，如“网站导航”等，索赔额是40万元左右。其目的估计还是想要和对方协商解决。

　　就在上一个案子的处理还没有出来的同月内，另一知名网站搜狐也因网站内容涉嫌抄袭被e龙告上法庭。

　　这两起官司因为关系到知名网站，以及率先涉及网站之间的经营纠纷而备受瞩目。对于小网站告大网站的做法，是秋菊“俺就是要讨个说法”的执着，还是一种炒作？业内莫衷一是。在当时，很多网站的页面都是互相模仿制作的，中国网络业还没有明确的法规，且网络著作权的界定确实是个难点。在“灰区操作”的困扰下，靠的只是从业者的自律。

　　IT门户：相煎何急？

　　Chinabyte诉天极、赛迪咨询诉Enet

　　大众门户网站被别人告了，而中国四大IT门户网站也不甘寂寞，在岁末凑了个官司热闹，相互之间打起了官司。11月17日，天极网在自己一周年生日之时收到了同行业兄弟chinabyte的生日“礼物”——Chinabyte将天极网告上了法庭。这是一份很尴尬的礼物：“天极网大量抄袭Chinabyte的作品，还注明所抄作品版权归自己所有。”这是自《互联网站从事登载新闻业务管理暂行规定》出台后，第一桩与此相关的网络官司。

　　同样的故事也发生在另外两个IT同业兄弟之间。11月21日，赛迪咨询发现一份7月27日在赛迪集团旗下的《中国计算机报》上首次发表的产业咨询报告，7月31日赫然出现在硅谷动力的网站上。原告方赛迪咨询以硅谷动力侵犯著作版权为由，要求被告硅谷动力赔偿损失1．5万元人民币，并在媒体公开道歉。被告方硅谷动力声称，硅谷动力是在上海一家网站上转载这份报告的，并非直接转载自《中国计算机报》。硅谷动力拒绝赔偿和道歉。

　　个中滋味令人回味，据说是为了预演上市之争：专业门户网站是继大众门户网站上市后最有可能上市的，而IT门户网站又是中国发展得最健全、最有可能率先被纳斯达克接纳的网站。据传这几家公司都在暗中运作上市事宜，时至今日不知道上市对这些IT门户网站还有多大的诱惑力。

　　李逵李鬼：不见不散？

　　上海东方网诉山东东方网

　　像李逵遇上李鬼一样，这个官司的出现很有戏剧性：7月31日，一个网友在输入eastday．com（上海东方网）时不小心在eastday后面都加了“s”，结果发现了另外一个″东方网″，其中文名称与上海东方网一致，而其域名和网站标志也与上海东方网有惊人的相似。其次，该“东方网”下设的9大频道与上海东方网开通时首批推出的频道名称一字不差，所有栏目名称都有很强的相似性。另外，该“东方网”在页面设计、布局以及所使用的字体、颜色等方面都与上海东方网大同小异，整体风格极其类似。

　　经实地调查发现，5月27日下午4点，上海东方网上网测试20多天后，该中心就申请注册了eastdays．com的国际域名，6月5日又申请了国内域名。就此，上海东方网股份有限公司法律顾问发表声明，称济南开发区梦幻多媒体网络技术开发中心大量复制、抄袭东方网的内容，严重侵害了上海东方网的合法权益，造成了经济及名誉等损失，并要求该企业立即停止侵害行为。

　　而济南梦幻认为：原告对中文名称“东方网”、域名eastday．com和eastday．com．cn并不享有任何法定权利；原告在起诉状中所称的理由完全不能成立。因此，请求法院依法驳回原告的所有诉讼请求。

　　一宗看似简单的域名纠纷案件就这样变得扑朔迷离，谁是李逵谁是李鬼？究竟谁是合法的东方网？还是在法庭上不见不散吧！

　　网络侵权：没完没了？

　　彼阳诉新浪、“唐吉诃德”状告搜狐网

　　彼阳是一家网站的编辑，真名叫孙光海。3月27日他独家发表在天极网上的原创文章《263侵权案：三方现身讨说法》，7月被新浪以“来稿”的名义刊登。作者“彼阳”以个人名义对新浪提起诉讼，并迅速引起业界的关注。

　　11月7日下午，双方庭下达成和解。新浪承认侵犯彼阳《263侵权案：三方现身讨说法》一文的著作权，并以适当方式向原告公开赔礼道歉，赔偿原告经济损失，合计人民币6162元。

　　彼阳的官司首开个人胜诉商业网站的先河，值得一笑的是恰恰是有关侵权案的文章被侵权。也许是受彼阳胜诉的影响，11月24日，搜狐也被一作者告上法庭，引出了第一个追究“超级链接”侵权责任的官司。

　　状告“搜狐”的刘先生讲，他发现搜狐网站未经他允许，竟擅自将他1995年翻译出版的《唐吉诃德》一书的三种版式在网上全文登出，刘先生认为搜狐的做法严重侵犯了他作品的著作权，因此他要求搜狐立即停止侵权，公开道歉，并赔偿经济损失10万元。而搜狐矢口否认。原来，刘先生通过搜狐的搜索引擎找到的3个版式的《唐吉诃德》实际上存在于搜狐所链接的3个不同的小网站上。

　　互联网的官司总给人一些新鲜，照这样看来，互联网里的一些特有元素，包括内容、域名、网页风格、中文名称、超级链接等等，都有可能成为官司的理由。没有法律的规定和约束，涉及上述元素的官司将会没完没了的打下去。

　　网络官司：打出晴天

　　综观2000年的网络官司，数量急剧增加之外，还引出了好几个第一案，涉及的面更加广泛。从内容、域名、网页风格、中文名称、超级链接等，涵盖了互联网特性的方方面面，已不是靠从业者的自律就能解决的。在无数个面红耳赤、无数个对簿公堂之后，网络官司终于打出了一片晴天。

　　11月份，信息产业部颁布了《互联网电子公告服务管理规定》、《互联网站从事登载新闻业务管理暂行规定》；12月20日，《高法就网络著作权纠纷案适用法律若干问题的解释》出台，网络版权和著作权等终于有法可依，上述未完成的案件也终于有法可依。在互联网已经成为全球信息化核心力量的进程中，一直滞后于发展的政策，终于由旁观进入了实际操作，开始为产业发展提供实际的推动力，毕竟发展是一切的最终标准。

　　随着网络的普及，越来越多的人会像阮小姐那样去面对形形色色的网络官司。告别人、被别人告——他们在重复，也似乎想证明什么。从12月21日起，我们可以放心的大声向对侵犯我们权力的网站说：我要告你！
