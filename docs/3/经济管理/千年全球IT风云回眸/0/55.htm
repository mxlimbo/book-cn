2000年硅谷十大风云人物

　　仅仅在十年之内，硅谷里的影响力从社区领导人身上，戏剧性地转移到了那些在全球舞台上运作的高科技公司CEO身上。

　　历史演变到今天，决定影响力的是选举总统而不是市长的能力；是与中国建立贸易关系而非建设一个新市区的能力，是影响白宫而非市政厅的能力。

　　为了确定谁是最有影响力的人物，墨丘利新闻报在自1979年以来每10年一次的调查中，访问了政界、商界、劳工界、学术界和慈善事业领导人。结果得出了由大人物们自己选出的40名最具影响力人物的名单。它显示出高科技企业领导人是如何跃入经济力量和政治影响的范围的。

　　这种影响力的转移，为一个曾经被各州和联邦政客们忽视了地区，开辟了更广阔的前景。硅谷亟待解决的交通问题成了格雷.戴维斯州长优先考虑的问题之一，它的高科技政治议事日程在华盛顿受到了欢迎。

　　尽管他们取得了成功，硅谷的新的影响力代理人们却有一个限制了他们影响力的共同特性：高科技领域的需求迫使他们把公司的日程放在首位，很少有时间去理会民间事务。但一旦参与进去，这些领导人就用高科技企业所钟情的分散风格，来攻击社区问题。他们集合在一起来解决一个问题，无论它是一个学校保证金或是有问题的慈善事业。但是，问题处理完毕他们跟着就解散了，直到下次出现问题时再重聚。

　　桑特克莱拉县前主管人之一迪安尼.麦肯纳说：“你需要一个特别项目时打电话给他们，他们就会为它做出贡献。任务完成之后，他们就走了。”结果是，硅谷里最有影响的人物没有组织一个有凝聚力的架构。而在其它城市有可能刺激社区改革运动的提议，如把奥克兰A’s带到桑特克莱拉，未能得到多数人的支持。

　　墨丘利新闻报的调查发现，人们在硅谷里谁是最有影响力的人和他们的影响是如何发挥作用方面，有着惊人一致的看法。在前十人当中，其中七个是高科技行业的领导人物，一个是政界人物。另外两人分别是墨丘利新闻报的发行人和它母公司的首席执行官。

　　如下是这十大影响人物的名单。

　　克雷格·贝瑞特(Craig Barrett)

　　英特尔公司总裁兼首席执行官，他凭借自己的制造业专长使英特尔成为业界效率最高的个人电脑微处理器制造商，现在他正带领着在美国最有价值的公司中排名第三的公司，进入新市场和开展一场吸引优秀教师到硅谷里的学校任教的活动。

　　约翰·钱伯斯(John Chambers)

　　思科公司总裁兼首席执行官，在他任职期间，思科公司的市值由90亿美元上升至4410亿美元，在美国公司排名中位列榜眼。而且思科的增长导致硅谷住房和交通网络需求的增长，他的地位因此飞速上升。

　　约翰·多尔(John Doerr)

　　Kleiner Perkins Caufield&Byers公司风险投资家，他是硅谷与华府之间的枢纽，并领导了把高科技议事日程变成美国的议事日程的运动。

　　罗恩·冈萨雷斯(Ron Gonzales)

　　圣约瑟市市长，因领导硅谷最大的城市使“海湾地区快速运输系统”通向圣约瑟市，这件海湾南部头等大事为他带来了不低的知名度。

　　卡尔·哥瓦迪诺(Carl Guardino)

　　硅谷制造集团总裁兼首席执行官，他把一个无目标的组织转变成了一个在加州首府萨克拉曼多和政坛人物中，推动硅谷议事日程的举足轻重的商业集团。

　　杰·哈里斯(JayHarris)

　　墨丘利新闻报主席兼发行人，经营报道硅谷当地新闻的日报，最近又在联合道路(United Way)和海湾地区快速运输系统(BART)项目中扮演更重要的社区角色。

　　斯科特·麦科尼利(Scott McNealy)

　　Sun微系统公司主席兼首席执行官，在经营为互联网提供动力的电脑服务器的主要制造商公司的过程中，脱颖而出成为高科技行业里反微软派别的精神领袖。

　　吉姆·摩根(Jim Morgan)

　　应用材料公司主席兼首席执行官，领导公司击退了频繁来自海外的挑战，使他所在公司保持了头号半导体生产机器生产商的地位。他为慈善和社区服务承担的义务，在高技术同行中树立了权威。

　　卢·普莱特(Lew Platt)

　　惠普公司前主席兼首席执行官，在去年12月退休前是硅谷深受尊敬的政治家。

　　托尼·利德尔(Tony Ridder)

　　Knight Ridder公司主席兼首席执行官，14年前离开圣约瑟市时，他作为墨丘利新闻报发行人，是该市主要公民义务推动者。当他重新回到圣约瑟时，把墨丘利新闻报的所有者Knight Ridder公司的总部从迈阿密迁移到了圣约瑟市区。

　　风云人物名单的高科技行业倾向与十年前的名单相比有着戏剧性的变化。这次前40名当中有20名是高科技公司里的高级管理人员，而十年前的名单中只6人，而6人当中只有慈善家和惠普公司创始人大卫.帕卡尔德被列入前10名。
