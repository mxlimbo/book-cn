2000年IT业最吸引人的10大事件(一)

　　回首2000年，几多忧愁、几多欢喜!这一年里，人们从盲目再度回到了理智!也许我们大家都还清楚记得，曾经喧嚣一时的Y2K(千年虫)问题，使得我们差点带着足够的食品，躲到防空洞里，待Y2K走后再出来。好在我们现在对它了有了更常识性的了解。也许人们还记忆犹新，一路飞涨的互联网股票因无法获利，而遭到人们的唾弃!大多数人都还认为软件巨人要想推翻那个捏造的事实裁决书，难!比什么都难!但当我们收到他或她爱我的邮件时，还未打开情书附件时，心里不免一阵窃喜，可是打开后，又得到了什么呢？请看以下的2000年IT业10件引人注意、发人深省的事：

　　1.‘.com’们撒手人间。人们开始总是认为，像传统的旧经济一样，只要能坚持某种商业模式——提供某种产品或服务，只要有人愿意掏钱，则这种模式在新经济中也照样适合。但当一个又一个的互联网公司烧完了风险投资的资金和IPO(首次上市)融资后，他们再也没钱维持生计了。也没法像传统经济一样挣钱时，他们便开始自吞苦果了。别了!eve.com、boo.com、furniture.com、pets.com、mothernature.com及等等后来者。好好安息吧!
　　
　　2.音乐在网上遭遇版权法。今年以来，互联网出现的数字音乐格式逐渐成为一种严重的威胁。不仅在理论上担心。这不，五大唱片公司也慌了神。Napster开发出了免费进行音乐交换的软件，并迅速在全美国流传开来。当然，它也因此而进了法庭，唱片行业协会本来是打算把这个释放出来的魔鬼重新收到瓶里去的。并对Napster进行起诉，但这宗案子得待明年才能了结。到时可能情况已经发生了变化。尽管今年5月，法官作出了对MP3不利裁决，但是否可行，还需试目以待。现在音乐行业担心的互联网完全改变了其商业模式。

　　3.‘爱虫’病毒。5月4日，对系统管理员来说，都是一个极糟糕的一天。IS曾多年教导他们，小心不要打开自己不熟悉的电子邮件附件。但一些人还是迫不及待地打开了‘情书’，没想到这封情书如此狠毒，它不仅删除了受害者的图片文件，并隐藏了视频文件。在亚洲，它打开发挥了它的淫威。并通过微软的Outlook的，邮件地址迅速进行传播。据专家估计，这起源于菲律宾的病毒对美国一半以上的大型企业及上千万的电脑造成了伤害。但因该国没有这方面的法律规定，只好放弃对他指控。

　　4.FBI的食肉动物侵吞了人们的隐私。由美国联邦调查局开发的，别名为‘食肉动物’的电子邮件监视软件，受到美国国内提供自由和隐私的人士强烈反对。今年更是成为官方侵犯隐私的代名词。本来‘食肉动物’就是政府的监视和窃听情报用的。但FBI却称它不会侵犯隐私。但是安装在ISP商那里的这个软件不仅可对未过滤的电子邮件进行监测，还可对所有通过这些ISP的邮件进行监视。上月，一专家对‘食肉动物’的评测报告立即招徕人们对他的一阵臭骂。

　　5.拍卖UMTS成了欧洲的淘金潮流。像任何好的项目一样，任何掘到第一桶金的人的动作都很神速。但随后而来淘金的人可能就没有那么幸运了。拍卖建立在UMTS(全球移动电讯系统)基础上的3G(第三代)无线网络技术经营许可证成g国家有利可图的工具，可是在此之前谁会想到有这种结果呢？英国通过向四家拍卖优先权，结果获得320亿美元的收入，然后德国紧跟其后，拍卖许可证，也收入了448亿美元。但在荷兰，情况就变得有些不行了，它只拍卖到了24亿美元。在意大利，结局就更惨了。分析家开始考虑政府的贪婪尺度，及这些电信公司如何来弥补这些巨额投资的损失。瑞典政府因用银行贷款的方式，而不是采用交钱拿证的方式而受到人们的称赞。但政府对每一位申请用户收取1万美元的费用，而对UMTS的收入税也高达15%。公司也未赚到任何便宜。
