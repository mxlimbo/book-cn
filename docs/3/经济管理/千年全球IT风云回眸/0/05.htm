2000年“10大病毒”


　　英国的杀毒软件商Sophos PLC在本周二发表研究报告，列出了2000年最具破坏力的10种病毒。

　　尽管今年5月份出现的“爱虫”病毒给全球各地的电脑用户造成了巨大的损失，但据Sophos公司称，“爱虫”病毒在2000年的“10大病毒”排名中只能屈居第二。　　

　　2000年最具破坏性的病毒是“Kakworm”。虽然早在今年1月，人们就发现了这种病毒，但它的感染率却一直居高不下。今年的每个月，该病毒的感染率都能稳居前三甲。　　

　　根据Sophos公司的统计，2000年最具破坏力的10种病毒分别是（括号后是感染率）：Kakworm（17%），爱虫（14.5%），Apology-B（8.9%），Marker（6.5%），Pretty（5.6%），Stages-A（3.5%），Navidad（3.4%），Ska-Happy99（2.3%），WM97/Thus（2.1%），XM97/Jin（2%）。
