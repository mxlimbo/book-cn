2000年中国网络9大产品

　　产品一：中文域名

　　最厚利的产品。谁说互联网不能挣钱？谁说互联网只有电信挣钱？除非我们的网民、我们的企业上网从来不需花钱。其实，中国的网民是负担最重的，他们花的钱就是“互联网不挣钱”的最大讽刺和最大的驳论！随着网民数量激增，中国互联网已经开启了巨大的钱眼，中文域名就是2000年的最佳产品，完全是传统商业模式无与伦比的互联网收入模式。“几天内超过50万申请；近三个小时有300多万申请”......以每个域名每年200元计，就是每年数亿以上的规模。这样的题目，让小学生也能算出来。关键是谁能够垄断性地把住收钱的关口。以后，这样的商机会越来越多，否则，为什么各个部门都要争先介入互联网，留取“买路钱”。

　　产品二：ICQ

　　最矛盾的产品。爱一个人，用ICQ；害一个人，也用ICQ。尤其是孩子和学生。现在，ICQ市场如烈火一般，商家们都在为如何建立收钱模式而绞尽脑汁，而不用多久，人们就得为ICQ造成的负面效应和社会问题而焦头烂额。ICQ就是以这样独特的方式，推动着互联网的普及；也以这样的方式，创造着新的社会隐患。我们就是用这样的双刃剑，奋力开辟着中国互联网市场。

　　产品三：微软研究院

　　最善包装的产品。微软研究院当然是产品，而且是微软2000年主推产品的重中之重。反垄断官司的失利，公司核心层集体换班，市场策略无法调整，创新产品没有出台，改变形象成为首要任务。而研究院无疑是最好的题材。因此，微软最大的公关人员，不在公关公司，不在公司市场部，而在研究院。因此新闻频频，特色无限。

　　产品四：PDA

　　市场最实在的产品。互联网公司的喧嚣没有为自己带来多少喜讯，但是他们推动出来的2000多万网民，却为其他领域创造新的市场。电信公司自然是近水楼台，中文域名也是坐收渔利，广告收入模式大开，PC厂商也沾尽甜头。但最热闹的还是掌上终端的争夺。围绕PDA、商务通、掌上电脑的市场大战是2000年硬件行业的最大热点。

　　产品五：PIV

　　最失意的产品。任何一位IT人士都不会忘记那充满激情的“奔腾”时代，不会忘记英特尔是产业总导演的美好岁月。但是今天，这位导演越来越远离我们关注的舞台，PIII、PIV、还是PV，至少我已不再关心；400M还是1G，至少我已经不太清楚；威盛、AMD雷鸟还是Transmeta公司“Crusoe”，至少我已不在乎。因此，至少我们应该明白，一个互联网遭遇寒东的时刻，绝不是原有的一切回归春天的时刻。时代变了，就不会回复。作为Wintel时代的半边天，Windows 2000与PIV一起，共同享受新时代的失落。

　　产品六：Net

　　口气最大的产品。作为PC时代的垄断者和网络时代的失意者，微软抛出.NET计划，要一统网络天下？对这个网络界有史以来最大胆和最狂妄的计划，许多人都会首先笑起来。经历了反垄断公司的重大失败后，微软能给人们这样的“轻松一刻”实在是难得。.NET，是概念还是产品？还是两者兼得？反正这也是2000年整个软件唯一夺目的名词。作为制造业的软件业大有夕阳西下的架势，但作为服务业的软件业正在崛起，.NET无疑是这场重大变革的前奏，但也只是Linux模式的后缀。在18号文件的激励下，中国的软件能够如何应对变化，.NET就是一场迫在眉睫的考试。

　　产品七：WAP

　　最烧钱的产品。目前为止我们还只能通过在一定的时间、固定的地点（家中或办公室的电脑）上网，局限性极大。随时随地获取信息是一个巨大的理想。而且是必然能够实现的理想。但是，在梦想与必然之间，还要有巨大的代价。当我们的资本家把大把大把的钱扔进去，最早的产品首先首先是概念。WAP、iMode、CDMA、第三代、3G等，2000年金钱的推动下，无线领域已经是概念的丛林，但还需要艰难的跛涉。

　　产品八：杀毒软件

　　最挣钱的产品。互联网是病毒传播的天然温床。5月4日，“我爱你”病毒全球泛滥，美国国防部、中央情报局、英国国会都不能幸免。互联网商机无限，依靠病毒，就是巨大的钱眼。作为PC时代的热门产业，互联网使病毒软件越发成为一个经久不衰的产业。金山毒霸沸沸扬扬登场，国内杀毒市场重新洗牌。“个体户”时代面临互联网最强大的冲击。趋势、诺顿等国外产品也强力渗透。看来，以后每年评点，病毒折腾的事件越来越“精彩”。杀毒软件也应该是保留节目，相信会常评常新。

　　产品九：eBook

　　最轰动的产品：3月14日，恐怖小说之王Steve King在网上推出一本仅仅70多页的中篇小说。半个月之内，个人获得版税超过40多万美金，而传统出版版税只有1万左右。这一轰动事件，使得eBook（电子书或叫掌上阅读设备）概念迅速升温。夏天，博库网在国内推出eBook，紧接着方正也开始了中国的eBook新梦想。一场“阅读的革命”正在酝酿，作为全新的消费品，eBook无论对硬件业还是网络业，都有巨大的商机。当然，谁是英雄，答案在未来。
