2000年中国网络7大概念

　　当资本家和股民被市场跌麻了神经、跌碎了信心之后，再好的概念都不灵了。但是，概念是永恒的，哪怕只是冬天里的概念，比如ＡＳＰ、ＩＤＣ、Ｐ２Ｐ等等。互联网在概念的波涛中前进。

　　概念１：纳斯达克　概念之王

　　在互联网领域，再大的概念也大不过纳斯达克。再大、再新、再牛的概念，最终也都是为了获得纳斯达克的认可。同时，无论是ＩＰＯ、ＯＴＣ还是“期权”等等充满诱惑的概念，也只有等到纳斯达克本身灵验时，才能显灵。

　　概念２：中国概念股　概念中的概念

　　因为法律禁区，所有远到西方ＩＰＯ的中国互联网公司，都只能成为纳斯达克概念中的概念。到８月份，几家门户跌到６元以下，概念从此变成“垃圾”。

　　概念３：B2B　最扶不起来的概念

　　５年Ｂ２Ｃ狂热催生出４００家上市公司，形成１万多亿美元的股票价值。下一个１万亿靠什么？几乎所有大投资银行都把赌注押在了Ｂ２Ｂ身上。但逆势之下，Ｂ２Ｃ一片狼籍，但Ｂ２Ｂ也难以扶起。与投资家、媒体花费的工夫来说，Ｂ２Ｂ是２０００年最辜负大家期望的概念。

　　概念４：传统企业转型　最迷魂的概念

　　互联网公司度日如年，多少人为之痛快，但是再顽强保守的人，也不敢否定互联网本身。即使隔岸观火的传统企业，也得赶互联网的潮。于是，媒体和大众都将．Ｃｏｍ的热爱“移情”到传统企业身上，“传统转型”成为新的时尚。风险和机遇一体，传统企业将会越来越能理解互联网公司的遭遇，认清楚大家可能都是难兄难弟。

　　概念５：数码　最抢手的概念

　　公司名称添加“数码”，可摇身一变，赶上新潮；房地产项目改名“数码港”、“数码城”更是一字千金；还有“数码大厦”、“数码中心”……在多少互联网遭受唾弃，大幅贬值的大潮中，“数码”概念是为数不多能够“守住身子”的新概念。

　　概念６：ASP　最迷乱的概念

　　未来某天，我们使用的ＰＣ（包括各类上网终端）就像今天的电话一样。所有的数据处理、软件安装与升级以及相关服务，都不再需要用户自己动手，而是由一个类似“电话局”的地方统一完成。这个美好的情景不是空想，而是ＡＳＰ（应用服务供应商）的终极故事。可惜目前的ＡＳＰ，还只是挂了一个“羊头”。

　　概念７：P2P　最震撼的概念

　　Ｐ２Ｐ（ＰｅｅｒｔｏＰｅｅｒ）是２０００年互联网带来的唯一震动整个世界的创新。Ｎａｐｓｔｅｒ公司的软件可以使用户不通过服务器，就能彼此交换硬盘中的音乐文件，避开了传统版权的法律限制，使得“盗版”合法化了，美国人真是“盗版有术”。当然，一切还只是开端。
