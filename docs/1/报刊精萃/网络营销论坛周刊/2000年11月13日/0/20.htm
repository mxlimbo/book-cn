超市趟后感：现状一角

swellstudio/文

　　刚从超市购物回来，想了些问题，过程：

　　这家超市离我出发点300多米，此次购物花了半个小时，目的是为解决晚饭与夜宵，买了两包方便面，一包青豆，一包花生排和一条高尔牌香烟，花费28元。

　　所需时间 + 商品费用 = 消费者为一次购物付出的代价

　　补充：还应包括以下选择项 - - 运输费用（坐车、打的等）、体力消耗、外出风险（遇到交通事故、打劫等）

　　消耗 / 商品费用 = 消耗比率

　　显然，消费者想在一次购物中达到最低消耗比率，两个办法：

　　一、 一次购买较多的商品，尽管会增加体力消耗、运输费用，但消耗比率总能够达到最低；

　　二、 直接从消耗处解决问题，想办法降低运输费用、缩短购物时间等。

　　假设我们在一次购物的过程中没有其它的获得（如在购物时获得快感等），只有花钱买回来的商品。那么传统的方式明显的存在时间、体力消耗太大的问题，都市人生活节奏快，办事都讲究个效率，开口闭口时间就是金钱，老年人出门不方便，动一动都会累着…… 为此，也出现了一些速送店之类的电话预定服务，此为后话。

　　几番周折的商品：

　　拿方便面举例，由生产厂家生产、加工、包装，分销商订货送到超市、商店，生产商为了拥有更多的分销商，会打广告、搞促销活动，通过宣传自己的品牌、保证方便面的口味和抵抗其它竞争对手来赢得市场，处在二级的分销商也会积极推销自己库存的商品，使它们买的更好。一包方便面从出厂到仓库到超市，然后被许多消耗者挑来挑去，换了几个摊位，终于被我买了下来泡了吃了。

　　成本问题：

　　厂家和商家把大把大把的钱花在了广告、门面上了，广告要做的好，楼要盖的高，门面要装修的好，还要雇人来卖……这些钱少不了，而且也会增加商品的价格，他们也为降低成本伤脑筋呢。

　　前阵子，人们从电子商务看到了希望：

　　理想的情况下，消费者只需花很短的时间在网上挑选所需的商品，将订单发送到厂家，再由厂家通过某种方式送到你的手中，使消费者只需为商品付费。按理说这时商品的价格应该比传统的低，但实际上要高。目前电子商务网站为了争取市场份额，打广告烧钱烧的厉害，有些还要为消费者付送货费，商品的成本自然上升，再加上国内网上消费需求很低，BtoC的电子商务公司死了一堆。结果大家都开始转形，从网上商城到做单一商品的垂直性网站，也有的提出了连锁经营的模式，而模式是有基础的，不能乱套，失败的例子很多。

　　希望很快随着网络神话的破灭而破灭。

　　………………现在我在一边吃面，一边思考small business的问题，一边回答网友的D3D问题，忙呀…………大家也一起来加入这个讨论吧，集思广益，我喜欢讨论。

　　1、 做电子商务现在比较适合做什么商品？

　　2、 网上购物还能带给我们什么？

　　3、 网络营销是传统营销的变向还是延伸？

　　4、 消费心理对购物的影响。

　　
主题：回答问题

老李背着手/文

　　1、 做电子商务现在比较适合做什么商品？

　　从具体商品上很难说清楚。但从商业角度，卖这样几种东西是可以赚到钱的。

　　第一是主控渠道的产品，在电子网络环境下，主要克服的困难就是如何通过电脑提高管理和运营效率。

　　第二，是非服务增值类产品（可能定义的不准确），比如文化和品位属性较强的东西例如一种名字很长的法国香水儿，又比如索尼walkman的电池盒儿的盖子。这些东西只须要相对较少的网下人力成本，而且一般不容易买到。要注意的是，商品的精选和对市场及人文的揣磨，在涉及到对外贸问题时应考虑外汇操作等等。

　　第三，在传统行业利润较高，但流通性不是很大的产品。比如珠宝。一般的业主承受压款的能力有限，而珠宝的选择又是很有个性的事，通常在几百种样式之间都挑不出满意的。就运营上来说，运营珠宝的钱可能一年才能周转几次，大量的成本损耗在了店面和库存上。目前成熟运作此类站点的还没有（个人网站不算）。要害是数立良好的形象和较强的专业知识。

　　以上是单就产品而说的一些参考。

　　2、 网上购物还能带给我们什么？

　　说实话，网上购物带不来啥。还是带给用户产品，老板收钱发工资养活自个儿和一帮手下。不过前一段时间，网上购物的概念给我们当了一次电子网络的普及老师。不仅把观念在一定泛围内灌输给受众，还筛出了一些专业的网络人才，这批人将是第二批吃旁蟹的人。

　　3、 网络营销是传统营销的变向还是延伸？

　　无论怎么定义，营销就是营销。只是采用了不同的生产工具。在新的生产工具的参与之下，对传统带来改革，同时也触发了或者将触发一些新的营销或者服务的旦生。

　　4、 消费心理对购物的影响。

　　这个话题可以写本儿书了。这儿说不清楚。
