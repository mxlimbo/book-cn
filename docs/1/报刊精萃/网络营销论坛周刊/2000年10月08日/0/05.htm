营造虚拟世界的忠诚

sundy/文

　　每当听到有人夸耀他们的网站当月又有多少的点击数时，别提我有多恼火了。这不但因为点击数并不能准确衡量访问量，而且从长远来看，对于电子商务而言，数量远不如忠诚那么重要。是的，全世界的人都可能点击访问你的网站……但是就一次。

　　营造忠诚

　　网络忠诚意味着你让用户有足够的理由回访你的网站。这种忠诚首先要从信任开始，访问者根据他们自己的标准做衡量，看你是否准时发布信息，看整个过程是否有如传统方式那么便捷而且有保障。

　　一旦你得到这种信任，你就赢得了开拓市场的无限商机。在网上，你可以同客户建立一对一的个性化联系，甚至可以得到他们的允许直接开展推销！

　　网上忠诚经由以下方法得到：

　　　网站本身的内容
　　　为用户度身定制的服务
　　　为吸引访问量和回访率而特制的系列计划

　　一、完善网站

　　花点时间对网站做些检查和批评。找一个对你的网站完全陌生的人，让他看看是否有趣味有吸引力？看看通过站点是否能很快了解你是谁、你做什么？或者是否有太多生涩冷僻的专业用词把人吓跑？

　　友好的界面，便捷的导览系统对吸引回访率大有裨益；你还得有丰富的内容，什么信息也没有，光是罗列一张推销清单，没什么比这种网站更糟糕了。另外也要尽量加强网站与访问者之间的交流。

　　有了这样一个内容丰富、交互功能强劲的网站，访问者就可能会接受你那些有针对性的推销行为。竭尽一切可能使得访问者更方便留下其E-MAIL地址，或者使其更多的了解你。我不理解为什么很多站点上有关公司的信息那么模糊，好象一边希望我买他们的东西一边又要对我隐藏什么……不要这样！

　　同时，确保你的网站有明确的隐私政策，任何时候，一旦你开始询问某人的个人信息，实际上你就开始涉及隐私。如果你在草拟隐私政策上需要帮助，请参照网上资料TRUST和DMA.

　　二、度身定做

　　访问者告诉你个人信息，绝不表示你就能给他们疯狂发送各种E-MAIL。首先得征求同意！这是一切行动的前提，对建立维护用户忠诚度至关重要。最好还能告诉他们为什么要这样做，其实这就是传统的“对我有什么意义”法则。

　　那么究竟对他有什么意义呢？你当然可以沿用老例，提供某些折扣、免费服务等，但我建议你作进一步研究，看看用户到底真正最需要什么，看看怎么样可以给你的用户带来更多的方便或者趣味。

　　无论在网上或者通过E-MAIL，应该尽量使信息的发送更加个性化。例如CDNOW.COM这家网站，提供“我的CDNOW”栏目，籍此根据每一位注册用户的情况在网上和E-MAIL中提供个性化的服务内容。用户通过标明喜欢什么样的音乐，建立自己的“我的CDNOW”档案；而网站就籍此收集建立起庞大的用户数据库！

　　三、帮助建立“网上忠诚”的计划

如果你不愿意自己来建立这份忠诚，你可以找专家，市场上有很多这类服务，专门帮助吸引“回头客”。

　　例如，使用Mypoints.com、Rewards.com或者Freeride提供的服务，用户只要做点简单的事比如阅读E-MAIL、浏览一个站点等就可以赢得积分。这些积分可以兑换小商品或者获得免费旅行。类似的还有Cybergold.com，但它可以直接兑换现金而不是积分， E-centives则是发放个人储蓄奖券。

　　使用这些站点的服务有助于吸引访问量，留住访问者。但请记住，访问者的忠诚度最终是由你所提供的服务而决定的。可以这样认为：激发访问量 + 赢得信任 + 价值&服务 = 无限忠诚的网上用户

　　多说一点：“请”, “谢谢”, “对不起”, 那么你就对了！
