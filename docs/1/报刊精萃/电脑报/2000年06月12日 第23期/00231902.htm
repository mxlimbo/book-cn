《 追新一族 》

　　上周的软件更新信息可以说是激动人心的，不但很多老牌的、为大家所熟知的软件推出了功能更强大的新版本，同时飞雪也发现了不少实用的新软件，且听我一一道来：
　　 新近的ICQ 2000a Beta v4.31 Build ＃3143（http://www.icq.com）改善了网络连接，并且增加了防火墙的设置，让用户感觉使用更加顺畅。而AOL的AIM（AOL Instant Messenger） for Windows 4.0.2000（http://www.aol.com）也新增加了当离开状态的伙伴返回时予以通知和自动升级控制等功能。另外，AIM的一个新附件AOL IM Speech 2000（http://www.the－hitman.com/wtm）利用TTS引擎（Text－To－Speech）可以为你即时读出当前信息，“公用略缩语筛选”还能解读BTW（By the way）之类的网络常用语，是一款不错的软件，至少能让你在和朋友用英文联系时锻炼一下听力：）
　　大名鼎鼎的nullsoft也有了新动作，WinAmp 2.63（http://www.winamp.com）在有了更好的多语言支持、更加出色的音频流控制、WMA 编码器转码WAV/MP3/MOD、充实了内置搜索引擎minibrowser的内容等等，正如它的主页所说的：“yet another swell version of the world's greatest audio player has been let loose out of the corral（世界上最棒的音频播放器的另一个增强版已经出笼）！”飞雪不知道它是不是最好的，不过，它确实以出色的MP3播放效果和众多的支持插件俘获了一大批铁杆用户，免费发放更让一些老牌厂商视之为洪水猛兽。
　　WinGate 4.0 RC2（http://wingate. deerfield.com）的发布给通过局域网上网的用户带来了好消息，NAT（Network Address Translation）技术让用户可以在任何平台（包括Windows、Macintosh、Unix和Linux等）上共享Internet连接、允许无线和以太网连接、扩展了防火墙技术以及一系列的网络访问控制新技术，都会让你感到局域网上网更轻松了。
　　在这个“网络为王”的时代，当越来越多的软件加强与浏览器之间的结合的时候，WinZip也终于推出了一个相关附件——WinZip Internet Browser Support Add－On Build 3124（http://www.winzip.com），它能在利用浏览器下载压缩包资源后进行接管，自动转到目录并打开。
　　飞雪在这里还向大家推荐一款非常不错的CD播放器——Quintessential CD 2.1.017（http://www.quinnware.com/qcd），小巧而功能强大，新的Internet CDDB2控制（与前一版本相比，这是唯一的改进，据说是为即将推出的2.2版本更好使用这个功能作测试）。
　　其它的国外软件更新还有著名的专业MIDI和数字音频编辑器CakeWalk Pro Audio 9.03 Update For Win9x（ftp://www.cakewalk.com/pub/Patches/Cakewalk/9.03/Pro_Audio）、优秀的E－mail软件Eudora Pro 4.3.2.7（http://www.eudora.com）、功能强悍的、已经支持GeForce2的显卡/显示器配置工具PowerStrip（http://www.entechtaiwan.com）和RealPlayer 8.0 Basic Beta2（http://player.broadcast.com）等。
　　国产软件方面也有不少好消息，《超级兔子魔法设置》Magic Set v3.3（http://superr.yeah.net）正式版已经发布，新功能有黑客木马程序扫描和清除、网络地址IP转换和有关DirectX的设置，本栏目第21期提到的注册表减肥功能据称已经完成，不过鉴于安全性方面的考虑，暂时仍然没有加入。另外，开发小组已正式决定将Magic Set的版权、源程序及所有的东西进行转让，555，飞雪在这里只能对他们说“祝你们成功”了！
　　方便你管理和编辑电子文档的《良友收藏家 3.44》（http://ourbooks.csw.cnshare.net）改进了存盘功能，减少死机后丢失数据的可能；校正窗口大小，对话框中不再有滚动条；标题栏改为单一的当前记录的标题。
　　曾经推出过《海天一色聊天室》的海天公司又有了新作，《海天一色网络游戏系统》 1.0标准版(http://www.haitian.com.cn)采用了JAVA技术，目前包括连珠、升级两个游戏，具有游戏、聊天等多种功能。
　　东日软件将《东日－通讯录》 6.0（http://sunisoft.126.com）作为正式作品发布了，这是一款拥有强大查找和导出功能的个人通讯录管理软件，新版调整了软件界面、增加列表显示、加强信息容量和修正了前版中“导入”功能可能导致泄密的BUG。
　　另外，五月初推出的新兴下载软件《中国网爪》（http://chinaclaw.yeah.net）现在有了测试b版，其多地址下载功能倒是让人感觉挺新鲜，不过飞雪怀疑它存在的必要性——显然还不如加强镜像站点搜索和管理功能，不过从它一系列的功能改进和增强来看，这个软件也正在逐渐走向成熟。
　　看到以上这么多功能强大而实用的软件，飞雪不禁想起一句著名的广告词——“科技让你更轻松！”
