《 文件分割工具集锦 》

　　1.1toX 1.04
　　性质：共享软件，327K
　　下载：http:∥perso.infonie.fr/chass
　　功能简介：1toX文件分割工具支持CRC校验、密码保护、拖放等，但是它的工作界面采用的不是英文，而是某一种欧洲语言，不过有明显的工具图标按钮，并不影响它的使用。单击界面第一个按钮可打开选择文件窗口，通过其中的选择框找到要分割的文件后双击，出现设置窗口。在其中的“Cible”框右端输入存放分割后文件的文件夹，在“Type de 1toX”(分割的类型)中复选“Batch”项，使得在分割文件的同时自动生成一个合并文件的批文件，单击“确定”按钮，即可完成分割工作。下次合成文件时只需要在存放分割后文件的文件夹运行合并文件的批文件即可自动完成。1toX功能较单一，只能够作分割、合并文件的最基本操作而不具有其他功能。
　　2.Arclab Packager MK1 2.1
　　性质：共享软件，977K
　　下载：http：∥arclab.com/products/packager/download/apack.zip
　　功能简介：Arclab Packager Mk1的图形操作界面非常的近人，它简洁大方的界面给笔者很深的印象。运行后出现选择界面，其中的两个按钮非常直观地显示出相应的功能。要分割文件，单击窗口上部的按钮；要合并文件，单击窗口下部的按钮。分割文件时，从“Source”区的驱动器按钮选择要分割文件所在的驱动器，在“Target”区选择分割后的文件存放的文件夹，在“Disk format”区选择生成的小文件块类型，有1.44M、2.88M、1.2M、ZIP100、自定义等几种，单击“确定”按钮即可完成分割。合并文件时同样要先选择小文件块所在的文件夹和存放合并后文件的目录，单击“确定”按钮即可完成合并文件。Arclab Packager MK1操作简单，可以直接将大文件分割到软盘上。但是在分割大文件的时候，不能创建合并用的批文件，因此目标计算机中必须也安装有Arclab Packager MK1才能进行合并文件。
　　3.Splitty、l．7
　　性质：共享软件，285K
　　下载网址：http:∥www.basta.com/Software
　　功能简介：Splitty具有数据校验功能，在“File to split”框设置要分割文件的路径名。在“SPlit files destination folder”框设置存放文件块的文件夹，通过“Split files maximum bytes”框设置文件块的大小。在“Split files name prefix”框设置分割成的文件块名称前缀。还可设置“Prompt before overwriting”（覆盖之前提醒）、“Operate in the same folder”（在同一文件夹中操作）、“Generate split files name”（生成分割文件名）等。单击“Apply”按钮即可开始分割操作。要合并块文件时，打开“Splitty”的工作窗口，选择“Merge”标签页，在“Mastesplit file（.fsm）”框设置块文件的控制文件路径名，这个后缀名为fsm的文件时程序在分割时自动生成。在“Original file destination folder”框设置存放合并成的大文件的目标文件夹。单击“Apply”按钮开始合成过程。也可以单击合并功能页中的“Verify files”（验证文件块）按钮对分割结果加以验证。Splitty的界面简洁，具有数据校验功能但是不能生成合并用的批文件。
　　4.File Splitter l．0
　　性质：共享软件，182K
　　下载:http：∥www．download．com／splitter．zip
　　功能简介：可以说File Splitter是针对低版本Windows操作系统设计的，可以运行在Windows 3.X上，而且安装、操作界面也是典型的Windows 3.X格式。不过它能将大文件分割为多达99个小文件块，并能将这些文件块合并为大文件。保存文件块的时候，程序能智能分析磁盘空间，加以提醒。分割的过程中能够产生合并批文件。File Splitter较简洁和实用，其操作过程相当简单，但是不能将大文件直接分割到软盘上，而且分割速度也较慢。
　　5.Part It 2.01
　　性质：共享软件，327K
　　下载：http:∥www.skylarkutilities.com
　　功能简介：Part It在分割大文件的同时，能够创建一个合并可执行文件，可以运行这个文件在没有安装Part It的计算机上将文件块再还原成源文件。Part It能够显示所使用的软盘状态信息，包括驱动器是否准备好，驱动器中软盘的卷标、序列号、容量等。Ppart It程序的操作界面十分豪华，设置项目十分丰富，在细节上考虑较多，但是缺乏数据验证功能，同时无法向硬盘上分割文件。
　　6.文件切割机Cut＆Copy Tool 1.33
　　大小：共享软件，208K
　　下载：http:∥www.netease.com/～laoyang
　　功能简介：Cut＆Copy Tool是一款国产的文件分割工具，在操作上支持拖放功能、切割和合并的文件的长度是没有限制的，几百G都可以，截取及切割实现可视化操作，真正环保型软件，无须安装、直接运行、支持鼠标右键，在文件浏览器中按鼠标右键即可运行文件切割机。此外可从大文件中截取出一小段来，例如截取VCD片断的功能是其一大特色。运行CutnCopy.exe，后进入主画面，将要切割的软件直接拖到被切割文件名对话框中。在文件长度条右边的框，输入切割后的文件长度，单位为KB，在显示小文件尺寸大小的框中，按Mouse右键可弹出设置软盘参数。进行切割时，按下MOUSE右键，弹出菜单，点中“Go进行切割”，将执行切割功能。如果要执行截取功能，选择“显示截取文件界面”，会出现两个文件长度状态条，第一个状态条是选择起始位置，第二个状态条是选择结束位置。按右键弹出菜单，点中“Go执行截取”，将执行截取功能。对于合并文件操作，由于切割的文件是无损切割的，可用任何拷贝工具包括DOS的Copy命令将其合并。此外提供了批合并功能，只需将要合并的文件，用鼠标拖到文件框中即可，在“按文件名排序”选择文件自动排序，或按加入的文件先后顺序排列。点中“执行文件合并”即可合并文件了，按“取消文件合并”取消合并操作。
　　7.文件分割机KNIFE 2.0 BETA 1
　　性质：免费软件，38K
　　下载：http:∥www.newhua.com/hj_china.htm
　　功能简介：这个国产文件分割机的特点就是文件非常的小，不过也能完成基本的文件分割合并操作。它是运行在DOS状态下的。运行时在“X:\KNIFE>”提示符后输入：“KNIFE <回车>”，按屏幕提示输入要分割的文件路径和文件名然后回车；输入分割后文件的字节数然后回车，即可把文件分割成几个。文件恢复时把KNIFE所带的RST.COM copy到要合并的文件块目录中，然后键入“RST <源文件名> <回车>”即可。
