《 阿飙专线 ---FIRE WORKS直通车(五) 》

　　通过Import命令FireWorks能导入哪些格式的文件？
　　FireWorks可以导入Macintosh的标准图形格式（PICT）、Freehand 7.0或者8.0的标准图形格式（FH8、FH8）、Illustrator的图形格式（AI7、AI8）、PhotoShop的各个版本的图形格式（PSD）、CoreDraw 8的图形格式(CDR)、FireWorks的批处理脚本（JPS）、Windows和Macintosh版本的PNG文件（PNG）以及GIF、 JPEG、 BMP、 TIFF、TGA, LRG, ASCII格式的文字, RTF格式的文字。
　　通过Export命令，FireWorks能输出那些类型的文件？
　　FireWorks可以输出Macintosh的图形格式(PICT)、GIF、JPEG、TIFF、BMP、PNG、Domino Designer Image Wells (Windows 和 Macintosh版本)、Illustrator的图形格式、HTML文件包括支持 JavaScript rollovers、Image Maps、Dreamweaver 的Behaviors、Dreamweaver Libraries 和样式层叠表单CSS文件以及FLASH的SWF动画文件。
　　FireWorks支持数码相机和扫描仪吗？
　　FireWorks支持工业标准Twain，只要利用File/Scan/Twain acquire命令就能够从数码相机或者扫描仪中方便地获取数字文件。
　　FireWorks的标准文件格式是什么，为什么使用它作为标准文件格式？
　　PNG是FireWorks的标准文件格式。PNG图形格式是W3C推荐使用的一种可扩展的Web图形格式，PNG文件格式采用无损压缩技术，并且支持索引、256级灰度、真彩色（24 bit－RGB）并带有一个可选的8 bit Alpha通道。PNG文件还能够存储gamma和chroma数据，能够对图形色彩进行改进以适应不同的平台。
　　是不是所有的浏览器都支持PNG文件？
　　虽然PNG文件格式是W3C推荐的Web图形格式，但是由于这种文件格式太新了，还未被大量的软件支持，因此并不是所有的浏览器都支持PNG文件格式。不过，最新版本的IE、Netscape、Microsoft Office、FreeHand和PhotoShop都能够读取该文件格式。如果你希望设计的网页能够在老版本的浏览器上正常浏览，建议使用功能强大的Export命令将图形输出为GIF或者JPG格式。
　　FireWorks能和那些软件协调工作？
　　FireWorks是DreamWaver和其他HTML编辑软件的最佳搭档。当用户在FireWorks中制作Slice images和 创建 JavaScript Rollovers时所形成的代码能够直接在DreamWaver的Behaviors中使用。在DreamWaver中用户可以直接调用FireWorks对图形进行编辑处理。FireWorks和FreeHand也能协调工作，在FreHand 8 中用户可以随意编辑并存储PNG文件格式的图形。
　　在FireWorks中能够直接打印图形文件吗？
　　FireWorks是个能够对Web和屏幕图形进行优化处理的图形设计工具。用户可以在FireWorks中直接对图像进行打印处理，也可以将FireWorks的文件（PNG或者TIF）导入FreeHand 8中进行打印处理。
　　我可以将带有图层的PhotoShop文件导入FireWorks中进行处理吗？
　　FireWorks在导入PhotoShop的PSD文件时能够完美无缺地保留住图层。不仅如此，FireWorks也能够保持Illustrator和FreeHand的图层或页，并将其转换为Frame。
　　我能够在FireWorks中使用第三方的Plug－ins(插件)吗？
　　FireWorks支持许多第三方的Plug－ins (插件)，例如：Alien Skin’s Eye Candy，而且令人惊喜的是，在FireWork中自带了一个Alien Skin’s Eye Candy3.1插件。
　　在FireWorks中如何设置临时文件目录？
　　FireWorks的临时文件目录设置和PhotoShop极为相似。在File菜单中选中Preference命令，在Preference设置对话框中选择Folders标签，在Scratch属性窗口中的Primary和Secondary选择框中选择主磁盘和备用磁盘驱动器，FireWorks会将它们作为临时文件存放处。
　　FireWorks中如何设置Plug－ins(插件)的目录？
　　FireWorks的Plug－ins文件目录设置和PhotoShop极为相似。在File菜单中选中Preference命令，在Preference设置对话框中选择Folders标签，在其中的Additional Material属性窗口中，用鼠标单击PhotoShop Plug－ins旁的Brower按钮，在弹出的目录选择对话框中选择Plug－ins所在地目录，单击Ok按钮确定选择即可。
　　在Windows 95中安装完FireWorks后，启动FireWorks，在软件初始化过程中，在没有任何提示的情况下，FireWorks异常退出，请问这是为什么？
　　这是因为你的Wiindows的System子目录下有一个老版本的Comclt32.dll文件在作怪。这个老版本的dll文件是Windows自带的，如果安装了IE 4.0或者更高的版本后，系统会自动更新这个文件。你也可以到微软的站点上下载一个名为Com32upd.exe的文件，运行它即可解决这个问题。
　　我的计算机上有256M内存，安装有Windows 98。可FireWorks在初始化过程中,系统为什么会突然崩溃呢？
　　这是因为某些版本的FireWorks无法访问128M以上的内存地址空间，因此在256M的计算机上运行FireWorks可能会出错。此时应做如下处理：
　　1.用鼠标单击“开始”按钮
　　2.单击“程序/附件/系统工具”菜单
　　3.选择“系统信息”工具
　　4.在“系统信息”面板上单击“工具”菜单，选择“系统配置工具”
　　5.在“常规”窗口中单击“高级”按钮。
　　6.在“限制内存到”对话框中，设置内存为“128M”
　　7.在窗口中单击“确定”按钮，然后重新启动Windows 98。
　　为什么在运行FireWorks时，无法显示常用工具栏，而且有时工具栏上的工具按钮会消失？
　　而且这是因为在FireWorks目录下的FireWork.ini文件出现了错误。应删除FireWork.ini这个文件，然后重新启动FireWorks。
　　为什么在Windows 98的资源管理器窗口中双击FireWorks生成的PNG文件，会引起资源管理器出错？
　　这是因为在默认状态下，在资源管理器中打开PNG文件时，系统会以View/As Web Page方式打开PNG文件。而在IE中，系统中只预留了4K的缓存来读取PNG文件，由于PNG文件包含了许多数据库信息，但这些信息超过了缓存的容量时，便会引起出错。安装IE 5.0或者更高的版本即可解决这个问题，用户也可以安装ACDSee等其他图像浏览软件来取代IE作为PNG文件的默认浏览软件。
