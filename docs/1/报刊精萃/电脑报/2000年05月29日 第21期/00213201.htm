《 品牌机升级 BIOS 也超频 》

　　超频是许多电脑爱好者梦寐以求的事情，超频成功能给玩家带来心理上极大的愉悦和满足，那种心跳的感觉真是难以形容。但对于品牌机用户来说，DIY（特别是超频)最大的障碍是缺少资料。由于各品牌机的主板都是自己生产或是OEM定制的，无任何超频资料，只有自己想办法。下面我把超频的经历写出来，供拥有联想品牌机的用户在超频时作一参考。
　　一、品牌机配置情况
 　　联想1＋1系列天琴959电脑，PⅡ233、32M SDRAM、Seagate3.2G、丽台S680显卡、Legend－V主板。升级前外频只有66MHz，倍频支持2.5×～5.5×。
　　二、扩充内存，提高系统性能
 　　在Win98多任务环境下，系统内存容量对系统性能有着重大影响。虽然32M基本能应付，但当窗口开得较多时，仍需要更多内存支持系统高速运转，否则会由于频繁的磁盘交换而降低效率。天琴959主板上有三个168线内存插槽，原机配有两条LGS 16M SDRAM，另一个空着，于是我另购置一条64M SDRAM插上，系统内存达到96M，目前的应用程序足以应付自如了。
　　三、 升级主板BIOS软超频
 　　在我升级主板BIOS之前，曾进入BIOS设置修改倍频3.5为4，使内频达到66.6MHz×4=266MHz，超频成功（万一不成功，可清除COMS重新开机）。但是由于外频没有改变，系统性能提高并不明显。于是就想升级主板BIOS看是否能增加外频。说干就干，马上打开机箱仔细观察，发现主板左下角有LEGEND－V的标记，后又用ctbios.exe应用程序测得BIOS信息为“QDI，LEGEND－V BIOS V1.1SL 11/24/97，Award Modular BIOS v4.51PG，i440LX－2A69JQ1DC－00”，主板制造商的主页为http://www.qdigrp.com。我马上驱“猫”上网，发现已有最新BIOS升级程序“L5_V20SL.BIN 01/05/99”，立刻下载，并在该网页同时下载BIOS刷新程序AWDFLASH.EXE。首先制作一张系统启动盘并拷入BIOS升级程序L5_V20SL.BIN和BIOS刷新程序AWDFLASH.EXE。进入BIOS 设 置，记下先前改动过的任何设置，然后在LOAD SETUP DEFAULT 这项中恢复。在 BIOS特性设置中，设置“Boot Sequence”为“A，C，SCSI”，并在CHIPSET FEATURE SETUP 中将“System BIOS Cacheable”设为Disable，并关闭内部和外部Cache，保存并退出BIOS设置 。让系统由启动盘启动，然后在DOS状态 下键入以下命令 : 　A:\>AWDFLASH L5_V20SL.BIN ( 回 车 )
　　然而随后却出现“Insufficient Memory”的错误信息，后又根据《电脑报》1999年合订本下册第 259页文章“升级主板BIOS时‘内存不足’处理一例”的办法，结果还是出现 “Insufficient Memory"！ 几天后，当我再次按照LEGEND QDI主页上的说明书的步骤，再次刷新LEGEND－V主板BIOS。当输入软盘中正确的BIOS文件名L5_V20SL.BIN时会出现“Insufficient Memory"的错误信息，而当无意中输入与软盘中不符的BIOS文件名L5_V20HL.BIN时，却出现了正确的“Do you want to save BIOS?(Y/N)"信息，按Y，输入文件名L5_V11SL.BIN保存原BIOS，然后就会出现“Source file not found!"。升级还是不能成功。既然原来的BIOS文件保存下来了，于是就根据《电脑报1999年合订本》下册第115页制作了BIOS紧急恢复盘，以防万一，毕竟我是第一次单独升级我的联想品牌机主板BIOS。
　　我为这次主板BIOS升级准备了2周，可是没有成功，心里有些不甘心。接下来几天我给好几个网站发E－mail，包括联想公司，联想回复如下：“建议您最好到当地的联想维修站修复您的BIOS，联想不建议用户自行升级或刷新计算机的BIOS。”我在浙江的西南山区龙泉，当地没有联想维修站，最近也要到一百公里外的丽水去升级。我想是不是有别的办法呢？如此日思夜想了好几天，又翻书又上网，偶然看到QDIFLASH.EXE，我想是不是QDI主板升级BIOS就认QDIFLASH刷新程序呢？看好了就干，马上下载此程序替代系统启动盘中的AWDFLASH。插入软盘，重新启动，在DOS状 态下键入以下命令:A:\>QDIFLASH L5_V20SL.BIN( 回 车 )
　　屏幕显示“Do you want to save BIOS?(Y/N)”。提示的问题是你是否需要把原来的BIOS从系统中存入软盘中，我选择“N”，因为原来的BIOS文件在上次已经拷入 盘 中。然后出现“Are you sure to program?(Y/N)”信息，按“Y。开始刷新。经过揪心难熬的5秒钟后，出现“Flash write ok!”，“Please Power off or Reset System！”按“RESET”键，重新启动计算机进入系统的BIOS设置程序，用LOAD SETUP DEFAULTS（加载默认的设置内容）来用新的BIOS将CMOS内容升级。再进入“CPU SPEED SETTING”项……哇！外频选项从66MHz至83MHz都有了，而且倍频也多了。根据先外频后倍频的原则，马上选取最高的83MHz外频，倍频选取3.5，经过几次试验，最后定在75MHz×4=300MHz。
　　系统历经我的“严刑拷打”，使用至今运行正常。从我的经验看，尽管是品牌机，只要认真研究配置，特别是主板情况，做到胆大心细，是完全可以通过超频来提高系统性能的。我的联想内频从233MHz到300MHz，感到运行速度明显快了许多。不超白不超，超了没白超！这些天总算没有白忙活。欢迎大家来信指导与交流，我的E－mail：625@371.net

（浙江　曹隆枢）
