《 秘技锦囊 》

　　法外行
　　游戏中输入以下密码：
　　OLASH = 无限弹药
　　OLAIRHEAD = 飞行模式（用X和C键控制上下）
　　OLCDS = 超级地图模式
　　OLREDLITE = 定住敌人
　　OLJACKPOT = 得到所有物品
　　OLGPS = 显示你的坐标（当你按Tab时出现在屏幕的左上角）
　　OLBOUNCE = 超级跳跃模式 开/关
　　OLWIMPY = 自动装弹
　　OLTOMBSTONE = 自杀
　　OLGUSHER = 无限油
　　OLIMYELLA = 无敌
　　OLER = 加满生命值
　　OLHIDEOUT = 跳关
　　OLTOWN = 跳关
　　OLTRAIN = 跳关
　　OLCANYON = 跳关
　　OLMILL = 跳关
　　OLSIMMS = 跳关
　　OLMINER = 跳关
　　OLCLIFF = 跳关
　　OLRANCH = 跳关
　　OLEATHER = 隐形
　　OLSTINNETT = Gatling Gun
　　OLSHAW = 武器威力加大
　　OLSHADOWS = 喷气包（按A升起）
　　OLZIP = 传送到某个角落
