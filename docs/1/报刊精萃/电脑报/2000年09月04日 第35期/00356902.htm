《 {《老Voodoo也能玩新游戏》补遗} 》

{ 看了Chan Yung在《电脑报》发表的文章《老Voodoo也能玩新游戏》之后，立即拿出MDK2，QUAKE3等游戏进行实战，没想到的确有用，不过游戏速度很慢，一点也不过瘾。现在有一个很简单的方法，且听我说来：只要把下面E文COPY到autoexec.bat中就OK了，第一行是调芯片频率，最好别超过“60”。
　　set sst_grxclk=58
　　set sst_fastmem=1
　　set sst_fastpcird=1
　　set sst_pcird=1
　　set sst_video_24bpp=1
　　set fx_glide_swapinterval=0
　　set sst_screenrefresh=70
　　set sst_swap_en_wait_on_vsync=0
　　试一试，是不是快多了，随便说一句http：//cbleem.126.com这有绝版的Voodoo驱动FOR WIN98，是仅有的最新驱动了。
(修罗王一平)}
