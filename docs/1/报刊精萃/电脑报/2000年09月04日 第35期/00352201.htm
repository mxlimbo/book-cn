《 {滤镜之旅(六)} 》

{ 这次给大家介绍的是Kpt6，大家可到http://heaven.netwalk.net.cn/download/program/graphic.html去下载（它可是个庞然大物呀）。这一套系列包含了10个滤镜。我们这次重点介绍下面的几个滤镜。
　　 KPT Equalizer 可以用来对图像进行调节，制作梦幻的效果。
　　 KPT LensFlare 创造反射、光晕和透镜反射等辉光效果。
　　 KPT Goo 一个很强的涂抹工具。可以将图像像流质一样进行涂抹。
　　 KPT SkyEffects 自然天空生成器，制作逼真的天空效果。
　　 KPT SceneBuilder 在Photoshop中插入三维模型，并渲染着色。
　　安装了滤镜之后，可以在Photoshop的菜单Filter→Kpt6找到滤镜。为了能够在有限的篇幅中讲清这5个滤镜。我们将滤镜组合起来制作效果。
　　至本期为止，“阿飙专线——Photoshop直通车之滤镜之旅”系列就结束了，欢迎大家继续向本栏目投稿。

一、江山如此多娇
　　原本比较平淡的群山，加上均衡调节和辉光效果之后，变得具有一定的梦幻色彩，效果如图1所示。这个作品运用了KPT Equalizer制作梦幻效果和KPT LensFlare制作辉光效果。
　　1. 首先打开一幅山景图片，这张图片是Photoshop 5自带的一张图片。
　　2. 选择菜单Filter→Kpt6→KPT Equalizer。
　　3. 在界面中将参数调整至如图2所示。
　　在本实例中，将均衡器的256像素滑杆调整到－47％，128像素滑杆调整到78％,64像素滑杆调整到－47％，32像素滑杆调整到78％。预览区的群山已经变成那种梦幻效果了，按下打钩按钮。
　　4. 下面要给群山加上一道辉光。选择Filter→Kpt6→KPT LensFlare。
操作界面窗中，在预览区中拖拽红色辉光到相应的位置，按下打钩按钮。如果对于该辉光不满意，可以单击预设参数。在预设效果中选择喜欢的辉光，按下打钩按钮，再进行调整。
　　整个画面就制作完毕了。

二、日出
　　细心的读者会发现，这里没有给出原图。其实，这张图片根本没有原图，全部是利用Kpt6做出来的，如图3所示。背景的日出图是用KPT SkyEffects 滤镜制作的“苹果”则是使用KPT SceneBuilder 滤镜调用三维苹果模型制作的。苹果的手臂和融化的下部是使用KPT Goo滤镜涂抹而成的，酷吧。
　　下面我们就来实战这个图片的制作过程。
　　1. Photoshop中选择菜单 File → New新建一个空白文件。大小为宽度（width）640像素，高度(height)480像素。
　　2. 选择菜单Filter→Kpt6→KPT SkyEffects，系统弹出滤镜界面，如图4所示。
　　3. 单击预设参数按钮。在左边天气选择中选择Sunrise（日出），然后选择一张Autumn（秋日图）。单击OK。回到主界面，再单击OK即可。
　　这里我们把预设参数的几个模版介绍一下，界面如图5所示
　　 Day(白天)  Night( 夜晚)  Sunrise( 日出) Alien Skies(另类天空)  Special FX (特效)  User Presets(用户自定义效果)
　　4. 这样我们就完成了一个日出的天空。为了下面制作的方便我们新生成一个图层来制作苹果，在Layer图层面板中单击新建按钮新建一个图层，并将其作为当前操作图层，如图5所示。
　　5. 下面我们使用一个三维模型来制作一个苹果。选择菜单Filter→Kpt6→ KPT SceneBuilder，系统弹出滤镜界面，如图6所示。按住File菜单不放，选择其中的Open命令，打开Kpt6目录下的Object目录下的Apple.ptc文件插入一个苹果模型，并且拖拽模型到适当的位置。
　　6. 单击Interactive Texture（交互材质）标签，如图7所示。再单击屏幕当中的Click here to render（单击此处渲染）可以看到一个光溜溜的苹果，因为我们没有赋予它材质或颜色。
　　7. 单击苹果的身体，再单击Use Color右侧的颜色框，为苹果指定红色。同样再单击叶子，再单击Use Color右侧的颜色框，为苹果叶子指定绿色。（如果想在苹果上贴材质可以单击Use Texture（使用材质）选定图片来作为材质。）按OK结束操作
　　8. 苹果完成之后我们就可以开始制作苹果的手脚了。确保当前图层是苹果图层。选择菜单Filter→Kpt6→KPT goo，使用其中的手型工具在苹果上涂抹出下部，如图8所示。
　　9. 当画手指的时候因为手指比较细，所以可以把手指的笔刷调细。单击笔刷将Brush Size笔刷粗细调整到17像素，就可以拉手指了。
　　10. 画好以后，OK，整个画面就做好了。}
