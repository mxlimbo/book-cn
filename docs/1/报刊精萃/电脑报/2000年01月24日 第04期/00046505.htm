《 {秘技锦囊} 》

{　　科隆战记 3
　　当你要打头目时，会出现一段对话，此时按人物变换键就可以不断升级。
　　游戏中当和人对话或观看告示牌时(也就是游戏画面下方有出现讯息的时候)，按以下键：
　　F1 = 增加 HP 及 SP
　　F2 = 增加金钱并升级
　　F3 = 启动“小辣椒”加速模式(按一下启动,再按便解除)
　　当凯恩把 “日光降临”练满时启动“小辣椒”加速模式，便可不费吹灰之力就学到本世纪最帅气的绝招“地狱怒火”。
　　雷曼2
　　游戏中输入Globoxrules，就可得到所有力量
　　创世纪 9：升腾
　　在游戏目录中找到 default.kmp 这个文件，用写字板之类的文本编辑工具打开它
　　将下列指令加到[cheat Commands〗底下
　　alt＋shift＋i = toggle_avatar_invulnerable
　　alt＋shift＋l = toggle_avatar_fly
　　在游戏进行中即可使用秘技
　　alt＋shift＋i = 圣者无敌
　　alt＋shift＋l = 圣者飞行
　　还可以在[cheat Commands〗后加上以下密码：
　　pass_one_hour
　　unpass_one_hour
　　sunrise_sunset
　　pass_one_minute
　　unpass_one_minute
　　toggle_sun
　　toggle_wind
　　toggle_storms
　　toggle_avatar_fast
　　如果你用的是1.07版或以上，你可以在default.kmp中加入以下几行代码
　　recharge_mana
　　fill_spellbook
　　spells_0－2
　　spells_3－5
　　spells_6－8
　　法术共有八级,每级四种,共计三十二种,每种法术都需要至少一类的药材搭配其卷轴，要在五芒星阵执行束缚仪式(bind spell)后才能使用。而药材一共有八种,由于背包以及袋子的空间非常有限，使玩家到游戏中期常常有袋满为患之苦。在此提供一个特殊方式,玩家只要每种药材带一个即可，就再也不用上山下海找药或是花钱买药材了。
　　在五芒星阵执行束缚仪式，依法术规定放置所需药材及卷轴后，点燃蜡烛,会要玩家输入法术咒语。此时先用游标指向药材及卷轴，按住鼠标左键将点到的东西移到玩家身上，等到药材及卷轴都捡完了之后,再输入咒语，之后你会发现法术已经记忆成功，而卷轴及药材却完全没有消耗。
　　让怪物兽对你视而不见
　　打开背包（按B键），这时怪物们就会对你视而不见，你就可以随意狂打他们了。为了不影响视线，你可以把背包拖到屏幕边缘。
　　魔法十字军
　　在游戏时，按下Enter并键入下列代码：
　　－EMBIGGENME 上帝模式
　　－CRAZYGUY 全部的符咒和超自然的力量
　　－WHOAH 飞行模式
　　－UBERLOAD 随机的跳关
　　－SHOWFPS 帧频显示
　　－CALCFPS 帧频计算
　　－DEBUG 显示DEBUG信息
　　－BOUNDINGBOX 周围的物体被拉进箱子
　　－3DNOW！ AMD's 3DNow!支撑开关
　　CSWIRE 风景开关
　　圣剑大陆
　　按F8输入oh vincent后按Enter键,再按F8键输入:
　　hp filling = 体力全满(战斗中才能使用)
　　mp filling = 法力全满(战斗中才能使用)
　　level up = 等级上升一级
　　wahaha = 直接过关}
