《 {轻松备份Windows CE} 》

{　　如果你在使用掌上电脑的时候，突然遇到掉电，或者其他紧急情况，而你所有的资料都没有备份，这将是多么可怕的情形啊。其实你可以试试bUseful Backup Plus，这可是一个重量级的 Windows CE 备份软件，用起来非常简单，可以给你无后顾之忧的感觉。
　　bUseful Backup Plus可以备份你的Windows CE系统的全部设置或者指定的部分设置，并可以把备份文件压缩，生成自解压备份程序，这样你可以不用安装bUseful Backup Plus即可恢复你的备份了。你可以选择单独的一个文件、数据库或者注册表进行备份和删除。备份的文件可以存储在闪存卡或者是任何可重写存储设备上。下次等你要恢复备份时，还可以同时完成对多个掌上电脑的设置。当然，其中的程序的重新安装需要安装在相同的驱动器上。在你进行备份时，bUseful Backup Plus会自动为你建立一个记录文件，记录发生的错误、备份的文件等信息，方便你的查询。
　　使用bUseful Backup Plus也非常简单，你首先在软件中选择要备份的资料，指定备份资料时，软件界面分为两个部分： System 和 Files(如^04050301a^)。System 是系统资料，也就是地址簿、工作计划、注册表等资料，所以说，如果你只想备份和 Outlook 有相关的资料，只要选 Databases 就可以了。
　　Files 是文件系统，就是所有文件可以整个备份起来。( 如果用 CF 卡备份要注意，因它会连 CF 卡下的档案也进行备份，所以说，如你不打算备份 CF 卡内的资料，要在 File 目录中把 “储存卡” 的钩去掉。 )
　　很简单，是不是？赶快去down一个来试试吧。
