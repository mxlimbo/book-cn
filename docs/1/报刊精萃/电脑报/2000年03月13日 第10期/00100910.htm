《 {浏览器新贵KATIESOFT} 》

{　　上网时，我们经常打开多个IE进行浏览，这样虽然可以减少等待的时间，但是资源消耗很大，有时甚至使机子不堪重负，以罢工告终。Katiesoft就是针对这种情况，以其独有的一屏四窗口方式，解决了窗口与资源之间的矛盾。
　　Katiesoft是一款准浏览器软件，它必须配备IE 4.0或Netscape 4.0以上的浏览器才能使用。从http:／／www.katiesoft.com下载其最新的6.0.92版本，运行安装程序后，即可进入该程序。
　　◆各菜单项功能如下：
　　File 包括打开文件，打开另一个Katiesoft，打印等常用命令。
　　View 其中，“1 pane”意为打开一个窗口，“2 pane”意为打开两个窗口，其余类推；“Rotate panes”意为窗口之间位置旋转。
　　Katiesoft Option 其中“Pages”包括每一个窗口默认的首页、默认的搜索引擎（http:／／www.altavista.com；http:／／www.hotbot.com；http:／／www.yahoo.com；http:／／www.excite.com），当然你也可以手动设置自己喜欢的首页和搜索引擎；“Settings”中，选中“Recall my last session”，则表示每次起动都进入上次退出前所处的网址；“Use fetch instead of reload”则表示窗口所显示的内容始终与网站上的一致，这对于网上炒股特别有用；“Auto advance every 1 minute”表示每隔一分钟就自动显示群组中的另四个相关网址，其中间隔的时间可以由自己设置。(^10020501a^)
　　Internet Option 直接调用IE的属性设置。
　　Mail、news 就是打开OE。
　　Control 包括对四个窗口进行停止、刷新、返回默认主页及搜索等操作。而“Pop up windows”是针对那些烦人的Cookies而设置的，它可以禁止广告窗口以及设置打开广告窗口的方式。
　　Slideshow指输入20个URL网址并取一个群组名称，Katiesoft已内置了20个这样的群组，以后想去哪一方面的网站，只要点击“Slideshow”菜单项中的相关群组名即可。
　　Favorites 不但包括了IE中的收藏夹内容，还预先分类设定好了搜索网站、新闻网站等地址，相当方便。
　　Snapshot 这种“快照”方式是收藏夹功能的延伸，它能把你一次所显示的四个网址记录成一个ktml文件，这样你只要打开一个ktml文件就相当于同时打开了四个网站。
　　Panes 除了全屏显示所选窗口外，还能使后台运行的程序转到前台，占用四个窗口中的一个窗口所处的位置。
　　Quote 、Search、News 都是对所需内容的快速搜索。
总的来说，Katiesoft是一款相当不错的浏览器，它的四窗口、Slideshow、Snapshot以及对Cookies的阻拦都相当实用，特别是有时我们在浏览一个主页时想去其中所指的一个链接，但又还想看该主页的其它内容，这时只要把该链接拖到另一个窗口就解决问题了。Katiesoft的确给我们带来了方便与快捷,是一款不可多得的浏览器软件。}
