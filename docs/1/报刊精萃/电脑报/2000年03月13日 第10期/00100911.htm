《 {图标任我抓} 》

{　　从可执行文件或动态连接库文件中提取图标的软件有很多，但是想要直接从屏幕上“抓”图标的工具就非Icon Catcher 3莫属了。它独特的抓取功能一定会博得你的喜爱。
　　运行后出现界面，你可不要下意识的直接去按第一个按钮啊（相信多数人都会这么做），看清楚，那可是“Exit”呀，真是够与众不同的了，我第一次运行的时候它就给我先来了个下马威。它亦支持从可执行文件或动态连接库等文件中提取图标，更有甚者，它可以把指定目录甚至整个系统中的图标全给您翻出来，你要做的一切就只是选一下“Scan All”即可，够狠吧。
　　“Capture”可是它最值得骄傲的功能，点击它后就会出现一个透明的矩形窗口，移动一下才发现原来它是一个放大镜，可供你仔细的选择须要抓取的区域，想得还真是周到，不过由于是放大显示，图像看起来会很粗糙，这没有关系，图像实际上还是很圆滑的。选定区域后，双击，此图标就会出现在Icon Catcher工具栏下面的图标显示窗口中，在放大镜上单击右键即可取消。
　　在“OPTION”选项中，你可设置一些与软件相关的参数：
　　Icon Size用来调节所抓图标的大小，对于一般的WINDOWS图标，用其默认的32×32绝对没问题；
　　Output Format可以指定输出的文件类型，用＊.ico就行了；
　　Background Color不用多说，是设置背景色的；
　　Interface Language比较有意思，我安装的版本只有英文选项，你可以点下面的连接直接到网上去下载你所需的语言版本；
　　File for Scanning用来设置从哪些文件类型中抓取图标，相应的打勾即可；
　　Performance用来调节CPU占有率和复制的解析度的大小，就用默认的最大吧；
　　Switches中的选项是一些不重要的参数，只管打勾就行了。
　　它是一款共享软件，未注册版本不提供存盘功能，真是太遗憾了。你要是真的喜欢它的话就请到网站www.helexis.com注册，或与软件作者联系support@helexis.com.
　　此软件你可以直接从http:／／www.helexis.com／ic／iconcatc.exe下载，大小为418KB。}
