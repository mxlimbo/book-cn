《 低端网络激光打印机的选购 》

选购低端网络激打的必要性

　　随着Internet的进一步发展，网络打印技术的发展也相当快，网络打印机的性能得到了很大的提高。网络打印机在提高设备使用效率、提高打印速度和质量、提供个性化的打印以及降低整体拥有成本方面具有一系列优势，今天，网络打印机已经成为用户在添置OA设备时首先考虑的外设之一。
　　HP早在1996年底就在中国市场上推行“网络打印”的概念，但时至今日，中国的网络打印市场并没有真正蓬勃发展起来。在当前许多的办公坏境中，用户仍利用PC服务器来实现共享打印，这种看似不需另外投资的所谓“网络打印”存在许多弊病，实际上并不是真正意义上的网络打印。由于PC本身还要充当网络服务器，这必定会影响到整个网络的运行性能，无形之中，被占用的PC成为了一台公用的设备，该PC使用者的工作效率必将大打折扣。尤其是在打印业务量较大或PC本身不稳定时，打印进程的延误必将给工作带来更大的损失。利用打印服务器实施低端网络打印，其价值并不仅是体现在打印作业能力的迅猛提高，更重要的是实现了网络打印管理的合理性。此外，独立打印服务器＋打印机的网络打印可将作为打印服务器的PC解放出来，更为有效地利用资源，保障原PC使用者充分自由地使用。

选购时应注意的问题

　　如果单从打印工作原理看，网络激光打印机和普通激光打印机确实没什么特别的不同，但在网络环境中工作时，网络激光打印机比普通激光打印机具有更高的工作效率，而这正是普通激光打印机望尘莫及的。下面让我们来看一看选购网络激光打印机时应注意的问题。
　　1．分辨率：分辨率是购买所有打印机都应关注的一项指标，单位是DPI（Dot Per Inch：中文含义是每英寸介质上可打印的点数）。网络激光打印机的分辨率概念与普通激光打印机并无不同之处，同样是DPI越高越好，但是由于网络激光打印机工作重点是强调输出速度，因而对输出精度无过高的要求，过高的分辨率反而会使输出速度变慢。就当前市场的网络激光打印机而言，其分辨率普遍为600 dpi，高档产品可达到1200 dpi。作为中小企业，选择600 dpi的网络激打就已足够，没有必要追求过高的分辨率。
　　2．打印速度：网络激光打印机的输出速度指标用PPM（Pages Per Minute：每分钟输出页数）来衡量。这与普通激光打印机相同，所不同的是网络激光打印机更注重输出速度这一指标。一般而言，分辨率相同，输出幅面相同的产品，打印速度越快，说明其档次越高，价格自然越贵。在网络打印机中，12 PPM打印速度的产品只能算是低速产品，快的可达到24 PPM，一些最新的高速产品打印速度高达28 PPM。对于我们今天介绍的低端网络激光打印机，其打印速度一般在12 PPM到16 PPM之间。影响网络激光打印机输出速度的因素有两个：1内置的处理器工作频率及运算能力。它是打印机具有高速打印性能的重要条件，常见的是采用RISC处理器，工作频率一般在50MHz，高速产品可达到166MHz；2内存配置大小，以及最大能扩充到多少。一般而言，内存容量越大，打印速度就越快。此外，在不需要额外增加资金投入的情况下，部分厂商还采用内存增强技术来提高网络激光打印机的输出性能。如HP为其激光打印机研制的MET（Memory Enhancement Technology：内存增强技术）,利用它能更加有效地提高打印速度。
　　3．输出幅面：网络激光打印机目前有A3、A4两种输出幅面，一般A4幅面的完全可以满足办公要求。而A3幅面的算是大幅面产品，从价格上看，A3幅面的产品比A4幅面的产品差不多要贵一倍左右。
　　4．网络连通性：作为网络打印机，其良好的连通性自然是很重要的因素，只有具备良好的连通性，才能保证网络激打与各种设备进行连接使用。检验一台网络打印机连通性的好坏，主要是看其连入网络后能否正常工作。不能仅从某一特定网络连接的好坏来断定其连通性的优劣，而应该根据在各种网络中（如以太网、Novell NetWare、AppleTalk等）的使用情况判断，只有这样才能保证打印机具有良好的连通性。
　　5．外设管理软件：它是使用网络打印机最重要的辅助工具，用户可以通过它完成打印机的安装、配置和管理。部分产品还可以通过Web浏览器界面进行远程管理工作，节约了时间和费用。

几款典型的低端网络激打产品

　　网络打印机在网络普及的今天已经越来越显示出它的巨大威力，笔者就结合网络打印的技术及使用特点，给大家介绍几款网络激光打印机的性能指标。
　　●联想LJ2312P/2412P网络激光打印机
　　联想的网络激光打印机新品LJ2312P/2412P，是联想公司在普及网络打印行动中，专门为中小企业推出的通用型产品。它们通过外置独立打印服务器＋打印机实现网络打印，而它们所具有的10页～12页/分钟的打印速度以及5000元以下的价格优势，令中小企业用户有了选择激光网络打印机的可能。
　　与传统的利用PC服务器实现共享打印相比，LJ2312P/LJ2412P具有灵活、可靠、实惠的巨大优势。它所提供的高速双向并口与USB接口，真正解决了移动打印的难题，做到即插即用，很适合中小企业及工作组的网络打印需求。此外，对于追求高速网络打印的用户，联想提供了内置网卡LJ8212N，网络打印机可通过网卡直接与网络相连，其数据传输能力很强。
　　联想的LJ2312P(如图1)网络激光打印机可以达到600×600 dpi的打印分辨率，LJ2412P更达到了600×1200 dpi的高分辨率。此外，它还具有丰富的灰度级专业图像输出，从而满足了图像输出的需求。LJ2312P的内存为2MB，LJ2412P有4MB的内存，安装SIMM可扩充到36MB，使联想激光打印机的打印速度提高，可以达到12页/分钟，而且提供了少于45秒的预热时间及15秒的首页打印时间。纸张适应技术是联想网络激打所具有的一大特色技术，中国地域广阔，南北气候相差甚大，打印纸的生产厂商技术实力参差不齐，同时，大多数的办公环境灰尘多、湿度大，国外很多激光打印机产品在初入中国市场时都无一例外地遇到卡纸的问题。联想采用纸张适应技术，对P系列打印机进行了全新的防卡纸设计，通过平滑的走纸通道、全新的硒鼓机构和特殊的防静电处理，把打印机的卡纸率降到最低，即使发生卡纸，也非常容易取出，不会像其它打印机一样，由于取纸困难而对硒鼓等精密部件造成损害。
　　●HP LaserJet 4050网络激光打印机(如图2)
　　HP LaserJet 4050采用了分辨率增强技术（REt）和HP的超精细碳粉技术，在全速引擎状态下仍能提供真正的1200 dpi打印质量，具有133MHz微处理器及8MB的内存，保证了大流量的数据传输，达到了每分钟16页的高速打印。这款网络激光打印机还配备有增强的控制面板帮助信息，它能及时提示打印机出现的问题，可为你节省时间。HP LaserJet 4050打印机广泛应用于以太网10/100TX、Microsoft、Novell Netware、Unix以及其他流行操作系统，通过打印服务器与网络相连，能以与网络相同的速度传输打印数据，几乎可达到每秒10MB/100MB的速度。这不但是打印机连网形式的一种革新，更是打印速度的一个质的飞跃。同时，通过HP Web JetAdmin，它还能实现轻松安装和管理，网络管理员能通过Web浏览器界面进行远程安装、配置和管理打印机。用户只需从桌面进行各种管理工作，也节约了时间和费用。应该说，这款高性能、高质量的激光打印机在其同价位机器中是赢家。对于任何一个需要工作组打印机的人而言，HP LaserJet 4050的速度和质量使得它成为你的首选。
　　●EPSON增强型智网通EPL－N2010激光打印机(如图3)
　　EPSON增强型智网通EPL－N2010装备了全新的高速机芯，可以将A4纸打印速度提高至每分钟20页，这在我们介绍的几款打印机中打印速度名列前茅；同时，它配备超强的166MHz处理器以及标准的8MB内存（可扩至256MB），可充分满足你大量打印输出的需求，同时也可以满足你存储字型等需要。在软件方面，EPSON采用先进的激光打印机页描述语言ESC/Page，保证了高速打印。另外，它在Windows 98/95/32和Windows NT 40操作系统下的打印驱动程序中开发了“作业平衡”功能，可根据计算机性能的高低，自动调节计算机和打印机的数据处理差额，合理分配工作量，从而有效地提高了办公效率。
　　为适应商业用户业务坏境的网络发展需求，EPL－N2010还特别装备10 Base T/100 Base TX内置网卡，无论Windows 98、Windows NT还是Mac用户都可通过网络共享其超凡打印能力，使中型工作组网络打印更便捷，工作效率更高。在打印质量上，它采用EPSON灰度调整技术MicroGrayl200，通过对墨点尺寸的调整，产生1200 dpi高分辨率输出，实现128级的灰度等级，颜色过渡平滑细腻，使打印图像可达专业效果；此外，采用分辨率增强技术BiRITech，通过对线条边缘的修正，避免小字符放大时出现的边缘锯齿，确保图形中的细小曲线能精确打印，许多含有小字符、照片的文档资料、CAD图纸均可清晰鲜明地再现。此外，根据环境温度，打印机机芯还可自动调整墨粉浓度，确保了打印的高精度。
　　特别值得一提的是，EPSON增强型智网通EPL－N2010还提供A3幅面的优质宽幅打印。无论是计划书、报告，还是宽行报表，大至A3幅面的，小到明信片尺寸的，均可随意输出，可打印的介质类型包括普通纸、信封、卡片、标签甚至透明胶片等，尤其适合政府部门、现代商业办公企业及CAD用户对打印输出的要求。
　　●LEXMARK Optra s1855激光打印机(如图4)
　　LEXMARK Optra s1855激光打印机采用了133MHz NEC 4300－133 RISC处理器，从预热到打印第一页不超过10秒钟，而且打印速度可以达到18页/分钟。说到打印质量，它具有真正的1200×1200 dpi的高分辨率，图像质量令人满意。LEXMARK Optra s1855激光打印机内置有8MB的内存，不过使用者还可以选择SIMM来扩充内存，最大容量可以达到132MB，支持2lGB硬盘。如此大的内存容量，你可以用它来存储字型、规格、表单和宏等，大幅度提高了打印机的作业处理能力。在打印机的使用上，LEXMARK Optra s1855激光打印机具备缩版并印、碳粉节约模式与省电模式等功能，让你的打印轻松又省钱。
　　LEXMARK Optra s1855激光打印机同时具备高速双向并口与RS232C串口，拆装都十分方便。作为多人使用的办公室共享资源，这款网络激光打印机所提供的两个基于PCI的内置式解决方案端口（ISP），可以支持多种选择。同时，它的Marknet S适配器可以支持包括DLC/LLC、Novell、AppleTalk、TCP/IP在内的一系列网络协议，此外，它还直接支持多种网络操作系统，例如Novell NetWare、Microsoft Windows NT Server V31/35/40、IBM OS/2等，对于网络打印来说，使用起来非常方便。在网络打印管理方面，它有开放的MarkVision打印机管理程序，同时支持开放式的打印机管理。此外，它还可以自动集成到多种网络管理应用程序中。
　　●佳能LBP－800激光打印机(如图5)
　　佳能LBP800是这几款低端网络打印机中价格最低的，仅2800余元。它的外型设计充满时尚感，它的打印质量更令入瞩目，打印速度为每分钟8页，如配合佳能独有的自动图像精细技术，它的分辨率可高达2440×600 dpi，这也可以说是在我们介绍的几款低端网络打印机中分辨率最高的了。除此之外，它所具有的全新打印机驱动程序，能支持Windows 98/95及Windows NT，兼容性极高，对联网使用的多用户来说，能很方便地使用。佳能LBP－800可以通过AXIT 1610进行各种网络连接，支持10Base－T和100Base－TX以太网标准，支持的网络平台为Windows 95/98/NT。不过它与我们所介绍的网络打印机还是有一定的不同，它需要另配网卡及其它配件，算起来也不便宜。
