《 专家坐堂 》

　　为什么不能正常关机

　　哈尔滨读者刘福荣问：去年十月购得厦华学生电脑一台。前不久正常关机后，没断开电源线。这时突然停电又立即来电，以后不论是软关机还是按“POWER”键关机，都不能断开主机电源。请问这是什么原因？
　　答：电源断开又马上通电会产生瞬间高压，作用在计算机电源上会在其内部产生巨大电流，从你的描述上看，软关机和强行关机都失效，我认为开关电路损坏的可能性很大，开关电路损坏后导致不能正常处理主板和ATX按键开关产生的关机信号。条件允许的话，建议更换电源。

Win98播放CD时无声音的解决

　　北京读者夏又权问：自从操作系统从Win95升级到Win98后我一直用解霸5.0播放CD和MP3。近期我用Powerpoint制作课件时，Win98自带的CD播放器不能播放出声音，而播放器显示读盘正常，音量开关也能正常调节。而在系统升级之前，在Win95下播放器能正常使用。重新安装声卡驱动程序无效，已确认光驱到声卡的连线正确。这是什么原因？
　　答：通常从主机传到音箱的声音是模拟信号（USB音箱等数字设备除外），普通音乐CD存放的为数字形式的信息，从这些数字信息到变成我们听到的声音，需要一个D/A（数/模）转换，光驱自身带有D/A电路，可以把CD上的数字信息直接转换成模拟信号，再通过光驱和声卡的连线把模拟信号传到声卡上（也可在光驱的耳机插孔直接输出）。不同品牌的光驱，D/A转换的效果不同。Win98允许在播放CD时直接把光盘上的数字信息读入内存，通过CPU和声卡来完成D/A转化，你可以通过在“控制面板／多媒体／CD音乐”中查看“在这个CD－ROM设备上启用数字CD音频”是否被选中，用超级解霸来播放CD音乐也是此道理。如果系统启用了数字音频，但由于某种原因，播放CD时如果仍然在光驱上进行D/A转换，那么音箱中就不会出现CD歌曲。

红外线接收图标为何时有时无

　　读者阳道钧问：有时安装Win98后，控制面板里会出现一个红外线接收者的图标，可有时重装系统后又看不见。请问它为什么时有时无？
　　答：现在的主板上都带有红外线接口，和主板其它接口一样（COM口、USB口、IDE口等），在CMOS中都有对它们的设置，可以打开也可以关闭，如果在CMOS中关闭了主板的红外线接口服务，那么在安装Win98后，红外接口将不会显示。

显示器不能显示红色怎么办

　　读者罗永问：我的一台显示器有大半年没用了，现在使用时发现色彩很淡，红色变成黑色，像某些游戏，片头的背景是红色，而字是黑色，我就看不见字，请专家指点一下是什么原因?有什么办法可以修复。
　　答：如果你用的是新型显示器，带有RGB三色独立调整系统，可以直接进入显示器设置菜单，把红色值调大；如果你用的是早期老式模拟显示器，这种现象应该是红色电子枪电压不够，射电子束不能正常打到显示屏产生的。打开显示器后盖，在电子枪接口电路板上有红、绿、蓝三色调整电位器，可以把红色枪的电压稍稍加强。在打开显示器后盖时一定要注意安全，建议将显示器送到专业维修部门处理。

Outlook Express 5.0怎么“吃”标点？

　　广元读者岳军问：我装了Outlook Express 5.0后，在用中文写邮件时，发现每打一个标点符号再打中文时接下来的中文会把前面的标点符号覆盖掉。我只能把光标插入要打的标点符号的位置重打标点。请问这是什么原因？该怎么解决？谢谢！
　　答：这是IE5同王码4.0不兼容的表现，如果输入法习惯用五笔的话，建议使用智能陈桥五笔输入法。

VXD文件丢失后的处理

　　读者徐弘问：最近电脑在启动时每次都会出现如下提示：Cannot find a device file that may be needed to run windows or a windows application. The windows registry or SYSTEM.INI file refers to this device file,but the device file no longer exists. If you deleted this file,on purpose try uninstalling the associated application using its uninstall or program. If you still want to use the application associated with this device file,try reinstalling that application to replace the missing file——meme.vxd. Press a key to continue。我该怎样解决?
　　答：一些软件在安装的时候需要在SYSTEM.INI及注册表里加入信息，使系统在启动时直接读入某个VXD文件，你可能最近安装了一些软件，而在使用过程中不小心把一个叫meme.vxd的文件删除掉了，这样系统在启动时找不到SYSTEM.INI或注册表中指定的该文件，就会出现以上的英文提示。建议你查找一下近期安装的软件，把meme.vxd文件重新拷入Windows下的SYSTEM目录。如果找不到该文件，在系统启动后能正常工作的前提下，可以考虑把SYSTEM.INI文件及注册表中有关meme.vxd文件的项目及键值删除。

对话框显示不全的解决

　　包头读者周婷问：我的Winzip等许多程序的对话框显示不全，右边和下边都有部分看不见，致使好多功能无法使用，不知如何解决？我试过改变分辨率也无效（我的系统是Win95 OSR 2，显卡是Trident 9685，2M显存）。
　　答：这种现象是系统设置的问题，如果对话框中的字体设置得过大，右边和下边就会有文字看不见。你可以在桌面上单击鼠标右键，选择“属性”中的“外观”，把“方案”改为“Windows默认”，即可自动把对话框中的文本改回原来的大小。

为什么一开机就出现一窗口及软键盘

　　苏州读者李红梅问：我的机器是装的Win98，每次开机后D盘的program files窗口和微软拼音输入法的软键盘就出现在桌面上，但启动菜单里是空的。请问我该如何处理？
　　答：进入Win98后执行的文件不一定非要在启动菜单中指定，启动菜单中的快捷方式只是执行文件列表的一部分，详细的启动执行文件列表在注册表HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Run及RunServices中指定，你可查看其中是否有打开D盘文件夹的键值，如果有请将它删除（也可以通过MSCONFIG中的启动来查看）。系统启动后自动打开输入法，是因为默认语言为中文的结果，只要把默认语言改为英文，重新启动即可。

Word 97打开后任务条为什么消失

　　重庆读者徐晓燕问：为什么我只要一启动Word 97，Win98的任务栏就会自动消失（已排除设置为“自动隐藏”）？
　　答：Word 97可以工作在普通窗口和最大化窗口两种模式下，任务栏消失的现象多出现在前者，因为普通窗口的窗口大小是可以调整的，如果在普通窗口状态下把窗口调至充满屏幕，并且在任务栏的属性中关掉了“总在最前”，那么在启动Word 97后，任务栏就会自动消失。你可以把普通窗口调整至合适大小，或者是在“最大化”窗口工作，同时要注意使任务栏“总在最前”有效。

提示RAM PARITY Error的缘由

　　焦作读者李雪涛问：我的计算机为MMX200，1.2G硬盘，32M内存，华硕TX97－E主板，显卡为S3，最近一段时间开机无法启动，出现Ram Parity Error的提示，请问该问题如何解决？
　　答：这种情况多出现在内存质量不稳定的兼容机上。为保证内存数据的准确性，一些内存自身带有奇偶校验的功能（同并口串口的校检是同一原理），如果内存不稳定，在奇偶校验时就会出现错误的提示，如果条件允许的话，建议更换内存以确认。另外，如果内存本身不带有奇偶校验的功能，在CMOS中又把内存的奇偶校验功能打开，系统也会提示奇偶校验错误。

硬盘为什么“吱吱”作响

　　南京读者芮成晨问：我在去年10月买了一台品牌机，近期经常出现蓝屏，重装系统格式化执行至12％时，硬盘发出“吱－吱－”异响，并且出现“Trying to recover allocationunit 65,288”后死机，重新执行FORMAT无效，FDISK出现“No Fixed Disk Present”，请给予指点。机器配置为PⅡ266，4.3G硬盘。
　　答：你的来信中没有指明蓝屏的类型，一种蓝屏为常见的VXD、0D或0E错误，另一种可能是系统无法读取硬盘数据而出现的蓝屏。从现象上看，可以肯定你的机器硬盘出现了物理损坏，系统无法正确格式化硬盘，FDISK也无法识别硬盘，将正确的分区数据写入硬盘，请联系机器经销商。

怎样快速输入日文和中文？

　　潘阳读者任香问：我是一位日语工作者，经常要在文档中输入日文，当输入一篇日语文章时，总要不停地在五笔和区位之间切换，日语字符的区位码查起来又很麻烦，请问是否有简单的方法来完成这一工作？
　　答：有一种比较简单的方法，就是把每个日语字符当做一个词组来处理，在五笔输入法的自定义词库里为每个日语字符编码，这样输入时就不必在各种输入法之间切换了。

显卡驱动程序为什么无法在NT下安装？

　　安徽读者赵浩霖问：我在用NT组网，改写显示驱动程序时，系统不允许修改，提示没有权限，可我使用的是Administrator登录，有所有的权限，请问这问题怎样解决？
　　答：如果你在NT下安装的显卡驱动程序是为Win9X设计而并非NT版本时，系统会提示没有权限而无法安装驱动程序，请你改用专为NT操作系统设计的显卡驱动程序。

盘符顺序改变该怎样恢复？

　　温州读者李新源问：我参照贵报1999年第50期“软件世界”，把C盘的MSDOS.SYS文件拷入Win98的启动盘，用该启动盘可直接进入电脑，但是我突然发现很多软件都不能用了，经查证，发现是原来的光驱盘符从F盘变为了G盘，请问我该怎样把盘符还原？
　　答：1.44M的软盘容量很有限，微软把常用的一些外部命令打包到一个名为EBD.CAB的文件中，Win98的启动盘在启动时会自动利用内存创建一个虚拟的硬盘，然后把EBD.CAB这个文件解压至虚拟硬盘中，这样就可直接使用一些常用的外部命令。CDROM可以在启动盘加载，也可以在Win98启动的过程中加载，但这两种加载方式都是位于虚拟硬盘的建立之后，所以盘符要向后移一位。因为启动Win98后，所有的外部命令都在硬盘上，因此虚拟盘的存在也就无意义了。你可以手动把启动盘CONFIG.SYS中有关建立虚拟盘、AUTOEXEC.BAT中解包CAB的命令删除，也可直接将Ramdrive.sys及EBD.CAB文件改名。注意要在AUTOEXEC.BAT最后加上“SET COMSPEC=C:\COMMAND.COM”及“SET PATH=C:\WINDOWS;C:\WINDOWS\COMMAND”。
