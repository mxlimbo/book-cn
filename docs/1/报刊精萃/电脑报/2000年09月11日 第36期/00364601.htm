《 优秀的网络监视工具：C4U 》

{　　一、什么是C4U?
　　C4U是一款优秀的网站监视软件。它可以替你去“浏览”指定的网站或网页，看看它们有没有更新，哪些地方做了更新，新增了几张图片、几个链接、几个E－mail地址，文本有几处变化，更新的内容中，是否包含有你指定的关键字。如果有，C4U会立即显示几行简短的报告，并放上新内容的链接，点击它，就可以在浏览器或C4U窗口中查看具体内容，从而节省大量宝贵的上网时间，尤其当同时监视多个网站时，效果更加明显。很多朋友喜欢关注新闻事件，用C4U来跟踪各大新闻网站，就非常方便。还有的朋友，可能正在从事某种研究，那么，利用C4U跟踪关键字的功能，效果也非常理想。
　　二、C4U的使用
　　1.添加站点
　　使用C4U，你必须先上网，在你做其他事情的同时，它在后台监视网站或网页，并随时报告最新变化。另外，你还得先将要监视的站点网址告诉C4U，这就叫“添加站点”。有四种操作方法可选：
　　（1）点击浏览器中的C4U按钮。安装C4U后，这个按钮会出现在浏览器右上角，点击它，就可以将当前正在浏览的网页或网站放到C4U的监视列表中（C4U左上部小窗口）；
　　（2）在C4U窗口中，点击Add（添加）按钮，然后，在对话框第一格填网址，第二格填网站或网页名称；
　　（3）在C4U窗口的监视列表中，右击鼠标，选择弹出菜单上的“Add Location”（添加站点），余下操作同（2）；
　　（4）在C4U窗口中，点击菜单“Location|Add Location”，其后续操作同（2）。
　　对于每个监视站点，在监视内容方面，我们还可以进行不同设置。在上面四种操作中，都将打开一个相同的对话框，我们可以在那里进行设置，其中各项简要说明如下（如图）：
　　Location (URL)：指定网站或网页的网址。
　　Name：给网站取个名称，它将显示在监视列表中。
　　Notify Me When：当站点有以下更新时，让C4U通知你：
　　●Text Change：文本变化，当网站上的文字内容有变化时，C4U将通知你；
　　●New Link：新增链接，当网站上有新链接放入时，C4U会通知你；
　　●New Image：新增图片，当网站放上新的图片时，C4U会通知你；
　　●New Email Address：新增E－mail，当网站上出现新E－mail时，C4U会通知你。
　　对这四项，你还可以指定变化数量达到某值时才通知你，其缺省设置为1，意思是，只要有一处新文本、一个新链接、一个新E－mail地址、一张新图片加入，C4U就立即报告。
　　Found New Key Words：查找新的关键字。它可以用于监视像“手机上网”这样的关键字。当C4U在网站上发现更新内容中包含指定关键字时，它会通知你，并将相关内容的链接告诉你。关键字中可以运用“or”和“and”逻辑运算符。
　　Ignore Periodic Changes：忽略周期性变化。选中此项后，像旗帜广告这样的自动循环变化，C4U就不会作为网站更新来骚扰你。如未选默认设置，笔者建议你选择，因为很多网站都放有这种东西。
　　Flash the application icon when this page is changed：如果网页更新，就闪烁程序图标。选中后，当C4U发现指定网页有变化时，它在系统任务条中的图标会闪烁，以提醒你马上前去浏览。
　　2.检查站点
　　添加站点后，接下来就是检查站点。需要说明的是，默认设置下，C4U每次启动后，会自动检查各站点有没有更新，但接下来就不再自动检查了，除非你用鼠标右击它在任务条中的图标。选择“Check Every……”中的某一项，让C4U每隔15、30……分钟自动检查一次。当然，你也可以在需要时，手动检查站点。
　　（1）检查监视列表中所有站点。按一下工具栏中的Check（检查）按钮就行了。如果你想暂时不检查个别站点，鼠标右击站名，选择Disable即可，下次选择Enable，又可以恢复检查。
　　（2）只检查监视列表中个别站点。鼠标右击站名，选择Check Now（立即检查）即可。
　　检查站点后，C4U会针对每个网站的情况，给出一份简短的报告，虽然是英文的，但格式固定，内容简单，一般都看得懂，其中蓝色的文字，就是更新内容的链接，点一下就可以查看具体内容。下面说明一下检查时站名前出现的各种图案的意义，这有助于你快速判别站点的更新情况。
　　●黄脸：表示你要求跟踪的网站或网页已经更新了。
　　●蓝脸：表示某个网站自从上次检查后，还没有更新过。
　　●问号：表示某个站点还没有检查结果，C4U即将或正在检查它。
　　●绿脸：表示该站点暂时禁查，即上面说的选了Disable。
　　●灰脸：表示该站点不允许访问，这往往是由于线路阻塞等原因。
　　三、使用技巧两则
　　1.监视浏览器书签中的站点
　　有相当网龄的人，浏览器书签中一定珍藏了不少精彩网址。现在，你可以用C4U来监视它们，以更快获得感兴趣的信息。利用C4U的网址导入功能，可以轻松达到这个目的。操作方法如下：
　　点击菜单“Import Favorites|From Browser”，这时，会弹出一个对话框，你收集的酷站都在这里，要监视哪些，直接挨个点击就行了。不过，一次只能加一个！
　　2.与搜索引擎结合使用
　　将C4U与搜索引擎结合使用，可以获得与某主题相关的更多、更新的信息。现举例如下：
　　假如最近你在关注环境保护方面的信息，你可以这样操作：上网进入某个搜索引擎（如雅虎），查询关键字“环境保护”，待搜索结果出来后，点击浏览器中的C4U按钮，将该网页加入C4U监视列表中，以后，你就可以随时通过C4U，检查搜索结果中有没有新的与“环境保护”相关的条目，一旦有，点它一下即可！
