《 {永远的彩虹——街机模拟器Raine} 》

{　　想必很多玩家玩游戏都是从街机起步的吧，街机在我们很小的时侯就有了，想起当时为了一币通关，一天到晚苦练技术的情景，那真是一种美好的回忆……
　　好了，现在我就开始向大家介绍一款顶级的街机模拟器——Raine，这个模拟器主要是支持80～90年代TAITO,TEMCO等著名厂商的游戏机板上开发的游戏，Raine的最新版本是Raine20000819（官方主页http://www.Rainemu.com/）（大小1004K,下载地http://www.geoshock.com/getfiles/Raine20000819.zip）
　　相应的游戏ROM下载后，不用解压，直接拷贝到Raine目录的rom子目录里面，当然你也可以到天极网游戏世界emu－zone(http://lone2.yeah.net)去下载。
　　当大家下载了模拟器后，解压，双击程序图标，然后就开始进入Raine的街机世界：
　　如图所示为Raine的主界面从上到下选项为:
　　Play Game 玩所读取的游戏
　　Change Game 读取游戏
Control Setup 改变键盘设置/摇杆类型
Screen Setup 改变屏幕设置
Sound Setup 声音设置
　　Game Setup 各种设置
　　Quit Raine 退出Raine
　　在这些选项当中要注意以下几点：
　　1.Control Setup(改变键盘设置/摇杆类型)
　　点击Control Setup，开始手柄设置，你只需要点击Joystick Select，然后就出现了很多手柄的列表。现在Raine已经支持绝大多数手柄，甚至世嘉和超任的手柄。如果你是用的PS改装手柄，你就选Psx pads port 1一项，然后试一下你的手柄是不是可以控制了。如果可以了，就按下Done，结束手柄的设置。
　　2.Sound Setup（声音设置）
　　Raine模拟器现在支持大部分游戏的声音，但是有些游戏还是不存在声音，比如Taito F3机板的游戏（我最爱玩的雷龙也没有声音，5555）。当然除了一些本身模拟器还不支持的游戏这个因素以外，在模拟器中你需要在sound Setup菜单中设置你的声卡。
　　如果你采用的是Soundblaster PCI64声卡及其兼容系列的话，则选择Ensonic Soundscape模式。要是你和我一样用的是Yamaya 724声卡的话,你就在Sound Setup里选Sound Blaster。
　　3.Screen Setup（改变屏幕设置）
　　你可以通过模拟器中的自动或者手动方式来调节模拟器运行的速度，采用自动方式，模拟器将自动进行速度的调节。如果你觉得自己调节比较习惯的话，可以采用手动方式，其方法是在游戏中按Page Up或者Page Down来调节。有一些复杂的游戏，比如Taito F3机板的游戏需要速度比较快的电脑，并且采用自动调节帧速度才能够保持游戏正常的运行速度。
　　模拟器显示尺寸的默认设置是320×240，可能有的机器由于显卡的原因只能够看到某些游戏的一部分画面，那么就在游戏中按“P”键暂停当前的游戏，然后再使用方向键来调节画面的位置，直到你觉得满意为止。
　　4.如果你觉得游戏的速度还是不能让你满意，你可以尝试进行以下的操作，看有没有效果：
　　1.关掉声音；
　　2.将Screen Stup里的Vsync Palette这项设定为off。
　　进行完游戏设置否，就在Change Game里选定一个你想玩的游戏，然后Play Game,进入游戏标题画面。
　　按数字键3 ： 投币；4：投币；1：1p开始； 2：2p开始。
　　此外，Raine还设定了其他很多游戏热键：
　　0 截屏 （保存在Screens目录下，pcx格式）
　　F2 储存游戏进度
　　F3 切换保存的Save Slot(0－9)
F4 读取游戏进度
Pageup 加速
Pagedown 减速
ESC 返回Raine的主界面
P 暂停游戏 （暂停时，可以调整游戏画面尺寸Up上卷 Down 下卷 Left 左卷 Right 右卷）
　　经过上面的介绍，你是不是对Raine这个街机模拟器有了比较详细的了解了呢？那么就开始永远的彩虹——你的Raine之旅吧……}
