《 微软dot－Net策略初见成效 》

{　　微软正在开始展示它的具有远大目标的dot－Net策略的几个部分，尽管这个策略完整的平台还要在一两年之后才能完成。
　　微软方面希望通过本月末的四种服务器应用系统的发布来详细描述公司目前关于dot－Net计划的进展。微软方面称还有更多的产品将提供给市场：有三种服务器在今年年底上市并且明年年初第8种服务器将问世。早在今年6月份的时候，微软就公布了dot－Net平台，包括应用工具、下部基础结构、服务器软件、构建模块服务（比如基于WEB的数据储存）、驱动程序软件等，而最初的焦点问题是最前面的两部分。
　　在9月26日企业发布会上,微软准备发布SQL Sever 2000，Host Integration Server 2000 和 Exchange Server 2000。而微软称在这个月初及时完成的Windows 2000 Datacenter Server届时也会在发布会上向公众亮相。其他四种服务器：BizTalk Server 2000，Commerce Sever 2000, Application Center 2000 and ISA (Internet Security and Acceleration) Server 2000,现在还都是BETA版。除BizTalk外,其他的服务器都必须今年年底上市。而BizTalk则安排于明年早期发行。
　　“很显然,微软的dot－Net计划已在开发界激发起相当大的震撼”，咨询公司META集团的一位分析家威廉如是说。微软的开发工具产品经理大卫说：“在系统的每一个层次里,我们都已进行了一些颇具特殊意义的改变以适应下一代WEB服务的发展。”　　(广州　曾鸣)}
