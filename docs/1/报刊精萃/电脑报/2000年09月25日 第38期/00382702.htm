《 {硬件瞭望台} 》

{　　威盛Samuel2 CPU跳票
　　威盛已经放弃了它在Samuel2 CPU上的野心。大家可能忘记了： 威盛承诺在2000年底把其工作频率提升到1GHz。但是，推出Samuel2样品的日期已经推迟到12月，明年上半年，Samuel2将只有 650MHz－800MHz。
　　DirectX 8.0今年之内发布
　　Microsoft在日本已经用ATi Radeon显卡展示了其DirectX 8.0的特性。正式版的 DirectX 8.0 承诺在今年发布。至于下一个 DirectX 版本 DirectX 8.1应该是在2001年下半年发布，而DirectX 9.0 在2002年下半年发布。
　　ELSA 推出新的影雷者MX显卡
　　ELSA已经推出采用大卡设计的影雷者MX显卡。大卡规格的影雷者MX显卡，在配备及性能上与现有的影雷者MX显卡并没有任何差异，二者的售价也完全相同。
　　菲利普也要出声卡
　　QSound Labs实验室今日宣布Philips在其新的三款声卡上采用了 QSound 技术，它们预计在今年秋天发布，新声卡的名字是 Rhythmic Edge、 Seismic Edge和Acoustic Edge。 Acoustic Edge 是支持5.1声道的PCI声卡；Seismic Edge可以支持双喇叭、三或四喇叭； 而Rhythmic Edge只是一款传统的立体声声卡。
　　Promise推出 SuperTrak100与 SuperTrak66
　　6通道的 SuperTrak100 磁盘控制卡和4通道的 SuperTrak66卡均配备有 RAID管理工具、SuperCheck超级检测工具, 可以提供存储设备的监测和所有SuperTrak本地网络任何地方的阵列控制。支持RAID级别为0,1,1＋0,3,5，并支持热插拔。
　　AMD CPU也可实现SMP
　　AMD将在2001年1月至2月间推出一款特别的AMD 760MP芯片组。AMD 760MP 将可以使任何Socket A处理器做各类SMP(即多处理器功能)配置。换句话说，雷鸟甚至是毒龙都可以做SMP。 AMD没有把其低价的处理器排除在SMP系统之外。因此这可能形成一个低价服务器市场。
　　威盛发表 Apollo Pro 266 及 KT266
　　威盛 正式发表新一代 Socket 370 及 Socket A 主机板芯片组, 分别是 VIA Apollo Pro 266 及 KT266 芯片. 两款芯片组同样支援新一代的 DDR 266 DRAM，内存带宽达 2.1G/s。
　　DOOM3：现今显卡都不能达到每秒60帧
　　id Software的首席程序师John Carmack声称，即使是目前最快的PC，也不能让DOOM3在全场景场失真以及打开所有特效的情况下达到每秒60帧的速率。}
