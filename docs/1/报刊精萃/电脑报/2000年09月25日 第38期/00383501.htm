《 {显卡测试软件——3D Mark 2000} 》

{　　对于硬件发烧友而言，还有什么比挑选一块称心的3D加速卡令人心烦呢？要选择一块价格合适、性能稳定、具有特色的显卡不是件容易的事。所以，一些软件厂商开始开发专业的3D测试软件，这其中比较著名的有ZD soft的3D Winbech和Madonion公司的3D Mark系列。与3D Winbech所不同的是，3D Mark更加贴近游戏用户，而实际上，现在游戏应用占3D加速卡一个非常大的方面。
　　全新的3D加速芯片上市,不但带来了全新的技术,也使传统的3D测试软件不能满足需要了,所以Madonion公司继续推出了支持更多特性的3D Mark 2000。下面，让我们先看一下3D Mark 2000的特性。
　　1.增加了对DirectX 7.0的支持。
　　2.为Intel和AMD处理器进行优化的全新的3D引擎。
　　3.提供多种细节的游戏测试。
　　4.全新的材质渲染测试。
　　5.新增的复杂多边形测试。
　　6.新增环境凹凸映射测试。
　　7.可选择纹理的色深。
　　8.支援Microsoft Windows20000。
　　3D Mark 2000的界面和操作与3D Mark 99/MAX没什么差异,没有什么新的功能,这不能不说是一种遗憾。
　　3D Mark 2000测试配置最小要求：Intel MMX 166或以上处理器，64M以上内存，Windows 95/98/98 SE/2000操作系统，Microsoft DirectX 7.0以上，一块支持DirectX 7.0的3D加速卡。
　　●测试设定
　　在进行测试之前，我们需要先设定好测试环境。3D Mark 2000的设定工作包括：
　　(1)编辑描述
　　在这里为测试命名，以区别于其他测试项目以及一些个人的资料等等。
　　(2)Select Tests:选择测试
　　3D Mark 2000的全部测试需要十分钟时间才能完成，对于等不及的朋友可以根据需要选择测试的项目。其实，如果你只是希望快速了解显示卡的游戏性能，你可以只选择Game1和Game2游戏的测试。事实上，3D Mark的最后得分也就是由这两个游戏的测试决定的。另外要注意，“Visual Test”项中的“Image Quality”(画质测试)功能只有在3D Mark 2000 Pro版本中才能进行。
　　(3)Options设定
　　在这个选项里可以设定一些测试具体方案，比如循环测试和测试时是否播放音乐等等。
　　(4)Display and CPU Setting：显示卡与CPU设定
　　这个可以说是3D Mark测试的主要控制面板。在这里我们可以更改最重要的测试设定，包括测试用显示卡、解析度、材质色深、Z－buffer深度、缓冲区和CPU的优化功能。
　　●实际测试
　　整个测试分为游戏测试(Game1,Game2)、CPU速度测试、像素填充率测试、多边形处理能力测试、贴图速度测试、凹凸映射测试。还有一项图形质量测试需注册为3D Mark 2000 Pro方可进行。
　　3D Mark 2000对诸如GeForce 256等这样支持硬件T＆L的芯片有良好支持,在设置中如果显卡用到了硬件的T＆L加速,这样的显卡在3D Mark 2000中的得分要较其它芯片多。在系统信息中,你可以查看到自己的芯片是否在硬件上支持T＆L或其它3D特效。不过实际测试发现，3D Mark 2000对DirectX的T＆L支持并不好，因为在3D Mark 2000中Madonion采用了自己的3D引擎，所以采用硬件T＆L并不比PⅢ优化快多少，这也是许多人对GeForce系列T＆L性能提出质疑的根据。这只有希望在下一版的3D Mark中，得到完美解决了。
　　此次Madonion为3D Mark 2000设计了两个全新的游戏场景。Game 1测试是一个飞行模拟游戏的场景,起伏的山峦基本已无法看出是由三角形拼接而成的,而树木外形与我们在3D游戏中常见的完全不同,每棵树都有各自的特征,很难看出有完全一样的。当测试最高细节时,满山遍野都是各式各样的树木,连远景也没有“缩水”。不过如果没有速度较快的CPU或硬件T＆L支持的话，速度根本无法令人接受。
　　当设为最高级别时,每帧最高三角形数目居然达到了7万以上,如果没有硬件T＆L加速的话,现有的流行3D加速卡都运行困难。
　　Game 2运用了MADOnion的Max Payne's光源及其它一些渲染效果,不过除了像木桶、水井、灯柱之类圆柱体已经处理得非常圆滑外,并没有太多的特效,灯光的效果总是不能让人满意。当测试级别为最高时对圆柱体的处理已非常圆滑了,让人无法想到这是由多边形即时处理而成的。
　　由于Game 2每帧中使用大量的物体和光源,虽然三角形用得并没有Game 1中多,但加上更大的纹理,使得速度比Game 1要慢。
　　除了主要的两项测试外,3D Mark 2000还有CPU,像素填充率,纹理填充等测试,其中,CPU 3D Mark测试法较特殊,它的得分由Game 1和Game 2得分相乘后得到,当然,Game1和Game2的帧数取最高细节时的帧数。
　　在原3D Mark 99 MAX的基础上,3D Mark 2000新增了对G400环境凹凸映射的支持。在纹理尺寸方面,3D Mark 2000支持最大64M的纹理测试,不过如果没有256M的内存或是64M显存版的加速卡,是无法通过本测试的。
　　设置好了测试时的选项，只需要在主界面的那个“RUN”按钮上一按就可以进行测试了，测试时间不是很长，测试时的画面也很精彩，大家就当是一边看动画一边做测试。测试完成后,你就可以查看自己配置的3D Mark得分了,如果机器不是太COOL,可别抱太大希望哦!对于拥有PⅢ级处理器或硬件T＆L图形卡的用户而言，得分将比普通机型高一个档次。
　　对于未注册成为Pro的版本,Result Browser是无法使用的,这非常不便,不过MADonion也提供了在网上查询结果的方式,只需点击Online ResultBrowser就可以了,还可以在Madonoin网上与其它网友交换数据,看看与自己同配置的网友得分如何,如果太低,再去看看硬件升级的建议。
　　除此之外，你可以通过3D Mark 2000欣赏到一段使用Mark 2000引擎制作的动画。由于采用了大量的特效和3D技术，效果震撼人心！在测试之余，笔者建议大家千万不要错过这段魅力十足的演示片段。也许再过一年或者半载，我们就能领略到这么酷的游戏啦！
　　●结论
　　3D Mark 2000是一个非常优秀的3D测试软件。它支持目前最先进的3D技术(如T＆L功能)，操作快捷方便，测试画面赏心悦目，一扫以往专业测试软件的枯燥感。详尽的性能和功能测试可以让我们直观地了解显示卡的性能。唯一令人遗憾的是，尽管3D Mark 2000以Gamers Benchmark自称，但它毕竟不是真实的游戏，也并不能代表显示卡在真实游戏中的性能。其次，3D Mark 2000是一个基于Direct3D界面的程序，它无法为我们提供显示卡的OpenGL性能。之所以选用3D Mark 2000，更多的只是想测试显示卡的规格和功能而已。
(EX3D)}
