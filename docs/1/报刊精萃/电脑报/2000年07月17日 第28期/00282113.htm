《 追新一族 》

　　随着高考尘埃落定，暑假正式到来，正是出门旅游的大好时机。小编我也早有些按捺不住，只是无技可以脱身。咳！真是后悔自己当初作学生时去的地方太少，学弟学妹们可千万不要再犯我这样的低级错误！须知读万卷书，还得行万里路！切记切记！
　　言归正传，下面还是让我们来看看上周的软件龙虎榜都有哪些上榜内容。
　　“闪”亮登场
　　尽管没有得到官方证实，Macromedia Flash 5.0 Beta（http://www.macromedia.com）已经可以在少数软件站点里提供下载了。支持XML，可以完成丰富的电子商务功能；嵌入了HTML文件格式和链接，从而有效地设计、配置、丰富了文字WEB应用；最重要的是其全新的编辑方式，通过新的JavaScript，以采用更接近自然语言的方式，有效地生成非常完善的应用程序……真的让人迫不及待想一睹正式版的“芳容”！
　　图像捕捉
　　著名的抓图软件 HyperSnap－DX Pro 和 Flash 32 在上周均有所更新。HyperSnap－DX Pro 提供了专业级影像效果，能让你轻松地抓取屏幕画面、使用DirectX技术的游戏和DVD画面，并且可以去除不必要的背景，在最新的3.62版(http://www.hyperionics.com)中增加了黑白转换功能，并改进了版本自动更新功能；而 Flash 32 不单可以捕捉任何形状的画面，更可将捕捉得到的画片储存成105种不同格式，甚至可以将一组连续画面制作成动画，还提供了多种滤镜方便制作各种特别效果，在其最新版本3.03 (http://www.logipole.com/indexe.html)中修改了在以前的版本中抓取最前端窗口及保存时存在的问题。
　　文件管理
　　Drag and File 这一强悍的文件管理工具于近日推出了4.52c Gold版(http://www.canyonsw.com/dnf.htm)。它可以完全替代Windows自带的资源管理器，除了具有基本的复制、粘贴、移动、删除等功能之外，还建有Zip压缩/解压程序，支持FTP连接，提供了相当方便的磁盘按钮及多窗口操作方式，并可在程序中直接打开“我的电脑”、“回收站”、“控制面板”等等。在其最新的Gold版中还集成了另一组件 Drag And Zip。
　　文件分割工具 1toX (http://www.logipole.com/indexe.html)具有“化整为零”的强大功能，能按照用户的要求将一个大型文件分割为多个文件，以适应不同的存储介质。1toX具有简单易用的操作界面，支持CRC校验、密码保护及拖放操作，并提供了普通、批处理文件及可执行文件三种分割方式，在每个分割片段中还可加入注释。在其最新的2.0版中可同时对多个文件进行操作。
　　在众多的十六进制的编辑工具中，WinHex(http://www.sf－soft.de/winhex.html)可算是一个佼佼者。文件小、速度快，能同时对文件和磁盘(支持FAT16、FAT32和NTFS)进行编辑，可做Hex与ASCII码的编辑修改，具有多文件查找替换、文件比对分析等功能，并支持一般的数值运算及逻辑运算。其最新的9.5版增加了磁盘克隆功能，支持多个位置的热键切换，各项设置参数也改为保存在注册表中。
网络工具
　　著名的离线浏览工具 Offline Explorer (http://www.metaproducts.com)近期频频更新，又推出了Pro 1.3.272 Final，主要解决了以前在进程恢复、已下载网站更新以及解析Javascript时存在的BUG。另外它改进了项目统计时的文件中断处理和Win2000中的IE自动检测，对已下载完毕的项目以粗体显示，并增加了对浏览器NetCaptor的支持。
　　远程登录客户端软件 Windows Telnet Project (http://ctelnet. magnitka.ru)支持 VT100、 ANSI，能让用户同时运行多个进程，支持本地打印，并自带有个人Ftp服务器。其新版1.7.0着重改进了剪贴版程序，只响应Windows内部的剪贴信息，还增加了FTP文件传输起始命令，并对用户输入程序进行了修改。
　　国产精品
　　最新版的网络伴侣 iCompanion 2.02(http://onekey.yeah.net)重写了邮件检测部分，使用了线程技术，并修正了Windows 2000下不能关机和不能启动默认浏览器、第一次连接成功时程序长时间失去响应及退出或下线时出现非法操作等问题。
　　目前在国内，OICQ拥有最多的用户。而最近，一个与OICQ非常相似的名叫“即时语”的即时聊天软件OMMO Beta 0711(http://www.ommo. com)悄然向它发起了挑战。这个由深圳文赛信息网络有限公司出品的软件在功能上比OICQ更多，支持显示朋友在线信息、即时传送信息、即时交谈、即时发送文件和网址。会自动检查是否已联网，如果你的电脑已连入Internet，就可以搜索网友、显示在线网友，还可以显示新闻、天气情况等等。
　　全新的木子MP3播放器 Aries Lar 3.3.0(http://mzstudio.easthome.net)出炉。这一新版本经作者重新编写代码，修正了在某些装NT/2000的计算机上不能正常使用的问题。其新的DLL插件支持方式使得支持的文件类型更多，并容易扩展。作者还添加了许多方便的操作，如特别制作的歌曲管理功能，让你可以对拥有的歌曲文件进行分类管理；几乎所有窗口都支持拖动操作和菜单的快捷选择；各个面板都做了改变，界面更加友好。
