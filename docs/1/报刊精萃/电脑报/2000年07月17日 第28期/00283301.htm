《 也谈[巧改同维电视卡] 》

　　我在2000年第10期电脑报的硬件周刊中曾经看到一篇《巧改同维电视卡》的文章。正好本人也有一块同维TV003型电视卡，于是照上文的说法改进我的电视卡，可是没有成功。由于当时手中也没有万用表等测试工具，只好作罢。
　　前不久，我终于有了自己的DIY利器“万用表”，于是重新开始这项工作（实在是由于电脑中的电磁干扰太严重，电视卡无法正常工作，不得已而为之）。
　　首先，我仔细观察了电视卡，这块卡的金手指很简单，只是在卡的反面有两两并列的八个引脚。我将电视卡插入电脑，开机后测量了八个引脚，发现其电压分布为：
　　引脚编号： 1 2 　3 4 　5 6 　7 8
　　电压（V）： 0 ＋5 　＋12 0 　＋5 0 　＋5 0
　　知道了电压就好办了（《巧改同维电视卡》一文只告诉了1至3号引脚的电压，并且只对这三个引脚加了电，这就是我上次改装不成功的原因）。具体做法如下：
　　1.在市场上买一只功率大于20W、带有＋12V和＋5V输出的直流电源。现在电子市场上常见的符合要求的是“开关电源”，质量很好，而且可为DIYer今后维修其它电脑配件提供高质量的电源。我买的是上海复旦大学生产的开关电源，规格是：＋－12V（1A）、＋－5V（1A）、＋－5V（5A，电压可调），170元，不错吧？如果嫌贵，可买规格简单一些的。
　　2.找一个大小合适的硬纸盒（便于加工），电脑上的一些卡类配件的盒子很合适，也可自制。
　　3.将电视卡的引脚1接电源的负极（如果开关电源的每个电压接口都有各自独立的负极，则必须将用到的每个接口的负极都连到电视卡的引脚1），2、5、7接电源的＋5V（1A）（这三个引脚可共用一个＋5V接口，但前提是此接口的供电必须充足），引脚3接电源的＋12V（1A）。其它的引脚可不管，也可接到电源的负极。我是将电线焊接在卡上的，由于此卡为ISA接口，而ISA接口很快将会被淘汰，所以不如干脆将其完全改装。
　　4.将电视卡固定在硬纸盒中，并留出电视卡的输入/输出接口，再将红外线接收头固定在硬纸盒前面。注意还要在纸盒上留一些散热孔。
　　5.将硬纸盒的外面包一层纱窗用的金属纱（注意不要碰到电视卡），然后将其接地（如接地良好的电脑机箱或暖气片），以防电磁干扰。此步骤也可省略。
　　到此，改造全部完成。检查一下电源线连接是否正确，然后连接好信号线，无须任何调试即可收看电视节目了。另外我想说明一点，改造电视卡会有一定的风险，请你小心，如果没有把握，请不要动手。此方法仅供参考，一切不良后果请自负。如果我的改造有什么不当之处，或你有什么更好的方法，也请与我联系。我的E－mail是danfa@21cn.com。　

　（湖北　辜　翔）
