《 {计算机主板技术简介} 》

{ 一、 微星的“Live BIOS”技术
　　目前普遍使用的BIOS刷新方法过程烦琐，且有几个难点：要求用户了解计算机启动过程，能够正确地辨识BIOS类型、版本号。用户必须明确自己主板的芯片组、跳线情况、版本号等，否则可能选错BIOS升级文件；在DOS环境下进行，要求用户了解DOS命令行格式，还要懂得DOS引导机制和内存分配的相关知识；要求用户了解CMOS原理、BIOS的内容和层次等知识，否则可能在刷新过程中对有关选项作错误的设置；DOS下仿图形的刷新界面不够友善，可能导致误操作。 以上的几个难点之中，任何一处出现问题都将导致刷新失败，造成灾难性的后果。
　　微星的“Live BIOS”技术主要面向不能自如地应用传统方式刷新BIOS的普通用户，它将刷新过程移到Windows下进行，并将上述各难点一并封装在一个“黑匣子”中，使普通用户绕开专业性很强的技术问题，只需接触简单友善的Windows风格界面就能完成BIOS刷新，以最大限度地减小刷新BIOS失败的风险。“Live BIOS”可使用于Windows95/98/Me/ 2000与NT 4.0等平台上。目前已有三款微星主板支持这一功能： 815E Pro （MS－6337） 、815 Pro （MS－6326）和K7T Pro （MS－6330），而微星预计将来推出的所有主板都支持此项技术。
二、 AGP 8×
　　Intel宣布新一代的图像界面规格——AGP 8×，为下一代的个人电脑及工作站订下新的显示标准。
　　AGP (Accelerated Graphics Port) 是由Intel 公司所制订的显示界面，速度由最初的AGP 1× (264 MBytes／sec) 到现在的AGP 4×(1 GBytes／s)，因为AGP拥有高速频宽，广受众多显示芯片厂家的支持，推出支持AGP 4×的不同产品，以满足用户对图像运算、高画质要求的需求。Intel 宣布的AGP 8×，依旧使用 32－bit 的架构，而速度方面则提升至533 MHz ，支持2GBytes／s，是AGP 4×的两倍。速度的提升，即代表了显示芯片制造商能利用AGP 8×的优点，充分发挥显示芯片的效能。
三、Voice Doctor
　　Voice Doctor是主板的一项新技术，它可以监测系统在启动阶段硬件的工作情况。在系统启动的时候，系统会自动诊断硬件配置，其中包括CPU、内存、显示卡、硬盘、软驱和键盘。如果出现了异常的情况，我们就可以听到一些提示的声音，警告用户系统出了故障。
　　当主板出现问题即可用多国语言（包括汉语 、 英语 和日语等）报告问题来源情况，大大方便了 DIYer 的工作，是目前一项新型的安全保护技术！
四、FOC
　　FOC（风扇开关控制）负责在CPU运行的时候检查风扇和CPU是否处于正常状态。在系统进入操作系统以前，H／W可以检测CPU风扇发出的信号，得到它们旋转的信息以后将这个信息发送到BIOS中。如果BIOS得到CPU风扇旋转的信息，它才进入正常工作状态。如果不是，它将通知系统，并立即切断电源。因此，这样一来就可以保护CPU，以免出现过热的情况。这项技术技术是专门为AMD的CPU设计的。
五、ATA/100标准
　　作为下一代硬盘的新标准它可以让硬盘控制器与硬盘缓存之间以100MB／s的传输率进行数据交换，即硬盘的外部传输率达到100MB／s，引用昆腾的话来说，ATA/100接口标准作为一种新的高速硬盘接口，提高了硬盘数据的完整性与数据传输率，对桌面系统的磁盘子系统性能有较大的提升作用。这一规范已经得到了英特尔公司及各大硬盘厂商的支持，并已生产出相关产品。ATA/100接口和数据线与DMA／66一样，也是使用40针80芯的数据传输电缆，并且ATA／100接口完全向下兼容，这意味着凡是支持DMA／33.DMA／66接口的设备完全可以继续在ATA/100接口中使用。ATA/100协议还包含了CRC特性，它能够有效提高高速传输中数据的完整性和可靠性。
六、整合技术
　　自从整合技术在1999年“大行其道”以来，主板整合技术成为了一大新的发展趋势。其原理是将一般单独配置的 AGP 显卡、 PCI 声卡、 PCI Modem 、 PCI 网卡、 IEEE——1394 等设备接口集成在主板上，以提高产品的兼容性和性能价格比。
　　目前比较流行的做法是在主板上集成一块 AMR （Audio Modem Riser ）专用插槽或 CNR （ Communication and Networking Riser ：通信网络提升器）接口以及集成IDE控制器来支持更高的硬盘标准等等。比如，i815主板集成了声卡、显卡以及CNR槽在主板上。还有些主板集成ATA/100控制器来使主板支持ATA/100，并且有的还带有RAID功能。
七、主板能源功能的改进
　　主板能源新功能的改进主要体现在 STR 新技术的发展， STR 技术原出自于笔记本电脑技术，其前身是 STD 技术。STD 就是 Suspend to Disk 的缩写，意思是“ 挂起到硬盘 ” 。其具体过程是将系统一般是 Win98) 运行时的当时状态和相关系统信息保存到存储设备 ( 硬盘 ) 上，此时系统耗能极小，再次开机时可省去大量的系统自检和启动时间，从而迅速恢复到关机前的状态。STR即为Suspend to RAM，意思是“挂起到内存”，原理和STD一样，只是将信息存储在内存上。
八、另外一些方便功能
　　1.PC’99 技术规格：这是由微软、 Intel 等公司共同制定推广的一项业界标准。 PC'99 的认证规定极多。如对总线设计发展的限制，人体工学环保等。在主板设计方面主要规范了产品的设计要求，它提出主板的设计必须符合人体工学的要求。产品布局必须合理，以保证安装者能正常装配使用主板。此外主板各接口必须采用有色标识以方便识别。这些都无疑大大的方便了使用者，使不熟悉主板设备的使用者也能尽快的安装好相应的接口设备。
　　2.多类型CPU主板：这是指一些主板采用多种混装 CPU 接口以适应不同的 CPU 产品。具有代表性的产品是精英公司的“双子星”系列主板产品，该类主板采用Slot 1＋Socket 370 双接口设计，可自行选用 Slot 1接口或Socket 370接口的CPU产品，以达到节省转接卡，方便使用的目的。不过双 CPU 接口在使用时只能择一而用，不能同时使用多个CPU，这一点不同于双、多CPU主板。目前也有不少厂商的主板产品采用了 CPU 双接口设计。不过随着Slot1 CPU的淘汰和 Socket 370 CPU 的兴起，该技术流行程度已大不如前。
九、结束语
　　主板制造商之间的竞争是非常激烈的，大家都在尽力开发更新的技术以使自己的产品在市场上具有优势而处于领先地位，以吸引更多的用户，提高收益。
　　展望将来，各大主板芯片厂商将在围绕前端总线速度的进一步提升上做文章。VIA虽领先一步，但Intel也自有打算，200MHz FSB(系统前端总线)已离我们不远，DDR 主板也已经推出。而在厂商方面，采用同种芯片组的主板性能相差很小，所以，他们还将继续采用并完善各种技术以及推出新技术，使之在激烈的市场竞争中，占尽先机，成为自己独特的卖点。总之，主板的发展方向将是“更快、更全、更强、更方便”！
　　(四川　钟全德)}
