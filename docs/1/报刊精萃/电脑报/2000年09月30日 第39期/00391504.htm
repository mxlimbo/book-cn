《 {虚　拟　之　王 ——Paragon CD Emulator 1.02} 》

{　　用过Logicraft的Virtual CDROM 1.0、Farstone的Virtual Drive2000 5.X、IMSI的CD Copier 2.X等虚拟光驱软件吧？那你就不应该错过Paragon的CD Emulator 1.02。它是一款由Paragon Technology Systems公司制作，运行于Windows 95/98、Windows NT 4.0、Windows 2000下的虚拟光驱软件。虽然Paragon CD Emulator起步较晚。它有着更稳定的性能、更完善强大的功能，而且几乎秉承了前辈的所有优点。
　　同以往的虚拟光驱软件不同，Paragon CD Emulator是基于模拟SCSI设备来实现对光驱的虚拟。大概正是这个缘故，其运行时也非常稳定，并有较低系统资源占用率和较高的执行效率。
　　Paragon CD Emulator 1.02支持添加多个虚拟光驱，并可随意更改盘符；支持数据格式的压缩。值得一提的是，你甚至能在运行过程中随时添加、删除虚拟光驱，而且不需重新启动计算机就能立刻生效，极大地方便了操作。
　　Paragon CD Emulator 1.02有很强的光盘模拟能力，能够很方便地制作并虚拟多种格式的光盘：常见的数据及游戏光盘；带音轨和数据的CD混和光盘；纯CD光盘。虽然，Virtual Drive2000 5.X和CD Copier 2.X也能制作带CD音轨混和光盘，但Virtual Drive2000 5.X只能把混和光盘制作成一个数据映像、一个CD映像，不能合二为一；而CD Copier 2.X又不能识别某些混和光盘（CDI类）。Paragon CD Emulator 1.02则能很容易地为混和光盘制作映像，并能稳定可靠地虚拟成混和光盘。
　　Paragon CD Emulator 1.02增加了方便实用的映像文件编辑功能。你可以轻松自如地在制作的CDI映像文件上增加、删除CD音轨；可以很容易地制作一张CD音乐光盘；还可以从外部导入一个ISO文件制作你所需要的光盘。这个功能可以说是Paragon CD Emulator 1.02最大的一个特色。
　　在制作CD音轨映像时，Paragon CD Emulator 1.02还支持对音轨的MP3压缩，有效控制了映像文件的大小。只要安装了较新版本的媒体播放器，你就能用CD播放器很好播放。怎么样，Paragon CD Emulator 1.02的种种优点足以让你心动了吧！快去http://216.71.163.73/download/vcdetr.zip下载一个来感受一下吧，相信你不会失望。(四川　彭勇)}
