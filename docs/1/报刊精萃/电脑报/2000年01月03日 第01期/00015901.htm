《 {Novell网一线多机上Internet} 》

{　　不少单位或部门正在使用Novell网络。本文介绍如何使网络用户采用一线、一账号而多个用户同时上Internet。
　　一、基本条件
　　Novell网络用户平台为Windows 98浏览器为IE 5.0。
　　二、安装、配置Windows 98
　　安装Windows 98时必须是作为Novell网络用户安装Windows 98，而不是一般单机安装。
　　对于须要专门运行Novell网络应用软件或要使用非ODI方式进行Novell网络登录的用户， Windows 98安装完成后可设置双重DOS启动。完全不会影响原来的Novell网络功能和使用。
　　Windows 98的安装应在进行了Novell网络登录后进行。 这样在安装完成后自动成为Novell网络用户，无需过多的网络配置。
　　安装Windows 98要以ODI方式登录Novell网络。为简要说明问题，这里将Config.sys、Autoexec.bat、Net.cfg文件中必需的命令行举例如下：
　　Config.sys：
　　LASTDRIVE=Z
　　Net.cfg：
　　Link Driver NE2000
　　　PORT 340
　　　IRQ 5
　　FRAME Ethernet_802.3
　　NetWare DOS Requester
　　　FIRST NETWORK DRIVE = H
　　　NETWARE PROTOCOL = NDS BIND
　　Autoexec.bat：
　　CD\NWCLIENT ；Novell 客户软件所在子目录
　　SET NWLANGUAGE=English
　　LH C:\TOOLS\NWCLIENT\LSL.COM
　　LH C:\TOOLS\NWCLIENT\NE2000.COM
　　LH C:\TOOLS\NWCLIENT\IPXODI.COM
　　LH C:\TOOLS\NWCLIENT\VLM.EXE
　　H:；进入第一个网络驱动器
　　Login LWF；以LWF身份进行网络登录
　　简要说明：NE2000兼容网卡的接口地址为340H；中断号5。因现在硬盘很大，通常分为C、D、E、F盘，光驱为G盘。因此第一个网络盘符从H开始。其它命令的解释和文件来源就不在这里赘述了，请参考有关书籍。
　　如果登录成功，就可准备开始安装Windows 98。否则应检查Novell网络的软、硬件。
　　网络登录必须包含有本机的光驱。 否则无法从光盘上进行Windows安装。登录成功后从光盘上开始安装Windows 98。
　　安装Windows 98 过程中应注意的是：当安装基本结束，重新启动机器进入Windows 98后，提示进行Novell网络登录。在登录名称栏目中输入正确的网络用户名。并且检查网络服务器名是否正确，输入正确的口令。在“我的电脑”中应能查看到网络服务器的目录。
　　在通过拨号上网或以其它方式与Internet网络相连的微机上，应在安装Windows 98 时在“Internet 工具” 中选择安装“Internet 共享连接”。其它网络客户机不必安装。
　　三、Windows 98 网络配置
　　按上述方式安装好Windows 98 后，还要继续增加一些协议和服务。 其目的是为了让Windows 98的对等网络与Novell网络共存。
　　首先用鼠标点击“开始”，在“设置”栏内打开“控制面板”。双击“网络”，打开后检查是否已安装上述所需协议。若没有安装，点击“添加…”→选择“协议”→点击“添加...”→左栏选择“Microsoft”－右栏选择相应的协议－点击“确定”。重复选择“添加…”，安装所有所需的协议，计有NetBEUI协议、TCP／IP协议、IPX／SPX协议。
　　返回配置菜单，应出现“TCP／IP”→“网络适配器”、“NetBEUI”→“网络适配器”、“IPX／SPX”→“网络适配器”等相应行。
　　再点击“添加…”→选择“客户”→点击“添加…”→左栏选择“Microsoft”→右栏选择“Windows 网络客户”→点击“确定”。
　　为使对等网络中可以相互访问，共享资源，还要安装网络服务。点击“添加…”→再点击“服务”→点击“添加…”→左边栏目中选择“Microsoft” ，右边栏目中选择“Microsoft 网络的文件和打印机共享”。按“确定”按钮，回到网络配置。按下“文件和打印机共享”按钮，选择“允许其他人访问我的文件”和“允许其他人打印到我的打印机上”。注意：不能选择添加“Novell网络文件和打印机共享”服务。否则不能安装“Microsoft 网络的文件和打印机共享”服务。Windows 98对等网络将不能实现文件和打印机共享服务。如果提示已经安装了“Novell网络文件和打印机共享” 服务， 请删除它。然后再重新安装“Microsoft 网络的文件和打印机共享”服务。
　　此外，对每台客户机要设置网络标识，以便于相互连接访问。在网络配置的“标识” 标签下， 输入本机的计算机名，如 LGW、LGW2、LGW3等等，其它两栏可不输。注意，这里的计算机“标识”名与Novell网络的用户名可以不同。因为它代表的是Microsoft 网络的“标识”名。按“确定”，完成网络服务配置。
　　硬件和软件安装完成并正确配置后，重新启动机器。完成了Novell网络用户登录后， 在Windows 98的桌面上打开“网上邻居”，可以看到Novell网络映射盘符和联网的各台微机。为了其它微机能查看本机的磁盘文件和共享本机的其它资源，必须将磁盘（文件目录）设定为共享，允许其它用户访问。当然可以设定访问口令以增强安全性。具体做法是打开“我的电脑” 后， 用鼠标点击要打算共享的磁盘（目录）。再从“文件”下拉菜单中选择“共享”。选定“共享”和规定“共享名称”。如果网络上微机较多，应注意共享名的区别。
　　四、配置一线多机上网
　　如果用户不是安装的Windows 98第二版或是愿意采用其它一线多机上网软件，可以选择在连接Internet的微机上安装WinGate、SyGate、WinProxy...等软件，读者可参考本报去年以来的多篇介绍文章。
　　本文就使用Windows 98第二版的“Internet 连接共享”作简要说明。注意，只要在与Internet网连接的客户机上安装“Internet 连接共享”。
　　在安装Windows 98时，从Internet 工具中选择“Internet 连接共享”。也可在安装结束后从“控制面板”→“添加和删除程序”→“Windows安装程序”→“Internet 工具”中再选择“Internet 连接共享”。提示要插入光盘。
　　安装完Internet 连接共享后，须要进行配置。从控制面板上点击Internet 图标，配置Internet 属性。在Internet 属性的连接标签下，局域网（LAN）设置框内会出现一个“共享（H）”按钮。点击它后出现Internet 连接共享的配置菜单。应当选择：“启用Internet 连接共享”；“在任务栏显示图标”。 这样在Windows 98桌面的右下栏目中会出现一个地球和计算机小图标， 表示启动了Internet 连接共享。用鼠标右键点击它可以随时改变设置。
　　在“连接Internet 时使用：”滚动框中选择与Internet 连接的方式。框中应出现“拨号网络适配器”或其它连接适配器，例如ISDN等。
　　在“连接到我的家庭网络时使用：”滚动框中选择与内部网络连接的方式。框中应出现“NE2000 Compatible”。
　　此外，须要对共享微机TCP／IP协议的相关属性进行配置。在控制面板中点击“网络”，在“配置”标签下与TCP／IP协议有关的有三项。
　　1.TCP／IP（Home）→ NE2000 Compatible；
　　2.TCP／IP（Shared）→ 拨号网络适配器；
　　3.TCP／IP → Internet 连接共享
　　用鼠标选中一项，再点击属性进行配置。应按上述顺序进行配置。
　　当窗口出现后选择“IP地址”标签。选中“指定IP地址”。在“IP地址” 栏中填入规定的IP地址。
　　如果“启用DNS”，“域”不须填写。主机为本机的标识名称（机器启动时显示的，登录名称，应与该机网络设置中“标识”中的名称一致），如lgw等。
　　未强调说明的标签下的设置，应保持为空白或缺省值。最后选择确定。
　　连续选择“确定”，退出网络配置。重新启动Windows 98后设置生效。
　　说明
　　IP：MS网络内部IP地址。GW：MS网络内部网关地址。
　　DNS：域名服务器地址 。WINS：WINS 配置。
　　Host：主机名。登录名：Novell网络客户登录名。
　　如果设定了IP地址，子网掩码均为255.255.255.0。
　　五、上网软件的配置和使用
　　以IE5.0为例。 共享微机上仍是使用拨号网络连接，其它客户机必须设置使用局域网连接。
　　从控制面板上点击Internet图标， 进入Internet 属性设置。在连接标签下点击“局域网配置”按钮，选择“自动检测设置”。不要选择“使用代理服务器”。
　　一般应用软件不须特别设置。
　　首先打开共享微机， 包括Modem电源。按上述安装、配置好后启动任何一台客户机的浏览器，共享微机应能自动拨号（已建立拨号网络，设定用户名和密码）。共享微机与Internet网连接后，便可从任一台客户机上网浏览。
　　客户机之间还可以相互共享资源，直接传递、拷贝文件。}
