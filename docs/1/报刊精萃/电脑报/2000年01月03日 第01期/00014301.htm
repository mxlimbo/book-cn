{　　一、ActiveXControl Pad 1.0 让你轻松在网页中加入ActiveX
　　ActiveXControl Pad 1.0是个免费网页编辑工具，大小约2.7M。它可以让你建立一般HTML无法建立的互动及多媒体网站。你可以轻松地在网页中加入HTML Code、ActiveX、HTML Layout以及VBScript或JavaScript。
　　从功能上看，ActiveXControl Pad有以下特点：ActiveX Control Pad使用HTML文件作为主要的编辑环境，所以你仍然可以用HTML Source Editor直接编辑HTML；你可以使用ActiveX Control Editor在网页中加入ActiveX控制项，如：TextBox、Scrollbox。然后，你还可以编辑控制项的内容，ActiveX Control Pad会在HTML里面加入Object标签(Tag)；ActiveX Control Pad加入了HTML Layout(排版)的概念。HTML Layout是一种“所见即所得(WYSIWYG)”的设计方式。你可以正确地指定控制项的大小、位置并排列它们；利用Script Wizard你可以指定控制项的“事件(Event)”以及动作。你也可以在VBScript或JavaScript上建立自订的Script。ActiveX Control Pad会自动在HTML中加入Script标签。

　　三、Dutch's HTML Frames 让分割页框的制作变得轻松
　　这是个大小为2.4M的共享软件。通过Dutch's HTML Frames 1.0，网页设计者只要点点选选，就可以制作出符合需求的分割页框，完全不需自己动手键入Tag，快速又轻松。
　　分割页框(Frame)是目前网页上极为普遍采用的做法，它可以在同一个网页上同时显示多份文件，对于浏览者来说，不必在不同的网页之间频繁切换，非常方便。不过，对网页设计者来说，分割页框的工作并非那么轻松，特别是尺寸、配置等格式上的调整，那可是很费时间的。
　　Dutch's HTML Frames可以让设计者通过图形操作界面建立网页上的分割页框，而且包括各种页框大小、页框连结(frame URL)、边界间隔(margin)、卷轴属性(scrolling)、边框(border)等设定一应俱全，皆可供设计者任意微调，并自动输出成HTML Code。因此，设计者完全不必键入任何标签(Tag)，甚至不须了解HTML语法，便可简单快速地制作出符合需求的分割页框。
　　事实上，Dutch's HTML Frames也同时是一个文字式的网页编辑器，设计者可以直接在编辑区上修改HTML Code，而且可通过语法标签清单以点选的方式加入Tag，省去原本一些烦人的“key－in”动作。另外，Dutch's HTML Frames也提供“In－Line Frame”的功能，可以制作出IE所支持的“floating frame”。

　　四、可以加快网络传输及浏览速度的FastNet99 3.0
　　FastNet99是个大小为2M的自由软件。它让你的电脑上有一份自己的IP地址清单，大大减少浪费在查询DNS(域名管理系统)上的时间。
　　影响网络传输速度的因素众多，除了线路畅通与否之外，和硬件功能是否调节到最佳状态，都有直接关系。因此像MTUSpeed和iSpeed等软件都是针对它而调整设定，以期发挥硬件连线时的最佳效率。不过，还有一项会影响网络传输速度的因素也值得留意，即连线时花费在查询IP地址上的时间。
　　用户都了解要去浏览一个网站、下载文件、连上IRC甚至收发电子邮件，这些动作都要靠连接上某个IP地址来完成。由于IP地址均为数字组成，很难记忆，因此用户多是键入URL的文字地址(如www.toget.com)，然后再通过连上DNS主机，来确定与文字URL相对应的数字IP，最后才完成连接的工作。因此，任何DNS上都会有一份清单，列明这样的对应表。全世界的网络IP何止千万，因此就算是再大的ISP，其DNS也不能掌握全部的IP。若用户键入的URL刚好不在名单之列，只能再询问更上层的DNS。世界上只有5个称为“大师级”的DNS有最完整的清单，因此连线时往往有一部分时间，是花费在来往于这些不同层级DNS的查询上。FastNet99的设计便是让用户可以在电脑里储存这样一份清单(虽然算不上是“大师级”)，以减少用户花在DNS上查询的时间，让连线速度无形中加快不少。
　　FastNet99有几个方法来建立IP地址清单。首先是该软件安装后本身就有一个近七百多个IP地址的清单(称为“host”档)，用户可以直接汇入Windows目录底下的同名文件。另一个是用户可以随时加入常去常用到的URL，FastNet99只要“确认”(verify)无误后，便会自动加入原有清单，下一次连线时便无须再向外部的DNS查询，直接参考本端电脑里的清单即可。第三个方法是FastNet99提供了“扫描”(scan)IE的“我的最爱”和Netscpae的“历史名单”(history)及“书签档”(据说下一版才会支持扫描IE的“历史名单”)，自动“确认”后也会加入电脑里的清单中，这样让用户连接常用的地址时，可以提高速度。最后则是如你有办法弄到DNS上的清单，可用软件里的“host merge”程序，让自己的电脑变成与DNS无异，把花在来往于DNS上的时间减到最低。
　　就一般用户来说，其实无须这么大的清单，而FastNet99可让用户自行加入地址的设计，便是让不同用户根据自己连线的习惯和需要，创造属于自己的小型DNS，不用太浪费网络资源。想要改进连网速度的用户，不妨考虑试试FastNet99的效果如何。最后要特别指出，这款软件无需常驻内存，只要更新名单后，电脑会自动参考之。常驻内存唯一的好处只在于让用户连网时，能随时加入自己喜欢的IP地址，以备日后之用。
　　下载网址：http:∥www.toget.com.tw/article/accelerater/19990205_1035.html。

　　六、制作网络广播节目的简单工具RealAudio Encoder (PC) 3.1
　　RealAudio Encoder是个大小仅824K的自由软件。它采用了让网络也能播放广播节目的最新技术。现在最新版的RealAudio Encoder不但能将现有的声音文件转换为网络可以播放的格式，而且使用28.8K的“猫”也能收听到立体声；如果用ISDN或者LAN收听的话，更能获得接近CD的音质。
　　RealAudio Encoder是一个很容易操作的网络广播节目制作软件。它可以将现有的声音文件，或者立体广播节目转换为RealAudio格式。然后，就能通过RealAudio Server播放，用RealAudio Player收听。
　　RealAudio Encoder可以转换的现有声音文件格式包括.wav、.au、.pcm。如果你所要制作的格式不是以上所说的格式，就必须先用其它的软件转换，才能用RealAudio Encoder制作成广播节目。另外，也可以将广播节目连接到电脑的声卡上，直接将广播节目用RealAudio Encoder转换为RealAudio格式。

　　九、功能强大的离线阅读软件 BlackWidow
　　BlackWidow是一个功能强大的离线阅读软件。当然，你可以使用BlackWidow直接进行网页浏览的工作，由于它的界面设计与Internet Explorer十分相似，所以使用很方便。
　　在BlackWidow中浏览网页时，你可以指定在网页中的哪些部分将显示给你看，例如，可以指定只要看网页上的文字、图片等等。另外BlackWidow还有很多神奇的功能，当你在浏览网页的同时，BlackWidow便可以将对方整个网站的架构图勾勒出来，这样就不需要花费太多的时间搜寻网站的内容，只要通过架构图便可以直接找到自己所需要资料网页的所在位置了。

　　十、网站Mirror(镜像)制作专家 FtpRacer 99 1.2
　　如果你对网络不陌生的话，相信一定听过“Mirror”(镜像)这个名词，Mirror就是将远端网站中的东西复制一份到你的局域网Server或者是较近的服务器上，而FtpRacer 99正好是个专门制作Mirror的专家。
　　FtpRacer 99是个1.4M的共享软件，专门用来进行Mirror工作的软件，特别是针对FTP Server。FtpRacer 99可以将远端服务器中的资料自动下载到你的局域网络中的FTP Server中。除了一般的下载工作外，FtpRacer 99还具有排程功能，通过一些简单的设定，FtpRacer 99便会定期自动地与远端FTP Server连线，并将两端的文件同步化，并且可以保有相同的储存架构。
　　在FtpRacer 99中还有一些相当实用的功能，比如说可以检查两端同名文件的大小，ZIP压缩格式中文件的内容。另外，还提供了文件压缩的工具，让你下载的文件不会在服务器中占太多的空间。
　　FtpRacer 99还支持远端登录管理的工作，可以在FtpRacer 99之中利用Telnet的模式登录到远端的FTP Server中，然后直接控制整个Mirror动作的进行。这对网管来说是再方便不过了，因为你将不需要亲自到服务器面前就可以进行文件的维护工作。它还同时支持多个FTP Server登录管理及Mirror的能力。你可以同时管理多个Mirror工作的进行。另外，整个Mirror的工作结果以及进度报告也可以通过IRC与E－mail两种管道进行回报，可以随时掌握最新的工作状况。
　　本版的FtpRacer 99支持多种语系的作业平台，包括英文、中文繁简体、日文、意大利文、法文、德文以及俄文等等，让你可以在不同语系的作业系统中通行无阻。
