{　　Nerf ArenaBlast
　　按“～”然后输入以下密码：
　　god = 无敌
　　allammo = 每把枪999颗子弹
　　fly = 飞行模式
　　ghost = 穿墙
　　walk = 关掉飞行和穿墙
　　杀气冲天
　　按回车键，然后输入GETGOLD，就可得到金钱
　　便利速食店
　　把店名改为Yuki按Enter键再进行征人，不久之后徐怀钰就会来应征
　　苍穹守护者
　　游戏中按以下键：
　　Q = 加生命值
　　W = 加魔法值
　　地狱门
　　修改捷径成C:\Program Files\智冠科技股份有限公司\地狱门\Game.exe yoki
进入游戏按 Enter键会出现一个可输入密技的对话窗
　　FIGHT =无敌
　　MAGIC =可学得全部的魔法,使用时不扣魔法值
　　RAISE ＃=各主角升级一次 (＃为要加的级数)
　　GETOBJ ＃ =得到物品 (＃为物品编号)
　　GOLD ＃=得到金钱 (＃为要加的数目)
　　FOG =迷雾开关
　　MAP =游戏地图
　　SETEVEN =设定事件
　　VIEWEVEN =观察已设事件
　　VIEWMSG =观察讯息流向
　　SCREEN =截取屏幕画面
　　BUY ＃=买卖系统测试 (＃为要测试的项目)
　　印地安纳·琼斯与恶魔机器
　　游戏中按F10，然后在命令窗口中输入以下密码：
　　talkit_marion on = 无敌
　　talkit_marion off = 取消无敌
　　urgon_elsa = 所有武器
　　azerim_sophia = 加生命值的物品
　　nub_willie = 过关提示
　　mem = Show Memory
　　version = 显示游戏版本
　　polys = Show Polygon Rates
　　明星志愿2
　　在选择女主角的地方, 选择自订
　　名字姓输入: 侯
　　名字名输入: 湘婷
生日输入: 8月27日
　　血型输入: A (或皆可)
　　这样在游戏一开始金钱会有50万元，所有属性起始值为5000点
　　Pharoah 法老王
　　让人口统计数字中包括无家可归者
　　游戏中按住[Ctrl]＋[Alt]＋[Shift]＋C，然后输入“help homeless”
　　立刻过关
　　游戏中按住[Ctrl]＋[Alt]＋[Shift]＋C，然后输入“pharaoh's tomb”或“Pharaohs Tomb”
　　霹雳英雄榜
　　无限赚钱
　　至业途灵处购买霹雳紫水晶与长石粉
　　再至秦假仙处炼成铜仙丹……炼制成本共300金
　　铜仙丹卖掉可得640金　净赚340
　　由于物品无限,此法可无限使用
　　七个王国2
　　在游戏中先输入 !!＃＃％％＆＆　然后输入以下秘技:
　　CTRL－B = 修复建筑
　　CTRL－, = increase frythan combat skill
　　CTRL－! = increase frythan combat skill
　　CTRL＋M = 显示地图
　　CTRL＋T = 得到所有科技
　　CTRL＋U = 切换 Mortal/Immortal 君主模式
　　CTRL＋A = 出错信息模式
　　CTRL＋D = AI信息
　　CTRL＋　= 得到1000食物
　　CTRL＋C = 得到1000金钱
　　CTRL＋Z = 快速建造模式
　　CTRL＋; = 在选中的城市中得到10个居民
　　CTRL＋＋ = 能量全满
　　CTRL＋J = ???
　　ALT＋CTRL＋E = 损失10点声望
　　ALT＋CTRL＋R = 得到10点声望
　　ALT＋CTRL＋K = 选中的建筑遭受20点破坏
　　ALT＋CTRL＋J = 选中的建筑修复20点耐久度
　　ALT＋CTRL＋X = 损失1000金钱
　　ALT＋CTRL＋C = 损失1000食物
　　轩辕剑叁
　　隐藏武器
　　在游戏开始的“威尼斯广场”上连续敲钟20下，就可以得到攻击力＋70的“绝情猫袖箭”。
　　无条件使用绝招
　　在战斗时，行动值满了选单出来后按4即可无次数限制使用绝招。
##2 帝国时代2
无限农田
在组队对战模式中，当盟军选取农民造田时，地面上出现农田0％的轮廓之后，停止建造。然后用自己的农民上前将农田建造完成，你会发现这块农田已无粮食容量显示，实际上已经变为无限粮食的农田。
注：使用此秘笈带来的后果请自行负责^O^。}
