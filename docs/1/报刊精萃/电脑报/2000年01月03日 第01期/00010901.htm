《 {春色满园关不住--2000年软件业发展展望} 》

{　　按照大多数人的看法，2000年是一个新世纪的开端，在这意义重大的一年中取得好的成绩，无疑将给未来的十年、百年的发展开一个好头。为了抓住这个千载难逢的市场契机，从软件巨头微软，到代表民族软件产业的金山、铭泰，各个软件厂商都使出浑身解数来研制自己的XXX 2000，准备在2000年施展一番拳脚。
　微软的大手笔
　　2000版软件宣传最火、声势最浩大的莫过于Windows 2000了。试用版本在1999年已经看到，正式版尚在千呼万唤、犹抱琵琶半遮面之中。“这是一个既能满足个人用户需求又能满足企业级应用需求潜力，将使NT比过去更为强大的、灵巧稳定的新一代操作系统。”这是著名IT人士Zdent对Windows 2000的评价。操作系统Windows 2000中80％的源代码都经过重新编写，它的系统功能有了很大的扩充，稳定性得以进一步的增强。
　Windows 2000新增的高级功能主要有:
　　1.实用的多语言技术。Windows 2000中多语言技术的出现，终于使用户能够在同一套Windows系统中查看、编辑和打印不同语言的文档。而Windows 2000 Professional的多语言版本，甚至能够在同一套Windows系统中，为不同的用户切换到不同语言的用户界面上去(包括菜单、对话框和帮助文件)。
　　 2.强大的IE5.0 。Windows 2000中的许多方便的功能与IE5.0是分不开的，与IE4.0相比，IE5.0增加了很多更体贴、更高效的功能。
　　3.支持VLM（超大内存）。
　　4.活动目录。活动目录是Windows 2000服务器版中最重要的功能。它是Windows 2000 Server可扩展和调整的目录服务。它存储有关网络对象的信息并使管理员和用户可以方便地查找和使用该信息。活动目录的目录服务，将结构化数据存储作为目录信息逻辑和分层组织的基础。
　　5.管理工具 。Windows 2000中的管理工具是由Windows NT 4.0服务器版发展来的一组管理程序，对于系统管理员来说，这些工具是必不可少的。
　　6.更广泛的软硬件兼容性。
　　当然，我们并不能因此忘掉去年出尽风头的Linux操作系统，它们在传统的操作系统面前充满了勃勃的生机。虽然它们具有强大的功能与无限潜力，Red Hat与OpenLinux现在都还不是大部分用户的理想选择。但是应该注意Corel Linux，一个去年底出现的新套件。Corel计划要将Debian套件与KDE图形使用接口结合，再加上新设计的工具，使Linux更容易安装、使用与设定。如果有更多Windows应用软件的Linux版本，将会对它的发展有很大的帮助，但是现在只有少数几家大厂商承诺要支持Linux。其中Corel已经推出了WordPerfect办公室套餐软件、CorelDraw绘图软件的Linux版本。每个人都在猜测Linux的命运。不过，如果它继续维持便宜、稳定的特性，而且变得更容易使用，肯定会引起一场操作系统的大风暴。
　　Microsoft 在2000年的大手笔，不仅仅是Windows 2000，还有Exchange 2000和SQL Server 2000。Exchange 2000基于Windows NT Server的基础上，为企业间的信息交换，协同合作提供了更加强大的解决方案。作为SQL Server 7.0的升级版本，SQL Server 2000将充分利用Windows 2000操作系统中的目录服务，使数据库的管理变得更为容易，并预计在明年上半年上市。
　网上争夺忙
　　一直与微软抢夺浏览器市场的网景通讯公司的动作则显得有些迟缓。被美国在线公司（AOL）收购一年后，尚未推出第五版的 Netscape Communications浏览器软件，而其测试版可能要再等两个月才发表。
　　Netscape Communications 4.7版与IE 5.5版看起来差别不大，但技术差距却很大。为迎头赶上微软，网景正设法从平地再建高楼，以更苗条、更标准化的技术架构，取代网景传统的软件。新版软件由独立的组件组成，更方便程序开发人员用来搭配他们的应用软件。据悉，Netscape推出延迟的原因之一，可能是为引进新功能，包括一种使用者接口技术，称为延伸使用者接口语言（XUL），还有合并邮件与实时传讯功能。
　　虽然推出日期数度延后，但AOL互动服务事业组总裁Barry Schuler表示，该公司仍支持此浏览器。AOL在新年过后会宣布新的浏览器策略。
　　可以看出，软件发展的趋势，就是更友好，更方便快捷，让用户更易于使用。最近，Net2Phone公司发布了Net2Phone 10.0测试版，新版软件对原有版本的界面进行了修改，使按钮的效果更加直观，并增加了网络传真和语音邮件的功能。Net2Phone软件的语音邮件功能，可以让用户编辑语音短讯并以附件形式发送。相信它的正式版在2000年就能和大家见面，上网打电话会变得更容易。 CMGI投资公司与Akamai公司将合作开发智慧型网络软件，使网络内容能更快地传送，让广告自动送到特定消费者，作市场分析等，将对网络商店更有利。
　　去年末以低价位发行的《东方网神》世纪号，能帮助大家更加方便地安装畅游网络所需要的各种各样的工具软件，浏览器、下载软件、FTP软件， mIRC、ICQ等聊天必不可少的工具，为大家省去很多安装和设置不顺利的麻烦。在《金山快译2000》的刺激下，《东方快车》也加快了它的“前进速度”，推出《东方快车》世纪号，并以29元的低价发行。看来在2000年两大翻译软件将会继续“较劲”，希望能研制出真正能够满足用户需要的全文翻译软件。
　图形软件有变化
　　图形制作软件相对来说就相当多了，大家熟悉的就有ADOBE的PhotoShop系列、COREL的CorelDraw系列、微软的PhotoDraw和 ULEAD的各个系列。估计在今年的图形软件市场将会出现十分激烈的竞争。这是因为ULEAD和ADOBE的合约将在2000年到期，而PhotoShop中的不少应用技术均来自ULEAD，在2000年解禁之后的ULEAD势必会推出一个全新的产品。以ULEAD 的一贯风格，我们相信这一产品将会是小而精巧，并且功能将会十分强大，到那时PhotoShop的地位将会受到挑战。
　　随着多媒体在网络中的广泛应用，图像编辑压缩技术也就更为大多数人重视了。东芝将于2000年1月14日上市面向个人用户的动态图像编辑软件“AQU－Pa!!”。该软件的最大特征是，可以在动态图像上增加文字声音等信息，并采用国际标准的动画压缩技术MPEG－4来压缩数据。MPEG－4与目前普遍使用的MPEG－1方式相比，压缩后的数据容量可以减少到1/20以下。该软件向用户提供视窗，只要点击画面上的编辑按钮，就可自动启动电子邮件，将编辑好的数据作为附件生成新邮件。由于输出格式采用MPEG－4规格压缩的独立规格。压缩后的数据显示需要专门的显示软件。
　网页制作工具继续激战
　　网页制作的流行是近几年的事，现在制作网页已经没有人只用EDIT来编辑了，取代它的是一系列的工具，其中以用微软的FrontPage系列和Macromedia 的Dreamweaver、Fireworks、Flash为多。在去年底，这些软件均进行了升级，编辑网页的功能明显强大了，特别是FrontPage2000，它的图像编辑能力提高了不少，支持Word和PowerPoint文件的直接插入，可以在最短的时间内做出一个像样的网页，大大方便一般的网页制作的初学者。Macromedia公司对自己产品内的相互支持就做得更绝了，在Dreamweaver 3中的图像编辑中，将会默认调用Fireworks 3来编辑。这样看来，在今年Macromedia还会推出Flash4的升级版，里面的“对象”也将会用调用Fireworks 3来编辑，这样它就做到了产品的大会师，有机会垄断网页制作工具的市场了。这样一来ADOBE 和COREL的日子就更不好过了，它们将很容易又失去一批用户，所以这两家公司很可能会在它们的下一代网页编辑工具中、仿效Macromedia，巩固自己的市场。而就微软来说，面对Macromedia公司的攻势，它很可能会采取暂时妥协的策略，今年在FrontPage 2000中加入补丁，对使用Macromedia产品来编辑的网页加以支持，增加兼容性。
　电子商务带来的革命
　　电子商务去年很火爆，但这还只是一个前奏，2000年电子商务会在世界，特别是中国更加风行。许多公司都意识到这一点，纷纷出台了相应的方案。
　　美国太阳微系统公司（Sun）于去年12月6日发布了新版本Java标准和Java开发工具。Sun发售的“Java 2企业版”包含了多种该公司开发的软件技术，为软件开发商利用广受欢迎的Java编程语言，开发商用软件提供了更为一致的环境。“Java 2企业版”还包括一本内容详细的用户手册，为客户编制基于Java的电子商务软件，提供了极大的方便。
　　微软公司与Sun也不约而同地发布了新的XML技术。微软公司发布了BizTakl架构，该架构能够帮助不同行业使用的电子商务系统，例如银行业或制造业等，都可通过这种XML网络标准交换资料。而Sun发布的技术结合了XML与Java程序设计语言的技术，可以让软件开发人员构建应用软件，同时运用两种技术。
　　借这股东风，保护电子商务的防毒软件也应运而生。NAI近日发布了WebShield for Solaris4.0新版本。WebShield for Solaris 4.0是在网关上保护电子商务的新一代防毒软件。WebShield for Solaris是第一个集防护Internet病毒、恶意代码和目录安全于一身的软件。
　　相信在2000年，随着电子商务的发展和普及，以上技术会被广泛地应用，并不断地发展和更新。
　　今年的软件市场的竞争，将会达到白热化的程度。产品功能上的相互比拼，将会比以往都激烈。同一公司内的产品的相互联系会越来越密切，而对其他公司产品的兼容性将会做得更好。所谓适者生存，到底哪些公司将会脱颖而出，我们拭目以待吧!}
