《 {软件扫描} 》

{ 共享类
　　Cool Edit 2000
　　Cool Edit 2000是Syntrillium公司出品的顶级数码音乐处理软件，其强大的功能令人叹为观止。它内置了目前世界上最完备的音效，提供了三十多种特殊效果，对世界上绝大多数的音乐文件格式提供了支持，具有完整的声音与音效的处理功能。对于已经完成的乐曲或音乐作品，可以经由各种特效处理，剪接编辑而成为新的作品，处理速度上是相当的流畅。同时也支持外接键盘及其他各种外部音效设备。目前，其最新版本为Cool Edit 2000 v1.0。
　　评测室点评：Cool Edit在操作上非常简单，即使没有任何电脑基础，使用Cool Edit 2000也不会有太大的困难，真正的障碍反而是你的音乐细胞。可惜的是，似乎没有办法直接处理MP3文件，得先转换成WAV文件才行。
　　ICQ 2000a
　　ICQ在新世纪的钟声即将敲响之际，终于推出了ICQ 2000，这对广大ICQ用户来说无疑是一大喜讯。总体的感觉是，在功能上有所扩展和增强，界面和整体设计上考虑更加周到，更富于人性化。在界面上最显著的变化是新增了“Quick shortcut bar”快捷操作条，可以自行定义，方便地实现你常用的各项操作或调用其他的应用程序，此外还增加了ICQ邮件及今日信息按钮；在功能上针对初级和高级用户分别进行了细化，新增了帮助信息卡片功能，提供了两组不同的消息发送模式，并可以对没有上线的朋友发出邀请。目前，其最新版本为ICQ 2000a Alpha version。
　　评测室点评：ICQ 2000目前仍处于测试阶段，但已经有了许多可喜的变化。想要尝鲜可得抓紧，因为ICQ在初期会对使用ICQ 2000的用户数量有所限制，定期开放。记住先得安装ICQ 99a。
商业类
　　Director
　　用Director可以轻松地制作出高品质的网页动画、光盘启动画面及丰富的简报、广告、产品展示、教学指导、游戏等各种多媒体形式。除了可以处理最基本的图形和文字信息之外，它还支持对音频、视频等各种媒体形式的合成处理，让你的创造力毫无约束、任意发挥。
　　它具有可视化的集成开发环境，支持直接的拖放操作和强大的文宇编辑语言；预置的100多种Behavior资料库可供用户直接套用，轻轻松松就能做出令人心仪的作品，开发效率极高。多变的视觉效果是Director最令人欣赏的地方。在Director中所有的对象都可以经旋转、倾斜等变换而变得生动无比。而动态的alpha channel能够以透明度与渐变的方式，完美地控制视觉效果的展现。支持Internet标准则是Director的另一个重要特征。Director能够在网络上发布丰富的多媒体作品，直接整合现有的HTML文件信息，支持多媒体的超文本链接方式，可与XML直接进行数据交换，制作出具有“智慧”的导航框架结构。目前，其最新版本为Director 7.0。
　　评测室点评：创建和发布多媒体产品的工业标准，是进行多媒体开发的绝对首选，非常适于电子商务领域的开发。
　Dreamweaver
　　伴随着IE的免费大放送，捆绑其中的FrontPage一度成为了网页设计的主流，人们对它有太多的抱怨却又不得不凑合着用。可伴随着“能让你美梦成真”的Dreamweaver的出现，情况大有改观。
　　它具有“所见即所得”的集成开发环境，内置各项强大的功能让你轻松地做出各种交互式的网页和动态效果；新增了快速标记编辑器(Quick Tag Editor)，可对页面窗口中元素周围的标记进行编辑；改进的HTML查看器(Inspector)现在支持对行进行编号；可自动同步远程和本地的网站的版本；在历史面板(History Palette)中可以重复多个步骤，并可进行保存；其开放式的对象嵌入功能使你可以将各种需要的数据信息直接嵌入到网页中，另外还支持各种外挂插件。Dreamweaver迅速崛起，成为了网页开发平台的首选。目前，其最新版本为Dreamweaver 3.0。
　　评测室点评：能为网页的设计与实现提供快速的、专业级的解决方案，网页开发人员的必备工具。}
