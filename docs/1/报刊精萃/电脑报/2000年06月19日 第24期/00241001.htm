《 离线浏览大师--TELEPORT PRO 》

　　Teleport Pro是Tennyson公司开发的一款功能强大的离线浏览器。它可以根据你的需要下载一个站点的全部文件(甚至可以为你做一个镜像站点)，也可以根据你的设定进行有选择的下载。通过Teleport Pro Scheduler，你可以进行自动下载，只要对你要下载的时间及一些参数进行设置，电脑便可根据你的要求在指定的时间自动进行下载。如果你的机器还没安装Teleport Pro,可到http:∥www.tenmax.com/pro12.exe去免费下载。
　　一、如何下载网页
　　●关键术语
　　1.开始地址(Starting Address):这是用户在进行一个下载时必需输入的一个HTTP地址，它可以是一个域名也可以是某网页或文件的地址等。Teleport Pro不支持FTP或新闻组地址的下载。
　　2.链接深度(Link Level)：这是相对于开始地址而言。开始地址页上的链接页的深度为1，开始地址页上的链接页上的链接页的深度为2，其它依此类推。一般我们设置成3到5，太大的话，下载的信息量将特别大。
　　3.下载工程文件：在你要开始下载之前必须创建或打开一个工程文件，它的扩展名为tpp。这文件包含了你的各种下载设置以及已经下载的各种文件的目录及更新日期的详细记录。
　　●下载过程
　　Teleport Pro利用查找算法从一个开始地址开始快速的查找Web页，对找到的页进行验证（是否符合要求）和分类，把符合要求的文件取回。
　　Teleport Pro对取回的文件根据Web页进行分类，它把每一个Web页以一个文件夹的形式在Teleport Pro左边的窗口以树结构的形式表示出来。而每一页上的各种文件则在右边的窗口列出来。
　　Teleport Pro对下载的各文件情况在工程文件里进行详细的记载，所以不会重复下载，下载也可分多次进行。
　　二、创建工程
　　创建一个下载工程最简单的方式是使用新工程向导（New Project Wizard）。执行“File/New Project Wizard...”（或单击按工具栏上的新建按钮）来启动这个向导,这个向导分四步进行设置：
　　1.选择下载的方式
　　Teleport Pro共六种下载方式供选择（图1）：
　　（1）Create a browsable copy of a website on my hard drive：如果仅仅是为了离线浏览某Web站点，这种方式将是最简单和最快的一种。Teleport将从你指定的Web站点把符合你要求的文件下载到你硬盘的一目录下。
　　（2）Duplicate a website,including directory structure：这种方式主要用于做站点镜像。它将把网站的整个目录结构及文件复制到你的硬盘里。
　　（3）Search a website for files of a certain type：如果你仅仅需要图形文件、背景图片、声音文件甚至ZIP文件等，你就选择这一种。它仅仅取回你所要求的文件。
　　（4）Explore every site linked from a central site：如果你只是想知道站点的结构则选择这种方式。它仅仅取回文件的名称而不取回文件的内容，这样它的下载速度非常快。然后你可以根据下载的文件名称来进一步的决定下载哪些内容。
　　（5）Retrieve one or more files at known addresses：下载指定地址的文件。在这种下载模式里，你可以输入多个Web页的地址（一行一个），Teleport仅仅取回指定地址的Web页。
　　（6）Search a website for keywords：跟第一种方式相似，只是在根据它是否含有要求的关键字来决定是否下载。
　　2.输入开始地址
　　在图2所示窗口中的第一个文本框里输入一个合法的地址(必填),在第二个文本框里为该下载取一个直观名称(可选)。在一个工程里可以包含多个开始地址，如果想再添加地址，在工程向导完成后，可通过“Project/New Starting Address...”来添加多个开始地址。点击“下一步”，进入第3步（图3）。
　　3.指定下载的文件类型
　　四种选择分别代表的含义分别是：
　　●Just text（仅下载文本）
　　●Text and graphics（下载文本和图形）
　　●Text、graphics、and sound（下载文本、图形和声音文件）
　　●Everthing（全部下载）
　　如果你所登录的网站需要账号和口令，则在Account和Password分别进行输入。
　　4.完成设置
　　单击完成按钮，在保存文件对话框里为工程取个名称，同时Teleport以此名称创建一个文件夹，下载下来的文件均存放于此文件夹下。
　　三、开始下载
　　点击工具栏上的开始按钮便开始下载（图4）。Teleport以10个线程来同时下载10个文件,下载指示器上的10个灯的不同颜色及形态来表示各线程的下载状态。如果你想中止下载，点击工具栏上的暂停、停止或中止按钮。其中如点击暂停按钮，则暂停下载，再次点击则继续下载；如点击停止按钮，Teleport便不再创建新的下载线程，但必须等待当前的10个线程完全下载完毕后才停止；如点击中止按钮，则所有的线程都被中止，没完成的线程则本次下载被取消。不过这两种情况，对文件的下载都不会造成影响。Teleport都将把文件下载的情况记录在工程文件（＊.tpp）里，下次下载时，它都能从正确的地方开始继续下载。
　　四、离线浏览
　　下载完毕，现在你可断开连接，进行离线浏览。只要在Teleport左边窗口的工程名称上单击鼠标右键弹出快捷菜单，选择“Open”命令，就可启动你的浏览器进行离线浏览了。也可以选择Teleport右边的文件窗口里的文件夹或网页进行同样的操作。
　　五、下载参数详细设置
　　前面利用向导对下载也进行了一些简单设定，如果想作进一步的设定或对以前的设定作修改。执可行“Project/Project Proerties...”菜单进入工程属性窗口进行各种下载设置。Teleport最常用的是“Summary”和“File Retrieval”两选项。图5所示即是“Summary”选项。点击中间的“Clear Project Database”按钮将清除下载的各种文件及清空工程文件数据库里的所有下载记录，重新回到开始下载之前的状态。点击“File Retrieval”进入图6所示的设置下载文件类型的选项窗口。
　　如果选择第一个选项，你可以输入一个数值，所有比这个数值大的文件将不会被下载，这样可为你节省下载的时间。如果输入数值为0则表示对大小无限制。
　　如果你选择第二个选项则可指定下载的各种文件的类型和大小。点击“Add...”命令按钮会弹出一个小菜单来增加各种设置。其中前面6种是Teleport Pro为我们定义好的。你如不满意则可以自己定义，选择“User defined...”进入自定义对话框（图7）。在下载文件类型框里你可输入你想下载的各种文件类型，可使用通配符如＊.gif等。点击“OK”按钮可完成设置。也可利用“Edit”和“Delete”按钮对已经添加的各种设置进行修改和删除。
　　六、计划下载
　　一个站点的内容常常是非常丰富的，下载时常常要经过漫长的等待；如果我们能把下载的时间安排在深夜，不仅网络下载速度快而且费用低，当然我们还可以安心睡我们的觉。通过Teleport Pro Scheduler可以很容易实现。可以分以下几步：
　　（1）把Teleport设置成自动断线。即下载完成后自动断线。
　　执行“File/Connections...”打开Connections对话框。
　　（2）按前面所讲的方法对要进行下载的站点建立工程文件（可建立多个）。
　　（3）启动Teleport Pro Sheduler,进行下载时间的设置。执行“Project/Schedlue...”。
　　（4）添加工程。执行“Edit/Add Project”命令或点击工具栏上的添加按钮（上为一个＋号）把所要执行的工程一个一个加进去。
　　然后工程列表选中第一个工程，执行“Edit/Edit Project”命令，选择Schedule选项。选中最后一个Just once并输入开时下载的时间，点击确定即可。
　　（5）激活Schedule。执行“File/Activate”命令（或点击工具栏上的开始定时下载按钮），此时在任务栏上的系统托盘上也多了一个Shedule的图标。这时你可关闭“Teleport Pro”和“Schedule”窗口，到时下载便会自动进行。　　(浙江　方建文)
