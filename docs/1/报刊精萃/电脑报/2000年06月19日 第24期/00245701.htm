《 业界新闻 》

柯达推出数码影像服务新功能

　　随着数码科技在影像领域的推广，柯达超相像推出了数码证件照、开心随意贴和无底片复制三种不同影像服务功能。由柯达公司研制的柯达超相像(中国版)数码系统是采用专业数码影像技术最新开发的集照片扫描、屏幕预览、数码编程和即时输出于一体的新型设备、其最显著的特色是：先拍摄或扫描照片，可通过屏幕预览系统为用户提供预选择，再经过数码编辑确定所需求的尺寸和格式，最后由柯达具有专业型的4700热升华打印机，把客户选定的照片打印出来。数码证件照不但可选中哪张印哪张，还可同时输出多种尺寸的彩色或黑白照片，重印放大照片需底片，还可制作五彩缤纷边框的相片影像贴纸，颇受人们青睐。据悉，第5000家柯达快速彩色店近期落户成都，新配柯达超相像数码系统将使柯达快速彩色”影像服务网络更精彩。

IBM 笔记本电脑加入支持 Linux 行列

　　IBM 正大力推广 Linux 操作系统，不仅在高阶服务器采用这种新型的Unix系统，也将 Linux 引进笔记本电脑。
　　Gartner 分析师 George Weisse 说，蓝色巨人大力推助 Linux，有助于提升 Linux 在企业计算机界的可信度，也对 IBM 有好处。
　　但 Linux 要广获企业应用之前，必须先鼓励全球性的软件开发、服务与支持活动、改善资料回复及其它弱点，并提供专业服务级的保证。
　　IBM 表示，稍后会供应内建 Linux 的 AS/400 系统，但未透露进一步的信息。此前，蓝色巨人已在 Netfinity 服务器中采用 Linux，并已开始在 S/390 系统上测试 Linux。
　　此外，IBM 将在最新型的 ThinkPad A20m 与 T20 笔记本电脑中，装置 Caldera OpenLinux eDesktop 2.4 版，预计第三季度会认证特定的ThinkPad 机型，搭配 Red Hat、SuSE 和 TurboLinux 版的 Linux 操作系统。

摩托罗拉与IBM、东芝
签署蓝牙技术开发协议

　　摩托罗拉(Motorola)最近推出第一批以蓝牙技术为架构的产品，并与国际商业机器(IBM)以及东芝(Toshiba)签署协议，携手开发这项短距无线通讯技术，透露出公司积极开发蓝牙技术的决心。
　　在蒙特卡罗举行的蓝牙技术大会上，摩托罗拉也推出一款蓝牙无线汽车电话装置，让驾驶者无须手持话机，也不用绕着一大堆耳机话线，就能够轻轻松松打电话。摩托罗拉也表示计划和IBM与东芝共同开发最新技术。
　　IBM与东芝是蓝牙特殊开发集团(Bluetooth Special Interest Group)初期的赞助厂商。该集团专门从事蓝牙技术的测试与开发工作。
　　摩托罗拉预计将在今年年底前推出蓝牙技术的移动电话与耳机，并于明年夏天前推出蓝牙技术的汽车产品。
