《{Url2htm帮你轻松收集网址}》

{　　你想过把收藏夹中网址收集起来吗？甚至把整个硬盘中的网址都收集起来，随便地复制、自由粘贴，甚至存成一个网页。有了这样一个包含你所有喜好的网址的网页，再访问某个网站时直接点相应的名称就行了，实在是很方便。存成网页还有一个好处是：把你的收藏网页，拷到一张盘上，到另一台机器上也照样能用，如果你是在单位和家中都上网的话，这个功能是不是太有用了。
　　Url2htm就是能实现上述功能的一款软件。该软件是绿色软件，无需安装，点击图标就能运行(如图)，双击目录选定要扫描的目录，再选"开始扫描"就成了，如果想生成网页，点击“生成网页”，指定一个文件名，存盘即可。用这个软件，你还可以自己制作一个简单的网站引擎。该软件功能不多，所以只有228KB的大小，是免费软件。下载网址：http:∥ozc.myetang.com(山东　黄家贞)}
