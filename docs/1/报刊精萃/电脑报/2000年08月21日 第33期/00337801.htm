《 {奔驰FC-815A+评测报告} 》

{　　i815芯片组显然是今后一段时间内主板芯片组市场上的主流产品，它与BX芯片组相比，在支持的功能上有所增强，如支持Ultra DMA/66，AGP4×等技术。因此无论是在性能上，还是在扩展性上，815主板应该比BX主板更具优势。另外，因i815芯片组正式支持PC133，所以它能完整提供66/100/133MHz的前端总线，这也是它的特点之一。所以，从技术方面的情况来看，i815芯片组的主板已具有取代BX芯片主板的实力。正因为i815芯片组在技术上有所突破，并且又是Intel主推的产品，所以各主板生产厂商都纷纷跟进，深圳新天下公司也反应迅速地推出了自有品牌奔驰FC－815A＋主板。
产品简介
　　目前，Intel 815芯片组分为两个版本，分别是i815/Solano 和i815E/Solano2。这两款芯片组都使用了相同的北桥芯片，被称为图形存储控制中心（GMCH）。两者的区别主要是在南桥芯片上，其中i815使用了ICH1（I/O控制中心），编号为“82801AA”。 ICH1可以支持UltraDMA/66和AC'97，提供两个USB接口。而i815E采用的则是ICH2，编号为“82801BA”，相对于ICH1增加了对ATA100技术的支持，可以提供两个UltraDMA/100控制器，并且支持四个USB端口，以及通讯和网络升级卡（CNR），集成了更多的网络功能。因此，i815E/Solano2在性能及扩展性上更具优势。这次新天下公司送测的奔驰FC－815A＋主板，其南桥芯片的编号为“82801AA”，所以它是一块i815/Solano芯片组的主板。
　　奔驰FC－815A＋主板采用了黄色的PCB板，ATX结构，做工和用料都很精细。而且板上的电气元件用料实足，图形存储控制芯片上安装了散热片，所有连接器的布置完全符合PCI规范。这款主板上提供了六条PCI插槽，取消了对ISA设备的支持；一条AGP 2×/4×插槽（支持Sideband模式）、三条168线DIMM内存插槽，支持PC66/100/133MHz的SDRAM，最高可支持512MB的内存；两个USB接口;一个CNR插槽，但只支持Soft Audio Modem卡；两个Bus Master IDE连接埠，最多可支持四个Ultra DMA 33/66设备 ；整合了Intel i752图形加速芯片，支持AIMM （AGP Inline Memory Module，AGP板上内存升级模块）功能；板载创新PCI128声卡（CT5880音效芯片）。除此之外,随主板还附送一本说明书和一张驱动光盘。说明书为用户详细介绍了主板的规格，跳线以及BIOS的设置情况。另外，对于奔驰FC－815A＋主板上的一些人性化设置，像BIOS升级简便设置、双BIOS功能等，说明书上也作了简略介绍。但由于是全英文的，普通用户可能会略感不便。
　　奔驰FC－815A＋主板采用了Scoket 370架构，支持Celeron及Coppermine系列的CPU。由于首次加入了对 PC133规范的支持，所以它能支持66/100/133MHz的前端总线，同时还具有异步系统总线/内存时钟调节能力。这样用户就可以使用PC100与PC133 SDRAM。特别是对于赛扬用户而言，i815芯片组主板的性能还要优于ZX/BX以及i810芯片组。 因为它可以在66 MHz 前端总线 下运行赛扬处理器，而同时把内存频率保持在100 MHz，这样就提高了整个系统的性能。
　　由于奔驰FC－815A＋也像i810芯片组主板一样在北桥芯片中集成了i752图形处理器，这使得它的适应能力更强。它既可作为一块整合型主板，满足低端用户的需要，同时，还可以通过附加3D图形加速卡的方式，来满足高端用户的需求。当用户在奔驰FC－815A＋主板AGP插槽上附加3D加速卡时，北桥芯片内置的i752加速卡会被自动屏蔽掉。当用户在使用内置显卡子系统时，系统将会自动从内存中挤出2MB作为显卡内存，若有需要可提高至64MB。另外，你也可以通过AIMM（AGP Inline Memory Module，AGP板上内存升级模块）接口，插入显存扩展卡来扩充显存容量。
　　奔驰FC－815A＋主板的倍频和外频调节采用的是跳线和BIOS相结合的方式，提供3～8倍频的调节能力。由于Intel很早以前就开始在CPU上加上了倍频锁，因此倍频调节对于815主板来说相对意义较小。奔驰FC－815A＋主板的外频调节是分成三个频段来实现的：在66MHz的频段上，可选频率有66/68/75/78/80MHz；在100MHz的频段上，可选频率有100/105/110/113/117/121MHz；而在133MHz频段上，则有133/138/140/150MHz可选。一个频段中的外频选择，只需在BIOS中就可以完成。而各个频段之间的切换，就必须通过跳线来完成了，这样做起来多少有点麻烦。奔驰FC－815A＋主板上没有提供CPU核心电压以及I/O电压的调节。另外，BIOS中有一个“System Memory Frequency”的选项，通过它我们就可以实现前面提到的内存异步功能。
　　奔驰FC－815A＋主板上还加入许多人性化的设计。如升级BIOS时，只需按下“Alt＋F2”，主板即可自动搜索软盘中新的BIOS升级文件，这对于没有升级BIOS经验的朋友来说是非常合适的，减小了升级失败的危险性。而且这款主板也采用了双BIOS设计，当一颗BIOS芯片受到破坏而不能启动时，只需在开机时同时按下机箱面板上的Power和Reset键，即可启用另一颗BIOS用来启动电脑并恢复被破坏的BIOS。
性能测试
　　这次的性能测试基本上还是和以往一样，主要是通过Winbench99、Winstone99和CC Winstone2000等测试软件来了解奔驰FC－815A＋主板在性能上究竟有如何的表现。测试是在使用板载i752显卡和附加Geforce256 3D图形加速卡两种情况下进行的。而且针对这两种情况，分别在100MHz和133MHz外频下对系统性能分别进行了测试。
　　测试平台
　　主板：奔驰FC－815A＋
　　CPU：Intel Coppermine 550E(FC－PGA)
　　内存：KingMax 128M SDRAM
　　硬盘：昆腾Fireball Plus LM 20G
　　显卡：讯英Geforce256 32M SD 5.33版
　　系统：Win98SE
　　在使用Winbench99对CPU部分、磁盘子系统以及图形WinMark测试后，从测试结果我们可以发现，奔驰FC－815A＋主板的板载i752显卡确实是限制系统图形性能发挥的一个瓶颈。测试数据表明它的3D性能实在有限，只能适用于那些对3D性能要求不太高的环境，这也是类似于i810的整合型主板的一个通病。当然，提升系统3D图形性能最有效的办法就是附加性能更好的3D图形加速卡。在这一点上， i815芯片组的主板比i810主板在扩展性上更具有优势。测试数据也很好地印证了我们的观点，在附加Geforce256 3D图形加速卡后，在同一外频下，系统的2D商业图形WinMark和高端图形WinMark的得分分别有近一倍和50％的提升。在附加3D图形加速卡后，奔驰FC－815A＋主板就能够很好地满足高端用户的需求了。另外，由于i815芯片组是Intel第一款支持PC133规范的芯片组，所以奔驰FC－815A＋主板在133外频下的表现也是我们关注的一个焦点。从测试结果来看，133MHz外频下系统性能比100MHz下有了大幅的提升，Intel在经过众多挫折后终于作了一次正确的选择。另外，从商业磁盘WinMark和高端磁盘WinMark的成绩来看，磁盘子系统的性能已经达到了Ultra DMA/66的要求。
　　Winstone99是一个基于应用程序的测试软件。它包括三个测试子项，其中我们最常用的是Business Winstone。它基于流行的商用软件，并采用套件的形式。它的测试指标对于办公用平台较有说服力。CC Winstone2000的工作原理和Winstone99相似，但适用于配置较高的系统，主要用于图形图像处理、多媒体平台的测试。它们都是用来衡量PC的整体性能的测试软件，但两者的侧重点不同。从奔驰FC－815A＋主板Winstone99和CC Winstone2000的测试成绩来看，基于奔驰FC－815A＋主板的系统的表现是比较出色的。特别是当系统处于133MHz外频下时，整体性能都表现得更为优异，这多少得益于133MHz 外频下内存带宽的提高给系统性能提升带来的影响。另外一点，i815芯片组的系统架构和i810、i820系列芯片组一样，也采用“Hub Accelerated”架构，即在传统的南桥和北桥之间采用266MB/s的专用数据端口连接，这种架构可以使PCI总线的数据传输率最高能够达到266MB/s，多少会为系统性能的提升带来帮助。
　　Quake3的测试再一次证明了我们前面的观点，在3D游戏日益流行的今天，板载的i752显卡确实显得有些力不从心了。如果你是一个游戏爱好者，我们劝你还是去买一块3D性能更好的图形加速卡吧。另外，无论是在133MHz下，还是在100MHz下，两者的测试成绩相差并不大。特别是分辨率越高，它们之间的差距就越小。出现这种情况的原因，我们在前几期的主板评测中已经作了分析。到目前为止，还没有更好的方法来解决这一问题。也许要等到几个月后,DDR产品正式上市，这种状况才会有所改观。
超频测试
　　现在可以说超频是非常流行的，主板的超频能力已经成为一块评价主板的重要因素了。就奔驰FC－815A＋主板超频选项的设置而言，它在这方面做得还不够好。我们在前面就已经提到了，它仍然采用的是跳线和BIOS相结合的形式，对于普通用户而言就比较麻烦一点。不过，在测试中我们发现在超频情况下，这块主板表现得比较稳定。我们就是在这块主板上把Coppermine 550E超到了733MHz，并跑完了所有的测试。即使是在140MHz外频，CPU超到770MHz的情况下，我们运行了一下Winstone99和CC Winstone2000，没有发现异常情况，超频非常成功。150MHz外频下能够完成自检，但不能进入系统，其原因可能是多方面的。除了在设置方面有点不足之处以外，就奔驰FC－815A＋主板的超频性能以及超频后的稳定性而言，我们总体评价还是非常不错的。
兼容性测试
　　从近期的主板测试情况来看，我们还没有发现什么兼容性问题，特别是对于Intel芯片组，发生这种情况的可能性就更小了。但不怕一万，就怕万一，一旦发生就后悔莫及了。因此，我们还是认为有做兼容性测试的必要。
　　目前装机的用户，大多数都选择了兼容机，所以出现不兼容问题的几率就要大一些。在这些兼容问题中，最容易出现的是内存条与主板不兼容。因此，我们的兼容测试就把重点放到了内存条上。出现内存条与主板不兼容问题的原因是多方面的，有可能是主板设计上的缺陷而造成主板与内存不兼容，还有可能是内存条本身的质量问题所引起的。我们挑选了目前市面上常用的几种品牌的内存条，像现代、KingMax以及金条等，试着把不同的内存条混用插在奔驰FC－815A＋主板的内存插槽上,并插满所有的插槽。从测试的情况来看，我们还没有发现奔驰FC－815A＋主板与内存有不兼容的问题，说明它在内存兼容性方面确实做得不错。
总　　结
　　从奔驰FC－815A＋主板的整个测试情况来看，它的表现是非常不错的。奔驰FC－815A＋主板除了能为用户提供i815芯片组所具有的功能以外，还能为用户提供一些个性化的设置。像前面提到的BIOS升级简便设置、双BIOS功能等，这在主板设计越来越缺乏特色的今天，是不多见的。这也说明，新天下公司在这款主板上确实是花了一番心思的。}
