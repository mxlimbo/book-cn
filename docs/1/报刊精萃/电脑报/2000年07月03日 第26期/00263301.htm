《 真正实现光驱的 DMA33 模式 》

　　我是一个初级DIYer，近日购得《用电脑PC DIY》2000年2期，其中一文《光驱“脱胎换骨”记》（以下简称《脱》文）读后感受颇深，该文作者用的是ACER 632A光驱，该光驱支持DMA33模式，但是在开机时却显示为MODE 4，显然未能用上DMA33，后来竟然通过刷写Firmware才用上了DMA33。我看了颇不以为然，这种情况说不准是主板与光驱配合不好。我用的也是ACER 632A，一开机显示为UDMA MODE 2，显然已是DMA33了，那实际速度如何呢？这次我得测试测试。打开Norton Utilities，选择“System Information”，测试光驱。放入一张正版软件盘，一测试，啊？只有10.9M/s，这就是DMA33吗？难道我的主板也有问题？不可能！那是驱动程序问题？也不对，在Windows下光驱驱动用的可都是Win98自带的！
　　不管三七二十一，我先驱猫上网，进入驱动之家，拖下ACER高速和低速两种驱动程序，下网装载测试，问题依旧。难道真的必须刷写Firmware？关于Firmware，我可只知道在MODEM上刷写后可提升速度，加强对各种协议的支持，光驱嘛，我就不知道了。翻开光驱说明书，我左看右看、上看下看，硬是没有找到可升级Firmware字样，再把光驱包装盒找出来，看了半天，终于在外包装盒上看到了一句“FLASH MEMORY FOR Firmware UPDATING”，哈哈！显然可以升级，理论可行。《脱》文作者实践在先，理论＋实践都已经有了依据，我只好冒险。我立即驱猫上网，进入ACER中文网，硬是找不着光驱Firmware升级程序，而且驱动程序一栏只有驱动程序名，却没有该驱动的用途说明。想起《脱》文中曾说到ACER的官方网站ACER.COM，立即键入WWW.ACER.COM，到底是官方网站，各方面介绍都比较详细，找了一通，终于找到了ACER 632A的Firmware升级文件，该文详细介绍了如何升级驱动、Firmware等，立即拖了下来。按照说明先装载该光驱的DOS驱动程序，重新启动到DOS模式，运行该升级文件，只见屏幕显示正在查找可升级硬件，只过了十多秒，屏幕显示一行字“UPDATING SUCCESSFULLY”，我吓了一跳，这就完了（又一次冒险），也没有让我备份一下，就这样刷新了，要是不管用，就得为我的光驱开追悼会了。
　　不管它，重新启动，自检找到光驱，继续进入Win98，我稍觉宽慰，看来是成功了，打开了“我的电脑”，放入一张光盘，双击该盘符，鼠标就不动了，等了几秒种，一点反应都没有，死机了。我重新启动，再进入Win98，重新放入一张光盘，这次更糟，打开“我的电脑”就死机了。糟了，这下我的光驱“光荣”了。正考虑着如何如何，突然想起《脱》文曾说过：去掉光驱DMA选项后正常读盘。再次RESET，进入Win98，进入设备管理，去掉DMA选项，再次怀着惴惴不安的心情重新启动，进入Win98，放入光盘，双击盘符，该盘内容迅速呈现出来。好了！正常！再用Norton Utilities测试，数据传输率由原先的10.9M/s上升到33M/s，我的光驱终于真正实现DMA33了!再经几次测试，复制传送速度明显提高，令我大感欣慰（设备管理中光驱逻辑单元号由原先的32UP变为32VP）。
　　这次升级，先苦后甜，更让我体会了DIY的乐趣，也让我得到了一些经验，开机显示为UDMA MODE 2并不等于你的光驱已工作在DMA33模式下，当然也可能只是我的ACER 632A的问题，但为什么在Win98选中DMA选项就会死机，这究竟是何原因？是否因为升级Firmware的原因，还望各路大虾不吝指教，我的E－mail地址是neilm@21cn.com.

(江苏　莫祝华)
