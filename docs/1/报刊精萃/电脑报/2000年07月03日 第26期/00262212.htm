《 巧用Animation Shop2.0的向导制作动画 》

　　Animation Shop 2.0以其简单的操作和强大的制作功能迅速成为了我的最爱，它有两个较好的向导可以帮助我一步一步地制作和学习。下面我就利用这两个向导通过动画制作的方式来介绍一下它们的使用方法。
　　一、用Animation Wizard向导制作动画
　　1．启动该向导　选取“File\Animation Wizard”命令或单击工具栏上的“Animation Wizard”按钮；
　　2．设置画面尺寸　可以设成全部画面都使用第一幅画面的尺寸(Same size as the first image frame)或者全部画面采用自定义的尺寸(As defined here)；
　　3．设置画布颜色　可以把全部画面设成透明(Transparent)或者不透明(Opaque)，当然，颜色设置可以自行设定；
　　4．设置画面的位置　如果加入的图片与画面的尺寸不相同，可以把加入的较小的图片设置在画面的左上角(Upper left corner of the frame)或者中心位置(Centered in the frame)。画面的空白部分用画布颜色填充(With the canvas color)或者用前一个画面的内容填充(With the preceding frame’s contents)；最后把图片按比例缩放至画面大小(Scale frames to fit)；
　　5．设置动画选项　选择使动画一直播放下去，直到手动停止它(Yes, repeat the animation indefinitely)或者指定播放次数(Play it on times)。还要设置每幅画面显示停留时间(Set the display time for each frame in the display time),这个设置的数值可以在后面通过“Animation\Frame Properties”来修改；
　　6．把图片加入到动画里　单击“Add Image”按钮选择要加入到动画里的图片，完成后，还可以取消动画里的图片、调整图片的顺序；
　　至此，我们已经建立起了一幅动画框架，单击Finish按钮关闭向导页，就可以通过工具栏上的“View Animation”按钮或“View\Animation”来浏览动画。
　　二、用Banner Wizard向导快速制作横幅
　　1．启动该向导　选取“File\Banner Wizard”或单击工具栏上的“Banner Wizard”按钮；
　　2．设置背景　可以设置成透明背景(Transparent background)或其他颜色作背景(Opaque background)，也可以使用一张图片作为背景(Use a background image)；
　　3．设置横幅尺寸　可以使用因特网上常用的六个统一尺寸之一作为横幅尺寸(Use a standard banner size)。如果在第二步里使用了一张图片作为背景，可以选择该图片的尺寸作为横幅的尺寸(Same as the background image)，当然，也可以自定义尺寸(As defined here)；
　　4．设置动画时间　设定动画每幅画面的显示时间(Set the display time for each frame),默认的值为10秒，在设定显示时间后，通过“Animation\Frame Properties”命令对单幅或整个动画的显示时间进行修改，设定的值将在每幅画面的下面以“D:n”的形式显示。设置每秒显示动画的几幅画面(Set the number of animation frames to display per second)，在前面两个设置确定后，将决定动画有多少画面(每幅显示时间/10×每秒显示画面幅数)；还可以设置动画是否循环播放及播放次数；
　　5．输入横幅文字　在文字框中输入将在横幅中显示的文字，并设置好文字的字体、字号等；
　　6．设置文字颜色　指定字体的颜色(Opaque text)或者用一张图片的剪贴图颜色作为字体颜色(Image text)；
　　7．设置文字的迁移　从“Transition Name”下拉菜单中选择一种迁移类型，可供选择的有背光(Backlight)、跳跃(Bouncing)、阴影(Drop Shadow)、旗(Flag)、高亮(Highlight)、字幕(Marquee)或轮形(Wheel)；选择变化方式后，还可以按特殊要求对迁移效果进行自定义(Customize)；
　　至此，已经完成了一幅横幅动画了，单击Finish按钮完成并关闭向导页，就可以通过工具栏上的“View Animation”按钮或“View\Animation”命令来浏览动画了。
