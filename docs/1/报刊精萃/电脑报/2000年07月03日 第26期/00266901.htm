《 秘技锦囊 》

　　笑傲江湖
　　轻易发出“九连击”： 按下“式”后进入招式选择，首先使用一次中央的普通攻击，在角色发动攻击时，按任何一个招式，再不停地按画面上的任何一个位置（如地面），就可连续使用角色最大行动值（即九次）的普通攻击！
　　剑侠情缘Ⅱ
　　在中都欧阳家的欧阳密室里可救出两名女子，右边那名女子会给你一万左右的金钱，只要你在她给你金钱后在原地狂按鼠标就会每次得到一万左右的金钱（注意：一、必须狂按鼠标；二、必须原地）。此秘技不易成功，使用前请先记录，并多试几次。
　　独闯天涯
　　首先进入游戏主界面，然后依次敲入summeryan便可进入密技模式（会在屏幕左上角出现一张笑脸^_^）：
　　1隐身术：输入HIDE即可将主角隐身
　　2补血术：按下HOME键即可将血补满
　　3复制术：首先把需要复制的东西抓起来（从物品栏中拿起），然后输入COPY便会在地面上出现一件同样的东西……
　　4分身术：输入SELF即可创建一个主角的“克隆”，如此便可以吸引敌人的注意力，同时攻击力也增倍！
　　幕府将军——全面战争
　　在游戏中输入以下秘技（1.01以上的版本此秘技无效）：
　　·matteosartori 地图全开
　　·daggins 地图全开
　　·muchkoku. 拥有无限的KoKu
　　·conan. 建筑单位能力提升并对KoKu产生负面影响
　　精灵幻境
　　主角一开始都有一颗低喃的精石，在游戏途中找到光之键便可以到秘密商店交易。 例如：艾克瑟到玛哈洞窟打倒精灵使会得到光之键，而低喃的精石说到沙理纳斯森林便可到龙角之地。此时在世界地图下到沙理纳斯森林将萤幕往左移往龙角，会发现多了一个地点可以去。那儿有蘑菇三兄弟可以交易，有不少平常买不到的东西，如：铁矿石，魔法白银，等等。其中最重要的是矮人族之戒，一定要买。让主角装三个矮人族之戒几乎无敌。
　　赚钱妙法：
　　在佛西亚内取得奇迹之水，撤退之后再进去，这样可反复取得奇迹之水（奇迹之水可卖五十元）
　　无限移动：
　　1首先召唤你要的精灵数量。
　　2在第2回的时候让精灵移动最后一个和精灵使。
　　3先让精灵使移动，后选精灵。
　　4选到精灵的时候，窗口上会有一个“召唤精灵”的快速键。
　　5按下去，会出现另一个窗口，选“ok”，这样精灵使可再度行动（可使用无限次哦！）。
　　机械战神（Deus Ex）
　　编辑deusex\system目录下的user.ini文件，把所有的“t=”改为“t=talk”（你也可以使用其它字母，不一定是t）。然后在游戏中按“T”键，会在屏幕底部出现一行“Say：”文字，删掉所有字符，只留下空白的一行（每次输入密码时都要这样）。
　　输入“set DeusEx.JCDentonMale bCheatsEnabled True”（没有引号）激活作弊模式，然后就可以使用下列密码了：
　　god = 无敌模式
　　invisible = 隐形
　　behindview 1/0 = 第三人称视角 开/关
　　fly = 飞行模式
　　walk = 飞行模式关闭
　　iamwarren = 打开EMP区域
　　allskillpoints = 得到所有技术点数（Skill Points）
　　allweapons = 得到所有武器
　　allammo = 重新装满弹药
　　tantalus = 杀死当前的目标
　　opensesame = 打开当前的门
　　legend = 隐藏菜单
　　allhealth = 生命值全满
　　allenergy = 能量值全满
　　allcredits = 10000金钱
　　MDK 2
　　先用“～”键打开控制台
　　mdkNewGame（1，12）= 跳到第1关（第2关至第9关依次类推）
　　mdkNewGame（11，1）= 跳到第10关＃1 （特别的）
　　mdkNewGame（12，1）= 跳到第10关＃2 （特别的）
　　God（1）= 无敌模式 （比较容易输入 ^_^）
　　Slow－Mo模式
　　当你使用六脚狗Max时，按住射击“Fire”键不放，然后按4次“equip up”键，如果成功的话屏幕会显示Slo－mo！。如果要取消这个秘技，重复以上步骤就行了。
