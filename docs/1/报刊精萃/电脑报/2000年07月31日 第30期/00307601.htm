《 升技SE6主板测试报告 》

　　Intel i815芯片组的主板已经全面上市，我们电脑报评测实验室也在前几期的报纸中对几款i815芯片组的主板进行了测试。从i815芯片组的主板测试情况来看，它在性能上相当出色，完全可以取代BX芯片组。特别是Intel i815E芯片组，由于采用了新的南桥芯片——Intel 82801 ICH2，(编号为“82801BA”)，相对于原来的ICH增加了对ATA100的支持，可以提供两个Ultra ATA／100控制器。同时它还支持4个USB接口，并具有通讯和网络升级卡（CNR），集成了更多的网络功能，因此在性能表现和功能上都更为出色。这次，我们拿到了ABIT（升技）公司新推出的基于Intel i815E芯片组的主板——SE6，下面我们就来看看这款主板在性能及兼容性方面究竟表现如何。
产品简介
　　升技SE6主板采用了黄色的PCB基板，ATX结构，做工和用料都很精细，所有连接器的布置完全符合PC’99规范。主板上提供了6条PCI插槽、一个AGP（2×／4×）插槽（支持Sideband模式）、3根168Pin DIMM内存插槽，支持PC 100MHz／133MHz的SDRAM，最高可支持512MB的内存；两个USB接口，并允许从主板上扩展另外两个；一个CNR插槽；两个Bus Master IDE连接埠，最多可支持4个Ultra DMA 33／66／100设备；整合了Intel i752图形加速芯片（支援AIMM功能）和AC'97音效子系统。另外，主板上还有一个VL－Bus接口，通过扩展接口支持TV－OUT和液晶平面显示输出。
　　升技SE6主板采用了Scoket 370架构，支持Intel PentiumⅢ 500MHz～1GHz及Celeron 566MHz～677MHz的CPU。作为Intel 8XX中的一员，i815芯片组的系统架构和810、820系列芯片组一样，也采用“Hub Accelerated”架构，即南桥和北桥之间采用266MB／s的专用数据端口连接，这种架构可以使PCI总线的数据传输率最高能够达到266MB／s。而且也取消了对过时的ISA设备的支持，真正做到了PC’99规范。由于在i815芯片组中，Intel首次加入了对Intel PC133内存规范的支持，所以它能提供66／100／133MHz的前端总线，而且还具有异步系统总线／内存时钟调节能力。因此升技SE6主板既支持66MHz外频的Celeron系列处理器，也可以使用100MHz或133MHz外频的Coppermine处理器。对于用户来说，即使手中的Celeron处理器无法超频到100MHz外频，也可以像采用VIA芯片组的主板一样，让CPU运行在66MHz外频下，而把内存的速度保持在100 MHz，以提高系统的性能；也可以让133MHz外频的Pentium Ⅲ和原来的PC100 SDRAM配合，充分利用手中的资源。但是i815提供的异步内存总线功能并不完善，因为我们发现当CPU 运行在100MHz频率下时，不能使用133MHz的内存。即使如此，我们也很高兴Intel在i815芯片组加入这种功能。
　　由于升技SE6主板也像i810芯片组主板一样在北桥芯片中集成了i752图形处理器，这使得它的适应能力更强，可以很好地满足OEM厂商的需要。同时，也可以通过附加3D图形加速卡的方式来满足挑剔的游戏玩家的要求。在升技SE6主板AGP插槽上附加3D加速卡时，北桥芯片内置的i752加速卡将被自动屏蔽掉。当用户在使用内置显卡子系统时，系统将会自动从内存中挤出2MB作为显卡内存。你也可以通过AIMM（AGP Inline Memory Module，AGP板上内存升级模块）接口，插入显存扩展卡来扩充显存容量。
　　另外，在升技SE6主板上你还可以找到一个CNR插槽，它与我们常见的AMR插槽不同。AMR只支持Modem的连接，对于即将来临的宽频网络时代实在是不够用。而CNR的诞生则把问题解决了！CNR除可以连接专用的MODEM卡外，还能使用专用的家庭电话线网络（PHONE PNA），以及10／100 Base Ethernet LAN等，未来即使使用宽频网络也不愁占用到其他PCI插槽。
　　升技SE6主板的外频和倍频可以同时通过DIP开关和BIOS来调节，这在以往的主板中是不多见的。它们能提供从4.0到最高可至10.5 倍频的调节能力。外频选择比较丰富，支持的最高FSB达到了153MHz。这些外频可以使用DIP开关来选择，也可以在BIOS中进行调节。当然，最好的方法是把DIP开关设置成默认值，然后在BIOS中进行调节，这样调节起来比较方便。另外，BIOS中有一个“System Memory Frequency”选项，通过它我们就可以实现前面提到的内存异步功能。CPU核心电压调节也可以在BIOS中实现，其调节范围为1.30V～1.80V之间，是到目前为止我们所见过的i815芯片组主板中调节范围最大的一块，并且可以以0.05V的幅度进行增减。除此之外，升技公司还专为超频爱好者设计了一个叫SoftMenuⅡ的软件，通过它可以快速方便地实现超频，而且操作并不复杂。
性能测试
　　由于升技SE6主板支持Intel PC133内存规范，并且可以通过附加3D图形加速卡的形式来提升系统3D性能，因此，此次性能测试就是分为系统外频在100MHz或133MHz，以及在使用内置显卡或附加3D图形加速卡这两种情况下性能到底有何差异而进行的。测试中所使用的主板、显卡等的驱动程序我们都更新为最新的版本，以求获得最好的性能。
　　测试平台：
　　主板：升技SE6
　　CPU：Intel Coppermine 550E(FC－PGA)
　　内存：128M 金条
　　硬盘：酷鱼Ⅱ 15.3G
　　显卡：大力神GTS2 32M 5.32版
　　系统：Win98SE
　　ZD Winbench99可以说是我们使用频率最高的测试软件了。从它的测试数据来看，当外频总线从100MHz提高到133MHz后，由于系统带宽也相应地由800MB／s提高到1.06GB／s，这对于系统性能的提升是非常显著的。升技SE6在133MHz外频下与CPU相关的CPUMark99和FPU WinMark99测试数据的提升尤为明显，提升幅度高达34％。从商业图形WinMark和高端图形WinMark的测试结果我们可以看出，在使用板载显卡和大力神GTS2显卡这两种情况下，系统的3D图形性能之间的差距也是非常明显的。由于板载的i752显卡的3D性能实在有限，只能适用于那些对3D图形性能要求不高的场景。因此这时你可以通过在升技SE6主板上附加3D图形卡的方式来提升系统的3D性能，以满足自己对3D图形性能的要求。升技SE6主板支持Ultra DMA ／66，甚至还支持新推出的Ultra DMA ／100技术规范。但是由于受到硬盘内部传输速率的限制，Ultra DMA/100对我们来说还比较超前。不过，随着新式具有Ultra DMA/100接口硬盘的推出，你的机器将会发挥出更高的效率。通过测试数据，我们可以发现磁盘子系统无论是在商业磁盘WinMark，还是在高端磁盘WinMark中的得分都达到了目前DMA/66的水平。
　　Winstone99和Winstone2000是衡量整个系统性能的测试软件，它得分的高低可以反映出整个系统究竟处于何种水平。测试结果表明，升技SE6在133MHz外频下整个系统的性能比100MHz外频下要高出许多，这主要得益于133MHz 外频下内存带宽的提高带来系统性能的提升。
　　Quake3作为3D游戏的经典之作，一向是我们最偏爱的测试软件之一。因为通过它，我们可以对整个系统在实际应用中的表现有个大致的了解。从升技SE6的测试数据我们可以看出，在使用板载Intel i752显卡和附加附加大力神GTS2显卡这两种情况下，AGP部分所表现出来的性能差距是非常显著的，这与我们的预测结果不谋而合。这主要是由于Intel i752在3D性能上的缺陷造成的，在当前3D游戏日益盛行的今天，Intel i752确实显得有点力不从心。虽然它还可以通过AIMM接口，插入显存扩展卡来扩充显存容量，提升系统在3D方面的性能，但用附加3D图形显示卡的方式来提升系统性能的效果更显著。另外，这些测试数据也可以从一个侧面说明，升技SE6在扩展性方面确实要优于以前的i810芯片组主板，完全可以满足高、低端用户的需要。
超频测试
　　是否能够超频成功取决于多方面的因素，当然主板是其中一个重要环节。就超频方面所提供的设置而言，我们觉得升技SE6主板就做得不错。它不仅为用户提供了丰富的外频选择，而且超频设置方式也很灵活（可以通过DIP开关调节，也可以在BIOS中设置）。另外，升技SE6主板具有的异步系统总线／内存时钟调节功能也有助于超频成功。事实也证明升技SE6在超频方面确实表现得不错，我们就是在这块主板上成功地把Coppermine 550E超到了733MHz。经过两天的测试，没有出现什么异常情况，系统很稳定。接着我们再把外频提高到137MHz，启动机器，频率显示为754MHz，然后顺利进入系统。我们运行了一些测试软件，没有发生异常，超频成功。再把外频设置为140MHz，开机后能够通过自检，却无法进入系统。在把CPU核心电压调为1.65V后，顺利进入系统，但在运行测试软件过程中系统报错，超频没有完全成功，需要通过其它的方法来加以改进。但就升技SE6主板在超频方面的表现而言，我们认为是很不错的。
兼容性测试
　　兼容性是大多数购买主板的用户比较关心的问题。因此在本次升技SE6主板测试中，我们就加入了兼容性测试。在分析了常出现兼容性问题的部件后，我们把本次兼容性测试的重点放在了内存条和电源上。内存条是计算机内部的重要部件，虽然内存条在生产上有严格的技术规范（如VIA的PC133技术规范）。但由于生产内存条的厂家太多，每个厂家在生产内存时采用的内存颗粒不同，再加上制造工艺上的差异，就造成了内存条品质上的细微差异，这就是出现主板和内存条不兼容现象的客观因素。再有就是由于主板设计上的缺陷，造成主板DIMM插槽出现与内存不兼容。我们选择了多种品牌内的存条，如市面上比较普遍的现代、金条等，并试着把不同的内存条混用插在升技SE6主板上，都没有出现不兼容现象，说明升技SE6在内存方面的兼容性不错。
　　在选购电脑时，一般用户往往只关注CPU、显卡之类的产品，常常忽略了微机电源在系统中的重要作用。尤其是在硬件升级换代频繁的今天，随着处理器和显卡处理能力的飞速提高，它们集成的晶体管数量的急剧增加，这都直接导致对电能要求的增大。同时还有高速硬盘，更不用说为散热添加的各种风扇，因此微机电源在整个系统中的作用日益突出。在这次兼容性测试中，我们选用了六个不同品牌的电源来与升技SE6主板进行搭配，它们包括长城ATX—250S、台达“大水牛”—200PB、百盛电源、保利得环保电源、金河田ATX—300以及世纪之星电源。从它们与升技SE6主板的搭配情况来看，都还不存在兼容性问题。
总　　结
　　从整个测试情况来看，升技SE6主板的表现是非常优秀的。由于它是采用Intel i815E芯片组，支持PC133内存规范，并可以通过附加3D图形卡的形式来提升系统3D性能，因此在整体性能和扩展性上优于Intel i810芯片组主板，可以满足高、低端用户的需要，具有广阔的市场前景。到目前为止，i815芯片组的主板已经全面上市，而怎样在同类型产品中找到一款性价比较高的主板，确实是一件伤脑筋的事情。通过前面的测试，我们可以对你说，这块升技SE6主板主板真的是不错。你也不妨到市场上去比较一下，然后再作出自己的选择。
