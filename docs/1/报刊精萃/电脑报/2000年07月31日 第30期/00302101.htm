《 {追新一族} 》

{　　前一段时间各大软件公司的航母级产品纷纷推出拳头产品新版本或者测试版本，如微软公司的IE5.5、Adobe的Photoshop6 beta3、Macromedia的Flash5 beta等。与这些大手笔的行动相比，上周的软件业界显得相对平静一些，除了几个优秀的软件推出小的升级版本外基本没有什么大的风浪。
“防火”利器
　　天网防火墙推出了个人版2.0 beta(http://sky.net.cn)这个测试版本重新优化了系统，减少占有系统资源；增加了一个非常灵活的规则设置系统，用户可以直接下载设置好的一系列安全规则，不用做任何调整即可使用；增加了对端口说明，其中端口说明中已经包含了大部分常用木马程序所有的端口；增加了声音报警机制；增加了日志的保存功能；允许用户自己定义是否一开机就启动防火墙。
“管家”更新啦
　　Windows Commander(http://www.ghisler.com)的强大功能是众所周知的，配套光盘工作室的各位兄弟更是爱不释手。自 4.03版与大家见以来，很长一段时间都没有再见它的新动向了。进入7月中旬，Windows Commander连续出击，在发布4.50版后几天又推出4.51。与4.01版相比，4.5版修正个100多个BUG，并增加了70多个命令和功能，例如可以在压缩包中修改文件名、增加压缩包“全部解压并运行”的功能等。这两次升级使Windows Commander的可靠性和操作性能得到了很大的提高。
“导航者”凑热闹
　　在微软推出IE5.5不久，Netscape Communicator也发布了其最新的版本，大有一比高下的意味。不过Netscape Communicator 4.74(http://home.netscape. com)给我们带来的新东西太少了，它允许用户在退出程序时清理所有的邮件目录，WinAmp 和RealPlayer 也分别升到了2.6与7.0。总的来说这个版本并没有多大的改进，看来Netscape Communicator要想赶上IE还需要多努力！
闪客的兄弟
　　Flash在网上的使用越来越广泛，即使不懂得使用的新手，也希望通过Flash的特效来装饰自己的网页。SWiSH 1.5(http://swishzone.com)是一个非常简单的Flash文字动画制作程序，现在你可以在动画中插入声音文件，可以调整水平与垂直字距，可以设定在动画下载完以前循环播放特定的动画等。除此之外还增加了几个新特效以及改良了对象放置。怎么样，不需要安装Flash，你也可以轻轻松松地制作出非常炫目的Flash文字动画特效来了。
其　它
　　Media Jukebox是一个简单、高效的音乐文件管理及播放软件，不但可以抓取音轨并将其保存为MP3或WMA格式，还可以将MP3或WMA文件刻录成光盘。这次Media Jukebox更新为5.0.578。另外它还具有强大的网络功能，可以从网上直接收听广播或下载音乐文件, 也可以把自己的作品发布到网上。Media Jukebox 5.0.578(http://www.musicex.com/mediajukebox)除了修正了一些Bug外，现在完全支持代理服务器，支持刻录数据光盘，可以输出HTML或是ASC Ⅱ的播放列表文档，在刻录光盘时还可以选择刻录74分钟或是80分钟的光盘。
　　CloneCD是强大的光盘复制工具，与其他同类软件不同的是CloneCD可以1:1地完全复制光盘。为了读取带有复制保护的盘片，需要能够读取磁轨隐藏信息的光驱，目前几乎所有ATAPI CD－ROM都支持这一点。CloneCD 2.7.1.1(http://www.elaborate.byte.com)增加了NT内核模式驱动，即使不是Windows NT/2000的管理员也可以使用该软件；CloneCD.vxd与系统配合得更好(Windows 9x)；大幅度提高了性能与稳定性；改进了安装和反安装程序；界面有所变化；增加了一些CD－RW的支持。
　　在众多的FTP工具软件中，FlashFXP也算是一款相当强悍、快速的FTP传输工具了。FlashFXP的操作和程序界面与其他的FTP软件也没有太大的区别；从功能来看，它支持续传，拖曳操作方式，支持站点间的文件传送，可以设定网站和本地文件夹的对应关系等。FlashFXP 1.2 build 623(http://flashftp.castlegardens.com)增加了上传文件名的大小写转换和文字显示控制方面的功能，同时做了一些Bug修正和功能变化的工作，对FTP上传和下载功能模块进行了较大的改良。
　　另外，ZipMagic 2000(http://www.mijenit. com)也从V3.0.1.186升级到V3.0.1.355。虽然变化并不大，仅仅修正了菜单、热键等几个小问题，不过ZipMagic 2000的确是一个值得我们关注的软件。它可以大简化您的ZIP文件的操作方式，您可以不需要把ZIP文件解压出来就可以使用里面的文件了。对于ZipMagic，每一个ZIP文件都被看作一个文件夹，你可以随时打开它们，执行里面的程序。
　　功能强大的MPEG影片压缩编辑工具软件系统PowerVCR可以使您的PC机具备录制和编辑影音的功能。这次推出的PowerVCR Ⅱ(http://www.cyberlink.com.tw)支持MPEG－1和MPEG－2压缩格式，具备录制实时优质的MPEG－1和MPEG－2影片的能力，支持DVD画质和5.1环绕立体声系统，而信号源可以是各种视频捕捉设备。}
