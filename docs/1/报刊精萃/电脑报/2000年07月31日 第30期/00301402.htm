《 自动找回丢失的WINDOW桌面 》

　　大家在使用Windows的过程中，经常会遇到Explorer崩溃后丢失桌面外壳（Shell）的情况，每次都要手工恢复桌面外壳，这会很烦！其实，只要在注册表中做一点小小的修改，那么每次在你的Explorer崩溃之后，Windows就会自动恢复桌面。运行注册表编辑器regedit.exe，找到下面这个注册表子键：
　　如果操作系统为Win9X的话：
　　HKEY_LOCAL_MACHINE \ Software \ Microsoft \ Windows \ CurrentVersion \ Winlogon
　　如果操作系统为Windows NT 的话：
　　HKEY_LOCAL_MACHINE \ Software \ Microsoft \ Windows NT \ CurrentVersion \ Winlogon找到该子键后，修改键值：AutoRestartShell（如果它不存在，就在右边窗口点击鼠标右键，新建该键值）。这个键值的类型为REG_DWORD类型，把该值设置为1（0表示禁止自动恢复），这样在Explorer崩溃之后，Windows就会自动恢复桌面了。　　（湖北　刘煜）
