《 媒体播放器试用报告 》

　　Windows视窗操作系统中自带的媒体播放器“Windows Media Player”想必大家都已经非常熟悉，但相信喜欢用它的人一定不多。现在，随着Windows系统的逐步升级（从Windows95、Windows 98到最新的Windows 2000），Windows Media Player也不断进行改进，但变化都不大，并没有推陈出新的感觉，与用户的期望始终存在着较大的差距，相较于同类的其他多媒体软件，无论是在功能、界面及可操作性上都有着诸多的不便。现在，微软公司对此有所意识，宣称将很快推出全新的下一代媒体播放器。
　　在微软刚发布了基于新技术的Windows Media Player v7.0预览版不久，我们评测室的软件评测人员就从微软的主页上将其Down下来，为读者朋友在使用前做一个试用报告。
　　名称：Windows Media Player
　　版本：7.0 Preview
　　大小：6.09MB
　　发布时间：2000年3月27日
　　开发者：Microsoft (http://www.microsoft.com)
　　本次试用所使用的系统环境：
　　硬件环境：CPU Celeron 300A、内存64M、显卡Winfast 3D S310、硬盘Quantum 8.4GB、DDN专线128Kbps；
　　软件环境：简体中文Windows 98 第二版、IE5.0；
　　上网时段：18点～22点。

安　　装

　　为稳妥起见，用户在安装前最好关闭当前系统中正在运行的所有应用程序和活动进程。运行下载的“wmp_preview.exe”文件，在弹出对话框中分别连续两次点击“Yes”及“下一步”。在安装时有两个选项：Reinstall all components（全部重新安装）和Update newer components only（只安装升级新组件），我们为了充分感受其新特点，所以安装时选择了前者。然后，可以在复选列表框中选择需要安装的组件，基于上述同样的原因，我们将每一个选项都选定。Windows Media Player v7.0全部安装完成后，占用的磁盘空间大约为17.2MB，最后只需重新启动系统即可正常使用。

界　　面 　

　重新启动系统后，发现桌面上已自动添加了“Windows Media Player”的快捷方式图标，而在任务栏右端的系统图标列中出现添加了在线更新提示图标。双击快捷方式图标后，竟然需要等待大约一分钟媒体播放器才载入完毕，这主要是因为Windows Media Player的默认运行模式为在线工作模式，启动时需要从网上下载部分数据信息的缘故。如果将“File”菜单中的“Work offline”（脱机工作模式）选定后，关闭媒体播放器再重新运行，这时启动速度就相当快了。
　　Windows Media Player的运行界面与以前的版本比较起来显然存在较大的区别，它提供了一个“All－In－One”的整合操作界面，整体上给人一种美观大方的感觉，显得新颖别致。Windows Media Player主界面为多页切换方式，通过点击窗口左边并列的七个按钮即可进入不同的子页面，它们分别是： Now Playing（当前播放浏览页面）、Media Guide（网上媒体资讯向导页面）、CD Audio（音频CD复制页面）、Media Library（媒体资料库管理页面）、Radio Tuner（网络广播/频道向导页面）、Portable Device（便携设备数据复制页面）和Skin Chooser（外壳设置页面）。在主窗口右上方并排的三个按钮依次为显示/隐藏均衡器及其他设置、显示/隐藏播放列表、随机播放设置，最右边为播放列表选择；主窗口下端为通用播放控制条，其上分布（从左到右）有：播放/暂停、停止、静音、音量调节、上一首、快倒、快进、下一首及播放进度条，最右端为显示模式（完整/压缩）切换按钮，所有的操作均可通过菜单或快捷键来实现。

设　　置

　　在Windows Media Player 的主界面中选择“Tools”——>“Options”即可进入设置对话框，共分七个页面：
　　1．“Player”页：设置启动页面（默认为Media Guide）、自动升级检查频率以及相关网络设置；
　　2．“Network”页：选择使用哪些网络协议及进行代理服务器设置；
　　3．“CD Audio”页：设置CD音频数据复制质量；
　　4．“Portable Device”页：设置将音乐复制到便携式设备的质量；
　　5．“Performance”页：设置网络连接速度和缓存以及视频硬件加速功能；
　　6．“Media Library”页：设置其他应用程序及网站的访问权限；
　　7．“Formats”页：设置与Windows Media Player相关联的媒体文件类型。
　　Windows Media Player提供了完整窗口和压缩窗口两种显示模式，可通过右下角的切换按钮，以及“View”菜单的“Full Mode”和“Compact Mode”选项进行切换；在“View”菜单的“Now Playing Tools”子菜单中设置可显示的内容有：播放列表、标题、设置窗口及窗口大小控制条；并且通过“Play”菜单的“Shuffle”和“Repeat”选项分别设置随机和重复播放。此外，在当前播放页面（Now Playing）中还有两个非常重要的设置：均衡器（Graphic Equalizer）和视频设置（Video Settings，），但前提是首先必须选择“显示设置”窗口，窗口间可通过“向前/向后”进行按钮切换。在均衡器设置中提供了默认（Default）、流行（Pop）、摇滚（Rock）及自定义（Custom）四种模式，并通过“Balance”（平衡滑块）调节左右声道；在视频设置中可调节显示亮度（brightness）、对比度（contrast）、色调（hue）及饱和度（saturation）。 主要功能试用 　　微软宣称全新的Windows Media Player 7.0将为用户带来第一流的优质服务、无比完善的强大功能。事实到底如何，现在请看我们对其各项功能进行的试用，其主要功能包括以下几个方面：
　　1．多媒体播放功能：
　　作为一个老牌的媒体播放器，Windows Media Player支持几乎所有的媒体文件格式，包括Mp3、Wav、Midi、Rmi、Aif、Snd、Au等音频文件格式和Mpg、Avi、Dat、Mov、Ivf等视频文件格式。此外，它还支持通过URL（统一资源定位）打开网上媒体文件进行实时播放。在播放音乐时，“Now Playing”页中增加了多种可选的二维或三维动态效果显示，非常酷。在播放视频时，图像窗口大小可自动适应调整，并支持全屏播放（快捷键Alt＋Enter），在各种显示模式下均能保持清晰和稳定的图像。
　　2．多媒体资料库管理功能：
　　使用过媒体播放器以前版本的用户一定清楚，它们都不支持清单播放，并且一次只能打开某一个文件，使用起来非常不方便。现在，新版的Windows Media Player不仅支持自定义播放列表，而且提供了更为强大的媒体资料库管理和维护功能，支持曲目的搜寻、排序、添加、删除和移动等操作。通过“Tools”菜单中的“Search Computer For Media”选项(快捷键Ctrl＋E，)，可自动将你的电脑中的所有媒体文件信息存入资料库中，并会根据不同的专集、艺术家和流派分别进行分类。通过点击“New Playlist”可直接创建播放列表，并支持列表文件的导入和导出。
　　3．网上信息收集功能：
　　Windows Media Player 7.0新增加的“Media Guide”(网络媒体向导)就好像是一张图文并茂的电子版新闻快报，它依托于WindowsMedia.com网站(http://windowsmedia.microsoft.com/default.asp)，每日将网上的各种最新的影视、音乐等焦点信息及时呈现在用户面前。用户还可以方便地对自己感兴趣的网上音频或视频信息进行下载、播放、搜寻、查找等操作，但前提是你必须已连接上互联网。
　　4．网络“收音机”功能：
　　Windows Media Player 7.0新增加的“Radio Tuner”(网络广播频道向导)就好像是一个网络调频收音机，它同样依托于WindowsMedia.com网站。通过Radio Tuner，用户可以快速、直接地与超过1500个以上的网络广播频道连接，享受基于Windows Media 格式的高质量声音传送服务，当然前提还是你必须已连按上互联网，并且连线速度相对较快。
　　5．Audio CD音乐播放及复制功能：
　　除了支持Audio CD音轨（CDA）播放之外，该新版本还增加了CD音乐复制功能。这种功能允许用户将需要复制的音乐CD放入光驱后，在“CD Audio”页中选定你想要复制的曲目，点击“Copy Music”按钮即可进行复制。它还会自动以Windows Media特殊的压缩格式（wma）对音轨压缩后保存到硬盘上。微软称该款最新版能在保持CD音质的前提下提供比Mp3高一倍的压缩比，但经我们试用后发现其压缩比与Mp3基本相仿，并没有显著的提高(也许还是预览版的缘故)。
　　6．便携设备数据复制功能：
　　针对目前市面上掌上电脑、Mp3随身听等各种便携式设备的日趋普及，微软也及时在其新一代媒体播放器中加入了向便携式设备传输数据的功能。在“Portable Device”页中，选定你想要复制的曲目和设备，再点击“Copy Music”按钮即可实现信息的传输。
　　7．“换肤”功能：
　　以前的Windows Media Player在外观上显得过于简单朴实，长期以来也没有什么变化，给人一种“老土”的感觉。但在新版中，它借鉴了时下非常流行的如Winamp等软件使用的 “换肤” 功能。除了默认的界面和以前常用的界面，它还提供了两种可选的外壳（Skin，）。切换外壳的具体操作步骤是：首先进入“Skin Chooser”页面，在列表框中选择你喜欢的外壳，然后点击“Apply Skin”按钮即可。如果你想要试试其他更多的外壳，可点击“More Skins”按钮直接从网上下载。

试用总结

　　通过对Windows Media Player 7.0的多次试用，我们认为微软此次针对媒体播放器的“大手术”在一定程度上是相当成功的，主要体现在以下三个方面：
　　1．进一步完善和统一：在广泛支持各种常见的媒体文件格式的播放基础上，又增强了媒体文件管理维护功能，特别是扩展了对网络媒体的支持。
　　2．简便的操作：清晰、简洁而强大有力的用户界面，确保了几乎所有常用操作都只需鼠标点击即可实现，相对于以前版本，在易用性上有了很大的进步。
　　3．个性化的设置：完全用户自定义界面的设计带给用户更大的自由选择空间，增强的媒体资料库管理功能也为用户根据自己的个人喜好建立自己的收藏夹提供了方便，因此，其界面设计更具人性化。
　　当然，目前它也还存在着一些不足，比如对机器配置要求较高、占用系统资源较多、与其他声音事件并发时常出现冲突以及其他一些BUG，也许是它还处于测试阶段的缘故，至少现在它还不会取代Winamp在用户电脑中的地位，但是基于对Windows Media Player 7.0的初步了解，我们对它的未来前景还是比较看好。
　　有兴趣的读者朋友可以到微软的网站上去查看详细信息，(http://www.microsoft.com/windows/mediaplayer/en/download/Playerv7.asp)。最后，我们还是希望Windows Media Player 7.0正式版能够早日推出！
