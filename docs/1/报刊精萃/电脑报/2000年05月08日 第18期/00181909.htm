《 多功能浏览器 Fantasia Web Explorer 》

　　Fantasia Web Explorer是一款精巧的多窗口浏览器，类似于国外的NetCaptor和国产的TurboIDE。它也是基于IE内核，在同一窗口内可以打开多个页面，而且数量没有限制，唯一的限制就是机器本身的资源了。与同类软件相比，Fantasia Web Explorer自然有过人之处，否则也就没有必要推荐给大家了。
　　Fantasia Web Explorer最主要特点就是节省资源，不少拨号上网的朋友为了提高上网效率，都习惯同时打开多个IE窗口浏览，如果稍不注意，很容易导致Windows 系统资源耗尽（就是出现“程序没有反应”的错误），而多窗口浏览器的作用正是为了解决这个问题。据软件说明文件中提到：在Fantasia Web Explorer里面开20个页面，和开3个IE浏览器消耗的资源是一样的，并且页面数目不限。软件在状态栏中还提供了实时系统资源显示，可以让你更加有效地掌握系统资源的分配情况。一旦发现资源低于50％，我建议你还是及早收手为妙，否则很可能悔之晚矣。 　　此浏览器的界面及控制都制作得很完善，完全兼容IE标准界面，包括按钮设定，地址栏自动填充功能，状态栏链接提示等，可以轻松上手。在选项中还提供了“总是打开新页面”和“自动激活新页面”的选择，前者可以让你可以一口气在地址栏里连续输入你想要访问的URL，而Fantasia会在后台自动为你输入的每一个URL打开一个页面进行浏览，后者则允许连续点击你想要访问的URL，系统会自动在后台打开你的连接而不会切换当前页面。这个选项对上聊天室、讨论区、Web BBS等一个页面上有许多链接的网站很有用。
　　在浏览网站时，你是否经常为不请自来的广告窗口感到特别无奈？以往只能手工关闭或使用专门的广告过滤软件，使用Fantasia后，这些烦恼将烟消云散，因为Fantasia内置了页面过滤功能。只要在要关闭的广告窗口标签上点击右键，选择“加入弹出窗口过滤列表”，下次在访问这个网站，Fantasia会自动将广告窗口关闭，让你省心省力，当然，Fantasia也提供了过滤列表管理器，让你可以自由增删要过滤的Web页面。
　　除了上述浏览器功能外，Fantasia还内置22个中英文分类搜索引擎（新闻、MP3、软件、驱动……），可以轻松查找你需要的资料，并且搜索引擎可以通过下载数据包来实现更新。此外，Fantasia还附带了一些网络工具，包括帮助你建立本地DNS服务的本地解析信息管理器、常用IP工具包、IE与NetScape书签转换、独立关闭或打开页面的图片显示等等，真可称得上一专多能。
　　功能如此丰富的浏览器，其下载压缩包只有600余K，解压后也不过700多K，而且属于免费的绿色软件，没有广告条，无须安装，解压后可以直接使用。当然，完美的软件是不存在的，Fantasia也不例外，除了应该增加成批打开网站的功能外，软件的稳定性还应进一步增强（当然，由于以IE为内核，Fantasia的稳定性很大程度上取决于IE的稳定性）。但是，瑕不掩瑜，如果你与我一样对这款多功能浏览器感兴趣，不妨到作者主页（http://xiaxin.heha.net/mysoft/Fantasia.zip）下载最新版，一试为快。

　　（福建　林　斌）
