《 移动的BYTE 》

　　现在人们用于交换的数据文件、应用程序、影像文件越来越大，动辄就上百MB。用软盘作为交换性存贮载体的时代已一去不复返，因为所交换的数据量极大，人们往往用硬盘作为交换存贮载体，但硬盘是非常脆弱的，不适合作为移动性交换存贮载体。其实，主要用于交换数据的便携式存贮设备早已出现了，只不过投资较大罢了。早在五六年前，就出现了Zip，高达100MB的容量，多样化的接口界面选择（共有内置SCSI、外置SCSI、内置IDE及并口LPT），不错的性能让消费者很快的就接受了它。近年来，容量高达250MB的Zip250也出现在市场上。与此同时，其它的新兴交换性存贮设备也纷纷上阵，容量为120MB的LS－120 以较低廉的售价，向下兼容1.44MB磁盘的特性在早期占据了一定的市场，最近容量高达2GB的Jaz交换性存贮设备以其空前的大容量，多用户接口，接近硬盘的数据传输速度闪亮登场。其它的例如CD－R/RW、MO、抽取式硬碟、HiFD等等都纷纷上市，来抢占这片领域。面对如此众多的交换性存贮设备，简直让人眼花缭乱。为此我们在这里为大家作一次综合的介绍。

普及型产品——CD－RW

　　毫无疑问，当你的硬盘已经装不下你辛苦收集的文件、图片和程序时；当你需要保存自己亲手制作的多媒体程序时；当你急需保存重要的文件备份的时候，你首先会想到刻张光盘来保存。 　　由于光驱现在是个人电脑中不可缺少的一个部件，光盘的流通和使用的几率远远超过了软盘。而且现在光盘刻
录机的技术比较成熟，早期的CD－R（一次性光盘刻录机）已经被能重复写入的CD－RW刻录机代替。价格也下降到2000元以内，一次性刻录盘片的价格也在4元～15元之间，当我们想扩充存储设备时，自然而然会将其列入考虑的范围之内，毕竟自己刻盘已成了一种时尚潮流。
　　虽然CD－RW的功能具有四倍读写的能力，但是在格式化光盘时，是以一倍速去格式化，这点应算是RW唯一的缺点，不过随后就能享受四倍写入的快感。随意的将资料存入、删除，表现一样优良。目前CD－RW的刻录机支持所有 CD 格式，对目前的 CD－ROM 保持全面向下兼容。此外，制作出来的CD－RW光盘可以用DVD驱动器播放，从而保证了未来的兼容性。
　　CD－RW的优点在于良好的兼容性，用它作为存贮设备几乎可以走遍天下。要把CD－RW称为移动存贮设备未免牵强一点，因为它既不便于携带，安装也比较麻烦。但是CD－ROM的普及程度和存储介质（CD－RW光盘）的价格确实是CD－RW无可比拟的优势，CD－RW是目前普通用户最好的选择。 参考价格：驱动器2500元，盘片60元。

专业用户的选择——MO驱动器

　　MO十分类似于活动硬盘，常见的MO驱动器分为3.5英寸和5.25英寸。5.25英寸的MO容量有三种：230MB、640MB和2.6GB，3.5英寸的MO只有230MB和640MB两种。在转速方面，目前MO最低转速3600转，最高4643转，当然越高的转速，相对的售价上也会比较高。MO驱动器很有意思，它的性能和容量成正比。而且，MO驱动器保持向下兼容，小容量的盘片可在大容量的驱动器上用。市面上的MO有五种界面，分别是IEEE 1394、SCSI、USB、LPT、IDE。
　　一套标准的SCSI接口MO设备包括一台外置MO驱动器、一块与Adaptec AHA15XX系列兼容的SCSI ISA插卡、一根用来连接外置MO驱动器的专用SCSI电缆、一根内置的SCSI连接排线、还有MO专用电源插头。MO驱动器是一个白色的长方形盒子。正面有一个电源指示灯，插口右上方是弹出按钮，同时也是盘片操作指示灯。背面有两个标准的SCSI连接插口，电源接口，还有手动的DIP跳线开关，用来设置如PC或MAC模式、SCSI类型、SCSI－ID号、HDD或MO模式等等。我们试用的是一台型号为Slim FN640，SCSI接口的富士通3.5英寸640MB的外置MO驱动器，这款MO的接口界面是SCSI－Ⅱ，2MB数据缓存，支持碟片的格式有128 MB、230 MB、540 MB、640 MB四种；最大容量640M；转速为3600RPM；平均寻道时间为35ms；同步传输率最大3.9MB/s。
　　MO 在使用上十分方便，插好SCSI卡，连接上MO驱动器开机以后，在“我的电脑”里面出现了一个可移动的磁盘，这就是MO驱动器的盘符，插入一张MO光盘，你就可以像操作硬盘一样使用它了。打开它的时间要比打开硬盘慢一些，但速度远远快于软驱，而且你基本上听不到咔嚓的读盘声。甚至声音还小于一般光驱的读盘声音。我们计算了一下MO拷贝文件的时间：从硬盘完全拷贝640MB的MP3文件大约需要28分15秒左右， 1.44M文件拷贝不到十秒，速度挺快的。
　　在MO盘上删除文件就同在硬盘上操作一样快捷，格式化MO盘片也同样迅速。你还可以直接在上面执行应用程序，如安装大型软件、运行大型游戏、播放多媒体文件等等，速度同硬盘相差不远。
　　MO驱动器在PC和苹果电脑上都能很方便的使用。以前MO驱动器和盘片的价格很贵，几乎成了苹果机用户的“专属设备”。但现在状况已完全改观，MO驱动器（SCSI接口）单台的价格已经下降到2500元～2600元左右，640MB单张的盘片60元左右，230MB单张盘片才30元。加之读写速度快、性能可靠（一张盘片可读写保存上十年）、携带方便等优点，MO倒是一种不错的选择，特别是现在软件容量越来越庞大，多媒体文件泛滥成灾的年代。MO的弱点是不能读取普通光盘，所以多用于一些特殊行业，例如现在出版界几乎就是MO的天下。 参考价格：驱动器2900元，盘片80元。

软驱二世——Zip100 驱动器

　　现在，市场上交换性便携式存贮设备比较多，如果以容量大小来区分的话，Zip 算是一款主打低阶市场的产品。以Zip250 USB的表现的确让人有耳目一新的感觉，算是低档储存市场中相当强势的代表性产品。但若以价格衡量，Zip100则更有优势。
　　Zip100也有多种接口界面（USB、内接SCSI、外接SCSI、内接IDE及打印机接口），由于接口界面的不同，Zip100 同Zip250 一样，分为Zip100 USB、Zip100 SCSI、Zip100 IDE等这几类产品。由于接口是打印并口的Zip100 所以速度比较慢，已不那么常用了。
　　这款NEC公司的Zip100 IDE的外型同1.44MB的软盘驱动器非常相似，大小也差不多。不仔细看，还以为它就是软盘驱动器呢！它不支持热插拔功能，但它的数据传输速度比USB接口的Zip要快些。在安装Zip100 IDE时，不需要安装驱动程序，直接接入开机，系统会自动识别Zip100 IDE。在对Zip100 IDE进行数据存取时，也非常方便，同软驱操作差不多。它同Zip250 USB一样，提供对数据的软件写保护（不需要安装特别的软件,系统自带00/04/28），用软件对磁盘进行写保护，而不需要使用机械的写保护开关。对磁盘进行写保护并指定一个口令，这样必须先输入口令，否则无法添加、编辑或删除其中的文件。还可对Zip100 IDE进行读／写保护，这样对磁盘进行读/写保护后，除非用户输入了指定的口令，否则不能对该盘进行读写操作。当然，若你忘记了自已所设的密码，那你为了使用这张Zip 磁盘，唯一的办法是对Zip 磁盘进行完全格式化，这样你将丢失原磁盘上的所有数据。
　　在操作系统方面，Zip100 IDE兼容Windows 95, Windows 98/2000, Windows NT及Windows 3.X。它只能使用100MB的Zip磁盘,对于250MB 的Zip 磁盘无法识别。若你把250MB的Zip 磁盘插入Zip100驱动器中，它会自动弹出。它不兼容1.44MB的软盘，让人感到有点遗憾。Zip100 IDE的最大数据传输速率为1.4MB/s，在进行数据备份时，不会让人有等待太久的感觉。同样，我们也对Zip100 IDE作了一个小测试。在进行50MB的数据写入时，Zip100 IDE花了1分23秒，而Zip250 USB用了1分10秒，速度相差并不大。只不过在大容量数据存取时，同Zip250 USB的差距比较大。但这并不说明Zip100 会被Zip250 系列取代，它们的价格不一样，面向的市场也不一样，Zip100 IDE面向的是更低的交换性存贮设备市场。
　　在这里，还为大家介绍一下Zip100 USB,它同Zip100 IDE规格差不多。但由于它是USB接口，它支持热插拔功能。能在不关机的情况下，安装Zip100 USB。它需要驱动程序，但这些都是自动的，非常简单。值得说明的是，在系统兼容性方面，它不同于Zip250 USB。它仅兼容Windows 98/2000, Mac OS 8.1及以上版本，对于Windows 95及Windows NT来说，不能使用Zip100 USB。
　　Zip 驱动器以其较大的存贮容量、多样化的界面选择（USB、内接SCSI、外接SCSI、内接IDE及打印机接口）、不错的能让消费者很快的就接受了它。现在，软驱多已被排除在标准配置以外，许多系统电脑厂商（康柏、Apple等）都已将Zip列为标准配备，许多主板都支持以Zip来开机，加上Zip安装简单使用方便的特性，Zip几乎成为软驱明正言顺的接班人。 参考价格：驱动器1000元，盘片110元。 潮流和时尚——Zip250 USB 　　Zip250有多种接口类型，其中有USB、SCSI、打印机接口、ATAPI,以及 PCMCIA(笔记本接口)等。根据接口类型的不同，Zip250 分成Zip250 USB、Zip250 SCSI、Zip ATAPI(IDE)等这几类产品。它承袭了Zip100的优点，在性能上更加强悍了，整体设计也更加可人。Zip250系列中最新的成员是Zip250 USB，这款Zip250 USB 是Iomega 公司生产的，它采用了类iMac的外型设计，通身蓝色并且还是半透明的，同乳白的机箱、键盘、显示器构成了鲜明的对比，给人以耳目一新的感觉。
　　Zip250 USB背后除了有一个USB插槽外，还有一个PCMCIA接头，这个接头让Zip250 USB的应用层面更加的扩大，不但是桌上型电脑连笔记型电脑也可以使用250USB。Zip250 USB每分钟高达2941转的高转速使它的耗电量达到750mA，远超过USB接口的供电量（USB仅能供应500mA），因此一个Zip250 USB的电源供应器是必不可少的。Zip250 USB的电源供应器相当的小巧可爱，更重要的是这个电源供应器支持多国电压。需要说明的是，在使用PCMCIA接口连接笔记本电脑时，并不需要外接电源。特别应注意不要将USB数据缆线和PCMCIA适配器同时连接到Zip250驱动器上，否则它不能工作。
　　Zip250 USB是一台USB界面的Zip磁碟机。热插拔、随插即用是USB的特色，不用拆机壳，没有烦琐的软件安装设定，可以说是一步到位，相当简单。在使用方面，Zip250 USB就如同一般的磁碟机一样，使用者可以直接在上面增删档案存取资料，不须要特别的学习就可以上手。
　　在兼容性方面，Zip250 USB不仅能用250MB Zip磁盘，也能用100MB Zip磁盘，全系列的Zip250都向下兼容100MB Zip磁盘。不过，有一点要注意的是Zip250是以小读写头去读取高密度碟片，对於Zip250磁片会有比较好的读取效能。如果你用的是Zip100磁片，此时的存贮访问速率就同Zip100设备差不多。这在下面的小测试中，我们将会看到。相反，若把Zip250MB的磁盘放入Zip100的驱动器，它会自动弹出，不能被识别。在这里，提醒大家一下，最好不要把非Zip 磁盘插入Zip250 USB中（例如软盘），这样会严重损坏Zip驱动器。
　　对于交换性的存贮设备而言，往往最关心的是它的数据转输速率。由于Zip250 USB的USB的设计，这使得它在资料的传输上摆脱过去打印机接口性能不佳的困扰，在数据写入方面，整体上比40×光驱仅弱30％左右。以240MB数据传输为例，Zip250 USB为5分钟，而光驱也花了3分5秒。而在数据读出时，Zip250 USB为4分40秒，更接近光驱速度。在进行数据备份时，这种速度不会让人有太久的等待。
　　同Zip100 相比，在进行大容量数据存贮时，Zip250 USB的速度提高了近50％左右。但在存取较小容量的数据时，速度差不多，这同它们的速度指标是一致的。在此，有一个问题要注意，Zip250 USB在对100MB的Zip磁盘进行存取时，速度会明显下降，同Zip100 驱动器的速度差不多。特别是有时向 Zip250 USB 驱动器的 100MB Zip 盘写入特大文件时，系统可能呈现挂起状态。此时你可以通过检查按钮上的绿色指示灯是亮还是不规则闪烁，来确认 Zip 驱动器是否正确工作。在这里，我们当然提倡使用250MB的Zip盘。
　　在关于操作系统的兼容性方面，Zip250 USB兼容Windows 3.1、Windows 95/98/2000、Windows NT 3.51及以上版本、MacOS System 7.0 及更高版本。 参考价格：驱动器1980元，盘片210元。 大肚能容——艾美加Jaz 2GB 　　早在五六年前Iomega就推出了Zip，它高达100MB的容量多样化的界面选择（共有内接SCSI、外接SCSI、内接IDE及印表机连接埠），以及不错的效能让消费者很快的就接受了它。现在Iomega 已推出了容量高达2G 的portable ultra SCSI 驱动器——Jaz 2GB。
　　Jaz 2GB每张盘具有2GB的存储空间，能够储存2000张640×480 pixels的图片，3个小时以上的CD唱片音效资料，以及111分钟的Video 2X CD－ROM资料，保持与前款Jaz 1GB兼容，速度是Jaz 1GB驱动器的两倍。 　　Jaz 2GB整机成墨绿色，与我们身边的冷漠、灰白色的计算机形成了鲜明的对比，使人感觉很舒服。作为一种移动存储设备，它的重量不足1公斤。在Jaz 2GB的整套配件包括一根数据电缆，一个外部电源适配器，一个50HD—25D电缆转换器以及一个Jaz Traveller SCSI，这些都是你安装Jaz 2GB都要用到的。
　　在安装Jaz 2GB之前，你必须确定你的机子上安装的是Microsoft Windows 98、Windows 95、 Windows NT、Windows for Workgroups 3.11、Windows 3.1、DOS 6.0或更高版本的系统，或者是Mac OS System 7.0或更高版本的系统。Jaz 2GB的安装方法与我们平时安装软驱、硬盘差不多，只不过Jaz 2GB采用了SCSI接口。如果没有SCSI接口卡，则必须用到上面提到的Jaz Traveller SCSI，它是一个并行口与SCSI接口转换器，把它连接到Jaz 2GB背面的SCSI接口，再在它的另一端接上数据电缆，数据电缆的另一端接到计算机的SCSI接口上，硬件安装就算完成了。启动计算机，根据提示完成Jaz 2GB的软件安装，整个安装过程就完成了。不用拆机壳，也没有烦琐的软件安装设定，相当的简单。
　　但在这里必须提醒大家注意一个问题，我们在上面也提到了，那就是Jaz 2GB采用的是Ultra SCSI接口。也就是说，为了充分发挥Jaz 2GB的性能，你必需有一块SCSI接口控制卡。因为我们在试用中发现，通过Jaz Traveller SCSI转换器，把Jaz 2GB连接到计算机的并行口上的时候，Jaz 2GB读写速度要比接SCSI接口时下降许多， Jaz 2GB速度快的优势在这种情况下根本得不到发挥。这个问题也是目前采用并行口界面的便携式移动存储器所存在的一个普遍问题，原因是由于受到了并行口传输速率的限制，不能实现数据的高速传输。
　　对一个高性能的存储器来说，除了容量以外，其数据传输率也是衡量其性能高低的重要标志，因为没有人愿意把宝贵的时间浪费在大量的数据读写上面。但对Jaz 2GB来说，高达2GB的容量，如果读写速度不尽如人意，那么你将发现备份数据也是件十分痛苦的事情。Iomega 公司在Jaz 2GB上已注意到这个问题，他们在增大容量的同时，也注意到了数据传输速率的提高。与前款产品Jaz 1GB相比较，Jaz 2GB平均数据传输速率达到了7.4MB/s，最高可达8.7MB/s，转速为5394 RPM，平均寻道时间为10毫秒读，12毫秒写，这么快的速度，再加上2GB的容量，Jaz使你能够以硬盘的速度获取资料，甚至能在硬盘出现故障时让你通过一个Jaz活动硬盘引导你的系统，这不正是我们所梦寐以求的吗。
　　综上所述，我们可以看出，Jaz 2GB以其优异的性能，必将在当今互联网及高频宽应用日益普及，各种信息朝着“数字化”和“多媒体”这两大方向不断转型，电脑用户从网上下载的信息量有增无减，而数字摄影、数字影像处理等技术又发展蓬勃的今天占有一席之地。如果你需要进行大量数据备份，或者需要移动存储，那么Jaz 2GB将是你的一个不错的选择。 参考价格：驱动器3950元，盘片1050元。

轻巧灵便——Clik! 40驱动器

　　现在，我们来介绍一款专门用于笔记本电脑的Clik! 40 PC Card驱动器。这款Iomega公司出产的Clik! PC Card驱动器非常小，呈长方形，厚度同一张磁盘差不多，用户可以轻易地将驱动器和磁盘随身携带。Clik! 40使用笔记本电脑的PCMCIA插槽。它专门用于笔记本用户进行数据共享、交换。Clik! 40使用很简单，只须要把它插入笔记本的PCMCIA插槽中，不用连接其它的数据线及电源线，系统会自动识别它。在使用时，Clik! 40的自带软件Quik Sync会自动对系统的重要数据进行备份，并对你的最近修改文档进行跟踪。
　　Clik! 40驱动器使用容量为40MB的Clik! 40磁盘，40MB 的容量对于笔记本用户来说已足够了。它的数据传输速率为620KB/s，纵然备份40MB的数据，所用时间也不超过2分钟。它消耗电能很少，在不进行数据存取时，Clik! PC Card驱动器会自动在3秒钟内进入睡眠状态。在兼容性方面，Clik! 40驱动器只能使用在Windows95/98系统下。它的接口类型为PC TYPE Ⅱ/ATAPI,同Clik! Mobile驱动器是兼容的。
　　Clik! 40同样有采用USB接口的型号，适用于台式机用户。在目前的移动存储设备中，Clik! 40的轻巧便捷是其它产品无法相比的。 参考价格：驱动器2100元，盘片150元。 结语 除了上面我们介绍的几种驱动器外，市面上还有很多其它的设备可以用于数据的备份和转移，例如：LS－120、Flash Card驱动器和具有文件传输功能的MP3播放器等。这些设备或者技术过时，或者不够主流，有的甚至不是专门的存储设备。不管怎么说，越来越多的用户正在开始使用移动存储设备，假以时日，它必将和CPU、硬盘一样，成为个人电脑中不可缺少的配件。
