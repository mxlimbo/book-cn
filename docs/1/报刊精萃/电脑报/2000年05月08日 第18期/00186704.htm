《 黑暗中，有一双眼睛 》

生化危机3：最后逃亡

　　日版名称：BioHazard 3: Last Escape
　　美版名称：Resident Evil 3: Nemesis（复仇女神）
　　推出时间：1999年
　　系统平台：PC（移植中）、PS（可模拟）
　　主要角色：
　　卡洛斯·奥利维拉（Carlos Oliveira）：算是男主角。有南美血统的史泰龙式硬汉，Umbrella公司“处理生化危机事态”（UBCS）雇佣兵，是全系列《生化危机》中最强悍的配角，与吉尔共同开始了“最后逃亡”。
　　妮克莱·吉诺瓦（Nikolai Ginovaef）：UBCS的小头目，《生化危机3》中的最大反角。
　　《生化危机3》的故事开始于利昂抵达拉孔镇前大约两个小时，那时G病毒还没有完全蔓延开来。玩家扮演的是《生化危机1》的女主角吉尔，自从查尔斯被排挤出警局后，希望彻底调查真相的她，预感到可怕的威胁离自己越来越近，这感觉与在那神秘别墅中的一模一样……
　　总的来说，《最后逃亡》基本沿用了《生化危机2》的引擎，为了使冒险区域更广阔，采用了大量户外地形，此外还提供了更丰富的“物品组合设计”以及躲避攻击的“回避系统”等特性。尤其刺激的是，本集新加入一个手持火箭筒的“追踪者”，亦步亦趋的尾随着吉尔，经常冷不防从什么地方冒出来（包括破门而入，这在前两代中是从没有过的），打玩家个措手不及。游戏之所以叫“最后逃亡”，与它有着直接的关系（追踪者一出现，想不逃都不行）。
　　《生化危机3》看起来有一些另类，对于生化迷来说，游戏前一半的情景很接近《生化危机2》，后一半才算真正开始。《生化危机3》的突发性事件明显增多，突出了惊怵感和所谓“恐怖幸存”的基调。相较《生化危机2》来说，普通模式的难度有所增加，简单模式的难度则大幅度降低。如此设定的好处是当老鸟竭尽全力浴血奋战时，菜鸟玩家也可以从容大爽一把，可谓 “各得其所”，肯定会受到各个层次玩家的欢迎，仅这个细节就可以看出卡普空的功力。
　　《生化危机3》的剧情要比前作短一些，而且只有一个主角。不过，卡普空尝试了设定分支剧情，就是说假如玩家解谜顺序不同，剧情发展会稍有差异，当然，对总体故事结构并无根本性影响。总之，《生化危机3》虽然没有辜负玩家对它的期待，但与《生化危机2》相比，多少有点原地踏步的感觉，感觉更像一部资料片。
　　从《生化危机1》起大家可能已经注意到了这样一个有趣的现象，就是《生化危机》的英文名字有点乱，日版是“BioHazard”，美版改成了“Resident Evil”，《生化危机3》是系列中第一部加上副标题的作品，更夸张的是，“BioHazard 3——Last Escape”（最后逃亡）和“Resident Evil 3——Nemesis”（复仇女神）单从名字上根本就看不出来它们是同一部游戏！这并不是卡普空故弄玄虚，造成此种情况的原因是版权问题，据说在《生化危机》销往美国以前，老美有部电影的名字就叫“BioHazard”，这样游戏就只好改名。
　　隐藏剧情：
　　《生化危机3》加入了一个类似“赏金猎人”的迷你游戏，玩家可任选一个角色，从电车站开始行动。角色身上捆着一个定时炸弹，必须在规定时间内到达规定地点才能解除，否则的话……在路上杀死敌人可获得奖励时间，顺利到达后视成绩获得数目不等的“奖金”，然后用这些奖金购买武器，比如无限弹药机关枪、火箭筒等在剧情中使用。

别传：代号维罗妮卡

　　日版名称：BioHazard: Code Veronica
　　美版名称：Resident Evil: Code Veronica
　　发售日期：2000年
　　系统平台：Dreamcast（不可模拟）
　　虽然卡普空并不像史克威尔或南梦宫那样与索尼结成了“战略伙伴关系”，但由于土星3D机能较弱的缘故，卡普空在一代之后便抛弃了它，让世嘉的拥护者好不失望。新一代家用电视游戏机的“先行者”DC推出后，其3D性能全面超越PS，卡普空终于答应再为其专门制作一部全新的《生化危机》游戏。
　　游戏主角仍然是那位在《生化危机2》中陪伴我们与僵尸战斗的美女克莱尔。逃离并炸毁Umbrella公司后，她和里昂两人分手，继续寻找哥哥查尔斯。三个月后，克莱尔在欧洲某个废弃军事基地进行搜索时，突然落入一个神秘组织手中，被囚禁在某孤岛上……她醒过来，发现自己被大批丑恶的僵尸及怪物包围，更糟的是孤立无援，只能用手中的枪摆脱危机……
　　剧情和玩法同样遵循了所谓“《生化危机》标准”，由于DC的手柄和PS差距较大，所以操作加入了180度突然转身等新要素。游戏还是一男一女两位主角，为了弥补前几集“情感”部分较弱的缺失，《代号维罗妮卡》中加入了男女主角的一段生死恋，意欲给这套充满血腥恐怖的系列增加一丝柔情，不过感染力还是无法与RPG相比。好在玩家也并不是太在意，抱怨最多的倒是DC手柄用起来不如PS习惯，不管怎么说，《代号维罗妮卡》体现了独立于传统“生化”系列的风格，今后甚至有可能顺着这个思路单独发展出一部系列来。
　　隐藏剧情：
 　　《代号维罗妮卡》中，玩家能玩到一个限定时间的迷你战斗游戏，使用的角色包括查尔斯和《生化危机1》中出现过的一些角色，在规定时间完成任务，可得到弹药无限的火箭筒。另外，还有一个第一人称视角模式射击迷你游戏，你可以用狙击步枪等武器进行游戏。

外一篇：枪神

　　日版名称：BioHazard: Gun Survivor
　　美版名称：Resident Evil: Survivor
　　发售日期：1999年
　　系统平台：PS（可模拟）对应光线枪
　　主要角色：
　　就在里昂、克莱尔和吉尔在拉孔市被大批僵尸包围时，另一个地方，一架直升飞机不明原因坠落到地面，命大的幸存者慢慢从地上爬了起来，他已经丧失了记忆，脑海中唯一有印象的只剩下自己的枪法，站起身，这个人缓缓向噩梦走去……
　　很难给这部游戏一个准确的评价，因为根本不知道它究竟从什么地方冒了出来，而且怎么看都像一部骗钱的东东。糟糕的操作（幸好有敌人弱智的AI作为补偿），差劲的画面效果和随处可见支离破碎的贴图虽然并不能说明卡普空不是作FPS游戏的料，但他们肯定没有认真去作。靠着整个系列的威名，它居然也占据了日本游戏销售排行榜榜首一段时间。假设你是个忠实的生化迷并且希望换换口味的话，《枪神》倒也不妨一试。

即将到来：零点

　　日版名称：BioHazard 0
　　美版名称：Resident Evil Zero
　　发售日期：未定
　　系统平台：Nintendo 64(可模拟)
　　N64虽已有英雄迟暮之感，但在美国市场还能苟延残喘，所以据说卡普空决定为它开发一部《生化危机》游戏，取名《零点》。虽然这个消息难免让人有点怀疑——因为N64在日本成绩实在太可怜了，不过从目前来看，至少有板有眼，所以我们还是宁可信其有。
　　《零点》的女主角是《生化危机1》里的瑞贝卡，男主角叫比利·科恩（Billy Cohen），一位退役海军军官，因被指控犯有多项谋杀罪被捕，后来从监狱中逃出。玩家可以在游戏中随时切换男女主角。传言《零点》将是全系列难度最大的，一直伴随着我们的“超时空储物箱”也将被取消，不过你可以在两位主角之间交换物品。至于其他消息嘛……只能去问卡普空了。

遥遥无期：生化危机4

　　发售日期：未定
　　系统平台：PS2(不可模拟)
　　曾有未经证实的消息说卡普空有意在《生化危机》正传发展到四代时转移到PS2平台，并结束整个的故事，给玩家作个“了断”。游戏核心可能围绕罪魁祸首——Umbrella公司总部展开，希望在E3召开时可以得到相关消息，让我们期待吧。
