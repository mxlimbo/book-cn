《 SyGate使用点滴 》

　　1SyGate的安装简单。可以说按照默认设置一路按“Next”也可以安装成功（当然，你必须先安装好网络和Modem），可以说接近Windows98第二版所提供的共享Internet连接工具，只是在客户机上设置IP地址要稍微复杂一点，但方法和第10版文章“使用Windows98第二版建立共享Internet连接”最后介绍的“手动配置客户机的IP地址”相类似。但SyGate的管理功能要强大得多。
　　2SyGate的使用没有什么限制。首先，SyGate的安装提供了服务器安装和客户机安装（安装的时候选择“Server mode”或者“Client mode”）。但其实客户机不安装SyGate的客户端程序也同样可以，使用上也没有什么限制。客户机只要安装了TCP/IP协议（即使客户机不是Win95/98操作系统），就可以进行任何访问操作。
　　3SyGate自动拨号。如果当你有任何Internet访问要求时，可以自动启动SyGate Manager管理程序，选择“Tool/Configuration”菜单，打开设置窗口，然后在“Connection Method”栏中选择“Through Dial－Up Networking”（通过拨号连接），在“Dialup Entry”栏中选择拨号网络，如163或169，然后勾选“Enable Dial－on－Demand for Client”（自动拨号），但要注意，必须在“Input user name and password manully”栏中输入拨号网络的用户名和口令。
　　4SyGate自动断线。在设置（Configuration）窗口中的“Auto Hang Up”栏中输入一个时间，但在这个时间中，没有Internet数据下载和上传，SyGate将自动断线。 　　5限制访问站点。在SyGate中，可以非常方便地限制对某些站点的访问，选择“BWlist Editor”菜单，编辑你想限制的站点列表，这样，网络中的用户就不能访问这些限制的站点了。
