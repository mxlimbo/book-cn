《 专家坐堂 》

提示“Verifying DMI pool data”后死机的解决

　　苏州读者xuyong502问：最近计算机启动时，自检通过，然后出现“Verifying DMI pool data ”的提示就死机了。我已经排除了病毒的可能，望能给我指点！
　　答：这是个相当棘手的疑难问题，据笔者所知不少人都遇到过相同的情形，即验证DMI数据时死机，而且很多情况下都没有好方法能够修复。如果你的机器还能从软驱启动，那么问题可能与DMI无关，解决起来要容易一些，可用系统盘向C盘重传系统，即执行“A:\>sys c:”，绝大多数情况下都能解决。如果不能从软驱启动，则可能是刚刚安装的硬件与系统中其它设备有冲突，或存放DMI数据的EEPROM出现故障，或者EEPROM已装满而无法存入更多的数据，另一种可能原因是电源存在故障。你可以试试拔掉声卡、网卡等暂时不用的设备后再启动，或者清除CMOS设置后再启动，还不行的话可换个电源，能够启动之后首先要做的事情是升级BIOS。所有方法均不行时就只有更换BIOS芯片或者整个主板了。

为何一读某些光盘即死机？

　　读者fhxyy问：我的光驱为华硕40速。读了几百张光盘，但有几张盘总是读不起。比如说《电脑报99精华版》，A盘在别的机器上一读就出，但在我的机器上，光驱刚读出盘的编号就死机，按CTRL＋ALT＋DEL也不能关掉应用程序。直接弹开光驱门则系统显示：A Fatal exception 0E has occurred at 0167:BFF9E463。The current application will be terminated这是为什么？
　　答：从现象来看，你的光驱应该没有问题，可能是因为有的光盘上有自动运行的程序，而这些程序在你的机器上运行会出现非法操作。你可以在“设备管理器”中调出光驱的属性，在设置中去掉“自动插入通告”前面的小钩，确定后重启系统，此后插入光盘时系统就不会自动运行光盘上面的程序，也就不会出现非法操作了。

如何打印prn文件

　　读者阿涛问：Word 97中的打印对话框有两个选项，可以输出到打印机，也可以输出到文件形成一个扩展名为prn的文件，这个文件是不是可以以后打印，怎么打开这个文件？
　　答：prn文件可以在以后打印，它存储了打印文档时所有送往打印机的数据，因此在打印时只须直接将其拷贝到打印机就行了。假设打印机接在计算机的第一个并口上，可在MSDOS窗口中键入并执行以下命令：“COPY /B TESTPRN PRN”或“COPY /B TESTPRN LPT1”。需要注意的是，目标打印机必须与打印文档时选择的打印机型号相同。

怎样在NT下安装调制解调器

　　读者FOX问道：我的电脑安装了Windows 98及NT Sever 40，想用NT来上网，但是我的调制解调器是内置式的，位于COM3端口，因而在安装时系统无法检测到该调制解调器（系统只能检测COM1、COM2端口），请问应如何设置才能使NT能检测到COM3口上的调制解调器？
　　答：在NT下安装驱动程序之前，要确保内置调制解调器使用的端口没有与其它设备发生资源冲突，例如不能与主板上的串口使用相同的资源，必要时可更改CMOS设置，或者调整内置调制解调器的跳线。建议为内置调制解调器使用下面的标准配置之一：COM3（I/O=3E8h，IRQ=4），COM4（I/O=2E8h，IRQ=3）。设置好端口之后，NT在启动时NTDETECT应能自动检测到内置调制解调器使用的端口，在控制面板中添加新的调制解调器，添加过程中选择新的端口，并在此端口上安装厂商提供的驱动程序，或者安装速率最接近的标准调制解调器的驱动程序。如果NTDETECT没有检测到端口，则说明可能该内置调制解调器的兼容性不好，或者端口存在着资源冲突，可再次调整端口的设置，然后试试在控制面板中手工添加一个端口，最后安装调制解调器的驱动程序。

开机时为什么不显示UDMA66等问题

　　读者杨平生问：我有三个问题请教：1．我手头有一块精英双子星P6BAT－A＋主板（支持PC133，VIA芯片），支持UDMA66，还有一块昆腾Plus KA 91G硬盘（7200转，也支持UDMA66）。但是在开机时BIOS并不显示UDMA66，而显示LBA Mode 4，BIOS能正确识别我的硬盘，我用的数据线是不会错的，因为在别人的电脑上是好的。如果改用精英双子星i810主板，开机时BIOS显示DMA33，两块主板在CMOS里都设为Auto，经销商说是主板BIOS不支持，硬盘的实际传输速度是66MB/S。不知是不是这样？2．LBA Mode 4和CD－ROM Mode 2有什么区别？3．我不知如何将主板跳到133MHz的外频，我只找到主板上的AUTO 66/100和100的选择跳线，跳线到AUTO 66/100时可以在CMOS里设置CPU的外频有66、75、83MHz，但是不知如何跳线才可以在CMOS里设置CPU的外频到100MHz以上，如112、133MHz等？
　　答：1．昆腾Plus KA系列硬盘支持UDMA66，P6BAT－A＋也支持UDMA66，通常说硬盘的工作模式时并没有LBA Mode X的说法，因为LBA是表示硬盘扇区的一种方法，而不是指硬盘接口的传输模式。传输模式有UDMA 4/3/2/1、DMA 2/1/0和PIO 4/3/2/1/0等几种，如果BIOS的确显示LBA Mode 4，那么应该是指UDMA Mode 4，而UDMA Mode 4就是UDMA 66，因此BIOS正确识别了硬盘的传输模式，只不过显示上有点问题。你可以在Windows 98安装VIA的IDE驱动程序，之后就可以使用硬盘的UDMA模式了。
　　2．同样的道理，CD－ROM Mode 2实际是指UDMA Mode 2或PIO Mode 2，目前较新的光驱都支持UDMA33，最低也支持PIO Mode 4，因此CD－ROM Mode 2应相当于UDMA33。
　　3．你只能在CMOS中设置66、75、83MHz，这说明你的CPU的标准外频是66MHz的，P6BAT－A＋在这种情况下最多能通过跳线设置100MHz外频，如果你使用标准外频是100MHz的CPU，那么就能在CMOS中设置112、133MHz了。

如何才能自动播放数据光碟

　　读者解凯问：我的光驱的自动读盘功能最近突然没有了，但CD盘能自动读，系统管理器中光驱选项的自动插入通告已选定。请问除了重装Windows 98外如何能恢复？
　　答：Windows 98自动播放CD和数据光碟的功能可以分别控制。在光驱属性中选定自动插入通告后，可以单独指定对CD盘或数据光碟是否启用自动播放功能。为了恢复自动播放数据光碟的功能，你需要安装Windows 98光盘上“Tools\Reskit\Powertoy”目录下的Tweak UI，安装方法是用鼠标右键单击tweakuiinf，从关联菜单中选择安装，安装完毕后从控制面板中启动Tweak UI，然后在Paranoia一栏中选定“Play data CDs automatically”。

DLL文件版本不正确的原因

　　广昌读者周利众问：我最近重装Windows 95，版本400950B,并将IE30升级为IE50，后来又安装了主板驱动程序补丁(华硕P3B－F)。现在机器启动时，系统提示运行的DLL版本不正确，但系统运行正常。重装系统前没有这种情况，请给予指导。
　　答：在安装主板驱动程序的补丁时，请确认该补丁推出的日期要比你所使用IE5安装盘源文件的日期要新。动态链接库文件（DLL文件）自身也有不同版本及不同语言的版本，如果主板驱动程序中的某个DLL文件的版本低于IE5中同名DLL文件，或者是两者语言不同，安装时主板驱动程序带的DLL文件可能覆盖了原来的文件，这样一些软件在运行时就会提示DLL文件版本错误。

Word怎样实现自动滚屏？

　　四川读者马致问：我在用Word 97阅读一大文件时总要不停地翻页，听说Word 97可以利用微软鼠标自动滚屏，我没有微软鼠标，是否可以实现这一功能？
　　答：在Word工具栏中选择“工具→自定义→命令”，然后在左边的活动框中选择“所有命令”后，在右边活动框中选中“AutoScroll”，接着按住鼠标左键不放，把它拖到工具栏的任一个地方，看到一条粗黑线时松开。这时可以用鼠标单击“自动滚动”后即可看到文档自动不停地滚动了。上下移动鼠标可以调整文档滚动的速度。

关机速度慢的解决

　　重庆读者邓德嘉、何昌松问：我的电脑配置是K62－266、ASUS P5A主板、32M内存、MAXTOR3 21G硬盘，操作系统为Windows 98。目前系统运行良好，但就是关机速度非常慢，请指教，谢谢！
　　答：Windows 98有快速关机的设置，在系统关机时，如果由于一些特殊原因（比如突然断电、死机等）没有正常关机，下次运行时系统会自动禁用快速关机，这样关机的速度就会很慢。你可以通过手动的方法关闭禁用快速关机选项，具体做法如下：在Windows 98中运行程序Msconfig，单击“高级”进入高级设置，把“禁用快速关机”前面的钩去掉。
