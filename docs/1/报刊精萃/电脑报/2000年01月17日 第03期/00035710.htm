《 {NT上的免费邮件服务器IMS} 》

{　　电子邮件是现代人不可缺少的一种联络方式，在Intranet上除了建立Web服务器与Ftp服务器，如何选择并安装一个邮件服务器，也是非常关键的。在Windows NT操作系统上，微软公司提供2种邮件服务器，一个是Option Pack所带，一个就是Exchange Server,但二者都比较庞大，安装起来非常费时，配置起来也不简单。The EMWAC Internet Mail Services for Windows NT (简称"IMS")是一个Internet上提供的免费邮件服务器，支持Smtp、Pop3等常用协议，具有许多实用功能，安装简单，管理方便，将它作为Intranet上的邮件服务器，不失为一个很好的选择。下面，就给大家介绍IMS的安装与配置。
　　下载：
　　IMS是免费软件，可以从http://www.emwac.ed.ac.uk/pub/IMS/IMSi386.zip下载获得，目前的版本是0.8X。随机文档是http://www.emwac.ed.ac.uk/pub/IMS/IMShtm.zip。
　　安装：
　　以管理员的身份在Winnt服务器上登录，将下载的IMSi386.zip释放到系统目录\winnt\system32下，在该目录下分别执行命令：
　　SMTPRS －INSTALL
　　SMTPDS －INSTALL
　　POP3S －INSTALL
　　成功后，进入“控制面板－服务”，可以看到服务列表中添加了以下几项：
　　EMWAC POP3 Server
　　EMWAC SMTP Delivery Agent
　　EMWAC SMTP Receiver
　　Windows NT用户权限配置：
　　进入“域用户管理器”，建立一个本地组IMSUsers，在其中添加所有可以使用IMS电子邮件服务的用户，并分配IMSUsers组“Log on as a batch job”权限。
　　建立邮件存放目录，并分配权限：
　　建立一个目录，用于存放往来邮件，比如d:\inetpub\mail，并分配Everyone对该目录Full Control权限（如^03050402a^1）。
　　配置IMS：
　　进入控制面板，可以看到一个EMWAC IMS图标，双击它，启动配置程序，分为Diretory，Misc，Lists，Aliases，Logging五个功能页面（如^03050402b^2）。现分别对五个功能页面介绍如下：
　　（1）Diretory功能页面
　　在Mailbox处输入上面建立的邮件目录，选中Automatically Create Mailbox,其余项目默认。
　　（2）Misc功能页面
　　在Add左边的框中输入邮件服务器的Ip地址或者域名，单击Add加入到Accept Mail For中。如果在多个网络中传送邮件，将网关地址填入SMTP Gateway处。
　　（3）Lists功能页面
　　在这个功能页面中，可以创建邮件列表。
　　（4）Aliases功能页面
　　可以建立邮件地址的别名。
　　（5）Logging功能页面
　　在这里设定日志文件的种类，日志文件位于c:\winnt\system32\emwac\的日志目录下面。 经过以上服务器端的配置，IMS邮件服务器就可以启用了。在客户端，将POP3与SMTP都设置为邮件服务器的IP地址或域名，马上可以自由地收发邮件。}
