《 {FLASH 32--多功能抓图、动画制作工具} 》

{　　笔者用过不少抓图软件，认为这些抓图软件的功能都较为单纯，除了“抓取”和“保存”外，剩下的都是些用处不大的功能。最近从网上下载了这个Flash 32抓图工具，试用后感觉相当不错，该软件不仅能抓取任意形状，还可以用抓取的图像制作成多种动态图像文件，从而大大满足你的创意要求。该工具可从http://www.cnvnet.com/download/d/flash32e.zip或者http://www.perso.infonie.fr/chass下载。
　　一、 主要功能
　　该软件可以运行在Win9x/2000/NT4.0环境，主要有如下功能：
　　1.支持抓取任意形状，并可保存为用于多种平台的105种图像文件格式。
　　2.可以从剪贴板、Internet Cache、WebCams和文件中抓取图像，还可以抓取MS－DOS全屏幕，并能以图像或文本格式保存。
　　3.可以将抓取的图像制作成按钮、图标、动画光标、动态GIF、屏幕保护、Windows墙纸和开关机画面、Html页面等。
　　4.支持抓取动态屏幕，并可将抓取的多幅连续图像制作成AVI、MPEG、动态GIF、FLI和FLC格式的文件，该功能可以使您录制屏幕操作过程，方便了演示作品的制作。
　　5.丰富的图像处理功能，如旋转、变形、颜色处理、调整大小、3D效果、添加边框和文字等。
　　二、启动程序
　　Flash 32的安装非常简单，按照默认设置，一路按“Next”即可完成安装。之后，在Windows任务栏上生成一个照相机样式的快捷图标，可以在该图标上单击左右键弹出Flash 32的操作菜单。在弹出的菜单上指向“Capture”项，出现如^03020604a^1所示菜单，该菜单主要用于选择抓取方式。包括：“Full screen”(抓取全屏幕)、“Region...”（选定抓取的区域）、“Select a windows with the mouse”（用鼠标选择抓取的区域）、“Select a windows in a list”（从列表窗口中选择抓取当前打开的窗口）、“MS－DOS full screen capture”(抓取MS－DOS全屏幕)、“Clipboard capture”（从剪贴板中抓取）、“Capture from the Internet cache”（从Internet Cache中抓取）、“Capture an image in a file”（从文件中抓取）、“Capture from a WebCam”（从WebCam中抓取）等。
　　如果在该图标上单击右键后鼠标指向“Setting”，弹出用来对Flash 32进行有关设置的菜单。主要有“Captures”（与抓取操作的有关设置）、“Hotkey”(设置抓取热键)、“Images Format”(设置图像格式)等项，一般情况均可使用默认值。
　　三、抓取操作
　　1.常规抓取
　　Flash 32支持一般抓图软件的所有抓取功能，如抓取全屏幕、活动窗口和活动窗口中的菜单栏、工具栏、客户区等任何区域。可以连同光标一起抓取和抓取下拉菜单。如要抓取全屏幕可简单地按抓取热键“Print Screen”，也可以选取左键菜单的“Capture/Full Screen”项。如要抓取活动窗口，单击“Capture”菜单下的“Select a windows with a mouse”，然后在要抓取的活动窗口的标题栏上单击，如要抓取其它区域只须在该区域上单击即可。
　　2.抓取任意区域
　　使用Flash 32可以抓取屏幕上的任何区域。当抓取矩形区域时，还可以自行定制抓取区域的大小，这为制作有固定尺寸的图标和Windows开关机画面时提供了方便。从左键菜单的“Capture”菜单上单击“Region”，弹出用于选定抓取区域的对话框，包括：“Rectangle”(矩形)、“Round rectangle”(圆角矩形)、“Ellipse”(椭圆)、“Polygon”(任意多边形)和“Hand made”(任意形状)。
　　其中矩形、圆角矩形、椭圆的抓取相同，选中单选钮后，单击“OK”，在要抓取的区域中按住鼠标左键不放，从左上向右下拖出一个区域，松开鼠标后所抓区域即进入到Flash 32窗口。在抓取矩形窗口时，如选中“Pixed with size”单选钮，可以在其右边的框中指定抓取区域的大小，此时鼠标带有一个指定尺寸的矩形，抓取时，将鼠标指针移至要抓取的位置，再按组合键“Alt＋Z”。如选中“Polygon”单选钮可以抓取任意多边形，此时，用鼠标在要抓取的多边形节点处单击可选定抓取的多边形，结束抓取时双击左键。
　　3.抓取MS－DOS屏幕
　　Flash可以将抓取的MS－DOS屏幕保存成图像格式的文件或仅将屏幕中的文字以文本文件的格式保存。选择“Capture”菜单上的“MS－DOS full screen capture”，再单击“开始\程序\MSDOS方式”切换到MSDOS方式，在屏幕中显示要抓取的图像或文本，然后按“PrintScreen”键，切换回Windows，所抓图像已进入Flash 32窗口中。
　　此外，从左键“Capture”菜单中选取相关命令，还可以从剪贴板、Internet Cache、WebCams和DLL、EXE等文件中抓取图像或图标。
　　四、制作动画光标
　　使用Flash 32可以将抓取的图像直接制作成按钮、图标、动画光标、动态GIF、Windows开关机画面、屏幕保护、HTML页等，原来要用多个其它软件才能完成的功能，Flash 32全都包括了。这些功能的操作命令大部分在软件主窗口的“Create”菜单下，操作十分简单，只要在完成抓取之后，再从这个菜单下选取相关命令，按照提示操作即可完成整个制作过程。
　　比如我们用抓取的图形制作一个动画光标，可以先抓取一幅大小为34×34的图像，进入到Flash 32的主窗口后，打开“Create”菜单，从“Animated Cursor”下选取一种动画光标的颜色(此时如抓取的图像不是34×34，Flash 32将会自动调整，并给予提示)。然后单击“Add”按钮增加其余的图标，在“Name”框中指定动画光标的名称，在“Title”框中指定标题，再单击“OK”按钮完成制作。
　　五、创建动态图像文件
　　软件支持抓取动态屏幕，可以将抓取的多幅连续图像制作成AVI、MPEG和动态GIF文件等。可以选择抓取全屏幕、指定窗口和屏幕中的某个区域。该项功能的另一个用途是你可以录制操作过程，这为用户制作演示作品提供了方便。下面我们以制作一个动态GIF文件为例，看看其制作方法：
　　鼠标左键单击任务栏图标，在“Create an Animation”菜单下选取“GIF”，然后在弹出对话框的“Animation target folder”栏中输入保存文件的文件夹，在“Animation file name”框中输入要创建的GIF文件名，在“Repeat capture every”框中指定抓取每幅图像的时间间隔，在“Capture mode”下拉列表框中选定抓取方式，此处选择“Select a region of the screen”，之后单击“OK”按钮，此时任务栏上出现一个带红色圆环的图标。这时就可在屏幕中播放要抓取的动画文件，用鼠标在播放的画面上选取一个区域，然后单击任务栏上的图标开始抓取(再单击一次则是停止抓取)。抓出图像后，可以对每一幅图像进行修改，单击“Suppress”按钮，可以取消某幅图像，完成后单击“Create”开始创建。
　　六、其它功能
　　该软件还提供了强大的图像处理功能，如对抓取图像进行灰度转换、调整大小、变形、缩放、旋转、应用3D效果、添加边框和文字等，功能异常丰富，足可满足一般图像处理的要求。此外，还可以利用“Edit”菜单下的命令将抓取的图像输出到剪贴板或从剪贴板中输入图像，也可以将处理好的图像直接E－Mail给他人等等。}
