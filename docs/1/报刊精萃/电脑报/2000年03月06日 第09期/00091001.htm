《 千年之礼——2000版软件撷英 》

　　2000年到了，这本来没有什么特别激动人心的，从根本上来说，它不过是新的一年的开始，但人类这种动物就是比较喜欢回顾历史，然后再展望未来。回顾历史一般就是10年一小结、百年一大结，一千年，当然就是一个不得了的大大结。 　　软件行业，在这样一个历史关头当然也不能免俗，很多著名的软件都轰轰烈烈地推出了所谓的2000版。这些著名软件的2000版，意味着将过去的所有版本的辉煌都包含在其中，但更多的是通过软件中包含的新技术和新观念寄托了对未来的无限希望。这里，我们就给大家介绍一些国内国外2000软件精品，并提出使用建议。同时我们也给出了性能上可以和它们匹敌的类似软件。

操作系统

　　Windows 2000

　　软件类别：商业软件
　　软件大小：随版本而易
　　相关网址：http:∥wwwmicrosoftcom/china/windows2000/
　　当历史跨入一个新的千年的时候，Windows 2000扮演了一个恰如其分的角色：它将带着软件行业进入一个新的历史时期——纯32位软件时代。Windows 2000是继DOS、Windows30、Windows95后的又一个新时代。
　　Windows 2000的英文版已经在2月17日静悄悄的发布了，中文版则要等到3月20日。如果仅仅是从个人使用的角度来看，软件在Windows 2000中运行得更流畅了，而且系统很少有崩溃的现象发生。但Windows 2000的升级成本很高，因为除去Windows 2000的本身的费用，为了能真正地发挥Windows 2000的效能，你必须改善你电脑的配置：主要是CPU（推荐PⅡ400以上）、内存（推荐96M以上）、硬盘（需要大而且快的硬盘）这三样的配置。如果这三样配置比较弱的话，Windows 2000就将会使你的系统慢得无法忍受。但无庸置言，从2000年开始，Windows 2000将会在我们的电脑系统中流行十年。
　　推荐：如果你是一个电脑发烧友，你不妨一试Windows 2000，但对于我们大多数用户来讲，还是等一等比较好，因为Windows 2000太新了，其他的软件和硬件都没有作好准备，而且还要等待微软解决Windows 2000最初出现的一些问题。

　　WPS 2000中文集成办公系统继承了WPS 97操作简便、功能齐全、实用方便等优点。与WPS 97相比，WPS 2000增添了许多实用功能和最新技术，是一套各方面性能都较突出的集成办公软件。软件增强了编辑排版、文字修饰、表格和图象处理功能，兼容更多的文件格式，支持不同的汉字内码，可以编辑处理文字、表格、多媒体、图形图像等多种对象，是一套具有字处理、多媒体演示、电子邮件发送、公式编辑、对象框处理、表格应用、样式管理、语音控制等诸多功能的大型集成办公系统。

　　CCED 2000

　　软件类别：共享软件
　　软件大小：138MB

　　这是一款从DOS年代大家就已经耳熟能详的国产表格软件，和WPS共同撑起了办公软件的一片天，在国内，连当时红透全球的Lotus也不能和它相比。但当微软的Excel中
文版进来以后，CCED就日渐式微。现在，CCED 2000又准备重整雄风。
　　CCED 2000集文本编辑、表格制作、数据处理及数据库功能、图形图像功能和排版打印为一体的综合办公及家庭事务处理软件。CCED 2000全面支持以往的CCED和WPS文件，对Word、Lotus 1－2－3、DBF数据库以及HTML文档格式都提供了足够的支持。操作方式还可以自行定义，用户学习和使用起来都非常方便和快捷。
　　CCED 2000的计算公式可设定为自动运算类型，在自动运算公式所涉及到的数据被修改后，立即更新运算结果。程序还提供了文本框、图像框、图形框等多种内置编辑对象，支持OLE技术，在文档中可插入如图像、声音、影像等各种OLE对象，并且CCED 2000文档本身亦可插入到其他软件所提供的文档中去。软件还提供了包括通用类、财会类、计算类、行文类、演示类以及纸型类等丰富的模板库，使用起来将会十分轻松。
　　推荐：CCED当然是性能价格比很高的软件，功能也比较全面，使用它当然能够满足大多数商业和办公的需要。
　　相关软件：华表、Excel

多媒体

　　Cool Edit 2000是一个功能非常强大的音乐编辑制作工具，它能够完成录音、编辑、合成、效果处理等多种任务，Cool Edit 2000可以将电脑变成你自己的家庭录音棚。使用Cool Edit 2000，我们能够通过CD、卡座、话筒等进行声音录制，或者通过与专业作曲软件Cakewalk Pro Audio的配合来合成音乐，然后再进行降噪、扩音、剪接等编辑处理，还可以添加立体环绕、淡入淡出、3D回响等音效，制成的音频文件可以保存为wav、snd和voc、MP3、rm，或者烧录到CD上。

　　推荐：功能又要强大、使用又要容易的音乐编辑制作工具，除了Cool Edit 2000，就很难找了。

网络工具
