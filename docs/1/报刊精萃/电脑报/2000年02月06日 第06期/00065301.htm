《 星型局域网组网指南 》

　　引言

　　九十年代初SUN公司就提出了著名的“网络就是计算机”的口号，当时对此能深刻理解的人还为数不多，大多数人仅仅把它当作是SUN公司的宣传策略。随着计算机的迅速普及和网络（特点是Internet）的迅猛发展，不但证明了SUN公司的高瞻远瞩，更为这句著名的口号做了最好的演绎。
　　对大多数非专业的计算机用户，网络还只是计算机的一个别名，即使对一些专业人士，网络也只是一些抽象的概念。现在单位甚至家庭都拥有一台以上的电脑，如果能把这些电脑以很少的代价互连起来，既能起到共享资源（文件、打印机、Modem、ISDN等）作用，又能节约经费开支。

　　一、入门篇

　　首先了解一下网络的定义：利用通信设备和线路将地理位置不同的、功能独立的多个计算机系统互联起来，以功能完善的网络软件（即网络通信协议、信息交换方式及网络操作系统等）实现网络中资源共享（Resource Sharing）和信息交换的系统。其实对非专业人士而言，只要知道网络就是用线（双绞线或同轴电缆等）连接数台计算机以达到互相通信和资源共享即可。那么什么是局域网(LAN)呢？按照IEEE的说法，局域网可定义为：“A LAN is a data communication system allowing a number of independent devices to communication directly with each other，within a moderately sized geographics area over a physical communications channel of moderate data rates.”译过来就是：局域网是一个通信系统，它允许数台彼此独立的计算机在适当的区域内，以适当的传输速率直接进行沟通。由上述定义可知，局域网与计算机的种类没有关系，LAN的主要概念是“在适当的区域、以适当的传输速率直接进行沟通”，也就是说LAN上任何两台计算机间应该都是没有界限、可以直接互通信息的。
　　从传输距离上看，LAN一般都在同一栋建筑物，甚至只局限于建筑物的同一层或几个房间。从传输速率上看，最通用的以太网（Ethernet）是10Mb/s，也有100Mb/s、1000Mb/s等。从网络拓扑结构看，最常见的有三种：总线型、星型、环型，本文只介绍星型网络结构：这是现在局域网最流行的一种结构，其中心为一个共享式集线器（Hub)或一个智能交换机（Smart Switch)，呈星状结构将计算机互连起来（如图）。这种结构的最大优点是网络稳定性好，不会因为一台计算机的故障而影响整个网络通信。另外，此种网络结构的可扩充性也很好，要在网络中增加一台电脑，只需用网线将其连入Hub的一个RJ45槽口即可。当然此种结构也有缺点，一旦Hub出现故障，整个网络就会瘫痪。
　　下面我们再介绍组建10BaseT(10BaseT中的“10”表示网络的传输速率为10Mbps,“Base”表示基带传输，“T”表示传输介质为双绞线。)局域网所需的硬件：
　　计算机：至少一台服务器和若干台工作站。
　　网卡：从速度上可分为10M、100M、10/100M自适应三种。从总线分，网卡可分为ISA、PCI两种。ISA网卡属较老的一种，价格虽然便宜，但有容易和其它设备（声卡、显卡、硬盘保护卡）的中断、I/O地址冲突的缺点。PCI网卡不存在这个问题，但价格较贵。对于不太熟悉电脑的用户，推荐使用PCI网卡。
　　网线：对于星型网络结构一般采用非屏蔽双绞线，两边接头用RJ45。
　　集线器（或交换机）：这是星型网络结构中专用的设备。从接口数上可分为8、16、24口等几种，可根据自己的需要选择。从功能上分，Hub可分为主动式、被动式、智能式三种，可根据对网络通信的要求选择。

　　二、实战篇

　　1网卡的安装和配置
　　网卡的安装和配置是整个局域网组网过程的第一步，上面已经提到网卡按总线标准可分为：PCI，ISA两种。如果买到的是PCI网卡，一般在安装和设置中，基本上不会出现什么问题，只需按照安装过程中的提示进行即可。如果买到的是ISA网卡，安装过程中可能会碰到一些硬件设备冲突问题，可用随卡带的驱动盘中的设置/诊断程序（文件名可能有几种：“diag”、“modi”、“setup.exe”）来重新设置中断、I/O。例如现在绝大多数网卡都是以UM9008芯片为主控芯片，在驱动盘中有一个名为mod9008.exe的文件，用来设置网卡的工作方式，可将网卡设置为PnP方式或跳线方式。当把网卡改为跳线方式后，必须用另一个名为diag9008的文件来调整网卡的参数（中断、I/O地址等）。一般网卡出厂时，中断被设为3，I/O地址设为300H。
　　以上介绍的是在DOS下对网卡的诊断和设置，当网卡插入主板，重新启动计算机后，还必须进行Windows的设置。如果启动机器后，Windows报告检测到新的硬件，可按照其提示进行网卡设置。也可以在启动后在“控制面板”的“网络”选项中选择“添加→适配器”，然后选择厂商和网卡型号，按“确定”后在网络窗口中会显示添加的网卡设备，同时还会安装相应的网络协议和网络客户。双击“网络用户”在出现的对话框中可以选择登录到某个Windows NT的域，以及登录选项。关于Windows NT的有关内容，限于篇幅，本文不作专门介绍。
　　最后来看一看在网络对话框中“标识页”的内容，实际上是给你的机器取网上的名字和划分工作组的地方。首先，在“计算机名”中取一个名字，就是你的机器在网上的代号，其他人可以在“网上邻居”中用这个名字访问你的机器；其次，还要决定你的机器在哪个工作组中，在同一局域网中工作组的名字必须相同。在“访问控制”中，如果是NT的用户，选择“用户级访问控制”；如果非NT用户，选择“共享级访问控制”。
　　2.非屏蔽双绞网线制作
　　网卡安装/配置好后，接下来就是制作网线了。首先，用压线钳上的剥线刀在距网线顶部2厘米处绕线割一圈，将绝缘线剥下，露出四对双绞线。四对双绞线中的每条芯线都标有特别的颜色，而且这些颜色有公认的规定（最常用的规格为EIA/TIA 568B）。对调第3、5两根线后，从左至右依次为“白橙”、“橙”、“白绿”、“蓝”、“白蓝”、“绿”、“白棕”、“棕”。然后将网线剪齐并插入到RJ45接头中，注意一定要插到底。接下来就是将RJ45接头插入压线钳中的压接槽中，把压线钳使劲压一下即可。网线的另一头可以依法制作。另外，在购买双绞线时一定要分清是哪类线（见表一）；网线侧RJ45（“公头”，8槽）不要与RJ11（电话线接头，6槽）混淆，RJ45各脚功能见表二，集线器端的RJ45（“母头”）各脚功能正好相反，即传输改为接收，接收改为传输。
　　网线做好后，将网线的一头插入需要入网机器网卡的RJ45中，另一头插入集线器（Hub）中，然后可以试着在各自的“网上邻居”中查找对方的机器名（也就是在“标识页”中所起的计算机名）。如果能看到别人的机器就说明网络通讯成功，局域网组网到此结束，下面将说明局域网如何实现资源共享和Internet上网共享。

　　三、应用篇

　　1文件共享
　　这是网上资源共享最常用的方式，可以通过下列途径实现资源共享：点击桌面图标“我的电脑”选择你要共享的驱动器或者目录，然后在上面点击右键，选择菜单中的“共享”选项，在出现的对话框中输入共享名，可以使用缺省名称，也可以自己随便起一个。访问类型有三种：“只读”、“完全”、“根据口令访 到局域网上。
　　2打印机共享
　　打开“我的电脑”，双击“打印机”，选择“添加打印机”，按“下一步”，选择“网络打印机”，再按“下一步”，在“网络路径或队列名”编辑框中填入相应的内容。比如我单位一台Workstation的电脑上，连有一台HP激光打印机，那么编辑框中就应填入“\\workstation\hp”。然后选择正确的打印驱动程序，安装后就可实现打印机共享。
　　3.Internet资源共享
　　Internet上网共享的含义就是在局域网中拨号上网，或通过ISDN上网，其他局域网中的机器都可以共享该机器的拨号连接。共享上网只需在所有的上网机器中安装TCP/IP协议和在那台拨号上网的机器上安装一个特定的软件，即可实现。这个特定的软件可分为两种，一种是通过Proxy Server（代理服务器）实现共享的WinGate；另一种是通过网关实现共享的SyGate。这两个软件可到下列网址下载：http://www.sygate.com，http://www.wingate.com。关于这两个软件的设置涉及到一些专业知识，一般用户不需了解，本文也就不作介绍。对此感兴趣的读者可到这两个软件公司的网址去查阅，或参阅相关书籍。
　　最后就该配置局域网中其他要共享上网机器的TCP/IP协议。在桌面的“网上邻居”图标上按鼠标右键，再按“属性”按钮，打开网络对话框，双击“TCP/IP→网卡的型号”，出现“TCP/IP属性”对话框。以我单位校园网为例来说明该属性对话框选项的填法，“IP地址”栏填写：202.121.153.4（根据自己申请的IP地址填）：“子网掩码”栏填入255.255.255.0；“网关”栏填入：202.121.153.1；“WINS配置”填入：202.121.154.5、202.121.154.2；“DNS配置”：主机填入Jzhang，域名填入shiep.net.cn；DNS服务器搜索顺序填入202.121.154.5、202.121.154.2。网络共享设置就这些了，现在可以重新启动机器，然后就可以试着拨号上网浏览了。如果是通过Proxy Server上网，还需下载MS Proxy Server 2.0代理服务器软件，可到微软公司Proxy Server站点下载。使用代理服务器有两个好处：一是可利用其大容量的缓存提高上网速度，二是可以起防火墙作用。Proxy Server的设置和管理较为复杂，已经超出了本文的范围。其实，网络是个实践性很强的东西，必须亲自动手组建一次，才会对网络的一些要领有更加深入的理解。另外，组网过程中会遇到很多小问题，有些在本文中已涉及，还有一些需要读者自己解决，相信聪明的读者是不会被这些小问题难住的。
　　（上海　张戟　魏国强　钱敬荣）
