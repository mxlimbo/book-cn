《 三款上网好工具 》

　　1．Go Network Express Search

　　从某种意义上说，上网就是搜索自己感兴趣的资料。通常只用一个搜索引擎是不能满足搜索要求的。但是，从一个搜索网站到另一个搜索网站又是一件艰巨无比的工作，最糟的是大部分搜索工具不够丰富，不能让人满意。
　　Go Network的全新搜索工具Go Network Express Search则可完全满足你的要求，它收录了300多个网站及搜索器，提供了极为详尽的搜索结果。Express Search过人之处是它能够与Internet Explorer配合得天衣无缝，它在IE的工具列上加入一个搜索表格，你只要键入要搜索的资料，一按Go它就会为你在网上进行地毯式搜索。另外，在网页内你只要在搜索的字词上按鼠标右键，再选择Express Search，你就会得到同样效果。通过Express Search，你可以作多类型的搜索，新闻网站、下载网站一应俱全。你当然可以储存搜索结果作日后参考之用，你甚至可以输入密码，储存你自己的私人设定，可谓十分周到。

　　2．LapLink FTP

　　有时，网迷简直想闭上眼也可以完成文件的传送！虽然FTP并不怎样高深，但如果传送的文件很重要，尤其是那些机密文件，为了确保文件传送顺利，我不得不格外留神，免不了会妨碍其他工作，害得我不知花了多少宝贵时间。
　　现在有了LapLink帮手，事情就简单得多。它提供了一个类似Windows文件管理器的操作界面，简单易用，只要用鼠标拖动文件，就可以轻轻松松把文件及文件夹从FTP服务器搬到自己的硬盘。我可以一边看电视一边传送文件。不过，最方便的还是它可以帮我加入新FTP网址的Wizard小精灵功能，只要在工具列表选择Connect，再按Help Me Add a New Site即可。你还可以同时开启四个窗口，Internet窗口显示出FTP网址；Contents窗口显示指定的FTP的内容；My Computer窗口就显示硬盘的内容；Contents窗口则可以显示指定文件夹的内容，形式基本与Windows文件管理器大同小异。
　　本人在此极力推荐LapLink FTP是因它实在是一套实用的软件，既能满足网上老手，也适合刚刚才开始用FTP的新手。

　　3．1Jump

　　1Jump是一个完全免费的网上浏览伙伴，也是一个可以帮你找到所需的商业及市场资料的好帮手，只要键入一个公司名称，就可以马上得到详尽的资料。换言之，想知道市场情况时，就不需要键入一连串繁复的关键字。1Jump视窗会出现在浏览器窗口之上，你可以用公司名称、公司简称，甚至是地区编号搜索，1Jump的资料库储存了超过百万家主要公司的名称，毫无疑问，你会找到各公司的官方网址、主要产品、专利资料、雇员情况，总之可轻松找到一切与该公司有关的资料。1Jump也有很完备的资料搜集工具及投资组合管理，足以同时处理多项本地及海外的投资。
