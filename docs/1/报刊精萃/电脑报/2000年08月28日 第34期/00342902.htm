《 {极光旋貂Wheel Mouse} 》

{　　极光旋貂是罗技电子采用独步业界的光学技术, 取消传统机械滚球全新设计的新型鼠标, 它使光标定位更精准可靠。采用创新的 “网际风火轮”, 使浏览网页更加便捷快速，全中文化的软件接口, 让鼠标按键设定更容易。
　　这款新产品有着与众不同的特色，光学定位, 精准可靠。流行炫丽、深蓝色圆弧造型使鼠标更酷。同时针对国内用户使用了全中文化驱动程序接口。
　　系统需求:IBM PC平台, Windows 95, Windows 98/Windows 2000, 或 Windows NT 4.0 (滚轮功能) 可使用USB端口或鼠标端口(PS／2 port),光驱(CD－ROM)。 }
