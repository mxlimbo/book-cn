《 {USB技术在品牌电脑中的应用} 》

{　　一、 USB走向品牌电脑
　　USB（Universal Serial Bus）通用串行总线，早在1997年就已经在主版中出现了，Intel 440 LX芯片以及更早的TX、HX主板就提供了这样的接口，直至去年才开始普及，据Dataquest预测，到2001年，百分之百的笔记本电脑和个人电脑都将是通用串行总线（USB）兼容产品。到2003年，与USB兼容的个人电脑的用户将要购买近5亿件USB电脑外设。
　　目前的市面上各种各样的USB设备比比皆是，如USB的键盘、鼠标以及音箱等等。USB成了即插即用设备的代名词，但是不是说只要你买的设备是USB接口的，一切连接、应用都会像宣称的那样简单、易用呢？
　　二、USB设备真的是省心的设备吗
　　计算机产业在我国的发展历史是从“攒机”开始，这样，很多人就形成这样一种观念，各种计算机硬件的接口都是标准的，将它们配搭在一起就可以工作。而USB技术作为一种新的接口标准，目的是使计算机与外部设备的连接变得更加规范、更加快速和更加容易操作，这就给人带来一种错觉，使用USB设备，是一件省心的事。但实际上并不如此。
　　软件方面：首先你的操作系统如果是老的Windows 95，恐怕你是用不成USB设备。然后，看你的BIOS对USB支持是否到位，不到位则你在DOS下无法使用USB设备，如果在DOS下不能使用键盘，那你什么也做不了。
　　硬件方面：从系统结构上，标准的主板只提供两个USB口，你只能连接两个USB设备。为了连接更多的USB设备，就必须使用USB HUB（集线器），这样在用户的桌面上又多了一个设备和一大摊子线。另外，某些USB设备可能会对系统供电带来影响。
　　也正是因为这些原因，希望升级的用户，最多只能零零散散地配备少量的USB设备，而不能随心所欲地全部使用。不少的整机厂商也只能宣称其产品配备USB接口，而不敢全面配备USB设备。因而以即插即用为代表的USB技术的优势始终没有发挥出来。
　　三、配备USB设备——增强电脑的易用性
　　现在，绝大多数的品牌电脑都配备有USB接口，一般普通用户购买了USB设备后，可以非常简单地安装使用。还有一些厂商的品牌电脑，配备的外部设备甚至都是USB接口，这样，可以最大可能地减少用户在安装、升级、使用上遇到的麻烦，增强电脑的易用性。这方面的代表产品就是联想的“天禧”家用电脑。下面，我们就以联想天禧电脑为例，来看看USB设备是如何在品牌电脑中应用的，又给用户的使用带来那些方便。
　　统一的USB设备插口
　　以前，为了方便用户安装和使用电脑的各种外部设备，品牌电脑的各种接插头和线缆常常使用用特殊的颜色来标志。在“天禧”电脑中，USB成为所有外设统一的接口方式，用户不需要判明设备需要接到哪个插接口，而是随便哪个USB接口都可以，还可以什么时候用就什么时候接。
　　集成的USB HUB
　　USB HUB的作用就就点像电源插板，可以提供多个USB接插口来连接各种不同的USB设备。USB HUB的特点决定它一定要摆在电脑桌上面，这样才可以方便地插接各种USB设备，但这又给用户增加了麻烦。现在，一些显示器厂家将USB HUB集成到显示器内，而一些PC整机厂家选择了将它内置于主机内的方案。联想天禧电脑则是将USB HUB与显示器底座半固定地集成在一起，这种方案可以解决HUB零散摆放的问题，同时也解决HUB供电的问题。
　　联想天禧的USB HUB采用与显示器底座半固定方式是比较独特的，避免将USB HUB集成在显示器或者主机中的种种不足，USB HUB与显示器的电路相互独立，不改变显示器的电路设计。而天禧这种分体式USB HUB既有专门设计的供电，又非常利于升级，而且采用转动式设计，用户可在对HUB插拔操作后将HUB移至显示器后面。这样就最大限度地发挥了插接时的便利性，又降低了整机一体化方案中各种USB连线被牵动的可能性。
四、品牌电脑中的USB技术难关
　　将USB技术运用到一个完整的系统中，并使用户能应用自如，不出问题并不是一件容易的事。比如Windows98操作系统对USB的支持并不是无误的，实际上只有Windows98SE（第二版本）才能做到这一点。举例来说，当你使用全部USB的键盘和鼠标的时候，你将键盘插到主机上时，系统会像往常一样向你报告发现一个新部件，并要求你选择如何处理，是“取消”还是“下一步”。此时的USB键盘、鼠标尚未装到系统上，因此用户是无法操作的。
　　主板BIOS也是这样，很多主板BIOS都宣称对USB的支持，但实际上却不完全，比如在DOS支持不够，不支持即插即用等。现在我们看看联想天禧电脑是如何做的。
　　电路设计
　　首先要按照系统功能的定义来确定整个HUB电路及相关器件类型，虽然HUB电路是有标准电路的，但由于提供非常全面的功能，性能可能大打折扣。品牌电脑为了追求实用性和品质，因此在电路设计上去掉不必要的功能，而重点保证品质和实用性，始自保护动作电流，休眠后的功率损耗，满负荷的电压衰落等。
　　电磁干扰
　　USB HUB另外一个技术问题是EMI，这是一项衡量电器设备在使用中对外界电磁辐射的技术指标，目前国家在这方面已有详细的规定，以保证设备的使用不会对用户其它电气设备产生干扰。而USB HUB在这方面的技术实现上确属难解决的，因为HUB电路中时钟震荡电路的工作频率高达48MHz，频率越高，电磁辐射就越强。
　　可靠性设计
　　在工程上，为了将用户使用USB设备遇到问题的可能性降到最低，围绕HUB要进行多种可靠性设计，包括各种硬件设备的插拔试验，各种应用环境的插拔试验，各种应用软件的兼容性测试等。}
