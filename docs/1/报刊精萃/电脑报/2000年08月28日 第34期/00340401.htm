{　　高雷：“我是《电脑报》的一名很普通的用户，我们这（黑龙江省牡丹江市）每周收到报纸的时间都要等到周五，最快也要周四，我想能不能在哈尔滨建一个分印点，以满足黑龙江及东北部分地区的广大读者，谢谢你们啦！”阚维中：“非常遗憾到现在我还没参加过俱乐部组织的任何活动，《电脑报》每周一发行而在我们这里却要在周四或周五才能收到……上次大庆联谊会的通知是6月26日发出，我却是在7月2日才知道，而联谊会已经召开，唯一能参加活动的机会却在眼前失去了……”
经过多方努力，南木高兴地报告众读者朋友一个特别消息：从2001年开始，《电脑报》将在北京、南京、长沙设3个分印点，再加上邮政提速，全国各地的的读者朋友们基本上能在星期一就收到当周《电脑报》啦！另外，《电脑报》一直可以破月破季订，很方便哦！
汪滨：“我是一位电脑报忠实的读者,每逢邮政局送来报纸,我都要认认真真的阅读,不错过任何精彩的文章……在阅报过程中,我注意到可以加入《电脑报》读者俱乐部，不但可以和其他读者进行相互交流,而且可得到电脑报社的种种优惠购物，并可参加其组织的各种活动。但是,我不知道怎么样才能成为《电脑报》读者俱乐部会员,望请南木给予指导。”yltw：“我是贵报读者，我想知道如何才能成为会员”qijiongning：“我是浙江宁波市的读者，我想问一下如果我邮购的话，从我寄出汇款单到收到物品要多长时间？盼回复，谢谢。”
