《 {解开千禧之谜——初览Windows Me新功能} 》

{　　Windows Millennium Edition（以下简称Windows Me）是微软最新推出的面向家庭用户的PC操作系统，也是我们非常熟悉的Windows 9x的升级版本。微软给这个操作系统确定的目标是：更强的稳定性、更简便的家庭网络功能、更好的多媒体工具、更简单的上网工具。从这个定位出发，Windows Me的安装界面中可以看到，这个系统应该是一个简单而健壮的系统。因为面对的家庭用户并不需要很高的运算能力，而从实际的使用中我们也可以看到这一点。不管孰是孰非，现在让我们看看Win Me系统为我们提供的新功能。
一、焕然一新的操作界面
　　Windows Me与Win98不同，它大量借鉴了Windows 2000的界面风格，比如漂亮的界面显示字体、透明的图标、使用鼠标拖曳图标时图标样式、鼠标菜单的淡入淡出效果等。它也提供了个性化菜单，可以将访问频率低的菜单项隐藏起来，以便于快速访问常用的应用程序。对于开始菜单和任务条，系统提供了新的“任务栏和开始菜单”设置项目（如图1所示）。在其中可设置开始菜单中的列表项目，以创建用户的个性化操作菜单。
　　同时，系统也提供了全新的“文件夹选项”设置，它同样借鉴了Win 2000的模式，主要的功能增强在“查看”选项中，增加了“隐藏受保护的系统文件”、“自动搜索文件夹和打印机”选项。这些都是从家庭用户的角度考虑，避免了可能发生的误删系统文件的情况。如当进入驱动器文件列表窗口中可以使用“隐藏该驱动器的内容”命令，这样你就不会看到该驱动器所有的文件列表。
二、软/硬件兼容性
　　软件兼容性：
　　在应用方面，Windows Me同样可以运行以前Windows 9x系统下的多数应用程序，不过由于新的系统取消了DOS实模式运行方式，虽然对于一些老的纯DOS程序不再兼容，但是也让系统更稳定可靠，减少了死机的可能性。一些基于此种方式运行的应用程序，如Partition Magic、Norton Ghost等将不能在此环境下使用。但可通过Windows Me创建的启动盘启动到DOS实模式环境下使用。此外，一些与系统底层结合非常紧密的应用程序也将不能顺利运行，比如著名的Norton Utilities 2000等。否则，系统会给出提示信息，不推荐继续运行。
　　硬件兼容性：
　　因为Windows Me推出得更迟，所以在硬件上支持的设备也更加多。从Windows Me的硬件设备选项中便可略见端倪。虽说这增加了系统的体积，但也总比我们翻箱倒柜找驱动程序，或者用可怜的Modem从网上下载来得强。
　　此外，Windows Me还提供了对FireWire的支持。FireWire最主要的特点是：适应性强，采用FireWire 技术，可将几乎所有的数字式设备包括打印机、DVD机、数字摄像机，甚至微波炉等等连接成一个真正的家庭网络。这构成了一个全新意义的数字式系统，以后我们的家庭肯定是“一切尽在网络中”了。在这一点上充分表明了Windows Me是微软面向家庭消费者用户设计的目的。
三、丰富的在线帮助与支持
　　Windows Me中新提供的一项非常强大的功能就是详细的在线帮助（如图2所示）。在新系统中，所有的帮助都被集成到了“帮助与支持”中。这是微软第一次采用HTML格式作为系统操作的顾问以及疑难解答中心。其提供的说明非常详细，而且提供了运行所关联的系统工具的快捷方式，使你可以快速调用相应的程序，而不需要从开始菜单中选择。
四、高效、安全的系统四宝
　　月光宝盒——“系统还原”
　　“系统还原”可以说是Windows Me面向广大用户提供的一项最实用的Windows维护工具。其主要的特点是：不需要你对系统维护有多深的了解，而且也不需要你掌握什么知识和会使用什么工具软件，只需要简单的几个步骤就可以将发生冲突甚至在崩溃边缘的系统挽救回来(见图3)。通过这个工具你可以取消所有针对Windows所做的设置更改，并恢复系统到还原点之前的设置环境。简直同《大话西游》里周星驰所拿的“月光宝盒”相差无几。有了它，你可以让系统恢复到以前保留过的任何状态。同时它还可以在Windows Me的安全模式下执行，这样又增加了成功恢复系统的机会。
　　快速引导——“神行太保”
　　Windows Me加快启动的一个法宝就是Fast Boot（快速引导）。要支持这项功能，需要新的硬件支持，老式的机器都不行。在Dell的新款Dimension个人电脑中，预装Windows Me据说启动速度不到30秒。而相比之下Win 2000的速度要慢许多，尤其是系统提供的“保存设置”，有时会让你有过了一个世纪的感觉。
　　Win Me的“宙斯盾”
　　Windows Me另一个值得称道的是系统提供的安全性。这里的安全是系统针对系统自身运行的重要文件的保护。新的系统采用了名为“PC Health”的在线监测维护功能。这项功能的重要组成部分之一就是采用了类似Win 2000提供的系统文件保护功能（SFP）。它取代了以前版本的系统文件检查器。这项功能会在系统启动时自动运行，它可以实现系统运行所必需的文件禁止被覆盖和删除。不过，这个工具并不保护你的注册表文件和ini配置文件。所以如果你把system.ini给删除了，那就只有通过其它途径来恢复了。
　　自动更新——百变金刚
　　AutoUpdate（自动更新）功能可以让你在上网时，在系统内悄悄地执行更新：检查重要的档案更新版本，并帮你下载与安装，让你的麻烦减到最低。当然，如果你不想让盖茨常来“打扰”你的电脑，也可以把自动更新程序关闭。
　　注意：在使用“系统还原”功能后，Windows Me会在硬盘根目录下创建一个具有隐含属性的文件夹“Restore”，并将所有的备份文件保存到其中。当Windows Me运行时间较长，且用户在系统中安装并删除了大量的应用程序后，这个文件夹会变得非常的庞大，而且它有时会将一些无用的文件自动保存到其中。某次使用的光盘文件中发现有病毒，当终止操作后使用扫描程序扫描硬盘时，竟然发现“Restore”文件夹中已经有了光盘染毒的备份，而且是使用了不可识别文件类型。幸亏杀毒软件功能强大，否则后果不堪设想。
五、家庭网络向导
　　互联网时代的到来对操作系统提出了更高的要求，这也是Windows Me的重头戏。随着计算机硬件的不断更新，家中拥有多台计算机将会很常见。但对于外部设备，通常只有一套，那么如何实现资源共享呢，最好的方式就是组建家庭网络，如Windows对等网，这样可以共享打印机、驱动器等硬件设备，而且可以实现联机操作，可以共享Internet连接。
　　在Windows Me中，系统提供了一个简单的操作向导，你只需要选择相应的硬件设备，就可以轻松创建网络共享，在这方面Windows Me做得甚至比Win 2000的智能化程度还要高。因为我们知道，对于一个初级用户来讲，网络设置所带来的难度可能是所有普通用户的Windows操作中最高的。现在让我们看看这项功能的具体用法：启动“家庭网络向导"，首先是一个欢迎界面，跳过之后是“安装选项"，程序会询问你需要编辑当前系统中的家庭网络设置还是创建在其他计算机中使用的家庭网络安装软盘。对于使用Internet连接共享，需要在与Internet连接的计算机上设置使用本机上的Internet，并选择当前可用的连接方式。之后，设置家庭网络中的其他计算机可以使用的共享Internet连接网络连接设备（一般为网卡，如图4所示），然后就是设置计算机和工作组名称、设置文件夹和打印机共享、设置访问授权口令等。之后，程序会询问你是否创建家庭网络安装盘，由于创建的家庭网络安装盘可以用于家庭网络上，实现与其他计算机的Internet连接共享。当在关联的界面中单击“完成"按钮可结束家庭网络向导的创建，程序会安装相应的支持文件，并会重新引导系统启动。之后，你就可以享受家庭网络带给你的共享快乐了。对于家庭网络上的其他计算机，可从家庭网络启动盘中直接运行“setup.exe"进行安装和自动设置。
　　当家庭网络创建完成后，在Windows的任务条中会多出一个Tray图标。使用鼠标右键单击该图标，在关联的菜单中提供了显示Internet共享连接状态、设置连接共享方式、禁用连接共享、隐藏任务栏图标等项目，根据自己的需要直接选择即可。如果你选择“选项"命令，系统会给出如图5所示设置界面，在其中提供了是否启用Internet连接共享、选择用来访问连接共享的方式、选择用来访问家庭网络的网络适配器等项目，直接选择即可。
　　最后说明一点，在Windows Me中，仍然需要用户自己设置IP地址、子网掩码之类的东西，要是能变成自动识别就更好了。
　　六、.COM时代到来
　　与我们预想的一样，在Windows Me中依然捆绑了IE 5.5。它增加了IE的打印预览功能（如图6所示），效果相当不错。在系统中提供了另一个网络工具是MSN Messager Service，这是一个类似ICQ的工具，而且还可以定时检测Hotmail.com用户邮箱。不过程序提供的功能远没有ICQ强大，再加上对于国内用户而言MSN的访问速度实在不敢恭维，预计国内的使用者不会很多。
　　Windows Me在数字多媒体领域的功能也是相当强劲的，详细介绍请看本期第23版。(河北　张熙平)}
