《 {站长周记} 》

{　　最近，一方面市场是越来越冷淡了，另一方便新产品刷新率却是越来越高了。本来这一个星期站长还比较平静，可是当站长看到又有假货出现在市场上的时候就愤怒了。
　　这里先为大家报道一下关于假货的消息：
　　最近市面上一些如“Wescom”五彩套件专用电源、“亮丽2000”的“长城”电源，其实这两款电源并不是长城公司产品，而是某些人为谋取暴利而把普通电源改名换姓后来冒充长城电源进行销售，每台长城PC电源上均有长城电源注册的专用商标“HOPELY”。
　　G400也出现了水货，真货与水货的最大区别为，真货均为中文包装，并贴有“MGA 中国大陆专销”标志。另外上面还标有唯一的一组编号，这是与包装盒内“产品质量保证书”和“产品保修证书”后面贴的标签完全一致的，包装盒内还有一本《高级用户使用手册》。另一个区别是，正品包装盒左上角用粗体标出了显存的大小“16M”或“32M”字样，颜色为金色，而水货包装盒上则没有。
　　大白鲨光驱也有许多假货，从外观上来看有以下主要几个特征：从包装上来看真货假货很难分别出来，而从光驱的形状上就可以分辨出来了，假光驱面板和真品有很多不同点：1假大白鲨的面板是非常平直的。而真品大白鲨光驱在光盘挡板下面都有一道弧线。2假货的紧急弹出孔是在音量调节钮的右上方；真大白鲨2代光驱的孔位在音量钮的左上方；而大白鲨4代的孔位在面板的右部，比较好分辨。
　　上面三种情况大家要仔细看清楚了。
　　说说市场吧，硬盘还是缺货缺得厉害，价格也开始上涨了，站长观察发现这次涨价有这样两个特点，一是5400转硬盘的价格涨得最多，这可能跟市场上5400转硬盘价格比较低有关系，相比之下，7200转硬盘的价格变化较小。二是20G主流硬盘涨价幅度要比其他容量的硬盘要少一些，估计和20G主流硬盘的存货量较大有关系。不过，大家也不要慌，雨过总会天晴的，相信不多久价格又会回落的，因为各个厂商的新型号硬盘已经发布了，比如西捷已经发布了酷鱼Ⅲ硬盘，这可是支持ATA/100的哟。
　　此外，据说有的主板厂商已经指出了VIA KT133 Athlon芯片组在安装了三条 PC133内存后便会有不稳定的情况发生。其实站长觉得这也没有什么，但是有这样一个小BUG总觉得不大好。
　　CPU
　　CPU的价格天天在跌，跌得我们大家心花怒放。钻龙600现在的价格只要530元了，真便宜呀，真不知道它的底价是多少。钻龙650只要670元，钻龙700只要890元了。钻龙系列这一周价格普遍下降了30元左右，估计不久后会有更高频率的钻龙上市，现在钻龙CPU降价可能是为了新品做铺垫，也有可能现在这些钻龙的价格还会进一步下跌。但是AMD的高端产品雷鸟的销售形势不容乐观，现在雷鸟700的价格为1140元。需要注意的是最近AMD将钻龙的倍频彻底锁掉了，所以大家想买可超频钻龙的时候要注意这个问题，这里站长将未锁倍频和已锁倍频钻龙的图片放出来，大家仔细看它们针脚的区别。
　　Intel自然不会看着AMD将市场份额越做越大，赛扬、PⅢ这些统统都一直在降价。现在赛扬Ⅱ600/566为760/690元，赛扬500/466/433的价格为690/620/610元，但是赛扬Ⅱ533还要810元，这可能和它拥有良好的超频性能有关。PⅢ733/667/600EB为1620/1530/1420元。不过由于100MHz外频的PⅢ有着良好的超频性，所以现在有的商家将这类CPU超到133MHz外频当成133MHz的PⅢ来卖，大家看仔细了。
　　内存和硬盘
　　CyrixⅢ 550也上市了，不过性能不大好，其价格也要550元，所以卖得不好了。HY 64M的内存只要520元了，HY 128M也只要1030元了。KingMax的价格几乎没有什么变化，64M还是要590元左右。内存的价格稍微向下跌了一点，要知道这可是内存价格长期稳定以后的下降呀，不知道是不是有什么预兆？
　　硬盘的价格则是一路上扬，其中10G以下的硬盘很难找到了。涨价的原因是硬盘缺货，这也是国际市场的行情。IBM、昆腾和希捷硬盘都有10～30元的涨幅。希捷的10.2G现在为780元。20.5G也涨到了860元，酷鱼Ⅱ20.5G的价格从1050元涨到了1100元。大家看了这个价格是不是觉得好吓人，要知道前几天5400转20G的硬盘也只要700多元呀。
　　显卡和其他
　　好长一段时间显卡市场只有显卡品牌之争，这是因为市场上好像只有NVIDIA的显卡卖，大家只是打着不同的招牌而已。现在3dfx也向显卡厂商提供芯片了，这样一来大家就可以有更多的选择了。
　　市场上现在的热点是GeForce2 MX，各个厂家纷纷推出了这款显卡，价格也直线下落，站长觉得GeForce2 MX是一款非常成功的显卡。1000元左右的价格绝对超值，其中耕GeForce2 MX Golden版已经换用了Elite MT 55ns显存，比以前使用的Winbond 55ns显存超频能力更胜一筹。
　　现在也有几款Voodoo系列显卡上市，游戏发烧友们期待的显卡“极品”Voodoo5 5500价格高达3000多元，而2000来元的Voodoo3 3500TV也很有特色，主要是可以进行视频捕捉和回放。而重新登台的Voodoo3 2000价格降到了780元。3dfx的产品在兼容方面做得不错，使用老主板的朋友在升级显卡时可以考虑。
　　其他方面好像只有DVD－ROM是一个热点，因为它一直在跌价，哈哈！先锋16× DVD终于有货了，型号上DVD－105SZ吸盘式，价格要1260元，托盘式的ACER 16× DVD也是这个价格。NEC的12×价格是999元，市场还到了一批松下6× DVD要850元。在这里站长还是要提醒一下——注意全区解码的功能。 }
