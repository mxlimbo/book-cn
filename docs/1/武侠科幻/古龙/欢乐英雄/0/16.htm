第六章（三）

　　郭大路也已从桌于底下窜出。

　　燕七松手点心掉下来叁四个，郭大路伸手接了两叁个，同时张大了嘴，一个软软的儒米烧卖正好不偏不伤掉在他嘴里。

　　这下虽然并不是什麽了不起的武功但却配合得又紧凑，又巧妙、简直令人叹为观止。

　　卫夫人居然也叹息了声，说道"看了你们这两手功夫，我就算让你们晚点东西也算值得了。"

　　郭大路叁口两口就将烧荚吞下去笑道"这人倒总算还有点民心。"

　　他开始吃第二个烧卖的时候，燕七也已吞下了个包子。

　　能吃得这包子可真不容易所以嚼在嘴里的滋昧也象是特别好，燕七笑道"这包子真好吃却不知是用什麽做馅的？"

　　卫夫人微笑道"包子和烧卖都有两种肉。"

　　郭大路道"那两种？"

　　卫夫人道："一种是虾仁鲜肉的。"郭大路道"还有种什麽肉："

　　卫夫人道"老鼠肉毒老鼠。"

　　老鼠本来是可以吃的但毒老鼠吃下去却能要人的命。

　　郭大路吃下去的烧卖，好像已停在嗓子眼上·再也不下去。

　　他本来还想问问·他吃的烧卖是哪种馅做的，现在却已用不问他忽然觉得四肢发软脑袋发绿。

　　再看燕七张脸竟已变成死灰色而且渐激发黑。

　　卫夫人还在微笑。

　　郭大路正想过去·忽然觉得她象已是到了很远很远的地方，一张脸渐渐变得模糊不清·渐渐什么也看不见了。

　　他只觉得燕七已冲过来抱住了他，道"临死之前我有个秘密要告诉你。"郭大路道"什……么秘密？"

　　燕七道"我──"

　　他还没有说出自己的秘密·就已倒下。

　　就算他说出，郭大路也听不见了。

　　人为财死鸟为食亡。

　　这句话并不太对。

　　有的人并不太在乎财宝绝不会为了钱拼命，却往往会为了好吃而死。

　　你是不是觉得这种死法很冤枉？

　　等你俄得发晕时·说不定也会觉得不如死了算了。

　　但他们为什麽会挨饿呢？

　　朋友当然是为了朋友。

　　"为朋友而死的人，是绝不会下地狱的。"

　　但朋友若都在地狱里他们也许宁可下地狱，也不愿上天堂。

　　自占艰难喉死。

　　死的确可以算是最可伯的事了。

　　那意思就是你已完了，已完全消减了从此不再有希望你的肉体很快就会腐烂你的姓名也很快就会被人淡忘。

　　世上还有什麽比死更可怕的呢？

　　死了若还得下地狱那当然更可怕。

　　但地狱究竟是什麽样子谁也不知道。

　　那地方想必很黑暗，非常黑暗……

　　黑暗。

　　黑暗得让你非但看不见别人·也看不见自己。

　　郭大路连自己都看不见。

　　他只感觉到自己的眼睛已睁开了。

　　但自己究竟在什麽地方？究竟是不是存在？他却完全本知道。

　　"不知道"的本身就是种恐惧也许就是人类最大的恐惧。

　　人们恐惧死亡·岂非也正因为他根本不知道死了究竟是什麽样子的。

　　郭大路也不能不惧，几乎已恐惧得连动都不能够动恐惧本教是人类永远无法克服的

　　过了很久郭大路才听到自己身旁仿佛有个人在呼吸。

　　但那究竟是不是人的呼吸声他还是不知道。

　　在如此黑暗中任何人都已无法再对自已有信心。

　　幸好他还能相信一件事，燕七活时既然跟他在一起就算死了也一定还是会跟他一起。

　　有些朋友好像永远都分不开的无论死活都分不开。

　　所以郭大路壮起胆子道"燕七…是不是你？"

　　又过半晌黑暗中响起个很虚弱的声响"是小郭吗？"

　　郭大路总算松了口气。

　　只要有朋友跟他在起无论死活都没关系了。

　　他身子开始往那边移动终于摸到了只手，一只冰冷的手。

　　郭大路道"这是个是你的手？"

　　手动了动立刻将郭大路的手握紧。

　　然後听到燕七虚弱的声音道"这是什麽地方？"

　　郭大路道"不知道。"

　　燕七道"我们是不是还活？"

　　郭人路叹了门气道"不知道。"

　　燕七也叹了口气道"看来你活时是个糊徐人死了也是个糊涂鬼。"

　　郭大路却笑了笑道"看来你活时要臭我死了也要臭我。"

　　燕七没有说话却将郭大路的手握得更紧。

　　他平时本是个很坚强的人但现在却象是要倚赖郭大路了。

　　也许他本就在倚赖郭大路了、只不过平时直在尽力控制自己个人只有到了真正恐惧的时候才会将自己真正的情感流露出来。

　　郭大路沉默了半晌忽又问道"你猜我现在最想知道什麽？"

　　燕七道"想知道这里是什麽地方？"

　　郭大路道"不对。"燕七道"想知道我们究竟是不是还活？"

　　郭大路道"也不对。"

　　燕七叹道"我现在没有心情猜你的心事你自己说出来吧。"

　　郭大路说"我只想知道你的秘密。"

　　燕七道"我？……我有什麽秘密？"郭大路道"你临死的要告诉我的那件秘密。"

　　燕七的手忽然缩了回去沉默了很久才带笑道"到现在你还没有忘记？"

　　郭大路笑道"无论死活都不会忘记。"

　　燕七又沉默了很久·才缓缓道"可是现在我已不想把那件事告诉你了。"

　　郭大路道"为什麽？"

　　燕七道"也没有为什麽只不过……只不过……"

　　他这句话还没有说完·前面那无边无际的黑暗忽然亮赵了点阴森森、碧鳞鳞的火光。

　　鬼火惨碧色的火光下仿佛有个人影。

　　也许不是人影是鬼影。

　　他看来飘飘荡荡的站在那里·好像上不天下不地。

　　郭大路忍不位问道"你是人？还是鬼？"

　　没有回答，这也不知是人还是鬼的影子·忽然又向前飘了过去。

　　无论他是人也好是鬼也好总是这无边黑暗中唯一的点亮

　　只要有一点光就比黑暗好。

　　郭大路沉声道"你还能不能走？"燕七道"能。"

　　郭大路道"我们追过去好不好？"

　　燕七叹道"无论如何·我想总不会比现在这情况更坏的了。"

　　鬼火在前面飘荡好像故意在等他们。

　　郭大路已找了燕七的手再握紧道"你拉我千万莫要放松无论好歹我们都要在起。"

　　他们的力气还没有恢复身子还有点麻烦。

　　但无论如何他们总算已站了起来·跟那点鬼火往前走。

　　前面是什麽？

　　是天堂？还是地狱？

　　他们既不知道也不在乎因为他们总算还能手拉手往前走。

　　等他们渐渐可以走得快点的时候，前面那鬼火速度也加快了。

　　鬼火突然如流星般闪，忽然消失。

　　四面又变得完全黑暗。

　　没有光没有声音。

　　他们只能听得到自己心跳的声音心跳得很快。

　　两个人都已感觉到自己心跳的声音·心跳得快。

　　两个人都已感觉出对方的手心里在冒冷汗。郭大路道"你不用害怕，假如我们真的已死了，还有什麽好害怕的？假如我们还没有死就更不必害怕了。"

　　一个人叫别人莫要害怕的时候他自己心里定在害怕。

　　燕七道"我们是继续往前走？还是退回去？"

　　郭大路道"我们是往後退的人麽"燕七道"好·不管好歹·我们先往前面闯闯再说"

　　两人的手握得更紧大步向前冲出。

　　突听声大喝·道"站住"

　　喝声响起黑暗中突又闪起了七八点鬼火。

　　阴森森的火光飘飘荡荡的悬在中空。

　　他们已可以看到前面有张很大很大的公案。

　　案上有个笔筒还塔很多个本子也不知是书？还是帐簿？

　　一个人正坐在案後翻本帐沥。

　　他们还是看不清这人的面目依稀只看出这人好像长很长的胡子，头上还颤顶古代的皇冠。

　　刚才那鬼影也在公案旁，还是上不天，下不地的吊在那里手里好像拿块很大的木牌。

　　难道这就是招魂牌？

　　难道这地方就是森罗殿？

　　上面坐的就是阎王？

　　他仍不知道谁也没有到过森罗殿谁也没有看见过阎王。

　　但他们却巳感觉到种阴森森的鬼气令人毛骨惊然。

　　上面坐的阎王居然说话了。

　　那声音也阴森森的带鬼气·通"这两人阳寿未尽为何来此？"

　　那鬼影子道"因为他们犯了罪。"

　　阎王道"犯的是何罪？"

　　鬼影子道"贪饱之罪。"

　　阎王道罪在儿等"

　　鬼影子道"男人好吃必定为了女人好吃必定为娟此罪列为第七层地狱永世不得吃饱。"

　　郭大路突然大声道"说谎的罪更应该该打入十八层地狱……"阎王拍桌了喝道"大胆，在这你也敢如此放肆？"

　　郭大路道"无论你是人也好是鬼也好只要冤枉了我我都非放肆不可。"阎王道"冤杆了你什麽？"
