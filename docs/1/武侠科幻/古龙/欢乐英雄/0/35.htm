　　林太平道"我在想……春天好像已经过去了。"郭大路点点头·道"不错，春天已经过去了，昨天刚过去的；"

　　郭大路微笑道"你难道不知道麽？昨天那位小站娘跑走的时候春天岂非也已跟她一起走了麽？"

　　提花篮儿上市场。

　　提花篮儿，空钱袋。

　　怎麽回去见爹娘？"

　　郭大路看了看他，又看了看那小站娘悄悄的退了回去。

　　春风温柔的就象是情人的眼波。

　　如此美的春光，怎能不让朋友来同享？

　　郭大路终于伸出手轻轻的敲门。

　　没有回应。

　　敲门声再大，还是没有回应。

　　燕七怎会睡得这麽死？

　　郭大路大声唤道"太阳已经晒在头上了，还不起来"

　　背後却有了声音·是王动的声音。

　　王动道"他不在後面院子，也不在厨房。"

　　郭大路道"你真的什麽都没有说"

　　王动道"去找杯酒喝喝。"

　　春去何处从来没有人知道
