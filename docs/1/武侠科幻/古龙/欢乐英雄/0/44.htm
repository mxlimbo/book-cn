第二十六章　玉玲斑

　　她竟然就是玉玲斑。

　　没有人能相信这是真的事没有人愿意相信。

　　这温柔善良纯真的小姑娘真就是那凶狠泼辣骄横的女煞星

　　每个人的目光都盯在她脸上。

　　她垂头，发已凌乱心也似巳碎了。

　　郭大路心里突也不禁有了怜悯之意，长长叹了口气苦笑道"你是他母亲选中的媳妇，却是他父亲的仇人？世上真有这麽复杂的关系？你……你一定是在开玩笑。"

　　他当然也知道这绝不是玩笑，但却宁愿相信这不是真的。

　　玉玲斑笑得更凄凉路然道"我明白你的好意只可惜世上有些事就偏偏是这样子的。"

　　郭大路还是不明白。

　　五玲斑垂头，道"陆上龙王和我们玉家的仇恨已积了很多年，二十年前就发过誓定要亲眼看到玉家的人全都死尽死绝。"

　　郭大路失声道"你父亲是不是他……"

　　他不敢问出来因为如果玉玲斑的父亲真是死在陆上龙王的手里，杀父的仇恨，就没有别的人能够解得开了。

　　玉玲斑却摇了摇头，道"我父亲倒不是死在他手上的。"

　　她目中又露出了怨恨之色，冷冷接道"因为他就算有天大的本事也设法子再杀一个已经死了的人。"

　　玉玲斑道"我母亲不姓五姓卫。"

　　郭大路苦笑啪哺道"我早就说过，一个人太有名也不是件好事。"

　　玉玲斑道"也并不太坏。"

　　郭大路道"世上有很多人都是那麽样活，而且活得很好。"

　　玉玲斑道"但我们玉家从来没有那样的人，玉家的声名也不能从我这代断绝。"

　　郭大路道"现在你母亲呢？"

　　玉玲斑默然道"已经在去年去世了。"

　　她咬嘴唇，道"她临死的时候还怕陆上龙王放不过我，所以特地去找她的妹妹。…"

　　玉玲现也开没有再看他·但眼泪却巴悄悄的流了下来─…·

　　人就是人

　　晴空如洗碧万里。

　　没有暴风雨。

　　五玲斑凄然笑道"我明白你的意思我本来也想这麽样做的可是…。"

　　她垂下头垂得很低轻轻的接道"现在无论怎麽样做，都已太迟了"

　　不错现在已太迟了。

　　机会一错过是永不会再来的。

　　她本就是个不肯低头的人。

　　红娘子垂下头，知道自己的确已不必再说了，也不能再说下去。

　　红娘子道，"你说。"

　　玉玲斑望王动的窗口轻轻地问道"你的确很能了解别人但却为什麽好像偏偏不能了解他呢？"

　　屋里为什麽忽然没有了他的呼吸声？

　　陆上龙王斜倚在他的虎皮软蛇上盯王动就象要在他脸上钉出两个洞来。

　　连王动自己都觉得脸上似已被钉出两个洞来。

　　他从未看见过这麽样的眼睛从来未看见过这麽样的人。

　　陆上龙王应该是个什麽样的人呢

　　可是他想错了。

　　他看到陆上龙王，就知道无论谁想激起他的怒火都很不容易。

　　无论谁都不会怪他失礼。

　　陆上龙王道"你们究竟要什麽？"林太平道："我们耍的现在我们已经有了。"

　　他握住他妻子的手充满了幸福和满足。因为他要的是自由、爱情和快乐现在他全都得到。

　　这绝不是别人赐给他们的，也绝没有任何人能给他们。

　　你若也想要自由、爱情和快乐就只有用你的信心、决心和爱心去换取，除此之外绝对没有别的法子。

　　绝对没有。就因为他们明白这道理，所以他仍才能得到。所以！

　　他们永远都很快乐。

　　谁说英雄寂寞？

　　我们的英雄就是欢乐的．
