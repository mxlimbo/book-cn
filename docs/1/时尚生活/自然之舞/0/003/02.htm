飞行伞之特性
（飞行原理）

　　飞行本身上毫无任何动力，它之所以能够飞行，除了伞衣充满空气后，显出特殊的形状外(飞行翼)，全赖飞行员本身的控制，结合大气中种种特性(空气动力)飞行。传统式的降落伞，既一般降落伞在空中只能产生下坠阻力，毫无升力可言，而飞行伞在空中飞行中会产生速度和升力，而且它的速度和升力远较它的阻力大。因为飞行伞在购造上，伞衣内层结购设有气囊，在未充满空气前并无实质的棱角，但当伞衣内层气囊充满空气后，伞 的前缘就显出轮廓(棱角)，如此飞行伞在空中飞行时将相对的气流由翼面上下分别引开流动，阻力与对方的风力平行，重量与伞翼上方空气相结合使飞行伞产生速度前进。
