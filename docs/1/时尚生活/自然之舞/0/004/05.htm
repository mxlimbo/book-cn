逆水而上去溯溪

　　炎炎夏日，踩著清流碎石，缓步涉入清幽宁静的溪谷中，由下游往上游溯行，探寻一条溪流的生命之源，看它的生命过程诸如鸣咽低吟、汹涌壮阔、飞瀑急湍、绿碧深潭，甚至难见的景观都出现。这就是为什麽许多人舍坦荡正路不走，宁愿采溯行溪谷的方式，因为在溯源寻根的过程中获得确实太多了。

　　一、溪谷分类：以高度和行程来区分如下等级：

　　1.初级溪谷：

　　一千公尺以下，一天或一天半可以往返，流程起伏不大，适於新手的信心建立与兴趣培养，如大屯山区部份溪谷。

　　2.中级溪谷：

　　千公尺至二千公尺，二天至三天行程，具相当程度的瀑布、深潭、峡谷地形，颇具困难度且须露宿的行程均称之。可进一步让新手了解溯溪的实质意义，如插天山区溪谷。

　　3.高级溪谷：

　　二千公尺以上，需四、五天以上行程，有高大瀑布、深潭、峡谷地形，具高难度且须紧急露宿的行程均称之。须有技术熟练和体能良好者尝试，如台湾各高山溪谷。

　　二、溪攀方式：依诉求目的有如下两种：

　　以地域研究为主的溯溪：如调查某山区的所有溪流并兼综合调查其棱脉、溪谷、岩壁和步道及动植物人文生态资源。这种方式可说所有的溪谷都是未知的处女地，更无任何资料可供参考，必须动员很多人，经年累月利用假期逐一探勘始能完成。

　　漫无目的的溯溪：兴趣相投的山友合组小队伍，伴行溯溪，尽兴而上，这是目前国内最盛行的方式。但此方式缺乏研究精神，实难建立完整溯溪学及资料。

　　三、溯溪的装备

　　由於潜在性的危险，须要有一些专用装备来保护身体和物品，尽量减少外来的伤害和破坏。原则上其装备是登山装备加上攀岩器材、水上设备，再予以防湿处理。入门初学者最好跟随溯溪团体或专家，可省去闭门摸索，装备上亦可减轻团体装备的购置，然一些基本装备仍不可无：

　　溯溪鞋：为溯行者必要的装备，不要使用磨损率大的草鞋和较无阻滑作用的矶钓鞋，要使用不织布底潜水布质料的溯溪鞋，既有阻滑效果又可收保暖之实，选择上以两指式抓地力较强为佳。

　　护腿：为潜水布质料，除可防寒外，又可免於杂木石头碰伤、割伤腿部之虑，分长统和短统两种，长统除护小腿外并可护膝。

　　头盔：可用轻便的攀岩用岩盔，或国内的工程帽替代。头盔对溪攀滑落及落石袭击，有保护头部免於直接的伤害。

　　直式背包：以能塞溯溪装备、基本登山携带用品等，太大太小均不适合。

　　防水内袋：溯溪有时得在深潭峡谷泳渡，背包下水是常有的事，要使背包的东西达到防湿的效果，单靠东西用塑胶袋绑起，仍是不够的，因此背包内加装一个防水内袋是必要的。

　　排汗内衣：溯溪的衣物讲求快乾保温，市面上一种Polyester 100%的保温排汗内衣可满足溯溪者的需求。

　　切忌穿牛仔裤而以具伸缩性大的运动裤即可。以上各用品在登山钓鱼店均可购得。

　　四、溯溪的计划和执行：

　　溯溪是集合登山、露营、攀岩、野外求生等综合技能的一项全身运动，危险性相对地较其他活动为高，因此从溪流的选择、队伍组成、溯登季节的考虑、资料的搜集、路线的决定、交通问题的解决、溯溪装备的整理、粮食计划与采购、医药品的准备、溯行人员职务分配，对於一个筹划者或参与者不可不知。
