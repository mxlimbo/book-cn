极限女王Viviane

　　从美国parachutist杂志98年版的节选：
　　“这又是一个新的记录，空间中心队的Viviane Wegrath成为在ESPN极限运动会上首个获得奖牌的女运动员。空间中心队获得本次大赛的亚军。获得这个亚军并非易事，因为Viviane因为一次脚腕的受伤而被迫在比赛前休息了五个星期。”

　　Russ Francis是跟随Viviane拍摄全程空中滑翔赛况的ESPN记者。他说“Viviane的技术足可进入世界一流的行列。她的准备，她的意志，她的紧凑足可证明她以为此奋斗了多年。”作为橄榄球联盟冠军，前 San Franciscos 49人队的成员，Russ知道极限运动的亚军意味着什么。

　　Wegrath说她进行了大量的药物治疗来世自己在比赛中保持状态。她的风格就是良好的技术，与队友的完美合作以及高雅的类似于舞蹈的动作。

　　“我喜欢空气中飞翔的感觉,"Viviane说：“这是一种完美的感觉。我热爱这个运动。”
