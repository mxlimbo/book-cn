WTO协议之美国利益

作者: 小路
( November 18, 1999, 02:08 PM )

　　美国的农业和商业利益是促使美国签订美中协议的主要原因。

　　美国的工商界把中国，这个世界上人口最多的国家看作是他们未来最有利可图的市场，并且美国的农业部门希望开放中国市场可以帮助美国的农业经济从目前不断衰退的窘境中摆脱出来。

　　参与协商签订协议的美国贸易代表声称：美国国会目前需要讨论通过这一永久的中国普惠国待遇，而不再是以往每年讨论通过最惠国待遇。美国贸易代表发言人对于美国国会能否通过这一永久的普惠国待遇表示乐观。

　　积极的商业组织

　　美国主要的商业组织可能会在协议签署方面可能发挥很大作用，这些组织包括美国商会，美国商业圆桌协会，以及美国CEO（执行总裁）组织。

　　美国商会一直在关注该协议，并声称他们会组织强大的游说团以支持中国进入世贸组织并获得普惠国待遇。

　　美国商会主席托马斯说：“美国共商代表团将向国会阐明为什么中国进入世贸组织对美国未来经济增长和增加就业机会方面意义重大。”加入游说团的公司包括波音，美国最大的出口商之一，98年的出口额为2650万美元。

　　中国被认为是世界上最大的飞机市场，在未来的20年内可能购买总值1200亿额美元的商业飞机。波音一直是中国入世的支持者，当然条款需要体现“各方公平”。

　　“中国入世的协议可以为美国工农界带来很多就业机会，这对于美国人和他们的家庭的未来都有好处。”波音的总裁Phil Condit说，“它为美国的商品和服务打开了一个巨大的新市场，我有绝对信心美国的优秀的企业会在未来的一年中会赢得大量的新业务。”Phil Condit极力劝说国会批准这一协议“这样可以使美国从这个开放市场的协议中获得全部利益。”

　　中国的农业市场

　　作为协议的一部分，中国同意削减农业进口关税14.5%-15%。许多美国的农业组织一直支持中国加入WTO，这样可以让他们的商品进入这个市场，并且这个协议也突出反应了他们的利益。

　　“我们绝对支持中国加入WTO，无论需要我们做什么，我们都将支持它。”美国农业联邦总署的发言人Chris Noun说。

　　中国巨大的尚未开发的市场为美国的农民和农业部门带来希望，美国的农业显然没怎么享受到美国经济的空前繁荣。由于98年亚洲地区性经济危机，亚洲出口市场大幅下滑，给美国的农业经济带来巨大打击。

　　“这是一个新市场。在中国有十多亿消费者，并且他们的消费高质量农产品，加工食品和初级产品的能力在不断上升，坦白地讲美国急需这样一个市场。”Chris Noun说：“美国农业经济从亚洲危机中回复过来还需要很长一段时间，但是每一个市场对其都是有利的。”
