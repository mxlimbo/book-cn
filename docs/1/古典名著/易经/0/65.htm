《易经·系辞上传》

第一章

　　天尊地卑，乾坤定矣。卑高以陈，贵贱位矣。 动静有常，刚柔断矣。方以类聚，物以群分，吉凶生矣。 在天成象，在地成形，变化见矣。

　　鼓之以雷霆，润之以风雨，日月运行，一寒一暑，乾道成男，坤道成女。

　　乾知大始，坤作成物。

　　乾以易知，坤以简能。

　　易则易知，简则易从。 易知则有亲，易从则有功。 有亲则可久，有功则可大。 可久则贤人之德，可大则贤人之业。

　　易简，而天下矣之理矣；天下之理得，而成位乎其中矣。
