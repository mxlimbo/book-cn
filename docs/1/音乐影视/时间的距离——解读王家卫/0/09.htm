东邪西毒

贾志俊

　　在王家卫的作品中，我特别喜欢“东邪西毒”！倒不是因为王氏其它的作品不好！事实上王家卫的每一部作品，都是难得一见的佳作！但是我却独爱“东”片，因为他和王氏其它作品有着显著的差异；“东”片虽不像王氏其它作品那样自由、颓废，甚至有些虚无飘渺，像是现在都市生活的预言！而“东”片却像是精雕细琢、匠心独具，可以传世的复杂工艺品一般；远离都市深入大漠的场景，却依然吐露出那份特有冷漠、孤独及疏离感！在感情上的描绘，丝丝入扣，有着一份比较以往作品的更多复杂及细腻；除了提供一个思考空间外，更透过剧中人物彼此的关连性，及哲理般的对白，告诉我们一个解决感情问题真实且真正面对的答案！（这点和王氏以往作品有着相当大的差异）

　　杜可风的镜头呈现出的那种疏冷、精致却又有些梦幻的风格，及其间夹杂着些看来略带粗沙粒的沙漠摄影，和蒙太奇氏打斗场面，水中闪闪的倒影，编织出一幕幕犹如诗画般的意境。（我甚至觉得杜氏在本片中每一幅剧照，都可以作为单独的摄影作品来看待！）无怪乎可以摘下威尼斯影展的奥赛拉摄影特殊贡献奖。而透过杜的镜头，也似乎更可以深入王家卫影片剧中人物感情世界 其中最精彩的两段便是在慕造访西毒在屋，屋内鸟笼的影子，不时的映在慕的脸上！暗喻慕的处境犹如鸟笼一般。其后，慕与西毒相拥而眠，慕将欧当作"东邪"，而欧则幻想慕为其大嫂。当慕将手伸进欧的内衣，镜头出现的竟是黄药师，而欧转身而去，竟面对的是大嫂。上下的连接镜头中，早已将两人的心中世界清清楚楚的交代！（我这才发现，原来镜头也会说话！）

　　王家卫的电影美学，若是没有杜可风，必定失色不少！然而张叔平的美术设计，加上陈勋奇的配乐，几乎使我们忘记这是一部武侠片！？看了"东邪西毒"之后，我才惊觉其实我无须羡慕像"原野奇侠"那样美的西部电影，原来中国的西部竟也是如此的迷人！
