韩与莉亚：走私者的爱情故事(下)

　　结果蓝道还是不值得信任。

　　说什么带我们去吃点心，却把我们交给达斯维德。

　　维德身旁的那个不是波巴·费特吗？难道就是他找到我们的？韩从拷问室出来的时候想着。

　　突击士兵把韩拖回牢房。

　　「我觉得糟透了。」他对丘伊说。丘伊扶他躺在床上。

　　莉亚也被送进来了。「他们为什么这么做？」她问。

　　「不知道，他们跟本不问我问题。」这时蓝道和两个守卫走进牢房，蓝道解释说维德要的是路克，他们只是饵，维德还答应把莉亚和丘伊交给他处置。

　　韩愈听愈气。「你真是把我们害惨了。不是吗，好朋友？」说完韩翻身下床，给了蓝道一拳。他把蓝道打倒在地上，却也挨了守卫一枪托。

　　蓝道爬起身来和守卫走了出去。「你对人还真有一套。」莉亚轻抚着韩的伤，说道。

　　韩、莉亚、丘伊以及丘伊背着的崔皮欧被带到一间奇怪的房间。维德和费特站在房间的另一端。「现在又怎么了，伙伴？」韩问蓝道。

　　「你要被送去冷冻碳化。」蓝道回答。

　　听到这个，愤怒的丘伊开始攻击身旁的突击士兵。

　　韩阻止他，说道：「停下来，丘伊，这样帮不了我。保留你的力气，我们还有机会的。」

　　韩看了一下莉亚，「公主，你要照顾她，听到没有？」丘伊总算静了下来。

　　韩转身看着莉亚公主，看着她悲伤的眼睛。韩走向她，给她最后的，深情的一吻。突击士兵硬生生拉开两人，把韩拉到房间中央的圆台上。

　　「我爱你。」莉亚告诉他。

　　「我知道。」韩回答，看着眼泪滴下莉亚的脸颊，韩不忍心再伤她的心。

　　圆台慢慢的向下降，韩看着他的朋友，他的爱。圆台降到底了，韩觉得四面八方都有冷风吹来，他心中最后一个念头是：她爱我…

　　…韩跌到地上，全身冷的发抖。有人扶他坐起来。韩挣开眼睛却什么也看不到。

　　那人的声音很奇怪，他说：「放轻松，你才从冷冻碳化中解冻出来。」

　　韩伸手摸了摸自己的脸，发出一声呻吟。

　　「嘘，你得了冬眠症候群。」那人又说道。

　　「我看不见。」韩吃力的吐出这几个字。

　　「你的视力会恢复的。」

　　「我在那里？」韩问道。

　　「贾霸的宫殿。」韩还是认不出那个人的声音，伸手去摸那人的脸，却发现那是一个头盔。

　　「你是谁？」韩问。

　　只听那人脱下头盔。一个熟悉的声音说：「一个爱你的人。」

　　「莉亚！」韩喜道。

　　莉亚吻了韩，说「我要把你救出这里。」

　　韩在莉亚的搀扶下站了起来，安静的宫殿突然回荡着笑声。那是贾霸的笑声。

　　韩试着说服贾霸放了他们，但贾霸还是命令手下把韩从莉亚身边拉开。

　　韩被拖到地牢中，却发现丘伊也被关在那。韩一边试着从丘伊的拥抱中挣脱出来，一边听丘伊说着他被碳化时发生了哪些事。隔天韩和丘伊被带到贾霸面前，路克和莉亚也在那里。贾霸要把韩、路克和丘伊送到沙丘海去喂沙雷克。

　　由于韩的视力还没完全恢复，在沙雷克坑那儿向贾霸反击的行动韩帮不上什么忙，但他仍然从沙雷克的嘴边把蓝道救回来，还意外的将波巴·费特送到沙雷克的肚子里。

　　解决了贾霸，一行人离开塔图音，飞向集结中的反抗军舰队。

　　又一次任务简报，但这次韩的心态和上次完全不一样。之前马丁将军问韩是否愿意率领一只突击队时，他毫不犹豫就答应了。大概是受了莉亚的影响吧，韩自嘲着。现在马丁将军正在前面向大家解释突击队的任务：潜入安铎的卫星，摧毁第二死星的护盾产生器。

　　「不知道他们找谁来执行这个任务？」莉亚低声问韩。

　　韩有些尴尬。

　　他正想着该怎么告诉莉亚，马丁将军突然问他：「索罗将军，你的小组准备好了吗？」看到莉亚不但不生他的气，而且还自愿加入他的小组，韩松了一口气。

　　通过帝国的安全检查后，突击队降落在安铎的森林卫星上，在韩的率领下朝目标走去。走没多久他们就遇上了帝国的哨兵，韩决定由他和丘伊亲自解决，但他出了点差错，让两个侦查士兵骑飞行机车逃了。

　　「嘿，等等。」韩看着莉亚和路克骑着剩下那部飞行机车追过去。

　　韩焦急的在原地等着，但是只有路克回来。担心莉亚的他命令突击队员们先走，他、路克、丘伊和两个机械人出发去找莉亚。

　　他们没有找到莉亚，却被一群伊渥克抓到他们建在树上的村落。韩很高兴的发现莉亚也在这村落里，但韩随即了解到她是宴会的来宾，自己却是主菜。

　　所幸路克运用了一点原力的技巧让伊渥克放了他们，并邀请他们参加部落会议。会议上崔皮欧述说着反抗军的故事，韩轻轻搂着莉亚，她的头靠在他的肩上，闻着她的发香，韩静静的听着。

　　听完反抗军的故事，伊渥克们决定帮助反抗军。

　　韩和族长商量好后，回头不见莉亚和路克。韩走出茅屋，见到莉亚正看着路克离开。他走到莉亚身边，发现她在啜泣。

　　「嘿，怎么了？」

　　莉亚擦了擦眼泪。「没事。我…只是想独处一下。」

　　「没事？少来，告诉我到底怎么了。」

　　「我…我不能告诉你。」

　　「你和路克说了？他才是你说话的对象？」韩失去理性了。

　　「我…」

　　韩生气的转身走开，想了一想，又转身走回莉亚身边。「我很抱歉。」

　　「抱着我。」莉亚说。韩紧紧的把莉亚抱在他怀里。

　　在伊渥克的帮忙下，突击队找到了帝国基地的后门，却遇到帝国的埋伏。在反抗军及伊渥克和帝国军队的混战中，韩和莉亚来到基地的门旁。韩尝试修改门锁的线路，身旁莉亚掩护着他。突然一发雷射击中莉亚的手臂。她痛的叫了一声。

　　「我看看。」韩关心道。

　　「不是很严重。」莉亚说。

　　「不准动。」帝国士兵从莉亚身后喊道。

　　韩举起双手，看到莉亚从怀中拿出一把枪。

　　他和莉亚四目交投，「我爱你」韩说。

　　「我知道。」莉亚笑着。

　　「手举高，站起来。」帝国士兵命令道。

　　莉亚转身向帝国士兵射击。

　　顺利炸毁护盾产生器后，韩帮莉亚包扎手伤。天空上第二死星爆炸成一颗大火球。

　　看着莉亚担心的脸色，韩安慰说：「我确定那东西爆炸时路克不在里面。」

　　「他不在。我感觉得到。」

　　韩吸了一口气，「你爱他，不是吗？」

　　「对。」莉亚笑着说。

　　好吧，韩想。「没关系，我了解。他回来以后我不会挡在你们中间的。」

　　「韩，他是我哥哥。」莉亚又笑了。

　　什么，韩糊涂了。哥哥？那他们不是…？那路克不是…？

　　莉亚亲了韩一下。她甜甜的笑着。

　　韩懂了：她是爱我的，她是爱我的！韩也笑了。他也吻了莉亚。
