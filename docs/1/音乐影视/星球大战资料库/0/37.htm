星球大战前传中的明星们

　　在演员使用上，卢卡斯此次也与从前有所不同，当年的哈里森·福特名不见经传，整个片中没有什么大牌演员。《星球大战前传》的几位主演虽也谈不上一流，但都是家喻户晓的人物，男主角里安·尼森因主演《辛德勒的名单》一片而享誉国际影坛，这次他长发垂肩，络腮胡须，据说迷倒了不少女影迷。年方１８岁的美少女妮坦莉·褒曼在《星球大战前传》中扮演阿美达娜皇后，当年她因《这个杀手不太冷》而一鸣惊人时才１２岁。９岁的杰克·利沃伊德在片中扮演举足轻重的阿纳金，以前他曾因与施瓦辛格一同主演《铃儿响叮当》而出名。几个配角也大有来历，比如一流影星塞缪尔·杰克逊。最妙的是制片人、录音师都在片中走来走去。卢卡斯自己没有出现，这多少让影迷有些失望。
　　互联网影迷评选最佳场面：

　　１　水下城市
　　２　奎伊与达斯·摩尔的决斗
　　３　纳布星球的最后决战
　　４　逃亡时机器人自动修复飞船的场面

　　最佳演技
　　阿纳金的扮演者杰克·利沃伊德
