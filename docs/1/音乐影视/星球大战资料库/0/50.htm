黑武士与黄药师

　　作品受到欢迎，就可能刺激作者推出续集，不过有两部经典，我更赞成制作“前传”：第一是武侠小说射雕三部曲，可惜金大侠不会再写了；第二是美国电影《星球大战》三部曲，其前传影片已经面世，而黑武士的崛起正是其中重要内容。

　　１９７９年公映的《星球大战》（ＳＴＡＲ　ＷＡＲＳ），以及后来推出的第二集《帝国反击》（ＴＨＥ　ＥＭＰＩＲＥ　ＳＴＲＩＫＥＳ　ＢＡＣＫ）、第三集《杰迪归来》（ＪＥＤＩ　ＲＥＴＵＲＮＳ），被称作科幻电影的里程碑，影响广泛而深远。８０年代中，我家所在的大学校园搞“美国电影周”，让我小小年纪就接受了原装正版的《星球大战》，作为看电影的启蒙。海湾战争之后，不服输的伊拉克拍过一部展示武力的纪录片，竟配上了由约翰·威廉斯爵士创作、伦敦交响乐团演奏的《星球大战》主题曲，因为近二十年来的音乐作品中，只有星战主题是这样气势雄壮、深入人心。

　　《星球大战》系列最重要的意义，在于电影科技运用的开创精神，另外其情节也很吸引人。这个讲述银河系正邪力量对抗的剧本由制片兼导演卢卡斯自己创作，结束时并没有留下多少拍“续集”的伏笔，倒是背景故事还有发掘的余地，譬如黑武士达斯·瓦德的来龙去脉。看这片子的时候，随着情节“往后”展开，对这大反派传奇背景的了解，却是点点滴滴“向前”增进——感觉正像读射雕时跟随郭靖、黄蓉的历险，而渐渐沉醉于桃花岛往事、五大高手恩怨的零散倒叙，直至心驰神往而感意犹未尽。作为介于黑暗与光明之间的角色，黑武士的魅力直逼东邪黄药师。

　　用当今最棒的特技，包装本来就具有深深观众基础的“背景”故事，成为“制作前传”这个天才主意。卢卡斯的好朋友、也是满脸胡子的斯皮尔伯格，拍过大名鼎鼎的电影印第安纳·琼斯三部曲，后来卢卡斯又导演了电视剧《印第安纳·琼斯少年时》，因此他拍前传的经验，大可让《星球大战》迷们放心。最近电视上播出《星球大战》前传的制作花絮，饰演童年黑武士的小朋友说：“人们会忘了泰坦尼克号！”——有一种天真而直率的狂妄劲儿，让我们这些老朋友忍不住在心里叫好。
