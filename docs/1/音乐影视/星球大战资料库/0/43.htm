《星球大战前传》让卢卡斯“晚节不保”？

　　被称为世纪末最后的大片《星球大战前传：幽灵的威胁》昨日登陆广州。许多《星球大战》的影迷、卢卡斯的追随者们看了首场演出后深感失望，觉得在这１５年中，卢卡斯的电影除了在制作技术上更先进了一些外，其他方面不仅不见长进，反而陷入不该有的拙劣和庸俗。人们期盼已久的《星球大战》新作只是一部场面浩大、技术先进的平庸“儿童动画片”而已！

　　《星球大战》被公认是世界电影史上最著名的影片系列之一，首次推出是在１９７７年。１５年前，卢卡斯在完成《星球大战之六：斋殆武士归来》之后，便远离这一系列。已完成的３部影片，十几年来历演不衰。１９９３年，卢卡斯重返摄影棚，开始拍摄《星球大战前传》，《前传》故事发生时间比１９７７年公映的《星球大战》的故事早３０年，讲述的是银河系中一群邪恶势力企图控制星际贸易，负责维护正义与和平的斋殆武士去排解纠纷，他们意外结识奴隶之子、天生具有特殊宇宙能量的少年安尼肯并收他为徒。他们怎么也没想到，安尼肯长大后成了邪恶的黑武士……

　　整部故事情节简单，运用的是好莱坞科幻片最常用的模式：邪恶势力破坏秩序，正义力量出面维护和平。模仿自东方功夫的武打造型，在这部超现代的科幻题材影片中显得极其拙劣。影片表现的“英雄主义”更是浅薄、单一，让人感觉出美国人惯有的那种作为世界主宰者和拯救者的狂妄。

　　《星球大战前传》的所有精彩之处，似乎都在外在形式上，以浩大的场面和离奇古怪的场景、服饰，给观众造成强烈的视觉刺激，取悦现代观众的流行趣味。虽然它采用的是最先进的数字化技术，在音乐和服装上都大下工夫，但依然无法掩饰其内容的空洞与贫乏。

　　《前传》一问世，就受到不少观众的批评。认为《前传》是《星球大战》系列中最差的一部，是卢卡斯的一大败笔，可谓卢卡斯“晚节不保”。
