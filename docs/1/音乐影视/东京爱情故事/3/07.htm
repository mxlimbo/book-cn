第七幕 爱媛


S t a g e 7-1 < From Vol.7 >
背景: 公司营业部

< 完治正走回营业部,却发现莉香坐在他的座位上,不知在写着什么,
完治疑惑的走了过去 >

完治问莉香: 你在作什么?
莉香: 等等...好了。

< 说着莉香拿起了手上的纸站起身走到旁边,完治连忙也跟了过去 >

莉香问对面的同事: 有没有完治的(指印章)。
同事回答说: 有。

< 莉香过去拿了印章盖了下去,完治在旁边看着纸上写的字 >

完治边看边念: 请..假..单。

< 莉香走向和贺部长的办公室,完治赶紧从后面追上,拿走莉香手上的请假单 >

完治对莉香说: 我没说要请假。
莉香指着完治大叫: 啊.....

< 完治坐下来办公,不理会莉香,莉香就伸手拿了完治桌上的一叠文件,跑了出去
,完治尴尬的看了看同事,不好意思的赶紧追了出去 >

完治大声说: 还我。
莉香: 不要。
完治大喊: 喂!
莉香跑到楼梯旁,拿紧了文件说: 交换!
完治着急的说: 那是重要的文件。
莉香指了指完治手上的请假单说: 那也重要。
完治挥了挥手上的请假单说: 哪儿(重要)?
莉香把手放在心口上说: 我的爱都在那上面。
完治生气的指着请假单说: 不要随便放在这种东西上。
莉香不高兴的说: 这种东西!! 这么想的吗?
完治: 不是! 这种东西不会有『爱』的。
莉香反问说: 那么哪儿有? 给我看!
完治: 给你看?
莉香认真的说: 完治爱我的证据。
完治为难的说: 在这边吗?
莉香点点头说: 对!

< 完治站着不知该如何表示,脸上满视为难的表情 >

莉香突然笑了起来说: 在想什么低级的事嘛! ...才不是!
完治不高兴的用力挥着请假单大声说: 那么是什么? ....要我怎么办吗?
莉香也大声的回说: 怎么办!? 那种事也要我说吗?

< 莉香说着就把手中的文件向楼下丢去 >

完治大叫着: 啊....
莉香悠哉悠哉的说: 掉下去了...就像我们的爱情...
完治对着莉香说: 你啊....

< 说着完治赶紧下楼去捡文件>

完治边下楼边对莉香说: 还有话要说....你等着...
莉香心情愉快的回答着: 会一直等的!

S t a g e 7-2 < From Vol.7 >
背景: 清晨,完治家中

< 完治和三上正在熟睡,传来了一阵阵急促的门铃声,完治连忙爬了起来,
要去开门,没想到却一脚采到三上的身上 >

完治: 来了。
三上同时大叫: 好痛。
完治回头看了三上说: 你还在啊!
三上问完治: 谁来了?
完治: 会拼命按(门铃)的,全世界只有一个。

< 完治说完打开门,门口果然出现了莉香的身影 >

莉香: Good morning.
完治指着莉香对三上说: 看!
莉香看了高兴的问: 什么? 什么? 知道我会来吗?
完治一边走回床上一边有声无气的回答: 是,是。
莉香高兴的对完治说: 来接你了。
完治诧异的出了声: 噫....不会是现在去爱媛吧!
莉香: 答对了,恭喜。
莉香: 经松山,板仓去爱媛。
莉香说着把两手放在脸上,作出可爱的表情说: 而且还是和这么可爱的女孩一起。
完治: 别开玩笑了。
莉香: 你说要去的。
完治: 是我随便说的。
三上问莉香: 怎么,想去爱媛吗?
莉香点点头: 嗯!
三上说: 什么都没有的地方。
完治躺在床上连忙接着说: 对,对,告诉她。
莉香不理会的说: 要去。
完治只好无奈的说: 知道了,知道了。
莉香听了高兴的说: 真的吗?
没想到完治接着又说: 下个月去。
莉香不高兴的说: 不行! 现在不去就没有意义了。
完治反问说: 什么意义?
莉香: 因为...
完治: 因为?
莉香说不出口: 因为....算了,只是开点玩笑。
完治不解的问: 啊?
莉香解释着说: 不知道我和你一起去旅行,你会不会高兴。
完治: 我没说不行。...怎么会突然想去。
莉香转变着话题说: 算了。两个男人早上起不来,只是来叫醒你们。
莉香伸手撕开完治脸上的贴布并对完治说: 晚安。
完治大叫说: 好痛。

< 莉香说着就转身拿起皮包匆匆的走了,留下三上和完治两个人在房里 >

S t a g e 7-3 < From Vol.7 >
背景: 完治家中

< 完治安慰完关口刚回到家,听见电话铃声,赶紧过去接 >

完治: 喂!
电话那端传来三上的声音: 是我! 刚和莉香分手,大概快回去了。

< 完治默默的听着 >

三上: 温柔的去迎接她吧!
三上没听见完治的回应,不禁问: 喂! 有在听吗?
完治: 罗唆! 你的话能听吗?
三上: 我知道! 但是也该考虑一下她的心情。
她叫你去安慰关口是因为爱你。.....她安慰我也是因为爱你。
爱媛的事也一样,她相信去了那里可以弥补和你之间的空隙。

< 完治听到这,突然把电话给挂断了 >

完治自言自语的说: 我知道。
完治烦恼的继续说着: 这些事我知道!

< 完治叹了口气躺下身,左手不小心碰到莉香早上留下的东西,完治顺手拿了出来
,却看见了那些东西是莉香准备旅行用的日常用品,不禁想到了和莉香的对话 >

< 莉香: 明天就去。 >
< 完治: 什么? >
< 莉香: 当然是去爱媛,明天请假去。 >

< 完治想到这,连忙把莉香早上留下的公文袋中的东西倒出来,没想到是机票 >
< 完治看到这些东西,忍不住心情就激动了起来,放下机票就冲了出去要找莉香 >
[ 这时主题曲的旋律响起了 ]

< 另一边,莉香正从商店中买了包子走出来,慢慢的走回家 >
< .... >
< 完治跑到莉香家前面,坐在上楼的楼梯口等着莉香,正在发着呆时,听到有车声,
抬头一看,莉香也正好回来了,完治站起身等着莉香走过来 >

莉香走了几步,发现完治站在前面,忍不住高兴的叫着: 啊! ...完治!
莉香跑到完治的身前说: 等很久了吧! 很冷吧!
说着莉香把热包子贴到完治脸上说: 买了这个! 暖活吧!
买了三个肉包,一人一个加一半。

< 完治静静的注视着莉香 >

莉香继续说着: 对了,三上好像比较有精神了,你那边怎样?
莉香看见完治没有回答,又问了次: 完治?

< 完治一句话也不说,只是伸手把莉香紧紧的抱着 >

莉香: 怎么了,完治? .....包子会被压扁。
莉香: 喂....会痛.....
莉香: 完治?

< 完治一直没回答,只是静静的拥抱着莉香,好像想永远抱下去似的 >
