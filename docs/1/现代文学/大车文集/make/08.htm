一九九三年诗歌作品

诗歌


——题记

可以这么说，诗歌的存在，是一个奇迹。
我们都是这个奇迹的受益者。
在越来越令人难以适应的现代社会里，我们应该说，感谢上天， 因为我们至少还拥有一样东西。
——那就是诗歌。

更多的时候，我觉得自己更像一名匠人，而不是诗人。
因为我所做的一切，仅仅是将一大堆字粘合在一起，并打磨成自己喜爱的形状。

１９９３年１２月１日晚

有关于爱情和死亡
（组诗）

Ⅰ
　
像次小小的死亡
又似乎留下了什么
我就这样浮出水面
来到陌生的你面前
　
风卷着帘子
看不见的外面
世界正一点点毁灭
你靠在黑黑的门口
提着昏黑的油灯
我们相隔遥远
影子却在天花板上交错晃动
　
月上水面的时候
我坐在门口
太阳在屋后沉下去
我的脸在光中变幻不定
你在床边长久凝视着我
如同辨认远游回家的丈夫
　
　
Ⅱ
　
一个下午
慵懒的阳光
我坐在藤椅中
读着古人的书
　
他们早已死亡
但油墨未干
发出的清香使我疑惑
然后我做着梦
梦中栩栩如生的他们
站在以后耐心等着我
　
醒了以后夜已黑尽
倒挂着的残月
冷冷看着我
　
　
Ⅲ
　
梅花落尽
最后一朵飘浮如你的脸
疲惫地笑着
任凭风吹落
　
我伸出手
你的手指就在掌中剥落
一层又一层缓缓褪去
　
像一个苍白的影子
总是落在我的视线之后
　
从此
我的一生不再有你
半夜淡淡忧伤
伴着早醒的花和我
　
　
Ⅳ
　
我会和树林一起腐烂
那时花已开放
幼苗伸开小手迎接
春天又会再来
而对于我
一切将成为过去
　
生命是只过于苍惶的鸟
它的阴影在我的眼前一闪即过
而我
还未来得及看清它的形状
　
隔我很远的地方
是一片静寂的沉默之地
那是一个思考的好地方
而平时 我们的情感和思想
总是频频为语言所打扰
　
来到那里
我未说出的话
将和秋天的树叶一起剥落
使我们成为独立着的
那一枝枝树干
在黑夜之中
犹自闪闪发光
　

一个陌生的小女孩


　
没有回家的那个小女孩
她一个人沿着冰冷的大街
走向一个沿未决定的终点
她的眼黑得像郁金香
吸引着鸽群飞来飞去
道旁开着满眼的大丽菊
像散开的绒布围在她的周围
　
小女孩低着头
世界仅仅是她眼里的反光
她的茫然不知像是固守枝头的
几片树叶
秋雨之中格外夺目
　
什么时候她会从街上走过来
没有人知道
她自顾自走在笔直的路上
怀抱着一个尚未决定的开始
　

每一个日子


每一个日子
都是无法再续的歌
我们围坐在这年的冬天四周
如同松果围绕着松树
鱼儿包围着河水
　
每一个日子
如同花瓣
在我们的掌中层层剥落
我们停留在玻璃窗的外面
呵出的热气
使镜面变得模糊
　
每一个日子
都是前来告别的朋友
他共享过我的骄傲和卑俗
在他离去的时候
没有留下字条
却碰响了家门口的风铃

９３＼１２＼１ 晚
