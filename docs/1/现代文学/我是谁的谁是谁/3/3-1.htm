刺　激

--1--

　　感观上的刺激并不难求，心灵上的创伤却难以弥合。虽说时间是一剂良药，它可以让我们忘记一切，但有些东西却永远都难以忘怀，那是一份"刻骨铭心并永远感动心灵深处"的情感。

　　我想，令本篇主人公深受刺激的，并不只是他与两个陌生女人之间的什么关系，而是他被他深爱着许久的女友抛弃这一令他无法忍受的事实。虽然他在回忆此事的时候只是一副轻描淡写的口吻，但是，我们不难看出，这种轻描淡写，淡化的只是他自尊心受伤害的程度，因为，在人生最为自然的两种状态（酒醉与性爱）下，他撕去了"大男人"伪装，让我们看到了他隐藏在内心深处的痛苦与欲望。
