梦幻骑士

　　　　

　　"梦幻骑士"是我的英雄--唐·吉诃德。

　　我得到这个木刻，在一个偶然的机缘里。

　　有一次不当心，将吉诃德手中那支矛弄断了，这更像一个刚刚打完仗的他。

　　去年在竹东深山里的清泉。小丁神父将彼德奥图和苏菲亚罗兰主演的这张名片放给我看时，我一直没有受到如同书本中的那种感动，直到那首歌：《未可及的梦》慢慢唱出来的时刻，这才热泪奔流起来。

　　既然吉诃德象征了一种浪漫的骑士精神，身为半个西班牙魂的我，是应该拥有一个他的。
