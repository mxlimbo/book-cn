花生漫画全纪录——花生50年（2）

　　虽然已经有研究者已经把花生漫画作为一个记号语言学方面的课题，我们却不想把对“花生爸爸”的评论写成一篇博士论文。只是再也不能听到大人小孩对着日报上的查理.布朗开心地吃吃笑了。这是全球最幽默漫画系列，在75个国家、用21种语言在2600份报纸上发表过。对全球3亿5千5百万读者而言，“花生”触动了他们心中最深的那根弦。它带给华侨人们想回家吃午餐的愿望，在《花生》世界里，生命的意义就是“回家睡个好觉，希望明天会更好”。而小史努比的愿望则是当一个伟大的小说家，可它所有的小说开篇就是“那是一个狂风暴雨的黑夜......”

　　《花生》和美国三代孩子们一起长大，直到走进今天这个空前繁荣的社会，它从一个独立的角度看待儿童的世界，认为这个世界也是个完整舞台，有自己的喜怒哀乐和人生哲学。在《花生》之前，所有漫 画里的孩子被无一例外地描绘成啕气和父母捣蛋的不懂事的小顽皮。“舒尔茨给我们一个活生生的儿童世界，他们和成人一样有着许多的失落、抵制和悲伤......他们甚至和成人一样有着理想的幻灭和觉醒挫折以及自我否定的过程。”

　　虽然以小孩子作为主角，但《花生》是给任何年龄的人看的漫画。事实上，它受到更多成年人的欢迎。漫画中的角色个性鲜明:查理.布朗做什么都抢购，莱纳斯总是抱着一条毛毯，露西摆摊提供心理咨询......不仅令人印象深刻，而且让读者在他们身上找到自己的影子。其实，除了男主角查理.布朗的性格跟最初没有什么不同之外，每个角色在这50年里都有或多或少的改变，这也算是一种“成长”吗。例如，露西最初是一个挺讨人喜欢的小姑娘，后来脾气就变得自私、霸道；而史努比原来还算一只安守本分、用四条腿走路的小狗，不过自从幻想自己是飞行员或是别的什么之后，就变得更加人性化了。

　　这种跨越代沟的特质，无怪乎父母看着《花生》长大，而他们的孩子仍然会喜欢它。1967年上演的地铁音乐剧“好人查理.布朗”今年2月仍会重演；而从1965年起，年年对话都会播出获得艾美奖的“查理.布朗的圣诞节”，这部漫画的内容至今已被拍成四个版本的电影，建成了三个主题公园，而售出的书更是多达3亿本！

　　“有时候，晚上我躺在床上睡不着，我问‘为什么我这么倒霉？’有一个声音回答我，‘没什么是你独自随的，只是碰巧这回是你罢了。’”——这是老舒尔茨赋予《花生》平静而又深刻的一面。”花生爸爸“说，他喜欢和”成长“有关的题材。”做小孩并不容易“，虽然已经77岁高龄，但他仍然对孩提时代的忧虑和不安记忆犹新，”外面的世界如此可怕，上学也很惨，不是教师找你麻烦，就是给大的同学欺负。如果长大以后都将这些苦恼忘记，那么成年人就会对儿童的问题视而不见。其实，小孩和我们一样，在现实生活中挣扎求存。”
