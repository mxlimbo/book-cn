"拳王"吴宝气

　　吴宝气是个酒仙，一闻到酒，双腿必然特别沉重。他的人生座右铭是：酒醉英雄汉，饭胀傻脓包。喝酒的人不挑酒，啥子酒都能成为他饭桌上的珍品，茅台、五粮液自然不摆⑴了，吴宝气肯定敬仰，可惜没喝过几盘。每日陪伴他的是"有点打脑壳"的"跟斗酒⑵"，或者是红苕秧秧、甘蔗皮皮、玉米须须酿制成的杂酒。在没有酒的日子里，实在惹毛了，醪糟水都要舀上两碗，以解"饿"酒之急。吴宝气逢人便说："我这个肚皮是个酒糟凼凼，啥子酒都装得。"

　　喝酒的人不挑菜，泡菜豆瓣都可以下酒，在这一点上，吴宝气有点遗传。当年他那个酒鬼老爸与他对月畅饮，下酒菜只有桌中间一颗油光锃亮的花生米。父子俩每喝一口，便无限神往地朝桌上狠盯一眼，谁也不舍得伸手。酒过三巡，一瓶老白干马上见底，吴宝气实在抵挡不住来自花生米的煽诱，他刚想跐手，就遭老爸咣了一下。老爸瞪着被酒熏红的眼睛，喝骂道："你龟儿光吃菜，不喝酒！"

　　吴宝气成家后，更是黄鼠狼下耗子——一窝不如一窝，经常为每顿的下酒菜发愁。不过他倒是把祖传喝寡酒的绝技全盘继承下来，并且发扬光大。在确实"饥寒交迫"的时候，只要有酒，要啥子烂花生米？他只需将一根5寸长的铁钉子插进墙里，喝一口酒，顺手把钉子拔出来嗍一口，再喝，再嗍，津津有味，往返无数……在他的心中，那颗锈迹斑斑的大铁钉和一根还有点啃头的卤鸡腿没什么区别。

　　吴宝气不但嗜酒如命，而且还是个划拳高手，自封"巴蜀拳王"，打遍天下无敌手。凡是不知天高地厚来挑战的，在吴宝气的神拳面前连火候都没摸到，便丢盔卸甲，屁滚尿流。吴宝气的拳经很简单：逮你的手型、嘴型，逮你出拳的套路，并且胸有成竹，就像韩国队打中国队一样，赢得你瓜兮兮的，哭兮流了。吴宝气的"拳"是最传统的"乱就乱"，他对现在流行的各类酒令不感兴趣，因为那些新潮酒令花里胡哨，趣味低级，严重影响吴宝气对嘴型手型的判断。

　　这天，吴宝气刚刚关了工资，连忙来到路边一个小火锅店，准备扯它一斤枸杞，顺便烫几串洋芋片、海白菜（是要比嗍铁钉子好点）。他急涝涝地把自行车靠在墙角，跟斗扑爬地来到一口空锅前，边吞口水边对老板吼道："赶快把火点起！"

　　此时，邻桌的一对新潮男女差不多已经喝到位了，男的赤裸上身，袒露出胸前的纹身；女的粉腮赤眼，鬓发早乱。二人开始划拳："我们两个谁傻瓜？你傻瓜呀我傻瓜……"男的似雄狮怒吼，女的如银瓶乍裂。换一种过门："红红的太阳蓝蓝的天，绿油油的稻田一望无边，乡下的农民爱看三级片。"两人四目圆睁，恨不得鼻子对鼻子，眼对眼，大叫一声："好多钱的三级片？"

　　吴宝气不屑地乜斜着这对菜拳，鼻孔里"哼哼"喷着冷气。哪想到那对狗男女得寸进尺，划拳声一浪高过一浪，旁边的吃客纷纷避让，连连买单。酒到高处不胜拳，二人像发情的土鸡："两只小蜜蜂啊，飞到花丛中啊……"

　　吴宝气实在忍无可忍，拍岸大怒："你们两个恐怕要下蛋。"

　　空气顿时凝固，吴宝气也晓得自己过于扯靶子⑶了，正在后悔自己嘴巴有点贱，没想到对方根本没发燃，男的说："哥子，我两个来一盘，不用三打二胜，一拳定输赢！"

　　咋个不可以呐？正中吴宝气下怀。我"巴蜀拳王"未必虚哪个嗦？两人拉开架式，吴宝气说："你刚才跟这位女士用左手划拳，如果你不是左撇子，请你还是用右手，这样比较尊重人。"对方边点头边说："哪个输了，就把面前的一瓶酒干完。"

　　吴宝气暗喜：你娃娃死得早，我已在旁边观察了半天，老子专门逮你的五魁首。说时迟那时快，过门一唱，吴宝气将拳一收，大吼一声："五……"

　　对方确实将手掌全部叉开了，但吴宝气还是输得闭声闭气，那人的右手有六根指拇。

　　空肚皮喝酒经不得风吹，回家的路上，吴宝气边蹬自行车边呕，边呕边埋怨："格老子遇到个六指拳魔，哪里搞得赢？就像国奥队遇到巴林，哪晓得对方有超龄……"

　　1999-11-09
