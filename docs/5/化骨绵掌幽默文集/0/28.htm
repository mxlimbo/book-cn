十九、都是庸医惹的祸

　　近日，贾老练的霉运不断，先是老婆紫薇赌气离家出走，后是在麻将桌上尽捡些大炮来点，就连在大街上走路，也莫名其妙地摔了个狗吃屎。额头上吊起个包不说，关键是左手大拇指的指甲被彻底掀翻，血溅嘉陵江。老鬼正色警告："赶快去打一针破伤风，免得丢了老命。"

　　贾老练急忙跑到隔壁的小诊所包扎伤口，哪想到那个干虾儿医生说："必须从拇指的第一个关节截肢，否则后果不堪设想。"

　　贾老练坚决不答应，高呼："我不要做残疾人！"

　　几天过后，伤口果然开始溃烂，贾老练急忙找到干虾儿。医生仔细检查过伤口，大惊失色道："如果你再不截肢，整个左手就要废掉了。"

　　贾老练泪流满面，哀哀地央求道："能不能再给我的左手一次机会？"

　　干虾儿骄傲地抬起下巴，轻轻地摇摇头。无奈，贾老练只得勉强同意截肢。哪想到干虾儿医生说："毒性已经串到大拇指的根部，要截肢就得把整个大拇指切掉。"

　　贾老练慌了："啥……啥子？那咋个得行？我还要拉小提琴。"

　　"你用大拇指拉小提琴嗦？"干虾儿不解地问。

　　"揉弦、换把位都需要大拇指在琴柄上款起⑴噻！"

　　"再耽误下去，你的整个左手都要切掉，还拉个铲铲琴？不过……"

　　"不过啥子？"贾老练心头燃起了希望。

　　干虾儿医生凑近贾老练，神秘地说："最近我一直致力于断头术的研究，还没有获得成功。不过，移植一根拇指还是非常有把握的。"

　　"从哪里移植？"贾老练听得毛骨悚然。

　　"把你的大脚趾头移上来，虽然不太美观，但也比没有强。"

　　贾老练含泪同意了。于是，干虾儿医生迅速切下贾老练的左手大拇指，又将他的一个大脚趾头趁热接了上去。半个月过后，拆线一看，手术非常成功，只是嫁接的大脚趾显得又粗又短。

　　老鬼参观了贾老练的"大拇指"后，说："你快拉一曲《月亮惹的祸》，看影不影响你的艺术表现力。"

　　贾老练托起小提琴，潇洒自如地演奏了起来。果然，"大拇指"对演奏水平没有任何影响，甚至揉弦揉得比过去更有味道。美中不足的是，贾老练的演奏过程经常被"大拇指"的阵阵奇痒所中断。

　　贾老练解释道："我以前有严重的脚气。"

　　老鬼惊呼："干虾儿医生的医术真有点过关，连脚气都能接着健康发展。"

　　由于大脚趾头是掌管身体平衡的，缺少了一个大脚趾的贾老练，现在走路有点跛。

　　⑴款起：搁着、挡着
