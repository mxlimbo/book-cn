16、You remove a dynamic disk from a Windows 2000 Professional computer and add it to the Windows 2000 Professional computer on your desk.  The moved disk is a simple volume.  You are unable to see the volume from your Explorer.  What should you do?

1.  Restart your computer after completing the disk installation.

2.  From the Disk Management, right-click the disk labeled Foreign and choose Import Foreign Disk. <Correct>

3.  Convert the new volume to NTFS.  Rescan Disks from the Action Menu of Disk Management.

4.  From the Action Menu of the Disk Management snap-in, Rescan Disks.

Explanation :

Your computer will identify the dynamic disk as coming from another computer.  You must import the disk rather than the usual procedure of Rescan Disks applied to brand new physical disks.

 

17、You purchase an infrared-ready printer to use it with your personal digital assistant (PDA) and your Windows 2000 Professional desktop computer.  You also purchase an IrDA transceiver that connects to your desktop's serial port.  How do you install and configure this transceiver?

1.  Enter setup mode for your computer and change the mode of the serial port to IrDA or Infrared. Use the Add/Remove Hardware wizard to detect the device. Use Control Panel's Wireless Link icon to configure the device.

2.  Enter setup mode for your computer and change the mode of the serial port to IrDA or Infrared. In Network and Dial-up Connections set up the wireless link by referencing the infrared port which shows up after you attach the transceiver and startup the computer.

3.  Let Windows 2000 Professional's Plug and Play detect the transceiver.  Use Device Manager to configure the properties of the device.

4.  Use the Add/Remove Hardware wizard to detect the device. Use Control Panel's Wireless Link icon to configure the device. <Correct>

Explanation :

The transceiver attached to a serial port can only be detected by a manual Add/Remove Hardware wizard. Plug and Play cannot detect the external infrared device. 

 

18、A hardware technician reports that he is unable to install the driver for a new hardware device he needs to install on a client computer.  The computer is running Windows 2000 Professional.  When he attempts to install the driver, he receives an error message reporting that unsigned drivers cannot be installed.  When he views the Driver Signing Options dialog box from the Hardware tab of System Properties, the option to ignore file signature verification is disabled, so he cannot select that option.What must you do to enable this option so the technician can disable file signature verification?

1.  Make his user account a member of the domain's Administrators group.

2.  Make his user account a member of that computer's local Administrators group. <Correct>

3.  Make his user account a member of that computer's Power Users group.

4.  Instruct him to use the Add/Remove Hardware icon in the Control Panel.

Explanation :

By default, in the Driver Signing Options dialog box, the option to apply the file signature verification setting as a system default is selected.  If this option is selected and the file signature verification setting is set to Block installation of unsigned files, the options to ignore or display a warning message for unsigned files will be disabled for any user who is not a member of the local Administrators group.  You must make him a member of the local Administrators group so he can override the system default.

20、Your enterprise has employees with finger dexterity impairment.  Which setting must you employ to minimize repeated keystrokes typing errors?

1.  SerialKeys

2.  FilterKeys <Correct>

3.  Keyboard configuration in Utility Manager tools

22、You log on using the local Administrator account to a computer running Windows 2000 Professional.  You want to enable Advanced Power Management (APM) support.  However, when you open the Power Options icon in the Control Panel, there is no APM tab.  What is the most likely reason this tab is not available?

1.  APM is only valid for laptop computers.

2.  APM can only be enabled via Group Policy.

3.  The computer has a BIOS that is ACPI-compliant. <Correct>

4.  APM must be enabled from the BIOS.

Explanation :

If a computer has a BIOS that is compliant with the Advanced Configuration and Power Interface (ACPI), ACPI will automatically enable Advanced Power Management (APM).  You cannot disable APM in this case, so the APM tab is disabled in Power Options.

 

23、You have a small company with 25 computers.  You have TCP/IP as the only protocol on this Windows 2000 network.  You do not have Internet access and the IP addresses are handled automatically with a DHCP server.  You decide to provide limited Internet access for these 25 computers with the Internet Connection Sharing (ICS) feature.  You setup an Internet Service Provider (ISP) and a 56K modem. You specify Shared Access in the Dial-up Connections Property page of the modem connection.  You cannot make the shared service work.  What mistake have you made?

1.  You cannot have a DHCP server on this Local Area Network (LAN). <Correct>

2.  You put the Internet Connection Sharing on the Internet connection rather than the office LAN connection.

3.  You failed to set all computers to automatically receive IP addresses.

4.  You failed to setup DNS service with the ISP.

Explanation :

You have a DHCP server on this LAN.  You must remove the DHCP server so that the ICS can be the DHCP allocator.  The computers on the LAN were already set to automatically receive IP addresses because of the DHCP server.

27、You configure Performance Logs and Alerts to collect performance data for the Processor, LogicalDisk, Network, and Memory objects for a computer running Windows 2000 Professional.  You notice that the counter Memory\Pages/sec has increased dramatically since you created a performance baseline.  Also, the counter Server\Bytes Total/sec. is much lower than in the baseline. 

What should you do to increase the counter Server\Bytes Total/sec and decrease the counter Memory/Pages/sec?

1.  Upgrade to SCSI disk drives.

2.  Increase the size of the pagefile.

3.  Upgrade to a faster processor.

4.  Add memory. <Correct>

Explanation :

When the counter Memory\Pages/sec increases while the counter Server\Bytes Total/sec decreases, most likely there is not enough physical memory to support network operations.  This can be solved by adding memory to the computer.  Increasing the size of the page file, upgrading to a faster processor, and upgrading to SCSI disk drives may improve performance but not as much as adding memory will do.

30、Your project team members are using an Access database to store project data.  You plan to work on the database while you are on a business trip.  You configure your laptop, which is running Windows 2000 Professional, to use offline files.  Access is installed on the laptop.  The Access database is stored in a shared folder on a computer running Windows 2000 Server.  That shared folder is configured for Automatic Caching for Documents.  You open the database while your laptop is docked.  When you undock the laptop, you discover that there is not a copy of the database in the Offline Files Folder on your laptop.Why should you do to enable the database to be used offline?

1.  Access the shared folder and select the option Make Available Offline from the context menu of the database file.

2.  Configure the share to allow Automatic Caching for Programs.

3.  Configure Group Policy to allow the caching of files with the extension .mdb. <Correct>

4.  Add the file type .mdb to the File Types tab of the Folder Options dialog box on your laptop.

Explanation :

You must configure Group Policy to allow the caching of files with the extension .mdb.  By default, files with the extensions .slm, .ldb, .mdb, .mdw, .mde, .pst, and .db cannot be cached.  You can override this by entering the file extensions that you do not want to be cached in the Group Policy setting \Computer Configuration\Administrative Templates\Network\Offline Files\Files not cached.All file types other than the ones you designate can then be used offline.
