MCSE2000认证简介

Windows® 2000途径的MCSE应试者必须通过五门核心考试和两门选修考试，这些考试有效可靠的衡量了设计和实现解决方案中的技术熟练程度和专门技术。（对于已经通过Windows NT® 4.0考试的人，短期内可以进行附加的“加速”考试。）业界内的专家参与了这些考试的开发，它们反映了微软产品是如何使用在世界各地的组织中。Windows 2000 MCSE 途径正在代替 Windows NT途径；两者所必需的核心考试有所不同.

可以作为选修考试的核心考试在一次认证中只能算作一门。换句话说，如果应试者在一条途径中获得了某门考试作为核心考试的学分，那么在该途径中，他不能以这门考试作为选修考试而获得学分。 

　

MCDBA认证简介

 微软认证数据库管理员（MCDBA）证书是为那些实现和管理SQL Server®数据库的专家准备的高级认证。该认证适合于那些设计物理数据库、开发逻辑数据模型、通过Transact-SQL创建数据服务、管理和维护数据库、配置和管理安全、监视和优化数据库、安装和配置Microsoft SQL Server的个人。

    MCDBA认证适合于
    数据库管理员
    数据库分析员
    数据库开发人员

　　通过获得高级MCDBA证书，个人可以证明自己拥有领导企业成功地设计、实现和管理Microsoft SQL Server数据库所需的技能。

　　MCDBA应试者应该拥有至少一年使用Microsoft SQL Server的经验。关于获得微软认证数据库工程师证书的其他要求，请浏览MCDBA的要求。

MCDBA的要求

　　微软认证数据库管理员必须通过三门核心考试和一门选修考试，它们可以对技术的精通和专业技能进行有效、可靠的测量。这些考试随着业界专业人员的输入而开发，它们反映了微软产品在全世界的企业中的使用。
