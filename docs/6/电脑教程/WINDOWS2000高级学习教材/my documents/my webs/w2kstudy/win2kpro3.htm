31、Your enterprise wants one department to have mandatory user profiles. There are 20 persons in this department.  What method should you use to assign profiles to each user?

1.  Save the model account's ntuser.dat to the workstation's %SystemDrive%\Documents and Settings folder as ntuser.man.

2.  Use Folder Redirection to store each user's Documents and Settings folder on the Domain Controller.  Rename the ntuser.dat to ntuser.man.  At each log on, it will be written to their desktop.

3.  Configure a workstation environment for the account.  Copy the ntuser.dat file to each user's home directory as ntuser.man.  Use offline folders technology to bring the saved folders and files to the workstations.

4.  Create a model user account and configure a workstation environment for the account. Create a copy of the resulting ntuser.dat file named ntuser.man in the NETLOGON share of the domain controller. <Correct>

Explanation :

While local user profiles are stored locally on the computer that the user logs onto, the mandatory user profile is stored on a network server and downloaded when the user logs on.  Group Policy settings take precedence over user settings. NETLOGON share is the  %SystemRoot%\SYSVOL\sysvol\domainname.com\scripts folder.

33、You want to add a Service Pack to the Windows 2000 Professional system.  You will update using distribution disks with your CD-ROM drive and your floppy drive.  What command will you use?

1.  update.exe <Correct>

2.  winnt32.exe /s:folder_name /unattend:folder_name\answer_file

3.  update /slip

4.  winnt32.exe /unattend /s:source files

Explanation :

If you do not update from a slipstreamed distribution point on the network, then you will have to apply the service pack with update.exe each time you add a service.

Update /slip is for slipstreaming the service pack onto the installation files at a network distribution point. From this point, installations can be performed without a separate installation of the service pack.

36、You have established an infrared link with your office partner's laptop.  Both computers are running Windows 2000 Professional.  When are you able to send files from Word and from Visio on your laptop to her laptop?

1.  A data connection must be established from within each application using the Save as option and naming the target computer's folder.

2.  A data connection must be established for each program using the Control Panel Wireless Link icon.

3.  A data connection must be established for Word from within the application's Send to and another established for Visio from within its Send to.

37、Your Windows 2000 Professional system runs three applications at the same time in a manufacturing environment.  After using System Monitor, you determine that the paging file could use some adjustment for performance enhancement.  Sometimes the applications actually pause while waiting on the virtual memory.  You have two hard disks in this computer.  The second one holds user data, plus the boot partition.  What are your options for the paging file? (Choose all that apply.)

1.  Set the Virtual Memory Manager option to demand paging with clustering.

2.  Create a second paging file for the second hard drive. <Correct>

3.  On the Virtual Memory dialog box, decrease the space in memory the Registry can reserve.

4.  Move the one paging file to the active, system partition. <Correct>

5.  Increase the paging file initial size <Correct>

Explanation :

Even though you can move the one paging file to a separate disk than the boot partition disk (because it avoids competition between the read and writes of the system files and the paging file), it is also effective to add a second paging file to the boot partition because the two paging files will share the activity and still give some performance relief for the boot partition.You can move the one paging file to a different disk than the boot partition.  In this case, since the boot partition is on the second disk, you would be moving the paging file to the first disk.

40、Your Windows 2000 Professional computer has three physical disks supporting a database application. Your recent purchase of these disks was based on optimal performance. Disk alignment is not an option for you at this time because of the time involved.. What other system setting can you configure to further improve system performance on this symmetric multiprocessing (SMP) computer?

1.  You can use file compression to improve I/O reads and writes.

2.  You can modify the registry to bypass the global I/O counters and the Task Manager I/O counters. <Correct>

3.  You can use the diskperf -yv command in the startup script.

4.  You can use the Split IO/sec counter to monitor for disk fragmentation.

Explanation :

The Task Manager updates the shared measurements of process I/O on an SMP system.  An application, such as a database that makes considerable use of process I/O and global I/O, will slow down the overall system with the Task Manager measurements.  To modify the registry, create the I/O System subkey if your registry doesn't have it so that the path looks like this: 

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\I/O System.  For the new CountOperations entry, the value should be set to 0 as a REG_DWORD type.

The diskperf -yv command enables the counters for logical drives.  Its use would increase the burden on the processors rather than lighten it.

 

41、You created a local user account for a temporary employee on her Windows 2000 Professional computer.  The computer has an account in the domain.  You are the computer's local administrator and you are a domain administrator. You want to increase security with Account Lockout Policy.  You notice that the effective settings are not changed after you set up the Account Policies and shut down the computer and start it up again.  The Local Setting is changed. What does this mean?

1.  You must be a member of the Domain Admins rather than the local computer administrator if you wanted to effect this account policy change.

2.  Before the policy settings take effect, you must shut down the computer and start it back up.  Effective settings will then match the Local settings.

3.  This computer is part of a domain, even though the user has a local account.  The effective settings show you the domain policy is overriding your settings for Local Computer Policy. <Correct>

4.  You should have set the Account Lockout Policy in the User Configuration, Security Settings, rather than in the Computer Configuration, Security Settings.

Explanation :

Account Policy is always set at domain level if the computer belongs to the domain.  You have attempted to set a policy in the Local Computer Policy.  The fact that you are working with a user who has a local user account is irrelevant.  The computer is affected by the domain settings. 

Having Domain Admins privileges while you are working on local computer settings doesn't change this fact.

43、You want to connect to the corporate Local Area Network (LAN) in Princeton, New Jersey.  You are accessing the LAN from a motel in Charlotte, North Carolina, using a Windows 2000 Professional laptop.  You do not have a connection preconfigured.  What do you need to do?

A.  Set up two connections:  a dial-up to your Internet Service Provider (ISP) and next, a Virtual Private Network (VPN) to the corporate LAN in Princeton. <Correct>

Explanation :

Since you do not have a valid Internet IP address when you are on the road, you'll need to connect to your Internet Service Provider (ISP) first.  That is the first connection you'll create in Network and Dial-Up Connections.  Next, you'll need to connect to the Remote Access Server (RAS) in Princeton.  That is the second connection you'll create.  You'll reference the first connection in the second connection setup routine.
