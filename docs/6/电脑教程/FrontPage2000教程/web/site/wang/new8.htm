第八课

动态效果处理集锦
大家都知道FP是一种很典型的静态网页设计工具，并且它自身的优点很大程度在网站的管理上，所以你想要使用FP来制作大型的动态网页那就错了。

不过鉴于设计上的需要，FP还是提供了不少的动态网页效果的处理，虽然一些已经变的很“古老”了，但对初学者来说也是很重要的。下面笨笨熊基本上总结了一下：如果有不足之处，还希望大家能来信指教。

1>最常用的鼠标滑动变色效果：

Format / style sheet，进入CSS样式表窗口

其中的默认格式是：<style><!---></style>

在Style标记对中间写：<style><!-a--></style>

点下面的 style按钮，在弹出的窗口中设定

Color前景色为 blue，OK。

在a{}标记后面写：a:hover 如：<style><!-a{}a:hover--></style>

同样点击 style 设定颜色为 ：red

这时最后的显示应该类似：<style><!--a{}a:hover{}-></style>

OK以后，在预览状态下鼠标滑动到连接上时，你能看到连接已经变成了红色。

看似非常简单的效果目前在网页可是应用最广泛的。

由于起到了强调的作用，因此无论是大网站（新浪网易搜狐）还是一般的小网站都应用了它们。

2>实现文本的动态效果。

选种格式文本。

Format / animation在随后的级联项中选择你喜欢的格式。

不过这种效果是在网页载入时显示出来的。

3>设定网页的切换过渡效果。

当然这种效果是针对整个网页而言的。

Format / Page transition

在左边的事件类型中选择：page enter（网页载入）page exit（网页关闭）或是 site enter （站点载入）site （站点关闭）。在右边的效果里，自己选选就可以了。

4>插入动态按钮：

将鼠标停在需要出入按钮的地方，insert /active elements/hover button，在窗口中 button color中设定按钮的颜色，effects：列表中选动态按钮的变化效果。

至于里面的效果，笨笨熊就不多说了。

在某些网页中我们经常会看到一些按一定方向（左右）滚动的信息，其实这些不是什么高深的技术。
