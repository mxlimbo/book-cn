第十课

实现FTP上传
网站作好了，当然要将它们放到服务器上面。你应该听说过关于FTP的东西，其实所谓的FTP（文件传输协议），是一种类似于HTTP的传输协议FTP服务器听起来很专业，也即我们申请个人主页时得到的个人空间。

当然那时是许多人同时使用一个或是几个服务器，比如网易100M空间的FTP服务器是：go.163.com。

现在对FTP服务器应该有个初步的认识了吧。

当然我们需要知道的还远不止这些。

下面笨笨熊简要的讲一下FTP工具。

由于FTP服务器的存在，相应的专业传输工具自然也就少不了了，目前比较流行的是CUTEFTP/BFTP/FTP control/网际传神等等。不过我们现在正在使用FP作网站，而本身FP也同时提供了上传功能，虽然在功能上显得单调一些，但是毕竟不用再花很长时间来独立学习什么FTP客户软件的使用。

好了，我想看了上面的部分你已经对上传有一定的理解了。

我们开始了：

1>在FrontPage explore 中 File / publish frontpage webs

2>在网站发布窗口中点击 more，填写网站发布以后的目录

如：http://www.51step.net/

当然你可以选全部发布或是只更新改动的网页。

3>OK以后，弹出FTP服务器地址及FTP的相应目录：

如：51step.net /home/u51step/www/

4>输入登录用户名及密码。

这时FP开始连接服务器完成上传。
