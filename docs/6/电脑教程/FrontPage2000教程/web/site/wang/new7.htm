第七课

文本操作及列表格式
通过上面的学习大家应该对网页已经有了一个初步的认识，作网页当然不能单单停留在功能上，至少要把自己的家园装扮的漂亮一些才好。

网页以文本为主，就先说说普通文本的格式了。

1>文本格式操作：

1）普通文本的格式：

选种需要格式的文本，右击选font properties

Fonts：设定文本的字体。

Font style：字体的样式，Italic（斜体）Bold（粗体）等等。

Size：字体的大小。一般的网页主题内容字体为2/3号字。

effexts：underline（下划线）strikethrough（中划线）

color：字体的颜色

如果还不能实现效果的话，你可以通过字体的CSS style 来实现，直接在字体属性里面，点击 style按钮，分别进行相应的设置。

2）连接文本的格式：

既然是网页，当然在其中连接文本占了大部分。所以对连接文本的格式自然是很重要的。

再者，由于通过网页属性设定的连接颜色使网页中所有连接都是一样的。当然有时我们可能需要某些连接使用别的颜色，这时 CSS实现的 style属性就显得特别重要了。

选种连接对象文本。比如：试验文字，右击选hyperlinks properties，进入连接属性窗口，点击 style按钮，选择 color 标签，设定前景色为新的连接颜色。

当然你还可以试者对其他的属性设置。

2>文本的列表格式（List）：

选种需要格式的文本，Format/bullets and numbering

1）使用自定义的图片作为列表项。

在列表格式窗口中选 image bullets标签，用右边的浏览钮找到使用的图片。

2）使用标准的列表项：plains bullets，FP提供了四种格式

3）数字列表项，使用数字来标定列表的顺序。Number要改变列表的其他属性，可以在列表的任意项上右击选择：List properties
