第五课

建立超级连接和书签连接
既然作网页，超级连接一定是很重要的了。如果你还不太懂的话，简单说就是通过鼠标点击打开目标窗口的东西，当然可以在当前窗口中覆盖进行刷新，也可以通过弹出新窗口的形式来打开。

1>目前在网页中用到的连接包括以下几种：

1>文本连接对象：是最常见的。

2>图片连接对象：只不过把连接对象改成了图片而已。

3>热点连接：通过图片的热点实现连接的。

4>电子邮件连接：点击以后实现发送邮件的功能。

5>书签连接对象：实现在页面内部的连接。

2>下面笨笨熊将细致的说明这几种连接的用法：

1>文本连接：对文本对象实现点击连接。

用鼠标选定需要作连接的文本。通过菜单Edit / hyperlinks或者直接使用快捷键：CTRL+K实现。

在弹出的窗口中选定连接的对象，注意一定要从网站的根目录起始，不能使用浏览按钮从硬盘作连接，那么就成了绝对连接，当用FTP传到服务器上，恐怕显示的仍然是打开：D：\file等。

在Target Frame 选项框中可以设定目标的打开方式：

_blank：在一个新开的窗口中打开网页目标。

_top：如果没用框架的话，效果同上面是一样的。

_self：在当前窗口打开。

_parent：在父窗口打开。

default：在本窗口中打开。

2>图片连接对象：

用图片作为连接对象。

同文本连接一样。CTRL+K建立，当然你还可以在图片属性中完成。鼠标选种图片，右击选Image properties，在Link框中输入URL地址或是通过浏览按钮选定目标文件。不过在这里有一个缺点，当我们用图片直接连接时，FP会在图片的周围加上边框，默认情况下很粗，所以很不美观的，为了统一和一致，我们可以通过设置在图片属性中用CSS的style来实现。

右击 Image properties，进入图片属性窗口，点击下面的 style按钮，在其中的Borders标签里，设定边框的颜色，边框形式和边框粗度。

3>热点连接：

用热点（Hot spot）为图片作连接也是经常用到的。鼠标选定图片对象，这时在编辑窗口的最下面会自动出现图片编辑工具。点击热点工具（长方形/原形/多边形等），在图片上绘制热点范围，完成后弹出连接窗口。下面的就一样了。

4>电子邮件连接：

在网页中你或许经常看到电子邮件连接。

点击以后，浏览器会自动调用邮件客户端软件例如：FOXMAIL，OUTLOOK。其实是很容易实现的。在连接窗口中的URL地址不在使用HTTP，而变成了 Mailto:zeng.zeng@263.net。如果你还想标定一下邮件的主题，或是邮件的抄送和暗送对象可以使用下面的语法mailto:zeng.zeng@263.net?subject=友情连接&cc=51step@163.com&bcc=51step@163.com

其中的CC是抄送对象，BCC是邮件的暗送对象。只要换成相应的邮件地址即可。

5>书签连接：

除了上面的连接以外，还有一种在页面内部实现的跳转连接。这种连接主要适用于比较长的网页作内容上的导航。当然你也可以直接用它连接到某页的指定的位置。

实现的方法是：

1）先建立目标书签

选定需要连接到的文本。

edit/bookmark，输入书签的名字。OK这时文本会用下划曲线标出来代表不同于其他文本。

2>作连接

CTRL+K，进入连接窗口，如果你是在当前页面内作连接的话，直接在Bookmark下拉菜单中选择对应的书签名即可。如果是连接到指定页的指定地方，可以先选择目标网页，这时该页的所有书签也都自动导入到Bookmark里。或者你干脆自己来写：http://www.51step.net/index.html#new

其中#是用来提示书签的起始位置的，当然不能丢了。

好了，现在你已经掌握了所有的连接形式。
