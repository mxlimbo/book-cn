Chapter 10

The WFC Application Package

There is one Microsoft Windows Foundation Classes for Java (WFC) package we have yet to examine: the Application package com.ms.wfc.app. This package consists of an assortment of classes that at first appear to be unrelated. This chapter concentrates on time-related classes, the multithreading classes, and the Registry class. You'll find the code for the sample files in this chapter in the Windows Applications subfolder on the companion CD.
