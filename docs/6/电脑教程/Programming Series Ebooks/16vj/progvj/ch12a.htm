Chapter 12

Accessing Databases

Chapter 3, The WFC Utilities Package, described the Windows Foundation Classes for Java (WFC) Input/Output io package, which allows Microsoft Visual J++ programs to read from files, write to files, and copy files. One task not described in that chapter—one that Visual J++ 6 programmers will invariably find themselves faced with—is the accessing of data from a database. This chapter deals with that task.
