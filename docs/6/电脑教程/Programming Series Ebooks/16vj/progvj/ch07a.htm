Chapter 7

Menus and Dialog Boxes

The RTFEditor application in Chapter 6 demonstrated how to graft a couple of buttons and an edit box onto a rich edit control to build a fairly functional Rich Text Format (RTF) text editor. However, RTFEditor was limited in what it could do because it was missing two fundamental weapons in the user interface arsenal: menus and dialog boxes. In this chapter, we'll start over and build an editor with the standard menu across the top, a toolbar just underneath the menu, and dialog boxes that are displayed on demand. We'll even add a context-sensitive menu to the editor. (Of course, this editor application won't have all the features that you would expect from a commercial editor. As an exercise, you can add features to the editor later if you want.) All the examples in this chapter are located in the Windows Applications folder on the companion CD.
