Chapter 3

The WFC Utilities Package

The WFC utility package com.ms.wfc.util consists of a collection of classes that make your console applications easier to write and more trouble-free. In this chapter, I'll present some of the more useful classes contained in the utility package.

We'll begin with the Debug class, taking some time to discuss how this class integrates with the Microsoft Visual J++ 6 debugger. From there, we'll go through a series of collection classes, starting with the arrays class. All the code discussed in this chapter is located in the Windows Applications folder on the companion CD.
