Chapter 20

Creating Transactional Components with Visual Basic 6.0

In Chapter 19, we learned about working with transactions within ASP Web pages. In this chapter, we'll see how to work with transactions within Microsoft Transaction Server (MTS) components that have been called from ASP Web pages. We'll use Microsoft Visual Basic 6.0 to develop these MTS components. We'll also take a quick look at transactional C/C++ components.
