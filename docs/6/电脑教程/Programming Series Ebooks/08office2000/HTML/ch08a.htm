Chapter 8

Timesheet System

In Chapter 6, I showed you how to use the Chart component to build dynamic chart-based reporting solutions. In Chapter 7, I showed you how to use both the PivotTable component and the Chart component to build rich, interactive data analysis and reporting systems. In this chapter, I will show you how to combine three components—Chart, PivotTable, and Spreadsheet—to build a common business system: one that tracks the time that employees spend working on their projects.
