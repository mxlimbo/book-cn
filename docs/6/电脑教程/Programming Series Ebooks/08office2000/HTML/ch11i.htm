Summary

This chapter showed you how to build a custom function add-in for the Spreadsheet component. I discussed how to implement simple functions, functions that take ranges for input arguments, and functions that access external resources and have their own function-editing user interfaces. Using these techniques, you can build powerful function add-in libraries for the Spreadsheet component, providing your users with your own advanced analytics.

This is the last chapter that describes a complete solution involving the Office Web Components. Chapter 12, the final chapter of the book, will describe how to deploy solutions using the components, including how to make the components automatically download and install in a web-based application.
