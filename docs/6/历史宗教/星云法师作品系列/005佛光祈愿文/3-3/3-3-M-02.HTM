慈悲伟大的佛陀！
今天是岁残年尽，
我们的人生又过去了一年。
这一年来，
在我的生活里，
尝到许多的酸甜苦辣；
在我的遭遇里，
看到许多的人情冷暖。
慈悲伟大的佛陀！
所幸仰仗您的庇护，
让我生起勇气面对现实，
一次又一次渡过险难；
所幸承蒙您的教诲，
让我提醒自己不要失望，
平安走到一年的尽头。
慈悲伟大的佛陀！
有时我会怀疑：
人生的意义究竟在那里呢？
有时我会思索：
人生为什么有那么多的苦难？
有时我也会难过：
我对人真心，
竟然遭来别人对我的假意；
有时我也会不平：
给人帮助，
反而被人怨怪被人歪曲。
每当想起佛陀您的开示，
我就悔悟这是自己的业障深重，
所以我就会提醒自己
应该用责人之心责己；
我就念及是自己的诚意不够，
因此我就会要求自己
应该用恕己之心恕人。
慈悲伟大的佛陀！
感谢您的至理名言，
让我消除怨恨之气，
重新拾回欢喜自在。
佛陀！
这一年来，
我每日缠绕在执着厚茧中无法开脱，
我经常陷身于无名漩涡里不能自拔。
虽有师长教诲，
无奈讲时似悟，对境生迷；
虽有善友提携，
总是忠言逆耳，错失良机。
佛陀！现在
当愚痴的邪风吹来的时候，
我知道要抱紧佛法真理的磐石；
当嗔怒的烈火炽盛的时候，
我懂得应泼洒柔和忍耐的法水；
当贪欲的洪流高涨的时候，
我明白需开启喜舍布施的闸门；
当骄慢的高山隆起的时候，
我会得去运用谦虚尊重的巨铲。
佛陀！伟大的佛陀！
我要用三无漏学对治三毒的弊害，
我要用五停心观消除烦恼的阴霾，
我要用八正道分抹去邪见的污垢，
我要用六波罗蜜渡过人生的苦海。
慈悲伟大的佛陀！
今天一年又过去了，
往昔所造的恶业，
我要在您的法座前能够深深忏悔；
过去一切的恶行，
希望在您的加持下能够改往修来。
仰仗您的威德，
让我的陋习随着岁月而流逝；
仰仗您的威德，
让我的善心在忏悔之中萌芽；
仰仗您的威德，
让我的智慧在惭愧里面生根。
慈悲伟大的佛陀！
炮竹声中辞旧岁，
万象桃换将更新。
一生罪业污秽的我，
至诚地跪在您的座前，
祈求您给我新生，
祈求您给我再起。
慈悲伟大的佛陀！
祈求您满足我衷心的愿望，
慈悲伟大的佛陀！
请您赐给我美好的一年。
请您赐给我美好的一年。
