为孕妇祈愿文　　　　　　　　回目录

慈悲伟大的佛陀！
弟子怀着祝福的心向您祈愿，
祈愿即将作母亲的她们，
都能获得佛陀您的加被。
因为，
她们现在虽有喜悦，但不安宁；
她们现在虽有快乐，但有牵挂。
佛陀！
一个即将作母亲的孕妇，
她们有患得患失的心情，
她们有生男生女的挂碍。
这些复杂的感觉，
都希望佛陀您能施给她们无畏的勇气，
赐予她们无惧的信心。
孕妇，
这些即将作伟大母亲的人，
她们为人类绵延生命的河流，
她们为世界带来无尽的希望，
她们忍耐十月怀胎的茹苦含辛，
她们忍耐多少晨昏的身心不适，
她们用青春岁月换来新生命的形成，
她们用血液辛苦浇灌新生命的长大。
甚至于她们忘记为自己的容貌妆扮，
只为了殷殷期盼新生命的到来；
甚至于她们忘记了自己的幸福时光，
只为了苦苦等待小儿女的到来。
慈悲伟大的佛陀！
即将降诞人间的幼儿，
需要慈母的胎教，
所以祈求您，赐给这些待产的母亲们
慈和柔软的气质，安详沉静的个性，
明理通达的美德，自在安然的心情。
希望这些年轻的母亲们，
芙蓉的面孔永远绽放微笑，
如花的玉貌永远释出安详。
更希望这些准妈妈们，
能用慈眼等视一切人，
能用善言谈论世间事。
慈悲伟大的佛陀！
让他们亲子的心能够，
亲连子，子连亲，
亲子之间，心心相连；
让他们母子的缘能够，
母连子，子连母，
母子之间，善缘相续。
慈悲伟大的佛陀！
请您赐予力量、勇气、信心，
给这些将为人母的妇女，
让她们减少身体的痛苦，
让她们远离身心的压力，
让她们拥有亲友的照顾，
让她们得到生产的平安。
慈悲伟大的佛陀！
祈求您让他们所生的男孩，又壮又胖又乖巧；
祈求您让他们所生的女儿，又白又美又端庄。
更祈求您，伟大的佛陀！
让她们的新生儿，
将来都能感受母亲如山高般的恩情，
将来都能体会母亲如海深般的爱心，
将来都能领略母亲如春晖般的美德，
将来都能感谢母亲如秋月般的圆净。
更祈求佛陀您加持这些孩子，
将来能继承母亲的信仰，
将来能担当家族的传灯，
将来能显扬前辈的庭训，
将来能作为后代的模范。
慈悲伟大的佛陀！
祈求您让天下的慈母和儿女，
在您的庇佑之下，都能够
身心轻安，生活正常，
日夜安全，一切如意。
慈悲伟大的佛陀！
祈求您让世间的慈母和孕妇
，在您的加被之下，都能够
慈心爱物，结缘布施，
聪明智慧，祥和欢喜。
慈悲伟大的佛陀！
容我再一次
万千哀愍的祈求您，
让天下的孕妇，临盆的时候无痛无恼，
让天下的慈母，生产的时候无惧无畏，
让新生儿，平安地降诞人间，
让好宝宝，快乐地与世见面。
慈悲伟大的佛陀！
这是人间美好的善缘，
这是家庭幸福的聚会，
慈悲伟大的佛陀！
请您降福给孕妇和胎儿吧！
慈悲伟大的佛陀！
请您接受我诚恳的祈愿，
请您接受我诚恳的祈愿。
