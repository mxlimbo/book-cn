慈悲伟大的佛陀！
我们弟子
祈求您万分地垂怜世间的儿童，
祈求您万分地加护世间的孩儿，
佛陀！您可看到世间上每一个角落，
有多少不幸的儿童，
他们受恶毒的鞭打，
他们受暴力的残害，
他们在羞辱中无能脱困，
他们在无助下暗自哭泣，
慈悲伟大的佛陀！
儿童是国家未来的主人，
儿童是人间未来的希望。
星星小火可以燎原，
年幼的儿童安能不会为人类造福呢？
一滴雨水都能滋润万物，
可爱的孩子为什么不能普利天下的苍生呢？
伟大的佛陀！
希望世间上的人
对孩子都能视若珍珠宝玉，
希望世间上的人
对孩子都能赐予慈颜爱语，
希望世间上的人
对孩子都能倍加爱护教导，
希望世间上的人
对孩子都能尊重鼓励帮助。
我们要发挥
幼吾幼以及人之幼的精神，
我们要怀抱
亲吾亲以及人之亲的胸襟，
给予孩子们的生活要无忧无虑，
给予孩子们的身体要无病无痛，
给予孩子们的心理要无烦无恼，
给予孩子们的爱心要无微不至，
让所有天下的儿女都能在美好的环境中成长，
希望天下的儿童能有快乐的家庭，
希望天下的小孩都有慈善的父母、亲人，
慈悲伟大的佛陀！祈求您让每一个孩子
穷苦的，都能获得安乐；
饥饿的，都能获得保暖；
无依的，都能获得关怀；
苦难的，都能获得解救。
让每一个孩子的童年，
充满了天空中灿烂的星星
让每一个孩子的童年，
洋溢着园圃里花朵的芬芳。
慈悲伟大的佛陀！
天下所有的儿童幼小的心灵里，
都希望有一个无泪的睡眠；
天下所有的孩子无邪的思想里，
都希望有一个尊严的童年。
慈悲伟大的佛陀！
希望大人们的粗暴严厉
能改为慈悲柔和，
希望大人们的无明责怪
能变为光风霁月。
让世间上所有的孩子们，
都能徜徉在青山绿水的怀抱中，
尽情地玩乐，
广泛地学习。
让世间上所有的孩子们，
都能安住在光明喜悦的世界里，
父母是他们的朋友，
老师是他们的亲人。
让每一个儿童都能像善财童子，
有五十三参的机会；
让每一个儿童都能像妙慧童女，
能表现伟大的智慧；
让每一个儿童都能像均头沙弥，
能得到长老的慈爱；
让每一个儿童都能像罗喉罗，
能萌发向上的信心。
慈悲伟大的佛陀，
让您能接受我们至诚的祈愿！
让您能接受我们至诚的祈愿！
