规劝青少年祈愿文　　　　回目录

　　慈悲伟大的佛陀！
您可知道，
    我们现在的社会多么混乱啊！
    我们现在的人心多么险恶啊！
    我们几乎每天都在恐惧中生活！
    我们几乎时时都在苦难里挣扎！
多少的儿童只知道顽皮嬉戏，
    不知道读书上进；
多少的青年只知道任性玩乐，
    不知道忠孝仁爱。
一些在外游荡的孩子们
    能欺则欺，能骗则骗，
    能偷则偷，能抢则抢；
一些课业不佳的学童们
    自暴自弃，自我毁伤，
    自怨自哀，自我践踏。
他们颠倒人生观只想自己，
    却不为别人着想；
他们错乱价值观短视近利，
    以致于贻害社稷。
他们的行为
    让有为有守的青年们耻于为伍，
    让勤学上进的青年们感到蒙羞。
多少的父母为子女而伤心难过，
多少的老师为学生而灰心失望，
多少的仁人看到世情不正而唏嘘伤感，
多少的义士看到社会乱相而徒叹奈何。
伟大的佛陀！
祈求您慈悲垂慈，
    让我们的社会能够安定，
    让我们的人心获得净化，
您所教诲我们的慈悲爱心，
    如果人人都能具备，
    到处无缘大慈，同体大悲；
    那将是多么美妙的社会！
您所指示我们的忍耐柔和，
    如果大家都能奉行，
    从生忍、法忍到无生法忍，
    那将是多么安全的人间！
您所鼓励我们的觉悟正道，
    如果大家都能体证，
    从自觉、觉他到觉行圆满，
    那将是多么清净的国土！
您所明示我们的般若智慧，
    如果大家都能拥有，
    从闻、思、修进入三摩地，
    那将是多么圆满的世界！
慈悲伟大的佛陀！
祈求您：
    让他们早日将丑陋的恶习抛除，
    完成健全的人格；
    让他们早日从愚痴迷梦里觉醒，
    探求生命的真谛；
    让他们早日从向外贪求中回头，
    找到内心的宝藏；
    让他们早日认识自我真如本性，
    享受无尽的喜悦。
慈悲伟大的佛陀！
    请您让迷失的青少年们早日回头，
    请您让堕落的青少年们早日改心，
    请您让冥顽的青少年们早日换性，
    请您让偏执的青少年们早日转身。
慈悲伟大的佛陀！
多少的父母为子女的生活而含辛茹苦，
多少的师长为学生的教育而夙夜匪懈，
多么希望佛陀您能
让天下的子女们都懂得世间上
    一丝一缕，物力唯艰；
    一粥一饭，来处不易。
    让他们产生感恩的心念，
    让他们培养惭愧的美德。
多么祈盼佛陀您能让天下的学子们都懂得：
    他们在享受国家社会的成果，
    他们在享受父母亲人的余荫。
但是，
    他们贡献给社会国家的是什么呢？
    他们回报给父母师长的是什么呢？
慈悲伟大的佛陀！
    让懵懂无知的人早日回头吧！
    让冥顽不灵的人早日觉悟吧！
慈悲伟大的佛陀！
    请您接受我至诚的祈愿。
慈悲伟大的佛陀！
请您接受我至诚的祈愿。
