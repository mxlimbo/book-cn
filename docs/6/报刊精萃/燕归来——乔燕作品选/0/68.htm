如何应对性骚扰？

　　近年来，性骚扰成了一个热门话题，越来越多的人开始自然而然地谈论它--这在几年前还对"性"字闭口不言的国人来说，无疑是一种观念上的进步。可是，人们越来越多地在公共场所遭遇到各种隐蔽的性骚扰给许多人带来了难言的困惑、烦恼和心理压力。那么，什么是性骚扰呢？

　　顾名思议，性骚扰是对别人进行与性有关的骚扰，它显然不是出自对方的需要和愿望，而是不得不被动面对的尴尬。它包括包含性内容的行为、举止、语言等，只要不是对方明确表示需要的，并且已引起别人的紧张、不安、难堪、尴尬等情绪及心理反映的，都属于性骚扰。

　　许多人以为性骚扰主要是男性针对女性发生的，其实不然，有相当比例的性骚扰由女性发起，比如在办公室穿着过于暴露的衣服而给男同事带来紧张、妨碍及被动性联想和不情愿的生理反应的，也属于性骚扰。当然若男同事不反感，反倒很欢迎，则不属此列。性骚扰也可在同性间发生，比如有"性兴趣障碍"的人将"性兴趣"指向没有"性兴趣障碍"的同性，或者在饭桌上、办公室里，有人讲的荤笑话给在场的某人（包括同性与异性）带来不快、不适等。

　　由于许多性骚扰具有隐蔽性，一些不良感觉只有当事的人能感觉到，因而性骚扰的取证和如何判定有一定难度，人们对它的认定标准也并不完全统一。

　　近年来的一些调查资料，以及我在热线工作中的体会表明，性骚扰发生在办公室、发生于同事、上下级之间的比例很大，有些人由于顾忌到面子、自身的发展与利益而忍气吞声。这对一个人的心理健康是相当不利的。因而如何应对时刻有可能出现的性骚扰、及时调整心态、避免因此带来的心理障碍和伤害是自我发展的重要一环，也是现代人必修的新课题。

　　既然性骚扰很有可能在自已身上发生，那么如何应对性骚扰呢？

　　首先要明确性骚扰是别人对自已的侵犯，尽管这种骚扰有时没有实质的行为。既然是对自已的侵犯，我们当然有足够的理由来有效地保卫自已，如义正辞严地警告、批评等。这不是我们的错因而不要责备自已，否则就是用别人的错误来惩罚自已了。

　　其次，对性骚扰者要勇敢地说"不"，明确的拒绝可以给对方强有力的警示：我不是那种能让你为所欲为的人，你必须尊重我！千万不要采取容忍退避、模棱两可的态度，那会使对方更加得寸进尺。许多人受到伤害甚至人身攻击正是由于这种软弱的态度给了对方鼓励。

　　第三，要尽可能保留证据，以控制对方，如把他写给你的便条、送的淫秽画片或书刊、录像留下来，这些都是证据。对于来自上级的性骚扰，此类证据尤为重要，必要的时候，它们就成为保护自己人格、尊严、工作、利益的重要砝码。

　　将自已的处境告诉可以帮助你的家人和值得信赖的朋友，求得帮助和支持。这帮助和支持既会是方法、方式上的，也是情感与心理上的，因为这种情况下人的心理压力是可想而知的。

　　必要时向上级或有关部门反映，如单位的妇女委员会、工会的女工委员会等。

　　在进行以上的自我保护的同时，要积极地进行心理咨询和心理治疗，尤其是在心理有了压力、自尊心与自信心降低、内心一片混乱的时候。及时的心理咨询可以使自已免受不应有的伤害，提升自尊与自信。
