花与婚姻

　　现如今，"婚姻脆弱"的说法很普遍，无论是在朋友聚会的餐桌旁，亦或是在电台的热线节目中，此言会不时灌入耳鼓。那一日，便听闻一新结识的朋友如是说："为什么要结婚？结婚是为了什么？当人在恋爱的时候总是习惯于把爱人客体化（理想化），这绝对是个自我陶醉的过程。而当两个人真正在一起时，幻想逐渐破灭，真正的悲剧就形成了。人世间可以称得上悲剧的无外乎两种情况：欲望得不到满足，于是便痛苦；欲望得到满足，于是便厌倦。"

　　我曾把婚姻比作花。

　　花是有生命的事物，可萌芽、生长、吐出芬芳，但也会在生命的尽头枯萎、死亡。花自有爱花人日夜牵挂，既使是山野之花也有大自然规律的疼爱——土壤、阳光、雨露、清风。

　　婚姻也是有生命的，从相识到爱恋、再到建立两个人的家庭，然后分出两个方向：或者恩爱白首，或者风吹烟散。

　　看看我们周围，哪桩婚姻令人羡慕、哪桩婚姻已奄奄一息，便可知道婚姻的命运与婚姻中的人有何关联！

　　正如那位朋友所言，恋爱时很多人耽于理想、幻想、保留了过高的期望、没有做好接受平淡的婚姻生活的准备，等现实逼人醒来才倍感失落，抱怨、指责、懊丧、把责任推为婚姻的不可信与脆弱；而那些携手人生、共历风雨的伴侣却从不把矛头指向对方，而是最大限度地彼此适应、互听心声、使两颗心越帖越近。
前一种人只知热切地期望花开却未用心养护，而后者则怀不苛求花香花色之心只细细培植，因而前者易因花淡朵小而失望、后者常因花鲜色艳而感恩……

　　花儿能感觉干、湿、冷、暖，固有其脆弱的一面，但爱花人因花的脆弱而放弃种植者决不是真的爱花；婚姻在满足人的生理、心理、伦理、社会生活等需要的同时最大限度地保护了人的隐私，它作为一种特殊的人际关系，本身并无知觉，我们有何权利避开人的因素而去感叹婚姻呢？

　　正因为美满婚姻不易得，才更需抱有理性婚姻观念、怀着对生活的信任与热情、完善自已、了解别人。认真地寻求，求得后在恋爱中相知，统一对婚姻的认识，婚后不断校正座标，共同探索情感的真谛。

　　记得高尔基在他的《人》中反复重复一句话："一切为了人！一切在于人！"婚姻的前途不正在于人本身吗？

　　让我们以爱花之心去爱护婚姻吧，用感叹婚姻的精力用来反躬自省！
