一条路上的阳光

也许会在路的尽头遭遇慈祥
于是安然走过山岗
走过一片褐色的微笑
走过不能预想的荒凉

两座山峰不是真正的高度
山谷里暗藏着空旷的灵魂
进入只是一场虚惊的开始
别惧怕尘土之外会有另一种死亡

到底能有多远
一条路上的阳光
一个方向上的凝望
老人的回忆深过少女的幻想
城市的凌晨比乡村的夜晚还黑

到底离慈祥还有多远
一条路上的阳光正被眼泪侵犯
世上只剩下未来不忍心把人欺骗
心灵的宫殿该在路的尽头出现

　1999年3月5日
