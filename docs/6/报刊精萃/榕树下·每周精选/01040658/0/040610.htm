酒状

○潘乙宁(海南)
　　　　　　 

    酒状不是喝酒渴得很疯的那种人。酒状酿的酒谁喝了都说“有
劲。”早上，小墟集市了，酒状把铺面的门板一块块卸下来放好，
搬出一张大桌子和几张小凳子小椅子，酒坛就放在大桌上，桌上还有
装着炒花生或花生饼的玻璃瓶。几张小桌小椅子供喝酒人用的。酒
状的酒酿得好，酒色有些淡黄，揭开酒坛的盖子，你不会立刻可以闻
到那酒的芳醇，你只是慢慢的才能闻到，好像香又好像并不浓，甚至
还有点薄荷的感觉，但确实又不是，你只是感到一些凉凉的气味钻入
你的鼻子，然后才在你的肌肤中慢慢渗开。
    你是不懂喝酒也想喝了。酒状酿酒是祖传的，据说明末到现在，
几代了？他也说不清。人们都说，他的酒之所以酿得好，是因为他家
里那口井的水好。那口井我小时候见过，井口很小，水很清，清得有
点碧绿，透彻之中却似藏着什么一般，一看，就让人觉得凉快。酒
状的酒酿得好，喝酒的人就多，都是些熟客。那些渔民早上上海后，
把鱼虾交给妻儿拿去卖了，洗个澡，多数到酒状的店里，要上一杯酒
几颗炒花生，便一边无边无际地说些闲话一边饮酒了。酒状话很少，
他是把整个身子躺在大大的背椅里，微闭着眼睛，不知道是听别人讲
话还是闻酒的醇味。酒状的身后有两块门板，刻着一副对联：“一
目清泉酿美酒，香；半杯浊酒醉千人，妙。”据说是几代之前县里一
位有名气的人题的。我记得，每年大年三十，酒状总是用红漆重新临
摹那幅对联，很虔诚也很神圣的样子。小时候，我时常去替父亲买
酒，酒状总是多舀一些，还摸着我的头慈祥地问：“今天老师教什么
书？”我讲了，他又叫我背书。我背出来，他便从玻璃瓶里抓几
颗花生给我。有时，我背不出来，他便说快回去背书，仍是抓几颗花
生给我。有一次，他叫我读他身后的门板上那幅对联，我读了一遍，
他又重复一遍，不像是读，像唱，像是在唱琼剧一般。酒状酿酒从
不给外人看，除了他妻子之外。他妻子在结婚的第三年便死了，他也
没有再娶，别人说，酒状，再娶一个吧。他说好的好的，可一直没有。
我想他可能把酒当成妻子——那自然没有再娶妻的念头了。他总是
夜里才酿酒的。有好事者也常偷偷去看，但也没有看到真正的酿酒方
法。只看到，他酿酒前，先点几柱香，拜了酒神（那酒神我见过，其
实只是一个小木匾，里面贴着大红大红的纸，写着“酒神正位”，两
旁贴着一幅小对联，好像是这样的：“酒能怡情，七情六欲缘酒生；
酒亦治病，大病小疾随酒去。”）。拜过酒神，又拜井神，然后拐进
厨房，关起门，他便是在那里酿酒的。过了个把钟头，在外面偷看
的人便可闻到酒的醇味，那味道渗在空气中，人们禁不住要尽力地呼
吸，想把那香味都吸给自己，人们都说，吸酒气比饮酒还要醉人。
于是，夜夜有人去吸酒气。酒状从不喝酒，有人问：酒状，你酿的
酒那么好，为什么从没喝一口？他回答说，每夜我都吸了很多酒汽，
那都是酒的精华。那倒是，酒的精华他都吸了，喝酒还有何味？
但他又确确实实酿出一坛坛的好酒，给我们的父老乡亲增添许多怡情
治病的好处。酒状除了酿酒，就喜欢捧着一本砖头厚的书来看，那
书是用宣纸装订起来的，上面手抄着一些古诗，也不知道这“书”已
是牛年马日的了，他看得也很有滋味，微闭着眼，似是睡了，嘴里却
念念有词。有时候，他偶然也铺几张纸在那卖酒的桌上，写上几个
“福”字“寿”字。但即便这两项爱好，其实也只是偶而为之的。
酒状对我很好，从小我就有个印象，每次买酒或从旁边经过，他总是
不厌其烦地问我读几年级了，或者说背首诗给爷爷听。我不知道他是
不是觉得我可以读点书。但我记得，我小时候是很顽皮的，一点都没
有能读书的样子。我考上大学，酒状也很高兴，他轻摸着我的头，
连声说这下子好啦这下子好啦。最后，他用瓶子装一瓶酒，递给我
说，拿回去给你爸庆贺一下。其时，我父亲已戒酒多时，但我还是
接过那瓶酒，父亲后来也没有喝，直到酒状死了，才在他的灵前祭他。
酒状死的那夜跟往常没有两样，夜风很平和，我们正坐在屋子外面
等待彩票开奖的消息，不久，便听说酒状死了。他死在灶旁，死时，
酒气正往外冒，芳醇飘浮在故乡的上空。我不知道人们是怎样发现
他死的，据说那夜他酿酒第一次没有关房门。我听到他死的消息时，
好像没有特别的感觉，脑中只崩出一个念头：以后那些酒民去哪找他
的酒来喝？直到家人和左邻右舍都跑去酒状家只剩下我一人面对着
黑乎乎的街道和房子时，尤其我后来再走进酒状的家时，我才真正感
到，酒状死了。他怎么舍得死呢？怎么舍得离开他的酒呢？
    出殡那天，他的酒民都来了，他们把酒坛里剩下的酒一杯一杯地
祭在他的灵前，他生前没有喝过一口酒，死后，就给他一回豪饮吧。
我斗胆做一挽联，敬献于他灵前：
 
   生不求富死不求贵但求一斗糟米一桶清水酿出半杯浊酒醉他千人 
   哭已无声唤亦无言唯愿一斛美酒一卷古诗再造平常生活怡你来生 
 
　　酒状，我不知道这幅对子是否合你意：来世，再做酒状。
