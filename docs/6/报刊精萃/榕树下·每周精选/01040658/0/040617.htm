五月的少年

○商略(浙江)
　　　　　　 

五月的少年打马而去 
越过黑暗穿过诗歌 
沿着通向天际和云朵的道路 
沿着一路向日葵的歌唱 
沿着一路仆仆风尘 
给我留下 
一个黑蝙蝠翅扑不已的 
易碎的暗夜 
 
五月的少年打马而去 
风雨来时 
我的肋骨隐隐作痛 
黑夜会是一枝破空的箭 
把我钉在单薄而冰冷的苍穹 
我听见泥土下面 
有人叹息。有人腐烂 
有人窥视。或静观 
 
五月的少年打马而去 
踩着月色和一夜露水 
让马蹄敲打着寂寞的星空 
风和阳光被甩在了身后 
我十年前的影子 
杀了那个灰暗的春天之后 
你将给我带回怎样的好消息
