谒陈子昂读书台

○ 陈义怀

　　在好几回擦肩而过之后，一个秋天的下午，我终于来到了位于射
洪县金华山的陈子昂读书台。不过距离最初的那次遥望，中间已横隔
了十几年的岁月。那时候，我在心里吟诵着“前不见古人，后不见来
者，念天地之悠悠，独怆然而涕下”的诗句，胸中涌溢着宇宙人生的
感叹，我不知道写下这首诗的人会有着怎样的人生。这个问题一直象
谜一样吸引着我，而今天我终于有机会来寻觅答案了。  
　　汽车在一处人迹稀少的小镇上停下来，我问路边的人：读书台在
什么地方？一个正在打麻将的女人朝背后努了努嘴。我顺着她指的方
向望去，那是一片长势不错的山林，秋阳下隐隐露出一角屋脊。经过
一座小木桥，爬过几十级台阶，一条小路从林间浮现出来。林子里很
幽静，参差的树影间落下点点光斑。微风拂过，光影摇曳，沙沙的声
音一路响过来又远去，仿佛是一千多年前的那个气宇轩昂的年青人正
在林中漫步。那时候，大唐王朝的权柄掌握在一个名叫武则天的女人
手中。关于她，尽管历史上有许多污秽的传说，但至少在政治上她还
算一个明君，除此之外，我们不能要求更多。而这个正在世外桃园般
的林子里读书的年青人也许不会想到，他一生的起落沉伏与这个女人
有着千丝万缕的联系。然而，谁也无法穿透岁月的迷雾看清生命的轨
迹。此时的陈子昂风华正茂，他满腹诗书，一腔抱负，如同剑在匣中，
待时而出。这和那个时代的许多年青人一样，他们胸中洋溢着一股生
命的激情，希望佐明君，匡江山，扶社稷，建不朽之功业，留万古之
盛名。你只要看看几十年后李太白写下的诗句就会明白：仰天大笑出
门去，我辈岂是篷蒿人。这一年太白已是四十多岁了，他以为腾飞的
时日终于到来，不禁欣喜若狂，其实唐玄宗早就给他定了位：御用文
人。陈子昂的命运也不过如此，他的一支笔虽然可以扫尽六朝的颓糜
之风，但在武氏王朝的棋盘上，他仍然只不过是一颗无足轻重的棋子
而已。而子昂的悲剧在于，他始终没有认识到这一点，所以，他只能
在蓟北的行营中悲愤地写下那首《登幽州台歌》，留下屈子投江般的
千古绝唱。当然这是后话，这与那个正在金华山上饱读诗书的翩翩少
年还隔着几十年的人生岁月，他还可以尽情地规划自己的人生。  
　　林间小路一直延伸到金华山的最高处，其上有一个呈倒葫芦形的
两进的小院落，这便是子昂少时读书的地方了。院中多古柏，皆右旋
而上，想当年它们大概是听过子昂的朗朗书声的吧。院子大概很久没
有修茸过了，显得有些破旧，陈列也很简单，只在四壁的墙上有一些
介绍陈子昂生平的壁画和文字，一眼就能看出是很有些年头的东西了。
唯一值得一看的是院子正中那块刻着《与东方左史虬修竹篇并序》的
石牌，正是在这篇奠定了陈子昂文学史地位的诗文中，他向六朝浮华
奢靡的文风发起了挑战，文章一开始就说：东方公足下，文章道弊五
百年矣！其声震聋发聩，千年而下余音不绝。跨过一道圆形的拱门，
来到后面的小院。院子里越发显得清静，只有几只鸟雀在草丛中跳跃
啄食，见有生人光临，一闪身便飞进林子里去了。更寂寞的是那块有
名的“臭石头”，他一直在这里陪着子昂的英魂。相传这块石头是将
子昂迫害至死的当时射洪县令段简所变，只要一敲击石头就会散发出
一股难闻的臭气来。其实，这只是后人为了逞恶扬善的心理慰籍而已，
就象岳飞庙前跪着的秦桧。实际上，在历史上那场善与恶正与邪的真
实较量中，邪恶却最终是占了上风。有时候，正义与善良实在是很脆
弱的东西。子昂一生少年得志，壮年坎坷，末了却落入小人之手。堂
堂一代文宗，最终竟死在区区一个县令手里，想来实在让人感慨万千。
关于段简罗列罪名捕杀子昂的原因，历来有不同的说法。一说是因为
段简贪财，据《新唐书》陈子昂本传里记载：县令段简贪暴，闻其富，
欲害子昂。家人纳钱二十万，简薄其贿，捕送狱中。此说颇多疑点，
段简区区一个县令何以能对陈子昂横加迫害？当时子昂虽然致仕归隐，
但他曾经是武则天身边的红人，并且解官以后还享受着右拾遗的待遇。
二说是谓段简乃受武三思的指使害死陈子昂的。唐人沈亚之在《上九
江郑使群书》中说：陈死于枉，皆由武三思嫉怒于一时之情，致力害。
此说似乎不符合通常的政治斗争的逻辑，在历史上武三思与陈子昂既
没有公怨也没有私仇。作为一个官场老手，武三思是不会出此下策对
一个已经归隐的旧臣下手而授人以柄的。第三种说法是段简是以“附
会文法”的罪名坑害子昂的。唐代严禁私修国史，而子昂归隐后却私
自撰写《后史记》，这于是成为段简罗织罪名的原由。然而段简迫害
子昂的动机何在？况且，人书俱在，岂能容段简一手遮天？因为段简
面对的不是一个普通人而是名满天下曾经做过右拾遗的陈子昂。其实，
关于陈子昂的死，到是他自己在诗中透露了一些消息。在《陈子昂集》
中有一首题为《宴胡楚真禁所》的诗云：人生固有命，天道信无言。
青蝇一相点，白璧遂成冤。清室闲逾邃，幽庭春未暄。寄谢韩安国，
何惊狱吏尊。这首诗写于作者被囚禁的监狱，时间大约是在公元700
年的冬春之际，也就在这年的晚些时候，陈子昂被害，年仅四十三岁。
从诗中看，陈子昂是知道自己必死无疑的，至于为什么会遭此恶运，
他隐约说道：青蝇一相点，白璧遂成冤。白璧当然是作者自喻，但青
蝇是谁？是段简还是有另有其人？这就是历史的疑案了。当地有一个
传说，武则天听说段简要害陈子昂，曾派了八员女将从京都千里赶来
相救。但当她们赶到时，陈子昂已经成为刀下的冤魂了。传说虽然美
丽，悲剧却已无可挽回……  
　　我的思绪在历史的迷雾中徜徉，不觉已是日斜西山了。涪江在夕
阳的余辉下静静流淌，我的眼前突然闪现出一艘缓缓驶来的小船。一
位长髯飘飘的老人站在船头，他面色凝重，遥望着暮霭沉沉的金华山
不禁悲从中来：陈公读书堂，石柱仄青苔。悲风为我起，激烈伤雄才。
这个人是杜工部，此时是公元762年的冬天，唐朝历史上两位大诗人
在此对话，但他们中间已隔了六十多年岁月的沧桑，大唐王朝也在安
史之乱的风烟中满目疮痍，摇摇欲坠了。一千多后，一个无名小辈来
到这里，他只是来缅怀一段尘封的历史，但从中却咀嚼到了同样苦涩
的滋味。
