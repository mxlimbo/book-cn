短歌ABC

○西湖歌舞(安徽)


A  
我凝视镜子里面的硬伤  
仿佛那是另一个人的皱纹  
春天的雨水打湿了窗棂  
 
如果我醉在浪尖，以天空为镜  
海浪的阴影将无休止地退缩  
去抚摸灯火通明的肉体  
 
不知不觉，一些细小的事物  
发展了爱与同情的面部运动  
如今都沉溺在石头里。我已迟到…… 
 
但真实的情况是，我就站在自己的屋子里  
不是在镜子的深处，而是在生活的表面  
我的想象总是不着边际  
 
B  
再多给一点时间，我就会更加从容地  
登上分别多年的山巅。我多么需要  
新鲜的空气持久地滋润心肺  
 
我曾经珍藏斑斓变幻的诗情画意  
把读书当作追赶博大的思想。真的  
多年以前，那是一种幸福…… 
 
可我必须在鸡鸣之前离开，不再怀念。  
火把后面是忧郁的墓碑  
过去的毕竟过去了，是花粉也是灰烬  
 
我将回到庸常的生活，回到另一种阳光下  
隐没在人群里，比众多的绿叶更象绿叶  
比此起彼伏的回声更象回声  
 
C  
给我面具，我就能听到童话  
一些鲜活的动物穿过朦胧的天空  
穿过风，乘一缕月光来到我的房间  
 
后半生能收割几回这样的呓语？  
我的村庄周围亮起空荡荡的田野  
我知道，那是睡意正浓的村庄  
 
有关昨天，我的臆想已进入无限生长的时光  
生活的重量遗留下怀乡病  
鸟巢空了，所有的回答不可改变  
 
再一次，再一次开始这丢失童贞的远行  
刻骨铭心的诺言是一把斧头  
白银的斧柄在光芒之上
