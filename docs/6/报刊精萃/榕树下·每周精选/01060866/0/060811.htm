朋友・兄弟

○ 杨川


    大哥，今夜我无论如何是睡不着觉了。屋外电闪雷鸣，之后又是
瓢泼大雨，整个世界都是雨声。我读你的小说，还读林弟的小说。我
这儿有许多名作家的小说，我都读厌了。我觉得他们描述的世界离我
太远了。我情肯读你和林弟的小说，有一种温暖和亲切感从你们的文
字中流进我心里。
    穿透耳膜的雨声让我猜想着你在这雨夜里会干些什么？雷电中，
乡村的电被拉了阐。黑暗中，你点燃了早已备好的烛。在你那低矮、
闷热、蚊子飞舞的小屋里，你正伏在你那张开裂崩缝的木桌上，在方
格稿中构筑你自己的世界，描述着一个又一个人物的命运。而这些人
物清一色全是如你一样的壮族。而这些人物的环境永远都在山清水秀
的壮族山寨。林弟不用猜，这会准在梦中。在这些个我们可以用来写
作的日子里，是他最劳碌的日子，他的学生们又将进入中考的严峻时
刻。他的写作总在忙里偷闲，或假期里很少的几天空闲中。然而他总
能出好作品。他的作品给人一种少而精美的姿态。那一年一家省刊还
把他当文学新秀特别推出。
    大哥，你对小说的操练是完全来之于生活。你笔下的人物，讲着
乡言土语，或半生不熟的汉话，甚至反装句。浓郁的壮族特色，乡巴
佬的炊烟和稻田里的香味全被你搬到了纸上。你的小说永远溢出那种
新米煮在锅里的清香。你知道你走不出你那风景秀丽的壮乡，你知道
你对城里的事物难得读透。五十多年的乡村生活，你只能透悟乡下人
的喜怒，壮族寨子里人们的悲欢离合，这些是你的长项。我和林弟无
论如何也达不到你那种境界。
    林弟是造小说的高手，但他也只能在他的视野中构筑他的世界。
在他性情深处缔结他偏爱的情结。他的小说扬溢着才气、诗意，美且
沉重。让人在阅读中得到美的享受。他那些奇巧的构思，精美的句子
让我即爱又怕。王小波说过∶“好的小说会让读者喜欢，让写小说的
人害怕。”当我以读者的心态阅读你的充满乡土清醇的小说时，读林
弟那充满美感、哲理的小说时，我会拍案叫好！可我用写者心态看你
和林弟的小说时，我感到茫然。我问自己∶为什么写不出乡情浓郁如
大哥那样的小说？为什么写不出如林弟那睿智、精美的文字来？或许，
小说、乃至句式、气势都是拒绝复制和效仿的。这样我只好依旧我行
我素地拼凑自己的文字、句式，让文字形成小说。幸好大部份作文都
换成了烟钱，也没给大哥和林弟丢脸。否则我就无脸夹在你和林弟之
间充老二了。
    大哥∶你知道，我最先认识的是林弟。九五年我的小说和他的小
说相遇到了同一本杂志上。至今我仍然认为，那一期杂志上最好的小
说第一数他那篇，嘿嘿，第二当然是我的那篇。那时我才到这个小县
城。我是自己把自己下了岗来的。当时下岗这词还在它妈肚里没生出
来。我上了家庭主男这岗，煮饭、接送上幼儿园的女儿，也开始实施
我二十岁时的写作梦。我多方打听得知这个县只有二个小说写得好的
人，一个就是在边远山区、壮族寨子里的民办教师你，另一个自然是
林弟。可我们都没见过。读林弟那篇小说，我真正读懂了他文字后面
的情节，他是个能用小说反应心情，层次较高的人，我以为他至少也
有十多年的小说操练史，我甚至以为他是中年人。后来得知他才二十
多岁，工作没几年的中学教师，真让我吃了一惊。并且产生了极想结
识他的冲动。
    在一次改稿会前，我终于见到了他。林弟还是个文静秀气的小帅
哥，举止稳重，言谈柔和自然，还有几分乡村中学教师的纯朴。真有
点人如其文，清新、洒脱、耐看。那个时候我暗想∶他那厚重的文学
功底和对小说的领悟是怎样钻进他大脑的？老道的文笔，年轻的岁数，
这已经超越了我佣常的判断认识。我心里就对他说了句∶“仙人！”
那年我三十六岁。比林弟长十岁多一点点。
    那时林弟也没见过你。尽管同在一县，但相距遥远。改稿会报到
后我和林弟就被地区文联的车送到了离市里很远的一个风景区。我们
一直在编辑老师那儿打听你究竟到了没有。晚上篝火晚会，我再次问
编辑老师，那位老师诧异地问∶“你们一个县的，会没见过？”那位
老师指着默默坐在一边的你∶“那，就是他。”
    那时你沉静地坐在篝火旁，一双深陷在额下的眼晴在火光中闪动
着亮光，一看便知是典型的壮族人。我们握手、相识。大哥∶那晚你
给我的印像是什么？你肯定不知道。这么多年我都没对你说过。那晚
我仿佛见到一颗坚而铁实的古榕树根。给人一种凝重、沧桑而又深沉
的感觉。第二天我从你目光中又一次印证了我的感觉。
    在风景区的改稿会，地区请来了省内、外一些刊物的编辑，意在
让我们这些写者的稿纸有处可卖。我感觉更象一次物资交易会，老编
挑选产品，写者出售产品。然而清高的文人和老编都不会用这种语气
去形容这次活动，美名曰∶改稿会。呵呵，大哥，我这个人有点俗嘎，
但实在哦。
    你的作品被选中了，但要改。那是省里一家很有影响力的刊物老
编对你的要求。在风景区住着时你就改了一回，临到回城老编还让改。
多他妈的折磨人。而你整个改稿会期间都在改稿，特别是回城，我俩
同住一宾馆标间，你通夜改稿，让我都替你气不服。这些老编真他妈
的会折磨人！你也真是太老实了。连林弟也说这太不划算了。
    我抱了一大档案袋六七篇小说去，才卖掉一个六千字的短篇，余
下的又都抱了回来。气得我直骂这些老编有眼无珠。林弟被选中一篇，
也是叫改。林弟说回去改了再寄。后来改了寄去，如肉包子打狗，有
去无回。我一直怀疑那些受地区文联邀请来的老编，有的真正不是来
选稿，只是骗吃骗喝而已。在随后的几年里我还遭遇过国家级的刊物
老编到云南省文联骗吃骗喝，把作者辛苦手抄之作带回去说发，结果
自然是肉包子了。唉！大哥阿，云南人憨，云南人真诚，云南人极易
相信别人。我们走自己的路吧，别再轻易相信别人。
    还记得吗？我们从改稿会上回到县城后，一起去逛报刊亭，我买
了本外省的省级文学刊物，我看了几眼就肯定我写的不比他们登出来
的差。第二天把从改稿会上带回来的稿，按地址寄了去。三个月后稿
费就和样书一起寄来了。所以，人应该自信，自己选择刊物乃至编辑。
这叫主动出击！千万别被动等候。
    从相识到相知，五、六年了。这只是一个婴儿从生到幼儿园的短
暂的成长过程。然而我们却走过了好漫长的一段岁月。这种漫长的感
觉来自于生活的艰难。大哥，你至今还是一个拿着低微工资的民办教
师。或许你将是民办教师的终结者，纪念碑。在简陋昏暗的乡村教室，
用你所有的真诚和学识去教导那些半句汉语都不懂的孩子，那些走出
大山，走进城里、走进大学学堂的壮家子弟，他们对世界的最初认知，
对汉字的领悟就来自你的引导。你的弟子一批批走了，你却永远地留
在了山里，用你的真挚和生命的岁月，继续着你的教育事业，也继续
着你的小说人生。
    大哥，我并不欣赏你的这些近乎于伟大的精神。在我的思辩中，
这只是一种牺牲自己的愚昧精神。我们的媒体和民众主流偏见都以褒
奖、畅导的理论唆使民众树立牺性自己、奉献社会的精神。你的奉献
没有回报，你那一两百元的低微工资意味着剥削和不公正！逆来顺受
的中国人，要什么时候才能觉醒到理直气壮地去争取自己的公正、平
等、合理呢？大哥，你的思想在这方面还在睡眠中？醒醒，好吗？我
们是人，要争取自己平等、公正的人格！要对一切不平等的，不尊重
人的价值的人说∶“滚你妈的头！”
    说到林弟，我听到最多的一句话，被我称之为屁话的话就是∶年
轻有为，前途无量。
    林弟确实年轻，才华横溢。做一校之长，忙得脚底板翻天。一年
难得几天闲，放寒、暑假别人有十天半月的休闲时光。他可能就没这
么幸运了。忙！忙！！！这种庸常的忙碌消耗了他这几里年的生命时
光，这就是他的有为。我的看法是∶如这般有为倒不如无为还好。林
弟是个天生文学种，全身上下都透出小说的细胞。如果，他能无为于
那些过眼烟云的事物，静下心来操练文学，几年内，中国又多了一个
知名作家。这才是他最长项的、最完美人生价值的取向。可这只是我
的想法，我也知道林弟此时大有∶“人在江湖身不由已”的无奈。毕
竟他得为社会做这份事才能养活自己。我坚信文学的冲动、爆发，早
就暗布在林弟身体每个细胞中，总会有石破天惊的一天。大哥，让我
们共同期待这一天，让我们做好分享林弟大收获喜悦的准备吧。
    大哥∶你会不会觉得我太看重文学、太看重小说了。
    当然啦，我们弟兄三人就是因小说相识、相知，从文友――朋友
――到兄弟情的阿。文学是我们共同的语言，不讲这还讲啥呢？
    我们活着，都不容易，我们选择文学，把她当自己第二个老婆。
活一生，爱一世。喜欢也罢、讨厌也罢，选择了，就相拥着走过一生
吧。我们不能选择权力、也无能选择金钱，只能选择廉价的一支笔、
几叠纸所能构筑的虚拟的美好、虚拟的痛苦、虚拟的所有喜怒哀乐。
我们这样活着，一起往前走。
