走看梅雨

○陈富强

　　梅雨是江南的一张名片。没有梅雨的江南不是真正的江南。 
　　梅雨通常是淅淅沥沥的，在窗外懒洋洋地下着，我醒着的时候在
下，我睡着的时候她也在下，等我醒来了，她依然在下着，于是，这
漫天的雨下得屋子里也潮了，瓦房顶上的草湿了干，干了又湿，年久
失修的老屋，在这个季节里不堪雨淋，天上下着细细密密的雨丝，屋
内也落着滴滴嗒嗒的雨点了。树已经绿了，被雨一次又一次地洗着，
更显绿了，在眼前浓浓淡淡地洇开去，满目都是绿色了。我坐在窗前
听雨，雨在茫茫的天空下看我。我就想走进雨里去了，去看雨。 
　　我真的走进雨中是在路上，车子在跑着，天幕像裂开了一个大口
子，雨像决了堤的水一样从高处倾下来，这时，车子就如行在水里了，
从车窗里望出去，白茫茫一片，只见雨溅在玻璃窗上，水雾把我的眼
睛蒙住了，路旁的树和花草一律保持沉默，它们在雨中已经没有开放
的姿态了，原先的婀娜现在让位于雨落成的花朵了。这还是梅雨么？
梅雨会是这样的放肆，这样的不顾一切么？梅雨是女人的眼泪，是细
细的，断断续续的，柔情万种的，怎么可以豪放到不要一种优雅的姿
势，落成如此倾盆，莫非梅雨也有说不出来的苦，以雨洗面。我坐在
车内，可是我又是在行走着的，我看到了梅雨在平坦的田野里一望无
际，也看到了梅雨在山谷间若有若无的飘荡，我穿行在梅雨中间，听
到梅雨在高空的叹息，我想梅雨一定是有心事了，她郁积了一年的心
事就在这些日子里全部化作雨水释放了，这是梅雨，也是泪水了。被
梅雨淋过的路边的树和花草更见翠绿，花色清纯地在绿叶间荡漾着，
山清清地在我的视线里向后倒去。在这样的雨里的江南，泥土也会飘
渺出清丽的芬芳，如果有谁插下一截木桩，来年一定也会生根发芽。 
　　再一次走看梅雨是在端午节了。我办完了事去推自行车，就发现
后轮没有气了，这时，雨在哗哗地下着，一点也没有停下来的意思，
在梅雨季节，是不能指望雨在短时间内会停下来，她不下个淋漓尽致
是不肯的竭手的，我推着车子沿环城西路往回走，走过这条路的人都
知道，这是一条绿草如茵，树木葱茏的路。路上少行人，我走进雨里，
雨就疯一样地往我的雨衣上倒了，我听到雨点砸在雨衣上沉闷的卟卟
声，也能感觉到这种声音带给我的一丝丝疼痛，树叶和花枝无奈地听
任雨一遍又一遍地浇灌着，我很为这些花草树木担心着，它们没日没
夜地被雨水浇着，总会有一天，要喝醉了。远眺雨中的房子，墙上的
藤蔓依旧不屈不挠地向上攀援，黑瓦间流下的水像瀑布，有着木格子
的窗子紧闭着，这屋子的主人透过木百叶窗，看得见从自家屋顶上流
下的飞瀑了。 
　　雨在路上堆积起来了，这里一滩，那儿一滩，车轮碾过时，水就
飞起好看的水花，这种无色的花朵的开放比昙花还要短暂，没有颜色，
但花瓣四射，在行人的视线里飞成一朵，然后转瞬即逝。水花的开放
无比自由，她可以在车轮底下开放，也能在我的脚下开成一路细细的
碎花，我踩在积水成溪的路上，踩一脚就能踩出一朵水花，我行一路，
就踩出一路的花朵了。我独自走在雨中，有一路的花朵伴随着我开放，
梅雨带给我的一些伤感也从脚下溜走了。 
　　我走在梅雨里，我想起了童年的乡村了。那时节也有梅雨，但记
忆中的梅雨是丝丝小雨，是很少这样的瓢泼大雨的，我扯一块塑料布，
赤足走在青石板路上，雨打在脚背上，轻轻的，柔柔的，像是一根羽
毛在轻拂着，田野里的河水涨了，倒灌进稻田里，鱼儿就游进田间的
水沟去了，我站在田埂边就能看到一尾尾的鲫鱼在水中快乐地游来游
去，鱼尾泼起的水花，鱼头昂出水面，我只要凭我的一双站手在水沟
的一端布成网状，也能活捉它们，它们在河里的敏捷此刻已经全无，
它们一定是被柔情的梅雨冲得晕头转向了，它们没有想到在它们的前
方有一双小手在等着它们。我用青草穿起鱼儿，拎着往家走。我走过
一户农家，农家的主人问我是什么鱼？我自豪地大声说是鲫鱼，主人
说好煎鲫鱼汤了。我又走过一户农家，主人问你摸的是什么鱼？我依
旧自豪地大声说是鲫鱼，主人说好做红烧鲫鱼了。这时我才晓得原来
鲫鱼是有好几种做法的。梅雨季节过去了，乡村的晒场上就会摊满从
衣箱里翻出来的衣裳和棉被，落了一季的梅雨，屋子里发出潮湿的气
息，棉衣里也会有白色的细绒样的花花，这是梅雨过后留下的痕迹，
必须在太阳底下晒上几天才会去除梅蒸气息。 
　　乡村的房子多是平房，所以特别会产生梅雨天特有的痕迹，城里
的房子多为高楼，梅雨的侵蚀相对要小一些。我的眺望已经看不到老
家的屋子了，地上会不会依然长出绿色的细绒样的花来？只是田野的
水沟里大约不会再有鲫鱼的游动了。 
　　我在梅雨的端午走着，我想起了多少年以前的那个端午节，一个
诗人投进了江里，淹死了。那天，有没有今天这样的雨如注？如今，
哭泣的梅雨是不是为他而下？为了写作这篇短文，我曾经是那么刻意
地想找到一句古诗的作者，这句古诗是“黄梅时节家家雨”，可是我
记不起它的作者了，就像我记不起多少年以前的那个端午，他在投进
汩罗江时的眼泪是如何让寂静的江水大声地喧哗。我只好去超市买了
一把艾草插在门上，告诉自己，端午节来了。 
　　我这样一直走下去能走到哪里？这梅雨如果真是女人的眼泪，要
把走在路上的所有人都给淹没了。如果我停止走动，梅雨，你能停止
哭泣么？
