猪头肉

○袁宏伟

　　我小的时候，爸爸妈妈在一处中学当老师，微薄的工资要养活一
大家人。上有老，下有我们三个还小的孩子。所以，家里平常很少吃
肉。母亲精打细算，一分钱掰成两块花，可是，到了月底还是捉襟见
拙。有时，自己磨点豆桨，到外面买来斤半油条，及马蹄烧饼，家里
就像过节一样。吃完了都舍不得把嘴角的疙瘩擦去。 
　　家里的小孩有时生病了，母亲就会拿一个攒了很久的鸡蛋，煎一
个荷包蛋，外加上一碗面条。小病号吃得嗤嗤拉拉的，狼吞虎咽。而
旁边的兄弟姐妹就会眼巴巴的瞅着他，嘴里咽着口水，还问：“面条
香么？好吃么？”小病号总是顾不上回答，不一会的功夫就会把面条
鸡蛋消灭掉。围在周围的孩子们见没有多少戏了，就会失望的散去。 
　　不过，碰上大点的孩子吃“病号饭”时，多半会把“病号饭”分
给大家吃。尽管每人只有一筷子或者半筷子面条，一块很小的鸡蛋，
可是大家都吃得格外香甜。等把这碗面条和鸡蛋吃完，病仿佛也好了
一大半。所以，当时我和弟弟妹妹总是盼望着生病，而且是生大病。
这样，就可以吃到母亲做的荷包鸡蛋面条。那时，谁生病了，就会让
不生病的孩子们羡慕的要命。 
　　还记得在过年时候，学校里破天荒的弄来了一批猪头。要分给老
师们过年。但是猪头的数量很少，按老师们的人数来分，肯定不够。
所以，大家研究来研究去，决定还是按照最古老的办法，就是“抓阄
儿”。包了纸团，纸团里写了字的就是有猪头的，没有字的，就没有。
等到把“阄儿”做好，老师们一个一个都蜂拥而至，磨拳擦掌，都希
望自己的运气好一点。当时，我爸爸是学校的校长，妈妈是学校的图
书室的管理员，爸爸决定，由我和弟弟出马，去抓“阄”。妈妈还有
点担心，怕我们小孩子不会抓，被大人们欺负。可是，爸爸已经定下
来了，他说：“不就是个猪头么？有没有它咱们照样过年。让孩子们
去，抓着了，大家没有话说，抓不着，就会给别的老师们一份希望。
大家都不容易。如果咱们亲自去抓，大家有的可能怀疑咱们在‘阄’
里做了手脚。”妈妈只好嘱咐我们几句，让我们去了。我和弟弟蹦蹦
跳跳去了。 
　　现场非常热闹，很多的人围着那个盛满“阄”的箕簸。搁在一边
的猪头们排着整齐的队伍，一个个的呲牙咧嘴，肥头大耳，仿佛冲着
人们在笑。由一个德高望重的，学校里的“贫委会”老大爷主持。他
很严肃把箕簸用力的颠了颠。然后又用手伸进去搅了搅，又仔细的检
查了一下“阄”的数量，一切都正常。等到老大爷喊了一声“好了。”
人们争先恐后的冲上前去，一个比一个还要迅速的捡起“阄”来。老
大爷被人们推得趔趔趄趄，但还没有忘记自己的职责，大声的喊着：
“一家派一个代表来，别乱，一个一个的来，双职工的派两个代表。
小孩子们先抓。”可是他的声音马上就被那些抓到的或者没有抓到的
人们的惊喜和沮丧的声音所代替。 
　　我和弟弟根本就靠不到前。弟弟非常机灵，他低下头，拉着我，
从大人们的腿缝里钻过去，穿过层层的“腿”我和弟弟才看到了那个
箕簸。老大爷看到我们，马上说：“来，来，你们是不是替你们爸爸
妈妈来抓阄的呀？”我点了点头，我颠着脚尖，勉强才够到了放着
“阄儿”的桌子。在大人们的各种各样的手下，我好不容易扯到了一
个纸团，弟弟因为个子实在太矮，够不到桌子，在下面急得哭起来了，
有一个好心的老师把弟弟抱起来，弟弟含着泪水，认认真真的挑选了
一个纸阄儿。 
　　我和弟弟拿着那枚“阄儿”，谁也不敢打开，一直跑到猪头那里，
有一个老师专门负责来分猪头，拿着有字的阄儿去，他就会把猪头给
你。他看到我和弟弟气喘喘吁吁的跑过来，笑着问：“怎么？你们阄
儿中了？”我把弟弟手里的阄儿和我的手里阄儿一块交到他的手里，
说：“叔叔，你给我们看一下，我们的‘阄儿’是不是有字？”他笑
着接过去，慢慢打开，我的心怦怦乱跳，弟弟甚至捂住眼睛。他看到
第一个，笑嘻嘻的说：“恭喜，有一个中了。”哇，我和弟弟立刻跳
起来，欢呼起来。他又打开了第二个，瞪大了眼睛，有点迷惑的说：
“又中了。”我和弟弟也愣了半天，然后，我们忍不住就互相拥抱起
来，我抱着弟弟，不知应该怎么样表达自己喜悦心情才好。弟弟搂着
我的脖子，一个劲的嚷：“姐，你掐掐我的肉。”我说：“你欢喜糊
涂了？掐你肉干什么？”他忽闪着一对大眼睛说：“我好知道我是不
在在做梦呀！”那位老师，笑了：“快把这两个猪头拿回家吧，你就
知道是不是做梦了。” 
　　我和弟弟一人抱着一个猪头，欢天喜地回家了。 
　　爸爸把我们抓中的猪头放到了门前大树上挂了好几天，说是放在
外面冻一冻，收拾起来容易些。其实，是向别人显摆。那些没有抓到
猪头的老师们，时常会有人来参观一番，盯着那两个猪头，满脸都是
艳羡的神情，然后感叹一番，“唉，小孩子的手就是‘毒’呀。不服
不行。”不管谁来了，爸爸和妈妈都会把我和弟弟在客人面前猛夸奖
一番，说是我们的手气如何好。客人们也会拉着我和弟弟的小手，说
是童子手，干净，抓什么来什么。当即，即有好几个亲戚来我们家，
说是单位抓“阄儿”抓自行车，让我和弟弟去。说是借用小孩子的好
运气。吓得父母替我们拦了，亲戚们还老一阵子不高兴。 
　　那年过年，爸爸把两个猪头收拾得干干净净，一个煮了一锅，放
在外面冰起来，当作冻肉吃。拌上大蒜，放上酱油和醋，卷着煎饼吃，
味道要有多香就有多香。另一个猪头把耳朵割下来，炒了一个菜，剩
下的就细水长流了，做成了腊肉，来客人的时候，才会拿下来，割一
刀肉，炒菜吃。 
　　那个年，因为我们的好手气，而过得富足，过得畅快。现在想起
来，其实，那会的猪头上好多都是肥肉，用现代的观点，就是多脂肪，
多胆固醇，是时髦小姐们和太太们避之不及的，可是，就算到现在，
我最爱吃的依然是猪头肉，煮得烂烂的，肥瘦搭配均匀，切得细细的，
用捣好了的蒜泥拌了，加上酱油，米醋，葱花和姜丝，再配上黄瓜丝，
用烧饼或者山东煎饼一夹，咬在嘴里，满嘴里流油，特有的猪肉香味，
令你满颊生辉。它让我感到生活着是多么美好。
