灯笼果

○敲字落花


    圆溜溜地似脂球，青珀珀地似橄榄。一咬，酸汲汲的汁儿直激脖
喉眼，令人好笑。小核，如碎米粒，味苦。
    如果，你再进一步仔细观察，会发现它的表皮还印有一圈圈银白
色的道儿，檀香色的杠杠，如描绘地球的子午线。我想不通，光凭这
一点，也根本沾不上灯笼的边呀，谁家的灯笼是绿色的？名字起的不
好。
    这物儿，南方怕没有。
    大北方，也仅仅在我的故乡哈尔滨市郊见过。它，掛果在树，坠
千颗万颗绿珠，拇指儿甲壳大。这种树类似茶树，簇生，漫过膝盖，
长势稠密，笔直地茂似一堵绿墙。
    我们学校旁就有一围果园，植一片这种树。
    果园是最能激发孩子们行窃意念的诱惑之地。
    管理果树的园丁知道这个理儿，立下一项特别的规矩：凡是孩子
们来，可以自由到园内摘吃水果，不用花销一分钱，吃多吃少，这倒
要看你自己的肚皮大小啰。但是不允许带走一颗一枚。
    我们知恩图报，不白吃。做小工，以工换物儿。帮果农捉虫子、
灭砬剌蛊、洋虫罐、剪余叶、打扫院子。
    守园子的艺工是两位老人。
    一胖、一瘦；一恶、一善；一懒、一勤，都姓王。小孩嘴甜不吃
亏，我们就都喊王大爷。冷不丁，有时，会冒出二个人同时应声。总
不见胖大爷挪动那席火炕，憋在饭桌后面吧叽吧叽地吸旱烟袋。每一
次，都是瘦大爷招呼我们。胖大爷不爱动，还指派我们给他递水，捧
盆子，涮碟子碗筷……，背地里我们咬牙切齿地喊他“老地主”。
    二丫是瘦大爷的闺女。小嘴特别能摆乎，会唱小令。唱孟姜女。
唱小白菜。唱傻小子坐门墩。
    我们听了好感动，想哭。
    这时，胖大爷便用铜烟袋锅使劲拷炕沿儿，骂一句：“没出息！”
    瘦大爷过来打圆场：“咳，和小孩子斗什么气。有憋尿憋屎的，
没有憋泪儿的。”
    胖大爷就不言语了。
    他会编蝈蝈笼，一房一室，用秫秸扎的，方的圆的，扁的，塔形
……。有的是原色，有的被涂成檀酱色，小小巧巧，大不过巴掌。每
年他必定送我们一人一个，决不多给。
    后来，我们都长大了。二丫考上了林学院，离我们住家不远，来
玩，说着，笑着，叨叨地就讲起了胖大爷的种种不是。
    二丫听得眼珠瞪得大大的，猛拍大腿：“哎呀，瞧你们，太小心
眼了吧。你们错怪了胖大爷。”
    我驳道：“谁错怪谁呀？”
    “胖大爷可不是你说的那一种人。他是荣军，双腿抗美援朝时被
鬼子的炮弹炸开了花，截去了一半。他行动不便，只好一天天呆在炕
上，遭罪呀，没有腿的人哪”
    误会你十年。
    胖大爷！
    “咦，你咋不吃灯笼果，嫌酸？这可是新品种，先酸后甜，一泡
浆。”
    我尝了一颗，果然，酸酸地激凛人，提神儿，而后释放出一股淡
淡的回甜。“真的是不一样吗？”我选了一颗大点的，捧在手心，左
瞧瞧，右看看，觉得它似另外一种被放大了的东西：一珠绿泪儿。
