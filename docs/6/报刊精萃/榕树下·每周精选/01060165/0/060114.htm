摊破浣溪纱

○风与城


    陶朱初见南浦时，是在公司的新进人员培训课上，满座尽皆凝神
正襟，企图博得良好印象，唯独这名面孔秀丽的女子神态自若。偏是
这样他才注意到她，讲到“客户永远是对的”，停下来提问，如果客
户错了又当如何？旁边楚翘撇一撇嘴，显是对这样幼稚的问题不屑一
顾，陶朱目光望向南浦，喏，就是你。我，南浦回过神来，还好她尚
懂得答，请参照第一条。陶朱略一晗首，不知如何反倒松一口气。
    数日后名单分配下来，南浦与楚翘俱在陶朱这一组，当晚他作东，
三个人去西餐厅吃牛排。席间楚翘眉飞色舞，南浦却只是淡淡笑容，
并不多话。陶朱问宋小姐，南浦便答石经理，几个回合下来两人仍是
拘束，陶朱于是说，私底下你可以叫我名字，石陶朱。南浦随口应道，
很少听到三个姓的名字。陶朱吐出一口烟圈，说来话长，总之是与身
世有关。楚翘把握机会插进来，名以言志，石经理你是想学想学范蠡
财色兼收吧。陶朱从容答道，但凡男人，谁个不想。那你如今可算得
偿所愿了，楚翘察言观色，犹要再拍上一掌。
    南浦听得骇异，不过一个部门经理，值得她如此不遗余力？许是
南浦自己久已脱离社会的缘故。她自幼便无做女强人的愿望，大学毕
业嫁给百川，先还替他的小公司管理杂务，后来发展迅速，一应业务
外包，她便名正言顺回到家中。百川笑，不怕我在外面花天酒地？南
浦揽镜自照，或者等我人老珠黄。
    谁知竟一语成谶，南浦尚未老去，已有大把青春貌美之少女自动
送上门来。在他口袋拣出那只俗气的蕾丝花边袜，南浦不禁诧异，人
说男人之财富与品味成正比，如何欣赏女人偏偏不在此例？分手时南
浦十分平静，倒是百川悔恨不已，苦苦牵住她的衣角。南浦狠心并未
回头，不是不相信他一时失足，但她宁肯抓紧这唯一输得心高气傲的
机会。
    万幸尚无小孩，没有无辜受害者，百川仍是给了大笔费用，南浦
不动声色照单签收。她并未动用分毫，或者早该自食其力了，在家蛰
居这半年，她已明白婚姻失败不是陈百川一个人的责任。南浦决定出
来工作，填履历时十分惴惴，众多应征者，唯独她是一片空白。或是
久已不用的营销文凭起了作用，竟然意外得以通过，当晚南浦暗下决
心，定要做它个风生水起。
    自此南浦与楚翘便做了陶朱的左右二臂。南浦自知底气不足，十
分谨慎，除去同事例行的招呼，并不与他人闲谈是非。楚翘又不同，
她在公司活跃异常，上下逢源，对客户则手腕灵活，无往不利，端的
是个利害人物。南浦自叹弗如，但不知何故石陶朱似是有意照顾，凡
事指点不说，交给楚翘的单子做得辛苦，而她的却往往水到渠成。几
月下来，南浦长进不少，业绩竟也能够与楚翘平分秋色。
    一日傍晚陶朱吩咐下来，楚翘可以按时下班，南浦则陪他前往酒
店。这虽不是第一次分头行事，楚翘走时却斜斜睨过一眼，该名大主
顾举足轻重，你打醒十二分精神。南浦慌忙应了一句是。路上果见陶
朱神情肃穆，在酒店咖啡厅落了座，犹要补上几句，这次非比寻常，
我也并无一分把握，南浦你务得为我争气。南浦听了不禁心酸，谁不
知养兵千日用兵一时，这话无非暗示受了他的恩惠，当得舍命相报了，
可是终究敌不过好奇，陶朱平日并不罗唣，不知什么大人物驾到？
    那人姗姗来迟，相貌温和，气度不凡，南浦一见之下却宛如当头
棒喝。耳听得陶朱介绍，这位是IT巨头陈百川先生，南浦只是作声不
得。百川与她四目相接也是一怔，毕竟见过世面的人，即刻恢复正常。
互相恭维奉承之后渐渐转入正题，陶朱问道，不知陈总今年在广告方
面有何打算？百川微微一笑，石经理你哪里找来宋小姐这样人材？陶
朱立刻会意，我有点事离开一下，南浦你陪陈先生聊聊。
    南浦略为抬起眼睛，那凄惶的神色叫两个男人看得心下为之一动。
陶朱是千般不忍，百川却是万般无奈。昔日最懒与人交际的南浦如何
会选这一行，百川只觉千错万错在己身，呆坐半天终是问出一句，你
现在可好？南浦心下想，这话可怎么回答，好与不好岂是一个字说得
清。忽然瞥到陶朱犹站在不远以眼光向她鼓励，顿时想起自己身份，
当下说道，多谢陈先生关心，工作尚可。南浦握着玻璃杯的身体慢慢
后仰，百川知道他与她已不只是隔着一张桌子那么远的距离。
    饭后百川要送她回家，南浦并无拒绝，张姓客户与陈姓客户有何
不同，她已有把握心下坦然。二人步出酒店，南浦意外发现陶朱的车
候在门厅，于是回头向百川笑道，不劳陈先生，我们还得回公司将细
节拟出。
    次日合同签订下来，公司上下一片欢腾，陶朱将楚翘与南浦叫至
办公室，分别作出升职与加薪的奖励。楚翘喜形于色，南浦却是一贯
的宠辱不惊，谁料陶朱生出误会，独独留下南浦细细解释，你是新人，
无谓太出风头。南浦点一点头，她岂会看重那些，只是明白他的良苦
用心，又多添一分感激。出得门来果见楚翘春风得意，四下夸耀即将
搬入公司付首期的新居。南浦天真地问，陶朱如何有这般权力？楚翘
愕然，你不知道？他是公司总经理，董事会成员。
    天！南浦满面通红，这年头凡挂个主管职务一律以老总相称，哪
里料到他竟是真命天子。楚翘向她投过怪异一瞥，象在判断她是愚昧
或作状，而她要到这时才恍然大悟，难怪同事都以这等眼色相待，怕
不是怀疑有什么不可告人之内幕。
    陶朱自己也讲不出道理，为何当初偏偏挑中南浦。楚翘是公司出
重金自对手处挖来，参加新人培训不过走走过场，但南浦，他一见她
便没来由地另眼相看。下决定时还是颇费过一番踌躇，不知该放在底
层磨练或是直接委以重任，犹豫再三终究力排众议留在身边。后来她
的表现没有叫他失望，虽说他有意栽培，但也得她是那块料。他安慰
自己，有经验的管理者可一眼看出，宋南浦似只璞玉，稍加雕琢日后
必成大器。
    陶朱渐渐感到自己已对南浦情意暗生是在公司的股东大会后，那
日他得知，经过一系列繁琐的过户手续，石总成为石董已指日可待。
楚翘第一个前来恭喜他，这女郎，不知哪里的消息如此迅即。多年苦
心经营眼看得成正果，陶朱心下也难捺欣喜，他叮嘱楚翘未成定局不
得声张，可是终究忍不住私自与她前去酒吧庆祝。楚翘混着香气与酒
味的嘴唇直倾到他耳边，她唤他，陶朱，如今你还有何心愿未了？陶
朱胸中波澜起伏，百感交集，从一名家境贫寒，母亲改嫁的穷小子挣
扎到现时地位，其中艰难唯有自知；但与此时荣耀相比，所有一切竟
已不值一提。如果说还有何遗憾，当是没有一位自己所真心爱的红颜。
以他现今条件，这该不是什么难事吧？
    陶朱望着楚翘精致的五官，他知道她对他觊觎已久，只要他一个
眼神，软玉温香立时满怀。但在此刻他忽然极之渴望南浦那不带任何
欲望的清秀面庞。或许越是不可得到，越是想要征服，陶朱被心下强
烈的愿望驱使，不顾楚翘难堪，扔下她直奔南浦住处。
    南浦对陶朱的到来并未表现任何诧异，她似有种处变不惊的天赋，
陶朱想起那日在酒店南浦投过的仓惶一瞥，原来这些日子他一直缠绕
于心，挥之不去。小小公寓陈设简单，南浦递过一杯热茶与淡定笑容，
恭喜石董。陶朱汗颜，连她都已知道，这上下怕不只差贴出布告。他
对楚翘的大嘴暗生恼恨，忽然握住南浦的手由衷说道，南浦，你与楚
翘真是不一样。南浦心下一颤，欲待推开，他的手却温热有力，挣无
从挣。你也这么说，可见不是空穴来风了，她缓缓将手抽出，我倒宁
愿象楚翘，不要赋人口实。别把不相干的两件事混在一起，你的业绩
有目共睹，谁敢质疑。是吗，南浦反问，若我洁身自好，谣言自然平
息；若是被不幸而言中呢，哪有功劳这回事。
    陶朱站起身，南浦，不要似只刺猬，仔细考虑再作回答。两人对
视半晌，他忽然拉过她温柔地拥入怀中。她的秀发受热气吸引，丝丝
缕缕贴过来，贴至他的脸上，陶朱深吸一口气，你知道我会等你，任
何时候。南浦抿紧双唇，默不作声，眼见得心上硬壳一寸寸裂开，眼
见得他放开她，一步步走到门口。
    南浦彻夜未能成眠，陶朱与百川的面孔轮换在她眼前出现。这两
个人何其相似，一样的呼风唤雨，一样的一帆风顺。她忽然羡慕邻居
那对小情侣，或许唯有两手空空之时，才懂得彼此信赖是最大财富。
如果石陶朱尚是他们中的一员，她怕是会得情愿与他一起奋斗的吧？
又或者她还是楚翘那样不知天高地厚之年轻女郎，一早也已乐得合不
拢嘴。不不，宋南浦，经百川一役，切勿再轻信任何人。
    天亮时她已有了决定，不过打一份工，做不下去换一家即可，南
浦倒是第一次体会到靠自己双手的好处。走到陶朱的办公室，楚翘正
好自里面出来，她似笑非笑，石总今日出差，怎么你不知道？南浦不
去理她，将辞职信轻轻放在桌上。回来整理手头工作，偏又听到她与
女同事窃窃私语，攀上高枝变了凤凰，自然不须再做这份牛工。南浦
实在忍无可忍，走过去轻声说道，可不是，许多人盼还盼不到。
    出得这一口恶气，南浦心中痛快不少。呵，从此又不必朝九晚五
听人闲话，她舒适地安坐家中翻开报纸。可是且慢，如何心头隐隐失
落，难道做得一阵竟已不忍放手？工作无疑是令人美丽的，陶朱聚精
会神的姿态可说明一切。但教她留恋的是人还是事呢，南浦的眼睛慢
慢落在电话机上。她可是还在期待什么？
    铃声骤然响起时南浦倒吓了一跳，她的脸上依次掠过欣喜、矛盾
与迟疑的复杂神色，宛如与情人斗嘴的少女不知该否妥协。只听一把
陌生的女声说道，是宋小姐吧？南浦略为失望定一定神，请问哪位？
我是陶朱的太太，名叫周音尘。南浦纵是已做决定仍然大吃一惊，失
声问道，陶朱已经结婚？那边闻言也是一怔，公司现任总裁正是我父，
宋小姐你不知道？每个人都拿这话来问她，你不知道你不知道，仿佛
所有事只她蒙在鼓里，南浦一时只觉孤立无援，险些落下泪来。石太
太把语气放得温和，陶朱今日成就得来不易，他不见得会为你放弃到
手事业，宋小姐你是聪明人，不用我教你怎么做吧。南浦恼恨交加，
又不知如何解释，只是说不出话，那边却以为她尚在犹豫，便又说道，
宋小姐若是经济上有困难，那倒可以商量。南浦终于冷冷答，不，我
不要钱，我要的你给不起。
    暮色渐渐沉下来，沉下来，阴霾无处可逃，如纠缠不休无法苏醒
的噩梦。南浦双手颤抖，忽然提高声线问自己，你还在期待什么，嘎？
以为他会说，我丢下一切跟你走？她失控地大笑，继而俯下身，将脸
深深埋入掌心。不不，她没有哭泣，她一早已哭不出来，宋南浦庆幸
她未如楚翘般投怀送抱，此刻尚余半分自尊，可以自浴室找出一罐眼
霜，细细抹掉昨夜留下的黑圈。
    公寓的门被大力拍响，陶朱焦急地在外面叫，南浦，你在吗，为
什么不应我。奇怪，他还有何话好说，以为他权倾天下，谁料竟连自
由也没有。南浦镇定地前去开门，陶朱立即拥住她，楚翘说你今日找
过我，不及回公司我便赶来你处。他的眉目踌躇意满，原来误会南浦
的决定。我有惊喜送给你，他说，猜一猜是什么。惊喜？南浦苍凉地
想，这个世上已无任何东西能叫她吃惊与欢喜。陶朱顾自从兜中掏出
一串钥匙放入她手中，南浦，这已是你名下物业。他仿佛期待她的感
恩戴德，南浦脸上肌肉不受控制，果然笑起来，石陶朱，这可算是求
婚？陶朱微微一怔，随即答道，自然，如果你愿意的话。南浦笑意更
深，定定望住他，不行，你不能重婚。这有何妨？他不假思索脱口而
出，你不也离过婚？
    南浦笑容僵在唇边，身不由己退上一步，原来他有备而来，早已
打听清楚。她的头脑电光火石突地变得澄清，那日陶朱有意撇开楚翘，
安排她与百川单独见面，不过也是预谋中事吧？到底南浦只做了他手
中一枚棋子。南浦思前想后手足冰凉，好一个陶朱公，竟真叫楚翘说
中，古来女子扮演的角色不外两种，有用时充当利器，无用时权作解
语。陶朱犹在理直气壮滔滔不绝，你要一个名份，我不会让你委屈，
过得两年，我便与她离婚……可是要我稍安勿躁，等你功成身退，双
宿双飞？南浦怒火填胸，不待回答，抡起一掌重重掴在他脸上。
    南浦决定离开这座城市，世界没有大到遇不上的地方，但能够容
人转一转身也是好的。一干人等踪迹全无，唯有楚翘前来与她话别。
这个女郎倒是可爱，自她们没有利害关系立时视南浦为密友。她向南
浦道歉，对不起，我曾经以为……南浦微笑打断她，没有关系。楚翘
即刻释然，絮絮讲起公司闲话，陶朱做了总裁，同事挂念南浦。南浦
听在耳中，只觉一切已不再与她相关，但楚翘忽然写下一个地址，我
已搬家，记得与我联络。南浦只觉眼熟，细细瞧去可不正是那日陶朱
想送给她的物业。她向楚翘笑笑，恭喜你达偿所愿。楚翘并不疑她如
何得知，略红一红脸便算是承认。呵，只需换个名字便可馈赠新人，
如同股票过户那么简单，南浦忽然心生另种好奇，楚翘你漂亮本事，
为何定要依靠男人？楚翘倒是百无禁忌，谁爱做那份苦工，每月所得
不过有限，付了分期又不够买花，横竖强装笑脸，不如对住一个人。
一时说得兴起，又问南浦道，你可听过周音尘这名字？哦，石太太。
可不是，她愿付钱让我离开她丈夫。你如何答她？楚翘伸出手指，我
要这个数。他们恁地有钱？南浦讶异。有没有是一回事，愿不愿给又
是另一回事。此话怎讲？楚翘笑，南浦你又扮天真，我一年能花多少，
不过廿三十万，况且青春有限，我又能卖得几年？周音尘装装样子罢
了，这年头但凡男人袋里有钱谁个不是如此，睁只眼闭只眼只当少赚
几个也便算数。南浦听得感慨，可是犹觉心内之结未解，喃喃念出声
来，却听见自己讲的竟是甲之蜜糖，乙之砒霜。
    光阴荏苒，宋南浦离开不觉已经数载，本城经济更加繁荣，霸主
新贵轮番登场。这日晚报头版登出消息，财富论坛择日举行，石陶朱
先生已获邀请。音尘自他手中取过报纸，发什么呆，不认识自己名字？
一面却也难掩得色，翻来覆去细细阅读。陶朱疲倦地走至窗前，音尘
的声音紧追不舍，此时气候正好，不若我陪你同去？陶朱头也未回，
冷冷说道，我要取道杭州，你跟去做什么。
    五月的杭州春光明媚，陶朱公事已毕，忽然心血来潮，吩咐司机
驶去西湖。此时已近黄昏，游客仍是如织，陶朱信步走到石舫，叫老
板泡上一壶龙井。季节转暖，本地人早已换上薄衫，唯他仍是领带西
服丝毫不苟，一眼便知偷得浮生半日闲。老板殷勤地亲自送上托盘，
老板，茶来了。陶朱不禁失笑，此老板与彼老板，原是没有分别。回
过头再去看风景，天色又暗下一分，几个游人却兴致不减，犹要租船
游湖。陶朱闲闲望去，只见一个纤细的背影，他的心底似被重物击中，
连忙追出门外。眼见只差几步的距离，那船却缓缓荡漾开去，耳边尚
余隐约笑语，转眼已在湖光山色间。
    陶朱伫立岸边，双目濡湿，要到做了帝王以后，他才明白自己对
这个女子，始终不能释怀。多年来无法稍减的悔恨与隐痛都似有了交
代，那日若是他抛开一切，此刻与她泛舟湖上的，怕才轮到他陶朱公
吧？暖风吹过，水面的细纹便如某个刹那的盈盈眼波，一层层涌上来，
涌上来，直至将他满心的怅惘重重叠叠化为一句轻轻的低语：送之南
浦，伤之若何！
