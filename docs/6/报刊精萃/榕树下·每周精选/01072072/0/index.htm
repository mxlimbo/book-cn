【编者的话】

生和死和我有什么关系？ 
风中有驼铃传来 隐约还有花香 
也许是个蒙面的阿拉伯女子 
簪一朵蔷薇花 面纱飞舞 长发飞舞

　　备好十二件纱罗的衣裳 
　　从柔白到粉紫 
　　籍以洇湿月光下的故事
