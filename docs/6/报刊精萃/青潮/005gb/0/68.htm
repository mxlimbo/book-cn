夏日剧情


唐不遇

脸孔黧黑的爱神
她一定到过夏日的海滨
在一个凄美的爱情故事中
那美丽的女主角突然珠泪涟涟

愚蠢的设计者，危险的代言人
三妻四妾被你迎娶进门
但柴米油盐却走投无路
直到今日，那最初的盟誓
仍在泛着白沫的波面四处漂泊

一个人痛骂另一个人的虚伪
暗藏的魔鬼掐断他的脖颈
他紧紧拥住情人的肉体
灵魂绵软无力，战栗地说"飞翔！"
而心中滚过无数奇怪的念头
像雷声被雪亮的闪电押赴刑场

多少人说出言不由衷的话
那是因为爱神用一些蜂蜜把他收买
我们是否已进入角色？--不像是演戏
那么你已用沉默把自己的身心抛出
