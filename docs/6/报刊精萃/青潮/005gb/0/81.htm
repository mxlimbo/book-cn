十月的天空


明眸点墨

纷乱的意象
写满十月天空的纯净
云 只为别人洁白

所有的骨胳饥渴难耐
在草叶尖和飞鸟的羽翼
寻找流水 以及
丰满的稻谷
而十月的天空依然站着
不经意地刻意
点缀火一般的阳光

又一次在了你的包围
十月的天空
眼睛的背后是你
你的背后呢
我在十月天空下阖上双目
假装看不见
自己的后脑

山一般地跌跌撞撞后
我才真正了解了
一粒流沙的伟大
抚摸过往
嫩草便钻出十月的天空
而稻谷
在荒漠里生动地枯萎

给我一把合适的刀子
让我 堵住伤口
