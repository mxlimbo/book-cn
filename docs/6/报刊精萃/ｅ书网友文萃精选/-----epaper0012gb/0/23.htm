写 在 岁 末

雨果（whq26@163.net）

　　我有一条怪诞的生命轨迹
　　因为,曾经与你和她交汇… …
　　又突然间回到了原地
　　纵然不懂珍惜 纵然一落千丈
　　我无语 怪自己 自由放纵… …
　　我祈祷 无怨无悔
　　来吧!所有的惩罚…
　　去吧!自由飞翔…

　　什么是责任 什么是疯狂
　　什么是生活 什么是幸福
　　我不知道
　　我不知道自己要的是什么?
　　事业?金钱!爱情!?
　　NO!!!我只要活着…

　　随风飘动 伴云停驻 花开花落又一年,
　　和雀轻唱 陪燕齐舞 潮来潮往又一岁.
　　回顾以往好象不曾拥有什么 只有短暂拢上心头….

　　 ――雨果26 尽心下
　　 2000年岁末
