□...信息一体化解决方案

　　Unified Message的应用渐渐在国内兴起，通过计算机将语音、传真、数据全部集成在一个统一的信息平台将会被越来越多的用户接受和喜欢。作为全球性通讯接入设备供应商，Eicon为开发CTI应用的软件开发商提供了开发 Fax Server 和 Voice Mail Server的最佳平台。

　　Eicon Diva Server BRI、Diva Server 4BRI 和Diva Server PRI系列卡是各 CTI 应用开发商开发Unified Message方案的最佳通讯平台。Diva Server 系列卡内置高速RISC CPU 和集成的DSP 技术，将满足您的一切通讯需求。它能支 持ISDN 、PSTN、GSM 等各种通讯请求；集成的DSP 技术能支持CTI应用的 一切应用需求，如Fax Server 、IVR、DTMF 、Voice Mail Server 等；在 开发界面上，Diva Server 卡支持标准的CAPI 2.0 和TAPI 3.0 开发界面，同时 我们的CAPI ASSIST 开发工具更能降低您的开发工作量。

在欧洲和北美， Eicon拥有众多的软件合作伙伴，利用Diva Server 卡来开发各种Unified Message的应用: 如Fax Server , Voice Mail Server , Mobile Data Server 等产品 ，著名的公司如Cycos AG，Ansid，Tobit, GFI, Fenestrae, Vipcom都是Eicon的 合作伙伴，目前他们的软件和我们的Diva Server 系列卡在欧美市场上都得到 了广泛的应用。

　　方案结构图： http://www.bisdn.net/eicon/eicon54.htm

　　本拓扑图就是在欧洲市场上一个普遍流行的应用方案。通过Diva Server 系列卡跟公司的PBX相连，它将公司的Fax , Voice，Email各种信息 都跟 Microsoft Exchange 联系在一起，用户通过统一的Email 形式能接收Fax , Voice 等各种信息，使用户在任何时候都能方便地接收各种信息(不管你 在办 公室还 是在出差)，用户发送传真仅需在自己的PC 上轻轻一点，Diva Server 卡就会将 你所要发送的文件自动发出去，减少了您很多的工作量。

当然，除了此类应用外，Diva Server 系列卡能支持各类CTI应用的开发， 跟普通的语音卡和 Fax卡相比，Diva Server 卡具有自己独特的优点：首 先， Diva Server 系列卡 内置了普通的语音卡和传真卡的一切功能，同一块卡上就 可以同时支持语音(IVR) 和传真功能；另外，Diva Server 卡还能支持TCP/IP 的数据访问，可以支 持 PSTN 56K V.90， GSM， ISDN 等各种通讯访问，在 上图中我们可看到Diva Server 系列卡还可充当远程访问服务器和Internet访问 服务器的功能， 由此使得企业的任何通讯要求都得到了很完美的解决；其实 ，在远程访问服务器市场上，有不少ISP将我们的Diva Server PRI卡用来充当 ISP接入服务器，WAP访 问服务器等；在使用上，我们的Diva Server 系列卡 也 非常灵活，每种卡在一 台机器上都可最多插4块(一台最多120个通路，最少2个 通路)，能适应各类企业的需求。

　　目前，Unified Message 应用在国内市场上还刚刚兴起，我们真诚地邀请 各类软件开发商，系统集成商跟我们合作，共同开拓这个市场，我们相信此类应用在不远的将来将在中国得到广泛的应用。
