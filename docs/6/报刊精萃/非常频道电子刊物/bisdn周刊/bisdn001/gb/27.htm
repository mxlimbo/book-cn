□...VOIP 企业应用解决方案

用户需求：我们公司是一个跨国公司，我们经常要和我们国外的同事联系，所以国际长途电话费非常昂贵。由于我们和国外总公司之间，已经有了一根数据DDN专线来进行数据传输，随着VOIP技术的日益成熟，我们想利用现有的 DDN国际专线，来打国际长途电话，以节省公司庞大的长途电话费开支。

Eicon博士：您的需求，也完全可用我们的Diva Server Voice VOIP网关来实现。采用Diva Server Voice 卡和您现有的程控交换机相连，然后连上您现有DDN专线上的路由器，您打向总公司的电话将不收任何费用。

方案结构图： http://www.bisdn.net/eicon/eicon44.htm

Eicon博士：从上图中，我们的Diva Server Voice 卡和您企业的程控交换机相连，对交换机作一个简单的配置后，使得您单位的职员在其分机上拨某个号码(如09)后，使交换机知道该电话要和您总公司联系，该拨号便传到和Diva Server Voice 卡相连的线上，我们的Diva Server VOIP网关便将该呼叫转化成 H.323的IP呼叫，同时我们的VOIP网关软件可设置该呼叫应传到哪个IP地址，我们可设置所有的该类H.323呼叫将通过路由器传到总公司的Diva Server VOIP网关(其实，总公司可以用别的标准H.323网关，Diva Server VOIP网关 能和业界大多数标准的IP网关进行互通)，总公司的VOIP网关接收到该H.323呼叫的电话号码后，将自动通过交换机转到相应的分机上，接通电话后，便开始了正常的语音通讯。所以，通过该方案，在拨号时，您只需拨上您所设置的前缀号码(如09)加上对方的分机号码，就可接通您国外同事的电话，无论你打多长时间的电话，将不会增加公司任何的电话费用。这应该是一个最经济的节省您电话费的方案。根据您的需求，本方案可以采用我们的Diva Server Voice 4BRI卡(8个并行通话)或Diva Server Voice PRI卡(30个并行通话，最多120个 并行通话)来进行通讯。
