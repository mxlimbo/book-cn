□...DW中的项目列表应用一览

　　类似于word中的编号列表和项目符号列表，在dreamweaver中也有五种形式的项目列表：无序列表、有序列表、定义列表、目录列表和菜单列表。下面就这五种项目列表进行对比和简要的解释：

-- 项目列表形式 HTML代码 备注
无序列表

  • 无序列表

  • 有序列表

  • 定义列表

  • 目录列表

  • 菜单列表
<ul>
<li>无序列表</li>


<li>有序列表</li>
<li>定义列表</li>


<li>目录列表</li>
<li>菜单列表</li>
</ul>

  • <ul type=disc/circle/square>则可选择全部列举项的标记符号为圆点、圆环或小方块。

  • <li type=disc/circle/square>则可选择这个及其後列举项的标记符号为圆点、圆环或小方块。

  • 将光标置入想改变标记符号的列举项中，选择菜单text>>list>>properties或按右键在快捷菜单中选择list>>properties

    ,在弹出的对话框中改变style的选项也可以改变列举项的标记符号。

有序列表

  1. 无序列表

  2. 有序列表

  3. 定义列表

  4. 目录列表

  5. 菜单列表
<ol>
<li>无序列表</li>


<li>有序列表</li>
<li>定义列表</li>


<li>目录列表</li>
<li>菜单列表</li>
</ol>

  1. <ol type=A/a/I/i/1>则可选择全部列举项的标记符号为大写字母、小写字母、大写罗马字母、小写罗马字母或阿拉伯数字。

  2. <ol type=A/a/I/i/1>则可选择这个及其後列举项的标记符号为大写字母、小写字母、大写罗马字母、小写罗马字母或阿拉伯数字。

  3. 将光标置入想改变标记符号的列举项中，选择菜单text>>list>>properties或按右键在快捷菜单中选择list>>properties

    ,在弹出的对话框中改变style的选项也可以改变列举项的标记符号,改变Start Count的值则可改变项目列表序数的起始值。

  4. <ol type=a start=4>则项目列表序数会从d开始。
