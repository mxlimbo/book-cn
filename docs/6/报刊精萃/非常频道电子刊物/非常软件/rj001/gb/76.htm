□...苹果新版Mac OS X问题不少

　　【ChinaByte综合消息】苹果公司为苹果电脑设计的最新操作系统Mac OS X是自从Mac苹果电脑于1984年面市以来，苹果公司对其电脑操作系统进行的首次彻底翻新，市场对它的期待值很高，Mac OS X预计在本月下旬问世。但是据消息人士透露，这套软件存在许多缺陷和不足，很难令电脑用户感到满意。

　　据介绍，Mac OS X最大的问题是不支持DVD播放和刻录，其次还有一些小的问题，如安装在Mac笔记本电脑上使用时则不支持休眠功能等。

　　安装了新版Mac OS X的用户如果需要播放DVD光盘，必须关机启动9.1版的苹果电脑操作系统才行。专家指出，仅这一项缺陷就有可能使众多苹果电脑用户对新版Mac OS X敬而远之。这位专家说：电脑用户一般的确很少用到DVD刻录功能，但是读盘还是经常要用到的。谁会愿意需要重新开机才能读DVD光盘呢？

　　消息人士透露，Mac OS X中存在的这些问题要等到今年夏季以后，苹果公司才会着手解决。届时，苹果公司将推出一个修正版本的Mac OS X(开发代号为Puma，美洲狮)，预装在新出厂的Mac电脑中提供给电脑用户。

　　苹果公司代表拒绝对Mac OS X的开发情况发表评论。(责编：赵秀芹）
