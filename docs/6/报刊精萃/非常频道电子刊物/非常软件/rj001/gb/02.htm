□...5分钟发Email1万封，Linux系统真好用

摘自：网易

　 　北海道日本电气软件和Thirdware于12月26日宣布，双方已共同开发出新的电子邮件发送系统"Fast Mail Appliance Server on Linux 1.0"。该系统的开发成功意味着通过电脑服务器设备以及在操作系统上采用Linux，就能够以较低的成本引入大规模电子邮件发送系统，5分钟便可发送1万封电子邮件。该系统将于2001年2月开始推向市场。

　　而原有的电子邮件发送系统要实现同样高的处理性能，必须使用UNIX服务器。而使用UNIX服务器势必将导致成本的飙升。对此，负责开发的Thirdware表示，"以2台UNIX服务器设备组成群集来管理3000～5000个用户帐号时，大约需要花费5000万日元"的费用。而使用比UNIX服务器成本要低的PC服务器，"若要发送5000封以上的电子邮件，PC服务器肯定会出现'死机'现象"(Thirdware)。

　　Fast Mail Appliance Server既有于价格上和低价位PC服务器对等的优势，又具备足以和UNIX服务器设备相媲美的大容量处理能力，因此，它不失为一种集2种服务为一体的电子邮件发送系统。
