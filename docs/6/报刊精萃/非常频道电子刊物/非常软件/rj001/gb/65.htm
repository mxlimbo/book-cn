□...微软发布Back Office Server 2000

　　本报讯 Microsoft公司日前发布了面向分支机构、企业部门和中型企业的软件包BackOffice Server 2000。该服务器软件包包含Windows 2000 Server、Exchanger 2000 Server消息接发、SQL Server 2000数据库、Internet Security and Acceleration(ISA)Server 2000、Systems Management Server 2.0和Microsoft Host Integration Server 2000等服务器软件的标准版。

　　软件包还包含简化防火墙配置和连接许可设置的Internet连接向导；提供软件包自动监视的Health Monitor 2.1；以及对系统问题进行智能分析和修复的Bayesian Trouble shooters。
