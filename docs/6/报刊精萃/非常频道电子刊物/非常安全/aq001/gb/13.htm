□...Microsoft 安全公告 (MS01-012) (MS,补丁)

涉及程序：

vCard

描述：

Outlook和Outlook Express中的vCard存在缓冲区溢出漏洞

详细：

Outlook 和 Outlook Express 中存在一个名为“虚拟名片”（vCard）的组件。发现该组件中包含一个未经限制的缓冲区，容易造成缓冲区溢出漏洞。

如果一个恶意用户构造一个包含恶意代码的虚拟名片，并发送给其他用户。当接收者打开该虚拟名片，将可能导致两种情况发生：一、导致邮件客户端程序崩溃，接收者需要重新启动客户端程序并删除存在问题的邮件来使服务恢复正常；二、导致邮件客户端程序在受影响的机器种运行任意的恶意代码。

注意：该漏洞的唯一限制是：接收者打开恶意的邮件。

受这个安全漏洞影响的组件是 Outlook Express 的一部分，而 Outlook Express 又是 IE 浏览器的一部分，因此，要想彻底修补这个安全漏洞，就应该升级浏览器，而不是仅安装用于电子邮件客户端程序的补丁。

受影响的系统：

Microsoft Outlook 98

Microsoft Outlook 2000

Microsoft Outlook Express 5.01

Microsoft Outlook Express 5.5

解决方案：

下载并安装补丁：

http://www.microsoft.com/windows/ie/download/critical/q283908/default.asp
