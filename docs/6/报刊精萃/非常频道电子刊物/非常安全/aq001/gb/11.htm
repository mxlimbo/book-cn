□...Microsoft 安全公告MS01-010(MS,缺陷)

涉及程序：

Skins

描述：

Windows 媒体播放器 Skins 文件下载漏洞

详细：

在 Windows 媒体播放器 7 引进了一个叫作"skins"的功能。Skins 是一个可下载文件，用户可以用它来改变媒体播放器的用户界面。Windows 媒体播放器的 Skins 安装在"C:/Program files/Windows Media Player/Skins/skin.wmz"目录下。

当用户从一个恶意的 WEB 站点下载 Windows 媒体播放器 Skins 文件（.wmz），如果该站点中的 .wmz 文件中包含了一段可执行的 java 代码，那么将会在 skin.wmz 目录下保存这段代码。利用该漏洞恶意的用户就可以在本地受影响的系统中执行任意命令。

受影响的系统：

Windows Media Player 7

解决方案：

下载并安装补丁：

http://www.microsoft.com/Downloads/Release.asp?ReleaseID=27961
