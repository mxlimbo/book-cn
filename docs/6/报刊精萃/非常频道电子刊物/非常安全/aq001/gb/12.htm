□...Microsoft 安全公告 MS01-011 (MS,缺陷)

涉及程序：

Windows 2000

描述：

向Windows 2000域控制器发送畸形的请求导致服务器拒绝服务

详细：

运行在 Windows 2000 域控制器中的核心服务存在一个安全漏洞。如果恶意攻击者发送特定类型的畸形的请求到域控制器，将可能导致服务器拒绝服务。当服务器接受到这种特定类型的服务请求，需要消耗大量的系统资源来进行处理。如果一个恶意的攻击者发送大量的畸形请求到受影响的服务器，将耗尽所有的 CPU 资源。这样控制域处理以后提交的请求将可能非常慢，甚至根本不进行处理。

如果攻击者停止发送畸形请求，服务将恢复正常。理论上利用该漏洞可以造成一个拒绝服务攻击，不过一般该攻击不会占用超过75%的可用 CPU 资源。已经正确登录到域中的用户将不会受到该漏洞的影响。如果域中有多个域控制器将会对受影响的系统起到负载均衡的作用。

受影响的系统：

Microsoft Windows 2000 Server

Microsoft Windows 2000 Advanced Server

Microsoft Windows 2000 Datacenter Server
