3

　　"姥姥，姥姥……"

　　还没进屋，张小开就嚷嚷开了。

　　土屋里一点动静都没有，要在往常，张小开的姥姥早就拄着拐杖颤巍巍地从屋子里出来迎接自己的外孙了。

　　"怎么回事？你姥姥不在家吗？"

　　白雪奇怪地问。

　　"我也不知道。"

　　张小开走上前去，推开了半掩着的门。

　　三人忐忑不安地走进光线昏暗的屋里。

　　"姥姥。"

　　张小开又喊了一声。

　　但屋子里一个人都没有。

　　"啊！"

　　白雪突然惊叫起来——她的脚踩着了一个软乎乎的东西。

　　是一只大黄狗，它躺在地上，一动不动，眼睛瞪得大大的，好像看见了什么特别可怕的东西，充满了恐惧。

　　"阿黄、阿黄，你怎么啦？"

　　张小开蹲下身子，将大黄狗抱在怀里，大声叫道。过去，每次他来的时候，大黄狗都冲他摇尾巴，将脑袋在他的脚脖子上蹭来蹭去，可现在，它却一动不动，再也不会叫唤了，张小开无法接受这个事实。

　　"它死了，好像是被吓死的。"

　　白雪让自己平静下来之后说道。

　　"你们看，这桌上还摆着碗筷，盘子里的菜和高压锅里的米饭都还没有吃过。给人的感觉好像是马上就要开饭的时候你姥姥一家就突然不见了。"

　　杨歌分析道。可是，人怎么会像水蒸汽蒸发一般突然消失呢？这无论如何也说不过去。

　　"怎么会呢？他们不会遇到什么不幸吧……"

　　张小开将阿黄放在地上，心里堵得慌，直想掉眼泪。

　　"不会有事的。"

　　杨歌把手放在张小开的肩膀上安慰道。但他的心里，知道这句话其实一点分量都没有。

　　"唉呀，你们看！"

　　白雪指着头顶的电灯叫道。电灯突然亮了，然后又灭了，接着又亮了起来，又灭了……一明一灭，好像有谁在拉电灯开关玩。

　　"走，咱们快离开这间屋子。"

　　杨歌当机立断，对白雪和张小开说道。

　　三个少年逃离了土屋，来到街上。这时，他们惊讶地发现，整个小镇的每座房子里的灯都在一明一灭着。

　　冥冥中的那个超自然生物正在向他们显示它的力量。刚刚发生的一切，只不过是这场恐怖游戏的序曲。
