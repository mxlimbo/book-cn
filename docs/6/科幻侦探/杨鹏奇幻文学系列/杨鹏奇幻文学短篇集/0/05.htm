狼人传说

　　一天上学的时候，张小开发现了一根狼毛，经白雪签定，是一根真的狼毛，也就是说学校里藏了一只可怕的狼。

　　从此以后，每天早上大家都会发现有些不寻常的地方：比如黑板被爪子抓破了、实验室里的兔子被挖去了心脏、教室一片狼籍……

　　三剑客对此进行调查……

　　在一个月圆之夜，他们听见学校实验室里传来了一声狼的嚎叫声，三剑客冲到实验室里，发现了一只狼，正在啃一只兔子……杨歌要袭击它，白雪制止了他。当月亮隐入云中时，那狼变成了人，竟然是他们的生物老师。

　　三人对此继续调查，找到了生物老师的儿子Ａ及其邻居，得知生物老师的生活非常不如意，因为没钱，妻子背叛了他，和一个大款跑了，在家中也让人瞧不起……极度的压抑使他变成了一只狼，但他非常的善良，从来不轻易伤人，所以每次变狼之前总是把自己反锁在屋里……

　　有一次，屋子不小心被一个教职工打开来了，失去理智的狼人冲到大街上伤人……三剑客阻止他……行人吓得四散奔逃……狼人在人群中觅见了大款和他的妻子，扑向他的妻子，大款舍其情人而去……就在狼人要伤害其妻时，一个美丽的女孩出现，挡在了狼人与其妻子之间……

　　原来，这个美丽的女孩是狼人的崇拜者，她给予了狼人许多的爱，在她的感化下，狼人变成了原来的人，与美丽女孩相爱了。

　　生物老师再也没有变成过狼人，他和美丽女孩过得很幸福，三剑客参加过他的婚礼。
