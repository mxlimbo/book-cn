第二章

　　L4殖民地群。

　　这个区域同样也充满了许多从小行星带运来的资源卫星。不过和L3殖民地群不同的是，这里的资源卫星较小（即使如此，它们的直径也有数百公尺之大），几乎每一颗卫星都有注册登记。只不过所有权都掌握在少数人的手里。

　　其中最具代表身份的就是，温拿家族的主人卡特尔·拉巴伯·温拿。

　　这位年仅十六岁的少当家，目前正在一颗几乎已经停止开采的废弃卫星上面。

　　透过蓝色的玻璃面罩望出去，眼前放着几具被排列整齐的棺材般的大型胶囊舱。总共有四具，里面装的是被誉为最强的MS——钢弹。

　　卡特尔不发一语，只是透过大型胶囊舱的玻璃静静的看着钢弹的脸。在这个区域之内，虽然还是要戴上头盔才能呼吸，不过多少还是残留了一些空气。也正因为如此，太空衣内的自动净化功能才能正常运作，让他能够长时间停留（理论上，几近于无期限）。

　　卡特尔像是要试验它的界限一般冲动了起来。不过并不是要测试太空衣的性能。

　　在离开眼前躺在"棺材"里的战友的那一瞬间，他忍不住又停了下来。

　　"和平的时代来临，钢弹只能功成身退了……"

　　这时，通讯机传来叹息的声音。卡特尔转头搜寻，和他同样穿着太空衣的迪欧·麦斯威尔就在不远的地方漂浮着。

　　在玻璃面罩下，他的笑容依然是那么开朗而熟悉，可是眼神中所流露的忧伤，还是道出了他对钢弹的依恋。

　　就在数月之前，曾发生了白色獠牙的残党打算劫持被藏匿的钢弹的事件。

　　由于号称天下无敌的钢弹的存在，让恐怖分子产生了"拥有钢弹就是胜利之保证"的幻想，所以才会发生那次的劫持事件。

　　卡特尔首当其冲，遭到恐怖活动的波及。没多久，其它的钢弹驾驶员也纷纷被卷入了事件的漩涡之中。

　　幸好，在事态尚未演变到一发不可收拾之前，事件总算落幕了。

　　可是，只要钢弹存在地球的一天，难保以后不会再发生类似的情况。就因为这五架MS拥有高科技的性能，以致于随时都可能变成另一场战争的导火线。

　　经过劫持的事件之后，卡特尔不得已做出放弃钢弹的结论。等这个资源卫星结束开采作业之后，便将它们一并丢向太阳烧毁。由于这里距离太空殖民地太近，若是将它们丢着不管，说不定将来会受到引力的影响而撞击殖民地。丢向太阳只是防范未然的作法。

　　另一方面．从地球上中东民族独特的宗教观来考量，太阳是万物之母，人类应该胸怀感谢，将属于她的奉还给她，或许这种终结的方式还能增加一点神秘的传奇色彩吧。

　　卡特尔和迪欧两人透过玻璃面罩互相凝视对方。卡特尔努力挤出笑容来回应迪欧。

　　他的确做到了，可是表情还是掩不住内心的遗憾。

　　迪欧喜欢卡尔特这种毫不掩饰的个性。他了解，就算再怎么依依不舍也是无可奈何了，况且这种结局并非卡特尔所愿。

　　"算了，别再说了！我们快把她们和废弃的卫星一起丢向太阳吧。"

　　"好……"卡特尔简单的回答。

　　这个时候，说什么都是多余的了。

　　两个人利用惯性飞向天井附近的铁柱，并牢牢的抓住。要离开卫星就必须从这个地方出去才行。

　　出口处停了一艘小型的太空梭。

　　迪欧与卡特尔打开出口的气闸，很快地又将它关上。这次他们并没有再回头，因为他们不想表露出依依不舍的样子。

　　这不仅是对"战友"的一种尊敬，而且套句迪欧常说的话："要拿出男子的气概来！"

　　卡特尔按下舱内的遥控器。

　　装设在资源卫星上的推进器开始冒出火光。喷嘴所射出的力量将资源卫星缓慢的向前推移。目标——在远方散发出白色光芒的太阳。

　　从太空梭的窗口目送资源卫星逐渐远离，卡特尔和迪欧的内心百感交集。

　　明知在太空中不可能听得见推进器的声音，但是他们还是不想错过任何可能听到的细微的声响。

　　他们各自低语着——

　　"再见，我的沙漠钢弹……"

　　"别了，伙伴……"

　　当然，这些别离的话语是无法传到资源卫星里的"战友"了。

　　资源卫星里躺着四架钢弹。

　　飞翼钢弹O式——型号·☆☆☆

　　G-00W0　　地狱死神钢弹——型号·☆☆☆

　　G-01D2　　重武装钢弹改——型号·☆☆☆

　　G-01H2　　沙漠钢弹改——型号·☆☆☆　　

　　G-01SR2　　本来第五架钢弹，型号·☆☆☆

　　G-01S2的二头龙钢弹也应该在里面的，可是驾驶员五飞却说什么也不肯轻易地放弃他的分身——"哪吒"。

　　"五飞还是没把她带来……"迪欧感慨的说。

　　"嗯，希洛和特洛瓦倒是很干脆的就送来了。"

　　卡特尔这么回答，眼睛还是没有离开渐行渐远的资源卫星的光点。

　　"不晓得大伙儿现在怎么了……"迪欧怀念地微笑着。

　　算一算，他们几个人也有好几个月没见面了，或许这辈子永远不会再叙了吧。毕竟，每个人都有自己的生活方式。

　　要大家再度共聚一堂，恐怕也只有等到……

　　迪欧急忙挥去脑海的想法。虽然只是随便想想，不过也够令人厌恶了。

　　（我想到哪里去了！人类好不容易才进人和平岁月呀……）

　　没错！要这些具破坏性的钢弹驾驶员再次聚首，就只有等战火重新燃起，人类的自由与和平受到威胁的时候了。

　　而且，出乎迪欧意料之外的，那个时候就在不远的未来了。

　　因为地球圈统一国家情报部"预防者"所发现的Neo·钛尼姆破片，就是即将把人类的未来带进混饨黑暗时代的"不祥的第十三号星座"的零件。
