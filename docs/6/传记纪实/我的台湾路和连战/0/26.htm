我的外祖父赵祥霖

　　「我外祖父叫赵祥霖，字润生，陕北米脂人。说他是个高人，一点也不为过。他是个大儒，身为杰出的教育家、中医专家和玄学大师。他不为官、不为名、不为利，一直过着隐居的生活。」
　　
　　陕西北部处于黄土高原地带，是中华民族古代文明的发祥地之一。
　　
　　这里物华天宝，人杰地灵，有中国历史文化名城延安，有「塞上江南」的榆林，有在崇祯十七年率百万大军东渡黄河，长驱北上，包围燕京，彻底摧毁了明五朝统治的闯王李自成的故乡米脂。
　　这块土地悠久的历史，特殊的地理位置，构成了雄奇壮丽，粗犷豪放，淳朴感人的民风：
　　
　　青线线那个蓝线线，蓝个英英采， 　　
　　生下一个蓝花花，实实的爱死人。 　　
　　玉谷子那个田苗子，数上高梁高， 　　
　　一十三省的女儿数上蓝花花好。 　　
　　正月里那个说媒，二月里定， 　　
　　三月里交大 钱四月里迎。 　　
　　三班子那个吹来，两班子打， 　　
　　撇下我的情哥哥，抬进了周家。 　　
　　蓝花花那个下轿来，东张西又照， 　　
　　找见周家的猴老子，好象一座坟。
　　你要死来你早早地死， 　　
　　前晌你死来后晌我蓝花花走。 　　
　　手提上那个羊肉怀揣糕， 　　
　　我冒上个性命往你家跑。 　　
　　我见我的情哥哥呀说不完的话， 　　
　　咱们俩死活哟长在一搭。
　　
　　1892年阴历四月五日，我外祖父出生于「兰花花」的故乡，一个传统于渔、读、耕、樵的书香门弟。其父怀治翁为前清秀才，功名增生，并在县试中获案首。博学多才，熟研经史又潜心中医、阴阳之说。
　　
　　我外祖父自幼就在这种环境中受到启蒙，熟读四书五经，善书会写，以「神童」之誉名传乡里。当时因为世事变幻，科举废除，他选择了米脂县高等学校，师从于陕北大学究李鼎铭先生，后来他与李鼎铭先生宜师宜友，来往甚密。在米脂县高等学校毕业后，创办了龙镇高等小学堂，他亲任校长。由于他治校有方，广揽社会名士讲课，龙镇高等小学堂名噪陕北，一时称盛。不但本县的弟子纷纷报名入学，邻县横山、榆林、子洲、绥德的学子也远来求学。
　　
　　我外祖父不但在精于教育，同时中医和玄学更是一代高人。培养出一大批革命志士和学者专家，中共高级干部中就有高岗、汪洋、李成远等。中级干部及学者专家有一百多人。但他一直隐居乡里，究竟是他理性的执意维护自己的天性，还是天性让他无意之中必然做出此选择，这是无法知晓的事。不管怎么说，在他生活的时代，他走着一条别人无法挤上的路。
　　
　　我对外祖父的经历了解不多，但我自己在治学及传统玄学方面的入门完全是得益于他老人家。
　　
　　连战曾生于黄城、长于黄城，所以听我讲陕北的人事，非但不陌生，而且是饶有兴趣，他又问：「你外祖父曾指点过毛泽东？」
　　
　　「是，我曾听我母亲谈过一些，这要从我外祖父的学生高岗说起。」
　　
　　高岗原名叫高崇德，一九○五年春出生在陕北横山县武镇一个家境贫困的农民家庭。他从小放过羊，打过柴，到了上学的年龄他与其兄高崇义慕名前往离家六十里地的龙镇高等小学堂就读。高崇义为人老实，不善多言。在我外公的眼里，高岗调皮活泼，但非常聪明而有胆识，认为他以后必成大器，出将入相之才。所以在生活上给予接济，在学业上严加管教。一有空闲，我外公就喜欢将高岗拉在身边，给他特别的教导，引导他去闯天下，他也将校长也就是我外祖父视为父辈，对我外祖父的话十分听从，直到他以后出任中共中央束北局书记以及中央人民政府副主席时，遇到不能决策的事情，还立即找我外祖父。
　　
　　高岗长到二十一岁时找到共产党，于1926年夏天加入了党的组织，并接受派遣到西北从事兵运工作。后来在他的一再要求下，省委一度派他前往甘南作军事工作。他很快就在军事上显示出了他的卓越才华，于1930年后担任了中共陕西省委军委交通员。
　　
　　1932年7月初，来到陕甘边区与刘志丹一起举行了暴动，开展了轰轰烈烈的游击战争，领导了创建陕甘边革命根据地的工作、担任了中国工农红军第二十六军第四十二师政委。
　　
　　1935年初中央红车被迫举行大撤退。
　　
　　万里长征向陕甘边这边走来时，高岗已经是西北革命军事委员会副主席、前敌总指挥部政委。到是年九月红十五军团成立时，他又担任了军团政治部主任。在全国革命根据地遭到重创，几乎全部丧失时，只有陕甘边根据地蓬勃发展，成为中央红军和中共中央的归宿地。这是颇使高岗引以为骄傲和自豪的事。
　　
　　高岗也经常向部属的同志们津津乐道的说「陕北救中央」、「同时也救了毛泽东」等故事。
　　
　　当时中央红军已经面临生死存亡的关口，我祖父告诉高岗，中央红军只有进陕北，才能扭转乾坤，以毛泽东和蒋介石的命理来分析，蒋介石乃青龙转世，有为王之名，但气数转落，必定是退走海外，而毛泽东是黄龙出世，只要获得黄土高原的息养，必定为一开国之君。高岗采纳了我外祖父的意见后，迎接了中央红军。
　　
　　毛泽东率领的中央红军到达陕北后不久，高岗就将毛泽东介绍给我外祖父认识。毛泽东常与我外祖父促膝谈心，请我外祖父点拨时局和运势。
　　
　　对我外祖父的指点，毛泽东是非常感谢。他在陕北时期，亲自提拔高岗担任中共陕北省委书记、陕甘宁保安司令部司令员。中央书记处开会时，毛泽束微笑着指着高岗说：「我们初到陕北，人生地不熟，高岗同志为中央机关的安全出了大力，将来在保安树碑时，要给高岗同志树一块最高最大的碑。」
　　
　　很快，抗日战争爆发了，毛泽东又提议高岗担任中共陕甘宁边区委书记兼保安司令部司令员，中共陕甘宁边区中央局书记，中共中央西北局书记，陕甘宁边区参讥会议长，陕甘宁晋绥联防军副政委，代理政委，是毛泽东为首的中共中央眼鼻子底下的一员大将。
　　
　　毛泽东离开陕北后，直到进京和开国大典，一直听取我外祖父的建言，也与我外祖父保持书信来往。
