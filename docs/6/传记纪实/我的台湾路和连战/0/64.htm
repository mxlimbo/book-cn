连战出任行政院院长

　　同时，朱婉清也来了电话，告诉我连战出任行政院院长，这次她的电话非常简短，最后她冷冷地问了我一句：「大师，什么时候来台北？」
　　
　　她可把我问住了，我什么时候来台北？我不是「来」台北，我是「回」台北，我归心似箭啊！
　　
　　但这时候我有急事需要前往泰国，待泰国办完事，立即就回到了台北。
　　
　　连战已经顺利出任行政院院长，现在心目中的唯一大事，就是我的爱人杜文。
　　
　　当我抵达台北机场的时候，杜文前来接我，小别后的聚首，杜文的热泪蕴涵着我们的千言万语，我们终于熬到了这天。
　　
　　第二天，我就随朱婉清去了行政院院长连战的府上，一进他家，我看到玄关内的桌上放着一大丛兰花，桌下摆放着一盆水，这盆助连战走马上任的甘露，就是我在新加坡电话中要求朱婉清转达的指令。
　　
　　我向朱婉清说了最后一道指令：在连战家入门玄关口，桌下放一盆水；在朱婉清家供奉的弥勒佛前也放一盆水。
　　
　　这天连战家有很多朋友到访，连战过来给我打了个招呼就去应酬其它朋友了，但从他家人的眼神中我察觉到了阵阵阴云，这阵阴云并不是笼罩在连家，而是连战过河后拆桥的信号，是即将向我以怨报恩的阴云。
