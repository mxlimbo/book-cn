二十七、鹦鹉——老资格的地监哨

１．档案摘抄

　　鹦鹉，俗称“鹦哥”，“八哥”。鸟纲，鹦鹉科。种类甚多。

　　头圆，嘴强大，上嘴弯曲。羽毛色彩华丽，有白、赤、黄、绿等色。舌肉质而柔软，经训练，能模仿人言的声音。主食果实。多栖息热带森林中，营巢于岩洞或树穴内。分布于美洲、澳大利亚和我国南部等地。

２．学舌冠军

　　从古到今，鹦鹉学舌的出色本领，引起人们的莫大兴趣。

　　相传唐代时，长安富豪杨崇义在家中被杀，地方官到他家中踏勘，一只笼中鹦鹉忽然开口说话，念叨一个叫“李弇”的姓名。地方官心生疑云，一查，李弇是杨家邻居，便带来盘问，果然是凶手。鹦鹉因报案有功，被唐明皇赐了个“绿衣使者”的封号。这个故事见于笔记《开元天宝遗事》。

　　类似的事在国外也有发生。１９８４年３月，美联社曾报道一则新闻：在美国得克萨斯州的贝敦，某人家夜晚被撬窃。受害者报告警察说，他家被盗窃时，有一只鹦鹉在场；被盗以后，鹦鹉不断重复这样一句话：“到这儿来，罗伯特，到这儿来，罗尼。”根据鹦鹉提供的这两个名字，加上从现场取得的指纹，警察很快就破了案，抓住了两个惯窃犯，一个名叫罗伯特，一个名叫罗尼。

　　更有趣的是一位叫朱·海特的英国妇女饲养的一只鹦鹉。１９８０年的一天，这只鹦鹉在树林中飞迷了路，被一个农民捉住。鹦鹉到农民家后，反复念叨一个六位数字。农民感到很奇怪，他试着按这个数字拨电话，果然找到了鹦鹉的女主人！

　　美国动物生态学家佩珀伯格女士训练了一只原产于加蓬的灰色鹦鹉——著名的亚历克斯，它能说出主人教的物品的名称，准确率达到８０％。有一天，佩珀伯格带着亚历克斯去见兽医：接受肺部手术。当她把亚历克斯留在医院要离开的时候，它却说：“你过来。我爱你。我后悔了。我想回家。”显然，这只鹦鹉认为自己犯了某个错误而被主人抛弃了。但令人费解的是，它是怎样在这个非常恰当的时刻把它平时单独学会的这４句话连起来一下子说出来的呢？

　　１９８１年，美国曾举行过一次别开生面的动物“说话”比赛。赛场上，数千只各色鸟儿竞相学舌，一只名叫普鲁德尔的非洲灰鹦鹉夺得冠军。它一口气“说”了１０００个不同的英语单词。

　　有一只俄国鹦鹉能用英语背诵莎士比亚的第八首十四行诗，还能用英语朗诵普希金的一首诗。

　　在下面的故事中，鹦鹉已不单纯是在“学舌”了。罗丝到法院提出与丈夫卡洛斯离婚，理由是丈夫有外遇，根据则是家里的鹦鹉波兹开始模仿一个女人的尖笑，还学会了一些新词。波兹被带到法庭作证。法官向它出示一个涉嫌进过卡洛斯住室的年轻美人的照片。“亲爱的，我爱你！”波兹尖叫道。法官问：“谁爱卡洛斯？”波兹回答：“鲁比爱卡洛斯，鲁比爱她的宝贝。”（鲁比是卡洛斯的秘书）。法官判定波兹的证词成立，并准予其主人离婚。

３．“防空兵”改行当“警官”

　　对于好“学舌”鹦鹉，大概没有人不知道。但对它的从军经历，就未必都能说得上来了。在人类航空史的早期，鹦鹉就当过“防空兵”。

　　由于鹦鹉十分灵敏，经过严格训练后能及时报警。在第一次世界大战期间，尚未发明雷达，德军经常空袭法军，使法军损失惨重。后来，法军就把训练过的鹦鹉放在埃菲尔铁塔上，让鹦鹉充当侦听德军飞机空袭的哨兵。当德军飞机从远处起飞，人们还未察觉时，鹦鹉就已敏锐地听到低微的马达声，立即飞回法军指挥部报警，使法国人民及时躲避了一次次空袭。

　　现在有了雷达等先进的探测手段，在防空作战中自然不必再烦劳鹦鹉了。不过现在鹦鹉又有了新的职业。在美国洛杉矶警察部队中，有一个奇特的“警官”，是一只叫皮尔特的鹦鹉。它佩戴着当地警察署颁发的警官证，在街头执勤。它的职责是提醒孩子们在穿越马路时注意安全。
