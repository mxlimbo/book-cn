□...微软发布小企业应用软件包

网络世界

　　本报讯 Microsoft公司日前发布了Small Business Server 2000小企业软件包，它是面向有50台以下PC的小企业的集成软件包。Small Business Server 2000是一个集成了Internet接入、电子邮件、协同计算、共享打印和数据管理的软件包。

　　软件包包含带SP1的Windows 2000 Server操作系统、带Outlook 2000客户应用的Exchange Server 2000、Internet Security and Acceleration Server 2000、SQL Server 2000和FrontPage 2000。此外它还集成了传真和调制解调器共享服务。
