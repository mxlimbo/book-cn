□...选择一款称心的图像浏览器

挑挑拣拣看图片


　　现在各种各样的图像浏览器琳琅满目，而且功能都比较丰富，选择一款适合自己的图像浏览器，能够使本来就异彩纷呈的图像世界变得更加绚丽多彩。不要犹豫了，下面就来选一款吧：)

简单易用的Easy Picture V3.01

　　在Easy Picture V3.01中，可以实现对浏览图片进行旋转和镜像操作，并且保存为一幅新的图片。EasyPicture较其他同类软件还提供了很多丰富而且实用的功能，比如说可以实现拷贝当前屏幕图像并保存下来，允许在一行同时显示15幅图片等。Easy Picture还提供了一项非常特别的功能，就是可以针对当前图片进行亮度、对比度和光线效果的调节，而且还可以对图片进行边界范围修饰，以及图像的偏移量调节。另外，还可以选择当前图片任意范围，实现复制，而且其操作又非常简单。

　　如果你对图像编辑没有特别的要求，大部分的工作是图片浏览，那么Easy Picture无疑是个得力的小助手。

小巧玲珑的JPEG File Explorer V1.5

　　说JPEG File Explorer小巧玲珑是有道理的，因为它的体重只有527KB。JPEG File Explorer是个与众不同的图片浏览器，它采用与Windows资源管理器类似的管理界面，只要在左侧的目录区域找到存放图片文件的路径，在右侧的预览窗口就会显示当前目录下的所有图片文件。JPEG File Explorer支持AVI、BMP、GIF、ICO、JPG、TIFF等图像格式，并可以轻松实现在BMP、GIF和JPG之间进行格式转换。在使用JPEG File Explorer的时候，图像文件可以采用缩略图方式显示出来，而且还可以复制到剪贴板上，支持拖拽操作。除此以外，还提供了幻灯片播放，图像文件的复制、移动、更名、删除等常用的文件管理功能。

　　如果平时经常从网上下载图片的网友，可以使用JPEG File Explorer来进行图片文件的浏览和一些简单的文件操作等。

快速浏览器
PolyViewV3.41

　　PolyView支持众多的图像格式，而且可以实现全屏幕浏览、幻灯片播放等一些图像浏览器最常用的功能。另外，在PolyView中，还提供了一些简单的图像处理功能，比如可以完成对当前图片的灯光效果处理，改变亮度、对比度，以及色彩校正等。在PolyView中，还提供了对图片进行滤镜效果的设置，加入特效，比如将当前图片设置成为浮雕效果等。除此以外，PolyView还提供了诸如翻转、镜像、快速复制、改变图像尺寸等多种功能。更加有意思的是，PolyView还可以制作GIF动画，另外还提供了和扫描仪的接口，可以直接从外设采集图片。

　　需要强大而且丰富图像浏览功能的朋友，不妨一试PolyView，它将带给你一个崭新的图像浏览空间。

功能强大的S-PicView V2.0

　　S-PicView是一个简单易用而且功能也非常不错的图像浏览软件，提供了小图像的预览功能，可让使用者更容易找到所要的图形文件。S-PicView支持读取包括BMP、I?CO、JPG、GIF、JIF、PGM、PNG、TGA、TIF/TIFF、XIF、WPG等多种图像格式，而且还提供了图像的一些编辑功能，比如可以基于图像的三原色进行调节，还可以设定图像的锐化程度、明暗度等。另外，32-bitS-PicView还可以针对当前图像改变色深、加入滤镜效果、旋转图像、镜像、定义图像尺寸，以及已经转换RGB格式图像到灰度效果等。

　　S-PicView是一个功能完善而且强大的图像浏览器，非常适合需要对图像格式转换和一些简单处理的朋友使用。

最佳选择
ACDSee3.1

　　ACDSee是目前最流行的32位图像浏览器，它的主页地址是www.acdsystems.com/products/acdsee/，可以以关联方式快速打开图像文件，支持图像文件的放大与缩小、图像快速跳转，可以自动适应屏幕窗口的大小，提供图像的全屏幕及幻灯片显示方式。它支持直接调用外部程序来实现对图像文件进行编辑等操作。ACDSee不但提供了强大的图像浏览功能，而且提供了一个优秀的图像文件管理器，它的图像浏览界面，可以在类似Window资源管理器界面的窗口中对图像文件进行各种操作，比如可以注释图像、图像文件的格式转换，以及进行复制、剪贴、删除、更名、查找、转存、打印等普通的文件操作。另外，ACDSee还提供了支持数码相机的接口，可以方便地查看、转换数码相机的Kodak图像格式。

　　ACDSee拥有最完善和强大的功能，在图像浏览器中应该是众多朋友的最佳选择。
