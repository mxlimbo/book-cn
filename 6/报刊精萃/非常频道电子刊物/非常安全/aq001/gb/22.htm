□...Microsoft 安全公告(MS01-015) (MS,缺陷)

涉及程序：

IE

描述：

Microsoft IE中的多个安全漏洞

详细：

在 IE 安全策略中提供一个高速缓存装置，用来储存用户本地机器中需要下载和处理的文件。设置该缓存是为了使保护高速缓存区中文件内容的物理地址。这样，用户使用 web page 或 HTML e-mail 访问信息时，将通过正确的 IE 安全设置。

发现在这种IE安全机制中存在一个漏洞，攻击者可能通过 web page 或 HTML e-mail 来监听高速缓存区中文件内容的物理地址。通过发送恶意的信息，攻击者可以在本地计算机中打开高速缓存区中文件内容，这样攻击者通过编辑 HTML 帮助文件(.CHM)来执行任意代码。

该补丁另外修补了下列三个漏洞：

1）安全域认证漏洞。该漏洞能够引起一个恶意的网站管理员非法读取访问该站点的用户计算机中的任意文件。

2）Windows Scripting Host 中存在一个与安全域认证漏洞类似的漏洞，不过该漏洞需要通过 IE 来利用。

3）通过 IE 调用 Telnet 存在一个安全漏洞。设计中，用户可以通过 IE 来调用 Telnet。但是在 Windows NT 4.0 或 Windows 2000 中，如果 Telnet 客户端是作为 Services for Unix (SFU) 2.0 的一部分安装，这样 WEB 站点指定的任意命令行选项都可以通过 IE 来调用 Telnet。SFU 2.0 中的 Telnet 客户端提供一个选项，逐条的创建一个 Telnet 会话日志。如果攻击者利用该选项来启动 Telnet 会话，那么当用户启动计算机时，将会自动执行日志所记录的信息。

受影响的系统：

Microsoft Internet Explorer 5.01

Microsoft Internet Explorer 5.5

Microsoft Windows Scripting Host 5.1

Microsoft Windows Scripting Host 5.5
