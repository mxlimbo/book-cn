Part II

Windowed Applications

In Part I, we concentrated on Microsoft Visual J++ console applications. These types of programs are okay for accomplishing simple tasks. Modern programs, however, require the use of colorful windows that use sophisticated controls.

In this section, you'll learn how to create windowed applications. Chapter 4 covers non-platform-specific windowed applications. Because there is only so much you can do using generic techniques, the balance of Part II is devoted to Microsoft Windows-specific graphical user interface (GUI) components.

By the time you've finished Part II, you'll be in a position to write sophisticated and professional Windows-based applications.
