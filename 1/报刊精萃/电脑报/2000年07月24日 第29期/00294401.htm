《FOXMAIL 3.1 BETA 新鲜出炉》

　　Foxmail曾被许多电脑报刊及软件下载网址授予十佳共享软件、五星级软件等殊荣，其受欢迎的程度由此可见一斑。2000年3月1日，Foxmail以1200万元的代价加盟博大互联网公司。这曾令许多用户对Foxmail的前途感到担心，不过事实证明这种担心完全是多余的。全新的Foxmail 3.1 Beta已经在张小龙先生和博大公司的共同努力下于2000年7月17日诞生了（正式版将在大约1个月后问世），本人在新软件问世的当天就对它进行了下载、试用，结果令人相当满意。新版本又在老版本的基础上增加了许多功能，如自定快捷工具栏、多内码的自动识别、全面兼容Outlook Express等，所有这一切已经使它超越了国内外同类软件的水平，我们甚至可以毫不夸张地说Foxmail已经成为了广大国人收发电子邮件的最佳选择。现在就让我们一起来看看Foxmail 3.1 Beta到底有哪些新增功能吧。
　　一、自定快捷工具栏
　　大家都知道，资源管理器、IE、Word、Excel等软件都提供了自定义工具栏功能，广大用户可根据自己的需要对快捷工具栏上的命令按钮进行添加或修改，从而满足个性化的需要。如今，Foxmail 3.1 Beta也提供了这一功能，它允许广大用户按照自己的使用习惯任意增加或者删除系统提供的工具栏按钮，更能符合不同用户的实际需要。
　　要使用Foxmail 3.1 Beta的自定义工具条功能，我们只要双击工具栏或执行“查看”菜单中的“工具条设置”命令，打开“ToolBar Setup”对话框（如图1所示），然后从“分类”栏中选择不同的命令类型，系统即会将该种类型的所有命令按钮列表显示出来。广大用户若拟将某个没有显示出来的命令按钮添加到Foxmail 3.1 Beta的快捷工具栏上，只须使用鼠标将它拖到快捷工具栏的适当位置即可，而将快捷工具栏上现有的某个按钮拖离快捷工具栏，即可将它从快捷工具栏上删除（有关操作与Office的自定义快捷工具栏功能非常相似）。
　　另外，利用“ToolBar Setup”对话框下方的有关命令选项，我们还能对Foxmail 3.1 Beta快捷工具栏的显示状态进行必要设置，如是否显示按钮名称、按钮与文字说明的排列方式、是否采用浮动按钮、当鼠标停留在按钮上时，按钮是否由灰色变为彩色、是否采用透明按钮（无方框背景）等，广大用户可根据自己的实际需要加以设置。
　　二、多内码的全面支持和自动识别
　　大家都知道，在Foxmail 3.0 Beta2中，尽管系统提供了对BIG5等汉字系统的支持，但这一功能却并不能自动实现，我们每次收到港台朋友发送来的BIG5码邮件之后，还必须使用手工对它们的内码进行转换才能正常浏览，虽然不影响显示，但操作却略显繁琐。新版的Foxmail 3.1 Beta已经彻底解决了这一问题，它既增加了对各种双字节内码文字的支持（主要包括简体中文GB2312、日文Shift－JIS、韩文KSC及繁体中文BIG5），又提供了对这些内码的智能化自动识别功能，能够直接将最后的结果显示到用户的面前，再也无须用户手工进行转换了。此外，对于这些不同内码的邮件，我们还能正常进行打印，如果有相应输入法的支持，我们甚至还可以直接在Foxmail 3.1 Beta中撰写简体中文GB2312、日文Shift－JIS、韩文KSC及繁体中文BIG5等不同内码的邮件，这就极大地方便了用户与海外朋友的交流。
　　另外，在使用Foxmail 3.1 Beta的多内码支持和自动识别功能时，我们还应注意以下几点：
　　1.Foxmail 3.1 Beta的多内码支持和自动识别是以操作系统的多语言支持功能为基础的，也就是说只有当用户的操作系统具备了多语言支持功能之后，Foxmail 3.1 Beta才能自动识别不同双字节内码并予以显示，若操作系统不具备多语言支持功能，Foxmail 3.1 Beta的该功能也就无法启用。
　　提示：Windows 2000用户如果需要添加多语言支持功能，只需双击控制面板中的“区域选项”，打开“区域选项”对话框并选择“常规”选项卡，最后从“系统的语言配置”列表框中选择需要添加的语言模块即可；Windows 95/98用户若拟添加多语言支持功能，则必须下载微软公司相应的多语言支持包，然后直接运行这些多语言支持包即可达到目的。

　　韩文http:∥www.bodafox.com/download/Kopak.exe
　　繁体中文http:∥www.bodafox.com/download/Twpak.exe

　　2.目前，自动识别是按照概率的方法估算的，所以对于十个字以内的小文本，其识别可能不正确。广大用户如果能够确认信件的语种，直接选择相应语种的效果可能更好（选中需要识别内码的邮件，然后执行“邮件”菜单“字符转换”子菜单中的有关命令即可，如图2所示）。
　　3.目前此功能只能用于中文Windows 95/98以及Windows 2000的各种版本中。
　　4.除了自动识别汉字内码之外，老版本提供的手工转换汉字内码功能在Foxmail 3.1 Beta中依然存在，对于那些操作系统不具备自动识别功能的用户来说，我们仍可通过执行“邮件”菜单“字符转换”子菜单中的“GB→BIG5”和“BIG5→GB”命令实现GB和BIG5码的相互转换。
　　三、全面兼容Outlook Express
　　老版本Foxmail的兼容性的确不尽如人意，我们要想实现Foxmail与Outlook Express的资源共享还真得费一番功夫，不过这对Foxmail 3.1 Beta来说已经没有任何问题了。Foxmail 3.1 Beta对软件的兼容性进行了扩充，实现了与Outlook Express的全面兼容，无论是导入Outlook Express的账号、邮件，还是将自己的邮件、通讯簿导入到Outlook Express都不在话下，这就极大地方便了用户的使用。
　　提示：在Foxmail 3.1 Beta中导入Outlook Express信息的方法：
　　1.导入Outlook Express账号
　　所谓导入Outlook Express账号也就是将用户原来设置在Outlook Express中的用户名、密码、POP3等信息直接导入到Foxmail 3.1 Beta中，以便直接利用Foxmail 3.1 Beta收取这些账号中的邮件，免去用户重复进行账号设置的工作。为此，我们只须执行Foxmail 3.1 Beta“账户”菜单中的“导入Internet账号”命令，Foxmail 3.1 Beta即会自动对用户原来保存在Outlook Express中的账号进行分析，并添加到自己的账号列表中，然后我们就能在Foxmail 3.1 Beta中直接收取这些邮件了。
　　2.导入Outlook Express信件
　　注意，这里导入Outlook Express信件可不是将某一封邮件从Outlook Express导入到Foxmail 3.1 Beta中（该功能在老版本中已经实现），而是指将Outlook Express中的所有邮件一次性全部导入到Foxmail 3.1 Beta中。为此，我们只须执行“邮箱”菜单中的“导入Outlook Express邮箱”命令，系统即会将Outlook Express中的所有邮件全部“克隆”到Foxmail 3.1 Beta中，效率相当高。
　　3.导入、导出Outlook Express地址簿
　　在Foxmail的地址簿中，我们只须执行“工具”菜单“导入”子菜单中的“Web文件”命令，即可将Outlook Express的地址簿信息导入到Foxmail的地址簿中；而执行“工具”菜单“导出”子菜单中的“Web文件”命令，也可实现将Foxmail的地址簿信息保存为“.web”格式，供Outlook Express等其它软件使用。
　　由此可见，我们只须综合利用这几点功能，就能将Outlook Express的账号、邮件、通讯簿等内容通通转移到Foxmail 3.1 Beta中，然后我们就能在Foxmail 3.1 Beta中“重温旧梦”了，这就为彻底取代Outlook Express铺平了道路。
　　四、提供了打印预览的功能
　　Foxmail 3.1 Beta为了进一步方便用户对邮件的阅读、管理，特意在传统邮件打印功能的基础上又新增了打印预览功能，从而提高了Foxmail的“档次”。
　　此外，Foxmail 3.1 Beta的打印预览功能还支持多内码预览和打印、多种比例预览显示、可控制打印份数、打印邮件头信息等优秀功能，完全可满足广大用户的任何需要。
　　五、增加Foxmail整体转移功能
　　去年底，我为了迎接Windows 2000的到来而对计算机进行了一次“大扫除”――将所有有用信息全部备份在光盘上（当然包括Foxmail的整个目录），然后重新分区、格式化。可是当我将光盘上的Foxmail目录拷贝回硬盘时，它居然不能运行！当时可把我给吓坏了，后来才发现自己在还原Foxmail的路径时出现了一点小问题，原来是虚惊一场！这虽然没有给我带来什么损失，但它也暴露了Foxmail的不足。当时我就在想，要是Foxmail能像其他绿色软件那样随意移动就好了。没想到这一“幻想”居然成为了现实。Foxmail 3.1 Beta提供了整体转移功能，它允许我们将Foxmail目录整体转移到其他任何路径下，无须任何设置，程序即可正常运行，这就大大方便了广大用户的使用（该功能对那些C盘吃紧的用户来说尤其有用）。
　　六、其他改进
　　除了上面已经介绍的内容之外，Foxmail 3.1 Beta还提供了其他许多改进之处，如：
　　1.全面增加了鼠标轮浏览的功能；
　　2.增强了一些鼠标右键的控制功能；
　　3.重新设计了信息提示框；
　　4.可选择将Foxmail最小化时是否在任务栏上显示；
　　5.在程序托盘上增加了动态的图标提示；
　　6.收、发邮件时采用不同的动态图示；
　　7.增强了对Windows 2000的支持能力，如透明提示窗口等；
