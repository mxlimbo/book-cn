《 秘技锦囊 》

　　诺克斯(Nox)

　　游戏中，按F1调出控制台，输入‘Racoiaws’激活作弊模式，然后输入以下密码：
　　help cheat = 显示所有秘技
　　cheat ability = 重新设置用户能力
　　cheat goto {waypoint | x y} = 调到命名的地点
　　cheat health = 生命值全满
　　cheat mana = 魔法全满
　　cheat level ＃ = 跳关
　　cheat spells = 得到所有魔法
　　cheat gold ＃ = 增加金钱
　　set god = 无敌且魔法无限

　　模拟人生(The Sims)

　　进入作弊菜单
　　按住Shift, control, alt, c这四个键不放
　　然后输入以下密码：
　　KLAPAUCIUS = 1000美元
　　WATER_TOOL= 能在房子周围放水
　　SET_HOUR ＃ = 设定时间（1－24）
　　SET_SPEED ＃ = 设定游戏速度（－1000－＋1000）
　　INTERESTS = 看市民的个性和兴趣
　　AUTONMY ＃ = 改变市民想法 (0－100)
　　GROW_GRASS = 改变草的高度 (0－150)
　　MAP_EDIT ON = 编辑地图开
　　MAP_EDIT OFF = 编辑地图关
　　ROUTE_BALOONS ON = 打开基本教学
　　ROUTE_BALOONS OFF = 取消基本教学
　　LOG_MASK = 设定事件纪录
　　DRAW_ALL_FRAMES ON = 描绘所有帧
　　DRAW_ALL_FRAMES OFF = 取消描绘所有帧
　　TILE_INFO ON = Show tile info
　　TILE_INFO OFF = Hide tile info
　　SWEEP ON = Enable ticks
　　SWEEP OFF = Disable ticks
　　! = 重做上一个秘技
　　; = 分开多个密码指令
　　history = Dumps family history file
　　edit_char = Open Create－A－Character screen
　　lot_size ＃ = Change lot size to ＃
　　快速赚钱
　　开启秘技模式后, 输入klapaucius，然后再开启秘技模式输入 !;!;!;!;!;!;!;!;!;!
　　依此类推,最多可以重复60次

　　风色幻想SP: 封神之刻

　　不要做任何动作静待30秒～60秒，然后你将看到环形视窗向右旋转，不要做任何动作再静待30秒～60秒，然后你将看到环形视窗向左旋转，向左旋转后，使用键盘输入fy(会有一个提示音)sp(会有一个提示音)四个字母，然后你会听到一,秘技模式打开：
　　F2键 =等级不变，99级实力
　　F3键 =可以看见隐藏宝物
　　F4键 =可以直接得到全部的徽章
　　F5键 =我方的角色全灭
　　F6键 =敌方的角色全灭
　　F7键 =滑鼠游标所指的角色HP，MP，POW全部补满
　　F9键 =滑鼠游标指到的角色立刻死亡

　　热舞2000

　　叫出隐藏模式的方法
　　打开游戏目录中 sam.h 的文件，选NOTEBOOK, 找到这一段
　　ms_bExtraModeIsShowed=0
　　ms_bLastExtraModeIsShowed=0
　　ms_bJamModeIsShowed=0
　　ms_bLastJamModeIsShowed=0
　　把所有 =0 改成 =1，再存档就可以了
