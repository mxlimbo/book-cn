《 网上Linux乐园 》

　　Internet是Linux的诞生地，Linus Torvalds当初创建她时，就是在Internet上，在全世界各地的UNIX高手、编程奇才的帮助下，以Internet为联系媒介，由分布在世界各地的成千上万的计算机爱好者一起努力的结果，所以认识Linux，学习Linux和应用Linux的最好场所就是Internet。下面所收集到的资源，希望能对和笔者一样正在学习、使用Linux的读者们提供最大的帮助。

　　1.国内的专业Linux网站(GB)

　　中国自由软件库：http://freesoft.cei.gov.cn/，中国Linux的资源宝藏。
　　中国Linux论坛：http://www.Linuxforum.net/，国内Linux爱好者的一个相互学习和交流的场所。
　　Linux技术支持网站：http://www.Linuxaid.com.cn/，Linux在中国的正式网站，提供技术支持服务，有很多源代码可供下载，也有很多面向初学者的资料。
　　Linux中国：http://xchina.cLinux.org/，包括Linux程序下载，窍门，技术资料，内容丰富。
　　中国Linux组织的网站：http://www.Linux.org.cn/，它们已经有了自己的 PLinux 。
　　Linux中文世界：http://www.cLinuxworld.com/，中文门户网站，设有论坛、新闻栏目，还提供主页空间。
　　中文GNU/Linux网站：http://www.cLinux.org/，收集了各类Linux中文资料。
　　中文Linux计划：http://lark.ihep.ac.cn/，中国Linux的先驱――陈向阳和方汉老师负责维护该网站，内容包罗万象，有中文软件，软件仓库等。
　　Linux中坚站：http://Linux.arecool.net/ ，有很丰富的中文分类文档。还有论坛，编程等。
　　Linux操作指南：http://Linux.ccpi.gov.cn/Linux/，有很多Linux安装、管理、释疑和指南类文章。
　　金山卓越Linux站：http://Linux.joyo.com:3000/，有很多内容，还可以打包下载Linux公报。
　　Linux晋江加油站：http://home.jjinfo.com/personal/Linuxboy/，Linux发展介绍及资源荟萃。
　　中文Xteam Linux的网站：http://www.xteamLinux.com.cn/
　　中文TurboLinux的网站：http://www.turboLinux.com.cn/

　　2.国内的专业Linux网站(BIG5)

　　香港Linux用户协会：http://www.hklug.org/，了解香港Linux的好网站。
　　Linux@Taiwan：http://www.Linux.org.tw/，台湾的Linux爱好者组织，CLDP的家，列有翻译的Linux HOWTO 中文版。
　　Linux in Information Networking：http://Linux.inhk.net/，包含新闻、初学入门、技巧、资源和讨论等。
　　中文化Linux计划：http://chinese.Linux.org.tw/，为台湾中文化Linux计划(Chinese Linux Project)正式网站。
　　Chinese Linux Extensions Website：http://cle.Linux.org.tw/CLE/，中文Linux扩展(CLE)，基于Red Hat和KDE的中文解决方案。
　　LinuxFocus在线杂志中译版：http://ccss.cc.nccu.edu.tw/
　　台湾中原大学Linux Server资源：http://evon.el.cycu.edu.tw/，列有Linux简介及操作说明。
　　探索 Linux：http://www.tvds.com.tw/Linux/，定期推出 Linux 应用的相关文章。同时有线上讨论区。
　　Linuxer Magazine：http://www.Linuxers.com.tw/，一本不错的Linux网络杂志。

　　3.国外Linux发行版本网站

　　RedHat：http://www.redhat.com/，大名鼎鼎的”小红帽"的网站，内容很多，也是搜索Linux各类资源的上好网站。
　　Slackware：http://www.slackware.com/，Slackware Linux的网站， 学习Slackware的好地方。
　　Debian：http://www.debian.org/，Debian Linux 的官方网站。
　　S.U.S.E：http://www.suse.com/，S.U.S.E. Linux 的官方网站
　　Turbo Linux：http://www.pht.com/，Turbo Linux 国外的官方网站。
　　Caldera：http://www.caldera.com/，Caldera 的官方网站。

　　4.国外著名Linux网站

　　Linux Online：http://www.Linux.org/，最权威的Linux网站， 文章讨论无所不包，软件硬件应有尽有。
　　Linux国际协会( Linux International )：http://li.org/，有大量的Linux资源列表。
　　Linux：http://www.Linux.com/，学习Linux的最好网站，也是Linux使用经验的汇聚地。
　　HowTo：http://www.Linux－howto.com/，查找Linux－Howto的最佳网站，包括了最新最全的Linux－Howto的各种版本。
　　Linux help online：http://www.Linuxhelp.org/，最大最全的Linux在线帮助网站。
　　Linux Sources：http://www.Linuxresources.com/，大而全的Linux网站，有新闻，软件，文档，支持等等，内容应有尽有。
　　《Linux 公报》：http://www.Linuxgazette.com/，著名的免费在线杂志，每月一期，介绍各种应用指南、技巧，是Linux用户交流中心之一，还可以提问题，绝对应该一读。
　　《Linux focus》：http://www.Linuxfocus.org/，免费电子杂志《Linux focus》，特色是每期集中讨论一个话题。
　　《Linux Journal》：http://www.Linuxjournal.com/，很棒的Linux专业杂志。
　　SUNSITE ：http://sunsite.unc.edu/Linux－source/，包含了几乎所有Linux分类的源代码。
　　Linux Box：http://Linux.box.ck/，资源全面的Linux宝箱。
　　Linux Headquarter：http://www.Linuxhq.com/，Linux总部，主要负责核心的开发工作。
　　Linux Support Line：http://www.Linuxsupportline.com/，Linux网上技术支持的网站。
　　Linux Source Driver：http://lsd.Linux.cz/，Linux驱动程序的源代码。 　　http://www.kde.org/，KDE的大本营。
　　Apache：http://www.apache.org，全球占有率最高的Web服务器：Apache的网站，学习它的最好场所。
　　Linux Kernel：http://www.kernel.org，Linux核心的网站，有着很齐全的Linux的核心的各种版本。
　　Sendmail：http://www.sendmail.org，Linux世界下最广泛使用的软件之一，这里就是它的老家。
　　希望本文所列出的这些Linux资源，能给正在学习使用Linux的读者们提供一些帮助。
（山西　冯彬然）

BBS资源

　　现在，几乎在所有的大中型BBS网站，尤其是各高校BBS，都开辟了Linux专栏或专区，以下并未将所有这些网站列出，只提供几个国内水平较高和上站人数较多的BBS网站。
　　http://bbs.tsinghua.edu.cn 水木清华BBS
　　http://bbs.ustc.edu.cn 上海交大BBS
　　http://bbs.sjtu.edu.cn 中国科技大学BBS
　　http://www.sina.com.cn 新浪网BBS
　　http://bbs.nease.net 广州网易BBS站

新闻组资源

　　同BBS一样，许多的新闻组网站也都开辟了Linux专栏或专区，下面只列出国内的几家优秀的新闻组网站。
　　中国自由软件库新闻组：news.freesoft.cei.gov.cn，由宫敏博士主持，人气很旺，Linux爱好者的交流好场所。
　　CLinux计划：news.cLinux.ml.org，是国内规模较大的新闻服务器，有关Linux的内容也不少，其中的cn.bbs.comp.Linux是各大学BBS站Linux版与News的连接。还有许多英文的新闻组。
　　中国科技大学新闻组：202.38.64.2，国内最大的Linux新闻组之一。
