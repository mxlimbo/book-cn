《 浏览器的错误信息 》

　　在网上浏览网页时常常会出现一些错误提示，现将一些网上常见的错误类型列举如下，希望对各位有所帮助。
　　401 UNAUTHORIZED：
　　表示你必须有一个正确的用户名称及密码方能浏览对方网页。例如浏览一些收费的网页就会出现这个信息。
　　403 FORBIDDEN PAGES：
　　所指向的页面实际存在，但只允许有特定权限的人访问，如果你有权限，却出现此提示，试着重新输入口令。
　　404 NOT FOUND：
　　最常见的出错信息，通常是要访问的页面不存在或被移动到别的地方去了。遇到这种情况时，在URL地址栏中，用后退键向前删，直到最近的一个斜杠线，再回车，以找到下级页面的链接。
　　500 SERVER ERROR：
　　这个错误通常由对方网页程序设计错误而引起。
　　503 SERVICE UNAVAILABLE：
　　你所要访问的页可能是存在的，但暂时不能访问，通常是系统忙引起，可稍后再试。
　　BAD FILE REQUEST：
　　可能在网页上填写表格的时候填入了不正确的资料，导致在处理资料的时候出现错误。你可以按“BACK”键返回，然后进行修改。
　　FAILED DNS LOOKUP：
　　用户所键入的URL位置不能被翻译成IP地址，这种错误通常是因为网站负荷过重。
　　FILE CONTAINS NO DATA：
　　所访问的页面是存在的，但当前文件为空白，通常由于网页制作者正在上传该页面引起，可稍后再试。
　　HOST UNKNOWN：
　　无法找到主机。有时一些站点移走了，却没有留下转移的链接。
　　NNTP SERVER ERROR：
　　如果你的网页浏览器不能找到新闻组的服务器，此信息便会弹出。原因可能是该服务器已关闭，或用户键入错误的服务器名称。
　　SITE UNAVAILABLE：
　　有四种情况可能产生这个信息：
　　（1）太多用户
　　（2）网站因维修而关闭
　　（3）电话线的噪音太大
　　（4）该网站根本不存在
　　TCP ERROR ENCOUNTERED WHILE SENDING REQUEST TO SERVER：
　　当网络传送一些不合法或不完整的资料时，就会产生这种错误，你可以过一会再尝试连接该网页。
　　TOO MANY USER：
　　这表示网站已经饱和，不能接受更多用户了，唯一的方法是过一会再访问。

　　(南京　小白)
