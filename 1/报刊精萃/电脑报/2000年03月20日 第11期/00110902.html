《 道听突说 》

　　不用突突说，本周最亮的软件明星也该是Windows 2000中文版，终于在今天看到了它的庐山真面目。虽然英文版早已问世，但作为专门针对中国人的版本，它依然吸引了我们所有的目光。突突唯一的愿望是这个中文正式版没有beta3版那么多的BUG。在微软的网站上，突突看到了这样的提示：“对于运行了Windows 98或Windows 95的家用电脑用户，微软强烈建议他们等待代号为“Millennium”的下一版消费型操作系统。”呵呵，那么多的家庭和个人用户，在期待着Windows 2000，却被告知要继续等待，是不是有点让人心烦呢？这个被称为视窗千禧版的操作系统，一直犹抱琵琶，但在微软的强烈建议下，倒是让人有了更多的兴趣。突突去网上寻了寻，却发现了这样一条消息，据报道，视窗千禧版缺少核心网络功能。微软悄悄地从其Windows千禧版客户操作系统中去掉了局域网功能，另外，这种操作系统还将不支持一个接入微软Active Directory的界面。这一举动使那些目前使用Windows9X平台的企业客户，看起来除了向Windows2000转移外几乎没有其它选择。有意思的是，微软还准备在2001年把目前彼此分开的商用和消费类操作系统统一到一种代号为“Whistler”的操作系统下，这种操作系统将使用基于NT架构的源代码。这些报道目前并未得到微软公司的证实，看起来的确有些有趣。到底该做何选择呢，突突当然会先试Windows 2000为快。消费型的操作系统？让我们期待着吧。其实大家应该还听到了另外一种声音，那就是目前只卖28元的Lindows，真诱人的选择。
　　突突今天好像没有写惯例的不着边的闲话，当然是因为要说的东西太多。在Word Perfect的免费下载策略奏效后，Corel公司准备推出可供免费下载的Linux版影像处理软件。 Corel称，Linux版的Photo－Paint 9将于今年夏初问世。Corel在Linux版的绘图软件产品开发上居于领先地位，但目前面对的最大劲敌就是随其它Linux版本软件赠送的Gimp，而且，擅长绘图软件的Adobe也开始设计Linux版软件，将来也势必是个强劲的对手。与此同时，Corel已开始测试Linux版Word Perfect Office 2000软件包，这套软件包括文字处理软件、个人数据库、电子表格、演示文稿软件以及Corel版的Linux操作系统。可以看到，Corel正在致力于将Linux转变成流行的PC机操作系统，所有的这些消息，对于钟情于Linux系统的玩家们当然是天大的福音了。 　　还有一个好消息，SUN公司已经宣布，SUN最近将公布一系列的程序源代码，这些程序包括Solaris操作系统、JAVA程序语言、广被人使用的Network File System（共享文件的网络软件）等。公布JAVA的源代码当然是希望替即将推出的JAVA更高版本造势，这两个新版本分别是Forte for Java Internet Edition与Forte for Java Enterprise Edition，预计将在今年下半年问世。SUN公司的发言人说得很有意思：“我们希望软件设计师充分利用Forte源代码，根据自己需要随心所欲改写。”
　　呵呵，写完道听突说的感觉很棒，因为突突又可以轻松一下了。哦，忘记了做一个大大的预告，软件版将会在下期推出一个详尽的Windows 2000专题，为你解决在Windows 2000安装、使用中遇到的各种问题。呵呵，一定要记得看了。看来突突又没得玩了。
