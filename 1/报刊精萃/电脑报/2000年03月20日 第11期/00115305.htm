《 双服务器和服务器网卡 》

　　笔者为很多单位改造过网络，现总结两种简单易行的提高网络质量的方法：双服务器和服务器双网卡。
　　一、双服务器
　　要求：工作站能运行装在任一服务器中的应用程序。
　　（1）两个服务器均为NetWare 3.12，实现资源共享，服务器名为Server1和Server2。 　　Server1、Server2各自用双绞线连到交换机或者HUB的RJ45端口上。
　　（2）服务器软件设置：修改Autoexec.ncf 　　1Server1的autoexec.ncf： 　　file server name server1 　　ipx internal net 1234 　　load c:\server 3.12\pcntnw.lan iop=8800 frame=ethernet－802.3 　　bind ipx to pcntnw net=abc 　　mount all 　　2Server2的autoexec.ncf： 　　file server name server2;(应不同) 　　ipx internal net 4570;(应不同) 　　load c:\server 3.12\pcntnw.lan iop=8800 frame=ethernet－802.3 ;(视网卡而定) 　　bind ipx to pcntnw net=abc ;(应相同) 　　mouat all 　　load monitor
　　(3)检查工作站和服务器的连接，从工作站以Supervisor登录，运行Syscon，在Available Topics下的“chang Current Server”选项，如工作站第一台服务器连通，则出现服务器名“Server1”，按下“Ins”键，若第二台服务器连接正常，则出现“Server2”。此时若选择“Use Information”，则出现两个服务器的名称和登录身份。
　　(4)修改系统登录原稿System Login Script 　　从工作站以Supervisor登录，运行Syscon，选Available Topics下的“SuperVisor Options”，再选择“System Login Script”，在文件前加入如下内容： 　　Attach Server1 /supervisor 　　Attach Server2 /supervisor 　　Map F:=Server1 /sys:Login 　　Map G:=Server2 /sys:Login 　　存盘后F为Server1映射的盘符，G为Server2映射的盘符。
　　重新启动两台服务器，以普通身份登录即可。
　　二、服务器双网卡
　　因为每一种硬件占用系统的一个port（端口）和int（中断），所以两块网卡设置的port和int应不同。
　　（1）设置网卡地址 　　用网卡的驱动程序，设置网卡的port和int，设其中的一块为port=300，int=3；另一块网卡为port=340，int=5。
　　（2）修改Autoexec.ncf 　　假设网卡为NE2000兼容的，Sn2000，TP－link、Accton等网卡，Load install在system option中的edit autoexec.ncf files中： 　　file server name fs 　　ipx internal net 12345 　　load NE2000 port=300 int=3 frame=ethernet－802.3 　　bind ipx to NE2000 net=a 　　load NE2000 port=340 int=5 frame=ethernet－802.3 　　bind ipx to NE2000 net=b 　　mount all 　　Load monitor 　　存盘退出。
　　把一部分机器连接网卡1上，另一部分连接网卡2上，即可。

(河南　陆凤兆)
