《 网上对弈何处去 》

{　　围棋是中华民族源远流长的国粹。
　　Internet推动了围棋在全世界的传播和发展，它打破了时间和空间的限制，使全世界的围棋迷都可以不分昼夜地享受围棋的乐趣。网上对弈已成为网络时代棋迷手谈的乐园。下面介绍一下Internet上的对弈场所。
　　国外对弈站点
　　国际互联网围棋服务器（IGS，Internet Go Server 的简称），http://igs.joyjoy.net，IP地址：210.134.78.91，端口：6969或7777。IGS是世界上历史最悠久、平均在线人数最多的对弈网站。它的程序功能和段位系统极佳。客户端软件是WINIGC，全英文界面，国内围棋网站大都有WINIGC下载。IGS的平均在线人数为500人，各种水平的棋友都可以找到适合自己的对手。IGS和由它衍生的NNGS(No Name Go Server的简称)类围棋服务器极大地推动了网络围棋在全世界的发展。
　　无名围棋服务器（NNGS），http://nngs.cosmic.org，端口：9696。NNGS在网络围棋界首创围棋打谱机器人，功能强大，它的网上名字叫GoBot。NNGS一般有20至30人在线，以欧美棋迷为主，棋力较弱，绝大多数是级位棋手。
　　传奇围棋服务器（LGS，Legend Go Server的简称），http://www.lgs.hinet.net，IP地址：210.65.1.245，端口：9696，属于IGS/NNGS类围棋服务器。LGS上也有一个打谱机器人。LGS有英文和中文（Big5）两种语言选择，有段位升降系统，是一个相当成熟的围棋服务器，但在线人数不是很多，一般晚上有20到40人，级别一般在二段以下。
　　互联网围棋（WWGO，World Web GO的简称），http://www01.tjsys.co.jp/，是目前世界上最大的JAVA对弈站点。提供会员服务。一般有60到70人在线，目前有英文版、日文版。
　　国际互联网围棋俱乐部（WING，World－Wide InterNet Gokaisho的简称），http://wing.gr.jp，端口：1515。他有中、英、日三种语言版本，是一个相当成熟的围棋服务器，但到该站下棋的人数较少。一般晚上在线人数在20到30人左右，棋力多为级位。
　　国内对弈站点
　　中国围棋网（CWS，China Weiqi Server的简称），http://www.gogame.cn.net，IP地址：202.98.15.15，端口：9696，是国内创建较早的围棋对弈站点。段位系统较好，在线棋手以K＊（打完20盘比赛获得系统确认的级位棋手）为主，有段位的棋手较少，一般在线人数为70人左右。中文客户端软件CWC，界面友好，操作方便，功能强大。
　　世界围棋网（WGS，Worldwide Go Server的简称），http://www.weiqinet.com，IP地址：202.98.15.25，端口：9696。WGS围棋服务器速度很快，运行稳定，段位系统做得很不错，系统的开放程度很高，下棋聊天的氛围很宽松，国内围棋服务器中独有的打谱机器人（GoBot或weiqibot）使你能够轻松自如地打谱、研究棋局等，还有一些自主开发的十分有趣的命令。一般20人左右在线，棋力参差不齐，有业余高段棋手，也有K＊棋手。
　　中国天元网（CTN，China Tianynan Net的缩写），http://www.weiqi.sh.cn，IP地址：210.96.210.159，端口：8888。CTN是国内在线棋迷人数最多的IGS/NNGS类围棋对弈网站，一般有200人左右在线对弈，高手也比较多。
　　通过Internet对弈，可以说是网络游戏中的一种。许多提供网络游戏服务的游戏站点均把围棋列为其发展的重点。
　　联众网络游戏世界，http://www.gl.net.cn。凭借着“联众”成功的市场运作和先进的技术（对弈服务器和聊天服务器分开等），赢得了许多网上棋迷。“联众”的界面非常简单，容易使用，但缺乏段位系统，而且对弈系统存在一些Bug，考虑得并不严密和周全，平均在线人数为750人左右。
　　中国游戏中心围棋站，http://go.chinagames.net，人气比较旺盛，平均130人左右在线。经常有职业棋手下多面打指导棋，这在国内首屈一指。客户端软件“围棋大决战”，界面一般，操作简便，摆棋的功能很强，支持多面打，升级较快。但他们的段位系统不够严密。
　　中国围棋网上棋院，http://www.weiqi.263.net.cn，是中国棋院今年建立的官方围棋网站，IP地址是：210.78.145.230。必须使用网上棋院提供的专门程序才能下棋，虽然这类软件提供了下棋的主要功能，但考虑得不周到，限制比较多，速度比较慢，下棋的人数很少。}
