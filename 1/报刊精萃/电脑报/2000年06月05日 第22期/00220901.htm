　　3.Office 2000 的剪贴画缓冲区溢出缺陷修补程序包
　　该“Microsoft 剪辑图片库”修补程序包解决了一个可能在“剪辑图片库”中导致缓冲区溢出的缺陷。缓冲区溢出可能导致“剪辑图片库”崩溃，或者让用户计算机容易受到恶意黑客创建的有害代码的攻击。恶意代码（内嵌于“剪辑图片库”下载的文件之中）可能对存储在用户计算机中的信息造成损害。
　　“剪辑图片库”和“在线剪辑库”（英文）使用了一种特殊的文件格式 ――CIL 格式 来下载剪辑。在某些情况下，内嵌于剪贴画 CIL 文件内的一个非常长的字段可能在“剪辑图片库”软件中导致缓冲区溢出。发生缓冲区溢出时，软件可能崩溃或导致任意代码在用户计算机上得以执行。该缺陷中的主要危险在于当用户打开一个附件（其中包含有特殊形成的剪贴画文件），或从恶意黑客的 Web 页上下载 CIL 文件时，将发生缓冲区溢出。
　　注意：Microsoft Works 2000、PictureIt! 2000（英文）、Greetings 99、Greetings 2000（英文）、Home Publishing 99 和 Home Publisher 2000 （英文）也受此缺陷制约。
　　该补丁的下载链接地址是http:／／officeupdate.microsoft.com／isapi／gooffupd.asp?TARGET=／china／downloaditems／office／CagPkg.exe
　　下载文件大小为293K。
　　4.Office 2000 ODBC 驱动程序安全性缺陷安全更新
　　加入了新的 IISAM 文件更新。 Microsoft 以前发布了一个对 ODBC 的安全更新，用来解决 ODBC 数据库驱动程序中的安全性缺陷，该缺陷会影响 Excel 2000 的用户。这一安全性缺陷同 ODBC 数据库驱动程序中的 IISAM 组件相关，并且利用这一缺陷在 Excel 2000 的查询功能下可执行类似于 Office ODBC 驱动程序安全性缺陷中所述的恶意操作。Microsoft 已经研究出对这一特定安全性缺陷的解决方法，并已将相应的 IISAM 文件加入到本次的更新版本中。
　　注意：Microsoft 建议所有 Office 2000 和 Excel 2000 用户都要用该安全更新来更新系统。对公司用户，我们建议你在使用任何更新程序或修补程序前先与你的系统管理员联系。
　　本修补程序需要用于 Windows 95、Windows 98、Windows NT 3.51 或 Windows NT 4.0。本更新适用于 Office 支持的全部语言。
　　该补丁的下载链接地址是http:／／officeupdate.microsoft.com／isapi／gooffupd.asp?TARGET=／downloaditems／JetCopkg.exe
　　下载文件大小为4765K。
