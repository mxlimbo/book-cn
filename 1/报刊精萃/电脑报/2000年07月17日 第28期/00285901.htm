《 AutoCAD 2000在建筑设计中的应用--模板的定制 》

　　笔者在使用AutoCAD 2000的过程中利用模板及可修改功能，将自己习惯使用的界面、命令以及常用的一些设置集中设定在user模版中，每次画图时只要调入该模板就不必再执行繁琐的状态设置，从而减少了大量的重复劳动，大大地提高了工作效率。在此，笔者以AutoCAD 2000安装到D:\AutoCAD 2000为例将定制user模板的过程介绍如下，希望能对大家有所帮助。
　　一、菜单文件acad.mnu的设定
　　1.利用“AutoCAD 2000 migration assistance”下的“menu ＆ toolbar porter”，可将原AutoCAD R14菜单(acad.mnu)中的内容加入到AUTOCAD 2000菜单(acad.mnu)内。
　　2.用编辑器打开D:\AutoCAD 2000\Support\acad.mnu文件，在pop11段落后增加pop12，＊＊lzj，在其下将自己常用的命令按前面的格式编辑好。这样，在下拉菜单“help”后，将增加“lzj”菜单项。
　　3.汉化菜单：将acad.mnu文件[]内的英文更换成中文。
　　4.快捷键设置：在acad.mnu文件的最后为快捷键设置，可按此格式将常用命令作成快捷键方式。例如：ID ZoomIn[CONTROL＋“Q”]。
　　二、简化命令的设置
　　1.利用“AutoCAD 2000 migration assistance”下的“Command Alias(pgp)porter”可将原AutoCADR 14菜单(acad.pgp)中的简化命令加入到AutoCAD 2000菜单(acad.pgp)内。
　　2.用编辑器打开D:\AutoCAD 2000\Support\acad.pgp文件，可修改或增设简化命令。例如：c copy，t trim。
　　三、界面设置
　　1．以“Use a template”方式，选择“acad.dwt”，进入AutoCAD 2000。选择下拉菜单tools下的options，进入options设置界面。
　　a.选择display项
　　勾选display scroll bars in drawing window项，选设置上下、左右滚动条。
　　勾选display screen menu项，则可调出右菜单Screen menu。
　　点选colors项，设置屏幕背景颜色，设为黑色（black）。
　　点选fonts项，可设置命令行的字体与字体大小。
　　修改Crosshair size项的大小可改变十字光标的大小。
　　b.选择user preferences项
　　点选right－click customization，将弹出鼠标右键设置窗口。选择Default mode和Edit mode repeat last command则可取消鼠标右键快捷菜单。（右键快捷菜单是AutoCAD 2000新增的功能，但笔者使用后觉得反而不方便、繁琐，增加了不必要的选择、确认操作，降低工作效率。）
　　2.工具条设置
　　a.选择下拉菜单View下的Toolbars，进入Toolbars设置界面。
　　选择new项，输入Toolbar Name，例为lzj。确认后，即出现一名为lzj的空工具条。选择Customize项，进入Customize toolbar设置窗口。通过Categories下的下拉条可在不同的工具条板块间切换，找出自己所需的命令图标。用鼠标选上，同时按住鼠标左键，将图标拖至lzj工具条内。这样，就能创建出自己独具特色的工具条。
　　b.工具条设置好后，只是存在于D:\AutoCAD 2000\Support\acad.mns内，但这是个临时文件，如果修改了acad.mnu，acad.mns将被刷新，设置好的工具条内容将被冲掉，因此需将acad.mns内的工具条设置内容复制到acad.mnu内。用编辑器打开acad.mns，将＊＊＊TOOLBARS下的＊＊lzj工具条设置段复制到剪贴板，再打开acad.mnu，将剪贴板的内容粘贴到＊＊＊TOOLBARS下。
　　3.存图设置
　　选择下拉菜单tools下的options，进入options设置界面。选择open and save项。
　　a.由于目前AutoCADR 14仍在广泛使用，为了方便交流，应将图形保存类型设置为AutoCAD R14的＊.dwg格式。即将save as项选为AutoCAD R14/LT98/LT97 Drawing（＊.dwg）。
　　b.自动保存及备份设置。勾选File Safety Precautions下的Automatic save和Create backup copy with each save，并将自动存盘时间Minutesbetween saves项设为30分钟。
　　四、字体的设置
　　1.在D:根目录下建立名为fonts的文件夹，将所需字体拷至其下。
　　2.设置AutoCAD 2000的字体路径：打开AutoCAD 2000下拉菜单中tools下的options选项，点选files项，双击下面窗口中的support file search path项，再点选其下的D:\AutoCAD 2000\fonts项，单击browse，选择D:\fonts即可。另外，也可将AutoCAD 2000等软件的字体路径设为此路径，这样便可避免重复安装字体文件，节省硬盘空间。
　　3.打开下拉菜单Format下的text style设置窗口，将数字设置SHX Font项选为txt.shx,勾选Use Big Font，将汉字设置Big Font项设为hztxt.txt。
　　4.解决钢筋符号的输入与显示问题：将结构软件pkpm下的txt.shx拷贝到D:\fonts下，重新启动AutoCAD 2000，执行dtext命令，按下“Alt＋131”与“Alt＋130”（131，130为小键盘数）就可写出漂亮的一级、二级钢筋符号。
　　五、图层、线形设置
　　选择下拉菜单Format下的Layer，进入“层”管理界面，设置以下图层、颜色及线形。
　　Name 　　Color　　 Linetype
　　钢筋层 　 red 　　 continuous
　　梁线层　 cyan 　 dashed
　　轴线层 　 red 　　 center
　　标注层 　 green 　 continuous
　　汉字层 　 white continuous
　　六、尺寸标注设置
　　1.选择下拉菜单Format下的Dimension Style，进入Dimension Style Manager设置界面。
　　2.点取new项，设置new style name为bz，并设为当前项。
　　3.选取lines and arrows项，将arrowheads下的1st和2nd均设为Architectural tick，Leader设为None。
　　4.选取Primary Units项，将linear Dimensions下的Precision设为0，将Round off设为10。
　　七、保存设置
　　至此，user模版设置完毕，需要保存设置，选取下拉菜单file下的save as项，将保存类型设为AutoCAD Drawing Template File（＊.dwt），文件名设为user，保存后在Template Description对话框中填入“设计模版”字样，即完成全部设置。同时在D:\AutoCAD 2000template下将自动生成名为user.dwt的文件。
　　以后画图时，只需以“Use a template”方式，选择“user.dwt”，进入AutoCAD 2000，以上设置即可再现，你可舒心地进行下一步的设计，真是省事、省心，快试试看吧。

（福建　刘志军）
