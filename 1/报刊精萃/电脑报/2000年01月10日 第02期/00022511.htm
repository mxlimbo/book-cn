《 {硬件t望台} 》

{　　显示卡方面：
　　台湾的低端显卡制造商Sparkle，将制造基于Savage 2000芯片的SP410显卡。这家公司将成为继Diamond之后第二家生产Savage 2000的板卡厂商。SP410和ViperⅡ功能基本相同，但Sparkle可能提供全系列的产品。SP410将有16／32／64MB、芯片运行频率为166／183／200MHz的各档次产品，预计显存将采用SDRAM和SGRAM。Sparkle还将制造GeForce显卡，但显卡配置不详。
　　Creative Labs Annihilator新闻组有一些关于Creative GeForce SDR 和 GeForce DDR 显卡边带寻址（Sidebanding ）支持的信息。Creative图形视频部的高级技术专家Bill Ball在新闻组说，他们正在制作开启／关闭边带寻址支持的程序。不知出于什么原因，nVidia在GeForce的驱动程序中不支持这个功能。在以前的TNT或TNT2也不支持这个功能，但不知道这个为GeForce定做的程序是否可以在TNT和TNT2显卡上使用。
　　关于Side Band Address，许多朋友还不熟悉，我在这里程序口罗 嗦两句。Side Band Address（以下简称SBA，边带寻址技术）和DME（Direct Memory Execution，直接内存执行）是AGP总线的两大功能。DME我就不多说了，直接调用主存存放纹理，也就是AGP纹理，被3dfx宣判为死刑的技术。:) 下面讲讲SBA。
　　PCI设备是通过PCI地址／数据总线（Address／Data）向主板发送地址请求，然后等待请求返回，之后才能发送下一个地址请求。因此大多数时间AD总线是闲置的，浪费了带宽。而AGP设备地址请求和应答是同步乱序进行的。举个形象的例子，AGP设备可以发出三个连续的地址请求（A1， A2， A3），而收到两个请求应答（D1，D2），在发送第四个地址请求后（A4），收到第三和第四个请求应答（D3和D4）。简而言之，AGP设备的数据传输是同时进行的，无须等待。SBA技术充分利用了带宽，提高了显示子系统的性能。
　　GeForce支持Sidebanding功能。但程序还不知道什么时候出来，而且是否支持其他的显卡如TNT／TNT 2或其他厂商的GeForce还未可知，这里有一个办法可以给3.62驱动程序增加Sidebanding功能，你不妨试一下：
　　修改注册表如下：[HKEY_LOCAL_MACHINE\Software\NVIDIA Corporation\Global\System] DISABLEAGPSIDEBAND=dword:00000000
　　但这是否能“Enable Sidebanding”不得而知，有人测试用此法后Quake3提高了3fps。其实如果我记得不错的话，用Powerstrip应该可以开启nVidia显卡的Sidebanding功能。方法是输入命令行pstrip.exe ／sba:off (或 ／sba:on)
　　nVidia驱动问题还真不少，在Athlon＋Geforce系统上，如果使用3.56以上的GeForce驱动，系统将不支持AGP2×，nVidia的Nick Triantos提供了解决方法：
　　修改注册表如下：
　　REGEDIT4
　　[HKEY_LOCAL_MACHINE\Software\NVIDIA Corporation\Global\System]
　　″EnableIrongate2x″=dword:00000001
　　测试系统：Asus K7M 主板、 Athlon 600、 Asus V6600 GeForce显卡。 用WCPUID测试3.56～3.65版本的公板驱动通过。
　　CPU方面：
　　Intel最新的Coppermine处理器已经发布有一段时间了，但人们对Coppermine还不是很了解。那么什么是Coppermine？Coppermine和PentiumⅢ有什么不同？
　　Coppermine其实是Intel为新的P6内核所起的代码名――铜矿，不过不要误会，芯片内部并不是铜导线互连，仍然是铝技术。Coppermine采用了先进的0.18微米工艺制程，在片内集成256KB L2 Cache，最高支持133MHz 外频。而老PⅢ是0.25微米工艺，只支持100Hz外频（超频另当别论）。
　　Coppermine的命名法则是这样的，在时钟后面加特定的英文，这样不仅表示具体的型号，而且可以与老的PⅢ区别开来。例如，PⅢ600E，PⅢ633EB。这里，E代表是Coppermine处理器，EB代表支持133MHz FSB的Coppermine。
　　FC－PGA（Flip Clip）封装的Coppermine只有500MHz和550MHz两种型号，也就是PⅢ 500E或PⅢ 550E。它们都只支持100MHz外频，倍频分别为5.0和5.5。Slot 1版本的Coppermine时钟为600MHz～800MHz。
　　由于使用新的 VRM 8.4 规格及多了一个 RESET 信号脚位，新封装的FC－PGA Socket 370 处理器与老的 Socket 370 主板不兼容，因此现有BX主板无法支持Socket 370的Coppermine。工艺的进步使得Slot1版本的Coppermine内核电压降低为1.65V，因此不少BX主板也无法支持Slot 1 的Coppermine。因此理论上只有你的BX主板支持1.65V的内核电压，才可以支持Slot 1的Coppermine。
　　那么是不是非要买新的i810、i820主板才能支持Socket 370 Coppermine呢？把成熟稳定的BX主板仍掉岂不可惜？当然不必，聪明的工程师总有解决的办法。除了Iwill已经上市的Slocket－Ⅱ转接卡外，著名主板厂商技嘉也正在制作一个Socket 370－Slot1转接卡GA－6R7＋，它不仅可以支持现有的Celeron处理器，还可以支持Intel的Coppermine处理器和VIA未来的Joshua处理器。
　　主板／芯片组方面：
　　Tomshardware网站披露，AMD改进型的750芯片组增加了一个新功能，就是北桥芯片的“Super Bypass"功能。
　　那么什么是Super Bypass呢？简单地说Super Bypass可以减少内存和CPU之间不必要的内存延迟。AMD的文献宣称Super Bupass所节省的内存延迟时间不小于25％。要知道，任何一个芯片设计者的目标就是能使芯片在与内部总线通信时延迟时间最小。
　　AMD 751（北桥芯片）负责CPU、SDRAM、AGP总线和PCI总线所有的通信。另外，751芯片还包括内存请求组织（Memory Request Organizer，MRO），MRO控制CPU、PCI、AGP和内存总线的数据。当Super Bypass不起作用时，MRO从主存中存取数据是一个漫长的过程。当Super Bypass启用后，MRO可以忽略某些过程，因此减少了不必要的时钟浪费，从而提高了系统性能。当然这要满足一定条件，AGP和PCI在进行数据传输时，Super Bypass必须起作用。据AMD说，在一般的系统运算中，Super Bypass在90％～95％的时间内处于活动状态。不幸的是，早期版本的AMD 750不支持Super Bypass。
　　想知道你的Athlon主板是否支持Super Bypass吗?你可以用Athlon Chipset Id V 0.30检测一下。如果你的 AMD 751芯片组是Revision C版本，那么理论上就支持Super Bypass。Athlon Chipset Id V0.30版还可开启Super Bypass功能。因为某些改进型750芯片组的Athlon主板（具备Super Bypass功能）BIOS中并没提供开启／关闭Super Bypass的设置。
　　使用“／B＋”开启Super Bypass功能，相反“／B－”关闭Super Bypass。“／?”显示帮助。推荐在DOS实模式下使用。例如“chpid／B＋”就是Enable Super Bypass功能。
　　遗憾的是，只有检测结果为Revision C和Stepping 5时这个程序才能允许使用Super Bypass功能。用Athlon chipset ID测试AMD公板显示为“Revision C stepping 1”。在华硕K7M主板上显示为“Revision C stepping 4”。不知道哪些幸运儿有stepping 5的Athlon主板。据AMD证实只有revision C和stepping 5的AMD751芯片组才具备Super Bypass功能。因此不推荐在低于stepping 5的Athlon主板上强行使用这个程序开启Super Bypass功能，轻则可能造成系统崩溃，重则系统损坏。
　　内存方面：
　　新千年前夕，高高在上的内存价格终于出现了下降的趋势。对于PC制造商和消费大众这绝对是个好消息。
　　内存大厂Kingston在本月首先发动价格战：大多数内存型号都降价了。品牌机内存和打印机专用内存降价20％～30％，普通的PC 100内存降幅高达51％。众所周知，Kingston是业界著名的内存厂商，具有举足轻重的地位，Dataquest 内存分析家Jim Handy称Kingston“领导RAM市场”。
　　Kingston内存产品部经理Lisa Dreher说：此次内存降价是因为前段时间内存供不应求，厂商恢复大批量生产，供货量的增加导致内存价格开始下跌。
　　虽然Kingston不直接向消费者出售内存，但Dreher说零售市场将很快对Kingston的降价作出反应。最近Kingston 64MB PC100 RAM零售价为95美元, 128MB为225美元。
　　Dataquest的分析家Handyodule认为Kingston 128MB内存的价格极具竞争力。他表示这体现了Kingston作为一个知名厂商的生产能力，并分析Kingston仍将大批量生产市场上最流行的内存条。
　　不知Kingston的降价是不是内存降价的前兆？
