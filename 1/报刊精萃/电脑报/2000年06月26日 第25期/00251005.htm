《 一步一步学SecondCopy 2000 》

　　SecondCopy 2000是一个常用的文档备份工具，这个工具的安装文件不足1MB，非常小巧，但是程序提供的功能很完整，包括定时备份和手工备份两种方式，而且程序支持向远程计算机备份文件。SecondCopy采用Wizard（向导）和用户自定义两种设置方式，分别适合初学者和具有一定计算机知识的用户使用。SecondCopy采用在线驻留方式，默认设置随系统启动自动运行，可以在系统后台完成备份操作，不需要用户干预，使用起来很方便。另外程序可以很好地支持Win9x/2000/NT系统，提供口令保护备份策略文件、以分组方式管理备份策略文件、支持创建分卷备份、支持对备份文件压缩、提供备份文件校验等基本功能，是一个非常典型的备份工具。
　　一、软件的获得与安装
　　SecondCopy 2000的官方站点是http:∥www.centered.com/，在此你可以得到程序的最新下载版本，本文中使用的版本是最新的6.0。下载的文件是一个可执行文件，直接运行它，并连续单击“Next”按钮即可轻松完成软件的安装。安装完成后程序会自动设置随系统启动而自动运行，其标志是在系统任务条中添加一个Tray图标。
　　二、以“Express setup”方式创建备份项目策略文件与文档备份
　　1.单击程序在任务栏中的Tray图标，此时会出现如图1所示的SecondCopy 2000程序主界面。
　　2.单击界面中的第一个按钮“New Profile”，程序给出如图2所示设置界面。该设置界面中提供了两个复选框，其中“Express setup”表示使用程序提供的设置向导创建备份策略文件，“Custom setup”表示由用户自行定义备份策略，包括设置备份文件类型、可排除备份类型、设置压缩比、文件同步方式等。程序默认使用“Express setup”，在此我们也选择使用它，单击“Next”按钮继续。
　　3.此时出现如图3所示界面，在该界面中的“Source folder”文本框中设置需要备份的所在文件夹，可单击“Browse”按钮，并在关联的“浏览文件夹”窗口中选择。该文本框下边的“Include sub folders”复选框表示连同当前文件夹下的子文件夹一同备份。单击“Next”按钮继续。
　　4.程序给出如图4所示界面。在该界面中的“Destination folder”文本框中设置保存备份文件的文件夹或驱动器。具体设置方法同设置“Source folder”操作相同，在此不再重复。单击“Next”按钮继续。
　　5.此时出现如图5所示界面。在该界面中提供的是当前创建的备份策略文件运行时间设置。在“Frequency”项中提供了备份时间间隔设置，其中包括：“Manual”手动激活、“Every few minutes”每隔多少分钟自动备份、“Every few hours”隔多少小时自动备份、“Every few days”每隔指定多少天自动备份。根据自己需要选择，并在该项关联文本框中设置具体循环周期。此外该项中的“Do Not Run Before”表示在设定时间之前不进行备份操作。该界面中的“Also run at”项中提供了两个复选框，分别表示在系统启动和关机时进行一次备份操作。“Do not run on these days”项的含义是在设置的天数内不进行备份操作，根据自己的实际情况选择即可。单击“Next”按钮继续。
　　6.最后程序给出的是“Finish”（结束）界面，在该界面中只提供了一个文本框，在其中输入新创建的备份策略文件的名称，之后单击“Finish”按钮确定，返回程序主界面，此时可以看到我们创建的备份策略文件图标已经添加到界面窗口中了，同时SecondCopy 2000开始首次运行当前的备份策略文件执行备份操作，你也可以看到程序的Tray图标会动起来。
　　三、以“Custom setup”方式创建备份项目策略文件与文档备份
　　使用“Express setup”方式备份的操作很简单，如果你需要自己定义备份操作的具体方式，需要使用“Custom setup”方式进行，以下是具体步骤：
　　1.在图2所示界面中选择“Custom setup”复选框，单击“Next”按钮，在关联界面中设置备份文件所在文件夹。
　　2.单击“Next”按钮，此时出现如图6所示界面，在其中提供了备份当前文件夹下的所有文件和子文件夹“All files and folders”、只备份所选择的文件和文件夹“Only selected files and folders”两个复选框，如果选择后者，需要在界面中的“Include specifications”和“Exclude specifications”列表框中分别设置可备份和可排除备份的文件。选择方法非常简单，单击相应项目中的“Select”按钮，程序会给出备份文件夹中的所有文件列表，选择使用窗口文件列表前面的复选框就可以了，此外在“Type of files”下拉文本框中提供了系统中注册的所有文件类型列表，你可以直接选择指定类别的所有文件。设置完成后单击“Finish”按钮返回。
　　3.切换到界面下边的窗口，依次在其中设置保存备份文件的文件夹、备份时间间隔等。单击“Next”按钮，程序给出如图7所示界面，在该界面中的下拉文本框中提供了备份方式选择，其中包括：单纯的文件复制“Simply Copy”、复制文件并删除目的路径下的旧文件“Exact Copy”、移动文件“Move”、压缩备份文件“Compress”、压缩并更新压缩备份文件“Exact Compress”、同步文件夹“Synchronize”等，根据自己的需要选择相应的备份方式，并完成相应的备份设置，由于不同的方式提供的设置项目不同，在此就不一一给出了。此外，在该界面中还有一个“Advanced Properties”按钮，在其关联的设置界面中包括的项目极为丰富，如使用口令保护、使用拨号网络登录到Internet、备份前运行指定的应用程序、使用硬件配置文件过滤当前的备份策略等，你可以根据自己的使用环境对备份策略进行控制。
　　4.切换到下一界面，就是设置备份策略的名称，完成备份策略等操作，这与前面的操作是一样的。
　　四、编辑备份策略文件
　　对于存在的备份策略文件，程序提供了编辑功能，不过编辑方法与创建备份策略文件的操作是完全一样的，首先选择需要编辑的策略文件，之后单击界面按钮条中的“Properties”按钮即可。
　　五、创建备份策略文件组
　　SecondCopy 2000提供了管理创建的备份策略文件功能，创建方法非常简单，在程序界面中的组列表项上单击鼠标右键，程序会给出一个关联菜单，选择“New Profile Group”按钮，程序给出“New Group”对话框，在其中输入新建的组名称，确定后返回程序主界面，你就可以看到新建的组标签项了。
　　如果需要将创建的备份策略文件归类，可直接使用鼠标拖曳其图标到相应的组标签项上即可。在此要注意的是，程序默认的“All Profiles”标签项中给出的是所有分组中的策略文件，如果在这里删除，那么在其所属分组中策略文件也就被删除了。
　　六、创建SecondCopy 2000安装盘
　　SecondCopy 2000提供了一项特殊功能，使用这项功能可以将程序的安装文件和当前创建的所有备份策略文件打包并生成SecondCopy 2000安装文件，当运行创建的安装文件时，不但可以重新安装程序，而且可以恢复所有的备份策略文件，这项功能在同类工具中还是不多见的。如果需要使用该项功能，可单击界面菜单条“Tools”项下拉菜单中的“Create Second Copy Setup Disk”命令，并在关联的对话框中确认，在浏览文件夹窗口中选择保存创建文件的磁盘或文件夹，之后单击确定按钮即可。
