《 {倾情之作金秋献礼――2000年《电脑报配套光盘》第3期9月发售} 》

{　　风风雨雨，《电脑报配套光盘》也已走过3个春秋。这当中凝聚了多少电脑报人的心血与汗水。“精品，做国内一流的电子出版媒体”是电脑报媒体室从不松懈的目标。在本世纪末的最后一个金秋，以什么来回报一直关心、支持我们的众多读者？无以为报，只有这册经过浸渍了我们心血的配套光盘。本光盘定位于国内广大的计算机初中级用户，指导活学、活用电脑，提倡在生活与学习中学好软件，用好软件。
五大板块　冲击无限
　●软件乐园
　　软件面对面：让你接触电脑报软件评测室的第一手试用资料，最新最全的软件点评。
　　软件追踪：要用就用最新，要用就用最好的。密切关注软件动向，即时向大家介绍最新的软件。
　　软件学苑：一个庞大的软件，复杂的功能，我的天！不用担心，到“软件学苑”来，诙谐的语言，活跃的气氛，使用软件从此轻松。
　　本期专题：每期推出一个话题，深入全面地介绍和比较相关软件技术及产品。
　　软件挑刺：大家介绍都说好，我偏要说它不好！让众人看看到底谁说得更有理。
　　技巧点滴：软件就是用的，可是怎么才能用得好，用得轻松，看看别人是怎么说的吧！
　　异域行动：用惯了Windows不妨试试别的操作系统，最近Linux不是挺热的么，你不想了解，不想学习吗？来参加我们的“异域行动”吧！
　●编程社区
　　新手入门：都说“万事开头难”，又说“师傅领进门，修行靠个人”，我们也来做一做抛砖引玉的工作。
　　代码讲解：看看人家写的经典代码，学习别人解决问题的方法，而且代码改一改就能为我所用，何乐而不为呢?
　　开发新思路：天天用这些软件咱都用烦了，“有点创意好不好？”
　　作者访谈：听听软件作者说什么，他们是怎样构思一个软件的，他们又是怎么推广的，软件商业化是如何操作的……你想了解吗？
　●硬件天地
　　看图识硬件：“眼见为实”，看看图硬件轻松搞定，从外到里全面认识。
　　硬件应用文萃：再怎么说也得说到用上面来。为什么都是TNT2，他的就比我的好？看看人家的，想想自己的，从此不再迷惑。
　●网海纵横
　　网络学园：网上冲浪怎能没有像样的工具?请到“网络学园”来，在这里可以学到你所需要的工具软件使用方法。“网络学园”能让你轻松驾驭一切，从此网海泛舟有了好帮手。
　　网络生活：来自互联网的全新生活体验，展现网民风采。赶快行动吧，做信息时代的新新人类。
　●艺术时尚
　　艺术长廊：艺术与技术在这里找到了最佳的结合点，充满艺术魅力的技术、需要高超技艺的艺术，技巧、心得，美工大师“现身说法”。
　　时尚软硬件：今天流行什么？抢眼的MD，还有和手机互发短消息的OICQ。生活真的很精彩。
光盘精彩内容曝光
　　本期光盘内容精彩纷呈，而且考虑到为了给读者减轻负担，把三张光盘浓缩为两张精华版，零售价也可从38元降到25元。然而内容却更加精彩：首先收录了现在最为热门的DirectX 8.0 beta版、Microsoft　Media Player7.0（正式版）、RealPlayer 8.0 plus（完全版）、Win2000 SP1 (英文版)。
　●开发区
　　Microsoft Visual Studio 6.0 SP4 (中英文版)
　　这是Microsoft Visual Studio 6.0开发平台的最新升级包，解决了Visual Studio 6.0中的已知问题。保持了与旧版本Visual Studio兼容的问题；Service Pack 4中还包含了Service Packs 1、2、3中所有的更正的内容等。
　　ActivePerl 5.6.0.616 for win32
　　ActivePerl是在Win32环境下运行PERL的最佳环境，支持IIS等Web服务器，既能用于商用Web环境，也可以和一些小型Web服务器软件配合使用，可以在Windows下快速建立Perl CGI支持环境。
　　MySQL 3.23.22－beta for Linux
　　一个完全免费的Internet数据库系统，可以与PHP、C等语言结合，作为网上交易发布系统的后台数据库支持
　●图形软件
　　Paint Shop Pro 7.0b4
　　一套功能强大的图像制作软件，支持众多的图像格式，提供了Layer功能等。在最新版中附加了Animation Shop 3，加强了动画制作能力；增添了几个自动设定的功能，包括为专业用途自动调整颜色、亮度、饱和度等；可调整色谱曲线图，让你可以找到自己想要的图像色调。
　　CompuPic 5.1
　　IrfanView32 3.21
　　E－Icons 3.16
　　Swish 1.51
　　HyperSnap－DX 3.62
　●网络特区
　　Internet Explorer 5.5
　　企盼已久的IE 5.5正式版终于推出来了。不过IE 5.5对一般用户的改进不大，只是现在你可以使用“打印预览”查看网页在纸上的输出效果了；但如果你是一个网页开发人员，一定会兴奋不已，IE 5.5的可编程性有所增强，由于IE具有改进的DHTML功能，因此在编程方面会更灵活，网页也会更美观。

　　FlashGet 0.88
　　netants 1.22
　　foxmail b3
　　oicq 0725P2
　　HotDog 6.5
　　ommo 1.0 Build_1013

　●办公软件
　　Office 2000 Service Release 1－a
　　这是微软针对Office 2000在使用中出现的一些问题以及Windows2000的新特性发布了中文版Office 2000 的SR－1a版本。这次升级是围绕着安全性的提升进行的，它将Office 2000系统组件升级到了Windows 2000的系统安全级别。
　　StarOffice 5.2 for Linux
　　Linux下的强大的文档编辑器。它几乎能完成MS Office的所有功能，但又是完全免费的，随着Linux系统的普及，StarOffice是否会成为MS Office的终结者呢，我们将拭目以待。StarOffice也有for windows版本。
