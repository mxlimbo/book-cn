《 {移动办公不是梦之电脑互联篇} 》

{ 一、串口或并口互联
　　两台计算机之间的互联，可以通过机器本身的串、并口进行互联。通常，我们可以采用串口或并口进行互联；采用并口互联，速率较快。采用串口互联，速率较慢。通常情况下，如果要简单实现两台计算机之间的互联，为了达到较高的传输速度，建议采用并口方式进行互联。

　　2.准备软件安装
　　Windows 98安装光盘，在台式机上：
　　在Windows 98的控制面板中，选择“添加/删除程序”项，双击进入，选择“Windows 安装程序”→鼠标选中“通讯”，点击“详细资料……”进入，再选择“直接电缆相连”软件包，之后选择“确定”回到上一级窗口，再选择“确定”。这时，系统要求插入Windows安装光盘。用户插入光盘后，系统会自动将直接电缆相连的软件包安装好的。
　　在笔记本电脑上，同样的安装过程再做一遍。
　　至此，所需软件安装完成。
　　3.配置
　　第一次配置直接电缆相连步骤：
　　选择：“启动→程序→附件→通讯→直接电缆相连”，系统出现窗口所示。
　　选择“更改”，以定义此计算机是作为主机或客户机。作为主机时，表明此计算机为服务提供者，当选择作为主机后，程序进一步要求用户提供直接电缆连接的方式。
　　从提供的端口中选择一个。如果以上端口均不是你正在使用的端口，还可以通过“安装新端口”方式，加入新的端口。
　　正确选择后，点击下一步，则出现窗口。
　　如果用户对安全性要求较高，还可以通过“设置密码”，加入密码保护功能。点击完成。至此，完成软件的配置。
　　以后，如果要通过直接电缆相连，就可以直接从“附件→通讯→直接电缆相连”直接启动连接。这时，作为主机的服务开始运行。
　　如果长时间发现不了客户机，主机还会提示用户检查客户机是否正在运行。
　　同样的步骤也要在笔记本电脑上运行，机器进入等待状态，等待客户机的访问。
　　第一次配置好直接电缆相连后，以后如果要使用直接电缆相连，用直接电缆连接好两台计算机的并口或串口后，就可以直接从“附件→通讯→直接电缆启动”相连，即可开始使用。通过电缆相连的机器相互找到后，就可以相互使用对方共享的资源了。共享信息包括文件、打印机、Internet共享入网等资源。
　　注意：有时通过直接电缆相连可能找不到客户机，用户应该查看电缆的端口，确保正在使用的端口同电缆连接的端口相同。
二、网线直通互联
　　1.准备硬件
　　首先要准备一块10M或100M以太网卡，Accton或D－Link的10M网卡，几十元一块，很便宜。这块网卡是为台式机准备的。再准备一块PCMCIA的网卡，安装在笔记本上。最后再准备一根长2米（<100米）以上的双绞线，两个RJ45头。
　　使用网络直通线时，网络的接法同使用HUB是不一样。由于没有通过HUB，所以，在直通线中，须用交叉连接法。以下简单介绍交叉线的连接方法。
　　在介绍连接方法之前，首先让我来定义一下RJ45头的正面和反面。我们把有弹性小片的那一面定义为反面。正面拿上RJ45接头，金属压线的一头朝前，从左到右，顺序为：12345678。通常情况下，网线中共有8根线，在10M网络下，传输数据仅仅使用1、2、3、6四根线，另外两对线则起平衡干扰的作用。不过，在制作网线的过程中，1和2必须是一对绞在一起的线，同样3和6也应是一对绞在一起的线。
　　对于RJ45接头制作方法：RJ45接头接法国际标准有两个可选方案，1.568B标准：白橙、橙、白绿、蓝、白蓝、绿、白棕、棕；2.568A标准：白绿、绿、白橙、蓝、白蓝、橙、白棕、棕。选择哪种标准，主要看使用什么厂家的接插件产品，如信息插座、配线架等。如果用户是通过HUB互连的，那么直接按标准进行网线制作。
　　2.硬件的安装网卡
　　在台式机上安装好网卡，启动计算机，系统会自动找到网卡，提示安装网卡的驱动，将网卡所带的驱动程序插入软驱，网卡驱动安装成功。
在笔记本电脑上，将PCMCIA卡插入PCMCIA插口，PCMCIA卡支持带电插拔，用户可以放心大胆地插入卡，这时，Windows会自动找到新加入的硬件，要求用户提供卡的驱动程序，将随卡所带的驱动程序安装好。
　　重新启动笔记本电脑。如果启动后看到SysTray中多了一个PCMCIA卡的小图标，则表示PCMCIA卡已经安装成功。
　　PCMCIA卡有很多种类型，我们可以选择一块PCMCIA网卡，也可以选择网卡和Modem二合
一卡。
　　3.安装正确的协议
　　安装协议的方法是：选择“控制面板”→加入NetBEUI协议、TCP/IP协议。
　　注意：如果要使用共享的目录及文件，要求用户在安装协议时，要安装NetBEUI协议，这样，通过共享方式，用户可以直接在网上邻居中找到对方的计算机。否则，无法找到对方的计算机。
　　点击“文件及打印共享”，选择“允许其他用户访问我的文件”及“允许其它计算机访问我的打印机”。这样，别的用户就可以访问本地的文件，笔记本电脑也可以使用台式机的打印机了。完成以上操作后，网络配置。
　　再在“标识”中将两台计算机设置到同一个工作组中。这样，在桌面上的“网上邻居”中就可以看到对方的计算机了。
　　以上所有配置操作完成后，点击“确定”，这时，系统要求将Windows 98安装光盘插入CD－ROM中，安装程序会将新加入的一系列软件包安装到系统中，然后，系统要求重新启动。
　　重新启动系统后，将公共资源共享，也可将打印机共享。
　　笔记本电脑的配置：
　　笔记本电脑的配置同台式机完全相同。本文不再叙述。
　　注意：如果两台计算机之间只安装TC/IP协议，尽管双方都可以ping通对方。但是，不会看到对方的计算机，也看不到对方的共享目录。要相互看到对方的目录及文件，应安装NetBEUI协议。
　　只安装了TCP/IP协议的用户，如果要进行文件传输，则可以通过FTP服务及客户的方式进行文件传输。这就要求在其中一台计算机上安装一个FTP服务器，以便另一台电脑可以通过FTP登录到台式机上。当然，也可以在两台计算机上同时安装FTP服务器。实际上，两台电脑之间的地位关系是完全相同的，任何一台都可以作服务器，也可作工作站。
　　4.资源共享
　　在台式机上，建立一个目录public。将共享文件拷贝到这个目录中，以备笔记本电脑访问。也可以将打印机设置为共享方式，这样，就不必将打印机搬来搬去了。
　　用鼠标右键点击硬盘，从菜单中选择“共享”，出现共享设置的对话框，设置相应的共享名称，如public，设置访问类型为只读、完全或根据密码访问等三种方式之一。如果选择了根据密码进行访问，则要求输入密码。这样，当来自笔记本电脑的用户要访问台式机的文件资源时，要求用户输入正确的密码，如果密码不正确，则台式机拒绝向用户提供访问。
　　使用网卡通过直接连线进行访问时，其实质同使用HUB的两台计算机进行相互访问是一样的。只不过是网络连线的接法不同。只有两台计算机的情况下，不必要花钱再购买一个HUB。

三、通过电话线拨号互联
　　Windows 98提供了一个拨号网络服务器，通过它，也可以实现台式机同笔记本电脑的连接，只不过，通过Modem连接两台计算机，速度较直接使用并口进行通讯要慢。但是，如果两台计算机之间的距离比较远，那么，采用Modem将两台计算机连接起来，就是一个必要的选择了。
　　通过Modem拨号连接实现两台计算机的互连非常适合两台计算机之间的距离十分远的情况。
　　1.硬件需求
　　Modem一个，内置、外置均可。PCMCIA Modem卡一块，电话线两根，分别连接到每一个Modem上。
　　2.软件安装
　　通过Modem拨号访问，首先需要一个拨号访问服务器。Windows 98提供了一个拨号访问服务器的软件包。如果你使用的操作系统是Windows 95，那么，你得安装Windows 95的一个服务包。用户可以从微软站点下载一个软件包，名称为：mdun12.zip。安装它后，就可以使用拨号网络服务器了。以下以Windows 98为例，在台式机上添加一个拨号网络适配器，具体实现办法：
　　首先在台式机上安装拨号网络服务器：
　　点击“我的电脑→控制面板→添加/删除程序→Windows安装程序→选择通讯”项。
　　再点击“详细资料”，选择拨号网络服务器。之后“确定”。
　　系统会自动提示用户插入Windows 98安装光盘。放入CD－ROM后，系统将从相应的压缩文件中安装刚才选定的软件包。
　　至此拨号网络服务器软件包安装完成。
　　此时，打开“我的电脑→拨号网络”，从菜单中选择“连接”，会看到菜单中多一项“拨号网络服务器”。
　　3.配置拨号网络服务器
　　Windows 9x要作为拨号网络服务器，需要选择用户的拨入方式为：“允许拨入”状态。对于拨号网络而言，密码保护是必要的。所以用户应该设置密码保护。更改密码可以通过点击“更改密码”。输入用户登录时的密码即可。
　　注意，用户还要选择服务器的类型，配置时，点击“服务器类型”进入拨号服务器类型的选择，选择服务器的类型为“ppp : Internet , Windows NT Server , Windows 98”方式。
　　此时，点击“拨号网络服务器”，则服务器进入拨号服务器状态，等待远程用户的拨入。也即进入服务就绪状态。此时，在SysTray中可以看到一个小的电脑图标。
　　笔记本电脑的设置：
　　首先，要安装一块PCMCIA　Modem卡。安装步骤：不必关闭笔记本电脑，直接将Modem卡插入笔记本的PC槽中，Windows 98会自动发现卡，系统提示安装Modem卡的驱动程序，用户只要简单地将随卡所带的驱动盘插入软驱，选择从磁盘安装。这样系统会自动安装好Modem的驱动程序。
　　其次，安装拨号网络适配器：
　　打开“我的电脑→控制面板→网络→添加→适配器→厂商”，选择安装Microsoft的拨号网络适配器。确定后，系统会自动安装。
　　笔记本电脑拨入设置：
　　首先，要建立一个拨号连接，建立连接的过程如下：
　　选择“我的电脑→拨号网络→建立新连接”
　　输入台式机Modem所连接的电话号码，选择服务的类型为“ppp:Internet,Windows NT Server,Windows 98”，这种设置支持TCP/IP、NetBEUI协议，用户可使用基于Internet的所有应用，也可以在拨号成功后，看到服务器上的共享目录、共享文件等。注意：使用电话拨号的方式进行两台计算机之间的互联，传输文件时，速度很慢。如果用户觉得速度不够快，可以选择使用ISDN，它可以提供128K的传输速率，有条件的用户还可以通过ADSL连接，利用它可以达到很高的传输速度（1.5Mbps以上）。
四、通过红外口互联
　　如果用户的台式机及笔记本均有红外端口，这时就可以使用红外端口进行互联。实际上通过红外设备互联同使用串口互联基本相同，只是系统使用的驱动程序不同。
　　首先，在计算机启动时，进入BIOS设置，选择红外端口，使其处于Enable状态。这样，在系统启动以后，Windows 9x会自动发现红外端口，并将它与串口、并口作同类设备对待。
　　注意：如果是采用红外端口进行互联，在两台电脑互联时，两台计算机之间的距离不要放得太远，而且两台机器之间不要放置其它的东西，以保证红外线不被中间的物体阻挡。
　　使用红外端口相连，通常情况下，有专用的通讯工具，使用这些工具进行双机之间的文件复制及其他资源共享。
　　由于目前台式机配置红外口的还不多，本文就不对此进行深入讨论。有兴趣的朋友可以自己找两台有红外端口的电脑试试。}
