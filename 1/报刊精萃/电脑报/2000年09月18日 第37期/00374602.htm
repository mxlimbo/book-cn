《 网上作品防抄一法 》

{　　有人说：“上了网的作品就没有版权”，这话虽有些偏激，但也不无道理。毕竟上了网的作品，特别是普通文字作品更容易被人转抄，只要点击Ctrl＋C→Ctrl＋V就行了。要在网上发布作品，又不想被人随意转抄看起来似乎是一个两难的选择。现在提供一种解决上述问题的办法，希望能起到抛砖引玉的效果。其解决办法所用的工具就是微软的网上动画制作软件Flash 5。
　　下面是使用Flash 5来保护作品的步骤：
　　1.新建一个Flash 5文档，把Layer 1改名为protect；
　　2.指定第1帧为关键帧，打开Frame Actions面板，并输入以下语句：
　　loadVariablesNum (″protect.txt″,0);
　　3.指定第2帧为关键帧，打开Frame Actions面板，并输入以下语句：
　　if (/:protect==1) {
　　gotoandstop(4)
　　}
　　4.设定第3帧为关键帧，并输入“请到某某网站观看本作品”，再打开Frame Actions面板，并输入以下语句：stop;
　　5.设定第4帧为关键帧，从这里开始放置你的作品。
　　另外再制作一个文本文件，命名为protect.txt，在文件里输入＆protect=1，注意这里有一个＆，表示这是一个变量。
　　如果没有在SWF中指定文本文件的路径，在上传时应注意把输出的SWF文件和文本文件放置在同一目录内。但为了更好的保护自己的作品，本文建议把存储变量的文本文件和SWF文件分目录存放，这样可以使作品更安全。
　　现在解释一下本文Flash 5中Action的作用，在第1帧中添加的Action，是告诉Flash 5读取一个名为protect.txt的文本文件中的变量。第2帧中的Action是判断读取的portect.txt中变量protect是否为1，如果为1的话就跳至第4帧。第3帧的Action是在第2帧判断为否时停止运行，而且提示到经授权的网站观看作品，这样就使Flash 5文件在此中断，自然也就没法看到后面帧中的内容了。文中protect.txt、变量＆protect=1均可由读者按个人喜好命名。
　　用上述方法可以保护文字作品，对图像和Flash作品也有一定作用，用它可以很方便地保护自己的作品只在指定的网站上被观看，而未经授权的转抄将只能看到“请到某某网站观看作品”，即可保护自己的作品，又作了免费广告，这不是很好吗？
　　用Flash 5保护作品的思路很简单，相信大家还会找到其它的好办法。(袁应崎)}
