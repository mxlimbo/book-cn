《 网上路况先知道 》

{　　当我要上网时，如能知道此时网络的联机状况，以确定上网还是不上网，就像了解了路况报导，得知大塞车时我就不会开车出门了，那该多好！这里就给你介绍两款这样的好工具。
　　1.VisualRoute
　　VisualRoute可为你提供网络联机的实时分析报告。你只要输入地址后马上可以告诉你现在的联机状况！
　　VisualRoute是一个用Java写成的软件，它需要Microsoft Java VM才能执行。所以建议软件配置为Microsoft IE 4.01或Microsoft Java VM或SUN Java Runtime Environment，这三个只要有其中一个就可以了。
　　首先连上线，激活该程序，然后在左上角Host/URL中输入欲联机的地址，按下Enter键或者点一下Host/URL旁边那个绿色小箭头，程序就会自动Ping你要求的网址并将结果(网络联机的实时分析报告)显示在右边的窗口中。使用十分简单。
　　报告窗口分为两部分，上面为文字报告区，下方为表格区。文字报告会将所联机的总节点数传回并报告是否有资料在传输中遗失或被忽略。如果有这种情形则在下方表格的Err字段中将会有红色或黑色的数字出现，告诉联机者到底是哪个节点出现了错误。表格中的IPAddress、NetWork、Location这三个字段则代表连结的节点的详细信息；ms和Graph是连结此节点所花费的时间，以千分之一秒计算。时间越长代表速度越差。再下面就是一个世界地图，你可以用鼠标左键放大、右键缩小它。
　　在Edit选项中的Snap Table As Text可将报告储存成文字文件，以供日后参阅。
　　每一次执行VisualRoute会要求输入注册码，如果你是试用就点选Run Trail按钮。如果你已经注册了请输入注册码并点选Run。
　　操作系统：Windows 98/95/NT；文件大小：2.3MB；试用限制：30天；注册费用：29美元。
　　下载地址http:∥www.visualroute.com
　　2.NeoTrace
　　在网络上下载软件的时候，经过的节点越少，受到的阻碍与延迟就越少，因此，下载的速度就越快。所以说，寻找一个又近又快速的节点是相当重要的。
　　要想掌握你自己的计算机与联机节点间经过了多少地方、传输速度如何等详细资料，并且以直观的图标方式呈现，那么就非NeoTrace莫属了。
　　下载neotr122.exe之后，直接执行即可进入安装程序。安装完毕后，首先要求你确定自己的位置。请在country栏中填入自己所在的国家，用两个字母表示，我们应该填“cn”；在city栏填入自己所在的城市；后面两栏分别填上邮编和省份名即可。然后按下“look up”，你所在的具体位置、详细经、纬度便立即显示在右边的窗口中了。
　　这时候你便可以点右边的“ok”键进入NeoTrace的主界面了。
　　你会看到它的address框内已经内建了一些网络节点。当然，你也可以自行输入你想要的。输入完毕后，按Enter键，程序就会自动帮你侦测从你的计算机到该网站或该节点所经过的地方以及传输情形等等。最后分别用图显示、列表显示、节点显示、图表形式显示给你，使你一目了然。
　　程序还提供了其他详尽的设定，你可以依照需求来自行设定。
　　操作系统：Windows 98/95/NT；文件大小：784KB；试用限制：30天；注册费用：24.50美元。
　　下载地址http:∥www.neoworx.com
(湖南　陈楚楚)}
