《 为你的网页加把锁 》

{　　Applet Password Wizard是一款专门为网页加密的软件，经过它加密处理过的网页，来访的客人只有取得了我们亲自发放的用户名和密码才能够浏览。
　　一、软件的下载安装
　　Applet Password Wizard为共享软件，当前版本v3.0，文件大小906KB。我们可以到软件的主页（http:∥www.coffeecup.com/）去下载最新的版本，也可以到以下链接地址http:∥210.77.155.248/soft16/password30.exe去下载。下载的是个自解压安装文件，直接运行该文件便可进入安装。安装完毕，在开始菜单的程序中会生成“Applet Password Wizard”快捷项。点击该快捷项，Applet Password Wizard即被启动（图1）。
　　二、软件的使用
　　Applet Password Wizard有四个标签页面，实际上也就是制作Applets程序的四大步骤。
　　1.“General”标签页面：该标签页面有三个分组框。首先，我们在“Applet Dimensions”分组框中可以调整Applets的大小，实际上就是设置网页上出现的用户名和密码输入框的大小。接下来我们在“Link”分组框中填写默认的URL，这样，当用户输入了正确的用户名和密码之后，URL所指向的网页便会呈现在用户的面前。最后，我们可以在“Colors”分组框中设置Applets的正文和背景的颜色。
　　2.“Login”标签页面：该标签页面共有三个需要填写的项目。当用户注册前，在“Pre－Login Message”下的正文框中填写的内容将显示在Applet最上方的信息框中，比如你在这里可以填写：请你输入用户名和密码。当用户注册成功后，在“Login Message”下的正文框中填写的内容将显示在Applet最上方的信息框中。在“Link”下的正文框中你可以输入一个特定的URL，当用户输入的用户名和密码有误时，将被导入URL所指向的页面，在该页面中，你可以给用户一些有关的提示信息。
　　3.“Users”标签页面：如果你授权给许多用户允许他们访问你的加密网页，那么，你在这里可以很方便地管理这些用户，你可以在这里添加、删除或者更改所有用户的资料。
　　（1）添加新用户　点击“New User”按钮，弹出“NewUser Properties”对话框，在该对话框中依次填入用户名、密码、网页地址，这时你在“Users”的列表框中会看见一个新增加的小图标，图标的名称与用户的用户名相同;
　　（2）更改用户资料　双击列表框中需要更改的用户图标，弹出“User Properties”对话框，在该对话框中你就可以更改用户名、密码和网页的链接地址；
　　（3）删除一个已有的用户　在列表框中选中准备删除的用户，然后点击“Delete User”按钮即可；
　　（4）批量增加新用户　这里提供了一个从文本文件直接导入批量用户的功能。你首先需要在记事本中填写好各个用户的信息，填写规则如下：每个用户的信息各占一行，依次填写用户名、密码、URL和目标帧，中间用逗号分隔。下面是一个典型的例子：guest(用户名),guest（密码）,http:∥www.here.com/guest（URL）,_parent（目标帧）。含有许多用户信息的文本文件编好以后，我们就可以把这些信息直接导入Applet了。点击“Load Users”按钮，弹出“Load Users”对话框，点击“Browse for .txt file”按钮，通过浏览找到需要导入的文本文件，双击该文件，文件中列出的所有用户一下子就全部被导入Applet了。
　　4.“HTML”标签页面：该标签页面中的列表框中有一段HTML代码，该段HTML代码实际上是对Applet的指令，告诉Applet 在Internet上应该做的每一件事情。如果你是一位网络高手的话，在这里可进行你认为必要的修改。我们按下“Copy HTML to Clipboard”按钮，列表框中HTML代码就会被复制到剪贴板上。接着我们用网页编辑器打开需要加入Applet程序的网页，把这段HTML代码粘贴到该网页上即可。如果这时用浏览器浏览该网页，你可能会发现刚才加入HTML代码的位置出现的是一个长方形的灰块。这是因为缺少了一个.class文件(joylock.class或者button.class)的缘故，这个文件是个Java程序，Applet要想正常工作离不开它的支持。Applet Password Wizard已经为我们准备好了这个文件，点击列表框右边带下划线的“joylock.class”字符，joylock.class就会被自动调出来，这时你只要把这个文件拷贝到加入了Applet程序的网页所在的目录下就可以了。现在我们再来看看效果如何。刷新刚才出现了长方形灰块的网页，哇，出来了（图2）！赶紧输入用户名和密码试试看吧，轻轻一按“Login”按钮，一下子就进入了我们刚才在“Users”标签页面中为用户设定的网页；再输入一个错的用户名和密码试试，我们被导入了错误信息提示页面。
　　到此为止，可爱的Applet小程序编制完毕并通过了测试。这时候我们就可以放心地把它上传到网上去担当重任了。需要提醒大家的是在上传加入了Applet小程序的网页时，千万不要把joylock.class文件忘记了。另外，上传joylock.class文件一定要用“binary”模式上传，否则，该文件将被损坏。
　　三、结束语
　　看了以上的介绍，你是不是觉得这个能让我们顷刻之间成为老鸟的东东很酷？但是，如果你没有注册成为Applet Password Wizard的正式用户，Applet在网下虽然会为你出色地表演，但它到了网上就罢工了，直到你为它掏出了白花花的银子。
(山东　周少军)}
