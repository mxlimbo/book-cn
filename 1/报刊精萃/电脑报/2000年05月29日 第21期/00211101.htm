《 System Safe Ⅱ--系统保护专家 》

　　System Safe Ⅱ是一款强大的系统保护软件，与其它类似软件不同的是，该软件提供了对系统重要文件的分级保护，即按文件的重要程度分别备份到不同的目录下，还特别对用户系统文件提供了双重保护，并能生成一个备份报告文件，用户可以通过查看这个报告了解有哪些文件被保护和备份，这对于文件恢复十分重要。System Safe Ⅱ提供了Win9x崩溃之后的安全恢复功能，同时还可以大大减少Win9x中经常出现的“非法操作”之类的错误。另外软件操作相当简单，只需点几下按钮即可完成处理。
　　用户可以从http://www.newhua.com.cn下载该软件，执行下载后的软件包，一路按“Next”即可完成安装，重新启动后，System Safe Ⅱ将会与Windows集成到一起，此后，每当计算机启动时，System Safe Ⅱ会自动对重要系统文件进行扫描诊断并报告它们的状态，一旦发现系统文件受损，将会自动修复和覆盖，这种诊断功能也可以在DOS环境下通过运行相关文件来进行。
　　软件运行后，首先出现如图1所示的窗口，这是一个用于系统诊断的操作界面，有一排带图标的快捷按钮，这几个按钮其实包括了软件的全部功能。用户可以点击按钮进行各种快捷操作，另外也可以在进入软件的主窗口中后再进行。为保证Win9x启动失败后的安全恢复，用户应在此创建一张系统启动盘，单击“Format Floppy Disk”按钮会自动调用Windows格式化程序，注意选中格式化对话框的复制系统文件复选框，右侧的“Shrinking Windows”复选框表示是否启用软件的动画收缩界面，“Introduction Music”复选框表示是否在软件启动时播放音乐，现在单击“Main Menu”按钮，进入软件的主操作界面，如图2所示。
　　主界面的左侧显示了当前被保护的重要系统文件和其它文件的状态，包括文件数目和最近更新的日期和文件总数目。右侧的按钮用于进行各种处理，具体的说明如下：
　　Update Critical Safe：更新重要系统文件，包括Io.sys、Msdos.sys、User.dat、System.dat、Autoexec.bat、Config.sys、Win.ini、System.ini和各种重要的DLL、SYS、VXD、DAT、INI文件。其中，前者被备份到System Safe Ⅱ的安装目录，后者被备份到另外一个目录，默认为C盘的Ssafe目录下，值得注意的是该软件还特别对用户系统文件Autoexec.bat、Config.sys、System.ini和Win.ini提供了双重保护，即在Ssafe目录下也会存放一份上述四个文件的拷贝。当System Safe Ⅱ第一次运行时，会自动进行文件备份，以后，每当计算机启动时，将会自动对备份文件进行诊断和检测，如发现问题将自动修复。建议以后要经常执行该功能以保持文件的及时更新。
　　System File Lab：保护其它文件，包括各种DLL、SYS、VXD、DAT、INI、DRV、DAT文件等，但被Update Critical Safe保护的文件除外，所保护文件将被备份到一个名为SYS_BIN目录下。执行后在出现的对话框中单击Start按钮开始检测，完成后再单击System File Lab按钮弹出如图3所示对话框，在该对话框中用户可以有选择地对指定文件进行备份，只需钩选左上相应复选框后，再单击“Add selected file type or types to System Bin”按钮即可，以后如发现文件受损或丢失，可以在右侧的“System Bin”列表框中选中相关文件，再单击“Restore selected files”按钮进行恢复。须要注意的是，Win9x中此类文件相当多，如钩选所有复选框，需要有足够的硬盘空间，这只要查看本对话框右下提示信息即可。
　　View Safe Contens：查看Ssafe目录下被保护的重要系统文件的内容，如发现受损或丢失，可以进行覆盖或恢复。在打开的对话框中，选中欲恢复的文件，再单击Restore Selected Files，文件恢复自动进行。
　　Safe location：查看重要系统文件的存放目录，可以更改。
　　File Scan：用于进行系统文件的自动检测，完成后生成一个文件syscheck.log，该文件存放在System Safe Ⅱ安装目录下。
　　Last File Scan Log：查看最近的检测文件syschcek.log内容。
　　Last Update Log：查看最近更新的重要系统文件Ssafe.log的内容。
　　System Safe Ⅱ安装后会自动在安装目录下生成一个名为Scansys.exe的文件，另外还会在系统根目录下生成一个批处理文件Scansys.bat，如果系统崩溃造成Win9x无法正常启动，可以利用事先制作的启动盘启动计算机，之后，在DOS环境下执行上述文件即可。
　　最后，须要说明的是，为保证System Safe Ⅱ在计算机启动时自动执行，以便对系统文件进行自动更新和检测，必须确保打开自动扫描功能，可以在主界面中选择“Action/File Scan”菜单，检查是否选中“ON”选项。另外，当重要系统文件丢失后，还可以到该软件网站（http://www.isoglobal.com）下载，在软件启动界面中提供了直接访问网站的连接。

　　(河北　王金柱　张世军)
