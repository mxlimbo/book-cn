第五幕 生日


S t a g e 5-1 < From Vol.5 >
背景: 温泉旅馆

< 莉香和完治出来在雪地上散着步 >
< 这时莉香捧起地上的雪,往完治的身上一丢 >
< 完治也捧起一堆雪,往莉香的背后一丢,两人高兴的玩着雪球 >

这时完治说: 好冷! 回去吧!
莉香摇头说: 再待一会儿!

< 莉香向双手呼了一口气,想温暖一下双手 >

莉香: 完治,我昨晚梦见你了。

< 完治讶异的嘿嘿笑了两声 >

莉香: 完治也梦见我了吗?
完治: 没有,我梦见吃蛋包饭。
莉香点了点头说: 我在梦中跳土风舞。
完治: 咦?
莉香张大了手比着: 大家围成一圈跳,和以前交往过的人轮流跳。
完治: 有几人?
莉香: 保密,你会吃惊的。

< 完治呆了一下,灰心的低下了头 >

莉香看见了,就说: 我胡说的,没有那么多。
莉香: 接着轮到你,我伸出了手...
完治: 嗯...

< 莉香的表情突然变得哀怨了 >

莉香继续说: 这时候音乐停了,你不见了,完治!...我觉得很孤独,就哭了起来..
莉香不高兴的说: 可是这时候你却梦见吃蛋包饭。
完治: 可是...

< 莉香这时绕着完治的身旁走了一圈,停在完治的面前 >

莉香: 来跳舞。
完治: 嗯...
莉香: 继续做梦...

< 莉香牵起完治的手,开始跳起了舞,口中并轻轻的哼起歌曲 >

莉香: I was dancing with my darling
To the Tennessee Waltz
When an old friend I happened to see..
( 背景也奏起了 Tennessee Waltz )

< 突然,莉香把头靠在完治的胸口,脸上露出了哀伤的表情 >

莉香: 不要丢下我一人。
莉香抬头继续说: 不能和别人在一起。

< 完治注视着莉香,紧紧的抱住她 >

完治: 不会再放开了。

< 莉香脸上露出了幸福的表情 >
< 两人在雪地中继续跳着舞,四周雪地一片寂静... >

S t a g e 5-2 < From Vol.5 >
背景: 完治家中

< 完治和莉香正在剥橘子吃 >

莉香: 你家的橘子真多,好吃!
完治: 家里寄来的。
莉香: 在爱媛!
完治: 嗯!
莉香兴奋的说: 完治出生的城市。
莉香转头问完治: 是什么样的地方?
完治: 什么样的地方啊....
完治想了一下,突然说: 有照片。

< 说着就去找照片 >

莉香高兴的说: 要看,要看。
完治: 在这边...

< 完治把一个盒子放在桌上,轻轻的打了开来,拿出里面的照片袋,倒出照片来 >
< 完治一眼看到最上面的婴儿照片,赶紧把它收起来 >

完治: 这个不行。
莉香笑着说: 看到了。

< 莉香一张张的看着照片 >

完治也拿起张照片,好像没印象的说: 这是什么!
莉香: 乡下...
完治: 真不好意思..
莉香: 我喜欢那种地方。
这时莉香拿起一张照片,问完治说: 这是哪里?
完治: 我小学念的学校。
莉香: 你在这里求学?
完治: 玩得比用功的多。
莉香: 玩什么?
完治: 踢罐头、荡秋千比谁荡得高...

< 莉香明白的点点头 >

完治: 也在后山玩游戏。
莉香: 嗯!?
完治: 作秘密基地藏宝贝。
莉香: 宝贝?
完治: 汽水空瓶、祭典时买的棉花糖棒...

< 莉香高兴的笑着 >

完治: 现在会想说这是在做什么嘛。
莉香向往的说: ...是一个平静的城市。 ( 这时背景画面渐渐转往爱媛 )
完治: 天空像蜡笔画的一样蔚蓝,暑假时有盂蓝盆舞大会、捞金鱼、放烟火...
莉香羡慕的说: 真好。
完治: 你知道吗! 坐纸箱从山坡上滑下去...

< 莉香点点头表示知道 >

完治大声的说: 很刺激的!
莉香: 不会受伤吗!
完治: 不会! 只有衬衫会被风吹涨...
完治得意的说: 不是我自负,全班我最会抓独角仙了。
莉香: 有独角仙吗?
完治: 有啊,下雨后很多。下了阵雨后,吹一声口哨叫朋友一起上山去...
完治: 好怀念唷!
完治接着说: 啊...对了..毕业的时候在校舍的柱子上刻下名字,刻了6年3班永尾完治..
完治讲到这,不禁失望的说: 可能已经没了...
莉香想了想后说:去看吧！
完治: 嗯...
莉香: 我也在旁边写下名字。
完治: 很远的。
莉香: 想去嘛! 想去你出生的城市。
完治: 那么...下次休假时带你去。
莉香高兴的说: 真的吗?
完治点了点头说: 真的。
莉香把小指伸出来说: 要作约定。
完治也伸出了小指,互相勾了勾说: 作约定。
莉香高兴的看着照片说: 有一天....两人一定会去。
完治也同意的说: 嗯!

S t a g e 5-3 < From Vol.5 >
背景: 公司地下室

< 完治正在整理着高尔夫球棍,莉香刚好走了下去,看见完治 >

莉香: 嗨! 你在啊!
完治: 我在这里。
莉香调皮的说: 不要等我喔!
完治虚心着回说: 谁会等你?

< 莉香露出诡异的笑容看着完治 >

完治: ....对了,你今天要来我家吗?
[ 你看,还不是露出了马脚,想邀莉香就说嘛 ]
莉香假装讶异的说: 什么!? 想要作什么色情的事吗?
完治急着辩解说: 不是。
莉香: 下次去借色情录影带来看。
完治不以为然的说: 什么嘛!
莉香作出夸张的表情说: 很厉害的!
完治好奇的问: 厉害吗?

< 莉香附到完治的耳边悄悄的说着话,完治听了后乾笑了几声 >

莉香指着完治说: 怎么脸红了。
说着双手贴近完治的脸颊说: 好热,好热。
完治把莉香的双手拉下,不好意思的说: 嗯..啊...晚上一起吃饭....
莉香: 啊!
完治继续说: 我今天...
莉香作出很抱歉的动作: 可惜,我约好了...
完治: 嗯...
莉香: 我们部门的大原先生请我。
完治酸溜溜的说: 要去吗?
莉香用力的点头回答说: 要去。
完治: 为什么?
莉香反问说: 要是女生请你,你也会去吧!
完治大声的说: 我才不去! 说什么话嘛!!
莉香笑嘻嘻的说: 会去的,完治很有人缘的。
完治自言自语的说: 我一个人好了。
莉香作出不小心听到什么的动作,把手靠在耳旁说: 咦...说什么啊?
完治: 没说什么。
莉香: 是嘛! 那我就要去了。
完治赌气的说: 去啊!!
莉香噘起嘴说: 我会去!!!
完治: 去啊!
莉香: 去了....

< 虽然嘴上说去了,可是实际上莉香并没有行动,反而走到了桌旁 >

莉香想了想,又对完治说: 不过....要是你肯说,拒绝他也可以。

< 完治不理莉香,自己在口中哼着歌 >

莉香看了完治这种态度,不禁说: 是嘛! 是这样嘛!!
莉香: 大原速度快,去喝酒也许就会动手了。...我是很纯情的。....大原要求的话也许会答应他...

< 完治还是不理莉香,莉香就把扩音筒拿起来大声说 >

莉香: 怎么办? 怎么办? 永尾完治的危机! 危机! 进场了...是大原....小心啊....这样下去...

< 完治这时听入了神,忘记了做事,莉香看到了高兴的说 >

莉香: 嗯...永尾完治无法做事!? 在动摇,在动摇了...
完治转身又好气又好笑的对莉香说: 你啊...
莉香: 终于要说实话了吗? 莉香!!不要去,留在我身边...
完治夺下莉香手上的扩音筒对莉香说: 吵死了!

< 两人对望着,不禁笑了出来 >

完治很正经的说: 今天是我生日。
莉香笑着说: 我知道,我会买东西回去,在家里等。
完治吃惊的说: 那...大原...
莉香一脸淘气的说: 我们部门里没有大原。
说着向完治挥了挥手说: 再见...

< 莉香说完就上楼去了 >
< 留下完治一个人呆立着,脑海中想着莉香刚才的恶作剧,脸上一付不解的表情 >

S t a g e 5-4 < From Vol.5 >
背景: 公司营业部

< 完治整理完文件,正准备回家,这时响起了电话声,完治顺手接了来听 >

完治: 喂, Heart Sport 营业部。
电话那端传来熟悉的声音: 永尾。
完治吃惊的说: 关口。

< 另一边,在回家的路上,蛋糕店的前面,出现了莉香的身影 >
< ~~~ 原来是莉香在观察着蛋糕... >

莉香问蛋糕店老板: 好吃吗?
老板: 当然。
莉香开玩笑的说: 真的吗! 要是不好吃的话,我会恨到你孙子那一代。
老板面带笑容很客气的说: 好的,好的!
莉香: 那好吧!
决定了后,莉香又看了看蛋糕,自言自语说: 太大了吧! 两个人吃不完。

< 不过莉香还是买了这个蛋糕,正当莉香提着蛋糕在去完治家的路上 >
< 这时莉香刚好看见在对街的完治 >

莉香高兴的叫着: 完治。

< 正当莉香高兴的准备到对街的时候,却看见了完治跟关口两人碰面,
不禁停下了脚步,只能目送着两人一起离去 > ( 这时响起了主题曲旋律 )
< .... >
< 过了不久,莉香到了完治家中,放下了蛋糕,而另一边,完治和关口在酒吧中.. >

关口: 什么都不问吗?
完治: 说出来会轻松些,我会听的。
关口不着边际的说: 人.....为什么会爱别人?.....只有痛苦....
完治安慰她说: 没有这回事,也有很多好事。
关口反问说: 什么事?
完治想了想: ....在一起会安心,还有...怎么说呢?....想努力,就像那样的心情。
关口: 但....我和完治你在一起时比和三尚在一起时更会感觉到....
关口打趣的说: ...或许我爱的是你。
完治不安的说: 不好....这样不好,不要说这种话,我也曾经爱过她,如今她这么说....
关口: 不是那样的意思!
完治: 我不愿看到她难过的样子。所以...

< 另一边,完治家中,莉香已经做好了一大桌的菜,把蛋糕盒打了开来 >

莉香看着蛋糕,自言自语说: 要先吃搂!
不过莉香想了想,又把拳头握了握,作出忍耐的动作说: 忍耐,忍耐!!!

< 镜头带回完治和关口这边... >

完治: 我该走了,她在等。
关口: 莉香吗?

< 完治没回答....关口点了点头,作出明白的表情 >
< 完治起身准备离去,顺手拿起了帐单 >

关口: 啊!...我来付!
完治: 没关系...
关口: 想在这儿多坐一会儿!
完治: 我付好了,...我会和三上再谈一次。
关口: 算了,不能再麻烦你了。
走吧! 回到莉香身边。快去.....
不快去...那么...
完治不知该说什么,只好应了一声: 那么...

< 完治正准备离去,关口啊了一声,完治回头看她 >

关口: 生日快乐!!!
完治: 谢谢。

< .... >
< 完治很快的回到了家,打开房门,走了进去 >

完治: 我回来了。

< 房里一片黑暗,像是没有人的样子 >
< 完治正失望的关起房门,走进屋内打开电灯时,突然听到砰的一声, 原来是莉香躲在旁边向他喷彩带 >

莉香高兴的说: Happy Birthday,完治。

< 完治看到莉香,不禁松了一口气,脸上也露出了高兴的神情 >

莉香跑过去拉着完治说: 这边,这边。
完治被拉到桌子旁,看到满桌的菜肴,吃惊的问莉香说: 全部自己做的吗?
莉香得意的说: 怎样!
顿了一顿,想起来说: ....蛋糕是买的。

< 莉香望着完治露出得意的笑声『 嘿嘿嘿嘿....』>
< 莉香和完治两人把电灯熄了,正在点着蛋糕上的蜡烛 >

莉香慢慢的数着: 四、五、六、七岁,小学入学,恭喜。...这时和三上相遇吗?
完治回答说: 没错 。
莉香继续数: 八、九、十、十一、十二、十三,中学入学,恭喜。...初恋?
完治露出不好意思的笑容指了指前面的蜡烛说: 更早,大概在这边。
莉香看了看,既羡慕又忌妒的说: 六岁! 真早!
莉香又数了下去: 十四、十五、十六,高中入学,遇见关口,而且一见锺情,恭喜。

< 完治不好意思的笑着 >

莉香笑了笑,继续数下去: 十七、十八、十九、二十岁,长大想做什么?
完治: 足球选手。
莉香: 可惜。
完治: 现在也这么想。
莉香笑了笑说: 是吗!
莉香继续把它数完: 二十一、二十二、二十三,到东京,这一年永尾完治的生平
有件最重要的事。...是什么?
完治: 遇见赤名莉香。
莉香高兴的说: 这暂且不谈....二十四岁。
莉香吹熄了手上的蜡烛,不满意的说: 我认识的完治的人生,只有这一根。
完治安慰说: 以后会增加的。
莉香点点头说: 嗯....!吹熄...

< 完治用力吹了一口气,想把蜡烛都吹熄,没想到还留下了一根 >
< 完治和莉香两人互看了一眼,完治说『开始』,两人一起吹熄剩下的蜡烛 >

莉香对完治说: 恭喜。
完治: 谢谢。
莉香: 来吃一吃! 也买了香槟。

< 莉香跑去冰箱拿香槟 >

这时完治说: 对不起,回来晚了。
莉香一边走到旁边去开香槟一边说: 没关系,没关系...
完治解释说: 想回家时客户忽然打电话来。

< 莉香停下了动作,完治看到了又改口说... >

完治: 不....客户忽然抱怨。...而别人都走了。我跑着回来的....对不起...。

< 莉香没出声也没有动作 >

完治一边站起身想去看莉香,一边说: 莉香...怎么了!?
莉香伤心的说: 为什么说谎?....和完治不相衬。...你不适合说谎。
莉香回过头来大声的说: 我讨厌说谎。 ( 同时响起了主题曲的旋律 )
莉香: 完治....讨厌那样的...

< 莉香伤心的转身去拿墙上挂的衣物离去 >
< 完治呆立着,当听到关门声时才想到莉香已经走了,赶紧追了出去, 只见莉香在前面跑着离开 >

完治一面追,一面大叫: 莉香~~~,莉香~~~~

< 莉香则流着泪,一句话也不回的向前跑着 >

完治继续叫着: 莉香~~~~莉香~~~~,等一下....莉香~~~~

< 完治终于在公园追到了莉香,完治抓着莉香的手臂,莉香也喘了两口气, 停下了脚步,但是莉香也把脸转向一边,不看向完治 >

完治道歉的说: 我说了谎...
莉香插嘴说: 虽然我这样不好.....但....我爱完治!!喜欢着你!!!
完治: 莉香。
莉香望着完治说: 好懊恼、好懊恼啊....完治...感情只有一个,没有两个,你放到了哪儿!!? 去滑雪 24 个小时,做工作、和朋友玩...我心里都只有完治一个人.. 你要全心说爱我,....要抓紧我,只看着我....
莉香流着眼泪说: 不然....我就离开!!

< 一直在莉香眼中打转的泪珠,这时流了下来,莉香抬高了头,不想让眼泪流下 >
< 过了一会,终于忍不住,把头垂了下去,让眼泪流出..... >
< 莉香伤心的流着泪离去 >

完治: 莉香.....

< 莉香没有回答完治,默默的走着 >
