第二幕 谣言


S t a g e 2-1 < From Vol.2 >
背景: 完治上班的路上

< 完治正走着的时候,看到了莉香,赶紧把大衣拉起来,想遮住自己不让莉香看到 >
< 经过莉香后,完治松了一口气,正在偷笑时,后面传来莉香的声音 >

莉香: 完治...

< 莉香拉住了完治,用手指指着树上说... >

莉香: 那个,那个....
完治: 嗯...

< 完治回头看见树上挂着一只鞋子,又看了看莉香,发现莉香右脚的鞋子不见了 >

完治: 鞋子怎么会挂在那里!?
莉香: 占卜天气嘛! ( 莉香举起右脚 )

< 完治跳起来拿了鞋子,交给莉香 >

莉香: 毕竟是男孩子!

< 两人接着一起走向公司 >
< 路上,完治看了一眼莉香的皮包,对莉香说.. >

完治: 早就想问了,你大皮包里装的是什么东西?
莉香: 爱与希望!!! ( 用力拍了拍皮包 )

< 这时莉香打了个呵欠.. >

莉香: 睡眠不足...

< 完治停了下来,脑海里浮现昨晚的事 >
< 莉香回头叫了完治一声 >
< 完治继续向前走 >
< 莉香偏着头看了看完治,然后嘟起了嘴巴说... >

莉香: 刚才你想逃!...如果我刚才没叫你,你会假装不知道的走掉...
完治: 我只是没发现。

< 莉香跑到完治面前,指着完治 >

莉香: 啊....不想见面!
完治: 不要胡说!

< 莉香噘着嘴不回答... >

完治: 谁叫你突然做那种事...
莉香: 我在卡车上提着皮箱突然跳车!?
完治: KISS...
莉香: 呀..呀...
完治: 什么?
莉香: 我义务服务,全年无休。
完治: 你同情我。
莉香: 嗯....
完治: 看到三上和关口接吻,所以...
莉香: 我会生气的...

< 莉香瞪着完治说... >
< 完治也回看莉香... >

完治: 那么为什么?
莉香又笑了起来说: 就算查电话簿也不知道。
完治: 嗯... ( 完全搞不懂莉香在说什么 )
莉香: 因为.... ( 没继续说下去 )


S t a g e 2-2 < From Vol.2 >
背景: 完治、莉香、关口在看完冰上曲棍球赛后,路上

莉香: 那...我走了...
完治: 嗯...再见。 ( 很高兴的说 )
关口: 还有工作吗?

< 莉香摇了摇头,发出表示没有的声音 >

关口: 那么大家一起去吃饭吧!
莉香: 可是... ( 看着完治 )

< 完治苦笑了一下,看了关口一眼 >

完治: 一起去吧!
莉香: 真的!!! ( 非常高兴的笑着 )

< 店里... >

莉香: 乾杯!
关口: 乾杯!
关口: 真快乐!
莉香: 值得吧! ( 指冰上曲棍球赛 )
完治插嘴说: 是我邀请的...
莉香不理他,对关口说: 冰上曲棍球好看吧!
关口点了点头: 嗯...
完治: 喂喂... ( 作出拨电话的动作 )
莉香: 喂喂... ( 作出接电话的动作和声音 )
完治: 是我邀请的。
莉香: 打错电话了... ( 作出切掉电话的动作 )
莉香转头对关口说: 是完治请客的呢...
完治继续说: 喂喂...
莉香: 是..是...

< 关口突然发出了啊的一声 >
< 完治望向关口... >

完治: 怎么了!
关口: 想起明天幼稚园的事,失陪一下。
完治: 好。

< 关口站起来,准备去打电话 >

莉香对关口说: 小心点,你不在时,完治也许会偷下印度迷幻药喔...
关口对莉香说: 你替我监视。
莉香: 是...

< 关口转身离开 >
< 完治看见关口离开后,压低声音对莉香说.. >

完治: 你什么意思吗?平常别人约会会有人跟来吗?
莉香: 什么..?什么..?你想做什么?
完治: 你就是这样,所以在公司会有奇怪的谣言。
莉香: 什么!?
完治: 我乱说的,没什么!

< 莉香放下了刀叉,瞪着完治,要他说出来.. >

完治: 是指你和和贺部长的事。
莉香: 喔...这件事啊...
莉香笑着说: 是真的,不是什么谣言。
完治: 为什么?
莉香: 爱上他没有办法。
完治: 但是他有妻儿..
莉香点了点头说: 我知道..
完治: 这不是畸恋吗?
莉香: 我知道..,但是我爱他...
完治: 只要爱,和谁都可以睡吗? ( 主题曲旋律奏起 )

< 莉香生气的看着完治,完治慌忙的低下了头.. >
< 这时莉香拿起了桌上的啤酒杯,把杯中的啤酒泼向完治... >

完治: 你....
莉香: 幸亏杯中的酒只剩一半。
完治: 你不要太过分了。
莉香大声的说: 那么你教我不会爱上人的方法...

< 完治张着嘴说不出话 >
< 莉香拿起桌上的毛巾丢给完治,对他说了声『笨蛋』后,转身拿起皮包,准备离开 >
< 莉香走了几步后,看见关口.. >

莉香: 对不起,我还有工作..
关口: 是吗?真可惜..

< 莉香没有回答关口的话,就离开了餐厅 >
< 关口向完治走过去 >
< 完治看着莉香离开,坐回了椅子上,顺手拿起莉香丢给他的毛巾擦了擦被弄湿的衣服,尴尬的看着不知情的关口... >

完治: 好热啊...

< 关口低着头没说话.. >
< 完治也说不下去,只能继续擦衣服... >


S t a g e 2-3 < From Vol.2 >
背景: 公司营业部,部长刚开完会走出来,莉香抱着资料跟在身后

部长大声叫: 后□..

< 后□没有回应 >

部长: 我一不在,他马上不见...

< 部长走向完治.. >

部长: 永尾..
完治抬头说: 是。
部长: 广告的事拜托你做了。
完治: 是。
部长对莉香说: 赤名,一半给永尾。 ( 指资料 )
莉香: 是。

< 部长继续去找后□ >
< 莉香把资料放在桌上,分成两份 >

莉香对完治说: 这边是北海道和关东的,拜托了。

< 莉香说完转身离开 >
< 完治拿起资料看了看,急忙对莉香说.. >

完治: 喂...没有分类表吗?
莉香: 有啊!但是我也要用。
完治: 一个一个找,要花三天的功夫。
莉香: 花三天不好吗!

< 完治看了看四周,低声对莉香说.. >

完治: 还在生昨天晚上的气吗?

< 莉香板着脸不说话 >

完治: 对不起,我道歉,昨天失言了..

< 完治弯腰对莉香深深鞠了个躬 >
< 莉香这时对完治笑了笑,然后对完治的电脑键盘用力的拍下去... >
< 完治惨叫了一声 >

完治: 消失了!!! ( 指电脑资料 )
莉香在旁边噘着嘴说: 我心灵的创伤可是一辈子都不会消失。
< 莉香说完后转身就走了 >

完治望着莉香的背影说: 真夸张....


S t a g e 2-4 < From Vol.2 >
背景: 咖啡店内

< 原本莉香和完治约好,完治要为失言的事道歉而请客,可是完治临时去见关口
,留了一张纸条在莉香桌上通知莉香,可是莉香没看见,还是去了咖啡店 >
< 莉香在店内打电话回公司,公司内没人接电话 >
< 莉香失望的放下话筒,走回座位坐下 >

莉香自言自语说: 还是回家吧!

< 这时外面下的雨变大了,莉香望向窗外,雨点打在玻璃窗上,顺着玻璃窗流下,
同时窗上也倒映着莉香的脸孔,两者融合在一起,滴落的雨点好像莉香的眼泪一样 >

莉香微笑着说: 好像流泪一样!

< 这时几位客人走进来,找不到座位 >

侍者对莉香说: 对不起,因为人很多,可以同席吗?
莉香: 我朋友不久会来...

< 莉香用手压着对面的座位,坚决的说.. >

莉香: 绝对会来..


S t a g e 2-5 < From Vol.2 >
背景: 酒吧中,完治一人独坐

< 完治正在喝着闷酒,这时从背后传来公司同事的声音.. >

同事: 永尾!..你不是和莉香在一起吗?
完治: 我有事。
同事: 可是莉香..去了!?
完治: 嗯... ( 这时主题曲响起 )

< 另一边,咖啡店营业时间结束了,把门关上了 >
< 莉香站在咖啡店外,搓着双手,在雨中等候完治 >
[ 哇! 好厉害,好痴心的莉香,怎么会这么能等呢? 佩服! 佩服! ]

< 这时完治从远方跑来,当看到莉香时不禁停下了脚步.. >
< 同时莉香也看见了完治,很高兴的叫着完治,并跑到完治面前.. >

莉香: 完治..,太好了,我以为你不会来了...
完治: 你一直在等吗?
莉香: 大概等很久,我没带手表,不知道时间,现在几点了?

< 完治抬起手,莉香看了看完治的手表,表上指着十点十分.. >

莉香: 这么晚!幸亏见到了!
莉香: 如果我迟到了,你会等我吗?就像莉香忠狗一样!

< 莉香摆了摆手指,装出忠狗的模样 >

莉香: 本来想回去,但是想到你请客的机会不知道下次还有没有?如果失去了这次机会,
也许会觉得很可惜!

< 完治接过莉香的雨伞,对莉香说.. >

完治: 我们去另一家...

< 莉香把头轻靠在完治的胸口 >

莉香: 不想去!好累喔!好像线断了一样,一点力气都没有,哪里也不想去...
莉香抬头望着完治说: 你看起来为什么那么遥远呢?
莉香: 你能来,谢谢!我很高兴!
莉香: 真的!我很感动的!
莉香又轻轻的说: 为什么会想再一起呢?...为什么一直这样想呢?...

< 莉香不再说话,转身离去 >
< 完治赶紧跑过去,把雨伞拿到莉香手上,莉香把它接了过去,继续走着... >
< 完治默默的注视着莉香离去的背影,而莉香却茫然着,
好像失了魂一样的向前走着.. >
