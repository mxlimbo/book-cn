第四幕 原谅


S t a g e 4-1 < From Vol.4 >
背景: 午夜,莉香家中

< 完治躺在床上,莉香在旁边轻敲着完治的胸口 >

完治不耐烦的说: 喂..
莉香: 再来一次...

< 完治转过头,继续睡觉 >

莉香: 不肯吗!完治!

< 完治不回答.. >

莉香: 算了...

< 莉香躺了下去,把完治的右手当枕头,也睡了起来 >
< ... >
< 清晨,莉香刚醒过来,一眼看到完治已经穿好了衬衫,正在打领带 >

莉香: 早安。
完治: 早安。
莉香笑着说: 起得真早,是要听收音机作早操是吗!

< 莉香这时也起了床 >

莉香: 我来泡咖啡。
完治: 不必,不必了。
莉香温柔的说: 外面很冷的。

< 莉香说完就去泡咖啡了 >
< 莉香和完治两人坐在沙发上正喝着热腾腾的咖啡 >
< 莉香看了看窗外... >

莉香: 天气真冷啊!
完治: 嗯!

莉香: 不知道下雪了没有?

< 完治没有回答,好像在发着呆,莉香看了觉得奇怪 >

莉香: 已经夏天了!
完治: 嗯!
莉香: 该买泳衣去海边了!
完治: 嗯!
莉香: 完治是笨蛋!
完治: 嗯!
莉香: 我好可爱喔!
完治: 嗯!

< 莉香看了看完治 >

莉香: 怎么了?
完治突然回过神来: 嗯~~~~
莉香: 你在发呆。
完治: 没什么。
莉香: 后悔吗?
完治: 什么事?
莉香: 和我变成这样。
完治好像一副不明白的说: 变成这样!?
莉香: 想起关口吗?
完治: 噫...
莉香: 拥抱我的时候。
完治: 嗯...
莉香: 你说『嗯..』的时候,表情很容易让人知道。
完治: 不要胡说!
莉香愉快的回答着: 是~~~~


S t a g e 4-2 < From Vol.4 >
背景: 夜,完治家中

< 完治躺在床上,手中抛接着新产品的鞋子,在想着心事 >
< 这时完治看了看时间,凌晨 2:05 分,完治忍不住去打了电话 >
< 另一边,莉香家中,莉香正在为她种的花浇水,这时电话铃声响了.. >

莉香: 喂喂...
完治: 我是永尾。
莉香惊讶的说: 完治!!
完治: 我问你...
莉香高兴的说: 什么...?
完治: 通常这时候你都会打电话来,今天怎么没打?
莉香: 怎么了?
完治: 要让你体验一下半夜接到电话时,我的心情。
莉香: 胡说,你其实是想听我的声音。
完治: 笨蛋,不要乱说。
莉香淘气的说: 那么..为什么!!
完治: ...今天去了 Alpen...
莉香: 怎样了?
完治: 很顺利,根本不用你帮忙,你一点价值都没有。
莉香失望的说: 啊....这样啊!
完治: 嗯...你和那家店的主任有什么关系吗?
莉香: 怎么了?
完治: 因为...
莉香: 什么?
完治: ...对了,有没有找我的手表?
莉香: 嗯...这是你第一次送我的礼物。

< 莉香拿起桌上完治留下的手表把弄着 >

完治: 不要随便决定!
莉香: 拿来做护身符。

< 这时莉香看了看时间 >

莉香: 啊...已经是这种时间了。
完治大声的说: 喂...想据为己有吗!
莉香: 反正我什么都不是。

< 说完切掉电话 >
< 挂了电话后,完治仍为新产品的事烦恼着,莉香则高兴的玩着手表.. >

S t a g e 4-3 < From Vol.4 >
背景: 公司地下室

< 莉香正整理着网球拍,觉得后面有人来了,回头一看,原来是完治.. >

莉香: 嗯...? ( 不知完治有什么事 )
完治: 你打电话给 Alpen !
莉香: 嗯!!
完治: 不要你多管闲事。
莉香: 只是打个电话...
完治大声的说: 叫你不要管。

< 莉香这时在箱子上站不稳,差点跌倒,完治连忙扶了她一把 >

莉香: 谢谢! ...怎样了?
完治: 怎样! ...总算谈好了。
莉香高兴的说: 太好了!!
完治: 太好了!?
莉香高兴的说: 嗯...

< 完治叹了口气,烦恼着该如何对莉香启齿 >

完治: 我说啊...
莉香疑问的说: 嗯...?
完治: 还是算了。
莉香: 什么事?

< 完治没有回答,正在烦恼着 >

莉香走到完治前说: 令人担心,这个样子绝对令人担心。
完治只好说出来: 这次休假你有空吗?
莉香高兴的说: 约会吗!!
完治: 约好和关口、三上见面。
莉香失望的说: 因为怕不好受,所以叫我也参加吗?
完治: 我没那样说。
莉香肯定的说: 就是那样。
完治: 那就算了,不麻烦你。

< 说完完治转身准备上楼 >

莉香迟疑着对完治说: 上次看见关口...
完治停下了脚步: 是吗!
莉香: 关口现在和健一交往...

< 完治没说什么,莉香看了后,轻声的说.. >

莉香: 你知道吗?
完治: 知道。
莉香: 啊~~~原来是这样。难怪....
完治: 难怪怎样?

< 完治又走了下来 >

莉香: 没什么!
完治: 说啊! 这样会让人担心。
莉香挂着笑容说: 因为关口的事,所以那天晚上会和我睡,不然...

< 完治不知该如何辩解... >

莉香: 嗯...这样就明白了..。这样的话,那天晚上的事就此扯平了。

完治: 不要随便决定。

< 完治不知该如何辩解... >

莉香: 嗯...这样就明白了..。这样的话,那天晚上的事就此扯平了。
完治: 不要随便决定。
莉香: 这样你轻松了吧! 我很轻松了...
完治: 你说谎的吧!!
莉香: 不是说谎,是真的。

< 完治两眼注视着莉香,想从她脸上看出什么,但只看到莉香脸上的微笑 >

莉香: 就这样决定了。

< 莉香说完转身离去,准备上楼,但走了几步回头对完治说.. >

莉香: 可以陪你去。
完治: 嗯...
莉香笑笑的说: 要和关口见面吗? 我会做中间人的...

< 完治看着莉香的笑容,不知这个女人心里到底在想着什么... >


S t a g e 4-4 < From Vol.4 >
背景: 公园中

< 和三上及关口分手后,完治和莉香两个人走在公园中... >

完治: 喂,不回家吗?
莉香: 完治~~,关口的事怎么办?
完治: 根本就没什么事了。
莉香: 『喜欢』,你还是喜欢她。
完治转移话题说: 刚才真是对不起,只是...你突然那么说,让人觉得着急。关口的事并没有怎么样。

< 莉香坐到秋千上轻轻座着 >

莉香不肯说出真正的心意: 没关系。我也是因为无聊所以才会和你睡,并不是有了爱情。
完治: 是假的吧!
莉香: 嗯...?

完治: 是假的吗?...一直只是玩弄我吗?

< 莉香温柔的看着完治片刻,转过头望向远处 >

莉香: 是啊!...我说爱你,你就着急...
莉香笑着继续说: 啊~~~,真快乐。

< 莉香看完治沉默着,就跳下秋千,注视着完治 >

莉香: 怎么了?

< 完治没回答 >

莉香关心的说: 怎么那么悲哀的脸!
完治: 为了玩乐就可以睡的女人吗?
莉香不明白的问: 你说什么?
完治: 网球鞋的契约也是...
莉香讶异的说: 嗯....
完治: 不是和他睡觉才得到工作吗!
莉香: 你在说什么!!
莉香沉下了脸,生气的大声说: 你在说什么!!!

< 莉香注视着完治,不想和他辨解 >

莉香生着气说: 我要回家了。
完治: 等一下,还没说完。
莉香赌气的说: 和谁在一起是我的事,你管不着...
完治大声的回答: 不能不管。

< 莉香停下了脚步,回头对着完治说 >

莉香: 奇怪了! 你说话乱七八糟的。...我们根本一点关系都没有。
完治: 因为...
莉香插嘴说: 只是因为完治是个温柔的人,所以才会担心我,这样而已。

< 两人沉默的对望着 >

莉香: 好了,忘掉吧!
完治: 嗯...
莉香: 是几时?...啊,对...是机场...从那时起到今晚的事全部都忘掉。
完治: 哪有那么简单!
莉香: 回到起点,把回忆全部忘掉。

< 完治望着莉香,莉香强忍着悲伤对完治笑了笑 >

莉香: 晚安...

< 莉香向后退了几步 >

莉香: 再见...

< 莉香向完治挥了挥手 >

莉香: Bye Bye.

< 说完莉香转身离去 >

完治: 等一下。
莉香: 才不等。

< 莉香转身飞奔而走,这时脸上终于流下了泪水 >
< 莉香走后,留下完治一个人在公园中独自发着愣... >

S t a g e 4-5 < From Vol.4 >
背景: 完治家中

< 完治坐在床上,独自看着时钟的秒针一秒一秒的走 >
< "终于走到十二点了,今天就是 28 号,是莉香出差回来的日子了", 完治心里这么想着 >
< ... >
< 清晨,完治一口喝完了鲜奶,赶着去上班,心想可以见到莉香了 >
< 完治急忙的在路上跑着,回头看见莉香正和同事走过来,于是停下了脚步, 挂起笑容等待莉香走过来,准备向她打声招呼 >
< ...莉香和同事走过完治的身边 >

完治: 早安。
同事: 早安。

< 莉香没有回答,继续和同事聊着天,当完治不存在似的 >
< .... >
< 公司中,完治坐在座位上发呆,旁边传来同事的声音 >

同事: 啊...赤名小姐。

< 完治抬头一看,原来是莉香拿礼物来营业部办公室分给大家 >

莉香笑着对大家说: 礼物。 ( 用手指了指 )

< 莉香把礼物拿给和贺部长 >

莉香: 部长,广告顺利办好。
部长: 辛苦了。
同事们拿着礼物高兴的说: 啊...这是什么!
部长对莉香说: 还顺利吧!
莉香: 是! 本来担心的,总算...
渡边拿着礼物招呼完治说: 完治,来这边。

< 完治走了过来,正想开口对莉香时... >

莉香马上对部长说: 失陪了。


< 莉香说完立刻就离开了,完治连讲话的机会都没有 >

渡边队完治说: 完治,吃哪一个?
完治没心情吃,于是说: 对不起,肚子不舒服。

< 完治说完急忙跑出去想追莉香,看见莉香正在等电梯,就走到莉香身后 >

完治满脸笑容的说: 回来了啊!

< 莉香不理他,脸上一点表情都没有 >
< 这时电梯到了,莉香走进电梯,按下了开关 >
< 完治站在电梯门前,欲言又止的说... >

完治: 我说...

< 电梯门这时关了起来,莉香从头到尾都没说一句话,完治站在电梯门口懊恼着 >
< ... >
< 公司下班后,完治走回自己的座位上,正在烦恼着,这时脚却轻易的踢到了一个东西,拿起来一看,原来是个冰筒 >
< 完治轻轻的打开了冰筒,里面事一个小雪人,雪人身上放着一个牌子, 上面写着"完治",完治看到这些,心里不禁激动了起来,口里叫了声"莉香" >
< ... >
< 莉香家中,电话铃声响了好久,莉香终于去接了电话,电话是完治在公共电话庭中打来的 >

完治: 喂喂...

< 莉香没有回答,两人沉默着....完治过了一会终于忍不住了 >

完治: 想见面。
莉香轻轻的点点头回了声: 嗯...

< ... >
< 广场前,完治正坐着... >
< 这时传来踢罐子的声音,完治一看,是莉香来了 >

莉香笑着说: 我回来了。 ( 主题曲响起 )
莉香: 完治。
完治看见莉香的笑容,心情安定了下来,说: 什么嘛! 出差要说一声!
莉香: 你又不是部长,为什么要跟你说!
完治: 是吗? 是吗? 人家是担心你!
莉香可爱的说: 你担心我吗!
完治: 我心跳得很厉害呢!
莉香: 应该去看医生吧!
完治: 我不是心脏有毛病!

< 这时完治作出打电话的动作 >

完治: 喂喂...
莉香作出接了电话的动作: 喂喂...
完治: 网球鞋的事很顺利。
莉香: 是吗!
完治: 你的功劳。
莉香: 完治不会待人处世,所以我要想办法帮忙。
完治这时忍不住又说: 不用你帮忙。
莉香: 是吗!!!

< 这时莉香向完治伸出了左手,手上带着完治的手表 >

莉香摇了摇手表说: 你看...太大了,太大了....
完治笑着说: 说谎。
完治看着微笑的莉香说: 只要你还带着,不是不会忘记我们的事吗?
莉香听了,回嘴说: 可是,我现在同样还是能用说『爱你』时的心情跟你说『再见』的。

< 完治摇了摇头表示不要 >

莉香高兴的说: 你那么喜欢我吗!...我一点都不知道,是那样的吗?
莉香走了几步说: 原来真是这样。但是我也是有脾气的,不会那么简单的原谅你。
完治: 我会努力的。
莉香: 如果我在半夜寂寞的时候,你会飞也似的赶来吗?
完治: 会飞奔而去。
莉香: 如果我在喜马拉雅( Himalayas )山顶打电话给你要你来接我呢?
完治: 会去接。
莉香: 如果要你带热腾腾的黑轮来呢?
完治比了个大手势说: 我会带一大堆。
莉香: 如果我要在家里开 Beatles 的演唱会呢?
完治: 我会去请他们来。
莉香: 那 John( Lennon )呢?
完治: 我会替他唱。
莉香: 如果我要你施魔法使天空中马上出现彩虹呢?
完治: 这个...大概就没办法了...
莉香: 那么...就不行了..
完治: 不过,我可以施魔法。
莉香: 怎么样的?

< 完治靠了过去,深深的吻了莉香 >

莉香: 我喜欢你。
莉香: 你喜欢我吗?
完治点了点头: 嗯...
莉香: 说『我爱你』。
完治: 我爱你。
莉香: 要叫名字。
完治点点头,深情的说: 我爱你,莉香。
莉香幸福着笑着,也说: 我也爱你,完治。

< 莉香这时把自己的围巾解了开来,把一半披到完治的脖子上 >

完治: 对了,谢谢你的雪人。
莉香: 不客气。.....手好冷喔!

< 莉香搓着双手 >
< 完治握住莉香的双手,把它们放进自己大衣的口袋中取暖 >

莉香: 完治。
完治: 嗯?
莉香: 我一直在想,你绝对会爱我的。
完治: 你说得对。

< 看着莉香的笑容,完治也忍不住笑了起来 >
< 两人高兴的笑着,沉浸在幸福的气氛中 >
