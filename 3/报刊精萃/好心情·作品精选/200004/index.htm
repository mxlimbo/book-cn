:: 特别推荐 ::
　　雪　花 [花谢]
　　搞不懂是下了雪天才冷的，还是天冷了才下雪的。
　　就如同分不清是大雪飘在我的窗外，还是我的世界原本就在大雪里。

:: 挚爱慈亲 ::
　　明月千里寄相思 [曾诚]
　　有一种叫思念的情感，无论漂泊了多长时间，流浪了多远路程，都会在这样的时节盈动于心，润湿于眶，让心找到归依，让爱有了港湾。
