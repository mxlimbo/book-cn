《 上网冲浪好工具――Ativa Pro 》

{　　Ativa Pro是一款出色的网络带宽监视工具，它能够让你对自己所处的网络状况了如指掌。另外，它还附带了一些非常实用的功能：可以为你进行网络参数的优化，让你快猫加鞭；可以帮你自动ping远程服务器，让你的网络长连不断；可以帮你检查邮箱，让你有信早知道；可以通过internet时间服务器校准你的系统时钟。
　　一、软件的下载与安装
　　Ativa Pro是一款共享软件，文件大小763KB，最新版本v3.05，未注册的版本可以免费试用10天，下载地址：http:∥www.ativapro.com。下载的是个自解压安装程序，直接运行该程序便进入安装界面。安装完毕，你可以通过开始菜单程序中的快捷项启动该程序。另外，你也可以不进行安装，手工解压缩后，直接运行其中的AtivaPro.exe便可启动Ativa Pro（如图）。
　　二、软件的设置和使用
　　1.软件的基本功能设置
　　点击“Cmnds”按钮，弹出“Ativa Pro Command Preferences”对话框，该对话框中一共有六个标签页面。
　　(1)“Lauch”标签页面：在该页面中，你可以设置Ativa Pro接通或断开Internet自动启动的程序。
　　(2)“Net Time”标签页面：在该页面中你可以让Ativa Pro为你校准系统时钟。
　　(3)“Auto Connect”标签页面：在该页面中你可以设置Ativa Pro自动拨号上网的时间和断线下网的时间。
　　(4)“Notifications”标签页面：在该页面中可以设置Ativa Pro执行各项任务时或者当网络传输的数据达到预设的数值时发出声音提示或文字提示信息。
　　(5)“E－mail Check”标签页面：在该页面中可以设置Ativa Pro自动检查邮箱的时间间隔、发现新邮件后给出语音提示并显示出新邮件列表。在这里需要输入欲检查邮箱的POP3、用户名和密码。不过，它只能检查一个指定的邮箱。
　　(6)“Auto Ping”标签页面：定时地ping你的接入服务器，可以避免因用户长时间没有响应而被服务器中断网络连接的现象。如果你想启用该功能，请选中“Auto Ping the following host”项，并在其后设置自动ping服务器的时间间隔，你还需要在该项下的正文框中输入你的服务器地址。另外，你在该页面中还可以进行ping测试，以检查你的网络状况。方法是在“Ping Test”下的正文框中输入你欲连接服务器的地址，然后点击“Ping”按钮，通过列表框中显示出的回应信息，你便可以了解到欲连接服务器当前的状况。
　　2.软件的基本参数设置
　　点击“Prefs”按钮，弹出“Ativa Pro Program Preferences”对话框，该对话框一共有四个标签页面。
　　(1)“Preferences”标签页面：“Start Ativa Pro when Windows starts”项：如果你想让Ativa Pro随着Windows一起启动，请选中该项。“Start minimised in system tray”项：如果你希望Ativa Pro启动后即缩至任务栏的系统托盘中，请选中该项。“Display Floating Bandwidth windows”项：如果你希望单独出现一个网络带宽检测窗口，请选中该项。“Ativa should always appear on top of other windows”项：如果你希望，Ativa Pro总是显示在其它窗口的前面，请选中该项。
　　(2)“Graph”标签页面：在该页面中可以设置网络带宽检测窗口曲线图中各种图形数据所显示的颜色，你可以把默认的颜色改换成自己喜爱的颜色。
　　(3)“Monitor Connection”标签页面：在该页面中你可以设置时间提醒功能，当你上网时间达到预设的数值时，Ativa Pro便会给出声音提示。该功能可以帮你控制上网的时间。
　　(4)“Web Update”标签页面：该页面提供了在线版本更新功能，不过该功能只对正式用户开放。
　　3.网络参数优化
　　点击“Tweak”按钮，弹出“Ativa Pro Tweak DUN Settings”对话框。在该对话框中可以对很多的网络参数进行优化，这样可以大大提高你的上网速度。对于网络新手来说，可以直接使用Ativa Pro提供的优化方案，只要点击一下“Set Optimimum”按钮即可。如果你对各种网络参数比较熟悉，可以打开“Tweak Speed”和“More Tweaks”页面，自行设置其中的各项网络参数。Ativa Pro还提供了恢复系统初始设置的功能，有了这个保险开关，你就可以放心大胆地修改各项网络参数了。万一修改错了，你只须点击一下“Set Basic”按钮即可。需要说明的是由于这里的各项设置都会对注册表进行修改，所以必须重新启动计算机设置才能够生效。
　　4.Ativa Pro连接报告
　　Ativa Pro能够为用户提供一个详细的网络连接报告。点击“Report”按钮，可以打开“Ativa Pro Connection Reports”窗口。在该窗口中，可以全面地了解到你所处的网络状况，以方便你日后制定出适合自己的上网计划。在“Log Details”窗口中，可以了解到每次上网的时间、时长、连接速率、发送和接收的字节数。在“Performance”窗口中，除了可以了解到日志中所有的信息之外，还能够了解到你每次上网时使用的IP地址以及你接入服务器的IP地址。另外，这里还给出了你每次上网的综合效率，单位为％，通过该指标，你可以清楚地了解到最佳的上网时段。用鼠标右键点击窗口中红色的长方体就可以调出详细信息的显示窗口。在“Weekly Stats”窗口中，通过柱状图形，为你提供了一周内每天接收和发送数据的比例。在“Transfer Stats”窗口中，你可以通过图表了解每个月内数据接收和发送的详细情况。在“Summary”窗口中，Ativa Pro会给出你当日网络连接的报告摘要，使你对当日的网络状况一目了然。
　　如果你感兴趣，不妨下载一试。}
