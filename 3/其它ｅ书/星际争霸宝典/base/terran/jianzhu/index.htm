Terran建筑详细资料：
Supply Depot（房子）
 

花费资源：100
建造时间：40
生命值：500
装甲类型：重
装甲：1
建造快捷键：S
生产部队需要有足够的房子，一个房子可支持8个单位。最好在人口上限没到之前就建造。
Command Center（主基地）
  生产：

SCV （工人）

花费资源：400
建造时间：120
生命值：1500
装甲类型：重
装甲：1
建造快捷键：C

基础建筑，SCV采来的矿和天然气都要送回这里。SCV在这里生产。达到一定技术等级后主基地可以增加雷达站和核弹发射井两个附加建筑，但一个主基地只能选择一个。
ComSat Station（雷达站）
  特殊能力：


雷达扫描

需能量：50

花费资源：5050
建造时间：40
生命值：750
装甲类型：重
装甲：1
需要：Academy（研究院）Command Center（主基地）
建造快捷键：C（选取主基地后）
雷达站使你拥有雷达扫描的能力。雷达扫描可以让你在一定时间内看到任何一部分地图的情形，同时在扫描区域里的隐形部队也会被发现。
Nuclear Silo（核弹发射井）
  特殊能力：

核弹2002008

花费资源：100100
建造时间：80
生命值：600
装甲类型：重
装甲：1
需要：拥有Cover Ops Center（谍报中心）的Science Facility（科学工厂）Command Center（主基地）
建造快捷键：N（选取主基地后）
制造核弹的场所，制造后可由特工释放。
