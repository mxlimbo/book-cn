战网命令列表

 

/help----用于查询,可列出此战网上所有的指令
/whois +ID----查目前这个玩家在哪个游戏以及正在干什么
/stats +ID----查某玩家的资料料(主要是战绩及Disconnect记录)
/me +字句----自己试试看:)
/ignore +ID----封人给讨厌的人一个大叉叉,他就不能与你谈话了
/time----显示当前时间
/whereis + name----显示此人所在频道
/whois + name----显示此人所在频道(与国际连通时可显示对方ip)
/who + game's name----显示在此游戏中的人
/stats + name----显示此人的战绩
/away + 信息----别人和你私聊时会就出现你所输入的信息
/status----显示当前有多少人在战网
/team create + name----建立name组织( /team create + 你的名 + 你要建立的队名)
/team info +组织name----显示组织名单
/team list----显示所有已建队的队名
/team add + name----邀请某人加入自己的组织
/team join + 组织名----加入组织(别人必须已经邀请了才能加入)
/team del + name----删除组织
/help team----显示所有team命令
alt + q----退出星际
alt + n----复制人名
alt + v----可知道玩家join and left 的状况
/w +ID+ 加你想讲的话----可以和正在里面玩游戏的人说话
