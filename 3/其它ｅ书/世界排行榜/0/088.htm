17个拒绝接受重要角色的电影明星

　　1．马龙・白兰度
　　拒绝了影片《金手臂》（1955）中一个吸毒成瘾的音乐家的角色。后来该角色由弗兰克・西那特拉担任，西那特拉的表演出色，一炮打红，从此确立了在电影界的地位。

　　2．詹姆士・卡格尼
　　拒绝了影片《窈窕淑女》中杜利特尔的角色，此角色后来由斯坦利・霍洛韦担任。当时电影公司答应给予高达100万美元的报酬，可卡格尼不愿打破他平静的退休生活。

　　3．蒙哥马利・克利夫特
　　克利夫特起初对影片《日落大街》（1950）中的男主角――一个年轻的作家――表现出十分强烈的兴趣。可最后他还是拒绝了这个角色，因为他认为他的观众不会喜欢看到他同一个比自己大35岁的妇女恋爱的场面。后来威廉・赫尔登和格劳丽亚・斯旺森分别扮演了这部受到广泛赞誉的影片中的男女主角。

　　4．贝特・戴维斯
　　拒绝了影片《飘》（1939）中女主角郝思佳的角色。她原以为影片中的搭档将是埃罗尔・弗林，而她最不愿意同此人合拍电影。此角色后来由费雯利担任。

　　5．柯克・道格拉斯
　　拒绝了《巴罗猫》（1965）中基德・希林的角色。后来李・马文因担任此角而荣获奥斯卡金像奖。

　　6．W・・菲尔兹C
　　影片《奇才奥兹》中的主角，一个骗子，本来是专为菲尔兹创作的，可菲尔兹拒绝出演此角，据说是因为米高梅公司只答应给他75000元的报酬，而他要求100000元。但根据菲尔兹的经纪人所写的一封信所说，菲尔兹是为了集中时间，写好自己的剧本《不能欺骗老实人》。最后由弗兰克・摩根扮演了这一角色。

　　7．简・方达
　　拒绝了影片《波尼和克莱德》（1967）中波尼・派克这一角色。当时方达在法国居住，她不想为拍此片而长期离开自己的寓所。后此角的扮演者是费伊・达纳韦。

　　8．加利・格兰特

　　拒绝了扮演影片《让我们相爱》（1960）中的男主角（琼马克・克莱门特）。该角后由伊维斯・蒙坦特担任。格兰特拒绝的原因是他不愿同影片中女主角（阿曼达・德尔）的扮演者玛・丽莲・梦露合作。

　　9．威廉・霍尔顿
　　拒绝了在影片《纳瓦隆的手枪》（1961）中扮演主角的邀请。霍尔顿曾要求公司支付75万元的报酬，但公司没同意，此角后来由格利高里・派克担任。

　　10．艾伦・拉德
　　拒绝出演影片《巨人》（1956）中的主角林克，因为他感到自己年龄太大，不适合扮演此角色。最后由詹姆士・迪恩担任。

　　11．海迪・拉马
　　拒绝了影片《卡萨布兰卡》（1942）中伊尔莎的角色。结果担任此角的英格丽・褒曼同男主角博加德的精彩表演使该片获得了巨大的成功。

　　12．伯特・兰卡斯特
　　拒绝了影片《班赫》（1959）中的角色，后来该其中的主人公班赫由查尔顿・海斯顿扮演，他因担任此角而荣获奥斯卡金像奖，这成为他辉煌的电影生涯中的一个新的里程碑。

　　13．默纳・洛伊
　　拒绝了在影片《一夜风流》（1934）中担任女主角，她以为该岂不会成功，但最后克劳德特・科尔伯特因演此角而获奥斯卡奖，在该其中担任主角的还有克拉克・盖布尔。

　　14．保罗・穆尼
　　拒绝了影片《高高的山岭》（1941）中的角色劳伊・厄尔，穆尼说他对扮演匪徒的角色已感到厌烦。该角后由博加特扮演。博加特把这角色演得出神入化。

　　15．乔治・拉夫特
　　拒绝出演影片《高高的山岭》、《马耳他之鹰》和《卡萨布兰卡》中的主要角色。结果全由博加特承担。这三个人物也成了博加特一生中所扮演过的最为成功的角色。据说拉夫特之所以拒绝出演《马耳他之鹰》是因为他不愿同当时还默默无闻的导演约翰・赫斯顿合作。

　　16．罗伯特・雷德福特
　　拒绝了影片《毕业生》（1967）中的主要角色本・布莱道克，可这一角色却使达斯汀・霍夫曼一举成名。罗伯特说他无法把握布莱道克的天真和无知的特点。

　　17．埃娃・塞恩特
　　塞恩特选择角色的挑剔是出名的，她不随意接受AE？约。但她拒绝接受出演影片《夏娃的三张脸》（1957）中的主角的这一决定显然是个极大的错误。结果乔安娜接受了此角，获得奥斯卡奖。
