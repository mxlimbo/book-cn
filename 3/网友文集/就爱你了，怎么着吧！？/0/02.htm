《居天有我》- 和天骄

这个国家的诗人已全部死去（天骄） 居天有我 （小爷儿.汤）
我不要做这个国家的诗人
过于悲痛的汉字
过于悲痛的神
除了泪滴，一切都是空的
除了干旱，干旱是如此无力
风沙之外还是风沙
石头之外还是石头

神伤尚未散尽
天一下子就黑了
大黑暗，披在大地身上
我看见大黑暗的尽头
是更大的黑暗

请拴住各自的性命
请不要随意走动
哀鸿已全部死去
它们死于青春！
我不要做这个国家的诗人
这个国家的诗人
如今已全部死去

对于我，世俗的美已经足够
庸俗的无数种化名
我只取幸福

我将娶幸福为妻
红尘白日，生儿育女
看我把剩下的日子过完
结庐而居的这一份平安呵
对于我
平安已经足够

我将娶幸福为妻
这世俗的美
又近，又累
我只为它写作
而不去做这个国家的诗人

我心如哀鸿
只告诉天空

有一首自远古致今日的长歌
无息无尽从未停过
没人能解读歌中的文字
更无法复述乐律的顿挫
但奇韵荡澈心宇
如踏苍凉大漠畅舞
似撼悠悠雄关豪歌

阅历五代的贞朴
叹尽秦汉的雄魄
今日天下有何可为
日月亦为多余的荧火
朝夕更是无奈中的烦琐

诗人本是俗物
神洲只纳清歌
仙人已乘风归去
把酒方去大惑
莫谈词语阕句
神韵天机无破
感死才炼得魂拓

恋世间温情万种
取俗为大道解脱
心弈淡漠

君可知
唐宋可渡安日?
诗仙依旧流落
绝句从未人为
你奈何
求得欢容更寂寞

君不闻
天地之间歌未尽
碌碌尘尘索索
问天命
赐你玩词烹句,为庸人小佐

你心无境
居天有我
