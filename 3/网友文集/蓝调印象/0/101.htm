缘 来 是 你

　　当年她和他在一起时感情极好，虽然时常吵架，但也很甜蜜，谁能保证年轻气盛的他们在一起能不拌嘴呢。和他腻在一块的时候，她对他说：如果她对他用手做一个动作，就表示她爱他。他一点不解风情问她好好的做什么手势，说出来不就得了~她耐心的说："这个竖起大拇指食指和小拇指，窝起中指和无名指的动作在聋哑人中就表示'我爱你'的意思。大拇指表示'我'，食指表示'爱'，小指表示'你'。如果我们见面不方便说话的时就可以做这个表示。将来，等我们年老了没力气说话的时也可以做这个手势。有一天，如果我们分开了，也可以做这个表示呀~"她咭咭的笑，又故意板起脸："我若做出这个动作，就表示我依然爱着你这个笨家伙！"他也笑了，说好吧，"等你做这个手势的时候我就做这个手势，"他竖起了拇指和小指把其他的手指窝起来。"这是什么怪手势？！这不是个'六'嘛！"她瞪着他的手，他揽过她在她耳边细语："这个'六'的含义不一样，它比你的动作少一个手指，只有拇指和小指，表示只有我和你，爱已经放在心里了，我们会永远在一起，永远爱着你......"

　　其实他挺喜欢她的，喜欢她的这些怪念头，喜欢她的精灵古怪。有时很多人前，见到她，她一脸正经的走过去仿若不认识他，手底下却做着那个聋哑人的'我爱你'的动作。他也一本正经的走过去，做一个'六'的动作。

　　但他们也时常争吵的，两个人脾气都倔，谁也不肯让谁。有一次说好去青岛玩一路还好好的，在栈桥却吵起来。他说他本来想找一个乖巧听话的女朋友，谁知道她那么脾气这么倔这么大！她恼怒道："本来？我本来还想找一个比我高很多的人呢！最起码有1.8，你有吗？！"他也恼了，他的确不高，其实1.75也不算矮了，但是和1.7的她站在一起就显得差不多，尤其她总不肯放弃穿高跟鞋，两人站在一起似乎她比他还高。触到隐痛处他反唇相讥："我是没1.8，我要找的是个小巧玲珑的女朋友，要那么高干吗~！"她脸色也变了，两个人在栈桥吵了了淅沥哗啦。

　　分手几年以后，他又来青岛。这次是和他新的女朋友在一起，应了他的话，他找的是一个仅有1.56小巧玲珑，乖乖巧巧的女朋友，站在一起到是般配之极。两人来栈桥边上看海，同样的地方，不同的人，他不禁有点感慨。真的有点想她呢，虽然现在的女朋友很好，如同他设想的一样。可是太顺从他，太乖巧，什么只依着他的意见，爱他，他却觉得少了点什么……他想是不是自己太不知足了呢？可是，好想她！想她的精灵古怪，想她的俏皮开朗，甚至想她那倔强的小脾气，天啊……他竟然揽着自己的女朋友又想起了她！

　　在中山路上逛街，意外的遇到了她！本来嘛，五.一谁不出来玩，可是就那么巧！她也有了新的男朋友，个子高高的，挎着胳膊，两个人急匆匆的走着。这个人大概有1.8多，模样也过的去，她这回该很喜欢吧~他心里酸酸的想。她一脸疲倦，是累了么？远远的看到她无精打采的样子，他就禁不住担心，那个和她挎着的家伙没看见么？真过分！如果是以前和他在一起，他看到她累总会替她拿东西，给她买可乐，讲笑话给她听，唯一的凳子也会给她坐让她休息……近了，近了，越走越近了，他禁不住有点紧张，又装做若无其事的样子，浏览着一路的专卖店。她茫然的看着前面，忽然眼睛一亮，她也看到了他，微微振动一下，立刻装做没有看见的样子。两对人在来来往往的人群中慢慢的，进行着陌生人一般的擦肩而过的仪式，他和她避开视线，又禁不住去看对方，再微微的颔首。他们的伴侣被路旁的店铺所吸引着，没注意到这一切，毕竟东西比人好看多了。

　　走过去了，就要走过去了……感受着两个人擦肩而过的，微妙的气息…..他甚至觉得自己闻到了她身上熟悉的香水，淡淡的柠檬的清香，她瘦了，头发是快留到腰了么？他没来得及看清楚，她曾说过那是为他留的呢…..他的头发还是短短的，好象微微的染了点酒红，以前他穿的很素，现在居然穿葡萄红的T血，靛蓝的牛仔裤….不过也很帅，身边的是他的女朋友么？很秀气呢….象个洋娃娃，他一定很爱她吧……她心里酸酸的，总之，这个女孩一定比她乖巧听话，过去为什么她总是和他拌嘴呢？为什么她就不肯让着他一点点呢？….

　　他禁不住回头去张望她的身影，身边的女朋友抬起头问他："怎么了？""看看我们错过了什么店……."其实是错过了她吧，自己默默的说，他心里忽然湿湿的，他看到她低头匆匆的走着，似乎没有任何的情绪的波动….微微叹气，她仍是个倔强的女孩啊…..忽然间，她回头了，她回过头迅速的向他这边张望了一下，他向她微笑着。她在那一瞬间笑了，笑的阳光灿烂的，象以前一样俏皮的露出了两个小虎牙，她冲他无声的做了一个手势，那是竖起拇指食指和小指的动作，他在那一瞬间里泪湿盈眶…..年轻时候的爱恋与悔恨，彷徨与无奈，梦想与踌躇，那些长长的思念的夜，无数个酒醉的日子，睡梦中呼喊的名字……

　　他不知道自己呆站了多久，女朋友奇怪的晃着他的肩膀，问他怎么了怎么了，他恍惚的回过头，女朋友扳下他的手，忽然笑了，说你看你手攥的这么紧干吗，他茫然的低头看，自己刚才高举的右手，紧紧攥着的，居然，居然是个"六"………

　　都以为爱情是自己能设计出来的，只要画出模型，建造出来，就是最甜美的爱。可是，爱却总是千变万化，难以琢磨。不称心如意的那个，却也爱了，也不知不觉的爱过万水千山，缘 来 是 你 ……………
