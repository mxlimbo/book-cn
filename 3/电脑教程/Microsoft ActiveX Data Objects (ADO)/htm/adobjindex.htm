Index 对象

            

表示数据库表的索引。

说明

如下代码创建新的 Index：

Dim obj As New Index

使用 Index 对象的属性和集合，可以：

  • 使用 Name 属性标识索引。

  • 使用 Columns 集合访问索引的数据库列。

  • 使用 Unique 属性指定索引关键字是否必须唯一。

  • 使用 PrimaryKey 属性指定索引是否为表的主关键字。

  • 使用 IndexNulls 属性指定索引字段中有 Null 值的记录是否有索引项目。

  • 使用 Clustered 属性指定索引是否分簇。

  • 使用 ParentCatalog 属性指定拥有索引的 Catalog。

  • 使用 Properties 集合访问特定提供者的索引属性。

注意   如果在已追加到 Tables 集合的 Table 对象中不存在 Column，则会在将 Column 追加到 Index 的 Columns 集合时出现错误。
