Users 集合

         

包含目录或组的所有存储 User 对象。

说明

Catalog 的 Users 集合代表所有目录的用户。Group 的 Users 集合仅代表具有特定组成员身份的用户。

Users 集合的 Append 方法对于 ADOX 是唯一的。可以：

  • 使用 Append 方法将新用户添加到集合中。

其余的属性和方法对于 ADO 集合是标准的。可以：

  • 使用 Count 属性返回包含在集合中的用户的数目。

  • 使用 Item 方法访问集合中的用户。

  • 使用 Delete 方法从集合中删除用户。

  • 使用 Refresh 方法更新集合中的对象，以反映当前数据库的模式。

注意   在将 User 对象追加到 Group 对象的 Users 集合之前，与被追加对象具有相同 Name 的 User 对象必须已存在于 Catalog 的 Users 集合中。
