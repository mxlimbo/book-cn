ADO 动态属性

ADO 动态属性总结

属性 说明
Name 指定 Recordset 对象的名称。
Unique Table、Unique Schema、Unique Catalog Unique Table 指定一个允许进行更新、插入和删除的基本表的名称。

Unique Schema 指定模式，即表的所有者的名称。

Unique Catalog 指定目录，即包含表的数据库的名称。
Resync Command 指定用户提供的命令字符串，Resync 方法发出该字符串用于刷新在由 Unique Table 动态属性所命名的表中的数据。
Update Resync 指定在 UpdateBatch 方法之后是否进行隐式 Resync 方法操作，以及如果这样，该操作的范围。
