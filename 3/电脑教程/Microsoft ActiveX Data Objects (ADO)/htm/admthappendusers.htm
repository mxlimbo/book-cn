Append 方法（Users）

      

将新的 User 对象添加到 Users 集合。

语法

Users.Append User[, Password]

参数

User   变体型值。包含要追加的 User 对象，或要创建和追加的用户的名称。

Password   可选，字符串值。包含用户的密码。Password 参数对应于 User 对象的ChangePassword 方法所指定的值。

说明

Catalog 的 Users 集合代表所有目录的用户。Group 的 Users 集合仅代表在特定组中具有成员身份的用户。

如果提供者不支持创建用户，将出现错误。

说明   在将 User 对象追加到 Group 对象的 Users 集合之前，与被追加的对象具有相同 Name 的 User 对象必须已存在于 Catalog 的 Users 集合中。
