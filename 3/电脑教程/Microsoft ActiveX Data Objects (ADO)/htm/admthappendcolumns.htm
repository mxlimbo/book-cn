Append 方法（Columns）

      

将新的 Column 对象添加到 Columns 集合。

语法

Columns.Append Column [, Type] [, DefinedSize]

参数

Column   要追加的 Column 对象，或要创建和追加的列的名称。

Type   可选，长整型值。指定列的数据类型。Type 参数与 Column 对象的 Type 属性相对应。

DefinedSize   可选，长整型值。指定列的大小。DefinedSize 参数与 Column 对象的 DefinedSize 属性相对应。

注意   如果在已追加到 Tables 集合的 Table 中还没有 Column，则会在将 Column 追加到 Index 的 Columns 集合时出现错误。
