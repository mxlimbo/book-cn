When you upgrade NT 3.51 to NT 4.0, what should you do?

A) Install from NT 4.0 CD-ROM and install NT 4.0 on the same directory of NT 3.51

B) Run winnt32.exe under NT 3.51, and install NT 4.0 on the same directory of NT 3.51

C) Run winnt32.exe under NT 3.51, and install NT 4.0 on the different directory of NT 3.51

D) Install from NT 4.0 CD-ROM and install NT 4.0 the different directory of NT 3.51

Answer: A, B


What is the minimum amount of hard disk space required to install Windows NT Server on an x86-based computer?

A) 125MB

B) 110MB

C) 130MB

D) 135MB

Answer: A


You want to upgrade your Windows NT 3.51 Server to 4.0, preserving the existing configuration settings. What are two ways to perform this upgrade? (Choose two)

A) Boot from NT 4.0 setup boot disks. Install the Windows NT 4.0 files in a different directory from the one that contains your NT 3.51 files.

B) Boot from the NT 4.0 setup boot disks. Install the windows NT 4.0 files in the directory that contains your NT 3.51 files.

C) Start Windows NT 3.51 server. Run Winnt.exe from the command prompt. Install the NT 4.0 files in the directory that contains your NT 3.51 files.

D) Start Windows NT 3.51 Server. Run winnt32.exe from the command prompt. Install the Windows NT 4.0 files in the directory that contains your NT 3.51 files.

Answer: B, D


You want to migrate the files and user accounts from a NetWare server to a Windows NT Server computer. What do you need to install on the Windows NT Server before you can perform the migration?

A) RIP for NWLINK

B) Gateway Service for Netware

C) SAP agent

D) FTP

Answer: B


Which of the following file system support DOS and Win16 applications, as well as has the read/write operation?

A) FAT

B) NTFS

C) HPFS

D) CDFS

Answer: A, B


The operating system files on a Windows NT computer are stored on the:

A) SYS: volume

B) Boot partition

C) System partition

D) C: drive always.

Answer: B


Your network has 10 machines, all of them are installed NT Server as a stand alone server to build a workgroup. Users complain that It's too bad that they must reconfiguration the desktop and reconnect the network when they logon from different machine. How to solve the problem?

Answer: Change one machine to the primary domain controller. And add all the machines into the domain. Create a server based profile for every domain user.


Someone wants to do a tape backup only with the registry, what should they do?

A) Select the differential option

B) Select at least one volume that contain the registry file

C) Stop the server service

D) From a remote computer to do the backup

Answer: B


Your network has 40 Windows NT Workstation computers. The computers all belong to a workgroup named SUPPORT. You want to install a server computer that is configured as a PDC. The domain name will be CITYCORP. You want all of the workstation computers to participate in the new CITYCORP domain. You also want them to be able to use global groups and user accounts from the CITYCORP domain in the local groups on the workstations. What must be done so that the workstations can become members of the CITYCORP domain? (Choose two)

A) A Windows NT Workstation administrator must reinstall each workstation and add it to the domain.

B) A Windows NT Workstation admin must log on at each workstation and use the Network option in Control Panel to configure the workstation to join the domain.

C) A Domain Admin must use Server manager to add a computer account for each NT Workstation computer.

D) A Domain ADMIN must use User Manager for Domains to create accounts for the Windows NT Workstation computers.

Answer: B, C


Your network has one Windows NT Server computer, 11 Windows 95 computers, and six Windows NT Workstation computers. You want to use the Windows NT Backup program to do a nightly backup of all the files on the Windows NT Server computer, including the Registry. You want to do a weekly backup of all workstation data files. What is the minimum backup hardware required to perform these backups?

A) One tape drive attached to the NT Server only.

B) One tape drive attached to the NT Workstation only.

C) One tape drive attached to the NT workstation and one drive attached to the 95 computer.

D) One tape drive attached to the NT Server and one tape Drive attached to the 95 computer.

Answer: A


You wish to view the roles of each of the Windows NT computers on the domain, as well as the operating system and version they are running. From a domain controller, what should you use?

A) Server manager

B) NT Explorer

C) The Network Option in Control Panel

D) Network Client Administrator

Answer: A


You want to create a backup copy of the entire Registry of your Windows NT Server computer. You want to include the security settings and user accounts. How can you do this while the computer is running?

A) Use NT Explorer.

B) Use the NT backup program.

C) Use Disk Administrator

D) Use Server Manager.

Answer: B
