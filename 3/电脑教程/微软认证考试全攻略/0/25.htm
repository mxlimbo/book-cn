70-063　MCSE Passed Win95 exam

I passed the Win95 exam this morning. Thanks to this list for valuable information.

The best resource for this exam is the Windows 95 Resource Kit by Microsoft Press. Virtually everything on the exam is covered in this book, which is very well written. If you find the book too long, then at least read the beginning and end of each chapter.

The test was similar to the free exams at Microsoft's website. Download pepmcse.exe, asm1.exe, asm2.exe, asm3.exe from http://www.microsoft.com/mcp

I think I got several questions on the actual exam that were the same or similar to those in the free Microsoft exams, which I felt were far superior to Transcender.

If you are preparing for the exam be sure to look at Microsoft's exam objectives at http://www.microsoft.com/Train_Cert/mcp/exam/prep/PG70-063.htm

For braindumps, go to the following sites:

http://www.bnla.baynet.de/bnla01/members/robsch19/

http://walden.mo.net/~rpraib/mcse.htm

http://www.mindspring.com/~trevorcoverley/main.html

http://www.iwaynet.net/~ryans/mcse.html

Here are some things you should study:

Know how DOS applications, Win16 applications, and Win32 applications interact

Know about processes and threads

Know how user profiles and system policies work and where the relevant files are located on an NT network and on a Netware network

Know about EMF spooling; Point and Print in general and for Netware

Know the F4,F8, etc. keys and how they are used at startup

Know which clients and protocols are needed to setup 95 machines for NT networks and Netware networks

Know hardware/software requirements for installing Win95 and how to troubleshoot problems in installation

Know remote administration tools such as Netwatcher, Registry Editor, System Policy Editor as well as what are needed to enable those tools on remote computers

Know how the Briefcase works and how to use it

Know how to set up Dial-Up Connection to the Internet and how to troubleshoot Dial-Up Networking problems

Know a few keyboard shortcuts and how to customize the desktop

Know how to setup Exchange and MSMail postoffice

Know several ways to fax a document (I think there are six listed in the Resource Kit)

Know how to optimize performance with disk utilities

Know how to customize/optimize DOS applications, MS-DOS Mode, conventional memory, etc.

Know how to optimize performance for a File and Print Sharing 95 computer.

Know a few general keyboard shortcuts ( F1, Alt-Esc, Alt-Tab, Shift while dragging file, Ctrl while dragging file, etc)

Know about WINS, DNS, LMHOST, IP addresses and host names in general

Know when to right-click (hands-on really helps)

Know about Plug and Play ( I don't remember what was tested as I scored lowest in this area )

Know about ICM printing support

Know about the Registry and its components

Know about longfilenames and Netware

On questions with more than one answer, you were told exactly how many to choose. I didn't know the answers to some questions but was able to eliminate two choices out of four.

Make sure you get plenty of hands-on experience with all of the above, even if you only have a standalone.

That's about all I can remember. If you are looking for braindumps, the websites listed above have some useful braindumps. Be wary of incorrect answers in the dumps though.
