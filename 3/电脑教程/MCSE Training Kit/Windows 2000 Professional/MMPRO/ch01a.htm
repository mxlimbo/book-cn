Chapter 1

Introduction to Windows 2000

About This Chapter

This book was written to prepare you to install, configure, and support Microsoft Windows 2000 Professional; therefore, this chapter helps you to understand Windows 2000 Professional and where it fits in the Windows 2000 family of products. It presents an overview of the Microsoft Windows 2000 operating system and the four products that make up this family. The Windows 2000 family of products consists of Windows 2000 Professional, Windows 2000 Server, Windows 2000 Advanced Server, and Windows 2000 Datacenter Server.

Before You Begin

You need no special preparation to complete this chapter.
