Chapter 5

Using the Registry

About This Chapter

The Microsoft Windows 2000 operating system stores system configuration information in a hierarchical database called the registry. This chapter presents an overview of the registry and introduces Registry Editor, which is a tool that allows you to view and modify the registry.

Before You Begin

To complete this chapter, you must have

  • A computer that meets the minimum hardware requirements listed in "Hardware Requirements."

  • Windows 2000 Professional installed on the computer.
