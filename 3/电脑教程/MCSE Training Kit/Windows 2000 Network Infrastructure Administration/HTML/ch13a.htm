Chapter 13

Implementing Certificate Services

About This Chapter

Certificates are fundamental elements of the Microsoft Public Key Infrastructure (PKI). Certificates enable users to use smart card logon, send encrypted e-mail, and sign electronic documents. Certificates are issued, managed, renewed, and revoked by certificate authorities. In this chapter, you will learn how to install and configure certificates.

Before You Begin

To complete this chapter, you must have

  • Installed Microsoft Windows 2000 Server

  • Installed Active Directory

  • Installed Domain Name System (DNS)
