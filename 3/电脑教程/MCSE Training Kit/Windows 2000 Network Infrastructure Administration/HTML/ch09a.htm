Chapter 9

Implementing Windows Internet Name Service (WINS)

About This Chapter

Although Microsoft Windows Internet Name Service (WINS) servers are not needed in a network consisting entirely of Microsoft Windows 2000-based computers, they are crucial in most Transmission Control Protocol/Internet Protocol (TCP/IP) networks containing computers based on the older architectures of Windows NT 4.0, Windows 98, or Windows 95. In this chapter, you will learn how to implement WINS on your network.

Before You Begin

To complete this chapter, you must have

  • Installed Windows 2000 Server with TCP/IP
