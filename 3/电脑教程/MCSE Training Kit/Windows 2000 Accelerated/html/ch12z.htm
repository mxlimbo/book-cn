Chapter 12

Managing Remote Access

About This Chapter

Microsoft Windows 2000 remote access has been improved. This chapter presents an overview of the new protocols for use with remote access. It explains the new options and interfaces in Windows 2000 that are used to connect computers and configure protocols correctly to meet all your remote access requirements. The chapter starts out with a review of Internet Protocol (IP) routing and explains how information is actually transmitted across the Internet. It concludes with a look at the various security methods you can use to keep your network secure.

Before You Begin

To complete this chapter, you must have

  • Two Microsoft Windows 2000 servers with local area network (LAN) connectivity
  • Installed Windows 2000 Server on one of the two computers (Server01 would be preferred)
