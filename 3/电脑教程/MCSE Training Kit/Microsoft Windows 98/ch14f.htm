Lesson 2: Logging On to a NetWare Server

Before you can log on to a NetWare server, you have to install and configure the necessary software. This lesson provides information to help you install and configure the IPX/SPX protocol, install and configure Client for NetWare networks, install directory network services, and configure logon.

After this lesson, you will be able to:

  • Install and configure the IPX/SPX-compatible protocol.

  • Install and configure Client for NetWare Networks.

  • Install directory network services.

  • Configure logon functions.

Estimated lesson time: 60 minutes
