Lesson Summary

ScanDisk is a disk analysis and repair program that has been upgraded to recognize and fix issues with FAT32 drives and compressed drives. ScanDisk runs automatically after an improper shutdown unless it is disabled through Msdos.sys.

Windows 98 Disk Defragmenter stores an application's executable file and its supporting files in a contiguous area of disk space, thus reducing access time for files on the disk.

Task Monitor monitors programs you use and records their disk access patterns when they start. It also records the number of times you use programs. Disk Defragmenter uses this information to favor more frequently used programs in optimizing the disk.

Disk Cleanup improves system performance by removing temporary and deleted files from a hard disk. Disk Cleanup automates the maintenance task of deleting unneeded files.

The Maintenance Wizard speeds up your applications, deletes unnecessary files, and scans hard disks for errors. Schedule Tasks allows you to run any program at a desired date and time.
