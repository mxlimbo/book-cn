Setup Script Structure

A setup script contains sections that are identified by headings. A heading is enclosed in brackets and is followed by one or more items. A setup script will contain at least two sections, headed by:

  • [Setup]

  • [OptionalComponents]

Note

If a section heading has no entries listed, Setup may generate an error and fail to continue.

The [Setup] Section

The [Setup] section sets parameters for control of the Setup process. Items with a value of 1 are enabled and items with a value of 0 are disabled.

The following sample [Setup] section lists settings that are commonly used for an unattended install: 

[Setup]
EBD=0; Setup will not create the emergency startup disk
VRC=0; Windows 98 files copied regardless of existing version
ShowEula=0; The license agreement will not appear
UNINSTAL=0; Do not allow user to specify Uninstall options and do not create backup files for uninstalling
