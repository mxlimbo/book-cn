Creating Hardware Profiles

A hardware profile is a record of a specific configuration of hardware in the computer. Mobile users need flexible hardware configurations. Users plug their portable computers into docking stations in the office and use them in undocked states away from the office. The docking stations may include additional hardware, such as CD-ROM devices or network adapters. The undocked computers might use external modems, rather than network adapters.

To address this issue and other situations when you need to establish several hardware configurations for a single computer, Windows 98 uses hardware profiles to determine which drivers to load when the system hardware changes.

Windows 98 automatically creates one profile for a portable computer in a docking station and a second profile for its undocked state. Windows 98 detects the status of the computer—docked or undocked—during startup and configures to the corresponding profile.

Creating Hardware Profiles Manually

You can create and modify hardware profiles through the System icon in Control Panel. The Hardware Profiles tab lists all profiles that have been created, including the profile for Original Configuration which updates automatically to reflect all hardware additions, removals, and changes.

If you want to create a new profile:

  1. Select Original Configuration, click the Copy button, and then give it a new name.

    After you create a new profile, you change it to establish the desired configuration for this profile. You must restart the computer with the new hardware profile to have the changes implemented. When you restart the computer, Windows 98 prompts you to select a hardware profile to load.

If you want to configure a new profile:

  1. Restart the computer with the new profile, and then double-click the System icon in Control Panel.

  2. In the System Properties window, click the Device Manager tab to view all installed hardware devices.

  3. Select the device you want to disable or remove from this profile and click Properties. The General tab of the device properties window offers two profile options:

    • Disable in this hardware profile prevents the drivers for that device from being loaded during startup.

    • Remove from this hardware profile removes the device and its drivers from the configuration in the profile and can only be reversed by reinstalling the drivers.

  4. Select the desired profile option and click OK.

  5. Repeat steps 3 and 4 for each device that you want to change in this profile.

Note

If you disable Plug and Play devices, the resources (such as IRQ and DMA) for those devices are available for other devices automatically. If you disable a legacy device, however, you must also remove the device from the hardware list in Device Manager and physically remove the hardware from your computer in order to free those resources for other devices.

You should use hardware profiles only to indicate different hardware configurations. Identical profiles slow the startup process because Windows 98 startup cannot determine which of two identical profiles to use, and will force you to select a profile to load.
