Creating and Deleting Partitions

You create and delete partitions using the command-line utility Fdisk.exe (see Figure 3.1) located in the C:\Windows\Command folder. Fdisk is also on the emergency startup disk you create with Windows 98. You can use Fdisk to:

  • Enable large disk (FAT32) support.

  • Create and delete primary and extended partitions.

  • Create and delete logical drives within an extended partition.

  • Designate an active partition.

Figure 3.1 Fdisk options

Caution

Actions performed using Fdisk are not reversible. If you repartition a hard disk with Fdisk, you delete all the files on the original partitions. Be sure to back up all data files on a partition before using Fdisk.

If you want to create a partition on your hard disk using Fdisk, perform the following procedure:

  1. Click Start, then Run. Type in Fdisk and press ENTER.

    – or –

    Start your computer with your emergency startup disk. At the command prompt, type Fdisk.

    If your hard drive is 512 MB or larger, a screen appears, prompting you to enable large disk support. If you answer Y, any new partitions you create will be FAT32 partitions. If you answer N, any new partitions you create will be FAT16 partitions.

  2. In the Fdisk Options screen, press 1, and then press ENTER. The Create DOS Partition or Logical DOS Drive screen appears.

    Partitions must be created in this order:

    • Primary partitions.

    • Extended partitions.

    • Logical DOS drive in an extended partition.

  3. To specify the partition type and size you want, follow the instructions on-screen, and then press ENTER.

    You specify the partition size as a percentage of disk space or in megabytes of disk space. If you specify a percentage of disk space, include a percent sign (%) after the number.

You must delete partitions in this order:

  1. Any non-DOS partitions.

  2. Any logical drives in extended partitions.

  3. Any extended partitions.

  4. Any primary partitions.

Caution

If you delete a disk partition by using Fdisk, all the data in that partition is destroyed.

If you want to delete a partition, perform the following procedure:

  1. In the Fdisk Options screen, press 3, and then press ENTER. The Delete DOS Partition Or Logical DOS Drive screen appears.

  2. Press the number for the type of partition you want to delete, and then press ENTER.

  3. Follow the directions on the screen, and repeat the steps for deleting any additional logical drives or partitions.

If Fdisk cannot delete a non-MS-DOS partition, quit Fdisk, delete the non-MS-DOS partition by using the software used to create it, and then restart Fdisk.

Fdisk cannot delete NTFS logical drives created by Windows NT in an extended partition. Use a Windows NT-based utility to delete these drives.
