Files Needed to Boot Windows 98

Although there are hundreds of files that Windows 98 needs in order to run correctly, a few of them are of particular importance because they start Windows 98. These files include:

Name of File Function
c:\io.sys Boots into real mode Windows 98. It also loads real mode drivers and TSRs specified in Config.sys and Autoexec.bat. This file is marked hidden, system, and read-only.
c:\msdos.sys Controls the boot process and compatibility with applications that require this file to be present before they can be installed. This file is marked hidden, system, and read-only.
c:\config.sys Sets and specifies system environment variables, loads real-mode drivers and TSRs in real mode. This file is optional.
c:\autoexec.bat Sets and specifies commands to be executed in real mode before protected-mode Windows 98 initializes. This file is optional.
C:\Windows\win.ini, C:\Windows\system.ini Holds some configuration information for Windows 98. Most options that were present for Windows 3.x are now stored in the Registry.
C:\Windows\win.com Initiates the Windows 98 load phases.
