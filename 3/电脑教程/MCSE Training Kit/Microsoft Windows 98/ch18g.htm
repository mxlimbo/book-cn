Practice: Installing Dial-Up Networking

In this practice, you will install Dial-Up Networking using the Add/Remove Programs Wizard.

    To install Dial-Up Networking

    1. In Control Panel, double-click the Add/Remove Programs icon.

    2. Select the Windows Setup tab.

    3. When the list appears, select Communications, and then click Details.

    4. Click Dial-Up Networking to select it, and then click OK.
