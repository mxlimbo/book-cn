Chapter 7

Configuring the Windows 98 Desktop

About This Chapter

This chapter discusses the interface enhancements provided by Microsoft Windows 98, the integration of Internet Explorer 4.0 in Windows 98, and the role of Internet Explorer 4.0 on the Windows 98 desktop. The chapter also presents information about the Active Desktop feature, which you can use to display active, or real-time, information such as a stock ticker or online newspaper.
