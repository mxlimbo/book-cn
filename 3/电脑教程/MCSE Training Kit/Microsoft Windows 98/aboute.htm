Features of This Book

Each chapter opens with a "Before You Begin" section, which prepares you for completing the chapter.

Whenever possible, lessons contain procedures that give you an opportunity to use the skills being presented or explore the part of the application being described.

The "Review" section at the end of the chapter allows you to test what you have learned in the lesson.

Appendix A, "Questions and Answers" contains all of the book's review questions and corresponding answers.
