Practice: Installing Windows 98

In this practice, you will upgrade your system from Microsoft Windows 95 by installing Microsoft Windows 98.

    To install Windows 98

    1. Insert the Windows 98 Setup CD-ROM.

      A dialog box appears indicating that your current version of Windows is older than the one on the CD-ROM and prompts you to upgrade.

    2. Click No.

      The Windows 98 CD-ROM window closes.

    3. Close any programs that are currently running.

    4. On the Start menu, click Run.

      The Run dialog box appears.

    5. Browse to the Setup.exe file located in the Win98 directory on your Windows 98 CD, and then click OK.

      The Windows 98 Setup dialog box appears.

    6. Click Continue.

      The Windows 98 Setup program informs you that it is preparing to run Windows 98 Setup and checks your system.

      The License Agreement appears.

    7. Read the License Agreement, select I accept the agreement, and then click Next.

      Setup initializes your registry database.

      Setup prepares your Windows directory, checks for installed components, and checks for sufficient free disk space.

      You are prompted to save your existing system files.

    8. Click No, and then click Next.

      Setup prompts you to select a country for Internet channels.

    9. Select a country and click Next.

      Setup prompts you to create an Emergency Startup Disk.

    10. Click Next.

      The Insert Disk dialog box appears and prompts you to insert a disk.

    11. Click Cancel.

      The Start Copying Files screen appears.

    12. Click Next to begin copying files.

      When the file copy is completed, your computer restarts.

      Setup finalizes hardware setup and system settings.

      Setup is now complete and your computer restarts, again.

    13. Log on with your assigned user name. Click OK.

      Windows 98 builds a driver information database and updates personalized settings.

    14. Close the Welcome to Windows 98 window.

      You are prompted to automatically open Welcome to Windows 98 each time you log on.

    15. Click No.

      Your installation is complete.
