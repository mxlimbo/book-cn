Lesson Summary

Windows 98 is a 32-bit desktop operating system that is designed to support a wide range of hardware and software, while Windows NT Workstation 4.0 is designed to be the most powerful 32-bit desktop operating system. For many sites, a mix of both systems will provide the best business solution. If a company does not have the hardware infrastructure required by Windows NT Workstation, and does not require a high degree of local security, then Windows 98 may be the best client operating system.

The following points summarize the technical differences between Windows 98 and Windows NT Workstation 4.0:

  • Windows 98 supports more legacy hardware, has broader application and device compatibility, and a simplified upgrade installation. Although Windows NT Workstation 4.0 supports most 16-bit and MS-DOS applications, Windows 98 will ensure greater compatibility.

  • Windows 98 offers broad mobile computer support and Plug and Play capability.

  • Windows NT Workstation 4.0 performs multitasking more effectively than Windows 98.

  • Windows NT Workstation 4.0 is more reliable and secure than Windows 98.
