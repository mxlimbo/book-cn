Using Scheduled Tasks

Scheduled Tasks is an application for scheduling programs to run at specific times. You can schedule important maintenance utilities such as ScanDisk (see Figure 9.4), Compression Agent, Disk Defragmenter, and Backup to run at regular intervals and when there is the least demand on computer resources. With Scheduled Tasks you can also run other nonmaintenance applications including:

  • DVD Player

  • Internet Explorer

  • Microsoft Chat

  • Paint

  • Windows Update

Figure 9.4 Using Scheduled Tasks to schedule ScanDisk

An icon in the Scheduled Tasks window represents each task you schedule. You can modify the schedule for any task by double-clicking the icon for that task. A properties window opens with the following three tabs:

  • Task provides the source location of the task.

  • Schedule provides options to configure the frequency, date, and time that the task should be run.

  • Settings provides options for configuring the conditions for a task.

You can remove tasks from the schedule by selecting the task icon and pressing Delete.

Note

If you are upgrading a Windows 95 installation in which System Agent is installed, the Scheduled Tasks utility inherits the System Agent tasks.
