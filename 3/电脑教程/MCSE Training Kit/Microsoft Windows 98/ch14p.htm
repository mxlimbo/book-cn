Optional Practice: Mapping a Drive to a NetWare Server

In this practice, you will locate a NetWare server and map a drive to that server.

    To map a drive to a NetWare Server

    1. Double-click Network Neighborhood.

    2. Double-click NetWareServer.

    3. Double click the Sys folder.

    4. Right-click the Public folder, and then click Map Network Drive.

    5. Check the Connect as root of the drive box, and then click OK.

Long File Name Issues

To save files and folders with long file names on a NetWare server, you must install long file name support on that server. You can do this by loading OS/2 namespace (Os2.nam) on the NetWare server.

Note

Long file name support is automatically disabled when interoperating with NetWare 3.11 servers, even if OS/2 namespace is loaded, due to problems caused by the NetWare 3.11 OS/2 namespace. To enable long file name support for these servers, obtain os2nsfix from Novell and run it on the server, and then add SupportLFN=1 to the [nwredir] section of System.ini.
