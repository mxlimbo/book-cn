Recovering the Active Desktop

If the Active Desktop stops functioning correctly, Windows 98 unloads it and displays the Active Desktop Recovery screen (see Figure 7.7). This screen offers troubleshooting tips to help you restart the Active Desktop.

Figure 7.7 Active Desktop Recovery screen

If a failure occurs after adding a new item to the Active Desktop, follow these steps:

  1. Click Start, point to Settings, Active Desktop, and then click Customize my Desktop.

  2. Find the item you recently added, and clear its check box.

If Internet Explorer stops working, or if you must turn the computer off without selecting Shutdown on the Start menu, the recovery screen provides a Restore my Active Desktop link. It is best to try this before disabling any items you recently added. In most cases, selecting the Restore my Active Desktop link will correct the problem.
