Lesson 3: Loading and Updating User Profiles

User profiles load automatically when the user logs on. The type of profile depends on whether or not the user's computer is connected to a network.

This lesson discusses the location of local and roaming profiles when a user logs on, and the destination for saved profiles when a user logs off. This lesson also describes the procedure for changing the default profile.

After this lesson, you will be able to:

  • Describe the route of local and roaming profiles when a user logs on and off.

  • Change the default profile.

Estimated lesson time: 10 minutes
