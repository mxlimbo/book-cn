Practice: Creating a Custom Toolbar

In this practice, you will add a System Tools toolbar to the taskbar.

    To create a new toolbar

    1. Right-click an empty area on the taskbar.

      A pop-up menu appears.

    2. Point to Toolbars, and then click New Toolbar.

      The New Toolbar dialog box appears.

    3. Open the C:\Windows\Start Menu\Programs\Accessories folder.

    4. Select the System Tools folder, and then click OK.

      A new toolbar called System Tools appears. The toolbar has buttons for each shortcut in the C:\Windows\Start Menu\Programs\Accessories\System Tools folder.

Enhancements to the Minimize Function

In Windows 95, you click a program button on the taskbar to restore a minimized program. In Windows 98, you can click the program button to minimize the program as well as to restore a minimized program.

The Quick Launch toolbar also contains a Show Desktop icon that you can click to minimize all active programs, and to hide any open dialog boxes.
