Lesson 3: Resolving Hardware Conflicts

Windows 98 includes several tools to assist your troubleshooting efforts when hardware problems and driver conflicts occur. This lesson provides an overview of Troubleshooters. It also presents information about Device Manager, which provides a single source for viewing your hardware devices and drivers. In addition, this lesson discusses Automatic Skip Driver Agent, which provides a method of resolving device driver conflicts.

After this lesson, you will be able to:

  • Resolve hardware conflicts with Troubleshooters.

  • Use Device Manager to resolve device driver conflicts.

  • Use Automatic Skip Driver Agent to resolve device driver conflicts.

Estimated lesson time: 30 minutes
