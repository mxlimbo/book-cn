Configuring Hard Disk File Systems

The disk cache in Windows 98 is dynamic and requires no additional configuration. However, you can use the File System Properties sheet (see Figure 6.5) to guide Windows 98 in optimizing the disk cache for your specific type of computer.

Figure 6.5 The File System Properties sheet

If you want to configure the hard disk cache:

  1. From the System icon in Control Panel, click the Performance tab, and then click File System.

  2. Select a configuration based on the typical role of the computer:

    • Desktop computer. A normal computer acting primarily as a network client or a stand-alone computer, that has more than the minimum required RAM and is running on power, rather than battery.

    • Mobile or docking system. Any computer with limited memory. Select this configuration for a computer that has limited RAM and is running on battery, so the disk cache will be cleared frequently.

    • Network server. A computer used primarily as a peer server for file or printer sharing, with adequate RAM and frequent disk activity. This selection optimizes the computer settings for a high amount of disk access.
The Read-ahead optimization setting, also configurable on this screen, sets the amount of RAM used for read-ahead optimization and controls how much extra data is read during normal file requests.
