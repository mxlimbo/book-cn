Lesson 3: Sharing Resources on an Intranet

The Windows 98 CD-ROM includes Personal Web Server, a tool that you can use on an intranet to create, maintain, and test Web sites and quickly share documents across different operating systems.

This lesson discusses the capabilities of Personal Web Server, as well as the procedures for installing and configuring this application.

After this lesson, you will be able to:

  • Install and configure Personal Web Server.

Estimated lesson time: 30 minutes
