Lesson 3: Selecting Windows 98 as a Client Operating System

Windows 98 and Windows NT Workstation share many features that make them both effective client operating systems. However, there are important differences that should be considered when deciding on which system to install.

This lesson compares the features of Windows 98 to those of Windows NT Workstation.

After this lesson, you will be able to:

  • Compare Windows 98 to Windows NT Workstation 4.0 to determine the appropriate client operating system for a given work environment.

Estimated lesson time: 10 minutes

Windows 98 and Windows NT Workstation differ from one another in their architecture. Figure 1.1 summarizes system requirements, hardware and software compatibility, and rates the stability and security for both operating systems.

Figure 1.1 Windows 98 compared to Windows NT Workstation 4.0

The following table gives a more detailed comparison between Windows 98 and Windows NT Workstation 4.0.

Component Windows 98 Windows NT Workstation 4.0
Minimum hardware standards 486DX-66 MHz 486/25 processor or better
Memory Minimum: 16 MB Recommended: 24 MB Minimum: 12 MB Recommended: 32 MB
Software compatibility Supports most Windows and all MS-DOS applications. Supports most Windows and MS-DOS business applications. No support for applications that breach Windows NT security (any application that directly accesses hardware).
Hardware compatibility Supports current devices and most legacy devices and drivers. Supports most current devices and many legacy devices. Does not support legacy drivers (a legacy driver is a real-mode MS-DOS driver).
Power management and Plug and Play Built-in APM and ACPI support and Plug and Play for both mobile and desktop systems. Not standard in Windows NT 4.x and earlier: APM and Plug and Play support are available from some OEMs.
Performance Preemptive multitasking for 32-bit applications. Slightly better performance on 16 MB systems. Preemptive multitasking for all applications. Significantly better performance on 32 MB systems.
Reliability/stability Improved over Windows 3.11, Windows for Workgroups, and Windows 95, but not as high as Windows NT. Offers the highest level of stability. The operating system and all applications run in protected memory space.
Security Support for server-based validation of logon process and folder-level permissions for users connecting over a network. Complete protection, down to the file level for both local and remote users.
