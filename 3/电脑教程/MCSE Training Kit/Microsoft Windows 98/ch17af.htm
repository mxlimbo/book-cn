Using NetMeeting

Windows 98 includes Microsoft NetMeeting (see Figure 17.8), a powerful conferencing tool that enables you to communicate through intranets or the Internet in real time. NetMeeting incorporates video, audio, and data conferencing functions based on industry standards so that you can communicate with people who use compatible products.

Figure 17.8 Using Microsoft NetMeeting

NetMeeting offers these conferencing options:

  • Audio conferencing.

    NetMeeting serves as an Internet telephone with high-quality audio. This option requires a sound card, a microphone, and speakers.

  • Video conferencing.

    You can hold face-to-face conversations with high-quality video. This option requires a video capture card and camera added to your computer.

  • Multipoint conferencing.

    You can share information from one or more applications on your computer, exchange graphics, draw diagrams on an electronic whiteboard, send messages or take notes, and send files to other meeting members using the binary file transfer capability. This option requires the equipment listed for both audio and video conferencing.
