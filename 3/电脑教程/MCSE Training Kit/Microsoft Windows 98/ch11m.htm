Lesson Summary

Setup scripts automate Windows 98 installation, reducing installation time and costs. You can use setup scripts with CD-ROM installations, push installations, and pull installations.

By running Microsoft Batch 98 and selecting the options you want, you create a setup script that you can use to run Windows 98 Setup with minimal user input. After you create the setup script, you can run Windows 98 Setup by specifying the name of the script as a command-line parameter.
