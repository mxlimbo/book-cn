Optional Practice: Configuring Sharing for Microsoft Networks

In this practice, you will install file and printer sharing for Microsoft Networks and share a folder on a computer. Then, you will connect to a shared folder on a second computer.

To complete this practice, you will need to have a second computer.

Exercise 1
Installing and Using File and Printer Sharing for Microsoft Networks

In this exercise, you will install file and printer sharing for Microsoft Networks. You will then share a folder on your computer.

    To enable sharing

    1. Right-click Network Neighborhood, and click Properties.

      The Network dialog box appears.

    2. On the Configuration tab on the Network property sheet, click the File and Print Sharing button.

      The File and Print Sharing dialog box appears.

    3. Select I want to be able to give others access to my files and I want to be able to allow others to print to my printer(s).

    4. Click OK to return to the Network dialog box.

    5. Confirm that File and printer sharing for Microsoft Networks appears in the network components list.

    6. Click the Identification tab.

      The Identification property sheet appears.

      Record your computer name in the following space:

      Answer

    7. In the Computer Description text box, type the possesive form of your first name followed by Computer to indicate that it is your computer. For example, if your name is Linda, you would type Linda's Computer.

    8. Click OK to close the Network dialog box.

      A dialog box appears prompting you to restart your computer.

    9. Click Yes.

      Your computer restarts.

    To create and share a directory

    1. Log on to the network.

    2. Click Start, point to Programs, and then click Windows Explorer.

      A Windows Explorer window appears.

    3. Click the icon for drive C: in the left pane of the Windows Explorer window.

      The right pane of the Windows Explorer window displays the contents of your drive C:.

    4. Click the File menu, point to New, and then click Folder to create a new folder.

    5. Type Public Folder for the folder name and then press Enter.

    6. Right-click the Public Folder folder, and click Sharing.

      The Public Folder Properties dialog box appears.

    7. Click Shared As.

    8. In the Comment text box, type This folder is on computer_name where Computer Name is the name of your computer.

    9. Under Access Type, click Full, and then click OK.

      A hand appears underneath the Public Folder folder in theWindows Explorer window indicating that it has been successfully shared.

Exercise 2
Connecting to a Shared Folder

In this exercise, you will work with a second computer to map a drive to a folder shared on another computer.

    To map a drive to a shared folder on another computer

    1. Create a Public Folder share on your second computer.

    2. On the Tools menu of Windows Explorer, click Map Network Drive.

      The Map Network Drive dialog box appears.

    3. In the Path text box, type the UNC path to the second computer's shared folder.

      For example, if the second computer's name is COMP1, the UNC path to the shared folder would be: 

      \\COMP1\PUBLIC FOLDER

    4. Click OK.

      In the left pane of the Windows Explorer window, scroll down until the mapped drive appears.

      What name has Windows 98 assigned to this mapped drive?

      Answer

    To create a file on a mapped drive using Notepad

    1. Click Start, point to Programs, point to Accessories, and then click Notepad.

      Notepad starts and opens a new file.

    2. Type This file is being created over the network on my second computer.

    3. On the File menu, click Save As.

      The Save As dialog box appears.

    4. In the Save in box, click My Computer to display your local drives and network drives.

    5. Double-click the drive mapped to your second computer's shared folder.

    6. In the File Name text box, type Created by First Name (where First Name is your first name).

    7. Click Save.

      The file is saved in your second computer's Public Folder folder.

    8. Exit Notepad.

    9. In the Windows Explorer window, display the contents of the drive you mapped to your second computer's shared folder.

      What is the name of the file on this drive?

      Answer

    10. In the Windows Explorer window, display the contents of Public Folder.

      What is the name of the file in this folder?

      Answer

    To change access permissions

    1. In the Windows Explorer window, right-click C:\Public Folder, and click Sharing.

      The Public Folder Properties dialog box appears.

    2. Under Access Type, click Read-Only, and then click OK.

    3. In the Windows Explorer window, click the drive that you mapped to your second computer's shared folder.

    4. Double-click the document you created.

      Notepad starts with your document loaded.

    5. Make a change to the document.

    6. On the File menu, click Save.

      Notice that the Save As dialog box appears.

    7. Click Save.

      An error message appears. Why can't you save the file?

      Answer

    8. Click OK to close the error message.

    9. Click Cancel to close the Save As dialog box.

    10. Close Notepad.

    11. When prompted to save your changes, click No.

    To disconnect a mapped drive

    1. From the Tools menu of Windows Explorer, click Disconnect Network Drive.

      The Disconnect Network Drive dialog box appears.

    2. Click the drive that is mapped to your second computer's shared folder, and click OK to disconnect from the shared folder.

      A dialog box appears warning that you may lose data if you continue.

    3. Click Yes.

    To withdraw a shared folder from the network

    1. In the Windows Explorer window, right-click Public Folder, and then click Sharing.

      The Public Folder Properties dialog box appears.

    2. Click the Not Shared button, and then click OK.

    3. Click Yes.

      The hand underneath the Public Folder folder, indicating that the folder was shared, is gone. The Public Folder folder is no longer shared.
