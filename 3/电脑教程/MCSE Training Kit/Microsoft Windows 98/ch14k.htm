Optional Practice: Logging On to a NetWare 4.x Network

In this practice you will configure a Windows 98 computer to log on to a NetWare 4.x server and install Microsoft Service for NetWare Directory Services.

Answers to the questions in this practice can be found in Appendix A, "Questions and Answers", located at the back of this book.

You will need the following information to complete this practice:

  • Name of NetWare server

  • Preferred directory tree

  • Context

  • NetWare user name

  • NetWare password

  • Domain user name

  • Domain password

Exercise 1
Logging on to a NetWare 4.x Server

In this exercise, you will configure a Windows 98 computer to log on to a NetWare 4.x server.

    To install the Client for NetWare Networks

    1. Open Control Panel, and double-click Network.

      The Network dialog box appears.

    2. On the Configuration tab, click the Add button.

      The Select Network Component Type dialog box appears.

    3. Click Client, and then click Add.

      The Select Network Client dialog box appears.

    4. From the list of Manufacturers, click Microsoft.

    5. From the list of Network Clients, click Client for NetWare Networks, and then click OK.

      The Client for NetWare Networks appears in the list of installed components.

Note

Notice that the IPX/SPX Compatible Protocol also appears. If you install the Client for NetWare Networks on a computer that does not have IPX/SPX installed, Windows 98 will automatically add it.

    1. From the list of network components, select Client for NetWare Networks, and then click Properties.

      The Client for NetWare Networks Properties dialog box appears.

    2. In the Preferred Server list box, type NetWare_Server_name.

    3. In the First network drive list box, select N, and then click OK.

    4. In the Primary Network Logon list box, select Client for NetWare Networks, and then click OK.

      Windows 98 copies the necessary files from the installation media.

      You are prompted to restart your computer.

    5. Click Yes.

    To log on to a NetWare 4.x Server

    1. In the Enter Network Password dialog box, type NetWare_user_name and password, and verify that the Login Server is correct.

    2. Click OK.

      The Set Windows Password dialog box appears.

    3. Re-enter your password and click OK.

      The Enter Network Password dialog box appears.

      Why are you prompted to log on again?

      Answer

    4. Enter your Domain_user_name and Domain_password, and click OK.

      Your domain logon script executes, followed by your NetWare logon script.

    5. Click Start, point to Programs, and then click MS-DOS prompt.

      An MS-DOS prompt appears.

    6. At the MS-DOS C:\> prompt, type N: and then press ENTER.

    7. Type CD \PUBLIC and then press ENTER to change to the \Public folder.

      The N:\Public> prompt appears.

    8. Type nlist server /b and then press ENTER to receive a list of Novell servers.

    9. Type exit and then press ENTER to exit from the MS-DOS prompt.

Exercise 2
Installing Microsoft Service for NetWare Directory Services

In this exercise, you will install Microsoft Service for NetWare Directory Services.

    To install Microsoft Service for NetWare Directory Services

    1. Open Control Panel, and then double-click Network.

      The Network dialog box appears.

    2. On the Configuration tab, click the Add button.

      The Select Network Component Type dialog box appears.

    3. Click Service, and then click Add.

      The Select Network Service dialog box appears.

    4. Click Microsoft in the Manufacturers list, click Service for NetWare Directory Services and then click OK.

      The Network dialog box appears.

    5. Scroll down and verify that Service for NetWare Directory Services is listed under The following network components are installed.

    6. Click OK.

      Windows 98 copies the necessary files from the installation media.

      You are prompted to restart your computer.

    7. Click Yes.

      After Windows 98 reboots, the Selecting NDS Tree dialog box appears.

    To log in to a NetWare server running NetWare Directory Services

    1. Click on your preferred directory tree, and then click Select.

      The Enter Network Password dialog box appears.

    2. Click Advanced.

      The Advanced Login Settings dialog box appears. Notice that Log in to a directory tree is automatically selected.

    3. What other information appears in the dialog box?

      Answer

    4. Click Log in to a bindery server.

    5. What information changed inside the dialog box?

      Answer

    6. Click Log in to a directory tree, type context, and then click OK.

    7. In the Enter Network Password dialog box, type NetWare_user_name and password and then click OK.

    8. Click OK.

      The Set Windows Password dialog box appears.

    9. Re-enter your password and click OK.

      The Enter Network Password dialog box appears.

      Why are you prompted to log on again?

      Answer

    10. Enter your Domain_user_name and Domain_password, and click OK.

      Your domain logon script executes, followed by your NetWare logon script.

    11. Click Start, point to Programs, and then click MS-DOS prompt.

      An MS-DOS prompt appears.

    12. At the MS-DOS C:\> prompt, type N: and then press ENTER.

    13. Type CD \PUBLIC and then press ENTER to change to the \Public folder.

      The N:\Public> prompt appears.

    14. Type nlist server /b and then press ENTER to receive a list of Novell servers.

    15. Type exit and then press ENTER to exit from the MS-DOS prompt.
