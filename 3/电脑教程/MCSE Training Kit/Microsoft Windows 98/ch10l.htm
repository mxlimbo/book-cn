Editing Msdos.sys

Windows 98 Setup creates a hidden, read-only system file named Msdos.sys in the root of the computer's boot drive. This file contains important paths to other Windows files, including the registry files. Msdos.sys is also necessary for compatibility with applications that require this file to be present before you can install them.

Msdos.sys supports an [Options] section, with which you can tailor the startup process. You can edit Msdos.sys with any text editor (see Figure 10.3) after you remove its read-only attribute.

Figure 10.3 Editing Msdos.sys with Notepad

Caution

There is an extra section of commented text at the end of Msdos.sys that is required for compatibility with virus protection programs that expect this file to be at least 1,024 bytes.

Entries in Msdos.sys include those in the following table:

[Paths] section:
Entry Defines the Location of
WinDir= The Windows 98 directory as specified during Setup.
WinBootDir= The necessary startup files. The default is the directory specified during Setup; for example, c:\windows.
HostWinBootDrv= The boot drive root directory.
[Options] section:
Entry Enables
AutoScan= ScanDisk after an improper shutdown. If set to 0, ScanDisk does not run. The default value is 1, which causes Windows 98 to run ScanDisk automatically after improper shutdown.
BootDelay=n The initial startup delay in n seconds. The purpose of the delay is to give the user sufficient time to press a startup option key. The default is 2.
BootGUI= Automatic graphical startup into Windows 98. The default is 1 (enabled).
BootMenu= Automatic display of the Startup menu. The default is 0 (disabled).
BootMulti= When set to 1, your previous operating system can be loaded through the Startup menu (When you boot your previous operating system, Msdos.sys is temporarily renamed Msdos.w40). When set to 0, you cannot boot your previous operating system. The default is 1.
BootMenuDelay= The number of seconds to display the Startup menu before running the default menu item. The default is 30.

Note

For a complete list of Msdos.sys entries, refer to Appendix G, "Contents of the Windows Msdos.sys File."
