Using Windows 98 Troubleshooters

Windows 98 Help includes troubleshooting aids called Troubleshooters for solving problems with various system components. Troubleshooters are interactive help files that prompt you for input regarding the problems you are experiencing and the results you obtain after taking suggested actions. For instance, the Startup and Shutdown Troubleshooter can help you troubleshoot the boot process with the tools described earlier in this chapter.

The Hardware Conflict Troubleshooter (see Figure 10.6) helps you locate the source of a problem with conflicting hardware devices, and suggests corrective actions. There are numerous other Troubleshooters for components and functions such as networking, modem, printing, PC cards, sound, and memory.

Figure 10.6 The Hardware Conflict Troubleshooter in Windows 98 Help

When a problem occurs with a hardware device, the properties sheet in Device Manager for that device will include a Troubleshooter button that starts the appropriate Troubleshooter.

If you want to use a Windows 98 Troubleshooter, perform the following procedure:

  1. On the Start menu, click Help.

  2. On the Contents tab, click Troubleshooting, then click Windows 98 Troubleshooters.

  3. Select the category that corresponds to the problem.

    In the right pane of the Help window, you are prompted for the type of problem you have. Answer each question, and take the actions the troubleshooter suggests.
