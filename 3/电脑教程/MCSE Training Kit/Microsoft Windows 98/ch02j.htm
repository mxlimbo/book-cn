Lesson 2: Upgrading from Windows 95 or Windows 3.1x

Windows 98 performs Setup in five phases. Most of these phases are the same whether you run Setup from Windows 95 or from MS-DOS. The five phases of running Setup from Windows 95 are:

  • Phase 1—Preparing to run Windows 98 Setup

  • Phase 2—Collecting information about your computer (this phase also creates the emergency startup disk)

  • Phase 3—Copying Windows 98 files to your computer

  • Phase 4—Restarting your computer

  • Phase 5—Setting up hardware and finalizing settings

This lesson discusses the steps involved in upgrading from Windows 95 or Windows 3.1x.

After this lesson, you will be able to:

  • List the five Setup phases for Windows 98.

  • Run Setup to upgrade from Windows 95.
