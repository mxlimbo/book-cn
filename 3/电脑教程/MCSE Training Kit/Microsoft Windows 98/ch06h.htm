Configuring Display

The Display icon in Control Panel opens the Display Properties dialog box (see Figure 6.3), which provides options for configuring screen resolution, color depth, appearance settings, and screen savers. Higher resolutions increase the screen area and may be required by some applications, but can also make fonts and icons uncomfortably small for some users. You should ensure that the users you support can view the selected resolution without difficulty.
