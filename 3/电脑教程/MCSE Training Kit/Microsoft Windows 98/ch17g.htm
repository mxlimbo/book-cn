Practice: Configuring Internet Explorer

In this practice, you will set Internet Explorer options such as the connection type, proxy server settings, news, and email information.

    To configure Microsoft Internet Explorer

    1. Double-click the Connect to the Internet icon on your desktop.

      The Internet Connection Wizard dialog box appears.

    2. Click I have an existing Internet account through my phone line or a local area network (LAN) and then click Next.

    3. Click Next.

      The Set Up Your Internet Connection screen appears.

    4. Click Connect using my local area network (LAN) and then click Next.

      The Proxy Server screen appears. You are asked if your LAN uses a proxy server to connect to the Internet.

    5. Click No and then click Next.

      The Set Up Your Internet Mail Account screen appears. You are asked if you want to set up an Internet mail account.

    6. Click No and then click Next.

      The Set Up Your Internet News Account screen appears. You are asked if you want to set up an Internet news account.

    7. Click No and then click Next.

      The Set Up Your Internet Directory Service screen appears. You are asked if you want to set up an Internet directory service account.

    8. Click No and then click Next.

      The Complete Configuration screen appears.

    9. Click Finish.
