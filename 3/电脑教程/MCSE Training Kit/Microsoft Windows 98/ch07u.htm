Lesson 3: Using Active Desktop

The Active Desktop in Windows 98 integrates the Windows desktop with Internet Explorer. With Web view enabled, you can use the Active Desktop interface to put active content from Web pages or channels on your desktop. Examples of active content include stock reports, weather reports, and other news reports. This lesson describes the procedures for configuring Active Desktop items, adding Web sites to the Active Desktop, and recovering the Active Desktop.

After this lesson, you will be able to:

  • Configure the Active Desktop.

  • Add a Web site to the Active Desktop.

  • Recover Active Desktop.

Estimated lesson time: 25 minutes
