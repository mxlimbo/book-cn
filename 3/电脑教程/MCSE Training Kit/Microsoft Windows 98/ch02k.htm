Phase 1—Preparing to Run Windows 98 Setup

After you start Setup.exe, the opening Windows 98 Setup screen appears (see Figure 2.2).

Figure 2.2 Opening Windows 98 Setup screen

The following information appears on the left side of the screen:

  • The five Setup phases, with the current stage highlighted

  • The time remaining to complete Setup

  • A description of the current Setup activity

Setup checks for anti-virus software, and, if any is found, displays a message prompting you to disable it. Setup then creates Setuplog.txt, which records information about the setup process as it occurs, in the root directory of drive C.

After you click Continue, Setup creates a temporary directory named C:\Wininst0.400. Setup expands and copies the files that are required to run the Setup Wizard from Precopy1.cab, Precopy2.cab, and Precopy 3.cab into C:\Wininst0.400.

Accepting the License Agreement

After the Setup Wizard has loaded, the License Agreement appears. You must accept the License Agreement to continue Setup.
