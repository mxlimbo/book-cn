Practice: Configuring Internet Explorer with a Start Page

In this practice, you will set Internet Explorer to open with a start page.

    To open Internet Explorer with a start page

    1. Start Internet Explorer.

    2. On the View menu, click Internet Options.

    3. The Internet Options dialog box appears and displays the General tab.

    4. In the Address box, type the location of your start page.

Configuring System Policies

To prevent users from running Internet Explorer normally, or from running any other program, you should apply system policies to the kiosk computer so that all items on the desktop are removed, and Internet Explorer is the only program that users can run. Use System Policy Editor in Registry mode to configure the computer to retrieve its policy file from its local hard disk, so that policies still apply even if a server is unavailable.
