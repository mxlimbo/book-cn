Lesson 2: Windows 98 Features

The new and enhanced features of Windows 98 provide a faster, more powerful, and more manageable work environment. Windows 98 offers key improvements to the Windows environment, advanced support for new hardware standards, and improved communications and networking support.

This lesson summarizes the improvements to the Windows environment provided by Windows 98.

After this lesson, you will be able to:

  • Describe the key improvements implemented in Windows 98.

  • Describe the new hardware standards supported by Windows 98.

  • List the network and communications features supported by Windows 98.

Estimated lesson time: 10 minutes
