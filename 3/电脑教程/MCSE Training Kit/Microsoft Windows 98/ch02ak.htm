Procedure for Removing the Windows 95 System Files

To remove the Windows 95 system files, you can delete the Winundo.dat and Winundo.ini files manually. However, the Remove Windows 95 system files (Uninstall info) and the Uninstall Windows 98 options will remain on the Install/Uninstall tab of the Add/Remove Programs dialog box.

If you want to remove Windows 95 system files, perform the following procedure:

  1. In Control Panel, click Add/Remove Programs.

  2. Click the Install/Uninstall tab, select Remove Windows 95 system files (Uninstall info), and then click Add/Remove.

    The Windows 98 Uninstall dialog box appears, stating that if you remove these files, you will no longer be able to uninstall Windows 98.

  3. Click Yes.

    A dialog box appears informing you that the files have been successfully deleted.

  4. Click OK.

    The Uninstall Windows 98 and Remove Windows 95 system files (Uninstall info) options no longer appear on the Add/Remove Programs Properties page.
