Lesson Summary

The Active Desktop integrates the Windows desktop with Internet Explorer, enabling you to put Web content—such as graphics, HTML documents, and ActiveX controls—directly on your desktop.

Configuring an Active Desktop item involves adding the item to the desktop, positioning it, and resizing it. If the item added is a URL pointing to the Web site, you can subscribe to the site with the Subscription Wizard. When you subscribe to the site, Explorer checks the site for updates and either notifies you of the new content or automatically downloads it to your computer.
