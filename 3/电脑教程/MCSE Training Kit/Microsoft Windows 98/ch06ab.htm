Summary

The following information summarizes the key points in this chapter:

Configuring Hardware Settings

  • Control Panel is a folder containing icons with which you open property sheets to customize the computer's configuration.

  • Use Control Panel to access the Add New Hardware Wizard, which allows you to install hardware devices that are not Plug and Play compatible.

  • Use Control Panel to access the System icon, which allows you to create and modify hardware profiles.

  • Use Control Panel to access the Display icon, which provides options for configuring screen resolution, color depth, appearance settings, and screen savers.

Configuring System Settings

  • With virtual memory, Windows 98 keeps some information in RAM while it swaps, or temporarily moves, other information to a virtual memory swap file. Configure virtual memory with the Virtual Memory dialog box.

  • Use the File Systems Properties sheet to set the hard disk cache and the CD-ROM File System cache, and to configure floppy and removable disk drives.

  • Use the Accessibility Options icon in Control Panel to configure the computer for users with vision, hearing, and mobility needs.

Creating the Software Environment

  • Use the Add/Remove Programs Properties sheet to install and remove components and applications.

  • When you remove programs with Add/Remove, their entries in the registry are marked as deleted. Once a day upon startup, Microsoft Registry Checker scans and backs up the registry. If Registry Checker contains 500K or more of keys marked as deleted, it will optimize the registry the next time you start your computer.

Using the Registry to Configure Windows 98 Settings

  • The registry is a database used for storing software and hardware configuration settings. Windows 98 creates the registry during Setup and updates it when changes are made to hardware or software configuration.

  • The components of the registry consist of subtrees (analogous to the root folder of a disk), keys and subkeys (analogous to folders), and values (analogous to files).

  • To alter a registry value, you can use the Registry Editor, although Microsoft does not recommend or support making changes directly to the registry when an alternative exists.
