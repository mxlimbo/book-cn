Lesson 2: Using Web-based Troubleshooting Tools in Windows 98

As discussed in Chapter 9, "Maintaining Windows 98 in a Stand-alone Environment," Windows 98 provides a number of Web-based resources that assist you in accessing technical support and obtaining the latest versions of device drivers and system files. These tools are also useful when troubleshooting Windows 98 in a network environment. This lesson provides an overview of the following tools: Windows Update, Windows Update Technical Support, and Windows Report Tool.

After this lesson, you will be able to:

  • Use Windows Update to update drivers for devices on a network.

  • Use Windows Report Tool to resolve complex problems.

Estimated lesson time: 15 minutes
