Troubleshooting PPTP Connections

The following table lists some common PPTP connection problems that can occur and some recommended solutions.

Problem Solution
PPTP doesn't work after ISDN installation If you install an ISDN card after installing PPTP, the ISDN installation procedure may overwrite some of your DUN files. Remove and reinstall DUN using Add/Remove Programs.
Cannot ping PPTP server The PPTP server probably has PPTP filtering enabled, which prevents pinging it. Contact your PPTP server administrator to disable filtering.
Cannot log on to target network If you can connect to the PPTP tunnel server, but cannot perform a domain logon to the target network, verify that you have configured the connection to your PPTP server correctly.
