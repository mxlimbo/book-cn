Launching Applications from the Desktop

With the Web style desktop enabled in Windows 98, you can select objects using hover select, and start them using single-click execution.

Using Hover Select

When you position your pointer over an object using hover select, the pointer behaves in the same way as when it is positioned over a hyperlink in Internet Explorer. The pointer changes to a hand, and if you leave the pointer over an object, the object becomes selected.

Using Single-Click Execution

Using single-click execution, you can start a program or open an object by clicking the object once. For example, if you click My Computer on the desktop, a window opens displaying the contents of My Computer. Clicking a shortcut to a Microsoft Word document opens the document in Microsoft Word.

Users who are familiar with the Windows 95 interface may require a period of adjustment to this method of starting a program. For example, with the Windows 95 interface, you can rename an object by selecting it, then clicking the object again. However, if a user does this with single-click enabled in Windows 98, the selected program or object opens.

If you want to rename an object using single-click execution:

  1. Right-click the object.

  2. Click Rename on the context menu.

  3. Type in the new file name.

  4. Press ENTER to save the new name.
