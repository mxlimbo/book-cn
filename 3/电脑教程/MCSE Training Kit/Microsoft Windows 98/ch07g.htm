Taskbar Enhancements

The taskbar (see Figure 7.2) has been enhanced to facilitate starting tasks and switching between them.

Figure 7.2 Windows 98 Taskbar

Enhancements to the Windows 98 Taskbar include:

  • Multiple toolbars

    You can customize standard and user-defined toolbars and place them on the taskbar or anywhere on the desktop.

  • Enhanced Minimize function

    Click programs on the taskbar to toggle between maximized and minimized window display.

Multiple Toolbars

The Windows 98 taskbar supports multiple toolbars. Toolbar bands, which appear on the left side of each toolbar, allow you to move and resize each toolbar. You can also add toolbars to customize your desktop.

You can select from the following standard Windows 98 toolbar options:

Toolbar Function
Address Adds the address bar from Internet Explorer.
Links Adds buttons for popular Internet Explorer links.
Desktop Displays the desktop icons as toolbar buttons.
Quick Launch Provides buttons to start Internet Explorer and Outlook Express, to Show Desktop, and to view Active Channels. Active Channels is a full-screen window that you can use to view channels. Channels are Web sites that automatically deliver content to a client computer on a regular schedule.
New Toolbar Creates a customized toolbar of your choice.

When you add a toolbar, you can place it on the taskbar, have it float on the desktop, or position it on any side of the screen. You can create a toolbar for any resource that you can access, including local folders, network folders, Internet resources, and intranet resources.
