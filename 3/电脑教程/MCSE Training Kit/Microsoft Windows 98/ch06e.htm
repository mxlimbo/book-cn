Using Control Panel to Configure Windows 98

Figure 6.1 shows many of the icons available in Control Panel. You can open Control Panel from the Start menu (Setting, Control Panel) or My Computer.

Figure 6.1 Windows 98 Control Panel

Windows 98 Control Panel includes the icons in the following table:

Use This Icon To
32-Bit ODBC Administer ODBC (Open Database Connectivity) Data Sources, including user, system and file data sources. View information about installed ODBC drivers and core components.
Accessibility Options Alter various settings to meet the needs of users with hearing, movement, and vision disabilities. Change the system display for better contrast and alter the functions of the keyboard.
Add New Hardware Automatically or manually add device drivers for hardware that is not Plug and Play.
Add/Remove Programs Add and remove programs and Windows 98 components. Create a startup disk.
Date/Time Change the system date, time, and time zone. Enable the computer clock to adjust for daylight savings time.
Desktop Themes Choose various themes that modify wallpaper, screensaver, sounds, icons, fonts, colors, and pointers based on themes such as nature.
Display Specify the background, screen saver, and visual appearance of Windows 98 interface and programs. Configure display adapter, monitors, desktop icons, color management profiles, and Active Desktop settings.
Fonts View, add, and remove fonts.
Game Controllers Add, remove, configure, and test game controllers.
Infrared Add and configure infrared devices. This icon appears in Control Panel when an infrared device(s) is installed.
Internet Configure Internet Explorer properties.
Keyboard Specify the speed of cursor blink and character repetition, keyboard language, and keyboard type.
Modems Install a new modem, configure existing modems, or remove an unneeded modem.
Mouse Customize the button settings and the pointer scheme and motion.
Multimedia Specify or change settings for audio, video, MIDI, and other multimedia devices.
Network Add, remove, and configure network clients, adapters, protocols, services, and file and printer sharing. Provide network identification information and specify share or user level access control.
Passwords Change passwords, enable remote administration, and change user profile settings.
Power Management Add, remove, and configure power schemes.
Printers Add, remove, configure, and manage printers.
Regional Settings Specify international settings such as the country and language, number and currency formats, and date and time formats.
Sounds Change system and program sounds assigned to events. Add, remove, and configure sound schemes.
System Examine and configure system hardware. Create hardware profiles. Configure virtual memory and file system settings and other system device properties.
Telephony Configure telephony drivers and dialing properties.
Users Set up and manage multiple users on a computer.
