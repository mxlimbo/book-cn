Troubleshooting Multiple Display Support

If you encounter problems using multiple displays, use the following troubleshooting guidelines to aid in resolving those problems.

If you can't see any output on the secondary display:

  • Confirm that the device has been activated through Display Properties.

  • Confirm that the correct video driver is selected for the secondary adapter.

  • Confirm that the secondary display has been initialized during restart. If not, check device manager for the status of the video adapter.

  • Physically switch the order of the adapters in the PCI slots (this requires the primary display to also qualify as a secondary display).

If the "Extend my Windows desktop onto this monitor" option is disabled, confirm the following:

  • The secondary display is highlighted in Display Properties.

  • The secondary display adapter is supported.

  • The secondary display has been detected.

If you have problems running a program using multiple displays:

  • Run the program on the primary display.

  • Run the program full screen (MS-DOS) or maximized (Windows).

  • Disable the secondary display to see if the issue is specific to Multiple Display.
