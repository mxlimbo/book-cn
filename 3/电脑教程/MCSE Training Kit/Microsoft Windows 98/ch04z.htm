Practice: Configuring Multiple Displays

In this practice, you will configure multiple display devices.

To complete this practice, you must have multiple display adapters installed.

    To Configure Multiple Displays

    1. In Control Panel, double-click the Display icon.

    2. In the Display Properties dialog box, click the Settings tab. Numbers in the framed areas indicate monitors configured for the system. Frame 1 is for the primary (main) display; frames 2 through 9 are for the secondary displays.

    3. Select the display adapter for the primary display. In the Colors menu, select the color depth. In the Screen Area menu, select the resolution.

    4. Click frame 2 to activate it. In the Display menu, select the display adapter for this secondary display. Click the Extend my Windows desktop onto this monitor box to place a check mark in it. In the Colors menu, select the color depth. In the Screen Area menu, select the resolution. If there are additional secondary displays, repeat step 4 for each one.

Note

You must have multiple display adapters installed and configured to have these options on your Settings tab.

    1. You can change the monitor positioning on the virtual desktop by dragging the frames in Display Properties. The positions of the frames dictate the relative positions of the displays to each other. If one screen is located above the other, then the mouse pointer must be moved up or down to reach the other screen.
