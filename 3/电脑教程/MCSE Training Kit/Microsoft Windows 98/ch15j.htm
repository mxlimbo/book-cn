Enabling Through Users Icon

You can also enable user profiles by double-clicking the Users icon in Control Panel. There are several advantages to using this icon to manage profiles:

  • You can create user profiles, including user names and passwords, without logging on as each user.

  • The Add User Wizard provides more options than those that are available with the Passwords icon.

  • You can configure settings for each user rather than for all profiles.

After you enable user profiles, double-clicking the Users icon in Control Panel opens the User Settings property sheet (see Figure 15.3) and displays the User List of all user profiles available on the local computer.

Figure 15.3 User Settings property sheet

You can modify an existing user profile or create a new user profile. Select the items you want to add to the profile from the following list:

  • Desktop folder and Documents menu

  • Start menu

  • Favorites folder

  • Downloaded Web pages

  • My Documents folder

After you select the items from the list, choose between two options for creating the profile folders:

  • Create copies of the current items and their content.

    Copies the default user folders and their contents for the selected items to the newly created profile.

  • Create new items to save disk space.

    Creates empty folders for the selected items. Contents are added when the new user logs on and chooses the desired settings.
