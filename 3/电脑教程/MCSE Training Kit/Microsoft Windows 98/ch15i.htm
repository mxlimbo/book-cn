Enabling Through Passwords Icon

One way to enable user profiles is with the Passwords icon in Control Panel.

If you want to enable user profiles, perform the following procedure:

  1. Double-click the Passwords icon in Control Panel, then click the User Profiles tab (see Figure 15.2).

  2. Select Users can customize their preferences and desktop settings.

Figure 15.2 User Profiles tab

With this method, Windows 98 creates a default profile that contains the settings in use at the time that you enable user profiles. New users have the option to save their settings when they log on. If a user chooses to save the settings, Windows 98 creates a user profile for that user name.

On the User Profiles tab, there are two User Profile Settings options:

  • Include desktop icons and Network Neighborhood contents in user settings.

  • Include Start menu and Program groups in user settings.

You can select either or both of these settings when you enable profiles, and that selection determines which user settings apply to user profiles. If you select neither of these settings, the user profiles contain only the user-specific registry settings.
