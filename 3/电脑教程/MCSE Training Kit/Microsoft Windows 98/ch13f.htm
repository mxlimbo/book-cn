Lesson Summary

Windows 98 is fully interoperable with Windows NT. A Windows 98 client running on a Windows NT network can log on to a domain, share resources with a computer running Windows NT, gain access to Microsoft BackOffice applications running on a Windows NT Server, and download user profiles and system policies.

The Windows 98 Client for Microsoft Networks supports the Microsoft Distributed File System for Windows NT Server, which provides a single tree structure for accessing multiple shared resources located on different servers on a network.
