Troubleshooting Connection Problems

If you suspect the problem is with the connection settings, you can enable a log file (Ppplog.txt) that records connection and negotiation data for PPP connections.

If you want to enable Ppplog.txt, perform the following procedure:

  1. In Control Panel double-click the Network icon.

  2. Select Dial-Up Adapter and click Properties.

  3. On the Advanced tab, select Record a log file in the Properties box.

  4. In the Values box set the value to Yes.

  5. Click OK.

The information contained in the ppplog.txt file may help the remote server administrator troubleshoot the problem.

Note

For information about the Ppplog.txt file, refer to Appendix K, "How to Interpret the PPPlog.txt File," as well as the Microsoft Windows 98 Resource Kit.
