Lesson 1: An Overview of System Policies

This lesson provides an overview of system policies. System policies offer a powerful mechanism for network administrators to increase control and manageability of desktop computers across a network. You can use system policies as part of your remote administration program in order to manage your domain or network more efficiently.

As a support professional, you need to understand what system policies can do and how they affect computers running Windows 98. You can use Windows 98 system policy files with a Windows NT or Novell NetWare network server when Windows 98 is used as a client on either network.

The decision to implement system policies should be based on a thorough understanding of their value and limitations and when it is appropriate to use them.

After this lesson, you will be able to:

  • Describe what system policies are.

  • Describe how system policies are implemented.

  • Identify situations in which it is appropriate to use system policies.

Estimated lesson time: 20 minutes
