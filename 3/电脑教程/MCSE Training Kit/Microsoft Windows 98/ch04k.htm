Configuring Power Management

The Power Management icon in Control Panel (see Figure 4.2) is used to configure power management.

Figure 4.2 Power Management properties

Power Schemes

Windows 98 comes with preset power management configurations, called Power Schemes. Each scheme can be configured to meet the power management requirements of a particular system.

If none of the preconfigured schemes is appropriate, you can either change the properties of an existing scheme or create a new scheme.

If you want to change the properties of an existing scheme:

  1. Double-click the Power Management icon in Control Panel.

  2. Select the scheme from the list of Power schemes.

  3. Adjust the settings for the System standby, Turn off monitor, and Turn off hard disks options as required.

  4. On the Advanced property sheet, determine whether to show the power meter on the taskbar and whether to have the computer prompt for a password as it comes out of standby.

  5. To create a new scheme, adjust the settings as outlined above, click Save As, and then provide a name for the new scheme when prompted.

Note

The settings on the Advanced property sheet apply to all power schemes.
