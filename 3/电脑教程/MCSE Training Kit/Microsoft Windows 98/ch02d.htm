Lesson 1: Preparing for Installation

Preparing to install Windows 98 involves determining hardware requirements, preventing software conflicts during Setup, dealing with issues that may arise from dual booting with Windows 98, and selecting the Setup method you will use. This lesson addresses the steps necessary to successfully install Windows 98.

After this lesson, you will be able to:

  • Determine if your computer meets the minimum Windows 98 hardware requirements.

  • Identify the types of software that might interfere with installation.

  • List the guidelines for dual booting Windows 98 and another operating system.

  • Choose the appropriate Setup method that you will use to install Windows 98.

Estimated lesson time: 15 minutes
