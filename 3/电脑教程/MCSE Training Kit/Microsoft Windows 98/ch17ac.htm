Optional Practice: Installing and Configuring Personal Web Server

In this practice, you will install and configure Personal Web Server.

Exercise 1
Installing Personal Web Server

In this exercise, you will install Personal Web Server.

    To install Personal Web Server

    1. Click Start, then click Run.

    2. In the Run dialog box, type D:\Add-ons\Pws\Setup.exe (where D: is the name of your CD-ROM drive).

      The Microsoft Personal Web Server Setup dialog box appears.

    3. Click Next.

      You are prompted to select a configuration level.

    4. Click Typical.

      You are prompted for the location of the Web publishing home directory.

    5. Click Next.

      A status bar appears showing the progress of the installation.

    6. Click Finish.

      This completes the installation. You are prompted to restart your computer.

    7. Click Yes.

      The computer restarts.

    8. Log on to Windows 98.

Exercise 2
Configuring Personal Web Server

In this exercise, you will configure Personal Web Server.

    To configure Personal Web Server

    1. Click Start, point to Programs, Internet Explorer, Personal Web Server, and then click Personal Web Manager.

    2. The Tip of the day dialog box appears.

    3. Click Close.

    4. The Personal Web Manager window appears displaying publishing, monitoring, and statistical information.

    5. Click Web Site.

    6. The Home Page Wizard appears.

    7. Click on the Wizard.

    8. A Security Alert dialog box appears warning that your information may be seen by other people.

    9. Click In the future do not show the warning for this zone and then click Yes.

    10. Select a template for your home page and then click Next.

      You are prompted to create a guest book.

    11. Click Yes and then click Next.

      You are prompted to create a drop box.

    12. Click Yes and then click Next.

    13. You are prompted to customize your home page.

    14. Click Next.

    15. Internet Explorer opens and displays a template.

    16. Maximize Internet Explorer.

    17. Type in a title for your home page.

    18. Type in your name and e-mail name.

    19. Scroll down and click Enter new changes.

    20. Your home page appears with the title you typed in.

    21. Close Internet Explorer.
