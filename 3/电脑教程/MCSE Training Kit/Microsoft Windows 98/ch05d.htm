Lesson 1: The Windows 98 System Architecture Overview

Microsoft Windows 98 is a 32-bit operating system that provides built-in Internet connectivity, Plug and Play hardware support, high performance, robustness, and backwards compatibility with Windows 95. This lesson presents an overview of Windows 98 architecture and discusses how Windows 98 performs multitasking.

After this lesson, you will be able to:

  • List and describe the core system components of Window 98.

  • Describe key features of Windows 98 architecture.

  • Describe two methods of multitasking in Windows 98.

  • View a list of tasks running on your computer.

Estimated lesson time: 30 minutes
