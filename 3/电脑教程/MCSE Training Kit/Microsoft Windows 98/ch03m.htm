Optional Practice: Creating Partitions

To complete this practice you must have an extended partition in which you can create a logical DOS drive, or a primary DOS partition with free space on your hard drive in which you can create an extended partition. In this practice you will create and format a new partition.

Exercise 1
Creating a Partition

In this exercise, you will use FDISK to create a new partition from free space on your drive.

    To create a new partition

    1. Click Start, point to Programs, and then click MS-DOS Prompt.

      An MS-DOS prompt appears.

    2. At the MS-DOS prompt, type fdisk and press ENTER.

      FDISK prompts you to enable large disk support.

    3. Type N and then press the ENTER key.

      The FDISK main menu appears.

    4. In the Enter Choice box, type 4 and then press ENTER to view your current partition information. Record the information in the following table.

      Partition Status Type Volume Label Mbytes System Usage
                   
                   

    5. Press ESC to return to the FDISK Options menu, and then press enter to accept the default selection (Create DOS partition or Logical DOS Drive).

      The Create DOS Partition or Logical DOS Drive screen appears.

    6. If you have not already created an extended DOS partition, type 2 in the Enter Choice box, and then press Enter. If you already have an extended DOS partition, go to step 7.

      This will create a new extended DOS partition.You are prompted to enter the size of the partition you wish to create.

    7. Type 550 to create a 550 MB extended partition, press Enter, and then press Esc to continue.

      You are prompted to create a logical drive in the extended partition.

    8. Press Enter to accept the default size when prompted to create a logical drive.

      A screen appears indicating that you have created a logical drive.

Note

Fdisk may adjust your partition size to accommodate your hard disk's geometry.

    1. Press ESC to continue, and then press ESC again to exit FDISK.

      You are prompted to restart Windows before the changes you made will take effect.

    2. Press ESC and then restart your computer.

    To verify the creation of the partition

    1. Start an MS-DOS prompt.

    2. At the MS-DOS prompt, type fdisk and press ENTER.

      FDISK prompts you to enable large disk support.

    3. Type N and press ENTER.

      The FDISK main menu appears.

    4. Type 4 and then press ENTER to view your current partition information, and record it in the following chart.

      Partition Status Type Volume Label Mbytes System Usage
                   
                   

    5. Press ENTER to view the Logical Drive information, and record it in the following table.

      Drive Volume Label Mbytes System Usage
               

    6. Press ESC to return to the FDISK Options menu, and then press ESC to exit FDISK.

Exercise 2
Formatting a Partition

    To format the new partition

    1. Open My Computer

    2. Right-click the drive letter icon and click Format.

      The Format dialog box appears.

    3. Under Format type, click Full.

      Leave the Label box empty.

    4. Click Start.

      A message box warns you that all files will be destroyed and asks if you want to format this drive.

    5. Click OK.

      A progress bar appears and when the format is complete, a summary box appears.

    6. Click Close to dismiss the summary box.

      A message box appears and suggests that you run ScanDisk before using the newly formatted partition.

    7. Click OK.

      A help window for ScanDisk appears.

    8. Close the Help window.

    9. Click Close.

      The partition is now formatted and ready for use.
