Practice: Configuring a Local Printer in Windows 98

In this practice, you will configure a local printer with the printer's Properties sheet.

    To configure a printer

    1. Click Start, Settings, and select Printers.

    2. In the Printers folder, right-click the icon of the printer that you wish to configure.

    3. Click Properties.

      The tabs that appear indicate which properties you can configure for the selected printer.

Note

Printer properties include port and graphics settings, paper and font options, and other printer-specific device options. All printers will display the General and Details tabs, but other tabs will display only if your printer has the capability to change other properties.

The following tables explain the General and Details tabs.

General Tab

The General tab contains basic printer information.

Use This Option To
Comment Provide any desired information, such as to clarify when the printer is available or what it is used for. This information is copied to a remote user's computer when connected to your shared printer.
Separator Page Print a separator page between print jobs. Default is None. Simple prints a page with text; Full prints a sheet including graphics.
Print Test Page Print a page to verify that the correct driver and port settings are in use. Also lists what files are in use by the printer driver.

Details Tab

You can use the fields in the top part of the Details tab to specify or change a port and driver for the selected printer.

Use This Option To
Print to the following port Select a port for the printer output. If you select File, the print job does not go to a printer. Instead, Windows 98 creates a file with an extension of .prn. Windows Explorer lists these files as Printer Files. These files are fully rendered files for the selected printer and cannot be edited.
Print using the following driver Change printer drivers.
Add Port Add either a Network or Local port.
Delete Port Delete an existing port.
New Driver Select a new printer driver.
Capture Printer Port Redirect print jobs sent to local ports to network printers.
End Capture Stop redirection of a local port to a network printer.

Note

Printing to a file has several uses. For instance, if your printer is not working properly, you could copy the .prn file to a disk and use it on another computer connected to the same type of printer.

You can also use .prn files to prepare work for a printer to which you do not have access. For example, you might be hiring an offsite printing service to print your work. You would add the appropriate printer using the Add Printer Wizard and configure it to print to File, then give the .prn files to the printing service.

You use the options at the bottom of the Details tab to specify Timeout settings as presented in the following table:

Use This Option To
Not Selected Specify how long Windows 98 waits for the printer to come back online before it reports an error message. Increase this setting if you have a very slow printer.
Transmission retry Specify how long Windows 98 tries to send data when the printer cannot accept it before displaying an error message. Increase this setting if you have a very slow printer.

Setting Up Multiple Configurations

You can set up multiple configurations for a single printer. For instance, if you have a printer that prints in both PostScript and non-PostScript printer languages, you can set up two configurations for the printer by running the Add Printer wizard and supplying the same physical location of the hardware, but a different local printer name for each printer language.

Note

PostScript is the language most often used for high-resolution printing and for producing camera-ready copy for publishing.
