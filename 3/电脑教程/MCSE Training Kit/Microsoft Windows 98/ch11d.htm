Lesson 1: Installing Multiple Copies in a Network Environment

Installing Windows 98 on one computer is simple. Installing Windows 98 on multiple computers on a network requires careful planning to ensure that the process minimizes disruption and costs. Before you install Windows 98 in a network environment, consider developing a multiple installation plan. This involves setting the goals for the timing and scope of the installation process department-wide, and updating the company's related policies and standards.

This lesson describes how to plan and select an installation method for installing multiple copies of Windows 98 in a network environment.

After this lesson, you will be able to:

  • Plan for multiple installations.

  • Describe the three installation methods.

  • Choose an appropriate installation method.

Estimated lesson time: 25 minutes
