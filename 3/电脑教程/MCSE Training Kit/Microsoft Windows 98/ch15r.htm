Lesson 4: Configuring User Profiles on a Network

When implementing roaming profiles, the location of the profiles is different depending on the type of network.

This lesson discusses the differences between implementing roaming profiles on a Windows NT network and a Novell NetWare network. This lesson also describes the procedure for implementing mandatory user profiles.

After this lesson, you will be able to:

  • Describe the differences between implementing user profiles on Windows NT networks versus Novell NetWare networks.

  • Implement a mandatory user profile.

Estimated lesson time: 10 minutes

When you implement roaming profiles, any computer on the network that a user logs on to configures to the settings in that user's profile. On Windows NT servers, the profile is stored in the home folder, while in Novell NetWare 3.x Networks, the profile is stored in the SYS:\MAIL\<user_id> directory (see Figure 15.5).

Figure 15.5 User profiles on Windows NT and Novell NetWare networks

Important

The system clocks on the user's computer and the logon server must match for the roaming profiles to work properly. You synchronize clocks by using the net time command. System clocks automatically synchronize when logging on to a Novell NetWare network.
