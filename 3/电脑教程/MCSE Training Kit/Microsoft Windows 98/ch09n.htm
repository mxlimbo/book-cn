Using the Maintenance Wizard

The Maintenance Wizard schedules the running of system tools such as Disk Defragmenter, ScanDisk, and Disk Cleanup, allowing you to automate the following tasks:

  • Speed up programs

    Rearrange programs stored on your hard disk to speed up the opening and running of these programs.

  • Scan hard disk for errors

    Check file folders and disk surface for errors and fix them automatically.

  • Delete unnecessary files

    Delete temporary files to free up disk space.

Figure 9.3 The Maintenance Wizard

If you want to start the Maintenance Wizard, perform the following procedure:

  1. Click Start, point to Programs, Accessories, System Tools, and then click Maintenance Wizard.

After using the Maintenance Wizard (see Figure 9.3), each task appears with the label Maintenance in the Scheduled Tasks window in My Computer.
