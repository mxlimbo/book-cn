Practice: Creating a Web Page Subscription

In this practice, you will use Internet Explorer to subscribe to a Web page using the Subscription Wizard.

    To create a Web page subscription

    1. Using Internet Explorer, go to a Web page you wish to subscribe to.

    2. On the Favorites menu, click Add to Favorites.

    3. Click Yes, but only tell me when this page is updated.

    4. Click Customize to launch the Subscription Wizard.

    5. Select the desired e-mail option for this subscription, and then click Next.

    6. If prompted, enter your email name and your e-mail server name, click OK, and then click Next.

    7. Enter any necessary user name and password information for the site, and then click Finish.

    8. Click OK on the Add Favorites dialog box.

Configuring a Web Page Subscription for Offline Reading

You can set up subscriptions so that you do not have to stay connected to the Internet to get updates. Instead, you can schedule Internet Explorer to activate and download changes to subscription pages at designated times, so that you can read the changes offline at your convenience.
