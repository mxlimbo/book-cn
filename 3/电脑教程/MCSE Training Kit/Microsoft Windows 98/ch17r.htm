Using Security Zones

You can control access to the Internet by dividing the Web into zones. A zone is a grouping of sites with the same security level. You assign a web site to a zone, and Internet Explorer notifies you about potential security problems related to the site according to the security level for that zone.

Security levels also determine how Internet Explorer reacts when you connect to a page that has components such as ActiveX controls or Java applets, which produce animation and other multimedia effects. In addition, security levels can determine several other options, including whether or not a user can download files or launch applications from a Web page.

Security Zones

Internet Explorer has four predefined zones that have default security levels.

Zone Default Security Level
Local Intranet Zone Medium: Warn before running potentially damaging content.
Trusted Sites Zone Low: Do not warn before running potentially damaging content.
Internet Zone Medium: Warn before running potentially damaging content.
Restricted Sites Zone High: Exclude content that could damage your computer.

If you want to configure security zones, perform the following procedure:

  1. On the Internet Explorer View menu, click Internet Options.

  2. Click the Security tab.

  3. Select the zone you wish to configure from the Zone menu.

  4. Click the desired security level for that zone.

You can modify the security level for each zone, which gives you more control over security options. In addition, you can customize each security level by choosing Custom, then clicking Settings.

You can add sites to the local intranet, restricted sites, and trusted sites security zones. All Web sites that are not included in one of the other zones are assigned automatically to the Internet zone.
