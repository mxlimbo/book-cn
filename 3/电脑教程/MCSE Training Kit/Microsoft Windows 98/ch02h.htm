Selecting a Setup Method

You can install Windows 98 by upgrading an existing Windows 95 or MS-DOS operating system. You also can install Windows 98 on a newly formatted hard disk or to a new folder.

The recommended procedure for installing Windows 98 is to run Setup from within Windows 95. This method retains your current settings and requires the least amount of user input.

You should run Windows 98 Setup from MS-DOS when:

  • You want to change the current settings.

  • Setup from Windows 95 fails, and you are unable to start Windows again. This could happen if the failure occurs during the file-copying phase.

  • You want to install Windows 98 on a reformatted hard disk or to a different folder than your current Windows folder.

  • You want to upgrade Windows 3.1x.

Note

If you set up Windows 98 from MS-DOS using either a network server or local CD-ROM drive, then real-mode network or CD-ROM drivers must be loaded.

Starting Setup

Windows 98 Setup differs from Windows 95 Setup. Windows 98 requires only five Setup steps, whereas Windows 95 required 12. In addition, hardware detection and enumeration take place after the first restart in Windows 98. In Windows 95 Setup, hardware detection occurred before the file-copy phase. Performing hardware detection after restarting your computer makes Setup more reliable and reduces the chance that your computer will stop responding.

When you run Setup from a CD-ROM drive, network, or removable hard drive, the necessary Driverx.inf files are copied to the local hard disk prior to the first restart. This gives Windows 98 access to the installation media in order to continue Setup after your computer is restarted.

You can start Windows 98 Setup using one of the following methods:

  • From CD-ROM

    Run Setup.exe from the Windows 98 compact disc CD-ROM. Windows 98 requires approximately 175 MB of free space on FAT32 drives, and aproximately 225 MB on FAT16 drives for typical installations.

  • Over a network

    Copy the Windows 98 Setup files from the Windows 98 CD-ROM to a shared directory on a network server, and then run Setup.exe over the network. This requires approximately 115 MB of free disk space on the server, and approximately 130 MB on the local computer.

  • From the hard disk

    Copy the Windows 98 Setup files to the hard disk and then run Setup.exe.
