Summary

The following information summarizes the key points in this chapter:

Win32 Driver Model

  • The Win32 Driver Model (WDM) provides a common set of drivers that allows hardware developers to develop one device driver that will function in multiple Windows-based operating systems.

  • The five layers to the WDM are the Legacy virtualization driver, the Device class driver, the Device mini driver, the Bus class driver, and the Bus mini driver.

Using OnNow Power Management

  • OnNow is a power management feature in Windows 98. When OnNow is enabled, the computer is ready for use immediately when the user presses the On button. The computer appears to be off when not in use but responds to wake-up events, such as a telephone ringing, or software that tells the computer to "wake up" at a specified time.

  • System requirements for OnNow include Advanced Configuration and Power Interface (ACPI). Peripherals must meet ACPI specifications, and applications must support power management and be Plug and Play-compatible.

Supporting Universal Serial Bus Devices

  • The Universal Serial Bus (USB) is a low- to mid-speed bus that you can use for a variety of peripherals, including mouse devices, keyboards, joysticks, scanners, audio devices, and digital cameras.

  • Because USB is fully Plug and Play compatible, you can add USB peripheral devices without having to restart the computer to enable the new device.

Implementing IEEE 1394 Serial Bus Devices

  • IEEE 1394 is similar to USB, except that it supports devices requiring much higher bandwidth.

  • IEEE 1394 devices are Plug and Play compatible, so when installing an IEEE 1394 device, you do not have to restart the computer to enable the new device.

Multiple Display Support

  • Multiple Display Support is a Windows 98 feature that supports several graphics adapters running as many as nine monitors on a single computer.

  • Multiple display devices can be used as a single, expanded desktop. Multiple display is useful for business applications, desktop publishing, presentation graphics, and games.

  • To configure your computer for multiple displays, you must first install additional display adapters.
