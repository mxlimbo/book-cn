Lesson 1: Windows 98 Overview

This lesson provides a brief history of Windows operating systems and provides information related to the capabilities of Windows 3.1x, Windows 95, and Windows NT.

After this lesson, you will be able to:

  • Describe fundamental differences among the common Windows operating systems.

Estimated lesson time: 5 minutes

There are three common Windows operating systems utilized in organizations today: Windows 3.1x, Windows 95, and Windows NT.

Windows 3.1x (including Windows for Workgroups) is a 16-bit operating system that requires Microsoft DOS (Disk Operating System). Windows 95 was released in order to add the functionality of a 32-bit operating system, and to dramatically enhance the user interface. One of the key features of Windows 95 is backwards-compatibility with Windows 3.1x.

Windows NT is a true 32-bit operating system with a completely different architecture to that of Windows 3.1x and Windows 95. Windows NT is an operating system for users who need stability, security, and the ability to utilize various types of processors such as Alpha, MIPS, and Pentium. However, Windows NT does not provide complete backwards-compatibility with 16-bit programs.

Windows 98, the newest Windows operating system, provides an upgraded feature set to Windows 95. It is fully compatible with 16-bit programs, and, as such, is an alternative to Windows NT as a 32-bit operating system.
