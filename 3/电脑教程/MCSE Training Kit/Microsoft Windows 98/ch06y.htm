Registry Editor

You should use a Control Panel icon, such as Add New Hardware, to make modifications to system settings. However, if it is essential to modify the registry directly, you can use a tool called Registry Editor (Regedit.exe), located in the C:\Windows folder, to view and modify current registry settings.

Viewing and Altering Registry Values

Registry Editor (see Figure 6.14) is divided into two panes. Keys and subkeys are listed in the left pane, and value entries are listed in the right pane. In the left pane, there is a plus sign (+) beside a key that has additional subkeys. The plus sign (+) changes to a minus sign (-) when the list is expanded to show the subordinate entries.

When you click a key or subkey, the associated value entry appears in the right pane, and the entire path of the selected registry key appears in the Registry Editor status bar.

Figure 6.14 The Registry Editor

If you want to alter a registry value:

  1. Double-click the value name in the right pane of Registry Editor.

  2. For ab data types, enter your changes in the Edit String dialog box; for binary data types, enter your changes in the Edit Binary Value dialog box. Then click OK.

    New keys or values are created with the Edit/New commands.

Caution

Microsoft does not recommend or support making changes directly to the registry when an alternative exists. Most changes to the registry should be made through Control Panel. Unlike the Windows NT Registry Editor, there is no Read-Only Mode option in Windows 98. Changes to the Windows 98 registry are immediate and may be irrevocable. If you must make a change to the registry, Microsoft recommends backing up your current registry first.

Registry Editor Commands

Several useful Registry Editor commands facilitate work with the registry. The Registry Editor contains a search function. From the Edit menu, click Find and type the string you wish to locate. You can search keys, value names, and value data for the desired string.

The Registry menu has the commands: Export Registry File and Import Registry File. Use the export command to save total or partial registry data in a *.reg text file, and use the import command to restore it on the same or on a different computer.
