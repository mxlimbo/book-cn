Summary

The following information summarizes the key points in this chapter:

Windows 98 Overview

  • Windows 98, the newest Windows operating system, provides an upgraded feature set to Windows 95.

  • Windows 98 is a true 32-bit operating system that features full support for 16-bit applications. Although Windows NT also will support most 16-bit applications, its architecture does not support full backwards-compatibility.

Windows 98 Features

  • Improvements to Windows 98 include improved setup, file system support and reliability, integrated user interface, and reduced cost of ownership.

  • Windows 98 provides support for new hardware standards, including OnNow, USB, IEEE 1394, ACPI, DVD, WDM, and multiple display devices.

  • Windows 98 introduces improved support for networking and communications features such as NDIS 5.0, Windows Sockets 2.0, WBEM, PPTP, and VPN.

Selecting Windows 98 as a Client Operating System

  • Windows 98 is designed to support a wide range of hardware and software, while Windows NT Workstation 4.0 is designed to be the most powerful 32-bit desktop operating system. For many sites, a mix of both systems will provide the best business solution.

  • Because Windows NT Workstation is more reliable and secure, some business customers may be best served by deploying Windows NT Workstation.
