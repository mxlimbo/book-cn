Chapter 3

Windows 98 File System Support

About This Chapter

Hard disk management can be one of the most important aspects of user support. Your knowledgeable and regular use of Windows 98 disk management tools can enhance computer performance and reliability.

This chapter discusses how to partition a hard disk, as well as the capabilities and limitations of FAT16 and FAT32 file systems. The chapter also presents information to help you enable FAT32 and convert FAT16 partitions to FAT32 partitions. Finally, the chapter describes the procedure to implement disk compressions.
