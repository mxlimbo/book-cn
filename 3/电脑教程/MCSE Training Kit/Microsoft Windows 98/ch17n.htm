Lesson Summary

The Internet is composed of high-speed data lines connecting regional commercial networks. An intranet is an internal corporate or organizational network that uses Internet technologies to let employees browse and share electronic information. An intranet is a private version of the Internet's World Wide Web, but it's available only to people inside the organization.

After you subscribe to a Web site, Windows 98 can notify you when a Web page has been updated. You can configure your subscriptions to download Web pages for offline reading.

A channel is a Web site that makes it possible to receive up-to-date Web content. Once you subscribe to a channel, its content is updated periodically by the provider and is sent directly to your desktop.

Microsoft NetShow is an IE component that streams multimedia content over networks. NetShow provides a continuous data flow with little wait time before content playback.
