Five–Phase Setup in MS-DOS

Several installation steps are the same, whether you run Windows 98 Setup from MS-DOS or from Windows 95. They include:

  • Creating the Emergency Startup Disk (ESD), part of Phase 2—Collecting information about your computer

  • Phase 3—Copying Windows 98 files to your computer

  • Phase 4—Restarting your computer

However, when you run Setup from MS-DOS, the following additional steps take place:

  • During Phase 1—Preparing to run Windows 98 Setup:

    Real-mode ScanDisk performs a routine check on your system.

    The Mini.cab file is expanded, and its files are copied to the C:\Wininst0.400 folder. This file contains the run-time version of Windows required for Setup.

  • During Phase 2—Collecting Information About Your Computer:

    You choose the Install Directory.

    You select the Setup type (Typical, Compact, Portable, or Custom) and, depending on the Setup type, which optional components are to be installed.

    You specify user and computer information.

  • During Phase 5—Setting Up Hardware and Finalizing Settings:

    Hardware detection of non-Plug and Play devices is performed after Plug and Play enumeration and before Setup restarts your computer for the second time during the finalizing settings portion of this phase.

Note

For a more detailed comparison of setup from Windows 95 and from MS-DOS, refer to the Setup Summary in Appendix D, "Setup Summary."
