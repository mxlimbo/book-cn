Lesson 4: Sharing Resources with NetWare Users

Windows 98 users can share resources with NetWare servers and clients in the same way as in a Microsoft network. The configuration process, however, differs for NetWare, and the access control level is limited to user-level access. This lesson explores both of these considerations.

After this lesson, you will be able to:

  • Install and configure File and printer sharing for NetWare Networks.

  • Share resources on a NetWare network.

  • Describe common problems and solutions in Windows 98 interoperability NetWare.

Estimated lesson time: 15 minutes
