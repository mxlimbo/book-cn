Creating a Dial-Up Networking Connection

After installing Dial-Up Networking, installing and configuring your modem, and setting up any needed dialing locations, you are ready to create a dial-up networking connection.

Figure 18.6 The Make New Connection Wizard

If you want to create a new dial-up networking connection, perform the following procedure:

  1. In My Computer, double-click the Dial-Up Networking folder.

  2. Double-click the Make New Connection icon. The Make New Connection Wizard appears (see Figure 18.6) and prompts you for the following:

    • A name for the connection

    • A modem with which to connect

    • The phone number to dial

After configuring the new connection, you can start it by double-clicking the connection icon in the Dial-Up Networking folder.

Using AutoDial

Dial-Up Networking also provides an AutoDial feature. If AutoDial is enabled, Windows 98 starts Dial-Up Networking when you attempt to connect to a network resource that is not available on the physical network. A message appears, asking if you would like to use Dial-Up Networking to connect to the resource. Some examples of actions that start an AutoDial connection include:

  • Trying to use a persistent mapped network drive connection.

  • Double-clicking a shortcut to a remote file or printer.

In the Dial-Up Networking folder, you can disable the prompt that asks you if you want to use Dial-Up Networking when you are attempting to connect to a network resource.

If you want to disable the AutoDial prompt, perform the following procedure:

  1. Open the Dial-Up Networking folder.

  2. On the Connections menu, click Settings.

  3. On the General tab, select Don't prompt to use Dial-Up Networking.
