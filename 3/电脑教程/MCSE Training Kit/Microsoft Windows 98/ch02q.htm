Lesson 3: Installing from MS-DOS

There are situations in which Windows 98 must be installed from MS-DOS. Installing Windows 98 from MS-DOS follows the same five phases as upgrading from Windows 95. However, some additional steps are added.

This lesson describes when it is necessary to install Windows 98 from MS-DOS, as well as the five–phase Setup process.

After this lesson, you will be able to:

  • List appropriate instances in which to install Windows 98 from MS-DOS.

  • List the additional steps required to install Windows 98 from MS-DOS.

Estimated lesson time: 15 minutes
