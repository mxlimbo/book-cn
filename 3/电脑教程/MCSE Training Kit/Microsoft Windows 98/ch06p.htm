Configuring Accessibility Options

Computer operation can be especially challenging for users with vision, hearing, or other physical disabilities. Support personnel should be familiar with system settings that provide better access for users with special needs.

The Accessibility Options icon in Control Panel is one method for customizing some sound, keyboard, mouse, and display settings. Use this method if you are familiar with the available options.

Important

If Accessibility Options was not included in your Windows 98 installation, add it through the Windows Setup tab of Add/Remove Programs.

The Accessibility Wizard automates configuration through dialog boxes that explain options that are available through several Control Panel icons. With this wizard, you can save the configuration in a file and then transfer it to other computers.

The Accessibility Wizard is located in the Start menu (Programs, Accessories, Accessibility).

Microsoft Magnifier

Microsoft Magnifier is a display utility that makes the screen more easily readable for users with vision limitations. The magnifier creates a separate window that displays a magnified image of a portion of your screen. Each user can move and size this window to individual preferences.

Microsoft Magnifier is located in the Start menu (Programs, Accessories, Accessibility), and can also be started through the Accessibility Wizard (see Figure 6.8).
