Lesson 5: Multiple Display Support

Windows 98 includes support for using multiple display adapters and monitors with a single computer. This lesson describes the uses for multiple displays, configuration of multiple display support, and troubleshooting suggestions for problems that might occur.

After this lesson, you will be able to:

  • Describe uses for multiple displays.

  • Configure multiple display devices.

  • Describe multiple display support issues and their solutions.

Estimated lesson time: 20 minutes

Before you continue with the lesson, run the Chap4b.exe animation located in the Animations folder on the Supplemental Course Materials CD-ROM that accompanies this book. This animation provides an overview of Windows 98 support for multiple display.
