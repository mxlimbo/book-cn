Procedure for Uninstalling Windows 98

If you meet the uninstalling requirements, you can uninstall Windows 98 and restore your Windows 95 or MS-DOS operating system.

If you want to uninstall Windows 98, perform the following procedure:

  1. In Control Panel, click Add/Remove Programs.

  2. Click the Install/Uninstall tab, select Uninstall Windows 98, and then click Add/Remove.

    The Windows 98 Uninstall dialog box appears, stating that you are about to remove Windows 98 from your computer and restore Windows 95.

    Click Yes to continue with the uninstall procedure.

    A dialog box appears informing you that your disk(s) will be checked for errors.

  3. Click Yes.

    ScanDisk runs on all nonremovable hard disks, and another dialog box appears asking if you want to continue.

  4. Click Yes.

    The computer restarts in MS-DOS mode and the Uninstal.exe program runs automatically. This program is located in the C:\Windows\Command directory. The MS-DOS program restores the Windows 95 files and settings contained in Winundo.dat and then prompts you to restart the computer.

  5. Once the computer restarts, Windows 95 will be restored and usable.

Important

If you have installed any programs after setting up Windows 98, you will have to reinstall them.
