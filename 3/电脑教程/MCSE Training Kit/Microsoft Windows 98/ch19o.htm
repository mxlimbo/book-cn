Using Troubleshooters

Windows 98 Help includes Troubleshooting tools that can help you diagnose and solve problems using a question and answer format. Different responses to the questions will lead you to different troubleshooting solutions.

The Troubleshooters also include help for these network-related topics:

  • Modem

  • Print

  • The Microsoft Network

  • Dial-Up Networking

  • Direct Cable Connection

If you want to use a Windows 98 Troubleshooter, perform the following procedure:

  1. On the Start menu, click Help.

  2. On the Contents tab, click Troubleshooting, and then click Windows 98 Troubleshooters.

  3. Select the category that corresponds to the problem you have encountered.

  4. In the right pane of the Help window, Troubleshooters prompts you for the type of problem you have. Answer each question, and follow the suggestions.

Note

If the topic you want is not listed under Troubleshooters in the Contents listing, look in the Help index. Some troubleshooting topics do not appear in the Troubleshooters listing.
