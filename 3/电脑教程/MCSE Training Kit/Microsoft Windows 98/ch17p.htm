Content Ratings

The Recreational Software Advisory Council (RSAC) maintains a rating system of levels of language, nudity, sex, and violence on Web sites. By using these ratings, you can prevent Internet Explorer users from accessing sites that rate unacceptably to your organization.

Figure 17.4 Content Advisor for Internet Explorer

If you want to enable content ratings, perform the following procedure:

  1. From Control Panel, double-click Internet.

  2. In the Internet Properties dialog box, select the Content tab.

  3. In the Content Advisor box, click Enable.

  4. Type in a password and click OK. The supervisor password must be used to disable ratings, or to override a site restriction.

  5. On the Ratings tab (see Figure 17.4), click the category for which you want to set a restriction.

  6. Move the slider to set the rating for that category.

  7. Continue until you have set ratings for all the categories you want, then click OK.

    A dialog box appears to tell you that Content Advisor has been installed.

  8. Click OK.

  9. Click OK to close Internet Properties.

Note

Recently viewed sites will continue to be available until the current open Internet Explorer window is closed.

Configuring Ratings Settings

There are four categories for which you can configure ratings: language, nudity, sex, and violence. You can configure each category individually for five rating levels, Level 0 through Level 4. Complete descriptions of each level appear in the Content Advisor dialog box. In addition, clicking the More Info button will display the Internet page for the rating service.

Note

Because not all sites are rated, you can also use the General tab of the Content Advisor dialog box to restrict access to unrated sites. You can also use the supervisor password to allow users access to unrated sites.

Rating Systems and Bureaus

By default, Internet Explorer uses the RSAC rating system. You can use another rating system by copying the .rat file from the rating system into the C:\Windows\System folder. You can choose to use the additional rating system by adding it to the list of rating systems.

If you want to use the additional rating system, perform the following procedure:

  1. In Control Panel, double-click Internet.

  2. In the Internet Properties dialog box, select the Content tab.

  3. In the Content Advisor box, click Settings.

  4. Type in a password and click OK.

  5. From the Content Advisor dialog box, select the Advanced tab.

  6. Click Rating Systems.

  7. Click Add and browse the system folder to locate the new rating file.

  8. Select the file and then click Open.

  9. Click OK three times to accept the changes.

Different rating systems use different methods to determine a given Web site's rating. The rating label information may be within the HTML document as a tag, within the HTTP header, or it may be obtained from an independent Web server, or ratings bureau, that distributes rating information. If a rating system uses a ratings bureau, you can add it to the list of bureaus in the Content Advisor dialog box.

If you want to use a ratings bureau, perform the following procedure:

  1. From the Content Advisor dialog box, select the Advanced tab.

  2. In the Ratings bureau drop-down box, type the name of the bureau you want to use.

  3. Click OK.

Storing Rating Information

A file called Ratings.pol, located in the C:\Windows\System folder, stores rating information.

Note

Despite its extension, Ratings.pol cannot be viewed or edited in System Policy Editor.

Your supervisor password is encrypted and stored in the following registry location: 

HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Policies\Ratings\Key

If you forget your supervisor password, you can run Registry Editor and delete the Key value above. The next time Windows 98 requires the supervisor password, the system will prompt you for a new one. You can remove all ratings by deleting the supervisor password and deleting Ratings.pol.
