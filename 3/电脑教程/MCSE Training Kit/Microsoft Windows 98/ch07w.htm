Practice: Configuring an Active Desktop Item

In this practice, you will add, move, and resize an Active Desktop item.

    To add an Active Desktop item

    1. Right-click an empty area of the desktop, point to Active Desktop, and then click View as Web Page.

      The Active Desktop is now enabled, and the Internet Explorer Channel Bar appears.

    2. Right-click an empty area of the desktop, and then click Properties.

      The Display Properties dialog box appears.

    3. Click the Web tab, and then click New.

      The New Active Desktop Item dialog box appears. It tells you that you can visit the Active Desktop gallery on the Microsoft Web site to preview and install Active Desktop items.

    4. Click No.

      A New Active Desktop Item dialog box appears.

    5. In the Location box, type c:\windows\readme.htm and then click OK.

    6. Click OK to close the Display Properties dialog box.

      The Readme.htm file appears as an Active Desktop item on the desktop.

    To move an Active Desktop item

    1. Move the pointer to the top edge of the Active Desktop item.

      The title bar of the Active Desktop appears.

    2. Click and drag the title bar to the top center of your desktop.

      The Active Desktop item is moved.

    To resize an Active Desktop item

    1. Position the pointer over the edge of the Active Desktop item.

      The pointer changes to a double-headed pointer.

    2. Drag the border of the Active Desktop item to change the size.
