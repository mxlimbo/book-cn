Lesson 1: Interoperating with NetWare Overview

Integrated networking is a key feature of Windows 98. This lesson provides an overview of Windows 98 interoperability with Novell Netware, including the capabilities of Windows 98 computers running Microsoft Client for NetWare Networks on NetWare networks. This lesson also describes the procedures for installing and configuring the protocols necessary to connect to Novell NetWare networks.

After this lesson, you will be able to:

  • Describe Windows 98 interoperability with Novell NetWare.

  • List the capabilities of Windows 98 clients running on Novell NetWare networks.

  • List the requirement to interoperate with Novell NetWare.

  • Install and configure an IPX/SPX-compatible protocol.

Estimated lesson time: 10 minutes

Windows 98 supports interoperation with NetWare version 3.x. These versions of NetWare use a database called the bindery to organize network objects such as users, groups, and print queues. Each server has a bindery, which it does not share with the other servers. Windows 98 also supports NetWare 4.x servers that emulate the bindery used in earlier versions of NetWare.

In addition to bindery support, Windows 98 includes support for NetWare Directory Services (NDS), used in NetWare versions 4.x. These versions of NetWare implement a network-wide, hierarchical directory structure for network objects called a tree. All NetWare 4.x servers on a network can share the same NDS tree.

A Windows 98 computer running Microsoft Client for NetWare Networks on a NetWare network can:

  • Log on to a Novell NetWare bindery-based server or Netware Directory Services (NDS) tree with Microsoft Service for NetWare Directory Services installed.

  • Connect to file and print resources on NetWare servers.

  • Run most NetWare 3.x utilities (with some exceptions, like nwuser and rprinter).

  • Share resources with NetWare clients.

  • Download roaming user profiles and system policies from a NetWare server. Roaming profiles are for users who log on to more than one computer.

To interoperate with Novell NetWare, you must install a protocol, a client, and, if using NDS, an additional service.

  • For NetWare 3.x and 4.x systems that are bindery-based, you install the internetwork packet exchange/sequenced packet exchange (IPX/SPX)-compatible Protocol and Client for NetWare Networks.

  • For NetWare 4.x NDS systems, you install the IPX/SPX-compatible Protocol, Client for NetWare Networks, and Service for NetWare Directory Services.
