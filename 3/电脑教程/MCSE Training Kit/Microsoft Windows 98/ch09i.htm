Lesson 2: Performing Maintenance Tasks

Regular maintenance of your computer system improves performance and increases efficiency. This lesson discusses several tools provided by Windows 98 to make accomplishing and scheduling your maintenance tasks easier.

Both the Windows-based and MS-DOS based versions of ScanDisk have been upgraded to provide disk-checking and repair capability for both FAT16 and FAT32 file systems. Disk Defragmenter now includes an application optimization option that decreases the time it takes to start a program. The Disk Cleanup wizard, a new tool in Windows 98, removes temporary and unneeded files.

The Maintenance Wizard is also a new tool, and it provides a quick and simple method for running ScanDisk, Disk Defragmenter, and Disk Cleanup unattended on a regular schedule. Using the Scheduled Tasks wizard, you can schedule any application to run at a specific time.

After this lesson, you will be able to:

  • Use ScanDisk to troubleshoot your hard disk.

  • Use Disk Defragmenter to optimize your hard disk.

  • Describe the function of Task Monitor.

  • Run Disk Cleanup to maintain system performance.

  • Use Scheduled Tasks to run any application at specified times.

  • Use Maintenance Wizard to run maintenance utilities at specified times.

Estimated lesson time: 30 minutes
