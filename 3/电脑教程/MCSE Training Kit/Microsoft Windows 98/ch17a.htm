Chapter 17

Using Windows 98 and the Internet

About this Chapter

This chapter discusses using Windows 98 with the Internet and intranets. It presents information to help you learn how to browse the Internet and intranets efficiently, how to control access to Internet content, and how to set up security to protect your network from unauthorized access. It also describes how to set up and manage Personal Web Server so you can create and maintain Web sites and share resources on an intranet. Finally, this chapter discusses how to use Microsoft NetMeeting and Outlook Express to communicate on the Internet and intranets.
