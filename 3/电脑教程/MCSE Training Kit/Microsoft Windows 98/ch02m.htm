Phase 3—Copying Windows 98 Files to Your Computer

After collecting the necessary information about your computer, Setup will prompt you to start copying the Windows 98 files to your computer, as seen in Figure 2.5.

Figure 2.5 Copying Windows 98 Files during Setup

During the file-copying phase, Windows 98 Setup does not require your input.

Caution

Do not interrupt file-copying. If Setup is interrupted during the file-copying phase, Windows may not run when you restart your computer. If you are upgrading Windows 95, Windows may not start because there will be a mixture of Windows 98 and Windows 95 files on your computer. If you are installing to a new directory or newly formatted hard disk, Windows 98 will not have all of the necessary files to start. Therefore, you will need to restart Setup.
