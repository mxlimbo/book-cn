Developing a Multiple Installation Plan

You can perform a pilot installation on a small group. By performing this pilot test and then projecting the results of the test to the entire network, you can determine the time and resources required for the network to migrate to Windows 98.

The following tasks are necessary for an effective multiple installation plan that ensures rapid installations with minimal disruptions.

  • Determine your installation goals—specifically the number of computers on which to install Windows 98 and the expected time required to complete the installations.

  • Budget the resources for personnel and tools required to meet your goals.

  • Obtain approval for resources and the multiple installation process.

  • Update the company's hardware and software standards lists.

  • Update the company's policies and practices manuals or guidelines for use of the PC and network.

  • Notify the users that the installation will take place.

  • Provide training for new users.

  • Document the specific configuration and use of each computer. Document any problems or deficiencies with the system that require further attention.
