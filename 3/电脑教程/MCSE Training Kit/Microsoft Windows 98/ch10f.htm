Practice: Using Registry Checker

In this practice, you will back up the registry, view the contents of a backup file, and recover from a corrupted registry.

Exercise 1
Backing Up the Registry

In this exercise, you will use the Registry Checker to create a backup of a working registry.

    To use the Registry Checker to create a copy of the current registry settings

    1. Click Start, and then click Run.

      The Run dialog box appears.

    2. Type Scanregw.exe and then click OK.

      The Windows Registry Checker dialog box appears indicating the System registry is being scanned.

      The Registry Scan Results dialog box appears indicating that no errors were found and that the registry has already been backed up today.

    3. Click Yes to create another backup.

      The Windows Registry Checker dialog box appears, and the backup begins.

      When the backup is complete, the Windows Registry Checker dialog box appears, informing you that the backup is complete.

    4. Click OK.

Exercise 3
Recovering from a Corrupted Registry

In this exercise, you will use the Registry Checker to restore a known good copy of your registry files.

    To create a corrupted registry

    1. Click Start, and then click Run.

      The Run dialog box appears.

    2. Click Browse, select the Reg.bat file located in Labfiles\Chapter10 on your Supplemental Course Materials CD-ROM, and then click OK.

      This batch file corrupts the registry by replacing one of the registry files with an incompatible file.

    3. When the batch file finishes running, restart your computer.

      When the computer restarts, the Microsoft Windows 98 Startup Menu appears.

    To repair a corrupted registry

    1. Note the warning in the center of the screen.

      Answer

    2. Press ENTER to boot to a command prompt.

    3. What switches are available for ScanReg? (Type ScanReg /? at the command prompt to view the switches.)

      Answer

Note

If you choose to run ScanReg without any switches (as prompted earlier), it will scan the registry, determine that it has been corrupted, and automatically restore a good registry. In the step below, you will use the /restore switch so you to can choose which backup to use.

    1. At the command prompt, type ScanReg /restore and then press ENTER.

      The Microsoft Registry Checker utility checks your system files for corruption. When the check is complete, a list of system backups is displayed. The list is sorted with the most recent backup at the top.

Caution

When choosing a backup *.cab file to restore from, it is best to choose one that is marked as having been started, unless you know for certain that one of the other backup files is reliable.

    1. Select the most current backup, and then press ENTER.

      Your current system files are backed up, and the backup file you selected is restored.

      The You've reverted to a Good Registry dialog box appears.

    2. Press ENTER to restart the computer.

    To examine the results of the registry restore
