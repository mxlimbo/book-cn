Lesson Summary

Using virtual memory, Windows 98 keeps some information in RAM while it swaps, or temporarily moves, other information to a virtual memory swap file. The total amount of virtual memory is the sum of your computer's RAM plus the unused disk space. You configure virtual memory with the System icon in Control Panel.

You use the File Systems Properties sheet to set the hard disk cache and the CD-ROM File System cache, and to configure floppy and removable disk drives.

You use the Accessibility Options icon in Control Panel to configure the computer for users with vision, hearing, and mobility needs.
