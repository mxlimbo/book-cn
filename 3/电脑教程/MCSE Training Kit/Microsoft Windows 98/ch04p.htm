Adding a USB Device

USB is a Plug and Play bus, which means that you can add or remove devices from the bus while the computer is turned on. USB identifies and enables the device automatically. Because USB is an external bus, users do not have to open the computer to add a device to the bus.

When a device such as a new keyboard is added to the bus, the following events occur, all of which are transparent to the user.

  1. The hub notifies the host controller that a new device exists.

  2. The device is assigned a unique bus address.

  3. The operating system loads the required device driver.

  4. The device is configured.

  5. The device driver is initialized.

  6. The device is now ready for use.

From the user's perspective, after the keyboard is plugged into the bus, the keyboard works.
