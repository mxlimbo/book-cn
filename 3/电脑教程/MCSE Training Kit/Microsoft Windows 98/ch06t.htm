Installing and Removing Applications

The Install/Uninstall tab of Add/Remove Programs Properties is a central location for adding software to the computer and for removing unneeded applications (see Figure 6.10).

Figure 6.10 The Install/Uninstall tab of the Add/Remove Programs Properties sheet

Adding Programs

When you click the Install button on this tab, Windows 98 does most of the work for you. You need only to insert the floppy disk or compact disc that contains the program. You do not have to indicate which drive; Windows 98 searches for a file called Install.exe, first in the floppy drives, then in the CD-ROM drive. You can also install programs from a network drive.

If Install.exe is not found, Windows 98 searches for Setup.exe. If neither program is found, Windows 98 prompts you for the location of the Setup program.

Removing Programs

With the Add/Remove button, you can remove any application that appears on the list above the button. Only programs that have been written to take advantage of this feature can be removed through this window.

Clicking the Add/Remove button runs the original setup program for the selected application. You can then add or remove additional components of that application, or the application itself, through the application's Setup program.

When you remove programs with Add/Remove, their entries in the registry are marked as deleted. Microsoft Registry Checker scans and backs up the registry once a day upon startup. During this scan, if Registry Checker determines that the registry contains 500K or more of keys marked as deleted, it will optimize the registry upon next boot. Optimization reclaims wasted registry space, reserving 4K worth of data to preserve the registry key's header.
