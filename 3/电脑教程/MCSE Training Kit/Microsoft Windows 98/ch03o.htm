Lesson 3: Long File Names in Windows 98

Long file name (LFN) support in Windows 98 eliminates the 8.3 file naming convention imposed by MS-DOS. For each long file name, Windows 98 also generates an MS-DOS 8.3 file name for the use of applications that cannot read long file names.

This lesson discusses how Windows 98 generates long file names and MS-DOS file names. It also describes common problems with file names and their solutions.

After this lesson, you will be able to:

  • Describe how Windows 98 generates long file names.

  • Describe how Windows 98 generates MS-DOS file names.

  • Describe common troubleshooting issues with long file names.

Estimated lesson time: 15 minutes
