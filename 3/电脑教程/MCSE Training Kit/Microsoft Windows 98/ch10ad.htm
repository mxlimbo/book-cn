Practice: Using Dr. Watson

In this practice, you will start and configure Dr. Watson. You will also create and examine a Dr. Watson log file.

Exercise 1
Starting and Configuring Dr. Watson

In this exercise, you will run and configure Dr. Watson.

    To start Dr. Watson

    1. On the Start menu, click Run.

      The Run dialog box appears.

    2. Type drwatson.exe and then click OK.

      The Dr. Watson icon appears on the right side of the taskbar.

    To configure Dr. Watson

    1. Right-click the Dr. Watson icon, and then select Options.

      The Dr. Watson Options dialog box appears.

    2. Under View, select Open new windows in Advanced view, and then click OK.

Exercise 3
Examining a Log File Created by a GPF

In this exercise, you will use Badapp16.exe to generate a GPF. You then examine the log file created by Dr. Watson and compare it to the log file you created in Exercise 2.

    To generate a GPF

    1. Run the Badapp16.exe file located in the Labfiles\Chapter10 folder of your Supplemental Course Materials CD-ROM, and then click OK.

      The BadApp window appears.

    2. On the Action menu, click GP-Fault.

      BadApp generates a GPF, and a dialog box appears indicating that a GPF has occurred. Notice the progress bar at the bottom of the dialog box. This is generated by Dr. Watson and indicates that Dr. Watson is generating a log file.
