Lesson 1: Setting Up Dial-Up Networking

This lesson discusses Dial-Up Networking. With Dial-Up Networking you can connect to a remote computer or network through a modem or ISDN adapter, rather than through a network interface card.

To set up Dial-Up Networking you need one or more compatible modems or ISDN devices and sufficient hard disk space to install the Dial-Up Networking software.

After installing and configuring your hardware devices (ISDN, modems, etc.) you can use the Dialing Properties utility to define different locations from which you dial. For example, you can create two different locations (home and office) and enter information for each location. Then, whenever you create a new dialing location, Dialing Properties automatically adjusts the telephone number based on the location from which you are dialing.

After this lesson, you will be able to:

  • Install Dial-Up Networking.

  • Install and configure modems and dial-up adapters.

  • Install and configure an ISDN adapter.

  • Configure a dialing location.

  • Create a Dial-Up Networking connection.

Estimated lesson time: 60 minutes
