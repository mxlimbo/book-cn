第一节 FLASH4 用途及新功能介绍

作 者 : Kooler

　　 FLASH 是 MACROMEDIA 公司专门为网络设计的一个交互性矢量动画设计软件，网站设计者可以使用 FLASH 随心所欲地为网站设计各种动态 LOGO ，动画，导航条以及全屏动画，还可以带有动感音乐，：）完全具备多媒体的各项功能。

　　你还记得前几年上网的感觉吗？静态的页面，慢吞吞的传输速度，一页简单的网页 都要下载几分钟，更谈不上动态的网页下载了。

　　随着网络技术的不断发展，网络带宽的不断提高，人们对静态的网页已经不满足了，在 1995 年， SUN 公司开发了 JAVA 程序，采用既定的 APPLET 程序能够在网页上设计出动态的多媒体小程序，并在短时间内风行全球。

　　但是 JAVA 的专业性及设计上的局限性让网页设计者感觉到不能够充分反映自己的设计意愿，总有点美中不足，于是乎 Macromedia 公司 -- 多媒体制作的大哥也在 98 年的下半年隆重推出 FLASH3.0 ，并在前段时间推出功能更强的 FLASH4.0 。记得我在 98 年 11 月的时候下载试用后，当时的感觉就是 -- 以后网络的世界就是多媒体的世界，而 FLASH 就是网上多媒体制作的利器。为什么这么说呢？你看看 MACROMEDIA 公司对 FLASH4 的介绍吧：

　　 1 、矢量动画

　　 FLASH 做动画是矢量的，不论你怎么放大，图片质量不会改变。基本上不用考虑到象 GIF 、 JPG 等位图格式的压缩。另外 FLASH 还有将位图转换成矢量图的功能，：）酷吧。。。

　　 2 、支持 MP3 音乐压缩格式

　　原来 FLASH3 只支持 WAV 格式，现在 FLASH4 已支持 MP3 压缩格式，用 FLASH4 做出来的带 MUSIC 的文件更小了。：）酷吧。。

　　 3 、 BUTTON

　　 FLASH 提供灵活的 BUTTON 制作功能，根据 UP ， OVER ， DOWN ， HIT 四个状态的灵活设置，可以制作出极为动感的 BUTTON ，在各状态下还可以加 MOVIE 哦，另外 MOVIE CLIP 还支持 HTML 、 FS COMMAND ，可以任意的定义它的 ACTION ，这就是应该就是所谓的交互性吧。。：）酷！

　　 4 、 MOVIE CLIP

　　 MOVIE CLIP 应该说是 FLASH 电影里的一个元素，不过实际上它也算是一个小电影，可以随意的加入 MOVIE 里面，呵呵。。大电影里带小电影，酷吧。。

　　 5 、 LAYER

　　用过 PHOTOSHOP 的都知道 LAYER 功能的强大，几乎可以让人随心所欲制作各种你能够想得出来的图片，同理， FLASH 是动画制作软件，用它你也可以随心所欲的制作你能够想的出来的动画。。。嘿。。酷！

　　 6 、 TIMELINE

　　看名字就知道是个控制时间的东东，：） TIMELINE 是以 FRAME 为单元的，每个 FRAME 还可以设置各种 ACTION ， LABEL 以及 FSCOMMAND 。你在设计的时候别忽视了这个东东哦，在动画制作上对时间的灵活运用可有得研究着呢！

　　 7 、 ACTION 和 FS COMMAND

　　 FLASH 的交互性的大部分设置就在 ACTION 和 FS COMMAND 里，通过对 ACTION 和 FS COMMAND 的设置，你可以随意的设置各事件发生的效果，还有对变量及函数的设置，你可以为所欲为。这里可深奥着呢…… :)
