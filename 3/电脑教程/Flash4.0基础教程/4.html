第四节 菜单详解 (1)

作 者 : Kooler

　　今天讲菜单，为了避免不同汉化软件所译的菜单名称不同，我就以英文版的菜单来讲解。

　　 FLASH4 的菜单共分 9 项（呵呵，废话），那我就开始依次讲咯。

　　 FILE（文件） 菜单包含以下命令：

　　 NEW ：建一个新电影。
　　OPEN ：打开一个存在的电影。
　　OPEN AS LIBRARY ：将已经存在的 .FLA 文件中的 LIBRARY 中的各元素导入当前的 LIBRARY 中。
　　CLOSE ：关闭当前 MOVIE 。

　　“ IMPORT ”、“ EXPORT MOVIE ”、“ EXPORT IMAGE ”这三条命令可以实现 FLASH 和外部应用程序之间文件的交换，输入输出各种格式的图象、动画以及声音文件。

publish setting （此处为重点，要考试的啊，请仔细研读，咳……）：这个命令是在你做完 MOVIE 后，在向 WEB 发布前的设定命令，所弹出的对话框中有三个子选单：“ FORMATS ”、“ FLASH ”、“ HTML ”。

　　（ 1 ）“ FROMATS ”是设定你在 WEB 发布 MOVIE 的格式选项。一般来说“ FLASH ”和“ HTML ”两项是必选的，而你另外加选其他图片格式（ JPG ， GIF ， PNG ）时， FLASH 就会自动生成一段 HTML 代码，这些代码使浏览者没有安装 FLASH PLAYER 时，可以看到相应格式的图片。

　　对于 Windows 和 Macintosh 自动播放程序选项没有设置项，所以在出版设置对话框中也就没有自己的标签。 选取“ Use Default Name ”选项时，则输出的文件名使用当前默认的文件名，你也可以自己定义一个不同的文件名，取消“ Use Default Name ”再在相应的位置改名就 OK 了。

　　（ 2 ）在“ FLASH ”中可以对在出版过程中制作的 swf MOVIE 进行选项设置。在输出使你可以有多个选项选择，包括图像及声音压缩设置，还可以保护你的电影不被其他人输入利用（呵呵，版权所有，盗版必究！）

　　在“ FLASH ”中有以下几个选项：

　　 Load Order （装载顺序）：这个选项只影响第 1 帧，包含“ TOP DOWN ”和“ BOTTOM UP ”两个选项，由于网络的速度和电影文件大小的问题，浏览者不可能将电影在短时间内全部下载观看，采用这个选项可以先在电影中提取一部分进行播放，使用这个选项确定 MOVIE 开始下载时先后播放的内容。

　　Generate Size Report ：这个选项可以生成一个文本文件，我们可以用作缩小 MOVIE 文件尺寸的指南。这个报告显示在最终的文件中 MOVIE 不同部分的字节数。报告的名字和输出 MOVIE 的名字相同，但是文件的扩展名为 .txt 。

　　Protect From Import ：保护你的电影文件，防止其他人从网络上下载 swf 文件后再输入 Flash 中使用你的 MOVIE 。

　　mit Trace Actions ：令 Flash 忽略当前 MOVIE 中的 Trace 指令。 Trace 指令会让 Flash 打开一个输出窗口，显示某些内容。

　　JPEG Quality ：可以调整你的 MOVIE 文件中位图的质量，一般来说，图像质量低，文件 SIZE 就小；图像质量高，文件 SIZE 就大（注意设置最好不要低于 60% ，不然图像会有失真的现象）；如果输出图像中没有位图，本选项就没有任何作用。 Audio Stream （流声音） and Audio Event （事件声音）：这两个选项分别确定输出流声音和事件声音的取样速度和压缩比。使用这些设置只有当你没有在声音属性对话框中对个别声音进行设置的情况下，或者当你打开 Override Sound Settings 选项的情况下才能应用。这时的设置会和你在声音属性对话框中对个别声音的设置一样的。如果 MOVIE 中没有流声音，那么这个设置就没有什么用。 Override Sound Settings 让本对话框中的设置覆盖并取代你在声音属性对话框中为个别声音进行的设置。 Version 指定输出 MOVIE 的版本号， Flash 4 新增功能在用早期版本的 Flash 出版的 MOVIE 中是不能工作的。
