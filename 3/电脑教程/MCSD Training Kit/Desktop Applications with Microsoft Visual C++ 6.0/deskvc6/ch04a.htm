Chapter 4

Creating the User Interface

About This Chapter

This chapter explores some of the elements of a typical Microsoft Windows application user interface. You will learn how to edit application menus and toolbars, and how to add code to handle user selections. You will learn how to update the interface to communicate information to the user. You will also learn how dialog boxes are used in an application, and how to use the dialog editor to create a dialog box template.

Before You Begin

Before you start this chapter, you should have read Chapters 2 and 3, and completed the exercises in the text.
