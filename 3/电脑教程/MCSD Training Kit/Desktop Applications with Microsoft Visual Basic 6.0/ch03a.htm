Chapter 3

Validating and Processing User Input

About This Chapter

In this chapter, you will learn to respond to user input. The technique you choose will be based on the type of interaction you want to provide. You can validate data at the form level, which is a centralized style of validation. The alternative is field-level validation, which validates data based on each data element within a user interface.
