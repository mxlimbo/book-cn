Chapter 11

Debugging and Error Handling

About This Chapter

In this chapter, you will learn how to avoid errors in your applications by implementing error handlers. You will also learn how to debug your applications to fix logic and run-time errors. These concepts will be presented for development of both front-end applications that run in the presentation layer of an n-tier application and middle-tier components.
