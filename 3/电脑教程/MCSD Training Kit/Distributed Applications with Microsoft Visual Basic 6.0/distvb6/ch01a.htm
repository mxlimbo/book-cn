Chapter 1

Application Design Concepts

About This Chapter

In this chapter, you will learn about the process of designing a distributed application in Microsoft Visual Basic using Microsoft Solution Framework design concepts. You will study the issues that you should consider in planning your design. In addition, you will learn about source code control using the Microsoft Visual Source Safe library management system and how to connect to a database using the Microsoft Visual Data Tools.

Before You Begin

To complete the lessons in this chapter, you must have:

  • Installed Visual Basic 6.0 Enterprise Edition.

  • Installed Microsoft SQL Server 7.0 Enterprise Edition.

  • Installed the Pubs sample database included with SQL Server 7.0.
