格廷根大学
Georg-August-Universitat Gottingen

通讯地址：wilhelmsplatz 1，3400 Gottingen, Germany

　　

　　1733年初，奥地利君主、德意志神圣罗马帝国皇帝查理六世批准汉诺威选帝侯、1714年起任英国国王的乔治二世（Georg August Ⅱ）关于在格廷根建立一所大学的计划。1734年，汉诺威枢密院官员、德国大臣明希豪森（Geriach Adolf von Munchhausen）代表乔治二世创办了格廷根大学，并将由乔治二世出资开办的这所学校定名为格廷根乔治·奥古斯特大学。同年10月14日，学校开始上课，1734-1735学年的注册学生人数为147名。但是直到1737年9月17日，格廷根大学才正式举行开学典礼仪式。

　　格廷根大学的学术机构自1980年起以专业领域划分，除人文科学学院（以历史学和语言学为主）和社会科学学院（以政治学和社会学为主）外，其他院系都以一个学科为中心，开展教学与科研工作，全校共开设71个专业。该校大多数专业采取科学研究与教学并重（语言学、外国语和教育等专业以教学为主）、寓教学于科研之中的独特教育方针，现以各院系的主要科研项目为例加以说明。

　　1.神学院 下设6个教研室和一个研究所。主要研究课题是下萨克森州教会史、早期基督教作家、非圣经教科书辞典、叙利亚圣经用语索引、基督教考古学与拜占庭艺术史等等。

　　2.法学院 包括一个教研室和5个研究所。重点研究项目包括古代罗马法与古代哲学、国家理论史、刑法典史、欧洲法、比较法（劳工法与经济法之比较）、刑法程序、犯罪学、环境保护与原子能法、税收法等等。

　　3.医学院 下辖19个中心（包括多家医院），大学医院拥有1,500个床位，每年接待4万病人，在下萨克森州颇有声望。

　　4.人文科学学院 下设22个教研室和4个研究所。重要的历史与文化研究项目包括中欧和北欧古代史与文化，下萨克森州地方史，18世纪启蒙思想史，斯拉夫研究与东欧史；其人文学科研究课题是东方和亚洲语言文学与文化，文学主题与动机，人种学研究，伦理学和应用伦理学（生态学、心理治疗和经济）的基本问题。

　　5.社会科学学院 包括3个教研室和2个研究所。该院的应用学科研究主要涉及工业和管理的结构改革，现代技术的文化含义，地区分析与下萨克森州南部的选择规划，国际关系与政府比较（美国、法国和中东），教育史与美学教育，格廷根教学法总目，宣传工具与大众传播学，体操与运动史，等等。

　　6.教育学院 设有8个教研室和一个研究所，主要研究课题包括教育理论与教育哲学、社会主义史、现代社会中的儿童生活、成人教育与业余教育、旅游业与教育、计算机科学教学法、教科书与教学仪器，等等。

　　7.数学系 该系3个研究所开展的研究项目共分3大类14项，较重要的有非参数统计法、随机过程、综合分析、微分方程、代数数论、物理学中的数学问题、应用信息理论，等等。

　　8.物理学系 下设8个研究所和一个天文台。共开展9大类研究项目，如金属玻璃、非平衡超导体、重离子核物理学、听觉心理物理学、半导体结构与化学反应、量子场论、统计物理学、地磁学、日地物理学、合金结构、分子与表面物理学、太阳物理、晶体生长与材料研究等。

　　9．化学系 3个研究所分别研究电化学、X射线结构分析、非金属导体的开发、安全工艺与环境、量子力学计算、光化学、仪器分析、高压反应、激光在工业程序中的应用，等等。

　　10.地质学系 下设6个研究所，研究领域广泛，其中有工业与城市系统、应用矿物学、沉积研究、环境地质化学、水天地质学沉积研究、构造地质学与结构地质学、古生物学与地层学、环境沉积与工程地质学，等等。

　　11.生物系 9个研究所开展6大类研究项目，如植物生理学、植物生物化学、生态地植物学、动物分类学与生态学、生理学与细菌生物化学、发展生物学与遗传学、人类生态学与社会生物学、临床与环境心理学、生理心理学、组织心理与小组研究等。

　　12.林学系 10个研究所集中研究森林破坏问题，森林生态学系统平衡的必要条件，林业的自然、经济和工艺基础，热带亚热带林业，树种保护，林业与木材工业，环境与生物工艺学。

　　13.农学系 11个研究所的研究课题包括农作物生长生理学，环境对土质的影响，植物综合保护，饲料理与草地经济学，增殖生物学与耕作工艺学，现代牲畜生产的卫生问题，农业与环境政策的规划与评估，农村开发中心社经济问题，热带牲畜传染病的控制等。

　　14.经济学系 下设3个教研室和8个研究所，共开展11个科研项目，如经济理论与政策（包括收入、增长、贷币、信贷、国际经济、开发、环境等领域），福利与保健，法律与经济学、市场与消费、工业管理与商业规划，银行与金融、应用计量经济学等。

　　格廷根大学堪称德国的人才摇篮，首先是因为这所著名学府直接培育了一大批杰出人才，其次是由于在它所聘用的专家学者中有不少人为人类的科学事业作出了特殊贡献。

　　在格廷根大学培养的科学巨匠中首推医学博士科赫（Robert Koch），这位高材生作为结核杆菌、霍乱病原体和炭疽杆菌的发现者，于1905年获诺贝尔重量学和医学奖。

　　奥托·瓦拉赫（Otto Wallach）是有机化学奠基人之一韦勒（Friedrich W?hler）的学生，是格廷根大学第一个在教授任期内荣获诺贝尔奖（1910年化学奖）的学者。

　　劳厄（Max von Laue）学生时代从师数学泰斗之一希尔贝特（David Hilbert）学习数学，后来因提出了X射线衍射理论于1914年获诺贝尔物理学奖。

　　在其他毕业生中，朗缪尔（Irving Langmuir）1932年获诺贝尔化学奖，布特南特（Friedrich Butenandt）1939年获同项奖，博尔恩（Max Born）1954年和他的学生分享诺贝尔物理学奖，女物理学家格佩特迈尔（G?ppert-Mayer）1963年与人同获诺贝尔物理学奖，等等。

　　在格廷根大学学习和工作过的诺贝尔化学奖得主还有理查德（T.William Richards,美国人，1914年获奖），能斯特（W.Hermann Nernst, 1920年获奖），德拜（Peter Debye,1936年获奖），奥托·哈恩（Otto Hahn,1944年获奖），艾根（Manfred Eigen,1966年获奖），赫贝格（Gerhard Herberg,1971年获奖）。

　　“数学王子”高斯（Johann Carl Friedrich gauss）作为格廷根大学的学生（学习3年）和教授（工作48年）曾使格廷享有“数学界麦加”的称号。

　　还值得一提的优秀毕业生和著名学者是农学教授泽多夫（Wilhelm Seedorf）、地理学专家克勒佩尔（Rudolf Kl?pper）、医学教授波佩（Hanno Poppe）、神学博士齐默利（Walther Zimmerli）、化学教授和1988-1989学年在任副校长的策克（Axel Zeeck），等等。

　　韦贝尔（Wilhelm Eduard Weber）毕业于哈勒大学，但却先后在格廷根大学工作长达48年。他被聘教授后两年，便成功实现了人类首次电报通讯，后来为发展该校的物理学和化学作出了积极贡献。

　　曾去格廷根大学求学的世界名人颇多，其中就有我国的朱德同志（1923-1925年在那里学习），1984-1989年在任的原联帮德国总统威茨泽克（Richard von weizs?cker），德国柏林洪堡大学的创始人威廉·洪堡（Wilhelm von Humboldt）和他的弟弟亚历山大·洪堡（Alexander von Humboldt）。

　　格廷根大学的学生宿舍多集中在北郊新校区内，宿舍大致分为男生、女生、留学生、单身母亲与孩子、家庭式等几大类，其中“学生村”和“学生新村”以单人房间为主。学生宿舍的房租低于私房租金，但只能满足1,000多人的需要。学校设有各种风味的食堂和快餐店，其中规模最大的是中心食堂，每年供应约140万份“份饭”。
