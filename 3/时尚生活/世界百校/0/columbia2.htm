哈维里亚那天主教大学
Pontificia Universidad Javeriana

通讯地址：　
Carren 7° No.40-60, Bogotá, D.E. Colombia

　　地处哥伦比亚首都波哥大的哈维里亚那（宗座）大学（以下简称哈大）是该国创办的第一所大学，也是拉丁美洲地区最古老的高等学府之一。

　　哈大的教学、科研的系及专业有：

  1. 自然科学和技术科学

    2. 该领域包括3个系，即建筑与设计系、自然科学系和工程学系。

  2. 人文科学和社会科学

    3. 共有社会科学与教育系、法律系、社会传播系、教会法规系、哲学系、交叉学科研究

    系、神学系、宗教科学部、现代语言部、人文科学与社会科学系（设在卡利分校）等10个系（部）。

  3. 保健科学

    4. 保健科学包括医学系、牙科学系、护理学系和心理学系。

  4. 经济管理科学

哈大在波哥大和卡利各设一个经济管理科学系，开设的本科生和硕士生专业均有企业

管理、会计学、经济学、硕士生专业还有商业法、国际贸易、农村开发、非政府机构的管理与指导等。
