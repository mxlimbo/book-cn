沉醉于自己的唯心
 


    很多孩子都学我的样子，攀上高台来游戏。我坐在其中，仿佛是他们的领袖。用很欣慰和赞许的眼光看着他们小而敏捷的身体鱼贯在我的身边。
    他们此时应该还没有什么理想，但是却对于未来充满着幻想和对长大成人的一种渴望。
    从他们的眼神里，我看的出来，他们觉得我像个英雄，他们希望像我一样放纵。
    他们之所以想，是因为他们也明白我这样子是叛逆的。虽然小，但他们的头脑里依然有淡淡的观念在作祟。
    他们比我有观念的多。

    我低头吐了口烟。迷茫中，我看到了自己的脚甲上的蔻丹，红暗暗的，一粒粒的小点。有些褪色了。象青春一样。但蔻丹若是褪了颜色，可以涂上一层更艳丽的，青春就不可以了。
   
    一个孩子凑过头来看我，和我挨得很近，芳香的呼吸吹向我，我专注的看着他晶莹的肌肤。
    我曾经比他还向往未来。

    两个天使般的美丽少女，穿着短裙在广场的一角聊天，表情神秘而甜美。

    风汩汩如水，开始有些凉意了。我抬头看，猛然发现天空里连鸟都没有了。

    不锈钢的果皮箱在橘色街灯下，亮晶晶的放着寒光。象凶器。

    美术馆的广场。 ( 2000/06/27 一年前某个夏日夜晚的日记 )
