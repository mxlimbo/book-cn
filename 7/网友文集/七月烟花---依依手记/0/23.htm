奔向你

　　前几天明珠930又放了《bodyguard》，我没赶上看。第二天，有一个和我一样对这部电影发烧的网友和我说起来，心里有小小的遗憾，虽然已经看过很多遍了，可居然不厌倦，反而更留恋。

　　那年在学校，和英语系的同学一起看他们系作为功课放的原版英文电影，昏天黑地看了一下午的，就是这部惠特妮休斯顿和凯文科斯特纳主演的《bodyguard》，又译作保镖，或护花倾情。

　　那天下午，在四处拉着沉沉的猩红落地窗帘的电教教室里，完全没有中文字幕的英语对白，让人听得不是非常明白。可当最后一幕，画面上只有一个空寂冷清的机场，一架正在缓缓起动准备滑向跑道的飞机，一个身处爱情而不知如何面对只能竭力克制的女人，飞机的螺旋浆转起来，轰鸣声中，背景缓缓响起那首《i will always love u》，眼睛就没有选择地模糊起来，汹涌得直至看不清那个女人是怎样被那个会永远保护她的男人紧紧揽入怀中的。

　　其实更喜欢的还有这首《run to u》，虽然它只是剧中的插曲。第一次响起的时候，是那个身为保镖的男人在沉沉夜幕中独自面对空空的房间里满墙的投影，看着那个身为歌星万众拥戴的女人唱这首歌的MTV。男人的脸上映着闪动的光影，那个需要他保护的女人向他伸出双手，与他在两个界面里深情对视，音乐不停重复：I wanna run to you ，I wanna run to you ，Won't you hold me in your arms ，And keep me safe from harm? Iwanna run to you，But if I come to you ，Tell me will you stay orwill you run away 。那个身负保镖角色的男子，在这一刻心里是否特别想扮演一个爱人而不仅仅是保镖的角色？除了保护她的人身安全，还永远呵护她的感情？"I wanna run to u"，这仅仅是呼唤，还是一个女子发自内心的奢求？我不知道，对于那时二十岁太年青的年纪，这样的问题似乎太复杂了。

　　今天，看着一些文字，心里仿佛响起这种呼唤，突然特别想听这首歌。特地又找到这首歌下载，来回的听的是来自女人心底的声音：我想奔向你，告诉我你会停留还是逃走。多年前看完电影后的那个问题又浮上来，问自己。其实女人的内心永远是柔软和不堪重击的，如果说有不同，只是表现得不同而已。无论心流浪到哪里，总有可以牵系她的一些东西。

　　昨晚躺在床头看书，一行伤感的文字从灯下跳入眼睛：为什么在爱的时候，心里也是孤独的？合上书，我想起一句话：人可以承受孤独，但决不要寂寞。孤独可以是一种意境，而寂寞却是一种衰落。身处爱中，孤独时可以写下想念的文字，而寂寞的时刻，连心都是荒凉的。如果有天一个女子爱上了你，请求你的保护，请一定不要给她寂寞的感觉。就象席慕蓉在诗集中写到的那样："在年轻的时候，如果你爱上了一个人，请你，请你一定要温柔地对待他...... "　　

　　如果你爱她，可否有过这样的时刻。你可否曾抚摸着她的头发，听她讲述所有的梦想；可否曾在她忍不住伤痛的时候，为她抹去如雨点般坠落的泪珠；可否曾与她在宁静的夕阳里深情相拥，无语深深对视；是否与她在心灵遥相呼应的某个瞬间，不约而同地在同一段音乐沉醉；或是否在汹涌的人群中感受到她无助的眼神，从身后温柔地环抱住她；可曾情愿抛却一切矜持与虚伪，告诉她你已经是爱情的俘虏，并企求她永远不要给你自由？

　　我只想知道，听着这首歌，在这样似曾相识的时刻里，你的心是否和我一样深深地被感动过。如果有天当我奔向你，面对面站在你的面前，亲爱的你是否会为我作一次永远的停留。

依依
　01/10/2001
