Windows2000Server使用详解

 

　　2001-4-29 7:00看了一下亿唐个人主页网站统计系统，我站在社会科学→哲学类排名名列前茅，操作系统统计中，浏览“宇宙之子金星海”http://yzzz.126.com的朋友，使用 Windows 98　50.49% ，其它操作系统　28.33% ，Windows 2000　17.73% ， Windows 95　3.05% ，Windows NT　0.41% 。
　　如果您爱机的配置较新，有128M内存的话，推荐使用Windows 2000 。不推荐使用品牌机，品牌机的特点就是在最需要花钱的地方给您省钱，其性能可想而知。
　　本文使用详解是实际安装的记录，尽可能的解说详细，以方便您对照安装。
　　使用了一段时间后，系统速度会变慢，这时就考虑重装系统了。减少重装系统的一个方法是装一个反删除软件，但较烦琐，不如不装，到时直接重装系统就行了。

　           　一　重装系统前的准备
　　（一）下载最新的驱动程序。
　　如1、下载主板驱动，VIA威盛4-IN-1最新驱动4.30版For Win9x/ME/NT4/2000（2001年5月3日发布）包括IDE驱动3.011版、VIA AGP驱动4.05b版、IRQ Routing驱动1.3a版、VIA INF驱动1.20a版，新加入了反安装VIA IDE Miniport驱动的功能。
　　2、下载Leadtek丽台系列显卡WinFox配套应用程序最新1.9多语言版For Win9x/ME/NT4/2000（2001年4月18日发布）WinFox配套应用程序是Leadtek新推出的一款一流的显卡设置及功能选项调整工具。这是最新的1.9版，推荐拥有Leadtek系列显卡的朋友们更新这款工具 。
　　（二）作好系统的备份
　　如1、输入法的备份。以拼音加加系统备份为例，
　　用户除了可以单独对词库进行备份外，还可以对拼音加加系统文件进行整体备份（包括用户词库、各种自定义编码以及所有设置选项），以便在重新安装拼音加加时原有的一切设置都可保持下来。系统备份以及使用备份文件的方法是：
　　单击状态钮，打开输入法设置菜单；
　　光标指向“管理工具...”，右侧弹出子菜单；
　　单击选择“生成备份文件...”，将会生成系统备份文件，文件的名称与存放位置可由用户自行指定。
　　单击选择“从备份文件恢复...”，将可从指定的文件中恢复原有的拼音加加系统设置。
　　2、字体备份，自造字，必须备份，才能保存下来。
　　3、其他准备，如准备绿色软件。
　　所谓绿色软件，主要指不用安装就能使用的软件。有两种绿色软件。
　　（1）本来就不用安装即能使用的绿色软件。
　　如Fantasia Web Explorer2001预览版412，MyIE build 316，丁氏彩票预测系统 v，isobuster0996……
　　（2）重装系统前安装，重装后不用再安装就能使用的软件。
　　如　中国网爪最终测试版 ，DLExpert(下载专家)0.98 ，最好的浏览器之一51357最后测试版，飓风浏览器iwander210.exe，飓风邮件通320，winrar。
　　（3）windows20000server不能正常使用的软件
　　如卜算子.三颂个人人气运势分析预测 v1.2，开心助手--记录计算器450120，McAfee VirusScan 5.13(杀毒之星+洪恩毒盾)简体中文零售版……

       　　二　重装windows2000server
　　原则：能不装的，全不装，以减少空间。保存安装光盘，需要时随时可以补安装。
　　（一）确认计算机能从光驱启动。把win2k安装光盘放入光驱，开始，关机，重新启动计算机，按DEL进入cmos设置，移动光标到Advanced　Bios Features，把First　Boot　Device由HDD-0改为cdrom，ESC退出，移动光标到save&exit　setup，按enter，再y确认，退出cmos设置。
　　重启，底部出现boot from　cdrom，表示从光驱启动；press　any　key　boot　from　cdrom，告诉你，若要从光驱启动，请按任意键，就按任意键吧。接下去就从光驱启动，开始安装了。
　　（二）前期安装
　　1、setup　loading　files。
　　2、windows2000server安装程序，
　　按enter开始安装。
　　3、出现用户许可协议，按F8同意。
　　4、搜索以前版本，找到后询问，按ESC选择全新安装。
　　5、显示现有磁盘分区，按D选择删除所选磁盘分区C盘。
　　6、要求确认删除，按enter确认。
　　7、按L再次确认删除。回到5的画面，但C盘已经删除，标示为未划分的空间，光标在“未划分的空间”，按enter。
　　8、格式化所选分区，格式化类型，选择NTFS。如果是98、2000多重启动，则要选FAT。依照我的看法，不如装win2000一重启动干脆，不兼容的软件，就不用。如果有特殊用处，当然还是多重启动，因人而异。然后格式化。
　　9、安装程序正在将文件复制到windows2000安装文件夹……
　　出现“安装程序无法复制文件disk_.chm”，按ESC跳过；
　　“安装程序无法复制文件display.chm，按ESC跳过。
　　10、复制完成，自动重启，按DEL进入cmos设置，把First　Boot　Device由cdrom改回HDD-0。此时不改的话，重启时问你press　any　key　boot　from　cdrom，此时不按键盘，仍可从HDD0硬盘启动，不过迟早要改，不如现在改了。
　　（三）中期安装
　　1、安装程序正在检测和安装设备……
　　2、区域设置
　　要改变系统或用户区域设置，请单击“自定义”
　　单击“自定义”，再单击输入法区域设置，删除多余的输入法。如我，只保留了微软输入法，并添加中文简体内码。单击确定。跳出：
　　不能删除当前正在使用的中文（中国），它将在下次重新启动或注销后重新登录是被删除。
　　单击确定。系统的语言设置中增加中文繁体，若要使用微软拼音的手写板，则还要添加日语。单击确定，下一步。
　　3、自定义软件，写入姓名，单位可不写。
　　4、输入产品密钥。
　　5、授权模式。每服务器或每客户，我选每客户。
　　6、计算机名和Administrator系统管理员密码。阶级名可以是默认，如一人使用计算机，可以不设管理员密码。如设了管理员密码，就必须妥善保管密码，不能忘记，弄得自己的计算机，自己都用不了，或又要重装。
　　（四）后期安装
　　接下来安装windows组件。以我的设置为例。
　　1、internet信息服务。
　　在internet服务管理器，world wide　web服务器，公用文件，文件传输协议（ftp）服务器前打上对勾，表示选取安装。
　　2、windows media服务，不选。
　　3、附件和工具。
　　（1）多媒体，只选音量控制。
　　（2）辅助功能向导，不选。
　　（3）附件，选取代码转换工具，画图，剪贴板察看器，写字板，字符映射表。
　　（4）通讯，全选。
　　（5）游戏，不选。
　　4、管理和监视工具，不选。
　　5、脚本调试器，不选。
　　6、其它的网络文件和打印服务，不选。
　　7、索引服务，不选。
　　8、网络服务，只选简单icp/ip服务，域名服务系统（DNS).
　　9、以下消息队列服务，远程安装服务，远程存储，证书服务，终端服务，终端服务授权等都不选。
　　（五）完成安装
　　1、日期和时间设置，单击下一步。
　　2、网络设置，windows正在安装网络组件，请稍候。
　　3、正在安装组件。
　　4、正在完成最后任务。
　　5、您已成功完成了windows2000的安装，如果驱动器内有cd，请将其取出，要重新启动计算机，请单击完成。
　　6、重启，出现windows2000配置服务器，选“我将在以后配置这个服务器”，单击下一步，不选启动时显示该屏幕。
　　安装全部完成。看C盘，已用982MB。

　          　三　安装驱动程序
　　1、主板驱动
　　VIA威盛4-IN-1最新驱动4.30版For Win9x/ME/NT4/2000（2001年5月3日发布）包括IDE驱动3.011版、VIA AGP驱动4.05b版、IRQ Routing驱动1.3a版、VIA INF驱动1.20a版，新加入了反安装VIA IDE Miniport驱动的功能。
　　安装前先去INSTALL文件只读属性，再用记事本修改，把CUSTOMIZE=0，改为CUSTOMIZE=1，把DMADEF=0改为DMADEF=1。单击setup安装。
　　板载声卡的安装，单击ac97120a的setup开始安装。
　　2、安装显卡驱动。
　　我将小影霸TNT2M64，32M显存的显卡bios刷成丽台的bios，故安装丽台s32win2000驱动，一路next完成Leadtek丽台系列显卡WinFox配套应用程序最新1.9多语言版For Win9x/ME/NT4/2000（2001年4月18日发布）的安装。
　　3、modem驱动
　　我这是金网霸3623内猫。
　　开始，设置，控制面板，系统，硬件，设备管理器，单击pci简易通讯控制器，单击驱动程序，单击更新驱动程序，选搜索适于我的设备的驱动程序，选指定一个位置，单击浏览，指向驱动程序位置，单击确定，显示windows找到更好的驱动程序，单击下一步，显示“没有找到数字签名”，单击“是”继续安装。
　　4、拨号网络
　　双击桌面连接到internet图标，显示internet连接向导，选取手动设置internet连接，next；选取通过电话线和调制解调器连接，next；写入区号，确定；再确定；选取使用区号和拨号规则，写入电话号码――163，next；写入用户名，密码，next；连接名：连接到163，next；您想现在设置一个internet邮件账号吗？选否，next；要立即连接到internet，请选中……，选与不选均可，单击“完成”。
